### Post by inameiname on 2011-06-18
I know this is an older thread, but I figured I should update it. 

I figured out I could just repackage the debian package with only what I want and upload it to Launchpad (or just install the repacked deb). Its all about knowing which files to keep, and which not to. Trial and error. Not a big deal for me anymore, but at least I know. Just remember to repack using the same name, as if you don't, you might find the same files in the same packages if ever you decide to install the original. Of course, you could just copy and paste each screensaver (and all of its files) directly to the appropriate folders. Kind of a lengthy thing to do, though.

I'll be sure to mark this as solved.
