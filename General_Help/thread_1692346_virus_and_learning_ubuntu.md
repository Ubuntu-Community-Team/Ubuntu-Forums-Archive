### Post by seawolf167 on 2011-02-21
Welcome to the forums!

There are tons of references and guides out there, and Google is your best friend to find them.

One that I suggest is the [ubuntu pocket guide]("http://www.ubuntupocketguide.com/index_main.html"). It was written for an older version, but essentially all the basics are unchanged.

As for virus', they do exist, but the simple fact that Windows desktop installs outnumber *nix installs by what? 100 - 1? means that most virus' are written to target Windows. A good anti virus you can install is clamtk.

You can either install from the terminal or by searching in the software center (Applications -> Software Center -> search for clam -> then install the anti virus package)

Just to add to the info you've already been given...

  The main reason for clamtk is that you don't want to xfer a windows virus from your machine to a windows machine, or partition if you dual boot. for example, if you download a song via linux & that song has a windows virus. It would have the same effect as being shot with a cap gun. Now if you xfer said song to windows...
 Although I'm not advising it. The way I look at it is, windows has it's own anti-virus for such a case.
