### Post by Ramses de Norre on 2006-03-22
Ubuntu doesn't make software release updates between two ubuntu releases, this is because they want to make a stable OS and updating software and libraries too much will take down the stability.
Only security updates are made (and a few others that doesn't affect the stability).
It's possible though to install firefox 1.5.1.
I don't think, however, that the 1.0.7 has any security risks.
If you want cutting edge software you should update to dapper (only if you know what you're doing! It's still development) or use debian.
But I'd advice to stick with ubuntu breezy and don't worry about the older releases;)
