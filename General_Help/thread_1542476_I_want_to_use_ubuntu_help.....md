### Post by endotherm on 2010-07-30
well, first off, 64bit is not a problem. the only issue I've had with it, is trying to compile some game emulators (PCSX2 for instance). I've always had good luck with hardware support, but there is a minute possibility that you may need a driver from outside the ubuntu ecosystem, in which case it could be problematic if no 64bit copies of that drivers dependencies exist. 
these are both pretty rare circumstances. I'm not sure why they posted that, but it won;t likely be a problem. 

the ubuntu installer will walk you through creating a partition. here are some additional resources:
[http://ubuntuforums.org/showthread.php?t=282018](http://ubuntuforums.org/showthread.php?t=282018)
[https://help.ubuntu.com/community/HowtoPartition](https://help.ubuntu.com/community/HowtoPartition)
[http://www.easy-ubuntu-linux.com/ubuntu-installation-606-7.html](http://www.easy-ubuntu-linux.com/ubuntu-installation-606-7.html)

as for a tutorial, check your forums mail, and you will find a message from UbuntuGeek with a link to a "getting started" manual. there are also several great sticky's in the Absolute Beginner section of the forums. 

welcome, good luck, and have fun
