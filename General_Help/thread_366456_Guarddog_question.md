### Post by Qew on 2007-02-21
As said, reinstall Guarddog and then either set up the firewall by ticking what you want to access (DNS, HTTP, HTTPS, POP3, SMTP, etc), or go to the last tab called Advanced (sorry, this is from memory, because I currently am not using Guarddog), and check the box that says something like "disable firewall". Click apply and you should be done. Then uninstall Guarddog if you don't intend on using it.
