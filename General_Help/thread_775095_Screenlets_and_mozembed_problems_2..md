### Post by deathsshadow77 on 2008-04-29
actually I find that the latest version is pretty stable in general. Screenlets have really come a long way in the last couple months.

to get the latest version you can add it to your repositories
go to system->administration->software sources
go to third-party software tab and click the add button
copy and paste the following
```
deb http://ppa.launchpad.net/gilir/ubuntu hardy main universe
```

or 
visit this site
[http://www.screenlets.org/index.php/Download](http://www.screenlets.org/index.php/Download)
