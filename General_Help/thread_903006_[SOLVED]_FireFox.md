### Post by cooke77 on 2008-08-29
Yezinki, look via the 'K' icon, bottom Left corner.
If you installed it in Kubuntu, it will be listed......under "Internet".
Did you use "Adept Manager" to install it?
If so, it will definitely be there.
There are 2 versions...of FireFox 'available'. (Will be also listed in "Internet"....of Adept Manager. Scroll down List, to locate it. It should now be 'ticked'...since you have already installed it.)

One is FireFox 2, the other just FireFox. (This is the latest version,really.)
To un-install it, just 'untick' it...and use "Apply Changes" button...to bottom Right corner.

Print this out, if you can, to use as a 'reference-point'...to work from.

To re-install, just 'tick' it again.....then, 'apply changes'. Again.
