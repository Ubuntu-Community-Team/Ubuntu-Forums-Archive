### Post by directhex on 2012-04-23
Security patches to Ubuntu 8.10 ended *two years* ago. There have been dozens and dozens of remotely exploitable holes found in software on 8.10 since then.

You should *seriously* ask yourself if what you're doing is a good idea, because if this box is put on a network where other humans can see it, it'll be broken into faster than a Windows XP box.
