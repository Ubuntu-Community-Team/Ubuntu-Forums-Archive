Pressing Alt+Home will bring the current tab to your home screen. 

If searching is what you're after, you can also set up keyword searches so you can search from your URL bar. For example, I have a bookmark with the keyword "g" (just the letter g) that points to [http://www.google.com/search?hl=en&source=hp&q=%s&aq=f&oq=&aqi=](http://www.google.com/search?hl=en&source=hp&q=%s&aq=f&oq=&aqi=)

Now when I type anything into the URL bar starting with a g, it opens a Google search for that thing. For example, typing "g how do I fix my stupid broken Honda?" will open a Google search page for "how do I fix my stupid broken Honda?" Here's a pretty good tutorial on setting up keyword bookmarks in Firefox: [http://lifehacker.com/196779/hack-attack-firefox-and-the-art-of-keyword-bookmarking](http://lifehacker.com/196779/hack-attack-firefox-and-the-art-of-keyword-bookmarking)

Let me know if you use another browser, I think I can get it to work in all of them.

Cheers.
