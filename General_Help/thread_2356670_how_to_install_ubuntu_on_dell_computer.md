### Post by yancek on 2017-03-25
You should have a manual that came with the machine which should tell you which key to use to access the BIOS to change the boot order.  You should also see the key on boot for a few second, something like 'press F2 to enter setup'.  F2 is pretty common for Dell but it varies.

What are you going to dual-boot Ubuntu with?  You don't say what you currently have.  If it is windows 10, I would suggest reading the link below which is the official Ubuntu documentation.

[https://help.ubuntu.com/community/UEFI](https://help.ubuntu.com/community/UEFI)
