### Post by fragos on 2010-04-02
Right click on the desktop and the appearance preferences will appear with the background tab selected. Backgrounds aren't changed with themes. There is also a theme tab. Select a new theme. That should be all that's required unless you installed a special docking application. In that case you'll need to uninstall it or use it's configuration to disable it.
