### Post by xXx 0wn3d xXx on 2006-06-05
You should be able to configure a terminal and show desktop key by going under System > Prefrences > Keyboard Shortcuts. To get to a virtual terminal Press CTRL + ALT + F1-F6. You can use any key from f1 to f6. F7 will bring you back to your desktop. Shift + Backspace restarts X.
