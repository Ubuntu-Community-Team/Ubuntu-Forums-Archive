### Post by ipixel on 2005-05-16
To select 'expert' install options:

Start from the CD.
At the boot prompt, press TAB to see a list of all available kernels.

Depending on the architecture (i386, PowerPC, etc.) you will see a list of images, and some of those have the word "expert".

Type the name of the appropriate 'expert' selection - for your computer - and press return. 

The installation is basically the same, but the main difference is that at every stage you get to configure it!
