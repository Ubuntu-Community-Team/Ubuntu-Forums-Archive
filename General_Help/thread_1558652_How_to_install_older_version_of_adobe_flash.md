Are you running 64-bit Ubuntu, or 32-bit?  If you are running 64-bit, you will also need to figure out how to use the [nspluginwrapper]("http://freshmeat.net/projects/nspluginwrapper/") to use the 32-bit version with 64-bit FireFox.
