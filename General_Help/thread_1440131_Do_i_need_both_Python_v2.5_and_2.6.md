### Post by dimoftheyard on 2010-03-27
You can find out which packages need python2.5 by trying to delete it. If your system is anything like mine (Kubuntu 9.10), then you don't want to delete python2.6, because a lot of packages need it. Most of the packages requiring python2.5 on my PC seem to be part of KDE, so if you don't use that maybe you don't need python2.5
While I do recommend learning Python to anyone interested, I do not think you need python3.0. It certainly won't help you do uninstall one of the others.

That being said, normally you won't need to 'clean up' your laptop like that, because your package manager will automaticly uninstall any unnecessary package that was just installed as a dependency of other packages.
