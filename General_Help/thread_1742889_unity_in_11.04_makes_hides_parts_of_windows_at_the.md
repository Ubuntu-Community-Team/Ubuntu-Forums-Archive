### Post by sikander3786 on 2011-04-29
Install ccsm.

```
sudo apt-get install compizconfig-settings-manager
```

Start it by creating a launcher at your desktop with the command,

```
/usr/bin/ccsm
```

Make sure the Ubuntu Unity Plugin is enabled.

**[COLOR="Red"]Edit:[/COLOR]** I hope you were not talking about fullscreen windows as they reside in Global menu and the workaround mention above has nothing to do with that.
