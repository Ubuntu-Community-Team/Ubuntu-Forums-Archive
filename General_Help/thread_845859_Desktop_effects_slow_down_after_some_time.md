### Post by sayakb on 2008-07-04
Did you recently enable any plugin? If so, open ccsm (System->Preferences->Advanced desktop effects settings)
Click on **Preferences** and click **Reset to defaults**.
If you donot have ccsm installed, at a terminal:
```
sudo apt-get install compizconfig-settings-manager
```
