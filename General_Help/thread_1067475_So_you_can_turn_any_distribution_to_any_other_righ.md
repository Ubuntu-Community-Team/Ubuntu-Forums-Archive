### Post by geraldm on 2009-02-11
distributions can customize packages to be compatible with the package options that they have selected.  Also, in the software world, there are an abundance of incompatible packages.  A distribution takes a package at a particular time, and
then usually tests for compatible uses.  A small change in one package can "break" compatibility with other packages.  If you are using ubuntu, you may find compatible packages with other debian distros.  If you are going from ubuntu to fedora or vice versa, expect some problems.

### Post by snova on 2009-02-12
It's conceivable, though not for the faint of heart (or non-guru) to perhaps go from, say, Ubuntu -> Debian... but Ubuntu -> Fedora would be close to impossible.

I only say the Debian transition might be possible because they use the same package management system at their heart- so you might be able to change the repos and reinstall most/all of the packages from the Debian ones. But the Fedora one would be nearly unthinkable, because it's a completely different distro family.

And, honestly, changing from any distro to another would be more or less a reinstallation.

On the other hand, if you'd heard you can go from Ubuntu -> Kubuntu -> Xubuntu and all around, you can- but that's because they're all the same distro.
