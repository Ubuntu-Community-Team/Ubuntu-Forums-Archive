### Post by soccerboy on 2008-07-02
There are different types of PDFs.  Depending on how you make your PDFs, if you put an image in PDF, it should appear almost the same, text in PDF depends on what is installed, because inside the PDF stores the text.  Advantage over .doc is that most people have a reader for PDF and PDFs are generally non-editable.  Mostly, how a PDF shows up depends on the rendering backend of the PDF viewer.  So, PDF will look different from program to program.  PDF rendering is not exactly the same on linux as on windows but you can get pretty close if you install the adobe plugin from their website or the ubuntu repositories.  But still not exactly the same.
