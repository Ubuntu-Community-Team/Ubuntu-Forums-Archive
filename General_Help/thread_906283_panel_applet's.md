### Post by Genius314 on 2008-08-31
Panel applets... The only one I can think of is [Vista Menu]("http://www.gnome-look.org/content/show.php/Vista+Start+Menu+for+Gnome+Panel?content=71425"), which is a theme-able main menu. The default theme is based on Vista, but it can be changed to look more original.

Another menu is [Circular-Application-Menu]("http://code.google.com/p/circular-application-menu/"), which is a nice circular menu, (although not a panel applet).
