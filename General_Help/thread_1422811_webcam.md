### Post by darolu on 2010-03-05
There are a lot of IM apps that support webcams, including Empathy (Karmic's default), emesene and kopete. The problem is the version you install with Ubuntu doesn't support them because they were not considered to be stable enought when these packages (the deb files) were created.

One way to do this, is downloading any of these programs from their respective sites, compile and install it yourself.
If I recall correctly, I did read it was possible to make the Ubuntu version ones to make webcams work by installing the needed libraries only, I'm not sure though.

Edit: I just downloaded the "tarball" of emesene, it really is a binary file, webcams work perfectly, download it, untar it and run ./emesene from terminal or create a launcher:
[http://www.emesene.org/download.html](http://www.emesene.org/download.html)

The site says you can use de Debian package too.
