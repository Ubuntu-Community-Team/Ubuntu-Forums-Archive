### Post by urosg3 on 2009-06-18
Emerald theme is just window boders. You must import theme thru Emerald theme manager, System/Preferences/Emerald theme manager.

The second way is to import GTK theme if you use Gnome. Find some nice theme under GTK2 menu in Gnome-Look and install it via Appearance window.

### Post by ActiveFrost on 2009-06-18
For GTK themes, don't unzip anything ( they usually come as tar.gz ). Open Theme manager ( Desktop settings ) and click Install - browse to your newly downloaded theme file ( .tar.gz ) and select it .. should work :)

### Post by CatKiller on 2009-06-18
> **W-Unit said:**
> ...navigate to the folder I unzipped the theme to, and there doesn't appear to be any file in that folder or any of its subfolders that is recognized as a "Theme Package."

The file you downloaded was a theme package. You don't need to extract it. You can just drag-and-drop that file onto the theme manager.

Use System -> Preferences -> Emerald Theme Manager if you want to select a different Emerald theme.
