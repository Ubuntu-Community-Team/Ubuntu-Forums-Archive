### Post by UmlautBanana on 2011-07-19
I installed Ubuntu 11.04 through Wubi, and now I want to install it 'for real', with a live CD. I have a few questions.

If I install the live CD, will the Wubi installation be wiped out? I now have the option of Windows 7 and Ubuntu at boot-up; will there be two Ubuntu options after live CD installation or will there be more options under Ubuntu?

Is there any way to transfer my programs, files, and settings from the Wubi installation to the live CD one?

---

### Post by bcbc on 2011-07-19
If you install with the live CD it will be completely separate to the Wubi install (still available, but only through the Windows boot manager).

Only if you uninstall "Ubuntu" through Windows Add/remove programs, will the Wubi install be removed.

To transfer the Wubi to a partition install... see [HOWTO: migrate wubi install to partition]("http://ubuntuforums.org/showthread.php?t=1519354")
