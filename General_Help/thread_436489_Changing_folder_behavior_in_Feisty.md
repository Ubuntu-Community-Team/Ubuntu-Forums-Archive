### Post by adamthompson on 2007-05-07
When I double-click on a folder, I'd like it to open in a new window. Right now, it opens in the same window (i.e., bad, bad Microsoft behavior). I had this set up on Edgy, but I can't seem to get it in Feisty.

Nevermind; I figured it out. Within the folder, Edit -> Preferences -> uncheck "Always Open in Browser Windows". Then you have to close the window and reopen it.
