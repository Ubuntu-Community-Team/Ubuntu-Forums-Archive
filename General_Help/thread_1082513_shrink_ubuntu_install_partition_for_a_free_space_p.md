### Post by 73ckn797 on 2009-02-27
If you have the Ubuntu LiveCD you can boot into it. Go to System/Administration/Partition Editor.
From there you can select the Ubuntu partition and resize it. The remaining free space can be partitioned and formatted to whatever you want.

Back things up first, just in case. You probably will not have any problems.
