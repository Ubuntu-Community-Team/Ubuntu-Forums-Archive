### Post by orb9220 on 2007-03-29
Look in the docs and see what type of modes it supports.

Can you set it to MSC? which is a standard usb device and shows up as one.
Is it say MTP? This a special mode used by apps like Windows media player and does not show up in windows explorer as a usb device.

Amarok does have MTP support for my sansa e250. My player does allow me to change into either in my settings. MTP or MSC which is fortunate for me.

also did you look in the /media folder to see if it is there?
