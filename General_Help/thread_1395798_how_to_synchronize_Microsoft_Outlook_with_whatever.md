Evolution is a Linux Personal Information Manager that as is close an application as there is to Outook. It is maintained by Novell, also the developers of Suse Linux. Syncing data is supported through industry standard file formats and protocols, such as iCal, LDAP, CSV, WebDAV and via Microsoft Exchange Server. Syncing to a Palm Pilot is built into the Gnome desktop with Gnome-Pilot, which can even be called from within Evolution.

Some features may not look exactly the same as they are presented in Outlook, but they are there if you take the time to learn Evolution. It's not so simple that just somewhat knowing how to use Outlook will make you an Evolution master. Both are extremely complex groupware/personal information managers.

[http://www.novell.com/products/desktop/features/evolution.html](http://www.novell.com/products/desktop/features/evolution.html)

And of course there is Mozilla Thunderbird/Lightning or Sunbird. If you want to keep the learning curve easy, this would be the way I would consider. Lightning is a calendar plug-in for the Thunderbird email client. Thunderbird has many plug-ins to customize your usage and experience.

[http://www.mozillamessaging.com/en-US/thunderbird/](http://www.mozillamessaging.com/en-US/thunderbird/)
[http://www.mozilla.org/projects/calendar/lightning/](http://www.mozilla.org/projects/calendar/lightning/)
[http://www.mozilla.org/projects/calendar/sunbird/](http://www.mozilla.org/projects/calendar/sunbird/)

An alternative is Chandler, which is not supported by Ubuntu, but can easily be installed in Ubuntu. It is sponsored by the Open Source Applications Foundation.

This project attempts to re-define what a PIM or groupware should be, even allowing the download of the server application, which they call a hub. This system works cross platform with Windows, Mac as well as Linux. It's an interesting approach that doesn't strive for compatibility but promotes adoption of their platform instead.

[http://chandlerproject.org/](http://chandlerproject.org/)
