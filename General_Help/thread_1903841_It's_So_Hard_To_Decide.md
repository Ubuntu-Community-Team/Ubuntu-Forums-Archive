### Post by MARP1961 on 2012-01-03
People all have their own opinion on this subject!

I think the way the menus and settings work in Kubuntu is way too complicated. You have to search everywhere for everything!

Ubuntu have recently introduced the Unity desktop which is designed for minimalists and lovers of big , chunky icons. This said, I do prefer it to Kubuntu.

My advice would be to install Ubuntu 11.10 but then experiment with the different desktop environments within Ubuntu. It is very easy to simply install Xubuntu Desktop, Kubuntu Desktop, Gnome Shell or Lubuntu Desktop. You can install them all via the Software Centre or Synaptic Package Manager. Once installed you just log out then click the gear icon next to your user name on the login screen. Here you will see all your new desktop options as well as standard Ubuntu with Unity. You just log back into your new DE.

Kubuntu, Xubuntu, Lubuntu are all Ubuntu under the 'skin' anyway. If you have a newish computer, try Gnome Shell. If it's old and somewhat lacking in graphics then try Xubuntu or Lubuntu.

Try them all!

### Post by topsites on 2012-01-03
I don't foresee most people simply jumping from a Windows environment to a Linux one overnight just nice as can be, more so the longer someone has been using Windows the longer the process is going to take and while it could take weeks and months in some cases, I dare say for most people the conversion is likely to take a few to several years.
This isn't like riding a bicycle where in 3 days you're proficient, switching to Linux takes a commitment that for some will last a lifetime so the first thing you need are some darn good reasons why because once the conversion is complete you will not be able to return to windows, or not without considerable hassle.
That having been said Ubuntu does offer a dual-boot installation where you can run Ubuntu alongside Windows, it is this which I recommend using if you're looking to get your feet wet, having a dual-boot system is an excellent way folks can use either OS anytime to help them decide along the way, while getting used to it, and to see if it's really something you'd want.
The installation itself is fairly straight forward and to my understanding will not corrupt your Windows install, the only problems I had were no different than from a straight install and did not affect my working version of Windows although I do very highly recommend a FULL Backup!

### Post by t0p on 2012-01-03
Regarding the OP's most recent problems with a "disappearing" Kubuntu etc - I would suggest you don't use Windows to manage your partitions.  Just stick in your Kubuntu (or Xubuntu or whatever) CD and restart the computer.  Choose option to install Kubuntu, and use the installer's partition manager to decide where to install to.

If you're installing Kubuntu side-by-side with Windows on the HDD (ie dual-booting), make sure your Windows files are backed up first.  It's unlikely that anything will go wrong - but if it does, you don't want to lose all your mp3s or whatever.  In fact, backing up your files should be a regular part of your computer use, regardless what operating system you use.  Computers crash, fact.
