### Post by Jay11 on 2008-09-10
Well, I was going to stop posting but I decided to reply for anyone else thats had problems with the 7300 LE. I did a fresh re-install, pulled all the updates, then sudo apt-get install envyng-gtk, CPed the xorg.conf from my first post to /etc/X11/xorg.conf, then ctrl+alt+f1, sudo /etc/init.d/gdm restart, works perfect. All of the other methods I've tried havn't worked to well..mostly because it didn't detect my monitor from the beginning. If your monitor isn't detected automatically, use gksu displayconfig-gtk to select your monitor.
