### Post by mb_webguy on 2009-02-26
Look through your settings and see if you can't find something like "USB settings".  On my v3xx, I have Connections->USB Settings.  You should see options for Sync and Data Connection, and maybe Memory Card.  If you select Memory Card or possibly Data Connection, when you plug in the phone via a USB cable, it will be detected as a USB storage device, just like a flash drive.  Then you can simply use the file manager to transfer files.
