### Post by Frogs Hair on 2010-11-11
I can't help with the folder path , but I can help with installation. Menus can be found @ gnome look.org

1. download and save package 
2. right click the menu start button and select properties to enter menu settings
3. select the install button next to the picture of displayed  menu
4. navigate to the folder where the menu package was saved
5. select menu package click open on the bottom right of the screen
6. choose the menu and button from the drop-down  menus and select ok from the bottom right of the settings box
7. restart as needed

You will notice many pre-installed menus and buttons displayed on the drop down menus when you enter settings.
