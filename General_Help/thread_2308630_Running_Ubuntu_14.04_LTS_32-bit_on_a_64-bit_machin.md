I was gifted a secondhand laptop, with an Intel i3 processor, which implements a 64-bit architecture. The OS is Ubuntu 14.04 LTS 32-bit.

What are the effects of running a 32 bit OS on a 64 bit processor? Does this simply mean I can't use 64 bit programs, or are there potentially more serious consequences?

### Post by cbraxton on 2016-01-05
You would not be able to run 64-bit software but the vast majority is also available for 32-bit systems. Since Ubuntu includes PAE support the OS won't be limited to 4GB memory but you may get better performance running 64-bit. You would need to wipe and reload the computer to go there. So it's a judgement call really. If there is not a lot of customization and you're just starting out with it, a clean 64-bit installation is probably the ticket.
