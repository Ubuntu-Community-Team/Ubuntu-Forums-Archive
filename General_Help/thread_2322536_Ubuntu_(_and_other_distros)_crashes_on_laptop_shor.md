### Post by grahammechanical on 2016-04-28
Ubuntu Gnome 14.10 is out of support. What about Ubuntu? What version is Ubuntu? 

Often when we have the latest or close to the latest hardware we need the latest versions of Linux in order to get the latest Linux hardware driver modules and the latest proprietary video drivers. The Nvidia web site recommends driver 361.42 for that video adapter. I do not know if an older version of the Nvidia driver will support that 760M video adapter. You will need the latest version Ubuntu & its flavours to get Nvidia 361.42.
