### Post by Serenitude on 2007-11-05
It would make no sense at all to install Spybot and AVG using WINE. You are not subject to Windows viri, and installing them would just be an utter waste of time, space, and possibly (probably?) bork you system by giving Windows programs real-time control of your Linux environment. If you ARE still interested in installing AVG, though, I understand there's a Linux version available for free installation from the AVG website ;)
