### Post by prince_niceguy on 2007-11-23
When the grub option screen is displayed. Press e, then again press e. This will go into edit screen. Now, using back space delete quiet and splash option. This will allow you to show details of boot up process.

Now, press enter, press b. Boot up process will happen. It should give some idea about where the error is happening.
