### Post by kidders on 2007-04-20
Hi there,

> **Bru 88 said:**
> should I run 32 bit or 64bit?There would be no point in buying a 64-bit processor, only to force it to perpetually emulate 32-bit operation. If you'd prefer to install a 32-bit OS (which many *do* seem to prefer), you should probably buy a 32-bit processor.

> **Bru 88 said:**
> Are there any known issues with the selected equipment?The best way to be sure you're buying the right stuff is to wander around the internet for a while, and see what turns up. You're bound to find others who have made similar hardware choices. :-)

> **Bru 88 said:**
> Does going with the 8600GTS card make sense or do I not need directX10 support?I suppose that depends on what you'd prefer. Many users like to retain ongoing DirectX support, so Microsoft stuff will continue to play nice with their hardware. It's not something that would make a great deal of difference to *me* ... but don't let that influence you!
