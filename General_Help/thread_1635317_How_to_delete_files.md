Hey Dan,

The 10.10 Netbook Edition is unfortunately pretty unfinished in many ways. I would reccomend installing Ubuntu Desktop from the Software Center, then log out, find the box where you can select different desktop environments on the bottom of the login screen select Ubuntu Desktop Edition, then log in.

Anyway, to do it within the netbook edition, you need to open the Applications overlay, then search for and launch Nautilus. You can use it to delete, move and copy files.
