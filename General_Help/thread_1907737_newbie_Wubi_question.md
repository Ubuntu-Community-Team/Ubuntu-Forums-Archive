### Post by bcbc on 2012-01-12
Running wubi.exe standalone gives you the option of different ubuntu 'flavours': ubuntu, kubuntu, xubuntu etc.

To run a different release, you need the release-specific version of wubi.exe. You can find different versions at:
[http://releases.ubuntu.com/11.10/](http://releases.ubuntu.com/11.10/)
[http://releases.ubuntu.com/11.04/](http://releases.ubuntu.com/11.04/)
[http://releases.ubuntu.com/10.10/](http://releases.ubuntu.com/10.10/)  (Support ends for this in April 2012)
[http://releases.ubuntu.com/10.04/](http://releases.ubuntu.com/10.04/)
