### Post by BinaryFeast on 2009-08-17
Sometimes when that happens, you can use this: [http://sourceforge.net/projects/btmgr/](http://sourceforge.net/projects/btmgr/). Is is a boot manager. You burn it to a disk and boot from it, then in turn use it to boot from the CD.  It has worked for me when an old PC gave me trouble booting a Linux CD, despite having changed the BIOS boot options.
