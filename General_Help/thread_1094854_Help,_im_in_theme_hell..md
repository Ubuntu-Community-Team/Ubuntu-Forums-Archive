### Post by Sarai the Geek on 2009-03-13
Hmm...
In addition to what all the folks above said, make sure that you DON'T untar a theme before dragging and dropping- just drop the .tar file right in. Also, if the filename of the theme you downloaded ends with .tar.bz, change it so it says .tar.gz. 

Oh, also...
GTK = the stuff that's inside the window (buttons, icons, window color, etc) + panels & menus
Metacity, Emerald, etc = The window frame including title bar
GDM = The login screen

Hope that helps :)
