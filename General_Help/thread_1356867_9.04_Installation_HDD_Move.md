The partitioning program gparted should be able to clone your current partition to your new hard drive. You can install it with this command:

```
sudo apt-get install gparted
```

[**This walk through**]("http://gparted.sourceforge.net/larry/move/move.htm") should help you get the job done.

Fair warning: Don't do anything silly like delete or reformat your existing partition until you've copied it, or you will lose your data.

Hm, I guess you're right. Apparently you can't copy a partition that is mounted (which of course includes your system partition). I'm not entirely sure WHY that's the case, since you'd only be reading from the partition and not writing to it, but I assume it's because it doesn't want to try and copy a partition that could be changing as it copies.

In any case, a LiveCD should be able to handle it fine.
