### Post by oldsoundguy on 2009-03-26
Linux is NOT Windows.  It has a lot of it's OWN programs.

[QUOTE
Vuze (If you want this for the VIDEO download .. nope!
Bitdefender (Not needed (NO AV needed))
Daemon tools ??
iTunes (there is a Linux alternative, but not sure how well it works.)                                         
twhirl ??
Google chrome (NOT available for Linux yet)
limewire (There is a substitute.)
camtasia studios (unknown)
Advanced System Optimizer (not needed, Linux is self optimizing.)
mozilla thunderbird (available in the Package Manager .. I use it)
adobe cs4's (NOPE .. currently only Photoshop CS2 works in a program called "WINE")
winrar or 7zip (NOT NEEDED, there are different file systems in Linux but there are compression/de compression programs there. Since RAR is not just WINRAR.)
irfanview (great mid level photo program but NOT in Linux)
microsoft office (another one that can install and work in WINE)(but there is also Open Office or you can opt for Lotus Symphony)
audacity (in the Package Manager)
vlc media player (in the Package Manager)
adobe fireworks (NOPE)
everest home edition (there is a command line built into the system that will tell you your hardware installed and those green checks in Package manger will tell you what is installed.)

[/QUOTE]

Linux is an operating system .. and is NOT a SUBSTITUTE FOR WINDOWS.  If you love your Windows only games, it is NOT for you.

### Post by lovinglinux on 2009-03-26
Vuze - if you just want a torrent client, then Deluge is the best alternative in my opinion. Easy to setup, nice interface and has the most important features. If you need the video thingy, then you could use Miro, which also has torrent support (not as good as Deluge).

Google chrome - you can try Chromium [http://ubuntuforums.org/showthread.php?t=1099166](http://ubuntuforums.org/showthread.php?t=1099166)

limewire - do you really need this kind of program? Stay with torrents, is more secure. 

camtasia studios - there is gtk-recordMyDesktop and others

irfanview - digiKam is much more powerful than infranview, but if you want something simple there are other options, including Picasa

As already mentioned, some Windows programs will work when using Wine, which is a Windows emulator that allows to install/run .exe files inside Linux. But this is not the best way to go. If you can find a Linux counterpart for the program, then you should try it instead of using Wine.

This will help you find other softwares [http://linuxappfinder.com/](http://linuxappfinder.com/)
