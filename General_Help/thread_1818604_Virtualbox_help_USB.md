### Post by hyfanious on 2011-08-05
I use this method:
in ur "Synaptic Package Manager" search this:
```

oracle VM virtualbox

```
after installing go to "User and Groups"
and then click "Manage Groups"
find "vboxusers"
do double-click on it and mark ur user name (and other user that may use virtualbox)
click Ok , restart ur system
everything must work fine
