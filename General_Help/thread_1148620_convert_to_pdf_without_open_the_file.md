### Post by ikacer on 2009-05-04
no. .doc files are displayed/read differently on each computer. Unless you found a program that renders the file in exactly the same way as the original it won't look the same.
If the file was made in Microsoft Office, you can open with MO and save it as a PDF there. MO can do that if you install this plugin:
[http://www.microsoft.com/downloads/details.aspx?FamilyID=4d951911-3e7e-4ae6-b059-a2e79ed87041&displaylang=en]("http://www.microsoft.com/downloads/details.aspx?FamilyID=4d951911-3e7e-4ae6-b059-a2e79ed87041&displaylang=en")
