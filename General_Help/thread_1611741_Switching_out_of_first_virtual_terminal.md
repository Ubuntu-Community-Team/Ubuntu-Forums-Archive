### Post by jwbrase on 2010-11-02
Generally Alt-F7. Under certain conditions (if X has crashed or something) it will sometimes be on a higher virtual terminal, which on my machine at least is often number 9, in which case it will be Alt-F9.

You only need the Ctrl key if you're in a GUI. When you're at a terminal that's running in text mode, you only need Alt-F7 (or the function key for whatever terminal you're switching to).
