### Post by ELF_O8 on 2008-12-26
Can't really help much but to tell you that Beryl was fused with compiz to make Compiz Fusion. Ibex comes with the simple compiz configuration manager. If you want more, you can use ```
sudo apt-get install compizconfig-settings-manager 
``` to see if that gives you more options.
