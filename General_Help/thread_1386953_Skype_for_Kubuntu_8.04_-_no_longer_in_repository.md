### Post by chip616 on 2010-01-22
OK, a friend in the [CompuServe Linux Forum]("http://community.compuserve.com/n/pfx/forum.aspx?msg=129847.11&nav=messages&webtag=ws-linuxforum") found the solution for me:

(1) Get [skype static version]("http://www.skype.com/download/skype/linux/choose/").

(2) Create a directory and untar the file

(3) Run the binary skype from that directory.
