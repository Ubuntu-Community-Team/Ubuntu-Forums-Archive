### Post by sayakb on 2008-09-20
You could invoke the help center by pressing F1 at an idle screen (say, your desktop).
If you want a help button, right click on the desktop, select Create Launcher. Type name as **Help** and command as **gnome-help **and press OK. Now drag n drop this on the panel.
