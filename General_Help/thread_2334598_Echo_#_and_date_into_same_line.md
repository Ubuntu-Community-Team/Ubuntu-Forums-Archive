### Post by raleigh3 on 2016-08-20
How can I echo # and the current date into the same line using a script ?

This is not working.

```
echo '#' & date >> new.sh 			 		
```

---

### Post by SeijiSensei on 2016-08-20
```
echo "# $(date)"
```

That concatenates the hash mark with the string generated by the date command.  The $() construct is the equivalent of the older, now deprecated "backtick," the ` character above the tilde on the keyboard.  You need to use double quotes so the $(date) operator is evaluated and not treated literally.  The command
```
echo '# $(date)'
```
will return the string "# $(date)" since single quotes mean that the enclosed string should be treated as a literal.

---

### Post by steeldriver on 2016-08-20
Alternatively, you could incorporate the # into the date command's format string e.g.

```

date '+# %c' >> new.sh

```
