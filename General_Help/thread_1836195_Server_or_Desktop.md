> **mbaker2311 said:**
> I installed - I think - Ubuntu server.  But then I read that it comes GUI-less, and I definitely have a GUI.  Is there a way to see if your Ubuntu install is a server install vs. a desktop install?

Thanks!

If you got a GUI right after installing, you don't have server.

The primary difference between Ubuntu server and Ubuntu desktop is the presence of the 'ubuntu-desktop' meta-package. A server can be converted in to a desktop simply by installing this package. Going the other way around is a bit trickier.
