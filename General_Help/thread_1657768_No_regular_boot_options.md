As a temporary fix, at the Grub menu press "e" to edit the menuentry and remove "single" from the "linux" line. That will allow you to boot to the normal mode. Press CTRL-x to boot.

Once on Ubuntu, run "sudo update-grub" then check /boot/grub/grub.cfg to see if the normal mode is back in your menu.

### Post by drs305 on 2011-01-01
This is the point where I recommend purging and reinstalling Grub2. It's not a difficult process if you just cut and paste the commands. It will remove all the Grub2 files and then replace them. 

If you are able to boot to your normal Ubuntu installation (without resorting to a LiveCD), accomplish steps 2-5 of the following guide. You don't need to 'chroot'.

If you have to boot with a LiveCD to get to an ubuntu OS, run all the steps in the chroot section.

[http://ubuntuforums.org/showthread.php?t=1581099]("http://ubuntuforums.org/showthread.php?t=1581099")

I would recommend not using all the linux options initially. You can add them to the /etc/default/grub file once things appear normally in the menu. Of course, add the necessary options that you need to have the basic OS work during the reinstall. You might want to make a copy of /etc/default/grub and put it in your home folder for reference.
