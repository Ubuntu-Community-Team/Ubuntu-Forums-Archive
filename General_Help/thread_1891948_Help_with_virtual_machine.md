### Post by gennatolls on 2011-12-06
I think you will need to buy a new Window 7 Install DVD... the joys of windows. I don't think your laptop disk would work since you already have an install for that license. In the past when I used windows, the installer could tell if you had windows on that disk previously allowing you to buy the upgrade instead of full install (or you could do a registry hack to trick it which is supported by Microsoft itself). In your case, everything will be emulated in a virtual machine so the installer won't be able to see that on your physical HDD anyways. I know Parallels for Mac could just boot your Windows Partition from within OS X. You might check to see if VMware can do that, I use VMware and its definitely worth the money. Good Luck.
