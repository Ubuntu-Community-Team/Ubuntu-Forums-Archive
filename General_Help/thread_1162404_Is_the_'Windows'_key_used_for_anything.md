### Post by SuperSonic4 on 2009-05-17
Nope, the windows key is just an ordinary key which is assigned to the windows menu before the keyboard is shipped. On Linux it's often called the Super key but it can be assigned to a different command as a modifier like shift.

For example on my laptop:

Super + T = Terminal
Super + W = Browser
Super + F = File manager

and so on
