### Post by sisco311 on 2010-01-27
Open a terminal (Applications - > Accessories -> Terminal)
and paste the following command into it:
```
sudo dpkg --configure -a
```
press Enter to run the command. You will be prompted for your password. Type it in and press Enter again.
