### Post by whatthefunk on 2012-11-17
If youre using Kubuntu, then try sticking with the default torrent program, ktorrent.  In Ktorrent, click on the **Plugins** icon in the top toolbar and activiate **Scheduler**.  A new icon will pop up in the toolbar called **Bandwidth Schedule**.  Clicking on that will bring you to the schedule interface.  In the top toolbar, click **New Item** and set it how ever you want.  It might be best to set it to stop all torrents during the day.

Hope this helps.
