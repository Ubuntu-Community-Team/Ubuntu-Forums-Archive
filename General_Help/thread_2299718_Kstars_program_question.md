### Post by oldos2er on 2015-10-20
The setup wizard that runs when the program is first started should've asked your location. If not, you can run the 'Set Geographic Location' tool: [https://docs.kde.org/stable5/en/kdeedu/kstars/config.html#setgeo](https://docs.kde.org/stable5/en/kdeedu/kstars/config.html#setgeo)
