### Post by grahammechanical on 2016-05-01
> thats a ttf.mscorefont-installer

That is a Microsoft End User Licence Agreement (ELUA) that we have to agree to before those Microsoft TrueType fonts will download. You may have accepted the EULA on the previous version that you upgraded from but accepting the terms of the agreement is also required when upgrading to the next version. It seems.

The message to check our internet connection is the most obvious reason for a failure to download some package. But if the internet connection is fine we have to look elsewhere. And the fault could be that we have installed some PPAs that do not exist for 16.04 and so Apt is getting errors when trying to connect to the software repositories.

Run these two commands & see what errors there are. If you do not notice anything, then post the output of those two commands in this thread. Enclose the output in QUOTE tags.

```
sudo apt update
sudo apt upgrade
```

Regards.
