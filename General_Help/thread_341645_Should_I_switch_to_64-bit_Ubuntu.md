Technically, yes, all the same applications that run on a 32-bit processor can run on a 64-bit processor, but in reality, this isn't the case. Although 64-bit is up and coming, it's still not very widespread, so a lot of applications don't run properly just yet. I would stick with 32-bit for the time being. Widespread support for 64-bit processors is still a few years away, I think.
