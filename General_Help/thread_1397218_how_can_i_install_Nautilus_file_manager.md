### Post by jocko on 2010-02-03
If you really have ubuntu (and not xubuntu or kubuntu) you should already have nautilus. It is gnome's default file browser, and also the program that displays the desktop. If you see the desktop and can browse the file system you have nautilus. If not, you install it by typing this command in a terminal:
```
sudo apt-get install nautilus
```
Or open up synaptic (System-->Administration-->Synaptic package manager) and search for nautilus.
Or, to install the complete ubuntu desktop (gnome+default applications):
```
sudo apt-get install ubuntu-desktop
```
