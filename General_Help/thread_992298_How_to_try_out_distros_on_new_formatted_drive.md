### Post by Titan8990 on 2008-11-24
If you have ever installed Linux before you should have no problems at all (or any OS for that matter). 

Drop the drive in (if it is PATA you may have to configure the jumpers) and proceed to install Linux on it (assuming that it already formatted in EXT3).


I can tell you that installing a distro like Puppy Linux will be slightly more difficult but will give you a better understanding of the entire OS installation process.

### Post by snowpine on 2008-11-24
Hi there, most distros these days (including the ones you mentioned) are available as Live CDs (or from a USB thumb drive--Google "unetbootin"). These allow you to test the distro before installing to make sure a) you like it; and b) it supports your hardware.   

Once you find a distro you like, it really is as simple as running the installer and telling the partitioner to "use the entire drive." It should autodetect your new, blank hard drive. 

Xubuntu is a great choice because the forums here are the best you'll find. :) 1ghz with 512mb of ram is not that bad, so you have many options.
