### Post by lukeiamyourfather on 2014-11-25
Assuming we're talking Android here. If the phone is setup to lock then it will cut off USB transfers when it locks, or least that's my experience with a Nexus 5 phone. This can be an issue if you start a transfer that will take 15 minutes but the screen is set to turn off in 5 minutes (at which point it locks itself and cuts off the USB transfer). If that's happening either disable the lock temporarily, keep the phone awake manually, or set the inactive screen timeout to something longer like a half hour.
