hmm that im not sure on.. but you can always use "the gimp"

find the wallpaper you want and know what size it is example:

say its 1024x768
go to applications > graphics > gimp imaging

goto file > new and enter the size the wallpaper is

now fill the background the color you want, and then create a new layer on the "layers channels paths" window, theres a button for it.. then open up your wallpaper, copy and paste it onto the new layer

then use the opacity slider on the layers window to get the desired effect, save and set as wallpaper

im not sure if there is a more simple way to do it how you want it but this gives the same effect
