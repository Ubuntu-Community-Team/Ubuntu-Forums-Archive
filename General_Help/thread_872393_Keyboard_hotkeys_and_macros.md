From the top panel:
System->Preferences->Keyboard Shortcuts

Macros are usually done from the command line (aka terminal) with bash.  Bash is the default language used when using the terminal.  Here are a few links that might help:
[http://ubuntuguide.org/wiki/Ubuntu:Feisty](http://ubuntuguide.org/wiki/Ubuntu:Feisty)
