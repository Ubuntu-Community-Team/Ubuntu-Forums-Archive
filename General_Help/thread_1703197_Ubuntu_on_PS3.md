Which PS3 (fat/old or thin/new) used to have some impact on this question.

Why?

The old/fat PS3 hardware was advertised as being Linux compatible.

Yellow Dog Linux was used in universities for the PS3 clusters that they created.

The PS3 system is Linux based.

Running a distro on it is problematic due to the minimal (256MB RAM) that the PS3s run.

Some months ago Sony created an upgrade that caused ALL PS3s to not be able to run user installed Linux distros (there are multiple court cases in progress throughout the world due to this.

Less months ago, someone found & made available the security keys to the Sony PS3 system. (There are now some court cases in progress due to this as well.)

So, the bottom line is, if you do want to run a Linux distro on your PS3 for some reason (know that it will be slow & inefficient as they always are unless you are running a cluster) then you can (on the fat or thin PS3).

Have a web search & you will come up with the technical info' that you need to know pretty quickly.

Just search for "how do you install linux on a ps3" & you will find all that you need. :)
