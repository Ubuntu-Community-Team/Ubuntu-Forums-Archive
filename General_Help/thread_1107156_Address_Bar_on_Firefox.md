### Post by Rallg on 2009-03-26
This is a Firefox feature, rather than an Ubuntu feature (meaning, that you can find the answer on Firefox-related help sites).

Try this: In the Firefox URL bar, go to

about:config

You will see a warning page. Accept the risk, then locate this line:

browser.urlbar.maxRichResults

then change the default (probably 12) to 0.

There are other ways to customize Firefox, as you can see from the configuration page.
