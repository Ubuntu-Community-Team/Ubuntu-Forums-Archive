To change the scrollbar size, install gnome-color-chooser via Synaptic.

run it from System > Preferences > Gnome Color Chooser.

Go to the "Specific" tab and you will see "Scrollbars".

Tick the checkbox for Width, and set it to whatever you'd like. Mine is 10.
