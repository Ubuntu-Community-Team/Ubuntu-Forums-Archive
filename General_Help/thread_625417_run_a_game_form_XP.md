### Post by the yawner on 2007-11-28
I was able to play my WarCraft 3 installed on the Windows partition via Wine. (You just need to know the complete directory to run the .exe file) As for other windows programs, it all depends on how well they work with wine.

=====
Edit:
Just be sure NTFS write is enabled. I once played a whole campaign thinking I have saved the game because the that's what the program said. But upon checking there was no file saved, and I assume it has to do with NTFS write disabled.
