When you upgrade/install 11.04 or 11.10 you should get a panel on the left side of the screen called the Launcher. Did you not see across the very bottom of the opened Dash four more icons? They are shaped like a house, a grouping of ruler, pencil and pen, a page with the top right corner folded over, and an icon of musical notation.

That is most strange. I also wonder how it is possible that you did not have Gnome 3 installed in 11.10 because Gnome 3 is the desktop environment that ubuntu 11.10 is built upon. Up to 11.04 Ubuntu was built on the Gnome 2 desktop environment.
