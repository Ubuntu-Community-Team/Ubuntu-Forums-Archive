Storm, you said that it was set to boot from cd, so I'm not clear if you were in the bios and didn't know it, or what.  

On some computers, as you're rebooting, press F12 repeatedly while it's booting.  It should take you to the boot device menu.    Here you can set it to boot from cd.  Then either:

1) With the xp install CD in the cd rom, you might be good to go.

2)  While booting, press F8 repeatedly per blue's instructions.
