### Post by tramir on 2008-04-22
Or you can use the MozTrayBiff extension ([http://moztraybiff.mozdev.org/](http://moztraybiff.mozdev.org/)), which provides a "minimize to tray" feature combined with a mail notification icon. I personally use this together with the Mailbox Alert extension ([http://tjeb.nl/Projects/Mailbox_Alert/index.html](http://tjeb.nl/Projects/Mailbox_Alert/index.html)), for more customized alerts.

Hope this helps...
