### Post by rocksockdoc on 2010-11-21
Relatively new to Ubuntu, and would like to find the Ubuntu equivalent of the "open terminal window here".

The desired use model is to navigate in Nautilus to a particular folder, and then right click to select a menu item "Open terminal window here".

The terminal would open up to that directory for command line actions.

---

### Post by oldos2er on 2010-11-21
Install nautilus-open-terminal. You'll need to either logout and log back in or reboot to activate it.

### Post by oldfred on 2010-11-21
If I am wanting to edit a file from sudo I just right click, open with other applications, Use custom command and type in gksudo gedit. Then next time it is one of the commands available and I do not have to retype.

To change to a folder in terminal I just copy and paste the path, cntl -l (el) to convert path to copyable string and type cp space, then cntr, shift v in terminal.

### Post by WienerWuerstel on 2010-11-21
You could try [Nautilus Terminal]("http://www.ubuntugeek.com/nautilus-terminal-an-integrated-terminal-for-nautilus.html"). With Nautilus Terminal you have a Terminal in every Folder you want, without going into the Gnome Terminal.
