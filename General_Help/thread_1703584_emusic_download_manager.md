### Post by stef3 on 2012-03-22
emusic's own FAQ suggests this java based download manager:
[http://www.kallisti.net.nz/EMusicJ/Download](http://www.kallisti.net.nz/EMusicJ/Download)
see: [http://www.emusic.com/info/help/downloading/#dlm56](http://www.emusic.com/info/help/downloading/#dlm56)

I am running Ubuntu 11.10 64 bit, and the installation was smooth: I just double clicked on the file, told the archive manager to extract everything into a folder I created, navigated to the folder in nautilus, right clicked on emusicj, in the properties I checked the executable box, then double-clicked emusicj to run it.
Done, deadline beaten, thank you kallisti
