### Post by Mike'sHardLinux on 2006-11-29
In Synaptic, you can right click on the package you installed, choose properties, then click on the Installed Files tab, and see all the files that were installed. The binary(ies) will generally be installed under /usr/bin. You should be able to easily figure out which file is the one you type at the command-line to run the software.
