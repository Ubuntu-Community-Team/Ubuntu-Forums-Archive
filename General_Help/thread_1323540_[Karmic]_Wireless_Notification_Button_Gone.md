### Post by ramjet_1953 on 2009-11-11
First up, try this:

1. Navigate to [COLOR="Blue"]System>Preferences>Startup Applications
[/COLOR]
2. When the window opens scroll down the list looking for [COLOR="Blue"]Network Manager[/COLOR].

3. If it is in the list, make sure there is a tick in the box for it to start on your login.

4. If it wasn't ticked, after putting-in a tick, log out and back in again.

Hopefully, this will cure your problem.
If Network Manager isn't in the list you will need to add it.
