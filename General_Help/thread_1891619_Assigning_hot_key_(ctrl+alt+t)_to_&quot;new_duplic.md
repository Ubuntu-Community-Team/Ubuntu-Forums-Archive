,except that instead of assigning the hot key (ctrl+alt+t) to the command "new terminal" (current default), I want to assign the hot key to the command "new duplicate tab", which is the hot key used by Tab Mix Plus for the Firefox extension on Windows. However, on Ubuntu, this hot key is assigned to "new terminal". How can I reassign it to the command "new duplicate tab?

### Post by claracc on 2011-12-06
You can find the key combinations in the menu: system --> preferences --> keyboard combinations . (Lucid Lynx 10.04)

In the window popping up you have a list of action and key combination, there you can change them as you want.
