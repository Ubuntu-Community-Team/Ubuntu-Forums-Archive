### Post by sirjoebob on 2009-01-24
I am assuming you have the compiz config settings manager (CCSM) installed. if so, go to system>preferences>compizconfig settings manager.

Go under the "Desktop cube" and there is a "transparent cube" tab. change the settings under there. Alternatively, you can Google things like this with good success.

### Post by sirjoebob on 2009-01-24
select the check box for "transparency only on mouse rotate"

If I understand your question correctly, this should solve your issue. It makes the cube transparent when rotating but opaque when sitting still.
