### Post by dupuy on 2007-10-12
I posted the following in another thread ([http://ubuntuforums.org/showthread.php?t=189300](http://ubuntuforums.org/showthread.php?t=189300))

> You need to have Kopete 0.12 or later, and you'll need to know the name of the chat room and exchange number (invites to chat rooms aren't supported). What's an exchange number? I guess it's something about the AOL server supporting the chat. I found out the exchange number by using Pidgin (ex-GAIM) to connect to the chat room; somehow that client can figure out the exchange all by itself.

Once you have these, you select Join Chat... from the AIM menu, just like it says in the manual. What, you can't find the AIM menu? Neither could I, but it requires a right click on the little AOL buddy icon at the lower right of the window. Having finally done this, it doesn't seem to do anything much (there's a dialog box where you enter the chat room name and the exchange). However, after figuring out what the exchange for the chat room was, it did *finally* work for me.

Not exactly what I'd call easy to use, but I'm just glad it does actually work

I have no idea how you would make an AIM chatroom with kopete, but you can at least join one, following the above instructions.  You'll need to use GAIM/Pidgin to get the exchange to use.
