### Post by Impavidus on 2020-07-13
??

The installation procedure is:
- Make backup copies of all your important files on an external drive and safely remove and unplug the drive.
- Download the Ubuntu .iso file of your choice and save it to a temporary location.
- Check the integrity of the .iso file (can be skipped if downloaded via torrent).
- Create a live disk, usually a small usb drive, with a specialised tool. A dvd is possible too, if you like the classical way.
- Boot the live disk and make sure everything works.
- Partition the hard drive (unless you want the installer to do so automatically).
- Install.

See: [https://help.ubuntu.com/community/Installation](https://help.ubuntu.com/community/Installation)

### Post by Impavidus on 2020-07-14
If you can't boot the computer from the external drive, that can be because you didn't select the right drive to boot from, the usb drive is not plugged into a port that the computer can boot from (some computers are picky, so try a few ports) or the live disk was not properly prepared and not recognised as bootable. See this tutorial on creating a live disk on Windows: [https://ubuntu.com/tutorials/create-a-usb-stick-on-windows#1-overview](https://ubuntu.com/tutorials/create-a-usb-stick-on-windows#1-overview)
