### Post by tgeer43 on 2009-09-03
I'm not familiar with those applications but if they were installed from the repositories either through 'Add/Remove' or 'Synaptic Package Manager' then they will update through 'Update Manager'.

If you downloaded and installed them yourself then you have two possible options. If the maintainers have a repository then you can add it to your Software Sources and then you'll be able to update through the Update Manager. If not then you will need to download and install the newer version. Uninstallation of the old application is sometimes required but usually not.
