### Post by realzippy on 2010-06-26
Here is a short HowTo:

Get Enemy Territory (needed for TCE):

```
wget -c http://ftp.games.skynet.be/pub/wolfenstein/et-linux-2.60.x86.run
```


Get the patch:

```
wget -c http://ftp.games.skynet.be/pub/wolfenstein/et-linux-2.60-update.x86.run
```

Install ET:

```
sudo sh ./et-linux-2.60.x86.run
```

Install the patch:

```
sudo sh ./et-linux-2.60-update.x86.run
```

Get TCE:

```
wget -c http://download.softpedia.ro/dl/089a6cbf31e75f50073ef397c16d0e10/4c264d31/500019968/linux/games/true.combat.elite_0.49-english.run
```

Install TCE 

```
sudo sh ./true.combat.elite_0.49-english.run 
```

Do not start the game when asked in the installer!
Start game:

```
tc-elite
```

or use menue.

Edit:
Sound does not work,didn`t mention first 'cause watched soccer.Seems to be a known issue:
[http://ubuntuforums.org/showthread.php?t=5246&highlight=territory&page=37](http://ubuntuforums.org/showthread.php?t=5246&highlight=territory&page=37)
