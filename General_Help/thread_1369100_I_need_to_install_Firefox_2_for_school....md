### Post by rainbowagent7 on 2010-05-31
I resolved the issue by doing a fresh install of Hardy. In  Applications>Other>ScreensandGraphics you can click on the monitor  icon to the right of "model" and it gives you a list of makes. Mine  wasn't on there of course (I'm using a Dynex, which was probably made  after Hardy was released), but I was able to set it to "Generic LCD  Panel 1024x768", rebooted and it worked like a charm.
