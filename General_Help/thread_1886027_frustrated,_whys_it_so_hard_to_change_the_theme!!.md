### Post by jjex22 on 2011-11-24
Hi there and welcome to the forums! Unfortunately the user interface you have at the moment is unity- you may have seen this in lots of posts. Whilst it is a very intuitive interface, for those of us who like things exactly how we like them, it's about as customisable  as a pencil sharpener, which is why I only log into it basically to help people with Unity issues.

The great thing about linux is that if you don't like a user interface, you can change it, and there's LOTS out there. 

Most people getting annoyed with Unity move to Gnome shell - this is the windows manager behind Unity,and is far more customisable - you can get it from the software centre.

For a desktop that's I suppose out of the box closest to modern windows (ignoring personalisation) Then I'd go for KDE - again downloadable from the software centre, this is my personal favourtie, but that's just preference - it's also the heaviest on resources - I wouldn't recommend it on a machine with less than a  gig of ram. 

The final one i'd chuck in is LXDE - again from the software centre, this is very resource light, very configurable and functional - it's by far the quickest, but it doesn't have the visual effects of the others.

There's of course a lot more out there but these are easy to get and install from the software and are the most supported on these forums, so a good place to start for the beginner. To switch between them you just log out and select them from the cog next to the sign in box on the log in screen.

Try **Gnome-Tweak-Tool [Advanced Settings]**, it makes theming a bit easier. Remember latest Ubuntu_Unity uses Gtk 3 engine, being relatively new, isn't as customizable in terms of themeing as was Gtk 2. Still Advanced Settings helps. 

You can find themes for Gtk 3 [**here**]("http://gnome-look.org/").

To install Gnome-tweak-tool run the command below in the Terminal:

```
sudo apt-get install gnome-tweak-tool
```Hope this helps
