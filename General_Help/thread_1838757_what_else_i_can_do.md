### Post by sisco311 on 2011-09-04
You can't change the permissions of a file located on a filesystem which does not support Unix file permissions, like NTFS or FAT.

EDIT: BTW, if it's a script then you don't need execute permission to run it, you only need read permission:
```
sh ./patch.sh
```
