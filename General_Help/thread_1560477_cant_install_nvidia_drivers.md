### Post by pania on 2010-08-25
I am using nVidia driver 256.44 but I installed by using the ppa from XUpdates, this is the page where you can get the details.

[https://launchpad.net/~ubuntu-x-swat/+archive/x-updates]("https://launchpad.net/%7Eubuntu-x-swat/+archive/x-updates")


 Click on Technical details about this PPA and you will see the information about the ppa add this to your software sources list and then update. Make sure to read all the details first, I have been using this driver with lucid 10.04 for about 2 weeks without any problems.
