Thanks.

a) MBs can vary wildly with price, just make sure it has all the features you need - ones with inbuilt NVIDIA graphics are good as Ubuntu supports them.

b) Wireless cards can be PCI - or these days USB devices are popular (check the forums for models that are supported)

c) 64 bit processors must go in a 64 bit MB, then you can install 64 bit Ubuntu to get better performance than with 32 bit (normal) Ubuntu. Almost all apps are now available in 64 bit (but may not yet perform any better). There is a whole sub-forum on this subject here.

My PC has a 64 bit MB (with inbuilt NVIDIA graphics) and I run a 64 bit Sempron CPU, but I still use 32 bit Feisty because there are less "complications" than currently with the 64 bit version (I am awaiting Gutsy to be released to see if things are better).
