### Post by Scott753 on 2009-11-22
If you mean, would you be able to install Windows 7 to a separate partition while having only the Ubuntu OS, I am pretty sure that is possible. The option should come up when you insert the Windows7 installation CD.

However, installing Windows will create a whole new set of problems, namely, that you will have to reinstall grub afterwards, as in:

[https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows](https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows)
