Hi. I'm not really into localization, but at first glance it seems obvious that you have DE locale set for hda1, but not for hdb1 and that's the reason you are having problems. Have a look:
/dev/hdb1 /media/host ntfs-3g defaults,nls=utf-8 0 0
/dev/hda1 /media/50_01_62 ntfs-3g defaults,locale=de_DE.UTF-8 0 0

Notice the "locale=de_DE.UTF-8" part? You should have this for hdb1 as well and you should be good then.

Hope this helps.
