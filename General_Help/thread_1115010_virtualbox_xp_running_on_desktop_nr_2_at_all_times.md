### Post by zekica on 2009-04-03
In theory, it is possible to run games inside virtual machine with a slight performance loss.

However I haven't heard of any virtualization software that supports acceleration of DirectX API, so the short answer will be no. VirtualBox currently supports accelerated 3D for OpenGL applications/games. I haven't tested this, but it should work.

You should setup a dual boot system, but I know that it is inconvenient - you have to close all applications whenever you want to play a game.
