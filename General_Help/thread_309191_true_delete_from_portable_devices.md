### Post by Mirielle on 2006-11-29
I had the same problem. It can be fixed easily:

Start Konqueror.

Settings > Configure Konqueror... > Behavior

Tick this option: Show 'Delete' context menu entries which bypass the trashcan

I suppose you would've found it yourself without much effort, but anyway. :) You could also just press Shift + Delete.
