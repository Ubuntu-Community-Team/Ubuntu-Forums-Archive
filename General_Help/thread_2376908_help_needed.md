### Post by ajgreeny on 2017-11-07
Depending on how long ago it was that you last used Ubuntu you may be familiar with synaptic, the original package manager of Debian and thus Ubuntu when it started.

Synaptic is still available and many users, me included, believe is better than any other GUI package manager available for Ubuntu.
So if that name is something you recall, give it a try by installing it and using it rather than whichever of the software centre apps you are now using.
