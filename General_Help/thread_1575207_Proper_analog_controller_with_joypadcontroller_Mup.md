### Post by sendblink23 on 2010-10-16
I've just tested on Ubuntu 10.10 x64 using the Mupen64plus v1.5 provided by Ubuntu Software Center
And it works 100% fine, i have just tested all these games with both analogs: 

Perfect Dark - my controller works exactly like this: [http://www.allgame.com/game.php?id=26461&tab=controls](http://www.allgame.com/game.php?id=26461&tab=controls)
The Legend of Zelda - Ocarina of Time
GoldenEye 007
Banjo Kazooie
Mario Kart 64
Super Mario 64

My games are all US versions

With my PS2 usb controller, are you sure you configured correctly your controller?

just incase the way I set my controller...
*D-pad - exactly as N64 was using the D-pad
*Analog - exactly as N64 was using it *Left side Analog (Y - up, down - X left, right)
*C buttons - I used my right Analog for them(C-up - up, C-left - left, C-down - down, C-right - right)
*Z button - L2 button of PS2
*L button -  L1 button of PS2
*R button - R1 button of PS2
*A button - X button of PS2
*B button - Square button of PS2
*Start button - Start button of PS2

On my controller I have to press the "Analog" button(on my controller its between Select & Start buttons) to use both analogs.. so if you have that button.. USE IT! before making the buttons configurations. So eitherway its suppose to work on your controller as well this configuration... if it doesn't then your controller is not fully compatible... go to the creators website to see what you can do: [http://code.google.com/p/mupen64plus/](http://code.google.com/p/mupen64plus/)
