### Post by tszanon on 2007-06-07
Maybe you can get what you want by binding the Meta key to the Win key.
It's under System > Preferences > Keyboard, then Layout Options tab > Alt/Win key behavior, set Meta is mapped to Win keys. Maybe that'll let you set a shortcut.
