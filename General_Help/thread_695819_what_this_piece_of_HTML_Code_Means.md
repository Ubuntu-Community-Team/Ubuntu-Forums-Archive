I'm pretty META tags are there to be read primarily by spiders. They can be helpful but they're probably not the only thing taken into account. Anymore, the content seems to be more important. Especially having a relevant title and a solid first paragraph will be important, but it's also good to have solid META tags ... never hurts anyway.
