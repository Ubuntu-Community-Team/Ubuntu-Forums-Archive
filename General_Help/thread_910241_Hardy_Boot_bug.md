### Post by mlentink on 2008-09-04
1. Don't Panic!
2. It is unlikely that this is in fact a bug, though it could be. Did you install software or change system settings before this started to appear?
3. You should be able to boot into recovery mode. When you start up your machine, it comes up with the GRUB-menu (GRUB aka The GRand Unified Bootmanager), which will allow you to chose to boot your normal kernel or, almost always directly below that, recovery mode. Have you tried this already?
