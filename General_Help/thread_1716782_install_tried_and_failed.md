### Post by seawolf167 on 2011-03-29
Welcome to the forums!

Please check a couple things:

[LIST=1]
[*]Ensure that the cd/dvd is flagged as "bootable" (for example - download the .iso file, then use ImgBurn [on Windows] to "Write Image File to Disk", don't just burn the .iso on a cd as a file)
[*]Ensure that BIOS is setup to boot from your cd/dvd drive (during POST, you should see a message saying "To Enter Setup, Press [SOMEKEY], where SOMEKEY is usually [F2], [F8] or [DEL])
[/LIST]
