### Post by yancek on 2014-09-25
I imageine you have seen link below which takes you to a page explaining how to install wine-pipelight in Ubuntu/Mint and at the bottom is another link "enabling plugins" which explains how to enable each plugin.  If you are running this from an Ubuntu/Mint DVD, you can do it if you have enough RAM.  If you reboot the DVD, you lose everything and have to do it again each time you reboot.  The Live medium is read-only and nothing is saved on reboot.  You could use software such as unetbootin to put a bootable Ubuntu on a flash drive and select the persistence option and you would then be able to save changes within the limits of the persistence file, 1-4GB persistence should be enough.

[http://pipelight.net/cms/install/installation-ubuntu.html](http://pipelight.net/cms/install/installation-ubuntu.html).
