### Post by charlie cat on 2012-06-18
This works on my system:
Go into System Settings -> Keyboard Layout.  In the bottom right corner there is a button saying "Options".  In the window that pops up, expand the "Key(s) to change layout" node.  From there, you should be able to set your key combination.
