### Post by coffeecat on 2015-09-19
You can install the voice recognition app in Google Chrome: [https://chrome.google.com/webstore/detail/voice-recognition/ikjmfindklfaonkodbnidahohdfbdhkn/related?hl=en](https://chrome.google.com/webstore/detail/voice-recognition/ikjmfindklfaonkodbnidahohdfbdhkn/related?hl=en)

It works just fine, but I don't think there's a way to dictate directly inside another app. You have to export your text file, but you don't have to use the cloud if you don't want to. It offers the option of a download of a text file to your computer if you prefer.

If you need to install Google Chrome, go here and download and install the deb which will add the google chrome repository to your sources for updates: [https://www.google.com/chrome/browser/desktop/](https://www.google.com/chrome/browser/desktop/)

I managed to install the Voice recognition app in Google Chromium as well, but I haven't worked out how to allow access to the microphone in chrome://settings/contentExceptions#media-stream as the app is telling me. To be honest I didn't try too hard as it is working for me in Chrome, and allowing microphone access was simply a case of OK-ing a popup in Chrome. Perhaps someone else can advise how to set up Chromium for Voice Recognition if you prefer Chromium over Chrome.
