### Post by Tamlynmac on 2010-12-06
Guess the answer lies in what type of document you wish to scan - text,  image, both, what app(s) your planning on using to edit it with and what  format you plan to save the file in after editing. For example, you  could save a written document in text and open/edit it in OO, then  export it as a pdf. Or, by installing pdfedit from the Synaptic package  manager you can edit a .pdf file.

Perhaps, a little more information regarding edit apps, what type of  documents are being scanned and the final desired saved format - might  help to better answer your question.
