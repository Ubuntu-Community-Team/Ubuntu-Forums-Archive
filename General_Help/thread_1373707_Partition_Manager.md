### Post by lmarmisa on 2010-01-06
Gparted is the program that you need. Download it from Synaptic (System -> Administration -> Synaptic Package Manager).

If you need to manage not only GNU/Linux partitions but also NTFS partitions, download the package *ntfsprogs* too.
