:confused:  Much to my chagrin I am unable to install the desktop themes that I downloaded. They have file names ending in ".tar.gz". How do I do this?

---

### Post by ad_267 on 2009-08-01
Go to System - Preferences - Appearance and drag and drop the theme file onto the window or click the install button. A .tar.gz file is a compressed archive, think of it as what a .zip file is in Windows.

Sometimes that method might not work if the file isn't structured correctly. If so you can right click on it and extract it and see what's inside.
