### Post by fragos on 2007-12-25
FSB stands for Front Side Bus.  Faster is always better but there are many factors that impact performance.  For example 1GB of memory is much faster than 256MB because it makes the use of swap space on disk rare.  You mostly see the difference when changing between applications.  I'm not sure a super fast FSB would ever yield a performance difference you'd easily notice as a user where.  Some speed enhancements show up better when measured rather than when experienced.  Remember that with laptops, faster can mean shorter battery life.  My 1420n has an Intel Core 2 duo T5250 1.5GHZ, 667MHZ FSB, 2MB L2 Cache.  There were faster CPUs available but I stuck with the basic package and it performs very well.
