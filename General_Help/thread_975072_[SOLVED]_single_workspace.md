### Post by amitabhishek on 2008-11-08
I find multiple workspaces extremely irritating, is there any way to have single workspace a la Windows? Please let me know.

---

### Post by the8thstar on 2008-11-08
Right-click on the workpace icon on the bar; some preferences menu should appear with columns and lines. Bring them all to value 1. You'll end up with only one workspace.
