### Post by Rocket2DMn on 2008-04-01
Simple Backup is under System->Administration->Simple Backup Config
Sometimes people have problems not seeing new programs appear in these menus, so restarting usually helps with that.
Alternatively you can run it from the terminal with
```
gksudo simple-backup-config
```
