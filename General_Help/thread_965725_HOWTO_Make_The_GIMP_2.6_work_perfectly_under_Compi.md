### Post by Qrjusz on 2008-10-31
As I haven't seen a solution for this one nowhere in the net and on this forum I decided to post it here, maybe somebody will profit from my solution. I know that it's not perfect but since it works...

1. Run The Gimp
1.1 Palce the "Layers" and "tools" windows so that they're entirely inside the main GIMP window
2. Run Compiz settings manager (right click the Compiz fusion icon)
3. Select the "Window decoration" menu
4. Next to the line "decoration windows" there is a little green +. Click it. And then click the "tools" window of your GIMP
5. repeat it with Layers window
6. Now do the same but with the "shadow windows" line for both windows.
7. et voilà both windows should now stay on top all the time
