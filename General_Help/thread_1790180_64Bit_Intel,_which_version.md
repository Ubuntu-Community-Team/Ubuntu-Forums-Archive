### Post by CharlesA on 2011-06-24
AMD64 is just the name of it, like i386 is intel's name.

It'll run fine on any 64-bit CPU.

### Post by jfreak53 on 2011-06-24
So I'm fine leaving the current i386 installed then?

---

### Post by CharlesA on 2011-06-24
If it's working fine, you can just leave it.

Ubuntu will use a PAE kernel if you've got more then 4GB of memory, so if you don't really have a specific need to upgrade to 64-bit, you don't have it. :)
