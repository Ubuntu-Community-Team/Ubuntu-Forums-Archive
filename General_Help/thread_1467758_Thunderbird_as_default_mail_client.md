### Post by boluc91 on 2010-05-01
Had the same problem...

This Thunderbird plugin seems to fix it:
[https://launchpad.net/libnotify-mozilla/experimental-indicator/alpha1](https://launchpad.net/libnotify-mozilla/experimental-indicator/alpha1)

You have to right click->save as the xpi file and then open it with thunderbird, otherwise firefox will try to install it as a firefox plugin.
