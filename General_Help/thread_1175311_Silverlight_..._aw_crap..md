### Post by directhex on 2009-06-01
To give a little more detail:

The current stable version of Silverlight is 2.0. The current stable version of Moonlight is 1.0. Moonlight 1 can view Silverlight 1 sites. Moonlight 2 is needed for Silverlight 2 sites. The Moonlight team are now at the stage where they have a functional pre-alpha for Moonlight 2, but it's major work to implement.

My recommendation if you want to help is to use the latest preview xpi ([http://go-mono.com/moonlight-preview](http://go-mono.com/moonlight-preview)) and file a bug on Novell bugzilla about sites that fail to work.

Oh, and before anyone says it: Moonlight 2 already includes some Silverlight 3 features.
