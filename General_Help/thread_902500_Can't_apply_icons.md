### Post by FuturePilot on 2008-09-15
It's an old theme. Starting with Gnome 2.20 they changed some of the icon naming conventions meaning that if you were to use an older theme that was designed for a Gnome version prior to 2.20 the name of the icon that it is looking for won't be available. Then it will just fall back to the default Gnome icons.
