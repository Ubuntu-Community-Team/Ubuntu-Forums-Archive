### Post by geirha on 2008-10-24
I assume Mac4Lin is a theme?  Themes for ubuntu may be a bit confusing. There are different types of themes; metacity themes which changes the window borders, gtk themes which changes the inside of the windows (buttons, dropdowns, scrollbars etc.) and there are icon themes which changes how the icons look. Some theme packages contain all three types of themes, changing "everything", but some only contain two or one type of themes. You're probably lacking an icon theme for that one.

### Post by northern lights on 2008-10-24
Check on [http://www.gnome-look.org]("http://www.gnome-look.org") and download some icon theme you like. Navigate to "System > Preferences > Appearance". Open the folder you download the theme to. Grab the theme file (most likely *.tar.gz), right-click-hold it and drag and drop it into "Appearance". It will now be installed. Thereupon select "Customize theme" and change the icon part.
