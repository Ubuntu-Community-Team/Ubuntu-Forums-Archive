### Post by marshag63 on 2010-09-06
How about Slitaz?

Video of Slitaz running on 450 MHz computer with 256 mb Ram.
[http://www.youtube.com/watch?v=0EYC2eBqKXs](http://www.youtube.com/watch?v=0EYC2eBqKXs)

Web Site:
[http://www.slitaz.org/en/](http://www.slitaz.org/en/)

You can either take the harddrive out and install as mentioned above or you can use Plop boot manager run from a floppy and then you can use a USB drive.  I believe you can download all packages you need with another computer and save to a USB as well.  

See:  7. Running without harddisk installation from Floppy, CD, Windows boot menu, Syslinux, LILO, GRUB, network

[http://www.plop.at/en/bootmanager.html](http://www.plop.at/en/bootmanager.html)

Run from Floppy with Disk Image:  [http://www.plop.at/en/bootmanager.html#runflp](http://www.plop.at/en/bootmanager.html#runflp)

Plop boot manager is a little bit of work, but once you setup your floppy, you can boot any old computer with a floppy drive once you set the BIOS to boot from floppy.  

Hope this helps :)

MarshaG.
