Link to bug report.

If you lose connection to the internet because Firefox is losing your IP, it appears the ubuntu firefox modifications 0.9rc2 add-on is causing the problem. I recommend disabling it so you can connect normally without having to sudo dhclient every time you restart Firefox.

This bug currently affects my cable internet connection.
