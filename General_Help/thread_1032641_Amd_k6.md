If fast performance is your #1 goal, then yes, Fluxbuntu is quite a bit "lighter" than Xubuntu. Here's a thread where I share my experiences with it: [http://ubuntuforums.org/showthread.php?t=855596](http://ubuntuforums.org/showthread.php?t=855596)

If security is your #1 goal, Fluxbuntu is not an "official" branch of Ubuntu, and has not received any updates/bug fixes in over a year. I would agree with 3rdablum's recommendation of the latest version (8.10) of Xubuntu, in this case. 

There is some security risk involved any time you are connected to a network or the internet. However, most users on this forum would probably agree that Linux is quite a bit safer than Windows in this regard. Viruses that are written to exploit Windows security flaws should not affect a Linux system.
