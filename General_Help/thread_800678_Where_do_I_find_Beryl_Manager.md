### Post by Sef on 2008-05-20
Beryl is unsupported.  Instead use Compiz-Fusion, it comes in Hardy Heron and Gutsy Gibbon.  You can download the manager from Add/Remove.  Applications > Add/Remove > Change "Show" to 'all open source applications' or 'all available applications' > search ccsm > check the box > apply changes > apply > close.

Note: 'all open source applications' = Universe, the applications are only open source; or 'all available applications' = Multiverse, the applications are both open source and proprietary.

To set up Compiz-Fusion, read [Forlong's Blog]("http://forlong.blogage.de/article/2007/8/29/How-to-set-up-Compiz-Fusion").
