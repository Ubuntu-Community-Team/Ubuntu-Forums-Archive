### Post by herald on 2007-08-18
> **ironmatar said:**
> i tried puting the cd back in but it now does not start from cd either which is my problem  so i need a way to format so it will or reset boot device somehow.

Most BIOS these days allow you to press a key to enter a boot menu.  This should allow you to bypass the hard disk and force the CD to install.  If this doesn't work, check the jumper settings on your HD and your CD rom to make sure they aren't both set to slave or master or whatever.  If you have a cable that has one black connector and one grey connector (or the connectors are different colors) try setting the drives to cable select.
