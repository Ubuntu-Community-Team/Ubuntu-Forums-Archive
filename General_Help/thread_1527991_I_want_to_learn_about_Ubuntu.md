### Post by kyleabaker on 2010-07-12
I think your request would be more appropriately placed in the [General Help]("http://ubuntuforums.org/forumdisplay.php?f=331") section (maybe a mod will move it for you).

To really break down how the Operating System works, and understand it, requires some knowledge of programming, but I think you can get most of the history and technical information without having to understand much about programming.

Some reading I would suggest for you is the following (in order):
[http://en.wikipedia.org/wiki/Ubuntu_(operating_system](http://en.wikipedia.org/wiki/Ubuntu_(operating_system))
[http://en.wikipedia.org/wiki/Debian](http://en.wikipedia.org/wiki/Debian)
[http://en.wikipedia.org/wiki/Linux_kernel](http://en.wikipedia.org/wiki/Linux_kernel)
[http://www.freeos.com/articles/3102/](http://www.freeos.com/articles/3102/)

If you're asking about learning a programming language Ubuntu is written in, you'll be starting way over your head. But just to be informative, Ubuntu is written in a lot of different languages. Different programs are written in different languages such as C, C++, Python, etc. The answer to this type of question can be found here:
[http://ubuntuforums.org/showthread.php?t=705754](http://ubuntuforums.org/showthread.php?t=705754)

Good luck and happy reading. ;)
