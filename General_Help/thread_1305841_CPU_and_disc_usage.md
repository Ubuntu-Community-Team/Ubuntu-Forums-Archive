I dont have the good solution, but I can advice perhaps two things
First, register for launchpad, confirm the bug (change the status) and post your findings and what printer you have, etc.  If no one reports it, it aint gonna be fixed.

Secondly as a workaround, disable cups from starting in system > administration > boot-up manager.

Start it manually if you need it:
```
sudo /etc/init.d/cups start
```
and stop it again with 
```
sudo /etc/init.d/cups stop
```

If that doesnt help, you may need to kill the gs process
```

sudo killall gs
```

As for the diskspace, look where its going using the disk usage analyser. Its probably a ton of temp files generated by gs that you can safely delete from /var/log or somewhere.
