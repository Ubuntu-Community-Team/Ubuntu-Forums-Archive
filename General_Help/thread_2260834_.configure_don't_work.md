### Post by Yellow Pasque on 2015-01-15
This program (version 3.3.x) is available in the repository, so unless you absolutely need 3.4.0, the best way to install is:
```
sudo apt-get install pgpool2
```

To build from source:
```
sudo apt-get install libpq-dev build-essential wget
cd ~/
wget -O - http://www.pgpool.net/download.php?f=pgpool-II-3.4.0.tar.gz > pgpool-II-3.4.0.tar.gz
tar xzf pgpool-II-3.4.0.tar.gz
cd pgpool-II-3.4.0/
./configure
make
sudo make install
```

Then you can continue following the guide.
