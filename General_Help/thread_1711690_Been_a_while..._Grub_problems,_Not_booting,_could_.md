### Post by wilee-nilee on 2011-03-23
I think the script looks good, as far as I can see. If you resized the windows partition with another partitioner then the disk manager in w7 then it just may need a chkdsk. 

So how did you partition the W7, and was it rebooted to see if it booted, if you used the Ubuntu partitioning?

You can reload the MS bootloader to the mbr to see if windows boots with its own bootloader as well.
Boot the windows install or recovery disc choose the language then hit r for repair, navigate to the repair command terminal and run.
bootrec.exe /fixmbr
If you want to run a chkdsk here you would instead run
chkdsk /f/r

Grub can be reloaded to the mbr when needed from a live cd.
[https://help.ubuntu.com/community/Grub2#Reinstalling%20GRUB%202](https://help.ubuntu.com/community/Grub2#Reinstalling%20GRUB%202)
