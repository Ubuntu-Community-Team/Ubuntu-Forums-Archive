### Post by elvinatom on 2008-05-01
I think the problem with a 128MB machine is that the CPU is slow. Even if you install a modern light-weight Linux on it, the applications your mom will run are up-to-date and will be more RAM and CPU demanding than the box can provide. I run Ubuntu on a 900MHz Duron and Firefox alone take 10sec just to start up - never mind flash. You might just be better off reinstalling 98 from scratch - at least you are rid of all the junk-ware, spyware and viruses. If you insist to go Linux, I suggest Fluxbuntu or even better the really fast "Damn Small Linux" if you're up for a setup challenge. Then you got true lightweight from head to toe. If you're a geek to become consider recompiling the kernel and throw out everything you don't need, but be warned - that is not easy for a beginner!
