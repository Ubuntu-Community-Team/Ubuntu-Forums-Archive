### Post by drs305 on 2011-11-05
I don't know what is causing it but if you have compizconfig-settings-manager there is an option to open a window with a specific size.

ccsm >  Window Management > Window Rules > Size Rules tab.

You can 'grab' the window class (such as 'chromium') and then specify the size for chromium windows.
