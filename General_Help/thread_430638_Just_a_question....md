### Post by Hendrixski on 2007-05-02
Oh that's an easy one.  The proprietary drivers for most graphic drivers are... well.. proprietary

And therefor they're this kind of black whole that no developer can see into, test against, or even guarantee that will work.  So Ubuntu distributes with open drivers that if someone has issues with they can file a bug (for example using Launchpad) and then it will get fixed.
