### Post by Copperline on 2010-05-22
Hi - I have noticed Lucid does run hot at some times...especially when using Firefox as the CPU (somehow and for some reason as yet unknown to me) needs to run at 100% regardless of what I'm doing, which causes the system to jump sometimes.

56/57/59 isn't excessive for a core2duo pro...I'd be expecting around 50/55 under load so perhaps it's a touch higher than expected but it doesn't indicate a problem to me. If you notice it creeping up then you could try re-applying your thermal grease (Arctic Silver 5 is a good choice IMO) and seeing if that makes a difference. I think, though, it's peculiar to Lucid. Have you monitored what the CPU is doing whilst you're using Lucid on it? My guess is you're probably experiencing the same as me...which is 100% CUP use when running the basic system and Firefox with 'Google' displayed.

Copperline
