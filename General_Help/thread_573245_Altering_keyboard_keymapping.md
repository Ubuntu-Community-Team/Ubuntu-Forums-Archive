### Post by strabes on 2007-10-11
I don't know how to do exactly what you asked, but I do know that you can use the windows key with other keys for shortucts. In gnome-keybinding-properties (System > Accessories > Keyboard Shortcuts) you could set the keyboard shortcuts to the corresponding keys on the keyboard, except using the windows key instead of the FN key. For example, on my laptop, FN+PageUp is volume up, so if my FN key were broken I could use WIN+PageUp instead. I hope this makes sense and helps.
