### Post by mcduck on 2012-02-18
You need to install [Ubuntu-Tweak]("apt:ubuntu-tweak") or [CompizConfig Settings Manager]("apt:compizconfig-settings-manager").

With Ubuntu-Tweak, the option is in Tweaks->Unity Settings->Launcher Hide Mode (set it to "Auto Hide").

...and with CCSM, it's in Desktop->Ubuntu Unity Plugin->Behaviour->Hide Launcher (set it to "Autohide").

I already messed up a 11.10 install using CCSM. I don't want to do it again.

Ubuntu Tweak has much less options, and is thus safer to for those who are not sure about what they are doing. On the other hand it has only a very small fraction of the features and the power CCSM has.

CCSM is safe to use as well, but only if you actually pay close attention to every message and warning it gives you as it will allow you to enable plugins and features that may conflict with each other. So clicking "OK" without knowing what you are doing sure is a bad idea. (Of course that's *always* a bad idea, regardless of what program you are using and what you are doing... ;))
