### Post by mcduck on 2011-08-31
The [PS3 Media Server]("http://code.google.com/p/ps3mediaserver/") is rather excellent media server to use with PS3, and it's multiplatform so it runs just fine on Ubuntu.

PS3 will connect to many other media servers as well, including XBMC, but the PS3 Media Server has very good transcoding features so if you have powerful enough computer to use as the server you won't need to worry about compatibility issues with PS3 and different audio/video codecs.
