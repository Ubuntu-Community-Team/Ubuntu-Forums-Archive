### Post by seawolf167 on 2011-06-15
This is because you are using 11.04. Most of the guides out there that you will read will be for earlier versions. In 11.04, if you click the power icon then select System Settings you should be able to see the Administration options.

Also - at the login screen you can select to show the Ubuntu Classic Desktop which will match most of the old[er] guides.
