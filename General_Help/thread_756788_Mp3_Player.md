### Post by djrobthaman on 2008-04-16
> **xcvcx said:**
> Amarok mysql database. I hear it's better but i can't get it too work and in a guide on how to set it up it says /configure and some other things.

I'm not sure what you mean, but to get amarok to work all you need to do is install it (go to add/remove programs in your ubuntu menu and search for amarok).  Once it is installed, run amarok and the first time it turns on it will show a configuration wizard type pop-up.  Just keep clicking the buttons that say or at least sound like ok and you're done.

The only thing is that it doesn't play mp3's out the box.  But that's really easy to fix too because the first time you try to play one a message will come up that says something like "Amarok can't play mp3's yet, click ok to fix it" Click the button, restart amarok and you're good to go.
