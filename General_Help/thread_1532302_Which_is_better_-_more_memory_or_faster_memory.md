### Post by cascade9 on 2010-07-16
It probably is right, but without knowing the motherboard model it ishard to know for sure.... 

As for 2GB DDR400 vs 3GB DDR 333- the glib answer is 'if you want to use 2GB+, then of course 3GB is the way to go. RAM is a lot faster than swap files'. I tend to not use more than 1-1.5GB unless I'm doing a huge amount of multitasking (not fun to do on a P4) or ripping videos, so for me I'd take the DDR400 option everytime ;) 

BTW, from memory the 1st of the intel chipsets to support DDR400 (and dual channel) RAM was the i865/i875. Considering that moving to 3GB would stop dual channel RAM working, that makes it even more likely that you would be better off with 2x1GB DDR 400. One other thing that could affect things is the FSB or the CPU- to use DDR 400, the CPU has to be a 400MHz FSB model.

### Post by Mark Phelps on 2010-07-17
Depends on the memory controller in use on the board.  

On some older boards, when you use 2 sticks of memory, the controller can run it at 1T.  But when you add a second or third stick, it throttles that back to 2T -- and there's nothing you can do to change that.
