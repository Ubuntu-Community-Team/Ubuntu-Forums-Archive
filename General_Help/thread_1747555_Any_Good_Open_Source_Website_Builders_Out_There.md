### Post by SeijiSensei on 2011-05-02
[Joomla]("http://www.joomla.org/") and [Drupal]("http://drupal.org/") are the big names in the field.  [WordPress]("http://wordpress.org/") is more oriented to blogging.  All of these can be installed on Ubuntu if you have a "LAMP" (Linux+Apache+MySQL+PHP) or "LAPP" (Linux+Apache+PostgreSQL+PHP) stack.  MySQL configurations are more common, but I prefer PostgreSQL for an SQL server.  Here's a guide on installing [LAMP on Ubuntu]("https://help.ubuntu.com/community/ApacheMySQLPHP").
