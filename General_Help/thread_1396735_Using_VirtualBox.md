### Post by oshunluvr on 2010-02-02
The short answer is "Maybe."

First of all W7 requires a LOT of memory and video resources. Unless you're using a very heavy duty machine, you won't find much usefulness. 

XP works very well in vbox with moderate resources. If you require USB access you'll need to use the non-OSE version from the virtualbox website. Also, games or other programs that require direct access to hardware may not work well or at all.

I use vbox to update my Harmony remote and play on-line poker. Works great for those apps!

You'll find lots of infor on the web if you're looking to run a specific program. Do a websearch for your game and vitrualbox and see what comes up.
