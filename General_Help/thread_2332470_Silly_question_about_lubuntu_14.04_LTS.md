### Post by &amp;KyT$0P# on 2016-08-01
> I know it has support until 2017 (next year) but how exactly does it work..?
The LXDE components won't be supported but the base system, being Ubuntu 14.04, will be supported until April 2019 same as Ubuntu 14.04.

> If I install lubuntu 14.04.1 it still has support or the support goes for the 14.04... or the updates just go to the last version 14.04.4?
Lubuntu 14.04.1 is a pre-built ISO of Lubuntu 14.04 with the same base kernel version that shipped with 14.04.  It's still supported and will remain supported same as the rest of Ubuntu 14.04.
Later point releases have later kernel versions, I would not recommend using unless you are required to in order to make use of your hardware.  (I've found this whole HWE thing to be messy.)

Basically the only real difference between point releases is base kernel version and how upgraded are the packages it ships with.

Your question is not completely clear but I will answer the best I can.

Shortly after it reaches EOL all the repositories are moved to archived servers so you will not be able to get any updates not even security.

You can redirect your repository source list to the archives to get updates but all the software will be old they do not get anymore new updates once moved to the archives so your system would be at great risk if you go online with it, I personally recommend that no one ever uses an EOL system.

Yes. Likewise if you install 14.04.5. The Ubuntu base will be supported till 2019 for both 14.04.1 and 14.04.5.
