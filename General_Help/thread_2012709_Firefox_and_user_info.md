Google gives money to Mozilla because the default search engine shipped with Firefox is Google. Every time you do a search in Firefox, Mozilla gets money. There is no conspiracy here. If you don't want to be tracked by Google, don't use their services. Use DuckDuckGo as search engine and Ghostery to block site trackers and other bugs like Google Analytics, as already suggested. 

If there is someone to be afraid of is the web sites you browse and not the Mozilla Foundation, which is on your side. They fight for privacy and an open web. That's why we suggested those add-ons.
