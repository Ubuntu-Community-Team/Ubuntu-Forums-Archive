For the other queries:-

1.  Minimising FF should not mean you lose the conky window, unless you use the minimise all windows icon in the panel; when using gnome, that also happens to the conky window, but not if you minimise only FF.

2.  Don't start conky in terminal but either automatically when the session starts, and I'm not sure how you do that in xubuntu, or use the Alt+F2 run dialog box if you want to start it manually.
