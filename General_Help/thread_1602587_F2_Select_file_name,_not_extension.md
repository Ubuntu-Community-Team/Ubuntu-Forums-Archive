### Post by osjak on 2010-10-21
The Nautilus file manager in Ubuntu 10.04 had a nice feature to select the file name without its extension when a user presses F2. Since file extensions hardly ever change, this simple feature allowed me to press F2 and start typing the new file name right away without trying to position the cursor and/or delete the current name.

I just updated to Ubuntu 10.10, and it seems that Nautilus has lost that handy behaviour. I scanned its preferences and cannot find a setting for this. Any idea on how to get it back?
