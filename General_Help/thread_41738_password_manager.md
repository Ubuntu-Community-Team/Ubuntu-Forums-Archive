### Post by matthew on 2005-06-16
Ooohhhh... Sorry, not having used roboform I made a false assumption. What Revelation will do is allow you to save passwords, credit card info, and other sensitive information is a very secure, encrypted manner and have them readily available for you. It won't automatically fill in web forms for you. I'm sorry for the misunderstanding.

It seems to me that Firefox will do what you want already, won't it? Go to Edit->Preferences, then click on the Privacy tab on the left. Open the section called Saved Passwords and make sure the box is checked that says Remember Passwords. I think you can even set a master password that has to be entered every time Firefox is started up before the feature will run.

### Post by shadow07 on 2005-10-02
I too am looking for a similar package.  I have Roboform for my Windows machines, but I am unable to find anything for Gnome.

The problem with the Firefox password manager is that it looks for specific HTML field names "username" and also "password".  The problem is that not all web developers use these standard names for fields.  When this happens, Firefox will not prompt you if you wish to store the username and password in it's password manager.

With products like Roboforms, or Citrix's Password Manager, it detects most (if not all) username and password fields in a web form.

The problem, however, is that no one has created a port for one of these applications.

This is not a major piece for me and prevent me from making the switch to Ubuntu over Windows.  But, it is a nice to have.
