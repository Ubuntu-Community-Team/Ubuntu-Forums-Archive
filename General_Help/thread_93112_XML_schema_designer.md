### Post by charlesbrooking on 2005-11-22
[QUOTE=graabein]Hi, can anyone recommend me a gnome application to design XML schemas (.xsd)?[/QUOTE]

Check out the "Web Tool Platform" plugin for Eclipse, which supports auto-complete and has a graphical view for XSD. My use of the plugin has mainly been limited to XML and XLST documents, but that support has been good. I use Eclipse for Java development as well, so the plugin is convenient for me. If you have no other use for Eclipse, you may be able to find a more simple application for your purpose.

Later
Charlie
