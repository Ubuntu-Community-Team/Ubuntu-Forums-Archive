### Post by ad_267 on 2008-11-07
That program is for bitmap fonts. From the website it can export these:
    * PSF2 Linux console fonts.
    * HEX fonts (see online help). 

I'm not sure what programs you'll be able to use those fonts with, but you would put them in your .fonts directory.

I think you'd be a lot better off using FontForge. There's a tutorial at [http://fontforge.sourceforge.net/editexample.html](http://fontforge.sourceforge.net/editexample.html)
