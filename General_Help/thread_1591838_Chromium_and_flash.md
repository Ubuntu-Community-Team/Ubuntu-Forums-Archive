### Post by VanceVEP72 on 2010-10-09
When you say Flash, I assume you mean the Flash Player plugin?  If so, the easiest fix for me was to open the page containing the Flash object in Firefox and let Firefox install the plugin.  Then close Firefox and re-open the page in Chromium. Voila!

### Post by xjesse on 2010-10-09
Chrome *stable* actually includes the Flash plugin so if all else fails with Chromium.

Here:
```
sudo aptitude install flashplugin-nonfree
```
