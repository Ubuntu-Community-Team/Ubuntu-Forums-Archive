### Post by grahammechanical on 2016-03-18
Imagine a reset of Ubuntu to a default install that prevents a user from dual booting Windows 10. When Windows 10 does that to a Linux installation Microsoft can claim that as the machine was not sold with Linux on it, then the customer has nothing to complain about. 

Linux developers are in an entirely different situation to Microsoft developers. At present we can easily over-write Windows when installing Linux. Having such a restore to factory default option will offer more opportunities for users to destroy their Windows OS. When Linux is the factory default then the situation will be different.
