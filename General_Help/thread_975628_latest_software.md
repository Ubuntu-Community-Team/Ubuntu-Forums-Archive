### Post by aysiu on 2008-11-08
Ubuntu updates its software with every release (every six months). So if you update to the next release, you'll get the latest software.

### Post by magnus0 on 2008-11-08
Ubuntu only provides security updates, not version updates.

For the newest (maybe not so stable) software, there is [http://getdeb.net](http://getdeb.net)

### Post by edd07 on 2008-11-08
Or, if the software you're interested in has its own repository, you can add it to Third-Party Updates and you will receive the latest version. Each software usually provide instructions on how to do this.
