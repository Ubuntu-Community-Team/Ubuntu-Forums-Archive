### Post by ankspo71 on 2012-09-11
Hi, fdisk has an expert mode option to correct the partition order:
[http://journalxtra.com/linuxsanity/how-to-reorder-linux-drive-partition-numbers-2768/](http://journalxtra.com/linuxsanity/how-to-reorder-linux-drive-partition-numbers-2768/)

If you notice in the text below, the 'start sectors' and 'end sectors' indicate the true positions of the partitions on your hard drive. sda7 should have your first logical linux partion number (sda5), but it doesn't. One reason for this happening is because we can modify our partition layouts as we need to, but the system is unaware of the changes.
