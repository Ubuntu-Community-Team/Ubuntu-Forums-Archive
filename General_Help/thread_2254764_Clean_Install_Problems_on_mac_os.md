### Post by llamabr on 2014-11-29
In that case, it would depend on what kind of keyboard you have.  Obviously, you can't just substitute option for control.

Your keyboard might have a section shift key, or a function key, or AltGr key.  That's the one you want.

Otherwise, this will work way better if you have a friend who can lend you a usb mac keyboard for 5 minutes.

### Post by Bucky Ball on 2014-11-30
Welcome. Just a tip. 13.10 is no longer supported so I'd advise you start again with a supported release. Even if you did succeed in installing 13.10 it would be rather pointless as the repos are no longer up and therefore you would be using out of date software with no chance of update/upgrading (and out of date security with no available security updates). 

12.04 LTS (supported until 2017) and 14.04 LTS (2019) are your best options. 

Please see [HERE]("http://ubuntuforums.org/showthread.php?t=2229730"). 

Good luck.

* Update: I see you've installed it. I would still advise you start again. We can't give you much help with an unsupported release.

You might like to now have a look [HERE]("https://help.ubuntu.com/community/EOLUpgrades"). ;)
