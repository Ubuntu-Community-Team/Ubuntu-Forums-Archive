### Post by labinnsw on 2010-01-16
Not sure if you tried formatting the drive while re-installing 9.04. I think 9.04 uses ext 3 while 9.10 uses ext 4 file system.

If that does not work, have you tried installing 8.10 and then upgrading? (The drive must be formatted to ext 3)

If it cannot be done while re-installing, boot into the Live CD and format the drive with the partition editor (Gparted.)

I know these methods are tedious, but I would go to any lengths to be able to use Ubuntu.
