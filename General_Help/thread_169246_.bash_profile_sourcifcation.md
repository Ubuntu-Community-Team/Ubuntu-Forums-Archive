### Post by jrib on 2006-05-02
profile only gets sourced on login shells, .bashrc will get sourced for interactive shells that are not login shells.  So .bashrc will get sourced with applications > accessories > terminal.

If you need something to get sourced when you login to gnome instead of when you open a terminal, you can use ~/.gnomerc (you may need to create it)
