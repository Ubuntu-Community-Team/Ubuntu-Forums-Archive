### Post by howefield on 2010-03-10
There are many live CDs available for download which include anti virus scanning programs.

Here is one.

[http://www.sysresccd.org/System-tools](http://www.sysresccd.org/System-tools)

Another option is to install Ubuntu to a pen drive and install anti virus into it, as long as the other Windows computer can boot from USB you'll can scan it from the pen drive.

System > Administration > Startup Disk Creator.
