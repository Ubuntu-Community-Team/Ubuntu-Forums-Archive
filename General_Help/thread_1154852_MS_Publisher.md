### Post by Rackstar on 2009-05-10
It is possible to get MS Office running through wine. If you need more information about this, you can ask.

But using wine sometimes has problems, I don't know for publisher, but word works rather well.

Another option is to run Windows in a virtualbox. This means you run Windows as like you would run a program in ubuntu, you can then install MS office in windows. 
The program you need is VirtualBox. You can ask if you need more information.

But what you should do, is contact the school and let them know not everyone has MS Office and runs Windows at home. If they expect it to be so.

I hope this helps you a bit.

### Post by 3rdalbum on 2009-05-10
Microsoft Word documents can be opened and saved in Openoffice.org (comes preinstalled with Ubuntu), Abiword or Kwrite.

Microsoft Publisher, unfortunately, isn't really much use except for very simple work... therefore nobody in open-source software has reverse-engineered the file format. Your best bet for Publisher, if it's absolutely necessary to use it at home, is to install Windows into a virtual machine.
