### Post by zachthejones on 2009-11-26
I've been playing a bit on Tux Paint with my 2-year old. He loves the program and my wife and I are having a blast teaching him to use a mouse to use it. But it isn't fullscreen.

Does anyone know if there is a fullscreen version out there, or a way to tweak this one (from the regular repos)?

---

### Post by TryMe on 2010-06-01
If you're running Tux Paint by double-clicking an icon, check the properties of the icon and add "--fullscreen"  as an argument.

You can edit the proprieties of a applications menu item by going to:
System menu -> Preferences -> Main Menu

Command should look like:
tuxpaint --fullscreen
