### Post by slayerboy on 2007-02-28
If you're running from kubuntu to get gnome:
```
sudo apt-get install ubuntu-dekstop
```

if you're running from ubuntu to get KDE:
```
sudo apt-get install kubuntu-dekstop
```

if you're running from either to get XFCE:
```
sudo apt-get install xubuntu-dekstop
```

You could do like i do and have a base install of ubuntu and have both kubuntu and xubuntu installed all at the same time.

I prefer XFCE, that's why i showed you that you could do that.  The choice is yours.

### Post by Andy_D on 2007-02-28
If you don't like Kubuntu then you probably wont want to install kubuntu-desktop as that will give you essentially the same thing. Instead try:

sudo apt-get install kdebase

That should give you the essentials required to run a kde session. Once installed just select kde from the sessions menu at login.
