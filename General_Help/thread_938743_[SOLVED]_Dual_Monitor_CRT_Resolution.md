### Post by /////// on 2008-10-05
I just dug up my CRT monitor and decided to connect it to my computer. Using nvidia-settings, I managed to get it to work with my computer but the resolution can only go as high as 640x480. My main LCD monitor works fine and runs at its native resolution at 1440x900. How do I get the CRT monitor to run at a higher resolution like 1024x768? Also, after installing the second monitor so that both of the monitors use a separate X screen, I've noticed that whenever I click something, it takes a bit longer than before, why is this happening? I have an 8600GT.

### Post by /////// on 2008-10-05
Lol nevermind, just sorta figured it out. Turns out I had to gksu displayconfig-gtk in terminal which opened up the screen and graphics Preferences, then select the second screen and change the model of it to a generic screen of 1024x768. After I did this, I rebooted the X system and it gave me a very weird configuration in which both the screens were like one screen which I didn't want. It turns out that I still had the option as use a separate X screen but the xinerama option was turned on. I disabled that and rebooted the X server again and this time I got the second screen to be 1024x768 only I couldn't access the screen, I then went into the nvidia-settings again and set the screen as 1280x1024. I saved the file and rebooted the X server and this time for some strange reason it gave me what I wanted in the first screen which was an accessible 1024x768 screen. O.O
