### Post by akiro.yamamoto on 2006-03-04
Some of the desktop components like Yelp (Gnome Help) use the rendering engine
ans shared files from 1.0.7. So deleting or removing 1.0.7 will damage the entire system performance. That's why it's only in the upcoming Dapper release and not in
the breezy backports. ;)

### Post by aysiu on 2006-03-04
[QUOTE=Resurrection]I'm not quite sure what this means, but ok, if you guys say its tied to stuff, then thats good enough reason to me.[/QUOTE] Backports are more up-to-date versions of software that are available for the current (as opposed to the upcoming) release.

Ordinarily, Firefox 1.5 would be in the backports for Ubuntu's current version (05.10, codenamed Breezy Badger), but they're now just going to be included in the upcoming Ubuntu release (version 06.04, codenamed Dapper Drake).
