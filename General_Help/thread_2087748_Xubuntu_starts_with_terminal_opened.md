### Post by 2F4U on 2012-11-24
Maybe in one of the previous sessions you had two sessions open when you shutdown the machine. The default configuration, as far as I know, is to save the session, so on next start it would open the applications from the last session. Can be configured in Settings - Session and Startup - Session.

### Post by Impavidus on 2012-11-24
Maybe you forgot to close them the last time? Xubuntu remembers what programs you were running when you logged off. Check setting-mamager->session and start-up (or whatever the english version is called, I'm translating from dutch)->application autostart and session.
