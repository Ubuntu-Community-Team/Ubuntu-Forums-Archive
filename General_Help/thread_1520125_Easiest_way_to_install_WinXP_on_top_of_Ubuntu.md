### Post by teejaybee on 2010-06-29
You'll only be able to do this if you can find or make some free space into a separate partition. Then when you install windowsxp to this partition, it will probably want to load itself as default OS so you'll need an ubuntu cd or something like that so you can boot from it and run GRUB so that you can run ubuntu again once you're done :)
