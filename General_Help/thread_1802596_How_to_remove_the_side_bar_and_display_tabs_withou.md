### Post by nothingspecial on 2011-07-12
I don't think you can move it to the bottom.

Install compizconfig-settings-manager (why on earth it is not installed by default I will never understand)

Open it up and choose the unity plugin. In there you can change the icon size and set the thing to never autohide.

Or choose classic ubuntu at the login screen and it will look like 10.10
