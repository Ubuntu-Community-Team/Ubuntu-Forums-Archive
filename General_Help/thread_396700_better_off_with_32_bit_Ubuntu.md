### Post by DrOlaf on 2007-03-29
I have an AMD64 laptop, which I dual-boot (triple-boot if you count Windows XP) into 32-bit Edgy with Gnome, and into 64-bit Kubuntu. Both work fine, although to be honest I don't see much of a performance difference between the two (I know, I know, I'm comparing apples with oranges here).

If you install 32-bit on your machine you will have access to the 32-bit repositories instead of the 64-bit ones, so installing software won't be a problem. There are plenty of threads in this forum that will describe the range of software available in the 32-bit and 64-bit repos. I have installed Skype on my 32-bit partition, and it works just fine.

### Post by smokey edgy on 2007-03-29
I too have an AMD 64-bit machine but run the 32-bit version. I was somewhat reluctant to use the 64-bit version. From what I've seen, the 32-bit based packages seem reasonably more stable with certain ones having trouble running in 64-bit (Flash player, various codecs, etc). Though I'm sure there are workarounds, I'd say to avoid more hassle, the 32-bit version is the way to go (in my opinion). 

Good luck!
