### Post by danwood76 on 2009-05-06
The software repo for 9.04 will have a complete set of updated packages, so to go from 8.10 to 9.04 would require a full upgrade (full mirror).
A partial upgrade would leave you with broken packages.

You will have to download the complete repo.
