mchaley,

By default, NX uses the system's authentication methods to allow users access.  It can however use a NX database for authentication instead.

A very quick walk-through for setting up NX on Ubuntu is available here:

[http://www.nomachine.com/ar/view.php?ar_id=AR12D00432]("http://www.nomachine.com/ar/view.php?ar_id=AR12D00432")

While the link refers to NX 2.0, the instructions should still hold for NX 3.0 installation.

There are also more general instructions for installation and server administration here:

[http://www.nomachine.com/documents.php]("http://www.nomachine.com/documents.php")

You can also search the NoMachine KnowledgeBase here for other issues that you may encounter:

[http://www.nomachine.com/support.php]("http://www.nomachine.com/support.php")

Please let me know if you are more successful in trying out NX by using the above information.
