### Post by Frogs Hair on 2013-04-10
Install Downloaded Themes or icons in 12.04-12.10

_Install the Gnome Tweak Tool / Advanced Settings for making theme selections from the software center . _ 

Use only GTK 3.6 themes for 12.10-GTK 3.4 for 12.04  Check this in the theme description at the download location. 

Download and fully extract the theme packages and hold the folders on the desktop until needed . (Right Click Extract Here)

(Single User) Create a .themes and .icons folder in home / hidden folders . Use Crtl + H to open hidden folders in Nautilus.

(All Users) Use gksudo nautilus in the terminal to open  Nautilus as root and navigate to > File System / usr / share / themes or icons .

Place the folders in either Location . Use The Tweak Tool/ Advanced Settings to make theme selections . Logout - in may be required before themes are visible in Advanced Settings . 


There are theme PPAs available if you don't want to install manually.  [http://www.webupd8.org/2012/12/2-beautiful-dark-themes-for-gtk-36-or.html](http://www.webupd8.org/2012/12/2-beautiful-dark-themes-for-gtk-36-or.html)

Not giving you a GUI for something as frivolous as installing new themes; a process which is easy anyway (and actually, unchanged since the Gnome 2 days - the Appearance panel in Gnome 2 would only install the theme into your home directory, so if you ran a program as root it would look like Windows 95).
