one of my dual-boot system can access the wireless internet service in my area, and the second one cant.

what should i do to have an ubuntu (first computer) - dualboot (second computer) internet sharing  ??

---

### Post by jimv on 2008-08-10
Install Firestarter on the first machine, and then you can use it to enable and configure internet connection sharing in Ubuntu.  Once you install it, you can open it from Applications > Internet > Firestarter.
