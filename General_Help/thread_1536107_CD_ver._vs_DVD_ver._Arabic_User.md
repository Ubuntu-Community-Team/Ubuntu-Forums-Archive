### Post by libssd on 2010-07-22
Assuming your machine has reasonable hardware resources (at least 1 gb RAM and 32gb disk), use the DVD version, which is a more complete distribution.

Just to clarify: are you planning to install on an 8gb flash drive, or use it to install from to a larger HDD? I have done both; full Ubuntu installation on an 8gb flash memory device takes up about 4gb, assuming you choose manual partitioning, and assign a relatively small (128mb or less) swap partition.

If you are just using the USB drive to install from, 4gb is plenty big enough.
