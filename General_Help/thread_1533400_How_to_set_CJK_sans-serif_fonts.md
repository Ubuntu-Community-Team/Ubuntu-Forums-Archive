### Post by Welton on 2010-07-17
Hello, everybody.

I installed the CJK language packs (and I uninstalled the ttf-wqy-zenhei font pack so the other CJK fonts don't get ugly), but for some reason, when displaying Chinese (or sometimes Japanese) text, the font that is displayed is not sans-serif. On my Ubuntu, this affects Firefox, Opera and Google Chrome. No matter how many times I uninstall and then install again, this font still appears.

So, how to set to display CJK texts in sans-serif font?

---

### Post by Welton on 2010-07-18
I solved this by removing ttf-arphic-ukai and ttf-arphic-uming and then installing ttf-wqy-microhei. All through Synaptic.
