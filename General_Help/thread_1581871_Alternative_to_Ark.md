### Post by fmmarzoa on 2011-02-04
Hello,

I come with a solution for the crash:

sudo aptitude uninstall unrar-free
sudo aptitude install unrar

This will install non-free unrar version, but it works.

More information on this bug report:

[http://www.mail-archive.com/debian-qt-kde@lists.debian.org/msg36217.html](http://www.mail-archive.com/debian-qt-kde@lists.debian.org/msg36217.html)
