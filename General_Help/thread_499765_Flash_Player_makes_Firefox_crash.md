Same here.  This is mentioned in other forums (can't tell you which one's right now).  It seems that doesn't happen if flash player is loaded in another tab.

A suggested workaround is that you load a flash file (either local or streamed) in another tab and then minimize it (but leave the flash player running).  I have used this workaround and it works fine.
