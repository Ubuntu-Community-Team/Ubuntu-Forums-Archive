### Post by matt_risi on 2006-10-18
A real explaination:

Ubuntu uses the Gnome desktop. (Goes for a cleaner design, minimalistic but very robust, in my opinion.)

Kubuntu uses the KDE desktop, which is a more flashy type design. 

(I really suggest trying both, you can install both apon install of Ubuntu)

Xubuntu is more for use of lower-end machines, and uses a light desktop manager.

Edubuntu is simply Ubuntu with more educational tools and accessoroies.
