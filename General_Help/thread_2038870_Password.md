### Post by papibe on 2012-08-07
Hi rickm1945.

You can set auto login by going:
```
System Settings -> User Accounts
```
there press the unlock button, and then you can turn on 'Automatic Login'.

Note that you will still need to use your password to do administrative tasks, like install software for example.

Hope that helps, and tell us how it goes.
