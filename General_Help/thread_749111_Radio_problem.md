### Post by randcoop on 2008-04-09
If you go here, for example: [http://www.bbc.co.uk/radio/aod/mainframe.shtml?http://www.bbc.co.uk/radio/aod/radio3.shtml]("http://www.bbc.co.uk/radio/aod/mainframe.shtml?http://www.bbc.co.uk/radio/aod/radio3.shtml")
You'll see on the left a clickable link that says, "Listen using standalone RealPlayer"  If you click on that, a window should open asking you what program you want to launch.  Browse to /usr/bin/ and then choose kmplayer.  Then kmplayer will play the stream for you.
