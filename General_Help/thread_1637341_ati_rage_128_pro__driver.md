### Post by Mark Phelps on 2010-12-06
You're not getting any replies because -- basically -- there is nothing really that you can do.

If you're seeing a graphical desktop, instead of just text lines on a black screen, the only working video driver is already installed.  That would be the default open-source driver.

There are no other drivers that you can install that will work with your card and recent Ubuntu versions.
