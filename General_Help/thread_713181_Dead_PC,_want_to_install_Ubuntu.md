### Post by jcwmoore on 2008-03-03
Most PC's will boot off of a CD before trying to boot from the HD, if you stick the CD in and it does not boot for the CD, you will need to hit F8 or F12 or something like that to get into the bios menu, from there change the boot order so that the CD is before the HD.

as for wiping the HD, Ubuntu will give you the option to automagically do that during the install.
