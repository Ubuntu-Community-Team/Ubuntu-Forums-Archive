### Post by franknfurter2 on 2020-07-17
Dear pantanz,

as I know, shortcuts are first captured by the application that is on top. If the application has not defined a shortcut, then the shortcut is forwarded to the system. If the system has defined that shortcut, it will do the defined action.

So for example, if you would open the picture folder in your open firefox session (if this is possible, I dont't think so), you have to define a shortcut in firefox. I don't know, if this is possible. For security reason this might be blocked by firefox.

If you want to open the folder in nautilus, you have to define a shortcut on the system that opens nautilus and within that the folder.

So the solution depends exactly on what you want.

Frank
