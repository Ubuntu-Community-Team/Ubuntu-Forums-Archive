### Post by IanB2 on 2007-05-21
Is it possible to run the KDE desktop in preference to the Gnome desktop in Ubuntu?
I'm sure I've read somewhere that this is possible.

---

### Post by rax_m on 2007-05-21
Yes it is.. if you have some kde applications on your system already installed then most likely you already have KDE installed. 

At the login screen, there is usually a sessions button. If you press that you can normally select a KDE session. If it's not available then you may have to install the KDE desktop over ubuntu (which is quite simple and you don't lose your gnome desktop).

---

### Post by ghell on 2007-05-21
If you only want KDE then you might want to get Kubuntu instead of Ubuntu for future installs. If you just want the choice of KDE or gnome (or xfce) just install kubuntu-desktop (or xubuntu-desktop) on top then you should be able to select it from the "sessions" option on the login screen.
