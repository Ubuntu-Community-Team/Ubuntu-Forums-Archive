### Post by ali999 on 2010-05-20
get compiz. ```
sudo aptitude install compizconfig-settings-manager
```
activate it by going to system>preferences>apperance and then visual effects. then go to system>prefences>compizconfig setting manager to make it the way you want. This program made me fall in love with ubuntu when i started, the rest is now history.

fyi...if you can't enable the extra settings, you probably have to get the correct video drivers
