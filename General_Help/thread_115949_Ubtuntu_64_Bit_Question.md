### Post by Jave27 on 2006-01-11
There are some apps that haven't been ported over to AMD64 yet, but the chief complaints seem to be the lack of Flash (doesn't bother me), and wine.  I've been running it successfully for a while and only boot into Windows now for Civilization 4 (which kinda works in Linux 32 if you have Wine and Cedega), DVD Backups (do-able with wine and the 32-bit libs apparently, but such a hassle), and Quickbooks (accounting package - supposedly works with Crossover Office, not sure if that's available on 64 or not).

If you don't mind filing the occassional bug report, then AMD64 is a good choice.  If you'd rather have stability over (a little extra) speed, then stick with the 32-bit versions.
