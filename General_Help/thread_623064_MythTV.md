### Post by #Reistlehr- on 2007-11-25
I like it a lot. There are a lot of 3rd party themes you can install. My favorite is MythMCE - a take off of the Windows MCE interface. 

But setting it up, functionality, navigatability, and such are all really easy and straight forward. You shouldn't really get lost and wonder what button you need to hit. 

On average a 30 minute program without re-encoding on my system takes up about 1 gig. If you use ffmpeg or mencoder, you can shrink it to about 200mb's and keep good quality.

Personally, i wouldn't suggest using a cheap tuner. It all depends on the tuner itself. If you buy a crappy tuner you'll get grain/noise in your image, and it would persuade you to not buy the good one. If you buy a good tuner and still are not satisfied, always a good reason to return it. I have a Hauppauge PVR-150, and i love it. Came with a remote.

LIRC (For the remotes) was a little tricky to set up when you're not used to it. But there is a lot on the internet to help you.

### Post by foolsh on 2007-11-25
I set up a MythTV box on a 2.8Ghz P4 with 512Mb RAM.
The tuner card was cheep ati tvwonder, this card has a software decoder so if my computer was any slower I don't think it would have been very good at decoding the stream, and probably would have dropped frames like crazy.
If you have a hardware decoding card you can get away with a lower end system.
just so you know I tried setting up a windows based media center while I was trying this out and mythtv is the only one that worked with my cheep card. so check the specs on your tuner card and ask questions to see if it is a hardware or software decoder.
Yes you will need a lot of hard drive space the format that mythtv records on takes up a lot of space, but there are ways to setup mythtv to recode the files to smaller dvix files after they have been recorded.
I would gather as much info about your intended system as possible and ask more questions before committing any money and time to something you might not be happy with.
