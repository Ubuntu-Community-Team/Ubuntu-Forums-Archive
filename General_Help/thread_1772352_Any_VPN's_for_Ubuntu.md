### Post by gjonatha on 2011-05-31
Hi there,

 I have tested vpnc for ipsec and it works fine.

 You will need to install the package:
sudo apt-get install vpnc

 Then just do:
sudo vpnc-connect

 It will ask for the gateway address and user information. You can also add a vpn connection from the network manager.
