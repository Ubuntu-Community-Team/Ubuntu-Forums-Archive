### Post by rmil on 2012-05-22
Unfortunately this has to be done manually
first you make backup of your data base folder /var/log/mysql

than you remove installation of mysql5.5
```
sudo apt-get purge mysql-server
```that will uninstall everything that connects to mysql (phpmyadmin, python=mysqldb....)

Now you install from source code following procedure under this link
[http://www.geeksww.com/tutorials/database_management_systems/mysql/installation/downloading_compiling_and_installing_mysql_server_from_source_code.php](http://www.geeksww.com/tutorials/database_management_systems/mysql/installation/downloading_compiling_and_installing_mysql_server_from_source_code.php)
