### Post by tomcat2007 on 2008-09-28
After having installed Ubuntu and several other flavors of Linux on a number of different computers, I have come up with a few items on a wishlist I would love to see incorporated into future releases:

1.  An option on the network applet to utterly and thoroughly disable IPV6 (or reenable it if necessary.) 

2.  A standard wireless interface icon and applet that doesn't magically change and/or lose functionality when you update the network configuration.

3.  An eth0 that stays down and gone from ifconfig when you have ifdowned it and then updated the wireless network configuration.

4.  Configuration save and load points.  This would allow you to save your computer's configuration files (or possibly a specific subset such as network) then restore them if you muck it up.

Perhaps some of these things already exist and I don't know about them, this was just something I was inspired to post after fighting with a laptop which initially connected to wireless & worked flawlessly, but since changing the IP addy from DHCP to static, it no longer recognizes the wireless interface, no matter what I do.
