### Post by QIII on 2014-05-23
The latest version of Flash for Linux is, unfortunately, not the latest version of Flash.  Adobe dropped development of Flash for Linux some time ago.  Websites requiring the latest Flash will often report that Flash needs to be updated -- and, unfortunately, you can't update to the latest version.

Chrome has pepperflash installed by default.  It is currently under constant development, so you might avoid that message by using Chrome.  If you don't want Chrome, you can use Chromium, but you will have to install pepperflash.

Irritating, I know.  But we can't do much when outside entities don't support Linux.

Best wishes.
