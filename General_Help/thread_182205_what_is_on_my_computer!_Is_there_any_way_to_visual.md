### Post by nanotube on 2006-05-25
another thing: you can clean your synaptic/apt cache by running command 
```
sudo apt-get autoclean 
```
and you can get a visual representation of what's on your hd and how much space it takes up by installing and using program "filelight" 
```
sudo apt-get install filelight
```
well, that ought to do the trick. :)
