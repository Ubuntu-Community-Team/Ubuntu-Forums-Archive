### Post by agarrett5 on 2009-10-05
If i were to do a fresh install of ubuntu how would I be able to keep my settings?  is there a profile file or folder which I would need to copy over?

---

### Post by 3rdalbum on 2009-10-05
Copy over the hidden files and folders inside your home directory. Their names begin with a period (.) and they contain your per-user settings.

As long as you *don't elect to format the partition* that contains your home directories, then the fresh install of Ubuntu will retain the existing contents of your home directories. However, **do not use this as an excuse not to make a backup first!**  I've never had a problem with the retaining feature in Ubiquity since it was added, but this is no guarantee that it will work for you.
