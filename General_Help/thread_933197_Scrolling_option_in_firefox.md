### Post by Grimnir512 on 2008-09-29
In windows whilst using Firefox, pressing the mouse wheel would bring up a circle with an up and down arrow in it, and when the mouse was moved up and down itself the page would scroll.

This doesnot appear to be in Ubuntu, is there any way to enable it or is it a windows option?

I am running Ubuntu 8.04.

---

### Post by kpkeerthi on 2008-09-29
Enable autoscrolling:
Firefox > Edit > Preferences > Advanced > Use autoscrolling
