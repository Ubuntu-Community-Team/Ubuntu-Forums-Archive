### Post by dtfinch on 2007-01-13
Right click an .mp3 (or whatever you prefer) file and click properties. In the Open With tab, you can add bmp and make it the default handler for those files. If you just add it without making it the default, it'll probably appear in the context menu.
I don't know about changing the appearance, but it'll probably involve finding a skin you like. I still use xmms.
