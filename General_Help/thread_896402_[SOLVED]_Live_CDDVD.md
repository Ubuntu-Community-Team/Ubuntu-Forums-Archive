### Post by jolx on 2008-08-21
if you select Ubuntu Live CD it will mark all the packages that come on the ubuntu live cd but arent installed when you install ubuntu (ubiquity, ntfsprogs, gparted, etc) to be installed. if you select ubuntu/kubuntu/xubuntu desktop it will mark all the packages that come with the ubuntu/kubuntu/xubuntu desktops (gnome, kde, or xfce) to be installed
