### Post by sifuhall on 2010-05-03
I've been wanting to try Ubuntu for a while now so last night I setup a dual boot (windows 7 & Ubuntu 10.04).

I installed on a Dell Inspiron 1720 with 2gb ram and 2 disk drives (160 gb & 300 gb).

Setup went great out of the box.  No issues with my video card or wireless networking (these were my previous sticking points with linux many years ago).

I setup chrome as my preferred browser, and I'm already loving Ubuntu.

I was looking for equavalents of my favorite programs, editplus and mediamonkey when I stumbled on a website that showed me how to setup wine so I could use editplus and mediamonkey.

Everything seems to be working great, but I have two issues.  If I can get these resolved I can see ditching Windows in the near future.

1.)  My printer is a networked Brother MFC9440CN.  Ubuntu was able to find this, but the closest driver was the MFC9420CN.  I tried using this but the test document failed.

2.)  I have a Maxtor SAN storage.  Ubuntu can see the device, but can't browse the shares.

Any suggestions on how to address these two issues will be greatly appreciated.

I'm looking forward to learning more Ubuntu.  It's been a long time since I was excited about an OS.
