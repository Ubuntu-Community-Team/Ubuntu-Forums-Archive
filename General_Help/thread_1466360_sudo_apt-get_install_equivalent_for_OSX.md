### Post by srs5694 on 2010-04-30
For open source software, you could look into [DarwinPorts](http://darwinports.com/) or [MacPorts](http://www.macports.org/) systems. These are akin to the FreeBSD Ports system (or Gentoo's Portage, which is inspired by FreeBSD Ports). They make it easy to download and install source code-based packages. I've used DarwinPorts to install a couple of packages, but I'm far from an expert on these systems, so I don't know their precise limits.
