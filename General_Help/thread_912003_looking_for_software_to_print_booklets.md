### Post by Infinite Indefinite on 2008-09-09
I have printed booklets on Ubuntu using Adobe Reader 8, which has a booklet printing option in its print settings.  (Incidentally, Adobe Reader 8 works with older versions of Ubuntu as well: I've been able to use it to print booklets on Edgy Eft).

If you have a printer that does automatic duplex printing then you should have no problem.  I haven't tried it with manual duplex.
