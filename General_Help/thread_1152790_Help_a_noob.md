### Post by _Purple_ on 2009-05-08
Go to places from the panel menu. Do you see the external HDD in that list? 
You can use MSN account with the im client pidgin. Application > Internet > Pidgin Internet Messenger. You can also download amsn. 
To download amsn, go to application > Accessories > Terminal and type 
```
sudo apt-get install amsn
```

Once it is installed, it will appear under Application> Internet.
