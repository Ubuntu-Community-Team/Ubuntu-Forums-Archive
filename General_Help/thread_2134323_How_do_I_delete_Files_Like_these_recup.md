### Post by papibe on 2013-04-10
I only see recup_dir.1:
```
drwxr-xr-x  2 root         root          4096 Apr 11 08:09 recup_dir.1
```
You can't easily delete it because it is owned by root.

To delete everything on the directory tree, with no way to recovery it back, run this command:
```
sudo rm -rf recup_dir.1/
```

Hope it helps. Let us know how it goes.
