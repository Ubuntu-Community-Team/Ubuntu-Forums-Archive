### Post by mahershi on 2010-05-24
Hi!

I have run 64-bit Ubuntu on my desktop (Pentium Dual Core = PDC for this note) and on my notebook (PDC). It worked well on both those machines.

Both my machines are simple beasts (64-bit PDC, 4 GB, large SATA hdds, Intel graphics chipsets built onto the motherboard .... detected in both cases).

My desktop has a DLink USB wifi ... and thankfully that works.

My laptop has a broadcom wifi built into the system and .... that works too since 9.10!!! Earlier it needed some goofing about, but there was always help on the forums and in the wiki.

That said, I did run 32-bit desktop on the notebook once. But, that was short-lived. Everything worked fine, but, deep in my mind I wondered why I wasn't harnessing the full 64-bit power of that beast ... so within days of installing the 32-bit desktop, I switched to 64-bit desktop.

Finally, I haven't noticed any performance differences between 32-bit and 64-bit Ubuntus on the laptop.
