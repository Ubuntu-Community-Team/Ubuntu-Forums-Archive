### Post by updatelee on 2011-11-11
Well I got transmission working again

sudo apt-get remove --purge transmission transmission-common transmission-gtk
sudo updatedb
locate transmission
sudo rm -R [anything related to transmission]
# dont forget ~/.config/transmission and ~/.cache/transmission
sudo apt-get install transmission

then use /usr/bin/transmissio-gtk to start transmission

been working overnight so I'll call it good. Odd things were it doesnt seem to have a built in list of blocklist anymore so I had to find one online. 

[http://blocklistpro.com/download-center/p2p-ip-filters/](http://blocklistpro.com/download-center/p2p-ip-filters/)

enabled webif and lowered the default port number and Im good to go.
