1. Wubi doesn't really partition your drive like a normal install would. Instead it makes a folder called ubuntu on the drive you have chosen and then adds ubuntu to the bootloader for you. All you need to really do is run the installer, restart, select ubuntu when your pc restarts and there you go.

2. If you choose the uninstall.exe or try to reinstall wubi it will delete your kubuntu.

3. You should check out the FAQ section on the Wubi site here [http://wubi-installer.org/faq.php](http://wubi-installer.org/faq.php)

If any of my info is wrong then feel free to correct me, but I am pretty sure I am correct on this.

Edit: For number 2. If you want to save your kubuntu install just rename the ubuntu folder or in your case it might be called kubuntu, I am not sure as I have never installed kubuntu.
