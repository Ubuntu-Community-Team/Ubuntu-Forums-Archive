### Post by ibuclaw on 2008-05-15
Where is the .bin file?

All I can find is [setup.exe]("http://www.wordpressthemegen.com/software/setup.exe")

But with bin files in Linux, what you do is open up a terminal and type in (at the file location)
```

chmod +x filename.bin
./filename.bin

```
