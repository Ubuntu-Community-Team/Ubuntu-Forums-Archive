### Post by guyver on 2007-10-11
I've been trying to get the same information.  So far silence or just being ignored.

Debian is by far waaaaaaaaaaaaaaaay easier to floppy install than Ubuntu.  Not sure why Ubuntu can't do this.

Go to Debian's site.  Download all 5 floppy images and build them onto floppies.

Boot from floppies (and make sure you are connected to a cable modem / router / etc.)  Once your system gets an IP address, installation is done from getting files over the Internet.

I made a dedicated Linux box this way, but I'm wanting to try and modify this so that I can do Xubuntu instead.

Getting Debian up and running is so easy, it makes Ubuntu look hard (and it is if you don't have the ability to boot from a USB or CD/DVD drive).

Go here for Debian Floppy Images (bottom row of links):  [http://www.debian.org/releases/stable/debian-installer/](http://www.debian.org/releases/stable/debian-installer/)

As for trying to use the Debian disks to install Ubuntu, go here (but it's dated):  [https://help.ubuntu.com/community/Installation/WithFloppies](https://help.ubuntu.com/community/Installation/WithFloppies)
