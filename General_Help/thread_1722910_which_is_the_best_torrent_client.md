### Post by newbuntuxx on 2011-04-07
Install wine using the ubuntu software center. And run utorrent.

Utorrent is by far the best torrent client out there. I actually tested vuze and utorrent on my ubuntu 10.10. 

In vuze, even after opening the ports that it needs (on ufw and my perimeter Gateway), and the torrent status shows green (green happy smiley face) I still can't go past 200 KB/s. 

As for utorrent, I don't need to open any port at all! Yet, it manages not to only work, but it works with blazing speeds (over 800 KB/s,)

I actually tested both clients (vuze and utorrent)for the same torrent at the same time (on two ubuntu machines from the same connection). Utorrent is by far better.

If you check their site, they talk about their connection management technology and how it puts less stress on the internet pipe etc...but with more performance...it is true, as far as my tests show.

Moral of the story: install wine, and use utorrent.
