### Post by hasimir44 on 2007-05-29
best practice is to use "sudo" before any commands that need to be run as root.  I won't lie though, if I have a bunch of commands to run as root, I cheat and login.. you can login as root 2 ways that I know of: 

```
sudo su -
```

and..

```
sudo -i
```

be careful..
