### Post by johnwillis on 2008-05-21
Presumably you already have a webserver set up for hosting this stuff... if not you need the following:

-Apache 2.X - Webserver
-MySQL Server - Database Server
-PHP 5 - Programming Interpreter
-PHP5-MySQL - Extension to allow PHP to read MySQL

Then look into PHPBB3 for forums, very easy to set up. But your database needs to be set up and assigned a user first. You may find the package "MySQL-Admin" useful.

I use Joomla (Content Management System), Moodle (Course Management), PHPBB (Forums) and others on a daily basis. And everything you need is open source.

You can always set up a test server in VirtualBox/VMWare to test your findings.

Have fun,

-J
