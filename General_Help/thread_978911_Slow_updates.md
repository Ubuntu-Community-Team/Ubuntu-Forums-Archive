### Post by cariboo on 2008-11-11
The server you are accessing for updates may be the problem. To pick a faster server, go to System-->Administration-->Sowtware Sources. Select the Download From dropdown and select other, then click the Select Best Server button. this will test all the servers and suggest the the fastest server, select it and you should now be able to download updates faster.
