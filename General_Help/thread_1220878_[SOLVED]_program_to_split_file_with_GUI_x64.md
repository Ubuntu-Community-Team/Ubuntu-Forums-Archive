What sort of file, how many pieces, what size pieces, and why?

Actually, none of that matters as you can use the Archive-manager, file-roller to do this.  It is hidden in the Applications > Accessories menu, but you can get it with Alt+F2 and type file-roller in the box.  Choose new in the application, and then Other Options in the dialog that appears.
