### Post by LaRoza on 2008-03-25
Try this, changes are bold:

```
jelmoh@jelmoh-laptop:~$ cat /etc/fstab
# /etc/fstab: static file system information.
#
#  -- This file has been automaticly generated by ntfs-config -- 
#
# <file system> <mount point>   <type>  <options>       <dump>  <pass>

proc /proc proc defaults 0 0
# Entry for /dev/sda3 :
UUID=271f0129-fb5a-46a5-a342-f6229b7afa10 / ext3 defaults,errors=remount-ro 0 1
# Entry for /dev/sda1 :
**#UUID=1054C7B854C79F3A /media/sda1 ntfs-3g defaults,locale=en_US.UTF-8 0 1**
**UUID=1054C7B854C79F3A /media/sda1 ntfs-3g defaults,locale=en_US.UTF-8, umask=000 0 1**
# Entry for /dev/sda2 :
UUID=5ea00f78-54f5-46f1-95ed-62cbc074187c none swap sw 0 0
/dev/scd0 /media/cdrom0 udf,iso9660 user,noauto,exec 0 0
jelmoh@jelmoh-laptop:~$ 

```

To edit this file run:

```

gksudo gedit /etc/fstab

```

Remember to comment out the line and add the change. If the change isn't what you want, or something doesn't work, just uncomment the old entry and delete the new one.
