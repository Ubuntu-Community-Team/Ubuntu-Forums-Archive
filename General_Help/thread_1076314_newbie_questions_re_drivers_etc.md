### Post by davec64 on 2009-02-21
The good news is you can try before comitting it to the Hard Disk by downloading the Live CD and booting directly from this.
It gives you a fully working system running from the CD and memory (it will be a little slow as it'll be reading info from the CD).
You don't have to worry about drivers as there built ionto the system, but by running the live CD you can check everything works.
If something doesn't work search this forum for that specific piece of hardware on how to get it to work.
 Just remember Linux isn't a replacement for Windows, it's different and it will do things differently from how you've got use to! :)

Have fun playing!
