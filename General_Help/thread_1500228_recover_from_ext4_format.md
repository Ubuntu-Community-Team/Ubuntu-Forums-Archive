### Post by cryptotheslow on 2010-06-02
Hard to say without knowing what was exactly done during the install.

If a basic format of the secondary drive was done followed by an install of Ubuntu to that drive then the first 5 - 10GB where the install is will be unrecoverable.

Search on these forums and Google for Photorec - there are extensive howtos, it's a useful app for rescuing files when you have truly hosed your partition and file table info. Read up well first before diving in.
