### Post by Soarer on 2007-06-10
Hi & welcome.

Are you using the cursor (arrow) keys to move up & down? The mouse won't work at that stage.

Other than that, the most common problem is that the CDROM isn't burned correctly. It needs to be burned at a very slow speed (the slower the better) although it never gave me a problem.

Finally, you might want to try the latest version (Feisty, 7.04) rather than Dapper 6.06 - it may work better for you.

It should just load & run, so I would say there is something wrong with the CD you have.

Let us know how it goes.
