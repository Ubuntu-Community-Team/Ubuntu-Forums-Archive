### Post by JigglyWiggly_ on 2011-12-01
So on this one IBM keyboard next to the arrowkeys there is this window looking key which lets me swap between the desktops in the virtual machine(ctrl + alt f(x))

Now I can't obviously just press ctrl + alt + f1 on the machine itself or it will switch the main machine's desktop session.

So how do I do this? Virtual box btw

---

### Post by DroidVPN on 2011-12-01
You can try one of this

1. HostKey + F1 (where HostKey is often the right Ctrl key)

2. First press F1, then Ctrl and then Alt. Your host OS won't pick it up but the guest OS will.
