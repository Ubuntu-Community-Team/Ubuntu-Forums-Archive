### Post by Affendi on 2011-03-12
Hello, my mouse stopped working a day or two ago and so I've been using the virtual mouse (shift + NumLock, number keys on number pad move mouse, 5 clicks).  So I've been using a lot of keyboard shortcuts to do everything since using the number pad is a little slow. But I think I must of hit a strange key combination which makes 5 on the number pad do a middle click instead of a left click (unfortunately this means the difference between changing focus to another tab and closing the tab).  Naturally I'd like to know how to put it back, but also how to do it again, and how to right click with this feature too.
As far as I can tell this virtual mouse thing is heavy wizardry, I can't find documentation for this feature anywhere.

---

### Post by Affendi on 2011-03-12
I've found it, it's called Mousekeys, and I got it working again.
I must have hit the * key on the keypad, this switches it to middle click mode.
/ switches back to left click mode
- switches to right click mode
0(del) to select/drag
. (ins) ends select/drag
