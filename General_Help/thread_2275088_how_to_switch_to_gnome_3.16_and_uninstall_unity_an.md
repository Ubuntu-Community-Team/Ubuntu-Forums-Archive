### Post by dino99 on 2015-04-24
there is no ubuntu released using gnome 3.16 right now. You need a vivid installation, then add the gnome3-team staging ppa, and after updating the archives, upgrade all the 3.16.1 packages; then deactivate the ppa. That is what i'm using here since a couple weeks, all fine.
ppa:gnome3-team/gnome3-staging
