### Post by EmanresuusernamE on 2008-09-17
Simple, when you install Ubuntu, tell it NOT to use the entire disk.  Have it resize your Windows partition, and have it then install Ubuntu on the remaining disk space.  It should have an option to install onto only part of your drive.  BACKUP your windows data before you install onto your drive though, just in case.  You WON'T be able to get anything from windows back if you use the entire disk.
