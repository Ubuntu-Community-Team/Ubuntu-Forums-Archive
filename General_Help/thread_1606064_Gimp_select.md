Hi all,
I'm trying to select a precise space in a number of photos. 1380px by 1380px. I find using my mouse is a pain. is there any way to make a 1380 by 1380px select box tool which I can use to select a given space quickly?

---

### Post by Vermind on 2010-10-26
You mean like take the rectangle select tool,
go to the tool options tab, and tick the box labelled "Fixed:" and change the dropdown to "Size" then put 1380x1380 in the box under it.

You can do the same with the crop tool if you want to crop instead of just simply select, discarding the area outside the 1380px square.
