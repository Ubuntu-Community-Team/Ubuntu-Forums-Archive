### Post by WorMzy on 2011-03-19
The flash plugin changed. It now stores temporary files in the browser's cache folder. On firefox, this is in the profile's folder.

The default is ~/.mozilla/firefox/xxxxxxxx.default/Cache, where xxxxxxxx is a random string of characters.
