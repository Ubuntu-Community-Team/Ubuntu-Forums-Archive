### Post by dandnsmith on 2009-05-08
If you're trying to use the scanner across the network, it's worth investigating [SANE](http://www.sane-project.org/), which used to be the means of using an awful lot of scanners with linux. I feel that I should say that, when I tried it some years back, despite configuration instructions I couldn't get it to work with a scanner attached to a Windows PC across the network.

[http://solutions.brother.com/linux/en_us/index.html](http://solutions.brother.com/linux/en_us/index.html)
