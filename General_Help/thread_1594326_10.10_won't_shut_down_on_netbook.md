### Post by hariseldon122 on 2010-10-13
Mine is LG X130.Until I try to shut it down everything works perfectly. I found this:
[http://projectgus.com/2010/10/ubuntu-on-asus-eeepc-900ax/]("http://projectgus.com/2010/10/ubuntu-on-asus-eeepc-900ax/")

 "the Ubuntu Netbook Remix 10.10 installation went smoothly. Until the system froze when restarting after installation.

The freezing problem persisted, with freezing/lockups on shutdown, restart, suspend or hibernate. The RT3090 wireless was the problem. To fix it, you can install [the DKMS kernel module package]("http://www.halibutdepot.org/how_to_build_rt3090_for_ubuntu_lucid/") containing ralink’s proprietary source drivers. I just downloaded the prebuilt DKMS package on that page, I didn’t build it myself. Then I blacklisted the OSS ralink modules:

Appending to /etc/modprobe.d/blacklist.conf

# blacklist other Ralink modules in favour of 3090 DKMS mod
blacklist rt2860sta
blacklist rt2870sta
blacklist rt2800pci
blacklist rt2800usb
blacklist rt2x00lib
blacklist rt2x00pci
blacklist rt2x00usb"

Anyone try the method above?

### Post by Likeale on 2011-02-28
> **andrewhong951 said:**
> I'm having this same problem, but I don't know what to do with information to fix it. Can someone please help me? I'm totally new to this whole linux thing.

1º Download [http://stat.case.edu/~jrt32/how_to_build_rt3090_for_ubuntu_lucid/rt3090-dkms_2.3.1.4-0ubuntu0~ppa1_all.deb](http://stat.case.edu/~jrt32/how_to_build_rt3090_for_ubuntu_lucid/rt3090-dkms_2.3.1.4-0ubuntu0~ppa1_all.deb)
2º Open it with Ubuntu Software center
3º Install it
4º Open Terminal
5º Write sudo gedit /etc/modprobe.d/blacklist.conf
6º add
# blacklist other Ralink modules in favour of 3090 DKMS mod
blacklist rt2860sta
blacklist rt2870sta
blacklist rt2800pci
blacklist rt2800usb
blacklist rt2x00lib
blacklist rt2x00pci
blacklist rt2x00usb
7º Save
8º Restart computer
