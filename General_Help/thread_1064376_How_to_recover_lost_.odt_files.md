### Post by personjerry on 2009-02-08
If in fact you did delete the file permanently (lost file handles) then you should stop all applications immediately, and don't save anything. The files may still be on your hard drive, which may be recoverable, unless some new object overwrites it. Thus if less programs are running to write files onto hard drive, the less chance that one of those files overwrites your important files.

There are lots of freeware programs that can recover the files. Google is your friend.

OTHERWISE... get the files from the trash bin?
