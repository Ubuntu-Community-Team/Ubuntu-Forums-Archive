Well, some people are having issues with 10.04, and others arent. I would suggest, if you have the space, to install 10.04 clean on a separate partition. That way, if you have issues, you can simply boot back to 9.10 to use your computer/fix the problems.

I might use this as an opportunity to suggest setting up a separate partition for /home. In case you dont know, your /home folder is where all your pictures, music, and data is stored. The cool thing about a separate /home is that you could have 10.04 and 9.10 installed, and have them sharing the same /home folder, making your data accessible from either install. You can set this up using the manual option on the partitioner when installing. Ask for help here if you need it :)
