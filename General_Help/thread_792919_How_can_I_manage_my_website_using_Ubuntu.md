### Post by freymann on 2008-05-13
> **Old74th said:**
> How can I manage my website using Ubuntu? Does anyone out their actually use a Dreamweaver type software on Linux for their websites?

 There are some nice editors for building web sites available with ubuntu.

 Bluefish and Quanta Plus are my two favourites. You can install them through Synaptic Package Manager quite easily. They don't come with any premade templates but are great for editing/building code.

 In project mode Quanta Plus will move files around via ftp between your computer and the remote server (I can't remember if Bluefish does this too in project mode).

 Or try FileZilla and gFTP as separate FTP programs.

 You could also create a 'local ftp folder' on your desktop and work with the files as if they were on your local computer. I think that's under Places > Connect to Server.

 If you search in Synaptic Package Manager for HTML you will get a whack of hits. Try some of the software out and see what works best for you!

 And don't forget GIMP can be used to edit your images (resize, crop, change formats). It works great.

 I haven't found anything like Dreamweaver or Front Page or Web Easy Professional in linux land, but that doesn't mean they aren't out there.

 -cheers!
