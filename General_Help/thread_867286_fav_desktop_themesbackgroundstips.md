### Post by Happy_Man on 2008-07-22
I'd have to say that if you want to get an amazing desktop, start with the theme. Find one that you really really like, and build around it. For example, my Linux desktop is based off of Futurelooks, which I find immensely appealing. In Windows, I use a darker theme to help contrast the game from the window decorations. So build your theme with a purpose, and you'll get something nice, guaranteed.
