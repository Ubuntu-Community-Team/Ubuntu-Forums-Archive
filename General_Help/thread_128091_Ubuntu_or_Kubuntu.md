### Post by mirshafie on 2006-02-11
It doesn't matter at all if you use Ubuntu or Kubuntu. It's all the same. Ubuntu comes with Gnome + the package ubuntu-desktop which is a set of GTK applications, KDE comes with KDE + kubuntu-desktop which is a set of KDE applications. You can install either of them easily whenever you like, no matter which one of them you originally chose.

In Linux you can use *lots* of different window managers and desktops, and even combine them in many ways. Gnome, KDE and Xfce are all window managers that use the X window system. That means that they all essentially work the same way, so you can use GTK applications on a KDE desktop and KDE applications on a Gnome desktop.

I personally recommend KDE. I used Gnome before and hated KDE, but since KDE 3.5 I use KDE because in my opinion Gnome has been left behind in functionality. I also think many KDE applications such as k3b (dvd/cd burner), Konqueror (browser/file manager), amaroK (music), Kate (text editor) or yakuake (terminal) are a lot better than many Gnome applications. As I mentioned before you can use all these on Gnome aswell but they're slightly faster on KDE.

**plexi50**
Do 'sudo apt-get install xubuntu-desktop'. If it adds a lot of annoying extra applications to the menu you can either delete them manually or if you're really lazy you can just unistall it by 'sudo apt-get remove xubuntu-desktop' or if you grow to like Xfce, 'sudo apt-get remove ubuntu-desktop'.
