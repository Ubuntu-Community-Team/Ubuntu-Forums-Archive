### Post by dcstar on 2010-04-19
With the upcoming release of a new Ubuntu version, the usual situation will occur where the main software repositories will get heavily overloaded in the weeks following the official release.

As usual these forums will then fill up with people complaining about the performance of their updates/downloads during this period because **they** have not changed their repository from the install default, so if **you** don't want to suffer these upcoming problems:

[SIZE="3"]Go into your Software Sources and select a close regional repository - or if your ISP actually hosts a repository then select that.[/SIZE]

Doing this will ensure that your own system updates more reliably and also the load on the main servers is reduced - a win for all Ubuntu users.
