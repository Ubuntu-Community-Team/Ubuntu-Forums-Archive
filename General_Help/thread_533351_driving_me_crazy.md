### Post by jamvegan on 2007-08-23
I used to have the same problem.  I tried various things to fix it, but what finally worked (and this is a rather extreme solution, you probably want to hold out for a less absolute suggestion) was doing a clean install of Feisty, then installing flashplugin-nonfree via Synaptic.  No more problems with Flash.  In fact, even when I had it kind of working before, video and sound never synched properly, but now it's perfect.

I did not do a clean install because of the Flash issue, but it was a nice side-effect that Flash now works. :-)
