### Post by elmuchacho on 2012-12-16
Hello,

We use the 12.04 LTS Ubuntu as a distribution embedded in our products, but we don't need all the Unity bells and whistles, and occasional bugs. 

So we'd like to install lubuntu, which is lighter, instead of Unity. My question is, does 12.04 LTS cover lubuntu as well, i.e will lubuntu be as maintained as the standard Ubuntu, or is it only Ubuntu with Unity that is the official 12.04 LTS ?


edit: I should have read the manual first :p

[https://wiki.ubuntu.com/Lubuntu](https://wiki.ubuntu.com/Lubuntu)

Please note that Lubuntu 12.04 is not an LTS (5 years support), but a 'standard' release, supported for 18 months. This is simply due the fact there is insufficient 'man-power' to commit to 5 years of support.


That is a pity.
