Greetings


Try :

> sudo rm -vf /var/lib/apt/lists/*

Or:

> sudo rm /var/lib/apt/lists/* -vf

sudo apt-get update




Or:

> sudo rm /var/lib/apt/lists/us.archive.ubuntu.com_ubuntu_dists_trusty_main_i18n_Translation-en
