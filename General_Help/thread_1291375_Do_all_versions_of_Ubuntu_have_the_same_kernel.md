### Post by BQAggie2006 on 2009-10-14
As far as I know, all versions within a release (i.e. Ubuntu/Kubuntu/Xubuntu 9.04) are developed with the same kernel version, though I'm not sure if there is any differing modifications between the versions (different modules, drivers, etc). However, similar versions not within the same release (i.e. Ubuntu 9.04 vs Ubuntu 9.10) will not have the same kernel version.

### Post by petersohn on 2009-10-14
Ubuntu and Kubuntu use exactly the same package repositories. The difference is only in what packages are installed by default (e.g. KDE vs. GNOME packages).
