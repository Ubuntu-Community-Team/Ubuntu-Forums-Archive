### Post by Yoooder on 2007-05-24
The biggest hurdle you'll have is getting Ubuntu onto a flash drive.  I haven't installed Linux to a thumb drive in a while, and it wasn't terribly easy the last time I did.

If you can get Ubuntu running on a thumbdrive (google it or ask around here for that specific task) then getting quake to run on it shouldn't be difficult.

I really don't expect it to be too terribly hard, flash-linux installs are quite popular now and I'm sure there's some good tutorials (and btw, there's some distros that are explicitly for flash media). 

As for the kernel requirement, no worries.  Kernel 2.0 is ancient...  everything is 2.2 at the absolute minimal, and most distros run 2.4 or 2.6
