### Post by kazztan0325 on 2012-01-29
Hi qwertyjjj,

You can install Language Packs, Fonts and Input Methods for Spanish with **'Language Support'**.

Once those are installed, you log out and log back in, then you would be able to add Spanish Input Method with IBus Settings.

Hope this helps.
