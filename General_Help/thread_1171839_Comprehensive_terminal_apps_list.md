### Post by maheshasolkar on 2009-06-03
Only recently found this.. a command line twitter client called ttytter!

  [http://www.floodgap.com/software/ttytter/](http://www.floodgap.com/software/ttytter/)

It is a Perl script, you can just download and run!

```
  % cd ~/bin
  % curl -o ttytter http://www.floodgap.com/software/ttytter/ttytter.txt
  % chmod +x ttytter
  % ttytter -user='username:password'
```

And you are on!

PS: I used ~/bin because that is where I keep my executables.
