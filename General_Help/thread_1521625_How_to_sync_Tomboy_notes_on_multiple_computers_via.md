### Post by darolu on 2010-07-01
Open Tomboy, go to Edit - Preferences; then click on the "Synchronization" tab on top; then select the option that says Service: **Tomboy Web**; after that set the server input text to: **[https://one.ubuntu.com/notes/](https://one.ubuntu.com/notes/)** and click "Connect to Server"; it will open the UbuntuOne website where you'll have to log in... you shouldn't have any trouble from there but just in case, I'm including this link =)

[https://wiki.ubuntu.com/UbuntuOne/Tutorials/Notes](https://wiki.ubuntu.com/UbuntuOne/Tutorials/Notes)

Good luck.
