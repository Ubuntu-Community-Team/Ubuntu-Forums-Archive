### Post by coldraven on 2011-08-15
If you are using Firefox select Edit>Preferences and you can choose the folder for downloads.
Surprisingly, by default it is called "Downloads"!
In Nautilus you can add a bookmark to a folder and it will appear in your "Places" menu.
Or do as TeoBigusGeekus suggests.
