I've been very happy with Ubuntu so far, after 2 days I'm positive I can make a full switch from Windows.

Beagle, Tracker and GDS are just too cumbersome and making them index my root directory is probably unthinkable (and I just need the filename). Plus I need to use my custom scripts to update the database (Xlocate). While in Windows Everything's database is always up to date as it monitors everything and still has a tiny memory footprint.
I wonder if there is such a program out there, and am very sorry if so, or why nobody has thought of making one.
