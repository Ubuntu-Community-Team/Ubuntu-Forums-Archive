### Post by bcbc on 2013-02-20
Yeah I've seen that bug before - I submitted a patch but it hasn't been taken up yet. But there should have been a log file showing the failure (the application doesn't show so it's not clear that anything actually happened).

Anyway... glad you got it sorted. Be aware that Wubi installs are more susceptible to file system corruption and so you should ensure backups are very current (backed up off the virtual disk e.g. under the /host folder, another partition or external drive, or synched to a cloud).

If you ever find things freeze up, use [Alt+SysRq R-E-I-S-U-B]("http://en.wikipedia.org/wiki/Magic_SysRq_key#Uses") to safely reboot.
