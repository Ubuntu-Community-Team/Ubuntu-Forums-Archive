### Post by -kg- on 2011-08-10
That's a good question...doesn't make a bit of sense!

But I have something for you to check.  Log on to the Live CD, then pull up "System Settings/Power Management/Power Profiles".  See which (if any) have the "Suspend Session" check box checked and if so, uncheck that box on ALL of them that have it checked.

That's the only thing I can think of that would "log you out" after a certain time period.  Actually it almost does sound like a Suspend.  If you were just logged out it seems like you'd be presented with a "Log In" dialog instead of the necessity to reload it from the disk.

It might be something else, but that's all I can think of at this point.
