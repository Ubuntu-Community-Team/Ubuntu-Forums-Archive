### Post by jon.reeve on 2009-03-13
Puppy Linux is a tiny distribution that's made for old or slow computers. I used to run it on a 500MHz laptop that had 128M of RAM, and it ran every bit as fast as a modern computer with a fat OS. Puppy's package management, software suite, etc, aren't *as* good as Ubuntu's, and it requires a tiny bit of learning curve, but it's blazingly fast (especially with fluxbox), and has a couple features that Ubuntu doesn't, like the ability to sleep/wake instantaneously. I used a flavor called NOP, which I thought was a lot prettier-looking than the standard issue Puppy.
