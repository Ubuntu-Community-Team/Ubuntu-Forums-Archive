Is there somewhere to go lookup what the latest kernel version is for each release of Ubuntu? I want to be able to look up for Dapper Drake and amd64 what the latest kernel version is and then do the same for let's say Intrepid and amd64, etc.

### Post by unutbu on 2009-02-22
Go to [http://distrowatch.com/table.php?distribution=ubuntu](http://distrowatch.com/table.php?distribution=ubuntu)
The kernel version is listed on the line that begins with "linux".

Alternatively, you can go to [http://packages.ubuntu.com/dapper/linux](http://packages.ubuntu.com/dapper/linux). There are links at the top of that page which allow you to select various Ubuntu versions.
