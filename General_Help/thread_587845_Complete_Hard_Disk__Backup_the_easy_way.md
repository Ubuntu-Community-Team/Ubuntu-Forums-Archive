Hello,
 I have Fiesty installed on a Toshiba Laptop. I need a few Windows programs so I thought I'd Install XP and dual boot till I could get rid of it. First I needed to backup Fiesty just in case.
I went around & around for three days trying to get a backup disk image. I'm obviously not proficient  enough yet to solve this problem. I finally remembered that the Acronis True Image rescue disk used with XP was supposed to be system independent. I booted into that disk, it faultlessly used all my hardware, showed all drives, etc. I simply pointed and clicked and it backed up Fiesty complete (70Gb) to a USB Hard Disk in four hours. I tested the verification and restore features, all was well. Acronis recognizes many different file systems it seems.
I know this may be a "chicken" way out but it'll work until I can get better at the terminal commands and I don't have to worry about screwing up everything.
