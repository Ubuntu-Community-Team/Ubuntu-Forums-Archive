### Post by dukebytes on 2009-01-20
ls -l shows the "long listing"  and  ls -a will show all files including those starting with . (hidden types)

ls is a very non dangerous command - you can try using ls with different arguments (and several at one time) to see how it looks and what it shows you until you find the one you like.  i use ls -al all the time ;)
