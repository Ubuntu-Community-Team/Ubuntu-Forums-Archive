### Post by Impavidus on 2016-09-22
Ubuntu may be more suited for weak hardware than Windows, but it's not a drop-in replacement. It's completely different. I can't say whether you'll like it. Ubuntu itself isn't much lighter than the latest Windows versions, so on weak hardware the lighter flavours of Ubuntu, like Xubuntu and Lubuntu, may be better. They often perform better than Windows on weak hardware.

1: You can create a live usb and try Ubuntu or a different flavour without installing it. This won't do anything to your hard drive, you can just reboot and remove the live usb and your computer will boot Windows as if nothing has happened. Do you like the user interface? You can tweak it extensively and try the other flavours. Does it feel fast? After installation it should be a little faster still.

Installing is a different matter. You'll have to make room on your hard drive where you can create a few new partitions for Ubuntu or you have to add another hard drive to your computer. The procedure depends on whether your Windows is installed in traditional BIOS mode or in UEFI mode.
[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)
[https://help.ubuntu.com/community/UEFI](https://help.ubuntu.com/community/UEFI)

2: Difficult to predict. It depends also on your graphics hardware.

3: Obviously there is no MS Visual Studio or MS Office on Ubuntu. There are alternatives, similar, but not identical. Java, and consequently Android Studio, are available.

4: Most things can be done in the GUI, and everything can be done in the terminal. If you don't like terminals, you can live without. Unless you break something, as the terminal is often the only place to fix a broken system. I prefer the GUI if I want graphical output, but stick to the terminal if I want to give mostly input or if there's very little I/O altogether. Or I use tools that use terminal input and graphical output in separate windows.
