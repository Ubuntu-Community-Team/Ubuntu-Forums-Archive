### Post by tkiesel on 2006-03-25
Hi jive_turkey,

(Finally I've had a valid reason to say that!)

Ahem...

To set the default program for use with a file type like a .mp3 file, you right click on a file of that type in the File Browser.  Select the option at the bottom titled Properties.

Then, you choose the Open With tab, and therein select a program that Ubuntu will use to open files of that type.  Then hit the Close button after you've selected the program you want, and you're set!!

Have a great day!
