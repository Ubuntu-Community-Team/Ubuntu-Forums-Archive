### Post by AdotB on 2010-03-20
If all you need is to record names and into a spreadsheet, then you probably don't need a database. Just use OpenOffice Spreadsheet (aka Calc).

It should already be installed in Ubuntu, and in the Applications menu:
Applications -> Office -> OpenOffice.org Spreadsheet
