### Post by ddjolley on 2012-04-12
A few weeks ago, I was turned on to Simple Scan by this forum. My purpose was to be able to scan documents and save them as a PDF file. Initially, I loved it.  It does all I need to do and it works out of the box.  However, after using it for a while, I have discovered that the size of my PDF files are huge when compared to the size of files others are producing.  For example, our federal tax return which is 104 pages in length was sent to me as a pdf by our accountant.  The file size was less than 0.5 MB.  If I were to scan a single page document using Simple Scan, the resulting file would be larger than that. So, my question is this:  How can I produce smaller pdf documents using Simple Scan?  I want to do this because the whole objective is to scan documents so that they can be attached to emails.  These huge file sizes are pretty much a deal killer.  Thanks for any help.

      ... doug

### Post by Morbius1 on 2012-04-12
Don't know if this will help but:

[1] In Simple Scan select Document > Scan > Text ( I think Photo is the default and it's default resolution is at least twice that of Text ).

[2] Don't Print to PDF, use Save as.

---

### Post by ajgreeny on 2012-04-12
Go to the Documents ->Preferences in the menu and you can change the dpi etc etc of the output file to suit your needs.

### Post by ddjolley on 2012-04-12
It appears that the key to solving this problem is to select 'text' rather than the default, 'photo'.  Major difference.  Thanks to all who contributed.
