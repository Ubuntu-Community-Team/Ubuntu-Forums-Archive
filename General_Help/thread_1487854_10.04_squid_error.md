[SIZE=2]
[/SIZE][FONT=Arial Narrow][SIZE=2][COLOR=Red]Remove the older version of squid,[/COLOR][/SIZE][/FONT]

```
sudo apt-get autoremove squid
```

sudo /etc/init.d/squid3 restart
sudo /etc/init.d/squid3 reload
```
