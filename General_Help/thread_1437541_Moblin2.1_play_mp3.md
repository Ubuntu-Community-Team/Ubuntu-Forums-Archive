### Post by darolu on 2010-03-24
Uhmmm isn't Moblin based on Fedora? if you have a Moblin based on Ubuntu (or debian) you may be able to add Ubuntu Multiverse repository and install "ubuntu-restricted-extras", that package adds support for restricted multimedia formats.

Other option is to download all the green-bullet items listed here: [http://packages.ubuntu.com/karmic/ubuntu-restricted-extras](http://packages.ubuntu.com/karmic/ubuntu-restricted-extras)

If it's based on Fedora, you'll be needing to add their repos or download the RPM's they use to add restricted-formats support.
