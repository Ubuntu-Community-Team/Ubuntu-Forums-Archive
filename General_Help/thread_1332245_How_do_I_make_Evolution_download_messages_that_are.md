### Post by mocoloco on 2009-11-21
Actually if it's pop then you just need to make sure the messages are still on the server.  If Outlook is set to download messages then they are removed from the server and you can't get them on another system.  You'll need to set Outlook to leave a copy of the messages on the server.  Actually you'll need to set Evolution to do that too, then either system can get them.
