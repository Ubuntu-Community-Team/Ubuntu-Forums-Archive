### Post by 3rdalbum on 2009-04-12
Just install kde-core and run it in Sessions on the login screen. All your Gnome programs will still be available. The KDE menu won't show some of the Gnome administration and preferences programs but you can run them if you know their name (for instance, the System Log program is called "gnome-system-log").
