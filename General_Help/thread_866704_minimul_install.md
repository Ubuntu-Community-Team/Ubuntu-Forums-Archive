### Post by eriqjaffe on 2008-07-22
> **pedrom169 said:**
> what programs would you recommend me using?

Openbox?
any login manager?
panel?Depends on how lightweight you want things, and how comfortable you are with configuration via the terminal and via text file.

For me, once you've gotten the CLI going, would be to install to following:

Xorg - kind of important if you want a graphical interface.  ;)
Openbox - Spartan, but great.
Obconf - GUI configuration tool for Openbox.
Obmenu - GUI menu editor for Openbox.
Xterm - terminal emulator, beats having to toggle between TTYs.
PCManFM - my file manager of choice, it can do quite a bit without being too resource-heavy.
LXpanel - it's forked from fbpanel.  Pypanel is also nice.
Firefox - or whatever web browser you choose.  Firefox is a bit of a heavyweight, though - on my old P2 laptop, I use Opera.
Wicd - my personal favorite tool for managing network connections - made setting WiFi up on my laptop a breeze.
Slim - lightweight login manager, although you may prefer to just log in at the command line and startx.
