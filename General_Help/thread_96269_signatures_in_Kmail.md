### Post by hvbakel on 2005-11-28
Go to the kmail settings, there you can add new identities for yourself, each with it's own signature. When composing a new message, you can use the pull down menu to set your identity (you may first need to select the 'identity' checkbox in the 'view' menu). Then add the signature via the 'attach' menu. The identities are actually quite versatile, you can even set them for mail folders, so that all messages that you send or reply to from a certain folder are always sent with a certain identity.
