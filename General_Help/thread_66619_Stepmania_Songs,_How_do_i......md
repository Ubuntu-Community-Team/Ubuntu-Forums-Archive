### Post by gerbman on 2005-09-18
[QUOTE=Eibo]Um i downloaded StepMania and then downloaded a song package and it still says i have no songs how do i put the songs into STepMania?[/QUOTE]

What is the song "package"?  Songs need to be put in the "songs" directory in Stepmania's install folder.  Each song needs its own folder in that directory.  [This](http://www.ddruk.com/downloads.php?subcat=57) is a good source of songs.  Unzip the different mixes to Stepmania's "songs" folder.  That worked fine for me.  Post any problems and I'll try to help out.

I think this is not the right place to post questions like this, since it's not actually linux problem at all. [-X 

But anyway if you have song packages(smzip), then you have to put them into [HTML]~\stepmania\packages\songpackname\file.smzip[/HTML]
and if they are extracted files then they go to:
[HTML]songs\mixname\songname\[/HTML]
they work in the same way in linux and in windows, so if you have stepmania problems like this, you could use bemanystyle forums or ddruk forums etc.  ;-)
