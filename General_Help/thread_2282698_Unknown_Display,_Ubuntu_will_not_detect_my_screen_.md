### Post by grahammechanical on 2015-06-16
The Displays utility is useful if we are running with an open source video driver but if we are using a proprietary video driver then it is best to use the settings manager that came with the proprietary video driver as the Displays utility becomes less than useful when a proprietary video driver is used.
