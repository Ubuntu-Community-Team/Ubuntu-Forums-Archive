### Post by erkki70 on 2005-04-15
Hi,
I guess thatyou're using Ubuntu PPC version and browsers are crashing because you miss the flash plugin.
Installing flash plugin will allow you to access the sites but it might be that you won't see any animations because the flash plugin is a bit dated...
So first, activate universe repository then in synaptic search packages with flash as a keyword and install them.
Your Firefox browser won't crash after this but I don't know about epiphany or galeon though.
You will find more install infos here:
[http://ubuntuforums.org/showthread.php?t=14777](http://ubuntuforums.org/showthread.php?t=14777)
and searching for flash in the Ubuntu ppc forums.

Good luck and hope this helps.
