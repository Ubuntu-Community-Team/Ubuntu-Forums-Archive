### Post by hikaricore on 2008-12-14
If you're running any version of Ubuntu later than gutsy this is what you need to do:

> wget mirrors.kernel.org/ubuntu/pool/universe/i/ircii-pana/bitchx_1.1-4ubuntu4_i386.deb
sudo dpkg -i bitchx_1.1-4ubuntu4_i386.deb

There ya go ^_^
