### Post by ajgreeny on 2007-06-01
Are you sure you are logging on to the internet with the same ISP as the smtp server?  Most ISPs will not let you send mail if you are logged on with another ISP.  The only way you can do so is to set up your email account to use the mail server of the ISP which logs you on to the web.  This situation can be a real pain if you are using a laptop on lots of different  wireless accounts unless you can change your email to a web based one like googlemail, etc.
