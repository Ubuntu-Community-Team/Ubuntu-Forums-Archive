### Post by Magnentius on 2008-04-20
No, K3B does not support burning to multiple drives simultaneously. When you start burning, the main K3b window just locks up so you can't start burning other disks. Which is imho a bug, why else would you have multiple DVD burners? Has anyone a workaround?

I found a feature request @ [http://k3b.plainblack.com/message-board/burning-to-multiple-drives-at-once](http://k3b.plainblack.com/message-board/burning-to-multiple-drives-at-once) so I guess k3b can't do this yet.
