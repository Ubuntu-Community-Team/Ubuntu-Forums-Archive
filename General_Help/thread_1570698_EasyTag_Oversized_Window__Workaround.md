### Post by Gary_inNYC on 2010-09-08
For some reason, EasyTag isn't usable with certain desktop resolutions such as 1024x768, which my old Thinkpad t40 still uses.  

Since this is an old issue, reaching 3 years*, I'm assuming a fix for this is not coming, or is not a priority.  Clues as to a fix led me to the easytagrc file, but upon playing with the options, I wasn't able to reproduce the original solution*.  

The application is too awesome not to use in my opinion, so the workaround I conceded to was merely unmaximizing the window (right click its taskbar entry, then Unmaximize).

It's not a fix, but it does stop the annoying window jumping behavior completely so that it's usable again.  If window placement is an issue, just alt+drag it :D

I'm not gonna call this solved, as this is just an ugly workaround.
