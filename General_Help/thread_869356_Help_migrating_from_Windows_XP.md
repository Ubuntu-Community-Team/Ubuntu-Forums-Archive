### Post by lordadi on 2008-07-24
Hi,

> Evolution (is there an easy way to migrate emails, contacts and calendar?)

The way I did it, was to import my Outlook file (*.pst) into Thunderbird and then find TBird's mbox files which can be imported into Evolution.

This is a rough way to do it [LinuxQuestions.org thread]("http://www.linuxquestions.org/questions/jds-61/outlook-to-evolution-244811/").

Or you can try [Outport]("http://outport.sourceforge.net/").
