### Post by LordNeo on 2011-05-25
Well, those are some "special" installs for some requirements:

Server:
Installs a minimal CLI (command line interface) with internet wich is more than enought to install a web server, mail server, ftp server, file sharing server and stuff like that. It uses barely no resources so you can spend all your CPU/RAM in running the services. It doesn't install a Graphical Enviroment, and it's not meant for "End User" but for a 24/7 server computer.

Cloud:
Mostly the same as a server install, but this focus on creating virtualized machines services, so you can "create" several server machines all running from inside only one physical machine. You could google for "virtualization" and "cloud computing" if you need more info.

Best regards
