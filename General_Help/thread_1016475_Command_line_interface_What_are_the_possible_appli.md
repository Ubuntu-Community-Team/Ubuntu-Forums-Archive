### Post by cariboo on 2008-12-19
For an organizer have a look at [calcurse]("http://manpages.ubuntu.com/manpages/intrepid/man1/calcurse.1.html") and have a look [here]("http://www.linux.com/feature/58720"), for a run down on text based email clients.

For a simple calendar at the prompt type:

```
calendar
```

have a look at:

```
man calendar
```

for more info.

Jim

### Post by x1a4 on 2008-12-19
fbi - image viewer (framebuffer only; will not work in a terminal)
weechat - irc client
vim - text editor
[benmp3]("http://sourceforge.net/projects/benmp3/") - mp3 CD and Wav player
console-freecell - freecell cardgame
mc - midnight commander file manager
[MJS]("http://mjs.sourceforge.net/") - MP3 Jukebox System
cmus - audio player
ogg123 - ogg vorbis player
mpg123 - mp3 player
mp3blaster - mp3 and ogg vorbis player
ogginfo - show ogg file info
mpd - music player deamon; server that allows to play audio files, like flac, mp3, ogg vorbis, wav and aiff.  
mpc - mpd client
rtorrent - bittorrent client
finch - instant messenger 
mutt - powerful email client
fdisk - partition manager
sed - stream editor for filtering and transforming text
gawk - pattern scanning and processing tool
raggle - rss reader
screen - screen multiplexer
