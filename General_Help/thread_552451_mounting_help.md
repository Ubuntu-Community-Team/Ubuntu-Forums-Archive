### Post by Happy_Man on 2007-09-16
Well, since Ubuntu doesn't want to make it for you, open a terminal and type this: ```
sudo mkdir /media/DVD
```

Or, if you don't want to do it that way, press alt-f2 and type "gksudo nautilus", then navigate to the /media folder and create a new folder in it called "DVD" (has to be uppercase, Linux names are case-sensitive).
