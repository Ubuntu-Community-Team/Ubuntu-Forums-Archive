### Post by mc4man on 2010-12-31
> 3- Any drive or partition I open, it's icon will be placed on desktop.
open configuration editor (in terminal gconf-editor
apps -> nautilus -> desktop, uncheck 'volumes visible'

> anyhow when deleting a file or folder it will do it without warning. sure it is in trash but it is better to have a warning message.
nautilus -> edit -> preferences -> behavior -> trash, enable 'Ask before...' (or access in terminal w/
 nautilus-file-management-properties

#1 happens sometimes, will never change
#4 is avail. on r. click
#2 isn't 'auto', but again r. click
