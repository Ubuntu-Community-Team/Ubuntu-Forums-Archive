### Post by Mark Phelps on 2011-02-01
You have only two options ...

First, if the machine came preinstalled with MS Windows, there is likely a function key (usually F11) you can press to restore the machine to its original condition -- presuming that you did not erase the recovery partition when you installed Ubuntu

Second, again, if the machine came presintalled with MS Windows, it might have come with a Recovery CD -- which does essentially the same thing.

If you don't have either of these, and the machine came preinstalled ... then you will have to contact the PC supplier and see how you can obtain a set of full-restore CDs (or more likely, a DVD).  Tell them that your machine crashed with a virus and you need to completely restore it to its original condition.
