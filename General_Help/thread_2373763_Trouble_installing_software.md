### Post by Autodave on 2017-10-09
Are they .deb files?  If so, the easiest way is to go to the software center and download *gdebi *and use that to install those programs.

You need to understand the Linux is not Windows. It functions quite differently. With very few exceptions, and program that you are going to install requires "dependencies". These are other programs needed to make the program you want, to operate. Without the dependencies, it will not function or even install. *gdebi *will take care of the dependencies required. Again, the program that you want will need to have a .*deb* extension.
