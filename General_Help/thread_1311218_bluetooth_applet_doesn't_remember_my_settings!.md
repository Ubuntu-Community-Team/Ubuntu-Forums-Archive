### Post by corruptor1972 on 2011-02-13
You could use a different Bluetooth manager.  I use Blueman, which has a disable option that is remembered.

Search for 'blueman' in the Ubuntu Software Centre and install.

You can get rid of the default Bluetooth manager by uninstalling the 'bluetooth' package.
