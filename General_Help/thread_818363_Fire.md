### Post by Shazaam on 2008-06-04

### Post by cyclonedr on 2008-06-04
Hey man open the CompizConfig Settings Manger by going to System->Preferences->Advanced Desktop Effects Settings.  Now Click on animation.  Make sure your on the close animation tab.  Double click the first line and select burn.  Set the duration to about 400 or so, you can edit this later.  You don't have to have fire enable for this to work.  If you want the effect to work when you minimize the window go to Minimize animation tab and double click the first entry.  Select Burn and set the time to 200 or 300.  Hope this works for you.
