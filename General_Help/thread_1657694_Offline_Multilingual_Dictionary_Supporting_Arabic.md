### Post by Akhnatoun on 2011-01-01
For whom seeking to have an offline multilingual dictionary , "**GoldenDict**" is a great software that can translate between several languages at once, these are some of its unique features:

[LIST=1]
[*]Support of multiple dictionary file formats, namely:
   Babylon .BGL files, complete with images and resources
   StarDict .ifo/.dict./.idx/.syn dictionaries
   Dictd .index/.dict(.dz) dictionary files
   ABBYY Lingvo .dsl source files, together with abbreviations.
[*]Support for Wikipedia, Wiktionary, or any other MediaWiki-based sites to perform lookups in.
[*]Scan popup functionality. A small window pops up with the translation of a word chosen from another application, similar to Babylon in Windows.
[*]Support for global hotkeys. You can spawn the program window at any point, or directly translate a word from the clipboard by highlighting the word and pressing ctrl+c+c "you're able to change this hotkey at anytime".
[/LIST]

GoldenDict is available in the main Ubuntu repository.

For people looking for offline dictionaries, they can find them free at Babylon website [http://www.babylon.com/free-dictionaries/](http://www.babylon.com/free-dictionaries/)

For whom looking for Arabic dictionaries, you can download my best collection of Arabic dictionaries that are working directly with GoldenDict without any conversion from this link [ArabicDict.tar.bz2](http://www.4shared.com/file/fes4fNVQ/ArabicDicttar.html). Just use them by extracting the file and adding the extracted directory to the Dictionary Sources of the program.

Hope this be helpful ):P
