### Post by freerick on 2007-03-13
I had problems installing Flash from the repos as well. The solution that I found was to install Swiftfox, which is a modified version of Firefox, which appears to be optimized for your architecture. After I installed it, I went to YouTube and it popped up the yellow bar, saying I need flash. I clicked on it and all was well!

Do a Google search for Swiftfox or check if it's available in Synaptic.

(obviously any flash-enabled site will do, not just YouTube :-) )
