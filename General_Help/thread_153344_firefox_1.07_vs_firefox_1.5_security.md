### Post by Rebeca on 2006-03-31
There have been a few changes security-wise from 1.07 to 1.5, and you can look them up here: [http://www.mozilla.org/projects/security/known-vulnerabilities.html#firefox1.5.0.1](http://www.mozilla.org/projects/security/known-vulnerabilities.html#firefox1.5.0.1)

If those don't seem too bad for you, I would suggest you take in count some of the bug fixes in 1.5: [http://www.squarefree.com/burningedge/releases/1.5.0.1.html](http://www.squarefree.com/burningedge/releases/1.5.0.1.html)

Either way, it's really easy to upgrade firefox using automatix.
