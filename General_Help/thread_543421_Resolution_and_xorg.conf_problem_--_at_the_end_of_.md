### Post by _simon_ on 2007-09-05
With nvidia-settings you have to run it with root privileges to save changes.

Before you make any changes though, backup your xorg

```

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup

```

```

sudo nvidia-settings
```

And make sure you press save to save the changes.

### Post by Steve1961 on 2007-09-05
Try this.  Open a terminal and type something similar to the following:

gtf 1280 1024 75

This equates to 1280x1024 with a refresh rate of 75.  Replace with the values for your monitor.  This generates a modeline similar to:

Modeline "1280x1024_75.00"  138.54  1280 1368 1504 1728  1024 1025 1028 1069  -HSync +Vsync

Insert this in your monitor section.  i.e.

Section "Monitor"
	Identifier	"HP w1907"
	HorizSync	30-83
	VertRefresh	55-75
        Modeline "1280x1024_75.00"  138.54  1280 1368 1504 1728  1024 1025 1028 1069  -HSync +Vsync
	Option		"DPMS"
EndSection

Then in the screen section of xorg replace all your instances of 1400x900 (or whatever you're using) with the first section of the modeline, in my case that would be "1280x1024_75".

Make sure the values you use, and the modeline you generate, all correspond to the resolution and refresh rate you want.

Once done, save xorg.conf and restart x
