### Post by nl4m on 2010-04-23
Try updating it through the ethernet cable. It may fix the problem. Try "pinging" a site. Click on "System" - "Administration" - "System Monitor" and click on the "Resources" tab. Now go download a large file and return to the "Resources" tab to see if the connection is dropped. Or maybe try reading the error logs in file /var/log. Hope these ideas help.
