### Post by coffeecat on 2007-05-31
I know you didn't ask this specifically, but a couple of related tips.

It's worth installing the same fonts in both Windows and Linux for minimum hassle when going from OOo in Linux to Word in Windows. If you haven't already done so it's worth installing the MS core fonts which will give you Times New Roman among others. I think the package is msttcorefonts but I'm not in Ubuntu atm so I can't check.

Be aware that page breaks and line breaks **may** come out slightly differently when you take a document from Linux to Windows. So do your final tidying up in Word. On one occasion I spent a long time making sure page breaks were exactly where I wanted them in a several hundred page document in OOo in Linux and then found that they came out differently **in the same release version of OOo in Windows set up for the same printer**. :(

One way round this is to save the document as a .pdf file. You can do this in Open Office but not in Abiword. (Go to File > export as PDF). You won't be able to edit the pdf file but at least when you print it out WYSIWYG.
