### Post by taurus on 2008-11-01
Click a torrent link.  Another window will open and highlight "_O_pen with" option and on the drop down window on the right, pick transmission if it's on the list.  Otherwise, browse to /usr/bin and pick transmission from there.  Then, highlight the third option "Do this _a_utomatically for files like this from now on." so transmission will start automatically each time you click a torrent file.
