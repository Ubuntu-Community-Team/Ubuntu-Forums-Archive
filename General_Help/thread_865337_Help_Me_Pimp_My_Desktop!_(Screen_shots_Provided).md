### Post by RuleMaker on 2008-07-20
1) Change your desktop background
2) in "system-->preferences-->advanced desktop effect settings" click on desktop cube, expand the skydome (if isn't) and chose your background (the background behind the cube)
3) change the side pictures of cube in the same menu
4) change the cube opacity while rotating to 30
5) install some good GTK theme, i recommend the "Intuition".

### Post by Sealbhach on 2008-07-20
If you install Ubuntu Tweak

```
sudo apt-get install ubuntu-tweak
```

it gives you the option to install a Simplified Compiz Configuration Settings Manager which will show up in the System/Preferences menu.

I find that much easier to use for setting Animations and Window switcher.

You can also download themes from [www.gnome-look.org](www.gnome-look.org) and drag them into the Appearance window and they will install just like that without much effort on your part. Just select the Customize button and you can experiment all you like with the appearance.


.
