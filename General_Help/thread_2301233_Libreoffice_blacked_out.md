### Post by Dennis N on 2015-11-01
Searching for themes within Libre Office (Tools > Options > Personalization) only searches Firefox themes (these used to be called Personas) that decorate the toolbar only and won't affect the text backgrounds. These are the same ones you can find in Firefox's Tools > Addons.

If you haven't tried changing the Ubuntu theme in whatever theme changing tool you are using (like Ubuntu Tweak), you should try a different one of those and then you may see some improvement. A combination of this with the Firefox themes may give the best result.

For Firefox themes, I suggest you get the exact name of the one you want first (use Tools > addons in Firefox, or visit the Firefox site), then enter it in Libre Office's search tool.
