### Post by beebelo on 2007-08-10
I'm not familiar with Emprex (not that it means anything), so I did a google search for "emprex dvd linux" and discovered that your problem is not uncommon.  There was one poster that said it started working when he changed the jumper to 'cable select'.

In another post someone was talking about proprietary firmware issues, as in, the manufacturer designed it for windows only.  (There are modems and printers like that--they require the windows operating system to function.)

So do some searching.  Use the exact model number.   Try the jumper trick if it's not already set for cable select.

Hope it works!  Be sure to post back if you solve the problem.
