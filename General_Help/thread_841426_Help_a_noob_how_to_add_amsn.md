### Post by dorkdork777 on 2008-06-26
What does the downloaded file end with? If it's a .deb, you can just double-click it and select 'install package', and it'll do the work for you. If it isn't, then the install will be more complicated. For this reason, I just stick to .deb installers for the most part - they're easier. So delete your non-DEB amsn file; this is how to get the DEB.

So, if you don't have the .deb installer, you can find it here:
[http://www.getdeb.net/app/aMSN](http://www.getdeb.net/app/aMSN)

Select your system architecture at the bottom (32 or 64 bit - if you're not sure, go for 32 bit) then hit the download button. Once it's done, you can just double-click the .deb file, and click 'Install Package' - the rest will be done for you.

It's a good idea to remember getdeb.net or bookmark it, as it's an excellent resource for DEBs.

### Post by Taxman415a on 2008-06-26
twin-08, start over and do it the easy way. In Ubuntu, just go to the Applications menu then Add/Remove. Then in the search box type amsn. Then check the box on the left. Then hit apply changes. It will install for you and you can run it from there and it will show up in your applications menu to run in the future. 

In Ubuntu you don't generally need to download installers separately.
