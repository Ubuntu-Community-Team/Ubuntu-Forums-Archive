### Post by chrispche on 2008-04-27
Added some themes from gnome-look.org. How do I enable emerald? Clicking on them in emerald does nothing. Do I need to restart?

---

### Post by akshar_patel_47 on 2008-04-27
Nope... Just add the code 
```
emerald --replace
```
as a command in system->preferences->sessions->startup programs add
and then restart your computer.
Now if you click on any emerald themes in emerald themes manager they will change automatically.
