### Post by Kir_B on 2010-05-01
I'm a bit of a noob myself, but the only thing I can think of, is that perhaps you've got yourself a broken package that is a dependent of the Transmission application.

Have you checked the Synaptic package manager's "broken" section?

Open the "synaptic package manager" and select "custom filters" (near the bottom left), then click on the "broken" tab, to see if there is a broken package in your system. If there is, select the option that will fix it. I had one quite a while ago that caused my system to use an extraordinary amount of resources with some applications, so this may be the case for you too. I hope this the cause of your problem. If not, your going to require  a more experienced person than I am.

I hope that I was helpful. Best of luck, Kirby

P.S. use the "force Quit" option when ever you have to. You can add it to your panel, via a right click on the panel, then selecting "Add To Panel" and then select the "Force Quit" application. It's quite handy for run away programs, which doesn't happen very often, but when it does, Its perfect.
