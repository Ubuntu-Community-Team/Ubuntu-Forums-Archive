Hi there,
I have found on this forum many threads about Volumouse which is a great app for Windows.
So I have decided to make Volumouse for Linux.
This is a Python 3 program that allows you to change the volume with the mouse wheel using it at the four corners of the screen.
If you are interested, please download it there : [https://github.com/pzim-devdata/volumouse](https://github.com/pzim-devdata/volumouse)
