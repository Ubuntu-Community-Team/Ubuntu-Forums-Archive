Does the Canadian Dictionary you selected have a checkmark in front of it?  If not, you need to add the dictionary, not using a wizard as in previous versions, but through the Tools > Extension Manager.  In version 3 you need to download the dictionary you want then use the extension Manager to Add it.

Give that a go and let us know how it works.

if that doesn't work or apply to your situation, try the following:

Open your document - select all of the text and then Format > Character > and language of the dictionary you wish to use.
