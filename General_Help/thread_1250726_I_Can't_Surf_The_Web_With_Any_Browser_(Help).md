### Post by cariboo on 2009-08-26
You put your question in the wrong section, I moved it here to General Help.

Open an Applications-->Accessories-->Terminal and type:

```
ping www.google.com
```

if that doesn't work type:

```
ping 74.125.155.105
```

If you can ping by number but not by name you have a DNS problem.

Post back when you have tried the above.
