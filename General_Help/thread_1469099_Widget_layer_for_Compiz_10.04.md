### Post by jjsonp on 2010-05-01
In 9.10 I used to autohide my top & bottom panels, then install compiz, then select the widget layer.

### Post by WorMzy on 2010-05-01
Have you installed compiz-fusion-plugins-extra? You need that to get the widget layer. It appears on mine, so if you have that package, try reinstalling it.
