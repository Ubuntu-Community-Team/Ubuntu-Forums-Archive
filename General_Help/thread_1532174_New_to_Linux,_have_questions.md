### Post by Mark Phelps on 2010-07-17
> **Beardface said:**
> I've heard that when installing Ubuntu that it prompts you to partition your hard drive. I don't think that this is the case.?
If you're planning on using the traditional dual-boot approach, in which each OS has its own separate set of partitions, do NOT allow the Ubuntu installer to shrink the Vista partition.  Doing so runs the risk of corrupting the Vista OS and rendering it unbootable.  Instead, use the Vista Disk Management utility to shrink the Vista OS partition, making room for Ubuntu.

Then, boot from the Ubuntu CD and use the "largest free space" installer option to allow it to do its own partitioning in the free space you created.

As to free partitioning tools that run in Vista, look for EASEUS Partition Master.  They provide a free download version.
