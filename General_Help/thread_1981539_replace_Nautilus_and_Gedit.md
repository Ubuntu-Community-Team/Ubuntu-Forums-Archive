### Post by drs305 on 2012-05-16
Both appear to be in the standard repositories. Of course, you'll have to install a lot of dependencies as well, which should come along automatically.

After installing, go to Preferred Applications to see if they are an option. Otherwise, right click the type of file you'll use with them and set them as a preference that way.
