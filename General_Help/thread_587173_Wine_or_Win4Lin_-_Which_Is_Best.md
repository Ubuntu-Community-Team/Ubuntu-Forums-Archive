### Post by alek01 on 2007-12-13
Which one is best depends on which Win sofware you want to run. All the Win on Linux packages have limitations because the best Windows implementation is done by Microsoft!
On my Ubuntu Gutsy I've just tried the trial of CrossOver (It's the commercial version of Wine) available on Automatix. Firstly, you can only get as far as Win 2000 and for many soft the system tells you that a Win98 "Bottle" (or virtual machine) would be better. Second, the latest soft are unlikely to work. Office 2000 would be 100% fine but Office 2003 has some limitations. I installed Office 2003 (Office11) in a Win 2000 Bottle and Outlook couldn't connect to the internet. The system told me a Win98 bottle would be better. But it didn't solve the problem.
So, before deciding, I would list the software I want to run on the Win emulation and try the alternatives. Even Win4Lin which requires a licensed Windows disk, will not run 100% of the windows software.
It might better to learn a new Linux software than try to teak a Windows platform on Linux.
My problem is Outlook. Evolution is a lot inferior in following-up tasks and Email and most importantly it sync poorly and with uncertainties with my Blackberry. I have not given-up yet, but it may happen.
