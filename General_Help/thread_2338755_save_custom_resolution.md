### Post by Bucky Ball on 2016-10-01
Welcome. I use arandr myself. It is a GUI for xrandr basically. Install that from the repositories, set up the resolution and you will find a 'Save' icon or you can save your configuration via File> Save As from memory.

You can also install via a terminal with

```
sudo apt-get install arandr
```

Which release of Ubuntu are you using?

* PS: When I was using this method, I set up a configuration in arandr, checked the command that created, made a launcher and used that command. That way, when I switched on the machine I could just double click the launcher and the resolutions would be set. No need to open arandr.
