### Post by spacesamurai on 2009-05-12
Given the advice up until now, check to see what shell your user has. 

#echo $SHELL

If the answer is /bin/bash, you need to copy the skeleton file

cp /etc/skel/.bashrc ~/.bashrc

Else if you are using another shell, depending on the shell you may not be able to use .bashrc.
