### Post by Sef on 2006-05-05
My favorite is software to burn with is K3b.  Download it from Synaptic or the Terminal.

Synaptic 

System > Administration > Synaptic Package Manger > Search > Hightlight > click apply.

Terminal

Applications > Accessories > Terminal

sudo apt-get update

sudo apt-get install K3b

Two sites with info on installing:

1) [http://monkeyblog.org/ubuntu/installing.html#navigating_the_terminal]("http://monkeyblog.org/ubuntu/installing.html#navigating_the_terminal")

2) [http://www.psychocats.net/ubuntu/installingsoftware.php]("http://www.psychocats.net/ubuntu/installingsoftware.php")

### Post by glotz on 2006-05-05
Nautilus has inbuilt CD burner. Make your selection and browse the menus. Or if you want to burn an .ISO file, just right click in and say BURN BABY BURN!

Likewise, if you want to copy a cd, just pop it in and right-click on the icon that appears on the desktop and select "copy".  A mixed-mode or audio cd will be handled by serpentine, the audio cd creator that is built-in to Ubuntu.
