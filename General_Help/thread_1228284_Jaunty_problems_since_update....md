### Post by guillaume_u on 2009-07-31
Hi,

No problems for me and, I guess, for most people. The only piece of advice I can give you is to use the _[version of Firefox from Mozilla]("http://www.mozilla.com/en-US/firefox/all.html")_. It's nicely packaged (no compilation required) and you get the very latest updates (I run version 3.5.1). I have Thunderbird likewise. 

The only drawback is that this is a per-user installation, unless you run Firefox as root every time you want to update.
