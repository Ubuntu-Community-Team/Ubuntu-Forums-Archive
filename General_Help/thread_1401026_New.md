### Post by TheShader on 2010-02-07
Hey there, considering my own experiences. Gnome is a lot easier for beginners. It's simpler than KDE. You can switch to KDE when you get some experience. The best way of learning Linux is experience... Try different things, download different distros and try them all.

About the drivers, I don't think you would have any problems... Latest Linux kernel is ok with most of the hardware. Just boot with the live CD and check what is working. Oh and a little thing to consider: If you connect to the internet with wifi, your wifi card could be restricted and you would need to download the driver for it on Ubuntu. But if you don't have a working wifi, how can you download the driver for it lol? :p prepeare a LAN cable just in case... Good luck bro!

### Post by audiomick on 2010-02-07
Another vote for Gnome for a beginner. Less choices to customise it = less chance to break it... ;)

If you are concerned about messing anything up, have a bit of a play with the live CD "try without changing" first. It is slow, but a completely safe way to look at Ubuntu before you commit.

As far as learning goes, my experience is that you only really start learning when you make a conscious decision to use Ubuntu for everything, and only boot into windows for things that really don't work on Linux, like maybe a particular game, or when you have to get something done to a deadline and don't have time to find out how to do it on Ubuntu (and that doesn't mean "can't be bothered right know").

For info, the stickies at the start of the "absolute beginners" section are a good start.
