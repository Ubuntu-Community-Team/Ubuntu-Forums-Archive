Found some source code for MEradio here: [http://java4me.blogspot.com/2009/01/meradio-shoutcast-radio-streaming.html](http://java4me.blogspot.com/2009/01/meradio-shoutcast-radio-streaming.html)
Is it the same one?

If yes, it looks like the .db file was created by the Java "Record Management System":
[http://en.wikipedia.org/wiki/Record_Management_System](http://en.wikipedia.org/wiki/Record_Management_System)
[http://developers.sun.com/mobility/midp/articles/databaserms/](http://developers.sun.com/mobility/midp/articles/databaserms/)

So it's stored as a "byte array".
Maybe it's possible to find a java RMS editor somewhere. Otherwise creating an editor for it shouldn't be too hard. (Or even improving the source code so it stores the settings you want.)
