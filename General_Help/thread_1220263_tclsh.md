### Post by ea1776 on 2009-07-23
Nevermind, I figured out a way to install tcl.  I went through the Ubuntu GUI.

System->Administration->Synaptics Package Manager

Once that was open, Edit->Reload Packages
Then, select tcl for installation and it will download/install automagically.
