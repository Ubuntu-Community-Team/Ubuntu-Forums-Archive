### Post by emelur on 2008-03-05
I've been trying to move my references from EndNote X (Win 2k) to bibus (Gusty). Exporting references from endnote and importing them in bibus within the Windows environment works fine. Using the same procedures for importing them to bibus in Gutsy does however end up whit just half the EndNote library being imported!

Has anyone else had this kind of trouble...?

---

### Post by emelur on 2008-03-05
Solved it myself! \\:D/ 

For the record: bibus does not recognise all the different types of references in EndNoteX. For example, 'online database' and 'newspaper article' seemed to cause trouble. I simply removed them from the .txt file and suddenly, importing references was a smooth business!
