### Post by sgosnell on 2010-08-02
Ubuntu is a Linux variant, so yes, it's compatible.  MS Office, though, is a Windows application, and won't run under Ubuntu.  It will run to some extent through wine, but it's not worth the trouble.  Open Office is a Linux application that is almost totally compatible with MS Office, though, and will work fine for most of the things most users need.  I regularly use it to deal with Excel spreadsheets and Word documents at work, and have had no issues.  Some people report problems with formatting when using complicated documents, but I haven't seen any problems.
