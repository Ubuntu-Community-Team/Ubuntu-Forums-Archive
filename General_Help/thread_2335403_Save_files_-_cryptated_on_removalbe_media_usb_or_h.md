### Post by mcduck on 2016-08-28
Take a look at EncFS, it allows you to create encrypted folders, which can be located pretty much anywhere, on removable drives or even in cloud storage like Google Drive... My preferred tool for using it is [Gnome EncFS Manager]("http://www.libertyzero.com/GEncfsM/").
