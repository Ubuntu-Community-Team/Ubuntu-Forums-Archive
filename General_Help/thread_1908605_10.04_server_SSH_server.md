### Post by cryptotheslow on 2012-01-13
Use Synaptic Package Manager (apt-get from the command line for the server) to install **openssh-server** and **openssh-client**

Make sure to use a very good/strong password or preferably keyfiles for authentication - especially if the box is internet facing.

Clearly you don't need to install the server package on the desktop if you only intend to ssh **from** that to the server - just the client needed in that scenario.
