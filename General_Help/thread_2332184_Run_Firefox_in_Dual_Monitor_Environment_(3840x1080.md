### Post by kyrithis on 2016-07-29
Fullscreen mode, by nature, takes up a "full screen". The way most fullscreen applications are made is that they can only cover one monitor while in fullscreen mode. I'm afraid this is simply the way programs work with monitors. To my knowledge, there is no way to extend Firefox to both displays in the way you'd like it to be done.
