### Post by whoop on 2010-04-06
It uses XFCE instead of GNOME, but xubuntu is a little bit bloated by gnome packages so it is not as snappy as it could/should be. It's still snappier than Ubuntu though.
So it is good for old(er) machines. Also it could prove to be useful when Gnome 3.0 is released and users that don't like it could switch to XFCE (it will basically be the old GNOME when GNOME 3 gets released).
