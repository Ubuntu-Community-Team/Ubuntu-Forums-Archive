Wine is a reverse-engineered Windows compatibility layer. The key thing to take from this is 'reverse-engineered'. It is not and probably never will be completely compatible with Windows.

For programs that do not require 3D acceleration, you are much better off installing Windows on a virtual machine. Why not just install Windows? Because you can take a snapshot of your virtual machine and if it ever screws up, easily restore it to the snapshot state. Also, you can do your web browsing in Linux and not have to worry about hitting a drive-by malware site. The idea is to use Windows as little as possible in order to minimize exposure to viruses.

No, this isn't as user friendly as Windows.
