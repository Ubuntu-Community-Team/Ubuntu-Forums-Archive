### Post by Wayne_V on 2009-06-11
the i386 version should run on both systems.  only use the AMD64 version if you have a 64 bit (AMD or Intel) processor and want to run it in 64 bit mode (you can stick with the i386 version on 64 bit cpus if you wish).

sounds like your second system may have USB issues.  google the motherboard model number and look for known issues.   check if there are any bios updates as well.  also check the bios settings for anything usb related that may seem relevant.
