> **bradsorensen said:**

> **bradsorensen said:**
> I am nervous because I know Windows in and out and the software, where are some good places to get software and games for Linux Ubuntu? Things like wallpapers, screen savers oh very important I do allot with digital pictures and both watching dvd movies and creating dvd slide shows. I also do allot of burning/ripping music cd's and dvd's to make backup copies.

You can download several screensavers from the repos, just launch synaptic or adept, and search for screensaver.  :)

As far as wallpapers there are thousands of great images at: [http://www.gnome-look.org/](http://www.gnome-look.org/)
(i even saw some of the Ubuntu CE backgrounds listed there)

There are several cd ripping software titles in the ubuntu repos.

And for ripping and backing up dvds, I would suggest installing [Automatix]("http://www.getautomatix.com/wiki").
It has a few different dvd rippers as well as video editing software you might find useful.

> **bradsorensen said:**
> Now down to business.
I don't know how to install software and or games in Ubuntu, any pointers? Is is like in windows were you click on the file and it installs for you?

You can install tons of software directly from the ubuntu repositories, either from the Add/Remove Programs function, synaptic (or adept package manager in kubuntu), or using command line with apt-get.  There are several unsupported repos availible from different sub communities and users you can add to your sources.list to add software that ubuntu doesn't offer by default.

You may also want to check out: [Ubuntu Click and Run]("http://www.getdeb.net/")
A nice site with a browseable archive of software packages that you can't find most anywhere else.

> **bradsorensen said:**
> Can I run vmware in Ubuntu to run windows xp software I cannot get for Ubuntu Linux?

You should be able to do this without any trouble at all, and if you run into trouble there are several guides in the howto section of the forums that should be able to clear anything up you can't figure out.  Though you need to know that direct rendering programs (3d games/software) are not supported in the free versions of vmware, I can't say how they run in the newer pro versions of vmware.
