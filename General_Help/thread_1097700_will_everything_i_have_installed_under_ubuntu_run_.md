### Post by snowpine on 2009-03-16
Yes, Ubuntu and Xubuntu are the same core system, so the same applications run on both. You can test Xubuntu easily by entering the following terminal command:

```
sudo aptitude install xubuntu-desktop
```

Then, log out, and at the login screen, click Sessions and choose Xfce. If you want to go back to the original Ubuntu desktop, click Sessions and choose Gnome.
