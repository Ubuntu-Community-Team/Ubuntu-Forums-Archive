### Post by jazzmuzik on 2006-04-25
I haven't used streamtuner so I can't speak specifically to that, but generally speaking there are "restricted" audio formats which you can enable that aren't installed by default. Once you do that you can stream mp3 from the web and save it to a file. For example, the libmpg123.so plugin for xmms has a "save stream to disk" option.

[https://wiki.ubuntu.com/RestrictedFormats](https://wiki.ubuntu.com/RestrictedFormats)
