### Post by Jesse Tustin on 2011-04-11
Tired of installing ndiswrapper and everything to set up a wireless card? Try this!(might only work with Broadcom)
Go to System > Administration > Additional Drivers. If you know the name of the brand of it (mine was Broadcom) it should come up. Install the driver. Tada! You've installed the driver simply!
