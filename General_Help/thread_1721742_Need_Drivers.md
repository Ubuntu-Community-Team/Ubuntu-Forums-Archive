### Post by 3Miro on 2011-04-05
In Ubuntu you don't search the internet for drivers. You go to System -> Admin -> Additional Drivers and install from there.

Internet comes only if you don't find what you need in the repository (Additional Drivers). However, the sad news is that if it is not in the repos, then it has a high change of not working at all.
