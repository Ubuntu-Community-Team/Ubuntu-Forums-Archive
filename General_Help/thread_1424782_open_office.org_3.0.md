### Post by claracc on 2010-03-08
You have to use openoffice math. In openoffice writer you go yo insert, object, formula, there it appears a splitted window, you enter special symbols in the bottom window (this is the window for math, tick in see, selection in order to select symbols and special math characters, also you have to study the tutorial for using openoffice math). The subindex in math is made with _ character, superindex is ^.To write greek symbols, you put %fi or %DELTA.

Writting sub_index or super_indexes in open office writer, you have to use some specific keys. For sub_index, key is ctrl+alt+B, for super_index you can define keys in tools, custom, keyboard or see there if there is any key defined for superindex.
