### Post by tgalati4 on 2009-12-01
Find someone with a Tiger DVD and rebuild your system.  With tweaking and RAM (1.25 GB or better) Tiger 10.4.x runs well and will give you the best user experience.  If you need to run Windows programs, buy a used IBM thinkpad ($200-$300US).  You can dual-boot that with Ubuntu and everything (I mean everything) will work out of the box--minimal tweaking.

Trying to get windows to run on PPC under Linux is an edge case--I would like to know who has done it?
