### Post by DaimyoKirby on 2011-09-27
Okay, scratch that. I figured out how to fix it:
1. Right click on *Web Browser*, and select *Properties*.
2. Click *Add a new empty item* from the window sidebar.
3. In **Name:**, type "google", and if you have Google Chrome installed, "Create Launcher **Google Chrome**" should show up in the drop-down that appears; when clicked, everything necessary should auto-fill.
4. Click *Create*.
5. Delete the default *Web Browser* icon; if you want to keep it, just move Chrome above it, and the Web Browser button should be preserved in a hidden menu below your new Chrome shortcut on the sidebar.
