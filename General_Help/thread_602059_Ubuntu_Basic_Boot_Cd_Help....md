### Post by froy02 on 2007-11-04
You have to allow from the bios that the first boot device CD drive.  When your desktop starts it tells you to hit a key.  Some say F2, some Delete and your laptop is probably says F2 for setup and F12 for options. 

When you get to the bios setup find the boot sequence and make the cd-rom as first boot device.  It would not change anything in your set up except it would first look for a bootable  cd and if there is none it would boot to the other dvices, usually your hard drive.
