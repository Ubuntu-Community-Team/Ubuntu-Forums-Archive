Hey there,
I had a slow Firefox (*v3.5.3 installed via [UbuntuZilla]("http://ubuntuzilla.sourceforge.net")*) while switching tabs on a Jaunty AMD 64x, plus nvidia graphics driver (v180) and Compiz enabled. I've googled across the Ubuntu & nvidia Forums & I saw many people having the same problem while crying like hell, like me and struggling with Compiz, nvidia driver versions, xorg.conf, etc.

Before you get through all those threads, please see if you have any problem with a brand new Firefox profile (*firefox -profilemanager*). If no, you might want to read this on.

I had no problem with a clean firefox profile, so I started to examine my main problematic profile for installed addons & I've got those Bad Plugins!
Let there be a list of Bad Plugins you may find, here's mine:
> 
- [Mozilla JetPack]("https://jetpack.mozillalabs.com/")
- [Vacuum Places Improved]("https://addons.mozilla.org/en-US/firefox/addon/13878")
- [YSlow]("http://developer.yahoo.com/yslow")
