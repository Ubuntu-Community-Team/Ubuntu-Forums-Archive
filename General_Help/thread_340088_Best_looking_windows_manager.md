It also looks cool in a way though to be able to work without having any panels or things on the screen.  Can decent functionality and speed really be implemented through such a menu based system?

I suppose though that Fluxbox can be used in conjunction with Gnome or totally by itself... what is the easiest way to install it on a Ubuntu system?

Fluxbox has a taskbar (running applications) and a systray (notification area). It also has a desktop switching button.

Here's how it is laid out:

desktop switcher ------ taskbar ------ systray ----- clock

The nice thing about the Fluxbox menu, is that you can design the layout yourself, putting in subdirectories where you want them and putting apps where you want them.. much more flexibility than the gnome menu.
