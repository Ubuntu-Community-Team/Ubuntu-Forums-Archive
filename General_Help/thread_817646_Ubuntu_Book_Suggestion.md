### Post by spiderbatdad on 2008-06-04
You can DL the official ubuntu handbook: [http://vntutor.blogspot.com/2007/07/official-ubuntu-book-download-free.html](http://vntutor.blogspot.com/2007/07/official-ubuntu-book-download-free.html)

It is .chm and needs to be converted with libchm-bin
```
extract_Libchm <inputfile> <outputfolder>
```Note the capital "L."

It is easiest to start by creating a folder for the downloaded zip, extract the package in that folder, cd into the folder, and run the above command.
The process creates the output folder, containing all the html pages.
