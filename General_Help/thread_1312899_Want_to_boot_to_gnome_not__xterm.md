### Post by iski4kix on 2009-11-03
I figured it with the help of this post
[http://ubuntuforums.org/showthread.php?t=852852](http://ubuntuforums.org/showthread.php?t=852852)
So in my xterm I typed gnome-session which brought me to my gnome desktop then I changed my login option from automatic to ask me what user to login as then when I rebooted I was able to reselect GNOME at the bottom of the login screen
