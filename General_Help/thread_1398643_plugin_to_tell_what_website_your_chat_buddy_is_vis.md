### Post by dfreer on 2010-02-05
> **MindFusion said:**
> Finding out what websites they're browsing would theoretically require you to use hacks, such as keylogger with an instant report, but that is illegal, as not only the use of such remote hacking tools is illegal, but invading the personal privacy of people is as well.

Not particularly; depends on how the OP wishes to obtain this information. Let's say User A **wants** to allow User B to know which sites he's browsing, for whatever reason. You could install a plugin to User A's Firefox that announces the top level domain of the currently active tab to D-BUS, similar to the way Amarok can announce what song is currently playing.

User A's Pidgin could then have a plugin that would update the clients IM status/away message to show what website he is browsing. User B would need no hacking tools or anything, just a compatible IM client.

Notice that this entirely depends on User A *wanting* to share this information with User B, which is what I hope the OP's situation is. Some user's like to announce what song they are listening to, and although I wouldn't want to announce my browsing history myself others (including OP) may want to.
