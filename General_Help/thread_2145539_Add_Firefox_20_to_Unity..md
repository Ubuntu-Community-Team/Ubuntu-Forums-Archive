### Post by Frogs Hair on 2013-05-15
The current version of Firefox in 13.04 is 20 and updates come via the update manager. If the FF icon is not in the launcher on the left start typing the Firefox into dash which opens when the cursor is moved to the top left corner. Select the icon when it appears to open the application. When the icon is in the launcher right click and select lock to launcher. The software updater will offer the option to install updates.

Ubuntu Manual: [http://ubuntu-manual.org/](http://ubuntu-manual.org/)
