### Post by llama320 on 2008-06-30
```
 compiz --version 
```

worked for me. it'll output a bunch of stuff (kind of like when you do compiz --replace) but the last line should be the version. For reference, I'm using the one from hardy repositories and i've got 0.7.4

good luck
