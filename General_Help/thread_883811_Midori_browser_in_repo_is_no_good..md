### Post by wkulecz on 2008-08-08
In case you've read Marcel Gagne's "Cooking With Linux" column in Sept 2008 issue of Linux Journal and want to try the Midori web browser which he reports as the only one to pass [http://acid3.acidtests.org](http://acid3.acidtests.org) don't waste your time with the version 0.0.17 in the repo as it fails badly.  The default Firefox 3 does much better.

Links2 is great if you want to go lean and mean for text.

--wally.

Edit:  the current Midori version is 0.0.19 if you want to build it yourself (I don't have time) the article didn't mention which version he ran, but it couldn't have been 0.0.17
