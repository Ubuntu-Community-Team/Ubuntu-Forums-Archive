### Post by MaxIBoy on 2009-03-11
Compiz runs instead of GNOME's default window manager (assuming you're using GNOME, of course.) GNOME's default WM is Metacity. 

If you switch from Compiz to Metacity or Metacity to Compiz (i.e. turn the desktop effects on or off,) all windows will be moved to Workspace 1. It's annoying, but I think right now it's unavoidable.

If that's not what you did, something else is going on.
