### Post by Impavidus on 2014-09-01
Install the package **flashplugin-installer** using the software centre, synaptic or the terminal. You can also install **ubuntu-restricted-extras**, which will include flashplugin-installer together with some other non-free software, if you want it. Finally, there is google chrome, which includes a flash player by default.
