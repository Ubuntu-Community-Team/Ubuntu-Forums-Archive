### Post by pytheas22 on 2009-01-28
It's possible that the CD you burned is bad.  Please try burning another one, at low speed.  You can also check the CD for defects; there's an option for this at the first menu that you see when you boot to the Ubuntu CD.

Another option would be to install using [wubi]("http://wubi-installer.org/"), provided you have Windows on this computer.  wubi doesn't require a CD.

If neither a second CD nor wubi work for you, please tell us the exact make and model of your computer, as well as how much memory you have, so we can see if there are any known issues running Linux on it.

### Post by thatsnotthefloor on 2009-01-28
If you already know you want to install Ubuntu and have no need to try it first, I suggest giving the alternate installer a try. It's a text-based installer, just as a heads up, so you'll be installing in a terminal setting. It's pretty straightforward though.

[http://www.ubuntu.com/getubuntu/downloadmirrors#alternate](http://www.ubuntu.com/getubuntu/downloadmirrors#alternate)

Also, I'm not sure what exactly is going wrong with the livecd install, but I'd also check your hardware's compatibility, just to be on the safe side. You can find a list of compatible hardware here:

[https://wiki.ubuntu.com/HardwareSupport/](https://wiki.ubuntu.com/HardwareSupport/)

I hope that helps and good luck beginning your journey with Ubuntu!
