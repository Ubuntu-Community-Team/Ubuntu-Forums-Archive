### Post by Who on 2006-01-25
In short, no.
If you devoted your life to it, you may be able to get Visual Studio to run under wine ([www.winhq.org)](www.winhq.org)).

Linux has some of it's own development environments. If you kn ow c# then you'll probably like monodevelop ([http://www.monodevelop.com/Main_Page](http://www.monodevelop.com/Main_Page))
In my experience, VB is a bit of a no-go area, though the mono project ([http://www.mono-project.com/Main_Page](http://www.mono-project.com/Main_Page)) are working on a VB compiler too...

For less MS orientated languages the web is your playground! Some titles to start with would be
Eclipse - general purpose IDE
emacs or vim (the two are apparently hugely different, and have huge fanbases, but are not like any IDE most people have used before - kind of glorified text editors...but better)
Glade
Kdevelop

The list continues!

Basically, if you need Visual Studio, you really need Windows AS FAR AS I KNOW- sorry

Who
