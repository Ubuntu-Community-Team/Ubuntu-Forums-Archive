### Post by merlinus on 2007-08-05
IIRC, you cannot use chown (or chmod) on non-linux partitions.  You can set permissions via the entry in /etc/fstab.

Also, it should be, for example:

```

sudo chown -R username:usergroup /media/sda7

```-R sets the ownership or permissions recursively.

-merlin

### Post by merlinus on 2007-08-05
THe name of the group that you want to have the permissions.  username has to be a member of that group.

In this example, user mctek and all members of the group mctek would have ownership, and read and write permissions.

```

sudo chown -R mctek:mctek /media/sda7

```

System/Administration/Users and Groups is where the users and members of groups are managed.

-merlin
