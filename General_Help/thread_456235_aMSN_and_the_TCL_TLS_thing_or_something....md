### Post by MoLE on 2007-06-02
Hi Lufkin, welcome to Ubuntu.

What are you trying to achieve by using the tls.1.5.0 package?  Your subject title alludes to aMSN - so I assume you're trying to install the aMSN client for use with Windows Messenger service.  Did you know that Pidgin (previously known as GAIM) which is the instant messenger bundled with Ubuntu - also works with MSN?

If you still want to install amsn - it is in the universe repository.  You will need to go to System --> Administation --> Software Sources, and make sure you have the universe repository checked.

Then you can go to Add/Remove programs and type in amsn, check the box and install.  You shouldn't need to muck around with installing any libraries, as the package management system should do all that for you.
