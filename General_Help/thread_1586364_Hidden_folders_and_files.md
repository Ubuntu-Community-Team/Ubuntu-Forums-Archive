### Post by cj.surrusco on 2010-10-01
Adding a period to the beginning of the filename will make it hidden. If the filename cannot be changed, you would need to follow this procedure:

1. Create a text file called ".hidden" in the folder that contains the file you want to hide.

2. Add the names of the files you want to hide into the .hidden file.
