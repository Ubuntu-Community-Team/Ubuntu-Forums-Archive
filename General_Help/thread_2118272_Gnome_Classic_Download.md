Perhaps you would like synaptic. It is "graphical management of software packages". However, I believe the application you are looking for is gnome-session-fallback. There should be gnome gnome2d unity unity2d in 12.04 I believe after you install gnome-session-fallback

open a Terminal window and copy in it the following command:```
sudo apt-get install -y gnome-panel
```then log out; when the login prompt appears, choose "Gnome Classic" from the menu.

### Post by hawthornso23 on 2013-02-20
Install gnome-session-fallback. It gives an environment with the look and feel of gnome 2 but it uses GTK3 under the hood so it is fully supported and works nicely in 12.04 which is GTK3 based.
