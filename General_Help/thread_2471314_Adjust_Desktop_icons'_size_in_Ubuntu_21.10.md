### Post by deadflowr on 2022-01-26
Extensions are now controlled by a separate app called Extensions which should be installed.
Just search for it in the applications  menu.

In Extensions it's the Desktop Icons, click on the cog icon to open it's settings.

From there I think you can control size by setting them to small, medium, or large. (May have other names like tiny or something)
I'm not sure if you can set a specific pixel-like size.
