### Post by themusicalduck on 2010-09-18
Easiest way to do this -

Open the Youtube page and wait for it to download fully (so the seek bar is red). Then open the file browser and go to File System, then tmp.

Look for the video file in there, it'll start with FlashXX and then have random letters and numbers after it. Copy the file to your desktop and rename it to whatever you want.

### Post by Chris1274 on 2010-09-18
Youtube downloader is available through synaptic. You can then download vids from the command line:
```
youtube-dl <video url>
```
