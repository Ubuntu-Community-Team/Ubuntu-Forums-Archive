### Post by drmrgd on 2012-07-11
This might be a stretch and not what you're looking for, but try right clicking on the window name in the Task Panel, and putting a check in the "Maximize" box.  

In the, I've had some issues with various apps opening oddly, and I found that if I mess with that menu, I can usually fix it so that the application opens the way I want it too.  Like I said, though, this might not fix your issue.
