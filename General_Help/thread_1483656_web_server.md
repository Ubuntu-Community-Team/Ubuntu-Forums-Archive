### Post by darolu on 2010-05-14
In Ubuntu is really easy, you can install it via Software centre, Synaptic or terminal with apt-get or aptitute.

I suggest you to install the whole LAMP server though, it's always good to have mysql and php ready if you need them, open Syanptic, click on Edit and then click on "mark by task" and select the LAMP option, apply changes and that's it.

Compiling from source as jerenept said is a very good idea to get maximum performance from your server and to use the latest version of it, but if you are asking here about how to get apache installed on 'linex' make me say I do **NOT** recommend this method to you, at least not yet. Installing from Synaptic works great, is easier and faster (which is the best part) so you can get to work right away.
