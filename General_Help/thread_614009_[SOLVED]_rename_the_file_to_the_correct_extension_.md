### Post by Hopsakee on 2007-11-18
I tried the 'save as' option. It didn't work. I did found the solution though.

erase the .ods extension all together

so:  "adminstration.ods" becomes "adminstration"

When I used Windows there was some option 'hide extensions'. Maybe I've also turned this option on in Ubuntu. Which means my files are actually named adminstration.ods.ods. I don't know, but removing the 'ods' works. So in a pragmatic way I'm satisfied.
