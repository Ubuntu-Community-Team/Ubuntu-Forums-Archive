### Post by mardawi on 2008-04-28
> **elonlon said:**
> OK whit number 2 i don't need help i successes
but why i don't have 4 desktops
and on the effects i can't change between the desktops in like square

To have 4 desktops (workspaces) right click the workspaces switcher (should be in the right-bottom corner) and choose preferences. then make columns 4 instead of 2.

for the cube, you need to have "Advanced Desktop Effects Settings" utility in your System>Preferences, if it's not there, you can install it using:

```
sudo apt-get install compizconfig-settings-manager
```

Then open  System>Preferences>Advanced Desktop Effects Settings and enable desktop cube effect.
