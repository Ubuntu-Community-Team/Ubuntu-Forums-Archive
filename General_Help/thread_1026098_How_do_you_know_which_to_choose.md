### Post by oldos2er on 2008-12-30
There's no need to create a LiveCD for each of them. There're some simple commands you can run to try different DEs.

 For Ubuntu, "sudo apt-get install ubuntu-desktop"

 For Xubuntu, "sudo apt-get install xubuntu-desktop"

 For Kubuntu, "sudo apt-get install kubuntu-desktop"

 To change to a different DE, logout, under the 'Sessions' menu choose which one to run, then log back in.
