### Post by kostkon on 2008-02-18
The next version of *Ubuntu*, *Hardy Heron (8.04)*, will use *PulseAudio* as its sound server and you will be able to control the volume level of every application independently (including *Flash*).

For the time being the only option is to install the *Flashblock* extension in *Firefox* and block any *Flash* adds, videos etc. you don't like.
