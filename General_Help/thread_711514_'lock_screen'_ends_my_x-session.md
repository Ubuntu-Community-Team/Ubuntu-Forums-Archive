### Post by PewterHydra on 2008-02-29
I recently upgraded (fresh install) to Gutsy. One thing that really irks me, but I can't figure out the reason for it's occurrence is that when I attempt to lock the screen it ends my x-session and logs me out instead. My personal computer, running the same version, doesn't have this problem so this isn't some obvious you're-clicking-the-wrong-button type thing. 

(This is annoying as heck because every time I leave my desk at work I have to lock it, which means restarting everything I'd been working on once I get back.)

Has anyone heard of this bug and, perhaps more importantly, how to fix it?

---

### Post by PewterHydra on 2008-03-04
If anyone else starts having this problem, here is my solution, though the explanation escapes me. 

It looks as though some hardware (likely my dual-head setup) just doesn't get along well with the screensavers, so if I disable the screensaver (uncheck:  preferences > screensaver> activate screensaver when computer is idle) life goes back to normal and I can lock the screen without x crashing. 

Yay!
