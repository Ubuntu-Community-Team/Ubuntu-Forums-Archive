### Post by simpsonatwork on 2008-09-22
The name of the plugin which does that is Scale. Just enable it in CCSM. If you don't have CCSM install it via Synaptic for instance, "compiz-config-settings-manager". Then run it: System > Preferences > Advanced Desktop Effects Settings.
