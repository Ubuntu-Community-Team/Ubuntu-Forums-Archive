### Post by Ataris on 2009-02-09
Extract whatever tar file you are using until you get to the main directory of the theme file (meaning when you open the folder that's where you start seeing files).

Navigate to Home, use Ctrl + H, find the .themes folder, and cut and paste the theme folder into that directory.

This is the best method to use when installing themes.
