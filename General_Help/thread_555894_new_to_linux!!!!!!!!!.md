Welcome to the forums. Like the other people said, use the livecd first and see how you like and see if it runs ok on your computer.

Regarding the dialup internet, I haven't used dialup in a while so I can't offer any advice. Someone here should be able to help you out with that if needed (also, the forum search tool is really good - you'll likely find some threads that will help).

Regarding "running normal programs on ubuntu" - it sounds like you mean can ubuntu run the same programs that you run on windows? The answer to that is yes and no and maybe. Yes, I'm funny. What I mean is that some programs like Firefox web browser or Thunderbird email client have both windows and linux versions (there's many others too). So for those programs the answer is yes you can run the same programs. For other software you might use on windows there is not a version of it for linux - like Quicken, Photoshop, (name a popular commercial app). And the windows version of these apps won't run directly on linux (more on this in a bit). However there is almost always a linux program that will be basically the same, and sometimes even better than the windows equivalent. So for example, take MS Office. An OK program, but costs big bucks. Linux has openoffice. It offers pretty much the same functionality as MS office and it's free. Lastly, there IS a way to run windows apps on linux. There is a free application named wine that can install and run lots of windows programs. You can check their site at [http://www.winehq.org/](http://www.winehq.org/)  to see what apps are running. You can also use and application like virtualbox or vmware server (both free) that you can use to set up what's called a virtual pc on your computer and install windows on the virtual pc. Then you can start the virtualpc and install and run any windows apps. This option may not be for you because you said your windows is expiring soon. But if you can get your windows activation problem sorted out, then the virtual pc method is good way to go if there are certain windows-only apps you want to use.

Good luck and have fun! And don't be shy about asking for help - the people on the forums here are super helpful and friendly.
