 Vik, 9.04 is no longer supported and the repos have been taken down. If you like the older Gnome 2 that you are accustomed to, 10.04 has another year and 4 months of support remaining. The next LTS version, 12.04, will have 5 years of support, but it will also carry Gnome 3 (just like version 11.10 does).
 
The suggestion to try Xfce (Xubuntu) was a good one. If you like a desktop more similar to Windows XP, but much more efficient and faster, Lubuntu is also a great choice. Either way you keep the Ubuntu system running, but you have a different desktop environment. You can also add those interfaces to your current Ubuntu install via synaptic (Xubuntu-desktop I believe) and log out and log back into whichever DE you wish.
