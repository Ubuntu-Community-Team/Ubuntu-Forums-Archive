### Post by handy on 2006-03-05
I have only used Ubuntu with broadband, & have found that if I don't atleast turn off **ipv6** in **Firefox**, I can not connect to the internet!!

To do this: 

1.  Enter **about:config** in the Firefox address field.

2.  Enter **ipv6** in the filter field.

3.  Double click on the only visible line to change it's boolean value to **true**.

Exit out of that tab in Firefox & try to connect to a web site now?

It works for me on the previous & the current version of Firefox.
