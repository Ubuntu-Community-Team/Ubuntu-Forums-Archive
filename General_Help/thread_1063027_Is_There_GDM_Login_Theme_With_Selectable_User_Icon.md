### Post by jonlemur on 2009-02-07
There are several.  If you go to [gnome-look.org]("http://www.gnome-look.org/index.php?xsortmode=down&page=0&xcontentmode=150"), you can find a bunch of downloadable gdm themes, and you can probably tell from the thumbnail if they have icons.  You'll probably have to set the icon in System->Preferences->About Me, and there's an icon in the corner that you can set.

### Post by mb_webguy on 2009-02-07
There are a couple already installed.  Go to System->Administration->Login Window, go to the Local tab, and select one of the ones that mention a face browser in the description (i.e. either "Happy Gnome with Browser" or "Human List").  You can change the image used, but I can't remember how just at the moment.
