### Post by DJ_Peng on 2008-10-26
I suspect it's because the image code is referencing an image on your (or his) computer, and even getting it into Thunderbird the image is no longer in the same place relative to your work. If you have a place where you can store it on the web and show it on a web page I'd try putting it there and then edit the HTML file to get the image from that very specific location. Just keep in mind that anyone who gets the email through a program or service that doesn't support HTML email messages won't see the picture in the signature.
