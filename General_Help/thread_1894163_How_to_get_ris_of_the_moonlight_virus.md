### Post by directhex on 2011-12-12
You think you're being clever, but you're not.

If you installed Moonlight in Chrome from an extension, remove it from [here](chrome://plugins/)

If you installed Moonlight in Firefox from an extension, remove it from Tools/Addons/Extensions

If you installed Moonlight as a package, run "apt-get remove moonlight-plugin-core

And I'd recommend you get your hyperbole filter serviced as soon as possible, I think yours is malfunctioning.
