 I don't usually use cd cover apps, but one easy way to find out now and in the future is (I assume you're using Firefox):

Go to [http://mycroft.mozdev.org/download.html](http://mycroft.mozdev.org/download.html) and search for "freshmeat" in the search box called "Find Search Plugins".

In the result page, click the "Freshmeat project" link, and this will install a search plugin for Freshmeat, one of the biggest collector of organizer of software in linuxland. Now, use that search engine and search for the terms you need, like "cd cover" .

You'll get results from freshmeat , and when you find one you like, see if it's in synaptic.
If not, see the instalation instruction's in the software's homepage.
