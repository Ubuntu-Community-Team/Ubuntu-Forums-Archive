### Post by Pollox on 2010-02-27
I would think you could just use the built-in file browser, Nautilus, to do this.  If you go to the Places menu and click "Connect to Server" (or in Nautilus the File menu and "Connect to Server"), you can connect to a remote computer via various methods.  I've only used the ssh option to access files on a remote computer, but there's a couple ftp options as well.  Of course, that's assuming the other computer is already correctly set up to share a folder.
