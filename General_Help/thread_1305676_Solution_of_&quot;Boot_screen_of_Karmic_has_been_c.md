### Post by Tapas Bose, India on 2009-10-30
Hallo everybody.
I had written in the post of the link [http://ubuntuforums.org/showthread.php?t=1299876](http://ubuntuforums.org/showthread.php?t=1299876) that my Karmic boot screen had been changed due to installation and un-installation of mythbuntu-gdm-theme and I could not get my Karmic boot screen back. Unfortunately nobody gave reply. Although I myself can able to solve the problem.
As the post is closed so I am writing a new post to help any other friend or Ubuntu user. During installation of mythbuntu-gdm-theme another software package was installed, which is xsplash. Which also un-installed during un-installation of mythbuntu-gdm-theme. To solve the problem I installed only xsplash. It installed with another software package naming ubuntu-xsplash-artwork. Then the problem was solved and I got my Karmic boot theme back.
Have a nice day...
