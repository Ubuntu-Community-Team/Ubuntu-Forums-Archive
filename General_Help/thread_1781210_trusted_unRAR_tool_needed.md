### Post by Mark Phelps on 2011-06-13
In my own experience, when unRAR fails toward the end of a single file archive, that typically means the either the checksum is bad or the password you entered for the archive file is incorrect.

Yeah, I know it SHOULD tell you up front if the password is incorrect, but sometimes, it goes all the way to the end of trying to expand the archive -- and then fails.

If these are movies you're getting from the Internet, and you're supplying a password given by the poster, this generally means the password provided is incorrect.
