### Post by efflandt on 2010-10-19
Does it do the same in Windows?  Does your printer have a web config menu to set various printer defaults for saturation of each color, etc.?  Otherwise if you right click on your printer in System > Administration > Printing and go to Properties, is there Printer Options that could make such adjustments specifically for Linux?

When I first got my Lexmark C543dn its colors were too saturated for my taste.  Maybe fine for bold brochures, but not photo realistic.  So I downloaded some images that were combination images with a bunch of things in them for adjusting colors, etc.  And I played with adjusting settings on the printer web config itself until the printed images appeared most balanced and natural, close to a photo.  It took some experimenting because changing one setting sometimes required adjusting another setting.  But I must have done a pretty good job because on the Ubuntu test page with Linux defaults I can see discrete differences between all the 10% steps, except maybe between 90% and 100% yellow.
