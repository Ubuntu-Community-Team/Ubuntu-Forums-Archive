### Post by kyphi on 2008-04-13
As far as I know there is no equivalent in Linux.

Sending spam back to spammers for a great number of reasons is not a good idea.  You would just be contributing to spam traffic yourself.  Fortunately there are many ISPs that have blocked that portal as well as instituting spam blocks at their own level.

If you must have this programme, you could try to run it via Wine or CrossOver Linux.

Another solution would be to get Mailwasher Pro from Firetrust which will do a lot more for email security than any other programme I know.  Mailwasher Pro will run on Wine and CrossOver.
