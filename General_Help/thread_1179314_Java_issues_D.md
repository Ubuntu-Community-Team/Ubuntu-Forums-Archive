### Post by kpkeerthi on 2009-06-05
> I've installed Java using the Add/Remove functions, selecting Sun Java Runtime 6.

You need **sun-java6-plugin**. Java 6 Runtime environment will not get you browser plugin. Search and install **sun-java6-plugin** from Applications -> Add/Remove and restart your browser.
