### Post by simonodell on 2010-04-22
Hi all,

I was looking for a way to emulate the windows CTRL-ALT-DEL functionality and I have noticed there are others also looking for this. 

I came up with a simple solution which I would like to suggest might be cool for future versions;

1. Goto : System > Preferences > Keyboard Shortcuts

2. Click "Add"

3. Give it a name, i.e. "ctrl-alt-del"

4. Enter the command "gnome-system-monitor"

5. Ctrl-Alt-Del is already assigned to logout, say yes when it asks if you want to reassign it.

Bingo! now when an app is playing up, you can hit ctrl-alt-del and use gnome-system-monitor just as you would use taskman in windows to kill the malfunctioning process.

Hope this helps others, all the best!
