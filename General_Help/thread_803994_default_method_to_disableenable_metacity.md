What is the normal way or default method to disable/enable metacity themes(GNOME's default window manager)???


I usually use the 2 commands below but I was wondering if there was a simpler method??
 
To enable Metacity:

```
Alt+F2, then type
metacity --replace
```

To enable Emerald:

```
Alt+F2
emerald --replace
```

### Post by ryanhaigh on 2008-05-22
You could create launchers to do those commands. Just right click on the desktop>create new launcher. Call them whatever you want and use the commands you posted as the commands for the launchers. Once finished you can probably drag the launcher to the panel if you wanted to or open the menu editor and put them in there.
