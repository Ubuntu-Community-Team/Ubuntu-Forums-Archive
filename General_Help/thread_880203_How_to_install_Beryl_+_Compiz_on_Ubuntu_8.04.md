### Post by mcduck on 2008-08-05
Compiz is already installed by default. You can enable it in System/Preferences/Desktop Effects.

You may want to install Emerald & compiz configuration manager to use compiz/emerald themes and access all compiz settings.

Beryl doesn't even exist any more, it was merged to Compiz.
