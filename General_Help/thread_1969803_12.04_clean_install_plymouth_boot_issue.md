### Post by Derek Karpinski on 2012-05-23
This should force plymouth to load earlier (this might add a few seconds to your boot time)

```
echo FRAMEBUFFER=y | sudo tee /etc/initramfs-tools/conf.d/splash
```Then:

```
sudo update-initramfs -u
```And for the hell of it (probably don't need it):

```
sudo update-grub2
```If your resolution is bad, try this:

[http://ubuntuforums.org/showpost.php?p=11930648&postcount=2](http://ubuntuforums.org/showpost.php?p=11930648&postcount=2)
