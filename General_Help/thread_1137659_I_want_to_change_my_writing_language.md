### Post by fballem on 2009-04-26
> **hamamshady said:**
> I already done that you in windows I could click ALT + SHIFT and it changes I only want to know the magic keys in ubuntu

I write in English, French, and Portuguese. I have two keyboards defined - one with Accents (US International with Dead Keys) and one without (US). I'm assuming that you have something similar (English and Arabic).

To make the switch, I've added a keyboard indicator to my top panel. When I need to switch, I can then click on the keyboard indicator and select the keyboard that I want to use. If this is what you're after, then here is how to do it:

1) Right-click in the top panel, then select Add to Panel from the menu.
2) You will get a long list. (see screenshot). Scroll down to the Keyboard Indicator.
3) Click on the Keyboard Indicator and you will see that the Add Button is enabled. Click on the Add button and an icon for the keyboard indicator will be placed on the panel.
4) If you don't like where the indicator is on the panel, then right-click on the indicator, select Move, drag it to where you want on the panel, and click again.
5) To lock the location, so it cannot be moved, right-click on the indicator and select Lock To Panel from the options.

To use, click on the keyboard indicator and it will switch to the next keyboard that you have defined. To make any changes (such as designating one of them as a default), right-click on the indicator and select Keyboard Preferences.

Hope this helps,
