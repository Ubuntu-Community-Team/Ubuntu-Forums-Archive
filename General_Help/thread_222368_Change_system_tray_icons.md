### Post by wieman01 on 2006-07-24
I am using KDE, and you are - in all likelihood - a Gnome user. So I cannot help you with the fonts.

However, I have just been through a system tray exercise so I can share my findings with you.

Programs (Gaim, etc.)use their own tray-icons which are either found in the "/usr/share/icons/<your icon theme>" directory or in the program's own directory (e.g. Gaim's) under "icons". 

You can edit these icons using Gimp or similar tools and change the background according to your needs. You have to ensure that you don't change the NAME of the icons but you can edit them (and even replace them) as you like.

Hope this helps a bit.
