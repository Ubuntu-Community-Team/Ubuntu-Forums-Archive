Hi Luke

Not to sure about your first question. I'll let someone else answer that. 

For cloning Windows (XP or Vista), use NTFSClone. NTFSClone is included with Clonezilla (this option is set as default) and G4L (Ghost for Linux). Out of these two, I prefer G4L although Clonezilla does backup faster but G4L restores quicker (from my tests anyway :) ). 

If you want to backup Ubuntu (or other distro), you can use PartImage which is also included in Clonezilla and G4L (also PING). Hope this helps.
