### Post by doulos564 on 2011-10-18
I've found some alternative products...

These two I'm looking at:
Tellico (open source via standard methods)
Readerware from ([http://www.readerware.com/index.php](http://www.readerware.com/index.php)  -  pay product)

Both of these can search internet sources (e.g. Amazon).
Both will search a list of things (e.g. ISBN)
I created a list like this:
   terminal
   cd .alexandria/(book collection)
   ls *yaml | cut -d "." -f 1 >$HOME/Desktop/book.txt

Then used the file Desktop/book.txt to populate the list in the product.
Tellico would only take 100 entries at a time.  It also wouldn't find some books, but on subsequent searches for them would.

Readerware seems more mature and thoughtful.

Others that I haven't investigated yet:
[http://www.collectorz.com/book/](http://www.collectorz.com/book/)  (pay)
[http://www.librarything.com/](http://www.librarything.com/)     (free online social media site)

Hope this helps.  If you look at any, please post your opinions back as it may save us all some time.

### Post by dumbmatter on 2011-10-22
A word of warning to those of you importing from Alexandria: multiline comments are not correctly imported, so you have to set those manually. You can find all your books with multiline comments by running

    grep \|- *.yaml

in your ~/.alexandria directory.
