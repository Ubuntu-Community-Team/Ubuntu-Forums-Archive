### Post by MaxIBoy on 2009-05-09
Typically, developers don't like to release precompiled versions of alpha software. It changes too fast to be worth the trouble, and if you can't compile from source, you probably aren't going to be able to troubleshoot the inevitable bugs that show up (and the reason for alpha releases is so people can help the developers to find bugs.)


Anyway, if you understand the risks, there will often be an ubuntu PPA for the software.
