### Post by y-lee on 2008-04-22
If you installed Google earth thru synaptic or add and remove programs you should wait until the update hits the repos. It takes a little while for it to be repackaged and added to Ubuntus repos.

AS to bin files if you downloaded them to your desktop or home folders you usually  have to change the permissions to execute. If you can see the file on your desktop or in your file manager then right click it and choose properties, click on the permissions tab and be sure execute is checked for at least owner. I usually click owner group and others. After doing so click close.

Now you should be able to double click the bin file to start it. :)

Edit: changing permissions can be done thru the command line in the terminal but I figure you want the easy way ;)
