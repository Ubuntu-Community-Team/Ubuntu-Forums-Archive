### Post by enoelrocotiv on 2011-03-04
I had the same issue and log messages. The problem was simple to fix but it was hard to diagnose: **a configuration file was broken**. In my case .profile was broken by a pip completion script. I saw it on .xsession-errors:


```

$ cat .xsession-error
[...]
/home/USER/.profile: 27: Syntax error: "(" unexpected (expecting "}")
[...]

```

I fixed .profile and I could log in.
