### Post by jocko on 2009-02-21
> **kaelonlloyd said:**
> I've set up a dual boot pc with XP and ubuntu using wubi. At the moment i got comodo antivirus and firewall on my windows side but no protection on my linux side. I've read that an antivirus isn't needed for linux but im worried that a windows virus will get to XP when im on linux.
Can a virus find its way to xp through linux or not ? and if so how would i stop it?
Windows viruses don't run in linux, so you don't need to worry about getting infected while linux is running.
Of course, if you download an infected file in linux, and then try to open it in windows it will run... but as long as your antivirus program in windows is kept updated and scans files on access, you should be safe anyway...
