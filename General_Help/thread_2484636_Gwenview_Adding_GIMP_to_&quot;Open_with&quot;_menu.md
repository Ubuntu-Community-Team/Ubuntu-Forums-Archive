### Post by Dennis N on 2023-03-05
Linux Mint does not offer a Gnome desktop version. Did you install Ubuntu (which uses the Gnome desktop) or one of its flavors (Xubuntu for example)? In Ubuntu, the comparable application is **gthumb**. It has the same grid view of images. And if Gimp is also installed, it does appear in gthumb's "Open With..." option.
