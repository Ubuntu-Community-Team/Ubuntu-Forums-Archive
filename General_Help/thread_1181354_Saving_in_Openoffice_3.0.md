### Post by labinnsw on 2009-06-07
By default Open Office saves a file to the location where you are at when the file is created or edited. It does not however save a newly created file without giving you the option of choosing the name and location.

Try creating another file called "test", write in it "test." Close it saving it in the documents directory in another directory called "test."

If that works, I would say you inadvertently saved your other file in another location than the one you thought you specified.

Open the File Browser.
In the sidebar, click on File System
On the Tool bar click on Search
In the search field type the name of the file you created and press enter.

When you find your file, right click and select properties to see where it was saved.
