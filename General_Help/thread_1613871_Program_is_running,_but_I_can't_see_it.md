### Post by sikander3786 on 2010-11-06
Not all programs run successfully with wine.

See

[http://appdb.winehq.org](http://appdb.winehq.org)

to know if your desired program is supported and to what extent.

To see what is happening, go to Wine Configuration > Graphics and tick Emulate a Virtual Desktop. You desired program might show there and let you know what is going on(little hopes though).
