### Post by H3lp3r on 2008-11-26
When i go to install something it says An error as a occured. Then it says below that i must manually run 'dkpg --configure a' to correct the problem. What is it talking about and what do i do?

---

### Post by ibutho on 2008-11-26
You need to go to Applications -> Accessories -> Terminal and enter the commands
```
sudo dpkg --configure -a
```
