### Post by domino1241 on 2010-12-27
Hello All,

As you've probably found out if you searched for this on Google, the new FAFSA page ([https://fafsa.ed.gov](https://fafsa.ed.gov)) does not allow any Linux browser access to financial aid forms.

The solution I found is to install User Agent Switcher for Firefox ([https://addons.mozilla.org/fr/firefox/addon/59/](https://addons.mozilla.org/fr/firefox/addon/59/)) and restart the browser.  Right click a toolbar, like the navigation toolbar near the back button, and click 'Customize'.  Find the User Agent Switcher icon and drag it to the toolbar.

Now you can change the browser's identity by clicking the little globe on the toolbar and select, for example, Internet Explorer 8.

You should now be able to get beyond the first page of the new FAFSA website.

Good luck!

### Post by noah++ on 2010-12-27
It's unusual to encounter these blocks based on user agent string anymore. In the case of FAFSA, I am only getting a warning that my browser is unsupported. It doesn't block me from using the site.

Anyway, if you do have to use this workaround, choose the UA that is closest to your actual browser version. If you choose IE, for example, the site might serve up some browser-specific code that really *will *render your browser unable to navigate the site -- maybe in a subtle way you won't detect until you've laboriously entered page upon page of application data.
