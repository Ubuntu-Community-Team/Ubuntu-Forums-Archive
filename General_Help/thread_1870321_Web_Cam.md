### Post by 3rdalbum on 2011-10-27
Look for one that uses UVC. These should all work on Linux because UVC is a standard; much like how any USB flash drive works on any USB operating system, any UVC webcam will work on any UVC-compatible operating system with no additional drivers necessary.
