### Post by kurt18947 on 2011-11-18
To get the 'old style' desktop you need to install 'gnome-session-fallback'. Gnome shell gives you the Gnome 3 desktop.  'Gnome classic' looks *something* like Gnome 2 but doesn't really act like it.  There is a post some on this forum where there are directions to unload and load some packages & change some settings to get closer to gnome 2's functionality.  I can't lay my hands on it this minute.  

If you had Gnome 3 (Gnome-shell package) you could have tapped the 'super' key and you'd have gotten a launcher sort of like the Unity launcher.  I've tried tried most of the desktops and use Gnome 3 the most. You can also install Xfce-desktop or LXDE-desktop or KDE-desktop from the repos and select your choice on log-in.  It seems a fair number of Unity refugees have settled on Xubuntu or XFCE-desktop running on regular Ubuntu.
