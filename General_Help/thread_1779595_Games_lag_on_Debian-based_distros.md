### Post by mistyautom on 2011-06-11
This may be due to having more visual effects in Ubuntu as compared to other distros.  You computer will run the fastest by doing to following:

1. Right click the background and select "Change Desktop Background"
2. Select the "Visual Effects" tab.
3. Check the hole which says "None: A simple desktop enviroment without any effects."
4. Click close.

[http://www.dotdeb.com](http://www.dotdeb.com) is a great place to find deb packages.
