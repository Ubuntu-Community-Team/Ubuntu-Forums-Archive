### Post by mcduck on 2011-02-03
You already have one by default, it's set to same as your login password is and the keychain is unlocked automatically at the same time you log in.

If you want to change or unset the keyring password just go to System/Preferences/Passwords and Encryption Keys, right-click on the "Passwords:login"-keyring and select "Change Password"
