### Post by kingsidy on 2006-01-01
since u just downloaded it i assume that you want to burn it in windows. If you have a program like nero go into the recorder menu and select burn image, then go into the folder that contains the kubuntu image and select it. if you do not have nero, whichever software you are using should have a "burn image" option. if you do not have any burning software at all, i would recommend using dvddecrypter and use it to burn the image onto a disc, or google for an image or iso burner software.
