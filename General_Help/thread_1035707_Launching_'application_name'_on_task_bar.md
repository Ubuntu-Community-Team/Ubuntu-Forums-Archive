### Post by blackened on 2009-01-11
If you use Compiz, then you might try adding it to the "Skip Taskbar"' section under "Window Rules". You could match the window by title "Starting **wildcard character**", but this could have some unforseen consequences if you happened on a webpage that began with something similar. But hey, where's the fun in learning if you don't get to break things in the process right? :D
