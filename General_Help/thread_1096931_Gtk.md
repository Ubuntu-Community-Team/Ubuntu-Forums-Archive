### Post by Yashiro on 2009-03-15
GTK is a set of functions and commands that programmers can use in their applications to create a user interface. (windows, panels, buttons, sliders, icons)

It was originally created during development of 'The Gimp' paint program. It is now used as the base for the Gnome desktop and most of it's GUI based applications.
