### Post by jmszr on 2009-05-31
balagunner,

Try right clicking on Applications > click on Edit Menus > when the Menu comes up click on Sound & Video under Menus: > then under Items: scroll down and see first if Songbird is listed, and if so, is it checkmarked - if not, check it and close - that should do it. If thats"s not it, go to Synaptic and search for and remove Songbird and then go here: [http://www.getdeb.net/app/Songbird](http://www.getdeb.net/app/Songbird) and download and install from there - that should do it.
