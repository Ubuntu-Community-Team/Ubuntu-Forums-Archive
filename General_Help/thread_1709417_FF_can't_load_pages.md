### Post by +TheKillingName+ on 2011-03-18
It was just about to disable IPv6 from FF.
I've already disabled it from ubuntu preferences... 
For other people encountering the same problem, just do this in Firefox:

[LIST]
[*]Type **about:config** in the address bar, press Enter.
[*]Find **network.dns.disableIPv6** in the list.
[*]Right-click -> Toggle in order to set it as** false**
[/LIST]
