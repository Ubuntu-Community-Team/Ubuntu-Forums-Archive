### Post by blane2 on 2011-08-01
Virtualbox

download a distro's .iso mount with virtualbox and install.


if you want to skip the installation steps then just download a .vdi image and mount it (a fully set up system)

[http://virtualboxes.org/images/](http://virtualboxes.org/images/)

either way is fine. Images can save you time if you're trying out various distros to see what you like best.

If all you want is browser operations, I'd go for the smallest linux distro you can find. 

Puppy is a good choice... as long as guest additions are available.
