### Post by MAFoElffen on 2023-06-29
Please run the UbuntuForums 'system-info' script like this:
```

wget -N -t 5 -T 10 https://github.com/UbuntuForums/system-info/raw/main/system-info && \
chmod +x system-info && \
./system-info --details

```
Please post the URL of where it gets uploaded to a pastebin.

Running the script like that, with the '--details' option, in the sound section of the report, will tell us the detailed sound hardware and your systems current settings.
