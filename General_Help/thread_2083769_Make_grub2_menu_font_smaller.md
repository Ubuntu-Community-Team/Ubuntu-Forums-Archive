### Post by gilloz on 2012-11-13
After installing Ubuntu 12.04 some time ago, I had to change the grub2 menu font to a normal size as oppose to the very small font that had come up after the new installation of Ubuntu 12.04.  Today, I had to replace my Bios battery.  When the grub 2 menu came up at startup, the font was huge and I was unable to select Ubuntu because it was clear out of the screen view.  How can I change it back to what I had originally?

Update:  Found a solution.  Just did a sudo update-grub  in terminal, and grub menu font is back to it's original size.
