### Post by drs305 on 2010-04-12
wme,

Welcome to Ubuntu and the Ubuntu forums. Note we try to keep the language a little more gentle in these forums, even when discussing other operating systems.
As for getting to see the Grub menu, try holding down the SHIFT key during the early boot stages. It should display your menu and perhaps you can boot another kernel or into the recovery mode.

There are several links in my signature that may help, depending on your comfort level and computer knowledge.

If you still can't boot, here are instructions on how to boot from the Grub 2 terminal.
[https://help.ubuntu.com/community/Grub2#Using%20CLI%20to%20Boot]("https://help.ubuntu.com/community/Grub2#Using%20CLI%20to%20Boot")

Other sections of the guide provide instructions on how to reinstall Grub 2 if that becomes necessary, and there is a "boot info" script we can have you run that will help us if needed. 

Come back if you are still having no success and we can provide help. Run this script: [http://bootinfoscript.sourceforge.net/]("http://bootinfoscript.sourceforge.net/") and post the results here.
