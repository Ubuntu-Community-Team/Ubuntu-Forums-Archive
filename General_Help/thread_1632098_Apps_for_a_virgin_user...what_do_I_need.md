### Post by lobralleo on 2010-11-27
Welcome back to Linux!

The list of applications you need will depend on what you plan to do with your computer; Ubuntu already comes with a number of general purpose applications ready for use: you will get an office suite (OpenOffice), a web browser (Firefox), a mail client (Evolution), a document reader for pdf and other formats (Evince)  and the list goes on and on - you will find all them in your menu, and you can check the Ubuntu Software Center for many more.

To switch on the eye-candy, just go to System -> Personal -> Appearance: you can activate the fancy effects there. To control and tweak them, you can install ccsm or compizconfig-settings-manager (again, you will find these packages in the Ubuntu Software Center).

Have fun! :)

In this case, I would suggest to just start and play around in Ubuntu to test the various packages available and figure out what you want to do and what you will need. Of course, the forums and the net will be your friends for all inquiries :)

As I said, Ubuntu already comes with an almost complete array of applications for most everyday uses. Everything else can be found in the repositories and installed via Software Center, Synaptic or command line.

A short list of applications that I use mostly every day, just from the top of my head:
web browsing: Firefox, Google Chrome;
e-mail client: Thunderbird;
media player: VLC (also comes with a web browser plugin);
image editing: GIMP;
build panorama images: Hugin;
video editing: Avidemux;
audio editing: Audacity;
FTP frontend: gFTP;
application launcher: gnome-do.

Of course, the list cannot be complete, since your personal needs will shape your software list; for instance, if you want to delve into programming, you will need compilers and perhaps a programming environment; if you need to produce documents, you will probably use OpenOffice, or perhaps LaTeX; and so on.

### Post by SeijiSensei on 2010-11-27
I suggest burning both an [Ubuntu]("http://cdimage.ubuntu.com/releases/lucid/release/") and a [Kubuntu]("http://cdimage.ubuntu.com/kubuntu/releases/lucid/release/") LiveCD and trying them both for a while.  Loading programs will take longer since they'll be read off the CD, but otherwise you'll get a good sense of what's available in both major desktop environments.

I've used KDE, the desktop environment in Kubuntu, for a least a half-dozen years, probably more. I do add a few things to a stock installation:

Image Editor: GIMP
Media Player: smplayer
Web browser: Firefox (Kubuntu ships the KDE browser Rekonq)
Mail Client: Thunderbird (Ubuntu uses Evolution; Kubuntu uses kmail)
Camera Manager: digiKam

Install kubuntu-restricted-extras before you install smplayer.  You can install Firefox from the main menu and everything else using KPackageKit.  

KDE has a lot of eye-candy.  Use System Settings to see what's available.
