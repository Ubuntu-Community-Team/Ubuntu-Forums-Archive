### Post by az on 2004-11-21
Actually, universe packages are from debian.

Become a debian developer.

Add your package to debian (unstable) and when the Ubuntu build daemons try to compile it and succeed, it is added to the universe repository.

This is what is currently happening with Hoary.  Warty is already released, so only security updates are introduced.


This is why Ubuntu is good for debian.  More people adding stuff to debian.
