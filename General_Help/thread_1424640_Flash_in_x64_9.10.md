### Post by darolu on 2010-03-08
Easy and safe? open a terminal and type:
```
$ sudo apt-get install flashplugin-nonfree
```
If you don't like/want to use a terminal, open Synaptic (at System - Admin) and search for "flashplugin-nonfree" and install from there.

Another -not THAT easy- but safe way to get it, is going to [http://get.adobe.com/flashplayer](http://get.adobe.com/flashplayer) and downloading the .deb file for Ubuntu.
