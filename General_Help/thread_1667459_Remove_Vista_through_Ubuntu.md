### Post by bcbc on 2011-01-15
You have wubi so you can't remove windows. You can shrink your windows partition from within Vista and then migrate the wubi ubuntu to the new partition [http://ubuntuforums.org/showthread.php?t=1519354](http://ubuntuforums.org/showthread.php?t=1519354)

This assumes you don't have 4 primary partitions already. Afterwards you can do whatever you want with windows, but if you have only be using ubuntu for a little while then I recommend keeping both until you're sure - or create restore dvds.
