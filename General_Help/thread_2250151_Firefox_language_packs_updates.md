Hi

I use Ubuntu 14.04 LTS.
In the update menu (starts every time I login) there are many updates for languages packs used by Firefox. Those contain multiple languages that I do not need. 
I can disable the installation per language pack but how can I configure my system to only show the updates for the languages I use?

Thanks

### Post by Impavidus on 2014-10-27
The update manager will always offer updates to all language packs you have installed. If you don't need them (the packages, not the updates), you can uninstall them through the package manager.

### Post by vasa1 on 2014-10-27
> **iqdesigner said:**
> ... how can I configure my system to only show the updates for the languages I use?...
The system will offer updates for whatever is installed whether you use it or not.

If you still want to have packages installed but not updated, it may be possible to pin those packages or put a "hold" on them.

BTW, just how many Firefox language "packs" do you have? Apt-cache lists > 80.
