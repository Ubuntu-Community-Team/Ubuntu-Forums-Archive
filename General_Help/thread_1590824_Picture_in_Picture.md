### Post by ankspo71 on 2010-10-09
Hi,
You can have window previews by installing "compizconfig-settings-manager". After you install it, you will find the application in your system menu. Open that application up, then enable the windows preview feature. 

You can get more desktop effects if you install "compiz-fusion-plugins-main" and "compiz-fusion-plugins-extra" too.

Hope this helps.
