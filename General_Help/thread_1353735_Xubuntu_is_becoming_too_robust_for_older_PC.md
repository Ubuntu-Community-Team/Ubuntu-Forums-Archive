### Post by kellemes on 2009-12-13
I'd really take a look at [Arch]("http://www.archlinux.org/"), using it myself for years. Maybe not that easy to setup, takes some time and effort. But it's a one time effort, rolling release means you never have to reinstall the system again.. And you're having the most bleeding edge system available!

If you're not able or willing to spend the time setting up the system I can highly recommend [Zenwalk]("http://www.zenwalk.org/"). very easy to setup, very slim, very fast, very much up-to-date and a very friendly community to help you out.

Another way to speed up your current system is to simply install a lighter wm like [icewm]("http://www.icewm.org/") or [LXDE]("http://lxde.org/") (Yes I know it's openbox, but more..). Pretty complete dm's that work great on older systems.

Cannot remember ever having used Vector.
