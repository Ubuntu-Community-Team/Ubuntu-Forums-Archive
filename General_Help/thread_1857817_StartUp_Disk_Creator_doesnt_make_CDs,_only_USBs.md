### Post by An Sanct on 2011-10-11
> **Tone303 said:**
> Blank CDs wont appear in the "Disk To Use:" Section of start up disk creator. Only Hard Drives or USB Flashes will. Strange, guess ill just use Brasero to burn as image.

Exactly, startup disk creator creates startup USBs, to get a Live/install CD, burn the image with any CD/DVD burning tool.

PS. Please, make sure, to "Burn image" and not just burn the ISO file to the CD, as "Burn image" also writes boot sectors.
