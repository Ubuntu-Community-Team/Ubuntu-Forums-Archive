### Post by gfixler on 2008-02-16
We all know we can select a URL, middle-mouse on the icon in the address bar, and go there. However, last year sometime I was *also* able to middle-mouse on the big Google 'G' next to the search box, to paste whatever I had selected, and automatically search for it. I use mouse gestures, so my usual way of looking things up was:

1) select something
2) mouse gesture open a new tab (click, drag down, release)
3) middle-click on the 'G'

I don't like right-clicking, and choosing from a long list of things in there now the "Search Google for *whatever*" option, and I simply cannot find how to get this working again. Nothing happens now, so I have to select the text in the there, either hit delete on the keyboard, or right-click, and choose delete from a drop-down menu, then go select the text, make a new tab, middle-click in the box, and *then* click the magnifying glass. I also don't want to do the "I'm Feeling Lucky" method of middle-clicking in the page to auto-search for what's selected. I've config:'d that accursed middle-mouse option away long ago.

Any ideas? This option isn't in Tab Mix Plus (I've searched it countless times over), nor Firefox's options/preferences, nor in Google (it's all about the (@*$% URL bar ;), nor here in the UF. Help!

### Post by gfixler on 2008-02-17
I've found something just by fiddling that could possibly become my alternative. If I select some text, then drag it to the search box, it replaces what's in there, and submits the search, all in one shot. Also, holding Alt while doing this makes it pop up in a new tab, as unlike the middle-click method, I can't just gesture open a new tab, and middle-click on the search box's 'G,' but must drag, which can only happen in the same tab. It's something, for now.
