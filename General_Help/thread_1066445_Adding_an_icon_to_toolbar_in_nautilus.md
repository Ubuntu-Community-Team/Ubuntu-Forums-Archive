### Post by kanikilu on 2009-02-11
I don't know of any way to add an icon to the toolbar, however, if you always want the sidebar to show, try this:

- Run gconf-editor from the terminal (or ALT-F2)
- Browse to apps > nautilus > preferences
- Check the box next to the item "start_with_sidebar"
