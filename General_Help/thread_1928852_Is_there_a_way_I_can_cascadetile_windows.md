### Post by Paddy Landau on 2012-02-21
Not exactly. However, if you enable the Grid function in Compiz, you can send windows to the left, right, top, bottom, or chosen corner and thereby tile a limited number of windows.

Install [ccsm]("apt:ccsm") if you have not already done so.

[http://www.youtube.com/watch?v=6Df0CY4tkm0](http://www.youtube.com/watch?v=6Df0CY4tkm0)
