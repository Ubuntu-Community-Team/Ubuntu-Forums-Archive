### Post by lovinglinux on 2009-10-01
> **jokerfit said:**
> [http://packages.ubuntu.com/jaunty/all/deluge/download](http://packages.ubuntu.com/jaunty/all/deluge/download)

Are all of these safe to download from?

Yes, but you don't need to manually download and install them. All you have to do is use the built-in installation managers. Click the Add/Remove from the main menu and search for Deluge then select it and click "Apply Changes".

Another easy way is to install via command-line. Open "Applications >> Accessories >> Terminal" the copy and paste the following code:

```
sudo apt-get install deluge
```

Or you can simply [click here](apt:deluge) to automatically install Deluge from the official repositories.

For more info on installation methods visit [https://help.ubuntu.com/9.04/add-applications/C/index.html](https://help.ubuntu.com/9.04/add-applications/C/index.html)

### Post by Thomas Kenobi on 2009-10-05
Haven't tried Deluge myself, but I'll cast my vote for Transmission. Extremely lightweight and it has just those features you'll need, saving you time customising everything.

As for Azureus/Vuze I must dissuade you from using it. While it may be extremely customisable, allowing you to tweak almost every variable you could possibly want, it is also extremely resource heavy. On my system it used upwards of 300MB of RAM, while transmission used 10-15MB. Now that isn't a big issue on Windows, since the OS isn't particularly fast anyway and Vuze slowing it down some more isn't going to make a substantial difference, but on Ubuntu it represents a terrible waste of resources.
