### Post by thathatman on 2008-01-12
Hi, I'm using an Apple USB keyboard. It has F14-16 in place of Scroll Lock/SysRq/Pause keys. I found that I can't use the Alt+SysRq "magic keys" with this set up. Is this some way to make F15 into SysRq?
Thanks!

---

### Post by erpo on 2008-01-13
You could try going to Ubuntu menu->System->Preferences->Keyboard, choosing the Layouts tab, and using the "Choose..." button to tell the system you have an Apple keyboard instead of a generic model.
