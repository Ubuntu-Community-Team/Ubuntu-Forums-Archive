### Post by PRC09 on 2010-03-03
EXE. files are windows only files but you dont need internet to open them,just windows (or Wine).If you have windows already installed and have downloaded the drivers with the live cd,you need to be in windows to make the exe. files  work.Just boot into windows and go to where ever you have saved the downloaded files and run them from there.Hope this helps.....
