### Post by grahammechanical on 2021-11-21
Ubuntu 20.10 reached end of life 22 July 2021. The standard practice for end of life releases is for the repositories to be closed and moved to the old-releases server. That icon is possibly an indicator icon warning that Ubuntu Updater cannot find the sources or repositories for 20.10.

When was the last time that install was updated? Have you tried running Ubuntu Updater. It should be set to notify you of the next Ubuntu release which was Ubuntu 21.04. Ubuntu Updater might still offer you the option to upgrade to 21.04. If it does not make that offer then updating 20.10 and upgrading to 21.04 will be more complicated. Consider doing a fresh install of Ubuntu 21.10 and then upgrade to 22.04 next summer. You will then be on a Long Term Support release with a minimum of  five years support.

If you do manage to upgrade to 21.04 please keep in mind that it will reach end of support life in January 2022. 

[https://wiki.ubuntu.com/Releases](https://wiki.ubuntu.com/Releases)
