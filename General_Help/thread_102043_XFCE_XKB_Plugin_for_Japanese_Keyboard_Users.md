### Post by kairu0 on 2005-12-11
In case there are any others who use a Japanese keyboard with XFCE4 and have problems with switching keyboard layouts with the xfce4-xkb-plugin, here is my alternative .deb.

Specifically, the Japanese layout showed up twice as PC and JP, and so the last keyboard layout was bumped off of the list. I added a crude filter to remove the PC layout from ever getting added to the xkb plugin.

If you already have XFCE working with your keyboard, then you obviously don't need this package.
