I havn't seen the video because I am on a very slow connection, but I think the effect you are looking for might be "Dodge" in Compiz. If that is the case, then you need to open CCSM, which is available as System->Preferences->"Advanced Desktop Effects Settings".

[LIST=1]
[*] In CCSM, enable "Animations" in the "Effects" category.
[*] Click on the "Animations" applet, and select "Focus Animation" tab. [*] There is only one effect displayed by default, which is set to "Fade". Select this effect, and click "Edit".
[*] In the dialog that shows up, change the effect from "Fade" to "Dodge".
[/LIST]

That is it, basically. But you may have to play with the "Duration" setting in order to get a satisfying effect. 250 seems to be a nice value, so you can actually see the windows moving, but it doesn't get in your way.

Well, i think the first problem is a slight confusion about what the effect itself does. The Dodge effect, applied to the focus animation, creates an animation when clicking between one open window and another, not clicking on the close or minimize buttons. You can change those effects for close or minimize, but using dodge for that animation would not do much, as there isn't anything to dodge. The dodge is between to windows, one 'dodging' the other as you change focus to that window. Right now you have it set on fade, so change that to dodge and 250, then open up to non-maximized windows and click in between them.

I suggest to install compizconfig-settings-manager. Use synaptic and type "compizconfig" in the quick search entry field, it should show the settings manager and all related packages available. Mark and install as usual. Then you should find it in "System -> Preferences -> CompizConfig Setings Manager"

As to the "dodge" effect not being visible. I had to increase the time to 500 (!!!) in order to actually see the effect. That was on a Acer Aspire One Netbook, not the fastest imaginable computer, still fast enough to create the impression that "dogde" does noting. It still did, just much too fast. Please don't take the timing I used too literally, if it doesnt seem to work, just drag the slider quite a bit to the right. Then have another window open and click somewhere into it. If  the effect is too slow now, drag the slider back to the left, if it is visible, but still too fast (barely visible) drag it further to the right.

This worked for me, hope that helps
