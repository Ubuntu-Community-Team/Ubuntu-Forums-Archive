### Post by grahammechanical on 2016-01-23
Libreoffice has a Help system. Have you searched on "spell check?" Did you see the sub-menu "Activating for a language?" Those utilities should offer to install additional language support tools such as translation tools & other stuff.

### Post by Nuno_Abreu on 2016-01-23
Language support can be found on **myspell **specific package for your language.

Grab your Terminal and try to install this:
```
sudo apt-get install myspell-el-gr
```

You might have to configure the spelling language afterwards.
Tell me if it worked.
