### Post by Frogs Hair on 2011-02-20
Remove Awn , remove the Awn theme folder from  Home Folder > hidden folders .icon folder. This is where your latest theme is stored , so if you reinstall without removing it , the same theme from the previous installation comes back.

Remove the Awn folder from file system usr/share and reinstall.

### Post by kerry_s on 2011-02-20
open your file manager & press ctrl+h, that will show the hidden files/folders, go into .config & .gconf-> apps
they both have settings.
you need to quit awn before you delete the folders/files.


the 1 you can't delete is the original panel, that means the other panel has delete.
