### Post by tuxcantfly on 2007-04-29
> and it only works on Windows XP according to the official FAQ.

That's rather outdated info; it works on any windows version except Vista, currently

Also, the standard Wubi is rather limited when using Win9x (disk size is limited to 4 GB) so you may want to use this version: [http://www.cutlersoftware.com/ubuntusetup/wubi/downloads/experimental/wubi-7.04-netboot.exe](http://www.cutlersoftware.com/ubuntusetup/wubi/downloads/experimental/wubi-7.04-netboot.exe)

It's basically the same as the cd-based install, only it doesn't need a cd. Partitioning is required; it installs on a real partition, not a loopmounted file.

More info about wubi-netboot here: [http://ubuntuforums.org/showthread.php?t=427793](http://ubuntuforums.org/showthread.php?t=427793)
