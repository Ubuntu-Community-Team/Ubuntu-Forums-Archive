### Post by adult swim on 2009-03-14
once you install the older cups with the link that dcstar provided, you will want to open synaptic package manager and find the cups you installed.  make sure it it highlighted and then click on the package menu that is towards the top of the window.  click on lock version to keep cups from updating to the newer version the next time  you update your computer.
