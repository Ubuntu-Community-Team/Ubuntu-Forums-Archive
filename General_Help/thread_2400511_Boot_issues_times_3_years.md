### Post by oldfred on 2018-09-06
Those are now End of Life.
Best to down load current LTS long term support version either 16.04 or 18.04.
From Live installer in live mode you should be able to mount your old partition(s) and copy data to a flash drive or external hard drive.
Then you can reinstall Ubuntu. If older system, it will be BIOS. 
If from last 5 years it may be UEFI. And then how you boot install media is how it installs.

       Also link on how to create a  bootable DVD or USB flash drive, using Windows or Ubuntu, Min hardware requirements
