### Post by Yellow Pasque on 2013-04-30
> But what i actually was wondering. Is it so that most OS resources are used by GUI?
The Unity GUI and Gnome-Shell require basic 3D acceleration. If your GPU doesn't have sufficient 3D acceleration, then Unity/G-S start using your CPU to do all the graphics, which will cause a major slowdown on older CPU's.

> I mean does or can x/l-buntu 12.xx or 13.xx with the newer os under the hood (kernel 3++) weigh lower or same as the Ubuntu with Gnome but with lower under the hood os (kernel 2x)?
They can. 

> But then lets say i want some newer stuff. Like firefox etc. But i don't really see like i really need to upgrade to the newer version because of this. Because old one works perfect for me.
Well, if you can't install newer software that you want, then the old one does not work "perfect" then, does it? ;)
