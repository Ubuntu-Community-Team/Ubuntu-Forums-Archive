### Post by lah7 on 2013-08-29
Ubuntu (or newer ones for sure) have VPN abilities built right in. You should find them in the network icon on the menu bar and selecting "VPN Connections...". Assuming you know the VPN server and its credentials, you should be up and running right away.

Out of the box, I know PPTP is supported, but if this isn't the protocol that you used, you can install additional packages which offer the feature.

You could use a package manager like **Synaptic** to find various packages which offer a daemon or client which integrates and manages VPNs from your system if you're using a version of Ubuntu that might not have this.
