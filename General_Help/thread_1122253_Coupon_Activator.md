### Post by wfp on 2009-04-10
The software there trying to get you to install is most likely for Windows only. You might have some luck with the User Agent Switcher which is a Firefox addon you can get it here > [https://addons.mozilla.org/en-US/firefox/addon/59](https://addons.mozilla.org/en-US/firefox/addon/59)

After you install the Add-on and restart Firefox, you will see the Add-on under tools in Firefox then change it from the default to internet Explorer 7 (windows vista) All it does is fool the site in thinking your on vista.
