### Post by ibutho on 2008-04-08
The best way to find out what is wrong with a program if it doesn't work by clicking is to run a terminal and execute the commands used to start the program. This will result in error messages being printed on the screen. These messages will give you a clue on whats wrong. In your case run the command "frostwire" in the terminal and post back the error messages.
