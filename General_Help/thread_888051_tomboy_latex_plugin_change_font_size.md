### Post by urielgm on 2009-04-12
After a long time of trying different things this is what I came up with.

\[
\LARGE \emph{$x^2 \frac{p_n}{q_r}$}
\]

NOTE:
1) Between "\[" and "\large" there is only a carriage return.
   The same goes for each line after.
2) There is a space between \LARGE and \emph.
