### Post by Mark Phelps on 2011-10-26
If you're asking can you run the EXISTING Win7 install through a VM inside Ubuntu, the answer is NO.  You have to install it afresh inside the new Virtual Machine. 

And yeah, there have been hacks posted that ALLOW this to happen, but they end up corrupting and/or deactivating Win7 -- rendering it useless.
