### Post by wildmanne39 on 2011-07-07
Hi, I had the same problem, this is how you fix it.
```
gksu gedit /etc/default/grub
```
and change this line:

#GRUB_GFXMODE=640x480

to this:

GRUB_GFXMODE=640x480

Then save and exit the document.


Then do:
Code:
```
sudo update-grub
```
For better resolution at boot I suggest making the 640x480 800x600
