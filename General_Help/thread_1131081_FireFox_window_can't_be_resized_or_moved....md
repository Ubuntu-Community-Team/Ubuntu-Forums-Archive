### Post by radar2020 on 2009-04-20
A temporary fix for Firefox is to press F11 once or twice.  This will reset the screen so the max, min buttons are available again.

A permanent fix if you have compizconf settings manager installed is as follows:

Exit Firefox

SYSTEM>PREFERENCES>COMPIZCONFIG SETTINGSMGR>UTILITY>CLICK ON the word WORKAROUNDS which brings up a new screen and uncheck the  first box LEGACY FULL SCREEN SUPPORT.  Exit

Start Firefox
