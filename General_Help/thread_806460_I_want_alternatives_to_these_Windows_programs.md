### Post by x1a4 on 2008-05-25
Hi,

1- If you're using KDE, kget, otherwise gwget both of which are in the repositories.  Although by far the best download manager is the [Sun Download Manager]("http://www.sun.com/download/sdm/download.xml") which I highly recommend.

2- [Everest QDictionary]("http://www.free-soft.ro/qeverest/qeverest.html")

3- Use OpenOffice and save as PDF

4- K3B if you're using KDE, otherwise Brasero.  Both are in the repositories.  

5- [DTV]("http://hesdvb.sourceforge.net/") (view), [VDR]("http://www.cadsoft.de/vdr/") (record) or MythTV (view/record and more).  MythTV is in the repositories.

If you're not using KDE but want KDE applications, by all means you can install them, as long as you don't mind adding KDE libraries.

### Post by atomkarinca on 2008-05-25
If you want a local dictionary, gnome-dictionary would be your best bet. Open up Synaptic and search for "dictd". It's the dictionary server, with that install dict-freedict-deu-** (** being the languages you want). Then when you run Gnome Dictionary, under preferences, add a dictionary host, the server address being localhost and you're good to go.
