### Post by Lars Noodén on 2012-11-02
Your question is much more about MySQL than about shell scripting. I can't help but think that the problem is better addressed using perl or another scripting language with an API for MySQL.  Here is a short example which would work just as well for SELECT as it does for INSERT:
[http://ubuntuforums.org/showthread.php?t=2077261](http://ubuntuforums.org/showthread.php?t=2077261)

With your existing shell script, there's a small problem with calling date more than once as time passes.

```

date="$(date)"
DATE=$(date --date="$date" +%Y-%m-%d_%Hh%Mm) # Datestamp e.g 2002-09-21
DOW=$(date --date="$date" +%A) # Day of the week e.g. Monday
DNOW=$(date --date="$date" +%u) # Day number of the week 1 to 7 where 1 represents Monday
DOM=$(date --date="$date" +%d) # Date of the Month e.g. 27
M=$(date --date="$date" +%B) # Month e.g January
W=$(date --date="$date" +%V) # Week Number e.g 37

```
