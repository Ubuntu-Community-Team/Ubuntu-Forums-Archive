Then, download Hiren's Boot CD from [here]("http://www.9down.com/Hiren-s-BootCD-10-6-352820/") (use mirror servers at the bottom of this page), boot from it (into Mini Windows XP) and from the 'Recovery' Menu, choose 'Get Data Back for NTFS'. See if it can find your lost folders!
