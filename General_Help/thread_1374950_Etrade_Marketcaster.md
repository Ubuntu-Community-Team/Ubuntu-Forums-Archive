### Post by ionospheric on 2010-01-07
I found out how to get Etrade Marketcaster to work with Ubuntu 9.10. I tried to post this to an existing thread at

[http://ubuntuforums.org/showthread.php?t=121608]("http://ubuntuforums.org/showthread.php?t=121608")

however it is an archive thread that cannot be posted to so I am opening a new thread so that others can find this solution if they need to.

By default, Ubuntu 9.10 does not install the Sun version of Java. The Marketcaster does not work with the default Java version of Ubuntu.

The solution is simple: In Synaptic, search for "Java" and install sun-java6-jre or sun-java6-jdk and don't forget to install the sun-java6-plugin. Then restart your browser and Marketcaster should work. So much for the good news. The bad news: Stocks are down today..
