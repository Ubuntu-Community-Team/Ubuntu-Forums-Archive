### Post by testube_babies on 2006-05-07
Have you tried upgrading to the latest versions?

Add these to your /etc/apt/sources.list:
```
deb http://www.beerorkid.com/compiz dapper main
deb-src http://www.beerorkid.com/compiz dapper main
deb http://xgl.compiz.info/ dapper main
deb-src http://xgl.compiz.info/ dapper main
```

Then follow this tutorial:
[http://www.ubuntuforums.org/showthread.php?t=148860](http://www.ubuntuforums.org/showthread.php?t=148860)

And don't blame me if it breaks your system ;) I had to wipe my system clean once after trying this, so just be careful.
