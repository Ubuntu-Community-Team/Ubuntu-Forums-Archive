### Post by Cheesehead on 2009-02-02
The better way to install skype is using Synaptic, and the Medibuntu repository ( [https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu) ).

If you really want to use the downloaded .deb, you need to open it in gdebi or another deb-file manager. A .deb file is a package, not an executable.
