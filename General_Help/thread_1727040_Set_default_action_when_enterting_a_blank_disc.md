### Post by Kvothe on 2011-04-12
I have accidentally set my default option for a blank disc and told it not to ask me again. When I put a disc in, it runs that option and I can do nothing else unless I open up a different program manually. How do I access and edit this option again? Please respond as soon as possible.

---

### Post by spikoley on 2011-04-12
Open up Nautilus (Places> Home Folder) then go to Edit> Preferences and then its on the Media tab.

I found the solution on this [thread]("http://ubuntuforums.org/showthread.php?t=1287417"), so the credit goes to user OpenGuard.
