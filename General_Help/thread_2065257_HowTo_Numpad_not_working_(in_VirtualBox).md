HowTo: Numpad not working (in VirtualBox)
***********************************
1. Go to System->Preferences->Keyboard
2. Switch to "Mouse Key" tab, then REMOVE tick(checkmark) from "Pointer can be controlled using the keypad".

Tested on Ubuntu, BackTrack5R3-GNOME-64
