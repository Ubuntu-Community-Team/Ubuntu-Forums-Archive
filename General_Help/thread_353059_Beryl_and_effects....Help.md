### Post by _simon_ on 2007-02-04
To set burn (fire) on close:

Visual Effects -> Animations

Select the CLOSE tab.

Choose BURN from the drop down menu.

Don't worry if you haven't got the all options in the drop down menu that I have, I'm using the SVN.
