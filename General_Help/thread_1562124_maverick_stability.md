I'm using Maverick for everyday use.

But what is "everyday use"? For me it's visiting a few websites and reading my IMAP mail. There are a few crashes still and some of the daily updates don't always go as planned but I do have a Lucid installation that I can fall back to. I frequently copy all valuable data to a networked drive.

I'm not recommending that you use Maverick for everyday use though as I have seen enough posts to show that other users are experiencing _far_ more problems than I am. Different hardware will cause different problems. I get 100+ updates most days so a lot of problems are being fixed but there is still a fair way to go.
