### Post by steviefordi on 2010-02-25
I don't remember the exact steps but you need to enable USB support - which I think is in the 'settings' section of your virtual machine.

Once you have done this, you need to allow the usb device to 'pass through' as by default it will only be available to the host PC and not the VM.

I'm sure the VirtualBox help explains how to achieve this.
