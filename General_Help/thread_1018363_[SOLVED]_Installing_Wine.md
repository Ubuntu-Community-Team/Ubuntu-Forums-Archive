### Post by eBoxNet on 2008-12-22
Download wine via this link : [http://www.winehq.org/download/deb](http://www.winehq.org/download/deb)

after the download double click the new file on your desktop in order to install wine.

next thing is to download your poker game and double click it.
if it ask you then choose run with wine emulator.
if your poker game is compatible with wine the installation will finish and you will have a shortcut for it under your wine menu (applications->wine->game's name)
