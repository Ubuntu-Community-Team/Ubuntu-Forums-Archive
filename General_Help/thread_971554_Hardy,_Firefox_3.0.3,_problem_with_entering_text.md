### Post by jamesstansell on 2008-11-05
Your writing text problem most likely isn't related to your java issue so you would probably be better off to split this into 2 threads.

Hardy doesn't have the latest sun-java6 packages yet.  There is bug open about getting them but I don't know that it's being worked on yet.  The good news is that it does have the latest security fixes.  The recent update 10 (which is the version you're seeing on java.com) fixed a lot of things, but none were considered security-related.

A red cross failure?  I'm not sure I've seen that before for java applets.  What sites do you get that on?

BTW, this is the bug requested the backport of java 6u10 to hardy: [https://bugs.launchpad.net/ubuntu/+source/sun-java6/+bug/288658](https://bugs.launchpad.net/ubuntu/+source/sun-java6/+bug/288658)
