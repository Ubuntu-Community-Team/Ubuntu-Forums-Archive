### Post by Matthewslf on 2008-05-12
google for getdeb, once on their site search for compiz fusion. make sure you download the dependencies too. A site I like less but that still works is packages.ubuntu.com. Same dependencies rule applies. You should be able to find modem drivers at either of these sites too.

Edit: I'm not sure whether 7.10 uses compiz or compiz fusion. check then get the appropriate one. By the way, once you have the .deb files, install them by double clicking the files. You may get a bunch of dependancy errors, install the packages without errors and then those with should stop giving errors after a few packages.
