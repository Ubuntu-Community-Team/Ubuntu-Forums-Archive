### Post by SonWon on 2006-11-07
Many Linux users need their PDA/phone to sync.  Many Windows users are not switching to Linux because their PDA/Phone will not sync.  In my case I am using VMware to sync to Windows 2000 Pro.  Way to hard for the average Windows user.  If Linux is going to go mainstream then syncing with portable devices must happen.  If you are a developer or can help please consider supporting this project.


SynCE is not dead!

But it needs some help. Maybe from you?
Help needed

The SynCE projects needs people that...

    * Create a roadmap for SynCE and try to make the project follow it
    * Handle all bugs in the Bug tracker
    * Clean-up the patches for the ipaq driver and submit them to the linux-usb mailing list so we can remove the kernel-2.6-driver download.
    * Reply to people who ask questions on the forums and mailing lists
    * Improve the GNOME support: synce-gnomevfs, synce-software-manager, synce-trayicon
    * Improve the OpenSync plugin for synchronizing pre-WM5 devices
    * Make packages for various Linux distributions
    * Do other things that benefit the project

Are you interested or just want to add to the list above? Send a mail to David Eriksson.

David's email link can be found at [http://synce.sourceforge.net/synce/]("http://synce.sourceforge.net/synce/")

Thank you,
SonWon
