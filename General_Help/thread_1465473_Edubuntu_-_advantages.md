### Post by P4man on 2010-04-29
open a terminal and type

> sudo apt-get install edubuntu-desktop 

and it will turn your ubuntu into edubuntu. Simple as that.

 Or do the same thing using ubuntu software center (install "educational desktop for ubuntu")

BTW,expect ubuntu servers to be slow today.. everyone is downloading 10.04
