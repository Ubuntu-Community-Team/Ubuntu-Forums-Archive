### Post by drs305 on 2008-09-25
Try this. Right click on the file type and select 'Properties'. Then click on the 'Open With' tab. Highlight Bluefish and then select 'Remove'. Make sure all the Bluefish references are removed, if there are multiple entries listed. I'd close it out, then repeat to add Bluefish back.
