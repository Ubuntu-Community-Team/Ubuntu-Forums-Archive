Do you have build-essential installed?

> sudo apt-get install build-essential

Also, check the extracted folder for a README/INSTALL document.  Will have instructions and any required dependencies the program might need to install correctly.  Compiling from source is not recommended for someone new to Linux, unless you just really have to/want to.  Always check the Repositories 1st for a program.
