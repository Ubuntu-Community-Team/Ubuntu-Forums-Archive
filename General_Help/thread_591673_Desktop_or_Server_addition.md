### Post by jayson.rowe on 2007-10-25
By default the Server CD doesn't even install X or a Desktop Environment - you have to do that manually from the Command Line after install if you want it...plus the Kernel in the server edtion is tuned for Server performance (throughput) rather than Desktop Responsiveness...

Just because you want to run some server services doesn't mean that you have to run the Server version - I'd go w/ the Desktop myself - just my $0.02
