Press <Shift>+<2> to type the symbol "@".

If you can't figure out why it's not working by using the keyboard layout manager in "System > Preferences > Keyboard", you may need to use xev to figure out the keycode for the <Shift>+<2> combination by running this program in the terminal:```
xev
```
press the key combination and find it's keycode. Then use the program xmodmap to change the keycode to input "@".
