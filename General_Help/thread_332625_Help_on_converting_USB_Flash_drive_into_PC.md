### Post by jtlovell on 2007-10-17
I just started messing with mojopac this week.  I installed it onto my ipod and had no problems using it with Windows XP only; I have not tried Vista yet.  And you will not be able to use it on any Linux system.  You will be also limited to the amount of software that you want to install by the size of you thumb drive.  Such as if you want to install M$ office you will burn almost half you memory.  

The only down fall on using mojopac is when you are in the mojopac environment you will not be able to view the host computer files.  You will have to minimize mojopac to view the host computer files. 

I suggest trying it out!
