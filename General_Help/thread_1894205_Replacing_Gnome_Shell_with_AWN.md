### Post by yetiman64 on 2011-12-12
Please note AWN (Avant Window Navigator) is only a dock application, it does not do windows compositing or any such tasks. You can run it in gnome-shell or unity (or any Desktop Environment you choose to use) but the compositing you mention is done by the DE (or at least a component of it - like compiz or metacity etc) itself.
