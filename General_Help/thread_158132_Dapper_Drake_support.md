### Post by philipacamaniac on 2006-04-10
>Are security fixes provided? 

Yes.

>Are application updates provided, e.g. for firefox or openoffice? 

No, except for security updates, and through the *Backports* repository.

>Which applications are supported?

All applications in *Main* are fully supported by Canonical. Applications in *Universe* are supported by the community. Applications in *Multiverse* are from outside Ubuntu (e.g., Debian archives), and are not supported.
