### Post by grahammechanical on 2011-06-08
For the wireless connection issue you need to set the connection to connect automatically. Click on the network manager icon and select Edit connections. Select the wireless tab and select your wireless connection and click Edit and then tick the box. It is up near the top. It is easy to miss. You might want to tick Available to all users down near the bottom.
