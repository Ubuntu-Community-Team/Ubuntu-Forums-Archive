### Post by Tamlynmac on 2009-12-10
It appears to be supported by the HPLIP print driver - see [this]("http://hplipopensource.com/hplip-web/models/deskjet/deskjet_6980_series.html").

You can use this [sites]("http://hplipopensource.com/hplip-web/supported_devices/index.html") printer search to identify supported printers and which OS's/features work for the HPLIP driver(s).

Good Luck and hope this helps.

### Post by Dennis N on 2009-12-10
I have an HP6980 which is in the same series: Network printer - can use wireless or ethernet. Mine is connected with ethernet cable to my router. It works without any problems with Ubuntu. Easy installation - probably 5 minutes in the printer setup dialog to get it going.
