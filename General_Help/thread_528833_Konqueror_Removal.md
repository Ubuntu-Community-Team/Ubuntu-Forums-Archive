### Post by Arisna on 2007-08-19
I don't know whether it is possible to uninstall Konqueror and keep a working installation of KDE, but you can make Dolphin the default file manager by using Konqueror, ironically, and going to Settings > Configure Konqueror > File Associations > inode and changing the "Directory" settings, etc., to use Dolphin first.
