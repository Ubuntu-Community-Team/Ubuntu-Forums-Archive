### Post by joshier on 2008-08-28
[https://fedorahosted.org/liveusb-creator](https://fedorahosted.org/liveusb-creator)

OK, basically, what this application (available in windows and linux) does, is it downloads an ISO of fedora for you, enables you to select which USB stick you'd like to put fedora on, and then installs it for you.

After this, you can reboot, then select your USB stick and you'll have a fedora setup with a writeable partition that you define the size of within the first setup and bingo - a portable USB OS stick that even a novice can setup for!

Possible with ubuntu?.. I should thinkso.

### Post by DanJC on 2008-08-28
I have plenty of experience with Fedora's live USB creator (I tried for a while to get a good Live USB, before I discovered Wubi), and I can tell you that it is not as simple as it makes itself out to be.  Sure, putting Fedora on the USB stick is easy, but getting it to work is a whole different story.  After putting Fedora on the USB stick, I estimate you have a 1/10 chance of it actually working.  There's stuff you can try to make it work, like getting makeboot.bat and running it from inside the USB stick, but that doesn't work too often either.  I've only gotten it to work once, and it only booted once.  After making a text file to test the persistent memory, it refused to boot, and I was stuck booting into Vista.
*shudders*
