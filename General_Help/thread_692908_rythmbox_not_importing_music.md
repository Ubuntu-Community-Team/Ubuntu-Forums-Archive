### Post by src2206 on 2008-02-11
Welcome to the Community my friend :)

Do you have ogg codec installed? I think it is not :). This is the most common reason for RythmBox not to import music.

Check out the Ogg Vorbis official page and download the package for Gutsy. The other way around is to install AutomatiX2, which will give you an option to install all the codecs (both open source and closed source) required to play any media file.
