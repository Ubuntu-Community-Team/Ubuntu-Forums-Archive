### Post by OMRebel on 2006-10-22
Hey guys, I've got a bit of a problem with my top menu.  There used to be a seperator bar up there on the top right, but it is gone now.  I am also seeing that programs, such as Gaim, used to minimize to that task bar.  Doesn't happen anymore.  How can I correct this?  I hope this makes sense.

### Post by CatKiller on 2006-10-22
If you want a separator, right-click on the panel, selcet "Add to Panel..." and select Separator. Similarly, if some programs used to use the notification area, and now they can't, from the Add to Panel screen select Notification Area.

---

### Post by OMRebel on 2006-10-22
It was the Notification Area that was missing.  Added that, and it's working like a charm now.  Thanks!
