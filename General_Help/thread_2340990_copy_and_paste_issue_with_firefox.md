### Post by dragonfly41 on 2016-10-24
This is not a "stupid bug".
If you highlight plain text and right click you see only one option .. Copy  (Ctrl + C).
If you highlight a hyperlink and right click you see two options .. Copy link address _and_ Copy  (Ctrl + C).

Just try Ctrl+C to capture the text.
