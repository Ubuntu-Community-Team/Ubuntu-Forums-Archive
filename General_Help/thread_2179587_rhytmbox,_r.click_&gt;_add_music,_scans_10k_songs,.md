### Post by Kirboosy on 2013-10-09
Welcome to the forums!

Try this,

Edit-->Preferences-->Music-->"Library Location: Music files are placed in:"--> Click the Browse button. Navigate to where your collection is stored and add it. 
Also, make sure the "Watch my library for new files" is checked

Hope that helps!
~Caboose
PS Do you have the mp3 codecs installed? If not they can be installed through the following command
```
sudo apt-get install ubuntu-restricted-extras
```
