### Post by sdennie on 2008-04-26
It will depend largely on the games you plan to play.  I don't know of any VM software that fully virtualizes the video card so, I don't think you will have much luck running 3D games within a virtual machine.  However, the Wine project has had a lot of success running games and you may want to check it out: [http://www.winehq.org/](http://www.winehq.org/).  There is also a fork of Wine called cedega that's primary goal is to run Windows games on linux.  However, it requires a subscription.
