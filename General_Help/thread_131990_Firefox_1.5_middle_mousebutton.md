### Post by Rizado on 2006-02-17
Hi, I got a small problem, not a problem really but it's bugging me. In windows when I middleclick on a tab it closes, but with linux it just opens some strange page, usually with a word from something I've been reading. It does the same if I middleclick anywhere (except on a link, that works fine.) in a page too. Very anoying because I often middleclick by accidant and sometimes it's not possible to go back. Any ideas?

---

### Post by mustang on 2006-02-17
Open up firefox, type in **about:config** in the address bar. Type in "contentload" in the filter. Change the boolean value to false by double clicking it.
