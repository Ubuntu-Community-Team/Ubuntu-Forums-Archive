### Post by Senori on 2006-07-01
The "Computer" menu, and the Control Panels, can be found in GNOME CVS in the "slab" module. There is a Dapper .deb available in [this thread](http://www.ubuntuforums.org/showthread.php?t=205266) from cbudden, but it needs to be updated for some changes.

The theme found there is Industrial. The icon theme is also Industrial- the latter is in an RPM that you can find [here](http://mirrors.kernel.org/opensuse/distribution/SL-OSS-factory/inst-source/suse/noarch/gnome2-SuSE-10.1-31.noarch.rpm).
