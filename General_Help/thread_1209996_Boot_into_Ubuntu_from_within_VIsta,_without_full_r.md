### Post by earthpigg on 2009-07-10
you can run ubuntu _*inside*_ of windows as a native windows application, if you like. not emulaton, not a virtual machine, and not dual boot.

[http://www.ghacks.net/2009/04/02/portable-ubuntu/](http://www.ghacks.net/2009/04/02/portable-ubuntu/)

> One could think that running an operating system inside another operating system would surely demand lots of system resources. This is surprisingly not the case. Running Ubuntu portable requires less than 50 Megabytes of computer memory that are occupied by its processes.

screenshots linked above pretty much say it all.
