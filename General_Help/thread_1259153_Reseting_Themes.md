### Post by orlox on 2009-09-06
Themes installed by the user are stored at ~/.themes. You can navigate to that folder and delete anything you think is screwing things up. Also, if you have a custom iconset thats causing trouble, these are stored at ~/.icons.

Messing with this folders wont affect the default themes, cause this are well protected. The simplest way to set the thing to the defaults would be to erase both folders completely, but if you're unsure, you can just move them to your desktop, and see if things are working as you want them to.
