There is no set standard for HTML e-mail, what happens is that things are usually formatted to work with Microsoft products and anything else is just bad luck if you have problems.

Good web sites use basic HTML that meets standards and works with almost every e-mail client, crap web sites use fancy HTML that is only tested to work with Microsoft clients and usually does not meet the general HTML standards.
