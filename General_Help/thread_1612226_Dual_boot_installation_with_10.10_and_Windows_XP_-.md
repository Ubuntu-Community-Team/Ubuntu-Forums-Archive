### Post by trying_it_out on 2010-11-03
Don't know if this will apply to your issue as it stands, but anyway,

A good rule of thumb that I have used over the years when dealing with dual boot systems is - always install your Windows first, then install your Ubuntu linux second. This way you will avoid a lot of issues. Even if you have to adjust the size(resize) your windows partition to make room on your drive for Ubuntu or adjust partitions use Gparted from a live CD session.

 ie) Boot with install CD, choose try Ubuntu without installing, make your adjustments to your Hard Drive with Gparted or(Disk Utility), then restart, choose install and way you go. Grub should take care of everything else, automatically.

The only complaints Windows should make is the next time you boot into windows it will want to check for errors -Scandisk- usually on bootup, anyway, let it do its thing, and thats it...:)
