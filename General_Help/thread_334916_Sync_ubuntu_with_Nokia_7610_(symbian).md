### Post by CroEragon on 2007-01-10
This is weak point of Ubuntu, connecting with mobile phones. It can be done but with lot if work.

I have no experience with Nokias or any Symbian based phones (I have Sony Ericsson W850i) but i assume that you have flash card in phone and file transfer mode on your phone. That is your safest bet. You connect phone to USB and select File Transfer. Then it will use usual file transfer protocol same as using any other card reader and won't depend on phone software that may not be usable with Ubuntu. Apart from that i have no idea.
