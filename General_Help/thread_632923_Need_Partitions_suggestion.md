### Post by rsambuca on 2007-12-06
I recommend formatting it to whichever system you will be using the most.  If you will be using linux more, then format it to ext3 and use the fs-drivers for Windows (use the XP compatibility mode when installing).  If you will be using Vista more, then format it to ntfs.  Both ways work well, just a little slower when not using the native system.
