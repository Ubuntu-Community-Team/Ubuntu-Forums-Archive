### Post by kaamos on 2005-12-27
Try removing all vga=XXX parts from your /boot/grub/menu.lst file. So open a terminal and type```

sudo cp /boot/grub/menu.lst ~/
sudo gedit /boot/grub/menu.lst
```
And remove the vga parameter from the lines that have it. So a line like this> /boot/vmlinuz-2.6.12-10-686-smp root=/dev/hda3 ro **vga=835** quiet splash
is now like this> /boot/vmlinuz-2.6.12-10-686-smp root=/dev/hda3 ro  quiet splash
After rebooting the terminals hopefully work.

Hope this helps!
