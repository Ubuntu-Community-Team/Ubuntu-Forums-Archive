### Post by jbrown96 on 2009-11-09
Go to system settings-->Power Management. In edit profiles, there is an option to choose what happens when you press the power button. Note that you will need to do this for all of the profiles.

This is one of my favorite parts of KDE. If you spend a little bit of time setting up profiles for your laptop, it can work really nicely. There's so many more options than Gnome provides.
