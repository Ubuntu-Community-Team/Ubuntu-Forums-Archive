### Post by taurus on 2007-05-28
Okay, here is the problem.  You have /dev/sdb5 & /dev/sda2 in /etc/fstab when you should use /dev/hdb5 and /dev/hda2.  Therefore, edit /etc/fstab

```
gksudo gedit /etc/fstab
```
and convert those two partitions from

```
/dev/sdb5 /media/media ntfs-3g defaults,locale=en_US.UTF-8 0 0
/dev/sda2 /media/windows ntfs-3g defaults,locale=en_US.UTF-8 0 0
```
to 

```
/dev/**h**db5 /media/media ntfs-3g defaults,locale=en_US.UTF-8 0 0
/dev/**h**da2 /media/windows ntfs-3g defaults,locale=en_US.UTF-8 0 0
```
Save the changes and reboot.
