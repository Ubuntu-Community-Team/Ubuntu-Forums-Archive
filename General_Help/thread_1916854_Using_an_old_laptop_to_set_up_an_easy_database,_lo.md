### Post by llamabr on 2012-02-04
Ubuntu does seem to ship with all the features, but you don't have to use them.  Ubuntu is not gnome.  If you're finding that gnome is too heavy, and all you want is somewhere to park a browser or gnumeric or something, get a lighter window manager, like windowmaker:

[http://en.wikipedia.org/wiki/Window_Maker](http://en.wikipedia.org/wiki/Window_Maker)

damned small linux is good if you only have a very tiny disk, but that doesn't seem to be your issue.  Don't confuse small disk with small memory, or slow processor.  All different issues.
