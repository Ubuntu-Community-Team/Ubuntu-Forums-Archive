### Post by fooman on 2008-08-21
open firefox 3, navigate/surf to the web page you would like to have as your homepage. then go to:

edit > preferences > main tab

then simply click on the button that says "use current page".  if you want to use multiple pages...navigate to the pages (in different tabs),  then again go to the edit > pref > main tab and now the button will say "use current page*s*".  click it, close ...done.
