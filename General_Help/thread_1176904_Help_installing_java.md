### Post by MyR on 2009-06-02
Welcome to Ubuntu/Linux!

Go to Applications > Add/Remove
Change "Show:" to All available applications
Search: java
Check the box next to "Sun Java 6 Runtime"
Click apply changes. & enter your password

This should be the first place you look when you want to install something.  Most of the time you don't need to go to a website and download a program to install something as you would in windows.
