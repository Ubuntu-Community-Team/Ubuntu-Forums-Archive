### Post by C.S.Cameron on 2010-04-01
Don't try to upgrade or update a persistent install.
The kernel is part of a disk image that can't be updated.
It may seem like it is working but will just fill your drive with junk.
If you are not satisfied with the result of a persistent install then try a Full install ie install Ubuntu to your flash drive as if it were just another hard drive.
(disable your internal drive first though).
