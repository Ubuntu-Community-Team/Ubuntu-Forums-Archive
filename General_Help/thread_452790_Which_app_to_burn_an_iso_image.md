### Post by thk on 2007-05-23
Under the "places" menu, click the home folder icon. Now navigate to the iso file. Right click the iso file and select "write to cd/dvd".

### Post by vtel57 on 2007-05-23
In my Ubuntu, I use Gnomebaker. It's available in the Ubuntu repos:

```
 $ sudo apt-get install gnomebaker
```

Or you can search for it within the Synaptic graphic interface.

Luck!
