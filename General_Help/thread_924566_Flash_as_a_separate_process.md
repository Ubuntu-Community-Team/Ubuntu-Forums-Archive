### Post by fragos on 2008-09-19
I'm unsure if there is a direct way but if you download flash files they will play in Totem in a separate process. It appears that if you open Firefox preferences there is an Applications tab where you set the action for various file types. By default the action for Flash is "Use Shockwave Flash (in Firefox)". Click on the action and one of your choices is "Use Movie Player". I didn't try it but looks like that is the place for you to go.
