### Post by piquat on 2011-01-25
Coming from another newb, the easiest way (and will also teach you the least about what's really going on) is a Nautilus share.  Open up Nautilus and right click a folder, go to sharing options.  Guest, IIRC, will let anyone have access.  The windows machines on my network sees those shares w/o having to install a thing on them.

Guess I should add, Nautilus is the default, already installed, file manager.  I don't think it's called Nautilus in the menu though.
