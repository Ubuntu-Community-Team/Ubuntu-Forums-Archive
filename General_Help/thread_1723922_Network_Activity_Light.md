### Post by luigi_mb on 2011-04-07

> I am wondering if there is a Linux facility similar to the one found in windows wherein there is a little light, on the bottom of the screen, that shows net activity (sending and receiving).

Get "gnome-system-monitor" with Synaptic. Then right-click on a panel (upper or lower) and choose Add to Panel. Select System Monitor. Now right-click on System Monitor, select Preferences. and tick Network.
