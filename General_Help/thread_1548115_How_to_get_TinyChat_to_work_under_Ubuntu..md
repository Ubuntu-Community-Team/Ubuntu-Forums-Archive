This problem can be easily fixed by right clicking in the chat and going to global settings. That will open a help page. On that page in the right column click on "Global Privacy Settings panel" in the first article. This will open another page with the control panel in the top right section...click on the website privacy settings tab ( tv screen with an eye ) and add tinychat.com as always allow.
