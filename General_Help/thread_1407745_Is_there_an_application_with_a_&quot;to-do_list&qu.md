### Post by TennTux on 2010-02-17
People are not rushing to answer this post, so I will give it a try. I have a couple of suggestions some of which may seem trivial or obvious, but, I will offer them anyway.

The simplest is to use a text editor and write to a text file on a partition that all OS(s) can reach. It's crude but it would work. You could do the same with a word processor that have versions for all platforms like Open Office. Perhaps the most transparent would be some kind of web based app that was hosted on a server on the Internet. But, after viewing your post I threw a Java Application together in a couple of hours to do what you asked. You would need to install the Java Runtime version 1.6 or better. I am attaching a todo.jar file that contains 3 source code files and the corresponding executable code. It is a fairly simple application but it should do the job. Under windows with Java installed you should be able to double click the file to run it. Under Ubuntu you may need to right click and select open with Sun Java 6 Runtime.
