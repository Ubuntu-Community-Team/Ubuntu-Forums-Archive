### Post by pseudonym on 2006-12-31
Hi,

The [Readme]("http://us.download.nvidia.com/XFree86/Linux-x86/1.0-9746/README/appendix-d.html") at nvidia.com has all the X config options and how to add them to xorg.conf. Basically, you'll want to enable NvAGP and probably RenderAccel. VRAM only matters if you have a card which xorg has trouble detecting how much VRAM it has. Shouldn't be an issue for a FX5500.
