Guys & Gals,

  I am facing two different problems with the nvidia & nouveau drivers. With the nvidia driver, all seem to work just fine.... except one thing: cpu usage gets pretty high, especially when scrolling on any browser (e.g. facebook, youtube scrolling---->cpu goes up to 90+%, starts @~60%). Sometimes, trying to type text, therefore, gets preeeeeety slow. I am sure you get the picture. Yes, I have tried different desktops, other than the default Unity, but to no avail. Cloning my laptop's monitor to an external tv-monitor works like a charm, once I either use nvidia-settings (however, only works for one session and has to be re-done every time!--->I believe I read somewhere that this is the default behavior NVIDIA has chosen...anyways!) OR by manually editing the xorg.file, which I generated (see below). Cloning now works at every reboot/login-logout session. BTW, ```
 disper -c 
``` works great as well. I would be pleased if I didn't have the slow experience, at times, with this driver. Any ideas?

  Which brings us to the next issue: the nouveau driver. Cpu usage seems to be better, however, cloning my external monitor gives me a black tv-screen with "no signal detected" (connected via an hdmi cable, which has no problems whatsoever). Hence, in this case no cloning<--->no fun for me :(. Under "Displays" in Ubuntu, the two monitors are recognized, and choosing "mirror" still gives me a black external screen. I tried with "disper -c", as well. Still nothing. If I could get some help, to possibly clone my displays I'd be more than happy to stick with nouveau.

The laptop is not to be used for games. Basic usage, which includes surfing the Internet, office docs, "normal" work & movies. I am including some basic info, that I could think of. 
For anything further, please do ask me.
