### Post by Mockskin on 2006-05-31
I'm new here, and no linux guru, but my best guess ( based on breaking my alt and / keys repeatedly until I learned to use gconf-editor to set the shortcuts... ) is to completely default your keyboard prefs.

System -> Preferences -> Keyboard

Best way I found was to:
Go to the Layouts tab, click Add.
Select your layout ( likely just US English ) and click OK.
Now select the old layout and remove it.

( I found the reset to defaults button didn't work all the time, whereas the above method did, but what works for me may not work for you. )
Hopefully that'll help. I'd bet you broke it playing with the keyboard shortcut editor like I did.
