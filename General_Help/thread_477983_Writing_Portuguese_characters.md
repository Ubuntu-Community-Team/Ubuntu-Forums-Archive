### Post by tvst on 2007-06-18
being brazilian, i have the same problem. the "english international" layout in gnome is not the same as in windows, so the cedilla (cedilha, in portuguese) is not "acute accent + c" but "right-alt + comma" ("alt direito + vírgula"). it takes some getting used to...

i think you can manually change things by editing some config files. i'm not sure which exactly, but i think this can be done by editing/creating "~/.Xmodmap" appropriately.
