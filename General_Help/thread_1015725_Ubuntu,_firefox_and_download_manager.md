### Post by blazemore on 2008-12-19
I think it's a Firefox setting rather than an Ubuntu one.

Go to edit->preferences in Firefox.
Go to i think it's Content, or maybe something like Applications

And find the filetype pdf or whatever.
You can set the default behaviour, "always ask" might be most suitable for you.
