### Post by edhplus2 on 2010-11-09
I am using Ubuntu 10.10.

This might be what you are looking for:

There is a drawer applet that you can add to a panel. Right-click on an empty space in the panel, select "Add to Panel". Then scroll down to Drawer. Highlight it and click "add" (or just double click on Drawer.)  That will add it to the panel.

Then to add applications to the Drawer, right click on the drawer icon and select "Add to Drawer".  Select "Application Launcher: Copy a launcher from the applications menu." Highlight it and click "Forward" (or double click it.) For Rhythmbox: open up the "Sound and Video" choices by clicking on the triangle to the left. Scroll down, highlight "Rhythmbox" and select "add." That will add it to the drawer.

I don't know where to find touch-freeze and fusion-icon because I don't use those applications, so I don't think they are on my menu.
