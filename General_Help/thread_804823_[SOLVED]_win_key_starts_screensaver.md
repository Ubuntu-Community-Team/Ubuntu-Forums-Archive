### Post by lswest on 2008-05-23
check under system-->preferences-->keyboard shortcuts and under system-->preferences-->advanced desktop effects (if you have it installed)-->general options, and check the keyboard shortcuts to see if you accidentally set "lock screen" or somehow bound that key to start the screensaver.
