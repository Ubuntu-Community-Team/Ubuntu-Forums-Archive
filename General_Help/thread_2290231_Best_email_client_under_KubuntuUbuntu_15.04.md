### Post by buzzingrobot on 2015-08-10
Kmail is the resident default mail app in KDE.  You may see some displeasure expressed about it hanging around the web, but its current versions have behaved well for me. I don't know how it deals with Exchange or if it can import Exchange mail, if that's your concern about Outlook.

Be aware that Evolution is a complex Gnome app, including calendaring and other functions.  Installing it on KDE will  bring in a *large* number of Gnome dependencies.

A very limited number of mail clients are available for Linux and sadly, it doesn't seem their numbers will increase because many developers apparently think everyone is using a web interface for mail.
