### Post by aysiu on 2009-11-01
You can install the *xubuntu-desktop* package using Synaptic Package Manager or *apt-get*, log out, and then select Xfce from the session list and log back in again.

If you like Xfce, you can remove Gnome using this tutorial:
[http://www.psychocats.net/ubuntu/purexfce](http://www.psychocats.net/ubuntu/purexfce)
