### Post by HokeyFry on 2006-01-07
How do you stop icons from popping up onto the desktop when something is mounted? Like keeping a second partition or removable media (e.g. ipod, CD/DVD) from coming up on the desktop?

---

### Post by Madpilot on 2006-01-07
Applications menu --> System Tools --> Configuration Editor 

Then, in the Config Editor, apps --> nautilus --> desktop --> uncheck "volumes visible"

That should do it.

---

### Post by Norberg on 2006-01-07
System --> Setings --> Removable Media (looks like a blue cd) and uncheck the setings you don't want.
