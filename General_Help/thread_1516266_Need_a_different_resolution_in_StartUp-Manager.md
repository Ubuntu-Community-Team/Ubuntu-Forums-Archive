### Post by dcstar on 2010-06-24
> **Brausepaul said:**
> Hi,

I got a new widescreen TFT supporting 1680x1050. Upt o now I used StartUp-Manager to set the boot menu's resolution, but the tool only offers 1600x1200 pixels. Is there a way to have "my" resolution selectable and working?

No, those resolutions are native VESA functions and are very limited.

If you install **burg** you can select the available boot menu resolutions with the "R" key when the menu is displayed.
