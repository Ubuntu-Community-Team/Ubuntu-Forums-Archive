### Post by 3rdalbum on 2009-09-22
Skype is probably the best software for this; it's cross platform, it's available on Linux, and many people already have it. If you need a solution that doesn't depend on people downloading and installing Skype, then try Meebo (Meebo's voice and video relies on a third party Flash application called Tokbox that runs in-browser).

With Skype you will need to install and run the "padevchooser" package in order to set up Pulseaudio. Basically, run the Skype test call, go to the padevchooser icon in the notification area, and go to Volume Control. You'll see Skype listed in the Reocrding and Playback tabs; use the little arrow next to it to "move it" to the microphone and your headphones.
