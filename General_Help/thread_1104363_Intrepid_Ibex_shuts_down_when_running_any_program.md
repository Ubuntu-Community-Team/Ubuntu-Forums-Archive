### Post by jespdj on 2009-03-23
Random crashes are often a sign of hardware failure.

Try this: Test the memory by selecting memtest86+ in the GRUB boot menu. Let the memory test program run for a few hours. If it finds errors, then there's a big chance that the memory in your computer is failing.
