### Post by sudodus on 2011-12-16
I would suggest that you use the software that is included in Ubuntu. If you have the file browser Nautilus (default file browser in 'vanilla' Ubuntu), select from its menu system

File -- Connect to server -- SSH

and you will be connecting via sftp, the same service as by Winscp (but without the detour via Wine). You can save the session to the left panel of Nautilus, so that it is easily activated in the future.
