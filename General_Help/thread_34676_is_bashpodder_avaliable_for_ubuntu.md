### Post by weblars on 2005-05-16
I don't think that there are any ipodders available in the Ubuntu repositories yet. 

Bashpodder is a simple bash script. You can download and move it i.e. in a folder "Podcasts" in your home directory. The file bp.conf is used to store the URLs of subscribed podcasts. Just copy the URLs of the rss feeds seperated by a line break into this file. Starting the bashpodder.shell script will download the podcasts.

Visit [http://www.ipodder.org/directory/4/ipodderSoftware/linux](http://www.ipodder.org/directory/4/ipodderSoftware/linux) if you are interested in other solutions. Maybe jPodder is an alternative for you. It's a java program and can be dowloaded at [www.jpodder.com](www.jpodder.com)
