### Post by nice_like_rice on 2009-07-10
If you are looking to protect your children, many applications such as DansGuardian are available which allow web filtering etc.

As for screen capturing, I can think of many ways to go about this, I'm sure there is an application somewhere which allows this (ie periodic screen capture). If you are familiar with scripting (bash,python,perl), this is even easier to achieve.

To record the desktop, there are many possibilities, istanbull, XVidCap, make sure to configure them correctly or you will have hundreds of gigabytes of video saved very quickly. Add the screen capture to autostart, and let it run in the background, problem solved.

As I said, monitoring the computer is probably not the best option, Dansguardian or other similar apps allow web filtering, and can also record when certain keywords appear, and there are many very simple ways to monitor web traffic, which is a better way than keylogging/screen capturing.
