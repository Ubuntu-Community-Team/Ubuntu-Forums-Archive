### Post by MaxIBoy on 2009-12-16
Regardless of whether Ubuntu 6.06 is still supported, it is VERY out of date.

You can download a newer version of Ubuntu here: [http://releases.ubuntu.com/karmic/ubuntu-9.10-desktop-i386.iso](http://releases.ubuntu.com/karmic/ubuntu-9.10-desktop-i386.iso)

That link will let you download an ISO that does work with unetbootin. You can install unetbootin on Windows or on Linux as well. Finally, that image can fit on a normal CD and it will work on a DVD as well. You have many options for ways to install it.



Installing programs on Ubuntu is different than in Windows. This page can help you understand how it works:
[https://help.ubuntu.com/community/Repositories](https://help.ubuntu.com/community/Repositories)
Basically, instead of looking on the Internet for stuff to install, you can install it from a single trusted and safe place called a "repository." It is very similar to the App Store if you have used an iPhone, but you don't have to pay money for anything, and there are over 20,000 programs to choose from.

In Ubuntu, the program used to look through the repositories and install things is called "synaptic." Here is a simple tutorial to synaptic:
[http://www.simplehelp.net/2007/05/03/an-introduction-to-synaptic/](http://www.simplehelp.net/2007/05/03/an-introduction-to-synaptic/)

To know all the details about managing software in Ubuntu:
[https://help.ubuntu.com/community/SoftwareManagement](https://help.ubuntu.com/community/SoftwareManagement)

If you have 6.06 you can upgrade to 8.04.3 LTS without problem.

For your sound/flash issues:

wget [http://www.zoef-pc.nl/zooi/multimedia.sh](http://www.zoef-pc.nl/zooi/multimedia.sh)
chmod +x multimedia.sh
sudo ./multimedia.sh

This script will add the medibuntu repositories to your sources.list and install some packages.

You don't need the latest Ubuntu to get your machine a bit more up-to-date. And btw, Dapper is not outdated, it uses older software which works. New != Better.
