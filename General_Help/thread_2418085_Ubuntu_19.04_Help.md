### Post by Dennis N on 2019-05-01
Mounted or connected devices don't display on the Desktop as they did previously. I use the "Places" gnome-shell extension* to easily access a remote network storage and more.

What will display on the desktop: icons for files in your Desktop folder; icons for Trash Can and/or Personal Folder  (if activated in the Desktop icons extension). At this time, turning off the extension with it's little switch doesn't prevent Desktop folder items from displaying on Desktop; use its config dialog settings to activate the others.

*shows as "Places" on the panel, but the extension name is "Places status indicator".
