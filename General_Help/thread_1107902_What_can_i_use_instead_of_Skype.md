### Post by jespdj on 2009-03-27
If you're looking for a replacement for Skype that can use your Skype account and can talk to other people who use Skype: There is none!

Skype uses a proprietary (secret, encrypted) protocol, and the software from Skype itself is the only software that works with it.

There is Skype for Linux however, and the easiest way to install it is via the Medibuntu repository (see Jgonick's link).
