### Post by mcduck on 2006-01-01
If you right-click on panel you get options to delete the panel, create a new panel and add applications and stuff to panels. You can move panels and things around with middle mouse button (or right-click and select 'move', make sure that 'lock to panel' is not selected.) You can also set panel's size and transparency etc. if you right-click it and select 'properties'.

So just move everything to the bottom panel and delete the upper panel.

### Post by ardchoille on 2006-01-01
Go to the panel you want to delete. Right-click each applet and drag it to the panel you want to keep (you can drag and drop between panels). When the panel you want to delete is empty, simply right-click it and choose "Delete this panel". If you later decide you want the deleted panel back, simply right-click an existing panel, choose "New Panel" and then populate the new panel with the applets you want.
