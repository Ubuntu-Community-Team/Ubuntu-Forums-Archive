### Post by carcdrcdr on 2008-07-10
Some drives can append data to CD-Rs while others cannot.  Check your drives documentation to see if it can.

I recommend using CD-RWs if you plan to keep adding/changing data to a CD.

You may be interested in using k3b to manage your CD projects. To install:
```
sudo apt-get install k3b
```

Or use the package manager of your choice ;)
