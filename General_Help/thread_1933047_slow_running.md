### Post by Double.J on 2012-02-28
Hi there!

These things usually happen because of packages that get added over time; it may not be just one, but could be caused by several packages. 

I'd advise the first step to be opening up system monitor and have a look at what's using your resources - both amount of RAM and the load on the CPU.

Sometimes new packages can just be buggy and slow - is there any pattern of whej it's slowing down? is a certain browser/media play/office program in use?

As for a hardware cause of slowwing down, by far the most common is a failing hard drive. The symptoms of this are slow boot time, slow  to load applications and save files, but generally pretty much as usual once something is loaded, however, first things first, check what load is on the system.

Hope it helps
