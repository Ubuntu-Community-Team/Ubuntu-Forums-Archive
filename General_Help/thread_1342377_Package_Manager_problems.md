### Post by michy99 on 2009-11-30
Go to System->Administration->Software Sources and click on the Other Software tab. make sure the partner repositories are enabled.

### Post by mkvnmtr on 2009-11-30
Just a hint. Nearly every thing you need is in the repositories or in a repository you can enable. It is very rare to need to download something from the internet. If you are using a repository you will get updates but stuff from the internet will need to be uninstalled and replaced with the update.
