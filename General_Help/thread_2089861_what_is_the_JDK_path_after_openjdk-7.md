### Post by emonti on 2012-11-30
I'm attempting to install Spring Tool Suite, after I used unbuntu software center to install openjdk-7 jre, icedtea and openjdk-7 jdk.

The installation wizard needs a jdk path:

'The application needs a JDK. A Java Runtime Environment (JRE) will not be sufficient. Select the JDK path: '

What is the JDK path?

---

### Post by emonti on 2012-11-30
I took a guess and went with this:

/usr/lib/jvm/java-7-openjdk-amd64

It seems to have satisfied that particular requirement.
