### Post by louis_nichols on 2006-05-12
There is a configuration setting for this in the Configuration Editor, under Applications>System Tools.

Expand Apps, then go to Nautilus and you will find it. I don't remember the exact name, but it will be obvious. Something like "show volumes on desktop". Just uncheck that.
