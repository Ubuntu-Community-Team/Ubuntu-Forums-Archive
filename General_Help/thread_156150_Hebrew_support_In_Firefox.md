### Post by ekarni on 2006-04-06
In Firefox, Edit menu -> Preferences, select the "General" icon at the top left of the box, then the "Languages" button.  Use the pulldown to select Hebrew, and click add.  I don't think Firefox support depends on Ubuntu system-wide Hebrew support.
