In Ubuntu 7.10, there was a compiz plugin for window previews: when the mouse hovered over an entry in the bottom panel, a thumbnail of the windows' contents would appear. ### Post by Melk79 on 2008-06-13
You need to install the CompizConfig Settings Manager (CCSM) from Synaptic, if you haven't done so.  The setting is the same, "Window Previews," and it works great.
