### Post by prodigy_ on 2010-03-16
No USB? Sounds like a motherboard problem to me, but it's not necessarily BIOS. (Southbridge perhaps?)

Can't see how wiping CMOS and re-flashing BIOS can hurt you more when pretty much everything is already broken. But I'd start with making a backup of all data. And if BIOS update won't help, then it's service center time.
