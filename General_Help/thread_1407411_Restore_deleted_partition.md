### Post by pbrane on 2010-02-15
The best way to restore the partition is to re-create the partition with the EXACT same parameters as the original. Do not format. I find that it can be useful to have a copy of the partition data printed out to refer to.

If you only had one partition to begin with then it may be a simple fix using *fdisk* or *gparted*.
