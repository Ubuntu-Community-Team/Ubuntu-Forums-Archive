---

### Post by buntudawg on 2011-07-30
Not too hard to find.

Put:
```
alignment middle_middle
```in your configuration file.

See here:
[http://ubuntuforums.org/showthread.php?t=867076&highlight=start+conky+boot](http://ubuntuforums.org/showthread.php?t=867076&highlight=start+conky+boot)
on setting up conky to start at boot amongst other things.

Check here:
[http://conky.sourceforge.net/config_settings.html](http://conky.sourceforge.net/config_settings.html)
for some configuration settings.
