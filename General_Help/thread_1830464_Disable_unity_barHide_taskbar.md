### Post by phosphide on 2011-08-21
By using Classic, the natty taskbar isn't available. 

You could change the settings to autohide:

1. Go to the Ubuntu Control Center. (Hit Windows Logo and then search System Settings)
2. Click CompizConfig Settings Manager (Should be on the second row far right)
3. Under Desktop, click the Ubuntu Unity Plugin.
4. Then change the settings for Hide Launcher to the Autohide option.
