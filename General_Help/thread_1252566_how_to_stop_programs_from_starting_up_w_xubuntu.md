### Post by running_rabbit07 on 2009-08-29
> **keastes said:**
> i am running xubuntu 9.04 on on an old pentium 3 machine with 256 MiB of ram some how i got it set so xchat, Firefox, and terminal start with xubuntu. my question is how do i turn them off?

Go to system>Preferences>Startup Applications and find those programs and un check them. Also click on the options tab and uncheck the Automatically remeber running programs box if it is checked.
