[SIZE="4"]Linux Alternatives to Windows Software[/SIZE]

Welcome, this is the Linux Alternatives to Windows Software guide. In this guide you will find Windows software placed side by side with their Linux alternatives to give you a good idea of what kind of software Linux offers and to allow you to get familiar with the different kinda of Linux software that is available.

Linux applications are rapidly evolving so i will be changing and updating this guide periodically to ensure that it is up to date. If you find any incorrect information, spelling mistakes, dead links, or would like to suggest more information, more links or more sections then go ahead and PM me about it. If anyone has any questions or problems related to Linux software you can post it here if you feel so inclined. This guide is not intended to provide you with everything you need to know about Linux software, but rather a solid foundation on which you can build upon.

Some other good sources for information on Linux Alternatives to Windows Software is the [Ubuntu Guide Alternatives list]("http://ubuntuguide.org/wiki/Alternatives"), [Linux App Finder Alternatives]("http://linuxappfinder.com/alternatives"), and the [Open Source Alternatives]("http://www.osalt.com/") website. This guide focuses on the most used desktop applications which means that applications for KDE and other desktop environments may be excluded.

**_Contribution Format:_**

> **Category:**
Windows Software to [Linux Alternative(s)]("Linux_Alternative(s)_Website_Link") - Description of Linux Alternative(s).



**_[COLOR="Blue"][SIZE="4"]Internet[/SIZE][/COLOR]_**

**Web Browsers:**
Internet Explorer to [Firefox]("http://www.mozilla.com/firefox/"), [Opera]("http://www.opera.com/"), or [Flock]("http://flock.com/") - Firefox is the most used Linux web browser, used by default in most distributions. It's fast, secure, and has many extensions and themes. Opera is second place but it is still an amazing web browser, known for having thing that Firefox has through extensions out of the box such as Speed Dial and User Agent Switcher. Flock is a social web browser based off of Firefox so it inherits its speed and security while integrating social networking and media services such as Facebook, YouTube, Twitter, Flickr, Blogger, Gmail, Yahoo Mail, etc.

**Communication**
AIM, MSN or Yahoo to [Pidgin]("http://www.pidgin.im/"), or [aMSN]("http://www.amsn-project.net/") - Pidgin (formerly named Gaim) is a multi-platform instant messaging client. The software supports many commonly used instant messaging protocols (such as AIM/MSN/Yahoo/MyspaceIM, etc.), allowing the user to log into various different services from one application. aMSN is a free and open source MSN Messenger clone with all the features of MSN without needing Windows.

mIRC to [XChat]("http://www.xchat.org/") - XChat has all the features of mIRC without the price, the IRC client of choice for Linux users.

Microsoft Outlook to [Thunderbird]("http://www.mozilla.com/thunderbird/"), or [Evolution]("http://www.gnome.org/projects/evolution/") - Thunderbird is a powerful and easy to use application to organize, secure and customize your mail from the people who brought you Firefox. Evolution provides integrated mail, address book and calendaring functionality to users of the GNOME desktop.




**_[COLOR="Red"][SIZE="4"]Media[/SIZE][/COLOR]_**

**Music Players:**
Windows Media Player or iTunes to [Rhythmbox]("http://www.gnome.org/projects/rhythmbox/"), [Banshee]("http://banshee-project.org/"), or [Amarok]("http://amarok.kde.org/") - Rhythmbox is an audio player that plays and helps organize digital music. Originally inspired by iTunes, it is free software, designed to work well under the GNOME Desktop using the GStreamer media framework. Banshee can play, import, and burn audio CDs, and supports many portable media players, including iPod and Creative's ZEN players. Music stored on the iPod can be played without synchronization, and album art stored in the Banshee library is transferred to the iPod. Amarok's tag line is "Rediscover Your Music", and its development is based around this ideology. Amarok's core features such as the unique "context browser", integrated Wikipedia lookup and lyrics download help users to find new music, and to learn more about the music they have. Amarok also features integration with last.fm, giving users suggestions about what to listen to next and which artists may fit their mood, as well as with Magnatune integration, allowing no-cost full listening of all the music in their catalog, and DRM-free purchasing.

**Video Players:**
Windows Media Player to [Mplayer]("http://www.mplayerhq.hu/design7/news.html")/[SMPlayer]("http://smplayer.sourceforge.net/"), or [VLC]("http://www.videolan.org/vlc/") - MPlayer is a powerful video player for Linux that plays virtually every media file type you'll ever want to use. SMPlayer is a great front-end for MPlayer that makes everything a lot easier and adds a lot of nice features too. One of the most interesting features of SMPlayer is that it remembers the settings of all files you play. Say you start to watch a movie but you have to leave... don't worry, when you open that movie again it will resume at the same point you left it, and with the same settings: audio track, subtitles, volume and everything else. VLC is another video player that plays practically every file you would ever need but most people choose MPlayer/SMPlayer over VLC because it doesn't handle styled soft subtitles very well.




**_[COLOR="Sienna"][SIZE="4"]Safety[/SIZE][/COLOR]_**

**Virus protection/Firewall/Anti-Spyware Protection:**
[B][COLOR="Red"]
NOTE: You don't need virus protection unless you're sharing files with a Windows computer, and you don't need a firewall unless you're are setting up a server.[/COLOR][/B]

Norton or McAfee VirusScan to [ClamAV]("http://www.clamav.net/") - Under Linux, it is most likely that ClamAV will run in daemon mode, servicing requests to scan files sent from other processes. These can include mail exchange programs, files on Samba shares, or packets of data passing through a proxy server (IPCop, for example, has an add-on called Copfilter which scans incoming packets for malicious data).

Comodo Firewall Pro, Outpost Firewall, or Zone Alarm to [Firestarter]("http://www.fs-security.com/") or [Guarddog]("http://www.simonzone.com/software/guarddog/") - Firestarter is a GTK/GNOME desktop environment application, while Guarddog is a KDE desktop environment application. While these applications in and of themselves are not direct replacements for the Windows programs mentioned, they are GUI front ends for iptables, which is the actual firewall system that comes standard in Linux distributions. These GUI front ends make configuring and utilizing iptables much easier and convenient for the average user.




**_[COLOR="Green"][SIZE="4"]System Tools[/SIZE][/COLOR]_**

**Screen Recording:**
CamStudio to [recordMyDesktop]("http://recordmydesktop.iovar.org/about.php"), [Istanbul]("http://live.gnome.org/Istanbul"), or [XVidCap]("http://en.wikipedia.org/wiki/XVidCap") - recordMyDesktop, Istanbul, and XVidCap all capture your screen and do so even when using desktop composition.

**Application Launchers:**
Launchy to [GNOME Do]("http://do.davebsd.com/") - GNOME Do allows you to quickly search for many items present in your GNOME desktop environment and perform useful actions on those items. GNOME Do is inspired by Quicksilver  and GNOME Launch Box.

**Application Docks:**
ObjectDock or RocketDock to [Avant Window Navigator]("https://launchpad.net/awn/"), or [CairoDock]("https://help.ubuntu.com/community/CairoDock") - Avant Windows Navigator is a very customizable application dock for Linux with features such as support for launchers, task lists, and third party applets. Cairo-dock is an animated application launch bar for the desktop comparable to Mac OS X's Dock, or Rocket Dock. It is compatible with Compiz-Fusion, Beryl, Compiz, and also Xcompmgr, but it can run without a composite manager (with fake transparency).

**Disk/Image Ripping/Burning:**
DVD Shrink to [K9Copy]("http://k9copy.sourceforge.net/") - K9Copy is a small utility which allows copying DVD's on Linux.




**_[COLOR="Purple"][SIZE="4"]Productivity[/SIZE][/COLOR]_**

**Office:**
Microsoft Office to [OpenOffice]("http://www.openoffice.org/") - OpenOffice.org is a powerful open source alternative to Microsoft Office because it includes alternatives to all of the Microsoft Office applications while providing room for extensions to be built that add functionality such as PDF editing and additional spelling dictionaries.

**Image Editors:**
Adobe Photoshop to [GIMP]("http://www.gimp.org/"), or [Inkscape]("http://www.inkscape.org/") - The GNU Image Manipulation Program, or GIMP, is a raster graphics editor used to process digital graphics and photographs. Typical uses include creating graphics and logos, resizing and cropping photos, altering colors, combining multiple images, removing unwanted image features, and converting between different image formats. GIMP can also be used to create basic animated images in GIF format. It is often used as a free software replacement for Adobe Photoshop, the most widely used bitmap editor in the printing and graphics industries; however, it is not designed to be a Photoshop clone. Inkscape is an Open Source vector graphics editor, with capabilities similar to Illustrator, CorelDraw, or Xara X, using the W3C standard Scalable Vector Graphics (SVG) file format. Inkscape supports many advanced SVG features (markers, clones, alpha blending, etc.) and great care is taken in designing a streamlined interface. It is very easy to edit nodes, perform complex path operations, trace bitmaps and much more. We also aim to maintain a thriving user and developer community by using open, community-oriented development.

**Video Editing:**
VirtualDub to [Kino]("http://kinodv.org/") - Kino is a non-linear DV editor for GNU/Linux. It features excellent integration with IEEE-1394 for capture, VTR control, and recording back to the camera. It captures video to disk in Raw DV and AVI format, in both type-1 DV and type-2 DV (separate audio stream) encodings. You can load multiple video clips, cut and paste portions of video/audio, and save it to an edit decision list (SMIL XML format). Most edit and navigation commands are mapped to equivalent vi key commands. Also, Kino can export the composite movie in a number of formats: DV over IEEE 1394, Raw DV, DV AVI, still frames, WAV, MP3, Ogg Vorbis, MPEG-1, MPEG-2, and MPEG-4. Still frame import and export uses gdk-pixbuf, which has support for BMG, GIF, JPEG, PNG, PPM, SVG, Targa, TIFF, and XPM. MP3 requires lame. Ogg Vorbis requires oggenc. MPEG-1 and MPEG-2 require mjpegtools or ffmpeg. MPEG-4 requires ffmpeg.

**Text Editors:**
NotePad to [gedit]("http://www.gnome.org/projects/gedit/") - gedit is a simple and easy to use feature full Linux alternative to NotePad. It has handy features such as tabs and code highlighting.
