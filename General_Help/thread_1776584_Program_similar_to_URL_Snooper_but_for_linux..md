[http://en.wikipedia.org/wiki/URL_Snooper](http://en.wikipedia.org/wiki/URL_Snooper)

**URL Snooper** is a [Windows]("http://en.wikipedia.org/wiki/Microsoft_Windows") program to find [URLs]("http://en.wikipedia.org/wiki/Uniform_Resource_Locator") of [streaming media]("http://en.wikipedia.org/wiki/Streaming_media") and [data]("http://en.wikipedia.org/wiki/Data_stream"). This allows streamed files download through every [download manager]("http://en.wikipedia.org/wiki/Download_manager"). Its scope is the same as that of a [stream recorder]("http://en.wikipedia.org/wiki/Stream_recorder"). It uses [Winpcap]("http://en.wikipedia.org/wiki/Pcap") for packet capturing.
