### Post by Fingers &amp; Thumbs on 2008-06-16
I'm a little unsure as to how you have firefox set up.
Go to >Edit >Preferences, select the Content tab and you will see a button named "Colors" clicking this will bring up another window, you should be sure that there is a tick in the check-box that says "Allow pages to choose their own colors" as using your own selection doesn't work for all sites, I'm not sure why, I guess it depends on what type of code the site uses. Perhaps Greasemonkey would allow you to manipulate such sites to your preference.
