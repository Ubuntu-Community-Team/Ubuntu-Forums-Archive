### Post by ticket on 2012-04-22
gtkpod 2.1.0 works for me, on Ubuntu 11.10, using ipod nano 4th Generation.

Just use the "Add files to selected ipod" icon (from the icon menu bar).
I was able to select an .mp3 file and it was copied to the ipod.

I think you then have to hit the "save all changes" icon after adding the file(s).

The ipod was able to play the mp3 without problems.
