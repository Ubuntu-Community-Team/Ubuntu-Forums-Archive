I would like to scan films with Epson v600. [Image Scan for Linux by Avasys](http://www.avasys.jp/lx-bin2/linux_e/scan/DL2.do) works, but doesn't support Digital Ice (at least I can't find this option).

Is VueScan the only software for Linux supporting Digital Ice?

---

### Post by myoldhouse on 2011-04-04
In a word 'yes'. Vuescan is the only scanning software for Linux that supports Digital ICE (it is called Infrared in the program).

An alternative, although cumbersome, is to run Windoze in a virtual machine, under VirtualBox, on Ubuntu and use the Epson software that ships with the scanner there. I have done this in the past and it does work. 

I now just use Vuescan Pro (x64 version) directly on Ubuntu 10.10. I feel it was money well spent for the lifetime of updates. You can test it out for free to see if it suits you.
