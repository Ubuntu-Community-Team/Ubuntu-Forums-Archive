### Post by Tom--d on 2008-08-03
You can disable the shadows.

You need to have ccsm tho.
```
sudo apt-get intall compizconfig-settings-manager
```

Go to:
System > Pref > Advanced Desktop Effects.....

Then go to Window Decoration. 
Turn off shadows.

done :)
