### Post by worksofcraft on 2010-08-06
No I don't think that can happen.

Ubuntu is quite secure against viruses because most running software need explicit superuser permission to modify executables.

My understanding is that infections on Linux, if any, are in the form of Trojans and targeted mostly at servers by criminals who hope to run botnets.
