### Post by darolu on 2010-02-10
You can install Thunar (xfce's) or Dolphin/Konqueror (kde's) -dolphin being the newest-; I can't tell if they are "better" or not since it is a matter of taste, I personally like Nautilus more, but Thunar is great too, I can't give an opinion of Dolphin since I am not very fond to KDE but I suppose it works fine too.
You can always use the linux terminal to manage all your files.

You can install with:
```
$ sudo apt-get install thunar

$ sudo apt-get install dolphin
```

Or via Synaptic.
