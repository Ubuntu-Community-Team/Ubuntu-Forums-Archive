> **TheNessus said:**

W: GPG error: [http://ppa.launchpad.net](http://ppa.launchpad.net) intrepid Release: The following signatures couldn't be verified because the public key is not available: NO_PUBKEY 632D16BB0C713DA6

That means you do not have a key on your computer that verifies the files being downloaded. You still should be able to update by ignoring those errors. If you have security concerns than you should install the applicable keys for  [http://ppa.launchpad.net]("http://ppa.launchpad.net/")

Google it or search this forum on how to add the signature keys. If you still cant get it after that then we can help.
