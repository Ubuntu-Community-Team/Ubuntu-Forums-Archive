I tried this and it works, but it does not include messing with **autostart** file, also it does not start program **minimized**:
```
Open the file manager (by default PCManFM), go to /usr/share/applications. 
Right-click on the program you want to be added to your autostart programs, and click Copy. 
Now, go to ~/.config/autostart in PCManFM, right-click and click Paste. 
The program should be added to your autostart programs now, but double-check by clicking the LXMenu down left, click Settings, and then Desktop Session Settings.
You may have to create the autostart folder.
```

### Post by midnightramen on 2013-05-11
You may want to take a look here : [https://help.ubuntu.com/community/AddingProgramToSessionStartup](https://help.ubuntu.com/community/AddingProgramToSessionStartup) . I think it is a slight variation of what you may have already tried, but I think it is the best way to go for starting a program within a Gnome or Unity session . There are other ways for items to start on startup, placing the path of the program in /etc/init.d/rc.local for instance, but I don't think that is the path you are hoping to go with.
