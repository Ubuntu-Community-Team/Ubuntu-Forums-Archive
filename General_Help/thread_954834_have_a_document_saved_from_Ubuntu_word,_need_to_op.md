### Post by silverglade00 on 2008-10-21
The problem is that Ubuntu does not use Microsoft Word. It uses OpenOffice.org as the office suite. There are two ways you can deal with this. The first is to open it back up in Ubuntu and save it again as a .DOC file in Save As. The other thing to do is to go to [www.openoffice.org](www.openoffice.org) and download the free Windows version. You will be able to open it with no issues at all!

If this is a document that you need to share with others, I suggest saving as a .doc file. Otherwise, install the other suite.
