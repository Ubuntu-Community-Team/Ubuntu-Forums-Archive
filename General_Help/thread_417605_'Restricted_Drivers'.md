### Post by mhansen on 2007-04-21
It means the drivers are closed source (you get a binary), so the Ubuntu developers are at the mercy of the respective companies that release that code to make sure it works.  They can't make changes themselves...they can't even look at the code (well...technically) and they sure can't make changes to fix anything that's broken.  So therefore, they can't support it.
