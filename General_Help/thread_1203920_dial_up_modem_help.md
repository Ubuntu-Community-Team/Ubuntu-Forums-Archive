### Post by _duncan_ on 2009-07-04
Hi. I don't have any experience with your specific modem. But looking at terminal.txt, it seems the libraries needed for compiling are not installed in your system. These libraries are available from the live CD.

Just insert the live CD. Chances are, a dialog will appear asking if you want to use it as a package source (or something like that, can't remember the exact wording). Then install the package 'build-essential'.

Try compiling again.
