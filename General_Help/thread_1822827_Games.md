### Post by Grenage on 2011-08-11
If they are Windows games (and most games are), then you can try running them with WINE/PlayOnLinux.  Failing that, you'll need to either dual boot, or just stick with Windows.

There are some great games with a Linux client, but most mainstream games are only designed for Windows and consoles.
