### Post by kpkeerthi on 2009-05-04
Try installing from command line
```
sudo apt-get install nvidia-glx-180
```
... and the enable driver using
```
sudo nvidia-xconfig
``` ... and **Reboot**.

You can install nvidia-glx-173, if nvidia-glx-180 is troublesome for you.
