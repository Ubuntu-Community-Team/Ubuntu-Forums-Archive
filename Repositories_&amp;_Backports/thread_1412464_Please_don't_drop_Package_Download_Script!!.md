### Post by Bartender on 2010-02-21
The Package Download Script (PDS) tool, available in Synaptic, has been an absolute life-saver in my quest to make dial-up at home and Ubuntu workable.  I create a PDS from the dial-up desktop at home, then go into town with the laptop and get all the updates.

Without PDS I'm screwed.

I've been reading about how Synaptic, Update Manager, etc. will all be rolled into one tool.  Pleasepleaseplease include PDS in that new tool somewhere.  If it disappears I'll have to go back to Windows and that would suck.
