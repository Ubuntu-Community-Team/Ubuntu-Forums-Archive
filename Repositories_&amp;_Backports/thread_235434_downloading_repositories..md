### Post by SgtDude on 2006-08-19
This thread has some good information.

[http://www.ubuntuforums.org/showthread.php?t=233325](http://www.ubuntuforums.org/showthread.php?t=233325)

Basically the easiest way to do it is to order the DVD's from someone (there's a link in the linked thread above to a site that sells them), copy the files into a directory structure similar to the ones on the official archives, and then use debmirror to mirror the rest of the files (specifically dapper-update(s?) and dapper-security, or whatever distro you want).

I've done this with the Breezy distro, but I'm moving soon, and the machine with the mirror is currently being shipped.  Once I am reunited with my machine I'll post a more detailed how-to and probably build an entry in Ubuntu's wiki.  But if you're motivated, it's not too terribly difficult to knock out.
