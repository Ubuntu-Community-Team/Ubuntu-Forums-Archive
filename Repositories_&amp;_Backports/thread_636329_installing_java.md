### Post by ijason on 2007-12-09
man, i had a heck of a time getting java to work right on my machine too.

what i ended up doing was removing ALL java installed on the machine as listed in the System>admin>Synaptic     , then rebooted to make sure i hadn't missed anything.  then i downloaded (also through synaptic) the 'nonfree' java package. just do a search for 'java nonfree' to get the right one. 

it will install itself properly through synaptic. you should be able to get all the plug-ins you need from firefox after that. good luck!
