### Post by Sutekh on 2006-04-26
As far as I can tell, it won't be any different than just simply using the installer to do everything.

It is certainly not madness!  :)

I think a positive point is that you can completely build your Ubuntu system up the way you want it.  

Forget GNOME/KDE, I like to do the base install and then install some sort of light-weight window manager, and other programs I need... and nothing I don't need.  Runs incredibly fast.

A negative point is that it may be very hard to do this when new to Ubuntu/Linux.   I still come across things I need to install, that were part of the Ubuntu (GNOME) install.  That can be a pain.


I think the only reason I would suggest this method was if you downloaded the Ubuntu Install CD but now want Kubuntu installed instead, and not have to waste time installing GNOME.  Then I would do the base install and install Kubuntu from the internet repositories.

Does that help?

Installing the Ubuntu desktop after a base install is super easy, its pretty much one command that will install everything

By using this command you are doing what a normal installation does, installing everything.

### Post by Sutekh on 2006-04-27
I am currently experimenting with different interfaces before Dapper is released, so I can decide what I will use.

Currently I use **OpenBox** by itself.  I have an AMD X2 3800 with 1GB of RAM.  It runs ridiculously fast.  I have spent a lot of time customising it too, and it runs just the way I like.  

I really like **XFCE** (Xubuntu), it was quick and the plus is that its a pretty full environment, as Openbox is just a window manager.  I am also thinking of installing **Enlightenment 17** soon to give it a shot.

I've also tried **XGL/Compiz with GNOME**, which is very cool, so its going to be very difficult to choose what to use in the future.

I've tried **Fluxbox**, but didn't really get into it.  **IceWM** I haven't tried yet.


If I were you, I'd try them all out before you start customising Ubuntu and ammassing lots of data.  If you find one you like, then stick to and get rid of the others.  You might find it tough-going though, especially if you have tricky hardware (wireless, etc).  I think that's half the fun though, and the community here is always willing to help.

This s a good site to start at, if you want to check out some different interfaces, with links to the official sites, and screenshots

[Window Managers for X](http://xwinman.org/)
