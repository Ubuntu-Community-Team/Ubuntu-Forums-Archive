### Post by runenes on 2005-01-17
All three packeges are in [universe](http://www.ubuntulinux.org/wiki/UniversePackages/view?searchterm=universe) . 

you can use the commands
```

$ apt-cache search REG-EXP # to search for a package
$ apt-cache showpkg PACKAGENAME # to show info about a package

```

See 'man apt-cache' for more, and see the link above for how to add universe (but be warned, it's unsupported).
