### Post by tinny on 2008-07-30
NetBeans 6.1 is too new for the Hardy repositories (the Ubuntu team needs time to test etc).

NetBeans 6.1 is very easy to install / uninstall on Linux. (comes with both install and uninstall scripts, this is how I use it).

[http://download.netbeans.org/netbeans/6.1/final/](http://download.netbeans.org/netbeans/6.1/final/)

FYI: The NetBeans platform that the NetBeans IDE is built on has an internal update mechanism built in, so the internal modules that make up the IDE will be updated anyway.

Also, it looks like 6.0.1 is the version that is going to go into Ubuntu intrepid.

Basically I wouldn't worry about it if I where you, 6.0.1 is fine!
