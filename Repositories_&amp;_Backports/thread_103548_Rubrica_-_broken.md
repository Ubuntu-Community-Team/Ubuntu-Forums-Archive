### Post by Nkosi on 2007-07-04
Two years later to your posting if it still applies...
I installed Rubrica with some initial errors, but it's working fine now. I had to change the default name from "rubrica.rub" to "somethingnew.rub" and set the default path for Rubrica to find my new file, also set in Rubrica's Preferences.  I prefer Dlume, another even simpler address book, but it doesn't import as far I know. 

Rubrica has a few bugs, but it is amazing simple, fast, small and straightforward. Good luck.
