### Post by Golovko on 2004-12-12
I came from Debian to Ubuntu (and I love it). I am used to using dselect as my main package management tool. My problem right now is that dselect on ubuntu will only show installed packages. I can do apt-get install *packagename*, however I can't use dselect to search for an uninstalled package and set for install.

Is this in fact that way Ubuntu is set up with dselect, does anyone know of a way to change this? I guess I could use the synaptic package manager, but I like dselect and I am used to it.

Gol

### Post by Golovko on 2004-12-12
Well google turned it up, either *dselect update* or choose update from the dselect menu. I had only been using *apt-get update* which in the version of Debian I ran prior, updates packages listed in dselect as well.
