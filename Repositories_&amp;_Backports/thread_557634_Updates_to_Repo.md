### Post by 5-HT on 2007-09-23
Save for either security fixes or major bug fixes (which may just be patches and not version upgrades), updates will only be provided for the next Ubuntu version barring backports.
There have been some exceptions, but not often. 

Perhaps a rolling-release distro is better suited for your needs if you'd like constant updates once a new upstream package is released?
