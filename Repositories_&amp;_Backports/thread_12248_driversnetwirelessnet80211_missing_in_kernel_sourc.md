### Post by Paha Pukki on 2005-01-23
Hi -

I have built and installed the Ubuntu Linux 2.6.8.1 kernel but can't get my wireless network adapter to start with it. 

I checked the differences between my kernel and the standard Ubuntu kernel (2.6.8.1-3-386) and I found that the standard kernel has as /lib/modules/2.6.8.1-3-386/kernel/drivers/net/wireless/net80211 directory, which is missing from my kernel. 
Then I found that the Ubuntu linux-source-2.6.8.1 package I used for building my kernel does not have a drivers/net/wireless/net80211 source directory. 

Is this missing somehow or do I need to do something else to get the net80211 kernel modules built?

---

### Post by Paha Pukki on 2005-01-23
Ah well, nevermind. The problem was caused by the absence of two firmware files for my D-Link/TI ACX100 wireless network card in the /lib/hotplub/firmware directory. After copying them from the original Ubuntu kernel everything worked. I guess I have to figure out now why they were not automatically generated by the kernel build...

---

### Post by Paha Pukki on 2005-01-23
OK, found them. Both the ACX100 firmware files and the net80211 kernel modules are in the linux-restricted-modules-2.6.8.1-3-386 package.
