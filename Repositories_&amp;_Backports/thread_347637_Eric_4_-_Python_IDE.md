### Post by MWeather on 2008-01-02
That's for eric3. for eric4 you need to download eric4 from the project's website and install qscintilla2. It's not in the repos, though.

So download it from debian: 

[http://packages.debian.org/sid/libqscintilla2-3](http://packages.debian.org/sid/libqscintilla2-3)
[http://packages.debian.org/sid/python-qscintilla2](http://packages.debian.org/sid/python-qscintilla2)

After that it will compile just fine.
