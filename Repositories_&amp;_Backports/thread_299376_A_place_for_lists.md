### Post by HAARP on 2006-11-15
It would still be possible for evil-minded people to sneak into this list with "normal" packages and suddenly add malicious ones.

A better idea would be some sort of "packages-wiki" where everyone can add packages, list their files, test them, and mark them as ok/broken/malicious.
Or more like a digg-like site that promotes packages that have been voted good enough to a front-page, and let's them stay there.

Of course this wouldn't work with the larger repos with many packages that pose the same purpose (e.g. Doomsday / Yagisan's repos or kubuntu.org's latest stuff). It's more of an idea for a repository that carries all those small different packages.
