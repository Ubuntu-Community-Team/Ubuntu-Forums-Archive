[/QUOTE]It first has to make it into the current development branch (Dapper) and once it's there, if there are backport requests, it can be backported. Take a look at the guidelines on the backports forum and if these apps qualify, place backport requests for them. If accepted, they'll become available in the backports repository a while later.
