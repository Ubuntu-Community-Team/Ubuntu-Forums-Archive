### Post by howefield on 2016-10-17
16.04.2 would likely have the Yakkety kernel but not sure when it is due for release, you'll be able to add the Hardware Enablement Stack via apt or your normal method of installing software, ie you won't need a repo.

Bear in mind that only 16.04.1 and 16.04.5 is supported till 2021 with regards the HWE.

[https://wiki.ubuntu.com/Kernel/LTSEnablementStack](https://wiki.ubuntu.com/Kernel/LTSEnablementStack)
