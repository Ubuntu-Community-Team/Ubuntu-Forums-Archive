this thread is supposed to offer .debs for the latest **Geom**.
[Geom]("http://abulafia.fciencias.unam.mx/~canek/geom") is an interactive geometry program.

To find out more about Geom have a look at the [Screenshots]("http://abulafia.fciencias.unam.mx/~canek/geom/index.php?section=screenshots") or watch the [Screencasts]("http://abulafia.fciencias.unam.mx/~canek/geom/index.php?section=screencasts").

As Geom depends on [CCC]("http://live.gnome.org/Criawips/CriaCanvas") and there are actual no .debs for Edgy available, I will offer you both here.

You may have to run "export LD_LIBRARY_PATH=/usr/local/lib" in the terminal, before you can start "geom".

Have fun with this great software :)
