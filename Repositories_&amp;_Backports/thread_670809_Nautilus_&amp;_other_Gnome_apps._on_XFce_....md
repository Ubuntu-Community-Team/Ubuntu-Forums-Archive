### Post by someonestolemyname on 2008-01-18
I believe that Nautilus is very tightly integrated with Gnome. xfce is a completely different desktop environment. If you installed gnome, you should be able to choose Gnome Session in the sessions menu on the login screen.

To make ubuntu have gnome and all of the other default apps as in Ubuntu rather than Xubuntu, install ubuntu-desktop, then chose the session as above.
