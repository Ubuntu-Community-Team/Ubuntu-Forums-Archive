### Post by Sutekh on 2006-04-23
Try this website

[http://cargol.net/~ramon/ubuntu-dvd-en](http://cargol.net/~ramon/ubuntu-dvd-en)

You can download the entire main/universe/multiverse repositories to 3 DVD's and use them as apt-get repositories with
```
sudo apt-cdrom add
```

Just to re-iterate 3 DVDs.


There are also a variety of add on CD created by members of the community.

Such as this one

[Ubuntu Forums - We need an add-on CD](http://ubuntuforums.org/showthread.php?t=136955)

[Ubuntu Wiki - Unoficial Ubuntu 5.10 Add On CD](https://wiki.ubuntu.com/UnofficialUbuntu510AddOnCD)
