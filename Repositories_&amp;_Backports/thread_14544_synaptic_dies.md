### Post by mendicant on 2005-02-08
If you're running warty, all updates should be security updates (unless you also have other repositories in your sources.list).  Security updates also seem to have ubuntu in their name--but maybe that's just for all ubuntu specific changes made to a debian package.  If you're really curious, later versions of aptitude (not warty, unfortunately) have aptitude changelog <pkg name> from the command line as well as the command 'C' in the curses GUI.
