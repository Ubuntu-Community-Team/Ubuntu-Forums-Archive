### Post by Framli on 2010-10-11
*I'm posting this here, as it's related to repository management but it will fit more nicely in the natty testing forum when it opens.*

I test compulsively development releases of Ubuntu since Karmic Alpha2. A frequent issue I encounter, with all the repos I tend to add, is never knowing when PPA and other repos are available for a development release and being confronted to ugly 404 bugs while updating.

I've just hacked a small script to let me know when the repos I use support the release I'm testing :
[https://code.launchpad.net/~davidc3/+junk/repostory](https://code.launchpad.net/~davidc3/+junk/repostory)
```
bzr branch lp:~davidc3/+junk/repostory
```
