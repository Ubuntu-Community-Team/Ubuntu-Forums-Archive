### Post by cmost on 2007-09-17
Gnome is never backported to any Ubuntu release.  The only way to get the latest Gnome is to upgrade to the latest Ubuntu release (when it's released.)  Ubuntu almost always coincides its releases with Gnome releases.  Sure, it's possible, in theory, to install the latest Gnome in an older Ubuntu release but doing so would be so wrought with problems that it's simply easier to upgrade to the newest Ubuntu.
