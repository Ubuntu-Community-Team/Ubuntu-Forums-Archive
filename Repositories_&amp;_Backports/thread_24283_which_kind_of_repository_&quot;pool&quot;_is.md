### Post by Juergen on 2005-04-06
Pool contains ***all*** packages (for all architectures, dist-versions etc).
The dists then only 'point' to a subset of these packages.

Look here (and maybe read the whole thing ;-)):
[http://www.debian.org/doc/manuals/repository-howto/repository-howto.html#pools](http://www.debian.org/doc/manuals/repository-howto/repository-howto.html#pools)
