### Post by WW on 2007-04-06
It is in the **main** repository. One quick way to check this is to run Synaptic, and use the Search button.  In the list of packages, the orange Ubuntu icon tells you it is part of the main repository. (Packages in the **restricted** repository also get the icon, but there is nothing restricted about gstreamer.)
