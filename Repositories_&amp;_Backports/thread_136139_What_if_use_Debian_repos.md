### Post by az on 2006-02-25
The packages have the same format, but use different toolchains and library versions.  All the stuff is compatible at the source code level, but not at the binary level.  Prebuilt binary packages are made for the distribution they were built for.

There are a lot of packages in Warty that you cannot run in Breezy, for example.
