### Post by mojoman on 2006-12-28
The repositories you use are listed in /etc/apt/sources.list . To view it, simple type cat /etc/apt/sources.list and to edit it type sudo gedit /etc/apt/sources.list (use kate or nano instead of gedit if you use kubuntu or xubuntu). There are probably lines which you can comment out in order to activate access to some repositores (i.e. removing the # in front of the line). If you need to, there is a generator for sources.list so you can genarate your own and replace it (remember to make a back-up ;)  )

[http://www.ubuntu-nl.org/source-o-matic/](http://www.ubuntu-nl.org/source-o-matic/)
