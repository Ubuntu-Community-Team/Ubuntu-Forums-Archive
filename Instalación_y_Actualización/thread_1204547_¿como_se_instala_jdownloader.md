### Post by dnovai on 2009-07-04
Hola! no soy muy experto en esto de ubuntu, pero hace un tiempo también anduve tratando de instalar JDownloader. Tienes que bajar la carpeta que esta en la página principal de dicho programa (observa que sea la que es especifica para linux) y solo tienes que descomprimirla.
Luego abre un terminal y procura estar en el directorio principal de la carpeta que descomprimiste. Finalmente ejecutas en el terminal.

```
  java -jar JDownloader.jar
```

Y listo, se ejecutará el programa. Esto lo debes hacer cada vez que quieras ejecutarlo.
Espero te haya ayudado. Ojalá algún experto del foro pueda corroborar esta info o buscar otro modo de ejecutar el programa en ubuntu.
