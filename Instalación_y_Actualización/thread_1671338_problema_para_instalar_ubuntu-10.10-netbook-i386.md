### Post by RafaelG on 2011-04-19
[SIZE=2]Creo que lo que esta pidiendo es que crees el UBS stick con con el USB creator. Se supone que hay varias formas de instalar Ubuntu. Una de ellas es con Wubi que a largo plazo solo es una forma de emigrar paulatinamente desde windows ha Ubuntu.  Otra alternativa es instalar Ubuntu desde la BIO de tu maquina en una particion totalmente diferente de los demas OS (Normalmente utilizada).

La alternativa que tu preguntas es casi lo mismo que ingresar un CD con la imagen ISO (LiveCD) en tu lector para que realice la instalacion pero la diferencia es ingresar un dispositivo USB Memory para instalar la imagen ISO que se encuentra creada pero no es tan simple como copiar y pegar la imagen ISO en tu USB memory.

Para que la BIO reconosca la imagen ISO en tu USB Memory debes crear un USB stick, en el siguiente LINK podras aprender como crear lo solicitado ingresando a las siguientes alternativas:
