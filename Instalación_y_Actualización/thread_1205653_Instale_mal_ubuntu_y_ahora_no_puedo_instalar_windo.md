### Post by Carlos C on 2009-07-06
Hola. Este foro se centra en resolver problemas relacionados con Ubuntu. Si tienes problemas con Windows deberás buscar en otro foro. Comprendo que cometiste un error al instalar Ubuntu, pero instalar windows es un asunto que deberás tratar en otro lado.

Para no ser tan antipático te puedo sugerir una posible solución:  revisa tu BIOS, ya que seguramente tienes problemas al instalar windows xp porque tu disco es SATA. En general es posible modificar la manera en que se reconoce el disco, busca alguna opción parecida a "IDE mode". Con eso debieras poder instalar correctamente tu sistema microsoft. Ahora, si tienes problemas o dudas relacionadas con eso, por favor resuelvelas en otro lado, este foro es para atender consultas relacionadas con Ubuntu.

Saludos.
