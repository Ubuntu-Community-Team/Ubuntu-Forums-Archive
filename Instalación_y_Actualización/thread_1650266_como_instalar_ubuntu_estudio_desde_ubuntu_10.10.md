### Post by Carlos C on 2010-12-23
Hola. Por favor, la próxima vez que publiques en el foro recuerda hacerlo en el subforo adecuado. Cuando tengas dudas te recomiendo leer esto:

[http://ubuntuforums.org/showthread.php?t=1319475](http://ubuntuforums.org/showthread.php?t=1319475)

Respecto a lo que preguntas, Ubuntu Studio es básicamente una instalación de Ubuntu con aplicaciones orientadas a la multimedia y un kernel con baja latencia. Si eso no te dice nada te recomiendo buscar en google para que te hagas una idea. Todas las aplicaciones que están en Ubuntu Studio se pueden instalar en Ubuntu, al igual que el kernel. Sin embargo, si no tienes mucha experiencia quizás te resulte más fácil hacer una instalación limpia de Ubuntu Studio desde el cd de instalación. En realidad tienes varias opciones:


[LIST]
[*]Instalar Ubuntu Studio en una partición aparte, independiente de Ubuntu y Windows. Al iniciar el sistema escoges el sistema operativo que deseas usar.
[*]Instalar las aplicaciones de Ubuntu Studio en tu Ubuntu manteniendo todas tus configuraciones.
[*]Instalar las aplicaciones de Ubuntu Studio y además instalar el kernel de baja latencia. De esa manera escoges al iniciar el sistema el kernel adecuado y lo utilizas.
[/LIST]

Es necesario que escojas una opción antes de poderte ayudar. Si no entiendes algo de lo que te sugiero, por favor, pregunta.

Saludos.
