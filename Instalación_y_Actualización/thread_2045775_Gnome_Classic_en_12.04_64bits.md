### Post by jualin on 2012-08-21
Hola Mario,
Lo de 32 bits vs 64 bits depende en tu procesador. Si tienes un procesador de 64 bits puedes usar cualquiera (64 bits/32 bits). Uno de los objectivos de tener un sistema operativo de 64 bits es poder utilizar mas the 2^32 bits de RAM (4 GB). Con el de 64 bits el RAM es practicamente infinito 2^64 bits. 
Entonces, si tienes mas the 4 GB de RAM (y los quieres usar) entonces instala el de 64 bits. Si encuentras muchos problemas con el, trata el de 32 bits. 
Saludos desde Miami :)
