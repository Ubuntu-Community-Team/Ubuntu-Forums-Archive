### Post by bichopro on 2010-04-14
Autocad lo puedes usar desde la web y no tendrias para que haber instalado wintendo:
[http://butterfly.autodesk.com/](http://butterfly.autodesk.com/)

Si usas karmic koala o anterior descarga super grub disc: [http://www.supergrubdisk.org/index.php?pid=5](http://www.supergrubdisk.org/index.php?pid=5)

Baja la iso copiala y dale siguiente, siguiente , siguiente. Es muy facil e intuitivo.
Tambien puedes descargar un exe para instalar en windows y recuperarlo desde ahi: [http://www.supergrubdisk.org/index.php?pid=12](http://www.supergrubdisk.org/index.php?pid=12)

Tambien te puede servir esto: [http://www.pagondel.org/how-to-recuperar-grub2/](http://www.pagondel.org/how-to-recuperar-grub2/)
