### Post by jorval on 2009-07-01
Hola amigos, la página que indico es una descripción del método general para instalar programas en Mandriva Linux compilándolos desde el código fuente. Lo que encuentro excelente de esa guía es que explica con detalles en qué consiste la instalación de programas construyéndolos desde su código fuente. Su lectura te aclarará temas como:
1.- En que consisten los tres pasos que siempre se mencionan:
./configure ; make y make install
2.- Por qué algunos programas se saltan el primer paso ./configure
3.- La importancia de mantener el código fuente disponible después que se ha instalado con éxito la aplicación. Recomienda donde mantenerlo en Mandriva y en Ubuntu ¿dónde?
4.- Cómo desinstalar un programa que ha sido compilado e instalado desde el código fuente.
5.-Qué hacer si he borrado el código fuente y deseo desinstalar el programa.

[http://wiki.mandriva.com/es/Instalar_desde_c%C3%B3digo_fuente]("http://wiki.mandriva.com/es/Instalar_desde_c%C3%B3digo_fuente")

PD. Todo esto fue a raíz de los famosos drivers para hacer funcionar la red inalámbrica con Madwifi. Estos conceptos me aclararon muchas cosas.
