Conky is an amazing, lightweight text based system monitor that is configured using a simple markup language somewhat similar to HTML.  It will dang near make you breakfast if you tell it to.  It is extensible using anything from simple bash scripts, to scripting languages like python, ruby, perl, up to high level languages like Java and C++

You can think of Conky as the Linux equivalent of Rainmeter on Windows, only far more flexible and powerful.
