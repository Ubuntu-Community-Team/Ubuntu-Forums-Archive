### Post by bouncingwilf on 2011-08-20
Usually the modem comes with built in Windows software ( which you don't need)  and as such Ubuntu "sees" it as a CD ROM or thumb drive. If the Modem gets "mounted" i.e. you can see it on the desktop or in Places, then rt-click on it and eject it (leaving the modem in place) Wait 15 or 20 seconds and go to the network manager (on the top panel) and follow the install instructions.

If none of this works then come back to us with a little more detail about what happens when you insert the modem.


Bouncingwilf.

### Post by dave01945 on 2011-08-20
whenever i have had problems with mobile broadband not working with network manager i have used sakis3g it can be found here

[http://www.sakis3g.org/](http://www.sakis3g.org/)

it has everything compiled into it so just make it executeable and run 

```
./sakis3g --interactive
```

and it will prepare the modem and give you a list of access point or you can enter a custom one if it's not listed i always keep a copy on my computer found it very usefull
