Hola !
Para el dominio @ubuntu.pe se utiliza el servicio proporcionado por google apps, la manera de utilizar el correo a través de thunderbird será muy parecido a la configuración hecha para utilizar gmail

Puedes revisar este link donde se detalla todo el proceso para una correcta configuración:
[https://support.google.com/mail/troubleshooter/1668960#ts=1665018,1665141](https://support.google.com/mail/troubleshooter/1668960#ts=1665018,1665141)

Básicamente es ingresar tus datos de cuenta tal cual, thunderbird se encarga de practicamente todo el proceso ;)
