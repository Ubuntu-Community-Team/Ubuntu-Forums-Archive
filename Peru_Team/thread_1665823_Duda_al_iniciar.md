ubuntu mode rescue
mentest

bueno ahora si as actualizado, puede que se ya instalado otro kernel adicional y el sistema no borra el anterior pór si el nuevo falla tienes la opcion de iniciar el antiguo

y asi  constantemente ( los numeros y xx son referencia)
