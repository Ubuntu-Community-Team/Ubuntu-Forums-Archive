### Post by JohnAStebbins on 2010-04-21
Last official HandBrake release 0.9.4:
[http://handbrake.fr/downloads.php](http://handbrake.fr/downloads.php)
Also available in a PPA:
[https://edge.launchpad.net/~handbrake-ubuntu/+archive/ppa](https://edge.launchpad.net/~handbrake-ubuntu/+archive/ppa)
Note that the release doesn't work on ubuntu 10.04.


Nightly snapshots:
[https://edge.launchpad.net/~stebbins/+archive/handbrake-snapshots](https://edge.launchpad.net/~stebbins/+archive/handbrake-snapshots)
These work on 10.04.
