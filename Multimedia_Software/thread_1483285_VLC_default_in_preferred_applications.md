### Post by yesplease on 2010-05-15
SOLVED: there are 3 options to make VLC the default application.

1) VLC default in preferred applications, as in the posts above did not work.

2) What does work is to right-click on a file and choose properties (choose Open With, etc.)

3) Right-click and open with other application. Choose the application from the list. Tag 'remember this application for this type of files' did not give the expected result (i.e. opening the file for a second time, uses the old default).
