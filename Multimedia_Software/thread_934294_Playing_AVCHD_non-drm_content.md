### Post by whoopn on 2008-09-30
FIXED!

Ok in order to install this and play AVCHD files on Linux, Windows (XP or Vista), and presumably Mac in the future:

You must install CoreAVC's codecs ($8) and then install XP/Vista Codec pack for Windows

Linux:  Go to [coreavc-for-linux]("http://code.google.com/p/coreavc-for-linux/") and follow their instructions in the wiki.  Note I think you still have to purchase coreavc in order to make this work...but at $8 I think even the linux gurus can afford that.

Its not the best solution but its the best (and fastest) one out there.
