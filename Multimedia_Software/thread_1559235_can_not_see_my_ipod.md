### Post by Lars Noodén on 2010-08-23
Have you installed HFS support?  The Apple products use HFS instead of EXT.

[hfsplus](http://packages.ubuntu.com/lucid/comm/hfsplus) - Tools to access HFS+ formatted volumes
[hfsprogs](http://packages.ubuntu.com/lucid/comm/hfsprogs) - mkfs and fsck for HFS and HFS+ file systems
[hfsutils](http://packages.ubuntu.com/lucid/comm/hfsutils) - Tools for reading and writing Macintosh volumes
