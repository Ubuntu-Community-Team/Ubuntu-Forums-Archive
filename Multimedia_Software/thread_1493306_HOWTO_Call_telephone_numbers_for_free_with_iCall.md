### Post by chenxiaolong on 2010-05-25
Well, iCall now has a beta Linux client for it's free PC-to-telephone calling software.

It's super easy to install:

```
sudo apt-get install libqt4-core libqt4-gui libogg0 libtheora0
wget "http://www.icall.com/desktop/download/?version=beta&os=linux"
tar zxvf iCall-7.0-beta.tar.gz
cd iCall-7.0-beta
sudo sh INSTALL.sh
```

and that's it :D.
