### Post by funkydan2 on 2008-07-08
The best way to find out why a program doesn't appear to start is to run it from the terminal.
Go to Applications->Accessories->Terminal and then run 'tuxguitar' at the command prompt.  You should get an error message which will either help you to understand what the problem is or post it up here so that someone can help you with it.
