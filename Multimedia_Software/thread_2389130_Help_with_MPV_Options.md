### Post by SeijiSensei on 2018-04-12
Another vote for smplayer. 

If you highlight all the episodes using a file browser like Nautilus or Dolphin, you can right-click and choose "Enqueue in SMPlayer" to build a playlist that will let you navigate through the episodes. You can save the playlist for future use as well. SMPlayer remembers where you are in every video you watch, too.
