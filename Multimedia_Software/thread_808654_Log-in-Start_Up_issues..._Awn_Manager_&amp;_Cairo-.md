### Post by bogil on 2008-05-26
From the top panel you could go to:
System > Preferences > Sessions

For example, to automatically launch awn at log-in:
[LIST=1]
[*]Click the add button
[*]Type a label for it, example: awn dock
[*]On the "command", for awn, type: avant-window-navigator
[*]Comment (exeptional), type whatever you want.
[/LIST]

The same goes for  cairo clock; if you want to launch it at log-in just do the steps above, but on the command, type: cairo-clock
