### Post by markbuntu on 2008-08-05
You should have no problem dual booting if you follow the directions carefully.

Also, the latest 8.7 driver does support crossfire through the aticonfig command line and crossfire support is expected to be in gui Catalyst Control Center sometime in the next few months. 

Since ati was aquired by amd there has been a wholesale shift in ati's attitude towards and support for the linux community. They have opened their files and specs to the open source developers and are working very hard to bring their linux drivers up to the same capabilities as their windows drivers, which is their immediate goal.


As it is, the 4850 was the first ati card released with linux drivers on the same day though most vendors declined to include them in the driver software packages.
