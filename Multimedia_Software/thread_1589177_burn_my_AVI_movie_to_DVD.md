### Post by tommcd on 2010-10-09
> **inman2787 said:**
> I want to burn a movie to a dvd but the problem is that this movie is in the .avi format. Is it possible to burn a .avi file to a dvd. Also tell me how to get this to work. are you familiar with any program which will help me in this matter.
The easy and *free* (as in freedom and free as in beer) way to encode an .avi movie to DVD format and burn it to a DVD is to simply install **devede** from the Ubuntu repos:
[http://packages.ubuntu.com/lucid/devede](http://packages.ubuntu.com/lucid/devede)
[http://www.rastersoft.com/programas/devede.html](http://www.rastersoft.com/programas/devede.html)
Just open a terminal and run:
```
sudo apt-get install devede
```
Or use synaptic, or the Software Center, which ever you like, and you will be good to go.
The only other thing you will need is the popcorn!
Write back if you need more help.
