### Post by pwnprog on 2008-08-22
it's not a linux program, but it is a very lightweight windows program that can be run with wine quite easily. it is called foobar2000 and can be downloaded at this website [http://www.foobar2000.org/](http://www.foobar2000.org/) . 

i just downloaded an entire discography without and tags!! luckily this program saved my ***. if you bring the files from one album into the playlist window, highlight them, and right click, you can select an option to autodownload the appropriate tags for those files. it was quite amazing how it detected each tag perfectly!!

highly recommended
