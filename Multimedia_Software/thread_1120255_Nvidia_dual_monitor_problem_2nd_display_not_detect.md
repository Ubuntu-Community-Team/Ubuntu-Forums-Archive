### Post by Lunatico_22 on 2009-04-09
I'm not sure, but I think using the gnome display manager and modifying the Xorg config at hand made things not work. What I do with the nvidia drivers, after restoring the original settings for the xorg config, is running it as root. ```
sudo nvidia-settings
``` This way you can change the resolution and display configuration inside the driver and maintain the changes after rebooting the machine, very helpful when the EDID is corrupt. When connecting a 2nd display, use the nvidia driver to set it up, as root if you want to keep the changes. Display configuration - detect displays - configure - twinview - clones. 

Hope that helps.
