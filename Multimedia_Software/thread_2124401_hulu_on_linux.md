### Post by VeeDubb on 2013-03-10
It's probably overkill if all you want to do is watch hulu, but the bluecop plugin for XBMC works just fine on linux, which means that you can use XBMC to browse and watch Hulu.  

That said, it's probably easier to just watch it in the browser unless you're running a HTPC, which case you'd probably already be running XBMC.
