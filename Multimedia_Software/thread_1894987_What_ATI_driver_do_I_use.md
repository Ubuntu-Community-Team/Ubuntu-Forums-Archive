### Post by Mark Phelps on 2011-12-14
The ATI open-source drivers have improved a lot in recent Ubuntu releases, especially in 11.10.  So, unless you really need the restricted drivers, either for 3D acceleration or for CCC fan support, you should consider staying with the default open-source drivers already installed.
