### Post by namdung on 2008-12-30
Is Ubuntu recognizing the CD/DVD but is unable to play the multimedia content? Is it able to content of the CD/DVD? If not it's a hardware or mounting issue.

Ubuntu doesn't install proprietary audio/video codecs by default.

In *Add/Remove Applications* search and install for GStreamer codecs. Don't forget to select *Show All Available Applications*

This will enable u to play proprietary formats like mp3, wav, avi, mpeg etc.
