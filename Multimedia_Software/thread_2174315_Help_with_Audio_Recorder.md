### Post by ibjsb4 on 2013-09-13
You seen this ?

> Aditional guides and fixes:
If you are recording from a microphone, please check the [Input] settings in gnome-volume-control. The input volume is often too low.  Please study this picture: [http://www.futuredesktop.org/oneiric/images/picture-8c.png](http://www.futuredesktop.org/oneiric/images/picture-8c.png)
 You may need to change the sound configuration on some computer systems. Please take a look at this picture.
[http://bildr.no/image/1284507.jpeg](http://bildr.no/image/1284507.jpeg)
Install and start "pavucontrol" and set the configuration to "Analog  Stereo Duplex" and test the recording. This may fix the recording if you  get "empty" audio files.
 See also: [https://answers.launchpad.net/audio-recorder/+question/209038](https://answers.launchpad.net/audio-recorder/+question/209038)

Thats from the PPA.

[https://launchpad.net/~osmoma/+archive/audio-recorder](https://launchpad.net/~osmoma/+archive/audio-recorder)
