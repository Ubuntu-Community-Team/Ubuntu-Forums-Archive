### Post by Baldrick_NZ on 2010-05-03
Having used .mp3 format for most of my computer life, I thought I'd try out some of the open-source alternatives.

I tried .flac which gives very good sound quality, but with files on average of 25Mb per song would start using more disc space. So I tried .ogg, which also gives (to my ear) no appreciable quality difference (256kbps or 500kbps), but at a fraction of the disc space required for .flac files.

What I'm looking for is an .ogg version of 'MP3Gain' which will allow me to normalise all my .ogg files to virtually the same audio level, without any appreciable sound quality loss.

I've played with EasyMP3Gain (which has worked flawlessly for my .mp3 files), and although it has a listing for .ogg files, it won't normalise them. There is an .aac plugin in the repo's, but not one for .ogg's.

Any ideas what one might use?

Cheers.

### Post by Rhubarb on 2010-05-03
Try **vorbisgain** it's in the repositories, and does exactly what mp3gain does (but for ogg vorbis sound files).

I haven't tried out vorbisgain myself, but at a guess it'd be a command line app, so if you want to know how to use it, just run:
```
man vorbisgain
```
