### Post by tgalati4 on 2010-08-01
Well yes and no.  Using pulseaudio, applications and desktop notifications can mix audio and you can set custom audio levels for system beeps and bops and for various music and video players.

Playing youtube means that you are using Adobe Flash.  It's not well-coded, nor well-supported under Linux.  It's not open source and there is no hope of it improving.  So, it's not surprising that your audio doesn't work the way you expect it when using Flash.

Mac OS X uses Core Audio, which is a proprietary audio framework.  I don't know how well it works with Youtube and Flash, but it's probably better than under Linux.

Use whatever works for you.  Realize that Flash will always be crap under Linux and find other ways not to use it--like HTML5 websites.
