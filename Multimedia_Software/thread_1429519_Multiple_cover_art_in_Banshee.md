### Post by boombox1387 on 2010-03-20
If you have an album with multiple artists that is showing up several times in the Browser, you can group all of them together using the 'Compilation Album Artist' field.  For each song in the album, go to Edit Track Information, and check the box next to 'Compilation Album Artist:'.  In the text field under that label, give each song on the album the same value, like Various Artists, for example.

Once each track is identified as part of a compilation, each song with the same values for Album and Compilation Album Artist should get grouped together under a single album.  Hope this helps, if not, let me know.
