Installed ubuntu 8.10 ,  then activated nvidia driver. After reboot mouse can be moved, but in some seconds (20 sec) mouse and keyboard became freezed, however, ubuntu works ok (update notification window appeared).

When ubuntu uses "default video" driver, then mouse and keyboard works ok.

In case of nvidia in xorg.conf driver name is "nvidia", NoLogo = "true" also "glx" is enabled.
