### Post by AndyBoy_LV on 2009-03-06
Ok, i guess i have found my solution

First, check the available refresh rate in nvidia-settings. 

Run in terminal: ```
nvidia-settings
``` If you don`t have nvidia-settings installed, you can get one from synaptic.

My refresh rate was 62. Remember it, as you will need it later. 

So, let`s edit xorg.conf. In terminal, open it with ```
sudo gedit /etc/X11/xorg.conf
``` (Note: I use gedit as my text editor, you may use something else.)

Next step is to generate some Modelines. 

First, you must find a list of supported resolutions for your Nvidia card. I got all my resolutions from my Vista partition. 

Next, you must run this command in terminal:

```
gtf x y r
``` where x and y is a resolution you want and r is a refresh rate you got from the nvidia-settings command. For example, for resolution 800x600 run this: ```
gtf 800 600 62
```  

The output of the command would be something like this:

```
    	# 800x600 @ 62.00 Hz (GTF) hsync: 38.56 kHz; pclk: 39.49 MHz
  	Modeline "800x600_62.00"  39.49  800 832 912 1024  600 601 604 622  -HSync +Vsync

```

Add this to your xorg.conf to the "Monitor" section, just like that:

```
Section "Monitor"
	Identifier	"Configured Monitor"
  	# 800x600 @ 62.00 Hz (GTF) hsync: 38.56 kHz; pclk: 39.49 MHz
  	Modeline "800x600_62.00"  39.49  800 832 912 1024  600 601 604 622  -HSync +Vsync
EndSection
```


And also that resolution to the "Screen" section:

```
Section "Screen"
	Identifier	"Default Screen"
	Monitor		"Configured Monitor"
	Device		"Configured Video Device"
	DefaultDepth	24
    	SubSection "Display"
        	Depth           24
        	Modes   "1920x1200" "800x600"
    	EndSubSection
    	SubSection "Display"
        	Depth           16
        	Modes   "1920x1200" "800x600"
    	EndSubSection
EndSection
```

Do that for all resolutions you need, save the xorg.conf file and than restart your PC. If you did everything right, all your defined resolutions should be available in the nvidia-settings.

Good luck!

### Post by theasprint on 2010-12-23
Hi,

As my signature suggests, I can help.

Here is my way of solving Nvidia graphics problems. And most if not all of the people I helped actually solved their problem.

See, Ubuntu and Canonical actually does not support Nvidia graphics drivers for Ubuntu.
So therefore the drivers they provide under System>Administration>Hardware Drivers are not up to date and may be incompatible. 

So, you should actually use drivers from Nvidia instead.

1. Turn off/Disable the Nvidia drivers that Ubuntu provided under System> Administration> Hardware Drivers.

2. Go to nvidia.com and download the latest drivers for your graphics card.

3. Turn off your gdm (graphics mode)
```
sudo service gdm stop
```

4. Install your downloaded Nvidia driver
```
sudo bash <your Nvidia driver's directory>
```

5. Turn on your gdm
```
sudo service gdm start
```

The users in these 2 threads also have their problem solved:
[http://ubuntuforums.org/showthread.php?t=1648734]("http://ubuntuforums.org/showthread.php?t=1648734")
[http://ubuntuforums.org/showthread.php?t=1651377]("http://ubuntuforums.org/showthread.php?t=1651377")

If you still have problems, go to the thread in my signature below, and follow the instructions of TheRawGod.

* I consider my this way to be an official solution because it involves getting drivers from official nvidia website, which is the most updated ones.
Editing xorg.conf files may seem primitive and sometimes do screw up.

Good Luck :D
