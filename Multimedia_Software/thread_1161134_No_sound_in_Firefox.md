### Post by HappyFeet on 2009-05-16
Sometimes deleting the firefox config file can fix issues. Go to your Home folder and do ctrl-H. Then look for .mozilla file and delete it. Then restart firefox and a new config file will be made. Doing this though will reset all favorites and add-ons. Perhaps you will want to backup your config file first. But it is worth a shot.
