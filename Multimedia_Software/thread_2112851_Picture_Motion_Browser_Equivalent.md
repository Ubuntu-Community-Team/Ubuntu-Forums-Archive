### Post by sudodus on 2013-04-20
Welcome to the Ubuntu Forums :-)

You can use Shotwell or Digikam to import and organize photos and video clips from flash memory or a HDD in a modern camera. Openshot is a fairly simple video editor, that can also convert to different video formats.

If you need to import DV via firewire, you can use Kino in linux. It is possible but not easy to set it properly (or at least it was, when I tried it a couple of years ago). I think it is much easier to use Windows and Sony's software and easiest to use Mac OS to import the video via firewire.
