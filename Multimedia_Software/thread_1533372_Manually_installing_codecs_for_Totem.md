### Post by NFblaze on 2010-07-17
For Totem you can go to Synaptic Package manager and install the gstreamer codecs for good, bad, ugly and fluendo. That should cover all of them, but if it doesnt next time you play an unrecognized file Totem will search for that package and install it.
