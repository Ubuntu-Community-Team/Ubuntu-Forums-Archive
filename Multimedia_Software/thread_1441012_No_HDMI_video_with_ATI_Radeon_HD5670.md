I'm running Ubuntu 9.10 64bit with the latest updates and I just put in a brand new Radeon HD5670 video card.  Normal monitors work just fine, however Ubuntu isn't detecting my TV when I connect it via the HDMI port.  (I'm assuming it should show up in System -> Preferences -> Display)  

I've been googling for most of the day now, but I can't find much on how to troubleshoot HDMI video.  I have a laptop with Ubuntu on it and it's HDMI works great, so I'm not sure what the problem is.  You guys are my last hope, so any help would be greatly appreciated.  Thanks!


**Edit: Nevermind, I got it to work.**  Apparently the card will only work with either the HDMI or the VGA plugged in, not both.  On top of that, my TV (Toshiba Regza 32C7000) only accepts the input if it's on HDMI port #3.  I've got no explanation for that, but oh well.
