I got same error with my psd file from a MAC user, too.
Don't know why.
With Google I found at [http://ubuntuforums.org/showthread.php?t=87055](http://ubuntuforums.org/showthread.php?t=87055) :
"In my experience one common reason for people not being able to open valid Photoshop files is that they are in CMYK mode (for commercial printing) rather than RGB, greyscale or indexed mode. Even the 'lite' versions of Photoshop can't open them."

But the MAC user did say, the file was RGB already.

The solution in my case: The MAC user did save the file as "cs2 version" and voila - it opens successfully!

Good luck!
