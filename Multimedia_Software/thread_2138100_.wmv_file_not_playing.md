### Post by SeijiSensei on 2013-04-23
There is a class of WMV files that will not play in 64-bit Linux because the codec file needed for them is only available in a 32-bit build.  See [this thread](http://ubuntuforums.org/showthread.php?t=1705713) for details and some workarounds.
