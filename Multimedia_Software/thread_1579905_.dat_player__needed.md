### Post by Ghost|BTFH on 2010-09-23
You could try to right click on it and tell it to open with movie player, see if that works.

Basically, if you have all the gstreamer plugins/codecs/etc. a .dat file should be able to be played with movie player just fine.

Please note however, a .dat format can represent a whole lot of different file types and may not even be a VCD file.  Make sure you know the source of the file before you just try to play it.
