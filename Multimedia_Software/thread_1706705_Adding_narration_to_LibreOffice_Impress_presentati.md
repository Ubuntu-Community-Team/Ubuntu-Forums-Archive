### Post by gewitty on 2011-03-14
Does anyone know if it's possible to add a narration to an Impress slideshow?

The requirement is for a slideshow that will be exported to run on a web page. I can't find any way to record a narration in Impress itself, so I'm now wondering if I can export the presentation to another application where I can add the soundtrack, prior to transferring it on-line.

---

### Post by howefield on 2011-03-14
Have a look at the menu option Slide-Show  > Slide Transition.

It may do what you require but you'll need to have the soundtrack already recorded so you can add it using this option.

### Post by gewitty on 2011-03-15
Thanks for the suggestion.

I found an app called Videoporama, which looks promising. It still requires a separate sound track, but allows me to output my Impress show in a number of different video formats. This has certain advantages in that the file is effectively 'locked', and will play reliably on just about any platform.
