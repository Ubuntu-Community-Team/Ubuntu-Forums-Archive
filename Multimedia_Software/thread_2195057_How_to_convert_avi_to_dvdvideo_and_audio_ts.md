### Post by ajgreeny on 2013-12-22
If I understand you right, you want to convert an .avi file to a format that will allow you to burn to a DVD using k3b for use in a household dvd player.

You could use avconv (or ffmpeg) to easily convert the .avi to an mpeg2 (file.mpg) which you can then easily burn to a DVD which will play in a normal dvd player.

However, like The Spectre, I am also surprised that you are having problems using devede, as I have always found it to be a very capable application.
