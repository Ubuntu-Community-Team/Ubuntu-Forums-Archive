> **neilevan814 said:**
> I was wondering if anyone knew what the best way is to play a video in slow motion.

Dynamic speed alteration can be accomplished with MPlayer as well:

```

 [ and ]  Decrease/increase current playback speed by 10%.
 { and }  Halve/double current playback speed.
 backspace Reset playback speed to normal.

```

All the best,

Andrew
