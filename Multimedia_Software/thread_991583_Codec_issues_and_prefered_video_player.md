### Post by mc4man on 2008-11-24
> How do I make Ubuntu's preferred movie player VLC instead of Totem
That depends on whether your using 8.10 (intrepid) or 8.04 (hardy)

For intrepid
simply open nautilus (open your home folder)
then in top panel, edit -> preferences -> media -> and in the Dvd Video drop down choose vlc.

For hardy see here

[http://ubuntuforums.org/showthread.php?p=4986640#post4986640](http://ubuntuforums.org/showthread.php?p=4986640#post4986640)

Note for setting a player as default you can only use this method once, see link for how to add other apps as a choice for default player ect.

When you change vlc's launch command try using  vlc %f first.
