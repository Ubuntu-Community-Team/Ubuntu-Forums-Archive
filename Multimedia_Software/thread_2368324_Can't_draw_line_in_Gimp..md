### Post by ajgreeny on 2017-08-10
To try out your current configuration for gimp, make sure all gimp windows are shutdown, rename the hidden **.gimp-2.8** folder in your home as **.gimp-2.8backup** and restart gimp.

Try drawing a line now; does it work?  If so your old config was corrupt in some way, if it still does not work I am not sure what else to suggest except for purging gimp and then reinstalling to see if the whole installation was corrupt in some way.
