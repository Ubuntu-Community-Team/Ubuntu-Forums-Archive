### Post by Mrazek on 2009-09-04
Hi,
 
I recommend to test everything on any test machine(s) and then decide what component(s) to upgrade.
For example I currently run my testing backend on Athlon XP 2100 with 1 GB RAM and nVidia 6xxx and testing frontend on [EMAIL="P4@1.7"]P4@1.7[/EMAIL] with 1 GB RAM and nVidia 5xxx. On backend there are two cards: Pinnacle DVB-T/analog and Leadtek analog. I think both machines are powerful enough and so I think you can use for example Intel Atom, no worry.
