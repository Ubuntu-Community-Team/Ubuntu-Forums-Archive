Have got VLC Media Player to play all my CD's and DVD's. So could someone tell me how i can make VLC Media Player the default program anyone please

---

### Post by Revolutionary101 on 2010-01-07
Right click all of the files that you would use with VLC and go to properties. Then you should go to the Open With tab and select VLC as the program to open the file with. This will work for all files of the same media type so you need to do it to any file type you will play with VLC (ex. avi, mkv, mp3 ect.)
