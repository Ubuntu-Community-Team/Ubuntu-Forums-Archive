When I was using windows Itunes was my primary music player. One of my favorite feature is the active scan. Everytime I download music (legally of course) Itunes automatically adds it to my library. Ive tried some music players for ubuntu, but it requires me to add the music in manually after download. Is there any music player that has this feature?

### Post by Kirboosy on 2014-02-03
Rhythmbox has that feature. I think several others support it too but they just might not be on by default. 

In Rhythmbox you can set it up through Rhythmbox > Edit > Preferences > Music (tab) > "Music files are placed in" and"Watch my library for new files"

Hope that helps!
