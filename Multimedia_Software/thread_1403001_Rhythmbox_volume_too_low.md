### Post by midwesterndirt on 2010-02-09
I'm running Rhythmbox 0.12.6 in Ubuntu 9.10 64-bit and for some reason, the audio volume in this application is lower than anywhere else (mplayer, vlc, etc). I've checked the per application volume settings in the system audio settings, but everything looks normal. Replay gain shouldn't be the cause because all of these apps should recognize it equally. I've tried using the equalizer plug-in and turning all the bands up, which raises the volume to the appropriate level, but degrades the fidelity of the sound. Another indication of this problem is when I first play a song after opening it, the volume is normal but then drops after a second or two. I've found nothing in Rhythmbox bug reports, but due to the huge number of reported bugs with the app, I don't expect to see a solution there any time soon. No other media player works has the features that I want, so switching is not an option. Has anyone experienced this issue and found a solution?
