### Post by FraggedLocust on 2009-02-12
As far as cheese goes, you should see it under Applications>Graphics if you install it with Add/Remove. If you find that, you can just drag it onto your desktop, and ubuntu will create a shortcut for you.

Otherwise, I think you should be able to copy cheese to the desktop by running this command in a terminal.
```
sudo cp /usr/bin/cheese ~/Desktop
```
