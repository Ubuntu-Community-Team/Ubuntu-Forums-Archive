I used Asunder very happily under earlier versions of Ubuntu.

I had to install 12.04 from scratch after the upgrade stalled.

I use the Gnome interface.

Problems:

1) The refresh button does not show. Instead I have a button that says CDDB Lookup. How to I refresh when I change a CD?

2) When I insert a CD, I get the Audio Disk window and I can select "what to do". When I select "Other Applications" Asunder is not listed as an application. How do I add it to the list?

### Post by stephen67 on 2012-05-24
I found the problem

Preferences had /dev/cdrom

I changed to /dev/sr0

All works now!

Stephen
