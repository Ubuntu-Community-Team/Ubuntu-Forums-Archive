### Post by viktormas on 2011-04-20
VLC and Gnome Player are my favourite video players. Kaffeine is another one I use to use for a long time. You can install all of them and try them out with this simple command in the terminal:
```
sudo apt-get install vlc gnome-mplayer kaffeine
```
