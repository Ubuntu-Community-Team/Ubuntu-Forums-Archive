### Post by renzokuken on 2008-10-20
You can either install VLC which has all the necessary codecs built in

```
sudo apt-get install vlc
```

.....or, you could use medibuntu ( [www.medibuntu.org](www.medibuntu.org) ) to install the codecs seperately and then use whichever media player (e.g. mplayer, exaile, rhythmbox, totem etc) to play your files
