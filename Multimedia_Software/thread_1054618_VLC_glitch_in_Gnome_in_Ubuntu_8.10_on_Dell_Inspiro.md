This video: [http://www.youtube.com/watch?v=0v7ZiS9KbvM](http://www.youtube.com/watch?v=0v7ZiS9KbvM) documents a bug I've encountered using VLC on my Gnome desktop under a fully updated Ubuntu 8.10 system on January 29, 2009. I've had this problem since installing Ubuntu on my desktop. The video output for VLC and several other applications flickers and sometimes appears over other windows placed atop the video stream. In this example, a VLC video stream flickers atop a focused Firefox window. When the Firefox window closes, the video continues to flicker, but with less frequency. I have only been able to avoid this problem with VLC by running it in a failsafe terminal rather than on my normal Gnome desktop. I am using an unmodified Dell Inspiron 1526 notebook with a proprietary ATI graphics card driver. Viewing YouTube videos in Flash works just fine.
