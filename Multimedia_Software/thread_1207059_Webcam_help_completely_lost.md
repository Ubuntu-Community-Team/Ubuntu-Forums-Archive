### Post by Brian Vaughan on 2009-07-07
cheese is available from the Ubuntu repositories, compiled and packaged for Jaunty. You could install it with Synaptic Package Manager, under the System menu, or you could use Add/Remove... under the Applications menu.

I noticed this thread, because I wanted to try out a webcam I've had lying around. It took seconds to install cheese via Synaptic, and it automatically detected my webcam.

Compiling and installing from a downloaded tarball is rarely necessary.You may want to look at the instructions on installing software on Ubuntu:

[Adding, Removing and Updating Applications]("https://help.ubuntu.com/9.04/add-applications/C/index.html")
