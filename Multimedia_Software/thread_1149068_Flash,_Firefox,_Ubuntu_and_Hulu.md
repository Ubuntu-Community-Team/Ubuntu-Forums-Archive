### Post by mdg1019 on 2009-05-04
I was having problems with Flash working with Firefox and the new Netbook Remix. I'd installed the SWF plugin. I went to Hulu to watch a TV program and I got this HUGE play button on the screen. I pressed it and unfortunately nothing played.

It's been awhile since I've run Linux and I thought I'd give Ubuntu a try on my netbook, Acer Aspire One. After fumbling around for awhile, I came up with a combination that worked. I uninstalled the SWF plugin. Then I downloaded the .deb file from Adobe's site. I followed their installation instructions and received an error message that libcurl3 was required by the plugin, but not installed. So, I installed libcurl3 with the package manager and tried the installation instructions again. Voila. The plugin installed fine and I can now watch Hulu videos.

Hope this may help somebody else who runs into a similar problem. As I stated above, it's been awhile since I've run Linux, so there may be a simplier solution, but this works.:)
