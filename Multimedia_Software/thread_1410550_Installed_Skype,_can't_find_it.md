### Post by tekkidd on 2010-02-18
Go to the terminal and type in Skype. Then Skype should start. If it does right click on the menu (in gnome) OR right click on the "K" icon (in KDE) and choose edit menus. Then go to the internet tab and click "New Item" Then for the name type in Skype and for the command also Skype. The go and Google for Skype icon and download it. Now go back to the menu editor and click on the icon select custom icon and navigate to the location where you saved the icon. Then press ok. That ought to do it. And if you get a Skype unrecognized command error reinstall Skype.
