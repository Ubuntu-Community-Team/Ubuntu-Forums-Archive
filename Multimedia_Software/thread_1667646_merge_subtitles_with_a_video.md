### Post by solar george on 2011-01-15
Do you want to 'burn' the subtitles into the video or just have them available as an option?
For the former you're going to have to re-encode the video so I'd recommend handbrake (from handbrake.fr - they have a ppa with updated packages available) which is able to add a subtitle file to the video output in this way.
If you want the latter I'd look into [mkvtoolnix-gui]("apt://mkvtoolnix-gui") which will convert the file (without re-encoding) into a .mkv file which allows you to add optional subtitles to the file.
