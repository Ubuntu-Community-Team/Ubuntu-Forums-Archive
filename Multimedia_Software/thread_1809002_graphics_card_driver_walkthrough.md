### Post by jocko on 2011-07-22
> **lkjoel said:**
> Open up a Terminal window and type in it:
```
sudo apt-get install fglrx nvidia-current

```Reboot your system into full graphics mode.
No. That won't help in any way because of:
1. His nvidia card is way too old (released in 2000) to be supported by the nvidia-current driver (but it's probably supported by the legacy driver nvidia-96).
2. He's trying to use a radeon card (which is "only" 7 years old compared to the 11 years old nvidia card), so I'm assuming the nvidia card is laying on a table or in a box somewhere...

@rogerrabbitsclone:
The radeon 9800 is probably the best of those cards, and both of the ati cards are supported by the open source radeon driver. None of them are supported by the proprietary driver, so don't even try to install fglrx...

To get the radeon driver working, you should have to do absolutely nothing. It should work directly after a fresh install of ubuntu (but your attempts to get the nvidia card working may very well have messed things up).

But if you want help with getting the driver working you need to do a few things:
1. Delete /etc/X11/xorg.conf if you have created one. It is not needed, and if it's misconfigured it may cause problems.
2. Reboot and see what happens.
3. To see which driver is i use (find your gfx card in the output):
```
lspci -k
```
4. Post your /var/log/Xorg.0.log here. It will tell what the problem is, if there is any.
