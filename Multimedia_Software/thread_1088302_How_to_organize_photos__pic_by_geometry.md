### Post by yther on 2009-03-05
The command **identify** is part of the ImageMagick (sic) package.  Here's a sample output from it:

```
fedex desktop1.png PNG 1680x1006 1680x1006+0+0 DirectClass 8-bit 167.426kb
p1010001.jpg JPEG 3072x2304 3072x2304+0+0 DirectClass 8-bit 3.29335mb 0.300u 0:02

```

The output of that could be parsed, although not easily (to me) if the names have spaces.  It's possible the output format could be tweaked a bit; see the documentation.

I bet there are some image management utilities out there that can do this automatically... someone's bound to know.  :)  In fact, I believe [GQview]("http://gqview.sourceforge.net/") can sort images by dimensions.  It's a bit old, but it's still available in Ubuntu.
