### Post by lovinglinux on 2008-11-09
I don't like the new VLC. It takes some time to load and start with black screen video or glitches on two machines here, using Hardy, Intrepid or Windows.

I think smplayer is a really good option. I has several configurations and controls, a nice interface and play well. It also has excellent subtitle support, including SSA and you can change the charset to use subtitles in Portuguese.

If you are looking for simplicity, try gnome-mplayer. Is a simple and nice frontend to mplayer.
