There is a Band In A Box-type program in the Packages called MMA - Musical MIDI Accompaniment. It's a command-line based musical backup track generator. I've used it before and it's excellent. Unfortunately, Ubuntu only has a very old version, which didn't work with my jaunty.

I'm not having any luck with the current version 1.5 posted on their website, [http://www.mellowood.ca/mma/downloads.html](http://www.mellowood.ca/mma/downloads.html)

### Post by musiloco on 2009-12-05
I have installed MMA on Ubuntu 9.10.
I have just checked and I have the version 1.5. So I think I have installed the old version from Ubuntu repositories and then the newer one from the MMA&#347; website.
At first I did not know how to use it. Then I read the tutorials and I realized that MMA has no GUI, just command line. Then I installed LeMMA, a front end GUI and things started to get shape for me. Also you need timidity  or a similar synth to produce sounds (Qsynth and Totem also worked for me).
It is a decent replacement for BIAB.
