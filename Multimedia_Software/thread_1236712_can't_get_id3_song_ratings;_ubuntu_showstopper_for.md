### Post by benerivo on 2009-08-10
Your problem is really that you have just wiped the windows partition, and can no longer see the stars in linux. The only obvious fix that comes to mind is to either reinstall windows, or just try running mediamonkey in wine, so that you can create a playlist for each star rating you have. Then in amarok, you can open each playlist and batch rate all the songs with the correct amarok star rating (although i'm not sure where amarok would write this rating to as you suggest it doesn't read the existing id3 ratings).
