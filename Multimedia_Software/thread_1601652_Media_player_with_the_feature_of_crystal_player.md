### Post by bobwyajnr on 2010-10-20
I've been playing around with media players in Ubuntu recently.

XBMC wins hands down in my experience. Gorgeous interface I must say. I actually got it to light the DTS 5.1 channel light on my digital headphones (SPDIF)!! It has hands-down the best media player interface for Ubuntu/Linux. Once I had tried the BluRay playback plugin (and got this working) I fell in love!!

VLC, Mplayer and Xine all have interface layouts and features which match Windows media players from 10+ years ago. Stream information is rudimentary at best... I tried playing about with the latest version of Mplayer (compiling this from source) and didn't see much happening there to excite me...
