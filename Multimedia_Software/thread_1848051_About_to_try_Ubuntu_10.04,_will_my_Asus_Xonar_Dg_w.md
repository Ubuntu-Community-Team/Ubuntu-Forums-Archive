### Post by .... on 2011-09-22
The Xonar DG requires ALSA 1.0.24 for full use ( [http://www.alsa-project.org/main/index.php/Changes_v1.0.23_v1.0.24#CMI8788_.28Oxygen.29_driver](http://www.alsa-project.org/main/index.php/Changes_v1.0.23_v1.0.24#CMI8788_.28Oxygen.29_driver) ). Ubuntu 11.04 still uses 1.0.23 for the ALSA kernel module.

Either use Ubuntu 11.04 and upgrade ALSA : [http://ubuntuforums.org/showthread.php?t=1681577](http://ubuntuforums.org/showthread.php?t=1681577)
..or just use Ubuntu 11.10 (it's in Beta2 form right now, but should be released officially in a couple weeks)
