### Post by timcredible on 2009-01-16
are you sure the files are mp3 files?  if you ripped a file with ubuntu, the default file format is ogg (an open source format mostly superior to mp3).

also, make sure you unmount your ipod before disconnecting it (right-click on the ipod icon and choose 'unmount'), or whatever is in the cache won't get written to the ipod and it won't work.
