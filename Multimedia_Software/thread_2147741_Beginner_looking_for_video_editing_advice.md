### Post by bakelitedoorbell on 2013-05-24
I have been using **OpenShot** to edit videos for Youtube, and I think it could be used for your purpose.  It's fairly simple to use, although if you are new to editing video you may have to practice a bit.  You can install it from the Ubuntu Software Center.

Do the intro/outro clips change for each video?  If not maybe you could get a copy of them from the previous fellow.
OpenShot can use Blender to create simple animated titles if needed.
