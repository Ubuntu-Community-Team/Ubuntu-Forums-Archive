### Post by lovinglinux on 2009-11-26
Do you have the subtitle file in srt, ssa or other common format? If yes, then simply put the subtitle file in the same folder as the movie, with the same name as the movie file name. You also might need to enable subtitles in the player preferences.

I recommend using [smplayer](apt:smplayer) instead of the default player. It has an excellent subtitle support.
