### Post by WasMeHere on 2011-10-10
This kind of compressed archive is a product from this web site: 
[[COLOR="RoyalBlue"]_http://www.rarlab.com_[/COLOR]]("http://www.rarlab.com")

But often the built-in archiver in Ubuntu can manage to read and/or extract from various compressed file formats. Try selecting it from the *movie.part.xvid.rar* file in your file browser (right-click and select if it will not start by left-clicking). The other files should be in the same directory. I don't know about the sfv file.

Have fun

### Post by sammiev on 2011-10-10
I use rar which is already in the package manager and can be added in seconds. Then I just use this command from the Download directory containing the rar files.

rar x -y name1.rar

This will extract them and join then together. :)
