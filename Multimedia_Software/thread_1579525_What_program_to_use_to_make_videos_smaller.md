### Post by amsterdamharu on 2010-09-22
winff is a program that simplifies using ffmpeg. You check some boxes and off you go. FFMpeg is a command line program, that means that you have to run the commands in a terminal and read the man (manual pages).

I suggest installing ffmpeg and winff to use ffmpeg in a simpler way.
