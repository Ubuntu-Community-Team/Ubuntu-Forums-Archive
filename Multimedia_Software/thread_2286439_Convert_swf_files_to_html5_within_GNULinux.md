### Post by Yellow Pasque on 2015-07-12
I don't know if you're going to find a real easy solution, because swf is a pain in the butt. You may want to try playing the swf file in VLC and using this tool to create a FLV file. [http://www.unixuser.org/~euske/python/vnc2flv/index.html](http://www.unixuser.org/~euske/python/vnc2flv/index.html)
From there, you should be able to convert fairly easily to whatever you want: [http://superuser.com/questions/624565/ffmpeg-convert-flv-to-mp4-without-losing-quality](http://superuser.com/questions/624565/ffmpeg-convert-flv-to-mp4-without-losing-quality)
