### Post by ron999 on 2010-07-01
**dvd-slideshow** will make a slideshow from a bunch of jpg/png images and a soundtrack.
The result is a vob file which can then be authored into a DVD.
The vob file is mpeg2 video and ac3 audio. Choice of PAL or NTSC.

It's probably in the repository.
This is the website:-[http://dvd-slideshow.sourceforge.net/wiki/Main_Page](http://dvd-slideshow.sourceforge.net/wiki/Main_Page)
And some practical help here:-[http://dinsdalepiranha.wordpress.com/2007/10/16/how-to-make-a-video-for-youtube-with-linux/#comment-1025](http://dinsdalepiranha.wordpress.com/2007/10/16/how-to-make-a-video-for-youtube-with-linux/#comment-1025)
