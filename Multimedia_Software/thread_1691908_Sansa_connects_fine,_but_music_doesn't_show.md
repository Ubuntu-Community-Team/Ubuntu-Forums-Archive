### Post by Chronon on 2011-02-20
This can happen if you change USB modes.  When you sync files to your player in MTP mode they can end up hidden in a non-obvious location.  Usually, you need to use the same syncing method to get the music back off that you used to put it on.  If I am correct, you should set the USB mode back to MTP and then use an application that allows syncing to MTP devices to remove these files.  

Personally, I prefer MSC mode and just mount my audio players like a normal storage device.  This way I know where the files are located on my device.
