### Post by NovaWasp on 2009-10-19
In the live CD the drive should be present on the desk top.  Double Click it to mount it.  You should be prompted for a password.  Just hit enter.  There isn't a password in a live cd.  Now you may have permissions issues, but that's easily taken care of.  Open a terminal and type:
 
gksudo nautilus
 
If it asks for a password hit enter.  You should at this point be able to copy to any external device you connect.
 
Hope that helps.
