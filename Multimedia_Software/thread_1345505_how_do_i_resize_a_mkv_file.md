### Post by VertexPusher on 2009-12-04
Avidemux can read and write MKV files. If you want to reencode the video but keep the audio, load the MKV file into Avidemux, choose your video encoding settings and select "Copy" for audio. Choose MKV output format, then save.

You can use mkvmerge to separate the video first and then remux it after encoding, but there's no need to do so.
