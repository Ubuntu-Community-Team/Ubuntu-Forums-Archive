### Post by Bucky Ball on 2012-09-12
If you think Kino is difficult to use I would totally avoid Cinelerra. That'ss is intended to be a pro-level app similar in power to the cross-town heavyweights like Premiere, etc. If you have a spare month or two to learn it, go for it. You might take that long just getting it to work reliably.

What is difficult about Kino? Using it or getting it to work in the first place? Probably the most straightforward, easiest to use on the fly vid editor around for Linux.

Getting your machine setup to edit vid is usually the tricky part (getting the right codecs, drivers, modules, tweaks, etc).
