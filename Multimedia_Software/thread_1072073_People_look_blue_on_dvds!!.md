### Post by toolunious on 2009-11-04
For anyone still looking for an answer: I discovered that I had this problem since the upgrade to ubuntu 9.10. I think the problem is that somehow the Hue value is set to blue/0 or whatever the most left is on the Hue value-bar. You can fix this (with all media players in once!) by setting the Hue value in Totem. Vlc uses this value as well apparently.

 I think [Reset to Defaults] will do the trick as well. The hue bar is found from the menu bar clicking [Edit] > [Preferences] >> [Tab:Display].
