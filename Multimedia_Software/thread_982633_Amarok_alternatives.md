### Post by WWSmith36 on 2008-11-14
I use Amarok and fortunately it works perfectly in my machine.  Sorry to hear that you are having problems.

A famous winamp replacement for linux is xmms.  It can be found in the repos for dapper, feitsy, and gutsy (universe).  For hardy you can get the deb at launchpad
[https://launchpad.net/ubuntu/hardy/i386/xmms/1:1.2.10+20070601-1build2](https://launchpad.net/ubuntu/hardy/i386/xmms/1:1.2.10+20070601-1build2)

Try to install it with dpkg, install the dependencies with aptitude, and you should be set.

Hope this helps
