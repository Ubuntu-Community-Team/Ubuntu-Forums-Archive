> sudo apt-get install mp3splt mp3wrap mp3splt-gtk libmp3splt

then run with 

mp3splt-gtk

point it to dir containing .cue and .mp3 file, or else fetch cue online

there's also mp3directcut (wine), which is nice in that it displays the track in waveform fashion
