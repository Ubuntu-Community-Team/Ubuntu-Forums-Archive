### Post by mrtomservo on 2008-09-08
As a user of both nVidia and ATI cards, I don't think I can definitively say one is more stable than the other in Linux.  The only drawback that i've found with ATI is that if you use the fglrx drivers that you'll get screen flicker problems if you run GL apps with Compiz.  I can live with it, but some people can't.

I guess if I were to suggest one to a friend i'd go with nVidia.  But that's just because i've used it for longer and am more familiar with it's quirks.
