### Post by TheFu on 2015-12-04
Sorry. doesn't exist. Especially when you say "simple and intuitve."  Patches to most of the editing software projects are always appreciated.

For audio, **audacity** is the primary tool. 

For video, there are about 5-10 contenders. 

Paid Linux software generally is for professional use and runs in the $1.5K-$20K range.

Have you looked at alternativeto.net?  [https://alternativeto.net/software/windows-movie-maker/](https://alternativeto.net/software/windows-movie-maker/)
Be certain to check the possible output formats, if that matters to you.

I've been looking for a video editor on Linux for 15 yrs and still haven't found one that does what I need. Video editing is the one of two remaining tasks where Windows is easier than Linux for me. Plus, the video editor that I use will not run under WINE or in a remote VM install, so having a real Windows install is required here.  I should say that my needs are different than yours for this software.

Also, don't forget that production software, like OBS, might be useful too.  I use OBS to mux audio, and multiple video sources into those popular conference recordings seen all over youtube with the slides for 95% of the screen and the talking head in a corner ...
