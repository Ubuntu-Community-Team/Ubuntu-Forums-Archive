### Post by OutOfReach on 2008-07-15
[LMMS]("http://lmms.soruceforge.net") is a really good replacement for Fruity Loops, [Audacity]("http://audacity.sourceforge.net/") is an open source sound editor, [Hydrogen]("http://www.hydrogen-music.org/") is an excellent drum machine. I still haven't found a good replacement for Propellerhead Reason though. Audacity, and Hydrogen are available from synaptic, an older version of LMMS is also available in Synaptic, but you can find the latest version [here]("http://getdeb.net/app/Linux+MultiMedia+Studio")
