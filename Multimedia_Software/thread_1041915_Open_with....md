### Post by mc4man on 2009-01-17
Depends what your right clicking on.

In 8.10 if it's an icon for a mounted dvd video, vlc should be available. If not go to home folder -> edit -> preferences -> media and choose vlc from the drop down for dvd video. Then it should be available in the r. click on the icon.
(in 8.04 you need to manually add vlc to that drop down.

If it's a file then go right click -> open with -> open with other application and pick vlc from the list.
You can also set a file association (for double left click) by going r. click -> properties -> open with. (proceed to add if not seen there.
This needs to be done once per different file extension. 

For a folder go r. click -> open with -> open with other application. What you choose should be added to the 'open with' list. 
(if it isn't go back and pick something else, doesn't matter what. I think this just applies to 8.04, it's always 1 pick behind as far as adding to the list.

note that all folders are treated the same so don't change the default in the properties of any folder, it should remain with nautilus
