### Post by mardukvmbc on 2010-04-05
Hi everyone. I've been using (k)ubuntu for about 3 years without major issues until now.

Just bought a sony HDR-XR150 high def handycam. Thought since it had a USB connection (no firewire) that it would just mount as a drive and I'd be able to download the videos no problem.

Well, it turns out it's a big problem... all the apps capable of talking to video cameras seem to only be able to use firewire. The camera does indeed mount as a drive but none of the video files I found seem to work... I get some video on some of the files from vlc but no sound and nothing seems to be able to transcribe the video.

If anyone has any experience/success downloading from handycams via usb please let me know!

I'm currently on kubuntu 9.10 64 bit (2.6.31-20-generic)

Thanks

Update: Found a series of .mts files on the camcorder. Nothing would read them properly except kdenlive -- seems to be the only thing that works!

### Post by Bucky Ball on 2010-04-05
Like redrat said. Is there anyway you can change what file type you are recording to? It may be set to mts. Are you recording to a flash card? If so you could put that in the computer.  

Kino rocks for ripping stuff from a DV camera. Quick and easy once set up then edit in Cinellera or something else (or even Kino). It's in the repos.

Some vid cameras you need to convert to a readable common file type out of the camera and come with Windows (naturally) software to do this conversion.
