### Post by gandaran on 2008-12-28
the only wma format rhythmbox can't handle is lossless wma, to play the lossless format use a xine based player like amarok, mplayer can also play this format, when ripping a cd in windows media player just choose the normal wma format, you wont have any problem playing them in rhythmbox 
check if you have all gstreamer plugins installed or just install the ubuntu-restricted-extras, the 'ubuntu extras' package for intrepid 8.10 installs nearly all the gstreamer plugins in synaptic.
