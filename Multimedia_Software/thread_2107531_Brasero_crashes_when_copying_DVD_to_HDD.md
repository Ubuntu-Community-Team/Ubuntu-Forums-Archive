### Post by sudodus on 2013-01-22
I used Brasero for several years and liked it. But in 12.04 I've had problems.

Now I use *k3b* instead. It belongs to the KDE desktop environment, so installing it will bring a lot of kde infrastructure (system files). But if you have space for it on your hard disk drive, it is no problem.

```
sudo apt-get install k3b
```
