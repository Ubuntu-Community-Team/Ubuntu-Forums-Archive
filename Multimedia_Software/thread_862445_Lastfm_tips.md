### Post by CryptSphinx on 2008-07-17
you will need to set up a last-fm account to use it (go to the lastfm page and set that up first)

to use last fm on the desktop you can 

' sudo apt-get install lastfm '

from the terminal

or you can get it in synaptic by searching for last fm then installing from there.

once its installed, run it from the menu and the config is all wizard-straightforward.
