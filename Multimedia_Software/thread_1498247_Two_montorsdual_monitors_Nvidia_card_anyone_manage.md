### Post by gdonwallace on 2010-05-31
If you have not already done so, I would recommend downloading the nvidia-settings program from the repos.

I have dual monitors setup on my work machine, and use nvidia-settings to manage everything.  Works find for me.

My other suggestion would be to run the program from terminal and run it as root. (i.e. sudo ./nvidia-settings)  That way there should be no problems modifying the settings in this program and having them written out to the xorg.conf file.
