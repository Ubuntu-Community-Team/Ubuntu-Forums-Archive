### Post by gandaran on 2008-10-28
to install adobe flash just look for flashplugin-nonfree in synaptic package manager mark for install and click the apply button or go to adobe website [http://www.adobe.com/shockwave/download/download.cgi?P1_Prod_Version=ShockwaveFlash](http://www.adobe.com/shockwave/download/download.cgi?P1_Prod_Version=ShockwaveFlash) download the .deb package and double click to install, just make sure you haven't installed any other flash package or it wont work.
