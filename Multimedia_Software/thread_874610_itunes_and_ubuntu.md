You can get iTunes running on ubuntu with wine.

Works for me. I'm running Heron.

Download the installation file from apple. 
From the terminal 
wine iTunesSetup.exe

This will guide you through the setup normally.

When it's run, go to where iTunes.exe lives in the .wine directory.
~/.wine/drive_c/Program Files/iTunes

Then:
wine iTunes.exe

It works for me.
