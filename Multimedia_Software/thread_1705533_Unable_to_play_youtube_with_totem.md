### Post by lovinglinux on 2011-03-13
Get my extension FlashVideoReplacer for Firefox.

It allows to watch YouTube videos and a few other sites with a different plugin. Works best with gecko-mediaplayer.

It also allows to open the video with any external player and to redirect the pages to webm html5 page if you are using Firefox 4.

Make sure you get version 2.1.0, which has new cool features and improvements.

[http://www.webgapps.org/addons/flashvideoreplacer](http://www.webgapps.org/addons/flashvideoreplacer)
