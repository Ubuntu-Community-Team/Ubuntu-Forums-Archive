### Post by grege on 2010-04-22
other options

Decibel is a neat player that plays files not tags - although it can be configured for tags.

The new version of aTunes can also be configured to uses files instead of tags, it defaults to tags but can be reconfigured from the menus.

Decibel in in the Ubuntu repositories as decibel-audio-player

aTunes you have  to get from the author and manually install the deb. aTunes also needs a Java install.

[http://www.atunes.org/](http://www.atunes.org/)

The best audio player of all IMHO is Amarok, a KDE app that works fine in Gnome (ensure libxine-ffmpeg1 is installed).

Amarok uses tags to organize the library, but at the top of the left hand column you will find a file browser that allows you to bypass the lib and just load from files.
