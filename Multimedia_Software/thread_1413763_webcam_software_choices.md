### Post by zeroseven0183 on 2010-02-22
Have you considered Cheese (Webcam Booth)?
You can get it in the Ubuntu Software Center or type in the Terminal ```
sudo apt-get install cheese
```
You'll find it in the Sound & Video section of Applications after installation.

For more information about webcams, go to: [https://help.ubuntu.com/community/Webcam](https://help.ubuntu.com/community/Webcam)
Cheese: [http://projects.gnome.org/cheese/](http://projects.gnome.org/cheese/)
