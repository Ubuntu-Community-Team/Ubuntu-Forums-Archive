Dear musicians,

As we all know configuring a "studio" takes time, and it is not all through easy.

I would like to share my experience with Audacita, recording from external sound device.

I tried a lot of combination and the only one i got to work is the following.

JACK > AUDACITY

1. set-up jack
Start with making your jack set-up. Then close Jack.

2.
Open Audacita and use the same settings as Jack. Sample rate, buffer etc.
Make sure JACK is input and output. Close Audacity.

3.
Start Jack

4.
Start Audacity

5.
Record!

Hope that helps ya.
