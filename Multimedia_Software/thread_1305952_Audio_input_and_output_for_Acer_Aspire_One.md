### Post by irohaphoto on 2009-10-30
I am going to post this as a new thread because I think it may help people who may have audio input/output problems. I downloaded 9.10 3 days ago and I found that I was not getting any sound and the microphone was not working. I also thought that Skype did not have video but I learned how to check all those.
First I am using a Acer Aspire One KAV60 notebook dual boot with XP

If you are having trouble with sound goto your System> Preferences> Sound then click the Hardware tab and change the profile drop down menu I have mine set to Analog Stereo Duplex

Next for your mic:

in the same Sound popup click the Input tab and slide the slider over for the output volumn and the again for the input volumn be sure it is past the unamplified part and test using your external mic plugged in. Connector may need to be changed from Microphone 1 to Microphone 2 mine is set for Microphone 2. Then click Close to quit.

If you are comming from windows then Skype will be a little different
I downloaded the Skype for Ubuntu 8.04 just because I didn't know which one to try.

When you open the Skype window you get no menu bar like the version you get for windows for linux the main menu is a blue S on the lower left corner next to the green button is your add/search

Clicking on the main menu you can get a list of menus, Options will be for testing sound, video and other privacy settings.

Hope this helps

Have fun with 9.10!
