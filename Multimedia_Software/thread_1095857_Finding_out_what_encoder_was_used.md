### Post by x33a on 2009-03-14
you can find the info about the encoder only if it is included in the id3 tags by the ripper, afaik.

you can use any id3 tag editor to find detailed info about the tags, and songbird, amarok etc. also let you view the tag.

for even detailed info, you can install mediainfo.
