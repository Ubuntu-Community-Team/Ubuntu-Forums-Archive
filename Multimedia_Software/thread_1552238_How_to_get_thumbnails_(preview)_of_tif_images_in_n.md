### Post by tanoloco on 2010-08-16
I anser to myself :) hoping that this may help someone in the future.

Nautilus will create a thumbnail of tif images the first time you open it with gimp.

Obviously open and close all the pictures by hand could take a lot of time.

No idea at the moment if this could be automized using a script bash which open and close in gimp all the images in a folder just to permit nautilus to create all the thumbnails of a folder containing a lot of tif images.

No idea at the moment if there's a plugin of nautilus which can do it automatically.

Any suggestion is welcome
Cheers

### Post by cscj01 on 2011-11-03
In Nautilus preferences, choose size limit up to which you want to show preview.  Usually this size limit is much lower than most .tif images.
