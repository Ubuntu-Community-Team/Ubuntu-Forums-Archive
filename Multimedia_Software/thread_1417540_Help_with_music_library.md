### Post by cozmicharlie on 2010-02-27
What player you use really is a personal choice based on what features you need.  There are lots of really good players that range from the very simple (they just play your files) to those have tagging, organizing, converting integrated.  Rythmbox is the native player in Ubuntu and it works great, it has some integration for tagging and it can manage your ipod.  Others players include Quod Libet, Songbird, Banshee, Exaile and the list goes on. One of the most popular and probably the player that is most like itunes is Amarok.  It is for the KDE desktop but you can install it in Ubuntu gnome desktop.  If you also install soundKonverter, and K3b you can get converting, ripping and burning all integrated.  It requires a lot of dependencies so if you don't have much hard disc space you may want to go with something else.  If you tell us what features are most important to you then we can help to narrow your choices.  

Since you have all your files in itunes, make sure and install all the codecs you need.  This tutorial is excellent and will walk you through the process ([http://ubuntuforums.org/showthread.php?t=766683](http://ubuntuforums.org/showthread.php?t=766683)).  Make sure and install codecs for aac and alac (search in synaptic).

Hope this helps and enjoy.

### Post by ikacer on 2010-02-27
Choosing a music player depends on what you are looking for. There are quite a lot to choose from in, but if you have a really large library, you might want one that is light and fast.

One set of options is to use music player daemon (mpd), along with a front-end of your choice. Some popular ones are:
-Sonata - Python GTK+ Client
-ncmpcpp - NCurses Client

Another very lightweight media player worth checking out is [Goggles Music Manager]("http://code.google.com/p/gogglesmm/"). This one is very light and fast, manages large libraries well, and has good search capabilities. However, it is developed primarily for Arch Linux, so you'll have to get it from getdeb, as it's probably not in the Ubuntu repos.
