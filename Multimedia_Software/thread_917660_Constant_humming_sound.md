This doesn't sound like an Ubuntu issue to me; maybe broken laptop / power issues? Here's a couple of things you can check / try:

a) Switch off the laptop, unplug the mains, remove and reseat the battery. Restart and check if humming is gone.

b) If there is no humming, plug in the power adapter to check if humming appears. If it does, it is clearly a power related issue; you should get the power brick / socket checked.

c) Do you have Windows? If so, boot into Windows to check if humming is still present. If it is, then it's a hardware issue.

d) Boot off the live CD (This is a poor test, elimination only) and see if the humming persists... at what point does it start?
