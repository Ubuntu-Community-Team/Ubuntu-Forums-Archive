### Post by Tolstoy the Cat on 2011-06-14
I'm having awful trouble joining some avi files. The files were recorded on a Vado HD and have resolution 1280x720p, apparently. Each individual files plays just fine in mplayer.

If I use
cat file1.AVI file2.AVI file3.AVI > output.avi
the file size indicates that all 3 files have been joined, but when I view the output, it finishes after the first 2. If I join the first 2, then the second 2 and then try to combine the results, the output still ends after viewing the first 2 (the second 2 don't display).

Using avimerge creates files with no sound that can be played by VLC but not by mplayer. Using the -c option didn't help.

I had the idea of converting them to some other format with the same resolution using ffmpeg, but for some reason I keep losing resolution.

I'd be grateful of any further ideas.

Thanks.

### Post by FakeOutdoorsman on 2011-06-14
Can you show the output of:
```
ffmpeg -i file1.AVI -i file2.AVI -i file3.AVI
```
This will give some useful info about the files.
