### Post by Canaryguy on 2011-09-09
Hello,

For those free software lovers looking for an alternative to Skype here's one that I found.

It's called Linphone and you can download it from the site: [www.linphone.org](www.linphone.org)
In the same site you can create your sip which will be used as if it were your nickname so other people can add you to their contact list.

1º Once you have the sip open Linphone.
2º Go to Linphone>Preferences
3º Choose Manage SIP Accounts from the tags above
4º Press the Add button
5º Introduce after sip: your sip that you created in the first field. sip:name@sip.linphone.org
6º In the SIP Proxy address introduce: sip:sip.linphone.org
7º Press OK button and click on the Done button
8º Close Linphone and open it again. It will ask for the password that you chose when creating your SIP.
Done

In the main window of the program in the white area under Name and Present status you can make a right click with your mouse to add a contact.It will ask for a name (it can be any) and the sip.

When I try to use the program with the microphone integrated on my computer I get a lot of noise but connecting an external microphone like a headset everything works fine.

I hope you like it.
