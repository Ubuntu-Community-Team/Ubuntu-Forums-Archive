### Post by IntuitiveNipple on 2008-09-22
Does anyone have experience of, or can recommend, an external USB 7.1 channel sound device that works with Linux, for use with a laptop?

It will usually drive a 7.1 speaker system such as the [Creative Inspire T7900](http://uk.europe.creative.com/products/product.asp?category=4&subcategory=789&product=10321) when at home-base but might be used on-the-road too.

I've been doing some searching and not found many authoritative articles.

Things I *think* I need to know are what kernel module (driver) it would use (snd-usb-audio I suspect), what supporting utilities it might need, and what functionality might be unavailable without Windows drivers and utilities.

Probably the 'best' (most confidence-inspiring) device I've seen so far is the [Turtle Beach Audio Advantage SRM 7.1 Channel USB](http://www.turtlebeach.com/products/audio-advantage-srm/home.aspx) but I've not yet verified it works 100% with Linux.

Issues I've seen mentioned that I'd not have thought to check before buying are things like the fact it can't natively decode DVD digital surround-sound such as Dolby and DTS - it will only do pass-through to an A/V Receiver/decoder. That defeats the purpose of having a small, portable, USB device.

There's a thread in forums, "[Turtle Beach Audio Advantage SRM issues](http://ubuntuforums.org/showthread.php?t=600254)", where a couple of users mention trying one but it isn't clear if issues were resolved - no S/PDIF surround output.
