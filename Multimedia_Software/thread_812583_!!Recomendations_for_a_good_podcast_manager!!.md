### Post by bergersau on 2008-05-30
Bashpodder is as light as it gets. 
[http://lincgeek.org/bashpodder/]("http://lincgeek.org/bashpodder/")

Just add the desired feeds to the bp.conf file, one to a line and run the script.  It'll download into dated sub-folders for you.

Set up a cron job to run it automatically.

Though I run the script manually via a button in my gnome menu bar.

I prefer to delete podcasts manually - usually takes all of 2 secs to grab old dated subfolders and right click 'delete'.
