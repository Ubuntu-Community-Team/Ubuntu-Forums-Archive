### Post by tgalati4 on 2013-01-05
I keep most of my music on a [http://freenas.org](http://freenas.org) server.  It serves up UPnP, DNLA, mediafly music shares.  On each machine, point your music player library to the server and import (without copying).  That way each machine has local/fast searching, yet the music is always pulled from one place.

You can also use command line players like moc.
