### Post by ajgreeny on 2009-05-04
I think your url for the stream will need to be an audio stream, not just the web address it is from, as I think your example shows.  As an example here are two from the UK, one is an mp3 stream, the other a realplayer stream.

[http://media-ice.musicradio.com:80/ClassicFMMP3](http://media-ice.musicradio.com:80/ClassicFMMP3)
[http://www.bbc.co.uk/radio3/ram/r3g2.ram](http://www.bbc.co.uk/radio3/ram/r3g2.ram)

Both work without problem in rhythmbox.  I think you just need to go to the page from which you play the streams in Firefox and then right click on the stream link to get the proper address.  Interestingly, BBC has now started using a flash player linked into mplayer plugin for its streaming in order to show information, which is a bit confusing, but if you use the copy url from the mplayer part at the bottom of the player window, you can copy that straight into rhythmbox with no problem.  If you use another Firefox plugin I don't know how you can do the same thing but I'm sure it must be possible.
