### Post by c.los on 2008-12-14
The older Elisa (0.3.5) I'm pretty sure had UPnP support, both server & client, the new one (0.5) has not had the UPnP support implemented yet. I'm not sure what Fuppes is but I know you can mount a UPnP share with djmount ([http://djmount.sourceforge.net/](http://djmount.sourceforge.net/)). Try just mounting it then setting Elisa to use the mounted directory for searching audio/video.
