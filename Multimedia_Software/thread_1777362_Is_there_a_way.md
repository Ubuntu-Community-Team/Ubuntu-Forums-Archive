### Post by tilii on 2011-06-07
This is the way I choose to handle audiobooks. Bear i mind that I am not concerned with the real title of each track. I merely do it this way for speed of ripping as I find that many audiobooks don't always have detail on CDDB/Musicbrainz etc. I use Asunder by the way.
 

   I set the filename to be %L_%N (Album name_Track number)


   I manually type in the Artist name then for the Album name I type in Audiobooktitle01.. for the first disc.... then Audiobooktitle02 for the second etc etc.


   This way you can still tell the disc number and sort in the correct order even though the track numbers will reset to 01,02,03 etc when a new disc is put in.


   I find this works well and is obviously much quicker than perhaps typing in all the real track titles.
