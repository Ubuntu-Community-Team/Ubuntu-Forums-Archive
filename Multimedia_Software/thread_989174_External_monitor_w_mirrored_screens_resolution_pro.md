### Post by jecarr33 on 2009-01-23
I was having this issue.  For some reason I was not able to clone my monitors but was able to span them.  If you are using nvidia-settings, then the only way to get the "Clones" option on the configuration drop down is to enter it as root.

open up a terminal and type:

gksudo nvidia-settings

Once the settings open you should be able to enable or disable your second monitor.  Once this is done, you can use the configuration drop down menu and selects "Clones"

Once that is complete save that to your xorg.cong file.  It's always wise to save a back up of your initial settings first for later use.  

I hope this helps.
