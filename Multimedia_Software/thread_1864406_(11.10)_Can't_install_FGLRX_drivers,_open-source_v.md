### Post by Prodoc on 2011-10-19
It's a known issue: [https://bugs.launchpad.net/ubuntu/+source/fglrx-installer/+bug/871344](https://bugs.launchpad.net/ubuntu/+source/fglrx-installer/+bug/871344)

You'd be better off following the bug over there and possibly help out by supplying more info.

In the mean time try installing the proprietary drivers using the option "ATI/AMD Proprietary FGLRX driver" instead of "ATI/AMD Proprietary FGLRX driver (Post-Release Updates)". This has been known to work.
