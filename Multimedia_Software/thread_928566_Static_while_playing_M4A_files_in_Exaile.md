### Post by hobophobik on 2008-09-24
I was getting bursts of static during playback of iTunes DRM Free M4A files in Exaile.

Took me a while to find a solution, so in case it's of any use to anybody, I fixed it by running Exaile using "exaile --no-equalizer", as mentioned in this thread.
