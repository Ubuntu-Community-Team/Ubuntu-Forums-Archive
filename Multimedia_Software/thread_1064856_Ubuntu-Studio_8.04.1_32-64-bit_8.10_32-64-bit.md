### Post by wstout on 2009-02-09
It really all comes down to hardware and what will work best on your machine. I have 8.10 working well with the Studio components. From my tinkering 8.04 probably is a little more stable. If you are going to be using Ubuntu Studio for its intended purpose and doing a lot of audio editing etc., I really suggest the 64bit. I do some recording and there is a difference that you can tell between 64 and 32. 64 bit flash works as well as 32 for me and there is also new Sun Java 64 bit out there that hasn't been out long, not for sure if its in the repositories, but I have been well pleased with 64bit.
