### Post by eyeyoubin on 2010-04-14
Hello there.
There is a simple solution.
Go to Synaptic, install gstreamer0.10-plugins-ugly, few other packages will be automatically installed with it.
Remove all the music from the list, rescan the folder and you will be fine.
