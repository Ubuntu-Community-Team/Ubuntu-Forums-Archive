What mp3 in particular is this and what playlists does it use? I find songbird to be a nice simple player that works well on any platform really. Does regular static playlists and smart ones based on tags. Should do what you want, it's not in the repositories yet. Go to songbird [website]("http://getsongbird.com/"), download latest stable version to your home directory. Then simply extract it and double click on the songbird exectuable file located inside the extracted Songbird folder. You can make a launcher to this later.

The file you need to click in the extracted folder to be clear is (~ being your home directory):
```
~/Songbird/songbird
```

See addons for plugins that might help make syncing to mp3 player better, like FolderSync for instance.
