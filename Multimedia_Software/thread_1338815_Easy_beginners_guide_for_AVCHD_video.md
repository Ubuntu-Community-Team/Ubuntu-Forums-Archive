I don't know if there is a guide.

I just bought a Samsung F30, which produces the same type of file.  I use Video Converter, which is a GUI for ffmpeg, to convert to "dv" files, then devede can make a DVD from those files.  

I have also been using Cinelerra for video editing.  I think it's a fabulous program, although it takes some time to learn to use it.  From Cinelerra, I render to h.264 for uploading to youtube or vimeo.
