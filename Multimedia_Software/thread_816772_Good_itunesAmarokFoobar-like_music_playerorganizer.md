### Post by helino on 2008-06-03
I would definitely recommend Exaile, [www.exaile.org](www.exaile.org). It's similar to Amarok, but a little more lightweight and it uses Gnome as window manager. It should work great in Xubuntu. It also support "playlist-tabs" like Foobar does.
