### Post by theirishfozz on 2008-10-14
Hi

How did you share it on XP? Was it as network drive or did you have a custom client? Web Interface?

If it was just a network drive thats easy to mount in linux. Hop into XP (or check the dreambox docs) to find what path it is and google mounting a network drive in linux. Its fairly simple.

If its a custom client install wine (windows "emulator")

> sudo apt-get install wine

and then get hold of the client installer. Double click it to install on wine and you should be able to run it.

If its a web interface you should be able to access it in the same way as you did on XP through a file browser.
