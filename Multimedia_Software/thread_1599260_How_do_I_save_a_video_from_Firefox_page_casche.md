### Post by ron999 on 2010-10-17
Hi
Firefox stores streaming videos in a folder named **tmp**.
So when it's finished downloading you can 'copy' and 'paste' to put it somewhere else.

### Post by zuerston on 2010-10-17
Places==> Home Folder ==> up arrow 2 clicks from home, find tmp, drag movie out of tmp onto desktop when almost done playing watch for icon change and time left,Firefox will delete it from tmp but not from your desktop ,so drag it just before done playing,simple and it works great.
