### Post by ron999 on 2010-04-23
Hi
You can use mkvmerge.
It's part of the **mkvtoolnix** package in Ubuntu's repository.
It has a GUI, **mkvtoolnix-gui**. Fairly simple to use.
This will create an mkv file where you pick and choose which audio and video tracks it contains.
The mkv files are accepted by YouTube, but I don't know about other websites.

There's also MP4Box.
It's part of the **gpac** package in Ubuntu's repository.
This will make an mp4 file with your choice of audio and video tracks.
But I think that this is command-line use only. Not so easy to use.

### Post by Alan James on 2010-04-23
I am a video editor so I always like using an NLE to combine audio and video clips. I often use a Mac and Final Cut for doing this but have recently come across KDENLIVE, a very promising video editor for Linux. It doesn't satisfy all the needs of a professional video editor but it will allow you to visually merge the audio and video you are talking about. Try it out and see if it works for you.
