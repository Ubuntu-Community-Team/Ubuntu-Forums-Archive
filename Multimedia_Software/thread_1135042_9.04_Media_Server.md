### Post by LoermansA on 2009-04-24
Hi Jake,

I use Mediatomb as a UPnP mediaserver. It's in the repositories.

Give it a go. It has a web interface with which you can add/delete files and folders to it's database.

It doesn't do transcoding, but I hardly ever have the need to do so. The PS3 plays a lot of formats these days.
