### Post by tgalati4 on 2010-02-11
You can use any of the big music players (banshee, exaile, listen, rhythmbox) and duplicate artists and albums will show up (mispellings, caps vs lower case etc).  You can edit ID3 tags in any of these players.

Easytag allows you to rename files, directories, and move things around.  You can shut off automatic scanning which will speed things up.  Then only rescan once you have made a bunch of changes.

### Post by 2hot6ft2 on 2010-03-05
I used a couple tools to fix my collection

MP3diags to diagnose my collection and fix problems

mediamonkey from mediamonkey.com (it's a windows app but it may work in wine) to do the tagging, finding duplicates (by name and by content), synchronize tags and organizing them to a location like O:\Tunes\<ARTIST>\<TITLE>-<ARTIST>

There's also kleansweep for finding duplicates but I prefer using a media app for managing media files. It's in the repos.

Guess I should run mp3diags again after doing everything else I've done to my collection.

Oh yeah and TheRename to remove the track #'s it's a windows app but there's krename in the repos.
