### Post by tamirtoad on 2009-01-04
Hey guys,

I have a solution for this problem. I believe that the only thing was that banshee simply couldnt load the songs that had been loaded onto the ipod via itunes. i have an ipod nano 3g.

rythmbox can detect/delete/add new songs, but the disadvantage is that you dont get albumart.

banshee can add albumart, but it wont be able to load the songs that you put onto the ipod via itunes.

so heres what i did.

i opened rythmbox and dragged/dropped all files into a separate folder on my own computer. i then deleted every single file on the ipod.

then, i opened banshee and imported all of the songs in the separate folder. then i put those songs back on the ipod plus the songs that i wanted to add originally.

so pretty much from here on out im not gonna use anything but banshee to load songs onto my ipod. definitely not itunes.

i love coverart. i hope that helped.
