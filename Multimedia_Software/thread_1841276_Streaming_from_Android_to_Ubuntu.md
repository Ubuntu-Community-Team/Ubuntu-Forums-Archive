### Post by vfiz on 2011-09-11
The only program I know that works kind of like this is UPnPlay: [https://market.android.com/details?id=cx.hoohol.silanoid&hl=en]("https://market.android.com/details?id=cx.hoohol.silanoid&hl=en").  It allows you to stream music to other universal plug and play servers/clients.  I've only tried it with XBMC running on a Lucid-based HTPC, but it worked great.
