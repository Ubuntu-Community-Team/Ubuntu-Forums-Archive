### Post by RollerJ on 2009-10-10
[LIST=1]
[*]After you insert the Audio CD and have the tracks all listed in front of you, type the name of the track you want to rip in the Search bar above. You will see the other track names disappear with only the track you want to rip listed. Click on the Extract CD icon (it will still say "copy all tracks to library" don't worry it won't) and Rhythmbox will rip the single track only.
[*]:guitar:
[/LIST]
