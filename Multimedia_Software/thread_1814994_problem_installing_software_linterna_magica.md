### Post by hislordship on 2011-07-30
I posted a question on how to install linterna magica yesterday but got no answers so I figured it out elsewhere. Here it is for others to use.

Linterna Magica is a way to watch Youtube videos for those who wish to use free software only. Gnash doesn't work on youtube and the html5 trial doesn't always work so well either if you have a free version of the firefox browser. So it makes sense to install linterna magica for ethical reasons.

To do that you need to download the greasemonkey firefox add on first.

download this to your desktop: [http://download-mirror.savannah.gnu.org/releases/linterna-magica/linternamagica-v0.0.9-7.tar.gz](http://download-mirror.savannah.gnu.org/releases/linterna-magica/linternamagica-v0.0.9-7.tar.gz)

then follow the documentation here [http://linterna-magica.nongnu.org/#help](http://linterna-magica.nongnu.org/#help)
