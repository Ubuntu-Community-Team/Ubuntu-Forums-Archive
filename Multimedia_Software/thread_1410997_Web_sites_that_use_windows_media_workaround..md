I have found that if you use the mobile link on some web sites, you can listen to streaming audio that the site haves to ofter. 
For example [http://livefirefeeds.com](http://livefirefeeds.com) on the main page select  [http://stream1.livefirefeeds.com/](http://stream1.livefirefeeds.com/) then select the location you want to hear then select m3u format and the movie player will play the stream.
