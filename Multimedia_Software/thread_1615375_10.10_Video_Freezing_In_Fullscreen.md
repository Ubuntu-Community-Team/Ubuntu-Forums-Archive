### Post by Smari on 2010-11-16
I had the same problem and this worked for me:

```
sudo mkdir /etc/adobe 
sudo su 
sudo echo \"OverrideGPUValidation = 1\" >> /etc/adobe/mms.cfg 
```

I got this from [another thread]("http://art.ubuntuforums.org/showthread.php?t=1593363&page=2") here on the Ubuntu forums and it seems to be a common problem...
