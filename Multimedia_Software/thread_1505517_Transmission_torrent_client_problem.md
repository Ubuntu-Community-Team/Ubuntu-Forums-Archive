### Post by Aleksandar_B on 2010-06-09
I use Transmission torrent client to download torrents. I have noticed a weird and annoying thing, when I download a movie I usually don't mark all files in that torrent (extra subtitles, covers...) just the movie file. When the download has finished all files, even the ones I have not marked, have been downloaded. This happens only to some small files like before mentioned covers and subtitle files, not for example when there are two movie files in a torrent and I mark only one, only that one will be downloaded.

This isn't really a problem but I was wondering why is this happening?

---

### Post by Boondoklife on 2010-06-09
I think this has to do with how the torrent process works. Each person shares with the other "chuncks" of the torrent. In your case the small files were prolly part of the begining and end chunks so you got them by default.

This is not a flaw but just how the process works.
