### Post by volamoot on 2010-01-03
You should be able to play the .ogg file format by A)Opening the file at the file location, B) Using the import (or File Add) Feature that is on Amarok, Then selecting the file in the media library section, and either double clicking the item or by highlighting it and pressing play.

Good Luck
