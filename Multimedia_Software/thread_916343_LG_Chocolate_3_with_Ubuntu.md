### Post by Bakon Jarser on 2008-11-05
I'm a little late on the reply but I just got one of these.

Menu > Settings & Tools > Tools > USB Mass Storage. 

That will let you access it like a usb drive.  Put your music in the music folder.  I don't think it supports folders.  You could also use a program called bitpim.  Download the latest version from their site and run it as root.  It has reeeeeeeeaaaaaaaaallllly slow file transfer rate but you can sync the calendar, images, phonebook, and other stuff pluss add ringtones.

### Post by Bakon Jarser on 2009-01-12
> **mrzud said:**
> Placing the device in USB will allow you to see the files on the device, but it will not allow you to add files in a manner in which they can be played on the phone.

Following my instructions allows you access to the sd card only.  You can add .mp3 files (and only mp3 files, .wav and others don't work) into the music folder and the phone will be able to read them.  You can't add folders, it won't see any files in folders.  I assume you could also add pictures and other things to the appropriate folders if they are in the correct format but I haven't tried that. 

The only way I've found to access the phones internal memory is to use bitpim.  But once the mp3's are on the sd card you can use the phone to transfer them to the phone memory.

### Post by briandotcom0 on 2009-09-21
I have a chocolate 3, and I can add music (only tried mp3s) to the music folder.  plug it in and it recognizes a usb mass storage device.  If you have an SD card it only gives access to the card.  To access the internal memory, take out the card before plugging it in.  photos and videos work too.  Get Winff (sourceforge), it is a gui for ffmpeg, it has presets for converting music/audio for the lg chocolate 3, it has worked for me.  I cannot get bitpm to work for me.
