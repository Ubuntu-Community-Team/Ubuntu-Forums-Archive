### Post by cariboo on 2008-10-28
See as you are new to Ubuntu, you should stick wit Add/Remove Programs and Synaptic Package Manager. To install most of the plugins for media playback go to System-->Administration-->Synaptic Package Manager, click the search button and search for **ubuntu-restricted-extras** once the search is completed right click on the meta-package and chose install then click the apply button. Synaptic will take care of the rest, by downloading and installing the files.
