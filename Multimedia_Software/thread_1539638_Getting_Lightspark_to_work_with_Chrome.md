### Post by alexp-sssup on 2010-07-26
Verify that you have only browser-plugin-lightspark and that its version is 0.4.2.1. Restart Chrome from the terminal (after closing any running chrome instances) and see if any message related to lightspark is displayed. Lightspark should be displayed as "Shockwave Flash 10.0 r42" for compatibility reasons.

Also see this page [http://sourceforge.net/apps/trac/lightspark/wiki/Troubleshooting](http://sourceforge.net/apps/trac/lightspark/wiki/Troubleshooting)
