### Post by scratman on 2009-03-29
I have.  I installed the version of Wine in the Repositories, and set that up.  I then went to [http://www.mozilla.com/en-US/firefox/all.html](http://www.mozilla.com/en-US/firefox/all.html) and downloaded the latest version of Firefox for Windows.  I installed this in Wine, and then used Windows Firefox wo navigate to bbc.co.uk/iplayer and selected a programme that I wanted to watch.  I selected download, and it pointed me to the iplayer install page and advised that I had to install Adobe Air.  I clicked OK, and a few minutes later, Adobe Air started installing.  A few minutes later, iPlayer Desktop was up and running, and downloading the Grand Prix from this morning.
