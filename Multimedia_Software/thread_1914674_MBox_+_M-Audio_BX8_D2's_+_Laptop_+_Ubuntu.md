### Post by skinny honkie on 2012-07-18
Heya - 
I'm just here to add some observations, I don't have a full answer to your question.
I'm running 12.04 on a toshiba M5 w/60GB SSD and an original USB mbox with the focusrite preamps. As an audio output device, it's flawed but very interesting - sound quality is nice, and has a low noise floor. I find that the audio intermittently fades in and out of the left-hand output channel, but wiggling the usb cable generally fixes it temporarily. If I plug in a USB HDD into an adjacent USB port, the problem happens more frequently, so I'm guessing that this relates to power draw across the USB bus.

Without any tinkering, it does not function as an input device, but I haven't explored this to any depth yet. If I get anything happening I'll update the thread :)

Overall, it's nowhere near as solid or functional as an ice1712-based setup (eg delta 1010) on linux but the sound quality is good when it works.
