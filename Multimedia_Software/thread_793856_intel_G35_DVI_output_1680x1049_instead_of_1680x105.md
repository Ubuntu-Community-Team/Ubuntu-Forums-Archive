### Post by wkulecz on 2008-05-14
Due to lack of solutions for only getting 1600x1200 on a 1920x1200 LCD monitor using a DVI cable, I'd suggest using the analog VGA cable.  

To be honest, I can't see any image quality loss -- we've got a dual screen "clone" setup in the lab using a $50 50ft cable from Frys for the remote cloned desktop and it works wonderfully at 1920x1200.
