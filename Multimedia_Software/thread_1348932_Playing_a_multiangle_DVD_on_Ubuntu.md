### Post by mc4man on 2009-12-07
I would think mplayer could, either from the start or using the keyboard while playing ( maybe A on keyboard  ..?

or 
mplayer dvd://[COLOR="Red"]X[/COLOR] -dvdangle [COLOR="Blue"]X[/COLOR]

where red = movie title number and blue = angle number
