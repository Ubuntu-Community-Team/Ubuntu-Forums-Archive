### Post by lovinglinux on 2009-04-29
If you can play your TV with TVTime, then you probably can record it with [FoxMediaCenter](http://fmc.isgreat.org/), which is my media center extension for Firefox. It can record TV series weekly, schedule recordings manually and record on-the-fly. It also has xmltv support for scheduling, playlist manager and other stuff. I designed it to be a simple and lightweight media center.

The program is still beta, but it works great for me. So please don't discard it so fast. If you have issues, please ask me and I will try to help you. I suggest starting reading all the stuff in the site, so you can understand how it works, the requirements and limitations.
