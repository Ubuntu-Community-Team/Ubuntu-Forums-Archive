### Post by Cybie257 on 2009-04-17
First thing, turn off the Desktop effects completely. That seems to create a lot of video issues. Just last night, I was able to watch YouTube vids without any degradation when in there native size. Switching to full screen, things got choppy and out of time. When I disable desktop effects, full screen is now just about perfect.

System->Preferences->Appearance
   Visual Effects (TAB)
     Select "None". 

See if that fixes the issues. :)
