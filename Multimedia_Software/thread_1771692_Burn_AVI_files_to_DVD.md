### Post by akand074 on 2011-05-30
> **mrmjtudor said:**
> I have been searching the forum and elsewhere for advice on simply copying AVI files to DVD - no menus chapters etc just a simple copy.

Any ideas

Do you want to be able to play it on a dvd player? If so you can try using DeVeDe (from the software centre), put the settings for no chapters/etc. and have it make the disk structure (or ISO or whatever you prefer) then just burn the file(s) it makes to a dvd using any burner (brasaro is the default). Otherwise you can easily just make a data disk using brasaro.
