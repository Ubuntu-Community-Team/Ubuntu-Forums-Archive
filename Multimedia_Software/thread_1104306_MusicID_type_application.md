### Post by R4kk00n on 2009-03-25
If you just want your files to have correct tags and names try [MusicBrainz Picard](http://musicbrainz.org/doc/PicardTagger). It's in the repos.
And if you want something more interesting, there is [MusicIP Mixer](http://www.musicip.com/mixer/index.jsp). It can fix your tags, but its real strength is in finding similar music in your collection and creating playlists out of it. I haven't tried the Linux version (which is somewhat limited compared to Windows one), although I've extensively used the Windows version for more than a year.
