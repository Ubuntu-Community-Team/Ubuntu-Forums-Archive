### Post by Mark Phelps on 2011-05-16
You could split the original file into pieces and burn each piece to a DVD.  

Look around for HJ Split.  There is a java-based version that will run in Ubuntu.  You have to install the Java infrastructure.

But then, you will also need HJ Split on any machine you will intend to use to re-join the pieces.  They pieces can not be used individually.
