### Post by pb3 on 2009-12-27
I had the same issue.

Open terminal
Type Alsamixer
Highlight Headphone jack sense (use arrows)
use m key to toggle on/off

Check out this thread for more info if that doesn't solve your issues. 
[http://ubuntuforums.org/showthread.php?t=1315138](http://ubuntuforums.org/showthread.php?t=1315138)
