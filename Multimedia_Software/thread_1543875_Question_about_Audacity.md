### Post by diablo75 on 2010-08-01
I've been doing a little music production lately and I just finished recording a piece of keyboard to go with a song someone else recorded.  We've been collaborating online.

So I have two tracks, one keyboard and one the rest of the song (guitar, vocals, drums).  The keyboard track audio consists of a WAV file that is shorter in duration than the rest of the song, and doesn't have all the padding it needs at the beginning in order for someone else to take the track and slap it next to the original and they're already in sync.  As it stands now, my track comes in early if not for manually moving the audio further down the timeline.  So I was wondering if there is a quick and simple way to insert silence into any gap that comes before the audio file begins in the timeline, so that it can be exported back out as a new wav with additional padding at the beginning and allow others to quickly drop it into their own mixing software.

---

### Post by NFblaze on 2010-08-01
The File Menu bar has a  Generate > Silence

It lets you select for the duration of seconds you want.
