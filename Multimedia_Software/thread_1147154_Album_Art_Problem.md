### Post by canadiandude007 on 2009-05-04
Which program are you using?  Different programs behave in different ways, so it would be good to narrow down which ones are having issues.  You mentioned a few, but for example, are you getting an error?  Have you tried downloading album art different ways?  What have you tried?

For example using Listen, you can download Album art by right-clicking on a blank album cover or go to a menu item to download all missing album art.

For Amarok you can use script manager to install scripts to pull album art from other sources.  You can right-click on a blank album and then search Amazon for album covers.

We need more information from you.

Have you tried to launch the program from the command line to see if errors are generated when you attempt to retrieve album art?
