### Post by albeano2004 on 2008-06-04
Hi jaime256,
Please try the following:
-Go to System->Preferences->Removable Drives and Media
-Untick the box marked 'Import digital photographs when connected'
-voila!

Hope this works for you, if it does please post, for future reference.
If not, let us know.

### Post by xc3RnbFO8P on 2008-06-05
Try **File Management** System> Preferences> File Management 
If you cant see File Management goto
System> Preferences> Main Menu
In Main Menu System> Preferences, check box File Management
In File Management Media tab Photos choose do nothing.
