### Post by DizzyTech on 2008-06-27
To make sure all the tags keep, go to your Windows Media library and go to Library (the little drop down arrow, if you're in WMP11) >> Apply Media Information Changes.  Then, just copy all of them to a DVD or external hard disk for when you install Ubuntu.  When you're in Ubuntu, install the package "ubuntu-restricted-packages".
