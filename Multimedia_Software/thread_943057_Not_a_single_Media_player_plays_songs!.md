### Post by WWSmith36 on 2008-10-09
I think ubuntu does not come out of the box with mp3 support.  Legal issues and all.  You have to enable the multiverse repository in the software sources and install
```

sudo apt-get update
sudo apt-get install ubuntu-restricted-extras
```

Hope this helps
