### Post by sudodus on 2012-12-26
I use ***DigiKam*** to organize my pictures. It has more options than Shotwell, and it should be rather easy for you to make the slideshow you want instead of creating a huge impress file.

You can also use ***feh***, which can use a filelist, a file containing the names of the files you want to show in the order you want to show them.

You can create that filelist more or less manually with a standard text editor.

Start with a list of the files created with for example a ***find*** command line!
