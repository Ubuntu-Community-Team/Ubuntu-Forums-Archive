### Post by cariboo on 2008-11-03
If you need to run vlc as root use vlc-wrapper, have a look at the man page here:

[http://manpages.ubuntu.com/manpages/intrepid/man1/vlc-wrapper.html](http://manpages.ubuntu.com/manpages/intrepid/man1/vlc-wrapper.html)

It tells you how to use it.

There should really be no need to run it this way on your own computer, but if you need to run it as root on a server to stream media, this is the way to do it.
