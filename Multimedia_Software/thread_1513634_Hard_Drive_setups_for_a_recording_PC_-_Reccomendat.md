### Post by cchhrriiss121212 on 2010-06-20
I built my own machine for this last year, and have been happy using a single 7200rpm drive for everything. This is probably because I do not have huge amounts of audio files in a single project (max of roughly 30), or possibly because the effects are minimised (or not present) with jack. I saw the same 2 hard drive advice when I was researching but all the users seem to be on windows systems so I tried out with a single drive. But if you have the money I would definately go for an SSD for root and the rest for home.
What makes the biggest difference will be your soundcard, what are you planning on using?

p.s. these types of audio recording questions are best posted in the studio forum (even if you don't use studio), a lot of knowledgeable chaps in there
