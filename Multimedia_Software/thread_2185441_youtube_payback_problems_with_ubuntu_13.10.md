### Post by lah7 on 2013-11-02
Let's find some exact details first. **Which web browser are you using? What's your graphics? Do you have any additional drivers installed?**

I can suggest you try another web browser to see if the problem persists. If your entire session freezes up, this could be a bug with the graphics.

Alternately, you may want to try turning on HTML5 in YouTube to identify if this is a Adobe Flash issue, you can do this via [http://www.youtube.com/html5](http://www.youtube.com/html5) (assuming the web browser is HTML5-compatible) A web browser like Google Chrome (not Chromium) bundle their own flash player, so for all other browsers like Firefox, you will need to double check the flash plug-in is properly installed.
