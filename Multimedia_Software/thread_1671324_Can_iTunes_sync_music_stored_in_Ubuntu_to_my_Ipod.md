### Post by tommcd on 2011-01-19
If you want to boot Windows and have the itunes installed on Windows read the music on your Ubuntu partition, I don't think this will work since Windows will not read linux file systems.

Itunes will not work on Ubuntu. There are posts on the net about getting itunes to work on Ubuntu with wine: [http://www.jonathanmoeller.com/screed/?p=2091](http://www.jonathanmoeller.com/screed/?p=2091) However, from what I have read it does not work well. Running Windows apps on linux with wine is hit or miss.
You are better off using a native linux app to use your ipod. See this: [http://www.psychocats.net/ubuntu/itunes](http://www.psychocats.net/ubuntu/itunes)
AmaroK and Banshee are said to work well with the ipod.
I would also recommend installing the latest version of Ubuntu, which is 10.10.
