### Post by jaminthorns on 2010-07-12
You should try Audio Tag Tool, just search 'tagtool' in synaptic. Once you've opened it up, go to the 'Rename Multiple Files' tab and enter in a format like '<album>/<title>' or something like that. It will make more sense if you click the help button to the right.

### Post by Chronon on 2010-07-13
EasyTAG is a good program.  It allows both renaming of files based on metadata and tagging based on path/filename information.

If you set your filters correctly, you should be able to get information from the entire path of a file.
