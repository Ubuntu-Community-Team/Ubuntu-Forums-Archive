### Post by scotty2718 on 2010-08-24
Alternatively, if you would still like to have the notifications and tray icon (as I do), you can configure how the status icon works as follows:

Open Rhythmbox
Click Edit->Plugins
Scroll down to the "Status Icon" plugin and select it
Click the "Configure" button

You can now edit how you want the "notifications" and "status icon" to behave using the drop-down menus.  If you want Rhythmbox to quit when you press the close button and to always start maximized, you must change the "status icon" to something other than "Owns the main window"
