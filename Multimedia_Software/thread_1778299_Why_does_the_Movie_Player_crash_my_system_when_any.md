### Post by Crusader911 on 2011-07-15
New user here, running on a Commodore 64x with Ubuntu 10.10.  If I touch any of the controls in Movie Player it freezes my whole system.  I haven't found a solution, but I do have a tip for your computer freezing.  Hold down Alt and PRTSC while clicking each of these keys in order: R,E,I,S,U,B.  Pause a few seconds between each click.  It's BUSIER spelled backwards, or you can remember Reboot Even If System Utterly Broken.

If you just want to shutdown because you've had enough Ubuntu frustration for one day, replace the last letter with an O.
