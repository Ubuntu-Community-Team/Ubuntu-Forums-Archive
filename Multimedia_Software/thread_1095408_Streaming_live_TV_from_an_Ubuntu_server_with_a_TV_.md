I'd rather have the new machine run on Ubuntu, but I've been unable to find any program that will perform the needed Windows Media Center functions. I've seen uShare and Twonky, but neither of them seem to stream from TV tuners.

### Post by lovinglinux on 2009-03-13
I guess what you are looking for is [MythTV]("http://www.parker1.co.uk/mythtv_ubuntu.php"). You can replace the Windows Media Center server with Mythbuntu or Ubuntu+MythTV.

You could also check [XBMC]("http://xbmc.org/"). I'm not sure if you can stream live TV with it, but you certainly can stream media files. You could also replace the Xbox software with XBMC.

### Post by steefjeqv on 2009-03-14
Hi,

VDR is being integrated in XBMC (linux version). This will allow you to watch live TV on XBMC. At the moment it will require some patching and compiling. 

VDR was originally developed around the DVB standard. I believe it still needs a patch or a plugin to use ATSC (not sure though).
