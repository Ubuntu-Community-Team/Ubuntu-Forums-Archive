### Post by ajgreeny on 2008-08-18
Use Alt+F2 and the command ```
gksudo displayconfig-gtk
``` to get a clue what driver you are using.  With xorg in Hardy the ```
sudo dpkg-reconfigure xserver-xorg
``` is no longer of use, I'm afraid.

### Post by Kevbert on 2008-08-18
You could try manually modifying your xorg.conf file so that it includes a driver:
```
Section "Device"
	Identifier	"Configured Video Device"
       [COLOR="Red"]Driver          "ati"[/COLOR]
EndSection

```
Hopefully that should help (it's a driver that comes with the kernel).
