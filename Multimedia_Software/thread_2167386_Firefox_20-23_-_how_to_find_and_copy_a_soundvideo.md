### Post by SeijiSensei on 2013-08-13
Many videos are distributed using the HTTP [headers which prohibit caching]("http://www.w3.org/Protocols/rfc2616/rfc2616-sec14.html#sec14.9").  I believe this results in the cached file being deleted when it is viewed.
