### Post by mrksem454 on 2010-05-06
In Totem Movie Player, you can install multiple plugin packages like,
gst-plugins-base-the basic and essential plug-ins for GStreamer,
 gst-plugins-good-the plug-ins for most Open formats,
 gst-plugins-ugly-good-quality plug-ins that might pose distribution problems, needed for DVD playback and
 gst-plugins-bad-a set of plug-ins that need more work, needed for DVD playback as well.
