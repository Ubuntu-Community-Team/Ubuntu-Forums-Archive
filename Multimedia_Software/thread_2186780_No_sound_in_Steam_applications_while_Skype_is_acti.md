### Post by Shpitzick on 2013-11-09
Hello!

When I order my steam to set PulseAudio Default as the sound device, there is no sound in steam games but there is in Skype.
When I set it to ALSA Default there is sound in steam games but not in skype.

Any solutions?
Thank you!

Edit: SOLVED IT! YAY!
I followed [this]("https://support.skype.com/en/faq/FA10964/can-i-change-the-sound-system-used-by-skype-for-linux") to disable PulseAudio and set Skype to use ALSA, then set Steam to use ALSA. Now I can hear GLaDOS <3
