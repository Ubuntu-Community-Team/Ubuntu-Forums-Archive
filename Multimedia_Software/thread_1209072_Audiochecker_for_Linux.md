### Post by cascade9 on 2009-08-12
I've found audiochecker runs under WINE. Its not perfect, I get more 'could not qualify the source of this track' results, but thats the way it goes. 

Tau Analyzer is what audiochecker uses to detect the source of the files, and there is a .tgz version for linux-

[http://true-audio.com/](http://true-audio.com/)

Command line only though (unless theres a GUI for it somehwere, I'm looking LOL) 

BTW- audiochecker only tells you if it is a CD source, vinyl rips and I assume other lossless files, e.g. those sourced from DTS-HD master will fail (not that I've ever seen  a DT-master file in the wild)
