### Post by Rob Sayer on 2015-05-29
There's also dvd styler, also in the repos.  Never used devede personally.

The OP has the typical question from people who haven't done much encoding.  'Burning' an mp4 file to a dvd that can be played in an old dvd player is not a one step process.

Whatever you do, if you care about the video output quality at all, find the setting for 2 pass encoding and switch it on.  If you don't use 2 pass I guarantee the result will look like crap.

In the windoze world there are lots of cruddy "simple 1 click" tools for this.  They're all rubbish.
