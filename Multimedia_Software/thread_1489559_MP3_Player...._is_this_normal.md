### Post by sxmaxchine on 2010-05-21
the 200mb is probably the mp3 players OS. there are also files stored on it so the computer can recognise it as a usb.

You will notice when the size of the drive increases that amount you cant use increases as well. eg. a 128mb usb might only let you use 120mb while your 2gb mp3 only lets you use 1.8gb

### Post by julio_cortez on 2010-05-21
Unfortunately the bigger is the size of the disk, the bigger is the size you "lose" if the nominal size has been calculated with the "simple" mode.

The "right" size indeed is the one shown as [GiB]("http://en.wikipedia.org/wiki/Gibibyte")s instead of [GB]("http://en.wikipedia.org/wiki/Gigabyte")s.
