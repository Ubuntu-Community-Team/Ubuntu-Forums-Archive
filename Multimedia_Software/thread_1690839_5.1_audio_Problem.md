### Post by lidex on 2011-03-24
Through digital or analog?
Use the alsa-info-script to provide detailed information.
Run this command in a terminal:
```
wget http://www.alsa-project.org/alsa-info.sh -O alsa-info.sh && bash alsa-info.sh 
```
Choose the upload option and provide a link for the output.
[Terminal="Applications->Accessories->Terminal"]

### Post by lidex on 2011-04-05
First try updating your system. You are behind a couple of kernel versions:
```
sudo apt-get update
```
```
sudo apt-get upgrade
```
```
sudo update grub
```
Now move/rename ~/.asoundrc
**Reboot**

Re-run the alsa-info script so we can see where we are.

Models for alsa-base.conf - (we'll get to this if needed):
```
3stack-6ch
3stack-6ch-intel
auto
```
