### Post by irv on 2010-06-22
If you want to make a Ogg videos with Firefogg you need to install the plugin for firefox. If you go to the link below and about 50% into the document you will find a howto: It will tell you how to install Firefogg and how to use it to convert using video codes theora and audio codecs vorbis. Your video file will be locally encoded and not upload to any server. I believe it can then be incorporated into HTML5 with <video> tag.

[http://diveintohtml5.org/video.html ]("http://diveintohtml5.org/video.html")
