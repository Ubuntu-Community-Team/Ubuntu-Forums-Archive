### Post by dragos240 on 2009-09-17
Well. There are multiple tools for ubuntu for what you are trying to do. :) Also the latest version of itunes works with WINE. Open up a terminal and do this:
sudo apt-get install wine

and then after you hit enter, enter your password you use to login. From there, download the windows version of itunes, and you can use it.
