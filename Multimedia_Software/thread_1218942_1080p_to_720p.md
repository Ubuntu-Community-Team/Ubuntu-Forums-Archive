### Post by lykeion on 2009-07-21
For a simpler conversion with a GUI interface, you can use **avidemux**. 
Select Auto > "Playstation Portable (H264)" to get the resizing to 720p correct. You may want to change back Audio to "Copy" and Format to "MKV" 
at the left dropdown-menus.

The conversion will take a very long time (especially on older computer). So maybe you should try out the conversion on just a small piece of the movie first. 

You should be aware that all transcoding will downgrade the video quality.

### Post by FakeOutdoorsman on 2009-07-21
If you're using Ubuntu 9.04 Jaunty Jackalope you can try FFmpeg:
```
ffmpeg -i input.mts -acodec copy -vcodec libx264 -vpre hq -s 1280x720 -threads 0 -crf 24 output.mp4
```
If you just want to try a test encode of 30 seconds add the "-t" option to the input:
```
ffmpeg -t 30 -i input.mts -acodec copy -vcodec libx264 -vpre hq -s 1280x720 -threads 0 -crf 24 output.mp4
```
