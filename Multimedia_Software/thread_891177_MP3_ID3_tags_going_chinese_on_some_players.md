### Post by Gadgetech on 2008-08-18
Try changing the ID3v2 version in EasyTag from ID3v2.4 to ID3v2.3 (or vice versa) in Settings -> Preferences -> ID3 Tag Settings.

Using ID3v2.4 on my Sansa made everything show up as "Unknown".
