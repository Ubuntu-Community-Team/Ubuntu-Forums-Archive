### Post by gandaran on 2008-11-30
yes. with any player, totem, vlc, mplayer and many others (only DRM protected wmv/wma files don't work)
to install codecs, get the medibuntu repo installed, [https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)
install the ubuntu-retricted-extras and  w32codecs from medibuntu
type codecs or windows codecs in synaptic package manager search box and install anything that smells codecs there!
