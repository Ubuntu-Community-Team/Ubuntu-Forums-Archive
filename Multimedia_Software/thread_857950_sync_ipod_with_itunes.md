### Post by OutOfReach on 2008-07-13
Well you could sync your ipod with open source alternatives to iTunes, like rhythmbox (installed by default), banshee, amarok, and others. But if you REALLY want iTunes and nothing more, you'll need to install WINE (follow the instructions here: [http://www.winehq.org/site/download-deb](http://www.winehq.org/site/download-deb) ), then Download and install iTunes from the Apple website. Just a note: sometimes iTunes will not work flawlessly through WINE, so I recommend the open source alternatives to iTunes.
