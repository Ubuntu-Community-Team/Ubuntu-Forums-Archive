### Post by norwoods on 2009-03-31
run gksu nvidia-settings
in nvidia-settings, click X Server Display Configuration in the left column.  on the right you should have your two screens and a box next to the label: Make this the primary display for the X screen.  
click on the display you want to be the primary display.
click on the box next to the label: Make this the primary display for the X screen.
click Apply
click on Save to X Configuration File
click Quit 
reboot
