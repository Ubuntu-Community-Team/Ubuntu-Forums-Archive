### Post by Rhubarb on 2011-01-20
I've had success with this using a jabber server with empathy.

The jabber server that I use is called prosody
[http://www.prosody.im/](http://www.prosody.im/)
It works well with ubuntu.

Once you've got your jabber server installed and running, point empathy (the built in IM client in ubuntu) on both computers to your server.

Then add each other's contacts to each other, then right-click on the contact and select "Video Call" and there you have a live webcam feed of each other :)

Note that this will work fine once it's all set up - after which no internet access is required.
I have had a few occasional reliability problems with video chat in empathy on ubuntu 10.04.  - Ubuntu 10.10 runs rock solid :)

This solution is a little complicated to get going, but it works extremely well, and unlike "nearby" option in empathy this actually does work with a webcam.
