### Post by Tocci on 2008-09-10
I am a new user and was having the same problem.  I have had some luck by doing the following....

1  Set Rhythmbox to look in the shared folder for the music.  Edit-Preferences-   Music
2  Go to Edit-preferences-Playback     Check the box that says "use crossfading backend"
3  Play with the network buffer slider.  I put mine around 150 seems to work well unless I'm going online.  Then I have some playback issues.  I am streaming it to a wireless laptop, so that might be some of my issue.  

Hope this helps, let me know if you have more questions.  Like I said I am new to Ubuntu just trying to give back a little.
