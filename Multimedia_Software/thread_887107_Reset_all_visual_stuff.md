### Post by cariboo on 2008-08-12
All of your desktop settings are in the hidden files in your home directory.

I have a separate /home partition and when I upgrade or install a new version everything in my home directory stays as it is. I usually delete all of the hidden files when I install a new version except for ~/.mozilla and ~/.mozilla-thunderbird because I have had at times problems with different version screwing things up.

If you do backup your hidden files and install different desktop themes or other enhancements, when they get screwed up you can always revert to your backup and start all over again without having to do a reinstall.

Jim
