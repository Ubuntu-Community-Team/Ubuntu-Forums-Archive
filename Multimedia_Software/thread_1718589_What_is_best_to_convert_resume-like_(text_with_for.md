### Post by rocksockdoc on 2011-03-31
Apparently, OpenOffice, with the beta "Oracle PDF Import Extension", can edit PDF:
- [http://extensions.services.openoffice.org/de/project/pdfimport](http://extensions.services.openoffice.org/de/project/pdfimport)

In addition, the Oracle PDF Import Extension can save to "PDF/ODF hybrid  file" format, but I'm finding the formatting not to be as useful as I'd  like as it apparently saves to ODF odg format, instead of to .odt or .doc formats.
