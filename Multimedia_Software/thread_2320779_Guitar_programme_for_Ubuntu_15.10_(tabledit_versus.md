### Post by Finn bjerke on 2016-04-17
1) In Ubuntu 15.10 tabledit works under Wine using MIDI sound. Demo version has no sound youll have to pay for Windows version to get MIDI sound. TIMidity need to be installed too which is done from Ubuntu softwarecenter.
2) Guitar pro 32 bit version does not work in Ubuntu unless you tweak it, (its not overly diffifult) but printing option did not work properly 
3) Guitar pro does not support  64bits versions of Ubuntu, so you need a lot a tweaking in order to use 32bit architecture or alternatively dual booting using a 32bit Ubuntu side by side with 64 bit (I have not tried that) 

You can use sweet tuxguitar for bulk-converting files to midi which will be imported by tabledit. Tuxguitar is part on the canonical family.

All in all I recommend tabledit, its user friendly not overly expensive, the guitar pro 6 tech people are not helpfull at all. I installed tabledit  with no problem under Wine, unzipped installed and sound is there. The programme is user-friendly and intuitive, Guitar pro is not. Guitar pro has better sound but it is complicated to use. 

[http://www.tabledit.com/download/linux.shtml](http://www.tabledit.com/download/linux.shtml)

Youll find a lot of arrangements for stringed instruments all over the internet, classical guitar, bossa nova guitar, banjo, bass etc.   With tabledit youll get tabs and sheet music, it work very well and its not overly expensive. 

I love tabledit, I have no postive things to say about guitar pro 6 except nice sound. Its not user friendly and the support is arrogant and silly.

Music for tabledit here: (test the demo if you feel motivated) 

[http://freetabs.org/listtabs.htm](http://freetabs.org/listtabs.htm)
