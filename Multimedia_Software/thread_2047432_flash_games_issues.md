### Post by houseworkshy on 2012-08-25
If you are useing firefox with no-script there will be a bar at the bottom of the browser window.  On the right of that is an options tab. click on that.  That will open a window, in that window are several choises.  One of those is also called options.  Take a look at the whitelist in that and see what is allowed, personally I don't allow anything globaly and remove the lot.  If you do that too when you go to a new page the bar appears at the bottom of the page and clicking options shows you what scripts the page wants to run.  You then have the choice to allow temporarily or add to the white list. If everything is allowed on a page you visit the bar will not appear but you can still get to it via an icon to the left of the address bar, it's a sort of blue 'S' with a red sploge.
Another nice feature is that when a script is blocked middle clicking on it's name opens a new browser tab with links to find out about it, so even if you haven't got a clue you can quickly find out.  Quite interesting in a disturbing sort of way seeing just how much muck is out there.
Java is pretty much a virtual machine so even though linux itself doesn't suffer much from malware ( there is some ) browsers running on linux are no safer than they are on anything else.  Definately control java irrespective of the OS.  No-script does this and more, it's a good add-on which I recomend, I infer that the ubuntu maintainers  think so too as it is one of the few add-ons which is actually in the  repositories.  Another one worth haveing, especially for youtubeing and games is better privacy, that one deletes "lso's" which are very long term supercookies, some never expire at all. 

 You could also take a look at the browsers own preferances, it's less likely to be the cause as youtube works though it just may be a blocked attack site or wants to do more with java than you have set for. Sometimes clicking on a flash game that doesn't play live will give you the chance to download the .swf and you can play offline. 

If none of the above works this thread may help  [http://ubuntuforums.org/showthread.php?t=1712247](http://ubuntuforums.org/showthread.php?t=1712247)
