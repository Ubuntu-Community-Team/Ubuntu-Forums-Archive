### Post by ajgreeny on 2010-04-29
Desktop effects are the 3d desktop things you can get with compiz, such as the desktop cube, wobbly windows etc etc, which are enabled to "normal" on a default install if the system thinks your graphics card can run compiz.  The problem can be that systems can be not quite good enough to run compiz properly, and can then cause these sorts of graphics problems.

Go to System->Preferences->Appearance and set to "none" to see if things improve.  You can also get to that point from a right click on the desktop->Change Desktop Background -Visual Effects tab.
