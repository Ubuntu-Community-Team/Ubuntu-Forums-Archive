### Post by aeiah on 2011-08-02
it depends how fancy you want things to get. you could just set up a web server and point the document root to your video folder. people will just see a rudimentary list of urls leading to the video files, then they could just stream them with whatever video player their pc has.

you could write a script to create a web page with clickable thumbnails of the videos or something too i guess, but there's probably software out there for web servers that do this kinda thing.

or are you after something with embedded videos?

or you could set up an upnp server like mediatomb, but that would require the clients to have an upnp compatible player.
