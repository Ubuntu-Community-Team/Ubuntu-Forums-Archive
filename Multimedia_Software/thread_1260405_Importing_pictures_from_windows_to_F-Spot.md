Just wanted to update the above post.  I copied all of my pictures, videos, documents, etc. onto some disks. It not only imported the pictures, but it imported all of the name tags that I had with the pictures.  I haven't tried to import my videos or anything like that yet, but hopefully that will go just as smooth.
