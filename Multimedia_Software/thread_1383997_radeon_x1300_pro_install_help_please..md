### Post by sports.car.guy on 2010-01-17
If it is installed, before removing it try to solve it running slow. It might be worth it.

Did you write the xorg to use the fgrlx driver?

You can do so with

sudo aticonfig --initial

or

sudo aticonfig --initial -f

The -f writes a completely fresh xorg.conf.
