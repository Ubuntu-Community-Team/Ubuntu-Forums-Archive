### Post by SuperSonic4 on 2008-09-05
Does right clicking and then selecting *DVD Menu* work?

Personally, I'd use VLC instead of Totem, I find it far superior in every way.

```
sudo apt-get install libdvdcss2 libdvdread3 libdvdnav4 vlc
```

you may also need regionset ```
sudo apt-get install regionset
```

to set the region type ```
sudo regionset
``` in terminal and select a number

Source: [http://ubuntuforums.org/showthread.php?t=766683](http://ubuntuforums.org/showthread.php?t=766683)
