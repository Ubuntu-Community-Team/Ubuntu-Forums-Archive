### Post by Autodave on 2023-08-21
Have you rebooted the system?  If so, try turning it off altogether.  Let it sit for a few seconds and power up again.  If that still doesn't work, try disconnecting all USB hubs.  If still no good, if this has a removable battery, remove it, disconnect every cable, let sit for 10 minutes.  Put battery back in and power up.
