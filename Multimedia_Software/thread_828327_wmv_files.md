### Post by gandaran on 2008-06-14
any linux player can do it, totem, gxine, vlc, mplayer and many others.
for totem, you have to install the gstreamer plugins/codecs
for mplayer, you need the w32codecs package from the medibuntu repository.
vlc, just install it, it work's strait out of the box.
