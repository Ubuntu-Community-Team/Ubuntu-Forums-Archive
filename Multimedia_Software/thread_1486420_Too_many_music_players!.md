### Post by Miss Lemonade on 2010-05-17
I'm a recent linux convert, tankoo very much. This is my first post, please be gentle. I have a few particular needs for a music player, even in Windows I used multiple programs. Just looking in Ubuntu Software Center, there are 8 kajillion media players. I've installed a number of them, discounted the ones that totally crashed or killed my sound (why does that happen??) and kept looking for the ones that did what I need them to do.

Aside from just *listening* to music, I need to build playlists quickly from thousands of files to play for streaming to shoutcast servers. (In case you're wondering I DJ in secondlife) What I did like about Winamp is the search and playlist functions. I have my media library kinda spread out, and in Winamp I could type in, for example, "soundg" and immediately find all my Soundgarden files. The only Linux program I found that I didn't eliminate for the aforementioned reasons that does that is Rhythmbox, so I'm happy with that. I can build my playlists exactly how I did it before. Yay.

Now, on to streaming... The only thing I've been able to rely on is my old Sam Broadcaster running under Windows 7 on VirtualBox. Sam has a decent search function, but building playlists is ugly. What's really good about it is the output plugins, crossfading, and gap killer. For Internet radio, a little broadcast radio-like compression/limiting is just fine with me thank you very much, if all the songs play evenly. Normalizing and/or replay gain hasn't been as effective for me. Audio in VB is horrid! and SL is sometimes better, but I found I can listen to my stream now by putting the URL in audacious. Sweet.

Now, that's a lot to ask from a laptop to run at once. And it's sorta defeats the purpose of dumping Windows... so I've been searching for a solution to streaming. Mixxx looked promising, and when I did get it working,  I was able to start beatmatching right away. Fun!! But it's not reliable. Anything using jack audio has been unstable and glitchy. This is the stuff that made me run screaming from Windows. Grr.

Now I'm learning about MPD, which seems promising... but there are 16 million clients for it. Bah!

Could it be, would it be possible... to find one music player that does it all?? Wishful thinking, I know... 

~pulls hair out~
