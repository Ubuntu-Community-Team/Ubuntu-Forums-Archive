### Post by bmac on 2008-06-02
We may have differences regarding specifics - for example I don't require changing the photo file names prior to or during importing files from my camera. But I have found that G Thumb Image Viewer is quick and simple to use. Once opened I simply select file import photos and it provides me with the ability to:
Define the "Destination"
Delete or not imported photos from the camera during import
Keep original file names during import
Select which individual photos to import
etc.

This application has been extremely stable and seems to function well with not only my camera but multiple cameras belonging to friends and family...

Hope this helps....
