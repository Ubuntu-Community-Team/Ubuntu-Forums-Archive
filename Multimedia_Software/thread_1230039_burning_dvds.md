### Post by Bigtime_Scrub on 2009-08-02
I would just use Brasero which comes with Ubuntu. If you are trying to burn an .avi to playback on a standard DVD player then you will want a package called DeVeDe. This will author the AVI into a readable format the DVD player can read.

```
sudo apt-get install devede
```
