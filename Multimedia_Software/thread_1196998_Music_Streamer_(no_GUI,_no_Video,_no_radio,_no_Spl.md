### Post by mcduck on 2009-06-25
If you have a working streaming server on the server end, pretty much any player should be able to play the stream.

If you have just normal WWW server, nothing is going to be able to stream the audio but will instead load the audio files in the same way any other files are handled (requiring the player to download the whole file before playing it).
