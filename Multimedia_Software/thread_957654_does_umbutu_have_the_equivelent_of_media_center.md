### Post by lovinglinux on 2008-10-24
If you just need to record from a TV card, Mythbuntu might be too much for you, but if you need to store and share tv recordings with other devices than Mythbuntu is a good option. Keep in mind that it has a lot of configuration options and nuances. 

I use a different approach to media center, using Firefox and a few scripts, since I only record and watch recordings on a single PC. Look for the Firefox tutorial on my signature, where you can also find tutorials for a simple TV recorder and PVR-like recording. 

If a simple recorder is enough for you, but you don't want to use Firefox to manage your media libraries like me, you can use [[COLOR="Red"]**XBMC Media Center**[/COLOR]]("http://xbmc.org/"). While it lacks the ability to record videos, it has a simple and lightweight interface to manage and view your media.
