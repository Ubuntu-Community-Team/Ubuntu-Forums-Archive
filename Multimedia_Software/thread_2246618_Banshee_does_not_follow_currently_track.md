### Post by Rob Sayer on 2014-10-04
I've never actually used banshee much so I can't answer that part.  And I don't know of any linux music players similar to foobar because I think foobar is a kludgy, buggy, poorly documented piece of crap.  If I were running windows more I'd use vlc as a music player ... it's quite good in linux too, though it's not my default video player.

The good news is that there are tons of linux music players.  I actually use 2 of them.  Clementine usually because the playlist features do what I want.  Or Audacious for playing one track because it boots up fast.

There are lots of them in the ubuntu repos, which I recommend sticking to for new users.  The only music player requiring an external ppa to install that I'd bother with is deadbeef, but that's because it's fairly easy to get bit perfect audio with it, not for its playlist features.
