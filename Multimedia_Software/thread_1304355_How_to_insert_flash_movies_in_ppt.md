### Post by RichardLinx on 2009-10-29
You'll need to convert them to another video format if you want to run them from within OpenOffice.org presentation. I haven't tried running a video from within it before, but I'll recommend you try a few video formats and see which works best.

To convert the video use WinFF:
```
sudo apt-get install winff
```
