### Post by deadflowr on 2020-04-23
The package to install is flashplugin-installer.
This will grab the proper flash version for firefox, no need to run pepperflash.
Pepper flash is chrome version, which could be used with a hacky fix to run in firefox.

Flash is painful to use now. Purposefully painful.
Installing it is only half the battle as now to use it you must enable it on every site the will use it, for every firefox session.
It'll stay active for the web page you enable it on, but only for the current session.
I see it reverts back to off on a browser restart.


That said, it's dead technology walking.
It dies officially at the end of the year.
Any site that still uses it is probably not a site worth visiting.
