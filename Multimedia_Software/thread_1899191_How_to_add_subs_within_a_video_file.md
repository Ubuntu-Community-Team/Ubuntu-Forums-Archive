i was onto that as well.

the subs will need to be burnt in, apps like handbrake will do this easily if the op can get the srt to do it.

the biggest question is the source, what format is the op getting these flicks in and from what media.

if it's a dvd, use handbrake, under sub tab, select in drop down the desired sub, then hit add, to the right tick burn in.

this will be burnt in literally, no way to disable, that way, if the device doesn't support subs, this will work.
