### Post by Dreamsorcerer on 2013-07-12
I just wanted to post about an alternative to Canon's Liveview software, which I have started working on.

This software allows you to take photos with your DSLR through your computer, and preview those photos on your computer screen before saving.

You can download it from: [https://launchpad.net/liveview](https://launchpad.net/liveview)

I have used this with my Canon EOS 450D, but it should work with any camera supported by gphoto2.

It currently supports displaying a preview and taking photos. It lacks support for changing camera settings through the program.
If you find any additional bugs, please check the bug tracker and report a new bug if one does not currently exist ([https://bugs.launchpad.net/liveview](https://bugs.launchpad.net/liveview)).
