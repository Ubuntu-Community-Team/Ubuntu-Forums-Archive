### Post by OutOfReach on 2008-10-12
The default audio/video player in Ubuntu is Totem, Look under the Applications menu for 'Movie Player'.

There are other players other there as well, such as MPlayer ([http://www.mplayerhq.hu/design7/news.html](http://www.mplayerhq.hu/design7/news.html)) and VLC ([http://www.videolan.org/vlc/](http://www.videolan.org/vlc/))

Both are available in the repositories and can be installed like so:
```
sudo apt-get install mplayer
```
```
sudo apt-get install vlc
```
from a terminal, or you can search for them in Synaptic.

I recommend VLC.

### Post by nerktord on 2008-10-12
The default music player in ubuntu is Rhythmbox, it supports many functions such as: Playlist/Library, stream Radio/Podcasts, audio ripping, CD burning, extensive integration with other programs and much much more. I believe iT uses GStreamer media framework, so it'll support a variety of Audio formats. 

For Video content, I too recommend VLC media player. It supports many video/audio codecs and file formats and will normally work 'right out of the box'.. You can't really go wrong with VLC.
