### Post by gettinoriginal on 2008-12-13
1) Set a static IP each for both the desktop and laptop.
2) Share the folder/ drive within which the files lie.
3) Run the SMB/ CIFS client on the Ubuntu and you'll be able to access the shared folder/ drive.
4) Copy and paste.

See the Ubuntu documentation on how to browse Windows fileshares (SMB) here: [http://library.gnome.org/users/user-...etwork.html.en](http://library.gnome.org/users/user-...etwork.html.en)

Of course if they are both ubuntu machines (you don't specify), just email yourself your desktop music folder.
