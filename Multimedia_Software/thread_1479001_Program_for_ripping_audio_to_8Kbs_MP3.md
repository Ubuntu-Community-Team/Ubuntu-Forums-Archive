### Post by ron999 on 2010-05-10
OK
WinFF will let you batch convert those files to mp3 at 32Kbps. It is a GUI for ffmpeg.
Get it from here:-[http://winff.org/html_new/]("http://winff.org/html_new/")

When you're using it you should:-
1 Use '**Preferences**' to use a different target folder so that you don't just over-write the originals.
2 Select **Convert to... Audio** and **Device Preset...MP3**.
3 Use '**Options**' then in '**Audio Settings**' enter **32kb** in the '**Audio  Bitrate**' window.
