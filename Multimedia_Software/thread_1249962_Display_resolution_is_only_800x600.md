### Post by peepingtom on 2009-08-26
Have you installed the new nvidia drivers or are you just running the ones ubuntu came with? You can change resolution with the default ones by going to System -> Preferences -> Display. If you want to run the better drivers made by nvidia, go to Applications -> accessories -terminal and in terminal type ```
 sudo jockey-gtk 
``` to install/activate them. You then change resolution with nvidia-settings, found in System -> Administration -> Nvidia X org settings.
