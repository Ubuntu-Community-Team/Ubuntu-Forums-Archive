### Post by papibe on 2013-03-24
Hi StrictlyStupid. Welcome to the forums ):P

Most of the time the file organization does not matter, as most the organizing occurs in the media player based on the file's ID3 tags (metadata).

Are your files' tags well set up? If not there are several ID3 tags editors that can help you to do that.

Taking on the task of reorganizing your files would require that either the ID3 tags are properly set up, or that the file names are standardized and contain all the information you need for the organization (artist, album, etc).

Let us know which way you are going to go, and if you need more help with it.
