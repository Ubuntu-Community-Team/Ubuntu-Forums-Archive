### Post by BicyclerBoy on 2011-07-21
With twinview (linux) fullscreen is the whole virtual screen. Only native apps seem to understand the real size (gnome desktop etc)
You have to use windowed apps (not full screen) or setup a metamode with one screen turned off.
```
"metamodes" "DFP-0: 1920x1080_60, DFP-1: NULL; DFP-0: NULL, DFP-1: 1680x**1050**"

```
[B]This example metamode just (best guess) switches between the 2 screens (only 1 on).
[/B]
There are gnome keystokes to toggle thru' video modes
<ctrl>+<alt >+[<+> | <->]

where <> are keyboard keys, + means hold down all keys, [ | ] options one or the other.
.
This option line can be used to set the twinview order..
```

Option         "TwinViewXineramaInfoOrder" "DFP-0, DFP-1"
```.
You can **not** use the gnome display preferences tools with nvidia driver
