Yes, this is the classic problem of *Flash 9* not working well with *PulseAudio*.

One option is to install the *libflashsupport* package, but this may make your *Firefox* to crash a lot in pages that contain *Flash*.

Another option (I believe a better one) is to install *Flash 10*. You can download it from *Adobe*'s page. *Adobe* offers a *Flash 10* package for *8.04*.
