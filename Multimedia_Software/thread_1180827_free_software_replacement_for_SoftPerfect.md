I live in a hostel, with many Windows users. Some really liked ubuntu when I explain the features and all. But all here use a software called netscan (Softperfect Network Scanner) on Windows which scans the network for all computers and the smb shares, running multi-threaded. So, it can display shares of all computers really fast. But it fails to run on wine. As it is one of the software people here can't live without, they switch back to windows.
The nearest match I found is smb4k which has same features, but it doesn't auto scan the shares, so the user has to manually browse through the shares. So, anyone knows about a better Open Source alternative to netscan than smb4k?
