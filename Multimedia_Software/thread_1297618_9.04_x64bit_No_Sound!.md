### Post by VertexPusher on 2009-10-22

I guess a reboot really can work magic!  =)
There are two sound devices in your computer. During boot-up the system will randomly choose one of them and make it the default sound device to be used by applications. To make sure that the correct device is chosen *every time*, open a text editor and enter the following:
```
defaults.ctl.!card "Intel"
defaults.pcm.!card "Intel"
```
Save the file as ".asoundrc" in your home folder. This will make the Intel audio interface the default one. To make the ATI device the default one, replace "Intel" with "HDMI".
