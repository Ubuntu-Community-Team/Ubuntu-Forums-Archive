### Post by mgol on 2009-01-12
PulseAudio is default in Hardy and Intrepid. If You don't have any problems with that, don't change it. If You do (for example Avidemux and Skype seems not supporting it at all or have problems with it), You can change it to ALSA in System -> Preferences -> Sound. Don't use OSS if You don't have to, ALSA should deal well with most programs and OSS seems to have problems with supporting more than one source of sound (for example, music player and a game in the same time).
