### Post by |{urse on 2012-06-05
From what I'm reading (google) you need to install driconf (reportedly in the universe repositories) and enable S3TC texture compression in driconf. Also some ubuntu users report that this doesn't even fix the issue until libxtc is installed.

Hopefully these will help you or someone else can lend a hand as I've never personally experienced this issue.
