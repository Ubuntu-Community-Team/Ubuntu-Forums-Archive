### Post by littlepeon on 2010-05-07
hey there,

for a screen-recorder my recommendation is to use recordmydesktop as it is the most stable, feature rich out there.
u can use the gnome version frontend: gtk-recordmydesktop--there are many reviews/youtube-videos on getting it up and running.
i actually like the kde frontend better: recorditnow--there are reviews/youtube-videos on getting it up and running also--it includes countdown timer and re-encoding back-ends to convert your videos after recording. the only downside(not really but some may think so) is that you will have to add some more kde libraries to a stock install of ubuntu.

both of these are in the repository and you can add them with a simple: sudo apt-get OR you can chose them in Synaptic/Software Center.

these are both front-ends to recordmydesktop--if you are command-line literate, you could benefit from just invoking it at the command-line and using its functions/arguments to do the same thing as the front-ends i've mentioned. there are many reviews/tutorials if you google it.

there are other screen-recorders (istanbul or xvidcap) but none have the potential/features in my opinion.

have fun
