### Post by anonbeat on 2009-03-17
Hello,
    the first public release of the Guayadeque Music Player 0.0.1 is now out. As it mains features has the easy to catalog and manage your music collection. It supports audioscrobbling, shoutcast radio, lastfm information, tags editing, auto cover download to music folder, manual fetch of covers, etc

There is a package for ubuntu 8.10 available for download and the source code in [http://sourceforge.net/projects/guayadeque/](http://sourceforge.net/projects/guayadeque/)

See screenshots at [http://sourceforge.net/project/screenshots.php?group_id=250783](http://sourceforge.net/project/screenshots.php?group_id=250783)
