### Post by Holger_Gehrke on 2021-05-17
SpleeterGUI is just a a Graphical User Interface for a program written in Python using Tensorflow. So even if you're using the command line version of Spleeter you're going to need a rather beefy GPU. You can find Spleeter and another GUI for it that should work on Linux on the [Python Package Index]("https://pypi.org/search/?q=spleeter").
