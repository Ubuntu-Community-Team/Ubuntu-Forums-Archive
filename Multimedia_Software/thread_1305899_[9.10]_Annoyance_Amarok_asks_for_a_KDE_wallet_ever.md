### Post by damien_d on 2009-10-30
Hi All,

I have a small annoyance with Ubuntu 9.10.  I have installed Amarok, and every time it opens, it asks for my KDE wallet.

Om my media centre, this is especially annoying because the screen resolution (800x600 on a CRT), the wallet screen often falls behind the splash screen.

Is there a way to turn this off?  I have no desire to use last.fm and hence no need for the wallet functionality.

  -- Damien

### Post by lovinglinux on 2009-10-30
Go to "System Settings >> Advanced User Settings >> KDE Wallet" and untick the "Enable the KDE Wallet subsystem".

### Post by realzippy on 2009-11-02
deactivate last.fm in amarok,close amarok,open .kde/share/config/amarokrc:

[B]gedit .kde/share/config/amarokrc
[/B]

then set ignore kwallet to yes:

**ignoreWallet=yes**

When asked,then activate the option "Access always".

### Post by whymichael on 2009-11-29
If you force KDE Wallet to close,

   Right click the application on the panel > Close

Then Amarok will start up and then you can disable the services that use KDE Wallet.

   Settings > Configure - Amarok > Internet Services

   Uncheck Last.fm

If you want to use Last.fm, you can click the settings button (with the wrench on it) and you will have the option disable the use of KDE Wallet.

### Post by adeypoop on 2010-01-20
i found a way to open kwallet without requiring a password. this prevents the annoyance of entering my password when i start amarok but still allows me to automatically login to lastfm.

The way I did it was this, I installed the Kwalletmanager  using synaptic (as described by r-mo). Using the wallet manager I changed the password for the wallet to be a blank password. By having a blank password, the wallet gets open by any application requesting it automatically.
