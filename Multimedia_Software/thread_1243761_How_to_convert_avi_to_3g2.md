> **Prince Valiant said:**
> How can I convert an avi file to 3g2?  The file is anywhere from 1.5 to 2.5 hours long.

The best way IMHO is to use an appropriately configured FFmpeg. If you are happy to use h263 video and aac sound the Fakeoutdoorsman's guide should give you the required tools:

HOWTO: Easily enable MP3, MPEG4, AAC, and other restricted encoding in FFmpeg
[http://ubuntuforums.org/showthread.php?t=1117283](http://ubuntuforums.org/showthread.php?t=1117283)

The actual syntax will depend very much on the type of input file you are using and the capabilities of the device you are going to use for playback.
