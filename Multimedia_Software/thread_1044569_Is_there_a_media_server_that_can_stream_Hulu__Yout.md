### Post by HunterK on 2009-01-19
One program that I found very handy in windows was MediaMall "PlayON" software.  It is windows media server that "broadcasts" hulu / youtube / CBS / ESPN, etc streams to network media devices. (XBOX and PS3!)  I was able to browse and play the content of the sites directly on my ps3.  The media mall server basically downloaded the video and cached it on the windows PC and then streamed it to the PS3.

[http://www.themediamall.com/playon/](http://www.themediamall.com/playon/)

I miss this awesome little app now that I am on Ubuntu.  Is there anything like this in the open source world?
