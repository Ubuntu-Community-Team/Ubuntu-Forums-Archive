### Post by spegru on 2009-11-20
Delayed response but yes I have been doing this.
I used an old analogue tv cad with an s-video input and connected it to vhs. Mind you I discovered that lots of these are inputs only on the vhs end which is no good so I got a scart to s-video/ composite adaptor which you can set as input or output. Seems to work reasonably although I have some mild fine diagonal stripes that seems to spoil it a little - still quite watchable though
