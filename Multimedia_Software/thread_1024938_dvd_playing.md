### Post by 2hot6ft2 on 2008-12-29
The complete way
[http://ubuntuforums.org/showthread.php?t=766683](http://ubuntuforums.org/showthread.php?t=766683)

Or the short way
```
sudo apt-get install ubuntu-restricted-extras libdvdcss2
```
in a terminal.
Applications>Accessories>Terminal

Sometimes the libdvdcss2 wont install that way but it can be found here.

Intrepid
[http://packages.medibuntu.org/intrepid/libdvdcss2.html](http://packages.medibuntu.org/intrepid/libdvdcss2.html)
