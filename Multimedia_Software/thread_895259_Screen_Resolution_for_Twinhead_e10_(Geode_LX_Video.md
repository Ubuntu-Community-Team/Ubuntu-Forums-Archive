The screen works better in ubuntu 6.10, 7.04 and 7.10.

It gives a resolution of 1024x768, so I saved the xorg.conf from running the lappie under an older version


So if you backup your old xorg.conf

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.backup
```

Then edit it 

```
sudo gedit /etc/X11/xorg.conf

In the meantime, I get around the problem by adding two more panels and putting them at the bottom of the screen. The bottom one at 45px and the next one at 120px. Then you can put the standard "bottom panel" on top of them so that you can actually see it. Also it has the bonus that when you open windows in expanded view, they fill up only the bit of the screen you can see. If you don't use the panel trick you can use alt-leftclick to drag the window into view.

Also on login you won't see the "options" menu. But you can access this through F10

Finally you can't see all the list options under System>Preferences but if you right-click System you can choose "Edit menus" and cut down the amount of items in the list :)

### Post by phpmonkey on 2010-05-17
Thanks to the comments in [http://bugs.launchpad.net/bugs/433142](http://bugs.launchpad.net/bugs/433142) I have finally managed to get the correct resolution of 1024x600 :KS :KS :KS

I have it working in Ubuntu 10.04 (Lucid Lynx)

There is no xorg.conf by default, but I created /etc/X11/xorg.conf and put the following in it:

```
Section "Monitor"
	Identifier "Monitor0"
	VendorName "Monitor Vendor"
	ModelName "Generic monitor"
	HorizSync 28 - 51
	VertRefresh 43.0 - 60.0
	# run "gtf 1024 600 60" output on next lines
	# # 1024x600 @ 60.00 Hz (GTF) hsync: 37.32 kHz; pclk: 48.96 MHz
	# Modeline "1024x600_60.00"  48.96  1024 1064 1168 1312  600 601 604 622  -HSync +Vsync
	Modeline "1024x600" 48.96  1024 1064 1168 1312  600 601 604 622 -Hsync +Vsync
EndSection

Section "Device"
	Identifier "Videocard0"
	VendorName "Videocard vendor"
	BoardName "AMD Geode LX Video"
	Driver "geode"
	BusID "PCI:0:1:1"
   # Specify Panel size for Geode LX, corresponds with ModeLine above:
	Option "PanelMode" "48960 1024 1064 1168 1312  600 601 604 622"
EndSection

Section "Screen"
	Identifier "screen1"
	Device "Videocard0"
	Monitor "Monitor0"
	DefaultDepth 16
	Subsection "Display"
    		Depth 16
    		Modes	"1024x600"
    		Viewport 0 0
	EndSubsection
EndSection
```

I've also attached the file, so you can try it out for yourself. If you save it to your desktop, use the following command:

*[COLOR="Red"]WARNING: This will overwrite any (custom) xorg.conf file you already have there (backup first if in doubt: **sudo cp /etc/X11/xorg.conf /etc/X11/xorg-backup-(date).conf**[/COLOR]*

```
sudo cp Desktop/xorg.txt /etc/X11/xorg.conf
```
