Those would be flash games. ;)

Shockwave and Shockwave Flash are two completely different things. Shockwave content is in .dcr 
format (if I remember right, it's been years since I have worked with Director/Shockwave) while Flash uses .swf.

(When Shockwave became popular Macromedia started adding "Shockwave" as prefix to all their product names, "Flash" became "Shockwave Flash", Freehand online plugin became "Shockwave for Freehand" and so on..)

It's unlikely that Shockwave would ever be completely merged with Flash, it does things like hardware accelerated 3D graphics, IRC and loads of other stuff, has it's own scripting language (Actionscript in Flash, Lingo in Director/Shockwave) and so on. And n the end Flash content can be embedded inside Shockwave. I'd say that most of Shockwaves features that are useful in Flash have already been done (video & pixmap graphics support).
