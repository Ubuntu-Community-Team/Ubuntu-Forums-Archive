### Post by yonni on 2009-03-23
Here it is, use the guide linked to above because messing around with this and getting the wrong settings could seriously fry your graphics card and/or monitor if you get it wrong. My setup is for 1280x1024@60Hz which I'm absolutely certain that both graphics and monitor can handle

After this, you'll need to go into nvidia-settings gui and under "X Server Display Configuration" select the correct resolution and refresh rate. Opening up nvidia-settings as root ("gksudo nvidia-settings") and then hitting "Save to X Configuration File" under "X Server Display Configuration" will then cause the change to be persistent across sessions

Good luck :)
