### Post by quokka on 2009-05-06
Can't help with the discrete sound card, but I do know that the S/PDIF optical out on the rear of the Gigabyte EP31-DS3L motherboard works. I've just set one up with MythTV. Only used stereo, but I would expect 5.1 works. Ubuntu 8.10.This motherboard has RealTek ALC888 audio, which would imply that any board with this audio controller should work (if it has the TOSLINK connector).


PS: Rear Panel Optical S/PDIF out on ASUS P5KPL/1600 does not work on Ubuntu 8.10. This board has VIA VT1708B Audio.
