### Post by BicyclerBoy on 2011-02-07
NTSC colour is 59.94 field rate & not 60Hz.

What vdpau settings do you have configured in XBMC ?
The GT210 is not fast enough for advanced x2 de-interlacing so check you have not got playback profile set to max vdpau settings.
AFAIK it is not fast enough for some other feature set C features.
GT220 is the minimum recommended.

Do you have medibuntu repos ? ubuntu-restricted-extras libdvdcss etc ?

Have you read the ubuntu webpage about DVD playback ?
[https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)
