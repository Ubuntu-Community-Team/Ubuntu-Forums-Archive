### Post by fballem on 2008-05-11
In the old days, I had all of my CDs recorded in Windows Media Lossless format on my NAS. When I plugged in my Creative Zen Touch, I would open up Windows Media Player and be able to synch from the NAS to the Zen Touch. Windows Media Player would convert the WMA Lossless to MP3 automatically as part of the transfer. I did not retain the MP3 files on my computer, nor on the NAS - it was a seamless process.

In the new days, I have all of my CDs recorded in FLAC (which is lossless) on my NAS. Rhythmbox (sp?) is set to use this as my library. So the question is how do I get the CDs on to the Creative Zen Touch. I'm a newbie, so please be specific.

Many thanks!

### Post by fballem on 2008-05-18
Installed SoundConverter from the universe repository. Once installed, I used the program to make a copy of the FLAC files as MPG files.

Once completed, gnome2 can transfer these to my mp3 player.

It's a two step process, not as easy as Windows Media Player, but certainly workable.

---

### Post by fballem on 2008-05-21
The advantage to this approach is that:

1. In Windows, every time I had to reload my mp3 player, every track had to be converted - it took a long time (I have more than 2,500 tracks)

2. In the new world, because I have pre-converted the tracks, it takes less than an hour to reload my mp3 player.

Some care is required in setting up Sound Converter - in Edit | Preferences, there is an option to delete the files after conversion (definitely don't want that option). In addition, I placed the converted files into a separate folder, and I de-selected the option to create subfolders based on artist and album - the mp3 player uses a straight list of files plus the tags to organise things.
