### Post by Melcar on 2009-06-04
Try [PTS]("http://www.phoronix-test-suite.com/").  It's a complete benchmark suite and it makes it really easy to compare between hardware. If you want to test 3D, running games is pretty much the best way.  However, with those cards I would concentrate on 2D rendering performance instead; PTS has a GUI-Toolkits and a X rendering test suite that can help you with that.
