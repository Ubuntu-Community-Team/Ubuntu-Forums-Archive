### Post by coskierken on 2009-12-17
To install the driver, first in a terminal you must switch to a supersuser by the "su" command followed by your login.  It will prompt you for your password.  I would have that binary accessible on the desktop and drag it to the terminal window and it will populate it with single quote marks at the end and beginning.  Just backspace and delete them.  Then hit enter and the program should run.  Hope this helps.
