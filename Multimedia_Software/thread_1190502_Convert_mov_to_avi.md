### Post by angelsguitar on 2009-06-18
Have a look at [http://ubuntuforums.org/showthread.php?t=766683]("http://ubuntuforums.org/showthread.php?t=766683"), the famous Multimedia How-To for Ubuntu.  In that how-to, besides describing how to install the most common packages you'll need to playback any media type, you'll find instructions on how to install WinFF, a gui program to convert many formats.  You'll have to enable some extra repos to be able to install the program (specially medibuntu.org and the repo for WinFF)

I highly recommend you follow the entire how-to, but for what you are asking, installing ffmpeg and WinFF should be enough.
