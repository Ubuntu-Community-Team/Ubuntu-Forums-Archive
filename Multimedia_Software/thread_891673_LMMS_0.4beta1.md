### Post by eyecreate on 2008-08-16
I have made a quick package for those wanting to test LMMS 0.4 beta without trying to compile it. It was made with a quick checkinstall so no dependencies are written(which means you need to have the other packages installed already.)

Requires installed:
Qt >= 4.3.0
and should have:
libvorbis
SDL
libsamplerate
libsndfile
stk
and WINE for VST

[lmms-0.4.0_beta1-1_i386.deb](http://www.4shared.com/file/59265270/35112ff2/lmms-040_beta1-1_i386.html)
