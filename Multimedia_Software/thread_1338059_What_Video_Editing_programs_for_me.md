### Post by ilil on 2009-11-27
Well, there is no one-to-one replacement for MyDVD/Videowave/DVDit now. I speak about such features as (1) converting from any video format and (2) creating DVD menus. My program, Bombono DVD, provides easy creating DVD menus but lacks converting features (yet). But that work can be done separately with programs I mentioned earlier (WinFF, KDEnlive and so on).

But if you just need to put videos on DVD, one after one (like ConvertXtoDVD), you can try DeVeDe.
