### Post by Dyffo on 2010-06-22
I have downloaded a very old movie, this has been presented as a series of VTS Files. On my computer I can play this as a DVD using my VLC Media Player. However I would like to create a DVD which can be played through my DVD Player. Can anyone advise what burner is available for Ubuntu which will do this conversion. I have tried DeVeDe and Brasseiro without success. Any assistance would be much appreciated.

### Post by cchhrriiss121212 on 2010-06-23
Take a look at Xfburn, you just go to create new data composition then drag the files over.
In order for burned DVDs to be read by standard players they need to have folders like this:
AUDIO_TS (an empty folder)
VIDEO_TS (put all video files in here)
