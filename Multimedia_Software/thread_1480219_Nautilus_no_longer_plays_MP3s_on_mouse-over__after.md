I just noticed that after upgrading to 10.04 Lucid Desktop, nautilus no longer plays MP3 files on mouse-over. 

It does recognize the file as a music file as it displays the music note icon in the comic balloon next to the music note file icon when the mouse is positioned over an MP3 file, however no sound is played.

If I double-click the MP3 file, totem is invoked and plays the MP3 file just fine.

Any help is greatly appreciated.
