### Post by satanselbow on 2012-01-31
The gloss is obviously down to the paper surface texture not the picture itself...

To create the effect in software you can overlay a transparent white gradient from top left corner to bottom right corner becoming more transparent as it crosses the image - this can be implemented in the gimp quite simply.

Alternatively some lightbox gallery scripts have dynamic effects that would have a similar effect - this would be non-destructive so would not destroy the original scanned image - which should be backed up before processing anyway - always keep the originals.
