### Post by mcduck on 2009-05-20
Like it says on the page you posted (in red text!), "Note: Aimersoft DRM Removal Softwares work only on Windows 98 / 2000 / XP / Vista 32-bit;".

That's a Windows program, only made to run on Windows. There's no quarantee that Wine would be able to run any windows app, some things work and others don't. You should consider every app that actually works with Wine as being a nice extra, not something you could really expect to happen.

Wine or not, the basic rule is that every operating system only runs programs made for that operating system.
