### Post by bkbilly on 2010-05-10
When we are watching youtube videos and we switch to Fullscreen mode it shows us the animations we have from the compiz...
This is a bit annoying...
I found the solution:[INDENT]You have to install **CompizConfig Settings Manager**[/INDENT][INDENT]Open it and go to **Effects -> Animations**[/INDENT][INDENT]**Open & Close Animation tab**[/INDENT]Put this line on the end of the first **Open Effect**:
```
** & !(class=Firefox-bin)**
```Try a Youtube video and it should be showing no animation!
