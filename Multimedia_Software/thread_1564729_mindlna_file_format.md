### Post by huwjenjinn on 2010-09-14
Believe its MP3 for music, JPG for images and MPG for movies.

DLNA is pretty fussy about the format for MPG files so you'll need to make sure you've converted them just so. I've found **ffmpeg **invaluable in this respect. If you think your MPGs are not appearing then try re-encoding them using **ffmpeg **into a recognised DVD format.

There really isn't much documentation for minidlna but there are a few guides on the internet. Google is your friend here.
