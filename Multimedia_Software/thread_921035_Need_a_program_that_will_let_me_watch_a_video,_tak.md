### Post by timcredible on 2008-09-18
avidemux2 will allow for easy frame capturing out of a video (get one picture out of a video).  digikam will allow for easy slide shows of pictures (with audio if you want).  gimp will allow you to layer pictures like you asked.  gimp is already installed on all ubuntu systems, so, go to synaptic and install avidemux2, digikam, kipi-plugins, ogg123, mpg123, mjpegtools to install the rest of the software.
