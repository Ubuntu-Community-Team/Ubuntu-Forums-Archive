### Post by colascdave on 2012-05-06
OK, I'm only using one monitor but I did manage to eliminate overscan on my Panny 32" 720P LCD TV using a combination of  "ViewPortIn" and "ViewPortOut" with the 302.07 driver. I played with nvidia-settings commands until I finaly got something that worked. This is what worked for my TV:

nvidia-settings --assign CurrentMetaMode="DFP-0: 1920x1080 { ViewPortIn=1920x1080, ViewPortOut=1820x1020+50+30 }"

With that, I added the following line to the "Monitor" section of xorg.conf to make the settings "stick:"

Option "MetaModes"  "DFP-0: 1920x1080 { ViewPortIn=1920x1080, ViewPortOut=1820x1020+50+30 }"

Good luck, hope this helps!
