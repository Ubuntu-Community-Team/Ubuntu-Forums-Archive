### Post by uBeer on 2009-05-08
I have an Edirol UA-25 EX sound card and normal sound works with jaunty if you select oss instead of pulse and alsa (that is in advanced mode so with midi enabled, in non-advanced mode it should work with pulse and alsa). Don't know too well if midi does work, I don't do my music stuff on Ubuntu.

Too bad Ubuntu seems a bit slow with integrating the latest alsa, because there are some enhancements to the drivers to make the advanced mode work well.

But this is all about the UA-25 EX. If you buy the old one (the UA-25) it should work well.
Hope this helps!
