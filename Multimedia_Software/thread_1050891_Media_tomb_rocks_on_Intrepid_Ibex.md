### Post by Darkmoon_UK on 2009-03-07
Yep, I found MediaTomb to be excellent, too.

Heres my findings of trying to run 4 different uPnP servers between my Intrepid box and PS3:

MediaTomb - Works and free!
TwonkyMedia - Works but not free.
GMediaServer - PS3 gave me DNLA 7531 error.
uShare - PS3 gave me DNLA 501 error (despite using --dnla switch which they claim provides PS3 compliance)

Incidentally, I do have a niggling problem with streaming to the PS3, there are some very small, occasional pauses, maybe once every 2 minutes. Happens with both MediaTomb and TwonkyMedia, but they're wired 100Mb and the CPU load is very low. It's like its not buffering enough and then drops out for a second before catching up with the HDD.

But yeah, MediaTomb seems like a great product, well done that team! :popcorn:
