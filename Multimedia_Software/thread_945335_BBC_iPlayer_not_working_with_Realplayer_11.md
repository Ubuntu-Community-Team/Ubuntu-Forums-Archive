### Post by ajgreeny on 2008-10-12
I think the easiest way to proceed is to use the mozilla-mplayer plugin in firefox for the BBC iplayer radio.  I've never used it but I think vlc plugin should also work in the same way, so try both and choose the one you like best;  I use the mozilla-mplayerr plugin mostly, and if I want to keep a copy of the streamed music, I use audacity to record the soundcard output.  This will work only with streaming and it is not possible as far as I'm aware to download for later playing, as you can with Windows, hence the use of audacity.

If you want to use realplayer as a standalone, you can set the url of the BBC radio streams in realplayer favourites so that you do not need firefox open, and play radio that way, but you can't use the listen again feature using realplayer, you will need Firefox for that.

I hope that all makes sense to you.  If you need more info come back again.
