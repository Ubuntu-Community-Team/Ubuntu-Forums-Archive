### Post by Sin@Sin-Sacrifice on 2008-05-11
Check to see if the crossfader is enabled in the preferences.... if it is then turn it off. If that doesn't work you can use other media player such as VLC or Amarok (I love Amarok... if you have an ipod then you can do all your music swapping on it). They should both play m4a files. Make sure you have all the proper codecs installed and enabled.... you should though by default I think. About them not showing up.... you have to point rhythm box to the folder they're in via the music tab.... Import Folder button. Dive in head first man. There's tons of info out there.

### Post by olivine on 2008-05-11
In synaptic, search for Amarok, mark it for installation, and press 'apply'. It will install Amarok and several other packages it need. Then start Amarok from the desktop application menu (top left). It will ask you a few things to get the right settings. It should be very straightforward.
