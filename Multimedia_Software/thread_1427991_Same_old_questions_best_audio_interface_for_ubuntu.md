### Post by mkloch on 2010-04-25
Hi! I had the same problem in the past. Right now I've got [Lexicon Alpha]("http://soundsoflinux.blogspot.com/2010/04/lexicon-alpha-works-with-linux.html"). It can record two inputs simultaneously and works out of the box. It has no phantom power though. Lexicon Lambda and Omega are reported to work as well :) About firewire  vs usb thing - I think you should go for usb. And about usb 3.0, I'm not a pro, but I think audio interface has to support usb 3.0 standard to benefit from it.
