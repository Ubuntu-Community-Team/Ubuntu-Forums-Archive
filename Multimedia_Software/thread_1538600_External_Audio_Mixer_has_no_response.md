### Post by cchhrriiss121212 on 2010-07-25
From my searching skills it seems as though this device is not supported on any Linux platform. If you really want to ditch Windows then check the ALSA or FFADO (thats for firewire cards) database for cards that are supported and trade yours in. A firewire device will perform better than USB on Linux though it is slightly complicated to setup.
I wouldn't wait around hoping that this device will oneday become supported as it is unlikely.
