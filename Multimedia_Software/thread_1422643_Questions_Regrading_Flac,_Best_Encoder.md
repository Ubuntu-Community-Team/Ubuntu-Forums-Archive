### Post by mc4man on 2010-03-05
read 
man flac
lossless is lossless, the higher level #'s (than the default of 5) will take a bit longer and may result in a slightly smaller file
(look at the Synonymous with to gather what else is set and to what advantage if any

I use rubyripper quite often which has a default option for flac of --best
