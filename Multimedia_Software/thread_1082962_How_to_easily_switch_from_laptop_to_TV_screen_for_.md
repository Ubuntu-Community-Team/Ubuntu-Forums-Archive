### Post by Harmshi on 2009-02-28
If you want to switch displays from you laptop (with ATI graphics card) to your TV using CRT / VGA cable, read this:

[http://blog.harmjans.com/2009/02/28/how-to-switch-to-external-display-for-ati-on-ubuntu/](http://blog.harmjans.com/2009/02/28/how-to-switch-to-external-display-for-ati-on-ubuntu/)

Here you can find a link to a bash script that will instantly change displays from laptop screen only to Laptop screen + external VGA to External VGA only to Laptop screen only again etc...
You can make a launcher for the script or bind it to Fn + F5 using xbindkeys.
