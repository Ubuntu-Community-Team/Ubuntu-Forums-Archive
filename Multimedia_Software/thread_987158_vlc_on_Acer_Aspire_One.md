### Post by N2G(bn#7+ on 2008-12-07
Hi, I just noticed that no-one had replied to your post... 

I've just been mucking around trying to install things on the Acer Aspire One today and managed to get VLC installed under the basic version of Linpus Linux Lite that comes bundled with the machine.

VLC works fine for playing divX movies once you get it to work and the quality is fine (even playing from an external hard drive powered through the usb).

After enabling the XFCE right-click menu, all I had to do was:
```
sudo yum install fedora
sudo yum install vlc
```
and optionally:
```
sudo yum install mozilla-vlc
```

Useful instructions (the VLC one didn't work fully, but there's lots of useful material there) are available [here]("http://the.taoofmac.com/media/Acer/Aspire%20One/AA1notes.html").

On a side note, the Acer Aspire One Linux version is available in Australia for AU$340 (after a cashback from Acer), so it might be worth checking on the net for a cheaper one.
