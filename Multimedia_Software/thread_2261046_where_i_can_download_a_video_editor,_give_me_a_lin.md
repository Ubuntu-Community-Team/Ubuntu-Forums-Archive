### Post by Rob Sayer on 2015-01-16
Why don't you want to use the standard ubuntu repos and/or software center?  Is it because (as your previous posts indicate) that you're actually using antiX linux?

If you want a deb file just find the web site of the program and download it.

It's actually not that easy to recommend an editor.  Depends on what you want to do, the resolution needed, and the hardware.

There are some really good ones that are pretty hard to use for a novice.  I'd put lightworks in that category.  The downside of it being "pro quality" is that the devs assume that it'll be used by pros who don't need hand holding.  Openshot has a good ease/power compromize, as does avidemux.

Also, powerful NLE editors need *lots* of computer power.  Serious video geeks not only use mega powerful computers to edit, they often add an extra hard drive just to store the editor's temp files.  It's that bad.
