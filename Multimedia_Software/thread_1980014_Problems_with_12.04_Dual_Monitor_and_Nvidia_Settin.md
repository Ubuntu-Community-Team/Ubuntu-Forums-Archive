### Post by sunwukung on 2012-05-15
Just solved the problem, leaving for posterity. In a nutshell:
remove any old xorg.conf files, then run:

```
sudo nvidia-settings
```

set as required, reboot. Problem solved. 

I'd already tried entirely replacing the file several times (i.e unticking merge, running as root etc) but deleting the original files did the trick. @surfer, thanks for looking at the problem.
