### Post by varunendra on 2010-07-24
Close firefox, then try installing flashplugin-nonfree or flashplugin-installer from synaptic (uninstall all other flash plugins first, after closing firefox). You may need to enable multiverse or restricted software repositories in 'Software Sources'.

### Post by lovinglinux on 2010-07-24
You problem is very common and probably due to conflicting plugins. Install the latest version (1.0.9) of [FLASH-AID](http://flash-aid-extension.blogspot.com) extension for Firefox. It will remove conflicting plugins and install the proper flash version according to your browser architecture.
