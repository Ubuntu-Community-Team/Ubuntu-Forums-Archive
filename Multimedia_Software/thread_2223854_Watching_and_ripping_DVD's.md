### Post by Slopsbucket on 2014-05-13
This is just a Heads Up for anyone having problems.

I spent hours trying to get VLC to play a commercial DVD.
I followed all the instructions I could find about installing dvdread4 and then using a terminal to install css.h.

I experimented a lot and got nowhere until I realised that the css.h that Ubuntu gives you isn't worth the recycled electrons used to create it.

In the end the answer was so easy it's rediculous.

Go here:

[http://download.videolan.org/pub/debian/stable/](http://download.videolan.org/pub/debian/stable/)

Download the appropriate .deb file and then open your file browser and double click on the file to install it.

VLC now works like a dream and Handbrake rips the DVD's for me like it's supposed to.

Cheers,

Andrew.
