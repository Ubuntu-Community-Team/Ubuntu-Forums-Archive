### Post by helios16v on 2012-02-04
I figured it out, you can't. it's a known bug that when you try to reorganize a file it can't find where the file was to relocate it so it puts it back in unclassified files.

I switched to XBMC anyways so this information is no longer needed.

For anyone who is still using the classic Moovida with the better looking UI IMO then you can no longer get support for this. You must update to the newer UI in Moovida 2 which has the iTunes-alike windowed mode for organization and then the uneditable layout in full screen mode which I think is a huge downfall.

So to fix this problem if anyone else is having it, either update to Moovida 2 or switch media centre programs like I did.
