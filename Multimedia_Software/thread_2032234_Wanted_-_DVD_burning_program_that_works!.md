### Post by SeijiSensei on 2012-11-28
You should try K3b again.  I've used it for years without incident.

Tell it you want to make a New Data Project, then navigate in the top window to the source directory where the files are stored.  Drag the ones to be burned down into the bottom window that represents the DVD.  When you're ready, insert the blank DVD and hit Burn.
