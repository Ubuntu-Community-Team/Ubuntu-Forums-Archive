For copyright/restriced DVDs see [www.medibuntu.org](www.medibuntu.org) (the repository seems to be somewhat wacky atm)

In gerneal you require some codecs to play restricted media anyways.

After adding the repository (medibuntu has a howto on the named page) install DVD stuff via:
```
sudo apt-get install libdvdread3 libdvdcss2 ubuntu-restricted-extras libdvdnav4
```
