gtk-RecordMyDesktop does work on 8.04!, if it doesn't work for you, then provably you are missing a gstreamer plugin, check if all gstreamer plugins are installed or install the ubuntu-restricted-extras, this package contains the gstreamer plugins and a lot more, but I don't really 
recommend it, as it will also install open java, it's best you install the gstreamer plugins individually.
