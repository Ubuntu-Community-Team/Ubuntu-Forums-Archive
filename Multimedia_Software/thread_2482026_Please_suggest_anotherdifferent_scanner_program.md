### Post by iamjiwjr on 2022-12-16
Xsane (a deb in the repo) is free. Vuescan (deb from their website can be downloaded) is pay-for. If this applies or gives you another path to pursue - I have a Brother printer-scanner. For Document Scanner to work for me, I had to remove the automatic printer files (ipp-usb and sane-airscan).  My printer driver was already installed to work without them, so that was an easy install via Gnome Settings. Then I downloaded the manufacturer's scanner file, a deb file, and installed it (I use gdebi for that). Everything works great now.
