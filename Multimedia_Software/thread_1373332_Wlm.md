### Post by ajgreeny on 2010-01-06
It is true to a certain extent, some things will run but many won't, so it's best not to depend on it too much.  Anyway, having chosen to run ubuntu as an OS, why not try to use ubuntu's own versions of software you know, instead of trying to use windows versions.

There is almost nothing that can not be done in ubuntu that you can do in windows, so try thinking differently now; you may be surprised just how wonderful open source applications are.

If you really must try wine, it is in the repos like everything else you will need, so ```
sudo apt-get install wine
``` will get it for you.
