### Post by chunchengch on 2008-12-06
I refer to the thread [http://forum.ubuntu.org.cn/viewtopic.php?f=74&t=94276](http://forum.ubuntu.org.cn/viewtopic.php?f=74&t=94276) that posted on Ubuntu-cn to make a deb for the sake of easy installation.

This deb will add a plugin into the audacious, so it will show the corresponding lyric while playing mp3 with audacious.

Before playing mp3, you have to go to Application > Audacious > Preference > Plugins > General , and select the plugin ShowLyric , then open the preference of this ShowLyric plugin to set the colors of lyric you like, when all these are done, close audacious.

BTW, one important thing is the id3 tag of mp3 must be correct, otherwise, it would not show the correct lyric, you can use EasyTAG or other tag tool to check and edit the id3 tag;  if there is no corresponding lyric in the lyric website, the lyric won't show either.

Now, open mp3 file with audacious, you will find a window showing lyric, enjoy it!
