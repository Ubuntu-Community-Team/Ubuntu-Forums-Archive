### Post by djRobbieF on 2008-08-28
That's probably part of your Windows drivers as opposed to a hardware feature.

If you're using a program like Audacious to play your audio, you can activate plugins for that program.  Audacious has stuff for effects, plus things like vocal cancellation.

I think it's just going to be software-specific though, unless there's a program for Ubuntu that does it for any sound card--but I don't personally know of any such program.  It might exist though.
