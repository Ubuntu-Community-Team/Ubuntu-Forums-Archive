### Post by SeijiSensei on 2010-10-09
> **andrew.46 said:**
> If this file is failing the repository versions of FFmpeg and MPlayer might be time for an upgrade :).

mplayer is a rapidly moving target, and the repositories are always quite a bit behind the current SVN version.  rvm, the author of smplayer, maintains a [repository]("https://launchpad.net/~rvm/+archive/ppa") with fairly up-to-date versions of mplayer.  

I usually build mplayer from scratch using the most recent daily snapshot at [http://www.mplayerhq.hu/](http://www.mplayerhq.hu/).  After a "sudo apt-get build-essential" and a "sudo apt-get build-dep mplayer" it compiles without incident.

No particular reason except I haven't used SVN all that much.  Most of the time I just build a new mplayer when I've installed a new version of Ubuntu. At this point I have /usr/local on a separate partition, so I don't really need to build new versions very often.  I only do so a couple of times a year.
