### Post by Vladlenin5000 on 2014-10-07
Had you installed the recommend proprietary driver version from System Settings > Software&Updates ("Additional Drivers" tab) and you probably wouldn't have had that issue.
Apparently there was a minor kernel update and in that case you need to manually install the drivers again.
