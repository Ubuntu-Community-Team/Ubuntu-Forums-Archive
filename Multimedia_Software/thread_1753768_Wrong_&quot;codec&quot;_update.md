### Post by Paddy Landau on 2011-05-09
Open Synaptic Package Manager. Go to File > History. You should be able to see what was installed, and therefore go to uninstall it.

Alternatively, give up on Totem and use [VLC]("http://www.videolan.org/vlc/") instead. VLC tends to be reliable.
