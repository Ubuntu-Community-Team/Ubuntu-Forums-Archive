### Post by daodeltaforce on 2009-10-19
I realize the post question here in this thread is over a year old but it needs to be answered so that others can resolve this issue.  I am running Kubuntu 9.04.  I also had the same question and found a link to a Opera blog site that has the full solution.  I followed the instructions and installed Totem using Terminal and Totem played my Apple Quicktime QTL streaming file perfectly.  I did not have to adjust any preferences in Totem.  If you make Totem your default player in Kubuntu then you can simply click on the QTL file and it should play, but I have VLC media player set as my default so I just had to load Totem and navigate the file browser in Totem to add the QTL file, then select play.

Here is the link with the steps to install Totem:
[http://my.opera.com/ubuntunerd1/blog/how-to-play-quicktime-videos-and-other-media-with-totem](http://my.opera.com/ubuntunerd1/blog/how-to-play-quicktime-videos-and-other-media-with-totem)
