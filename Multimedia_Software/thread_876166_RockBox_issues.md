### Post by michaelaly on 2008-07-31
Rockbox can be a little slow on updating its' database. Navigate to the settings menu -> General Settings -> Database -> Update Now. Wait a few mins, say 5-8 depending on how your storage capacity. The database will be updated, it's just not always prompt.
Note : I'm running Rockbox on a Sansa player, but it should have the menus.
