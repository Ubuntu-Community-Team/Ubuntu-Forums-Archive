### Post by uptwobucks on 2010-01-10
I had this problem after installing desktop cube. If you have installed desktop cube, disable it by right clicking on the desktop then click "change desktop background", then "visual effects". Select "none" to see if this corrects the problem. I am able to play movie files by disabling it.

If you have not installed desktop cube, then I cannot help you, must be other issues.
