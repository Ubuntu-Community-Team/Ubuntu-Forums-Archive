### Post by jonnymccullagh on 2008-08-05
If you want to stick with these Adobe products you can run Photoshop on Wine there are guides around the web but Gimp is an alternative but it takes some time to get familiar with the GUI. CS2 works with the latest Wine (Google have been helping Wine with this) and there are some guides around the web to getting it running.
I have never tried Illustrator - alternatives include inkscape,Xara Xtreme,OpenOffice Draw ...
Dreamweaver - I used to run MX quite well under Wine (Crossover Office) and version 8 seems to run too but I do not use it much any more. I am using Aptana at the minute for editing code and uploading/downloading.
I was able to run Flash MX a few years ago in Wine (Crossover Office) but have not tried any of the newer versions lately.
