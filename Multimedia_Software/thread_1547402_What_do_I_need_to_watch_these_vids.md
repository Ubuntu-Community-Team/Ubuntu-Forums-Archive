### Post by Zeike on 2010-08-06
If these videos are flash, you should be able to get them to work by installing the 'flashplugin-installer' package.

If these videos are silverlight, which I suspect they might be, because they're on bing, your best bet is to use the moonlight plugin. I think you can install this with the 'moonlight-plugin-core' and 'moonlight-plugin-mozilla' packages.
