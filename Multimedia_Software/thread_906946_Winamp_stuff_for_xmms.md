### Post by dmizer on 2008-09-01
If you would like to see features in XMMS, you'll have to talk with XMMS developers here: [http://www.xmms.org/](http://www.xmms.org/)

A possible alternative for you:

Currently Audacious2 is in development, and the opportunity is ripe for you to put in some feature requests.  More information here: [http://audacious-media-player.org/index.php?title=Audacious2](http://audacious-media-player.org/index.php?title=Audacious2)

Be sure to describe the feature well, and never say, "like winamp" ;)
