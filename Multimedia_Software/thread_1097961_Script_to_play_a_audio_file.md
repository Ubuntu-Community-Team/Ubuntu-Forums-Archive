### Post by Neo_The_User on 2009-03-16
```

sudo apt-get update && sudo apt-get install mpg123
mpg123 (put your audio file here without parenthesis)

```

Example:

```

wget google.com/black-hole-sun.mp3
ls
mpg123 black-hole-sun.mp3

```
