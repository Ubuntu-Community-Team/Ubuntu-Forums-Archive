### Post by andrew.46 on 2010-02-24
Hi Mia1990,

If you are running an *svn* version of tovid this means that you are using the development version of tovid which is held in a subversion repository where multiple developers can work on the project at the same time. Running the command *svn up* will download the latest changes to the source code to your computer which you can then compile and install.

Does this answer your question at least in part?
