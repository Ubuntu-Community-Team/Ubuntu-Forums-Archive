### Post by mardurhack on 2011-08-23
Hi I've got the same problem on an Asus Eee 1201N with Ubuntu 11.04. Tried almost everything described here and elsewhere but couldn't find a solution. Instant playback works flawlessy in Windows XP. It's enough to enable "Mic Volume" in the Volume Settings in the Output tab (in the Properties window you can select Realtek HD Audio Input or Output, in the first one I checked the Microphone box and in the second one I enabled the Mic Volume slider setting it at max and unchecking "Mute"). I wonder if a sort of different driver is needed to get it working. Any ideas?
