### Post by spwilkinson on 2010-12-15
An update for those with similar problems:

Unfortunately for my ego, this was a trivial problem to fix, but fortunately for my sanity, it was a trivial problem to fix :-)

Ubuntu was outputting the screen correctly, but my tv was in 'widescreen' mode and not 'PC' mode as it should have been.
Changing the setting over resolved the issues immediately!

Regards
