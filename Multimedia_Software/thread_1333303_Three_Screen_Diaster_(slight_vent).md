So I feel the need to vent a little and hopefully get some help for my situation. I do appreciate you taking the time to read this and try to help me. 

So I have been a long time Linux user (Caldera, Slackware, Debian, and now Ubuntu). I took a brief hiatus from Linux as my primary machine and switched to a Mac for a couple years. Regardless, with the release of the i7 CPU's and Karmic I really wanted to build out my "dream" machine. 

I am using a quad screen NVidia Quadro NVS 450. I assumed since I really don't need anything to graphic intensive and the support for Linux this would be a good choice (I've used dual screen NVS models in the past). All I really want is some nice Compfiz stuff.

I don't want TwinView, I don't want Xinerama and I don't want multiple X Sessions, I just want it to work. Of course, since there are two GPU's on the single card they are seen as two separate cards. All I want is to have a single panel on the middle screen and be able to drag windows across screens. Just like I would in Windows by having the start bar on the center and the other displays as extensions. I would just think, HOPE, that Ubuntu would support this now. What is more sad is even with two screens in order for it to work I need to enable TwinView which just mimics a single large display. 

To make matters worse my Sony Vaio laptop with an external display running 9.04 can do this just fine. 

I tried an ATI 5750 card since it supports three screens on a single GPU but after loading the drivers (the ones that just prompted me after the install from Ubuntu) I have no X at all. 

This is a mess.
