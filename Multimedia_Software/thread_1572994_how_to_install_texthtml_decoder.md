### Post by dirty_harry on 2010-09-12
*.rm is a container for RealAudio and RealVideo, which are proprietary codes. Haven't seen it for a while.
[https://secure.wikimedia.org/wikipedia/en/wiki/RealVideo](https://secure.wikimedia.org/wikipedia/en/wiki/RealVideo)

Now you have the choice:
# you can install the [helix-player]("https://help.ubuntu.com/community/HelixPlayer") with Synaptic
# or install the RealPlayer for Linux developed by RealNetworks, owner of the Real-codec; probably more efficient but also a bit more work -> see [https://help.ubuntu.com/community/RealPlayerInstallationMethods](https://help.ubuntu.com/community/RealPlayerInstallationMethods)


I'm not sure, but I think there was a possibility to compile the VLC-player with RealMedia-Support... anyway.
