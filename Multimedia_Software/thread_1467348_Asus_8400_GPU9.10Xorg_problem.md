### Post by Fenris_rising on 2010-04-30
Hi Chaps

Well at long last I upgraded from my old X300 Radeon GPU to the Asus 8400GS. This was following comments seen on this forum about the fact that Nvidia cards were a bit better in performance and function.

So I installed it activated the drivers for it and after 20 minutes had both my screens working and compiz. Just like I had had with my old faithful ATI based card but with visibly more finesse.

Then out of no where something 'burped', after an hours use, and it all went downhill from there.

Problem one was being unable to get both screens working at the same time as they had before. I was left with one functioning screen which contained the desktop of both screens. In effect I had to scroll left and right to see the complete desktop.

Much head scratching followed. Drivers were removed, reinstalled, removed again and envyng given a go. All to no avail. I even tried the nouveou (spelling?) drivers with no success.

Using the Nvidia UI to set things up as I had the first time did keep throwing an error (as it had before when everything worked) about being unable to create a xorg.conf backup. But being spoiled by the fast install times of Ubuntu I was prepared to risk it.

A complete fresh install did follow just in case something was fundamentally wrong but, again, to no avail.

In the end I pulled up the xorg.conf file to look at. It contained around 15 lines of text. I then used the Nvidia UI to set everything up again but when I hit save I then chose to preview the file. This turned out to be a very long list of text that was to be the new xorg.conf file.

**It seems that the newly created file does not replace the old one due to the failure of the Nvidia UI to create the xorg.conf backup file.**

Using sheer cunning I copied and pasted the preview text into the old xorg.conf having manually created a backup (just incase). I now have both screens up at last and hope to have compiz working again.

Has anyone else had this problem? What did you do to solve it if you were able to?

regards

Fenris
