### Post by Th3Professor on 2011-06-09
I noticed that Netflix is a fairly popular topic in here, so having a thread with the answer in the subject might help reduce the many-other-thread-creations.

Putting the subject more accurately, Netflix will most likely never support Linux.

The best bet for getting Netflix working on Linux is through something like that Chrome plugin, or perhaps the Firefox alternative.

Chances are, streaming their vids in *nix will always require some kind of 3rd party app (or, I suppose more technically, 4th (or 5th) party app).

Unfortunately, the current "best" solution for bypassing Netflix's streaming limitations is using a Windows OS, so if it's on the computer, it's faster to just reboot and boot into it, watch a bunch of Netflix vids, then boot back into *nix.

I have no idea if it works on Mac systems, though I'm assuming it does.
