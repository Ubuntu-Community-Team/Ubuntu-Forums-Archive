How can i  add subtitles in Totem Player while playing an AVI ?

---

### Post by lovinglinux on 2009-04-13
Just put the subtitle file in the same folder as the avi file and make sure that they have the same file name (except for the extension). The subtitle will be loaded automatically when you start the player.

You might want to try the latest version of [smplayer](http://ubuntuforums.org/showthread.php?t=890228), since it has a much better subtitle support.
