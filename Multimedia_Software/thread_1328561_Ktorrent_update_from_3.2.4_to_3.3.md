### Post by Chronon on 2009-11-16
You could remove your current Ktorrent and install the new one.  You probably will need to extract the archive and run an install script.  However, you will no longer get automatic updating, bugfixes, etc. because you manually installed it instead of using the package manager.

The old version should be managed by the package manager, so you can install it there.
