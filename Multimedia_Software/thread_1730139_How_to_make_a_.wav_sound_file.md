As for software, I'd install Audacity, but there are likely many other programs you could use.

A Post Note:
FYI
I've been tinkering around with the program *Sound Recorder* and *Audacity*.

I've found that *Sound Recorder* .wav files seem to only work with Linux systems.
M$ Media Player won't play .wav files from *Sound Recorder* nor will MAC OS.
At least that is what say my friends to whom I've sent .wav files made from *Sound Recorder*.
Too bad, it's so easy to use.

However, *Audacity* seems to work on M$ OS (XP and older) and MAC OS.
