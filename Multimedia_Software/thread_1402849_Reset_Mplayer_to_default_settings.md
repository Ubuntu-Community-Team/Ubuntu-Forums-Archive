### Post by HappyFeet on 2010-02-09
Go to your home folder and do Ctrl-H. You will now be able to see your hidden config files. Look for the .mplayer folder and delete it. Next time you start mplayer it will go back to default settings.

If you want to remove mplayer, just go to synaptic package manager and search for mplayer. Click it and select **completely remove**. Then hit apply.
