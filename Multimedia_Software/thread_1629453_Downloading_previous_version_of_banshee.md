Where did you get 1.9 from? If you get it from a ppa you can downgrade it to the version in the Ubuntu repository by opening up Synaptic, highlight banshee and go to package > force version.

If not then maybe you can try to upgrade it by adding banshee's ppa to your sources list. Just google it.
