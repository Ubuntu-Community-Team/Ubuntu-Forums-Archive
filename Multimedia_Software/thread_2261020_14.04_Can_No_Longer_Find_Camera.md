I have 14.04 installed on my computer.

I use to have no problems loading the SDXC card from my camera so I could copy to my computer.

The last week, my computer was not able to find the card, or camera.

Today I got this message
```

Error mounting /dev/sdg1 at /media/allen/4588-AAD1: Command-line `mount -t "exfat" -o "uhelper=udisks2,nodev,nosuid,uid=1000,gid=1000,iocharset=utf8,namecase=0,errors=remount-ro,umask=0077" "/dev/sdg1" "/media/allen/4588-AAD1"' exited with non-zero exit status 32: mount: unknown filesystem type 'exfat'

```

Any help will be greatly appreciated.

### Post by mc4man on 2015-01-15
Is exfat-fuse installed?
```
sudo apt-get install exfat-fuse
```
