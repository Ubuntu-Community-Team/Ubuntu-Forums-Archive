### Post by Ulysses361 on 2009-11-06
Run the following commands in the terminal to install gnome-alsamixer:

sudo aptitude update
sudo aptitude install gnome-alsamixer


Then use the gnome-alsamixer controls to decrease volume on the PCM channel.
