### Post by VeeDubb on 2009-10-31
For the purposes of ipod support, you should seriously consider Banshee or Songbird.

Songbird, I'm not a huge fan of personally because even though it's been out for a while, the development is still too active, so it changes too quickly.


Banshee, on the other hand, has really fantastic ipod support, and VERY robust cover art handling.

One of the first things I do when I have to do a fresh install is

sudo apt-get remove rhythmbox  && sudo apt-get install banshee

### Post by Glucklich on 2009-10-31
The thing is... for every new stuff, you have to wait a while for the compatibility. In this aspect, simple stuff works the best. For example, if you had the old k7 to jack adapter for your car, it would be working right now. Since the jack doesn't care if you're using the 1st generation or the brand-new 5th. 
As for the computer, check iTunes compatibility with Wine. Don't know if it works but definitely worth the try, in case of Songbird and Banshee fail as well.
