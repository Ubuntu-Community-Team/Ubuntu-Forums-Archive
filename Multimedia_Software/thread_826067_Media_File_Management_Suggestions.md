Hello everyone,

I'm wondering if anyone knows any good media file management software solutions, especially open source applications.  I've been searching for some but nothing comes close to this commercial application.  [http://www.mediabatch.com/]("http://www.mediabatch.com/")

To quickly sum up the highlight features,
- All users are able to see any available metadata for files &#8211; frame size, codec type, total run time, bit rates and more. Any available detail of a file is revealed without even having to download it!
- View entire contents of a zip file before downloading it.
- Time-code based FLV viewer that allows real-time marker points to store notes and draw on the video!
- Directory Manager web application that allows you to manage your Media Batch lists easily and quickly.
- Your production team can not only see if a file has been viewed, but when and with I.P. stamping.
- Your team and your clients have the ability to post and review notes for each individual file.
- Integrated approval system so clients can officially mark off files.
- Built-in support for file uploading so clients don't have to mess with FTP programs.
- Admin can change column titles, lock out files or even hide columns altogether.
- Integrated version tracker to ensure your system has the latest features and updates.

If you guys know any other solutions that may work like the above features, let me know :)
