### Post by Onesimus on 2010-01-10
I did this using Audacity.  You can directly record from the audio input.

Yes, I know you didn't want to use an editor - but it does work reasonably well.  Selecting each track actually takes far less time than typing in the name of the track, author, etc.  This works particularly well if there is a couple of seconds of silence (or lower volume) between tracks - which tends to be the case.

Hope this helps.
