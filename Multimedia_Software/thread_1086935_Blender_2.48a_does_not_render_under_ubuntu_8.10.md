### Post by Lunatico_22 on 2009-03-04
I can confirm the fullscreen problem, I dont know how to fix it, but what I do is changing to another desktop with "Ctrl-alt-left (or right...) arrow", so the program stays in fullscreen mode in one of the four desktops and my other applications on the others.
I find this arrangement very useful.
 About the rendering, just in case... if you are doing so with Yafray, check in the render properties the "enable XML" box. If you dont do so, you will not see the rendering window until it is complete. Also, when the program is in fullscreen mode you have to press "Alt-tab" to change to the render window. 
About the crashes, maybe the program becomes unresponsive because you started to render something that is behind the blender window. Since you cant see it because of the fullscreen mode, you never notice about when the rendering started. If its not the case, I dunno what could be.

Hope that helps
