Guitar pro is an alternative however its not freeware, its a tad expensive but has very good sound, it does not import anything but midi, if you use Ubuntu 10.10 .. They are working on it they say. 

I have tried to install tableidt in wine not succesfull  Im afraid. Using dll files from windows is a little to advanced for me.. Thats propbably the way to make tabledit or powertab work under Wine. 

Best solution:
Import midi files in tuxguitar or better Guitar Pro 6, export midis from windows guitar programmes such as tabledit or giutar pro.
