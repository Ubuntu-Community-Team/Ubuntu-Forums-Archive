### Post by sadranyc on 2008-10-31
Hi! I'm pretty much a begginer so I'll try to help you with what I know.

Ipodlinux and rhytmbox are totally different things, the first goes on the ipod and the second one is an Ubuntu application.


I have a 5.5g 30gb ipod, and I've had a great experience syncing it with AmaroK. You can get it with 

sudo apt-get install amarok

It has support for playlists, real time fetching of album covers and lyrics, coverflow, and many other features.

BUT, the thing is that this player is made for kde so it'll have to download many libraries to work in gnome. Currently there's a project to "clone" AmaroK for the gnome interface, which indeed is very nice and also has support for ipod with an external plugin. 

The name of this application is Exaile, you can find more info here

[http://www.associatedcontent.com/article/947219/exaile_the_music_manger_for_gnome_thats.html](http://www.associatedcontent.com/article/947219/exaile_the_music_manger_for_gnome_thats.html)

Exaile's webpage: [http://www.exaile.org/](http://www.exaile.org/)
Hope I could help

### Post by manlypain on 2008-10-31
I have a 3rd generation nano.  It works great with RB.  I had to install "gtkpod"

When you plug in the ipod for the first time gtkpod will ask you why type of player it is.  When you have it set up (you actually put the information in your post) it will show up as a usb device.  just don't unplug it without ejecting it.  It writes the table files when you eject (really important).  

Also i had to plug mine into a windows machine at work with itunes installed and format the nano for windows.  gtkpod has a website that goes over all of this.

good luck
