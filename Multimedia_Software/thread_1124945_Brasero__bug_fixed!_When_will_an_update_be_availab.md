### Post by kostkon on 2009-04-14
You can fill a bug report [here]("https://bugs.launchpad.net/intrepid-backports/+bugs") asking *Brasero* to be backported from *Jaunty* to *Intrepid*.

You will need to have the Backports repository enabled in your software sources list (you can enable it in *System &#8594; Adminstration &#8594; Software Sources*) to get the update.
