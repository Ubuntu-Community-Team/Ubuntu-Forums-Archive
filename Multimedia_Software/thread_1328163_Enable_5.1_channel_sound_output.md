### Post by blazemore on 2009-11-16
This is from memory so actual labels may differ.

Right-click on the volume control icon.
Select Settings/options/properties/mixer, something like that
Go to the hardware tab.
make sure the correct output is selected.
Go to the mixer.
Ensure the sliders for the surround channels are displayed and not muted.
