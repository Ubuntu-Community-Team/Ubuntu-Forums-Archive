### Post by gwu777 on 2008-05-10
despite looking for this answer for days before asking, I have found the solution in the last few minutes while searching for another problem on ubuntu!!!

I went into system>main menu and added the icon "screen and graphics" to the other menu, then I was able to chose the correct graphic card for my laptop.

Surprisingly, the system was setup for chose driver by name with the correct driver, but it wasn't until I manually selected it that it became valid (strange!!!) Anyway, I restarted it and now compiz is working perfectly and I am enjoying avent windows navigator!!! Plus I can now logout without problems!!!

The question I have is why is this option "screen and graphics" which used to be in the preference in ubuntu 7.10 is now completely hidden. It is so useful for my problem but as well to change the resolution... All this without passing through commande line and xorg.conf. I am a happy man...
