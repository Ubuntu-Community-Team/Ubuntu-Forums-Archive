### Post by dugh on 2008-10-31
Ubuntu has the latest version of Java installed (version 6 update 10 build 33), but Firefox still uses the old plugin.  They have a new plugin now that has features like draggable applets, jnlp-based applets, and a quicker loading time.  But new applets like these won't work in Ubuntu:
[https://scenegraph-demos.dev.java.net/demo-applets.html](https://scenegraph-demos.dev.java.net/demo-applets.html)
