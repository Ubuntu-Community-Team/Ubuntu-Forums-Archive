### Post by evilsoup on 2013-04-17
I don't have a smart TV, I have a PS3, but that's a DLNA device so it should work the same way. Basically, yes, you set up your media server software on the computer and select which folders should be served up; and then you use your client (the TV) to select the server, and you can browse to the file you want to play.

I use ps3mediaserver (you may need to add [this PPA](https://launchpad.net/~happy-neko/+archive/ps3mediaserver)), which is as user-friendly as these things get. Don't worry about the name, it should work with any DLNA device.
