The following script I mostly swiped (and I don't think I got the --pos 0x998 right at all) from exactly where I forget now and saved it as 45custom_xrandr in /etc/X11/Xsession.d - but, it seems to be ignored anyway. Maybe I needed to add it in Manual autostarted applications in Default applications for LXSession to have any affect?

```
xrandr | grep VGA | grep " connected "
if [$? -eq 0]; then
    # Hisense HDTV is connected
    xrandr --output VGA --mode 1152x864 --pos 0x0 --output LVSD --mode 1680x1050 --pos 0x998
    if [$? -ne 0]; then
        # Something went wrong. Autoconfigure the HD Color Display Monitor and disable the Hisense HDTV
        xrandr --output VGA --mode 1152x864 --output LVDS --off
    fi
else
    # Hisense HDTV is not connected
    xrandr --output LVDS --mode 1680x1050 --output VGA --off
fi
```
