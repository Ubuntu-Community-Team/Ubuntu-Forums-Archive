### Post by BicyclerBoy on 2012-04-23
For extended desktop (windows like) you can use default unity desktop & nouveau YMMV.

But to get the best performance using nVidia driver..
The latest nVidia twinview is now meant to support multiple monitors (>2) on consumer cards.
Your pro-sumer CAD card should have supported Mosiac..
Your card has VDPAU feature set B1.

Don't use xinerama option (Xorg extension), this breaks openGL.
The "Rotate" option will affect all screens that are part of twinview setup.
The "Rotate" option has impact on openGL.

If you want to rotate one screen, separate X screens or Mosiac might be better.
Never used or have read any thread on mosiac tho.

To help debug post your file as attachment:
/var/log/Xorg.0.log
