### Post by shayguitarra on 2009-04-06
A lot of people recommend floola, but I had terrible problems with it on kubuntu (although that may have just been me.). If you're using kubuntu type gtkpod into adept and install gtkpod-aac. I've been using that with few problems purely for video transfer. You can tag your videos for 'skip when shuffling', as 'music video' etc.

Thanks for the tip on Handbrake, Wolfen 69. Conversion has been a bit of a pain on occasion with Avidemux.
