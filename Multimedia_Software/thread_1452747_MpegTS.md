### Post by Nikkus on 2010-04-12
Ok! I get it!!! Well for the subtitles!!! :)

Using ProjectX

Searching a few hours, find and test this thinks:

First of all, we need install ProjectX, see the link bellow:

[http://ubuntuforums.org/showthread.php?t=203330](http://ubuntuforums.org/showthread.php?t=203330)

To extract, i make a little and very very simple tuto, based in a spanish version([http://www.mundodvd.com/showthread.php?t=5375](http://www.mundodvd.com/showthread.php?t=5375)).. Let go!

1) Open the program! :P

2) File / Add(ctrl-o) and load your .ts file
[IMG]http://www.nikkus.com/projectx/002.png[/IMG]

3) Presettings / Presettings(ctrl-s)
[IMG]http://www.nikkus.com/projectx/003.png[/IMG]

4) In the presettings window, choice "output" and look the image:
[IMG]http://www.nikkus.com/projectx/004.png[/IMG]

5) Click in "subtitle" and put this options:
[IMG]http://www.nikkus.com/projectx/005.png[/IMG]

After that, go to File / Close(Alt-x), and click in prepare..., look:
[IMG]http://www.nikkus.com/projectx/006.png[/IMG]

Now, in ProcessWindow, click in action: demux, and after this click in the "play/pause" button, look:
[IMG]http://www.nikkus.com/projectx/006.png[/IMG]
