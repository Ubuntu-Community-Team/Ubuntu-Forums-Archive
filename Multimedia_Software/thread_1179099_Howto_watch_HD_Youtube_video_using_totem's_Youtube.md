### Post by kwokyinc on 2009-06-05
I am not sure whether this has been posted or not.

I found the Linux version Adobe flash player for firefox very sluggish, especially when playing HD videos (e.g. those from youtube). 

Downloading the flv / mpeg files from youtube and then watch them locally using other video players is an option but this method is not efficient.

Totem has a youtube plugin that allows direct viewing youtube's video. Unfortunately, currently it (totem version 2.26.1) does not play HD video from youtube. After searching Google, I found the following solution posted by a Japanese Linux user, which enable viewing HD videos directly from youtube from Totem. :P

I tried it and it works for me. Hope this post can help those who are looking for the same thing.  :P



[Patch to totem's youtube plugin (translated into English by Google)]("http://64.233.189.132/translate_c?hl=en&sl=ja&u=http://subtech.g.hatena.ne.jp/y_yanbe/20090423/1240550210&prev=/search%3Fq%3Dtotem%2Byoutube%2Bhd%26hl%3Den&rurl=translate.google.com&usg=ALkJrhjVWdqEC8y2nR1vAoGeNziX-uD0Pg")

### Post by lovinglinux on 2009-06-05
You can also install [Greasemonkey extension]("https://addons.mozilla.org/en-US/firefox/addon/748") for Firefox and load [this script](http://userscripts.org/scripts/show/41722). It will allow you to play YouTube videos with mplayer plugin instead of Flash, including HD, and also provides a link to download the video.
