I'm moving my music collection from my windows desktop to my newest ubuntu installation, but I have a large amount of windows media playlists, custom ratings, etc stored on the windows computer. ### Post by pythonscript on 2009-09-06
Apparently, wmp stores its ratings in a proprietary format within its configuration settings, so you can't extract them. The playlists, however, can be read by a bunch of other media players if you convert them to .m3u.
