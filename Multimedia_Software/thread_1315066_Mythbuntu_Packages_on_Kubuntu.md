### Post by fenian on 2009-11-05
[Click here]("http://mythbuntu.org/download/getmythbuntu.php") to download base mythbuntu packages.

After you install the package go to the menu System>Administration>Mythbuntu Control Centre.

Click System Roles and choose Primary Backend and Desktop Frontend.

Click apply and it will download the mythtv packages.

Next you need to set up the backend which is well described in the Mythbuntu manuel available [here]("http://www.mythbuntu.org/documentation/mythbuntu_8.10_installation.pdf").

That should get you started if you have further questions you are probably better off posting in the [Mythbuntu section]("http://ubuntuforums.org/forumdisplay.php?f=301") of Ubuntu forums.
