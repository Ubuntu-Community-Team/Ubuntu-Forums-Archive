### Post by sudodus on 2013-02-17
Welcome to the Ubuntu Forums :-)

An alternative to getting a new graphics card (unless it's broken) is to get a flavour of Ubuntu 12.04, that runs well enough with your present card.

I suggest that you try Lubuntu with the ultra light-weight desktop environment LXDE. I use it myself on aging computers, pentium m, p4, amd athlon and intel xeon. It really makes the graphics perform better (but of course, you won't get the eye-candy of Unity with compiz).

- Download the Lubuntu iso file, make a CD or USB install drive and test it live before decide if you want it

- or install lubuntu-desktop into your present installed Ubuntu system

```
sudo apt-get install lubuntu-desktop
```
and select desktop environment at the log in screen.

An intermediate alternative is Xubuntu with the light-weight desktop environment XFCE or
```
sudo apt-get install lubuntu-desktop
```
