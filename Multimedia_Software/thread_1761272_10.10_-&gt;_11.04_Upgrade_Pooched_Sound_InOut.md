### Post by lidex on 2011-06-26
Your kernel sound modules aren't loading. Which soundcard are you wanting to use?
Try modprobing.
```
sudo rmmod snd-ca0106
sudo modprobe snd-ca0106
```


```
sudo rmmod snd-hda-intel
sudo modprobe snd-hda-intel
```
