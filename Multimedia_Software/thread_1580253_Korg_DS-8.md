### Post by cchhrriiss121212 on 2010-09-23
According to wikipedia:
> By using one of the optional non-volatile (but battery-powered) KORG RAM  cards MCR-01, MCR-02, or MCR-03 this program memory can be expanded by  extra programs and combinations: 100/10 for the MCR-01, 200/20 for the  MCR-02, or 400/40 for the MCR-03. Programs can be backed up and received  via standard MIDI dumps.
So I'm guessing that any old flash card will not be compatible, but you could always try. It can store 100 programs internally anyway, which should be enough for most people.

What is it you want to do with the synth+PC? You could record it's sound output, or you could connect it using MIDI to use an pattern editor like Rosegarden.

### Post by cchhrriiss121212 on 2010-09-23
Yeah I saw that the cards are pricey, they cost twice that where I live, but that is because they are not being manufactured any more. There should be software out there that allows you to upload patches to the synth, but [this]("http://www.kidnepro.com/KN/Korg/DS8-707.html") is all I could find (for Windows/Mac only).

If you want new sounds then I would start making them yourself anyway, the presets and patches you will find for sale should just be a starting point for creating your own sounds.
