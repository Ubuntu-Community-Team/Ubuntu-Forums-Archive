What I found the problem to be after searching the Internet is not with Spectrum, but it's an digital rights license issue. It seems that Linux doesn't have a plugin that is licensed to play some forms of video streams. Windows is licensed for this. So you have to either find a plugin that will work on Linux that will allow you to play the stream, or play it on Windows. There is a trick that someone posted on a site that requires you to use Wine on Linux to run a windows plugin behind Firefox. This allows you to play the streams, it's a little complicated to get setup and may not be worth it unless you are only using Linux. I don't remember what website I found it on or I would post it.
