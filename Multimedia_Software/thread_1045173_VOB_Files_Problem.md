### Post by stderr on 2009-01-20
VLC should be able to show VOBs... as should Totem, although you might need to install some [extra codecs ]("https://help.ubuntu.com/community/RestrictedFormats/PlayingDVDs"). 

If you don't have VLC, you can get it with:

sudo apt-get install vlc
