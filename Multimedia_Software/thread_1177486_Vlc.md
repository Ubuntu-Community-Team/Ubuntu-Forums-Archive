For much of version 0.9.9 this is in fact the default action for vlc and this is the version that is shipped with Jaunty. The old single window behavior has been restored for vlc 1.0 rc versions so you have a few choices:

[LIST=1]
[*]Search out some of the PPAs holding the vlc 1.0 rc versions, or patched older versions of vlc
[*]Waiting a while and seeing if the new version, when it is eventually released, will be backported to Jaunty
[/LIST]
