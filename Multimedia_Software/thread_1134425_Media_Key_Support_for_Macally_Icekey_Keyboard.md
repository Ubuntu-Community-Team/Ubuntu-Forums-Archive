### Post by polyrhythmic on 2009-04-25
I too just picked up a Macally Icekey (in black), and at first thought there was no keyboard support.  However, if you go to System --> Preferences --> Keyboard Shortcuts you can remap the keys as needed.  On my 8.10 install eject was the only key that worked without remapping.

However, similar to what has been [reported with Thinkpads]("https://bugs.launchpad.net/ubuntu/+source/acpi/+bug/272292/comments/17"), mute reports as XF86ScreenSaver and will lock the X screen whenever you press it.  I'm still looking for a workaround for this bug...
