### Post by quixote on 2010-02-25
I can't answer most of your questions, but about the boot process, you have to load the X windows graphical interface in any case because of the application you want to run.  That's what takes most of the time.  I'm not sure how much extra gnome takes, but you'll make your life simpler if you don't worry about it :D .  Something like a media center is going to need some sort of desktop manager anyway.

Then, I would imagine that to get moovida launched on startup, all you'd need to do is add it to System > Preferences > Startup Applications.  You probably want it to run maximized which may require a switch appended to the command. People who know moovida could probably help with that.
