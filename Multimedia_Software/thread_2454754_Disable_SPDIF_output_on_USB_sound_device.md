Hello everyone,

I use a USB sound box by LogiLink to connect a sound system by USB on a dual boot Windows 10/Linux setup that also supports optical output using a S/PDIF connector. On Windows, the driver allows to disable the S/PDIF output as long as this port is not required such that there is no flashing red light. On Linux however, so far I found no way to disable it. I found some postings where a similar problem is described and the reported solution is running 'sudo alsamixer' and to mute the S/PDIF port on the respective card. Hence I tried to do the same, but running 'sudo alsamixer' and selecting the USB Sound Device by F6 does not give the option the mute the S/PDIF output, the port is simply not listed.

Is there any other way how to mute the S/PDIF connector or turn off the flashing red light?
