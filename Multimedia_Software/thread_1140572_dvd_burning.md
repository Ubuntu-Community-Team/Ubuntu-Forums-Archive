### Post by Ralphie on 2009-05-02
I don't know how you can cut an iso automatically, but I can think of a work around for you. 

Use a program called dvdshrink, it is a (freeware) windows program that works under wine pretty well.

You can either shrink the whole thing to fit on one disk, or you can re-edit it and cut it into two parts manually, saving each separately and then burning them separately for full quality. 

Another tip if you are shrinking it, remove extra audio tracks that you wont be using (there are check boxes to do this), doing this can sometimes cut down the size by a few hundred mb's
