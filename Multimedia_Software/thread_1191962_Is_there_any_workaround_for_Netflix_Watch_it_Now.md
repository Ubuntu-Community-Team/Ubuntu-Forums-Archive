### Post by frabjous on 2009-06-19
I think the problem is that Netflix uses Microsoft's Silverlight for DRM protection, which is a Microsoft product, and Microsoft isn't allowing it to be used with Linux, and darnitall if Netflix isn't in their pocket.

Some people have gotten it to work with a virtual OS, as with VirtualBox. Currently the only known solution, as far as I know. There are rumors that Netflix may change DRM methods, but I ain't holding my breath.
