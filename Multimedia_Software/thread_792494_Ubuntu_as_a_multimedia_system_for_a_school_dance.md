### Post by Eddie Wilson on 2008-05-13
I do this sort of thing all the time and I use Rhythmbox. It works very well for that sort of thing. I hook my computer system into an outside sound device,(Pevy sound system)and I set the q as a side panel. I like the library setup and you can even fade one song into the other. I've tried others and they were ok but Rhythmbox works the best for me. Its very stable also. When I used Windows for this  same purpose a long time ago I had some problems every once in a while but I've never had the first problem with Ubuntu or Rhythmbox. :guitar:
