### Post by EXCiD3 on 2008-06-19
Ok the first part worked...try using a different resolution in there. You shouldnt have seen any effect from the 2nd part because it was just removing things you werent using anyways.

Another thing to try is run "sudo nvidia-xconfig" in terminal to have it autoconfigure your xorg...but this was already done when you installed the driver in the first place so the xorg may not be any different than it was before your modifications.

Also try "sudo nvidia-settings" in terminal to edit your resolution. Make sure you Save to X Configuaration File and check the box to Merge. If thats not installed "sudo apt-get install nvidia-settings" in terminal should install it for you.

If you want the Nvidia logo (which wont make any difference) remove the NoLogo "True" line from your xorg.conf.
