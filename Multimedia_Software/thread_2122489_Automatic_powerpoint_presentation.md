### Post by Cheesemill on 2013-03-05
If you want to show PowerPoint files then your best option is to install Windows.

For anything except the simplest of presentations you will get formatting errors when trying to use any software except PowerPoint.

If you do want to try viewing the files with Lubuntu, then install LibreOffice Impress and see how far you get.
