### Post by bumanie on 2009-06-02
I don't use linuxmce at this point, but have done some reading up on it. It is a huge program, if one were to use all its functions - home automation, security, voip etc...etc. It can be used alone as a pvr/home theatre unit etc. You do need a nvidia gpu to use it - 6100 and above (I think). From what I have read it is easier to setup than mythtv - no dependency issues as is often found with myth, however, post installation it can take some time before it shows anything on a monitor/tv screen - so don't panic if the monitor/screen is black for a period of time. After that, there are a number of menus to setup, depending on what you need linuxmce to do. As far as I know it should be able to do anything your other pvr can do. No harm in giving it a try, but you should do some reading to make sure the hardware you have is compatible - it's hardware compatibility is more restricted than a 'normal' *buntu installation. There is also a dedicated forum for linuxmce if you get stuck, someone there would be able to help you.
