### Post by lykeion on 2010-10-24
[Openshot]("http://www.openshot.org") maybe? Install in Lucid and later with this command
```
sudo apt-get install openshot
```

If you use Lucid and like to install the latest version from PPA, go to [this page]("https://launchpad.net/~openshot.developers/+archive/ppa") and follow the instructions.
