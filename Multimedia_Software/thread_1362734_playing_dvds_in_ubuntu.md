### Post by StephenWoods69 on 2009-12-23
Hi,
   I just installed VLC media player and have found it very good.
To install open System->Administration->Synaptic Package Manager
and search for VLC.
If you decide to install it then you can set VLC up to load when you insert a DVD by opening Places->Home Folder->Edit->Preferences->Media tab
Then under media handling select the drop down for DVD video and select Open VLC media player.
VLC will then auto-play when you install a DVD.

Hope this helps.
