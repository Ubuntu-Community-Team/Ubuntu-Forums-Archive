### Post by cozmicharlie on 2009-07-17
The ATI drivers are a little tricky to install but not that difficult.  The driver under the Hardware manager in Ubuntu is old so install the newer version from ATI/AMD web site.  They have instructions and there are lots of posts in the forums to help.  As per the performance - I like the card.  I added it to a build for an HTPC and I was very satisfied with the performance.  It is a good card for the price and I believe you will be very pleased with the performance.
