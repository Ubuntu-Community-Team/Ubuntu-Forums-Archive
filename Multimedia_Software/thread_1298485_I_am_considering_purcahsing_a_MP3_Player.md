### Post by Chronon on 2009-10-23
SanDisk players have a good track record for supporting MSC mode (i.e. behaving as a UMS device).  The Clip+ supports this so you should be able to just drag and drop media onto it from your file browser.  There was a problem with gphoto2 preventing mounting of some UMS devices under Jaunty.  If you search around you can find workarounds for that, should you experience it.
