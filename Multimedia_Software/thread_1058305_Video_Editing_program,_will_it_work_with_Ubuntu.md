### Post by Dies on 2009-02-02
> **Miss_Hopeful said:**
> So I could download Windows Meida player/movie maker and it would work with Ubuntu? (Sorry, like I said, I know nothing about computers, or Ubuntu. Lol.)

No. Forget about Windows software.

a.) It more than likely won't work, or if it does work through wine it may not work right.

b.) You don't need it anymore. That's the whole point of using a different OS.  ;)

If you follow the instructions on that page, the applications that are already on your machine, "Movie Player", "Rhythmbox", etc. will be able to play almost anything you're likely to come across.

For very simple video editing try installing Pitivi or Avidemux, for more advanced stuff try simtaalo's suggestions or Cinelerra.

BTW almost all suggestions about software to use that you are likely to get on a Linux forum are going to be "Free" software and most you will be able to install with a few clicks using Synaptic Package Manager which is already on your system either in

Applications > Add or Remove Software

or

System > Administration > Synaptic Package Manager

Good Luck and read through the link I provided. ;)

### Post by issih on 2009-02-02
Bear in mind that things don't work like windows....Have a read of the ubuntu documentation:

[https://help.ubuntu.com/8.10/index.html](https://help.ubuntu.com/8.10/index.html)

That will let you begin to understand how things work around here..after that things will make more sense.

The advice about reading up on restricted formats is quite correct..essentially you need to install the ubuntu restricted extras package, and ideally the libdvdcss package too, after that most of your multimedia headaches will go away, as you will have flash player, java and the w32codecs for audio and video playback.

Take the time to learn how ubuntu differs from windows and you should get along just fine :)

good luck
