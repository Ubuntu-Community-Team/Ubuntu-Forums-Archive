### Post by Melcar on 2009-01-20
Type alsamixer in a terminal and you can access all your mixer settings. If you really want a GUI, you can install alsamixergui from Synaptic.  Not sure about Pulseaudio (it's only a sound server, so I don't know why it would have one).
