### Post by Geogriffith on 2008-05-18
Go to the Applications menu, select "Add/Remove", change the "show" drop-down to list "All available", and do a search for "flash". Check the tickbox for "Macromedia Flash" and click on "Apply changes". You might need to restart firefox.
