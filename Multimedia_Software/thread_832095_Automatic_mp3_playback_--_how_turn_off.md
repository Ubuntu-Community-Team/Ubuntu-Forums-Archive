### Post by mc4man on 2008-06-17
actually I think your talking about the totem-preview feature. You don't need to click on it, having the cursor over will start the preview. If you highlight an audio file and do a shift/delete then the preview continues even though the file has been deleted - when the song plays thru it'll be gone for good. Better to r. click/delete - r. click turns off preview. As far as disabling the preview feature haven't seen anything yet - maybe in configuration editor

edit : To turn off preview configuration editor -> apps -> nautilus -> preferences -> preview sound, edit key to never
