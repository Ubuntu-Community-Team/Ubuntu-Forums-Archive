### Post by QIII on 2017-11-03
Hello!

OpenELEC is a Linux distribution, an OS, in its own right.  It isn't a "program" to install in Ubuntu.  You can, however, dual boot and choose between OpenELEC and Ubuntu or run it in a virtual machine.  I wouldn't do the latter as VMs provide terrible graphics performance.

OpenELEC uses Kodi as its "desktop".  You can install Kodi on Ubuntu.

[This]("http://ubuntuhandbook.org/index.php/2017/02/kodi-17-released-install-in-ubuntu-16-10/") gives a pretty good tutorial for installing Kodi.

### Post by monkeybrain20122 on 2017-11-03
You don't need or want OpenElec. As Qill said it is just an OS dedicated for Kodi basically, I have installed it in a usb for a friend and the mouse and touchpad don't work and need to use keyboard to navigate so that was no go. It is much better just to install Kodi from the software centre. After that you'll have a an option to log into a stand alone kodi session in your logijn screen when you reboot.

There is also a kodi ppa if you want the up to date version [https://launchpad.net/~team-xbmc/+archive/ubuntu/ppa](https://launchpad.net/~team-xbmc/+archive/ubuntu/ppa)
