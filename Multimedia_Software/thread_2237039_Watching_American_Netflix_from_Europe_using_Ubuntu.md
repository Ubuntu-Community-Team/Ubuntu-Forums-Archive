### Post by yancek on 2014-07-30
Pipelight is software that you can use on some Linux systems to watch Netflix.  The link below explains how to install it on Ubuntu.

[http://pipelight.net/cms/install/installation-ubuntu.html](http://pipelight.net/cms/install/installation-ubuntu.html)

You also need to identify you browser as running a windows version.  Some information on User Agent Switcher is at the bottom of the page in the link below:

[http://www.webupd8.org/2013/08/pipelight-use-silverlight-in-your-linux.html](http://www.webupd8.org/2013/08/pipelight-use-silverlight-in-your-linux.html)
