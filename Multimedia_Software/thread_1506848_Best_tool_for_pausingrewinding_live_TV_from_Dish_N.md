### Post by lovinglinux on 2010-06-10
You might want to try my Firefox extension, [FoxMediaCenter](http://foxmediacenter-extension.blogspot.com). When you start a recording with it will make the file immediately available to watch in the playlist manager. So you can watch, pause and resume, using your favorite video player. 

Keep in mind this extension is still experimental.
