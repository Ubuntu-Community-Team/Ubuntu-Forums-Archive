### Post by pidou46 on 2013-06-17
Hi Salicet,

Personaly I use MuseScore, it work out of the box with intergrated synthetiser and a soundfont (not perfect but usable)

If you read the standard notation system, you will like it because the playback is controled by the score notation (loops, loudness, etc..), as you expect as a musician.

It is possible to integrate it in a more complex jack audio setup so you have some room to play with.

It is in rapid developpement so bug are relatively quickly solved.

There is also a windows and OS X version, so you can share with your freinds.

Last but not least it is in Ubuntu repository
