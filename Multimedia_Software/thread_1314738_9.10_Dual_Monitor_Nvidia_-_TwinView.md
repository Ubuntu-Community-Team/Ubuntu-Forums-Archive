### Post by jhayes on 2009-11-04
did you use the binary blob fron nvidia, or the driver upgrade from ubuntu?
i usually get better results from downloading the file from nvidia. 
something like 
sudo /etc/init.d/gdm stop

sudo sh ./NVIDIA-Linux-x86_64-185.18.36-pkg2.run

sudo reboot

then 

sudo sh ./Nxxx --uninstall
change permissions of /etc/X11 to
sudo chmod g+x /etc/X11; sudo chgrp jon /etc/X11
assuming jon is primary user.

then you can open it save settings etc. and have it work
