First, your first post said you wanted all your music transferred, no? Second, I didn't know you intended to network these computers. Yes, if they are networked you can just copy all the files. I believe Ubuntu has a default folder called "Music", and you can dump all the files there.

The bad news is that most music files will not play in Ubuntu without first installing some extras. Open the Software Center and search for "Ubuntu Restricted Extras" and install it. For AAC, try the gstreamer plugins.
