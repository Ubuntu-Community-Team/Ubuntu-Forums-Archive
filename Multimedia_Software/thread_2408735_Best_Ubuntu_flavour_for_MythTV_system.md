### Post by deadflowr on 2018-12-22
No.
There's no real recommendation of which flavor you want to run.
The closest is perhaps Xubuntu since that's been more or less the mythbuntu go to default for ages now.

On regular Ubuntu 18.04 it has an option for minimal installs which does what you want (not install cruft like libreoffice, etc)
But Xubuntu doesn't have that yet, But does offer a method of installing a minimal xubuntu install,
from Xubuntu developer unit193:
[https://unit193.net/xubuntu/]("https://unit193.net/xubuntu/")

### Post by Tadaen_Sylvermane on 2018-12-23
My living room HTPC is my Mythtv backend & direct boots gui to Kodi. All built on an Ubuntu server base. No other gui stuff is involved. You don't need a full desktop to run it unless you are stuck to the mouse and keyboard. One can use an app on your phone to remote control Kodi. Works great for me. I find the Myth front end software to be lacking.
