### Post by NotMarkus on 2010-05-21
Rhythmbox is a GTK app, so you can just change your GTK settings and they'll affect Rhythmbox.

Go to System > Preferences > Appearance,

or type ```
gnome-appearance-properties
``` into a terminal. Then, under the Fonts tab, change the Application Font size.
