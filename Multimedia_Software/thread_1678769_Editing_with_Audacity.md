### Post by In2Blues on 2011-01-31
I'm no expert, but I do use the program occasionally.

Once you have highlighted the section(s) that you want to remove, simply hit the "delete" button and it will disappear. That's how I always do it.

You can then save the new file (I'm assuming that you're working on a copy -- not the original -- in case something goes wrong).

Hope this helps.
