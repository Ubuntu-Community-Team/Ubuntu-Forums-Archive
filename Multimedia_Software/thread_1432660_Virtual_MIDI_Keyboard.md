### Post by babarosa on 2010-03-18
You'll find versions of vmpk here (I use it on **_Karmic_** and it is working very fine):

[https://launchpad.net/ubuntu/+source/vmpk](https://launchpad.net/ubuntu/+source/vmpk)
After "opening" the folder "0.3.0-2", go for "vmpk_0.3.0-2_i386.deb (314.9 KiB)". Right click and download it.

Here is another version (I guess the latest version):
[https://launchpad.net/~philip5/+archive/extra/+index?start=75&batch=75](https://launchpad.net/~philip5/+archive/extra/+index?start=75&batch=75)

Read on this site how to add this repository to your system. I do not know, if this version works on Jaunty too.
