### Post by habana on 2012-07-18
I have installed Banshee and set it as the default sound player in Details/Default Applications/Music. However, when I click on any MP3 file, it opens with Rhythmbox. If I right click on it, I can select and play it with Banshee but it is not the default. How can I fix this without removing Rhythmbox completely.

---

### Post by habana on 2012-07-19
I fixed it. I had to right click on a an MP3 file select Properties/Open with/Banshee/Set as default. I will mark this as solved.
