### Post by chiwi on 2010-05-10
If you are doing video editing, I assume you are also rendering under XP, then you'll need full power of your video card as well.

I've read somewhere (i'm not 100% sure though), that Virtualbox will not give you the whole set of features the your hardware supports, even though you installed the drivers. I believe this is a key difference between VirtualBox and VMWare for instance. I think VMWare supports all features if you installed the drivers.

I haven't tried VMWare yet, i'm a happy VirtualBox user who only uses it to open M$Office docs when OpenOffice is not an option. :)

What you are trying to do for sure it will work. How fast, I don't know. You may want to go to Virtualbox web site and read the documentation to see what's supported and what's not.

Good luck.
