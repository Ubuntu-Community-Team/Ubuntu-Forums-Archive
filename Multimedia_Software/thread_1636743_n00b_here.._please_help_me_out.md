### Post by wannacme16 on 2010-12-04
I assume that your install was an oem install (option available on the screen after the language in the install process) as a regular install requires account info (your username and password) to actually install the system. First, you're gonna need root control to install and remove packages. Top left of your screen should be your menu (app., places, system) under system go down to administration and choose user and groups. I assume also from the fact you already installed a new set of drivers that you have some root abilities so adding a user should be possible. Here is where you enter in your info and place whatever privileges you want for yourself such as root. This is assuming your install was an oem install, when you go to the user and groups you will see who holds the keys to this system (oem or ?????). If your install was not an oem install and you are not the one who holds the keys there is little you can do apart from a fresh install where you enter your info in the install process, otherwise your locked in whatever position your in. Linux is primarily a multi user system where the admin controls users and users privileges, sort of like daddy and kids. So first who's system is it? Who holds the keys?
