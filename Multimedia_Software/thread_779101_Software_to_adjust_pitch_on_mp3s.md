### Post by bcrowell on 2008-08-05
Soundstretch works great for me. I think audacity uses soundstretch's library to do this, so the results should be of identical quality. Using soundstretch from the command line is probably a lot quicker and easier than starting up audacity, importing the file, doing the pitch change, and then re-exporting the file.
