### Post by will198 on 2008-08-27
Hi all,

Finally i get the answer...

I have to disable this option:

	Option		"ConnectedMonitor" "DFP,TV"

I don't know why but when write this option my tft monitor go wrong.

In fact i only have to add this option to the device section:

	Option		"TwinView" "1"
	Option		"SecondMonitorHorizSync" "30-50"
	Option		"SecondMonitorVertRefresh" "60"
	Option		"TwinViewOrientation" "Clone"

The other options i belive are automatically detected,  and i don't have modify the other section (monitor, screen, etc)
