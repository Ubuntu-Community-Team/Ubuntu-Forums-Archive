### Post by gordintoronto on 2013-08-18
If you can figure out what command starts the Catalyst Control Center, you can start it with administrative rights by entering the command: sudo [catalyst command]

It's probably: sudo amdcccle

I suspect this will also let you set up two screens.

As a comment, when I clicked on the dash in Ubuntu 13.04, and typed "cata," I was given the option to run Catalyst Control Centre with administrative rights. If that did not happen for you, it might not be installed properly.
