### Post by Topfs on 2008-09-22
Hi, I and team-xbmc just wants to inform all you media lovers that XBMC Atlantis have now reached Beta1 and we have alot of exciting stuff on the table!

First of this is the first Cross platform beta and works on Xbox, Windows, Linux, OSX (both tiger and leopard)

News
    * OSX Tiger support
    * iTunes and iPhoto integration (experimental)
    * A new Hidefinition version of the beloved PM3 skin, PM3.HD
    * XBMC Live, a complete solution on a CD that will boot and run XBMC without touching your hard drive


XBMC Live will boot on x86 and sports support for MCE remote out of the box. although many more remotes, WiiRemote, J2ME and PS3 Blueray and Gamepad and any LIRC remote, is rather easily [installed]("http://xbmc.org/forum/showthread.php?t=37443")

Read more on xbmc.org and download on xbmc.org/download
[ubuntu specific installation guide]("http://xbmc.org/forum/showpost.php?p=185738&postcount=1")

Hardy specific:
```
sudo echo "deb http://ppa.launchpad.net/team-xbmc-hardy/ubuntu hardy main" >> /etc/apt/sources.list
sudo echo "deb-src http://ppa.launchpad.net/team-xbmc-hardy/ubuntu hardy main" >> /etc/apt/sources.list
sudo apt-get update
sudo apt-get install xbmc
```
