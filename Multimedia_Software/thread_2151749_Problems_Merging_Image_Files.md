### Post by ajgreeny on 2013-06-05
Gimp will do that without too much fuss.

Start a new empty image the size you want, then open all the pics you want merged into it and make sure they are all the same size (in pixels) by using **image ->scale** if needed, or you will have difficulties when you try to merge them together.

Now cut out the parts of images that you want with the appropriate tool for the shape you want, and using "feather edges" from the toolbox if necessary.


Finally copy each selected part image to the new image by pasting as a new layer so you can play around with positioning and so on as needed.

Jst play around for a while and I'm sure you'll soon see what the various options can do, and also that my suggestion is not the only way, but there are other ways as well to end up at the same point.
