### Post by sudodus on 2013-02-23
Welcome to the Ubuntu Forums :-)

I suggest that you start using ***DigiKam***. It is a powerful photo manager, that can help keep track of several thousand pictures. Shotwell is a simpler program, that comes with Ubuntu.

DigiKam belongs to the KDE desktop of Kubuntu, but it can be run without problems in any of the Ubuntu flavours. The necessary libraries from KDE will be installed automatically, when you install it.

```
sudo apt-get install digikam
```
