### Post by dv3500ea on 2010-05-11
You need to download a 'soundfont' file from the internet. you can search for 'free soundfonts'.

You then need to go into lmms and go to Edit -> Settings
click on the folder icon
Locate the file where it says DEFAULT SOUNDFONT FILE

### Post by bilbea on 2010-11-14
Or... search for, download and install package (apt-get, aptitude or synaptic) - fluid-soundfont-gm

Then start lmms - go to Edit -> Settings -> [folder icon]
Last one should be soundfont - point it at ...
/usr/share/sounds/sf2/FluidR3_GM.sf2

Done! :D
