### Post by johnaaronrose on 2012-12-20
I've written a GUI for get_iplayer in Gambas3. It allows selection of a  program followed by downloading it. It calls get_iplayer  with a --raw  parameter since it doesn't work without it. Its .deb  package (which  includes Gambas3 runtime) is  downloadable from:
[http://dl.dropbox.com/u/3928731/irecorder.deb]("http://dl.dropbox.com/u/3928731/irecorder_0.0.39-1_all.deb")

Selecting Default mode results in a TV programmes being recorded as a   .flv  file. Use Mobile Media Converter / HandBrake / VLC / WinFF if you   wish  to convert to another format (e.g. to play on the iphone). 

I'd be grateful if people who try iRecorder would let me know their   experiences with it, particularly if they wish to suggest enhancements.
