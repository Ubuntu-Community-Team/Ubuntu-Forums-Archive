### Post by Malcy on 2008-12-16
The easiest thing is to use an app like Handbrake or a script like mts2avi to transcode the AVCHD file into a format more widely used by editing apps. Then you can edit it in a NLVE like KDEnlive, Cinelerra etc.

Handbrake: [http://handbrake.fr/](http://handbrake.fr/)

KDEnlive: [http://www.kdenlive.org/](http://www.kdenlive.org/)

p.s. KDEnlive is supposed to be able to edit AVCHD files but I haven't managed it yet.
