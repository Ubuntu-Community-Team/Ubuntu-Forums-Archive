### Post by pytheas22 on 2008-05-04
If you've already installed Ubuntu and you're using an up-to-date version, accessing your Windows partition should be as easy as going to Places>Computer and double-clicking on the icon that represents your Windows partition.  You should not need to do anything else.

Until about a year ago, you would probably have had to create a FAT32 partition to share the files, but that's no longer necessary, since reliable NTFS drivers now exist for Linux.
