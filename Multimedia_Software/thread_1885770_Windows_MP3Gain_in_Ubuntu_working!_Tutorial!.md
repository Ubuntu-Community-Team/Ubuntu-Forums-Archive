### Post by utahman1971 on 2011-11-23
I installed the easyMp3Gain on ubuntu, and it kept freezing laptop so I had to restart laptop to use it, but it kept not working. 

This is a Tutorial on how to get it to run:

I wanted to use the real one that works on Windows operating system. So I went to where you could download the Mp3Gain from their website for Windows, and downloaded the executable full version file to my Ubuntu. I right clicked to properties and made it a executable. I then right clicked the file again, then chose to open with Wine. As soon as I did it, it opened as an install. I installed it like it was Windows. It works and no problems.

The only thing is when you open a folder you have figure where you files are. I got an external hard drive so I had to go to next thing right under desktop with + and extend it, then go to where it says media and extend. If that is where it is then select folders or folder to add mp3's. If you have your files under your username and music folder under ubuntu then it should be easier.

I hope this helps anyone else that wants to use the real mp3Gain for your mp3's.
