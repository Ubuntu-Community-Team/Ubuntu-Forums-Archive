### Post by TheFu on 2012-12-28
I've been editing videos for 10+ yrs - usually just cutting commercials out. There is no copy/paste as you describe, but you can mark scenes and insert new scenes inside videos using other tools. You just need a imported list of scenes to be inserted.  I've never used either tool you mentioned.  I think you want a "non-linear video editor."

I wish I could recommend a linux-based video editor. I cannot. I use Linux for everything in the process **except** editing.

For a linear editor, the easiest way is to create clips and save those into separate files, then merge the total video back together. Almost every video tool will merge multiple files into a single file - ****provided the resolution and frame rates for all material involved match.****

I have no idea what "previewing" has to do with any of this.

I hope I'm being clear. Ask for a better explanation if not.
