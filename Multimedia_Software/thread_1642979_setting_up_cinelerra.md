### Post by beccatoria on 2011-01-22
Hi there - hope this is still useful to you six weeks later.  ;)  

I have a similar issue - just the very bottom of the preferences with the "apply" "save" "cancel" buttons aren't visible.  

If, when you're finished changing your preferences, you simply hit "Enter", the dialogue box will close as if you saved everything.  

If you really need to use the keys, or access something below the level of the screen, I'm sure there's a more sensible way to do this, but the solution I've been using on the rare occasions it's an issue is that I have Compiz installed for extra desktop effects.  One of those is Super+a which displays all the windows, scaled, across the desktop.  I can then see what options are below the screen, which in turn means I know how many times to hit "tab" to cycle through buttons until I can hit "enter" to select said "below-the-screen" button.  

As I said, ridiculous workaround and I'm sure there's a more sensible one, but since it's the only program that gives me this issue, and I can just hit "Enter" when I'm done instead, I haven't bothered looking into a better solution.
