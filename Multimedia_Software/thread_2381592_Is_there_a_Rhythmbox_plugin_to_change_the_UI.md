"rhythmbox-plugin-alternative-toolbar">>>[https://github.com/fossfreedom/alternative-toolbar](https://github.com/fossfreedom/alternative-toolbar)
 The Github pitch for the plugin touts &#8220;enhanced play controls and interface for Rhythmbox&#8221;. That&#8217;s a rather modest description considering the improvements it makes. Modern, minimal, and music-focused is the alliterative menage à trios I think would sell the plugin  better.
[Found Here.]("apt://rhythmbox-plugin-alternative-toolbar") There is no need to add a PPA just install it in the repos.
Once the plugin is installed you will need to open Rhythmbox and enable it through the Tools > Plugins and enable the &#8216;Alternative Toolbar&#8216; option. You should be prompted to restart the app so that the changes can take effect (if you&#8217;re not just fully quit the app and reopen it).

You&#8217;ll find a small number of layout and features options in the View menu, including options to display the play queue, show/hide cover art, and toggle the sidebar.
