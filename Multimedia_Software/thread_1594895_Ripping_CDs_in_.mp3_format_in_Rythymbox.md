### Post by seventhsamurai on 2010-10-12
Rhythmbox cannot rip in mp3 format, it can only play mp3s. Ogg is the true open-source format and that's what it supports. But do not be disappointed.

You can use programs such as RipperX or RipOff to rip them to mp3, both of which are very good and easy to use. Make sure you install LAME (mp3 encoding library) from Synaptic Package Manager before installing either of these programs.
