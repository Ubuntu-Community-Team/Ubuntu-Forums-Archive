### Post by bakelitedoorbell on 2009-07-13
I assume you are running it using Wine?

Here's what you do.  Download and install this excellent freeware "glide to openGL" wrapper, which was created specifically for Diablo2:

[http://www.svenswrapper.de/english/index.html](http://www.svenswrapper.de/english/index.html)

In the glide-init.exe program, under "settings", check the box next to "desktop resolution".

In Wine Configuration, uncheck the box next to "emulate a virtual desktop"

Bob's your uncle.
