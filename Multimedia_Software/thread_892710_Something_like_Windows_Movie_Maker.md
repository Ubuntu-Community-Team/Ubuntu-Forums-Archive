### Post by eks on 2008-08-17
Yes, lot's of options:

Cinelerra
Kdenlive
Kino
Pitivi Video Editor
Avidemux
Open Movie Editor

You can search each of them on Synaptic. With [Cinelerra]("http://cinelerra.org/") you have to add a different repository. With Kdenlive, if you are using 64bits, you might [have some issues]("http://ubuntuforums.org/showthread.php?t=828093") (but if you are new to linux I would strongly recommend a 32 bit install in the beggining).

Either way, Kino is good for capturing and simple movie making features like the ones on windows. If you want more features you can try Kdenlive or Cinelerra.
