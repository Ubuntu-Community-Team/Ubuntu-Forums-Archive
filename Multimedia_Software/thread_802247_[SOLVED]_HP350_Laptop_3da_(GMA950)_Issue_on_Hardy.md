### Post by prshah on 2008-05-25
> **WunSick said:**

XORG.CONF
```

Section "Device"
	Identifier	"Intel Corporation Mobile 945GM/GMS/940GML Express Integrated Graphics Controller"
[color=red]	Driver		"i950"[/color]
	BusID	"PCI:0:2:0"

EndSection

```

Looks like an older intel driver. The new intel driver encompasses all chipsets; install it with ```
sudo apt-get install xserver-xorg-video-intel
```

then enable it by manually editing your /etc/X11/xorg.conf file, and change the above line (in red) to ```
	Driver		"intel"
```

Save it, exit, close all open programs, then restart your X server with Ctrl+Alt+BkSpace,
