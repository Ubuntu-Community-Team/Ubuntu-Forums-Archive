### Post by Pjotr123 on 2008-06-27
Apply these four steps:

1. disable visual effects:
[http://ubuntutip.googlepages.com/tipsandtweaks](http://ubuntutip.googlepages.com/tipsandtweaks)
(number 5 )

2. install full multimedia support:
[http://ubuntutip.googlepages.com/multimediaubuntustep1](http://ubuntutip.googlepages.com/multimediaubuntustep1)

3. make sure you're machine is running on the restricted driver (if there is one for your graphics card):
System - Administration - Hardware Drivers

4. adjust screen resolution (if applicable):
[http://ubuntutip.googlepages.com/bugsinubuntu](http://ubuntutip.googlepages.com/bugsinubuntu)
(number 2 )

That should do the job.  :-)
