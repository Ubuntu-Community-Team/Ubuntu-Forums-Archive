Hi,

I suppose it is NOT a bug.

Problem solved as follow;

Edit -> Preferences -> Window Management
Click on Reset Saved Window Positions to Default Values 

Restart Gimp and get 2 window mode (tools on left only)

Windows -> Single Window Mode (tools on both sides)

Now Single Window Mode starts.  Thanks
