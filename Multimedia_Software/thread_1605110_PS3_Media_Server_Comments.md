I'd never used it before today, but I threw it on my server and I rather like it.  Easy setup, just run a .jar file with java and it's got a nice little GUI and everything.  It ran just perfect out of the box too, all you need is mencoder and ffmpeg.  I'm playing an HD MKV file right now and I'm having no latency issues over wireless.  I just wanted to post this as more of a comment topic on the software because I think it's really cool that it works this well using Linux as a media server for my Playstation 3, :-)
