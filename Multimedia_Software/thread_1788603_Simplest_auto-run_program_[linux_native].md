### Post by BicyclerBoy on 2011-06-23
Do you mean you want to auto-run a Windows executable off a CD ?

Or do you mean make a native linux program auto-run off a CD ?

Some of the more nervous types don't like auto-running apps on any media because of the possible security risk.

The autorun process in Linux works the opposite to windows.
The Linux autorun is a User program setup on the PC by the user to allow execution of a certain application when a CD/DVD is mounted in the filesystem.
That application can be on the CD/DVD I believe.

man autorun
