### Post by sikander3786 on 2010-07-20
Hi.

For installing besides windows you need either to install through "wubi". That is a software present on every ubuntu cd that lets you install Ubuntu as a program inside windows. Just double click it and it will guide you till the end.
Or you need to dual boot your PC. For dual booting you need to select the install besides windows option at Ubuntu Installer Partitioning Stage. Here is the link.

[https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)

And for installing VLC just go to Ubuntu Software Centre in the Applications Menu and search for VLC and click Install. It is just a click away. For codecs, you need not to worry. Just play the movie in your favourite media player and Ubuntu will download the codecs automatically if they are not present in your system.

Ask if any further queries.

Regards.

Sikander.
