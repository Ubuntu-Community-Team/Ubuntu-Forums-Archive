It depends on the type of damage. If the HDD has failed sectors, no software will get the data back.
If you google for "ubuntu data recovery", there is a webpage just about that.  But know that if the file system is non-Linux native, then you should use tools from the other OS instead for better results.

And if something is important enough to keep, then it is important enough to have backups.  A backup is 3 separate copies, on 3 separate physical devices, with 1 at least 500km away.  Anything less isn't a backup.  Backups need to be validated periodically, preferably immediately and at least yearly after created.
