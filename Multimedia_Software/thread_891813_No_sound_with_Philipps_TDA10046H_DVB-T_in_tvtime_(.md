Normally I would be happy to use the card with kaffeine. The problem is that kaffeine just seems to pick up the DVB feature and the reception is very poor where I am. 
But when I use tvtime, it uses the standard setup (i.e. not freeview just terrestrial - and the picture is a lot better - but of course - there is no sound :)

I know mp3s and videos work fine with the installation - so the hardware must be ok - I've also tried to ensure that everything in the mixer is switched on and not muted - and to a good volume level. I've also checked the volume settings on the program and have tried to change the audio settings e.g. uk_i etc.

Still, no luck. If anyone else has had a similar problem (especially with the same type of card) could they kindly let us know.
