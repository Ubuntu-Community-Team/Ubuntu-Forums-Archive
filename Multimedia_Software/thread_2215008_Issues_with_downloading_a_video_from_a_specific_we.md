### Post by TheFu on 2014-04-04
Not all websites allow their videos to be downloaded. That is the way of the world. They make the URLs convoluted so much that a simple firefox plugin can't figure it out. In the old days, when a file was being streamed, a temporary file would be open and it was possible to grab it from /proc/$PID/fd.  **get_flash_videos** did it that way - try it.

If you really, really want it, then you can record your screen, assuming the PC is fast (results generally suck)
or
you can buy some external hardware to record video+audio from specific input types.  I have a Haupauge 1512 that does this. Sadly it has Windows-only software that is required, but it easily records 1080p and 5.1 audio from almost any source. If the source is high quality and the networking good, this can have amazing hidef results. It isn't cheap, however.
