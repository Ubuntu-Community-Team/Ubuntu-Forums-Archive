### Post by ron999 on 2010-06-09
Hi irish66
The Windows message was to tell you that you won't be able to change the region setting of your DVD-drive more than 4 times. After that it sticks.
 You then have to  return it to the manufacturer to be reset.

When you're using VLC media player with Linux it isn't too particular about the region setting of the DVD-drive.
So you might be able to play the wrong-region DVDs without having to change the region setting of the DVD-drive.

For Linux there's also a program called '**regionset**' which will tell you how many lives you've got left.
Maybe there's a similar Windows program to use with your laptop.
