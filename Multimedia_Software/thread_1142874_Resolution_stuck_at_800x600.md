### Post by HappyFeet on 2009-04-29
Try
```
sudo dpkg-reconfigure -phigh xserver-xorg
```
You can basically skip over the parts about keyboard and mouse. You need the Tab, Enter, and Arrow keys to navigate the setup windows.
