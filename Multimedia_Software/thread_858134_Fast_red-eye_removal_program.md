### Post by calraith on 2008-07-13
The only solution Google has revealed to me for removing red-eye natively in Linux is to use the Gimp.  There is a [script]("http://www.linuxjournal.com/article/6567") that will automate a big part of the process.  But as far as I can tell, there is no Linux native application capable of batch processing several photos.

However, I did just download and install a Windows program called [Red Eye Remover Pro]("http://red-eye-remover-pro.vicman-software.qarchive.org/") using Wine 1.1.1, which appears to be capable of batch processing.  Seems to work, but I have no pictures with red eye with which to test the software.  At least all the buttons work and it doesn't crash, and clamscan reveals no detectable malware, anyway.
