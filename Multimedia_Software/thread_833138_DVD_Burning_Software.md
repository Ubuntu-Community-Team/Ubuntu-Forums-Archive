### Post by Gina on 2008-06-18
The format is called DVD-Video.  It's a collection of files rather than one file per recording - not very elegant!

I use K3b in Gnome for all my CD/DVD burning (mostly iso images).  It's the only CD/DVD writing app I've found that will check the MD5Sum (easily) and verify the written data. I haven't tried it with DVD-Video recordings but I see no reason for any problem.  You can also make exact DVD to DVD copies, of course.
