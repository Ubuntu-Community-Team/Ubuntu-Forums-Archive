### Post by omniuni on 2008-12-17
Check out showFoto. It has some very nice features that I actually think Lightroom does not. For example, under Enhance => Restoration you'll find a quick way to reduce JPEG artifacts. Related is digiKam, which provides photo albums and advanced tools such as a light table to compare images.
