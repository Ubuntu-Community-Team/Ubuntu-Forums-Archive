### Post by starcraft.man on 2009-08-27
Best tag editor I know is EasyTag for GNOME, KDE also has kid3.

Both install from repos, same as name of program for package. I love easytag, supports CDDB lookup for albums, as well as simply editing the tag entries manually. For the more odd tags, use kid3, displays all tags not just ones expected.
