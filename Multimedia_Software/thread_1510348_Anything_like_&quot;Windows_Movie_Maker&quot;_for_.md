### Post by gkumar on 2010-06-15
If you're running Ubuntu 10.04 desktop, then the Pitivi Video Editor should already be installed, under Applications > Sound & Video > Pitivi. If it's not installed for whatever reason, it can be found in the Ubuntu Software Center. It's a simple video editor comparable to Windows Movie Maker.

Good luck!
