### Post by TheFu on 2017-01-02
I don't know anything about Macs, but VLC is **the** player for videos on Linux.  If the DVDs have DRM, you will need to load the DVD libraries that handle that in the extras. There are lots and lots of how-tos for it. Google easily will find that. During Ubuntu desktop installation, there is an offer to install "restricted extras" - if that was checked, it should be good to go.

[https://help.ubuntu.com/community/RestrictedFormats/PlayingDVDs](https://help.ubuntu.com/community/RestrictedFormats/PlayingDVDs)
[https://help.ubuntu.com/stable/ubuntu-help/video-dvd.html](https://help.ubuntu.com/stable/ubuntu-help/video-dvd.html)
[https://www.videolan.org/vlc/download-ubuntu.html](https://www.videolan.org/vlc/download-ubuntu.html)

More info than anyone should need on how to solve this.
