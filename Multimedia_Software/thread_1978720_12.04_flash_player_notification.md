### Post by SeijiSensei on 2012-05-12
What exactly are you trying to do?  Install Flash?  If so, first make sure the "partner" repository is enabled.  Open a terminal window and type "sudo nano /etc/apt/sources.list".  Find the line that begins with "deb" and has "partner" in it and delete the hash mark ("#") at the beginning of the line if it is there.  Save the file and exit with Ctrl-X.  Then run these commands:

```

sudo apt-get update
sudo apt-get install adobe-flashplugin

```

If that's not what you're trying to do, then please tell us what the button you're pressing is supposed to do.  It sounds like whatever script generated that dialog box expects you to be running plain Ubuntu rather than Kubuntu.

### Post by bbb13 on 2012-05-17
The problem was solved with:

```
sudo apt-get purge flashplugin-installer
```

and then install flash player through firefox extension [Flash Aid]("https://addons.mozilla.org/en-US/firefox/addon/flash-aid/")[U] , using wizard mode and selecting only "override GPU validation".
