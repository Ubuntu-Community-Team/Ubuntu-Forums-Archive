### Post by Mocto on 2009-12-24
Hello,

The plugin is available here : [http://bazaar.launchpad.net/~guillaume86/exaile/context-dev/files/head%3A/plugins/coverflow/]("http://bazaar.launchpad.net/%7Eguillaume86/exaile/context-dev/files/head%3A/plugins/coverflow/")
Just copy all this content to ~/.local/exaile/plugins/coverflow and the plugin should appear in your plugin list.

The plugin depends on [python-clutter]("apt://python-clutter") so you may have to install it, and I warn you that several people have dependencies issue preventing the plugin from working ([french forum]("http://forum.ubuntu-fr.org/viewtopic.php?id=340626&p=5")).

Good luck
