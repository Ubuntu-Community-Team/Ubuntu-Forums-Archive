### Post by Veector on 2009-08-05
I found a way to do sync ipod touch in ubuntu
Install Virtual Box(not the open source version<Virtual box OSE>)
Make a  windows xp virtul machine. Select the virtual machine settings and go to the usb tab enable usb controller, enable usb 2.0 (ehci) controller, add ipod touch from the buttons on the fight and edit the ipod touch filter, erase everithing but the vender id and the name. Install itunes in the virtual machine and the ipod touch will apera like it would on a normal windows machine.

This is the only way that i know of that will let you sinc your ipod touch iwhile using ubuntu. I personally dont mind because i use virtual machines all the time and im used to it.
