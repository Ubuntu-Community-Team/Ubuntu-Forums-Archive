### Post by SeijiSensei on 2017-11-14
You can create animated GIFs in the GIMP.  I do it all the time.  (I have no idea what file format the result produced in that video uses.)  You build up the image by layers.  To replicate his example, you'd create a layer with the "Pan Americana" text and use it as the background.  Then you'd create one or more layers that contain the other components of the logo.  With "transparency" enabled, you would end up with a result similar to his that you can export as a GIF from the GIMP.  

I use this technique to create animated avatars from anime programming  In my case I don't have to create the layers myself.  I just extract the ones I need, adjust the timings accordingly, and resave as a GIF like 

[img]http://www.takinganimeseriously.com/images/avatars/chiko-in-chains-bordered.gif[/img]

If you save that image to a file, and open it with GIMP, you can use the Ctrl-L command to reveal its eight layers.

There are other examples at [http://www.takinganimeseriously.com/images/avatars/](http://www.takinganimeseriously.com/images/avatars/) if you're curious.

### Post by coldraven on 2017-11-17
> animated images are frigging annoying
A charity site that I maintain wanted a "donate" button. I used ImageMagick to make it do the following. For ten seconds the button's text is "Donate" but then for two seconds the text changes to "Please", this cycle then repeats indefinitely.  I figured that it would not annoy but just be animated enough to catch the eye and hopefully make a donation. So maybe what you want is a subtle transformation in your logo that is just enough to lodge your site's name in the mind of the viewer.
