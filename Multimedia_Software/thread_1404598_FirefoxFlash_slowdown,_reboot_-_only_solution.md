### Post by lovinglinux on 2010-02-11
I had a similar problem when using a P4 single core processor, but it has been solved since I upgraded to a Core 2 Duo. Flash still uses too much CPU, but it is actually playable now.

I would advice to clean up the CPU fan. This helped a lot when I was experiencing the same problems and prevented flash from going nuts after a while playing the video. You could also try the Flash Optimization tips from [Firefox optimization and troubleshooting thread](http://ubuntuforums.org/showthread.php?t=1193567).
