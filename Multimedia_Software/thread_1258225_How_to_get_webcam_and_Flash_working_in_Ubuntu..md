### Post by hoppipolla on 2009-09-04
Dear GOD! Here I am sitting here with both working. Hehe :)

I'm sure a lot of you know how to do this but what the hell I'll bung it up here anyway!

Flash first - Fire up synaptic and search for "flash". Find the currently installed packages (if flash IS installed these should be **swfdec-mozilla** and **libswfdec** I think) and then... remove them!

And then install the "**flashplugin-installer**" package.

Now, for webcam, just use [www.ebuddy.com](www.ebuddy.com). Cheating I know, but hey it works.

Enjoy!

Hoppi :)


EDIT -- Hmm, even in ebuddy I had some problems with webcam... fingers crossed between that and the sometimes functional cam features on Kopete/aMSN, you should be able to get a webcam session going anyway! If anyone else has thoughts and ideas please post them! :)
