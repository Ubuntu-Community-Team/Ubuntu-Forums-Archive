### Post by Superkuh on 2011-06-28
QMMP is a small, resource light, actively developed audio player that is a familiar winamp clone (you can even use winamp skins). It'd be great for a netbook (or a desktop).

[http://qmmp.ylsoftware.com/index_en.php](http://qmmp.ylsoftware.com/index_en.php)
[http://code.google.com/p/qmmp/](http://code.google.com/p/qmmp/)

```
$ sudo apt-get install qmmp
```
