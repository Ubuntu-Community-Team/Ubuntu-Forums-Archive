### Post by xc3RnbFO8P on 2009-12-15
First it is important to choose picture with good colors,
like this picture, only thing that is yellow on this picture is the flower.
1) Use tool called: Select by color tool (shift- O)
2) Mode: Add to the current selection
3) Feather edges 
4) Threshold (higher number if the flower is the only thing that is yellow)
5) When you have finish select the color Select> Invert
6) Color> Desaturate
7) Image> Merge Visible Layer

Save As> Select File Type (By Extension)

If you got similar colors in the picture use **Free Select Tool**
