### Post by jamesrl on 2008-08-27
If you set your volume within exaile to something below ~75% (the actual system wide volume is irrelevant) the issue goes away.  It also goes away if you turn off the equalizer within exaile.  (Run Exaile with 'exaile --no-equalizer').
