### Post by lovinglinux on 2010-02-26
You can also remove any conflicting plugins and install the appropriate version of flash with an [automated installer created by carlee](http://ubuntuforums.org/showthread.php?t=1414595).

If that doesn't work, then close Firefox, go to your Firefox profile folder (~/.mozilla/firefox/<profile name>)and delete the file **pluginreg.dat** then start Firefox.
