### Post by Matt L on 2010-08-15
I don't have three monitors, but was successful in getting two to work on my GTS250.  This is Fedora 13.  Hopefully there is some similarity.

With the latest X server on Fedora, there is no longer a file /etc/X11/xorg.conf.  The server figures out most of the settings by itself.

The NVIDIA installer created a complete configuration file, which worked great for one monitor.  I attempted to follow the instructions on their site to enable Twinview, with quite limited success.  Then I found this in their forums:

[http://www.nvnews.net/vbulletin/showthread.php?t=72490](http://www.nvnews.net/vbulletin/showthread.php?t=72490)

> I recommended that you delete everything but the [FONT=Courier New]"Device"[/FONT] section and leave the rest up to the X server's autoconfiguration:Here is my complete resulting xorg.conf file:
```

Section "Device"
    Option     "TwinView" "True"              # [<bool>]
    Option     "TwinViewOrientation" "RightOf"     # <str>
    BusID          "PCI:1:0:0"

    Identifier     "Device0"
    Driver         "nvidia"
    VendorName     "NVIDIA Corporation"
EndSection

```It worked fine on one monitor before adding the TwinView options.  With this configuration, it works great on two monitors.  Maximizing a window maximizes it to a single monitor, and you can drag windows between monitors.  Desktop effects work great.  The best of all worlds.
