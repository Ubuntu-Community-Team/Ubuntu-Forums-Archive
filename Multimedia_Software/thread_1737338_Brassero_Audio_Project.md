### Post by varunendra on 2011-04-24
Seems like folder permission or disk space problem.
May we have screenshots please?

You may wish to check/confirm available disk space using 'Disk Usage Analyzer' in Application > Accessories.

And if you doubt a bug in brasero (it does do crazy things sometimes), you may wish to try K3B. It is the best free CD/DVD burner for linux as far as I know.

If you wish to avoid installing K3B (due to its huge size and unnecessary KDE modules associated with it), you may download [Slax]("http://www.slax.org/get_slax.php"), install it on a USB flash (installs like a breeze - just copy-paste two folders and run a script - in linux or windows) and run it from there (or choose from menu to load it entirely into RAM and run it from there - super fast speed!). Download size is 200MB approx.


PS: Slax is a complete (portable) desktop OS, K3B is just one of its native apps.
