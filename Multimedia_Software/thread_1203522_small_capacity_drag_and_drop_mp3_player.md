### Post by jasonsjunk on 2009-07-03
I have had good luck with the Sansa players.  I personally use the E280 which gives me 8GB of memory and they are only like $40 these days.  For wav support you would need to install rockbox on the Sansa, not many small cheap players support wav playback and with only 512mb like you wanted you wont be able to fit very much on there in wav format.
