### Post by shantiq on 2010-12-10
in case you have been experiencing problems using[ youtube-dl]("http://freshmeat.net/projects/youtube-dl") in recent times

try updating it this way maybe


```
sudo youtube-dl -U
```


and run

```
youtube-dl -v
``` to see the changes



and then try again

============================================================


it is a useful tool to watch a document which has been broken in say 10 parts
once you have them all you can create a m3u playlist and watch without having to go to your computer every 10 minutes 


one needs to copy all the urls to a .txt or.m3u   possibly any word document

then

```
youtube-dl -t -a (name of playlist)

```

  it also shields you from connection vagaries
