### Post by richlion2 on 2015-12-29
Hi, 

I just wanted to share my bad experience with VLC. I am on the most recent Kubuntu 15.10 version. I think VLC is currently a very unstable product and I suggest using SMPlayer. Here are my thoughts in VLC:
- you open a video file (any type, .iso, mpg, mp4, wmv, etc) and video plays. Open the same file again it gets stuck. Video stops, sometimes there is no sound. It can also block other video players from playing any content.
- I browse through my surveilance camera .mod files, so I need to use fast forward and edit. It's unstable, jittery. Any suggestions to untick some of the options are not helping. 
- Sometimes a reboot of the system helps VLC play again.
- the Record button saves the file (I need it to cut my videos), but no sound is recorded.

I completely switched to SMPlayer, it's got a different interface, but is a very stable product. The interface can be customized, so adding FF and Rewind buttons work, although they are not as simplistic in terms of using as compared to VLC. I use Avidemux for editing videos, both are so far doing a fantastic job.
