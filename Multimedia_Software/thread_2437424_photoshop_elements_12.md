### Post by rbmorse on 2020-02-24
There are several alternatives that run on Linux, however.  My personal fave is called darktable.  It's in repository and can be installed by opening the terminal and running the command:

```
sudo apt install darktable
```

If you try it and don't like it, you can remove darktable simply by running the same command, replacing the word install with the word remove.
