### Post by red13 on 2009-03-15
try installing pulseaudio device chooser from synaptic then goto applicatio> sound & video> it will be in there, start it up it will load up a icon in the task bar, open the program u want to use then on the icon left click and select volume control you will see a arrow next to a padlock under the playback tab click it and select source.
