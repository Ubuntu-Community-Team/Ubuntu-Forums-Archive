### Post by leonardo_neo on 2009-04-14
> **volter said:**
> I don't know how to turn on my microphone & when I calling to some one the call just turning off.How can I fix that?


Open skype. 

Press skype icon at bottom left. Select 'options'. Go to sound and devices.

For "sound out" and 'Ringing', select 'pulse'.

Try to make a test sound. If sound is OK, then proceed for the microphone issue.

For "sound in" is a bit tricky. Try to select Intel ICH6 (hw:ICH,0). 

This should work for most of the people. After selecting this try to make a test call. If test call is fine, then you are done. If test call doesn't work, then you should try all the options from "sound in" one by one and test that with make a test call. One of them should work with you.

Hope this helps. :)
