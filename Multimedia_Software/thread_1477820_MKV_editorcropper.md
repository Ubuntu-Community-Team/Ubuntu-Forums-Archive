### Post by ron999 on 2010-05-09
Hi
In the Ubuntu repository are packages 'mkvtoolnix' and 'mkvtoolnix-gui'.
The gui is a program called 'MKV files creator' aka 'mkvmerge GUI'.
You would use this to create a new mkv file from the one that you have already.
The GUI has options for 'Enable splitting ... after timecodes'.
So to remove part of your file you would make a new file that is split into 3 parts.
Then you would create one new file from just the 2 parts that you need using the 'add' and 'append' functions.
There is no loss of quality.
