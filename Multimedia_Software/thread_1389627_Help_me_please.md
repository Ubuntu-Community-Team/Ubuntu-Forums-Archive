### Post by hemimaniac on 2010-01-24
If there is enough room in the ubuntu partition you could create a folder there and move them to the ubuntu partition, format the other one then move the files there. Otherwise you could try to shrink the xp partition enough to make room for the files you'd like to keep, back them up to there, then once the xp is formatted out, expand the partition to re-take the xp space.
