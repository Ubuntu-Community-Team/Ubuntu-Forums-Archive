Hello,

While trying to get Ubuntu to "DO NOTHING" when I insert a CD, I filed a bug report: [https://bugs.launchpad.net/bugs/305964](https://bugs.launchpad.net/bugs/305964)

To recap the bug report:
Given the usability improvements in Ubuntu, I assumed the CD launching preferences would be there. Thus, I spent a few minutes looking for such preference there.

Thanks to Chris, I now know that this is configured via Nautilus.

Which leads to the following usability issue:
Should this setting be moved / copied to somewhere in System -> Preferences?

My reasons for thinking YES:

1. Inserting Media is a system wide event. It doesn't really have anything to do with Gnome / Nautilus. (I do understand Gnome runs the show, but still...)

2. Most System wide settings are in System -> Preferences. I think it would be a service to the end user if more settings were placed in a central place.

3. My first instinct told me the setting would be in System -> Preferences.

What do you think?
