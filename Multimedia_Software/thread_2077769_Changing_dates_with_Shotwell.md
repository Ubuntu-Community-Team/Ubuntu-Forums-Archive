### Post by eric-yorba on 2012-10-29
Shotwell lets you edit the photo timestamps in bulk.  Additionally, if you enable metadata writing in the prefs, it will update the metadata in the photo file itself.  (Note that Shotwell cannot write to RAW files or videos.)

The help page provides more info:
[http://yorba.org/shotwell/help/edit-time-date.html](http://yorba.org/shotwell/help/edit-time-date.html)
