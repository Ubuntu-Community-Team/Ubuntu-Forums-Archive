### Post by Yellow Pasque on 2010-03-01
Apparently, disabling/enabling the onboard modem (if you have one) sometimes works. [http://www.thinkwiki.org/wiki/AD1981HD](http://www.thinkwiki.org/wiki/AD1981HD)

Also, you can try specifying the model keyword:
```
gksu gedit /etc/modprobe.d/alsa-base.conf
```
Add the following line to the end of the file, then save and restart:
```
options snd-hda-intel model=thinkpad
```
(That suggestion is from the ALSA manual):
```
	AD1981
	======
	  basic		3-jack (default)
	  hp		HP nx6320
	  thinkpad	Lenovo Thinkpad T60/X60/Z60
	  toshiba	Toshiba U205
```
