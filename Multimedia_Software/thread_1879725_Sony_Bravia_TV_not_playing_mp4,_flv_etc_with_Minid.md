### Post by jon890 on 2011-11-12
Was using Minidlna with Sony Bravia KDL-32EX503 TV, it was Minidlna restricting the file types such as mp4, FLAC, flv etc, not Sony. Changed dlna server to Serviio problem solved, all file types now play including Freeview HD recordings transcoded to mp4 using Mythtranscode and Vlc. Serviio now recommended by Sony.:)
