The flash version installed by default by [Flash-Aid]("https://addons.mozilla.org/en-US/firefox/addon/161939/") can be detected by Chromium too and doesn't use the package manager. 

However, you need to fix the problem with the package from the repositories, because it probably will affect the installation of other applications.

Please provide the error output you get from the terminal when trying to remove flash. It usually suggest a command to fix the problem.
