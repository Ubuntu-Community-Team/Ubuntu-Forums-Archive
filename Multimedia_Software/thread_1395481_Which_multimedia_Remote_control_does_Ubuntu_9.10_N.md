In theory almost any remote and IR receiver will work if you got the time to set it up right. Lirc was designed originally for homebrew IR receivers and using pretty much any remote after all. It has grown quite a bit since then.

What I would suggest is any half decent Windows Media Center / Vista / 7 remote. Not some cheap one you get off ebay or one of the companies selling junk out of China either. Go to New Egg, Tiger Drect, Amazon, any reputable outlet like that and spend 30 bucks US on a plug and play Windows Media Center / Vista / 7 remote.
