### Post by Linuxdork on 2009-01-26
You want to find something like this: [http://www.slax.org/modules.php?action=detail&id=1039](http://www.slax.org/modules.php?action=detail&id=1039)  Backtrack is built on SLAX and should work in a similar way.  If you read the documentation for modules in SLAX it will tell you that you will pretty much need to rebuild the ISO after you add the module to it.  It's not to difficult but does require some studying.  Hope this helps.

### Post by Linuxdork on 2009-01-26
Yeah, if I remember correctly you have to copy the contents of the disk to a 'writable' (I.E. your hard drive) location and then you copy the module into the modules directory.  After that you run the makeiso or something like that command that is in the root of the 'image' and the will make the ISO for you.  Then just burn that.
