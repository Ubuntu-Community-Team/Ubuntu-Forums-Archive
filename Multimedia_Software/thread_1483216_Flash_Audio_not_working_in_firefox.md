### Post by Guitar John on 2010-05-18
System > Administraion > Synaptic Package Manager

Type "Flash" into the Search field

You will see "flashplugin-installer" is already installed.

Click on the box and select "Mark for Reinstallation."

Click "Apply."

When done, restart Firefox.  

It worked for me.
