### Post by mc4man on 2011-10-31
gnome3 is not longer using that, it's being replaced by an audio/video preview/player

To see - 
```
sudo apt-get install gnome-sushi
```

To use just highlight a file, audio or video, then press spacebar
To get playback the codec must have support in gstreamer already installed, otherwise sushi will just do nothing
