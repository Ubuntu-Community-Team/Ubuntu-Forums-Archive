### Post by SuperSonic4 on 2008-12-10
It shouldn't be necessary but there is the regionset package available@
```
sudo apt-get install regionset
```

to activate type in a terminal: ```
sudo regionset
```

and press the number key for whatever region you want (it does tell you). For PAL DVDs this will almost certainly be region 2 (Europe)
