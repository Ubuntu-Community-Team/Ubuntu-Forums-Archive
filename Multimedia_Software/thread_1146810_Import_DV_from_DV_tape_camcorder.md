### Post by bumanie on 2009-05-03
Kino does a good job, only problem is that it only supports firewire. May be you could buy a firewire cable, the camera most likely has a firewire port. I have a sony dv cam and kino works great with firewire, but unfortunately the cable has to purchased separately. A program called Cinelerra may be able to use a usb connection. Look[ here]("http://cinelerra.org/getting_cinelerra.php#packages") for installation instructions on ubuntu. Cinelerra apparently has a big learning curve when first used, but documentation states it is a professional quality program. It's worth a look if you don't want to buy a firewire cable.
