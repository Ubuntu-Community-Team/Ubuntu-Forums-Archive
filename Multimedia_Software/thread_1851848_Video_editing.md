I am only an amateur in video, and so far I'm happy using *OpenShot* for editing and conversion. I think it is fairly easy to use and it can do the things I want. If you need something more advanced, there are several other programs, that you can find browsing this forum or the internet. And let us wait for more replies to your question.

OpenShot is in the repositories, so install it via Synaptics or the terminal command
```
sudo apt-get install openshot
```
