### Post by ron999 on 2009-08-30
Yes you can.
Load the folder of 20 files into EasyTag.
Highlight the files that are to be modified - all of them.
Then type the artist name into the 'Artist' field.
Then right-click somewhere in that box and select 'Tag selected files with this field'.
Then save the changes by clicking on the floppy disk icon.

Likewise for 'Album', 'Genre' etc.
More information is here:- [http://easytag.sourceforge.net/]("http://easytag.sourceforge.net/")
