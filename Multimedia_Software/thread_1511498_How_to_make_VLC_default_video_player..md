### Post by bhishan on 2010-06-16
How do I make VLC media player the default video player, so that if I double click a video file VLC opens it?

---

### Post by WinterRain on 2010-06-17
Right click a media file and go to properties>open with> vlc. You will have to do this with every type of file.
