HandBrake will let you specify start and end times based on chapter, seconds, or frames.  So you can cut off the beginning or end of the video, but can not edit out portions in the middle with HandBrake. In the gui, these options are all where the chapter controls are.  There is a combobox there that is set to "Chapters" by default.  It can be changed to seconds and frames.
