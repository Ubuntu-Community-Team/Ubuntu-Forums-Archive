### Post by Whovian on 2012-07-12
Link at github to help you install (or what seemed described) feature.
[link]("http://mendhak.github.com/rhythmbox-tray-icon/")

```
wget https://github.com/mendhak/rhythmbox-tray-icon/raw/master/rhythmbox-tray-icon.zip unzip -u rhythmbox-tray-icon.zip -d ~/.local/share/rhythmbox/plugins
```Than just go into the plugins in rhythmbox and add the tray icon.
