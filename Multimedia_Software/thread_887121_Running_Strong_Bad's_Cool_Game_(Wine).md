### Post by tremby on 2008-08-23
yup! see my howto at [http://www.telltalegames.com/forums/showthread.php?t=6051](http://www.telltalegames.com/forums/showthread.php?t=6051)

a chap on there, demonfoo, pretty much figured out how to get it working, and i broke it down into easy steps and wrote it up into a howto. the crux is that the current release of Wine doesn't have a particular bug fixed and so a source patch needs to be applied before Wine is compiled.

demonfoo was kind enough to package Wine with the patch already applied just for this purpose, though (Hardy i386 and x86_64), so you can skip a great chunk of the howto. details are all there.

he and i have played the whole game through in Wine and it works flawlessly (less the disappearing eyes bug which is also affecting Windows users).

have fun
