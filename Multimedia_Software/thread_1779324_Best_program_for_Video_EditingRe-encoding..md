### Post by gordintoronto on 2011-06-10
I think Pitivi is installed as part of 11.04, so it would be easy to have a look at it. Make sure you install the "restricted extras" before doing anything else.

I like to add titles, do transitions, the "Ken Burns" effect from images, etc, so I find Cinelerra suits my needs. It has a larger learning curve to match the greater power.

No matter what program you try, you should be able to find text and video tutorials with a little Googling. It's well worthwhile to spend some time on tutorials instead of just jumping in to a program.
