### Post by FakeOutdoorsman on 2008-12-08
I personally use abcde.  It's command-line, but it's easy to use and does the job.  [andrew.46](http://ubuntuforums.org/member.php?find=lastposter&t=558538) has an excellent guide with sample configurations for FLAC, OGG, and MP3.  If you want lossless, then FLAC is the format to use:
[abcde: Command Line Music CD Ripping for Linux](http://www.andrews-corner.org/abcde.html)

Another option is to just make a copy of the cd.  This will create an image file that can be later used to burn a cd.  andrew.46 has written another excellent guide on how to do this:
[CD and DVD Writing from the Linux Command Line](http://www.andrews-corner.org/burning.html)
