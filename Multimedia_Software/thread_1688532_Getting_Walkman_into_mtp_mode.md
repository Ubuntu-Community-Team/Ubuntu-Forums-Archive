### Post by tommcd on 2011-02-15
MTP = Microsoft transfer protocol.
Does your player have an option to set it to MSC mode?
My Sandisk Sansa Fuze mp3 player works perfectly in linux, but you have to set it to MSC mode to get it to work in linux. This is stated on the Sandisk website.
Hope this helps.
