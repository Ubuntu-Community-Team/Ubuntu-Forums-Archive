MPlayer has this ability which can be utilised with a key combination from the commandline interface:

```

[ and ]
Decrease/increase current playback speed by 10%.

{ and }
Halve/double current playback speed.

backspace
Reset playback speed to normal.

```

or with these keys or from the menu in the SMPlayer gui. I attach a screenshot of the gui at work + menu open.
