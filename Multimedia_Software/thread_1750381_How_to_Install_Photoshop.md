### Post by ajgreeny on 2011-05-05
You could also persevere a bit more with gimp.  It looks a bit different from Photoshop, with separate windows for tools and layers, but in reality it still does just about everything that P'shop does, but for free.

In version 2.7 available from a ppa, it is possible to have a single window setup in gimp, but I found that it was hardly worth the change to a beta version where you had to go to the single window setup at each startup, and the arrangement within the window was never correct, with the tools taking much far too much space, so I have gone back to the stable version from the webupd8 ppa for 10.04.

In future it will probably be a lot better than now, but I have grown used to the current separate windows and quite like it.
