### Post by Atermoon on 2010-03-13
Here's what I did to organize my ~90 GB collection.

- Move all my music to "Music 2"
- Open Banshee
- Go to "Preferences", enable "Copy files to media folders when importing"
- Switch to the "Source specific" tab, change folder hierarchy to "Album artist/Album" and file name to "Number. Track artist - title" (Of course you can use whatever option you prefer)
- Import everything from "Music 2"
- Make sure all my files have been copied to "Music"
- Delete "Music 2"

Everything ended perfectly organized, but my absolutely perfect ID3 tags might have helped :P

There might be better solutions but this one did the job for me so I decided to share it ;)
