### Post by MonkeeSage on 2008-12-02
> **davidshere said:**
> My sound juicer now rips to files with an OGA extension.  Is this a new format, and if it is, what do I need to tell my Windows users with whom I share these files?  With OGGs, I just said, "Use the latest version of Winamp and you'll be fine."  Is it the same with OGA?

Usually the OGA extension is just an ".ogg" file (as supported by Xiph), with only an audio stream. I.e.,  it amounts to an ogg container with an audio transport stream encoded in the Vorbis format. Some people have taken to using the OGA extension to distinguish between "pure" ogg format, with vorbis audio, and OGM (a non-standard, unsupported extension to the ogg format which allows for multiplexing arbitrary audio and video streams into the ogg container).

In other words, it is an ogg container with a vorbis encoded audio stream--all supported ogg demuxers with vorbis decoders should parse and play the file just fine (e.g., winamp).
