### Post by Bufeu on 2012-07-23
You can run Sony Vegas 9 in Wine, works for me without any problem.
Install corefonts, fontfix, tahoma, msxml6, gdiplus, directmusic,vcrun2005, dotnet2, quicktime76, and if you want to use Windows Media Player codecs, wmp10. Then run the setup.exe.

EDIT: Found this on YouTube: [http://www.youtube.com/watch?v=-ppcceV2ypo](http://www.youtube.com/watch?v=-ppcceV2ypo).
