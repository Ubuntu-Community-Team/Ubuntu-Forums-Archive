### Post by cstudent on 2010-06-17
I've told Lucid to use Banshee as the default program to open when I plug in my 5th gen iPod. How do I reset it back to asking me with a dialog box what program to use when I plug it in? I can't find it anywhere.

Thanks.


FOUND ANSWER: Opened Nautilus under Edit>Preferences and Media tab. Changed Media Player to "Ask what to do".
