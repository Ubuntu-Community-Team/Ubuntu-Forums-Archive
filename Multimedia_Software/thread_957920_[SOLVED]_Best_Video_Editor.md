No, there isn't really any good Linux video editors. This best in my opinion is Cinelerra, but that still, in no way can match the adobe products..

Its not in the repositories that come in default so you will have to add it.....
```
wget -c http://akirad.cinelerra.org/pool/addakirad.deb && sudo dpkg -i addakirad.deb
```

Then open up scynaptic and search video editor, or cinelerra. And it will be in there.
