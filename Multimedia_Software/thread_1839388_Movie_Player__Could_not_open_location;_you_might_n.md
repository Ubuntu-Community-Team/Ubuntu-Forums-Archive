Hello all, not sure if this helps but I had the same problem and all the necessary files were installed.  The DVD was opening up into places instead of opening automatically into vlc.  However, I found that if you go to Places click on the DVD drive there is an option button on a tool bar labelled "Open vlc media player".  Once I clicked on this the DVD opened and the problem went away.:D
