### Post by mc4man on 2014-07-05
Then just go ahead & remove audacity & libid3tag0 packages
```
sudo apt-get purge audacity  libid3tag0
```
Then clean
```
sudo apt-get clean
```
check that there is no libid3tag.so.0 file anymore with the find command (if still there then remove manually
Then re-install
```
sudo apt-get install audacity
```
