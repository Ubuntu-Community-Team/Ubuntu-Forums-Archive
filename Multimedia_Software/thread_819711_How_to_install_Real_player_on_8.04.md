### Post by gandaran on 2008-06-05
get real player here,[http://www.real.com/linux](http://www.real.com/linux)

drag the file to your home folder and open a terminal 
first, make the file executable with the following command:

chmod a+x RealPlayer11GOLD.bin

second, run the executable as root and enter your password when prompted:

sudo ./RealPlayer11GOLD.bin
