---

### Post by benerivo on 2010-01-01
The 'hue' is wrong. Strangely, another thread with the same problem was immediately below this one (see [here]("http://ubuntuforums.org/showthread.php?t=1369334")). The solution was...```
Movie Player (Totem): Edit --> Preferences --> Display --> Color Balance
```
