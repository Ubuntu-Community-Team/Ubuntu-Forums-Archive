### Post by SeijiSensei on 2015-08-03
No, you don't need to compile ffmpeg.  You just need to add Doug McMahon's PPA:
```

sudo add-apt-repository ppa:mc3man/trusty-media
sudo apt-get update
sudo apt-get install ffmpeg

```

Some suggestions on rotating with ffmpeg here: [http://stackoverflow.com/questions/3937387/rotating-videos-with-ffmpeg](http://stackoverflow.com/questions/3937387/rotating-videos-with-ffmpeg)
