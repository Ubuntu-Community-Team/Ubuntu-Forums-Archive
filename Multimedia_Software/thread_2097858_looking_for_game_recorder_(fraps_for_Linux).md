### Post by myromance123 on 2013-01-17
Here's a simple list of recording applications I've tried:
1. (gtk)RecordMyDesktop
2. Kazam
3. GLC
4. Gamecaster

From my experience, there is nothing verging onto the simplicity that is Fraps. The best solution I've had is sadly:

# Using RecordMyDesktop (just for video)
# Using outRec (just for game sound)
# Using Audacity (just for my voice over, if necessary)
# Game must be in Windowed mode.

I've spent countless hours with all the applications up above, and that is the solution I've come to.

Kazam has the best POTENTIAL, but there is a serious problem rendering it unusable at the moment. It will leak into memory like crazy after several minutes of recording, and simply crash leaving you with your game stuck (this is tried and true in both Ubuntu 12.10 and 12.04 by myself).

GLC is unstable at best. Some games it will work great with! (mostly opensource games like the Cube engine games). Other's it won't even recognize (Torchlight, Rochard).

Gamecaster looks promising, but practically it's not done even a second of recording for me. I've tried it multiple times, but you may have better luck with it.

If you ever find a solution, please by all means, let me know!
