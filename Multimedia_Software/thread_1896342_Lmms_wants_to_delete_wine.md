### Post by N00b-un-2 on 2011-12-16
Linux Multi-Media Studio relies on Wine 1.2.  So if you want to install LMMS you have to uninstall Wine 1.3 which positively will cause Windows application breakage.  If you have a copy of Fruity Loops, I would strongly recommend you stick with that because it is well supported on Wine for one, and Fruity Loops is FAR more usable than LMMS.  LMMS is essentially a poorly written hack of Fruity Loops to begin with.  Although I am a supporter of open source software, there is nothing I hate worse than when a Linux application is written to use Wine, because its not truly a Linux application if it relies on the Windows API.  LMMS and Google Picasa are two of my biggest pet peeves on Linux actually.
In my opinion, there is no real Linux alternative to Fruity Loops, there's just a shoddy unofficial port which is hardly a worthy substitute for the real thing.
