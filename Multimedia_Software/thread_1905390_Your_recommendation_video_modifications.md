### Post by ianp5a on 2012-01-08
So far the best way I have found to join video files is with AviDemux 
It works like this:
"Open" a the first video file
"Append" then next.
"Append" the next and so on.
con) Each append video needs to be selected individually.
pro) Selection dialog is quick as it remembers the last folder. I joined 15 files in 30 seconds manual selection
pro) It does not ask about codecs it keeps the current settings
con) You cant see the names of files you have selected before you start the join process. So double selection or order mistakes are easily missed.
