### Post by Tom_ZeCat on 2008-08-11
This works in Windows and probably in Linux too.  Rename the m4a's extension to mp4 and it should play.  If not, there are conversion programs that will turn it into an MP3 (or into an ogg).  The one I use in Windows is named Switch (it's part of a set of programs named Super).  I haven't hunted for a Linux equivalent yet, but I'd bet there is one.
