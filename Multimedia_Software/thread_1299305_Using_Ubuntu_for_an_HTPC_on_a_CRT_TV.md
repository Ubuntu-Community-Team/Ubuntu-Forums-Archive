### Post by jbrown96 on 2009-10-23
XBMC is loosely based on Ubuntu, and it is really great for HTPC, especially if you have a Nvidia graphics card. You could always try the netbook remix if the resolution is really tiny. 
Boxee (which is based on XBMC) and XBMC are really good applications. They make versions for ubuntu, which would work well. XBMC also releases a liveCD, which is really great because the entire system will be designed for XBMC, and it won't be an added application to regular ubuntu.
