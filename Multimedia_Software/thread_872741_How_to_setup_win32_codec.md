### Post by warp99 on 2008-07-28
> **wejick said:**
> I have avifile-win32-plugin installed so what should i do next to get win32 file play in my totem and other player?

Install the win32codecs from medibuntu.org. Information is located in the Ubuntu help guide:

[https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)

These codecs work with mplayer. If you want the totem plugins then you need to add the restricted plugins. The Ubuntu help guide also has this information:

[https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)
