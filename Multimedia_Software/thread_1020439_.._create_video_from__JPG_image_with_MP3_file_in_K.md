### Post by richardjennings on 2008-12-29
Hi, I read this post yesterday whilst trying to work out something similar. In my case I wanted to make a video with just one picture all the way through. I ended up using ffmpeg.

here is an example:

```
ffmpeg -loop_input -t 1710 -i picture.jpg -i audio.mp3 -y -f flv -r 1 video.flv
```

where 
-t 1710 is the time in seconds
picture.jpg is the image
audio.mp3 is the mp3
and video.flv is the outputted video file

note picture.jpg and audio.mp3 should be in the directory you issue the command from, eg, your home folder as default.
