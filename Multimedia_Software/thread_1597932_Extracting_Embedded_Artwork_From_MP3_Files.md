I'd like to save the artwork from the file to my hard drive (as "cover.jpg" or whatever), so that programs that don't read embedded artwork can still show cover art.

### Post by ufleisch on 2010-10-16
> I'm using Kid3 for general tagging, and it supports embedded artwork, but doesn't allow me to save artwork already embedded in the file.
Kid3 can save embedded cover art. Just select the *Picture* frame and then click *Edit* (with Kid3 1.5 you can double-click on the picture), then click *Export*.
