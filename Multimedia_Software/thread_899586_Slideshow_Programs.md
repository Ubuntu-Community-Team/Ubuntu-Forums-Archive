### Post by Irihapeti on 2008-09-01
Open Synaptic Package Manager and search for "slideshow".

A number of programs are listed.

I haven't tried any of them, so I have no idea how good they are. But you may find something that does what you want.

I think it's possible to make a sequence of still images in Open Movie Editor, but it requires a bit of fiddling around. I think someone wrote a script to do this, but I'm very hazy about the details. I know it can be done in Kdenlive, but I had problems with images and sound going out of sync, so I don't use it now.
