Are there any Linux systems that incorporate ability to record from direct sound ? With  windows xp using Audacity you can record what is playing on the computer directly.

### Post by kostkon on 2014-02-03
If you are using Ubuntu, try the following:

Install the Pulseaudio Volume Control (pavucontrol) utility using the software centre. Open Audacity and start recording, then open pavucontrol, click on the recording tab and you'll see Audacity listed there. Set the device for Audacity to the *Monitor* of your soundcard.
