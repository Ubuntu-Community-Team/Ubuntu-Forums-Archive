### Post by Kiemc on 2012-04-23
Well, I see that no one has given an answer to this question, although you put it on different sites.

Unfortunately I myself can't give a solution to the problem, but I can tell you that I face the similar problem. I've tried several media players, and this problem everywhere occurs. I think the problem may be with the codecs or such (I don't know, I'm not a professional), and hopefully it will be fixed in the 12.04 version Lubuntu.

However, I can tell you something with which you can make a workaround. In Gnome MPlayer, you can set video picture settings pretty easily. (In VLC it's not so easy...) So, in Gnome MPlayer, go to View --> Video picture settings (the option name can differ, I'm not using an English version), so the last option is it. For me decreasing contrast with 40 points and saturation with 80 points is quite fine. (You may need to decrease it more, as I've set my monitor to be darker.) The point is that here you can change the settings to make the view acceptable. Unfortunately, it must be done every single time you start the application – but at the moment we don't seem to have a better "solution".

So the question is still open.
