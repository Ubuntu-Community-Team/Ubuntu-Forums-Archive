This is an improvement on a very old post

[http://ubuntuforums.org/showthread.php?t=617019]("http://ubuntuforums.org/showthread.php?t=617019")

My partial solution on Hardy is:

alt+F2 and run gconf-editor
go to apps/rhythmbox/plugins/mmkeys and uncheck both "active" and "hidden"

unbind the media keys in the audio section of System/Preferences/Keyboard Shortcuts

Bind the keys in Amarok in Settings/Configure Global Shortcuts

For reasons unknown the binding only works if the Amarok window is in focus
