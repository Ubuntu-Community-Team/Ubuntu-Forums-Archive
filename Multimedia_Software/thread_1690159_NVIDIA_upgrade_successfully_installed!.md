### Post by fester225 on 2011-02-17
I finally came across a set of instructions to do a relatively easy upgrade of my NVIDIA driver to 260.19.36.

[http://www.multimediaboom.com/how-to-install-nvidia-display-driver-260-19-36-in-ubuntu-10-10-maverick/](http://www.multimediaboom.com/how-to-install-nvidia-display-driver-260-19-36-in-ubuntu-10-10-maverick/)

sudo add-apt-repository ppa:ferramroberto/nvidia
sudo apt-get update
sudo apt-get install nvidia-current nvidia-current-modaliases nvidia-settings


Doing the upgrade through Terminal and the repositories sure beats doing it the way NVIDIA says to do it!
