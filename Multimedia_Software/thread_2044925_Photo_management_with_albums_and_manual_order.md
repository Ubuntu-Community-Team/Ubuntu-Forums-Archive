### Post by MalachiConstant on 2012-08-20
Hi all,

so far I've been using shotwell on Ubuntu. And I really like it's simple interface and how it integrates external editors such as Gimp. I could live without the option of creating albums as shotwell's events come pretty close to that functionality. But without the ability to manually order my photos it's completely useless to me. To me being able to create albums and manually order photos within them is the absolute basic functionality of a photo **management** software, only surpassed in importance by the ability to actually display the photos.

So I started to look for alternatives. But so far, I haven't found any. Gthumb apparently lost this functionality a few years ago (why?!) and even the mighty digikam falls short. The only program that supports manual ordering that I could find is Corel's Aftershot for Linux, which is closed source and costs 60$.

I can't believe there is no open source alternative!? Did I miss something?
