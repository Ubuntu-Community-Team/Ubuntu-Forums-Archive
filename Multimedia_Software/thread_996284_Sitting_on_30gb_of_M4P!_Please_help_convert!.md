### Post by damis648 on 2008-11-28
See [this page]("http://itunesm4ptomp3.wordpress.com/2008/09/16/convert-itunes8-music-to-mp3/"), the software creates a virtual CD burner, which iTunes burns to. As this happens, the software rips and re-encodes to MP3/WMA/WAV. This sounds like the easiest solution. I suggest that if you do this, you use MP3. It is a rough solution, but it works and is more automated than doing the DVD thing yourself. Just copy the music to the Windows partition, import to iTunes, get the account username and password to authorize iTunes for burning it, make a playlist, and burn!
