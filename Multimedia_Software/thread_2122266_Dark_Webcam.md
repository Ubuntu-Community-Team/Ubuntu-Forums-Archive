When my son and I skype, or if I use any other webcam software, the webcam has good lighting for about 10-30 seconds then switches to a much darker picture? What do I need to do to rectify this? I searched webcam on the forums and nothing came up at all, which I find strange.

### Post by coldraven on 2013-03-04
I'm not sure if the following still works in the latest Ubuntu but it's worth a try.
Install guvcview from the Software Centre.
Close all web-cam programs like Skype etc.
Run guvcview and see if there is a check box for "Auto levels" or something similar.
If so, uncheck it.
If necessary adjust the Brightness level.
Quit guvcview (no need to save) and try Skype again.
