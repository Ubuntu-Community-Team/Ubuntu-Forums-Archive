### Post by evilsoup on 2012-11-02
Adobe stopped supporting flash for Linux a little while ago; they're going to keep up security updates for the current version for the next five years (hopefully long enough for the transfer to HTML5), but we're not going to be getting any new versions from them.

Google have promised to support flash through the 'pepper' component of Chrome, but that's only available for Chrome, so if you need the latest flash (you probably don't), that's the best your going to get on Linux.

There might be specific workarounds for any problems you have, so ask on these forums if you find you can't do something flash-related.
