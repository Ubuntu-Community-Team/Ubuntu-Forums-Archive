### Post by Novalux on 2008-07-28
Well obviously a newer, more expensive card, will generally allow you to run games with higher quality settings. Portal can look pretty good with a newer graphics card, but it will run with lower end cards. I've got an nVidia 7950gt and Ubuntu/Compiz works great on it, and on my XP install it runs Portal, Half Life 2, etc. really good at 1400x1050. If my monitor supported 1280 resolution I would use it and it would run perfect at full settings.

Do a search on newegg.com for your price range and look at the comments. For example, this card can apparently run Crysis pretty well, which is a feat. [http://www.newegg.com/Product/Product.aspx?Item=N82E16814150287](http://www.newegg.com/Product/Product.aspx?Item=N82E16814150287)
