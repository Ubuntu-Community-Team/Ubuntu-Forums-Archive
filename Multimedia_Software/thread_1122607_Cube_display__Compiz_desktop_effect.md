### Post by Fenris_rising on 2009-04-11
Hi there

Not sure if your onboard graphics are up to it but you need to install compiz.

Go into synaptic and type compiz in the search.

If compiz is not selected in the list then select it.
Also select fusion icon, emerald, simple-cssm.
Synaptic will pick up all the other dependencies you need including the main cssm menu item. The simple cssm appears in the;

right click desktop>Change desktop background>visual effects

select custom; note the preferences box. This gives you the simple compiz setup list. Both this one and the main cssm icon will appear in the preferences menu as well along with emerald. For emerald you may need to DL some .emerald themes to decorate your windows with.

Ensure you have any proprietry drivers installed for your GPU if required.

Fusion icon appears under Applications>System tools and is the main configuration gui for compiz IE lots and lots of things to play with depending on your gpu. Click the fusion Icon it then appears top right of screen right click this to access all areas.

regards

Fenris
