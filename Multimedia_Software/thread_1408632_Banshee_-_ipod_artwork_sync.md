### Post by boombox1387 on 2010-02-22
The 'killall nautilus' was as workaround to get iPods working with Banshee 1.5.1 on Ubuntu 9.10.  A new version of Banshee (1.5.3) fixes this this issue (yay!)  You can find information about updating [here]("https://launchpad.net/~banshee-team/+archive/ppa").

As far as artwork goes, it sounds like you're running into [this bug]("https://bugzilla.gnome.org/show_bug.cgi?id=564185").  There might be some workarounds there that you can try.
