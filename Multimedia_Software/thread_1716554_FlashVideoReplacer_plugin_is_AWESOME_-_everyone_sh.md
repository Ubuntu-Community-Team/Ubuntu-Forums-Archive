### Post by nrundy on 2011-03-28
I have never been a fan of Adobe software. I find it of the lowest quality. Having to run Flash for the Web to work has always irked me. 

Well, with this plugin, I'm a big step closer to dumping Flash Player.

The plugin will automatically redirect you from a Flash-Player video to a WebM version if available. If a WEbM is not available you have the option of playing the video with a standalone player. I use VLC and it starts right up on the page and plays the video. Works great!

Thanks a lot lovinglinux. You do good work!

[https://addons.mozilla.org/en-us/firefox/addon/flashvideoreplacer/](https://addons.mozilla.org/en-us/firefox/addon/flashvideoreplacer/)
