Hi all,
I had this problem recently where the maximum volume was not as loud as it previously was in windows. I have seen this question asked a couple of times with no real solution.

The solution I found was go to System -> Prefrences -> Sound
There the output volume is set at 100% but can be increased further.
If you move that slider to the maximum then your volume will be as loud as it was before.

Sorry if this solution has been posted elsewhere, I searched when I had the problem but found nothing.

Hope this helps someone.
