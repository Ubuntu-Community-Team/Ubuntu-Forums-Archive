### Post by cozmicharlie on 2008-06-07
Do you have the music saved somewhere other than the ipod?  I am not sure what you mean by Banshee changed the software it sounds like it messed with the file system.  Have you tried accessing it with another package like gtkpod?  If you have the files saved or can access them to save them I would recommend reinstalling the firmware from Apple.  It is available on Apples web site.  Warning though, this will delete all your music but it will restore the ipod to the original state so you can use it.  If you have the music backed (or you don't mind reloading the music) up you can simply reload the music.    

FYI - if you are comfortable with using another system on the ipod I would recommend Rockbox.  If you install Rockbox it uses a normal file system (not the crazy system Apple uses) and you access it just like you would any other file - works with both Windows and Linux.  You can also play flac or other codecs that the native ipod cannot.  I use it on mine and I would never go back to itunes.

I hope this helps.
