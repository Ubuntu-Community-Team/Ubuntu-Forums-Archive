### Post by jawilljr on 2011-09-15
The_chef, I have a LG tv also. On mine there are two ways to fix it...

1. Set the aspect ratio to "Just Scan"

2. Set the input label of the HDMI input to "PC"

Either way it sets the TV to 1:1 pixel mapping.

What is happening is the TV defaults to "overscan"
