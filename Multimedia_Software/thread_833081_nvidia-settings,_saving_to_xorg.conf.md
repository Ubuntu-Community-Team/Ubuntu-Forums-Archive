### Post by aeiah on 2008-06-18
yea, gksudo or sudo nvidia-settings. the xorg.conf file is in the root file system (its at /etc/Xll/xorg.conf), and so you need root permissions to access it (this is what sudo / gksudo does in this instance, after you provide it with your password).
