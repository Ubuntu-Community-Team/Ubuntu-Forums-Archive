### Post by 67GTA on 2009-06-25
Having 20,000+ awesome, free, and open source software packages at your fingertips is a weird feeling after doing things the Windows way;) You will probably go cross eyed looking at all of the packages through the package manager. A less technical package manager is the Add/Remove app in the applications menu. Apt-get is the app that Ubuntu uses to install software. Sudo is used to give you temporary "administrator" powers to make system changes like installing software. When you use Synaptic or Add/Remove (frontends) to install something, they use apt-get (backend) to do it in the background.
