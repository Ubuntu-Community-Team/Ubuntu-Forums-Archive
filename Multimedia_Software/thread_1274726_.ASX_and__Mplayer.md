### Post by cor2y on 2009-09-24
if you want to record asf streams use mimms (its in the repos) or there are firefox addons like downthemall or download helper that allow you to save streams or even mplayer itself with its dumpstream syntax its been a while since i have used this so i cant remember it correctly see google for help but it goes something like this
```
mplayer -dumpstream -dumpfile file.asf http://www.whatever.com/file.asx
``` that should allow you to save the stream mimms works in the same manner as that.
