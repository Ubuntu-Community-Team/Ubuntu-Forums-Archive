Veljko, try running " sudo nvidia-settings " from the terminal - that might help you configure your settings. You should back your xorg before doing anything -> sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup This copies your current xorg file to back-up, and if things go terribly wrong you can restore your old file.

If you post the contents of your xorg.conf the folks on this forum might be able to help.
