### Post by roquefilipe on 2008-08-05
yes, it does, but the subtitle file must have the same name has the movie, except for the extension. Its location also has to be very specific. 

Check under Settings->Preferences, then Video->Subtitles/OSD with the Advanced options enabled under Subtitle auto-detection paths which directories VLC uses to search for subtitles files.

I think you can also drag & drop the subtitle into the video.
