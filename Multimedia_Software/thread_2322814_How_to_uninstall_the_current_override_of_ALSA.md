### Post by Yellow Pasque on 2016-05-03
> **Salvatore_Palomino said:**
> First of all, what is an override of ALSA? Second, how do I uninstall this override and install the newest one?

It refers to installing an updated ALSA package or building ALSA from source to "override" (overwrite/replace) the version that comes with Ubuntu. If you haven't done that, then no need to worry about it. Proceed to: [https://wiki.ubuntu.com/Audio/UpgradingAlsa/DKMS](https://wiki.ubuntu.com/Audio/UpgradingAlsa/DKMS)
