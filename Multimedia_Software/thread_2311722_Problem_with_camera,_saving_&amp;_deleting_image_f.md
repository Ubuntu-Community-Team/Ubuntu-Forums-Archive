### Post by germeten on 2016-01-30
It turns out when I created a particular file folder, I did it "as root." That put a lock on it and I was unable to edit or delete. Left clicking the menus finally made me see the "as root" option so I used it to delete the folder, remembering I'd probably used it to create the folder and not noticed. So that aspect of my problem has been fixed. I won't know if my kolour-paint problem is fixed until I try uploading photos again.
