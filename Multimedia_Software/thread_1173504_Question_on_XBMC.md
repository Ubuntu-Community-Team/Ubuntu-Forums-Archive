You can do that, with use of the Aegis skin it is a real nice workplace for all things multimedia ;-) (only wish the back button on my remote worked :-(  )
Troubles with it (on my system, AMD 4800 X2 64, 2 GB, Asus AM2 mobo, Gforce 8400 GT graphics):
1.  .MKV files crash the system often when stopping or rewinding.
2.  some files no audio (plays ok with SMPlayer?) but seldom
3.  When using GPU prepair for a HOT video card :-( 
4.  browsing media files kinda slow

and some very small minor annoying things but that is just my taste ;-)

Pro's
1.  uses GForce 8000+ GPU for HDMI high def (finally 1080i plays flawlessly)
2.  when on HD it upscales video real nice, pictures get real crisp!
3.  Inbuilt aligner for over-scan troubles! 
4.  Looks gorgeous!

maybe look at Elise, it is simple, stable and fast, just sort off limited, and the jaunty version will probably drive you nuts with it's checking for media each time it starts (hint, go make coffee, do lunch and it will be there ;-P ) But there is probably a way to turn that off.

And also using your normal desktop to browse and play works fine, prepare to install a whole load of media players for video, have not found a single one that plays them all flawlessly..

My choice atm. SMplayer for .MKV and .MP4 (don't forget to set the number of threads he can use if you have a multicore proc)
VLC for .AVI and rest.. 

good luck!
