### Post by Razmear on 2010-10-19
Got it fixed. 
Had to manually edit xorg.conf to set the correct freqs: 

Section "Monitor"

    # HorizSync source: xconfig, VertRefresh source: xconfig
    Identifier     "Monitor0"
    VendorName     "Unknown"
    ModelName      "CRT-1"
    HorizSync       30.0 - 95.0
    VertRefresh     50.0 - 160.0
    Option         "DPMS"
EndSection

reboot and now all modes are available. 

eb
