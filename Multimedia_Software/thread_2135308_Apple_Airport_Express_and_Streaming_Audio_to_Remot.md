### Post by TheFu on 2013-04-14
First, you need to know which protocols they speakers support, then look for software that provides that protocol as a server. No need for any new hardware, of that, I am certain.

If the remote speakers are controlled by some other hardware or software, then the protocol(s) supported by that software will be important to know and support. 

As for browser streaming, I've been happy with gnump3 for about a decade, but almost any media streaming server should do this.
