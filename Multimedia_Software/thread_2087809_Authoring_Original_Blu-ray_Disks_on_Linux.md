### Post by ejolson on 2012-11-24
I found this thread old thread

[http://ubuntuforums.org/showthread.php?t=1605152](http://ubuntuforums.org/showthread.php?t=1605152)

and was wondering if anyone has more current information.

Judging from what the search engines reveal, nothing substantial has changed on Linux in regards to blu-ray mastering and authoring of new disks from original video content for the last 3 years.

As far as I can tell, the only way to master a bluray disk is is to use

[http://www.videohelp.com/tools/tsMuxeR](http://www.videohelp.com/tools/tsMuxeR)

which hasn't been updated since 2009 and then burn the UDF filesystem using imageburn

[http://www.imgburn.com/](http://www.imgburn.com/)

running under Wine.  I think

[http://pel.hu/bdedit/](http://pel.hu/bdedit/)

also runs under Wine.

The combined functionality of these three programs would make a full featured mastering application, but it is impossible to use these programs to build such an application because they are closed source.  I'm not sure whether they are closed source because of NDA arrangments with the bluray consortium or whether for more mundane reasons.

So to repeat the earlier question, does anyone know about new developments for authoring bluray disks on Linux?
