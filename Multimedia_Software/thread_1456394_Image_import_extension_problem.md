When I import images from my camera Ubuntu assigns them upper case extensions i.e. .JPG (I assume it's Ubuntu's doing as in windows they're lower case)

This causes a problem when I come to upload them. For some reason photobucket doesn't recognise upper case extensions and changing each extension by hand takes ages!

How can I force it to give lower case extensions to imported photos?

The save action lets you specify the extension and as I use it to resize I've just combined both actions.
