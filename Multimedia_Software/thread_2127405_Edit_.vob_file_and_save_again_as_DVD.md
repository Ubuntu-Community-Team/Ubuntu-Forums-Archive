### Post by veroslav on 2013-03-20
You could change the output format to 'MPEG-PS' and set Audio and Video to 'Copy'. That will preserve the audio/video quality as-is and export as a new VOB/MPEG. As far as I know, you always export to a new file, I dont think its possible to directly edit the original VOB.
