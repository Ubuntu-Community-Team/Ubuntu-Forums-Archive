### Post by mastermindg on 2009-10-07
Are you referring to the ID3 tags? If so, you can just edit the tags and use a uniform file convention. It's a pain but it's worth it. Check out Audio Tag Tool. It does batch tag editing, simplifies the process.

### Post by mastermindg on 2009-10-07
Amarok 1.4 allows you to sort via filename rather than ID3 tags. Right click the column header and choose Show Column --> Filename. Not sure about 2.2, though.
