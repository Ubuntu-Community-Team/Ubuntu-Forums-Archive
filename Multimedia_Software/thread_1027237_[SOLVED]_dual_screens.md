### Post by enchesss on 2009-01-01
have you installed the latest nvidia drivers?

system -> administration -> hardware drivers -> activate nvidia



have you installed the nvidai control panel

system -> administration -> sybaptic package manager -> nvidia-settings


then configure your monitors


system -> administration -> sybaptic package manager -> nvidia X server settings


Make sure that your tv was plugged in when you booted (may need to reboot but maybe not)

You can play with these settings - need more info about tv size for detail if you need further help

sometimes the tv is hidden behind the monitor icon in X server display configuration. Just select it with the model arrows (ou may need to enable it and logout or reboot).
