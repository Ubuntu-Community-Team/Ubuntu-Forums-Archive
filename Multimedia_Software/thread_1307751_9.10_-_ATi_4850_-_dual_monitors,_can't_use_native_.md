### Post by MrNiceguy on 2009-12-08
Don't know if you're still monitoring this thread, but here's a suggestion:

If you're trying to set up your displays using the standard Ubuntu Systems applet, try using ATI's catalyst control center instead.  Hit Alt-F2 to bring up the "Run program" dialog.  Then type in "gksu amtcccle" without the quotes.  Put in your password when prompted.

I couldn't get dual-screen working on my ATI 4770 using Ubuntu's tool, but ATI's worked like a charm.  I've got dual-screen with compositing on both.
