### Post by scragar on 2009-09-04
Xorg.conf isn't always needed, try pressing ctrl+alt+F1 to get to a tty, and run:
```
sudo mv /etc/xorg.{conf,backup}
```To rename the xorg config file, then restart X.

Even if that doesn't work graphically Xorg will create a log and inside the log will be the config it created for itself when starting up(It won't save it by default though), if your graphics are bad without the config file just copy the section from the log and replace the parts required.
