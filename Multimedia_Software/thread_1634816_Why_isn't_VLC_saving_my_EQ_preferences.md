### Post by mc4man on 2010-12-01
Open vlc -> tools -> preferences. You'll see a checkbox for Allow only one instance.

Now click on 'Show Settings' - All
Go Audio -> Filters and enable the Equalizer (screen 1

Then in the filter list click on Equalizer (screen 2
You can input 1 custom setting in the Bands gain box, 10 values separated by a space
The preamp setting is the Global gain
You might want to first set the equalizer in the vlc player interface, get it to what you like and write down.

Then click save and close. When using vlc the equalizer and settings will be automatically enabled and set

Or you could search out threads on the system-wide pulseaudio equalizer and ck. out, Some like, some don't, though more the former
