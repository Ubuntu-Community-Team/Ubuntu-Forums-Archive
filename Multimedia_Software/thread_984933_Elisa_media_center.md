### Post by eternalnewbee on 2008-11-17
**Compiz** is the software that enables you to have a **3D desktop**. It is installed by default in 8.10. If you want to run **Elisa**, you should disable it. To do that, go to **Main Menu > System > Preferences > Appearance > Visual Effects**, and click **none**.
Hope this helps you.
