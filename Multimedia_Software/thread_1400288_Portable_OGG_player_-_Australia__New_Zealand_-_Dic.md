### Post by grumpthehermit on 2010-02-06
Just wanted to post a quick recommendation for a small portable 8gb OGG ( & mp3 ) that I purchased yesterday from our local Tandy store.

It's a "generic" Dick Smith branded player that supports OGG out of the box. Code on the back of the player is a A5146, it's similar in size / looks to an apple nano.

The packaging on the player notes OGG support but the website does not mention this at all sadly, the player claims to support the following ( I've only tested my OGG's so far )

Audio - MP3, WAV, OGG
Video - AVI (MPEG4/XVID)
Photo - JPEG/GIF
EBook - .TXT files

The player I bought is the 8GB Version, there is also a 4GB version, A 8GB "Video" version with a larger colour screen and a 16GB version, I have not seen the 8GB "Video" version or the 16GB version so I can confirm that those will have OGG support but I "assume" they will ...

The player has a 176 x 220 COlour TFT LCD display and a touchpad for all controls other than the off / on slider button with is located on top of the player above the screen.

This is a link to the Aussie sites selling this player, in Australia Tandy and Dick Smith mostly stock the same product Tandy sells allot of Dick Smith branded product.

[http://www.dse.com.au/cgi-bin/dse.storefront/4b6dff0d0149c0b42740c0a87e0106b1/Product/View/A5146](http://www.dse.com.au/cgi-bin/dse.storefront/4b6dff0d0149c0b42740c0a87e0106b1/Product/View/A5146)

If you read the reviews you will see there is a good one from an Aussie talking about Linux and OGG also.

The files I am playing have all been created by me using Rhythmbox using Ubuntu 8.10 using the highest quality setting possible and saving files as OGG and not OGA so my files are all massive and come up as 499kbs in Rhythmbox.

The player shows up as a portable media device in Ubuntu 8.10 and it's as simple as dragging music into the music folder once you have the player connected ( charges off the usb also ).

The only negative I have abot the player so fair is it seems to scramble the track order and in a few cases lists every track twice when viewing them under the "Album" heading.

When viewing and playing under the "Folder Explorer" entry everything seems fine so far.

All in all pretty good for $79 dollars, if anyone else has one I'd like to hear what they think of them ?

( PS - The supplied ear bud headphones are very low quality, most people would probably want to use their own or buy better ones )
