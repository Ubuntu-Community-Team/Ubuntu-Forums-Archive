### Post by Elfy on 2009-06-20
Open a terminal from the Applications/Accessories menu - install the restricted extras package

```
sudo apt-get install ubuntu-restricted-extras
```

You'll be prompted for your password - it will not show as you type - this is normal.

Have a look here also - should give you some extra help if needed - [https://help.ubuntu.com/community/RestrictedFormats/](https://help.ubuntu.com/community/RestrictedFormats/)
