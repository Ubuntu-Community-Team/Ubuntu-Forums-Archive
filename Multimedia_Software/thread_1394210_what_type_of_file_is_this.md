### Post by Uncle Spellbinder on 2010-01-30
WMA - Stands for Windows Media Audio. It is the file type promoted most by Microsoft, the company that invented it. It is the native format used in Windows Media Player, both on Macs and PCs. It competes with the MP3 and AAC formats.
