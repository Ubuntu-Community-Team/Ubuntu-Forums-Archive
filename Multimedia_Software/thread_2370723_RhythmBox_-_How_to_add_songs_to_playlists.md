### Post by mc4man on 2017-09-06
If the files you want to add have already been imported into RB you have to go to your  Music library view. Then select a track(s) > right click > Add to Playlist >  pick the one you wish to add to.

If those file(s) haven't been imported then I believe you must 1st. import into Rb, ect..

In other words it's not pick playlist > add files, it's pick files > add to a playlist
