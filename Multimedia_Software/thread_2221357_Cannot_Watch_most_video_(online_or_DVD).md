### Post by TheFu on 2014-05-01
The live-CD install on a flash drive is not a full install. It only provides a base OS to test out hardware compatibility and nominal web browsing. You can install packages as needed, but those will be lost at reboot, I think.

Install to a HDD, then use aptitude to resolve any dependencies if "software center" or synaptic or apt-get doesn't do it correctly. aptitude is smarter, IME.

Also ... depending on local laws in your country, you may not be allowed to install DRM packages legally - needed for watching DVDs. Ubuntu asks about this during the installation process.
