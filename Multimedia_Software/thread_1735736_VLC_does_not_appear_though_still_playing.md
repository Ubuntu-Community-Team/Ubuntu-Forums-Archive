### Post by satanselbow on 2011-04-25
Open you home folder in nautilus from the menu select View -> Show Hidden Items

navigate to .config -> vlc and delete the 2 conf files there.

When you restart VLC it will automatically recreate the default preference files therefore returning the original skin ;)
