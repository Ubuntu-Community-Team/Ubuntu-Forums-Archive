### Post by Isyaltut on 2010-10-08
I don't know about import manually, but banshee extension lyrics is pretty good. Just open terminal and type sudo apt-get install banshee-extension-lyrics. You must have internet connection to get it working. And if you enable banhsee to write metadata in to files, it will be saved permanently in to the mp3 file itself. It's pretty useful for me since I have iPod Touch and all of the lyrics is displayed correctly when I played on it.
