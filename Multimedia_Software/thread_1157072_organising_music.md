### Post by spacesamurai on 2009-05-12
Yes it is, made to work beautifully with Ubuntu. If you have your repositories set to default, you can install it as follows.

```
$ sudo apt-get install easytag
```

Once installed, you can run it as follows 

```
$ easytag
```

Hope this helps.
