### Post by lovinglinux on 2009-07-09
You shouldn't remove firefox-3.0 because it is the official package and there are applications like gecko-mediaplayer that depends on it. Removing firefox-3.0 also breaks the the searchplugins of Firefox 3.5.

Firefox 3.5 is not officially supported yet by Canonical. It's the community that makes it available through the universe repository too feed the Tamagochies :)
