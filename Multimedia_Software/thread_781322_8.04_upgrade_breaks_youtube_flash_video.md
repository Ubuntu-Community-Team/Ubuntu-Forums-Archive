### Post by Iskendar on 2008-05-04
Hi all, 

Since I've upgraded to Hardy, I can't play youtube video anymore in firefox 3b5. Tried reinstalling flashplugin-nonfree using apt-get, even removed it and installed from the tar.gz file from the adobe site. Checking in about:plugins shows:
```

Shockwave Flash
    File name: /usr/lib/firefox/plugins/libflashplayer.so
    Shockwave Flash 9.0 r124

```
But still I keep getting the following error on youtube:
```

 Hello, you either have JavaScript turned off or an old version of Adobe's Flash Player. Get the latest Flash player. 
```
Weird thing is that non-video flash sites work perfectly, and I've been able to play flash video from other sites like [http://www.tudou.com](http://www.tudou.com). Any ideas?

---

### Post by Iskendar on 2008-05-04
Found it. Adobe flash player now apparently has security settings you have to change via adobe's website. Youtube video plays fine afterwards. Found this by trying konqueror (no youtube) and opera, which launched a popup about adobe's security settings. Changed the settings and now youtube plays in all three browsers. Hurray for opera! Weird choice by adobe, you'd expect some settings interface plugin for the browser...
