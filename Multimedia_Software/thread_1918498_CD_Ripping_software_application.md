### Post by Rodney9 on 2012-02-01
Asunder - [http://littlesvr.ca/asunder/](http://littlesvr.ca/asunder/)

An application to save tracks from an Audio CD 
as WAV, MP3, OGG, FLAC, Wavpack, Musepack, Monkey's Audio, and/or AAC files.

Uses CDDB to name and tag each track
Creates M3U playlists
Can encode to multiple formats in one session
Simultaneous rip and encode
Allows for each track to be by a different artist
Does not require a specific desktop environment
