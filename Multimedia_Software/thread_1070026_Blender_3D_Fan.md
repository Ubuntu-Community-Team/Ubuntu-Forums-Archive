### Post by ragtag on 2009-02-14
Hi,

 The one you can find here [http://www.blender.org/download/get-blender/](http://www.blender.org/download/get-blender/) (scroll down to Linux x86-64) should work. At least the 32 bit version works on my Ubuntu machine, and the 64 bit version on my CentOS machine at work. The 32bit on in the Ubuntu 8.10 repositories works too. :)

Blender has seemingly almost no dependencies beyond working OpenGL (run 'glxgears' on the command line to test) and Python. So installing is as simple as downloading the package, unpacking it and running it (more or less).
