### Post by emptythevoid on 2011-03-01
I use XVidCap.  It works well enough.  The program will give you a red square you can resize to any shape - whatever is in that red box is what gets recorded.  You can specify what kind of frame rate you want, if you want the PC's (or mic's) audio recorded or not.
