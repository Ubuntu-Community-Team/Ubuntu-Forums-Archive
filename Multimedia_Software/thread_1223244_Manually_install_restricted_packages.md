### Post by mwcrowley on 2009-07-26
There's definitely a way to do this.
I haven't done this but I think it would work for you.
Also these packages should work for either Kubuntu or Ubuntu, if they don't please post back here

Go to [http://packages.medibuntu.org/](http://packages.medibuntu.org/) 
click on the your version of Ubuntu.  Click on the binaries you want.  That will give you a page with a download link and information on dependencies.  Getting the dependencies you need will be difficult because you are doing this by hand.  This is definitely possible.

I think your best choice is to find a distro with restricted extras already installed.  Even though you will have to reinstall I think that it will still save you a great deal of time.
[http://www.nixtutor.com/ubuntu/20-ubuntu-derivatives-you-should-know-about/](http://www.nixtutor.com/ubuntu/20-ubuntu-derivatives-you-should-know-about/)

[http://kiwilinux.org/kiwi/en/](http://kiwilinux.org/kiwi/en/)  This one looks promising but it only has English, Romanian and Hungarian installed.

If you search there is a distro out there with restricted packages preinstalled.  When you find it please post back, and I'll do the same.
