### Post by wolfen69 on 2009-01-15
i know K3B has "normalize" audio. after installing K3B you will also need libk3b2-extracodecs for mp3 support. go to system>administration>software sources and make sure all repos are checked off. then go to synaptic and install k3b and the extra codecs.
