There is [gcstar](apt:gcstar), which is a video cataloging app.

You could also try my media center extension for Firefox, called [FoxMediaCenter]("http://fmc.isgreat.org"). It can scan your media folders, create contact sheets of each video, create video thumbnails using Speed Dial extension, record TV, import xmltv data, track which programme and videos you have already watched and more. Any feedback would be much appreciated.

If you want something like YouTube, then you could install Apache/MySQL/PHP and run [Gallery2](http://gallery.menalto.com/), which is an excellent photo gallery application, but that also supports video with additional plugins.
