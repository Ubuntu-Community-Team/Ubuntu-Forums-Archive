### Post by b0b138 on 2008-11-17
two issues... 1. Even with the pass through, I don't think you can burn from a vm. 2. XP doesn't know what to do with a bin or iso for that matter. You would need burning software.

Easiest thing to do would be to move the file to ubuntu and burn it from there. Or, in XP download and install virtual clone drive. This will make a fake cd drive that will mount cd images. [http://www.slysoft.com/en/virtual-clonedrive.html](http://www.slysoft.com/en/virtual-clonedrive.html)
