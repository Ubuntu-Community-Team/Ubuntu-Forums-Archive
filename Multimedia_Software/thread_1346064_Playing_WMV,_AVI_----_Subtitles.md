### Post by JBAlaska on 2009-12-04
Open the terminal and do:
```
apt-get install ubuntu-restricted-extras
```

If that fixes the wmv thing, try installing VLC to play your avi's with subs (srt). Note they have to be in the same directory and named EXACTLY the same as your Avi file except with the .srt extension
