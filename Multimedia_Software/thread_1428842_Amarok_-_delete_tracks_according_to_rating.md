### Post by Hakimjo on 2010-03-14
Thanks for moral support.  The problem was solved in the following way:
- the filter does get confused when using another interface language than English.  e.g. In French "rating" is translated "vote" but the filter works on neither - so: switch to Englich interface if you want to filter rating
- one star rating does count for 2, since Amarok allows halve stars ...

these things applied, the filter and subsequent elimination of tracks works just fine.
