### Post by milesje on 2008-10-18
You can get the files from audible.com by following these easy steps.
1)purchase the books.
2)on the web page (My Library) click on the stream link for the book. (firefox will ask if you want to download the .pl file) 
3)once you have the .pl file open it with a text editor (gedit, vim) whatever you use
4)copy the url line (everthing inside the quotes) and past it into firefox. which will not download the .aa file.
5)use what ever software you normally use to sync files to your iPod

Note: you must have already used this mp3 device with audible.com before.
