### Post by snowpine on 2010-02-19
It depends whether or not the music is DRM from the iTunes store. Regular files it should be as easy as copying them over and installing the ubuntu-restricted-extras package so you can play mp3s. Rhythmbox is a popular mp3 player for Linux but there are many other choices too.

If the files are Apple DRM then the easiest solution is to burn them to CD and re-rip them on Linux.

### Post by litemirrors on 2010-02-19
Alright the first thing I would do if I was you is

1. Clean install Ubuntu 9.10
2. iTunes files with .m4p are DRM encrypted and cannot be played through anything other than the iTunes player.
3. m4a = unencrypted audio

Anyway the restricted extras will allow playback of various file types, however you also violate law basically by using them. However Fluendo [http://www.fluendo.com/](http://www.fluendo.com/) does sell a plugins package for 40 USD that's legal but I noticed they offer the mp3 plugin for free!

Let's say you do burn cds and copy them into Linux through Rhythmbox, well they'd be transcoded from already lossy files. So basically you get generational loss on the sound. Part of the reason I deleted all my iTunes library and went FLAC.
