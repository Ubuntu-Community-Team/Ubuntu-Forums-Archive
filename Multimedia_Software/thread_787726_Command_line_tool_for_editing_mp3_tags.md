### Post by kozlovsk on 2011-09-04
Like eyeD3 better, can convert different versions of ID3 tags at the same time, so there will be no several versions if tag present:
sudo apt-get install eyeD3
eyeD3 --to-v2.4 -t "Song title" -a Artist song.mp3
