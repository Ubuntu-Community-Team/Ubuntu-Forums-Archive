### Post by tuppe666 on 2010-12-03
Hi sorry to take so long. The answer is in the tags. Everything with MP3s is the tags. If you right-click(to get the context menu) you will see there is a edit track information one of the tags is "Compilation Album Artist" that is what you have to set all the tracks to. Just put a tick in the tick box and for an album with a guest[sic] singer just use the main artist name, or for "99 Disco" etc.  just type "Various Artists". You don't have to do them individually you can select several track or albums(of the same name) like you would in any other manager using shift or ctrl with a click, and edit the same way as before only you get a little layer button which allows you to change the same tag on many files at once.

Be aware though that Banshee is a player first and a tag editor along way down its list, as I said everything is in the tags. Choose a tag editor, most users in these forums seem to prefer easytag. I would say puddletag as its in development although as yet not as complete. If you have a little time and are more familiar with tags. I would suggest you try MP3 Diags which is hands down the best program for MP3s on any platform.
