### Post by BicyclerBoy on 2011-01-10
FYI
x264 is not a container it is a video encoding program.

It outputs video encoded in the format MPEG-4/10 (H264AVC) .
There are multiple levels of encoding complexity allowed.

The container could be mpeg2-ts, MPEG4, mkv etc.

It is all very confusing because people sometimes refer to: the codec, the container &/or a proprietary name.
Sometimes the container dictates the codec.
