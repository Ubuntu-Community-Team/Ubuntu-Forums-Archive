### Post by plh on 2008-07-02
OK, problem is solved: FN-keys are disabled in Hardy on this laptop model. Thus, the only way to switch to the other display is to use the menu System/Administration/Nvidia X server settings WITH THE LCD PLUGGED IN and running, and then to manually switch to the LCD using the menu.

Hope it can help whoever owns this type of machine and runs into the same trouble :) ...
