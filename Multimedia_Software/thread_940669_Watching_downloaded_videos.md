### Post by stromdal on 2008-10-07
I installed MythBuntu (Hardy) on a box some time ago, and I have no problem playing DivX and XVid AVI's. I haven't tried burning to DVD, but that shouldn't be a problem either.

Most likely, you haven't installed the w32codecs - a set of proprietary codecs needed for DivX /XVid playback. You can enable the proprietary codecs in "MYTHBUNTU CONTROL CENTER" / "Proprietary Codecs".

Also, you should check out this thread ([http://ubuntuforums.org/showthread.php?t=766683](http://ubuntuforums.org/showthread.php?t=766683)) for tips and tricks to get video/multimedia/DVD player/etc working in *ubuntu. MythBuntu is based on Xubuntu, so for Xubuntu-specific instructions, you should follow these.

I hate to say RTFM, but the MythBuntu installation manual is quite informative (although not complete) and if you're new to Linux/Ubuntu/MythBuntu/MythTV I really suggest browsing through it to learn some of the basics of MythBuntu/MythTV. It is huge, but you will probably be skipping A LOT of it due to relevance. You can find the manual at [http://www.mythbuntu.org/installation_manual](http://www.mythbuntu.org/installation_manual)

Hope this helps. If it doesn't help, or if you need help with a specific error message or such, please provide the relevant info/error message for better help.

Regards
