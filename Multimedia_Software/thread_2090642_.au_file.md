At this link you can read about au-files and download a sample file.
[[COLOR="Red"]http://file.fyicenter.com/14_Audio_.AU_File_Extension_for_Audio_Files.html[/COLOR]]("http://file.fyicenter.com/14_Audio_.AU_File_Extension_for_Audio_Files.html")

I can play it with ***mplayer*** which can be installed with
```
sudo apt-get install mplayer
```
