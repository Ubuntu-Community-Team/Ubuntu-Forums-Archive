First things first. If you download anything from a website to your computer, put it in /home/{*****}/Download (where: {*****} is your username). By doing so and keep doing it, you're absolute sure that everything you download can be foundin the Download folder. 
 
Secondly: Correct me if i'm wrong, but i thought that ruby ripper can be found in the Ubuntu repo. Go to system > Administration > Synaptic packagemanager. Do a search on ruby ripper.
 
As much as i can judge, saying that you downloaded it, and afterwards it was gone, myabe you didn't downloaded if completly (by mistake). Try downloading it again to your Download folder => [http://code.google.com/p/rubyripper/downloads/list](http://code.google.com/p/rubyripper/downloads/list). Check that folder if it is there. Be aware it's a tarball (.tar.bz2). You have to unzip it first by just right clicking on the file and select exctract here in the pop-up menu.
