### Post by syczu on 2008-06-29
I'm using Ubuntu and I can use shortcuts since I've installed amarok. See in amarok 'settings->global shortcuts configuration' and check your settings. I have set it on default and it works well.

### Post by dn* on 2009-02-22
To anyone still having trouble, try System->Preferences->Keyboard Shortcuts and disabling (by hitting backspace) any keys that might already have GNOME preset.
