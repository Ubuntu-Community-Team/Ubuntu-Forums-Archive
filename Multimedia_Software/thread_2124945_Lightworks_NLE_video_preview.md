indeed. I've been waiting for a very long time for a PROFESSIONAL video editing application to come on the Linux Platform.  I'm Beta testing Lightworks 11.5 on Ubuntu 10.5 and it's a very capable application. It supports all professional video formats and can even handle 3D STEREOSCOPIC clips.  It's definitely a class apart!

One thing disappointed me with Ubuntu is that in 2010 (ever since  version 11.04) they seriously messed up the IEEE1394 stack making DV  capture via FireWire impossible - worst -  they don't seem to be willing  to fix it, despite the numerous bug reports.

It has just been ported to Linux and still in Beta so for it may be a bit rough around the edges. But it&#8217;s the only professional NLE video editor for Linux.
