### Post by agentsmith23 on 2008-06-03
Try this. Go [here]("https://a248.e.akamai.net/f/674/9206/0/www2.ati.com/drivers/linux/ati-driver-installer-8.28.8.run") to download the driver for the 9250. Download it to your desktop then from a freshly opened console type "cd Desktop" then type "sudo sh ati-driver-installer-8.28.8.run".

That should get your video card running with the correct drivers. I'm not sure what kind of performance you should expect from that card with compiz but that should get the drivers working.
