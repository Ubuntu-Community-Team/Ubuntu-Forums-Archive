### Post by UKBB on 2011-06-26
I never have been able to get netflix to work on my Ubuntu pc. What I did was load virtual box and create a virtual Windows XP machine on it. It worked but once I put it in full screen mode I couldn't get out of it. I could watch the movie or show but couldn't control it (pause, ffwd, rwind etc..). I had to reboot the pc to get out of it. I figured that this was overwhelming the on board graphics so today I installed an Asus 210 Silent video card. Now I can go full screen with no issues and everything works. An added bonus is that the CPU isn't working as hard now to play the video so the fan doesn't ramp up and cause noise.
