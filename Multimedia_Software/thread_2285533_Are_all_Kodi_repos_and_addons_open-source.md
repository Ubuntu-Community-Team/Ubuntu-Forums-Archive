### Post by Rob Sayer on 2015-07-09
I've never actually heard of an open source repo.  I guess it'd have to just host open source software.

Note that sourceforge is a very well known "open source" site but they are somewhat notorious for hijacking source links so they can put adware in the package.

Pure open source is a lot less common than people think.  It's usually a combination of open source and closed source.  If your ubuntu install has ubuntu-restricted-extras it's mixed open and closed as well.  If you have firefox and want truly open source you'd better purge it and install iceweasel.

I very much doubt kodi/xbmc is 100% open source either.

Whether those plugins in that link are truly open source, I don't know.  I'd be more concerned about having to add them from an untested ppa.  They'd have to do something really important for me to do that.  I don't have any ppas in my source list and don't intend to unless I have to.
