### Post by Linux&amp;Gsus on 2009-04-09
I only can tell you about K3B and only in theory since I never created a multi session disc.

When you open K3B click on "New Data CD Project". Prepare what you want to burn -> drag from the top file browser window to the bottom empty project field, then click on "Burn". A window pops up, go to tab "Misc". There in the 2nd pull down menu called "Multisession Mode" you choose what you want to do.

In the tab "Filesystem" you can give your CD a name that'll show up when you pop in a computer.
When you are happy with everything go back to tab "Writing" and hit the button "Burn".

Hope that helps. But as I said I never burned a multi session CD or DVD myself. So, I hope that's doing it like I assume it should.
