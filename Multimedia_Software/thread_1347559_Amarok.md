### Post by gordintoronto on 2009-12-12
If you run System/Administration/Synaptic Package Manager, go to Settings/Repositories, and make sure the "multiverse" is enabled.

Then search for "restricted" and select "Ubuntu Restricted Extras," it will install codecs you need for most music playback, and also for playing DVDs.
