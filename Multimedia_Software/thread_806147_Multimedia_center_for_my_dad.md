### Post by tamoneya on 2008-05-24
depending on how comfortable you are with building computers I would just build one myself.  There are many mini itx boards that are designed to be used as media centers and therefore have hdmi built into the board.  when all is said and done it will probably cost 600-800 dollars depending on the specs.  You will be able to do whatever sort of remote desktop stuff you want since it will be a full ubuntu install and it should be pretty seamless since he already uses ubuntu.  If you need help you could give me some basic specs and I could pull together a newegg wishlist for you.
