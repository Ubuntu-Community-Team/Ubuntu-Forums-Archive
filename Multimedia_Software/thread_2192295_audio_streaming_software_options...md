### Post by tgalati4 on 2013-12-09
*vlc* has streaming capability.  *icecast* has a learning curve, but once you spend some time with it, it is quite powerful.

Once the icecast streaming server is set up, you can control it from anywhere with one of several clients such as:

idjc - graphical shoutcast/icecast client

I've only briefly played with *mixx*, but it is basically a DJ-streaming application, which allows you to mix voice and mp3 tracks and then stream them in a live environment.

mixxx - Digital Disc Jockey Interface
