### Post by Javich on 2010-06-24
Hey fidelandche, there are 2 approaches to your issue; if you connect the headset using the 1/4" plugs (like the earphones ones) or if you do get a headset. If you use the 1/4 plug, and you have your soundcard working properly on skype, then any will do. If you get a USB one, you have to make sure of the following things:
- It works generally on gnome (i.e. reproduce music, as an mp3 and record from it's mic)
- Plug it in (and make sure it works) **before** you start skype
- Use the skype free testing call service to test it


I use skype with my USB headset on ubuntu 9.10 (32bit) and it's a cheap one I got on radioshack.
