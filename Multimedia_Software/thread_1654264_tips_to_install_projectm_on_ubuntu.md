### Post by shantiq on 2011-06-07
and i have just found out that with **clementine** player 

in synaptic   or ```
sudo apt-get install clementine
```

**and it installs ProjectM directly on your system**


find it after at  ```
/usr/share/projectM
```


which already boasts a marvellous artist info feature if you click tools/visualizatios ProjectM is there by default wow


but it list a fairly staid bunch of milk files


so to boost that i have added 

 quite a few more (attached)


to do that open the zip cd to folder

and from terminal go   ```
sudo cp *.milk /usr/share/projectM/presets
```

**or** remove all the old ones and replace with this new selection
your choice

to remove

```
sudo rm *.milk /usr/share/projectM/presets
```  **then** copy the new ones in



you end up with a choice of 416 with the added ones; many of those are funkier more up to date than the default bunch
