### Post by rcmn on 2009-05-26
Finally a compact Gui for VLC 1.0 RC.I could not wait any longer ,here howto get it .

-first remove any VLC that could be installed.

then add the following repository

```

deb http://ppa.launchpad.net/kow/ppa/ubuntu jaunty main
deb-src http://ppa.launchpad.net/kow/ppa/ubuntu jaunty main

```

 import the PPA key:

```

sudo apt-key adv --recv-keys --keyserver keyserver.ubuntu.com 0x0e494dbb2f021ac1

```

finally intall the newest version.

found here

[http://webupd8.blogspot.com/2009/05/install-vlc-10-rc-in-ubuntu-from-ppa.html]("http://webupd8.blogspot.com/2009/05/install-vlc-10-rc-in-ubuntu-from-ppa.html")
