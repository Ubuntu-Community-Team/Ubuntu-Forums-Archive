### Post by gudeco on 2011-11-30
Hi there, My advice would be for you to use FL Studio itself through Wine (which is a sort of MS Windows ambient emulator for Ubuntu). You can benefit from the information in this thread [http://ubuntuforums.org/showthread.php?t=1260057](http://ubuntuforums.org/showthread.php?t=1260057). I've heard of Ubuntu Studio and some other programs such as Rosegarden or Fasttracker (very old), but I've never tried them, so I can't really tell you about the advantages and disadvantages involved. The one thing I can say is that if you are already used to FL Studio's interface, there is no need to change (although I am a great freeware enthusiast as well). Actually, the cool thing about Wine is that you'll never have to ask questions like this again, you can just run any Windows program in Ubuntu.
