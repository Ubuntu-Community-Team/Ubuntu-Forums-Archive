### Post by starcraft.man on 2009-07-10
Ok, I just had a look at this. The problem is by default the export feature doesn't add an extension, right click on the exported file and put .m3u on the end (please make a copy in another folder first, just in case). Then just go file > Open, select the playlist with the m3u on the end and open. It will be in the main pane, save it out with a name and voila.

I admit, it isn't very good. Disappointed by this feature, I'll look into it. There's alternatives of course btw, amarok, songbird, rhythmbox (default) and more.
