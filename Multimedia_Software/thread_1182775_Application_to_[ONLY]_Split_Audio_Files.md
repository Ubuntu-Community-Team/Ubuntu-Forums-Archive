### Post by logos34 on 2009-06-09
**mp3splt** (+ **mp3splt-gtk**) for mp3 and ogg.  Excellent app.

**vcut** to split oggs. (included in vorbis-tools pkg)

**mp3directcut** (wine).  mp3 only, but displays file as a waveform 

for flac I think the only way is too expand into wav, use something like **wavbreaker**, than re-encode to flac

hope that helps
