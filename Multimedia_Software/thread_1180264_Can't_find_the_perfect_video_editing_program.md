### Post by RedRat on 2009-06-06
If you are coming from Windows and use Adobe elements for video, you might want to try Kino. Unfortunately, I don't know if there are any HD editors for Linux. Kino doesn't allow the importation of the HD video. However, it seems to be able to read those dv files though. I import the HD video from my camera into Elements on the Window side and then move the digital video (dv files) over to Linux side. Oddly, Kino can read those files.

### Post by Backharlow on 2009-06-07
Cinelerra can definitely do hd, and any other res you need. Make sure you are using Cinelerra CV (community version) which is available here [http://cinelerra.org/](http://cinelerra.org/)
They have their own repo which has updated packages for each Ubuntu. So, no building from source necessary anymore. It is pretty stable, and on par with something like Adobe Premiere for Final Cut Pro.. a few gripes not withstanding of course. I was shocked to see it doing nearly real-time opengl based effects on my crap intel with crap nvidia card. You can even throw already encoded stuff at it, i.e.mpeg.
