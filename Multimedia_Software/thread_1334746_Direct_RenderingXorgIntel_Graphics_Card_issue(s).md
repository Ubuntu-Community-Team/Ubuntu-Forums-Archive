### Post by lidex on 2009-11-22
Xorg is not relied on in later versions of ubuntu as HAL is supposed to auto-detect. My understanding is that in Jaunty if you add info to xorg.conf ubuntu will use it. Have seen a lot of issues on intel chipsets. Definitely not the best option for linux. Give the drivers from this repository a go and report back.
[http://www.webupd8.org/2009/05/graphic-video-drivers-ubuntu-repository.html]("http://www.webupd8.org/2009/05/graphic-video-drivers-ubuntu-repository.html")
