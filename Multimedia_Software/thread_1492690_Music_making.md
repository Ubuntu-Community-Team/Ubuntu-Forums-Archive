### Post by cchhrriiss121212 on 2010-05-25
To have a go at music production on the linux platform you should try LMMS, as it is simple to use out of the box. For serious use, you can start using the jack sound server with for example: Qtractor to handle midi and phasex as a synth. Do want to use a midi keyboard?
Check the Ubuntu Studio forum for helpful music posts.

### Post by robério1 on 2010-05-25
I think Nted fits rather well your requirements. It will need the installation of TiMidity plus, who will be responsible for the actual sound-making. You will have to chose it in the Nted menus as your MIDI-out; there are 4 ports, any of them will do. Both programs can be installed by the Ubuntu program installer.
Good luck!

If you want something *extremely* simple, you can try Tuxguitar.  It is geared towards guitar use because it shows tabs, but it also shows sheet music and I use it to mess around with all kinds of compositions.
