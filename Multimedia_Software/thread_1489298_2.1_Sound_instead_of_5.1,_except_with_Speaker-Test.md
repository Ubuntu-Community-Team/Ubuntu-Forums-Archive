Upgrade or fresh install?
For volume control make sure these packages are installed:
```
sudo apt-get install --reinstall gnome-media gnome-media-common
```
and right-click on panel>Add to panel> Indicator-applet

I would go for an alsa upgrade via the alsa-upgrade link in my sig.
