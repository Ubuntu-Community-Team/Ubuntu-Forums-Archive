### Post by howefield on 2016-02-09
I'd suggest some command line applications, tools like convert and mogrify will easily and quickly crop or resize an image. Very small learning curve :)

Problem with cropping an image using a command line tool I guess is knowing exactly where to "position" the crop -  so I'd use gthumb or gimp. But speaking of Shotwell, looks to me that you can tell it not to import photos and then disable the sidebar, so what you are left with is the viewer plus editing tools.

There is also at least one add-on for Thunderbird which will resize images but not too sure if it is any good or not, but it is an option.
