### Post by lovinglinux on 2011-06-17
Install [Flash-Aid](https://addons.mozilla.org/en-US/firefox/addon/flash-aid/) and run the extension wizard. It will detect installed flash plugins, remove them and install the beta 64bit from Adobe Labs. Additionally, it will apply some tweaks that should improve performance and fix common issues.
