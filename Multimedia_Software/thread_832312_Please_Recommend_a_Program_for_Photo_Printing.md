### Post by muecker on 2008-06-17
This so feels like an FAQ, but I'm going to post it anyway and maybe it will become an FAQ ;)  Would somebody please give me a recommendation for a decent program for printing photos in Ubuntu?  Here are some of the requirements:

[LIST]
[*]I need to select the photo tray, which rules out a few programs
[*]I need to select the correct size, which rules out F-Spot.
[*]Optionally it would be nice to print multiple photos on a page, either the same photo multiple times or multiple photos (two 5x7s, several wallet size, etc.).
[*]Optionally it should auto-rotate the images for the best fit on the page.
[*]I would prefer to not pay for it because I have a handful of working programs in Windows already (they give them away with every camera you buy).
[/LIST]

My printer is an HP Photosmart 8200, although I would like to see a generic list of apps.  The main reason I mention that is because when I print from F-Spot I can select the photo tray but it doesn't print the correct size and my printer cancels the print job.  I think I previously tried Gimp and Picasa and neither would let me specify the print tray.

Not that it should matter but my printer is attached to a D-Link wireless print server so I can print from anywhere in the house.

This is so easy to do from Windows but the last time I tried from Linux I gave up.  It seems that most programs are designed for viewing, editing or uploading to the web and printing is an after-thought if it works at all.
