### Post by redradish on 2009-05-05
Hey, I've also missed features like that of Winamp, and have been music-player-hopping for years, but I've finally found one that's a good enough replacement of winamp for me! - Listen

It has the option for music to be queued and not immediately played when double clicked on, and plays all my FLACs perfectly. Only downside to using it as apposed to something like Amarok is it's relatively new and so lacks some of the top end features, but the interface is amazing (pretty much just like winamp, but without movable pieces), and seems to be progressing nicely :-) Amped.

Check it out 
[http://www.listen-project.org/](http://www.listen-project.org/)
