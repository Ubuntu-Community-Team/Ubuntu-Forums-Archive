### Post by mc4man on 2008-09-04
3 ways
go places -> home folder -> edit -> preferences -> media

go right click on applications -> edit menus -> highlight preferences and enable file management. Then access that thru  preferences menu

While inserting media hold down shift button. In the popup choose banshee from list, it will now be default.
