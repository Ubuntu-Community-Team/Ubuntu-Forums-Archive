### Post by dniMretsaM on 2011-08-09
Well you can use just about any converter and have it only add non-MP3 files. The easiest thing (other than a BASH script, but I don't know how to do that for this) would be to install SoundKonverter and use it. Select Add Folder and add the folder with your music stored on it. After that a window will pop up asking you what media types you want to add and simply un-check MP3. That should do it.
