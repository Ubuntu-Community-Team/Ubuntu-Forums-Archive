Your advice works in principle but I modify the steps as follows:-
1) Upload the photo with the person for posting on the new background image to a website offering free removal of background on image.  There are many online websites offering this service, such for exmple: [www.remove.bg/](www.remove.bg/)
2) After removing the background, download the edited photo.
3) Copy the background image on LibreOffice Writer and adjust its size
4) Open the download photo on LibreOffice Writer
5) Adjust the size of the person and move it on the background image to the position you need.
6) Save the image as .pdf file or select it with screenshot and save the file as .png format.

That is all.
