### Post by vitke on 2011-10-27
Yes, it's disgusting that one has to spend so much time fixing trivial things. Anyway, here's a cure for this one (after [http://askubuntu.com/questions/55878/all-desktop-settings-are-messed-after-logout):](http://askubuntu.com/questions/55878/all-desktop-settings-are-messed-after-logout):) in System Settings -> System Administration -> Startup and Shutdown -> Session Management turn -> On login choose Start with an empty session.
