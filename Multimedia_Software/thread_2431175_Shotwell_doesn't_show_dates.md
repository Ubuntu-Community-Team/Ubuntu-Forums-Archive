### Post by Autodave on 2019-11-14
Upgrading from one release to the next often causes problems like this.  I would suggest that you only use the LTS (long term service) releases and stay away from these short term releases.  The short term releases should be considered as beta releases.

I would download the 19.10 release and put it onto a bootable medium and boot and run from that just to see if the dates appear then.  If they do, then you can install 19.10.  Otherwise, if it were me, and I didn't have any bleeding edge equipment that I had to have the latest and greatest release, I would install 18.04 and keep that until 20.04 is release in 6 months.
