### Post by gandaran on 2009-03-31
which ubuntu 8.10 32-bits or 64-bits?
list of things to check
install all system updates
update apt-get/synaptic package list (sudo apt-get update)
install only one adobe flash plugin, choose either apt-get/synaptic package flashplugin-nonfree or the .deb adobe-flashplugin from adobe.com
make sure you don't have any open source flash installed like gnash or swfdec.
if you have flashblock addon installed disable it or remove it.
