### Post by mc4man on 2011-10-16
Don't have openshot installed but maybe try either of these -
change your version of libavcodec-53 to libavcodec-extra-53, also update libavformat to the extra version
The extra versions of libavcodec depend on libxvidcore4, the normal don't.

or 
add medibuntu to your sources & update - section C in link below, 
[http://ubuntuforums.org/showthread.php?t=1117283](http://ubuntuforums.org/showthread.php?t=1117283)
medibuntu's packages also support aac encoding, the ubuntu versions don't if that is something you need

I'd do the ubuntu repo upgrade 1st. to ck. if that takes care, then medibuntu if desired

If you have synaptic installed just search libav
