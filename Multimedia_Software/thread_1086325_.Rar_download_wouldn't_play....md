### Post by ChrisMP1 on 2009-03-03
No, relabelling does not do anything. If it worked on that file, it must have been accidentally labelled a .rar file to start with. I believe that Totem (the music/movie player) does not look at the extension, so all you did by renaming the file was tell the system to open it with Totem. Try removing the extension altogether, then right-clicking on the file and going to 'Properties' to see what kind the file was.
