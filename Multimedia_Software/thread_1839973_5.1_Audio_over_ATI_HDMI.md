### Post by BicyclerBoy on 2011-09-06
You should be able to output multi-channel LPCM & AC3 DTS & the HDA variants..but it all depends on the receiver capabilities.

First discover the audio devices/codecs.
aplay -l
aplay -L
then try to output audio test noise to various devices..
e.g.
speaker-test -c 2 -r 48000 -D hw:1,n
where n could = [0 - 9]
Note: speaker-test does not always exit cleanly so close & re-open the terminal.

Then can discover the audio capabilities of the receiver in the /proc filesystem..

### Post by BicyclerBoy on 2011-09-14
speaker-test reports the alsa user-space libraries.
It is generally accepted that alsa 1.0.24 user-space is a good minimum.

You should post your ubuntu distro version & the output of:
aplay -l

There are some launchpad ppa that allow alsa upgrades. 
If you are running 10.04 then this ppa can be used:
[https://launchpad.net/~team-iquik/+archive/alsa?field.series_filter=lucid](https://launchpad.net/~team-iquik/+archive/alsa?field.series_filter=lucid)

When you run speaker-test, close & re-open the terminal to be sure it works.
