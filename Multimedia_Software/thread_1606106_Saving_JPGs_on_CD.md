### Post by coffeecat on 2010-10-26
> **indytim said:**
> 1. Launch the burning application (like k3b).

So long as the CDs are already blank, (i.e. not used CD-RWs) there's an even easier way, for which you won't need an app like k3b.

1 - Insert blank CD

2 - When the "You have just inserted a blank CD..." window pops up, accept the default "Open CD/DVD Creator" by clicking on OK. A CD/DVD creator folder will open.

3 - Drag and drop the JPEGs.

4 - Click on "Write to disc".

There is one possible problem with this, but I stand to be corrected. I believe this way you get the Rock Ridge extensions which means that the CD will no be readable in Windows, for which you need Joliet extensions. For that you will need a burning app.

**Edit:** I shall correct myself. From the help notes for the Nautilus CD/DVD creator:

> The filesystem written to the CD will be readable with long filenames on all recent operating systems. Both the Joliet and the Rock Ridge CD-ROM filesystem extensions are used.
