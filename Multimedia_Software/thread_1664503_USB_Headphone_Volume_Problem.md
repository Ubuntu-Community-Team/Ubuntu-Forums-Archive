### Post by veefwoar on 2011-01-11
I got a solution happening.

Go into console type 'alsamixer'

With headset plugged in

Hit F6 and select the USB headphone driver from the list

This will give you the controls for the volume. Make sure the "Speaker" column is turned up to full.
