### Post by mc4man on 2008-08-16
To play from a dvd ripped to file (VIDEO_TS folder), open vlc -> file ->  choose open directory, navigate to the VIDEO_TS folder and open that. (not the .vobs inside) see screen
If you don't have a VIDEO_TS folder than you'll need to rip differently.
If vlc open on the movie or episode then go navigation -> title -> dvd menu
