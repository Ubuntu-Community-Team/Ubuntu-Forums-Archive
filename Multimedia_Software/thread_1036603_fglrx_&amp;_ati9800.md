### Post by Melcar on 2009-01-11
I have a similar card but on a slightly slower system (AMD XP 1800+, 512MB RAM, 9800PRO 256MB).  It's running Xubuntu 8.10 with the open source driver, and despite 3D performance being about half as with fglrx (already did countless comparisons myself with several chips) it can still handle Compiz and games incredibly well.  Movies play smooth as well.  The only difference between the drivers that I noticed was 2D rendering (radeon is noticeably faster).  
I doubt it's the card that's giving you issues (provided the drivers are installed right).  1GB of RAM is more than enough to run a regular 32bit OS, so that shouldn't be an issue either.  The CPU most likely.
