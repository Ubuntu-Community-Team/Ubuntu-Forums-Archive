I think you already have a utility for recording if you look at Accessories > Sound & Video > Sound Recorder

but as well you can install nice utility from source, open a terminal and type:
```

sudo apt-get update
sudo apt-get install recordmydesktop gtk-recordmydesktop
```

this allows you to record audio+video of your desktop activity.
