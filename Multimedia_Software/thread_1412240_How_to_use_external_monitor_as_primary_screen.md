### Post by josh6847 on 2010-02-21
I am running Ubuntu 8.04 on a laptop. I have a 19" Acer monitor and in the past have successfully configured my xorg.conf to handle dual monitor setup with 1024x768 on the laptop and 1600x900 on the Acer monitor. What I have done now is run the laptop through a KVM switch. Due to this, I would like to set the Acer monitor as the primary monitor and just shut the laptop. How can I project my laptop screen to my Acer monitor so that I can use the 1600x900 resolution available rather than just cloning my laptop and using 1024x768? Thanks.

---

### Post by NoBugs! on 2010-03-29
You should be able to use both screens by going to Display Preferences, and setting both to the right resolution, unchecking "mirror screens". Then hold Alt while dragging the top/bottom panels to the bigger screen, then you can just use the laptop as second screen.
