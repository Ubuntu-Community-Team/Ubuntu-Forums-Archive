### Post by RFGunslinger on 2008-12-15
I've had the best support from nVidia cards, especially if you want to run Compiz desktop graphics (wobbly windows, desktop cube, etc).  ATI support tends to lag a bit for newer cards, I think because they rely on an open source driver.
