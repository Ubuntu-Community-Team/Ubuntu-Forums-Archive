### Post by lidex on 2010-07-23
Here's another tip. You can install search engine plug-ins in firefox for a large number of sites. When a plugin is selected in the search bar, highlighting and right-clicking text will give the option in context menu to search for that text in that search engine/site.
Plugins here:
[http://mycroft.mozdev.org/index.html](http://mycroft.mozdev.org/index.html)
Results for "ubuntu":
[http://mycroft.mozdev.org/search-engines.html?name=ubuntu](http://mycroft.mozdev.org/search-engines.html?name=ubuntu)

(Some sites will offer to install a search plugin when your are on the site. Simply click the search box dropdown. Ubuntu forums need to be added manually. What you want here is not the default ubuntu forums but the '3rd party-google' which searches ubuntu forums using google. )
