### Post by Tyche on 2008-06-18
Open NVIDIA X Server Settings (System -> Administration -> NVIDIA X Server Settings) and select X Server Display Configuration.  On the right, choose X screen.  That will show you the number of colors, and the depth.  The physical size of your screen is shown above (in my case 1440 X 900)

Hope this helps.
