### Post by sydbat on 2009-07-10
This question is for any jukebox player (Amarok, Banshee, etc).

I have been wondering if there is a way to create a playlist that only has xMB and goes through the entire library to randomize songs (this is to burn a CD or put onto an mp3 player). Although I am not a big fan of iTunes, it has this feature, it works well and it saves time trying to pick out 200+ songs to burn/transfer...at least the last time I used it in Windows over a year ago.

For example, I want to limit my playlist to 500 MB. I want it to be completely randomized. With iTunes, I chose the various options in a smart playlist and it was (mostly) good. 

In Banshee I can set up a 'smart' playlist with the "Limit to xMB" and "Random" options, but it only 'randomizes' in alphabetical order (the playlist starts with A and goes to B, but does randomize those artists).

Is there any way to have this "Limit to"/"Random" option look through the entire library (over 20 GB, A to Z) and truly generate a random playlist?
