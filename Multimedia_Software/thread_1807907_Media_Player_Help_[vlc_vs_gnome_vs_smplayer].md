### Post by beew on 2011-07-19
Install mplayer  (and gnome-mplayer if you want) from this ppa.

[https://launchpad.net/~ed10vi86/+archive/tst]("https://launchpad.net/%7Eed10vi86/+archive/tst")

or you can install mplayer  from this one.
[https://launchpad.net/~ripps818/+archive/coreavc]("https://launchpad.net/%7Eripps818/+archive/coreavc")

Don't worry about the coreavc stuffs, they have nothing to do with the player.

Use gnome-mplayer as the front end to play videos, it doesn't have a lot of options but uses least resources.
