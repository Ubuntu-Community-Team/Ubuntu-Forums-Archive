### Post by Tinman131 on 2009-07-04
Well, I would think that--if the background is the only thing blurry--that it's an issue with trying to fit the image onto the frame required for a desktop wallpaper. 

Have you tried using the same image(s) with different resolutions (1024 x 768, 1280 x 1024)?  The might not fit your widescreen display perfectly, but try to find the one that scales well enough (so it's not distorted) to see if the image is still blurry.

Also, what kind of monitor do you have?  Are these problems being experienced by other users / expressed in customer reviews?  I've got a Dell 15" LCD and, every so often, things go blurry and I have to "reset to factory defaults."  Even though only the background is blurry for you, it might still be worth a shot to change some settings, reset the default, et cetera, to see if this is just a problem with the image you're using, the resolution you want, or something else.

See what you find with these suggestions and we'll go from there.
