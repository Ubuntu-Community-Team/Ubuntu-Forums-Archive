### Post by tgalati4 on 2013-09-11
There is no best.  There is crappy and less crappy.  Most of the file issues have to do with tagging.  Use *easytag* to fix tags so more files will show up.

I would read the reviews on Twonky and see if it does serve files better than the free alternatives.  Sometimes you really do get what you pay for.

I use mediatomb and I put it in the less crappy category.
