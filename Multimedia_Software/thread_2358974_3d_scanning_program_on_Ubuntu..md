### Post by Bucky Ball on 2017-04-19
Welcome. Why would it? It is [a Windows program]("https://artecgroup.zendesk.com/hc/en-us/categories/200451942-Compatible-hardware"). Not sure it even runs on Mac.

Your only hope would be to use Wine (search for and check Artec Studio on 'WineHQ') or to run Windows as a virtual machine and install the app in that. You may have a few issues getting the hardware required for 3d scanning running in a virtual machine, though. If you do a web search for 'artec studio linux' you might be able to glean what success others have had with this.

My advice? Keep a Windows install and use that for Artec Studio or find a native Linux alternative, if there is one. Good luck.
