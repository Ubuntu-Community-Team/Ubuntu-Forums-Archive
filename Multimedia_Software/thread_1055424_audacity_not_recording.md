### Post by solaralchemy on 2009-02-02
Ok i suggest you try this it seems a really easy fix. I did so many differnt things like download a new sound coding but I would have to look up the tread I found it on. Even after the download audacity would not work. it had something to do with changing all sound to ALSA. Yet after the fuss it still was no go.
Here what I suggest it was the last thing I did and got Audacity going.
Bring up the volume control menu. In the right there is a tab called preferences. A  menu will appear move down and check the box next to CAPTURE. back in the main menu with the sliders a tab will apear called RECORDING click on it and it will give you an addtional volume slidder that allows you to control recording level.
Good luck if this works for you then everthing else I did before was pointless. If it does not work then ill try to recall the other things I did before this last step.
