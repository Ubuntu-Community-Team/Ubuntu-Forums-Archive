### Post by Norman IV on 2009-03-15
Well, I fixed this on my own.

 I FINALLY found a post with relevant info: 
[http://ubuntuforums.org/showthread.php?t=845784](http://ubuntuforums.org/showthread.php?t=845784)

 It did not exactly match my problem but one of the replies discussed editing the /etc/pulse/daemon.conf file and changing the defaultchannels from 2 to 6. This worked, but I was not looking for this information at first because the speaker test correctly identified each speaker (I guess it does not use this pulse audio configuration)
