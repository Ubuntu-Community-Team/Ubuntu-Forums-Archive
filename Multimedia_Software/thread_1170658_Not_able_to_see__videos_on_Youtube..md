Or, if you don't want to use the command line(some people just don't like it):
-Open Synaptic (System > Administration > Synaptic package manager)
-Look for the ubuntu-restricted-extras package, mark it for install
-Click "Apply Changes"
-Wait for the package to download and install and it's done.
