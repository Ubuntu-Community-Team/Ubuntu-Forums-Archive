### Post by antieuclid on 2009-05-07
I have an old (~2002) computer that I'd like to set up as a media computer for my grandfather and myself. The first part is relatively simple: what is a nice lightweight multimedia-PC software that is very easy to use? It needs to be able to connect to folders on other networked computers, but it doesn't need to be able to do anything with traditional TV. My grandfather has Alzheimer's, so the simpler the better when it comes to user interface.

My other question is a bit more complicated. Because of his Alzheimer's, my grandfather gets rather confused when dealing with my collection of .avi TV shows. He doesn't understand that he can watch any of them at any time, and he keeps waiting for a commercial break to do things like go to the bathroom or get himself a snack. So I was wondering if there's anything out there that would basically allow me to set up a playlist that worked like a TV channel. So that even when I'm not there with him, he could just turn it on and it would have shows he liked depending on the time of day. I've tried just having a very long random playlist on repeat, but he likes keeping track of what time programs are on, and he found it frustrating. And weird as it sounds, it would be nice to be able to add commercial breaks between programs. Probably not actual ads, but being able to put up a slideshow of jpegs with random trivia and the occasional reminder to use the restroom for a couple of minutes between episodes would be a big help. I know it's a long shot, but is there anything like that out there?
