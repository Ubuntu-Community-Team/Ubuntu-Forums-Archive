### Post by Frogs Hair on 2012-09-18
Snow was removed from the Compiz extra plugins. E17 has snow and rain modules but they are very different. You would need a an SVN PPA or an E17 distro like Bodhi Linux to get the modules working.The E17 packages in the repo are incomplete and outdated and Compiz won't work with E17.  [http://www.bodhilinux.com/](http://www.bodhilinux.com/)  

There are Gnome 2 distros like Scientific Linux but I don't know if an older versions Compiz and plugins are available for it.      [https://www.scientificlinux.org/news/SL6.3-live](https://www.scientificlinux.org/news/SL6.3-live)
