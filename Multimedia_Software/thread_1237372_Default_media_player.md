### Post by rainwalker on 2009-08-11
System > Preferences > File Management (you may have to open the menu editor and check the box so it shows up, do this by right-clicking on the menu bar and choosing "Edit menus")
Go to the "Media" tab and you can set what happens for a ton of different media types.

It used to be so much easier, but all the settings got moved to this "Media" tab a while back.
