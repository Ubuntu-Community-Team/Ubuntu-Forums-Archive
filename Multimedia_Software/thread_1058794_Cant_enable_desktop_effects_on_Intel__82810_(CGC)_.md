### Post by redroad55 on 2009-02-03
Hi, First go to applications > system > admin. > software sources >> third party tab check for the presence of the medibuntu repository ... If it is not go here..
 [https://help.ubuntu.com/community/Medibuntu#Adding%20the%20Repositories]("https://help.ubuntu.com/community/Medibuntu#Adding%20the%20Repositories")

now go to synaptic and check simple-ccsm click apply .. then go to applications > system > preferences > appearence >> under the effects tab you now will have a custom selection > select it and you should be presented with a driver recommendation > allow this download > Log out and log back in > 

Post back with results and or questions .. Best to you
