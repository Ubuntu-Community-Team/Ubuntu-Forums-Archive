### Post by ottosykora on 2010-12-26
you just simply download the driver zip file from the canon website.
Unzip it and install the .deb package same as any other deb files. 
The rest will happen itself.

The only thing is, you have to switch the small switch on the backsite to external driver, then it will become on usb scanner not memory stick. And then you can use sane or what ever from the repositories to scann and work on your documents.

you can find the driver pack here:

[http://de.software.canon-europe.com/software/0037654.asp?](http://de.software.canon-europe.com/software/0037654.asp?)


the file you want to install is called : cndrvsane-p150_1.00-0.2_i386.deb


otherwise you can find some instruction here:
[http://lowerstrata.blogspot.com/2010/07/canon-p-150-and-linux.html](http://lowerstrata.blogspot.com/2010/07/canon-p-150-and-linux.html)
