### Post by lykeion on 2011-02-24
You can use a sound editor like Audacity to create a MP3 file with 1 hour of silence like this:
[LIST=1]
[*]Start Audacity
[*]Select Tracks > Add New > Audio Track
[*]Select Generate > Silence... > Enter duration
[*]Select File > Export... > Enter name and select MP3 Files as type > Click Options... and set MP3 options > Click Save
[*]Metadata seems unnecessary, just click OK
[/LIST]

But frankly the 1 hour silence method doesn't seem to be the best. My advice is to ask this question in the Mediatomb forums: [http://sourceforge.net/projects/mediatomb/forums](http://sourceforge.net/projects/mediatomb/forums)
