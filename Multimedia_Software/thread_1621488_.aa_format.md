### Post by cchhrriiss121212 on 2010-11-14
You have a few options: WINE or virtualbox could help if you just want to play the file, or you could try to convert the file using another machine ([see here]("http://stream-recorder.com/forum/remove-drm-audibles-audio-books-removing-copy-t878.html?t=878")). 

The best option would be not to buy anything with DRM, if you were unaware of the problem this practice causes before buying the audiobook, contact audible and ask for a refund. This is the only way companies will learn that no-one wants DRM.
