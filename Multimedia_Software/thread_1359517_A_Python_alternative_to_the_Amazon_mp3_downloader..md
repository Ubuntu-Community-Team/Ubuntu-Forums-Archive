### Post by sccolbert on 2009-12-19
So, I wasn't happy with the Amazon mp3 downloader for linux (because it sucks). And clamz is written in C with a bunch of dependencies. 

Thus, I've created a python downloader for .amz files using the crypto keys figured out by Ben Moody (clamz author). 

Its just command line only right now, but I will add a pyqt front end in the future. 

The only external dependency is PyCrypto, which is in the ubuntu repos. 

I always appreciate feedback!

GPL licensed
[http://code.google.com/p/pymazon/](http://code.google.com/p/pymazon/)
