### Post by Isamu715 on 2012-11-28
Why do you want to compile the Intel drivers by hand?
You should not attempt to compile things and install manually unless you know exactly what you're doing.
If you need updated Intel drivers you should rather look at this PPA: [https://launchpad.net/~glasen/+archive/intel-driver](https://launchpad.net/~glasen/+archive/intel-driver)
To update your drivers use:
```
sudo apt-add-repository ppa:glasen/intel-driver
sudo apt-get update
sudo apt-get upgrade
```
