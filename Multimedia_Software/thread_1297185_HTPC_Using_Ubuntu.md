### Post by steefjeqv on 2009-10-24
If you like to watch TV with XBMC, you can combine Hepi's ppa with the VDR team's ppa :


Hepi :

[https://launchpad.net/~henningpingel/+archive/xbmc]("https://launchpad.net/~henningpingel/+archive/xbmc")

VDR team :

[https://launchpad.net/~the-vdr-team/+archive/vdr-ubuntu-karmic]("https://launchpad.net/~the-vdr-team/+archive/vdr-ubuntu-karmic")

Just install the vdr package and the stream-dev-server package.
Scheduling your recordings and so on can be done using a web interface,
vdr-plugin-live.
