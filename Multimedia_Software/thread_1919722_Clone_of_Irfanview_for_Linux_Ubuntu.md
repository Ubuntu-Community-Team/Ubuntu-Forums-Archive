### Post by ajgreeny on 2012-02-03
I used Irfan-View in my windows days, and apart from Gimp it was almost the only graphics application I ever used.  I agree totally it is a terrific application.  If you go to the IV website you will find some questions about a linux version, but the developer says (more or less) that it is not worth it as it runs so well under wine.

I have to admit that he is correct; even though it is 6 years since I used windows, so I don't really know how much IV has developed on that platform, I looked recently at IV-4.3 and it is still largely as I remember it; very good.  However, for my purposes it is no better than gthumb.  Have a look at that if you don't know it and it may surprise you.

If you still prefer IV download the windows setup executables and run them under wine to install it.   You will also need the winetricks package to install the mfc42.dll (I think that is the missing file), or if you run windows version as well you can simply copy that .dll file to your ~/.wine folder in the same sub-folder as it was in the windows install.
