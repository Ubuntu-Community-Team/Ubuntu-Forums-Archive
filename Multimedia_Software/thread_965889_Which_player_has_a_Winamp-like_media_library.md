### Post by echostorm on 2008-10-31
Amarok is ridiculously good. It takes a minute to get used to the browser like theme, but after that... its incredible. It sorts by artist/album, etc. you can click on an album to get information, art etc... it also will show song lyrics for you.
In addition, it supports streaming audio from streamtuner (configure streamtuner to use %amarok instead of %xmms)and has a milkdrop style plugin for visuals (a little buggy but hey)
I was turned off by it for a few weeks, ill admit... being a hardcore winamp user at the time, but after i learned to really use it, I never looked back :)

### Post by southerngrey on 2008-10-31
I am assuming that you are using the default gnome desktop?

After trying to solve the same issue I have ended up using Rythembox as my music player.  It's library management isn't quite as good as winamp but it's better than the other options I looked at.  After a short time I stopped worrying about it as I can always get at what I'm after pretty quickly.

A lot of people like Amarok []("http://amarok.kde.org") but it is for the KDE desktop rather than Gnome which comes installed default on Ubuntu.  There are other options available in the repositories but all of them have downsides which on balance makes Rythembox the best option for me.
