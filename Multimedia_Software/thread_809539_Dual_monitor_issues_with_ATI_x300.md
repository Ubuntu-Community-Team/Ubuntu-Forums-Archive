### Post by corbett.jim on 2008-06-12
Issue : Dual Monitor support on Ubuntu 8.04 for ATI card.

Keywords : Ubuntu 8.04, ATI Technologies Inc Radeon RV100 QY [Radeon 7000/VE] card, two Dell 1901 FP monitors, Open source driver xserver-xorg-video-ati 1:6.8.0-1

 
Hi, 

I ran into similar issues with the "Screen Resolution" tool, but I hope my post will assist you.

Note : I am a beginner (so please do not expect miracles here :-) ). 

STEP 1 : Driver Install / Check 

I have a ATI Technologies Inc Radeon RV100 QY [Radeon 7000/VE] card. I figured this out by running the command lspci on the terminal window.

Apparently, this card is supported by the ATI driver : xserver-xorg-video-ati 1:6.8.0-1 , which is installed by default on Ubuntu 8.04. It can also be installed by running the Synaptic Package Manager. 

See Reference : [http://xorg.freedesktop.org/releases/X11R6.7.0/doc/radeon.4.html](http://xorg.freedesktop.org/releases/X11R6.7.0/doc/radeon.4.html)

Caution Note : From this reference 
[https://help.ubuntu.com/community/BinaryDriverHowto/ATI](https://help.ubuntu.com/community/BinaryDriverHowto/ATI) 
I came to believe that the proprietary ATI fglrx drivers *might* not support my ATI RV card. I tried the process described in that link for Ubuntu 8.04 but was not successful.

STEP 2 : Modifying xorg.conf

In reality I played with my xorg.conf file to come to the attached solution.

1. Link that really helped me : [https://help.ubuntu.com/community/RadeonDriver](https://help.ubuntu.com/community/RadeonDriver) (Scroll down to 'Getting Dual-head to work' and read the 'XRandR 1.2' section. 

2. This will lead you to [http://wiki.debian.org/XStrikeForce/HowToRandR12](http://wiki.debian.org/XStrikeForce/HowToRandR12) .The sample xorg.conf file there helped me immensely.

My final xorg.conf file is attached below for inspection. Note my ATI card has a single digital output from which a cable kind of splits it into two DVI outputs called DVI-0 and DVI-1.

Hope this is of some help.

Thank you, 

Jim Corbett, India.
