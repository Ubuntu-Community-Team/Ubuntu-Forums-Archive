### Post by starcannon on 2008-09-10
get it
```
sudo apt-get install recordmydesktop
```

use it with gui (Applications>Sound & Video>gtk-recordMyDesktop)
```
sudo apt-get install gtk-recordmydesktop
```

learn cli usage 
```
man recordmydesktop
recordmydesktop --help
```

use it with cli
```
recordmydesktop
```

enjoy and have fun, its a great little utility.
