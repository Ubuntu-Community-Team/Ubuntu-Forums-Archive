### Post by little_penguin on 2009-10-04
The Logitech Quickcam Pro 9000 uses the universal video class UVC driver (v4l2).
It works in v4l2 based applications such as Amsn, Ekiga, Skype, Kopete, luvcview and guvcview, etc. See the following links for more info:

[https://help.ubuntu.com/community/UVC](https://help.ubuntu.com/community/UVC)
[http://linux-uvc.berlios.de/](http://linux-uvc.berlios.de/)
