### Post by papibe on 2011-04-10
AFAIK, neither VLC or mplayer have an option for that.

If you really want to do this there's a way. You can use tools like ffmpeg or mencoder to create a new file with one of the subs hardcoded on the video. Then you can use the remain sub as an external file.

I hope it helps.
