### Post by timcredible on 2008-09-19
plug your canon into the pc and see if it pops up with an icon on the desktop.  if not, check to see if it can be seen with digikam (auto-detect camera).  if not, change your usb mode on the camera and try again.  usually only 1 of the usb settings on the camera will work, but so far every digital camera i've tried works in linux (about 10 of them).

also, if you use raw file format, then digikam will open those files (dcraw plugin is automatically installed when you install digikam).  if you don't like digikam with dcraw, then you can use rawtherapee, or gimp with dcraw or ufraw plugin.
