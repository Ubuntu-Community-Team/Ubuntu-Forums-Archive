### Post by tgalati4 on 2010-03-21
M-Audio products are generally well-supported in linux.  If you are doing a mixboard recording, then why would you need more than 2-3 tracks?

If you are doing a full-on studio recording, then yes, you would need multitracking of each instrument/vocal.  But for live recording, I use a Denon CD recorder.  I rip the tracks to WAV files then use Audacity for track splitting, fades, Some EQ, and any other processing required.

For recording 2-hour church concerts, I use both a Denon CD recorder and an M-Audio Delta 66 with a binaural microphone that records straight to wave file with Dynabolic live CD (dynabolic.org).  

Trying to record 8 tracks of live sound for later mixdown will give you headaches.  Ideally, you would have 2 mixboards, one for live sound reinforcement, and one for just recording mix.  You set the levels of the recording mix with decent, isolating headphones.  Then you only need a 2-channel M-Audio or PreSonus USB dongle or even your onboard line-in (if a Mac).  The Mac's Wolfsen codecs are OK for live recording.
