### Post by tamoneya on 2008-06-16
sometimes the sites themselves work fine under linux just the site developers dont support the platform and choose to lock linux users out.  This is often accomplished by checking your browsers user agent which identifies the OS and browser (Firefox with linux).  If this is the case you can easily get around this with user agent switching(pretend to be IE on windows): 
[https://addons.mozilla.org/en-US/firefox/addon/59](https://addons.mozilla.org/en-US/firefox/addon/59)

However if it is actually a plugin incompatability error this will have no effect.  Depends on the situation.  One solution that always works however is to install virtual  box or VMWare and run windows XP in an emulated environment.  Personally this is always my last choice since I cant stand the windows operating system,
