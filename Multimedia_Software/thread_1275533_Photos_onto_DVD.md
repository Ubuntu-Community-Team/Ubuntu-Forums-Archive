### Post by cotcot on 2009-09-26
No need to compile. It written in java. Download the tar. Extract it (right click on the tar and select extract here). Go in the folder. Open terminal and start it with ```
java -jar Jslideshow.jar
```

I was curious and have tried it. It starts and opens a window. Everything works untill I want to render the whole thing to a video. It then stops with a failure announcement that it could not find checkdependencies (although imagemagick and ffmpeg are installed on my box).
