### Post by lovinglinux on 2011-01-07
First, get [Flash-Aid]("https://addons.mozilla.org/en-US/firefox/addon/161939/") and run it. It will detect installed flash plugins, remove them and install the best option according to your system architecture and version. Additionally, it will apply some tweaks that should fix your problems. If you still get that issue after running Flash-Aid, then try to disable Compiz (effects) and check if it works.

BTW, are you using dual monitors?
