### Post by Robbyx on 2018-11-01
If you are no longer able to update your Harmony remote via the old harmony remote web site and you use congruity and concordance in firefox Quantum:
The author of Congruity advised me to try installing the 'Ignore X-Frame-Options Header' add-on in Firefox and see if that fixes the problem. It worked for me.

Here is a more official  link to the same point:

 [https://github.com/jaymzh/concordance/pull/18](https://github.com/jaymzh/concordance/pull/18)
