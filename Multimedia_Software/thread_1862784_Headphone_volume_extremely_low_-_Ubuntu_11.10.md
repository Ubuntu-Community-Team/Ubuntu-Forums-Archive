It seems as though Ubuntu 11.10 has fixed nearly every problem I had in 11.04, except it has introduced one small problem...

Every time I plug in my headphones in the jack, the volume for "<speaker>" in alsamixer gets set to 0 and as a result the volume for any music, etc. is extremely low. This can only be fixed by setting the volume for "<speaker>" to 100% again manually in alsamixer as using the volume control in the Ubuntu sound menu does nothing. This happens every time I plug in the headphones, so if I plug them in and change the volume in alsamixer and unplug and plug them back in again the problem reappears. Is there a way to fix this and/or have a script that sets the volume in alsamixer anytime headphones are plugged into the computer?
