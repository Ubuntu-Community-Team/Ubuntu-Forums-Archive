### Post by tommcd on 2011-02-01
Sandisk *ipod*??? Sandisk does not have a mp3 player named ipod as far as I know. If they did it would likely invite a lawsuit from Apple since Apple owns the ipod name.
What Sandisk model do you have?
Sandisk does make mp3 players. I have a Sandisk Sansa Fuze that works perfectly in linux. On my Fuze player you have to set it to MSC mode in the settings options for it to work on linux. So try setting your player to MSC mode if you can and see if it is recognized in Ubuntu.

Write back if you need more help.
