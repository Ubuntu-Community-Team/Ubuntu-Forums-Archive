### Post by Rumplesmigskin on 2008-06-26
I've been sniffing around for alternative video conferencing utilities for Linux. I found one interesting-looking candidate [here]("http://tcpcam.org"). Written completely in C, only really depends on SDL and libjpeg. Usage is also very simple: it simply binds to an IP address and a port. Only requires one user's port to be accessible to the WWW, which is always a plus. It seems to be a fairly recent program so try it out. I think it's great, but then again I'm weird like that. Check the website out to learn more about it.

There's also [VIC]("http://mediatools.cs.ucl.ac.uk/nets/mmedia/"), which is available in the repositories. It's in a similar vein but much more comple^H^H^H^Hnfigurable. :)
