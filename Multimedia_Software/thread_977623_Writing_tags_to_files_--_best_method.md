### Post by le_vainqueur on 2008-11-10
In XP I have been using Media Monkey to write my tags to mp3 files.  This is the best program I have found for this purpose.  It has an intuitive browser, and it properly writes the tags to the file so that they are the same regardless of the program or device the music is played on.  It also has a very nice album art search tool.

Today I discovered that the 6G iPod classic works in Ubuntu, so I want to be able to edit tags without going to XP.  I have tried Ex Falso, but the file browser is tedious at best.  Also, there does not seem to be album art support.  I have used Easy Tag in the past, and as far as I can remember, it was roughly the same.  Is there a better way to edit tags that I am missing?  What would be ideal is if they could be written within a media player in the same way as Media Monkey does it.
