### Post by SeijiSensei on 2014-02-06
If you're not averse to the command prompt, then both **ffmpeg** and **mencoder** can do this rather easily.  The exact syntax would depend on how the source video is encoded and what formats you wish to use for the output file.  You can find lots of help for these programs by searching Google.
