### Post by Fully216 on 2009-06-16
GIMP is a program that you can use to manipulate raster images.  Since I am guess you are not working with vector art, it will be hard to scale it to such a large size without it looking distorted regardless of what image manipulation program you use (ubuntu or windows alike).

You best bet would be to convert the image to a vector image (inkscape) and then scale it to the size you want.  If the image in complicated, this can be a difficult process.  Maybe a sample of the image might be helpful to see how easily this can be done.  VectorMagic is a website that does this for you, FYI.
