### Post by beew on 2011-11-22
Why do you want to do it the hard way? You can install the Nvidia driver by clicking "Additional Drivers" and it will download and install the correct version for your hardware. After that open a terminal and type
```

sudo nvidia-xconfig
``` and reboot and that's it.

You will have to reinstall the driver each time you have a kernel upgrade if you download from Nvidia and install it manually.

If you want up to date version you can just add the x-swat ppa and upgrade. 
[https://launchpad.net/~ubuntu-x-swat/+archive/x-updates](https://launchpad.net/~ubuntu-x-swat/+archive/x-updates)
Right now their version in 285.05, but the new stable version 290.10 will be in it probably in a week if not a few days.
