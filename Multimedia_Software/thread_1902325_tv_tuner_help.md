### Post by linuxisgoed on 2012-01-23
I recently bought a second hand pci tv card on ebay for £25 and it works great with mythtv.  It's called a "wintv-nova-td-500-dvb-t" model number : 84109LF made by hauppauge. Make sure you get the correct card for the correct tv signal you are trying to view on you pc.  If you have a satellite dish set up and want to watch freesat in the UK you need a dvb-s pci card or usb stick. 

If you only have an aerial on your rooftop like me then you need a dvb-t  (t is for terrestrial tv)  The duel cards enable you to record two channels at the same time or watch one and record another at the same time- can come in handy. 

Make sure you don't buy an old analogue card if that broadcasting method is being phased in your area.  
If you want to watch programs in HD then you will need to spend a bit more cash.  This link give a good selection of pci cards/ usb tuners that work with mythTV  in ubuntu.
[http://parker1.co.uk/mythtv_novafw.php](http://parker1.co.uk/mythtv_novafw.php)
or search for 'recommended tuner cards for mythTV
I'm using ubuntu 10.10 64 bit
