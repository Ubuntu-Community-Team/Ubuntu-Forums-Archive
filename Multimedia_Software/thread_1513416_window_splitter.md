### Post by xzero1 on 2010-06-20
Might not be what you need, but you can use GnomeShell  to view multiple workspaces. See [http://live.gnome.org/GnomeShell/Screenshots]("http://live.gnome.org/GnomeShell/Screenshots") for screenshots.

Though it has not been released (expect a few bugs), it might be useful.

To install on Lucid:
```
sudo apt-get install gnome-shell
```  (this will not be the latest version but it is the easiest to install)

To run:
```
gnome-shell --replace
```

To kill it hit Controll-C in the terminal running gnome-shell.

See [http://live.gnome.org/GnomeShell]("http://live.gnome.org/GnomeShell") for more info.
