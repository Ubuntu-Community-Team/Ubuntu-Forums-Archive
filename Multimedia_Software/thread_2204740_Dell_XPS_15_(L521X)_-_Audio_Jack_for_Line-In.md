### Post by weber-jonathan on 2014-02-10
Hello!

Since this is a hardware question I think the issue is limited to my system-configuration. I hope you can help me anyway:

I have a Dell XPS 15 (late 2012, "L521X"). It has an integrated soundcard with internal Mic, internal Speakers, a headphone-jack and a headset-jack (both according to the icons). On both jacks I can receive audio when I plug in my headphones.

Since I have a Tascam USB Audio-Interface, which is not working for recording in Ubuntu, I want to connect it (workaround) via Line-In to my Laptop.

The problem is: I can find no way to change the headset-jack (or at least one of them) from Line-Out to Line-In. I tried via the normal Audio Configuration (in Systemsettings), I tried Alsamixer and I even tried the tool "hda-jack-retask".
 
I am sure there must by any way to achieve this, since no Laptop would be shipped with two identical Line-Out jacks. That would not make much sense ;-)

1. Update to latest sound module/driver: [https://wiki.ubuntu.com/Audio/UpgradingAlsa/DKMS](https://wiki.ubuntu.com/Audio/UpgradingAlsa/DKMS)

2. Add this line to /etc/modprobe.d/alsa-base.conf
```
options snd-hda-intel model=dell-headset-multi
```

3. Reload alsa
```
sudo alsa force-reload
pulseaudio -k
```

Success?
