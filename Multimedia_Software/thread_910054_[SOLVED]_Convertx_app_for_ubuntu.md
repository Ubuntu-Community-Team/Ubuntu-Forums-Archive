### Post by annatar on 2008-09-04
Actually..you can run covertx under Ubuntu using WINE.
WINE is basically a piece of software that allows you to run windows apps in Linux.
Find and install it via synaptic, or use this command :
```
 sudo apt-get install wine
```

In case that (older) version of WINE doesn't work, see here for the latest:
[http://www.winehq.org/site/download-deb](http://www.winehq.org/site/download-deb)

After that, download and install covertx as usual.
