### Post by Yellow Pasque on 2014-01-20
Were you editing that file (/etc/pulse/daemon.conf) for some reason?
It's supposed to be enable-lf**e**-remixing.
```
gksu gedit /etc/pulse/daemon.conf
```
Find that line and change the 't' to an 'e'
Save. Quit. Log out and back in. If you're still having an issue, make a new log file:
```
rm ~/pulseverbose.log
LANG=C pulseaudio -vvvv --log-time=1 > ~/pulseverbose.log 2>&1
```
