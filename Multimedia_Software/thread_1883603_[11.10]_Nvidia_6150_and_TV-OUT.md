### Post by emagine1 on 2011-11-20
Hi Guys...

I Finally Solved the TV-OUT issue with my Nvidia 6150 (M2NPV-VM Motherboard). :guitar::lolflag:

Same as Windows this GPU doen't support Clone Mode (I Think). Then only one display is displayed at a time.

_**Relevants Links: **_
Driver Manual: [ftp://download.nvidia.com/XFree86/Linux-x86_64/280.13/README/index.html](ftp://download.nvidia.com/XFree86/Linux-x86_64/280.13/README/index.html)
Configuring TwinView: [ftp://download.nvidia.com/XFree86/Linux-x86_64/280.13/README/configtwinview.html](ftp://download.nvidia.com/XFree86/Linux-x86_64/280.13/README/configtwinview.html)
Configuring TV-Out: [ftp://download.nvidia.com/XFree86/Linux-x86_64/280.13/README/configtvout.html](ftp://download.nvidia.com/XFree86/Linux-x86_64/280.13/README/configtvout.html)
Option supported: [ftp://download.nvidia.com/XFree86/Linux-x86_64/280.13/README/xconfigoptions.html](ftp://download.nvidia.com/XFree86/Linux-x86_64/280.13/README/xconfigoptions.html)

I am using [current-version] of Nvidia Driver (v280.13).

_**To change of Monitor to TV (supports Full HD, but I like of HD size): **_
- Open Nvida X Server Settings (with normal user)
- Go in X Server Display Configuration
[COLOR=Red]**- In  Model, select your monitor (If not already selected) and in Resolution select off**[/COLOR]
- In  Model, select your TV and select auto or the resolution desired.
- Click in Apply.

_**To change of TV to Monitor: **_
- Open Nvida X Server Settings (with normal user)
- Go in X Server Display Configuration
[COLOR=Red]**- In  Model, select your TV (If not already selected) and in Resolution select off**[/COLOR]
- In  Model, select your Monitor and select auto or the resolution desired.
- Click in Apply.

**[COLOR=Red]I highlighted in red the third line, because if not done the image will not appear either in TV or Monitor[/COLOR]**
