### Post by mcduck on 2010-04-15
.mp4 is just a video container like all the others. If you get any fancy extra features then that's done by whatever program you are using (in this case the media center software you are running), and isn't in any way related to the video files themselves.

I'm not absolutely sure what is it that you want to do, but there are several applications available that are used to create DVD disks with menus, which of course allows you to use such video clips (or other icons) to start the actual videos.
