### Post by trickytrickytricky on 2008-12-24
I know of two programs that are supposed to allow you to edit video files in the same way windows movie maker does. They are "Open Movie Editor" and "Pitivi" You can install either one, or both, using the synaptics package manager. Just type the name of the one you want in the search field and it will find it for you. The package manager will install the program under Applications, Sound & Video.

Having said all that I am having problems getting them to work properly so I'd be interested to see how you get on with them.

With Open Movie Editor I find that as soon as I try to preview clips on the timeline the whole application closes and remembers nothing. As for Pitivi I think it's designed primarily for Fedora (I may be corrected) and on my ubuntu installation the program seems unstable. I can add clips to the timeline but not preview it and after a few attempts the program freezes.

All this could be down to something wrong with my installation so if you do try them let me know how you get on.

### Post by WinterWeaver on 2008-12-24
Most video editing solutions for Linux are very unstable in my experience. I've tried almost every app out there, and all of them had issues with crashing, or missing features etc.

The only one I've found at this point that does descent editing is Blender 3D. It's in the ubuntu repos, so you can install it via Applications >> Add/Remove.

It has a video Sequence Editor, but since blender has a different workflow from most other apps, it may take some time to get used to it. If you google "Blender3d Sequence Editor" I'm sure you will find some descent tutorials.

Good Luck

WW

Sorry, but games, color photo printing, and video editing are reasons you still need the abality to dual boot into windows.

Progress is being made, color photo printing is OK now, but only if you have the "right" printer.

Video editing on Linux is about at the level of what it was on Windows when Windows 95 came out -- if you have the same hardware the developers use, it usually would work if you futz around enough, if not, you sat there hoping for a miracle fix in the next upgrade.  At least with Linux you've not spent triple figures of money on buying the software initially and getting hosed over and over again buying upgrades.

On Windows XP Sony Vegas Movie Studio is pretty inexpensive and works very well on P4 or better.
