The answer to most of your questions is no. The GIMP does not provide the same features as Photoshop, and it certainly does not support Photoshop plugins. There is some overlap in features, but they are fundamentally different applications.

You can download legal 30-day trial versions of Photoshop (which may not function in Ubuntu via WINE) to compare and contrast the applications yourself.

I'm sorry to say this, but if you are doing a course which involves learning Photoshop skills, you cannot follow the course without using Photoshop itself. Both applications share common concepts (such as Layers), but the implementation of these concepts, as well as the workflow of each application, are often very different. You're just going to confuse yourself if you try to substitute Photoshop while doing that course.

### Post by mcduck on 2009-05-24
You can do exactly the same things with Gimp as you can do with Photoshop, but no, Gimp definitely is not Photoshop with tools renamed.. :D

If you are good with Photoshop, you'll have no problem working with Gimp. On the other hand, if your Photoshop skills rely heavily on easy single-task tools introduced in latest Photoshop versions (extract tool and such) and you don't know how the same things are done with basic photo manipulation techniques you'll have some troubles.

If you were going to take a Photoshop class you'd definitely want to use Photoshop. But you said it's actually HTML course, so you should be quite fine with Gimp if you have any basic photo manipulation skills.

For HDR photography I can only but recommend Qtpfsgui. Great HDR editor. I use it on both Linux & Mac. :)

### Post by hozzy on 2009-06-30
As you can see, Gimp and photoshop are not the same. I used photoshop for a while, got pretty good at it, then I found myself traveling with a computer that had ubuntu on it. Gimp was my only option. Now I use gimp and it works well.

### Post by bobince on 2009-07-01
> Alright I will be taking a basics to HTML course here in the fall and need to learn photoshop.Gah. This is what's wrong with so many of these courses.

Photoshop is a high-end creative tool for professional users, specialising in retouching high-resolution photographic material. It is not at all suitable as a simple bitmap editor for beginner web authors. There's enough to cope with learning HTML, CSS and JavaScript without having to pile the complex feature-set of PS on top.

But as &#8216;Photoshop&#8217; has become the generic verb for image editing, everyone is convinced they have to learn and teach it as the only image editing tool that exists, alas.

It's better to understand the principles behind working with bitmaps than it is to be tied to one particular tool. (Personally I'm not a big fan of either PS or Gimp, but I can certainly get by using either.) But that probably doesn't help you if your course is going to be testing for how well you remember where each button is on the Photoshop toolbar.
