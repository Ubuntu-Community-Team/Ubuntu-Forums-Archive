### Post by cchhrriiss121212 on 2011-02-27
> My expectations are high so I need something that will do the same (if not more) as Cubase 5.1.
Don't expect to find exact equivalents of professional apps that won't cost you anything, they don't exist. Linux audio production is a capable platform but it requires a lot of time to adjust for a Windows native user.

It would help if you explain exactly what features you need to have, as well as how much time you are prepared to spend setting things up and learning how things work.

If you are attatched to VSTs for example you will not be happy as they have limited functionality on Linux. That won't change anytime soon as VSTs are a proprietary standard. Linux does has it's own set of plugins: LV2, DDSI and LADSPA.

If you are still curious, the first thing you should do is install and configure jackd, then I'd recommend [Qtractor]("http://qtractor.sourceforge.net/qtractor-index.html") and possibly Hydrogen.

> ubuntu lacks the gadgets found in the studio version and could possibly be a pain to get it rite on your current setup
That is not exactly true, Studio is just the same as regular Ubuntu with extra packages installed.
