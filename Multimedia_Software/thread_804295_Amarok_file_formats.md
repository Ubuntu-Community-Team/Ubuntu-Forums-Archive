### Post by Sef on 2008-06-05
Out of the [Audio Format Support]("http://en.wikipedia.org/wiki/Comparison_of_media_players"), Amorak supports all of these except APE, and Musepack and AC3 are partially supported.

MP3 	WMA 	RealAudio 	Vorbis 	Musepack 	AAC 	AC3 	APE 	FLAC 	ALAC 	SHN 	WV

It does not support DRM.
