### Post by cottfcfan on 2012-06-07
On the other side of the coin, I have had compilation albums scattered all over my Clementine library.
Again by using exfalso & highlighting all the tracks in the Album, you can then add the tag "compilation" and set the value to "1".
After doing a full library rescan, Clementine then pulls all the tracks together in one Album:)
