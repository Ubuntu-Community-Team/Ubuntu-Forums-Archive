Hello Mohamed_Saber, and welcome to the forum.

I will start with a recommendation that you update or reinstall with a supported version of Ubuntu as 15.10 has lost support and it will get no further updates of software or security utilities.

However, even if you install a supported version, either 14.04 or 16.04, it will be very helpful to know the hardware you are using.  
Please tell us the CPU, amount of RAM, graphics card and if this is a laptop using wifi, the wifi adapter will be useful all which you can find with command ```
sudo lshw >hw.txt
``` then add the hw.txt file produced in your home as an attachment using the paperclip icon in the toolbar of the "Reply to Thread" reply box.
