### Post by pjd99 on 2011-08-07
Are Rhythmbox and Banshee available in the lubuntu repos? Both decent gtk/gstreamer players, but heavier than Audacious. Amarok is another, but requires KDE libraries.

XMMS2 is lightweight, though the interface is very much like Winamp.

I wouldn't go the wine option either. Although IMO, WMP is among the worst pieces of software on a default windows install. I wish I could purge it from my Win7 partition... Media Player Classic FTW!

### Post by SalHelder on 2011-08-08
You also have QuodLibet (which has plenty of interfaces that you can chose from), Listen and Exaile (among dozens of others). But if you are looking for something really cute and shinny don't expect it to be lightweight, you could try [Songbird]("http://www.getdeb.net/software/Songbird") but it's heavier than Rhythmbox, Banshee or Amarok. Because you are using Lubuntu I don't think you should use Songbird.

Anyway Audacious is themable go to View > Interface > Winamp Classical Interface.
