### Post by Diabolis on 2008-07-27
You need to install a client, graphic user interface, there are many.

```
sudo apt-get install esperanza
```
or
```
sudo apt-get install abraca
```
or
```
sudo apt-get install gxmms2
```

more about it: [http://wiki.xmms2.xmms.se/wiki/Clients](http://wiki.xmms2.xmms.se/wiki/Clients)

If play is not working through the command line, is probably because you have to specify a play list first.
