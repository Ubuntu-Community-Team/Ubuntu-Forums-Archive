### Post by Existentialist on 2008-12-15
The tag files are stored as a part of the individual music file.  The tags will not be lost unless you remove them or erase the song.

For future reference, there are programs that make tagging a little quicker.  I recently started using [[COLOR="Blue"]_Picard _[/COLOR]]("http://musicbrainz.org/doc/PicardTagger") to organize my tags, and rewrite the file names.
