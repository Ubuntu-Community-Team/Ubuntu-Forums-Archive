### Post by silkworm2.5 on 2011-05-08
If you are in the standard gnome desktop (two panels, one at the top of the screen one at the bottom) then go into the system menu, administration, then system monitor. Click on the system tab in this program to see all the info on your Ubuntu distribution. 

If you have only one panel and a bar at the side where your programs are listed, then click the button in the upper left, and type in "System Monitor" and click the first result. This is the same program mentioned above, and the details are in the same place.

### Post by mikewhatever on 2011-05-08
All versions of *buntu are Debian distributions, so that, if you want to download the installation file, look for .deb. That said, the Ubuntu way of installing Adobe Flash would be to open the Software Center, search for 'flash', select the appropriate package and click Install.
