Also, I don't really like its current embedded media players.

To get to the point: what I'm looking for are suggestions as to what system would work best to:

[LIST]

[/LIST]

All easily navigable using a remote and ideally quick to boot up.

### Post by m.rankovic on 2008-07-04
[http://www.entertainer-project.com/index.php](http://www.entertainer-project.com/index.php)

[quote="entertainer-project.com"]Entertainer aims to be a simple and easy-to-use media center solution for the Gnome and XFce desktop environments. Entertainer is written completely in Python using an object-oriented programming philosophy. It uses GStreamer's multimedia framework for multimedia playback. The user interface is implemented with the Clutter library, which allows sleek OpenGL animated user interfaces. Entertainer also uses other great projects like SQLite, pyIMDBb and iNotify.

The ultimate goal of this project is to create the best media center solution available for any platform. This means that Entertainer should be the best looking, most easy to use and most feature filled media center solution available. Not the easiest goal to achieve but we are working on it. [/quote]
