### Post by paulg on 2008-09-15
In this case you're better off with the NVidia card. I had read that there are some proprietary IP in the X1### ATI cards that is preventing those cards (i.e. X1200, X1900) from benefiting from the new driver improvements. The main benefactors are the current generations ATI cards.

You're not seeing recent comparisons because those cards are circa 2006 :) For the record I have a system with a 7600GT and a system with an X800pro. Both run reasonably well. The X800pro is a lot improved in the last 3 years. The 7600GT is more stable than the ATI card. I also understand that the X### series ATI cards have better driver support than the X1### cards.
