### Post by Precipitous on 2008-12-09
You can install the Last.fm client through Synaptic. As for setting up Rhythmbox, do the following:

[LIST=1]
[*]Go to the Edit menu and select Plugins
[*]Put a check-mark next to the Last.fm plugin, then click the Configure button
[*]Enter the user name and password for your Last.fm account
[*]Click Close (twice)
[*]Re-start Rhythmbox
[/LIST]
If you are still not able to get it to work, you might try using BMPX instead. It can be installed via Synaptic, and is (in my opinion) much better than Rhythmbox anyway. In fact, I think it has the best Last.fm integration out of all of the Linux audio apps I have ever tried.
