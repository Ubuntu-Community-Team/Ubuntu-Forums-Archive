### Post by Spice002 on 2012-08-21
Did you install ubuntu-restricted-extras? That contains the codecs and libraries needed to render MKV files. To install it, type in the terminal:
```
sudo apt-get install ubuntu-restricted-extras
```
For more information on it, refer to this page:
[https://help.ubuntu.com/community/RestrictedFormats/](https://help.ubuntu.com/community/RestrictedFormats/)
