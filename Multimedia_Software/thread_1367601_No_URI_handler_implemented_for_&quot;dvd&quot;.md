### Post by grgzfla on 2010-01-01
I had the same problem.  After some poking around, I found myself at the following link...

[https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)

Not only did it describe what the problem was, following the instructions to install the ubuntu-restricted-extras package and run the .../libdvdread4/install-css.sh script allowed me to watch "Kelly's Heroes" flawlessly!

Enjoy!
