### Post by opitzs on 2011-02-23
Install VLC (a very good media player), then go to Media>Open Network Stream and enter as stream address:
[http://mediaweb.musicradio.com/Show.asx?StreamID=2](http://mediaweb.musicradio.com/Show.asx?StreamID=2)

That is from the classic fm faq.
