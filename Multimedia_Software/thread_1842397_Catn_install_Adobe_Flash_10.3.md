### Post by Enigmapond on 2011-09-11
Very easy. Just install the Firefox add-on Flash-aid which was developed by one of our members. Run the script and install either the stable or the 11beta...it will install and uninstall anything that conflicts. Good Luck!

[https://addons.mozilla.org/en-US/firefox/addon/flash-aid/](https://addons.mozilla.org/en-US/firefox/addon/flash-aid/)
