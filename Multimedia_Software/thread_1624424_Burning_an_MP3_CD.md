### Post by eddier on 2010-11-24
Try WinFF.

Drag and drop all the mp3 files onto it and select wav as the output format.
Select convert and let it do its stuff.
 
Then do the same for all the flac files,again select wav as the output format.

Gather the resulting wavs together in one folder.

Use Brasero to burn the wavs to a cd.
