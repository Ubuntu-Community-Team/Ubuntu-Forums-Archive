### Post by llelectronics on 2012-01-08
You could try installing the application **handbrake** which should be able to rip your dvds to Mp4 H264. 
There are a lot of Youtube clips which explain very easily how to achieve this with handbrake if you have any trouble with it. 
(i.e. [http://www.youtube.com/watch?v=AadiU_3fGRg](http://www.youtube.com/watch?v=AadiU_3fGRg) )

If you recorder your DVD+RW with a DVD-Recorder (I mean those ones that you plug in TVs) you might need to fixiate the DVD+RW first before it can be read by your computer. (Consult your DVD-Recorder manual to get the details on how to do this)
