### Post by ad_267 on 2008-09-06
Banshee stores it's settings in ~/.config/banshee so they're still there when you reinstall it. Open the file browser and show hidden files by pressing Ctrl+H and then browse to the .config directory and delete the banshee directory.

Or from a terminal:
rm -rf ~/.config/banshee
