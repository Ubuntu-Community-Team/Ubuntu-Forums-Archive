### Post by fisheater on 2010-03-06
Thanks for the email, this one is solved.

Solution: easytag [HTML]http://easytag.sourceforge.net/[/HTML]

1. Select folder with MP3s in it.
2. Select all.
3. Change to Podcast (not in dropdown list, type it in).
4. Select save, then yes to save to all.
5. Enjoy the fruits of your labour.
