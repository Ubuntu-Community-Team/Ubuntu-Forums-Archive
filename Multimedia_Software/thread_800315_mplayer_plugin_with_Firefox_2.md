### Post by cor2y on 2008-05-19
You have two choices.
1) Save yourself some headache install mplayer with firefox3 support, for the missing plugins you can edit them by manually downloading the xpi files open with fileroller and edit the rdf files inside from firefox 2 to firefox 3 and they will then work in firefox 3.
2) Compile the mplayer mozilla plugin yourself you will needa  bunch of development files all available in synaptic., or look for a mplayer mozilla plugin deb that doesn't depend on firefox3 but only that firefox or mozilla browser is installed you might find that at the debian package archives
[http://packages.debian.org/mozilla-mplayer](http://packages.debian.org/mozilla-mplayer) (try the one in etch)

I know its a bit of a drag but those are the only ways i can think of
