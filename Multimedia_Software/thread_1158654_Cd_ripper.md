### Post by namd3r on 2009-05-13
It will if you have the right packages installed.

To check, open the program and go to Edit -> Preferences

At the bottom it should have a dropdown box with the available output formats, like ogg, mp2, or mp3 if it is available.

If mp3 isn't an option, you will have to install gstreamer0.10-plugins-ugly and gstreamer0.10-plugins-ugly-multiverse via Synaptic or apt-get.



Edit: before it will work, you will have to restart Sound Juicer/Audio CD Extractor.
