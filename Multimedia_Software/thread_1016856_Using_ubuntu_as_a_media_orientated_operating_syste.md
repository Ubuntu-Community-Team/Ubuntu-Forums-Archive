### Post by Macintosh SE on 2009-01-30
I actually did something similar with an old Dell PC, that was given to me with a Windows virus. Naturally, I zeroed the disk and installed Ubuntu. I have it set up with a wireless Microsoft keyboard and mouse (given to me after it stopped cooperating with Windows computers), so I can't help you with regard to controllers. However, my setup is similar to what you describe. I set it to login automatically (System > Administration > Login Window > Security > Enable Automatic Login). The computer plays videos on my TV via VLC, and opens them in fullscreen by setting an option in VLC preferences. You could probably do something similar with music. Although my setup is not as pretty as Mythbuntu or something similar, it was easy to set up.
