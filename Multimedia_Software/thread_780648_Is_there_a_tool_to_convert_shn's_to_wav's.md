### Post by AndrewB on 2008-05-04
There is a tool called shn2wav. you will need it to do the conversions.

Maybe start here:[Google search]("http://www.google.com/search?q=shn2wav&ie=utf-8&oe=utf-8&aq=t&rls=com.ubuntu:en-US:official&client=firefox-a")

My notes from using the tool:
> Navigate to the directory with your shn files and...
[folder with SHN's must be directly in home folder]

Code:

$ shn2wav

Then you will (hopefully) have a directory full of wav files, which you can then input into the burning app of your choice.

Hope this helps...
