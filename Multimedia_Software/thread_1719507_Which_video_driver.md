### Post by fela on 2011-04-01
You need to install the proprietary driver with the 'recommended' flag (ie. the current one), to get optimal performance out of your graphics card. E.g, 3D graphics acceleration is not possible without it (I don't think anyway).

### Post by wolfen69 on 2011-04-01
It depends. If you already have the correct resolution and are not going to need 3D acceleration for games and things like Google Earth, then you probably won't need any of them. But if you do need it, I would go with the recommended one.
