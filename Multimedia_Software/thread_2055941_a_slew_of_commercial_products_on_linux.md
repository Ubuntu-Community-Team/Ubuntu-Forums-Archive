Unfortunately, no, the Adobe Creative Suite for Mac will not work on Linux. You could transfer the license to Windows and run it in a virtual machine. Wine is another option, not a good one though.

> **jellyAM said:**
> 
i also use zbrush a fair amount and im wondering what some of the equivalent open source applications would be and what kind of experience people have had trying to run zbrush on wine if they've tried?


It looks like Zbrush will run with Wine but those ratings are not always accurate, especially for niche applications with fewer users. Show stopping bugs can go unnoticed despite the "platinum" rating which implies perfection.

[http://appdb.winehq.org/objectManager.php?sClass=application&iId=1152](http://appdb.winehq.org/objectManager.php?sClass=application&iId=1152)

Blender has some sculpting features but not near as advanced or polished as Zbrush. There are commercial products like Mudbox that have native Linux versions.

> **jellyAM said:**
> 
lastly the final two apps i find myself using all the time are ableton and reason im just wondering what people's experience on winewith those two app on wine have been?


Looks like Ableton runs with Wine, same thing as before though. There might be show stopping bugs lurking.

[http://appdb.winehq.org/objectManager.php?sClass=application&iId=2113](http://appdb.winehq.org/objectManager.php?sClass=application&iId=2113)

Generally speaking I wouldn't rely on Wine for day to day production if you can avoid it. Some applications really do run just fine with Wine but those are few and far between.
