### Post by ajgreeny on 2012-07-15
Perhaps you need to find why vlc is causing you these problems, but to close all vlc windows that are running use terminal command ```
killall vlc
``` or to just kill the errant window use command ```
xkill
``` and then put the cursor over the window and left click.
