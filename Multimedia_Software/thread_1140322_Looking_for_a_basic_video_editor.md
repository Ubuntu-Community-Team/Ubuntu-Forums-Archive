### Post by pytheas22 on 2009-04-27
[Cinelerra]("http://cinelerra.org/") has a pretty intuitive interface for basic tasks.  It also offers very advanced functionality if you learn to use it.  It's not in the normal Ubuntu repositories, but there are instructions on its website for downloading it.

VLC and avidemux also offer some very simple editing tools, but you may need more than that.
