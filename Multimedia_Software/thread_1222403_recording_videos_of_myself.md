### Post by enz1m3 on 2009-07-26
The default ubuntu app for that is Cheese, which is very nice. Just install from synaptic: "cheese" or in the console type: "sudo apt-get install cheese".

Here's some more info about webcams on ubuntu: [https://help.ubuntu.com/community/Webcam](https://help.ubuntu.com/community/Webcam)

I haven't had any problems with webcams yet in Ubuntu (I have laptop integrated ones and Creative ZX or something like that), but if you prefer, in this thread there are two webcams known to work out-of-the box: [http://ubuntuforums.org/showthread.php?t=275901](http://ubuntuforums.org/showthread.php?t=275901)
