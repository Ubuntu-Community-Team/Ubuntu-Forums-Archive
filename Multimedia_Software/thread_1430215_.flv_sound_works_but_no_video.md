### Post by mikemelen on 2010-03-15
open the system >> Administration >> synaptic manager there select swfdec-gnome and apply for installation. 

 *Swfdec*-*Gnome* provides tools to integrate Flash into the GNOME desktop. It contains a standalone Flash player.
