### Post by papibe on 2011-11-01
Hi timpiele.

Using a VGA cable should be fine. In order to set it as an external monitor use 'Monitors' (System -> Preferences -> Monitors)

Although most HDTV have VGA inputs and they support computer resolutions, make sure that the TV has some basic features to control the image (horizontal and vertical position for example).

Hope it helps,
Regards.
