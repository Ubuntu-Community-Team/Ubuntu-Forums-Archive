### Post by ajgreeny on 2018-02-20
It may be easier to use the ppa repository for LO, details of which are at [https://launchpad.net/~libreoffice/+archive/ubuntu/ppa](https://launchpad.net/~libreoffice/+archive/ubuntu/ppa)

I've been using this ppa for several years now with no problems for me, and I feel it is one of the most trustworthy ppas around, so no real need to fear it.

I have no idea about the quickstarter, I'm afraid as I never used it, but LO 6 does not offer it as an option and as far as I can see form LO 5.4.4.2. in 18.04 neither did that version.  I don't have any earlier versions to look at so I'm not sure when it disappeared, or even if it has gone now, but I did find one reference to it now being deprecated.
