### Post by junglejuice on 2008-08-29
ok ok so i can totally understand why no one was interested in helping me out with this one. 
it seems like i was i was able to fix it myself
all i did was
regenerate the xorg.conf file. obviously i lost some of my confirgured stuff doing this, but i havn't noticed any new problems as of yet. anywayz openGL apps are working fine again. here's the steps i took
in terminal run this command
```
sudo dpkg-reconfigure -phigh xserver-xorg
```
this rebuilds your xorg.conf file sets them back to defaults
then go to hardware drivers manager in the system group and enable hardware acceleration.
presto i can play openGL games again and use Blender hope it works for you too.
