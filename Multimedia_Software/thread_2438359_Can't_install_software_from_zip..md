### Post by mc4man on 2020-03-11
Inside your extracted folder is the synthesizer-v-editor file.
Right click on Properties > Permissions > check Allow executing file.. box, close properties.
Open a terminal, drag & drop that file into terminal, click on terminal to return focus, press enter on keyboard.
