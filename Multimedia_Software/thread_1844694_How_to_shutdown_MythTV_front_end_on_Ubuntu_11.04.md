### Post by jcobban on 2011-09-15
I cannot figure out how to shutdown the MythTV front end on Ubuntu 11.04.  This may be a Unity interaction problem.

MythTV has its own peculiar user interface which does not appear to cooperate with the window manager.  It does not respond to any of the usual commands to shut down.  When I hit ESC enough times a dialog pops up with two options, one of which is to shut down, but I cannot seem to select that option using the keyboard, even though it appears to be highlighted.  And of course because MythTV refuses to cooperate with the window manager I cannot use either the normal mouse operations to close the window, nor any of the usual alt or ctrl key shortcuts.

Why has nobody ever thought to wrap a standard GUI around this product?

I finally gave up and pushed the power button.  It was the only way I could get this program to shut down!
