### Post by TheFu on 2014-08-29
For screen captures there are many tools.  I use Imagemagick and the "import" tool for screen capture. There are probably GUI tools for that too. Not my style.

I suspect you really want video capture.  ffmpeg, avconv, recordmydesktop, and a few other options come to mind.  Nixie Pixel makes youtube videos using some tool that I can't recall. [http://www.youtube.com/watch?v=EjbtlzOY7S4](http://www.youtube.com/watch?v=EjbtlzOY7S4)

It is possible to mux audio recorded with **audacity** with video later using most video editors like **avidemux** later.
