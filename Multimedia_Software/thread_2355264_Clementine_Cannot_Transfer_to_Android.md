### Post by TheFu on 2017-03-11
Clementine is a player application according to my reading.  It isn't a media server/streamer.

There is a remote control clementine for Android, but it requires v1.3+ of Clementine on a desktop.
[https://play.google.com/store/apps/details?id=de.qspool.clementineremote&hl=en](https://play.google.com/store/apps/details?id=de.qspool.clementineremote&hl=en) says it is a remote for v1.3+ of Clementine. v1.2.3 isn't new enough.

Remote streaming is a feature request in the clementine bug tracker:
[https://github.com/clementine-player/Android-Remote/issues/20](https://github.com/clementine-player/Android-Remote/issues/20) 

Or am I completely missing something?

### Post by SeijiSensei on 2017-03-11
I just connected my phone (an LG Stylo) and copied all my music files to the Music directory in Android.  Actually, since I have an SD card, I put the Music folder on that.

USB connections to Android can be hit or miss in Ubuntu.  If you're having problems, you can browse the various threads here, or you can choose another method.  I use Wifi File Transfer Pro, an Android app which runs a web server on the phone.  Then you can connect to it with a browser on Linux and move files back and forth.  I paid $2.99 for it, but it was worth the money.
