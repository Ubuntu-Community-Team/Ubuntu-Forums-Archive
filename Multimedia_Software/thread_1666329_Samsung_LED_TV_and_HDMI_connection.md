### Post by mtbdrew on 2011-01-13
Just thought I might save someone some time if they are trying to use Mythbuntu with a Samsung LED TV.

If you run into the issue where the desktop is larger than the screen even with everything set to 1920x1080i and you tried a customized EDID file to no avail. Go into the menu on the TV and select Source List, Edit Name, then chose the HDMI and change the name to be either PC, DVI or DVI-PC. Either of these three setting should correct the over sized display issue.

Now if you happen to want audio over HDMI and are using the HDMI Computer connector that Samsung so nicely provides. Make sure you don't choose either of the DVI options above as these mute the audio coming over the HDMI connector and instead enable the pin audio jack next to the HDMI connector.

### Post by papibe on 2011-01-13
It's probably a TV setting. Most TVs are set to do a mild zoom on the signal they receive. It's very annoying and it seems inherit from analog practices.

Use your remote and look for a setting to change how the signal is being scan. In my Sony Bravia it's under the Display menu and needs to be changed to "full pixel". In my brother's Samsumg (if I remember correctly) is in the Picture -> More -> Picture Options -> Screen Size. There, try all options, but I think "Just Scan" is the good one.

Let us know how it goes.
