### Post by peachcolor on 2009-12-10
This is a lyrics plugin for Rhythmbox. Lyrics is downloaded and displayed automatically. 

**Project homepage**: install instructions and FAQ could be found here.
[http://code.google.com/p/sogou-lyrics/](http://code.google.com/p/sogou-lyrics/)

**Feature**:
[LIST=1]
[*]download lyrics from various sources
[*]load local lyrics automatically
[*]display lyrics embedded in the interface
[*]display lyrics using gnome-osd
[*]display lyrics in a auto scroll window, press left button and drag will move the window, press right button and drag will resize the window, press middle button and drag will locate lyrics in roller mode
[*]open lyrics file using system default editor
[*]pop choose lyrics dialog while no accurate lyrics found
[*]edit lyrics and download lyrics manually in context menu
[*]totally configurable through plugin manager
[*]keep up with updates, discover yourself
[/LIST]
