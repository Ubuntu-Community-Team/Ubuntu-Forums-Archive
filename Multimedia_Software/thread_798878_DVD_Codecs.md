### Post by gandaran on 2008-05-18
you do need all of these files installed, check in synaptic if all the gstreamer plugins are installed, if not install all, good, bad, ugly, ffmpeg & etc. libdvdcss2 is for playing encrypted DVDs. the gstreamer plugins are for the totem player, if you are planing to install another player, like mplayer (which I recommend) you'll also need to install the w32codecs.
