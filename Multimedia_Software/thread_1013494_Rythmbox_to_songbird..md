### Post by chrisamiller on 2008-12-17
I don't know of any scripts to do that, but if you've got an itch to scratch and a little coding experience, you could probably do it yourself.

- For starters, all of your info is saved in the file ~/.gnome2/rhythmbox/rhythmdb.xml

- iTunes also uses an XML file, so I'd probably start by looking at the source code for the iTunes->Songbird import extension.  It's probable that you can tweak it to read the rythmbox file instead
