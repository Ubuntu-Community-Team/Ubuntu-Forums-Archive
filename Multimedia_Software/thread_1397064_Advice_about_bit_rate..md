### Post by VertexPusher on 2010-02-03
> **BT1 said:**
> I was wondering, what bitrate should I copy them in? I forget if a CD's bit rate is 128-196 (or whatever), so I ask, is copying in a bit rate over 200 really beneficial? Does it decrease the chance for loss to copy in a higher bit rate? I have 320b copies of some songs, but this question has nagged me for some time.
Bitrate requirements vary a lot, even within the same song. That's why you should **never** encode at a constant bitrate. Many years ago CBR encoding was necessary because some MP3 players could not handle variable bitrate files. But this is a thing of the past.

In variable bitrate encoding mode, you'll be able to choose a quality setting, and the encoding software will allocate just as much space as required to retain that quality at all times. The local bitrate in a VBR file can be anything between 32kbps and 320kbps, it will go up and down to accommodate the complexity of the source material. Even if the result has the same size as a 128kbps CBR file, it will sound much better than that because the bitrate *distribution* in the VBR file is totally different.

> Also, what format should I rip them in? For reference, I use the songs for burning to CDs for my car, to listen to from my Sansa mp3 player, and also for listening to at home in a higher quality audio setup. I don't know much about .ogg formats compared to mp3 or anything, so I hand these questions up to you audio pros.
FLAC is great for archiving because it is lossless, but for everything else I would choose MP3 because it is the most widely supported format.
