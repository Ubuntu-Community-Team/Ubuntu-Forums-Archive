### Post by captaintrav on 2010-11-01
Go to the output devices tab and change the default output device to the internal audio by right-clicking on the name and checking "default".  I'm almost positive from the looks of that screenshot that it's trying to use an HDMI device as the default sound output.  I'm assuming you're not using HDMI audio.  You'll have to restart whatever program you're using after changing the default device.  On the playback tab, you should be able to move the audio stream between devices, as well. 

If you're SURE that the sound is being played back on the proper device, you might want to give a shot at updating alsa using the script in this thread: [http://ubuntuforums.org/showthread.php?t=1046137](http://ubuntuforums.org/showthread.php?t=1046137)

As far as the codec name, I'm not sure about that as ALC887 is not even mentioned on realtek's site.  It's possible the version of the motherboard you have doesn't actually have 883 as the spec sheet says, I wouldn't worry about it unless you examine the motherboard and see what's silkscreened on that tiny little chip.
