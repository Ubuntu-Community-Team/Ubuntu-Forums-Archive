### Post by TUBEHTPC on 2010-09-27
The following steps (designed for Ubuntu OS menus) solved my HDMI Audio problem on my Ubuntu OS, but should do the job on every Linux OS using ALSA Sound driver.

Furthermore, this a simple solution, using simple steps, suitable for new users (like myself) of the Linux distributions.
I hope it will be helpful.

1. enter ubuntu software center
2. install GNOME ALSA Mixer
3. enter GNOME ALSA Mixer
4. mark the empty IEC958 checkbox as well
5. reboot
6. enter System / Prefernces / Sound
7. choose Hardware tab
8. choose profile: Digital Stereo (HDMI) Output
