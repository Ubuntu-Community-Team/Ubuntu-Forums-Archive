### Post by Mahdi D on 2013-02-06
Hello there, Yeah, I know the way! Open Terminal ( CTRL + ALT + T )

run this command:

```
alsamixer
```

then, go to Front Mic, press ' C ' untill it's 0, and you wont hear your own sound anymore, that's how I fixed this.

Hope it helps, Regards!
