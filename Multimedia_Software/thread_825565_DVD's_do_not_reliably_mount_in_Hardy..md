In windows they mount fine.  In ubuntu, Mepis 7.0 and other distros using more recent kernels I cannot mount these and other normal dvd-rw's. These are not proprietary movie dvd's but mainly blanks on which I want to burn data.  The mini DVD's contain a .vro file, a video file the camera creates, that mounted fine in Mepis and windows.

### Post by natrone on 2008-07-25
I found the problem not to be Ubuntu but the actual recording device.  When I have finished recording to the disk I "finalize" it in the camcorder.  So far since I have done this I have not had any problem mounting the DVD's.  I hope this helps.
