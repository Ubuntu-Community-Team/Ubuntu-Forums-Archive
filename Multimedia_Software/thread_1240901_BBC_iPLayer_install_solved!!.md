### Post by dougalkerr on 2009-08-15
I accidentally found a link for installing the BBC iPlayer Desktop Player. I was having bother like a lot of Linux users and was searching for a solution when I stumbled upon a topic on a Google link. This led me to the BBC iPlayer download and install page. This is different to the simple player link which shows the Adobe Flash Player version needed for Linux, etc. I initially followed this and successfully installed Adobe Flash Player and reinstalled and reinstalled, etc because when I went back to try to play a program I was always getting the 'Download Adobe Flash Player to view this content' (or something) even after installing the Adobe Player. If you follow the instructions for installing the player to the Desktop, all problems seem to disappear. I did get a window saying something about content not being available or such but, I just closed it with no further problems.
I have listened to Radio and I am watching a documentary just now... great.
I am running Ubuntu Satanic Edition 9.04 on a Dell Studio 1537 laptop with 4Gb ram and an ATI HD3450 graphics card with 256 Mb ram with Compiz Window manager running with only the ever present window resize slow speed as the only issue but, this doesn't stop me from doing anything so far... I hope this solves a lot of problems for many peeps. I have also written to the BBC asking them to make the Desktop player installation link as noticeable as possible.
