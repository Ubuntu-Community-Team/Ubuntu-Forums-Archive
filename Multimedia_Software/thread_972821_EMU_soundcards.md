### Post by neufena on 2008-11-11
In case anyone is interested I fixed this, you need to install 'alsa-firmware' from Medibuntu ([https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)) then it works perfectly.
