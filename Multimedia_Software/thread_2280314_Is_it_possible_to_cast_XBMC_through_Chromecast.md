### Post by TheFu on 2015-05-29
Don't think so. "Casting" is done either through DLNA or the Chrome browser. With chrome as a browser and plex as the server, there is a web interface to all the media on a plex server. I don't know if XBMC has that capability. Never used it, if it does, that could be easier to use tonight.

I added a plex server to my XBMC/Kodi machine and wish I'd done it sooner.  The work you did to setup XBMC is NOT wasted. Plex has a similar requirement for TV and Movie organization. Plex takes 5 min to install with a web interface to set things up. I ran both on low power Atom-like box for longer than a year.

And if you do go with Plex, know that you don't need to pay for anything or even sign up for an account to use it on your LAN. The Plex management URL is http://{IP}:32400/manage/index.html - after you install it using a .deb package, it should startup - point a browser at that and setup media and other minor stuff.

Plex is the best DLNA server I've used.
