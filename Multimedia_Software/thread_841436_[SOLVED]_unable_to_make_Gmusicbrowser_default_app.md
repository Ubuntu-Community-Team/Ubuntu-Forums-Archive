default app for what? mp3 files?

If it is for mp3 files  then find an mp3 file, right click on it, choose properties and the open with tab. Then click the add button and choose Gmusicbrowser if it is listed there. If it is not listed there then you can write "gmusicbrowser" at the custom command box at the buttom and then add it.
