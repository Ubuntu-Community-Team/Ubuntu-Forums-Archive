### Post by Melcar on 2008-07-11
Hardy is probably loading vesa instead.  Make your "Device" section look like this:

```
Section "Device"
	Identifier	"Configured Video Device"
        Driver          "ati"
        Option          "EnablePageFlip"    "on"
        Option          "GARTSize"          "32"
EndSection


```

The EnablePageFlip option boosts 3D performance a bit (and God knows you need all the speed boosts you can get with this driver), but it may cause instability on some cards (it works fine on my 9800pro however).  The GARTSize option you may need if you play games or do other heavy 3D stuff, since the default value the driver uses is 8 and it may sometimes crash when it runs out of GART memory; don't bother setting it any higher than 32 (32 is the max anyway) since that will only eat up your system memory... also, this value is **not** the same as your GPU memory (many people confuse this sometimes).

Edit:  The glxgears thing is normal.  It's just telling you that it terminated unexpectedly (because you closed it).
