### Post by xtremethegreat1 on 2010-02-15
The software you are installing through the terminal, comes from the ubuntu repository. This is because the software there is made compatible with ubuntu first and then provided to us users. This is why, the software in the ubuntu repository is always a few versions behind that of the original site.

The testing takes some time, because of which a software cannot be added up in the repository right away.
