I did a little Googling but couldn't find any specific specs on either.  Which is the better quality?


<=== EDIT ===>

### Post by mike3244 on 2010-01-01
AC3 is Dolby Digital surround, DTS is an alternative surround system. LPCM is uncompressed stereo. It would appear that the software you are using doesn't do anything with the audio but re multplexes it back into the transcoded video. I would stick with the AC3 if I were you.
