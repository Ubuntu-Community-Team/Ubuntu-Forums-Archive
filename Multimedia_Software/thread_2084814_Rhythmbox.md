### Post by evilsoup on 2012-11-16
From [here]("http://www.webupd8.org/2012/10/rhythmbox-298-released-install-it-in.html"), in a terminal:
```
sudo add-apt-repository ppa:webupd8team/rhythmbox 
sudo apt-get update 
sudo apt-get install rhythmbox
```

IMO, you should only do this if you have a specific need for Rhythmbox.
