### Post by niyonk on 2008-07-27
Go to Applications->Add or remove Applications
In the search box type acidrip or k9copy.

k9copy compares to dvd shrink if you read in the program's description.
And i think it also has an option to back-up as an ISO.
Try both programs but i recommend acidrip more if you want to rip into AVIs or other formats.

Hope it helps :)
