I've been having a lot of problems with this Ubuntu Kylin. In this post I want to ask for answers for this one particular problem.

I will then test the disc on Windows computers, to make sure the disc can be used on Windows too. Something very strange then happens. All the MP3 recordings on the disc work just fine on Windows, usually. But lately, there is a particular recording, I would burn it onto the DVD using my Ubuntu, and, when I then insert the disc into a Windows computer, the MP3 file will show up on the Windows simply as "file" (a blank icon) and cannot be played at all. and yet, when I try the disc on Ubuntu, the MP3 file can be played without problem. I then burn this recording for a second time, onto a second disc, using my Ubuntu, and the same thing happens: when the disc is inserted into a Windows computer, the MP3 file shows up only as "file" (blank icon), and can't be played at all. But when the disc is entered into Ubuntu, the MP3 file shows up as MP3, and can be played without problem.

Why? And why only this file, not the other MP3 files?
