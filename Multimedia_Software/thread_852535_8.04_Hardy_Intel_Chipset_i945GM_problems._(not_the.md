I have experienced that huge font on the log on screen. It has something to do with the monitor you are using.

Laptop screen is fine.
At home I plug my laptop into my 19" screen and it's fine.

At university they have some Acer 17" models and when I use them the font for the username and password entry box is massive, one letter takes the whole box.

My guess is the monitor may be giving you the wrong dot per inch value. I have no idea how to fix it though.

About the Intel graphics drivers. There are 2 as far as I know. the "i810" and the "intel". i810 is old and obsolete (didn't support widescreen among other things), replaced by the newer intel one so you will be using that.

About your WoW problems, have you tried to see if any other games suffer the same glitches.

Some other fixes for the Intel 945GM is to disable FramebufferCompression (fixes problems with external monitors), add XAANoOffscreenPixmaps for dynamic touchpad configuration and virtual screen size for dualscreen in Compiz (max virtual size is limited to 2048x2048 with Compiz). You can also force the "intel" driver too in case it used the "i810" instead. eg:
```
Section "InputDevice"
	Identifier	"Generic Keyboard"
	Driver		"kbd"
	Option		"XkbRules"	"xorg"
	Option		"XkbModel"	"pc105"
	Option		"XkbLayout"	"us"
	Option		"XkbVariant"	"euro"
	Option		"XkbOptions"	"lv3:ralt_switch"
EndSection

Section "InputDevice"
	Identifier	"Configured Mouse"
	Driver		"mouse"
	Option		"CorePointer"
EndSection

Section "InputDevice"
	Identifier	"Synaptics Touchpad"
	Driver		"synaptics"
	Option		"SendCoreEvents"	"true"
	Option		"Device"		"/dev/psaux"
	Option		"Protocol"		"auto-dev"
	Option		"HorizEdgeScroll"	"0"
	Option		"SHMConfig"		"True"
EndSection

Section "Device"
	Identifier	"Configured Video Device"
	**Option		"FramebufferCompression"	"off"**
	**Driver		"intel"**
	Option 		"AccelMethod" 			"XAA"
	**Option 		"XAANoOffscreenPixmaps" 	"true"**
EndSection

Section "Monitor"
	Identifier	"Configured Monitor"
EndSection

Section "Screen"
	Identifier	"Default Screen"
	Monitor		"Configured Monitor"
	Device		"Configured Video Device"
	[B]SubSection "Display"
		Virtual		2048 2048
	EndSubSection[/B]
EndSection

Section "ServerLayout"
	Identifier	"Default Layout"
	Screen		"Default Screen"
	InputDevice	"Synaptics Touchpad"
EndSection
```


Sorry my post is of very little help for your actual problems but try out other games like Alien Arena (based off Quake 3 engine). If it's WoW specific then you can post it in the Wine section where people know more about it.
