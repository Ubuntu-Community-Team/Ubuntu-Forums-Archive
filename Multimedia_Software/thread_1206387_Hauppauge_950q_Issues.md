w_scan defaults to ***European DVB-T***, whereas your device states to be American(not only, but most probably..) ATSC.

pls read the fine manual and **use the correct commandline switch for ATSC frontends** and the commandline switches for the needed **output format**.

### Post by Donewithunix on 2009-10-04
Try the new firmware dvb-fe-xc5000-1.6.114.fw which you can read about at the bottom of this page:

[http://linuxtv.org/wiki/index.php/Xceive_XC5000](http://linuxtv.org/wiki/index.php/Xceive_XC5000)

Worked for a friend who is running ubuntu 9.04 after downloading and installing the latest v4l-dvb drivers using these directions:

[http://linuxtv.org/wiki/index.php/How_to_Obtain,_Build_and_Install_V4L-DVB_Device_Drivers](http://linuxtv.org/wiki/index.php/How_to_Obtain,_Build_and_Install_V4L-DVB_Device_Drivers)

Then used w_scan to generate a channels.dvb file for kaffeine (over the air using the antenna that came with the tuner):

w_scan -fa -A1 -k > ~/.kde/share/apps/kaffeine/channels.dvb
