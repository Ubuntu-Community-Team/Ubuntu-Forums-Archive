### Post by ArmenianLeader4 on 2009-07-31
You can use Add / Remove to download new codec packages if you have even limited internet, but you can use the Application called "Movie Player" to play most files, and if that STILL doesnt work, you can download RealPlayer 11 from the website for free. just save it to your home folder and unpack it with the terminal. You don't even need wine.
