I have used a combination of kino, kdenlive, open shot, to do a lot of the simpler stuff I need. I have also installed cinelerra a much more robust package. Cinelerra has a repo version in one of the deb derivative linux OS's called 'cinelerra-cv' but it is restricted in some ways compared to the full latest version you have to install from source manually. Try this link for more info  [http://cinelerra.org/1/](http://cinelerra.org/1/)   I find it has a great deal of flexibility and is closer to a prosumer type package than any of the run of the mill ones. Search with google for further info as well as there was a lot of enthusiasts posting blog info and tutorials for this newest Cinelerra I found.
