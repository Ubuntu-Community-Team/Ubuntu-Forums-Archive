### Post by cchhrriiss121212 on 2010-07-18
The site states that it works for Windows as a VSTi or OSX, if you are not experienced with either WINE or music production on Linux then you will have a hard time using this on Ubuntu.
If you must use this synth, then your best bet is using it with WINE+Reaper which is the best VST host you will get.
If you just want a synth that sounds like 8-bit video games then try out LMMS and run it in 8-bit mode. Or try using zynaddsubfx.
