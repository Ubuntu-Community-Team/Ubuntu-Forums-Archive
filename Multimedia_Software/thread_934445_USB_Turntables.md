### Post by jlh68 on 2008-11-05
A regular turntable and ADS Tech "Instant Music" plus Audacity will work with the 32 bit Ubuntu, and might work with the 64 bit Ubuntu.  You will have to use Audacity got to [edit] > [preferences] and then select USB input.  Plug in the ADS USB unit and then make the change in [preferences], if the USB unit does not show up right away wait a minute or so for Ubuntu to recognize the new imput source.

A side note: I emailed ADS Tech and asked if their device would work with GNU/Linux and ADS responded that it absolutely would NOT work with a Linux OS.  Well _I have used the ADS device successfully with Ubuntu and Audacity._
