Hi,

Best video results (HDTV, h.264, VC-1) can be achieved by using Nvidia graphics. VDPAU is supported as of the 8xxx series (512 mb video memory recommended).

You might want to check out yaVDR. It's an Ubuntu based VDR distribution (including XBMC among other things).
