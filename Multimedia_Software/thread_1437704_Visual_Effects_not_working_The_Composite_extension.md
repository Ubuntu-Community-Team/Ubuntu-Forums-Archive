### Post by tjones00 on 2011-01-16
Try sifting through your /var/log/Xorg.0.log and look for it tries to load the glx module and what sort of errors it returns. If some required module/extension/option is not being automatically loaded by x server you can try messing around with a Modules section.

[http://www.x.org/archive/X11R6.8.1/doc/xorg.conf.5.html](http://www.x.org/archive/X11R6.8.1/doc/xorg.conf.5.html)

Scroll down to module section.

Make sure glx + any other modules/extensions/options it might require (eg extmod) to work properly are loaded with your own Modules section.

If that doesn't work because the base x server is missing something or built in static form you can try compiling your own.

[http://www.x.org/wiki/CompileXserverManually](http://www.x.org/wiki/CompileXserverManually)
