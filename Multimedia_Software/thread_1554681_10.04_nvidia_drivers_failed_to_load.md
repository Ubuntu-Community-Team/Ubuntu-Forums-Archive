### Post by rnj13 on 2010-08-17
Well, I feel like an idiot now.

There is a well-known, well-documented issue using tv tuner cards and nvidia drivers in the same system that has to do with the amount of virtual memory the kernel uses for drivers.

The issue is documented here:
[http://www.mythtv.org/wiki/Common_Problem:_vmalloc_too_small](http://www.mythtv.org/wiki/Common_Problem:_vmalloc_too_small)

Although in my 10.04 installation, the grub configuration files appear quite a bit different (the instructions in that link are for an older version of grub I think), so I used the directions from this post, but only increasing my vmalloc=192M first and that fixed my issue:
[http://ubuntuforums.org/showpost.php?p=8623584&postcount=17](http://ubuntuforums.org/showpost.php?p=8623584&postcount=17)

I know this solution is posted in many places but that one little line in the output of dmesg is easy to miss!):P
