### Post by medior on 2013-11-01
I stopped using easytag. It has a feature called autocorrection where it scans your audio tags and changes them if it doesn't like them. This would be a nice feature if the user could choose when to use it, and if it would tell you exactly what was changed. But just having loads of tags changed and not knowing why or what has changed annoyed me a lot.
Nowadays I use puddletag (spread sheet style tag editor) or kid3. kid3 is a KDE tool which nicely shows all tags if more than one version is present. I also have eyeD3 installed to easily view tags from the command line (it also supports editing but I always used GUI tools for that).
