### Post by seqbes on 2010-08-16
The following tools may suit your needs:
 
[LIST]
[*][COLOR=Blue]**[Free AVI metadata editor]("http://abcavi.kibi.ru/?tk")**[/COLOR] - supports AVI native metadata. (Win)
[*]  [COLOR=Blue]**[Commercial multi-format metadata editor]("http://www.digitalconfidence.com/MetadataTouch.html")**[/COLOR] - supports native metadata for AVI, and XMP metadata for AVI and MP4. (Win)
[*] [COLOR=Blue]**[MP4 command line metadata tool]("http://atomicparsley.sourceforge.net/")**[/COLOR] - supports native metadata for MP4 (*nix, Win)
[/LIST]
