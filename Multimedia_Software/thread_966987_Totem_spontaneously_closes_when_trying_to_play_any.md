### Post by Maqz447 on 2008-11-01
Try to turn off desktop effects and see if the problem persists. If it doesn't, you have a problem with compiz that seems to be somewhat common. See here:

[http://www.ubuntugeek.com/fix-for-video-playback-problem-in-compiz-fusion.html](http://www.ubuntugeek.com/fix-for-video-playback-problem-in-compiz-fusion.html)

This workaround fixed it for me. Well, actually I don't know about totem, as I don't use it. But VLC, Mplayer and gxine are working well now.
