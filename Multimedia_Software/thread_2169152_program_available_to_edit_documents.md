### Post by Mark Phelps on 2013-08-21
> **openation1 said:**
> Hello everyone......... I want to know if there is a program to edit documents after I scan them and put them in my computer? 

Depends primarily on how you save the scanned documents ...

If you save them as image files (i.e., jpg, gif), they are actually pictures, not documents, and a document editor is not going to be able to do anything with them.

If you have a way to save them as plain text files, any text editor can then modify them.  If you have a way to save them as rich text files, a word processing app (LiberOffice, MS Word) would then be able to modify them.
