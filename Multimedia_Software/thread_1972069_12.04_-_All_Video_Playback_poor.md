### Post by christopher.suttles on 2012-05-03
Have all latest updates installed. Video playback is poor and often crashes. Have Nvidia driver 295.40 installed, running 2 GH Core 2 Duo, 4 GB DDR2, GTX 260M.

Using banshee, video playback stutters randomly and uses about 50-60% of CPU.

VLC is even worse, takes up about 70-80% of cpu.

Maybe my video playback is not being accelerated properly? Anyone know how to check?

(Video work fine in Windows before. .avi , mp4, .mkv are all affected)

### Post by christopher.suttles on 2012-05-03
Installing 
"sudo apt-get install vdpau-va-driver vainfo"

then running 

"vainfo"

Showed the VDPAU was not working

So...I checked if libva1 was installed via 

"sudo apt-get install libva1"

And it wasn't...installing that fixed issues. CPU usage is the same with banshee, but video playback is now flawless.

Don't know why it was not installed....nvidia current and ubuntu restricted extras were among the first things I installed with a clean install of 12.04
