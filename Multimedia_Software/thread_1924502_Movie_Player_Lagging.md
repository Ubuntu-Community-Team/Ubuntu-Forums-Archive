### Post by WasMeHere on 2012-02-13
Welcome to the Ubuntu Forums!

In order to help we need to know more about your computer and what you want to play.

computer
- cpu type and speed
- ram size and speed
- graphics card or chip

movie
- codec
- frame size (e.g. 1280x720)
- frame rate (e.g. 30 frames/second)

It is a problem with old hardware, that new versions of Ubuntu have desktop environments, that are too demanding.

You might be able to install a good proprietary driver, that solves your problem.

Otherwise try ***Xubuntu*** with XFCE (small footprint) or ***Lubuntu*** with LXDE (smallest footprint)!

- Download the iso files and test them from a CD or USB drive.

- Alternatively install those desktop environments into your present Ubuntu.
```
sudo apt-get install xubuntu-desktop
```
```
sudo apt-get install lubuntu-desktop
```
You can select desktop environment at the log in screen.

Have fun finding out :-)
