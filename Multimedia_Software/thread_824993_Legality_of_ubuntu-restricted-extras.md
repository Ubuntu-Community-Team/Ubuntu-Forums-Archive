### Post by mcduck on 2008-06-25
No, the codecs are not sold, just like most of computer software is not sold. Instead they are licensed to users, and that limits your rights quite a lot.

Having previously owned a legal copy of Windows, the codecs, or anything else, doesn't make it any more legal for you to use these codecs.

Fluendo sells a package of legal codecs for Linux. Some were already available in repositories (I think at least the MP3 codec that they are giving avay for free).

Or you could use the illegal ones as it's very unlikely that anybody would ever come to check if you are using licensed codecs or illegal ones.

(btw, you are _not_ using the same codecs as in Windows. You are using different codecs that code and decode to same format.)

It depends on what the license says. Since you don't own your copy of the codecs what you can and what you can't do with them depends on the conditions of the license. Most of the licenses only apply to the one platform ythat you have paid for.

Also, you are probaly misunderstanding the idea of a codec. The difference between different codecs is much like the difference between different web browsers; they are different programs made to viwew the same content. A codec is simply a program that codes/decodes stuff into or from a certain format.

The intellectual property stuff applies to the _format_, not the codec. (I'm _so_ lucky not to live in USA :D). For example everybody who makes a codec, program or device that uses MP3 files has to pay licensing fee to those who own the rights & patents to that format. This license only applies to that codec/program/device, not every codec that uses the same format. So if you have Windows Media Player (microsoft has paid the license for MP3) that doesn't allow you to legally use any program/codec you wish, only the WMP. If you install Fluendo's legal MP3 codec it's still illegal for you to use lame, even when both codecs use the same MP3 format.

(besides, you didn't buy a legitimate copy of Windows. You bought a legitimate license to use Windows with the limitations described in the license. You don't own your copyy of Windows, you don'tt own the files that you write with Microsoft Office (you only own the _content_ of those files) and so on. You only have limited right to use them while Microsoft still owns them.)
