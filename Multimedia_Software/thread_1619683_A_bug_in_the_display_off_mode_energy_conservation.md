### Post by slobodyan on 2010-11-12
Greetings to all lovers of Ubuntu!

After  installing Ubuntu 10.10 on a laptop Acer Aspire 5942G graphics card ATI  Mobility Radeon HD 5650, I encountered a funny problem, for which I  then found a temporary solution.

Problem description:
some time after the boot falls color monitor. Color depth is only 65000 colors or more. In all settings, all is well (and the color depth is there spelled 24bit). Color depth is reduced at some time after a reboot.

[COLOR=#000000]On the forums I found a very similar description of the problem and recommending off Compiz. [/COLOR]However, my problem is not solved - all the same color is greatly reduced at some point.

After further experiments with the screensaver was set: color spoils the energy saving mode and turn off the display!

After disabling the option "disable display through .." color is now always saved in the norm!

Here is a temporary solution to this strange problem.

[COLOR=#000000]I would be grateful if the developers will notice my post and mark it as a bug.

[/COLOR]Also I will be glad to help this text to those who encountered this problem as I do.

Thank you for your Ubuntu! :)
