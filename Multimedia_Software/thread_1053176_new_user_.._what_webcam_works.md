### Post by kostkon on 2009-01-28
Have you used any application to test if it works or not? If not, for a start, a good way to test if it works is to use *Cheese*.

In recent versions of *Ubuntu* *Cheese* comes by default. You will find it in *Applications &#8594; Graphics*. If you don't have it you can install it using *Add/Remove*.
