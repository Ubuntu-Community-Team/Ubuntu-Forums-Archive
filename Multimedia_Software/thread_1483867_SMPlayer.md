### Post by Dhanjal on 2010-05-15
I was just wondering whether there is some way to permanently turn on the Extrastereo filter in SMPlayer. I have to turn it on every time manually

---

### Post by rvm4000 on 2010-05-15
There are several ways.

In preferences -> advanced there's a field where you can enter "actions" to be run every time a file is played. Type there **extrastereo_filter true**

Another possibility, in preferences -> advanced -> options for mplayer, type **extrastereo** in the audio filters field.
