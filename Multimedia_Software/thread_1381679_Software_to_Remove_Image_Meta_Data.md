### Post by alswilling on 2010-01-15
In an archive thread from 2006 a question was posted that received no response. Although the thread is old, chances are that others have had the same question who may not have been able to find an answer. 

This post is simply to provide the name of a software program that will accomplish this task; one that is open source and can be used on Linux.

There are times when removing non-image elements, such as embedded code, etc., is necessary, although the meta data may provide important information in some cases and should not be removed.

There is a software program that will remove image meta data that is also compatible with Linux. The program is Jhead, and it can be obtained at 
[http://www.sentex.ca/~mwandel/jhead/](http://www.sentex.ca/~mwandel/jhead/)
