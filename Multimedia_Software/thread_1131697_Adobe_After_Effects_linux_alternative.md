### Post by Arijan on 2010-10-26
> **ajmorris said:**
> 

All three of these, also have debian repositories that you can add to your ubuntu repositories, to download them via apt.



This is not true. Proprietary software is not available on repos (Fuson, Shake, Nuke). You can purchase them and install manually (manual install stands for trials too)

Jashaka, Cinelerra, and Blender are not proprietary, they can be downloaded from repos.
There is also Piranha - the one of it's kind - but also proprietary.

[QUOTE=trusktr]

Do you think we can do this with After Effects CS5? That'd be awesome!

[/QUOTE]

Running Adobe over wine is too buggy - it is suggested by the vendor to use them with virtualization (VmWare, VirtualBox)

Just for the record for compositing on Linux: Mighty software set for film production was written initially for RHEL by Autodesk and it included Flame, Smoke, Flint, Flare, Inferno, Lustre - this software kit is superior, and system/hardware requirements are no less than enterprise supercomputer solutions.

For Linux fun desktop users who don't want to pay for composite apps,
Blender is stable but not so friendly solution;
Jashaka is still not stable enough;
Cinelerra is a pain and tough to get used to;
LiVES can be helpful, but lacks features and commodity.

So, U can try installing virtual Windows, set it up properly, and install your Adobe apps - it will work far more stable than on wine.
