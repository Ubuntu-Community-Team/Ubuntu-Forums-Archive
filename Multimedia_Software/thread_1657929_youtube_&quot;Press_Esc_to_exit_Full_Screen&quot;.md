### Post by lovinglinux on 2011-01-02
Gnash is definitely not the best option.

Get [Flash-Aid 2.0]("http://www.webgapps.org/addons/flash-aid") and run it. It will detect installed flash plugins, remove them and install the best option according to your system architecture and version. Additionally, it will apply some tweaks that should fix your problems. If you still get that issue after running Flash-Aid, then go to the extension Help tab, click the "Report" button and send me the results so I can analise your situation.
