### Post by little_penguin on 2009-01-11
You can hardcode the subtitles into the avi files using avidemux, then create a DVD with devede. You can also use ManDVD which lets you create the DVD and embed the subtitles at the same time. 

I'm not sure what program would create a DVD that enables you to select or deselect subtitles though.

### Post by binbash on 2009-05-18
You do not need to hardcode the subtitles if you are going to use.You have a couple of options : 

1- Some dvd players can read divx files so burn a dvd similar to this structure

-Foldername.srt

2- You can use devede to convert and burn the movie.It is very easy to use and you can put around 2 movies into a dvd.BUT do not forget to show to subtitle(s) to devede
