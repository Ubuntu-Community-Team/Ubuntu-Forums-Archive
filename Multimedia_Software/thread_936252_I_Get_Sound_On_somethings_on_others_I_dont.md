### Post by minky311 on 2008-10-02
First I would install ubuntu-restricted-extras by going into the synaptic package manager System->Administration->Synaptic. See if that helps for the mp4 and mp3 problem. As for the streaming problem you should install the mplayer plugin for firefox by going into the synaptic package manager and searching for mplayer and installing it which will also install the plugin for firefox.
