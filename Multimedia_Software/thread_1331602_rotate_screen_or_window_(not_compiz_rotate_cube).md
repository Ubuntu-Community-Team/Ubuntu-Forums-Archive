### Post by Dragon ELEMENTAL on 2009-12-14
Hiya, I stumbled here looking for a Compiz Fusion way to rotate indevidual windows / the screen.
There is a plugin in development that will allow doing this (and a lot more!).  "The Freely Transformable Windows plugin allows windows to be rotated, slanted, and scaled." @ [http://wiki.compiz.org/Plugins/Freewins](http://wiki.compiz.org/Plugins/Freewins).  However this is currently in development, and I won't recommend it for general use.

Until it's released, there is always SMPlayer (based on MPlayer core), which in my opinion is the BEST media player frontend available on Linux (after the 'XBMC for Linux' project).  It features rotating the video 90° both clock & counter-clock wise and also lets you flip the video horizontally and vertically (you can assign custom keyboard shortcuts or can use the right-click context menu).

You can get it here:
[http://smplayer.sourceforge.net/downloads.php?tr_lang=en](http://smplayer.sourceforge.net/downloads.php?tr_lang=en)
or directly:
[https://launchpad.net/~rvm/+archive/smplayer](https://launchpad.net/~rvm/+archive/smplayer)

Hope this helps!

^_^
