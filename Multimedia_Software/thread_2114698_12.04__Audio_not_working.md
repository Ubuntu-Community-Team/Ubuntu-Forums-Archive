### Post by hugheyst on 2013-02-10
UPDATE: I've figured it out. I ran the following command [thanks to [http://blog.zloether.com/2009/11/reset-sound-settings-in-ubuntu.html](http://blog.zloether.com/2009/11/reset-sound-settings-in-ubuntu.html)]:

```
sudo apt-get purge alsa-base && sudo apt-get install ubuntu-desktop
```

All is well now.
