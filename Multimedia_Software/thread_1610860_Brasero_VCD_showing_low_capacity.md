### Post by tg3793 on 2011-09-30
Yup I was having a similar problem. Trying to create and SVCD with Brasero. My .avi file was only a couple hundred megs in size and my CDR was fine.

But hey I got a work-a-round and thought I'd share it with you.

I'm using Brasero 2.30.2 and decided to try burning my video project to an image file first and then used Brasero again to burn the .cue file which of course calls the associated .bin file.

It came off without a hitch and I just put the newly burned VCD in my DVD player on the upstairs television and it worked.

Hope this helps others.
