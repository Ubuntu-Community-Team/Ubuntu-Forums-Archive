### Post by brydong on 2009-01-30
I'm looking for recommendations for a distro/setup. I'm building a media pc that will drive our only tv full time. We will NOT have cable of any kind so the record features of mythtv aren't needed. This will essentially replace current tv. Key uses will be:

- cataloging and watching avi's from local hard drive.
- ripping, cataloging and listening to music.
- ripping, cataloging and watching dvd's.
- watching streaming video from net
- listening to streaming music.

For this config, is mythtv still the best choice, using it simply as a front end? Or just stick with plain old vlc for videos. Is there a cataloging app for videos similar to what f-spot does for photos?

all suggestions welcome, thanks,
brydon

### Post by enosis on 2009-01-30
Hi brydong,

I'm trying to setup exactly the same thing (no cable, only local movies/tv series and streaming videos/music).

What's also important for me is the interface of the application, and to be honest MythTV looks really outdated/ugly.

I found a very nice looking media center, along with all the features that we need (I'm not sure about the ripping part), and it plays all kind of content easily.

I'm talking about xbmc, which you can find here: [http://xbmc.org/home/](http://xbmc.org/home/)

[CENTER][IMG]http://farm4.static.flickr.com/3144/2876723351_95e56df043.jpg?v=0[/IMG][/CENTER]

An more feature-rich version of xbmc is Boxee ([http://boxee.tv/index2.php](http://boxee.tv/index2.php)), which uses xbmc's base-code and adds social networking features, along with a much easier way to stream content from various sites.

[CENTER][IMG]http://i.i.com.com/cnwk.1d/i/bto/20090123/abc-on-boxee1_610x343.png[/IMG][/CENTER]

I'm super-happy with these two apps, but the only problem for me is stability. I was testing xbmc at opensuse, but it was crashing at random times. And since Boxee comes only for ubuntu by default, I decided to try kubuntu. 

All these happened the last 2 days, so I still haven't tried Boxee at my machine, nor xbmc's stability at kubuntu (still setting up drivers, etc). :)

Well, I hope this helped. I hope both of us will soon have a nice media center!
