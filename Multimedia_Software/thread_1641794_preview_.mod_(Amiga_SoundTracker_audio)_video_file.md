### Post by BSD_dad on 2010-12-09
Just bought Canon FS300 camcorder which records in .mod files. Running 10.10 netbook. Nautilus does not render a video preview icon for this file, it shows a musical note. .mov files show video still for icon fine. Doesn't help browse video clips needless to say. Anyway to preview this file type, either in different file manager or application? File Properties shows type as "Amiga SoundTracker audio". TIA!

UPDATE, SOLVED: found [this]("http://mail.gnome.org/archives/gnome-list/2007-October/msg00001.html") after more searching with mimes keyword. I only deleted the .mod lines and now Nautilus recognizes the .mod properly as an mpeg-2 file complete with video still preview icon.
