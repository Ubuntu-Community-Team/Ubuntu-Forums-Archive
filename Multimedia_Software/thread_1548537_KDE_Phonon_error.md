### Post by Yellow Pasque on 2010-08-11
It means you should run the KDE settings program (install 'systemsettings' package if you don't already have it). Then go to multimedia options and select pulseaudio as the default KDE sound device for all categories so KDE apps won't annoy you.
```
systemsettings
```
