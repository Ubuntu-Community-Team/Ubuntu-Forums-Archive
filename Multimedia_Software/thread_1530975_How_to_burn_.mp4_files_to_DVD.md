### Post by El_Belgicano on 2010-07-23
I would go straight to tovid, from the wiki page it is very easy to make up (or simply copy/paste) the right commands to the terminal of your choosing. And if you don't like the command line, there is also a simple GUI for tovid available.

I usually just write the list of commands to ~/.dvdcreate and run it.

Example:
- [Making a simple DVD]("http://tovid.wikia.com/wiki/Making_a_basic_DVD")
