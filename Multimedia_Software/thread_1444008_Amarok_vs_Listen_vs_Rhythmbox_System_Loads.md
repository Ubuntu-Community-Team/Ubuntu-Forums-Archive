### Post by 2hot6ft2 on 2010-03-31
I thought I would share something I just found out.

System is AMD64 Turion X2 32 bit OS 9.10 with 3 GB RAM

I decided to give Listen a try and noticed that the system load monitor was around 2/3 to 3/4. I didn't think to check the % of load until I started writing this but I did a comparison against Amarok and Rhythmbox.

I let the system sit with nothing else running, it's about 82 F right now and I played the same song (Dido - Honestly Ok) in each player. Also I let it cool down to idle temps. before starting the next player and waited until about 3:15 into the song to take the screen shots.

Here are the temps. all in Celsius.
Amarok
GPU = 67
CPU Core 1 = 46
CPU Core 2 = 46
HD Temp. = 46

Listen
GPU = 79
CPU Core 1 = 64
CPU Core 2 = 64
HD Temp. = 46

Rhythmbox
GPU = 63
CPU Core 1 = 43
CPU Core 2 = 43
HD Temp. = 46

None
GPU = 59
