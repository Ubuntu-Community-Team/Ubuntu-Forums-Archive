### Post by sikander3786 on 2010-11-21
Was there any special purpose of setting up xorg.conf as the modern X doesn't need it anymore.

Switch to tty1 by pressing Ctrl + Alt + F1 and login using your credentials.

Backup your existing xorg.conf by

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.old
```

Remove it

```
sudo rm /etc/X11/xorg.conf
```

Reconfigure your graphics

```
sudo dpkg-reconfigure -phigh xserver-xorg
```

Reboot

```
sudo shutdown -r now
```

And see if it improves anything.
