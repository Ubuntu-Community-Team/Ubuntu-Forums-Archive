### Post by drawkcab on 2011-08-19
I came across this new cross-platform project called FLIRC.  

The idea is that, instead of buying and configuring a special IR remote to operate XBMC/WMC/Boxee, you can configure the Flirc dongle via gui to work with your existing remote.

[http://flirc.tv/](http://flirc.tv/)

Flirc supports Windows, Mac and Ubuntu (.deb file updated via ppa). 8-)

### Post by handy on 2011-08-20
On a tangentially connected topic:- there is actually simple open-source software available that allows you to use an Arduino ($39-) with an IR receiver (~ $1.35) to read whatever button you push on whatever IR remote & the software gives you the code that the IR remote outputs.

Which means you can use any redundant (or otherwise) IR remote, to do whatever you program it to do with the Arduino (& compatible) boards, or any other suitable chips that you can program for that matter.

There is a good tutorial on the topic here:

[http://tronixstuff.wordpress.com/2011/03/30/tutorial-arduino-and-infra-red-control/](http://tronixstuff.wordpress.com/2011/03/30/tutorial-arduino-and-infra-red-control/)
