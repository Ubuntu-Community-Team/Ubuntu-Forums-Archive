### Post by cozmicharlie on 2008-07-25
It does close when you are recording.  The basics for using it are you select the area of the desktop (or the entire desktop), press record, do whatever you are doing that you want to record, stop, render and save the file.  The xvidcap web site has some good info and a few videos that show how to use it - this should help [http://xvidcap.sourceforge.net/](http://xvidcap.sourceforge.net/)

Enjoy
