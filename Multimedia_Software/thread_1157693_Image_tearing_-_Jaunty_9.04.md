### Post by rockinstlouis on 2009-05-13
If anyone is having a lot of tearing in Jaunty.. this is what I did..

first off I have a NVidia graphics card.. i've found if you don't have a 3D video card, you're pretty much screwed..

alrighty.. i'm sure the Ubuntu's latest "packaged" Nvidia drivers are ok to use, but I downloaded the latest version from Nvidia and complied it..

make sure the driver settings are set to the "best quality", and select all your 'sync to vblank' options

make sure your running the compiz settings manager and you set the 'sync to vblank' option in there..

i did all of this and still got tearing.. until i turned off my visual effects.. i'm going to experiment with a custom option just for my own curiosity..

hope this helps!
