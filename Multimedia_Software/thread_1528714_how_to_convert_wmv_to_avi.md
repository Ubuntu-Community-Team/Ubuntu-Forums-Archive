If you prefer a GUI solution to ffmpeg, there is always winff which is in the repos. Another GUI option is Avidemux which makes it easy to do simple filtering/resizing and clipping operations as well as transcoding. To get any of these to work, you will probably need to install the ubuntu-restricted-extras package and perhaps the win32codecs (or win64codecs) package from medibuntu.
