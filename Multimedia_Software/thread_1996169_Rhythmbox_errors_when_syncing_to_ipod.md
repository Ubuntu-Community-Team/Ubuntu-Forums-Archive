However, when I do sync there are a number of errors displayed and as a result the library says a total of “xxx” songs and the ipod displays around 20 less. ### Post by unknown user on 2012-06-11
I have now sorted out what the issue was with the error message that I mentioned before. It was due to a number of songs that were showing in my library but were not actually there. I have since removed the files in question from my library listing in Rhythmbox and now when I perform a sync with my ipod, the error message does not appear.

All sorted....
