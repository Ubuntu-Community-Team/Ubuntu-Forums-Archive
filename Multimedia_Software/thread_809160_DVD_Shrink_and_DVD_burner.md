### Post by aeiah on 2008-05-27
k9copy will help you shrink dvds to fit on 4.4gb single layer dvds. k3b or brasero should be able to take a dvd file structure (VIDEO_TS etc) and create a compatible dvd. they'll also burn your .iso files, although i find it much easier just to right click on the .iso file and select 'write to disc'.

i dont know if k3b or brasero nativley handle .img files, but there are command line converters that will quickly convert .img to .iso
