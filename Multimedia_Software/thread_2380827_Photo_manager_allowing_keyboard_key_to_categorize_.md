### Post by TheFu on 2017-12-22
My workflow is different from yours, but geeqie is the viewer/decider that I use.  Going through thousands of photos a week sucks using other tools that I've tried.  Geeqie does have a little scripting language which might be useful.  The built-in bulk renamer is good, but too many clicks to access, sadly.

I don't use it as a photo manager, having been burned multiple times in trusting any so-called "photo managers".  I trust the directory system and the metadata stored **inside** the actual files.  I do **not** trust metadata stored in some external file(s) or in an external DB.
