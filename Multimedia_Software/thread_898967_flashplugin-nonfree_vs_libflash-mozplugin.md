### Post by etnlIcarus on 2008-08-24
libflash-mozplugin is open-source and backward engineered. It's by no means official and it isn't 100%.

flashplugin-nonfree is the official one from Adobe and should be more reliable. However, with this latest release of Ubuntu which switched to pulseaudio, a bug in the nonfree plugin meant flash would crash for a lot of users.

If you search for, "flash crash", you'll probably find several hundred threads directing you to one of several solutions.
