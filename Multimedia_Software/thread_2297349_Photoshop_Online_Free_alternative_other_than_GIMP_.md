### Post by brian-mccumber on 2015-10-03
Photoshop is bulky too and it's interface is just as confusing as Gimp's if you have not used the software much. If you are wanting to do professional editing then you will need professional quality editing software and there is nothing light about that. I use Mirage for quick crops and re-sizes, but when I need to do some real editing, like cutting stuff out of images, blending images, or web graphics creation I use Gimp. 

You could always install Wine and then use Photoshop in Linux if Photoshop is the software that you know well.

### Post by Bucky Ball on 2015-10-03
... or you could run Windows as a virtual machine in Ubuntu and run Photoshop, or, if you must use Windows software, use Windows. :)

If you're not willing to use an alternative to PS, using Windows, one way or the other, is really your only alternative. Gimp is the one people will suggest as it is the one closest to Photoshop and is intended as the alternative for PS. Gimp is used in some commercial and production environments right along with Photoshop and is possibly less bulky than PS and slightly less complicated, only because it is not quite as feature packed.

### Post by yoshii on 2015-10-05
I am still trying to learn how to use GIMP.  But in the meantime, I use PhotoFiltre (the freeware version) which runs just fine in Wine.  You might want to try that. It's much easier to learn and it has the basic photo editing commands even though it's not as powerful as GIMP or PhotoShop.  Here's the link:  [http://www.photofiltre-studio.com/pf7-en.htm](http://www.photofiltre-studio.com/pf7-en.htm)
It's portable freeware.  Also there are plugins available for it as one install:  [http://www.photofiltre-studio.com/plugins/PPP_III-en.exe](http://www.photofiltre-studio.com/plugins/PPP_III-en.exe)
