### Post by driftertx on 2009-11-16
From: [https://help.ubuntu.com/8.04/installation-guide/ia64/hardware-supported.html](https://help.ubuntu.com/8.04/installation-guide/ia64/hardware-supported.html)

Graphics Card Support

Ubuntu's support for graphical interfaces is determined by the underlying support found in X.Org's X11 system. Most AGP, PCI and PCIe video cards work under X.Org. Details on supported graphics buses, cards, monitors, and pointing devices can be found at [http://xorg.freedesktop.org/](http://xorg.freedesktop.org/). Ubuntu 8.04 ships with X.Org version 7.3. 

So although this says for 8.3 it's the same policy now too except the xorg version now I think is 7.6That forum specifically deals with nvidia cards in linux. More than like your card will work fine and to get more acceleration you can install the nvidia 'proprietary' driver which works pretty well.

I did some searching for you, there is a forum that x.org recommends for help with nvidia cards, [http://www.nvnews.net/vbulletin/forumdisplay.php?f=14](http://www.nvnews.net/vbulletin/forumdisplay.php?f=14)

Good luck.
