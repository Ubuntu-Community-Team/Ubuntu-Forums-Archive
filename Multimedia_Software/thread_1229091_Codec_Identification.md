### Post by mc4man on 2009-08-01
Mediainfo is for the most part fairly accurate
(( best way is to open mediainfo and drag your file into, otherwise  browse. 
The easy view is the best choice to open a file in, then switch to html for extended info, text for copying info

[http://mediainfo.sourceforge.net/en/Download](http://mediainfo.sourceforge.net/en/Download)

I'd use the 'official' , just pick the ubuntu version you're on, requires 3 packages, install from bottom to top (libzen0, libmediainfo0, GUI

If you do the ppa just install from synaptic or apt-get
