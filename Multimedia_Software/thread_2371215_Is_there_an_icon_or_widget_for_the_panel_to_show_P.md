### Post by TheFu on 2017-09-16
Plex MS is a system-wide service.  It is running, like any other service runs.  Always.  Steam is an end-user tool, not a system-wide thing.  Would you expect to be shown that apache or mariaDB are running in a GUI?  Nope.  They are system services and automatically start at boot time.  Plex Server is the same way.
