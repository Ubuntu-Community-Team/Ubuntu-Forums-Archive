### Post by poikiloid on 2010-02-25
try DigiWF.....it is awesome!!! lets you edit the xmp metadata of images and video of single files, and even multiple files, right from a right-click item in nautilus.  it is a very light-weight and sleek solution.

[https://launchpad.net/digiwf](https://launchpad.net/digiwf)

you can add the PPA repository ([https://launchpad.net/~svd/+archive/ppa](https://launchpad.net/~svd/+archive/ppa)) to Software Sources and install it via Synaptic so you get automatic updates, or you can just install the DEB package on the overview page.
