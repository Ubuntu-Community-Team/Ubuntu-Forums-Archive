### Post by BicyclerBoy on 2011-01-16
Are you running Ubuntu gnome desktop ?

If so navigate the desktop GUI menu
System/Preferences/Sound/

select input tab
check you have any inputs selected & not muted. Turn up the gain

Tap the mic with finger does the Input Level vu meter bar graph flash ?

Select different input connectors & retry above...
