### Post by somersetsimon on 2008-11-09
I've just installed Ubuntu 8.10 on a desktop that has an nVidia GeForce4 MX 440 with AGP8X card installed. My xorg.conf file doesn't mention any specific devices, just this

```
Section "Monitor"
    Identifier     "Monitor0"
    VendorName     "Unknown"
    ModelName      "Unknown"
    HorizSync       30.0 - 110.0
    VertRefresh     50.0 - 150.0
    Option         "DPMS"
EndSection

Section "Device"
    Identifier     "Device0"
    Driver         "nvidia"
    VendorName     "NVIDIA Corporation"
EndSection

Section "Screen"
    Identifier     "Screen0"
    Device         "Device0"
    Monitor        "Monitor0"
    DefaultDepth    24
    SubSection     "Display"
        Depth       24
    EndSubSection
EndSection
```

I installed nvidia settings and nvidia-xconfig.

When I run nvidia-settings it tells me

```
You do not appear to be using the NVIDIA X driver. Please edit your X configuration file (just run `nvidia-xconfig` as root), and restart the X server. 
```

If run 

```
sudo nvidia-xconfig 
```

I get message that says

```
Using X configuration file: "/etc/X11/xorg.conf".

VALIDATION ERROR: Data incomplete in file /etc/X11/xorg.conf.
                  Device section "Configured Video Device" must have a Driver
                  line.

Backed up file '/etc/X11/xorg.conf' as '/etc/X11/xorg.conf.backup'
New X configuration file written to '/etc/X11/xorg.conf'

```

I want to get at the settings in more detail so I can configure a TV output. Am I doing anything obviously wrong?

I hope it is supported! The main problem seems to be that I need to have the right info writen into the xorg.conf file. My installation of 8.10 onto my laptop (with a Trident Cyberblade driver) had the same problem. In previous versions I could call up display-config and select my card and driver and monitor settings and this would get written into the xorg.conf file. I got around this on the laptop by using a Live CD of 8.04, setting up the graphics card, saving the xorg.conf file, then copying that back into my 8.10 installation!
