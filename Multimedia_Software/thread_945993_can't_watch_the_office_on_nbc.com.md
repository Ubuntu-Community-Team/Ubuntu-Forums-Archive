### Post by Denestria on 2008-10-12
I don't know if Silverlight is needed to make the videos at NBC.com work or not but...  did you install Moonlight from the download plugin link on [http://www.go-mono.com/moonlight/](http://www.go-mono.com/moonlight/)?  I went to their website which says
> Note: These are currently built without multimedia support. **No video or mp3 playback is enabled** on these binaries.


The [http://www.mono-project.com/Moonlight](http://www.mono-project.com/Moonlight) site says
> These builds **do not include media codecs (video or audio**), for that, you must currently **build Moonlight from source code**.


So even if Moonlight would play the videos at NBC you would have to compile it from source code to get that capability.  This project is still in the development phase.
