### Post by SuperGrouper on 2010-03-16
Hi!  I had this problem with an external hard drive I used to use with Windows (I use a dual boot.)  I solved the problem by backing up the data on it onto another hard drive and formating it (in Windows.)  Then I restored the data onto it and reloaded Linux, and it read the volume as read/write instead of just read.

I'm not quite sure why it did that, but it worked for me with my external HD, so you might try it with your iPod if nothing else works (be sure to reinstall the iPod firmware via iTunes before loading Linux.)

You might also try formating it via a Mac (borrow one from a friend?) and using the same method if the Windows one doesn't work.

I'm not 100% sure this would work with an iPod, though, so only try it if you bought the insurance. ^^;  I once killed an iPod with experiments like this (involving RockBox, which no one ever told me would blast my model.)

If I were you and I had insurance on the iPod, though, I'd give it a try.
