### Post by SeijiSensei on 2013-09-08
The PS3 only supports a [limited number of formats](http://manuals.playstation.net/document/en/ps3/current/video/filetypes.html).  One alternative is to run [ps3mediaserver]("http://www.ps3mediaserver.org/") on the machine where the videos are stored and use DLNA on the PS3 to connect to the server.

The PS3 does support the DivX and XviD codecs.  I suggest converting to XviD in the AVI container.
