### Post by cor2y on 2008-10-16
Its Real Media Networks latest proprietary format.
If you want to play it you will need to use the windows version of the real media player the linux and mac version don't play the format.
And if you wish to convert it then you need to pay for the Real Producer/Player Plus (windows version only) which will allow you to convert it/burn to disc as vob etc.
Nothing in linux or outside of the real networks can play this format yet, it has not been reversed engineered yet by third parties.
