### Post by drmrgd on 2012-02-15
> **SFBrother said:**
> I'm new. I don't really know much about that stuff. Do you think it is possible to use this software called "Wine" to use my windows software with linux?

You can try.  Wine is hit and miss with some Windows software.  That is to say, some work, some work with little glitches and bugs, and some don't work at all.  You can use this site, [Wine HQ]("http://www.winehq.org/") to check the program that you want to run under Wine.  It'll tell you how good it will run and some instructions for installing it if available.  

I've used Fraps a bunch in the past for game recording.  There is no linux port of Fraps.  But, I did find this, which looked pretty decent:

[http://www.dedoimedo.com/computers/glc.html](http://www.dedoimedo.com/computers/glc.html)

The package is here: [https://launchpad.net/~arand/+archive/ppa](https://launchpad.net/~arand/+archive/ppa)

If you're interested, you can find instructions on how to install, you can check out the GLC wiki at github here:

[https://github.com/nullkey/glc](https://github.com/nullkey/glc)
