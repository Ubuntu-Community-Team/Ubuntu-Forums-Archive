### Post by sledtv on 2009-11-04
Hi

To resolve this problem use firefox extention user agent switcher ([https://addons.mozilla.org/en-US/firefox/addon/59](https://addons.mozilla.org/en-US/firefox/addon/59)).
and set like this


In Firefox, click Tools - Default User Agent - Edit User Agents. Click New - New User Agent.

Description: Quicktime 7.6.2
User Agent: QuickTime/7.6.2 (qtver=7.6.2;os=Windows NT 5.1Service Pack 3)

Click Ok - Ok.

Click Tools - Default User Agent - Quicktime 7.6.2.


[IMG][[img]http://img692.imageshack.us/img692/7192/screenshot008o.th.png[/img]](http://img692.imageshack.us/img692/7192/screenshot008o.png)[/IMG]

Then switch to quicktime user to see trailers and re-swith to default for normal navigation

Enjoy
