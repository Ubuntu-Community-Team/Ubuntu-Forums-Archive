### Post by MDSmith2 on 2009-04-21
I personally prefer compiling FlightGear from CVS source but you can download it from the Ubuntu repos by using:
```
sudo apt-get install flightgear
```
That should download FlightGear and all of it's dependencies (Took about 3 hours to download on my small machine).

You can start FlightGear by running ```
fgfs
``` and the command line arguments can be found by using ```
fgfs --help --verbose
```
Aircraft and scenery can be downloaded from the [FlightGear](http://flightgear.org) website.
