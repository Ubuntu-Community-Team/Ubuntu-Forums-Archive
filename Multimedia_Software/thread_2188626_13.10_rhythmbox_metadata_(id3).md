### Post by Yellow Pasque on 2014-01-26
Rbox can cause fits if mp3 files have both ID3 and APE tags (because it reads the APE tag last and that is what's displayed).
mp3val is a handy little utility to tell you what kind of tags you have.

If you're like me and just want to use ID3 tags, you can install python-mutagen package and run this little APE tag stripper python script on your collection:
```
#! /usr/bin/python
# -*- encoding: utf-8 -*-

#!/usr/bin/env python
import sys
import mutagen.apev2
for p in sys.argv:
	try:
		mutagen.apev2.delete(p)
	except Exception, e:
		print e
```
