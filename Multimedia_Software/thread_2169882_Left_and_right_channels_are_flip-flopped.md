To really answer the question, The Xonar XG is based off the Oxygen DSP Sound Chip which just recently made it into the Kernel with driver support. So be patient as the ALSA, Pulseaudio, and Kernel Team are trying to get it to work properly. If in doubt, upgrade. Try Ubuntu 13.04 if you aren't already using it. As time progresses, the drivers will get better. Promise. :)
