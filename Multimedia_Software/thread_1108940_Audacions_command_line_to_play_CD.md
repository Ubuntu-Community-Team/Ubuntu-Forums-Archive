### Post by mc4man on 2009-03-28
I don't believe there is any command to autoplay audio cd's with audacious anymore. (unless you revert back to a much older version (1.3...?)

You need to call the plugin in the current version which only seems available from the gui

On the other hand Amarok will work perfectly for autostart *and* autoplay from one or more devices.
