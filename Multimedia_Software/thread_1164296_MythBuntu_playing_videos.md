### Post by wyliecoyoteuk on 2009-05-19
Yes it is possible.
You will have to make sure that MythTV knows where they are, in the setup menus.
Also make sure that the viewer is set to "Internal" 9.04 sets the viewer to mplayer, which will cause you to lose control if you play an AVI for example.
Go through the menus and select setup, then video setup

You may also have to use Mythbuntu control centre to install the restricted codecs.
