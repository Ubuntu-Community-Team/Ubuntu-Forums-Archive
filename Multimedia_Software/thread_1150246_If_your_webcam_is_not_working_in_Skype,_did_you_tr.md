### Post by Keyper7 on 2009-05-05
Here's the story: video calls in Skype were not working for me ever since I upgraded from Hardy to Intrepid. My reasonable guess was that for some reason Skype wasn't supporting my webcam anymore. After some random googling and failed attempts to solve the problem (including the famous v4lcompat hack that had no effect for me), I stumbled upon [this link](http://forum.skype.com/index.php?showtopic=328901), which describes how to force Skype to use X11 instead of Xv.

Guess what? It worked. So all the time my webcam was working perfectly. The problem was with displaying video. So I'm leaving the link here as a suggestion. Is anyone here in the same situation? I'm very curious about the reasons for this bug.
