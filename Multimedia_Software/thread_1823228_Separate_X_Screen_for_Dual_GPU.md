### Post by belize1334 on 2011-08-12
It seems to have been a driver issue. I was running the ver 173 driver since it had better fan control so my tower was quieter. I switched to ver. current and the dual display came up perfectly. I then discovered the "coolbits" option in xorg.conf to make the new driver throttle the cards so now they run quietly and then throttle up when being used. It's nice to watch the thermal load and see the second card turn the fan on when I enable full-screen video on the associated monitor. 

Anyway, let this be a lesson to me to use the current drivers...
