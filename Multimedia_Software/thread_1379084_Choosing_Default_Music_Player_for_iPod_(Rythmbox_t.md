### Post by mac1984user on 2010-01-12
I recently installed Banshee and would like to set it as my default music player. Currently, when I plug in my iPod, Rythmbox auto loads and then Banshee cannot access the device. Is there a way to go about changing this setup so when I plug in the iPod, it's Banshee that loads it and not Rythmbox? Cheers.
 
*USING Ubuntu 9.10

---

### Post by Mze on 2010-01-12
1. Under System>>Preferred Applications>>Multimedia tab, select Banshee from dropdown list

2. In Nautilus>>Edit>>Preferences>>Media tab, make changes to media handling when they get connected.
