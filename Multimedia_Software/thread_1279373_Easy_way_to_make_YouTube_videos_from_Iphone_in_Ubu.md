### Post by mhedges48 on 2009-09-30
I've been shooting video on my Iphone and looking for the best way to get edited/merged movies onto YouTube. After much trial and error the below works great for me:

(1) transfer movies from Iphone to computer (either through basic USB, which works for me in Jaunty, or bluetooth).

(2) convert the .mov files to .flv using the program PyTube

(3) edit/merge/cut/etc the files using the awesome program Pitivi Video Editor.

(4) Upload to YouTub, and you're done.


hope this saves someone some time
