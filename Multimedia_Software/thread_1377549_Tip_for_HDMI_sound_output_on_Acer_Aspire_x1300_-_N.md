### Post by multanihl on 2010-01-10
Just thought I would post this in case anyone else had the same problem.  I set up a little Acer SFF machine in my living room, with HDMI cable going into the television.

I was going crazy trying to get the sound to output through the HDMI cable instead of the analog audio, and it wasn't until I switched the device to headphone-out in alsamixer (for some reason) that I was able to get sound to output through the HDMI instead of the regular audio out.

I don't think I'm the only one who bought one of those machines for this purpose, so I hope this helps!
