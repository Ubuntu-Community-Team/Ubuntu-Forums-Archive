### Post by speartip on 2013-05-21
First of all I would uninstall what you have already done.
In a terminal run the following:
```
sudo apt-get purge --remove gmusicbrowser*
```
Then install the official one from the repo.
Either use the ubuntu software centre, or from a terminal:
```
sudo apt-get install gmusicbrowser
```

The problem with using .deb packages or adding ppas to your software sources that you find on the net is that they are mostly untested and run the risk of breaking your system. While this is unlikely it can & does happen.
You are much safer sticking to the official repos, unless there is something you absolutely need that Ubuntu has not packaged, or you need a newer version of something.

Once you have installed the official version test it out & report back if you still have freezes.
