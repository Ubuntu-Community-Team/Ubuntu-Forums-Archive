### Post by tijmz on 2008-08-07
I know that [VLC](http://www.videolan.org/vlc/) allows you to build video playlists. There are likely many solutions out there that can do so ([Miro](http://www.getmiro.com/features/) might be interesting, too and is available from the repos, just like VLC). 

For the rest, I would say you could set things up so that the video output from your player always goes to a secondary screen (in your case, the projector), while the primary screen is accessible by remote desktop. Remote desktop is pretty easy to set up using Ubuntu and I would guess the separation of video content/desktop is also doable.

Sounds like a nice project, so if you get it running, do post how you did it :)
