### Post by arghya05 on 2010-10-04
ultimately solved the problem with 5.1 speaker recognized as stereo.just run the alsa script upgrade to load the the latest
ALSA driver 10.23 and  change the  /etc/modprobe.d/alsa-base.conf file adn add the following lines according to
options snd-hda-intel model=3stack-6ch-dig(based on model id)

now save the script and reboot the computer.Go to ulsamixer and scroll to exteme right and change the channel from 2 to 6.

now everything works
