### Post by wolfen69 on 2008-07-31
go to system>admin>software sources and make sure all repos are enabled. then reload. then open terminal and ```
sudo apt-get install flashplugin-nonfree
```

### Post by lordadi on 2008-07-31
Hi,

Try to install flash nonfree from the repos. Go to system > admin > synaptic package manager (it will ask for your password) and click settings and ensure that universe and multiverse are ticked. Then close the window adn click reload in synaptic. Then search for flash nonfree adn right click it and say mark for install.


Cheers,
