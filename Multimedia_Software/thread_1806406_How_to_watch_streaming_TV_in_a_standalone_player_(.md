### Post by ar-jar on 2011-07-17
I'd be interested in finding a solution to this too. I've been looking around a bit and there doesn't seem to be any simple solutions. If your video is on one of the supported sites, then this add-on might do the trick (my favorite site is not supported): [https://addons.mozilla.org/en-US/firefox/addon/video-downloadhelper/](https://addons.mozilla.org/en-US/firefox/addon/video-downloadhelper/). With this tool you should be able to download the file first and then play it. There are some script-based methods for e.g. YouTube as well: [http://www.webupd8.org/2010/03/2-ways-of-playing-youtube-videos.html](http://www.webupd8.org/2010/03/2-ways-of-playing-youtube-videos.html).

Pls let us know if you find something more general. I guess the content providers don't want their content copied so they try to hide it as well as they can... -A
