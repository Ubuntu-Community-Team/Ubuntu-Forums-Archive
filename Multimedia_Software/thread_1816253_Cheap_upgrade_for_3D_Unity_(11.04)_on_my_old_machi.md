Just upgraded one of the older machines in my household to Natty, and while this particular machine has performed and served me well through years, I suddenly realize that it might be aging - like me... :-k

After completing the upgrade, I was told that my hardware did not support Unity and I would be using classic Gnome. Kind of annoying and it bothers me a bit that this otherwise quite functional hardware is no longer supported by main stream Ubuntu releases - I thought, cheap computing was a part of the Ubuntu concept. Anyway, by adding Unity 2D I got sort of the right Unity feeling, but the 2D version is lacking behind the real thing when it comes to the new GUI tricks and polishing (kind of Windows 7 Basic feel :().

Not being quite ready to dump this machine, I started thinking about giving it a minor graphics upgrade - a cheap one! The machine is based on an ASUS motherboard **A7N8X-E** equipped with an AM Athlon XP 3200+ processor and 1.5 Gbytes of RAM.
