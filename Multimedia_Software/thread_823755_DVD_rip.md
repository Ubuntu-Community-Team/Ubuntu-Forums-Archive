### Post by llama320 on 2008-06-09
I use acidrip

```
sudo apt-get install acidrip 
```

It has subtitle functionality and gives good video quality once you get the settings right. Uses mplayer and mencoder by default, though I think you can change it if you need to

You can output in .avi and .mpg

One thing to be aware of is target file size can be a little misleading...It doesn't always hit the mark, so if you care about making it, say, 700 MB on the nose, this might not be the program for you. Otherwise I love it

Good Luck
