### Post by squaregoldfish on 2008-12-21
Sounds like a dodgy keyboard mapping in Ubuntu (US/UK/other). Where is your \ key on your keyboard? It's usually next to either shift or the Enter key (above right shift). Try both keys and see if you get a backslash out of one or the other.

If you find the key on a different key, you'll need to figure out which keyboard layout you have and adjust the setting in Ubuntu accordingly.

Steve.

### Post by squaregoldfish on 2008-12-22
If you run gnome-control-center, and select the Keyboard option, there's a Layouts tab in the resulting dialog.

Click on the giant + icon, and you can select a new layout. This has the very nice feature of showing you what the new layout will look like, so you can check it before you select it.

Steve.
