I think there is an option somewhere in Kaffeine to set the cache, ( the amount of input video received before display - a buffer). Haven't used Kaffeine for years, though. If not, try mplayer.

I run a cache of 8192 which removes all problems with "reception", even when I have poor connection/reception.

### Post by shertiger on 2010-06-20
I should have stated that this problem was due to the signal strength being too high. SBS worked becuase it has a low signal strength in comparison to the other channels. I therefore reduced the signal by pulling the tv connectino most of the way out from the socket.
 
This worked until I moved nearer to the transmitting tower. I then added an attenuator to reduce the signal strength. Hope this helps anyone else solving a similar prob.
