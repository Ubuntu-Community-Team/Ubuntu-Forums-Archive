### Post by Gazhole on 2009-12-11
Banshee looks in the audio file itself for coverart first, then a few different websites like last.fm if there isn't any available.

If you get the EasyTag program you can add coverart to the ID3 tag of the mp3, and Banshee *should* be able to read it.

I dont think theres a way actually in Banshee itself to add it like in Amarok though. I would suggest you use Amarok, but personally i prefer Banshee :P

Hope that helps!
