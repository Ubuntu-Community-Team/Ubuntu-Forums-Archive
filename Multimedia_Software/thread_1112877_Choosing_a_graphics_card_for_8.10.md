### Post by Melcar on 2009-04-01
Any ATI/nvidia card is good.  Any nvidia card would do, it just depends on what you expect from it in terms of performance.  For ATI, the new 9.3 driver is the last one that will have support for pre HD2xxx cards, so if you get something older and want continued support for it (i.e plan to use newer distros with it) then you're going to have to start looking into the open source drivers; these drivers are very good for things like movie playback and some 3D (they can run Compiz and nearly all open source games) but lack 3D performance and are only opengl1.3.
