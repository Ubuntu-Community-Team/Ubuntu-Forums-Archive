### Post by Agent.Logic_ on 2009-04-07
Have you tried editing the xorg.conf file?


First, backup your xorg.conf file. Open up the terminal and type:
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup
```
Enter your password. This will store a copy of your current settings, so that you can recover your settings if something goes wrong.

Now, type the following in the terminal:
```
sudo gedit /etc/X11/xorg.conf
```

Enter your password. This will open up the X Server configuration file in the text editor. Look for the line that says:
```
Section "Screen"
...
...
EndSection
```

Before the EndSection (which follows Section "Screen"), type this:

```
SubSection "Display"
Virtual 1280 1024
EndSubSection

```

Save the file and reboot. This will set the resolution to 1280x1024. :D

In case it does not work, or something goes wrong, boot into recovery mode and rename the **xorg.conf_backup** file back to **xorg.conf** so that you can atleast continue your work with current settings.
