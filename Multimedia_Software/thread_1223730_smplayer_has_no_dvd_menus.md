### Post by rvm4000 on 2009-07-26
From version 0.6.7 smplayer has support for DVD menus, but it's disabled by default (it's experimental). Go to Preferences -> Drives and check the option "Enable DVD menus".

You'll probably need a recent version of mplayer too.

You'll find smplayer 0.6.8 and a recent version of mplayer here:
[https://launchpad.net/~rvm/+archive/testing](https://launchpad.net/~rvm/+archive/testing)
