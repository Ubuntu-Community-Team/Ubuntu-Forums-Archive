### Post by ajgreeny on 2015-08-27
You obviously did not read the link that I posted.

Make sure you do read it now, and follow the instructions there to install libdvdread4 and then run command ```
sudo /usr/share/doc/libdvdread4/install-css.sh
``` which will install libdvdcss2, needed to read and play just about all commercial encrypted DVDs.
