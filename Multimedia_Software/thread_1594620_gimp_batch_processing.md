### Post by lykeion on 2010-10-12
Gimp doesn't have built-in batch procesing. However people write Gimp plugins and for batch processing DBP seems popular. Look at the info/screenshots to see if it's what you're after:
[http://members.ozemail.com.au/~hodsond/dbp.html](http://members.ozemail.com.au/~hodsond/dbp.html)

It's included in the package gimp-plugin-registry, which I think it's in the repositories, use synaptic to install.

Good luck.
