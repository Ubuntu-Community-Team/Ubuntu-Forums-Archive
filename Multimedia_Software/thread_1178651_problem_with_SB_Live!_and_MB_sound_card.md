### Post by kostkon on 2009-06-04
You need to set your sound playbacks in your sound preferences (i.e. System &#8594; Preferences &#8594; Sound) back to *Autodetect* and then install the *PulseAudio Device Chooser* utility from *Synaptic* in order to set your Live as the default output device in your system.

For more info check [this thread here]("http://ubuntuforums.org/showthread.php?t=1130384").

If you have any question regarding the above, don't hesitate to ask here.
