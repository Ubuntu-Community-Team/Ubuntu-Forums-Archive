Speakit stops reading long text selections.
Many people seem to be having this problem. When using Speakit! in a Chrome browser, myself included. But not anymore. I made a few changes to my setting in Chrome and it seems to be working now.
Here is what I did: I use the iSpeech engine and not the speakit!
I then set the voice to US English Male and the speaking rate to x2 390 words/minutes.
