### Post by gfg on 2009-01-18
I strongly recommend Banshee for use with an ipod. You can use it to syncn if you want to, and it's also easy to manage it manually. One feature that I found very neat when I used my ipod was that Banshee would automatically convert my songs to a compatible format if I dragged over some FLAC's or other unsupported formats.

Edit: Oh and another thing I like about banshee when using it with an mp3 player is that it shows how much space is available on the device, in a nice graphical way.
