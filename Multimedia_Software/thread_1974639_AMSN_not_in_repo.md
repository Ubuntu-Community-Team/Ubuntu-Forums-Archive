### Post by ubuntu27 on 2012-05-06
Apparently Alvaros Messenger (aMSN) was removed from Ubuntu 12.04 LTS since aMSN will not offer security updates up to 5 years (Ubuntu 12.04 will be supported for 5 years as a Long Term Support).

You can install amsn from the previous Ubuntu (Oneiric).

> **immoT said:**
> Install amsn on 12.04

go [https://launchpad.net/ubuntu/precise/+source/amsn/0.98.4-0ubuntu3](https://launchpad.net/ubuntu/precise/+source/amsn/0.98.4-0ubuntu3)

1) Builds
precise i386 / precise amd64
2) Built files
download:
amsn-data_0.98.4-0ubuntu3_all.deb
amsn_0.98.4-0ubuntu3_i386.deb
3) Install first amsn-data, then amsn (by doubleclicking)

Warning: this way installing package have never automatic updates.

[http://ubuntuforums.org/showthread.php?t=1954154](http://ubuntuforums.org/showthread.php?t=1954154)

[http://packages.ubuntu.com/oneiric/amsn](http://packages.ubuntu.com/oneiric/amsn)
