### Post by jashstud on 2009-01-28
I have been able to get my graphics card working properly.
I found this wiki to be extremely helpful. 

[http://wiki.cchtml.com/index.php/Ubuntu_Intrepid_Installation_Guide](http://wiki.cchtml.com/index.php/Ubuntu_Intrepid_Installation_Guide)

I followed "Installing the Restricted Drivers Manually" from a fresh install and fully updated Ubuntu 8.10 64 bit. 

Everything seems to be working properly, save a resolution issue with my login screen. 
The login screen appears only in the top-left corner of the screen and fills only about 1/4 of the screen.
It is usable but does not look very pleasing. 

Once I log in everything seems to be working properly.
