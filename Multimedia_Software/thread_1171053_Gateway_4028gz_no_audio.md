First time to post, long-time reader...
I have a Gateway 4028gz laptop, fully updated Ubuntu 8.04 but cannot seem to get any sound.  I have tried the various options under System/Preferences/Sound.  The laptop appears to have an Intel 82801DB-ICH4 - IEC958 chip.  Everything else seems to work fine.  Any help would be appreciated.

Thanks!

Update:  fixed the problem... Had to disable the following switches under the volume control for ALSA:
Headphone Jack Sense
Line Jack Sense
External Amplifier

Thanks anyway!
