### Post by Guitar John on 2009-12-16
I read somewhere that Windows does not recognize spaces.  So if your file is called, for instance "my tune.mp3" you would have to change it to "my_tune.mp3"

If you go to Edit > [Preferences]("http://soundconverter.berlios.de/Screenshot-Preferences.png") and tick the box called "Replace all messy characters" GSC will do that for you.  I don't know if this is the problem, but it seemed like a good place to start.
