### Post by LinuXGo on 2010-01-01
Well there might be some problem with the sound configuration, go to sound perferences and select the input tab. The input volume shouldn't muted nor shouldn't it be amplified and make sure the connector is correctly set. Next we'll go to the output tab, make certain that the connector is set and is selected to have no amplifier. (Ex.Analog Output/No Amplifier)
