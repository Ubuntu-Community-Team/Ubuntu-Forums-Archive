### Post by ron999 on 2010-12-22
Hi

I don't know how to do the job from the command line.
If nobody else comes up with a suggestion then **kid3-qt** will do it with a GUI.

File > Open Directory

Then highlight all the files in the left-hand window.

Tools > Convert ID3v2.4 to ID3v2.3

Then File > Save
