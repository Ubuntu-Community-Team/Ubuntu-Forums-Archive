### Post by dano1 on 2009-06-08
for those with sound muted on startup this may help. Enabled user and root in haldaemon group under user settings. Now sound starts with no mute. 

Using jaunty 64, alsa 1.0.20 still no system sounds but may be attributable to the method used to update alsa.

Hope this helps others : ).

edit:
hmmm.switched default sound card card to pulse, all other sound settings to autodetect and i'm good to go-system sounds work along with digital output in myth. Only took 2 months of fiddling around-can't wait for the next release ; )
