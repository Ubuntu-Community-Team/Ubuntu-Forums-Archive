### Post by davidmohammed on 2012-07-29
Hi,
  I've been working on porting an old Rhythmbox v0.1x plugin called [RadioBrowser]("http://www.programmierecke.net/programmed/rhythmbox-radio-browser.html") to Rhythmbox v2.96.

Features include, playing and searching for internet radio stations & ripping mp3 tracks from those streams.

Help is needed to test this port - so if you are feeling nervous - please wait for a "stable" release :)

If you want to help out - or are just curious(!) - unpack the zip into the folder ~/.local/share/rhythmbox/plugins

If you are familiar with Git use:

```
git clone https://github.com/fossfreedom/radio-browser.git

```
Any issues - please use GitHub.

[https://github.com/fossfreedom/radio-browser]("https://github.com/fossfreedom/radio-browser")

Thanks in advance for any help in testing.

P.S. - if you are a python dev who can help out with this port - please checkout the Master branch.
