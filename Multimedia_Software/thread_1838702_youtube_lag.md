### Post by the_blue_box on 2011-09-04
Hi ZetCoby

I had this problem and lag issues on youtube and other things, what I did was fiddle with the settings in CompizConfig Settings Manager.
If you don't already have it installed...

```
sudo apt-get install compizconfig-settings-manager
```The only other thing I can think of is are you running an up to date vision of flash? If not it may be a problem.

Also something I found the other day is [[COLOR=DarkRed]**minitube**[/COLOR]]("http://flavio.tordini.org/minitube") [COLOR=Black]I watch all my youtube [/COLOR]videos on it.
It can be downloaded from the software centre.

I hope this helps
