### Post by techrolla on 2008-07-10
If you are having sound problems in hardy, there is a simple solution:
```

# apt-get remove pulseaudio
# apt-get install esound

```

I believe that is what I did.  My system works amazingly now.  I can watch flash videos in firefox with sound, and listen to rhythmbox at the same time.

Pulseaudio isn't ready for the primetime, I'm not sure why it was included in hardy.
