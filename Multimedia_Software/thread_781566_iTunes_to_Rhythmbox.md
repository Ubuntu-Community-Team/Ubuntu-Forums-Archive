### Post by Simian Man on 2008-12-09
> **Matthileo said:**
> 
As to that I have two questions.
First, is it possible to import an iTunes library file into rythmbox (to preserve my ratings)?

Rhythmbox can import folders full of mp3s with no problem.  As far as I know, it will not be able to import any application-specific metadata like ratings.

> **Matthileo said:**
> 
And second, is it possible to configure rythmbox to import music from cds as 160kbps mp3s and how would I do this?

Yes you can import as any format gstreamer supports.


> **Matthileo said:**
> 
(less importantly, an impromptu third question)
Does Rythmbox support album art imbedded into the song file like iTunes?
Yes!  You can also overide its choice if it gets it wrong.
