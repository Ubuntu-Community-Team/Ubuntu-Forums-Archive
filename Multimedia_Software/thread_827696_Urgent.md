### Post by ad_267 on 2008-06-13
Install the ubuntu-restricted-extras package.

Go to system - administration - synaptic package manager. Search for ubuntu-restricted-extras and tick the box next to this then click apply.

This package should contain everything you need.

Read here for more info: [https://wiki.ubuntu.com/RestrictedFormats](https://wiki.ubuntu.com/RestrictedFormats)
