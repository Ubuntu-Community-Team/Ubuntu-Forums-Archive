### Post by Lars Noodén on 2014-03-04
In Firefox, I would recommend avoiding flash all together and using the "YouTube Flash to HTML5" plugin:

[https://addons.mozilla.org/en-US/firefox/addon/youtube-flash-to-HTML5/?src=api](https://addons.mozilla.org/en-US/firefox/addon/youtube-flash-to-HTML5/?src=api)

As far as I can tell, it works on all the videos, at least the ones I've run across so far.
