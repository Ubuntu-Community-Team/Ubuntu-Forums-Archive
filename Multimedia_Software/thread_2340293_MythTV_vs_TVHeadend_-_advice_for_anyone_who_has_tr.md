### Post by deadflowr on 2016-10-17
It's less about which one is better and more of which one is easier to setup.

Mythtv is actually really easy to setup.
But it requires you to use myth's own setup programs (and typically locally on the machine at hand works best) and doesn't explain well on how to move from setting window to setting window.

(Hint: if a window doesn't explicitly tell you to press Okay or Cancel simply pressing ESC will exit the window, but with the settings you set for whatever that particular page set applied.
I think that might be confusing for most, as most assume from historical context that pressing ESC cancels whatever thingy you might have just tried)
But once you get pass that setting it up is fairly simple and easy.


TVHeadend can be set from anywhere from any web browser.
but it's settings page(s?) is far more (seemingly) convoluted, but also quite a bit more nuanced, imo.


As far as frontend, myth really needs  to run with a mythtv-frontend. And not some 3rd party program.
(all 3rd party plugins or addons I've ever used with myth as the backend have been of lesser quality or lack the full capabilities that the true frontend has.)

As far as I know, TvHeadend has no frontend, per se.
so most 3rd party programs for it work quite well.
Since their is no standard bearer as to what features the frontend should have or not have. There is no gauge to judge them against. ( I guess)

But at the end of the day, once you setup the backend and tuner cards properly, they both run smooth.
Though my TvHeadend experience is far more limited to my myth experience.


Finally, alternatively In my own opine, if the desire is to run and watch tv on a single machine I would suggest neither myth nor TvHeadend and instead suggest lighter fare such as me-tv, smplayer, or kaffeine.
All 3 of these are extremely easy to setup and far superior in terms of user-friendliness compared to either myth or TvHeadend.


I doubt any of this will actually help.
Since it really comes down to your own personal preference.
But alas, no harm in trying out several or all options.
Good Luck

### Post by all_urkam on 2016-10-18
I had used mythtv for several years, and find it vastly (!!!) more difficult to configure than tvheadend. Having to use the mythtv-frontend for configuration is frustrating most of the time. My wife never liked or used it.

Now with tvheadend in the back and Kodi at the front, it is (almost) as easy as the good old tv set. All settings can be done at one place via browser.

Also, keep in mind that mythtv still CANNOT use the modern SAT>IP technology (which I'd strongly recommend to consider). I had asked the support question for sat>ip on the [mythtv forum]("https://forum.mythtv.org/viewforum.php?f=3")  more than 2 years ago - it is by far the most viewed topic on the forum - but no solution is in sight ([https://forum.mythtv.org/viewtopic.php?f=3&t=226](https://forum.mythtv.org/viewtopic.php?f=3&t=226) ).  

So, if you want sat>ip, then mythtv is NOT an option! Tvheadend does play very well with sat>ip
