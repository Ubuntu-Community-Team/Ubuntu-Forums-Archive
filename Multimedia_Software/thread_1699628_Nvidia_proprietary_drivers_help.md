### Post by tommcd on 2011-03-04
What version of Ubuntu are you using? You really should at least tell us what version of Ubuntu you are using and what nvidia graphics card you have.
I am assuming here that you are using either Ubuntu 10.04 or 10.10. 
If you have a GForce 6 series or newer card, install *nvidia-current*. [http://packages.ubuntu.com/maverick/nvidia-current](http://packages.ubuntu.com/maverick/nvidia-current)
 A GForce 5 series card would use *nvidia-173*. [http://packages.ubuntu.com/maverick/nvidia-173](http://packages.ubuntu.com/maverick/nvidia-173)
Write back if you need more help.
