### Post by ajgreeny on 2009-09-28
If the screen just suddenly goes to black, and then back to the login screen, it could be an overheating problem.  See if you can get any sensor package to run, such as **lm-sensors** or **mbmon** to see if temperatures are low enough.  Also check that the machine is not clogged up with fluff, and is sitting on a hard surface, not a soft fluffy one, which can block airways easily.
