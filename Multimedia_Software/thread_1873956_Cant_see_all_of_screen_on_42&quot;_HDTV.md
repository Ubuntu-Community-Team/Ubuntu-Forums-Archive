### Post by mitchell1992 on 2011-11-02
well i somehow fixed t. i used the overscan tool when i downloaded sysinfo from software center

for anyone else

sysinfo>NVIDIA>NVIDIA Display Settings>DFP-1 (GKK SONIQ) (I went through the tabs on left where mine said this) > go to overscan compensation where i found mine to be good at 114.
hope this helps anyone with the same problem
