### Post by AgentLinux on 2010-06-21
After installing Ubuntu 10.04, to my knowledge I had no audio, however I managed to fix it eventually.

Solution:
a) Adding data to a config file via the terminal.
b) Making sure the "Sound Preferences" are set correctly.

---
a) Open a the following config file in a text editor from the "Terminal" application, found under Applications --> Accessories 

gksudo gedit /etc/modprobe.d/alsa-base.conf

At the very end of the file, adding the line:
options snd-hda-intel model=6stack

Then save the file and reboot.


b) Setting the Sound Preferences:
Connecting the headphones to rear side of the computer cabinet. (Atm, I do not get audio from the front of the cabinet, I might have disconnected the mainboard cables due to noise some time ago.)

In the Sound Preferences: setting Output = Internal Audio Analog Stereo
In the Sound Preferences: setting Output --> Connector = Analog Output
In the Sound Preferences: setting Hardware --> Internal Audio = Analog Stereo Output

The "6stack" stated in the option, was found on this list.
[http://www.kernel.org/doc/Documentation/sound/alsa/HD-Audio-Models.txt](http://www.kernel.org/doc/Documentation/sound/alsa/HD-Audio-Models.txt)
I guess "6stack-digout" might work as well, though I have not tried it.
