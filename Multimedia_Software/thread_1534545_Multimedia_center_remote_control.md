### Post by steefjeqv on 2010-07-21
Hi,

I've succesfully used an X10 remote from ATI (Remote Wonder) using lirc to control my HTPC (which is a VDR setup).

Whenever I configure an Ubuntu-VDR, I just install the vdr-remote-plugin and use the remote that comes with the TV card. Thus avoiding the lirc stuff.

If you're looking for a dedicated HTPC have a look at this :

[http://www.yavdr.org/]("http://www.yavdr.org/")

You can also use their Launchpad repo to upgrade a regular Ubuntu.
