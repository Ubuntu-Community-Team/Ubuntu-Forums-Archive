Since the archived thread has never gotten any real answer but the issue itself is still very much existent ; please anyone?

I believe this is actually a generic flaw ; why do we put the graphic equalizer in single programs on the PC while it's much more logical to apply the eq to the sound output stream?
Think about it ; how does your HiFi or car-stereo EQ work? It's setting applies to CD play, radio tuner, external input, etc. One EQ setting on the device for all sound sources.
Then look at the PC ; all kinds of sound mangling on every sound source, while it would make MUCH more sense to have it system-wide.

To speak for my own needs ; the last.fm player does not have an EQ ; Rhytmbox doesnt have (a decent) one. So why solve it a thousand times? Solve it in the system. Simple. Clean.

Pretty pretty please? ;-)
