### Post by walterav on 2017-05-13
Which desktop environment are you running Unity/Mate/Gnome/KDE since it looks like the key combination is used by Workspace Switcher of that environment. If you are lucky you will find a way somewhere under Ubuntu System Hardware Keyboard shortcut to change it, other option is to tweak the windows manager like compiz/compton which listens for keyboard shortcuts and disable that specific option.

### Post by Bucky Ball on 2017-05-13
You could try changing the key combo. Open Blender> File> User Preferences> Input and you can change it there. Click the arrows next to the headings on the right.

In the left-hand column in 'Input' there is 'Select with' and you can choose to select with right or left. Presuming this refers to the mouse button, but I could be wrong. Any use?
