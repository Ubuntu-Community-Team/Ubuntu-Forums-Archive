### Post by TheFu on 2014-01-11
"best method"?  Well, that is highly subjective.

I have a dedicated XBMC/Linux machine with lots of extra plugins to support streaming of non-DRM video.  That probably is NOT the best method for you.  To much effort to get it up and working for most people.  You can load XBMC onto Ubuntu to get a feel for it, if you like.  

VLC is an option, but less user friendly. Entering URLs isn't fun for anyone.

For an average computer user, using a web browser is the "best method" - since it is easy.
