### Post by oldtraveler on 2009-01-12
After my new installation of Ubuntu 8.10 I notice that mid files seems to play best using RythymBox Music Player while GNOME player or Movie player is best for wav, mp3, avi, or wmv files.

Can I somehow set these applications to be the default players? I would like for them to also open embedded files of those types.

---

### Post by forger on 2009-01-12
The first time you right click over the file and select "open with other application..." and choose your application.
To make it a default, you right click over the file and choose Properties -> Open With
