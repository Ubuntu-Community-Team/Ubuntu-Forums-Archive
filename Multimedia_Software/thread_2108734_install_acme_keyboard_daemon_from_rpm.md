### Post by opiumpoetry on 2013-01-25
1. make sure Universe repo is enabled then run [COLOR="Blue"]sudo apt-get update[/COLOR] in Terminal.

2. install Alien rpm-to-debian converter with [COLOR="Blue"]sudo apt-get install alien[/COLOR] in Terminal

3. open folder where you downloaded rpm package, i.e. [COLOR="Blue"]cd Downloads[/COLOR] in Terminal

4. now run [COLOR="Blue"]sudo alien -i file-name.rpm[/COLOR] in Terminal

*download the rpm package here _[http://rpm.pbone.net/index.php3/stat/4/idpl/19855780/dir/pclinuxos/com/acme-2.4.3-10pclos2010.i586.rpm.html](http://rpm.pbone.net/index.php3/stat/4/idpl/19855780/dir/pclinuxos/com/acme-2.4.3-10pclos2010.i586.rpm.html)_

*more info here _[http://www.unixmen.com/convert-rpm-to-deb-file/](http://www.unixmen.com/convert-rpm-to-deb-file/)_ and here _[http://www.unixmen.com/howto-install-rpm-packages-on-ubuntu/](http://www.unixmen.com/howto-install-rpm-packages-on-ubuntu/)_
