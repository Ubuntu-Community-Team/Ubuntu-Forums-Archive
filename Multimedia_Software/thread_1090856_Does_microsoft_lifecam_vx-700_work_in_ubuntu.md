I just bought the Microsoft VX-700 at target. For the record, it is supported by XawTV, but XawTv can't capture from it. A program called luvcview available from synaptic that can capture stills, but no luck with movies. None of the other webcam programs available from synaptic worked with it...So my thought is that its either not supported fully YET or just not fully supported.

edit: A little more poking around - luvcview is actually part of a [Logitech project]("http://www.quickcamteam.net/software/linux/v4l2-software/luvcview") for Linux support for their webcams, which means they care about OSS, so I would probably go with a Logitech webcam.

### Post by Khar136 on 2010-06-02
I just bought a Microsoft LifeCam VX-700 (because it is so cheap).

Running Ubuntu 10.04 and with the VLC media player you can at least see the image.:P

Just run VLC, Click Media, Open Capture device, Play.

Also, the Cheese program (install via package manager) showed me what my webcam sees, and was able to take pictures and record a movie (although even on a fast computer it seemed to be struggling a bit).

Camoramo does not appear to be able to connect.
