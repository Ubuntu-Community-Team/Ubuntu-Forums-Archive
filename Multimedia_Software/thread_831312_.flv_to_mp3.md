### Post by lizzard on 2008-06-16
ffmpeg can do the job. It's in the repositories. To convert an flv file into a mp3-audio file just do: ```
ffmpeg -i inputfile.flv outputfile.mp3
```

### Post by Fingers &amp; Thumbs on 2008-06-16
I use a very nifty utility for audio conversion. It adds a "Convert audio File" item to the right-click context menu, however I have never used it with flv's it has never let me down with anything else.

To obtain it you will need to have universe enabled in the repos, and then install nautilus-script-manager, and nautilus-script-audio-convert. Either using synaptic or sudo apt-get.
