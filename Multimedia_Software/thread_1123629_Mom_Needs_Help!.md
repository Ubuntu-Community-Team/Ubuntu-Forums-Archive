### Post by Bucky Ball on 2009-04-12
You might find Kino good. You can find a lot of help for it too:

[http://www.kinodv.org/](http://www.kinodv.org/)

It is in the repositories. Cinelerra is also good, but not layman. You might like to have a look though. They are attempting to match Final Cut and Premiere I suppose:

[http://cinelerra.org/](http://cinelerra.org/)

This is not available in the repositories. You need to add to the software sources to make it so. 

The main issue is getting the camera to work, but if you're lucky and everything installs okay, should be fine. All depending on which version of Ubuntu you are using, getting permissions to use the camera is easy or hard! You are probably going to need Firewire (IEEE1394) cable and appropriate socket on your computer and camera to get the video in. Install Kino, plug it up and see what happens (switch the camera on, then boot Kino).

Hope that helps and good luck. :)
