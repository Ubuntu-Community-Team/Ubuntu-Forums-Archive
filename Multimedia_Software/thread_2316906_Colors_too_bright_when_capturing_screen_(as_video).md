_**I've had an issue with gtk-recordMyDesktop:**_
[http://ubuntuforums.org/showthread.php?t=1132929](http://ubuntuforums.org/showthread.php?t=1132929)

Very similar to this person's issue in this thread which didn't get an answer. I just want to let all know I was doing my playback in GNOME Mplayer and noticed my video output wasn't set. I'm using Lubuntu 15.10 and recordMyDesktop. There are no settings to change the actual color or brightness other then the recommended video quality (left it @100) *in RecordMyDesktop*. I actually changed the video output in GNOME Mplayer [Edit>Preferences>Video Output = x11] 
[B]
Now the super bright .ogv video clip is now working showing my desktop as it was recorded. [/B]

Just posting this since I didn't see this other thread with an answer and Google didn't have anything else listed for this issue.
