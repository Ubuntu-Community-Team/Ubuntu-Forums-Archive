When you say the "latest and greatest" version of compiz I presume your talking about the git version.Thats not really recomened unless you know what your doing as its for development and is buggy.Compiz comes preinstalled with ubuntu.

About the nvidia driver, think of it as playing games in windows.You need to have the driver installed or the game wont work.Simply go System --> Administration --> Hardware Drivers and tick the boxes.

This should explain things:

[https://help.ubuntu.com/community/CompositeManager/CompizFusion](https://help.ubuntu.com/community/CompositeManager/CompizFusion)

About the window boreders not showing, try ticking the window decerator box in advance desktop effects settings.
