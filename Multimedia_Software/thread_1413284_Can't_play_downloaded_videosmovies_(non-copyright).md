### Post by lovinglinux on 2010-02-22
There are open source alternatives, swfdec and gnash, but unfortunately  they don't work on several sites.

Most users will recommend to install the proprietary flash plugin from Adobe.

For 32bit see [this](http://lovinglinux.megabyet.net/?page_id=220#Removing-Conflicting-Plugins-2).

For 64bit see [this](http://ubuntuforums.org/showthread.php?t=1358591).
