To do this in Mint Linux and probably several other non standard ubuntu-desktop editions in the volume control panel you have the option for preferences select that and select External Amplifier which will add a switch tab to your volume control panel. Then in that switches tab uncheck external amplifier. It worked for me and I was having the same problem with dual boot where my regular ubuntu-desktop worked fine after switching the output device but my other desktops didnt have the same options
