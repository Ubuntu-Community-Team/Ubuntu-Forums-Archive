### Post by cariboo on 2008-09-26
If your dvd player plays .avi's then you just burn a dvd data disc. If you need to convert the files to a dvd format then you can use devede or tovid. Both packages are in the repositories and you can use Add/Remove Programs, Synaptic Package Manager and of course the command line to install them.
