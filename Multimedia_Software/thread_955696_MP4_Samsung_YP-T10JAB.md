### Post by kostkon on 2008-10-22
According to [its specs]("http://www.samsung.com/uk/consumer/detail/spec.do?group=homeentertainment&type=mp3digitalaudio&subtype=yp_t10&model_cd=YP-T10JAB/XEU&fullspec=F"), it is a [*MTP*]("http://en.wikipedia.org/wiki/Media_Transfer_Protocol") based player. Thus, you could use the media player that comes with *Ubuntu*, *Rhythmbox*, to put songs in your player.

You should make sure that the *MTP* plugin is enabled in *Rhythmbox*, by selecting *Edit -> Plugins*.

If you want to put videos in your device , you will not be able to do it using *Rhythmbox*.

For that, you could use [*Gnomad2*]("http://gnomad2.sourceforge.net/") or maybe [*Banshee*]("http://banshee-project.org/download/") (not sure about this. But you will need to use its latest version. Installation instructions are on [its site]("http://banshee-project.org/download/")). You can install *Gnomad2* using *Add/Remove* or *Synaptic*.
