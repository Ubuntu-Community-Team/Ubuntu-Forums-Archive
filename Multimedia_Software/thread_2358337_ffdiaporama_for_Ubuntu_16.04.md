### Post by wangji on 2017-05-29
you can now run any ffDiaporama version 2.1  or 2.2_devel on any mint-17/18  or any ubuntu.


[https://sourceforge.net/projects/toysbox/files/ffDiaporama-2.1-standalone/OldWineNewBottle-ffdiaporama-2.x-2014.deb](https://sourceforge.net/projects/toysbox/files/ffDiaporama-2.1-standalone/OldWineNewBottle-ffdiaporama-2.x-2014.deb)

[https://sourceforge.net/projects/toysbox/files/ffDiaporama-2.1-standalone/OldWineNewBottle-v2ffdiaporama-2.x-2014-amd64.deb](https://sourceforge.net/projects/toysbox/files/ffDiaporama-2.1-standalone/OldWineNewBottle-v2ffdiaporama-2.x-2014-amd64.deb)

[https://sourceforge.net/projects/toysbox/files/ffDiaporama-2.1-standalone/AnyWhere-ffdiaporama-2.x-2014-amd64.tgz](https://sourceforge.net/projects/toysbox/files/ffDiaporama-2.1-standalone/AnyWhere-ffdiaporama-2.x-2014-amd64.tgz)

 



   Either unpack AnyWhere...tgz  , then run /path_to_extractedAnyWhere/ffDiaporama/ffDiaporama



  or 

                   sudo dpkg-i OldWine...deb ; then execute in terminal: ffDiaporama 

 
   enjoy
