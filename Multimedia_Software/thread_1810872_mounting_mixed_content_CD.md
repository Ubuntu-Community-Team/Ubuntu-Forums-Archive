### Post by ExileAmongYou on 2011-07-23
I think the old bugs about mixed content CD's are more to do with those weird cases where someone has made a CD that is half audio CD and half data CD. Having a bunch of different file types (including audio) on a data CD is not the same thing, and shouldn't really cause any problems. It's likely that the CD didn't burn right.
