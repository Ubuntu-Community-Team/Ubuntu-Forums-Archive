### Post by tea for one on 2020-05-31
Rhythmbox 3.4.4 supplied with Ubuntu 20.04 has a setting where you can have **gapless** playback.

Especially useful for playing CDs recorded as live performances where one song melds into another.

Rhythmbox > Preferences > Playback > Checkbox Crossfade between tracks > Crossfade duration set to 0 seconds.
