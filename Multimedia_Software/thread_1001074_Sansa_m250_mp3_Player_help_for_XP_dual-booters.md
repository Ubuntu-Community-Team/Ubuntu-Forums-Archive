### Post by jardenp on 2008-12-03
I was going to add this to a similar thread about Sansa players and Amarok, but my problem didn't involve Amarok. Everything was fine with with my Sansa m250 until my wife put her workout music on it on our Windows machine and then I had to use a Windows machine to access work files on it. It basically turned the mp3 player into a "dual boot" if you will--what I could access on the Ubuntu machine wasn't visible on the Windows machine and vice versa. Before sharing my apoplectic reaction to what I thought was a reformatted drive (Goodbye work files! Goodbye Dennis Miller Show mp3s!) I searched these forums and someone pointed out the MSC/Autodetect USB settings difference. I found that my Ubuntu machine automatically takes it as MSC, but the Windows machine doesn't. So, to view the other half of the mp3 player--the half you put on there with Ubuntu--on a Windows machine, go to the players Menu --> Settings --> USB --> MSC or Autodetect. I don't know if I can access my wife's music with Ubuntu, but I'm alright with that. Hope this helps someone!
