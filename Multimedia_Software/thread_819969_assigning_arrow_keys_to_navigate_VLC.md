### Post by chewearn on 2008-06-06
Under VLC Settings > Preferences menu, find Interface > Hotkeys settings

Make sure when you start changing, you don't inadvertently set two actions to the same key combinations (the key combinations will not work if this happens, and VLC won't tell you what's wrong).
