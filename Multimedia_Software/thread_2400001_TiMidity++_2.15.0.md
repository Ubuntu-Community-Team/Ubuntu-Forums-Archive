### Post by Yellow Pasque on 2018-09-02
This is the wrong forum/site to ask for that. This site is for users supporting users.
You can file a wishlist bug on Launchpad, preferably using apport:
```
ubuntu-bug timidity
```
Or using the Launchpad web interface if you find that easier: [https://bugs.launchpad.net/ubuntu/+source/timidity/+filebug](https://bugs.launchpad.net/ubuntu/+source/timidity/+filebug)

I would not get my hopes up that 2.15 will appear in Ubuntu 18.10. It was just released and has yet to be packaged upstream in Debian: [https://tracker.debian.org/pkg/timidity](https://tracker.debian.org/pkg/timidity)
Also, Ubuntu 18.10 is past the Feature and Debian Import Freeze (August 23): [https://wiki.ubuntu.com/CosmicCuttlefish/ReleaseSchedule](https://wiki.ubuntu.com/CosmicCuttlefish/ReleaseSchedule)
