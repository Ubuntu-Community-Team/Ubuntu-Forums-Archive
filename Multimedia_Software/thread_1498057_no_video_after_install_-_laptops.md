### Post by Mark Phelps on 2010-06-01
This kind of problem is exactly why (1) you should NEVER force a new install on untried hardware and (2) LiveCD mode was invented.

Had you used LiveCD mode, you would have had a chance to try out the new OS version to see if there were any problems.

To help with diagnosing these, we would need to know the video cards/chipsets at work in each laptop.  So, boot with an Ubuntu LiveCD, open a terminal, and enter"sudo lspci" for each -- and post the results back here.
