Have a look at this post re a £22 WandTV stick: I have not tried but it look promising. Please try for yourselves. Uses VLC for H264. 

[https://www.dealextreme.com/forums/Default.dx/sku.8325~threadid.278942](https://www.dealextreme.com/forums/Default.dx/sku.8325~threadid.278942)

Based on the basic post::  [http://www.dealextreme.com/details.dx/sku.8325](http://www.dealextreme.com/details.dx/sku.8325)

"
talas 
My WandTV has the EC168 chipset and now works under Ubuntu! 

I did this on a Ubuntu system (9.04 x86_64) 

First install Mercurial: 

sudo apt-get install mercurial 

Then get the source for the driver and the v4l and dvb-system with mercurial: 

hg clone [http://linuxtv.org/hg/~anttip/ec168/](http://linuxtv.org/hg/~anttip/ec168/) 

When it completes downloading, go into the folder ~/ec168/ and run make. )takes some minutes....)

Hopefully the compiling goes well, then you will have a lot of modules in the ~/ec168/v4l/ subfolder. 

Then you have to download the firmware from [http://palosaari.fi/linux/v4l-dvb/firmware/ec168/dvb-usb-ec168.fw](http://palosaari.fi/linux/v4l-dvb/firmware/ec168/dvb-usb-ec168.fw) 

( This file can also be found on your driver CD, called EC168BDA.bin. This has to be renamed to dvb-usb-ec168.fw ) 

Put this file in /lib/firmware/ 

Then load the following modules from the ~/ec168-2/v4l/ folder: 

sudo insmod dvb-core.ko 
sudo insmod dvb-usb.ko 
sudo insmod ec100.ko 
sudo insmod mxl5005s.ko 
sudo insmod dvb-usb-ec168.ko 

Fire up VLC, open capture device, set it to dvb, select frequency of your local transponder, and enjoy the show."
 
Freq. set to 722000 kHz and bandwidth 8 Mhz. for our example.

### Post by geh on 2009-05-06
The Pinnacle 73e works fine in Ubuntu 9.04. Fresh install, and the loaded the firmware:

If you plug in the USB stick and get an error message like "firmware not found", you have to download the firmware:

cd /lib/firmware
sudo wget [http://www.wi-bw.tfh-wildau.de/~pboettch/home/linux-dvb-firmware/dvb-usb-dib0700-1.10.fw](http://www.wi-bw.tfh-wildau.de/~pboettch/home/linux-dvb-firmware/dvb-usb-dib0700-1.10.fw)

This downloads one file into the /lib/firmware directory. Now plug in the stick again, and it should load fine (without the "firmware" error message), i.e. you get an adapter0 if you do

ls /dev/dvb*

Then, dowloaded latest version of VLC and it played perfectly the H264 channels "freeview" in Norway (NRK). Open Capture Device, freq. 722 000 000 Hz, 8000 (KHz) and play. Off you go.

Perfect. Now all I need is to get VLC to search channels if you do not know the frequency. Will see, no big deal.

### Post by seventeener on 2010-10-03
Norwegian DVB-T broadcasts can be watched with the VLC player, here is my HOWTO:

**OS:** Ubuntu 10.04

**TV-TUNER CARD supporting DVB-T:** mine is the Terratec Cinergy HT PCMCIA, bought in 2007. No driver installation is needed, the card is recognised automatically.

**ANTENNA:** it's very important to have a proper antenna. This is one of the reasons why I couldn't find the broadcasts earlier. I live in the line-of-sight with the local transmission tower, this should be a very good area for reception (in Norway you can find the nearest tower on [this map]("http://www.ntv.no/Dekningskart")). I'm using a very simple antenna that came bundled with the card: just a 29 cm metal telescopic stick. And only at a particular angle and a particular length of the stick (about 20 cm) could I tune to the signals! It took many try-and-fail attempts with various antenna positions before a scan was successful. The broadcasting company recommends using at least a 10-element antenna [here]("http://www.rikstv.no/Kundesenter/Antennehjelp/").

**SOFTWARE CONFIGURATION:** first install necessary packages:
```
sudo apt-get install vlc ffmpeg w-scan ubuntu-restricted-extras
```
Then run w_scan:
```
w_scan -ft -c NO -X >> channels.conf
```
This will produce a file channels.conf, where the found channels are listed, if any. Now run the VLC player and open the file channels.conf from the player. You'll get a playlist with the channels. 

**WHAT YOU GET:** only the NRK tv and radio channels are sent open. The rest are protected and cannot be seen without a subscription, [read here]("http://www.ntv.no/modules/module_123/proxy.asp?D=2&C=19&I=1203&mid=168"). Both video and audio on the NRK channels work great. NRK1 comes in 1280x720, NRK2 and NRK3/Super in 720x576 resolution. Subtitles can be turned on and off.
