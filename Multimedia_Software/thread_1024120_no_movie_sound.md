### Post by namdung on 2008-12-30
I reckon u have installed all the codecs required to play the audio/video files. If not install the ***GStreamer ***codecs from *Add/Remove Applications*.

Is it a particular type of movie file or all movie formats having this issue?

Which application r u using to play the movies? Have u tried with VLC Player? 

Sometimes an application may lock the ALSA, so close all applications (including Firefox if Flash is playing) and try playing movie again.

Is audio playing fine? Even mp3, wav, flac ?

Have u checked the *Sound Preferences-->Music & Movies*? Usually *Autodetect* works fine, else try ALSA.

Sound Card? Ubuntu version?
