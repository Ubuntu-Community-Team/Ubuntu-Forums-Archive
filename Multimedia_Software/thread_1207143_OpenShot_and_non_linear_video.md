### Post by pdxken on 2009-07-10
I don't know about openshot but I finally just yesterday figured out how to edit videos.
:popcorn:
About a year ago I tried several and just got frustrated and lost interest.
This week the need became stronger so decided to give it a go again.

After installing kdenlive using synaptic the same old frustration came back until I discovered what I did wrong.
kdenlive is set by default to Hi deff. All I had to do was check the DV box instead of HDV and it works great! My computer is an older P4 2Gig. that seems too old for HD.

So far I have only tried a few clips but it seems to work fine with what I did, (attached several clips together with a title page with transitions and render it all down to an .mp4).
Try google kdenlive there is a nice review by Linux Magazine.

Ken.
