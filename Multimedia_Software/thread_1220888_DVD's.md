### Post by m_duck on 2009-07-23
[Playing restricted formats in Ubuntu](https://help.ubuntu.com/community/RestrictedFormats)

The main package that is needed for playing DVDs in libdvdcss2. In Ubuntu, the easiest way to get this is to add the medibuntu repository and install it like any other package.

[Medibuntu](http://www.medibuntu.org/)

I also tend to install ubuntu-restricted-extras, w64codecs (**or** w32codecs if using a 32 bit version of ubuntu) and vlc media player.
