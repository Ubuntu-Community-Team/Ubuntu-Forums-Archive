### Post by boombox1387 on 2010-06-09
> **Atermoon said:**
> Try Banshee with the Mirage extension (the package is called *banshee-extension-mirage*).

I second the vote for Banshee + Mirage.  Banshee is a pretty great media player, and if you use the Mirage extension with Banshee's Play Queue (use the 'Fill by Similar' option), it usually does a pretty good job of automatically picking similar music.

Banshee can also make good use of a song's Score, which is a number between 1-100 that reflects how much you've played and skipped the song.  If you choose to shuffle by score, Banshee will pick songs "randomly," but actually favor songs with higher scores, and it will get smarter as you skip songs.  This is a neat feature that I don't think iTunes has (at least it didn't when I last used it, back in the 7.x version).

### Post by Lrpbpb on 2010-06-09
Just as a side note for those who are new to Ubuntu, in order to install Banshee just go to the Software Center, and to install the Mirage plugin just fire up a terminal window (Applications>Accessories>Terminal) then just copy and paste the following into it (remember, to copy/paste inside a terminal window you use Ctrl+Shift+V/C, not just Ctrl+C/V): ```
sudo apt-get update banshee-extension-mirage
```
Press enter, restart Banshee if you had it open and voila.

P.S:I know you mentioned you liked rhythmbox, but I personally feel banshee is better (function wise). And Banshee developers are working to soon add full iPod/Phone support to banshee, similar to what currently exists in rhythmbox.
