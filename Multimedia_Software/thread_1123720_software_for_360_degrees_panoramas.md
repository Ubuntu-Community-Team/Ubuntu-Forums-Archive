### Post by conualfy on 2011-05-13
For simple image panoramics try Hugin.

It is a great peace of software, it can do the job by itself (only  needs autopano as a plugin). I just tried it in Ubuntu 11.4 and it works  like a charm. Use the Ubuntu Software Center to install it and use the Assistent tab to Load images, then do the Align, and finally press Create panorama.
