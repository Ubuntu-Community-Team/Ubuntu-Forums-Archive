### Post by Jonners59 on 2011-01-03
OK, for my next project I am looking to improve the media experience at home.

We have 1 central PC running 10.10 that we store all our docs and MM on.  It also has a Tv card in it, not set up yet.
We have another 5 PCs & Laptops of various types all running 10.10 and have local media player capability using VLC.  It is not very smart.
We also have all the usual other devices hanging off, such as Wii and a networked Tv...  I do have a Slingbox, to connect analogue media, but that does not seem to want to work any more.

OK, so what would I like to do...?
I think, what I would like is an app on the central PC that can sniff out all the media on the LAN, and file it in an easy to read and rate library that can be viewed by any machine, including the Tv.
I would like it to record and stream videos, DVDs, and Tv again library and stream

One of the problems we have is home videos are taken in portrait and when viewed remotely we can not rotate.  Would like to be able to resolve this too.

Remote control...

Access via a VPN for mobiles/remote location, such as on hols to access all the media and have it streamed.

I do not know what else a media server should do, but suggestions are sought on the server app, and how to do, etc.
