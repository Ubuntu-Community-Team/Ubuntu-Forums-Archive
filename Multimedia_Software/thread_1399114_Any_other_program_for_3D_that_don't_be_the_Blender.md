### Post by adam-red on 2010-02-06
2.5 alpha is very stable and has an updated interface which should bring in the windows crowd (the current interface is still based on IRIX). The new one looks very much like XSI and is much more friendly.

[http://www.blender.org/development/release-logs/blender-250/](http://www.blender.org/development/release-logs/blender-250/)

A lot of the VTMs on BlenderCookie are done in 2.5 now as preperation for the shift and theres a beginners series on modelling an alien from scratch starting with this one:

[http://www.blendercookie.com/2009/12/01/box-modeling-an-alien-character-in-blender-2-5-alpha-0/](http://www.blendercookie.com/2009/12/01/box-modeling-an-alien-character-in-blender-2-5-alpha-0/)
