### Post by cariboo on 2008-09-18
IF you haven't installed it you need **unbuntu-restricted -extras** to install them in a Applications-->Accessories-->Terminal type:

```
sudo apt-get install ubuntu-restricted-extras
```

This install the codecs you need to listen-to mp3's, watch videos, intall flash and java, as well MS core fonts.
