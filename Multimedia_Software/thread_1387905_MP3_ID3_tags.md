### Post by bravo sierra echo on 2010-01-22
I make CDs full of MP3s to play on my in-car CD player.

Unfortunately, it can only read ID3v1 tags - not the later v2 ones.

There are many programmes out there which edit ID3 tags, but one track at a time.

Can anyone recommend one with a batch editor, so that I can set it to work on my Music directory and have it change the ID3 tags on every single track to V1?

At the moment some of my music has v1 tags and some v2, so sometimes my car stereo displays track names and sometimes it doesn't.  Most annoying.

### Post by myth1914 on 2010-01-22
Musicbrainz.org has a good one.  Picard is the software.  It can be installed with synaptic as well.

I've also purchased a lic for Jaikoz (lin/win) and it has the Picard functionality along with other features.  IMHO, it was well worth the $28 or so for the lic.  Word of warning if you choose this route...  Only run batches of 10k mp3s/session
