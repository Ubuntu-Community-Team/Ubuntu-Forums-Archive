### Post by gordintoronto on 2010-09-23
Using Ubuntu 10.04, in Sound Preferences, the input tab is for your mic. There's a mute checkbox and a volume control.

There's another program called alsamixer which you could install from Administration/Synaptic. It runs from the command line, make the Terminal window big first. I like it because you can see "everything" at once.
