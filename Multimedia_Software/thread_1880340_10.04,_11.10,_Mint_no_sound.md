### Post by MoreOrLess on 2011-11-13
If you're using the open-source radeon driver, you have to manually enable hdmi audio (it's not the default option because it breaks some systems).
Manually add this to your boot line:
```
radeon.audio=1
```
If it works, make the change permanent in /etc/default/grub .
