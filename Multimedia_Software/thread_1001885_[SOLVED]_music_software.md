### Post by sirjoebob on 2008-12-04
I throw my vote in for Amarok, It is all I use for playing music. It has an itunes-like interface and is easily customized. Great thing about Amarok is that there is an information tab that automatically checks the artists entry on Wikipedia for info on them so it is easy to get some background while listening to music. Also, with Amarok you will need to install k3b to burn cds. IT is really easy to do but there are good tutorials to install k3b for amarok in ubuntu, Google is your friend.

More on the video side but works for audio to is VLC. VLC is great. Also, I would recommend adding the Medibuntu repository for easy access to restrcited codecs for things like mp3 playback. A link to that is in my signature.
