Step1: make sure you've got libdvdcss2 installed.
You can grab it from the medibuntu repository [here]("http://www.medibuntu.org/")

Step2: copy disc from your gnome desktop to make an iso image (that some players like smplayer will happily play), or burn the image to a backup DVD for safe keeping - so your original DVDs don't get scratched if you want to watch a movie.
