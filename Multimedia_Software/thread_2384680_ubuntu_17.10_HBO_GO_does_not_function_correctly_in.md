### Post by tinylagarto on 2018-02-13
The Flash issue should be independent of Wine.

Did you allow Flash in both Chrome and Firefox? Either from the settings or on the website itself.

Chrome comes with the Pepper Flash plugin included by default and for Firefox, if not already done, you need to install the Flash player from the repositories. The package is called:

```
flashplugin-installer
```
