### Post by Vladlenin5000 on 2015-03-03
Sorry, there's really nothing you can use for such purpose with a Chromebook regardless of the OS. It simply isn't powerful enough. You need to think about a different computer before you even start thinking about the capture device.

Use the Roxio's card system requirements as a guide:
> Intel® Core 2 Duo Processor 1.8 GHz or AMD Athlon x2 64 2.0 GHz, 2GB RAM
> **Dual Core 2.3 GHz above is recommended for 1080 30p/60i recording**
