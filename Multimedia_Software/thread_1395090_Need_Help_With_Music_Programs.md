### Post by tacantara on 2010-01-31
The Synaptic package manager has a program available for download called Sound Converter.  To find it, click on the System > Administration > Synaptic Package Manager.  When Synaptic opens, put soundconverter in the search box.  Click on the box to the left of the package name to select it, then click the green checkmark at the top of Synaptic to lock it in and start download.  The program launch icon should appear in the Sound & Video section of the Programs menu.

For more info on using Sound Converter, check this link out:  [http://www.ehow.com/how_5066158_convert-mp-wav-ubuntu.html](http://www.ehow.com/how_5066158_convert-mp-wav-ubuntu.html)
