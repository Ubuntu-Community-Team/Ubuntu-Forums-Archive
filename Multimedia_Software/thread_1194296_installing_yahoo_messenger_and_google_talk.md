### Post by Revolutionary101 on 2009-06-22
Problem #1

Just use Pidgin Internet Messenger. It comes with Ubuntu. Find it under Applications>Internet>Pidgin Internet Messenger.

Problem #2

Use Avidemux it comes with a nice GUI, offers many features and supports most video codecs. To get it just type this into Terminal

```
sudo apt-get install avidemux
```Hope it helps.
