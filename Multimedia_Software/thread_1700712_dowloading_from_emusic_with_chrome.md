### Post by nothingspecial on 2011-03-09
If emusicj manager is not listed in the "open with" box, choose open with other application, then choose custom command. In the box that opens up type ```
emusicj
```

If that doesn't work, emusicj may not be the command to open it. In that case, assuming emusicj has a menu entry, go system > preferences > main menu then find the entry for emusicj, select it then click the properties button and you will see the correct command.

Other than that, you can have firefox and chrome at the same time, if you prefer chrome then use that for browsing but use firefox to download your music.
