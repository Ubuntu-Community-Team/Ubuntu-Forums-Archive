### Post by robert shearer on 2012-11-01
You need to disable preview so the cd is not scanned.
From memory you open nautalis then >edit>preferences>media and check something like 'do not preview removable media on insertion"
(it is a while so my memory may be soggy as to the actual tab to access this but it is one of the preferences tabs...far right one perhaps.)

Once you have done that you should be able to insert a cd and it will show as an icon on your desktop.
You can then left click it and it will show the files in the file manager.
You can then right click a track/file and select open with.

If you have a preference to open it then right click the track and select 'properties' and then select the 'open with application' and select 'make this default'.

If you are using unity then access the system settings and it is under default applications (again from memory so experiment and search) :-)

If you don't find it post back and when I get home I will check on my Ubuntu install the actual tabs.

edit..Unity>Settings>Details>Removable media>Never prompt or start programs on media insertion
