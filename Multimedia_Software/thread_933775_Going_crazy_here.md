### Post by gandaran on 2008-09-30
to install adobe flash, open synaptic package manager, scroll down to **flashplugin-nonfree** mark for install and click the apply button
or you can run this command, open a terminal and type **sudo apt-get install flashplugin-nonfree**
if you installed any other flash like gnash or swfdec remove them.

### Post by arkticcool on 2008-09-30
There's is no need to get yahoo messenger itself, nor the non-free flash plugin. There are plenty of alternatives. Online messengers, ebuddy, and the multi-purpose messenger Pidgin (formerly Gaim), allow you to chat with your friends. Pidgin comes with your Ubuntu installation. Try your best to acclimatize to open source alternatives as much as possible, its why you got Ubuntu in the first place.

[http://ebuddy.com]("http://ebuddy.com")
Pidgin can be found under **Applications > Internet > Pidgin internet messenger.**
