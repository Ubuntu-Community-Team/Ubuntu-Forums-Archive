### Post by slogum on 2009-06-15
I have a pretty huge mp3-collection. The collection is ordered in A LOT of different folders, very systematic. I've always used explorer to organize my music under windows.

Then I started to use MediaMonkey. In mediamonkey you have access to your filesystem. You can see the folder three on the left hand side. You can browse the different folders, and just dobbeltclick to play the file. Or you can right click -> enqueue folder. Many more options.

The best part is that you can manipulate the file system. You can rename files, move files, move folders, rename folders and so on. And everything you do is synchronized with the file system.

A couple of weeks ago I converted over to Linux. The last weeks I've been searching desperately for a MediaMonkey-alternative, but I can't seem to find it. I used Amarok. In Amarok I can see the folder threes and browse around. But I cant change filenames or move folder. Close but not close enought :)

I also tested songbird, wich I love. Many nice addons and brilliant in-built browser for watching musicvideos on youtube, listen to radios, and so on. But I cant even see the file names in the playlist. And there's certainly not possibility to see the folder threes and browser around.


Are there any mediaplayers out there for linux that can satisfy me needs? Or are there any plugins for Songbird or Amarok that can to the job?


Anyone else in the same situation than me? :)

Let me know
