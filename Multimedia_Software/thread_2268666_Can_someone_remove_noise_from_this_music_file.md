### Post by marseille2 on 2015-03-15
As an alternative ... if you're just after listening pleasure, then the easiest solution is to play it in something like Audacious, and use its built-in 10 band equalizer (in the Output tab on the main toolbar) to cut the noise. It's non-destructive, so you won't change the actual file ... but you'll have to set the equalizer every time you play it.
