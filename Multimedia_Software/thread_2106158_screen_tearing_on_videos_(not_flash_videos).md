### Post by sudodus on 2013-01-20
I don't know what desktop environment you use, but I guess from the title [ubuntu]... that you are using Unity.

So I suggest that you try a lighter desktop environment

- Xubuntu with the light-weight XFCE or
- Lubuntu with the ultra light-weight LXDE

You can test them either live booted from a CD/DVD/USB drive or install the desktop environments into your present Ubuntu system
```
sudo apt-get install xubuntu-desktop
sudo apt-get install lubuntu-desktop
```
You can select desktop environment at the login screen. (Click on the small round symbol near the user names).

I can play high-definition video with mplayer as well as VLC with Lubuntu on aging computers even using the built-in driver (and better with the proprietary driver, if applicable).
