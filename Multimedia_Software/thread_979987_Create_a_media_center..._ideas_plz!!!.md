### Post by prshah on 2008-11-12
> **Dorian17xD said:**
> I have a Pentium 4 computer with a 80 GB HD and 256 mb on RAM and I want to use it as a media center system.


256Mb is too low. If you want to use it strictly as a media "server" it's OK, and you can run it without a GUI.

But considering you want to use it as a media center, you will need more RAM, especially if you want to do live TV recording. I'd suggest atleast 1GB, but I suppose you could do with 512MB in a pinch.

Hi,

I recently use an old P4 and convert it into a Media center using XBMC ([http://xbmc.org/](http://xbmc.org/)). So far, I think this is the best Media center over here for linux and other platform. it's really complete and user friendly. I invite you to try it. There is a live CD available.

To run XBMC you need a decent graphic card with enough RAM.

I suggest you to install Ubuntu Hardy since XBMC doesn't support Intrepid Ibex yet (still in process).
