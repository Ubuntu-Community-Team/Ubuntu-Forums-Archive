### Post by lovinglinux on 2009-04-04
Lots of people recommend VLC, which can play almost anything. I prefer smplayer, which is a frontend for mplayer and has great subtitle support, including SSA, subtitle search and download.

If you want simplicity, then gnome-mplayer.
