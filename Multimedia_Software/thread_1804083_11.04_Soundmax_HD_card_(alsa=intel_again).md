### Post by lidex on 2011-07-14
Use the alsa-info-script to provide detailed information.
Run this command in a terminal:
```
wget http://www.alsa-project.org/alsa-info.sh -O alsa-info.sh && bash alsa-info.sh 
```
Choose the upload option and provide a link for the output.
[Terminal="Applications->Accessories->Terminal"]

### Post by lidex on 2011-07-16
To edit:
```
gksudo gedit /etc/modprobe.d/alsa-base.conf
```
Remove this line:
```
options snd-hda-intel position_fix=1 model=3stack
```
Add the one mentioned previously. Time zone here is GMT+5 or US EDT. This time of year I work long hours many days.

When you see (snd-hda-)intel referenced within alsa it refers to the the specification set forth by intel for HD Audio, not to intel hardware specifically.It is the replacement for AC '97 and alsa loads the snd-hda-intel module to interface with it. Any number of hardware/manufacturers utilize it.
