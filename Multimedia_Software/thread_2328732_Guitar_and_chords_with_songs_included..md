### Post by Rob Sayer on 2016-06-24
I did a search out of curiosity and found little, which did not surprise me.  Linux just can't compete with Android for app development, and the fact that Android is Linux based doesn't matter.

There is an extremely out of date version of Ultimate Guitar for Linux that I wouldn't touch with a 10 foot pole.  It was last updated in 2006.  Linux backward compatibility leaves much to be desired, and I would assume it wouldn't work.

However, apparently the Amarok player will fetch tabs from ultimate-guitar.com, which is all the linux version of Ultimate Guitar did anyway.  I suspect the project was merged.

I can't offer any Amarok advice because I never use it.
