### Post by Toz on 2011-10-09
Hello and welcome to the forums.

It would really be helpful if you included more information, especially about your system, so that someone might be able to help you. At a minimum, the make and model of your computer and what, if anything, you've tried so far.

Because this is an audio-related issue, please open a terminal window, run the following command, and post back the link that gets generated.
```
wget http://www.alsa-project.org/alsa-info.sh -O alsa-info.sh && bash alsa-info.sh
```

This will list your audio hardware and its current configuration settings.
