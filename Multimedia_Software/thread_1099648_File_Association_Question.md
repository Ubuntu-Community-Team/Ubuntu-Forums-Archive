### Post by Precipitous on 2009-03-18
When I play a .pls stream (Winamp format audio stream via Orb), the music starts automatically in Gnome MPlayer, although Gnome MPlayer never launches and I get error messages stating that it can not open the file... I want to be able to play these streams in Banshee, but have not been successful in setting up a file association for .pls files. I have Banshee selected as the default multimedia application in System > Preferences > Preferred Applications, and have used Ubuntu Tweak to set as many audio formats to play in Banshee as possible, but still no luck. Can anyone help?

---

### Post by Precipitous on 2009-03-20
The problem ended up being caused by the Gecko Media Player plugin over-riding the default media player setting in Ubuntu. As soon as it was un-installed the problem went away.
