### Post by RedPenguin on 2011-03-22
I have a Compaq Presario SR1550NX desktop PC with an intergrated Sis 761 vid card.

Ubuntu seems to work perfectly on this PC except if you try to browse the web and scroll down the page, it's like everything blends in to each other, like, the top of the page and what's being scrolled merge into itself and you are forced to do an Alt-Tab or go to another window and back to see things normally.

Also, when going through windows, you see "leftover" pieces of the previous window.

I have done Xorg -configure to generate an xorg.conf file as I am using a Live USB and that seems to fix a fair amount of the video problems, except when I open some windows like secpanel, it's all messed up you can't even use the program, it's like snow on a TV, where you can't make the window out.

I think this video card should be support fairly well, I remember using Gentoo Linux on this PC years ago and never had these refresh problems and such.
