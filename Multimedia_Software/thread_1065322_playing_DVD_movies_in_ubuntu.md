### Post by markusf21 on 2009-02-09
Let me apologize in advance if there is an easier/faster way, but this is the way I know. Go to the [Medibuntu]("https://help.ubuntu.com/community/Medibuntu") site and add their repo. Then in a terminal type```
sudo apt-get install libdvdcss2 w32codecs
``` I would also get VLC, works very well for playing DVDs. ```
sudo apt-get install vlc
``` Then when you play your DVD just make sure you are using VLC.
