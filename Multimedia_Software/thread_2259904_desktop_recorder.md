### Post by Ko_Char on 2015-01-08
I use SimpleScreenRecorder, and like it very much. 
It is a lot better than RecordMyDesktop.
[http://www.maartenbaert.be/simplescreenrecorder/](http://www.maartenbaert.be/simplescreenrecorder/)
(I have one problem though. If I choose mp4 output, the nautilus file manager doesn't show thumbnail for the video file. Totem might say gstreamer error, but it runs fine in VLC or browser. It is just like openshot video editor. May be I'm missing something)

Some people say Kazam is also a good one.
