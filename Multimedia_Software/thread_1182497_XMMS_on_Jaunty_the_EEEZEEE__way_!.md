### Post by critter47 on 2009-06-09
No geeky stuff, no 'builds' necessary, it's this SIMPLE >>>>>>

To get XMMS working on Kubuntu Jaunty all you do is **copy** it from a working drive with an early disto like Gutsy/Hardy. Plug in ur old drive as a slave to copy from, get a file manager like Krusader-root mode to make it easy, search the old drive for all files & directories with XMMS in the name & **copy them to the same place** on ur new drive. If ur as lucky as me, not only will XMMS now work but u'll have it already loaded with ur old preferences & playlist - make sure u copy the 'hidden' .xmms folder in ur old home directory.
Done this on two systems, kept plugins by copying directories like xmms-cdread & skins - they all work like i'd left it !  I'm using XMMS 1.2.10
Only tried this on Kubuntu 9.04 Jaunty so maybe others can post their results.
If someone can get xmms-kde to work let me know.
Don't see how this can damage anything as XMMS seems to work as long as the necessary bits are in the right places.... but I'm only responsible for myself, my offspring & the odours near the TV :p
