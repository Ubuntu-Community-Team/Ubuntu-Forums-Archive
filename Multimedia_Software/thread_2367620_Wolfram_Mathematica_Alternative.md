### Post by Frogs Hair on 2017-08-01
Wolfram Language & Mathematica come free for non commercial use with Raspbian OS for Raspberry Pi, but I find only paid versions for other Linux operating systems. You could check out sagemath from the Ubuntu repository.

[http://www.sagemath.org](http://www.sagemath.org)

### Post by monkeybrain20122 on 2017-08-08
[sagemath ]("http://www.sagemath.org")and [maxima ]("https://launchpad.net/~maxima-developers/+archive/ubuntu/maxima-nightly") You can download a binary for sagemath from the link above. An old version of maxima is in the repo, maxima is also bundled with sagemath. Maxima has a gui called wxMaxima, which you can install from the repo if you choose the ppa/repo route to install maxima.
