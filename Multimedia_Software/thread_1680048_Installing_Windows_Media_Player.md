### Post by cchhrriiss121212 on 2011-02-04
Glad you like Quod Libet, it does not get much recognition, but IMO it is the best out there for a music library player. The Album art is downloaded from various websites, if you go to Music>Plugins you can see what sources are used, Amazon and a few others.
