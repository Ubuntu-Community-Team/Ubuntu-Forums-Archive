### Post by VanillaMozilla on 2010-11-18
There's also Denemo, and quite a few others.  Some months ago I looked at quite a few of them, including Finale for Windows.  I thought they were all quite cumbersome to write music with keyboard or mouse, and some barely worked at all.  Of the open-source ones, MuseScore is reasonably usable, and at that time it seemed the most mature by far.  Denemo has some promise, but it does not wrap the line for display.  You can only view the page by generating a page preview with Lilypond -- if you can figure out how to do that.

I haven't seen TuxGuitar or Qtractor.  Rosegarden doesn't seem to emphasize keyboard or mouse entry.  Audacity is excellent for sound file editing, but it has nothing to do with music scores.

If you open the ps file that Lilypond generates the first time you get a successful "compile" (don't know what else to call it), leave the file open while you are working. Work on the ly file some more, save it, and then compile it. The ps file will automatically update without having to manually re-load it.  

I'm just getting started in Lilypond. But for just keying in a melody line and wanting decent output, it seems mulch easier than Finale. It also transposes for you.
