### Post by jmate24 on 2009-12-13
is there 2 PCs involved. because if there is then you can take a folder (i.e. /home/yourusername/Desktop/myfolder) and right-click it and choose "sharing options" and select it to enable sharing. when the dialogue box appears click the "install service" button, then enter your password, restart your session then log-in, go to where that folder is, (i.e. /home/yourusername/Desktop/myfolder) and right-click it. then select again the "sharing options" and click both checkboxes and enter a name for the share (i.e. myfolder). then go to your vista pc and go to the "control panel" and select networking and internet options and enable file and folder sharing. 

then restart both machines.

finaly in the vista pc enable sharing for the folder that contains your music, videos, pictures and data. (yes there will be a warning, but it is ok to ignore it)

then go to your ubuntu machine and use the Places menu and look for Network, you should be able to see the MSHOME, Windows Network or Microsoft Network and click on it. there you should be able to find your vista shared folder and just copy and paste your data. (but it still will take a long time so i suggest going out for pizza)

there is a second way of doing it...

by using an external hard drive. that is at least 1.5 - 2x the size of your total data. but it still will take at least 3 - 5 hours if you are not using firewire IEEE1394
