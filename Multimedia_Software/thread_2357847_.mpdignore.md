### Post by #&amp;thj^% on 2017-04-06
Now I have not used this on Xubuntu...but this used to work for me.
My option is to drop a file named .mpdignore in every folder you want to exclude from indexing.

for example:
```

    echo "*" > </path/to/directory/to/be/ignored/>.mpdignore
```
