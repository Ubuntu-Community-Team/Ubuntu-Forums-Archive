### Post by BCW142 on 2013-08-23
You need to point MythTV to the new storage group. You need to set it up in mythtv-setup as a storage location or video location, you may also need to run the database on it after doing that (it should run by default). You need to move both files (.mpg & .png.mpg or whatever the picture is called) to get normal effect, but that isn't likely to fix the description. Not sure how to move that, but I do have a /Videos that is part of the storage I can view with MythTV as well as the more normal /mythtv/recordings. :D
