### Post by malachi on 2008-11-21
**_[ManDVD]("http://www.getdeb.net/app/ManDVD")_** is easy to use and gives good results.

**_[QDVDauthor]("http://qdvdauthor.sourceforge.net/")_** is a bit more complex to use, but it gives more you way more flexibility. (You can find QDVDauthor in the repositories.)

**_[Here's an article]("http://www.linux.com/feature/118910")_** to a pretty in-depth look at DVD authoring software in Linux.

### Post by Keyper7 on 2008-11-21
It should be noticed that most, if not all, of those DVD Authoring tools are actually GUI frontends for the command-line dvdauthor tool and, as far as I know, none of them offers the power and flexibility that you can have when you use dvdauthor directly.

Using dvdauthor needs patience, though. :)
