### Post by Spectre5 on 2009-11-22
You should be able to burn mp3 files as a music cd and play it via a CD player.  Make sure you don't burn them as data files, then some CD players cannot play them (but you could fit WAY more songs on the CD this way if your CD player supports this).  In Brasero, select "Audio project" for normal audio CD's.

If you want to convert them though...then there are MANY tools available.  A very simple and straight forward one is "Sound Converter."  You can find it in the repos as "soundconverter."

### Post by rlj1965 on 2009-11-23
One would not want to convert an mp3 (a lossy format) to .wav to burn it to cd. As Spectre5 stated, you can burn mp3's to audio disc however, you may want to start witha lossless file like .wav or .flc to get the best quality. Of course if all you have is an mp3 file as a source format, you will have to burn with that. there are many programs that run on ubuntu that will burn an audio disc like brasero, gnome cd master, k3b, etc. Google is your friend.
