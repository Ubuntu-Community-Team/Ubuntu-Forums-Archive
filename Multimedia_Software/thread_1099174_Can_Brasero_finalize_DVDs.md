### Post by lovinglinux on 2009-03-18
I'm not sure if you really need that feature, because I have burned some DVD's in the past and I think they worked on the DVD player. Besides, when I burn a rewritable DVD, there is a "finalizing" step in the process and if I try to burn it again, Brasero ask for wiping it first.

Maybe the problem is the avi file. I guess DVD players can only play avi files encoded with DivX or Xvid. As you probably know, avi are just containers. The content inside them can be encoded in several different ways. To make sure you could re-encode the avi file with Avidemux, using the MPEG-4 ASP (Xvid) profile, which is fully compatible with the required standards. You don't have to re-encode the entire video for testing purposes, just select a small fraction of the video with the bottom slider and the selection buttons ( "Selection: start" and "Selection: end").
