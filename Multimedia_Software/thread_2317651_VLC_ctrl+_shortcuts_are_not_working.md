### Post by zajcek on 2016-03-18

### Post by zajcek on 2016-03-18
I accidentaly found a workaround. If VLC window is maximized, CTRL+ shortcuts doesn't work, if window is unmaximized, CTRL+ shortcuts works. I guess it is Unity issue (or feature).

### Post by zajcek on 2016-04-25
Aaaargh. Once again I found out it is probably U[COLOR=#000000][FONT=Verdana]nity desktop bug. If we have global menu turned on, this specific shortcuts doesn't work. The workaround is to open media file, press F for fullscreen and F again. Than menu is not in global menu and the shortcuts start to work.[/FONT][/COLOR]
