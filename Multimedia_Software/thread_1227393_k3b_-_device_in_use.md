### Post by grantbow on 2009-07-30
Using Tools...Burn Image in k3b version 1.0.5 in Jaunty works great burning many discs one after another.  I just put new blanks into the drive as the burned ones eject.  This is possible by changing the counter in the lower left of the dialog box to tell k3b that I want more than one burned disc.

Now that I am using Karmic when I try to burn many CDs at once using the 1.66 version of k3b I get a dialog box between discs saying that the device is in use by another program.  I can click the dialog to quit the other program (in this case "nautilus") but the key reason I use k3b is to burn lots of discs without having to click anything.  I haven't yet worked to isolate the problem further.  I found some old related forum threads but not much that's recent.  There are 0 k3b package bugs at [https://launchpad.net/k3b](https://launchpad.net/k3b) and mostly forwarded bugs [http://bugs.debian.org/cgi-bin/pkgreport.cgi?package=k3b](http://bugs.debian.org/cgi-bin/pkgreport.cgi?package=k3b) so I expect the best place to file a bug would be [http://bugs.kde.org](http://bugs.kde.org)

Have other people encountered this problem?
