### Post by MikeMiracle on 2009-10-17
**The solution is to use WINE**

[LIST=1]
[*]Install Wine from Synaptic.(if you dont have wine already)
[*]Download MSVCP60.DLL from website such as   dll-files.com (Sound doesn't work without this dll)
[*]Copy MSVCP60.DLL to c:/windows/system32 in Wine.
[*]Download installer for Windows version of Webex from Webex Site.
[*]Install Webex player to Wine.
[*]play your file in Wine.
[/LIST]
