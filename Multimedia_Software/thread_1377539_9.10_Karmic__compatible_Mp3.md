### Post by mac9416 on 2010-01-10
Hey, ubuchignome,

I've been told that the latest Ipods are locked up tighter than a drum. I tried several FOSS apps on one that a friend got for Christmas, but with no luck. I finally ended up putting iTunes on an XP machine. :-/

As for Linux-compatible MP3 players, I've been told that Android phones such as the G1 do a fair job. I've never owned one myself, so I can only repeat what I've been told.

Let me make this recommendation: buy your MP3s from Amazon.com. To use the iTunes store, you must be running either Windows or OSX and have iTunes installed. In contrast, Amazon not only lets you download MP3s from Linux without iTunes installed, they provide a Linux download app packaged in a .deb. Can't beat that.

Good luck!
