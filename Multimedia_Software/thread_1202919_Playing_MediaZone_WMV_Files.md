### Post by t1m on 2009-07-02
Hi,

This is just for those of you, who, like me, have tried to play MediaZone WMV files on Ubuntu.

For me the MediaZone Flash player works (if Flash is working, of course). The problem comes when watching the Windows Media files...

Some (see [http://ubuntuforums.org/showthread.php?t=402394](http://ubuntuforums.org/showthread.php?t=402394) "wmv files not working") have been able to at least play the "try before you buy" files in mplayer. I don't think MediaZone has that option anymore. But if you are a member and try to play, say, a full Rugby match, mplayer will say "The stream is encrypted and decryption is not supported.".

When you watch a MediaZone WMV file on Windows, Windows Media Player itself asks you for a password. This allows it to un-encrypt the file. But mplayer can't un-encrypt these files, so it gives up with the error.

See [http://ubuntuforums.org/showthread.php?t=226355](http://ubuntuforums.org/showthread.php?t=226355) "Encrypted WMV's"

Sorry for the sad news. But I hope it answers a question that I had, too.

p.s. You could try watching MediaZone WMV's via Windows Media Player via Wine, but I haven't had any success.
