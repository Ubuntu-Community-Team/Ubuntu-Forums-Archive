### Post by lessmemorelove on 2009-11-25
i would use two programs for this - this is not the best solution but i feel it is the easiest to learn: lives and devede.

lives is a good video editor. there are a lot of extra features in it but the basic idea you want to accomplish sounds like selecting video and deleting it. you can do that with this program and then save the file as a dvd mpeg to add to an iso later. just remember to deselect "decouple video from audio" under the edit menu;)

the other program i would use is devede.  devede can make a dvd with all the dvd mpegs you made from lives. it is pretty straight forward to use. 

i get devede from the ubuntu repositories and lives from the getdeb.net repository.
