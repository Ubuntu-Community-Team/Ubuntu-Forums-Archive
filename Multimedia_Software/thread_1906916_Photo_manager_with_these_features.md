### Post by eric-yorba on 2012-01-10
> I would like to find photos based on their pixel dimensions.  Say, all pictures that are 1920x1080.
Shotwell will add this eventually, feel free to bug the mailing list, maybe then it will be added sooner rather than later.

> I would like to organize photos on the disk, according to *SOME* but not all tags.
Are you referring to the directory structure?  I'm not aware of any program that can do this.

> I would like to find photos that have no tags at all.
Shotwell can do this via the search mechanism.  Hit ctrl-S to create a new saved search, and you can build a search criteria of untagged photos.
