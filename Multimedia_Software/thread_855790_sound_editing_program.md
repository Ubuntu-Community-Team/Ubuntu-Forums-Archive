### Post by Tom_ZeCat on 2008-07-10
Every now and then I run across an MP3 that crashes Amarok.  It doesn't happen that often, but it can be annoying when it does.  The solution I found was to open the MP3 in the sound editor in Roxio on my Windows XP computer, save it as a wave file, and then re-save it as an MP3 in HiMatt compliant mode.  (I think it's called that -- it's the mode that's supposedly got the best sound quality.)   After that it plays fine in Amarok.  

I'm wondering if there's a sound editor for Linux similar to the one I have in Roxio.  It would be nice to be able to do this without having to go to my Windows PC.  Plus, a sound editor is nice if you want to delete the talking before a song from a live album.
