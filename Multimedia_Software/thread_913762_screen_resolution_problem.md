### Post by starcannon on 2008-09-08
That card while a little anemic should be able to play that game with no trouble at 1024x768, or 800x600 for that matter.

I haven't played that particular game, but if its like most I have played, you'll find that some user specific settings are stored in you /home/username directory. It may be in a hidden file so be sure to CTRL+H to view both hidden and visible files. When you find the folder or config file delete it, restart the game, and if its like most, it will create a new user file with all default options.

GL, I know thats not detailed, but its the best I can do since I haven't installed that particular game.
