### Post by TheFu on 2014-09-07
Plex media server - though disorganized music (like all in 1 folder) is a danger to the music. A corrupted folder could be bad for all those files.

Music will be organized by any of the ID3 tags - including genre - that will split up the country. Of course, I'd strongly suggest using an organization tool to make the music into Genre/Artists/Album/Track layout. This will speed up all sorts of things vs having 1,000+ files in a directory.

Never heard of DAAP - but Plex is a great DLNA server.  That means it works with pretty much any computing device (Bluray/tv/android/VLC / chromecast, etc.).

I don't use multiple accounts (actually, I don't use any Plex accounts), but plex does support others.
