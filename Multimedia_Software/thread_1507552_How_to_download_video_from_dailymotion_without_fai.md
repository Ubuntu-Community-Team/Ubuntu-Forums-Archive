### Post by jflaker on 2010-06-11
> **rudysuryanto said:**
> I use addon from Mozilla Firefox to download video from dailymotion, but sometime can suddenly stop downloading, then it can not resume. Are there software that run under Linux that can download video from dailymotion and can resume it, if downloading suddenly stop? Thank's.

install "clive".

It is a CLI only and the command for getting from youtube or dailymotion is 

```
clive "<Paste the URL from your web broswer>"
```

The program drops the video in to your /home/YourUserID directory.
