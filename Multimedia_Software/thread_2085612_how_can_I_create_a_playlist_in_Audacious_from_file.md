I will mark this as solved.

 I installed Clementine and by opening a playlist,
a music file can opened from the network directory and it will be added to the Clementine playlist.
 You can open a music file from the network that will play in Audacious, except Audacious will create a new playlist when it does, it will not add the file to an existing playlist
