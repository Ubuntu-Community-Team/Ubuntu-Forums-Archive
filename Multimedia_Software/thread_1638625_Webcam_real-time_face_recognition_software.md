### Post by g30rg3 on 2011-05-04
Linux doesn't have an inbuilt face recognition system, in fact I was thinking of working on one. There is the Open CV library, but it only has face detection as of now, and the face detection is frankly not very good, since it uses object detection to detect faces which makes many erroneous detections. You can combine it with a color detector and a neural network for improved results
