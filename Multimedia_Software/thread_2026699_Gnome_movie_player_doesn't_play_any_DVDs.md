### Post by bobsan on 2012-07-15
You need to install libdvdcss2 for Ubuntu to play DVD. When you are at it, you should also install libdvdnav and libdvdread4. I think you need to activate the medibuntu repository to install libdvdcss2, here is the instruction. [http://www.unixmen.com/how-to-add-medibuntu-repository-in-ubuntu-via-terminal-and-gui/](http://www.unixmen.com/how-to-add-medibuntu-repository-in-ubuntu-via-terminal-and-gui/)
