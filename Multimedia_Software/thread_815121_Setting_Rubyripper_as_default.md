2 things you can try:

System-Preferences-File Management-Media-"CD Audio", then see if RubyRipper is available as an option there.

If not, open your /media/ directory, then right-click the relevant CDROM, then choose "Open with another application"; in the list that pops up, either choose RubyRipper (if it appears) or click "Use a custom command", and enter the command for rubyripper.

Subsequently, rubyripper should show up as an option in the autostart as well as the File Management-Media option listed above.
