### Post by cody1233 on 2013-04-02
I assume you're just looking for a DLNA server?  If you have a desktop, you can try Rygel (in the software center).  If you're using server, try minidlna.  I haven't personally had any experience with either of these programs since i just use my wii with samba, but if you have problems with these, i can try to help.
