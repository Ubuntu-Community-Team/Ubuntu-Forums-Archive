since I've been waiting a long time for the stable release of Gimp 2.10, I removed Gimp 2.8, installed flatpack and then installed the Gimp using the following line:
```
flatpak install https://flathub.org/repo/appstream/org.gimp.GIMP.flatpakref
```

### Post by claus on 2018-05-21
I found out myself how to start Gimp 2.10 by using (inside a shell) the command 'flatpak run org.gimp.GIMP'.
