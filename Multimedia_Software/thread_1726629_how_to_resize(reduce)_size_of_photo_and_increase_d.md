### Post by howefield on 2011-04-11
Gimp is fine for occasional resizing but maybe a bit cumbersome for such a small job, an alternative would be to install nautilus-image-converter from either Synaptic Package Manager or the Ubuntu Software Centre.

This will give you a right click resize option from within Nautilus. Right click the image and select "Resize Images.." and enter details eg, pixel size, into the dialogue window, ect, ect.
