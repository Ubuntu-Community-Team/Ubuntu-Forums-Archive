### Post by jbrown96 on 2009-02-23
i don't think you will be able to do that. If you know anything about file systems, it's helpful to think about CDs and DVDs as having their own file system. The two cannot be directly copied because they have different formatting. If you are simply copying the disc for backup purposes, then you have two options. First, create an image (.iso) of your audio disc. Then you can burn that image file to a DVD. It won't play in cd players but you will have an exact copyt that perfectly preserves the metadata and gaps in between tracks. The other option is to copy the individual files from the audio Cd and then create a data DVD. This also won't play in a CD player, but the disc will burn faster since there is less data. 

I recommend the first method, so you can burn the image file to a CD-R later.
