### Post by Legendário on 2009-10-18
I will leave here my solution, just in case anyone-else needs it. I had to run a live session of Ubuntu Hardy, install nvidia restricted drivers and then, copy xorg.conf to a usb drive to later restore it on my Jaunty install. This was the only way I could have optimal resolution on my jaunty box.

Hope it can help someone else.
