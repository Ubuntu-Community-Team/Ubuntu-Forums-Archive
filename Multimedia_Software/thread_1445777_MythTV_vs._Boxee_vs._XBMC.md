### Post by Crazy_Land1 on 2010-04-25
you are right, both boxee and xbmc only play media.

Boxee is built off of xbmc but boxee has gone the way of social where xbmc is still just a media player for the home. From what i have been reading tho both xbmc and boxee are looking at adding media capture in the future.

So for media capture the best i have played with is mythtv
As for running xbmc and boxee under windows i had to much problems with that so i run xbmc media boxes using the live cd from xbmc and the live cd from XBMCFreaks, XBMCFreaks live cd is used on a ION board and they have set up the cd for those boards.

So one route you can go is like i did, list of set up here

Windows 7 system with media capture software hooked up to cable for recording
XBMC in livingroom
XBMC in recroom
XBMC in bedroom 1
XBMC in bedroom 2
Windows home server hooked to the network serving up all media, music,pic's and videos to all XBMC's and all pc's in the house.

Windows 7 i record on then FTP the files to the windows home server.

But i can't remember isn't Mythtv a samba server as well, if so then set up a Mythtv box and then set up some xbmc or boxee boxes and use those to read from the Mythtv box
