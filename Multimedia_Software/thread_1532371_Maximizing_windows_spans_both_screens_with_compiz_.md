If I enable compiz, maximizing a window (or playing a movie fullscreen) will span both windows: not really convenient. 

If I disable compiz, all works fine. Maximizing a window will span only 1 window (the one it is located on). I do not really need compiz, but having it would be nice, especially since my hardware should easily be able to run it.

I have a Radeon HD4850 graphics cards, en tried this with both the opensource and the ATI closed source drivers.
