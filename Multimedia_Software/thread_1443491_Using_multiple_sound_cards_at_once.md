### Post by Kazurik on 2010-03-31
Hey, I was wondering if it was possible to direct the audio output of a program to a specific sound card?

For an example in windows I would have my built in sound card set as my default device. That way my music would play through my speakers. But then in skype I would choose my 2nd sound card for its output as I have my headphones plugged into that card. This would allow me to have only skype calls go to my headset while all other sound, music/games, would play through speakers. Is this possible in Ubuntu?
