### Post by TenPlus1 on 2008-09-29
To make your own DVD movies (with menus) you can use DeVeDe, and to play DVD movies (with menus) you can use SMplayer or VLC, but I do recommend you add the medibuntu repositories and install the libdvdcss libraries...

Go here for medibuntu: [https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)

### Post by teddybairs1 on 2008-09-29
I usually use ffmpeg to convert my videos to ntsc-dvd mpeg and then use ManDVD to create the menu and assemble the DVD into an .ISO which can be burned to a DVD. So far I haven't had any issues except burning the DVD directly from ManDVD. I always have to create the ISO first (ManDVD has an option for that). This usually takes up a lot of space on the hard drive though. I recommend at least 12-15 GB free.
