### Post by cilynx on 2009-10-24
You're looking at jumping into a huge world of information.  My best advise (as a long-time web developer) is to get to know HTML and CSS by hand before you do anything else.  No WYSIWYG tool, either open source or commercial, actually lives up to it's promise: you always wind up tweaking the code in the end.  

Learn the code first so you don't spend the next year trying to figure out why What You Got is not What You Saw.  Once you've gotten the code down, go back to the visual tools and see which ones line up best with your strengths and weaknesses.
