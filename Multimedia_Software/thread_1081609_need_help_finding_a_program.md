### Post by SuperSonic4 on 2009-02-27
Normally you can install most software by doing ```
sudo apt-get install <package>
``` in the terminal or you can use synaptic to search for a package

I've heard good things about sound juicer but not used it myself:

```
sudo apt-get install sound-juicer
```

Myself, I use k3b without any issues but then I run kubuntu and putting k3b in gnome will require some KDE libraries:

```
sudo apt-get install k3b
```
