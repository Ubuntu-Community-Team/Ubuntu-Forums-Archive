### Post by SuperSonic4 on 2009-04-28
k9copy's is pretty simple, I'll load up a annotated diagram in a couple of mins :]

1. Select the source, this will probably be a dvd from your drive

2. Select iso as the output, next to it will be the location of the output iso

3. Press copy to start
