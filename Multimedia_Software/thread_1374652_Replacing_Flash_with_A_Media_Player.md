### Post by m0o on 2010-01-07
You can use Greasemonkey scripts like "[Youtube without Flash Auto]("http://userscripts.org/scripts/show/50771")," which uses embeded browser players like mplayer, totem, vlc, or gnome-mplayer. It also supports multiple browsers. 

There's also the Media Player Connectivity Firefox extension that lets you use external players to play Flash video content.
