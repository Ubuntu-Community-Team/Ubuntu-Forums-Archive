### Post by cariboo on 2009-04-12
I can't use slacker.com, because I don't live in the right country. I looks like it is a Flash application, so all you need is to install it. I would suggest you install the ubuntu-restricted-extras meta package. It will install Flash, Java, MScorefonts and most of the codecs you need to play most types of audio/video media. The ubuntu-restricted-extras package is in the repositories and you can install it using Add/Remove Programs, Synaptic Package Manager and of course the command line.
