### Post by scar1 on 2011-09-07
To be a little more accurate with the fault description it was almost like the colour was going in and out and contrast was coming in and out also.

Like the old school macrovision encryption.

Anyway I resolved this by changing a setting in boxee;
Media >> Video >> Advanced >> H/W Acceleration [unticked]
Media >> Video >> Advanced >> Render Method >> Software

All working ok now...
