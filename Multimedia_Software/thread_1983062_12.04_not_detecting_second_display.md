### Post by sambolinux on 2012-05-19
I had the same problem  and searched in vain. I found the problem lies with the kernel from ubuntu repositories, so download the kernel and install from ppa mainline and you should be fine.
**FIRST** go here([http://kernel.ubuntu.com/~kernel-ppa/mainline/]("http://kernel.ubuntu.com/%7Ekernel-ppa/mainline/"))  and download the header files and kernel image for your system i.e.: i386 or amd64. You will need these files : [linux-headers]("http://kernel.ubuntu.com/%7Ekernel-ppa/mainline/v3.3.5-precise/linux-headers-3.3.5-030305-generic-pae_3.3.5-030305.201205071254_i386.deb"), [linux-headers-***_all.deb and ]("http://kernel.ubuntu.com/%7Ekernel-ppa/mainline/v3.3.5-precise/linux-headers-3.3.5-030305_3.3.5-030305.201205071254_all.deb")[linux-image-.]("http://kernel.ubuntu.com/%7Ekernel-ppa/mainline/v3.3.5-precise/linux-image-3.3.5-030305-generic-pae_3.3.5-030305.201205071254_i386.deb")
**SECOND** create a folder called kernel your home directory and cut and paste these three files there in the kernel folder.
**THIRD** open a terminal and navigate there by **cd ~/kernel** then  paste** sudo dpkg  -i *.deb**
**FOURTH** after the installation reboot and it should be the first kernel listed
