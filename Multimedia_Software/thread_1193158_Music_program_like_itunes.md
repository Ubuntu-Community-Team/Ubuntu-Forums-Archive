### Post by davidryderuk on 2009-06-21
Hi,
There are two other music applications like itunes I've heard about. The first is banshee, which is quite good.

[http://banshee-project.org/](http://banshee-project.org/)

The other is Songbird, which is still in an experimental stage, but is developed by the same people behind firefox so it should be interesting.

[http://getsongbird.com/](http://getsongbird.com/)

If you want to install banshee its already in the repositories so just search for it in synaptic package manager. If you want to install Songbird you can use the Getdeb website.

[http://www.getdeb.net/](http://www.getdeb.net/)

Additionally although it is a windows application (a really good one) and its more like an interactive radio station than a music player, you might want to look at spotify. You can follow the instructions on their website to get it installed.

[http://www.spotify.com/en/help/faq/wine/](http://www.spotify.com/en/help/faq/wine/)

If you have never heard of wine you might also want to read about it first.

[http://www.winehq.org/docs/wineusr-guide/what-is-wine](http://www.winehq.org/docs/wineusr-guide/what-is-wine)

If you have any issues with the application try switching the sound driver for wine to OSS (can be found under the Audio tab of the "Configure wine" application).

### Post by davidryderuk on 2009-06-21
Depends where you got it from. If you got it from the Getdeb website, just double click on the .deb file and click install in the resultant window. This will produce a link in your applications menu as well as checking for any dependencies.

If you got it from the actual songbird website then you will probably be able to extract the tar.gz file into your home directory (although there is probably a better place to extract it) and then click on the file called "Songbird" inside the extracted folder and choose to run it.

Personally i prefer using deb files, because they are less messy than tar.gz files.
