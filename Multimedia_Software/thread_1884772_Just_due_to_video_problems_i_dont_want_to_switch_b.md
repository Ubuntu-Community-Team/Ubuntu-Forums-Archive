### Post by beew on 2011-11-22
512 mb of ram is pushing it especially if you try to watch videos and do something else at the same time. Try Lubuntu 11.04 and see if it gets better. Ubuntu 11.10 is quite buggy for me and I expect many surprises which I don't want. Better wait a few months for the bugs to clean up. Also if you have a weak system forget about the Software center, it is a resource hog at startup. Get synaptic (it comes preinstalled in Lubuntu I think)

Also for flash and youtube install minitube and the flashvideoreplacer addon, this way you can watch videos on Youtube and a few supported sites without flash, which would ease up resource use quite a bit.

For minitube get it from ppa 
[https://launchpad.net/~nilarimogard/+archive/webupd8]("https://launchpad.net/%7Enilarimogard/+archive/webupd8")
Ubuntu's version is always outdated and is broken (if you use 11.04 or below)
