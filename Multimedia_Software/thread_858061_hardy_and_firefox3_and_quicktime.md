### Post by Hoaxe on 2008-07-13
i don't have a problem, this is more of a post for others.

I wanted to watch some quicktime videos on apple's site, but it wasn't quite working. not at all actually. all i would see is the first frame, and it would freeze at that instant. i tried getting the mozilla-plugin-vlc package, but that didn't work for some reason. So i tried the Mplayer package for mozilla based browsers and just my luck it worked great.

not only that, but the interface for mplayer in my FF3 is just sweet. 

here's the code:

```
sudo apt-get install mozilla-mplayer 
```

just in case someone wanted to know how, google had returned weak results so i decided to post this, hope it helps someone
