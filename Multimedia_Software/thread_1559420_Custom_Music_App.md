### Post by Jason Cook on 2010-08-23
I am looking for a music playing app that has some very specific features. I want to play music for a large group of people. I would like the listeners to  vote on what songs to play. The interface should allow users to vote for songs in my library. It would then play the song with the most votes after the current song finishes playing and reset the vote counter for that song.

Some other useful features, but not essential, features would be preventing a song from playing twice until a pre-defined amount of other songs get played. The ability to import a song from a flash drive or other device and vote for it would also be useful.

If no one knows of an app that will do this, I could help (or hinder?) someone else in writing this app. I have only used bash.

After think about this some more, using this a front-end for VLC sounds like a decent idea. That means that it can play a lot of file types and would remove the need to write an entirely new app.
