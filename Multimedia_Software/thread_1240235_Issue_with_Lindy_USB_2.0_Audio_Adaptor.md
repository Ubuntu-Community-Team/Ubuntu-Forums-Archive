### Post by dp_miller on 2012-08-07
For info - I just bought one of these for my Ubuntu 12.04 setup.

Plugged in and it "just works". You simply select the new 'Speakers' option that appears in the sound settings.

Note I'm using it as 2 channel analog as I had a need for headphones and I'd stupidly bought an amp without a headphones socket. I don't think you'll have any joy if you need the more advanced features as this would require a driver.

But if you just want an additional 2 channel audio out, this works fine.
