### Post by virgult on 2008-05-08
If you go to Firefox's homepage, somewhere in the plugins section, there is a cool feature that allows you to download any kinds of multimedia content from a webpage, even if it's embedded in a flash animation. The format should be h264; if you need an mpg stream, consider re-encoding it with Avidemux.
