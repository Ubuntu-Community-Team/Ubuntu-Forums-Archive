### Post by Ublis on 2008-12-19
I'd go with [MP3Tag]("http://mp3tag.de") (under Wine) and Amazon with some manual postprocessing as cover source. I've tried automated tools, but they often give really bad results, ranging from a completely wrong cover to the right cover but with a huge border around it. Also many ID3 tag editors will have some issues with writing tags properly (EasyTag for example would sometimes not recognize tags it had written itself), while MP3Tag seems reliable.
