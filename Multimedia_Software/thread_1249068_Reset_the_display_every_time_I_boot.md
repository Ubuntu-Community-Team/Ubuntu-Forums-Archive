```
### Post by walterdf00 on 2009-08-31
I had a similar problem using mythbuntu with a via board.  I had to change the video settings every time I logged in.  I tried a couple things that seemed to fix it.  I used
```
sudo dpkg-reconfigure xserver-xorg
