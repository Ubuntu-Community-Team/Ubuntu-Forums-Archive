### Post by PhilGil on 2010-03-04
> **joehc said:**
> 
Is it possible that some image you just can't get any information about, just Image Type: jpeg (The JPEG image format)
Width: 640 pixels
Height: 400 pixels
?
Is it because a get it through a msn transfer?
Yes, it's possible for an image to have no EXIF data.  The image source needs to embed the data, and photo editing programs can strip it out while saving an edited image.  In Photoshop, for example, if you use "Save for the web" after editing an image the EXIF data is stripped out.  If you use "Save As", the EXIF data is preserved.
