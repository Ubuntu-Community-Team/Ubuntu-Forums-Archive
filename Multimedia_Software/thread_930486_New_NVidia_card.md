### Post by annatar on 2008-09-26
Of course you don't have to reinstall...
 
Ubuntu should automatically detect your nvidia card load the corresponding driver
Even if that's not the case,  ubuntu should prompt you to choose 
 
However, the default 'nv' driver is inferior(in terms of performance and supported card features) when compared to the offical nvidia driver
 
if you are using Kubuntu...see here for the easiest way to install the official drivers
[https://help.ubuntu.com/community/BinaryDriverHowto/Nvidia#Kubuntu%208.04%20Hardy%20Heron](https://help.ubuntu.com/community/BinaryDriverHowto/Nvidia#Kubuntu%208.04%20Hardy%20Heron) 
There is also some troubleshoot info on that page incase you have problems
