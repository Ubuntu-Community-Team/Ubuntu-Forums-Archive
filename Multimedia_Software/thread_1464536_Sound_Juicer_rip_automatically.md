### Post by mc4man on 2010-04-28
don't use sound-juicer (use rubyripper for auto ripping) but should be fairly easy.
Either go to File Management -> media -> cd audio -> use other application -> use a custom command or insert an audio cd while holding down the shift button and in the popup -> use other application -> use a custom command

use as a command
sound-juicer -a

then set sound-juicer as the default for audio cd's

(file management is a disabled menu item in system -> preferences - can be enabled from edit menus or accessed from nautilus -> edit -> preferences among other ways
