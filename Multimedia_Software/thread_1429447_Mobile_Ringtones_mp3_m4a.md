### Post by muckblit on 2010-03-14
One problem is that the phone may enforce 64kbps for ringtones but not for Music Player. Then you would have an mp3 listed but it would not play as a ringtone when you sample or use it.

Or a phone only takes mp3, not m4a.

Or only m4a.

Or only Very Low quality, 64kbps.

I pick a 30 second clip from an mp3 in Audacity and save that to mp3.

Then I run SoundConverter and set preferences to Very Low quality mp3 output. Audacity must be able to change bitrate but I have only used it once and didn't see bitrate options.

Convert to 64kbps. Save to my phone's microSD chip. I had problems with impossible to delete files when I used bluetooth so I pull the chip out and insert it into a usb card adapter to load the file into Ringtones.

It might be worth mentioning that I have a couple of new Samsung phones and it takes several hacky things to make external cards work. R450 Messenger formats the cards, but the other Samsung keypad phone on Cricket needs my R450 to format the 8G microSD card for it. Then if I bluetooth the files in, all kinds of funky things happen if I try to delete from either end so I just put the card in the usb reader.
