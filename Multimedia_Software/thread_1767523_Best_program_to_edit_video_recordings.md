### Post by adam22 on 2011-05-26
Kdenlive or Openshot. For the latest version of Openshot:

```
sudo add-apt-repository ppa:jonoomph/openshot-edge
			sudo apt-get update
			sudo apt-get install openshot openshot-doc
```

Verson 1.3.1 is the current release. It's simple to use, just import files and drag them on the tracks. You can trim clips and add effects/transitions.
