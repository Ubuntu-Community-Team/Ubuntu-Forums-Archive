### Post by Bill magee on 2009-03-22
This solved my sound problems in recordMyDesktop:

To record both microphone sound and game sound:

1. in System/Preferences/Sound set all options to Alsamixer including Default Mixer to SiS alsamixer
2. run /usr/bin/gnome-volume-control. In switches/preferences turn on Mix and Mic. Make sure microphone is unmuted and turned up (if you want mike sound. Set device for sis alsamixer.

Hope this works for you.
