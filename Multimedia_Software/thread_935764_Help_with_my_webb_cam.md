### Post by GeoMX on 2009-05-21
I have this webcam too, searching the web for information on how to install it on Ubuntu I concluded that it is not supported under Linux:

This webcam is not listed in USB IDs list:
[http://www.linux-usb.org/usb.ids](http://www.linux-usb.org/usb.ids)

It is not in the list of devices supported by the gspca driver (it supports some other Pixart Imaging Devices (Genius webcams)
[http://www.kernel.org/doc/Documentat...inux/gspca.txt](http://www.kernel.org/doc/Documentat...inux/gspca.txt)

So, if you're looking for a webcam to be used with Ubuntu, better avoid this one.
