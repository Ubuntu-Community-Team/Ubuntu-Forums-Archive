### Post by nleibert on 2012-05-11
Hi everyone,

I've been using Ubuntu 12.04 for a week now and everything has been really smooth besides playing flash videos...I chose to use Gnome Classic (No Effects) instead of Unity. The problem I'm having is when viewing any Flash video (such as on Youtube) I can't view it fullscreen without it slowing down to a crawl...infact some videos make my whole system choppy BUT it doesn't do this when I'm using Unity or Gnome 3. I've installed the Flash-Aid plugin but that didn't help.

Quick system specs are:

Intel i5 2500k 3.3GHz Processor
AMD Radeon 6870 1GB Video Card
8GB DDR3 Memory

I just don't understand why flash would be practically unplayable in Gnome Classic but not Unity? This issue has been driving me nuts because if it weren't for this problem, Ubuntu would be perfect for me.

---

### Post by nleibert on 2012-05-12
I found out the problem...incase anyone is in the same situation, I reinstalled Ubuntu and chose NOT to install the ATI drivers when the "Install Additional Drivers" popped up on first boot and now everything works perfect! I guess the ATI drivers for my Radeon 6870 do not work (even though they worked in the Precise Beta!).
