### Post by Gregory Petrosyan on 2010-03-04
I'd like to announce the release of cmus 2.3.0 here.

cmus 2.3.0 features gapless MP3 playback, native PulseAudio output plugin (which cures all the problems with PA ALSA emulation present in 2.2.0), very fast metadata cache and much improved compilations handling.

Not to mention tons of bugfixes since 2.2.0, which was released almost 3 years ago.

Please give it a try — [http://cmus.sourceforge.net](http://cmus.sourceforge.net)

### Post by VolatileMember on 2011-04-26
For all of you interested, there is a new version of cmus (2.4.0). For people who don't like to compile for themselves there is a PPA here:
[https://launchpad.net/~jmuc/+archive/cmus]("https://launchpad.net/%7Ejmuc/+archive/cmus")

Release notes from freshmeat:
> Compared to 2.3.x, cmus 2.4.0 features live filtering, mutt-like short filters, resume support, smarter string handling, long format options (including ones for bitrate/codec), HTTP proxy support, less CPU wakeups during playback, new RoarAudio output plugin, support for big-endian systems, and lots of different new audio sample formats
