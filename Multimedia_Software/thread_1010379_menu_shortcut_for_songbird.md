### Post by turtlepaul on 2008-12-13
Since Songbird isn't available through synaptic, I installed by manually downloading the package. I was wondering how to add menu and panel shortcuts for it. There is a file to launch the application, but that asks me to run it when I click on it. I was hoping for something a little smoother.

### Post by cdtech on 2008-12-13
Right click on the top panel, select "add to panel" select the "Custom application launcher" and just put the terminal command you use now into the "Command" location. Press the icon on the same window to change the default icon to a custom icon of your choice.
