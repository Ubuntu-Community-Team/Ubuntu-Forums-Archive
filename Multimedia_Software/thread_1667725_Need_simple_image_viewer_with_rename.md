### Post by smartbei on 2011-01-15
Sounds like gthumb does most of what you want - available in the repos.

If you want something more powerful, you can try the Shotwell Photo Manager that comes with ubuntu - it creates a DB of photos for easy management. Try them out [it only takes a second] and decide.

There is also F-Spot, which used to be included in ubuntu by default, but I think Shotwell is better.

lastly, you can try running Picasa on Wine - it is supposed to work fine, and if that is what you are used to, what could be better?
