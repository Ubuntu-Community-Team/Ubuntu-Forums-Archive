### Post by markbuntu on 2008-06-17
Streamtuner is great for streaming radio. It is fairly old and was written to use xmms which is not avialable in the repos but you can find out about it here:

[http://ubuntuforums.org/showthread.php?t=765609](http://ubuntuforums.org/showthread.php?t=765609)

It will work with totem but not with rythmbox. If you want something similar to streamtuner that works with rythmbox you can get tunapie. A new streamtuner is slowly making its way from debian to Ubuntu so prehaps the new version will work with rythmbox. Streamtuner seems to have more stations than tunapie. If you listen to internet radio streams, you really should have one or both of these.

Streamripper still works if you need to rip streams.
