some of you might be interested in this [many others probably know this already]

Audacious is now really stable in my recent experience
something which i did not find in earlier years


there used to be a way to colour the skin
but now it seems to have gone


reading around i realized it should be ok with wsz skins

no manual way to do this tho

but easy with terminal  if placed in a folder called Skins in your home folder

```
sudo cp  ~/Skins/*  /usr/share/audacious/Skins

```


will move as many as you want there

and end up with something like this   [[IMG]http://img3.imageshack.us/img3/1351/workspace1002m.th.png[/IMG]](http://img3.imageshack.us/img3/1351/workspace1002m.png)   [[IMG]http://img267.imageshack.us/img267/5662/workspace1003.th.png[/IMG]](http://img267.imageshack.us/img267/5662/workspace1003.png)
