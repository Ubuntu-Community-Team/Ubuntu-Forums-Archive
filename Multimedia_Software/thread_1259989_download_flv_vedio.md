### Post by krendar on 2009-09-07
I use a command tool called "youtube-dl". You can install it via synaptic.

Easy to use:

```
youtube-dl -t http://www.youtube.com/watch?v=8gpjk_MaCGM
```

The "-t" flag tell it to use the title of the video as filename. You can also add a "-b" flag to get the high-quality version of the video if it exists.
