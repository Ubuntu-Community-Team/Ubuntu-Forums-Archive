### Post by Guitar John on 2009-01-02
I have the Creative Zen Vision:M and my wife has the Zen V Plus.  For file transfer on both devices I use a program call Gnomad 2 which is available in *Add/Remove Programs* or in *Synaptic Package Manager*.

The program works as far as transferring any type of file that the device handles, but it is a bit labor intensive.  If you have an entire cd that you want to put on the MP3 Player, you have to move it in track by track.

I hope this helps.
