### Post by semaka on 2011-02-07
After 2 days of reading and trying i've figured out what really matters here, if anyone is interested.

This : HorizSync       28.0 - 55.0 VertRefresh     43.0 - 72.0 

Should be :  HorizSync 31.0 - 80.0 VertRefresh 55.0 - 75.0

Get your monitor supported frecvencies and write them in xorg.conf
The rest can be whatever nvidia thinks.

And one more thing, this is the result of using a cheap VGA cable, search for one with more wires in it.
