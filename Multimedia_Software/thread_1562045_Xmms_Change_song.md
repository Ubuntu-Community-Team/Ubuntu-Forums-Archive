### Post by tgalati4 on 2010-08-27
If you click the PL button (playlist) you can open a directory, then choose "Add" and it will add all songs in that directory.  It should cycle through all of the songs automatically, either in order or shuffle play depending on whether the shuffle button is pressed.

If you like minimalist players try mocp (sudo apt-get install moc).
