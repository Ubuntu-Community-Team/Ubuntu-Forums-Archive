### Post by shoup on 2013-03-05
I have just finished installing 12.04 and XBMC frodo onto a new system and the entire process was easy as could be.  However, when I hook up the system to my tv, a 40" sony, ubuntu 12.04 greatly overscans the picture so that I can not see any of the icons along the edges.  When I look a the display function, it recognises that the tv brand is a sony, but it says it is a 72" screen.  When I start firefox, there is the same overscanning, but when I go into XBMC, the screen is okay.  I am running an asrock z77 itx motherboard with an intel i3 3225 chip.  I can not find how to tell ubuntu that the screen is 40", I am hoping that could fix the overscanning.  Any suggestions or help is greatly appreciated.  

shoup

### Post by shoup on 2013-03-10
Just wanted to post the solution I found to my problem, it has nothing to do with Ubuntu or XBMC, but with my tv.  There is a setting under screen settings, display area, that has an option called full pixel.  Changing that setting allowed the full screen to be displayed.  Just wanted to post this in case it can help anyone else who experiences this problem.

shoup
