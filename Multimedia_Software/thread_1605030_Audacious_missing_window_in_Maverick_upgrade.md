### Post by freakengine on 2010-10-24
I just wanted to post this because I didn't see any similar posts.  After my upgrade to Maverick from Lucid, I could load Audacious and hear it but I couldn't see or control it.  I fixed it by entering the following in a terminal window:

audtool mainwin-show yes

Not sure why it defaulted to not showing the window after my upgrade since I'd never set Audacious up that way in Lucid.  So far this is my only issue with the upgrade.
