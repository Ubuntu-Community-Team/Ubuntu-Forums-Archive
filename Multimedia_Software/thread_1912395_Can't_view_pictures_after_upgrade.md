If you have a Windows pc nearby try putting the cd in it and seeing if you can see your photos. If not then the photos did not actually get burned to the cd. I had that happen to me once. If so, put the cd back in your Ubuntu laptop and see if you can see the photos. If so, just copy them (the entire folders) back to your laptop again. If not, then use your Windows pc to copy them from cd to a usb stick and then copy them from there to your Ubuntu laptop.
