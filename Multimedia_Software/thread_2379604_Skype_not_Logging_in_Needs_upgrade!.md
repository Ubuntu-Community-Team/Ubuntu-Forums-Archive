### Post by jnm2 on 2017-12-07
I noticed that when I started skype yesterday it didn't work. It asked for my password, and then just died. I didn't have time to look at it then, but today it was behaving the same, so I looked around. It turns out that MS servers are rejecting Skype of the wrong version now. I was with 4.3, which is the one currently in Ubuntu repositories, and there is no warning message, no 'You must upgrade" hint, it just dies, silently. 

So head over to the Skype Linux Download site and get the new version which has leapt up to version 8 point something. It might complain that its newer than the version in the repo, but go ahead and install. It will now work OK. 

Note: 64 bit linux only now. But the embedded images now work without requiring you to login at the Skype website. 

I really think MS could have just sent a message out telling people what's happening. It is, after all, a messaging app.
