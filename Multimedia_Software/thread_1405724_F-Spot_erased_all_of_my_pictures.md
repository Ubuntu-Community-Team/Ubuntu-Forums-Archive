### Post by kamaaina on 2010-02-13
Rename the Pictures folder to Pictures1. create a new Pictures folder in home.  Delete ~/.gnome2/f-spot/photos.db then re-import the pictures.  This has happened to me and its an inconvenience but at least your photos are still there.  An alternative might be digikam which is a kde program that you can install via synaptic.

Reply if any of that is not clear.  If anybody else has another solution I would like to hear it.
