While Exaile is running, it can also be controlled via command line arguments. These can be assigned to a keyboard shortcut in your window manager or desktop environment.

> exaile -n Play next track

exaile -p Play previous track

exaile -s Stop playback

exaile -a Play currently selected or queued song

exaile -t Pause or resume playback
