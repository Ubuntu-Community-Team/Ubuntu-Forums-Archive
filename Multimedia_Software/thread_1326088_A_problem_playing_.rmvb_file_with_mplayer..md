If you have a 32 bit install of ubuntu then add w32codecs package from medibuntu

You can just go here directly and download and install

[http://packages.medibuntu.org/karmic/w32codecs.html](http://packages.medibuntu.org/karmic/w32codecs.html)

If you have any rv30 files then in 9.10 the .so, (drvc.so) has been removed from w32codecs, though there is an easy fix
