### Post by grege on 2010-05-18
If you want to buy Touch apps then you will have to set it up with their software first, to set up the account. Once setup you can use the touch via wifi to buy your apps and can then ditch iTunes.

Rhythmbox will allow you to add and delete music from the Touch using simple mark, copy and paste. Once you do this you cannot then use iTunes again or it will have a sync hissy fit.

So find a Windows or mac box to set it up, preferably one that does not have iTunes currently installed. Then ditch iTunes when you are finished.

### Post by grege on 2010-05-19
@Alistair

You can legally fill an Ipod Touch with music without ever buying anything from iTunes. You can rip your own CDs or by DRM free MP3s from a variety of sources and add them with the iTunes software.

Rhythmbox allows you to add this music as well. As I stated above you use one or the other, but not both. 

If you choose to use Rhythmbox then you do not use iTunes again. If you are an Ubuntu user then you probably do not want to touch iTunes anyway and Rhythmbox now gives you that option, and frees you from the restrictions of iTunes.

But, you still have to set it up initially if you want to download apps via the WiFi connection.
