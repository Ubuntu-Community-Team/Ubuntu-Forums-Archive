### Post by stumbleUpon on 2009-03-09
The firefox addon DownloadHelper has this capability.

[https://addons.mozilla.org/en-US/firefox/addon/3006](https://addons.mozilla.org/en-US/firefox/addon/3006)

On loading a page with some flash content, an rotating icon appears in the navigation toolbar and clicking this icon lets you save the flash file to your desktop.

### Post by lovinglinux on 2009-03-09
Open the web site you want to download from, then click the web site icon in the address bar (the one just before the address) then select "More Information...". A window with several options will pop-up. Select the "Media" tab and a lits of available media will be displayed. Select the media you want. A preview will be displayed in the bottom panel. Click the "Save as " button just above the preview panel (on the right side of the screen). 

Please keep in mind that not all media will be available through this method or trough download extensions, specially if the web site really wants to deny direct access to the file.
