### Post by Mark Phelps on 2013-06-04
I have dual monitors, with different screen resolutions, connected in 12.04 -- and they both work fine.  But, I don't have an xorg.conf file -- because those have been deprecated and are not really needed anymore. ### Post by gordintoronto on 2013-06-04
Also run NVIDIA X Server Settings, see if there's anything useful or surprising in what it displays. For what it's worth, NVIDIA X Server Settings identifies my DVI-connected monitor perfectly, and did the same with the previous two monitors. If that's not true for your setup, I would suspect the cable.
