Currently I'm using Adobe Premier for this, but I'd like to have a Linux solution. So far the usual options cannot handle AVCHD video.

Until recently Cinelerra couldn't edit AVCHD natively and you needed to use an intermediate format.  A brief description of how to do this is at

[http://www.renomath.org/ejolson/video/avchd/](http://www.renomath.org/ejolson/video/avchd/)

Note that Cinelerra 4.1, Kdenlive and OpenShot all natively edit AVCHD on Linux.
