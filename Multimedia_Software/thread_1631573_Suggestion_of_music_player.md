### Post by emterroso on 2010-11-26
Hi.

I recently switched from Windows and I'm still looking for alternatives for some of the programs I used on Windows. The one that IMO was just perfect and I miss the most is Winamp. There are tons of players out there and I don't want to have to install and try each one of them. Is there some player with the following features?

1. Plays MP3 files stored on my HD (I suppose all of them do that, but not doing much more than that is a plus)

2. Have a media library with incremental search that shows data from ID3 tags, playcount, lastplay... (Rhythmbox gets close, but I miss the ability to configure the columns it shows)

3. Smart views/dynamic playlists/automatic playlists. Rhythmbox have this, but they're not as general as Winamp's smart views which could have very complex expressions in an SQL-like language.

4. Provides an easy way to change ID3 tags of multiple files.

5. Have keyboard shortcuts for playback functions.

6. Showing album covers is also cool, but really not a priority.

The least features besides these, the better.

Thank you
