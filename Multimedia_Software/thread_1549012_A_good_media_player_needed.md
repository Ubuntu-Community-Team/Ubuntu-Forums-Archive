### Post by kerry_s on 2010-08-09
i use gnome-mplayer for movies & music.

make sure you install:
ubuntu-restricted-extras

add the medi repos & install w32codecs

add "-lavdopts skiploopfilter=all" to the gnome-mplayer options.

if you experience stutter & have hyper threading enabled in the bios, try disabling it, also disable the execute bits compatibility if you have it.
