What should I check next to understand what's going wrong?

### Post by filofel on 2008-05-10
Case solved. In post
[http://ubuntuforums.org/showthread.php?p=4902387#post4902387](http://ubuntuforums.org/showthread.php?p=4902387#post4902387)
Zorael suggested to check that the nvidia driver wasn't modprobe blacklisted: And indeed it was.
/etc/modprobe.d/blacklist-restricted contained
```
# This file is used to disable restricted drivers
blacklist nvidia
```
After commenting out the "blacklist nvidia" list and restarting gdm, the nvidia graphic logo flashed on the screen, and the x server started properly.

Why nvidia was blacklisted (even though the nvidia driver was enabled in the restricted drivers (jokey-gtk)) remains a mystery to me - specially since the previous nvidia setup worked perfectly fine in Feisty. But heck...

Next step will now be to recreate the proper twinview dual head config, but that sure should be easier with the nvidia driver enabled ;-) .

Thanks Zorael!
