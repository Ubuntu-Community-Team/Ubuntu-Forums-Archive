### Post by WEPrechaun on 2008-07-13
Not sure if you have the same card but I have the same problem with a Sound Blaster Live CT4781. I was able to kill the static (hiss) by using the digital output (black) vs. the analog output (green) then went into the volume settings to enable SB Live analog/digital output jack. I learned that the volume was controlled by the Wave Surround slider. 

Also, check out this link:

[http://alsa.opensrc.org/index.php/Emu10k1]("http://alsa.opensrc.org/index.php/Emu10k1")
