### Post by jcolyn on 2010-08-15
Another alternative is Gnome Mplayer with the w32codecs, and libdvdcss2 installed.

To get the above in your package management search for gecko this will install Gnome Mplayer. Next enable Medibuntu (see link below) and search for w32codecs and libdvdcss2

[https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)
