### Post by markbuntu on 2009-03-16
If your sound is scratchy or stuttering you can edit these lines in the file /etc/pulse/daemon.conf to look like this

```

default-fragments = 5
default-fragment-size =25

```
You can try changing these numbers to get best performance from your sound card/chip. There are also some sound cards/chips that can only be fixed with an ALSA upgrade so you may want to consider that if this does not work for you.

### Post by psyke83 on 2009-04-30

Set your PCM volume back to the proper level. See Part C, Step 3 of my PulseAudio guide for more information.
