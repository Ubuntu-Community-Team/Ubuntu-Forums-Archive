### Post by Melcar on 2009-01-16
If you want a working card "out of the box" that will work with no user modification, you should go for an ATI r5xx based card like a x1650pro or even a x1950pro if you can afford it.  Those cards will have functional 2D/3D as soon as you install Ubuntu, although 3D is limited to opengl1.3.  Movie playback, Compiz, and most open source games should work fine.
If you don't mind installing drivers (not that it's hard anyways, just check the restricted drivers box), a nvidia 6xxx or 7xxx will do, or from the red camp, hd3xxx cards.  I don't know how well a card like a x1650pro would do with Aero though.
As far as brands, Sapphire is a very good ATI vendor.  For nvidia, I always liked EVGA and Leadtek.
