### Post by prince.buster on 2010-05-29
just sharing an observation in regards to the default install of audacity on ubuntu 10.04 lucid lynx.

it would just crash! all the damn time, and unpredictably. got my goat, i gotta say.

seemed like a latency issue, or something along those lines - because it occurred primarily when doing fast operations: e.g holding ctrl and clicking about on the sound file.

solution? use jack. the latest version of audacity (1.3.12 beta) plays quite nicely with jack - a whole lot more nicely than it does with pulseaudio, it seems. hasn't crashed since.
