### Post by Zorael on 2008-05-03
If you're using the proprietary driver, make sure you have the **nvidia-settings** package installed. Pop up Synaptic and search for it.

Once installed, you should get an Nvidia Settings entry someplace in your menus. There is an antialiasing tab in there.
