### Post by uberdonkey5 on 2008-12-15
depends what you want. Cinelerra is very non-intuitive but is supposed to be the best in terms of capability. There are 2 versions, the original, and a community supported version. Best to go for the second version. For details and tutorials:

[http://cinelerra.org/docs.php](http://cinelerra.org/docs.php)

[http://www.thesourceshow.org/node/11](http://www.thesourceshow.org/node/11)

[http://www.robfisher.net/video/cinelerra1.html](http://www.robfisher.net/video/cinelerra1.html)

[http://heroinewarrior.com/cinelerra/cinelerra.html](http://heroinewarrior.com/cinelerra/cinelerra.html)

[http://ubuntuforums.org/showthread.php?t=927376](http://ubuntuforums.org/showthread.php?t=927376)

Blender is a very high speck animation program, but also allows video editing:
[http://eugenia.gnomefiles.org/2008/04/20/video-editing-with-blender/](http://eugenia.gnomefiles.org/2008/04/20/video-editing-with-blender/)

Personally I use Avidemux because it is simple to use, has a lot of capabilities and is much much much much more intuitive than cinelerra. Though I have all three on my computer, I only use Avidemux.

HOWEVER, if you want to edit high def (HD) you may have big problems. Cinelerra is supposed to be able to edit it, but so far I have had no luck even loading HD files. Blender can also edit it, gonna try tonight. There is some script that has been written to utilise fmmpg to transcode HD into high quality AVI, but sound does not sync. Basically I think, for now, HD editing is poorly/not supported in linux. If anyone could contradict this I would love to know how (without writing the damn software myself I mean).

P.S.if you follow the instructions for cinelerra it installs ok, but you have to do it from source code. If you've not done this before you may find it a hassle.
