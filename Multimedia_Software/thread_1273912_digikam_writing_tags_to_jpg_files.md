### Post by Magellan on 2009-09-24
OK...I figured it out.

In case anyone else needs to do this, I found two methods that work.

1)  in the bottom right, there is an option behind the "More..." button to write metadata to the files.  

2)  Create a new tag (I called it transfer) and apply it to every photo (after checking the write metadata to file option).  This causes the metadata write to occur, then remove the tag (you can do this all at once when you are done).

In either case, You have to go album by album and select all the photos before taking one of the above actions.
