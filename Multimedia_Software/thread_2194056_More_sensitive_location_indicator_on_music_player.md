### Post by James Wilde on 2013-12-16
Not the most needed accessory, but I'd like to know if there is an mp3 player for Ubuntu 13-10 which is more sensitive than RhythmBox when it comes to the slider to locate one's position on a track.

The reason is that I am learning a language using mp3 files and sometimes need to move back just a few seconds to hear a phrase again.  RhythmBox has a rather rough and ready location control, and it's usually a matter of a minute or more that one backs when moving the slider.  Ideally one would like to be able to use the arrow keys.

---

### Post by ajgreeny on 2013-12-16
You could use gnome-mplayer which has arrows to move the position 10 seconds forward or backwards, as does mplayer used in terminal with the cursor keys.

VLC will also allow keyboard shortcuts which are all shown in the Tools ->Preferences dialogue box, eg Shft+Left goes back 3 secs.

I'm sure there are others but these are the ones I know of without searching.
