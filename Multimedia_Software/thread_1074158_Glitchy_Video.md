### Post by Mercury_Alpha on 2009-02-19
With most ATI cards currently, videos will flicker like that when compositing is enabled. The workaround is to either disable effects (which will probably affect Cairo) or to use a software video rendering mode like X11. Some media players don't allow you to select which renderer you use, but VLC does.
