### Post by cchhrriiss121212 on 2010-12-22
Try these:
LMMS - Am easy to use DAW like FL
Hydrogen - Drum sequencer
Mixxx - DJ mixer
Phasex, zynaddsubfx, bristol, AMS - synths
Qtractor - multitrack recorder with MIDI and plugin support
Jackd - low latency sound server

If you want tutorials, just type the name of the program into a google video search, you should find something useful. 

> does anyone have any experience with this, be it professional or amateur?
There are a few pro studios out there using linux audio, but they are in the minority. Plenty of amateurs around though, check the ubuntu studio forum. As of today, many of the developers are amateur musicians creating programs that they would like to use.
Pro-audio FOSS relies on jackd, which is complicated to use and set up (at first). It is getting better though, as are the programs mentioned above.
