### Post by ufleisch on 2010-09-21
The "Export" function of Kid3 only exports the data of the current folder. But you could use the advanced "Create Playlist" function (available since version 1.3), there is an option "Write info using" where you can use all metadata. Then you can create a playlist from all folders of your collection.
