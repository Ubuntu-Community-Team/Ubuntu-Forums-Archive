### Post by LeoSolaris on 2008-08-15
The applet makes a call to something in ActiveX in IE. It does something slightly funky to java in other systems apparently. I wonder if this would happen on a Mac.

The work around:

Use the [User Agent Switcher]("https://addons.mozilla.org/en-US/firefox/addon/59") addon for FF3 & the [MediaWrap]("https://addons.mozilla.org/en-US/firefox/addon/1879") addon. (The media wrapper is the important one. I just like adding an extra plug for the User Agent switcher because it is bloody useful...  now, if I could just get Netflix to operate with the media wrapper...)
