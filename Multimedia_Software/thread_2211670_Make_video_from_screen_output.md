### Post by TheFu on 2014-03-17
There are many screen capture programs. These are separate from the drawing programs.
guvcview is one - if you want to video external stuff - webcam-like.
ffmpeg is another - [http://www.commandlinefu.com/commands/view/148/capture-video-of-a-linux-desktop](http://www.commandlinefu.com/commands/view/148/capture-video-of-a-linux-desktop)
mencoder
There is an Ubuntu Guide for this: [http://ubuntuguide.org/wiki/Screencasts](http://ubuntuguide.org/wiki/Screencasts)
