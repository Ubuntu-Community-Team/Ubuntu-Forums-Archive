### Post by ssam on 2013-03-14
dcraw is a command line program that can convert a raw to an image. ufraw is quite an advanced graphical program that lets you open the raw and adjust it. ufraw uses dcraw to do the actual work. if you install the gimp-ufraw package, then when you have done the adjustments you can open it straight into gimp. otherwise with plain ufraw you can save it as a jpeg or tiff and then open that in gimp.

you might also want to look at a raw image manager, like darktable or rawtherapee
