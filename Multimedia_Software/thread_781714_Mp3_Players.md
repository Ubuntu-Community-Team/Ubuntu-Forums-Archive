### Post by yochaigal on 2008-05-04
you need Gnomad2.

There are two ways to install this program.

1) Open Synaptic Package Manager.  System > Administration
     
   Search for Gnomad2.  Select the appropriate check box. Hit Apply.


2) Open Terminal > Applications > Accessories

  Type
           sudo apt-get install gnomad2 
Press Enter
