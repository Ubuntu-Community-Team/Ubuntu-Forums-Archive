### Post by andrew.46 on 2010-10-10
Hi subhanil,

> **subhanil said:**
> Can anyone please suggest how can i convert video files into different formats? Can i use VLC for that? If yes how?

You can indeed use vlc to convert videos from one format to another, although you may eventually find this a little limiting. Simply open vlc and then select Media ---> Convert / Save, fill in the details of the input and desired output file and press the Convert/Save button to select the type of conversion.

All the best,

Andrew

### Post by nechus on 2010-10-10
Try this Nautilus script. It's very easy to use:
[http://gnome-look.org/content/show.php/Audio%2BVideo%2BImage%2BText%2BISO+Convert?content=92533](http://gnome-look.org/content/show.php/Audio%2BVideo%2BImage%2BText%2BISO+Convert?content=92533)

Be ready to use Synaptic to install some packages that the script will require.

You can also try Encode:
[http://www.launchpad.net/encode](http://www.launchpad.net/encode)

Encode is a gui to ffmpeg that allows you to convert audio and video files to various different formats.

And also have a look at DamnVid:
[http://code.google.com/p/damnvid/](http://code.google.com/p/damnvid/)
