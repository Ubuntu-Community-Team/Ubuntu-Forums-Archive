### Post by sccolbert on 2010-01-07
Hello, 

I'm happy to announce the first non-beta release of Pymazon: a python implemented downloader for the Amazon mp3 store. 

Improvements from the beta:
- Running download status indicator
- Various fixes for Windows
- Some code cleanup

Pymazon was created to be a simple and easy alternative for the Linux version of the Amazon downloader, and alleviate the pain of getting it to work with 64bit Linux. 

You can read about Pymazon at [http://pymazon.googlecode.com](http://pymazon.googlecode.com)

You can download from googlecode or the cheeseshop:

$ pip install pymazon

or

$ easy_install pymazon

It also works on Windows. 

Dependencies:
PyCrypto (it's in the ubuntu repos and the cheeseshop)
PyQt4 >= 4.5 (optional, only needed for GUI)

GPLv3 License

Cheers!

SCC
