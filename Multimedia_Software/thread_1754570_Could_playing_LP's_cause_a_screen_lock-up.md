### Post by VanillaMozilla on 2011-05-11
You certainly CAN do other things while you are capturing a sound file.  If you have an old computer and it is not able to keep up, it is possible that it might affect the sound file adversely, but maybe not.

In any case, no program should ever be able to lock up the system.  If it does, that's a system error.  In this case, it's also an error in whatever program you were running (probably the capture program).

If you want a recommendation for a sound capture program, I suggest Audacity.  That's widely used, and quite good.  Crashing is possible with old Windows versions, but I have had excellent luck with the versions in the Ubuntu repository.
