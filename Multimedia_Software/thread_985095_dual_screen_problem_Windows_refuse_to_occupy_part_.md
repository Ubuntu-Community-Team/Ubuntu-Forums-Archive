### Post by K_V_B on 2008-11-17
Hello all,

I've got a Dell Optiplex 755, which has integrated intel graphics. ( 82Q35 Express Integrated Graphics). I managed to configure it for dual screen use, using my two Acer monitors, but I have an odd problem: When I try to move a window to the second monitor (to the right of my primary one) it refuses to go all the way to the right. It get's stuck on the left side, about 1 third of the width of the monitor.
I can move my mouse everywhere, and can bring up the right-click menu everywhere.
I can maximize a window on the second screen, and it will maximize to the size of that screen, but offset a few hundred pixels to the left and down (ie, part of the window ends up on my primary monitor).

The system runs Ubuntu 8.10, which I installed today. I haven't really tweaked anything yet, so I'm just running the stock gnome. I used the System/Preferences/Screen Resolution to set the dual screen mode. My to screens are ACER X223wQ screens, each being 1680x1050 pixels.

What could be the reason for this behaviour? What do I need to do?

---

### Post by K_V_B on 2008-11-17
Strangely the problem solved itself shortly after I posted it.

What I did was:
System/Preferences/Appearance
- Visual Effects.
  - Try to change from "none" to "Normal". 

What then happend was that the system informed me that this was impossible, but strangely my whole desktop is now useable...
