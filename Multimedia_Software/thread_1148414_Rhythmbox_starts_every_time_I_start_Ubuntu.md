I've checked Ubuntu's settings for default startup apps, and Rhythmbox isn't even in the list, neither have I checked the option to save my session on shutdown (I'm not that new a user).

But I did a Google search on this and came up with this: [https://answers.launchpad.net/ubuntu/+question/700](https://answers.launchpad.net/ubuntu/+question/700)

Where do I go from here?

### Post by Bohemenian on 2009-05-06
I solved it. Really easy fix, just not the first thing I thought of. 

So if you ever experience the same problem; check the option for Ubuntu to remember what programs you had running on shutdown and close all apps you have running before shutting down. At the next startup, Rhythmbox wont start (it didnt for me anyways) and you can disable the option to remember the running programs on shutdown.
