[https://bugs.launchpad.net/bugs/238606](https://bugs.launchpad.net/bugs/238606)
You could try using the Alpha version of Flash 64-bit.  It has worked great for me, no more of those grey boxes.
[http://labs.adobe.com/downloads/flashplayer10.html](http://labs.adobe.com/downloads/flashplayer10.html)
