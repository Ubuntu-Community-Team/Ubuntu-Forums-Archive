**In Windows 7 x64 I had a fix to support an ***(Interpolated) ***resolution of: _1152x864_** 
 
**I modded the Intel GMA 3150 driver to support downscaling under Windows 7 x64 like this :P:**
*"Look for all keys in registry named: "Display1_DownScalingSupported"*
*Set all to 1*
*After reboot your computer supports higer resolutions.(Interpolated)*
*This is great for programs that need a higer resolution then 1024x600"*
 
That way all my programs could work and the Netbook could run like a notebook!
