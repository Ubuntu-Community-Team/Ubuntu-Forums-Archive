### Post by ajgreeny on 2011-01-28
No, I'm afraid you misunderstood.

Devede encodes the mp4 files into the standard AUDIO_TS and VIDEO_TS folders with all the normal .vob files and then archives all those into an .iso file.  You then burn an image of the iso onto a DVD and it will play in any DVD player.

Try it out.  It's very easy, though it is a long-winded process, taking a lot of CPU power, and maybe a few hours if there is a full 4.7GB of iso file to make.
