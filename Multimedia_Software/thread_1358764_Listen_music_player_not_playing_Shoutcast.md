Hate to break it to you but listen is not compatible with shoutcast streams. The PLS file that the shoutcast web page downloads contains ip address or the actual streaming server (and also the port number) If you extract that info and play that you'll be able to listen to streams using listen. I've used the same method to play shoutcast streams in XMMS2.
