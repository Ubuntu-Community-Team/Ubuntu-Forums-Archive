The xptv file contains your project. If you want an avi file you have to render your project into an avi file. Click Render and choose the Avi container. The output file is, for example, "my_school_project.avi". You don't need to change the settings, unless you are not satisfied with the result.
This is easier with Pitivi 0.15 because you have a few render presets you can use.
