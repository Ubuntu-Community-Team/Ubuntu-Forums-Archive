### Post by Chargaff on 2009-11-28
Hi,

I had a studio based on Linux for almost 10 years. Did everything I needed with it, record, edit, mastering, mixing, live surround performances for theater, writing ect...

I changed career, but I missed sometimes theses days... What distro have I used ? Ubuntu Studio has been on my studio computers for a few years. I don't know what it's like now, though. There are many option, you have to try... It's a very good idea to have a test machine.

The main software for studio work on Linux is without any doubt Ardour. Build your studio around it. Audio on Linux is wonderful, powerful and versatile, but not that easy for a newcomer...

Don't hesitate to post when you need help !

Good luck
