### Post by tacantara on 2009-10-25
I tried Banshee for my iPod Classic 160 GB, and I ran into some of the same problems that you did.  I finally gave in and installed iTunes on a Windows VM.  I use Sun Microsystems VirtualBox, and I set up a Win XP virtual machine.  This may be an option for you, as long as you have a good copy of a Windows OS that you can create the VM with.  If you decide to go  the VM route, download the software from Sun's website, as it has more features than the version in the Ubuntu repos.
