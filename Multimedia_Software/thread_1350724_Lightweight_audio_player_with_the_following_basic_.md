You might look at Audacious. It does not create a music database. You add files by browsing to your music directories. Loads and saves playlists. Similar to classic Winamp 2.xx - I use it.

The default skin is not that attractive - Refugee looks better and comes with it. Aucacious also uses any classic Winamp skins.
