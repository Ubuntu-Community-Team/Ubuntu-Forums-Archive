### Post by cariboo on 2008-11-25
I use a command line program called randomplay to play my mp3s. I've got about 10,000 music files and I would rather hear them randomly then in any set order. I have the program set to not repeat for 30 days. Randomplay is available in the repositories and can be installed using Add/Remove Programs, Synaptic Package Manager, and of course the command line.
