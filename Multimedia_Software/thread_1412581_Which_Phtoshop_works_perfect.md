### Post by foresthill on 2010-02-21
Wine most definitely runs Photoshop, often faster than Windows will on the same machine. There are workarounds for many of the bugs, like the rubber stamp tool not working properly. I run 5.5 and Elements 2. 5.5 is fine, once you install the ICC profiles so that the colors display properly. But sometimes the toolbar and boxes for layers, brushes, navigator, etc. will disappear mysteriously. This does not happen in Elements 2, nor do colors display wrong like 5.5 (until ICC profiles are installed).

There are minor bugs in all versions I have used, but no real deal breakers as far as I'm concerned. You just have to google the problem and there is always a solution out there.

Gimp is OK, but so different than PS, it's almost like learning how to use PS all over again (a major undertaking).

### Post by malty on 2010-02-21
CS2 run well in wine however, many essential plugins will not, Nik Colour Efex is one example. For eighteen months I have used CS3 in Virtual Box, runs well except, painstakingly slow.
I now use Bibble 5 Pro in 9.04 and except for Nik Efex (polariser is the main usage) and polaroid dust and scratch remover, I can do most things any photographer needs. Bibble has a growing list of plugins, Noise Ninja is brilliant,
Since installation of Bibble, CS3 is virtually redundant. It ain't free but so what, it does the job unlike the odd, clumsy Gimp and the short on features Digikam, which is only useful for shrinking pix ready for EMailing.
I now use VirtualBox simply to run scanning software and scanner, (Plustek, no drivers for Linux.)
