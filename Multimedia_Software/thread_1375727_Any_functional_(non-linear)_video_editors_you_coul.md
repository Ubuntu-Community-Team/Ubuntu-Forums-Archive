### Post by kelvin.illa on 2010-01-08
My experience:

**Pitivi** is very stable and looks clean but severely lacks features since it's only in version 0.13 right now.
[B]
Kdenlive [/B]seemed very good at first. However, the features--at least in my experience--were unpolished. For one, you can only attach one effect to a single clip; clicking on a clip to attach another won't do anything, but you can always drag an effect from another clip to a clip with an already attached effect to circumvent this limit. Also, the playback is very slow to point of being incomprehensible; add the fact that KDE libs have to be loaded as well. Lastly, it does look good in Gnome.

The playback of** Cinelerra (from the akirad repository) **is very slow to the point of incomprehensibility like the above, Kdenlive. 

On the other hand, **Cinelerra (from heroinewarrior.com)** has a perfectly working playback.  It seemed great until I tried to add a 'Title' effect which crashes the application for some reason. I tried to add to the video track a .png image with a transparent background as a work-around but it won't add.

By default, the interface of **LiVES** isn't in a non-linear video editor form but it does have one you can easily switch to (called multi-track mode). It crashes when I try to play the preview in this mode.These accounts are based on my experience and you can always disagree and say that 'it' works for you (well, good for you) but it *might* not change the fact that these aren't working for me.

Maybe there are still some video editors I could've missed so back to my title question: _Any functional (non-linear) video editors you could suggest?_ Otherwise, I'm sticking with Pitivi.

### Post by houseworkshy on 2010-04-18
There are several versions of cinelerra.  I found that the cinelerra cv was the one which worked best for me.  It does crash a lot but has a save and load backup functions which make the crashes less of a pain if one uses them a lot.  It is the best I have seen for Ubuntu.  Open shot is a nice looking frame on 9.04  which I'll try out properly on 10.04.  PiTv seems to depend on plugins for functionality and I can't find them.  Kdenlive is a bit like cinelerra in that it is a wonderful program which crashes often.  Lives, avidimux and open movie editor all also work and also crash on my system.  Kino is good for importing.  Overall, despite frequent crashes, my favorite nle for Ubuntu is cinelerra-cv by a long way with kdenlive second only because it has less functions.
Non-linear Video editing is a bit of a hole in Ubuntu currently but it's closeing.  We have the main funtions in several packages and some advanced features in a few but so far I've failed to find stability in any.  By stability I mean doesn't crash when one is in the middle of a project. Backup frequently whatever you use.  I've found it helps to break any project down into very small jobs, I know that that is one of the normal practices for video editing but for linux ( currently ) it is an adage which needs to be taken even further than usual.
