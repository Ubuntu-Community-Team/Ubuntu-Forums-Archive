### Post by dekinstoke on 2009-01-06
I think you will find the problems with sound in Paltalk Express are not local to your Pcs, but basically due to the fact that Express is still in Beta and not exactly stable.

I have had some success using the Paltalk Scene program under Wine in Ubuntu, with a little bit of tweaking... details below:

You can probably log into Paltalk and get Rooms List up running Paltak under WINE, but you can't access rooms right?

You need to install MSN Messenger 7 under Wine and probably Internet Explorer too (use IES4Linux).

MSN Messenger is available at [www.oldversion.com]("http://www.oldversion.com")

With these two installed I can access rooms, with sound and everything.

You will also need to get MS True Type fonts installed using a font server like xfstt otherwise it looks horrible.

End of the day, if Scene was supposed to run under Linux, the nice people at Paltalk would have made a Linux port ...wouldn't they?
