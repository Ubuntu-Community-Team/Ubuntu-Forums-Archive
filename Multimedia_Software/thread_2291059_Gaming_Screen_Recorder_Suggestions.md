### Post by TheFu on 2015-08-17
Use a hardware device that doesn't need a computer at all. It sits between the video output and the screen.  
[http://www.amazon.com/AGPtek-1080P-Click-Recorder-Capture/dp/B00MQQKDYE/](http://www.amazon.com/AGPtek-1080P-Click-Recorder-Capture/dp/B00MQQKDYE/)  $80.  No software incompatibilities to worry about. For audio input (smack is critical, right?), you'll have to use component inputs, not hdmi.

I also have a Hauppauge 1512 HDMI recorder, but it is Windows only for now.

If you don't need HDMI recording, there are lots of other options - especially when you don't need more than s-video resolution. In that situation, any Linux compatible video recorder will work fine.

I don't know anything about WINE.

### Post by TheFu on 2015-08-17
> **Slavaderp said:**
> I'm not looking for an XBOX or PlayStation recorder, just a desktop one. Do you mean that what you suggested would work for PC, too?

Yep. If it has HDMI or Component video out ... that will work. Zero impact to the PC running the game.

If you just want a screen recorder software - check out simplescreenrecorder - it is a GUI over ffmpeg. You could use ffmpeg directly - both of these options will obviously impact system performance.

The HW based solution is simple, doesn't impact performance, and "just works."  There is a price. $80.  This is cheap compared to the other HW+SW options which are $150-$2500.  Definitely read the reviews before purchasing. There are caveats about the output files, but nothing a bit of transcoding won't handle.
