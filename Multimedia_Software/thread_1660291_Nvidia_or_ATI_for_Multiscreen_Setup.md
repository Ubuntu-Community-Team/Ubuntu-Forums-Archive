### Post by tjones00 on 2011-01-05
I run two dual card multi monitor machines one ati and the other nvidia.

Honestly I'd say they're pretty much the same at this point in time. However, ATI did pull a sleezeball move recently dropping linux support for cards that were over 5 years old. 

Theoretically if you use linux you are suppose to support ATI in spirit since AMD was the company that released it's 64b technicals to the linux community allowing the creation of the amd64 (64b) kernels.

I'd go with the hardware that supports your ideal setup. Try the ATI card but make sure you can return it. If the ATI card doesn't work with the triple setup just send it back. Double and triple check the return policy before you buy.

Nvidia still seems better for video rendering in linux though. So the selling point for ATI is the triple display.

I believe ATI allows for more tweaking in windows for games as well trying to brand themselves more as enthusiast/tweaker hardware to set themselves apart from nvidia.
