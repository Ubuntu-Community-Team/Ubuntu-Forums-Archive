### Post by prince_sabin on 2011-01-04
handbrake can make mpeg4 format files easily(ipods or psp)

here's the ppa:

sudo add-apt-repository ppa:stebbins/handbrake-releases

then:
sudo apt-get update
sudo apt-get install handbrake-gtk


you can also try avidemux(it has more options, but is more complicated)

sudo apt-get install avidemux

Have fun :)
