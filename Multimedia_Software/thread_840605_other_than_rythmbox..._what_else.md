### Post by bakaeigo on 2008-06-25
You can still run amarok within gnome -- installing it through apt-get or aptitude will automatically download the required KDE dependencies required to run it.

```
sudo apt-get install amarok
```

### Post by cozmicharlie on 2008-06-25
You can't go wrong with Amarok and as bakaeigo said, it runs fine in gnome.  It sounds like you want an itunes like player so Amarok, Banshee, Exaile, Quod Libet, Songbird are all good choices.
