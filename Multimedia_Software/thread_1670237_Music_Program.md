### Post by cchhrriiss121212 on 2011-01-19
> **Evil-Ernie said:**
> Oh no! Didn't realise that! So what is the big benefit now of having Studio installed rather than plain Ubuntu?
Studio 11.04 will ship with a low latency kernel as well as all the extra software. But there is no real big advantage to using Studio over generic Ubuntu, you can find all the software in PPAs or the main repo. 

I think the best way to see Studio is as an introduction to FOSS audio/video/graphic software. As linux audio matures there is less tweaking and setup work required to get going, Firewire cards are now plug and play for example.
