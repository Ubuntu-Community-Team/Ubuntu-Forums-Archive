I just open "Nautilus" file browser and navigate to "/media".then "mouse right 'click&hold'" on "cdrom0" icon and select "copy" in the newly opened "mouse menu".then I navigate to the '/home/(your user name)/' folder where I want to place the disk folder and then "right mouse 'click&hold'"  and select 'paste'in the newly opened "mouse menu"then I 'rename' the newly created '/home/(your user name)/cdrom0' folder to the name of the dvd.I can play the newly created folder
with 'VLC media player' by just "mouse right 'click&hold'" on the your new folder's icon and selecting "open with" and selecting 'Open with"VLC media player"'.
VLC opens up and begins playing the "folder" exactly the same as if it was a "mounted DVD disk"or a 'Gmount-iso' mounted "ISO" file.
