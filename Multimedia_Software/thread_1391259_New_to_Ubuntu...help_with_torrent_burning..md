### Post by mjitkop on 2010-01-26
Hi, Brian.

My favorite application to make a DVD is DVDStyler. It is in the Ubuntu repository so you can easily install it right from Ubuntu. Here is the link to the web site for more information:
[http://www.dvdstyler.org/](http://www.dvdstyler.org/).

If your mp4 file contains a standard video compression, you can simply import your file in DVDStyler and it will be converted automatically to the DVD standard. You will also be able to make your own menu.

Here is a link with instructions to make a DVD:
[http://www.dvdstyler.org/index.php?option=com_content&view=article&id=70&Itemid=55&lang=en](http://www.dvdstyler.org/index.php?option=com_content&view=article&id=70&Itemid=55&lang=en)

I'm not saying this is the best DVD authoring application but this is my favorite.

I hope this helps.

Good luck! :D
