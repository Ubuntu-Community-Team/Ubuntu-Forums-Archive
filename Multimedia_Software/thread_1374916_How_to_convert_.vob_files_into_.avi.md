### Post by le1 on 2010-01-07
Install "Avidemux" from synaptic package manager or ubuntu software center, choose the biggest .vob file (..._1) from "video_ts" folder. Follow instructions.

If you have multiple audio tracks choose the one you want from Audio--> MainTrack and save it to file "your-name".avi
