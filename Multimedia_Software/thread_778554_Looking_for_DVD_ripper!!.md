### Post by benerivo on 2008-05-02
Sometimes dvdbackup can be handy. It has a console interface that allows you to select the title or chapter that you want copied, and then creates a dvd folder containing the video. Say for example you wanted to backup an episode of Family Guy, you can select the relevant title and it will take about 2 mins to copy the episode to the hard drive (no quality loss), then you can find the .vob file it has created and change the extension to .mpeg.

EDIT - Plus if you play the .vob file, you will also have the subtitles that can be displayed by vlc, kaffeine, you dvd player under the tv, etc.
