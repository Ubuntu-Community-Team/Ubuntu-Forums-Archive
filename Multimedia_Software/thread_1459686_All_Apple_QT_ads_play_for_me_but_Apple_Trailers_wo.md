> There now seems to be some version checking going on ... If I recall correctly there is a way to get the actual content locally and play it, but this is very step intensive ....berk

i found that by installing "User Agent Switcher" as a Firefox Addon gets around some of the more tedious and tiresome ways.

This method works with MPlayer.

goto > Tools > Addons > Get more addons

> At the top in the "search all addons" box type "user agent switcher"

It should be the top item in the results page 

> then click the green "Add to Firefox" 

> restart FF to complete the installation.

Then, > in the FF toolbar, select > Tools > Default User Agent > iPhone 3.0.

And you're done! Happy Quicktime viewing!
