### Post by Cheesemill on 2013-08-07
You can't run Netflix directly on Ubuntu, but running the following commands will install a patched version of Wine along with the Windows versions of Firefox and the Silverlight plugin...
```
sudo apt-add-repository ppa:ehoover/compholio
sudo apt-get update
sudo apt-get install netflix-desktop
```
