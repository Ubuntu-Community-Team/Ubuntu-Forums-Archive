### Post by hikaricore on 2009-01-11
Incase it hadn't crossed your mind, be sure not to do any read/writes to the card in question until you have a solution!

If I recall SD cards us a FAT format so anything that can recover files from a fat filesystem should work in this case.

### Post by Hal.9000 on 2009-01-11
I have had success with photorec which is part of the testdisk package. Just apt-get install testdisk and then sudo photorec. About a month ago I used it to retrieve about 100 photos that were accidentally deleted from an sd card.
Good luck
