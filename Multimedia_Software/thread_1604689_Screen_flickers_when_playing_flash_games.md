### Post by eival on 2010-11-01
adobe's only solution other than making sure you have the latest version of flash, is to disable hardware acceleration.

right click on the flash an select properties, its the first tab that pops up.

also sometimes flash games have updated version released that fix these types of issues, make sure your playing the latest version of it.


if nothing works it can always come strait down to your hardware just not being efficient enough to run whatever your trying to run smoothly.

also make sure you have the 3rd party hardware drivers isntalled, if you have Nvidia or ATI, ect.. they can fix these issues too.
