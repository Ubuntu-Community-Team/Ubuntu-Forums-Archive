### Post by kostkon on 2008-06-10
Yes, these media players don't recognise videos so you cannot use them to upload video to your device.

A solution would be to use [*Gnomad2*]("http://gnomad2.sourceforge.net/"); it's in the repos.

I think the new version of *Banshee* supports videos, so I assume that you will be able to use it to upload video to the *Zen*, although it needs  confirmation by someone else that this actually works.

There is a PPA repo for *Banshee* so you can easily install the latest version of it if you want.
