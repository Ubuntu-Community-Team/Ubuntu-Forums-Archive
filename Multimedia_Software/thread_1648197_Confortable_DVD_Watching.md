### Post by gobbles414 on 2010-12-18
I recommend that you install a program called VLC. It has a playlist feature which will allow you to add all of the files to a playlist, and I believe that the subtitles will remain active throughout the entire playlist. VLC is available in Ubuntu Software Center / Synaptic Package Manager or from VLC's official website. You may also wish the activate something called Medibuntu, and then install the libdvdcss2 package using Synaptic Package Manager...

[http://www.videolan.org/vlc](http://www.videolan.org/vlc)

[https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)
