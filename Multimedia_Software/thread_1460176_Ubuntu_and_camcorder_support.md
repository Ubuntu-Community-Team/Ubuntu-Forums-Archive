### Post by P4man on 2010-04-22
I have no idea, but why dont you try? You can run ubuntu live session off the cd (so even before installing it), and see what happens when you connect your camera. 

The upcoming version (10.04) comes with pitivi preinstalled, so it will run pitivi from the CD. Its a very basic videoeditor (think MovieMaker), but probably exactly what you need for simple editing for youtube. It supports all common formats.

Ubuntu 10.04 is available as release candidate now, the final version will be out next week. Of course you can also try the current version (9.10). it doesnt come with a video editor by default, but you can install it from the software center. Yes, still without having installed the OS. You will just have to reinstall it on every boot since nothing is written to your harddrive, so you lose changes every time you turn off your pc.
