### Post by Ulysses361 on 2009-10-30
@Muldrake: your sound issue is due to the following:

# ALSA Version
# ------------
#  
# Driver version:     1.0.18rc3
# Library version:    1.0.20

Your ALSA driver and library are not the same version. Please follow this procedure to upgrade to ALSA version 1.0.21:

[http://monespaceperso.org/blog-en/2009/08/31/upgrade-alsa-1-0-21-on-ubuntu-jaunty-9-04/](http://monespaceperso.org/blog-en/2009/08/31/upgrade-alsa-1-0-21-on-ubuntu-jaunty-9-04/)

Make sure that ALSA driver and library are both on version 1.0.21 after upgrading and rebooting.

### Post by neilg4rqn on 2009-11-25
When I installed 9.10 I could not get the sound to work. After a few attempts this is what worked.
1  Make a clean install of 9.10
2  Do not fiddle with pulse settings apart from making sure 
   nothing is muted. 'System/Preferences/Sound'
   (beware if you put a fader at zero it sets the mute ON).
3  Using 'System/Administration/Synaptic Package Manager'
   install 'gnome-alsamixer'
4  Use Gnome Alsa Mixer to control your sound. At first turn
   everything up and ON to see which combination of faders
   is required.

It worked for me and I now have Skype running along with the 
usual suspects.
