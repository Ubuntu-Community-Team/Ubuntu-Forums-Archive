### Post by ron999 on 2010-07-13
Hi
First thing will be to copy the DVD onto your hard drive as one file for you to edit.
If it's a simple DVD with a VIDEO_TS folder then **vobcopy** will do it.
The command is like this:-
```
vobcopy /media/cdrom0/VIDEO_TS
```
