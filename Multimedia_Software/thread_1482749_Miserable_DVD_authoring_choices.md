### Post by GuiGuy on 2010-05-13
I do a lot of DVD/ video editing and authoring, so have to maintain a Windows computer because Linux has always lacked robust and useful software for the task. At least it did when I last looked  about 18 months ago.

Over the last day or so I thought I'd have another look to see if there's been any progress. I limited myself to that software which had a GUI, as command line software doesn't enthuse me. Here's my take on what I found. 

**My Hardware & OS;** ATI graphics, AMD Phemom based MOBO with 8G RAM. Lucid Lynx 64 bit.

All videos in PAL standard, 16:9 aspect ratio is essential (for me):

**Quick snapshot;** all software I tried, except Kmediafactory was crash prone. Only Kmediafactory allowed me to complete a project. As we have come to expect from OSS and linux software especially, documentation and/ or help files were usually totally absent. Where they did exist they proved next to useless. 


**Bombono DVD - **
Cannot control aspect ratio - defaults to 4:3
Crashes if attempt is made to link menu item to video file


**Imagination**
OK as a slide-show maker, but tedious and frustrating to use. No automatic time synchronization between audio and slides. 
[B]
KMediafactory[/B]
Probably the best of a bad bunch. Seems to be the only one that allows control over aspect ratios, but unfortunately menus seem to be 4:3 only. Plays nicely with K3B for burning.

**ManDVD**
No Aspect ratio control. Seems limited to 4:3. Menu building is confusing. Crashes on rendering


**QDVD-Author**
Cannot control aspect ratio; again, it seems to be 4:3 only

Only seems to allow selection of groups of video files; I could not work out how to select a single video file.

Very slow file selection dialog. Insists on building thumbnails, which takes forever and then it promptly crashes.

**devede**
Simple and basic, but hangs as soon as a video file is added


**CONCLUSION**
Only Kmediafactory delivered any sort of result for me. However, as it cannot create 16:9 aspect menu screens it's unsuited for quality output results.

It does seem that if one wants or needs to do serious video work, a Windows or Mac OS PC continues to be essential.

NB: Happy to be proved wrong on any of the above or to be pointed to software not listed above.
