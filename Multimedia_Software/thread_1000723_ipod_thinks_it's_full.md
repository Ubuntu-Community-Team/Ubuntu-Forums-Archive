### Post by cozmicharlie on 2008-12-03
Ipods are odd the way they store files.  Try accessing the folders directly and see if their are any music files still in it.  
[LIST=1]
[*]Mount the ipod (normally just plug it in your usb slot)
[*]Open it by double clicking
[*]Their should be a folder called "ipod control" - open it.
[*]Now open the folder called "music"
[*]You should see a bunch of folders labeled "F, F01, F02 etc).  These folders hold the music files.  Open a few of them and see if they contain the any music files.  If so then for some reason, the programs in Linux are not accessing them (maybe they are drm).  You may have to manually delete them.
[/LIST]  A couple other options:
[LIST]
[*]Reload the original Apple firmware (available from Apples web site).
[*]Go back to a windows computer and delete the music files using itunes.
[*]Install Rockbox (the best way to use an ipod IMHO) and forget Apple altogether.
[/LIST]

Post back if you have problems.

Good luck!
