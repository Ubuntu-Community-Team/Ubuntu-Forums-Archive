### Post by cchhrriiss121212 on 2010-06-18
Firefox is an application (a web browser) so it won't play a sound file for you. If you just want to play the file, you could use audacious with a software synthesizer plugin. Here are instructions for that:
[http://linuxtechie.wordpress.com/2008/01/22/how-to-ubuntu-midi-playback-with-audacious/](http://linuxtechie.wordpress.com/2008/01/22/how-to-ubuntu-midi-playback-with-audacious/)

If you want to edit the file, you would need a sequencer like Rosegarden.
