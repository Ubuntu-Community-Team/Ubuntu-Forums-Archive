### Post by wbee on 2009-12-14
knottshawk,

This might prompt some disagreement.

That's because not that long ago,Creative would not share code with Linux developers,forcing those developers to work backwards.

That said,I got a Creative Soundblaster Audigy SE card for about twenty dollars.

LiteOn Player>VLC Media Player>(the card)>into loss leader 2.1 speakers gives me basic,cheap,KK friendly sound.

[http://www.tigerdirect.com/applications/SearchTools/item-details.asp?EdpNo=4459743&Sku=C44-3400](http://www.tigerdirect.com/applications/SearchTools/item-details.asp?EdpNo=4459743&Sku=C44-3400)

((I got a better deal from E Bay but TD has a good description and reviews.))

**Edit--For some reason,the KK driver does not disable the onboard sound and support the Audigy SE card,**until you disable the on board sound in your bios(or at least in mine).Reboot,and it worked for me.))
