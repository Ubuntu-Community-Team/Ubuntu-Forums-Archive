### Post by sv1cdn on 2010-01-09
Hello world!
Been using since summer 2009 the xserver no backfill from PPA: [https://launchpad.net/~ubuntu-x-swat/+archive/xserver-no-backfill](https://launchpad.net/~ubuntu-x-swat/+archive/xserver-no-backfill) and it was working fine with my ATI video card. No heavy CPU usage when restoring from minimized and a general performance boost were the results of that use.
But recently made an upgrade and realized that xserver-xorg-core package (check it from synaptic manager) has been upgraded to a jaunty-proposed version and the problems started to appear again!
So using Synaptic I selected this package and hit CTRL-E key in order to choose the nobackfill version. This way things got back to normal!
Take good care!
