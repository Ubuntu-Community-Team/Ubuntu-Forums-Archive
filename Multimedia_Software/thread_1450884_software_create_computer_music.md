### Post by cchhrriiss121212 on 2010-04-09
Yes there are plenty of high quality programs that you can download, including DAWs, synthesizers and everything else you would ever use. There is even Ubuntu Studio which is an off shoot of the Ubuntu OS that is tweaked for audio production.
LMMS is probably best for a new user, but I think what you want is some of the things in this ppa:
[http://ubuntuforums.org/showthread.php?t=1350304](http://ubuntuforums.org/showthread.php?t=1350304)

Some highlights:
Jackd sound server - This is essential for low latency audio work and allows you to connect everything up in a modular style of production.
Hydrogen - Drum sequencer
Phasex/zynsubfx/ams - Synths
Ardour - DAW similar to cubase
Qtractor - DAW + midi features

Check the studio sub-forum for lots of great info and troubleshooting
