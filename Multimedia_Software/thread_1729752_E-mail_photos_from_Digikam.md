### Post by cjhabs on 2011-04-15
gmailagent stopped development in 2008.

I use Thunderbird, but before that I used the default Evolution - both worked with DigiKam.
You need to configure the mail client so that it points to your mail server - I was using Evolution to manage my gmail account, so I know it works, but I don't have the details of how I did it.

As I was looking at your problem, I configured Thunderbird to read my gmail account - it took me only a minute to do - take a look at:
[http://mail.google.com/support/bin/answer.py?hl=en&answer=77662](http://mail.google.com/support/bin/answer.py?hl=en&answer=77662)
this is for v2 - I am using v3 so the setting has moved to the Edit tab, but Thunderbird did all the work for me. You can use this setup for sending the images, but still use the web interface for everything else. And configure Digikam to use Thunderbird in this case.
