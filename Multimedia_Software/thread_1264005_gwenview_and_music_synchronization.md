### Post by barna on 2009-09-30
There is a nice application, slideproject.sourceforge.net
written in Java. It can play .wav files (limited capabilities, like no fade-in/out, etc). But it can call external shell scripts at certain slides, so it is the easiest to start up amarok for example for audio playback, and pause it when a movie is coming.
