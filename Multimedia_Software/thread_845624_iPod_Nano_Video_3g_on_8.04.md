### Post by hayden92 on 2008-07-02
Hey, I'm no expert in Ipods, but you can get most of your questions answered here:

[http://www.apple.com/ipodnano/specs.html](http://www.apple.com/ipodnano/specs.html)

In regards to the question of what you need to get stuff on/off the Ipod, download gtkpod using the add/remove application or synaptic package manager.

I'm not trying to shame you by giving you these basic instructions (below), I just don't know how much you know about Ubuntu and installing programs on Linux

Downloading gtkpod is pretty self explanatory and I think you can work it out yourself, if not, here goes:

----------To install using the 'Add/remove' application (simpler)----------------------

Click on Accessories->Add/remove
Once it opens up, type 'gtkpod' into the search bar
Tick the box
Apply

NOTE: If the program doesn't show up, try changing the 'SHOW' feature to 'all available applications' if is is not already.
---------------------------------------------------------------------------------------


----------To install through Synaptic Package manager (more advanced)------------------

Open Synaptic package manager by clicking ->System->Administration->Synaptic Package manager
Or open the terminal by clicking ->Applications->Accessories->Terminal and type 'synaptic'

Once you have done that, search for 'gtkpod' and download all the package files you will need (there should be two or three)
---------------------------------------------------------------------------------------

I worked out how to use gtkpod without any help, it's pretty easy to figure out and you can just google for some help if you really need it ;)


I hope that helps you out
Let me know if it does
