### Post by gali98 on 2009-09-29
If you use the normal Ubuntu (i.e. GNOME) then you already have a pretty good burner.
Just go to Applications->Sound and Video->Brasero Disk Burner

If you want something better, then I recommend K3B like the user above. It's really great.
