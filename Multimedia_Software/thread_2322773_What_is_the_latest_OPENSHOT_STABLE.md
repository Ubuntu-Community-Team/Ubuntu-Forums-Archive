### Post by T.J. on 2016-04-30
The current upstream version from Openshot is 2.0.7: [http://www.openshot.org/download/](http://www.openshot.org/download/).  You can try the PPA if you like.  PPAs can be buggy, but if the version in 16.04 is already broken, what do you have to lose?


The current Debian Stable upstream is: 1.4.3 [https://packages.debian.org/jessie/openshot](https://packages.debian.org/jessie/openshot).  Debian Stable packages are older, but they have been debugged better than Ubuntu's.  I might be able to help you port it or debug the 2.0.7 if the PPA doesn't work out for you.
