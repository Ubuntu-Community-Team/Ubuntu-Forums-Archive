### Post by papibe on 2012-03-12
You can set nautilus arrange items by modification time, and then reverse the order.

The problem is that when you double click on them, you enter the picture viewer (eog). The latter does not communicate in any way with nautilus and does not have any setting to sort the pictures using file modification time.

My suggestion: install gThumb. It does has an option to browse the pictures in that order.

Hope it helps.

### Post by papibe on 2012-03-12
Sorry If I mislead you, it wasn't my intention.

What I wanted to explain you is that, although gThumb doesn't 'coordinate' with nautilus either, it has its own internal order.

In gThumb go to:

```
View -> Sort Images -> by Modification Time
```
In order to work, it may be necessary to refresh the view (F5) or even restart gThumb.

After that you are all set.

I hope this time the above information can help you.
Regards.

That is odd since I got it working using modification times.

Did you try cheking:
```
View -> Sort Images -> Reverse Order
```
Also, try either restart gThumb and/or reloading the pictures (Ctrl+R).

Once on fullscreen, you go to the next picture by pressing the 'space bar', and going back to the previous using 'backspace'.
