### Post by Jonor on 2013-10-14
No need to use Rhythmbox for your Clip.
First make sure the USB mode on the clip, found in Settings>System Settings is set to MSC which is much easier and quicker IMHO.
Go to the source folder of your music and copy the tracks to the clip board and then paste them into the MUSIC folder on the Clip.
I suggest giving at least 5 minutes for every GB pasted, USB is slow.

Are you using a simple file browser such as Files, which is in the Ubuntu Software Centre ?

### Post by Nistegmos on 2014-07-03
you may want to reformat the Clip if you can, and reset it to USB Mass Storage mode instead of MTP mode.  Each mode has it's own storage on the Clips, so you'd be wasting space by having leftovers on the MTP mode side.  But I could be wrong about that.  I was never 100% sure either way.  But once you get a fresh system with USB Mass Storage mode, then it's all good. And you can use any file manager on almost any OS (as long as the file system is FAT32, which it usually is), and put your music files into the music folder.  

Good luck.  Check the SanDisk Sansa Clip forums.  There is tons of good info there.
