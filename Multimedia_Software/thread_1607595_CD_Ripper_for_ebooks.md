### Post by rykel on 2010-10-30
What I did in the past was to use CD Juicer (the Ubuntu default CD Ripper) to rip the CDs, and then use EasyTag to edit the mp3 metatag information in batches. Finally, I rename the actual filenames as a batch using gprename.

The above works for me, but yes, I wish that there is ONE CD Ripper that does all of the above - or, since all 3 programs are free software, combine their codes into one single application.

CD Juicer is also hard to use because their "Edit Profiles" requires me to understand what those archaic words under "Gstreamer Pipeline" mean, in order to change the quality of the final mp3.
