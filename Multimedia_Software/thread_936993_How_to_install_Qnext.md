### Post by cariboo on 2008-10-03
Just double click on the file you downloaded, File-roller will open, click extract, extract it somewhere in you home directory. Open Nautilus (Places-->Home Folder) double click the directory you just created, double-click qnextclient and you running. Make sure you have java installed. If you don't, install ubuntu-restricted-extras. This will install java, flash, mscorefonts as well as the codecs you need for most audio/video file playback.
