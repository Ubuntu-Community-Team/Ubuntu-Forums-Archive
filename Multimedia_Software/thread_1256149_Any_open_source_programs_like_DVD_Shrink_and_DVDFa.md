### Post by lykeion on 2009-09-02
Some general info on DVD ripping in Ubuntu:
[[COLOR="Blue"]RestrictedFormats/RippingDVDs[/COLOR]]("https://help.ubuntu.com/community/RestrictedFormats/RippingDVDs")

Regarding DVD Shrink and DVDFab; I don't think there are any "clones" of these programs. 
However there are some very good dvd transcoding applications like:

**mencoder** (command-line) [part of mplayer] 
Homepage: [[COLOR="Blue"]http://www9.mplayerhq.hu/design7/info.html[/COLOR]]("http://www9.mplayerhq.hu/design7/info.html")

**ffmpeg** (command-line)
Homepage: [[COLOR="Blue"]http://ffmpeg.org[/COLOR]]("http://ffmpeg.org")

Info: [[COLOR="Blue"]Video transcoding with ffmpeg/mplayer[/COLOR]]("http://videotranscoding.wikispaces.com/?token=23a6250c99d7f8cc9bbbbd70ce80c39f")
or search the forums (or google) for how to use mencoder/ffmepg

**dvdrip** (gui) [a.k.a dvd::rip]
Homepage: [[COLOR="Blue"]http://www.exit1.org/dvdrip[/COLOR]]("http://www.exit1.org/dvdrip")
Info: [[COLOR="Blue"]https://help.ubuntu.com/community/DVD::Rip[/COLOR]]("https://help.ubuntu.com/community/DVD::Rip")

**handbrake** (gui)
Homepage: [[COLOR="Blue"]http://handbrake.fr[/COLOR]]("http://handbrake.fr")
Some info here: [[COLOR="Blue"]http://netcodger.wordpress.com/2009/01/13/dvd-transcoding-with-handbrake[/COLOR]]("http://netcodger.wordpress.com/2009/01/13/dvd-transcoding-with-handbrake")

Also DVD Shrink & DVDFab can be installed and run with **wine** (but I haven't tried it myself):
[[COLOR="Blue"]HOWTO: Install and Run DVDfab Decrypter with wine[/COLOR]]("http://ubuntuforums.org/showthread.php?t=132336")
[[COLOR="Blue"]Wine AppDB: DVDFab Decrypter[/COLOR]]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=14346")
[[COLOR="Blue"]HOWTO: Install DVD Shrink with Wine in Breezy[/COLOR]]("http://ubuntuforums.org/showthread.php?t=78611")
[[COLOR="Blue"]Wine AppDB: DVD Shrink[/COLOR]]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=2230")

Good luck! :)
