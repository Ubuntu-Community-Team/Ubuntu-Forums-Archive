### Post by nermaljcat on 2008-11-06
Try using (on the command line): ```
mplayer -vo x11 <video file>
```
If that works, edit /etc/mplayer/mplayer.conf and change the vo option to include x11. eg. vo=fglrx => vo=fglrx,x11

This fixed it for me.
