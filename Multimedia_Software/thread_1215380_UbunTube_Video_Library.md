### Post by nesh87 on 2009-07-17
Hey all,

Ive created a new program (python + pygtk) that allows you to create a video library. You can import youtube links or local videos. Its still in its alpha stages, still needs some work but is useable.


Overview (from the website):
> 
UbunTube is a graphical video library which is capable of importing youtube files or local files into a single application. UbunTube is written using GTK+ with the use of glade and pygtk.




A few screenshots:
[[IMG]http://img16.imageshack.us/img16/6363/listl.th.jpg[/IMG]](http://img16.imageshack.us/i/listl.jpg/)  [[IMG]http://img441.imageshack.us/img441/1235/player.th.jpg[/IMG]](http://img441.imageshack.us/i/player.jpg/)  

[[IMG]http://img16.imageshack.us/img16/7443/fullscreenc.th.png[/IMG]](http://img16.imageshack.us/i/fullscreenc.png/)


Features:
* Import Local Videos
* Import YouTube links
* Create Thumbnails and fetch video details
* Play videos (fullscreen implemeted) and default controls
* Remove videos
* Tray Application
* Video thumbnails may be overriden

Bugs:
* Mplayer refuses to start silently when fetching details

Future Implementation:
* Probably change the name, its taken.
* Implement the sidebar
* Add config files
* Implement video seeking when playing
Requests are welcome
