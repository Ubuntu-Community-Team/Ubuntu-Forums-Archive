### Post by SeijiSensei on 2013-12-15
Silverlight is a proprietary Microsoft technology.  There is an open-source clone called "Moonlight" but it lacks some of the key features of Silverlight, in particular support for its "digital rights management" code.  Oftentimes, you are stuck using Windows to view Silverlight videos.
