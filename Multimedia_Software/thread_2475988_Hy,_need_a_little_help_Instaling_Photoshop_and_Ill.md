Some people have gotten really old versions to work under WINE, I understand.  "Really old" is about 15 yrs old versions.

There are many reasons to used Adobe software, but it comes down to "I make my living with Adobe software."
If you don't earn a living with the software, I'd recommend switching to a native Linux tool ASAP. It will initially suck. Change always does, but after a few months you'll have figured out how the Linux versions accomplish similar things.

Not really the same, but I stopped using MS-Office around 2008 and started using OpenOffice, then LibreOffice.  Fortunately, the change to a "ribbon" control in MS-Office made the change to OO much more necessary.  I'd been using MS-Word since the DOS 5.0 days and wasn't interested in learning their new interface.  Plus, 99% of the time, HTML output is what I should be using when strict page layout wasn't needed.  I worked in document management and publishing.  The rules for the best way to store information is to use the most common format possible and only add minimal layout when needed.  Eventually, the layout must be fully controlled - say for a magazine advertisement - that's about the only reason to use strict page layout tools.  HTML has sufficient layout controls for 95% while keeping the actual data in a standard format and allowing insertion of external media/content easily.
