### Post by cotcot on 2010-04-14
> **gagea said:**
> Thanks a lot. Is there any book apart from the online tutorials for Blender.
Yes there is book called [[COLOR="Red"]Essential Blender[/COLOR]]("http://wiki.blender.org/index.php/Doc:2.4/Books/Essential_Blender"). Be aware that this book is for blender 2.4x. Blender 2.5 will be available somewhere in summer. The user interface changed a lot. That will make this book outdated for 2.5. 
Once version 2.5 is out you will still be able to use 2.49. So my suggestion is to learn blender 2.49 with the book and switch to 2.5 afterwards. Once you are confortable with 2.49 the switch to the user interface of 2.5 is not a big issue.
