I just stumbled across Easytag. Wow, what an awesome app. I'm using the CDDB to find songs and update all the tags and file name. The one thing I want to do but haven't figured out how is to not have the track number included in the file name when I update the tags with CDDB. 

Does anyone know how to configure things so that the track isn't automatically part of the file name?

---

### Post by cforput on 2012-02-18
Marking as SOLVED. I figured it out.

1. Highlight files you want to change
2. Click on Scanner
3. Select Rename Files and Directories
4. Up in only %T
5. Close
6. Click on the disk icon for Save File(s)
7. Answer yes, you want to rename

Presto, all done!
