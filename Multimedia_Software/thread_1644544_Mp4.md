### Post by lykeion on 2010-12-13
You can use Avidemux.
* Open first mp4 in Avidemux (select Use safe mode if asked)
* File > Append (Select next mp4)
* To save, set Format dropdown to MP4, File > Save > Save Video... and enter a filename, like combined.mp4
