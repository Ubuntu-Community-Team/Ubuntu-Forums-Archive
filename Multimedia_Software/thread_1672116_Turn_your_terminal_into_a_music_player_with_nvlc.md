### Post by shantiq on 2011-01-21
if you link this with nautilus-terminal you end up with a very quick way to play tunes from inside a folder

and the controls are really fast    keys   a and for volume

up and down arrows for playlist
left and right arrows for moving forward on a track
Ctrl+s  for status


possibly faster to manipulate then the gui








[CENTER][IMG]http://img534.imageshack.us/img534/6910/screenshot2nf.png[/IMG][/CENTER]









Quick way to install nautilus-terminal is here



```
sudo add-apt-repository ppa:flozz/flozz 

```
```
sudo apt-get update 

```
```
sudo apt-get install nautilus-terminal 

```
NOTE :After Nautilus Terminal installation you have to restart nautilus using the following command from your terminal : 

```
nautilus -q && nautilus & exit 

```
and then you can control it (hide/reappear) with Ctrl+Shift T and can be customized from the thingie on the right 


from    [http://www.ubuntugeek.com/nautilus-terminal-an-in](http://www.ubuntugeek.com/nautilus-terminal-an-in)...
