### Post by Arup on 2009-11-04
One word, Logitech, the Phillip chipset work out good with GSCPA drivers. MS uses Microdia on some of their cams, notably VX-6000, you don't really get good image quality. There are other variants with Microdia chipset, some with LED lights. Those LED lights never come on unfortunately so lots of tweaking needs to be done for a usable image.
