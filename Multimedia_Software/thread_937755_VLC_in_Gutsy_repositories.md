### Post by DJ_Peng on 2008-10-04
There's actually an [update]("http://www.ainotenshi.org/2008/09/15/vlc-092-released/") in the works. If you're using an alternate skin I strongly suggest you switch to the default skin  before you run the update. I had some [pretty serious issues]("http://nancib.wordpress.com/2008/09/17/about-the-new-vlc-ppa-builds/") because I didn't do that.
