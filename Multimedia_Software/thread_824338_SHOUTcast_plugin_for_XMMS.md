### Post by markbuntu on 2008-06-11
I don't know of any plugins but if you are running xmms then you can use streamtuner. If you are using xmms2 or alsa or pulseaudio try tunapie. 

Streamtuner was written for xmms which is deprecated, i.e. no longer supported, a new version is on its way that will work without xmms.

These are available in the repos so you can use synaptic to get them.
