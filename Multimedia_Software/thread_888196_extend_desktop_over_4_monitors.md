### Post by Geoscientist on 2008-08-12
After failing with a pair of radeon cards, we have opted for a pair of nvidia 8600 cards. 
With very little fuss, it is set up with 4 monitors. Each pair of monitors is configured in twinview. That is, a pair of independent screen pairs. So the cursor moves through the line of 4 monitors, but individual windows can only be dragged within each distinct monitor pair.  

Is it possible to extend the desktop over all 4 monitors?

---

### Post by Geoscientist on 2008-08-12
In nvidia X Server settings, configure the 4 monitors as X screens, and tick the checkbox for enable Xinerama. 

Desktop extended over 4 monitors!
