### Post by cariboo on 2008-11-24
IF you are using Intrepid you shouldn't need to edit your /etc/X11/xorg.conf file. Does System-->Administration--NVIDIA X Server Settings detect your monitor properly? If so you can adjust your monitor resolution using the above utility. THe changes won't take place until after you restart X. TO restart X press Ctrl-Alt-Backspace.

If you are using Hardy press Alt-F2 and type:

```
gksu displayconfig-gtk
```

to adjust your resolution. Remember to restart X.

Jim
