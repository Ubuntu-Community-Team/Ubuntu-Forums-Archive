### Post by ghostborg on 2009-07-27
I use DVDfab HD Decrypter free, works well under Wine.
Use NT4.0 setting.
Rips and shrinks to an ISO.

K9copy has an Assistant which walks you through the process.

Handbrake .93 is awesome. Look again they have a GUI.

I use an AMD quad core and it encodes an typical movie in about 30-40 mins. Xvid AVI ,2pass,1500bit rate AC3 passtrough.
