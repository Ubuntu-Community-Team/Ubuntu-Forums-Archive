### Post by FakeOutdoorsman on 2012-12-22
MP4 container support for softsubs is not very good. Perhaps the device can handle MKV container which is more flexible. If you're referring to hardsubs (where the text is "burned" onto the video) see [this thread](http://ubuntuforums.org/showthread.php?p=12415136#post12415136). I haven't tried it, but in that same thread evilsoup mentions that special characters are not handled correctly.

### Post by evilsoup on 2012-12-23
Just for clarification, the special characters thing refers to the filename of the subtitle file (commas and apostraphes seem to throw it off); I have had no problems with punctuation etc *within* the actual subtitle. I don't know how it handles non-latin characters, but I wouldn't have thought that would be a problem.
