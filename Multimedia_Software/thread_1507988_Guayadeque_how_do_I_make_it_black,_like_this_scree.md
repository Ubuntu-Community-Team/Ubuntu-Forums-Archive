Guayadeque uses the system theme colours. If your theme is dark it will look dark. So if you want it to look use a dark theme.

In the guayadeque thread it was explained long ago how to make a theme be loaded only for one application so you can have that look only for guayadeque. Dont remember exactly who posted that info but you can try finding it searching for the word theme.
