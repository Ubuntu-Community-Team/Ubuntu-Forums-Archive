EDIT: Nm, fixed it. Changed:

```
    Screen      1  "Screen1" RightOf "Screen0"
```
To
```
    Screen      1  "Screen1" LeftOf "Screen0"
```

Can't believe I missed that, sorry for the post - Although hopefully someone else as stupid as me will have the same problem and see it one day. I had tried swapping the Screen#'s around, but not changing RightOf to LeftOf *hits head*.
