### Post by Oscareightyone on 2011-01-23
If you are using the standard ubuntu installation there should be a program called brasero disk burner installed. Just choose 1:1 copy and choose. You can burn it too a file or onto another disk. If you burn it to a file, you can burn it onto another disk later. This is probably the easiest program to do this in.

Applications==>Sound and Video==>Brasero disk burner
