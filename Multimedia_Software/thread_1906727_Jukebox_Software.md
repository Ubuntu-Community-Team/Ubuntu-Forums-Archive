### Post by hellarough on 2012-01-10
Man I am searching for something similar. What I really want is a web interface to control the music playing on my server/stereo that allows for random/dynamic playlist from any device on the local LAN. 

As for you, here are a couple of different configurations that seems to be pretty common and fulfill most users needs:

1. MPD  + [[COLOR="Red"]your client of choice[/COLOR]]("http://mpd.wikia.com/wiki/Clients")

2. Rhythmbox + [[COLOR="Red"]plugins[/COLOR]]("http://live.gnome.org/RhythmboxPlugins/ThirdParty") (specifically rhythmote or rhythmbox-gwt)

3. Subsonic (the jukebox feature allows local playback)

but if you want a touchscreen jukebox like at the bars I haven't seen much for Linux in that realm. Perhaps you can use WINE to run the windows apps. I am interested to see what other ideas there are out there.
