### Post by p-dh on 2009-11-26
Hi Nick,

It's fairly easy to add any Shoutcast stream you want.
1. Find the stream and copy the address, which should include the IP and port # (e.g. [http://208.53.158.48:9538/](http://208.53.158.48:9538/)).
2. In Songbird, open a location (under the File menu item) and paste in the address.
3. A new tab will open. Once it starts playing (you may need to click on a link if the page requires it), you will see the stream address listed.
4. In the listing next to the address, you should see an icon under the heading "Download". Right click on the icon and choose "Save to playlist..." Choose whatever list you want, you can even create a new one.

You need to remember that manually added Shoutcast streams may stop working when the server changes the port number for that stream. You'll need to go through the process again.

Let me know if this helps.

Paul
