### Post by ghostborg on 2009-10-27
Check your sound settings if you can find them  eg-mic,line etc. to make sure they are not muted.
Check the application that you were using at the time to see if that is still muted also- I do that all the time.
Check your speaker volume too. I know but had to say it.
 If all else fails reboot. Leave more system info for other more knowledgeable people. Like your distro 8.10, 9.04.etc. sound card info.
