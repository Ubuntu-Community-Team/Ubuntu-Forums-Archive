### Post by SeijiSensei on 2011-11-20
Aegisub is pretty much the gold standard for [anime fansubbing]("http://forums.animesuki.com/forumdisplay.php?f=17").  It supports [SSA/A** subtitles]("http://en.wikipedia.org/wiki/SubStation_Alpha") which allow for various fonts and the ability to locate multiple subtitle streams on-screen at once.  The Matroska (MKV) container format, and software that uses libass like mplayer, are fully compliant with these subtitles.
