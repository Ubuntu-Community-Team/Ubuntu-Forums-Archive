### Post by circlemaster on 2008-08-28
Hi.

I have been using Audacious as my music player for as long as I can remember and now I want to extend my experience, if anyone knows how to do this I would be grateful:

When I go to sleep I want Audacious to play, and fade away after a set time (30 minutes for example).
When I am to wake up I want Audacious to start playing again.

---

### Post by circlemaster on 2008-08-29
I solved it myself.
Enabled the "Alarm" plugin that comes with Audacious and that solves the wake-up part. Then I start playing a song and type in a terminal "sleep 30m; audacious -s" where you can change 30m (30 minutes) to 1h (1 hour) or whatever duration you wish it should play before stopping.
