### Post by sports.car.guy on 2010-01-29
Most of the Dell Laptops need additional packages for the buttons to work. Check the repositiories and see if there is a package that provides the functionality for your laptop. I know my D600 Dell has been like that in the past, haven't updated it yet so not sure if it will be for 9.10, the same case I have experienced before.

### Post by xboxswimmer on 2010-02-01
Solved! I went to System: Preferences: Keyboard and chose the Dell Multimedia USB Keyboard as my layout. All the buttons work perfectly now!
