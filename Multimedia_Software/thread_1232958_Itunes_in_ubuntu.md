### Post by martinbaselier on 2009-08-06
If you install wine you can then use it to start windows programs. 

about wine:
[http://wiki.winehq.org/FAQ](http://wiki.winehq.org/FAQ)

about wine and itunes:
[http://appdb.winehq.org/objectManager.php?sClass=application&iId=1347](http://appdb.winehq.org/objectManager.php?sClass=application&iId=1347)

A nice open source alternative for itunes is songbird. It uses about as much resources as itunes does. It's based on firefox so it's extendable with plugins in the same way. It's nice player with lots of options. It's not in the repositories though, so you would have to download and install it yourself from their website. 

I personally prefer banshee. It's smaller and has less options, but it's perfect for just playing music in the background.
