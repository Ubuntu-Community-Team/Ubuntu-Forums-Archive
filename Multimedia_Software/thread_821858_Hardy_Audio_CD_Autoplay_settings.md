### Post by Johnny K on 2008-06-07
In previous versions of Ubuntu, the program located at *Preferences > Removable Drives and Media* would allow me to set, change, or disable autoplay settings for audio CDs. For example, I could set my preferences so that totem always started when an audio CD was inserted.

In Hardy Heron, the "Removable Drives and Media" program no longer allows me to do this.

It seems that the mysterious disappearance of this GUI feature would hurt Ubuntu's usability, and in order to help improve Ubuntu I wanted to bring this issue to light.
