### Post by lovinglinux on 2010-09-12
The problem is that you are using Ubuntu 8.04, which does not have flash version 10. It says you have, but is a fake version.

Additionally, don't install multiple plugins.

Get [FLASH-AID](https://addons.mozilla.org/en-US/firefox/addon/161939/), an extension to remove conflicting flash plugins and install the proper version according to your browser architecture.

If you still can't run flash properly after running FLASH-AID and if you have a 32bit system, then you get the latest version from [Adobe directly]("http://get.adobe.com/flashplayer/").
