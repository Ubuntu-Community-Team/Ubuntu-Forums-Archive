### Post by ajgreeny on 2009-07-24
I've given up on mozilla-mplayer, which I assume is what you are using, and moved to gecko-mediaplayer.  It seems much better and does not crash, which the mozilla-mplayer plugin was prone to do a lot of the time.

Give it a try.Uninstall mozilla-mplayer first and then install gecko-mediaplayer.  It will bring in some dependencies, including gnome-mplayer which is actually a useful app in its own right, I have found, and is what I use now to listen to realplay streams from the BBC.  I used to use realplayer, but did not like it and was glad to get rid of it.
