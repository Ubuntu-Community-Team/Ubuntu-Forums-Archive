### Post by lovinglinux on 2009-09-13
If you just want to watch TV, then tvtime should work for you. There is also xawtv and others.

If you want a full features media center, then there is MythTV, Freevo and others.

If you want a simple media center, with EPG, xmltv support, reminders, automatic recording of weekly TV series, playlist manager and other stuff, then I recommend my extension for Firefox, called [FoxMediaCenter]("http://fmc.isgreat.org"). It still experimental, but fully functional.
