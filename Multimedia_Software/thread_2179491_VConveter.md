### Post by NeebxKN on 2013-10-08
I have just finished writing a simple python script that I thought I would share with the community. This script can be used to convert multiple .wmv files in a directory to .mp4 files in the same directory. It is very simple to use all that is needed is the full path to the directory. 

Example:
/home/username/Videos/

Requirements
avconv Version 8.6+
Python

This would be the proper syntax the script takes including the trailing slash. There is a lot of improvements that can be added to this script I will be working on some of them when time lets me. However if you want to write some improvements yourself you are more than welcome to do so. I will also be adding more conversion formats this just the one that suited me at this moment since i needed 200+ .wmv files converted and I did not want to do it manually. Avconv is required to use this script because this is the program that is called to do the converting. Anyways Enjoy the script and post back if you have any questions.
