### Post by dark fader on 2011-02-01
Is anyone effectively using an Ubuntu system as a media center?

I have an old Mac Mini which got itself a corrupted hard drive just after I bought myself a new one. I rescued it with an Ubuntu install and it seems to run more smoothly than OSX did.

Now I could reinstall OSX, get an EyeTV USB adapter and get an Apple remote, but I did that before and it was painfully slow. And the hardware, being for a Mac, would be a lot more expensive.

I'm wondering if I could replicate the same functionality with Ubuntu, without scaring off my non-geek other half.

I need to be able to pause and rewind TV. (Record isn't as important). Remote sync with an iTunes library would be nice but not essential. It would be good to do this with a remote control and/or wireless USB keyboard with track pad or track ball.

So, is anyone doing this effectively? Is it possible to set up an Ubuntu media center that 'just works' (in that way that a Mac sort of almost does, if you squint a bit and don't mind all the random pauses, crashes and necessary reboots)?

Or more simply, has anyone got full digital TV functionality from a cheap USB adapter? If so, I'd love to know the model and software you're using.
