### Post by tgalati4 on 2014-06-02
If you are running a Gnome2 desktop environment, such as MATE, then you will see the tabs inside the window.  gpodder would be difficult to use without those tabs.  I don't run Unity, so I presume they show up in the top of the display (Mac style menus).  If they don't then you have an issue.  You should see something like:

Podcasts   Subscriptions  Episodes Extras View Help
