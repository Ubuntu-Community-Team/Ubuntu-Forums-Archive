### Post by markbuntu on 2008-05-14
what I do is pick to the stream in firefox and when it starts up use CTL-c to copy the url and then drop it into the rythmbox "add new station" entry box with CTL-v and then see if it works. Rythmbox will ask if it can look for new codecs if it can't open the stream and synaptics opens so you can pick them and install them. If the station streams in a number of formats I try the differnet ones until I find the one that works. A few stations are using a new Windows format that this does not work with but a great majority do.

good luck.
