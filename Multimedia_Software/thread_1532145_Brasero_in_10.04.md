### Post by Don DeGregori on 2010-07-16
I was able to get Brasero to rip selected CD tracks under Audio Project.
Here are my steps.

Don


Note:  Copying a track right from the source CD will not work in Basero under Audio Project. Under Data Project this will work, and many steps can be cut. But Audio Project has more options just before burning.	


1.	Under Places, Music, create new folder
2.	Insert Audio CD you want to rip and burn. Shows on Desktop.
3.	Audio Disk window, choose Open Folder, OK
4.	Select track, right click, Copy
5	Click Music folder in left pane
6.	Open newly created folder
7.	Right click, Paste, then Back arrow
8.	Continue this process till all wanted tracks are copied
	to new folder. Then close folder
9.	Right click on Disk icon, Eject, remove disk
10.	Insert Blank CD. Will see icon on desktop
11.	Click Applications, Sound & Video, Brasero Disk Burner
12.	Click Audio Project
13.	Click green plus.
14.	Double click Music folder
15.	Open newly created folder
16.	Highlight all tracks you want to burn.
17.	Click Add
18.	Repeat step 16.
19.	Click Burn
20. 	You should now see Properties of... (your CD drive)
21.	Select Option you like
22. 	Click Burn, or Burn Several Copies
23.	Burning CD window shows, and burning starts
24.	Success when finished
