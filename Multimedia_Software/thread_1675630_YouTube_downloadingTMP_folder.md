### Post by lovinglinux on 2011-01-25
In the most recent versions the folder is indeed the cache instead of the tmp. But some web sites still save in the tmp folder. YouTube for example.

Best video downloader in my opinion is [Video Download Helper]("https://addons.mozilla.org/en-US/firefox/addon/3006/"). However, some sites don't store videos in the cache, because they use real-time streaming protocol.

Keep in mind that storing videos from some web sites like YouTube is against the service EULA.
