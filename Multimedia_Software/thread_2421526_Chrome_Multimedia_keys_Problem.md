### Post by webaake on 2019-06-23
Google Chrome 75.0.3770.100 (and since version 73.x) has taken control of hardware multimedia keys on keyboard. I'm trying to disable this in Chrome with no success. This is what I've done: 
1. Load chrome://flags/#hardware-media-key-handling in the Chrome address bar; this should open the right flag when loaded.
2. Set the Hardware Media Key Handling flag to Disabled.
3. Restart Google Chrome.

And I've unistalled a bunch of addons. And I've checked the remaining 3-4 addons (no multimedia addons installed at all) for possible keyboard shortcuts.

So following all advice I've found nothing that works to stop Chromes new behaviour. In Ubuntu for example, I've set the hardware volume keys on the keyboard to respectively mute, lower and raise the volume. Chrome disables this and instead loads a new default page where I was playing a youtube video. Funny thing is; the volumes adjustments works as well! Sometimes new "features" only brings problems!

Anybody got some luck with this on Linux Ubuntu?
