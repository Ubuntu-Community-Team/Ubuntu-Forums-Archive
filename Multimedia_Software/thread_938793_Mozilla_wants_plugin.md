### Post by concord on 2008-10-05
There's a .dcr file embedded in the page.  This is an interactive media file saved in the Shockwave format for distribution on the Web; similar to a Flash (.SWF) file, but created with Adobe Director rather than Adobe Flash; often used for creating Web games and other highly interactive content. As far as I know there is no version of the player for GNU/Linux yet.
