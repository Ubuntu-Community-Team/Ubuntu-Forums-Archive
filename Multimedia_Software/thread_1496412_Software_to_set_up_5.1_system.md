Hi!

I have a simple analog 5.1 sound system connected to my onboard sound card. It is connected through three component 3.5mm cables. If I understand correctly, that method is called Component, it's one cable for Front L/R, one for Subwoofer/Center, and one for Rear L/R.

When I play back DVDs or DVD-A with this setup (using VLC), some DVDs (e.g. Mike Oldfield's Tubular Bells 2009 5.1 remix, or Jackson Browne's "Running on Empty" DVD-A, or 300) sound fantastic: there is ample usage of all channels including the subwoofer. These mixes are stellar.

Other DVDs (e.g. Metallica's 5.1 mix of the Black Album, "Foreigner - 4" DVD-A, the European DVD of "American Beauty") have an equally great surround mix, but there is hardly any sound coming from the Subwoofer! Everything comes from the tinny little speakers, including drums, bass etc., making the recordings sound awful.

I don't think these DVDs were mixed to sound that bad! Could there be a problem in how my system addresses the 5.1 system? Besides my computer, I have a DVD/SACD player that allows me to customize which kind of 5.1 system it's connected to, e.g. I can set my Front/Center/Rear speakers as **large** or **small**, or Subwoofer on/off.

I haven't connected that player to a 5.1 system yet, but I would assume that if I specifically tell it I have small (i.e. no bass part of their own) Front/Center/Rear speakers, it will change the way that bass is distributed through the system, with more Subwoofer usage.

Long story, I know - now here's what I wonder: do you know if there's a way to configure how my computer addresses the connected 5.1 system? Currently, the only software that I can even get to use 5.1 properly is VLC media player (like I said, for some DVDs). When I select 5.1 output in my Sound Preferences (I have it set to Stereo Duplex), the Subwoofer is off by default.

Any ideas?

Thanks!
