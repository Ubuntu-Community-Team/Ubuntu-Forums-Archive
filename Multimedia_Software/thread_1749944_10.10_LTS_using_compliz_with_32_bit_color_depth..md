### Post by realzippy on 2011-05-05
24bit color *is* 32bit...
Many modern desktop systems (OS X, Gnome, KDE, Windows XP/Vista/7, etc.) offer a 32-bit color option (given a suitably modern video card), but in that context, 32-bit color refers to 24-bit TrueColor with 8 bits for an alpha channel.

from:
[http://en.wikipedia.org/wiki/Color_depth](http://en.wikipedia.org/wiki/Color_depth)
