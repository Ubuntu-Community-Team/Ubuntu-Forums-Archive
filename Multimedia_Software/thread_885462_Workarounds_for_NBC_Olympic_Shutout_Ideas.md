### Post by AR_Kozz on 2008-08-10
As I'm sure millions of other linux users have unpleasantly suddenly discovered, the live streaming broadcast of the 2008 Olympics turned out to be in an extremely limited codec, Microsoft's new Silverlight. 

Because Silverlight has no working linux ports, the streams are thus unavailable to any pc's except those running Windows and Mac. And because NBC has an exclusivity contract they are allowed to use geoblocking to prevent us from watching, say, Canada's streaming which is via wmv.

So far I've tried 3 things:

1) Wine. It wasn't till I tried Wine with flashplugin.exe in place that I discovered the issue was about Silverlight; in linux the message simply says windows and mac only. Unfortunately, Silverlight will not install under Wine (root certificate issues apparently). According to the WineHQ App database, Silverlight is rated "garbage." So there goes Wine.

2) Moonlight. Microsoft has supposedly lent a hand on this porting of Silverlight to linux, but not with much effort apparently because it was expected by mid-year but is still nowhere near finished. Even to watch buggy video, it has to be compiled which is quite a chore in its case.

and

3) cctv5. China Central Television streams a sports channel over the net. 
here's a url of one of their casts:

mms://218.56.57.38/cctv5

I have an issue of convincing Firefox not to try to use Totem whenever I click a link to mms video, so it only works with embedded mms, if you have that same trouble try this link:

[http://www.thoughtsofanordinaryman.com/2008/08/watch-beijing-olympic-games-2008-live.html](http://www.thoughtsofanordinaryman.com/2008/08/watch-beijing-olympic-games-2008-live.html)

That other link on that page, of Macedonian sports, is crap, there's no Olympics there.

Also tried using Canada proxies to get their video, but got restricted region messages anyway

Anyone else know something I don't about the Silverlight format, or any other good streams I haven't found yet... I've just spent a good few hours looking
