### Post by ad_267 on 2008-07-24
Ok well I've tried videos on both the southpark and cnn site and they both work for me. They're flash videos but might not work because you have a different flash player installed.

I installed the flash player by installing the ubuntu-restricted-extras package. This installs the flashplugin-nonfree package as well as other useful packages. You can read about that here: [https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)

To install flashplugin-nonfree go to System - Administration - Synaptic Package Manager. Click Settings - Repositories. Make sure there is a tick after universe, restricted and multiverse. Click close then Reload. Search for flashplugin-nonfree or ubuntu-restricted-extras. Mark the package for installation and click apply.
