### Post by SeePU on 2010-09-28
Are you using the default open source driver?  Did you install the restricted binary driver?  

If you're using the FOSS radeon driver (i.e. the open source driver is already installed and you just need to 'choose' it or leave as default for use), then you won't have HDMI audio as it's not supported yet.

If you installed the binary driver, that would be a different story.  I think it works but you might have to google if there's issues.... maybe try "Ubuntu" + "HD 5450" + "popping sound" or other terms until some hits look useful?
