### Post by 4Orbs on 2009-11-19
More than a year ago I quit ripping subtitles onto my avi files because they would cause a green line and noise. Now I just download the subtitle srt files from one of the free subtitle websites and keep the srt file in the same directory as the video file (with the same file name as the movie). I don't know if this is the problem with your ISO files, but it sounds as if it might be.
