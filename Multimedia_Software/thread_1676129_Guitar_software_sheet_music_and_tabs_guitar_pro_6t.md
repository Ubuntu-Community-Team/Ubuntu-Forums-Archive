### Post by Finn bjerke on 2011-01-26
Guitar tab programmes can play guitar music while showing sheet  and tabs. Very nice Indeed.  You can find Bach bourre or Beatles blackbird on the net and have it played for while showing the fingering and the sheet music. :guitar:

Now can Ubuntu match Windows programmes ? YES THEY CAN

1. TUXGUITAR from the ubuntu universe can be downloaded thru synaptic, the prorgram can import most guiar files thru the "batch convert" function. It allows you to convert files from:

Import: 
Powertab, Tabledit, Midi, guitar pro. 
Export
tuxguitar file format and guitar pro file format

That means you can use tuxguitar for import from windows programmes. The tuxguitar programme is a little primitive but OK for learning

GUITAR PRO 6
is a commercial program it wil cost you around 50$ its a professional programme with many function and very nice soundbanks. The import function does not work under Ubuntu 10.10 - however tuxguitar can batch convert to guitar pro file format.  

Alternatively you can install powertab or tabledit under wine. Its too complicated for me. I use guitar pro 6 which is very nuice. Tabledit is much more user freuindly imho. 

Coclusion you can use any tab from the windows world by combining tuxguitar imports with Guitar pro 6 or just use tuxguitar.
