### Post by QIII on 2013-11-23
Moonlight is dead.  When it was alive, it did not contain the proprietary components needed to actually work with DRM-protected Silverlight content.

Silverlight itself is not long for the world, as Microsoft has pretty much shelved it.

You don't have to "go back" to Windows.  There's not an either/or choice you have to make.  You can dual boot and use both Linux and Windows.  Using both is allowed!  :)

I have Linux machines and Windows machines for just such reasons.
