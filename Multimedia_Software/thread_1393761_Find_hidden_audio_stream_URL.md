### Post by howefield on 2010-01-29
> **bo_n_tulsa said:**
> Is there a way to figure this out?

1. Go to [http://krmg.com/mobile/](http://krmg.com/mobile/)
2. Save the Listen Live! (MP3) link.
3. You want the file KRMG.pls.
4. Import this file into rhythmbox and you'll see the links to the stream appear in your radio section.
