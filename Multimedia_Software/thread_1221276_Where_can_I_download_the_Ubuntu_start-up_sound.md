### Post by igorzwx on 2009-07-23
read this:

[https://help.ubuntu.com/community/OpenSound](https://help.ubuntu.com/community/OpenSound)
**Ubuntu Intrepid/GNOME 2.24 (and later)**

 Starting with v2.24, GNOME uses libcanberra for system sounds. Ubuntu builds the libcanberra packages to use ALSA output by default. A rough outline of building libcanberra from source for use with OSS4 (or any gstreamer backend) can be found [here]("http://www.4front-tech.com/forum/viewtopic.php?t=3225&start=26"). This hasn't been tested very thoroughly at this time (especially on Ubuntu 8.10), so please leave feedback/suggestions in that thread.
