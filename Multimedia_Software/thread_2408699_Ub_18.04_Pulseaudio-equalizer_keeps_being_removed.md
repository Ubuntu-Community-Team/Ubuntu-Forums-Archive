### Post by Frogs Hair on 2018-12-21
The PAEQ was last supported by the [webupd8]("http://www.webupd8.org/2013/10/system-wide-pulseaudio-equalizer.html") team and there is no way they can fix any bugs anymore the last I read. I suspect the EQ uses a dependencies that interfere with the updates thus the application is removed to install the updates.

> Please note that PulseAudio Equalizer is no longer maintained so if you  find a bug or if it doesn't work for you and the fix isn't trivial,  there's nothing I can do. Update: for a modern, maintained PulseAudio  equalizer you can try PulseEffects - read about it

Pulseaudio Equalizer is no longer actively maintained, but there is [https://github.com/pulseaudio-equalizer-ladspa/equalizer](https://github.com/pulseaudio-equalizer-ladspa/equalizer)

Both of these use the same LADSPA plugin from package swh-plugins Multiband EQ (mbeq, 1197) [http://plugin.org.uk/ladspa-swh/docs/ladspa-swh.html](http://plugin.org.uk/ladspa-swh/docs/ladspa-swh.html)

The same Steve Harris plugin is also mentioned in Pulseaudio documentation for using the LADSPA sink module, with an example command of how to load the equalizer module [https://www.freedesktop.org/wiki/Software/PulseAudio/Documentation/User/Modules/#index47h3](https://www.freedesktop.org/wiki/Software/PulseAudio/Documentation/User/Modules/#index47h3)

The Pulseaudio-equalizer-ladpsa package provides a very basic gui for setting equalizer levels
Levels are saved in Pulseaudio configuration default.pa file like this: 

```
.nofail
load-module module-ladspa-sink sink_name=ladspa output.mbeq_1197.mbeq sink_master=alsa_output.pci-0000_00_1f.3.analog-stereo plugin=mbeq_1197 label=mbeq control=5.3,2.6,2.6,-8.5,-10.5,-11.2,-16.0,-14.7,-6.6,-5.7,-3.0,3.0,6.7,7.3,7.3
set-default-sink ladspa output.mbeq_1197.mbeq
set-sink-mute alsa_output.pci-0000_00_1f.3.analog-stereo 0
.endif
```
(pulseaudio sink "alsa_output.pci-0000_00_1f.3.analog-stereo" shown in this example will be similar name with different numbers on other systems)

If user previously had similar configuration from pulseaudio-equalizer, or can work out EQ level settings without help from gui, it is possible to use just the pulseaudio default.pa configuration and LADSPA plugin to run the equalizer
