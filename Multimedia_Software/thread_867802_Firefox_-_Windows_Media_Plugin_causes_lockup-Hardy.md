### Post by linux6994 on 2008-07-23
When attempting streaming audio I have found that the Firefox Windows Media Plugin causes the PC to lockup. Power reset is required to get things back.

I have found that all works well having the Windows Media Plugin disabled and having mozilla-plugin-vlc installed. Using VLC and its plugin works great. Under Firefox>preference>applications change Windows Media apps to vlc media plugin. I also saw this in Intrepid also.
