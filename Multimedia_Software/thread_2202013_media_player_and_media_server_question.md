### Post by ian-weisser on 2014-01-28
Not that I know of.
Most servers stream those very large files, and some media players (iTunes) are designed to work from local files, not to receive streams.
But quite a few media players in the Ubuntu repositories handle streams very well, and a few are command-line only.
They are in Software Center.
