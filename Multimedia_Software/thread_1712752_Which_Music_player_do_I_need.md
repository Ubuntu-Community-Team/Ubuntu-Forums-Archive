### Post by VastOne on 2011-03-23
Clementine is good..  Has many of the functions you are looking for, and adding to the library is the best I have seen - But it can get crazy memory leaks with use

Guayadeque is buggy, can develop massive amounts of memory leaks ... It is ok if you do not mind following ....

GMusicBrowser is the best in that it allows you to create your own music experience with layouts created by others or you can design your own.  These layouts can cover everything from the player to the tray to even a desktop addition like what Conky provides - It is even skinable

It also has a built in tag editor that is as good as nay I have seen and you do NOT need to refresh the library for tag changes to take effect.  Side note, for tag editing, you should really check out puddletag.. the best there is...

GMusicBrowser also has plugin support and an active developer who is available on the IRC and forums.. And it uses the least footprint of memory and CPU of all as it was designed just for large libraries, handling my 50,000 track library at 80Mib that NEVER changes ...  

The links to GMusicBrowser and Puddletag are in my sig line
