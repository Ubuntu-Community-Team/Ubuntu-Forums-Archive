### Post by markbuntu on 2009-05-29
The sound scheme in 8.04 is way different than on 7.10. There is a quick start guide to getting sound set up here
 
[http://ubuntuforums.org/showthread.php?t=997506](http://ubuntuforums.org/showthread.php?t=997506)

If you are still having problems you can go here for some very extensive troubleshooting help.

[http://ubuntuforums.org/showthread.php?t=843012](http://ubuntuforums.org/showthread.php?t=843012)

If you had made and changes in your etc/asoundrc or ~/.asoundrc file when using 7.10 you may want to move the file or rename it and start with the new defaults and get everythign working before trying any of that again.
