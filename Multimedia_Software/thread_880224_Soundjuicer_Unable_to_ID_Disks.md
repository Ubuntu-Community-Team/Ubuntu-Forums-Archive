### Post by luks on 2008-08-06
MusicBrainz doesn't have any Disc ID attached to that album. In this case, click on the "Submit Album" button, it should offer you to either add a new album or search for an existing one. So search for "Ivoryline" and select "There Came a Lion". After this, SoundJuicer should be able to look up the CD.
