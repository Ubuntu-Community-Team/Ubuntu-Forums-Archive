### Post by foobnix on 2011-02-23
**  &#1057;hangelog: **


[LIST]
[*] A simple tag editor with the possibility of batch processing tags in many files (Dmitry rind)
[*] Improved integration with last.fm (Favorite Tracks, Top Artists, Top Tracks, biography)
[*] Pop-up messages on the currently playing song (optional)  (Dmitry Kozhura)
[*] Automatic download online music
[*] Editing titles and delete files, folders directly from the player (Dmitry rind)
[*] New button to activate random playback and sequencing (Dmitry rind)
[*] Open a folder in the file manager (adjusted) (Dmitry Kozhura)
[*] Improved volume control
[*] Accelerated and altered search online music ( [fain182]("https://github.com/fain182") )
[*] Optimized Drug-and-drop  (Dmitry Kozhura)
[*] Made a cache for disk imaging from the internet: CD covers, album titles, lyrics  (Dmitry Kozhura)
[*]  Displays technical information about the currently playing file, as  well as the status of the player in the status bar  (Dmitry Kozhura)
[*]  The ability to copy a list of information about the tracks in a format  suitable for putting some on the tracker (D  Mitry Kozhura)
[*] Increasing the speed of your player (a quick start, exit and nekorotye other functions)
[*] Set up translations in different languages ( [Oleg "Eleidan" Kulik]("https://launchpad.net/%7Ehelh-saintman") )
[*] Implemented a lot of suggestions and fixes many small bugs.
[/LIST]
