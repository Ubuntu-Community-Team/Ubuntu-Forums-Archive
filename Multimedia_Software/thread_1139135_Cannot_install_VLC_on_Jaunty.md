### Post by coskierken on 2009-04-27
You can do it anytime.  Just create the disk you want (64 or 32), boot, and then manually do the partitions.  You will see your current partition, but just use it. It may say do not use, but arrow down and hit which file system you want and hit the format button.  Your first partition should be / from the pull down menu.  I would put a second for your home partition, but it is not necessary if you don't want to.  If you have any data, be sure to save it off as you are reformatting the disk.  

If you have the hardware(over 4G ram and 64bit proc), give 64 bit a shot.  It depends what you use it for.  If you rip quite a few movies down to mp4, it is worth it if you have the hardware to support it.  I have ripped a movie via Handbrake 64 in as little as 17 minutes and that was just using 1 of my cores on my Q9550 Intel.  DvdRip supports using all cores at one time, so I will be playing with that next.  

Any problems you have can generally be worked out.  Just search on the forums for answers and have a little patience.  Have fun.
