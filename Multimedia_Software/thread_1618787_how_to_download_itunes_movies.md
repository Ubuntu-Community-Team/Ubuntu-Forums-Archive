one way

wget -U QuickTime/7.6.2 [COLOR="Blue"]<insert url here>[/COLOR]

you may need to add an h before resolution

Ex. - picked one just now
```
wget -U QuickTime/7.6.2 http://trailers.apple.com/movies/independent/ongbak3/ongbak3-tsr1_[COLOR="Red"]h[/COLOR]720p.mov

```
