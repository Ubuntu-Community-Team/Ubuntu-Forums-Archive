I have an LG KU990 "viewty" cell phone, and, in case it is of help to anyone, I am posting here a link to a shell script which uses ffplay & ffmpeg to convert videos to the format & screen size of the phone: [http://www.clip.dia.fi.upm.es/~mcarro/pieces/](http://www.clip.dia.fi.upm.es/~mcarro/pieces/)

I am not a seasoned shell programmer; hence the chunk which takes care of deciding whether it's the video width or height which has to be selected to be scaled to the phone's screen width or height is not specially resilient.  If anyone comes up with a more elegant solution, I'd be happy to incorporate it in the script.
