### Post by eg0n on 2010-02-08
What type of audio file are you trying to edit?
According to its [faq]("http://live.gnome.org/Rhythmbox/FAQ") Rhythmbox has different [requirements]("http://live.gnome.org/Rhythmbox/TagWriting") for each audio type.

I think that the best way to handle your tags is by using Easytag. It's a little bit complicated but once you've learnt how to use it, you could do whatever you want within few seconds.

### Post by Aleksandar_B on 2010-02-24
I'm also having this problem. I can edit .mp3 format tags in Rhythmbox but not .wma tags. EasyTag doesn't seem to support .wma format, so is there another solution?

EDIT:
I have found a solution for editing .wma tags in Ubuntu. It's a program caled "Ex Falso", you can get it with Synaptic. It works in most cases, although I have encounterd some problems when after editing tag with Ex Falso, it's still not recognized in Rhythmbox nor when clicking on song and going properties and Audio tab.
