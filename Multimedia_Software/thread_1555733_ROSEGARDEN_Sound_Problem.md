### Post by cchhrriiss121212 on 2010-08-18
Rosegarden is just a sequencer so it doesn't make sounds on it's own, you need to connect it to a synth using jack. Some good synths are zynaddsubfx, phasex and qsynth (for sound fonts).
If you want a simpler interface try using LMMS, no jack required and it can do all you're asking for out of the box.
