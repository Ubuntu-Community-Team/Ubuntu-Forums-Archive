### Post by ubukool on 2010-06-11
Hi friends,

Earlier today I discovered that I could use Amarok to manage my ipod and, sure enough, when I connected my ipod to USB, Amarok saw it and listed the contents under 'local music'.

Later, I disconnected the ipod and when I connected it again, Amarok wouldn't list it under 'local music'.  I can open the ipod with Amarok, but all it does is add all the files to the playlist, it won't let me manage them.

I'm using a 4th generation ipod nano and Amarok version 2.3.1.  Can anyone suggest how to get Amarok to see the ipod as a media source again?

---

### Post by ubukool on 2010-06-11
I've found how to make Amarok see the ipod.  If you select Amarok ---> play media, you get a dialog box with a number of sources on the left.  If you select 'ipod', it puts tracks in the playlist but ipod comes up as a manageable resource as a local media source, which is what I wanted!

I don't know why it stopped recognising it automatically, but I don't mind as long as I have a method to make the ipod manageable through Amarok.  I hope this helps others who may have the same problem. ;)
