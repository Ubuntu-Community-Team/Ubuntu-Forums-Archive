### Post by Kirboosy on 2013-09-27
Ah that clears it up. I did the same thing to my test box and I do have the same issue as you now. You could try installing another webcam utility. 

GTK UVC Video Viewer should do the trick.

If you are on Ubuntu 13.04 you can install it with the following command

```
sudo apt-get install guvcview
```


Here is the launchpad page to the project. 

["guvcview" package: Ubuntu ]("https://launchpad.net/ubuntu/+source/guvcview")


Hope that helps!
~Caboose
PS You may wanna file a bug report with the Ubuntu devs to see if its something they might fix.
