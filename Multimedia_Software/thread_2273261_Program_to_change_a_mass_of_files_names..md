### Post by coldraven on 2015-04-12
Some multi-rename programs will show you the result of your renaming before actually doing the renaming. This is important because you can really mess up your files, big-time! Like you accidentally give them all the same name for example. 
I would recommend that you copy your Tragically Hip Discography '87 to '04. file to a different folder before you experiment with multi-renaming.
I have used Krusader which is a KDE app. If you install it in Ubuntu it will drag in a lot of dependencies and might be overkill for your needs. Multi-rename is just one of the functions of Krusader, you can easily manage to delete everything if you are not used to two-pane file managers.
[http://www.krusader.org/](http://www.krusader.org/)
