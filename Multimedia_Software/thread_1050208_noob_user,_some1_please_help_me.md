### Post by howefield on 2009-01-25
Install ubuntu-restricted-extras with Synaptic package manager.

Go to System > Administration > Synaptic package manager and in the search field type ubuntu-restricted-extras and tick the check box to install. Then apply.

  
This page is worth a read

[https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)
