### Post by alejandro.mc on 2009-12-05
Best to use Handbrake. If you are using 8.04 LTS you can only use the program from the terminal. To do so you have to install the program first. But there is no .deb for 8.04 LTS because the programer didn't have 8.04 LTS, he is already in Jaunty, so what to do?


1. Download and extract the file HandBrakeCLI from:

[http://sites.google.com/site/archivosamc/](http://sites.google.com/site/archivosamc/)

2. Follow the instructions here to install the program (skip extraction since you are downloading a different file, the rest follow exactly):

[http://en.flossmanuals.net/Handbrake/InstallingUbuntu](http://en.flossmanuals.net/Handbrake/InstallingUbuntu)

3. So now you have the Handbrake program working. How it should be used to extract a MKV from a DVD? Here they have a bunch of instructions and guidelines:

[http://trac.handbrake.fr/wiki/CLIGuide](http://trac.handbrake.fr/wiki/CLIGuide)

YOU MUST READ TO DO THINGS PROPERLY!!!

More questions? Try the forums from Handbrake!

[http://forum.handbrake.fr/](http://forum.handbrake.fr/)



Cheers!

Alejandro.
