### Post by sdowney717 on 2009-10-01
using a geforce 8400gs and running karmic beta alpha and using a CRT monitor.
this puts the tv screen off to the right. You can open up your app and then drag it to the right and it will show up on the TV.
you will know it is working if you get part of your background image showing up on the TV.

you can view changes immediately when using nvidia - settings by clicking the apply button. 
I had trouble when using their save x configuration. I had an empty config and it needed a device section to even show the preview xorg page.
