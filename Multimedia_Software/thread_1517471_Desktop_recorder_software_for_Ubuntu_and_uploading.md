### Post by swamytk on 2010-06-25
Desktop recorder software is highly useful in teaching, tutorial and knowledge sharing side. That too in Open Source communities, the video tutorials are famous through YouTube. You can also make your own desktop recorded video tutorial using gtk-recordMyDesktop software in Ubuntu. This tool has matured a lot to make it easy to use and quality output.

More about this application: [http://karuppuswamy.com/wordpress/2010/06/25/application-desktop-recorder-software-for-ubuntu-and-uploading-to-youtube/](http://karuppuswamy.com/wordpress/2010/06/25/application-desktop-recorder-software-for-ubuntu-and-uploading-to-youtube/)

### Post by VH-BIL on 2010-06-25
I also use Record My Desktop as a software developer to show clients new functionality of applications or Web Sites for approval or to show progress. It is better to send a video then sending an alpha version to a client which may have bugs. I find it very useful.
