### Post by Arminius on 2009-10-02
think I've solved it, think my damn firewall was treating my ps3 like an invader, I foolishly assumed that cause they were on the same router it would just let it through.

so if anyone else has the same problems find the IP address under network settings on your ps3, look for pings from that on your firewall, allow it and it should work
