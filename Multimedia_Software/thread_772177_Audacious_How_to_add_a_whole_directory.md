### Post by fabiops on 2009-06-01
Hello!
You've probably got the option
"Use XMMS-style file selector instead of the default selector" selected.

Go to Preferences > Appearance and Unmark "Use XMMS-style file selector instead of the default selector".

Now you should be able to select a directory and add its whole content 
by simply highlighting it and clicking att the "add" button.

That's what solved the problem for me!
Good luck, Cheers
