I did delete the specified files in my home directory anyways, but we still don't see the card showing up as a recognized device.  
The device is seen, however:
```
$dmesg | grep -i via | grep 82xx
[    9.239968] VIA 82xx Audio 0000:00:11.5: PCI INT C -> Link[ALKC] -> GSI 22 (level, low) -> IRQ 22
[    9.240959] VIA 82xx Audio 0000:00:11.5: setting latency timer to 64

```
However it seems that the driver either is not loaded or is not loaded successfully.  For example
```
$dmesg | grep -i alsa
