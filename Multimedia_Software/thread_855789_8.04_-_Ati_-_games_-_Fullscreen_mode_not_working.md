### Post by markbuntu on 2008-07-10
If you are using compiz, make sure you have unredirectfullscreenwindows checked in the General section of System/Preferences/Advanced Desktop Effects Settings.

Also, changing the xorg.conf file will not make any changes in the proprietary fglrx driver configuration.

### Post by wonk-o-matic on 2008-07-10
Got this from [http://www.runescape.com/p=kKmok3kJqOeN6D3mDdihco3oPeYN2KFy6W5--vZUbNA/kbase/viewarticle.ws?article_id=751](http://www.runescape.com/p=kKmok3kJqOeN6D3mDdihco3oPeYN2KFy6W5--vZUbNA/kbase/viewarticle.ws?article_id=751)
and it worked for me:
To get my full-screen working, I did: 
   cd /usr/bin/X11
   sudo cp Xorg Xorg-backup
   sudo sed -i s/XINERAMA/ZINERAMA/g Xorg
Then reboot.
