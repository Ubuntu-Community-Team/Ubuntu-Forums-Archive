### Post by ssam on 2014-03-23
Musicbrainz is usually pretty complete and well maintained compared to most other databases. It would be great if you could add any missing albums you have to it. If banshee does not let you upload the metadata you can use [http://musicbrainz.org/doc/MusicBrainz_Picard](http://musicbrainz.org/doc/MusicBrainz_Picard) , then banshee should be able to pick it up (I have had a few CDs that I have had to do this for).
