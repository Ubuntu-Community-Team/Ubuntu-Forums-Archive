System -> Administration -> Synaptic Package Manager

Then Settings -> Repositories

Then just click the selector box by the repositories you want active.

**I've included screenshots for your reference.**

Once you add them, click the Close button, and then click the Reload button to tell your system what is available from those repositories.

Once you have done that, you can install VLC or whatever else you need from them.
