### Post by FuturePilot on 2010-01-10
Is there a way to configure the bitrate when Banshee transcodes a file when transferring it to an mp3 player?

I was transferring some songs to my mp3 player, some of which were Flac, and since my mp3 player doesn't support Flac Banshee went ahead and transcoded them for me....to 128kbps(!) Mp3. EEEK! ](*,) If I must use Mp3 I would prefer to not have such terrible sounding low bitrates.

### Post by FuturePilot on 2010-01-11
Ok, I figured it out. It's not intuitive at all. For anyone else that wants to know you have to go under Edit &#8594; Preferences &#8594; Source Specific &#8594; Audio CDs, select MP3 LAME Encoder from the drop down and adjust the bitrate. Then you can change it back to whatever format you had it on before (if you had it on something different to begin with). Really, that setting makes no sense in that location for mp3 player transfers since what I'm doing has nothing to do with CDs.

### Post by FuturePilot on 2010-01-26
Apparently if you set the import setting to anything other than MP3, (I prefer Flac) it defaults to 128kbps MP3 when transcoding. So apparently you have to change the import format to MP3 before transferring songs so that it will use the specified bitrate. How annoying. ](*,) There must be an easier way.
