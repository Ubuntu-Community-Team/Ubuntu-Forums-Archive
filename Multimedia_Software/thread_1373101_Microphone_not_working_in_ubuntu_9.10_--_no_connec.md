Hello! I can't seem to get my microphone working; I've looked at several similar threads and tried the solutions given there, but they haven't worked. They tend to involve something like this:

right click the speaker icon --> click "Sound preferences" --> go to the input tab --> click the **connector option**, etc. 

This is where I hit a block; I don't *have* a connector option. Under "Choose a device for sound input," the only option is "Internal Audio Analog Stereo." (By the way, I have no idea what that means.) My microphone is plugged in to the same jack I used when I had Windows Vista, but I've also tried the other jack and it doesn't work either. Does anyone know what I'm doing wrong?

If it makes any difference, I'm trying to get it working for Skype, so if anyone has further suggestions related to that, they would be much appreciated. Thank you!

### Post by limeforisannah on 2010-01-05
Never mind, it's working now! I still don't have a connector option, but I fiddled with it a bit and now it's working. I think under the "Hardware" tab, it was set to "Analog Stereo Output" instead of "Analog Stereo Duplex." So I set it to "Analog Stereo Duplex" and it worked. I now feel silly. ^_^;;; Anyway, if anyone else made the same mistake, there it is.
