### Post by Menschenfresser on 2011-07-30
Hello Forum!

This is not a support post. I simply wanted to share so google can pick it up, since I don't have a blog.

I wanted to try digikam for a long time, but having all my pictures nicely taged and rated in f-spot made it difficult to switch. Looking around, I could not find anything that migrated those infos from one database to the other, so I wrote a script that does.

It is in bitbucket with (I hope) easy to follow instructions:

[https://bitbucket.org/rolandgeider/f-spot-to-digikam](https://bitbucket.org/rolandgeider/f-spot-to-digikam)

The script copies the tags, the ratings and the comments from f-spot to digikam. It was tested with the current stable versions of both programs (F-Spot 0.8.2 and Digikam 1.9.0) but could work with slight older ones.
