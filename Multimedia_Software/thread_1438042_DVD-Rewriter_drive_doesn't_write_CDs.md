A PC recently donated to me has a DVD writer drive that doesn't appear to write CDs. DVD-R and DVD+Rs are fine.
It has the model number "HL-DT-ST DVD-RAM GSA-H30N". Googling suggests this is made by LG and is capable of writing CD-R discs (the ones I'm using are just standard 700MB up to 52x, the standard types really).

Are there any commands I can key in to see if the drive is capable of writing CDs, or if there is any sensor that can tell if there is a problem with the drive itself?

### Post by wkulecz on 2010-03-24
Odds are the drive is defective.  They really are multifunction devices, where one function can fail while the rest work fine. 

I've a good number of RW drives that are now only usable as readers with either CD or DVD writing non-functional, or worse, work just enough to ruin the disk you try to burn.
