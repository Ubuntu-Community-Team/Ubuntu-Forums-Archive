### Post by sdowney717 on 2009-05-17
first off you bought a card which ATI will no longer support. x1300 support is dropped forever by ATI. Additionally, the Ubuntu OS has moved onto versions of xorg in Jaunty that the last ATI driver will not work with at all. The only video driver that will work with that card is the built in 2d free ati ubuntu free driver.

To make that card work either run 
Intrepid with the FGLRX ATI driver or
Jaunty with the ATI Ubuntu Free Driver.

To make it work with free driver in any ubuntu version,
Boot to a command prompt
Run 
 sudo dpkg-reconfigure -phigh xserver-xorg

to make it work with FGLRX driver
install the ATI driver using build package

then run
  aticonfig --initial 
You should get a return that says driver file initialized.

Here is my xorg and I am using an x1300 card

