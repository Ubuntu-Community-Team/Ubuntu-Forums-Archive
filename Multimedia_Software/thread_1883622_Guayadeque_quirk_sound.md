### Post by anonbeat on 2011-11-20
> **sblackwell said:**
> ubuntu 11.10 when i load the play list in Guayadeque when each song starts about 5 to 10 seconds in the song you hear this annoying quirk sound once. it is not a bad file because you can play it from another player and this doesn't  happen.

You have enabled to save lyrics embedded in the track. When guayadeque finds the lyrics it saves into the track and as the file changed while playing it makes this litle sound but this will happend only the first time you play the file and the lyric is found. The next times the lyric will be found embedded in the file.
You can disable this behaivour from Preferences -> Lyrics
