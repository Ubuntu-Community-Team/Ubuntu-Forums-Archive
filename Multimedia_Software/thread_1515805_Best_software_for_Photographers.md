### Post by SoFl W on 2010-06-29
I hope digikam is working out for you.  I came across this tonight if you are interested.  I use gnome so I am not sure if KDE is the same.  The default image viewer (open with image viewer) is called actually "eog" (eye of gnome) not 'image viewer', in the package manager there is an option for eog plugins which includes exif data.  Is aperture the same as the f-stop value you are looking for?  If so the plugin has it. 

When you open Image viewer (eog) goto Edit-> Preferences-> Plugins(tab)  to turn on the Exif option.  There is a configure button to make adjustments for some of the plugins.  Turn on any of the other plugins want.
Next Goto View and check Side Panel for basic exif information to the left of the image provided you turned it on in the exif plugin configuration.
Next goto Edit-> Toolbar and move the properties icon to the toolbar.   The properties button will bring up *a lot* of information about the picture.
