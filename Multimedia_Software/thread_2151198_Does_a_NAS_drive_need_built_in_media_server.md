---

### Post by zylx on 2013-06-04
It depends what protocols (smb, dlna etc) are supported on your TV, but generally you need DLNA (multimedia server) to stream videos, music and pics to PS3 or TV. However, almost any modern NAS has the ability to install additional software, so you can replace the default media server.
