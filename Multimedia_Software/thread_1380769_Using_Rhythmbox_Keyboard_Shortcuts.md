Windows logo Key + F1 = Launch Media Player (Open "Rhythmbox") (i mostly turn of monitor to save electricity :popcorn: )

Windows logo Key + F3 = Play/Pause

### Post by HedgeMan on 2012-05-13
Perhaps rhythmbox-client can help you. I just configured fluxbox for the following Keyboard Shortcurs:

~/.fluxbox/keys

```
Mod4 space :ExecCommand rhythmbox-client --play-pause
Mod4 Right :ExecCommand rhythmbox-client --next
Mod4 Left  :ExecCommand rhythmbox-client --previous
Mod4 Up    :ExecCommand rhythmbox-client --volume-up
Mod4 Down  :ExecCommand rhythmbox-client --volume-down
```HTH

Regards,

HedgeMan
