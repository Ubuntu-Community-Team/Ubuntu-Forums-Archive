### Post by cyclone5uk on 2009-03-10
I've just started using Acidrip to copy some cartoon episodes from DVD into Xvid so I can watch them on my netbook.

Everything works fine except for one problem. Even though I set the desired file size at 175mb, it's ending up with a file that's 290mb.

What gives? I have the audio set for mp3 and the video set to Xvid.

---

### Post by cyclone5uk on 2009-03-10
Okay solved this one myself. When you change the desired file size from the pre-set acidrip doenst always register it. Seems you need to type in the new size TWICE to force it to recognise the new size.

If anyone's interested - I've been getting some really nice clean results by setting it to 175mb with Xvid/mp3 doing a 2 pass encode. This is for 25 minute cartoon episodes.
