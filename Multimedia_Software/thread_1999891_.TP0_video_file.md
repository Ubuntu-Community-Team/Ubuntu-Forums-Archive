Have you tried running it through avconv or ffmpeg? If not, try (from a terminal I'm afraid)

avconv -i input.tp0 -acodec copy -vcodec copy output.mkv

This will copy the audio and video streams (ie, it's lossless), and remux them into an MKV file. However, .TP0 doesn't seem to be supported by avconv, so this probably won't work...
