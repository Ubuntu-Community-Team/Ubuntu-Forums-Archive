### Post by cor2y on 2008-05-18
Its one of the gstreamer plugin it has problem rendering mpeg4 videos (h264,X264.Xvid, lavc mpge4, dixv etc)
You have two choices keenish27 either install totem-xine (replacing totem-gstreamer first) or follow this fix located here
[http://ubuntuforums.org/showthread.php?t=799065](http://ubuntuforums.org/showthread.php?t=799065)
To replace totem-gstreamer with totem-xine
```
sudo apt-get install totem-xine;sudo update-alternatives --config totem
```
Select totem-xine as the default media player
