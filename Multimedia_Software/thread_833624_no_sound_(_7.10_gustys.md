### Post by Zerophase on 2008-06-20
Problem solved! the answer is installing linux backports modules. first you have to enable the gusty backports in the software sorces menu. then you need to type in the terminal   sudo apt-get linux-backports-modules-generic or i386. then you reboot and it should be good. hell yeah!! now i can watch a movie :)
