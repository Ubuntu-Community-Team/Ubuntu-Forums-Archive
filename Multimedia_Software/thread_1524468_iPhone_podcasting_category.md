### Post by Flash858 on 2010-07-06
If you set up your podcast feeds (you need to use mp3, as ogg is not supported by the iPhone) in Rhythmbox, then drag and drop to the iPhone, it works perfectly. The "album" art even shows, which is a LOT more than I can say for iTunes... ;)

---

### Post by Flash858 on 2010-07-06
OH, and if you need to use the google downloader, a workaround would be to get the same feed setup in rhythmbox, get a single episode downloaded (overnight maybe for video), and then copy or move the individual episodes to the same folder you use in Rhythmbox. 

By default, it just uses the "music" folder in the Home directory, but that is easily changed. I use a folder I made called "Podcasts", that is also in the Home directory myself.
