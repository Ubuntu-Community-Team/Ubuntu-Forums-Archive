### Post by 000zero on 2009-02-24
Never mind i fixed the problem.  When the grub boot list came up i booted the ubuntu recovery console and then from there chose the "attempt to repair x server" option and it worked.

Hope this helps anyone else if they have a similar problem, also be forewarned if you have a dell inspiron 700m with a intel 855 gme vid card DONT try to use the extended desktop option.
