### Post by shantiq on 2014-09-23
Welcome to Ubuntu Jim !
what needs to be done is to link wav files to Totem player [now called Videos]


1- click on 3 line icon top right of page in FF
2 - pick Preferences/Applications
3 - find wav and pick Videos


[ATTACH=CONFIG]256626[/ATTACH] 

although it says default it is not always left that way in my case VLC had taken it over but did not work so wav with "Videos" fixes it.
