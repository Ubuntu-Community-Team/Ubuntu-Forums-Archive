Just another tip for the future, if you have trouble with a Linux app, try running it from the command line and then observe the output. Oftentimes when there is a problem, the output will be very instrumental in diagnosing it. If nothing else, I have used the troubling output to search the web when I haven't been able to solve the problem right out.
