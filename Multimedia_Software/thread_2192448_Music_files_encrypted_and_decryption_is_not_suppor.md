### Post by cryptotheslow on 2013-12-07
Have you been playing or accessing these files on a Windows machine? WMA is a Windows Media format and if you are getting "encrypted" messages then the files are protected by DRM (digital rights management), There are pieces of Windows software around that will remove the DRM provided you still have the relevant licences for the files.
