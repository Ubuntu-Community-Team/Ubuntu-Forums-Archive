### Post by mc4man on 2009-08-03
from one of the links 
> Audio CDs

Audio CDs do not use WAV as their sound [COLOR="Blue"]format[/COLOR], using instead Red Book audio. The commonality is that both [COLOR="Blue"]audio CDs and WAV files have the audio data encoded in PCM.[/COLOR] WAV is a data file format for a computer to use that can't be understood by CD players directly. To record WAV files to an Audio CD the [COLOR="Blue"]file headers must be stripped and the remaining PCM data[/COLOR] written directly to the disc as individual tracks with zero padding added to match the CD's sector size.

On the audio cd it's a cdda, on (ripped) or [COLOR="Blue"]viewed[/COLOR] from pc it's a .wav
Though it's been a while, IIRC when an audio cd is viewed (explored) on a windows install you'll see .cda files which are just small marker files for the tracks, which when ripped will be .wav's


[http://en.wikipedia.org/wiki/Rainbow_Books](http://en.wikipedia.org/wiki/Rainbow_Books)
