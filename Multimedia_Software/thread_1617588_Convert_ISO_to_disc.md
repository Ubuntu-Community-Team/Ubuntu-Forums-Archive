### Post by Sylos on 2010-11-10
This is a long way round but here is what I would do:

1. Install AVIdemux and use it to open the VOB files and then convert them to a smaller format.
2. Use something like qdvdauthor, TODISCgui or DVDstyler to reconvert the files with some custom menus back into the relevant DVD compliant format and create the iso as a smaller file.
3. Burn the DVD.

This will take a while (particularly if, like me, you have an old 2ghz athlon with 1g RAM) but it should get the job done.

Hope this helps.
