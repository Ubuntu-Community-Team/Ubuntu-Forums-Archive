### Post by Lars Noodén on 2013-06-21
I don't know about JPEG but you can [make nice HTML based slides using S5](http://meyerweb.com/eric/tools/s5/)

If you do include images make sure you are using the right tool for the job.  Lately I've been seeing an epidemic of badly compressed JPEGs.  If you are working with a logo, graph, chart, or black and white photo, then GIF or maybe PNG is the way to go.  Only use JPEG if it's got millions of colors and few lines or other areas of sharp contrast. Bitmapped text is especially likely to turn out fugly in JPEG.
