### Post by slick666 on 2011-05-25
This is comming from a guy that **used** to work with a multi-monitor setup. But I haven't done this since 10.04. I ended up using nVidia card with just two monitors. The nVidia card treated the two monitors as ONE big screen. Gnome then chopped it up into 2 seperate desktops. This was great since when I maximised a program is maxed to a single screen and didn't split across the two. I could full screen a VM and it would just sit on the one screen. I would imagine if you want to use a TV tuner to watch some shows what ever application would behave the same.

I hope this helps and FYI if you're looking for a TV tuner use the hauppague 950Q.
