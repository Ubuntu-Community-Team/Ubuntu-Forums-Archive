### Post by binary10 on 2010-04-30
Ok have worked out that the visual desktop effects issue is causing this, which has been mentioned elsewhere. Disabling visual effect makes the BBC iplayer 'play' button work.

This has been kicking around for soooo long I would have thought it would have been a non-issue in 10.04.
