### Post by csisgro on 2008-06-10
I got an e250 also. Worked fine is MSC mode then one day every time I selected MSC it would reboot. So after trying many things went to forums on their web site and downloaded the latest firmware for the e250 hardware version I have (the site tells you how to identify which firmware you need). I installed it from safety boot up mode (Hold, rec and power all at the same time - forces a usb mode which writes to a reserved area where you copy the files you download then reboot the sansa and it upgrades itself) or whatever they called it an voila everything working again. Might help going through this procedure even if you already have the latest version.
