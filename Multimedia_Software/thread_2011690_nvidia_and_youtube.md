### Post by ajgreeny on 2012-06-27
If you do that you will not get flash videos from some other sites that are either

[LIST=1]
[*]not yet using html5, or
[*]do not work with the flash-video-replacer add-on.
[/LIST]
I assume you must be either signed up for html5 or are using the replacer add-on.

To get rid of the blue? colour you can try right clicking in a flash window and in the settings dialog disabling graphic acceleration.  It may overcome the problem.  Aklternatively you can use a previous version of flash (11.1.102.63) which you can get from lovinglinux at [http://ubuntuforums.org/showthread.php?t=1953796](http://ubuntuforums.org/showthread.php?t=1953796).  Scroll down the post a bit to find the links for32 & 64 bit versions, but be aware of the security considerations of the earlier flash version.
