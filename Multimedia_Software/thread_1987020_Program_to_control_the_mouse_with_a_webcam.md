### Post by peyre on 2012-05-25
My hands have been bothering me from too much mousework, and I'd like to find ergonomic solutions that will let me take some of the load off them and let 'em heal.

Ideally, I'd like to find something that will respond to hand gestures: move the hand=move the cursor; tap down=left-click; spread the fingers=right-click; or whatever.  Most of the software out there uses the head (or face) to control the mouse, which is clumsy, but it does marginally work.

I've had some passable results with eViacam in Windows at work, though the dwell-click bites big time when I'm doing anything other than casual surfing (it'll click on a link, or move the cursor out of where I'm trying to type, because I haven't moved for 5 seconds).  You'd think they'd offer the option of substituting a hard left-blink for left-click, hard right-blink for right-click, etc., but none of the (free, at least) software in this category seems to offer that.  Still, I'm getting some results in Windows.

But I'm getting nothing in Ubuntu.  Mousetrap sounds great but doesn't work.  At all.  eViacam offers a .deb installer, but when I open it in eDebi, it whines about dependencies, demanding old versions of libcv2.1, etc. which are no longer available in the Ubuntu repository.  Installing from source also snarls in dependencies, and I've had to add eViacam to the list of programs I've failed to install from source.  (It's a near-perfect record so far!  Raceintospace is the only exception to date.)

So, does anyone know of something I can use to control the mouse cursor with my webcam?
