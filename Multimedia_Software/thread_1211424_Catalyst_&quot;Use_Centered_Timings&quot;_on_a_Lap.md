### Post by aphirst on 2009-07-12
In the Windows ATI driver, there is an option under "Panel Settings" called "Use Centered Timings".
In short, if the physical display on the laptop's screen is 1280x800; with this setting enabled, if the screen resolution is changed to 1024x768, the new "desktop" is shown centred on the laptop's display, with no scaling and a black border.
This setting is very useful for games which do not have wide-screen modes, since having the game stretched across the screen looks dreadful.

I haven't access to my Linux Hard Drive for my current laptop at the moment, but my friend with a HD2600 can't seem to find an equivelant setting in his Catalyst Control Centre (In Arch Linux).

My question is, is this setting present for anyone with a HD4000-series GPU, as is in the laptop I wish to get this summer. Clearly, on this laptop, I wish to use this "centered timings" option.

Obviously, the setting might not be called that, but I think i've adequately described *what it does* in order for it to be found.
