### Post by toddr on 2009-08-28
As best I can tell, TNT.com uses flash for their streaming video content. Do you have the Flash 10 plugin installed? If you don't, you can install it easily. In Applications go to Add/Remove and search for "flash" and install the Macromedia Flash plugin. While your at it, you could also install the gstreamer ffmpeg video plugin. You might also try installing the "Ubuntu restricted extras" package. This one contains Flash and Java and a couple of others. Hope this helps.
