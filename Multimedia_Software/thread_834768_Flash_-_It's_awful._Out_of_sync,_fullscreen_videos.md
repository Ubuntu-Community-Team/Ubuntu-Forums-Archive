### Post by Sirron on 2008-07-06
I agree, something has to be done about this. The BBC and others seem to think that providing content as streaming flash video constitutes linux support; yet it relies on buggy proprietary software. You could try Gnash, I haven't myself as it wouldn't work for me strangely, but it's not a full solution as it is not fully compatible with Flash 9.

The nearest thing I can find to a solution is to install Wine, then install the windows version of firefox, and the windows version of flash. Whenever I want to watch fullscreen flash content, I open up the windows firefox and magically it's fine in that. The performance isn't exactly the same as windows, but it's much better than the linux version.

Considering we're getting better performance in Wine than with the native port, the situation is pretty dire.
