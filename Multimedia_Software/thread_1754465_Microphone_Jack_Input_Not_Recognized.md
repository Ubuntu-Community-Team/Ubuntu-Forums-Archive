### Post by lidex on 2011-05-11
Did you reset the changes you made? If not, do so then use the alsa-info-script to provide detailed information.
Run this command in a terminal:
```
wget http://www.alsa-project.org/alsa-info.sh -O alsa-info.sh && bash alsa-info.sh 
```
Choose the upload option and provide a link for the output.
[Terminal="Applications->Accessories->Terminal"]

```
options snd-hda-intel model=ultra

Save and close gedit.

sudo rm /etc/modprobe.d/alsa-base.conf.save
