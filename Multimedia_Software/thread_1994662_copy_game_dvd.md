### Post by sprocket10 on 2012-06-03
The difficulty in making a copy of a game disk depends on how new it is.  Since it is a DVD I'm guessing < 6 yrs old.  Some publishers use anti-cracking 'tricks' such as purposefully creating a predetermined series of faulty sectors (I think that's the correct word) on the disk, causing many cracking programs to give an error and fail.  The best way around this is to use some commercial software that devotes staff towards towards continually updating their cracking methods to &quot;keep with the times&quot;.  The best company I can think of is SlySoft [http://www.slysoft.com/en/](http://www.slysoft.com/en/)  They've probably got something that will work for you, but there are no doubt others.  As with other cracking procedures, sometimes you may simply have to wait until someone figures a workaround.    I would avoid downloading cracked game files (could be loaded w/ malware :( ) and simply make an ISO image from your own legal copy and mount the image when you wanna play.  I would think you're OK legal-wise as long as you're using it for personal backup purposes.  For example, I have some PC games whose authors have since either stopped supporting said games or closed down completely and I don't want to be tied down to using a CD years later when few, if any, laptops (what I primarily use) have optical drives.  Therefore, I've ripped ISOs and mount the ISO files using this free software ([http://www.slysoft.com/en/virtual-clonedrive.html](http://www.slysoft.com/en/virtual-clonedrive.html)).  This is fine as long as you're not illegally providing your stuff to others.
