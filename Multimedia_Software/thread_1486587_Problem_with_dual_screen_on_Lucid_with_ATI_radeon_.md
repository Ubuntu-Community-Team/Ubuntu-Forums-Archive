### Post by nickdc on 2010-06-19
I've also had a problem with dual screen since installing Ubuntu. In my case both screens work, but on extended desktop the panels etc appear on the right hand screen instead of the left (if I reverse the connectors I get the same problem in Windows). Also, much more annoyingly, mouse and pointer activity, particularly scrollbars, are quite unresponsive. I've spent some time on this forum looking for solutions, but as a newbie to Linux most of the potentially helpful stuff I can't really understand. The main thing I've learned is that the problem is almost certainly related to my graphics card, a radeon 1950. ATI don't support Linux on that card and the open source drivers are far from perfect as yet. Apparently they are being developed fast, so maybe in a few months there'll be an upgrade that sorts the problem - that's what I'm hoping, anyway. Meanwhile I'm forced to continue using Windows on this machine.
