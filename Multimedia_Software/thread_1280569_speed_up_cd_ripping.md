### Post by dummy910 on 2009-10-04
I like and use **Audio CD Extractor** in Gnome. Super easy to use and rips in several formats(.wav, .ogg, mp3 etc).

**To install:**
Add-Remove, search **Audio CD Extractor**, after installed, look for an icon in your Applications>Sound&Video titled something like AudioCD-Extractor(I don't have it installed on the machine I'm posting on so...)
**[color=red]or[/color]** from a terminal:
**sudo apt-get install sound-juicer**

Homepage: [http://www.burtonini.com/blog/computers/sound-juicer](http://www.burtonini.com/blog/computers/sound-juicer)
