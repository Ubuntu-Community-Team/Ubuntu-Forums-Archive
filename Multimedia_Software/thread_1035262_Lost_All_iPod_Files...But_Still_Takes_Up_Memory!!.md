### Post by timcredible on 2009-01-09
sounds like the ipod database is corrupt.  rhythmbox and amarok will work with an ipod also, you could try them and see if there's an option to rebuild the database.  but if the ipod database is gone or corrupt, then maybe you could just copy all the mp3 files to your hard drive (places->computer->your ipod), delete them off the ipod, initialize the database (via rhythmbox or amarok) and then re-copy them onto the ipod might work

keep in mind that ipods rename all mp3 files to other names (it's a kind of sequence like file1.mp3, file2.mp3, etc.), so you won't recognize the songs by their filenames, but i would think that the mp3 tags are still in the file.
