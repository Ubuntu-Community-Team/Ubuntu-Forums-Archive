### Post by purplecar on 2008-10-09
Been using Ubuntu for my theatre sound design business for sometime and came across this thread while trying find Linux software for the triggering of sound cues in a play [http://ubuntuforums.org/showthread.php?t=693876](http://ubuntuforums.org/showthread.php?t=693876)

The implementation of sound in the theatre has quite a few specific requirements if to be executed artfully and is different then DJing.  I tried using TTtrigger and while it was pretty good it did not fit all of my needs and was not something I could hand off to someone else to run.  
    
I have now found a solution that works very well.  There is a program called SOUND CUE SYSTEM that runs very well under wine.  Currently I'm running it on Ubuntu Studio under (or through) Wine configured to use jack as the audio driver sending Cues out to different ports in my Firewre card.  Here is a link [http://www.soundcuesystem.com/](http://www.soundcuesystem.com/)
It's not free but it's very reasonably priced and it works as well as Q-Lab for my designs and it is significantly cheaper then "industry standard" SFX.  

Version 10 won't work as I think it's GUI is built with VISTA in mind but any version under 10 will rock.  

Took me a awhile to find a solution I could use professionally so I hope this thread helps other sound people.
