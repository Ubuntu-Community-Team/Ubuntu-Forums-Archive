### Post by dv3500ea on 2010-09-08
I use Music/Artist/Album/tracknum - name.extension
only because that's how Rhythmbox rips my CDs.
I organise my music through Rhythmbox and not by using the file system (because then I have to actually worry about this hierarchy.
I very rarely access my music files directly using the file browser.
