### Post by TE7 on 2010-07-15
The GTK2 version fails after you add files, wait a minute, and then use the scroll bar.

The qt4 version works. It's for KDE, so you have to install the libqt4 file first. The link follows:

[https://sourceforge.net/projects/easymp3gain/files/](https://sourceforge.net/projects/easymp3gain/files/)

Download the .deb files to unique folders, open a terminal, cd to  each folder and type in

sudo dpkg -i *.deb
