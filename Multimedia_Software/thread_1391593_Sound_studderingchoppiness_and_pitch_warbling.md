### Post by audebruin on 2010-01-31
I have the same issue. Rhythmbox playback is choppy and wobbly.

The problem is gone when I change the "Configuration profile" in Pulseaudio Volume Control to one of the other 14 options and set it back to Analog Surround 5.1 Output + Analog Stereo Input (i am using Asus A7N8X Deluxe onboard sound and a Creative 5.1 set).

I upgraded from Ubuntu 9.04 to Ubuntu 9.10 using Synaptic.
