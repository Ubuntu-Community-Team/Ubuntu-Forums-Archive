### Post by rgb1701 on 2011-03-28
Install Wine per

[http://www.winehq.org/download/deb](http://www.winehq.org/download/deb)

then install DVDFab at

[http://www.dvdfab.com/download.htm](http://www.dvdfab.com/download.htm)

You must first rip the disc to hard disk with DVDfab (actually a convenience if traveling) then you can play the movie with SMplayer or VLC.

[http://smplayer.sourceforge.net/downloads.php?tr_lang=en](http://smplayer.sourceforge.net/downloads.php?tr_lang=en)

[http://www.videolan.org/vlc/](http://www.videolan.org/vlc/)

Would you really see a difference on such a small screen if you just played the DVD?  DVD's can be played straight from disc with VLC.
