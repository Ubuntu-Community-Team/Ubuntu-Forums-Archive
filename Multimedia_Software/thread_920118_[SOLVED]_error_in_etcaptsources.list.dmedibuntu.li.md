### Post by aysiu on 2008-09-14
Just paste this command into the terminal: ```
sudo rm /etc/apt/sources.list.d/medibuntu.list
``` Make sure to paste it instead of retyping it.

I don't know what happened the first time, but if you have already manually added the Medibuntu repositories to your /etc/apt/sources.list, you don't need the /etc/apt/sources.list.d/medibuntu.list any more.
