### Post by Creative2 on 2008-07-24
you can use fuoco tools 

or if you prefer a simple bash script 

read documentation on my signature

for bash script save like mpg2avi.sh (if you need of NTFS FORMAT you have to change -ofps 25 in -ofps 30 )

```
for f in *.mpg; do lame ; mencoder "$f" -ofps 25 -ovc xvid -oac mp3lame -lameopts abr:br=128 -srate 48000  -vf scale -zoom -xy 720 -xvidencopts fixed_quant=4  -sub SUBTITLE -subfont-text-scale 4 -o  "${f%.mpg}.avi"; done
```
