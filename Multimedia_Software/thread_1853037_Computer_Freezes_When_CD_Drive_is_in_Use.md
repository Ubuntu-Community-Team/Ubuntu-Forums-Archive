### Post by WasMeHere on 2011-10-01
I have Ubuntu 10.04 (lucid lynx LTS) on my main computer, so I am quite used to it. I have never had any problems with optical drives, except one that broke electronically. I have also installed it on several other computers and helped friends. Of course, something may be wrong with the drivers, but it might be the hardware too.

I suggest that you do some testing.

1. test reading from the CD/DVD
2. test using a live usb-system if you can use the CD/DVD to read & write
3. borrow another CD/DVD drive and try in your computer to read & write

This way we can figure out if the error is in the hardware or software

Good luck
Olle
