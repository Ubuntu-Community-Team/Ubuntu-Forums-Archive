I first noticed this problem about 36 hours ago -- everything worked fine before that.  The problem exists whether I'm using wireless or ethernet.  However, Chrome still streams video with no problems.  I prefer Firefox and hope to find a fix for this.

I'm running Ubuntu 10.04 on an ASUS EeePC, model 1000HA.
