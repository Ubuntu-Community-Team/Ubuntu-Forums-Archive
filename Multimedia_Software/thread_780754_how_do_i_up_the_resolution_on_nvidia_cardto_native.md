### Post by peakshysteria on 2008-05-03
Have you tried to run sudo displayconfig-gtk from a terminal? Then Ctrl + Alt + Backspace to restart x (or reboot). Had the same trouble with my ATI card. But now I'm finally running the right resolution (1280 X 1024). Looks to be something wrong with the Screen and Graphics Preferences manager...but give it a spin...

But my LCD tv can't detect my Ubuntu yet. Not sure why!?
