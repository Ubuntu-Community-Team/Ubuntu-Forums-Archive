### Post by gandaran on 2009-03-15
open synaptic package manager, scroll down to mozilla-mplayer, mark for install and click the apply button.
you'll need to install the w32codecs package from [medibuntu]("https://help.ubuntu.com/community/Medibuntu") for mplayer, add the repository to synaptic to get the codecs package.
