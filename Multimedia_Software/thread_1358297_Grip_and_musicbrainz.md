### Post by WileyGaia on 2009-12-21
The value to put into the "DB Server" input box is "freedb.musicbrainz.org".
You will need to hit the "Initiate/ Abort dsic DB lookup" button for it to go fetch the tags.
The CGI path remains the same i.e. "~cddb/cddb.cgi"
