Try this ...

Delete the /etc/X11/xorg.conf you created

then run this command 

sudo Xorg :1 -configure

which will create  xorg.conf.new in your home folder. Copy this file as root to /etc/X11 renaming it as xorg.conf

Then edit the line     Driver "vesa" to Driver "sis" save and restart and see what happens.

### Post by realzippy on 2011-02-12
> **ookami.kb said:**
> On this driver it shows an error:
No valid modes found - check VertRefresh/HorizSync

..so make the "monitor" section in xorg.conf like this:


```
Section "Monitor"
 Identifier "Configured Monitor"
 HorizSync 28-72
 VertRefresh 43-60
EndSection

```

...if done already,increase Hsync/Vrefresh values.
