### Post by Major_Kong on 2008-12-19
My 2 cents:

If you used Winamp, audacious is the way to go.
If you want something more like Itunes, or something similar, use Amarok (if you're using kde) and there's not much to discuss. But if you're using gnome it gets a bit tricky... There's (at least in my opinion) no mature jukebox type player for gnome, even if there's a wide range of choice. 
I'll just focus on two of the most used: Rhythmbox is the default audio player, but lacks some (many?) of the nice features of Amarok, and the interface could be better*. Banshee is a better audio player, but it hogs more resources than rhythmbox. So there's really no 'right answer' for gnome, it will depend more on the user than with anything else.
If you wanted something among the lines of foobar2k... try using wine to run it. There's no real replacement.

Bottom line (Windows -> Ubuntu):
[B]
Winamp -> Audacious
Itunes (or close to it) -> Rhythmbox/Banshee/Songbird/etc. (it's trial and error here)**
Foobar2k -> No replacement avaliable[/B]



*for example, something that bugs me a little is the volume control widget...

**if you're using kubuntu, it's amarok and there's not much to discuss.
