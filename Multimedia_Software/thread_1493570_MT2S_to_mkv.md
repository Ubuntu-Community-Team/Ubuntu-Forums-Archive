### Post by Judo on 2010-05-26
There are multiple ways, but I'd say using Handbrake is the best.

Handbrake is one of the better x264 frontends, even x264's developers like it.  You can set the resolution and file size you want and the presets have pretty good quality.  (I'm not sure if this is default or not, but you should probably copy the audio, rather than convert it.)
