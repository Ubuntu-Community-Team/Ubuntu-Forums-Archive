### Post by SRTS on 2008-04-26
Hi, on ubuntu 7 I had "direct rendering: yes". I know that installing compiz usually turns that "yes" to "no", so I didn't install it.
However, on ubuntu 8 it seems compiz is installed by default, disabling your ability to use 3d graphics.

so, if your 3d-apps or games suddenly stopped working, make sure to uninstall compiz first!

edit: don't worry about metacity. unlike ubuntu 7, in HH it seems metacity survives the deinstallation of compiz, and won't leave you with a broken system that requires reinstallation!

### Post by RacerII on 2008-04-26
When you are running compiz , it does not mean 3d is disabled.
However some chipsets do have a problem with running compiz and other 3d apps at the same time.

Example , on my nvidia box i can run compiz and have direct rendering.
On my laptop with a crappy intel accelerated chipset i can run compiz and have direct rendering.

On those 2 systems 3d apps run fine with compiz.

However on my ati system , it doesnt play nice.
