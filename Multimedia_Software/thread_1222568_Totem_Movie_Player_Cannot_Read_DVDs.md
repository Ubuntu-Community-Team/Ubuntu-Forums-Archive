### Post by miniyak on 2009-07-25
you probably need to reinstall the restricted extras again. This can be done though the add/remove program (search "ubuntu"), some of the other installation procedures can be more successful however. 

This is expected behaviour unless you already have the extras installed and still having issues. Plenty of wikis and threads on the topic getting proprietary formats working already exist such as this one [https://help.ubuntu.com/community/RestrictedFormats]("https://help.ubuntu.com/community/RestrictedFormats")
