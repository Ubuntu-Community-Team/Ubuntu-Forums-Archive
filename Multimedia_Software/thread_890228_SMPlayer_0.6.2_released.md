### Post by rvm4000 on 2008-08-14
SMPlayer is a front-end for MPlayer, developed with the Qt-4 libraries (required at least Qt 4.2).

[http://smplayer.berlios.de/](http://smplayer.berlios.de/)

Most important changes since 0.6.1:

 * Added an audio equalizer (please read Audio_equalizer.txt).
 * Added a dialog to search and download subtitles from opensubtitles.org (See Finding_subtitles.txt).
 * The toolbars can be configured. The user can remove, add or change the order of the buttons. Right now it can only be done by editing manually the configuration file. A graphical editor will be added in the next version. See Configuring_the_toolbars.txt.
 * The single instance port can now be chosen automatically by the application.
 * Added the Slovenian translation.

A more detailed changelog can be found [here](http://smplayer.berlios.de/forums/viewtopic.php?id=573).

There are two packages for Ubuntu:

[smplayer_0.6.2_i386.deb](http://prdownload.berlios.de/smplayer/smplayer_0.6.2_i386.deb) (compiled on Ubuntu Feisty)

[smplayer_0.6.2_amd64.deb](http://prdownload.berlios.de/smplayer/smplayer_0.6.2_amd64.deb) (compiled on Ubuntu Hardy)

I'd also like to remember that it's recommended to use a recent version of mplayer.
