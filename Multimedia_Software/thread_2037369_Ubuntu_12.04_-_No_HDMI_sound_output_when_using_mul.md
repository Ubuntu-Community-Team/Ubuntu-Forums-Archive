### Post by ranger1021994 on 2012-08-04
Try this :
1) Go to Sound settings (volume icon)
2) Go to output tab
3) Select HDMI (If this doesnt work then try out all the other options)

### Post by Tiede on 2012-08-05
Log out with the cable connected, and the TV powered on.
Then, just log back in, and try to choose HDMI out in the sound settings again. It should have suddenly reappeared.
