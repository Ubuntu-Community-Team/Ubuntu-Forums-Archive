### Post by Malcy on 2009-03-28
Trying Spotify out under wine in my Jaunty 64 test install and it just keeps crashing, freezing solid. I'm not the only one with this problem and it isn't related to wine config setup or mp3 decoder issues. 

The version available at present is 0.3.13. I have Spotify running happily on my main Intrepid install. When I checked that install, I found that the client is version 0.3.11, so something has happened in between. I replaced the 0.3.13 version of the client in my Jaunty install with a copy of the 0.3.11 client from my Intrepid install and it is now fully working as well.

So make sure that you use V 0.3.11 to install Spotify under wine until they fix this problem.
