### Post by Terryaaa on 2008-07-16
My problem is solved ! In recovery mode you must select from one of four options: Resume normal boot - repair broken packages - drop to root shell prompt - try to fix "X" server

I ran: "repair broken packages" and then" try to fix "X" server" followed by "resume normal boot" One of the two automatically fixed the problem!

I am up and running.
