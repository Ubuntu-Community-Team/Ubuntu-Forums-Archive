Everything was working ok using the default drivers but once i started using the nVidia proprietary drivers that's when the problem started. I've tried the 331 listed in "additional drivers" in settings and have also update to the latest Linux driver from the nVidia website.

Anyone any suggestions?


[EDIT] Apologies it would appear problem was caused by my KVM. I changed to a direct cable to my monitor and it resolved the issue.
