### Post by thewolfman on 2009-06-20
G'day Mate,

if you wish to use any KDE ap in Ubuntu you must make sure that all the support files are added with it, Ubuntu is Gnome based and works with GTK so any ap for KDE needs whatever pulled in with it.

Try installing a few more aps from KDE like Amarok (music) or Kontact (PIM prog very good!) and try reinstalling Konquerer and Kong-KIM that way any missing bits should be added automatically.

Also when installing individual packages with Synaptic you should look and see if it is suggesting something else as a dependency which you can change in the settings menu of Synaptic. After marking a file right-hand mouse click and see what else it may want.

Have a good one.
