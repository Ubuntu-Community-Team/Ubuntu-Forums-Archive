I am assuming that your are using the default install with the pulseaudio sound server? You can sort it out manually by right clicking on speaker icon on the top bar and select preferences. Under the input & output tabs you can select which devices you want. There is a package called earcandy, which is supposed to automate this sort of device switching but I found to to be rather flakey.
