Some time ago, Oracle (Sun) Java JRE 6 update 21 has become available. Not in the Ubuntu partner repository, though; that repo still contains update 20. 

Probably because update 21 contains no security bug fixes, but repairs only functionality bugs: [http://www.oracle.com/technetwork/java/javase/6u21-156341.html](http://www.oracle.com/technetwork/java/javase/6u21-156341.html)

When you're plagued by functionality bugs in update 20, you can still install update 21 manually:
[http://sites.google.com/site/easylinuxtipsproject/java#TOC-INSTALL-MANUALLY](http://sites.google.com/site/easylinuxtipsproject/java#TOC-INSTALL-MANUALLY)
