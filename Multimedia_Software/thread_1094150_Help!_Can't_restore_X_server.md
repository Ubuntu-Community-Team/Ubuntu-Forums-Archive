if you change 

Section "Device"
        Identifier  "Configured Video Device"
        Driver "nvidia"


to

Section "Device"
        Identifier  "Configured Video Device"
        Driver "nv"

that should restore your config file to the generic nvidia driver.

you can edit this from using nano 

sudo nano -w /etc/X11/xconf.org
