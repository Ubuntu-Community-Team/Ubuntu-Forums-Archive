### Post by Erlander on 2008-07-28
I am using a DVico Fusion usb dvb tuner.  Mine is about 4 years old and is no longer supplied.  It works out of the box on current Linux distributions.

[http://www.linuxtv.org/wiki/index.php/Main_Page](http://www.linuxtv.org/wiki/index.php/Main_Page) is your best place to start.  Support for many TV tuners is built into the linux kernel with more being added all the time.

I'm not sure whether you are using DVB or NTSC there in Thailand.
