### Post by ajgreeny on 2012-02-13
Sound-juicer (in the repos) can do that.  You set the output file type in the Edit >Preferences menu item.

RipperX can also do it with similar ease, and I expect many others as well, though I have only used those two, so try both, or more, and see which you prefer.
