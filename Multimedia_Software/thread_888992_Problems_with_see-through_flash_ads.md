### Post by silkstone on 2008-08-13
The problem undoubtedly lies with Adobe Flash. There are bugs in the current version, so transparency doesn't work. (e.g. The clock in the top-right of the BBC home page should not have a black background.)

The latest Flash 10 beta 2 seems to fix this but can cause other problems, so perhaps the best idea is to put up with it and hope that the release version of Flash 10 will provide a cure.

This is something over which Ubuntu has no control - it's a poor Flash plugin which is down to Adobe.
