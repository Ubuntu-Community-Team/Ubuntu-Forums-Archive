Now I am typing from my Jaunty machine.  If when you install Jaunty everything does not just automatically work, you can also install and try the following application.  I used it on older Ubuntu versions before Ubuntu got better at detecting hardware and installing the correct drivers - it saved the day for me:

> EnvyNG is an application written in Python which will download the

(according to the model of your card) from ATI or Nvidia's website
and set it up for you handling dependencies (compilers, OpenGL,
etc.) which are required in order to build and use the driver.
