### Post by devcaka on 2010-10-25
Hi,

If you have installed Nvidia drivers, you should also have a Tool called Nvidia X Server Settings under System>Administrator.

Select your screens and click on Configure Button. Select Twinview which allows you to move windows from one screen to another.

Good Luck
