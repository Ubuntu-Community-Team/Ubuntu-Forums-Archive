When I tested this out the first time round it kept on crashing over and over again...I found that there were bugs in Xinerama! 

The first  bug was that it couldn't map negative values for the mouse position. Ie. I would move my mouse to the top left but as soon as I moved it onto a surface mapped as negative by serverlayout it would die.

Secondly, it would just segfault randomly :(

Therefore...I searched for a fix and fortunately found one here: [https://launchpad.net/~jared-bunting/+archive/xorg-xserver-650539](https://launchpad.net/~jared-bunting/+archive/xorg-xserver-650539) so I just installed that and hey presto! I started working :)
