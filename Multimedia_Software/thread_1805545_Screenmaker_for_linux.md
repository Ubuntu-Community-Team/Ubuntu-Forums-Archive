### Post by ankspo71 on 2011-07-16
Hi,
There is a tool for Linux called "gromit" that makes annotations on the screen. It doesn't work good for me though, because it only makes scribbles when I try to use it, but it might work good for you. You can find gromit in Kpackagekit, Muon, Synaptic etc.

To run gromit:
press Alt+F2 then type gromit

Then you can control it with these keys:
Pause:       toggle painting
SHIFT-Pause: clear screen
CTRL-Pause:  toggle visibility
ALT-Pause:   Quit Gromit.

Hope it helps.

Here's a link to the homepage of gromit: [http://www.home.unix-ag.org/simon/gromit/](http://www.home.unix-ag.org/simon/gromit/)
