### Post by wauvil on 2009-09-29
I have been using the latest version of ubuntu with the NVIDIA 180 driver for some months now and have always had to turn off desktop effects so that my video would not be choppy in mplayer, smplayer etc.  Just in case anyone has the same issue, try updating your driver the the latest one, 185. I just recently did this and noticed no choppy picture at all with desktop effects enabled.  Sometimes a little work in terminal can give big rewards in the end
