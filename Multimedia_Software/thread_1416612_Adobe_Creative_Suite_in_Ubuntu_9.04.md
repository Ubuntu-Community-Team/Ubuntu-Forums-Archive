### Post by snowpine on 2010-02-26
Hi Jen! I use Creative Suite sometimes for work. Here is how I installed it in Ubuntu:

1. Install VirtualBox in Ubuntu
2. Create a Windows virtual machine (using a legal copy of Windows of course)
3. Install Adobe CS in the virtual machine
4. Install VirtualBox Guest Additions and create a shared folder for a seamless experience

Good luck!
