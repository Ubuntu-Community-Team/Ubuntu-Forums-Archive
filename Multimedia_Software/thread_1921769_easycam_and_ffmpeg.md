### Post by FakeOutdoorsman on 2013-02-25
Try a recent real [ffmpeg static build](http://ffmpeg.org/download.html#LinuxBuilds) (32 and 64 bit versions available for kernel 3.2.x+, and 64 bit available for kernel 2.6.26+). The so-called "ffmpeg" version in the Ubuntu repository is not from FFmpeg despite the name. No need to install or anything. Just download the archive, extract, and run:
```
wget http://ffmpeg.gusari.org/static/32bit/ffmpeg.static.32bit.2013-02-01.tar.gz
tar xzvf ffmpeg.static.32bit.2013-02-01.tar.gz
./ffmpeg -f video4linux2 -video_size 640x480 -i /dev/video0 -vframes 1 -qscale:v 2 image.jpg
```
