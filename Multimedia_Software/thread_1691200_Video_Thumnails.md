### Post by aeiah on 2011-02-21
i think totem is the default thumbnailer. you can install other video thumbnailers and set a different one as default (eg, mplayer or ffmpeg) if you want to fiddle. if you're sure the videos are all the same then this might be more effort than it's worth.

renaming the file always worked for me (rename with a space at the end, then rename back). i think eventually i changed to mplayer thumbnailer but i use the default one again on this lucid install and its fine.
