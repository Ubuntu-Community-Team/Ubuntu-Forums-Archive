### Post by LuridCinema on 2010-02-03
With Openshot , Cinelerra, ffmpeg & Blender you will have an extremely powerful arsenal of video tools. Lookup installation howtos on each for your distribution. I feel that for video production Cinelerra is hard to beat. Openshot has some cool features and I found Blender looked hard to figure out, but it does have a following for it's NLE tools.

One thing I can say is make sure you have your project settings correct in Cinelerra before you import anything into the timeline, that will cause crashes if you are off.  I found Cinelerra most stable on my install when I render out as Quicktime for Linux ( .mov ) . I will then convert to any other format needed w/ ffmpeg
