Hello friends, I'm a newbie in this forum and this is my first thread.

I use an Acer Aspire 4315 notebook computer. It has an Intel GM965 graphics card. With Gusty, Hardy and Intrepid I was able to enable desktop effects (compiz fusion) without any problems. But, in Jaunty desktop effects no longer work.

Under System -> Administration I looked for a third party driver but it was no use. 

Can anybody please help me? How do I enable desktop effects?

---

### Post by nyk on 2009-05-12
That particular video card is blacklisted right now. To enable it, go to System > Administration > Software Sources and enable the Jaunty Proposed repository. Then run Update Manager and install the proposed compiz updates
