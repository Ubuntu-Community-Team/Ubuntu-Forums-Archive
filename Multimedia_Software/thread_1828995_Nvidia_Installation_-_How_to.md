### Post by BicyclerBoy on 2011-08-20
The driver readme has an excellent (best)  write-up..
[http://us.download.nvidia.com/XFree86/Linux-x86_64/280.13/README/index.html](http://us.download.nvidia.com/XFree86/Linux-x86_64/280.13/README/index.html)
It is good to know how to do this & complete it a couple of times but..

A better way to get your nVidia fix is from pre-compiled package archives ppa like
xorg-edgers launchpad ppa
x-swat team ppa
There are others related to mythtv etc..

The reasons it is better:
- package management
- no dependencies problems
- driver does not break with every kernel update.
- don't need to stop X server gdm & console login compile etc..
- it works
- package management.
