### Post by deadflowr on 2020-07-16
Mythbuntu as a flavor is done.
But the mythtv packages are very much still actively under development.

Now to install mythtv you need to first install any ole Ubuntu flavor and then install the mythtv packages.
Somewhere, though it's been awhile, they recommended a lighter desktop like Xubuntu as the base.
And then install the mythtv packages you need on that.

This page lists the basic packages:
[https://www.mythtv.org/wiki/Installing_MythTV_on_Ubuntu]("https://www.mythtv.org/wiki/Installing_MythTV_on_Ubuntu")

Note the one package that did get removed was the mythbuntu control center.
Though all the tools it did are still doable, if only a little more involved, imo.
