### Post by SeijiSensei on 2013-02-01
You don't tell us what the output format is, but if you would be satisfied with XVid+MP3 in the AVI container and hard subs, you can use mencoder as an alternative.  You'd have control over the size of the subtitles as well.  Read [this guide]("http://en.gentoo-wiki.com/wiki/Mencoder#XviD") for details.  If you're creating something that is worth archiving, use two-pass encoding.
