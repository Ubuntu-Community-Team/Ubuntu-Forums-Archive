### Post by CoreAffinity on 2009-09-10
I got it to work. I found this writeup ([http://ensode.net/ati_radeon_xpress_200m_linux.html](http://ensode.net/ati_radeon_xpress_200m_linux.html)) that explained that the ATI driver does not work with only sideport memory... and that is how I had my BIOS configured. I re-installed the ATI drivers, changed the BIOS settings to use 128M UMA instead of sideport memory (also 128M, by the way) and all is well.

I did not experiment with the amount of video memory to use, or a mixture of UMA and sideport.

Hopefully, a future ATI driver update will correct this issue and I'll be able to use sideport memory instead of main  memory.
