### Post by antek on 2009-12-27
Hey,

Bumping a somewhat old subject, just to bring some more info (and questions) about this card (this topic is high on google when entering nxp713x). I didn't fully run it though, but it seems it's more or less compatible with other cards with the same chipset. You can select a chipset in the 'card' argument during modprobe saa7134:

```
$ modprobe saa7134 card=67
```

With card 67 (Beholder BeholdTV 409 FM) I had best results: good video, but completely no sound. Other cards (e.g. 44 - Empire PCI TV-Radio LE) had the sound in a good quality, but no video :P. To get the list of cards, use:

```
$ dmesg | grep saa7134
```

If anyone knows the solution for this please share ;)
