### Post by pytheas22 on 2008-10-04
[This site]("http://www.ventanazul.com/webzine/articles/openvpn-ubuntu-and-hulu") might help you.  It will get OpenVPN installed.  You'll need an American VPN server to connect to, though.  If you know of anyone with a web server in the United States, they'd probably be able to hook you up.

You can also change your Firefox preferences to use a [public proxy server]("http://www.publicproxyservers.com/"); if you find one in the United States, it would probably work.
