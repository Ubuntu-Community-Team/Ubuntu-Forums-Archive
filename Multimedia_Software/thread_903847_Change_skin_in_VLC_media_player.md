### Post by mc4man on 2008-08-28
It's fairly easy in ubuntu, should be same for kubuntu 
First in konsole (or terminal for gnome

 ```
cd ~/.vlc; mkdir skins2
```

Then place your downloaded skin(s) in there (home dir. -> .vlc (the complete .vlt

open vlc and and from settings tab choose -> switch interface -> skins2.
That will open the default skin. (for skins2 interface

Right click on the upper bar area and choose -> select skin and select from the list the skin you want. (will become the skin used when choosing skin2 interface vs. default wxwidget from above

you can also set in vlc -> settings -> preferences -> main interface -> Skins

You can change from wxwidgets interface to skin interface as above or..

To set as overall default for vlc while in settings (click advanced options) go  interface, highlight Main interface and in box on right set as in screen (change  - interface module 

Beware that some skins are unpredicable, I'd try them out first by selecting from wxwidgets interface before setting as default
