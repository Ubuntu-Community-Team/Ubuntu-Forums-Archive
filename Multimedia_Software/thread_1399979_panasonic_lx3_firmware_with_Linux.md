### Post by mooscape on 2010-02-06
I have successfully installed the firmware upgrade (v2.1) for the Panasonic Lumix DMC LX3 from Ubuntu Linux 9.10 (there is no official download available).

Read the following and follow the Apple MAC instructions carefully. I used File Roller to unzip the file. Procede at own risk.

> [http://panasonic.jp/support/global/cs/dsc/download/LX3/](http://panasonic.jp/support/global/cs/dsc/download/LX3/)

No problems were experienced and I now have many new and usefull features added to my camera.
