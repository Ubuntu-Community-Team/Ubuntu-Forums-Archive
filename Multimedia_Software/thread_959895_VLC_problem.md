### Post by scragar on 2008-10-26
I have seen that problem only when watching certain types of video, in avi's or ogg this doesn't appear to be a problem, but it is common in MP4s or MKVs.

The only solution I have found is to set the video a few seconds earlier than where you want in those video formats.
