### Post by kostkon on 2008-06-05
You can use *Rhythmbox* for this purpose. If you want to control it from a web browser then you can install the [*Rhythmweb*]("http://web.vee.net/projects/rhythmweb/") plugin. 
Alternatively, if your mobile device supports J2ME then you can use the [*Remuco*]("http://remuco.sourceforge.net/") plugin to control it directly from it. 

*Remuco* works with many media players. It currently supports:

Full support:
[LIST]
[*]Rhythmbox
[*]XMMS2
[/LIST]

Basic Support:
[LIST]
[*]Amarok
[*]Banshee
[*]MPD
[*]Totem
[/LIST]

I am pretty sure one of these two options will cover your needs :)
