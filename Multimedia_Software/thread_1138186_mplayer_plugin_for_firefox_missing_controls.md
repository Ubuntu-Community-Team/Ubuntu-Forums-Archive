### Post by linuxuser9999 on 2009-04-26
After I upgraded to Ubuntu 9.04 the mplayer plugin for firefox 3.09 is missing the video controls when I play video which used to be there when I had Ubuntu 8.10 and below. I can see the video but I can't rewind, stop, forward, pause, etc. I also get the same problem when I play audio files.

I tried uninstalling and reinstalling the mplayer plugin but the video controls still don't show up. 

Is anyone having this problem? Can you tell me how I can fix it?

---

### Post by linuxuser9999 on 2009-04-28
I fixed this problem by uninstalling the totem-mozilla plugin for Firefox.  I guess Ubuntu reinstalled it (which I removed when I was using 8.10 and 8.04) when I did the upgrade to 9.04. Even though the application preferences for Firefox was set to use the mplayer plugin to play videos the totem plugin kept taking over but after uninstalling it the mplayer plugin is now being used and I can see the controls.
