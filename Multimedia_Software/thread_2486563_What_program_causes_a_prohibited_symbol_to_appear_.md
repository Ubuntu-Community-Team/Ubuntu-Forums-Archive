[COLOR=#232629][FONT=-apple-system]Because the icon it appears when a key is pressed, the system registers the press. This appeared after the latest updates. Previously, the keys were configured only inside Kodi, since it does not react to system settings there, they were removed through the "settings>key combinations" menu item. They are still disabled there, but the volume keys now adjust the system volume, not the Kodi volume (the corresponding icon appears instead of the error icon). Re-assignment fails, attempting to assign causes this icon to be called, but the assignment fails.
