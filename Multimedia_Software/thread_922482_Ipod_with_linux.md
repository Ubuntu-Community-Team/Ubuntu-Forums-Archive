### Post by cozmicharlie on 2008-09-17
You did not state how you installed it and what version you installed.  I assume you installed it in Wine and the latest itunes.  Ipod support in itunes via Wine is possible but it appears to be difficult.  You may be able to get it to work with older versions and the patch mentioned here [http://appdb.winehq.org/objectManager.php?sClass=version&iId=11372](http://appdb.winehq.org/objectManager.php?sClass=version&iId=11372).

It may be easier for you to use one of the native Linux packages like gtkpod (available via Synaptic).
