### Post by cchhrriiss121212 on 2010-11-25
I haven't used LMMS in a while, but there should be an option in the preferences menu that lets you set latency on a sliding bar. There are other tweaks you can do like switching to a lighter DE (like LDXE or IceWM), installing a lowlatency kernel and disabling compiz and wireless processes. I'm not sure what cpu you have, but you can try setting cpu scaling to "performance" (right click on the top panel and you should see an option to add a scaling applet).

I think an external device is only worth it if you want to record from another source, an Edirol UA-25 would be good if you ever want that.
