### Post by boombox1387 on 2010-03-20
I second the vote for Banshee.  When I first switched to Linux, I tried to get iTunes working for a long time.  It was really buggy and slow through Wine, so I ended up moving to Banshee.  It wasn't an easy process for me, but it was well worth it.  When I made the switch, Banshee didn't have a working iTunes importer, so I had to manually update ratings and playcounts.  Now that Banshee has added the importer, moving from iTunes should be a much less painful process.

If Banshee doesn't meet your needs, Songbird is probably the most iTunes-like media software for Linux.  Last I tried, it wasn't quite as snappy as Banshee, and it doesn't blend in with the rest of the desktop as well, but it's definitely a good media player.

If you get iTunes working through Wine/Crossover, that's great, but personally, I think a native, open source media player really is the way to go.
