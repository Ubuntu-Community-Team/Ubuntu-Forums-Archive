### Post by IanW on 2008-12-23
I just get the pre-compiled debs from [GetDeb](http://www.getdeb.net).
They've had 1.6.1 since Dec 15th.

(be sure to select Hardy/Intrepid & 32/64 bit architecture in the website's chooser so you get the right build)

Then install with GDebi by double-clicking on the deb files (qdvdauthor-data first, then qdvdauthor)
