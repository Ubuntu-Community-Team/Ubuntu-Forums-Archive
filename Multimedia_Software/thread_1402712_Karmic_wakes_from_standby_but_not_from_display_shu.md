### Post by ColJep on 2010-02-09
I have a computer that uses an Asus P5QL-CM motherboard with built in Intel GMA 4500 graphics.

It will not wake up if I allow the system to shut down the monitor. I can only recover by using the reset button. If I let the system put the computer in sleep mode then the normal quick press of the main power button restores everything without any trouble.

I suspect the Intel graphics but cannot find anything through Google or the forums. Has anyone come across this problem?

I did once try to recover from a shut down monitor by switching of the power and restarting but I had to reinstall after that. A good reason for a separate /home partition!

I added a separate Nvidia video card and the problem has gone. Problem confirmed due to Intel video.
