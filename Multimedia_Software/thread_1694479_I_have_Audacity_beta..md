### Post by cchhrriiss121212 on 2011-02-24
Try using the LADSPA plugins from software centre, they will be picked up automatically. If you are a new user, get used to installing through the package manager.

To do what you asked (which I don't recommend), you can use the sudo command to give yourself access to system folders. Like this:
```
sudo nautilus
```
then copy the files wherever you like.
