### Post by markbuntu on 2008-10-04
If you are using PulseAudio, you need to change the output device these apps are using in the PA Volume Control. You can change the default device by right clicking on it in Output Devices, or move the stream in the Output by right clicking on it and choosing move stream. If you want more info on that, you can scan through my guide here. Just read the parts that seem interesting to you:

[http://ubuntuforums.org/showthread.php?t=843012](http://ubuntuforums.org/showthread.php?t=843012)
