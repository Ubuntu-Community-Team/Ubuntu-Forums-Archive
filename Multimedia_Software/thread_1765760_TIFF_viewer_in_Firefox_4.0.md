On the US patent office website patent images are shown in TIFF format. Since I've upgraded, the TIFF viewer I had installed (Mozilla Tiff Plugin) has stopped working. I tried reinstalling it - through the .deb file - but no dice. Anyone got any suggestions on what else I could try?

### Post by Rhubarb on 2011-05-29
There's a firefox plugin that may work for you:
[https://addons.mozilla.org/en-US/firefox/addon/aspator/](https://addons.mozilla.org/en-US/firefox/addon/aspator/)

I haven't tested it, but it claims:
"2. With USPTO patent drawings directly. No need to install TIFF viewer."

It should work with firefox 4 as well.
