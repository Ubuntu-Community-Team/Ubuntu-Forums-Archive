Hello Ubuntu forum!  I'm groovydaddy, and I'm a green-ear newbie to Linux.  :)  Feel free to send me a PM if you want.  I'd like to network within the forum to obtain new contacts.

Now, to the digital camera part.  I have a Fujifilm Finepix A610 camera.  Fujifilm told me there is no Linux driver for this camera, so I have to use it on my Windows to download the pictures.  NOT SO!!  There is no driver available, but it is possible to download the pictures using the "Camera" app.  The A610 shares the same owner's manual with the A800, which is supported by Camera.  I downloaded Camera, hooked up my A610 to the USB, and BAM!  Photos without Windows!!

If anyone else has been driving themselves crazy trying to get this particular camera to work with their Ubuntu flavor Linux the way I have, install Camera and you'll be able to download pics.  I hope this helps.  :D  Be sure that your camera is hooked up to the USB and powered on before you open Camera, or Camera will not recognize the camera.  :)
