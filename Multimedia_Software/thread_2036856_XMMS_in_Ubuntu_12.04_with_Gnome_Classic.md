I just thought I'd post this in case anyone would still like to use XMMS which is much similar to old school winamp.

The files shown are for i386 systems and I expect others would be needed for other systems.

All the files came from [https://launchpad.net](https://launchpad.net) bar Debian.tar.gz which came from [www.schultz-net.dk](www.schultz-net.dk).

I installed the four "lib" files then the XMMS file.

The Debian.tar.gz file is simply a skin.  To install it or others go to your home folder.  Press Ctrl H, open the .xmms folder then Skins folder and put it in there (no need to open / unzip the file).  Open XMMS and install the skin via options ---> Skin browser.

I also had to change the audio output setting in XMMS options ---> preferences

The XMMS installer automatically makes an entry in the gnome classic menu.  I have tried launching XMMS from terminal in "unity" and it does work but you would have to add it to the menu should you wish too.

These are the i386 files to search on launchpad:

libgtk1.2-common_1.2.10-18.1build2_all.deb

libgtk1.2_1.2.10-18.1build2_i386.deb

libglib1.2ldbl_1.2.10-19build1_i386.deb

libaudiofile0_0.3.2-0ubuntu4_i386.deb

xmms_1.2.10+20070601-1build2_i386.deb

Enjoy
