Not sure if kino can deal with .mov As long as the camera can be connected via firewire, it's worth a go (kino only works with firewire. > sudo apt-get install kinoKino can output raw dv so it can be edited either in kino or with another editor - lives and cinelerra (although complex to learn) are both decent editors.
