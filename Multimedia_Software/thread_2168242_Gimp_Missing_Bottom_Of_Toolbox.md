### Post by QIII on 2013-08-17
Hello!

In the menu bar in the main GIMP dialog, click Windows -> Dockable Dialogs -> Tool Options.

Drag the tool options dialog into the area where you see "You can drop dockable dialogs here" by clicking and holding the "tab" (not the window decoration).

Let us know if that helps!
