### Post by aurelix on 2010-05-04
> **ahalin said:**
> Did you ever get an answer to this?

In theory it's really simple, all you have to do is go to the digiKam settings and tell it to store the tags in the image files, not only in its DB. 

Perhaps you have to update your images' tags once so that the file headers are rewritten with the currently assigned tags. The simplest way would be to add a dummy tag to all images. (may take some time)

Once you did this, when you import the files into F-Spot, the tags will show up under "Imported Tags".
