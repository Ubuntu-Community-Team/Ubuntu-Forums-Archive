### Post by hoosemon61 on 2013-09-17
I have a couple Sansa Clips that I use primarily for audio books and for podcasts.

I get podcasts from two computers, one running Ubuntu 12.04, and the other running Windows 8.

I'm using Gpodder on each platform.

All files copied to the player from either computer are visible on the player.

When plugged into either computer, I can only see the files copied to the player with that computer.

What's happening here, any ideas?

---

### Post by Yellow Pasque on 2013-09-17
I had that problem with my Clip too. It was caused by Windows using MTP while Linux was treating the device as a regular USB disk (or vice versa).
There's actually a setting on the Clip ("USB Mode") to pick either MTP or USB, but the default is to Autodetect and let the OS decide.

---

### Post by hoosemon61 on 2013-09-19
Cool, it worked.  I set it to MTP mode and the files saved using Windows are now visible on my Linux computer.

Thanks!
