### Post by tom56 on 2008-05-14
You need libdvdcss2 (esentially a plugin for reading copy-protected DVDs) The easiest way to install it is to add the Medibuntu repository ([http://www.medibuntu.org](http://www.medibuntu.org) - follow the instructions here) and then install libdvdcss2 from Synaptic Package Manager. Whilst you are there, install vlc, which is a great program for watching DVDs and other video files.
