### Post by bgeraghty on 2009-10-28
"Stereo Mix" was the logical capture device from windows that allowed you to record anything that was to be output to speakers.

PulseAudio might be the key if you are looking for something similar to that. It is supposed to work with Audacity, according to the wiki.

See "Recording example using PulseAudio and Audacity" at the bottom of this page:
```
https://wiki.ubuntu.com/PulseAudio
```
