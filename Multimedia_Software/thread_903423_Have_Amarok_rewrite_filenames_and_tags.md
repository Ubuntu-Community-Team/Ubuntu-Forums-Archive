### Post by treehouse on 2008-08-28
I'm not up to speed on album art, but I think there's a way to embed it in the mp3 metadata, although if you already have the image files in your folders you might just rename it as a hidden file?

For filenames and tags, I would personally go with a seperate program. I use Easytag, but there is also Quod Libet and a few others that do batch edits of both filename and metadata, and can lookup track info through CDDB. What I like in Easytag (I think Quod Libet does this too) is if you have all the info in either the filename or metadata you can make a filter to mass rename all the files according to whatever format you like.
