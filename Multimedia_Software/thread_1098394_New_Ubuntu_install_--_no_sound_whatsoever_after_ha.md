### Post by Therion on 2009-03-17
Since you have an Audigy card, try this... It's worked for me in the past:

Right-click on the Volume icon and select "Properties".  
Do you see a "Switches" tab?  
If so, click on it.
Do you see a check-box for **Audigy Analog/Digital Output Jack**?  
If so **clear** that checkbox (meaning, remove the check).

See if you have sound now.
