### Post by coldraven on 2016-01-02
Whoever created the pdf has set a password to stop anyone editing it. You will have to contact them and ask for a version with no password.
From LO's help screen in the File>Export as PDF>Security tab:
> **PASSWORD**   Assigns a password to prevent users from making unauthorised changes.
 You should only use passwords that are hard to find by other persons or programs. A password should follow these rules:
 
[LIST]
[*] Length of eight or more characters. 
[*] Contains a mix of lower-case and upper-case letters,     numbers, and special characters. 
[*] Cannot be found in any wordbook or encyclopaedia. 
[*] Has no direct relation to your personal data, e.g., date of     birth or car plate. 
[/LIST]
 The open password must be entered to open the file.
 The permission password must be entered to edit the document.
