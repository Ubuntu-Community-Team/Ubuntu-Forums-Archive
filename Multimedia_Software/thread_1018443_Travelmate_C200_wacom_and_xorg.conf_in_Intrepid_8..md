### Post by Favux on 2008-12-25
Hi MadeR,

Have you tried the Laptop Testing Team's site:

[https://wiki.ubuntu.com/LaptopTestingTeam/]("https://wiki.ubuntu.com/LaptopTestingTeam/")

Why did you comment out the serial.conf stuff?  Try these commands to see if you have an active wacom module:
```
lsmod
modinfo -d wacom
```
if you don't see "wacom" it isn't there.  Kernel updates break the wacom kernel driver.

If you want to reinstall Wacom, Loic2 has some backports he wants folks to try:

[http://ubuntuforums.org/showthread.php?t=870640&page=41]("http://ubuntuforums.org/showthread.php?t=870640&page=41")

posts # 401 and # 416.

Also I'm pretty sure the "cursor" stuff does nothing for you and you can comment it out.

You might also want to look at my post for a guy with a C310, post # 14:

[http://ubuntuforums.org/showthread.php?t=983666&page=2]("http://ubuntuforums.org/showthread.php?t=983666&page=2")
