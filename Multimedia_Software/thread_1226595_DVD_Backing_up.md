### Post by 67GTA on 2009-07-29
I use k9copy. Make sure you have the Medibuntu repo, and install libdvdcss2 to circumvent the copyright stuff. I like K9copy because it will keep the original DVD menu. It will shrink and output an ISO image or audio/video files. I would also recommend K3b for burning.
