### Post by varunendra on 2012-10-18
Welcome to the forums muddy1 :)

Assuming you are using default Ubuntu with Unity, please go to
**System Settings > Details > Removable Media**.

There you can set "Music Player" to "Ask what to do" (default), or check the checkbox for "Never prompt..." (disabled by default). If required, you can also set the default behaviour for other media using "Other Media.." button.

Hope it helps.
