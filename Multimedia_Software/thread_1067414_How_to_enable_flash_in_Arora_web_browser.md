### Post by iopo on 2009-02-18
As the arora web page explains
[http://code.google.com/p/arora/](http://code.google.com/p/arora/)
plugins (including flash) will not work if you are using qt4.4 (i.e. if you are using kubuntu with kde 4.*).
The website also explains that the problem is solved if you install qt snapshot (if you follow the link "BeginnerStepByStepInstructions" it will explain you how to do it). However, qt snapshot is the unstable pre-release version of qt, so you may end up messing up your system.
