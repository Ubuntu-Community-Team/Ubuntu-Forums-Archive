anyhoo once you have the file system you desire, so as not to confuse rythmbox by moving alot of folders arround causing it to "know" that there is a file there (when actually you moved it)
you can just look at the main screen under the music catorgory, click on one song to highlight it, and control all to highlight every song.  Right click and select remove, BUT NOT MOVE TO TRASH that will delete your library from your computer not just rythbox.  After that just go into edit, pref.
and select the highest up folder that your library will be found under, (like the music folder for example) and you may want to check the monitor my library option so if you were to add anything into the library folder it would be included in rythmbox without the need to import it manualy.  This should keep it from having import errors when ever you delete a file form your library, but of that Im not certian.   anyway just do that and click ok and it should start re-indexing and importing your library.
