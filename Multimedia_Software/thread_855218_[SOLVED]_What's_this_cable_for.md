### Post by NilsE on 2008-07-10
With the labeling you provided it indicates that the cable is used to hook your PC up to a display device like a TV that accepts "component" input. There is multiple ways to cable video. The following is a list of the more standard ones in order of worst to best

composite = one cable with an RCA jack carrying the video signal

Svideo = multiple cables within one cord with a multi contact DB type connector

component = 3 cables with RCA jacks for multiple signals

HDMI = full digital signal with one DB type connector.
