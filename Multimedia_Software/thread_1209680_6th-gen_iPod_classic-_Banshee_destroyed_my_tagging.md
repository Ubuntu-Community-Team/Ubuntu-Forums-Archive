### Post by tacantara on 2009-07-10
I've used Banshee myself in the past to sync an iPod Classic, and I ran into some of the same problems that you're encountering.  I concluded that the best way to sync an iPod is with (unfortunately) iTunes.  I had to install it on my XP virtual machine since Wine doesn't play well with the most recent versions of iTunes.

I honestly think that Banshee is a great media player/organizer, but the folks at Apple make it impossible to use their products with anything else but their software.  Unless anyone else out there knows of a better (i.e. Linux based) solution, I'd say use iTunes.

JMHO
