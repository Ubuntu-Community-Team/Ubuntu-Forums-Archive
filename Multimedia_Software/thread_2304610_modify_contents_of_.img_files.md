### Post by brian-mccumber on 2015-11-28
FuriousISO mount is available in the Ubuntu software center - [https://apps.ubuntu.com/cat/applications/furiusisomount/](https://apps.ubuntu.com/cat/applications/furiusisomount/)

It seems to be able to mount .img files and then you can access the files in the file browser.
