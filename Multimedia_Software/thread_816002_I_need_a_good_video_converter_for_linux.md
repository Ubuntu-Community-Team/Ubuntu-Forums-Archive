### Post by gandaran on 2008-06-02
I recommend ffmpeg, install the ffmpeg only from the medibuntu repository,
ffmpeg from the normal synaptic repositories is a bit limited on codecs.
if you want a gui for ffmpeg, just google **winff**, there's a .deb package for ubuntu in winff homepage.
other applications worth looking into are **avidemux** and devede, both can be found in synaptic.
