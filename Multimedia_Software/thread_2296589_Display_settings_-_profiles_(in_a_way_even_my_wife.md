### Post by Bucky Ball on 2015-09-27

Not related to desktop environments.

I have a dual-monitor setup. Sometimes I only want the monitor on, not the laptop screen, so I installed arandr (it's in the official repos/SCentre)> configured it to show just the monitor> saved that setting in arandr.

Next, you can check that saved file and copy the command it is using> create a launcher> paste in that command as the command> place launcher wherever you like. One click and I have just the monitor screen, no laptop screen.

So ... you could do the same and create launchers for the various setups you need to use. :)
