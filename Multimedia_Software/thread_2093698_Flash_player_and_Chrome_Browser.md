### Post by monkeybrain2012 on 2012-12-11
You shouldn't need to install anything, chrome comes with its own flash.

To see if it is enabled, open chrome and type "about://plugins" in the address bar, you should see two versions of flash: a system version and a chrome version, you should disable one of them.
