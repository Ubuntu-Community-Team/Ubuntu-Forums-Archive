Hi,

I've just bought a new monitor - Viewsonic VA2213w and am trying to set it up with Lucid.

The resolution required is 1920x1080 @ 60Hz.

I have an NVidia GTS250 with the proprietary driver v195.36.24 installed.

I can set the right resolution in the nvidia-settings app, but it will not survive a restart. X always reverts to 1024x768.

I have manually edited xorg.conf now to try to force the mode. The GDM login screen displays at the correct resolution, but on logging in X changes the resolution.

Xorg.0.log shows the resolution being correctly set to 1920x1080 initially, then right at the end for some reason it is reset to 1024x768.

I can't for the life of me figure out how to get this to work correctly.

Any ideas?
