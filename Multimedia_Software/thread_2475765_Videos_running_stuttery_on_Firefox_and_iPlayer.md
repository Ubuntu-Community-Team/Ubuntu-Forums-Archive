### Post by TheFu on 2022-06-07
For issues like this, I usually start by checking that the software is using hardware acceleration provided for playback by the GPU. Not all do, so I avoid those.  For example, I don't use firefox to play videos.  Firefox is my main "secure" browser, so for something like video playback, which I consider entertainment, I'll use something else - usually mpv.

Using software with hardware video decoding support is most important on slower systems with limited RAM.
