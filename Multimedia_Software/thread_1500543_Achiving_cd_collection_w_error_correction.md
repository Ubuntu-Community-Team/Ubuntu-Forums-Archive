### Post by ron999 on 2010-06-03
Hi
I use RubyRipper with Karmic Koala.
This rips each track 2 (or more) times and compares them for differences caused by reading/ripping errors.
But if a CD is damaged I don't think it can repair that type of error.
There's information here:-[http://wiki.hydrogenaudio.org/index.php?title=Rubyripper]("http://wiki.hydrogenaudio.org/index.php?title=Rubyripper")
