### Post by Infastyles on 2010-01-13
[SOLVED]

I guess all the threads I read explaining how to get this working made the assumption that I should know to stop gnome before doing a xserver reconfigure

Here's how I got it to work.

sudo /etc/init.d/gdm stop
sudo dpkg-reconfigure xserver-xorg
sudo /etc/init.d/gdm start

substitute gdm for kde if ya use that
