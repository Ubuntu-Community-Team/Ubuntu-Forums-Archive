> **zaphod777 said:**
> I was wondering if there would be any advantage to trying to install the nvidia drivers or if you even could on a server install.

You could, but if you're using it solely as a DLNA server, say with ps3mediaserver, it shouldn't matter at all.  The server will just be sending a stream of data to the display device.  How the stream is decoded and displayed depends on the graphics hardware on that device.
