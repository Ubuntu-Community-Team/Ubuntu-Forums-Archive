### Post by Musicalmidget on 2009-05-03
Having successfully converted almost fully from Windows to Ubuntu myself, I'm now looking into doing the same with my father.  He seems quite keen on the idea and we've installed Ubuntu on another partition of his hard drive to try out, but we're having problems finding the right software for him to use.

The most important thing he uses the computer for is recording old vinyl and cassette tapes, cleaning up the audio and then exporting digital copies, which are in turn added to his music collection and synchronised with his iPod.  For years, he's been using a Windows product called 'Audio Cleaning Lab' by MAGIX, but obviously he will no longer be able to use that if he makes the switch to Ubuntu.

We've come across Audacity, which looks similar in many ways to the software he's already used to so we're going to give that a try.  The big problem however, is that it has one killer feature that he really needs and to my knowledge, Audacity does not.  This feature is a spectrum analysis that not only plots the waveform of a section of audio, but also allows him to manually identify clicks and pops and remove them with a select and delete tool.  Audacity will plot spectrums, but it doesn't appear to offer the same kind of functionality.

We've also seen a program called 'Gnome Wave Cleaner' that looks to have an analysis tool more like the one he's used to, but there doesn't seem to be any way of manually removing pops and clicks from within the analyser.

Finally, we've also found Ardour which reminds me a lot of Pro Tools, but in all honesty it looks to be quite complicated and I wouldn't even know where to begin with that.

So, I'm wondering if anyone here is aware of any other software that will work on Ubuntu that we haven't considered?  That spectrum analysis tool is really proving a pain to find in any other software, but I suspect it will ultimately be the one thing that will or wont persuade him to take the leap and switch.

I should point out that I have looked into using the existing software through Wine, but this did not seem to work.  We had partial success using the Windows software in a virtual machine, but if the only way to do what he needs is to run Windows virtually, I have to wonder whether switching would be the right thing to do?

I should also point out that the simpler the software, the better for him.  That said, he's not against learning something new as long as it's not going to be too overwhelming.

We'll be trying Audacity at some point over the next few days.  In the meantime, if anyone has any alternatives to recommend, we would very much like to hear them.
