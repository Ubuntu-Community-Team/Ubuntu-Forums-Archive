### Post by tgalati4 on 2015-02-27
Welcome to the forums.  New user and server are not normally used in the same sentence.

*sudo* means "Super User Do".  It's simply used to execute a program with root or administrator privilege.  And yes it is used extensively when setting up a server.

```
sudo apt-get update
sudo apt-get upgrade
```

Just setup 14.04 desktop machine and play with it for a while.  You can add a lot of services to the desktop machine.  When you get stuck, search in the Popular Pages link below and if you can't find the answer there, ask a single, specific question in the forums.
