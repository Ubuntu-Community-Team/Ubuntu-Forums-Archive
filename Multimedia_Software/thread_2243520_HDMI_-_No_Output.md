Unfortunately, AMD/ATI dropped Linux support for that adapter quite some time ago, so there is no driver to update to.  You will need to stick with the open source driver installed with Ubuntu.  There is nothing to be done about this, since proprietary drivers are supplied by OEMs, not OS distributors.
