### Post by quixote on 2009-10-01
According to this site, [psychocats.de]("http://www.psychocats.net/ubuntu/itunes"), who are generally a very good resource for many things to do with linux, the short answer is "no."

There are workarounds, however.  One is to try some of the more or less compatible programs and see if they work for you.  There's a list on [wikipedia]("http://en.wikipedia.org/wiki/Comparison_of_iPod_Managers#Linux").

The simplest thing, even though it may not seem so at first, is to install VirtualBox, create a virtual windows machine inside it, install windows on that using the spare install CD you have lying around :-? , and just do your iTunes stuff in virtualization.  It's a slightly longer start up time than just clicking on one program under Win or OSX, (you start vbox, then windows, then the itunes program), but for that you can use a virus-free OS you control for all your other work.

Sun-virtualbox is in synaptic (System > Administration > Synaptic), so it's an easy, just click, type of install.  (Don't get the virtualbox-ose because it doesn't have good USB support yet.)

### Post by chipper_farley on 2009-10-15
I decided to bail on iTunes all together.  I'm going to have to convert some of my music to another format which can be done with tools like Tunebite (tunebite.com).  A lot of iTunes music acually has no DRM and will play fine in Banshee so you'll want to test out your library.  Also, any files you ripped from a CD will play too.  Just take your files and set them somewhere your Ubuntu machine can access them and try them out.  For me, most worked with no changes.  I'm using Banshee for my player and it's great.

Amazon has started offering MP3 downloads from their site and have an Ubuntu downloader program that installed like a charm on my machine.  Then I use bashee from there.

Good luck!
