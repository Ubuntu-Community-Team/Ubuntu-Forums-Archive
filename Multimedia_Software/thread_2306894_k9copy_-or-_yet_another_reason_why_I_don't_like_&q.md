### Post by rebeltaz on 2015-12-19
I know y'all are tired of my complaining about "upgrading" to the lastest OS version, but every time I turn around, I find something else messed up by the "upgrade"... I know - stop turning around.

So I seem to have learned that k9copy was discontinued back in 2011. That never bothered me, since it worked. Since it worked, there was no need to "update" it. Hence... I never knew it was abandoned. Yesterday, I went to use that program on my laptop only to notice that when I "upgraded" the OS to 14.04 LTS, apparently the system took it upon itself to remove k9copy. Why? And why remove a program that works from the repository just because it is no longer "updated" by the developer?

A little research and I found that k9copy was reborn so to speak as k9copy-reloaded - [http://k9copy-reloaded.sourceforge.net/index.html](http://k9copy-reloaded.sourceforge.net/index.html)

Unfortunately, this version requires building and since it is a KDE based program, it requires tools I don't really feel like installing/learning. 

SO.... my question is this... Since this is no longer available to download, is there a program that can function similarly? In my case, I used k9copy to rip a single selected video track with it's associated audio track(s) from the DVD to the hard drive in ISO format. I would then, at a later date, feed these ISO images to handbrake for processing into m4v files. 

I can still use k9copy on my shop computer - which I should apparently NEVER "upgrade" - but I still need something on my laptop. Thanks.
