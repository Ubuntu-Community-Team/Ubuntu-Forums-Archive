### Post by Raffles10 on 2009-12-14
[DeVeDe]("http://www.rastersoft.com/programas/devede.html") works very well, it can convert all types of video file and creates an iso that can be burned to dvd. 

```
sudo apt-get install devede
```

I use Brasero to burn the iso's but any dvd burner should do. :)
