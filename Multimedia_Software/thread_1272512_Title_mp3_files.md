### Post by Matt Yun on 2009-09-22
I use [Audio Tag Tool]("http://packages.ubuntu.com/hardy/tagtool"): apt-get install tagtool

This utility can do single or batch renaming of mp3 files based on ID3 or ID3v2 tagging, and vice versa.
