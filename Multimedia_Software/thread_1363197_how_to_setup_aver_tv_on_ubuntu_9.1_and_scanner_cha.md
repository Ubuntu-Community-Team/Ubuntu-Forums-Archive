### Post by jimmers on 2009-12-24
I have Aver Tv up and running on Jaunty, it is AVerTv Digi Volar X, after searching many forums,
I just plugged it in and in a Terminal typed:

sudo apt-get install dvbscan
sudo apt-get install w-scan

I use Kaffeine to watch TV, and it is good  just using the mini Antenna that came with the dongle,
but for ultimate performance it is recommended to use an External Antenna.

Hope it works for you.
