### Post by kansei on 2008-06-18
> **jasonpeinko said:**
> i really dont want to reformat, and i still want to use my computer for other things mainly its a gamming setup.

go to synaptic and search for "myth" (search only name, not description). There should be a few myth packages, then some prefaced with "ubuntu".. these are meta packages that install myth and preconfigure it, ubuntu-mythtv-frontend kinda assumes a frontend is the primary use of the computer.

But really just install whatever myth packages you need (frontend, backend, plugins, the database stuff) and it should all work reasonably well.

You can also install mythbuntu from synaptic and I strongly recommend this. The mythbuntu config wizard can step you through getting everything set up, even remote controls and IR blasters!
