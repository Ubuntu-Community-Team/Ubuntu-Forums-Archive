Thanks for the screenshots.
As far as I can see this is perfectly normal.

Files are always grayed out if you want to select a folder.

Files you add through the menu (or if you select a music folder in the settings and scan, all files get added recursively) get added to the library and not to the current playlist.

You have to switch to a different Browser like for example the search, paned or album browser. There you can select the tracks you want in your playlist and rightclick -> add to playlist ...

Or select Music>Browse Library>Some Browser to open a seperate browser where you can serach through your library and drag&drop the songs in you playlist in the other window.

If you are more of a playlist type of guy you might like Exaile.
