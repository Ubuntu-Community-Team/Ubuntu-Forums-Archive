### Post by knight2 on 2014-03-09
**PROBLEM:**
On Ubuntu 12.04 LTS Handbrake would not read DVD titles. Also Acidrip would give error message "Mencoder interrupted by user"

**SOLUTION:**
To install libdvdcss2 using libdvdcss.deb, in Terminal type command:

**sudo /usr/share/doc/libdvdread4/install-css.sh**

Hope it helps.
