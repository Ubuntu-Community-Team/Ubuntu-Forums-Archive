### Post by soulmatic09 on 2009-05-25
I've got to say, I think this is the best media library manager I've used. 

I've tried Rhythmbox, XMMS, Audacious, Amarok, and even Songbird, but I think Quod Libet takes the cake. Especially if you've got a very large collection of mp3s. It's quick, stable, responsive, and has a tray icon controller.

If you don't like the other players, I'd definitely give it a shot. You can do a search through synaptic for it.
