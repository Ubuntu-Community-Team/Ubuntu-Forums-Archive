### Post by PaulW2U on 2018-04-09
OS and application updates are made available whenever the Ubuntu developers decide to release them. Depending on how often you update your system you may get just one update or 100 updates. Most users won't really care about whether they are updating the OS or its applications. Depending on which application you use to download and install updates you may not be able to separate them anyway.

Within an Ubuntu release, applications only receive important bugfixes and security updates and are not updated to a new version. However, Firefox, Thunderbird and Chromium are generally always updated to the current major or minor version.

Chrome, as opposed to Chromium, is not an Ubuntu application so will be updated by Google whenever they decide to release an update.
