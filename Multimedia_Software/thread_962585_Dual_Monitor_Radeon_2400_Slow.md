### Post by mmbella on 2008-11-04
After a lot of hunting I got it to work - Here's how 

### DUAL MONITOR SETUP ATI RADEON 2400 WITH TWO DELL 2009W SCREENS ###

 IN THE TERMINAL

 $ sudo apt-get update

 $ sudo apt-get install linux-restricted-modules-generic restricted-manager

 $ sudo apt-get install xorg-driver-fglrx

 $ sudo depmod -a

 $ sudo vi /etc/X11/xorg.conf

    Add or confirm that Driver "fglrx" is in the "Device" Section in the xorg.conf file

 $ sudo aticonfig --initial -f

    My xorg.conf file looks like this. I had to add new screen resolutions and the virtual resolution
---------------------------------
# xorg.conf (X.Org X Window System server configuration file)
#
# This file was generated by failsafeDexconf, using
# values from the debconf database and some overrides to use vesa mode.
#

# For example:
#   sudo dpkg-reconfigure -phigh xserver-xorg

Section "ServerLayout"
        Identifier     "Default Layout"

This did not do the trick I still had to mirrored monitors when I wanted one big desktop. Make these modifications -

$ aticonfig --list-pairmode

$ sudo aticonfig --add-pairmode=1680x1050+1680x1050

$ aticonfig --list-pairmode

Restart the X server or reboot the system. You will need to go into (Ubuntu) System-Preferences-Screen Resolution
 and set the new screen resolution for the Big Desktop.

This should work.
                                                                                                                                    I hope this helps others who are having the same problem.
