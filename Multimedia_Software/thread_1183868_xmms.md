> You don't -- no one is maintaining xmms.  Use audacious -- it does about the same thing

Not entirely correct - you can still get xmms.  Audacious does not play shn.  Follow this tutorial ([http://ubuntuforums.org/showthread.php?t=833164](http://ubuntuforums.org/showthread.php?t=833164)) or search the forums and find a .deb for xmms and install it.  Then install the shn codec and the xmms-shn plug in and enjoy your music.  The tutorial is for Hardy but it works on all versions of ubuntu.  Don't be concerned that it is all command line - just cut and paste and you will be fine.  Post back if you have any problems.  

Also, I saw your other post on converting - once you install the shn codec you can use SoundKonverter to convert shn to wav.

Enjoy!
