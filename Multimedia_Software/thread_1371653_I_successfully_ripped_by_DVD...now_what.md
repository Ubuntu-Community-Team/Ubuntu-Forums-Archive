### Post by N00b-un-2 on 2010-01-03
DVD::RIP is a fantastic program, but it is only intended for ripping video from DVD and it creates a standard AVI file, 1.4GB by default.  I have used dvdrip to back up my entire movie collection onto my HTPC which is attached to my TV.
So in short, I think you are missing the point.  dvdrip is not intended for *copying* your dvds so much as *ripping* them to AVI files.  However, you can use another program called *DeVeDe* to create a burnable .iso file with the video files you ripped.  You can then use Brasero to burn the ISO.  Kind of a roundabout way of accomplishing things, but what you are trying to do is illegal, so there is no program available that will allow you to copy DVDs directly.  The last application I knew of that would do so was DVDxCopy Deluxe for Windows, and 321 Studios (the software company that made it) was *VERY* quickly issued a cease and desist order and threatened with several lawsuits.  I wonder if you might even find a copy of it floating around on the torrent sites still...

in a terminal type
```
sudo apt-get install devede
```
