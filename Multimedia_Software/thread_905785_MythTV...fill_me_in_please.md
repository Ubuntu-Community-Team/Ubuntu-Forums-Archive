### Post by lovinglinux on 2008-08-30
Follow this tutorial [http://www.parker1.co.uk/mythtv_ubuntu.php](http://www.parker1.co.uk/mythtv_ubuntu.php)

Unfortunately, MythTV is not easy to configure. 

A capture card works pretty much like a VCR. It depends on which input slots you have. For example I have a very old and basic card which has RF input for regular antenna cables and Composite/S-Video input for setup boxes. So I can watch pretty much everything or even plug a camera in it If a had one.

Modern capture cards can also have digital input slots.
