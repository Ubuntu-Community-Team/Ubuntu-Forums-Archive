### Post by SebbyT on 2009-05-19
Aha! I *think* it's a conflict with Compiz/Desktop Effects. Go to System > Preferences > Appearance, then to the Visual Effects tab and select the option: None. I was immediately able to shift+click to select concurrent layers and ctrl+click to select non-adjacent layers. So far so good...
