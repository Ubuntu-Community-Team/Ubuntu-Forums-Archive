I used Synaptic Package Manager to install the gimp importer for raw camera images (ie gimp-ufraw).

It imports raw images from my Nikon Coolpix 8800 all right, however if I make any changes and then click on OK, the original file appears to be overwritten by the modified version, without any option to Save As using a different name. The modified image then opens automatically in Gimp.  If Gimp is closed the resulting file is the version that was modified using gimp-ufraw, consequently the original version is lost. 

The obvious workaround is to make a copy of the original file beforehand, and then open the copy using gimp-ufraw, but this seems rather clumsy.  Why isn't there a Save As option?  Am I missing something here?
