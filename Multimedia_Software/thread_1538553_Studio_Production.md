### Post by cchhrriiss121212 on 2010-07-25
#1 - I don't recommend trying to run any music software through WINE (you will lose functionality) if that is what you mean. Reaper does seem to work well according to some on this forum.
#2 - There is lots of great quality FOSS audio software for Ubuntu, try looking at whats in [Ubuntu Studio]("http://ubuntustudio.org/") for starters.

Unfortunately you m-box is not supported, so unless you want to buy a supported device you will need to stick with Windows. Someone actually [made their own driver]("http://www.zamaudio.com/?p=97") for this card, but you will need to compile it into the kernel yourself.

### Post by pepun on 2010-07-26
I also wouldn't recommend running such music or audio software through wine.
but i can totally recommend ardour, which is very similar to pro tools and a real good piece of software ([http://ardour.org/](http://ardour.org/)).
it's even possible to compile it with vst-support so that you can use some plug-ins designed for windows. this uses wine though. but i think it is still much better than to run the whole thing through wine.
