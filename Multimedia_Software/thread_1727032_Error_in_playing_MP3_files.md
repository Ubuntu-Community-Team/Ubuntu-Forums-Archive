### Post by papibe on 2011-04-12
As far as I know, the default media player should offer you to install the necessary codecs (like in this [tutorial]("http://www.psychocats.net/ubuntu/mp3")). If not, you can try (from [this]("https://help.ubuntu.com/community/RestrictedFormats/") help page):
[LIST]
[*]Go to Applications &#8594; Add/Remove...
[*]Set Show: to All available applications
[*]Search for ubuntu-restricted-extras and install it.
[/LIST]
I hope it helps.
