### Post by Favux on 2010-06-21
Hi viMitch,

I have the P&T working fine in Lucid.  I even have a touch toggle script so Button1 acts the same way as in Windows.  Only problem is there is a little lag in Gimp with the stylus.  But I think that only affects a couple of us.  Not sure why yet.  I've minimized it.

For Karmic and Jaunty you'll have to compile and install at least linuxwacom 0.8.6-2.  See the [linuxwacom HOW TO]("http://ubuntuforums.org/showpost.php?p=6546012&postcount=1").  It's easier than it looks.  For the xsetwacom script see [post #1112]("http://ubuntuforums.org/showpost.php?p=9492033&postcount=1112").

Still maybe some teething troubles in Lucid but I think they're licked now.  This time you'll again need to compile linuxwacom 0.8.6-2 (at least) but just for the wacom.ko, see [post #833]("http://ubuntuforums.org/showpost.php?p=9485092&postcount=833").

Also see [post #384]("http://ubuntuforums.org/showpost.php?p=8165182&postcount=384").
