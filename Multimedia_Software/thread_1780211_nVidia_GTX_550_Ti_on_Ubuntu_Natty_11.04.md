### Post by BicyclerBoy on 2011-06-29
From reading through the nvidia driver downloads "Supported Products" tab..
You need version 270.41.06 (or later) to support GTX550Ti..
This version may not be in std distribution repos.

From the forum posts, it seems the Jockey tool (additional hardware drivers) does not work properly.

These ppa have later version of nvidia driver if you don't want to use the nvidia installer method.
[http://www.ubuntuupdates.org/ppas/27](http://www.ubuntuupdates.org/ppas/27)
or xorg-edgers ppa
