### Post by maphew on 2008-12-15
according to this message on the vlc forum, [http://forum.videolan.org/viewtopic.php?f=16&t=53015&p=175507&hilit=firefox+plugin#p175507](http://forum.videolan.org/viewtopic.php?f=16&t=53015&p=175507&hilit=firefox+plugin#p175507), the only way to get playback controls is to a) edit the web page and add javascript to create controls, or b) install the Firefox Greasemonkey extension and put the javascript for controls there (though how to carry this out is left as an exercise for the reader).
