### Post by Geogriffith on 2008-05-18
If you install the "Adblock Plus" addon for firefox, there won't be a commercial messing up the embed code, and you can watch the video just fine.

Open up a Firefox window, go to "Tools">Add-ons>Get add-ons, type in "adblock" and search, and select and install "Adblock Plus". When you restart firefox, select the "Easylist USA" option.

You can always disable adblock when you don't need it, if you prefer to keep ads on most sites (if you think they deserve the ad revenue, etc). But the way that videoegg sets up their advertising messes up the video, for some reason, and this is the easiest way to get around it.
