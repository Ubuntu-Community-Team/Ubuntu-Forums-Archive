### Post by richierich1986 on 2013-05-19
A good things to look at would be ESI Juli@ XTe (PCIe) or the original ESI Juli@ PCI. 
Also, M-Audio Audiophile 2496 PCI works great and is still available after all these years.

Other than these three, I wouldn't dare to give any advice, although I hear some Roland (Cakewalk)
USB soundcard modules work flawless. This is not own my experience though. I've never used any of them.

The ESI Juli@ XTe is nice though, because it has convertable inputs. You can use the RCA plugs
or, if you switch the board around, TRS (balanced) jack inputs! Very good quality sound, especially,
if like me, you're using studio grade monitor speakers, which work with balanced jack and XLR inputs.
