### Post by chewearn on 2009-12-29
Did you install SMPlayer via Ubuntu repositories?  The version in the Ubuntu repositories tends to be outdated.  If you have the confidence to use PPA, I suggest getting it directly from the SMPlayer developer's PPA.  I have good results from it.

Link to SMPlayer PPA:
[https://launchpad.net/~rvm/+archive/smplayer]("https://launchpad.net/%7Ervm/+archive/smplayer")

For better integration with SMPlayer, it is also suggested to use a patched MPlayer:
[https://launchpad.net/~rvm/+archive/mplayer]("https://launchpad.net/%7Ervm/+archive/mplayer")

For the record, this info came from:
[http://smplayer.sourceforge.net/downloads.php?tr_lang=en](http://smplayer.sourceforge.net/downloads.php?tr_lang=en)
