### Post by drroc on 2017-05-27
I have an Ubuntu box connected to a 7.1 channel AVR via SPDIF cable.  Kodi works great and can send pass through audio to the AVR. AVR correctly detects the pass through audio and get I nice surround audio.  

Did anybody get Chrome to work with surround sound audio?  It would be great to get surround sound when playing YouTube Red or Amazon Prime videos via Chrome.  Right now, I only get PCM stereo.

### Post by drroc on 2017-08-01
Ok, so the trick was not to use SPDIF, but to send audio via HDMI to the receiver.  The sound in Ubuntu control panel had to be configured to HDMI 5.1.  The receiver then correctly displayed that it was receiving multi-channel 5.1 audio.  After that, I was able to listen to surround sound tests in youtube.  Everything more or less worked, except that the front stereo also played on the corresponding rear speaker as well. (e.g. left front, played on left rear as well).
