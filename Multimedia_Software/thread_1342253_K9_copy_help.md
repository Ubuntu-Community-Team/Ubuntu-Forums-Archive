### Post by benerivo on 2009-11-30
You should be able to get the tabs back by right clicking on any remaining tabs, which should show a list of all the sections you can make visible / remove. If this doesn't work, you can always delete the configuration file which in this case is /home/john/.kde4/share/config/K9Copy. I'm not sure about the 'insert a recordable dvd' problem.
