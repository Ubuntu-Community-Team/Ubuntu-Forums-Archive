### Post by Rodney9 on 2010-12-17
1. in vlc select View and check "Advanced controls" (if unchecked)
2. click the red dot to start recording and click it again when you decide you want to stop recording
3. the files are huge .mp3's in your home folder (Places -> Home folder)
4. to change the record dir: in VLC go to tools -> preferences -> input and codecs -> and look for "record directory and file name"
