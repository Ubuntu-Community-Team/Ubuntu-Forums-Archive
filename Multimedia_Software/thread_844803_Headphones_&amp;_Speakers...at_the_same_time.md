### Post by cenwesi on 2008-06-30
Have this same problem on my HP 2133 laptop. 
There are two ways you can fix this:
#1: install the latest/stable alsa sound - **[http://www.alsa-project.org/~valentyn/Alsa-sound-mini-HOWTO-4.html#ss4.1](http://www.alsa-project.org/~valentyn/Alsa-sound-mini-HOWTO-4.html#ss4.1)**

#2: Install kernel 2.6.25.7. That fixed my problem. I will suggest you do the same and you can always go back to your current kernel if it does not work.
