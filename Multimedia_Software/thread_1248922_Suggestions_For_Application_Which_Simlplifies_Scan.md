### Post by andrewc6l on 2009-08-25
I did the same with a smaller number of pictures (about 30). I still did scanning & cropping manually, but I found one good GIMP plugin was Deskew ([http://www.cubewano.org/gimp-deskew-plugin/](http://www.cubewano.org/gimp-deskew-plugin/)). Contrary to its description, it will straighten out almost any graphic image where an edge is slightly off horizontal.
