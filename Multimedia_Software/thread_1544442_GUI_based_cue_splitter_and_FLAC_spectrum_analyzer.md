### Post by Yahoé on 2012-03-26
Flacon is indeed a great choice, fast and reliable.
However for the lastest version download directly from the Flacon site [http://code.google.com/p/flacon/downloads/list](http://code.google.com/p/flacon/downloads/list) or better yet use the ppa **ppa:flacon/ppa **by typing in a terminal:
sudo apt-add-repository ppa:flacon/ppa && sudo apt-get update && sudo apt-get install flacon
