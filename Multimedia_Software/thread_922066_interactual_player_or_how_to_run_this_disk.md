----  Section 1:  System Requirements
-----------------------------------------------------

Minimum System configuration

*  Intel Celeron/Pentium or AMD Duron/Athlon processor, 400 MHZ
*  Windows 98SE, ME, 2000, XP
*  64MB of RAM (2000 and XP: 128MB of RAM)
*  4MB Graphics Card, 800X600 resolution, 16 Bit color
*  Direct Sound compatible Sound card
*  4X DVD-ROM drive (UDMA enabled)
*  Direct Show compliant DVD decoder software installed 
*  Direct X 7.0
*  Internet Explorer 5.0

Recommended System configuration

*  Intel Celeron/Pentium or AMD Duron/Athlon processor, 600 MHZ or higher
*  Windows 98SE, ME, 2000, XP
*  128MB of RAM (2000 and XP: 256MB of RAM)
*  32MB (or higher) Graphics Card, 1024X768 (or higher) resolution, 32 Bit color 
*  Direct Sound compatible Sound card
*  16X (or faster) DVD-ROM drive (UDMA enabled)
*  Direct Show compliant DVD decoder software installed 
*  Direct X 8.1 or higher
*  Internet Explorer 6.0 or higher


InterActual player 2.0 is NOT a DVD decoder or player.  It is an enhanced Interactive
environment which requires a fully functional DirectShow compliant DVD decoder installed.

InterActual Player 2.0 and InterActual ROM content is not compatible with Macintosh operating systems.

Section 2:  Before Installing the software
-----------------------------------------------------
Shutdown any applications that are running, and disable any screen savers before 
using or installing the InterActual Player.


Section 3:  Installing InterActual Player
-----------------------------------------------------
Insert the DVD into the DVD-ROM drive on your computer.  The InterActual player
should automatically prompt you with an installation dialogue.  If this does not occur:
for Windows 98SE, ME, and 2000, double click the "my computer" icon on your desktop.
Right click your DVD-ROM drive and select "Explore".  Double click the Install.exe file.
for Windows XP, Click the "Start" button and select the "My Computer" icon.
Right click your DVD-ROM drive and select "Explore".  Double click the Install.exe file.


Section 4:  Running InterActual Player
-----------------------------------------------------
Starting InterActual Player:

Double click the InterActual Player icon on your desktop or click the "Start" button, 
select "Programs", select the "InterActual" Folder then launch "InterActual Player" 
within the "InterActual Player" folder.

For more information on using the InterActual Player, click
the "?" mark button at the top left on the main player window.

Exiting InterActual Player:

Click the "X" button located at the upper right corner of the main player window.


Section 5:  Uninstalling InterActual Player Software
-----------------------------------------------------
To uninstall the InterActual Player Software on your computer, select the Control
Panel and double click the  "Add/Remove Programs" icon. Select "InterActual Player"
then select click the Add/Remove button.  Another method is to click the "Start" button, 
select "Programs", select the "InterActual" Folder then launch "InterActual Player Uninstall" 
within the "InterActual Player" folder.
