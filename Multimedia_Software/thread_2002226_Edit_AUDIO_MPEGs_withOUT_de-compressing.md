### Post by VanillaMozilla on 2012-06-12
Audacity will allow you to open a file directly, rather than opening a copy of it.  I assume it can deal directly with compressed files in this way.  This method is likely to be extraordinarily fast.

On the minus side, I have no idea how Audacity would handle a crash if you're not working on a copy.  So just in case, I would make temporary copies as backups.
