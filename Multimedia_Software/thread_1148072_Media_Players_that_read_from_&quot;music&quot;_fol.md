Hi!

I had the EXACT same issues with Rhythmbox and Banshee, I just wanted to see my music in the directory structure that I organized on my drive. After questioning around I found that Exaile would be the one that fits my expectations the best.
I have tried to set up Sonata, but it was a bit complicated as you have to configure MPD (Media Played Daemon) as well, and I'm a bit of a newbie and didn't have the time time to research it properly, so I'm still struggling with Sonata to get it work.
I've also found **gMusicbrowser** to be quite a powerful player, that has lot of features and customization options. You can check it out too, see if you like it.
Also, in a different thread one of the guys suggested a way to set up Banshee to organize the music the way it is on the drive. You can check it out here: [http://ubuntuforums.org/showthread.php?t=1078732&page=11&highlight=sonata]("http://ubuntuforums.org/showthread.php?t=1078732&page=11&highlight=sonata")
