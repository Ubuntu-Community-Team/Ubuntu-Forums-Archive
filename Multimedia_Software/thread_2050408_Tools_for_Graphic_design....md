### Post by griphon on 2012-11-22
You can try this softwares

- Inkscape : Vector Drawing (Svg)
- Gimp : Photo/Graphic Editor
- Mypaint : Painter
- Synfig : 2D Animation
- Pencil : 2D Animation
- Scribus : Publishing Software
- Calibre : Publishing Software
- Blender 3D : 3D modeling/animation
- Lightworks : Video Editor
- OpenOctave : DAW/Music Composer
- Kompozer : WYSIWYG web authoring
- BlueGriffon : WYSIWYG web authoring (CSS3 + HTML 5 support)
- Bluefish : Text Editor with zen
- LAMP (XAMPP) : Web Local Server
