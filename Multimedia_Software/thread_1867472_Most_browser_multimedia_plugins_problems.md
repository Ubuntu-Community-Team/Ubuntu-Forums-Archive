### Post by beew on 2011-10-23
Well this is what I do for Firefox. I disable all the totem plugins except for quicktime (they appear separately in the Firefox plugin menu) and use gecko-media player for everything else (well I actually install the helix player plugin for realplayer since gecko doesn't work very well, but I hardly come across sites that only offer the realplayer option)

This is done once and for all when I set up FF so there is no need to manually switch between plugins.(see screenshot to get an idea) so even though you do have to disable things but I think the first point is satisfied. 

Having done the step above the rest of the conditions all work out.

Don't know about Chromium though, I don't use it.
