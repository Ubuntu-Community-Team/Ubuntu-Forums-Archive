### Post by squaregoldfish on 2009-03-13
Amarok documentation, including user guides, can be found in the [Amarok Wiki]("http://amarok.kde.org/wiki/Main_Page").

If you're using Gnome as your window manager (and if you're using default Ubuntu, you probably are), Amarok takes a while to load because it's a KDE application. Consequently, your machine has to load the KDE subsystem before it can start Amarok. If you were running KDE, and tried to open a Gnome application, you'd have the same effect.
