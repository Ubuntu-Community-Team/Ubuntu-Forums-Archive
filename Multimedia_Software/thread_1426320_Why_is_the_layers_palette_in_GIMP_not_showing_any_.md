### Post by Yetch on 2011-02-24
I had the same problem. I dragged the layers dialogue outside the paths, channels, layers, undo window so it became its own seperate window and this solved the problem. I'm using Win7 so I don't know if this will work in Ubuntu. I also wanted to keep layers as a dialogue in that main window though so I clicked the arrow next to the word "Layers" and in the dropdown box I unchecked "Auto follow active image", exited, then re-checked it. This also solved the problem for me.
