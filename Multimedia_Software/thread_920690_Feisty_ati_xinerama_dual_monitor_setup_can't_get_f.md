### Post by nanotube on 2008-09-15
Hi all,

Ok, so here's the deal. I am trying to get a dual-monitor setup going on my dell inspiron 5150, with an ati mobility radeon 9000 (M9) 64mb video card.

First, I tried using the mergedfb method, as per this tutorial:
[http://ubuntuforums.org/showthread.php?t=301961](http://ubuntuforums.org/showthread.php?t=301961)

But was unable to get beyond the "clone" mode. It appears to be due to this bug: [https://bugs.launchpad.net/ubuntu/+source/qpsmtpd/+bug/91586](https://bugs.launchpad.net/ubuntu/+source/qpsmtpd/+bug/91586)
which seems to be only fixed on gutsy...

So, I moved on to trying the xinerama method, as per this tutorial:
[http://ubuntuforums.org/showthread.php?p=1773624](http://ubuntuforums.org/showthread.php?p=1773624)

This worked, in as much as I got the desktop to /extend/ onto the external monitor, rather than clone the main display.

However, I still cannot get it to use the full resolution of my external monitor (1920 x 1200), instead it is stuck at 1600x1200 (same resolution as my built-in lcd panel).
