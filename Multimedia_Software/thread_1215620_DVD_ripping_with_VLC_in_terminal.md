### Post by iheartubuntu on 2009-07-17
Here is the instructions for ripping. Very easy to do in VLC...

**VLC 0.9.6 and newer**

   1. Open DVD. Go to the Media menu and choose Convert/Save. Then pick the Disc tab and adjust the Starting Position to the proper Title/Chapter. (You can find the proper Title/Chapter by viewing the disc with Media/Open Disc first.) Click Convert/Save.

   2. Enter file name and start ripping. Check File and enter a file name ending in .mpg. Click Save. On a P4 - 2.6 ghtz dual core system with 3GB ram it only took a couple of minutes!
[B]
OPTIONAL[/B]

   3. If it didnt rip the correct video, open a program called OGMRip, click "Load" to load the dvd into the program. Note which title it chooses for the video stream. This is what you will use up above in step one.

Installing OGMRip...

[http://www.getdeb.net/app.php?name=OGMRip](http://www.getdeb.net/app.php?name=OGMRip)
