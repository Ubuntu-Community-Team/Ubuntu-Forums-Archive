### Post by fermulator on 2009-09-26
With the mplayer-plugin in Firefox, sometimes the cache doesn't completely finish, you're not not able to "Save As" the file that is cached.

I've always wondered where the heck it stored the temporary cache file (because it has to be somewhere on disk).  I checked ~/.firefox, ~/.mplayer, etc.

Turns out, it's simply: /tmp/mplayer****!

Enjoy.
