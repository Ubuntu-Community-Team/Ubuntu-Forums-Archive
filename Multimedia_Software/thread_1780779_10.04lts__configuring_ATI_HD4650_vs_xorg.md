You should try using this Firefox Add-on:

[https://addons.mozilla.org/en-US/firefox/addon/flashvideoreplacer/#install-beta](https://addons.mozilla.org/en-US/firefox/addon/flashvideoreplacer/#install-beta)

It is made by Lovinglinux, an Ubuntu member & big contributor to our community.

If you do a search you will find a thread on the add-on in this sub-forum.

[edit:] I did the search for you:

[http://ubuntuforums.org/showthread.php?t=1487327&highlight=flash+video+replacer](http://ubuntuforums.org/showthread.php?t=1487327&highlight=flash+video+replacer)
