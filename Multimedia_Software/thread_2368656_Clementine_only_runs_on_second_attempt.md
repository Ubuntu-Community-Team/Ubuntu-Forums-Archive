### Post by mc4man on 2017-08-15
By default clem will open unactivated to a status icon (indicator). If that's not available in cinnamon  then obviously you wouldn't see it.
The 2nd attempt activates the already open clem
To change, in clem > Tools > Preferences >  Behavior > uncheck 1st. item, 'show tray icon', click Apply
