### Post by Yeti can't ski on 2008-12-01
I installed and successfully used SopCast with Ubuntu Intrepid Ibex using the following package for Sopcast 0.4.0: 

[http://cid-45806b8a57d785f7.skydrive....0-2|_i386.deb](http://cid-45806b8a57d785f7.skydrive....0-2|_i386.deb)

It was just a matter of downloading, double-clicking the file and then basically letting it install itself. It is a bit "scary" to install a package outside from repositories but so far everything is OK. 

The full instructions can be found here: 

[http://ph.ubuntuforums.com/showthread.php?t=929879](http://ph.ubuntuforums.com/showthread.php?t=929879)

In that thread you can also find other debs for earlier versions of Sopcast. 

Once the program is installed, it can be launched from a link automatically created in the Audio/Video Menu. After that, search a channel, double-click it and then "Launch".

Resolution is not fantastic. Bear in mind that Sopcast has a P2P structure just like a torrent. However, it is fine for everything I need (watching a football match of a foreign country once in a while).

If you decide to give it a try, let us know if it works. 

Good luck!
