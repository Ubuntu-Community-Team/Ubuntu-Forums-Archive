### Post by shantiq on 2014-01-03
default package on the[ site]("http://www.google.co.uk/intl/en_uk/earth/download/ge/agree.html")   requires [repackaging]("http://askubuntu.com/questions/366438/how-to-install-google-earth-64bit-in-ubuntu-13-10-ia32-libs-dependency-error")



so i have done this and[ uploaded it]("http://ubuntuone.com/22Fs9fXnhIROgVBp4ykxY4") to my Ubuntuone



simply double-click on deb package   or  cd to folder where you downloaded it    and run   ```
sudo dpkg -i google-earth-stable_current_amd64_Ready_for_13.10.deb
```
