### Post by panopticon on 2010-07-20
I assume you're satisfied with Rhythmbox, and I am not going to persuade you to change to an alternative media player. 

However, Banshee has tag editing which should, if the right option is checked, write your changes to the music files themselves. 

You might want to try that, and then re-load your library in Rhythmbox to see if it reads the tags the way you edited them through Banshee.

It's best done to edit your tags on Linux, and play them on Linux. WMP reads tags differently and sorts files differently.
