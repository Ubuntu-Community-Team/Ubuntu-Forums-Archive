### Post by Erik1984 on 2014-02-09
You should check first if your system supports the latest version of Flash for Linux.

```
cat /proc/cpuinfo | grep -i sse2
```

If the output of the above command is empty your processor is not capable of running the most recent version of Flash for Linux.
