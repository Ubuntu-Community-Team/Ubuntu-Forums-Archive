### Post by ajgreeny on 2012-01-13
I suggest two things.

[LIST=1]
[*]Enable the stable firefox ppa repository for 10.04 which will mean you run the latest Firefox.  [https://launchpad.net/~mozillateam/+archive/firefox-stable](https://launchpad.net/~mozillateam/+archive/firefox-stable)
[*]Install the flash-aid add-on for firefox which will automatically get the latest flash for you, install it, andmake sure there are no confklicting packages still on your system.
[/LIST]
Bear in mind that flash for Linux is still not going to be as good as it is on Windows, though it is improving with time.  HTML5 is probably the answer in the longer term, but for now flash is something we all have to manage with as best we can.
