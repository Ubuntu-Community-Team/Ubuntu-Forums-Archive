### Post by SeijiSensei on 2020-07-14
If you're okay with an animated GIF instead of an MP4, you can use the GIMP.

Open the initial image in the series, then use Open as Layers and select the remaining images.  Use Export and give the output file a .gif extension.  Choose the "Animation" option when prompted and set a reasonable duration like 3000-5000 ms.  If you want to change the order of presentation, you can open the Layers tool (Ctrl-L) to reorder the images. 

It's possible to convert a set of images to an MP4 with ffmpeg. [https://hamelot.io/visualization/using-ffmpeg-to-convert-a-set-of-images-into-a-video/](https://hamelot.io/visualization/using-ffmpeg-to-convert-a-set-of-images-into-a-video/)

You can also convert an animated GIF to an MP4 with ffmpeg, or use one of the various [online tools]("https://www.google.com/search?q=convert+animated+gif+to+mp4").
