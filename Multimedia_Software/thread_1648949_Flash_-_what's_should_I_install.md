### Post by cchhrriiss121212 on 2010-12-19
>  Do they do everything Adobe-Flash does without any headaches?
No.

The best option is just to install flash then use the flash-block or noscript addons with firefox if it bothers you. If you are new then I recommend installing ubuntu-restricted-extras which has flash as well as a great deal of commonly used codecs (mp3 etc.), it will be updated along with the rest of your system.

Don't expect the same level of performance on Ubuntu that you had on Windows though, flash can use up to 100% of mid to low end cpus.
