Check this out. Ubuntu can now support Netflix Streaming Movie Downloads -- uh, sort of. The way it works is that **it doesn't work**. Instead of running it on your workstation, you run it from this Roku little box:

[http://blog.wired.com/gadgets/2008/05/review-roku-net.html](http://blog.wired.com/gadgets/2008/05/review-roku-net.html)

Now, if someone could just hook this little device so that one could split and capture the video output, or send it through the home network back to another Ubuntu workstation in a window, then that would be something really neat as well -- especially if you could control the unit from Ubuntu and see it back on Ubuntu.
