### Post by paechan on 2009-05-15
I myself am having some resolution issues in jaunty, but I made some headway by setting the right virtual resolution in xorg.conf. I still don't know how to set modes, but this helped me get closer to the right resolution. Edit the file xorg.conf, from a terminal window:

```
sudo gedit /etc/X11/xorg.conf
```

Find the "Display" section and edit the last part so that it contains the virtual subsection:

```
Section "Screen"
	Identifier	"Default Screen"
	Monitor		"Configured Monitor"
	Device		"Configured Video Device"
	SubSection "Display"
		Virtual	1440 900
	EndSubSection
EndSection
```

You can try giving this a go, or else you can try adding that xrandr line in Preferences > Startup Applications.

I would very much like to see your xorg.conf btw.

Good luck
