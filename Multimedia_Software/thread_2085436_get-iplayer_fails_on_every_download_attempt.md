### Post by ron999 on 2012-12-16
> **stobio said:**
> Could any one highlight where I am going wrong please?
How do I update flvstreamer please? Is this the right file or should I be trying to update rmtpdump?
Hi
Looks as though you're using get_iplayer from PPA, that's good.
It uses RTMPDump instead of Flvstreamer.
Latest version is RTMPDump-v2.4. No need to update it.

For BBC local radio sometimes _**MPlayer**_ is needed too.
So make sure it's installed...
```
sudo apt-get install mplayer
```

Then, same as before...
```
get_iplayer --type=radio --pid=p011d2td
```

(If you've a part-downloaded show, delete it first).
;)
