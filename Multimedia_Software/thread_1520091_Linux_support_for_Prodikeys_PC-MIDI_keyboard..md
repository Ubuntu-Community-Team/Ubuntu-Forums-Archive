### Post by MIJ-VI on 2010-06-29
Back in 2007 mr.thraz started a thread titled...

**need help with a midi problem im investigating.**
[http://ubuntuforums.org/showthread.php?t=492069&highlight=prodikeys](http://ubuntuforums.org/showthread.php?t=492069&highlight=prodikeys)

...in which he was trying to track down Linux support for a type of USB-based PC/MIDI keyboard hybrid from Creative.

[IMG]http://images.prodikeys.com/products/images/product_prodikeyspcmidiusb.jpg[/IMG]
[http://www.prodikeys.com/products/prodikeys_PC-MIDI/specification.asp](http://www.prodikeys.com/products/prodikeys_PC-MIDI/specification.asp)

Exhibiting the same foresight as numerous other PC hardware makers, Creative only supports Windows for use with this device. 

However, thanks to Mr. [FONT=monospace][/FONT]Don Prince <dhprince.devel@yahoo.co.uk>...

Kernel v2.6.35-rc2 /drivers/hid/hid-prodikeys.c
[http://www.linuxhq.com/kernel/v2.6/35-rc2/drivers/hid/hid-prodikeys.c](http://www.linuxhq.com/kernel/v2.6/35-rc2/drivers/hid/hid-prodikeys.c)
