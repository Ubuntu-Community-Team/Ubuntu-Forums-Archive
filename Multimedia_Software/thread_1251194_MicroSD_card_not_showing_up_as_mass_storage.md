### Post by bobbycheetah on 2009-08-31
Issue is resolved. Found other threads referring to a download available at pinstack.com - called BB Storage Activation new.rar   I had to register at the site to get the download.  AFAIK, the file is from RIM and don't know why it is not available more readily from other sources.  Apparently, refurbed, unlocked, hacked, etc. BB's have their USB mass storage connectivity disabled - somewhere in a lower level, and this file re-enables it.  Hope this helps others out there.
