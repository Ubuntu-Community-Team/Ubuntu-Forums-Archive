You should check that the HD 5450 allows you to connect both HDMI and Displayport at the same time.

I have an older HD 4290 which has VGA, HDMI, and DVI -- and it does not allow BOTH the HDMI and DVI ports to be connected, instead, one of them has to be VGA.
