### Post by lemon-cluster on 2014-02-19
I run the Mediahuman Youtube to MP3 application on 12.04.  It makes a neat job.       
In the main the files generated play well on media player and on my ipod clone but not on my Sandisk MP3 player which locks up. 

When I look at the MP3 file properties, it shows all those generated by Mediahuman as having a bitrate of 32 Kb/s.

I believe that the bitrate (Tag?) is incorrect, although the only device that is upset is my Sandisk Clip Zip player.

### Post by tgalati4 on 2014-02-19
Install *easytag* and fix the ID3 tags.  Open a terminal:

```
sudo apt-get install easytag
```
