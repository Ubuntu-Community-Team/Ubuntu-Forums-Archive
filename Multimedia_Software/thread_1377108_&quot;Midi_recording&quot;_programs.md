### Post by markbuntu on 2010-01-10
You can find the ubuntustudio meta packages easier with Synaptic package manager. it is a big download and you need to set up jack which can be tricky. 

You might want to just get rosegarden which can bring you some more immediate results.

There are a lot of how tos and wikis around about setting up and using ubuntustudio and jack. You should look at some of them before doing anything to avoid confusion and frustration.
