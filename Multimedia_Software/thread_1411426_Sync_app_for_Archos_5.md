I am using  Rhythmbox now with the Archos5 and it picked up the device with no issues.  You need to have  Rhythmbox already open before connecting the device and have the plugin enalbed for MTP portable players.. I checked off both, one was for MTP and other for IPOD just in case.

I am syncing my podcast with gpodder. You just select podcast prefereces and type of device, filesystem based mp3 player, it is in the dropdown.. Then you select a folder to sync with. I just created a folder called podcast in the music folder on the device.

The only thing is you cannot have  Rhythmbox and gpodder open at the same time of course.  

The only function so far  I was not able to do is have playlist in  Rhythmbox  sync with archos, so at this time I am just managing the playlist on the device itself.

I believe you can also sync podcast from  Rhythmbox  to the device; but I like the features of gpodder much better.  The podcast in  Rhythmbox show up as default folders in your home music folder.

Basically just:
- checked all my playlists under the auto-sync list  playlist tab

- Removed all auto-conversion rules

-  Device config tab; Sync tracks to: \Music\<Album  Artist>\<Album>\<Track:2> <Title>
check "Copy  Playlists", under options check "Library Playlists"; Destination  directory: \Playlists\
check "Use only the first Genre... "
check  "Use only the first value...  "
