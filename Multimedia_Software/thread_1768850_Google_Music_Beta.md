**Chromium and the window manager**

The window manager is  responsible for handling the user's interaction with multiple client  windows. It does this in a manner similar to that of other X window  managers, by controlling window placement, assigning the input focus,  and exposing hotkeys that exist outside the scope of a single browser  window. Parts of the ICCCM (Inter-Client Communication Conventions  Manual) and EWHM (Extended Window Manager Hints) specifications are used  for communication between clients and the window manager where  possible.

The window manager also uses the XComposite extension to redirect client  windows to offscreen pixmaps so that it can draw a final, composited  image incorporating their contents itself. This lets windows be  transformed and blended together. The window manager contains a  compositor that animates these windows and renders them via OpenGL or  OpenGL|ES.

[**Google Music Manager Finally Launches On Linux**]("http://www.omgubuntu.co.uk/?p=17888")
> **Google have announced the release of Google Music Manager for Linux. **

[[IMG]http://cdn.omgubuntu.co.uk/wp-content/uploads/2011/07/Selection_002-500x385.jpg[/IMG]]("http://cdn.omgubuntu.co.uk/wp-content/uploads/2011/07/Selection_002.jpg")
The Various Steps of Google Music Manager

[Google Music Beta]("http://music.google.com/"),  which was launched back in May, lets users upload as many as 20,000  tracks for free access and streaming through the web and mobile devices &#8211;  wherever they are in the world.
[[IMG]http://cdn.omgubuntu.co.uk/wp-content/uploads/2011/07/Pass-Out-Tinie-Tempah-Music-Beta-Google-Chrome_009-500x384.png[/IMG]]("http://cdn.omgubuntu.co.uk/wp-content/uploads/2011/07/Pass-Out-Tinie-Tempah-Music-Beta-Google-Chrome_009.png")
 
Music is cached for offline play on both the desktop and mobile devices.
 
At the time of Music Beta&#8217;s launch Google only provided Windows and  Mac version of &#8216;Google Music Manager&#8217; &#8211; their desktop client for adding  and seamlessly syncing your music folder with Google Music Beta. With  the release of Google Music Manager for Linux, everyone* is now free to  take advantage of the cloud-music-storage service.
 
Those of you already signed up/using the service can grab the linux  installer &#8211; provided as both 32bit and 64bit .deb packages &#8211; by hitting  the &#8216;Add Music&#8217; button to the top of the player window.
 
***Download** | [Google Music Manager for Linux]("http://music.google.com/") (32bit, 64bit .deb)*

**Although Google Music Beta is  &#8216;restricted&#8217; to US users, requesting an invite form your Google account  via a US proxy &#8216;gets you in&#8217;. Just head to &#8216;music.google.com&#8217; via a US  proxy and request an invite. You only need to use the proxy to request  the invite; all other steps, such as using the service, work wherever  you are.*
