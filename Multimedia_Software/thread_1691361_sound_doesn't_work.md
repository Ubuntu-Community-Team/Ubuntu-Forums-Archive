A few things:

1. Does it work in any other apps?
2. If so, is VLC on Mute?
3. If not, is VLC's volume turned down?
4. If not, if you go to your sound icon in the top right, click on it, go to Sound Preferences, click on Applications tab, is the volume on for vlc there?
5. If so, in VLC does an MP3 file play?
6. If not, go to Tools -> Preferences -> Audio - is audio enabled?
7. If so, what is the Output set to?
