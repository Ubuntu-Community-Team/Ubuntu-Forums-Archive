### Post by meteorrock on 2013-01-19
Just use the terminal. I found mine in the dash, just use the search  function. You can drag and drop it into your unity launcher if you are using the latest unity desktop for next time. 

After you open up your terminal, use these commands.

```
 sudo apt-get update && sudo apt-get remove gnome-screensaver && sudo apt-get 
install XScreenSaver && sudo apt-get install electric sheep 
```After dragging and dropping in that code in your terminal, use the search function and find your newly installed xscreensaver app with the electric sheep screensaver in that app.
