### Post by Dennis N on 2013-05-18
The file type is set in the Export dialog (reached from File > Export, or File > Export Selection). Once a file type is selected (and quality if applicable), Audacity keeps your settings until you change them to something else. That's it.
 
Reference:
[http://manual.audacityteam.org/man/File_Export_Dialog](http://manual.audacityteam.org/man/File_Export_Dialog)
