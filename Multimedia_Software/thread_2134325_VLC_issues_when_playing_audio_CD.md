However, when switching between tracks (either by clicking the skip buttons or by allowing the current track to end), VLC:

[LIST]
[*]displays the correct filename in the window title bar 
[*]changes the play/pause button to a play button which does not work 
[*]lists "--" as the length of the track 
[*]is able to be exited out of, but the icon in the bar at the top of screen still exists and cannot be gotten rid of

[/LIST]

If the track changes because the current track has ended and the next track is being moved to, the change is usually (but not always) successful.  If the track changes due to my hitting a skip button, the change is usually (but not always) un-successful.
I can play CDs fine by using Rhythmbox, but I am hoping to make VLC usable, since that is what pops up automatically when I insert a CD.
