### Post by tgalati4 on 2009-08-05
Some newer CD players can play mp3 files.  Just burn one disk of the small files and test it.

Instead of using VLC, try using moc to listen to audio books:

sudo apt-get install moc
Go to the directory where the files are located:
mocp

If you combine the files, then it will be difficult to fast forward through the chapters.  Some mp3 players can't handle large mp3 files.  Best to leave them as small files.
