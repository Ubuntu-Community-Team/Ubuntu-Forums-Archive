### Post by HappyFeet on 2009-08-07
OK. Go to synaptic and completely remove flash. Then go back to the adobe link I gave you and download the .tar.gz flash file. Right click it and select "extract here". Copy the **libflashplayer.so** file. Go to your home folder and do Ctrl-H to show the hidden directories. Go to the .mozilla folder and open it. Right click anywhere in the folder and dreate new directory. Name it **plugins**. Open plugins folder and paste the libflashplayer.so file.

Next, open a terminal and do:
```
sudo apt-get install mozilla-mplayer sun-java6-plugin
```

Report back what happens.
