### Post by SeijiSensei on 2011-08-22
If it's a well-made animated gif, this is going to be difficult.  Open the gif in a graphics editor like GIMP, and you'll see the frames as separate layers.  Animated gifs often use a differencing algorithm to remove all the redundant pixels in each frame.  That means the intermediate frames won't usually be complete images.  

If all the frames are complete, you can save the individual frames and see if a video animator will combine those.
