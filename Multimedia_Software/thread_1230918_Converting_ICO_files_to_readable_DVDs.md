### Post by mc4man on 2009-08-04
> files on my Jaunty Jalopy that are *.ico

While I don't particularly like or use brasero I can't see any way it would create a .ico from anything ( .ico = microsoft icon images
even if you tried it would warn you

What it would make is a .iso ( no matter what ext. if you 'force' one

To make a playable dvd from a .iso you need your burning app to burn to a dvd video format, not as a data dvd

(( if they do have a .ico ext. change to .iso and see if a player will play (vlc, totem, ect

edit
Just to see I forced brasero to use a .ico ext, then when done renamed to .iso, plays fine
A .iso will have a disk icon, a .ico a picture icon
