### Post by Auslegung on 2008-04-29
Amarok is really good, and even though it's KDE it'll still work on GNOME.  You'll need to reference this post to get Amarok properly configured to work with an iPod: <[http://ubuntuforums.org/showthread.php?t=431217&highlight=unmount+ipod+amarok+post-disconnect+command]("http://ubuntuforums.org/showthread.php?t=431217&highlight=unmount+ipod+amarok+post-disconnect+command")>.  Listen media player looks promising but my Amarok is so tweaked out that I don't want to spend the time to configure something else the same way.  Exaile is also nice, pretty much Amarok for Gnome.  All three I've mentioned handle podcasts well, but my vote is Amarok, even though I ocasionally want to punt it.
