### Post by JBAlaska on 2009-11-24
Try [K9Copy](https://help.ubuntu.com/community/K9Copy) it's in the repo's so just start synaptic package manager and put k9copy in the search bar.

Or do this in a terminal:
```
sudo apt-get install k9copy
```


If your using Gnome it will install the KDE libs as well, but imho there are several KDE apps that are superior to Gnome ones.
Krusader, dolphin, K3B etc..
