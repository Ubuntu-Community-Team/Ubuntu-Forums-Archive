### Post by manimal347 on 2008-10-12
I hate to be elitist, but why are you using Limewire? It's really not a good P2P client, and as a Linux user, you have so many better options that are open source and created by approachable people.

FrostWire: Limewire source code changed so you get the search features of LimeWire Pro.

Amule: Edonky2000 Network- obnoxiously slow and unreliable like Limewire, but lots of rarities. Has a better rep. in general.

Bittorent: Try Deluge, Transmission, Ktorrent, or Utorrent under Wine.

SoulSeek: Nicotine+

Ares: It runs under Wine quite well.

Pan: If you have access to usenet binaries through your ISP
