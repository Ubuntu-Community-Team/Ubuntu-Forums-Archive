### Post by shantiq on 2018-01-07
to view metadata use mediainfo   or exiftool    both in repository
then go into folder

and enter

```
mediainfo *
```  or```
 exiftool *
```



as regards replaygain   there is [http://manpages.ubuntu.com/manpages/precise/man1/replaygain.1.html](http://manpages.ubuntu.com/manpages/precise/man1/replaygain.1.html)
never used it myself but it is there
