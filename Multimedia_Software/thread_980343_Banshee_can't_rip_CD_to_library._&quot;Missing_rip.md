### Post by cariboo on 2008-11-12
Which option were you using to rip cd's. Go to Edit-->Preferences-->Audio CD and select how you want the ripped version saved. If you use MP#(Lame encodedr) make sure you have lame installed. To see if lame is installed in a termihnal type:

```
locate lame
```

If you don't get any output, you need to install it.
