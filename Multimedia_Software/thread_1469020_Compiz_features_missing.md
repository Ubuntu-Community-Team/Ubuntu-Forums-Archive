### Post by Dkkline on 2010-05-20
For all you google'ers here is a quick way to get all the extra plugin's for compiz:

1.) open a terminal ("Applications -> Accessories -> Terminal")
2.) type: ```
sudo apt-get install compiz-fusion-plugins-extra
```.
3.) open compiz settings ("System -> Preferences -> CompizConfig Settings Manager") and they should all be there!

Have fun!
