### Post by mcduck on 2009-08-10
No, only way to run Shockwave in Linux is to install both Shockwave and windows version of some web browser with Wine.

Adobe has told that they have no intentions to ever port Shockwave to Linux.

(besides, it's becoming more and more obsolete anyway, as all the really important features it has have been ported to Flash. And the ones that haven't been ported yet (like hardware-accelerated 3D graphics) are already under development)
