### Post by gradinaruvasile on 2011-04-09
A new version of Skype (2.2.0.25, still beta) for Linux was released recently (april 6).
It has support for call waiting/holding and improved audio/video. And a feature called Skype Access (connect to paid wifi hotspots and pay with skype credits) that i dont need (it involves paying you know...). Anyway it is detailed in the release notes.

Download (the Ubuntu 10.04 option should work for all versions):

[http://www.skype.com/intl/en-us/get-skype/on-your-computer/linux/](http://www.skype.com/intl/en-us/get-skype/on-your-computer/linux/)

If the .deb wont work, the static version is an all-inclusive unpack-and-run build, should work on any recent(ish) Linux if you have older Linux or the debs wont install/run for some reason.

Release notes:

[http://blogs.skype.com/garage/2011/04/skype_22_beta_for_linux_with_s.html](http://blogs.skype.com/garage/2011/04/skype_22_beta_for_linux_with_s.html)

BTW i still have to use the LD_PRELOAD workaround for my gspca-based webcam.
