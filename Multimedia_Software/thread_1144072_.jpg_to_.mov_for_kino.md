### Post by FakeOutdoorsman on 2009-04-30
FFmpeg can do this, but it is command-line:
```
ffmpeg -i images%d.jpg -r ntsc output.dv
```
The *%d* refers to the numerical sequence of your input images: images001.jpg, images002.jpg, etc.  The *-r* is your frame rate and can be: pal, ntsc, or an interger. You can output to several formats, but I assume Kino prefers DV. If you want a non-command-line solution, perhaps *Stopmotion* can also do this, but I've never used it.
