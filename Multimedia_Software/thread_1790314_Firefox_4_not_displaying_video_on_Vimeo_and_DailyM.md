### Post by lovinglinux on 2011-06-24
Install [Flash-Aid](https://addons.mozilla.org/en-US/firefox/addon/flash-aid/) and run the extension wizard. It will detect installed flash plugins, remove them and install the best option according to your system architecture and version. Additionally, it will apply some tweaks that should improve performance.

### Post by lambart on 2012-11-21
Update: Although the Vimeo flash player was loading with a click on the player (i.e. to tell Flashblock to load the .SWF file), it apparently needed to load further flash crapola from vimeo.com.

Adding player.vimeo.com to my Flashblock whitelist fixed the problem. I realize this won't help the original poster since s/he did not report having Flashblock installed. But hopefully it will help someone else. And if you don't have Flashblock installed, you should! Unrestricted Flash is a security issue not to mention an invitation to obnoxious advertising nightmares.
