The native resolution of both my monitors is 1440x900. 
I'm using Twinview, and that works perfectly, I have compiz on and the cubes function well, pretty much everything I want for normal desktop use works perfectly. 

I setup wine, and used it to install a few games, when the games initially ran, they were stretched across both monitors, I fixed this by adding the "metamode" option to my xorg.conf

However, now when I run a game, it runs on my left monitor (at the right resolution,) but the right one just gets completely disabled. When I exit the game I need to restart my x server to get everything back to normal. I would like to be able to see Mangler, and music, and Pidgin etc. on my spare monitor while being in game. 

Basically what I want to know is; if there is a way to make games run at 1440x900 on one monitor, while the other monitor runs at 1440x900 and stays active?

