### Post by holotropik on 2009-01-18
Hi,

I am new to Ubuntu. I wanted something different than Windows for a change and to learn something new. So I set myself a project to build a studio laptop with Linux.

Anyway.

I have spent the last few months building my Studio laptop into something i can use to record audio and other various studio tasks.

I thought i might share some knowledge i have found through this thread if thats OK?

In summary, after some tweaking I got Ubuntu 8.10 to acknowledge and use my Korg Zero8 mixer with internal firewire audio card in this mixer. I am stoked that this worked considering I was lead to believe that unless the audio card was on the "supported" list it would not work.

I was able to use freebob driver to connect the audio card/mixer and have it run fine.

Of course I had to use the rt kernel -16 instead of the generic kernel in Ubuntu 8.10. But, now I have to work out the nvidia driver problem and I will be sooo happy!

I will post more detail and answer any questions. This forum has helped me to sort out this project so I will put back as much info as I can ;)
