### Post by gpeck157 on 2010-05-15
I was having the same problem on my new installation of 10.04 but I solved the problem by going into the Synaptic Package Manager. I did a "Quick Search" on Internet Radio. One of the packages that came up was "rhrythmbox-radio-browser". I installed this then did a complete shutdown. When I boot up again "Radio" in Rhythmbox now works.

Good luck...
