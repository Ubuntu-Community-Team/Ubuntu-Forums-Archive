### Post by mike.headon on 2010-07-10
Please also see this thread:-
[http://ubuntuforums.org/showthread.php?t=1359632&highlight=record+streaming+audio](http://ubuntuforums.org/showthread.php?t=1359632&highlight=record+streaming+audio)

See the post from "lessmemorelove".
The important components are Audacity and Pulse Audio Volume Control (Pavucontrol). You will also need Lame if you want to save as MP3.

This worked fine for me.
Note that you record as a WAV file (beware - huge) and convert to MP3 when you save it, then discard the WAV. 
My next project will be to discover if you can record direct to MP3:o
