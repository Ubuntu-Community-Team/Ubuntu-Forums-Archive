Hi
If you're using Firefox you need the plugin [SIZE="4"]**gecko-mediaplayer**[/SIZE]. It's in the Ubuntu repository.

If you already have another plugin installed (such as mozilla-plugin-vlc) then uninstall it first.
Look in Firefox > Tools > Add-ons > Plugins to see what's installed.

When you install the gecko-mediaplayer it installs plugins for DivX, Quicktime, RealPlayer and Windows Media Player.
