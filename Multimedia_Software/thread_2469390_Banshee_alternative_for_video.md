### Post by TheFu on 2021-11-28
**smplayer** is a front-end to **mpv**. mpv is amazing.  I've never used smplayer.  No clue about tagging.  That's dependent on the media file header ... I use mkv for media files, so **mkvpropedit** can add tags.  It really depends on the video content whether tagging is useful.  Movies, TV Shows have tags that are different from high-school sporting events or home movies. Just be certain that the tagging tool doesn't lock up all the data in some non-standard DB, never to be released.  It is best to get any tags actually stored in the file header whenever possible.
