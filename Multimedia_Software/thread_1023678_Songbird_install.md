### Post by caro on 2008-12-29
If you downloaded Songbird to your desktop, you should be able to double click on the tar.gz to launch and extract the files.  Save Songbird files to your home directory and then click on the Songbird file to launch the app.  You can create a launcher in the panel to make it easy to launch.  Good luck.
