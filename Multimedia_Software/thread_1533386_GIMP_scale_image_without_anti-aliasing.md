### Post by higashi on 2010-07-17
I want to make an 8-bit looking image that has very large pixels. My solution was to make a 50x50 image, zoom in, draw it with the 1x1 pixel pencil tool, and then scale it to whatever size i want.
Everything went well except, when i try to scale it the anti-aliasing kicks in and blurs everything. Is there a way that i can disable anti-aliasing?

---

### Post by tudor117 on 2010-07-17
Image -> Scale Image -> Quality: Interpolation -> None
