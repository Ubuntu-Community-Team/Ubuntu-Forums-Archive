### Post by johnsmoke on 2012-07-22
Never transferred video to my computer before, I thought I'd run into more problems, but none at all. Kino is such a great easy to use program! I captured my video, then imported the AVI to Devede which is also a great piece of software, super easy to use - only thing it didn't have that I would want is the ability to add custom chapters, at specific times, at the moment it only allows chap marks every 5 minutes, which isn't bad either..... Anyway, this is GREAT software for any of you thinking of doing video editing. My first time using firewire and it was great. Now I just need to learn Openshot to try and sink in my external audio to the video.
