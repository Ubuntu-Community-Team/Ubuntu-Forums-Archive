### Post by SeijiSensei on 2013-01-14
Are these commercial DVDs?  Did you install kubuntu-restricted-extras to give K3b the extra bits of software it needs to decode them?

Some recent DVDs use additional copy-protection measures besides normal CSS.  (Disney is a notorious example.)  Some of these DVDs can be unreadable on Linux.
