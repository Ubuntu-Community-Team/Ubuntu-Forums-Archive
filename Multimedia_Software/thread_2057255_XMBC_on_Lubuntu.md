What lubuntu ? like 11.10 
If 11.10 use this ppa
```

sudo add-apt-repository ppa:nathan-renniewaldock/xbmc-stable

sudo apt-get update

sudo apt-get install xbmc

```



If 12.04 and above 
```
sudo apt-get install xbmc
```
