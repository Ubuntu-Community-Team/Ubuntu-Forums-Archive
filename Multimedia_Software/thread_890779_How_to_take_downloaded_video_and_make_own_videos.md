### Post by eggdeng on 2008-08-15
There are lots of variables here as regards codecs and formats and the learning curve will be fairly steep.
A good place to start might be with Avidemux
```
sudo apt-get install avidemux
```
The wiki is at [http://www.avidemux.org/admWiki/index.php?title=Main_Page]("http://www.avidemux.org/admWiki/index.php?title=Main_Page")
