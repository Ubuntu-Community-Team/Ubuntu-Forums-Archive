### Post by nelwin on 2009-10-02
Try EasyTag. I've only been using it for a few days on my Ubuntu 9.04 64-bit system but it's been great. You can install it from the terminal with the following, minus quotes:
"sudo apt-get install easytag"
Check out their site @ [http://easytag.sourceforge.net/](http://easytag.sourceforge.net/)
Good luck! :)
