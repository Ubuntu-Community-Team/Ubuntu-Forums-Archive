### Post by Rodney9 on 2012-02-15
Vkeybd Virtual MIDI Keyboard
This program allows you to use the Linux AWE32 driver, OSS MIDI sequencer,
or ALSA MIDI sequencer and provides "virtual" keyboard (the musical type),
which can be "played" using the keyboard or mouse under X11.

Available in the Software Center.

You can look at this page for other types - 

[http://soft.udm4.com/downloading/ubuntu_keyboard_sound_effect/](http://soft.udm4.com/downloading/ubuntu_keyboard_sound_effect/)
