### Post by ajgreeny on 2012-02-12
It's needed for both, or at least, the codecs are needed, but not necessarily the w32codecs package.   You may find this the easiest way to get these codecs, but it is not totally necessary to use that package.

If you install the appropriate ***ubuntu-restricted-extras** package, which  adds many of the **gstreamer** packages, you can get just about the same effect, in codec terms.
