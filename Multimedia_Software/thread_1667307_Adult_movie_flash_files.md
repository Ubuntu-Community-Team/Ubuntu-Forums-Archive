### Post by cfhowlett on 2011-01-14
When flash videos are played in the browser, a .flv file is created in /tmp.  Typically the file is named FLASHAABBCCdd or something similar but always with a "FLASH" prefix.  This /tmp file is retained so long as the browser tab stays open.  Once the file is fully downloaded, it can be renamed, copied, moved or manipulated like any other movie file.  

So far as renaming a GROUP of files, there's a way.  Google is your friend.

I'm not saying HOW I know, but I know.
