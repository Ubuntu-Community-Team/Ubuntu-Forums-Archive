### Post by akand074 on 2010-08-15
You could just right click on a .mp3 file and click properties. Then go to the "Open With" tab and choose which program it opens with. Now all .mp3 files will open with that program. Another way to do it (maybe the easiest way with a good GUI) is by using Ubuntu Tweak. Its just a small program that allows you to change those kinds of settings real easy. 

```
sudo add-apt-repository ppa:tualatrix/ppa
```
```
sudo apt-get update
```
```
sudo apt-get install ubuntu-tweak
```

The first line adds ubuntu tweak to your repositories, the second line updates your repositories so that it recognizes that you added something, and the last line installs ubuntu-tweak for you, from there you could go ahead and play around it shows up in the "System Tools" menu from the main menu. 

I hope that solves your problem
