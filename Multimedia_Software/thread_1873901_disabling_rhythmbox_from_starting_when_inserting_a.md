### Post by 741Baus on 2011-11-02
Hi Baldrick_NZ 
              Try Home-Edit-Preferences-Media-CD- and change to do nothing I have this marked to all media that is inserted .
                    Peter

### Post by robert shearer on 2011-11-02
and for 11.10...
System settings>Removable media... select for desired cd/dvd actions **or**, for a catch-all option, check the box for "Never prompt or start programs on media insertion"
