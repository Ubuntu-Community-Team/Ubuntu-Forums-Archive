### Post by markbuntu on 2010-04-19
This might be possible using the open source drivers but if you are trying to use the proprietary drivers there is a good chance it will not. This is because both the nvidia and ati proprietary drivers replace some of the X libs with proprietary versions that are incompatible with other drivers.

So, what cards are you trying to do this with?
