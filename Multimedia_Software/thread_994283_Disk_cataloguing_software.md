### Post by cariboo on 2008-11-26
There so many disk cataloging programs, that easiest way is to go to System-->Administration-->Synaptic Package Manager and click the search button and enter **disk catalog**

This will list almost all of the disk cataloging programs available.

I use Griffith to catalog my dvds.
