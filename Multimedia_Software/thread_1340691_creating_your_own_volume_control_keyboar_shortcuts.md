### Post by donato roque on 2009-11-29
I use Ubuntu 9.10 karmic and I have my volume controls configured to use keyboard shortcuts.  I went to System-->Preferences-->Keyboard Shortcuts.  In the window pop up, select the row item you want to edit and type the keyboard shortcut you prefer.  Volume controls should be under Sound.
