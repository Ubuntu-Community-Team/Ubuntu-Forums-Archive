### Post by cybrsaylr on 2014-10-12
The Opera Forums came to life helping find the solution here.

Seems the fault lies with Opera 12.16 which distorts color, when running 12.04LTS. However Opera 12.16 runs fine with correct color, when running 14.04LTS. Opera Forums suggest using beta Opera 25 but that won't play any videos with either 12.04 or 14.04! You need to install  the pepper flash plug-in (pepperflashplugin-nonfree), which was only available in the Ubuntu 14.04 repositories but not 12.04 repositories! Once pepperflashplugin-nonfree was installed beta Opera 25 plays all videos fine with correct color with 14.04LTS.  The beta Opera 25 version is snappier and faster than Opera 12.16 but bookmarks are completely changed around that I'm still attempting to figure out what they did.
