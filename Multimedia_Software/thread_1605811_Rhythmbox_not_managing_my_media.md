### Post by Mr Bean on 2010-10-28
I find that Rhythmbox only imports files with id3 tags. So if a file has no tag for "artist" you can't get it to show up in the library however hard you try. 

I found a program called "Ex Falso", available in the Ubuntu repositories, that does a decent job at editing id3 tags. With the artist tag defined, Rhythmbox imports music no problem. 

"If I change a song name or artist the file name doesn't change" I'm not sure that Rhythmbox ever had that functionality. I think it only changes the tags. Ex Falso can also rename files if you don't mind using a separate program.
