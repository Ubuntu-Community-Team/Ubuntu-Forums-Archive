### Post by labinnsw on 2009-06-05
If the volume control is not in the notification area, add it by:
right click on a panel and select "Add to panel"
Scroll down to Volume control.
Select it then select "Add"
Now double click the Volume control icon to activate the Volume control.

You select a device by clicking File >> Change Device

Configurations are done by clicking Edit >> Preferences

This is based on Ubuntu 8.04, hopefully it is still true for 9.04
