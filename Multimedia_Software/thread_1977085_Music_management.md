### Post by Enigmapond on 2012-05-09
If you are really running 6.10 this suggestion won't be of any help however, if you are up to date or at least to 10.04 then try Guayadeque. Very versatile player. Great with portable devices and with very large libraries. It's really the best Ubuntu has to offer IMO...

```
sudo add-apt-repository ppa:anonbeat/guayadeque && sudo apt-get update && sudo apt-get install guayadeque-svn
```
