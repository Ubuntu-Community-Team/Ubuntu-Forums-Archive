### Post by Maheriano on 2009-10-29
Tools - Options - Interface - Start
Click OK when the box comes up, the advanced interface will automatically be selected for you and then it'll ask you to restart Vuze. Do that and it'll come back up with the new interface to search right inside Vuze.

To avoid having to do this every time you open it, either go to the Vuze website and download version 4.2 or upgrade Ubuntu to 9.1 and it'll automatically come down as a package update.

```
From [http://mydailytech.com/post/old-interface-in-vuze/](http://mydailytech.com/post/old-interface-in-vuze/)

# Go to the Tools menu
# Click on Options
# A new window will appear
# Look in the item list on the left for &#8220;Interface&#8221; and click on the plus arrow beside it
# Click on &#8220;Start&#8221; in the list that appeared
# Click on the &#8220;Show&#8221; button beside &#8220;Display Vuze UI Chooser&#8221;
```
