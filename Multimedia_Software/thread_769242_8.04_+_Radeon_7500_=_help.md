### Post by Rocket2DMn on 2008-04-26
I'm not really sure what that means, you may want to try reinstalling the backend part of compiz (it may prompt you to uninstall more parts of compiz, it should be safe to do so, it just needs to be reinstalled, too).  First disable compiz by enabling metacity
```
metacity --replace
```
Now uninstall:
```
sudo apt-get remove --purge compizconfig-backend-gconf
```
Then reinstall (I will include compiz itself just in case)
```
sudo apt-get install compiz compizconfig-backend-gconf
```
Restart X with CTRL+ALT+BACKSPACE, and now try to load compiz again
```
compiz --replace
```

### Post by Rocket2DMn on 2008-04-27
I'm not sure you are going to be able to get any more FPS than that with compiz since your card is so old.  If you need to do something that is graphics intensive, like play videogames, you will want to disable compiz first by using metacity, then re-enabling compiz when you are done.  I suggest making launchers in your panel to do this so you don't have to type it out every time.
Right click the top panel somewhere and hit Add to Panel and double click Custom Application Launcher.  Name one Compiz and use the command 
```
compiz --replace
```
Click OK.  Now double click again and make one called Metacity and use the command
```
metacity --replace
```
