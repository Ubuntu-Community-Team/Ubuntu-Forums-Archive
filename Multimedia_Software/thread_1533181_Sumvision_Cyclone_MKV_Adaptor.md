### Post by Vectorferret on 2010-07-17
That mkv adaptor will work with any current generation Ubuntu (Ubuntu, Kubuntu or Xubuntu). Ubuntu uses Gnome and Kubuntu uses KDE, so if you have a preference pick that one. To share a folder for the adaptor just right click on it and choose properties and then the share tab (in Ubuntu, I don't personally know how in Kubuntu). Ubuntu will automatically install the necessary libraries to share it.
