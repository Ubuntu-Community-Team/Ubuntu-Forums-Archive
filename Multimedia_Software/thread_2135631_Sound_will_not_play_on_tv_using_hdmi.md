### Post by Mark Phelps on 2013-04-16
When AMD chips are onboard with laptops, typically, the HDMI port is best controlled using their drivers.  If you haven't tried it already, see if Additional Drivers is offering the AMD restricted drivers.  With those installed, you should be able to get integrated audio/visual controls through the HDMI port.
