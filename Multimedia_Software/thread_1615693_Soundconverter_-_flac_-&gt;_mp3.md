### Post by papibe on 2010-11-07
Most mp3 players use ID Tags to sort/play/search etc. ID Tag are stored inside audio files, and usually are embedded into the files when you rip the CD. If not, there are several Tag editors you can use to add Tags to audio files.

My guess is that Sounconverter didn't copy the tags along to the converted files. Maybe that app has an option for that, and you just need to reconvert using it.

If not, you'll need a new software that preserves the Tags. I'm sorry but I can suggest you any because I don't know them that much.

Good Luck!
