### Post by itsr0y on 2009-08-23
For a long time, I could not get fullscreen to work in Flash player on Firefox.  The same problem was described in  [post 615121]("http://ubuntuforums.org/showthread.php?t=615121").  I just found out the solution:

Go to **Preferences**
-> **CompizConfig** Settings Manager
-> **General Options**
-> _Disable_ **Undirect Fullscreen Windows**

Now fullscreen works!
