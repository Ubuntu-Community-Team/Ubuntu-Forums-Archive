### Post by stef2n on 2009-09-12
Hi, everyone!

I am pleased to announce you about a new plugin, Pidgin Embedded Video, which we have developed. This plugin will allow you to watch videos from Youtube and Vimeo directly into the conversation window, without opening any other browser window. This functionality is similar to a feature of Yahoo! Messenger. Check out the page of the project for more details: [http://code.google.com/p/pidgin-embeddedvideo/](http://code.google.com/p/pidgin-embeddedvideo/)

If you like it or if you want to give us some suggestions, we are happy to receive any feedback from you.

Regards,
Stefan
