### Post by JayPeeJay on 2011-09-21
Get Inkscape.  It is a vector graphics program.

Use the pen tool to draw a path, then open the stroke dialog.  You can set either end as an arrowhead (many different kinds).  You can also make the line a dashed line.  Couldn't be simpler or quicker.

### Post by BentFX on 2011-09-22
My suggestion would be Inkscape. This took about 5 minutes. Gotta remember "Extensions > Modify Path > Markers to Stroke Color" to control arrowhead colors.
