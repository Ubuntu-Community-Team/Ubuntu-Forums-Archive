### Post by cariboo on 2008-12-22
I would suggest you install the ubuntu-retsricted-extras meta package, this will install most of the codecs you need to play most types of media. I would also suggest installing w32codecs from theMedibuntu repositories. To enable the Medibuntu repositories go [here]("http://help.ubuntu.com/community/Medibuntu"), and follow the instructions for your version. Scroll down and install gpg key also.

You can install the ubuntu-restriced-extras meta package using Synaptic Package Manager or the command line.
