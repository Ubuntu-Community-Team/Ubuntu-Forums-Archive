### Post by BrettD on 2009-12-22
I've got a similar problem at the moment.  I would suggest starting here: [https://wiki.ubuntu.com/X/Troubleshooting/IntelPerformance](https://wiki.ubuntu.com/X/Troubleshooting/IntelPerformance) to narrow down the cause of the issue.  If that page does not turn up a solution, try running the LiveCD on your machine to see if things work properly.  If 3D performance using the  LiveCD is good, the problem could be caused by a configuration or upgrade issue (I suspect this is the cause of my problem).
