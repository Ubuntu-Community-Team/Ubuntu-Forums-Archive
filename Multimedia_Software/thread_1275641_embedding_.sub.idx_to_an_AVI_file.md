### Post by yabbadabbadont on 2009-09-26
I didn't think that the avi container allowed for that.  If you wrap your files in a matroska container (mkv), then you can include the subtitle files too.  (You can even embed fonts to be used with the subs if you want)

### Post by plh on 2009-09-26
thanks for replying.
I found a workaround: 
using avidemux ( [http://en.flossmanuals.net/Avidemux/ExtractingDVDSubtitles](http://en.flossmanuals.net/Avidemux/ExtractingDVDSubtitles) ) I can extract an SRT file from the DVD, and then embed the subtitle into the video using mencoder.
