### Post by shaunthesheep on 2013-11-16
A new Lightworks for Linux Beta v11.5 is now out with a much wider range of export formats than previously. AC-3 audio can now be imported without having to transcode to wav. It is now possible to export a YouTube optimised file that is uploaded to your YouTube account as it is being exported so that it is fully uploaded by the end of the export. Export to DVD, BluRay and many other formats are now available. Export times are rapid because Lightworks uses all available cores of CPUs 100% to effect rendering.

The release of this Beta is an important moment for Linux. There is now a fully functioning professional NLE available on the Linux platform. 

Deb files for *buntu and Linux Mint and RPM files for Fedora and OpenSuse are available [here]("http://www.lwks.com/index.php?option=com_kunena&func=view&catid=19&id=56304&Itemid=81#56304"). 

The Windows and Linux versions of Lightworks are now fully synchronised in their development cycle and there will thus be much more frequent releases for Linux than previously.

Please spread the word.
