### Post by barthel on 2009-10-11
This is for folks using vlc in Karmic or grabbing the vlc 1.0 series from a PPA.

One of the changes in the 1.0 series is that the "image" video module has been replaced by the "scene" video filter. The options are the same, except that the "--image-out" prefix for the module is replaced with "--scene" for the filter.

I'm posting this here in hopes that it will provide a relevant search result for anyone else looking for the image module. The videolan wiki does not yet reflect this change and all my attempts to google led nowhere. (I finally got an answer via email.)

Cheers!
