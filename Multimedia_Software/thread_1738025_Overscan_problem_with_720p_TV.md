### Post by alexeix on 2011-04-24
Hi,

I've connected my Acer Aspire Revo 3610 to a 32" 720p TV, which has a resolution of 1366 x 768, and I am experiencing an overscan issue.
I'm running Ubuntu 10.10 with the latest nVidia drivers, but when I go into the nVidia settings and try to set the correct resolution, there is nothing which matches the TV.

Ubuntu seems to be correctly identifying the Panasonic display, but it is setting the resolution to the closet available value - resulting in overscan.

I haven't found a fixe for this problem, but I have found reports of it dating back several years.
Has the issue still not been addressed?

Does anybody have a proven solution?

Thanks in advance!

### Post by alexeix on 2011-04-28
No replies, but I finally found the answer myself.

The solution is to switch off Picture Overscan in the TV's display menu.

See [http://www.fixya.com/support/t1739141-panasonic_viera_doesnt_display_computer]("http://www.fixya.com/support/t1739141-panasonic_viera_doesnt_display_computer")
