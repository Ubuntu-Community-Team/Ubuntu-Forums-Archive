### Post by 3WrKB4C on 2013-11-11
Ok, I got it.
You can add external collections (i.e. collections contained on removable devices) by selecting
Settings -> Configure DigiKam -> Collections
then it is quite intuitive. I just did it. 
I guess that, in order to share the database, you must also move the database to the removable disk. So from the same dialog select Database and locate it on the removable file (I didn't need this, so you may need to test it).
Hope this may help.
Cheers,
Luca
