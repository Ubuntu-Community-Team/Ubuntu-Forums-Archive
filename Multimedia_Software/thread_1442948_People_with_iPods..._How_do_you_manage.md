However, the only problem that I'm facing right now is with my iPod... I have an iPod Classic 6G, and I would really love to be able to manage it as I used to in Windows XP and iTunes.

First thing, I want to be able to use Apple Lossless (.m4a) on Ubuntu, is that possible? Finally, is there a music player or an application that can efficiently import Apple Lossless files to iPods. It would be nice if there is a way to upload movies and podcasts too.

I believe that iPods should be well supported by Ubuntu, because, really! Who doesn't have an iPod! I Know that we should blame Apple for that, but isn't this why we're Ubuntu users? We can manage and find our ways around stuff without the need of blaming anyone. I really prefer FLAC over Apple Lossless because it has better compatibility, and I even looked for ways to hack my iPod to force it to play FLAC, and learned about an open-source application called Rockbox, but sadly it only works on iPods up to 5.5G... If there is another way, hacking the iPod is still an option for me!
