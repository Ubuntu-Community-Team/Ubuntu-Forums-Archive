### Post by jerenept on 2010-05-01
I think this has been addressed before..

open Software Centre, search for 'ubuntu-restricted-extras' and 'libdvdread4' and install them.
You will need to install CSS- open a terminal (Applications-Accessories-Terminal) and type 'sudo /usr/share/doc/libdvdread4/install-css.sh' (without quotes) and enter your password when requested. That shold help you out.
