### Post by rakuna on 2010-08-11
I think what you will need to do is go to Startup Applications (System -> Preferences -> Startup Applications) and create a new launcher.

The details that you need to fill in can be found by adding the rythmbox application link to the desktop or panel and copying it's properties.
