### Post by bark50 on 2010-07-13
Barrrrt, I don't think we're going to get an answer for our problem.  My workaround was to uninstall Banshee and add an alarm plugin to Rhythmbox.  You can find the instructions here:  [http://nedrebo.org/code/rhythmbox/alarm_clock/]("http://nedrebo.org/code/rhythmbox/alarm_clock/")  If you want to give this a shot, before following the instructions, check synaptic for "subversion" and install it if it's not already installed.  For me, this shutdown alarm for Rhythmbox works well.

By the way, more Rhythmbox plugins are available at [http://live.gnome.org/RhythmboxPlugins/ThirdParty]("http://live.gnome.org/RhythmboxPlugins/ThirdParty")
