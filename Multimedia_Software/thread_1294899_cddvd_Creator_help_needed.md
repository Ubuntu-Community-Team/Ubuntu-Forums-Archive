### Post by mharrison on 2009-10-18
I believe if you try to add data to a disk that was finalized (or closed, depending on your software), it will only show the last data that was burned.

My suggestion would be to format the disc, reburn the data you originally burned, and check your burn software for the option to leave the disk open after the burn so you can add more data to it later.

Not sure how to do this in Linux personally, as I have never done it, but I know that most Windows software has this option/ability.
