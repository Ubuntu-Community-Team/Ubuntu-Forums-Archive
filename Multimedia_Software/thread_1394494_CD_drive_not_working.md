### Post by labinnsw on 2010-02-06
1. Check your /media directory. It should look something like this.
[ATTACH]146259[/ATTACH]

2. Check /etc/fstab. It should like something like this.
```
# /etc/fstab: static file system information.
#
#  -- This file has been automaticly generated by ntfs-config -- 
#
# <file system> <mount point>   <type>  <options>       <dump>  <pass>

proc /proc proc defaults 0 0
# Entry for /dev/sda1 :
UUID=42ab818f-b1c2-468c-b1fa-4ec714495e10 / ext3 relatime,errors=remount-ro 0 1
# Entry for /dev/sda2 :
UUID=f3442d9e-2901-435d-940f-7ed53c364c0f none swap sw 0 0
# Entry for /dev/sda3 :
UUID=c7d708f4-3f46-47c5-9c23-e9cd39f0e223 /media/Storage ext3 relatime,errors=remount-ro 0 1
/dev/scd0 /media/cdrom0 udf,iso9660 user,noauto,exec,utf8 0 0
/dev/scd1 /media/cdrom1 udf,iso9660 user,noauto,exec,utf8 0 0
/dev/fd0 /media/floppy0 auto rw,user,noauto,exec,utf8 0 0
/dev/sdb1 /media/backup ntfs-3g defaults,locale=en_AU.UTF-8 0 0
```
