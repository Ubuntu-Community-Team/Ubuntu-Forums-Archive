### Post by &amp;KyT$0P# on 2016-07-28
That shows that you got the package from [https://launchpad.net/~mc3man/+archive/ubuntu/trusty-media]("https://launchpad.net/~mc3man/+archive/ubuntu/trusty-media"), which has a compatible libde265 that looks to being overridden?

Install ppa-purge, and use it to remove that other ppa -
```
sudo apt-get install ppa-purge
sudo ppa-purge ppa:strukturag/libde265
```

Then run [FONT=Courier New]sudo apt-get update[/FONT] and then try again.

Does it work this time?

### Post by mc4man on 2016-07-28
You could simply remove the strukturag ppa as you've nothing installed from it but something else is in play here. 
Are you using 14.04? If so then somehow you added the strukturag ppa & changed it from trusty to utopic, add-apt-repository would have added it as trusty, not utopic.
So if on 14.04 (seems reasonable to assume you are) then just remove the strukturag ppa, update your sources & the plugin will install.
(- or edit it back to trusty as the strukturag ppa doesn't conflict with the trusty-media ppa & does provide a gstreamer plugin.
