### Post by Sin@Sin-Sacrifice on 2009-12-15
Sub? As in sub-titles? If so and there's a .srt file in the folder then that's the sub-titles and they're only recognized when they have the same name. Ex: Fight Club.avi and Fight Club.srt will work. Fightclub.avi and Fight club.srt will not. If you're splitting the file then you need to split the .srt as well. Copy the text in the .srt from the time-line where you cut the movie and copy it to a second document with the same name as the second video. EX: Splitting Swordfish.avi to Swordfish1.avi, Swordfish2.avi will need to have Swordfish1.srt and Swordfish2.srt. By the way there's more sub files than srt but they all work the same. It's just a file with the time the text is suppose to show on the screen and the text that shows. 

If there's no subtitle file then the subs were hard encoded and you can download a .srt file for the movie... just Google "subtitles for 'your_movie'" and download and split the file as needed.
