### Post by ron999 on 2010-02-17
Hi
I have used a program called 'dvd-slideshow'.
It's in Ubuntu's repo.
It produces a VOB file to burn to a DVD, but it's very easy then to convert that to a suitable format for YouTube.

I got help from this blog here:-[http://dinsdalepiranha.wordpress.com/2007/10/16/]("http://dinsdalepiranha.wordpress.com/2007/10/16/")

There's more information here too:-[http://dvd-slideshow.sourceforge.net/wiki/Dvd-slideshow_0.8.0]("http://dvd-slideshow.sourceforge.net/wiki/Dvd-slideshow_0.8.0")

EDIT
dvd-slideshow has an -flv option to give the output as a flash video instead of VOB

### Post by ElSlunko on 2010-02-27
> **niceguy123 said:**
> Hi,

Any news on this? 

Thanks.

Haven't finished any single project but here's what I've discovered so far :

Imagination -- Pretty good for simple slideshows. Transitions are easy to apply, but not very "fancy". Unfortunately you can't build menus so you'd need another app for a fully interactive DVD. Exports to .vob, .ogv, .flv, .3gp.

Mistlex -- Simple interface that allows you to create menus & titles. I haven't found many transitions but there's a plugin interface. You can add image slides or videos so combining this with a fancier app like openshot to generate the video files might work out. Not sure if it supports animated backgrounds for the menu system.

QDVD Author -- An incredibly robust DVD creating app. However there were so many options & it seems to have been written for KDE that I was a little overwhelmed. I'll get back to hitting this one harder at a later date.
