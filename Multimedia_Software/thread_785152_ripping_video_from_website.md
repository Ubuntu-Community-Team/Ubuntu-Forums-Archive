### Post by bryncoles on 2008-05-07
you can actually just navigate nautilus to your '/tmp' folder, and when you load that web page, the flash video will show up as a file with a name like 'flash7864321'. copy this file to your desktop (of documents folder or wherever), rename it and its yours!

this seems to work with any youtube-style content, and it works with the link you posted!

note, the file will deleet from '/temp' when you close that webpage, but not from wherever you copy it to!
