### Post by Backharlow on 2009-06-07
show k9copy

Description: DVD backup tool for KDE
 k9copy is a tabbed tool that allows to copy of one or more titles from a DVD9
 to a DVD5, in the*same way than DVDShrink for Microsoft Windows (R).

Looks to me like the app you are trying to use is really a DVD "ripper", which is used for extracting video from a disk image (a DVD or image file from a DVD). You are trying to do the reverse. I would adopt an application designed for authoring (creating) a Video DVD. Once you do that, if that application refuses to transcode (change format) your file, I recommend using Avidemix, to first transcode your file to type that the authoring environment can work worth. Your basically trying to get your file to DVD compliant MPEG2.
