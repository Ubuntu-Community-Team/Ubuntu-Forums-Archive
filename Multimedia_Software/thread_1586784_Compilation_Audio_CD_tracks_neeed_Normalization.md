### Post by lykeion on 2010-10-02
I think both k3b and brasero have normalize plugins. Check out the menus in these and you'll find some way to enable the normalize  plugin. In braseror it's Edit > Plugins. You might get a popup to install the package normalize-audio in k3b.

### Post by cybrsaylr on 2010-10-03
Yep, I found it in Brasero and K3b. 
In K3b it's under Settings > Configure K3b > Programs.
The normalize plugin is checked on both so I assume that means it should take care of normalizing the volumes. I haven't burned anything yet to confirm this yet but know some files had varying sound levels needing correction.
