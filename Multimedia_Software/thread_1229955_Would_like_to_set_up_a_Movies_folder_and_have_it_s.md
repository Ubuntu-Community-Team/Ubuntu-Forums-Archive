### Post by fballem on 2009-08-02
> **yabbadabbadont said:**
> If you bookmark the folder in Nautilus, it will be listed in the Bookmarks subfolder on the Places menu.  Is that good enough?

Exactly what I was after. To do this:

1. Create a folder in the home folder called 'Movies'
2. Select Places - this will open Nautilus
3. Navigate to the 'Movies' folder.
4. Open the 'Movies' folder.
5. From the menu, select Bookmarks > Add Bookmarks
6. The 'Movies' folder should appear when you select Places from the main menu.
