### Post by -random on 2008-05-16
Funny things about vlc pause:

1. When you press space for a second too long, it picks up 2 or more spaces, thus pausing. . . and unpausing :P

2. If you select the window the pause doesn't work. For the pause to work you have to click on the video itself to kind of select it, and it then picks up spacebar.

If you think something is wrong with your Vlc, just re-install it in the package manager. You can also try running vlc in a terminal to see if any errors occur while running it.:mad:
