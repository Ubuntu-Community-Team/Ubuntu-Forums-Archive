### Post by norwoods on 2009-02-10
presumably you are talking about the nvidia-settings program.  if so, you must run the program with root permissions to set the configuration file.  you can do this by opening a terminal in the Applications--Accessories menu and starting nvidia-settings in the terminal window by typing gksu nvidia-settings followed by the enter key and the entering the password followed by the enter key.
