### Post by mayeco on 2008-01-11
Que no los confundan con eso que Ubuntu Linux y Software Libre son diferentes.  [Ubuntu Linux ]("http://www.ubuntu.com/community/ubuntustory/philosophy")**es** Software Libre por lo que al ser miembro de Ubuntu o de las Comunidades Locales de Ubuntu estas siendo participe y activista del Software Libre, solo que tienes Ubuntu como herramienta para promocionarlo y darlo a conocer.

De que otra forma se puede promocionar el Software Libre sino con su mayor expresión, el Proyecto GNU + Linux = Ubuntu Linux.

No hay existe mejor forma de promocionar el Software Libre y ser activista que en una Comunidad Linuxera.
