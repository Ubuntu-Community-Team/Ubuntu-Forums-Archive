### Post by Ptero-4 on 2008-10-17
También hay otro ubuntu que se llama xubuntu y utiliza el escritorio xfce (este es más ligero que el KDE o gnome). O puedes usar el opengeu (antes se llamaba geubuntu) que usa enlightenment 0.17, el fluxbuntu que lleva fluxbox, el icebuntu que lleva icewm o el ubuntu-lite que lleva lxde. Todos estos se basan en la estructura básica de ubuntu y usan las mismas herramientas base para configurar el sistema, lo que varia es el tipo de escritorio (y por lo tanto de apariencia inicial y desempeño general) con el que viene, el cual puedes cambiar o complementar con cualquier otro una vez tengas el sistema instalado.
