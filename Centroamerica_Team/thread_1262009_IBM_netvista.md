Si, riomacho tiene razon. El ISO alternativo (aunque dure tanto bajandose) es el mejor.

Para correr el disco...

**Insertalo en tu PC.** Una vez insertado, reinicia o apaga

**Modifica el orden del BIOS.** Para que el CD sea el primero en bootear

**Sigue los pasos.** Son realmente simples, la unica diferencia es que son mas detallados para mayor rendimiento. 

Si no quieres hacerlo asi, bajate el [Wubi Installer]("http://wubi-installer.org"). Dura casi lo mismo que el ISO alternativo, pero instala Ubuntu como aplicacion de Windows. Personalmente, yo bajaria el LiveCD (tengo una copia de respaldo :) ), pero eso te toca a ti.
