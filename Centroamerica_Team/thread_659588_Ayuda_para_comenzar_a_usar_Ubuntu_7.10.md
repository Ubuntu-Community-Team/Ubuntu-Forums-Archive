### Post by riomacho on 2008-01-07
En este caso hay que usar el otro CD (Alternate Install CD).  Seguro que el CD que quemaste esta bien, pero el procesador no tiene la potencia para instalar desde el Live CD.  Entonces con el alternate install CD, usted puede ir directamente a la instalacion, y en modo de texto si fuera necesario.

Aqui esta la pagina del UCR:
[http://ftp.ucr.ac.cr/ubuntu-cd/gutsy/](http://ftp.ucr.ac.cr/ubuntu-cd/gutsy/)

Busca el archivo indicado mas abajo, en su caso seria:

PC (Intel x86) alternate install CD

Y es importante 2 notas mas
- Revise el MD5sum ya con el archivo completo
- Hay que quemar un imagen ISO, que es un proceso un poco diferente.  

Mas detalles disponible:
[https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)
[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

En Español:
[https://help.ubuntu.com/community/ConseguirUbuntu](https://help.ubuntu.com/community/ConseguirUbuntu)
[https://help.ubuntu.com/community/ComoGrabarIso](https://help.ubuntu.com/community/ComoGrabarIso)
[https://help.ubuntu.com/community/ComoVerificarIso](https://help.ubuntu.com/community/ComoVerificarIso)
