### Post by oldfred on 2012-03-04
It is not a SATA issue although sometimes BIOS settings can make a difference.

From Ubuntu liveCD or USB download and run the boot infoscript. This may be able to repair your install also.

Boot Repair:
[https://help.ubuntu.com/community/Boot-Repair](https://help.ubuntu.com/community/Boot-Repair)
You can repair many boot issues with this or post the link to a run of boot info script so we can see your exact configuration.
