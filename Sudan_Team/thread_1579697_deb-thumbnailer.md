### Post by zero-n on 2010-09-22
[Deb thumbnailer]("https://launchpad.net/deb-thumbnailer") places the respective application icon over the standard  .deb icon, as demonstrated below, providing that the .deb file itself  contains the application icon.



[IMG]http://www.ubuntugeek.com/wp-content/uploads/2010/07/Deb-thumbnailer_1.png[/IMG]


**Install deb-thumbnailer:**
Run the following commands in terminal
```

sudo add-apt-repository ppa:deb-thumbnailer-team/ppa
sudo apt-get update
sudo aptitude install deb-thumbnailer

```**Remove deb-thumbnailer:**
Run the following command in terminal
```

sudo aptitude remove deb-thumbnailer 

```[COLOR=Red]**Note:**[/COLOR]
after installation you need to log out & in


[source]("http://www.ubuntugeek.com/deb-thumbnailer.html")
