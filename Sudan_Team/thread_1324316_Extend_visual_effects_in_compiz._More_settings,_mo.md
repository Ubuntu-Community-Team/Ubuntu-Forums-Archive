### Post by Y2J on 2009-11-12
If you want to work on a fancy Ubuntu operating system, all you have to do is enable Extra visual effects from Appearance preferences (System->Preferences->Appearance preferences->Visual effects). However, by default you will get only some basic visual effects. To get access to all extra visual effects (including the Compiz Desktop Cube and more), you will have to install compizconfig-settings-manager from the synaptic package manager. Then you will be able to access all hidden visual effects settings from System->Preferences->CompizConfig Settings Manager.

```
 sudo install compizconfig-settings-manager 
```
Done!
