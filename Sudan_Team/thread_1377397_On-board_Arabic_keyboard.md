my work while my keyboard have non-working key so i found a solution which is 

onboard keyboard it comes with Ubuntu but i think you may need to show it's menu 

by:

1- Right click on main menu.

2- select [COLOR=Red]**Edit Menus**[COLOR=Black].

3- Select Universal Access from the left.

4- From the right select[COLOR=Red]** OnBoard**[/COLOR] & [COLOR=Red]**onBoard Settings**[COLOR=Black].



but the onBorad doesn't have Arabic language by default so we need to add it manually:

1- Download the package from the attachment. ( thanks to [/COLOR][/COLOR][/COLOR][/COLOR][revoltism]("http://ubuntuforums.org/member.php?u=80665") )
[COLOR=Red][COLOR=Black][COLOR=Red][COLOR=Black] 
2- Extract it's content to 
[/COLOR][/COLOR][/COLOR][/COLOR]```
/usr/share/onboard/layouts
```[COLOR=Red][COLOR=Black][COLOR=Red][COLOR=Black]

3- All you need now is select Arabic layout from onBoard Settings.
Applications >> onBoard Settings

and then choose layout tab and select [COLOR=Red]**Arabic**[/COLOR]
