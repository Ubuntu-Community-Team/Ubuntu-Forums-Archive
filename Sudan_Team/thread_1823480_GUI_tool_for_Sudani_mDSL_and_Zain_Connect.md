### Post by zero-n on 2011-08-12
If didn't like connect to Internet using a console command you can use these tools.

[COLOR=Red]**Zain Connect:**[/COLOR]
This file sent to me by [toloykhan]("http://ubuntuforums.org/member.php?u=1054960") so credit goes to him.
1- Download the package from this link:
```
[http://www.mediafire.com/?84invg495a65aw4](http://www.mediafire.com/?84invg495a65aw4)
```2- Open your terminal and go to package download location.

3- Extract package content:
```
tar -xvf Zain_Connect_Linux.tar.gz
```4- Enter extracted package directory:
```
cd Zain_Connect_Linux
```4- Add execution to Install script:
```
sudo chmod +x install
```5- Run the script and follow the steps if you prompter to enter location just hit enter to accept default:
```
sudo ./install
```6- You will find the application in [COLOR=Red]**Applications**[/COLOR] >> [COLOR=Red]**Internet**[/COLOR] >> [COLOR=Red]**Zain**[/COLOR] if you are using Unity just open the dash type [COLOR=Red]Zain[/COLOR] and press enter.


[COLOR=Red]**Sudani mDSL:**[/COLOR]
1- Download the package:
```
[http://www.mediafire.com/?aihi6qbx8tg8ooc](http://www.mediafire.com/?aihi6qbx8tg8ooc)
```2- Open your terminal and go to package download location.

3- run this command if your have 32-bit machine:
```
sudo dpkg -i CrossPlatformUI-V2.1.2-SudaniOne-i386-Ubuntu.deb
```or this command if you have 64-bit machine
```
sudo dpkg -i --force-architecture CrossPlatformUI-V2.1.2-SudaniOne-i386-Ubuntu.deb
```4- Run the application from [COLOR=Red]**Applications**[/COLOR] >> [COLOR=Red]**Internet**[/COLOR] >> [COLOR=Red]**SudaniOne**[/COLOR] if you are using Unity just open dash and type [COLOR=Red]Sudani[/COLOR] and hit enter.
