### Post by zero-n on 2010-08-25
run the following commands :

```
sudo add-apt-repository ppa:sabily.team/ppa
``````
sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys D1A0D2FB
``````
sudo apt-get update
```enjoy Sabily's repository
