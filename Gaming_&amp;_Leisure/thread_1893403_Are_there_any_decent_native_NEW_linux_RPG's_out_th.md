### Post by Telengard C64 on 2011-12-11
> **JohnSmithNot1882 said:**
> Emulators regardless of their claim, Wine Is Not an Emulator, never run Windows smoothly and always have issues.

It's not just a *claim*. Wine really isn't an emulator at all, and it doesn't run Windows.

Suppose you want to run Windows applications on your computer. What do you need?

You need Windows libraries and core executables to provide the services and environment required by Windows applications. Problem is the libraries and core executables from Windows don't run on Linux. What to do?

Create from scratch libraries and core executables which run natively on Linux and which provide the services and environment which Windows applications need to run.

Congrats, you just invented Wine. Sort of  :p
