### Post by dfreer on 2010-01-20
bsnes can be compiled for 64-bit. ZSNES is strictly 32-bit only, but will still run on 64-bit systems just like any other 32-bit binaries if you install the right libraries. +1 for bsnes though, it requires a decent processor but is probably the most accurate SNES emulator around. Snes9x just had a new release as well, if the other emulators aren't what you are looking for.
