 Many big companies like id and Epic don't really like people repackaging their installation routines hence it might be difficult to find Debian packages to install those games.  For now you'll just have to go the route of downloading the games and installing them through the routines provided.  I would suggest installing them to /usr/local to make uninstallation easier.  I know you can download ET since it was released free but UT2K4 is a commercial game so you'll have to purchase it.  It should contain a Linux installer on the CD (UT2K3 did).  Check [here](http://ubuntuforums.org/showthread.php?t=5153) for some cool Linux games (long live Nethack!).
