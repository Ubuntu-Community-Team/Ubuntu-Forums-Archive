### Post by detrate on 2008-09-05
You can modify these in your player settings dialog if you're not comfortable with editing configs.


However, if you prefer configs, you can find more information on the [Nexuiz crosshair cvars here](http://www.nexuizninjaz.com/toolz/cvar/index.php?search=crosshair).

If you edit your config.cfg it'll be overwritten on next play of the game.  So if you want your changes to stick / load, you must create a file called 'autoexec.cfg' in your ~/.nexuiz/data directory.
