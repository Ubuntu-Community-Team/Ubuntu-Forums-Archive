Hi all, 

I'm a long time simutrans player (since around '98 ), but haven't played for a while.

The other day I got into a heated debate with my boss over why he should try playing simutrans rather than OpenTTD, and I went to show him why, when I realised there wasn't a simutrans package available!

Realising this didn't help my case, I decided to make a simple package, which I hope some may find useful.

(I have looked through launchpad, and there does seem to be a package, but I'm not sure if it's not in the repository yet, or if it's just in Hardy or something, but I'll post mine anyway)

For those in the know, this includes:
Simutrans 99.17.1 (latest "stable")
Pak 64
Launcher from the Applications > Games menu

.. it launches in 1024*768 fullscreen, which I thought would suit most people - some time I'll try to get around to writing a version of simustarter that will run on Linux... simuconfig.tab is in /usr/share/simutrans/config if you want to edit it.

You can download the package from my blog. ([link]("http://www.timwintle.co.uk/blog.pl/Personal/simutrans-for-ubuntu.html"))

there's no source package, and it includes the 32 bit version (which is recommended even for those, like me, on a 64 bit machine - see instructions on the blog post.
