### Post by MonkeyBoy on 2007-06-21
If you go to >System>Administration>Login Window>Security you can activate automatic login which will bypass the login window.

To start Mame you probably need to go to >System>Preferences>Sessions then you can add your mame as a startup program.

I hope this helps.  It sounds like a cool machine you have there.  :)
