### Post by RabidWeezle on 2010-11-24
For non-free (but still cheap)

Unreal Tournament (all except unreal tournament 3, but that requires at least a gig of ram anyway)

Neverwinter Nights Diamond (on gog.com for less that 10 dollars, DRM free, digital download) Has a linux port, play online or over lan.

**For free:**

The Mana World - Free to play mmo in anime 2d.
```

sudo apt-get install tmw

```

Frozen Bubble
It's like Bust a Move, except it has online multiplayer if you have someone's ip to play.
```

sudo apt-get install frozen-bubble

```

Wolfenstein: Enemy Territory
Good game, but good luck getting the sound working.

Armagetron or Gltron, I can't remember which one, but one of them are multiplayer at least. And very fun if you like the old movie tron with the light cycles. Check the ubuntu software center for these 2.

BZflag: Personally my favorite linux (and windows/mac/etc) online game. It's a 3d tank game where you pick up power up flags that do different things with really cool maps and lots of players.
```

sudo apt-get install bzflag

```

Warzone 2100 - RTS
Free port of the playstation 1 game of warzone 2100 made completely free and now has online play with a nice lobby system. Basically it's like controlling a bunch of veicles you design and build in factories in a war for oil (sounds familier)on a far away planet. While it is top-down view, the units/land/buildings/etc are all 3d. Good music too. It's fun customizing your units.

```

sudo apt-get install warzone2100

```

There's much more than that of course, these are just ones I find were my favorite that could run on 512 megs of ram. Good luck on your quest :)
