### Post by efikkan on 2010-08-02
If you got an nVidia card, enter "nVidia X Server Settings", choose "DFP-0 (LPL)" under the "GPU 0" category, and then select Centered under "GPU Scaling Method".

I'm pretty sure AMD got a similar setting.
