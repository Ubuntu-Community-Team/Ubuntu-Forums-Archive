### Post by zooter on 2006-07-04
I have found a great program for emulating the mouse and keyboard with a joystick or gamepad on linux.  I think it could get rid of the need for joystick configuration stuff in individual programs.  As long as the joystick is recognized by linux, it should work.  One problem with this program, however, is that it's not made as a package for ubuntu.  The program is called **qjoypad**.  Here is the website:  [http://qjoypad.sourceforge.net/](http://qjoypad.sourceforge.net/)

I use it on mandriva and it works great with the exception of a few bugs.  I have emailed the developer and he said that he's too busy to work on it anymore and is hoping more people can work on it.  

I also use qjoypad to watch streaming video of the internet through my tv.  I have a usb extension so I can use my gamepad like a tv remote.

I am posting this message because I think something like this should already be available in every distro but, for some reason, it's not.

I hope you all find this info useful and that either qjoypad or another program similar is developed and made available as a package for ubuntu.  I'm also hoping that this idea gets posted as a 'sticky' so that more people can work on a program that is very useful and yet not easily available on linux at the moment.
