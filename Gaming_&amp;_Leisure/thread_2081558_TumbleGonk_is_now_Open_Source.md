### Post by brooss on 2012-11-07
TumbleGonk is a causal ‘tile swap’ style game with polar based tile layouts that provide a unique gameplay style.

The game code is now licensed under GPLv3 with CC-BY 3.0 art.

[youtu.be/6tR638VHdIo]("http://youtu.be/6tR638VHdIo")

You can find more information and get the source from the sourceforge project page [http://sourceforge.net/projects/tumblegonk/](http://sourceforge.net/projects/tumblegonk/)

Detailed build instructions for **Ubuntu** are on the wiki [http://sourceforge.net/p/tumblegonk/wiki/Home/](http://sourceforge.net/p/tumblegonk/wiki/Home/)
