### Post by Arthur_D on 2011-02-21
> **Semi_Nerd said:**
> Thanks guys... I do appreciate it, but I was looking for something along the lines of, "This is how you make windows games work on Linux..." I will check out these other suggestions tho.:)
You can use Wine to fake a Windows environment. Far from perfect, but works surprisingly well with some games, and incredibly bad with others.
Point is, game developers usually don't supply Linux versions of their games, and to expect Linux to somehow magically run all Windows games shows that you haven't quite caught the idea that GNU/Linux is a completely different platform, both technologically and philosophically.

However, chances are that you'll be able to find at least a few Linux games you are satisfied with, or get some Windows games running well in Wine.
