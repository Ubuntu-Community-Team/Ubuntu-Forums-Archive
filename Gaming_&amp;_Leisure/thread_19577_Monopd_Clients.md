### Post by TravisNewman on 2005-03-12
Gnome and KDE have cooperated to the point that running KDE aps in Gnome or vice versa works quite well. It may take a little longer to start up, since it's gotta load some KDE libraries, but it'll work. You can install KDE from the Universe repository, or any of the libraries you may need if you don't want to install ALL of KDE. I would suggest you give Gnome a good shot before installing KDE and never using Gnome again.
