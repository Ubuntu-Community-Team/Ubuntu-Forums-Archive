### Post by Sofox on 2011-10-27
Currently we have Ubuntu Software Center and Desura, along with a variety of online game purchasing sites like Gameolith and Indievania. It will be difficult to introduce another game purchasing method to Linux users, especially one that involves installing something and hasn't even started building up momentum.

In order for it to succeed, you'd need to have something unique about it, something that solves a problem and does something that nothing else does... and I think you've already put your finger on what that could be.

If we could have a Steam-like platform, only with selling Windows games and with Wine built in, we could open up a whole new market. The idea is that users buy games on the program, but the games are Windows games and the program automatically takes care of Wine settings and the like. What's more, no game first goes up for sale before it is first comprehensively tested to ensure that it works well with Linux and Wine, and so the users can buy in complete confidence.

With the huge amount of Windows games available, I'd say this method can make it easier for Linux users to play a wide range of games, and solves a problem in the process.

Good idea, Feel free to use any ideas I put in to the Mega Games Packs, [Windows]("http://ubuntuforums.org/showthread.php?t=1869429") (WINE) And [Linux]("http://ubuntuforums.org/showthread.php?t=1858387")
