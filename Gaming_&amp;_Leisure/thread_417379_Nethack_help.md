### Post by engla on 2007-04-21
You should really learn the standard nethack controls. They are available to see in the in-game help (press ?). The standard doesn't need the numpad (I play some with my laptop), but uses the letters hjkl,yu,bn. Now that might seem strange but after a while you learn.. it is the same arrangement as in the old 'vim' editor.

### Post by shmeeter on 2007-04-21
I agree.  On a laptop, without a keypad, learn the hj-etc way.  You can't go diagonally with the arrow keys anyway.  The biggest problem becomes how to kick, untrap, stuff like that, but you just use ctrl-d, #u(ntrap), etc.  Once you get used to it it's not so bad.
