> **WaveMyBlackFlag said:**

the topic of this thread is unrelated to your problem, please find out if the problem is for all programs, or if it is just games.
Then make a new thread under Absolute Beginners Talk, if the sound issue is global for all programs, if it is just games then I would be amazed but this is the correct subforum for it then.


If you have more than one audio output (like onboard soundcard and usb headphones) and use pulse audio, then get from Synaptic Package manager a program called Pulse Audio Device Chooser (padevchooser) (I think). Pulse can direct sound to different outputs and pulse audio device chooser lets you pick which output each program uses.
You might need to minimize a game you are playing,
open puse audio device chooser
left click on the icon that appears on the icon tray
select volume control
set the main output as default
set the game you are currently playing to switch to the other output.

First place to go is System &#8594; Preferences &#8594; Audio and try out all the options.
