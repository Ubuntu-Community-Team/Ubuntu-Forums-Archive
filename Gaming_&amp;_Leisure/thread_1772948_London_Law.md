### Post by MunkyJunky on 2011-06-02
If you're running the 'classic' Ubuntu interface (not Unity) look under Applications > Games. 

If you're running Unity (Ubuntu 11.04's default) open up the dash (super key, or click on the icon in the top left of the screen) and type 'London Law' into the search bar.
