### Post by KIAaze on 2009-11-19
Flight of the Amazon Queen is available in the repositories too and is pretty fun. (uses scummvm too):
```
sudo apt-get install flight-of-the-amazon-queen
```

Otherwise, there's also Adventure Game Studio: [http://www.adventuregamestudio.co.uk/](http://www.adventuregamestudio.co.uk/)
Usage made easier under GNU/Linux with the Adventure Game Goddess: [http://feags.sourceforge.net/](http://feags.sourceforge.net/)

And by the creators of Machinarium (thx for the tip. I only played samorost 1 so far. :) ):
[http://www.amanita-design.net/samorost-1/](http://www.amanita-design.net/samorost-1/)
[http://www.amanita-design.net/samorost-2/](http://www.amanita-design.net/samorost-2/)

Sowjet Unterzoegersdorf: [http://www.monochrom.at/suz-game/](http://www.monochrom.at/suz-game/)
