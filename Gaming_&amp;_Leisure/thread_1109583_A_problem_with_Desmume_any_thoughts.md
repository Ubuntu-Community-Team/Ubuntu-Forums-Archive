### Post by Chemical Imbalance on 2009-03-29
The only thing you need to do with no$gba is start the .exe and then point the location to where your game is.  It should work fine.  You don't need to download any bios files or anything like that.  You will however need to set the save type like the other person said.  I'll check out that game and see what settings work.


EDIT:

okay to get that game working, all you need to do is after you start it up and it says you have the wrong save type, just go to Options, Emulation Setup, and set NDS Cartridge Backup Media to EEPROM 64K bytes.  For every game you try with this emulator, try each option in the NDS Cartridge Backup Media until it loads.


Here's the difference between just plain no$gba and no$gba + nds2xgl2


[ATTACH]107989[/ATTACH]

Google for a download of nds2xgl2 and place it in your no$gba directory.  To start it up choose nds2xgl2.exe

It basically gives you a bigger window to play the games in and allows other options.
