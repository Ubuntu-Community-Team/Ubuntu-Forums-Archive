### Post by Toxicity999 on 2006-03-05
You have Several Choices here... (I have played this game.)

1)Install wine and download the Windows FlashPlayer Standalone 7.

sudo apt-get install wine

And... I'm not quite sure where I obtained the Windows Standalone player but I'm pretty sure it available on the Macromedia site still. [http://www.macromedia.com](http://www.macromedia.com) search for Standalone

after you get it there should be an EXE in the zip, extract it and run it with wine.
(Right click it, Run With Other application, click the arrow near bottom and type wine.)

This will start the player where you can then run loader.swf to start the game.

2)Install Linux Flashplayer 6 Standalone

See... This is a real trade off... Most people get better performance running native instead of via wine... BUT this is a way outdated flash version...

You can also get this from the Macromedia site [www.macromedia.com](www.macromedia.com) by searching for Linux Standalone (or other variations I forget exactly what I searched for.)

3)Install the Mozilla Flashplayer

sudo apt-get install flashplayer-mozilla

Restart Firefox if running, aces. But, the catch is for me anyway Performance is VERY laggy after the first few minutes of play... Then again all of these options seemed to follow this trend... I ended up running it off my Winbox... It's not a huge problem as I don't play this hardcore or anything... if its even possible in this game (heh.) I only really play for my girlfriend whos really into it. and, I must admit it can be fun at times... I mean you can expect lag flash isnt exactly choice one for semi streaming 50mb games haha...

wish you luck.
