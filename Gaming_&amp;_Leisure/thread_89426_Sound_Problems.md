### Post by lateo on 2005-11-12
try first to disable 'choose menu element' sound event
go to :
system>preferences>sound, sound event, user interface, 'choose menu element'
and choose no sound at all.

this will prevent esd from 'keeping' sound for a few seconds. most of the time this is enough for games to run with sound.
