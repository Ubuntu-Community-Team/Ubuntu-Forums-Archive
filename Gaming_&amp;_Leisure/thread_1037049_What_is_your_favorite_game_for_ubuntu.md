### Post by jaminux on 2009-01-22
Wyvern ([http://www.cabochon.com](http://www.cabochon.com)) is my favorite game for Linux. It is a highly addictive 2D fantasy MMORPG based which draws inspiration from DnD and NetHack.

Not only is it my favorite MMORPG for Linux but I also think that it is the best free 2D MMORPG around. No others I have seen seem to be half the quality.

To install it type the following in the terminal:

```

wget http://www.cabochon.com/downloads/jex/client/platforms/PureJava/Standalone/install.jar
sudo java -jar install.jar
```

This initiates the install GUI, I recommend installing it in the default location /wyvern/client as you get funny bugs installing it in other places

After the install:

Wyvern tries to read and write settings from its directory, which is owned by root, so if you try and run it as a normal user then you get some very annoying GUI bugs and the client will refuse to save your settings. To avoid this right click and create a launcher on the desktop. In the name field type Wyvern, in the command field type gksudo Wyvern; then press ok. If you wish the icon to have an image one can be found in 
/wyvern/client.

When using this icon you will now be prompted for the root password like on the synaptic, meaning that you are executing the program as root and therefore can read and write from the directory Wyvern is installed in.

Now the game is installed you can go to [www.cabochon.com](www.cabochon.com) and create a character using the create character page in order to play (I recommend making a Giant, Dwarf or Rakshasa to start off with as they are the easiest races).

------------------------------- WARNING --------------------------

Wyvern has very strict (and sometimes strange) rules. I would suggest reading them before you start playing in order to avoid getting banned.

Home any of you that try it have as much fun as I have, it really is addictive.
