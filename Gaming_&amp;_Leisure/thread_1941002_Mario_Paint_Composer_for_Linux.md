### Post by Toz on 2012-03-14
> **HydroCity said:**
> Can anyone tell me where to get mario paint composer for linux? or a good alternative for linux? thanks

Mario Paint Composer works fine in wine. Make sure you have wine installed (look for it in the software centre) and download the program from [http://www.unfungames.com/mariopaint/]("http://www.unfungames.com/mariopaint/").

Once unzipped, right-click "Mario Paint Composer.exe" and select the option to open or run with "Wine Windows Program Loader".
