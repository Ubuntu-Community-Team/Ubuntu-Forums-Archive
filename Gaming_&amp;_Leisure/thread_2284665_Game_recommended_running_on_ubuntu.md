### Post by oldrocker99 on 2015-07-01
You should be able to run: Warzone 2100, uqm (the Ur-Quan Masters), Battle for Wesnoth, 7kaa (Seven Kingdoms Ancient Adversaries), Freeciv, probably 0AD, and a bunch more.

To help you in your search, open a terminal and type this:```
sudo apt-get install synaptic
```

Synaptic is pretty much the best package manager, and is **much** faster than the Ubuntu Software Center. It also has a very good search feature. Search for "strategy games," or "arcade games," of "first person shooter" or really, whichever type of game you're looking to play. There are very few real-time strategy games, you should know.

Steam has some big downloads, because Steam has some big games, but so do the Ubuntu file repositories, and I have had to wait on downloads to finish with both. Steam has some crackerjack games which are free to play (and to install), including Team Fortress, DOTA2, Robocraft, WakFu, Destination Sol, and several others. Steam has over 1000 Linux games, from indie to AAA games, and they frequently have sales to make buying a bit more affordable.

It would help if you also posted the output of this:```
lspci
```
This will allow people to see what your laptop has going for it.

### Post by giga+bytes on 2016-07-17
Check out GOG.com; it's a Steam-like service owned and run by the makers of the Witcher games, but has none of the things I despise Steam for (such as unacceptable TOS, mandatory required client, online login requirement, etc.). They have a large selection of games (not nearly as large as Steam, but new games are added often), *all DRM-free*, and many are fully compatible with Linux, particularly Ubuntu. System requirements are shown for each game, so you can make sure your computer should be able to handle it before buying, and you can see how big a download each game is. I get all my games from there and have been completely satisfied with their service!
