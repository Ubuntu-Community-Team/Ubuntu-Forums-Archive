Hi,


You like linux ? You like games ? They you'll like playing games on Linux easily !!


Basically, PlayOnLinux is a Wine interface. It will help you create a separate prefix for each apllication. On some game, they are already scripts done, so you don't have to do much things...

But they are also exclusive tools, my favortie is "WineVersion", so you can use different wine versions for different games !!!


The second version is very good, but they are still some lack (no debug console, no scritp editor... you have to edit the files yourself and use terminal...). Anyway, it's a very good program.


Bonus TIP :

If you use **Steam**, you can share maps/mods/etc with a Windows installation. To do it, simply make a symbolic link (ln -s, you can use third button of mouse in nautilus) from "/home/$USER/.PlayOnLinux/wineprefix/Steam/drive_c/Program Files/Steam/Steamapps" to you STEAMAPPS folder on Windows !! Secure, simple, and spaceless !!!
IMPORTANT : The Steam folder on Windows should be a FAT32 partition !!!




LINK : [http://www.playonlinux.com/en/](http://www.playonlinux.com/en/)


ENJOY !!
