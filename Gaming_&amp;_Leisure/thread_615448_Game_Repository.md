### Post by hikaricore on 2007-11-17
This is unlikely to happen officially from Canonical servers.
From a software license and manpower standpoint this just isn't feasible.

However there are user created repos in abundance such as [DarkMagez]("http://mirror.randumb.org/darkmagez/repo/dists/edgy-darkmagez/") **outdated** edgy repo which had up to date packages of several games. 
Similar repos exist for certain projects as well as other user created game sets, tracking these down are probably your best shot.  You could even start your own with some sever space/bandwidth.

Alternatively you can just keep track of the updates on [GetDeb.net]("http://www.getdeb.net") from time to time as they update many gaming packages often.
