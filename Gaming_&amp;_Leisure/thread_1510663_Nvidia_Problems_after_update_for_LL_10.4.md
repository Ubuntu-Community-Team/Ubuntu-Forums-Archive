### Post by acidviews on 2010-06-15
Have you recently updated ur system only to find that ya Nvidia is no longer working?

Fear no more as it is only a small problem and you don't need to do much to fix this!
Simply go to System > Admin >Synamptic.

Search for Nvidia Kernel and sort by latest.
Choose 195.36.24 and or try Nvidia kernel common 2 *  +1Ubuntu2

perhaps this is a problem only found by me at this stage but i felt i would share the solution anyway just in case it helps somebody. 

Me my mate are both running ubuntu 10.4 and we had problems getting our wow game to run because opengl was failing a further look i found that the Nvidia kernal did not update with the rest of the pakage.
