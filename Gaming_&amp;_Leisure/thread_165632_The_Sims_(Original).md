### Post by bluevoodoo1 on 2006-04-25
You could try with nautilus. Navigate to your CDrom, right click on the setup.exe and select wine. If wine is not present, choose custom command and type in wine. You won't see any potential error messages as you would with the command line, but this is another way to try it. [Winehq.com]("http://appdb.winehq.org/appview.php?appId=768") doesn't say much if the game works or not.

### Post by DoktorSeven on 2006-04-25
The Sims has been notorious for not running via Wine or Cedega -- generally the installer doesn't work and the game itself doesn't get far.

I do seem to remember *someone, somewhere, somehow* getting a preinstalled (via Windows) version of The Sims working, but I can't find out how.  Makes me want to try it out myself.

There was a version of The Sims created by Transgaming especially for Linux (basically the Windows version and a patched Wine to run it) but it's long since out of print (it came with a version of Mandrake Linux quite a while back).
