### Post by Fatesauce on 2012-10-29
I discovered an odd compatibility issue with minecraft and Oracles Java 7 vs OpenJDK 7.  I reinstalled Ubuntu 12.04 recently and for the first time i installed Oracles Java 7 instead of OpenJDK 7, which i used in my last install.  Minecraft would not work.  After log in, I would be presented with a blank grey screen reminiscent of the old WWW days when you attempted to view a java applet on a website without having java enabled.  

Minecraft only needs Java 6 to run, and with Java 6 from Oracle, it worked perfectly.  The Technic mod pack though, needs Java 7.  It works fine with OpenJDK 7, but the same scenario with Oracle Java 7.

Interested to read some feedback on this.  It seems a bit less than average that the official Java 7 release is inferior to the open source release.
