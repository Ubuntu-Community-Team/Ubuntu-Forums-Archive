### Post by LinuxFox on 2009-05-28
SuperTux version 0.1.3 doesn't need a 3-D card.  It has Software and Open GL rendering modes.

SolarWolf is a 2-D arcade-styled game and doesn't need a 3-D card.

Kobo Deluxe is a 2-D shooter.  Like SuperTux, it has both OpenGL and Software rendering.

These games are under the Add/Remove menu.
