### Post by Footissimo on 2006-07-13
1) ```
sudo sh tremulous-1.1.0-installer.x86.run
``` :) (i.e. sudo = as root, sh = run the following).

2) Go into the game, change 'source' to internet..then refresh

3)```
cd /usr/local/games/tremulous
sudo sh uninstall
```Should do it

### Post by Footissimo on 2006-07-13
I presume those'll be the beta maps on the derelict server..see [here](http://tremulous.megavoid.de/index.php?module=download) to download them and unzip them into ~/.tremulous/base (the dot signifies that it's a hidden directory and the "~" means in your home directory) - that particular directory is where the user configuration and preferences are held (rather than the game itself which will be installed systemwide) when unzipping, just right-click and click on show hidden directories :)
