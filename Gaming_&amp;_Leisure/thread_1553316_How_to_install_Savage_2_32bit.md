### Post by Cresho on 2010-08-14
put the file in a known directory like....

open up a terminal and cd into the directory where its installed

example..

```
cd /home/ted
chmod +x /home/ted/Savage2Install-2.1.0.1-i686.bin
./Savage2Install-1.3.99-i686.bin
```

if you want all users to have access then you put sudo in front of the install command
