### Post by oldrocker99 on 2015-01-13
You may be in luck; winehq shows the Windows demo, at least, with a Platimum rating, meaning that it runs flawlessly. While Google shows a "free" download link for the game, it's very likely to be infected, and viruses under wine are just as nasty as viruses under Windows. It is legally available;) for Windows on Desura, with no DRM,  for $19.95, and also offers a free demo. If you don't have wine installed, do this:
```
sudo apt-get install wine
```

Then CD to the directory where you have downloaded the .exe file for the game, and type this:
```
wine (programname).exe
```

The installer should come up, and just choose the defaults. It **ought** to run pretty well.
