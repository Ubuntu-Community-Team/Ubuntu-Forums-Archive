### Post by hikaricore on 2007-06-25
None of the other Harry Potter games run under WINE.  This one is brand new.  So I'm guessing no.

Also it is NOT useful what-so-ever to install directx under WINE.  Wine has it's own implementation
of the directx libraries and it WILL NOT use the ones you install (if you even manage to) unless you
otherwise force it to.  Forcing it to will not result in anything thing aside from the probable crash of
WINE itself.  Some games need specific libraries from directx, but these are often minimal and only
to satisfy a game/application that checks for them and they aren't physically used after the check.
