### Post by ygarl on 2005-10-03
I like Fluxbox and Windowmaker. Both have MUCH smaller footprints than XFCE, Gnome and KDE.

They won't copy your menu items over, so make sure you note your fave program name (Windowmaker is slightly less extreme - making shortcuts for Firefox, XTerm, and a few useful bits and bobs. Fluxbox pretty much leaves the menu bare except for a terminal and some Flux configuration options - so you HAVE to make some notes before you try it on your fave programs!)
Both of them can be tarted up to look pretty sexy, but come 'out of the box' pretty minimalistic - and using very few resources!

Of course, they may not load much of the Gnome/KDE services either so you may find yourself manually starting some services if you want to run some things requiring those things (find out the easy way - try them!)
If all else, it gives you a chance to experience some other windows managers. You can always uninstall them afterwards if they don't work for you!
