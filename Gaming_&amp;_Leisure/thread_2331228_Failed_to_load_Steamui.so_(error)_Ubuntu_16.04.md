### Post by efflandt on 2017-01-04
daa-goddy, for GTX 1070 you likely need newer drivers than might be included in 16.04. But if you used a .run file from Nvidia website, not sure how to remove that.

Normally you would add the graphics-drivers ppa (see [https://launchpad.net/~graphics-drivers/+archive/ubuntu/ppa](https://launchpad.net/~graphics-drivers/+archive/ubuntu/ppa) ):```
sudo add-apt-repository ppa:graphics-drivers/ppa
sudo apt-get update
```Then use "Additional Drivers" to chose a version (newest at the moment is nvidia-375).

Current official release: `nvidia-370` (370.28)
Current long-lived branch release: `nvidia-367` (367.57)

Although, if you have hybrid graphics (Intel and Nvidia), Additional Drivers might not find nvidia drivers automatically, so you might need to manually install (which would install related files). First line here would clear out any previously installed nvidia package (but not .run files):```
sudo apt-get purge nvidia*
sudo apt-get install nvidia-375
```
