### Post by acej1995 on 2011-01-30
Go to **Application**s -> **Ubuntu Software Center**
Go to **Get Software**, and in the search bar type in **PlayOnLinux**.
Install that, update it, and make sure you have _**3.88**_

Once you downloaded that, go to **Applications** -> **Games** -> **PlayOnLinux**.

Click the Install button, and try to install StarCraft 2 Through there.  If that doesn't work, try doing it with **CrossOver Gamer**, it's a purchasable program, but you can try the 7 day trial, at least to see if it works, and if you like the game through Ubuntu.

### Post by acej1995 on 2011-01-30
When you click **apply**, a window should come up, usually about 1-10 seconds later, just like an installation wizard, click forward, and it will ask you if you use a **Digital Download** or a **CD/DVD**, and go on from there.

EDIT~: If you use the **Digital Download**, go to [http://www.us.battle.net/](http://www.us.battle.net/) Login, click **StarCraft 2 Wings of Liberty**, and click **Manage game**.  Click the W**indows Installer**, and it will download the Blizzard game installer, then redirect the PlayOnLinux to there when it prompts you to Browse for an **Executable** (Exe)
