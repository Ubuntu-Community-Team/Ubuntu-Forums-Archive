### Post by deadflowr on 2017-12-05
Any game that can use the gamepad can be played with the gamepad.
The only real trouble will be that each game may have its own configuration settings you will need to set.

One game, depending on overall system hardware, you might want to test run with would be supertuxkart.
( i say system hardware because it's a 3d game and does require somewhat more modern hardware for a cleaner gaming experience.)

It's in Ubuntu's archives, so at least that would be one game you could try.
