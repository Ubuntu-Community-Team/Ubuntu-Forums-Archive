### Post by cgreen on 2006-02-08
Not all shoot-em-up, but here are five games that have kept me out of bed on more than one occasion:

armagetron - 3D Tron-like high speed game

bzflag - a 3D first person tank battle game
Can be a bit buggy in solo mode (run via bzflag -solo n where n is the number of opponent bots)

frozen-bubble - Pop out the bubbles!
Perfect on occasions when the brain is just ticking over quietly

scorched3d - A 3D version of the classic DOS game Scorched Earth
Apocalypse mode may puke trying to find smoke-orange.bmp in which case fix it by adding a symbolic link from any other .bmp file in the same directory. In other words, paste these two commands one after the other into a Terminal session:

Note: If the location of the smoke-orange.bmp file in your error message differs from the path below then change it to use the path your machine reports instead!
> cd /usr/share/scorched3d/data/textures/explode/
sudo ln -s exp00_10.bmp smoke-orange.bmp

planetpenguin-racer -  another 3D racing game featuring Tux, the Linux penguin

Tastes good this Ubuntu, I think I'll have another cup.
