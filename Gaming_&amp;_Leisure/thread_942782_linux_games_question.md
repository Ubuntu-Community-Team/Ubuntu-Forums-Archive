### Post by Naiki Muliaina on 2008-10-09
Use 'cd' to get to the file in terminal (if its in the top level  of your home folder skip the cd). For me i download all files to my desktop, so it was on my desktop. If its on your desktop then follow the commands below.

```

cd Desktop
sudo chmod a+x armyops250-linux.run
sudo ./armyops250-linux.run
```

That did it for me :)
