### Post by maitrebart on 2011-02-12
I fixed the problem by editing the **~/.etracer/options** file with the following options:

```
set fullscreen false
set x_resolution 1280
set y_resolution 960
set x_resolution_half_width false
```

where the size of the window will be 1280x960. You may want to fit your own.

Afterwards, you may want to change these options from the Configuration menu.
