### Post by alexforcefive on 2008-06-27
from [http://zerowing.idsoftware.com/linux/doom/](http://zerowing.idsoftware.com/linux/doom/)

> The following files need to be copied from the win32 install CDs
to your base/ directory (by default, /usr/local/games/doom3/base)

base/pak000.pk4
base/pak001.pk4
base/pak002.pk4
base/pak003.pk4
base/pak004.pk4

so pop your cd in the drive, find the folder named "base", and copy those files into /usr/local/games/doom3/base
