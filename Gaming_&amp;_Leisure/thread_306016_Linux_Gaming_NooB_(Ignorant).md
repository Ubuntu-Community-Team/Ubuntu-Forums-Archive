### Post by ButteBlues on 2006-11-26
One bit of advice, since you are obviously comfortable with partitioning.


Create a separate partition for /home (even if using the Ubuntu Desktop CD, this CAN be done very easily). It'll give you that "easily restartable" feeling simply because all your application data (Firefox bookmarks, etc.) lie there, and if you need/want to reinstall Ubuntu, you don't have to worry about reconfiguring your applications.

No, it doesn't.  Gnome and KDE do not use more resources, they just preload certain programs into memory.  DE's like XFCE and Windowmaker actually end up using more resources then the "bloated" Gnome and KDE during multi-tasking.  

Please see: [http://ktown.kde.org/~seli/memory/](http://ktown.kde.org/~seli/memory/)
and pay close attention to [http://ktown.kde.org/~seli/memory/#section8](http://ktown.kde.org/~seli/memory/#section8)
