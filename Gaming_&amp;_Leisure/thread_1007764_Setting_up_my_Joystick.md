### Post by mastermindg on 2008-12-10
Bus 002 Device 011: ID 0f30:010b Jess Technology Co., Ltd 

Device 1 Jess Tech GGE909 PC Recoil Pad
  6 axis, 12 buttons, 0 hats, 0 balls

Plug-n-play with Ubuntu 8.04. I prefer the ZSNES emulator. To get this joystick (and probably any other) to work, simply open up the emualtor, save configuration and then edit:

~/.zsnes/zinput.cfg

change joy_sensitivity=64

This will allow you to use the arrows and/or the dual joysticks.
