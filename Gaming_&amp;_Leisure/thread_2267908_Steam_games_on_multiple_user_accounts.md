### Post by R33D3M33R on 2015-03-05
Put games in a folder where both users will have access to it and add this newly created library in sisters Steam account. After that turn on Family Sharing: [http://store.steampowered.com/promotion/familysharing](http://store.steampowered.com/promotion/familysharing)
