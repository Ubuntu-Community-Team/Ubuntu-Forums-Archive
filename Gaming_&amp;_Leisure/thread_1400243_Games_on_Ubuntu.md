### Post by beren.olvar on 2010-02-06
There are not many games with native support for linux, but there are a few good ones. You should check those also.

I think a well known commercial game with linux support is Never Winter Nights. So if you like RPG try checking that out.

For the windows games you can try installing "wine" from the repos. It is an application that allows to run windows programs. Of course, new games not always run properly, but you can check their applications database ([http://appdb.winehq.org/](http://appdb.winehq.org/)) to see if your favourite games are supported.



cheers, and good luck!
