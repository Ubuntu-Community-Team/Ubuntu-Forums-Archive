### Post by MaximB on 2010-03-23
First check that your Video Card drivers are installed (for Nvidia and AMD/ATI use their proprietary drivers from their website OR the ones that come with Ubuntu).

If you are using Ubuntu 64-bit you need to install the 32-bit libs (but that doesn't seems to be the case here).

Also you can change the Gamma/brightness from the options menu, the game is very dark by default, so you will need to close all light sources inside your room, they might reflect the monitor and make the game hardly visible.
