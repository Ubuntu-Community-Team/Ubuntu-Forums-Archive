### Post by ubuntu27 on 2010-06-08
Hello tgaston34. It sounds like you don't have Flash player installed.

What Linux distribution are you using? Is it [Ubuntu]("http://www.ubuntu.com/"), [Kubuntu]("http://www.kubuntu.org/"), or [Xubuntu]("http://www.xubuntu.org/")?

And are you using 32-bit or 64-bit OS?


I will assume that you are running Ubuntu Linux 10.04 LTS (a.k.a Lucid Lynx) 32-bit.

Open _Synaptic Package Manager_ at System/Administration/Synaptic Package Manager.

And look for the **[ubuntu-restricted-extras]("apt://ubuntu-restricted-extras")** package and install it.


Alternatively, install **[xubuntu-restricted-extras]("apt://xubuntu-restricted-extras")** if you are using xubuntu. And install [kubuntu-restricted-extras]("apt://kubuntu-restricted-extras"), if you are using Kubuntu.


Also, I recommend you to visit the [Medibuntu]("http://medibuntu.org/") website where it explains about media formats and how to install them.

[https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)
