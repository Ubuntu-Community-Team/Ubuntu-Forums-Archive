### Post by kvarley on 2009-02-27
Urban Terror is a fast-paced first person shooter (FPS), with good graphics and amazingly addictive gameplay.

#First goto the downloads page:[http://www.urbanterror.net/page.php?6](http://www.urbanterror.net/page.php?6)

#Then simply find the section on the page where it says 
"The .zip installer for mac and linux users", select your download location and get the archive.

#Unzip.

#Then when in the directory you have downloaded, select the file which suits your architecture.

Linux 32bits executable: ioUrbanTerror.i386
Linux 64bits executable: ioUrbanTerror.x86_64

#Make sure you allow the file to be executed as an application by right-clicking on the file and going in to "Properties" and then into the "Permissions" tab, tick the box and hit Ok.

Now just double click and it should load! Enjoy!
