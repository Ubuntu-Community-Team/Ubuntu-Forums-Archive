### Post by 5nak3 on 2009-06-26
Ok the problem is most likely that you are running compiz (fancy desktop effects)

Click applications -> add/remove -> type in "compiz fusion icon" select the program and install it.

Now before you play the game try this:

Click applications -> system tools -> compiz fusion icon -> the compiz icon should load up in the top taskbar.

From there

Right click the newly appeared icon -> select window manager -> select metacity and you should be good to play without any minimizing.

If you want compiz back after, just right click the blue compiz icon again -> select window manager -> select compiz -> right click the icon again -> reload window manager

And you're done.

Hope that helps
