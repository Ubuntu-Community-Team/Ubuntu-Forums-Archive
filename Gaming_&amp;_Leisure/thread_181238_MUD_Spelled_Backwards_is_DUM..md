If you're looking for MUDs, [http://www.mudconnect.com](http://www.mudconnect.com) is a good place to look--it has a huge database of MUDs which you can search by categories and features.

Personally, I used to play quite a few MUDs, especially Star Wars MUD and the Unofficial Squaresoft MUD (UOSSMUD)... Now, I've kind of gotten away from playing any, though I do log onto UOSSMUD from time to time to gain a level here and there, but mostly for the people there.

UOSSMUD has a very unique system of advancement, based on the system from Final Fantasy Tactics. You can find it at uossmud.sandwich.net, port 9000. If you decide to try it out, my character's name is Gunn on there, and feel free to ask me about anything whenever I happen to be on.

### Post by Sye d'Burns on 2006-05-25
[MudMagic Mud Client For Windows & Linux]("//http://www.mudmagic.com/mud-client/") - (This is the client I recommend for Linux at least)

Current Version: 1.8 - February, 2006

The Mud Magic Mud Client is written in gtk and runs on Windows, most Linux distros and MAC OS X. The source code is free for download and open source. Our sourceforge page contains the project backend for developers. The client supports MXP, MSP, MCCP, and ZMP with PCRE regular expression handling and Python script engine. The plugins shipped with the client currently include an automapper, note system, and database front end. 

The client is modularly designed for expansion. From the start of the project development, the structure, design and layout of the source code has been created with expansion in mind. MudMagic Host services hires mud developers of the community to perform major upgrades to client. Developers that wish to make permanent modifications to the source can use CVS through our sourceforge page to submit recomended modifications.

Basic features include triggers, alias usage, linux and windows compatibility, per-session/per-game profile preference storage, global preference savings, and much more.

As for muds I'd recommend [mudconnect]("http://www.mudconnect.com") as Steggy suggested. Personally I like to play [Discworld]("http://discworld.atuin.net/lpc/").
