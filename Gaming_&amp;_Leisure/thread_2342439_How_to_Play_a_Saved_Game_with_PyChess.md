### Post by KenUBF on 2016-11-06
I've been playing chess the last several months with my dad who has been teaching me. We use PyChess and sometimes when we cannot finish a good game we try to use the save feature. We both save the .pgn game files that are created but when we try to load these games later it seems as if only one person is able to view the board as it was saved. For example, the first time we tried playing a saved game I loaded my game file and I could see the board with the pieces as we left them, but he got a fresh board, with all pieces in the starting positions. The opposite recently happened, where I saw nothing but a fresh board and he was able to view our last positions on the board. I must be doing something wrong. I'd love to be able to save games and play them later with this program. How do I do that? Thanks!

For a little more detail this is the process we both use when trying to load a saved game: We open the PyChess game window, click Game in the menu, Load Game, then choose the saved .pgn file that was created when we saved our games, a small window pops up that says Open Game, except for one of us it doesn't show our saved positions, just starting a new game.

### Post by oldrocker99 on 2016-11-07
This is a reported bug:[https://bugs.launchpad.net/ubuntu/+source/pychess/+bug/1571913](https://bugs.launchpad.net/ubuntu/+source/pychess/+bug/1571913)

The fix appears to be to download PyChess 12.4 from the PyChess download page::[http://www.pychess.org/download/](http://www.pychess.org/download/). The version you have is 12.2.1.

Good luck, and beat your dad!
