### Post by newfy78 on 2008-03-12
I installed Ubuntu 7.10 AMD 64 bit last week.  I've been a windows user my whole life and I have to say I am loving Ubuntu.

I installed Assault Cube and Urban Terror without any problem and I am able to run them....for 5 minutes.

After about 5 minutes both games change from full screen to a smaller window.  They are not minimized but change to a window that is about 1/3 the size of my desktop.

The game is still running but I can't move my mouse or use my keyboard.  I end up having to reboot to get out of it.

I don't think this is a problem with the game because it happens to both assault cube and urban terror.

Any ideas?

---

### Post by newfy78 on 2008-03-12
I found the answer.

Silly really.  My screen saver was set for 5 minutes and even though my screen saver didn't come on, it was causing the problem I described above.

I'm surprised that nobody knew the answer right away.

Anyway.  If anybody else has this problem, just set your screen saver to something higher (e.g. 2 hours).

As a side note I think it is pretty dumb that my OS can't recognize that i'm using my keyboard and mouse in the game (and therefore enables the screen saver)

Peace.

Newfy.
