### Post by ManOfLinux on 2011-05-24
If you install the virtual box guest os extensions in safe mode and check install 3d support (experimental)(thats not it exactly but it should be easy to see) you get 3d acceleration in windows in that virtual.
I would choose xp sp3 or micro xp for the os. I have tried tiny windows 7 but it was much slower even after I gave it access to 4 gigs of ram and all my processors. 
Btw if you have problems with the copy protection of your games choose the passthrough option for the dvd/cd drive under storage in the virtual box menu. That way the virtual machine gets direct access to the dvd, I dont know if it will work for all games but it worked for battlefield 2142. 
Mouse issues. Now some games you will not have this in(fate), but in BF2142 when you get to the screen where you choose your spawn point the mouse cursor goes away, but before that its fine. 
To fix this turn off mouse integration you will get your cursor back. the problem is when you are through playing and go back to the game menu, you loose your cursor again, so if you turn back on mouse integration you get a cursor. kinda of pain but the game is playable.
Also when I played a friends copy of "ho chi min trail" I had to turn down the mouse sensitivity in the game way down otherwise it would just jerk around. 
So there are some issues with FPS's but games like fate should be very playable. I think this will only get better as time goes on.
Thanks VirtualBOX!!!!!!!!
