### Post by skymt on 2006-11-11
Controls: Arrow keys move your ship, Z fires an energy beam, X performs an action that depends on the game mode. The top row of icons is the four different modes. Try them all and experiment to see what they're like.

By the way, Zerocool, please don't dig up old threads like this. It would have been a better idea to start a new thread.

Lag: I know this is a bit late, but there are a few ways to speed rrootage up:

* Use the -lowres option (run it as "rrootage -lowres"). It gives you a much smaller gameplay area, but a much higher framerate.

* Use the -accframe option. It enables an alternate frame management algorithm that may work better (or may not).

* Get better hardware. I know, it's obvious, but I said it anyway. :rolleyes:

### Post by Frem on 2006-11-24
It's supposed to slow down.
You can run "rrootage -nowait" to run at normal speed, but the game basically becomes impossible.

[QUOTE="man rrootage"]-nowait
When there are a large number of bullets on the screen, the game is  slowed  down. This option prevents this slowdown so that the game speed is constant.
[/QUOTE]
