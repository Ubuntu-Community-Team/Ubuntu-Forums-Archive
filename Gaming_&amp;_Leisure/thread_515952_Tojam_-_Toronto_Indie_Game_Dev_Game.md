### Post by ADT on 2007-08-02
Anyone here heard of the Toronto Game Jam. Its a 3 day game developement session where a group of people get together in different teams and make a load of cool little games. Then they put them up on the Toronto Game Jam website:

[http://www.tojam.ca/](http://www.tojam.ca/)

May 2007 has been the second tojam. And there are some great games. 

The top game this year was XiQ (It has a linux build for download available)

From tojam website:

> XIQ is a game about trapping. Your avatar can fire in 4 directions, creating lines. If those lines form a box, the badguys in the box are destroyed and you score points. Move with the arrow keys or joystick and shoot in all four directions with WASD. Once you fire a bullet, it creates a trailing line. This line forms the edge of a box. Use four lines to create a box, trapping triangles inside.

The weirdest game has to be Tinker, its about a Cat that needs to get oil over itself so it can swim in the sea and save dolpins???
It is a flash game so it plays in a web browser.

Tojam is definitally worth checking out.

It would be great if it happened in other cities as well.
