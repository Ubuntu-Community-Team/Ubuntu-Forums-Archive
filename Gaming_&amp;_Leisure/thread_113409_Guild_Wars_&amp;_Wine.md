### Post by qferret on 2006-01-07
If you buy Cedega, all you are paying for is:

1. binary install
2. Point2play  (GUI frontend...there are free ones available now too...Grapevine is pretty decent)
3. The ability to vote on what games to try supporting next

Other than that, the free CVS source is the same product.
When it comes to a product that depends on which way the wind is blowing, as to whether a program will install and run under it, I can't see spending $ on it. Supporting the work of the developers and allowing them to keep going is a "research grant", not a software license....
