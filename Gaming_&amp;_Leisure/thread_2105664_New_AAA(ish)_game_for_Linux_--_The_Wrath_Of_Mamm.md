### Post by pieman711 on 2013-01-16
I've finally managed to get my Game into a presentable state and ready for release. I hope I didn't raise too many hopes with the thread title. It is a simple 2d maze game a bit like chips challenge. 
I've made it my self and am entirely self taught at programming. It's take about 10 years on and off (although a LOT more off than on) and has gone through severl iterations starting in QBASIC with ascii art via blitz basic to finally a linux compatible C++ release with full graphics  and based on the Allegro 5 libs. 

Features:
8 levels included
Different difficulties
challenging game play
Colourful 2d graphics 
A map editor, which should be fairly easy to add new maps to your game

I made the engine myself and all of the graphics. The map editor is called Mappy and is really easy and fun to use.  

More details are available in the readme. 
Let me know if it works and any feed back will be appreciated greatly!

The game binaries and assets are available at:
[https://sourceforge.net/projects/thewrathofmamm/files](https://sourceforge.net/projects/thewrathofmamm/files)

The source code is available at:
[http://sourceforge.net/p/thewrathofmamm/code](http://sourceforge.net/p/thewrathofmamm/code)

The map editor "mappy" is available at 
[www.tilemap.co.uk]("http://www.tilemap.co.uk")
--this runs in Wine perfectly or there is an older version that can be compiled in linux. 

Enjoy :)
