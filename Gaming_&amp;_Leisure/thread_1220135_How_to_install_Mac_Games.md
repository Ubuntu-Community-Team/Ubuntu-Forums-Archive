### Post by damis648 on 2009-07-22
Linux is neither binary compatible with OS X nor is it API compaible, some BSDs are binary compatible but not API compatible. In any case, most Mac games are Windows games running in Cider (a modified version of wine, or more specifically darwine), so you would be better off going with Wine or Crossover or something.
