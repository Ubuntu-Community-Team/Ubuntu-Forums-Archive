### Post by leech on 2007-06-14
With how much activity Gens (Sega Genesis / Mega Drive / Sega CD / 32x emulator) is getting these days after megamaced had created some Ubuntu packages for the latest versions, I thought it'd be a good idea to start a list of other emulators that have lost a lot of their developers due to various reasons and get them going again.  I know this isn't linuxgames.com or happypenguin.org, but perhaps with the amount of traffic in here, we could get some of these projects some help.

I'll start a list and we can go down it and add to it.  

Virtual Jaguar [http://icculus.org/virtualjaguar/](http://icculus.org/virtualjaguar/)  Atari Jaguar.  As the URL implies, this is hosted on the almighty Icculus's page.  It looks like it's slowed down due to that evil thing called real life.

FM-TOWNS Bochs [http://web.archive.org/web/20051214222524/http://fmbochs.emuvibes.com/](http://web.archive.org/web/20051214222524/http://fmbochs.emuvibes.com/) this is actually a Windows emulator for the Japanese FM-TOWNS computer.  I couldn't find a Linux version, but it uses bochs and the source code is available, perhaps someone could port this.  It has a lot of patches for bochs, though it looks like it's for a rather ancient version (1.2.1, Gutsy has version 2.3)

Dega-sdl [http://www.emulinks.de/emus/](http://www.emulinks.de/emus/) Sega Master System and Game Gear.  The Windows version is at 1.12, the Linux version linked here is at 1.07.  Also according to this post [http://ubuntuforums.org/archive/index.php/t-450920.html](http://ubuntuforums.org/archive/index.php/t-450920.html) the source needs to be changed a bit to even compile anymore.

XSteem [http://steem.atari.st/](http://steem.atari.st/) Atari ST.  This emulator is really good, though we also have Hatari, which only recently got STe support.  XSteem is not open source yet, though when I was on the mailing list years ago, the author had said he intended to release the source code for it.  [http://www.freelists.org/archives/steem/07-2003/msg00039.html](http://www.freelists.org/archives/steem/07-2003/msg00039.html) a post from 2003 stating that they were going open source.....

Well, that should start us off.

Leech
