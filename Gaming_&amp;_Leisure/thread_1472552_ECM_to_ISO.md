### Post by Rustybolts on 2010-05-04
```
sudo apt-get install ecm
``` Install ecm
In terminal navigate to folder containing image then
```
unecm [COLOR=Red]ecmfile[/COLOR] [COLOR=DeepSkyBlue]outputfile[/COLOR]
```
