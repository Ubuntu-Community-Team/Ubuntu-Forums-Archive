### Post by zugu on 2006-11-21
Solved: the installation directory should have been the current directory before running "wine HEROES3.EXE". Also, I had to enter winecfg and tell it where my CD-ROM is. The game performs poorly under Wine, so I tried to run it under Cedega and it works just fine.
