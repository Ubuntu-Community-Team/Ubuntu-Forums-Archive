### Post by phoenix5002 on 2008-03-11
I used to play Tribes2 a lot, and I still hold it to be probably the best multiplayer game I've ever played.
So I was excited when I found this game:
[http://legendsthegame.net/index.php?m=content&pageid=21](http://legendsthegame.net/index.php?m=content&pageid=21)

It looks extremely similar to tribes2 and it runs natively in Linux and Windows.  It even has voice binds like tribes2!! and it allows you to select custom Tribes1 or Tribes2 bindings so wherever you come from you will be comfortable.  It's still in development so it doesn't have vehicles yet, and there is only male characters, but it's great to see some of the old maps again like dangerous crossing.
Really brings me back....

Anyway, it doesn't seem to be that popular yet.  But, I hope it gets as many players as Tribes2 had back in the day.

Just thought I would share it with everyone.
