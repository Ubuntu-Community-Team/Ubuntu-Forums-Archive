### Post by Sammi on 2008-03-13
You need to install new ATI drivers. All the info on the topic is found here: [https://help.ubuntu.com/community/BinaryDriverHowto/ATI](https://help.ubuntu.com/community/BinaryDriverHowto/ATI)

Easy method for doing it to use the Envy application for automatic download and installation. Very intuitive program to use: [http://www.albertomilone.com/nvidia_scripts1.html](http://www.albertomilone.com/nvidia_scripts1.html)

But please note that Envy is not officially supported, and there is no dear mommy, if it makes your system break.
