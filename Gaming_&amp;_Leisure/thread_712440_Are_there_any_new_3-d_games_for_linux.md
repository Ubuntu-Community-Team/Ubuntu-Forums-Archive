### Post by noerrorsfound on 2008-03-03
> **eragon100 said:**
> Urban Terror is closed-source i believe, but i'm not sure.
You're correct.
[http://www.urbanterror.net/e107_plugins/content/content.php?content.5](http://www.urbanterror.net/e107_plugins/content/content.php?content.5)
> Urban Terror can be freely distributed over the internet, unmodified, without charging for the product. **Urban Terror uses the Quake 3 SDK license. This means the game code (the .qvm's in zpak000.pk3) are closed source** and can only be distributed electronically over the internet, not on cd/dvd. The non-code data files of Urban Terror (zpak000-assets.pk3) can go on any medium however. The full license texts come with the installer, they outweigh the information provided here.
The only thing that's open source is the executable used in place of Quake 3's retail executable. For the executable, a modified ioquake3 is used.
