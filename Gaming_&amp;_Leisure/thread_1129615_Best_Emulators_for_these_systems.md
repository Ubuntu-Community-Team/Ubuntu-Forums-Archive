Hi There i use the follow emulators :

Mednafen : Nes , PC-Engine , Wonderswand , etc ( Ubuntu Repo )

Mednafenfe : Front-End for Mednafen ( [Site]("http://sourceforge.net/projects/mednafenfe/files/") )

Snes9x-gtk : Super Nes for Gnome ( [PPA BearOSO]("https://launchpad.net/~bearoso/+archive/ppa") )

Vbam : VisualBoyAdvance-Merge ( [PPA ~ripps818]("https://launchpad.net/~ripps818/+archive/ppa") ) , the executable is called "gvbam"

DOSBox : MS-D.o.s Emulator without gui / front-end( Ubuntu Repo ) 

GR-Lida :  Front-End for DOSBox , ScummVM , etc ( [site]("http://laisladelabandonware.iespana.es/proyectos.php?id=grlida"))

Gelide : Front-End for Emulator Very Util ( [Site]("http://gelide.sourceforge.net/") )

PcSX-df : Sony PlayStation 1 emulator ( Ubuntu Repo )

pSX : Sony PlayStation emulator , the one thing you must configure are the Input Controls , nothing more . ( [Site]("http://psxemulator.gazaxian.com/") )

ePSXe : Sony PlayStation Emulator ( [Site]("http://www.epsxe.com/download.php") )

Gens : Sega Master Drive / Genesis Emulator with GUI in GTK for Gnome ( [Site]("http://sourceforge.net/projects/gens/files/") )

Mupen64Plus : Nintendo 64 Emulator with GUI ( [Site]("http://code.google.com/p/mupen64plus/") ) 

lxdream : Sega DreamCast Emulator with GUI ) ( [Site]("http://www.lxdream.org/download.php") )

Yabause : Sega Saturn Emulator ( Ubuntu Repo )

sdlmame : Machine Arcade Emulator ( Ubuntu Repo )

sdlmess : Too much home system emulator , go and see the list of System Emulated , how : Nes , SuperNes , Sega , Sony , etc  ( Ubuntu Repo )

Basilisk II : Macintosh 68K Emulator ( Ubuntu Repo )

DesMuMe : Nintendo DS Emulator ( Ubuntu Repo )

Fceu + gfceu : Nintendo NES emulator + Front-End for GTK ( Ubuntu Repo )

Wine : Its not emulator , its an api , to use Windows Software under GNU Systems ( Ubuntu Repo )

PlayOnLinux / PlayOnMac : Front-End for Wine

Cedega / Cider : Modified Version of Wine from TransGaming Company , to use "Games" of MS Windows , under GNU Systems

ScummVM : Virtual Machine for Graphic Adventure Games , how : Broken Sword 1 & 2 , Day Of Tentacle , Sam & Max , etc ... ( Ubuntu Repo )
