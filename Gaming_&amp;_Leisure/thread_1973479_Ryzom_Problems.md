### Post by Erik1984 on 2012-05-05
Hi,

I can't help you with the 12.04 version specifically but I got Ryzom to work in 10.04 a few months ago by downloading the client from SourceForge.

- Get this file (it's large!) [http://sourceforge.net/projects/ryzom/files/ryzom_client.7z/download](http://sourceforge.net/projects/ryzom/files/ryzom_client.7z/download)
- Get 7zip, if you don't have it already: ```
sudo apt-get install p7zip
```
- Extract the client (now it's much larger ~ 6GB!)
- Navigate in the terminal to the folder that contains the file ryzom_client
- Type: ```
./ryzom_client
```
