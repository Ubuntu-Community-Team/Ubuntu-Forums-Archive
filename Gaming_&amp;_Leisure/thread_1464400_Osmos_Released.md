### Post by Perfect Storm on 2010-04-28
> **ELD said:**
> Forgive me but i don't quite know what pkgbuild is but if it contains the full game then that is illegal?

No, it do not contain the game. 
It helps to install it easy with two commands. Just drop the full game in a folder with the pkgbuild files, then;

makepkg
pacman -U <package>

Then everything is installed, dependencies, desktop icons and cli launcher done.

The demo version of the pkg automatically downloads the demo and installs it.
