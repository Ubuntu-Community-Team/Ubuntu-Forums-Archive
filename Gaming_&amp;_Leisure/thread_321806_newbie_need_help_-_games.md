### Post by scrooge_74 on 2006-12-19
go to [http://www.winehq.org](http://www.winehq.org) and check if your game can run under wine

You can install wine from the repositories of Ubuntu or the wine repository to have the latest version, sometimes one version may run the windows program and next one will give you trouble.

Read all the info on wine so you can make the necessary adjustments. Sometimes it involves luck, but most of the time you have to pay attention to error messages to be able to correct problems.

But not always you will have games working.

Good luck and have some fun
