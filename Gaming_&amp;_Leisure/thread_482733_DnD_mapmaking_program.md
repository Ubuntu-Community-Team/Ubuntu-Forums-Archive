### Post by jjbean on 2007-06-24
There is the program [Inkscape]("http://www.inkscape.org/index.php?lang=en"). It is a SVG graphic program. It could draw maps. It is in the repos. I just downloaded [AutoREALM]("http://autorealm.sourceforge.net/index.php"), and it installed and ran with wine. It did give some font errors on install, and I have not played with it enough to see if it completely works. If you want to buy one there is [Dundjinni]("http://www.dundjinni.com/"). It is written in Java, and from what I have read it will run in Linux, but you have to run the installer on a Windows machine and then copy the program over.
