### Post by Yohumbus on 2007-07-05
well I know that I have use the qt version and I never had any problems with it, so there might just be some small fixable bug there... but I don't know much about the problem since I've never seen it. The one thing I would have to suggest is that if you always have an internet connection while you play then you should try playing via telnet at nethack.alt.org You wont be able to use any gui but I have found that after just a couple of games you get used to the console (I used to only play gui). Even though you will be playing console at NAO I think it is much more enjoyable since you have a lot more bones files and the website can give you a lot of statistics about the game and your characters.
