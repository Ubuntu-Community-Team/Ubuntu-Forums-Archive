### Post by Gondar on 2006-08-07
He is not asking for a game with light guns (as in weight) :P. He's asking about light gun games (think Duck Hunt for the NES).

As far as I know you should be able to use the Actlab Light Gun [http://www.act-labs.com/products/gun1.htm](http://www.act-labs.com/products/gun1.htm) on Linux (as it emulates a regular mouse, can anyone confirm this?). You could also check lik-sang and see if they have any light guns.

MAME has light gun support so you should be able to play some wicked arcade games. FCEU also have light gun support (NES emulator) and possibly others.
