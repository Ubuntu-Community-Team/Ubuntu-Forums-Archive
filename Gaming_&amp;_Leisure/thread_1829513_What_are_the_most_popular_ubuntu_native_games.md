### Post by drawkcab on 2011-08-20
I like Wesnoth and FreeCiv for turn-based strategy.

Urban Terror is no counterstrike but it can be fun sometimes.  

Netrek is probably one of the oldest network multiplayer games still in existence but it is really fun.  Hard to find players these days but the old regulars still show up from time to time.  Steep, steep learning curve.  Teamplay is a must.

Check out play deb where you can find some titles not available in the repos:

[http://www.playdeb.net/welcome/](http://www.playdeb.net/welcome/)
