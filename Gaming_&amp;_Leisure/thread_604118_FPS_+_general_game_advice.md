### Post by Rhubarb on 2007-11-05
Assault Cube would be perfect.  It's slim, runs on slow gfx hardware, and always has some people playing on line all the time.
If your ping is 300ms the game is still very playable, as it uses a lot of client side prediction.
[http://assault.cubers.net/](http://assault.cubers.net/)

If you've got a bit more gfx processing juice, then try Sauerbraten
[http://www.sauerbraten.org/](http://www.sauerbraten.org/)

These two games are completely open source FPS games.
