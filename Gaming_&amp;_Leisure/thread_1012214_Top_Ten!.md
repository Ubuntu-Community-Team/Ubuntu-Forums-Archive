### Post by spy_1134 on 2008-12-15
I can't give you a top ten but I can recommend a few games to you.

Battle for Wesnoth: A turn based strategy game
> sudo apt-get install wesnoth

I have no tomatoes: try to blow up other fruit wih bombs
> sudo apt-get install tomatoes

Open Arena: First person shooter
> sudo apt-get install openarena

Gweled: Bejewled/Diamond Mine clone (Menus are partly in french)
> sudo apt-get install gweled

Frets on Fire: Requires a fast pc. Guitar hero type game
> sudo apt-get install fretsonfire

Frozen-Bubble: Shoot bubbles that are the same color
> sudo apt-get install frozen-bubble


Thats just to name a few. Check out the ubuntu community help page for more games. [COLOR="Blue"][Click here for Community Help page.]("https://help.ubuntu.com/community/Games")[/COLOR]
