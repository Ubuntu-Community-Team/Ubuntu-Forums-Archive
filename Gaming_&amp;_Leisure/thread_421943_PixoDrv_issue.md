Hi there,

I had this same problem.

To fix it rename the hidden folder in your home directory named .armyops250 to .armyops250OLD.

Once armyops loads, it will see that it is not there and create it.

I found this information at: [http://forum.americasarmy.com/viewtopic.php?t=256955](http://forum.americasarmy.com/viewtopic.php?t=256955)
