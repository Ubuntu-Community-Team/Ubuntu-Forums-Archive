### Post by Vadi on 2009-04-06
Ubuntu should work (I have it on a similar laptop), but Xubuntu (different from ubuntu, designed for low-end machines) would work better.

You can install windows games in ubuntu without virtualbox, you can use wine for that. Not all games are compatible though - you can check the list of them here: [http://appdb.winehq.org/](http://appdb.winehq.org/)
