I take issue with this, and I apologize if it has been addressed, because I didn't read much past it.

This statement shows you really don't know what you're talking about. ID Software is hardly out of business, and they've released quite a bit of source code to their games.

Case in point is Doom. It is rereleased every year with a new package, using the same old original wads and more often Doom95 every time. That's not counting the numerous official ports made and sold.

There are a number of source ports available freely, and even enough freely released material that you don't even need to buy the original versions. Heck, the shareware version of Doom1 is still freely available, and all you need is that .wad file to play KDitD. By releasing DooM's source, ID gave it the ability to survive past 1994. It has been freely ported to nearly every system not officially ported, and a number of source ports have built it up to the point that it's no longer even remotely similar. Mouselook, jumping, crouching, etc... things that came afterwards as FPS games evolved are now freely available.

Doom still sells. Millions of copies later, after the source was released, it sells. 

Releasing the source freely is not the same as releasing the game freely. The quoted statement proved you don't understand that fact. The artwork, levels, weapons, monsters, and all other content of Doom is still commercial, save that in the shareware .wad (which is freely released). Furthermore, it still sells quite well for a game that's seen some 14 years on the market, 14 years straight without a single time that you weren't able to purchase a legal copy. 

It's longevity is due to the fact that it is open sourced. Bugs have been fixed, features added, online play expanded and kept active, and mappers able to make new maps that far exceed the original engine's capabilities. If all we had was Doom95, Doom would be a curiosity from the past, not a raging commercial success.   

While I never really left Doom myself, Quake seems to have followed that same formula. I have no doubt that Doom3's engine will be released eventually as well, and Q4, and so on. It's a proven successful tactic. 

Alternately, you can look at Baldur's Gate, a closed source game. It was rereleased for a while, but is now just another old title. It's fanbase has moved on, save a few diehards who stick with every game series. The only thing it has going is an attempted free, open source implementation of the Infinity Engine. And I lost track of that project years ago. 

Source code is not the same as an entire game. Releasing the source to a popular game is a good way to keep interest alive. It's worked well for Id software.
