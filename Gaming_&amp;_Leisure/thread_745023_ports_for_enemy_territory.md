### Post by russ18uk on 2008-04-04
Outgoing depends on the server (27960 is default) so on the server list you might find any number of ports since some hosts will use multiple "servers" (more like dedicated servers) on each machine/net connection. Incoming it's 27960 if you want to leave it default.

Should say that 27960 is for virtually all Quake 3-engined games. I think MOHAA was different but can't remember. Serious Sam, Quake 3, RtCW/ET, CoD should all use the same default port.
