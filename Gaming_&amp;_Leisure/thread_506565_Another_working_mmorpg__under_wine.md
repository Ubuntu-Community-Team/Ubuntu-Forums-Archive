### Post by Ripfox on 2007-07-21
Edit: I found two mmorpgs that works under wine with a little tweaking.

[http://www.helbreathusa.com/index.php](http://www.helbreathusa.com/index.php)

Helbreath: Heldinian was one of my favorite windows mmorpgs, so I tried it with wine and got it working with these winecfg settings:

Windows Version = XP
Graphics = uncheck EVERYTHING and set vertex shader support to none (uncheck "allow pixel shader" too!
Audio = ALSA, 
"hardware acceleration" = FULL
Sample Rate = 44100 
Bits Per Sample = 16
Uncheck "driver emulation"

ONE ISSUE: when you click the skills tab (looks like a hand in the lower right hand corner)
I have experienced an extreme slowdown.
SOLUTION: use the hotkey f8 to check on skills then hit it again to return to game at normal speed.
ALSO: create your account  from the website, NOT the in game "new account" tab at login screen. 
This seems to freeze the client.
There is two weeks free then 9.95 a month

Second one: [http://gekkeiju.kicks-***.net:8090/homepage/](http://gekkeiju.kicks-***.net:8090/homepage/)

(ok the forum wont let me post the link because of the word *** (alternate the word for a donkey in the url)
gee I hope this doesn't get me in trouble here :)

DISCLAIMER: This game or at least the creators seem a little sketchy...don't know about some of the game content, and there is some vulgarity to say the least. If anyone sees anything else messed up about the game content, let me know. 

This should work with the same settings as Helbreath, but when you are done installing it drops a file on your desktop called "Gekkeiju Online patcher" just double click this file and it will patch fine.
I set the resolution to the same as my desktop, put it in directx 9 mode, checked fullscreen, checked broadband, and the game works great.
This one is free, as far as I can tell.
