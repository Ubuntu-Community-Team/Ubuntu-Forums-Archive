### Post by wahman143 on 2006-07-19
> **firezip said:**
> Well how do you? I installed Ventrilo and now I want to know how I can run it.
Find your wine "windows" directory (on my Gentoo box, it's /home/user/c/)...using mine as an example, I do like such:

```

$ cd ~/c/Program Files
$ wine <name_of_prog>

```

Wine does NOT always make icons for the progs.

HTH,
W.
