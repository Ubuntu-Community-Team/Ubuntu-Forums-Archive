Nah don't use Godot. GDScript is ugly, hard tolearn, and used only in Godot, not only that but engine is not as good as Unity.

You can run Unity on both Windows and Linux and it also can compile games to both of these platforms. Learning from [documentation]("https://docs.unity3d.com/Manual/index.html") didn't work for me and may be a bit hard so I'd suggest you to use any [tutorial]("https://gamedevsuccess.com/development/coding-role-playing-game/"). They are much easier to follow. [YouTube]("https://www.youtube.com/results?search_query=unity+tutorial") is also great source of tutorials. Honestly you choosing anything else than Unity in 2020 is a mistake. This engine is just awesome and very easy to learn.

I've made great things in Unity and im looking to go back to it when I have time.
