### Post by oldrocker99 on 2013-07-15
No, I think it's the current Source engine. I had had some major graphics problems (the Valve logo was a pink rectangle, and the text was unreadable) until I found this fix:

sudo [your favorite text editor] /usr/bin/steam

Add this line anywhere that doesn't start with a #:

export MESA_GL_VERSION_OVERRIDE=2.1

Restart Steam and the problem is fixed. This is on Steam's end; it had a problem with Mesa.
