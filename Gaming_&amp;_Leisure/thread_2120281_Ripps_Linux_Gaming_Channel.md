### Post by ripps818 on 2013-02-26
Hello Ubuntu Lovers! 
I've started a [Youtube channel]("http://www.youtube.com/user/ripps818") with a specific focus on playing games on Linux, trying to prove that gaming on Linux is not only possible, but fun. 

The channel is still pretty new and I only have a few videos of myself playing *Minecraft: Feed the Beast* and *FTL: Faster than Light*, but I plan on expanding onto other games in the near future. I have a bunch of games that I've gotten from the Humble Bundles and I'm probably also going to do some games that work well in Wine (mainly because Linux doesn't really have a huge selection of popular games yet). Not to mention any of the Valve Steam games as they get released. I'm already working on my recording setup for TF2, I just got to tweak GLC for some performance issues.

I plan on doing a tutorial on how I've setup my recordings with GLC and I already have a github repo that I've setup with all the helper scripts I've written to make video recording easier (I'll release the link when I'm sure that all the scripts are working properly and have some halfway decent readability comments in them).

I just wanted to share and hopefully get some feedback on how it's going. I don't have very many viewers right now, and I'd like to know what I'm doing right, what I'm doing wrong, and how I can improve. If you like what you see, please leave a comment, subscribe, and/or share.

[http://www.youtube.com/user/ripps818]("http://www.youtube.com/user/ripps818")

Thanks!
