### Post by dorath on 2007-07-07
Previous threads in this forum have suggested adding the -opengl flag when starting the game.  Have you tried doing so?

For example:```
wine "[path to warcraft3 launcher]" -opengl
```
If you haven't tried that flag, please try.  If you have, please let us know.:D

Also, cogdah has some useful links and tips on [this thread (link)]("http://ubuntuforums.org/showthread.php?t=493618").
