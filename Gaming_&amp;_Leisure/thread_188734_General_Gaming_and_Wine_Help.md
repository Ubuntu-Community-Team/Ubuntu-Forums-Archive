### Post by eqisow on 2006-06-04
Well, I'm afraid that according to the [Wine Application]("http://appdb.winehq.org/appview.php?versionId=4768") database, Rogue squadron isn't going to work anyway.

As far as your problem goes, I can't say that I've ever encountered it before. What version of Wine are you running? Perhaps running the latest version would help. You can see the [Wine wiki]("http://wiki.ubuntu.com/Wine") for that.

The other problem I could see is that the games are checking to see if DX is installed and that check is failing, even though Wine has DX compatibility. Does the install give an option to install Direct X? You could try doing that to get the games to bypass the check.

*Disclaimer: The above solutions are guesses.* ;)
