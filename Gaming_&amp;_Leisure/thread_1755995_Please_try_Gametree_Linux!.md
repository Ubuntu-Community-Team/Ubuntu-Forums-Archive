Personally I see no big reason to use gametree.

I registered a while ago for pure interest. If you check out their documentation it becomes obvious that the Gametree only targets Intels CE4100 platform in order to run games on low spec multimedia boxes.
Mainly OpenGL ES stuff focused on PowerVR (you might want to google after Intel CE4100 yourself).

I guess the mostly targeted gametree developer / user is someone who wants to port a casual game with relatively low requirements to the CE4100 (for example something like World of Goo, Angry Birds and stuff like set).

Beside the fact that the Cedega 3D code is **very** old, they are now even more limiting 3D features by focussing on PowerVR / OpenGL ES stuff.

Unless you really are aiming to port any game to CE4100 downloading gametree is a mayor waste of time.
