### Post by Kirboosy on 2013-04-04
Hi there!

The type of file that you are looking to download is the [minecraft.jar]("https://s3.amazonaws.com/MinecraftDownload/launcher/minecraft.jar") file. You will then need to make sure that you have [java ]("https://help.ubuntu.com/community/Java")installed on your system. Once you have done that you can run minecraft!

I'm not sure how familiar you are with the terminal in Ubuntu. So I will assume you need a full walk through.

```
cd Downloads
```
```
java -jar minecraft.jar
```
