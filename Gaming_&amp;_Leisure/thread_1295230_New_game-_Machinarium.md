### Post by sonikmh on 2010-03-04
Hello,

you can use exe2swf inside linux, there is a linux binary inside the archive for that.

Just extract the downloaded exe2swf archive somewhere, get inside folder where it is extracted (= where the file 'exe2swf' resides) with terminal and simply run this command:

```
./exe2swf <path to *.exe file>
```

I've just tried that and it's working fine - I can run full Machinarium in a browser! :-)

### Post by kalos on 2010-08-19
> **pornee said:**
> Can you install 32 bit on 64?

For anyone who picked this up recently, I've had some problems on 64-bit. It works fine when I run the game the first time. After I save and quit that first time, every time I run the game I just see a black screen.

The problem is that the game requires you to run it from its installed directory. So instead of making a main menu entry pointing to the executable, I made it point to the following script:

```
#! /bin/bash

cd /home/$USER/data/apps/games/Machinarium
./Machinarium

```

(You'd have to update the path and chmod u+x the script.)

If you played really far the first time, then you can rescue your saved games by looking in ```
~/.macromedia/Flash_Player/#SharedObjects/*/#localWithNet/
```

Move #localWithNet/home/.../Machinarium/ to #localWithNet/Machinarium/
