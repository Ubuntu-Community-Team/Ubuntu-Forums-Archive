### Post by jimwilk on 2008-04-08
> **AFOH said:**
> Gweled is a nice clone of Bejeweled, but I'm still trying to get Zuma running on Wine, because I couldn't find any clone.

I know this trick of mine works with Klikit (a Kubuntu derivative) so it might well work with Ubuntu too.

When I used Gweled with PCLinuxOS, the high score was recorded. However, in Klikit, the high score could not be shown. But, I found in /var/games a file called gweled.easy.scores. Using sudo type privileges, i right clicked on that file and then Properties. I changed the User to my username instead of Root. Heh Presto, when I play Gweled, the high scores are now recorded under Klikit.
