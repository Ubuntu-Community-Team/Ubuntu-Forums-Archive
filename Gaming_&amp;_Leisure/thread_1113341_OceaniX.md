### Post by -=Andru=- on 2009-04-01
This game is a free Linux version of the original windows version from SunRay Games.

Features:
- Magnificent underwater world
- Cool power-ups to acquire
- 30 puzzling levels
- Binaries for x86 and x86_64
- Nice graphics and music
- Multilanguage support (EN, DE, FR, ES, RU, UA), using default language of your system

[Download link](http://www.sunraygames.com/files/dl/OceaniX.zip)
