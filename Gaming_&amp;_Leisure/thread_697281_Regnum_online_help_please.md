### Post by nille on 2008-02-15
Just place the file rolauncher in the directory you would like Regnum to be installed into. Then make sure the file is executable (I think it already is):

[B]cd /your/regnum/install/directory/
chmod +x rolauncher[/B]

And then start the launcher by typing:
**./rolauncher**

If you get a lot of output ending with a (core dumped), try this instead:
**MALLOC_CHECK_=0 ./rolauncher**

Also, make sure to check out the Linux-part of the Regnum forum:
[http://regnumonline.com.ar/forum/forumdisplay.php?f=15](http://regnumonline.com.ar/forum/forumdisplay.php?f=15)
