### Post by Tedd81 on 2007-11-06
I have just installed hexen 2 but my graphics card is letting me down.
you will need .pak files from at at least the demo version of the game.

hexen 2 is avaliable and supported from [http://uhexen2.sourceforge.net/](http://uhexen2.sourceforge.net/)

Here is how i installed it:

- install alien using Synaptic (just search for alien , 8.68 is the version i'm using)

- from [http://uhexen2.sourceforge.net/download.html](http://uhexen2.sourceforge.net/download.html) download the hexen 2 .rpm    ([http://prdownloads.sourceforge.net/uhexen2/hexen2-1.4.2-1.x86_64.rpm?download](http://prdownloads.sourceforge.net/uhexen2/hexen2-1.4.2-1.x86_64.rpm?download))
to your home folder.

- open a terminal navigate to your home folder and convert the .rpm to a .deb. 
  sudo alien -d hexen2-1.4.2-1.x86_64.rpm 

- in your home folder there should now be hexen2-1.4.2-1.x86_64.deb. install this  how you please. ( i just double clicked)

you should now have Hexen2 installed but  missing the .pak file. You will need to obtain file pak0.pak  from either the installation cds or or lots of googing. I copied the file from my windows machine that is running hexen demo.
Once you have the file move it to:
/usr/games/hexen2/data/pak0.pak 

You should now have a running version of Hexen 2.
Aplications>games>hexen2 or hexen2 from a terminal.

hope this is of help
