### Post by Sealbhach on 2009-02-15
> **earthpigg said:**
> doesn't forcing architecture mean that it cannot be uninstalled via synaptic?


Yes, I believe so I can't find it in Synaptic. But you can uninstall from the command line. First, find the installed name of the application:

```
sudo dpkg -l | grep world
```

Brings up:

> ii  worldofgoodemo                       1.40    A physics based puzzle/construction game


So you uninstall by using the -r option and the installed name:

```
sudo dpkg -r worldofgoodemo
```


.

### Post by ploum on 2009-04-22
> **crazyfuturamanoob said:**

Not really. You can use them without 3D hardware but it will be software rendered (using MESA) thus you will have bad performances and it will affect the global responsiveness of the system.

For small effects, this is not a real issue because processor are fast enough.
