### Post by ColdFFF on 2009-11-04
[DirectX]("http://en.wikipedia.org/wiki/DirectX") is a set of Windows APIs specifically designed for handling multimedia tasks like gaming.

The closest open api is OpenGL, but other alternatives exist (see the wiki link here [http://en.wikipedia.org/wiki/DirectX#Alternatives](http://en.wikipedia.org/wiki/DirectX#Alternatives) )

As you (hopefully) already know, Ubuntu is not Windows and therefore does not natively run Windows apps, including games. The only way to get around this is to use WINE - a Windows Emulator. WINE does emulate DirectX as well.

To check if your game will run under WINE, check the [AppDB at WineHQ]("http://appdb.winehq.org/").

Refer to the [Wine Forum]("http://ubuntuforums.org/forumdisplay.php?f=313") for more help on that though
