### Post by talknerdytome on 2008-08-27
Sure...it took me a while to figure out too. I've only been using Ubuntu for a couple of weeks. Let me backtrack here and see if I can remember what I did...

I'm going to assume you're new to Linux too since I'm not sure how much you do/do not know, so if I insult your intelligence here I apologize :)

You'll want to download the "2.955 MFH-Alarian Full Release" first. There is a version for 32 bit and 64 bit. You'll also need unrar to decompress the file, so in the terminal run "sudo apt-get install unrar" if you don't have it already. 

Extract the 2.955 file (into your home directory is fine, it will make its own folder). Now navigate to the new Alarian folder you have, there should be a FretsOnFire file in there with no extension, open that. (if you'd rather do it from the terminal just cd into the Alarian folder and run ./FretsOnFire). 

You should be able to run the mod at this point, but that version is a bit buggy. Go back to [the link]("http://www.fretsonfire.net/cgi-bin/ikonboard.cgi?act=ST&f=11&t=24664&st=3160#entry285528") and grab 2.956. Once you download and extract this file, just copy all of its contents and paste them into your Alarian folder, tell it to replace. Then go back to that page again and download 2.963 and do the same thing, extract and copy into the Alarian folder. 

Run FretsOnFire again and make sure everything is operational. You should be up to date now. Rock out :guitar:
