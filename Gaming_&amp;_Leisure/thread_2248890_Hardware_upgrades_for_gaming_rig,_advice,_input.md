### Post by dannyboy79 on 2014-10-18
physX in general in linux isn't really all there. in fact it was only just 1 week ago friday that the newest physx core sdk was released for linux. Borderlands 2 and TPS both use an older custom version of physX and even with my GTX 760 the game still only uses CPU NOT the GPU for PhysX.

If you are wanting to upgrade your machine for gaming in linux I would highly highly recommend an nvidia card, the 750Ti or higher.
