### Post by Grey Dog on 2007-05-10
I really enjoy AlephOne/Marathon Infinity for a shooter.
There is always a good gang of people online to play with.

--DOWNLOAD:
[http://trilogyrelease.bungie.org/](http://trilogyrelease.bungie.org/)              Marathon Infinity
[http://sourceforge.net/projects/marathon/](http://sourceforge.net/projects/marathon/)           AlephOne 

--ADDITIONAL PACKAGES:
sudo apt-get -y install libsdl1.2-dev libsdl-mixer1.2-dev libsdl-net1.2-dev libboost-dev

--STANDARD COMPILE of AlephOne
./configure
make
sudo make install

--COPY MARATHON FILES INTO A1 DIRECTORY
cd /usr/local/share/AlephOne
sudo cp -r ~/Desktop/Marathon/* .
