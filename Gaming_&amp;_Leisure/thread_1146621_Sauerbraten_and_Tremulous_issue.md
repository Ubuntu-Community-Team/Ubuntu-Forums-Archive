### Post by Melcar on 2009-05-02
Are you using desktop effects?  If so, try turning them off.  This is a bug relating to the window manager and some video drivers.  It happens to me in certain games on my machines running Xubuntu + Compiz + open source ATI drivers.  Besides turning Compiz off, setting the game to a lower resolution also fixes it for some reason.
