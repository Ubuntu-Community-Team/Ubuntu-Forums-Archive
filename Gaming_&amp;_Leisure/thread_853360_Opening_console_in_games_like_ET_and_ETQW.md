### Post by mriedel on 2008-07-12
> **Akingboy said:**
> Its the button one left from number 1.

That button is probably being treated as a dead key, allowing you to type stuff like ê. 

Go to System -> Preferences -> Keyboard, then click on Add and add the keyboard layout of your choice using the "Eliminate dead keys" variant. Don't forget to set it as default afterwards.

You won't be able to type é, è, ê, î etc. using dead keys any more though.
