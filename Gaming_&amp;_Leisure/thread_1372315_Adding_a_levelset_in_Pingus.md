### Post by marn on 2010-01-04
As Pingus isn't currently being worked on (and my kids love playing it) I have been looking into how to add levelsets.

There are over 150 levels in the pingus file - some work, some may not. A way of accessing them is here:

[http://ubuntuforums.org/showpost.php?p=4306694&postcount=6]("http://ubuntuforums.org/showpost.php?p=4306694&postcount=6")

but I like an "in game" solution (the above solution only allows one level to be played at a time)

Adding a levelset in /usr/share/games/pingus/data/levelsets/ will enable levels to be enabled in the game, meaning they have to be solved in order. (That is where the halloween levelset ist)

Either you can add them all using this method: 

[http://forum.ubuntuusers.de/topic/pingus-levels/#post-1119274]("http://forum.ubuntuusers.de/topic/pingus-levels/#post-1119274")

copy the code into gedit and save as a *.levelset in the above file, or make a selection. The desert levels seem to be solvable (I haven't done them all myself) as probably most of the levelfiles in /usr/share/games/pingus/data/levels/playable are.

Below is a levelset that I created for my chilren. Extract it and save as root in the levelsets file (/usr/share/games/pingus/data/levelsets)

There is also a volcano worldmap, but I haven't tried to get that working yet.

Marn
