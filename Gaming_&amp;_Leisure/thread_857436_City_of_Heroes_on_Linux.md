### Post by NovruzeliH on 2008-07-12
1 make a copy of your CoH directory (ex. C:/Program files/City of Heroes/) from your Windows Install. if you have different computers burn it to a DVD-R or couples of CD-R's. If you dont have a windows install around dont worry there is another way.
BURN City of Heroes FOLDER TO DVD

2 make sure you have set the Nvidia Drivers installed. Link to NVidia Install

3 Download cedega_4.4.1-1.i386.tgz into /usr/portage/distfiles/ Download transgaming-fontinstaller_1.0_i386.tgz into your Home directory. Download transgaming-mozctlinstaller-1.0-1.i386.tgz into your Home directory.

4 install cedega, type in console emerge cedega

5 create enviroment

6 copy COH to the linux instalation 1. Navigate to Places--> Home Folder--> TransGaming_Drive --> Program Files. 2. Right click and select 'Create Folder'. Name it "City of Heroes" -without the quotes. 3. If you have it all on a DVD-R Copy the entire contents to the City of Heroes folder and move on.

or Download [CohUpdater.zip]("ftp://client.coh.com/US/CohUpdater.zip") and extract it your Home directory. Then run cedega /home/YOUR_USERNAME/desktop/cohupdater.exe
