### Post by Irritant on 2010-12-16
It's out!  Please read the README and INSTALL files for how to install the game.  We have made some changes that should be for the better for Linux users in getting the game properly installed and running.

Here is the official press release:

COR Entertainment has released Alien Arena 2011 for Windows and Linux/Unix.

The latest version of the open sourced, freeware deathmatch shooter features a host of updates and features, including ragdoll physics, revamped in-game IRC client, true type font support, several new maps, a host of bugfixes and tweaks, and new music by renowned musician Paul Joyce. We have also implemented the use of AutoTools for easier and more consistent Linux/Unix/MacOS compilation.

Some of the new features include: 

[list]
[*]Ragdoll physics using the Open Dynamic Engine
[*]GNU AutoTools implementation
[*]Revamped in-game IRC client
[*]True Type Font support
[*]Two brand new maps, one revamped map
[*]Player models and skins updated
[*]New music by Paul Joyce
[*]Numerous bugfixes, tweaks, and optimizations
[/list]
