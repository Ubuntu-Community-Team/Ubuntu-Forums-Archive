### Post by Rytron on 2009-01-14
***Solved:***

**Vice C64 Emulator**
_To Start A Game Normally:_
File
Smart-attach disk/tape
Select Game(game does not need to be in a zip file)
**Autostart**

_To add second disk/tape(when you are asked on screen):_
File
Smart-attach disk/tape
Select Game
**Open**
