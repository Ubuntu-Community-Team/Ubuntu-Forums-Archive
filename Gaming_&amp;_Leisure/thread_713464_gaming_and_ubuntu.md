### Post by justin whitaker on 2008-03-02
I'm torn in answering this.

There are tons of "Get X game running on Ubuntu" threads in the gaming section. A quick search would have found that. 

Next, would it kill you to actually use, you know, real spellings in your posts? There is no way anyone is going to take gaming on Ubuntu seriously if our posts make us look like we can barely communicate.

I'm a gamer, and I can actually use actual sentences and spelling to express myself. You should try it. Quite liberating. 

The actual answer to your question is that with the right drivers and a little bit of work, Counterstrike:Source will run on Ubuntu. It runs on Wine, Crossover Office, and Cedega...although I have found that Steam games work the best overall on Crossover office. 

Ventrilo can also run, but you need to pass a command to wine via the terminal to make sure that you have sound in both the game and in Vent.
