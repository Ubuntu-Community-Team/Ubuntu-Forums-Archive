### Post by EuclideanCoffee on 2020-02-22
This is a licensing issue. The mod simply renders the Strife game data to your computer. What you are replacing is the engine for Strife, but the developer does not provide the game data for you to play without purchasing the game and using its game data.

So you will need to go to Steam, purchase Strife, and then use the game data to play the game on strife-ce. I'm sure that's probably not the answer you wanted. Sorry.
