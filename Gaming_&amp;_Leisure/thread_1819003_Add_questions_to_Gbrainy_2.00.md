### Post by R33D3M33R on 2011-08-05
You could edit 
```
/usr/share/games/gbrainy/verbal_analogies.xml
```or

```
/usr/share/games/gbrainy/games.xml

```For help with editing XML files see this: [http://www.w3schools.com/Xml/xml_syntax.asp](http://www.w3schools.com/Xml/xml_syntax.asp)
