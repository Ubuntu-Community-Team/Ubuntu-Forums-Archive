### Post by der_joachim on 2006-08-24
I have been tinkering about a bit. I have Guild Wars running at this point, albeit with a few issues. First, the music plays at double speed. Furthermore, I hear no sound effects. Second, my mouse just disappears. It does work though.

I can run around in cities and explorable areas. I have not tried any missions yet. Chatting is possible as well. Fighting is possible, but not very easy without a visible mouse pointer. 

My specs:
- Athlon X2 4200+
- 1 GB of RAM
- NVIDIA GeForce FX5600 with 128 Megs of RAM. Latest official drivers from the repositories.
- NForce 3 chipset with AC97 compatible sound chip.
- Wine version 0.9.19, compiled by hand, using the Ubuntu guide at WineHQ.

Settings:
- Windows 98 emulation
- Allow DirectX apps ... Checked
- Enable desktop Double Buffering ... checked
- Allow window manager... NOT checked
- Vertex shader support ... Emulation
- Pixel shader support ... checked
- Sound driver ... ALSA
- Hardware accel ... Standard
- Bitrate ... 44000
- Bits per sample ... 16
- Driver Emulation ... Checked

Feel free to experiment with these settings and please share your experiences. :D
