### Post by mister_k81 on 2013-05-29
what type of game do you want to make? If you are looking at making a 2D platformer or 2D overhead game, I suggest trying out MelonJS: [http://www.melonjs.org/](http://www.melonjs.org/) with the Tiled map editor: [http://www.mapeditor.org/](http://www.mapeditor.org/)

This is an HTML5 framework that uses java script coding, so it works in just about every browser, regardless of the platform. And you can make some pretty nice things with it. There is a basic tutorial here: [http://melonjs.github.io/tutorial/](http://melonjs.github.io/tutorial/) and the Tiled map editor has a Linux build. Check out some examples here: [http://www.melonjs.org/gallery.html](http://www.melonjs.org/gallery.html)
