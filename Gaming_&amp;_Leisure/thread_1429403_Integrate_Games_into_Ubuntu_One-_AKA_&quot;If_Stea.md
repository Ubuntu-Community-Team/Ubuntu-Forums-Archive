### Post by Ellipsis on 2010-03-14
Canonical is already planning to sell software through the Software Center in Ubuntu 10.10 and this naturally includes games. So with all the gamers out there demanding Steam for Linux: Canonical might already be quenching that demand.  But what about Steamworks? 

Steamworks allows features such as achievements to be added to games and allows the use of the "cloud" for save data. These features add a lot of value to games and in the case of achievements make one consider buying/playing games you might not have considered. Since Ubuntu already has a "Cloud" system (Ubuntu One) in place and is getting ready to sell games why not develop a tool that allows both cloud saving/sharing and achievements. 

Such a tool should of course by default be integrated into the pre-installed selection of open source games and allow commercial/open-source developers to make easy use of it. Canonical would be in charge of moderating the achievements system.
