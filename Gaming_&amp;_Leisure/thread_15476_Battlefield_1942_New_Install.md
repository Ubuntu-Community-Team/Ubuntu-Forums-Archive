### Post by crun on 2005-02-16
Sapper2ID, Wine alone probably isn't enough (unless ofcourse it's mentioned in the Loki Installer) - you will need Cedega (formally WineX). This is a commercial fork of Wine, which you can get at [http://www.transgaming.com](http://www.transgaming.com) (unlike wine you have to pay for it though, unless you want to compile it from cvs). Most Loki installlers for non-native games require Cedega.

For more info on running Windows games with Wine or Cedega, try [http://frankscorner.org](http://frankscorner.org).
