### Post by desertboy on 2008-03-22
I got it to compile from svn, was really easy in the end

sudo apt-get build-dep hedgewars
then I was missing
fp-utils-gfx so I added through synaptic

then grab svn untar it, cd to the directory,

cmake .
make
sudo make install

the new graphics and sounds are great
