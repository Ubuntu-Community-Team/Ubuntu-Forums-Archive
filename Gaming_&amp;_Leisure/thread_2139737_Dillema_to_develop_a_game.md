Hallo everyone. Well... I know it may sound weird, but I'm developing a game using Construct Scirra. It's a free, opensource, powerful 2D game creator, quite practical. It doesn't require any knownledge on programming. Sadly, it only does games for Windows, using DirectX. Both software and generated games work very well under WINE, but I'm not sure...

My dream was create a game, and with a program like that, it became possible. It was supposed to be a 100% free opensource game, for both Windows and Linux (and more platforms, if it was possible...), totally free of proprietary stuff. And since I began in the Linux's world, it would be nice contribute with something too. But since DirectX is still proprietary, but seems like it won't be possible...

I'm not a person to program codes, I'm just a person who like to draw sprites and has a dream to create the first game, just for fun. Of course there are "some" game engines, but they have some problems:

- Stencyl, the best option and runs on Ubuntu, also only export for Windows and Flash, both proprietary;
- Game Develop, even quite hard and lacks practice, is still with some bugs;
- And there's Game Editor, though "opensource" (not 100% sure about this), it's hard do something with this.

For now, the game will continue on Construct Classic specially, since it's opensource, there's still hope of some kind person would edit the code, and make a Linux version, to create games for Linux and using OpenGL/SDL.

Well, what do you think about this, guys? Please, sorry for annoying.
