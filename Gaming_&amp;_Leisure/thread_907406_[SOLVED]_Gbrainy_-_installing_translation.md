### Post by viciouslime on 2008-09-01
Gbrainy already comes with the German translation. I'm using it right now :)

All you need to do, is tell it to run in German. The easiest way to do this, will be to create a separate user account for your mum. Then go to System/Adminstration/Language Support and install the German language support for your system. Now log out, before you log in as your mum, look on the options for the login screen (bottom left corner) and set the language to German. Then, enter your mum's username and password and it should ask if you would like to make German the default language for her. Say yes. Now every program will be in German :)
