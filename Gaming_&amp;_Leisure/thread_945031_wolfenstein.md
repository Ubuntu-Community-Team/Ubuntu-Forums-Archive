### Post by MaxIBoy on 2008-10-12
Gtk+Radiant is really the only choice for doing serious mapping work. It hasn't really been in development since 2006, but then again, Quake 4 came out in 2005, and that's the latest game the editor supports.

This program is a great piece of work. I swear by it. It's a fork of ID Sofware's in-house Quake 3 editor, which has been much improved. In other words, you get tools which are more powerful than the ones used to make Quake 3 itself.
