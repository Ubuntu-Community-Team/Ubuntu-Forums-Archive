### Post by ziekfiguur on 2010-10-10
This might not really be on topic, but c++ is not really that much harder than java. So far the java games i played were mostly pretty buggy. Python however really is significantly easier.
That being said, it would certainly be possible, just take a look at freecol (even though i think it's to buggy to play) which is written in java. Another problem with java is that the vm might have bugs that you can't fix, and you don't know which vm your users would use. For that reasons i would suggest trying to learn c++, which really isn't that hard, or use python. However when using python you might hit some serious performance issues.
