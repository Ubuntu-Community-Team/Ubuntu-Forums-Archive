### Post by happysmileman on 2008-01-25
Yeah in fact I've just got Doom3 and the expansion pack working on my OpenSuse install a few days ago.

It goes a fair amount faster on Linux than in Windows (it needs 2/3 the RAM Windows does and 2/3 processor speed for minimum requirements), you don't need the CD in the drive to play (but you DO need a valid serial number), and I've gotten to the 9th level so far, playing for a total of about 7-9hours.

I've had one crash while loading the second level, very soon after I started playing, but other than that I've had no problems, and it was autosaved anyway, since it does that at the start of each level.

My frame rate seemed to drop at the very start of each level for a few seconds, but my graphics card is pretty much the bare minimum supported, and after standing still in the elevator for about 3-4 seconds the frame rate always went back to normal.
