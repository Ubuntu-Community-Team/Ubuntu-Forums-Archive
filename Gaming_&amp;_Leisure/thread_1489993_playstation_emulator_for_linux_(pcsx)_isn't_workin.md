Welcome to the community, I know it's hard to deal with not getting the answer you want right away, but I promise with more use of Linux, you gain a LOT more patience. Or go back to Winblows. :popcorn:

So you want to play Playstation 1 games on your Ubuntu computer. No problemo my man, you have a couple options:

[LIST]
[*]**ePSXe**
Great solid emulator, but quite a hassle to get configured with all the different plugins. However, if overall customization is what you are looking for, look no further.

To install:

Open 'Software Sources' under System>Administration on the toolbar
Click the 'Other Sources' tab
Click the 'Add' Button at the bottom
Copy and paste this URL into the APT line:

```
deb http://ubuntu.global-web.us/karmic binary/
```

Afterwards, reload your software list when it asks you to, then open a terminal and put in the following:

```
sudo apt-get install epsxe
```

After it is done you should have ePSXe in Applications>Games in the toolbar


[*]**pSX**
Another fantastic emulator, but with limited configuration options. Starts up right away, don't even think there is need for BIOS (you should know what that is if you are attempting PSX emulation), and it runs VERY solid.

To install:

After installing the repository for ePSXe, you can also install pSX in the same manner:

```
sudo apt-get install psx
```

Also will be under Applications>Games


[*]**PCSX**
Can't really help you here, because ePSXe and pSX have always worked fine for me, PCSX seems to be just a bunch of issues IMO. If you REALLY want to use PCSX, the best way is to install it with a .deb, found [here]("http://launchpadlibrarian.net/14262787/pcsx-df_1.816-1_i386.deb").


[*]**WINE**
I've also read that ePSXe 1.7.0 runs pretty good under WINE, which is what you need to run Windows (.exe) executable files under Linux. To install WINE, it's just as easy as:

```
sudo apt-get install wine
```

EDIT: If you are having issues running .exe files because Ubuntu says it's not safe, do the following:

Right click the file, and click 'Properties'
Click the 'Permissions' tab, and near the bottom, check the box next to 'Allow executing file as a program'

There has been a serious change with how Ubuntu allows .exe programs to be run it seems, and you are required to do this for most .exe files you may have downloaded.

[/LIST]

I hope this helps you out and gets you on your way, just don't give up! I've been using Ubuntu for 4 years now, and while it may be a pain sometimes, trust me, it's VASTLY superior to Winblows in many ways. But that's just me, it's really not for everyone.

Always remember, don't be shy to ask questions, and don't get discouraged when you don't get the answer right away.
