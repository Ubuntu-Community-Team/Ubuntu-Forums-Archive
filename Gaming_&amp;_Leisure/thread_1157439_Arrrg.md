### Post by eragon100 on 2009-05-13
> **ruzkie said:**
> have you been able to run it nicely before? that card is very bad for games

Urban Terror is based on the now 10 (!) year old Quake 3 engine.

It should run fine, were it not for the fact that they are some massive 3d performance regressions in the intel drivers, so you'll have to wait for them to be resolved. Right now a lot of invasive changes are being made to the driver infrastructure, so it's slow as molasses. It's even worse in the new ubuntu 9.04, so don't upgrade to solve it :(

It will most likely be fixed before ubuntu 9.10, Karmic Koala. If you want to be able to play Urban Terror before then, I am afraid you will have to downgrade to 8.04 "Hardy Heron", sorry :(

If it's not a laptop, you can also go and buy yourself a crappy $30 nvidia GeForce 8 dedicated card, it will run the game fine, and if it's a Geforce 8, say a 8400 GS, it can also decode HD video, also on linux. I am very happy with my own card, a nvidia 512 MB GeForce 8 8600 GT. It cost about $75, but it's nice for linux games. Urban Terror runs at max settings at 1680x1050 res with 60 fps. Because I capped it at 60 :D
