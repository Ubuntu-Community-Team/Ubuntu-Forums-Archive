### Post by T3h_Dohtem on 2008-12-30
Just a quick news flash. I'm a big fan of linux and Anarchy Online, so this new channel was very exciting to me. Whether you have already tried AO or think you would be interested (There is a howto [here]("http://ubuntuforums.org/showthread.php?t=291330")) a new community has been born centered around Linux and AO.

A cross server bot has been set up that links Linux users ingame to each server as well as to an IRC channel.

You can join from ingame by doing: /tell Linuxbot !join

You can also join outside of the game and talk to players on the game servers by joining up in IRC at:
Server: irc.funcom.com
Port: 6667
Channel: #Linuxbot

I have found that there are actually a lot of Linux fans and programmers in AO, so this was very cool to me, and I thought I would share. :)
