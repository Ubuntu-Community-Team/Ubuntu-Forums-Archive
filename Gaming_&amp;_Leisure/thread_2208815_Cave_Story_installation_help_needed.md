### Post by oldrocker99 on 2014-03-02
If you decompress the file with a file manager, you'll have a directory with the same name as the program. Open that directory, and right click the doukutsu file, select "Properties" and then "Permissions" amd make sure the executable box for the file is checked. Then right-click the program and select "Open." Or open a terminal, cd to that folder, and type

./doukutsu

and you should be OK.

And welcome to the forums. The only way to get answers is to ask. And lo, you'll get answers.
