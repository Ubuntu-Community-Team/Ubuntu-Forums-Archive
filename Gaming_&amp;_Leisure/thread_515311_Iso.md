### Post by hikaricore on 2007-08-01
Many intallers work if you copy the contents of all discs to a single folder. (example: make one on your desktop for them)

It's easy to make ISO files in linux as well as mount them (mounting is part of the OS),
but I suggest my first idea first as mounting multiple ISO files ripped from discs seems
a bit insane just to install one application.

Also check [The WINE Application Database]("http://appdb.winehq.org") first to make sure the game/program runs under WINE.
