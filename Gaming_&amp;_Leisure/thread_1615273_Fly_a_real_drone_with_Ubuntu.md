### Post by oregonbob on 2010-11-06
I thought I would mention the most fun I've had playing on Ubuntu. A company that makes a Quad helicopter that sells for around $300 uses Ubuntu for much development. They offer an open source API that includes a flight control program so you can fly the drone using Ubuntu and a joystick or game controller. I already compiled the example control and it works great on my Ubuntu, it is so cool and fun!

A YouTube video showing Ubuntu/joystick compile and flight:

### Post by oregonbob on 2010-11-07
There are a number of multiplayer games in development and that is one main reason they made it an open source project. If you search youtube for "ardrone" will can see some games in development. I have personally noticed drone development in general is moving at a very face pace. You might check out [http://ardrone.parrot.com/parrot-ar-drone/usa/ar-games](http://ardrone.parrot.com/parrot-ar-drone/usa/ar-games) and here is a Wall Street Journal writeup about drones [http://online.wsj.com/article/SB10001424052748703631704575551954273159086.html](http://online.wsj.com/article/SB10001424052748703631704575551954273159086.html)

Open source is a unique approach augmented reality gaming that makes development an exciting process. I have watched it enough to see it is moving at breakneck speed.

I bought one of the machines, flown it and it is worth the price now but I expect that price to drop like all new technology does. It is a blast to fly. The most popular way to fly is using an iPhone as the controller. That works great too but I already like the joystick on my ubuntu laptop better.

Update: I forgot to mention that in addition to controlling the drone with my Ubuntu linux, the drone itself is actually running linux! One can telnet into the drone system and poke around, make setting changes (though you might void the warranty) similar to iPhones, etc.
