Inside the directory with the files, use ```

```

Then to play the game, simply doubleclick the "ioUrbanTerror.i386" and be on your way. The one you mentioned (ioUrTded.i386) is a server. And, if you are on 64 bit, you need to use the 64 bit version (ioUrbanTerror.x86_64).
