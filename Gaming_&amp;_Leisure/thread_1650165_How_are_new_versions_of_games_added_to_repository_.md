If you hope for Warsow 0.6 to arrive in the Repos you will have to wait for Ubuntu 11.04. Because Ubuntu is not a [Rolling Release]("http://en.wikipedia.org/wiki/Rolling_release") Distro and only adds Security Updates if they are really needed. That means if you want a new Version of Game/Program XYZ from the Repos you have to wait for the next Ubuntu Release or use a [PPA]("https://launchpad.net/ubuntu/+ppas") because they mostly just add Bug Fixes and Security Updates to the Repos after a new Version of Ubuntu is released.
