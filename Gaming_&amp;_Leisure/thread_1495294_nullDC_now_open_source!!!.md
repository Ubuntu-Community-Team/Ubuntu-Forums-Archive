### Post by Sega Loco on 2010-05-27
The ever popular nullDC Sega Dreamcast emulator is now an open source project. This is great news for the Linux gaming community. I've personally started a (currently unpublished) fork meant for cross-platform compatibility, but if anybody else is interested in working on the official version, let the guys **[here]('http://code.google.com/p/nulldc/')** know! If you instead want to help me out with my cross-platform fork, let me know and I will send you a copy of whatever I have done so far. (Note, my code is much neater than the official stuff, but still a bit messy. Be sure to tell me if you fix anything and want me to merge it with what I have. I won't be around much this summer as I am working, but I will try to get to any questions or comments.
