### Post by jacksaff on 2008-09-19
pychess has a fics interface but it's still very new and buggy and not yet that featureful. Eboard is better, as is jin. The best client out there is babas (IMO) and it runs fine with wine.
For eboard, sudo apt-get install eboard
For Jin, google for it and download. It's a java file so just click to run - it doesn't need to be installed.
For babas
sudo apt-get install wine
Go to [http://www.babaschess.net/](http://www.babaschess.net/) and download the xp version, click and wine should install it for you.
