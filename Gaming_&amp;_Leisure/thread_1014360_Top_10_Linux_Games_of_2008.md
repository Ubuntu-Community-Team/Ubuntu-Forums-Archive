### Post by Brian Vaughan on 2008-12-19
> **shirteesdotnet said:**
> I admit I have not tried Flight Gear yet. The graphics on their site look nice! I used to play MS Flight Simulator when I was in XP a couple years back. Can Flight Gear compare?
In general, yes. The fundamentals -- the flight simulation and graphics -- are as good as MS Flight Simulator X. I found the graphic performance in FlightGear significantly better than in MS FS X.

There's a good variety of aircraft included with the basic package available in the Ubuntu repositories, with more general aviation and small planes, but fewer large planes. More planes are available for download from the main website.

The basic package only includes scenery for the San Francisco Bay Area, but you can download scenery for the entire world from the main website.

Downsides: the interface for FlightGear is much more limited than for MS FS X. In particular, if you don't like the defaults, you have to select the display mode, the aircraft, the starting location, and some other features, via the command line, or by creating a .fgfsrc file in your home folder. (There's a graphical launcher program, but it doesn't seem to have been packaged for Ubuntu yet.) There aren't the elaborate tutorials and game modes that are present in MS FS X. It's tricky to download the scenery files and install them correctly. Some of the aircraft have incomplete panels. And, as I recall, there was one package of scenery object files (the "Industrial" set, I think) that was missing, and I had to download and install separately that package from another website.

None of that should be too much of a burden for someone used to tinkering with GNU/Linux. FlightGear *does* work reasonably well out-of-the-box, and once you've got it configured to your liking, it's really good, with more options than MS FS X, and it's more amenable to modification.
