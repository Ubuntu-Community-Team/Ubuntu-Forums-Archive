### Post by James Paige on 2010-05-13
Here is how I do it.

First go to System->Administration->Synaptic Package Manager

Type your password to get into synaptic.

Go to Settings->Repositories

Make sure that the check mark by "Community Maintained Open Source Software (universe)" is checked

Close the "Software sources" window.

Now click the "Reload" button in the top left.

When the reload finishes, search for "pcsx"

When you find the pcsx-df package, click on the checkbox and choose "Mark for installation"

If it asks you to install any support packages, that is okay. (I can't remember if it does for this program)

Then click the "Apply" button in the toolbar at the top, and it will download and install the packages.

Close out of Synaptic

You will find PCSX in your Games menu.
