### Post by Carpentr on 2011-04-14
Check out [http://www.topmudsites.com/]("http://www.topmudsites.com/") and find a MUD you want to try. If you click the red "info" button it'll give you  the server address and port number.

For example, Avatar MUD's Telnet Address is listed as:
avatar.outland.org:3000

So the server address is: avatar.outland.org
The port number is: 3000

To open up the server in GNOME-MUD or KildClient there  should be a file button up top (not exactly sure as I haven't used either in a while) and  there should be a button in the file menu that you can click that will pop up asking you to input your MUD's details.

I hope this helped. If you have any more questions feel free to ask.
