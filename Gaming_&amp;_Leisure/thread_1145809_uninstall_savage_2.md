### Post by bkaplan on 2009-05-03
If I'm not mistaken, there is an uninstall-Savage2.sh file in the /home/user/Savage2 directory.

If you can't double-click it to run the uninstaller, open a terminal window and navigate to the Savage 2 folder.  Run it by typing ./uninstall-Savage2.sh

Hope this helps.
