### Post by mivo on 2008-01-23
Automatically, no, but you should be able to do this with the menu editor that comes with Ubuntu quickly. Try this:

[LIST]
[*]Go into System -> Preferences -> Main Menu.
[*]Highlight "Games" on the left side.
[*]Click on the "New Menu" button and type in a name (this creates a sub-menu).
[*]Drag and drop the games from the right panel into/onto the sub menu folder.
[*]Delete the duplicate entry by highlighting it and pressing the delete key.
[/LIST]

Hope this helps. :)
