### Post by Perfect Storm on 2008-05-26
It's not source code but the whole game packed ready to launch. It's only the code that's  Open not the graphic (the code can be obtain under more downloads). tar.gz or bz2.gz are just packages like a .zip file, it can contain everything.

```
cd ~/Desktop
tar zxfv bolzplatz2006-1.0.3-linux.tar.gz
cd bolzplatz2006
sh bolzplatz2006.sh
```

the game requires java to run, you need to install sun java first.
