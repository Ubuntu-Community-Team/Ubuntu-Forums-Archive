### Post by information_entropy on 2007-11-24
There are several different versions of X-COM UFO Defense.

The original version was  for DOS or Windows 3 computers

The later versions (like those in the X-Com collectors set)
 were redone to work in Windows 95/98 computers 

The original DOS version works great in DOSBOX :)

The Win95 versions produce the Win32 error if you try to run them
in DOSBOX

I tried to run the Win95/98 versions in a Virtual WinNT4 machine
It did not work :cry:

AFAIK Win95/98 versions will not run in DOSBOX:sad:
