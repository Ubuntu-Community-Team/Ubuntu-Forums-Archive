### Post by noerrorsfound on 2007-08-03
As it says: Check your firewall settings. The documentation says you need to forward ports 28763 UDP and 28764 UDP. You either have a software firewall (such as Firestarter) installed, or a hardware firewall such as a router.
[quote=README.html in your AssaultCube directory]Server port is fixed at 28763, with 28764 being used for the server info requests (both UDP).[/quote]

### Post by Mr Anderson on 2010-08-15
Hi All, i stumbled across this I know its old but i can help here. 
 
Creating a server is actually very easy if you visit [www.fragmasterclan.com]("http://www.fragmasterclan.com") you will see a tutorial that guides you through the process in Windows. For Ubuntu and linux you should edit your server config file to contain the settings as suggested by the tutorial. 
Also remember if you are using version 1.1.0.1 of Assualt cube you must have each command in your config file on a serperate line this is different to all earlier versions and is causing much confusion. 
 
With regard to the "unable to ping the master server" error this is deff your firewall it is the most common source of problems amoungst new server operators. Try (for a short time) removing all firewalls ie pluging directly in without routers ect in the way, this will show you its a firewall thing, and if using Mandriva or dare i say it Windows ensure your machine firewall is disabled.
