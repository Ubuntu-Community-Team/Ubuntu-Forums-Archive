### Post by Perfect Storm on 2009-12-02
Most games won't work in Linux as most games are build for Windows.
That said, there are some games that run Linux natively.
You can search here:
[http://gwos.org/doku.php/start](http://gwos.org/doku.php/start)
[http://happypenguin.org/](http://happypenguin.org/)
Would be better if you tell us which kind of games you like to play or genre you prefer.


That said; It's possible to get some window games running in Linux.
Take a look at; [http://www.winehq.org/](http://www.winehq.org/)
Some games may take some work to get running while others is seemingless running out-of-the-box....and others wont run.
Before you start on wine, please read up on it as it is not so straight forward, you'll find many information here (go through the stickies there);  [http://ubuntuforums.org/forumdisplay.php?f=313](http://ubuntuforums.org/forumdisplay.php?f=313)
