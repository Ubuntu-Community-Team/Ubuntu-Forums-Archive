### Post by Carpentr on 2012-07-13
Type this in the terminal:

```
chmod u+x PlaneShift-v0.5.9.1-x64.run
./PlaneShift-v0.5.9.1-x64.run

```

You should then be prompted by instructions, within the terminal, that will guide you through the installation process.
