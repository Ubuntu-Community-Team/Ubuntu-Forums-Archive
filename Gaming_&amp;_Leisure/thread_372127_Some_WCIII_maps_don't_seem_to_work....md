But when I try to play some of the custom maps I have (DotA, etc.) it won't let me. **EDIT:** I just realized that I think it's a version incompatibility issue.  When I first tried to connect to Battle.net it downloaded a patch which I assumed was the newest one, but now that I look at it I've noticed it says v1.07 instead of v1.20 or whatever the newest version is.  I downloaded the newest patch from Blizzard's website but when I try to install it with Wine I get this error: "ERROR: unable to create file 'C:\Warcraft III\BNUpdate.exe' Path not found"  Can anyone help me with what I'm supposed to do to get it to work?

**EDIT AGAIN:** Alright I figured it out, the folder was named "WarcraftIII" and I had to have it named "Warcraft III" with the space.  That got it working!
