### Post by pogokero on 2008-01-09
i had this same problem (gusty, battle for wesnoth). the game would suddenly drop into a window...and then go back into full-screen mode.  it's quite annoying.

the solution i found was to change the resolution in-game to the desktop resolution (which for me is the native resolution for my laptop monitor, ie 1280x800).  after this, the game looked better and it didn't drop into windowed mode anymore.

hope this helps.

### Post by sceaduwe on 2008-08-09
I see this post is a bit old, but i'm posting for anyone else who has the problem. I had the same problem and a google search brought this page up.

[https://bugs.launchpad.net/ubuntu/+source/wesnoth/+bug/105249]("https://bugs.launchpad.net/ubuntu/+source/wesnoth/+bug/105249").

I set my screensaver to one minute and started playing Wesnoth. One minute later it exited fullscreen. So I guess the way to solve the problem is to turn off the screensaver while playing.Hope this is usefull.
