I get this error constantly when using Steam.  Reinstalling did not help but I've found a way to get Half-Life 2 based games to work most of the time.

If I run Half-Life 2, Counter Strike: Source, Day of Defeat Source, of Sin Episodes: Emergence (all Half-Life 2 based games), and I get that error, here is what I do

I run the original counter-strike, click new game, let it load up a game so that I am in a map of CS by myself.  Loading up a multiplayer game will work too.  Then I quit out and try to run the Half-Life 2 based game.  80% of the time, it works.

If it doesn't work, running another Half-Life 2 based game works.  I'll load up Sin, load my quicksave, quit out of Sin, and then suddenly Half-Life 2 will work properly.

It's a huge pain and it's been happening for about 6 versions of Wine now.
