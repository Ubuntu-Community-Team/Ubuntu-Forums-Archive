### Post by slux on 2006-02-28
Second life Linux client is in beta, you'll probably find it at icculus.org but it's supposed to still be quite buggy.

The Sims has been ported to Linux, sort of. It's one of those Wine-ports that transgaming did at one point. Works all right although it's missing the per-user configuration features some people like. You can only get it as part of the old Mandrake Linux Gaming Edition box and it comes in an rpm but is relatively easy to install on any distribution.

And Lincity isn't a Sims equivalent, it's a SimCity equivalent and Simutrans is a Transport tycoon deluxe equivalent. :P
