### Post by splitretina on 2005-12-14
I spent awhile trying to compile Scorched3d 39.1 on Breezy. I got it to make but it seg faulted when run (SDL Parachute Deployed). I had nearly given up, but at the last minute I tried:

```
$ sudo alien Scorched3D-39.1-i386.rpm
```

Worked like a dream. That was the RPM from [http://www.scorched3d.co.uk/](http://www.scorched3d.co.uk/)

Just thought this might save someone some time.
