### Post by viciouslime on 2007-09-30
The first time you use an iPod of any kind you MUST connect it to a windows/mac os computer. Preferably windows as you won't be able to use it on linux properly otherwise.

This "formats" it. That doesn't just mean format the file system, it also creates the ipod database etc. Without this the ipod cannot be used. Once you've connected it to a windows system running itunes then you'll be able to use it with linux no problem :)
