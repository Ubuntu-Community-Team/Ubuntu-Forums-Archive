### Post by lil.deimos on 2010-10-10
Hi, 
I just released this app. It's written using Qt (>4.5.3). Here some features:
search with YouTube and Google Image
Display results in an opeGL scrollable grid view as thumbnails with description
Download all at once, slide show, set as wallpaper (not yet in kde 4), save / restore  searches
Import media also from a local directory.
Here a screenshot

[IMG]http://mediadownloader.cz.cc/portal/wp-content/gallery/mdfeatures/screenshot-media-downloader-v1-4-1.png[/IMG]

There is some work to do yet, but its stable. Also there are few translations and if you want to contribute you are welcome [IMG]http://ubuntuforums.org/images/icons/icon12.gif[/IMG]
Sources and deb packages are available for both i386 and amd64 here [http://mediadownloader.cz.cc]("http://mediadownloader.cz.cc/")

have fun
