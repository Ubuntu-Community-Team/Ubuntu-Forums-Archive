### Post by mivo on 2009-10-26
If you want a native client, Neverwinter Nights is still one of the prime choices. There are hundreds of player-made campaigns, and for a few bucks you can buy Bioware's Infinite Dungeons mods for a Nethack-like random experience. It is not free, but very inexpensive. May well be the only RPG you will ever need. :)

The [Penumbra]("http://www.penumbragame.com/index.php") trilogy is fantastic for just $20, and there is a native Linux version too.

For a better overview, start here: [http://en.wikipedia.org/wiki/Linux_gaming](http://en.wikipedia.org/wiki/Linux_gaming)

I maintain a Windows machine for gaming because there is just too much good stuff I don't want to miss out. To support gaming on Linux, you have to buy the few commercial games that are released for Linux. That is the only way.
