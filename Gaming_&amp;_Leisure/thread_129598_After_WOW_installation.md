### Post by moccah on 2006-02-14
Try to change the settings in the World of Warcraft/WTF/Config.wtf file.
I`m using: 
```
SET renderer "OpenGL"
SET gxApi "opengl"
SET gxColorBits "24"
SET gxDepthBits "24"
SET gxRefresh "60"
SET gxResolution "1280x800"

```
