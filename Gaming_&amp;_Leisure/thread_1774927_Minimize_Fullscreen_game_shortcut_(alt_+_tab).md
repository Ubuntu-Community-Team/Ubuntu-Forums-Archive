### Post by MunkyJunky on 2011-06-05
Alt+Tab isn't a minimise key exactly, it's the application switcher. When the game minimises, you're switching the current active application to the desktop itself. 

If alt+tab isn't working, try super+tab. You could also possibly set up a keyboard shortcut to go to the desktop.
