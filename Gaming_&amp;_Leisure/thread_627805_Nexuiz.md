### Post by BreathEasy on 2008-01-01
Also, I would recommend deleting the **.nexuiz** folder from your home directory (it's hidden, so if you're using Nautilus enable the SHow Hidden Files option somewhere in the menus). That will remove the config file so next time you start Nexuiz, it will start from scratch with defaults, just in case an option was selected which stuffed things up before.
