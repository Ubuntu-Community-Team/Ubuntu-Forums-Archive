### Post by mikedep333 on 2008-03-29
If you are running the same version of a game but for different platforms (such as nexuiz 2.3 for linux and nexuiz 2.3 for windows) it almost definitely will work. It only may not work if the game is very incomplete, or one of those crappy mac ports.

If you are running a game under wine on the linux box and natively on a windows box (and again at the same version), it will run unless there is a bug in wine which prevents it from running.
