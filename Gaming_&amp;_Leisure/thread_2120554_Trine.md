### Post by thnewguy on 2013-02-27
I don't think the original Trine (Windows version) will work properly in WINE. The WINE software really isn't up to the graphics processing required. However, as others have pointed out there is a native version of Trine for Linux so there isn't any reason to try to run the Windows version.

Not sure if anyone else has had this problem, but I found the original Trine would run fine on older versions of Ubuntu, but I couldn't get it running on 12.04 (and newer). Trine 2, on the other hand, works fine with 12.04 assuming your video card is up to the task.
