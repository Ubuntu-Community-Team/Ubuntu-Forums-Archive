### Post by DirtDawg on 2008-11-23
> **Grishka said:**
> or use fusion-icon.

I use fusion-icon as well. FYI: To install
```
sudo apt-get install fusion-icon
```
To start fusion-icon, go to Applications>System Tools>Fusion Icon. This will create an icon on your upper panel. Right-click and under Select Window Manager, switch from Compiz to Metacity. Switch back to Compiz any time you want fancy graphix back.
