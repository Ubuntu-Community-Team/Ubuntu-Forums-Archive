### Post by Nevon on 2009-11-03
If you installed it via your package manager you could simply deselect it in there and have it removed. If you installed it via some custom way, which I believe is the case with Savage 2, most likely there is an uninstallation script in the same place as the installation script. If not, then your best bet would be to ask the people in the Savage 2 forums.
