### Post by Carbon Based on 2006-10-16
The best realism shooter that I know of for linux would be True Combat: Elite, which is actually a mod for Return to Castle Wolfenstein: Enemy Territory. Both are free, and TC:E has pretty much all the elements you mentioned. It's frequently discussed in these forums, so search around. You can find install instructions here: [http://katanoon.nl/Ubu-e/install.php](http://katanoon.nl/Ubu-e/install.php)
The only error in that HOWTO is that you no longer need to patch as described in step 5, just download the .49 .run file from the link and execute it.

### Post by Matt Yun on 2006-10-16
I heartily second True Combat: Elite.  

1) Download the [Enemy Territory installer]("http://ftp.games.skynet.be/pub/wolfenstein/et-linux-2.60.x86.run").  After you save the file, chmod +x it and run.

2) Download [TC:E v. 0.49]("http://hydrogen.cert.ucr.edu/ftp/pub/mitch/true.combat.elite_0.49-english.run").  After you save the file, chmod +x it and run.

The websites for the game are [truecombat.com]("http://truecombat.com") and [truecombat.us]("http://truecombat.us").  Both sites have Linux support forums.

Also, [America's Army has a linux client]("http://americasarmy.filefront.com/file/AASF_Direct_Action_v25_Linux_Full_Install;49654"), but I've never played it.
