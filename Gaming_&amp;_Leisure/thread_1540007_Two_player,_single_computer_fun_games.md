### Post by Arthur_D on 2010-07-27
SuperTuxKart - a fun racing game with nitro and other collectibles. I'll recommend to use two input devices for this game, as you have quite a lot of keys to use and keyboards have commonly a limit on how many keys you can hold down at a time.
Slingshot - turn based gravity game where you need to use the gravity of planets to hit your opponent with your weapon.
Neverputt - a turn based golf like game with some crazy levels. Definitely fun for multiplaying on the same computer. :)

You can find all those games in the Software Center.
