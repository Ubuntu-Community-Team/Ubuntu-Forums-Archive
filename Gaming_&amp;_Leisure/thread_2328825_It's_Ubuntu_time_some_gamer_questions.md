### Post by Impavidus on 2016-06-24
Wubi was a method to install Ubuntu inside Windows for easy install and, in particular, uninstall. It's no longer supported, doesn't work with recent Windows systems and is not what you need anyway. What you need (maybe) is wine. That's a compatibility layer to run Windows applications in Linux. It doesn't always work. Check [here]("https://appdb.winehq.org/") what other people's experiences were with your games. It's also possible to install Windows in Ubuntu using a virtual machine, but the performance will not be good enough for most modern games.
