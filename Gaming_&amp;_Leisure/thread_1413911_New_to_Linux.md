### Post by Pollox on 2010-02-23
> **CrAsHnBuRnXp said:**
> 
4) I dont know how to install anything in Linux


There are several ways to install things.  A great method is to go to the Applications menu and click on Ubuntu Software Store.  There is also Synaptic Package Manager for more advanced users.

> 
5) If I install Ubuntu, is there a way I can convert my other hard drives into a linux file system without formatting?


Ubuntu can read fat32 and ntfs drives, so there's no need to reformat them.  If you want to use a linux file system like ext4 on one of your drives, it will require you to reformat it and Windows will not be able to read it.

> **CrAsHnBuRnXp said:**

5) If I install Ubuntu, is there a way I can convert my other hard drives into a linux file system without formatting?  
No. The only way to convert a Windows NTFS file system into a linux file system (ext3, ext4, etc) is by reformatting the drives on a linux machine, or from a live partitioning CD like Parted Magic: [http://partedmagic.com/](http://partedmagic.com/)
