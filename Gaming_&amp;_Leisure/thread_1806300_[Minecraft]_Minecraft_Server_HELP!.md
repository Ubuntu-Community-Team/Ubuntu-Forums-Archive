### Post by Jepic Phail on 2011-07-17
I am not sure if I understand your question. If you are asking about the physical location of server.properties, ops, whitelist, banlist, etc., they are located in your Minecraft server directory (where your minecraft_server.jar is located). If you are asking about how you can ban, whitelist, op people in-game, you can use the [server commands]("http://www.minecraftwiki.net/wiki/Commands"). If you want something more fancier you will have to install [mods]("http://www.minecraftwiki.net/wiki/Mod#Server_Mods").
