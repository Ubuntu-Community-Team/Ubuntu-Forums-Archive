### Post by TheFu on 2023-05-27
> **jonfisher said:**
>  I seriously doubt the i5 chip in my laptop will do better, cause aren't i5 chips normally slower than i7 chips like my desktop has (?)

The generation of the chip matters.  Generally, a Core i7 2 generations behind can be replaced by a Core i5 of the current generation for similar performance.  
If you have a 4th-gen Core i7, then a 6th-gen Core i5 is a little faster.
If you have a 8th-gen Core i7, then a 10th-gen Core i5 is a little faster.

The same works for Core i3 CPUs too.  So a 4th-gen Core i7 can be replaced by an 8th-gen Core i3.  There are many different CPUs in each generation, so a really fast Core i3 will do a little better than the Core i5 2gen behind or Core i7 4gen behind, but it still generally works that way.

You can look up specific performance by searching for the *CPU model* + *passmark*.

For example, I have a Core i5-750 (1st Gen).  That has about a 2500 passmark score with the current passmark tests (passmark values seem to change annually, so we can't compare old scores to new scores).  An
[LIST]
[*] Intel Core i3-7102E  or 
[*] Intel Core i3-4100M @ 2.50GHz or 
[*] Intel Core i7-2640M @ 2.80GHz 
[/LIST]
have about the same passmark.  For 1st gen CPUs, manufacturers tend to push the performance higher.

[https://www.cpubenchmark.net/high_end_cpus.html](https://www.cpubenchmark.net/high_end_cpus.html) has a bunch of 7500 and higher passmark systems, sorted by performance. If I pull some CPUs for around 10,000 passmarks, 
```
[LIST]
[*]Intel Core **i3-10**320 @ 3.80GHz    (6%) 10,057
[*]Intel Xeon Silver 4110 @ 2.10GHz (6%) 10,028
[*]Intel Xeon W-2125 @ 4.00GHz      (6%) 10,028
[*]Intel Core **i7-8**750H @ 2.20GHz    (6%) 10,012
[*]Intel Core **i5-11**45G7 @ 2.60GHz   (6%) 10,012
[*]Intel Xeon E5-2650 v2 @ 2.60GHz  (6%)  9,984
[*]Intel Core **i5-10**400T @ 2.00GHz   (6%)  9,974
[/LIST]

```
We can also see how it is possible to get a great Core i3 that is just as fast as some Core i5 CPUs from the same generation.  Pay attention to this detail, so you don't get screwed buying your next system or building your next computer.

Well, 90°C is hot, but all CPUs will throttle themselves when needed.  As for other parts of the case being 90°C, you need better airflow.  Start by cleaning the dust out, cleaning off the case fans, ensuring you have push and suck fans to get the flow going from the bottom-front to the top-back across as many components as possible. If you have multiple HDDs inside the case, perhaps a fan just to suck air over the HDDs is needed?

If you have furry pets, their hair will get inside and cruft up everything.  OTOH, if you have a snake or fish, that wouldn't be too much of a concern, unless you shed or have dry skin that gets sucked into the case.  I flake a bit, but those just show up in the keyboard, not inside computer cases (so far).

How many fan headers exist on the motherboard is different for each system.  Nobody here can answer that for you. Get out the motherboard manual and read it.

If the CPU is getting really hot, it might be time to pull off the fan on it, clean off the old thermal paste/grease, and apply new.  There are lots of youtube videos for how to do this.  It may be useful to get a better fan + heatsink for the CPU too.  I've always been happy with the free kit that comes with the CPU, but I'm not gaming 22 hrs a day and don't have heat issues.
