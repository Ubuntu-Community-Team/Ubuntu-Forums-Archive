### Post by Veikk0 on 2011-05-13
I believe you're trying to install the Windows version of Minecraft. In Ubuntu you just download the Linux version (minecraft.jar) from here [http://www.minecraft.net/download.jsp](http://www.minecraft.net/download.jsp), make it executable (right click on minecraft.jar -> Properties -> Permissions -> Allow executing file as program) and open the file with Java.

Alternatively, you can try this Minecraft installer [http://ubuntuforums.org/showthread.php?t=1726735](http://ubuntuforums.org/showthread.php?t=1726735).
