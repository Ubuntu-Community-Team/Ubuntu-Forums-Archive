### Post by dfreer on 2009-04-22
Logitech Dual Action Gamepad:
[http://www.logitech.com/index.cfm/gaming/pc_gaming/gamepads/devices/288&cl=us,en](http://www.logitech.com/index.cfm/gaming/pc_gaming/gamepads/devices/288&cl=us,en)

I have 3 of these never have had any issues with them. The main thing to look for is that it uses a generic USB HID driver, if it plug and plays in windows xp it should plug and play in linux too.

### Post by vahnx on 2009-05-16
I'm in Jaunty and my 360 controller light seemed to be blinking until I press a button then it just stayed lit. I did ```
sudo apt-get install jscalibrator
``` then ```
jscalibrator
``` and it was detecting it. So I guess it has native joypad (at least 360) support.
