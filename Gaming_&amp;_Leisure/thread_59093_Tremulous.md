### Post by endy on 2005-08-22
Yes I tried it today and it's very polished, all I'd suggest to anyone about to play it is at least skim through the [manual](http://tremulous.net/manual/) first as it's not just a FPS.

It is a total conversion of Quake3 but right now it still requires Quake3 installed to work, however according to [this forum post](http://www.tremulous.net/phpBB2/viewtopic.php?t=374) the next release will be stand alone :)

Saying that though, as it doesn't use any maps or artwork from Quake 3 perhaps you can download and compile the recently release Quake 3 source code and have it running that way... But form what I've heard so far compiling Quake 3 on linux is still a bit tricky so may be best to wait for the next Tremulous release :)
