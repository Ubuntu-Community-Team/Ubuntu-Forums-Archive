### Post by SomethingUniqueHere on 2007-02-12
To address each of your questions in turn:
1. If you can get along fine with wine then theres no need to get cedega.  I haven't tried either, i used to dual boot but now i just play UT2k4 and Doom3 native, but i would imagine its much much easier to get games running with cedega.
2. As i understand it, you can subscribe to cedega for certain time intervals, during that time you can download the latest release (which is now yours forever) and you're allowed to vote on what they focus their development on (ie, which games do they make sure work) and you get tech support.  for more info go to [transgaming]("http://www.transgaming.com/")
3. I think Cedega works by translating windows API calls, often with less overhead.  I've heard alot of people say they get (marginally) better frame rates with linux but it depends on the game but in practice, changes in performance isn't a concern.  You're always going to have a bit more trouble getting them to run in linux but the only way to get more developers to code in linux friendly games is to game in linux

Hope that helps you out!
