### Post by Tarvok on 2009-07-03
Recently, I decided to try playing Ultima 8 in Dosbox again. I'd read about others having absolutely no problem playing this game in Linux Dosbox, and wanted to see if I could get it working properly.

My problem has always been a mouse problem. Sound works great, everything runs nicely, smoothly, etc. The problem comes when I try to jump, which involves both mouse buttons being pressed at the same time. It just didn't seem to work.

Well, this time, I was screwing around with the buttons, just running my fingers over them... and the Avatar jumped! It isn't anywhere near reliable, but it seems Linux *can* send Dosbox the proper "both buttons pressed" command (it works perfectly in Windows on my computer). It's just really particular about what it will register as such, which means the game still isn't playable... but there's a bit of hope here.

I'm always hoping someone else has had a similar problem, and managed to solve it. I've never actually seen anyone. Maybe this time.
