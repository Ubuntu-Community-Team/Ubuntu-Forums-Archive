### Post by amohanty on 2005-07-18
command line:
Applications->System Tools->Terminal

On the prompt:

>> tux[TAB]

thats the tab key. On pressing tab you should get a list of all apps whose names start with tux. Then type the full name and press enter.

HTH

AM
