### Post by alecjw on 2006-12-16
I installed battle for wesnoth a few days ago and really liked it, but multiplayer on the server didn't work. This is becuase the ubuntu edgy version is 1.1.8, wheras the server uses 1.1.12, so it just crashes.
I was so determined to play online that I upgraded to feisty. However, there are two problems with that:
1. Obviously, the average user doesn't want to upgrade to an unstable verison of Ubuntu just to play a game
2. Sooner or later, the official server will probably require the current version, 1.1.13, then we'll all be screwed.
So basically, apt can't keep up.

What I'm suggesting is an ubuntu wesnoth server, which always uses the version in the apt repos. There would be the other advantage that you can play against people who you know, rather than strangers.

Maybe there could even be a server for each version, a feisty.wesnoth.ubuntu.com, edgy.wesnoth.ubuntu.com maybe....

So anyone like this idea?
