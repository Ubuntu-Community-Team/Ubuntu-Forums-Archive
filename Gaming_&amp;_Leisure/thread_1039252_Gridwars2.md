### Post by 5nak3 on 2009-01-14
i've got it and play it regularly, great game (well i think so)

there is no story or mission, it is a simple pick up and play game. I like the simplicity and aim of the game, but it is mastering the game which takes time and skill, especially when the screen starts to get really crowded.

Personally, i've never been one to think graphics make a game, and so for me, games where the game play comes first are the real winners. In this one, despite the poor graphics, the effects on screen are nice, and the game play is something anyone can understand, mastering it however is a different story.

If you are looking for a game which has depth and story etc.. etc... look elsewhere. Want a game to play while drinking a coffee or waiting for a file to finish downloading/uploading etc... you can't go wrong with this.
