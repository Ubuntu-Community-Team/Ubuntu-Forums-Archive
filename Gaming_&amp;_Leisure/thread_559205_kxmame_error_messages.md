### Post by DoktorSeven on 2007-09-25
If the files you have are patches to full ROMs, you'll need the full (parent) ROM to run the patch one.  Also, some ROMs depend on others such as BIOS files for Neo-Geo or the PlayChoice-10 ROMs, for example.

Finally, some ROMs may have been redumped and older versions won't work in the newer versions of xmame/SDLmame.
