> **Jamsers said:**
> For one, there's a difference between free and open source? Please elaborate... so does that mean... that, um, you can have something that's not free, but still open source? Or something that's free, but not open source? I'm genuinely confused.There are two kinds of "free". There is "free" as in "free beer" and "free" as in "free speach." Just because a game is free does not mean it is free because it doesn't follow both definitions of the word "free".

There is also a difference between "free software" and "free-software" (aside from the hyphen).

Some real world examples would be Red Hat Linux. You can pay for it to get support, but it is an Open Source operating system. The Zimbra email server is Open Source, but if you want the support, you buy a license. Doom was a commercial game, but is now Open Source (and Doom3 is about to get released as Open Source pretty soon, if it hasn't already, I haven't been following it).

Open Source just means that the entire source code to the software is made completely available and has no direct correlation to if it is commercial or not.

Now, you also have software that does not cost anything but is closed source, for example iTunes or Winamp. You can download them and install them on as many machines as you want, but you are not privy to look at their source code.

I hope I did a good job of explaining the differences. If not, I can try to do a better job.
