### Post by nefrin on 2009-06-11
The only other flight sim that I have seen that can run natively under Linux is Xplane ([www.x-plane.com](www.x-plane.com))

It's not f2p, and it can be quite pricy (70$ American last check). Also, the install is huge (70GB+ for a full install, only a few GB for a basic install; pretty much the extra size is all scenary).

Speaking to an avid MSFS user who has tried X-Plane on the windows platform, he was not overly impressed, and thought that MSFS (either 2004 or FSX) was a better buy based on:

1) Graphics
2) User Friendliness
3) Flight Physics (X-Plane boasts realistic, but they felt funny to him)

Flight gear is your best bet to start with. The issue I've had is my good set of dual joysticks is not supported under ubuntu :(
