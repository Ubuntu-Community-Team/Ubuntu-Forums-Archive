### Post by FranMichaels on 2008-03-08
You should be able to run pretty much anything.

[http://frotz.homeunix.org/frotz/](http://frotz.homeunix.org/frotz/)
It's in the Ubuntu repos too.

As for text adventures, the fancy term for them now is interactive fiction. Archive here
[http://www.ifarchive.org/](http://www.ifarchive.org/)

Also, install the bsdgames package, it contains a port of the first Adventure game (also known as Collosal Caves)
Just type advent in terminal after installing


If you want to run your C64 stuff (anything, not just text adventures,) grab VICE
[http://www.viceteam.org/](http://www.viceteam.org/)

### Post by FranMichaels on 2008-03-08
> **Rytron said:**
> 
I just got frotz from the synaptic manager.
How do I start it and play a game?

I just downloaded this game:
[http://www.ifarchive.org/if-archive/games/zcode/AllRoads.z5]("http://www.ifarchive.org/if-archive/games/zcode/AllRoads.z5")

Now all I need to know is how to start it in Frotz.

It's easy.

Open a terminal and type
(if it's in your desktop folder)

cd Desktop
frotz AllRoads.z5

The other option is right click on the AllRoads.z5, and make it open with frotz. If it's not in the list, click custom command type frotz.

Just FYI, from terminal you can view the manual for the app
type
man frotz

press q to exit the manual
