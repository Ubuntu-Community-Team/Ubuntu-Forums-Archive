### Post by birdy on 2008-08-02
[Numpty Physics]("http://numptyphysics.garage.maemo.org/") is a free and open sourced clone of the popular Windows 2D puzzle game [Crayon Physics]("http://www.kloonigames.com/blog/games/crayon"). It is great fun and very addictive. 

Unfortunately Numpty Physics is only provided as a [deb package for hand held architectures]("https://garage.maemo.org/frs/?group_id=638"), and there is a [Windows port]("http://thpinfo.com/2008/numptyphysics/"). So, I’ve made a Ubuntu (Hardy Heron) deb package available [here]("http://sonofjon.wordpress.com/2008/08/02/numpty-physics-ubuntu-package/") for those interested.

Enjoy!
