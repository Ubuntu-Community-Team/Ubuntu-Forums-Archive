### Post by thatguruguy on 2011-08-31
It means that the engine that runs the game will be free.  The program that supplies the lighting, physics, etc.  All of the art, sounds, character designs and other stuff that creates the "look and feel" of the game will still be copyrighted and will still be owned by id.

Basically, it means that you can now freely make your own game based on the Doom3 engine.
