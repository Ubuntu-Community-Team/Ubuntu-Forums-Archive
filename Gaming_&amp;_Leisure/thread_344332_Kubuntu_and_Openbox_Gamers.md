### Post by dbd on 2007-01-22
Both Halo and Counter Strike are commercial products, and there is therefore no legal way to get them for free. If you want to find out how to steal them then please go somewhere else, these forums are not for software piracy, but the support of free software (free as in speech, not necessarily as in cost, although Ubuntu is 100% free of charge).
 
Also both games are pretty old now so you should be able to buy either of them pretty cheaply now. Counter Strike and Counter Strike Source both run very well in Linux (once you've done some work setting them up) but you may have some problems getting Halo to run well.

If you want some free games you may want to check out this list of games, most of which are free both in the sense that you don't have to pay any money for them and that you can look at the code to work out how they work and you are free to change the code to make them play any way you wish :)
[http://doc.gwos.org/index.php/Native_Games](http://doc.gwos.org/index.php/Native_Games)

And if you want a free deathmatch FPS I'd recommend Nexuiz:
[http://www.alientrap.org/nexuiz/index.php](http://www.alientrap.org/nexuiz/index.php)
