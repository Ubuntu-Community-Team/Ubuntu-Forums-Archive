### Post by ViperKnight on 2007-01-16
The User settings (like key binds, gfx settings and stuff) are stored in:
/home/*USERNAME-HERE*/.ut2004/

The actual game files are stored in:
/usr/local/games/ut2004/

That's the default directory however^^.  If you picked somewhere else then the game files location will be different.
