### Post by Mr. T on 2007-09-20
Two ideas come to mind:

(a) Make sure you're running the latest version of WINE

(b) You can force HL2 to run using a lower-grade renderer by going into steam, right click Half-Life 2, select Properties..., select Launch Options and enter the following:

-dxlevel 70

This forces the game to run using the DirectX 7 renderer, which looks like crap but would be very well supported in WINE by now. Run the game, get past the area with all the flares, then when you're well and truly clear, quick, remove the option you added and run again.

There's a chance HL2 will continue to run in DX7 despite removing the option, so if you were originally using one of the more powerful renderers, try going back to the Launch Options and putting in

-dxlevel 80
or
-dxlevel 90
