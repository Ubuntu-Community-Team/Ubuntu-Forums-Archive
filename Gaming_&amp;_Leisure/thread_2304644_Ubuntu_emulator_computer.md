There are a lot of emulators available for ubuntu, retroarch covers a lot of them.

I have set up an arcade cab with Lubuntu and the Mahcade frontend, several other frontends are available.
I recently switched to linux Mint XFCE because it is getting increasingly difficult to get Daphne, the laserdisk emulator running on 64 bit ubuntu, but on linux mint it runs out of the box.
