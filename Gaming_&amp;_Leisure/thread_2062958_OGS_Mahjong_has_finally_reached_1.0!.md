### Post by kornerr on 2012-09-26
After 2 long years of development OGS Mahjong has finally reached 1.0!

[Video]("http://www.youtube.com/watch?v=XdOzD_Hc2lQ")

**Final 1.0 release**
[LIST]
[*] 3 game modes: Mahjong Solitaire, Shisen-sho and Shisen-sho with gravity.
[*] More than 150 layouts. Layouts format is compatible with KMahjongg.
[*] Support for multiple tilesets.
[*] 4 themes: “Classic”, “Neo-classic”, “Flowers”, “Distros”.
[*] Support for background scenes.
[*] 3 scenes: “Room”, “Room Lite” and “Inside the computer”.
[*] Save and load.
[*] Hints and shuffle.
[*] Unlimited number of undos.
[*] Camera animations and dynamic camera (cursor tracking).
[*] Layers highlighting.
[*] 5 languages: Russian, English, German, French and Spanish.
[*] Adapting the game settings to your computer configuration during the first game launch.
[/LIST]

**Installation instruction:**
[LIST]
[*]**apt repository:** deb [http://packages.opengamestudio.org](http://packages.opengamestudio.org) stable ogs
[*]**Game:** apt-get install ogs-mahjong
[*]**Additional music:** apt-get install ogs-mahjong-music-addon
[*]**Editors:** apt-get install ogs-mahjong-tools
[*]**Everything:** apt-get install ogs-mahjong-full
[/LIST]
Downloads for other Linux distributions and Windows can be found at our homepage.

**Homepage:** [http://opengamestudio.org](http://opengamestudio.org)

Engoy and let us know what you think of it! :)
