I have honest to god no idea where to ask for help about this annoying problem. Hope someone here can direct me in the right direction.

Anyway here we go.

I am running a CS 1.6 Dedicated Server on my Xubuntu computer. Me and som mates play games on it all the time and it works like a charm overall but...

I get this message everytime i run the server:
[FONT="Lucida Console"]
"could not connect to Steam servers"[/FONT]

As you all may know, that means that there won't be any VAC on the server. 

I have read different forum threads about this problem and they all say that the problem is caused by the firewall. But I think that I have disabled my firewall by choosing the network adapter that isn't in use in Firestarter. 

(In Firestarter I have chosen my WLAN connection as my Ethernet device, but I get my internet from my wired connection. So I think I have disabled my firewall that way)

I don't sit behind some router or anything.

Any help on this problem would be greatly appreciated!

EDIT:
Problem solved, a friend of mine pointed out that the server ip wasn't correct. To fix this problem I just added +ip <correct ip> and everything worked :)
