### Post by steve8track on 2007-09-23
You could try DosBox to run the same games you used to run on Dos in the past.

As far as a LAN, if you got a second computer in the room, a crossover ethernet cable should work so you wouldn't need an actual router/hub/switch.  Another option would be to ask around for people with old computers they don't want anymore that still run DOS, and you could run the original game you liked.  You may be able to set up a null modem connection for network play on two old computers that don't even have ethernet.

Also, you could try WINE for running a windows first person shooter that you may know of, or PCSX for a playstation game you may know of.

Good luck
