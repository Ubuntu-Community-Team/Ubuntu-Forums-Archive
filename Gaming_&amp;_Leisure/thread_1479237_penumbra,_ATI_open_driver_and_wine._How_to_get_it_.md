### Post by madjr on 2010-05-10
Hi guys

been struggling a bit with penumbra and the ATI open driver

i have an Ati mobile xpress 1250


but i got it working bit better now


A) **native instructions:**

things i changed:
800x600
texture quality: medium
shadows: off
post effects: off
vsync: on
depth of field: off

these settings seem to work the best for my card





B) **Wine instructions:**

install wine or install playonlinux (which will install wine for you)

download windows version

install it

run it

similar options as native

restart it


*note: it may log you out of gnome sometimes, just log back in.*
note2: even better speed, use the cpu frequency applet and go full cpu (or on demand)


compare the 2, see which works best


native is doing well for me at the moment.

i actually tried penumbra in windows vista and many times it was bit glitchy.. so if it's going to be like that i prefer to stick with my linux. The open drivers are also getting better.


anyway if you want **full speed**, the only options for now are:
-installing ubuntu 8.04/8.10 or mint5/6 with ATI binary drivers using envy
-booting into windows
-wait for the open drivers to get better and asking the driver developers to work on penumbra performance/glitches! (anyone got a link???!!! :0)
