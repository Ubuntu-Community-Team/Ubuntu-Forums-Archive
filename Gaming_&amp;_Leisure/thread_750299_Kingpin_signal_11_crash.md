### Post by Macubus on 2008-04-27
That is a problem that has plagued the native client since I can remember (Mandrake 7.1 days?)  

I have found the game plays considerably better under Crossover Games than it ever did with the native Linux binary.. If memory serves, it was basically ported because one of the programmers (Ryan Feltrin if memory serves) got turned on to Linux..  It was ported over only a few days, and was never updated to 1.21, so I'm always amazed anyone ever gets it running this far out..  

If you already have a subscription to Crossover, you can get the Games part for free..  Pretty cool.

Another alternative would be to try it under stock wine..  It should work that way too with no tweaking.

Hope this helps.
