### Post by cgolson84 on 2010-12-06
WoW works great in 10.10.  For me it requires no messing with the settings at all.  Just install the newest beta of wine (1.3.8) and then install WoW from the battle.net installer.  Some people get better performance by running it in opengl mode but I just run mine how it is default installed and i get a perfectly playable framerate with most of the eye candy.
