It is hard to be any clearer than the above links but the steps are basically..

1. Install the open source opendjdk or Oracles java (Oracles java gave me better performance but it is proprietary software so if that is a problem for you use openjdk)
2. Download the minecraft.jar file from [https://minecraft.net/download](https://minecraft.net/download)
3. Right click the downloaded minecraft.jar file and select properties - in the permissions tab check "Allow executing file as program" is enabled.
4. Double click the minecraft.jar (or right click and select "Open with Oracle Java" ect) to open the login window - enter your account details and away you go....
