### Post by nchase on 2007-12-29
Pidgin is installed by default. If you do not have it installed, go to your terminal and type "sudo apt-get install pidgin"

or

use the synaptic package manager and find pidgin and select it to install it

or

use the add/remove programs tool, find pidgin, and select it to install it.

edit: once installed, you can select pidgin from the main menu. it is listed under "Internet."
