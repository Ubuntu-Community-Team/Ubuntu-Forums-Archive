### Post by lakersforce on 2007-04-14
My expirience is that it runs better under linux.

Like I described elsewhere the performance boost actually seems to be incredible...almost unbelievable.

On my old graphicscard I had to run Doom with lowest setting with 640x480 under Windows for it to run smoothly. Under linux I could go up to medium settings with 800x600 and it ran smoothly.

Since I got my new graphicscard (7600 GS) I can go all the way up to ultra high and everything in advanced settings maxed out and it runs smoothly.
And the rest of the computer is at least as old as Doom3, if not older.
