### Post by pardalet on 2013-02-03
[_Hairy Tales_](http://hairytalesgame.com/), an Action/Puzzler game developed by Arges studio, is built with Unity3D (version 3.5). Developers are reluctant to upgrade to version 4, and thus supporting Linux, because they're not sure if it would be worth the cost.

In order to gauge the real interest of the Linux gaming community they put up a poll on the game's forum in Desura:
[http://www.desura.com/games/hairy-tales/forum/thread/linux-support](http://www.desura.com/games/hairy-tales/forum/thread/linux-support)


You're all invited to go there and vote if you want to play this game natively (I certainly do! :P)


*Seen on GamingOnLinux: [http://www.gamingonlinux.com/articles/hairy-tales-to-be-ported-to-linux-if-enough-interest-is-shown.1700](http://www.gamingonlinux.com/articles/hairy-tales-to-be-ported-to-linux-if-enough-interest-is-shown.1700)
