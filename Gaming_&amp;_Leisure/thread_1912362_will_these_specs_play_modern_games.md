### Post by oldrocker99 on 2012-01-20
You could try installing Lubuntu, or, if you are already running, go to Synaptic or the Software Center and install lubuntu-desktop. Lubuntu uses the very light-weight LXDE interface, which will allow a usable (i.e., non-Unity or GNOME 3) desktop with low CPU requirements.

If you're looking to play shooters, you're in luck; there are more free, native linux shooter games available than just about any other kind.
