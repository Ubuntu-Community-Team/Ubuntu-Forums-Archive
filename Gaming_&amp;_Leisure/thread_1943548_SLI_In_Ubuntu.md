### Post by lite1979 on 2012-03-19
From the [wiki]("http://en.wikipedia.org/wiki/Scalable_Link_Interface") on Scalable Link Interface:

> In order to use SLI, a motherboard with an [nForce4]("http://en.wikipedia.org/wiki/NForce4"), [nForce 500]("http://en.wikipedia.org/wiki/NForce_500"), [nForce 600]("http://en.wikipedia.org/wiki/NForce_600") or [nForce 700]("http://en.wikipedia.org/wiki/NForce_700") SLI chipset must be used, although with the use of hacks[[23]]("http://en.wikipedia.org/wiki/Scalable_Link_Interface#cite_note-22") one can make SLI work on motherboards with [Intel]("http://en.wikipedia.org/wiki/Intel"), [ATI]("http://en.wikipedia.org/wiki/ATI_Technologies") and [ULi]("http://en.wikipedia.org/wiki/Acer_Laboratories_Incorporated")  chipsets. NVIDIA has stated that only their own chipsets can allow SLI  to function optimally, and that they will not allow SLI to work on any  other vendor's chipsets. Some early SLI systems used Intel's E7525 [Xeon]("http://en.wikipedia.org/wiki/Xeon")  chipset, which caused problems when NVIDIA started locking out other  vendor's chipsets as it limited them to an outdated driver set. In 2007,  Intel has licensed NVIDIA's SLI technology for its [SkullTrail]("http://en.wikipedia.org/wiki/Intel_SkullTrail") platform, and select motherboards supporting the [Intel X58]("http://en.wikipedia.org/wiki/Intel_X58")  (Tylersburg) chipset have unlocked SLI capabilities. Not all X58  motherboards support this technology, as NVIDIA offered it to  motherboard manufacturers at the cost of $5 per motherboard sold.[[24]]("http://en.wikipedia.org/wiki/Scalable_Link_Interface#cite_note-23") NOTE: As of the release of the [AMD 900 chipset series]("http://en.wikipedia.org/wiki/AMD_900_chipset_series"), SLI can be run on an AMD mainboard provided that it has the 970, 990X, or 990FX chipset on the board. [[25]]("http://en.wikipedia.org/wiki/Scalable_Link_Interface#cite_note-24")

I guess the important part of that is making sure your motherboard supports it and that both your video cards have identical GPUs. I'm currently using an nVidia nForce 4; I forget which chipset, but I remember it being SLI ready. I'm currently seeing a little bit of choppiness when running Half-Life 2 at full resolution, and I'd like to see if this helps you as well.
