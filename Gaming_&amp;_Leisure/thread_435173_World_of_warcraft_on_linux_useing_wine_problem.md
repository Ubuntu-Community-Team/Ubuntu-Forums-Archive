### Post by hikaricore on 2007-05-07
One thing you may want to try (most howtos mention this, some don't), turn off all full screen glow effects and set all graphic settings on low.  Basically clearout/lower everything in video settings including resolution.  This will need to be done in d3d mode.  Restart the game in opengl mode after saving the settings in d3d mode and see if it's made a difference.  Also if you're using beryl or compiz, it's pretty standard to disable them before starrting wow or your hardware will most likely be stressing.
