### Post by sudodus on 2013-01-16
I have the opposite problem. I can't find a chess program, that I can beat ;-)

Don't download them, install them from the Ubuntu repositories. Then you get versions that work.
```
sudo apt-get install gnuchess
```
will install a text mode application and
```
sudo apt-get install glchess
```
will install a graphical user interface.
