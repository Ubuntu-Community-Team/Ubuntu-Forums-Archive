### Post by Technophobia on 2009-07-24
Greetings Ubuntu Gamers,

This Sunday Ubuntu Game Night(UGN) will be playing Savage 2([http://www.savage2.com/en/main.php](http://www.savage2.com/en/main.php)). The start time like always is at 1800 GMT (2 PM EST) and will run as long as people keep playing. Once we have or chosen a server this thread and the MOTD in our IRC channel will be updated.

For more information please visit [http://gwos.org/doku.php/event:start](http://gwos.org/doku.php/event:start) or join us in our IRC chat room.

**#UGN on irc.freenode.net**

---------------------------------------------------------------

_**For new Players**_

In order to play this game you must first create a **_free_** account here:
[LIST]
[*][http://www.savage2.com/en/create_account.php](http://www.savage2.com/en/create_account.php)
[/LIST]

Once you have an account the next step is to download the client here:

[LIST]
[*][http://www.savage2.com/en/download.php](http://www.savage2.com/en/download.php)
[/LIST]

After you have the client downloaded you must run the installer. Open a terminal and navigate to the file you downloaded and then run the command below.

[CENTER]sudo ./Savage2Install-2.1.0.1-i686.bin[/CENTER]

The last step is start Savage 2 by going to your Main Menu > Games Menu > Savage 2.

---------------------------------------------------------------


See you there.
