### Post by B0rsuk on 2006-07-16
Hey

It's time we should get serious about getting linux supported as a gaming platform. Games are not only very fun (and exercise your brain, keeping it from aging!), but also a major factor behind linux adoption.
We need commercial games to run on linux, because, to be blunt, they tend to be of much higher quality than free/oss games. People like games, A LOT.

----------
Problem: only few commercial games run on linux.
My suggested solutions:
-------------

# **Post on game's forum:** search for linux thread if there's one, write a message. Or create new topic if necessary.
No need to rant, just say you don't want to pay for windows, or you just like linux so much. Tell them you'll buy another game (name it! spread the word about games that work with linux !) if this particular upcoming game won't work with linux.
Not only you have a good chance of reaching developers if you post on their forums, but you get other users intrigued. They may give linux a try. Some of them will post into your 'linux thread', which is good because it keeps bumping the thread, and sends a message there are linux gamers.

# **play already existing linux multiplayer** games.
This makes it much easier to spread the word about linux and needs of linux gamers. Neverwinter Nights 2 petition is getting somewhere (some attention from moderators) only because original NWN has a large linux playerbase.
Why is n2n multiplatform petition so big ? Because linux nwn1 is so popular  ! [http://www.ataricommunity.com/forums/showthread.php?t=444382](http://www.ataricommunity.com/forums/showthread.php?t=444382)
Why majority of gamers coming to bioware site seem to use linux ?
[http://www.bioware.com/_poll/view_poll.html?pollID=136](http://www.bioware.com/_poll/view_poll.html?pollID=136)
Because nwn1 was a massive success and created large linux playerbase.
Supply creates demand.

# **petitions signed by 20 people, especially on dedicated page impress no one. You need legions !** To get legions, you need lots of linux gamers ! If another game of this developer works well with linux, it should be easy to get support from them. If you try a petition, do it on developer's/publisher's forum.
Also, get support from linux gamers with similar interest. If upcoming Enemy Territory:Quake Wars looks good, linux UT2004 players will be interested. 

# **in multiplayer games, ALWAYS add linux to your name.** I'm changing my nickname from .b0rsuk to b0rsuk@kubuntu so I don't look like a hipocryte. I think ubuntu is known enough that even relatively clueless people will know what it is.
This has two benefits. First, people see their fave game can be played with linux, and they see how many linux players are there.
Second, other linux players will leave cover and also add linux to their nickname. It may be mostly because they use Only True Distro, but it works. It makes interest in linux as gaming platform even higher.
This is particularly common fashion in tremulous, at times about 50% people have linux tags in their names. You can even see someone like painsaw@freebsd

# Documentation, we need documentation !** If you get a game to work with wine, add a note to wine application database !** Help other people ! Don't let Microsoft mentality consume you ! [http://appdb.winehq.org/](http://appdb.winehq.org/)
