### Post by AndyCooll on 2007-03-23
> **gwoodard said:**
> okay, how do you play games from wine?



Wine is just the "middleman" so to speak. It sort of interprets expected Windoze code to that understood by Linux. There are a few ways to play Windoze games.

1. By using the terminal (as mentioned above).
2. I've created a shortcut to my game (which in effect is utilising terminal code):
```
wine "C:\Program Files\Sports Interactive\Football Manager 2007\fm.exe"
```
3. Sometimes you can just double click on the exe and Wine will kick in.
