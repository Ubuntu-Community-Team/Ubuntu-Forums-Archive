The problem is that while Windows is case-insensitive, Linux is not. Because of that, problems caused by bad coding can sometimes be glossed over in Windows, but will be a deal-breaker in Linux.  In the case of Mugen, you will need to edit some files to make sure that cases are used consistently in order to make it work under Linux.

For more details, you can read [this thread]("http://ubuntuforums.org/showthread.php?t=660745").
