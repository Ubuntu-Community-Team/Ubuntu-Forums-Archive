You'll need to install the proprietary driver as you are currently using the open source NV driver.
[http://www.nvidia.com/object/IO_18897.html](http://www.nvidia.com/object/IO_18897.html) (Your card is listed here in the supported hardware list)

**Driver download**: [http://www.nvidia.com/object/linux_display_ia32_100.14.11%20.html](http://www.nvidia.com/object/linux_display_ia32_100.14.11%20.html)

I don't believe the version included in the Restricted Driver dialog supports the newer GT series cards (I could be wrong).
