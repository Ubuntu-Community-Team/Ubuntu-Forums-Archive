### Post by Yfrwlf on 2008-06-18
> **Posterus said:**
> can someone help me out on installing the demo? i got the .sh file and idk how to install it

The GUI way is to right click on the file and select properties, go to permissions, and check the box to make it executable.  Then, single or double, depending on your options, left click on the file to run it.  It should pop up with a box asking if you want to run it, run it in a terminal, cancel, etc.  You can just click "run".  I installed my demo to my own user directory```
/home/myusername/games/Penumbra
```You will need root access to install it to secure locations, so I'd recommend putting it someplace like that or some other place that you have user access to.
