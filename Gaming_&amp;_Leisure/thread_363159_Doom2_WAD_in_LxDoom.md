### Post by jpkotta on 2007-02-16
On all of the Dooms I've used, there is a default directory(ies) and/or a command line switch to tell it where the WADs are.  So you RTFM and figure out what the switch is and/or what the directory is.  For DoomsDay, it is the -file switch and I don't think there is a directory.  SDLDoom looks in /usr/share/games/doom, IIRC.
