[B]
You cannot play any Windows games on Steam on Linux. Those games need to be made for Linux. Steam is not like WINE.[/B]

Steam supports 3 platforms, but not all games are supporting all 3 platforms.
That means that a game which is Windows only (lets say Portal as that game is currently free to play on Steam), then you cannot install it on the Steam for Linux beta. You have to install it on the Windows client.

So if a game which is Linux only, you cannot install it on a Mac client of Steam, but you will be able to see the game in your Library under "all games".

I have both Linux and Windows games on my Steam account, bought Trine2 yesterday, not all games are supported on Steam for Linux even if they have Linux clients readily available, you have to check on Steam if they got it supported for Linux (yet).
