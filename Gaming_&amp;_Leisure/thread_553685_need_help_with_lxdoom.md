### Post by DoktorSeven on 2007-10-24
[http://freedoom.sourceforge.net/](http://freedoom.sourceforge.net/) - Freedoom
Also can search for and download Doom 1 shareware (ep1 of Doom 1).

To run (from terminal):
**lxdoom -iwad /path/to/doom.wad**
replacing /path/to/doom.wad with the actual path and WAD name.
