### Post by eragon100 on 2008-03-16
Disable compiz if it's activated. I had exactly the same problem with it.

If it's already deactivated, just play the game in a screen, and set the desktop to the same resolution as the game. That way, it looks fullscreen, and it probably won't crash. :popcorn:
