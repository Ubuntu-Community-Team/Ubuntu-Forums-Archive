### Post by meng on 2006-09-18
It's in the repositories, so you needn't download it separately.
sudo apt-get install flightgear
OR
sudo aptitude install flightgear

You'll need universe repositories enabled. Search the wiki if you don't know how to do this.

### Post by PilotJLR on 2006-09-19
Flightgear is good stuff... but if you want a REAL simulator, and I mean one to blow away Microsoft FS, then check out X-Plane.
[http://www.x-plane.com/](http://www.x-plane.com/)

It's unfortunately not free... the demo lets you give it a try, though it disables the joystick after 5 minutes.
It runs extremely well on the Fedora Core 5 system, so I assume it would under Ubuntu as well.

If you're familiar with Part 141 training environments, then this is similar to a modern PCATD. If you pay extra for the hardware controls, you can actually certify X-Plane to be loggable as PCATD dual recieved (with a cfii present, of course)!

Check it out
