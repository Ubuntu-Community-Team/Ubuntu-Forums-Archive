Hi there Dirk!

To clear up the bit about free source, see FAQ link for the site:[http://www.openoutcast.org/en_faq.php](http://www.openoutcast.org/en_faq.php)

Here's one of the FAQs:-
- Will "open Outcast" come for free?
Yes.
The idea behind the development of oOC is creating a game that's fun and available to everyone with the option to modify it. Because of this we will make all of the code open Source so you can use the open Outcast framework or parts from it for your own games without any charge (being mentioned in the credits would be fair though ;)).


Following link is for download for tar files and for Sourcecode + Gamedata (about 66.9 Mb).
[http://www.openoutcast.org/en_techdemos.php](http://www.openoutcast.org/en_techdemos.php)

[Edit] Note source code for demo only, still under development.
