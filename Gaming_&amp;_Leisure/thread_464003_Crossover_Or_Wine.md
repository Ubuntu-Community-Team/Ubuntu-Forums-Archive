### Post by pay on 2007-06-04

### Post by cogadh on 2007-06-04
Wine has a more frequent update cycle (monthly) than Crossover, which can be both good and bad. While the updates almost always fix issues, sometimes they can create new issues. With Crossover, the code base is a little more static, though not necessarily more stable. Basically, if Crossover says it's not supported, then it won't work, but it might work with the latest version of Wine.
