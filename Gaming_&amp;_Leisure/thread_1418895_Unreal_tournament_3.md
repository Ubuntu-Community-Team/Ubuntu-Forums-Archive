### Post by Cabs21 on 2010-03-02
UT3 works on wine BUT you need to install it, set your settings, resolution, character name, character, etc,  Then you need to try a single player game and see if your mouse mapping is working correctly.  If it is not you need to create a mouse mapping script in the WINE registry to make it work.  This causes a problem as your mouse mapping does not turn off while in the menus and this makes your mouse useless so make sure you know your hot key commands.

Moues mapping for those that dont know is in simple terms is the placement of the mouse and how it responds to inputs.  EX)  I move my mouse to the left end of my screen.  the mouse moves until it reaches the end of my desktop then stops even if I keep moving the physical mouse the mouse on the desktop does not go on to infinity.  This is a problem when in a shooting game like UT3 where you are always looking around with the mouse.  Without proper mouse mapping settings you will only be able to look left or right as far as your mouse can travel across the screen, meaning if I am getting shot on the left and I look left if the mouse hits the edge of the desktop I can not keep turning left.  Changing the mouse mapping settings to lock the mouse in place will return the mouse to the center of the screen but still input a left movement so if I turn left I will never have the mouse hit the left side of the screen since it continually return to the center.

Any tech smart people who would like to correct me feel free, otherwise I will not learn.
