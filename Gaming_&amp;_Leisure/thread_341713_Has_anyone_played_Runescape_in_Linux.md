This is a small how-to on how to run Runescape on Linux natively by tempering with the official Runescape client for windows. This method is developed and released into public domain by me (tcp / ethoxyethaan). After following several steps you will be able to receive a standard java executable file and this will allow you to play Runescape outside the browser and without the need of plugins.

First, we need to install p7zip and the openjdk jre
On Debian-based systems(Including Ubuntu etc), this can be done by opening a terminal and typing:

> sudo apt-get install p7zip-full openjdk-6-jre


Now, change to your home directory, by typing "> cd ~".
Then run these lines:

> wget [http://www.runescape.com/downloads/runescape.msi](http://www.runescape.com/downloads/runescape.msi) -O $HOME/runescape.msi
7z e -o$HOME/rs_client $HOME/runescape.msi
7z e -o$HOME/rs_client/cabfile $HOME/rs_client/rslauncher.cab
mkdir -p $HOME/runescape/bin
mv $HOME/rs_client/cabfile/JagexAppletViewerJarFile.* $HOME/runescape/bin/jagexappletviewer.jar
mkdir $HOME/runescape/images
mv $HOME/rs_client/cabfile/JagexAppletViewerPngFile $HOME/runescape/images/jagexappletviewer.png




You can then run

> /usr/bin/java -Djava.class.path="$HOME/runescape/bin/jagexappletviewer.jar" -Dcom.jagex.config=http://www.runescape.com/k=3/l=en/jav_config.ws
 jagexappletviewer $USERNAME/runescape/images


to launch Runescape.
Note that, on my Debian system, $USERNAME is empty. You can simply replace it in the above command with your account username(If you aren't sure, type "echo $HOME" in a console. you will see "/home/*xxx*", *xxx* being your username.

> A good idea would be to create a new text file, and paste this into it:

#!/bin/bash
/usr/bin/java -Djava.class.path="$HOME/runescape/bin/jagexappletviewer.jar" -Dcom.jagex.config=http://www.runescape.com/k=3/l=en/jav_config.ws jagexappletviewer $USERNAME/runescape/images


> Save it as "launchRS.sh", right-click properties and make it executable, then run it by double-clicking. You can even make a shortcut to that on your desktop or in your startmenu.


I found this tutorial to be excellent; It worked great for me, and should help with browser-issues.
Let me know if anything needs clarification, or I made a mistake somewhere.
