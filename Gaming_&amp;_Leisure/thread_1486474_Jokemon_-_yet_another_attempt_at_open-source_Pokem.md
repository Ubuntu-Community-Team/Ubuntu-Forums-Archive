### Post by gnuwtey on 2010-05-17
This is a half-finished project I made for my Computer Science class. It never really got anywhere because I decided that it was too tedious for me to complete all by myself, so I've decided to post what I have done on here.

(I'm not making this part of the Buntoids/FreeMon project simply because it's written in Java and I don't think the developers would appreciate maintaining two different languages at once. That and they haven't been active for the past few weeks.)

For the open source release, I decided to name the engine "Jokemon" as there's a running joke (as far as I can tell) that any Java version of a program previously released in another language should be named the original name of the program, with the first letter changed to a J. ("Jython" is an example of this).

I want to create a SourceForge or GitHub page for this, but I have no idea where to start with this, so for the time being, have a [zip file with all the code in it]("http://www.mediafire.com/file/mitilj2zm4z/jokemon.zip").

Currently, there are two working parts of the program - a command-line battle engine, and a GUI overworld engine. (Only one of them works at a time so far. In order to make the other part work, comment out the first part.) I haven't got audio working just yet, but that'll come later.

The controls in GUI mode are: arrow keys for movement, space for the A button, and X for the B button. I haven't done a Start menu yet.



Another reason for me to upload this project is that I want to know what I'm doing wrong in my programming. (And I'm *definitely* doing a *lot* of things wrong. I have this feeling. I just don't know *what* exactly it is.)
