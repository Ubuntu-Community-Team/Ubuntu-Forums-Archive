### Post by plech.d on 2010-06-03
Hi all,

I know this is an old thread, but just in case anyone's still having trouble with this - this is what worked for me on Ubuntu 10.04 LTS 64-bit:

1) Follow the "Before Installation" part of this tutorial:
[http://gwos.org/doku.php/guides:64bit:et_wolfenstein?s](http://gwos.org/doku.php/guides:64bit:et_wolfenstein?s)[]=enemy&s[]=territory

2) Run "sudo linux32 ./et-linux-2.55.x86.run"

The installer should start now.

I hope this helped some of you. Enjoy!
