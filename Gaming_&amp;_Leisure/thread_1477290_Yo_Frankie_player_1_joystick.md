### Post by thordom on 2011-04-26
Found out you can play this game via the Blender software (you may need to install this).

Then you run blender with an option to disable joysticks, such as the one below.

[B]blender -nojoystick /usr/share/yofrankie-bge/levels/start_menu.blend
[/B]

Then from the menu select Game and then Start (or press P), this will run the game in the top right frame.

That's as far as I got #-o
