I am surprised no one mentioned this. I tried it quite some time ago and it was a great game but had problems with ATI cards, so i had crashes.

Anyway the game is called: **Babylon 5: I've found her**. 

It's on a russian domain server now. They made it for Win and Mac, but apparently there is a linux port as well (it used to be back then when i tried it on win as well). And i checked their forums a bit and it seems it runs on Ubuntu 10.04 well. 

I have to say it's been a while since i tried it but i can still remember that it had a story and very nice grapahics and special effects. as i remember the only issue i had was my ATI card crashing. I wrote a review and browsing some old files it popped into my eyes. so if you guys &gals have some time surf the web and give it a go. if it really works it might be a nice change form all those Q3 engine games :-)

From the site i am not quite sure if they are selling it now or what, but it used to be freeware (opensource project or something...).
