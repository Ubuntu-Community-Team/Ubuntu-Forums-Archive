### Post by slave-zeo on 2010-09-01
Hi there. I started building the daily SVN of the Dolphin gamecube/wii emulator. These builds are for x86_64 and i686 on Ubuntu 10.04. 

I wrote an automated script to build and upload it to my archive located at [http://www.alphaboxmedia.net/dolphin-emu](http://www.alphaboxmedia.net/dolphin-emu) It runs every morning around 9am. 

My build script also posts to my twitter account whenever the builds are ready. My twitter username is slavezeo. You can follow me if you want a heads up.

The build is not a .deb package it is a compiled binary in an archive. To use it just decompress it using the archive manager and put the 'Linux-x86_64' or 'Linux_i686' folder somewhere. Inside the folder is a couple of more folders and a 'dolphin-emu' binary. Double click the binary and you should be off to the races.

Make sure your system is fully up to date or this binary will not work.

I have only tested it on my system and a couple of virtual machines. So please let me know if it works as prescribed or not. If it doesn't work I'll try to help out where I can :) I've had a lot of fun with this emulator and i hope you will too.

Again the web address is [http://www.alphaboxmedia.net/dolphin-emu](http://www.alphaboxmedia.net/dolphin-emu)
