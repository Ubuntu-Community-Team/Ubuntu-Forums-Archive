### Post by eragon100 on 2009-06-21
> **jenova_skill said:**
> Fresh install on Cpu. I just dunno if theres some default settings I should change.... nething I should download.

You have a fresh install? You shouldn't have to change anything in the x-server config file, but you do need to download and install the official nvidia closed source drivers if you want to have desktop effects and 3d acceleration. (in other words: if you want to be able to play any games.)

Go to System -> administration -> Hardware Drivers, select the nvidia binary driver, and Activate it.

If you have already installed the official drivers, then there is nothing else you should need to do :wink:
