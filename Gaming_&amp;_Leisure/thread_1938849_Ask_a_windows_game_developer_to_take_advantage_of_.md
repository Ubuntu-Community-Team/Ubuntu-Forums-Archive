Not many...

But there is a new one that is getting momentum and a good number of developers (big and small) are willing to explore and take advantage of: **nacl**.

_**What is Nacl you ask?**_

Native Client (NaCl) allows you to port your existing apps (C, C++, C#, etc.) to the web while **maintaining just one code base**. Spacetime Studios ported their multiplayer online game Star Legends in **less than two weeks** from an existing code base of more than half a million lines of code.)

Some early adopters of the technology already include **Square-Enix, Unity Technologies, ubisoft, Bungie**, etc. (video presentation)
