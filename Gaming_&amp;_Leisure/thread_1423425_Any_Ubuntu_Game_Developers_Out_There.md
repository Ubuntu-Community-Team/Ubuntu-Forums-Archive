### Post by RootsLINUX on 2010-03-08
I'm a game developer. I've spent the last 5 years working on [Hero of Allacrost]("http://www.allacrost.org/"), an open source, cross-platform RPG. I've done 100% of my development for it on Debian and Ubuntu. In terms of scale, Allacrost is less ambitious than 0 AD (which I have been following for a long time) but still a pretty large project. The code is finally maturing though and we're getting to the point where our focus is shifting towards scripting and content creation, which is pretty exciting. We released an unstable beta of the game in its current state about a week ago and we'll announce an official release probably in a few weeks. Check if out if you get the chance. We're always looking for more contributors to help out as well.
