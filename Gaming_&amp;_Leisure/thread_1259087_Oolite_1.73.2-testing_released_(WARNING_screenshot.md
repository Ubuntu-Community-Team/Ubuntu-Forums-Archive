### Post by Diziet Sma on 2009-09-05
Oolite-testing is now up to version 1.73.4.

Deb packages (Jaunty and Hardy) for i386 and amd64 Oolite 1.73.4-testing have now been released.

Despite the 'testing' label this release is actually more stable than the current 1.65 'stable' release. The update includes a huge number of bugfixes, support for shaders and normalmapping, nebulae, stellar coronas, procedurally generated planetary textures, joystick hat-switch support, lots of additional features available for creators of expansion packs (OXPs), language support (German, Italian, Russian and French OXPs added already) and much much more...

An Autopackage installer version is also available. So far it has been successfully installed on:
[LIST]
[*]Ubuntu 8.04 & 9.04
[*]Fedora 11
[*]Mandriva Linux One 2009 Spring
[*]openSUSE 11.1
[/LIST]
A 64-bit Autopackage installer is now available also.  It has been tested successfully on the following 64 bit distros:

[LIST]
[*]Mandriva 2009
[*]Ubuntu 9.04
[/LIST]


Deb packages are now available for 9.04 Jaunty and 8.04 LTS Hardy via repository and direct download at [http://debs.oolite.org/](http://debs.oolite.org/)

Autopackages are available from [GURPO (BerliOS)]("http://developer.berlios.de/project/showfiles.php?group_id=3577")
or
See my signature to go to the Oolite Forums bulletin board for download links.

BerliOS has 1.73.3 available as both Jaunty debs and an Autopackage installer, as well as installers for Mac and Windows, and source code...

The new Autopackage has been successfully tested so far on: 

[LIST]

Re "snazzy effects", maybe I should have emphasised more that the pretty ships and stations in the screenshots above are not part of the core game yet...(once all core ships have been remodelled, we hope to include them as an option in the next stable release). The 12 ships and 1 station done so far, are currently available as Oolite eXpansion Packs (OXPs). The easiest place to get all of them right now is via [http://aegidian.org/bb/viewtopic.php?t=6586](http://aegidian.org/bb/viewtopic.php?t=6586)  I don't think there's a man page for the game, but you will find 3 PDF files, (a readme, a Reference Sheet, and an Advice for New Commanders) installed along with the 1.73.2 game.

Advice, trouble-shooting assistance and information are also always available at ["The friendliest board this side of Reidquat&#8482;"]("http://aegidian.org/bb/index.php") (How many boards can honestly claim they've NEVER had a flame-war? We haven't...:D) The Oolite board is very active and everyone is more than happy to be helpful, with none of the "RTFM noob!" attitude seen on many boards. There is also a FAQ and a huge amount of info and background available at the [Oolite Wiki.]("http://wiki.alioth.net/index.php/Oolite_Main_Page")

The Wiki also maintains a (more or less) up-to-date list of the [more than 200 OXPs]("http://wiki.alioth.net/index.php/OXP") available, covering additional ships, stations, missions, flavour packs, HUDs, replacement sounds, planetary textures, languages, and more... there are also plenty more OXPs currently under development or being upgraded.

The only other thing I would advise is to make sure you have the latest available graphics drivers for your system... many problems that we are regularly asked about can be resolved simply by updating drivers.
