I noticed a submarine simulation game "Danger From The Deep":
[http://dangerdeep.sourceforge.net/]("http://dangerdeep.sourceforge.net/")
It looks very interesting and has a Debian package.
Question: Did anyone try to install that .deb with Ubuntu ? Is there any problem ?
tia for any comment or suggestion.

I went to the happypenguin.org website and searched for "Danger from the deep"
In the forum, someone mentioned that in order to work with Anjuta 5.10, one has to download the Mandriva package:
dangerdeep-0.1.0-1mdk.i586.rpm, then convert it to a .deb with "alien" and then installed the created .deb package using "dpkg".
It works nicely. I played the game a few minutes to test it and it's **excellent**
A must.
