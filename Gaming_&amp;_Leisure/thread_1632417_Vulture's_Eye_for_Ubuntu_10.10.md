Hi all,
For anyone who might be interested I just installed Vulture's Eye 2.2.100 and thought I would share the packages needed.

The source code can be downloaded here 
[http://www.darkarts.co.za/vulture/downloads](http://www.darkarts.co.za/vulture/downloads)

You'll need p7zip to unzip it.

```
 sudo apt-get install p7zip 
```


Then install the following packages libsdl-ttf2.0-dev libsdl-mixer1.2-dev bison flex

```
 sudo apt-get install libsdl-ttf2.0-dev libsdl-mixer1.2-dev bison flex 
```

To install Vulture's Eye, you'll need to run the command make nethack-home from the directory of the source code.
