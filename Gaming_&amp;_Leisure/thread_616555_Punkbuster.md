### Post by Polygon on 2007-11-18
Punkbuster works in linux. The linux ports of quake 4 / quake wars and stuff use punkbuster i believe, and evenbalance has a punkbuster updater binary for linux. Also punkbuster was in the americas army linux port, and it still works in version 2.5 (the last linux version)

BUT

If you run a windows game that uses punkbuster in wine, it will  most likely NOT work and kick you because its trying to hook into windows API's and stuff, and since wine is not 100% perfect it runs into problems and goes OMG HAXOR and kicks you.

evenbalance has also stated that if any linux port of a game uses punkbuster, evenbalance will make punkbuster work for that port of the game.
