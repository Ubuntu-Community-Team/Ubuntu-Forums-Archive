### Post by DoktorSeven on 2007-08-03
Open Synaptic from the System Administration menu.  Go to Settings->Repositories and check all the Downloadable From The Internet sources on the first tab.  Close and hit the Reload button.

Note that this is on the version I have in Feisty, it may be different for older versions (e.g. see [here](https://help.ubuntu.com/community/Repositories/Ubuntu) for 6.06LTS).
