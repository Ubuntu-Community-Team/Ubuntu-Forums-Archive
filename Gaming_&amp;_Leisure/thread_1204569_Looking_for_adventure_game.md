### Post by BoyOfDestiny on 2009-07-07
Get [scummvm]("http://www.scummvm.org/"). 
> ScummVM is a program which allows you to run certain classic graphical point-and-click adventure games, provided you already have their data files. The clever part about this: ScummVM just replaces the executables shipped with the games, allowing you to play them on systems for which they were never designed!

Ubuntu includes 2 adventure games for scummvm: Flight of the Amazon Queen and Beneath a Steel Sky in the repositories.

Other games have been released for free as well, and would be nice if Ubuntu would package them up too someday...
[http://www.scummvm.org/downloads/#extras](http://www.scummvm.org/downloads/#extras)
