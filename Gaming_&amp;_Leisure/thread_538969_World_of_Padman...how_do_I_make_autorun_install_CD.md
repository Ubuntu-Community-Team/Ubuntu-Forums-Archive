### Post by hikaricore on 2007-08-30
Linux doesn't really have an autorun ability persay.

However both Gnome and KDE can autolaunch nautilus/konqueor windows when a cd is inserted.
But if you are looking for a way to run the game on other people's computers you will have to burn the contents of the installer to the CD or actually install it on said computer from the CD.
