### Post by dgrafix on 2008-08-15
Depends, but the game usually has to be compiled on each platform (with some exceptions, like platform independent binaries, interpreter (-bleaaagh!-) languages etc..) as long as there is no platform specific pains to work around (eg. Direct-X, win32 etc). 
Then you have the task of professionally packaging it (an installer per -major- distro) and supporting it when people have problems. (and for linux that usually means an influx of noobs trying to 'run' before they can even 'apt-get / yum')

This is why a lot of professional developers jump ship. However for myself because i develop using linux now and i am using pure cross platform, non GPL stuff i am prepared to try it and see how it goes.
