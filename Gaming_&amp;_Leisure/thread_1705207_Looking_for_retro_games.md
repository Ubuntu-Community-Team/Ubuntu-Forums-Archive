Is there a web page that lists games, there compatibility with linux and how to get them working? my Googling has brought up mixed results.

Ultima VII (The Black Gate and Serpent Isle) both play natively on Linux (or Windows or Mac, for that matter) using Exult, which is in the repositories. On installation, it'll ask you for the path for either/both games. Ultima VII is still available; try Google...

Also, gog.com sells ten or so years old games for low prices, and a pretty large number run very well under Wine. There is, in fact, a thread listing those games which DO run well under Wine, and there are some fantastic older games; Baldur's Gate I & II, Icewind Dale, Heroes of Might and Magic III (they have IV, but III is the better game), Arcanum (dwarves with guns!), Fallout, and many more, of most genres.

Seriously, though, search through with Synaptic, because I believe there is a Zeldaish game out there, and there are a number of climb & jump games as well, if retro is what you're looking for. Old Amiga games can (sometimes) be run with uae, in the repos. 

Happy gaming. I do admit that I (after 3 years of Ubuntu only) do dual boot with M$W7, solely to play games. It's the one thing that Windows does better than any other platform. EVERYTHING else I do on my PC, I do with Linux.
