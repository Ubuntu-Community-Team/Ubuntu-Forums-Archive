### Post by DoktorSeven on 2008-02-11
No, Model 2 games generally do not work at this time, MAME devs are still working on it.  Some games will get to title screens and maybe even farther, but emulation is still not complete at this time.

And SDLMame is exactly the same emulationwise as mainline MAME.  The actual emulation code is NO different, since they use exactly the same codebases.  The ONLY difference is the way each one renders the emulation onscreen and interfaces with gamepads, etc (i.e. Windows GDI/DirectX/whatever technique it uses vs. SDL).  So if it works/doesn't work with mainline MAME, it will be exactly the same with SDLmame.
