> **maryllchan said:**

you can use the terminal - **sudo apt-get install neverputt**

or you can use the add/remove utility in the applications menu

or you can use synaptic package management tool in system-administration menu

or you can use .deb files (double-click) you can get them from [www.getdeb.net](www.getdeb.net) for example

or you can use alien to convert .rpm files to .deb files

or you can get the source code and compile it yourself

the choice is yours! :D
