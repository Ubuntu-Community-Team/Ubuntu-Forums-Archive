### Post by stevejesus on 2008-02-12
Hey everyone,

For anyone interested in Sega Saturn emulation using an 64-bit ubuntu system thats not interested in building it from source, I have made a package.  You can get it here;
[http://www.myfootinyourass.com/Ubuntu/64/]("http://www.myfootinyourass.com/Ubuntu/64/")

This .DEB was built on a dual-core machine, and judging by the stuff that scrolled by when I configured it, it showed the system type as SMP.  SO, this may only work on SMP machines?  If anyone has a single-core athlon 64, let me know if this works out for you.  Likewise, anyone who installs this package let me know how this works out.

If there is a need out there, I will start making these packages available as yabause updates.  Also, once I get my PIII machine set up (I just moved) I will start building 32-bit packages as well.

PLEASE support this emulator.  Sega Saturn emulators have always been very far behind.  This is our best chance to get a fully compatible Saturn emulator in Linux.  It's not quite thre yet, but its getting there.
