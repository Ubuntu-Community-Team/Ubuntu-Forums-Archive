I have got a great idea how to speed up and increase the number of games in Linux. My idea is that we create a special web site, maybe called Linux Gaming Database in which we post our created 3d models, which can then help Game developers to speed up their process of creating games.

By creating 3d models, I'm mean that we ordinary Linux users use 3d programs like Blender and we create different models. Like creating cars, characters, buildings, cities, planes, environments and all other things which we would like to see in games. Then we post those models to this site and game developers can look for things that they want to have in their game. This way we simplify work for game developers. So they do not need to spend hours and hours for creating 3d environments. They only need to do the hard part writing the program.

I have got this idea from game GTA in which you can modify the game in your own way using GTA mods. Here you can see the example how we should create Linux Gaming Database: [http://www.gtagaming.com/downloads]("http://www.gtagaming.com/downloads")

With Linux Gaming Database we can all give our contribution of making new Linux games.
