### Post by cyb3rkn19ht on 2010-01-22
L-Echo is a free clone of Echochrome a PSP/PS3 game. This is a video of L-Echo in action: [http://www.youtube.com/watch?v=-9M15RBDUDY](http://www.youtube.com/watch?v=-9M15RBDUDY)

I just recompiled L-Echo on Ubuntu 9.10 64-bit. Here is a link to the working zip: [http://www.beverstock.net/lan/lEcho.zip](http://www.beverstock.net/lan/lEcho.zip)
You will need freeglut3, which is available through apt. 
```
sudo apt-get install freeglut3
```

Enjoy!
