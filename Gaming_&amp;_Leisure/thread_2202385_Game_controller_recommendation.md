### Post by leo-ms on 2014-01-29
Honestly in-terms of compatibility, specifically with games that are released on Steam, you might be better off using an Xbox 360 controller since that's the controller that most developers are likely to test out with their games (and some games don't let you rebind controller buttons etc). I can't comment on how well a PlayStation 3 controller will work with most games on Ubuntu but I certainly know that the PlayStation 4 controller seems to work mostly fine with some games on Ubuntu.

### Post by xREDEMPTIONx on 2014-01-29
Check out the logitech F310 or F710 (wireless). They support dinput for older games and xinput for newer games by simply flicking a switch on the controller. The xinput mode emulates an Xbox360 pad which the majority of newer games support as mentioned above. You will get out of the box support in Ubuntu in either mode. It has the same buttons as the 360 controller as well so games that have actual button icons ingame will work perfectly with it. I have a dualshock 3 and 4 which I also use in bluetooth mode in Ubuntu but they require extra software such as qtsixa, xboxdrv and ds4drv to get the emulated xinput mode support. Dinput is supported with both controllers over USB OOTB. I use the dualshock 4 the most because it is just amazingly comfortable but at the end of the day my recommendation stands with logitech pad for no fuss, no setup required gaming.
