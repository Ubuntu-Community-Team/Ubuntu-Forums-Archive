### Post by Quake on 2004-11-30
Some Games are already Available for Linux, such as:

Quake 3: Download the Linux patch [ here](http://files.filefront.com/SEARCH/;2621770;;/fileinfo.html) and copy  pak0.pk3 to the base directory of where you installed the Quake patch.

UT2004/2003: The Linux installation is on the last CD.

Doom 3: Linux Binary: [http://files.filefront.com/SEARCH/;3750938;;/fileinfo.html](http://files.filefront.com/SEARCH/;3750938;;/fileinfo.html).

UT: Install the [Full Patch](http://www.3dgamers.com/dlselect/games/unrealtourn/ut-install-436.run.html) with the original CD in the Drive.

Enemy Territory: Download it here [http://files.filefront.com/SEARCH/;1063789;;/fileinfo.html](http://files.filefront.com/SEARCH/;1063789;;/fileinfo.html).

Tribes 2: You will probably find someone on ebay selling it...


Several Linux gaming sites: [Linux Games](http://www.linuxgames.com),  [Tux Games](http://www.tuxgames.com/), [Happy Penguin](http://www.happypenguin.org/)

For all the other games not available for linux, you can play them with Cedega.
