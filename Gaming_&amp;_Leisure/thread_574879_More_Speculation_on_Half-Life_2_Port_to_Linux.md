### Post by cogadh on 2007-10-23
They wouldn't have to port Steam just to get HL2 on to Linux, you don't actually need Steam to run HL2, they just programmed the game's copy protection to make you use it. However, if Valve is going to be serious about Linux, then they should port Steam and add a section to it for Linux games, like the sections they already have for individual game companies. Id could sell their Linux stuff through it, as well as anything Valve ports. 

The additional benefit of doing this would mean that they could clearly show exactly how much money they make off of selling Linux games. Most current Linux ports require the purchase of the Windows version then downloading a native client or using an alternate client provided on the install disk. Since the initial purchase was not of a "Linux only" version of the game, there is no way to show that the game was bought for Linux and not Windows. Having Steam on Linux could change that.
