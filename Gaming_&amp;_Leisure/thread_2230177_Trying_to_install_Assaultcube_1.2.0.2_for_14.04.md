### Post by oldrocker99 on 2014-06-17
Assault Cube is in the Ubuntu repositories, and is installable with Synaptic, or the Ubuntu Software Center, or opening a terminal and typing:
```
sudo apt-get install assaultcube
```

It will ask for your password, and shows nothing when you type it in. That's all there is to it. Installing software and getting updates from the command line is one of the great joys of using Linux.
