### Post by jim_deadlock on 2012-06-01
I would suggest installing Ubuntu "alongside" Windows (you'll see this option during install) which will give you a dual boot, so if you have any problems with Ubuntu you can always revert back to XP and no harm done.

I mention this mainly because of the other thing to watch out for: if you have an ATI/AMD video card you may need to fiddle around with manual drivers before it will give you a GUI and it can be frustrating (I speak from personal experience). If you have NVIDIA or integrated graphics it should be plain sailing.

Also, make sure you have a wired connection during installation - you can go wireless once you're up and running.

Oh and another thing, if your hardware is low-spec and you find that Ubuntu is a bit sluggish, all you need to do is switch to a lightweight desktop (eg XFCE) and you'll notice the difference.

Once you've built a PC you'll never go back :)
