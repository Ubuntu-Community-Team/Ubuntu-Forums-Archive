### Post by Chesamo on 2010-10-24
[http://www.petitionspot.com/petitions/steamchat](http://www.petitionspot.com/petitions/steamchat)

I've drawn up a petition to Valve to open the Steam Chat API. From the page:
> Valve's Steam software has revolutionized the face of modern gaming. In particular, the social gaming network of Steam Community provides one of the best methods of communicating and connecting with like-minded gamers. The in-game overlay, in particular, seamlessly enables in-game chat with other Steam users through Steam Chat (also available outside the overlay). However, it is currently impossible to sign into Chat any way apart from loading Steam on a supported platform, which isn't always possible. If Valve developers created a public API (application programming interface) for Steam Chat, users would be able to sign in to Chat using most chat clients (Pidgin, Adium, etc.) as well as opening up new functionality to Steam API-based software (such as My Steam on the iPhone).

This petition does not ask that the Steam Chat system be open-sourced, though that would be nice. All we ask is for the ability to chat with our friends... Being given the freedom of choice in how we communicate.
If this were to happen, libpurple plugins would be able to be created, meaning Pidgin and Empathy can communicate with Steam Chat. Meaning one need not be signed into Steam itself in order to communicate with Steam friends, meaning one need not install WINE on their system just to use Steam Chat (it is presently the only reason I have WINE on my Linux laptop -- I can only game on my desktop, which is running 7/Ubuntu dual-boot).

[http://www.petitionspot.com/petitions/steamchat](http://www.petitionspot.com/petitions/steamchat)
