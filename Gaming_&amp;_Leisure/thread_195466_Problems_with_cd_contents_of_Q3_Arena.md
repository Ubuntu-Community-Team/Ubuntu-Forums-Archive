### Post by spyke01 on 2006-06-12
Hi guys, i was attempting to get my q3 install for linux going and ran into a snag. The disc mounted with the Q3Arena label, but wasnt seeing the contents correctly, i tried it in both of my drives, but it stayed the same, i was fixing to restart but thought id try to umount and mount it again and it worked, thought id share this for anyone experiencing problems

unmount cdrom
```
umount /media/cdrom
```
mount it
```
mount /media/cdrom
```
