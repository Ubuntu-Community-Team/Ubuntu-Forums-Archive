### Post by jgates0 on 2010-10-15
My first post, but I noticed there was little documentation on NHL gamecenter live so I thought I would post my success.  I was a little hesitant to jump in and pay the $159 for it with no documented success, but thought I would give it a try anyways (figuring that it is only flash, so there is no reason it shouldn't).

I am running Ubuntu 10.10 64bit with both chrome and firefox on an Asus UL50A series laptop.  Adobe flash player version 10.1.85.3 is installed.  According to speedtest.net my download speed is 2.24Mbps.

I am able to run gamecenter live fullscreen with 4 games displaying in mosaic mode and have no picture or sound issues.  The picture quality is pretty good.  I would rate it slightly better than standard definition, but definitely better than free streams.

For any Ubuntu users who are curious gamecenter passes in flying colors.
