WoW is prolly one of the best wine game around, install SOMETIME have issue (and even then, not so often) but if you have an arleady installed client, just run it directly on 100% stock wine from the ubuntu software center.

it MAY (most often it go fine) have issue with updating/installing, realy, if you happend to encounter these issue, a windows xp virtualbox is a good workaround, and then you can game on your linux OS.

if you want some better assistance, i will be glad to help you, just drop me your favorite IM's account in private message

Edit: Once installed, one good thing to do so the game runs optimaly is to  change to OpenGL  (direct X is more than playable, but OpenGL bring it even better!) 
To do so, go in your WoW folder, subfolder "WTF"   open the config.wtf with gedit (or other)   and add    SET gxApi "opengl"             pretty much anywhere in the file. though for clarity reason, i personaly place it right before SET gxMonitor.
And it is strongly recommanded to play in Windowed, or Full Screen Windowed,   Traditional full screen with ANY wine game tend to be terribly buggy (mostly when alt-tabbing) and can in some case (not necesarely with WoW) **** up with your screen resolution if they are not the same.  Realy, Both kind of window mode work wonderfully and makes it more enjoyable, yet, up to you to see what fits better for how you play. someone who dont alt-tab at all wont mind full screen in most cases
