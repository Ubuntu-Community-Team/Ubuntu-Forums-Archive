### Post by oldrocker99 on 2014-05-18
First, there is an open source trucking game called Rigs Of Rods, but I just tried to install it and got a 404 error on the repository. However, Steam has a Linux version for sale of the quite popular Euro Truck Simulator 2, which I have and is certainly what you may be looking for. Here's a guide for beginners on YouTube:
[http://steamcommunity.com/sharedfiles/filedetails/?id=155811128](http://steamcommunity.com/sharedfiles/filedetails/?id=155811128).

Second, there is a program called wine, which will run many (but by no means all) Windows programs. There is also a user-friendly GUI for wine, playonlinux. Both are in the repositories, and the fastest way to install them is to open a terminal and type

```
sudo apt-get install wine playonlinux
```

PlayOnLinux will be in your Games folder, and it has a raft of programs it can install and, mostly, get them running.

### Post by mister_k81 on 2014-05-24
Well, one of the absolute best trucking games you can get on PC (Windows or Linux) is Euro Truck Simulator 2: [http://store.steampowered.com/app/227300/](http://store.steampowered.com/app/227300/)

The game is currently $24.99 on Steam, but it does go on sale a lot for much cheaper. The game does use European truck models and is set in European countries. But it does have an option to switch the traffic to the right hand side of the road if you want to give it a North American feel. The game is great though, definitely one of the best trucking sims you can get on the PC. And it even runs natively on Linux with no Wine required.
