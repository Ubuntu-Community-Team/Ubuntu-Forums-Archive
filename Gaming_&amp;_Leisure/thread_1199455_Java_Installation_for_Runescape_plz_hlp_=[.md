### Post by magmon on 2009-06-29
ok, its real easy if you know where you're going. First, go to system-> Administration->Synaptic package manager. Then, in the search bar, type java. Install the java 6 runtime, then scroll down and install the java 6 plugin. Runescape and all other java material should now work! If it doesn't, PM me and I'll see if I can help you further.

A quick note, you will have to restart any browser that was open during the installation.
