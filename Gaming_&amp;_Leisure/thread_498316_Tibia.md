### Post by troseph on 2007-10-04
Extract the files from the gZip provided from Tibia's site to your desktop then, to run the Tibia Binary:

```

$ cd Desktop/Tibia
$ chmod 755 Tibia
$ chmod +x Tibia
$ ./Tibia

```
