### Post by JonnyM on 2008-06-20
First of all, the latest version on AA for Linux and Mac is 2.5.0, I don't think there is anyone still playing 2.3. 
The semi-transparent logo on startup is caused by compiz, You need to turn off all desktop affects in the apperance settings.
The stripes you see on all the textures are caused by a bug in the Nvidia drivers packaged with Hardy. Using synaptic packagemanager install Envy or EvnyNG and use this tool to install the latest Nvidia driver.
