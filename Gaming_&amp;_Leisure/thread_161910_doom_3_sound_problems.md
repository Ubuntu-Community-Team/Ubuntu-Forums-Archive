### Post by deathbyswiftwind on 2006-04-18
Okay this is what you need to do

run in your terminal

doom3 +set s_driver oss

this will cure the problem and you only need to run it like this the one time. the same works for quake4 since they are the same engine just in case you ever get it all you do is enter

quake4 +set s_driver oss  and it cures that problem.
