### Post by DoktorSeven on 2007-10-24
Either:

from terminal: **VisualBoyAdvance /path/to/game.gba**

or install visualboyadvance-gtk and run **gvba** for a GUI.  (Not sure if this package is in older versions of Ubuntu -- it's in Gutsy.)
