### Post by acoustibop on 2007-12-05
pSX is really easy to install.  You'll need a graphics card with basic 3D capability.  Install libgtkglext1 - it's in the repositories, and is the only extra dependency pSX needs on a Ubuntu system.  Download the [pSX file]("http://psxemulator.gazaxian.com/"), tar it in the location of your choice (your home folder is favourite - just save the file there, open a terminal and enter tar xvpf <name of pSX file), add a Playstation BIOS, cd into your new pSX folder and enter ./pSX.

Or, check [dfreer's thread]("http://ubuntuforums.org/showthread.php?t=394097") - he maintains a repository in which he includes pSX packages.
