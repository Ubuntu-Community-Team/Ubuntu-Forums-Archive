Thanks!

You need to select the 32 bit client (the 64bit HotS crashes at launch).
On the launcher:
Settings -> Game Settings -> Heroes of the Storm section -> Launch 32-bit client

Performance is generally fine in game, although it does out of memory and crash after a game or two. I've taken to restarting it after each match.
[https://appdb.winehq.org/objectManager.php?sClass=application&iId=15978](https://appdb.winehq.org/objectManager.php?sClass=application&iId=15978)

You can check which version of wine you are running by executing "wine --version" at a terminal.
