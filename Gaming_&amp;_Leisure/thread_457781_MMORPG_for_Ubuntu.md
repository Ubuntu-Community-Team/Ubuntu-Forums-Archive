### Post by cisforcojo on 2007-05-29
To install the Mana World:

```
sudo gedit /etc/apt/sources.list
```

add this to the end of it:

deb [http://bertram.ifrance.com/](http://bertram.ifrance.com/) ubuntu edgy

(I tried changing edgy to feisty but it doesn't work)

Save & Close

```

sudo apt-get update
sudo apt-get install tmw

```

Say yes to everything and you're good to go. Should show up in Applications->Games.
If not, I'd suggest trying 'tmw' in a terminal.
