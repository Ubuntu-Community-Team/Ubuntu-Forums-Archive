To install it from Synaptic go to the menu item System --> Administration --> Synaptic Package Manager.  First, make sure that you have the universe repositories enabled...to do that in Synaptic go to System --> Repositories.  Select Ubuntu 5.04 "Hoary Hedgehog" (Binary) and hit the Edit button.  Make sure "universe" appears somewhere in the Sections field.  Once you do that, close the repositories window, search for scummvm, check the box, then hit Apply.
