### Post by justin whitaker on 2006-10-19
This is a very resource intensive game...and it uses the latest physics and DirectX technology...on top of what is basically an Unreal engine. 

Maybe Cedega will get it to install, but I doubt it will run. 

What you could try is:

1. Use Wine to install the physics driver.
2. Use Wine to install the latest direct X (you are not supposed to do this!)
3. Change the Wine version to XP. 
4. Try to install it.

If the CDs won't load the game, try copying all content to a folder and installing from there.

Good luck, let us know how it goes.
