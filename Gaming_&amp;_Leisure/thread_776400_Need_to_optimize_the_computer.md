Well if you want some real speed you could shut down gnome and load up Xwindows.Hit CTRL+ALT+F1 and then type:

> sudo /etc/init.d/gdm stop

> xinit -- :0 vt8

Then you can just cd to a game directory and boot it up with command line.You should notice a slight speed increase,for me I have to play Quake 4 this way because its slow other wise.
