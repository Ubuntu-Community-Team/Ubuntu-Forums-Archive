### Post by BigSilly on 2008-12-26
I can tell you for sure that Vice City works perfectly with the latest WINE and has done for a while. I still have my original copy and it installed and plays even better than it did on XP for me. The movie doesn't play at the start though. You have to press a button to skip into the actual game, but otherwise the game is perfect. You can access the movie of you wish in the GTA folder in your /home/.wine directory that is created when you start a new game.

Can't speak for the rest though sadly. I tried to get PES6 to work a while ago but couldn't. If that changes I'll post up for sure. :)
