### Post by cogadh on 2007-06-04
The Steam version of COD2 needs to be run from Steam, the retail box version does not. According to CodeWeavers:
> Currently CrossOver only "officially" supports running Call of Duty 2 under Steam. The stand alone CD version is not supported due to CrossOver lacking support for CD copy protection.
Basically you will need to get a "no-CD" crack in order to run the retail box version of the game.
