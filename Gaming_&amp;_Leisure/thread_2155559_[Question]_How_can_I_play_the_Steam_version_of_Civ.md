### Post by oldrocker99 on 2013-06-18
If you install PlayOnLinux (I heartily recommend that you do so), you can use its built-in Civ 5 script, which will ask you if you want to use Steam, installs the best version of wine, Steam, and the various other Windows dependencies. You'll have to enter your emailed Steam code, since it treats each new installation of Steam for Windows as a "new machine."
