Well first thing you should do is pass all the Cedega tests. Once you put in a CD, there is an option to mount, then you need to navigate and point to the Setup or Install .exe. There is no automatic install like in Windows. Check the unoffical Cedega wiki for some help. And since you are a subscriber, you should be able to get help from transgaming.
