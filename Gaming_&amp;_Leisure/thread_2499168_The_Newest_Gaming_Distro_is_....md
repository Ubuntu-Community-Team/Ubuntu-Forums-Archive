### Post by Tadaen_Sylvermane on 2024-07-19
I'm always on the hunt for the best for both programming / coding & gaming. And honestly they are all about the same. Minor differences here and there. Same general performance of course I don't measure beyond how it "feels" when I'm running a game. I've found Debian stable just as capable as the latest Noble LTS. I've had success with Zorin and Fedora Silverblue. Arch of course works. With few exceptions I haven't had any exceptional experiences gaming on Linux. All fairly similar. Hardware is the only limiting factor. In my case with AMD it's a bit better but performance wise its no different than Windows performance, maybe a bit better despite going through Wine / Proton + whatever front end if needed. I use Bottles flatpak for Battle.net stuff.

I will say that little things work on the newer software better than the older. In my case with Battle.net games on Debian Wayland Gnome if the screen goes into powersaving it crashes my game when the screen wakes up. The current Ubuntu Noble has no issue with this and resumes flawlessly. Other than that I haven't had any kind of life changing experience on any specific distro over any others.
