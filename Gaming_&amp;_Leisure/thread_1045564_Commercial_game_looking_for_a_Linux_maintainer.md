### Post by Vadi on 2009-01-20
Hi all,

[**Caster The Game**]("http://www.casterthegame.com/") is looking for someone to help port the game to Linux. Along with setting up a cross-distro distribution method for it, and provide support for the game.

'Porting' the game will be relatively easy - the game was originally developed on Linux, and then ported to Windows and Mac. The developers just don't have enough time to setup and support the Linux distribution, so they're willing to let someone help them with that. The more native games the better, right? :)

Some technical details about the game: sdl is used for image loading and input handling, openGL for rendering, and [irrKlang]("http://www.ambiera.com/irrklang/") for audio. 

On the distribution side, support should be available for the mainstream distributions (OpenSUSE, Ubuntu, Fedora) and any that you're familiar with. A human-friendly method of installing and updating the game is preferred - so .debs and .rpms, or a graphical [MojoSetup]("http://icculus.org/mojosetup/") installer would do, but a text-based shell script wouldn't.

**A portion of the game's Linux sales will go to the maintainer - so if you're seriously interested, [please drop a line]("http://www.elecorn.com/caster3d/links.html") (include any experience with packaging/maintaining if possible).**

Thanks for helping Linux gaming grow!
