### Post by Cappy on 2007-12-22
Not a bad game! I'm running 64-bit ubuntu so I had to use getlibs to get the missing dependency.

A few problems:
Dark parts of the dungeon are too dark - it's impossible to tell what kind of object I see. This may be intended. No way to adjust in settings.
Wizard module crashes.
Controls are pretty good but I had a hard time figuring out that the number pad was movement. I was expected WSAD. Be nice to have the "controls" section just so it can be seen what the default keys for everything are.

Overall very fun ... gameplay is a bit like Diablo imo.
