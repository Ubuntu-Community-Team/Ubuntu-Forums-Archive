### Post by cogadh on 2007-06-14
With Wine/Cedega/CrossOver you can play almost all Windows games that require DirectX 8 or lower (or no DX at all for the really old games). Some DirectX 9 games will work, but they often don't run very well. None of theDirectX 10 games will work at all. Check Wine's application database for whatever game you want to try to see how successful it is: [http://appdb.winehq.org/](http://appdb.winehq.org/)
