### Post by HoKaze on 2009-08-21
> **charlieg said:**
> Looks like somebody started one, but I don't know how playable it is:
[https://sourceforge.net/projects/a3dspaceshooter/](https://sourceforge.net/projects/a3dspaceshooter/)

I decided to test it but the source doesn't include a Linux Makefile and thus I was unable to compile the source for it. I did get to test out the windows version and this project is far from playable yet. Basic movement in 8 directions, models for arwing and some terrain/asteroids exist, as does shooting (and destroying objects) as well as some collision (namely asteroids explode if the arwing hits them but nothing else. No damage, no death, no crashing if you hit the surface of this tunnel rock you fly though).
Long, long way from being even a proper demo, but it shows lots of potential. The fact that the project's homepage isn't even done yet and it was started in 2007 doesn't bring much hope to us wanting a star fox-like game for Linux.
