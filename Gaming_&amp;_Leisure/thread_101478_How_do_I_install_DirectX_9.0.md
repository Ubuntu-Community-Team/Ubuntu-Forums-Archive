### Post by ubu-for on 2005-12-16

2. Install winetools via Synaptic.

3. Start winetools

4. Choose: "Base setup"

5. Install everything (IE only once!)

6. Choose: "Modify the Wine configuration"

7. Choose: "Windows ME"
