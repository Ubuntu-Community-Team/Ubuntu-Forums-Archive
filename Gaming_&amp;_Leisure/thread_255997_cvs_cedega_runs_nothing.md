### Post by DoktorSeven on 2006-09-14
It's very likely that hardly anyone uses it, and many who do have figured that it won't run very many games.  

Probably everyone else is just using the current version of Cedega (legally or... *ahem*... otherwise :( ), or sticking with Wine (I am just using an older version of commercial Cedega from several months back, when I let my subscription lapse).

I would assume that besides the whole copy protection thing, though, that cvscedega would be fairly similar in compatibility with commercial Cedega, and the Cedega Wiki lists [NFS2SE](http://cedegawiki.sweetleafstudios.com/wiki/Need_For_Speed_2_SE) as playable, [SC3K](http://cedegawiki.sweetleafstudios.com/wiki/SimCity_3000) as non-working (best to find the native port!), [minor issues with B'42](http://cedegawiki.sweetleafstudios.com/wiki/Battlefield_1942) (nothing about the installer though), but nothing about CFS or MM.  

Cedega has always been *extremely* hit-or-miss with me.  I look at it and Wine this way -- if it runs a game, that's great, but if it doesn't, oh well, no big deal.  In other words, I don't rely on it to run anything.  I know many wish it did a better job because they want to play their Windows games without being in Windows, but what can you do...
