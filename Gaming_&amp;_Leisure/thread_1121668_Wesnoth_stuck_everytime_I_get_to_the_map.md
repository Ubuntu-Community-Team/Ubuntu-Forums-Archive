### Post by Pasdar on 2009-04-10
Nevermind, after running it from terminal I saw what the problem was. When you install the game it doesn't install the music and the compaigns together with it. You have to install those separately. If anyone gets the same problem, what you need to install are "wesnoth", "wesnoth-all" and "wesnoth-music".
