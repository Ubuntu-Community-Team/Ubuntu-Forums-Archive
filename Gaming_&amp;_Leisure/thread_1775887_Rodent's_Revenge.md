### Post by nechus on 2011-06-05
Do you remember this old game that was part of the Windows game pack? I first played it when my dad got me a Packard Bell computer, which was loaded with Windows 3.1.

I came across with this game by chance yesterday and was beside myself when I discovered that it runs perfectly under Wine!! Windows users can't play it anymore on 64-bit systems.

You can download it from [http://www.pcgaming.ws/viewgame.php?game=rodents_revenge](http://www.pcgaming.ws/viewgame.php?game=rodents_revenge)

Extract the zip file, right click on the exe file, check the "Allow to run this file as a program" box (I hope those are the exact words in English) and and then open it with Wine. Listo!

It's a very simple game. You are a mouse and you have to trap the cat in a single square by pushing the blocks with the arrow keys. It will turn into a piece of cheese you can eat.

When you run the game, a blank window also opens. You can minimize it.

Have fun!
