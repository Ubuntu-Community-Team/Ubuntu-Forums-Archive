### Post by Perfect Storm on 2006-09-30
It's properly because you hit run ut2004 button in the installation window (classic error to make). So you are experience permission problem which is why you can only start the game with with root command (sudo). That is a no-no to do so.

uninstall ut2004 and remove .ut2004 from your home folder. Install ut2004 and exit the installer. Everything should be fine then. (and it also solve your shortcut/launcher problem).
