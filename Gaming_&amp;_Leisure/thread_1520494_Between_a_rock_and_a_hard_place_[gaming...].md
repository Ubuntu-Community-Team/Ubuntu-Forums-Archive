### Post by fetuspuppet on 2010-06-29
Youtube has some awesome tutorials on there for installing WINE.

More importantly though what types of games do you like? FPS? Strategy? Racing? Lots of stuff will run in Ubuntu, but one of my friends is always saying Linux=NOTFUN

You can make it fun though, but you get to learn a ton of stuff in the process.

If you like the old school games from the 90's you could try something like [SCUMMVM]("http://www.scummvm.org/") to run lots of the old scumm based games. (Sam and Max, Full Throttle, Maniac Mansion, Monkey Island.)

If you like racing games I think there are a few really good ones out there.

You can use WINE to install many Windows games, but there are a few that don't work, but a lot actually run faster in Linux!

If you're adventurous check out this site as well - [http://www.torquepowered.com/](http://www.torquepowered.com/) It's a developer site, but it has a good list of games that will or should run without having to install Wine or setup and emulators at all on Linux.
