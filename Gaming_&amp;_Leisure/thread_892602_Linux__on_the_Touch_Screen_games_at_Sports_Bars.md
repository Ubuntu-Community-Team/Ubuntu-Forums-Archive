I know which [touch screen]("http://www.monkeysarcades.com/Touchscreens_s/53.htm") games you are talking about. I had to update the hard drive with newer software and from what I remember I believe Red Hat is installed on them. You generally find the Merit Megatouch [Gametime]("http://www.monkeysarcades.com/Merit_Megatouch_Countertop_Arcade_Game_s/126.htm") in most bars, but you can also purchase them for home use as well. In regards to your question, it is not possible to get the games that you see in the [Gametime’s]("http://www.monkeysarcades.com/Merit_Megatouch_Gametime_Countertop_Touchscreen_p/mrtgt08.htm") onto your PC because the hard drive in all [Merit Megatouch]("http://www.monkeysarcades.com/SearchResults.asp?Search=merit&Search.x=0&Search.y=0") machines are locked with a security key that plugs into the board and will only run if the key matches with the software installed on the [Gametime]("http://www.monkeysarcades.com/Merit_Megatouch_Countertop_Arcade_Game_s/126.htm"). From my understanding the only way you can attain the software for updates is if you have an operator license and call Merit and they will send it to you in the mail. But those games are programmed to run on touch screen functionality so installing them to your computer would not make much sense unless you’re using a touch screen monitor.
