### Post by xer0kill on 2008-05-13
Indeed, compiz + games = not that great

The game I play often, [Urban Terror]("http://www.urbanterror.net") runs very poorly, even in fullscreen, if Compiz is running. My FPS meter may say that I'm getting a solid 60fps, but it is clearly running rather clunkily.

The desktop cube is cool, the other effects are cool, but in the end they just make things not perform optimally. 3d modelling applications take a big performance hit as well. If I have compiz running while using gtkradiant (q3 map editor), after about 10 minutes the program gets so slowed down it's basically unusable. If I run Metacity, everything runs great. It's a pain in the *** to keep switching back and forth, IMO.
