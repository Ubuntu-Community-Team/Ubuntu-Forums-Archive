### Post by hardran3 on 2008-08-12
PSP:

-Requires no extra hardware to "unlock" its true potential. You just have to peruse some readme files and apply yourself :)

-Plays AVC movies up to 720x480 (actually higher than its 480x272 screen, good for the TV out on the slim). Also makes a half decent mp3/aac player OOTB. Install LightMP3 and it can also play FLAC and OGG.

-Once you are on a Custom Firmware you can play pretty much any game from the Atari 2600 to the PS1 on it. (Gunstar Heroes for the Genesis. Get it. Play it. Love it.)

-Games have pretty amazing graphics for a handheld.

-The PSP homebrew scene is quite active. Check out QJ.net and maxconsole.

-Web browser sucks.

-Supports WPA wifi

DS:

-You have to buy a flashcart to "unlock" the NDS to do more. Add another $50 to your price.

-No media functions to speak of :(

-Emulators on DS? Not so much. :(

-Has more quirky games. Brain training and CrossworDS are my faves. FFIV remake. Need I say more?

-Not much for homebrew :(

-Web browser sucks.

-Onls supports WEP wifi :(

I personally have a Phat PSP on the latest Custom firmware, and a DS with an Acekard2. I would honestly recommend the PSP unless there are games on the DS you want, only because the PSP has so many more functions. Sony made a great product, and with a CFW, it is amazing.

### Post by anjilslaire on 2008-08-22
To convert movies, install ffmpeg and do the following:

```

ffmpeg -i INPUT -f psp -r 23.5 -s 368x208 -b 768k -ar 24000 -ab 128k OUTPUT.MP4

```

There are GUI methods, but I've found the above is easiest for me. Just replace INPUT for your source file name, and OUTPUT for your desired final file name.

I just keep some common scripts I use in a personal "howto" file and paste into a konsole when needed.
