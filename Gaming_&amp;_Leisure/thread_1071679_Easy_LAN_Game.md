### Post by ahood on 2009-02-17
I have to say...Battle for Wesnoth is easy to setup for LAN play.

Basically, it involves...

Step 1. Install Battle for Wesnoth client and campaigns/maps from Ubuntu repository.

Step 2. Install Battle for Wesnoth server on the host machine from Ubuntu repository.

Step 3. Start Battle for Wesnoth server on the host machine.

Step 4. Start Battle for Wesnoth client on the host machine and select Multiplayer.

Step 5. Start Battle for Wesnoth clients on other machines and join the multiplayer game!

See [http://ryanorser.com/?p=361](http://ryanorser.com/?p=361) for more details.

Battle for Wesnoth is not a flashy game when it comes to 3D, but the gameplay (fantasy turn based strategy) is hugely fun! My son (aged 9) is now addicted to Battle for Wesnoth, and this is a guy who has access (and loads of games) to a Wii and a NintendoDS.

Because the system requirements are low, it has high probability of working on just about any machine. We play it on an old Compaq Presario 700 laptop with a 1.5 Ghz AMD Athlon (original), 512 Mb RAM, and an S3 video.

The multiplayer allows each player to either fight for themselves or can play in teams. There is even a map where players don't fight amongst themselves, but rather just survive waves of attack of increasing strength adversaries. 

Alternatively, I recently played a multiplayer game of Warsow on my desktop system (AMD AthlonXP 1.8 Ghz, 1 Gb RAM, GeForce 7800) it was a lot of fun. The graphics are cartoony, but I liked it, and the gameplay was good.

### Post by donkyhotay on 2009-02-17
Depends on what kind of games you want:
Nexuiz = FPS deathmatch
Battle for Wesnoth = turn based strategy
Globulation 2 = real time strategy
Wormux = worms-style artiller action game
Scorched3D = 3D artillery game
BZflag = tank oriented FPS
Those are some of the best games for a LAN party in the repos. A few other LAN games I like but aren't in the repos is:
World of Padman = FPS with excellent team-based modes
Enemy Territory: Quakewars = class-based objective oriented FPS (commericial game with free demo)
Wolfenstein: Enemy Territory = class-based objective oriented FPS (commercial freeware)
