### Post by kettuz on 2010-10-13
It seems this guy is working on a native rewrite of the CRPG Arcanum game engine: [http://arcanum.game-alive.com/category/openarcanum/]("http://arcanum.game-alive.com/category/openarcanum/") It is heavily work-in-progress, but seems to be alive despite the lack of very recent updates.

On these boards the maker is asking which OS people are running. At the moment there are very few people speaking up for linux crowd. I think it was best if more people were to show him that there are more of us -- we wouldn't want this to become Windows only, WINE-run game, wouldn't we?

So, if you're interested in this effort, please go and add your OS information here: [http://terra-arcanum.com/phpBB/viewtopic.php?p=230124#230124]("http://terra-arcanum.com/phpBB/viewtopic.php?p=230124#230124")!
