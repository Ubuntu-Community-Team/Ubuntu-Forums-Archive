ok, i've had a lot of trouble getting RTCW to work natively under linux

i could load the game, look at the main menu, change all the options and settings

maybe if i patched the wine install to 1.41 it would be playable and stop crashing!!!
but to cut a long story short, i got the patch installed, the windows version keep crashing at the same spot, but i got my native install working!!!!!! heres how its done


U MUST HAVE WINE INSTALLED!!!!!
Installing RTCW(original 1.0)

1: Download these files
windows patches(u must get these 2 files)
[U][ftp://ftp.idsoftware.com/idstuff/wolf/win32/Wolf_Update_133.exe](ftp://ftp.idsoftware.com/idstuff/wolf/win32/Wolf_Update_133.exe)
[ftp://ftp.idsoftware.com/idstuff/wolf/win32/Wolf_Update_1_41.exe](ftp://ftp.idsoftware.com/idstuff/wolf/win32/Wolf_Update_1_41.exe)

linux install + patch
[ftp://ftp.idsoftware.com/idstuff/wolf/linux/wolf-linux-1.41-3.x86.run](ftp://ftp.idsoftware.com/idstuff/wolf/linux/wolf-linux-1.41-3.x86.run)
[ftp://ftp.idsoftware.com/idstuff/wolf/linux/wolf-linux-1.41b.x86.run](ftp://ftp.idsoftware.com/idstuff/wolf/linux/wolf-linux-1.41b.x86.run)

2: Install RTCW with wine

3: Once installed with wine run the 1.33 windows patch, this must be done because the 1.41 patch wont install without it, once thats done, patch to 1.41 with your windows install

4: install the RTCW linux client and patch. then copy the 3 pk3 files needed
pak0.pk3
mp_pak0.pk3
sp_pak1.pk3

then type wolfsp/wolfmp in your console and your up and running!!!!

this way the only way i could get the game to play.
i have not tested the GOTY edition, only the original
now im happly playing on my netbook, high detail, running around 38-50 fps lowest fps ive seen is 25

hope this helps someone!
