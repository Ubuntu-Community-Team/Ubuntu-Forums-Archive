### Post by hikaricore on 2006-08-20
> **Icon41 said:**
> Awesome thanks, it reminded me of zelda and graal and I wanted to check it out but i didnt know how to install

Well if you're runing debian or any distro of ubuntu you can simply download the file and open it from your file manager.  If you do not have a file manager installed you can use: "dpkg -i stendhal_0.52-1_i386.deb" without the quotes from a term window in the directory you downloaded it to.  Depending on your system setup and user permissions you may need to be root to do this so do "sudo dpkg -i stendhal_0.52-1_i386.deb".

If you're not using a distro that supports debian packages you will probably have to download and compile the source or check around for a prepackaged build which your system can install.

Installed fine on my system so I'm not sure what java libs need installed exactly.
