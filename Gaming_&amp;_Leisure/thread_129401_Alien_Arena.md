### Post by Mikecore on 2006-02-13
To install:

1)  Download alienarena-2006ge-x86.run  ( mirror on thier site )

2) run the install script

```


cd /into/the/dir/you/saved/the/game/in
sh alienarena-2006ge-x86.run


```

then I had a problem with no sound. I found a post on thier fourms that said to 

```


cd /usr/local/games/alienarena2006

mkdir old

mv crx old/

mv crx.sdl crx


```


I then started the game and everthing was fine. The game is nice like a wierd UT
thing going on, But I wont complain it was ported to linux :)
