### Post by Ferrat on 2009-07-20
yes and no, for some people there can be a gain on Linux, for most there is a small loss in performance, gains comes from a few places like if WINE (the API layer/interpretor that lets you play WOW on linux) ignores some "useless code" you get a "false boost" and other stuff but there are places where the WINE code is just more effective than the Windows code and there for runs faster. 
Try it out, might like Linux more like many do but don't expect a boost by switching, expect a loss but WoW is more than playable on Linux as long as you have a ATi or a nVIDIA graphic card, Intels have some problems most of them at least. 
With tweaking you should be able to get at least 75-80% of your FPS in Windows when running under WINE on Linux, when I played I got about 90-110% FPS depending on where in the world I was, so some places I gained, others I lost a little but I've raided and even recorded while raiding without any problems on Linux =)
