### Post by evets25 on 2008-07-17
from commandline, first run "winecfg" so that wine sets up it's directories and whatnot, then to run a program you type "wine nameofprogram.exe". So for instance, for me to run diablo, I run "wine ~/storage/games/Diablo II/Diablo II.exe". Hope that helps.
