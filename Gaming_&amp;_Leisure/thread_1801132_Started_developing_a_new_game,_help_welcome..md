### Post by didu on 2011-07-10
Hi guys,

I'm a fan of a board game called "Ricochet Robot" ([http://en.wikipedia.org/wiki/Ricochet_Robot](http://en.wikipedia.org/wiki/Ricochet_Robot)). This game has very simple rules and can be played by many people together.

I thought it'd be very easy to represent this game as a simple search problem. So, I went ahead and wrote a python version of the game. The code is hosted on sourceforge at:

[https://sourceforge.net/projects/botsolver/](https://sourceforge.net/projects/botsolver/)

The code so far is unpolished, but has the main functionality of the game. I thought that someone else might also be interested in the game and its development.

So, if you are interested in this game or just want to have some fun developing a free game. Please let me know and I will make you a developer for this game.

Have fun!

A good way to find developers for a SF.net project is to post in the Help Wanted section of SF.  You can access it by going to your project, then clicking Admin>Features>Help Wanted.  

Hope that helps.  :-)
