### Post by Monster_user on 2006-04-03
Wine is compatible with most programs, that are 2D, or use OpenGL.

However, it is still Alpha build (very incomplete, and can be difficult to setup.), and has bugs.

Windows Solitaire, Notepad, Microsoft Office, Adobe Photoshop, Internet Explorer, Quicktime, StarCraft, and Quake/Doom engine based games (Half-Life1, SW:JK2 JediOutcast, ST:V EliteForce) seem to be the most compatible. (Disney paid big money for Photoshop compatibility).

**[Cadega](www.transgaming.com)** has better game compatibility, than just plain old Wine. Cadega uses Wine with proprietary technology, such as SecuRom. It also has significantly better DirectX compatibility. 

You may also be interested in **'[ReactOS](www.reactos.com)'**. Which is an Operating System, that is practically based on Wine, and an OpenSource WinNT kernel.

It includes a few usefull apps, you can install the ReactOS explorer, onto Ubuntu. Then use it in place of the Windows Explorer (usefull for programs that need it).
