### Post by Sofox on 2012-07-13
For those who missed it the first time, the goal is to support Linux and potential Linux game pitches on the  Kickstarter and other Crowdfunded games by bringing them more exposure and attention. This is so that proposed Linux games reach their target get made, and pitches see the importance and support for Linux games and consider add Linux support to their pitch.

Previously, we maintained a big thread where we kept editing the first post with new additions. Since the edit lock has scuppered this, then until Inquisitor and I work something out, the list (which has already gained some attention on Reddit) will be maintained  here: [https://help.ubuntu.com/community/KickstarterGames](https://help.ubuntu.com/community/KickstarterGames)

So for this thread, we need everyone posting any Kickstarter or IndieGoGo or other Crowdfunded gaming projects that support Linux. Any new projects posted on this thread will be easy for forum members to see, and will then be added to the Wiki list and tweeted on LinuxGamers ( [https://twitter.com/linuxgamers](https://twitter.com/linuxgamers) ), all to achieve for maximum exposure.

Remember, if you're not sure if a project supports Linux, immediately ask the developer. Even if the answer is negative, the developer now knows another person considers Linux support important (many Crowdfunded projects have supported Linux due to constant requests). Also, sometimes game developers intend to support Linux but strangely don't announce it on the front page.

As always, feel free to use my handy tool here to make quickly formatted Kickstarter or IndieGoGo entries, both for the forum post and for the Wiki: [http://sofoxcentral.com/projects/LinuxKickstarterScraper/](http://sofoxcentral.com/projects/LinuxKickstarterScraper/)
