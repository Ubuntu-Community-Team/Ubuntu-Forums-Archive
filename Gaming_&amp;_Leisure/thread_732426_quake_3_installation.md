### Post by disturbedite on 2008-04-02
i actually recommend that people not use quake3 for linux.
after id released the q3 source code, development continued under the project name ioquake3.
ioquake3 is simply q3 with continued development just like id would do if they were still developing it.  it has so many more bug fixes & what not.  its great.  it does require compiling, i do it from svn, but its easy.

btw, ioquake3 does require the original q3 pak0.pk3 file though.
