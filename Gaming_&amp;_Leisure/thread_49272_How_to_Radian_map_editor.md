This is my first tutorial I hope you find it helpfull !

So you want to make maps for one of your favorite games do you, in that case you will need a map editor. The one I use is called Radiant,
Go here [Radian](http://www.qeradiant.com/?data=dlselect) and select your OS and what games you want to make maps for, once you have done that click on "Check for files".
You will see your search results and the appropriate file to download, Click on download. You will end up with a file called linux-radiant-1.4.0.run.
Next from the console change to the directory that you downloaded the file, ```
cd Desktop
``` 
then in the console type```
sudo sh ./linux-radiant-1.4.0.run.
``` that will start the install. Agree to the license agreement. 
The install path has to be the same as your installed games in my situation > /usr/local/games/
If your games are installed in your home dir then I don't think you need to run the install as sudo but I'm not sure. 
Check the options you want then begin install. 
If everything went ok you can start Radiant directly after the install.
Ok install complete next you need to set the editor up .
