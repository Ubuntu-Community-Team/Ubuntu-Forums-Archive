Hi, I'm not sure if this is the correct forum to post this in, but here goes....

I have a jukebox which I made out of an xbox running xbox media center, a psone lcd screen and an xbox IR remote. I mounted the whole caboodle into a box and hacked a controller to make external button controls.

This works great, but the screen is a bit naff and the controls are less than intuitive, so I would like to make a new jukebox.

I want to use Linux as the OS, but although I have been using Ubuntu as my sole OS for around a year, I still consider myself a noob.

Here are the specs I want for my jukebox, maybe I could get a little help?

1. Firstly, I want the jukebox to be completely standalone. I have an old laptop that I will be using for this project and it will be mounted in its own box and used soley as a jukebox.

2. When I say standalone, I don't want any streaming nonsense. I want to be able to stash my laptop safely away from beer spillages when I have parties, and not have to have it running to stream music to my jukebox. It must be completely self contained. I currently use ftp to transfer music to my xbox based jukebox.

3. The user interface must be totally idiot proof. I want a 'pub style' jukebox that only allows basic search functions and a queue button. I want the thing to boot straight into jukebox mode, the OS must be hidden from the user.

4. Yes the queue button! My main problem with my current jukebox is that it is too easy to accidentally cancel the currently playing track and play the selected track instead rather than add it to a queue. I want this to be impossible to do unless you use the....

5. ...remote control! I want to be able to use an IR remote too. Must be IR not RF because of my learning remote.

6. Touch screen interface.

Basically, I want a device that will sit in my living room and behave exactly like an old fashioned jukebox but capable of holding a lot more music. People should be able to use it intuitively and not be able to mess the settings up.

My current xbox based gadget has most of these features, but it has a tiny screen, uses arcade buttons as the controls, and I can't disable the 'instant play' feature, which means drunk people never remember to queue songs properly. It's also the size of a house...

I have found some software that will do everything I ask, but it is for windows and I don't want the long boot times and instability. It's supposed to be fun!

If all else fails, I'll use that, but I would vastly prefer a linux based device.
