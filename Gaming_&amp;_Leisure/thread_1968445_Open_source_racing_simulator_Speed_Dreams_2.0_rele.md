### Post by ocirne94 on 2012-04-29
Hello!
After a pretty long development cycle the Speed Dreams team proudly announce the release of the open source racing simulator Speed Dreams 2.0. The base package and several add-on packages are available for free download.
Visit [www.speed-dreams.org/release-20-en.html]("http://www.speed-dreams.org/release-20-en.html") for more info and a detailed list of changes (quite impressive, believe me)!

You can get Speed Dreams here:

* **Source code**: [http://sourceforge.net/projects/speed-dreams/files/2.0.0/](http://sourceforge.net/projects/speed-dreams/files/2.0.0/)
* **Free install via Desura** game service (**binaries**): [http://www.desura.com/games/speed-dreams/](http://www.desura.com/games/speed-dreams/)
*** PPA**: [https://launchpad.net/~speed-dreams/+archive/ppa]("https://launchpad.net/%7Espeed-dreams/+archive/ppa")

See you soon on [www.speed-dreams.org]("http://www.speed-dreams.org") !

Ocirne
of
the Speed Dreams team
