### Post by acoustibop on 2007-09-20
Why not try [pSX]("http://psxemulator.gazaxian.com/") instead?  For my money it's a much better Playstation emulator, and the only configuration necessary to get good sound is increasing the values on the latency sliders.

It's easy to install and get running, as well - the only extra dependency in a standard Ubuntu install is libgtkglext1 - in the Universe repository.  Tar it into a folder, add a BIOS. a game or two, configure your controller and you're good to go.
