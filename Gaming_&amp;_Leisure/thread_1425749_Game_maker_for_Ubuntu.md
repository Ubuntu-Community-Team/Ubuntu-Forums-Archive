### Post by Simian Man on 2010-03-09
CrystalSpace is a game engine, not a game maker.  A game engine is a library and set of tools that allow you to write your own game in a programming language.  A game maker is an application that allows you to make a certain type of game in a point & click fashion.  Linux has tons of game engines, but I don't know of any game makers.
