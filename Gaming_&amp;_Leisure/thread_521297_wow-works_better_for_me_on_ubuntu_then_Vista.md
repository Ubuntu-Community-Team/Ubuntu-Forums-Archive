### Post by mikedep333 on 2007-08-09
It sounds like you have a memory issue if the OS affects the game and if having other applications affect the game. In that case, you are referring to swapping/paging. Lag refers to the network being slow.

Anyway, this does not surprise me the least. Ubuntu uses up 200 to 250 megs of ram by default. Vista uses up 600 megs of ram by default (although I slimmed it down to 400 on my machine) plus I presume you have some OEM or 3rd party bloatware. That leaves you with only 300 megs of ram or so to play WoW and use your other applications instead of 800 to 750 megs. Quite a difference.

BTW. What type of configuration are you doing? When I think of configuration to reduce memory usage, I think of things like disabling services and uninstalling things like 3rd party apps and TCP/IP v6.
