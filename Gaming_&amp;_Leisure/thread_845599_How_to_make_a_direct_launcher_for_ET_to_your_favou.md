### Post by lillypop on 2008-06-30
Many of you will kow how to do this but i had trouble finding any Info on how to make a Launcher for Wolfenstein ET to my own server with my screen resolution and server password included.
I was fed up with either having to search server lists for my server or type Ip then having to reset my resolution from the standard 800x600 everytime i connected.

So heres how to make your launcher

Right click on desktop select the make launcher option.

Name your application mines called ET [=RW=] 

In the command type this:  home/Username/enemy-territory/et +set r_mode 6 +connect server ip +password 

I use /home because thats the directory my et is alternatively you and use the command
./et +set r_mode 6 +connect server ip +password blah

I use the password because i have private slots on my server If its not required just delete that line.
 you can change the r_mode to which ever resolution setting your preference is r_mode 6 = 1024x768 r_mode 4 = 800x600 etc.

To have the et icon displayed you will find in your enemy territory folder ET.xpm you will need to copy and paste this to your pictures folder.
Then in launcher click the icon and browse to pictures select picture click open at the bottom of teh window and the ET icon will display.
Then add a comment if you wish in the comment box click ok you then have you quick launcher to your favourite server.
