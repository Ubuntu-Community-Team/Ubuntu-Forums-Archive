### Post by Kirk Schnable on 2012-11-21
I personally use OpenVPN, but I wouldn't exactly call it a Hamachi alternative. Hamachi is P2P, while OpenVPN is a client-server setup, and all traffic goes through a central VPN server. 

I have not experimented with this program much, but you might want to have a look at Tinc.  It's P2P like Hamachi. [http://tinc-vpn.org/](http://tinc-vpn.org/)

Kirk

### Post by Horbo on 2013-02-02
Install Hamachi from [http://secure.logmein.com/us/labs](http://secure.logmein.com/us/labs)

Then install Haguichi (a very nice GUI for Hamachi) here: [http://www.haguichi.net/download/](http://www.haguichi.net/download/)

Then just run Haguichi - if you can work Hamachi in Windows you can work Haguichi.
