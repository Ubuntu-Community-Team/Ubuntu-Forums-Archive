### Post by ubudog on 2010-11-05
Warzone 2100 is a great game.  Make sure you get the latest one, it has all the new features and bug fixes.  I think it uses autotools (am I wrong?) to compile, so you can just get the source from their website.  Un-extract it with the archive manager.
You can compile it with the following code.

```
./configure && make && sudo make install
```

You can use the terminal, or you can use ALT>F2 and select the "Run in terminal" option.  Just copy and paste that code. :).
If there are any errors, let me know.
