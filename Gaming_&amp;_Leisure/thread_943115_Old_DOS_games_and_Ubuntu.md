### Post by WWSmith36 on 2008-10-09
Windows95 / DOS era games can actually run in linux with a program called DOSBox.  You can find it in the standard repositories.  You can search for it in add/remove or

```
sudo apt-get install dosbox
```

I am not certain if that particular game will be compatible.  I use to for an oldly but goody, WORMS+
