### Post by DarkAmbient on 2013-08-28
Not the 1.6.2 version, but it does with the latest "official" which is 1.5.2. The 1.6.# got a different game-files-structure you could say, and I think it's only temporary till 1.6.2 becomes the default/official latest one. To much work for only a short period of time to "support" 1.6.2 right now.

You/the player needs an Mojang account with Minecraft bought and connected to it to be able to play Minecraft. You can get one on [http://minecraft.net](http://minecraft.net), It's a one time fee.

You can still use my app to download/install Minecraft, update lwjgl, find links to useful Minecraft-sites or translate ingame-glyphs etc. You need to download 1.6.2 manually from [https://s3.amazonaws.com/Minecraft.Download/launcher/Minecraft.jar](https://s3.amazonaws.com/Minecraft.Download/launcher/Minecraft.jar), but this new "file-structure" means that 1.5.2 will be preserved. So to launch 1.5.2 use my app or the starter found in Unity-dash / menu. To start 1.6.2 launch that file you downloaded from link. You can enter worlds created with 1.5.2 with 1.6.2, not the other way around. (I think)

edit: If you deceide to mod Minecraft using Minecraftier, then it won't affect 1.6.2. You need to manually download the mod and follow the instruction on how to install it.
