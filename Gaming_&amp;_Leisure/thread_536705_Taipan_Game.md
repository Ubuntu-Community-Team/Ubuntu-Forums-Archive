### Post by FranMichaels on 2008-08-19
Is this it?

It can be run under [sdlmess]("http://rbelmont.mameworld.info/?page_id=163") if you have the required apple2 bios and a taipan disk image. You may have to *google* a bit to get sdlmess up and running, it's a little confusing at first...

With that, it can be run as
```
./mess apple2p -s6d1 somedisk.dsk
```
Replacing somedisk.dsk with the real filename.
