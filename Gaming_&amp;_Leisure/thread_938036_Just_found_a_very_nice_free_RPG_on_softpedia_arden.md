### Post by tgfcoder on 2008-11-22
Thanks for all your nice comments. binbash is correct, the game is still under development, and I've just finished high school so I have some free time to work on it. 

Did any of you try playing Nyx's world? Sempridge is my favourite, but Entarya is a little more challenging, so try that out too! 

There'll be lots of reworking before the next version -- it's around v1.3 now, which means a lot has changed since the last source release. (nearly half a year ago) 
More improvements/additions include Internet interaction and a wider variety of content (monsters, weapons, items, spells, summoning--and an interface update (some stuff is/was UGLY.)

Lucky for us Linux users, you get the latest releases seeing as I don't compile Windows binaries often!
