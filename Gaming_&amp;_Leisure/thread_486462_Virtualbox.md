### Post by eXisor on 2007-06-28
To actually answer you're question (unlike the other responders)....

No, virtuabox does not support 3d technologies. Since it provides only a virtual 2d graphics driver all 3d games are currently unfeasible.

VMWare does support limited 3d (directx 8 - I think) so maybe give that a go if you want legacy gaming in a virtual machine.
