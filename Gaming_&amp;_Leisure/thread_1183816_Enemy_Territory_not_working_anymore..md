### Post by Barky on 2009-06-10
It looks like your default resolution is 1680x1050 in-game but ETQW realizes your monitor can't go that high so it auto closes. Check your config file in the ETQW directory to find a default resolution setting so you can change it.

### Post by darthmob on 2009-06-10
the path to your config should be *~/.etqwcl/base/etqwconfig.cfg*.
the resolution is set with the command r_mode (1280x1024 is r_mode 9; lower values stand for lower resolutions).

PS: it's enemy territory: quake wars you are trying to start. [enemy territory]("http://www.splashdamage.com/content/wolfenstein-enemy-territory-barracks") is developed by splashdamage as well and free to play and available for linux.
