### Post by RomeReactor on 2007-01-26
Some games accept ALT+ENTER to change between fullscreen and windowed. It would probably be best to have your games run at lower resolution than your desktop, so as to not bleed offscreen. Not sure if this works for you, though. Also check out each game's README; some games require peculiar commands to run in a window...
