### Post by cogadh on 2008-06-30
All you should need to do is set the installer as executable: 
```
chmod +x RegnumOnlineInstall_32
``` or if you have the 64-bit version:
```
chmod +x RegnumOnlineInstall_64
``` then run the installer like this:
```
./RegnumOnlineInstall_32
``` or if you have the 64-bit version:
```
./RegnumOnlineInstall_64
```
