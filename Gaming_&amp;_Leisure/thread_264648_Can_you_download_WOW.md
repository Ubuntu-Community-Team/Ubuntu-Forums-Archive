### Post by skymt on 2006-09-25
> **WalmartSniperLX said:**
> Microsoft OWNS the copyrights, making it legal.

Microsoft isn't emulating anything. You are, on your XBox.

Another example: Virtual PC. It's emulation software, made originally by Connectix, later bought by Microsoft. It allows PowerPC Mac users to run x86 Windows programs. Through emulation, obviously. Connectix doesn't own the copyrights to every Windows program, but nobody sued, and Microsoft liked the program enough to buy it and continue development.

And another: Apple's Rosetta technology, bought from some company I can't remember the name of. It's emulation software used to run PowerPC Mac programs on Intel Macs. See above reasoning.

EDIT: Basically, if you own the software, don't work around copy protection, don't download it from an unauthorized source, and aren't breaking the EULA, emulation is legal.

The fourth qualification (EULA) is interesting. For example, Nintendo explicitly disallows copying of their games for any purpose without written. Therefore, all emulation of commercial Nintendo games is illegal.
