### Post by Vadi on 2009-07-24
No, you do **not** install it as root. 

Just right-click on the file, properties, permissions, set it as executable. Then double-click on it to start.

If you install as root or into /usr, you'll into various sorts of problem that the installer was not designed for. Such as the above need to run the installer as sudo, which is a horrible security idea.
