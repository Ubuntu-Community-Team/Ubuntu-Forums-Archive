### Post by michaelzap on 2009-07-06
On the [Urban Terror download page]("http://www.urbanterror.net/page.php?6") there's a zip file for Macs and Linux. You apparently downloaded the Windows executable. You need to get the Linux program if you want to play this in Linux.

The instructions are pretty simple, and also found on the download page:
> Just unzip the zip and make a shortcut to the ioUrbanTerror executable.
Mac executable: ioUrbanTerror.app
Linux 32bits executable: ioUrbanTerror.i386
Linux 64bits executable: ioUrbanTerror.x86_64
In linux, you need to make it executable first: right click it, properties and tick the 'allow execution' box.

You can run the program right from your home directory after unzipping it.
