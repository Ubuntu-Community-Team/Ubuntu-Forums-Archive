### Post by oldrocker99 on 2019-06-05
Steam games are located in the /home folder, in the hidden file .steam.

Never, ever install anything except programs in /. It's just for system files and programs. Steam itself is installed in the root folder, but everything else goes in the /home folder, which is the **ONLY** directory that the user has complete read/write access. Your /home folder is for pictures, music, downloads, etc. It's always larger than the root folder.
