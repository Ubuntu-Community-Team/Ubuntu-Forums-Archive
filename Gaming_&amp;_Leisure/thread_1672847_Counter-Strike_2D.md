### Post by MadCatmkII on 2011-01-21
Quite simple, you just need to give it two extra parameters:

```
-full -24bit
```

The 24-bit is important because the game tries to use 32-bit colour depth by default even though (as far as I know) X doesn't support that. Have fun!
