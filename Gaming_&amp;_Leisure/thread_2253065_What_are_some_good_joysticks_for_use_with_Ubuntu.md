### Post by dannyboy79 on 2014-11-17
check out penguin recordings on youtube, he recently got a joystick and stated it was working in ubuntu 14.04. logitech extreme 3d pro i think it was.  personally i use a wired xbox 360 controller. it's very well supported in ubuntu either using the default xpad driver (works in games like trine, trine 2, borderlands 2, borderlands tps)

i've been told by a friend he needs to use the xboxdrv userspace driver for his wireless xbox 360 controller (hence the reason I prefer wired, less things to go wrong since it's true input over the usb bus whereas the wireless receiver has to receive a signal and then process that signal, then the wireless receiver sends that signal thru the usb bus-WAY MORE COMPLICATED IMO)
