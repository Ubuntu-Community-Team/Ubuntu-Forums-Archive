### Post by ojdon on 2011-03-15
Unfortunately, there is hardly (if any) big budget games with a massive committed development team. They are mostly hobbyists working on Indie games. Though there are quite a few modern Indie games that might not have the beautiful graphics but some awesome gameplay, Minecraft, World of Goo and Braid are just a handful of games I've enjoyed on Linux. :)

Funnily enough if there are games that have ports for both Windows and Mac then you're more than likely to be able to play the Windows version through WINE like a few other members have suggested previously. Call of Duty 4 works fine for example. Check out the AppDB[http://www.winehq.org]("http://www.winehq.org")to see which Windows applications run through WINE.
