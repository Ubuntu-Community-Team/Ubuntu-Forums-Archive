### Post by Sutekh on 2006-08-07
CVS Cedega is what used to be known as WineX, does not have code such as copy protection and texture compression.

The Cedega Timedemo is a fully-featured version of Cedega, it's all there.  The catch is it only works for 14 days, and you get the great big watermark over the top of the game.

I have read that the current version of Wine is far ahead of the CVS Cedega.

If you are interested in trying Cedega, use the Timedemo.  If you don't like Cedega or don't want to subcribe, try using Wine.  It's far easier to install, and will probably work better than the Cedega CVS.
