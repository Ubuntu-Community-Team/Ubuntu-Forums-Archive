### Post by christhemonkey on 2006-10-01
Install wine and try with that.

Allthough in no way is it guarenteed to work.


```
sudo apt-get install wine 
```

Then run;
```
wine /path/to/championship/manager/executable/setup.exe
```

Replacing with actual path to executable.



Please join the [www.winehq.com](www.winehq.com) app database and report if you can get it to run/install ok.
