### Post by Flaringo on 2006-10-07
Hello. Since I am very bored today, I'm going to make the ultimate games list for Linux! It's going to include all the "big" games that works with Linux -- free or not, native clients or emulated! 

I am currently working on it, but I need to test how it'll look first, and what changes I'll have to make. It's really simple (BBcode xD).

So here it is:




                                                   [size=14]**The games list for Linux**[/size]





[size=8]MMORPGs[/size]

**World of Warcraft**
[www.worldofwarcraft.com](www.worldofwarcraft.com)
[img]http://www.armchairempire.com/images/Reviews/pc/world-warcraft/world-warcraft-9.jpg[/img]
Free: No, about 20-30$ and 15$ a month. (has a free trial)
Compatibility: Works well with Wine, Cedega, and works very well with Crossover(needs confirmation). Got it running almost perfect with wine myself, and I have not tested it with Crossover because I quit playing it.

**Runescape** (this game is browser based, and requires a java-plugin)
[www.runescape.com](www.runescape.com)
[img]http://koti.mbnet.fi/jonneweb/img/pelit/kuvat_/runescape.jpg[/img]
Free: Yes, but you need to become a premium member to be able to reach certain high-level areas.
Compatibility: I have not tested it myself yet, but it should work fine as you as you've got a java-plugin for you browser.

**Planeshift** (under development)
[http://www.planeshift.it/](http://www.planeshift.it/)
[img]http://images.apple.com/downloads/macosx/games/role_strategy/images/planeshift_200501071830.jpg[/img]
Free: Yes
Compatibility: I've had some problems myself when trying to install and update it, and when I got it running, it ran really slow. Quite some time since last time I tried it, though.

**City of Heroes**
[http://www.cityofheroes.com/](http://www.cityofheroes.com/)
[img]http://www.armchairempire.com/images/Reviews/pc/city-heroes/city-heroes-014.jpg[/img]
Free: No, about 20-30$ and 15$ a month. (has a free trial)
Compatibility: Works with Wine(not sure how well, though), Cedega (got it running very well) and I think it runs on Crossover.

**A Tale in the Desert**
[http://www.atitd.com/](http://www.atitd.com/)
[img]http://images.apple.com/downloads/macosx/games/simulation_and_sports/images/ataleinthedesert_200409171147.jpg[/img]
Free: Well, all I know is that there's a monthly fee, but I did not figure out how much by looking at their webpage ... (need information on this one) (Has a free trial)
Compatibility: Native linux client.

**Guild Wars**
[www.guildwars.com](www.guildwars.com)
[img]http://soeren-weber.net/data/2005/01/06/guildwars-action.jpg[/img]
Free: About 20-30$, but no monthly fee.
Compatibility: Works very well with Cedega, not sure about Wine or Crossover.

**Anarchy Online**
[www.anarchyonline.com](www.anarchyonline.com)
[img]http://vaultmedia.ign.com/rpgvault/image/article/661/661294/anarchyonline05102601a_1130295031.jpg[/img]
Free: The base version (the game without ANY expansions) is free until 2007. I'm not sure about the expansions, or if there comes any monthly fee with them.
Compatibility: Should work well with Cedega, not sure about Wine or Crossover

[size=8]FPS[/size]

**Cube**
[www.cubeengine.com](www.cubeengine.com)
[img]http://www.cubeengine.com/images/screenshots2/screenshot_754937.jpg[/img]
Free: Yes.
Compatibility: Native Linux client, so it should work perfectly fine.


[size=0]**(C)Flaringo 2006**[/size]
