### Post by sethmahoney on 2006-07-12
Some games work under linux with wine, others work with cedega.  But some games don't work at all.  You can find out which games work with cedega, and how well, here: [http://cedegawiki.sweetleafstudios.com/wiki/Category:Games](http://cedegawiki.sweetleafstudios.com/wiki/Category:Games)

Cedega isn't free though, but you can download and use the CVS version for free.  I think there is a script for doing it automatically somewhere on this forum.

### Post by darrenm on 2006-07-12
Hello, HL2 yes with Wine, not sure about the others.

Linux is fab in many areas but if you are a huge gamer you sometimes need the commercial properties of Windows.

But hey, persevere, spread the Ubuntu word and soon developers will start writing games properly again and they will be easy to port from one platform to another.

If you are new to Linux then you have made the right choice with Ubuntu. Get ATi working with Xgl/Compiz and wow yourself and your friends.

If you follow the ATi binary driver wiki on here ([https://wiki.ubuntu.com/BinaryDriverHowto/ATI](https://wiki.ubuntu.com/BinaryDriverHowto/ATI)) then you can't go wrong with the ATi driver, I prefer the officially supplied ATi drivers from support.ati.com as I'm not too much of puritan.
