### Post by pytheas22 on 2008-07-14
When I downloaded the installer, it wasn't executable, so clicking on it would not do anything.  This may have been where you went wrong, or maybe not.  In any case, I'm going to recommend that you follow these steps to make sure it's install properly:

1. download the installer (or find where it is right now on your computer if it's still there) and move it to your desktop.

2. run these commands:
```

cd ~/Desktop
mv armyops* ~/armyops
cd ..
chmod +x armyops
./armyops
```

3. now an installer will open.  Choose the default options.

4. after it's installed, you should be able to start the game with this command:
```

cd /home/joseph/armyops
./armyops
```
