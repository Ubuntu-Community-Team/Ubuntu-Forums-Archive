### Post by MaximB on 2007-11-07
Peragro Tempus is a new mmorpg game under heavy development that promises a lot and has a native Linux client.
It now has a playable testing demo that you can test from the game server or from your computer (via local server).
The graphic and content are constantly improving and every few days they release a new test build ! 

To run the game with localhost (if the server is unavailable)   simply Run script server.sh , then client.sh and enter localhost as address.

Their forums are not very active BUT their IRC channel is.
And if you can help developing their game please apply at the IRC channel, they can really use your help in every aspect of the game

Game website : [http://www.peragro.org](http://www.peragro.org)
Game Forums : [http://nntp.cyanox.nl/index.php?c=2](http://nntp.cyanox.nl/index.php?c=2)
Very active IRC channel : #peragro on irc.freenode.net
Screenshots : [http://www.peragro.org/index.php?q=gallery&g2_itemId=31](http://www.peragro.org/index.php?q=gallery&g2_itemId=31)
Download Linux test build : [http://linuxbuild.peragro.org/](http://linuxbuild.peragro.org/)
Wiki : [http://wiki.peragro.org/index.php/Main_Page](http://wiki.peragro.org/index.php/Main_Page)
Installation guide (works with Gibbon) : [http://wiki.peragro.org/index.php/Ubuntu_gutsy](http://wiki.peragro.org/index.php/Ubuntu_gutsy)
