### Post by Nevon on 2008-12-11
There is a Linux executable that comes with the download if you download it from the website, and not from a torrent. But even if you downloaded it from a torrent, you should be able to make your own executable fairly easily. Just open start_menu.blend in blender and then go to file -> save runtime. Then just save your file in the game root directory (I don't know if it matters, but that's what I did anyway). Mark it as executable and then run it by navigating to your game root and type:
```
./<executable name>
```
