> **Sarteck said:**
> The above screenshot was from running Diablo II:LoD under WINE 0.9.31.  I just downgraded to 0.9.26, and I'm having no shadow glitch.

Sometimes Wine will have what is called a regression.  This is where in attempting to fix the handling of new software or badly supported software, effects a currently working piece of software in a small/major way.  As with any work in progress such as Wine this can happen from time to time.  So test out newer versions of wine if you must, but if something like this seems wrong again, just downgrade back to the working version and you should have no trouble.  :)  Hope this helps.
