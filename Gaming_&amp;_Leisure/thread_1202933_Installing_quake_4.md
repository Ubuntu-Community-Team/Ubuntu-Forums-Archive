### Post by SerenityKill3r on 2009-07-02
You just need to install the game using quake4-linux-1.4.2.x86.run.

then you need to copy the required files off the 4 disks into the installation directory, usually /usr/local/games/quake4.

When you start the game you will be asked to install the cd-key, which is found on the back of the booklet in the box.
