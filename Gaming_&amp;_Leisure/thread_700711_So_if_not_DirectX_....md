### Post by DoktorSeven on 2008-02-18
No, generally you don't ever install DirectX when a game asks (and pretty much every Windows game will ask you on install, even in Windows, even if you already have the latest DX version!).  Go look at a thread/site regarding how to set up WoW in Wine, you'll generally need to enable OpenGL mode for best performance.
