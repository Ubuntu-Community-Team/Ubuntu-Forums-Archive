### Post by sakuramboo on 2010-06-11
There are a lot of different ways to run .bin files. One way is to right click on the icon, go to Properties -> Permissions and check mark "Allow executing file as program". Close and then double click the file.

Another way is to open a terminal where the file is located and run the following...

chmod u+x <file name>
./<file name>

or

sh <file name>
