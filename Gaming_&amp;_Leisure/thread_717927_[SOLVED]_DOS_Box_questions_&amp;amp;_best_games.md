### Post by DoktorSeven on 2008-03-07
As for your issues about DOSbox capturing the mouse, CTRL + F10 will release the mouse from any DOSBox game that captures it, and CTRL + F9 will kill DOSBox.  The only way to really go back to the prompt from a game is if the game has an "Exit to DOS" or similar function.
