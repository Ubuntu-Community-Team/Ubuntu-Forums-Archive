### Post by TheFu on 2022-06-10
> **ellie0303 said:**
> i have a pc and a ps4 but i use my pc mostly so if i could have some game recommendations as I'm getting bored of the ones I'm playing now thanks.

There are thousands, but chances are you have a specific type of game that you prefer.  Lots of people use emulators to place their old games. I enjoy playing the true Atari 2600 games from my childhood, but I double tank-4 would be as entertaining for people who didn't play it as a child and knows all the software bugs that made it easily win-able by experts.  And Space Invaders for mindless brainwashing or Yars Revenge.  Of course, these are only legal if you have the original cartridge from that era, which I do.

I also enjoy some old DOS games like Starflight, F-15, or the original, non-networked, Warcraft that was shipped on a CDROM with magazines from the time.  Same for early Command & Conquer versions, though I haven't played those in some time.

There are a bunch of classic games like tetris clones, Sokoban, Sudoku, Chess variants, if you like that sort too.  Most of these run great inside a virtual machine.
With 64G RAM and a recent Core i7/i9 computer, you can play many Android games too. Java is a hog, so the minimal requirements for happy play tend to be much higher than most games.

So, what do you like?
