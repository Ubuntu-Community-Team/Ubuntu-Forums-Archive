### Post by ahaslam on 2007-07-04
It does run well, but I'm somewhat disappointed with performance. Some say it's fine & comparable to Windows, I guess this depends upon your hardware & expectations. For me it doesn't max out any resource but only averages 20fps. Some say this is ok for a RPG, but I'd expect a lot more from a 3.15G C2D & 7950512GT. I'd suspect that machines with lower specs would also hit the same level (to a point).

I haven't got a copy of Windows to run a comparison, but if you'd be happy with 20fps give it a shot under Ubuntu, otherwise use your Windows install.

For more info (though I'm sure you've seen it): [http://ubuntuforums.org/showthread.php?t=349764](http://ubuntuforums.org/showthread.php?t=349764)
