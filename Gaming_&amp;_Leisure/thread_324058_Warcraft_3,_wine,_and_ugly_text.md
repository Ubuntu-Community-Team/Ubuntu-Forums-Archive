### Post by Afoot on 2006-12-25
What you need to do is to start the game with the "-opengl" switch, for instance:
```
wine war3.exe -opengl
```

If this doesn't do it, then you're probably not running the latest version of wine. You can get it here:
[http://wine.budgetdedicated.com/archive/index.html](http://wine.budgetdedicated.com/archive/index.html)

Good luck! :)
