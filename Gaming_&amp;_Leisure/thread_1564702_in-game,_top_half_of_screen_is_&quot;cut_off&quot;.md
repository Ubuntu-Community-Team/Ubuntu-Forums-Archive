### Post by gerowen on 2010-08-31
Couple of questions.

- What graphics card are you using?
- Are you using the proprietary drivers (if available) or the open source ones?
- What resolution is your screen set to?
- Does this happen no matter what resolution the game is set to?
- Are these native Linux games or Windows games running through Wine?
- Are you running multiple screens?

It could be any number of things, my first suggestion would be look at the refresh rate and resolution the games are changing your display to and see if you can find any patterns such as certain things that make it worse, etc.

Not very helpful, but I hope it's a start.
