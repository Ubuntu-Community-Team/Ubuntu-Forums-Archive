### Post by calimer on 2014-03-27
Platinum Arts Sandbox Game Maker Professional Edition features in game editing, coop editing, dynamic lighting, crossplatform support and advanced networking with the goal of adding visual scripting and many more features listed below.  Users will be able to rapidly create games of many different genres and require little to no prior coding experience.  Sandbox is available for Windows, Linux, Mac with Android support in the works and is based on the Cube 2 Engine.

There is still a free and open source edition available for anyone interested in trying it :)  It is also in the Ubuntu Repository which is not maintained by us.
