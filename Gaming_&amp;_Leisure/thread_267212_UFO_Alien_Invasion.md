### Post by skylark on 2006-09-28
The development pace of UFO:AI has been picking over the last few months. A new version was released yesterday. Grab it at: [http://ufo.myexp.de/]("http://ufo.myexp.de/"). 

The game is inspired by the original XCOM:UFO series of games (turn-based squad tactical combat). It is however fully 3D (uses a modified Q2 engine) and is entirely open-source (everything including the graphics are under the GPL license). Definitely worth checking out, although I'm biased since I've recently been helping out with some coding for it :) 

The game is far from finished though. The 2.0RC5 release should be interpreted more as a release candidate for a "technical demo" of the game rather than a full game. That said it is very playable in both single- and multi- player modes. There are still many half-finished features and a few bugs though so be warned. The longer-term development plan for the game is very exciting, with a new story-line being worked on for "2.1".

The community behind this project is very friendly and responsive. The UFO:AI forums and IRC channel (#ufo:ai on the freenode server) are usually quite busy along with the souceforge bug-tracker for the project. If you have some ideas or want to help out with the project please drop by :mrgreen: At the moment we are in need of some more graphical artists (for new models/maps, especially model textures) but anyone is welcome to help out.
