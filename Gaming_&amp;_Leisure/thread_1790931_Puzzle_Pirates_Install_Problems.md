### Post by Toz on 2011-06-25
If you're clicking or double-clicking the icon on your desktop to start the application, then you need to change the properties of the icon for it to work properly (doesn't look like the install program properly configures the icon).

I don't use ubuntu (use xubuntu) so I hope the instructions make sense. Right-click the icon and select Open With -> and select Gedit (I believe). Change the text **@client_name@** to read **yohoho**.

Save the file and try again.
