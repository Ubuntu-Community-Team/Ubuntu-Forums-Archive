### Post by mastablasta on 2015-01-05
try extracting files (possible if you have GOG version): [https://appdb.winehq.org/objectManager.php?sClass=version&iId=22375&iTestingId=80799](https://appdb.winehq.org/objectManager.php?sClass=version&iId=22375&iTestingId=80799)

for some reason older versions of wine give the game platinum rating. playonlinux makes it easy to use older Wine versions.

if you use wine to install the game, make sure you create the 32 bit wine prefix. if you have  64 bit OS by default wine will create 64 bit prefix.
