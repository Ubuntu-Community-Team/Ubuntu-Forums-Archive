### Post by hikaricore on 2007-07-28
Well that's not entirely correct.
There used to be (and probably still are) private servers running on a leaked/stolen beta server.
Such servers were buggy, outdated and obviously incomplete.

Connection to these servers required hacked or downgraded WoW installations that violated the TOS.
The groups working on, running, and providing downloads of these servers were slapped with a cease and
desist order from Blizzard's lawyers, which forced them to either close or go deep underground.

This is not the place to be asking about anything of this sort, but I thought I'd state the truth.
