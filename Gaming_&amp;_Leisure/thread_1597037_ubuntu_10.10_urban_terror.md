### Post by kaldor on 2010-11-06
> **slinky_41 said:**
> How do you load Urban terror.

- Download Urban Terror at their website ( [www.urbanterror.info](www.urbanterror.info) )

- Get the .ZIP for Mac and Linux here: [http://www.urbanterror.info/docs/texts/110/](http://www.urbanterror.info/docs/texts/110/)

- Extract it wherever (I use /home/username/Games)

- If you are on 32 bit, right-click "ioUrbanTerror.i386" and mark it as executable. For 64-bit OS, do it on ioUrbanTerror.x86_64.

- You can then double-click and run the game via one of those executables.

Hope this helped.
