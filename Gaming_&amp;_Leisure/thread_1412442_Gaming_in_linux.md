### Post by OgreProgrammer on 2010-02-25
> **rvndmnmt said:**
> 
Basically has a hard time loading multi-disk games.  Can't get the installer to recognize the drive after I mount the other disk.

Here is what I do. Rip the ISO of disks number 2 and up and place them in a directory on your hard drive. Usually the first disk is copy protected and it doesn't work. Anyway, we are not looking to steal anything. You can delete the ISOs after installing. I use gmountISO to mount each image when the time comes. 

So after the first disk is used and it asks for a second, I use wines cd eject command in terminal to force the disk to unmount.

```
wine eject -a
```

You can do this with each in turn as well if you don't want to image the disks. But its fussier sometimes.

Using gmountISO from the repository, I then fake mount the second disk and continue the install. And so on for each one until its done.
