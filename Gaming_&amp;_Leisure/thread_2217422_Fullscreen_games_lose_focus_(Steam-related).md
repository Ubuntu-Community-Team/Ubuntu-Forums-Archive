### Post by Ubutuxer on 2014-04-17
I've been using many PC's with Ubuntu 13.04 and 13.10 as a Steam gaming computer. There is a rather annoying issue where the full-screen game becomes out of focus (I assume?), because I can suddenly see the desktop mouse cursor instead of the game's own cursor. The game won't respond to any key presses at this point, since it doesn't seem to be focused anymore. I have to press Alt+Tab to get back to the game. It doesn't crash the game, but it's slowly getting annoying since this happens with every game, at least from Steam, and would seem to be completely random. Sometimes the game works for 10-20 minutes, sometimes it goes out of focus every 5 minutes. I am unable to figure out whether this has something to do with the Steam notifications, but I've disabled in-game notifications so I would think not. Some games where this happens: Broken Age, Broken Sword 5, The Novelist.

The only thing I'm certain is that it's not hardware specific, since I've seen this glitch on 5 or 6 different Ubuntu PC's so far. These computers include various CPU's, motherboards and video cards ranging from Intel HD Graphics to NVIDIA and ATI. There doesn't seem to be anything in common between these PC's. I can reproduce the issue on 13.04 and 13.10 with all updates installed to the 64-bit Ubuntu and Steam. I haven't tried any earlier Ubuntu versions with Steam and I am about to test the lubuntu-desktop window manager soon. Maybe it's a Unity/GNOME issue after all.

Anyone else playing full-screen games, especially Steam games, on Ubuntu? Maybe someone has a fix for this?
