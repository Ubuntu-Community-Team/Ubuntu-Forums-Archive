### Post by cor2y on 2007-04-03
The troubleshoot section recommends this
** No 3D acceleration **

 If you see output that looks like this: 

> 

$ fglrxinfo 
display: :0.0 screen: 0 
OpenGL vendor string: Mesa project: [www.mesa3d.org]("http://www.mesa3d.org/")

Then try these two commands:
> 
 mkdir -p /usr/X11R6/lib/modules/dri 
ln -s /usr/lib/dri/fglrx_dri.so /usr/X11R6/lib/modules/dri
