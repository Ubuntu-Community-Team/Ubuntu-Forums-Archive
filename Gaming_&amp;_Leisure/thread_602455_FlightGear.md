### Post by Robin T Cox on 2007-11-04
They are already installed. You just need to start Flightgear with the command (entered at a terminal):

> fgfs --aircraft=

followed immediately by the aircraft name. You can get a list of the available aircraft by entering the command:

> fgfs --show-aircraft

And a list of all the commands and options is available with:

> fgfs --help

I'm typing these from memory, so please excuse any errors as it's a little while since I used Flightgear.

Don't forget that there is a complete manual for Flightgear at:

[http://www.flightgear.org/Docs/getstart/getstart.html]("http://www.flightgear.org/Docs/getstart/getstart.html")

Also, you can download additional aircraft at the main Flightgear site (see Manual section 2.2).

Hope this helps.
