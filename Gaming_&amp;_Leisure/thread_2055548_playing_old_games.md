### Post by sienile on 2012-09-09
For Windows games you can use WINE. You can select the option to open the EXE with WINE in your file manager, or use the terminal method:
```
wine oldgame.exe
```

### Post by Stonecold1995 on 2012-09-09
Yes, Wine should work for old Windows games and DOSbox should work with DOS games.  Depending on the game, you may have to use a different emulator then DOSbox.  For example, the old Touhou Project games from 1996 for the PC98 (which ran DOS if I'm not mistaken) should be run on an emulator like Anex86.

If they don't work through Wine/DOSbox, you may have to install Windows in VirtualBox.  If the games are old, then the overhead of VirtualBox shouldn't slow the game down, and it should still be able to run at full speed.
