### Post by MaximB on 2009-12-30
I've posted about Age Of The Decadence in the past here : 
[http://ubuntuforums.org/showthread.php?t=692868](http://ubuntuforums.org/showthread.php?t=692868)
And here :
[http://lgn.linux-hardcore.com/age-of-the-decadence/](http://lgn.linux-hardcore.com/age-of-the-decadence/)

For those who don't know about this upcoming game yet, it's a deep single player cRPG with choices and consequences, several endings and outcomes to each quest and Fallout style combat. 
Once the game is finished they will try to port it to Linux, they use Torque Game Engine - so it should be no problem.

A few days ago they released a Combat Demo which surprisingly runs much better in Wine then on any Windows version.
None of the Windows bugs are present (after the little patch they released) and it runs flawlessly !

You can download the Combat Demo from here :
[http://www.irontowerstudio.com/forum/index.php/topic,1221.0.html](http://www.irontowerstudio.com/forum/index.php/topic,1221.0.html)

You will also need to apply the patch (just overwrite the .exe file with the new one) to play.

Note that this is only a Combat Demo in which you play a gladiator in the arena to fight for money and glory !
This is however not the "real" demo with quests, smart dialogs, story etc...

I don't know if they would release a Linux combat demo, they need to fix the Windows bugs first (which like I've said, are not present when running the game with Wine under Linux).

[B]There is also a Linux thread on their forums that you can ask for a Linux client (to make them try harder to port the game to Linux) :
[http://www.irontowerstudio.com/forum/index.php/topic,1235.0.html](http://www.irontowerstudio.com/forum/index.php/topic,1235.0.html)[/B]

**So test the Combat Demo and enjoy !**
