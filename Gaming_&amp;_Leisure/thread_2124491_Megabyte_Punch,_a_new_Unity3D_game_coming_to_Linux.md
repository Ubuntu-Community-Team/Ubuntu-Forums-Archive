### Post by Luckiboy on 2013-03-11
Megabyte Punch is an in-development game made by Reptile Games, a Dutch starting indie game studio. The game is built in Unity3D. The beta isn't available for Linux yet, but on Steam Greenlight (see link) the developer ensures there will be a Linux version. 

Link to the game: [http://megabytepunch.reptile-games.com](http://megabytepunch.reptile-games.com)
Link to the developer: [http://reptile-games.com](http://reptile-games.com)
Link to Steam Greenlight: [http://steamcommunity.com/sharedfiles/filedetails/?id=93137992](http://steamcommunity.com/sharedfiles/filedetails/?id=93137992)

PS: the game still needs a lot of yes votes on Steam Greenlight to be released on Steam ;)
