I don't know if there is a better place to post this, so...

The netpanzer game recently had to change its master server. Anyone getting the package from the repositories would have no way of knowing this and would give up in frustration. 

The file 'netpanzer.ini' needs to have the masterserver entry changed to 'netpanzer.selfip.net'.

More info at: [http://www.netpanzer.info/](http://www.netpanzer.info/)

Is there a better way to pass this on to the package maintainers?
