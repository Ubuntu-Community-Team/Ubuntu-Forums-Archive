### Post by oldrocker99 on 2012-10-29
Legends of Yore doesn't "install;" it runs from the extracted directory. What you need to do is open the Properties for the file legendsofyore.jar, go to the Permissions tab, and check "Allow executing file as program."

The legendsofyore.sh file is already executable, and I got the game running by:

cd path/to/Legends-Of-Yore
./legendsofyore.sh

I hope this helps.
