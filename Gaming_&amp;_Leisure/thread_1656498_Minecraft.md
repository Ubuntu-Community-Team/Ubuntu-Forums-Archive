### Post by AliceKingsley on 2011-01-06
A big part of the appeal for me is the exploration. There are no paths, no maps, and the world is infinite. (The first time you get within rendering distance of an area the terrain is generated.) I also love that you can change almost any part of the terrain with a bit of work. 

Another fun activity is building. You can make very interesting structures, houses, hideouts. I suppose after a while doing so might lose some appeal. However, people have taken on some wonderful, elaborate, challenging projects. 

The start of the game reminded me a bit of Myst - you're dropped onto an island with no instructions, and you just have to start exploring. (I wonder if anyone else had that train of thought when they started playing.)

And I like the lack of arbitrary objectives. It means the game is never "over", you just have to decide on something to do next. It makes it way more immersive than most of the games I've played, despite the world being made of cubes.

### Post by AliceKingsley on 2011-03-23
> **wog said:**
> I'm just trying to figure out how to run a server now and make it available for particular friends to play on. I suspect I'd have fewer problems if I wasn't also using this same machine for my computing life, too.

I've been able to run a sever without much difficulty. 
```
java -Xms1024M -Xmx1024M -jar minecraft_server.jar
```
Obviously, replace "minecraft_server.jar" with the path to wherever you put the file. 

From there, you should be able to use white-list.txt to add the players you want. My only issue is that if I move white-list.txt and the rest of the files the server generates, I can't redirect the server. It just generates new copies.

If you have any other issues, I might be able to help.

Edited to add: 
The big hangup I had with getting the server set up was getting a static ip address. I gave up on it and just use my current ip, and presumably I'll have to send an updated ip to the people I play with if it changes. It hasn't been an issue in the few weeks I've had it up, though.
