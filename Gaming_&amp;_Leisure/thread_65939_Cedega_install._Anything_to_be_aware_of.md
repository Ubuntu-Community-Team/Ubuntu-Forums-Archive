### Post by frank_b on 2005-09-15
Easy thing to do.

Just get the debian packages

"cedega_[version]_i386.deb",
"transgaming-fontinstaller_[version]_i386.deb" and 
"transgaming-mozctlinstaller_[version]_i386.deb"

(I didn't bother getting "Point2Play"...)

and follow the instructions in [http://digital-conquest.ath.cx/wiki/index.php/Cedega_4.2.1-1_howto#How_To_Install_Cedega](http://digital-conquest.ath.cx/wiki/index.php/Cedega_4.2.1-1_howto#How_To_Install_Cedega)

Note: the most recent version of cedega, 4.0 or something like that, didn't work in my Ubuntu because it depended on more recent versions of some files than the versions available in the Ubuntu repository, so I installed version 4.3.2-1 instead and it's working fine.
