### Post by KingBahamut on 2005-09-02
After a long time the Boson team has released a new version of their Real-Time Strategy game. Boson is game like Earth 2140, it is based on top of the KDE, Qt and kdegames libraries. This time, in addition to the usual packages, an installer with precompiled binaries which doesn't depend on Qt/KDE is availiable. This binary is linked statically against Qt, KDE, and several other libraries, meaning that you do not require an installation of these libraries.
  Highlights of this release include LibUFO port for user interface, graphical improvements such as realistic water, new effects and fully operational multiplayer support.
