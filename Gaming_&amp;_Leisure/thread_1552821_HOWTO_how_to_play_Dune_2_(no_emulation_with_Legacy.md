### Post by frenchn00b on 2010-08-14
(1)  Dune 2 , first screenshots, what is it?
[http://www.youtube.com/watch?v=tppjzT-su0Q](http://www.youtube.com/watch?v=tppjzT-su0Q)

[COLOR="Red"](2) We do not need wine nor any other emulators, we are using LINUX at 100% full speed. No wine emulator. It is a PURE LINUX product, called dune legacy ![/COLOR]

(3) download the file:
[http://www.myway.de/richieland/dunelegacy_0.96_i386.deb](http://www.myway.de/richieland/dunelegacy_0.96_i386.deb)
other archi: [http://dunelegacy.sourceforge.net/wiki/index.php/Main_Page](http://dunelegacy.sourceforge.net/wiki/index.php/Main_Page)

then 
```
dpkg -i dunelegacy_0.96_i386.deb
sudo apt-get -f install 
```

(4) 
```
mkdir /home/myusername/.config/dunelegacy/data -p
```
myusername which is your username

then copy the files into this just created folder (into data):
```

Dune2$ ls
ATRE.PAK   DUNE2.ICO  DUNE.CFG  ENGLISH.PAK  FRENCH.PAK  HARK.PAK   INTROVOC.PAK  MERC.PAK     OPTIONS.CFG  SAVEFAME.DAT  SETUPENG.DIP  SETUPFRE.DIP  SETUP.PIF  VOC.PAK
DUNE2.EXE  DUNE2.PIF  DUNE.PAK  FINALE.PAK   GERMAN.PAK  INTRO.PAK  MENTAT.PAK    ONETIME.DAT  ORDOS.PAK    SCENARIO.PAK  SETUP.EXE     SETUPGER.DIP  SOUND.PAK

```

(5)
Enjoy, they completely made all available and working
that's an amazing cool legacy dune 2 they made!! What an amazing work!

type :
```

dunelegacy

```

Enjoy !
