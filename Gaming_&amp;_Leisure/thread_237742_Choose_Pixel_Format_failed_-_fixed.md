Hi, 

The error is gone! OpenGL now works! \\:D/ 

I've had a "choosepixelformat failed" problem while running opengl games in wine for quite awhile. I have had the latest Nvidia drivers installed and glxinfo shows that direct rendering is happing.  I changed my xorg config in /etc/X11 to have the default display setting at 24 instead of 16. This fixed the probelem. 

I thought i'd let you know and hope that this might fix your problem (i have noticed that there are thousands with this issue)
