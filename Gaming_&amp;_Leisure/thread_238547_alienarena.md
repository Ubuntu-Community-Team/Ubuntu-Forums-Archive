### Post by Perfect Storm on 2006-08-18
Open the terminal then:

```
cd Desktop
sudo sh alienarena-2006ue-x86.run
```

Just follow the default settings. _BUT_ don't hit the **Run Alien Arena** button, instead exit the installer (the reason why is that you'll run into permission problem if you do so).

```
/usr/local/games/alienarena2006/AlienArena
``` 
to launch the game or applications tab ---> other
