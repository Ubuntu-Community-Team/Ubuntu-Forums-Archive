### Post by cgolson84 on 2011-11-04
First.  Binding of Isaac is a "roguelike" game.  That means they don't allow you to save the game and when you die you start from the beginning....this includes quitting while playing.  Quit = death.

I don't think they have fullscreen enabled in the linux version yet.  It doesn't seem to work for others either.

The only problem I have is that no matter what I try the game is very choppy when there are more than 2 things on screen.  I have tried running it in both a 32 bit and 64 bit distro, tried using a newer version of flash (11) and tried different Desktop Environments (XFCE, LXDE and KDE).  The game is almost unplayable for me with how choppy it is.  I should point out I am running a core i7 processor with a nvidia geforce GTX 560m graphics card...so it's something to do with the game.  I HAVE noticed that in the game if you right click the window you can see that it's using it's own built in version of flash (Flash 10) despite what you have installed.  I am wondering if this is what is causing the issue (a bad version of flash wrapped in).

I hope they get this worked out as I would like to be able to play it in Linux and not have to boot Windows for such a basic game.
