### Post by KaeseEs on 2006-08-01
'Dapper' ( or 'Dapper Drake' ) is the codename for Ubuntu 6.06, which is the most recent version.  You are probably running it.  If you want to check, you can open /etc/apt/sources.list with your favorite text editor.  Each line that starts with 'deb' should contain the word 'dapper' if that's what you're running.
