### Post by WienerWuerstel on 2010-11-22
PlayOnLinux is a GUI that helps you install Windows Games on Linux. It creates a new Profile for each supported Game and helps you install it and getting it to work. And because every Game you install trough PlayOnLinux has it's own Profile it's easier to maintain the Games and to tweak them without breaking the other Ones you have installed.
