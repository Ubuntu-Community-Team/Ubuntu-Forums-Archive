### Post by sffvba[e0rt on 2015-03-11
[PlayOnLinux]("https://www.playonlinux.com/en/") is a "front-end" for Wine (in other words it uses Wine to run Windows applications).  It makes Wine easier to use as there are pre-made scripts that set up Wine to work automatically with the windows application you are trying to use. 

As for LoL on Linux, it is possible but there has been a few bugs making it very difficult to play (I am not sure if the bug affecting the in-game store has been fixed) and every time there is a game update there seems to be new breakage.  What helps is if you keep an eye on the [main Wine site]("https://appdb.winehq.org/objectManager.php?sClass=version&iId=19141") as well as they will often show workarounds found by others when things break.

Best of luck and report back once you have started to try and have specific issues.
