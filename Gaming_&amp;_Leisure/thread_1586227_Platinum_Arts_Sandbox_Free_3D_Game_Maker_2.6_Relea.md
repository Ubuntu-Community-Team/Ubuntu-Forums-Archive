### Post by calimer on 2010-10-01
The 2.6 version for the popular 3D Game Maker Platinum Arts Sandbox Free 3D Game has been released!

For anyone that didn't know this project is actually developed by Coders using Ubuntu!  Not everyone is a linux user but the majority are.  The goal is also to remove all the NC content and feature it on the Edubuntu DVDs which has already entered talks!

Here is the scoop:
Platinum Arts Sandbox Free 3D Game Maker is available at [http://SandboxGameMaker.com](http://SandboxGameMaker.com) Key features include the race car vehicle simulator mode, additions to the RPG maker, new maps, widget menus, updates to the MovieCube machinima mode, cutscenes, in game video recording, and more. Also a Sandbox Social network is under development. Check out the homepage for more information.

In addition [http://FunFreeGamesHQ.com](http://FunFreeGamesHQ.com) has been created as a site to host Sandbox games and projects. It is under construction but will feature a Sandbox games section. It also features free flash games, tower defense games and free game downloads. To have worked posted on the site contact [http://SandboxGameMaker.com](http://SandboxGameMaker.com)

Feature List:
* Vehicle mode. Features NOS (/newent nos) and add bots for opponents to race! 
* For those of you wanting a human playermodel type /playermodel 9
* Lots of new maps including racetrack by Mortar and International Airport, Space, and a race car track for vehicle mode by Esequiel!
* Maps are now sorted by gametype in /packages/base
* New widget menu interface in the works. Compile with -DNEWGUI to activate
* The RPG now features melee combat, charged weapons such as fireballs and bows, ammo, teleports, triggers, cutscenes, savegames and movable boxes.
* Video recorder menu for easily recording videos! If you post them on youtube please include our webpage link :D
* Lots of new textures. Also beginning of the move to remove all Non Commercial content to make it an easier process to release commercial games! The goal for 2.7 is to be completely NC free.
* Lots of various Moviecube updates and improvements for creating your own Machinima.
* Sandbox Book (will be posted in the wiki) and many more tutorials on the way soon.
* SMD and IQM model support
* Lots of other various features and bugfixes!

About Platinum Arts Sandbox Free 3D Game Maker

Platinum Arts Sandbox Free 3D Game Maker is a 3D game creator and 3D game design tool based on the Cube 2 engine that allows users to quickly and easily create and edit their own worlds in game, even cooperatively. It is free, open source, and easy to use for Kids and Adults. It features a create mode, RPG Maker, Sidescroller Maker, MovieCube for Machinima, and Vehicle Mode for making race cars and other vehicles.

Feedback on this project is more than welcome.  Hoping to see some more linux games come from this :)
