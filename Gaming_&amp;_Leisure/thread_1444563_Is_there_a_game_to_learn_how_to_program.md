### Post by donkyhotay on 2010-04-01
I learned most of my coding by simply reading through the source code of other open source programs and tweaking things. Find a game you like, think of something you want to change about it (preferably small to start with), then start messing with it. You'll mess up a lot (especially at first) but eventually between that and reading various guides online you can teach yourself quite a bit.
