### Post by oldrocker99 on 2014-11-08
PlayOnlinux, [http://www.playonlinux.com/en/download.html](http://www.playonlinux.com/en/download.html) (it is available from the Ubuntu repositories, but is not the latest version) has an installation script for The Sims 3 (read the instructions listed in the installation page about how to start the game!) and should download Steam for Windows, from which you can then download The Sims 3 and probably play it well. You do need to this first:
```
sudo apt-get install wine
```

Good luck!
