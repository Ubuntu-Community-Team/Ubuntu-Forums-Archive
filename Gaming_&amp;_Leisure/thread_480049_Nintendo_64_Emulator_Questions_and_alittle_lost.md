### Post by BigSilly on 2007-06-21
I use Mupen64 myself, and can't say I've had the same trouble as you. Could it be a problem with where you got the file from? Try downloading it from Mupen's official site [here]("http://mupen64.emulation64.com/"). Easiest way to use it is to download the file, then double click it, which takes you to the extraction tool. Extract it somewhere, anywhere you like really, and then go there and double click the launcher. It's usually a diamond shaped icon. That starts Mupen64. I put all my roms in a folder called "N64 Games", and then pointed Mupen64 at it in the Configure, then Rom Browser settings. Make sure you leave all your roms zipped up. 

Honestly this is how all emulators work, pretty much. Mupen64 is pretty good too, and has fairly good compatibility. They've just released an update on Windows, so hopefully it won't be too long before us Linux users get it too. As a side note, where did you get TrueReality from? I've never even heard of it!
