### Post by Tomatz on 2008-11-07
Found a nice Linux benchmark tool and thought it would be good idea if we compared results of the wolf et benchmark :)

**Get it here (debian/ubuntu package 1.4):**

[http://www.phoronix-test-suite.com/?k=downloads](http://www.phoronix-test-suite.com/?k=downloads)

The .deb just installs scripts and dependency's so once it installs we must do a couple of other things.

**Open a terminal and type:**

```
phoronix-test-suite benchmark et

```
This will download the wolfenstien et benchmark and run it... simple ;)

After it has run it will give you an option to view the results in your default web browser. Then just post the results here!


If you want to run additional tests, **open a terminal and type:**

```
phoronix-test-suite list-tests
```

Then just

```
phoronix-test-suite benchmark <TEST NAME>
```

to install and run them.
