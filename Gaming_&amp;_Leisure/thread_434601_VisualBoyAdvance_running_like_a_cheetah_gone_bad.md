### Post by Extreme Coder on 2007-05-06
Hi Tux0r,
Are you using the CLI to launch a game? If so, then I'd recommend trying to run the command
```
gvba
```
It's a GUI for VisualBoy Advance, just like in Windows. Too bad it isn't added to the GNOME/KDE Menu by default. Anyway, after you run the GUI, go to Options -> Frameskip -> 0 .
This will make the game run at 100% speed with no frame skipping.

Hope this helped!
