### Post by bernardfrancois on 2009-11-14
If you right-click on the package in synaptic, and you choose properties, you can find a list of installed files in one of the tabs. One of them is the executable which can be run from a terminal window. Usually this file is located in a specific folder such as /usr/bin, /usr/local/bin, /usr/sbin, ...
