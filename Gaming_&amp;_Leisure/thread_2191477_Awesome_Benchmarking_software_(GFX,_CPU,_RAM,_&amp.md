### Post by dannyboy79 on 2013-12-02
There's a great software suite that can be installed which allows for easily benchmarking various aspects of your computer. It actually runs great in my Xubuntu 12.04.3. All I had to do was install the deb file and ensure I had the php related dependencies ```
sudo apt-get install php5-cli 
``` and I was golden.

It allows you to install many tests from Unigine like Valley, Heaven, Sanctuary and others. It easily allows for testing your machine using Team Fortress 2, Xonoitic, Urban Terror and a ton of other games. Get the deb here: [http://www.phoronix-test-suite.com/](http://www.phoronix-test-suite.com/)
The version in the official Ubuntu repo's is very old and just returns a php error and I solved by uninstalling that version from the repo's and installing from the deb from their website. I will be starting another thread to compare unigine results similar to the GPUTest and glmark2 thread
