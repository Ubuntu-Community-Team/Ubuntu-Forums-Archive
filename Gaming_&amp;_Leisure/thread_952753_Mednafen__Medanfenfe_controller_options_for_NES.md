Hi, launch mednafen and a nes game.
Press alt + shift + 1
Mednafen makes it easy to map both your controller and keyboard if you like.
After that you are all set. 
Press F1 to find out the other hotkeys.

### Post by Arylon on 2008-12-14
You should be able to find the Mednafen documentation on your computer at file:///usr/share/doc/mednafen/mednafen.html

If you need to reset Medafen to its default settings, just delete the file at ~/.mednafen/mednafen.cfg  a new file with default setting will be automatically created.

You mentioned the Command key, are you using a Mac keyboard? If you have a Mac notebook, try holding down the fn key while pressing F1.
