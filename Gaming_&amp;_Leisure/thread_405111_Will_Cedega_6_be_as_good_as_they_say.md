### Post by justin whitaker on 2007-04-11
Ok, I cancelled my subscription earlier in the month, so I still have access to the latest release. I downloaded it, and started the install. I was one of the folks that ran into the dash/bash problem with 5.10, so I was surprised when the install went off without a hitch.

I really do not have that many games, and those that I do are running fine on Crossover. I basically have a copy of System Shock 2 via Underdogs, and Guild Wars. 

First up: System Shock. The download does not install, per se. Running SystemShock2.exe via Cedega extracts the cab files into a directory. From there, you can run an oggsetup.exe, or Sshock2.exe. If you run the Sshock2.exe the game graphics will load, but no sound. You have to use the oggsetup.exe first, and then run Sshock2.exe. 

The game does run, flawlessly in fact. You just can't actually install it in Cedega's slick GUI because the game does not actually install as far as Cedega is concerned. 

So, that's a 7-8/10 for a title that was listed as "garbage" on WineHQ.

I'll let you know how GW goes.
