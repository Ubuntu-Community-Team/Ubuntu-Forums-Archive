```

```

You can also download the .deb file of the game and easily install it just by double clicking on the file and then select install package.

### Post by mikedep333 on 2009-05-03
"I have no time to try and select which game is good a which is not."

which games are good and which is not is highly subjective.

But there are recommended lists by many people. You can go through that list and identify the packages for them. Then put those packages names into a script, eg:

sudo apt-get install supertux-stable nexuiz tremulous
and continue with more packages (supertux-stable, nexuiz, and tremulous are packages)

Here are some threads with lists of good games. (not every game listed is in the repos)
[http://ubuntuforums.org/showthread.php?t=1014360&highlight=list+games](http://ubuntuforums.org/showthread.php?t=1014360&highlight=list+games)
[http://ubuntuforums.org/showthread.php?t=1070829&highlight=list+games](http://ubuntuforums.org/showthread.php?t=1070829&highlight=list+games)
[http://ubuntuforums.org/showthread.php?t=1074685&highlight=list+games](http://ubuntuforums.org/showthread.php?t=1074685&highlight=list+games)

EDIT: This is also probably a very good list if you sort by popularity.
[http://appnr.com/category/game/order/popcon-desc](http://appnr.com/category/game/order/popcon-desc)

Also, btw, keep in mind that you do not want to install EVERY game in the repos. Many are incomplete or only command-line based. Your games menu will get cluttered. Try to install recommended/popular ones only.
