### Post by lukeiamyourfather on 2013-08-19
Open a terminal, type in "sudo" and then drag that installer to the terminal window, hit enter. The "sudo" command means to do something with root privileges (administrator privileges in the Windows world). It will ask for a password and use the same password you use for your account.
