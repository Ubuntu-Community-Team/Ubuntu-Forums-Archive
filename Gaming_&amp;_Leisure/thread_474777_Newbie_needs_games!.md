### Post by ZeroXR on 2007-06-16
If you're a fan of old school shoot-em-up games in the same vein as Ikuraga but with a retro graphic remix, check out the Linux port of rRootage in the Ubuntu Add/Remove Program menu with the filter set to All Open Source Programs.

The only keys you need are the arrows to move your ship, [Z] to fire your laser, and [X] for your special (which varies from each game mode.

The techno sound track goes so well with the retro graphics rendered in OpenGL. I'd say it's pretty technical in skill for an old style game. :-)
