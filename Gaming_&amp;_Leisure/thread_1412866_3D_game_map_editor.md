### Post by shazbut on 2010-02-21
Not sure what you mean by terrain manager, so the following could be completely irrelevant...

Sauerbraten has in-game map editing, which actually works better than it sounds. Its map building methods are a bit different to the old bsp tools ala Quake engine etc, but it seems quite speedy once you learn how it works.  It also allows multiplayer map editing, which can be fun, or a PITA depending on how cooperative the other players are.
