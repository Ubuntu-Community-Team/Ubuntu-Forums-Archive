### Post by acoustibop on 2008-03-12
I use pSX, EmosSuck777, because it runs so well.  Despite the fact that it doesn't use videocard hardware rendering, it probably doesn't make your CPU work any harder than ePSXe would, so using ePSXe - or PCSX, the other (also plugin-based) Playstation emulator with a Linux version - probably wouldn't get your laptop running any cooler.

Don't forget that pSX works differently to PCSX and ePSXe - it's about accurate rather than enhanced emulation - and is a good 4 years on from where they left off.

I've had ePSXe and PCSX running in Linux before, and I can assure you - not only do they not work as well as pSX, but they're a real pain to install, configure and get running in Linux.

You find new ways to use a graphics card (or sound card, for that matter). Not everyone has computers that can push a reasonable FPS with an emulator, using the cpu alone.
