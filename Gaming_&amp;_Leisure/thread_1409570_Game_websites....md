### Post by donkyhotay on 2010-02-19
> **jblaven said:**
> Yes,

Some are flash type games.

Ok... *some* are flash type games. What are the others that aren't flash? Flash games are (fairly) benign since they're within your browser. Some flash games can be a privacy concern (read up on [flash cookies](http://www.eff.org/deeplinks/2009/09/new-cookie-technologies-harder-see-and-remove-wide) for more information) but generally no more then any other site that uses flash like youtube. Now games that you download/install are a bigger concern. Generally this requires root access (i.e. bypassing your systems first/best security) and can give you malware. Even games that don't install with root access *can* do some bad stuff (like root around in your home folder) but this is pretty rare. It's generally a matter of common sense, if you go online to "install-a-virus.com" type sites to download your games, you're just asking for trouble. On the other hand if you go somewhere reputable that has high recommendations from other people then you're probably fine. Downloading/installing games from the ubuntu repositories is always good since those are files that have been checked out by ubuntu community as a whole. Also downloading/installing FOSS games is good because with access to the source code you (or at least your geeky friend you trust) can review the code and look for anything malicious. Having linux is by itself more security then anything you can do to a system running windows, but just remember there is no such thing as immunity and a little bit of common sense goes a long way. Also, as someone mentioned before there are virus programs for linux, however they are primarily designed to run on a server and search for windows viruses to prevent them from spreading between windows systems. They aren't really designed to protect your linux system from linux viruses.
