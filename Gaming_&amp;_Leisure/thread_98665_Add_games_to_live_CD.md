### Post by RadixLecti on 2005-12-04
[QUOTE=kaaskop]I'm wondering if it's possible to add games to a live CD so my kids can bring the CD to school or friends and play there favorite games without installing any software?[/QUOTE]

Sure. There are lots of sites on how to remaster live CD's (most notably Knoppix). Thing is, if the games are more demanding than, say, pacman, they're probably going to run slower than treacle. Both the OS and the game will be using the same RAM-memory.

Next thing is if the game uses 3D-hardware acceleration. If it does, you'd have to include the appropriate drivers on the live-CD.
