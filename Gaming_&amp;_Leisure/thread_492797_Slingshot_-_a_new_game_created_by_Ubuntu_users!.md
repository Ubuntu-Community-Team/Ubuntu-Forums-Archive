### Post by musther on 2007-07-05
Slingshot is a new game for linux/windows, it was created by a couple of Ubuntu users.

We've just released our first version, 0.8p, and have a package available for Ubuntu.  This version is believed to be stable, works on Ubuntu, Kubuntu, and presumably Xubuntu - however it is a little slow on older machines (future versions will make it faster, so if it is a little slow for you, stay tuned!).

Basically the game consists of two spaceships which have to shoot one another through a field of planets, the gravity of the planets pulls the shots around.  The planet fields are randomly generated (just to keep you on your toes).  The gravitational effects are what make the game interesting and great fun!

**Check out the official website (the usual screenshots etc) at:**
[http://www.slingshot-game.org](http://www.slingshot-game.org)

**Here is a direct link to the Ubuntu package:**
[http://downloads.sourceforge.net/slingshot-game/slingshot-0.8.1p.deb](http://downloads.sourceforge.net/slingshot-game/slingshot-0.8.1p.deb)

**If you're running 64 bit Ubuntu, see this page:**
[http://www.slingshot-game.org/Downloads/Linux/64deb](http://www.slingshot-game.org/Downloads/Linux/64deb)

Have fun!
