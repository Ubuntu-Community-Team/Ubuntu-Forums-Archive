### Post by Boni Boy Blue on 2011-04-18
It's pretty easy to turn off the EQ. Just open the EQ window from "View" menu and there should be a check-box with "enabled" next to it. Make sure that is unchecked to deactivate the EQ.

### Post by flight878 on 2011-05-07
The equalizer can help improve the sound quality, especially for those that have substandard speakers.  One of the best equalizer presets IMO has been Winamp's "Full Bass and Treble".  The workaround provided by another poster however does not work properly because the frequencies tuned in Winamp are distinct from those in the Banshee EQ.

Here are the frequencies and corresponding gain settings for the original Winamp "Full Bass and Treble" preset:
[FONT=Courier New] [FONT=Verdana]60 Hz ->  4.4 dB
170 Hz ->  3.6 dB
310 Hz ->  0   dB
600 Hz -> -4.8 dB
 1 kHz -> -3.2 dB
 3 kHz ->  1   dB
 6 kHz ->  5.1 dB
12 kHz ->  6.7 dB
14 kHz ->  7.4 dB
16 kHz -> 10.9 dB[/FONT]

[FONT=Verdana]To use these settings in Banshee, the closest equivalent settings (based on the different frequencies) are:
[FONT=Courier New][FONT=Verdana]29 Hz ->  0 dB
 59 Hz ->  7 dB   [/FONT][FONT=Verdana](not quite here, I like a little more bass; it would normally be set to 5)
119 Hz ->  4 dB
237 Hz ->  2 dB
474 Hz -> -6 dB
947 Hz -> -3 dB
 2 kHz ->  0 dB
 4 kHz ->  2 dB
 8 kHz ->  6 dB
15 kHz -> 10 dB [/FONT]
[/FONT][/FONT]
[FONT=Verdana]Comparing the settings in Banshee and Winamp, the sound using a pair of Sennheiser headphones was quite close.  You'll have to create the preset with these settings.[/FONT] 
[/FONT]
