### Post by Rodneyck on 2006-12-14
I love Ubuntu, but there is some initial setup and tweaking that needs to be done, so hunker down in the threads for some reading. Those include multimedia codec setup (you can't watch DVD's out of the box), graphic drivers, hardware that is not detected, and any cool 3D effects from either beryl or compiz.

I did find a very good transition distro the other day called Sabayon. It uses KDE (which I do not like) but as pointed out previously, is much like XP. Sabayon takes it to the next step and includes a start button and menu system that is much like XP, only better and faster IMO. The DVD has a livecd so you can see it for yourself before installing. The DVD also comes with tons of applications already installed. There is also a lighter CD you can use as well.

The best part about Sabayon is it comes with the 3D effects (Beryl) already installed with a handy walk through that detects your system and offers hardware acceleration settings in the initial startup. It also includes all the codecs (multimedia) settings installed and many other features you have to manually set up in Ubuntu. Sabayon also detects many more driver specific apps. My zen micro music player was detected in Amarok right off the bat, not so with Ubuntu. 

So give both Ubuntu and Sabayon a try, they each have a liveCD you can play with before deciding.  Good luck.

### Post by zekopeko on 2006-12-14
AUTO HIDE

to get the auto-hidden panel not stick out go to System > Prefrences > Menu Layout.

then go to system tools and there choose Configuration Editor.

now you have Conf. Editor in Applications > System Tools.

the other option is ALT+F2 and the type gconf-editor just to run the program and not to put it in your Apps Menu.

Open it and then go to apps > panel > toplevels

there find your panel (there should be 2; something like panel0 and top_panel_screen0)

click on the one you want to autohide and in the little window look for auto_hide_size.

it should be something like 6, just change it to 0 et voila! you have a TRUE auto-hidden panel
