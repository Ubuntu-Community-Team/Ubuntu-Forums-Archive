### Post by Rinzwind on 2010-09-12
You can play several windows games in Linux if you have the original CDs/DVDs. Install PlayOnLinux and have a look at the list.

It uses wine and wine currently has a list of 4508 games.
Have a look here: [http://appdb.winehq.org/objectManager.php?bIsQueue=false&bIsRejected=false&sClass=application&sTitle=Browse+Applications&iItemsPerPage=25&iPage=1&sOrderBy=appName&bAscending=true](http://appdb.winehq.org/objectManager.php?bIsQueue=false&bIsRejected=false&sClass=application&sTitle=Browse+Applications&iItemsPerPage=25&iPage=1&sOrderBy=appName&bAscending=true)

There are 2 pokemon games btw:

Pokemon NetBattle	
Online pokemon battle game with full selection of monsters and moves from all the pokemon games from green to emerald

Pokemon World Online
This game give you the possibility to create an avatar and to play exactly like on Game Boy. But many of the other people are real person exactly like you. You can catch pokemon have fight with other pokemon and chat with other players.

### Post by lobralleo on 2010-09-12
Hi and welcome! :)

Actually, a number of pretty nice games is available under Ubuntu and in general for Linux, most of them being readily available in the Ubuntu repositories.
You can find many more through the Web; some links that I found useful to start with are:

[http://www.linuxgames.com/](http://www.linuxgames.com/)

[http://www.happypenguin.org/](http://www.happypenguin.org/)

[http://www.lgdb.org/](http://www.lgdb.org/)

Plus, you could be able to run several Windows-based games under linux using Wine (which is also available in the repos):

[http://www.winehq.org/](http://www.winehq.org/)

And also check PlayOnLinux:

[http://www.playonlinux.com](http://www.playonlinux.com)

Some mainstream games are also released natively for linux platforms (in my experience, a good example in the recent past has been Neverwinter Nights).
Your level of satisfaction with games in Ubuntu will likely depend on what you are interested in; you will not be able to run the latest hits from the big companies (usually), but there is definitely enough choice in genre and titles for many hours of play.
Have fun! ;)
