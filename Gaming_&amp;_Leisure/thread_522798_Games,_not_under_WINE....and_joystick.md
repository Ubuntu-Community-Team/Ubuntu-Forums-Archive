A USB PS2-Style controller that I think works amazingly well (I have three):

The Dual-Action Gamepad (some models come without the Mode Switch, all mine have it):
[http://www.logitech.com/index.cfm/gaming/pc_gaming/gamepads/devices/288&cl=us,en](http://www.logitech.com/index.cfm/gaming/pc_gaming/gamepads/devices/288&cl=us,en)

The Rumblepad 2 (same thing but with rumble, historically known not to work in linux but there is some experimental rumble features in the newer kernels, dunno if it works now):
[http://www.logitech.com/index.cfm/gaming/pc_gaming/gamepads/devices/264&cl=us,en](http://www.logitech.com/index.cfm/gaming/pc_gaming/gamepads/devices/264&cl=us,en)

The main thing to check for in a controller is that it uses an USB HID compliant driver. Basically, if you can plug it into a window machine without needing to install a driver, it should work in ubuntu as well.
