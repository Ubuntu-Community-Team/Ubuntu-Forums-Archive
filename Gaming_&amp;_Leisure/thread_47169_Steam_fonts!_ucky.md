### Post by smiley2billion on 2005-07-23
This error is due to the font Tahoma not being installed.

To fix this simply install the tahoma fonts via wine.

It can be downloaded here:

[http://www.lernnetz-sh.de/html/schule/software/software/font/tahoma/tahoma.exe](http://www.lernnetz-sh.de/html/schule/software/software/font/tahoma/tahoma.exe)  

then in console:  $ wine tahoma.exe

This should let you see the text on steam.
