### Post by fastb on 2005-10-16
Find the config file of call of duty.
Now once you've opened it with a text editor find full_screen "1"  somewhere. Now replace that 1 with the 0. Save the config file. Run the game. Change the resolution to 1280x1024. Close the game and change the config file to what it was before.0 to 1. The problem is solved. 
Note that you have to have the same resolution in the game and desktop.
