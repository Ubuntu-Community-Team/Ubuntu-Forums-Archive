### Post by acoustibop on 2008-05-31
I think there probably will be a Linux version sooner or later, but I'm not holding my breath.  Considering there was more than a month and a half from the official announcement that there would be a new version to the actual Windows release, I don't think they're in any hurry.

If, when it's released, I find myself with some time on my hands for all the messing about getting it set up and configured, I might give it a try - but considering how far ahead pSX is by now, it'll probably only be novelty value for me.  Out of the thirty-odd Playstation games I own, I don't think there's a single one that doesn't run well in pSX and better than in ePSXe.  But several of them won't run on ePSXe, and more run very badly.

I liked ePSXe back in its day, despite the games that wouldn't run on it, the slowdowns, graphical faults etc and, contrary to many emulator users, I enjoyed messing around with the plugins etc.  But when pSX is so much simpler to use and works so much better - no brainer!
