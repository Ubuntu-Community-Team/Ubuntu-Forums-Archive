### Post by cchhrriiss121212 on 2011-01-28
Dolphin works quite well for many games now, and there is no difference between the Linux and Windows versions. It is open source so progress is made rapidly, try testing out the latest svn build, or use this up to date PPA:
[https://launchpad.net/~glennric/+archive/dolphin-emu]("https://launchpad.net/%7Eglennric/+archive/dolphin-emu")

Choppiness can also be the result of underpowered hardware as Dolphin requires at least 3GHz dual core CPU.
