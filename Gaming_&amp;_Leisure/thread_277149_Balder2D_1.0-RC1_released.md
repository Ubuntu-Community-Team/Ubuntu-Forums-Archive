### Post by holomorph on 2006-10-23
heh, excelent.  Care to make some?  The latest release has a gimp python-fu script that makes level creation pretty dang easy, just create a 'geometry' layer (black and white, white=wall), a 'background' layer (anything you want, it will be the image that gets displayed), and a 'spawns' layer (red pixels are spawn locations, anything else is meaningless).  Run the export script, and put the three resulting files in their own directory under the 'Maps' directory.
