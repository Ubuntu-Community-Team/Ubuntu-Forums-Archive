### Post by crane on 2005-09-05
[Quake2 Lost Marine](http://www.planetquake.com/meanarena/quake2_lostmarine/) is a mod for Doom 3. It seems to be a pretty cool game although I haven't played a lot of it yet. Just thought I'd share!!

 It's a first release and the only downloads I've found are windows executable. I tried running under windows but was unable. I don't think my windows install has the 1.3 patch for doom3.

 After booting back to linux, I copied the quake3_lostmarine file from my windows partition to my doom3 folder in linux and ran the game with no problems.
