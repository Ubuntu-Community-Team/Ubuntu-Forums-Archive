### Post by K.Mandla on 2006-11-12
If you right-click on the file and select Properties, you should see a tab that says Permissions. Find the box that says "Allow this file to be executed" (or something like that). Then you should be able to double-click and the file will run. 

If that doesn't do the trick, open a terminal window where you have the file and try this command.

```
sh doom3-linux-1.3.1302-sdk.x86.run
```
