### Post by Sand &amp; Mercury on 2009-10-28
Quake 2 linux binaries floating around. If you're lazy though, it works fine if you just install wine. QuArK is pretty old too, so I think it'll work okay too.

Wine just in case you weren't aware is a program that adds a layer of compatibility with Windows programs to Linux. You can install it through the synaptic package manager, just look up 'wine' on the list, tick it and hit apply. 

Alternatively, a quicker way is to just open up the terminal (Apps > Accessories > Terminal) and typing 

sudo apt-get install wine

### Post by DoktorSeven on 2009-10-28
If I'm reading it right Quake Army Knife/QuArK is a level editor; you can use [GtkRadiant](http://www.qeradiant.com/cgi-bin/trac.cgi) on Linux for this (binaries are on [GetDeb](http://www.getdeb.net)).

There are a variety of Quake II engines out there if you search for "quake II linux", even a [Java version](http://www.bytonic.de/html/jake2.html). :)  Should be no need for Wine.
