### Post by FranMichaels on 2008-10-23
Sure. Go to Applicatons -> Add/Remove, make sure it's set to search All Apps, and type midi.

From there click rosegarden. 
For playback, timidity is the best way to go (unless you have hardware midi support), I recommend the fluid synth sound font over the freepatches, there is another thread about that though. Have fun.
