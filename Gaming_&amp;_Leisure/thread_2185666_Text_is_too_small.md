### Post by sudodus on 2013-11-04
Welcome to the Ubuntu Forums :-)

Usually it is possible to increase the size of 'everything' shown by the web browser, including the font, with the hotkey ***ctrl +*** 

It can be repeated, and it can be reversed with ***ctrl -*** and reset the the standard size with ***ctrl 0*** (control zero).
