### Post by fishbulb1022 on 2010-05-09
In Mupen64 on the computer you have the game saved on, open the rom and load your saved game. Then, in the Mupen64 window (not the game window) click emulator > save state as, and put it somewhere you can find it. Copy that file to the new computer, open Mupen64 and start up the game. Then, again in the Mupen64 main window, click emulator > load state from, and select the file. Once it has loaded, save the game as you would normally.
