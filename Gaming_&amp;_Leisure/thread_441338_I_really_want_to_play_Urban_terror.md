### Post by noerrorsfound on 2007-05-12
1. Download **both** ioUrbanTerror and Urban Terror zip files (click the link that says linux/mac).
2. Extract ioUrbanTerror by right clicking and then clicking *Extract Here* (it will make a directory called *ioUrbanTerror_1.0.zip_FILES*).
3. Extract Urban Terror into the Linux-i386 directory inside of ioUrbanTerror's folder (if extracted to desktop, it will be ~/ioUrbanTerror_1.0.zip_FILES/Linux-i386).
4. Make *ioUrbanTerror.i386* (in your Linux-i386 folder) executable by right clicking on it, clicking Properties, click the Permissions tab, and then check the box that says *Allow executing file as program*. Alternatively, you may do this from the command line by typing (if extracted on desktop) *cd ~/Desktop/ioUrbanTerror_1.0.zip_FILES/Linux-i386* and then *sudo chmod +x ./ioUrbanTerror.i386*.

That's it. Run Urban Terror by clicking the *ioUrbanTerror.i386* executable that is in the Linux-i386 directory.

Just download version 4.1 and extract the folder anywhere. Set the ioUrbanTerror.i386 to executable by going to permissions and check the "make executable " box. Double click it and it boots right up.
