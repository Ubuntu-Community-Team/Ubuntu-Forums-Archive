### Post by NovaWasp on 2010-12-23
Those of you using the xbox 360 wired controller are likely using the xboxdrv driver.

You can configure the driver to map key board keys.  The following will control well for Braid.

```

--ui-buttonmap  a=XK_space,b=KEY_LEFTSHIFT,x=KEY_LEFTSHIFT,y=XK_y,rb=XK_r,lb=XK_l,start=KEY_ESC,back=KEY_F1,dl=XK_Left,dr=XK_Right,du=XK_Up,dd=XK_Down
```

This is of course just the --ui-buttonmap options.
