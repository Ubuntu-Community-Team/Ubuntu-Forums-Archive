### Post by patrick295767 on 2007-01-19
So, how to :
  
1/ Install kxmame or gxmame or advancedmenu [http://advancemame.sourceforge.net/](http://advancemame.sourceforge.net/)
[http://www.zophar.net/frontends/mame.html](http://www.zophar.net/frontends/mame.html)
  
2/ create mame folder
  
3/ google roms 
then get all the roms for killer instinct 1 
(version 1.5  1.4 1.3 ...) 
and copy them as zip format in the mame folder you just created
 
lower case is important to be !
  
4/ get the file : 
ki.img
or
kinst.chd
make a zip of it if it s not as zip yet
  
5/ mkdir mame/kinst
 
6/ 
mame/kinst/kinst.zip
should be thre

7/ 
run the kxmame for instance
 check the options > folder of roms to be mame
dont be surprise kxmame prefer ZIP format to non-zip
so everythg is zipped : cool no ?
  
8/ update databse of games
  
9/ double click on killer instninct 1
 
10/ play
