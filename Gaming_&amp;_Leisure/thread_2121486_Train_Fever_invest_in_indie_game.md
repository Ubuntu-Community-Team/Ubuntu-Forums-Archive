### Post by Oligarf on 2013-03-02
With Steam selling Linux games now, doors have opened in many ways. Gambitious makes it possible to invest in game development, so too in new Linux games. It looks like an actual cross-platform game coming to Linux will get funded: Train Fever.

Train Fever is a modern-day Transport Tycoon (business simulation) based on an engine with a great innovative scope. Specialized in procedural content and urban simulation. 

Pre-alpha trailer: [http://vimeo.com/55925430](http://vimeo.com/55925430)

For more information, see: [http://gambitious.com/ideas/2496-train-fever-pc-mac-linux](http://gambitious.com/ideas/2496-train-fever-pc-mac-linux)

My personal interest? I would like to see the game being made so I invested 100 euro.
