### Post by maheshjr2000 on 2009-04-28
OpenGL 2.1 not available:

If you have an intel card. Your cards are NOT supported.

If you run 
# glxinfo | grep "OpenGL version"

and the result is 3.0 and you STILL get the error:
Check [http://forums.savage2.com/showthread.php?p=267148](http://forums.savage2.com/showthread.php?p=267148) FIRST before posting with your problem. Use the solution posted. 

Please post any other common problems AND solutions in this post so new posts do not have to be created.

Using ATI drivers for 9.04:

Savage 2 1.5 does NOT work with these drivers to solve this problem run this after installation:
dedicated_server.sh
It will update Savage 2 and you will be able to play.
