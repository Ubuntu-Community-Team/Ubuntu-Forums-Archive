### Post by disturbed1 on 2007-11-15
Running desktop effects? (Compiz-Fusion etc...)
Turn them off, then try again.

Also, if you have a program running through Wine in the background, I've notice some errors while doing this with different overlay outputs (VLC, OpenGlL games). Check the system monitor to be sure.

You could also try following the instructions given in the commandline just after the error ;)

> currently ignored for resolution filtering. **_*Set r_useSDLModes to 1**_* if you want it
Initializing SDL subsystem


If you've fiddled with the .cfg file at all, perhaps you changed something that shouldn't have been changed. Hopefully there's a backup made for you to restore to, or attempt to delete it. Worse case, you'll just have to reinstall the game.
