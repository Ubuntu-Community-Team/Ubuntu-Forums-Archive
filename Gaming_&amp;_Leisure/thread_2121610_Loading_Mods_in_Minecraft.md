### Post by I8NY on 2013-03-04
You can add non-steam games into the steam library by clicking Games> Add a non-Steam game to my library but that is another topic to discuss later.  The Linux version of Minecraft is the jar version that you have.  To get a clean install, you will need to delete a folder called ".minecraft" that is in the home folder (you will need to set show hidden files to see it).  It also has your saved worlds in there so back them up before deleting the folder.  The other option is to force minecraft to update while logging in.  Click options when it asks you to log in to view the force update button.
