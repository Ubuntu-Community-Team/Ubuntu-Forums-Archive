### Post by MrBlond83 on 2006-10-10
Also if you encounter problems with the instructions on evolvedclan.com please describe them and I'll do my best to try and help. I installed using those instructions and had no problems, as all the files there are binaries and executables and nothing that requires compiling of any sort. Basically it's just: install savage from the .sh in the .sh.gz file, update to 2.00c using the patch on liflg, and then update the game using the SEP-3T mod (which updates the game to the latest version, as it seems S2 did not make an update for linux for that version).

I'm running a pretty average Ubuntu system and did not need to install anything except those files, but if you get error messages about any library missing I'd use apt-file to locate those files. Also, of course you need 3d drivers installed and working :)
