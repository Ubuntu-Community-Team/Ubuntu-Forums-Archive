### Post by arinlares on 2009-09-16
[http://www.playdeb.net](http://www.playdeb.net)

They've got a bit of everything.  I have personally played AssaultCube, and Sauerbraten (unfortunately, this one only on Windows, as my Ubuntu-running netbook doesn't run it too well).  I highly reccommend both of these.

Playdeb.net is an unofficial Ubuntu gaming repository still in beta testing, but I've gotten some use out of it's repository over the past couple days with no problems.  If it is open, Linux-compatible, just request it in the getdeb bug report section of Launchpad, and they'll likely add it.
