### Post by Epsilon on 2007-09-08
Theres a ton of tutorials on the net for allegro and whatnot, learning is free.
But if you're willing to shell out some money I'd suggest you get the "Game Programming All in One" book. It teaches you the basics of how to create tank games, side scrollers or vertical scrollers like 1942.
Then when you feel you've mastered that, you can move on to "3D Game Programming All in One", it's centered around the Torque 3d engine which is Linux native.
