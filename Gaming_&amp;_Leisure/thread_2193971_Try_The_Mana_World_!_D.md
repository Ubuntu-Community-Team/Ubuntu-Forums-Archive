Hi I have wanted to talk to you about this little but big game, its a MMORPG ([https://themanaworld.org/](https://themanaworld.org/)) , with a unique and great community 
here are some screenshots taken from [https://wiki.themanaworld.org/index.php/Screenshots](https://wiki.themanaworld.org/index.php/Screenshots)

This game has many clients but the best one to play it is Manaplus, you can download Manaplus here [http://manaplus.org/](http://manaplus.org/)
it works for almost every OS (android tables and phones too !! :D)
Just go here [http://manaplus.org/ubuntu](http://manaplus.org/ubuntu) , open that deb file [http://repo.evolonline.org/manaplus/ubuntu/manaplus-addrepo_1.3_all.deb](http://repo.evolonline.org/manaplus/ubuntu/manaplus-addrepo_1.3_all.deb)  and the run this commands on terminal ```
sudo apt-get update
sudo apt-get install manaplus
```


you also need to register , here [https://themanaworld.org/registration.php](https://themanaworld.org/registration.php) and just log in using Manaplus and make a character , if you need any kind of help just type in the game chat ```
@wgm "Something here"
``` and probably a Game Master would show up to help you :D .
