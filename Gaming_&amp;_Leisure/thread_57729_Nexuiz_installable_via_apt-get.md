### Post by smack on 2005-08-22
If you got a installer that's zipped you might have gotten the windows version.

It's super easy to install. Just download the .run file from here:
[http://liflg.org/?catid=6&gameid=64](http://liflg.org/?catid=6&gameid=64)
Save it to your home directory.

Then open up a terminal and type 'sudo -s' to get a root shell.

You should still be sitting in your home directory. Just type 'sh nexuiz_1.1-english-2.run'. That'll run the installer. Use tab completion so you don't have to type the whole thing also.  :wink: 

Good luck.
