### Post by Donshyoku on 2006-04-02
Possibly the first game, at least the first major game, to utilize Cairo in a web browser.  Duke Nukem Forever (after about 10 years) has been completely rewritten to run in your web browser.  It claims to be OS-independent and browser-independent (as long as it has Cairo support such as FF 1.5, IE7, or Safari)!

[http://arstechnica.com/reviews/games/forever.ars](http://arstechnica.com/reviews/games/forever.ars)

They tested it in Ubuntu even!  Looks nice!
