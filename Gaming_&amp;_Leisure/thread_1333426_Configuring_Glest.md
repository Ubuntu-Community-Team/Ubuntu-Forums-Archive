### Post by josephmc on 2009-11-21
I actually figured this out while posting for help! I debugged the bash script that launches glest and found it was putting the config file in 

```
~/.config/glest/glest.ini
```

You need to run glest first to get it there. After that you can edit settings.

Most notable is the screen resolution settings. 

Hope this saves somebody the time it took me to figure this out!:)
