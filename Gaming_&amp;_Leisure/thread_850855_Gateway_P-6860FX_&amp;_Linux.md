### Post by justin whitaker on 2008-08-09
I have one of these. Fantastic laptop for gaming...for Linux? Not sure. 

Desktop Live CD runs well on this, but the disk does not configure the video card correctly, so you are stuck with 800x600 until you actually install Ubuntu and fiddle with it.

Mandriva does a little better...you can get 1048x768 out of the box, but again the drivers are not configured correctly, so back to fiddling around. 

Everything else works: wireless, bluetooth, cd drive, etc. 

Although, it has to be said: apparently Gateway did something weird with the video card, because people seem to be having issues with Nvidia's drivers on this laptop.
