### Post by Dannypb on 2015-10-19
Hi,

I created my first commercial game for Android using Unity.  It's an educational game called Space Bus.  I have created a tarball for a linux version and am offering it free to anyone who wants to play in on Linux.  I have tested it successfully Ubuntu 14 64 bit.  I also tried to run it on Centos 6 64 bit, but my glibc is too high to run it for some reason, and I don't really have the time to mess with it.  It has a 32 and 64 bit executable.  

The download page is here: [http://gamer8games.com/deploy/SpaceBus/](http://gamer8games.com/deploy/SpaceBus/) .

Instructions for running an x86 or x86_64 file can be found here: [http://askubuntu.com/questions/223228/how-do-i-run-a-game-with-a-x86-extension](http://askubuntu.com/questions/223228/how-do-i-run-a-game-with-a-x86-extension) .

I made my first commercial try as an educational game, so that if it stunk, I could say, "well, its just an educational game!"  I don't think it's too bad.  I could be biased though :)  The android version uses TTS (text to speech) to read the text while you blow stuff up, so it's easier to retain.  I didn't implement TTS for linux or the web player version.  

Anyway, just wanted to give back to the community.

Dan

oh yea,, the controls may be set for mobile.  When you start the game, go to settings and select keyboard / mouse.  Deselect the other options.
