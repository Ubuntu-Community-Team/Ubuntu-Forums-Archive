### Post by aidave on 2015-01-22
Howdy folks!

Many years ago I built a little helper app called [Kabikaboo]("http://kabikaboo.launchpad.net"), which had its ups and downs and still works just-good-enough for me not to do any development on it.  The latest version really needs to be in the repository but I'm not too great at packaging.  Anyways now I'm back into game development, and have made it a requirement that it all be done from within Ubuntu/Linux.  This is part of the toolchain I'm using:

[LIST]
[*][MOAI]("http://www.getmoai.com") - Lua/C++ engine
[*][ZeroBrane Studio]("http://studio.zerobrane.com") - Lua editor
[*][GIMP]("http://www.gimp.org") - Graphics
[*][Tiled]("http://www.mapeditor.org") - Map editor
[*]TexturePacker - Sprite sheet creator
[*]...and more!
[/LIST]
All of these work on Linux and almost all are opensource except TexturePacker!  So this game will eventually end up on Linux, Windows, OSX, and mobile - but the best part is I don't have to leave Ubuntu to do the coding!  We also plan to kick back 10% of our crowdfunds to the opensource tools we use and will opensource our game if we are successful enough.  Here is a dev article that talks a bit about this toolchain:
[LIST]
[*][How We (Painfully) Migrated our Art Assets into Sprite Sheets]("http://www.indiedb.com/games/dont-be-patchman/features/how-we-painfully-migrated-our-game-art-assets-into-sprite-sheets-for-speed")
[/LIST]
And more development updates and videos:
[LIST]
[*][http://www.indiedb.com/games/dont-be-patchman/news](http://www.indiedb.com/games/dont-be-patchman/news)
[/LIST]
