### Post by digitaldiamondbaseball on 2011-03-13
Just a heads up: There is a new baseball simulator available called Digital Diamond Baseball for those folks that enjoy baseball simulation games (e.g., Strat-O-Matic, Diamond Mind Baseball).  The game is written in Java and runs on ubuntu if the Java Runtime Edition 6 is installed.  

The game supports the following features:


[LIST]
[*]Import and play with players from any season in baseball history.
[*]Import and play with players from the current baseball season as it unfolds.
[*]Automatically generate season schedules, 5-game playoff schedules, or 7-game playoff schedules.
[*]Import the actual schedules used during the regular season.
[*]Import the exact lineups and starting pitchers that were used for each game during the regular season.
[*]Fast play a day, week, month, or the entire season.
[*]The ability to add your own custom play-by-play calls!
[*]Generate, view, and save HTML reports of the simulated statistics.  Available reports include standings, leaders boards, team statistics, and player statistics.
[*]View and save boxscores and play-by-play accounts during a game, or at anytime after the game is completed.
[*]Create and save custom lineups that you can use when you manage your favorite team, or that the computer manager will use during fast play.
[*]Customize the game by including your own ballpark images.
[*]Configure over 35 different options that control how games are simulated.
[*]Use the Matchup Screen during the game to find just the right pinch hitter or relief pitcher for the situation.
[*]Easily backup and restore libraries.
[*]Consult a comprehensive help system to learn how to take full advantage of the game.
[/LIST]


Lots of videos about the game can be found here:

  [http://www.youtube.com/user/DDBBAdmin](http://www.youtube.com/user/DDBBAdmin)

A free trial version is available that is fully functional except it only allows 3-inning games.  The full version is only $9.95.  Visit [http://digitaldiamondbaseball.com](http://digitaldiamondbaseball.com) for more information.

Enjoy!
