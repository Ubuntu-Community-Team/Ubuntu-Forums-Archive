### Post by flaugher on 2007-09-29
Celestia IS cool :)

The current copy in the repositories is old and the cmod files don't work (many moons and spaceships don't show up). Get the current version from the website (Google for it) and be sure to download the LINUX version. It comes down as a bin file. Right-click, make it an executable file under properties ... permissions ... then run it. Couldn't be easier. It installed fine for me under Ubuntu running on a Dell D620 and all the planets work.
