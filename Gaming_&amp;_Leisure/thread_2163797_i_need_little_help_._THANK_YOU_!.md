### Post by regor210 on 2013-07-19
You could try “Play on Linux” they have a warcraft III :Reign of Chaos and The Frozon Throne installer.

my experience with Play on Linux has been it works about half the time.

Play on Linux works by downloading a version of Wine that can be made to work with your game or app . Then they modify that version of Wine as needed to make it work. This is all done inside Play on Linux so the Wine you use for everything else is not used. 

Where I run into trouble is not all old game installers are the same so what works for version v.34 will not work with v.98.

But worth a try.

I would check in the Ubuntu software center 1st If its not there you can find it here

[http://www.playonlinux.com/en/](http://www.playonlinux.com/en/)
