Launch the application from a terminal and paste the error output here please.

As for a bios, this is not a bios for your computer, but a bios image made into a file.

For the playstation it will be: SCPH****.BIN

Where the **** is the four digit code based on the time and location the ps model was released.

These are not legal unless you own a ps, and then they're only technically legal.  Do not ask about them here again.

You can find them easily if you try.
