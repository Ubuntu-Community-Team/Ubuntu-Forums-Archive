### Post by karth on 2008-04-11
RunUO is really great and stable, as far as I recall, for having used it for more than 2 years now (I dropped maintaining a shard a few months ago).

It supports all clients up to 6.0.x but not UO:KR (the new client with a fancy 3D engine).
It may require a few skills in object oriented programming (it's C# based, yeah I know, M$ and all...\\:D/) to modify/add scripts, but has a comprehensive reference (but still, the main classes are hardcoded if you download the binaries).

RunUO kind of requires a Windows host to run (.NET technology), but there's a Mono port called SunUO that can run on linux boxes. Still, the two communities are "at war", so don't expect any SunUO support on the RunUO boards (I don't know anything about the reason, though, but that's not the point).
SunUO seems to be completely compatible with the RunUO scripts (as far as I know).
