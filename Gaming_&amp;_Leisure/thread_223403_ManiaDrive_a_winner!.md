### Post by Mickeysofine1972 on 2006-07-26
Hi 

Do you long to rattle around the kind of road systems that only the british government could design?

Do you dream of crashing a Clio without breaking the eggs in the boot?

Then ManiaDrive is for you!

This kewl game isn't new but has some very challenging levels and a good old fashioned 'Fun' theme to boot!
> Yes thats right kids! its FUN!


Download it now from:
[http://maniadrive.raydium.org](http://maniadrive.raydium.org)

It unzips into a folder with the binaries already compiled and working. I have run it on Kubuntu 6.06 and it runns fine!

You can run it by running the .sh scripts in the main folder or you can create an icon to point to the 'maniadrive.static' binary in the game folder, whilst remebering to set the working path to that folder too!

Mike
