### Post by almostlinux on 2009-08-21
The XP on servers are retained using your etkey. Save it somewhere else before you format and replace it when you reinstall the game.
 
It's here: 
 
~/.etwolf/etmain/etkey
 
where ~/ is the location of home of the user you usually run the game as.
