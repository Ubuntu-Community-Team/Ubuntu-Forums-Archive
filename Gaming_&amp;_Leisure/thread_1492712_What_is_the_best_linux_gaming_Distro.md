### Post by holastickboy on 2012-12-12
Indeed a way to resurrect an old thread! That being said, a lot has changed since this has been started.  Benchmarks show that KDE 4.9 is the best window manager to use when gaming on both AMD and Nvidia graphics cards. You can install that on any Ubuntu based distro with apt-get install plasma-desktop or apt-get install kde-full

On Amd, the benchmarks show that there is not much difference in performance.  With Nvidia, KDE is by far the best to game with.

Source:
[http://www.phoronix.com/scan.php?page=article&item=ubuntu_1210beta_desktops&num=1](http://www.phoronix.com/scan.php?page=article&item=ubuntu_1210beta_desktops&num=1)

Note: while these benchmarks are based on a slightly older version of unity, tests have shown that the newer version of Unity have not improved performance, and therefor is still relevant.  Gnome 3 does pretty well, but if you are really looking for 100% performance, drop it for KDE (just dont forget to turn off full screen effects in control panel).
