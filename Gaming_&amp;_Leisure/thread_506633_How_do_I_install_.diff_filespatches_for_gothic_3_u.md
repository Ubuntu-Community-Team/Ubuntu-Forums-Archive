### Post by hikaricore on 2007-07-21
I can't check the application database for wine at the moment as it is giving me errors, but I'm pretty sure the .diff files are to be applied to the code of WINE itself.

Meaning you will need to download the WINE source code, apply the patch, then compile and install WINE from your patched source.
You may want to read about patching here (or search the forums): [http://en.wikipedia.org/wiki/Patch_(Unix)](http://en.wikipedia.org/wiki/Patch_(Unix))

I don't often use patch files personally so I don't want to give you the wrong info
