### Post by Perfect Storm on 2006-03-18
.sh is a script file and is not distro specific it, it can run (hopefully) on every linux distros.

```
sudo sh <paths>/<name of the file>.sh
```

eg.
sudo sh /media/cdrom/ut2004.sh
