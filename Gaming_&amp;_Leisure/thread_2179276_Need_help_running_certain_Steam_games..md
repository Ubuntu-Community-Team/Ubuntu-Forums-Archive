### Post by Ikesurmarth on 2013-10-07
Hello, guys, I have a problem on Steam. Whenever I try to run a game utilizing the [Source Engine]("http://source.valvesoftware.com/"), such as Half-Life 2, Team Fortress 2, or Portal, I get an error stating that I need some OpenGL related thing called S3TC. After seeing [this Ask Ubuntu post]("http://askubuntu.com/questions/231176/how-do-i-install-s3tc"), I ran this command in the terminal before running Steam:

```
[FONT=Ubuntu Mono]force_s3tc_enable=true steam[/FONT]
```

The games COULD run, although they were slow and had a greenish tint to them. Back when I had Ubuntu 13.04 (unity), they ran perfectly at decent settings. Here are my hardware specs:

OS: Linux Xubuntu 12.04.3 x64
Graphics: Intel HD Graphics 3000
RAM: 4GB
CPU: (2nd Gen) Intel Core i3 @ 2.30 GHz
