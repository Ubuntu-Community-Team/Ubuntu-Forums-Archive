### Post by efflandt on 2015-06-05
Steam games that are compatible with Linux (or can run in wine) that use the Steam cloud and item servers (typically multi-player games) will have your achievements and items regardless of which OS you run them on. For example when the Linux partition at the far end of my drive started failing, I temporarily run the games in Win7 until I got a new drive. So everything was there after installing Steam and the games in Win7 and when later switching back to Linux.

Single player games or games under development may only store status of your current game locally, so if you want to play those on a different system, you would need to figure out where those game saves are and either copy them to the other system or figure out some way to share them between systems.

I am not familiar with EzCast Pro, but the $35US Chromecast device works with Google Chrome in Ubuntu or certain apps on Android tablets/phones, like Hulu, Netflix, YouTube, etc. (and possibly Google Chrome beta in new enough Android version) and probably Apple devices too (I have not owned an Apple device since my Apple ][+ in early 1980's). Note that Chromecast likely works best using its included HDMI extender to get a stronger WiFi signal away from the HDTV. Chromecast is actually better than some devices that mirror because your computer or phone acts as a control to have streaming Internet video go directly to the Chromecast device for less WiFi bandwidth than relaying it though your phone, etc.

### Post by oldrocker99 on 2015-06-08
Personally, I don't get paid to use Photoshop (I use GIMP, which does everything **I** need in a graphics processor), or MS Office (when LibreOffice does everything I need from an office suite), or even play games. 

Steam for Linux has over 1000 Linux games, including several I had bought when I was dual-booting: The Witcher 2, CIV V, X-COM:Enemy Unknown, Borderlands 2, Half-Life 1 and its expansions, the Orange Box, Empire:Total War, Crusader Kings 2, Europa Universalis IV, and more. If you have bought a Windows Steam game, and install Steam for Linux, the Windows games which do have Linux versions will already be visible in your library and available for installation, and the Steam Cloud Save will work just the same. Because of Steam's commitment to Linux (SteamOS itself is, like Ubuntu, based on Debian), Linux gamers can expect more AAA games to be released as time goes on, and a number of upcoming games will be coming out for Linux, Mac and Windows. When Europa Universalis IV came out last summer, it was out for Linux, Mac and Windows the day of release, as were the Kickstarted Wasteland 2, Pillars of Eternity, and the upcoming Torment:Tides of Numenara. This will become more common as the months pass.

I do use Wine to play a few older games GOG games, to install Steam for Windows so I can play Skyrim and Oblivion (which both run great), and to use QuickPar, which has features that the Linux PyPar doesn't have.

For streaming media, there is the cross-platform PLEX Media Server, which is in the Ubuntu repositories, and works with Roku, which also has the app, which other streaming players most likely have as well.

There is almost always a Linux program with similar features to a Windows program you use; you may need to learn how to effectively use that Linux program, but, after all, you did have to learn how to use that Windows program, too.

Welcome to the world of Linux, which, you will find, is not a cheap Windows replacement for geeks, but is an **upgrade** from Windows. Have fun!
