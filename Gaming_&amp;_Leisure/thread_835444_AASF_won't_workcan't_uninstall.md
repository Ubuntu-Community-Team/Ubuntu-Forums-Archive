### Post by Pathfinder_ on 2008-06-22
If it can't find libstdc++.so.5 that probably means you don't have it installed. Start up Synaptic Package manager and search for libstdc and install it. I think you'll need the 32bit version but I'm not 100% sure.

### Post by Cappy on 2008-06-22
Use
```
sudo apt-get install libstdc++5
```

If you are using 64-bit Ubuntu Hardy use
```
sudo apt-get install ia32-libs
```

You can find out what packages contain a library or file using [http://packages.ubuntu.com/](http://packages.ubuntu.com/)
