### Post by geppy on 2005-10-24
All of you guys complaining about how id couldn't get ALSA working really need to think about that.  Do you think they'd release a client that sucked as much as that?

No.

It's your own damned fault for using an OS that enables dmix.  dmix is like ESD for ALSA, and it probably sucks a fair bit more than ESD.  If you want to fix the problem, either use OSS, or edit your /etc/asound.conf or ~/.asoundrc so that you don't have to put up with dmix any more.

Then file a bug report and tell the Ubuntu guys to use JACK + oss2jack + ALSA2JACK. ;)
