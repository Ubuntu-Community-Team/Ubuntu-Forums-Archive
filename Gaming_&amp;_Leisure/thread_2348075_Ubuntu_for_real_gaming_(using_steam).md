### Post by oldrocker99 on 2017-01-01
Welcome to the forums!

There are well over 2000 Linux games on Steam, from indie to AAA. The Windows games you want to play **may** be playable on Linux using Wine. CIV 6 is coming out for Linux. There are dozens and dozens of superior FPS games: Borderlands 2, Bioshock Infinite, Painkiller, Metro, Serious Sam, and many others. Mafia III has a Gold rating on the Wine database:[https://appdb.winehq.org/objectManager.php?sClass=application&iId=11920](https://appdb.winehq.org/objectManager.php?sClass=application&iId=11920), and Doom does too:[https://appdb.winehq.org/objectManager.php?sClass=version&iId=33711](https://appdb.winehq.org/objectManager.php?sClass=version&iId=33711).

You *may* want to dual-boot to play Windows games natively; I used to, but I'm a 100% Linux gamer now, only playing Windows games via Steam for Windows installed with PlayOnLinux, and playing 90+% of my games using Steam for Linux.

### Post by schmied on 2017-01-25
Any game that doesn't require DirectX 11/12 has a good chance of being playable through Wine.
For instance, Doom 2016 uses OpenGL and Vulkan, which are both supported by Linux, so it's no surprise it runs well in Wine. id Software specifically always supported Linux gaming, but then Bethesda came along and... well, didyouknowwhat.

I find it sad that developers aren't moving over to open source technologies (such as OpenGL or Vulkan), especially since DirectX 12 runs so well and doesn't make your games slower than 11 whatsoever (sarcasm).
