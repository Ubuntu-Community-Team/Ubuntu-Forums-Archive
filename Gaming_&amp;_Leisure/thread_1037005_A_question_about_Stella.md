### Post by BigSilly on 2009-01-11
Just right click on the start menu, and select "Edit Menus". It's pretty self explanatory from there on. You'll probably find that once you enter the command, the Gnome menu will select the icon automatically. If not, you should be able to easily locate it in usr/share/pixmaps. Sometimes some apps just don't add a menu entry, but it's always a doddle to add one yourself.

Good luck!
