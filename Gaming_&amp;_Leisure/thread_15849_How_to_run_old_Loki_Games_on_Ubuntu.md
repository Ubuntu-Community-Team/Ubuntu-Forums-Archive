### Post by Uuranor on 2005-02-17
I hope this can help someone ... It is the result of some google search ^__^

[list=1]
[*]Download the last patch from *[ftp://sunsite.dk/mirrors/lokigames/updates/](ftp://sunsite.dk/mirrors/lokigames/updates/)*
[*]Execute it with *bash patchname.run* as root
[*]On command line write *export LD_ASSUME_KERNEL=2.2.5*
[*]Launch the game ...
[*]Enjoy!
[/list] 

I've tried it with Simcity 3000 and it works!!
If it doesn't work, try with LD_ASSUME_KERNEL=2.2.4.
I used hoary and kernel 2.6.10, but I think it works with warty.
