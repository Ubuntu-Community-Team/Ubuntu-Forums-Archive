### Post by Ranko Kohime on 2014-04-06
There is what oldrocker99 suggested, which is fine for the basic social stuff.

As far as having a full Steam install on say, a USB key, that's slightly more involved.  Downloading and extracting the .deb file from Steam should yield the basic executable that brings up Steam, and you might then have to link ~/.local/share/Steam to the USB key's appropriate directory.  It can be done, but no mistake: it is a bit of work.
