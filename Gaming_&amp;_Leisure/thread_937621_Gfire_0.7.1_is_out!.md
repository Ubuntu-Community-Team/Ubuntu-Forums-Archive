### Post by chewit on 2008-10-04
Gfire 0.7.1 is now released, this will continue the support of the 0.7.x series. It is an important release, which fixes and improves 0.7.0.

New Features:
- 'About' window expanded
- Improved the games list downloader, it now deletes the old games list once the new one is downloaded.

Bug Fixes:
- html escape function expand (improved stability)
- Fixed a memory leak (performance increase)

Games List version: 1 October 2008

You can download the deb, rpm or source file from [here]("http://gfire.site40.net/?page_id=24")
