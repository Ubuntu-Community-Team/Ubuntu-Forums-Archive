### Post by brian-mccumber on 2015-10-20
I posted my machine specs above it's nothing special but it plays the games I like so that's good enough for me.

You can play Borderlands coop and Dead Island is also coop and you can can play those two games with people running Windows too. Also Minecraft has multiplayer functionality and there are a million MC servers that can be played on if you don't want to go through the hassle of setting up a server. This is kind of biased because I am only listing multiplayer games that I actually play here. 

There are a lot of good titles at Steam and there are more added regularly. I don't use Wine, when I stopped using Windows I stopped using it and I imagine my machine would have a problem running Wine and the game at the same time so I just stick with good games I can find for Linux. I do have a bunch of good titles like Diablo 2, Fallout 3, and Oblivion on disks but they are only for Windows so I don't play them anymore. I think as the game devs start to realize that there are alot of people running Linux versions who like to play games the support for games that play on Linux will begin to grow and so will their profit margin, so I imagine there will be many more triple A titles available for Linux in the future.

If you can't tell I am a huge fan of open-world sandbox type games. I used to play the linear story line driven games but after I got a taste of open world goodness it's hard to go back to linear games.
