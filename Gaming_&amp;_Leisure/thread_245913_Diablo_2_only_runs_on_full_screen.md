### Post by User_Program on 2006-08-28
In your wine config it should be in /home/YOURUSER/.wine  there is an option in there called "DXGrab" = "Y"  change the "Y" to "N"   
There could be a command line option but I'm not sure what it is.

Running Diablo this way will make the mouse go crazy and you will prob get more mad with playing this way... at least I did.    So I just stuck to full screen, The only time I changed it was to take a screenshot.
