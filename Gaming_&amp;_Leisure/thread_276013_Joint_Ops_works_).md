### Post by Jimmy_r on 2006-10-12
I cannot believe this!
I have been trying to get this game working under wine a few months ago, completely fruitless.
A while ago i tried it on the demo version of cedega, but it was unplayable.
I was reading a while ago that wine has improved much recently, so on a fresh install of edgy i installed wine.
Just for fun i installed joint ops:typhoon rising, my favorite game of all times, without any real hopes of it working.

Well, first it did go as expected. Not even the installer worked, just a black screen.
I choose the install directX option, then tried again. The installer worked great this time.
So i tried to start the game. It gave me a lot of "Caps support for directx9 is nonexistent at the moment!"
I almost gave up, but went into winecfg and choose windows xp instead of win 2000.
This time it played the intro, then shoved the menu.
So i choose a training mission and it started! 
Ok, some textures was crap and no sound, but it actually was playable and i have not tried any tweaking yet.

I just wanted to let you know, if there is anyone else who had given up on this game on wine earlier.

Wow, the Wine developers rocks!

I have not even tried if the multiplayer mode works, but i did read somewhere that punk buster did not get along very well with wine yet.

Any input on this? Dare i connect to any servers as of now, or will punkbuster think that wine is some kind of cheatware? I would hate to get banned now when it is finally working :(
