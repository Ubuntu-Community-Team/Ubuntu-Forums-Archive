Welcome to the Linux gamer's club !  I tend to agree there is probably no need for a howto, as installing these games is quite easy.  Of course there are the usual snags that some people hit, but nothing that you can't figure out in a couple of minutes. For example, one general pitfall with many games installers is that at the end of instalation it offers if you want to start the game ( a la windows style installers), and usually you say yes, right ?  Wrong!!!! the installer runs as root, but you don't want to run the game as root as this causes lots of problems (crashes, etc). So the tip here is to tell the installer to ext, and afterwards run the game normally from the menu or command line. My point is that these sorts of things don't require a howto.

UT2004 is my favorite multiplayer game, and it runs great on my 6800 ultra, at 1280x1024 and all detail settings maxed out ! Rock on
