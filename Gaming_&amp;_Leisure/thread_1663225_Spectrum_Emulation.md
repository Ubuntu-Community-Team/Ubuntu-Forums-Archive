### Post by lykeion on 2011-01-09
* Java Spectrum Emulator (play Elite online with this emulator)
URL: [http://www.twinbee.org/hob/play.php?snap=elite](http://www.twinbee.org/hob/play.php?snap=elite)

* Spectrum Arcade (online ZX81 game emulator)
URL: [http://nologinrequired.no-ip.org/SpectrumArcade.cls](http://nologinrequired.no-ip.org/SpectrumArcade.cls)

* Spectemu (ZX81 emulator, haven't tried it)
Ubuntu packages: spectemu-x11
Sourceforge page: [http://sourceforge.net/projects/spectemu](http://sourceforge.net/projects/spectemu)

* World of Spectrum (list of emulators)
URL: [http://www.worldofspectrum.org/emulators.html#unix](http://www.worldofspectrum.org/emulators.html#unix)

Also try to search this forum with keywords like "spectrum" "emulator", there are a bunch of threads which can be helpful.
