### Post by EuclideanCoffee on 2020-02-18
I don't use or recommend ROMs, but the information you are looking for can be described in this article.

[https://www.addictivetips.com/ubuntu-linux-tips/play-arcade-games-on-linux-using-mame/](https://www.addictivetips.com/ubuntu-linux-tips/play-arcade-games-on-linux-using-mame/)

Simply, you move files to 

/usr/local/share/games/mame/roms

And to start the program, you write the following command in a terminal:

$ mame
