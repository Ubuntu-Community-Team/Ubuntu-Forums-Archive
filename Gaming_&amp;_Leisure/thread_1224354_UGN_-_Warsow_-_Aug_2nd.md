For new Players

In order to play this game you need to have warsow installed. On a default 9.04 system, the repo version is adequate. 

  * sudo aptitude install warsow warsow-data

The last step is start Warsow by going to your Main Menu > Games Menu > Warsow.

For users who have never played the game, Warsow is an Ego shooter. There is significant jumping, dashing, Wall jumping, double tapping, and jump moving that is not all that unlike ut2003/ut2004 styled DM play. Those who can and do experience motion sickness, this probably isnt the game for you.
