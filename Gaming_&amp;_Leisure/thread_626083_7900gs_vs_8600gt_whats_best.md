This is generally too little memory, or bad memory imposing a cap.

As a test, try to boot from the LiveCD and install nvidia-glx-new and enable it.  Then ctl-alt-bksp to restart the GUI and it should work.  If it does not your card is defective, or you have a system conflict.  Start cleaning up IRQs.
