### Post by fasteddy86 on 2008-10-31
You can't run the MFH mod from FoF directly. MFH is a standalone mod. Note that you don't have to put the MFH files in your FoF folder, either. Put it wherever you want. To run it, run the FretsOnFire file (the one without an extension) in your MFH folder. To make it easier, I put a shortcut to it on my kicker (I use KDE).
