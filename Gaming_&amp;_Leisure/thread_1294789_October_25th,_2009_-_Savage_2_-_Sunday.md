Gamers,

The **[SIZE="2"]#UGN[/SIZE]** (Ubuntu Game Night) game of the week is [SIZE="2"]**Savage 2**[/SIZE] by S2 Games. 

Savage 2 ([http://www.savage2.com](http://www.savage2.com)) is a team based multiplayer only hybrid of FPS, lite RPG, and RTS.

Every UGN event is on [SIZE="2"]Sunday[/SIZE] at [SIZE="2"]2 PM[/SIZE] EST (1800 GMT). The length of the event varies between 2-4 hours.

We will enjoy collecting your soul.

----------------------------------------

_For new Players_

In order to play this game you must first create a free account here:

[LIST]
[*][http://www.savage2.com/en/create_account.php](http://www.savage2.com/en/create_account.php)
[/LIST]

Once you have an account the next step is to download the client here:

[LIST]
[*][http://www.savage2.com/en/download.php](http://www.savage2.com/en/download.php)
[/LIST]

After you have the client downloaded you must run the installer. Open a terminal and navigate to the file you downloaded and then run the command below.

[LIST]
[*]sudo ./Savage2Install-2.1.0.1-i686.bin
[/LIST]

The last step is to start Savage 2 by going to your **Ubuntu Logo Menu > Games Menu > Savage 2.**


----------------------------------------

For more information about UGN or the event please visit [http://gwos.org/ugn/](http://gwos.org/ugn/). We also have an IRC chatroom that you can access here [http://java.freenode.net//index.php?channel=UGN](http://java.freenode.net//index.php?channel=UGN)
