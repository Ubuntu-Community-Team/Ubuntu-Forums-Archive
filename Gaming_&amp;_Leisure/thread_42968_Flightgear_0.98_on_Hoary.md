### Post by tristan on 2005-06-20
I'm not sure if Flightgear classifies as a game or not, but I've wasted lots of time with it so it's definitely not productivity software. An awesome piece of software!

The problem is that there's only hoary synaptic access to Flightgear 0.95, and a lot of improvements have been made in 0.98. 

I just followed the download links from the flightgear page and grabbed myself the slackware tgz for flightgear 0.98. I converted this to a deb using alien (I love that piece of software too), and installed it with dpkg.

I then started the whole thing with the command **fgrun** which brings up a nice gui for all the settings. Make sure you enter the correct path for the executable **/usr/bin/fgfs** and FG_ROOT ** /usr/share/FlightGear**.

Then, just pick a plane and start flying!
