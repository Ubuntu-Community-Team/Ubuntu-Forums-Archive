Gravity Games is pleased to announce the third public release of Galactic Vice:  Alpha 2 Pre-Release B.  Galactic Vice is a *free*,  real-time strategy game for both Windows and Linux that features two  races at war.  Vital mistakes made by a corrupt human empire lead to a  catastrophic war against an alien race; one in which humanity is pushed  to the brink of extinction.

Galactic Vice is under development and this release features new terrain, a map editor, new music, and better graphics! (that's *really *watered down, but you get the point)

### Post by Conoktra on 2011-09-14
In firefox, on the link Right Click -> Save Link As.  Choose where you want to save it (GvWebInstaller_linux).  Once it is saved you have to do this in a terminal: "chmod +x /path_to_download_folder/GvWebInstaller_linux".  Now you can run it in the terminal or even just double click on it.
