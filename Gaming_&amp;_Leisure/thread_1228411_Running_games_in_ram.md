Run in RAM? everything runs in RAM? RAM is ment just for that, running stuff from it for a speed gain.

### Post by del_diablo on 2009-08-01
Games never load enough of their data into the ram. They load a few models and similar, but never a decent amount. Ex the entire gameworld or all shaders or all models, etc. And games also usually lack the option to actually load the entire game into the ram on loading.
Loading an entire game into ram however would result in about no loading times and quite the speedup on it.

The main reason you want to do this in a game is because i think it would eliminate loading times, and it would make those 4 unused extra gigs of ram any use :P
Well, lets say you play a game that loads all the time. Well then loading it into ram would be golden. If a game however only uses a little load time at startup, then its not really worth it :P
