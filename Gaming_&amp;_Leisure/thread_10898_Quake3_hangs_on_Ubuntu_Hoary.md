### Post by Braese on 2005-01-12
try this in root console:
echo "quake3.x86 0 0 disable" > /proc/asound/card0/pcm0c/oss
echo "quake3.x86 0 0 direct" > /proc/asound/card0/pcm0p/oss
echo "quake3.x86 0 0 disable" > /proc/asound/card0/pcm1c/oss

after this, start quake3 in normal console, if worked, you can copy this in an bootscript
hope it works for you, in ET it doesnt worked for me

### Post by Ranime on 2005-05-10
Thanx for that!! I was trying for days to get Q3 running!!

I made a bash script to start Quake 3 with an called it *q3.sh*
You can make it with your favorite editor like *gedit*:

```
esdctl off
sleep 2
quake3
esdctl on
``` 

Save the script in your home directory, and *chmod* it to 775 from a terminal.
run the script from a terminal like this:

```
./q3.sh
``` 

And enjoy!
