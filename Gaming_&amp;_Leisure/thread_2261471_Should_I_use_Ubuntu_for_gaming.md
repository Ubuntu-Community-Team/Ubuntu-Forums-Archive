### Post by oldrocker99 on 2015-01-19
I have been gaming on Linux since 2008. A very few games at first, besides Neverwinter Nights, but, after the Humble Bundles, Desura, and Steam, there are so many great games now that I can be picky about which ones I buy, instead of buying every Linux game I saw. That always didn't work out; some pretty awful Linux games were released, along with some swell ones.

By the way, all the advice you'll see on Linux forums about nVidia and ATI are true. nVidia cards give far more usefulness for Linux gaming. The basic difference is that the closed-source, proprietary nVidia drivers are pretty great, while the open-source noveau drivers aren't so good. ATI's proprietary fglrx drivers are notoriously bad, and the open-source radeon drivers are pretty good. Intel drivers are all open-source directly from Intel, and work pretty well.


I, after dual-booting with Windows (strictly for games that wouldn't run with wine) for two years, deep-sixed that annoying, porous, buggy, vulnerable OS and gained a lot of HD space. I'm not looking back, either. In just the last six months, there have been several AAA games released: The Witcher 2, CIV V, X-COM, Europa Universalis IV, Empire:Total War, Euro Truck Simulator 2, Planetary Annihilation, Borderlands 2, Civilization:Beyond Earth, Cities in Motion, Tropico 5, and more. Who needs Windows for gaming?

Not I!
