[http://www.ubuntuforums.org/showthread.php?t=372164&highlight=ladbrokeshttp://www.ubuntuforums.org/showthread.php?t=372164&highlight=ladbrokes](http://www.ubuntuforums.org/showthread.php?t=372164&highlight=ladbrokeshttp://www.ubuntuforums.org/showthread.php?t=372164&highlight=ladbrokes)

I finally got **32RedPoker** working in IES4LINUX if I use the INSTANT PLAY/NO DOWNLOAD version. So for now I've given up trying to make the "downloaded version" work. I'm still interested in suggestions to make it work though.

_Getting Microgaming "instant play" version to work:_

I installed a couple of plug-ins from: [http://www.tatanka.com.br/ies4linux/page/Plugins](http://www.tatanka.com.br/ies4linux/page/Plugins)

Adobe Shockwave player
Adobe Authorware player

Now I can play 32RedPoker's "Instant play" version with no errors at all. This should work with other Microgaming sites as well because they use the same software.

I still can't get **Ladbrokes** "instant play" to work in IES4LINUX though.  When trying to load a table it never finishes the loading, but **Ladbrokes** "download version" works 95% with Wine. It only has a few graphics errors in the top of the client.
