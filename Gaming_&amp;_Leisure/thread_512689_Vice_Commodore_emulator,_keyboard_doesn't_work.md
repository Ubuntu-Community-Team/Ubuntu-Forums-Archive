### Post by Robin T Cox on 2007-07-30
The keyboard will work, but it seems not for all games. For example, I can't get it to work for Impossible Mission. To set it up, there is a control on the menu that comes up when you hold down your right mouse button. 

The available keyboard is via your numeric keypad. To use it, your Num Lock needs to be On. Use the number keys plus the 0 key on your numeric keypad.

See the VICE Manual for more info.

[http://www.viceteam.org/vice_2.html#SEC11]("http://www.viceteam.org/vice_2.html#SEC11")
