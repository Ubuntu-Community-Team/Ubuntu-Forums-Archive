### Post by smack on 2005-09-27
I launch mine like this. You can paste that in to a custom application launcher also. Right click a menu bar and click 'Add to Panel'. It's at the top. It'd be too annoying to have to type the launch command in a terminal each time you want to play.

cedega "/home/smack/.transgaming/c_drive/Program Files/Warcraft III/war3.exe"
You'd need to change the home directory to your home directory also.

I'd use the CD instead of a crack also. They come out with new war3 version all the time and finding the latest crack is always a super-pita.

Also, about it not finding the CD. Try enabling or disabling your DMA and rebooting. When I first turned on DMA on my CDROM war3 wouldn't find the CD until I rebooted(don't know why).
