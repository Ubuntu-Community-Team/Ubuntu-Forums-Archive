### Post by hikaricore on 2006-12-13
SVGALib is very outdated video support for X in linux.

Try a newer emulator such as: FCEU

Make sure you have universe and multiverse enabled (you may need to search for more info on this as I don't have time to type it all out) in your sources.list and run from a terminal:

```
sudo apt-get install fceu
```

Punkrockguy318 has also release a gui frontend for it here:

[http://www.ubuntuforums.org/showthread.php?t=195413&highlight=nes](http://www.ubuntuforums.org/showthread.php?t=195413&highlight=nes)
