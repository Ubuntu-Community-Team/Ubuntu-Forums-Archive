### Post by eragon100 on 2008-08-27
Are you still running NWN via wine? the installation guide for the linux version of NWN platinum (with the windows cdrom) can be found here: [nwn.bioware.com/downloads/linuxclient.html]("nwn.bioware.com/downloads/linuxclient.html")

Have fun :wink:

The other question:

Right click on "applications" and select "Edit Menus"

Go to applications -> wine -> Uninstall Wine Software and select the game you uninstalled. If you remove it there, the menu entry should be gone as well.
