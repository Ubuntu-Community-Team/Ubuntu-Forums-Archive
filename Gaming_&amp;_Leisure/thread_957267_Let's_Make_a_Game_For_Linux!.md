### Post by Abras on 2008-10-24
Wow, you obviously have some talent there. I'm sure some more graphics/sound-oriented people could come up with some criticisms, but I'd say you could join just about any open-source project of your choice. Getting someone to program your idea may be a bit more difficult (since people --programmers particularly-- always prefer to work on their own ideas), but still entirely possible. I would definitely suggest visiting [Free Ganer]("http://freegamer.blogspot.com/"), which is pretty much the epicenter of open-source game development. And eve if you don't find any willing participants there I'm sure the people there will be happy to tell you where you can.

I also have a few tips about finding participants. 
1. Show off as much of the art and music you already have done. People are more likely to join a project that already has quite a bit of work put into it.
2. Write up some kind of design document, which you can use to flesh out how the game will work when finish. It makes the project run more smoothly and also encourages participants. I'd be willing to help you with that if you wanted. 
3. Focus on developer-oriented websites. This particular forum has a lot more game players than developers.

OK hope that helps,
Abras

PS Have you ever contributed to any game projects before? Just curious...
