### Post by forrestcupp on 2006-08-14
I kept trying to install mohaa with the loki installer.  The install went well, but the game never worked.  I kept getting an error about that the opengl subsystem couldn't be found.  I saw that this is a common error, and I tried all of the solutions such as renaming the /usr/bin/tls directory to no avail.  I knew I had the latest nvidia drivers installed properly, and I couldn't find any other solutions.  I installed the dev packages for opengl and openal, and now mohaa works like a charm.  I don't know if that is what did it, but I haven't done anything else that would change anything.
