### Post by RMuscaritolo on 2010-11-17
From what I understand, it really is a technical issue first, than a marketing decision second...

The technical issue is really about support. Linux greatest strength, is also its greatest weakness here. *Flexibility and Diversity* How, as a company, do you offer support and compatibility with a system that is ever-changing and very commonly modified? Not to mention the differences between Distros in general. 

This all leads to the marketing problem. There just aren't enough gamers on Linux for anyone to really want to tackle these technical problems... which is funny, because a lot more people would absolutely try Linux if it had a good gaming platform... so it's sort of a chicken before the egg bit, lol.

I really think the entertainment component is missing from the Linux desktop mission. Once big gaming companies finally start to push into the Linux territory, it may very well take off and become a real alternative for most people... I can only HOPE!

BTW, there are already a few programs that you can use to play Windows games on Linux. If WINE is a little too difficult for you, take a look at "Crossover games". Still a lot of bugs, but it's worth a try =)
