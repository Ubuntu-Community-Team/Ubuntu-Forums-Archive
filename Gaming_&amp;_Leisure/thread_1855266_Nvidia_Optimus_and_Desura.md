### Post by dsnettleton on 2011-10-05
I'm not sure whether anyone else has this problem, but I have a laptop with an Nvidia card that uses Optimus (which is not supported under Linux). Right now, the only way to manage this is with an Open Source program called Bumblebee. I highly recommend the program. It's a godsend. But when I want to run anything on my Nvidia card (everything else defaults to the integrated graphics on my processor), I have to use the command "optirun" followed by the command to run the program.

When I use the Linux beta for Desura, games are launched automatically from the Desura shell when I choose to run them. Does anyone know if there is a way to create custom commands allowing me to "optirun" my games through Desura?

[Edit]
That, and users here can benefit from what I learn. I found the option myself. Just right-click a game in your list, then select "Properties." Type in "optirun" in the box marked "User Launch Options." Easy as pie. I wish I'd found it sooner.
