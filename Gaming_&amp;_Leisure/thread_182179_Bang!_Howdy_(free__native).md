### Post by polpak on 2006-06-18
Enable the multiverse repositories [https://wiki.ubuntu.com/Repositories/Ubuntu](https://wiki.ubuntu.com/Repositories/Ubuntu)
Install Sun Java ([http://wiki.ubuntu.com/Java](http://wiki.ubuntu.com/Java))

Then set sun java as the default with:
sudo update-alternatives --config java

It should work fine after that.

As for the game it self. It's very fun, though not nearly as "social" as their original game Puzzle Pirates.
Very stratigic though.. Pretty fun all in all.
