### Post by Bölvaður on 2010-05-08
for me it doesnt sound like a driver issue, but to be sure try some other 3d game which you** already have installed**.

Things to try:

when ingame press: ctrl+alt+enter, go into the settings for video and change to some other resolution than 800×600

if ctrl+alt+enter did not do anything you can accomplish the same by the following:
find the file ~/.warsow-0.5/basewsw/config.cfg (this is a hidden folder inside your user's home directory)

find and set

seta r_mode "9"

and check if it works

if it worked then go back into the file and find and replace this one also
setau name "^9I ^1<3 ^9U"
