### Post by Carpentr on 2012-06-06
Hey everyone,

With all the excitement about the Humble Indie Bundle going on, I thought I would make a post regarding a different matter. Bryan Lunduke (one of the hosts on the Linux Action Show) has recieved enough monthly contributions to release all of his software as open-source. 

Several (formerly proprietary) pieces of software have been released. Linux Tycoon, and 2299: The Game have been released. BLABA (a game creation tool), and Illumination (another software creation tool) have also been released as open-source. I purchased Illumination before it went open-source and it was interesting to play around with at the very least. 

Linux Tycoon is also fun to play, too. You build your own Linux Distro day by day.

You guys can check out the details here:
*[http://lunduke.com/?p=3448](http://lunduke.com/?p=3448)*

Hopefully a few folks find this interesting!
