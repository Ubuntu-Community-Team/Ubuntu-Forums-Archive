### Post by tastygrue on 2010-11-26
Ok, so what I have done to get Lincity-ng to work on my netbook is to open a terminal, which you can find through Applications --> Accessories --> Terminal, and type in
```
lincity-ng -f -S 1024x600
```Afterwards, you can just open up lincity-ng from the menu and it should work in fullscreen.  But from my experience, lincity-ng has been slow on my netbook, so I use the simpler lincity, which although is older, uses a lot less graphics, and is subsequently faster.  I hope this helps to solve your problem.
