I decided i would write this guide for everyone cause i looked around alot, granted im new to ubuntu but i couldn't figure it out for the longest time, i ran across EEEpc guide /w Montana linux and kinda combined this with ubuntu stuff i read to get it too work. Hope this helps some people out!;)

Whats Needed:
Ioquake3 - Engine Download for Linux (x86) -[ http://ioquake3.org/get-it/]("http://ioquake3.org/get-it/")
(Save this in Home/ (Username) )

Take Baseq3 and Mission Pack folder from windows install (thats what i did)

Install

Go to your Terminal and type : sh ioquake3-1.36-7.1.i386.run

This should bring up the installer just push ok or start, that takes a sec then when you go to you Home/username folder it should have a folder called Ioquake3.

Go into this folder and you should see Baseq3 and missionpack folders, look into baseq3 and u see 3 .so files... In this folder copy all your .pk3 files from your windows baseq3 folder, You could use thumb drive or something to get them into linux. Do the same thing with Missionpack folder just copy them.

Now just double click ioquake3.i386 In Ioquake3 folder!

Enjoy /w sound! :popcorn:
