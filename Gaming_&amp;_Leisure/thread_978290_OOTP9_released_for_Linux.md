### Post by eriqjaffe on 2008-11-11
Yeah, I know, it's commercial, but it's still nice to see a developer support Linux at all.  I ran it through Wine, but don't need to do that any more (hooray!) - the native build is notably faster than running in Wine, or even running in Windows on the same hardware.  Hooray, indeed!

[http://www.ootpdevelopments.com/board/ootp-9-general-discussions/175115-out-park-baseball-9-released-linux.html](http://www.ootpdevelopments.com/board/ootp-9-general-discussions/175115-out-park-baseball-9-released-linux.html)

$39.99 gets you a license that allows for two installations, and the intent is for future versions to have cross-platform licensing, so you can have the game on your Linux box at home and your Windows system at work.

Honestly, I've been playing this game for years now (I started playing when they released OOTP 5), and it's fantastically absorbing.
