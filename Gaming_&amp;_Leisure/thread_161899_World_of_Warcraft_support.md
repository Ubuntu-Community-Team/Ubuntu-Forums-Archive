### Post by Rizado on 2006-04-19
To be honest it runs much better in wine than cedega. You have to apply some patches though. But it's not hard. 
The easiest way is to do is download the latest cvs version (usually works better and is stable.) apply some patches and compile. Copy wow directory from windows and run it in opengl mode, either by editing a file or start it with -opengl.
With cedega I've only had problems, no mouse, crashes, unable to install patches etc.

I get a few fps lower in linux than windows but when I'm in places with lots of people like IF or SW linux works better.

And a link to the patches.
[http://appdb.winehq.org/appview.php?versionId=4031](http://appdb.winehq.org/appview.php?versionId=4031)
