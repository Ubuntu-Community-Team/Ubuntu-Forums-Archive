### Post by noerrorsfound on 2008-09-29
The Flash slide show is overkill. All that information (which really isn't much) could have been listed on the page with plain text which would save bandwidth and make it more accessible.

Why are you all so against giving these games ratings? If you remove the rating system from this website then there won't be anything unique about it and you might as well go to some other Linux game site like [The Linux Game Tome](http://www.happypenguin.org/) and just visit [http://getdeb.net/](http://getdeb.net/) if you need an "easier" installation than extracting a tarball. But since those debs install things to system directories it causes problems for some games—one that I know of being AssaultCube—because the games can't write to their directory (the next version of AssaultCube will store that stuff in the home directory). In cases like that, deb packages just cause more problems than they solve.

I voted yes to keeping the rating system. If you don't care about ratings, ignore them. Maybe some parent wants this kind of thing for non-commercial games so he or she can decide what to let his or her child or children play. Just because they're not useful to you doesn't mean they won't be useful to someone else.
