### Post by Technophobia on 2009-08-05
Hey people who use Ubuntu and play games,

This **Sunday** UGN will be playing **Urban Terror** ([http://www.urbanterror.net/news.php](http://www.urbanterror.net/news.php)) ouuu. Urban Terror is a free team based first person shooter where you shoot people and they bleed to death.

As always UGN events start at **1800 GMT** (2 PM EST) on **Sunday**.

For more information about past and future UGN events please visit [http://gwos.org/doku.php/event:start](http://gwos.org/doku.php/event:start)

See you there or in the IRC at **_#UGN on irc.freenode.net_**. If you don't feel like setting up a IRC client there is always the web based one here [http://java.freenode.net//index.php?channel=UGN](http://java.freenode.net//index.php?channel=UGN).

---------------------------------------------------------------

_For new Players_

1. Download the game from here [http://www.urbanterror.net/page.php?6](http://www.urbanterror.net/page.php?6). Make sure to get the **.zip** and not the .exe.

2. Extract the .zip file anywhere

3. Right click ioUrbanTerror.i386 inside the extracted folder and go to properties. Next select the permissions tab and click the check box next to "Allow executing file as program". Click close.

4. Run UrbanTerror by double clicking on ioUrbanTerror.i386 inside the extracted folder.

5. Play

If you need any help just ask either in this thread or in the IRC chat room.

### Post by 569874123 on 2009-08-08
Before trying any of these methods u should do what the main topic says and run the game at least once.

Method 1

This command also works in console
If u cannot open console add this line to ~/.q3a/q3ut4/q3config.cfg and save it.To add it correctly move all lines below the second one, one line lower and insert the code in the third line.

```
bind x "toggleconsole"
``` 
After adding that line to the file and saving the file u can press x to open and close the console.


```
/password ugn; connect 99.10.224.70
```

Plz keep in mind that before pressing enter, the last line in the console should look like this(if u r doing it this way)

```
]/password ugn; connect 99.10.224.70
```

Method 2(for those that do not like consoles)
Install xqf

```
sudo apt-get install xqf
```
Create the following directory 

1-Run xqf. 
2-Go to edit and press the add new server button or alternatively press ctrl+n. 
3-Select quake 3 arena as the game in the right side of the box and in the left paste this ip ```
99.10.224.70
``` then u can click ok.
4-After that select "favorites" in the left side of xqf and the server should appear, if there r too many than u could always quick filter it with the ip or the name of the server.
5-After the server is selected right-click it and click on connect.
6-After that a screen will popup, in command line select the file u want to execute, being it ioUrbanTerror.i386 or ioUrbanTerror.x86_64
7.Repeat 4 and 5
8. Enter ugn as the password and ur good to go :D

[http://forums.urbanterror.net/index.php/topic,7710.0.html](http://forums.urbanterror.net/index.php/topic,7710.0.html)
It's a modified XQF. It includes 4.x support, custom server vars filtering, number of players filtering, Hobbes' refresh patch, and legacy PB and BE support.

To watch the tutorial u will have to rename the file in UrbanTerror/q3ut4/demos in the zip to anything without lower case chars without altering the file type, for example:1.dm_68 or TUTORIAL.dm_68.
Then go to demos and choose the tutorial.
