### Post by tekkah56 on 2015-12-25
Installing EA Origin through wine would also help if there are no specific programs for linux
```
$ sudo apt-get update && sudo apt-get upgrade
$ sudo add-apt-repository ppa:ubuntu-wine/ppa
$ sudo apt-get update
$ sudo apt-get install -y wine1.7
```
After you finished installing Wine download EA at their website an walla! You've got yourself BF4. It may not be perfect since it has not been tested on Wine yet. Here is a thread on the Wine forms about this topic [https://forum.winehq.org/viewtopic.php?f=8&t=20186](https://forum.winehq.org/viewtopic.php?f=8&t=20186)
