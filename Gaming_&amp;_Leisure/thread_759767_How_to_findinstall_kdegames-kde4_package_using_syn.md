### Post by disturbedite on 2008-04-19
no you don't have to "kde's GUI".
click on Sections --> All
then highlight something, and type kdegames-kde4.
be aware though, installing kdegames-kde4 will install ALL kde4 games.  if you only want some, or know which ones you want specifically, then you don't need to install kdegames-kde4.
