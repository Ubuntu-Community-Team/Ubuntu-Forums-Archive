That's very strange.  I've also had problems getting it to work under WINE.    Possible the same problem - Click play, screen goes black like the game is loading, but it crashes before anything is displayed on the screen.  However, it works almost flawlessly for me in Cedega (occasionally it crashes or freezes when zoning/traveling/loading).  

I installed the game without CDs.  Downloaded the generic launchpad installer from the SOE site, then ran that in Cedega with the SWG profile, and the /game:starwars variable.  This loads the SWG launchpad, which then downloads the entire client and all hotfixes/patches.  You can also copy all required files from a previous installation since the game doesn't need to be "installed," it just needs all it's files in the correct directory.  So, to save download time, just copy the files from a Windows install or a previous Linux install, then do a Full Scan on the Launchpad.

It runs faily well - FPS is as good as in Windows.  Some issues are crashes when traveling, mouse/view issues when in 3rd person mode (scroll back into first person to fix), and /logout will freeze the client at the character select screen occasionally.

EDIT: Well, after a little more testing, it's not working "flawlessly" - JTL (space) doesn't seem to work at all; I crash every time I try to launch my ship.
