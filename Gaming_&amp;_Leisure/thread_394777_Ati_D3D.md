### Post by Death_Sargent on 2007-03-27
first of all set

```
Section "Extensions"
	Option	    "Composite" "Disable"
EndSection
```

should be

```
Section "Extensions"
	Option	    "Composite" "0"
EndSection
```

this might not seem important but it really can be

secondly what are you using to run these games

if cedega then just run the diagnostic tool and everything should start working.

For anything else you need to make sure that 
A. you have the correct driver dlls in the correct place
B. Configure for windows 2oo or higher
C. check the cideo config files to make sure they have the correct data

In the end it would be simpler to just use cedega since it will auto profile your card. 
I know you can play WOW but i have had problems with counter strike and such.
