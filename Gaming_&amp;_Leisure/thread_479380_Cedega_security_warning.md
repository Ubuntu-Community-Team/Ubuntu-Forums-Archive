### Post by xzero1 on 2007-06-20
I was all set to try Cedega, I got the 32-bit binaires to run on my 64-bit Ubuntu. But their signup screen page has no security encryption so any personal information entered here will be sent unencrypted. There is also no website validation so make certain you are on the correct page before using it. To be fair you are redirected to a secure website for the payment. Still, if you value your personal information, consider yourself warned!
