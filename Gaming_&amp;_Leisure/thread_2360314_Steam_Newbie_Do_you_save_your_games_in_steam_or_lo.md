### Post by #&amp;thj^% on 2017-05-03
This depends on the game. Games that use Steam to share data are marked with Steam Cloud in the Steam store but some only sync stuff like stats, so make sure to look up the game's information first.

For games that do not use Steam Cloud, usually the best way to transfer saves is to copy the savefile from one computer to another. This is stored in a game dependent location.
BTW Dropbox synchronization, I never tried this myself but it should be possible. Most games have their savegame data somewhere in you personal home directory, often inside the "my games" folder.

And Steam has a Backup feature: [https://support.steampowered.com/kb_article.php?ref=8794-YPHV-2033](https://support.steampowered.com/kb_article.php?ref=8794-YPHV-2033)

### Post by mastablasta on 2017-05-04
bare in mind that sync is not backup. if savegame get's corrupted, the backup will be corrupted as well. if you want backup the best way to do it is to have versioned backups. there a  few apps that will do that (Luckybackup, Areca, Duplicati, rdiff-backup... to name a few).

what i do is i copy games to sync folder, sync it, then turn of the syncing client.
dropbox and such might have file versioning available, so you could use that as well to do the backups.
