### Post by T.J. on 2016-03-26
"Decent graphics" are highly subjective.  

There are many commercial and non-commercial games available for Linux.  They do not require WINE.  If you are looking for open source titles, there are a few such as 2048, 0 AD, or Battle for Wesnoth.  Some of the commercial ones are AAA titles: like XCOM2, Saint's Row 4, Alien: Isolation, Middle-Earth:  Shadows of Mordor.  Even Tomb Raider is scheduled to be added soon.   The number of titles for Linux is increasing in large part due to Steam.  I highly recommend using the Steam service if you are a Linux gamer.  The Steam games have virtually identical appearance to their Windows counterparts.  Every Linux game I have bought from Steam has been EXACTLY like the Windows version.

There are also other ways of running games other than WINE.  Using a Linux KVM and a copy of Windows, you can have 100% compatibility if your computer has a second video chip and supports IOMMU.  I use a KVM to run Windows games (that do not have Linux versions) and to write code in Visual Studio for work, all running under Linux's control.

If you want more information, please ask. =)

### Post by Molnr_Dniel on 2016-04-14
What genres are you looking for? And where do you draw the line for "decent graphics"? I have noticed that you wrote descent, that game actually has Linux source ports. :)
We played Urban Terror a lot, it is using the ioq3 engine based on Quake 3. The game is similar to Counter Strike, but you do not have to pay for your weapons and it has cool trickjumps (the developers even call it "Hollywood tactical shooter"). It is completely free. Another good free FPS game is Warsow. You can not even tell that it is based on the Quake 2 engine. It plays like Quake 3, it has Clan Arena and other game modes built in. Unreal Engine 4 has native Linux support so we have a bright future ahead of us. Heavy Gear Assault is just about to release a new build with a Linux dedicated server and client.

You can always filter games on Steam:
[http://store.steampowered.com/browse/linux/](http://store.steampowered.com/browse/linux/)
