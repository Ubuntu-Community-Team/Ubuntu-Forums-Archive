[COLOR=#333333][FONT=sans-serif][EDIT] For clarification, [B]these are not illegal files, nor is this condoning any sort of piracy. These are open-source modifications to get the game running on Linux properly.

About a month or so ago, roadrunner shut down their services or something else happened. That doesn't matter. What matters is that a webpage hosted a few packages that fixed functionality in running that game natively on Linux. It was used in many a tutorial from years ago up until today. It appears that it was not mirrored anywhere else. So its sudden disappearance is a bit of a concern for having a good experience running NWN on Linux. There are three files needed, and I have managed to score one of them (nwmovies-latest.tar.gz).[/FONT][/COLOR][COLOR=#333333][FONT=sans-serif !important]The two files missing are:

[/FONT][/COLOR]
[COLOR=#333333][FONT=sans-serif !important]cursors.tar.gz
md5: 7be935418f0ececb5660f53b7a902b38
[/FONT][/COLOR]
[COLOR=#333333][FONT=sans-serif !important]The website that no longer exists is: [/FONT][/COLOR]
[COLOR=#333333][FONT=sans-serif !important][http://home.roadrunner.com/~nwmovies](http://home.roadrunner.com/~nwmovies)
[/FONT][/COLOR]
