### Post by Telengard C64 on 2011-12-19
Rogue is included in the [bsdgames-nonfree](http://packages.ubuntu.com/search?keywords=bsdgames-nonfree) package. If the *multiverse* repository is enabled you can install it with the following command.

```
sudo apt-get install bsdgames-nonfree
```

If you need help playing:

[A Guide to the Dungeons of Doom](http://www.rots.net/rogue/guide.txt)

Have fun  :D
