### Post by Cappy on 2007-07-08
if you install it to /usr/local/games you have to run the installer with sudo.
```

sudo sh shellscriptname.run
```
Oh, and be sure you copy the script from your CD to your hard drive to run it.

I personally install ut in a folder in my home directory with sudo.
