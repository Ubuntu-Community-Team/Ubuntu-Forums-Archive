### Post by julia4 on 2015-01-21
I can only comment on what worked for me. Hopefully it will help. I'm using Ubuntu 14.10, but I think Nautilus works the same as in 14.04. 

I went into the Nautilus menu and under Edit -> Preferences -> Behavior tab -> Executable Text Files (in middle section of page). Check the box for either Run executable text files when they are opened  OR the box for Ask each time.
 
That enabled me to run both an older x64 version of Firestorm and Singularity in Nautilus without installing either viewer. If that doesn't work, I also got both viewers running (without having to manually change the settings) in Thunar and PCManFM file managers. You could try installing one of those file managers as a backup.
