```Played a few levels, then Braid was finished.  This one proved difficult, because there was both a software conflict and a hardware issue.  As others have pointed out, Braid likes to jack ALL your monitors, not just one.  To run it properly, you need to be running on one screen.  Second, it doesn't want to run without S3TC texture compression.

That can be enabled, if you're in a region where it's legal, through driconf.

```
sudo apt-get install driconf
```You start driconf by typing its name into the Terminal, then go to the image quality tab and click yes for the top option.  After that, Braid booted to a black screen, where it sat for a few minutes.  First time setup operations, I presumed, because it started running fine after that.

Not to seem preachy; I just wanted to compile all the good methods into one post.
