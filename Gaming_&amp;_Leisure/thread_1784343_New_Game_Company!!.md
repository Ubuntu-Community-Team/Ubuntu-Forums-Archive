### Post by Galilei on 2011-06-16
Hey everyone!

In February of this year, me and my friend put up a gaming company of our own (a dream we've had for a very long time). 

As of now, we are developing a game for Android 2.2. The game is developed with Linux (Ubuntu, in fact), which means that the game will also work with Linux "out of the box", although we are going to compile a version to OsX, Windows, and any other platform that we can manage. 

Our current game is developed with smartphones and tablets in mind, but it translates surprisingly well to desktops as well. We hope to release the game in the Ubuntu Software Centre in addition to the Android platform in the very near future. The game is supposed to be done in about two weeks, so here's hoping that nothing serious gets in the way of things. 

Anyone with at least a HTC Desire caliber phone though, should definitely check out our game. We've put up a blog were we will share details about our games, company, and ourselves. Since Ubuntu seems to be heavily invested in touch based UI's etc. our game would seem to make a perfect fit with the Ubuntu OS. I have an Apple Magic Trackpad, so we'll see if we can translate the game mechanics to that device. If not, the game works perfectly well with the mouse. Be as it may, in a few weeks at least the Android version of the game should be ready!


Please check-out [www.lightsperfections.blogspot.com](www.lightsperfections.blogspot.com) for more information on our company Lights and Perfections, comment, and send us mail. We're stoked, and hopefully you are too!
