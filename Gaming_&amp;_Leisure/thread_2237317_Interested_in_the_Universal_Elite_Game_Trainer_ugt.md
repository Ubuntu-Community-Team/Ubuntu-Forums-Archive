### Post by s-parschauer on 2014-08-01
The GPLv3 licensed Universal Elite Game Trainer "ugtrain" became the  only game trainer which is actually working on Linux as it supports  dynamic memory with simplified discovery and automatic adaption. For  that it uses LD_PRELOAD and libraries to hook on malloc() and free(). It  bypasses address space layout randomization (ASLR) of the heap, the  stack and with its position independent executable (PIE) support also of code and data addresses within the  executable. This way it works on latest Ubuntu releases which have PIE for everything per default. The support for position independent code (PIC) will follow.  ASLR/PIC/PIE support has been merged into latest scanmem as well and  ugtrain integrates scanmem. The reason for this is to avoid the need for  root permissions as parents may ptrace their children as regular user  and scanmem is plain the best memory scanning tool on Linux.

Ugtrain is developed by example and provides easy to use configs for  FOSS games as these allow cheating. This makes ugtrain a completely  legal Open and White Hat Game Cheating tool. The configs are used to  demo and test ugtrain.
It claims to be an education project as it documents how advanced game cheating methods work for everybody.
