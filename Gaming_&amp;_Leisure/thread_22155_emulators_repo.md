### Post by cdhotfire on 2005-03-26
1. Go to synaptic then Settings -> Repositories.  
2. Click "Add". Choose "Ubuntu 5.04 "Hoary  Hedgehog"" and  check all the markers at the bottom.  
3. Press "OK".  
4. Do the same for "Ubuntu 5.04 Security Updates".  

That should give you multiverse universe and all hoarys reps. Zsnes is in there and so is mame. Well thats all i know if anyone else would like to add go ahead.
