### Post by Felson on 2007-11-15
There are a few fun free ones. (ones I find fun anywho)
Globulation2 
Falcon's Eye (remake of nethack)
Glest
Wesnoth
To name a few that I like. There are some native commercial titles, but they are few and far between. For those you will need to buy them online, as you are not going to find them in a box store. WINE can run some newer games pretty well, but I find that setting it up is a headache. And then there is the whole Emulator thing. See my sig for that. I personally spend most of my gaming time in those for now.
