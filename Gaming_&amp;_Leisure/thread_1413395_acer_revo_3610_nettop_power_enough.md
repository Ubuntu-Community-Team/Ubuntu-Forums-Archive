### Post by tommcd on 2010-02-23
What nvidia video card is on the Acer nettop? 
I used to play Doom3 quite well on a nvidia 5200. I would think any nvidia card from the 7000, or 8000 series could play Doom3 reasonably well.
Do you currently have Ubuntu installed on the Acer nettop? If so, then just install some games and see how well they run. You should probably be able to play a game like Nexuiz for example:
[http://www.alientrap.org/nexuiz/](http://www.alientrap.org/nexuiz/)
There is a Windows installer for Nexuiz also. So if you do not currently have Ubuntu installed and are using Windows, then try installing Nexuiz in Windows and see how well it runs. If Nexuiz runs ok on Windows, then it would likely run just as well, if not better on linux.
