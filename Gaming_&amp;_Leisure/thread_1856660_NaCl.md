### Post by Kodeine on 2011-10-08
Salutations!

If you haven't heard it's Google's attempt to allow usage of C and C++ (C# support is planned) in the browser. So this means we mite be seeing some intense web applications in the hopefully not to distant future and this of course means it's cross platform. I find it hard to see this as anything exciting for Windows users but for Mac and of course for us Linux users then only sliced bread is better. Obviously this is being developed with games in mind but there should also be some nice utilities such as image editors, office applications and perhaps 3D modelling software.

Currently it's in it's first baby steps. Quake and a open source FPS entitled 'Nexius' have been ported using the NaCl SDK. There's also NaClbox which allows you to play old dos games in the browser.

Excited?
