### Post by jacksaff on 2007-04-24
There really isn't a great internet chess server client in linux at the moment. You can get all the functions you need with xboard. Eboard is a bit more user friendly with a gui for most things and jin (java, so cross platform) is ok too but none of them come close to babas or the icc's proprietry software for windows. Knights looked promising but seems to have been abandoned. I run babas in wine, which is a bit win98 ugly and requires an older version but is the best I can find.
Fics is the best free site by far if you're serious about chess. A lot of newbies get put off though because they can't find a client program that's easy to use from the time they first run it. It's hard to see all the features it has unless you have a good, graphicla client or are prepared to learn it's command line interface. 
ICC is great but a bit expensive and they don't have a good linux client so I won't give them any more of my money. Generally has a lot more players than fics at any given time and pretty much all of the top players who aren't getting paid to use other software play there.
