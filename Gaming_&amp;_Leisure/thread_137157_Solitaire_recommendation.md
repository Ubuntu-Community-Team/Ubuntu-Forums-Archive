### Post by Sembiance on 2008-03-16
Hello

The website World of Solitaire ([http://worldofsolitaire.com]("http://worldofsolitaire.com")) has 'La Belle Lucie' on it (along with 31 other games).

It works on many Linux browsers including: Firefox, Opera and Konqueror.

It's 100% free with no ads nor required registration.
It's all done with JavaScript so it doesn't need flash or any installs.

PySol has a lot too, but I don't like how it's kind of ugly. World of Solitaire is a lot nicer looking :)
