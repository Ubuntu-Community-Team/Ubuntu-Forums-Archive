### Post by RabidWeezle on 2009-09-15
My personal favorite FPS games that you can easily play on that system with good single player and multiplayer:

Quake 2 - Good storyline, awesome multiplayer, A++. Not just a tech demo at all. You will need to buy the game for the pak files.

eduke32 - Good duke nukem 3d port with 3d model support and high res textures. Duke has some of the most... unique gameplay I've seen in my life. You will need to buy the game for the grp file.

doom legacy (all dos doom games)- Take that old game and put a new face on it, throw in some 3d models and whip it into GL mode with md2 models, very awesome port, though I think you will need to compile it yourself. Personally I wish some certain games that were made on id's engines would go opensource that haven't yet so we can port some old but good games to linux (I'd still buy their games for the media needed to run the game), including star trek elite force, jedi knight/jedi knight 2, american mcgee's alice, etc.

Other than that, all I can do is point to the wine appdb and looking up your favorite games. I suspect all quake 3 engine games should run rather well in wine. Personally if Turok 1, serious sam, and alien vs. predator had a linux port, they would be on my list aswell.
