### Post by epsolon77 on 2010-12-02
I have a minecraft server running in ubuntu 10.04 server.  After installing the Java JRE packages, simply download the minecraft_server.jar file from the downloads page from minecraft.net then enter the script listed on the minecraft page to start the minecraft server.  Make sure port forwarding is setup for external users to be able to access it, and also make sure the firewall will let you trough.
