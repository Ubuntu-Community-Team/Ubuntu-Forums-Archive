### Post by DoktorSeven on 2008-01-06
Configuration files are generally in /home/[your username]/.[name of executable]/ (or as a file instead of a directory).  So check ~/.neverball/ (neverputt stuff is put here too), specifically ~/.neverball/neverballrc -- deleting it should reset your settings.  (~ is a shortcut character used for /home/[your username]/ and is easier to type :) )

Note that files and directories starting with a dot are "hidden", so if you are looking for them with a GUI file manager, you'll need to tell it to show hidden files/directories.
