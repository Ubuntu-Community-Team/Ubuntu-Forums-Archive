### Post by desicratn on 2008-03-07
Just download version 4.1 and extract the folder anywhere. Set the ioUrbanTerror.i386 to executable by going to permissions and check the "make executable " box. Double click it and it boots right up.
There is no need with 4.1 to do anything else it is all done for you now. SUPER easy.


[http://www.urbanterror.net/page.php?6](http://www.urbanterror.net/page.php?6)
"Just unzip the zip and make a shortcut to the ioUrbanTerror executable.
Mac executable: ioUrbanTerror.app
Linux 32bits executable: ioUrbanTerror.i386
Linux 64bits executable: ioUrbanTerror.x86_64
In linux, you need to make it executable first: right click it, properties and tick the 'allow execution' box."
