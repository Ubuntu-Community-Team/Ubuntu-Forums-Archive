### Post by oldrocker99 on 2014-10-31
OK, hit CTRL-F2 and type
```
sudo nautilus
```

I'm assuming that Nautilus is your file manager.

You can then open the file with the Archive Manager, and then extract the contents to the proper directory on the file system, which you apparently have already found.
**This can be a *dangerous* operation! At the very minimum, close the file manager as soon as you're done!** 

Hope this helps.
