### Post by hikaricore on 2007-08-15
**Windows software does not run _natively_ under Ubuntu/Linux.**

You will need to install WINE or simialar API compatability software to even have a chance at possibly running such software.  It does look like Live For Speed works /w WINE:

[http://appdb.winehq.org/appview.php?iAppId=2582](http://appdb.winehq.org/appview.php?iAppId=2582)

But this will not always be the case with such programs.  If you can't live without your favorite
games and applications that don't work under WINE, you really should be dual booting.

Info on WINE / How to install: [http://help.ubuntu.com/community/Wine](http://help.ubuntu.com/community/Wine)
Helpful WINE info: [http://ubuntuforums.org/showthread.php?t=497332](http://ubuntuforums.org/showthread.php?t=497332)

--

Also if you like racing games you may want to check out some free/open source titles: [Vdrift]("http://vdrift.net/") and [ManiaDrive]("http://maniadrive.raydium.org/").

You can find Ubuntu packages of both here: [http://www.getdeb.net/category.php?id=3](http://www.getdeb.net/category.php?id=3)
([http://www.getdeb.net/app.php?name=ManiaDrive](http://www.getdeb.net/app.php?name=ManiaDrive) / [http://www.getdeb.net/app.php?name=VDrift](http://www.getdeb.net/app.php?name=VDrift))

And you can find more racing titles on the UDSF Native Game/Racing page: [http://doc.gwos.org/index.php/Racer_Games](http://doc.gwos.org/index.php/Racer_Games)
