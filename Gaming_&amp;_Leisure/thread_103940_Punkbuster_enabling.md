### Post by ratsalad on 2005-12-15
I know these games are supposed to come with Punkbuster, but I've found that you need to download the pbsetup tool to get it working properly. 

```

wget http://websec.evenbalance.com/downloads/linux/pbsetup.run
chmod +x pbsetup.run
sudo ./pbsetup.run

```

and add your game(s)  :)
