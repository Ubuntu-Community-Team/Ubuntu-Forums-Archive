### Post by thenailedone on 2018-10-02
Hello ladies,

So there is another way to get EVE Online to run under Linux (and this time you have all the DX11 bling to boot).

To get Steam and Steamplay with Proton to work with all titles have a read here - [Introducing a new version of Steam Play]("https://steamcommunity.com/games/221410/announcements/detail/1696055855739350561") (NOTE that there is no need to enable to BETA anymore as it is now in the client proper already).

Once you have Steam up and running and you have a new enough graphics driver installed:
1. Search for EVE in the Steam store and install
2. Run EVE, it should fail (this is needed to make the required prefixes etc.)
3. Follow the instructions from [thomas-lange/GetEveToWork.md]("https://gist.github.com/thomas-lange/520ebeb899666f6f0354915b7b934582") (thank you so much for this write-up Thomas Lange)
4. Profit (run EVE again and go through the installation. I have found that I had to stop and re-start the client a number of times to get it installed and ready).

Enjoy :KS
