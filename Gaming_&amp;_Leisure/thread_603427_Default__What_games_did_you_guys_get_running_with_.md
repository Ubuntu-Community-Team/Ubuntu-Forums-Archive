### Post by Ferrat on 2007-11-06
A big problem for some games 

Guild Wars 
Neverwinter Nights 2 
Homeworld 2

is the fact that after 0.9.45 some big changes to the dx drivers in Wine uses a Z value that works great on nVIDIA but not on ATi, ATi renders invisable models and textures with games using this specific value. 
Not clear if all ATi cards are affected but many are.

As for the 8.42.3 drivers they do boost preformance but are somewhat unstable overall on some cards and seems to have some small problems with later kernels
