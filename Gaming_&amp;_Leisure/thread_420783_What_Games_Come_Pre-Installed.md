### Post by AndrewRiedi on 2007-04-24
Checkout a program called Wine.  

[WineHQ]("http://winehq.org")

Use the application database to check if applications run, how well, and what tweaks you may need to use.

[WineHQ AppDB]("http://appdb.winehq.org")
