Hi everyone. My project team released a new version of our RPG, Hero of Allacrost, a few days ago and I wanted to spread the word around. We've received a lot of useful feedback and accolades from Ubuntu users on this forum in the past, so I wanted to make sure this community was aware of our new release.

We're trying to better embrace the "release early, release often" mantra in open source development so this is not an official release but more of a beta. Still though it is fully playable and takes me about half an hour to beat from start to end.


**Download**
1) Visit our main website at [http://www.allacrost.org/](http://www.allacrost.org/)

2) Find the download link and read the instructions. Make sure you download the "unstable 20100228" source file.


Because this is an unofficial release, we haven't invested in doing any packaging into a .deb, .rpm, or other format. Our last official release ( 0.2.2 ) is available in .deb format, but its much older ( October 2008 ).


**Preview**

Here are several older screenshots and some videos to give you an idea of what the game is like and help you figure out whether or not you want to give it a shot. This new release contains a lot more content and features than our last.

Feedback is very much appreciated. Also if you're interested in helping out the development team in any way (we could use it), you can ask me or check out the "Contribute" link on our website. Thanks, and enjoy!

### Post by RootsLINUX on 2010-03-10
> **Noraphalem said:**
> Wow!!! Nice oldschool console like rpg! Damn, wenn is it ready for everyday use!? :grin:

Yeah, I started this project long ago because I always wanted to make a game like that and there were none that I knew of on Linux. Not sure how you would clarify what "everyday use" is and when it will be "ready". Our last official release, 0.2.2, is stable but its not much more than a tech demo. The current release we're working on, which had an unstable beta released a week and half ago, is like a "mini-RPG". It has a plot, sequence of events, characters, betrayals, etc. It took me about 30 minutes to play through from start to finish.

The full shebang which will be something on the order of 20+ hours of gameplay won't be finished for at least a couple years (at LEAST). But after we finish this current demo that we're working on, which is our last, we'll begin working on the full game and will release one playable module of it at a time. We're shooting for each module being about 30 minutes of gameplay. So think of it like a TV series, where you get it delivered in pieces over a span of time.

### Post by Åtta on 2010-07-12
I've played the game for a bit now, and I have a few things I would like to mention.

First, I don't know if you're planning to do this for the proper releases, but you should really de-couple the editor from the game. I have to install around 100mb of Qt4 headers to install the game.

Like you said it might, the game segfaulted when I tried going from one area to another. 

Another thing I noticed was that I recognized some of the tiles. I know some of them come from a free tile set, but I just know I've seen that potion icon somewhere else... But I could be mistaken.

And lastly, when I installed the game, I couldn't just run it by running "allacrost". I had to figure out that the binaries were stored in /usr/local/games/ and run them from there.
