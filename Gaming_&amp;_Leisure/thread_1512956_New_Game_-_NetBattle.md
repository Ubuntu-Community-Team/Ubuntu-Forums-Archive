Hello everyone,

I'm going to have some time on my hands soon and I thought I'd port and finalize an abandoned game project.

What is NetBattle?

NetBattle is a network-based multiplayer arena combat game.
Basically each player controls a character with a sword and bow and they fight in an arena. The game includes an easy to use map editor to create custom arena. The game was almost done but had a few missing features, bugs, and needed some core engine clean-up.

It is originally programmed in Java with the simple JGame engine but I would like to port it to python with pygame.

Just wanted to hear your thoughts on whether I should attempt this or not.
