### Post by simzer on 2010-12-18
Hi,
I'm developing a game called 4dtris. Now I released a new beta version and I would like to invite you to a bit of beta testing.

The game is a tetris game, but in four dimensions. It's inspired by Blockout, a retro 3D tetris game. It's written in C, uses openGL  and SDL and released under GPLv3.

Suggestions/bugreports are welcome!

Project blog: [http://sourceforge.net/apps/wordpress/dtris/](http://sourceforge.net/apps/wordpress/dtris/)
Download: [https://launchpad.net/4dtris/+download](https://launchpad.net/4dtris/+download) (can be installed via PPA or .deb)
