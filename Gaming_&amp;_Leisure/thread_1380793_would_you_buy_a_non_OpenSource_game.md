### Post by MaximB on 2010-01-14
Not only will I buy, I've already bought many.
Not to vain I run the Linux Gaming News website (see my sig).

It's like the chicken and egg problem.
Many people won't move to Linux because there aren't enough good games for it, not the AAA games they play on Windows.
And the game companies say "there aren't enough Linux players to port our games".
they are very wrong because the indies who port their games to Linux make profit from this move.

In any case - if we buy Linux games, we tell the gaming companies that there is money in Linux gaming, so more companies will create more games - and eventually (with time) the gaming problem will be solved or reduced and many new people will use Linux, thus Linux will be supported better overall, more non-foss programs will be created for it.

Not that FOSS is bad, but proprietary **GAMES** are **usually** better.
