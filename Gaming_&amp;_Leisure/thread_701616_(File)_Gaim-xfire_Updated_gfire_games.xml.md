[*Old - See above* Update: April 7, 08]

Hey guys,

For those of you who use the xfire plugin for gaim/pidgin, and noticed the database out of date (people are playing a number instead of a game name), here you go. It's updated up to April 7, 2008.

Attached is a tar file containing an updated gfire_games.xml so that pidgin will correctly display such stuff.

To install:

1) Untar the file. Right click and untar it or put it on your desktop and in the terminal/console/konsole run: tar -xvf ~/Desktop/April-7-08-gfire_games.xml.tar.gz

2) Put it yourself via your favorite file manager/browser in /home/YOURNAME/.purple/ (notice the dot) to replace your current one. Alternatively, after the command in step 1, the command will be one of these two: cp ~/Desktop/gfire_games.xml ~/.purple  **or** cp ~/gfire_games.xml ~/.purple

3) In pidgin, go to account-->xfire-->reload game id list.

If you do not want to sign up for an account here at Ubuntu to dl the file, you can also find it [http://frozenfox.freehostia.com/gfiregames.html](http://frozenfox.freehostia.com/gfiregames.html) here at my page.
