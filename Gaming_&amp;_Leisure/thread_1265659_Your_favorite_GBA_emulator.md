### Post by mister_k81 on 2009-09-14
I generally just use [Mednafen]("http://mednafen.sourceforge.net/releases/") for playing GBA games. There's also a nice frontend for it [here]("http://ubuntuforums.org/showthread.php?t=813785&highlight=mednafen").

Another emulator to look out for is VBA-M, which is a fork of the VBA emulator. I believe it has a native GUI and a bunch of other improvements over the "current" VBA emulator that is out there. I haven't really tried it myself, but there is a 32 bit .deb file here:  

[http://sourceforge.net/projects/vbam/files/](http://sourceforge.net/projects/vbam/files/)

I'm not sure of there are any 64bit .deb files out there, or if this is the latest build (there doesn't seem to be any one established website for this project), but you might want to give it a try anyway.
