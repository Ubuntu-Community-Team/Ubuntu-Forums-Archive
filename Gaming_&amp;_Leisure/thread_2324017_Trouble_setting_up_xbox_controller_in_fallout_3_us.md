### Post by ZoiaGuyver on 2016-05-10
Afair Big Picture was shut off when running in Wine due to it requiring DX10. Not sure if that has been changed as I no longer use Wine to run Steam and have since changed to the Linux client.

If I remember right though there was also problems with the gamepad due to the same requirement (DX10). If you have a look on [wine.org]("https://appdb.winehq.org/") you may be able to find the bugs and add your comments in the hopes they get looked at. Hope this helps a little.
