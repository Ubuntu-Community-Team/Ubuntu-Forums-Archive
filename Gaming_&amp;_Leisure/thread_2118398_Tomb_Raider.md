The first game you linked to is Tomb Raider: Underworld, It was released on PC in 2008 and it is a traditional 3D third person adventure game: 

[http://en.wikipedia.org/wiki/Tomb_Raider:_Underworld](http://en.wikipedia.org/wiki/Tomb_Raider:_Underworld)

The second one is Guardian of Light, which was released in 2010 for PC: 

[http://en.wikipedia.org/wiki/Lara_Croft_and_the_Guardian_of_Light](http://en.wikipedia.org/wiki/Lara_Croft_and_the_Guardian_of_Light)


The full versions of both these games are for sale on Steam: 

[http://store.steampowered.com/app/8140/?snr=1_7_7_151_150_1](http://store.steampowered.com/app/8140/?snr=1_7_7_151_150_1)
[http://store.steampowered.com/app/35130/?snr=1_7_7_151_150_1](http://store.steampowered.com/app/35130/?snr=1_7_7_151_150_1)

But are for Windows only. The only way to run them on Linux is to install the Windows version of Steam through Wine and play them that way.

They're both pretty good games, though I like Guardians more. Pretty cool to see them run on a webbrowser. I can't test either one in Linux because I am using windows right now :oops: But they are both playable in Firefox on Windows 7. I will have to test them out on Linux later.
