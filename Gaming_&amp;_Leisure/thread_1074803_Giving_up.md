### Post by AdamShumpisxXx on 2009-02-19
Are you running it in DirectX or OpenGL? Are you using Wine, Cedega, etc...How can someone help you with so little details? I'll tell you something. I hate WOW. But I have installed Ubuntu on so many computers and the fact of the matter is a lot of people play it. I have gotten it to work just as well if not better than when running it on Windows (any version). Your first issue is the ATI graphics card. Ubuntu does not usually play very nice with ATI. It takes some work but most of the time you can always get it working to it's fullest. Having owned the x1650xt previously I know it's possible to get done what you want done. First things first...

1) Make sure you have your graphics card driver PROPERLY installed. You may be getting Compiz to work but that does not mean you have everything set properly. Most of the time you're going to have to go into xorg.conf and manually edit some things. That's just the reality of us ATI lovers. Hopefully now that AMD has taken over we'll get much needed love. DRI2 coming soon!

2) Make sure you have WOW running under Wine using the OpenGL graphics driver. This setup seems to be the best for me. Please follow this guide:
[https://help.ubuntu.com/community/WorldofWarcraft]("https://help.ubuntu.com/community/WorldofWarcraft")

3) Don't ever quick on Ubuntu. In the long run you'll find yourself coming back. Save yourself the computer work :p. If necessary do what I do. Set up 3 partitions. 2 seperate 15GB (basically) partitions. One for Windows (I use XP64) and one for Ubuntu (I use 8.10). And create the last partition as a basic storage / game install / Windows page file depository. I have a 500GB SATA 3.0GBPS HDD so this solution is more feasible for me.

Basically I'm just trying to help. That's what it's all about. Show me a Windows forum (that you don't have to pay for) that gives better and more universal help from it's users and creators alike. You won't find it.
