### Post by gazornonplat on 2010-08-05
Hi there, starting a project named Ubuntu GameSpace. Basically, the intended goal is to create a games digital download service similar to Steam and Direct2Drive for release in the Ubuntu Software Center. I'm looking for programmers and other experienced members to help in the completion of the project, as I myself am only trained in graphic design. In comparison to other Digital Download services, this will be fairly lightweight, as it is entirely web based, and no actual game content is hosted, we will simply provide demo downloads and links to purchase games from their original vendors. The idea behind this is to allow for a concise and easy to navigate/purchase catalogue of the very best games available for Ubuntu, with the possibility of social features later on. All help/feedback/contribution is extremely welcome and highly appreciated.

You're mostly on the right track here, let's say the image 2 was a final version and someone was using it. If they click the "Download Demo" button then that will download a demo from a url on the official website of the game (in this case DEFCON by Introversion Software), thus saving them the bother of hunting down a linux version of the demo and finding a download link. If they click the "Purchase" button then it will take them to a page (again) on the games official website where they can buy a copy from the original vendor, be it download or physical delivery. This is basically to simplify the process of finding decent games for Ubuntu and hunting down a place to purchase them. It will be web based, hosted on my own site with a Prism client available from the Ubuntu Software Centre. Did that help explain it at all?


As for the colour suggestion, yeah I will change it. This was just a very quick (about an hour or so) mock up to stir up some interest and get some people to work on it, it will change a lot after the project has gotten seriously underway. I think this is something a lot of Ubuntu gamers would use, so I'm hoping I can find the right people to help make it a reality.
