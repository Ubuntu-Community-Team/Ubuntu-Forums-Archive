### Post by NewJack on 2008-03-24
Did you download the .deb files from [www.getdeb.net](www.getdeb.net) or did you grab the tar.gz version?  Try grabbing it from getdeb.  Put UFO in the search box and it'll pop up.  You need to download the following .deb files:

ufoai
ufoai-data
ufoai-music
ufoai-tools (Not sure if I needed it, but installed it anyway)

You don't need the ufoai-server file, unless you are looking to run a game server.
