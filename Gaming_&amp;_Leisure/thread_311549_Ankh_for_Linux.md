### Post by msak007 on 2006-12-03
Well I just [read]("http://www.linux-gamers.net/modules/news/article.php?storyid=1868") that a game called [Ankh]("http://www.ankh-game.com/") (LucasArts style point and click adventure) is being considered for a Linux port. The developer ([Runesoft]("http://www.rune-soft.com/main.php")) will port the game if **200** pre-orders are placed. Only 200! Unfortunately the [site taking the pre-orders]("http://www.ixsoft.co.uk/cgi-bin/web_store.cgi?ref=Products/de/RSANKH01DV.html") is in German only so I can't order the game or I would. Does anybody know an English site where it can be ordered? Or are there any German speaking users who are interested in the game and want to pre-order it? I would love to see it ported, and would hate for the opportunity to be missed because a low pre-order count was not met due to language barriers.
