### Post by eight.coffee.beans on 2014-12-05
1. Upgrade to newest version of Ubuntu (optional)
2. Install PlayOnLinux ```
sudo apt-get install playonlinux
```
3. Start PlayOnLinux and in tab 'Games' you have option to select TES: Skyrim if I'm correct and following on-screen instruction you should be able to run the game
4. If step three doesn't work, install Steam inside PlayOnLinux and install TES: Skyrim inside of Steam, but I really don't know how playable the game will be with this step
