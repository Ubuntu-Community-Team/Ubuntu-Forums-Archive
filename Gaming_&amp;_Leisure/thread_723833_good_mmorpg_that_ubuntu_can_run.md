### Post by gonesolo on 2008-03-28
> **echinacea said:**
> the only MMORPG I play is city of heroes. I tried wine, didnt work. I tried cdega, didnt work.

has anyone else any success getting this to run?


I'm the app maintainer for city of Heroes on the Wine APPDB. I can confirm COH does run with wine but it does have some bugs.

1: don't install from CD - the patcher will hang - download the client from ncsoft and install using that.
2: The chat window has a bug = the first typed character is missed. also you need to press ESC twice to get back to character control.
3: trying to change your costume (at the tailor) will cause COH to crash - don't do it.

Also COH will run in Cedega.

However CodeWeavers have just released Crossover Games, which is based more closely on WINE, and they support the WINE project. Try the free trial it may work out better for you.(although I've not tested COH on Crossover yet)
