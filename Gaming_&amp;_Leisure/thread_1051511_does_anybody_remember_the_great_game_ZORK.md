### Post by FranMichaels on 2009-01-27
> **trixman said:**
> is it safe to download these games if they are not in the package manager.

thanks to all for the kind help
;)

Yes, you don't actually run the program it comes with. 
Frotz in the Ubuntu repository does.

Think of it as getting a pdf or an mp3 or ogg and playing it with your favorite Free software program.

You don't install the dat file, nor can it ever execute anything. It is merely interpreted by Frotz.

For a new user, I'm impressed you are actually concerned about security :)

For a better understanding of what frotz is and how it works
[http://frotz.sourceforge.net/](http://frotz.sourceforge.net/)
