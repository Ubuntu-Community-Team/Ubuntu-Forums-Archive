### Post by Sammi on 2007-02-25
If you say exactly which games you want to play, then maybe we can give you better support and suggestions for that specific title.

Keep in mind that many Windows games are based on DirectX, which is a proprietary closed-source technology that MS owns. Because of this these games will never run natively in Linux. There are, however, projects that are trying to reverse engineer DirectX, like Wine, Crossover, and Cedega. Both Crossover and Cedega are based on Wine. Today some DirectX based games run well when using these applications, while others don't.

This is a good reference site for Windows games that work in Wine: [http://appdb.winehq.org/](http://appdb.winehq.org/)
