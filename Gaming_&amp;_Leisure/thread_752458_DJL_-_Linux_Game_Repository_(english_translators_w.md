You will need Python 2.5 and Python-Qt4 libraries to launch DJL.
	
DJL manager is a free software (GPL) written in Python 2.5 for the operating system GNU / Linux, It is similar to Steam by Valve software.

[IMG]http://djl.tuxfamily.org/captures/depot.png[/IMG]

It provides, via repsitory,  downloading, installing and removing games in DJL subdirectory in a fully transparent manner (but without managing dependencies).
It can also import desktop shortcuts, place them in own subdirectory (to be able to run games already installed on the system).

A number of free(GPL) or free-of-charge games (more then 40 now) are available in the repository. You can add more via a menu and submit it to the developer.
The list of games is updated regularly from the Internet, it is not fixed.

DJL can [COLOR="Blue"]download and decompress / install packages tar, tar.gz (or tgz), tar.bz2, zip or GUI installers such as . package or. run (Loki installers)[/COLOR]. But these are only decompressed, without questions, the aim is that the installation is done in a fully transparent manner, without user intervention.

The games can then be removed in one mouse click.

Each entry in repository contain a [COLOR="Blue"]description[/COLOR], [COLOR="Blue"]screenshot[/COLOR] and[COLOR="Blue"] link to homepage[/COLOR] to help you choose which ones to install.

Once the games are installed, they can be launched from the main interface, which lists not only games installed from the repository, but also imported shortcuts.

DJL has[COLOR="Blue"] RSS[/COLOR] for some Linux Gaming Sites.[COLOR="Blue"] IRC-chat[/COLOR] for players.

DJL has Gnome/KDE integration, fully support (include repository entries) **French**,** Russian** and **partially English (beta)**.
