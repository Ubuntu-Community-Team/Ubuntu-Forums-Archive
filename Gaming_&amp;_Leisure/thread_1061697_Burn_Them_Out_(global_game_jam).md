### Post by bernardfrancois on 2009-02-06
Hi,

During the global game jam last weekend, I made a game with a team of 2 other people in 48 hours.

This is the short description of the game:
> You'll need to fight fire with more fire: try to save the sheep which are trapped in the barn!

It can be played under Ubuntu as well, just make sure you have the packages 'python' and 'python-pygame' installed, and then run 'python src/Game.py' after unpacking the files from the archive that can be downloaded here:
[http://www.globalgamejam.com/games/burn-them-out-0](http://www.globalgamejam.com/games/burn-them-out-0)

Feel free to post any comments or questions about how to get it working on Ubuntu.

Note that the game was made in 48h, and that the levels aren't really superb (the levels were done in the last hour of the competition).
One of these weeks, I'll make a more convenient level editor and probably some levels as well.

A known issue is that the game doesn't work on computers without a working sound card. A fix has already been made, but hasn't been put online yet.
