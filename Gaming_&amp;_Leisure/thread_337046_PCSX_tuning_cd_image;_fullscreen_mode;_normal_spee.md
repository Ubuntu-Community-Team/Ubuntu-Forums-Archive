I believe both of these things can be fixed under Configuration->Graphics.  If you're using Pete's MesaGL plugin (I'm not running this in Ubuntu, so I'm not sure what comes preinstalled, but this is the most common plugin, and you could certainly install it yourself), then you can click on the fullscreen button to go full screen, and you can click on "Use FPS limit" to limit the speed.  You may need to fiddle with the FPS limit setting to find the correct speed.

Hope this helps,
