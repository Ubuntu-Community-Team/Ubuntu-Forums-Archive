### Post by oldrocker99 on 2013-08-07
I successfully install Skyrim using the PlayOnLinux script, which asks you if you purchased it on Steam, then installs in its own bottle Steam for Windows, which you'll then have to activate via email, since it's treated as a new machine as fas as Steam is concerned. Then you open the library and select Skyrim and it will install.

Basically what mastablasta just said; the script makes it pretty easy. I've become quite a fan of PlayOnLinux. In fect, I use it more often than my paid-for Crossover Pro.
