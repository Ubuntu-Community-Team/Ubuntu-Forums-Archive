### Post by celtic426 on 2009-10-29
It depends on each game, but generally you install games just like you would install any other software.  For that file you downloaded, you want to right click it and say extract here.  That will make a folder in the same directory will some files in it.  Check out those files and see if you can see any easy install thing, like a file ending in .sh (this is an installer script).

Also, check out [http://getdeb.net]("http://getdeb.net") for .deb files of games that are really easy to install.
