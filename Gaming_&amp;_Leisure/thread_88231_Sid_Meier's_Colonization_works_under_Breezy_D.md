Things you need:

1. Colonization: [http://www.the-underdogs.org/game.php?id=210](http://www.the-underdogs.org/game.php?id=210)
2. Wine 0.9: 
a) Add this repository to your sources.list: [http://wine.sourceforge.net](http://wine.sourceforge.net) binary/ - for example by issuing the command 'sudo kwrite /etc/apt/sources.list'
b) sudo apt-get update
c) sudo apt-get install wine

Works like a charm for me. Any Colonization fans here :) ?
