### Post by dahtoy on 2005-06-27
the scroll around screen occurs when you've specified a viewport in the display subsection of the screen section of your xorg.conf. This can be rectified by setting ViewPort 0, ie,

```
Section "Screen"
    Identifier  "Screen0"
    Device      "ATI Graphics Adapter"
    Monitor     "Monitor0"
    DefaultDepth 24
    #Option "backingstore"

    Subsection "Display"
        Depth       24
        Modes       "1280x1024" "1024x768"
        ViewPort    0 0
    EndSubsection
EndSection

```
