### Post by CelticWarrior on 2020-05-12
Welcome.

Source: [https://www.quora.com/Can-you-run-Linux-on-a-Xbox-One-S?share=1](https://www.quora.com/Can-you-run-Linux-on-a-Xbox-One-S?share=1)

Quoting:
> The short answer is no.
The  longer answer is that any code that attempts to run on the Xbox One  needs to be cryptographically signed using a recognized key. If the  software is not correctly signed, the Xbox One will not run the code. In  order to run Linux, the boot code would have to be correctly code  signed so that the boot strap system in the Xbox One will launch the  Linux boot loader. To date, no one has been able to crack the Xbox One&#8217;s  cryptographic signature system to allow reliably booting home brew  software.
