### Post by cisforcojo on 2007-07-02
Hmm, PSP works perfectly for me (and quickly)

What kind of USB ports do you have? 

About the file deletion, how are you deleting the files? As sudo or as a normal user?
If as sudo, you're fine. If as a normal user, be sure to check out the .Trash folder. It's hidden but it definitely exists. All your files are there. Be sure to ALWAYS  delete the trash folder before you unmount. I've had problems with it before and I had to reformat the memory stick. Now that I empty the trash folder, no problems.
