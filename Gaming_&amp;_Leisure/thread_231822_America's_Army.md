### Post by Dropknee on 2006-08-12
This can do the trick to update your PB in America's Army

```
sudo chmod -R 777 /usr/local/games/armyops
```

Because most ppl intall this with sudo, you need to change the permission to the armyops folder and all the subfolder to be writen, with this you can save your changes in game and PB can update.
