### Post by Vgui on 2011-10-25
Unfortunately one of the main weaknesses of Ubuntu and Linux in general is playing Windows games, especially newer ones. You might get by with Wine or Crossover (which help to run Windows games) for older products, but you won't have any luck with something as new as DiRT 3, as shown here:

[http://appdb.winehq.org/objectManager.php?sClass=application&iId=13122](http://appdb.winehq.org/objectManager.php?sClass=application&iId=13122)

So yeah, looks like you'll be stuck with Windows 7.

I do however recommend dual booting with Ubuntu, as regardless of cost savings it's a much funner and nicer operating system to use once you get the hang of it. I personally keep a copy of Windows 7 stripped clean and purely for games, and everything else (software development, media, writing, web browsing, banking) I do in Linux.

### Post by sanderd17 on 2011-10-25
To be honest, you really don't save that much with buying one Ubuntu computer.

There's a lot of politics going on behind the scenes, and MS (as the biggest player) wins most of it, so they get manufacturers to create cheaper computers if they ship with MS.

The only way you can win money using Ubuntu is if you build your own system, or if you're a company that needs a lot of computers (in which case you can negotiate with the manufacturers directly).

In Europe, you can get a refund if you never boot Windows, but the procedures and the hours of calling you have to go through will probably cost you more money than you can gain.

And as said above, games is certainly not a strong point of Ubuntu.

But you could try Ubuntu on your old pentium. I wouldn't use Unity, but the ligher Unity 2D will probably run smooth. Or if you want a really snappy system, try Lubuntu. I have Linux with a pretty gnome shell environment running on a P4 with 1GB of ram at home. And it's certainly snappy enough for normal use (it's not Ubuntu though, but a kind of custom build OS).
