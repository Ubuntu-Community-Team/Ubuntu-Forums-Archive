### Post by Jengu on 2006-10-29
If you already have UT2004 installed on a windows partition, I can confirm copying it over to the linux partition and untarring the latest linux ut2004 patch into the folder you copy it to will also work (the latest patch contains new copies of the executables and the game's data is the same between windows and linux versions, so it works). This also works for Darwinia and a bunch of other games with Windows and Linux versions.
