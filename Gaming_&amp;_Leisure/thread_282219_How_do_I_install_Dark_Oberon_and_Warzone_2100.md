### Post by Techrogue on 2009-10-01
.deb files are for ease of use, because they automatically install components and add menu items. What I do is extract the files to their own folder in /opt and link to the corresponding executable by editing the menu. I would recommend against using alacarte; instead just right-click on your menu in the panel and choose "edit".

You should be able to install warzone by typing 

> sudo apt-get install warzone*

That will install the game, music and levels.
