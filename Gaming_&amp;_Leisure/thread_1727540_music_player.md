### Post by tommcd on 2011-04-13
The most Winamp-like music player in linux is Audacious, which is in the Ubuntu repos. Audacious does support some virtualizations; but I don't think you can set them as desktop wallpaper though.

There is also XMMS, which is also similar to Winamp. However XMMS is not in the Ubuntu repos.
