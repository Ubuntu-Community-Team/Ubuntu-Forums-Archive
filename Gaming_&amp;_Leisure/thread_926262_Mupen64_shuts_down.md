### Post by tjwoosta on 2008-09-21
if you have compiz enabled try disabling it before playing the game


to disable compiz press alt-f2 and type 
```
metacity --replace
```

(its important to not use the terminal, use alt-f2 instead)

after your done playing the game do this to re-enable compiz
 press alt-f2 and type
```
compiz --replace
```
