### Post by Filipek on 2009-12-11
I have played WoW extensively and only above Wine with reasonable results. I used PlayOnLinux as well with the similar results. The advantage would be you don't have to deal with configuration ... and I didn't have to do that because luckily, my WoW played with Wine "out-of-the-box" :-) 

It is good to know that linux "costs" you something about 10fps in the game - that is not definitely problem when you would have had 50 fps on Windows ... but if you have a machine capable of 20fps on Windows, linux would kill your gameplay (and I am not talking about fps-stressful environment like raids and stuff).

to 5nak3: Dual boot is always an option but if you don't want to steal anything, you'd have to buy license to OS which you don't like and that might bother someone. Linux is great and I love it but I also have the opinion to run applications in the environment they were written for ==> dual boot a Windows empty, just for games.

### Post by 5nak3 on 2009-12-11
@Filipek: I only made the comment about dual booting as many PC sold usually have Windows installed and thus allow the user a license to use it. Furthermore, I made the assumption that the given the OP's relatively low post count and previous questions asked they are new to ubuntu which leads me to assume they must have been using another platform before thinking about ubuntu and given that Windows is the most used OS it was a logical assumption that the user has a Windows license somewhere. 

As I result I feel that my inference in using a dual boot was perfectly adequate.


Back on topic however, OP as you seem to have more than enough power to run WoW. Again, my point while it was made given the fact I didn't know your system specs, I still stand by it, for one reason.
Whenever I've tried to use Wine/Crossover/Playonlinux I've found that there are little hangups and little tweaks that are needed to make things work.
I'll state again that because I've never tried WoW I am not an authority on the subject, however I feel that it would be best to have a look at [http://appdb.winehq.org/](http://appdb.winehq.org/) get a feel for the rating people have placed for WoW what workarounds/problems you may encounter. Read some tutorials or watch some youtube videos if any on how to get WoW running on linux.

If you feel confident in doing so go for it, you clearly have the power to do so, and I do not think frames per second will suffer too badly.

But running and installing WoW on windows (if you have that option) I feel will save a lot of hassle, unless of course the installation of WoW on linux is indeed a pain/tweak free matter.
