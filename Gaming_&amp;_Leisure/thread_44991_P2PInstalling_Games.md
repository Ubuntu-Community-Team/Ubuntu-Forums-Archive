### Post by artinla on 2005-06-29
Did you install Cedega from inside P2P?

I know that if you don't have a default cedega version, P2P won't work right.  The best way is to click the "versions" tab in P2P and click "get latest version".

If you have a downloaded version which you didn't get from transgaming, you have to edit a file to enable it.  It was buried in the readme somewhere.

Otherwise, I would look at your CD drive and make sure that it passes the system test in P2P.
