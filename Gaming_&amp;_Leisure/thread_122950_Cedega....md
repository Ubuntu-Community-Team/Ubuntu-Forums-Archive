### Post by danhm on 2006-01-28
Once you have downloaded the demo (if it is compressed, extract it first):

Open a console.
Browse to the directory you downloaded the demo to (use the cd command; "cd directory name") -- you start in /home/username
Type "sh cedega_timedemo_installer". It'll open a nice little installer window.

And that's it.
