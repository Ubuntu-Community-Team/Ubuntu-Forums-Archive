### Post by T.J. on 2016-09-22
Sounds like a bad patch session.  This happens rarely, but can even on Windows.

If you have no games installed, just delete the .steam directory in your home folder, and restart Steam.

If you do, we need to be more selective and back them up first so you don't need to download them again.  You will need to save the steamapp and userdata folders under .steam/steam. Delete the rest of the steam files.  Then restart steam.  Once it is patched, move the steamapp and userdata folders back in place.  Then click install in steam for each game and it will discover the existing files without downloading them again.
