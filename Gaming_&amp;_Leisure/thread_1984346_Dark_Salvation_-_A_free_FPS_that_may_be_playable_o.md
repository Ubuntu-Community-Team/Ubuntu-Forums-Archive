### Post by Sofox on 2012-05-21
Hey guys,
I haven't looked into this yet, but there's this Windows Single Player FPS game that recently became free: [http://www.darksalvation.com/order.html](http://www.darksalvation.com/order.html)

When looking around, I came across this: [http://www.moddb.com/games/dark-salvation](http://www.moddb.com/games/dark-salvation) where in the comments section, a guy asks if he can play it on an updated id tech 3 engine (which of course is open source and available for Linux). The reply from the studio is: "Email us and ask how to do it." (this is back in 2010 by the way)

If I'm reading this right, this means that means that there's a single player FPS out there (which Linux is very short on) which is not only free, but has the possibility of being run on Linux with what should be an only minor amount of configuration.

I am very busy for the next while, shouldn't even be typing this, but if someone wants to play a decent single player FPS on Linux, somewhat similar to Painkiller, give this a look.

By the way, I found out about this studio because they're currently doing a Kickstarter for a [new FPS]("http://www.kickstarter.com/projects/mangledeyestudios/metal-savior"), that has Linux support as an "exceed funding goal" dealie.
