### Post by mjkerpan on 2009-08-10
The binary version of Quake 3 for Linux may well be a collectors item, but ioquake3 plus the Win32 datafiles probably run better on modern systems than the original Linux release would, it would be a silly thing to get unless you are, in fact a collector. If you're just a normal gamer, buying a copy of Ultimate Quake from id, which includes the first three Quake games and costs $30 is a much better deal. Just copy the PK3s off the Q3 CD into the proper folder and you're good to go. Plus you get Quake 1 which is a better game anyway, especially when run in a fancy engine like Darkplaces ;)
