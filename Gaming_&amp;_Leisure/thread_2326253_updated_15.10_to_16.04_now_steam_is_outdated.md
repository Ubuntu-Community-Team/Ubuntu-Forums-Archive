### Post by oldrocker99 on 2016-05-30
Did you install Steam from the Ubuntu repositories?```
sudo apt-get install steam
```
If you installed Steam any other way, you'll run into errors.

Try```
sudo apt-get remove steam
```

Then reinstall the best way, which is from the repos.
