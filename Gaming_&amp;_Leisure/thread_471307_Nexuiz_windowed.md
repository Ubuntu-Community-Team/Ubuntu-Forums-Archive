### Post by jdunn on 2007-06-11
I want to try out Nexuiz.  I installed it from the repositories but when I run it, the screen is black.  I think its because it runs at 800x600 default.  My LCD screen doesn't support 800x600.  How do choose to run Nexuiz windowed from the command line?

---

### Post by hikaricore on 2007-06-11
```
nexuiz +vid_fullscreen 0
```

For further command line options always remember your man pages.  ^_^

```
man nexuiz
```
