### Post by Phobia on 2007-04-07
I have seen some independent companies make games for Linux. I think the biggest problem with getting games on Linux is the DX api. Most  game engines offer only DX rendering and not openGL rendering. The fact that MS has had DX10 is just trying to get people to switch to vista. The only thing that DX10 offers over DX9 is Shader Language 4.0 using HLSL. You can get shader language 4.0 in openGL using GLSL.  The Linux community is growing and i think there is a good market for more game to be made for Linux.
