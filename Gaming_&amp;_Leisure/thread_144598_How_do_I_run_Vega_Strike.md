### Post by kudu on 2006-03-15
[QUOTE=Abhorsen]How do I run Vega Strike on Ubuntu?  I intalled the file using Synaptic, But how do I launch it?  This is my first time trying to install a file, so forgive me if this sounds really, really stupid. . . Also, thank you intelikey for helping me last night on IRC[/QUOTE]

Type in terminal: ./vegastrike

You can also use smeg to create a proper menu launcher.

Applications -> System Tools -> Applications Menu Editor

Click on Games in the left-hand panel and select the New Entry button  below middle bottom of window. Enter whatever you want, with a launch command, for instance ./vegastrike, and it will place a menu item in the Games folder. You can copy the VegaStrike icon from the VegaStrike folder to /usr/share/pixmaps and assign it to your new Vega Strike launcher when you create it. I can't remember where the icon resides, so use synaptic to check where all the VegaStrike files are located and you will find it. You might have to restart X, might not, for it to show.

Sounds complicated, but it's a piece of cake.
