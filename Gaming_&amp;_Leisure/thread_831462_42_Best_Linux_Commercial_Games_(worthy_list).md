### Post by cogadh on 2008-06-18
Neverwinter Nights is a really bad example. The problems it has have very little to do with the fact that you use a Linux installer with a Windows CD/DVD. In fact, the installation itself is relatively easy, but getting the game to work once installed is a different story. The client does not work all that well on certain Linux systems. Most of the major issues have to do with the audio library the client uses being very out of date and the client is incompatable with more recent versions. That and the game's cutscenes don't work at all by default, but can work after performing some Linux gymnastics. That situation would be no different if the Linux installer were included on the NWN CD/DVD, it would still be pulling the same asset files from the disk that the Windows installer does and it would still be installing the same faulty library and binaries.
