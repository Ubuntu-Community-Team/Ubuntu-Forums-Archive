### Post by homerhomer on 2011-08-02
Here's a quick modified version of the minecraft script from the minecraft wiki. I downloaded it and noticed it had some issues.  I updated it .

**Prerequisite**
- You need to have Sun Java installed.  The open source one might work, but on the minecraft website he says, get the Sun one. 

- You also need to have your video card working with opengl or graphics will be very slow. 

** What the script does **

- This script will install minecraft.jar into "/opt/minecraft" folder and then link it to your apps, this way all you need to do is go to apps ( upper left Ubuntu icon, search ) and search for it and then drag it into Unity, sorry only tested it in 11.04 

- This script also takes care of the icon and setting up sound with pulse. 

To install just type: sudo ./minecraft_installer.sh --install-client

Good luck.
