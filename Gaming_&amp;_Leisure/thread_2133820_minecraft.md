Is there any way to play minecraft on ubuntu 12.10 without installing any openjdk file.

---

### Post by Horbo on 2013-04-10
Yes, install Oracle Java 6.  MC doesn't like Oracle Java 7.


```
sudo add-apt-repository ppa:webupd8team/java
sudo apt-get update && sudo apt-get install oracle-java6-installer
```
