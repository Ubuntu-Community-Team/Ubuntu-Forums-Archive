Any help would be greatly appreciated.[/QUOTE]When you installed et, it installed a copy of punkbuster already, in my case it was installed to /usr/local/games/enemy-territory/pb/. I changed to that directory, ran```
sudo chmod +x pbweb.x86
```, then ```
sudo sh pbweb.x86
``` and it updated fine for me.
