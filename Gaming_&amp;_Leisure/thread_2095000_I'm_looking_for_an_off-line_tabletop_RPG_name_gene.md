### Post by thnewguy on 2012-12-16
There is the D&D Character Generator at [http://dnd3rd.sf.net](http://dnd3rd.sf.net)
It will produce random names and the entire generator is contained in a single Java Script file. This means if you go to the website and save the web page using your browser you can use it off-line. The project also has a zip file you can download with all of their code and documentation.
