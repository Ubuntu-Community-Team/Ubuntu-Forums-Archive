### Post by cosborn72 on 2009-07-05
You have several options on Ubuntu.  First, you can choose a game from the Ubuntu repositories.  Here is a list.

[http://www.getdeb.net/category.php?id=3](http://www.getdeb.net/category.php?id=3)

There are lots of other games available for Linux as well, you can browse some here.

[http://www.linux-gamers.net/modules/wfdownloads/](http://www.linux-gamers.net/modules/wfdownloads/)

Your next option is to install WINE, a framework that allows one to run windows programs.  To see what games work with WINE, check here.

[http://appdb.winehq.org/](http://appdb.winehq.org/)

I hope this helps!
