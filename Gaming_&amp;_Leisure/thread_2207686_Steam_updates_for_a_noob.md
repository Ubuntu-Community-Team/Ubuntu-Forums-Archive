### Post by oldrocker99 on 2014-02-24
The line that it wants is a URL, e.g. [http://name.of.archive.com](http://name.of.archive.com)
But this method doesn't automatically add the key. It's better to use a PPA, which downloads and installs the key for you: 

Open a terminal and type:

sudo apt-add-repository ppa:[name of ppa) 
[The one you probably want is  **sudo apt-add-repository ppa:x-swat-updates/ubuntu**]
[ppa stands for private personal archive, and is how you can get the latest and greatest versions of files]
enter your password

Then:

sudo apt-get update

sudo apt-get upgrade

and any newer video drivers will be installed. After video driver upgrades, it's a good idea to reboot.

Let it be known if you have any problems, and good luck!
