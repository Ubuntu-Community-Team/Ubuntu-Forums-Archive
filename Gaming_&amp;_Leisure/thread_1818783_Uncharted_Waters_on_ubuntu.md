A virtual machine is so you can run another operating system in a window essentially, so if you are running Ubuntu you can bring Windows XP or Vista or whatever up in a window, it works 99% the same way in the window and is independent of your host oerating system and everything you do to it only effects the file stored on your hard drive which represents the hard drive of the virtaul machine.

The game recommends Windows XP so for best results then use a Windows XP virtual machine, however this pre-supposes you have a genuine and licenced version of Windows XP to install of course.
