### Post by buttons on 2007-04-16
The open-source Guitar Hero clone [Frets On Fire]("http://fretsonfire.sourceforge.net/") has been updated to 1.2.451!

Some of the changes in the new version:
[LIST]
[*]HOPOs (Hammer-Ons/Pull-Offs) fixed
[*]Memory leaks cleaned up
[*]Bug fixes
[/LIST]

For me, the game no longer crashes while running Beryl (still not recommended, but nice!).  I was able to play for several hours today without a single crash, in fact.

I should take the time to mention this fabulous game works famously with any Guitar Hero-compatible PS2 guitar and a USB adapter.  These are much cheaper (and tons easier to find) than the Xbox 360 X-Plorer guitars.  I have used mine for about a week now with no problems whatsoever.

The full version is available [here]("http://downloads.sourceforge.net/fretsonfire/FretsOnFire-1.2.451-linux.tar.gz?use_mirror=osdn").

and

The update (if you have 1.2.438 already installed) is available [here]("http://downloads.sourceforge.net/fretsonfire/FretsOnFire-1.2.451-linux-update.tar.gz?use_mirror=osdn").

Please do not forget to rock out.
