### Post by endy on 2005-04-19
I've encountered the same problem, hopefully it will be fixed int he upcoming 1.3 patch. In the meantime to load a mod add "+set fs_game" to the command line followed by the name of the folder the mod is stored in.

For example "doom3 +set fs_game lms" for last man standing :)
