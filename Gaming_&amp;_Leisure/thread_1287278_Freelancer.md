Hiya! 

Glad to see others are also playing Naev, just installed it myself a few days ago great game!!!

playdep.net actually does a nice single click install of the game for you if you can't be bothered to go through the hassle.

In any case I'm also glad that 3rdalbum mentioned escape velocity because I also used to play it back in my younger days :)

I found out that you can actually run escape velocity on linux as well though.

Ok, you need EV: Nova (the only installment made for windows if I am correct), and you need crossover for linux, so this isn't free. 

But if you have both of those little gems you can easily play escape velocity nova on linux with relatively little hassle :)

Oh and if you dual boot, install EV: Nova on windows as well, and just have your save file saved to usb stick to continue your adventure no matter the operating system :)
