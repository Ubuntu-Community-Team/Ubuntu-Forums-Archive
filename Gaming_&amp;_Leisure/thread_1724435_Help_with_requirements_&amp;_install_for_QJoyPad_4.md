The [Requirements](http://qjoypad.sourceforge.net/doc/c64.html#AEN66) for QJoypad 4.1.0 and my solutions are (Please review):

Requirement is Trolltech's QT
Solution is libqt4-dev - Qt 4 development files

Requirement is XWindows
Solution is libxtst-dev - X11 Record extension library (development headers)

Requirement is A Linux computer and a Linux-compatible gaming device
Solution is my PC with a Saitek X52 Flight Controls and Logitech Extreme 3D Pro

Requirement is A Linux kernel with joystick support
Solution is **I'm working on it.**
