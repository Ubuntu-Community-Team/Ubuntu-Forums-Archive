### Post by Talikar on 2006-02-28
Install your graphics card's drivers. If it's not working very well it won't run anything at any decent speed (as far as I know of). I am of course "assuming" you have a nvidia card from gt6600.

Anyway, if you don't know how to install them get automatix (link below) and install it (follow the directions) then open it, supply the password (when it asks), then once it's fully loaded check the nvidia box and push the ok/apply/whatever (:P) button. This should install your nvidia drivers for you, reboot/restart your computer and try zsnes again, if it doesn't work then I'm not 100% sure (although trying wine with the windows version may work).
