### Post by N.N. on 2008-05-22
Wolfenstein: Enemy Territory has a large player base, see the following statistics: [http://www.onlinegamingzeitgeist.com/games/](http://www.onlinegamingzeitgeist.com/games/).

If you install it, though, make sure either to stick with the original version (2.55) or update the game to the latest version (2.60b). If you just download the 2.60 installer and stick with it, you'll find yourself exposed to major security vulnerabilities and unable to connect to a lot of servers.
