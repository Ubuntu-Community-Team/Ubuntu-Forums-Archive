I believe you mean the Gamers Arena. if that is the case, then I can confirm that the vast majority of games mentioned there should work without problems. use the installation guides if in doubt. now Windows games, like Doom 3, need Wine to work. and I don't mean you need to get them drunk. it's a program for running Windows software. the compatibility is not very high, but Doom 3 is confirmed to work. see [http://appdb.winehq.org/](http://appdb.winehq.org/) to check if a given Windows program will work.

### Post by Perfect Storm on 2008-06-08
Note, some Video cards are better for gaming on linux.

1. Nvidia - best support, best performance
2. ATI - expect some hassle to get games to work.
3. Intel - more or less forget if the game require 3D acceleration.
