### Post by oldrocker99 on 2019-06-15
It is likely that that folder is hidden. In *NIX, all files that are hidden start with a dot, like .this. That will not show up normally.

In whatever file manager you use, open that folder and press CTRL-H. That should make the folder you're looking for show up and say howdy.
