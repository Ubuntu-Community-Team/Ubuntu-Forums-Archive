### Post by eragon100 on 2008-04-25
<snip>

You have to disable the internet with the network icon in the system tray every time you want to play, because if you have internet, the game contacts the authorization server to verify the key. It will offocurse give an error saying it's already in use. Right-clicking on the network tray icon and disabling the network connection with a single mouse click solves this problem immediately. After you have played, simply reactivate internet by right on the same button in the network icon as you did when you started playing. Your internet connection will be reastablished in about 2 seconds.

Have fun! :)
