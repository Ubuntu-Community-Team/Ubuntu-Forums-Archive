### Post by ckovach on 2010-05-10
The problem is an ambiguity in how the question is worded. The part of the sentence that's modified by "only" is unclear.  If by "you only smile when you whistle" it means "the only facial expression you make when you whistle is a smile", then answer C is correct. On the other hand if it means "the only time you smile is when you're whistling," then D is correct. The second interpretation seems more natural to me, so I agree with Pistolpete2, that D is  the better answer, though it's ambiguous.

Interpreting it this way, the question says happy=> whistle and smile=> whistle from which answer C, smile => happy, clearly doesn't follow (because  happy => whistle doesn't imply the converse whistle => happy).

On the other hand, if we replace *only* with *always*, which unambiguously implies whistle => happy, then the answer would be C. I suspect this is what the author of this question intended to say.

### Post by Big_Croc7 on 2011-02-11
NO NO NO NO! ARGH!

Sorry.

It doesn't say 'you smile if you whistle', it says 'you only smile when you whistle'.  This is VERY DIFFERENT!!!

It does not mean you smile whenever you whistle; it means that, if you are smiling, you must be whistling.

In more symbolic terms, the second part of the statement can be written as

smile -> whistle

NOT as

whistle -> smile

You only smile if you whistle; so if you are smiling, you must be whistling.  But the reverse does not hold, i.e. if you are whistling you need not be smiling.  The 'a -> b' symbol means 'a is true implies that b is true', not 'b follows on after a'.

(Of course, this is neglecting the whole business of actually being able to do both at once, which I agree is rather difficult - I've just tried it.)
