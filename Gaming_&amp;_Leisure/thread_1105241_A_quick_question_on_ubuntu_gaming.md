### Post by hikaricore on 2009-03-24
Linux/Ubuntu gaming is far from being in its infancy but many game developers do not release Linux ports of their games.
Some virtual machines support 3d rendering which is almost always required for modern games but it's not quite at the level of being able to play anything you please.
If you absolutely must have the very latest version of GameX which relies on Microsoft's vendor lockin (DirectX) dual boot or have a seperate machine for Windows gaming.

Be sure to check the [WINE Application Database]("http://appdb.winehq.org") to see if the games you need can be run through WINE.  
And post any WINE related questions here: [http://ubuntuforums.org/forumdisplay.php?f=313](http://ubuntuforums.org/forumdisplay.php?f=313)
