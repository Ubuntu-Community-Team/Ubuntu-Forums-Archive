### Post by azraelck on 2007-07-18
I've wanted to put out my own CRPG since I was about 7 years old. I'm actually working on it now; trying to learn to program in C++ as well as doing the design work. It's actually fun seeing it come together on paper, to the point that if I could program I'd have the core mechanics worked out within a week or two. I've got the setting done as well, but I have to draw up the maps and so on, and detail it to make it come to life. 

I don't know if I'd want to do it for someone like EA or some such, because I know that bean counters, idiots, and what have you would ruin any good work done by anybody on the project (not just me!). But then it may be just as fun, so long as I'm allowed to do my job. Not having done it commercially, I wouldn't know.
