### Post by svbh07 on 2008-05-09
I was bored, so I whipped this up over the course of an hour or so. Yes, it's terrible that it took me that long.

A bit of an explanation: I'm big on D&D. I've got every rule book and adventure module I could get my hands on. All in all, there are over 20 gigs worth. Anyways, my group hasn't met for too long and I was getting withdrawal symptoms. Today was the last day of class before finals, so I whipped this up.

Notes on the program itself: This program was started using pico in a terminal and finished with Notepad++ in Wine. It was coded in Ada95 and compiled using Gnat 4.2. I've got a 32-bit Intel Celeron laptop running Ubuntu 8.04. A binary as well as the source is included in the archive if any prospective downloaders can't run it for some reason. The source, including empty lines and the GPL info at the beginning, is 150 lines. Yes, I GPL'ed a <150 line program. Have fun. If you find any bugs, let me know. The program is rough, but as I said, I made it in a short time. It's not exactly optimized, either.

Stupid me, I forgot the most important part! This program will generate a number of die rolls of the user's choice. It can generate anywhere between and including 1 and 100 d4, 6, 8, 10, 12, 20, or 100 rolls.
