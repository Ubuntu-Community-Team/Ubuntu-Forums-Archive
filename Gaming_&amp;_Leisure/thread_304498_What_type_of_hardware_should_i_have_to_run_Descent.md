### Post by Peepsalot on 2006-11-21
I would think that 800mhz should be plenty power to run Descent.  

To me, it would seem a waste of money to upgrade your computer only to run DOS games.  There must be a better way.

Maybe there is something you can tweak to improve efficiency.  I have never tried dosbox so I can't say much about that.  

Do you have proper video drivers installed in Ubuntu?
Have you tried other emulators like [bochs](http://bochs.sourceforge.net/)?
Have you considered setting up a virtual machine running DOS in VMware or similar?  Maybe there is something that can handle it better than DOSbox.

And of course, you could always try running DOS natively by giving it it's own partition, and selecting DOS from the GRUB boot menu. ;)
