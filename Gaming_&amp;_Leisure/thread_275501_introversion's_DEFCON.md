They are going to release a GNU Linux port, I think in the form of Linux binaries that will require the Window's files to work, but in the mean time, it works almost flawlessly under WINE in Ubuntu! I was shocked to discover, based on previous WINE experience. But it's gotten a lot better!

It crashes every time I try to JOIN a game, but if I start one it runs fine. And there are lots of players since it's so new, so people hop on new games fast.

Check it out!
