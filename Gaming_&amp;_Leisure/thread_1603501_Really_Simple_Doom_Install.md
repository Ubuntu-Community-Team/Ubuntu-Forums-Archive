### Post by ronnielsen1 on 2010-10-22
It depends on the version number of doom you have. If it's the dos version it will play happily in dosbox. If it's a later version wine should take care of it. Do you have the cd, floppies or whatever or just the wads?
Here's other ways of installing it

[http://www.gamers.org/dhs/helpdocs/Linux-DOOM-FAQ.html](http://www.gamers.org/dhs/helpdocs/Linux-DOOM-FAQ.html)

> In fact there is several ports of doom in the ubuntu repositories :

freedoom - free game files for the 3D game DOOM
lxdoom - Unix port of boom, an enhanced version of DOOM
lxdoom-sndserv - sound effects server for lxdoom
lxdoom-svga - svgalib binary for lxdoom
lxdoom-x11 - X binary for lxdoom
prboom - clone of the legendary first person shooter Doom
doom-wad-shareware - Shareware game files for the 3D game DOOM
doomlegacy-data - Data files needed by doomlegacy
doomlegacy-sdl - A port of the Doom engine that supports OpenGL
doomlegacy-x11 - A port of the Doom engine that supports OpenGL
