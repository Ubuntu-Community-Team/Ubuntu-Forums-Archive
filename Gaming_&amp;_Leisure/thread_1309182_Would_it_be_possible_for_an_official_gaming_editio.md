### Post by MaximB on 2009-11-01
There is a gaming distro called Linux-Gamers : [http://live.linux-gamers.net/](http://live.linux-gamers.net/)
It's a live cd full of free Linux games.

But proprietary games is a whole different issue...
There are many small and big games for Linux and there is no one place to get them all like with steam.
There are a few Linux game shops like TuxGames and LGP, but they don't cover all games and they don't sell games that only have download versions (they do sell download versions but for games that also have a box versions), so they do miss quite a lot of good games.

I think it would be good to have one place to buy them all - but it's a huge project to make, and even bigger it would be to integrate it into a distro.
