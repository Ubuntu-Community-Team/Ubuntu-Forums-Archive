[SIZE="3"]**Here are some sites that have a ton of games for Ubuntu**[/SIZE]. (I personally like PlayDeb Repository)

[http://www.playdeb.net/](http://www.playdeb.net/)
Here is a list of games in the repository
[http://www.playdeb.net/available_games.html](http://www.playdeb.net/available_games.html)

A list of almost 2000 games for Ubuntu (site is currently down for maintenance, check back)
[http://www.playubuntu.com/](http://www.playubuntu.com/)

Ubuntu Gamer Arena
[http://gwos.org/doku.php](http://gwos.org/doku.php)

There are a lot of good games that run on Ubuntu.
**[SIZE="3"]12 good Linux games[/SIZE]**

**#1 Tremulous** (tremulous.net)
A team based first person shooter (FPS). Tremulous is free to download and play.
In Tremulous there are two teams: Aliens and Humans. Aliens can have such awesome feats as wall-walking (dizzying to say the least), pouncing from great distances, and spraying poisonous gases. Humans can obtain advanced weaponry and armor, and plant strategic devices such as turrets to defend their base.
Tremulous features a unique upgrade system, where players from each team must make a certain number of kills to gain new upgrades (aliens can evolve into more powerful aliens, and humans can obtain better items and gear).

Install:

```
 sudo apt-get install -y tremulous
```

**#2 Gridwars **(caiman.us/scripts/fw/f2317.html)
A retro top-down-shooter game, but any assumption that it's an old game is evaporated once you see the incredible liquid visuals... do try this one out!

Download and install from: [http://dlx.caiman.us/pageC2.php?nr=2...GridWars38.zip](http://dlx.caiman.us/pageC2.php?nr=2...GridWars38.zip)

**#3 VDrift** (vdrift.net)
A racing game with a focus on drifting. It's only interesting to players who enjoy this genre. Warning: It's only fun if you use joystick or gamepad!

Install:

```
apt-get install vdrift vdrift-data
```

**#4 World of Padman** (worldofpadman.com)
This is a cartoon-like first-person shooter which combines fast gameplay with vibrant colours. It uses the Quake3 engine, which is well supported by current hardware.

Install: padworld.myexp.de/index.php?files

**#5 Nexuiz** (alientrap.org/nexuiz)
This is a great Quake3 mod while using some of the coolest weapons since Half Life 2!

Install: alientrap.org/nexuiz/index.php?module=downloads

**#6 Scorched 3D** (scorched3d.co.uk)
Similar to the old Missile Defense games. But... using a 3D graphics world, network play and terrain interaction. In same cases it&#8217;s just more fun to nuke a path to your enemy and sink their base into the ocean... if you're into this kind of thing.

Install: [http://ubuntuforums.org/showthread.php?t=39655](http://ubuntuforums.org/showthread.php?t=39655)

**#7 Glest**(glest.org)
A great real time strategy game (RTS) in 3D , has some great graphic features and good imagination! Check it out.

Install: glest.org/glest_board/viewtopic.php?p=8976

**#8 Flight Gear**(flightgear.org)
Free flying and relaxing. Sometimes you just need to kick back and enjoy the breeze.

Install:

```
sudo apt-get install flightgear
```

**#9 Battle for Wesnoth** (wesnoth.org)
A turn-based strategy game with a fantasy theme. Fight a desperate battle to reclaim the throne of Wesnoth, or take hand in any number of other adventure

Install:

```
 sudo apt-get install wesnoth -s
```

**#10 RTCW Enemy Territory **(planetwolfenstein.com/enemyterritory)
Return to Castle Wolfenstein: Enemy Territory (full install) - Wage war in team-based combat. Wolfenstein: Enemy Territory is an online multiplayer game, where the players interact with each other over a network in two teams (axis and allies) to defend or destroy mission objectives.

Install:

Download from: ubuntusoftware.info/scripts/et-installer.sh

**#11 Alien Arena 2007 **(red.planetarena.org)
A great FPS, aliens, evils and more. Keep shooting, don't stop, run ! This is a standalone and multiplayer deathmatch.

Install:

Edit the file /etc/apt/sources.list to include the line "deb [http://ubuntusoftware.info/](http://ubuntusoftware.info/) feisty all" repository .


```
sudo apt-get update

sudo apt-get upgrade

sudo apt-get install -y --allow-unauthenticated alienarena2007
```

**#12 Action Cube **(splashdamage.com)
ActionCube is a total conversion of FPS called Cube. Set in a realistic looking environment, as far as that&#8217;s possible with this engine, while gameplay stays fast and arcade-like. This game is all about team oriented multiplayer fun.

Install:

```
sudo apt-get install -y actioncube
```
