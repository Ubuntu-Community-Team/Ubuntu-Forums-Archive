### Post by Apex_G on 2008-09-25
Hi, I know exactly what you're talking about. When I originally made the switch to Ubuntu/Linux; I found most characters didn't work, and some screen-packs and levels are configured only for Windows. 
Because the majority of Mugen players use Windows, it puts us at a disadvantage. 

If you want to try and use you Screenpack, then put the downloaded-folder inside the 'Data' folder. The open up Data/Mugen.cfg (open the file as text).
Then look for the following, and add accordingly:

Motif = data/[COLOR="Red"](Put screenpack file name here)[/COLOR]/system.def

Then save and exit. Try running mugen, and if it doesn't start......contact me. If your feed up, and just want a Linux version pre-tweeked, I could e-mail you my 
'old-school Street Fighter 2' mugen remake. It runs fine, but can sometimes mess with the sound. 

p.s. Hope this response isn't to late :)
