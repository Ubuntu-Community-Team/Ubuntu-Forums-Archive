### Post by Spastic on 2007-05-30
i think Game Guard is the major problem setting up maple story without a virtual machine of some sort.  its a root kit that check all your  running processes and such to ensure you aren't cheating. but i think the only way for game guard to come back with the proper results is to actually run it from windows (virtualized or native boot)

i used the guide mentioned already to help me setup maple story,  the ram allotment for the virtual machine is a variable you have control over - you could try it with only 256m allotted, the game doesn't  much.  i give mine 768 since i have more to play with, if you are experiencing slowdowns try playing with your Ubuntu screen resolution I'm not sure why but it seemed to be the difference for me.
