### Post by themerion on 2007-07-22
Speed in Starcraft is a deeply rooted wine-issue. If you check wine's application database, it says that Starcraft needs a DIB-engine in order to be faster. There's been some fuss about it, but nobody has really made one yet, and it seems to be an issue which has been around for some years (!)

However, one of Wine's summer of code projects this year is just to create a DIB-engine. We'll see how it goes...

Wine's bug page for the DIB-engine: [http://bugs.winehq.org/show_bug.cgi?id=421](http://bugs.winehq.org/show_bug.cgi?id=421)
