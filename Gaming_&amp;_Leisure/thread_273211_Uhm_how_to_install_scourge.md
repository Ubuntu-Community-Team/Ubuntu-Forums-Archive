### Post by francisco_athens on 2006-10-18
If you just want to use the package installer:
First:
```
sudo apt-get install libfreetype6 libgl1-mesa libglu1-mesa libsdl1.2debian libsdl-mixer1.2 libxmu6
```

Then get the .package file [HERE]("http://sourceforge.net/project/showfiles.php?group_id=98006&package_id=104986")

Execute the package (you don't have to be sudo) to install and have fun!

if you want to build the latest SVN source, I've made (very Ubuntu biased) instructions on my wiki:
[HERE]("http://herenaforge.org/tiki-index.php?page=SCOURGE_Building_From_Source")

I would recommend the CVS build over v.15 as there have been lots of great [fixes and enhancements]("http://scourge.sourceforge.net/contribute.html") since.
