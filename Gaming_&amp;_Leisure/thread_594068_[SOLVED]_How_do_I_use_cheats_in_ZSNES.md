### Post by mysticmatrix on 2007-10-27
How do I use cheats in ZSNES on Linux?
In Windows, I can put .CHT file in same directory by same name as that of ROM file. This doesn't appears to work in Linux.
Any help?

---

### Post by FranMichaels on 2007-10-27
Sure.

Drop the .cht file in ~/.zsnes

In nautilus press ctrl+h to show hidden files and folders
or use the view menu.

.zsnes

has all your saves, screencaps, etc. Very handy for backups and the like.

Make sure the name matches the rom (like you said ;)), and you'll be all set. :KS
