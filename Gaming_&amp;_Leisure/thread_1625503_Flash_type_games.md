### Post by theraje on 2010-11-19
If you can't get Flash on your system, you're a bit SOL.

Theoretically, you could download the .swf file (Flash's "exe"-equivalent extension), and use a disassembler to get the code.... but then you would have to convert the code to some other language, which would take a lot of time.

So yeah. If Flash won't play nice on your system, you're not going to get far fast.

Looks like those games are distributed as Windows Flash Projector files (exe).

The flash player is bundled with flash projector games, so all you need is [wine]("http://en.wikipedia.org/wiki/Wine_(software)") to load the Windows exe executables in your Linux system. 

You can install Wine from the Software Center.
