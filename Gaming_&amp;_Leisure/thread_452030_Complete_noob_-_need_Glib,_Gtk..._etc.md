### Post by benmoreassynt on 2007-05-22
I think GTK and all those things MUST be installed already if you are using Gnome in Ubuntu, so the problem is elsewhere.

You should install things using Synaptic Package Manager - don't try to do it yourself. Synaptic is really easy to use, and is the way to install almost all software in Ubuntu. There is no equivalent of the Windows install.exe method, but Synaptic is much better once you are used to it. Find out about adding repositories to Synaptic, and take it from there.
