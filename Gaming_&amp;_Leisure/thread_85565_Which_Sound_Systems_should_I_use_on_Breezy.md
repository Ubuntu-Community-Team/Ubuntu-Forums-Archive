### Post by fragmental on 2005-11-03
I use ALSA, but on some cards it can be nicer to use a different sound system because the card or chipset doesn't support hardware mixing or multiple sounds or some other reason.  You might get better answers about sound in a different forum though...like "video and sound" [http://ubuntuforums.org/forumdisplay.php?f=114](http://ubuntuforums.org/forumdisplay.php?f=114).  Also, make sure you check the help or the wiki or the unnoficial ubuntu guide.  

My sound works fine in ut2004.  I think it always worked fine, but I did do some odd mojo with my sound system in hoary.  I didn't touch the sound system when I updated to breezy.  I think some of the settings were reverted, but I haven't had any problems so far.  ut2004 sound still works.  You might also try getting the openal library from the repositories, but I doubt that really matters.  Won't hurt.  There are also some sound settings in the game you can change.
