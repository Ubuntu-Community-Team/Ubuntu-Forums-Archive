### Post by grahammechanical on 2014-01-15
What is Myst? Oh, it is a game. I do not play games. Has this game been ported to Linux? Has it been packaged in the Debian package format that Ubuntu uses?

[http://en.wikipedia.org/wiki/Myst](http://en.wikipedia.org/wiki/Myst)

If a game has not been ported for Linux then it will not run natively in any Linux distribution. Sorry. If we are in possession of a legal Windows version of the game then we can install, through the software centre, something  called wine. 

[http://www.winehq.org/about/](http://www.winehq.org/about/)

How well does a Windows application run in wine and so on Ubuntu? That all depends. Search the wine database for information about the application and see the ratings that are given.

[http://appdb.winehq.org/](http://appdb.winehq.org/)

As for Myst you will see something like this.

[http://appdb.winehq.org/objectManager.php?sClass=application&iId=4264](http://appdb.winehq.org/objectManager.php?sClass=application&iId=4264)

There are other results for Myst in the wine AppDB.
