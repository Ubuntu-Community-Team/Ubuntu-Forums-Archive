**LAST UPDATE:** 2012-01-31

[_Video_]("http://www.youtube.com/watch?v=hwYpyY7zhTI&feature=player_embedded")

**Changes since 0.8:**

[LIST]
[*]Completely new environment for the playfield.
[*]Support for shaders in materials of tiles and scene.
[*]The possibility to select the environment and the tileset separatly.
[*]Support for kmahjongg-1.1 layout format with non-standard sizes.
[*]A new, more user-friendly game settings interface.
[*]Dynamic language switching. French localization.
[*]Installer for the Windows version.
[*]The layout editor with support for kmahjongg-1.1 format.
[*]Three new music tracks in the additional music archive
[*]And a lot of bugfixes.
[/LIST]

**Install packages:**
1. Add the following to your **/etc/apt/sources.list**:
**deb [http://packages.opengamestudio.org](http://packages.opengamestudio.org) stable ogs**
2. Refresh APT list of packages:
**apt-get update**
3. Install the game:
**apt-get install ogs-mahjong**
4. Install additional music for the game:
**apt-get install ogs-mahjong-music-addon**
5. Install layout editor if you wish to edit/create your own Mahjong layouts:
**apt-get install ogs-mahjong-layout-editor**

**Source code:** _[Mahjong Sources]("http://sourceforge.net/projects/osrpgcreation/files/Mahjong/0.9/ogs-mahjong-0.9-src.tar.bz2")_

Packages for other Linux distributions and Windows can be found at our **homepage**: _[http://opengamestudio.org](http://opengamestudio.org)_
