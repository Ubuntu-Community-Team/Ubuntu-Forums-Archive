### Post by Coldbird on 2006-12-09
*RESOLVED*
For the Community... So they also know how the Bunny jumps...

1. Get yourself a the Xploder-Cheating Device for PSX (I tried others... but they didnt work...)
2. Insert the Disc into your Comp and copy the File PSX.EXE onto your Harddrive... (Do not make a ISO of the Disc and try booting it... THAT WONT WORK)
3. Switch your GPU Plugin to SoftX... as its the only one capable of properly displaying the Xploder Menu... Weird...
4. Start ePSXe and choose to load PSX.EXE Files... Then choose the previously copied PSX.EXE... The Xploder should boot...
5. Choose to turn your Second Memory Card into a X-Key Card... (This Memory Card will be rendered useless as it stores your Cheat-Data...)
6. Add / Edit / Choose Available Cheats and return to the Main-Xploder Menu and choose to start the Game... A Screen should popup telling you to exchange the Game-CD... At this Dialog press F1 to create a Savestate...
7. Restart ePSXe and revert your GPU Plugin to the Plugin you really wanna use In-Game...
8. Repeat Step 4
9. Press F3 to revert to the Savestate you created Last Time... Press ESC afterwards to return to the ePSXe GUI.
10. Choose to swap Discs and there ISO / CDROM... depending on where you got your Game stored... You should automatically return to the "Game"...
11. Press X (even if Screen is black... thats because of the GPU plugin being unable to render the Xploder Menu properly...) and the Game should boot with the Cheats applied...

-- Sidenotes about this Way of Cheating --
Unlike with the Use of PEC (The Cheating Device available for Windows Users...) - This Way of cheating is done in the PSX RAM... meaning that if you got Cheats activated and you create a Savestate while playing... The Cheats also will be saved...
This can be useful at some Points... but also deal Trouble at other Points...
Good Points would be that - if you use Savestates... - you will always have your Cheats activated :-)
Bad Points could be - for Example - in Breath of Fire 4 if you got a Unlimited HP Cheat saved in PSX RAM...
Cause some Battles you are ment to loose in BoF4... and if you got unlimited HP you cant loose the Battle... and therefore have a neverending Battle stopping you from advancing in the Storyline...
Ways to avoid this Problem is, for example... Doing Backup Saves using the Normal Memory Card Emulation... cause with Memory Card Emulation the Cheats do not get saved... So... if you really have to get rid of the Codes you would simply load the Savegame the normal... (old) way... by loading them from the Virtual Memory Card :-)
But keep in Mind that you have to boot the Game without a cheat-activated Savestate and without Xploder Pre-Booting if you want to get rid of the Codes...
