I haven't posted about new games for a long time now, that is because I have finally found a job and don't have much spare time now.
My apologies.
But now and then I come across a game that you might enjoy and don't know about so here you go... I give you...Nord ! 


Nord is a free (as beer) java based massively multiplayer online game.
In Nord you can build your own villages and visit the villages of other players.
You can also buy "premium" buildings (in order to support the developers and for prettiness, it does not give you any privilege over other players).
This game is for interaction between players, chatting and making friends.
I didn't have much time to check this game more so if you play I hope you give us more info about it.
It also has a mobile client.
