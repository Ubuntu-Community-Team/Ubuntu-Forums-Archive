### Post by AdamShumpisxXx on 2009-03-01
You could just boot the emulator up without running a game. This will load up the PSX BIOS you are using (more or less the Playstation "Desktop"). Then just navigate to the memory card just like you would with a real Playstation and delete the saves you want. Unless however you are talking about save states exclusive to that emulator in which case try the previous instructions given by _Svensk023_.

### Post by AdamShumpisxXx on 2009-03-01
It's a ZIP file containing ZIP files for all the best Sound Plugins for ePSXe. Extract all the ZIP files onto your Desktop. Choose one to try (example = spuNull2_135.zip) and then extract the DLL file from it into your Sound Plugins folder of your ePSXe install. Open up ePSXe and then change your preferences to switch from the Sound Plugin you're using to the one you just installed. If that doesn't work then try another one by repeating the process with another one of those ZIP files. If none of them work then get rid of ePSXe and get a better emulator. Hahaha.
