### Post by holomorph on 2006-04-08
yes of course you can.  But you can do better if you really need all your resources for running the game.  When I want to run warcraft III under wine (or even native games like enemy territories) on my old desktop PIII 450, I log out of gnome and log into a session just running openbox.  No taskbar or any of that stuff I don't need (since it's gonna be running full screen anyway), Really works nicely.  I suspect if I had XP on this machine and tried to run Warcraft III, it would probably even work less well, though I don't have a lot to base that on.
