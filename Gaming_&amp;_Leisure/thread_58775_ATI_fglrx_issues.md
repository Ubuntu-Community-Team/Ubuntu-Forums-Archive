### Post by evilghost on 2005-08-21
Was easy as cake for me, before I moved to an NVIDIA 6800GT.  I don't think you mentioned the card you're using.  Are you sure your card is supported by the Catalyst drivers (see [http://www2.ati.com/drivers/linux/linux_8.16.20.html)?](http://www2.ati.com/drivers/linux/linux_8.16.20.html)?)

From what I remember, it was as easy as:
```

sudo apt-get install build-essential
sudo apt-get install linux-headers-`uname -r`
cd /tmp
sudo wget http://www2.ati.com/drivers/linux/ati-driver-installer-8.16.20-i386.run
sudo killall gdm
sudo sh ./ati-driver-installer-8.16.20-i386.run

```
