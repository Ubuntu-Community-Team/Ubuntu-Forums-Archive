### Post by seethru on 2005-10-13
according to the unofficial cedega wiki, AoE 2 works but with quite a few issues that can be rather annoying.

> Tels: The demo installs fine. It also runs okay, but has some issues with the fonts (almost unreadable), is quite slow, and has black areas in briefings. In addition, the demo does not let you change the resolution, so I could only test it on 1024x768.

Thode: Tested on Cedega 4.1.1. The game installs, and patches with 2.0A properly, but fails to start. Using a nocd patch get's the game running, with the same issues as Tels mentions that the demo has. Changing resolution results in a picture freeze, and infinite sound looping.

However, 4.1.1 is fairly old and it's possible some of the issues have been resolved since then. I unfortunately don't have a copy of AoE 2 that I can try for you.
