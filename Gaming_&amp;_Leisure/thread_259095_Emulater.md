### Post by hikaricore on 2006-09-17
you need to tweak the configuration first.  and start with a simple game like mario 64...  assuming you own the cart for that one ;)

start by disabling the audio and only working with the video, this keeps the guessing to a minimum on what is crashing the games.  also atleast for me mupen seems to crash if I try to run more than one game in the same session.  so if it does this to you, after running and stopping a game just exit and relaunch.

anyway once you get decent fps and such working with mario64 and get your sound working, then and only then should you move on to a more advanced game like zelda.  I believe there is a special video mode for the zelda games which is only supported by a couple video plugins.  so look into it before flipping out here that emulator you didn't pay for, & the games you don't own, aren't working correctly.
