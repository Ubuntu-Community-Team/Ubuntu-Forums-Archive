### Post by Cappy on 2007-07-25
Hi, thank you for your detailed post. Geforce 4 is only supported by the nvidia legacy so you should
```
sudo apt-get --purge remove nvidia-glx
```
and
```
sudo apt-get install nvidia-glx-legacy
```
(reboot or alt+control+backspace) and see if that fixes your problems.
