It depends, you may have to kill your gnome bar (Which will restart it) or just go to Applications -> Games it's most likely there (In a native form, I'm not sure if wine puts icons there as diablo is there and I have it installed) so if you're running wine, you should just go to the game directory (somewhere in ~/.wine/drive_c/Program\ Files/) directory and execute the exe file using "wine somefile.exe" without the quotes in the console.
