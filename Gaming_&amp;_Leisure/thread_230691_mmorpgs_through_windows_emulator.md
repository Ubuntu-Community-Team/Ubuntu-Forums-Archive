### Post by AngryKidJoe on 2006-08-06
The problem with running a windows emulator is that the emulation layer eats up performance on a good many games. I play World of Warcraft using Wine ( [http://www.winehq.com](http://www.winehq.com) ) and I only run at about 50-70% of what Windows puts out.

But, I'm working on tweaking settings to see what I can find to make it run faster. Even have a Linux WoW Guild (see sig). But the way I see it, you have two choices: Bite the bullet and go with Micro$oft; have the games run the way they're supposed to but support a monopoly (HAVE YOU READ ABOUT VISTA?!) or bite the bullet and run linux, support opensource - and risk the games running slower.

At the moment I'm dual booted. Depending what I'm doing when playing WoW, that's how I choose my OS. At least until I get WoW sorted out on Linux.
