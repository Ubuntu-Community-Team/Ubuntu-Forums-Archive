### Post by Jeremy23 on 2007-01-21
Yup, just copy-and-paste the following into a terminal (Applications &#8594; Accessories &#8594; Terminal):

```
et +set fs_game tcetest +set com_hunkmegs 256
```

Then, you should be able to run the game. Once you've run the game with the terminal like that, it should also run fine when you access it from the Applications &#8594; Other menu.
