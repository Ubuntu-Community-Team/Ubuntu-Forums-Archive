I googled and it has been ported to run under cygwin ( [http://www.mail-archive.com/cygwin-ports-general@lists.sourceforge.net/msg00552.html](http://www.mail-archive.com/cygwin-ports-general@lists.sourceforge.net/msg00552.html) )

Download the setup.exe, works like a package manager. It's been awhile since I used it, so you are on your own there in installing and running stuff...
