### Post by LinuxFox on 2009-06-11
Doom, as in the original game?  You could try a source port like PrBoom.  PrBoom is under Add/Remove.  It's been a while since I used it via the repository, but I think the IWAD file (DOOM1.WAD, DOOM2.WAD, etc) needs to be in the home folder.  

I have PrBoom compiled to my home folder to play Freedoom, so the IWAD is located in the "/share/games/doom" directory.  I'm just remembering from the when I first tried it using Add/Remove.

Edit: I just tried the "-IWAD" command with the shareware Doom and it works for me.  Install PrBoom and create a shortcut to PrBoom and add "-IWAD /home/user/DOOM1.WAD" without quotes to the launcher, replacing "user" with your Ubuntu username. Make sure the IWAD file is correct.

### Post by noerrorsfound on 2009-10-17
Freedoom isn't an actual clone of Doom. The Doom engine source is used but all assets (graphics, levels, sounds) are different. Therefore, it's fine legally.

You can still buy Doom from id Software as a downloadable copy: [http://www.idsoftware.com/games/doom/doom-ultimate/](http://www.idsoftware.com/games/doom/doom-ultimate/)

I don't think John is worried about losing any money from people playing Freedoom instead of buying Doom. He probably thinks it's a neat project.:P I think $20 is a lot for the original game, but the real Doom is still better than Freedoom.
