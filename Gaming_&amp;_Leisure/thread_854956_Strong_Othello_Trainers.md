### Post by vprasad on 2008-07-10
Are there any strong othello trainers compiled (ie. readily available .DEBs?) for/compatible with Hardy? I know some of the stronger trainers are Windows based and I have mentioned them here, but I'm looking for native Linux trainers, so I'm making this post in the Gaming/Leisure forum rather than the WINE forum.

As far as I've found, there's:

Strongest-- Benedetto Romano's SAIO ([http://www.romanobenedetto.it/Saio.htm](http://www.romanobenedetto.it/Saio.htm)) which is a Windows app and is just a demo (trialware? WINE compatible? Any native Linux variants?)


Chris Welty's Ntest ([http://othellogateway.com/ntest/Ntest/index.htm](http://othellogateway.com/ntest/Ntest/index.htm)) which is a Windows app but has source code available (WINE compatible? Is there a native compiled Linux version?)


Richard Delorme's Edax ([http://abulmo.club.fr/edax/index.htm](http://abulmo.club.fr/edax/index.htm)) which has a native Linux version.


Gunnar Andersson's LZebra ([http://radagast.se/othello/download3.html](http://radagast.se/othello/download3.html)) which works on an older version of RedHat, but requires tweaks and library mods.
