### Post by blueturtl on 2007-05-11
Can you post your hardware specs?

SLI'd Geforce cards suggest your system is fairly new, so maybe it could be a hardware support issue. Enemy Territory is a more demanding game so my bet is that it either there is a bug in some device driver, likely the video ones that is causing the lock-up, or something in your hardware isn't functioning properly (bad RAM, or bad mobo).

Also are you running a 3D-accelerated desktop (Compiz or Beryl)? I see tons of threads on the forums about Beryl causing lock-ups. Compiz causes some as well, but seems to play nicely with native apps. If you are running with desktop effects you could try disabling them.

Does the game crash every single time you play, or is there variation on the time or place of crash?

edit: If a crash-happens you can try hitting ctrl+alt+backspace to see if it's only X that has crashed (this will restart X). If that doesn't work then the system probably does need to be hard-reset...
