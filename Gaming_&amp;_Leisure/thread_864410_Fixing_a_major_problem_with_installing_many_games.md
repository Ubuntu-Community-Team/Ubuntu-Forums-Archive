### Post by z00s on 2008-07-19
If you have any problems installing any old Loki Games, id Software games, Garagegames games, etc. try installing libgtk-1.2 from synaptic and see if that corrects the problem.  Installing it fixed over a dozen games for me.

You are running a x86 machine with glibc-2.1
Operating System: Debian GNU/Linux (or

Is the typical error message you'll receive if you're trying to install any of those games without gtk-1.2.
