Lucas chess is a super chess app that has tons of features. It has been available for a while using wine or tweaking a linux install with variable chance of success. Now just download the linux sh file and type 

```
bash LucasChessR130a_LINUX.sh

```

download it link below:

[https://lucaschess.pythonanywhere.com/downloads]("https://lucaschess.pythonanywhere.com/downloads")

then just set your language and go. It even gets in your menu-games automatically
