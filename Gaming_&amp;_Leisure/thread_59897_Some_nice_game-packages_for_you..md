It's not a matter of difficulty. Unless marked explicitly so, random deb packages on the Internet tend to be for Debian, and can have bad effects on Ubuntu. Debian and Ubuntu use the same format for packages, which is what leads to the misconception that it's safe to run all debs on Ubuntu.
