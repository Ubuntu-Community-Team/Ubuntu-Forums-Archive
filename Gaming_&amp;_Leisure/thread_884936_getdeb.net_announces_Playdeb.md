### Post by Vadi on 2008-08-09
"getdeb.net, a leading provider of new and updated programs for Ubuntu is announcing the start of a gaming repository for Ubuntu. The repository (codenamed "**Playbuntu**") will provide all games available on getdeb.net in an easy to use repository format - enabling you to receive updates for your favourite game via the Update Manager tool."
[B]
Full artcile & digg here: [http://digg.com/linux_unix/getdeb_net_announces_Playbuntu](http://digg.com/linux_unix/getdeb_net_announces_Playbuntu)[/B]

I say, about time! This'll make things so much easier. It'll be like the Medibuntu of gaming :KS

**Update**: Playbuntu is now Playdeb; beta out Sept 1st ([news]("http://digg.com/linux_unix/Playbuntu_is_now_Playdeb_beta_starts_Sept_1st")).
