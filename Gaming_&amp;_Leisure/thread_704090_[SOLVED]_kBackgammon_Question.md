### Post by iheartubuntu on 2008-02-26
Just wanted to resolve this thread.

I installed "GNU Backgammon" found at [http://www.gnubg.org/](http://www.gnubg.org/) by opening up the Synaptic Package Manager, then doing a search on "gnubg". Click both of the search results to install them.

I noticed that GNU Backgammon doesnt add a program icon in the games menu so you''ll have to do it yourself. 

Go to System > Preferences > Main Menu

Click games on the left side, then click the "New Item" button.

Put whatever you'd like for the name. I typed in "gnuBackgammon" and for the command type in "gnubg -w" and pick whatever icon you might like.

That should work for you. If it doesnt, try going into the terminal and typing "gnubg".
