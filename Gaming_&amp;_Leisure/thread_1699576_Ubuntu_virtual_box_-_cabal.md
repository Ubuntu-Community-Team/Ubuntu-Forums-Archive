### Post by marl30 on 2011-03-03
If this game need decent hardware 3D acceleration, it might play very slow in Virtualbox or cause Virtualbox to crash each time you try to start the game. I see why you want to try this route, which is because Winehq gave it a garbage rating for installing in Wine. 

Setting up Virtualbox is matter of installing Windows. You'll need to allocate the amount of disk space, RAM, video RAM in the setup process.  After installing Windows, restart Windows in safe mode, then install the VirtualBox extensions. 3d Acceleration will only install in safe mode. Restart, then install the game. 

Like I said, there is no guarantee that the game will work or will be playable, since Virtualbox is actually emulating hardware resources rather than using your computer's as Ubuntu is doing.  

It's worth a shot though. 

Good luck. :)
