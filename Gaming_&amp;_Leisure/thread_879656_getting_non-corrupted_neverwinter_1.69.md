### Post by milton1 on 2008-08-04
After many attempts, I managed to get neverwinter SoU patch 1.69 downloaded without corruption. I am posting this in case others are having the same trouble. For me, the trick was to use wget instead of going through the website.

```
wget http://files.bioware.com/neverwinternights/updates/linux/169/English_linuxclient169_xp1.tar.gz
```

This can also be used to get the patch for HotU by replacing "xp1" with "xp2" in the code above.

Happy Gaming!
