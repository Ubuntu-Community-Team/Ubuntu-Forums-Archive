### Post by keltic dave on 2011-02-27
Uninstall sun java reinstall open source java. download the minecraft.jar place it anywhere in the same folder create an empty file name it minecraft.sh open it add 

```
java -Xms512M -Xmx1024M -jar Minecraft.jar
```

save it then right click properties permissions make it executable. Double click it and click run and it should work.
