### Post by Perfect Storm on 2007-09-19
1) Some games aren't GPL and therefore not in synaptic and/or the people behind the games won't allow their games through other than their site. Another reason can also be that noone made a package for it or it havn't been tested or accepted (due to strong policy how to make official packages into Debian/Ubuntu).

2) Well, both yes and no. If you don't trust the site, don't use it. If it's open source you can check the codes and see if there's anything malicious. Also if you install the game as user instead of system wide, you'll minimize the risk tremendously.

3) Depends, if the game opens a port. On default Ubuntu have all ports closed. If behind a router then no.

Said that, go for populare linux games if you are unsure.
