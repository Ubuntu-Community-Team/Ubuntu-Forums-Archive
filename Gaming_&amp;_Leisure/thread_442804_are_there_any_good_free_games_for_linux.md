### Post by B. Gates on 2007-05-13
Right.

A tar.gz file is simply a compressed file, analogous to a zip file more or less. Decompress it by running the following in a terminal:

tar -zxf FILENAME.tar.gz

OpenGL is the 3D renderer that is used by a lot of games in Linux (and Windows/OSX for that matter). For it to work properly, you need to make sure you have got the latest version of your video card drivers installed in Linux - instructions vary depending on the type of card.

Finally, I have my own favorite free FPS:
[http://www.urbanterror.net](http://www.urbanterror.net)

Quiet a few servers, you can play against Windows/OSX players too. :)
