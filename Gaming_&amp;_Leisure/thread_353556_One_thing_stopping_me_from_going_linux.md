### Post by Oblivion Knight on 2007-02-04
To get some Linux games you can go to Applications-Add/Remove and do a search(or just select the "games" section on the left) for a list of games you may want to try, just be warned that some of these don't work. Additionally you can go to System-Administration-Synaptic Package Manager and search for "game", this will give you a large list of games you can try downloading. Also, I highly recommend downloading Sauerbraten from cubeengine.com, I know this isn't what you asked for(it's an opensourcse FPS), but it is a great game for playing online. I hope that helps.

### Post by RomeReactor on 2007-02-05
Hi. [Abuse]("http://www.labyrinth.net.au/~trandor/abuse/") is a nice 2-D sidescrolling shooter; you'll find it in Synaptic (search for **abuse**. Or install it from the command line:

```
sudo apt-get install abuse
```

I highly recommend [Battle For Wesnoth]("http://www.wesnoth.org/") (also in Synaptic, search for **wesnoth**), though it is a RTS game and is rather long.

**EDIT:** Actually, Wesnoth is a **Turn** Based Strategy game :biggrin:
