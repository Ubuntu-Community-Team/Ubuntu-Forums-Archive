### Post by jonifen on 2006-06-29
As far as I know, players should only be the only folder in /baseq2 unless you've made screenshots or demos (in which case more folders will be created). You should have a file called config.cfg if I remember correctly. If not, manually set the settings in a file called autoexec.cfg (you'll have to create this) in /baseq2 which will automatically override anything else saved.

You'll need to know the commands though. If you get stuck though for the commands, give us a yell, I'll dig my old Q2 config out :)
