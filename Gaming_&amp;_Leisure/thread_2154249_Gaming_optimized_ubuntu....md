### Post by CatKiller on 2013-06-14
If you really want to dedicate all the resources to gaming, don't install a desktop environment at all. Just launch an X session for that game. It'll be just like running DOS in 1992.

LXDE is my personal favourite of the low-resource DEs. You can mix-and-match, though; a file manager from here, a window manager from there, whatever you like. It all depends on how interested you are in the whole thing.
