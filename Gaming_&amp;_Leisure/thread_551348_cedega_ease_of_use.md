### Post by cogadh on 2007-09-18
> 
**Cedega** - Designed for games tho some will still run better in Wine or vice versa
Design stolen from Wine and includes some very flaky hacks to get games working. Also has some of the worst customer support ever seen and overcharges for that lack of support. Not the best choice if you are an Open Source purist.
> 
**Wine** - Designed for general Windows games and apps not geared towards anything specifically as such sometimes does not run games/apps(respectively) as good as Cedega/Crossover(respectively) or Vice versa sometimes better then Cedega/Crossover
True, not designed for any particular type of application, but that is only because it is designed to be a complete implementation of the Windows API in Linux, including all of the DirectX functions required for most Windows games. Wine is updated on a monthly basis and each update usually includes greater support and functionality for Windows programs. It is also worth mentioning that Wine is the source project for CrossOver and the stolen base code of Cedega. As such, Wine will always be a few steps ahead of both in terms of the completeness of its Windows implementation.
> 
**Crossover** - Office apps mainly and will occasionally run a game or such better then the others tho not usually 
Actually, CrossOver has been steadily adding game support for the last year and a half. Additionally, CrossOver is the major financial backer of the Wine project. It benefits from and contributes to the further development of Wine, so it is the better choice for the Open Source purists, if you feel you must pay for a product.

And since the OP asked about ease of use, every response should include each user's opinion on that topic, not necessarily "facts". As for my opinion, yes Wine is harder to use at first, but only because it doesn't have a nice GUI to do the work for you. If you are not afraid of learning how to actually use it, it is very easy to work with. If you are not willing to learn Wine, then I strongly recommend CrossOver before Cedega, but that really has nothing to do with either's ease of use (both are equally easy to use). CrossOver is true to the spirit of the Open Source movement and continues to be a benefit to the project that it is spawned from. Cedega took from the Wine project and has given nothing back in return, then has the balls to charge people money for a product that they didn't actually develop in the first place.
