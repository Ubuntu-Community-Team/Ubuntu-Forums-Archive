### Post by MaximB on 2008-02-18
I always try to provide you with good native games (some in development) every few days.
Now I'm asking you to try and find unknown (or not so well known) native GNU/Linux games.
Those games could be playable or in development.
But please don't provide us with games that we all already know about like quake, doom, nexuiz, freeciv - they are all good but we heard about them before.

The games can be free and open source or commercial.
But please don't post abandoned or very old and inactive projects.

Anyway, try to find a few games by yourself.

**Some tips  :**
1. If you find a game that uses the Torque engine there is a huge chance of it being ported to GNU/Linux, so don't hesitate and look at the game forums for mentions about "Linux", if you did not find it, be sure to post/ask about it.
2. If you see  that the game uses DirectX only, there is no real chance of it being ported to GNU/Linux (unless it's open source).
3. if you see that the game uses OpenGL - there is a chance for a GNU/Linux port.
4. if you see a Mac client there is a chance for a GNU/Linux client.
5. unfortunately if the game has a PS3 client it doesn't say anything about the possibility for a GNU/Linux client.
6. if you *think* the game might have a GNU/Linux client, try to be sure about it before posting.

Good luck.

### Post by Sukarn on 2008-02-18
I just saw [Road Fighter]("http://gaming.gwos.org/doku.php/games:alphabetical:r:road_fighter#road_fighter") on the [Ubuntu Gamers Arena]("http://gaming.gwos.org/") a few minutes ago.

It is a remake of [Konami's ROAD FIGHTER](http://en.wikipedia.org/wiki/Road_Fighter), originally released in 1984.

Its got the [source code]("http://www.braingames.getput.com/roadf/download/RF-sources.zip") on its [web page]("http://www.braingames.getput.com/roadf/default.asp").

The web page says that the source code is "compatible with Win32, LINUX and Mac OS X".

Also, the web page mentions that it requires "SDL, SDL_mixer, SDL_sound, SDL_ttf, SDL_image and SGE" to compile, which can be obtained from the [libsdl website]("http://www.libsdl.org/").

I haven't managed to compile this game yet, but I think its quite unknown. I enjoyed the original a lot when I had it on a video game more than 12 years ago, and I look forward to playing it again once I've managed to compile it.
