### Post by oldrocker99 on 2015-02-26
Just about any DOS game runs very well with DOSBox, which is needed for Windows and Mac machines to run them as well. Install it this way:
```
sudo apt-get install dosbox
```

Then to run a DOS program, cd to its directory and
```
dosbox name-of-program.exe
```

That should do it. All the old DOS programs GOG.com sells have DOSBox wrappers, for all platforms.
