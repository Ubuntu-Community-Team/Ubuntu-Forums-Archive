### Post by Frem on 2009-05-21
When you create a game, there's a checkbox that will prevent the game from being advertized to the main game list server. If it's unchecked, other people won't see your game in the server list. I forget exactly what it's labeled, and I don't have Nexuiz installed right now.

The game is client/server based. You can run a server and play over a LAN just fine. For a LAN game, just uncheck that aforementioned checkbox and have other people connect to your IP. The Linux and Windows versions shouldn't have any trouble talking with each other, provided that they are the same version number.
