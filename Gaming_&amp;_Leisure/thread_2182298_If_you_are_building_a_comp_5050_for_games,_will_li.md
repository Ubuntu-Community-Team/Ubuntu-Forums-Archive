If you play PC games, realistically you will at least need Windows XP, optimistically you will need Windows 7 for PC games.  Let's not even discuss that crap chute called Windows 8.  If you don't believe me go to Steam's web site and take a look at the list of Linux games compared to Windows games.  Windows has always had better PC games than Linux. Just recently I tried installing Ubuntu on my Windows gaming rig (Triple Core, Radeon HD 7700, 4 GB DDR3) to test out Dota 2.  On my Windows PC rig Dota 2 can handle just fine at 80 FPS.  The Linux port of Dota 2 can barely push 20 FPS on the same rig.  It is not as optimized as the Windows version.

Wine is definitely not a solution for every game.  There are a handful of them that will run really well under wine but for the most part I agree -- it is often more trouble than it is worth.  I heard Blizzard games run really well under wine, and I had lots of luck with Valve titles in the past (HL2, Portal, TF2, etc.) but that's no longer necessary as Steam for Linux is a reality now.

It really depends on which games matter to you.  There a lots of great ones available via Steam that can run natively on Linux now.  In fact, most of my Steam library is available natively on Linux these days.

I still keep a Windows partition on my desktop for a few games (I still haven't played through the Crysis titles or Alan Wake...and I probably need to play more Borderlands2), but I have kids and rarely get to game anyway. If you're serious about gaming you will likely (and unfortunately) still need a Windows partition, but that is definitely changing.  The more that people can ditch Windows entirely the better the support we will see for Linux.  I think the last time I even booted into Windows was when I upgraded my computer's hardware a few months back.  Prior to that...I couldn't even tell you.  Then again, I have been using Linux in some form for over a decade and as my main OS for a good 7 years or so.

I would see which games you are seriously interested in playing and then see what support is available for them.  Some great Valve titles are available for Linux now, and most indie games come with Linux support.  Beyond that you will probably still need Windows.
