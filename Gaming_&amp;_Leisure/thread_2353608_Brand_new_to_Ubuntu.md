### Post by oldrocker99 on 2017-02-23
Welcome to the forums!

First, type this:```
sudo apt-get purge steam
```

Then, since really the only way to correctly install Steam is this:```
sudo apt-get install steam
```

That will bring in all the other files that Steam needs to run along with the installation.

Good luck!
