### Post by thedevnull on 2008-12-03
Like FPS  games and Open Source?  Nexuiz project is looking for developers, modelers, artists, etc!

The Nexuiz project is looking for new contributors to join its growing community.  Nexuiz is a fast paced futuristic FPS (First Person Shooter) that redefines cool.  The project seeks:

*Engine developers 
*QuakeC developers 
*Animators/3D player animations
*Skin modeler
*Concept artist
*Others community members

If you want to be involved in a cool Open Source project that RULES, Nexuiz is for you!

### Post by detrate on 2008-12-03
The wiki for commands posted above is a bit out of date and divVerent has actually stopped making builds so that package file is a tad out of date as well.  For the _very_ latest, you should check out Nexuiz from SVN.  [Here's a quick tutorial on how to do that](http://alientrap.org/forum/viewtopic.php?t=3061).  It's not the most elegant script but it work :-P.  Nexuiz is compiled with fteqcc.

For a searchable up-to-date (2.4.2, svn soon ;o)) list, check out this [cvar and command list](http://www.nexuizninjaz.com/toolz/cvar).

Another good resource is [OUNS Wiki](http://ouns.nexuizninjaz.com/) which includes [this overview of Nexuiz QuakeC](http://ouns.nexuizninjaz.com/dev:quakec) written by the head developer divVerent.


Other good resources would be the following IRC channels:

#nexuiz on irc.quakenet.org
#nexuiz.editing on irc.quakenet.org
#nexuiz.ninjaz on irc.quakenet.org
#alientrap on irc.anynet.org
