### Post by acoustibop on 2007-08-30
In fact, talking of console emulators, there are also Nintendo 64 (MUPEN64), Playstation (pSX Emulator, ePSXe, PCSX) and Sega Genesis/32x/CD (GENS) emulators.

Some of these (MUPEN64, ePSXe, PCSX) are plugin based, which makes them somewhat awkward to set up and configure; GENS needs compiling, but if you check this forum, you should be able to find .deb files for it.  pSX Emulator is very easy; you just need to tar the downloaded file into a folder (anywhere you like), install libgtkglext1 (in the repositories), add a Playstation BIOS and a few games and go.
