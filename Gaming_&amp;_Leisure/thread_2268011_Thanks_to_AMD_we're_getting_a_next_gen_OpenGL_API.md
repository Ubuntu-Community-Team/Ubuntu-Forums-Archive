### Post by R33D3M33R on 2015-03-05
The Khronos Group, which takes care of the OpenGL standard, has announced a new OpenGL API named [Vulkan]("https://www.khronos.org/vulkan") (previously glNext). The benefits of the new API will be lower driver overhead, more direct control over the GPU and lower CPU usage. The group has confirmed that the new API is based on AMD's Mantle, which will probably be discontinued. 

Congrats to AMD for pushing gaming on Linux to the next level!
