Make sure, your /home/username/.quake4 file is completely owned by you. So you can write to it when running as a user.
 You could try creating a text file in your /home/username/.quake4/q4base folder.

 In the text file just type your full cd key in on the firest line (no dashes or spaces) and save the file as quake4key.  That is how mine is stored.
Worth a try!
