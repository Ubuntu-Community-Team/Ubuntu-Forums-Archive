### Post by GFree678 on 2007-08-14
The config files for OpenArena reside in your home folder under .openarena

Delete this folder to start again fresh. Removing and installing the game doesn't do anything since the config files aren't stored in the game folder.

### Post by DirtDawg on 2007-08-14
In the file browser(Nautilus) click View>Show Hidden Files. Or CTRL+H.

Any time there's a "." in front of a file name, that means it's invisible.
