### Post by pqwoerituytrueiwoq on 2014-11-18
i am using the snes9x_next library for retroarch
i am using this ppa [https://launchpad.net/~hunter-kaller/+archive/ubuntu/ppa](https://launchpad.net/~hunter-kaller/+archive/ubuntu/ppa)
i compiled the latest version of the snes9x_next library for multi-tap support, if you use the non-next snes9x library you can only use 1 player

snes would use 2 simultaneous directional buttons for diagonal and that does work for me, i use a couple gtron game pads i got on ebay for about 12 (that was a 2 pack)
