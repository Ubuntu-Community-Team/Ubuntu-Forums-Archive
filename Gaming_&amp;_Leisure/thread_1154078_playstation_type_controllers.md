### Post by Warpnow on 2009-05-09
Some gamepads are read as HID (Human Interface Devices) and therefore are universally supported without drives. They read like a mouse.

They are basically zero-customizable but work with any system. You just change the game keys to the ones the controller inputs. If the game doesn't support chaging keys, though, you'd be screwed.
