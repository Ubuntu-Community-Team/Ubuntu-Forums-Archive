### Post by DariusTriplet on 2005-10-17
It's no problem to install; just don't make the same two mistakes I did:

-Make sure you install as root
-Make sure you have two terminals open; one running the script and one for umounting the CDs.

As for updating it, I -think- you'll need a Loki installer.
