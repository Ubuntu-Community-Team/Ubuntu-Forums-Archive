SCOURGE v17, an OpenGL RPG in the spirit of rogue, SlashEm, etc. has been released!!
The latest version of scourge features:
-a full savegame system with screenshots, saved maps, etc.
-high score saved to web after everyone in the party dies
-new graphics and models
-new sounds
-new spells, professions and capabilities
-map can be moved with right-click and drag
-chapter 10 of the story (Mothrazu!)
-many bugfixes, including the 'party mass starvation' fix

### Post by francisco_athens on 2007-03-07
> **TheIdiotThatIsMe said:**
> Is there a place to find a .deb for it?

There is no deb for it.  There is however a linux installer package for it.

you'll likely need to get the support libraries using apt-get or synaptic

here is the command line to install the libraries with apt-get. 
Open a terminal window and:
```

sudo apt-get install libfreetype6 libgl1-mesa libglu1-mesa libsdl1.2debian libsdl-mixer1.2 libxmu6 libsdl-net1.2

```

then get the scourge-0.17.x86.package autopackage here:

[http://sourceforge.net/project/showfiles.php?group_id=98006](http://sourceforge.net/project/showfiles.php?group_id=98006)

make the package executable by right clicking on it, select permissions tab and checkmark the box allowing it to be executed.


we are in the process of making the artwork GPL compliant so that it may be available via the repositories...

Francisco
