*[EDIT]* [b]LATEST VERSION:
 - added IRC chatrooms.[/b]
*[/EDIT]*


Sierra recently pulled the plug on several of it's online gaming servers, among those games was Tribes2.

However, the good people over at:
[http://www.tribesnext.com/](http://www.tribesnext.com/)
have been working hard since the authentication server went down to get a new and improved authentication server up.  This one is completely free to everyone!

They finally finished and you can download the full game as well as the new patch and once again play the most epic online game ever conceived!

Although Tribes2 is originally a windows game, it is primarily run with OpenGL so running it under wine works great.  Also, Loki games made a linux port of the game which can be found here:
[http://t2.giantfriend.com/](http://t2.giantfriend.com/)
I've had trouble getting it to work though as it was made for older libraries, you can see the progress that we've made over at this thread:
[http://ubuntuforums.org/showthread.php?t=709780](http://ubuntuforums.org/showthread.php?t=709780)

But at least for now I would just recommend downloading the windows version from the first link I posted along with the multiplayer patch (also in first link) in the downloads section and simply run it in wine.  It's an older game so I don't think you should notice any performance loss when running in wine.  Run it with the command line parameter -online to play online or -nologin to play offline.  Also if your new to the game welcome!  and don't feel bad if you suck lol, because this game has a very steep learning curve, we were all newbies once.  and please don't mind the low graphics it's a very old game.  stick with it and enjoy!

[About Defense Turret]
Also, just in case theres any Tribes2 newbies out there looking to play, you might want to get DefenseTurret (T2's anti-cheat program) so you don't get kicked out of some servers.  Don't use it yet though, for some reason I don't think the new multiplayer patch is ready for DT, or maybe it doesn't need it for some reason.  So you don't need to worry about it yet, just play the game normally.  But in case it is ever needed, I just want to feel like I covered it, so here is the link in case you ever need it.
Defense Turret:
[http://files.filefront.com/DefenseTurret+version+115+win32/;1825168;/fileinfo.html](http://files.filefront.com/DefenseTurret+version+115+win32/;1825168;/fileinfo.html)


If you enjoy Tribes2 and want to see the community grow, then please make similar threads on your favorite forums to spread the word around.  Thanks.
and I will see you in game!

### Post by perillux on 2009-01-07
The online community was huge back in its glory day.  The max players allowed in a server used to be 64 (i think its higher now) and there was always several servers that were constantly 60-64 players.

There was a browser that each player had a page and could edit in game and you could view other players, there was also clan pages and official clan tags.  There was also in game irc chatrooms, and a forum used to exist.
There are still plenty of players even when the servers went down players still found ways to connect to their favorite servers, but now that it's free I can only imagine that our numbers will grow again.
Also, keep in mind that this new server is in BETA, so eventually we will see a return of the browser, chat, and forums.  But for now they just released the basic server so that we can continue to play while they work on finishing up the patch.

But yes even now you should easily be able to find servers to play on with around 16-32 maybe more players.

It's a sweet game but be aware it's got a bit of a learning curve so I recommend playing the tutorial before jumping online. Tribes2 isn't a quake-match you can just jump on and expect to figure out in 30 seconds. Unlike many other FPS's everyone has a permanent jetpack that you fly around with which is a large part of the game. Because momentum exists in the game it can take a bit of practice before you're ready to do some real dogfighting. Then of course there are all the vehicles, deployable equipment, base equipment, player loadouts, etc. that give you so many options and make the game fun, but are confusing (and will be your demise) if you don't know what you're doing.

### Post by perillux on 2009-01-08
5nak3:
It should have two shortcuts, Solo & LAN  and Online.
At any rate though, you don't need either.  Simply run the game from command line, for example, if you have the default paths type this in terminal to run online mode:

wine /home/userName/.wine/drive_c/Dynamix/Tribes2/GameData/Tribes2.exe -online

It's the same executable for online and offline.  use the "-online" option for what else.. online.  and use the "-nologin" option for single player.
