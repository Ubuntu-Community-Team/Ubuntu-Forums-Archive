### Post by monkeymind90 on 2011-07-26
I hope by third party site you mean legitimate retailer, rather than pirate. Most games for Windows do not work for Ubuntu (or any other flavor of linux) but there are a number of option. 

The first is Wine. Wine is a program that fulfills most of the dependencies of Windows programs on linux, so you can often get something to run by using wine. [Here the link to the wine page on GTA 4.]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=14723&iTestingId=34390") Doesn't look to promising based off of that, but that was two years ago and there have been several wine updates since then. Download the most recent version and take a look at this [page]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=8757").

Another option if you still have the Window 7 install disc or a code for it is to use a virtual machine. You can use software like [VirtualBox]("https://help.ubuntu.com/community/VirtualBox") to install windows within Ubuntu and play it on there. This may cause some slowdown since its a virtual machine and it cannot access some of the features from your video card. Depending on your system this may or may not be an issue. 

You might also check out [playonlinux]("http://www.playonlinux.com/en/"), although I've never had much luck with them.

The last option is to play other games. There is a decent set of fun games for linux that work natively (more or less). I'd check out websites like [playdeb.net]("http://www.playdeb.net/welcome/") (very easy install) or the [Linux Games Tome]("http://happypenguin.org/") (poorly organized but very complete).

Good luck, wish I could help more.
