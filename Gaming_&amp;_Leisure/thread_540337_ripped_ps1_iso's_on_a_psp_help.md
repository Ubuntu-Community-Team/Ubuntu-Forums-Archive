### Post by anjilslaire on 2007-09-01
You need custom firmware on your psp. For example, I'm running 3.52 M33-4, which is an unofficial firmware based on official firmware v3.52, but with the ability to run homebrew code.
Check out these sites:
[http://www.pspmod.com/forums/firmware/](http://www.pspmod.com/forums/firmware/)

### Post by Fonon on 2007-09-01
You need to convert it to a PBP file using popstation or something similar.  Then, put it into your GAME folder.  In the GAME folder, create a folder with the game name, and then put teh PBP into that folder.
