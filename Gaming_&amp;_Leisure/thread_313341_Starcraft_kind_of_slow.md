Anyone else play starcraft (via cedega) and notice it is a bit slow?

My system:

AMD Turionx2 
1 Gig ram
Radeon Xpress 1100

The game may be choppy. To improve performance you can run the game with a different priority. eg nice -n 20 StarCraft.exe or select "Decrease WineServer Priority" in the configuration of the StarCraft shortcut when using Point2Play.

That seems to have fixed it
