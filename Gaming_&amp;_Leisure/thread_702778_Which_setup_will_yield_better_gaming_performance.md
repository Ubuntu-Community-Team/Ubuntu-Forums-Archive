### Post by ph1 on 2008-02-20
It depends on the kind of game you're playing.

If you're playing games like Civilization IV, Rome Total War, or other games that require a lot of memory but have less demanding graphics, then you'll want the computer with the better processor and more RAM (probably your laptop).  

If you're playing a game like a first-person shooter that has more demanding graphics, your cards are nearly neck and neck as far as clock speeds go, but the GeForce has more memory and supports a newer version of OpenGL.

So I guess I vote for the laptop in either case.....Though I do agree that the hard drive speed is a bottleneck, you can get a new hard drive if you find it's too slow.
