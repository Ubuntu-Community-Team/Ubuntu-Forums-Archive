### Post by justin whitaker on 2007-03-29
> **Faaron said:**
> I'm having trouble editing the Config.wtf file for World of Warcraft. I installed Wine using Automatix, found Config.wtf through 'winefile', but I can't open it. I'm not sure how to find it without going through 'winefile', either. Any help would be greatly appreciated.

You need to unhide the wine folder in Nautilus (it's in the menus), and navigate to the WoW folder. Open config.wtf with the text editor of your choice, and save it.
