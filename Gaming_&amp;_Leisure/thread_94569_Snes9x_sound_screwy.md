### Post by mcduck on 2005-12-11
All applications don't add themselves to Gnome Menu. Open a terminal and type 'snes9express'. If that works, you can add it to menu using Applications/System Tools/Menu Editor, or you can make a desktop/panel launcher for it yourself. Right-click on desktop and select 'Create Launcher' or righ-click on panel, select 'Add to panel' and then 'Custom Application Launcher'.
