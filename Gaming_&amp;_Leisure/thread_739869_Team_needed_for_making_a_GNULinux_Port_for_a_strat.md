### Post by MaximB on 2008-03-30
I have been speaking to Cateia games, to company who is now working on 2 games for GNU/Linux named : the Legend of Crystal Valley and World under Siege: European Front.
Some time ago they have made a Medieval strategy game for Windows named : the Kings of the Dark Age.
I have been talking about making a GNU/Linux port for the game and they said that if a team of programmers (C++ and Python) could be found they will RELEASE THE SOURCE CODE and the art under the GPL to that game in order for us to have a GNU/Linux client.
Unfortunately the porting won't be easy at all, they've said that likely the code needs to be completely rewritten, but I see at as  an advantage because we can add new things that weren't in the original game and improve the game.
    
Game website : [http://thekingsofthedarkage.cateia.com/](http://thekingsofthedarkage.cateia.com/)
Thread about the source code : [http://cateia.com/games/forum/viewtopic.php?f=6&t=41&sid=8e80e247a8694670f7366455eac7dc28](http://cateia.com/games/forum/viewtopic.php?f=6&t=41&sid=8e80e247a8694670f7366455eac7dc28)

I am not a programmer, I am merely trying to bring more games to GNU/Linux in any way I can, be it by sending email for game companies or by asking the source code for old games.

So please if you are a programmer and can help, sign yourself here !
I will help you form a team that will bring another good game for GNU/Linux.
