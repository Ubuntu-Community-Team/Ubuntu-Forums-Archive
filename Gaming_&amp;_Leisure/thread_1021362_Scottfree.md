### Post by Vadi on 2008-12-25
Ah. Well it would be best if you installed stuff from Ubuntu, not Debian. Better compatibility then. You can install it by [clicking here]("apt:scottfree").

Unfortunately scottfree is one of the less user-friendly programs out there. To run it, you need go to Applications&#8594;Accessories&#8594;Terminal, type in "scottfree <location of database file> <location of save file>".
