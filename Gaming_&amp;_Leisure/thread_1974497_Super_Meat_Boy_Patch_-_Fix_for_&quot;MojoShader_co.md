[SIZE="3"]**## INFO ##**
This is a patch created by Icculus, the developer responsible for handling the Linux port of SMB. If the game won't run or if the game crashes due to a "MojoShader compile failed!" error, this should fix it.

If you've been trying to run the game but it crashes when starting up, this should also fix that.

I did not create the patch, I just created this simple installer for it. The installer does not contain any game data and it will not run if you don't already have the game installed.

**## RUNNING THIS SCRIPT ##**
To run this script, all you should need to do is double-click **supermeatboy-patch.run** and put in your admin password when prompted. Once you've done that the patch should install and the game should run normally thereafter.

If you can't install the patch graphically, you can run it from a terminal using this command:
```
sudo ./supermeatboy-patch.run
```
