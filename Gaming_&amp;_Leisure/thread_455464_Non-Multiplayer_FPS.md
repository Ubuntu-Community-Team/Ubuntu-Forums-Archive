It's in the 64-bit repos. Just install it that way.
```
sudo apt-get install libsdl1.2debian libsdl1.2debian-alsa
```
You may also need to install **libsdl1.2-dev**

But they're all in the repos, and you should *never* go downloading and installing files if they're already in the repos.
