### Post by hikaricore on 2007-07-08
The MDF file is a CD image file, and MDS is the corresponding catalog file.
These are made to the best of my knowledge with Alcohol software.

You can't run these files because they are not programs, you can however convert
them with [mdf2iso]("mdf2iso.berlios.de/") which is also in the universe repos.

Though some might be curious how you have mdf images and don't understand what they are
so I suggest if this pertains to any unlawful activity that you keep further discussion off the forums.

For information on the applications and games that run under wine, please see the official database: [http://appdb.winehq.org](http://appdb.winehq.org)
