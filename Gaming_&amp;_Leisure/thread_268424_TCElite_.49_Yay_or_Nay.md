### Post by DirtDawg on 2006-09-30
I played the new release tonight and I have mixed feelings about it. Since it's my weekend, I have time to waste posting about it! 

On the one hand, there are many great improvements including better weapons response and balance, tweaked maps, and no more "single-shot-suprise". I also like the new widescreen, neat-o smoke graphics, and the ability to chuck those 'nades even further.

On the other hand I had issues playing. I seemed to get 'stuck' in doorways more often. Worse, I had terrible fps. My graphics card is not large (64MB), but that was enough to run the old version. One of the cool things about the old version was the ability to run the game on less-than-modern computers. Granted, with the dynamic lighting off, and all the options set to 'low', things run fine. But that means the graphics 'upgrade' is actually a 'downgrade' for me. 

Overall, I think the benifits outweigh the problems. I vote 'Yay'.
