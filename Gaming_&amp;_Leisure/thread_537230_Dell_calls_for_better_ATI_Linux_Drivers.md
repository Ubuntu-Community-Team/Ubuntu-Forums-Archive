### Post by cogadh on 2007-08-28
I think you might have misunderstood what I was saying and may have misunderstood Dell's statement on Ubuntu support. The reason you can't find ATI linux drivers on the Dell site is because Dell currently doesn't sell Ubuntu machines with ATI hardware, they only sell Ubuntu machines with Intel and Nvidia hardware. If you purchased a machine from Dell with Ubuntu pre-installed, you can get support for Ubuntu. However, if a Dell user takes it upon themselves to install Ubuntu on a machine with ATI hardware, they are on their own in supporting it. I don't think that really counts as a deficiency in Dell's support, since they are actually supporting the hardware as they sell it. Your machine was obviously purchased with XP on it, so Dell supports XP-related issues on it, not Ubuntu.
