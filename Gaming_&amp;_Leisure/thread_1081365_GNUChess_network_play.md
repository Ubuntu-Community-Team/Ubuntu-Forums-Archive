### Post by jacksaff on 2009-02-27
I thought gnuchess was a chess engine - a program that just plays chess. You need another program to interface with gnuchess if you wnat any kind of human-usable interface.

You probably want xboard which you can use to interface with gnuchess but also connect to an internet chess server such as FICS. Even better are eboard or jin, both of which have more advanced graphical interfaces. 

For me, best bet on linux is still to use babas chess, which is windows software but runs well under wine.

With all of these you want ot go to the fics site and register a nickname to play. Embarrassingly, as I write this 
[http://www.freechess.org/](http://www.freechess.org/) 
is down...
But it really is the best place to play chess on the internet!
