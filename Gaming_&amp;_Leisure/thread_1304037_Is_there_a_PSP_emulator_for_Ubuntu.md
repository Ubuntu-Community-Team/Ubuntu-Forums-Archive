There are several:

[http://code.google.com/p/jpcsp/](http://code.google.com/p/jpcsp/)  Cross platform
[http://code.google.com/p/emu-sam/](http://code.google.com/p/emu-sam/) Windows (Use wine!)

I haven't tried the last three, but the first two are pretty decent. Npl was correct, homebrew (better than pretty much all the commercial games out) work on the emulators available, but because those commercial games are so much different emulation is slow work. 

There are some games that will however, some like pspplayer can do a couple of them. I am a homebrew developer so I could be biased, but playing them in an emulator isn't much fun! 

Go install custom firmware and enjoy some quality homebrew!  :D
