Type this into your firefox browser's address bar (assuming you use firefox);
```
about:plugins
```

See which version of java is listed there, if any.  If no version of java is listed then you need to open synaptic and search for sun java 6 and install the one that has plugin in the name.  For it to show up you need the universe/multiverse repositories enabled, which can easily be done in ´Software Sources´.
