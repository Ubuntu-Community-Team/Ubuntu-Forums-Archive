### Post by acoustibop on 2007-08-06
AFAIK there aren't any, El Lance-O, although there is an application called joy2key (in the repositories) that allows you to map joystick/controller controls to keyboard/mouse functions.  Don't think it works the other way round, though.

BTW as far as Playstation emulators are concerned, have you tried [pSX emulator]("http://psxemulator.gazaxian.com/")?  It's the only Linux Playstation emulator that's still in development, and to my mind is far better than ePSXe - apart from anything else, it's simple to install and configure, and needs no plugins.

However, it also doesn't support mouse input.  However, since it's still in active development, there is the possibility that this may be included at some point - there have been requests for this.
