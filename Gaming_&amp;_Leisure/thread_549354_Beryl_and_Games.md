### Post by cogadh on 2007-09-12
Beryl is usually safe to install and use, but it can interfere with gaming in general and Wine in particular. Just make sure you switch the window manager back to Metacity before you start a game and everything should be fine. That can be very easily done by right-clicking on the Beryl Manager icon on the panel.
