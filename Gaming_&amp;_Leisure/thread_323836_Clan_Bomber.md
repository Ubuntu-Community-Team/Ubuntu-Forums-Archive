### Post by shane2peru on 2006-12-22
Pretty neat game.  Just open a terminal window **Applications - Accessories - Terminal**  and type ```
sudo apt-get clanbomber
```  After it finishes just open up **Applications - Games - Clanbomber**  If it isn't there, you should be able to type clanbomber in the terminal to run it.  Or type clan(tab) and it will show you the options.

Shane
