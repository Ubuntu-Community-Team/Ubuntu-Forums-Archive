I recently discovered PuzzlePirates and I'm hooked.  The only trouble is, I haven't figured out how to make the downloaded version run.

### Post by towsonu2003 on 2006-12-27
I think it was either this ```
/usr/lib/jvm/java-1.5.0-sun/bin
``` or this ```
/usr/lib/jvm/java-1.5.0-sun
```
