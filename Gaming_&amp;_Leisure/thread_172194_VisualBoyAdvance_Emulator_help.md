### Post by Thorin on 2006-05-08
You can also run it from the terminal. So if I wanted to play a game that was saved on my desktop, I would open the terminal and type

```
vba /home/me/Desktop/game.gba
```

and voila! If you want to see the list of options for vba, just run vba, without pointing vba at a rom. Hope that helps!
