### Post by zanjabeel5 on 2008-01-30
I'd like to share with you the first game I made

very simple game:
colored tiles start to appear from right of the screen, as the game advances, speed increases.
you should press first letter of each tiles color  r,g,b,y 

[IMG]http://ubuntuforums.org/attachment.php?attachmentid=58064&stc=1&d=1201696567[/IMG]

till game over (my personal best was 46 :)  )

[IMG]http://ubuntuforums.org/attachment.php?attachmentid=58065&stc=1&d=1201696653[/IMG]

My goals were to:
- learn basic game development and sdl
- refresh c++ knowledge
- please my little sister 
- have something to share with others

to run the game you should have these libraries installed (default in ubuntu I guess) :

 libSDL-1.2.so.0
 libSDL_ttf-2.0.so.0
 libSDL_image-1.2.so.0

then just extract and run ./go
otherwise if you'd like to develope (fix bugs) and compile run make in the game folder
