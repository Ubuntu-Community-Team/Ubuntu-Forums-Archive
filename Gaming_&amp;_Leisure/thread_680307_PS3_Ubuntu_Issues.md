### Post by big dizzle on 2008-01-30
I've got 7.10 installed, and I've been reading up on installing linux on PS3, but I haven't actually done anything w/a PS3 yet.  Here are some suggestions though (keep in mind I haven't done any of this yet, so these are only guesses at where I would start if I wanted to fix the problem):

1) No clue about that internet thing.  I haven't written any scripts yet, but I imagine it can't be hard to write a script that will run that command that you are using to fix your internet every time it boots into ubuntu.

2) For fixing your graphics issue, have you tried going to  System>Administration>Screens and Graphics and changing the resolution from there?  If changing the graphics there doesn't work, check to see if you have the proprietary nVidia drivers installed (go to System>Administration>Restricted Drivers Manager).  If you aren't able to change the graphics and resolution settings, you could try using a program called Envy which detects your video card and runs a script that installs the correct driver for you ([http://albertomilone.com/nvidia_scripts1.html](http://albertomilone.com/nvidia_scripts1.html)).

3) I don't use flash player.  I was having troubles getting it to work write.  When I would go to youtube for example, and try to watch a video, a message would come up saying that you need to install blablabla and instead of choosing adobe flash, I chose the second one on the list (I think it was gnome something) and that seemed to work well.

That is the best help I can give you, but I have a couple questions for you though.
