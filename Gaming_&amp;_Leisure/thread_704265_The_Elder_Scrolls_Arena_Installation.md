### Post by Golem XIV on 2008-02-25
Is the Arena installation file executable (.exe)?  If so, copy it to the DosBox working directory (the one you are using as drive C:) and run it from there.  If it's a compressed file, extract it using file-roller or some other decompression utility and then copy it over to the working directory.  You will probably have to run some kind of configuration program in any case to set up the sound card.

There is more detail here:

[http://www.uesp.net/wiki/Arena:Running_Under_DosBox]("http://www.uesp.net/wiki/Arena:Running_Under_DosBox")

while it's oriented towards Daggerfall the info should be useful for Arena also.

I would recommend that you also read up on how to configure DosBox and/or get yourself a GUI front-end for it.

Good luck!
