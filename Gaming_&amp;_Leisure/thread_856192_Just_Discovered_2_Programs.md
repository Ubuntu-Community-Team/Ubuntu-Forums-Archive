### Post by jonian_g on 2008-07-13
Cedega is very good. It runs great the games that it officialy supports, but it's only for games. I use it for about 1 year to play NFS, Warcraft, Hitman and GTA, all of them run perfectly.

CrossOver is the commercial version of WINE. It is developed by Codeweavers which had helped a lot building WINE. If you go at [http://www.winehq.org/](http://www.winehq.org/) you will see that they have a link at the upper right corner for Codeweavers. CrossOver runs really good many programs and games as well. The only bad is that it doesn't use the latest version of WINE. I used it for some time but because of that I got back to non-commercial WINE.
