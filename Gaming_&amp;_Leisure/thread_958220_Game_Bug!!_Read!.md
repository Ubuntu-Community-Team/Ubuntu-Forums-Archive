### Post by ajgreeny on 2008-10-25
Probably just a conflict between compiz and the game in some way.  Get a copy of the simple script from [Forlong]("http://forlong.blogage.de/article/pages/Compiz-Switch") which allows you to switch compiz on and off with a single click, and keeps whatever custom settings you have set for compiz.  You can then simply stop compiz when you run the game(s).
