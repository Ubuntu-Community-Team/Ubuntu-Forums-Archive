### Post by Malikith on 2007-07-19
Performance wise, you may notice a small boost, depends. It definitely will not be slower. If you run it on a server with no Xorg at all, yeah, it'll probably be about 5-10% faster and alot more stable obviously. Hell, to be honest, it may be 5-10% faster running with xorg anyways. I haven't tried running a game server with xorg. But either way, you will not be disappointed.

As far as the version of Ubuntu, thats really up to you, do you want to update the server often? If so, just run with 7.04. If you don't want to update at all and just remain on one version, use the LTS releases.

And for your wireless card, I am not sure, I say try it with a live cd and see what happens, if it doesn't work, try researching and trying out ndiswrapper.
