### Post by ajgreeny on 2016-01-09
Like Dennis N, I am a fan of **audacious**; it's small, works brilliantly, and has none of the overkill or bloat that some other music players have.

However, it does not have a CD ripper which **rhythmbox** does.

If you are command-line able try **abcde** for ripping CDs; with an appropriate config file, or files, if you want to rip to different output sometimes, you can manage ripping in great detail, and also rip to several different file formats simultaneously.
