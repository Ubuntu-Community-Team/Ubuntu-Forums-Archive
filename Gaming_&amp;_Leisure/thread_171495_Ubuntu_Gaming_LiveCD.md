### Post by leech on 2006-05-06
So I was just talking to my brother, while he has been trying out different versions of Monopoly on several different emulators.  Then I thought, well, he should try out the Linux version, atlantik/gtkatlantic.  But he only has Windows on his PC, and wouldn't want to install Linux because all he really uses his computer for is to look up games on the internet and play games on it.  

So then I was thinking, well, why not an Ubuntu Games LiveCD.  That way he could just pop it in, start up whatever Linux game he wanted and we could play them multi-player.  I know there is Knoppix games, but Knoppix tends to have a rather bloated install.  I think if we took a Dapper LiveCD, remastered it with taking out things like Open Office, Gimp, etc. and putting in gtkatlantic, abuse, scorched3d, etc.  It would be fantastic.  

What does everyone else think?  Plus it could be a decent showcase for some of the gaming capabilities that free software can provide.  Though the one problem I really see with this is we'd have to set up auto-detection of nvidia and ati cards so that the fully accelerated drivers could be loaded.
