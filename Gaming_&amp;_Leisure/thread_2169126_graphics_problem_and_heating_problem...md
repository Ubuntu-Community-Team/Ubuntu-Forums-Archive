### Post by QIII on 2013-08-20
Hello!

Your machine uses Intel/ATI hybrid graphics.

Unfortunately, the equipment manufacturers have not provided good support to Linux for these systems.  Since yours is an Intel/ATI hybrid, your best bet is probably vga_switcheroo, which will allow you to switch between your Intel GPU and your ATI GPU.  However, you will not be able to use the ATI proprietary driver and will need to stay with the default open source Radeon driver.

For more information, please see [this wiki page](https://help.ubuntu.com/community/HybridGraphics).

If you have any questions, be sure to ask!

Best wishes!
