### Post by ekilfoil on 2010-05-06
Hello everyone,

I wanted to let you all know that we will be releasing a beta version of Mangler 1.2 on May 9th.

It's been 6 months since we released 1.0 and there have been a lot of new features added since then. If you haven't been keeping up and you're still using 1.0.x, you're going to be very happy with this upgrade.

There are *tons* of new features, but the most important ones are:

* admin capabilities
* master volume control
* chat support
* user mute, mic mute, global mute
* rhythmbox integration support

We'll be releasing the new version Sunday evening and it should be available in the Mangler PPA late Sunday or early Monday. Please feel free to report any bugs (preferably on our forums at [www.mangler.org](www.mangler.org) or in our bug tracking system).
