### Post by c1981 on 2009-02-11
All ID Software games are linux compatible - by downloading binaries from ID's FTP.

Examples: Doom 1,2,3 Quake 1,2,3,4, Wolfenstein (return to), Wolfenstein: Enemy Territory (is completely free but can be quirky in terms of sound configuration) - all other oldies (Hexen, heretic etc) also apply.

Open source games based on ID engine's:

OpenArena
Nexuiz
Warsow

Unreal also offers a number of Linux binaries for their games; as does another company - though i forget the name - that's just made their first attempt.

Other fun games:

Neverball / Neverput

Feeling retro? ---> Dosbox plays most things up to Doom II perfectly.

Wine also runs Windows games on Linux - even Crysis can be made to work.

Also - nosing around in the apt repositories will turn up quite a few games.

Last but not least: 3D support in Virtualbox, VMGL and other projects are 
quickly bringing 3D acceleration to virtual guest OS's - Quake 3 is already possible in a virtual Windows XP running on virtualbox; the overhead is being dealt with (slowly but surely).
