### Post by PacSci on 2009-04-20
The computer seems like a pretty good deal to me. WoW probably has about the same system requirements on Ubuntu - what you lose by running it under Wine, you gain due to Linux's overall efficiency - but 2 cores, 4GB of RAM, and a 500GB disk would most definitely be enough. You might need to install some proprietary drivers after it ships, though, for the graphics card.

If you buy it, make sure you check out this page on the Ubuntu wiki: [https://help.ubuntu.com/community/WorldofWarcraft]("https://help.ubuntu.com/community/WorldofWarcraft").
