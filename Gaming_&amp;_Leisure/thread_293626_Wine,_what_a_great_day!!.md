### Post by pkslot on 2006-11-05
Since 1999 i've been hooked on this great game called Homeworld, you know, on and off. Sometimes i install it, just to walk down memory lane and have a good time playing a great strategy based game.

Today i felt the urge to have a little go again, so i decided to try it out in wine, witch i've never gotten to work with anything. When i ran breezy badger and dapper drake, it never worked for me, but lucky me, in edgy eft it just don't have any issues to run against me (or my hardware).

I simply ran the "wine /media/cdrom0/HWsetup.exe" command and it installed nicely. When i ran the "wine /media/cdrom0/HWmenu.exe" it just worked like a dream. I configured it to run an ok screenresolution in the game, running in openGL. Then, almost shaking of sweet expectation, i started a skirmish game and pow!!!!!! Damn it felt good.

Well just thought i'd share my happiness with you :mrgreen: 

Btw, i'm running Edgy eft on an older machine:

Amd Duron 700 mhz
Nvidia geeforce2 mx
512 sdram
2 HD, 40 gb and 100 gb
Asus motherboard with onboard sound

So edgy *can* run smooth on older machines too.
