Get Wine 0.9 from [http://winehq.org](http://winehq.org).  They provide a debian/ubuntu repository there.  This is a later version that 20040615, and should work the same or better.

Once installed simply type "winecfg" at the commond line to set it up, and then continue with the instructions.
