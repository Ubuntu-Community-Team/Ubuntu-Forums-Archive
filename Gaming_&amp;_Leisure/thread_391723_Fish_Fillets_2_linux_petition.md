### Post by B0rsuk on 2007-03-23
Fans of good puzzle games may remember Fish Fillets - a great, polished and original puzzle game. 
```
sudo apt-get install **fillets-ng**
```

Originally commercial, it didn't generate enough profit so it was released under GPL license. It easily became the most polished game in repositories. And no doubt the best GPLed puzzle game. It's available on multiple systems, even quite obsure ones like freeBSD or solaris.

--------------

**Fish Fillets 2 is out**
[http://www.fish-fillets.com/pages/features.html](http://www.fish-fillets.com/pages/features.html)
The problem: only windows version is available.
On top of that, the 'demo' can be activated to become a full version. This is bad because it contains some kind of DRM junk. **It doesn't work in wine** because the copy protection interferes and suggests that the game is being run in a debugger.

I mailed
*support at altargames.com* .  They said that at the moment there isn't enough interest in linux version of FF2, but if there were more people like me, it would be great.
So If you enjoy Fish Fillets ( fillets-ng ), and would like a linux version of Fish Fillets 2, either post on their forums
[http://forum.altargames.com//index.php?showforum=21](http://forum.altargames.com//index.php?showforum=21)

And spread the word !

or send a letter to email mentioned above.
