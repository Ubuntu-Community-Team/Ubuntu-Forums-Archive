UPDATE:
it looks like the last version of GMAMEUI was released in 2008, so i might have been incorrect in saying that it is still actively developed. it may be a dead project. however, here is a FAQ about frontends for mame on linux/OS X:

> Got any compatible front ends for Linux? OS X?

Yup! I know of several that support SDLMAME on Linux: Mame Executor, MAME Plus! GUI, QMameCat/QMC2, Loemu, WahCade (which also works on OS X), XMAMEGUI (which is in Java and should actually work on all OSes), KXMAME, and GNOME Video Arcade. On the Mac, check out MAME Plus! GUI, MameTunes, EmuLaunch, and MacMAMEInfoX.
