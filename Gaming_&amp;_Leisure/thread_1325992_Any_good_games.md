### Post by handy on 2009-11-14
Install DJL, it will give you access to 122 games, last I looked.  They aren't all FPSs, thought they are certainly there too.

If you do install it, look up in the menu & you will find an item that has a list of game dependencies, I suggest that you spend the time & download them all.

They all go into ~/.djl/libs/ & are shared by any of the games that you download, if of course they require them.

DJL has a few other tricks up its sleeve, & a web site & forum.

It is in the Ubuntu repo's so it certainly isn't hard to install or remove.

### Post by gradinaruvasile on 2009-11-14
Linux native (q3 engine based):

True Combat:Elite - my favourite - CS like, but more realism based
Urban Terror - CS like game, good gameplay
Smokin Guns - Wild West shooter, original and fun
AlienArena - its Q3 with aliens and improved graphics
Enemy Territory - WW2 FPS,
OpenArena - Quake 3 clone
Nexuiz

Also CS 1.6/Source (and other HL2 based games) runs really well under Wine.

Note: You might need to disable PulseAudio for most of these games to work properly.
Also i hope u have Nvidia graphics... Cause the open source (and afaik the proprietary Ati too) drivers may cause system crashes. The only really stable (3d wise) drivers are the Nvidia proprietary ones ATM.
