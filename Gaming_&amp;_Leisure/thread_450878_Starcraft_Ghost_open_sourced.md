### Post by hartz on 2007-05-22
Developers of all kinds of software, including games / entertainment, reuse code.

A lot.

So even if Ghost did not get published, you are sure to find that they re-used portions of the code in later products, and even it may contain portions of code of other earlier products.

Releasing that code under an open-source license is fraught with risks, and since they are in the business of making money, there is little if any incentive and much preventing them from releasing "unpublished works"
