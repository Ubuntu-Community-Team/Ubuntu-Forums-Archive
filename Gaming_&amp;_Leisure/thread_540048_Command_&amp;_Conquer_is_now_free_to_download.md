### Post by Kumeelyun on 2007-09-01
It was just announced that in honor of the 12th anniversary of the Command & Conquer franchise, Electronic Arts is releasing C&C Gold as free ISO downloads on their site ([both GDI and NOD discs are available here]("http://www.commandandconquer.com/intel/default.aspx?id=61#NewsMain")). They give instructions and link to the XP patch.

According to [Wine's database]("http://appdb.winehq.org/appview.php?iAppId=166"), the game has silver status with version 0.9.16., but  garbage with 7.04 running Wine 0.9.43. I'm curious as to how it holds up with 0.9.44. If no one responds in this thread with their experience, I'll have to try it myself. I have the Command & Conquer First Decade collection, but this could prove to be another nice choice for Linux RTS fans if someone can get this running well with Ubuntu.
