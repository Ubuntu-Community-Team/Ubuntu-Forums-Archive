### Post by nhandler on 2008-06-01
There is no iMovie for Linux. The best alternative I have found is kdenlive. I found that it was very easy to use, and it had enough features for me to edit my movies. It is in the repositories, so you can install it through Synaptic. Or type 'sudo apt-get install kdenlive' in a terminal to install it.

You shouldn't need to install .tar.gz files often. Most packages you will need on Ubuntu are in the repositories, and they can be installed through Synaptic. [Here]("http://monkeyblog.org/ubuntu/installing/#source") is a guide on how to install .tar.gz files.
