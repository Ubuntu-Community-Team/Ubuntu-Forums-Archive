### Post by janfsd on 2007-04-03
for boost just install libboost-filesystem:

```
sudo apt-get install libboost-filesystem-dev
```

About cegui, if you are running edgy, you'll probably need to download the source at: [http://www.cegui.org.uk/](http://www.cegui.org.uk/) and compile it yourself. That's because you need a newer version of cegui than the one provided in the repos.

If you are on amd64, I've made a deb: [http://rapidshare.com/files/24201270/cegui_0.5.0-1_amd64.deb](http://rapidshare.com/files/24201270/cegui_0.5.0-1_amd64.deb)
