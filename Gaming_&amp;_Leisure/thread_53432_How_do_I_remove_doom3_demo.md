### Post by No6 on 2005-08-01
[QUOTE=PrimoTurbo]Doesn't it leave files around, it's not just in one directory? Is that all I have to do? There is no uninstall function? I can run doom3-demo in terminal? Don't I need to do something about that? Thanks for your help. :)[/QUOTE]

There is no uninstall. It's not like windows as there is no complicated registry etc. There are just 2 directories and one file to delete, and that's it all gone, they are...

The installation directory (/usr/local/games/doom3-demo by default)
The settings directory which is .doom3-demo in your home directory.
The link to run the program which is /usr/local/doom3-demo.
