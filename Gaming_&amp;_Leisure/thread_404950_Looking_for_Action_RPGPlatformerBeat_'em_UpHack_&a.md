### Post by Garyu on 2007-04-12
[www.dofus.com](www.dofus.com)
Dofus is an RPG that runs from flash. Just download and load the .html file to start the game. Looks nice, but since it is a MMORPG you need to pay a small fee to get full access (you can still get limited access for free to try out the game).

From the same team, also check out wakfu, which will be released this summer.
[www.wakfu.com](www.wakfu.com)

Neverwinter Nights has a native client that you can download from Bioware. You need a serial key to get it running though. Very nice RPG indeed.

Americas Army is a 1st person shooter, but still, there is a native client and it is very much fun. The newest version with vehicles to run and stuff like that does not have a native client and it does not work with wine (at least not last time I checked), so you need to look around for the linux installer. All for free though.

Wormux may not be exactly what you're looking for, but very fun if you have a friend to play with on the same computer. 

And maybe not so much an RPG, but Battle for Wesnoth is native and quite entertaining.

I am sure there are others. Looks for posts by Artificial Intelligence, he has a good gaming list in his signature.
