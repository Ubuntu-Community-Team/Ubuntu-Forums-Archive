### Post by hikaricore on 2007-05-22
I've had this happen with a number of games.
Try running them from a terminal and see if there's any error output when you exit them.

What the issue turned out to be fore me was the game was crashing on exit before (or when) it could change the resolution back.  If this turns out to be the cause for you, atleast it gives you somewhere to start.  Looks for graphic related error output or segfaulting.
