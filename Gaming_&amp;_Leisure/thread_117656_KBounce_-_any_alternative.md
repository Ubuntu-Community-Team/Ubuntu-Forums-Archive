### Post by mario.rimann on 2006-01-15
Hi all

There is a game named "KBounce" that comes along with the KDE. Now I'm using Gnome and would like to see this game on Gnome. Is there an alternative? Or is there a possibility to use KBounce under Gnome?

Short description of the game: There is a empty play field with just a ball in it. The ball is bouncing from wall to wall. Your job is to "cut" the balls way by placing new walls across the field. So with time the ball has less and less place - if more than 75% is "saved" by the player, it goes to next level (faster ball).

### Post by jasmuz on 2006-01-15
Yes you can run Kbounce on Gnome...as long as your system has installed kdelibs.

do this at the terminal;
sudo apt-get install kbounce

And it will fetch everything you need.

### Post by gnapse on 2009-08-19
Have you tried [IceBreaker]("http://mattdm.org/icebreaker/")? It's even better than the original jezzball in many respects (maybe kbounce has similar improvements, I don't know) but it won't clutter your gnome environment with kde libraries, and it's a much smaller download to install.

In ubuntu it is in the repositories, so a simple

```
sudo apt-get install icebreaker
```will do the job in no time.
