Easier yet would be using a command line tool, wget.

Open a terminal, then

$ wget pathtofile.zip

to stop press ctrl-c

to continue:

$ wget -c pathtofile.zip

you don't have to retype all of that stuff in a terminal, as you can access your history with the arrow buttons, then add "-c".

d4x is available in Synaptic and a great front-end for wget.

I don't have gFTP installed currently so I don't recall exactly how it works.

Hope this helps. I'm not one to push folks onto terminals for everything, but wget is so basic that I tell even noobs to use it. Plus a terminal looks very professional and sexy on the desktop. LOL
