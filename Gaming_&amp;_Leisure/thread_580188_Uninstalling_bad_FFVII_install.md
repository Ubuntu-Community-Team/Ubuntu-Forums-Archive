### Post by cogadh on 2007-10-18
If that was the only thing you installed with Wine, then just delete the .wine directory in your home directory. If it wasn't the only thing you have in Wine, the run "uninstaller" in the terminal. That will launch the Wine uninstall utility, which should be able to remove it.
