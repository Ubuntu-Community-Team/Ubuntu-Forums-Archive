### Post by Lystrodom on 2007-03-15
Step one: Download the .run, which you can get [here]("http://www.gamershell.com/download_2355.shtml"), among other places.
Step two: In the terminal, navigate to where you installed it.
Step three: ```
sh et-linux-2.55.x86.run
```
It will ask you for your root password, if you don't have one, you can set it up with:
```
sudo passwd
```

I think there's a way to do it without a root password, but this is the way I did it.
