### Post by AndyCinDallas on 2011-04-08
I've usually just gone into the browser cache and copy the file elsewhere.

Example - open Nautilus and go to /home/username/.mozilla/firefox/randomname.default/Cache

Sort the files by the date modified (or size - whatever helps you find the video-file itself) - then right-click and copy to another location.
