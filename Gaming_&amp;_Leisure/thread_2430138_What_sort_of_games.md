### Post by oldrocker99 on 2019-10-29
To quote Wikipedia, there are over 5,000 Linux games on Steam. ```
Valve officially released Steam for Linux on February 14, 2013. As of May 2019 the number of Linux-compatible games on Steam exceeds 5,000. With the launch of SteamOS, a distribution of Linux made by Valve intended to be used for HTPC gaming, that number is quickly growing.
```

Not only native Linux games, but hundreds of Windows games that run very well with SteamPlay, with an application layer called Proton. Some of them run **better** on Proton. You can find out which do, and which don't, here: [https://www.protondb.com/](https://www.protondb.com/)

Have big fun seeking Gaming Goodness (tm)!

### Post by qualtch on 2019-11-07
As mentioned earlier, there is a good bunch of games natively on Steam and the amount has been growing quite well during recent years. You can also check [https://www.protondb.com/]("https://www.protondb.com/") if you can run games with Proton on Steam. I have been playing some using it and they work if not out of the box, with minor tweaks that users have provided in the comments sections in ProtonDB. Then there is also Lutris which contains ready-made script bundles to help setting up Windows games. Youtube has pretty good list of guides in how to use Proton and Lutris to try out gaming. One company that does native Linux ports of games is also Feral, whose current offering you can check out here: [https://www.feralinteractive.com/en/linux-games/]("https://www.feralinteractive.com/en/linux-games/").
