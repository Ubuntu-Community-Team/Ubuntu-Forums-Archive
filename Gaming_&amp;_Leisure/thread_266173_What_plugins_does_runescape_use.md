### Post by Mr Frosti on 2006-09-26
Probably the game needs a higher version of Flash player, or Adobe Shockwave (which is similar to flash). The good news is that Flash player 9 is right around the corner for Linux. The bad news is that if it requires Adobe Shockwave, there is no, and no plans for a Linux client. 

I have heard that you can use WINE to install the Windows version of Firefox and then install these missing plugins.
