### Post by OpenMW on 2012-08-07
[img]http://wiki.openmw.org/images/thumb/f/f5/Screenshot016.jpg/800px-Screenshot016.jpg[/img]

**OpenMW **is a project that aims to completely create the rpg Morrowind in an open-source, cross-platform way. We're currently on version 0.16.0, with the full future release being 1.0.0. But before we can get there, there are certain challenges that need to be overcome.

Here's a **video** of the latest version, with its capabilities explained: [http://www.youtube.com/watch?v=ThfZrNVp5kQ](http://www.youtube.com/watch?v=ThfZrNVp5kQ)

At the moment, we're badly in need of MyGUI developers, or anyone who has done GUI programming in C++. If you're interested, you can take a look at this wiki page: [http://wiki.openmw.org/index.php?title=Contribution_Wanted](http://wiki.openmw.org/index.php?title=Contribution_Wanted)

We could also use anyone willing to **test** the game for bugs, and **research** the original Morrowind for formulas and fidelity. That's right, all you need to do to help is play games. :)

Our **downloads** can be found here for all systems: [http://code.google.com/p/openmw/downloads/list](http://code.google.com/p/openmw/downloads/list)

If you'd like to talk directly to the developers, you can register on the OpenMW **forums** here: [http://forum.openmw.org/](http://forum.openmw.org/)
