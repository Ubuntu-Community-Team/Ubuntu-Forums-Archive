### Post by Grenage on 2012-03-09
> **kstella said:**
> Okay. How do I do that?

Game data files for version 2 are (apparently) stored in ~/.lincity-ng/ which is a hidden directory in your home directory.  You can either type in the path (Ctrl-l in the file browser) or press Ctrl-h to show hidden directories.

If the file is small, you can attach it to a post, otherwise you'll need to host in elsewhere and post a link.
