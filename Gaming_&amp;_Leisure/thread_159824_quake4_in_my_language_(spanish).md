### Post by llonesmiz on 2006-04-13
Hi, I dont have the game so i cant test this myself but I guess it should work.
The game has several packages: zpak_english.pk4, zpak_spanish.pk4, zpak_french.pk4 and so on. By default the installer seems to copy the English one. You could try deleting the file zpak_english.pk4 and copying manually zpak_spanish.pk4 to the directory where the other was. If that doesnt work, you could try renaming zpak_spanish.pk4 to zpak_english.pk4.
Good luck.
