### Post by bswarm2 on 2016-09-20
I bought the newly released Sony 3001625 DUALSHOCK 4 USB Wireless Dongle to see if it would work system wide as a gamepad in both Linux and Windows. The dongle does work on Ubuntu 16.04 without installing anything. I can pair a PS4 Dualshock controller and it shows up in System Settings/Joysticks with all buttons and Axis working. It works for some Linux games (SuperTuxCart for one) but is not working with any Steam games.
I tried pairing the DS4 controller with the computers onboard bluetooth, it also shows up as a gamepad in System Settings/Joysticks but none of the controls work.
Is there any way to get it working in Steam?
Edit: Solved. Steam has to be in "Big Picture" mode. Even then, only some games have DS4 support.
