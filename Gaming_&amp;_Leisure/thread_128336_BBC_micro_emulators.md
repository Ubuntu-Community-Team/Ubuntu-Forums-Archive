### Post by Quirky on 2006-02-12
There is BBC emulation in Mess, which is quite good and some games are fully emulated, but is no longer updated (as far as I can tell). Mess should be installable from Synaptic, maybe.

Other emulators I know of are Windows only. B-Em [http://b-em.bbcmicro.com/](http://b-em.bbcmicro.com/) which is now available on Mac OS, not sure if a Linux version is on the cards.

The best emu there is is Beeb-Em.  It works very well in Wine - even sound - and this is the Emu I'd recommend, if you don't mind using Wine. [http://www.mikebuk.dsl.pipex.com/beebem](http://www.mikebuk.dsl.pipex.com/beebem)

(The source is also available for both of these if you fancy a challenge porting them to Linux!)
