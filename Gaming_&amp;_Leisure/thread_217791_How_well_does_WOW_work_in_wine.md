### Post by eqisow on 2006-07-18
To play WoW on Wine you will need a patched version of Wine. Look on the [Wine wiki]("https://help.ubuntu.com/community/Wine") under "Latest Wine with WoW Patch" for that. After you have that installed, see the [World of Warcraft]("https://help.ubuntu.com/community/WorldofWarcraft") wiki.

Once it's set up right, WoW should run flawlessly in Linux.

Note: I've set it up on 5 different computers with no problem.

### Post by eqisow on 2006-07-18
> When you say "flawlessly" -- Are you getting comparable frame rates?
Actually, yes. Frame rates are comparable across platforms. Loading times are noticably faster.

System RAM may have an effect, since the Windows API has to be loaded when running in Wine. Also, I suppose 3D acceleration in certain chipsets may not be as well supported as it is in Windows.
