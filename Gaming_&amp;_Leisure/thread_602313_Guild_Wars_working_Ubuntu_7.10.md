### Post by reagentz on 2007-11-04
I don't know how I did it, I never even needed to configure anything other than set WINE to run in "virtual desktop". I had the Guild Wars installer on my storage drive and thought I'd give it a shot. I setup the WINEHQ repositories in terminal, and apt-get install wine. Few moments later WINE 0.9.48 installed. After install I went to my storage drive, right clicked on GW.exe and then left clicked Open with application WINE. The launcher for Guild Wars loaded up and started updating. After the update I was sitting in the logon screen for name a password. Everything looked awesome, just like I remembered it under Windows. Once I logged in it start updating/streaming the data files. Before I launched all this I did edit my router to allow some ports to be open, it's under another Guild Wars Howto I was reading. Once I logged into the game everything was fine, no sound issues, graphic, or mouse issues.

I'm happily playing my farming toon (monk/warrior) under Ubuntu 7.10 now. Now only if NCsoft would come down on their still high price for expansions!

Edit: Turned off Virtual Desktop and set Guild Wars itself to run windowed, its the way I usually play it and it still runs fine.
