### Post by Cyhawk on 2008-08-04
I know its late to reply, but when you have a game folder, say FvF (future vs Fantasy, good mod) you would run the game like

./quake -game fvf

or whatever folder the mod is in

so like

./quake -game fortress
./quake -game arena

etc etc.

Check around Fileplanet for old mods for it.

Good single player ones I remember are:

airquake
FvF (Future vs Fantasy)
