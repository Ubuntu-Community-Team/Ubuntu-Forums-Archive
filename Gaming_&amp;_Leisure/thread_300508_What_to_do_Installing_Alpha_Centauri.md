### Post by Redache on 2006-11-15
Alpha Centauri only runs on Mac's and Windows machine's. I did some research as I wasn't sure but I can't see any mention of Linux compatibility anywhere. Try the Synaptic Package Manager if you want some things (You should be able to find Wine in there as well). 

Hope Wine Works for it! Great game.
