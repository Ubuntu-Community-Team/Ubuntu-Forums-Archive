### Post by RadicaX on 2014-06-07
[http://appdb.winehq.org/objectManager.php?sClass=version&iId=11690](http://appdb.winehq.org/objectManager.php?sClass=version&iId=11690)

Tiger woods PGA Tour 08 had a platium rating with the WINEHQ, meaning with Wine you could use it, and I seen 2012 had a silver rating, though both were done with slightly older versions of wine I believe. So if you do not mind using Wine, you might be able to get the windows version up and running on your Ubuntu. Though note, When they rate it like Silver, it has bugs, if they rate it gold, it still has little bugs, Platinum is normally without bugs, or pretty darn close to normal play to not notice the difference. So either way, Hope you like it.

P.S. Glad WGT works for you!
