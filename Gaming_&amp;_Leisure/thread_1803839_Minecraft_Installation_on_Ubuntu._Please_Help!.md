### Post by zami on 2011-07-14
Mmmm.... Minecraft....

With that .jar file...  Well first, let me suggest you tuck it someplace.  I just have a folder on my desktop called Games, and keep the minecraft.jar in there.  It's not an install file though, you wont delete it once the game is "installed", so just tuck it someplace handy.

Anyhow, on to making it "go".  (Assuming you have the correct Java goodness already installed, which it sounds like you should.)

Right click on minecraft.jar, and go into
Properties > Permissions > 
and check the box that says "Allow executing file as a program"
then close.

Now, right click on minecraft.jar, and in the dropdown menu select
Open with OpenJDK Java 6 Runtime 
(exact verbage might be a little different)

Hopefully that will do it!  You may find yourself looking at a blank box for a moment, or HTML jargon - just takes a moment sometimes.

-zami

### Post by NERDMAN! on 2011-07-15
download the jarwrapper package from the ubuntu software center. right click the jar file and go to properties, under permissions tab check the "allow executing file as a program" box, then go to the open with tab, click the add button, then click the + sign next to "use a custom command".
type "jarwrapper" in the box and click ok.
then close the properties window, put the jar file on your desktop and double click it.

thats all you need to do to run the game on ubuntu.

enjoy!
