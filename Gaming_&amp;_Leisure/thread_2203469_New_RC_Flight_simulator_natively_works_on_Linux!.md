### Post by ron14 on 2014-02-03
I was tired of never having a decent RC flight simulator on Linux, so I started writing my own.  You can play it for free at [(RonsRCSim.com)]("http://RonsRCSim.com"). It runs on ANY operating system that has access to 3D accelerated video and can run the Chrome browser.  You only need a Dual axis USB game controller to play it (logitech, xbox, or ps3 controller).  I just started this project in December, and I am constantly adding new aircraft and features, but I would like to hear what new features people would like to see next.  What should I improve first (Terrain, aircraft, options, ???)  There is a lot of work to do, but it is already very usable. :)
