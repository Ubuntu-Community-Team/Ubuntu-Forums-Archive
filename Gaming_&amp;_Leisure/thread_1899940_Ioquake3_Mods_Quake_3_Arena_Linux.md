### Post by DoktorSeven on 2011-12-25
Yes, all you should have to do is extract it in your baseq3 directory wherever ioquake3 is installed.  Make sure you haven't extracted it into a subdirectory inside baseq3, the "xcsv_bq3hi-res.pk3" file should be in the same directory as the pak*.pk3 files.

And it's not a mod you have to activate via the "Mods" menu, it will just take the place of the normal textures in-game automatically.  It does display some (very dim) text on the title screen at the bottom so you know it's installed and working.
