### Post by Holger_Gehrke on 2023-04-07
And if you already have the games in a version for DOS or Windows you could try running them through scummvm. That's an open source reimplementation of engines for various classic point and click adventure games. It's in the repositories (version 2.5 on Ubuntu 22.04) and there also is a snap of a more current version (2.7).
