### Post by ad_267 on 2008-07-13

To install assault cube go to System - Administration - Synaptic Package Manager. Go to settings - repositories and make sure universe and multiverse are ticked. Click reload and search for assaultcube then mark it for installation and hit apply. Then just go to Applications - Games - Assault Cube.

If you have a newer system you should also check out nexuiz, tremulous, alien arena, urban terror and wolfenstein enemy territory.

How to install anything in Ubuntu: [http://monkeyblog.org/ubuntu/installing/](http://monkeyblog.org/ubuntu/installing/)

### Post by ad_267 on 2008-07-13
Ok sorry that was my fault, I forgot that I installed assault cube from a debian package from getdeb.net so it shows up in the package manager.

So to install assault cube you can download the .deb files from [http://www.getdeb.net/app/AssaultCube](http://www.getdeb.net/app/AssaultCube)

You need to download the assaultcube-data package and install that first just by double clicking on the file then download and install the assaultcube package.
