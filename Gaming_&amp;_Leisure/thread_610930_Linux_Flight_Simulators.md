### Post by Quadunit404 on 2011-01-24
To take off in FlightGear:

1. Make sure you actually turn on the engines first. Checklists can be found on the [FlightGear wiki]("http://wiki.flightgear.org/index.php/Main_Page") or the aircraft help dialog. Be aware that for some planes, you may need to use the Flight Engineer view to start up the engines. Or you can just select "Autostart" in the aircraft menu if that's available.
2. After the engines and everything are on, release the brakes. This is usually done with Ctrl+B.
3. Taxi to the runway. Set your flaps to at least 5 (this can be done with the [ ] keys)
4. Increase the throttle to anywhere from 80% to max using your joystick's throttle or the Home key on your keypad after lining up with the runway.
5. ???
6. PROFIT!

### Post by stlu on 2012-09-25
I found **YSFlight** about a year ago, and it's not open source, but it doesn't cost any money to download.  the author has released it in 32-bit and 64-bit binaries.

The graphics detail is very low, but I highly recommend it for anyone who just wants to try out flight sims "out of the box".  It will work on older hardware, providing that OpenGL works with the graphics card.

Also, there is a network gaming mode which allows online flights as well as combat, enhancing the playability greatly.

Before I migrated from Windows to Linux, I had downloaded several simple freeware flight sims for Windows (there are a large variety of them), and YSFlight was the only one that had network play, which made it stand out from the rest.  I am glad it was compiled for Linux users.
