### Post by martinw89 on 2007-06-29
Hello everyone,
There may be more appropriate websites to post this as well so if anyone has some suggestions mention it in the thread and I'll post elsewhere.

For anyone that doesn't know what Pingus is, it's a clone of the old DOS game "lemmings".  More information can be found on the [Pingus website]("http://pingus.seul.org/").

I am not a developer, I've learned some basic Python and C++ but I currently don't even know how to program GUIs, so I think game development would be way past my capabilities :)  However, I am very interested in getting together a team of about 4-8 people who would be interested in developing Pingus to the point where it has a full playable story.  I can contribute too, although not in programming, but in documentation, maintaining a website etc.

As far as I can tell, the project is completely dead.  The devopment wiki does not even exist anymore, a google cache version shows the last update was in January 2006.  But from what I've gathered, these are the requirements a developer would need, along with some time requirements:
[LIST]
[*]Be able to develop using the Clanlib SDK
[*]Be able to devote time consistently, we don't want to have another dead project
[*]Be able to communicate regularly
[/LIST]

If you're interested, please respond in the thread or PM me.  Some tentative positions would be:[LIST]
[*]Level Designer (possibly 2 of these)
[*]Main developer, oversees everything else, contributes to core
[*]Animator
[*]In-game documentation
[/LIST]

I am also posting this on the Linux Game Tome. I will keep a running list of people interested and start getting back to people in a week or so, depending on how much of a response I get.  If there are only 2-3 people interested I will probably not continue.
