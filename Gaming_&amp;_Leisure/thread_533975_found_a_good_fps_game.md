### Post by kostkon on 2007-08-24
I am playing this, mostly online. It is made to be lightweight. It is very good.

Just one tip: if you install the game from *getdeb.net*, don't forget to change the permissions of the *packages* folder to be accessible to all users, because you will not be able to download and play new maps. The folder belongs to *root*, but you have to give also read/write access to regulars users. 

The folder is at
```
/usr/share/games/assaultcube/packages
```

also it is accessed through a soft link from 
```
~/.assaultcube/packages
```
