i have programmed a mini game called **squeeze**.
A screenshot of the game:
[[IMG]http://nailor.devzero.de/code/glgames/website/squeeze-screenshot-small.png[/IMG]]("http://nailor.devzero.de/code/glgames/website/squeeze-screenshot.png")

Graphics and Gameplay are comparable to Flash Games, but this runs directly on the Ubuntu Desktop.

**[This site]("http://nailor.devzero.de/code/glgames/website/squeeze.htm") describes in great detail how to download, install and control the game: **[http://nailor.devzero.de/code/glgames/website/squeeze.htm](http://nailor.devzero.de/code/glgames/website/squeeze.htm)
For the less patient:
Download: [http://nailor.devzero.de/code/glgames/website/downloads/](http://nailor.devzero.de/code/glgames/website/downloads/)
Install (intrepid or later): *sudo apt-get install ruby rubygems libopengl-ruby libsdl-ruby*
Play: cd $HOME/Desktop/glgames**; pwd; ruby -rubygems ./lib/tools/launcher.rb squeeze*
