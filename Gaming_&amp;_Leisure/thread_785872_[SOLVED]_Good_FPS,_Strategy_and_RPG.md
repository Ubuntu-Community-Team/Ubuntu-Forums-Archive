### Post by sirdilznik on 2008-05-07
> **Nxion said:**
> How does  Oblivion run with wine? Is it pretty fast?

It runs fine, I can't compare it to running it under Windows because I never played it under Windows, but it runs quite well on my fairly aging machine.  It randomly crashes occasionally (I think it's a problem with the game itself not wine) so save often, but yes it works quite well.

Edit: You will need to do a couple simple tweaks to get it to run reliably.  They are described in detail in the winehq wiki for the game.  I also replaced the water texture with a simpler but uglier one as the standard water texture was a MAJOR slow down.
