> I have USB easy touch ET-29 headset and I don't know how used them in the game - Enemy-Territory (configfile.cfg: seta snddevice "/dev/dsp")

change this line in your config file (in game-> autoexec.cfg)
seta snddevice "/dev/dsp1" //dsp is default , dsp1 is USB headset
