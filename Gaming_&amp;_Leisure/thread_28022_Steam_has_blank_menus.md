### Post by claydoh on 2005-04-19
[From Transgaming's game database]("http://digital-conquest.ath.cx/wiki/index.php/Half_Life_2"), you need to install their Mozilla ActiveX control in order to have html content in steam (which is why you have no text, etc.) Instructions on how to use it [are here]("http://digital-conquest.ath.cx/wiki/index.php/Cedega-4.2#New_Mozilla_Control"). If you are using Point2Play, I think it is set up automatically after installing the control. 

The Mozilla ActiveX control can be found in Transgaming's download area.
