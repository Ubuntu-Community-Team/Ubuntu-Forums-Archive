### Post by Catboy~ on 2009-01-16
[http://www.regnumonline.com.ar/index.php?l=1&sec=6](http://www.regnumonline.com.ar/index.php?l=1&sec=6)

You'll be wanting to download the one that says "Direct Download 64-Bits"

Once it's finished downloading, make a folder in your home folder called "regnum" (or anything really, it doesn't matter what it's called.) Drag the downloaded file into that folder.

Once it's in the folder, right click it and hit properties. Go to the Permissions tab, and check the little thing that says "Allow this file to run as a program." Hit ok and double-click the file. It should be all good from there :)

Good luck!
