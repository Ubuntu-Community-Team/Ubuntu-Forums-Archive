### Post by Sofox on 2011-09-29
The source engine has been ported to Mac, so we know that it does have an OpenGL version and is somewhat cross-platform already.

As for anything else, well Grenage has the right idea. The most ideal scenario for us Linux gamers is that Postal 3 is released on Nov 3rd, it has a native Linux port on release, and other Source games get ported to Linux as a result of the work done on making a native version of the engine. However, we face risks in whether Postal 3 will actually be released, whether it will have a Linux port close to release or even at all, and whether the porting work will be able to be used by other developers.

We'll just have to wait and hope.
