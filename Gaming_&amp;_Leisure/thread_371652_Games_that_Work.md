### Post by Sammi on 2007-02-27
You can not install DirectX in Linux. Microsoft owns DirectX and hell will freeze over before they make it installable in Linux.

However there are project, like Wine, Crossover, and Cedega, that are trying to make DirectX based applications and games work in Linux. The way I understand it they are doing this by implementing a compatability layer that translates system calls made by DirectX into "Linux language" by reverse engineering DirectX. This creates some overhead and the work is far from being done. Wine isn't even in beta yet.

Some games already work pretty well, while others don't. For example WoW, Warcraft 3, and all Source based games work pretty well. You can check out this page to find out what does and doesn't work in Wine: [http://appdb.winehq.org/](http://appdb.winehq.org/)
