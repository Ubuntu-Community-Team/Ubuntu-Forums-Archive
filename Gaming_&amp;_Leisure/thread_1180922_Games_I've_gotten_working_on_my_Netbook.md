### Post by Ferrat on 2009-06-07
Well would be fun to see what people have been able to get working on their netbooks, start with a description of your netbook and then the games that you have working on it and any need to know info =)

**Netbook**
Name: Compaq Mini 730EO 
CPU: Intel ATOM N270 (1,6GHz)
RAM: 2048DDR2
Graphic chip: Intel 945GME (256MB shared)
Running Ubuntu 9.04 Netbook Remix


**_Games working_** - With reasonable FPS and Screen size not interfering
**OpenArena** - Out of the box 
**Freespace2** - "Out of the box with linux installer and executable"
**Battle of Wesnoth** - Out of the box, meny missing exit due to cutting but ingame it's flawless
**Neverwinter Nights [WINE]** - Keep getting seg fault with native but works flawless with WINE, also resolution needs to be set manually in config file, movies work sometimes even with the odd resolution. 
**UFO: Alien Invation** - Out of the box needs windowed mode or you lose some view due to resolution. 
**Lincity-NG** - Works Out of the box if you use the command ***lincity-ng -g -f -S 1024x576*** (or your resolution) due to Lincity not liking the "weird" resolution.
**Homeworld 2 [WINE]** - Works out of the box but needs to manually set resolution in config file to make it look good.

**_Emulators_**
**Gens** - Out of the box
**Zsnes** - Out of the box 
**pSX** - Out of the box

**_Games not working and why_**
**Regnum Online** - Seg faults on start up and current fixes doesn't seem to work, this is an issue with Intel drivers I think, some people seem to have been able to make it work but I couldn't
**Guild Wars [WINE]** - Gotten it working by disabling pixelshader in winecfg, playable at a steady 5FPS, sounds worse than it is since Guild Wars gameplay isn't that FPS demanding. Wouldn't do PvP but missions and so on isn't really a problem since it's only the FPS that's slow and not the game itself (shitty drivers for 3D :P)
Working on getting the FPS up, want it at 15+ at least :)

If you know any solutions to this please let me know =)
