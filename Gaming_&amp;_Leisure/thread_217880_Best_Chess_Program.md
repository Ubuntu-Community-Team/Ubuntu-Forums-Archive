### Post by grim1234 on 2006-07-18
I think the best way to improve at chess is to play real people, of the same or slightly greater skill that you. To this end :

[Freechess]("http://www.freechess.org")

you will need a client program for the board. Use xboard : 

> sudo apt-get install xboard

After you've made an account you can connect to the server using :

> xboard -ics -icshost [www.freechess.org](www.freechess.org)

This will bring up the chessboard and the login prompt for the freechess server. 

This way you can play based on your rating, which adjusts as you win or lose. This way you should be able to find challenging opponents of about your level.

If you want to play against a computer then you could try gnuchess, but I haven't tried it. You can play gnuchess using xboard without problems I think.

Let me know how you get on.

:D

ps.

If the board appears too large then use :

> xboard -size medium -ics -icshost [www.freechess.org](www.freechess.org)
(can be small / large too)
