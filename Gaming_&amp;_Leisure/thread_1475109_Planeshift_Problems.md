### Post by sanone on 2010-05-24
Probably planeshift uses OpenAL which tries to connect to bluetooth device and displays the following error when you don't have a bluetooth device in your computer. 

```
bt_audio_service_open: connect() failed: Connection refused (111)
```

You can remove the bluetooth package and remove the error.

```
sudo apt-get purge bluez-alsa
```

Source: [https://bugs.launchpad.net/ubuntu/+source/espeak/+bug/379041/comments/8](https://bugs.launchpad.net/ubuntu/+source/espeak/+bug/379041/comments/8)

Btw it shows some shader (graphics) errors as well.. 

Goodluck
