### Post by Dark_Stang on 2009-10-25
Do you have compiz turned on? Try disabling desktop effects if you do and see if that helps.

Also, check your wine configuration. Applications > Wine > Configure Wine. Under the graphics tab you will see some options. Make sure "Allow the Window Manager to Decorate Windows" and "Allow the Window Manager to Control the Windows" are both checked.
