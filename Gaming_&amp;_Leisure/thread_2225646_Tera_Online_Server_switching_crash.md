I've been able to run Tera flawlessly ever since the latest Wine update. The only problem is I wanted to try a new character on a different server and when I tried changing servers the game crashes on me. Any ideas as to how to fix this? ### Post by jbaerboc on 2014-05-22
Alrighty found a simple resolution to this. To make Tera run a bit faster I had been closing the Launcher after the game started. Apparently this makes it unable to switch servers. I left the launcher running this time and server switch was no problem. So *meh* live and learn :-).
