### Post by jharris on 2006-07-14
Try [here](http://www.truecombat.com/intro.php?x=d), there is a list of mirrors to choose from. Don't forget to grab the 0.48 patch as well, otherwise you won't be able to play online correctly. Btw, you can launch the game from within ET's mods window or you can create a launcher with ```
et +set fs_game tcetest
``` in the command parameter.

I've found the full 0.48 run file: [http://www.liflg.org/?what=dl&catid=6&gameid=52&filename=true.combat.elite_0.48-english-5.run](http://www.liflg.org/?what=dl&catid=6&gameid=52&filename=true.combat.elite_0.48-english-5.run)
