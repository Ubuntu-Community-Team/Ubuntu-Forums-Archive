### Post by Version Dependency on 2011-11-25
I don't know any FPS that runs natively on Linux that compares to any of the recent CoD's.  But if you aren't anti-Wine, the CoD games tend to work well in Wine. 

Probably the closest native Linux FPS would be something like Urban Terror.  Graphics aren't mind-blowing but the game play is good and has a large amount of users.

PS: If you try to install any of the CoD games in wine, you may need to do what is suggested in this [post]("http://ubuntuforums.org/showpost.php?p=8313625&postcount=2") to get it to run.

### Post by HolgerB on 2011-11-25
Hi there,

I wonder why someone references Red Eclipse ? :confused:

Under Linux you mostly have more futuristical FPS games with deathmatch (Nexuiz, Alien Arena, Open Arena, Xonotic) which somewhat resemble UT, Q3A. The other type of games you find is something like Counterstrike (e.g. Urban Terror).

You might want to check out here:
[http://www.techdrivein.com/2011/07/top-10-fps-games-for-linux.html](http://www.techdrivein.com/2011/07/top-10-fps-games-for-linux.html)
or here:
[http://www.cahilig.net/2011/07/29/100-best-free-and-high-quality-linux-games](http://www.cahilig.net/2011/07/29/100-best-free-and-high-quality-linux-games)

Assault Cube is fast, somewhat fun but I found the graphic really annoying. 

True Combat - Close Quarters Battle is really great but the last time it was a PITA to get a running game (download here & there, patch this, modify that config and so on).

But back to your question:
As a sad fact there simply is no game similar to those Call of Duty or Battlefield type original for Linux. There isn't pretty much interest in Linux as gaming platform by bigger companies anyhow and all those AAA titles with great graphic require a lot of work to produce such nice looking content. Hence a COD7 for Linux is nothing we can really expect.

I had big success playing COD4 Modern Warfare ages ago under WINE. For newer versions you might check out WINEs AppDB.

HTH,
H.
