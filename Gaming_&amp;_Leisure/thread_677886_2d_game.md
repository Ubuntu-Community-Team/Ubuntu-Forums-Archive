### Post by ::: on 2008-01-25
Well, there's [OHRRPGCE]("http://hamsterrepublic.com/ohrrpgce/index.php/Main_Page.html"), which seems quite decent. It isn't Java, however: The game creation engine itself is written in Basic and games are scripted in some obscure language called "hamster speak". So, if you want to do your game in order to practice coding this might not be for you ... if you, however, just want to make a game with minimal (programming) effort OHRRPGCE might be worth a look.

If you want to code a game from scratch, I would recommend  [pygame]("http://pygame.org/news.html"), which is a SDL-Wrapper for Python (sorry, not Java either - but it's really great for making 2d-games)


If you want any specific information about how to program an RPG (as opposed to learning how to use a framework), you might want to have a look at [some roguelike develepment related articles]("http://roguebasin.roguelikedevelopment.org/index.php?title=Roguelike_Dev_FAQ"). It's not exactly the same style of RPG you are after, but maybe you can use some of that information.
