### Post by Carpentr on 2012-01-10
If you want to start learning C++ as your first language I'd recommend *_C++ Programming: From Problem Analysis to Program Design_* by D. S. Malik. It's not the easiest book to read, but it will teach you a lot. It used for one of my college courses.

As long as you pay attention and read the material you'll be fine. It's fun to learn and the book has some great exercises.
