Those specs should run Kerbal just fine; is the nVidia card enabled in the BIOS, or the onboard i3 graphics? Also, which nVidia drivers are you using? If you're using the nVidia drivers, you probably should be okay. If you're using the open-source Nouveau drivers, they are inferior to the nVidia drivers. This is the opposite of ATI drivers, in which the open-source radeon drivers are better than ATI's fglrx drivers.
