### Post by doorknob60 on 2009-01-02
Text based system for emulators? No. Put a lightweight Window Manager like Openbox on there, so it will still be a GUI, but it won't use a ton of RAM. If you're up to it, I recommend a Debian Netinst, Arch, or Ubuntu Server install, and build it up from there. Here 's a good wiki page that will be useful for any distro. Also you can IM me and I can help you set it up if you want. [http://wiki.archlinux.org/index.php/Openbox](http://wiki.archlinux.org/index.php/Openbox)
