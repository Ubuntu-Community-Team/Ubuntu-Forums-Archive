I just noticed that [Bit.Trip Runner]("http://bittripgame.com/bittrip-runner.html") has been added to the U.S.C. It's a real gem, and worth taking a look at.  As far as I can tell, the Ubuntu Software Center is currently the only place to buy it for Linux!
