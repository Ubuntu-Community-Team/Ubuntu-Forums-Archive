### Post by mario1o1 on 2009-01-30
hey i have the same problem, ive spent like 2 hrs today searching it up. you need to get the latest verison of java runtime environment and the web browser plugin. you might have the old version as you default. you can go to your firefox web browser and type in "about: plugins" in the url space to see if the latest one is your default. if it is not then go to terminal and enter "sudo update-alternatives --config java" and choose which one to use. then go to your web browser again and enable java. but if your running 64 bit linux like i am, the latest java web plugin is not available right now =(. hope this works.
