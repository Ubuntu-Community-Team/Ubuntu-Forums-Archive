### Post by hikaricore on 2007-04-09
You will need to install and configure WINE:
[http://wine-wiki.org](http://wine-wiki.org)
[http://winehq.org](http://winehq.org)

Ubuntu/Debian packages for WINE:
[http://wine.budgetdedicated.com/](http://wine.budgetdedicated.com/)

You may have some luck with Diablo, but don't quote me on that:
[http://appdb.winehq.org/appview.php?iVersionId=3498&iTestingId=8180](http://appdb.winehq.org/appview.php?iVersionId=3498&iTestingId=8180)

Diablo II runs near perfectly tho if you want to try it.  :)

Dreamweaver seems to work for some and not for others:
[http://appdb.winehq.org/appview.php?iVersionId=3482](http://appdb.winehq.org/appview.php?iVersionId=3482)

Search the forums for more info on wine and common issues with wine.
This will not be an easy journey, but many windows applications can be run in linux.

If dreamweaver is just a WYSIWYG html editor there are several opensource replacements to be found.
[http://ubuntuforums.org/showthread.php?t=293798](http://ubuntuforums.org/showthread.php?t=293798) (look under web design)

NVU is a nice one: [http://www.nvu.com/screenshots.php](http://www.nvu.com/screenshots.php)

Also you could try Scream in the repos.
