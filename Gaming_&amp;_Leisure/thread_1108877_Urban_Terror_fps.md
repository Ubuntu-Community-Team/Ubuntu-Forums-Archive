### Post by 5nak3 on 2009-03-28
the OP might be having this problems for a number of reasons though.

Maybe the driver isn't loaded properly, maybe he is usinga laptop (so no new gfx card), maybe high textures, trilinear filtering and compress textures might be enabled.

There are a number of reasons. If the OP is anything like me and playing on a laptop with a poor gfx card try this:

Lower the res to 800x600
Have medium quality textures
Bilinear filtering
Compress textures: yes
Turn off compiz
Turn off or limit the time that various settings such as brass and blood are shown

This in itself should allow you to hit 30+ FPS on average. Not great by any means, however, it makes the game playable, and I'm using a 5yr old laptop, with an ATi card...

Hope that helps.

### Post by rizzeh on 2009-03-28
create a new file, name it autoexec.cfg. Save it in /home/.openarena

You can load any config file from /home/.openarena directory by typing command in game console:
exec yourfile.cfg


File name autoexec.cfg executes automatically when the game starts.
