### Post by Naegling23 on 2008-10-30
Is hexen a quake or a doom based game?

Either way, both engines were open sourced, so you can find some variation of those to run on linux.  The "wad" file is a package of maps, skins, textures, etc used by the game engine.  You can find all sorts of wads in the dark reaches of the internet...honestly, I cant tell you where because im actually not sure...and they are usually not legal anyway.  However, if you actually own a copy of hexen, you should be able to extract the wad file yourself.  
If you really want to play hexen, might as well play the real thing.

disclaimer:  Im only going by what I know, I dont have hexen installed, so my facts may prove to be inaccurate.
