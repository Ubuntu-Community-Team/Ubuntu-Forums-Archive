### Post by WienerWuerstel on 2010-11-22
Do you mean Tekken Tag Tournament for the PS2? If so than you can play it with the PS2 Emulator called PCSX2. You can install it with a couple simple Commands.
```
sudo add-apt-repository ppa:gregory-hainaut/pcsx2.official.ppa
``````
sudo apt-get update
``````
sudo apt-get install pcsx2
```

### Post by WienerWuerstel on 2010-11-22
There are no Official Minimum Hardware Requirements. But you will need a strong PC to play PS2 Games with a decent Framerate.

Dual Core or Quad Core CPU (But no Atom)
Modern AMD or Nvidia Graphics Card/Chip (Intel is not strong enough)
1GB Ram or more

And if you want to know more about running PS2 Games on PCSX2, I would suggest you to go to [http://forums.pcsx2.net/](http://forums.pcsx2.net/)[]("http://forums.pcsx2.net/")
