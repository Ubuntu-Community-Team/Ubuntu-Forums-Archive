### Post by michael-piziak on 2015-04-11
I beat the game Mickey Mania, using Kega Fusion (Sega Genesis emulator).

Each level on the game is a throwback to a classic Mickey Mouse cartoon.  I watched each cartoon on youtube as I played the game through.

This is the most difficult Disney video game (to beat) I've played.  I seriously doubt a young child could beat this game.
