There is a nice little [PPA]("https://launchpad.net/~ika/+archive/ppa") available for Ika. Just open a Terminal and run those Commands and you have Ika up and running faster than i can say "Sephiroth".

```
sudo add-apt-repository ppa:ika/ppa
```
```
sudo apt-get update
```
```
sudo apt-get install ika ika-editor ika-engine ika-examples ika-games
```
