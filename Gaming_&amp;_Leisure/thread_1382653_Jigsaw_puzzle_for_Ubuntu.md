### Post by webmayo on 2010-01-16
Anyone looking for a jigsaw puzzle for ubuntu should take a look at version 2.5 of xjig. I have seen comments on another thread(now closed) about xjig being hard to use. This is no longer the case.

If xjig 2.5 is used with Xjig Manager, it is very easy to use, and the control of puzzle pieces can be set as you like.

Xjig Manager is a front end launcher for xjig 2.5 or later.

The options in this version are not available in the official version 2.4  in the repository.

xjig-2.5-2 and the manager are available as deb files from sourceforge.net
[http://sourceforge.net/projects/xjigmanager/files/]("http://sourceforge.net/projects/xjigmanager/files/")

Hope this is helpful to anyone looking for a jigsaw puzzle.
