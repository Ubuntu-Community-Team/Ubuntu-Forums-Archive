### Post by elebrin on 2005-10-21
- Point2Play is included if you buy their service
 - either way, you can install them through Point2Play or just run them with cedega
  - there are a few, but setup of cedega is real easy.  If you buy the support and dl the latest Point2Play package, just run:

sudo dpkg -i point2play-xx-whatever.deb

and it will install.  There is a script to install cedega from CVS but it dosnt always work.  Once you have point2play it should be in the menus somewhere, so start it up.  Run the tests, and then install cedega through it.  Then, read their forums about setting up games.

Good luck.
