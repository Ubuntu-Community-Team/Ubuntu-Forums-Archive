### Post by cogadh on 2007-09-20
Go to Wine's Application Database and look those games up. The information there will tell if they work and, more importantly, what you might need to do to make them work:
[http://appdb.winehq.org/](http://appdb.winehq.org/)

As for your keyboard problem, it can sometimes be fixed by checking the "Allow the window manager to control the windows" option on the Graphics tab in winecfg. Alternatively, you can try running the game in its own X session, which almost always corrects keyboard problems. Click on the "Stuff I've learned about Wine" link in my sig and go to the "Advanced Stuff" section, running the game in a separate X session is explained in detail there.
