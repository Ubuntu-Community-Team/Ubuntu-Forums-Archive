### Post by nille on 2008-10-21
> **jerome1232 said:**
> Linux 2.6.24-21-rt x86_64.

Honestly, installing the rt-kernel can help a lot, so try that! The rt-kernel is compiled with CONFIG_HZ_1000=y and some other settings (making it suitable for low-latency audiorecording.. and playing ETQW! ;)).

Just do:
```
sudo apt-get install linux-rt
```

Reboot and choose the rt-kernel in GRUB. And, since you installed your nvidia-drivers manually, you might be interested in the flags **-K** and **--kernel-name** when installing your graphics drivers. Check the manual for more info:
```
man nvidia-installer
```
