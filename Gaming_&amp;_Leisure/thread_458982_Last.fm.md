Looks like last-exit has a patch to enable it record the streaming, or so says [this guy]("http://raphael.slinckx.net/blog/2006-03-06/deskbar-and-leaftag-last-exit"):

*If you feel the need to save the song you&#8217;re currently listening on, you can use a secret patch (ask for it in IRC) that allows to dump the stream and nicely save the playing song when it has finished streaming under a given directory/artist - album - track.mp3*

Automatically split, that would be great, but hey, we can always use some sound editing software to do it ;)

Okay, so, I found the patch [here]("http://ubuntuforums.org/showthread.php?t=213185") and you can download the deb package (last-exit already patched) [here]("http://rapidshare.com/files/334255/last-exit_3-1_i386.deb.html").
