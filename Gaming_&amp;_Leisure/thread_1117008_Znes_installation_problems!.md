### Post by Chemical Imbalance on 2009-04-05
A huge number of packages are included in the Ubuntu repositories.  The repos are basically large storehouses of Ubuntu software that is officially supported, tested, and safe.  The windows way of downloading programs off of the internet is best to be avoided as the repos are safer than downloading off some random site.

Some ways to install in Ubuntu:
Open up Synaptic Package Manager under System, Administration
or
Use ```
sudo apt-get install "package-name"
``` in Terminal if you know the package's name.
