### Post by rogerdg on 2009-10-12
I've been having fun with this game today.  I was having difficulty because the menu with the save option on it kept disappearing.  When I pressed ESC to bring up the menu it would appear, then immediately disappear.  I finally managed to get it to stay on the screen after many attempts.  I went into the CONFIG option and changed the SYSTEM MENU KEY to =.  Now when I press the = key the save/configure/exit menu appears and always stays open until I exit or press ESC to close the menu.  This is much better than the default key the programmer chose for this function.
