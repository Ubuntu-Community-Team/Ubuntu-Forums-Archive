### Post by oldrocker99 on 2018-07-07
In the Ubuntu mouse properties dialog, set mouse acceleration to "slowest." that should solve your problem.

You might want to try PlayOnLinux instead of using a VM. ```
sudo apt install playonlinux
```.
It will likely run the games you want to play. It can't hurt to try.;)
