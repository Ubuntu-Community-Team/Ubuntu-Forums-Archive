Came across this game today: [http://www.warsow.net/?p=download](http://www.warsow.net/?p=download)

Its has a linux precomiled distro of the game that run natively on my kubuntu box with no hassles.

The game plays very cleanly, and is dam fun to play. If you are in South Africa, I know there is a local server sponsered by Web Africa, that is often populated in the evenings ( warsow.webafrica.co.za )

The game is very quake like, but loads of fun. And its 100% open source.

All you need to do is basicaly extract the tar file, and then start running the game with a simple ./warsow command from command line.

Hope you enjoy it.
