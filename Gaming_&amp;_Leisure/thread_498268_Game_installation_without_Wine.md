### Post by cogadh on 2007-07-11
Some Windows games (very few) can work natively in Linux, provided a Linux binary has been created to replace the Windows executable. The game also must use OpenGL instead of the DirectX API. Otherwise, you will need to use Wine or one of its variants to run Windows games in Linux.

As Feba said, Wine is not an emulator, it is an open-source implementation of the Windows API. Basically Wine allows Linux and the Windows executable to understand each other.
