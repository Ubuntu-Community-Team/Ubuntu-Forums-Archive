### Post by oldrocker99 on 2009-08-02
I remember it fondly from about 1999 for Windows. It was made open-source and it's alive and well for Linux. You can install it from Synaptic, too, if you want.  Gamefaqs.com has some user-written walkthroughs, which can be a help if you're just jumping into it. It can be a lot of fun with tons of explosion sounds and okay graphics. It basically slipped beneath most people's notice, with Command & Conquer occupying most people's minds, not to mention Warcraft II.

By all means install it. It is, right now, the best real-time strategy game available for Linux.
