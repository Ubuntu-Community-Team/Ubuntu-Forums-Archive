### Post by Technophobia on 2009-12-09
After some searching I have found a way to auto join a mumble server on boot.

Here is an example:

```
mumble mumble://mumble.server.net:64738/?version=1.2.0
```

or

```
xdg-open mumble://mumble.server.com:64738/?version=1.2.0
```

You can add this "xdg-open mumble://" link as a startup application or just a launcher. Thanks to whoever wrote xdg-open.

Enjoy.
