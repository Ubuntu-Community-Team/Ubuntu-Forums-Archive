### Post by damis648 on 2008-12-25
Is there no driver in System>Hardware Drivers?
If not, I recommend Envy. Open a terminal and enter:
```
sudo apt-get install envyng-core envyng-qt
```
and then launch Envy via the applications menu and let it install the driver for you.
