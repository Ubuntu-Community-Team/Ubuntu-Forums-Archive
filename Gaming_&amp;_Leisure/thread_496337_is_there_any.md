### Post by FoolsGold_MKII on 2007-07-09
It is possible to get BF2 working in WINE, but no-one has managed to get PunkBuster working for it as well and hence you'll be restricted to non-PB servers, which is no fun. I expect CoD would have similar issues, althought it might be easier to run since it's OpenGL-based.

Just download the linux version of WolfET. It's free, plenty of servers, native for Linux and uses the Linux version of PunkBuster. About as close to a Battlefield game as you're going to get on Linux.
