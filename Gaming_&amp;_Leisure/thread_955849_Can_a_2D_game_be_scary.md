### Post by Abras on 2008-10-23
In theory, yes they can. But, unfortunately, there aren't exactly a ton of 2D games that even attempt to be scary which can test that theory. I think this is partially because horror games weren't all that popular until Resident Evil, in the dawn of the 3D. So naturally, just about all of the horror games that have followed RE have been 3D.

Those kinds of "survival horror" games have always revolved around creating an atmosphere of fear and hopelessness, and never really on realism. Given that, I see no reason why a 2D game couldn't replicate that atmosphere. Actually, now that I think about it, the early RE games would have worked just fine as 2D top-down shooters (with the added bonus of less-clunky controls). 

And not only do I think 2D games can pull off scary, but I think games with absolutely no graphics, meaning text adventures, could be scary. I'm not a big text adventure guru, but I'm sure there are at least a few scary IF games. Actually, Interactive Fiction seems like an ideal platform for a scary game, since it combines the advantages of both video games and books.
