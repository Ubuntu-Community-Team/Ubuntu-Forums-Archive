### Post by Vadi on 2007-11-20
See if you can play the game? It should be installed already. If you can't, see if you can uninstall, and install again.

And yes, generally, after you install a game you *don't* want to start it from the installer - instead just finish with the installer, and then start it yourself.

The reason for that, I think but I'm not all sure, is that the installer runs as root - so if you start it as root, it'll write all it's config stuff into root's settings. Which then you won't be able to access. But if you start the game yourself, you'll be OK.
