### Post by hikaricore on 2007-03-20
If the games are supported by WINE or Cedega you should be able to run them.

But if they are heavy in graphic usage or require extensive d3d, your system may be underpowered to run them under wine.

I don't know the location of Cedega's supported list, but you can check the[ wine application database]("http://appdb.winehq.org") for the titles you mentioned.
