### Post by baylisscg on 2005-02-17
They are windows only unfortunately. They take the official drivers and adjust the internal, normally hidden from the user, settings to enable features / improve performace. Unfortunately windows drivers can include code that adjusts, read turns down, the settings for specific benchmarks to improve performance. The omega, and other tweaked, drivers turn this sort of thing off. 

You can get more info if you select "ATI Radeon" then "Omega FAQ" on the menu bar under the banner. Sorry.
