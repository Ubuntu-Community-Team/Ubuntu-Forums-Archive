### Post by koshari on 2007-04-10
same as normal except you need the wine path before the .exe path , for example, 

> wine /home/holto/.wine/drive_c/Program\ Files/uTorrent/utorrent.exe

the backslash is needed to indicate there is a space in the path."Program\ Files" = Program Files.
