### Post by ZOMGxuan on 2012-01-29
Pixelchase is an open-source, cross-platform game that I've been developing since 2007, but it is only recently that I've decided that it is polished enough to be shared on these forums. To describe it shortly, it's the game of tag on your computer screen. A longer description follows:

"Pixelchase is a simple 2-player game written in C++ using the Allegro game programming library. The objective of the game is to stay as the blue pixel, the runner, for as long as possible. The red pixel, the chaser, must attempt to catch the blue pixel by moving around, and the blue pixel must, of course, try to escape. Once the red pixel touches the blue pixel, their roles change, with the chaser now becoming the runner, and the old runner appearing randomly on the screen as the chaser.

The blue pixel gets points while running away, getting 1 point every 1/10 of a second. The first player to reach 1000 points wins. One can also choose to play against the AI to hone one's skills, or watch 2 AIs duke it out against each other. But nothing is as fun or funny as playing with one's friends."

Before you get bored, here are some (slightly outdated) screenshots:

Here's a short list of features:
Open-source
Cross-platform (Windows and Linux, Mac OS X compilation theoretically achievable)
AIs with configurable difficulty
In-game music
The most retro artwork you'll find in any game (as evidenced by the screenshots)!

If all this interests you, head down to [http://pixelchase.sourceforge.net](http://pixelchase.sourceforge.net) and hit the downloads section! For such a simple idea, it is surprisingly fun to play, at least for a short while. Try it and see for yourself! Easy-to-use installers are available for both Windows and Linux (.debs and .rpms). Hope you all enjoy playing, and feel free to leave comments on what you think about the game! :)
