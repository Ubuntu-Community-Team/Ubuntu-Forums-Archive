### Post by derekr44 on 2007-08-31
Check your realmlist.wtf file

You can find it by doing the following:
[LIST=1]
[*]Open your Home folder
[*]Select View at the top and check the Show Hidden Files option
[*]Open .wine
[*]Open drive_c
[*]Open Program Files
[*]Open World of Warcraft
[/LIST]

Your realmlist.wtf file should be there.  That displays the domain and ip of the WoW connection server.
