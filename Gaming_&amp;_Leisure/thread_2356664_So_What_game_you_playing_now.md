### Post by minecralex on 2017-04-10
I'm mostly spending my time playing Cities: Skylines nowadays (on steam), and if anyone is wondering, this is game is definetly in my top 5 if not top 3 best games I have ever played. This game will give you hundreds of hours of entertainment and yet is much cheaper on steam than the standard AAA games which usually go for 60 bucks. If there is one thing I'd recommend buying on steam is Cities Skylines. For anyone who doesn't know, Cities Skylines is a city building game much like SIm City, except it is much better than Sim City. Runs great on Linux too! 

Other than Cities Skylines, I also play a lot of minecraft, took me a while to understand how to get it to run on linux properly once I've switched over to Zorin from *gasp* Windows 10. Got tired of the updates, spyware, price tag, and all the usual crap that Windows usually is.

I also sometimes play GTA Online on my Windows 10 PC. (Although I spend most of my time on my linux machine, who said I was gonna delete Windows? Unfortunately, Windows still has the edge when it comes to those types of games, planning on trying to get GTA San Andreas to run on Wine/PlayOnLinux

Another great game that runs on Linux is "Rising World". This is one of my most favorite indie games, made by a tiny game studio in Germany, this game has some of the best graphics that I have ever seen. Unlike most indie games, this game is simply stunning in its graphics. The game is much like Minecraft as it has an infinite voxel based open-world procedurally generated map which the player has to survive in. It is also a sandbox. I won't go much into describing this game, but if anyone wants to know more, please search  the game up online, seriously, this is a very great game. Its only like 15 bucks on steam too.
