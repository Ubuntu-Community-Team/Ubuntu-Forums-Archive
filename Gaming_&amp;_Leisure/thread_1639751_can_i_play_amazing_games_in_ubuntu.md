### Post by TenPlus1 on 2010-12-07
A friend of mine has Ubuntu 10.10 installed on a dual-screen HD setup and has Wine 1.3 installed to run his Windows games...  So far they work pretty well with the odd glitch that can be fixed by visiting winehq.com

To install the latest wine do this:

1. Run Applications -> Accessories -> Terminal 
2. Type:   sudo add-apt-repository ppa:ubuntu-wine/ppa   and hit <Enter>
3. Close Terminal and Run System -> Administration -> Synaptic Package Manager
4. Click Reload button and then Search button, type "wine" and hit <Enter>
5. Select wine1.3 and any additional packages it will asks for
6. Press Apply to install and close Synaptic when finished
7. To install your game, right-click on the setup.exe or install.exe and run with "Wine Windows Program Loader" to install, most times this will work and your game appears in the Application -> Other menu.

Enjoy...
