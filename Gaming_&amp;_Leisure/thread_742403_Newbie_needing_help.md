### Post by forrestcupp on 2008-04-01
Are you using Ubuntu?  If so, go to System->Administration->Software Sources and in the 'Ubuntu Software' tab make sure all of the boxes are checked.  Then open a terminal and type
```
sudo apt-get update
```
Then try installing it again.
