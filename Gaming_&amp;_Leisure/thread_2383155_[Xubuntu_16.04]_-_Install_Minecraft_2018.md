### Post by vodkaspeed on 2018-01-21
For those new Ubuntu users (including Xubuntu, Lubuntu, and other flavours) having a hard time trying to get Minecraft running on their new OS.

Run these commands, one by one, by copying and pasting them in the terminal. 


[LIST=1]
[*]sudo add-apt-repository ppa:flexiondotorg/minecraft
[*]sudo apt update
[*]sudo apt install minecraft-installer
[/LIST]

I usually go through a longer process (e.g. installing java, creating a launcher, and other stuff) and always resort to searching for older guides, a lot of which are still relevant. Just running these commands should allow you to locate a fully executable Minecraft launcher.

Credit goes to **Ubuntu Made Simple**. 

[https://www.youtube.com/watch?v=sgmdTBsPVBU](https://www.youtube.com/watch?v=sgmdTBsPVBU)
