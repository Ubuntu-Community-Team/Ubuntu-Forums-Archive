### Post by meng on 2006-10-10
In order to get a Windows program running on linux, you need to use a Windows compatibility layer - wine or one of its offshoots (Crossover Office or Cedega). Cedega is probably the best bet. Windows programs do not run natively in Linux; neither do Linux programs run in Windows.
