### Post by cogadh on 2007-08-25
Look at your xorg.conf file for the "Module" section. It should look a little like this:
```
Section "Module"
    Load           "i2c"
    Load           "bitmap"
    Load           "ddc"
    Load           "extmod"
    Load           "freetype"
    Load           "glx"
    Load           "int10"
    Load           "vbe"
EndSection

```
We're looking for the **Load     "glx" **entry. If it isn't there, add it, then reboot.
