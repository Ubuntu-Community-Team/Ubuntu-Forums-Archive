### Post by lakersforce on 2007-04-21
For those of you who, when running sh et-linux-2.60.x86.run on a x64 distro recieve those annoying errors:

```
./setup.sh: 278: ...
./setup.sh: 289: ...
```

simply install this package:

```
sudo apt-get install ia32-libs
```

Viola! Your problems are now a thing of the past.
