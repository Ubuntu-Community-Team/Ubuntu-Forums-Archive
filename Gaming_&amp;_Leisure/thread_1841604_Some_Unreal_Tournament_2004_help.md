### Post by holastickboy on 2011-09-15
I have a few editions of UT2004 and can tell you exactly what to look for eh.  I have the latest anthology disc, and it does not come with the Linux installer on the disc (the anthology is the one with all Unreal games on it)

If you can find the original versions of the game, especially the CD version, it does come with the Linux installer.  I have UT2004 installed natively on my system, and it works no problems, so don't worry about WINE unless you are having some serious issues.  Eitherway, I recommend you also install the latest patch after a fresh install.

EDIT: Can confirm that the windows cd key works in Linux and Mac version of the game, they are all the same!
