### Post by tsav87 on 2007-07-26
Turn down the graphics?  HELL NO!! haha, All of the games run FASTER with the graphics turned all the way up than when the graphics where normal on a Windoze install.

Windoze has way to many hidden apps running which slows down the game, Linux dosn't have so many apps running and the game runs smooth with the graphics at full blast, shadows, reflections and all.

What I did to get CS working was having the latest version of Crossover from CoadWeavers installed and using it to install Steam (steamgames.com) then inside steam I bought CS:S in the steam program  (you can also get the original one for about half price).  After it downloaded/installed (it does it in one process) I ran the game, worked without any problems.  I did the same for HL2 and COD2.

One thing that I would stress, if you use Beryl or Compus, turn it off, otherwise it will not allow you to go full screen, basically it will show the panels while you play the game.

And if there is ever a time when you don't get any sound, run this command in the terminal.
```
killall esd
```

### Post by cogadh on 2007-07-27
I don't think anyone can really debate whether or not AA is a propaganda tool; the US Army flat out said from the start that they were making it to get young people interested in military service. However, what they ended up with is actually a very good FPS game. I don't know if it really works as a recruitment tool, but the game itself is no better or worse than any number of other military shooter games (Call of Duty, Battlefield, Medal of Honor, etc.). In some ways it is even better than those games. I found the whole training tutorial to be a very interesting take on military gaming, it gave it almost an RPG feel.

Besides, other than expense; which is really only a drop in the bucket for the US military budget; what is wrong with using the technology of the day to reach their target audience? The military already advertises on radio, tv, bilboards, magazines... even comic books. None of those advertisements actually show the "true horrors of war", why should this ad be any different?
