### Post by FG123 on 2007-11-14
Don't bother installing from the CDs, you don't have to. Just download the Steam client manually, install it, type in your account and start downloading the content from scratch through it.

If you have an existing installation on a separate Windows drive, you can copy all the GCF files from the Steamapps folder in the Windows Steam install, into the Linux steam install. That will save you from downloading.

Otherwise, if you still want to do things the way you already are, unmount the drive cleanly by right clicking the CD icon on the desktop and select Eject. Preferable to just ejecting by pressing the button on the drive.
