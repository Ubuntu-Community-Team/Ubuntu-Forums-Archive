### Post by mach-schnell on 2009-01-10
Any fan of Freespace 2, who has the original Freespace 2 cd's, should check out a project called "freespace open."  The people at Volition made Freespace 2 open source years ago, opening the possibility of added content (such as new campaigns, improved textures for old and new, etc.).  The project's homepage is located here: [http://www.hard-light.net/]("http://www.hard-light.net/")

Poke around to see if it interests you.  A detailed installation guide for linux can be found here [http://www.hard-light.net/wiki/index.php/The_fs2_open_on_Linux_Guide]("http://www.hard-light.net/wiki/index.php/The_fs2_open_on_Linux_Guide")  Again, please note that installing freespace open requires the original game data found on the cd's.
