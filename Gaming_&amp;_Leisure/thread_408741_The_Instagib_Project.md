### Post by blanky on 2007-04-13
Hey guys, for a while now I've been working on a game, it's been a year and two months since I started working on it. It uses the GPL'd Quake 3 Engine, specifically, the enhanced and patched [IOQuake3](http://ioquake3.org) engine. The game is called **The Instagib Project**. The game is pure GPL; both source and media. We actually use [Open Arena]("http://openarena.ws")'s GPL'd game media, so most of our stuff looks just like theirs, the only difference is the game play.

[Instagib](http://en.wikipedia.org/wiki/Instagib) is a common mod for many games which basically creates a game play where you're only given one gun, usually the gun in the game that shoots a rail type of shot (*Like the railgun in Quake 3*), you're given unlimited ammo, and one shot kills. Some might think this might get old fast and would be really cheap because people would get kills easily, well, believe me when I say it isn't. Because of that very fact, players have to have fast reflexes and have to learn how to be quick and precise to stay alive. The problem with Instagib is that it's usually just a mod to games, that's right, just a **mod**. There's usually an Instagib mod for every game that comes out that allows modification, and although some Instagib Communities might be pretty big, most are small and they eventually die out, I know because I was a victim of this, from the Jedi Academy, then Jedi Outcast Instagib Communities (*Which they called **disruption instagib***).

So it had always been my dream, before the Quake 3 engine was GPL'd, to buy the engine when I was older and make a 'Universal Instagib Game' that anyone would be able to play. Of course this was an ambitious dream, I was only 12 at the time and didn't know that the Quake 3 engine cost about $300, 000, and by the time I was old enough (*If I could even afford it*), the engine would be extremely outdated. Another issue was that I didn't even know how to program, I was just deeply in love with the Quake 3 engine, I was used to its ins and outs (*However, I never played Quake 3 itself, haha*). I then started to learn how to program, and after giving up on it a couple times I finally got the hang of it, then one day, miraculously, the [Quake 3 Engine was GPL'd]("http://games.slashdot.org/article.pl?sid=05/08/20/1329236"). Obviously I wasn't aware of iD's tradition to Open Source their engines after they've milked enough money out of it. By the time this happened, I was adequately familiar with programming. I quickly started tinkering with the source (*A real monolith, especially for someone relatively new to programming like me*). I didn't have any game media, so I used Quake 3 Team Arena Demo's game media up to version 0.5.

I released version 0.5 last August, and ever since then I completely re-made the entire game from the ground up. I switched the engine from CleanQ3 to IOQ3 and I started using Open Arena's game media, and created so many modifications. I guess now one can think of versions 0.5 and prior as a prototype of what I had in mind. Instagib is a very simple game type like I explain earlier, so I try to implement features I think (*And usually, know*) will spice things up, from personal experience.

I'm just glad I'm able to fulfill my dream, and now that I look back it seems like a miracle, I would have never imagined me doing what I'm doing now, or would have never imagined having the power of the Quake 3 Engine in my hands. What's more, I've always known the Linux community is short on games, and so I was able to 'shoot two birds down with one bullet', or however the saying goes.

I don't have any servers online yet because the game is barely known, usually when we play online we just direct connect. However, dedicated server binaries are available for those who want them. To play, just create your own server (*Click on **Start Server***) and set up your game. If anyone does start a dedicated server, I suggest they read [this]("http://instagib.jorgepena.be/wiki/Dedicated_Server"). A master server is indeed up, that my kind friend is running on his server (*Which is also running Ubuntu, 6.06 LTS*). So if you start a dedicated server, it should come up in everyone's Internet Games list. I just don't have enough money to rent/buy a server where I could host a game server, I could only afford the web server at the moment.

At the moment I don't have any Debian packages created for the game, but I do have a binary tarball which you could download and extract, and instantly play. I also have a source tarball, and I have information on the Subversion repositories so anyone could get the source to anything if they like, everything is under the GPL. A Windows installer is also available, and the game also runs on Mac but I haven't created binaries for that yet either (*I don't have a Mac, the only access I have to one is my friend's, through SSH*). The game actually should build and run on any of the platforms listed in the Makefile (*Or also seen [here]("http://ioquake3.org/?page=media")*).

If you have any problems whatsoever, please tell them either here or in T.I.P. Forums.

Here is the site: [http://instagib.jorgepena.be/news.php](http://instagib.jorgepena.be/news.php)

I really hope you like the game and I suggest you read the [wiki]("http://instagib.jorgepena.be/wiki"), as a lot of features are turned off by default. I hope you have a good time, and sorry for the long boring post, just thought I'd let you in on the history of the project. I posted here because I thought I would let you guys know of the game I'm working on, which runs on Linux natively. I would appreciate any constructive feedback.

At the moment it's only me working on the project, and just recently, the beginning of this week, my local friend became the game artist (*He did all of the menus and various other things*), which is why I've taken so long to release and work on things because it's just me working on the project. If you are in any way interested in it, please register and post on the forums, we would greatly appreciate it.

Thanks again guys, have a nice day.
