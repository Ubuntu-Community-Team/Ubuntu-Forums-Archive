### Post by evilghost on 2005-09-06
You're not going to be able to exceed the native resolution of that LCD, your max res is going to be 1024x768 without a "virtual desktop" which results in scrolling.  You'll need to drop the resolution of the HPS Simulations John Tiller games to 1024x768 or 800x600.  Also, last I remember there was a flag for Cedega that allowed the application/game to run fullscreen as opposed to windowed.

Hope this helped.
