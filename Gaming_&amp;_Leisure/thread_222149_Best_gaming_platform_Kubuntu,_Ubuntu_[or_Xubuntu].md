### Post by User_Program on 2006-07-24
I've tried out a couple of windows managers and really found no difference.  Though I do have a fairly up to date hardware.  I would say if games feel alittle sluggish when playing them in Gnome,KDE ect try running them without a window manager.  So at login to Gnome KDE ect you will select the game instead of the window manager.

Here is a how-to  [http://www.tldp.org/HOWTO/Linux-Gamers-HOWTO/x276.html](http://www.tldp.org/HOWTO/Linux-Gamers-HOWTO/x276.html)

It's under "4.2 Playing Games In X Without a Window Manager"

Sometimes I run them this way for benchmarking or if all I want to do it play a game.  I have notice only about 5 FPS difference.  But results on a low end system maybe much beter. (like low RAM or older video card)

### Post by ahaslam on 2006-07-24
Hi,

I use Gnome for everything but I've also tried kde & xfce on the same machine. As you found, kde used the most resources but I did find it a little quicker than gnome. Xfce uses fewer resources on my system than the other two and it fees as snappy as kde. The only reason that I stay with gnome is that I feel that it's more stable than xfce, which I have easily broken in the past. 

If spare resources are key, I'd recommend Xubuntu, otherwise plain old Ubuntu. Another suggestion would be to install Ubuntu & add KDE & XFCE via synaptic (not the (?)ubuntu-desktop packages, as these add bloat) and see what suits your needs.

Everyones opinions will differ for many reasons, I hope that you follow my second suggestion and find what's best for you.

All the best,
