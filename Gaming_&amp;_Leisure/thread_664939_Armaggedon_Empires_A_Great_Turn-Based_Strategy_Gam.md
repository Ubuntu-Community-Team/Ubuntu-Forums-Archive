### Post by plinydogg on 2008-01-11
Hey everyone,

As someone who keeps Windows installed on his computer almost solely because he is a gamer, I've been looking for fun  turn-based strategy games with high replay value. The first one I found was the excellent (but expensive) game Dominions 3. It's been mentioned elsewhere on the forums and so I won't elaborate on it.

But I also stumbled across a fantastic Windows & Mac only game that works flawlessly with Wine (I use version 0.9.46) called Armageddon Empires. It isn't mentioned elsewhere on the forums so I figured I'd mention it. It's a fantastic, super-addictive turn-based game set in a post-apocalyptic world. 

You can find more info about it here:

[http://www.crypticcomet.com/games/AE/armageddon_empires.html](http://www.crypticcomet.com/games/AE/armageddon_empires.html)

The only issue I've encountered running it in Ubuntu is that you have to set your top and bottom panels to "autohide" in order to have all of the game buttons available (for some reason, I can't resize the game window as I normally can). 

Has anyone else had any experience with this game?
