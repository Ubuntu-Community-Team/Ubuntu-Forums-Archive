### Post by szczur on 2012-06-07
When you start the game, there should be an wrench icon that leads to Option screen. There you can add teams by clicking plus sign below the "Teams" caption.
On that screen it is possible set everything you need. Remember that on Advanced Settings tab you can change key bindings if you want to.
