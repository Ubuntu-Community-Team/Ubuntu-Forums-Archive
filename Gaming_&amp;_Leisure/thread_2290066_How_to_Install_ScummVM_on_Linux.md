### Post by ajgreeny on 2015-08-09
Have you read the README file in that folder; they will often have full instructions.

However it will certainly be a great deal easier to use the repositories, and if you are running 14.04 you will get a later version than the one you seem to have, ie 1.6 as opposed to 1.2.

You should always look for applications and packages in the repos before you go searching on the web; it is easier and probably will save you from having to sort out a lot of dependencies that make installation difficult when done from source, which is what your folder seems to contain.
