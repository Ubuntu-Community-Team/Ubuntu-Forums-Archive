### Post by dfreer on 2009-09-19
My logitech dual action's (all three of them) work out of the box with ZSNES, mupen64, and pSX. Of course, getting pSX to run at all on Ubuntu has been the main problem as of late.

Are you sure that you are using the right controller model for the game you are attempting to play? Some games will refuse to recognize the dual shock controllers (the SCPH-1200/1150) so you have to use the model without the joysticks (SCPH-1010). Vice versa as well if I recall correctly. Also, make sure you plug the controller in first, then start pSX. Set the Device to your joystick, then map each key. It should work after that.
