### Post by RichardLinx on 2011-12-02
As most of you have probably already heard, the id Tech 4 (Doom 3) engine source code has been released under the GPL and the guys from [ioquake3]("http://ioquake3.org/") have already gotten to work on [iodoom3]("http://www.iodoom3.org/").

A lot of good things came as a result of the Quake 3 source being released, in particular, [ioUrban Terror]("http://www.urbanterror.info/home/"). Probably one of the best FPS games available on Linux.

Urban Terror was originally a mod, but became a stand alone game thanks to the efforts of the ioquake3 developers. This time around we have an engine capable of some remarkable feats. In addition to being the engine behind games like Doom 3 and Prey, it was also behind the *fairly recent* [Brink]("http://en.wikipedia.org/wiki/Brink_(video_game)") video game by Bethesda. The upcoming game Prey 2 will also be using a (modified) version of the id Tech 4. 

If ioquake 3 and Urban Terror are anything to go by, we can probably expect some pretty graphically impressive games on Linux in the coming years. Not to mention all the great Doom 3 mods we can probably expect (or hope) to become stand alone games available on Linux.
