### Post by donkyhotay on 2009-10-03
The mana world is currently the best *open-source* MMORPG out there and probably is what you will want to work on. The other suggestions posted above are either not open source or not actually MMORPG's. Now if you're at all insterested in working on an open source turn-based strategy/artillery game that plays online as a server/client system, I'm currently lead developer and project maintainer of an open source project called [tether](code.google.com/p/tether) that is trying to create an open-source version of [Moonbase Commander](http://en.wikipedia.org/wiki/Moonbase_Commander) and I wouldn't be opposed to some help. Finding an open source project that needs/wants help usually isn't hard, it's just a matter of finding a project that interests *you*. If you can't find anything that catches your interest you could always start your own project. Even if you don't fully develop an entire MMORPG on your own someone else may find the code useful (large parts of my project were lifted from a dead FOSS project) or someone else may take over from you and complete it when you've gained the experience you want.
