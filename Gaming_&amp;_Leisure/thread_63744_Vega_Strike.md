"Enable universe", I presume this means that to install vegastrke, you need to enable the "universe" repository of packages, so that the installer (apt or synaptic) can fetch the packages.  For details in how to enable universe, see in [http://www.ubuntuguide.org/#extrarepositories](http://www.ubuntuguide.org/#extrarepositories)
