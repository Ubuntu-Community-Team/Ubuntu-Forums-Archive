I am a complete n00b when it comes to linux and ubuntu and i am trying more and more to use ubuntu to get away from windows and i found Tremulous. Extremely easy to install and a good game to play on a rubbish machine.

Visit :: [http://tremulous.net/](http://tremulous.net/)

To install do this in terminal:

> 
chmod +x tremulous-1.1.0-installer.x86.run
./tremulous-1.1.0-installer.x86.run


Easy and you have yourself a great FPS for ubuntu!
