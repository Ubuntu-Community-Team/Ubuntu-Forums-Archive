### Post by wildboy211 on 2009-12-15
Im not sure if this is going to work for everyone, but most of the tutorials on getting a xbox 360 controller to work on ubuntu were complicated and i didnt understand them (just started using Linux 2 weeks ago). So i tried it my way, and it worked for me.

Ive tested this with Project64 with Wine, and MarioKart64 seems to work great. Ill probably be updating this in a few days as i learn more.

Well, this is how i did it:

You will need:
Xbox 360 Controller
Ubuntu 9.10
Wine
[Xpadder]("http://fileforum.betanews.com/detail/Xpadder/1148671810/1")

1. Install Wine (go to Ubuntu Software Center and search for 'Wine')
2. After Wine is installed, download and install Xpadder in a directory easy to get to (ie Desktop) using Wine
3. Plug Xbox 360 controller into USB port and you should be ready to go!

*Note: You must configure the Xbox 360 controller in Xpadder for it to work, to do this, follow this tutorial (takes roughly 5-10 minutes):
[http://xpadder.com/phpBB3.0.5/viewtopic.php?f=4&t=19](http://xpadder.com/phpBB3.0.5/viewtopic.php?f=4&t=19)

My system for comparison purposes:
Ubuntu 9.10
Intel Pentium 4 2.6Ghz
2GB PC-2700 Memory
500GB External Hard Drive with Ubuntu on 6GB Partition
64MB Onboard Video

If anyone has any problems let me know and ill try to help as best as i can. -Marc :D
