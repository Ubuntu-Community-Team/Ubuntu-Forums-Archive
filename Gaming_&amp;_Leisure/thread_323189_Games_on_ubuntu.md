### Post by Naegling23 on 2006-12-22
Unreal tournament 2004, quake 4, doom3, and soon to be ut2007, and quake wars are all native linux games.  This means they run without the need for special software like cedega or wine.  Any game that you need cedega or wine to run, might be more difficult to set up, or require some tweaking to get it running.  The linux native games run great with minimal fuss (although, it is harder to install them than windows).  Linux native games are just that, games for linux.  If you need wine or cedega, then it is a windows game "tricked" to play on linux. (it works, but its not designed to)
