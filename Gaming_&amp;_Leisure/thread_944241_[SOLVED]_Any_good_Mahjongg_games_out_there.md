### Post by jacksaff on 2008-10-11
You don't need all the kde games to get the mahjong ones.
Kmahjong and shisen-sho are the ones you want. The kde4 versions are MUCH prettier than the kde3 ones so get them.
They will pull in some kde4 dependencies but this is nothing to be feared on unix. The programs that don't pull dependencies simply have all the necessary code compiled in meaning every binary is bigger, every app has duplicated code and you end up with more bloat than if you installed a few libs.
There is no reason to stick with just gtk apps on gnome (or qt ones on kde) on any system modern enough to run these apps decently in the firstplace.
