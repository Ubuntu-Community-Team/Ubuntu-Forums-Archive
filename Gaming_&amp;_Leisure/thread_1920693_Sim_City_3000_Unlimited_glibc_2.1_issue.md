### Post by untanne on 2012-02-04
Hi, I run Ubuntu 11.10, Linux kernel 3.0.0-15, 64bit. I did the following to install Sim City 3000.

1. create a new VM with VBox
2. install Ubuntu 6.06.1
3. download installer (beta) from [http://www.liflg.org](http://www.liflg.org)
4. insert SC3u CD and make it available to the VM
5. execute installer as root
6. following the instructions within the GUI

Enjoy playing!
