### Post by seethru on 2005-10-15
Category: Emulator
Description: Atari ST emulator

Changelog:

 [LIST]
[*]Hatari now supports STE hardware emulation: STE palette, STE shifter (horizontal fine scrolling, split screen effects), DMA sound and STE joypads.
[*] Hatari can now emulate up to 14 MiB ST RAM instead of only 4 MiB.
[*] Support for ST parallel port joysticks.
[*] Improved GEMDOS HD emulation (added the Fattrib() GEMDOS call).
[*] Adding and removing a GEMDOS or ACSI hard disk should now work correctly.
[*] The screen conversion functions have been re-factored.
[*] The manual features now screenshots of the options dialogs.
[/LIST]
