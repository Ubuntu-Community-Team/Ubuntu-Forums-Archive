### Post by disturbedite on 2007-10-18
using zdoom/skulltag is super simple on linux.  i'd recommend skulltag in particular.

1.  grab the ubuntu base version from the skulltag download page.
2.  extract it anywhere you want.  put the wad(s) in ~$HOME/.skulltag or the folder you extracted st to.
3.  run st.
