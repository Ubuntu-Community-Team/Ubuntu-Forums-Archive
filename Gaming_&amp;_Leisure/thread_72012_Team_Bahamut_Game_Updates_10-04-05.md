Team Bahamut Game Update

**Schredderchess**

The world champion chess program [Schredder](http://www.schredderchess.com) has been released in a version for Mac and Linux. This is important because it is the first time a major software player in the chess world has released a version of its chess engine for Linux and the general public. There is a [demo version](http://www.shredderchess.com/download.php?file=shredderclassic.tar.gz) of the linux client available for download. It works on Linux x86 and is a 4 MB download.

**Battle for Wesnoth 1.0**

The Battle for Wesnoth team is proud to release version 1.0 of "The Battle for Wesnoth", a GPL'd turn-based strategy game with a fantasy theme and roleplaying elements. The game is available for download now for Windows, Mac OS X, and various GNU/Linux distributions.

The source code is available from [sourceforge.](http://prdownloads.sourceforge.net/wesnoth/wesnoth-1.0.tar.gz?download) You can also download it from synaptic or apt-get, although you may not get the latest version that way.
