I updated my how-to for setting up a modern arcade cabinet using SDLmame and the Wah!Cade front end to xubuntu 9.10. The old one was for 9.04 and was a bit dated. The new guide has an updated setup script and updated instructions.

It's pretty easy to follow and there is even a custom shell script I wrote to install the software and set up Ubuntu for arcade cabinet use.

Have a read and discuss here. Any suggestions are welcomed.

Link: [ABM.net : Setting Up a Modern Arcade Cabinet]("http://www.alphaboxmedia.net/arcade/")
