### Post by oldrocker99 on 2013-05-12
Desura, which predates Steam as a distributor of downloadable Linux games, has a number of free Linux games available. Just about every free game offered there is also in the Ubuntu repositories, which is the best place to start looking. The Ubuntu Software Center is an okay place to start, but I personally prefer Synaptic, as it's considerably faster at downloading and installation than the Software Center. If you make a list of software you want to install on the Software Center, then find those programs in Synaptic, the process will, IMHO, be faster.

And, BTW, there are LOTS of great free games for Linux, and LOTS of great commercial games, most of which are certainly worth paying for. Google "Humble Bundle" and see what's available for whatever you want to pay. It's a truism that Linux users have paid more per user for the many Humble Bundles than Windows or Mac users, proving that Linux users are HUNGRY for games, and, in the last 2-3 years, game developers have noticed that.

2013 is The Year Of Linux Gaming; I called it that in January, and I've seen nothing to dissuade me from stating that.
