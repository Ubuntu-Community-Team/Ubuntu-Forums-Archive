### Post by mrjerryk on 2010-05-19
The best way I have found is to run a media server on your Ubuntu PC.  I run fuppes and I am able to stream all of my media to my PS3.  


This is an easy guide to install it on 9.10 and 10.4.  

[http://ubuntuforums.org/showthread.php?t=1310511]("http://ubuntuforums.org/showthread.php?t=1310511")

The main thing to remember is to make sure your fuppes.cfg file is correct.  

You can also make fuppes start when you turn on your pc.
[http://fuppes.ulrich-voelkel.de/wiki/index.php?title=Startup_with_Init.d]("http://fuppes.ulrich-voelkel.de/wiki/index.php?title=Startup_with_Init.d")


Enjoy!

### Post by Wufei74 on 2010-05-19
I actually just set this up. Found a cool, easy tutorial for mediatomb via Google.

[http://linuxowns.wordpress.com/2008/04/28/stream-media-from-ubuntu-to-your-ps3/](http://linuxowns.wordpress.com/2008/04/28/stream-media-from-ubuntu-to-your-ps3/)

It was actually super easy, and I didn't have any trouble setting it up(complete newbie at linux, advanced Windows user).

My only gripe with it was when you click the "+" to add a song or folder, there's no signal to show you that its added. Normally that "+" would dissapear or become greyed out, but in here it doesn't. I added the same CD like 15 times when I first set it up :P.
