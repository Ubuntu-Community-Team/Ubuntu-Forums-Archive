### Post by mazelado on 2006-01-02
Open a console and go to wherever you downloaded the cedega_timedemo_installer file. Enter the following command:

chmod +x cedega_timedemo_installer

You should then be able to run the installer with this command:

sudo ./cedega_timedemo_installer

That should run the installer and you should be all set.
