### Post by Perfect Storm on 2008-09-26
[Scorched3D](http://gaming.gwos.org/doku.php/games:alphabetical:s:scorched_3d) - tank/worms game.
[Hedgewars](http://gaming.gwos.org/doku.php/games:alphabetical:h:hedgewars) - tank/worms game.

[Nexuiz](http://gaming.gwos.org/doku.php/games:alphabetical:n:nexuiz) - FPS game. This game can be scaled to work on older/slower hardware.
[Transfusion](http://gaming.gwos.org/doku.php/games:alphabetical:t:transfusion) - FPS horror game/Blood remake.
[Warsow](http://gaming.gwos.org/doku.php/games:alphabetical:w:warsow) - FPS, modified Quake 2 engine, so it's possible that it will run on older/slower hardware.

[Miniracer](http://gaming.gwos.org/doku.php/games:alphabetical:m:miniracer) - racergame included networkplay.
[Supertuxkart](http://gaming.gwos.org/doku.php/games:alphabetical:s:supertuxkart) - racer/gokart with up to 4 players.
