### Post by acoustibop on 2007-05-23
When you say "the Joystick calibrator program," which one do you mean, BigSilly?  There are a couple you can install from Synaptic, jscalibrator and joystick.  jscalibrator has a gui and is thus easy to use, unlike joystick, of which the two components have to be run from terminal.  However, if jscalibrator is what you installed, I found when I used it that it gave me problems with directional controls not working in some games/emulators.  When I removed it and installed joystick instead, the problems vanished.
