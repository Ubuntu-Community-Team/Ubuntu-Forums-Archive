### Post by Melcar on 2008-10-02
The driver you're using is "ancient".  The CCC in those earlier versions had a lot of bugs (resetting settings was one of them).  Using the latest version (you would have to install it yourself) should fix the issue.
Options like AA and AF can't be forced anywhere else other than in the CCC since fglrx uses its own configuration file (only refers to xorg.conf for basic stuff device initialization and resolution).
