### Post by willbe1kenobi on 2009-02-02
Yes you can. Just install a program called 'WINE' using Synaptic, and you should be on your way. 

Just a quick note - Wine is not perfect, as there are many windows games and programs that cannot be run using wine. 

I suggest taking a look at [http://appdb.winehq.org/](http://appdb.winehq.org/) to see what games/programs can run and what bugs you are likely to encounter.

But if there is a Linux port of the windows game you want to run in Linux I would rather play the Linux port
