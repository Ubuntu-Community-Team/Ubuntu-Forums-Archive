### Post by twigboy2000 on 2008-09-15
I've been using AutoHotKey under wine to duplicate the keystrokes between the windows.

[http://www.dual-boxing.com/forums/index.php?page=Thread&threadID=8720](http://www.dual-boxing.com/forums/index.php?page=Thread&threadID=8720)

You create the ahk script then wine the program to convert the script to an exe.  Once you have both of your WoW windows logged in to your characters, wine the exe that you created.

Enjoy!
