### Post by wererabbit on 2006-11-03
In Cedega, you just put your cd in the drive, it sees it - you pick the Title from a list of recognized titles or make your own if Cedega doesn't have it listed.  Then it runs in a window on your desktop thru Cedega.  

For either Cedega or Wine, check the compatibility lists before spending money on a game.  I've used NWN and UT2004 natively and they run extremely well.  Steam doesn't seem to work very well in Cedega or Wine if you download applications within it.  Instead, install Steam & Halflife/CS:S etc from CDs.  If you don't do this, chances are that the games won't run within either.  


*edit* I forgot to add this: 

Cedega creates a fake C:\ drive for each game you install - that way the game thinks it's in Windows and it sees the correct dlls in the right places.  All of this is fairly transparent to you unless you look in your Home folder and unhide these folders.  I've been able to add mods to my KOTOR by just navigating into the KOTOR C:\ drive that Cedega creates and then dropping the mods where they would go on a Windows box.
*/end edit*

Nvidia drivers for Linux have AA/AS support, but if you run XGL or Compiz to spruce up your desktop, you won't be able to enable AA/AS for your games at least from what I've encountered. 

I run Dapper right now - it's pretty darn stable and runs the games well.  I'm waiting for another couple weeks before moving to Edgy in order for them to get more kinks out.
