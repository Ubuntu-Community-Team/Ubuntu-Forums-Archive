### Post by cogadh on 2008-06-14
```
./configure --prefix=<prefix directory>
```
Just replace <prefix directory> with the path you want the app installed in.

EDIT - Ack, I think I misread what you were asking (I shouldn't post when I'm this tired). If the game is looking for libpng somewhere and you don't have it there, just create a symlink from where the game wants to find it to where it actually is:
```
sudo ln -s /path/where/libpng/is /path/the/game/wants
```


I'm not 100% positive about this, but you may be able to just use the Debian repository to get and install a precompiled package:
[http://allacrost.sourceforge.net/wiki/index.php/Installation_Instructions#Debian](http://allacrost.sourceforge.net/wiki/index.php/Installation_Instructions#Debian)
