### Post by anduweb on 2011-04-07
Well...there is a webbased configuration tool as the WGA works as an an usual AP.
By default it can be accesed on 10.10.10.254 (I have no idea why they used the (usual) broadcast ip).
But before you can access the RaLink interface you have to manually configure your Network card to an ip compatible with what the WGA has.. something like IP: 10.10.10.10 and subnet mask 255.255.255.0.

Since this seems to be built on a IE working base...you might need an IE to properly get the settings saved...
