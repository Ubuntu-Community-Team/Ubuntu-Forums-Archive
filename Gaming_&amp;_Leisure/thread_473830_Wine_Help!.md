When installing a program in WINE, where is the C: directory?  Because lets say you install a program in the default directory, where does it exists? C:\Program Files\"Program"? Because i need to find where a program I installed is...

---

### Post by hikaricore on 2007-06-14
The location is:

*~/.wine/drive_c/*

aka
[I]
/home/yourusername/.wine/drive_c/[/I]

---

### Post by cogadh on 2007-06-14
However, if you are looking for that directory in your file browser, you probably won't see it, since it is a hidden directory. Do a CTRL-H to temporarily unhide directories.
