[FONT=Arial, sans-serif][SIZE=2]**Djl is an open-source (GPL licensed) game manager written in Python 2.5 for the GNU/Linux **[/SIZE][/FONT]
[FONT=Arial, sans-serif][SIZE=2]**Operating Systems. It is inspired by Valve's Steam software for Windows.**[/SIZE][/FONT]
 
[FONT=Arial, sans-serif][SIZE=2]Djl makes it possible (via a repository) to download, install and remove a reasonable number[/SIZE][/FONT]
[FONT=Arial, sans-serif][SIZE=2] of games placed into a distrobution independent subdirectory (but without dealing with any [/SIZE][/FONT]
[FONT=Arial, sans-serif][SIZE=2]dependencies). It can also execute ,desktop shortcuts located in another directory....[/SIZE][/FONT]
