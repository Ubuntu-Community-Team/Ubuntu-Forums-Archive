### Post by Harleen on 2005-12-10
Actually there is a Linux alternative to the DEAMON tools. It's calles "mount". ;)

If you have a regular iso file and want it be mounted to /mnt, you should be able to do that with this command (without guarantee and untested):

sudo mount image.iso /mnt -o loop -t iso9660
