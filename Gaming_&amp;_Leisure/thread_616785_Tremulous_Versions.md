### Post by hikaricore on 2007-11-18
Those servers are just running a newer version of Tremulous than you have installed.

Software in the repos is only updated for serious isses such as bug/security fixes.
You are easily able to download a newer version and install it manually from the Tremulous website if you need a newer release.

---

### Post by por100pre1 on 2007-11-18
Those messages actually ask you to replace the official client with a backported client. If you are not going to be admin of one of those servers giving you the warning, just ignore the message.
