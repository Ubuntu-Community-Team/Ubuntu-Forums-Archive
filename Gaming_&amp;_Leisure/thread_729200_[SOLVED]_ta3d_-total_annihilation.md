### Post by zuzuf on 2008-03-30
Hi,

you can find the required resource files here:
[http://d-gun.com/files/](http://d-gun.com/files/)

(beware that these files aren't free, you need to own a copy of the game to use them. Currently TA3D use those files because we still don't have a complete set of free resources ... but that should come with time :) )

these are the images of Total Annihilation CDs, you need at least the first CD. To install the required files, start the install-sh script and follow instructions on screen.

If it doesn't work with these files, then you should post your ~/.ta3d/ta3d.log file on TA3D's official forums: [http://ta3d.darkstars.co.uk/forums/](http://ta3d.darkstars.co.uk/forums/)
