### Post by Vadi on 2008-06-14
If anyone's a gamer in here and frustrated when you can't alt+tab (or do whatever else) out a window: 

Apparently there's a shortcut you can set (preferences - keyboard shortcuts, "toggle fullscreen mode") that'll maximize a window and remove the borders. So starting a game in widowed mode + using the shortcut does the trick of getting normal fullscreen that you can get out of.

(not sure if this was mentioned before, but it works nicely here).
