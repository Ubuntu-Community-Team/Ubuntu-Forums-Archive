### Post by josh.bronson on 2007-04-01
I had some trouble with gnome-mud scrolling way past the end of the mud output. If you have a similar problem, you can try tt++ or Tiny Fugue. Remember to use the #split command if you use tt++, and run them in xterm, not gnome-session, as xterm is much faster.

You may also want to take the Bartle Test to find out what type of MUD you'd like:

[http://www.guildcafe.com/zQuest.php](http://www.guildcafe.com/zQuest.php)

happy mudding. As Stephen Colbert says, "Remember to shower." :)
