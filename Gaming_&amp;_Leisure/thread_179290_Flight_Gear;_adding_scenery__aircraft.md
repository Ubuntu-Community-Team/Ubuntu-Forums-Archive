Hello, all.

I installed Flight Gear, and wanted to know how to add extra scenery and aircraft.

I read the man page, and found references to $FG_ROOT/Scenery but I couldn't find the value of this variable.

A bit of trial and error, running fgfs with the log level turned up to debug, I managed to find that it is:

/usr/share/games/FlightGear

Hope this helps somebody else.

### Post by wrtrdood on 2006-05-19
Get the manual from the site.  It's a PDF and covers everything you want to know.  What I'll give you is just off the top of my head.

To get additional scenery, you have to create some directories and extract the scenery files to these directories.  I forget the actual name - check the manual.  Then you'll need to provide that path to the command when starting  fgfs.
