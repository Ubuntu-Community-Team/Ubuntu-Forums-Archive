### Post by Rhemat on 2006-09-27
I was able to resolve the problem.  For those of you who might have the same model of router as mine, port forwarding is under Application Level Gateway.  It'll have a list of preset games.  You could use Elite Force, which uses the same port numbers, or go to User and create a Quake3 profile.

-Rhemat
