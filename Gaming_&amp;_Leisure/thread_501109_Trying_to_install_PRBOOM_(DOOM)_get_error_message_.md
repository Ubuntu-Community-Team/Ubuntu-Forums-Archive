### Post by yabbadabbadont on 2007-07-14
PRBOOM is already in the repositories.  Install it using Synaptic.  Settings->Administration->Synaptic Package Manager.  Once in synaptic, Settings->Repositories  Make usre that the universe and multiverse repositories are enabled.  Click OK to save your changes, if any.  Hit the Reload button.  Now search for prboom and select it for install.
