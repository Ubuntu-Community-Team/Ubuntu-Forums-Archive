### Post by Rustybolts on 2010-08-01
I run from an aspire 5332 which has the gma4500m chip set, I can run Doom3, Penumbra series, Unreal Tournament 2004,Serious Sam 1+2, Darwinia, H-Craft.

There are not many(not any?) native Linux games it wont run try the demos out before any purchases first though.
