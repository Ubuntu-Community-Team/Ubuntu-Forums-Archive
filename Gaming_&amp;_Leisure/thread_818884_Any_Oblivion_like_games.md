### Post by sirdilznik on 2008-06-05
Are you running with:

"OffscreenRenderingMode"="fbo" ?

Because running the game with either pbuffer or backbuffer always causes the game to crash for me either immediately or within a few seconds of starting/loading.  Fbo on the other hand, works fine.  The game still crashes randomly sometimes but not all that often (save often)

rc3 works just as well for me as rc1 did.  I seriously doubt rc1 would be "better" than rc3 since nothing new has been added.  Wine has been in code freeze.  Bugfixes are the only difference between rc1 and rc3.
