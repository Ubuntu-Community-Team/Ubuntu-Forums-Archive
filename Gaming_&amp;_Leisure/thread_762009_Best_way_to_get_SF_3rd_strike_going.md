### Post by BigSilly on 2008-04-22
The files you are missing will likely be the .chd which you'll have in the wrong place, and yes, keep your game file zipped up. For the .chd, create a folder within your roms folder called sfiii3 which you'll notice is the same name as your zipped file. Stick the .chd in there and you should be ready to go. If you are trying to play Third Strike, take note that you'll be better off trying to use the NOCD version. SF3 has to create it's own nvram file on the first boot, which can take around 50 minutes to complete otherwise. The NOCD version will work instantly.

Let us know how you get on. :)
