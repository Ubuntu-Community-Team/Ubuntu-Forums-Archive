### Post by bstrawn689 on 2018-10-23
I recently decided to experiment with Proton. This is an option under Steam Play assessed through Steams settings menu. This handy little feature allows you to play Windows games on Linux, or at least attempt to, so I decided to try it out on a visual novel called Aselia the Eternal. And it failed.... Not wanting to give up, I looked through the Steam folder to find the executable file and tried using Wine and while it did not work at first after using the command, "winetricks mfc42" in the terminal to get a dll file I needed the game worked fine! Yay! I figured this out as before the game was released for steam it was playable through other means and was listed in winehq.com. 

Now for the question, is there any way to add mfc42 dll file to Proton? Or is this a lost cause? Thanks for the help!
