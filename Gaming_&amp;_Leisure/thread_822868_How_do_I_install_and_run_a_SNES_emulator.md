Hello, Linux has tons of emulators just like Windows. Just go to "Add or Remove Programs" or "Synaptic Package Manager" and search and install for Zsnes - it's my personal favorite super nintendo emulator. You can also install it by using a terminal and typing ```
sudo apt-get install zsnes
```

There are other snes emulators like snes9x  and bsnes but I highly recommend zsnes!
