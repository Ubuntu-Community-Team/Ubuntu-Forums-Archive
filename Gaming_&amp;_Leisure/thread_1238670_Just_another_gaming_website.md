### Post by GWBouge on 2009-08-12
Since I've started using Ubuntu I've really gotten into the open source/low cost/low budget gaming scene.  If anyone's spent any time hunting down such games, it's no secret that, while they may not be as 'pretty' as the big proprietary DirectX titles on average, they more than make up for it with gameplay.  The only problems I've found is that sometimes they're pretty hard to find, and (even some great ones) die off since they don't get much support.  UbuntuForums is a great place to start looking, but let's face it ... not every other distro/Windows/Mac user looks at (or even knows about) these forums.  There are some great lists out there, but they're typically pretty minimal, and they're really spread out over the net so you have to know what to google in order to find them.

So my proposal is simply an indie/garage game equivalent to websites along the lines of IGN or GameSpot, although (in a linux fashion) not nearly as bloated, clustered, or filled with stuff that just takes up space and draws attention from what you're actually looking for.  I'd like it to be based around the un-advertised titles from little known developers, and cover all platforms.  Game reviews, developer interviews, indie news, etc ... show your average Windows gamer there's more out there than he sees on TV and big-ticket websites, and that it's probably more in depth with more gameplay and features.  Now, I'm sure there's quite a few of these sites out there, but I honestly haven't come across any in the fashion I'm trying to pursue, so one more certainly can't hurt the cause, lol.

I have a *mostly* functional version of what I'm trying to accomplish up and running, although I'm looking to see if I can get any support/help before I buy the domain name, so it's hosted here:

<dead project;  addr removed>

I'm no writer, but I put a basic review up there just to test some things and do a little layout work.  If any one would like to help out something like this, let me know!  I'm obviously no professional when it comes to the graphic work, so if you can (and are willing to) do something on that end, great!  What I really need at this point is some content to get started, reviews from all areas.  From Nintendo to high-end 3D, old to new, RPG to FPS, seamless 3D adventures to side-scrolling puzzle games ... everything.  I'm even looking for some 'Press/News' type stuff up there to let readers in on what's going on in the low-budget scene, so if you have the means and desire to do something in that aspect, awesome.  There's a LOT more work that has been done on the back end of the website for any writers, although until I get some content up there there's not a whole lot I can sort out on the main pages and features.

Comments?  Ideas?  Suggestions?  Should I just scrap the idea?
