### Post by otoris on 2010-10-16
I would ask here -> [http://forums.dolphin-emu.com/](http://forums.dolphin-emu.com/)

I know in Windows 7 dolphin uses a DirectX plugin for graphics rendering whereas Ubuntu uses a OpenGL plugin for rendering. I haven't been able to enable anti-aliasing or any special graphics feature in Ubuntu with the OpenGL plugin in dolphin, so that could be problematic.
