### Post by cogadh on 2007-06-19
2 - 4 years of college to start:)

Game programming is not something that a single person can really do anymore. Games are actually made by teams of people. You have engine programmers, AI programmers, graphic artists, sound engineers, etc. all of whom each build a piece of a game. 

If you are really interested in learning about game programming, get yourself a book on general programming basics to start with. This may sound silly, but I have always found the Sams "Teach Yourself... in 24 Hours" books a good starting point. They are not as dumbed down as a "For Dummies" type book and they give you a good foundation to really build on.

### Post by Cappy on 2007-06-19
There are open source games, open source developer kits, and closed source mod tools for pre-existing games. You'll be wanting to modify something that has already been created. If you don't know ANY programming you're best bet is to create a mod for a pre-existing game. Lots of games supports mods like Neverwinter Nights, Oblivion, Morrowind, FPS games, Graal online, etc.

It's pretty much impossible to write a new game unless the game is 2D, a very simple 3D game with uncustomizable character models, or it is text based. And even those take a great deal of effort, skill, and programming knowledge.

### Post by TheTank on 2007-06-20
Let me be terribly honest and without wanting to hurt your feelings but forget it!
Games like morrowind and like cost millions and have large teams that develop for multiple years.
Often the games are based on each other. NWN is an ancestor of BG2 is an ancestor of BG1

If you just want to play around, great but anything else will (sadly) fail.

What I would suggest:
decide what you want to do: 2d/3d artist or coder
If you choose coder, first learn how to code. Game coders are among the best and you will need to become very good.
Once you have the coding down move on to creating games. Create simple ones and then make them more complex.
But make sure your code and the game is solid and well documented.

Also don't try to be a 1337 coder. An home made OpenGL engine will not get you a job.
A good and solid game will.

I know some guys that got into the industry buy making tools for NWN.

Another good thing might be to join a team of developers on an existing project.
Check out the projects on sourceforge and see if they need help.

I hope I do not sound to harsh. I would support anyone coming into the field but:
Shoot for the stars, but take it one step at a time.
