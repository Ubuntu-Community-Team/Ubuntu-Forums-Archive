> From Wikipedia, the free encyclopedia


Open source games are computer games assembled out of open-source software and open content.

All these titles are free software that are under a license such as the GNU General Public License or other similar licenses. Because the source code is available and modifications can be freely (re)distributed, most of these titles will run on many platforms.

For freely available games without a public license, see List of freeware games.
