### Post by HH60Gunner on 2010-03-08
Well I've been playing HoN Beta for about a week now.  The game is great and the linux client for it runs very smooth.  So far I think this is the best linux game to date that I've seen.  With the only competetion quality wise being warzone 2100.  The only downside of this game is that it is filled with a ton of elitist jerks who have all been masters of the game this one was based off of (DOTA).  My first few games and even a bit now I run into people who seem to have nothing better to do then to flame noobs.  Even though every game I join is named "noobs only", and ther server type set to "noobs only".  It seems to me that the idiots don't understand that when they join noobs only games they are in fact going to have noobs on thier team.  Instead of growing up and either mentoring newbies, or making thier way to higher level games they just choose to stay in the noob servers and flame noobs for being noobs in a noobs game.  Sometimes it's so irritating that I almost don't want to play, yet the fun factor of the game keeps bringing me back.  Hopefully when this game is released they'll make the server types reflect who can join.  (ie: some sort of crieteria for a noobs only game so pros can't come in and ruin the games.)  This game has a bit of a steep learning slope and it takes awhile to start learning what all your opponents abilites are.  So far I'm addicted and the only downside is the community.
