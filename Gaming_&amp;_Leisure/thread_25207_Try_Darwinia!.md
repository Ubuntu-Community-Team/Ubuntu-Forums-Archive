### Post by FoolsGold on 2007-04-08

Go into the lib folder in your Darwinia folder, rename "libgcc_s.so.1" to something like "libgcc_s.so.1.bak" (forcing the game to use your system libgcc instead of the supplied version). Run the game and all should be well.
