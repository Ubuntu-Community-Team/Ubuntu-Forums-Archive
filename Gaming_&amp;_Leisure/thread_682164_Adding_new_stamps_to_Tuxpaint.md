### Post by grantbuntu on 2008-07-19
I know how to get all the default stamps using synaptic.  Search for tuxpaint and select tuxpaint-stamps-default.  Install it (about 33 Mbs).  Enjoy lots and lots of animal stamps, clothing etc etc.  I bet something like 

sudo apt-get install tuxpaint-stamps-default

works as well.
