### Post by eivind on 2005-10-11
You didn't install it, you just decompressed it. What you have to do to play the game, is to compile the source. **Very general** instructions folllow - beware, they might not work! Open a terminal and do this:

1. sudo apt-get install build-essential
2. cd into the source directory (that's the one with all the .c and .h files)
3. sudo ./configure
4. sudo make
5. sudo make install

I haven't actually tried the quake source, so this is in no way bullet-proof, but this is how source files are normally made into something useful ;-)

Cheers,

Eivind
