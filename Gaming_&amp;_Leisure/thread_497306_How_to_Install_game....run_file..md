### Post by shad0w_walker on 2007-07-10
Right click on the file and goto

Open with.. -> Open with other application

You should get a popup box asking what program to use. At the bottom of the box you can enter a custom command (If theres no text area click the little arrow at the side)

Enter this command and click open.

```
sh
```

The installer window should popup after a second or two and your ready to go.
