### Post by tjwoosta on 2008-08-25

### Post by Ozor Mox on 2008-08-25
If you download the game from the website instead of using the deb, then you just need to untar it and double click the executable file. If it doesn't work, right click the file and click properties and make sure the permissions have it set to be executable. If it still doesn't work, open a terminal, cd to the directory, and type ./ActionCube (or whatever the executable is called) and press enter.

This is all from memory, anyone else correct me if I'm wrong!
