### Post by JDShu on 2013-02-24
You can try using wine, available in the repositories. Check out [www.winehq.org]("http://ubuntuforums.org/www.winehq.org"). But you'll have to understand that games written for Windows are unlikely to run anywhere near as well on Linux, if at all.
