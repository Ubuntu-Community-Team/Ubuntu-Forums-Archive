### Post by MasterJS on 2007-12-31
Alright, i figured out my problem with much luck. It turns out, after much searching on the world wide web, that XGL is not very friendly with other 3D applications as it is an application itself on top of GNOME, taking all of the direct rendering and allowing an indirect rendering of Compiz/Beryl. Well, an easy fix for this and anyone that uses XGL, is to create another user. In that user, you must create the file ~/.config/xserver-xgl/disable. Nothing has to be written in it, the file just has to be present. By creating this other user just for games, you can still have your normal user for all your other activities and keep all the special desktop effects.
