### Post by Anduu on 2006-09-16
Gaming can be quite tricky under Linux still.

Your first step should be to see if your game has a native Linux version.If your game has Linux support it is usually just a matter of going to the games website and downloading a special Linux installer.

Short of native support you would have to install/learn Cedega or Wine which can be a career in itself :p.

Your best bet is to learn by doing.Start with one game you have and go from there.Pick one that has a how-to in the gaming section and give it a shot.

### Post by MrBlond83 on 2006-10-09
Cedega is the way to go as far as gaming is concerned. WINE can be used to run several games well, but Cedega has far wider support, and runs most games better. Not to mention that WINE really does need a lot of configuring, while Cedega is more simple in that manner.

The down side being that Cedega costs money. So it really just depends on how much you hate having windows on your system. I have a dual boot system with Windows mainly for gaming, I'd rather reboot when I want to play a game than pay for Cedega (I play on Linux when I can - ie Unreal Tournament 2004, Battle for Wesnoth, Enemy Territory / True Combat Elite, Neverwinter Nights). Plus I heard that Vista has a free license for a year or something, no?
So I'd say if you're not short of HD space and games are an important part of your PC activity, install Windows on a second partition. That way you can run all of your favorite games (even Cedega doesn't run certain games), as soon as they come out, and at the best quality. And of course if a game has a native linux version run it on linux.
