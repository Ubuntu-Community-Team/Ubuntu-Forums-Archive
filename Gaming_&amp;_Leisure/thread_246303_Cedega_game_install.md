### Post by frodon on 2006-08-29
Hi ESPOiG,

If i understand well what you want to do is to install steam then install counter strike through steam, if it's the case i can assist you because it is something i did several times.

Here is what i did :
1- download steam from the steam website
2- open cedega, select "mnt" in the left cedega panel and create a shortcut to the steem .exe install file, then i click on the shortcut i created in cedega and press the play button.
3- Follow the steam install process
4- once done, if i remember well i deleted the shortcut i created to the install .exe file then create a new one to the STEAM.exe file.
5- Click on the created STEAM.exe shortcut and use steam to install counter strike.
