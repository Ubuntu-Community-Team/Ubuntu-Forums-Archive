### Post by AmirGooran on 2012-04-04
1. Extract the package into a folder
2. Open a terminal, and go into game folder with cd command.
3. ./bolzplatz2006.sh

Note: you must have java package in order to run the game.
Note: if there is an error while running the game,check log.txt file.if you see "couldn't load openal library" message, then you have to download libopenal1, liblwjgl-java and liblwjgl-java-jni packages from Ubuntu repository and copy/paste files "libopenal.so", "liblwgjl.so", "lwjgl.jar", "lwjgl_util.jar" from these packages into gamefolder/lib/lwjgl
I installed it in Kubuntu 11.10 without any problem.
Good luck!
