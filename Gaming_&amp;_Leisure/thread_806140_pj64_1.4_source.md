### Post by acoustibop on 2008-05-24
I don't think you can compile Project64 in Linux, bhuthogg - probably because it needs DirectX.

Try [Mupen64Plus]("http://code.google.com/p/mupen64plus/"). There is a binary distribution, but you'll need to run the install.sh included in the package; since Mupen64Plus is a plugin-based emulator, you'll also need to configure the plugins. Runs nicely on quite mid-range machines.
