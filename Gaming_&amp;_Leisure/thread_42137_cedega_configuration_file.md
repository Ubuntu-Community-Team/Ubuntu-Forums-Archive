### Post by equilibrium on 2005-06-16
~/.transgaming/config

you can setup cedega to run a windowed mode 'desktop' by editing the config file.

in the [x11drv] section

; Use a desktop window of the given size
;"Desktop" = "800x600"

just uncomment or add your own resolution.
