### Post by khelben1979 on 2009-05-08
First check out [this video]("http://tr.youtube.com/watch?v=gUf8LvcQ5Bg") from YouTube which demonstrates that the game is working together with Wine.

On WineHQ [they say]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=1638") that you need to set your Linux desktop to 1600x1200 before starting the game. Have you tried this?

I personally didn't think that this game works with Wine, but now I see that it does. Some games need to be run from a terminal and then invoking the -opengl parameter. This might be the case here also.
