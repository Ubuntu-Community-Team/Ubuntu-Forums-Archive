### Post by doas777 on 2009-07-05
I haven't had a lot of luck with virtualbox for games, so I use [vmware player]("https://help.ubuntu.com/community/VMware/Player"). I have found that the vmware player that ships with [vmware Workstation]("https://help.ubuntu.com/community/VMware/Workstation") works better however. you can just use the trial workstation to create your image, and then run it in player perpetually, even though the workstation trial is elapsed. you can create a vmware image for anygiven os that you have an install disk at [http://www.easyvmx.com/](http://www.easyvmx.com/) .

I also find that to play games, i first have to disable compiz, by running 
```
metacity --replace
``` before starting the game. directx fullscreen, and compiz window decoration sometimes fight eachother. 

i've also found that some games work best if I start with the vm not in fullscreen mode, and then switching to fullscreen after the game starts to load.  weird, wot?

good luck
