### Post by DoktorSeven on 2007-05-24
If the question is which is better, wine or PS2 emulation, both are not perfect at the moment.  Go check compatibilty lists at pcsx2.net and winehq.com and see which one best runs the game you're looking at.  Neither is going to be a perfect solution, and PS2 emulation is generally going to require a *much* beefier system for all of its games than Wine requires to play a certain game (PS2 emulation needs a very fast proc and a great videocard; Wine generally needs whatever Windows needs for the same game).
