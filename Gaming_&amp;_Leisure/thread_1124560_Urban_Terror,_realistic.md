### Post by Yvan300 on 2009-04-16
To install true elite all you have to do is download the run file from [http://www.liflg.org/?catid=6&gameid=52](http://www.liflg.org/?catid=6&gameid=52) after installing et of course :) . After it has completed downloading right click properties and under permissions check run as executable program. Then open a terminal, browse to the directory where you saved the .run file (eg cd Desktop), and type "sudo ./true.combat.elite_0.49b-english-4.run " without quotes of course and ur done :)
