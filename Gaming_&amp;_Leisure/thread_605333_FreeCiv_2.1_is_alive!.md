### Post by Mr. Picklesworth on 2007-11-06
It's been in Beta for longer than three full releases of Ubuntu, but [Freeciv](www.freeciv.org) 2.1 it is finally stable :)
[http://www.getdeb.net/app.php?name=FreeCiv](http://www.getdeb.net/app.php?name=FreeCiv)

There are a lot of improvements in this release, including quite a bit of sound and a huge visual refresh. The default tile theme is now very sharp and pretty, city names appear in a darkened box. There is a dependencies tree for the Science stuff, and while it is not the prettiest thing ever, it works well.

It no longer feels like using GTK for the game's user interface is a bad thing, as it now has a cool skin. I didn't know that was possible! Makes me like GTK even more than I already did; this thing looks surprisingly good, and has the bonus of being very accessible.
