### Post by oldrocker99 on 2016-08-26
My PCs have been bought on the cheap, mostly. I have a fairly low-to-mid-end rig, which I use for gaming (very successfully) and general computer use, and I'll add that, aside from running certain Windows programs, I can do everything I want my computer to do using Linux. I did dual-boot for a year or two, but Windows 7 was more much annoying and problem-prone than Ubuntu has ever been for me. Once X-COM was available, I deep-sixed that Windows partition and haven't looked back. 

AMD 8320E
MSI 970 Gaming mobo (for which I did have to edit grub to make USB3 work, otherwise very fine)
16GB 1600 RAM
nVidia 960


It is a truism that any hardware (except for **some** wireless adapters) is very, very likely to run on Linux, especially if the hardware is more than about six months old, without needing device drivers. Any Intel chip will do just fine as well. nVidia cards do require closed-source drivers, which are available directly from the Ubuntu repositories.

In other words, $1000 is just fine an amount to build a good Linux rig which will game very well. Mine cost ~$750.
