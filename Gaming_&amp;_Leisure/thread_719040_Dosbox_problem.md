### Post by pytheas22 on 2008-03-08
It's been a while since I used dosbox, but as I recall, you have to mount a location on your system as a drive inside dosbox using the mount command, which it sounds like you've successfully done.  Then you type the name of the drive you just mounted (e.g. c: or e: or whatever letter you chose), and press enter.  From there you can cd to the location of the executable you want to run, and run it.  If you need more help, please explain more specifically what commands you've run so far and what you're looking at in dosbox now.
