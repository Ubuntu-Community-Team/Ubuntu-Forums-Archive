### Post by donkyhotay on 2012-04-13
The problem with games like duckhunt isn't the game itself. That would probably be relatively easy to create, the problem is the 'gun' part of it. The old light zapper used for those NES used the refresh rate of the CRT screen to help determine where it was aiming at. It is physically incapable of working with a modern LCD or LED screen. I'm not really into that kind of game so I don't know if how more modern ones work that will work with LCD/LED screens or if they are compatible with linux.

### Post by KIAaze on 2012-04-15
The Wiimote might work. I've tried it as a mouse and remote control and was surprised how easy it was to install. But I used it without IR lights, so only accelerometers.
Without the IR lights, it's probably useless for aiming, but with them, it might work. (At least as well as it does on the wii.)

I don't remember which packages I installed and am not at home to test now, but just do a search for "wii" in synaptic and you'll find plenty.
I don't think I had to install anything additional from source. All just apt-get install. :)
