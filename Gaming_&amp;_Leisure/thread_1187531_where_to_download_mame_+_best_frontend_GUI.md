### Post by joezen on 2010-07-09
I'm using gxmame on an Intel Atom N270 w Lucid Lynx & its super easy...


just DL [http://surfnet.dl.sourceforge.net/sourceforge/gxmame/gxmame_0.35beta2-1_i386.deb](http://surfnet.dl.sourceforge.net/sourceforge/gxmame/gxmame_0.35beta2-1_i386.deb) then run a terminal with:

```
***cd Downloads | sudo apt-get install xmame-x xmame-sdl | ******sudo dpkg -i gxmame_0.35beta2-1_i386.deb***
```
