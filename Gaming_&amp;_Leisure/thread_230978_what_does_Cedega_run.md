### Post by Sutekh on 2006-08-06
You might have better success with [Wine](http://www.winehq.com/), which is kinda Cedega without the DirectX.  Plus it's free in every sense of the word, and easy to install.

You can browse their [Application Database](http://appdb.winehq.org/) to search for running programs like [Photoshop](http://appdb.winehq.org/appview.php?iAppId=17).  The DB has  lot of information, including the most successul version of Wine to use, and many other issues/limitations, etc.

If you want to give it a shot, then installing it can be done through apt-get/Synaptic, just folow the instructions on their site.

Hiya king,

I have used cedega in the past. I quit using it, when it turned out to not support ATI under AMD64 for the game I wanted to play (Battlefield2), but YMMV.

Using nvidia and i386 (i586) I could at least run several windows applications and games.
Plus you get a say in where emphasis for development of cedega will be in the next period using a vote system.

Still, it will cost you $5 per month for a minimum of 3 months. 

On the other hand, as Sutekh correctly states, if it runs using wine, why bother with the hassle of cedega? My advice?
Try wine first, then decide if you want to invest $15 or more in cedega.
