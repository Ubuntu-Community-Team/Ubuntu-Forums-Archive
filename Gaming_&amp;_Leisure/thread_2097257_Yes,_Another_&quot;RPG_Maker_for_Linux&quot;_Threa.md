### Post by Sofox on 2012-12-23
Open and sourced: [I'm not seeing many]("https://apps.ubuntu.com/cat/search/?q=rpg+maker&op="), if you could point them out, I'd appreciate it.

ntz: If you can compile from source, you're most of the way there.

Github is basically a way of storing source code online. If you go to the [Editor repository]("https://github.com/EasyRPG/Editor") for example, you can just click on the "ZIP" button and download the source code as a single file. Then just extract it, and follow the instructions in the INSTALL file to run.

Github can actually do a lot more than that (such as allowing people to contribute), but for now what I've said should be enough.
