Go to [http://winehq.org/site/download-deb](http://winehq.org/site/download-deb).  Ignore the first couple of paragraphs, because it is specific to Ubuntu (uses Synaptic) but use the instructions starting at "Installing from the WineHQ APT Repository with the console:".

Once that's done, type "winecfg" into a console.  It's a pretty graphical interface that should be understandable.
