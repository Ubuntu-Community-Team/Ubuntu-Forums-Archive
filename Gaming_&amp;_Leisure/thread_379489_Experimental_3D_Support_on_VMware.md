### Post by Tuffy on 2007-03-08
While I was surfing the net this morning I stumbled across this great find:
[http://www.youtube.com/watch?v=xF_CoXsXtk4]("http://www.youtube.com/watch?v=xF_CoXsXtk4")
As you can see, full DX 8.1 support is currently available for the beta version of VMware fusion. I was so excited when I saw this, so I searched for a possible sollution with the regular VMware Workstation. Lo and behold! 
[http://www.vmware.com/support/ws5/doc/ws_vidsound_d3d_enabling_vm.html]("http://www.vmware.com/support/ws5/doc/ws_vidsound_d3d_enabling_vm.html")
Im currently sitting on a windows machine, testing this out. well, it works perfectly. Im able to play deus ex with all the settings on high and no lag! no bugs! no problems! its magnificent!

As soon as im able, I will test if the procedure works on the linux version. since its basicly the same program im pretty sure my experiment will be a success.

this means, fellow linux gamers, we'll soon be able to play pretty much all games for windows on a linux without dual-booting! :popcorn: :popcorn: :popcorn: :guitar: :lolflag:
