### Post by Mike_IronFist on 2010-08-11
> **Gannin said:**
> Is there a way to get FluidSynth working with DOSBox so it'll handle the MIDI?
Well, you could install aconnectgui .
aconnectgui allows you to connect any program using MIDI to any other Program using MIDI, so you could try that. Open DOSBox, make sure it's running something that uses MIDI, and then open aconnectgui and see if you can connect DOSBox to Fluidsynth from the aconnectgui window.
