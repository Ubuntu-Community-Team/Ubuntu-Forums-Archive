Description: Stendhal is a full fledged multiplayer online adventures game like Ultima Online

Changelog:

    * added regrowing grain fields which can be harvested
    * added a windmill and an NPC who mills flour from grain
    * added a bakery and an NPC who bakes bread from flour
    * gave archer elf and archer kobold something to shoot
    * made iron casting take time

Dapper package:
[http://lamego.pinto.googlepages.com/stendhal_0.53-1_i386.deb](http://lamego.pinto.googlepages.com/stendhal_0.53-1_i386.deb)

NOTE:
You will need SUN's Java installed to run the game.
