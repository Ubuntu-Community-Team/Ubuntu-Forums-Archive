### Post by twin_57103 on 2008-04-01
I emailed the developer, asking specifically about hosting a private game.  Here is the response I got:

> On your Ubuntu computer: run Pioneers Server (pioneers-server-gtk)
Here you can select the game you want to play. Notice the server port,
you'll need that for the Windows computers that connect to the game.
Do not select 'Register Server', otherwise the game is not private, but
public. Start the server.
On the computers that join the game, select 'Join Private Game'. For
'Server Host', use the hostname of your Ubuntu computer (you can find
that with 'hostname'), and for 'Server Port' the number you've noted
earlier (default=5556).
You should be able to connect to the server now.
