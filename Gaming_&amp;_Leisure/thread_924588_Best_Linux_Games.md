### Post by Naegling23 on 2008-09-26
I only have one free game that I enjoy:
Frets on Fire - If you play the latest mods, the game is amazing!

As for paid games:
The current "best" as in shiny, new game is prob. Quake Wars Enemy territory.  When UT3 gets released, we can add that to the list.  A lot of people also like Eve online and Savage 2.
So, those are a few games from this current generation.  If you want to jump back a bit, we've got doom3, ut2004, Neverwinter nights and a few more.  Neverwinter nights is a great game that doesnt get its fair share of due (even though its trumpeted so much as a linux game).  Its big, huge even, and user created content makes it that much bigger.  It's deep, like D&D deep.  There is a lot to do and explore, once you start playing, you want to try out other character classes and do things in a different way.  It also can be played online, so you can either play cooperatively with your friends, or join a persistant world, and have basically an MMO.  Sure, the game is old, and the graphics are dated, but its an isometric view (similar to diablo) so you can get away with weaker graphics than you could with a fps.
