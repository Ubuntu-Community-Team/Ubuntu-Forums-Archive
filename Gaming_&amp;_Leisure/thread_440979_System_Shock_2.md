Unzip it, and run the OGGsetup (right click, open with Cedega). This unpacks all the files, and sets up the audio. 

Then, open up Cedega, and run the Sshock.exe as an installation (avoid Sshock2.exe-it doesn't do anything). It's annoying, you have to pretend to install it-it does not run any other way. Other than that, it runs great.
