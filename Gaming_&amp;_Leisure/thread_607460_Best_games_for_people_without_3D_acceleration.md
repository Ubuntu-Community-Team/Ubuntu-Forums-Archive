### Post by dahlek on 2007-11-10
**Arcade volleyball **is classic fun. It's nothing at all like volleyball, and has the "works", meaning you can play against an AI, against someone sitting next to you, or against someone on a LAN/Internet. This one outdoes the original DOS CGA game by improved features, such as 4 players, new themes, etc. This was originally a Compute! magazine game. I used to play this on a 4.77 Mhz IBM XT with no hard drive and 640K RAM:

[http://gav.sourceforge.net/](http://gav.sourceforge.net/)
sudo apt-get install gav gav-themes

**Liquidwar** is another cool game, IMO. "Blob melee" might be a better title - ever player controls a blob like mass and you have to eat the other blobs. There are actually some interesting strategies that you need to use to win and the controls are incredibly simple. It has the same options for opponents that gav has, you can play with other people at the same keyboard, someone online, or against an AI - or, a combination.

[http://www.ufoot.org/liquidwar/v5](http://www.ufoot.org/liquidwar/v5)
sudo apt-get install liquidwar

**Jump'n'bump** might sound really stupid, but it's actually really fun. Like the above, you can also play against any combination of AIs, people next to you, or folks online. You are a bunny, usually, and you have simply to jump on your opponent: this is like a frag - he pops and squirts out blood and respawns somewhere else on the board. Some things in this game aren't obvious, so if you have questions about how to turn certain features on or off, let me know. The game has no in-game menu, but I believe the ubuntu package installs something called "jumpnbump menu" by default which is a small GUI python script that lets you select some options, which level you want to play, if you want a LAN/Internet game and so on.

The levels package gives you other boards and on some of them, you aren't a bunny, you might be a Mario character for example. There are many many more levels you can find and download online.

sudo apt-get install jumpnbump jumpnbump-levels

**Freecraft** is still a favorite of mine. It's a Warcraft2 clone - Blizzard's big pre-Starcraft game. You can play against real people and AIs, but not against other humans sharing your keyboard. The same folks made other games but they all run slower on old computers in my experience.

sudo apt-get install freecraft
