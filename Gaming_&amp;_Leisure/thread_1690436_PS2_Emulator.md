### Post by richard.e.morton on 2011-11-13
I havent done it; but a quick search turned up a PPA for the software PCSX2 which is an emulator for windows and Linux

install the PPA - [https://launchpad.net/~gregory-hainaut/+archive/pcsx2.official.ppa](https://launchpad.net/~gregory-hainaut/+archive/pcsx2.official.ppa)
install the package - sudo apt-get install pcsx2.snapshot

remember emulators are a work in progress

dont forget that there are other great emulators such as these (and many more)
 - MAME - [http://mamedev.org/](http://mamedev.org/)
 - MESS - [http://www.mess.org/](http://www.mess.org/)
 - Dolphin (Wii and GameCube) - [https://launchpad.net/~glennric/+archive/dolphin-emu](https://launchpad.net/~glennric/+archive/dolphin-emu)
 - dosbox - [http://www.dosbox.com/](http://www.dosbox.com/)

[http://en.wikipedia.org/wiki/List_of_video_game_console_emulators](http://en.wikipedia.org/wiki/List_of_video_game_console_emulators)
