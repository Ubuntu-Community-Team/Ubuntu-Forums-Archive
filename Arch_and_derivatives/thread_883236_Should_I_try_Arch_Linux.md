### Post by smartboyathome on 2008-08-07
If you have the time to install it and configure everything how you like, sure. If not, don't bother. The guide you should use is [here]("http://wiki.archlinux.org/index.php/Beginners_Guide"), I used that when I first tried arch and it helped me a lot.

I wouldn't say installation is complicated if you only install using pacman. It is AUR which is the really complicated part. Some apps don't work when they come from AUR, and some apps there are old. If you are looking for a GUI for pacman, just use gtkPacman.
