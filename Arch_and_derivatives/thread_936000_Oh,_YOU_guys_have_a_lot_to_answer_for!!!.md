### Post by Canis familiaris on 2008-10-04
> **lukjad007 said:**
> I'll have to wait for a bigger HD. That will be soon and I'll be installing a bunch of OSes. Is it better to install Arch then Ubuntu or vice versa?

Install them in any order. But keep a [Super GRUB disk]("http://www.supergrubdisk.org/") in handy.
It will help you recover your GRUB just in case.

Also Remember these lines to add to /boot/grub/menu.lst:
```
#<insert name of the OS here />
title	<insert name of the OS here />
root	(hd0,1)
chainloader	+1
makeactive
boot
```
Replace <insert name of the OS here /> by the OS name and change the root partition accordingly.
In (hd0,1): 0 refers to the first hard disk and 1 refers to the second partition. So if Arch is installed in say /dev/sdb3 (i.e. 3rd partition of your second hard disk) then replace it as (hd1,2).
