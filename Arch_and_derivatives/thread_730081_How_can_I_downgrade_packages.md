### Post by fwojciec on 2008-03-20
By default pacman is not going to overwrite any files that exist on the system -- although overwriting can be forced using -f switch.  In this case it should be safe to allow pacman to overwrite the conflicting files -- you can do it by issuing a "pacman -Sy**f** kbd" command.
