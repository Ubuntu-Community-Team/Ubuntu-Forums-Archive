### Post by crimesaucer on 2008-12-03
[http://aur.archlinux.org/packages.php?ID=21601](http://aur.archlinux.org/packages.php?ID=21601)


I was a bit nervous to mess with my flash64 plugins that I had been using since they pretty much worked using those lib32's and the nspluginwrapper/nspluginwrapper-flash.....


..... but after I went ahead and removed my plugin from Firefox with:

```
nspluginwrapper -v -r ~/.mozilla/plugins/*
```

and also both the old packages so I could install the new flashplugin-alpha-64 package, I was glad I did it.


I'm really impressed at how much better the flash sites were. My nba.com home page finally works correctly with the drop down menus in the navi-bar working perfectly. I've seen nothing but improvements so far. I recommend it.
