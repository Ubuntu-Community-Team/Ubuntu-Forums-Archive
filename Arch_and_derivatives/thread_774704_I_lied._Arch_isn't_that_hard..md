Wow, I finally got my ATI proprietary drivers installed and working perfectly. :D Incase anyone else had the problem where they would crash/segfault as soon as you start X, you need to install X, then run Xorg -configure, THEN install the proprietary drivers and run "aticonfig --initial". Turns out if you do it in a different order, it messes up.

Anyways, I got arch up and running with gnome, and I'm excited now. :D I'm learning so much about Linux, and this isn't really all that hard at all. Sorry Ubuntu, but I think I've found a new toy ^_^;
