### Post by tsger on 2008-07-07
If you have problems opening folders using Gnome-Do 0.5 in Arch Linux, you may need to install xdg-utils. This will give you xdg-open, which is needed in order for the "Open" function to work in Gnome-Do.

[http://bbs.archlinux.org/viewtopic.php?pid=390454#p390454](http://bbs.archlinux.org/viewtopic.php?pid=390454#p390454)

Hope this makes someone else as happy as it made me!
