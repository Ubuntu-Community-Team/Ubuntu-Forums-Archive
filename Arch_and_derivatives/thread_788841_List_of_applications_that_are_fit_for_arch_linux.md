### Post by MONODA on 2008-05-10
This thread is just meant to be a list of applications that suit archlinux meaning that they can be installed using yaourt and not drag in a bunch of dependencies. The apps should also be arch-like meaning fast and simple. I will start:

1. cplay great CLI based music player
2. rtorrent CLI torrent app
3. Kazehasake Web browser
4. Osmo calender and task taker
5. Sylpheed email client
6. sakura terminal with tabs
7. emelfm2 file manager
8. pcmanfm file manager
9. geany IDE
10. leafpad text editor
11. mirage picture viewer
12. ePDFview pdf viewer
13. audacious music player
14. recorder simple cd/dvd burner
15. xpad sticky notes
16. openbox wm
17. obconf openbox configuration manager
18. obmenu openbox menu editor
19. gtk-chtheme change the gtk theme
20. mmaker menu maker for *box
