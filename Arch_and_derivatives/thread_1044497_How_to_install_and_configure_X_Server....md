That is the installation. It only gives you the base/core packages needed to get a root console.

You will have to create a user account, and do a full system upgrade, reboot.


Then you can add xorg and your video driver, configure xorg.conf, and then you can add gnome or whatever you use.

[http://wiki.archlinux.org/index.php/Beginners_Guide#Install_X](http://wiki.archlinux.org/index.php/Beginners_Guide#Install_X)


Just follow this guide from start to finish.


You can also look at this old guide here.... but make sure that you use the latest instructions from the official ArchWiki.

Old Guide to show you where you are at: [http://www.raiden.net/?cat=2&aid=276&pid=7](http://www.raiden.net/?cat=2&aid=276&pid=7)
