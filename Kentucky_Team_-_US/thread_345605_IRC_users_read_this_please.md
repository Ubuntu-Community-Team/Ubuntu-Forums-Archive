### Post by etank on 2007-02-12
IRC is short for Internet Relay Chat. If you have never used IRC before then I would recommend starting with Xchat. Xchat has a nice easy to use grpahical interface. 

Here are some links that will help you get started.

[https://help.ubuntu.com/community/InternetRelayChat](https://help.ubuntu.com/community/InternetRelayChat)
[https://help.ubuntu.com/community/XChatHowto](https://help.ubuntu.com/community/XChatHowto)
[http://www.irchelp.org/irchelp/new2irc.html](http://www.irchelp.org/irchelp/new2irc.html)

Please be sure to change the default port from 6667 to 8001. See the previous posts for more information on this.
