**Background**
So we are an Officially Approved LoCo Team now. With that comes the ability to use a sub-domain of ubuntu-us.org. We have been given [http://kentucky.ubuntu-us.org](http://kentucky.ubuntu-us.org) and [http://ky.ubuntu-us.org](http://ky.ubuntu-us.org) (both point to the same location) and we already have [http://ubuntu-kentucky.org](http://ubuntu-kentucky.org) (seperate from the other two and hosted by us). At this time it is hosted by Ubuntu. We are waiting now to see if they can make a DNS change and point to a server that we own so that it would be easier for us to maintain. Right now the site is using Drupal and has the Ubuntu theme applied to it.
