### Post by 3rdalbum on 2014-12-07
Ubuntu Touch System Requirements:
CPU: 1 GHz ARM Cortex-A9 or Intel Atom
Memory: 512MB - 1GB RAM
Storage: 4-8 GB
Multi-touch Screen

Your iPAQ:
CPU: Intel XScale 312 MHz
Memory: 64 MB
Storage: 64 MB (SD cards supported)
Single-touch screen

So no, I don't think you'll be running Ubuntu any time soon...
