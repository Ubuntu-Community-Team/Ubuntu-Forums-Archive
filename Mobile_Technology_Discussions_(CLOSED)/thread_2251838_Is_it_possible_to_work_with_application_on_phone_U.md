### Post by grahammechanical on 2014-11-07
The applications that we use in Ubuntu desktop are not suited to give a good user experience on a device with the screen size of a mobile phone or even a tablet. Ubuntu developers have the intention to have one Ubuntu code base.

When Ubuntu is pre-installed onto a mobile phone the user will experience an interface suitable for a mobile phone and applications to match.

When Ubuntu is pre-installed onto a tablet the user will experience an interface suitable for a tablet and applications that scale to the tablet screen. It will be similar to the phone User Interface but with added features that the tablet size screen makes possible. If the hardware of the tablet allows, then Ubuntu on the tablet will have Ubuntu phone functions.

When Ubuntu is installed on a laptop/desktop then the user will experience the usual Ubuntu interface (Unity) with the usual applications that we have in Ubuntu desktop. We will also be able to install and run the applications in the Ubuntu phone/tablet app store.

There is one further aim. When we connect a monitor, keyboard and mouse to Ubuntu phone or tablet then the mobile device will act like a PC and give the user the normal Ubuntu desktop experience with the added functions of a phone. And the apps in the Ubuntu for Devices app store. 

A lot will depend on the hardware specifications of each product released to market by the OEMs. and it all takes time to complete.

The applications in Ubuntu are developed by what are called third party developers. It is up to those developers to re-write their applications to run on mobile devices if they wish mobile device users to use their software. There is talk of Libreoffice developers porting Libreoffice to Android. How long this will take and how good Libreoffice will be on an Android device is something that only the future will tell. The same is true as to whether they will port mobile Libreoffice to Ubuntu for Devices. And of course, it will have to pass the acceptance tests to get into the app store just like any other app in the Ubuntu app store.
