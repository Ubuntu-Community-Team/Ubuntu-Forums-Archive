### Post by grahammechanical on 2013-09-08
Have you read the Ubuntu Touch wiki?
> 
**[SIZE=2]Can you make it run on my device xyz as well?[/SIZE]**

 In  order to understand the answer to this question you should get  aquainted to open source development in general. Even though the whole  idea of Ubuntu Touch is organized and driven by canonical it remains an  open source project and is characterized and limited by the same  constraints. The main work is done by volunteers and in case there is  nobody yet working on the device that you would like to have ported then  good luck. You can only wait (and hope) or start [porting]("https://wiki.ubuntu.com/Touch/Porting") yourself. You can also look if you device have already someone "working" on it. List of  [Devices]("https://wiki.ubuntu.com/Touch/Devices")

There are two places to look to find out if someone is working on porting Ubuntu Touch to your device

[https://wiki.ubuntu.com/Touch/Devices](https://wiki.ubuntu.com/Touch/Devices)

[http://forum.xda-developers.com/showthread.php?t=2239338](http://forum.xda-developers.com/showthread.php?t=2239338)

Remember, Ubuntu Touch is still developer preview code and the work being done by Canonical is for the purpose of providing Ubuntu Touch as a mobile platform for OEMs. Who knows how well it will run on ant device, especially a device that Ubuntu Touch has not had supplier tuning to get it running well?

Regards.
