### Post by ian-weisser on 2016-09-18
Creating a 'Mobile OS' is not the problem. Ubuntu and others already did that.
The problem is that an OS must be customized for each hardware platform (unlike the desktop x86 model, which is quite standardized by comparison).
The other problem is making installation (and recovery from botched installation) easy. No BIOS, which is why you flash devices. Make a mistake on a desktop, and simply slide in a different CD. Make a mistake on a mobile, and it's bricked without easy recovery.
