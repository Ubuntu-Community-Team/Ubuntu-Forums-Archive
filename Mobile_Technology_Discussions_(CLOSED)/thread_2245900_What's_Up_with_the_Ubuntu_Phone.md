[http://summit.ubuntu.com/uos-1406/all/](http://summit.ubuntu.com/uos-1406/all/)

[http://summit.ubuntu.com/uos-1411/](http://summit.ubuntu.com/uos-1411/)

It is not correct to call Ubuntu for devices "vapourware." The code is there and it is free to download and install on the 3 Nexus reference devices. It works. Ubuntu for devices has reached the stage where the images are ready or about ready to be handed over to two OEM partners for installation testing on the hardware they intend to ship as Ubuntu phones.

[https://insights.ubuntu.com/topic/phone/?cat=1189](https://insights.ubuntu.com/topic/phone/?cat=1189)

Ubuntu developers have been using Ubuntu for devices on the Nexus phones for months and they have been using it as their main mobile phone. Want a mapping app on your phone? This how the Ubuntu for devices OS will make it possible for app developers to write such an app.
