### Post by 3rdalbum on 2014-03-31
It is not possible to make one operating system image that will run on all tablets or all phones. Not possible at the moment.

Certain Nexus devices are officially supported as "development devices". Some other phones are supported by community ports. The real intent of Ubuntu Touch is to be preinstalled on new phone models.

The release of Ubuntu 14.04 does not change anything. You can already get Ubuntu Touch.
