Ubuntu Touch = Ubuntu for Devices = Ubuntu Phone

Ubuntu Phone = Mir + Unity 8 + app confinement + Over the Air (OTA) transactional updates + Click packaged apps + phone technology.

Ubuntu Core + Snap packaging = Snappy Ubuntu Core.

Ubuntu Personal = Snappy Ubuntu Core + Mir + Unity 8 + app confinement + OTA transactional updates - apt-get - deb packages.

Ubuntu Personal converged with Ubuntu Phone = "It is Ubuntu, Jim, but not as we know it."

There are Ubuntu Personal images available for installation in KVM or for flashing to a USB stick but not as an ISO image. It is not yet a desktop OS. Time will tell.
