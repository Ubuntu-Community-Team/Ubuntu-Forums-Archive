### Post by grahammechanical on 2015-03-22
Ubuntu phones are just that phones. The converged device that you (and I for that matter) are seeking does not yet exist. A Ubuntu tablet when it comes out will be a bit closer to our desired device. There is a video on discourse.ubuntu that shows the tablet UI switch from a phone UI to a tablet UI when a bluetooth mouse is connected. Applications go from full screen to windowed applications and back again as the bluetooth mouse is connected and disconnected. That video also shows applications like Libreoffice running in windowed mode.

I have installed Ubuntu Desktop Next on my desktop. This is the code base where convergence is being experimented with. At the moment it is like having a phone OS on a large desktop monitor. Although applications now launch as resizable windows and not full screen. I do believe that all the default desktop applications are there but I have yet to find a way to load them. The default web briowser is Ubuntu Web Browser. I am using it now. It is a default application in Desktop Ubuntu. It is a nice mobile web browser. It is not a function for function rival with the usual desktop browsers. And I cannot paste into this forum reply box otherwise I would have provided a link to that video.

So, my assessment is this: A Ubuntu superphone will need the right hardware and that in turn will need software that makes full use of the hardware. This software is still under development and it is my guess that we will see converged Ubuntu (Mir and  Unity 8) on the desktop before we see Ubuntu superphones on the market. It will all happen within the next year. Or maybe not.

Regards.
