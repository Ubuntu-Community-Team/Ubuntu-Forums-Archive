### Post by mooreted on 2015-02-07
Yeah, Ubuntu phone and tablet is just beginning. I suppose it can be done, but I don't think it's going to be easy. It might be a fun project. If you figure it out, you can teach us how to do it.

The problem is, a tablet and phone are not quite like a PC or laptop. The architecture is different, the internal hardware is proprietary, each OS has to be built especially for that particular device. Look at instructions for rooting a phone and you'll see it can be a bit involved. Do it wrong, and you have a fancy paper weight.

Most likely you'll have to find a developer that has done it on a particular device and buy that exact device.
