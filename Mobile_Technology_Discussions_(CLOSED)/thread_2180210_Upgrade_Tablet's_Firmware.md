Let's start with something more basic since it appears you don't know the answer to the above:

Can you plug your tablet into your Ubuntu PC with a USB cable?

If so, does the device automatically show up in "Files"?

Does the tablet have an SD or MircoSD slot?

I ask because sometimes it is as simple as copying a downloaded zip file of the firmware update to the device and letting it reboot.  You don't need to unzip the file - just move it over.

### Post by Mark Phelps on 2013-10-16
If you went to their support website and to get the firmware update, that site will indicate what is needed to do the firmware update.

Unfortunately, since this tablet comes with Android 4.1 and Win8 preinstalled, it's most likely that you will have to use one of those to do the update.

If you start messing around in Ubuntu trying to do this, you run the risk of turning the tablet into an "electric brick".

You should NEVER do a firmware update unless BOTH of the following are true:
1) You have a way to save off the current firmware to a device that can be read without using the OS on the device
2) You have a way to restore the saved firmware in the event that the updated goes badly
