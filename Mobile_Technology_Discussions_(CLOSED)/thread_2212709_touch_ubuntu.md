### Post by grahammechanical on 2014-03-24
Here is somewhere else to browse.

[http://insights.ubuntu.com/](http://insights.ubuntu.com/)

Keep in mind that official development is and has always been directed towards Ubuntu being pre-installed on retail mobile devices. First, smartphones followed by tablets then by the convergence of the Ubuntu mobile code base with the Ubuntu desktop code base and then TVs.

Official development is not directed towards making Ubuntu installable on every and any mobile device that is out there. That is an option open to community developers and is officially welcomed and approved because Ubuntu is open source.

See this interview with Mark Shuttleworth.

[http://www.mobileworldlive.com/interview-ceo-canonical](http://www.mobileworldlive.com/interview-ceo-canonical)
