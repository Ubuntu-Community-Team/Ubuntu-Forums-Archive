### Post by Nr90 on 2013-06-09
Well so far the aim seems to be to use as little hardware buttons as possible.
So I think you'll have to go through software at first (swipe down the volume icon and you get a mute button).

However, others have asked if there will be a possibility to use hardware buttons.
This seems likely to me, as implementation cannot be too difficult and provide more options to manufacturers.

One such buttons could be a switch like on the iPhone.
This would be down to the manufacturer I assume.
