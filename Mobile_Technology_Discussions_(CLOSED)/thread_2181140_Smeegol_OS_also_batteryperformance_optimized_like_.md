### Post by grahammechanical on 2013-10-16
What do you mean exactly by "fatter" and lighter?"

Ubuntu Touch is the basis for the Ubuntu ideal of convergence. The ability to turn a smart phone into a PC that will run the full Ubuntu desktop platform. From this blog we can see that the Ubuntu Touch platform now has Unity 8 in it.

[http://www.phoronix.com/scan.php?page=news_item&px=MTQ3OTA](http://www.phoronix.com/scan.php?page=news_item&px=MTQ3OTA)

Unity 8 is the Ubuntu Desktop user interface that will be the default user interface in Ubuntu 14.10 desktop. Over the next year Ubuntu desktop development branch will be getting the Ubuntu Touch code so that the desktop platform will have the Ubuntu Touch core applications and all the rest.

Ubuntu 13.10 desktop has the unity 7 user interface and that will not change for Ubuntu 14.04 LTS. But it is planned for convergence to be complete by Ubuntu 14.10 desktop and that means Unity 8 and Ubuntu Touch as part of the desktop code. 

So, if we are measuring 'Lightness' by the number of lines of code, then Ubuntu is certainly getting fatter. On the other hand Ubuntu desktop is benefiting from all the work being done on the mobile platform so that a Ubuntu phone has good battery life and a good user experience.

I am seeing a 10% - 15% drop in memory usage in 13.04 and 13.10 when compared to the original 12.04. So, in this sense Ubuntu is getting 'lighter.' And I think that Ubuntu desktop will get 'lighter' again when the full MIr replaces Xserver, Lighdm and Compiz in Ubuntu 14.10.
