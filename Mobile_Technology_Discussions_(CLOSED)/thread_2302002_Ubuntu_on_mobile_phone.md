### Post by grahammechanical on 2015-11-06
There are two sections of this forum where this question has been asked and answered before. The Mobile Technology Discussions section and the Ubuntu Phone & Tablet section. The answer is always the same. And it is "No."

Ubuntu Phone is not being developed to be installed on mobile devices in the same way we install Ubuntu on to a desktop or laptop machine. The Ubuntu Phone OS is developed for Original Equipment Manufacturers (OEMs) to pre-install on mobile devices for retail. 

The Ubuntu Phone OS is open source code therefore if someone wants to experiment porting Ubuntu Phone to their device then they can try. But Canonical is not working on this. 

[https://developer.ubuntu.com/en/start/ubuntu-for-devices/](https://developer.ubuntu.com/en/start/ubuntu-for-devices/)

### Post by Jordan_Roberts on 2015-11-12
The mobile market is not the desktop market. It will take time, but Ubuntu has a decent shot at becoming a player in the former, and that means taking away sales from Android devices. It means encouraging people to forego Android entirely and buy something with Ubuntu preinstalled (also because, if you were to buy an Android phone and throw Ubuntu on it yourself, this will still show up in the statistics as an Android sale and nothing more.) This is not the same as - for example - Dell preinstalling Ubuntu on laptops, because a laptop is a different kind of consumer device, subject to very different commercial dynamics.
