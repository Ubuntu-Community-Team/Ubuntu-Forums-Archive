Or should I install the regular Ubuntu (e.g. the upcoming 14.04) on them?

Ubuntu Touch is meant for devices that come with it preinstalled. It's not intended to be installed on third-party devices. Having said that, some Nexus devices can run Ubuntu Touch for the purpose of developing the system in anticipation of purpose-built hardware.

x86 PCs, including the Surface Pro, will use ordinary Ubuntu.

### Post by grahammechanical on 2014-03-10
You are not the first to ask this question and it has been answered many times in this section of this forum. All we can do is point you to the official web pages.

[https://wiki.ubuntu.com/Touch/Devices](https://wiki.ubuntu.com/Touch/Devices)

All the development being done for Ubuntu on phones/tablets will be converged into the Ubuntu desktop code so that there is just one Ubuntu code base. In fact developers have started to speak only of "Ubuntu" and not "Ubuntu Touch" any more. But this convergence strategy will not be complete until the release of Ubuntu 14.10 or even Ubuntu 15.04.

Do not expect more than the developers can deliver. Remember, Ubuntu is not developed by a massive commercial organisation but by a community of developers. For example, the core phone applications are all the work of community developers. There are limits to what can be achieved.
