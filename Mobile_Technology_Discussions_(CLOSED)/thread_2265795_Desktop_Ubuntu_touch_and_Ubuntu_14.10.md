### Post by grahammechanical on 2015-02-18
Can you imagine a Linux desktop user interface on a mobile phone device? It would be unusable. There must be differences even between a phone UI and a tablet UI let alone a desktop UI.

The default Ubuntu UI is Unity and the Ubuntu phone uses Unity but a version that has been re-written to run on the Mir display server. There are similarities between Unity on the desktop and Unity on the phone. There is a launcher for one thing. But of necessity Unity on the phone works differently. The user swipes from the four screen edges and uses touch to load applications and activate menu instructions.

I will put it another way. I would be happy to use a Ubuntu phone or a Ubuntu tablet but I would not like the Ubuntu phone UI as my desktop UI. And that is not going to happen. I would also be happy to use some of the phone core apps as replacements for desktop apps on condition that there were certain modifications. I can also see the benefit of having some of the phone scopes available on the desktop.

There are videos showing the phone in action. Look at those. A picture paints a thousand words.
