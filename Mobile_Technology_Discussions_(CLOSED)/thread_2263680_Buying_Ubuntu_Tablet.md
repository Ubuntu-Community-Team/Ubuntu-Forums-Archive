### Post by dotte on 2015-02-04
I've got an Acer Iconia W700P, as it's one of the few tablets out there that has an old-school x64/x86 i5 processor, which means full desktop versions of both Windows 8.1 and Ubuntu, which I've got setup in a dual-boot. 

Desktop ubuntu on a tablet can be a bit fiddly at times, as it's not designed to auto pop-up onscreen keyboards, auto rotate, or to always recognise swipe controls, but it does have full compatability with everything, so can't complain. 

I've thought about trying to setup a tri-boot with ubuntu touch as a third option, but it was such a pain getting the dual boot on the UEFI bios in the first place that I'm loath to try and fiddle with it again.
