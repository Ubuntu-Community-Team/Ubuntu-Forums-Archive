### Post by tgalati4 on 2013-01-28
You can run rhythmbox on your Ubuntu PC with the DAAP plug-in activated and then put a DAAP aware music player on the tablet.  For file access, I like ES File Manager.  It can see network shares and even play media.

You can also use a universal plug and play (uPnP) server on your Ubuntu box and install a uPnP aware media player on the tablet.  Although I haven't used it, there is:

ushare - lightweight UPnP A/V Media Server

I use freenas with a mediatomb upnp server and I can stream media to my Barnes & Noble Nook running Android.
