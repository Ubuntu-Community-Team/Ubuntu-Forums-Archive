### Post by Enigmapond on 2010-01-22
ClamAV is mainly for someone who runs a lot of things through Wine. Since Wine is a virtual windows set up, it can conceivably contract a virus. The normal Virus/Malware/Spyware can't install itself on a Linux platform.

### Post by lisati on 2010-01-22
I can only speak for myself: I have two reasons for considering antivirus for my Ubuntu installations. 

The first is that I sometimes run Windows, and consider it *useful* (but not *essential*) to have a tool available to help deal with malware, should something go wrong with my Windows installation. 

The other is that email providers (such as Yahoo, with whom my ISP has teamed up to proved my main email account) like you to have antivirus products installed, they sometimes even include a "thou shalt not transmit malware" clause in their terms and conditions.

### Post by running_rabbit07 on 2010-01-22
The Linux anti virus programs are for protecting Windows machines. A large amount of Linux systems are used as file servers and are unaffected by Windows viruses. There are some socially engineered malwares out there for linux, but AV programs will now spot them.
