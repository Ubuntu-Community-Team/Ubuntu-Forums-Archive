### Post by 2F4U on 2013-01-13
The first source of information would be the system log file, which may contain information about why Ubuntu crashes and reboots.
Did you recognize any pattern - besides Thunderbird - behind the crashes, e. g. applications that are always running when such a crash happens? 
Random crashes are often caused by bad RAM, so if you haven't already done run memtest for at least two iterations and see if it reports any problems.
