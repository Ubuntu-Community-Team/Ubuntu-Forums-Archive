no.. you cannot cluster two computers (yet) to automagically have one better desktop computer that performs better overall. Clustering is good for running very cpu intensive applications (I am talking calculations that take a very long time to complete on one computer) that are setup/programmed to be able to perform on a cluster. Normal applications like Linux desktop user applications cannot profit from such a setup (yet).
