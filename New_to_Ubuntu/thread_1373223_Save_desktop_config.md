### Post by njdove on 2010-01-05
In a default Ubuntu config with Visual Effects enabled you'll need to save two sets of preferences. First, the GNOME theme (controls, colors, window border, icons, pointer, and background image). You can save these settings as a Theme in the Appearance Preferences - click "Save As...".

You can also change the Visual Effects settings - lots of potential for showing off here. The best way to do this is to install the "Advanced Desktop Effects Settings" (ccsm) package. This will add "CompizConfig Settings Manager" to your Preferences menu. When you run it you can control a number of effects - try to make changes one at a time and don't change a setting unless you're reasonably comfortable with what you're changing. To save these preferences, click "Preferences" and add a new Profile to save everything.
