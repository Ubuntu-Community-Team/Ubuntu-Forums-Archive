### Post by wsxadf on 2013-06-30
Eclipse is well supported in ubuntu. you can download it easily by typing in the terminal

```
sudo apt-get install eclipse
```

or go to [http://www.eclipse.org/downloads/](http://www.eclipse.org/downloads/) and download the file you want to use.

If you want better experience when coding, I think it will be better to use lubuntu, kubuntu, or your custom ubuntu since your cpu is atom. I am currently using a netbook with atom 1.6ghz with ram 1gb. But default ubuntu is heavy for me, so I use ubuntu minimal installation with a dwm as my window manager.
