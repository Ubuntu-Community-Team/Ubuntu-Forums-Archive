### Post by glennric on 2008-05-07
If you log into Arch by typing root and enter, try that at the gdm login screen.  Wow, that is really bad though.  You have the root account enabled with no password.  Does Arch linux have their own forums?  You should go there to get help.

Edit:  By the way, if the root and enter thing doesn't work it is probably because you have root login disabled in the gdm settings.  That is the default, and so is probably the case.  You will have to create a user to log in, or figure out how to enable root login.  The latter is a bad idea.
