### Post by oldfred on 2019-01-29
Some systems do that, in promoting flash drive installer to sda.

You can try this:
       sudo grub-install /dev/sda
sudo update-grub

For future reference:
New versions of Ubuntu now use swap file, so no swap partition required. But if swap partition found it will use it.
Most desktop installs do not need a separate /boot partition. It becomes one more partition to manage. You do have to make sure it does not fill up. Ubuntu now housecleans out old kernels, so should not be an issue, but needs monitoring.
If only installing Ubuntu often better to use gpt. It is newer and has some udpate to make it more reliable, but not worth changing unless totally reinstalling.
