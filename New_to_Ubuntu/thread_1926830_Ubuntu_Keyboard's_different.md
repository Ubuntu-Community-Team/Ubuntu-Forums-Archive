### Post by coffeecat on 2012-02-17
Not weird. That sounds as though your system is set up for the UK keyboard layout. The @ and " on a UK keyboard are swapped around from the way they are on US keyboards and I guess maybe many other layouts. And the # key on a UK keyboard is where you find the \ on a US keyboard. UK-layout Mac keyboards are different again, but that's another story... :)

You need to select a different layout in Keyboard settings. If you need help with this, post back with which version of Ubuntu you are using.
