### Post by wilee-nilee on 2012-05-06
Generally a shared ntfs is what is advised with a windows and linux dualboot.

Does this computer have a recovery partition, if so they can save the stuff and reload the OS. Get them to clone the windows if they get a clean install this will save a lot of time in the future I use clonezilla personally.

Windows 7 has a imager/cloner already, if it is a OEM though it may be a one time shot. Make sure they have a recovery disc made in the backup section of W7 it may be a one time shot as well. If they have the install disc that works as well as a recovery for repair.

[http://clonezilla.org/](http://clonezilla.org/)
