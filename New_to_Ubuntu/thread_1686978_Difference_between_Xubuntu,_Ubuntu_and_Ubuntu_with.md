well ubuntu uses gnome and xubuntu uses xfce, yes, but its more than just what manages your windows and desktop. they're desktop environments, and this includes the default software. xubuntu will generally have lighter software for performing the same tasks than ubuntu uses. you're free to install any software you choose after installation though of course.

Xubuntu-desktop-

Ubuntu-desktop-
[http://packages.ubuntu.com/maverick/ubuntu-desktop](http://packages.ubuntu.com/maverick/ubuntu-desktop)

[http://packages.ubuntu.com/maverick/ubuntu-minimal](http://packages.ubuntu.com/maverick/ubuntu-minimal)

Xfce4-
[http://packages.ubuntu.com/maverick/xfce4](http://packages.ubuntu.com/maverick/xfce4)

Generally, xubuntu-desktop is heavier than minimal install + xfce4.
