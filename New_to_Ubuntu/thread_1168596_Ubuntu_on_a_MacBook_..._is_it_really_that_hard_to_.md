### Post by bugginmiami on 2009-05-24
You have to decide if you want to use Boot camp (truly tripleboot like you said) or virtualization (parallels or vmware fusion, not free).  I use parallels just because my predecessor at work bought it, and run win7 & ubuntu just fine in a window on my macbook. (full screen, youd be hard pressed to tell you didnt actually boot from it).  Bootcamp is free and works well, like multibooting a pc. The hardware is natively recognized, not emulated hardware like vm.
 If you run it as a vm, there are not a lot of hoops at all, its very simple. Boot camp a little more, but not bad (ive only done this with win7 and winxp, disclaimer!). I set up boot camp originally and havent used it as it runs vm's so well. Either way it sunds like you want to up memory to 2 gigs from the 1 that I believe that model comes with.
