### Post by buzzingrobot on 2013-09-27
One of the things that happens when a release reaches end-of-life is that the repositories that supply updates are turned off. 

Here is a brief snapshot of the release lifetimes ofUbuntu releases:  [http://www.ubuntu.com/info/release-end-of-life](http://www.ubuntu.com/info/release-end-of-life).

The 11.10 release reached end-of-life in May,  More here: [http://fridge.ubuntu.com/2013/05/10/ubuntu-11-10-oneiric-ocelot-end-of-life-reached-on-may-9-2013/](http://fridge.ubuntu.com/2013/05/10/ubuntu-11-10-oneiric-ocelot-end-of-life-reached-on-may-9-2013/). Note that page contains a link to guidance on upgrading 11.10 to 12.04.  However, I would recommend doing a clean install of the current 12.04.03 release to avoid potential upgrading hassles. ISO's are here: [http://releases.ubuntu.com/precise/.]("http://releases.ubuntu.com/precise/")

Kubuntu also has the 12.04.03 release.
