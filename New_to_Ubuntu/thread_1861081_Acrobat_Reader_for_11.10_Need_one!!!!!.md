### Post by Meelad on 2011-10-16
Follow this and your problem could be solved..

Open Software Centrum => Click on the Edit menu (in the top bar) => Software sources => Choose to Download from the Main Server and check the first four checks (if any of them is unchecked) => Click on the Other Software tab => Check all four checks and refresh the Software Centrum => Type Adobe Reader in the search box and it should be there for you to install..

Hope this helps..
