### Post by audiomick on 2010-03-17
It could be a problem with the CD drive, or it might be that the CD isn't good.

To be sure of the CD you should
check the md5sum of the iso before you burn it
burn it at the lowest possible speed
let the CD check itself from the "check CD" item in the CD boot menu

here is some info
[https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)
[https://help.ubuntu.com/community/Installation/CDIntegrityCheck](https://help.ubuntu.com/community/Installation/CDIntegrityCheck)
[https://help.ubuntu.com/community/GraphicalInstall](https://help.ubuntu.com/community/GraphicalInstall)

If your CD drive is really broken, look here (this is a link from the page that the first link above points to).
[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)
