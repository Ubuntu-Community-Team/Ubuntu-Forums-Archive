### Post by Hur Dur on 2011-01-21
Damn Small Linux is a dead project. It has been dormant for 3 years. As for installing it to the hard drive, you could probably do it the same way you would install any other Debian/Debian-Based distro. There is an article on the Wiki about installing it to the hard drive [ here](http://www.damnsmalllinux.org/wiki/index.php/Installing_to_the_Hard_Disk).

Alternatives:

Arch Linux (You need 128mb RAM or so for the install, but it will run with IceWM in about 26mb RAM. You can also get GNOME running on 64mb RAM, but speed isn't anything great.)

ConnochaetOS (Arch Linux for i586. It's already pre-configured.)

TinyCore (Very minimalistic distro)

Linux From Scratch (Patience needed, it can take days to set up on older computers)

Gentoo Linux (Patience Needed, and I wouldn't recommend it if you have under 256mb RAM, and a Pentium III processor. Else, it would be extremely slow.)

SliTaz (Tazpkg crashes on anything less than 128mb memory in my experience.)

Slackware (Similar to Arch Linux in speed)

Absolute (A custom Slackware with IceWM, made for older computers)

SalineOS (I have not tried this distro personally, but it uses XFCE, so it should run rather well on 128mb RAM, and an old Pentium 2)

BasicLinux (It's outdated, but it breathes life into old hardware.)

Puppy Linux (I personally only recommend this for Linux newbies. It is pretty limited compared to other small distros. It has small repositories, and is slow on less than 128mb RAM)

Debian Net Install (Can easily run on 64mb RAM with IceWM)

CRUX (Runs on very old computers. I've heard of it being functional with IceWM on a computer with 16mb RAM.)

Non Linux:
FreeBSD (Runs on some of the oldest computers still functioning. i.e. 32mb RAM, 486, Voodoo)

NetBSD (It runs on anything and everything. Even a toaster.)

KolibriOS (Runs snappily on 16mb RAM, and only requires 1 floppy.)

DexOS (In terms of functionality, it is terrible, but it will run like a champ on low spec systems.)

Light Programs:
Text Editor - Nano or Geany
Web Browser - Midori or Dillo
Package Manager - Pacman
File Browser - ROX
Desktop Environment - LXDE, Enlightenment
Window Manager - IceWM, DWM, Fluxbox, JWM
Music/Video Player - Mplayer or VLC
Games - Roguelikes
Image Viewer/Editor - ida or mtpaint

Also check out: [http://kmandla.wordpress.com/](http://kmandla.wordpress.com/)
