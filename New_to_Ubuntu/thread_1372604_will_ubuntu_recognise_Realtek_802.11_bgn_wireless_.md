### Post by Methuselah on 2010-01-04
> **jocelynbennicoff said:**
> how do i get a live cd to check it out?

You can download an .iso file from here:
[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)
Follow the instructions to burn it onto a CD-R.
(Also present at the above link)

When you have the disk you can boot a machine with the CD in the disk. 
Press F12 (usually...look out for helpful text) when starting to get a boot menu and select the CD.
Then choose 'try ubuntu without any change to your computer'.

It will run more slowly then when it is actually intalled but it will give you some idea of hardware compatibility.
I don't know if it will be practical for you to do this with a machine you haven't yet bought but there it is.
