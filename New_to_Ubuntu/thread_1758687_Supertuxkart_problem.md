### Post by Arthur_D on 2011-05-23
Hi, we had a [similar bug report]("http://forum.freegamedev.net/viewtopic.php?f=17&t=1412") over at the SuperTuxKart forums a little while ago.
Turned out in his case to be the OS reporting a bogus gamepad, which interfered with the game since all input devices can be used in single player mode.
You might be able to disable it in 'Options'->'Controls'.
