I... haven't used VMware for a long time, but I don't think it's something OUTSIDE of the host machine, but rather an add-on application you have to install within the hosted OS.  In Virtualbox, when you go to install it's own "Guest Additions" package, the way it does it is by mounting a fake CD-ROM on the host Windows machine and autoruns a setup.exe installation, which replace the default video drivers (for instance) and adds background services and features and stuff.  It probably works in a similar way with VMware...
