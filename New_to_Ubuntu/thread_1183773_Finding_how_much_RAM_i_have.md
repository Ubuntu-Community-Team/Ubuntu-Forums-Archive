### Post by yaroto98 on 2009-06-10
right click on a panel -> add to panel
pick the system monitor
right click on the system monitor on the panel
add RAM memory to the options
now you can see exacly how much RAM is free at any given time, PLUS if you left click on it, it'll give you your computer stats (including how much RAM you have)

### Post by Ms_Angel_D on 2009-06-10
Go System -> Administration -> System monitor and look under the system tab. Under hardware it will tell you how much memory your system has.

Also for a bit more detailed info you can look in synaptic for a program called hardinfo once installed go to System -> Preferences -> System Profiler and Benchmark
