### Post by crip720 on 2019-04-28
You can also just let Ubuntu's software update do it automatically.  Can be set daily or weekly,  It is already setup and will update the OS and all programs as needed.  It will notify and ask you  to install updates, just let it do it's thing and in a minute or two you have an updated system and can keep on working.  Will ask to reboot system once in a while, but that only takes a couple of minutes before you are back working.
