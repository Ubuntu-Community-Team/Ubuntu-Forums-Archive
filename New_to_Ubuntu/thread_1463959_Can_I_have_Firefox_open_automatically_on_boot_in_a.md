### Post by dv3500ea on 2010-04-27
System -> Preferences -> Startup Applications
click add
set name and command to 'firefox'

### Post by WinterRain on 2010-04-27
Go to system>preferences>startup apps and click the add button to add firefox. The firefox executable is in /usr/bin
