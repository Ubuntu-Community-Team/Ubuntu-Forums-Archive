### Post by unutbu on 2008-08-07
Edit>Current Profile
There is a bug in gnome-terminal (at least in my version) -- you may have to select Edit>Current Profile twice before the dialog window shows up.

Under the 'General' tab, untoggle 'Show menubar by default in new terminals'.
