### Post by MG&amp;TL on 2012-08-22
Yes, you can. Just let the installer resize the partition to the size you want, then install. When you reboot, you'll be greeted with a menu to use one system or the other. BACKUP YOUR STUFF before installation. If you make a mistake, you can reinstall an operating system over the mess, but files tend to be harder to recover.

To answer your second question, no, Ubuntu comes with more than enough software for most people. It has:

-office suite

-file manager

-web browser (firefox, by default)

-loads of other utility software, like backup.

However, we can't ship with DVD or Flash support for legal reasons, so you may want to install these after installing the base system. Feel free to ask for help on these forums if you need assistance with that.

For any extra software, you can install a myriad of other software from the software centre, anything from games to image editing, and 99% free of charge.
