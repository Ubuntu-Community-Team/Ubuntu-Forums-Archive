### Post by oldfred on 2014-01-21
Frankly if data is that valuable, you should be backing up anyway. Hard drives fail at any time, users make mistakes and erase data and many other issues. 
And to recover data you have to another drive of large enough size to copy recovered data to. And often better to have an image of corrupted drive or a third drive, so you only work on image.

There is no way to restore working Windows system as any part of NTFS partition will prevent it from working. You may if lucky be able to recover some files. I have uses photorec and it takes many hours to recover files only by extension. And then you have to take many days to sort thru files & recover file names.
