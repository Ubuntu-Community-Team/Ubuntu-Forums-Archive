### Post by 3rdalbum on 2009-01-31
If the computer is connected to the router via an Ethernet cable, then it will work out-of-the-box. If it's connected via a USB cable, get an Ethernet cable. If it's connected via Wifi, then that depends on what chipset your wifi card uses.
