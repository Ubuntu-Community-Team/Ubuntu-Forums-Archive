### Post by halitech on 2009-12-12
If windows is getting that bad on you, you may want to consider backing everything up that you want to keep. Reason I'm suggesting this is because your Ubuntu install is based on a filesystem inside a file on the windows hard drive. So, in other words, if something drastic happens to windows, say good bye to your Ubuntu install as well.

If you are wanting to go strictly Ubuntu, backup everything and format and reinstall ubuntu to the entire drive.
