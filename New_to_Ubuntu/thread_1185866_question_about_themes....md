### Post by Viva on 2009-06-12
What theme did you download? Theming in ubuntu is different to windows. Usually, you go to System->Preferences->Appearance and install the tar.gz file from there. Here is a simple reference for theming in gnome

GTK themes - Controls
Metacity/Emerald - Window Frame(Titlebar)
GDM - Login Window(System->Adminstration->Login Window-Local)
Icon themes - Icons

The process of installing themes is same for all except for the emerald and GDM themes.

### Post by 3rdalbum on 2009-06-13
Just drag the .tar.gz file to the Appearance window when you have the themes displayed. If it's not a complete theme, you can press the "Customise" button and you will find the theme element (controls, window borders, icons) there that you can apply.
