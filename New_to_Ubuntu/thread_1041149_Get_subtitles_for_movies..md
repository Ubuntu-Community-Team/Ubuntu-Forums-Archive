How to display subtitles

To show subtitles on top of your movie you have two choices: 1. Use a media player that supports subtitles, or 2. install a subtitle engine.

1. The quick and easy solution is to download the media player: VLC Player.

2. If you want to continue using your current media player, we suggest that you download DirectVobSub - it will also display special characters (including arabic) if you have trouble with those. It's a program that runs in the background. In order to display subtitles you simply rename the IDX, SUB, SRT, etc. file to the same as your movie file name (keep the extention name (eg. .SUB)) and place it in the same directory your movie resides.

Just rename downloaded subtitle file into same name as the .avi file, place them in the same folder, this way when you start the .avi subtitle will automatically load in VLC player. Use apt-get install subtitleeditor to download (guess what) a subtitleeditor and sync your subtitles to .avi if needed, thts all.
