### Post by nicedude on 2008-08-28
It is not hard but you must be careful and read up on what you need to do before just doing it. One thing I would tell you is that with newer laptops they usually no longer give you system restore CD's and instead use a part of your hard disk to store the restore data.  So you don't want to delete the system restore partition and instead you will need to just resize the Vista partition with the Ubuntu live CD and then reboot and install Ubuntu and it is pretty easy to get all that done and working ( I have done it 10 times at least on several PC's ).  So just google "Vista Ubuntu dualboot  setup" and read one of the many guides before you do this so you will have a good understanding of what you are doing.

Hope that helps and welcome to the Ubuntu family :-)

### Post by RolandFlagg on 2008-08-28
Hi, since you say that you are a complete beginner for ubuntu, I suggest you install with wubi first, this way you wouldn't have to worry about partitioning and losing all of your data.
You can read more about it here:
[http://wubi-installer.org/](http://wubi-installer.org/)
If you already have the latest live-cd, you can just pop it in under windows, and then there should be a choice that says "install ubuntu inside windows" choose that option and you are all set. Wubi is a much easier process than a full install.
This way, if down the line you want to delete ubuntu, you'll be able to use the "add and remove programs" in windows and get rid of it, dont have to worry about grub and stuff like that.
Good luck!
