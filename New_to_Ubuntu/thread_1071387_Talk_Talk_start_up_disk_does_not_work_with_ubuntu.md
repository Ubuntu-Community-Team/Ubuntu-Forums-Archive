### Post by Zill on 2009-02-16
George:  The TalkTalk CD is designed to work with Microsoft Windows, not Linux.  Although *some* MS apps can run on Linux via [Wine]("http://www.winehq.org/"), it is generally better to avoid this if you can. ;-)

The best way is simply to configure your router manually.

If you have a HG520b router supplied by TalkTalk then use the setup instructions supplied here:
[http://www.talktalk.co.uk/help/questions/TEC_02_25]("http://www.talktalk.co.uk/help/questions/TEC_02_25")

If you are using a different router then the information in the above link should still help you to configure your router correctly.
