### Post by tekkidd on 2011-06-28
They have IEs4Linux, however this is primarily for IE 4,5, and 6. IE 7 and 8 run in Wine but not very well. I would suggest that if you really need IE and want it to run without problems, use a virtual machine. 

Links: 
IEs4Linux: [http://www.tatanka.com.br/ies4linux/page/Main_Page](http://www.tatanka.com.br/ies4linux/page/Main_Page)

Internet Explorer Wine Page (All versions): [http://appdb.winehq.org/objectManager.php?sClass=application&iId=25](http://appdb.winehq.org/objectManager.php?sClass=application&iId=25)

VirtualBox (Virtualization): [http://www.virtualbox.org/](http://www.virtualbox.org/)
