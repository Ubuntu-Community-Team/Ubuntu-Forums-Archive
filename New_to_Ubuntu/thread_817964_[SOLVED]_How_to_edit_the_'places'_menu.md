### Post by wormser on 2008-06-04
Open up Nautilus> Bookmarks> use Add or Edit.  They will show up under places.

### Post by billgoldberg on 2008-06-04
This would be better asked in the pclos forums for obvious reasons.

Does pclos use nautilus?

If it does, open up nautilus (the filemanager) 

On the left side you should see the folders available under the "places" menu.

If you drag a folder in there, it will be listed under the "places" menu.

Well in ubuntu, it should work for pclos.
