### Post by Seq on 2011-06-09
> **Paqman said:**

> **fabricator4 said:**
> If by that you mean is Gnome 2 (aka "classic") going to continue to be part of Ubuntu, the answer is 'no'.  The Gnome team will now support and develop Gnome 3.

The "classic" gnome-panel is still a component of gnome 3. It is called "fallback" mode (versus "classic" in unity nomenclature). It is actually a fully supported component of the gnome 3 desktop. So yes, you will still be able to run the "classic" interface.


> **fabricator4 said:**
> None whatsoever.  Gnome 2 is a dead issue.  It will most likely (almost certainly) not be available as the fall-back GUI in future releases.

GTK dropped a lot of old cruft in version 3. Gnome also decided to drop their old cruft when moving to GTK3 and bumped their major version to 3 as well (to indicate the API changes). This version also has a new interface by default (gnome shell), but it is not your only option since gnome-panel was also ported to gnome3 (and included as part of the suite)

So yes, "gnome 2" is dead. That is not a bad thing. Software gets updated over time.

As for being available in future releases, it may or may not by default. It may need to be installed separately, or there may be a "gubuntu" build that ships gnome instead of unity (assuming unity's fallback in the future will be unity2d instead of gnome2 or gnome3-fallback).
