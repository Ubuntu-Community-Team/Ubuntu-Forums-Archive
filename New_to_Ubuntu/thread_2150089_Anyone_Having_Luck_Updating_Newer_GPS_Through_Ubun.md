### Post by Javelin Dan on 2013-05-30
[FONT=Arial][SIZE=2]Running Ubuntu 12.04 on a Compaq Presario dual core laptop.[/SIZE][/FONT]
 

 [FONT=Arial][SIZE=2]I was wondering if anyone has had success downloading updated maps for [SIZE=2]newer[/SIZE] Garmin, Magellan, or Tom Tom GPS navigators. My wife recently picked up a Garmin &#8220;nuvi 50LM&#8221; that's supposed to include lifetime map updates. All the brands say that they support only Windows or Mac. [/SIZE][/FONT] 

 

 [FONT=Arial][SIZE=2]I did some research and found that with previous models, you could simply use a card reader (with the mini SD card that comes with it), and update it using Ubuntu from a site that had different navigators listed by model number. But that post was a couple of years old and my model wasn't listed. The way it works now is; first you log onto the Garmin site, then you download the &#8220;Garmin Communicator Plug-in&#8221; (this is actually available from the add-ons menu in Firefox), then you plug in the device, wait for it to be recognized, register it with Garmin, and THEN find out if there are any map updates available. If you try to do any of this through Linux, it will tell you your OS isn't supported.[/SIZE][/FONT]
 

 [FONT=Arial][SIZE=2]I spent all my free time Memorial weekend [SIZE=2]installing[/SIZE] WINE and [SIZE=2]then[/SIZE] PlayOnLinux to first try Internet Explorer (8), then ultimately Firefox (Windows version). With Firefox, I could get in the site, get it to recognize that I had the Communicator plug-in installed, then it would tell me it was trying to recognize my device. That's were it would hang endlessly with a spinning wheel...[/SIZE][/FONT]

                        

[FONT=Arial][SIZE=2]I e-mailed Garmin and they told me that the USB charger cable they supplied wouldn't transmit data properly to register the device. I looked at one end and sure enough, it had a USB (flash drive) symbol with an &#8220;X&#8221; through it. WTF? I found another data cable I had that would fit but got the same results. [/SIZE][/FONT] 
  

[FONT=Arial][SIZE=2]Has any one had better luck recently updating either Magellan or Tom Tom with Ubuntu? It seems that if all I was trying to do was click on a download file for a given model instead of trying to communicate between two devices across two different platforms, then maybe the WINE thing would work &#8211; at least with a card reader.[/SIZE][/FONT]
 

 [FONT=Arial][SIZE=2]The sad irony in all this is that as I understand it, these companies use an open source mapping software to set up these things, then they force this proprietary Windows/Mac crap down our throats! GRRRR...[/SIZE][/FONT]
