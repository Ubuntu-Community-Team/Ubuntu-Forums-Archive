### Post by bodhi.zazen on 2009-12-16
You can run VirtualBox without X.

In anything resembling modern hardware you are not going to see a huge boost in performance mind you simply by changing window managers. Better off using light weight applications and disabling servers you do not use.

See : [http://xwinman.org/](http://xwinman.org/)

[http://www.linux.com/archive/articles/37552](http://www.linux.com/archive/articles/37552)

If you wish to see a lighter weight install try Crunchbang (Openbox) or Zenix (XFCE and Fluxbox)
