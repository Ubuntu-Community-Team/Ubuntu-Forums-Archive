### Post by coffeecat on 2010-03-23
Set the BIOS to boot from the CD. Then boot up from the CD and explore  Ubuntu in the live environment. This will be much slower than a proper  hard drive install, by the way. If you want to install Ubuntu, you simply double-click the  installer icon on the desktop. You can choose to erase Windows and use  the whole disc, or shrink the Windows partition and set up a  side-by-side Windows/Ubuntu dual-boot with Ubuntu on its own partition.

This is much to be preferred than using Wubi which installs Ubuntu  "inside" Windows.
