### Post by HereInOz on 2008-12-28
Tend to agree.  You are probably going to need at least 128MB RAM for Xubuntu.  The 600 processor will do OK, but the RAM will be a little too little.

Anyway, to your original question, it is probable that with a computer of that age, there is no function to boot from a USB flash memory stick, so that is out of the question. Can you not open the case and connect a CD drive to the existing IDE connection and run the install that way?

Could be your only option.
