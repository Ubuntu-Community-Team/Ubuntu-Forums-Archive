### Post by ibutho on 2008-07-21
You should be able to use the Firefox plugins to stream from Opera. Run Opera -> Tools -> Preferences -> Advanced -> Content -> Plug-in Options -> Change Path and make sure you add the directory that contains the Firefox plugins and then click on Find New.
