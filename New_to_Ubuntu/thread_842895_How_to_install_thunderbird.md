### Post by avtolle on 2008-06-27
You are running from CD, so I infer you have not installed Ubuntu yet. Is this correct? If so, installing Thunderbird (whether from the repositories, which is recommended, or how you are trying to do it) will install Thunderbird into RAM, as that is where Ubuntu is running. So, once you restart your computer, Thunderbird will not be there, and you will need to do it again (and again and again) until such time as you permanently install Ubuntu.

### Post by avtolle on 2008-06-27
Under System>>Administration>>Synaptic Package Manager. Use the search function, type in thunderbird, and after it searches, it should come up (it may be listed as mozilla-thunderbird, I cannot recall). Mark it for installation, click on Apply, and that should do it. Hope this helps.
