### Post by the-saint on 2009-05-09
I recently received the Jaunty Jackalope CD from Canonical and promptly tried to load it in my Sony Vaio Laptop which had Pre loaded Vista in it. In the process I screwed up my MBR and all I could see was Error 21. My laptop was neither booting Vista nor Ubuntu. 

It took two days for me to be back to normal 
The options i tried. 

1. Initially tried to run the recover C Drive. But this too did not help then I tried a Full Recovery. Nothing changed and I again was seeing the Error 21 in my Black screen. 

2. Then I downloaded the Vista Recovery ISO and wrote it on to a CD . Upon trying that although it was able to do some operations but again it failed to boot. Things were really going creepy and Felt like smashing the Notebook. 

Finally I came across a post that recommended using the Super Grub Disk 
3. I Downloaded the latest SGD and wrote it onto a CD. Hola... Things were getting better now.. I was able to load the grub and then running the Recovery option now set things right... 

Glad to be back from the nightmare... Still scared to install Ubuntu now..
