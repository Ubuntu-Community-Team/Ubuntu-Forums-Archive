### Post by swarup on 2010-10-23
I am installing Sun Java (sun-java6-jre), using the terminal window. The installation is all but complete. It downloaded everything, and has now reached a window where I have to "click" ok to accept the license to complete the installation. But obviously, in a terminal window one cannot "click" on the place where it says <ok>. I tried hitting "enter", also "tab", also "down arrow". But nothing worked. How to I indicate to the terminal window that I accept the licese agreement?

---

### Post by swarup on 2010-10-23
I figured out how to do it, thanks anyway! One just has to use the right or left arrow to make the <ok> turn red. And THEN one can hit enter, and the license agreement gets accepted. And the installation gets completed.
