### Post by dstew on 2008-05-17
dpkg-reconfigure xserver-xorg is not the best way to reconfigure your screen in Hardy. If you don't have a useable desktop, at the boot menu choose Recovery Mode, and choose xfix. That will get you a vesa-type desktop. On the desktop, check in System --> Administration --> Hardware Drivers for any restricted drivers you might need to install. Install and reboot.

If the display needs more configuring, press Alt-F2 to open a Run dialog box, and enter```
gksudo displayconfig-gtk
```You can now edit the display properties, including picking your monitor type by brand name. If you find your monitor there, it will give you all the appropriate resolution choices.
