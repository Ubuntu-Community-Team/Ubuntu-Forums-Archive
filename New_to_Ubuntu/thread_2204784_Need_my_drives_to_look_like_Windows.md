You can find them by pressing the Super (Windows) key + A. That will reveal icons for all the applications. If we press and hold the Super key an overlay will appear showing the keyboard shortcuts.

(It doesn't install though when you do an installation, like most other programs on the livecd do)

So, I would look and see if that is similar.

As for making them seen by both windows and linux, you'll need to set them as ntfs.

I don't know if you mean you want 4 partitions total, or 4 partitions from the unallocated space.
