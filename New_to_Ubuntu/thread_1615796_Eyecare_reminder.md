### Post by pme 72 on 2010-11-08
Also there is a nice and simple typing break reminder program natively installed in Lucid, 10.04. Go to System/Preferences/Keyboard. The last tab in Keyboard (Preferences) is Typing Break. You can set the work interval, break interval and whether or not to allow postponing of breaks. It has been working well for me since I enabled it a month or so ago.
