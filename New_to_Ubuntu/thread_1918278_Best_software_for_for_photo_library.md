These are OK if you only want to view your pictures. However, I have been using ThumbsPlus for virtually all my photo needs. It is simple and you can directly control various aspects of the photo, e.g., contrast, hue, gamma, etc. It is written for Windows but runs under Wine quite well. I think you might have to download a particular .dll for free--got to the Wine web page. Now the bad news is that you have to pay for it but I think it is well worth the money, I have yet to see such a simple and straightforward program in Open Source.
