### Post by Lisiano on 2012-05-12
It shows both actually.
[IMG]http://ubuntuone.com/6mXDjXppL5QF9RBkykiahi[/IMG]
Click the clock -> Time & Date Settings -> Clock -> Tick whatever you want
Or go into System Settings -> Time & Date -> Clock -> Tick whatever you want
Or open a terminal (Ctrl+Alt+T) and type in this command ```
gnome-control-center indicator-datetime
```
