### Post by ajgreeny on 2010-04-01
It depends on the source of the video and stream type.  It also will depend on the legality of the download, but generally it is worth looking in /tmp to see if there is a file of appropriate or random name which you can copy to /home before closing the streaming web page.  Not everything works like that, but worth a look.
