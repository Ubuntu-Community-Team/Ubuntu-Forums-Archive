### Post by Kirboosy on 2014-02-25
Hi Marcoman! Welcome to the forums!

I found out that there are some extra font settings not easily accessible from the settings menu. If you copy and paste **chrome://settings/fonts** in your URL bar you can access them. What is the value set for "Encoding"? Perhaps changing it to UTF-8 (and restart the browser) will help. You could also change all the font settings to another font if you wish.

Hope that helps!
