### Post by diablo75 on 2010-06-06
I couldn't find any entries about that particular Warcraft game, but one thing you might try is opening Ubuntu's System Monitor (equivalent to Windows Task Manager), view the processes running, and see if there is anything with the name wine in it running (sleeping?) in the background.  Then select it from the list of processes and kill it.  Then run Warcraft again.  This might do the trick for you, as it should "reset" wine.  If it does work you could even make a custom shortcut/launcher to put on your desktop/taskbar to use for killing Wine with a click in the future.
