### Post by Nepherte on 2008-09-28
This is how mine always looks like:
AuthUserFile /full/path/to/.htpasswd
AuthGroupFile /dev/null
AuthName "Password Protected Area"
AuthType Basic

<limit GET POST>
require valid-user
</limit>

Be sure to test it with a password without any special characters for now, I've had some problems with those.

### Post by tonybaqain on 2008-09-28
please add the following lines to your vhost .conf file :
```

<Directory "YourDirectory">
           AllowOverride AuthConfig
</Directory>

```

[*] note that **YourDirectory** is the directory containing the .htaccess file.

restart your apache.

and have fun :)
