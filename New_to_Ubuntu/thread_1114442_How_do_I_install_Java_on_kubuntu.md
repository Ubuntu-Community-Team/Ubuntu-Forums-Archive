### Post by taurus on 2009-04-02
Open a terminal and run

```
sudo apt-get update
sudo apt-get install sun-java6-bin sun-java6-jre sun-java6-plugin
```
If you get to a java licensing screen, press the Tab key to highlight the <OK> and Return to accept it.
