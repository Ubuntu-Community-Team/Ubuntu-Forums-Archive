### Post by coolbrook on 2009-03-23
See if you can get into safe mode to salvage your bookmarks.

[http://support.mozilla.com/en-US/kb/Safe+Mode](http://support.mozilla.com/en-US/kb/Safe+Mode)

If you can get in then export your bookmarks to a place where you'll know where to find them.

Consider creating a new default profile to see if you can get it going again and import your bookmarks.

[http://support.mozilla.com/en-US/kb/Managing+profiles](http://support.mozilla.com/en-US/kb/Managing+profiles)
