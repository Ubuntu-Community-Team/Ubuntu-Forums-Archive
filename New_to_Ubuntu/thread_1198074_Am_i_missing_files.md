### Post by freebeer on 2009-06-27
It sounds like what you have is the downloadable .ISO file, right?  If so, you need to burn that ISO file to a CD (or DVD, depending on which version you have).  You need to select "burn as image" (or similar).  From that you'll produce a bootable CD (or DVD) that can install the OS.

---

### Post by synapsys on 2009-06-27
Also remember to burn it slow, 4x or less, for best results. You could also run a checksum against the iso to make sure your download isn't corrupted.
