### Post by tamoneya on 2008-04-24
with linux the best way that I have found is to just try to play around with it.  Try installing an apache server.  Try setting up a dual boot. Compile a program from scratch.  This will allow you to learn a lot.  If you manage to break your system while experimenting it will provide you another chance to learn when you fix your system.  
Look around the forums and you should find many programs to try and install and play with.
