### Post by spartan_87 on 2010-03-31
Im not too familiar with Mint, but if it works like Ubuntu you should be able to do this:

Right click the desktop -> Create Launcher.
Type: Application.
Name: whatever you want, for example the name of the website.
Command: firefox [www.website.com](www.website.com) (replace "website" with the website you want it to go to.)
Then hit ok. That should do it.

### Post by bug67 on 2010-03-31
I am replying to this post from Linux Mint.

1.  Navigate to the web site that you want to create a shortcut to.

2.  Click and hold on the favicon in the address bar and drag it to your desktop.

Done.  :)
