### Post by gn2 on 2008-12-08
This is why the Live CD was created.

Place the Ubuntu CD in the drive, switch off then restart and it should boot from the CD, take the top option on the menu and you'll quickly find out if Ubuntu will run on your machine without any need for installing anything.

When you switch off nothing will be affected, it will be as if it never ran Ubuntu.

### Post by Bölvaður on 2008-12-08
oh the dreaded laptop. :mad:

Most drivers are included, so burning the live cd, changing the boot order of the computer in BIOS and restarting the computer with the cd in should give you pretty clear picture of how it is going to be.

The only drivers that are not included are some wireless cards and some graphical cards.

You should also use google to see how well your hardware is supported. imo intel wireless are sh111.
