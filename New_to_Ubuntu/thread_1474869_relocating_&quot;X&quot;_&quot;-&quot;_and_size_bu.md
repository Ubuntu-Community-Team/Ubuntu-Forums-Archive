### Post by flyfishingphil on 2010-05-06
FYI I've seen several requests for info on how to "relocate" the buttons in the 10.04 version and some have had some really long processes. In cruising around what is available I found an easy way to do the same thing without having to go thru the entire Applications>Terminal> enter a bunch of codes, etc.

If you want to put the buttons back on the right side, like we are all used to, simply go to System > Preferences > Appearances > Theme and select a theme that shows them on the right side.

Much easier for those of us that aren't "full speed" on making the program changes thru the Terminal.:rolleyes:
