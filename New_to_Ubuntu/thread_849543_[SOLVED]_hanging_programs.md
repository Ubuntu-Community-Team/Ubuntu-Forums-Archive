### Post by gn2 on 2008-07-04
Right-click in a panel (taskbar) select "Add to Panel" and select the "Force Quit" option.

You now have a handy little icon you can click on which lets you force applications to quit.

### Post by ibuclaw on 2008-07-04
If the program window goes gray, click the the usual close button and a **Force Quit** message will popup.

If your Desktop has frozen. press "**Alt+Ctrl+Backspace**" to restart X and return to tle login screen.

If that fails, use REISUB to safely reboot.
**Alt+PrintScreen+R**
And while still holding down "Alt+Printscreen", press in the following order
**E I S U B**
Make sure you allow your computer a brief moment to process the commands.

### Post by dizee on 2008-07-04
There is a cool little program called xkill that will close a GUI program that's not responding.

Pres alt+F2 and type **xkill** and the cursor turns into a skull and crossbones. Click on the program you want to close.
