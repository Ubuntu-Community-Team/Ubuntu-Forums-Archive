### Post by uRock on 2014-08-29
Hello and welcome to the forums.

In my opinion, [VLC]("http://www.videolan.org/vlc/features.html") is the best. You will need to install an application called libdvdcss2 in order to play some DVDs.

Both are available in the Ubuntu Software Center or can be installed via the command line by running the following command in a terminal.

```
sudo apt-get install vlc libdvdcss2
```
