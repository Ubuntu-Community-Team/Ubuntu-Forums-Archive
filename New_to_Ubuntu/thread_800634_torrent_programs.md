### Post by Paqman on 2008-05-20
The default client on Ubuntu is Transmission, which is so simple to use that it shouldn't be a problem. Azureus is available for both both Linux and Windows, but has a fairly bewildering array of features.

There is a Windows version of Deluge available, but it's noted by it's developers as being less stable than the Linux version.
