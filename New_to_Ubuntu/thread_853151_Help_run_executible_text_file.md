### Post by appi2012 on 2008-07-08
What did you do to install it?

if it's a bin file, you usually open a terminal, type:
```
cd [the folder it's in]
```
```
chmod +x [Name of Bin file].bin
```
```
sudo ./[Bin File].bin
```
