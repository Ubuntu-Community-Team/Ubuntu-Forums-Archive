### Post by Penguin Guy on 2009-04-16
This is one way in which Linux works differently to Windows. Windows copies your background picture and uses it, Linux just uses it. Therefore, you can't delete background images or they will not be able to be used any more. I would advise creating a 'Backgrounds' folder in your home directory.

There is a folder that is used globally that would be better but requires root privileges to access; **if you know what you're doing** then the official backgrounds folder can be found at: /usr/share/backgrounds
