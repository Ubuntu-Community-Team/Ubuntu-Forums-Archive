### Post by I2k4 on 2012-02-17
There is a famous problem with Atheros wi-fi cards on Acer netbooks, where the "power saving" setting can affect performance - I can't say I know anything about fixing it on Ubuntu, but suggest completely disabling it on Windows using the Device Manager.  Having done that, I don't experience any difference between internet performance on Linux and Windows.
