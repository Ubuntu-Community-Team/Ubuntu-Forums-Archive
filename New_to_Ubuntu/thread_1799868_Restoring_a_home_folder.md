### Post by nomko on 2011-07-08
Did you take a look in your trashbin? If it is your trashbin you can drag and drop the folder in your /home/{username}/ folder. Else you can create the folder by rigth clicking in nautilus and select new folder. It is not a system folder, it's just a folder which is created during installation to make some things easier for the user. It's a standard folder in your /home directory just like the download and video folder.

### Post by EVILCOOKIE on 2011-07-08
Just create another folder called Music, right-click it, select properties and click the icon at the prompt. I found a "music folder icon" at /usr/share/icons/gnome/256x256/places/folder-music.png.
