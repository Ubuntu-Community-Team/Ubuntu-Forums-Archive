### Post by Jazzy_Jeff on 2010-07-14
> **Pappy1911 said:**
> I'm thinking of buying a new laptop which does not have a CD/DVD drive.

What I want to know, can a memory stick be used instead of a CD/DVD disk to install a Linux distribution???

Thank you.

Yes it sure can. There are a couple of ways to do it. If you are using Ubuntu right now you can click on System > Administration > Startup Disk Creator to make it. Just download the image file you want to use and tell the disk creator where it is. You can also use a program called UNetbootin to make it as well.

Just remember to go into the Bios on the new laptop once you plug in the USB drive and tell it to boot from it.
