Is it possible to retrieve the wireless password that is already saved in Ubuntu 9.04?

### Post by Paddy Landau on 2009-10-02
Applications > Accessories > Passwords and Encryption Keys > Passwords.
Right-click the item and select Properties > Key > Password > Show password

### Post by oldmankit on 2010-10-04
A bit late, but there's an easy way.

Main menu -> search -> passwords and encryption

Find the network, properties, expand password, then 'show password'.
