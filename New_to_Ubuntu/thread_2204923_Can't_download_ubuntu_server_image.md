### Post by sudodus on 2014-02-11
Welcome to the Ubuntu Forums :-)

It might be a temporary error, and downloading might work again next time you try unless there is something wrong at your end of the connection. Remember to check that the download was good with md5sum according to the following link

[https://help.ubuntu.com/community/UbuntuHashes](https://help.ubuntu.com/community/UbuntuHashes)

If the error persists, you can try another version of Ubuntu Server. I'd recommend version 12.04 LTS (long time support until April 2017) or the newest version 13.10 with support only until July 2014, when you need to upgrade to the next version 14.04 LTS.

Or you can try with the Ubuntu ***mini.iso*** 12.04 LTS and use that to install Ubuntu Server.
