### Post by hesth on 2008-06-01
For anyone else who had this problem...

The WINE version in the synaptic is not the most up to date - and I only installed yesterday.

If you uninstall WINE in synaptic, then go to this page [http://www.winehq.org/site/download-deb](http://www.winehq.org/site/download-deb) and install the updated version, it works great.

Cheers!

PS I think is is valid for any .exe program
