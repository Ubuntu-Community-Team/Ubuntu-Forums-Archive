### Post by grahammechanical on 2011-09-05
Could you not export it as a PDF document. Any links will still work. You could then put it on a Ubuntu One account (assuming the document is less then 2GB) and publish the file. You get a http address that will let others download or open the document through a web browser.

I am experimenting with this method for a project that I am working on. Be aware that when you update a document on Ubuntu One the http address gets changed. An updated file is in effect a new document when it comes to allocating http addresses.

Regards.
