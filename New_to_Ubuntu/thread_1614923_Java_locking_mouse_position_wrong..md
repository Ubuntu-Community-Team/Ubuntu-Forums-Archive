### Post by outleradam on 2010-11-06
I have a java game which I would like to play in the browser on my multiple monitors.   The problem is when the game tries to lock my mouse cursor, it locks it 1 full screen from where it should be.  

If I put firefox on my left monitor, it centers the cursor locked into the middle of the right monitor.   If I put firefox on my right monitor, I can see the cursor moving on the far right side of the screen.


How can I correct this misplaced, locked cursor?  It is causing me to spin around and around in the game.  What is the fix for locking the cursor on multiple monitors?

---

### Post by outleradam on 2010-11-06
got it figured out.   Apparently java expects the primary monitor to be on the left.  Mine is on the right.  The fix was in changing my primary monitor.
