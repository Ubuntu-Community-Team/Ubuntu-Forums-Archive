### Post by huntwell on 2008-04-27
Universal,

I have personally found Envy to be far better than using any tutorial out there to get ATI drivers running good, and I have tried many. I recommend you use envy. I have installed both hardy and mint daryna and it works well on both. 

I have found other threads that explicitly instruct you to remove flgrx and xorg in order for envy to work properly. I have not had this issue since I went straight to envy after fresh install of hardy. If you need help with what to uninstall before installing envy, there are threads out there for that. I think the web site I used to download envy also has a tutorial on that as well, but I haven't read it for a while.

If you have Hardy, you want to install EnvyNG, if you have Gutsy, you will want to install Envy Legacy. I will provide a direct link to Alberto Milone's web site where you can download it directly.

Let me know how it turns out.
