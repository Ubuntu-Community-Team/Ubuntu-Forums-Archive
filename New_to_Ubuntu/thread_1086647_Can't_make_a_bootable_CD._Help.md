### Post by Elfy on 2009-03-04
If you are using infrarecorder - then it should burn it inthe correct manner. See this page - from actions - Burn as Image

[https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)

should be of help.

IF you haven't done so - check the md5sum before you burn it
[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

Then once it has burnt make sure that cd/dvd is set to boot first in BIOS unless you have a new enough pc/laptop to have a boot menu when you start.
