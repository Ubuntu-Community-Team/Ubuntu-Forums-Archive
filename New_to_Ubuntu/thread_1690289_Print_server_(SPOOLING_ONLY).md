### Post by Frogs Hair on 2011-02-20
It is possible to disable spooling in Win XP or Ubuntu . There is a catch , spooling allows the print job stored in a buffer until the device is ready .

I don't know if a print job initiated in Win XP would use the the Ubuntu spooler (for lack of a better way to phase it) Here are some links from my search.
