### Post by doriard on 2009-06-28
It appears there is currently no ALSA (Unix sound) driver for the Striker, but there is for the HT Omega Claro.  See the description here:
[http://ubuntuforums.org/showthread.php?t=205449&highlight=firefox+sound]("http://ubuntuforums.org/showthread.php?t=205449&highlight=firefox+sound")
The listing of available HT Omega ALSA drivers is here:
[http://www.alsa-project.org/main/index.php/Matrix:Vendor-HT_Omega]("http://www.alsa-project.org/main/index.php/Matrix:Vendor-HT_Omega")

Hopefully there will be an ALSA driver before too long.  It wouldn't hurt to request one from HT Omega company.
