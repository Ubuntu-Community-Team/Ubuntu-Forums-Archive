### Post by bumanie on 2008-10-30
Check that bluetooth is enabled under start up programs in sessions (System --> Preferences --> Sessions) it should be by default, but worth a check. Buetooth should be recognized upon plugging in a dongle. Look under System --> Preferences --> Bluetooth. If it is installed it should be in the list. I don't know xfce very well, but that is how it is on gnome.
