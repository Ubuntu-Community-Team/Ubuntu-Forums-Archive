### Post by lazyart on 2009-02-11
Launch "Configure Wine".

Press:

Shift-Tab
Right Arrow
Right Arrow
Tab
Tab
Tab
Space
Enter

That should disable the virtual deskop.  Now reopen the config program and change the desktop size.
