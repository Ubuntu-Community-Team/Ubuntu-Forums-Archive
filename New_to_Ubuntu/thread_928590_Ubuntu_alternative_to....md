Frostwire also uses the Gnutella network.

So you'll have as much (even more, since it's a free limewire pro version, not plain old normal limewire) results.

You can download a .deb from frostwire.com to install it.
--

An alternative could be "nicotine +".

It's a p2p client for the Soulseek network. It's great for the more rare music.

--

Or you could just use torrents to get your music.

--

There is also a program called "pytube media converter" that will allow you to download videos from youtube and save it as .mp3 .

Since youtube is fast and has almost all music known to Man, it's a great tool to get your music.

Pytube isn't in the Ubuntu repo's, but there are .deb packages available, so installation is easy.

--

Note: if you want to use the gnutella network but don't like the limewire/frostwire java based GUI, you can use "gtk-gnutella".

It's a Gtk based app, so it will use your gtk themes and icons. 

It's in the repo's, so download and install using add/remove, synaptic, apt-get or aptitude.
