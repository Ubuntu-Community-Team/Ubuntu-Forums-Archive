My suggestion, and this is purely out of experience, is to run disk cleanup with XP. Theres a second page on that app that not too many people know about. If you look there, theres an option to remove all but your last system restore file. On my parents old XP computer, it freed up nearly 12GB!

After you do this, run Defrag once or twice. You will be amazed at how much space you can clear up! :D

Ubuntu, at its smallest install is 2GB. You could technically install it on a thumb drive and run it from there!

If you wanted to actually use the system on a day to day basis, you would need around 8-10 gigs. This allows you to install updates and add new programs, both of which im sure you would want to do right away.



I dont spend too much time on Windows 7, but from the experience Ive had so far, its 10000000x better than Vista. ### Post by tom.swartz07 on 2010-01-06
if youre going to dual boot, the very first step is to make sure that you clean up your other OS files.

Think of it in terms of a car. Youre going to get work done on the cabin- new seats, cool new steering wheel, maybe even some flashy new lights or something. You wont take it to the shop with old soda cans and Burger King bags in it, would you?

Same applys here: Go through your Windows XP files, sort things out. Run a Defrag a few times. Delete any humongous files you dont need. Do that trick with the previous system restore files ([http://www.windows-help-central.com/delete-system-restore-files.html](http://www.windows-help-central.com/delete-system-restore-files.html)). THEN you can start to look into getting some new bling for your computer! :D


Even though this is an Ubuntu forum, many of our users here are coming from Windows, so we need to keep a keen eye to protect users like you so that you dont accidentally lose your data. 


Hope this helps!
