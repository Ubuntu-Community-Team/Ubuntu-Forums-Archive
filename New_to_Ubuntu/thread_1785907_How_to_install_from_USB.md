### Post by Jarramundi on 2011-06-18
Go to your BIOS on restart and change the boot order so that the USB is at the top of the list as a boot device. Should just start the installation as normal. :)

Remember its a good idea to allocate a few GB to a separate "/" (home) partition and a GB or so for SWAP, everything else is extremely simple
