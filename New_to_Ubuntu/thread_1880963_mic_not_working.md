### Post by Exonimus on 2011-11-14
Hello and welcome to the forums! :)

I'm not very knowledgeable on microphones, but searching the forums might help, and this topic in particular seems rather informative, it may be of help:
[http://ubuntuforums.org/showthread.php?t=1492643](http://ubuntuforums.org/showthread.php?t=1492643)

The first and most important thing to check is if the mic is set on mute. If you go to system->preference-> sound, then click 'input' and see if mute is unchecked. (I'm running a somewhat older version of ubuntu, so you have to forgive me for not knowing the exact location of this item on the new interface yet). You might also try opening up a command line interface, typing 'alsamixer', you'll get a graphical display with the volume bars of all sound devices. If you scroll to the right (there are usually more options to the right of the screen) you might need to see what the mic's volume is set on. If it's set on 0, that's obviously wrong :P

If that doesn't solve your problem, check out the aforementioned topic and see if it might be of help.
