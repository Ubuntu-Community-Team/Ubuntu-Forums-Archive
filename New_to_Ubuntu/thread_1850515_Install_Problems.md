### Post by lightwarrior on 2011-09-26
Usually Windows RAID drivers are supplied during win install (at least in my days).  You had to press F6 or something like that while all other drivers were loaded during Setup.

New motherboards usually have an IDE/RAID/AHCI SATA configuration menu.  Perhaps you only need to select RAID and follow your motherboard instructions/drivers.
