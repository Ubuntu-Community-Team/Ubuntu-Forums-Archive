### Post by mameshiba on 2010-12-20
I've brought my laptop to my family home for a week over the festive period. They have a wireless router, and I was hoping to be able to connect to the internet using this while I was here.

In the past, the laptop has usually scanned for network connections every time I booted up. I usually use a wired ethernet connection (which is what I'm currently using), but until this weekend it would usually find whatever wireless network there was and connect to that, disconnecting to it later after it had established both connections.

However, it had stopped doing this since I powered my laptop on (to read some comics, heh) on the train coming down. When I did this, it searched for connections as usual, found a few I didn't have passwords for, and then a message box came up saying 'You are now disconnected from the Internet'.

Since then, it doesn't search for wireless connections on booting up any more and I can't work out how to make it search for one.

Any suggestions? I can keep using the wired network, but I'd prefer not being tied down to one place, and to be able to use wi-fi in other places, too... ;)

I'm using Lucid Lynx on a Stylenote laptop with a Stylenote 10/100 LAN & Wireless 802.11G network card.

### Post by iosuna86 on 2010-12-20
Make sure that when you right-click on the network manager applet, the "Enable Networking" option is checked. 

It has happened to me before that that option has been unchecked by itself.
