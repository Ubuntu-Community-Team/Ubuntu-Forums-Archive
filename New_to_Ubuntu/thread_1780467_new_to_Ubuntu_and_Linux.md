### Post by Gone fishing on 2011-06-12
You may wish to install all the codecs etc. so that you can play mp3 watch movies etc. Open a terminal and run the following

```
sudo apt-get install ubuntu-restricted-extras
```

You can also do this graphically using synaptic or ubuntu software centre
