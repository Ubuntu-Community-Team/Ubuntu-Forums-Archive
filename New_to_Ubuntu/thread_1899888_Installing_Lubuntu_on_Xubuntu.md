### Post by lykwydchykyn on 2011-12-24
You just need to install the "lubuntu-desktop" package and its dependencies.

Then log out and change the "session" to LXDE.

### Post by alenis on 2011-12-24
Lubuntu works well on my laptop but turned out to be quite unstable on my Desktop computer. If you experience problems with it, especially with pcmanfm, you can try the Debian implementation of XFCE, which seems to be faster and lighter than the Ubuntu one, and LXDE, which has an older version of pcmanfm, less rich in features but definitely working.
