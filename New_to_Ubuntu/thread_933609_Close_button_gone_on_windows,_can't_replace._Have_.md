### Post by phidia on 2008-09-29
I don't know exactly how you set up the OS X theme or what files you edited.
You could try a couple different things: 1) System > Preferences > Sessions > Current Session tab you may be able to delete something there related to the theme you installed. 
2) Create a test user > sudo adduser test login to the test account and see if that works normally. If it does you can replace your hidden files (dot files-configuration files) one at a time-from your test account to your regular user and hopefully that will work.
