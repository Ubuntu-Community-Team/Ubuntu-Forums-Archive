### Post by spiderbatdad on 2008-12-02
The most noticeable difference to the average user will be the difference in desktop environment. Ubuntu uses Gnome, and Xubuntu uses Xfce. The Gnome DE is feature rich, but also resource intensive. If you love all the bells and whistles of the latest in desktop environments, you will miss Gnome. If you want only a nice functional desktop, due to a heavier reliance on command line tools, X might be more appealing and more portable.
