### Post by Idefix82 on 2008-09-12
In that case you can just open the file browser, select the folder that you want to show up in the places menu and press Ctrl+D. That adds the folder as a bookmark which will then show up among other bookmarks in the places menu. Alternatively, select the folder and go to Bookmarks->Add Bookmark.
I hope, that helps.
