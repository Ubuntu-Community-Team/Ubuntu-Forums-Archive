Hi and welcome, have you tried the nvidia-settings? Use the alt, F2 keys and enter the command ```
gksu nvidia-settings
``` and adjust your resolution there. If they are not installed then you can install via synaptic manager. You may also try and use the command by the same method 
```
gksu displayconfig-gtk
``` and set your resolution there.
