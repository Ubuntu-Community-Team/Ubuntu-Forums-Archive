### Post by ugm6hr on 2008-11-10
> **muted1987 said:**
> is there a way to install ubuntu over the internet or copy the disc to hdd so it boots and can install from there.

You can copy the CD to your HD and boot from that, but:
1. Copying a corrupted CD will not work any better than booting from a corrupted CD.
2. How can you copy the CD to HD without a bootable OS?

Net-install for Ubuntu requires a local network server (i.e. another computer).

Your options:
1. Get a ShipIt Ubuntu CD to install.
2. Get someone else to download and burn a new CD.
