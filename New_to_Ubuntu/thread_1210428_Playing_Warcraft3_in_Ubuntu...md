### Post by Troll_the_Great on 2009-07-11
> **Rabz said:**
> how would you go about installing war3 on an Ubuntu9.04 system??

For playing Warcraft III (which is a Windows game) in Linux you must first install wine, which is a windows emulator. After that, you just click on the .exe file and install it like you would normally install it in windows. Warcraft III runs flawlessly in Linux, so after you install it, you can play it, it doesn't need any special configuration.
To install wine, just go to Synaptic Package Manger and search for it.
Cheers!

EDIT: the only thing you should do is delete or rename the movies folder. The movies don't work.
