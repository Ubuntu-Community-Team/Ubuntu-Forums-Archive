### Post by Didius Falco on 2009-04-25
From Ubuntu.com:

> To upgrade from Ubuntu 8.04, press Alt+F2 and type in "update-manager -d" (without the quotes) into the command box. Update Manager should open up and tell you: New distribution release '8.10' is available. Click Upgrade and follow the on-screen instructions.After you've updated everything, you could simply repeat the process for 9.04, if the upgrade isn't on offer on the Update Manager - it is there on my 8.10 installation, though.

Much better idea to use the cd's though. Especially now, with 9.04 being so new, the servers are likely very busy.
