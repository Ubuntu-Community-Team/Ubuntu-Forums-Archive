### Post by kilroy423 on 2008-04-24
> and im still a n00b at linux extensions

For the most part Linux extensions are just for the users sake, Linux usually knows what to do with a file even though it doesn't have a file extension.

I'm not entirely sure about running this app since I have not used it, but you might want to see if you can run it from the cli.  If you can run the app from the cli then you can just create a shortcut with that command in it.  Just right click on the desktop and choose create launcher, name it whatever you want, and then enter the command to run your app.

Kilroy
