### Post by Christmas on 2011-06-09
To install it type in a terminal:
```
sudo apt-get update
sudo apt-get install flashplugin-installer
```
This will install the latest version.

Alternately you can download it from [here](http://get.adobe.com/flashplayer/), uncompress it, and copy the libflashplayer.so file inside ~/.mozilla/plugins (if the directory doesn't exist then create it).
