### Post by Ms_Angel_D on 2009-05-06
The public folder is used as a way share files on your network, the templates folder is where you can create blank document templates.

if you go into the templates folder then create an empty file called "text File.txt" then go to any other folder and right click creat document you'll see the option to make a new text document.

Both can safely be deleted
