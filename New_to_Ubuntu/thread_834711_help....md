### Post by Het Irv on 2008-06-19
Check your sources, System -> Administration -> Software Sources

You should have the first four check-boxes checked, if they aren't go ahead and check them.  When you click close, it should pop-up with the box about reloading, click reload and it should scan the repositories, it should be something like 40-50 files.  Then try to download and install again.
