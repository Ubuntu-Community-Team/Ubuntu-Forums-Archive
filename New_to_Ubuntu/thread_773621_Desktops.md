### Post by Shazaam on 2008-04-29
There are a lot of things to do.
1. Leave default setup alone.
2. Remove one (of 2) panels.
3. Make it look like Windows.
4. Put the panel(s) on the side.
5. Remove both panels and have a launcher for the terminal on the desktop.
6. Install a "dock" applet like awn.
7. Make the dock float around the desktop.
Lots more. :)

### Post by erginemr on 2008-04-29
As said, both KDE ([www.kde-look.org](www.kde-look.org)) and Gnome ([www.gnome-look.org](www.gnome-look.org)) are highly configurable.

You can also try alternative desktops (technically, window managers). I can suggest **fluxbox, icewm and enlightenment**, which, among others, have a quite different appeal to managing the GUI compared to Windows.

You can install both from Synaptic, and when you do, you will have options to load into the installed desktops.
