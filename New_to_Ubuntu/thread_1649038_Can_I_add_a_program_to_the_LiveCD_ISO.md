I second the persistent live USB. If your computer can boot from a USB. Once you boot from the live USB you can add any program you like, change the desktop wallpaper, change themes, etc. I have a 4GB USB that I carry around with me.

Go to the menu System > Administration > Startup Disk Creator and make the USB persistent.
