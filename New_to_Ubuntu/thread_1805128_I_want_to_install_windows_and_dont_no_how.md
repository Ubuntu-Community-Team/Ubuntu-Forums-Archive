### Post by LowSky on 2011-07-15
If all you want is a GUI

log in, then type

```
startx
```

if that doesn't work

```
sudo apt-get install ubuntu-desktop
``` and then reboot and try again

if you want to install windows, well that's another story, and it going to be very messy. First depending on the version, you will need to boot from the cdrom/dvd usually default but sometimes a F-key... pay attention during boot for the command on the screen.

then follow the Windows instructions.. could take 30 min - 3 hours depending on the version of windows you are using. don't forget windows doesn't come preloaded with drivers on any version before vista. and if it doesn't know what the device is, and you can't get on the internet to to allow it to search you might be out of luck unless you have the drivers already on other media
