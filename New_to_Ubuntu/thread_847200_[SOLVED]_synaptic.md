### Post by avtolle on 2008-07-02
AFAIK, you cannot do what you are trying to do. The way Synaptic works, as I understand it, is that you need to select each package to be installed; then, a check is made to see if additional packages which are dependencies are needed to be installed as well, and if so, the user is notified and asked if these should also be marked, the response to which is generally yes. Thus, each individual package needs to be marked for installation. 

You may, of course, mark several packages for installation before clicking on Apply, and all packages so marked will be installed in one session, rather than doing it one at a time.

### Post by Rhubarb on 2008-07-02
This is easy to accomplish:
Click on the first package in the list you want to install, then scroll down to the last package you want to install, press shift and click on the last package so all the packages in-between are highlighted.
Then just right-click and select "Mark for installation".

Easy :)
