### Post by snova on 2009-02-16
It means the CD was burnt incorrectly, or possibly that the image itself was a bad download.

Check the image: [MD5SUM on Windows]("https://help.ubuntu.com/community/HowToMD5SUM#MD5SUM on Windows")

And burn at a slower speed.

[https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)
