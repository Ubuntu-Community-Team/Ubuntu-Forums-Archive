### Post by colobix on 2011-10-29
Hi.,
Uninstall it, add the blender ppa and reinstall it.

Here's how you'd do it in the terminal:

sudo add-apt-repository ppa:cheleb/blender-svn
sudo apt-get update
sudo apt-get install blender
