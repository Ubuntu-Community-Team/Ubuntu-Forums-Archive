### Post by oldfred on 2014-11-17
Also what video card does system have.

I do have 64 bit Ubuntu running on an old 2006 vintage XP system with only 1.5GB of RAM. If I load two large apps will will go to swap. And I have to convert from Unity to fallback (screen looks like the old gnome2) as old video chip just does not have enough horsepower for Unity.
You can install Xubuntu, Lubuntu or possibly Ubuntu but probably need fallback.

Did you confirm down load was correct?
       [https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

Some systems have BIOS settings to turn on keyboard USB and it may be called something else related to USB, or not. With my older desktop both Windows & Ubuntu keyboard would work as they seemed to use their own drivers, but grub did not work as it uses BIOS defaults. Grub does not have many drivers and then relies on BIOS settings being correct.
