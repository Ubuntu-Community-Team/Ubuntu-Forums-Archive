### Post by Golem XIV on 2008-05-08
Assuming I understood correctly what you want:

System -> Preferences -> Advanced Desktop Effects Settings

Find and click on Window Decoration

Find the Command field and click on the brush icon to reset it to default
It should reset to **/usr/bin/compiz-decorator**

Close, Close, log off and back on.
