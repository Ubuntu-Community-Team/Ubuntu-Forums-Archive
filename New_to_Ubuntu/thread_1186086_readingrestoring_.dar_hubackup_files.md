### Post by Rubicon_82 on 2009-06-13
If you are able to install 'dar' from the universe repository, you will be able to restore your backup.  See the 'dar-docs' package for some tutorials on how to do this.

If you are unable to install the 'dar' package, you can simply manually download and extract the 'dar-static' package.  This will provide dar as a standalone executable which won't need installation.
