### Post by gordintoronto on 2010-07-09
The server version has no GUI. Unless you will run a high-volume production server, what you have done is a great starting point, which lets you experiment with various server features.

What kind of server do you want? For example, to make your desktop system a web server, just install Apache, Mysql and PHP. You can do that through Synaptic.
