### Post by Volt9000 on 2009-09-10
Howdy, and welcome to the forums and the Wonderful World of Linux! :)

There are TONS of places you can get software. By far the easiest is searching through the repositories. Click System > Administration > Synaptic Package Manager. From here you can search through thousands of applications, libraries, games, everything that will work on Ubuntu.

There are also websites out there that give you alternatives to Windows apps. A quick Google reveals:

[http://www.linuxalt.com/](http://www.linuxalt.com/)
[http://linuxappfinder.com/alternatives](http://linuxappfinder.com/alternatives)

But there are many more.

And of course, one of your best friends is the Search function on these forums. Chances are if there's a program you need, it already exists and someone has already posted about it, so try searching. If not, you can always feel free to ask.

### Post by tacantara on 2009-09-11
Welcome to Ubuntu.  As previously mentioned, the Add/Remove programs feature and the Synaptic Package Manager will provide you with thousands of software packages.  If you can't find it in either of those, you can download Linux compatible programs from numerous websites.  If you go that route, try to find packages with the .deb file extension, as they are the easiest to install (one click, just like Windows .exe files).  Ultimately, you will find something that you used in Windows that just doesn't have a fully-featured Linux substitute.  For that, I recommend installing Sun Microsystem's VirtualBox program (download it from their website, as it will have more features than the version available in Synaptic).  VirtualBox will allow you to run Windows as a virtual machine within Ubuntu.  For the little that I rely on Windows, it is a practical solution.  Enjoy the limitless possibilities of Ubuntu :)
