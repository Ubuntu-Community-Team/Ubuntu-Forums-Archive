### Post by gobbles414 on 2008-08-05
Apologies if these seem like stupid questions, but:

1. Which email software are you using in Ubuntu? Examples include Evolution and Thunderbird. Or, are you using internet browsers to check your email? If internet browsers, which ones are you using?

2. Do messages with other types of attachments send correctly via Ubuntu (e.g. Word documents of PDF files)?

3. If your using email programs and not internet browsers, are you using the same protocols in Ubuntu as you are in Windows? For example, are you using POP in Ubuntu but IMAP in Windows? Also, make sure that the ports in your email program match the ports listed in the official directions for Gmail and Hotmail.

4. How large are the archives that your trying to send? Do smaller archives also fail to correctly send in Ubuntu?

5. Is the address to which you're sending the same in both Ubuntu and Windows? I'd advise you to check for silly mistakes, like a *,* instead of a *.* in the address.

6. Which version of Ubuntu are you using?

7. **Please post the failure notification.** Feel free to change the email addresses listed in the notification, if privacy is an issue for you.
