### Post by mcduck on 2012-02-02
Not really for the individual files, but you should be able to use something like [zsync]("http://zsync.moria.org.uk/") to download only the corrupted pieces of the disc image again.

...and in the future, I really recommend using torrent downloads is possible, as the torrent protocol automatically checks for data integrity, making sure you always end with a working disc image...
