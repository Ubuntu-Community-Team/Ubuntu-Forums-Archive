### Post by deathadder on 2011-04-05
Some general Ubuntu Administration stuff can be found on the [wiki]("https://help.ubuntu.com/community/SystemAdministration") with a helpful [server guide]("https://help.ubuntu.com/10.10/serverguide/C/index.html") available too.

I generally start by having a look in help.ubuntu.com for the app I need for example, when I needed a LAMP setup, searching for LAMP throws back the [ApacheMySQLPHP]("https://help.ubuntu.com/community/ApacheMySQLPHP") guide first off.

Beyond the Ubuntu help pages I tend to use the following sites 

[http://www.debianhelp.co.uk](http://www.debianhelp.co.uk)
[http://www.justlinux.com/](http://www.justlinux.com/)
[http://www.debian-administration.org/](http://www.debian-administration.org/)
