### Post by nicedude on 2008-08-29
try using this command to get to the gnome sound properties and then select alsa for all the catagories

COMMAND TO RUN IN A TERMINAL TO OPEN SOUND PROPERTIES

gnome-sound-properties

Doing this should then have your system using Alsa instead of Pulse and it should apply to flash as well. Also you may want to look into installing flash player version 10 as it fixes allot of problems I had with flash 9

Hope this helps
