### Post by zrdc28 on 2013-05-23
Go to the help screen on firefox, "Restart with addons disabled" and see if that helps! If it helps disable them one at a time until you find the right one.
If than doesn't work you might try reloading Firefox and see what you get. first you need to "apt-get --purge remove firefox" then "apt-get install firefox"
