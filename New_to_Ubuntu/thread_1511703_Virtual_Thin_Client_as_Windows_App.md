### Post by yaaarrrgg on 2010-06-17
There's a couple VNC Java implementations (applets).  From a web browser, you can remotely log into the computer, and essetially control everything remotely.

[http://www.realvnc.com/support/javavncviewer.html](http://www.realvnc.com/support/javavncviewer.html)

It just requires a VNC server to be running on the Linux box, and users with privileges to access it.  

Ubuntu has a VNC server installed by default under (IIRC... sorry I'm on XP at the moment): 

System -> Administration -> Remote Desktop
