### Post by stoogiebuncho on 2010-05-26
Try going to the "Places" menu (beside "Applications"), and opening the "Computer" shortcut.  Is your ipod listed there?  If so, double-click it to open it and see if that makes it available to Rhythmbox.

If it's not there, open a terminal, type in 
```
lsusb
```
and post the output back here.
