### Post by ugm6hr on 2008-11-08
As long as you don't require specialist software, then Ubuntu should be a good choice for you.

Additionally, it is worthwhile looking at your hardware (e.g. printers, scanners etc) to ensure they will work well with Ubuntu.

This forum is very helpful if you run into any problems.

If you prefer, you can install Ubuntu alongside Windows while you get used to it (like having a Windows security blanket), provided you have enough HD space.
