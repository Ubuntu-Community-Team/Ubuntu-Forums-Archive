### Post by Hospadar on 2009-02-06
If you need a later version of IE, you could use a virtual machine running windows (i'd suggest VirtualBox, check the community docs link in my sig, search for virtualbox)

If the site just tells you "You need IE for this site" and doesn't let you do anything, chances are you don't actually need IE, and their MS tech people are just too lazy and dumb to test their site on anything else.  You can try getting the "user agent switcher" addon to firefox that spoofs an IE (or other browser's) user agent header so websites think you are using ie.
[https://addons.mozilla.org/en-US/firefox/addon/59](https://addons.mozilla.org/en-US/firefox/addon/59)
It's pretty straightforward to use.  Just pick a version of IE as your user agent and browse away.

I used to work for a major defence contractor who's time reporting website was "IE only", user agent switch, and it worked fine in firefox.
