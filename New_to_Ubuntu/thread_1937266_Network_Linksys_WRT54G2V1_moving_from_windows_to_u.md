### Post by Mark Phelps on 2012-03-07
To briefly answer your questions ...

As you already guessed, the Linksys setup software will not run in Ubuntu, so you can't use it.

The "network settings" are saved in the router.  So the network security key would be the same when attempting to connect from Ubuntu as when from Windows.  That much does not change.

What is different is the device drivers for the wireless device in your PC.  You will have to set up wireless operations in Ubuntu differently than you did in Windows.

What I would suggest as a start, is that you look in both the Networking forum and the Hardware forum using the Router model number and see if there are already threads there dealing with it.  If there are, you will simply be able to use the info in those threads to configure wireless on your PC.  If not, either of those forums is a good place to ask your wireless networking questions.
