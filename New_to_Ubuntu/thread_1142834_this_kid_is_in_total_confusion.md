### Post by decoherence on 2009-04-29
32 bit = i386, i686 and i*X*86 in general

64 bit = x86-64, AMD64 (even if it's an intel processor) EM64T, Intel 64

In the case of 32-bit name, the differences refer to various optimizations that were done when the code was compiled.

In the case of 64-bit names, these are just different market names or names that have been 'retired.' They refer to the same thing.
