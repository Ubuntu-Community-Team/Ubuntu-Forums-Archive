### Post by Uzi304 on 2009-05-16
Ok say you extracted the file to your desktop. Then in your terminal you would run: ```
cd /home/your_username/Desktop/hamachi-0.9.9.9-20-lnx
``` If you extracted it elsewhere, then just change the path accordingly. Also if "hamachi-0.9.9.9-20-lnx" isn't the name of the folder, then change that also. After you run the command I just gave you, then run the commands given by the Readme.
