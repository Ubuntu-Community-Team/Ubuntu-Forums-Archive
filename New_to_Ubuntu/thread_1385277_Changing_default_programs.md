### Post by mcduck on 2010-01-19
Just right-click the kind of file you want to associate with a program, select Properties, and then on the Open With-tab select (or add) the program you want to use.

For example the transmission/deluge change goes by right-clicking a .torrent file and selecting Properties->Open With->Deluge.

(note that right-click->Open With is a different thing, it selects the program for that time only instead of setting the default..)

Also, some default applications can be selected in the Nautilus's preferences, open a file manager and go to Edit/Preferences, and the settings are in the Media-tab. And of course Firefox has it's own settings as well, although it usually uses the default application if you haven't defined otherwise in Firefox's preferences.

For example, I use smplayer when I want to play a playlist at faster than normal speed. VLC also does that, but it resets the speed to normal for every playlist item.
