### Post by Peter09 on 2009-06-18
Hi,
right click on the network manager icon and select edit connections. Go for the tab for your wireless connection and open it up. There on the screen you will see a box to allow this to allow the connection for all  users. Check that and its should now login automatically.

Note I have not got a wireless device in-front of me so this is from memory.
