### Post by mcduck on 2012-10-05
Alt + middle mouse button allows you to resize a window from any point inside the window, so you don't need to be able to click at thew window corner.

Also the window border sizes are defined by the theme you are using, so you could just go to gnome-loook.org and find a window border theme that you like and that has wider borders.
