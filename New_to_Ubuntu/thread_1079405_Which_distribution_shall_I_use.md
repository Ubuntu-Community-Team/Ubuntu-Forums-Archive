### Post by novafluxx on 2009-02-24
I use Ubuntu as my main Linux installation, and I play around/learn with Arch now. I used Debian back in the old days (2003-2004). I think Arch is simple and clean, very clean.

I'd say for ease of use, and maximum ability to obtain community support when needed, use Ubuntu.

Arch is more for intermediate and advanced users, and its incredibly powerful as you can do virtually ANYTHING you want.

It has bleeding edge software, all of wich I'd consider stable releases.

Ubuntu has a pretty frequent release cycle, which can lead to problems updating software. You can always update to the latest release, however after doing that a few times, I've heard it can really bog down the system. So then you'd want to do a clean install of the latest, then you'll either need your /home on a seperate partition, or back it up.
