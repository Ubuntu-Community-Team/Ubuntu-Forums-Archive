### Post by Rocket2DMn on 2009-05-30
Apple hardware (PowerPC) is no longer officially supported by Ubuntu, but you can get the latest PPC port from [http://cdimage.ubuntu.com/ports/releases/9.04/release/](http://cdimage.ubuntu.com/ports/releases/9.04/release/)
Newer apple computers use Intel processors, so they are supported with the normal Ubuntu install CDs which is what you probably have in hand right now.  Since you are using older Apple hardware, you will need to get an installation CD from the link I just posted.  The Desktop CD has a Live session that you can play around in before installing, and the Alternate CD is just an installer CD.

### Post by Rocket2DMn on 2009-05-30
You will need to download the disc image onto a computer that works, and burn it to a CD (see [uhelp]community/BurningIsoHowto[/uhelp]).  Once the image is burned, you will put the CD into the Apple computer and boot from the disc (like you are probably currently trying to do, but you are currently using a disc that isn't supported on that hardware).  At that point you can either enter the Live session, or install to the hard drive.

### Post by 3rdalbum on 2009-05-31
If you want to boot from an Ubuntu CD, you need to hold down the "c" key as the computer starts up.

Note that the machine probably doesn't have enough memory for Ubuntu, but you could try Xubuntu, assuming the memory in the computer has been upgraded at some point in its life.

If it's a tray-loading iMac, you might be scuppered anyway as those have trouble reading modern CD-R discs. You would need to find the best quality CD-R available with the darkest dye.
