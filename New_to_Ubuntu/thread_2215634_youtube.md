### Post by Impavidus on 2014-04-07
You may need the flash player. Only part of the videos on youtube is available in html5. Open the software centre and search for **flashplugin-installer**. Install it. Then restart the browser.

Edit: flashplugin-installer is included by ubuntu-restricted-extras, so that will work too.
