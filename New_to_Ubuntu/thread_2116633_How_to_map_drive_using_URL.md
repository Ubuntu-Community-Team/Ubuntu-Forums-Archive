With windows 7, I would go to My Computer-> Map Network Drive -> Connect to a Web Site that you can use to store your documents and pictures - > then you put in the URL with the user name and password.

### Post by steeldriver on 2013-02-16
Hello and welcome

You have a number of options depending on the nature of the remote share (i.e. whether it is Windows based or Unix based)

Probably the easiest way to start is using the regular nautilus file manager, and selecting 'Connect to server...' from the 'File' menu. Then enter your remote share info and logon credentials.

Alternatively you can type a URL directly into the nautilus address bar (or equivalently, use the Go --> Location... menu)

Once connected, you can save the remote folder as a 'Bookmark'
