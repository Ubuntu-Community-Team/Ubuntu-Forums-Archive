### Post by Kopkins on 2012-09-10
Go to System Settings > User Accounts, or from the dash type user and press enter. At the top right you must unlock the window to make changes. Select the user that automatically logs in. There is a switch to select whether or not it logs in automatically. 

Additionally, you can go to System Settings > Brightness and lock, or open the dash and type brightness and press enter. From here, you can change how the computer will behave after the screen turn off or when resuming from suspend. Note that these changes apply for only the active user. If you want to make setting changes for a different user, you must login and make the changes with that user. 

When the computer resumes from suspend, you can have it require a password, and when it does this, there is a switch user button you can use to take you back to the main login screen. 

Best of luck,
Kopkins
