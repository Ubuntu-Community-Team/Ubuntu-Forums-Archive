### Post by Vined Adobo on 2010-08-27
I don't know how many times this can be said, but I'm not sure if it can be said enough.  I was saved by it again today.

Okay, this is kind of advanced, so if you're new, have someone who knows what they're doing help.  When you first install Ubuntu, make two partitions:  One for / and one for /home.  Okay this is OPTIONAL the first time and ONLY the first time you install Ubuntu, format the /home partition.  When things screw up (and they will) your home directory (/home) should be intact.  When things go bad, use the same names and passwords as when you first installed ubuntu to install anew but do not format your home directory.  Your /home directory should remain untouched. (at least for me, it does.)  You will have to re-install applications, and some settings, but your data (pictures, letters, notes, etc. should be just fine.

Now, don't rely on this to protect your data.

BACK IT UP to different media, MANY different media. (there are many fine tutorials and apps to do this!
