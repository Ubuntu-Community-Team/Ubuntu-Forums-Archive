### Post by Mark Phelps on 2011-11-08
I wouldn't trust Disk Utility reports.  Find out the make and model drive, go to the manufacturer's website and see if they have a drive testing utility you can download. Download that and run it.  If it says the drive is failing, then see about returning the drive under warranty for a replacement.  You would need to run the manufacturer's disk utility anyway to get warranty replacement.
