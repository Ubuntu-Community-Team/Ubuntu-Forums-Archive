### Post by Pawel Andrzej on 2009-02-20
I just noticed a Keyboard-Preferences quirk on my Thinkpad R40 running ubuntu 8.10 (Intrepid.)

I added Spanish and Polish keyboard layouts through the GUI:
System > Preferences > Keyboard > Layouts

Once I added desired layouts to the list I went to:
Layouts > Other Options > Use keyboard LED to show alternative layout

I checked the option "Num Lock LED shows alternative layout."

This setting didn't simply light up the Num Lock LED when I switched keyboards, but activated Num Lock in such a way that some alphabetical keys were re-mapped as numerical, in the following way:
/ --> +
> --> .
M --> 0
J --> 1
K --> 2
L --> 3
; --> -
U --> 4
I --> 5
O --> 6
0 --> /
O --> 6
P --> *

Half my letters suddenly disappeared. How annoying!
The simple cure is to deselect the option "Num Lock LED shows alternative layout" in the Keyboard Preferences GUI.

Not a big issue, but maybe this will save some confused person a little time.

If anyone knows about similar Thinkpad quirks, please share.
