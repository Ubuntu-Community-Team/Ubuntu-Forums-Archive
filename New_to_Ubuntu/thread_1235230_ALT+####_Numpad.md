### Post by CatKiller on 2009-08-08
The easiest way to do accented characters is with the Compose key (there is a way to enter Unicode numbers, but I can never remember what it is because the Compose key method is so easy). You can set the Compose key in System &#8594; Preferences &#8594; Keyboard &#8594; Layouts &#8594; Layout Options, but I think AltGr is the default.

To do é, you'd do Compose &#8594; ' &#8594; e.
è is Compose &#8594; ` &#8594; e.
ë is Compose &#8594; " &#8594; e.

And so on.
