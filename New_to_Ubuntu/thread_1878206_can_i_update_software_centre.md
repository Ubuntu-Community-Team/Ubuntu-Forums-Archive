### Post by Mark Phelps on 2011-11-09
Software center is just a front-end to the repositories that you have installed.  Everything that you can do with a GUI in Software Center, can be done with a GUI in Synaptic, or can be done on the command line with apt-get, aptitude, and/or dpkg.

So, to update what Software Center "sees", you update your repositories.  And, you can do that with Update Manager.  Have it check for updates.
