### Post by FredJones on 2008-05-11
I am now using Ubuntu 7.1 for 2 weeks. I am not thrilled with the look of Gnome, however. I made an Ubuntu virtual machine and I tried Xfce and it looks nice. Then I tried KDE and while it seems potentially even better, the reviews I wrote note that it can be slower and less stable than Gnome and speed and stability are VERY important for me.

My main issue is not so much with functionality as it is with the 'look' because my default Gnome is too 'chunky' and 'bubbly' and too much like Win XP. When I use Windows I use 2K and when I used XP temporarily, I used the 'classic' look which is like 2K--this look makes all the window elements smaller and leaves more space for window content--this is what interests me--I want to see as much content in my window as possible. I like to reduce the icons and borders and other elements.

Aside from that, I need speed--I switch windows a LOT and I have never had a machine, actually, where the speed of changing windows satisfied me. I always turn off all 'effects' so that when I click on a window it should just appear *instantly*. I would like as close to that as possible. :)

So what do people suggest I do to find the look and the speed I want? Which desktop manager do you recommend and should I be looking at different themes for them? Or perhaps other items are configurable which I am not yet aware of. I am not 100% clear on how best to adjust my system to get what I want. It seems that the menu fonts and icons for the same app are actually virtually the same between Gnome, Xfce and KDE, so my guess is that this is not the solution.

I have also read that KDE and GNome together can clutter things and maybe even cause bugs. I also had an experience myself once where installing a new win manager (I think it was KDE) did cause a problem IIRRC.

Doing research into this GUI issue I also discovered the existence of Gnome Office and KOffice--I am testing these now as I am not a huge fan of OpenOffice. Interesting that neither Gnome nor KDE install their own office packages by default.

I am also learning how to use multiple desktops, as this feature (even with my 2 physical monitor setup) seems great, ONCE you learn how to use it. :)

OK, any ideas are appreciated.

Thanks!

### Post by sdennie on 2008-05-11
One thing you could try other than the things you've tried is fluxbox.  It's not a complete "desktop environment" but, it's very lightweight and clean.  To install it:

```

sudo apt-get install fluxbox

```

After that, you will be able to select it from the login screen.

### Post by N.N. on 2008-05-11
Indeed, you can achieve greater speed and give your desktop a more minimalistic look by installing a lightweight Window Manager such as Fluxbox, Openbox, or IceWM (for which a ["Windows Classic"]("http://themes.freshmeat.net/screenshots/68144/73526/") theme is actually [available]("http://themes.freshmeat.net/projects/winclassic/")).

Forum member aysiu hosts a detailed guide on how you install IceWM: 

[http://www.psychocats.net/ubuntu/icewm](http://www.psychocats.net/ubuntu/icewm).

A detailed wiki entry on installing and using Openbox is also available: 

[https://help.ubuntu.com/community/Openbox](https://help.ubuntu.com/community/Openbox).
