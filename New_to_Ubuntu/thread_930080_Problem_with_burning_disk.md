### Post by jualin on 2008-09-25
Are you using Windows or Ubuntu to write the CD?
If you are using Windows then you shouldn't use the "Write to CD" option. You need an application that can burn .iso images such as Nero, Roxio, Clone CD. If you don't have any of those you can try [Infrarecorder]("http://infrarecorder.sourceforge.net/"), an open source burning application that works pretty well in Windows.  Good luck!
