### Post by QIII on 2010-04-27
The RC (Release Candidate) is already available.  It would be best not to install a Beta at all right now.

If you do normal updates, you will have the Final.  There would be no reason to  download and install again.

When you have the RC installed, either use the normal update GUI or just use the terminal:

```
sudo apt-get update
``````
sudo apt-get upgrade
``````
sudo apt-get dist-upgrade
```You should do this frequently while you are running any distribution.

There is some confusion about dist-upgrade.  If you do not use any flags or modify your sources list, dist-upgrade will upgrade the current distribution.  It will not install the next version.
