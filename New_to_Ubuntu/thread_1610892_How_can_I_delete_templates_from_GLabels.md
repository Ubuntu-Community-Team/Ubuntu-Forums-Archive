### Post by sdim on 2010-11-03
Mystery solved.
I e-mailed Jim Evins, the creator of the application.
He said that the templates created by the user are stored in the home directory, in the file called .glabels. You have to go to
View-->Show hidden files, of course first, so that the file is visible.
Many thanks, Jim.
Keep up the good work.
