### Post by NovaAesa on 2008-08-15
Click on Applications -> Add/Remove Software. Then do a search for LaTeX. There should be about 5 IDEs to choose from that come up ready for installation.

Alternatively, you could write the markup in a plain text editor (such as gedit) and then compile it using pdflatex.
