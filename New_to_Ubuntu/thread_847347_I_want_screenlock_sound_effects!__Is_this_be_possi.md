### Post by adamogardner on 2008-07-02
here's exactly how I did it:
  download the .wav file.
  select save on download dialog
  moved desktop icon for .wav file off of my desktop and to another directory
  system>pref>sound>sounds tab
  click on sound effect box that you want to change and select "select file"
  find the sound effect in the window that pops up(found where you put it)
  double click then that should do it.  
  this worked for me.  I feel lucky.
