### Post by t4thfavor on 2010-11-25
setup evolution to connect via IMAP to hotmail (im pretty sure it's supported) then make a folder to hold the 'old' mail (this is a matter of preference) next, press ctrl-a and then drag all the old email that you accidentally downloaded into the new folder you just created.

You can do this with the inbox instead of a folder, but sometimes it's nicer to sort it this way.

I have also forwarded all the mail to myself, but sometimes this causes you to have duplicates.

### Post by geeky_chris on 2010-11-25
Thanks for your responses everyone!

Was actually an easy non linux solution ... all the messages (4000+ of em! non-junk mail!)  When they were transferred the server deleted them and put them in the deleted items folder.  If you click the select all check box hotmail then shows a 'select all items in your deleted items folder' which then selects all the pages and pages of stuff.  You can then move this back to your inbox.

Simples!

Thanks for the ideas anyway... hopefully this might help someone if they make the same mistake as me.
