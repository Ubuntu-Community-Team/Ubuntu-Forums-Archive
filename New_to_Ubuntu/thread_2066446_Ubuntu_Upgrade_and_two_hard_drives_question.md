### Post by ajgreeny on 2012-10-04
There is no real reason to remove and then re-install the second hard drive when you install ubuntu if you choose "Something Else" at the disk preparation stage.  That will show you all the partitions currently on both disks and allow you to choose what partitions to use/reuse/delete etc etc.

Just make sure you know which disk is which before you install and it should all go smoothly.
