### Post by grahammechanical on 2015-06-03
> in fact, all partitions

That is the killer. We reinstall the same way we did the first install. In your case when we get to the screen that is asking for install options we select Erase Disk and install Ubuntu. That is the most simple method.

If we are re-installing and we want to keep the OS we are dual booting with then we select the Something Else option and things get a little bit more complicated.
