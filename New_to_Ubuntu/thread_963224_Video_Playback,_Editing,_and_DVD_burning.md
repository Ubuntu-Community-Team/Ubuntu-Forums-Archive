### Post by bumanie on 2008-10-30
Ensure you have codecs etc from [medibuntu]("https://help.ubuntu.com/community/Medibuntu").
1. > sudo apt-get install vlcVlc player plays nearly every codec there is.
2. Look at kino, for downloading and editing off a dvcamera. For already downloaded video, lives is good. For dvd conversion, devede. Once devede has processed the files it outputs .iso - you only need to right click the file and choose burn to disc.
3. Depends what else you want to do.

### Post by igknighted on 2008-10-30
I like Kaffeine for video playback.

As for editing, there aren't any spectacular options.  Kdenlive is probably the most promising, but is a little rough around the edges.  Without doubt Cinelerra is the most powerful if you need lots of features, but it can be complicated until you are used to the interface.

Get to know the terminal... it's so convenient for lots of tasks, when I'm on windows (like I am stuck until a bug gets fixed upstream so I can use my new laptop with linux) I feel so innefficient without it.
