### Post by Fraoch on 2011-05-11
Start a program using whatever way you want - the smaller Ubuntu button on the top left corner or the "Applications" button in the Launcher, this looks like a magnifying glass.

Once a program is started, its icon will appear in the Launcher.  When it's there, right-click it and select "Keep in Launcher" (there will be a checkmark beside "Keep in Launcher") as wojox described in post #2 of this thread.

### Post by CaptainMark on 2011-05-11
well, if its on the side your running unity, ubuntu's latest choice of desktop manager, it it very nice looking and really slick, but is in its infancy so is lacking features for now,

if its on the top then your running gnome, ubuntu's older desktop manager, whilst not being as visually appealing it has been around for 8 years or so and as a result is a bit of a veteran, its got loads of features and applets and is highly customizable, 

unity is the new default in ubuntu unless your graphics card is not powerful enough to handle it, then you are given gnome by default.  you can infact choose between them if when logging in, select your user but before you enter your password there is a dropdown box at the bottom and you can choose something like ubuntu desktop for unity and classic desktop for gnome, 

basically its your choice, and neither is a clear better choice,
