### Post by Vladlenin5000 on 2014-07-06
Radio Tray is a standalone app, ie, it works on its own and doesn't require additional software for the streaming protocols it already supports.
Differently, as you noticed, opening the page from where you usually listen a given Radio sation, informs you that there's something missing, the web browser alone isn't enough.
You can install Flash support along with other proprietary codecs not included by default in Ubuntu (neither in Windows) simply by installing the "ubuntu-restricted-extras" you can also find in the software center.
