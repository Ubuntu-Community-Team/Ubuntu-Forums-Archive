Alrighty. I guess youre right that there arent too many guides out there for 'point-and-click' guides for setting up a USB drive.

Ill help. Lets go through this guide, step by step. Im going to assume that you already have Windows installed, so we will use this guide here: [http://www.pendrivelinux.com/create-a-ubuntu-9-10-live-usb-in-windows/](http://www.pendrivelinux.com/create-a-ubuntu-9-10-live-usb-in-windows/)

If you have already downloaded the .iso, skip to step 2. If not, you could download the .iso from here - [http://www.ubuntu.com/getubuntu/download-netbook](http://www.ubuntu.com/getubuntu/download-netbook)

Following step 2, you need to download the file from here: [http://www.pendrivelinux.com/downloads/u910/USB-Installer-for-Ubuntu-v0.2.exe](http://www.pendrivelinux.com/downloads/u910/USB-Installer-for-Ubuntu-v0.2.exe)

This will allow you to navigate to the .iso file and install it to your USB drive. 


From there, its just a matter of waiting until its done, and then rebooting to the USB!

From there you could install Ubuntu.
The guide here is for the regular desktop version, but the steps are the exact same: [http://www.psychocats.net/ubuntu/installing](http://www.psychocats.net/ubuntu/installing)

After that, youre all set!

Good luck! Hope this helps!
