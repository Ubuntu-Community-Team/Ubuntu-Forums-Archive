### Post by Chronon on 2009-10-06
I think mp3tag has more features than EasyTAG, but they both do enough for me.  EasyTAG has good automation to automatically rename or retag whole collections (mostly what I have used it for).  I haven't used MP3Tag in a while, so I don't really remember too clearly.  

Picard is another metadata scanner/tagger, though it may be a bit "too automatic" for some people.  It uses MusicBrainz database to attach/correct metadata to your files.  You still have final say about which files it tags and which option to use if there are several matches.  It's not flawless, so you should make sure that you agree with its matches if you use this.
