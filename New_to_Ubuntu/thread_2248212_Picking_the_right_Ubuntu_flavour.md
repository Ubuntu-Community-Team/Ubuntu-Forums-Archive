### Post by Rob Sayer on 2014-10-14
Ditto on not installing 3rd party packages while installing, and don't install the ATI proprietary driver from Additional Drivers.  ATI doesn't support legacy cards in linux.  The open source video driver should be just fine.

I've run Linux Mint running MATE as a DE.  I actually didn't find it all that powerful or fast.  And don't, for God's sake, install ubuntu 14.10 until it's officially released as a non beta version.  I know guys who maintain unix/linux for a living who will not touch alpha/beta linux OS releases unless they absolutely have to.  It's definitely not something for beginners.

Linux Mint is pretty good for experienced users in general, though I tried Mint 13 XFCE once and it was a terrible bugfest compared to Xubuntu.  And their tech support is terrible.  I can't recommend mint to beginners for that reason.  Ubuntu is the only one for newbies IMHO.

With those specs you could run any of the 4 ubuntu DE versions.  I first installed the Unity version myself but I don't like it as much.  Frankly I didn't find it fast enough.  YMMV.  I'm mostly a Kubuntu user.  KDE is also big and heavy like Unity but you can seriously set it up to run faster, which Unity can't do.  Xubuntu and Lubuntu are lighter and faster, especially the latter.  Lubuntu is way faster than the rest but configuring it is more complex for new users compared to the others.  I've also found it the buggiest.  Xubuntu is a good balance of features and speed.

The best way would be to just dl and burn all the iso live CDs and try them for yourself.  Burn to a USB stick if possible.  It's much faster so it gives you a better feel.
