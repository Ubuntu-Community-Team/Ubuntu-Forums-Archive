I don't know the situation of that particular CD, but I know that Ubuntu does not have autorun, as it is a security hole. You want to look through the CD and look for where the linux driver is. Find some file that has a .deb extension, or maybe they have a readme file somewhere in there which tells how to install the linux driver.
