### Post by peter d on 2011-10-28
Boot up from the Live CD. Don't install yet. Then you can mount the drive and rescue your data from Windows.

When you do install you need to be aware that 8.04 is no longer supported (or at least not the desktop version). Try 10.04 or 11.10. If you have hardware issues 10.04 might be the better option although if you could run Vista I would have thought that 11.10 would be fine.
