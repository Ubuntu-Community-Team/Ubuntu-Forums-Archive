[FONT=Georgia][SIZE=3]Hello people, 

I am new here so I am happy if this is going to help .

A few days I was struggling to run my Atheros under Ubuntu and Voila !
When I turn on the WiFi ( Fn+F1) in Windows, then restart the PC, after the BIOS loading the WiFi led always turns off. So that means the BIOS is turning off the WiFi and when Ubuntu is loaded WiFi doesn't work.

[/SIZE][/FONT][FONT=Georgia][SIZE=3]Solution: Flash the BIOS with a higher version (I did with 1.16 it was 1.13 before). For more information look at Fujitsu Siemens site.[/SIZE][/FONT]
