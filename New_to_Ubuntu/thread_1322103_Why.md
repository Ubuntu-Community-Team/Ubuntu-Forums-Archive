### Post by Faolan84 on 2009-11-10
The best thing I've found you can do before installing an OS or doing any partition work whatsoever is to back up. I personally keep a separate external USB hard drive around that I use for periodic backups of my data.

here's what you do:
1: plug in your external USB hardrive into your box
2: load up a decent live CD and go to the commandline
3: make sure you mount all necessary drives
4: find out what your disk device names are and copy an image of your computer's internal hard drive to a file on the external hard drive. The command should look something like this
```

dd if=/dev/sda of=/media/sdb1/backup.img

```
5: shutdown your computer and remove the USB external hard drive
6: do any work you need without fear of data loss.
7: if something does go wrong you can plug in the USB external, load up your Live CD and do something like this in the terminal to restore your data
```

dd if=/media/sdb1/backup.img of=/dev/sda

```

It's really simple, and at it doesn't really take all that long, but if you ever need to do a recovery, you'll be thankful you have a full backup of everything on that particular hard drive include partitions and the MBR.

dd is a really great command to learn and it's one you should get to know real well. It will save you a lot of trouble.
