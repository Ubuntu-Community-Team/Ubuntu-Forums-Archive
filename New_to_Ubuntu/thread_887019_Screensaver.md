### Post by mc4100 on 2008-08-11
> **ramunenke said:**
> How do I set up ... some of those savory effects like the video [on this link]("http://http://www.downloadsquad.com/2008/08/08/3-linux-apps-that-make-me-hate-windows/").
Compiz is now installed by default if your hardware supports it ... which means if you can run it, then you already have it! So you don't need to worry about finding and setting it up :).
The effects can be tweaked with the Compiz Settings Manager (not installed by default). I believe it's...
```
sudo apt-get install compizconfig-settings-manager
```
and then you will have a new menu entry, which I think is under System -> Preferences.

### Post by rockerphil on 2008-08-11
well i'm personally still struggling with installing Compiz on my computer, but as far as some screen savers go you can try xlock and xscreensaver, which both come with some pretty decent screensavers. they can be installed with:

sudo apt-get install xlockmore

and:

sudo apt-get install xscreensaver

oh, one i also like myself is electric sheep, it can be installed with:

sudo apt-get install electricsheep
