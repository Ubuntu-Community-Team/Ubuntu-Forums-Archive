I use wicd and you can pick serveral wep keys (ASCII, hex, 64/128bit).

If you still have your Ubuntu CD, you can safely remove WICD using Synaptic Package Manager.

After that, insert your Ubuntu cd into the CD rom drive, and reload Synaptic.

Search for network manager and install it. You might also want the nm-applet (so it will appear in your top panel, you will need too add it to the panel I presume).
