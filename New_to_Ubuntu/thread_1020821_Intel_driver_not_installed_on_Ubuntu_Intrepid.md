### Post by Nepherte on 2008-12-24
I guess the "Configured Video Device" will use the intel driver. You can also explicitly add the Driver option:
```

Section "Device"
	Identifier	"Configured Video Device"
        Driver          "intel"
```

Remember to backup your xorg.conf file before doing any changes.

### Post by Nepherte on 2008-12-24
Something like that. The way X server nowadays works is that it auto configures/detects as much as possible, resulting in a very small xorg.conf file like you have. With recent versions of X, you can even run it without the xorg.conf file. On the other hand, you can explicitly say what xorg has to do by putting it all xorg.conf.
