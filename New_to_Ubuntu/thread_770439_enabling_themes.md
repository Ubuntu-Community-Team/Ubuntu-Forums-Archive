### Post by elmer_42 on 2008-04-27
Emerald worked in Gutsy, but now when I upgraded to Hardy it doesn't apply the theme when I click on it in Emerald. The way I fixed it was to change System -> Preferences -> Advanced Desktop Effects Settings -> Window Decoration -> Command. I changed it from the default "/usr/bin/compiz-decorator" to "emerald --replace". Then I pressed Alt+F2 and ran emerald --replace. I hope it works for you. :)
