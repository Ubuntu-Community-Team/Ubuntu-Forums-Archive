### Post by Sub101 on 2008-06-19
Splash screen you can change with: gnome-splashscreen-manager
The login window can be changed with the "login window" gui in the system menu.
The boot screen can be changed with "start-up manager".

The apps are all repos as far as I am aware. As for the others, you can change the menus by right-clicking on a menu. From there you can find each application and change its name and comment. Not sure if that is the sort of thing you are after though?
