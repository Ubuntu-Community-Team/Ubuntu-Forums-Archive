Mediawiki is the software that Wikipedia (and many other wikis) runs on; setting up a local server running it is not going to help you at all.

You probably want to look into something like Super Karamba, but for GNOME instead of KDE.

EDIT: Ah, that would be [gDesklets]("http://en.wikipedia.org/wiki/GDesklets").
