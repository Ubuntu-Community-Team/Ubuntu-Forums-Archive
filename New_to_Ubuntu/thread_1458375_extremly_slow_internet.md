### Post by QIII on 2010-04-20
The 32 bit Flash installer can be installed from the repos on both 32 and 64 bit versions of Ubuntu, and 64 bit Flash (beta) can be installed manually on a 64 bit version of Ubuntu.

I've done the latter since Jaunty without any problems.  It works just as well in Lucid, which I have been using as my primary OS since the first Beta.

A very few Windows-centric websites do complain about the 64 bit Linux version of Flash (Hulu is one where I have personally encountered this), but they are rare.

An informed answer is more helpful to the OP than FUD.

### Post by QIII on 2010-04-20
If you have 64 bit machine architecture, you should probably use a 64 bit OS.  

If you will remember, those who stuck with a 16 bit architecture soon found themselves left behind in the move to 32 bit architecture.

Memory aside, the 64 bit machine/OS architecture has advantages in things such as speed, precision of calculations, machine instruction sets, increasing software support, etc.

Put another way:  There is certainly no advantage to using a 32 bit OS when you have a machine that can operate a 64 bit one.
