### Post by barthel on 2009-09-06
Just got the Rival myself.

It sounds like you have the proper USB cable. The issue is that there are *2* USB mode menu items.

Under "Settings & Tools" there is the item "9. USB Mode". This is *NOT* the one you want.

Instead, under "Settings & Tools", choose "2. Tools", then "8. USB Mass Storage". You'll have to do this every time you connect to your computer. (Personally, I think this should be the 3rd option under the "9. USB Mode" menu item so it can be used as the default USB mode...)

HTH
