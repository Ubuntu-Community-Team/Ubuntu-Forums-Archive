### Post by westie457 on 2011-12-19
It is possible using the command line however you will have to upgrade a total of 3 times to get to 11.10 with the possibility of something going wrong with each upgrade. Personally I recommend that you wait until 12.04 LTS is released then upgrade. That is go from one LTS to the next LTS.

PS Also I would wait for a few weeks until most of the early issues get sorted out.

==============================================

The LTS you are using is supported until April 2013. The next LTS Desktop edition will be supported for 5 years, the same as the Server edition.

Nothing wrong with using using any version until it is no longer supported.

### Post by JRV on 2011-12-19
I agree with westie457's suggestion, but make sure you backup first.

There are big changes between 10.04 and 12.04.  The change from gtk2 to gtk3 is big, as is the change from gnome2 to gnome3/unity.

This upgrade may not go smoothly.  I am not going to chance it.  I think a fresh install is in order.
