### Post by Penguinnerd on 2012-01-12
Memory is not the only factor. The 1 Gb PC might have a faster hard drive, or a faster processor, or maybe it just has fewer drivers and stuff to load. 

As long as you have the minimum amount of ram, startup speed will probably not be helped with adding more ram. Having more ram only really matters when you start running more applications.
