### Post by diablo75 on 2009-02-03
I think what you're looking for is the Scale plugin.

Scroll down to the bottom in Compiz Config Settings Manager and find "Scale".  Not "Scale Addon"...I don't know what that does.

This captured image was very helpful to figure out.

I hope my figure out also help others.
There are 4 options in this menu (also in the captured image that diablo75 attached ;))

1. "Initiate Window Picker" -> shows all application windows in the current workspace
2. "Initiate Window Picker For All Windows" -> shows all application windows from all workspaces
3. "Initiate Window Picker For Window group" -> shows all same application windows to current highlighted application from all workspaces (For example, if currently gnome terminal is highlighted, this shows all gnome terminals from all workspaces).
4. "Initiate Window Picker For Windows on Current Output" -> I couldn't figure out the differences from "Initiate Window Picker", yet. --> If anyone figure out difference between 1 and 4, please reply it. It will help many others. :)

**A suggestion**: I think that "Initiate _Window_ Picker For All _Windows_", for example, are confusing because these two 'Windows' mean different things. Maybe, "Initiate _Window_ Picker For All **_Workspaces_**" are easier to understand because currently Ubuntu use the term "Workspace" for the latter 'Window'.
