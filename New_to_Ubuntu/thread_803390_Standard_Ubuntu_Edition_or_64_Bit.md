### Post by neurostu on 2008-05-22
64bit vs 32bit is a pretty big debate in these forums. The long and short of it is:
** Chose 32bit IF: **
You are more concerned with things working
You are not going to NEED more then 3 gb of RAM
You are going to be using mainstream applications

** Chose 64bit IF: **
You NEED more then 3gb of RAM
You will be running 64bit versions of applications
You are ready and willing to fight to get some 32bit applications running

If you are new to linux and just want things to work then use 32bit. Its many times easier to get working then the 64bit version of ubuntu.

I personally have a 64bit cpu and have 4gb of ram installed, however, for now I am using the 32bit version of the os. I have used both, but in my experience the 32bit version works better, requires less tinkering under the hood, and is easier to use.

That being said I love linux as I get to play with it and I enjoy learning how things work, so I will probably move to the 64bit version someday....

Its really up to you.

to answer your original question: Yes you can install a 32bit OS on a 64bit architecture without any issues. I've done it several times. (I think it actually easier then installing the 64bit OS, less issues with finding 64bit compatible drivers etc...)

Also check out this thread: [http://ubuntuforums.org/showthread.php?t=403064](http://ubuntuforums.org/showthread.php?t=403064)
