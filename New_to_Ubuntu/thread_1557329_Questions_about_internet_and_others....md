### Post by XubuRoxMySox on 2010-08-20
Ubuntu is kinda "net dependent" for updates and stuff. But you can get an older (but still supported) version on a CD with all the updates already applied, too. It uses alot less bandwidth than the newest version - especially while the current newest version is freshly released because it updates alot.

You can also set it up to update only once a week or once every couple of weeks, and download only security updates. That trims down it's bandwidth usage alot too.

With newbies I always try to set them up with an older release anyway 'til they get used to it, because they're not nearly as buggy and update-hungry as the new release is.

Hope that helps,
