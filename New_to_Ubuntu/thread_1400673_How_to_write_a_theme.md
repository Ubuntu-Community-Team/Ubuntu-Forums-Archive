### Post by MelDJ on 2010-02-07
try: [http://gnome-look.org/content/search.php](http://gnome-look.org/content/search.php)

or go to system-appearances-theme and edit the theme to become like the theme you want.
then go to your home folder and enable hidden files to be seen. go to the themes folder and you will see a folder with the name of the theme you made in appearances. compress it into a tar.gz to make it installable on other computers
