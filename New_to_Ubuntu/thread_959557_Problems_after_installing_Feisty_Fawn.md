### Post by Zill on 2008-10-26
Feisty Fawn is now obsolete and unsupported.

[https://wiki.ubuntu.com/Releases]("https://wiki.ubuntu.com/Releases")

I suggest you [download]("http://www.ubuntu.com/getubuntu/download") the latest version of Ubuntu, Hardy (8.04), and then try to run the live CD, rather than installing right away.  If this runs OK then you can click on the install icon on the live CD desktop to install the system to your hdd.

See how that goes then report back with any problems.
