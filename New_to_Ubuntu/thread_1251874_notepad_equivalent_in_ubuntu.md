You're kidding, right? Two reasons: One, you use Windows Notepad to write HTML and Javascript, rather than an editor that actually gives you some features. And Two, you're asking if there's a text editor available on Linux?

Well, there are plenty of text editors. Ubuntu comes with three text editors preinstalled and there are literally hundreds available. What operating system *doesn't* have a text editor?

If you're going to be writing HTML and Javascript, then the very least you should be using is Gedit - Applications > Accessories > Text Editor. It has code colouring and shows line numbers. But if you're serious about your craft then you should use something like Quanta Plus or Bluefish, that is a dedicated HTML editor.
