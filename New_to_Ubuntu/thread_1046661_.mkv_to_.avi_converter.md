### Post by andrew.46 on 2009-01-24
Hi adobrakic,

FFmpeg is trying to help you out:

> **adobrakic said:**
> 
```
[mp2 @ 0xb7d872f0]encoding 6 channel(s) is not allowed in mp2
```


If you do not wish to actually change the codec used by FFmpeg you could try as the program has suggested and decrease the audio channels, and perhaps also set a higher bitrate for the audio and video:

```
ffmpeg -i ThePianist.mpg -vcodec mpeg4 -b 512k -acodec mp2 -ac 2 -ab 128k ThePianist.avi
```

There is room there for more adventurous encoding but perhaps this might be enough and FFmpeg should fill in most of the other settings with fairly generous default settings.

BTW this is a different file from your previous matroska file? Settings will of course need to vary from video file to video file.

Andrew
