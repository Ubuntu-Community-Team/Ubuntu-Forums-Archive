### Post by ExileAmongYou on 2011-06-06
If you're just going to do a clean install, it shouldn't matter that your current install isn't recognizing the drive (assuming you already have the install cd). You just need to check that your motherboard's boot settings are set up right, so that it looks for the cd drive before the hard drive, and restart the computer with the cd in the tray.
