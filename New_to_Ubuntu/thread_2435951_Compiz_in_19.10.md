### Post by deadflowr on 2020-01-29
With the default gnome desktop Ubuntu uses?
No.
gnome shell is too tightly integrated with the mutter window manager.
Trying to switch to have it use compiz will break gnome shell.

To use compiz, use a different desktop environment.
Xubuntu, Ubuntu-mate and gnome-panel can all use compiz.
(And the unity desktop environment is built to only use compiz)

General thread on desktop environments window managers  et al, see:
[https://ubuntuforums.org/showthread.php?t=2415676]("https://ubuntuforums.org/showthread.php?t=2415676")
