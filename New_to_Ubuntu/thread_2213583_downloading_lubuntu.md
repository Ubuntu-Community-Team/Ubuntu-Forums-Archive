### Post by sudodus on 2014-03-27
A computer with Windows 7 is new enough to boot from USB, so you have also the alternative to create a USB boot drive (for example a pendrive). It might be more convenient, and a pendrive can be re-used many times for boot drives as well as for temporary data storage or transport of data.

See this link describing Unetbootin (a tool to create a USB boot drive from an iso file)

[Paul Sutton's Unetbootin how to]("http://zleap.net/unetbootln-usb-how_to/")

and this link with general information about booting from USB

[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)
