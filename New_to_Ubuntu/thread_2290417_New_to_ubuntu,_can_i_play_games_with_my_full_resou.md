### Post by mystics on 2015-08-12
If the game has a Linux version, it should run well enough. The problem will arise if you try to play Windows games. Wine (or PlayOnLinux if you want to keep things a little simpler) can play some Windows games, but performance may take a hit among other issues. I've had games that run smoothly, but certain features like Fullscreen and various graphical effects just break the game. 

Basically, know what you're getting into beforehand if you want to make Linux your primary gaming platform. If you still plan on playing a lot of Windows games and they don't have good ratings on the Wine Database mentioned above, then it would be best to dual-boot Windows and Linux and use the Windows side for your gaming. That will also help ease the transition rather than throwing you straight into Linux while having to get a new list of games to enjoy.
