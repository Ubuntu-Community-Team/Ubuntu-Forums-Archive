Are you using 64 bit? theres a known bug that makes life a pain in the butt.
If youre running 64bit, then go here and follow my guide: [http://ubuntuforums.org/showpost.php?p=8573604&postcount=2](http://ubuntuforums.org/showpost.php?p=8573604&postcount=2)


If youre not, or youre not sure, open the terminal (applications>accessories>terminal) and type
```
sudo apt-get install ubuntu-restricted-extras
```
itll ask for your password; dont worry if it doesnt show up when you type. itll download and install everything there then
