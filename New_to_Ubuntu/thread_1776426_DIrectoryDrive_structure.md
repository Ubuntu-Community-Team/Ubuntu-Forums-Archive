### Post by KL_72_TR on 2011-06-06
There is nothing equivalent of Windows in Linux. Linux see everything as a file and all files and directories are under the (or start from) [/] = root directory.
I think these two links can help you a lot: [http://http://www.pathname.com/fhs/pub/fhs-2.3.html]("http://http//www.pathname.com/fhs/pub/fhs-2.3.html") here you can find everything about Linux file system and [http://http://linuxcommand.org/]("http://http//linuxcommand.org/") the commands you need to work.
