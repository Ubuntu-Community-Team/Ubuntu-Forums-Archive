### Post by bumanie on 2009-01-14
HP have very good support for printers. [Go here]("http://hplipopensource.com/hplip-web/supported_devices/index.html") and check that your model is supported. If it is, plug it in prior to the installation of ubuntu and it should automatically be installed.

### Post by Amarsingh0793 on 2009-01-14
Not at all :)

Just plug the printer in, press ON and wait like 30 seconds and VOILA! In most cases, it will automatically install the printer without you even touching any config files, source code drivers e.t.c.

If you really do have a HP Printer, I recommend you install the "hp-lip-gui" package from synaptic. 

Have fun :P
