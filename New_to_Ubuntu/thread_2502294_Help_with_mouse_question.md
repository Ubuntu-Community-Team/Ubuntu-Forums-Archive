### Post by grahammechanical on 2024-11-08
It happens automatically. When the mouse pointer moves over a link the shape of the pointer changes to a hand with the index finger pointing up. It isn't actually a "grab thing." Try moving the pointer to the top of an application window. The pointer shape does not change. Now left click the mouse and you will have grabbed the application window and you can move the application.
