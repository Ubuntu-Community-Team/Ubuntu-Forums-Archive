### Post by thomz92 on 2010-01-20
do you have compizconfig settings manager installed? if not, do so. Afterwards, open it and go to the rotate cube plugin and check the box that says "edge flip move". That should do the trick. If you do not have the cube enabled, go to the desktop wall plugin, and in the last tab check the box that says "edge flip move".
