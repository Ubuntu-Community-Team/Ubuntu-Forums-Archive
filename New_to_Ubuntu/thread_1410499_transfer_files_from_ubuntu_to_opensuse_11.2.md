### Post by cariboo on 2010-02-18
There are three different ways to do it over a network:

[list=1]
[*] Install [OpenSSh]("https://help.ubuntu.com/9.04/serverguide/C/openssh-server.html") and use sftp
[*] Install [NFS]("https://help.ubuntu.com/community/SettingUpNFSHowTo")
[*] Install [Samba]("https://help.ubuntu.com/community/SettingUpSamba")
[/list]

There are other ways to do it, but three methods are the most common.
