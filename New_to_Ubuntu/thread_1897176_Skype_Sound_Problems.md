### Post by lechien73 on 2011-12-18
I had a similar problem with my wife's Acer Aspire laptop. The solution was to install PulseAudio Volume Control, and control the microphone as separate channels rather than one stereo channel. I've written up the problem[ here.]("http://blog.mattrudge.net/2010/03/08/microphone-problems-in-skype-on-ubuntu/")

Although that's written for an older version of Ubuntu, the same problem has continued with each subsequent upgrade, so I know it's still current.

Hope it fixes your problem too :)
