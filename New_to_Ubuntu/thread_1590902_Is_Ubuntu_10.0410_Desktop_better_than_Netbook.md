Hi,
Yes, it is true, Kubuntu Desktop and Kubuntu Netbook have been merged together for Kubuntu 10.10. 

Here is how to switch from desktop to netbook without having to log out:
[http://digitizor.com/2010/09/18/how-to-switch-between-desktop-and-netbook-workspace-in-kubuntu-10-10/](http://digitizor.com/2010/09/18/how-to-switch-between-desktop-and-netbook-workspace-in-kubuntu-10-10/)

I happen to like this better than the netbook desktop itself... but I have a large enough screen anyways... so I can't really say which setup I would prefer more on an actual notebook/netbook though.

### Post by snowpine on 2010-10-08
The only difference between Ubuntu Netbook, Ubuntu Desktop, Kubuntu, Xubuntu, Lubuntu, etc. is the desktop environment (user interface) and pre-installed applications. They are all the same core system "under the hood." I would recommend choosing the desktop environment **you** like best and that is most productive for you, and then install any extra apps you need from the Software Center.

### Post by lbrty on 2010-10-08
+1 to snowpine

Remember you can always install a different interface without reinstalling the entire operating system. By going to System>Administration>Synaptic Package Manager, you can add and remove different desktop environments. For example, type in any one of the following: *kubuntu-desktop* or *ubuntu-netbook* or *lubuntu-desktop* or *xubuntu-desktop *or *ubuntu-desktop*. If you install one or all of these you can switch between each one at the login screen to change your desktop environment.
