### Post by drs305 on 2008-09-25
Yes, you can download it from synaptic if you have the medibuntu repository enabled. Right now, both the googleearth site and the repo have a version of 4.3. Synaptic's version is at 4.3.7+ so it can't be too far behind.

I'm for simple, so if it's listed in synaptic it would just be:
```
sudo apt-get install googleearth
```
