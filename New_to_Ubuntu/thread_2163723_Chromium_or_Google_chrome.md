### Post by craig10x on 2013-07-19
Chrome also has a built in pdf reader (which comes in very handy) and it adds a ppa to your software sources so that you automatically get new versions and updates through the ubuntu updater...
As previously mentioned it also has an up to date flash plug in (pepper flash) and a few other codecs that Chromium does not have....and with Chromium (as with Firefox) you have to reply on the rather outdated adobe flash that you got installed with ubuntu restricted extras...Adobe no longer makes new versions of flash for linux BUT they have an agreement with Google to provide latest versions for it's Chrome flash plug in...

I have used both and now use Chrome exclusively...mostly because it is more a more complete package then Chromium...

### Post by craig10x on 2013-07-19
As far as tracking...what many people don't realize is that many home pages (including yahoo which i love to use as my home page) also track and so do various web sites you probably visit every day (including forums that have ads on them, etc...) the reason they track is simply to target ads specifically to your apparent interests and not for some nefarious reason as some here seem to think....
Also, as i said, the pepperflash plug in provides the latest flash versions...when you use Firefox or Chromium you are using the 2 year old flash on your system and Adobe no longer provides new versions for that on linux, only security updates...

But it's your decision of course...Chromium is fine but Chrome is more complete and more updated...The only reason Ubuntu is not considering it over chromium (as the new possible default browser) is because Chrome has a license agreement added to it that has to be said yes to when you download it...but then, so do microsoft fonts if you want to install them along with ubuntu restricted extras...so it is not really any big deal...
