Agian what do you mean it saves it in files? It only saves it if you save the files.

You should be able to go to "new" icon->"page in new tab" or you can use the more option in that new icon. Also check tools options and see if you have language set as HTML4 and DTD as transitional.

Kompozer is really not ideal. Linux doesn't seem to have any really good WYSIWIG editor. If Kompozer is a bit strange to you then you maybe could try Bluefish.
