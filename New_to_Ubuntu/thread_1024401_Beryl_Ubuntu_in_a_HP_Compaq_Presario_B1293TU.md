### Post by zephyrcat on 2008-12-29
I don't know specifically about that notebook model, but as far as beryl goes, you should be fine.

Beryl and Compiz actually merged to form Compiz Fusion, which is what is now in Ubuntu. It does have some additional system requierments, but pretty much any modern system should run at least the basics. Just go to System > Preferences> Appearance and enable desktop effects and you should be good to go.

### Post by igknighted on 2008-12-29
The actual hardware requirements for a composite desktop are actually pretty low.  What matters are that it isn't ancient, and that the driver supports the necessary functions.  Since intel uses an open-source driver, it typically has the best support for Compiz/Kwin effects.
