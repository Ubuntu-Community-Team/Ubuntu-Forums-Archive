### Post by arrrghhh on 2010-04-26
Well if you're up for it, building a PC would be the best way to go... Get the cheapest parts, exactly what you want, and you're not tied to an OS.

If you're not up for it, I would almost recommend (if you're going the Dell route) to get one with Windows - why?  I've heard horror stories of the "dellbuntu" version that Dell uses on their Ubuntu branded PCs, plus the hardware choices for their Ubuntu-specific boxes is usually very limited.  Perhaps that last part has changed - I hope so.

BUT - back to the matter at hand.  Unless you're doing some crazy graphics, playing the newest games, or running a server with a lot of clients connecting to it... that hardware you have is just fine.  Lucid Lynx (v10.04 of Ubuntu) is about to be released for the "final" version - If you can wait for it to drop (what... 3 days?) then grab that final edition and give it another whirl on that old desktop.  Hopefully sound works better for you.  If it doesn't, and you go for the full install and still have no sound - I'm sure someone here can help you :D
