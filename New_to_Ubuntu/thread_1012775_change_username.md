### Post by ibutho on 2008-12-16
You can change your username although sometimes this can result in undesirable effects (which can be easily fixed). I personally would do it whilst logged as a different user and make sure that the user whos details you want to change is not logged in. You can use the User and Group management GUI tool or use the CLI. If you'd lik to do it from the CLI, post back for detailed instructions.
