### Post by -kg- on 2009-02-01
According to [this Quick Start Page]("http://www.mondorescue.org/docs/mondorescue-howto.html#QUICKSTART"), you have to launch the application from the terminal as root, in Ubuntu's case:

```
sudo mondoarchive
```

Apparently, the software isn't big on graphics but sounds good enough.

If you want something just as good but a bit more on the automated side, try [Clonezilla.]("http://clonezilla.org/")  A very good [tutorial on both Clonezilla and Partimage is here.]("http://www.dedoimedo.com/computers/free_imaging_software.html")
