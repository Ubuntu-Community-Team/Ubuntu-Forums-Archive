Hello,

Onboard can be activated at the login screen by clicking on the "accessibility" icon (kind of like the [Vitruvian man](en.wikipedia.org/wiki/Vitruvian_Man") drawing) and then selecting "onscreen keyboard"

If you find the onboard keyboard is blocking the display (there is no docking facility within regular use, only login/lock screen), you can set the transparency to a comfortable level within onboard's options. This will allow you to see "thru" the keyboard.
