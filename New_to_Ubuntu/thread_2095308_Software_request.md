### Post by DuckHook on 2012-12-16
Ubuntu comes with so many, it is an embarrassment of riches. I believe 12.10 comes with Rhythmbox. Others prefer VLC, MPlayer, Banshee, Amarok. Tons more. Check software centre with search term "media players". It's Linux. Download them all and give them a spin.
