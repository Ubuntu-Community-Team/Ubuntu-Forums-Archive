### Post by chyneymon on 2008-06-29
Thought this little tip might help some newbies like myself.

Currently running Ubuntu 8.04 with Nvidia 8600 gt 512 mb.I have browsed previous posts regarding the sluggish feel with general things like opening/closing windows, leftover window artifacts, and slight jerkiness with fancy desktop effects using compiz. I noticed this even after a manual install of 173.14.05 (keeping in mind that I've also already tried using the restricted drivers).  I am still a n00b, but I found that the powermizer feature on the nvidia display driver 173.14.05 showed the GPU constantly downclocking. I thought this might've been the source of my nitpicky issues. Finally, while browsing I found a how-to page for automatically setting the gpu clock speed to maximum while on AC Power. I have to say, my system overall feels much smoother and more responsive now. Also, I love using all the compiz effects now without any noticeable slowdown or sluggishness when opening/closing or moving windows around, etc.

I followed this quick tutorial and everything is working perfectly. Can't complain with the buttersmoother effects. Thanks to whoever posted it. Enjoy!
