### Post by Sealbhach on 2010-05-05
In Nautilus, right click on the file and select "Open with other application", there is a tick box so that Nautilus will remember that choice for files of that type. That should do the trick.

### Post by _0R10N on 2010-05-05
You can set the default application for any type of file, following these steps:

Example: let's assume you want to change the default application for .doc files
1- Open nautilus and look for any .doc file.
2- Right click on it
3- Open the Properties option
4- Go to the Open With tab
5- Select or add the application you want.
