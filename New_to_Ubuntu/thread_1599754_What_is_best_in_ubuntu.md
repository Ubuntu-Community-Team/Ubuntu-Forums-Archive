### Post by eentonig on 2010-10-18
> **tweaktolive said:**
> will all windows apps, games work on ubuntu with an emulator ??

The better question would be

"Are there enough linux programs/alternatives available to accomplish what I want to do with my computer."

Linux is not compatible with windows. So windows programs will not run under linux. Yes, you can choose to use an emulator like Wine to run your Windows programs. But that seems stupid to me as a default approach. Why emulate an environment when you can use it natively?

For most day to day use, I recommend using linux. But if you have specific needs that require a program that doesn't have a decent alternative program under Linux, use windows.
