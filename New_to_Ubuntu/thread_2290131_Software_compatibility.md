windows based software will likely not run on Linux. there was some success using wine where a few applications can actually be run.: [https://appdb.winehq.org/](https://appdb.winehq.org/)

everything rated lower than gold on that page is usually a waste of time.

if you are already using open source on windows chances are there is a Linux version of same software. if not then you can either find an alternative or forget about using Linux. I mean if you are tied to windows 8 with software then it's best to stay with windows. 

Games usually still run better on windows.

### Post by buzzingrobot on 2015-08-10
Software written for one operating system won't run on other operating systems.  Tools like Wine in Linux create an environment running on Linux that attempts to simulate the environment that app would see on Windows.  Success varies.  Mastablasta has given you good advice.

### Post by howefield on 2015-08-10
You do not say if you have installed Ubuntu yet or are considering it, but another point to ponder is that many applications typically used on Ubuntu also have Windows versions, eg firefox web browser, thunderbird email client, vlc media player, libreoffice for documents and spreadsheets ect, pidgin instant messenger, gimp for image manipulation and 100s more. 

Installing on Windows might be a way to get used to the software as you get used to Ubuntu.
