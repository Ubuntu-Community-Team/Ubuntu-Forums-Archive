Working Stiff; Hi ! Welcome to the forum.

The most likely cause is a graphic card issue. Here is a method to try and see if you can boot.

1. in bios set the 1st boot priority to the cd drive.
2. insert the install disk into the drive, patiently wait -> stick figure/keyboard emblems ?
3.hit the escape key ->language screen->escape-> options menu -> f6 key -> boot options
4. in the boot options screen choose the "nomodeset" option; 
5. f10 to continue to boot. -> wait some more -takes time to de-compress.

do you now get the ubuntu desk top ?

If yes, we know what to do in the permanent install.
