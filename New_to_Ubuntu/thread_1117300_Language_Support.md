### Post by mkvnmtr on 2009-04-05
When you go to log in click on sessions and see if you can find someplace to pick your language. If not you might need to add some program support. In my experience using Spanish and English I have found quite a few programs that I needed. There are two sections in the package manager for internationalization and localization. Go through them completely and you will find what you need.
