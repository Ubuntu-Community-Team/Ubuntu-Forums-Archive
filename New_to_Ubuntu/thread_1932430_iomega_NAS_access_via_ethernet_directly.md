### Post by gandaran on 2012-02-27
it's easier to do it on ubuntu than windows!
from places » connect to server » choose windows shares (will not work with any other option for NAS servers) type the NAS IP address on server box and hit the connect button, if you have a username and password set ubuntu asks for them automatically.
