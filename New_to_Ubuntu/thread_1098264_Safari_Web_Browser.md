Epiphany rocks. And for those of who want to test Chromium here it is : 

> How to install Chromium on Ubuntu

Open your /etc/apt/sources.list file !
[QUOTE]
gksudo gedit /etc/apt/sources.list

Add these lines at the very bottom of it:
> 
deb [http://ppa.launchpad.net/chromium-daily/ppa/ubuntu](http://ppa.launchpad.net/chromium-daily/ppa/ubuntu) intrepid main
deb-src [http://ppa.launchpad.net/chromium-daily/ppa/ubuntu](http://ppa.launchpad.net/chromium-daily/ppa/ubuntu) intrepid main

then upload and install:
> 
sudo apt-get update
sudo apt-get install chromium-browser

The you&#8217;ll find Chromium in your Internet menu. [/QUOTE]
