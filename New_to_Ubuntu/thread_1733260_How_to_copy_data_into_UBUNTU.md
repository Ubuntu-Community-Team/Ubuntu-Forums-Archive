when you plug in the usb drive, it should be mounted automatically. by default a file manager (nautilus) will open and display the contents of the drive. simply select all or select your data that needs to be coming into ubuntu and paste it in your home directory or create folder in home.

if you are trying to copy else where then there will be permissions problem. check that first. otherwise its a breeze, as you will do with a normal pendrive !

### Post by Iusedtowearatie on 2011-04-19
If the data on your external HDD has some kind of structure, e.g. is sorted into music, pictures, documents etc, you might benefit from copying the relevant parts into the corresponding standard Ubuntu folders; music, video, documents etc.
The alternative is of course to bulk copy everything as has been mentioned earlier in the thread and the do the sorting afterwards.
