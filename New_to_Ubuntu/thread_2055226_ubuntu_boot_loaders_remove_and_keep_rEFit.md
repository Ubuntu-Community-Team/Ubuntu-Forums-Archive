### Post by 2F4U on 2012-09-09
Ubuntu needs a boot loader. If you want to use rEFit, the Ubuntu boot loader should be on the same partition where Ubuntu is installed. See here for more detailed information:

[https://help.ubuntu.com/community/MacBook/TripleBoot](https://help.ubuntu.com/community/MacBook/TripleBoot)
