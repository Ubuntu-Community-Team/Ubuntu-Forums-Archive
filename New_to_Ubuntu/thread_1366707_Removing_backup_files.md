### Post by presence1960 on 2009-12-28
Chris, in your file browser are you viewing Show Hidden Files? I ask because those files are backed automatically by Ubuntu and don't show in the file browser unless you choose Show Hidden Files either from the View on menu bar or by depressing Ctrl + H

Those backups take up 0 bytes and you can't open them from a file browser. (see my screenshot) 
Is there any particular reason they are bothering you? If you don;t want to see them disable Show Hidden Files in your file browser and you won't see them.
