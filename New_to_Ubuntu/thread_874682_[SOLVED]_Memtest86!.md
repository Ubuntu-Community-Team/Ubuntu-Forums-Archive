### Post by mcduck on 2008-07-30
It runs the test infinitely.

Generally, you should let it run for a long time, at eleast over night, to make sutre that your RAM is ok and working correvtly. But of course if you haven't had any troubles with your computer you don't need to use it at all..

Interpreting the results is easy: If you see any errors, either one or more of your memory sticks is broken, or has some compatibility issue with your motherboard. If you get no errors everything is fine.
