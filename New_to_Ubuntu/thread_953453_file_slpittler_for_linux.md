### Post by Malta paul on 2008-10-20
To use the Java version of HJSplit. You need to have Java installed. Then just put the HJSplit downloaded folder in your 'Home' directory. The folder will be 'hjsplit_g.jar' click on it and you will get a graphical display.

As ilrudie say you can also use the terminal, just follow the 'man' instructions. It a little harder to use but works well.

Have fun:)
