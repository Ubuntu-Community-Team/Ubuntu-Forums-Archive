### Post by sgosnell on 2010-02-23
The Ubuntu wiki page suggests installing the .deb from the mondo site, not the repositories.  Apparently the repos have renumbered the versions, leaving out a decimal, so the older version in the repos is reported as newer, when it's not.  You need the latest version, which is currently 2.2.9.2-1.
