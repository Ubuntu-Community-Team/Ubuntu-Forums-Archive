### Post by ajgreeny on 2013-10-15
What did you upgrade from?  If you're now using unity as your desktop the scroll bars are hidden away by the overlay-scrollbar system and all its packages, which I think you can still remove and use the standard scrollbars.

This was one of the (many) parts of unity which I simply could not live with, hence my now using Xubuntu 12.04.

See [http://ubuntuforums.org/showthread.php?t=2147684](http://ubuntuforums.org/showthread.php?t=2147684) for a thread with more details or search for "remove overlay scrollbars" at [www.googlubuntu.com](www.googlubuntu.com)
