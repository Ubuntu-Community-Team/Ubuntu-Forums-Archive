### Post by andrew.46 on 2009-05-30
Hi nimajiman,

> **nimajiman said:**
> I would like to install a tool for indexing/keeping track of my files and searching my desktop easily. I'm wondering what people find best and easiest to use and what will put the least stress on system resources.  [...]Are there other options I've missed?

If you want to step back a little from the gui offerings you could do worse than look at the set of gnu utilities known as 'findutils':

[LIST=1]
[*]**find** - search for files in a directory hierarchy 
[*]**locate** - list files in databases that match a pattern 
[*]**updatedb** - update a file name database 
[*]**xargs** - build and execute command lines from standard input 
[/LIST]

These utilities will place a minimal stress on your system resources but they do require some research and learning before they can be used effectively.

All the best,

Andrew
