### Post by QLee on 2010-09-24
Well, I guess I have to state that I agree with mcduck, gThumb opens pictures but it may just be a question of semantics, I think I understand what you want.

To achieve your stated goal "Is there a way to link my picture/image folder (thunar) so that it automatically opens with gthumb?", if I understood correctly, you'd probably want to edit the preferences of gThumb and set the folder you want as the folder to use when gThumb starts. (it's on the General tab) That way you could have the pictures displayed just by starting gThumb.

On the other hand, perhaps what you want might be to associate a picture type with gThumb so that when you click on that picture type in nautilus, it opens in gThumb. But it likely the above that you want.
