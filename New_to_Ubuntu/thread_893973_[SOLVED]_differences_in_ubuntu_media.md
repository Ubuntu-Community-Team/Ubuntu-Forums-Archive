### Post by Sorivenul on 2008-08-18
As far as I know, there is no difference in the installs.  Ubiquity performs the same actions with an install from DVD as from CD.  The extra packages on the DVD are helpful in the event of no internet connection.  I had to do this on my first Ubuntu install, and that was a while ago.  Good luck!
