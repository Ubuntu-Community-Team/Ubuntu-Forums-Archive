### Post by audiomick on 2010-01-18
Hallo.
I haven't done what you are talking about, but here are my thoughts anyway:

As far as the "pre-loved" machine goes, get as new as you can afford. If you want to update things that are too old, you might have to hunt around a lot to find the bits you need.

I think the CPU is less of an issue than RAM and HD size, particularly if you can get by with a server without a GUI.

As far as the HD size goes, I tried putting a bigger HD in a machine with an AMD K6 400 a couple of years ago, and the motherboard couldn't even deal with a 40GB HD, which was the smallest one in the shop at the time.

You will want to have as modern a network card in the thing as you can, I think. I don't know if there are issues with new network cards in old machines, but shoveling around large amounts of data on a slow network isn't that exciting...;)
