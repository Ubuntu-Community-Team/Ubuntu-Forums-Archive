### Post by JBA2337 on 2008-04-23
As a new Ubuntu user, I am doing my best to get used to some of the idiosyncrasies of the new system. Some things, like entering a password every time I want to change anything, become a bit tiring.

I figured out one way to help with the "entering password blues." I put my password on a sticky note, and then left the sticky note on the screen, where I can see it. Now to copy my password, all I have to do is to select it and then type CRTL-C. Then when I an asked to enter my password, all I have to do is type CTRL-V. This method works, even in a Terminal box.

I am concerned about leaving my password on the screen where I can see it. Is this a security risk?  Could someone, coming in via the Internet ever get a look at my desktop? If they could, they would see my password.

Thanks.

JBA2337

### Post by amazingtaters on 2008-04-23
Well I certainly wouldn't recommend it. And think about things this way. Whenever it's asking you for a password, that's a gate, and there is a gatekeeper keeping vigil over it. This gatekeeper lets you, who knows the password, in. That shady looking dude (read virus/malware/spyware) that keeps coming up to the gate doesn't know the password, and so cannot pass. It's all a safety measure, and one I know I'm appreciative of.
