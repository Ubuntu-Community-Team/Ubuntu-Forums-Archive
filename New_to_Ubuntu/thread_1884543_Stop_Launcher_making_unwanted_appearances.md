### Post by Old-un on 2011-11-21
I really enjoy running Ubuntu 11.10 except for the Launcher popping out when i accidentally move the mouse onto it. Is there anyway i can stop the Launcher from appearing unless i want it too?

---

### Post by roger_1960 on 2011-11-21
Hi

Using compizconfig-settings-manager (install it if not yet installed) you can go to the Unity Plugin section and adjust the "edge reveal timeout" which changes how sensitive the launcher is to the mouse touching the side (or corner or wherever you have defined will launch it in the "reveal mode" section)
