### Post by dmizer on 2008-07-30
alt+f4 does the same thing in Ubuntu as it does in Windows.  If this is not working, then the WoW window is not taking keyboard input (the same reason you can't alt+tab away from it).

You can try ctrl+alt+f2 (which will drop you into a commandline shell), and kill WoW from the command line.  Use ctrl+alt+f7 to get back to the desktop. Or you can try ctrl+alt+right-arrow which will give you a new virtual desktop from which you may also be able to kill WoW.

### Post by Yuki_Nagato on 2008-07-30
Consider switching to a terminal with "Ctr + Alt + F1"

Log in.

Use the command "top" to find the process name of WoW.

Then use the "killall XXX" command with XXX being the name of the WoW process.

EDIT: Yes, dmizer is right.  No point getting to the terminal with Ctr + Alt + F1 unless you know to get back with Ctr + Alt + F7.
