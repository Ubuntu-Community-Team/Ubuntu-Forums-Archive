### Post by diablo75 on 2009-03-29
The only concern I have with your setup is the amount of RAM you're laptop has.  It is highly recommend you have at least 384 MB of RAM to get a good experience out of the OS.  So you might consider installing Xubuntu instead (lower RAM requirements).

Other than that, I wouldn't be worried about drivers at this point.  Just download a Live CD and attempt to boot from it to see if the OS loads up okay before installing.  If you're planning on installing Ubuntu/Xubuntu ONLY, you might as well not waste time researching compatibility and just use the **Alternate **CD to install the OS and then see what happens upon first boot afterwards.  The Alternate CD is available for download from Ubuntu's website, you just have to look for the link to it.  It's a text-based installer that avoids trying to load all sorts of software like the Live CD does, but the full OS will be installed with the GUI.  It will likely run just fine.  As for drivers, can't install third-party drivers until after the OS is installed.  So you'll have to get that out of the way first.

### Post by diablo75 on 2009-03-29
Well, as they say with RAM, the more the better.  I got by on half a gig, but when I started running XP in a virtual machine, it was clear I'd need more to be happy.

I got buy (and could still get by) on one gig total for years and was always happy with the performance.  If you're going to put that much RAM in, I'd go with Ubuntu instead of Xubuntu.  You'd probably not notice much difference in performance, as the CPU in your laptop is a good speed.
