### Post by brucewagner on 2010-04-24
On the default Home (Start) page shipping with Ubuntu 10.04, the default search page does not automatically position the cursor (active insertion bar) on the text entry box for the search field...  AUTOMATICALLY.

Note that the page, google.com DOES position the cursor on the text entry field automatically.

For this reason, most users end up changing their default Start page to:  google.com    .....simply for that one little feature / benefit.   It makes searching so much easier....  one less click....   every time!

This is robbing Canonical of revenue, by the way.

If this could be fixed, I believe most users would leave the default Start page as is.... to support Canonical / Ubuntu.   We all want to help support Ubuntu....  but not at the cost of usability.

Another feature of  google.com  that the Ubuntu/Google start page does not have is...  the quick launch links at the top-left for, "Web Images Videos Maps News Shopping Gmail  more &#9660;"  ....which we've all come to reply on.

Couldn't these be added as well?
