### Post by t0p on 2008-08-15
I use the java app **Your Freedom** to do this.  It is available for download from [www.your-freedom.net]("http://www.your-freedom.net").  First of all, you'll need to install **OpenVPN**. It's in the repos, so get it through Synaptic or with terminal command:

```
sudo apt-get install openvpn
```

There will be no need to configure OpenVPN, its default settings are all that's needed for it to work with Your Freedom.  So you just install it.

Then you download Your Freedom from [www.your-freedom.net]("http://www.your-freedom.net")

On that site you'll also find a pdf Users Guide.  Download that too, it's most informative.

My internet service provider blocks us from using instant messaging.  Your Freedom allows me to get round this restriction.  It is also used by journalists in China to access websites blocked by the Chinese government.  So I think it'll work for you.
