### Post by Dennis N on 2014-09-26
> what would be the difference in the 32 bit and 64 bit as far as performance on the same 64 bit pc?
If I set both up on the same machine, had you try both, I doubt you could reliably tell which was the 64 bit and which was 32 bit. There is not enough difference to really matter in general usage.

In any case, you have to use the 32 bit installer. An Athlon XP2200+ processor is 32 bit so can it only use a 32 bit OS.
