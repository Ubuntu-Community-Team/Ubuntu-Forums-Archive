### Post by Bucky Ball on 2010-02-09
Put it this way, if you had three computers on a home network and a networked 2Tb external hard drive (for instance) you would use the networked hard drive as the server. That could contain all data and therefore accessible from all three computers.

You could replace the external hard drive with an old computer jammed with hard drives and no screen. Just plop it on a network.

A server is mostly used for hosting websites??? One of its jobs. Not the case in a large (or small) office situation serving several or hundreds of computers.
