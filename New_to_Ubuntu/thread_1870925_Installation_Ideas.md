### Post by mikechant on 2011-10-28
> Windows software cannot be installed on Linux. Exe files are specific to Windows and Linux will not be able to run them. 

This is not really the case; a significant amount of Windows software can be installed and run successfully on Linux using the 'Wine' application. I run a few items this way myself.

However, the particular program you mention, "Titanium Maximum Security" looks like the sort of program that protects Windows from various security threats and it is not relevant or appropriate for Linux (and it definitely won't run anyway, as per my comment below).

If you have other Windows software on CD, it **may** be possible to run it but you must install Wine first (it should be available in the Ubuntu software centre).

If you want to see if a particular Windows program will run with Wine, go to the Wine application database at this address:
[http://appdb.winehq.org/](http://appdb.winehq.org/)
and select the 'browse apps' option from the left hand menu.

Note that if you look up "Titanium Maximum Security" in the Wine application database, someone did try to install it just for fun and it was not installable.

It's always best to find a proper Linux equivalent program rather than running windows software with Wine, but sometimes there is no good equivalent and sometimes Wine helps fill those gaps.

Another possibility is to run Windows in a virtual machine under Linux, but that's a whole new topic...
