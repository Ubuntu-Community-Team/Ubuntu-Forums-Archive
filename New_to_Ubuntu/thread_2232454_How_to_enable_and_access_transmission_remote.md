### Post by cariboo on 2014-07-05
I use Qbittorrent, to enable the WebUI, click Options->WebUI. To access the web interface just point your favourite browser to:

[http://localhost:8080](http://localhost:8080)

Transmission should be the same, except the port number is 9091.
