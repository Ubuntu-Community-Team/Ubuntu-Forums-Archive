I take it xp still shows up in the partition but you just can't boot - is that correct?  You can get xp back by fixing the master boot loader - sorry I don't remember how to do that but if you search the forums (try mbr in search) you should find a post that explains how to do that.  It should have installed grub but for some reason it sounds like grub was not installed.  I would reinstall Ubuntu - make sure you partition manually.  Also, I personally see no reason to use easy Peasy anymore (just my opinion - I am sure others will differ).  The native Jaunty works very good on Netbooks.  Lots of good tutorials to follow ([http://tombuntu.com/index.php/2009/05/04/installing-ubuntu-904-on-the-eee-pc-901/](http://tombuntu.com/index.php/2009/05/04/installing-ubuntu-904-on-the-eee-pc-901/))
