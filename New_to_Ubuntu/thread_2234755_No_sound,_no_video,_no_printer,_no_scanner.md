### Post by slooksterpsv on 2014-07-20
If you have a 2GB or larger USB thumb drive, we can boot from USB otherwise you'll need a blank DVD. Here's the steps for USB and DVD

1. Download the distro version you want - Ubuntu, Kubuntu, Xubuntu, or Lubuntu - for your system I'd recommend 32-bit
2. Open Software Center and search for unetbootin - click to install it
3. Plug in your USB Thumb Drive (must be 2GB or larger)
4. Open Software Center and search for gparted - click to install it
5. Click on the menu icon and search for gparted, open the program
6. Click on where it says /dev/sda and change it to /dev/sdb - if you're unsure you can see the actual size of the drive (750GB is actually 690GB). SDB, in theory, SHOULD be your USB drive, if not, find it in the list.
7. Click on Device -> Create Partition Table - make is msdos
8. Click on Partition -> New -> Fat32
9. Finally click on the green checkmark.
10. Open unetbootin - go to the menu and search for it.
11. Select ISO, click on the ..., go to the Downloads, select the ISO file you downloaded e.g. ubuntu-14.04-32-bit.iso
12. At the bottom make sure the type is USB Drive, and select on Drive the /dev/sdb1 - if it was SDB that we formatted above, it could be sdc, sdd, etc.
13. Select OK and it'll make the live USB.
14. Reboot the computer and press the key to get into your BIOS (while the computer logo screen is up), it could be F1, F2, F9, F10, F11, F12 or ESC. - depending on the computer.
15. Once in BIOS go to an area where you can select the Boot Priority, change that to have your USB drive at the top and HDD next. Go to Exit -> Save & Exit

Let it boot and install.
For the DVD, it's much different, you'll need a blank DVD.

1. Download the ISO file for the distro you want to use, Kubuntu, Ubuntu, Lubuntu, Xubuntu, etc. 32-bit again still recommended.
2. After it downloads double click on it, it should open up a program to burn it.
3. Click on Burn.
4. Leave DVD in and Reboot.
5. Install.

If #2 doesn't open a program to burn it, go to the file manager, go to the file location, right-click and see what options it shows. If it's Ubuntu, it should show Brasero, Kubuntu not sure, Xubuntu Xfburn, Lubuntu not sure again.

### Post by buzzingrobot on 2014-07-20
Rosaleen, have a look at at this page from the Ubuntu site: [http://www.ubuntu.com/download/desktop](http://www.ubuntu.com/download/desktop). 

From there, you can download a good install image for 14.04.  When you cursor down a bit, you'll see instructions for transferring -- "burning"  -- the image to a DVD or a USB stick, in either Ubuntu, Windows, or OS X. If you can get your hands on a DVD, that's probably a bit simpler. 

After that, you need to boot from the DVD/USB stick. You do that by telling your machine's BIOS to boot from the DVD drive or the USB, whichever you're using. (If this is new territory for you, you should get your friend in to help, since that's how the magazine DVD was booted.)

A DVD can take a few minutes to boot, while a USB is noticeably faster.  You'lll see the options to "Try Ubuntu" or "Install Ubuntu".

Printers:  I have an HP printer that requires a driver that must be obtained from an HP site.  Ubuntu detects and recognizes the printer, but it won't work without that driver.  HP makes a little piece of software that it allows to be distributed with Linux distributions that will download and install the correct driver from HP.  If your printer does not work out of the box, I'd suggest turning to Google, and here, for help with a query about your specific model.
