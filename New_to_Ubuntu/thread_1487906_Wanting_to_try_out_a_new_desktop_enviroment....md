Whats the difference between a desktop enviroment and window managers? :p

---

### Post by malspa on 2010-05-19
In a nutshell:  A window manager is more minimal.  A desktop environment is more of the complete package.  For example, a window manager like Openbox doesn't come with a file manager; desktop environments do, like Xfce comes with Thunar, KDE comes with Dolphin, GNOME comes with Nautilus.
