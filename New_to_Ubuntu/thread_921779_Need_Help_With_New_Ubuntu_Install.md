### Post by spiderbatdad on 2008-09-16
Burn the .iso as an image to a cd. Defrag your windows partition. Shut down windows cleanly, and reboot with the disk inserted. Press f12 or enter your bios and set it to boot from cdrom.

### Post by Therion on 2008-09-16
Boot the PC normally
Put the Ubuntu CD in the drive.
When the drive spins up explore the CD using Windows Explorer.

When you look at the contents of the CD, do you see ONE file on it (filename.iso), or do you see several files and maybe a couple folders?
