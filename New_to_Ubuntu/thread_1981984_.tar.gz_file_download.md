As Toporagno mentions, you'll need [WINE](http://www.psychocats.net/ubuntu/wine) to run Windows programs like Ares.  Now might be a good time to also look at some of the alternatives that are native to Linux.  [Clementine](http://www.clementine-player.org/) is a good option.  It is in the repository and can be installed with a few clicks.  Are there specific features in Ares that you are looking to use?
