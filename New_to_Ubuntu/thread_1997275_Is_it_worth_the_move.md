### Post by Bucky Ball on 2012-06-05
Welcome. You can't create more than four primary partitions on any one physical hard drive, period, regardless of your computer's  brand or model.

You create an extended partition and put as many logical drives (partitions) in there as your hardware can handle. Three primary partitions and an extended partition with eight logical partitions inside that is fine. 

Extended partition like a big container for logical drives.

Thank the creators of the third-party, proprietary software for the standard format issue, not Linux. They have no power over that. But a hassle? Not really. Install ubuntu-restricted-extras and you're done for most things. Take about a minute. ;)

I suppose the pay-off for that is you're not dealing with 25 digit authentication keys for the piece of software you bought. Among other things ...

If you fancy a read you might find this interesting:

[http://linux.oneandoneis2.org/LNW.htm](http://linux.oneandoneis2.org/LNW.htm)

Enjoy your investigations and the learning curve.

(PS: One thing you will find is that most people here believe you should use what suits your working style best (OS bashing/flaming is frowned upon); this might mean, for an individual, using a bit of Linux, some Windows, mostly OSx, all in combination or separately. There is always the virtual machine option where you can have them all running in the one install!  Have fun ...)

### Post by Lars Noodén on 2012-06-05
I think you'll like it very much once you try it and find the move well worth it.  There are several ways to be very cautious in making your move.  The least invasive would be to try the live CDs.
