### Post by kestrel1 on 2009-05-24
Possible that the CPU is overheating or the PSU has a problem.
Check you cooling, make sure all fans are clean & running correctly. Ensure that all card (AGP, PCI, PCI Express etc) are seated correctly. Make sure that your ram is seated correctly. If you have more than one stick of ram, remove one & se if the problem continues. If so remove the one that is still there & replace with the other, retest.
CPU overheat or PSU could be the main reasons though.
