### Post by mastablasta on 2012-03-29
WindowsXP? that should be easy and problem free.
 
just to be sure though, first defrag the drive/partition where you plan to carve off the disk space at least 2 times. this will put all data parts on one part of disk
 
then do a chkdsk (check disk).
 
then partition the drive with windows disk manager just to be sure it all goes ok. simply create an empty disk space (unformated). 25-30 GB would do fine
 
then you are set to go, start install and install to largest empty disk space. the rest is clicking the next button and filling in some data.

### Post by Mark Phelps on 2012-03-29
> **ChrisPear said:**
> I would like to be able to boot into windows or Ubuntu and run some of my programs on both.

If, by that, you mean running the SAME program on both Windows or Linux, for the most part, the answers depend on which OS you're running.

While there is Wine (you should read up on this) which will allow SOME Windows apps to run in Linux, it involves a lot of work, and in my OWN experience, was basically a disappointment.

As to the obverse, running Linux apps in MS Windows, the answer is much simpler -- NO.
