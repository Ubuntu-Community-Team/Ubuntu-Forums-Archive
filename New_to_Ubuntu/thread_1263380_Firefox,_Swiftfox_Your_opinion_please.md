### Post by lovinglinux on 2009-09-11
> **eddski said:**
> I started using swiftfox becauses I read it was faster than firefox. Whether it is faster I am not sure, what I really want to know is are they one in the same? Do I need both for either to work? Can I delete one without messing up the other? Which do you prefer?

It's not recommended to delete the default installation of Firefox. It won't hurt to leave it there while you are using Swiftfox or Swiftweasel.

Swiftfox and Swiftweasel perform better than Firefox, because they are compiled from Firefox source code using some optimization techniques. For instance, there are versions of both for your particular processor family. So they run better than Firefox on your machine. 

While both Swiftfox and Swiftweasel are essentially just an optimized version of Firefox, Swiftfox is proprietary. So I would recommend Swiftweasel instead.

For more info visit [**[COLOR="DarkRed"]Firefox optimization and troubleshooting thread[/COLOR]**](http://ubuntuforums.org/showthread.php?t=1193567)
