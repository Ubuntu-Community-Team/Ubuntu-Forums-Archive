### Post by Jan_Erik on 2015-12-16
I am using the machine for a live show playback and I need to disable all system sounds so that they don't interupt playback. I do not find any "Sound" options in System Preferences. What is the best way to disable all system sounds?

---

### Post by howefield on 2015-12-16
Try left clicking on the panel sound indicator, select Sound Settings > Playback > System Sounds and mute by clicking on the sound icon.
