### Post by lovinglinux on 2009-09-19
> **Spiffworks said:**
> First I installed this repository in Synaptic
[http://ppa.launchpad.net/ubuntu-mozilla-daily/ppa/ubuntu](http://ppa.launchpad.net/ubuntu-mozilla-daily/ppa/ubuntu)

That's not a good idea. The ubuntu-mozilla-daily repository will install not only new final versions, but also unstable developing versions. Remove that repository from the software sources, then select **firefox** and **firefox-3.0** packages and select the option to re-install.

Then try to install **firefox-3.5** package.
