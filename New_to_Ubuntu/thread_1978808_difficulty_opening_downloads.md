### Post by Curtis6767 on 2012-05-12
Here's another dumb question, but have you tried to right-click on the file?

.deb files can often be installed via the Software Center. All you do is right-click the file, a menu opens, and you select the method you want to install.

If you use the Software Center to install the file, then you can use the Software Center to uninstall the file at some point in the future.

### Post by Shadius on 2012-05-12
Let's take one file at a time. For the *.deb* file you are trying to open, you can simply double click on it and select Install Package or you can open the file using Ubuntu Software Center. As Curtis6767 said, if you install a file using Ubuntu Software Center, you can remove it using Ubuntu Software Center as well. You can also install the file using the Terminal. See the link. 

[Installing .deb Files]("http://linux.about.com/od/ubuntu_doc/a/ubudg21t2.htm")

### Post by deadflowr on 2012-05-12
Definitely right-click on the deb file. Hopefully it will say open and ubuntu software center. However, there is a chance that it doesn't say anything(just open). This can be remedied by scrolling down to properties. Click on properties and select open with. Click on the select application and scroll down to ubuntu software center. Click, or double click ubuntu software center and it should become the default, or at least a choice, to open the app. from there it should be straight forward.
