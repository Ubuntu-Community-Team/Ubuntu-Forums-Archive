### Post by Kellemora on 2008-11-19
Hi Xieu

If they are NOT RUNNING, they are just using up a little hard drive space is all.

I'm from the opposite camp I guess, when I install a Distro, If it has the install everything option, I select that.  That way the package is there if I should ever need it.  Saved my can several times doing it that way too, like when cable was down and I needed a package to finish a project I was working on.  It was there!

Now back to your problem:  You can uncheck something in Synaptic and then say remove all associations or whatever comes up, complete removal.  Then see if something stopped working right.  If so, uninstall and reinstall it and it will reinstall the dependencies required for that part, without all the bloat of the other dependencies it didn't use.
