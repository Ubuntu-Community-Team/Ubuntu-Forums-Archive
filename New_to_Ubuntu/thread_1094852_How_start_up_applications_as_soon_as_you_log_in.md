### Post by WhiskyChris on 2009-03-13
Go to System - Preferences - Sessions and you will be taken to a list of programs that start up when you login. Click on Add to add a new one and give it a name and command. For instance if you use the Nautilus file browser you would type nautilus in the command box.

Pressing Alt-F2 will allow you to test the command you think is correct (for me I just need to type nau and it auto-completes to the full name required for the command).
