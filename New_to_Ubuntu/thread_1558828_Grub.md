### Post by ranch hand on 2010-08-22
You can run that boot info script from the Live CD.   I suspect that you do not have grub properly installed on the MBR.

There were a number of bugs in grub at release time and I do not think they have been entirely fixed in 10.04.

I am running 10.10 testing and grub here is a newer, better version without those problems.

The boot info script is safe to use, written by some very fine grub folks here on these forums.  It is used a lot for these problems.  It is very easy to use.  The directions are on the page linked by cavalier911.

Please use it.
