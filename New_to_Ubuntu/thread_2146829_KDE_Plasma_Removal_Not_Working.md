### Post by lykwydchykyn on 2013-05-21
In *theory*, all you should have to do is this:
```

sudo apt-get purge kde-plasma-desktop
sudo apt-get autoremove

```

If you explicitly installed any other KDE packages, that might complicate things.  But the idea of "autoremove" is that it's going to remove all the automatically-installed dependencies when the explicitly installed package that caused them to be installed is removed.
