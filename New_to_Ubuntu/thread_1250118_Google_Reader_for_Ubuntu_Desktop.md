### Post by mathay on 2009-08-26
There are many RSS feed aggregators out there. [Liferea]("http://liferea.sourceforge.net/index.htm") is one such aggregator for Gtk. If you use KDE, try [Akregator]("http://akregator.kde.org/"). 

All you should need to do to sync is go to the Good Reader settings, go to the Import/Export tab and then, "Export your subscriptions as an OPML file." Save the XML file and then just import them into whatever client you want to use.
