### Post by ajgreeny on 2013-04-28
Remember this is not Windows, so you do not need to go searching on the web for applications; you just install them from the repositories, using either ubuntu software center, the terminal with ```
sudo apt-get install <packagename>
``` or, my preferred way, using synaptic-package-manager.
