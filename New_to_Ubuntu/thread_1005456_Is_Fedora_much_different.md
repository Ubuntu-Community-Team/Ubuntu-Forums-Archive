### Post by PmDematagoda on 2008-12-08
Fedora is not completely equal to Ubuntu, however it still uses the same terminal commands(it is a Linux distro!:)). One difference you may notice though is that certain commands like fdisk, mostly administrative commands are unavailable as a normal user. In order to use the commands, you need to switch to the root account with:-
```
su -
```
However it is something that you get used to.

Another difference is the package management, Fedora uses rpms instead of debs, however it does have Synaptic/Update Manager like tools, so it won't be that different either.

In my opinion, Fedora is pretty good, especially since they tend to give you a bleeding-edge distro while still remaining pretty stable, though perhaps not as stable as Ubuntu.

### Post by opscure on 2008-12-08
Fedora has subtle differences, in that, the tool set provided is a bit different.  If you are a heavy terminal user this will be much more apparent.  Fedora's package manager is yum as opposed to apt-get in debian-based systems.  Also searching you would use slocate as opposed to mlocate, etc.

All and all, you should be able to find your way around the OS, but you may need to do a few google searches to get you rolling.
