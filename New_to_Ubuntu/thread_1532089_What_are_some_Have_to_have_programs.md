### Post by k3lt01 on 2010-07-16
VLC for multimedia (I find Totem is getting finnicky with what it will play while VLC will play everything I throw at it).
GoogleEarth (I'm a teacher so it's a brilliant tool for me).
Fieldworks (SIL Linguistics/Anthropology program just released to Ubuntu and is only at Alpha stage but I love it).
Firefox and Chromium, each has its own use for me as I keep my Automotive stuff (email etc) on Chromium while everything else is on Firefox.
Evince, PDF viewer and is part of the standard install.
Gbrainy, brain twister game part of the standard Lucid install and the only game I have on my laptop.
Pidgin, because Empathy has some pretty serious security flaws which I have informed Gnome about.
Skype, I have friends who only use Skype so I have it for them.
Meld Diff Viewer, I find this while trying to figure out how certain applications changed with newer releases. I use it often now.
GtkPod, the only application I have found that will load stuff onto my 6th Gen iPod 120gb Classic.

That's my must have list on my laptop. My desktop is set up as an entertainment centre so I can watch Dvds etc. so I would add DVD::Rip to that list.
