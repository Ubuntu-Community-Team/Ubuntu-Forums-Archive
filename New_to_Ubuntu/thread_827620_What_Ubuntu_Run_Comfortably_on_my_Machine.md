### Post by eriqjaffe on 2008-06-12
The RAM could probably be an issue, and will definitely not allow you to install using the standard Live CD (although you could install using the Alternate CD).

Other than that, I would imagine that 8.04 would run quite nicely on that hardware, although not knowing anything about the video card, I can't say if all the compiz glitz will run well.

Also, if it's a laptop, I'd try to determine what chipset is used by the onboard WiFi, as that tends to be one of the trickiest bits of hardware to get working in Ubuntu (well, any Linux distro).
