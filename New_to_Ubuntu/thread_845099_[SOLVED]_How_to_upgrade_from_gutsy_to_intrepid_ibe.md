### Post by sayakb on 2008-06-30
Since Intrepid is still in testing phase, upgrading to Intrepid Ibex from Hardy (or Gutsy->Hardy->Intrepid) may break your system. If you want to try out Intrepid, install it on a separate partition. Download the daily build here: [http://cdimage.ubuntu.com/daily/current/](http://cdimage.ubuntu.com/daily/current/)
