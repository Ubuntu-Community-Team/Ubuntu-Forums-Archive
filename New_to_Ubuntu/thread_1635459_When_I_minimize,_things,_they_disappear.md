### Post by ericstrom on 2010-12-01
Running Lucid for a while now and no problems. In the past, if I minimized something, it would show up as an icon in the lower panel. Now if I minimize (google chrome browser as one example), it just seems to disappear. My guess is it's still running, I just don't know where it is.

How can I tell if something is still running ?

How can I get back to the point where I minimize something and it keeps running AND shows up in the lower panel ?

---

### Post by qamelian on 2010-12-01
Right-click on the panel and select Add to panel. From the selection box that pops up find the applet called Window List and either double-click it or drag it to the panel.
