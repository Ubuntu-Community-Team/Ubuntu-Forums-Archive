### Post by CatKiller on 2009-04-15
In general, Ubuntu programs are kept stable for each release. Bugfixes and security patches are backported, but new versions with new features are not released until the next release of Ubuntu. There are other distros that regularly release bleeding-edge software, but Ubuntu isn't one of them. Some projects host their own repositories so that users can run a stable OS but have that one application updated to the newest version. Wine does this, for example. But unless you add in some software source other than the Ubuntu default, you won't get feature updates. Just the security ones.
