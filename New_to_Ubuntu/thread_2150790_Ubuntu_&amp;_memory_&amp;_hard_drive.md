### Post by Paqman on 2013-06-02
Should be ok. I run Ubuntu on an Atom-powered netbook that has 1GB RAM, and it's fine. If you want the regular Ubuntu desktop I'd suggest downloading the LTS version (12.04) and using the Unity 2D option from the login screen (click the icon next to your user name).

Alternatively you could try out one of the lighter alternative desktops such as Xubuntu or Lubuntu.

I'd say download them all, stick them on a USB stick or CD and give them a test run to see which you like.
