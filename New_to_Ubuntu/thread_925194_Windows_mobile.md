### Post by sayakb on 2008-09-20
Try installing the PC Suite (Motorola Phone Tools Software) using Wine. If you don't already have wine, at a terminal, type in:
```
sudo apt-get install wine
```

Note, not all apps work with wine. See: [http://appdb.winehq.org/](http://appdb.winehq.org/)
