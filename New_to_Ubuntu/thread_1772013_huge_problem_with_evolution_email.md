### Post by Zill on 2011-06-03
> **Loren Bliss said:**
> ...no problem whatsoever signing into Qwest email, nor has there ever been. Qwest (q.com) is what I've been using since I gave up on Evolution two days ago...
OK, thanks for that.  It is reassuring to know that your username and password is accepted by the Qwest server.  Now we need to check the settings in Evolution to make sure it sends the same data.

1)  Open Evolution and click on Edit, Preferences.
2)  Click on Mail Accounts, highlight your Qwest account and click Edit.
3)  On the Account Editor screen ensure the Identity tab is selected.
4)  Ensure that "Full Name" and "E-Mail Address" fields are correctly completed.  Note that the "E-Mail Address" field must show *exactly* the same address you use on the Qwest webmail screen - same case, no extra spaces or other characters etc.
5)  Select the Receiving E-Mail tab.
6)  Ensure the Server Type shows POP
7)  Server should be "mail.q.com"
8)  Username should be exactly the same as your email address above.  (Unless Qwest have given you a different username as well as an email address!)
9)  Security should show "No encryption" (Unless Qwest have specifically advised you to use encryption).
10)  Authentication should show "Password".
11)  Ensure Remember password box is checked.
12)  Click OK and exit preferences.

With Evolution set-up as above, click on the Send/Receive button on the main screen.  Evolution should ask for the password once then remember it for future use.  Any mail on the server should be received.

Please check this and see what happens.  Don't worry if errors occur about sending messages - we can set this up later.  Just see if you can click on Send/Receive without any POP errors.  Feel free to post a screenshot of your settings screens or any errors if this would help.

ps.  I have made some assumptions above about your account - if you do need non-default settings, such as encryption, then please advise.
