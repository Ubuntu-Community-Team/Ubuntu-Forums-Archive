### Post by SeijiSensei on 2014-07-05
If you've never used Ubuntu before, and your Win8 machine has 4 GB or more of memory, I suggest taking the virtual machine route to start.  Install the Windows version of [VirtualBox]("http://www.virtualbox.org/") along with the "Extension Pack."  Then create a virtual machine and install Ubuntu into it.  VirtualBox has simple "wizards" to handle these tasks, and you'll get a full-fledged version of Ubuntu to test out.  If you like it, you can consider options like dual-booting.

### Post by Mark Phelps on 2014-07-05
A major part of the limitation of trying to use Wine (or its companions) to play Window games is that Wine is unable to use the Windows video drivers, and that Linux video driver support is sometimes not up to the level of Windows video support, resulting in poor game performance.

In addition, Hybrid Graphics, which is common on prebuilt PCs that are used for Gaming, is not well supported in Linux, again, a driver problem.

And finally, the two apps that are most commonly used in MS Windows (Internet Explorer and Microsoft Office), work very poorly in Wine -- except for very old versions.

So basically, the more you depend on Windows stuff for daily use, the more disappointing will be your switch over to Linux.
