### Post by germanix on 2010-05-10
A really good book that I can recommend would be: "Beginning Ubuntu Linux" by Keir Thomas and Jaime Sicam which is a 730 page complete guide to Ubuntu. You will probably find it on Amazon or look at [www.apress.com](www.apress.com) as the book is published by Apress. I bought this book  in the third edition which was aimed at Ubuntu 8.04 LTS for then $39,99. This book is updated with every new version of Ubuntu. I do not know if the latest version of the book already cover 10.04 but all of the editions cover all of the basics one needs to know.. Also check ebay as you sometimes find new copies slightly shop soiled at reduced prices.
