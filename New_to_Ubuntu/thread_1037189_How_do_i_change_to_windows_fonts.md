### Post by jimmy the saint on 2009-01-11
For many of the sytem fonts go to system>preferences>appearance and click on the "fonts" tab.

For more detailed control install "gnome color chooser" with synaptic.  Be careful with gnome color chooser though, it has a lot of options and it is easy to get lost!
