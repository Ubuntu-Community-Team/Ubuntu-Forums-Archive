### Post by Armor Nick on 2009-02-15
If you want to use the windows/super key as a modifier (i.e. Win+E), then you'll have to enter gconf-editor;
Go to apps -> metacity -> keybindings
and type <super>E to create your keybinding.

Since you posted this in "Absolute Beginner Talk" Let me elaborate on LordIshael's info:

Go to System » Preferences » Keyboard Shortcuts
and you can define the "super" key (AKA "windows key") do do whatever you would like- Run programs, change desktops, etc. 

When he is referring to Compiz shortcut keys, those are shortcuts that are defined in the "CompizConfig Settings manager" Which you may have if you have turned on your shiny desktop effects. 

I think there are several tutorials on Compiz located in the how-to sections of this site.
