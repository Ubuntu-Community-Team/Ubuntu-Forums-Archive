### Post by yetiman64 on 2010-03-28
> **shazbut said:**

Yeah, that's not bad either. Thing is, all combos of view style and size can be set here. Rather than doing the proceedure above you can also "unhide" the file management icon on the system>preferences menu by right-clicking on the applications menu then choose Edit Menus, goto > System >Preferences and tick the File Management box. This puts an icon in the system > preferences menu for easier access to the settings at a later stage if needed.
