### Post by akiratheoni on 2008-06-15
Here's what I did. Go to Applications -> Accessories -> Terminal and copy each line one by one:

```
mkdir Applications
mkdir Applications/BitchX
cd Applications/BitchX
wget [http://www.bitchx.com/download/BitchX-1.1-final-linux.tar.gz](http://www.bitchx.com/download/BitchX-1.1-final-linux.tar.gz)
tar xvf BitchX*.tar.gz
./BitchX
```

That's what I did. It's all in a terminal window though and I dunno what to do but I'm guessing it was running :)

Hope this helps.
