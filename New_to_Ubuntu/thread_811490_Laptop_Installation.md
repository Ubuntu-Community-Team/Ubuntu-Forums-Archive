### Post by CameO73 on 2008-05-29
I assume you're talking about the Hardy Heron (8.04) LiveCD? If not, [download that first]("http://www.ubuntu.com/getubuntu/download") and try again.

You normally have two options to install: run the LiveCD and click on the Install-icon, or immediately select the Install option in the LiveCD boot menu. Do both options fail? Any messages or strange things happening (like flashing screen, text scrolling etc.)?

If the normal LiveCD fails, you can try the Alternative installation CD (get it [here]("http://www.ubuntu.com/getubuntu/download") ... remember to check the box next to the 'Check here if you need the alternate desktop CD.'-text). Check [these pages]("https://help.ubuntu.com/community/Installation") if you need more information.
