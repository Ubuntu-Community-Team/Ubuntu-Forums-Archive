### Post by mcduck on 2010-12-09
Yes, you can. Just like Windows programs, Windows viruses can't run on other operating systems.

(The last time I've heard of anybody testing this, even Wine didn't run most of the viruses correctly :D)

Besides, Linux has quite a bit more secure way of handling executable files, so even if the virus would be able to run on Linux you'd have to save it to your computer, make it executable, and then launch it yourself. And even then it would only have access to your home directory, and thus couldn't affect the rest of the system.
