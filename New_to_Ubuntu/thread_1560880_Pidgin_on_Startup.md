### Post by stoogiebuncho on 2010-08-25
If you open up System > Preferences > Startup Applications, you'll see that there is a "Options" tab that allows you to remember the currently running applications.  If you use that option, it stops using the list of startup programs and uses the list it created when you pressed the button instead.

I'm sure there's a way to get it to use the startup programs list again, but I've never had much luck with it.  What I would do is to open all of the programs you want to start on startup (and close everything you don't want to run on startup), and press that button again.  It should remember and load all those programs the next time you start.
