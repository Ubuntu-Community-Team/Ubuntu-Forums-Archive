### Post by giddyup306 on 2010-09-17
I figured it out just in case someone else runs into this problem.  I never did find an answer online, but I found it out through trial and error.

Pause all downloads then right click on set location.  Then mark local data is already there.  Then click on apply.  It will then go through the verification process.
