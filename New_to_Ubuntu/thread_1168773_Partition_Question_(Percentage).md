### Post by BandD on 2009-05-24
Well technically you can change it at anytime.  But messing with partitions is somewhat risky--there is always a chance you could wipe all the data on your drive (which would mean that you'd have to reinstall XP and Ubuntu).  So make sure you have everything backed-up first!!!!  Also make sure that you create any system restore CDs that may need to be created on the XP side of things before you install ubuntu.

You should also do a disk defragment on your XP install (maybe run it twice) before you mess with the paritions.  This is because NTFS doens't really oraginze files on the physical space on your hard drive very effeciently and so some files might need to be moved closer together so that you don't have to worry about them getting deleted.  

But if you let us know how big your hard drive is and how much space you already have used up, we'll be happy to give you our two cents about the size of each partition.
