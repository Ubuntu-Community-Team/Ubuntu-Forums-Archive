### Post by mike g on 2008-06-30
Ok, I stumbled on the answer myself, so far so good.

I will list here what I found, I hope you can follow this.

Even though I thought I set Firefox as my default browser it apparently was changed when I upgraded to 8.04.  I had to reset the default browser to Firefox.  

In 8.04 I did it as follows.....System > Preferences > Preferred Applications and then chose Firefox as the default browser.  I saved my changes and now everything seems to work.  (I hope, so far so good)
