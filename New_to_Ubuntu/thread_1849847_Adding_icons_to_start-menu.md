### Post by Birchle on 2011-09-25
> **cheatos said:**
> Can someone still tellme how to get a nice icon for that? Now it has some system default icon or so...
Go back into the menu item's properties (it should be the same screen you saw when you added the menu item) and click the default icon.  It'll pop up a window for you to browse to the image you'd rather use.  Select one, and it should be applied to the menu.
