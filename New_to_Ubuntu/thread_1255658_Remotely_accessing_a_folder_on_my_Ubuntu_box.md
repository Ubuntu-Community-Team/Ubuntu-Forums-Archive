### Post by ayenack on 2009-09-01
This can be done but there are so many pitfalls that it's hard to accomplish without either being on site or getting a pro to do it for you.

What I would suggest is signing up for some secure and encrypted remote storage and sharing the file that way. If you do a search in google you'll get loads of hits for on-line storage.
