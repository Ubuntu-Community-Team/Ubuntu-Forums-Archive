### Post by Hospadar on 2009-05-05
second that.

I also really like dvd::rip, its installable through the package manager.  There are many others as well, many of them available through the package manager.

Note also that if you want to rip encrypted dvd's (e.g. every dvd you didn't burn yourself) you'll need to install libdvdcss from the medibuntu repositories.

Follow this guide on adding their repos:
[https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)
then use synaptic to install "libdvdcss2"

### Post by Junkieman on 2009-05-05
Try some linux rippers :)

dvd::rip is great for ripping DVD program chains (the ifo,vob files)

acidrip is a all in one DVD to AVI ripper and converter

Thoggen is very easy to use, and rips to OGV files

TIP: Play the DVD first to unlock the drive before you rip :D
