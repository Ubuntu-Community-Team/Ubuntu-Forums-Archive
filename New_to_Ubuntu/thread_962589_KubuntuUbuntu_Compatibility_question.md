### Post by Titan8990 on 2008-10-29
Yes, this is true. They are both just different desktops that use different Window managers. As long as you have the dependencies for a say, a Ubuntu (Gnome) program then it will run under any window manager.


To install KDE desktop:


sudo apt-get install kubuntu-desktop
