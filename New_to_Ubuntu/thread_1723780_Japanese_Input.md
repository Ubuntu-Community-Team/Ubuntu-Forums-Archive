### Post by rosencrantz on 2011-04-07
Start System Settings. Go to Input Devices->Keyboard->Layout tab.
At the bottom of the window is the layout selection, add a Japanese keyboard. You can also define a global shortcut (upper right).
See the attached screenshot.
