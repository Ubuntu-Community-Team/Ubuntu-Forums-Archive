### Post by Elfy on 2009-02-10
You should be able to make a launcher - right click desktop - Create Launcher

You might need it to be an Application in Terminal from the drop down list - in the command box - put the path as well as the commnad you have been using

/path/to/folder/java -jar ted.jar

Try that - to start at boot - add the same to a new entry in Sessions - Admin - Preferences menu
