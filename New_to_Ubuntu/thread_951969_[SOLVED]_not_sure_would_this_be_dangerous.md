### Post by jerome1232 on 2008-10-18
add/remove only lists some programs, try synaptic (System-Administration-Synaptic Package Manager) it's a more advanced package manager than add/remove is. If you want to launch it, you can probably just type 'gufw', it will need root priviliges to configure the firewall though. If you wanted to remove it synaptic or (sudo apt-get remove gufw) should remove it.

If you want more info about gufw here is their website [http://gufw.tuxfamily.org/index.html](http://gufw.tuxfamily.org/index.html)

Well really it's not needed, by default Ubuntu has no programs listening for connections to begin with so all incoming traffic is dropped.
