### Post by sudodus on 2014-05-09
The installer can work from and to USB as well as from any other mass storage device (internal/external, IDE/SATA, HDD/SSD/flash drive). The requirement is that the computer's BIOS/UEFI system will recognize and accept booting from it.

See  [Installation/FromUSBStick]("https://help.ubuntu.com/community/Installation/FromUSBStick") which slickymaster already referred to.
