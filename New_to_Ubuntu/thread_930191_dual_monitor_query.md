### Post by Excalibre on 2008-09-29
I'm not positive exactly what you're asking but I think you're wondering if you can have two different virtual desktops on two monitors? So that you could switch them independently, and move one virtual desktop to another monitor, and so forth? If that's what you want to know, the truth is no, you can't. Your basic choices are to have one virtual desktop that's stretched across two monitors (so that you can drag application windows from one monitor to another, etc.) or two completely separate X sessions, which means you won't be able to move things from one monitor to another at all.
