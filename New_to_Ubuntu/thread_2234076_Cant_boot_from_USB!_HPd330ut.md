### Post by yancek on 2014-07-12
Are you having this problem with both of the HP computers or just one?  To boot from a usb/flash drive, you need to set it to first boot priority in the BIOS.  On my HP, there are multiple options to do this but only one of them works.  Check to see what usb options you have and if more than one, try them all.  Then again, it could be bad hardware.  You could try booting it from another machine if you have one available.
