[FONT=arial,helvetica,sans-serif][SIZE=2]**Why is the emulator running too fast?**
If the emulator is running too fast, make sure that the Synchronize...  option is checked on the Options->Emulator menu and that Sound is on.  If sound is off, there will be no synchronization and the emulator will  run at maximum speed. This should change in the future.[/SIZE][SIZE=2]   If you want no sound, mute the sound instead of turning it off.       [/SIZE]
[/FONT]
