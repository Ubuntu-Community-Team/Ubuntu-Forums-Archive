### Post by 3rdalbum on 2010-06-15
If you install the "ubuntu-desktop" package on your existing Netbook Edition system, you should get an option at the login screen to use regular Gnome as opposed to the netbook interface.

Ubuntu Desktop works fine on netbooks.

I use Docky as the dock.

### Post by Peter09 on 2010-06-15
It depends how you want to use it - I think netbook remix is very good for small low powered machines where you only have one or two windows open, say browser and email. It most probably is not so good for viewing multiple windows at once.
 
The good thing about it is that you do not need to struggle with small textual menus or menus that are so long they extend out of screen.
 
I personally really like netbook remix for my aspire one.
