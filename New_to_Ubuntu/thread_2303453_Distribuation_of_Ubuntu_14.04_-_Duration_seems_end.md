### Post by deadflowr on 2015-11-19
Sounds like there is a window hiding somewhere.
Normally if it hangs for a while it is because it needs user input to proceed.
This usually comes when the final stage arrives.
At this stage it needs to know if you want to remove old packages or keep them.

If normal Ubuntu look at the launcher and see if any extra launchers show, and if any of the launchers the are running might have an extra arrow and the left side (the will be really small)
Running apps will always have at least one arrow on the left side iof it's launcher If the launcher has 2 or more then the app has extra windows open somewhere.
Clicking on the launcher should expose all windows for the app if more thasn one window is open. You can then select the window you want to look at from there, by clicking on that window.

Hope it helps, or make sense.
