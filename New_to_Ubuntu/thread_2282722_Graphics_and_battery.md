### Post by Bucky Ball on 2015-06-16
*Thread moved to **New to Ubuntu**.*

Welcome. I suggest you install 14.04 LTS and post a new thread if you have any problems with your battery or graphics. You haven't used Ubuntu for a long time and a LOT has changed.

You may need to install the ATI driver, which is pretty easy now and we can help with that, or the open-source ones may work fine. 

How much RAM does that machine have? If it is an older machine, you might like to try something lighter like Xubuntu or Lubuntu. May improve the experience.

Download an .iso, burn it to disk or USB, boot from the media and 'Try Ubuntu'. See how it runs. If all good, install (with the icon on the desktop).  

Good luck with it.

PS: Backup anything you don't want to lose prior to installing and confirm your plan is sound. :)
