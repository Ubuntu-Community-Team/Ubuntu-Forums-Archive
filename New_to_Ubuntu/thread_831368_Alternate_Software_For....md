### Post by ad_267 on 2008-06-16
RAR archive support comes in the ubuntu-restricted-extras package I believe. See [http://help.ubuntu.com/community/RestrictedFormats](http://help.ubuntu.com/community/RestrictedFormats)

For 7z support you need to install p7zip of p7zip-full and then the built in archive manager in Ubuntu can handle these.

Most of the software you mentioned comes with Ubuntu already including a BitTorrent client, OpenOffice, PDF reader, CD/DVD Burner, Instant Messenger, Audio Player. This isn't like Windows where you need to install a whole lot of extra stuff.

Just run the live CD and give it a go.

If you're using MSN you may want to install emesene instead of Pidgin which is included with Ubuntu as emesene has better support for MSN.

### Post by RomeReactor on 2008-06-16
Hi.You probably don't need an antivirus, unless you want to scan a windows partition or scan email attachmetns before sending them to windows-using friends.

> **Johnathannn said:**
> An alternate program for WinRAR (OR if there's some kind of compression utility for linux)
Ubuntu has an archive manager; install **unrar**, **p7zip** and **p7zip-full** to get it to recognize more compressed files. Look for those programs in Synaptic (System->Administration->Synaptic), or to install them from the terminal (Applications->Accessories->Terminal):
```
sudo aptitude install unrar p7zip p7zi-full
```

> Tune-Up Utilites
Not really needed. You can disable services you don't want in 'System->Preferences->Sessions', though.

> ALL SLYSOFT PRODUCTS (AnyDVD, Clone DVD, etc...). PowerISO
Don't have experience with those.

> LimeWire PRO
You can get LimeWire from [their site]("http://www.limewire.com/download/version.php"); you should give [FrostWire]("http://www.frostwire.com/") a try before. I think you need Java to run either of them, so search in Synaptic for:

* sun-java6-bin
* sun-java6-jre
* sun-java6-fonts
* sun-java6-plugin

Or to install those from the terminal:
```
sudo aptitude install sun-java6-bin sun-java6-jre sun-java6-fonts sun-java6-plugin
```
As for these programs:
> Winamp

Windows Live Messenger

Bittorrent

Nero 8

Microsoft Office

Adobe Reader
You should try the programs already installed in your system before looking for other alternatives:

Winamp->RhythmBox (in 'Applications->Sound & Video')

Windows Live Messenger->Pidgin (in 'Applications->Internet).

Bittorrent->Transmission (in 'Applications->Internet)

Nero->Brasero (in 'Applications->Sound & Video); or, if you just want to burn data, load an blank CD or DVD in Ubuntu, double-click on the disc icon in your desktop and drag & drop files there, then press 'Burn to disc'.

Microsoft Office->Open Office (in 'Applications->Office)

Adobe Reader->Evince (you launch it by double-clicking a PDF file).
