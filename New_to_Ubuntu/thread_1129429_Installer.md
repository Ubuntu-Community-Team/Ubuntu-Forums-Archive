### Post by PacSci on 2009-04-18
Right-click the installer, select "Properties", then select the "Permissions" tab. At the bottom, there is a checkbox next to the "Execute" label that says, "Allow this file to be executed as a program." Make sure that it is marked, then try again and click "Execute" when the dialog pops up.
