### Post by Wizgot on 2011-08-11
When I right click on the desktop I get create "create Document". I would expect to find a drop down menu which lets you select a libre office document, spreadsheet, ect. This is not the case however because I get: No templates installed. How would I install these templates?

---

### Post by AlphaLexman on 2011-08-11
Put your files in the ~/Templates directory, then when you right-click your Desktop, they will show up in the 'create document' sub-menu.
