### Post by patchwork on 2010-03-21
The desktop livecd supports laptops.  What is the exact problem you are having?

If the cd is not booting, have you changed your bios settings to allow boot from cdrom before the hard drive?
Did you check the md5sum of the downloaded ISO?
If the cd is booting, have you checked the disk for errors?

Are you trying to install or run the live cd?

Also, is this the 32 bit or 64 bit version (your laptop will support either)?

If the cd is booting, but the screen goes blank, can you access the command line by pressing CTRL ALT F1?
