### Post by crazy bird on 2011-10-21
Hi there and welcome!
 
First recommandation is to install all necessary multimedia packages such as ubuntu-restriced-extras (can be found in synaptic: System>Administration>Synaptic packagemanager) and by adding the repository of Medibuntu ([http://medibuntu.org/repository.php](http://medibuntu.org/repository.php)). I found this site on the internet: [https://sites.google.com/site/tipsandtricksforubuntu/how-to-add-multimedia](https://sites.google.com/site/tipsandtricksforubuntu/how-to-add-multimedia), this will help you installing all necessary multimedia.
