### Post by thatguruguy on 2011-10-07
> **doppel.ganger said:**
> How would I edit the following image using GIMP to remove just the unnecessary  blankness and leave behind the logo?

Use the selection tool to draw a square around the circle, as close to the circle as you can get.  Then, right-click anywhere within the selection. A menu should pop up.  Choose "Image", then "Crop to Selection."  That's about as good as you're going to get, because there's no such thing as a round image.  They're always rectangular.

HOWEVER, that is a trademark. It's owned by someone.  Make sure you have permission before you use it for anything.
