### Post by Anuovis on 2009-12-26
In case Wine fails to do what you want...

If you can spend some money, you might check Cedega. It is a proprietary alternative to Wine.
If you have good PC specs and tons of RAM, you may try VirtualBox which emulates the whole Windows OS inside Ubuntu. Very resource hungry though.
