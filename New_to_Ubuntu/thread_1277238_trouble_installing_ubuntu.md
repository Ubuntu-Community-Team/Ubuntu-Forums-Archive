### Post by HereInOz on 2009-09-28
I am assuming that you downloaded the Desktop Install.  While this is a wonderful thing, and there are many other uses for the Desktop Install CD, I would recommend, assuming you want to install Ubuntu on your machine, and not just boot from the CD to see how it looks, that you download the Alternative Install rather than the Desktop Install.

I have had some machines which would absolutely refuse to load up the Desktop Install, but when the Alternative Install CD was used, it all went well.

I am not a real guru on the Desktop Install, I stopped using it some years ago - see above for the reasons - but I think that there are options to enable basic VGA mode, and things like that which can help in some circumstances.  Hopefully someone who knows the Desktop Install well will clarify these a little.
