### Post by 3rdalbum on 2011-11-27
Facebook has disabled its Vv (Voice and Video) chat for anyone not using Windows. It's actually powered by the Skype software so it's not simple to make a plugin.

Annoying.

You might be able to use Meebo ([www.meebo.com](www.meebo.com)) to start a Vv chat with a Facebook user, but I'm sure they don't do the "native" Facebook Vv chat. Instead it sends a link to the other user in a chat message, which then opens up a mini instance of Meebo when that user clicks it.
