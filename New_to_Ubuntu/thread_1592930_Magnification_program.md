### Post by robsoles on 2010-10-11
Welcome to UF.

Will a magnified patch around your mouse cursor do?

My boss preferred zooming the entire desktop in and out using [Windows]  key + mouse scroll wheel, before his eye operation, when I showed him the  magnified patch for his mouse cursor.

Have you installed compiz? ('System'->'Preferences'->'CompizConfig  Settings Manager' if installed, if not then open software centre and  type 'compiz' into search box).

Under Accessibility look for 'magnifier', check it's box, if a dialog  box opens then read it and choose the option that sounds like it enables  the magnifier. It doesn't dock (that I found) but it can be set up with shortcut buttons to turn it on and off.

Any good?
