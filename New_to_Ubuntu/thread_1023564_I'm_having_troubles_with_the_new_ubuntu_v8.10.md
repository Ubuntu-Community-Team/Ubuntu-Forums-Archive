Pls do the following changes

```
sudo gedit /etc/network/interfaces
```
```

auto lo
iface lo inet loopback

iface eth0 inet static
address 192.168.10.52
netmask 255.255.255.0
gateway 192.168.10.1

auto eth0
```

Replace with ur n/w settings.

Is 10.0.0.2 ur DNS server? If not, change the value.

Restart n/w services.
```
sudo /etc/init.d/networking restart
```

Now check if ur able to ping ur Gateway, DNS server and any public IP.
