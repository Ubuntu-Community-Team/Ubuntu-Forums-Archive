### Post by Peripheral Visionary on 2012-05-23
LXDE is an ultralight *desktop environment*. Lubuntu is not just "Ubuntu with LXDE tied on." It has specially chosen lightweight *applications* to go along with the lightweight desktop environment.  Ubuntu with LXDE would be "slower" because of the default *applications* used in Ubuntu. The majority of memory usage comes from applications, not as much from the d.e.
