### Post by Jon Monreal on 2010-04-24
Okay, in the menu go to System -> Administration -> Synaptic Package Manager. When it loads, click on any of the entries in the large pane, and just start typing libgpod.

You should see "libgpod-common" and "libgpod4". Click the boxes to the left of them and select "Mark for installation".

(And yes, to anyone who reads this, I know this can be done from the terminal much more easily, but I think introducing a new user to Synaptic is important.)

Since you're new to Ubuntu, is there anything else you need help with now, such as installing Flash? Please feel free to ask.

EDIT: Did you install gtkpod from the terminal or Synaptic (repositories) or from the site?

Also, I believe libgpod is already installed in recent versions of Ubuntu, so you should be able to just use Rythmbox (Applications -> Sound & Video -> Rythmbox) right away if you want to unless you would rather use gtkpod.
