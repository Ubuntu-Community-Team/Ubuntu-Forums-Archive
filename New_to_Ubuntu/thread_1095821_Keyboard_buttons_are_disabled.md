### Post by labinnsw on 2009-03-14
Go to System >> Preferences >> Keyboard
Select the layout tab
Select the keyboard model window and look for your keyboard.
Select your vendor and model if they are there.

If you cannot find your keyboard
Use Synaptic to install Keytouch and Keytouch editor.
Use these tools to configure the keys on the keyboard.

I think the additional information was included with the Keytouch installation.
