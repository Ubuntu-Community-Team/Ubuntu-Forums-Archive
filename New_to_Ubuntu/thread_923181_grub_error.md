### Post by hellion0 on 2008-09-18
Grub error 15 = File not found. Chances are, both operating systems are damaged to the point where Grub can't start them. Most of the time, I've seen that error on incomplete and broken Linux installs.

As above, try using your XP disc to restore the Windows bootloader. If Windows fails to load after that, you might have to start from scratch...
