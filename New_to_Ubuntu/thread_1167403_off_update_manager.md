Hi,

If you open up **System/Administration/Software Sources** and go to the **Updates** tab, you can set it to check for updates Daily, Every Two Days, Weekly or Every Two Weeks. You can even turn it off completely (_not recommended_) and not get any updates at all.

You can also set it to Install Security Updates without Confirmation, Download all Updates in the Background, or only Notify About  Available Updates.

As for the text, go to **System/Preferences/Appearance **and then the **Fonts** tab and pick whatever you want from there - there is a whole range of settings to choose from.
