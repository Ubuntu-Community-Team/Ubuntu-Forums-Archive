### Post by Mark Phelps on 2009-10-29
You're going to need a disk you probably don't have in order to restore the Vista MBR.

See the the link below, post #4, for more details.  It has a link to where you can download an ISO image you can burn that will generate a Vista Recovery CD -- which you can then use to restore the MBR:

[http://ubuntuforums.org/showthread.php?t=1198442&highlight=restoring+MBR]("http://ubuntuforums.org/showthread.php?t=1198442&highlight=restoring+MBR")

Many thanks to forum member raymondhenson for this very useful post.
