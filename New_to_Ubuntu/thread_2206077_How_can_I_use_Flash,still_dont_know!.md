### Post by deadflowr on 2014-02-17
Aye, either/or solution will work
The flashplugin-installer package will only install the flash plugin.
Where as the restricted-extras package adds a couple of other things, beside the flash plugin.
If you go with the ubuntu-restricted-extras, note that at somepoint you will be asked to agree to a EULA.
This part stumps people, because it can be hard figure out how to select yes.
to do so, try using the tab key, and the left right keys on the keyboard.
(Spacebar might also help)
