### Post by grahammechanical on 2011-02-25
Right click the network manager icon. Select Edit Connections. Select the Wireless tab. Select your wireless connection. Click Edit. Make sure that you have a tick mark in the box labeled Connect Automatically. Also put a tick mark in the box labeled Available to All Users.

By the way, there is a forum specifically for Networking and Wireless. And Ubuntu is supplied with help documents.
