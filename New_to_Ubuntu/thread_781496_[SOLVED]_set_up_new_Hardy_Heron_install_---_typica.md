### Post by Darkdelusions on 2008-05-04
For your multimedia codecs such as MP3, DVD, Flash, Quicktime, WMA and WMV

Open a terminal and type 

> sudo apt-get install ubuntu-restricted-extras

For graphics effect 

[https://help.ubuntu.com/community/CompositeManager/CompizFusion](https://help.ubuntu.com/community/CompositeManager/CompizFusion)

For Firefox plugs.

This one is completely subjective there are a handful I use. 

[Adblock]("https://addons.mozilla.org/en-US/firefox/addon/1865") Plus: Stops annoying banner adds on webpages.
