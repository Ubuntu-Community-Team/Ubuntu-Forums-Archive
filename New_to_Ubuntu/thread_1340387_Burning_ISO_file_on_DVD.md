### Post by jrrader on 2009-11-28
From the post, I just assumed he/she had the iso image (maybe from MSDNAA/EMLS), a working ubuntu install, and a broken XP install and wanted to burn the DVD from inside Ubuntu.  I download from MSNDAA all the time (always to Ubuntu) extract the ISO files with Wine, and burn them to CD or DVD with brasero.  I've done this with the 64 and 32 bit versions of XP and 7.  Never had an issue.  

If you are keeping Ubuntu, download super grub disk so you can get back into it after you install windows 7 - it's going to write over your master boot record.
