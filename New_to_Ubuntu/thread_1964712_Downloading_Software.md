### Post by na5h on 2012-04-24
> **snowman2795 said:**
> 
Also, having done a bit of reading, once I do install software, do I have a buncha commands I must learn to make this stuff go?

Most of the applications that you will ever need can be easily installed from the Software Center just by hitting the install button (and tapping in your login password).

Many applications that you download manually (from various websites) are also quite easy to install (for example .deb files) without "a buncha commands".

Using the terminal is a very quick way of getting many things done, though...and yes, some programs do require some basic knowledge on how to use the terminal.
