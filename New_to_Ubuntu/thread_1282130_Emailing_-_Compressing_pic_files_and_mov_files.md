### Post by nhasian on 2009-10-04
we need to know what email client you are using.  If you are using thunderbird, then you can use this add-on to resize pictures:

[https://addons.mozilla.org/en-US/thunderbird/addon/11004](https://addons.mozilla.org/en-US/thunderbird/addon/11004)

Resizing video on the other hand is more tricky and quite time consuming if you dont have a very fast CPU.  You will get the best compression ratio if you use the xvid codec.  You can use Avidemux:

[http://fixounet.free.fr/avidemux/](http://fixounet.free.fr/avidemux/)
