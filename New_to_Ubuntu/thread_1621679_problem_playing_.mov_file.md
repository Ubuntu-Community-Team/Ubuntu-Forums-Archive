### Post by akand074 on 2010-11-14
It's likely an issue with codecs. There are quite a bit that aren't installed by default in Ubuntu. I'd install VLC, its my favourite media player and it will install a bunch of codecs with it. Its available in the Ubuntu Software Centre. If you do have a wide range of videos you tend to watch, its probably a good idea to add the [medibuntu]("http://www.webupd8.org/2010/10/medibuntu-repository-is-available-for.html") repository and install the w32codecs or w64codecs as well.
