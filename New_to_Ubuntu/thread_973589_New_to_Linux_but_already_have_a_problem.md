### Post by taurus on 2008-11-06
Be careful if you use windows to download it since winRAR or some other programs would like to unpack it.  So, don't unpack it.  Just save it as an ISO file and burn it as an image file, not a data file.

Unpacking first and then burning it to a CD will cause the CD not to boot.
