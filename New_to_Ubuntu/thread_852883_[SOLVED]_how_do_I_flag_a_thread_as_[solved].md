### Post by iaculallad on 2008-07-08
The thread does not expire by itself, It's the OP's responsibility to mark it as SOLVED when the problem had already been resolved.

To mark the thread as SOLVED, use the "Thread Tools" menu, click on it to drop the menu down and select "Mark this thread as Solved".
