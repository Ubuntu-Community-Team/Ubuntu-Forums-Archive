### Post by arboreal on 2010-07-18
I was having the same problem.  emusicj worked fine for me prior to upgrading to 10.04  It took me a bit to realize that I hadn't reinstalled java.

You need to install a java runtime environment for emusicj to work.  The release notes for 10.04 are recommending openjdk-6, you can find it in synaptic.

[https://wiki.ubuntu.com/LucidLynx/ReleaseNotes#Sun%20Java%20moved%20to%20the%20Partner%20repository](https://wiki.ubuntu.com/LucidLynx/ReleaseNotes#Sun%20Java%20moved%20to%20the%20Partner%20repository)
