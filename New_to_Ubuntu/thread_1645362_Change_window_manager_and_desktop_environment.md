### Post by tkzv on 2010-12-14
Correction.
To switch to another DE I needed to:
1. From the user switching applet, which is also used to shutdown, reboot and chat select switch to another user. 
2. After everything closes and login screen appears, select &quot;Other...&quot; user. 
3. Type the same user name, press enter. 

The lists of available DEs and WMs appear only when the password field appears.

Very counter-intuitive :)

And for some reason choosing LXDE still brings Unity.
Choosing GNOME+OpenBox brings GNOME. But keyboard applet stopped working.
