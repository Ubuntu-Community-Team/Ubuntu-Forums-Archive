### Post by tjwoosta on 2008-05-08
install flash like this 

go to Appications-Accessories-Terminal
then type
```

sudo apt-get install flashplugin-nonfree

```

or alternatively you could find a page that requires flash and then click the notification bar at the top that shows up

or you could go to Sstem-Administration-Synaptic Package Manager and look up flashplugin-nonfree 

(there are three flash players available to you gnash,flashplugin-nonfree, or swfdec player  but the only one i can get to work is flashplugin-nonfree
