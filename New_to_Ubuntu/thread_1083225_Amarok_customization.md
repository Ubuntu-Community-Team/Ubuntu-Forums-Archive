### Post by Xiong Chiamiov on 2009-02-28
Despite what they say, amarok isn't really that customizable.  It uses the same colors as the rest of KDE (the QT colors), and you can find some themes [here](http://kde-look.org/index.php?xcontentmode=55).

If you're getting the wrong lyrics, either you have songs tagged incorrectly, or you're using a bad lyrics provider.
