### Post by westie457 on 2011-06-23
> **brawnypandora0 said:**
> Suppose I only use a USB external harddrive instead of an internal one. Is that problematic for my computer in any way? What if I accidently unplug the USB cable; would that result in an error?

Using an external drive does not cause any problems as in the past I have done exactly that with Ubuntu Studio. Also You can install to a memory stick either as a LIveCD with a persistence file to save docs, settings and things or as a complete system. 

As for accidentally or deliberately unplugging the cable, most definitely not recommended. At best you will mess up part of the file-system. At worst you will mess up Grub and possibly corrupt the boot partition.
