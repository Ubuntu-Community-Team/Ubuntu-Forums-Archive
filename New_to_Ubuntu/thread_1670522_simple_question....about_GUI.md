### Post by themusicalduck on 2011-01-19
If visual effects are enabled then it is Compiz. If they aren't enabled then the window manager is Metacity.

I don't think Gtk is a window manager as such but has to do with designing user interfaces for applications.

Beryl is an obsolete window manager that was merged into Compiz.

What is it you are looking to install from Gnome Look exactly? If it is just a theme, then it is often independent from which window manager you use. If it is an extra addon for Compiz then it can be complicated to install but it really depends which one. Remember, if you search through the Software Center, you can install extra Compiz effects through there.

There's no reason to mess with X for anything from Gnome Look.
