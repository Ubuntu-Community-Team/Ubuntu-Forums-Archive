### Post by SuaSwe on 2012-05-04
When you say you tried to change the file format, do you mean that you formatted the partition? Have a look on [the Community site's page]("https://help.ubuntu.com/community/DataRecovery") to start with; but beware there are no guarantees that you can recover the lost files. Additionally, keep in mind that if you write to the partition (saving new files onto it, etc) you reduce the chance of recovering the files that used to be present there.
