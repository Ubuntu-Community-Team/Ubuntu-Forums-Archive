### Post by RBarak on 2011-10-05
Today I'm writing instructions for Hebrew on Ubuntu 10.10. Thought I'd share...
:)


1. To install Hebrew:
System  - Preferences - Keyboard - Layouts - Add - By language - Language:Hebrew

2. To be able to change from English to Hebrew:
System  - Preferences - Keyboard - Layouts - Options - Key(s) to change layout

3. For Language support - refer to my other Howto at:
[http://ubuntuforums.org/showthread.php?t=1654337](http://ubuntuforums.org/showthread.php?t=1654337)
Good Luck
