> **Zenze said:**
> In the past when using Ubuntu with a Nvidia card I a have always been able to use the Hardware Drivers program to detect and install drivers for the video card. The Nvidia GTX460 was released shortly after Ubuntu 10.04, so it is not known by 10.04's Hardware Drivers program. You should either find a PPA that contains the Nvidia driver for 10.04, or manually install the driver from [www.nvidia.com](www.nvidia.com).
