### Post by ddrichardson on 2008-05-07
You need to add a line to your Xorg.conf that enables shared memory access. First open a terminal

Backup the file:

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.bak

Then edit it:

gksudo gedit /etc/X11/xorg.conf

Find the InputDevice Section that'll look something like this,

Section "InputDevice"
        Identifier      "Synaptics Touchpad"
        Driver          "synaptics"
        Option          "SendCoreEvents"        "true"
        Option          "Device"                "/dev/psaux"
        Option          "Protocol"              "auto-dev"
        Option          "HorizScrollDelta"      "0"
EndSection

Add another Option line that reads:

        Option          "SHMConfig"             "true"

### Post by annda on 2008-05-10
first of all, the touchpad is missing from the server layout section in your xorg.conf hopefully addind this line will help:

```
InputDevice    "Synaptics Touchpad"
```

restart X.

if it still isn't working properly, check if the touchpad is recognized as such:

```
cat /proc/bus/input/devices
```

you should see something like this:
```

I: Bus=0011 Vendor=0002 Product=0007 Version=25b1
N: Name="**SynPS/2** Synaptics TouchPad"
P: Phys=isa0060/serio4/input0
S: Sysfs=/devices/platform/i8042/serio4/input/input8
U: Uniq=
H: Handlers=mouse1 event8 
B: EV=b
B: KEY=6420 0 70000 0 0 0 0 0 0 0 0
B: ABS=11000003
```

it should be SynPS/2, not PS/2
