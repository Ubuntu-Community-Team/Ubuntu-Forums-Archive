### Post by tgm4883 on 2011-03-04
You shouldn't need to be editing root owned files every two minutes. Even if you did, you shouldn't be getting prompted for it every two minutes. 

You can login as root, although we can't show you how and I do not recommend it. I suggest you read [http://ubuntuforums.org/showthread.php?t=1486138](http://ubuntuforums.org/showthread.php?t=1486138)

### Post by maqtanim on 2011-03-04
tgm4883 has already said. Using a root account is not a wise idea. And you do not need to enter password in every 2 minutes. Ubuntu store the password for 15 minutes for root usage. Also apart from installing new softwares, update and uninstall those, you really donot need to access the root privilege. I myself use the root privilege once (or less) per day!

So I strongly recommend you not to use a root account for login.
