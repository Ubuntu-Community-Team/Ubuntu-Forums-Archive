### Post by rg_stephens on 2008-12-24
What connection manager are you using? 

I'm sure you've done a site scan to be sure that no other wireless routers are nearby on the same channel. A lot of things can cause interference, so try different channels. I don't know about this router, but if you go into the setup program, you can change power levels, transmit mode, and force a different connection speed, depending on what firmware you're using. I have no idea if this will fix the connection problem or not. 

If not this, it could be a driver problem
