### Post by az on 2008-05-05
> **shashwat0805 said:**
> i have a windows xp service pack 2 so all i need to lnow is hat can i have ubuntu and xp both simultaneously on my system

Boot the installer and make it install.  If you don't get to option to resize your windows partition, reboot into windows and defragement your drive.  Try the installer again.

By default, if the Ubuntu installer detects a windows system with enough free disk space, it will offer to shrink the windows partition down and install Ubuntu in the freed-up space automatically.
