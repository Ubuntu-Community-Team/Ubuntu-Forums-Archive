### Post by unutbu on 2008-12-13
Do you have a Windows XP CD, or another computer with which you can download and burn a SuperGrubDisk? 

If you have a Windows XP CD, then go here: [http://ubuntuforums.org/showthread.php?t=740221](http://ubuntuforums.org/showthread.php?t=740221)

If you can burn a SuperGrubDisk, then go here to download the CD:
[http://www.supergrubdisk.org/](http://www.supergrubdisk.org/) and here:
[http://www.supergrubdisk.org/wiki/UninstallGRUB](http://www.supergrubdisk.org/wiki/UninstallGRUB) for instructions on how to restore your Windows MBR.

If neither of the above options are available, then you could also perform the following (somewhat long) procedure:
1) Reinstall Ubuntu
2) Use these instructions for restoring the Windows MBR:
[http://ubuntuforums.org/showpost.php?p=4691608&postcount=2](http://ubuntuforums.org/showpost.php?p=4691608&postcount=2)
3) Then use your restoration disks
