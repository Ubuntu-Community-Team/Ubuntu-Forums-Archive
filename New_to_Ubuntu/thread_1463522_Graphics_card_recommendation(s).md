### Post by 3rdalbum on 2010-04-27
Grab a GT210 or an 8400GS. These are both Nvidia graphics cards, but remember when you buy a graphics card the brand name will not actually be Nvidia. It still uses Nvidia drivers, though, that you can install through the Hardware Drivers program on Ubuntu.

Don't bother with any ATI-based graphics cards - if the name contains "Radeon" then forget it.

The GT210 and 8400GS are low-end cards.
