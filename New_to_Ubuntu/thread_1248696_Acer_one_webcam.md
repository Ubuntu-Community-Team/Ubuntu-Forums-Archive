Hi John, 
 
Another option is to try install a custom kernal. There is a guy who has done a custom kernal for the aspire one which is optimised for it's hardware. Up until recently had been running the standard 9.04 kernal with no problems. 
 
I switched to the custom one which seem to get a bit better performence and a bit more out the battery. 
 
Might be worth giving it a try [http://www.aspireonekernel.com/](http://www.aspireonekernel.com/)
 
 
Also, a final though check. In Skype go to options > video devices and make sire enable skype video video is checked. There an option to test your webcam in there too.
