### Post by JKyleOKC on 2011-12-21
> **flyfishingphil said:**
> I ask because every time you set a password the window pops up regarding "remember password". Is it possible to view this file so I can copy it and do a printed list?It sounds as if this is happening when you browse the web, since the other normal applications seldom if ever ask you to set a password or whether to remember it.

If this is what's happening, which browser are you using? If it's Firefox, there's a dialog in the "Edit Preferences" area to view saved password data, and that brings up the entire list. By default, the passwords themselves are not shown, but there's a button to show them.

There's no option to print them, at least in my copy of Firefox, but you can do screenshots of the display, save those, and print them out.

The file that stores this information is encrypted, but I've not tried to crack the encryption or even identify the specific file since the display option suffices for my needs...
