### Post by squaregoldfish on 2009-02-11
I use Audacity for this. It's quite an advanced program, but it lets you do the simple stuff really simply. I'm in the process of converting all my cassettes to CDs, and use the following steps:

1. Record the whole tape into a file. It's worth doing a few short test runs to make sure the incoming volume is set to a good level.

2. Use the Noise Removal function to get rid of the background tape hiss. This makes a huge difference to the CD quality. It's really easy to use too.

3. Export the tape tracks as separate WAV files.

4. Use any CD authoring program to burn the WAV files to a CD. I use K3B, but any program should work fine.

Hope this helps,
Steve.
