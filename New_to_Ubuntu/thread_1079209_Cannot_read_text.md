### Post by yeats on 2009-02-24
> I need some help, please!

I was bored & I was stupidly messing around w/ the settings on my pc, not knowing what I was doing.

You know this now, but you must back up any files you decide to change.  It takes 1 second and can potentially save you hours:

```
sudo cp *file file.orig*
```

substituting the name of the config file for "file" here.  The backup can be called whatever you want, just so long as you remember what it is.  You probably would need to preface with sudo, depending on which file you're changing.


Remember what you changed?
