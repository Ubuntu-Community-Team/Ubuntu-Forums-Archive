### Post by 3rdalbum on 2010-02-13
Unetbootin doesn't install Ubuntu, it merely copies the ISO file to another drive and makes it bootable. The primary use of Unetbootin is to turn the Live CD into a "live USB" - that is, make a USB flash drive do the work of the CD. You can then use the flash drive to install Ubuntu to your hard disk.

Also, for obvious reasons, you can't erase the hard disk that your computer is currently running off!
