### Post by damis648 on 2008-05-22
It is called "Scale" And it is a plugin. Just make sure you have
compizconfig-settings-manager installed then goto System>Preference>Advanced Desktop Effect Settings
then find "Scale", make sure it is checked, go  into it, goto the Key Bindings Section, and set a hotkey.
