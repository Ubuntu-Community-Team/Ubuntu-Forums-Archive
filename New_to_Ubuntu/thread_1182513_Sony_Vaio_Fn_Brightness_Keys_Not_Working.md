### Post by Sealbhach on 2009-06-09
Hi, the jogdial is the round switch that controls volume etc.

I don't know of any way to get the brightness thing working. Perhaps you could install darker themes, like Dust, etc.  See the desktop screenshot thread for some inspiration.

To get the MAC address, open a terminal and run this command (just copy and paste it into the terminal):

```
ifconfig -a

```


The bit after "HWaddr " is your MAC address.
