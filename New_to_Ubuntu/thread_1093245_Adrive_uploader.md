### Post by cariboo on 2009-03-12
From the looks of it you don't have the java plugin installed. I would suggest installing the ubuntu-restricted-extras meta package, this will install the java plugin plus most of the codecs you need to view/listen to most audio/video media. Go to System-->Administration-->Synaptic Package Manager and search for ubuntu-restricted-extras and mark it for installation, depending on what you have installed already the package can install up to 60 different packages.
