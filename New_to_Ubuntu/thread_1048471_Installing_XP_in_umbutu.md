### Post by Thelasko on 2009-01-23
There's [a whole section of the forums]("http://ubuntuforums.org/forumdisplay.php?f=308") just on this topic.

Basically, you install VMWare or VirtualBox (VirtualBox is in the [repositories]("http://www.psychocats.net/ubuntu/installingsoftware")).  Once one of those is installed, you simply start them up, create a new virtual machine, and treat the virtual machine like real hardware.  You will require a copy of Windows to install on your virtual machine.
