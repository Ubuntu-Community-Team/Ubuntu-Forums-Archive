### Post by lazyart on 2009-05-13
It seems you can't install Office 2007 since Wine went to version 1.1.17, and it's currently in 1.1.21.  Either create a virtual machine to install it or use OpenOffice as mentioned.  The version shipped with Jaunty can recognize the .docx format introduced in Office 2007.
