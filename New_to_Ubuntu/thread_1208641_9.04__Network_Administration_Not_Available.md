### Post by tarps87 on 2009-07-30
I does seem like there are a few problems with it although it does seem to be supported by ndiswrapper.

Before you buy a new card there are a couple of other things to check

1)
```

lsmod
```
You are looking for ndiswrapper in the list, if it is not
```
sudo modprobe ndiswrapper
```
(I think you have already done this, but it is worth checking)

2)
```
sudo ifconfig wlan0 up
```
```
sudo iwlist wlan0 scan
```
It is possible the wireless card is powered off
If it finds any wireless networks then it is working, all we need to do it tell it to connect

3)
Try using the driver from the ndiswrapper website and then retry 2
[http://sourceforge.net/apps/mediawiki/ndiswrapper/index.php?title=BT_Voyager_1040](http://sourceforge.net/apps/mediawiki/ndiswrapper/index.php?title=BT_Voyager_1040)
