### Post by jamesjenner on 2011-05-04
Well I'm not a developer as such in Ubuntu but I do know that you need to provide some details as to what your interested in doing. 

What you need to consider is what areas do you wish to work in. Do you like the idea of working on widgets, are you interested in writing GUI based applications or maybe you wish to write utilities to perform specific tasks independent of a UI.

If you wish to develop for the UI then what area is important. For example if you wish to work on applets (gdesklets I think is the path Ubuntu is taking, but lot's of management programs provide applets of one sort or another) then which type you wish to work on depends heavily on where you need to start. If you wish to work on applications then you have to consider which framework you wish to work on, there are a lot of options. 

Also what knowledge do you currently have? Do you know any programming languages? Are you an experienced Unix/Linux user or are you knew to the whole Ubuntu environment?

Hey James,

UI means User Interface. GUI programs require the Graphical User Interface while terminal based programs are purely text based, sometimes not requiring user interaction. 

You have quite a few choices in this regard. You can write in C, C++, python, Java, etc. I would suspect that C and C++ are the common tools however python is quite popular as well (in fact gdesklets, the GUI embedded applets are written using python). 

I should note that you can also write shell scripts if you wish, this is a scripting language that runs in the shell (bash is the default one in Ubuntu, think of the Command Shell in Windows). Shell scripts are very powerful under Linux and is always useful to learn IMHO.

Cheers,
