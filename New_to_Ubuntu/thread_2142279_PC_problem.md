### Post by Mark Phelps on 2013-05-05
> **bapsummer said:**
> How do i start and turned off my pc quickly?

Any PC takes time to start up because there is lots of stuff that gets loaded into memory as part of the startup procedure.

The general ways to address this are as follows:
1) Startup tuning.  You can run an app that will monitor startup, record which processes are started, and how long it takes each of them to complete.  It's very complicated to analyze and, unless you are willing to take on a lot of work tuning the processes, is not going to accomplish much.
2) Hibernate the PC.  When you do this, the PC saves information loaded during startup, and when you shut it down and start it up again, it starts faster.
3) Use an SSD instead of a Hard Drive.  These speed up startup a lot -- but you pay the price in hardware since SSDs are a lot more expensive than hard drives.
