### Post by mc4man on 2009-01-29
Sometimes it seems a little tricky to use icons that aren't in the default folder when using 'create launcher'

When you click on the 'browse' tab you can browse anywhere, most times the icons will not show up in the window. When you click 'open' they will show up *back in the launcher window*. (if you pick a location with a lot of icons it will take some time.

Ex. 
want to get an icon from /usr/share/app-install.
See screens

screen 2 loading back in main launcher window after clicking open in above window
screen 3 - done
