### Post by t0p on 2008-10-31
The one major difference between Hardy and Intrepid is that Hardy is a "long term support" version and Intrepid isn't.  Hardy will be actively supported for 3 years, I think, whereas Intrepid will be supported for just 18 months.

Intrepid also has newer versions of software.  So, if you want to be at the cutting edge, install Intrepid.  But if a stable OS with longer support is more important to you, stick with Hardy.  Personally, I'm going to keep Hardy on my desktop machine, and I'll stick Intrepid on my Eeepc.  The best of both worlds!! :)

### Post by jaytek13 on 2008-10-31
Hardy having "long term support" just means that Ubuntu will continue to offer security updates throughout the period that it is supported. It is primarily directed towards businesses who don't want to perform OS upgrades every 6 months, but don't need the latest stuff. For the average user  I would recommend keeping up with the latest version.
