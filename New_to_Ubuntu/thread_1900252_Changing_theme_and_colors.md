Gnome-shell and Unity both came without much in the way of customizability. You need to install the utilities to do so. If you're using Gnome shell (which it sounds like you are), you'll need to install Gnome-tweak.
I personally can hardly believe the Gnome devs released 3 or 3.2 without this utility bundled, but as the Trafalmadorians say, so it goes.

```
sudo apt-get install gnome-tweak-tool
```

Or if KDE is really what you want

```
sudo apt-get install kubuntu-desktop
```
