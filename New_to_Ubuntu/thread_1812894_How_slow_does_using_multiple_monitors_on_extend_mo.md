### Post by Mark Phelps on 2011-07-27
The larger hit on system performance will come from allocating system memory to the graphics chip, not from the existence of multiple monitors.  If not much system memory is left, that can slow down your system.

And, yeah, if you decided to do 3D Gaming, and have three 30-inch flat panels, all maxed out in 2500x resolution, THAT would put a big demand on graphics -- but as already mentioned, that would almost entirely burden the graphics chip.
