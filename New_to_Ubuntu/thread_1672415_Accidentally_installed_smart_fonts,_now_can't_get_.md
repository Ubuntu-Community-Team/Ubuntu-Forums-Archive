Solution is as follows, uninstall smart fonts.  go to synaptic package manager, search for smart fonts, click to perminantly remove the packets, synaptic will inform you of all supporting packages, and you can remove them also, after applying the settings, restart the computer, the fonts then should return to normal.  IF you do not want to do this, go to the fonts section in ubuntu, and change the fonts used.
