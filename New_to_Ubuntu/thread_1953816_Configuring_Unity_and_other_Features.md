### Post by cbennett926 on 2012-04-06
The Unity Launcher is not yet configurable, although it will be supported in 12.04. You could try CCSM (Compiz Config Setttings Manager), but it normally has more problems than solutions. As to the search results you could filter it using lenses: Here's and article that might help! 
[https://wiki.ubuntu.com/Unity/Lenses](https://wiki.ubuntu.com/Unity/Lenses)
