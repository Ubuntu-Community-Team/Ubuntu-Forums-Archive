### Post by Monicker on 2008-05-01
Something is definitely wrong. I have installed Windows XP under VirtualBox. specifiying as little of 5gb of total space for the virtual hard drive, and still had room to spare for installing quite a few applications.  A default install of XP will not take anywhere near 10gb of space.

Verify the size of the drive for xp in the VirtualBox settings, and from within the XP vm find out how big it sees the drive as, how much space is used and available.

### Post by mlentink on 2008-05-01
There actually is, and I've done it before, based on the guidance of others in this forum. Give me some time to find out how I did it.

But there is one thing to note. You can create a virtual disk with a certain fixed space, as apparently you have done (and so did I , int hte first instance), but you can also create a disk with a certain maximum size, which will start out small, and expand up to the limit specified. That's what I would advise to do. My virtual XP is somewhere around 14 GB now, but can expand to 30GB if necessary. Funny to note that my ubuntu takes up about 9GB....
