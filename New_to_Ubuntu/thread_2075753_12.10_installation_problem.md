### Post by Bashing-om on 2012-10-25
hi Sohaib Afzal;

From the error "[  6.xxxxxxx] device-mapper: dm-raid45: initialized v0.2594b" it appears raid is a factor. 
Is windows operational under a raid array ? Software raid or hardware raid ?

If raided ...then alternate install cd must be employed as that disk has the tools to cope with raid.

else if no raid intended, meta data on the new disk will have to be removed and probably requiring a re-install of windows.
