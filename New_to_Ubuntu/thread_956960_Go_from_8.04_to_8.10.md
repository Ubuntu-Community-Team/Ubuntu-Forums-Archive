### Post by ptn107 on 2008-10-23
Wait 7 days until Ubuntu 8.10 is officially released.  Then go to System -> Administration -> Update Manager.  Click **check** and apply any updates that show up.  After that's done you should see a button in the update manager for a new distribution update for 8.10, just click it.

If you can't wait, open the run application box (alt+f2) and type **update-manager -d** in the box.  Even though intrepid is in the release candidate state, you should still be wary about installing it.  Back up important files before you proceed with the upgrade.

You can download the release candidate ISOs from here if you wish to burn your own CD.

32-bit:
[http://releases.ubuntu.com/releases/8.10/ubuntu-8.10-rc-desktop-i386.iso](http://releases.ubuntu.com/releases/8.10/ubuntu-8.10-rc-desktop-i386.iso)

64-bit:
[http://releases.ubuntu.com/releases/8.10/ubuntu-8.10-rc-desktop-amd64.iso](http://releases.ubuntu.com/releases/8.10/ubuntu-8.10-rc-desktop-amd64.iso)
