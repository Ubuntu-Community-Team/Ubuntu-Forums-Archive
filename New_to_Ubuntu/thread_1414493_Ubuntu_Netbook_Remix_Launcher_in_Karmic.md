> **kd.zombrainy said:**
> is it possible to add the netbook-launcher in UNR to the gnome panel in karmic?
I'm not sure I understand what you're asking here.

Do you want a drop menu launcher in the GNOME panel? Then right-click the panel in an empty spot> add to panel> then pick 'Main Menu' from the list. You'll get an Ubuntu logo that will drop down a program menu when you click on it.

Or are you asking if you can actually add 'netbook-launcher' to the panel...then I suppose you would right-click the panel in an empty spot> add to panel>then pick 'custom application launcher' and then in the command type 'netbook-launcher'...  Then you'll get an applet that will run 'netbook-launcher' when you  click on it... but this seems like an odd thing to want to do.
