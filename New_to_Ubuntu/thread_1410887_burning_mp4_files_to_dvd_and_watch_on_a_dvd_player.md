### Post by hemimaniac on 2010-02-19
to merge the files into one you can use avidemux by opening the first one, clicking ok to anything it asks to re-map or unpack, then each additional one use 

File > Append > then select the file

once all the files are added you can then do save project as, name it and wait as avidemux will splice it all to gether for you (with proper time maps)

then as the others suggested use devede to convert and burn it for standalone dvd pplayer
