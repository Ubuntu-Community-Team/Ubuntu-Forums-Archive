### Post by QIII on 2012-01-27
Some Windows-centric developers just do that.

There are some configurations you can change using about:config, but that's dangerous.

There is an add-on for FireFox called "User Agent Switcher" that will disguise you as a windows machine, but some applications will still not work if they are Windows specific.
