Beryl is old now, it fused with compiz to make compiz-fussion, which is installed by default, enable/disable it via: System>Preferences>Appearance then the **Visual Effects** tab.

Ubuntu is a fair simple linux to use, there are others that are good for beginers, but I wouldn't claim any of them to be the best, go for what you like best.

### Post by scragar on 2008-07-20
if your having problems using the features install the compiz settings manager

```
 sudo apt-get install compizconfig-settings-manager
```
you'll then find the advanced options in the same place, this let's you choose key bindings and turn on or off each feature you want.

### Post by carandraug on 2008-07-20

To get rid of the fire press Shift + Super (key with windows logo) + C  You can change it in the compiz settings manager but I believe that's the shortcut that comes by default.

EDIT
By the way, if you try the option to rain in the desktop, there's no shortcut to start and another to stop, you just toggle it on or off with the same shortcut. It took me forever to find that out.
