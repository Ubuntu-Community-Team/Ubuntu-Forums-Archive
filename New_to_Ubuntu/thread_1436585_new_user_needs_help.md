### Post by gordintoronto on 2010-03-22
Can you boot from a LiveCD?  It sounds like a hardware problem is at the root of the problem.

If the LiveCD works OK, Places/Computer should give you access to the hard drive.  You could look at the log files, described here:
[https://help.ubuntu.com/community/LinuxLogFiles](https://help.ubuntu.com/community/LinuxLogFiles)

With an eight-year-old laptop there are many things which could go wrong. Fortunately, if the hardware has failed, the cost of replacing it is low.
