### Post by tomtom1982 on 2008-12-14
Which version it is? Ubuntu 6.06 or Ubuntu 6.10?

It isn't able to upgrade directly from 6.06 or 6.10 to 8.10.

You can upgraee an Ubuntu 6.06 to Ubuntu 8.04 (over the net or with an Ubuntu 8.04 disk) and after this you can upgrade to Ubuntu 8.10 (over the net or with Ubuntu 8.10 disk).

But with Ubuntu 6.10 you have to upgrade to Ubuntu 7.04, than upgrade to Ubuntu 7.10, than to Ubuntu 8.04, than to Ubuntu 8.10.

In both cases a new installation will be much faster:p
