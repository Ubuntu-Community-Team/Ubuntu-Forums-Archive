### Post by BugBuster on 2009-07-07
I think ekiga should be able to do what you wish to, provided that all the other machines also use ekiga.
In case of a lan where there is no DNS available, u need to put in the ip address of the target machine.

U can set up with MS Netmeeting as well but that requires installing speex codecs on windows machine..also it will have only voice support, no text chat or video i believe.
