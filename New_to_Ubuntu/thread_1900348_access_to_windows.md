### Post by squenson on 2011-12-26
Hi,

I think it is not possible the way you have installed the two OSes 'side by side'. The only way I know to work in parallel with Windows and Ubuntu is to install one as a virtual machine inside the other one: one OS will be the host and the other one will be the guest. Go to VirtualBox.org for more information.

### Post by jerome1232 on 2011-12-26
Let me make sure I properly understand the question, You want to be able to run Windows from within Ubuntu correct?

Or do you simply want to be able to acess your Windows files from within Ubuntu?

The first is not possible, but.....

You can use a virtual machine which will emulate a computer, you can install windows into a virtual machine and you will be able to boot that virtual machine and  use it like it was it's own computer, the problem is it's taxing on your machine, you need a powerful machine to do it with lots of ram, and video isn't going to be good enough to do things like 3d gaming, although it's more than sufficient for most work/study needs.

The second is easy, You should be able to click on your home folder in the unity launcher, and it should be listed in the left hand pane under the devices column.
