### Post by Paqman on 2010-05-20
> **Satheesaran.S said:**
> 
1.Wat is minimum space required for Ubuntu Lucid Lynx?


For a single user, probably about 6-8GB, plus room for your data.


> 2.I am having a Laptop with single partition containing windows 7.
How could i install Ubuntu (Lucid lynx) in it?

During installation, it will ask you what you want to do with the drive. One of the options is to partition the drive and install Ubuntu beside Windows.

> 
3.While buyin a new Laptop,can we ask that dealer to create a seperate parition for Ubuntu? Is that possible?
 

Possibly, but you don't really need to (see above). Unless the dealer is very familiar with Ubuntu then there's no real advantage, as the installer will probably have to overwrite what they've done anyway.

Take a backup of your Windows partition before making any changes to partitions, just in case.

### Post by Bucky Ball on 2010-05-20
You need to shrink that Windows partition and make a new one in the free space you create. You can do this by running Ubuntu from the install disk and using 'Partition Editor' (System->Administration) or through Disk Management (from memory!) in Windows, or something like it. Not familiar with Windows 7..

VERY IMPORTANT: Before you shrink the Windows partition make sure you defragment it.
