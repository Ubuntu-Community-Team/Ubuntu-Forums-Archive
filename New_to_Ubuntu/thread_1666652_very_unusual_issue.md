### Post by cascade9 on 2011-01-14
If you failed the BIOS memory test, you may have blown your RAM. 

Reset the BIOS, from within the BIOS if yuo can get there, or by the clear CMOS swtich if you cant. Then try runnign memtest x86. If the BIOS memory test or memtest fails, you've given your RAM an error.
