### Post by blazemore on 2010-08-14
If you're using Ubuntu Desktop Edition, there's a Workspace Switcher in the bottom-right corner of the screen.

You can right-click on this and I think the option is something like "Workspace settings"
You can choose the number of rows and columns this way (2 rows 1 column).

The key combination to switch workspaces in this configuration would be "Ctrl + Alt + Down/Up"
