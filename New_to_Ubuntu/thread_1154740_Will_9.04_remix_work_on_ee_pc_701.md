### Post by bennachie on 2009-05-10
It will work fine, and the installation process respects the small screen (which is more than can be said for the vanilla version). However, when I tried NBR, there was a bug which was causing the GUI to operate at a truly glacial speed, and NBR chewed up noticeably more of that precious 4G SSD than did the vanilla Gnome version. 

All things considered, I think a vanilla installation is the best way to go. Just keep hitting OK during the installation process, and ignore the frustration of being unable to see some of the options - you can fix all the normal things afterwards. Do remember to set desktop effects to "none" so that you can continue to use the Alt-Click-Drag option, which continues to be needed for things like preference windows in Thunderbird and other external packages (Evolution is now quite good in that respect).
