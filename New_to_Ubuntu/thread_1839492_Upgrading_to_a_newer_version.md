Personally i prefer to back up my data (well i always have that anyways) and do a complete fresh install.

You can however do it through your update manager. (need to change settings to show new versions)

But if you do this make sure you are not conected to the internet via wireless, hardwire yourself for this incase of drop outs and trashing your machine.

However beware that 10.04 is a stable and LTS release so might be best to stick with it, unless you have a particaul reason to upgrade.

If you just want to see what the new versions are like then use a Live CD or a virtualisation from [www.virtualbox.org]("http://www.virtualbox.org")

### Post by oscarivera9 on 2011-09-06
Very good advice everyone's been giving you on this subject.  I agree with everyone else pretty much.  Let me explain why.  I have been using Ubuntu 11.04 for about two weeks so far and I actually do like it, however that wasn't always the case.  The reason I've only been using the newest version is that I just built a new PC about two weeks ago.  Prior to that, I had a Dell Precision 360 from 2004 that originally shipped with Windows XP and when I upgraded from Ubuntu 10.10 to 11.04 I encountered hardware issues.  I had to do a fresh install, but being a bit wiser from the experience I installed the 10.04 LTS because of the stability.  I never had a problem with 10.10 but I just figured the difference between 10.04 and 10.10 wasn't that significant and since the PC was rather old, I decided to keep its OS supported for a longer time than 10.10 would have, especially since I knew that it wouldn't be able to run 11.04.  Now that I have a new PC I can actually take advantage of 11.04 but like others suggested, you can always find ways to use the newest and latest of whatever it is that you need while staying with the 10.04 LTS.  
For example, I figured out (as was mentioned earlier) how to upgrade to Firefox 4 while still using 10.04.  Then, when Firefox 5 came out, I was able to upgrade pretty effortlessly, as if I were using 11.04.  If you go through the process of upgrading from Firefox 3.6 to Firefox 4 with an older version of Ubuntu, from then on your setup is ready to continue with automatic updates.  In other words, you would only have to go through that process once, after the first time it's as if you were using Ubuntu 11.04.
Then again, I don't know what reasons you have for wanting to upgrade.
However, please do follow others' advice and before you commit to the upgrade, try out whichever version you want on a Live CD just to make sure that a) you like it, and more importantly b) that your PC likes it.

Enjoy
