Dragon Natural Speaking 5 is Windows software. The easiest solution to your problem is to install the software on a Windows computer. ### Post by Dayofswords on 2010-04-05
to start:
Ubuntu is an operating system, an operating system run your applications and programs and does things in the background. Windows is also an operating system but the two are not the exact same. each have their own way of doing things.

Ubuntu is not installed like a normal program, it needs its own space on your hard drive. it works separately from Windows.

when it comes to installing ubuntu you need to burn it to a CD, you do this by downloading the ISO(a fairly large file at ~700MB). an iso is simple a copy of a CD. you use a ISO burner software like Infrarecorder([http://infrarecorder.org/](http://infrarecorder.org/)) if your doing this on windows. 

if you can not burn a CD you can request one through shipit([https://shipit.ubuntu.com/](https://shipit.ubuntu.com/)), but this can take weeks to receive as they as doing it for free, no shipping fee, nothing.

another option is to buy the CD from Canonical(makers of ubuntu), ([http://www.ubuntu.com/getubuntu/purchasethis](http://www.ubuntu.com/getubuntu/purchasethis)) is a little pricy if you simply want the cd

Do not buy Linux distribution(which ubuntu is) from ebay, this includes fedora, opensuse, gentoo, debian, etc. they are just scammers trying to make a buck.

i would recommend you find someone in your neighborhood who is tech savvy to help you with this

ALSO: ubuntu is different from windows, and will not use programs made for windows without some help of another program called "Wine", but this is not a full replacement, it may not work.
