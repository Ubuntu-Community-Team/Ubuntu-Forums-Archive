Yea, as mentioned. Install gdebi from the repositories then double-click one of the .deb file you have downloaded. Should work.

Would I be right in saying that the ones you can't install are .deb files you have downloaded manually and the ones you can install come directly from the official Ubuntu repositories? If so, different beasts and you can't relate one installing and the other not.
