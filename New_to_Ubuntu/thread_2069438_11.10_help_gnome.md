### Post by Frogs Hair on 2012-10-10
In _Ubuntu_ 11.10  the default desktop is Unity it also has the Gnome Fall-back/Classic Session available which can be selected from the session list in the login screen when installed.

In the Fallback Session Alt + Right Click is used to access the add to panel menu.  Knowing what the default desktop environment Black Ubuntu  11.10 is may help They have a forum also.
