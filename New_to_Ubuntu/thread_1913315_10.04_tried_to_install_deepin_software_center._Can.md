### Post by derrickruthless on 2012-01-23
I finally figured it out.

then browse to /usr/share/ and remove all traces of deepin software center (it should be one folder).

Run your updates and viola!

Ok, run this, but COPY/PASTE it so you don't accidentally wipe your system:

```
sudo rm -r /var/lib/apt/lists/*
```

then

```
sudo apt-get update
```
