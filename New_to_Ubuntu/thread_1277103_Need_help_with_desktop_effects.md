### Post by Paqman on 2009-09-27
You can't install proprietary graphics drivers on the LiveCD because, as you say, you have to reboot after you install them.

If you want a live session that you can install them on, create a LiveUSB using the "USB startup disk creator" and give it a couple of hundred MB for persistence.
