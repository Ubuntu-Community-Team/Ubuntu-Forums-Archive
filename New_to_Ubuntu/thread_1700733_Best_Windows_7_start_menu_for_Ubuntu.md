### Post by Mark Phelps on 2011-03-05
OK, I know I'm probably going to get flamed for saying this ...

But, changing Ubuntu to "look like" Win7 is really only going to make the situation harder for her.  Why? Because Ubuntu does not really WORK like Win7.

There are genuine differences between Linux OSs and Windows OSs.  Learning how to deal with those is one of the major stumbling blocks to switching over to a Linux OS.

Keeping the default look-and-feel of Linux OSs will actually help her switch because it will make the differences visible -- and that will actually be easier to learn.
