### Post by Cypher on 2008-08-12
I don't believe there are any native Linux flash applications out there..I run Flash 8 (renamed MX I believe) in Wine which works great. The newer versions of Flash though don't work so well in Wine.

Should you need to run the latest version of Flash and supporting tools, you can always run Windows in a Virtual machine in Linux using VirtualBox or VMWare..
