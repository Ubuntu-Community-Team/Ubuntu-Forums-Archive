### Post by bilalakhtar on 2009-12-01
What is the problem with booting from the live CD?
An alternative way to access your files would be to install EXT2-FSD or FS-Driver on Windows XP. These are drivers for accessing your Ubuntu partition from Windows. However, booting from the Ubuntu live CD and recovering the data is a better option.:popcorn:
