### Post by scragar on 2009-03-08
I've got it configured for tor so I can access .onion domains, it's not too hard to set up.
[list=1][*] ctrl+F2 = options, not hard.
[*]Add New Proxy
[*]Name isn't all that important, but you will need one.
[*]Hit proxy details tab
[*]Choose automatic, or manual, fill in details needed.
[*]Click URL Patterns Proxy.
[*]Add New Pattern
[*]Not hard to work out
[*]Click OK on everything
[*]Test it out.[/list]
Is that good enough, or do you want images? :p
