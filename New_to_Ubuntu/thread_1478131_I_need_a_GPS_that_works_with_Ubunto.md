### Post by keithweddell on 2010-05-09
There are probably a couple of things that you can do depending on your hardware, how much disk space you have and so on:

1.  You could "dual boot" - put Ubuntu and XP on the same harddrive and pick which one to boot each time you start the computer.  [This page]("https://help.ubuntu.com/community/WindowsDualBoot") has a lot of information.

2. Depending on whether your BIOS allows you to boot from a USB device, you may be able to install Ubuntu on the external hard drive - see [this page]("https://help.ubuntu.com/community/BootFromUSB") for more information.

3. You could install VirtualBox on Ubuntu and run XP inside it - see [this page]("https://help.ubuntu.com/community/VirtualBox").  

Hope these will get you started.


Keith

Edit - If there is a native Ubuntu solution, it would almost certainly be easier to go that way.
