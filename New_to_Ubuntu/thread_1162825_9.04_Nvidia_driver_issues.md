### Post by skymera on 2009-05-18
How did you install these drivers?

I suggest the Envy method. This has worked perfect for me, for years :)

```
 sudo apt-get install envyng-gtk 
```

Envy should be available in Applications > System Tools

Run that and choose Install Nvidia Driver.

This should install the driver and compile the kernel module for you :)
