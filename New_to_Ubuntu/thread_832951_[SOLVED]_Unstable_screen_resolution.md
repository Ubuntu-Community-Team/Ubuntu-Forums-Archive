### Post by alienexplorers on 2008-06-19
Make the changes noted and add the two lines in red.

> Section "Monitor"
    Identifier     "Monitor0"
    VendorName     "Delta" <--- Enter your monitor manufacturer here
    ModelName      "DE-570" <--- Enter you monitor model number here
    HorizSync       30.0 - 70.0 <--- Enter your monitors Horz Sync Here
    VertRefresh     50.0 - 120.0 <--- Enter you monitors Vert Refresh here
    Option         "DPMS"
    [COLOR="Red"]Modeline       "1224x768_65.00"  83.41  1224 1288 1416 1608  768 769 772 798  -HSync +Vsync[/COLOR]
EndSection

Section "Screen"
    Identifier     "Screen0"
    Device         "Videocard0"
    Monitor        "Monitor0"
    DefaultDepth    24
    Option         "AddARGBGLXVisuals" "True"
    Option         "TwinView" "0"
    Option         "TwinViewXineramaInfoOrder" "CRT-0"
    [COLOR="Red"]Option         "metamodes" "1224x768_65 +0+0"[/COLOR]
    SubSection     "Display"
        Depth       24
    EndSubSection
EndSection

Section "Extensions"
    Option         "Composite" "Enable"
EndSection

Save the file and do a ctrl+alt+backspace to reset your gdm.

### Post by aquavitae on 2008-06-19
The changes he refers to are in the xorg.conf file. Open a terminal and type ```
gksu gedit /etc/X11/xorg.conf
``` The make the changes, save and close. Reconfiguring the X server is just a matter of selecting an item from a menu. When you restart your computer, in the grub boot screen select recovery mode. It will boot up into a menu; one of the options is to reconfigure the X server. Give it a try before manually editing the xorg.conf file as it will make changes to it.

You need to get the log immediately after the display gets messed up, before restarting the xserver. If the display is too messed up for you to get the file then press ctrl-alt-F1 so switch to a different terminal, log in and type ```
cp /var/log/Xorg.0.log ~/
``` Press ctrl-alt-F7 to switch bach to the gui. That will copy the log file to your home folder so you can post it when you get a display back.

When you tried reconfiguring the xserver, how did you do it? Did you use the menu or did you use sudo dpkg-reconfigure xserver? Btw, are you using hardy? I've just remembered that older ubuntus didn't have the menu, so you'd have to use the command line.
