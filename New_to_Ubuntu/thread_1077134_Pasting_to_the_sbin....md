### Post by kellemes on 2009-02-22
> **Leo Dragonheart said:**
> I have a file I was told to past into a path such as /user/local/sbin, but it won't let me paste it so can someone please tell me how to do this thanx...

You need root privileges..
So you need to open your filebrowser from terminal (or ALT-F2) like so..
```
gksudo nautilus
```
Now you can paste to root protected places.
