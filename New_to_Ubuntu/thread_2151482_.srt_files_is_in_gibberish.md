### Post by papibe on 2013-06-04
Hi josef123.

It looks like an text encoding identification problem.

To see how the file is encoding run this on the terminal:
```
file yoursubs.srt
```
Then set the proper encoding in your player. In case of SMplayer is in:
```
Preferences -> Subtitles -> Encoding -> Default subtitle encoding
```
Hope it helps. Let us know how it goes.
Regards.
