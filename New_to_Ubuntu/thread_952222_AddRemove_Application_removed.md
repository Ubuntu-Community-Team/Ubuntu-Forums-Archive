### Post by t0p on 2008-10-18
> **J03y said:**
> Today I was upgrading to ubuntu 8.04 and suddenly i dont see the Add/Remove Applications software. Please tell me that this is a default change from 8.04. If it isn't then please can you tell me if i can download it and install it again? :confused:

You should be able to put it back by editing the menu.  Right-click on the top panel where it says **Applications** and select **Edit Menus**.

Now, on the left, click on **Applications** (at the top of the **Menus** column).  Now look at the bottom of the right-hand (**Items**) column: click the box next to **Add/Remove**.  Now close that window.  This should have replaced the **Add/Remove Applications** option in your **Applications** menu.
