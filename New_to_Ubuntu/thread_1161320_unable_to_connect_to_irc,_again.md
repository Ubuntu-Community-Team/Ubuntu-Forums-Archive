I found this at the Chatzilla web site:

[http://chatzilla.hacksrus.com/faq/](http://chatzilla.hacksrus.com/faq/)

> 2.2. How do I connect to an IRC network or server? 			 		           ChatZilla comes with a selection of popular IRC networks which can be displayed using the command /networks. Connecting to one of these networks is as simple as clicking the network's name in the list or using the command /attach (e.g. the command /attach moznet will connect to the "moznet" network).         
 		           Connecting to a network or server not listed is easy as well, with the commands /attach, /server and /sslserver. The examples below connect to the imaginary server "irc.example.org" using the normal port:         
 				           /attach irc.example.org
          /attach irc://irc.example.org
          /attach irc://irc.example.org:6667
          /server irc.example.org
          /server irc.example.org 6667         


You should be ablt to join using the correct port using the above.
