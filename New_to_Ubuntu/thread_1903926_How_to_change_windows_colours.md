### Post by Mariane on 2012-01-03
It seems to me that each time I install a newer version I end up asking this question... This time I have Kubuntu 11.10 and, once again, I cannot find how to change the colours. 

It used to be simple, in a program called kcontrol. A fake window appeared and you could click on a part of it, for example the top bar, and select a colour for it. When you were satisfied with the overall effect you clicked "Apply" and all windows would be displayed in your chosen colours. You could also change the colour of the task bar at the bottom of the screen without having to download I don't know how many themes before you found one which had the colour you wanted. You see what I mean? 

There's got to be a way of doing this in Kubuntu 11.10 but I cannot find it... Please help. 

I found packages kde-style-skulpture and kwin-style-dekorator and installed them but I cannot find how to run them. Typing kwin-style-dekorator or dekorator says "command not found". In "System settings" there is "Desktop effects" and "Desktop Appearance" but neither offers the opportunity to do this. I'm really puzzled. In KDE help center it talks about changing the colours as if it were an icon of the "System settings". But no such icon. 

Related: Do you know how to put a picture of your choice as the Desktop background, please? 

Mariane

### Post by McNils on 2012-01-03
systemsettings->Application appearance->colors

You can also use change the desktop theme
SystemSettings->Workspace Appearane->Desktop theme
