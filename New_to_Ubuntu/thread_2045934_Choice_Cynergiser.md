### Post by mikechant on 2012-08-30
> Please explain, what is a windows VM (virtual machine )? 

A VM is a program that runs under Ubuntu that 'pretends' to be a whole PC, so you can install another operating system such as Windows 'inside' it and Windows thinks it's running by itself on its own machine. You can then install Windows software inside the virtual Windows in the normal way; this allows you to run almost any Windows software perfectly without leaving Ubuntu.

I've got Windows 98SE running with VMware player, with working Internet access inside the VM.

You do need a reasonable amount of RAM to run a VM; graphics performance for games may not be acceptable; some hardware devices may not be accessible from within the VM; but generally it works pretty well.
