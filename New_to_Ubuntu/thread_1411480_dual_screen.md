### Post by Gone fishing on 2010-02-20
It depends on what video card you are using, if it's an nvidia this is easy, install the proprietary drivers using System > Hardware Drivers. 

Then install nvidia-settings using synaptic and then use this tool to set up your dual monitors.

### Post by rubenverweij on 2010-02-20
You can also look under Preferences->Display to see if your 2nd monitor is already detected. You can then enable it from there. If it's nvidia, you could also follow this guide: [https://help.ubuntu.com/community/BinaryDriverHowto/Nvidia](https://help.ubuntu.com/community/BinaryDriverHowto/Nvidia).
