### Post by Paqman on 2010-09-28
> **Benup said:**
> Heard of alot about python looking forward to learning it. Also reformat (sorry if the stupid question) is this were the hard drive is wiped completely?

Formatting a drive is when you create a new filesystem on it. That can be the same type of filesystem (eg: NTFS, FTA, EXT, HFS) as before, which is a reformat. If you create a new filesystem over the top of an old one it will wipe everything. If you've got a very full filesystem backing up the data and reformatting can be a quicker way to defrag it than using a defrag tool (and not all filesystems have defrag tools anyway)

> **Benup said:**
> Sorry for the double post but does Ubuntu auto update when a new version is released? or does it have to be done manually

New Long-Term Support (LTS) versions are released every two years, with intermediate releases every six months if you can't wait that long. When a new version is released you can upgrade through the normal Update Manager you get all your other updates through. On a decent broadband connection it'll take a couple of hours, and is all done automatically. However, since reinstalling Linux is so easy, a lot of people choose to just install the new release fresh, which obviously requires a bit more hands-on just like installing in the first place.
