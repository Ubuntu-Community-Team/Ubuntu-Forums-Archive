### Post by MrFaler on 2010-03-23
I had some troubles too after installing 9.10 with Wubi.
The fix you need should be similar to what I've done.

If you can't do anything on your system, and you are viewing this thread from another computer, then you should first boot into recovery mode.
Select root access with networking and go to ok.

First we'll uninstall your current driver:
```
sudo sh /usr/share/ati/fglrx-uninstall.sh
```

Next if on another computer, you'll want to go to the ATI website and look for your driver there.

[http://support.amd.com/us/gpudownload/Pages/index.aspx](http://support.amd.com/us/gpudownload/Pages/index.aspx)

When you found your driver and you are at the download button, the fun part begins; you get to type the link exactly like it is on the website. To do this you want to right click on the download button and then click on "Copy Link Location".

Now paste it in Notepad, or any other text editor.

Now you want to type this in the Ubuntu terminal:

Example: (this downloads the driver for the ATI Radeon Card.)

```
wget  https://a248.e.akamai.net/f/674/9206/0/www2.ati.com/drivers/linux/ati-driver-installer-10-2-x86.x86_64.run
```

Run the installer:

```
sudo sh ./ati-driver.....
```

Remember here to finish thre rest of the filename, and then enter your password.

Then Configure the driver:
```
sudo aticonfig --initial
```

If you get an error then you might wanna try the forced method:
```
sudo aticonfig --initial -f
```

Then:
```
Reboot
```

It should work now, if it doesn't you can once more boot into recovery mode root shell prompt (with or without networking).

Then this will get you to the ATI driver command line configuration:
```
aticonfig --help
```

Some of those tips, and most of the commands are from markbuntu. He gets the full credit he deserves.
