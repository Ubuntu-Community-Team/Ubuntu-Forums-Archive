### Post by cariboo on 2009-09-08
Go to System-->Administration-->Network Tools-->Port Scan, and enter localhost in Network Address. If you have not installed any services and don't have any torrents downloading, the only thing that should show is port 631, which is used the cups, the print daemon. Port 631 can only be accessed by localhost, so there is nothing to worry about.

If you want to check your router to see what ports are open go [here]("http://nmap-online.com/").
