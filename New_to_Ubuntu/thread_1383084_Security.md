### Post by MarkC502 on 2010-01-16
Thats a pretty broad question without many specifics as to usage. In general you can forget the word VIRUS exists as far as PC's. For the most part their are no glaring security flaws with Ubuntu like found in some windows deployments. Linux uses a command line based firewall called iptables which is already installed, you can install Guarddog or Firestarter to give you a simpler GUI to configure your linux systems iptable rules. 

In general you are much safer in Ubuntu than in Windows.

### Post by sgosnell on 2010-01-16
I have nothing at all installed, haven't for years, and never had a problem.  If you take even rudimentary precautions, you have no need of firewalls, anti-virus, or any other Windows junk, if you're just running a personal computer.  If you're in charge of a large network, then you need to know more, but for personal use, don't worry about it.
