### Post by Sheeba15 on 2009-05-18
Hello there!
 
Having just got my Acer Aspire One with Linpus Linux on it which I just didn't like I decided to try the ubuntu remix for netbooks which my daughter uses and loves.
 
However although the install went well and all was working correctly, if I switched to the standard ubuntu desktop rather than the netbook one I found that both panels would disappear and as I'm not familiar with linux in any form I wasn't particularly comfortable using the terminal to resolve it.
 
I read through these forums and found that quite a few people were having the same problem so I thought I'd post my fix for it as to me it seemed a whole lot easier than playing around with the commands when you're not familiar with them!
 
I actually installed it about 4 times before I found the fix! Simply when I'd installed it the last time instead of switching to the normal desktop straight away I did all the updates first. When I switched desktops after a cold restart everything went well and it's been working correctly ever since.
 
I can only assume that there must be a small bug in the remix version which is fixed by doing the updates first. So I'm now up and running and very happy to be getting to know ubuntu!
 
I hope this helps someone!
