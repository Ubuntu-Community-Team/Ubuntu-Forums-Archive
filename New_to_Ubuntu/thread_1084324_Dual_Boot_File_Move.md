### Post by presence1960 on 2009-03-01
Windows does not natively have the ability to read/write Linux partitions/drives. Download this .exe and install it in Windows. You will then have the ability to read/write files on linux partitions/drives. here is the link : [http://www.fs-driver.org/download.html](http://www.fs-driver.org/download.html)

P,S. Don't worry about the name of the file it will recognize ext3.
