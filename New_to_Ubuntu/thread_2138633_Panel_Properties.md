### Post by grahammechanical on 2013-04-24
You remember correctly but as you say that was several years ago in the days when the Ubuntu user interface was Gnome panel which was based upon Gnome 2. Ubuntu is now based on Gnome 3 but the developers choose not to use the Gnome 3 shell user interface. They developed the Unity user interface instead. 

The top panel in Unity is reserved space for certain kinds of applications to put app indicators to give notifications. What you are looking for does not exist any more on a standard install of Ubuntu. This may help you to revert. but then again it may not.
