### Post by wormser on 2008-06-02
Sounds like it could be a DNS issue.  To test this, ping by IP and by domain.  If the IP number works and the domain does not then it is a DNS issue.

Try [OpenDNS]("https://www.opendns.com/start/ubuntu.php").
