Linux offers many desktop alternatives. The most popular desktop environments are GNOME, KDE, Xfce and LXDE. These are rather large collections of desktop programs, rather than bare bones window managers such as FVWM, IceWM amongst numerous others. These environments present a GUI using a desktop metaphor. All of these environments allow the user to set many personal preferences and to perform common system management tasks. The Compiz window manager expands upon the concept of having multiple desktops by rendering each of the user's two dimensional desktops on the surface of a three dimensional cube.
