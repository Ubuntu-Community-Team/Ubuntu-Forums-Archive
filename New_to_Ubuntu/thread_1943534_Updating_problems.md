### Post by Frogs Hair on 2012-03-19
The further from the release date the more updates you will have when using  Wubi . You have many updates that won't be installed because they are obsolete already. 

The kernel you have is likely 3.0.0-12 and you would be using 3.0.0-16 if you update . When I help friends with Wubi the first thing we do when it boots up is run the update manager before installing anything . 

There have been many improvements since October and weather you update or not is up to you. A friend just installed with Wubi the other day and there were almost 400 Mb of updates. It is better to get the updates out of the way before installing Compiz , proprietary drivers and tweaking the system.

### Post by jerrrys on 2012-03-19
Open a terminal and enter:

sudo apt-get clean

sudo apt-get update

And please post any errors you get.  If no errors, in terminal enter:

sudo apt-get upgrade

[https://help.ubuntu.com/community/UsingTheTerminal](https://help.ubuntu.com/community/UsingTheTerminal)
