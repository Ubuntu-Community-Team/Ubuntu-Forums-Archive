### Post by falkTX on 2009-06-19
With Ubuntu, drivers aren't usually a problem (almost everything works out-of-the-box)

Try the "Hardware Drivers" in System -> Administration;
ATI drivers usually show up there and you have an option to install them

### Post by 3rdalbum on 2009-06-19
A mouse is a mouse is a mouse; if your mouse and graphics card work on the live disc, they will work on the installed system.

Your graphics card is probably running with 2D graphics acceleration, which is what the open-source driver supports. You can install ATI's 3D acceleration driver after install, if you wish. The "Hardware Drivers" program will give you the option to download and install the driver automatically.
