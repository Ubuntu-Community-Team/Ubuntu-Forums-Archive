### Post by HermanAB on 2011-11-13
Yup, the thing is that you got to consider the Linux security model, which is user based.  So after a user logged in and was authenticated, that user can then do things and save things.  Allowing another user to use that same session, breaks the security model.  

In a webmail system the data resides on the remote server and therefore doesn't break the security model, until the user wants to save an attachment locally.
