### Post by sydbat on 2009-05-05
Some more info would really help. 

Did you "install" Ubuntu into Windows (via wubi)? If so, then you can uninstall it like any other Windows program.

If you created a separate partition as a dual boot with Windows, then you shouldn't need to do anything...unless you want to get rid of Windows.

If it is the only OS on the computer you mention, then there is no reason to remove it, unless something went horribly wrong. Then you would reformat after saving what you can from your /home partition (using the LiveCD).
