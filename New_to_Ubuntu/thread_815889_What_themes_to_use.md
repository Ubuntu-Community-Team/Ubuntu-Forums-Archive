### Post by mcduck on 2008-06-02
There is no Beryl any more. It has merged with Compiz into the Compiz Fusion we are now using.

Anyway, there newer was much of a performance difference between the two, Bryl simply had more plugins and features but they are all in Compiz Fusion now.

When you are not using Compiz you can use GTK2- and icon themes just like with Compiz, the only difference is that you need to use Metacity-themes for window borders (Metacity being Gnome's default window manager).
