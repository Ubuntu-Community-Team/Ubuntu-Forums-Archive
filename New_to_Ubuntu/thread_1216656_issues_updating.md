### Post by scorp123 on 2009-07-26
> **nnamdi said:**
>  ```
  Could not resolve ‘proxy.aust-abuja.org’
``` Is your Internet connection working other than that?

Can you please try this:

Go through these menus:
[INDENT]*System > Administration > Synaptic Package Manager*[/INDENT]

Then, when ***Synaptic*** has opened, go through its menus:
[INDENT]*Settings > Preferences*[/INDENT]

A new window should have opened that offers a few tabs:
[INDENT]*General | Columns and Fonts | Colors | Files | Network | Distribution*[/INDENT]

... Go to the ***Network*** tab!

**What does it say there?**

My guess is that right there you have a invalid proxy configured .... because that is what your error message is saying. And this tab would be the most obvious place to configure a proxy for the "apt" package manager. So if there is an invalid proxy entry: please remove it. Refresh your repos ... and try the update again.
