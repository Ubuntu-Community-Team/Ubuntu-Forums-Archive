I need to update swfdec plugin in Firefox. But how? I'd also like to change it to Macromedia's Flash 9.0. How this can be done properly?

---

### Post by gandaran on 2008-04-30
open synaptic, look up swfdec, choose complete removal, find flashplugin-nonfree, check it for install and click apply.
