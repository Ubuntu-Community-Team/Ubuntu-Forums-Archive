### Post by anewguy on 2010-04-03
Just so you know, I don't know squat about Insyde H2O BIOS, but I do have something you might want to try:

- download one of the Window98SE or other boot diskette images and create the diskette, then copy the update .exe to it.  Make sure your boot sequence is set to allow a floppy first (or if no floppy, make a bootable CD) and reboot from the created media.  When the prompt comes up just type in the .exe file name.

I have done things like this in the past, but I have no clue if it will work for you or not.  I think once you get a bootable DOS like media with command.com, etc., on it, you should be able to execute the file.
