### Post by 3rdalbum on 2009-10-22
> **jtreg said:**
> nothing found in the Hardware Drivers screen although I have downloaded the driver to the Desktop, is there another step I have missed out... Nvidia GeForce 7200 GS, Hardy Heron.

The Hardware Drivers program works with software that is in the Ubuntu repositories, not with stuff that you download off the Internet manually.

So you must have an internet connection, you must enable all the repositories in the Software Sources program and you must have reloaded the package list (the Software Sources program offers to do it for you if you change the settings, or you can click the Reload button in Synaptic Package Manager).
