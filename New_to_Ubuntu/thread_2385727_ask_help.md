### Post by sudodus on 2018-02-24
It is a good idea to try Ubuntu (standard Ubuntu or an Ubuntu community flavour, Kubuntu, Lubuntu ... Xubuntu).

- **Try Ubuntu live**, booted from a DVD disk or USB pendrive. Do *not* install it at this stage. See this link,

[Try Ubuntu (Kubuntu, Lubuntu, Xubuntu, ...) before installing it](http://ubuntuforums.org/showthread.php?t=2230389)

- When you are ready to install Ubuntu, please consider a **dual boot** system with both Ubuntu and Windows, or at least create **Windows recovery disks or maybe even a complete backup** of your Windows system. This makes it easier to go back to Windows. Maybe there is some application (program) in Windows, that is not replaced by a good Ubuntu alternative.
