### Post by TheForumTroll on 2009-11-03
[GTK is a cross-platform widget toolkit for creating graphical user interfaces]("http://en.wikipedia.org/wiki/GTK%2B"). GTK has it's own look and feel. [Metacity is a window manager]("http://en.wikipedia.org/wiki/Metacity"). Emerald is a theme manager. It allows you to use some nice effects ;)

GTK is the tool kit (something that is used to create gui applications) with which the Gnome is made. Metacity is the window manager of Gnome (window manager means the software that manages the movement , maximising , minimising, etc related to windows  ) . Emirald is the window decorator (the program that draws window border maximise, minimise buttons etc  ). But it's not part of Gnome . It's part of another window manager named compiz . In Ubuntu , both metacity And compiz are used  . Metacity switch to compiz when you enable desktop effects . Unless you enable the emirald , it will still use the window decorator of metacity .
