### Post by grahammechanical on 2015-07-11
Your thread title contradicts this statement

> is how much can i limit the use of the mouse in favor of shortcuts and navigating with the keyboard

Press  what Microsoft calls the Windows key and what Linux users call the Super  key. Hold it down. Two things will happen. 1) The applications icons in  the Launcher will get a number which we can press to launch the  application and an overlay will appear that will list various keyboard  shortcuts that work with Unity.

When an application is open then  pressing Alt will cause a list of menu options to appear top left in the  top panel - Alt+F will open the File menu and the arrow key will scroll  down the list. Alt+E opens the Edit menu and so on. Items in these menu  lists can often be activated by Ctrl+letter combinations.

All  these short cuts date back to the days before the mouse was invented. We  had no choice but to use keyboard shorts to do everything. I have never got out of the habit of using Ctrl+C = Copy; Ctrl+X= Cut and Ctrl+V =  Paste. Ctrl+P could not be used for paste because it was already being  used for Print.
