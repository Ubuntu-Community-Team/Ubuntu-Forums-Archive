### Post by beastrace91 on 2009-10-18
Hey There,

.exe programs are for Windows only - they don't run on Linux (unless you use Wine but that is another story all together)

Anywho - blender has a Linux installer - go to add/remove programs in your KDE menu and search for "blender" or open kconsole and run the following command: ```
sudo apt-get install blender
```

Give it your password when it asks for it and then it will download & install blender for you.
