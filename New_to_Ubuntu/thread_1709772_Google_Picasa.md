### Post by user333 on 2011-03-19
It should show up under graphics -> picasa. Try to run the program by pressing alt-f2, type picasa, and if that works, then this can be fixed. First right-click the applications menu, and click "edit menus" After that, click on "graphics" in the left side of the window. If you see picasa, checkmark it, otherwise click "add new item" and use picasa for the command.

Let me know if you need more help!
