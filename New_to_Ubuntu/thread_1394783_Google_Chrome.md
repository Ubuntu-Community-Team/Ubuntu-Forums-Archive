### Post by Temposs on 2010-01-31
Go here:

[http://www.google.com/chrome/eula.html?hl=en&brand=CHMA&utm_campaign=en&utm_source=en-ha-na-us-bk&utm_medium=ha](http://www.google.com/chrome/eula.html?hl=en&brand=CHMA&utm_campaign=en&utm_source=en-ha-na-us-bk&utm_medium=ha)

Choose the Debian version, and make sure you choose the correct 32-bit or 64-bit version. Just download the file and double click it to run the installer.
