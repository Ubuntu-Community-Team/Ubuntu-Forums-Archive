### Post by whoop on 2009-05-27
Well I'm no wubi expert (never used it, most likely never will), but if i'm not mistaken wubi doesn't partition your hd, so it must be some kind of ubuntu image on your windows partition. That's not as secure as installing ubuntu to a separate partition. I know of no virus that infects/changes linux partitions from within windows. I know loads of viruses that mess up the windows file system and it's files. So with the wubi setup your ubuntu is just as safe as any file on your windows machine IMHO.
It's probably not as bad as I portray as the ubuntu image will not be infected itself, but it could be corrupted, deleted or even encrypted by some vicious windows virus.
