### Post by jerome1232 on 2009-01-05
If your talking about when you insert media and they pop up an autorun dialog, that means they have a script on the media's top level directory called autorun, autorun.sh, or .autorun

In the case of Windows media, it's a file called autorun.inf that invokes the dialog.
