If anyone had as many problems with pdftk as I did and is interested, here is the link to download the PDF Split and Merge (a.k.a. **pdfsam**) deb package for problem free installation with Gdebi Package Installer:

[http://packages.debian.org/sid/all/pdfsam/download](http://packages.debian.org/sid/all/pdfsam/download)

I used this with Windows XP, and it's _**AWESOME**_!!!  It couldn't be simpler to split and merge PDF documents with this utility.
