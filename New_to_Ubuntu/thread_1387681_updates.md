### Post by Perfect Storm on 2010-01-22
If the updater don't work, you can use the konsole/Terminal:

```
sudo apt-get update && sudo apt-get upgrade
```
There should be a fix in the updates that make the updater working again.

Firefox:
Application ---> Internet ---> Mozilla Firefox Installer
