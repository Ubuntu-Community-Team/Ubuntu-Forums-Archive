### Post by psoplayer on 2008-10-30
I think you will need to download the special USB version of the Ubuntu image if you want to run it directly from a USB flash drive. Unless you need to have a mobile installation (it's like the same computer no matter what boots off the USB drive) or have no CD drive (like the little bitty netbooks, i.e. the Eee PC), there's no reason to install it to a USB drive.

If you are unsure of adopting Ubuntu, there's nothing to lose by just installing it through the Windows-based Wubi installer. Instead of rebooting and installing from the CD, just put it in and run the install from within Windows. It won't repartition your hard drive or potentially make your Windows installation inaccessible but it's almost the same as doing a standard installation.
