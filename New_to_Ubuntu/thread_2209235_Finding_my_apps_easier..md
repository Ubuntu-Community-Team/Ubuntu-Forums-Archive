### Post by endlessinstant on 2014-03-04
You might also want to experiment with other desktop environments: Xfce, KDE, LXDE, etc.  Almost all of them will offer this feature stock.  You can easily install them from the software center by search for ie xubuntu-desktop or kubuntu-desktop.

From the log on screen, you can choose which environment you want to boot into so you can even have multiple environments on one machine if you're like me and have to change it up every now and then.
