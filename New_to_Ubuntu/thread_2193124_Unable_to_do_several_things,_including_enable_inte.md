### Post by grahammechanical on 2013-12-11
Ubuntu has six monthly development cycles during which the version is given a code name such as Feisty Fawn. We are now are Trusty Tahr. Ubuntu version numbers are based upon the month and year in which the version came out of development. So, Feisty Fawn was released in April (04) 2007. Hence 7.04. How new is your hardware? I found that Feisty Fawn installed fine with wireless working fine on the hardware that I brought in 2007. But I would not expect Ubuntu 7.04 to have drivers for hardware that has come on the market over the last few years.

There is another reason why it is better to use a newer version of Ubuntu. A lot of work has been done on the distribution to remove complexity and technical language. We no longer need to be a computer geek to set things up in Ubuntu.

You have a choice. You can download and install Ubuntu 12.04.3 LTS (Long Term Support) which will have support with security fixes until April 2017. Or install Ubuntu 13.10 (the latest) but it only has support until July 2014. It is your choice.

[https://wiki.ubuntu.com/Releases](https://wiki.ubuntu.com/Releases)

Regards.
