### Post by silkstone on 2008-08-27
Try dragging the zipped file to the Appearance window (System > Preferences > Appearance), or use the Install button at the bottom of that window.

You shouldn't need to unzip the package first!

### Post by mcduck on 2008-08-27
If installing by drag&dropping into the theme manager window doesn't work then you can just extract the package into ~/.themes.
