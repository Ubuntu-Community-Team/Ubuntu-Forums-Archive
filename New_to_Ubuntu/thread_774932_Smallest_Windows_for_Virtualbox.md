### Post by lwvmobile on 2008-04-29
If you have a pre-existing system and cd you can use nLiteos to remove chunks of Windows NT based OS systems and files from the installation media and create a new ISO. 

The only thing I have against TinyXP is it isn't exactly legal, if you care for that kind of thing. If you use nLiteos, at least its your own license key. Then again, modifying the content of the installation may be against the license rules in Windows, I was never sure about that, but I can acheive a 500MB base install with my nLite'd Windows XP Pro.

BTW, you can also streamline SP3(assumedly) using nLite into a SP2 or prior XP cd.
