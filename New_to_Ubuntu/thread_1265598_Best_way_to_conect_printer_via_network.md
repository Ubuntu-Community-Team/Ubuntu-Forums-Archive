### Post by zebraneo on 2009-09-13
I have a printer connected directly to XP, and I want to be able to use the printer via Ubunutu, What is the best and easiest way to connect the computer. The 2 computers are conected via modum. Ubuntu is wireless and Xp is connect via networkcable. 
Thanks so much everyone

---

### Post by NoaHall on 2009-09-13
System -> Admin -> Printer -> Server -> New -> Printer -> Network Printer -> Find -> Select Printer -> Select Type -> Choose driver

You might need to download a driver, if it's not there.
