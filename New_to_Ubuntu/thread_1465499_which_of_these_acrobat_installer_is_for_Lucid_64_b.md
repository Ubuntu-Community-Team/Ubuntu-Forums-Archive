### Post by Untitled_No4 on 2010-04-29
None of them is, probably because Adobe don't make a Linux 64bit Acrobat reader.

The easiest way to install Acrobat reader on a 64 bit in Ubuntu is to enable the partner repository in your package manager and then install acroread which will pull all the necessary packages to run it on a 64bit system.
