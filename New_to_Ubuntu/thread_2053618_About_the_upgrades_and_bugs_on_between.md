### Post by Paqman on 2012-09-05
If an upgrade has left your system in a less than pristine state, I'd refuse the upgrade. Back up your data and do a fresh install of 12.04. By that I mean download the 12.04 image, put it on a USB stick, boot from it and install it over the top of everything that's on your netbook.

When you get it installed, make sure you're using the Unity 2D desktop environment (click on the cog next to your name on the login screen).

### Post by critin on 2012-09-05
Continual upgrading over old versions isn't generally recommended.  One upgrade maybe, but more?  Not a good idea.  Go for clean installs then if there is a problem, you can pinpoint it better.

If you're not sure which one you'd like, check out the video reviews on you tube.  The reviews from actual talking reviewers, not those who are only showing off their desktops and music.
