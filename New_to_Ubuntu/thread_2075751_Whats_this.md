### Post by mag1strate on 2012-10-24
To get this to go away go to the dash and type "pass"

A menu option should come up called "passwords and keys"; select it.

Right-click where it says "Passwords:login" and select "Set as default".

or you can try and change the password by right-clicking where it says "Passwords:login" and select "Change Password".

I would choose the first option.
