### Post by Mariane on 2010-06-07
I had the same problem with a Dell Vostro 1520. I could install neither 9.10 nor even 9.04 on it. The machine was too recent or something, it looked like I could spend days fighting it and still end up with a messy install. So I installed Debian and it worked like a charm from the very first attempt. 

If you can use Ubuntu you can use Debian, they are not very different. You also have the choice between Gnome and KDE, etc. Debian is much more stable than Ubuntu 9.10 - Please don't take offense, anyone. You end up with a less "cool" but more reliable machine. 

You could also try to install Ubuntu 9.04 and wait a bit before switching to 9.10... If 9.10 won't install on your machine the chances are that even if you force it to install somehow you will still keep running into problems. You could handle these problems if you were an expert but if you were an expert you would not be posting in this section, would you?
