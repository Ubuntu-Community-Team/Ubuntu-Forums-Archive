How do you change between the metacity and compiz window decorator/manager without the compiz-fusion icon?

---

### Post by fooman on 2009-03-12
you can turn compiz on in a terminal with this command:

```
compiz --replace
```

turn it off with this one:

```
metacity --replace
```
hope that helps.

---

### Post by ggaaron on 2009-03-12
You can manually (from terminal, or Alt-F2 run command) run metacity --replace or compiz --replace.
