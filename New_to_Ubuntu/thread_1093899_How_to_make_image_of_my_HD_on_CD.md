### Post by tea for one on 2009-03-12
I have successfully used Remastersys.

You have the choice of creating a live CD (without personal data) or a complete image (including personal data). The software has a GUI which is pretty easy to understand.

The Remastersys software will also add the Ubuntu install software to the image so that if you completely trash your system, you can re-install including added packages.

As a guide, I simply created a remastered DVD of my OS only (without data) and it took approx 50 minutes and the size is around 3.6GB.

You may need to burn to a DVD if you have added extra software packages.

Here is the tutorial:-

[http://www.dedoimedo.com/computers/remastersys.html](http://www.dedoimedo.com/computers/remastersys.html)
