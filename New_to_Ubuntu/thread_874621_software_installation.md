### Post by northern lights on 2008-07-30
When looking for a particular application, or particular capabilities of a hypothetical application, it's generally advisable to check in the repositories or, if somewhere else, for a .deb file first.

Installing from binaries or source should really be the last resort.

Anyhow, here's some decent reading on software installation:

[Psychocats' tutorial: Installing Software]("http://www.psychocats.net/ubuntu/installingsoftware")
