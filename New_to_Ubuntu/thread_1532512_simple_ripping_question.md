To copy and paste within a Graphical User Interface (GUI) simply open the destination file folder (where you want the files to go) and open the originating folder or directory (where the files are coming from) highlight all of the files - if you are wanting to copy all of the files - by using the keyboard command

**[COLOR="Blue"]CTRL + A[/COLOR]**

**this will select [COLOR="blue"]ALL[/COLOR] of the files** - then simply drag them over to the destination folder **or** right click the highlighted files and select **[COLOR="blue"]Copy[/COLOR]** - go to the destination folder and then simply right click and select **[COLOR="blue"]Paste[/COLOR]**.

Copy and Paste

It's a basic computer function - if the files are already .mp3 then there is no conversion involved and using software to copy files is superfluous when one can merely complete the task with a few mouse clicks.
