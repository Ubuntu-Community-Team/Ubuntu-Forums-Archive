### Post by patlkli on 2015-09-08
Older versions of Ubuntu have reached End of Life/Support.
That means that there are no longer any updates, neither new software versions nor security updates.
Your Ubuntu 11.10 didn't receive any updates since April 2013, so it's probably vulnerable to a number of critical security issues, including the well known Heartbleed bug.

The oldest, still supported version of Ubuntu is 12.04 LTS, which is supported until April 2017.
However, I strongly suggest, that you try out your printer with Ubuntu 14.04 LTS, which will receive updates until April 2019.
