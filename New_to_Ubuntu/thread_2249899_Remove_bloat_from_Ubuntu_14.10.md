### Post by grahammechanical on 2014-10-25
A lot of stuff can be disabled but that is not the same as removing "bloat" for the code still stays on the machine. I am concerned that removing bloat will eventually disable the OS in some way. The fact is that this stuff is not going to go away. The Dash it set to become a possible replacement for the Ubuntu Software Centre.

We can remove the Amazon icon from the launcher and In the Dash we can filter the search results and restrict the search sources. We can also disable each of the Dash plugins. We get at them through the applications scope.

We have a lot more control and the developers would say we have a lot more choice. Which a lot of people will like because they live in an online environment. The removing of bloat is risky and what worked on an older version of Ubuntu does not necessarily work in the latest versions. Ubuntu is not designed to be an OS that can be dismantled and rebuilt in a different configuration by the user.

My personal opinion is that this is a subject that should not be discussed in a section called New To Ubuntu.

Regards.
