### Post by 3rdalbum on 2012-03-09

any help is welcome :] :):)

ATI doesn't support that card anymore unfortunately. The driver that comes with Ubuntu 10.04 should be capable of running the X1600, albeit with only 2D acceleration and no video decoding acceleration.

If you need 3D, or if 10.04 doesn't work at all with the card, you might have more luck with a more recent version of Ubuntu such as the current 11.10, or the beta of 12.04 if you are adventurous.
