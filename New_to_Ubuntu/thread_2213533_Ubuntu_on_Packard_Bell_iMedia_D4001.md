### Post by 3rdalbum on 2014-03-27
The reason why it was difficult to downgrade to XP is because XP doesn't directly support any hardware made after 2001. When faced with a computer as advanced as yours, XP has a panic attack.

Ubuntu is, of course, much more modern. It should run fine on your computer, which is not really that old. Same era as my computer now that I think about it.

What graphics card does it have? If it is an ATI RadeonHD 4000 series or lower, you should use Xubuntu or Xubuntu instead of Ubuntu (It will work, but be a little slow). Any Nvidia card will work fine with Ubuntu. Any Intel graphics will be okay with Ubuntu.

There is a slight possibility that something in your computer won't be compatible, but the chance is low. And you won't ruin your computer. If you are nervous or curious, you can run Ubuntu or any of its derivatives (Xubuntu or Lubuntu etc) from the CD to see what it is like, before installing to the hard disk. Xubuntu and Xubuntu can run any software from Ubuntu and you can even swap between them without a reinstall, so don't worry so much about which Ubuntu to install, just get one installed.

### Post by Rob Sayer on 2014-03-27
I've used both xubuntu and lubuntu.  I have lubuntu 13.10 on my 1Gb netbook but I'd probably recommend xubuntu to a novice.  There is a bit of a learning curve and you can expect to do somw configuring.  Lubuntu is somewhat harder for a novice in this respect, though it's faster.

But I think 3Gb ram is plenty for xubuntu.  Ubuntu with unity is a little heavyweight.  

Kubuntu with the KDE desktop is what I use mostly.  It's also considered heavyweight but you can configure it to run a lot faster, which you can't do with unity.  Kde can be configured to death, which some people like and others don't.

Linux can seem daunting at first but that's largely because it's all open source.  Unlike windows, they don't hide anything from you.  It's actually mostly a lot more straightforward.

The thing I'd suggest is to download kubuntu and xubuntu and lubuntu iso's.  Then do the md5 checksums on the iso files and burn them (usb stick is better because it'll run much faster).  Then just try the different desktops and see which one you prefer.
