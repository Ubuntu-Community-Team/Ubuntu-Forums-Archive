### Post by cholericfun on 2009-03-25
Hi,

as such there should be no harm using adobe flash player, you probably wont need it though, theres other flash plug ins for firefox that should do the job.
synaptic is what you want to use as a beginner.
its a graphic user interface for searching/downloading & installing software for ubuntu.
(System -> Administration -> Synaptic)
in synaptic you may want to (Settings -> Repositories)
in order to activate 3rd party software e.g. adobe flash player.

theres a few threads here for fresh installs. might give them a read if you didnt already. although i would recommend to download/install only updates and stuff you KNOW you need...

### Post by tom56 on 2009-03-25
The easiest way to install Flash is to simply visit a website that needs it, such as YouTube. Firefox will prompt you to install it, just follow the instructions making sure to pick "Adobe Flash".

### Post by Hospadar on 2009-03-25
ditto tom56, I always just go to some random flash game website, firefox prompts me to install flash, I say yes to the adobe flash plugin (there are some free-as-in-speech flash players but they don't work as well), and it's installed and working.

If you'd prefer to install it manually though (in synaptic or on the command line) the name of the package is "flashplugin-nonfree" (at least in the current version).  If you just search for flash or adobe flash in synaptic I'm sure you'll find it.
