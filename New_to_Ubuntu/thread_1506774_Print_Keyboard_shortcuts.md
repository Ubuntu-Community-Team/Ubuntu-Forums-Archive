Is there a way to save and print the customized keyboard shortcuts I have set up so I can view them in the event of a crash or freeze up?

---

### Post by apjone on 2010-06-10
I think these are the two files you need

Gnome Custom Keys:
/home/username/.gconf/apps/gnome_settings_daemon/keybindings/%gconf.xml

Gconf-Editor Custom Keys
/home/username/.gconf/apps/metacity/global_keybindings/%gconf.xml
