### Post by Steve.the.Goolie on 2011-09-08
Well, it turns out I had more problems like,
No window decorators
Unable to open anything on the "Places" menu
USB drives not recognised

ALL of these problems were confined to my main user account, so I created a brand new user and copied the hidden directories over to my /Home directory one at a time, logged out, then logged back in until the problem was solved.

The directory that did the trick and solved ALL my problems in one go was the .local directory.

I lost some customisation of the main menu, and a few  programme shortcuts, but I was happy to live with that.

Of course, I have no idea what the problem actually was!
