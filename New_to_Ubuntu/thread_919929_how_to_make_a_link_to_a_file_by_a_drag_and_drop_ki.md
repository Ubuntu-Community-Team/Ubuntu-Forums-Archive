### Post by dioltas on 2008-09-14
Try right clicking and selecting make link. Then drag the link (shortcut) onto the desktop.

That works in gnome. Are you using gnome or kde?

Edit:  A faster way to do it is to hold ctrl and shift and drag the file onto the desktop.

### Post by starcannon on 2008-09-14
Middle-Click drag the file to the location you want, choose make link from the choices given when you perform the middle click drag.

### Post by t0p on 2008-09-14
**OR!!** (cos there are always a multiplicity of ways to do anything in *nix) :) You want it on your desktop, right?  Right-click on the desktop, select **Create Launcher**, select type as **Location**, click **Browse** to find the file you want and select it clickety-click.
