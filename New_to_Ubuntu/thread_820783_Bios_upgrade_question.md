### Post by Hospadar on 2008-06-06
Make sure you read the bios revision documentation before you attempt this.  It's possible to cause your computer to become non-booting when you flash the bios.  It's not difficult, and it's designed to be updated now and then, but if it gets messed up there's really no way to fix it other than sending it to the manufacturer.

Also, I think there are some boot options that will allow you to boot on older bioses.  You may want to look into that.  I think the deal is that some linux kernel features require certain bios things, but these boot options will disable those features.  Some googling might help
