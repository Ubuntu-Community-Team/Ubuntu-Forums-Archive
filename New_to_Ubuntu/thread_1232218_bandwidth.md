### Post by The Cog on 2009-08-05
> **polarbear12345 said:**
> in computer networking bandwidth is max data rate or max signaling rate??

It normally refers to the data rate in bits per second. Using multi-bit symbols, the data rate can significantly exceed the baud rate.

Beware that many computer applications report their data throughput in Bytes/Sec rather than bits/Sec, which can be confusing.
