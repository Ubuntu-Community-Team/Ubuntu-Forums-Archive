### Post by BC59 on 2011-11-26
You can run Ubuntu exactly as you said.

Burn a CD with Ubuntu and boot your computer.

Then choose Try and not Install. It will start Ubuntu normally but without affecting your hard drive.

Now you have to search and find where is the windows partition and make the movements you want.

Try to not affect windows system files because you will cause a crash on windows. Sometimes when you make changes to such a partition, the computer starts a CHKDSK for various minutes.
