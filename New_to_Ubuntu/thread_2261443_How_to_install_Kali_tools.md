### Post by Sam_Osman on 2015-01-19
To install Kali tools you have to know the name of the tool you want and type "sudo apt-get install tool_name_here" without the quotation marks and underscores.  For example, if you want to install airbase-ng you would type "sudo apt-get install airbase-ng". If you don't know the name of the tool you want to install, just google search and find out what tools Kali comes with.  If all else fails you could try posting in the Kali forums.
