### Post by lswb on 2009-11-22
It is very easy to make it display a single picture. Just right-click on the folder, and select properties. When the dialog opens, click on the picture of the folder icon. A file selection dialog will open where you can navigate to the folder itself and select one of the pictures contained there as the new icon.

I don't know of an automatic way to make it display more than one picture but you could manually combine a few or use a screenshot of a preview page of some kind, then use that for your icon.
