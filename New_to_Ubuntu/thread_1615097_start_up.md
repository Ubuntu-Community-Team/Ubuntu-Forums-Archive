That probably means that the Xserver didn't start up correctly. This is actually the normal login screen without the graphics. Type in your username, enter and then your Passwort.

After that type startx. 

If this error was caused by a nvidiadriver and startx only doesnt work for you type:

sudo apt-get purge nvidia -glx

and try startx again
