### Post by kansasnoob on 2008-09-04
> I have ubuntu installed and have created a partition for the windows install, but the recovery disks from toshiba only give me the option to install over the whole harddrive. is there anyway to cheat this?

When you created that partition did you leave it "unformatted"?

If you format it as NTFS or FAT32 or anything else XP tends to see the whole disc as "used".

The area you're going to install to needs to be "unformatted"! Greyed out!

Look here:
