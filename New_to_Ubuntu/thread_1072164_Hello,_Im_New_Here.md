### Post by tarps87 on 2009-02-17
> **sthrnpagan said:**
> My Licensing beliefs are somewhat relaxed. Where may I obtain the ubuntu-restricted-extras and compiz?

The easiest way is with apt-get
Open a termanal (applications -> accessories -> terminal)
```
sudo apt-get update
sudo apt-get install ununtu-restritced-extras compizconfig-settings-manager
```
(This is only easier as I can remember the package names :))

The other way is under applications add/remove and then search for the program you want.
(You may want to select all applications in the drop down menu, it will make sense when you use it)

Once it loads, you'll find Synaptic Packet Manager on the toolbar (similar to startbar of windows) System->Administraton->Synaptic

This is where you can choose from a library of almost 30,000 programs to install.. "ubuntu-restricted-extras" is a meta-package that links to all the other goodies :)

Compiz can be turned on under System->Preferences->Appearance->Visual Effects, it'll probably ask you to install a proprietary Nvidia/ATI driver.. hopefully all nice and smooth ;)
