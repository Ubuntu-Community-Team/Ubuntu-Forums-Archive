### Post by prshah on 2008-08-22
> **Lorelei- said:**
> So now that I finally have a laptop running Ubuntu I am slowly getting to grips with command line, but I was wondering if anyone could recommend any good resources (online or books) to help a newbie get up to speed with command line quickly?

"man"ual and "info"rmation are your two best friends; Ex:```
man ls
info coreutils 'ls invocation'
```

To get a summary list of commands and actions, use ```
info coreutils
``` Navigate the cursor to the command for which you want more information, and press enter to read up on it.
