### Post by freeman2000 on 2009-04-20
I'm not quite sure I can help you, but I'll try.  I'm also running an HP laptop, a DV6000t series, with an AMD64 2.0 ghz duocore2 processor and 2gb ram.  I'm dual-booting between Ubuntu 9.04 and Windows Vista without any problems.  

The Power Management's main function is to conserve battery life/power when not plugged in, irregardless of what OS you are running.  The heat from the power adapter is more a function of what you are doing, independent of the OS.  For instance, my power adapter will run hotter when my battery is in charge mode than when fully charged.  Also, it will run hotter when the computer has to run harder, such as when dealing with lots of graphics, especially if I'm playing some music along with it, whether a CD or an online radio station.  If all I'm doing is email or searching the forum, it runs cooler. 

It doesn't matter what OS I'm running at the time.  Good luck.
