### Post by empty_spaces on 2010-05-10
Quite strange, because I experienced the stuck control key a couple of hours ago, so I restarted my laptop, and its been working ok since. Will keep monitoring to see if it comes back.

Now coming to your other issue about not wanting to reinstall every 6 months, here's a newsflash - you don't have to.
At the very least, with regular ubuntu releases, you should be able to run your system for 18 months, and 36 months if you stick to LTS.

I don't have a bad experience with upgrades, because I don't do upgrades. With upgrades, you are relying on the installer to correctly replace thousands of system files. The more customized your install, the more dicey your upgrade.

Ordinarily, since I always do fresh installs, you would think my system would be unusable for a day or two while I set up the new install. 
Not a chance, because that's where the planning comes in. I have already set up 2 partitions on my laptop that I alternate between at release time.
So I have a fully intact, working 9.10 install on one partition, and a new 10.04 install on another. Over a couple of weeks I slowly set up the 10.04 install in my spare time, and see if there are any serious bugs.
So I always have a stable, working 9.10 install to fall back on before I start using 10.04 fulltime.

Given your bad experience with upgrades, you should be looking to do something similar.
