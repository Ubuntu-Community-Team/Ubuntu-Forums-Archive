I'm completely new to Ubuntu so this is probably a really basic question, but the other day I was messing around with some settings and I think I changed something but don't know what.

Anytime a new window pops up it will usually go transparent.  This would be quite useful if I didn't want to use that particular window, but this happens with most windows as soon as I open them.  For instance, I opened the terminal window and began typing, only to have it go transparent while typing.  They are not completely transparent, but almost.  Usually moving the mouse over them eventually makes them re-appear, but sometimes it takes a few seconds before it comes back.  I don't know if this is a bug, like I said I was messing with some settings, and because I'm completely new to Ubuntu I figure it was a setting I changed.

Thanks in advance.

### Post by mbzn on 2010-05-11
I would advise you to go through your compizConfig settings manager(Settings>Preferences>CompizConfig settings manager) and look for 'Opacify' and disable that.

Edit: It's not a bug(Some people like it that way)
