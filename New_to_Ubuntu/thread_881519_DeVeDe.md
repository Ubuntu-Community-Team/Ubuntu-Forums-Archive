### Post by gn2 on 2008-08-06
Use the option in DeVeDe to "create a DVD .iso image ready to burn to disc", and choose a file name and location.
When it's finished creating the .iso image use Brasero to burn the created image to DVD. It works a treat.

### Post by Malac on 2008-08-06
> **Ginoxy said:**

Some players are designed to not play recordable discs. (Half-baked attempt to stop pirating I think).
 
I have one player in the bedroom which will not play recorded DVD of any type but the same discs play perfectly well in the player in the living room.
 
If you have only the one player then try a recorded disc you know works, say from a friend that's played it on their player, and see if that plays on yours. If not then you will need to get a new player........ (or some can be altered by adding a chip) :wink:
 
Hope this helps.
