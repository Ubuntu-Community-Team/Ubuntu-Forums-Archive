### Post by heyho on 2011-02-18
You should be able to run Ubuntu no problems on that machine.

Don't worry about drivers, as almost all of them (if not all) will be installed with Ubuntu, and any others can usually be downloaded and installed automatically after the install has finished.

Simply download the CD image (Personally I would recommend 10.04, as it is a long-term-support release) and burn it to disc, then boot from it.  The rest is quite self explanatory. 

Use the 32 bit, 10.04 from here:

### Post by fabricator4 on 2011-02-18
The driver page you pointed to appeared to be XP drivers.  They won't work for Linux/Ubuntu.  Don't worry about it, most likely Ubuntu will work fine straight out of the box.

Can you do a CD install onto that laptop?  If so, download the live CD iso and burn it to a disk.  Boot off the cd and click "Try Ubuntu".  It will be much slower than booting off the hard drive so do not worry.  If everthing appeas to be working OK click on the "Install Ubuntu" launcher and you're on your way.

I recommend 10.04 LTS as it's very stable and solid. The recommendation for Lubuntu is a good one but you might want to get both Lubuntu and Ubuntu 10.04 and try them both.

### Post by Old_Grey_Wolf on 2011-02-18
Ubuntu will run well on that computer; however, obviously not as fast as a modern machine. :)

I have it running on a 1.7 GHz single-core processor, 512 MB RAM, 60 GB Hard Disk, 12 MB video RAM.

You could try Ubuntu, Kubuntu, Lubuntu, and Xubuntu to determine what your uncle may be more comfortable using. 

You should also consider how often you want to update the computer. The LTS releases; such as, 10.04 are supported for 3 years. 

I would try running a "live CD" of one of the Ubuntu's of your choice to see how compatible it is with the hardware. If all goes well, you will not have to do anything concerning drivers for the hardware. The ATI Radeon graphics chip or the un-identified wireless chip may cause you some difficulty; however, post to the forum and someone will try to help you..
