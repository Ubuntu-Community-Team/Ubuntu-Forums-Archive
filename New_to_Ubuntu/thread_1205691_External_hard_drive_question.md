### Post by Paqman on 2009-07-06
> **LinuxFox said:**
> I just wanted to know since I remember being able to do this on Windows in 2005.

Old Windows games tended to be pretty self-contained, but modern ones install themselves into the operating system just like any app. If you just want to run some old DOS games or something then you should be ok, but generally you can't install software onto an external drive these days.

Also, as mentioned, USB external drives can be pretty slow.
