If you are installing Windows as a virtual machine "guest" operating system, you will need to register it with Microsoft, just as you would if you were installing it on a "real" computer.

Ubuntu requires no activation code, and you may install it in as many virtual machines as you wish. :)

### Post by empty_spaces on 2010-10-15
Hopefully, this answers your question.

Virtualbox is used to create virtual machines - ie Virtualbox let's you create virtual hardware such as a virtual hard disk to install on, a virtual network device to access the internet, etc.

So everything that you would need to do to install windows/ubuntu on an actual machine with physical hardware (including product key and activation), you would need to do the same to install Windows/ubuntu on virtual hardware created by Virtualbox

### Post by AndyM3 on 2010-10-15
VirtualBox, QEMU, Xen and the likes let you create *virtual machines*, which you can then run an OS on, just like a real computer. The guest OS usually won't know the difference, and with Windows you do have to enter a valid registration code, which requires having a purchased copy. You cannot install the same copy on both a real computer and a virtual one because Windows Genuine (Dis)advantage won't let you. And it's against the EULA.
Or you can crack it, of course, but that's **illegal** and strongly discouraged by everyone on this forum.
