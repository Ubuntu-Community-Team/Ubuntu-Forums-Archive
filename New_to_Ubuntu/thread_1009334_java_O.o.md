### Post by Therion on 2008-12-12
You can install Java, by installing the **ubuntu-restricted-extras** package.

From the terminal:```
sudo aptitude install ubuntu-restricted-extras
```

Or open your Synaptic Package Manager:

Search for **ubuntu-restricted-extras** mark the package for installation and then click Apply.
