### Post by gjoellee on 2009-01-19
Ubuntu will never come with Urban Terror installed by default. You can however download Urban Terror again, in .DEB format here:
The DATA package: [http://www.getdeb.net/download/3041/1](http://www.getdeb.net/download/3041/1) (must be installed first)
The menu package: [http://www.getdeb.net/download/3041/0](http://www.getdeb.net/download/3041/0) (this add Urban Terror to the menu)

Or you can install Urban Terror in an other way. Look here: [http://www.kshoster.net/content/howto-install-urban-terror-linux](http://www.kshoster.net/content/howto-install-urban-terror-linux)

You can use Ultimate Edition which is Ubuntu, but with a lot of more stuff pre-installed. If you download and install the gamer edition you will get a lot of more games install as well...including Urban Terror. See this link: [http://ultimateedition.info/ultimate-edition-20/](http://ultimateedition.info/ultimate-edition-20/)
