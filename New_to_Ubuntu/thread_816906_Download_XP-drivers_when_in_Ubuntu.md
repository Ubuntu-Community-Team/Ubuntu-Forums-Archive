So in XP there is no internet connection, but in Ubuntu there is. Can I download the XPdrivers while I'm online in Ubuntu?

Can I store the XP-drivers from Ubuntu on the XP-partition (NTFS)? Or must I create an NTFS partition which they can both read, first?

### Post by hyper_ch on 2008-06-03
yes, you can download them

if you install ntfs-3g you can store them directly on the ntfs drive

if you have usb thumb drive you can store the drivers on there and access it by xp

xp can make use of tar files, just not natively. Total Commander, a nice file manager on windows, has the ability to use .tar archives.
