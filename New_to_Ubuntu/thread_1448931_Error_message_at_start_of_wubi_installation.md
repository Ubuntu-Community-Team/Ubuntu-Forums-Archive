### Post by harry003 on 2010-04-07
When I ran the wubi installer under Windows XP, I immediately got a dialog box with an error message:


Exception Processing Message c000013 Parameters 75b6bf7c 4 75b6bf7c etc


The message offered to Continue - Cancel - Try Again but the box just kept popping up.

The machine is looking for removable media such as flash drives and SD cards. I had to push Cancel well over 100 times but eventually I satisfied it and moved on.

Don't give up. As frustrating as it is, I was finally able to get a good wubi installation and the problem has not turned up again. I think that you will be much happier with wubi than with virtual machines, which are slower and more cumbersome, as well as lacking in capability.

Good luck.
