### Post by CatKiller on 2011-07-19
To expand on the other posts, the LTS releases are supported for some time after release. As such, the software they install could be a long way out-of-date. There will generally be point releases that incorporate those updates at the time that the point release comes out.

This will also still need updating (just like all releases), but hopefully not as much.

- When you download Ubuntu 11.04 (say, from [http://www.ubuntu.com/download/ubuntu/download](http://www.ubuntu.com/download/ubuntu/download)), the downloaded file is called an ISO.

- Except for the rare update to the actual ISO file, the means for existing and new users to get the latest updates and bug fixes to 11.04 is via the Update Manager.
- Thus, even if you download and install the 11.04 ISO today, this file does not include all the updates and bug fixes made to 11.04.  You need to run Update Manager to get those.
- You will know when the downloadable ISO is updated because the version number will be noted on the web page – for example, it will be something like 11.04.1.  [I'm guessing on this last point.]
