### Post by drs305 on 2008-11-22
If it's your panel you've lost, do you have any other panels displayed? If so, right click, select New Panel, and when it is created you can either drag it to where you want it (l,r, top, bottom) or right click, select Properties, and set the position.

If by task bar, you mean you can't see open apps: right click on the panel, Add to Panel, and select "Window List".
