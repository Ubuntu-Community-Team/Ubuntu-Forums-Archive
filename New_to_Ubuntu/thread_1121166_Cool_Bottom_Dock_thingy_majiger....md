### Post by Jakey_TheSnake on 2009-04-09
I've always preferred Cairo-Dock to AWN. 

Cairo-dock: [http://prdownload.berlios.de/cairo-dock/cairo-dock_v1.6.3.1_i686.deb](http://prdownload.berlios.de/cairo-dock/cairo-dock_v1.6.3.1_i686.deb)

Cairo-dock plug-ins: [http://prdownload.berlios.de/cairo-dock/cairo-dock-plug-ins_v1.6.3.1_i686.deb](http://prdownload.berlios.de/cairo-dock/cairo-dock-plug-ins_v1.6.3.1_i686.deb)

Cairo-dock (extra) themes: [http://prdownload.berlios.de/cairo-dock/cairo-dock-themes-1.6.3.1.tar.bz2](http://prdownload.berlios.de/cairo-dock/cairo-dock-themes-1.6.3.1.tar.bz2)

You can run it at start up by adding the command "cairo-dock" to your System > Preferences > Sessions, and you can change theme and preferences once it has started by right clicking on it and from there it's pretty intuitive =)
