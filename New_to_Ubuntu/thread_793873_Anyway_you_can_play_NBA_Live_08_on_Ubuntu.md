### Post by TeoBigusGeekus on 2008-05-14
Install wine
```
sudo apt-get install wine
```
and then install your game.
According to this page
[http://appdb.winehq.org/objectManager.php?sClass=application&iId=6779]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=6779")
it works!
