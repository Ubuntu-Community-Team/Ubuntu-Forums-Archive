### Post by sisco311 on 2009-01-27
The admin password is your user(log in) password.
[uhelp]community/RootSudo[/uhelp]

You can change it in System->Administration->Users and Groups
or Applications -> Accessories -> Terminal:
```
passwd
```

If you have forgot your password:
[http://www.psychocats.net/ubuntu/resetpassword]("http://www.psychocats.net/ubuntu/resetpassword")
