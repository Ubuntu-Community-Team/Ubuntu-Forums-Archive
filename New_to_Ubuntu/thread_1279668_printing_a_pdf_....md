It may depend on your printer. The Canon laser printer I use has option for double sided, one two or more pages per side, top or side guttering etc. To print two pages per side I select print and then the page setup tab. Under that tab I select 2 pages per side.

### Post by expatCM on 2009-10-01
yes, spot on, thank you.

I went to File / Print / Page Setup and there selected the number of pages to be printed on a page.  It works.  The only important thing to remember is to change the page orientation to Landscape.
