### Post by gandaran on 2009-05-02
> **a.v.l said:**
> I have two desktop computers and a laptop. all running Ubuntu, but I can't work out how to network them so that I can press PRINT on any of them and my printer print out.

Perhaps someone has a simple "hold your hand" set of instructions I could use. Thanks.
this is easy if the printer is connected to one computer running ubuntu, (this PC must always be running) for ubuntu 9.04 jaunty on this PC (for other ubuntu versions it's more or less the same) open systen » adninistration » printing » server » settings » check-mark show printers shared by other systems and publish shared printers connected to this system, now you should be able to find the printer on the other computers and set it default printer.

alternately, if the printer has an ethernet port, you can simply plug it into your router and install it using the printer dialog.  It should automatically be detected if you select network printer.
