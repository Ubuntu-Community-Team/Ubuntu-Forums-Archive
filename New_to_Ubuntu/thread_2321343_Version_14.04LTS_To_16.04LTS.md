### Post by UbuntuAddict99 on 2016-04-26
Hello [**[COLOR=#000000]Rifqi_Arief[/COLOR]**]("http://ubuntuforums.org/member.php?u=2028974"), and WELCOME to Linux Ubuntu.

Now i agree with our fellow Ubuntuers, you seem to have received the beta release. But the final version is available now and the notification for the update is defined in the System Update settings.

System Settings -> Software & Updates -> Updates -> Notify Me Of A New Ubuntu Version. (The little drop/check box)

Now if it is set to any new version it will tell you soon as it can of any Ubuntu new release. But if it is for LTS Long-Term support versions only then it could be a bit longer. But it is irrelevant you can just download it now anyway. But PLEASE remember to back up your data before upgrading.

P.S. I have read yesterday that 14.04 LTS will still be supported until 2019 but i could be wrong. Only upgrade when you are ready and also within a reasonable time that is required to do so.

Happy Linuxing.
