### Post by Temposs on 2010-10-13
Using KDE would be probably be a pain if you're not already using it.

I would suggest that you install the **CompizConfig Settings Manager** app. It lets you manipulate the graphical effects of your desktop. You should use the **Opacity, Brightness, and Saturation** section in that app.
