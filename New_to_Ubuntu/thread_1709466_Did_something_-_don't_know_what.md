The "something" is that the preferences for "Open folder" type operations has been changed.

Do you have a folder on your desktop? If not, make one.  Right click on this folder and select "open with other application".  Make sure you have the "remember this application for folder files" checkbox ticked, then select "file browser" as the application, then click on the "open" button.  Everything should now be fine.
