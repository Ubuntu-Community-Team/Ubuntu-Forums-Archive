### Post by kurt18947 on 2011-10-09
You could create a live CD/DVD or Live USB drive and try without installing to check hardware compatibility. You will not be using any proprietary drivers initially.  If your machine will support booting from a USB device, a Live USB drive will boot faster and can have persistence to remember changes in setttings.  A Live CD cannot have persistence.  Lubuntu seems quite light.  I just installed it on a ThinkPad R31 with 512 MB. RAM and 1.1 Ghz PIII processor. With one chrome browser window open System Monitor shows 101 MB. RAM being used. Lubuntu & Xubuntu can both use Ubuntu's Repos and update mechanisms which work quite well.
