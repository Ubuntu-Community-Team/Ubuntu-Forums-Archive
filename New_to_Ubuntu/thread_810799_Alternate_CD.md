### Post by Famicommander on 2008-05-28
The LiveCD is a way to demo the Ubuntu desktop environment without actually making changes to your computer. You can try out the operating system to make sure you're comfortable, and then install it from there.

The Alternate Install CD is how you can install Ubuntu if your computer doesn't have the hardware capabilities to run the Live environment.

### Post by FuturePilot on 2008-05-28
The Live CD allows you to boot from the CD into a fully functional desktop. This is typically the method of installing Ubuntu. The Alternate CD basically uses the Debian installer which is a text based installer. You can't boot into a desktop with this CD.

Heh.  The LiveCD is 699 MB while the Alternate is 697 MB.  Not a very big difference :-)

Things you can do with the LiveCD that you can't do with the Alternate:
- Boot into a functional desktop, check that your hardware is recognized and working properly, get a feel of how Ubuntu looks and works.
- You can boot into a system that had the boot partiotion damaged so you can't boot into it the "standard" way and try to recover the damaged partition(s)
- Browse, e-mail, write, read, whatever, all without accessing the host system's disk (unless you specifically mount a partition in order to save your work)

Things you can do with the alternate CD that you can't do with the LiveCD:
- Install Ubuntu with a fully encrypted file system
- Install a minimal version of Ubuntu (without any graphical environments) in order to customize your installation
- Install Ubuntu on a system with too little RAM for the LiveCD to work reliably, or in some cases on a system where the LiveCD screws up graphic card detection
