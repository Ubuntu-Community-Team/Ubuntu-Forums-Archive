### Post by wojox on 2009-09-26
I've had this happen and had to hack it like so:

```
gksudo gedit /etc/X11/xorg.conf
```Take out  Option         "metamodes" "1152x864 +0+0"

Then add:

```
Option       "metamodes" "CRT-0: 1440x900 +0+0"
```

And reboot.
