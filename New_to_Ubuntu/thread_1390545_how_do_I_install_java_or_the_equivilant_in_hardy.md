### Post by sricks3 on 2010-01-25
If you're talking about the Java Runtime Environment (so that you can run programs and applets written in Java), the above recommendations should help you out.

However, if you want to do any of your own programming you'll be looking for a development toolkit. You have a couple of options here. The default package supported by Canonical is openjdk-6-jdk. If you want the package maintained by Sun, you'll need sun-java6-jdk. If you dig around a little, you can probably find a few other options as well.
