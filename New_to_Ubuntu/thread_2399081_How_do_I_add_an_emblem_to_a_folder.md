### Post by ajgreeny on 2018-08-21
Xubuntu is different from Lubuntu and uses thunar as file manager, which does, and has for a long time, offered emblems, whilst pcmanfm in 18.04 does not.

You could try using thunar as your file manager instead of pcmanfm, though I do not know if there are any "knock-on" effects from doing so.
