There are some, but why are you worried about this?  Linux is very secure and there is no need for a virus scanner. I've been contractually required to run AV on Linux a few times in order to have E&O insurance as mandated by a client. The mandate was to run it, keep both the software and the signatures current.  It never found anything.

### Post by TheFu on 2019-05-23
A little clarification.  /home != $HOME == ~/  

/home is just a directory, usually locked down to root only.

$HOME and ~/ is the HOME for the current userid.  There is absolutely nothing that says those need to be anywhere under /home/{userid}. Nothing.  In a corporate environment, that is actually very unusual since HOME directories are usually NFS mounts.

In a non-residential setup, always use **getent passwd** to determine where any specific userid's HOME is located. Never assume it is under /home/
