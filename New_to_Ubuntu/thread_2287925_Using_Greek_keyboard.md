### Post by sotiris2 on 2015-07-23
Simple stress accent is done by pressing ; key (next to L) then the vowel. &#964;&#972;&#957;&#959;&#962;
Diaeresis (&#948;&#953;&#945;&#955;&#965;&#964;&#953;&#954;&#940;) are done by pressing shift + ; and then the vowel. 
Stress and diaeresis is done by pressing shift +; , then the ; alone and then the vowel.

### Post by Vladlenin5000 on 2015-07-24
> **HappyAsHellas said:**
> The keyboard I have (Greek [COLOR=#ff0000]eliminate dead keys[/COLOR]) doesn't seem to work as I get &#949;;&#953;&#956;&#945;&#953; for some strange reason. Is there a different layout I should be using?

Your problem is marked in [COLOR=#ff0000]red[/COLOR] above. You need the exact opposite - with dead keys - in order for the aforementioned method to work. "Dead keys" are the keys with the diacritic symbols you need to combine with the letters to produce an accentuated character and they're called "dead" in this context because with this settings, on their own those keys produce nothing. They produce an accentuated character only when the character to be accentuated is typed after the (dead) diacritic symbol.

By selecting the option without dead keys you're effectively telling the system to NEVER produce an accentuated character and use diacritics as characters on their own.
