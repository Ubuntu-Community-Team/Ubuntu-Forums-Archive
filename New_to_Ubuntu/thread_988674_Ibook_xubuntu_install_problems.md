### Post by Keen101 on 2008-11-20
I don't own a mac, and haven't used a mac in awhile, so i can only give you ideas.

You must press the c key as soon as you hear the chime (bong).

if that doesent work, i've heard holding the command button down at boot will give you options to boot different os's and/or cd's.

also, assuming you can finally figure out how to boot a live-cd you may need to find a version that supports PPC processors. I think the latest versions only support intel based macs. There are some versions out there that run on PPC machines, but you might have to google for one.

When i tried booting a live-cd on an emac i think a regular live-cd didn't even get recognised, and it only worked when i had downloaded a PPC version.

### Post by cuvtixo on 2008-11-21
Explain what you mean by holding down "start" at the command line??

I've got a G3 ibook in front of me, and I happen to be planning to update from 7.1 Xubuntu PPC to 8.1.  
I get "press 'l' for linux and 'c' for cdrom" at startup.  There is no 'start' button, and if you see one, then it's not a command line!  You also might need to read up about "Open Firmware" It's kinda the Apple PPC version of BIOS, or the Macbook version of EFI, if you're familiar with that.  Open Firmware was also used with old Sun servers. 
Even Apple "Geniuses" aren't familiar with Open Firmware anymore, so please don't take it as an insult...
