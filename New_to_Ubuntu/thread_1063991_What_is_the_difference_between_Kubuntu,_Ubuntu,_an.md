### Post by Firestem4 on 2009-02-08
Typically the Xcfe Desktop Enviroment is a very lightweight and uses very little resources.

Ubuntu is a more Mac-like interface (imo), and uses more resources than Xcfe interface. 

Kubuntu actually uses more resources than either because it is a bit flashier, so it is a little more resource intensive. You could say it looks similar to a windows-style system, which it is. 

Each GUI has its own set of native Linux apps. Even though you can univsersally use any ^buntu variants programs, you have to download the libraries and extra files, so if hard drive space is an issue, this would be your only issue. 

Depending on how old the computer you are installing ^buntu on, you should be aware that while each linux GUI is lightweight (especially when compared to any windows or mac), KDE runs slower on much older systems compared to GNOME, or Xcfe
