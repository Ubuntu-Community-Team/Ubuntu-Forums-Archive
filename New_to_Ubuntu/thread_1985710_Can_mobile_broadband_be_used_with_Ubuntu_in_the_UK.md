### Post by raziel09 on 2012-05-23
Yes it is possible to use mobile broadband on ubuntu - i've been doing it since 9.04 but back then you needed to install the usb switch package for it to recognise that this is a modem.

In 10.04 you shouldn't need that package. Just plug the device in and wait for it to mount; you will be able to set it up in network manager by going to the mobile broadband and clicking on add and following the wizard.

You won't be able to install the windows\mac software that comes bundled with it but also won't be able to view your usage.
