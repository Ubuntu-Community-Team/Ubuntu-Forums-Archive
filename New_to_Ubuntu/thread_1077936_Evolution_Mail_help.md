### Post by Ms_Angel_D on 2009-02-22
In the Add/Remove Programs install Alltray then rightclick on your evolution icon and click properties under command enter this:

```
alltray evolution --component=mail
```

Now when you open Evolution it will automatically have a system tray icon ;)
