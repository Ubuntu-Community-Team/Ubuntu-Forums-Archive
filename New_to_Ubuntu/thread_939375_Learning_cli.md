### Post by NTolerance on 2008-10-05
This site does a fairly good job at explaining how the command line works in Linux:

[http://www.linuxcommand.org/](http://www.linuxcommand.org/)

Most Linux distributions use a program called "bash" for command line functions.  As you go on you'll see how "permission denied" is a GOOD thing.  It's what keeps Linux from getting ruined like oh so many Windows installations.  Look here for for info about how Ubuntu uses "sudo" for access to important parts of the system:

[https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo)
