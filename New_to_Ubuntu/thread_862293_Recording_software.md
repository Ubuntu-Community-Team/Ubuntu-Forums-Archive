Cheers

Welcome! Check out the Ubuntu Studio website: [http://ubuntustudio.org/](http://ubuntustudio.org/)

It is a version of Ubuntu custom-tailored to musicians!
Even if you don't want Ubuntu Studio, you can install any of the applications individually in Ubuntu/Xubuntu/Kubuntu. Audacity is a good, simple sound editor, and Ardour is a full-featured application similar to ProTools.

Good luck!
