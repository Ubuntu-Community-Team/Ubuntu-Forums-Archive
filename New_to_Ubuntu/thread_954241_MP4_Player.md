### Post by prshah on 2008-10-21
> **JoBangles said:**
>  When pasting MP3 file into MP3 Directory an error window pops up saying preparing to copy then after a while says input/output error. 

Had the same problem with an el-cheapo "MicroMax" MP4 player; turns out the battery was totally drained. Everything worked fine after charging it fully (6 hours / overnight). For some reason, if battery is low, I/O doesn't work, even though the battery is charging fine. Anything below 15% battery, I/O becomes unreliable.

Note that the "mp4" player will play only "converted" files; the included conversion program is Windows only, and the output format is some "unknown" format. This is in the case of my player and may/may not apply to you. No problems with MP3, WMV, etc, only videos.
