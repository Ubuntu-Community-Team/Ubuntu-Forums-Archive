### Post by angelsguitar on 2008-09-20
You could try using a download manager like Downloader for X.  The package's name is d4x; you could either install it using Synaptic Package Manager, or using a terminal with the following command:

```
sudo apt-get install d4x
```

Downloader for X has three configurable download speeds you can choose from.
