### Post by Idefix82 on 2008-11-21
You can do all the partitioning required when you start installing Ubuntu. You can then create new partitions in the unallocated space, delete existing partitions or resize partitions. You don't need to delete any data that you already have.
Once you start installing, feel free to report back here and to ask questions if you don't know what to do.

### Post by hyper_ch on 2008-11-21
before you partition your drives make sure to

(a) have backups (you should always have backups that are current)
(b) defrag the drive 2-3 times in windows
[(c) use the VISTA partitioner to make "empty disk space" if you're on VISTA]
