### Post by munkyeetr on 2008-08-20
One way to accomplish this is to do the following:

1) *Right-click* on a folder (any folder) and choose **Properties**
2) Select the **Open With** tab
3) Click the **Add** button, choose the program you want to use (xmms), and click **Add** again
4) IMPORTANT: Make sure you re-select **Open Folder** as the default action on the **Open With** tab.

Done.

Now when you right-click on any folder, you should have an option to **Open with "xmms"**
