The errors I was getting were:

Could not open location 'network://'
The specified location is not supported.

Now I got it fixed when I installed the packages I had on 12.04 onto 14.04 using:
[B]$  sudo  dpkg --set-selections < installed-software
$  sudo  apt-get -u dselect-upgrade[/B]
