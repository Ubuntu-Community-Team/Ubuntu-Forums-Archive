### Post by Dreamer Fithp Apprentice on 2014-03-22
Not at all sure I can. Sounds like you might know more about it than I do. However, since nobody has stepped up with an answer in 24 hours here's my first thought for what it is worth:

Try pavucontrol. Install it if you don't already have it. Run it (just "pavucontrol" from the command line). It has an "output devices" tab with a dropdown menu of devices. And I believe more than one way to set volume. And do make sure you aren't muted. And you might want to make sure of whatever device you are plugging into that jack. Good luck.
