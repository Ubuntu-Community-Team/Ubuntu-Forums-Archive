### Post by amauk on 2010-12-09
If you're using Evolution (which is the default mail client), then:
Edit > Preferences
Select the appropriate mail account, and "Edit"
Under the "Identity" tab, "Add new signature"

This brings up a nice GUI word-processor-like interface (inc. the ability to add images)
Make your signature however you want, and click the save button
