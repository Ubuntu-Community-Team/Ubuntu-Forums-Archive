### Post by coldcritter64 on 2014-07-27
VLC, use the Media menu > Open Directory. Add files to a directory open the directory with vlc and all media in the directory are added to the vlc playlist.
Or you can open the playlist editor and drag and drop files from multiple locations to the playlist there as well. Cheers.
