### Post by damis648 on 2008-08-09
You should **not** expect problems. The only differene is that you are now running a 32-bit OS on a 64-bit capable processor, so you just will not be using the 64-bit processing capabilities of the processor, but there is no real reason to switch to AMD64. Had I installed i386 already, I wouldnt bother reinstalling the 64-bit version.
