### Post by M7S on 2009-11-03
Hi. The DockbarX developer here. Since noone has answered your message I'll try to. DockbarX isn't just launchers, it replaces gnome's window list. All programs (that has windows), will show as icons in dockbarx, as well as the launchers you add. Multiple windows for the same program will use the same icon. If you launch a dockbarx launcher, the launcher will change to a program icon. It's pretty much like the superbar of win7.

On the question if DockbarX interferes with anything... there are a bug in the last build of global menu that makes dockbarx crash so you can't use those two together. Other than that, none that I can think of. 

I'm not the person to say whether DockbarX is good or bad, since I'm quite biased. I'll just recommend that you try it out and see for yourself.

### Post by M7S on 2010-03-02
The themes sets the gap. To intstall themes, simply copy the .tar.gz file to ~/.dockbarx/themes
You can change the themes from the preference dialog. Open it by right clicking the handle on dockbarx (sometimes you'll have to double right click, don't ask me why) and choose preference.
