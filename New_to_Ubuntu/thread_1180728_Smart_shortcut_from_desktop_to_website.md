### Post by TimeGuardian on 2009-07-08
Actually, its quite easy to do.
 
Just bookmark the dejaclick script using the yellow "star" icon on the deja toolbar, then find and open the properties of the Firefox bookmark you just created.  The file path that is associated with your new bookmark is the same one you should used from the command line.  It should start with the file:// protocol.
 
Nifty stuff.
