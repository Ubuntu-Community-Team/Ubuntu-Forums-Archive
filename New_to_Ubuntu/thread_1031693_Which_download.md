### Post by ELF_O8 on 2009-01-05
> The major difference between 64bit and 32bit is the amount of memory they address.
Right, 32 bit OS's can only address 4 GB of RAM. 64 bit ones, however, can address 18.75 **exabytes** of RAM. Like we'll ever need that much memory.
Anyway, the only real difference that 64 bit show is in programs that require lots of calculation: media conversion, image construction and cleaning, other math intensive programs. These programs see a significant performance boost in 64 bit OS's. This is because 64 bit operating systems have larger memory addresses. Be aware that 64 bit OS's require more memory than 32 bit, simply because of how efficiently and how often they use the RAM.
