### Post by SteveHarrisHolt on 2009-10-24
Finally got this sorted. I installed Kubuntu to a different disk and noticed that the keyboard was very slow (as has been quoted in other threads) but the mouse was fine (PS/2 mouse). I updated the motherboard BIOS then tried Ubuntu Server again and it was fine.

The only reason I can think of is that the motherboard BIOS update included a firmware update for the onboard USB controllers.
