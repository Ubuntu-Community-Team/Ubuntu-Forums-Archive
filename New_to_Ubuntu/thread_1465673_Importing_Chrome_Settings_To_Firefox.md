### Post by hweidi on 2011-05-25
**How to import Passwords from Google Chrome to Firefox:** 


In Google Chrome side: 
 
[LIST]
[*] Install LastPass extension on Google Chrome.
[*] You'll be prompted to create account, so create it.
[/LIST]
 In Firefox side: 
 
[LIST]
[*] Install LastPass Add-on and use same log in information for LastPass account you have created in Google Chrome.
[*] Go to LastPass "Tools" choose "Export to" --> "Firefox".
[/LIST]
 Your passwords are in your Firefox.
