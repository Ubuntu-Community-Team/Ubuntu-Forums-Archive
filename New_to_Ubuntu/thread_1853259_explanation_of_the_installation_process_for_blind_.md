Hi soundwarrior20. Welcome to the forums!

The community have a couple of web sites with help documents and tutorials:
[LIST]
[*][https://help.ubuntu.com/]("https://help.ubuntu.com/")
[*][https://wiki.ubuntu.com/]("https://wiki.ubuntu.com/")
[/LIST]
To make it a little easier, here are 2 links to important information: the system [requirements]("https://help.ubuntu.com/community/Installation/SystemRequirements"), and an installation [guide]("https://help.ubuntu.com/community/Installation"). In a nutshell you download the installation software, either burn a CD or create a bootable USB stick, and restart your machine so it boots from the device that you just created.
