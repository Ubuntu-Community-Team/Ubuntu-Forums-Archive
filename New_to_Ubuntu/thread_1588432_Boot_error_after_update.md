### Post by khelben1979 on 2010-10-06
Skip the updates to have a working system.

The next step would be to learn about the updates and what they do to the system. A kernel upgrade can break your system if the new kernel works badly with your Ubuntu system.

So try and untick any kernel upgrade packages and accept everything else.
