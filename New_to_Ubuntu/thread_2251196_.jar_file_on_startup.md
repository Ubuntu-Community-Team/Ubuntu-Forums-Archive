### Post by pissedoffdude on 2014-11-09
Hi,

No problemo.  Just write a simple script containing 
```
#!/bin/bash
java -jar your_jar.jar
```

Make it executable, and either run it as a cronjob or use Lubuntu's gui for startup applications if it has one.
