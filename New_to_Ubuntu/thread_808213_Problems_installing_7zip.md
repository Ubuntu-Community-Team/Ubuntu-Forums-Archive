### Post by Monicker on 2008-05-26
You won't see a menu item for 7zip, because its a command line application.  It does, however, get integrated with the gui archive manager once it has been installed.  Right click on a folder and select Create Archive, which will give you an option for making a .7z archive.  You can extract a .7z file by right clicking on it also.
