### Post by thomas144 on 2010-03-07
I have one piece of advice. Don't buy a computer with a Broadcom wireless card. My Dell Inspiron 1545 has a Dell 1397 card, with a BCM-4312 chipset or something. I have been trying to get it to connect to my network in Ubuntu for a while, but to no avail.
 
The basic difference between a 32-bit and a 64-bit architecture is that basically, the 64-bit architecture processes data in bigger chunks, so it can run faster. A 32-bit arcitecture is generally inferior, and you should probably go with a 64-bit system. By the way, 64-bit hardware works with both 64- and 32-bit operating systems. The bottom line is that 64-bit systems are preferrable and most things run faster on them. I hope that I gave you some good advice.

### Post by tacantara on 2010-03-07
If I were replacing any computer with the intention of running Ubuntu, I would check the Ubuntu wiki for hardware compatibility, then take a list of hardware known to run well with Ubuntu and go to my local "Mom and Pop" computer store (I happen to have a few around where I live).  From there, I would either buy a case (or re-use my current case if that's an option), the hardware, and build from scratch.  If building your own machine isn't an option, the "Mom and Pop" stores will usually do that for you.  You may find a price advantage in doing that, especially if you build it yourself.  Either way, you get a machine with Ubuntu installed.

### Post by walt.smith1960 on 2010-03-07
I have an IBM ThinkPad R61i that works great with Ubuntu 9.04/9.10. When I bought mine, there were several model #'s. Some had Win 2K drivers available, some did not. The models that supported Win2K/WinXP were a bit more money but mine has been great. The promo models only supported Vista (at the time). I don't know how well supported Linux would have been on those machines with (presumably cheaper) chip sets etc. Lenovo offers a driver matrix to show which models support which operating systems. It's not loading right now (Sunday afternoon) but if the notebook supports multiple versions of Windows including older, I'd say the liklihood of it supporting Ubuntu increase. My video, ethernet & WiFi are all Intel chips and work as expected.  

If I were buying a new notebook and wanted it to support Ubuntu, I'd probably  consider H-P, Lenovo Thinkpad or possibly Dell's "business line" The "business lines" tend to be more expensive but use higher quality more durable more mainstream components.  The "how cheap can we make this?" lines could be more finicky.  My experience and worth what you paid for it.:)
