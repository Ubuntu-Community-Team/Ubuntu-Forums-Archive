### Post by TheFu on 2017-01-11
Most ToS for social sites don't allow this type of scanning. You'll need hundreds-thousands of 

You'll need to look on github for tools like "[creepy]("https://github.com/ilektrojohn/creepy")" - would also suggest you follow some security-centric forums and check out youtube for how-tos. Many security conferences record their presentations about this stuff.  Any there are always well-tuned google searches, which can be extremely scary on what they reveal.

I've seen about 20 of these sorts of tools.  Most often, the newer tools are becoming subscription-based web services. They don't give away their code.
