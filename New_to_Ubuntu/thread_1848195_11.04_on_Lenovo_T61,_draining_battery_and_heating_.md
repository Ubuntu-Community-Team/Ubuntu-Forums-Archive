### Post by varunendra on 2011-09-22
As a starting point, you may use top command in terminal to see if there is some application using too much cpu power. For exapmle, compiz has been reported sometimes to be continuously using up 3 to 8% cpu even during idle modes.

Although there have also been some similar issues reported with newer kernels, so it may also be a bug if not any application.
