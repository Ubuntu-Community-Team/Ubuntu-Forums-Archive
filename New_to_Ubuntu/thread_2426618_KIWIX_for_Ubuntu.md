### Post by Frogs Hair on 2019-09-11
For the reader, you need to download .zim files then you can open the content in the viewer. Select the **All Files** text in the viewer and you will see a download option for various content which can be used offline later. To view content immediately select open after downloaded and for later use just select open from the list .It's possible to create a local server over a network. The downloaded content is stored in .local/share/kiwix


[https://wiki.kiwix.org/wiki/Content_in_all_languages](https://wiki.kiwix.org/wiki/Content_in_all_languages)
