### Post by ianhaycox on 2008-09-21
It uses the one you assign but avoids the need to setup port forwarding manually on your router.

In my case if I go to the port forwarding page on my router at [http://192.168.1.1](http://192.168.1.1) it shows the port forwarded that matches the number specified in transmission.

Note you need to have a upnp compatible and enabled router
