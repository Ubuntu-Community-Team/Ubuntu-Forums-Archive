The wubi is not perfect. I think of it more as a way to test drive Ubuntu to see if you want to do a full install. Problem is it sounds like a file in your ubuntu directory got corrupted. It is likely you will have to run the wubi to uninstall and reinstall, however, if you have the HDD space you can make a backup of all the files in the ubuntu directory before doing any updates and if it gets broken in the future just copy the files back and avoid the update that killed it.
