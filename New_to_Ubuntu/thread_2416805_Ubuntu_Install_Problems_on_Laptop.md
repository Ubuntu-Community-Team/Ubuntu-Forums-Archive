### Post by oldrocker99 on 2019-04-15
Download the .iso image from [https://www.ubuntu.com/#download](https://www.ubuntu.com/#download), then burn the image to a USB thumb drive, using Etcher, available on all platforms. Boot from it, and installation should go more smoothly.
