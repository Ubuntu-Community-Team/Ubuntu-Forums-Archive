### Post by TzaB on 2008-07-08
Go to /home/usr and click to view the hidden files. (Ctrl+h)
Find a folder with the name ".fonts" and paste the .ttf font that you want and do a reboot.
I think that your font will be ready for use :)

Note: If you cant find the folder .fonts, then you can create one.
