### Post by AndThenWhat on 2009-05-14
A really easy tool for XML development would be to simply use Gedit (Applications -> Accessories -> Text Editor) and go under (View -> Highlight Mode -> Markup -> XML) or HTML etc.

And you're good to go. I use it all the time for Java and HTML/PHP programming. Simple but effective.
