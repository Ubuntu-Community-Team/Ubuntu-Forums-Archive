### Post by sviola on 2011-02-21
You can change the font size by following these steps:
[LIST=1]
[*]Click on the wrench icon.
[*]Click "Preferences."
[*]Click "Under the Hood."
[*]Scroll down to the "Web Content" section and click the button labeled "Change font and language settings."
[*]Under "Fonts and Encodings," select a font and change the size there.
[/LIST]

That's it!
