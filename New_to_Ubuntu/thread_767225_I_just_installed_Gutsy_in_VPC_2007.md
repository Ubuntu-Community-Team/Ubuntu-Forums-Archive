### Post by 123456789123456789123456 on 2009-03-29
you said you installed it in Virtual PC.  Virtual PC, emulates very simple hardware.  If you are using an operating system other than windows, the hardware emulated is very standard.  This can cause several problems.  Ubuntu probably does not reconize the sound card VPC is emulating.  As far as screen resolution is concerned.  If VPC is emulating a standard VGA graphics card, Ubuntu will probably use a standard driver, which limits the screen resolution to 800x640 for some where around there.  To make sure, under system, display resolution, you get a window that allows you to change the resolution.

The best emulator program to run ubuntu under is VMWare.
