### Post by Dondermans on 2011-06-05
There are two versions of 'Ubuntu 10' (10.04 and 10.10), it might be worthwhile specifying which of the two you would like to use. I suppose you do not have Ubuntu installed at the moment, or you would not be asking this question. Do correct me if I am mistaken.

I don't know for a fact whether the Visioneer 8100 will work with either operating system, but a way to test this yourself would be to create a liveCD ([_Instructions_]("https://help.ubuntu.com/community/LiveCD")). Once you have done that you can boot from the CD and connect the scanner. I use a HP LaserJet Pro M1536dnf Multifunction Printer myself ([Specifications]("http://h10010.www1.hp.com/wwpc/sg/en/ho/WF06b/18972-18972-3328064-3328083-3328083-3974271-3974905.html")), which works like a charm.
