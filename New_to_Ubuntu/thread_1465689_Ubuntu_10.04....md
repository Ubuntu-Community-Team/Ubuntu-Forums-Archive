### Post by arrrghhh on 2010-04-29
> **joenewtzie said:**
> OH and I want to make sure I get update for laptops!

Whatever version you install, will continue to update based on that version.

Please do a ```
gksu update-manager -d
``` and click the "Distribution Upgrade" button in the top right.

See this guide for more info:
[http://www.ubuntu.com/getubuntu/upgrading](http://www.ubuntu.com/getubuntu/upgrading)

** Note - that guide has the "recommended" method.  Perhaps preferable over my command, but both achieve the same goal :D **
