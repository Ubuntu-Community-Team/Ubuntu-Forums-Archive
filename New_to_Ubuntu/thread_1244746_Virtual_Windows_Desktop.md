### Post by Xero Xenith on 2009-08-19
VirtualBox has experimental support for DirectX 9 and is probably your best bet at the moment, if you don't want to dual-boot. It lets you run Windows in an application - the Windows installation will use a virtual hard drive that is, in fact, contained within a file - you don't need a partition. You can also set it up so it can see folders on Ubuntu as network drives.

Get it here:
[http://www.virtualbox.org/](http://www.virtualbox.org/)

Note: you need an installable copy of Windows - preferably a genuine one. Morals FTW :)
