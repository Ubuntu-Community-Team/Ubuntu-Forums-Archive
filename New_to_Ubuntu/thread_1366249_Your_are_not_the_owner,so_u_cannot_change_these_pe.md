### Post by drs305 on 2009-12-28
You can open nautilus with admin rights by launching it with the command:
```

gksu nautilus

```

You will be running it with 'sudo' powers so you can change, delete, rename, move *any* file. Be careful if you are changing system files as this power will also allow you to mess up your system if you change something you shouldn't.
