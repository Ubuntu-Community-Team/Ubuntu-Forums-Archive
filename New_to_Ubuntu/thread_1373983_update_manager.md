### Post by mcduck on 2010-01-06
Since you just installed it's absolutely normal that you get a lot of updates. There has been quite some updates since 9.10 was released.

Once you get your system up-to-date the available updates will be lot smaller and not too frequent.

You can (read: should) safely install all the updates, they are not going to change how things work, they are just bug fixes and security updates. Ubuntu doesn't provide actual new versions for any software after release (and the Update Manager doesn't add any new software packages, only updates things you already have installed).
