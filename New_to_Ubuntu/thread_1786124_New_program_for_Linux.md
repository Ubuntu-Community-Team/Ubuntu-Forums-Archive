### Post by amauk on 2011-06-19
subscribe to the ubuntu-devel mailing list
[https://lists.ubuntu.com/mailman/listinfo/ubuntu-devel](https://lists.ubuntu.com/mailman/listinfo/ubuntu-devel)

Post to the list detailing exactly what your program does, and someone should let you know whether it would be suitable for packaging in Ubuntu (or upstream Debian)

Also if you're using any sort of source code management or revision control, let the developers know where they can pull the latest development branch

If you're not using any source code management or revision control, you may also want to register with launchpad for these services

Good luck
