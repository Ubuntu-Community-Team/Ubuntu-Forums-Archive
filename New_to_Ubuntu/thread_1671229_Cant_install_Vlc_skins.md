### Post by godspeedmav on 2011-01-20
If you open **VLC** > **Tools** tab > **Preferences** > **Interface** category > Check **Use Custom Skin** > Browse to the folder that you downloaded skin file to > then **Save** it. If it doesn't work try out the following;

[LIST=1]
[*]First download the skin file you want to change to.
[*]Open your **Home Folder**.
[*]Press **Ctrl+H** or **View** tab > **Show Hidden Files**
[*]Search for a file called **.local** (mind the . infront)
[*]Open that folder and click on **Share** > **VLC**
[*]I believe there is no folder called skins2, so create that folder by right clicking on the folder window > **Creat New Folder **and name it to **skins2**.
[*]Now drag the downloaded skin files to that folder called **skins2** folder.
[*]To change the skin of VLC, open **VLC** > **Tools** tab > **Preferences** > **Interface** category > Check **Use Custom Skin** > Browse to the folder that you copied the downloaded file then save it.
[/LIST]
