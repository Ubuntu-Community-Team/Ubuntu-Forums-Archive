### Post by lovinglinux on 2012-04-27
How did you install Firefox 12? You don't need to download it somewhwere else. It is already [available in the repos](http://packages.ubuntu.com/search?suite=all&section=all&arch=any&searchon=names&keywords=firefox), so all you need is an update and install the proper [firefox-locale](http://packages.ubuntu.com/search?suite=all&section=all&arch=any&searchon=names&keywords=firefox-locale) package. You can do that using the Software Center.

You might also need to use an extension like [Quick Locale Switcher](https://addons.mozilla.org/en-US/firefox/addon/1333/).

If you are looking for a translation extension, then see [https://addons.mozilla.org/en-US/firefox/search/?q=translate&cat=all](https://addons.mozilla.org/en-US/firefox/search/?q=translate&cat=all)

If you want to use beta, aurora or nightly versions of Firefox see [http://ubuntuforums.org/showthread.php?t=1712247](http://ubuntuforums.org/showthread.php?t=1712247)
