### Post by Zill on 2012-09-29
mark allyn:  Although I wish you well in your quest to find a "bundled" version of GTK, I think this may be difficult as there are so many dependencies, many of which vary with all the different versions of the installed OS.

As Ubuntu, along with all other Linux systems, requires regular updating to remain secure, it is designed to run, primarily, with internet access.

I suggest you should get your machine connected to the internet, either via an ethernet cable or wireless etc.  You can then install and update as required.  If desired (eg for security reasons) you can then disconnect the machine from the internet again.
