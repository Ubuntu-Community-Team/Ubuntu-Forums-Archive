### Post by SPazzo on 2009-11-16
Try checking the Md5sum.  Instructions on how to do that are [here]("https://help.ubuntu.com/community/HowToMD5SUM").  The MD5sum for that is:
```
0b4fa10a6f4995b3e2f729454c1462b6
```If it doesn't match, then it is corrupted and needs to be re-downloaded.

(From [http://www.ubuntupocketguide.com/download_main.html](http://www.ubuntupocketguide.com/download_main.html)
