Dual booting is where you have two operating systems running on one physical machine, you can not run both at the same time, where as with a virtual machine, you can run one operating system within another, e.g. Linux inside Linux, XP inside Linux, Linux in XP...lots of combinations.

### Post by Paqman on 2008-10-26

The important difference is that running two operating systems at once is very resource-intensive. Both systems will run very slowly unless you have a very powerful machine.

With a dual-boot whichever system you are on runs at full speed, but to switch to the other system you need to reboot.
