### Post by howefield on 2009-03-01
Open up compiz settings manager and check Animations Add-On, then check  Animations, and open it, select the open tab, highlight the top row in the animation selection which is probably Glide 1 if left it at defualt, and press the edit button.

Select the burn option from the drop down box, and then close.

Repeat for any further tabs that you want, eg Close Animation.
