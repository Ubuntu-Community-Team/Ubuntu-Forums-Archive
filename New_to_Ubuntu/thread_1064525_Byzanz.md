### Post by oldos2er on 2009-02-08
Open up Synaptic again, find the package byzanz, it should have a green box in front of it, meaning it's installed. Right-click on the package, click Properties, Installed Files. Look for an entry in /usr/bin, whatever its name is, try typing it into a terminal.

### Post by Vadi on 2009-02-09
Byzanz is a gnome panel applet. To add it, right-click on the top or bottom panel, select Add to panel, find byzanz and add it.

Or if you really insist on using the terminal, "byzanz-record" is the command you're looking for.
