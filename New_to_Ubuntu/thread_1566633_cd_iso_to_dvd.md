### Post by diablo75 on 2010-09-02
Don't think you'll have any trouble using any burning software to put a CD-bound ISO onto a DVD.  I've done it before.  Just right-click on an ISO file and hit "Open with Brasero" or whatever burning application you already have installed.

If you don't have any installed, I'd recommend GnomeBaker or k3b.
