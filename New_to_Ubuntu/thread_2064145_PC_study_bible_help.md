### Post by newb85 on 2012-09-28
I agree with both responses.  More detail is needed.

In general, if you can find a program native to Linux that meets your needs, you're better off with that.  WINE can be used in some cases to run Windows programs, but it's kind of hit or miss.

Personally, I recommend Xiphos.  It's built on the Sword project, so there is a vast selection of modules to install.  The downside is that virtually all the modules are older non-copyrighted works.  For example, you'll find KJV, ASV, and ESV, but not NIV, NASB, or NLT.
