### Post by ByteJuggler on 2008-09-04
Ubuntu's liveCD runs reasonably quickly provided you've got enough RAM.  When you've booted from the CD, you can either use the software from the liveCD or start the installation when you're ready.

### Post by ByteJuggler on 2008-09-04
> **32waystodie said:**
> 
-Once i've burnt the file to the cd and put it in am i just using the version ive downloaded or am i installing? so is it best to back up all my data and all of that as if your downloading.

Just to be clear - you need to burn the .ISO file onto the disc as an image, then you need to boot from it.  You must not just burn the file as a data file onto the CD, then it will not be bootable.  You should in any case backup all your data.
