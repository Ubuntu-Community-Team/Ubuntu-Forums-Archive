### Post by Lars Noodén on 2015-02-22
I would instead look at the Tor Browser Bundle (TBB) which provides everything pre-configured.  You can get it from the Tor project's web site and can install it in any directory you like.  It will not be taken care of by the package manager but will come with its own notifications when it is time for an update.

### Post by Lars Noodén on 2015-02-22
TBB provides about the only safe configuration.  It is Tor + Firefox.  IF you set up Tor and Firefox on your own, you will either end up with the same settings or have an unsafe configuration.  Anyway, if you allow Flash, you are as good as cancelling the benefits of Tor.  Its security is as bad as MS.  TBB will play webm.  TBB will play Ogg.  MPEG can be downloaded and played while the net is disconnected.
