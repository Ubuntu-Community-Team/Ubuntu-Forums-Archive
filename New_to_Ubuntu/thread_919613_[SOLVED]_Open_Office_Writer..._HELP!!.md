### Post by bumanie on 2008-09-14
Go to insert --> fields --> page count. I think page number will number a single page, whereas page count numbers successive pages. Also there is a wealth of help explanations in OO's "help" section. Also lots on the internet.
PS: Header/footer is correct. see below.

---

### Post by Mornedhel on 2008-09-14
The trick is to insert a header or footer first. This will be displayed on every page. Then use Insert > Field > Page Number into the header/footer to display the current page number on every page. Page Count displays the total page count IIRC.
