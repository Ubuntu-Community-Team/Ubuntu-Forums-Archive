### Post by cariboo on 2010-12-31
You change what program opens pdfs, by right-clicking on a pdf and selecting **Open with other application**, when the **Open with** windows pops up go to the very bottom and select **Use a custom command**, browse to where you have the foxit reader program located and select it.
