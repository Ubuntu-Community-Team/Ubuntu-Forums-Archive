### Post by ranch hand on 2010-05-17
I am not real clear on what window you are referring to, /usr/share/backgrounds or the System>Preferences>Background window.

Assuming it to be the System>Preferences>Background window you need to click on "Add" and navigate to the files you want added.

The images do not need to be in the /usr/share/backgrounds file to use them.  The advantage to them being there is that you are less likely to be changing that file a lot.  If you use a different file in your /home/<your user name> directory you do not need to be root to work with them.

I usually have a "wallpaper" folder there and use the images from there.

The remove option is nice in the System>Preferences>Background window as it just removes the from that and not from the directory in which it is stored.  That means you do not need a bunch of images you are not using in there at all but can put them back when you get the whim.
