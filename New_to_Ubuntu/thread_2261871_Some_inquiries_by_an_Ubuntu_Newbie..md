That's right. I will say that KDE, while incredibly full-featured, is resource-hungry. MATE (mate-desktop, my favorite) is much lighter-weight than KDE, and so are LXDE and XCFE. I like MATE because it has the best features (for me) than the other two, including a much better file manager.
