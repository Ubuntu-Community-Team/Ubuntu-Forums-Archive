### Post by nicedude on 2008-05-17
vi is a text editor vim is vi iMproved text editor. You probably will want to use Gedit most of the time as it has many more options compared to vi or vim. you will find in linux that there is always "more than one way to skin a cat" as there are many programs that do the same thing, since they are all free you can install and try things out until you find one for any given purpose that you really like. So when someone says use vim you can just as easily replace with gedit if you prefer. 

The one time vim comes in handy is it will work when you don't have graphics going and are just at a command line login screen to allow you to edit a text file that might fix a bad configuration situation, Like if your xorg.conf file is messed up.

Hope this clears this up for you

I have never used it but, you can start a vim tutorial by typing "vimtutor" on the commandline.  You may also want to install the gnome version of vim which has menus and such that can help you learn various commands:
