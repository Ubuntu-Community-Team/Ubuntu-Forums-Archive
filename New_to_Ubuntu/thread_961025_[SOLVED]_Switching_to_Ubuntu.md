### Post by NewJack on 2008-10-27
Well it's good to hear you have an interest in Ubuntu.  Let's take your questions one at a time.

1- Read these regarding Linux Viruses: [http://en.wikipedia.org/wiki/List_of_Linux_computer_viruses](http://en.wikipedia.org/wiki/List_of_Linux_computer_viruses) and [http://librenix.com/?inode=21](http://librenix.com/?inode=21) .  Basically, there are less viruses for Linux than Mac OS (And they are proof of concept and not out in the "Wild").  Unless you give root access to an unknown program, you should be fine.

2- There is Frostwire, which is built on the Limewire core:  [http://www.frostwire.com/](http://www.frostwire.com/)

3- There is WINE for Windows games/applications.  I haven't used it, but speed shouldn't be a problem.  Here is the homepage for WINE: [http://www.winehq.org](http://www.winehq.org) .  Also, check out the Gaming and Leisure section of the forums.  You'll get lots of help there.

4- It should be able to handle most everything.  If you are planning on having Compiz running and doing alot of multitasking, I would try to bump it up to at least 1Gig of RAM

5- Again, I haven't used any programs like that.  I have heard that Cinelerra might do the trick:  [http://cinelerra.org/](http://cinelerra.org/)

6- Ubuntu is a GNOME based distro which tends to be more stable than KDE, which Kubuntu based on.  KDE based distros tend to have a nicer GUI and lots of customization features.  You can run KDE based apps in Ubuntu, even though it is GNOME based.  Xubuntu is based on XCFE environment.  It tends to run well on older systems since it is a lighter desktop environment.

For your Mac, I might try Xubuntu first.

Hope that helped.

### Post by SteveLefty on 2008-10-30
To answer your second set of questions:

1. Do a Google search for that particular USB model name + Ubuntu.  That's normally a pretty good way to tell if other people have been able to get them to work.  Make sure the results you get mention which version they're using if they say it doesn't work (as in: if they were using a version of Ubuntu from 2005, the problem may have been fixed in a more recent version).

2. You can absolutely install Ubuntu on a blank PC.  Just start up off the Live CD that you've burned (I assume) and select to install.  It's extremely easy :)

### Post by SteveLefty on 2008-10-30
A 2.8 GHz processor is enough to do what you want to do.  2 GB RAM is also a good number.  Keep in mind that to encode / decode video and music you'll need as fast a processor as possible.  So, while 2.8 is definitely enough, you should see an improvement with each increase to your clock speed.  How much, I'm not sure, and I don't know where the line is in terms of price vs. speed increase.  You will most likely want a dual-core (or more!) processor because the video and music editing will take advantage of this.
