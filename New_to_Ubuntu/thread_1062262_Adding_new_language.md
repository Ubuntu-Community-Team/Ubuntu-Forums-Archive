### Post by wawacito on 2009-02-06
i don't know about arabic - but i added another language this way

system -> administration -> language support

check off the language of choice
check off "add support to enter complex characters"  (do you need this for arabic?)

once thats done you log out and log back in - there should be an icon of a keyboard visible next to where the day and time shows up (sorry im new too and i dont know what the name of that thingie is - is it a panel?) if you right click on there - it should say SCIM setup - thats where you setup the keyboard shortcuts to switch between languages.

hope that helps
