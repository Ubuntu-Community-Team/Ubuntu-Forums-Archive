### Post by houseworkshy on 2012-08-06
I'm not sure which operating system you are on or want to use.  If on ubuntu linux you could look in synaptic for rosegarden ( which has a website [http://www.rosegardenmusic.com/](http://www.rosegardenmusic.com/) ) and Ubuntustudio-audio, which may have a website which I can't find at the moment.  They are both free open source sound program collections, so you could just install them and see which programs you like.  Searching online for the bits within them which you'd want on the new machine would give the system spec's which you'd require.  
If you're not on linux you could try one of the live distros and do much the same.  This is a nice one [http://www.artistx.org/site3/](http://www.artistx.org/site3/)  It will run from the dvd, just put it in the drive and reboot, when finished just shut it down, remove the disc when it asks you to, press enter and your machine will boot up from whatever is installed on the hd unchanged.   The distro will run slower than it would if a hd install but well enough for you to get the idea of what you like.  Just found this on the aristx site [http://www.artistx.org/site3/Hardware-support/audio-hardware.html](http://www.artistx.org/site3/Hardware-support/audio-hardware.html)  might be helpful.
I'm a little out of date on which distros are best for media as I tend to look at the jam packed ones as software cataloges, prefering to install a fairly plain all-round system and then bung in what I like.
This is a good site [www.distrowatch.com]("http://www.distrowatch.com").  Searching with parameters "live media" and "multimedia" brings up a lot of results.  Best go only for those which are still supported.
+1 to ubuntu studio too.  Just had another look and it is now also a live disk ( so it can be tested running from the dvd drive ) It didn't use to be ... very nice.   I had an early version of this installed once,  7 or 8  something I think,  and it was good then.  May have to download that myself now.
So play with the software, research the bits you like, build to fit.
