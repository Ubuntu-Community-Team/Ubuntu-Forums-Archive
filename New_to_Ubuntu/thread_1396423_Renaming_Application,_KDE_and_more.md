### Post by KIAaze on 2010-02-04
Gnome and KDE are open-source, so the theming possibilities are pretty much unlimited.
It will just be more or less easy to do.

KDE usually offers more options.
You can add/remove/move window buttons for instance: System settings -> Appearance -> Windows -> Buttons.

I don't theme my desktop a lot, so I can't give you much help there, but I'm pretty sure that the same effects as Windows blinds can be achieved with GTK+ or KDE.
Maybe there's just a lack of artists. ^^

### Post by KIAaze on 2010-02-04
[http://www.panix.com/~elflord/unix/grep.html](http://www.panix.com/~elflord/unix/grep.html)
grep is basically a command-line search tool.
There's probably a GUI to search text in files recursively, but I don't know it. :P

Actually, there's also "ack", which is better than grep: [http://betterthangrep.com/](http://betterthangrep.com/)
I just haven't gotten used to using it instead yet.

To get source code:
-Download source package from official website
-Or use "apt-get source PACKAGE": [http://www.debian.org/doc/manuals/apt-howto/ch-sourcehandling.en.html](http://www.debian.org/doc/manuals/apt-howto/ch-sourcehandling.en.html)
