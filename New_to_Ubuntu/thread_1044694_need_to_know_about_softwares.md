### Post by Kellemora on 2009-01-19
Hi Jaf

In a nutshell, for every program available for windows, at great expense.  In Linux there are usually 4 or 5 sometimes more nearly identical programs, usually with more and better features.

And the nice part, 99% are FREE!

In Synaptic package manager, there are something like over 25,000 software programs you can install at the click of button!  There is a description of each one.  But, it's usually easier to just simply ask which Ubuntu program is LIKE this or that Winders program.  You'll often get several options to try out to see which one you like the best!

### Post by Kellemora on 2009-01-20
Hi PDT

It IS better, BUT and that's a BIG BUT, hi hi..........

In order to USE Ubuntu, you NEED to Keep a DOZE machine close by, for all the things Linux is not yet up to snuff about!

Not their fault, it's more the hardware vendors fault....
BUT even then, those 3rd party guru's who write drivers and the like for Linux are out to lunch all too often and forget MAJOR things needed in those drivers.

For example:  The printer driver for my color laser is BETTER than the OEM Doze print driver, more versatile and many more features.  However, those who wrote the driver must have assumed toner cartridges never run out.  The provided NO MEANS of replacing a spent toner cartridge.  The ONLY way you can replace a toner cartridge on this particular printer is to disconnect it from LINUX and connect it to a Windows Machine, wait for the driver to self-install, then select, change toner cartridge!

I have found very little support for the two primary human interfaces to a computer, the keyboard and mouse.  At best we have only the bare minimum to make the basic keyboard and basic mouse work.  Even the controller programs for the mouse is blatantly missing certain much needed commands.  Like support for the button under the scroll wheel.  It's default is Paste!  With no other options.  And no support for 120 key POS keyboards.

So, front office must continue to use Doze machines!
Of all the great programs for Ubuntu and Linux in general, the one and only useful accounting program is still one you must purchase.

The other thing is no scanner support for the top selling scanners!  That has nothing to do with Ubuntu and everything to do with the manufacturers tugging on Gates shirt tails!

Nonetheless, after only 1 month into Ubuntu, I converted my entire office over to Ubuntu, except for the couple of must have Doze machines.

Our production levels are much higher on Ubuntu than they ever were on the DOZE, even with all the headaches a noob suffers while learning this stuff.
