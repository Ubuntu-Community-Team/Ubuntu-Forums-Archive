### Post by NESFreak on 2009-04-04
> **kiridude said:**
> I have never heard of installing Ubuntu inside Windows. Did you boot from a cd to install Ubuntu?

this is because you installed ubuntu IN windows. You can remove it trough de windows add/remove thing. Follow the instructions. For a 'normal' ubuntu installation you have to boot your pc from the cd.

After uninstalling put the cd in your cd drive and reboot. When the bios boot screen appears there must be a key to press to select an alternative boot device. (the screen tells you which, probably F9) Start the live cd environment and follow the instructions.
