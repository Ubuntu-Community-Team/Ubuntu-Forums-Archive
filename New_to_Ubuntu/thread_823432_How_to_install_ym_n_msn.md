### Post by Fedz on 2008-06-09
I could be wrong but, I fear the OP mistakenly thought they could download and install the original MSN and YIM (.exe).

If you want to run .exe like in Window$ you can put [Wine](https://help.ubuntu.com/community/Wine) on in Ubuntu very easily from System > Administration > Synaptic Package Manager.

This will create a virtual C:\ drive allowing you to run Window$ programmes from within Ubuntu.

I doubt MSN will work but, YIM, Miranda, Trillian ...etc however will :-)
