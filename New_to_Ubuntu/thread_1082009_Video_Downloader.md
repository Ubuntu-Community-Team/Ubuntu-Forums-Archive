### Post by ajgreeny on 2009-02-27
You can also get the flash videos from your /tmp folder which I do with a shell script file.   After downloading the file (either watching it or not)  I run the script file and copy the flash file to another folder with this script:-
```
#!/bin/bash
cp /tmp/Fla* ~/Movies/
```
One click and it's done.  Not all flash files will play, however, as some need an oxo codec (or something similar), which I have not found anywhere.
