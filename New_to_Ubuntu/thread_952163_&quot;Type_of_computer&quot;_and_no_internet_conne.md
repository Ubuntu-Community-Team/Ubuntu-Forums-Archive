### Post by stephanvaningen on 2008-10-18
You can see in your computer's documentation what type of computer you have. You can also go to My Computer, right-click and click Properties: there somewhere you get some hints about the processor and if it is a 32-bit or 64-bit (since it mentions it often in the processor type when it's a 64-bit).

I would suggest you use the 32-bit version of Ubuntu anyway, since you probably want to use plugins in Firefox (like Flash), which often are not available for 64-bit... to give one example I can think of... So if you have a classic PC bought in a classic shop, just download the i86-32-bit version of Ubuntu and: enjoy
