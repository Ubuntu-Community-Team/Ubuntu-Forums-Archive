### Post by Cypher on 2008-05-06
In a true dual-boot scenario you have 2 OS' running separately so you have to reboot to go from one to another, there's no way around that.

However, you have the option of running Windows inside virtual machines like VirtualBox/VMWare inside Linux. Or you can run Linux inside Virtual PC/VMWare on WinXP..
