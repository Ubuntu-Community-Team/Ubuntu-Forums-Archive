### Post by nmcgrew on 2010-03-02
Is there a way to make my programs close by clicking the X as opposed to having to use the File and then Quit menu option?  Whenever I click the X in the navigation section [upper right corner] to close a program [out of intuitive habit] it just does that.  Closes it.  But it doesn't quit.  Like music players still continue playing, etc.  Really defeats the purpose of having the X.  Surely there must be a way to do this?

### Post by marshmallow1304 on 2010-03-02
Some applications minimize to tray instead of closing (Amarok does, maybe a couple other music players).  Check the settings for each program that does it; there may be a setting to change the behavior.

### Post by doas777 on 2010-03-02
only a handful of applications should display the behavior you describe. in my case, only music players and few other minor apps. in most cases you can configure that on an app by app basis, as marshmallow pointed out.
