You must have downloaded a corrupt Ubuntu LiveCD ISO file. You have to check the ISO file's MD5 hash first then compare it with the MD5 hash which comes with the LiveCD or from the website you downloaded it. Try to download this [windoze MD5 utility]("http://www.brandonstaggs.com/filecheckmd5/") and check it yourself.
If you got the correct MD5 hash, try to burn it at a lower speed (4x). Hope that will work for you.
