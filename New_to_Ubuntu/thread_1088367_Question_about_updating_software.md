### Post by lykwydchykyn on 2009-03-05
You should get a notification where there are updates.  If you want to check manually, you can load up Synaptic and do a refresh, then it should list upgradeable packages.

As long as you install software through the package manager (Synaptic/Adept/aptitude/apt-get) system, it will be kept up to date.
