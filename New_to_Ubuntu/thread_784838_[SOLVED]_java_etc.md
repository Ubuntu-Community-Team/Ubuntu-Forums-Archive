### Post by dyous87 on 2008-05-06
First go to:
[I]Applications>Add/Remove...

[/I]It may tell you that your list of available applications is out of date. Just click *reload. *On the top where it says *show, *click on the drop down menu and select [I]All available applications.

[/I]In the search box, type in: [I]ubuntu restricted.

[/I]When it shows up in the list, place a check next to *Ubuntu restricted extras (Not **K**ubuntu)*[I]!

[/I]Next type in *Flash *in the seach box. In the results, place a check next to [I]Macromedia Flash Plugin. (This will enable flash to play in firefox.)

[/I]Next type in *Java *and check off *Sun Java 6 Runtime *if it is not already checked.

Next you may want to install some video players for playing different formats and embedded videos.

In the search box, type in [I]movie player.

[/I]In the results make sure you have checks before *Movie Player, Mplayer Movie Player* and *VLC Player*. If not just check them off. 
You may also like to install Microsoft Core Fonts so that web pages using these fonts render correctly.
Type in *fonts *into the search box. When the results come up check off [I]Microsoft Core Fonts.

[/I]After all these changes are checked off simply click *Apply Changes *on the bottom to download and install them. From here on everything is taken care for you. It may require you however to accept a license agreement here and there for some proprietary stuff you are installing such as java or flash. Since we are downloading a lot of files, this process may take a long time depending on your transfer speed. You can however continue to browse the web while it is going.

Hope this helps, let me know if you have any questions.
