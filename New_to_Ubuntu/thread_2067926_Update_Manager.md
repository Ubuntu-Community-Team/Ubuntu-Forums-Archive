### Post by newb85 on 2012-10-08
If you're referring to the fact that you have to click "Install Updates", then yes, that's normal.  It used to be possible to set it up to do all that automatically, but that option has been removed from the GUI.  (It may still be possible by manually editing config files, but that's beyond my experience.)
