### Post by mbzn on 2010-05-16
The iso is a image of the cd, use the option "burn image to disk". That should get you booting if the cd is the first boot option in the bios. Also don't burn the cd too fast 'cuz faster=more errors, and due to the compression rate of the ubuntu cd every error is major. 

Hope this gets you going...
