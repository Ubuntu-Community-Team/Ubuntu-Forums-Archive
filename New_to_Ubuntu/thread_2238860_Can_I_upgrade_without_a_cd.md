### Post by nerdtron on 2014-08-10
Although it is possible to do a system distro upgrade, you'll be lucky if you have no issues after upgrading. It would be better to create a backup of your important files and just reinstall the latest ubuntu. You can use a USB flash drive as an install media which acts just like a cd.

If your current distribution is 14.04 already, just open the update manager, check for updates, apply all updates, then reboot. You should have 14.04.1 by now.
