### Post by lhb1142 on 2008-10-04
Somehow I managed to make OpenOffice.org the "default" application for opening PDF files. Of course, it cannot do so properly and the PDF files thus opened are unreadable.

I can open each PDF file individually by right-clicking and specifying that it should open with Document Viewer. Of course I have to do this each and every time.

What I need to know is how to make Document Viewer the default application for opening all PDF files as it was until this evening.

Any help will be much appreciated.

---

### Post by Shippou on 2008-10-04
Right click on a PDF file, then choose Open With Other Application (or something similar to that). Then choose the application you want the file to open by default, i.e. Document Viewer. The file will open and you can close it. 

There you have it. Everytime you click a PDF file, the Document Viewer will open it for you.
