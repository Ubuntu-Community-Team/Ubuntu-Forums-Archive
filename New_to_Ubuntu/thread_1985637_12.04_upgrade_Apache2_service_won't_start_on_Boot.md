### Post by kilowhisky on 2012-07-07
Alright after months of not really caring about the problem i finally figured out what it was. For some reason i was getting an SSL Cert error when i shouldn't have been. 

```
./boot.log:SSLCertificateFile: file '/home/<me>/.myCA/server_crt.pem' does not exist or is empty
```

So my solution was to move the cert to a location outside the home folder. Then it started working. Don't know why but it starts up on its own now.
