### Post by qamelian on 2011-02-13
> **Its_Rishi said:**
> I need a package with debian".deb" not as "tar.gz"
Most themes are packaged as tar.gz. If you are using the Gnome Appearance properties box, that's the format it more or less expects them in. You just drag the tar.gz file to the Appearance Properties box and drop it there. If the theme is correctly packaged, it will install.

### Post by Vaphell on 2011-02-13
1. gnome-look.org is the #1 place to go
2. you don't want themes in deb format from places where anybody can upload stuff. Debs require root to install so they can be used to plant anything in any place on your box. There was a case of malicious .deb on gnome-look.org not that long ago (iirc). Using built-in theme manager with tar.gz files as qamelian said is the way to go, debs should come from trusted sources only.
