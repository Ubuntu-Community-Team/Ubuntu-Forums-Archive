### Post by t0p on 2008-08-27
Since nobody else has told you this, I guess I better.

WEP is *insecure*!  Even if you have a real long key, it's possible to crack WEP in 5 minutes.  Go to youtube and search for "cracking wep" - you'll find a lot of videos showing skiddies doing just that.

WPA is much more secure (so long as you set a good strong pass).  So dump the WEP and score the WPA.
