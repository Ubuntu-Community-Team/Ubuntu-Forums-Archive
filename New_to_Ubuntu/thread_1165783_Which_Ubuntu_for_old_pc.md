### Post by XubuRoxMySox on 2009-05-21
Part of what makes a *buntu "light" or "heavy" is its *desktop environment*. Xubuntu uses a lightweight one which is why it runs better on older hardware than Kubuntu (the heaviest of heavyweight desktop environments, lotsa bells and whistles!).

By far the lightest and speediest of the ultralight desktop environments is [LXDE]("http://lxde.org"). Team Ubuntu is currently working on an LXDE-based *buntu (look for "Lubuntu" soon) which promises to be even speedier than Xubuntu on older hardware.

On my ancient machine I installed the Ubuntu-based [Crunchbang Linux]("http://crunchbanglinux.org"), which is "naked" Ubuntu (no desktop environment *at all* - only a lightweight window manager), and added the ultra-light LXDE environment. The two play nice together as long as I let PCMan (LXDE's built-in file manager) manage the desktop.

I didn't want to wait for "Lubuntu," LOL. So I "made my own Lubuntu" by customizing Crunchbang with LXDE.

-Robin

### Post by fluxlizard on 2009-05-21
try pclinuxos gnome edition
I have an old laptop 700 mhz 256 mb ram 4mb video card and it runs quite quickly on it. In fact, sad to say, but boot time, firefox, open office and the like start up only very slightly slower than they do on my ubuntu desktop on my main pc which I built last fall (3.0 ghz dual core 4gb ram, 756 mb video card). Which is why I recommend it on your machine. 

you get a full gnome desktop like ubuntu. It's not a compromise desktop like xubuntu (weird font size and lacks features of gnome). It's not a hacked desktop like puppy linux or somesuch. It's a full featured gnome desktop that will run nicely on your hardware

you get synaptic like ubuntu with a selection of thousands of packages, which maybe isn't as broad as ubuntu, but it's got all the same packages you are likely to use, and some that ubuntu doesn't include are included (visa versa is true as well though).

It runs quite nimbly even on old hardware, unlike ubuntu.

Or if you really like ubuntu get more memory used from your local mom and pop computer shop for cheap. If you can get 1gb ram in there, it will probably run ubuntu nicely- I have a 1000mhz machine that does, but only after I upped the ram to 1gb.
