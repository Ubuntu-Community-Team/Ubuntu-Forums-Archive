### Post by pedro3005 on 2009-09-02
LXDE consumes considerably less resources than GNOME or KDE, but you should only use it if your resources are too low for the above (please post PC specs). To get it, just run:
"sudo apt-get install lxde"
Log out, and choose LXDE at Change Session (or something like that).
