### Post by Het Irv on 2008-06-24
Unless it somehow affected the computer (short, surge, or something like that), it should be fine.  I don't think that is very likly though.  I would replace the CMOS battery and then boot the computer, set everything up if you need to and then unplug the computer from the wall (after a safe shutdown!).  If you can boot the computer up just fine you should be okay.  If not, it might be a more serious problem.
