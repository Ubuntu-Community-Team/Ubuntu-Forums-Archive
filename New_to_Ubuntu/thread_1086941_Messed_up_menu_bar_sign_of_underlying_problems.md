### Post by Toxicity999 on 2009-03-07
If you right click the desktop and click 'change background' it will open the customization tool, from there go to the visual effects tab, and select none. Doing this does *not* disable 3D, simply compiz, the effects renderer we use to make things pretty. no more real shadows, and otherwise fancy effects. but it should remedy that.
