### Post by yetiman64 on 2012-02-24
> **BlinkinCat said:**
> You normally don't see your password as it is entered - :)
^^^ This. Just type in the password as per normal. It is working, but a security feature of the terminal is to NOT give visual feedback for passwords.

It is possible to edit a system file to have asterisks show up, _*it is not recommended though*_ as any fault while altering/saving the file can leave you with a crippled admin account.
