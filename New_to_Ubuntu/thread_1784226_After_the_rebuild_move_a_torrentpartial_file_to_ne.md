### Post by CaptainMark on 2011-06-16
as long as you copy the database files over to the new install then you shoul be fine, for example if you use transmission then copy the folder /home/[you]/.config/transmission from the old hdd to the new, as long as all the files and part files are in the same path locations as before it will just continue as though nothing happened
