### Post by jeremy31 on 2018-06-06

---

### Post by #&amp;thj^% on 2018-06-06
I Agree with jeremy31
And you can do this via terminal command also with:
```
gsettings set org.gnome.settings-daemon.peripherals.touchpad natural-scroll false
```
