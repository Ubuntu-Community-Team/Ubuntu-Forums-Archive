### Post by Revolutionary101 on 2012-05-21

It will not update automatically and you can't add a new PPA to get it. You will most likely have to compile it from source code (takes a few minutes). Plus if you compile it you can customize it if you want, kinda technical but interesting at the same time. Here are some instructions:

[http://www.wikihow.com/Compile-the-Linux-Kernel](http://www.wikihow.com/Compile-the-Linux-Kernel)

I have done it a few times and I received no problems. Enjoy!

Tip: I wouldn't go crazy on customizing it because if you change some things it can mess up your install. Compile with caution.

### Post by qamelian on 2012-05-21
If you really need the newer kernel for some reason, there is no need to compile it yourself.  You can find them in the mainline kernel archive. Documentation  can be found at: [https://wiki.ubuntu.com/Kernel/MainlineBuilds](https://wiki.ubuntu.com/Kernel/MainlineBuilds)
