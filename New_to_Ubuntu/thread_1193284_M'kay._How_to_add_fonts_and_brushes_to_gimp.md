### Post by ubuntu27 on 2009-06-21
> **Xjjulix said:**
> o.o I click the link and I get "blank page":???:

It seems that You don't have Gimp's Documentation installed.

Install the package "gimp-help-en" for English, "gimp-help-es" for Spanish. If you speak antoher language, look for the packages named gimp-help in Synaptic Package Manager.

```
sudo apt-get install [gimp-help-en]("apt://gimp-help-en")
```

------------------------

You can also visit the online help:

[http://www.gimp.org/docs/](http://www.gimp.org/docs/)


[HOW TO ADD BRUSHES TO GIMP]("http://www.simmerspaintshop.com/forums/gimp-tutorials/1379-adding-brushes-gimp.html")
