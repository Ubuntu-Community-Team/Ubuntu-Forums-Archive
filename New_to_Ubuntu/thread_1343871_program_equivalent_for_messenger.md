### Post by Temposs on 2009-12-02
If you're using Ubuntu 9.10, the equivalent is Empathy, which is installed by default. With versions before 9.10, the default messenger is called Pidgin. Go to Applications->Internet->Empathy/Pidgin. You can set up any account you want like Yahoo or AOL or MSN or whatever.
