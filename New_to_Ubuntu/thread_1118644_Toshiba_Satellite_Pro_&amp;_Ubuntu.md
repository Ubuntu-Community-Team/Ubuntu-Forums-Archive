### Post by roger_1960 on 2009-04-07
Hi

Best plan is to download the desktop .iso file, burn it to a CD and then run the "live" disk to see if all your hardware works.  This does not affect your hard drive at all as it runs all off the CD - it will be a bit slow but should allow you to test everything before installing anything.

I have a A200 which works fine with the live CD (but I have not yet installed it on that PC (also Vista!)
