### Post by ibuclaw on 2008-11-12
Try booting into Recovery Mode and selecting the option "**Fix X**" in the console menu, then continue with the normal boot process.
If this resolves your issue, then I can follow you through how to setup your video card with the Xserver, if you use a NVIDIA or ATI PCI/PCI-Express graphics card.

If you are still having problems, and it is true that this happens with 8.10 only, then you may consider switching back to 8.04, afterall, no one is forcing you to upgrade.

I personally prefer my main machine to be as stable as possible and I can do without some certain new features.
