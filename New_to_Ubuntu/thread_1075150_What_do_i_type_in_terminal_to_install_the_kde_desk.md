### Post by perlluver on 2009-02-20
```
sudo aptitude install kubuntu-desktop
``` that will install the KDE desktop, and leave gnome available, go to sessions, before you login and select KDE instead of gnome.

---

### Post by sdennie on 2009-02-20
The command above installs KDE 3.5 but, if you'd like to try KDE 4 instead, you can use:

```

sudo apt-get install kubuntu-kde4-desktop

```
