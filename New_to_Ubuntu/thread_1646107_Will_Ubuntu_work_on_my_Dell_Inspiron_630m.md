### Post by rockerphil on 2010-12-15
ok, here's a quick rundown. Linux in general is known for being able to work very well on older hardware, so your computer being compatible shouldn't be an issue. as to which version of Ubuntu to pick i'd personally suggest Lubuntu if it's running on some pretty modest specs. if it's REALLY old i'd suggest taking a look at this tutorial

[http://www.psychocats.net/ubuntu/minimal](http://www.psychocats.net/ubuntu/minimal)

i've personally had a Dell Dimension L500cx with a 500 MHz Intel Centrino single core processor and 128 MB of PC100 SDRAM running a minimal install and it was quite responsive. it simply depends on what kind of resources the machine has. typically you'll need around 512 MB of RAM to run the standard Ubuntu, i may be wrong, but i believe Xubuntu has a minimum RAM requirement of around 256 and Lubuntu i believe can run on as little as 128 MB of RAM, and from what i've heard you can get a minimal install running on 64 MB of RAM. i hope this information helps,

Phil
