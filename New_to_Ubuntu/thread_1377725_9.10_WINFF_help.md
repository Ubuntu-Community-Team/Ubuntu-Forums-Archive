WinFF uses FFmpeg, but by default Ubuntu does not enable MP3 encoding in FFmpeg.  You have to manually enable it, but it's simple to do.  I'm unsure what version of Ubuntu you're using, but this should work for you:

[HOWTO: Easily enable MP3, MPEG4, AAC, and other restricted encoding in FFmpeg](http://ubuntuforums.org/showthread.php?t=1117283)

> **Excedio said:**

Search for 'libmp3lame' and ensure it's installed.

Installing the libmp3lame package will not enable MP3 encoding in FFmpeg.
