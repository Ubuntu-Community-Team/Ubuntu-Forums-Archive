### Post by Mark Phelps on 2012-02-06
If you're running from the disk (LiveCD) mode, I don't believe you can do a Wubi install that way; instead, you need to boot into Windows, right-click on the Wubi installer and select "Run as administrator".  That should start the Wubi installation.
