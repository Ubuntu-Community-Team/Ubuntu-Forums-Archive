### Post by Lateralis on 2011-05-01
This is a good question.  I think I've solved it, but would be nice if you could test. 

Open up a Nautilus window.  Can do that by clicking on the home folder launcher in the launcher bar.  Click File -> Connect to Server.  Input all of the relevant details.  Then, the shared folder *should* appear in the left hand Nautilus panel.

Additional: I've just successfully used this to connect to my user space on a computer cluster at work.
