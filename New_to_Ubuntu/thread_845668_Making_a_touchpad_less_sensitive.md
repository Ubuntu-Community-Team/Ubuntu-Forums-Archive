Its driving me nuts. This note has taken one hour to write. Its deleted itself five times and several events like curser moving to the wrong place in a sentence have occured. What can I do?

### Post by nkri on 2008-06-30
Is it because of tapping?  If so, go to System>Preferences>Mouse; in the Touchpad tab, unclick "Enable mouse clicks with touchpad."
If you still want tapping, one of the sliders in the General tab might help.

If it is not tapping, search around in System>Preferences>Mouse and see if you can find it.

Also, if "Mouse" is not in the preferences menu:
```
gnome-mouse-properties
```
All of this is assuming that you use Gnome.

Good luck!
