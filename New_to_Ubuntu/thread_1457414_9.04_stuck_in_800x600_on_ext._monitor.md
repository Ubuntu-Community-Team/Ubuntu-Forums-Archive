### Post by skyeric on 2010-04-18
Do you not have an option to change it under system>perferences>display?
Is it possible the display cannot show a higher resolution than 800x600? unlikely but i suppose possible..
Try this terminal command
```
 xrandr -s 1024x768 
```

or ```
 xrandr -s 1024x600 
``` if its that resolution you want.

hope this helps,
Eric
