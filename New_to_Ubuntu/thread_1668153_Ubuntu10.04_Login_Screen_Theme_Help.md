### Post by Perfect Storm on 2011-01-16
The new GDM login can't use those themes, AFAIK know you can't change theme on the new GDM. But you can change the background of GDM. The easiest way is though Ubuntu Tweak: [http://ubuntu-tweak.com/](http://ubuntu-tweak.com/)

Ubuntu Tweak ---> Login Settings


The new GDM uses gtk2 themes somehow, though some while back I found an very ugly hack to change it. But it far from safe.
