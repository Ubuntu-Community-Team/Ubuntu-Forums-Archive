### Post by Moop on 2008-12-12
Make sure that you don't have gnash installed too or it will conflict with flash. You should try the 64 bit flash. It works much better than the 32 bit on 64 bit Ubuntu. 

You would need to completely remove flash plugin non-free and then install the 64 bit version available here [http://labs.adobe.com/downloads/flashplayer10.html](http://labs.adobe.com/downloads/flashplayer10.html)

Installation instructions. [http://labs.adobe.com/technologies/f...t.html#install]("http://labs.adobe.com/technologies/flashplayer10/releasenotes_64bit.html#install")
