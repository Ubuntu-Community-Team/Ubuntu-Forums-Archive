### Post by mikewhatever on 2013-09-17
That link tells you that there is no Skype for Linux on the ARM architecture.
As you probably know, Skype and Flash are proprietary, closed source applications. Adobe created Flash, and Skype created Skype for Linux on i386. They have not done the same on ARM - don't know why, but that's life.
If you can not live without Flash and Skype, I suggest getting a Windows notebook, because most of the development for Flash and Skype is now done for Windows, and not for Linux.
