### Post by poltz66 on 2011-11-14
I had a quick question more related to the terminal. I'm looking for a shortcut in the keyboard to copy a result into the input. For example: I put "ls" to see what files I have in a directory. I get various results. And now I want to get one of those commands (which happens to be a long name), and use the command tar on it. I want to know if there is a keyboard shortcut that will help me scroll through the results. Kind of like pressing up and down arrows to browse through the command history. I know there's a way of doing it with a mouse, but I don't have a mouse because I'm in a laptop.
Thank you

### Post by elgordodude on 2011-11-14
The tab key

If you're in the directory you can enter the first letter or two of the file, press tab, and the rest will be written out for you.
