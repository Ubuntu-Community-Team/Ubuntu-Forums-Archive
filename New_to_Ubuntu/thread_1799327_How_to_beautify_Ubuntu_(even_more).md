What can you tell me about?#-o

what I always do after a fresh installation:
- change graphics driver
- install faenza icons
- pick a nice wallpaper
- add smooth window minimize and close animations with Compiz  (have to install that first of course)
- enable menu transparency in Ubuntu tweak (have to install that first of course)
- change the system fonts (all set to Waree 9, Ubuntu bold 9 for the window title)
- change the "chosen items" -color and the window color (a little)
- set the launcher icon size to 33

*then* compare it to the general windows- appearance and you will find the last one dead-ugly, tasteless and cheap.
