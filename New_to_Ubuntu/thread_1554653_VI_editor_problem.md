### Post by dumdidum on 2010-08-17
If you aren't familiar with Vim, you might try entering something like "vim test" in a terminal window followed by <cr>, then press <i>. Now enter some text, then press <Esc>. Type ":wq" followed by <cr>. Finally, look at your creation via "cat test"

check out vimtutor. if this all sounds too complicated, try a more beginner-friendly editor such as nano.
