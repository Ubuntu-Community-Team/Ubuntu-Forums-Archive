### Post by 3rdalbum on 2011-08-23
> **Merisi said:**
> I've decided after some thought to start using Ubuntu after trying it  out and I have several questions which I would be very grateful if they  could be answered.

1.  What is the best way to install Ubuntu alongside Windows 7?

Boot from the Ubuntu CD and follow the instructions to install Ubuntu using the "alongside" option. I'd recommend defragmenting from within Windows 7 and backing up your data before installing Ubuntu because it involves repartitioning - which can sometimes cause loss of data if something goes wrong.

> 2.  Does dual booting put excessive strain on a computer? I have an i-7  processor and 6 gb of ram.  When I tried Ubuntu a few weeks my computer  sounded over worked and it made me panic a little.

Your computer hardware is only running one operating system at a time, so dual-booting does nothing different to your hardware. If your computer sounds "overworked" then perhaps you are using some Windows software to control your fan, instead of letting the BIOS control it. The latter method is almost always better and will cause the fan to work properly no matter the operating system.

It's safe for the fans to run at their full capacity. That's what they're rated to. So no need to worry about that side.

> 3.  I've heard that Linux doesn't suffer from all the viruses that  Windows has but is it really safe to do online shopping?  Also am I  putting the windows side of my computer at risk by dual booting Ubuntu  alongside it?

Think about it: Windows is not safe for online shopping, because there could be a piece of malware running on your computer listening for your credit card details. If there's no virus, there's no danger. Therefore, if Ubuntu doesn't get viruses (and it doesn't, there are no Linux viruses at the moment), there's no danger.

Of course, make sure your wifi is encrypted otherwise any data transmitted over it can be sniffed by anyone within 200m radius.

You're certainly not putting the Windows side at risk, once you've actually installed Ubuntu. There's a slight risk while installing because of the repartitioning step - but it's probably about 1% chance of data loss, perhaps less. It's the same for any partitioning operation, not Ubuntu-specific. **Just make a back-up and you'll be fine in any case.**


> 4.  I also found Firefox a little bit blurry and I was wondering if it could be sharpened up?

Give it some time. Firefox won't change its font rendering, but you'll get used to it. Every font rendering platform does it a little differently, and the human eye sometimes takes a few days to get used to a new method of font rendering.

I got headaches and even migraines when I started having to use Windows XP at my work. I was too used to the Ubuntu font rendering.


> 5. Last but not least I'm really impressed with Ubuntu but it seems like  a very steep learning curve and I find it so overwhelming.  Is there  any quick way to mastering it?

That happens. Ubuntu's not really a steep learning curve, but it does require "unlearning" some Windows habits and accepting that you're a "newb" again.

I'd recommend reading as much as you can. All Linux distributions have a reference guide built in, called 'man'. If you want to know something about a command, just use man:

```
man apt-get
```

tells you a lot about the apt-get command.

Because the system is so transparent (it's open-source, after all) you'll always find more that you can learn about. Also, the system changes quite a lot in quite a short space of time, so there's always new things added and updated that you can learn about.

For some practical information about what's out there to install and set up, take a look at the Ubuntu Guide: [http://ubuntuguide.org/wiki/Ubuntu:Natty](http://ubuntuguide.org/wiki/Ubuntu:Natty)

Good luck my friend, I hope you enjoy the ride!

### Post by MG&amp;TL on 2011-08-25
For a first time user, there is absolutely no advantage to manually partitioning. Only choose that option if you've made a mess of it, or you need something special doing, like a separate home and root partition. There is no difference in speed or computing power between the two. You will get to choose how much space you give Ubuntu, if you choose 'alongside Windows'.

Good luck. Ubuntu's worth it. :)
