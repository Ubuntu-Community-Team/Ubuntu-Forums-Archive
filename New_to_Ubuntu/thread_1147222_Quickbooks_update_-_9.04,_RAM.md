### Post by cwmoser on 2009-05-03
FYI, I had a problem getting QuickBooks to run on my AMD PC runing 8.04 or 8.10 Ubuntu.
The thread describing this is at:
[http://ubuntuforums.org/showthread.php?t=1009051&highlight=Quickbooks](http://ubuntuforums.org/showthread.php?t=1009051&highlight=Quickbooks)

I found that Ubuntu 8.04 and 8.10 would not run QuickBooks if you had more than 2GB of RAM.  With 3GB, or 4GB Quickbooks would not start up; with 2GB Quickbooks would run.

Now that I have moved to the new Jaunty 9.04 64-bit edition, I revisited this issue.  My QuickBooks instance will run with 4GB RAM now.  The reasons might be:
- a change in Ubuntu 9.04
- that I am now using a 64-bit OS
- that I had to move from Codeweavers Crossover 6.2.0 to 7.1.0
