### Post by NightwishFan on 2010-04-05
The netbook remix is similar to Ubuntu desktop. It just uses an interface arguably optimized for smaller screens. Try it from a Live USB and see if you like it. If not, use the normal desktop edition. The difference is mainly in the interface and package selection.

### Post by snowpine on 2010-04-05
Eeebuntu is now called EB and is based on Debian, not Ubuntu. The new version (4.0) is still in testing according to their website: [http://www.eeebuntu.org/](http://www.eeebuntu.org/)
