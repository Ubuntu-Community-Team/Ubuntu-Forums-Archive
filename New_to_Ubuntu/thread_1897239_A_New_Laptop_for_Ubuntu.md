### Post by BBQdave on 2011-12-18
> **hongdou said:**
> I've been a long time Windows user and have only dabbled in Linux.

I have a few concerns for switching over to Linux full-time.  There are some program compatibility issues I'm worried about, as well as some hardware issues.

If you are looking at getting an inexpensive machine, I would go with one that has Windows pre-loaded and GNU/Linux hardware friendly.  You could then dual boot Windows and Ubuntu Linux.  You would have the program compatibility of Windows and the freedom to explore Ubuntu.

When you were more familiar with Ubuntu and comfortable with it's desktop and programs, you could then load it on your machine as the only distro (lose Windows).

I came from a Mac and GNU/Linux world.  As time went by, it just became a better fit to run GNU/Linux.  Now all my production and play is on a notebook with Debian GNU/Linux :D
