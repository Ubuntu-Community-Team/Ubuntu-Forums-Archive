### Post by iMisspell on 2010-04-01
You can not "open/import" windows programs in Linux (Ubuntu).
You could install _[wine]("http://www.winehq.org/download/deb")_ in Ubuntu, and try and "re-install" some of your windows programs, but not all programs will work. Search around the wine site if you really want to use a windows program. 

The best thing to do is find a linux program which will do the same thing as your windows program, and install that linux program.

As for finding file on your windows harddrives...
What version of Ubuntu did you install ?
Did you install Ubuntu with 'Wubi' of a dual boot ?
