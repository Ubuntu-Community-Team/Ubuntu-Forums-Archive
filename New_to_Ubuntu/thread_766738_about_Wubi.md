### Post by avtolle on 2008-05-12
It is my understanding that by installing via Wubi, Grub will not be interfered with; you will access Ubuntu under the Windows version in which you install it, e.g., if installed under XP, you would choose XP from Grub, then have the option to boot XP or Ubuntu from there.
