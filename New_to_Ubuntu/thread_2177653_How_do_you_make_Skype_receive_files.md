### Post by khelben1979 on 2013-09-29
It could be that your Skype don't have permissions to save the incoming file to your folder. 

If you click the green button with a + on it and then look at the: **General tab**, there is a box which is either ticked or not, and it says: **Auto-Accept Files**. Below that you got a box which says, in my case: bear, since that's the place of my home catalog. So you can click on that and change the file path where this Skype program are going to save down any incoming files, and make sure you click in a folder which belongs to your user, otherwise your Skype program will not have the correct file permissions to be able to download any incoming files in there.

Good luck! :)
