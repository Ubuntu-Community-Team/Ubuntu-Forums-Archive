### Post by amjjawad on 2010-10-10
The safest way is:

1) [http://www.ubuntu.com/desktop/get-ubuntu/download](http://www.ubuntu.com/desktop/get-ubuntu/download) - Download the latest version from this link.

2) Check MD5sum for the .iso image file.

3) Burn to CD or create Live USB

4) Run the LiveCD/LiveUSB without installing Ubuntu

5) If your Wireless USB will work, then you can install it without any problem
