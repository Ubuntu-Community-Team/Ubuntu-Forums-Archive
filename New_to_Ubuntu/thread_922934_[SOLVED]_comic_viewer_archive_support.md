### Post by kostkon on 2008-09-17
> **Furonne said:**
> I'm reading some comics with comix right now, but i'd like to know if there are any viewers with archive support? so i don't have to unpack all the time, i read lots thank you muchly.
Yes, the default document viewer in *Ubuntu*, *Evince*. If you rename the .rar files to .cbr and the .zip to .cbz you will get previews in *Nautilus* since you will change the MIME type of these files from compressed archives to Comic Book archives in Ubuntu and will open with Evince by default.
