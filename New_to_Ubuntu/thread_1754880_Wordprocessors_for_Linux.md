If you don't need math, you can skip those parts. Another great "FM" can be found on wikibooks: [http://en.wikibooks.org/wiki/LaTeX]("http://en.wikibooks.org/wiki/LaTeX"). This manual is a bit less math-centered. Once you read that, you will know that you can load packages with style files using the command

```

\usepackage{*packageName*}

```

And all default CTAN packages (which are quite a lot) can be found on the [CTAN website]("http://www.ctan.org/") together with their documentation. The documentation of those packages is really complete.

To install the complete LaTeX environment on Ubuntu (I prefer the Kile editor, but you could use any text editor and compile via the CLI) you just have to install 

```

sudo apt-get install kile texlive-latex-extra

```
 
Note that Kile already ships with the texlive-latex-base package. so if you use another editor, check that that package is installed too.

[*] good if you only need a few layout schemas which can be used for all your documents (e.g. I have a layout schema for my reports and for my presentations)
[/LIST]

Cons of LaTeX:
[LIST]
