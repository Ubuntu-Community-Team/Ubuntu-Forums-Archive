### Post by blakkandekka on 2008-04-28
For some reason FF2 won't run the Google toolbar if I parallel install it.  Both versions seem to share add-ons as well, which leads to problems.

I really only used the toolbar for the bookmarks as well so I've added the [Deng Google Bookmarks]("https://addons.mozilla.org/en-US/firefox/addon/3892") add-on which seems to work fine with FF3 and takes up a lot less screen real-estate as well.
