### Post by acrazyplayer on 2010-06-01
Next time when youre in the command line type "sudo gdm start" If that doesnt work then try "sudo /etc/init.d/gdm start" 

When you get into Ubuntu try opening up a terminal and typing in "sudo apt-get install ubuntu-desktop" just in case you've un-installed anything important this command will reinstall everything that Ubuntu normally has
