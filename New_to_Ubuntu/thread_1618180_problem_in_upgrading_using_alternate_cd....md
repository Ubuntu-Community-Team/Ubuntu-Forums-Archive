### Post by mcduck on 2010-11-10
The point of the alternate CD is that it gives you alternative install options and a text-based installer.

It's not intended to be a smaller download option, and if it was it wouldn't help you at all since you would need a disc with updated versions of all your packages anyway. So leaving stuff out of the alternate CD would simply mean that it wouldn't have everything required for install/update any more.

If you want to download as small amount of data as possible, then the best way is to simply get the computer connected to network and let it update itself. That way you'll only download the files needed to update your system and nothing else.
