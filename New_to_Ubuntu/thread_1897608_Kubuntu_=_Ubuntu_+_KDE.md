### Post by Lars Noodén on 2011-12-19
You'll have pretty close to Kubuntu.  Kubuntu actually has a few other applications in by default too.  If you want to try KDE more or less by itself then you can install the package [kde-plasma-desktop](apt://kde-plasma-desktop).  If you want to install the whole kit with all the extra applications that make up Kubuntu, then install the package [kubuntu-desktop](apt://kubuntu-desktop).  KDE is highly configurable, you'll like it.
