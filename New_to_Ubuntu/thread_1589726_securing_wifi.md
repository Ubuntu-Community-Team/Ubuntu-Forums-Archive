### Post by e79 on 2010-10-06
Also if your router supports it, would be a good idea to implement '**MAC address Filtering**'. This would allow only known MAC addresses (the ones you trust) to use your WiFi. It would add another security layer to your WiFi network. Of course, this can always be circumvent....but this is another story  :P
