### Post by nhasian on 2009-01-08
I think you want to remove the *Snapping Windows* option under Windows Management in the Compiz Config Settings Manager. if you dont already have the ccsm installed you can install it with:

```
sudo apt-get install compizconfig-settings-manager

```
