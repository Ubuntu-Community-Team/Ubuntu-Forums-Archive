### Post by howlingmadhowie on 2009-03-02
i'd say learning to use the shell would be great. you really start to recognise the power of a good command line interface when you start writing little shell scripts to do things like bulk renaming of files or whatever. 

there are a number of bash tutorials online. look for one about usage of bash first (a tutorial to explain cd, mv, mkdir, rm, pwd, touch ...) before trying one for bash programming (variables, loops ...). the next step after that would be to learn sed and awk. 

learning bash and playing around with your system will help you to learn where things are and how ubuntu ticks :)

lots of fun!
