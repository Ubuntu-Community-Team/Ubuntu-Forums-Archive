### Post by sandyd on 2012-09-07
If you want to install an app, first look in the Ubuntu Software Center. Applications installed by yourself (i.e. not adding a deb repository) are not updated/patched, and will have to be manually updated to avoid security problems and bugs.

 The repositories are always the first choice. Be aware that ALL other alternatives require a working knowledge of linux and the process can be fraught with trial and error (and more errors) ... That said, your system is your system, and ubuntu will allow you to make/do whatever you have the ability to accomplish.
