### Post by Mark Phelps on 2009-09-01
Unless you're an experience Linux person, you should not be downloading apps from the internet.  

The much simpler way is to use Add/Remove and select the app you want.  

A little more difficult is using Synaptic -- but it will handle dependencies for you.

A third way is download the package file itself (.deb file) -- which you can get from getdeb, but then, you will have to resolve any dependencies yourself.

Downloading source (which  I'm guessing you did) is MUCH more difficult as you have to install a development environment in order to build the executables.
