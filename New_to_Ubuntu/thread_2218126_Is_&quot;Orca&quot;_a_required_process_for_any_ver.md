### Post by grahammechanical on 2014-04-19
Orca is installed on every Ubuntu installation for the reason of accessibility for all potential users. Orca is installed on my system but it is not activated and therefore it is not using any system resources. We can activate or deactivate Orca through System Settings>Universal Access. There should also be an option to activate/deactivate the screen reader at login. I think it is the human icon in the top panel.
