### Post by glubbdrubb on 2009-12-29
Try gnome-do. 

You can get it in the software center or type into a terminal:

```
sudo apt-get install gnome-do
```

then run it from the Applications menu. Go to the preferences and put it in dock mode.

### Post by fabounet on 2009-12-30
<out of topic>
Docky is NOT a front-end to Gnome-Do, but "just" another dock.
Gnome-Do does not need any front-end, since it already has its own.
</out of topic>

<inside topic>
moreover Cairo-Dock will let you have desklets, that you can place on the widget layer, much like on OSX.
</inside topic>

<generality>
but as usual, the best is just to try them all, and pick the one that fits you.
Cairo-dock has its own PPA, see here for the weekly version :
[https://launchpad.net/~cairo-dock-team/+archive/weekly]("https://launchpad.net/~cairo-dock-team/+archive/weekly")
and here for the stable version :
[http://www.cairo-dock.org/ww_page.php?p=From%20the%20repository&lang=en]("http://www.cairo-dock.org/ww_page.php?p=From%20the%20repository&lang=en")
