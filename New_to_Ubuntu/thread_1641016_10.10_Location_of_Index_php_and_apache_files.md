### Post by mcduck on 2010-12-08
The default location is /var/www/

Note that it's owned by root, so you won't be able to drop files there like you can inside your home directory. But starting the file manager as root by running "gksudo nautilus" will allow you to move the files there.

This should also help a lot:
[https://help.ubuntu.com/community/ApacheMySQLPHP]("https://help.ubuntu.com/community/ApacheMySQLPHP")
