### Post by jimmy the saint on 2009-01-18
Sharing the connection through the ubuntu box is possible, see
[http://www.ubuntugeek.com/sharing-internet-connection-in-ubuntu.html](http://www.ubuntugeek.com/sharing-internet-connection-in-ubuntu.html)
but a little complicated.  It would be easier to just set time filters by ip in your router.  As for sharing files, you need to setup samba.  Samba is a linux/windows network protocol that allows windows to access files on a linux networked machine.
a simple tutorial here
[http://www.simplehelp.net/2007/05/19/how-to-share-files-and-folders-in-ubuntu/](http://www.simplehelp.net/2007/05/19/how-to-share-files-and-folders-in-ubuntu/)
