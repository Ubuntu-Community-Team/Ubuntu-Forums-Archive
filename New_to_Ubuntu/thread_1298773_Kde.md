### Post by howefield on 2009-10-23
Using Synaptic Package Manager, install the kubuntu-desktop package.

At login you'll be able to choose from the session menu, I think it is.

Note, you will have mixed menus, with gnome and kde apps, kde apps will work in the gnome environment and vice versa.
