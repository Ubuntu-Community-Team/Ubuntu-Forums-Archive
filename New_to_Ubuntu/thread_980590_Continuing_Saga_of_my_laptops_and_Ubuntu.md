### Post by candtalan on 2008-11-13
Just to mention that you do not need to actually install 8.10 in a pc to use the create usb stick. If you have an 8.10 iso file and an 8.10 Live CD that you can run (RAM minimum 384MB), then you can create the usb stick.

If you can download the 8.10 live CD iso file, then you can also create a live CD to use. If you are running software that can create an iso file image directly from a cd (I use k3b) then it could come down to first getting or borrowing a live cd 8.10.
