### Post by cortman on 2011-12-13
If you're using Gnome shell...

If you haven't already installed gnome-tweak, do so:

```
sudo apt-get install gnome-tweak-tool
```

Open it up (it'll be called "Advanced Settings"). Under the "Desktop" menu, slide "Show mounted volumes on the desktop" to On.
