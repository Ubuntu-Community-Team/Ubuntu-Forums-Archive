### Post by nishant.singh28 on 2010-01-16
OK. What do you man you mean by how to use Compiz? Simply go to your Ubuntu software centre and install the following:
Compiz
ccsm(Advanced Compiz Config Settings Manager)
Compiz Fusion Icon
You may also want to install Emerald Theme Manager.
Then, in System->Preferences, first select Appearance and enable Desktop Effects. Then Select ccsm and customise your effects. You can download .emerald files and use them using Emerald theme manger.
