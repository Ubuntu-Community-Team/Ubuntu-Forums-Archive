### Post by jenkinbr on 2008-11-26
Install CompizConfig Settings Manager:
```
sudo apt-get compizconfig-settings-manager
```

You will also need to make sure you have restricted drivers enabled.

you can start the settings manager by pressing <alt+f2> and typing 'ccsm' into the box that appears.
