### Post by jbrown96 on 2009-08-22
1) Installing firefox 3.5 won't replace 3.0. They will coexist. The thing to keep in mind is that the shortcut icon at the top will still point to 3.0, so you will need to change it. Right click and select properties. Then change the command to firefox-3.5 

I use 3.5 exclusively, and there aren't any issues with it.

2) install vim if you want to use it from a terminal. You probably want to use it as a standalone app in which case install vim-gnome.
