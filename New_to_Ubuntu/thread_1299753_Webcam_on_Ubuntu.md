### Post by murderslastcrow on 2009-10-28
You can use WLM 8.5 in Wine using the offline .msi file. There are many tutorials explaining how to do so. I did so successfully with my webcam. This way you can use Windows Live Messenger itself.

However, emesene, aMSN, and I believe Kopete all have video functionality. Not sure which one has video call functionality.

A great secondary option is to use Skype, which is a fairly simple and small download, for video calls. These are the only solutions I can think of at the moment, and I searched for a lot of info on it a few months ago and came to these conclusions.

Oh, and empathy should do video and audio, not sure if it can do both simultaneously, though.

### Post by 3rdalbum on 2009-10-28
The latest version of Emesene supports video on MSN, if you run the included compiling script to compile the support for it (it's easy, don't worry).

I personally use Meebo (online messenger). It open up a Tokbox video chat session. If both people are on Meebo it's completely seamless. If the other person is not on Meebo, then they get a link to click to join the chat. But it works very well, you can video chat to anyone on any IM network, and you can access all your IM accounts from any computer.
