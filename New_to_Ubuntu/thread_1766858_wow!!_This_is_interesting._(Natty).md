### Post by decoherence on 2011-05-24
> **mikeybinec said:**
> Anyway It did install and well I was expecting warp drive and photon torpedoes but nothing new..

Well, it IS a command line interface... it hasn't changed much in decades. What you did get by doing the upgrade are newer versions of the software on your system. Seeing as it's the server version, hopefully not too many things change and those that do, hopefully do so in a way you don't notice.

People who did the upgrade on the desktop version of Ubuntu did get warp drive and photon torpedos, mostly in the form of a shiny new graphical user interface. On a server, consistency between versions is generally a Good Thing.
