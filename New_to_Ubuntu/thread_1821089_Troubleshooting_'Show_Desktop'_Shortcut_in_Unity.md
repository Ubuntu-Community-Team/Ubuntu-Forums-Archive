I added a 'show desktop' icon to Unity dock using code found here:

(nice list of tweaks, by the way), to wit:
```
sudo apt-get install wmctrl cd wget http://webupd8.googlecode.com/files/showdesktop.tar.gz tar -xvf showdesktop.tar.gz && rm showdesktop.tar.gz sudo mv showdesktop /usr/local/bin/
```then dragged and dropped the shortcut to the launcher.

The shortcut now appears in the launcher but clicking it has no effect (though ctrl-alt-d does work)

Any ideas about getting this shortcut working?
