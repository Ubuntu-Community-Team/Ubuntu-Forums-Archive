### Post by Miljet on 2011-08-06
Whenever I do something like this, I usually copy everything that is not a hidden folder. In other words every folder that does not begin with a period. All of the configuration files and folders will be created as needed. The only thing that I copy from a hidden folder is the bookmarks backup from .mozilla/firefox/vkuuxfit.default/bookmarkbackup. The file you are looking for ends with .json. You only need to copy the latest one. After the bookmark file is copied to the correct directory, you open firefox, go to bookmarks, organize bookmarks, restore, and select the file you copied.
