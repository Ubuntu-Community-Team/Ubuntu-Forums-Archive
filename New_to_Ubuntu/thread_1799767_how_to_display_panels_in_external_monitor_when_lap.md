### Post by noviciate on 2011-07-08
I use a docking station for my HP laptop running ver10.04. Today I undocked to test the wireless router I just installed. I turned off the monitor setting for an external monitor while working to get the wireless connection. The wireless connection works but when I returned the laptop to the docking station and restarted the external monitor no longer has menu panels. A single desktop object is displayed and right clicking doesn't get a panel display menu.

I undocked the laptop and the laptop display has the panels as usual. Both the external and laptop displays are detected if I connect the external monitor to the undocked laptop. If I try to set the external as the leftmost (primary?) display it does shows the menu panels but the display is blurred and barely readable. If I keep that configuration and then again dock the laptop the external monitor has a clear display with the single desktop object and no panels.

Is there a setting that will again give panels in the external monitor when the laptop is docked?

---

### Post by noviciate on 2011-07-08
The problem was caused by the display of the laptop in the docking station being retained as the primary display that has the menu panels. I couldn't find a setting to make the external monitor the primary while the laptop was docked.

The solution is to [1] dock the laptop, [2] open the laptop to use that display with the menu panels to change the Monitors settings under Preferences, [3] verify that the external monitor has the correct resolution settings and is on, [4] turn off the laptop display, [5] keep the new configuration settings, [6] close the laptop while still docked. The external monitor becomes the only display and has the menu panels.
