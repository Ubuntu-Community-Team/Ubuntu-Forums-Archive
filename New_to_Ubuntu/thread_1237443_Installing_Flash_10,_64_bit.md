### Post by XCan on 2009-08-11
To put it a bit more simply, since the link provided above gives a lot of information how to cleanup other flash version. Extract the tar.gz file, inside you should fine a libflashplayer.so file. Put that file into ~/.mozilla/plugins . Restart Firefox and you should be good to go.
