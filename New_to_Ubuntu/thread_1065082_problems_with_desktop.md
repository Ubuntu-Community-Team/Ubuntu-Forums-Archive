### Post by Person98 on 2009-02-09
if you can open the system monitor you can end the processes that are giving you trouble (much like the task manager in windows), if is only those 2 programs that are giving you trouble you can try alternatives to them, opera or epiphany for a webbrowser and abiword for a word processor. To restart the panels you can press alt-f2 and type: xfce4-panel , that should restart the panels when running xubuntu or type gnome-panel if you are running ubuntu.
