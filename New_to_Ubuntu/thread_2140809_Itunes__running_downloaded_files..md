### Post by Impavidus on 2013-04-30
Welcome to Ubuntu and to these fora.

My bet is that you're trying to run .exe files. They won't run on Linux. Some .exe files are self-extracting archives, which the archive manger can handle, and therefore the archive manager pops up. If it's not an archive that won't work. For most things a Linux alternative exists and there are a few sites around that list them, for some others you can try wine. Wine is a compatibility layer to run windows programs on Linux, but success it not guaranteed. See [http://appdb.winehq.org/](http://appdb.winehq.org/) to find out whether others have had success with the programs you want to run.

Other files can be opened on Linux, if the right program is installed. For music and video files you may need to install some codecs and be advised that MS Office files don't always open perfectly, but usually readable.

First thing to remember: Linux is not Windows.
Second: You're always welcome for questions.

You can get many applications right in the ubuntu software center...when using an outside file, you need deb files as linux doesn't run the windows .exe files unless you run them through the wine program...
If you can though, better to use the deb files diirectly...

For example, Google Chrome is not available in the software center, but you can get the linux deb file from their website and then right click and install it through ubuntu software center...

### Post by Baldrick_NZ on 2013-05-01
Welcome aboard!

Firstly, if you're looking to use iTunes to download music - forget it - under Linux. It just won't happen, believe me I tried! Some success with getting the player to work, but the store - nah.
Use Google Music to download music instead, if available in your country. It has way more benefits than iTunes anyway, IMHO. There is also a 'frame' app that works pretty well with Google Music, which saves having a browser open when you're using Google Music.

If you want iTunes to transfer files to & from your iDevice, then both Banshee & Rhythmbox can do this. There is also a stand-alone app called gktPod, which I believe achieves this as well.

Incidently, do a search for 'Nightingale', which is an iTunes lookalike for playback. Nightingale is a fork of Songbird, which stopped support for Linux a year or so back.

Hope that helps.
