### Post by Peter Maunder on 2013-11-19
The current GNOME screensaver is just a blank screen. The normal Windows screensaver in contrast is similar to Xscreensaver. The default is to show a moving Windows logo.  There are options to choose a number of  moving images and the user's choice of their own pictures folder.  There are a number of questions devoted to Xscreensaver and how to install it.

Screensavers seem to be an emotive issue. You either prefer a blank purist screen or something more exciting. Perhaps the problem is in the name?
