### Post by P1C0 on 2011-03-27
It might be the graphics card. Booting into safe mode doesn't stress the gpu so much and even if you have some burnt capacitors on the card, it will work.
I've seen it happening on Windows XP.
Booting in normal mode -> freeze after some seconds
Booting in safe mode -> ok
