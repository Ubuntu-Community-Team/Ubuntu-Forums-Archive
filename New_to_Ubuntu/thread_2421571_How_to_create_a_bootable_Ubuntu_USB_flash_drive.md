### Post by C.S.Cameron on 2019-06-24
The main problem with Rufus out of the box is that there is no Persistence.
The drive is reset every boot, newly installed programs and settings disappear
Persistent installers include: UNetbootin, Universal and YUMI for Windows and mkusb for Ubuntu.
A Rufus install can be made Persistent, see: [https://askubuntu.com/questions/1126145/can-i-convert-a-live-ubuntu-usb-to-one-with-persistent-memory/1126305#1126305](https://askubuntu.com/questions/1126145/can-i-convert-a-live-ubuntu-usb-to-one-with-persistent-memory/1126305#1126305)
