### Post by grahammechanical on 2011-11-26
Your question is not clear. Please answer these questions:

What operating system do you use? Is it Ubuntu? What version? 11.10?

What do you mean by?

> i want to change my system operative

Do you want to install the Ubuntu operating system but do not know if you should download Ubuntu (latest version) 32-bit recommended or Ubuntu (latest version) 64-bit?

I have an Intel Dual Core 2.4GHz CPU. It is a 64-bit CPU. Your Intel Dual Core CPU will also be 64-bit. You can make sure by reading the user guide that came with the laptop. It should give you the specifications of your computer.

Having a 64-bit CPU means that you can run a 32-bit operating system or a 64-bit operating system. Both will work fine.

If you had a 32bit CPU then you would only run a 32bit operating system. A 64-bit operating system will not work on a 32-bit CPU.

The Ubuntu web site recommends 32-bit Ubuntu for people like you and me who are not confident in what we are doing. If we install a 32-bit Ubuntu then we will get a working Ubuntu and be happy whether our CPU is 32-bit or 64-bit.

If we install a 64-bit Ubuntu and our machine only has a 32-bit CPU, then the operating system will not work and we will not be happy but we will not blame ourselves. We will say that Ubuntu does not work.

When I first installed Ubuntu I installed 32-bit as I was unsure. Later as I gained experience I installed 64-bit Ubuntu.

A decision has been taken to change the Ubuntu web site next year to recommend 64-bit Ubuntu. This is because most computers sold have 64-bit CPUs as standard.

I hope that this helps. If you have Windows 7 on your machine and want to keep it as well as use Ubuntu then you should research how to install Ubuntu as a Dual boot system and find out what problems there might be when installing Ubuntu with Windows 7 already on the machine.

Regards.
