### Post by whitethorn on 2008-09-15
Hi,

I followed some guide somewhere and it said you just have to add these repositories to you  /etc/apt/sources.list

> deb [http://repository.cairo-dock.org/ubuntu](http://repository.cairo-dock.org/ubuntu) hardy cairo-dock
deb [http://ppa.launchpad.net/compiz/ubuntu](http://ppa.launchpad.net/compiz/ubuntu) hardy main


Then 
```
sudo apt-get update
```
```
sudo apt-get install cairo-dock
```

that should also take care of it.
