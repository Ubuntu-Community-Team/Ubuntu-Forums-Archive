### Post by bcbc on 2010-10-29
FYI WUBI checks the md5sum automatically, so this is totally unnecessary (it's a good idea if you are running a live CD or doing an install from CD though). WUBI also makes sure that the .iso you download manually is valid e.g. correct release etc. (downloading manually can speed up wubi installs if you place the .iso in the same folder as wubi.exe before running it).

The log file you see mentioned in that image can be accessed from the %temp% directory - it's called wubi-10.10-r197.log (or something similar). You can copy and paste it between [CODE ][/CODE] tags if you like and I'll take a look at the error.

Usually wubi installs very easily, but every now and then there are issues. Generally, you want to run as an administrator and have internet access, but sometimes there are still problems.
