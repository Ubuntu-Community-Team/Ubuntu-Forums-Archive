### Post by mcduck on 2009-04-28
In Linux you don't usually download anything yourself, but instead let the package manager handle both downloading and installing (as well as updating and removing) of everything.

For most commonly used software just go to Applications - Add/Remove...

To access all available software you should use Synaptic (in System/Administration/Synaptic Package Manager). Just find what you want, right-click ansd select "Mark for installation". When you have selected everything you wish to install/update/remove just click the apply-button.

Still, if you need to install something that isn't available in Ubuntu's repositories, the package you download will in almost all cases also include instructions how to install it.
