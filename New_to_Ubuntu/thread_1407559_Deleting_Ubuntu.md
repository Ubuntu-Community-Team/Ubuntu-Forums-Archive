### Post by abcuser on 2010-02-15
Download [Gparted](http://sourceforge.net/projects/gparted/files/gparted-live-stable/) partition tool and delete all partitions created by Ubuntu. Then reinstall Windows. This is in case you only have Ubuntu on computer. If you installed it inside Windows (using Wubi installer), then just go to Control Panel and Add Remove Programs and select and uninstall Ubuntu.

By the why what kind of problems would you like to solve with Ubuntu, that you were unable to solve with Windows?
