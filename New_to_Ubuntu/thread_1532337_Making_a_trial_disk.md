### Post by jmszr on 2010-07-16
gordon33,

After you have downloaded Ubuntu you need to verify the md5SUM: [https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM) and burn the Ubuntu image to a CD: [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto) . Then check the newly burned CD's integrity: [https://help.ubuntu.com/community/Installation/CDIntegrityCheck?action=show&redirect=CDIntegrityCheck](https://help.ubuntu.com/community/Installation/CDIntegrityCheck?action=show&redirect=CDIntegrityCheck) .

As far as games go - it depends on the game. Ubuntu has a considerable amount of Linux games available and some Windows games will run under Wine: [http://www.winehq.org/](http://www.winehq.org/) . You can ask in the Gaming & Leisure sub-forum if you have any specific game-related questions: [http://ubuntuforums.org/forumdisplay.php?f=93](http://ubuntuforums.org/forumdisplay.php?f=93)

Hope that helps.

Edit: You can also make a persistent flash drive install of Ubuntu (it will retain any changes you make to the install and allows you to save files etc.) You just boot from the thumb drive into a regular Ubuntu install (the flash drive serves as your hard drive) for all intents and purposes. Here's a howto to make one using a Windows computer: [http://lifehacker.com/5574276/universal-usb-installer-makes-a-persistent-thumb-drive-version-of-any-linux-os](http://lifehacker.com/5574276/universal-usb-installer-makes-a-persistent-thumb-drive-version-of-any-linux-os)

### Post by tarps87 on 2010-07-16
If you are looking for games and apps you would need to create a custom cd.  A live cd would not persist any data an is therefore an issue if trialling a system over a period of time.  If there PC will boot from USB a persistent USB install will be a better option.
