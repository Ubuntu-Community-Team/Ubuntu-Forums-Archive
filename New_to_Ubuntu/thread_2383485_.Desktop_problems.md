### Post by monkeybrain20122 on 2018-01-25
where is simi1.py? It doesn't look right that it is in /usr/share/applications (the .desktop file should be there, but not the binary or script that it invokes)

look for it with 
```
locate simi1.py
```

Also the I think Exec line should be
```
python /path/to/simi1.py
```

since you need to run the python script.
