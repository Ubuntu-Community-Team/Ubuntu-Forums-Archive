### Post by Frogs Hair on 2013-12-08
Hello and Welcome

The Flash Plugin Installer is in the software-center alone or along with the Ubuntu Restricted Extras package . Open the software center , search for the packages, and select install. The Ubuntu Restricted Extras package includes audio/video codecs as well.
