### Post by VeeDubb on 2011-12-03
> **DrMilo said:**
> Not what I wanted to hear but thanks!

Don't worry too much.  This is no different than if you were running a pure gnome3 desktop with gnome shell.  You'd then have gnome shell running on top of all of the gnome libraries.  

Unity is only the most superficial layer of the operating system.  Like almost everything on a linux machine, there are layers upon layers of functionality, and each depends on the layers below.  Unity just happens to depend on the same supporting libraries/programs/etc as gnome shell.
