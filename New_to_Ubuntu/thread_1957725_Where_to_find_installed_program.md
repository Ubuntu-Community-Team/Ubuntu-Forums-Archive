### Post by papibe on 2012-04-13
Hi sreeharitk.

'indent' is a command line tool. Open a terminal and run it like this:
```
indent your_program.c
```
change 'your_program.c' for the name of your actual C program.

Hope that helps, and tell us how it goes.
Regards.
