### Post by Xero Xenith on 2009-05-01
A (delayed) +1 to Gmail, but one extra note - if you would so desire, you can use IMAP instead of POP for Gmail. The difference being, POP simply downloads your messages and lets you do what you will with them, and nothing is synced back to the server (one-way), while IMAP syncs everything back to the server (two-way).

More info here:
[http://mail.google.com/support/bin/answer.py?hl=en&answer=75725](http://mail.google.com/support/bin/answer.py?hl=en&answer=75725)
