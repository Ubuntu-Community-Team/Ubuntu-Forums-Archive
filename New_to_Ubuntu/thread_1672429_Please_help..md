### Post by cariboo on 2011-01-20
I'd suggest you start in recovery mode, and once at the menu select failsafe X, this should take you to your desktop, but it will be running in low graphics mode.

Once at the desktop go to System->Administration->Hardware Drivers/Additional Drivers, and make sure you have the correct video drivers installed.

Once you done that, go to System->Administration->Synaptic Package Manager->Edit->Fix broken packages, and remove the broken packages. Once you have done that reboot and make sure things are working correctly.
