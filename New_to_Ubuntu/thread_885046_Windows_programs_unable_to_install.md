For your MS Office needs, I strongly suggest using a virtual machine rather than Wine. Office is so deeply tied into Windows that you'll pretty much end up installing most of windows anyway to get Wine to support Office.
Check out VirtualBox in the repositories, it's simple to set up and works well. You'll need a copy of Windows to install on the VM.
