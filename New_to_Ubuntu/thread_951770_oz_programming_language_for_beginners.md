### Post by phidia on 2008-10-18
[Python]("http://www.python.org/doc/2.3.5/tut/tut.html") is often recommended particularly for those using Linux. But what I mostly hear and read is that it's important to learn good techniques and ways of thinking right at the start rather than believing there is one best programming language.

Wikipedia does has a fairly complete description (with links) for [OZ]("http://en.wikipedia.org/wiki/Oz_(programming_language)") if you're interested in that.
