### Post by drs305 on 2010-04-01
Grub2 is a major revision of the bootloader and it doesn't use the same file structure. The menu is created from the contents of /boot/grub/grub.cfg but this file is normally read-only. The reason is that it is generated by various scripts. The main settings the user would normally change are located in /etc/default/grub.

There are a series of guides linked in my signature line that explain how Grub2 works and some of the common tasks the user might wish to perform.

Start with the Grub2 Basics post, which has links to other sites/posts at the end of the post.
