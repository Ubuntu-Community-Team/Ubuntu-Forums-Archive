### Post by Tatty on 2008-05-30
You are using the correct driver.  ATI stopped supporting cards older than the radeon 9600 a while ago so their official drivers for these older cards dont work with the latest xorg/free86 versions.  So you are stuck using the open source ati drivers, which are defined in your xorg.

Try reconfiguring your xorg.  I am assuming you are using hardy, so you cannot specify specific resolutions in it like you could in older versions - but what it will do is essentially "reset" your xorg.conf file so hopefully it should autodetect things a bit better.

First back up your xorg.conf so we can go back to it incase it all goes wrong, then do this:

```
sudo dpkg-reconfigure xserver-xorg
```
