### Post by didiercool on 2009-04-29

### Post by Kareeser on 2009-04-29
hananas, you can change the setting in compizconfig-settings-manager. You may have to install the package.

Uncheck the plugin "Workplace Switcher" (or disable the bindings Mouse4 and Mouse5, which switch between desktops)
