### Post by Old_Man_Unix on 2011-03-12
Confused (me):confused:

Firefox is a web browser.  It is used (mainly) to open (mostly) remote web sites that are written (mostly) in** HTML**.  It can also be used to open local HTML files.

Adobe Reader is a file reader.  It is used to open and read **PDF** files ("documents").

They are two different functions.

When you install the Adobe Reader,  it will install a ***plug-in*** in Firefox (and other browsers as well).  This is a separate function from the reader - although it is included with it.  The plug-in is supposed to enable your browser to open embedded PDF files (files that are embedded within the web page) in your browser window.  Very convenient if it always works.  Unfortunately it doesn't - at least not out-of-the-box.  

You can read all about it in the Firefox documentation here:

[http://support.mozilla.com/en-US/kb/Opening%20PDF%20files%20within%20Firefox]("http://support.mozilla.com/en-US/kb/Opening%20PDF%20files%20within%20Firefox")



Hope this helps and good luck.

reading the first line and if i'm not mistaken, you want firefox to be set as the default web browser again?
1. start firefox -> menu 'edit' -> preferences -> advanced -> general and click on 'Check Now' in the section system defaults. this way ff is set as your default web browser.
2. or you can go to system -> preferences -> preferred applications -> here you choose your preferred web browser.

reading a bit further:
in order to set a default application for pdf files you can right click on a pdf file and choose: open with other application -> choose preferred pdf reader and check 'Remember this application...'

see the other posts to set default applications for ff to use.

there are more ways to get the job done but this looks like a pretty easy way.
