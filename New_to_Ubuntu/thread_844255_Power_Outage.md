### Post by Canis familiaris on 2008-06-29
APT is the package management in Debian/Ubuntu systems. It facilitates installation, updation and removal of programs through centralised repositories.Also prevents conflicts and takes place of all dependencies.
Basically if you use Synaptic or apt-get commands or Applications->Add/Remove or even CNR to install stuff in Linux, you are using APT.

### Post by hyper_ch on 2008-06-29
apt is the package manager in debian-like system. If you use "Add/Remove Software", synaptic, adept, apt-get, aptitude you are just using some frontends for apt.

Redhat/fedora core use RPM as package manager

Gentoo uses emerge (or something like that)
