### Post by Maverick_Meerkat on 2010-08-04
Greetings,

F2 usually gives access to BIOS on Asus computers, you can also try esc, delete, or insert.

As a last resort, do not fear installing Ubuntu. Even if it's only for a test run. It's quick and should be easy to install. Furthermore, I haven't heard of anyone having difficulty uninstalling/removing it. 

Personally, I couldn't get the live CD of Ubuntu 10.04 to run on my laptop. So I simply installed using wubi and the .ISO directly from my HDD to a single partition running XP Pro.

I liked it so well, I later partitioned my drive and installed it on it's own partition as the default OS.

### Post by -kg- on 2010-08-04
According to your [laptop's user's manual]("http://www.retrevo.com/s/Asus-K40IJ-Laptops-review-manual/id/23511bh340/t/1-2/"):

> 2. Restart the Notebook PC and press [Esc] on bootup and select the optical drive (may be labeled as “CD/DVD”) and press [Enter] to boot from the (Recovery) DVD (or CD, in this case).

So for your particular laptop, the Escape Key (Esc) should take you into the boot selection menu.  Of course, the CD needs to be in the tray before you do the reboot.
