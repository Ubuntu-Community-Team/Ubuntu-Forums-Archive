### Post by Dennis N on 2009-12-23
The program you need to open an archive file is listed in Synaptic Package Manager as "file-roller", but is referred to as "Archive Manager" in the 'open with' dialog - right click on an archive file icon, and the top entry is "open with archive manager". After opening the archive, you can use the 'extract' button to extract the contents to any location. 

Check to see that file-roller is installed; it should there by default. If not, it can be installed with Synaptic.
