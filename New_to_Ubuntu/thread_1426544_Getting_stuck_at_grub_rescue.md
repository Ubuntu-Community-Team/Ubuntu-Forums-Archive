### Post by kansasnoob on 2010-03-10
Did you recently remove an operating system or something?

It says:

> Grub 2 is installed in the MBR of /dev/sdb and looks on the same drive in partition #5 for /boot/grub.

But there is no OS on sdb5:

```
sdb5: _________________________________________________________________________

    File system:       ext4
    Boot sector type:  -
    Boot sector info:  
    Operating System:  
    Boot files/dirs:   
```

The only OS I see is on sdb1:

```
sdb1: _________________________________________________________________________

    File system:       ext4
    Boot sector type:  -
    Boot sector info:  
    Operating System:  Ubuntu 9.10
    Boot files/dirs:   /boot/grub/grub.cfg /etc/fstab /boot/grub/core.img

```

So from the Live Desktop:

```
sudo mount /dev/sdb1 /mnt && sudo mount --bind /dev /mnt/dev &&sudo mount --bind /proc /mnt/proc && sudo chroot /mnt
```

```
grub-install /dev/sdb
```

If that returns any errors then:

```
grub-install --recheck /dev/sdb
```

That runs slowly so be patient, then:

```
exit
```

```
sudo umount /mnt/dev && sudo umount /mnt/proc && sudo umount /mnt
```

And reboot.
