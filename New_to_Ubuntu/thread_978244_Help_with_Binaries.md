### Post by Moop on 2008-11-10
You can right click on the binary file, choose Properties and then choose Permissions. Check the box for executing the file as a program. 

If you're trying to run windows games they may work better using wine. If you tell us the specific programs I'm sure someone can help. 

Generally speaking clicking on a executable isn't the linux way to install applications.
