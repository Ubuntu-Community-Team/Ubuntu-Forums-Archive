### Post by dhughes on 2008-08-17
Download option #1 the "x86 architecture" version since it will work on a 32-bit PC or a 64-bit PC.

 Next get and install Alex Feinman's ISO Recorder it makes it easy to burn the Ubuntu .iso that you downloaded and makes it easy to burn the .iso as a bootable CD, teh install CD. Just take notice there are three versions of this one for Vista (version 3), Windows XP (version 2) and version 1 which would be for "...Windows XP Gold (no service pack) and Windows XP SP1...":

[http://isorecorder.alexfeinman.com/v2.htm](http://isorecorder.alexfeinman.com/v2.htm)  <---Windows XP

[http://isorecorder.alexfeinman.com/Vista.htm](http://isorecorder.alexfeinman.com/Vista.htm) <--- Vista
