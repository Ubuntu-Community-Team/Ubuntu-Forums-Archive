Thank You.Ubuntu is an Operating System (OS), and it runs your computer hardware. It is NOT a Windows replacement, or an OSX replacement. It is a completely different OS based on the Debian OS, and uses the Linux kernel. More information [here]("https://help.ubuntu.com/10.04/about-ubuntu/C/index.html").
