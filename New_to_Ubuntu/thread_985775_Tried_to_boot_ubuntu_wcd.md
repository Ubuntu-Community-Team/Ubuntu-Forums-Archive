### Post by NewJack on 2008-11-18
First things first, before burning the iso to a CD, did you check the hash.  If not go to this page: [https://help.ubuntu.com/community/UbuntuHashes](https://help.ubuntu.com/community/UbuntuHashes) and make sure to check the hash for the version of Ubuntu that you downloaded.  

If you don't know how to do an MD5SUM, then check out this page: [https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

Hope this helped!
