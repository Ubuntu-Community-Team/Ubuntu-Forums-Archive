### Post by enchantedsky on 2008-05-24
Just do a Google search for the MS-DOS version of Doom and download it from a shareware website. 

It should run under Ubuntu. Just install Dosbox through Add/Remove programs, and setup the DOS emulator, and you're good to go.

I have run an old 1983 DOS game in Ubuntu, (Castle Adventure) and it runs great. There is no reason why Doom shouldn't work ----- you don't need a special Linux version of it.
