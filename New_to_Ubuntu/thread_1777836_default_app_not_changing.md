### Post by marcio_mi on 2011-06-08
usually the way I change the default application is through ubuntu tweak
try downloading it here if you want: [http://ubuntu-tweak.com/](http://ubuntu-tweak.com/)

once installed, you should see in the list on the left under "system" an entry called "file type manager". Clicking on that will give you the list of file types and their associated default software. you can try changing from there and see if it works.
