> **jason15417 said:**
> does an upgrade work or does an unistall then new version install have to be done i want the newest transmission installed and got a torrent half way done so should i wait or can i upgrade now?

upgrading will take longer as it will upgrade all of your packages, and install all of the new versions of your packages and the dependencies.

A reinstall will be quicker, but then you will have to re-install all of your apt-get packages.
