### Post by Paqman on 2012-06-16
Try [Gnome Nanny]("apt:nanny"), this kind of thing is exactly what it's designed for. I don't think it logs sites by default, but you can blacklist or whitelist sites. To log sites visited you could install a caching proxy like squid, but that gets pretty complicated.
