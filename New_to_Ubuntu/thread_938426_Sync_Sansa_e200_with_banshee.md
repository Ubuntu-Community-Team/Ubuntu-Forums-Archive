### Post by Mud.Knee.Havoc on 2008-10-04
I don't use Banshee, so I can't help you with that exactly, but I can offer a bit of hope!

I've got a Sansa e270 and find it very easy to add media to the player.  I don't bother trying to sync it with a program, though.  As soon as I plug it in via usb, it appears as a removable drive, so I can just drag and drop mp3s and movies to the player.

I also have Rockbox installed, which I recommend for Sansa owners (rockbox.org, it's a dual-boot OS that adds a lot of functionality to the player including nearly any audio format you can imagine, .mpg video, games, etc). 

The only thing I have synched to it is gPodder, which will copy all newly downloaded podcasts to the player with one click.
