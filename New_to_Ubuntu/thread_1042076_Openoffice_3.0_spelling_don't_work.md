### Post by redseventyseven on 2009-01-17
First of all, this isn't the Open Office support forum - you should visit [http://www.oooforum.org/](http://www.oooforum.org/) for enquiries about Open Office - especially OpenOffice 3.0 which if I recall correctly is not yet in the Ubuntu repositories.

However I may be able to help you with this problem.

Download the extension file from the page you suggested, and save it somewhere where you can easily find it again (e.g. the desktop). In OpenOffice, go to Tools > Extension Manager. You can then add your dictionary from there.

You will probably need to restart OpenOffice before you'll be able to choose your new language.

Hope that helps. If not, please direct your question to the OpenOffice community. I'm sure they'd be happy to help.
