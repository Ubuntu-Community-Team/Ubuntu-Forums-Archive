Hi, and welcome to the Ubuntu forums!

I don't use Aria (Aria2 seems to be the latest version), but from the description in Syanpatic Pakage Manager, it is primarily a command line tool.

I did find a web page describing a way to use it alongside Firefox, but it involves a little scripting:

[http://wael.nasreddine.com/2008/04/01/aria2-firefox-3.html](http://wael.nasreddine.com/2008/04/01/aria2-firefox-3.html) 

You can also get more information from the Aria2 web site at:

[http://apps.sourceforge.net/trac/aria2/](http://apps.sourceforge.net/trac/aria2/)

A third way to get information about it is to open a Terminal from **Applications/Accessories/Terminal** and type ```
man aria 
``` and if that shows no response try ```
man aria2
```

The man command is a great tool to get information about command and programs you have installed.

I hope this helps...
