### Post by SuperSonic4 on 2008-08-09
M$ changed the msn server recently
I'd imagine Kopete comes with a lot of KDE dependencies.

If you want an MSN client only I'd try emesene or amsn

```
sudo apt-get install emesene
```
```
sudo apt-get install amsn
```

emesene seems to be having problems displaying the contacts list and I suggest getting amsn from their website: [http://www.amsn-project.net/linux-downloads.php](http://www.amsn-project.net/linux-downloads.php)
since the new version can connect
