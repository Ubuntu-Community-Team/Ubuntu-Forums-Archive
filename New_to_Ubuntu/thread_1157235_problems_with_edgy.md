### Post by Dynaflow on 2009-05-12
> **Noobster666 said:**
> i downloaded it but it wont install,  i actually got that first but nothing was doing with it

Download the file (it'll have an ISO file extension).  You then have to burn it to a CD, but *not* as just data.  ISOs are disk images (an exact copy of the physical disk), and so you need to use the option in your CD-burning program under "burn disk image" or similar verbiage.  Otherwise, you won't have a bootable (or otherwise useful) CD.
