### Post by Jim_in_Germany on 2009-12-04
I use Scite as my editor of choice for small(ish) ruby scripts. 
It is very configurable, has syntax highlighting and you can launch your program from within it (by pressing F5).
For making a ruby GUI your best bet would be FXRuby. It is relatively easy to install and is well documented.
If you have a more exact question regarding installation or whatever, then please post back here.
