### Post by starcraft.man on 2009-06-27
> **TheStabe said:**
> I downloaded 70717-BurntOrangeIce.zip, and inside it is a .tar.gz folder and an .emerald file.

System > Preferences > Appearance. Once the menu opens, just drag and drop the tar.gz file onto the main window.

The emerald file is more complex, that is a replacement window manager that displaces metacity (the default one). I'd suggest you google emerald and read up on it. It isn't essential, the tar.gz is the bulk of the theme just stick with that for now.
