### Post by SeijiSensei on 2021-06-06
In KDE Konsole you can set the font in Settings > Edit Current Profile > Appearance. I prefer black on light yellow for readability.

You can also change the font size on the fly by holding down the Ctrl key and using the scroll button on the mouse.

### Post by grahammechanical on 2021-06-06
If you are using Ubuntu with the Gnome shell and Gnome terminal - open the terminal and click the icon of the three horizontal lines. Scroll down to Preferences. In the preferences dialog select Unnamed. Then tick Custom font. That will allow you change the font size of the default monospace regular font from 12 to whatever you want. Click on 12 and at the bottom of that dialog there is a slider to change the font size or you can used the minus ( - ) and plus ( + ) buttons.
