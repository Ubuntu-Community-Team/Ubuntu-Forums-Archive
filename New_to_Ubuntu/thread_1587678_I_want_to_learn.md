### Post by slooksterpsv on 2010-10-04
Learn both at the same time =D. I'd start with Python as it's an easy language, and helps to understand some complexities that are in other languages like polymorphism, inheritance, etc. (Don't let the big words stray you like they did me at first, they are important and make your life easier as a programmer)

### Post by slooksterpsv on 2010-10-04
If you click on Applications -> Accessories -> Terminal - when terminal opens you can either type in:
python
and start trying out python. Or you can go to Applications -> Accessories -> gedit
Type in the text I gave you above, click on File -> Save, save it somewhere (e.g. Documents folder would work), save it as: <whatever>.py - e.g. so firstprog.py

Let's use the above information, here's how you'd run it:
Applications -> Accessories -> Terminal
Type in the following (if we used the above name and save location):
```

cd Documents
python ./firstprog.py

```

That will run the program - you can use other items such as Eclipse to help write the code and that.

### Post by Zero++ on 2010-10-04
If you want to learn how to build computers buy a cheap one off of ebay or craigs list and take it apart. You might also want to grab a book on compute repair or A+ certification once you get your feet wet. A great web resource is PC Wiz Kid's Tech talk. He has great videos on all of the operating systems and some hardware stuff too!
