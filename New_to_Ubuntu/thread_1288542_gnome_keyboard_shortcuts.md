### Post by mprince on 2009-10-11
If Opera is your default browser, then you should be able to set it up using:

*System->Preferences->Keyboard Shortcuts*

Look for "Launch web browser" and change the shortcut.

(I'm using 8.04, so I guess this could have changed.)
