### Post by sinclair86 on 2008-10-11
Well im guessing you are trying to a fresh install? if so go here ```
http://wine.budgetdedicated.com/archive/index.html
``` that a deb package so all you have to do is download it put it to the usb drop it on the other computer and double click it.

if you want to copy the config files from one computer to another they are in the .wine folder. crtl+h its a hidden file in your home directory.
