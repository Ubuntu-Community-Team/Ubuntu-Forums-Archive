### Post by all2ez on 2008-09-03
The above links look great and should be very helpful to you.  It is also worth noting, in case you don't know, that you can learn a lot about most commands by reading their manual pages by entering something like:

```
man somecommand
```

where somecommand is the command you need information about.  for example,

```
man rm
```

will open the manual for the command to remove/delete files.   This shows all of the options associated with the command and some tips about how to use it.
