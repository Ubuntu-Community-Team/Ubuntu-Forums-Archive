### Post by Frogs Hair on 2011-06-02
If you don't already have a proprietary driver , type additional drivers into dash if using Unity or search for it on the administration menu in Classic Gnome.

When the program opens it will search for drivers , if one is located you will be able to install it  . Choose the recommended driver and restart after installation  . If no driver is found there is no current driver for your graphics card in Ubuntu other then the one in use .
