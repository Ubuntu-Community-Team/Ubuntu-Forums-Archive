### Post by andrewmv on 2008-12-26
*Handbrake* is definitely the easiest to use, and though I don't believe it's included in the Ubuntu software repository, there is an Ubuntu package on the website, so it should be pretty easy to install:

[http://handbrake.fr]("http://handbrake.fr")

The commandline tools *transcode* and *mencoder* should also be able to do the trick, but they're quite a bit more involved to use.

### Post by lhb1142 on 2008-12-26
The one I use is Winff [also known as Video Converter (Winff)]. You can find it in the Synaptics Package Manager if you have the Medibuntu and Launchpad repositories enabled (along with Multiverse and Universe).

Winff is an excellent program.

In addition, if you are interested, a great program for converting these mpeg and avi files to files suitable for burning to a DVD is DeVeDe. While this is also found in the Synaptics Pacakage Manager, the program there is an older version. The latest version can be found here:

< [http://www.getdeb.net/search.php?keywords=Devede](http://www.getdeb.net/search.php?keywords=Devede) >

assuming you are using Hardy; I do not believe it will work in Intrepid as yet.
