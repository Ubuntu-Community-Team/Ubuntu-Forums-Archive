### Post by germanix on 2010-12-26
So this is how you do it:

Go to keyboard - keyboard preferences, then to the tab "layouts". Then click on the tab "options". Once there find the listing "Key to choose third level". Select a key from the list that you want to use to get that symbol. Thats it.
PLEASE NOTE: WHATEVER KEY YOU CHOOSE, YOU HAVE TO HOLD IT DOWN AND THEN HIT THE ~ KEY TWICE FOR THE SYMBOL TO APPEAR!
