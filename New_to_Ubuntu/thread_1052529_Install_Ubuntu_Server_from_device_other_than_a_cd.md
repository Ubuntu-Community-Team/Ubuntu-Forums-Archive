### Post by anjilslaire on 2009-01-27
If you have another installation of Ubuntu running on a different PC, you can turn the CD into a bootable usb stick with the Create a USB Startup disk in Intrepid, or look up [unetbootin]("http://sourceforge.net/project/showfiles.php?group_id=222386") for Windows/linux
