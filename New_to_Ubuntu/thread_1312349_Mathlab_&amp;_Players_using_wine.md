### Post by lavinog on 2009-11-03
[https://help.ubuntu.com/community/MATLAB](https://help.ubuntu.com/community/MATLAB)
You shouldn't need to run matlab in wine.

You can also save yourself a license by using octave (a matlab equivalent)
The problem you may run into is that netbooks are not very powerful, so don't expect it to be quick.

Also make sure you have plenty of space if you are using matlab.  Since you have xp on your netbook, you should be fine...the netbooks with ubuntu preinstalled tend to have limited space.
