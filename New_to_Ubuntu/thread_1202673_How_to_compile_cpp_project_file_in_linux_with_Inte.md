### Post by QIII on 2009-07-02
To be honest, I'd recommend something like EMACS to begin with instead.  There is a lot of documentation on it, you can get all sorts of books about it (I recommend [www.powells.com](www.powells.com), which is located here in Portland), it is very common and your experience will be portable.

If not EMACS, I'm sure someone can recommend others.

EMACS can be downloaded from the repositories, as can all of the compilers for supported languages.

EMACS is configurable into a passable IDE, but I prefer Kdevelop and Sun's Netbeans IDE.
