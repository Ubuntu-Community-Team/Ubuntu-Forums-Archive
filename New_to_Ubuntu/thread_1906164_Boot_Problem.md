### Post by Rex Bouwense on 2012-01-08
Welcome to the forums.  Lets approach it from a different angle.  When you first turn on your computer with the Ubuntu CD already inserted and the Dell Splash comes up, press F2 (I believe it is F2 on Dells).  That will get you to your BIOS.  Maneuver to Boot (or Boot Order).  Change the first boot device to your CD Drive (whatever it is called).  Save your changes and back out. The boot should be from the CD Drive if there is a bootable CD inserted.  Since you have already booted with this CD, we can assume that it is bootable and that you have a good burn (at least for now.)
