### Post by Chronon on 2010-11-07
Songbird used to be available for Linux (there might still be nightly builds available).  But the project has decided not to support Linux in the future so it has been forked.  Nightingale is the project that hopes to continue developing the Songbird code base for Linux.  It doesn't seem like there's much going on there yet, though.

[http://getnightingale.com/](http://getnightingale.com/)
