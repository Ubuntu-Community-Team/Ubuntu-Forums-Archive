### Post by HotShotDJ on 2008-06-01
Nearly any brand will work because the USB Mass Storage protocol is standardized across platforms.  In order for a USB harddrive not to work with Linux, it would have had to be designed specifically for the purpose of breaking compatibility -- and shockingly, there are some that have done just that (Some Seagate Free Agent drives, and nearly all USB sticks with U3 software).
