### Post by sudodus on 2015-12-20
It depends on your computer. Some new computers have UEFI systems that are tweaked in order to make it hard to boot anything else than Windows. But it works in most computers to create a dual boot system (with Ubuntu and Windows). Please tell us about your computer, and it will be easier to help you,

- Brand name and model
- CPU
- RAM (size)
- graphics chip/card
- wifi chip/card

### Post by Rocky_Bennett on 2015-12-23
While you are still in Windows, go to your disk management tool and use the partition wizard to create a new sector on your hard drive. Before I set up my Ubuntu I did this and it made the whole installation so much easier. I set aside about 240 GB for Ubuntu, and then I made sure that I disabled fast start in my BIOS and the I inserted the Ubuntu DVD and restarted, and it all went real smoothly. Right now I am triple booting Ubuntu with Linux mint 17.3 and the Redstone version of Windows 10.

### Post by Image on 2015-12-23
Need to determine whether or not you are using UEFI as sudodus already mentioned. I installed it using a usb drive which is a bit faster than a cd.
Partitioning your HDD is key to a successful dual boot. I think I had to format the usb drive to fat32 for UEFI to recognize it at boot.

Let us know how it goes and good luck!
