### Post by jimreynold2nd on 2008-11-21
Hi Emio,

First thig first, welcome to Ubuntu!!! As I can see, you are a Windows user.prepare for some big change! And I mean **changes**. Look at the article in my signature.

To your problem, like Kosimo said, Pidgin is really good. And it's preinstalled.
   *pros*: Tabbed chat window, lots of protocol (Yahoo, AIM, MSN, GoogleTalk, ICQ, ...), lots of plugins that does things Y!IM never can, small, efficient and fast, ads-free (:P).
   *cons*: No IMvironment, no Voice, no Webcam, send file works 80% of the times, Doodle works 50% of the times.

Installing Y!IM via Wine is not recommended at this stage, considering you just moved in :).

Have fun!

### Post by steveneddy on 2008-11-21
Pidgen is actually a pretty good application for IM, although it lacks a few of the features commonly available on the Windows counterpart.

Pidgen can be found at

Applications --> Internet --> Pidgen Internet Messenger

Pidgen allows you to have multiple IM contacts on one messenger, all at the same time.

Like Yahoo!, AOL and ICQ....all on the same messenger and all in the same window.

### Post by steveneddy on 2008-12-14
The .exe file is for Windows PC's.

The Linux version of Yahoo! Messenger is out dated.

I suggest using Pidgin which should already be installed. It can be found by going to

Applications --> Internet --> Pidgin Internet Messenger

Great text messenger. No video or cute extras. Perfect for communication worldwide.

If you need video chat, you may consider using Skype. ATM it is superior to most video chat applications available for Linux presently.
