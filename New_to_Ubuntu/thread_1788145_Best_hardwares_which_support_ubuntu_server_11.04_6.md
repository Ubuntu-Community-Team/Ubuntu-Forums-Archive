### Post by cariboo on 2011-06-25
You can use almost any working system to install the Ubuntu server, I have it running on three different systems:

Server 1 Intel E5400, with 2GiB ram and 8 harddrives totaling 4TiB storage
Server 2 Intel P4 with 768MiB ram and an 80GiB hard drive
Server 3 Apple G4 with 128MiB ram and a 10GiB hard drive

The reason the server version can run on such a wide range of hardware, is that there is no gui in the default installation
