> **jcwerth said:**

I can't locate or work out how to make appear the main(top) menu bar - the one that has applications, system, places. 

That's the standard top panel on a Gnome desktop. However, what you have:

>  I have a panel running down the side....but that seems to be an additional menu.

...is the Unity interface. If you want to switch to regular Gnome then log out, and from the session menu at the bottom of the screen pick "Gnome" when you log in.

### Post by grahammechanical on 2011-04-01
Most of us are not familiar with the Unity interface. We will get more experience when we upgrade to 11.04 in a few weeks time. By then you should be the expert.

You should still be able to see the network manager icon on the top panel. It should be over on the right. If unity follows the style of Ubuntu studio (and I do not know if it does) then try clicking on the Ubuntu icon on the top left. My guess is that you will see a drop down menu that has lines for System as well as Applications and Places. If you select System a menu should slide out with Preferences and Administration on it. Then you can get to Additional drivers.

This is my guess based on how things work in Ubuntu Studio.
