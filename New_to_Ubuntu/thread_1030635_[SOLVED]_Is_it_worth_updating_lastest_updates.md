### Post by jimmy the saint on 2009-01-04
Unless you have added third party repos it is recommended to install all the updates.  Remember, the updates aren't arbitrarily added.  They are tested, retested, retested again and finally apporoved.  Often they address serious bugs that are reported or recent security alerts.  Not updating can leave you exposed to security holes as well as known bugs that are fixed for everyone else.... because they updated.

If so go to System > Administration > Software Sources and under the Updates tab make sure it looks like this:

Of course yours will say Hardy where mine says Intrepid. But the rest of the updates are generally safe and sometimes necessary for security reasons.
