Sure you can. There are at least two ways:

a) Install the 64bit version first, then run the 32-bit version in VirtualBox

b) Install each version to different partitions; you can share the swap partition though.

### Post by lukjad on 2009-03-20
It should be noted that while the 32 bit version of Ubuntu will work on today's 64 bit computers, in the future they will be incompatible with one another. Also, while the 32 bit Ubuntu can be installed on the 64 bit OS, the reverse it not true (that is to say that you can't install the 64 bit OS on a 32 bit machine).
