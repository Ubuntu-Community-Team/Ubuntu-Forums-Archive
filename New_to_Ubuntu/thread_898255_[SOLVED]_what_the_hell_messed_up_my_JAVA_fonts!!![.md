### Post by karatefish on 2008-08-23
I just worked out the same problem - check the language you've got set in your Interface settings. I had mine on Armenian, for some reason, once I changed it to English, all the items went back to normal.

Tools->Options, then open the Interface subsection, and click the Language options area.
