### Post by deadflowr on 2014-02-16
You can access the software sources through the Ubuntu Software Center also, if you don't have synaptic installed.
Edit > software sources > other software.

Newer releases such as Ubuntu 13.10 are even simpler, just open the program "software and updates".
(It's just a name thing, it's still software sources)
