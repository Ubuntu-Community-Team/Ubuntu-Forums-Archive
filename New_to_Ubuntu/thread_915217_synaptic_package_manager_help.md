### Post by Cheesehead on 2008-09-09
It also depends on *what* you added. Some packages are not applications, but are libraries (libfoo*) or plug-ins (foo-ad-blocker) or back-ends (foo-common) or automatic (foo-notifier). Also, many ordinary applications are command-line only, so no menu entry would be made.

If you added an app you're not familiar with, check the manpage to see if it's command line only.

If you tell us what you added, we can probably tell you the problem, too.
