### Post by the_doc on 2008-06-15
I like opera too, it's a good browser.  Assuming you have the .deb package for the latest version downloaded, then just right-click on it and select the first option **Open with GDebi Package Installer**, and the rest should happen automatically.  

If it complains about conflicts you may need to uninstall the old version first, that's happened to me a few times.  After that close and re-open Opera and you should have the latest version.

[http://www.opera.com/download/](http://www.opera.com/download/)

The file I got was "opera_9.50.etch-qt3_i386.deb".  All you have to do to install a .deb file is double click on it in nautilus. 

If your are upgrading from a version prior to 9.5 all your settings, bookmarks etc. will be preserved.

I have been using the weekly development snapshots since 9.5 beta came out.
