### Post by Bölvaður on 2009-03-15
the default one that comes with gnome now is quite good. [gnome-network-manager]("http://live.gnome.org/NetworkManager"). On my laptop I use tiling window manager so I need to activate my script to connect to a wireless network manually. But when I log into gnome it is done automatically, but takes longer time than with my script.


Im sorry if I cannot say anything else than "it is there".


You will probably be asked to make a keyring password after making a new connection. And after that the network manager looks first for connections you have already been on and ask you for the keyring password to connect.
