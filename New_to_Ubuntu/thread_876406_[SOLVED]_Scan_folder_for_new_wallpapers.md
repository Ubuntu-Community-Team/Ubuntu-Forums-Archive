### Post by aheckler on 2008-08-02
The folder Ubuntu uses for the pre-installed backgrounds is /usr/share/backgrounds/. So if you copy your wallpapers to that location, you should be able to see them through the right-click on the Desktop method.

EDIT: However, I'm not sure how you'd go about getting Ubuntu to use a different folder for the default location.
