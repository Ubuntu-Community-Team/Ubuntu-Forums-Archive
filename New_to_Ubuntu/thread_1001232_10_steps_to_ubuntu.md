### Post by threadbaredog on 2008-12-03
ubuntu live - try with no commitment

1. Go to the Ubuntu site: [www.ubuntu.com](www.ubuntu.com)

2. Download Ubuntu 8.04 (currently the latest stable version - says "ideal for large deployments" - ignore - that's the recommended stable version) and save it on the disk. It's a ".iso" file which means it's a "disk image" file.

3. Burn it on to a disk. Make sure you make it a "bootable", "system" disk. Your CD burner will have an option like "make a disk image" - use that. I made 2 un-bootable disks before I figured this out... ALSO IMPORTANT! MAKE SURE YOU WRITE AT A LOW SPEED FOR ACCURACY! (If you make a system disk like you're supposed to, your CD writer may do this for you automatically..) This took me another 2 disks! :-) Voila! You've got a disk with "Ubuntu Live" on it. (Believe it or not - this is a complete Operating System with everything you need to get going on it!) On the last disk in my box, I succeeded!

4. Stick in it your CD Rom player (it's probably in there already). Turn your PC off.

5. Now a slightly technical bit. DO NOT BE AFRAID! :-) You have to hold the <delete> key OR f8 OR the <shift> key OR...  (they tell you which on the screen but it disappears pretty quick.) whilst you start your computer. This gets you in to the little system set up thing built into the computer. What you want to do here is make sure that your computer boots from the CD before it boots from the Hard Drive. On my computer, I held down <delete>, then selected "Advanced Bios Features", then "Boot Sequence". You have to read the sceens (the instructions are all there, at the bottom of the screen probably) but they're pretty easy if you concentrate - and go with it. Fear not! :-) The basic rules are: you get a series of menus which you use with the up/down/enter/esc keys.  If you really want to, you can write down everything you do so that you can go back and change it back again afterwards but it's really not that scary after 5 minutes when you start to understand how it works.

6. When  you've done that, get out of the boot set-up thingy. Probably by pressing escape a few times. Save your changes if it asks you to confirm. You may have to re-start your PC here but probably not. Make sure your network cable is plugged in at this point - it allows Ubuntu to get information to configure itself.

7. Wait a bit and Ubuntu will appear on your screen. It's friendly! :-) Follow the instructions - select "try ubuntu without any change to your computer". Your computer just booted up the Ubuntu operating system. It's currently working from the CD rom so it's a bit slow sometimes. If you eject the disk and turn your computer off and on again, it will start Windows again. Don't worry.

8. Use Firefox. Visit your favorite sites. Then try the email program.  You won't be able to send your emails cos it needs setting up but you can pretend to write an email and check out the UI. It's really nice... :-)

9. If you wanted all this (for free!) plus a whole load of software for anything you'll ever need, you could install Ubuntu on your computer. (And keep Windows too if you wanted - a good way to do this is to get another hard-disk in your machine and install it on there, so it's completely separate from Windows.) Just follow the instructions when you choose "Install ubuntu"... if you want! :-)

10. You're there!.
