### Post by NightwishFan on 2011-03-26
Hold shift while booting until you see a menu listing Ubuntu kernels and a recovery mode. Boot into **at the latest** 2.6.32-28 (earlier is fine). Do not run this if you do not have a kernel that is .28 or earlier. When you are at your desktop run this command:
```
sudo apt-get remove linux-image-2.6.32-29-generic linux-image-2.6.32-30-generic
```

Then run:
```
sudo apt-get -f install && sudo apt-get upgrade
```

Then you should be set. Not sure what caused this in the first place.
