### Post by Rubi1200 on 2010-10-24
I don't see anything unusual, other than this:
> Grub 2 is installed in the MBR of /dev/sda and looks on the same drive in 
    partition #1 for (,msdos1)[COLOR=Red]/mnt/boot/grub[/COLOR]. It should say just /boot/grub

I recommend reinstalling GRUB using the following method from a LiveCD:

```
sudo mount /dev/sda1 /mnt
``````
sudo grub-install --root-directory=/mnt/ /dev/sda
```Don't forget to run

```
sudo update-grub
```once you are back in Ubuntu.

If this doesn't work, there is another method we can try.
```
