### Post by hyper_ch on 2008-08-18
if you can swap harddrives then

(1) use the harddrive that you want to install ubuntu to
(2) put it into a computer from which you can boot from cd
(3) unplug all the other harddrives in there (just to be sure)
(4) install ubuntu on it
(5) put the harddrive into your old computer
(6) most of the hardware change will be recognized... possibly some videocard errors and sound errors may appear
