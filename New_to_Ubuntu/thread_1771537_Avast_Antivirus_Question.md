### Post by Gone fishing on 2011-05-30
It is possible to run an on access scanner - I believe Kaspersky, Sophos and Antvir all do or did provide such solutions.

But why bother? I don't think there are any Linux viruses in the wild - anyone know different? have a look at:

[https://help.ubuntu.com/community/Linuxvirus](https://help.ubuntu.com/community/Linuxvirus)

It maybe worth having an AV if you share files with Windows users just to prevent them giving you infected files and you then passing them on. However, basically you are safe with Linux - not that we should be brainless and complacent and start executing any old script etc.
