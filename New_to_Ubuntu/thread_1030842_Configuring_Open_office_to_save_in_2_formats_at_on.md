From what I know having two files in different format is a bad practice and extra work/time to maintain. Similar to databases where you should keep information only once and not double up.

So maybe another approach for you is to think it though why you want to maintain two files with the same content but different formats. Maybe you find a simpler solution, e.g. the main file is ODF, on windows you can open ODF and then save to MS format, back on linux you would open the MS format and save as ODF again... just my 2ct.

*If you need to run with windows have you thought about using the ODF plugins?*
