### Post by snowpine on 2010-01-28
Your personal information can be recovered even if you delete it. The only safe way to protect your privacy is to use an application like [DBAN]("http://www.dban.org/") to completely wipe the drive, then reinstall Ubuntu.

Once you reinstall, I'd suggest setting up auto-login and typing up a quick how-to-change-username-and-password text document on the destkop. Unlike other distros (Fedora for example), Ubuntu **requires** you to set up a username and password during the install process.

Your friend is going to have to learn *something* about Ubuntu if he wants to have a positive experience. Linux is not Windows and all that.
