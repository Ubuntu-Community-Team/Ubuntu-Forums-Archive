### Post by anewguy on 2010-08-11
The version of VirtualBox available you install via Synaptic Package Manager is the open source edition (OSE).  You need to go to the Oracle/Sun site and download the free (but not open source) version called the PUEL version.  The OSE version doesn't support USB - the PUEL version does.
