### Post by donkyhotay on 2009-09-12
Easiest way to wipe everything is to simply put in the liveCD and do the same thing you did when wiping out vista. Instead of choosing the vista partition just choose the ubuntu partition. It'll wipe everything out and reisntall it. I recommend learning about what you did wrong this time before simply wiping it. It's better to try to repair problems then start over, but I do understand it's sometimes easier to start over. If you're experimenting it might be a good idea after reinstalling to create a VM for ubuntu and use that for 'experimenting' with. That way if you completely FUBAR you'll have only messed up the VM rather then the entire system.
