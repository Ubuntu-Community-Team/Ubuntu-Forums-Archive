### Post by CatKiller on 2011-08-26
Not with Optimus, it isn't. Don't install the proprietary drivers, they [won't work]("http://www.nvnews.net/vbulletin/showpost.php?p=2183477&postcount=2").

As I understand it, Optimus doesn't currently work that well, although there is a project called [Bumblebee]("https://launchpad.net/%7Emj-casalogic/+archive/bumblebee/") that's working on it. Some people have reported some success with it.
