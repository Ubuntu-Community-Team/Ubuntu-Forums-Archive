### Post by Paqman on 2010-09-27
That's naughty of them. 

So assuming you've extracted the archive to your desktop, open a terminal and enter:
```
cd Desktop
chmod +x ZendStudio-7.2.1-x86_64.bin
./ZendStudio-7.2.1-x86_64.bin
```

Not quite sure why there's two binary files in there though. If it doesn't work try the same with the other filename.

If it does work, you can create a menu item that will point to it so that you don't need to open a terminal to launch it.
