### Post by adobrakic on 2009-01-04
Going a little more in-depth from post above:

- Go to Menu -> Accesories -> Terminal
- Type in:
```

sudo apt-get install amsn --yes

```
- Enter your password (it won't show that you're typing it -- but it's there)

aMSN will appear in 
Menu -> Internet
