### Post by atomkarinca on 2008-07-12
You should first find a torrent to download. There are various websites for this, like [http://www.mininova.org](http://www.mininova.org) . You search something, find the torrent file you want, download it to your harddisk and point your torrent client to that file. You can associate the torrent client with .torrent files from your browser's preferences and avoid all those steps.

### Post by atomkarinca on 2008-07-12
As long as you won't do this to pirate something you can use Frostwire. Open up a terminal (Applications > Accessories > Terminal) and type:

```
sudo apt-get install frostwire
```

It should appear in your menu now.
