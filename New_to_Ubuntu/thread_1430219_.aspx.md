### Post by mcduck on 2010-03-15
> **sussexslanter said:**
> I see the page fine.  Near the bottom there are three download documents. There are download documents and one of them is a pdf.  When I press that link it asks me if I want to download it.  If I click download then all i get is an aspx file on the desktop which doesn't open.

That's a error on the server's end, not on your computer.

The server is supposed to read the .aspx file and generate normal HTML for your browser to view or to start the download of the file you requested. You are not supposed to get the actual .aspx file, and you really can't do anything with such file.

edit: Perhaps I should add that the only way to get the actual PDF file you tried to download is to contact that web site's maintainers and tell them about the problem & ask them to fix it or send the PDF file to you.
