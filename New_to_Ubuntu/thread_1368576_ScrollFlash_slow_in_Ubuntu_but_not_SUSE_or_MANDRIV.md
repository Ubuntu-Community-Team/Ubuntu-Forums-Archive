### Post by TBABill on 2009-12-30
Ok, here goes. I am a new Linux user as of a few weeks ago. I started with Karmic and it worked great in every aspect except for flash heavy websites (Facebook - Fishville, Farmville; flash game sites; websites like YouTube where lots of videos are listed to choose from). I tried the forums and found that many many users suffer the same problem from as far back as the LTS (possibly earlier than that). 

I wondered if it was just Ubuntu so I successfully TRIED (meaning the system installed and all drivers worked on my laptop) PCLinuxOS, Kubuntu, Fedora 12 KDE, Mint 8, Mepis (can't remember the others but I tried several others as well). All of these suffered identical problems with those same sites and apps.

Now today I have successfully installed OpenSUSE 11.2 and the problem is GONE! This is the only Linux version that, on my laptop, has outstanding performance with flash (seems better than Windows actually and all my processor temps are very low as well). The system seems perfect after tweaking drivers and some other minor issues most Linux distros suffer from machine to machine.

Question: What variant could exist to make those other systems NOT work well but for SUSE to scream on those same apps with the same machine. I did perform every available update and driver installation on each install. Is it KDE vs. Gnome? Is it the use of a different kernel? Something is causing a lot of frustration for many users, for years apparently only having workarounds to minimize the problem, but no solution offered.

I would offer a list of hardware on my system but I think that's irrelevant. Search the forums for various distros and there are lots of issues posted, and bugs reported, for the problem with flash and/or java slowing systems down, glitchy scrolling, cpu's running at or near 100%, etc.

I am only listing all of this because I loved Ubuntu and then the tweaks Mint offered, but I can't use a system where Internet apps most used in the household are hampered by the software.

Thoughts? Ideas? I am solely running SUSE now until I can confirm a fix is in place for Ubuntu to resolve it. SUSE is cool, but I really like how friendly Ubuntu (and Mint) is for new users.
