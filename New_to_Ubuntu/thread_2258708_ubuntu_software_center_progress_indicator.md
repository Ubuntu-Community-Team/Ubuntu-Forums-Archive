### Post by deadflowr on 2014-12-30
If you click on the progress indicator, it should open a window/move to a window that will show what's being updated/upgraded/installed.
If this is an on-going thing, then it's most likely indicative that you closed the software center, or shutdown the system before something you were trying to install was finished installing.
Do you remember the last thing you tried to install?

Another thing would be that it is a repository connection problem.
If you try to launch Software Updater, does it run ok, or do you can an error message; sometimes you will get a stop/warning icon in the top panel near the clock when the repositories are having issues.
