### Post by editrix on 2009-01-30
Which version of Scribus and which version of K/Ubuntu? I'm on 8.04 Kubuntu and have no problems with 1.3.3.11.

However, what I find is much easier is, I use Konqueror in icon view mode and go to my fonts directory--then they all show up as "previews." Or you can use KDE's font installer (if you're in KDE) as a previewer. The advantage is, you can't do anything in Scribus if the font previewer is open, so you're better off using another program to view them. There's probably some similar method in Gnome, if that's what you're using.
