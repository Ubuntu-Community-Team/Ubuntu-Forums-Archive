### Post by shifty_powers on 2008-05-09
well, joomla is designed to be installed directly to the webserver, so i wasn't entirely sure if it was appropriate, but it is a fantastic program.

it is actually a content management system, open source of course, that will allow you to use templates and a LONG list of plugins to create just about any site that you want. Used in the past, and it can be very powerful, but also quite easy to use. Depends what you want to do with it ;)

As for installing, you need ftp access to the server you're using, then you can configure it in a browser. see [http://help.joomla.org/content/category/15/99/132/](http://help.joomla.org/content/category/15/99/132/)

for an example site, visit [http://textualecstasy.com/](http://textualecstasy.com/) which a friend of mine did, with some help. have a look around ;) (colour scheme and templaet might not be to your liking, but there are LOADS of templates out there :D).

hope that gives you a bit of help ;)

edit: as for nvu, it is in the repositrories, so

```
sudo apt-get install nvu
```

in a terminal ;)

edit: ah ok, nvu is actually no longer maintained and has turned into kompozer funnily enough......
