### Post by fubbleskag on 2009-01-21
open firefox, open bookmarks, click Import/Backup, Import

browse to the firefox profile directory via your mounted windows drive, ie */windows/Documents and Settings/username/Application Data/Mozilla/Firefox/Profiles/oea9ih50.default*

find bookmarks.html
