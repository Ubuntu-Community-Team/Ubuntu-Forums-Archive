### Post by bobnutfield on 2008-05-12
OK, if you are running Windows XP, the tool you need is ntfs-3g which I believe is install by default, but if it isn't you can install it from Synaptics Package Manager.  It's called libntfs-3g23.  Once installed, you can just go to 

Places>Computer 

and click on the Windows partition.  From there, you should see your Pictures and Music folders.  Drag and drop is supported, so you shouldn't have any problem.

Bob
