### Post by em4r1z on 2008-11-03
> **LinuxFox said:**
> Ah, so it does apply to the repos and updates.If by 'repos' you mean newer versions, then you're wrong. You'll get security updates but your applications/libraries/kernel won't be as new as the ones present in a more recent release unless a fix requires the upgrade. Thus if you need a feature present in a newer application/kernel, you'll need to find a way to install it. For example by compiling a new kernel or using third party repositories to install a new version of an application, which might be troubling if it needs newer libraries than the ones installed and used by applications you don't plan to upgrade.

Honestly, I don't see the point of using the same set of tools for more than a year in a desktop environment for at the rate things evolve nowadays you'll be missing lots of useful features. An organization environment or a server it's a completely different story.
