### Post by rampageoberon on 2008-11-17
If you are behind a router you will have to open up port 80 for normal http traffic to your server. And you might want to give yourself a dynamic dns address.

have a look at [www.portforward.com](www.portforward.com) and [www.dyndns.org](www.dyndns.org)

### Post by louieb on 2008-11-17
Check out [DynDns.com]("http://www.dyndns.com/").  Its a free service.  If your router is like mine it can be set up to use your account. If not then you can use package **ddclient** to use your DynDns account.  [Lou's Home Server Stuff]("http://louieb.isa-geek.net/")
