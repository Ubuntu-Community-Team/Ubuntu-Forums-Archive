### Post by Larkspur on 2011-05-16
For an old-style menu, you can either log into Gnome Classic or, within Unity, run the command "gnome-panel" and also add it to Startup Applications.

For program menus, you can either uninstall the package or press F10 to open the first menu and navigate using the cursor keys.  Alt+[shortcut] also works (e.g. Alt+F for the "File" Menu).
