### Post by hanzj on 2009-03-31
Hello, 
I learned through a [post]("http://ubuntuforums.org/showpost.php?p=2126653&postcount=3") that we can scan for wireless networks by typing in the terminal:
```
sudo iwlist scan
```That command works, but newbies like me wonder whether there is a GUI way of doing this.  I prefer not to install any additional apps. Can the default-installed NetworkManager Applet scan for wireless network? According to [https://help.ubuntu.com/community/WifiDocs/WiFiHowTo#line-292](https://help.ubuntu.com/community/WifiDocs/WiFiHowTo#line-292) , NetworkManager *can *do GUI switching of networks.


[COLOR=Magenta]**UPDATE: I learned that I should "left-click" the icon in order to bring up the available wifi networks. Problem solved.**[/COLOR]
