### Post by Guille Damke on 2009-01-04
Hi,

I have used both versions, now in 8.10 (both 64 bit), which I prefered to install instead of upgrading. I think there is a big improvement in this last version. I have not had Java or Flash errors, now we can even install Flash for 64 bits GNU/Linux OS. For example, I really like the Network manager in the last version, it had a big improvement, I have the feeling of being using a more mature system under 8.10.
I think for switching back from 8.10 to 8.04 you have to do a clean install, that's what I did for a short test some days ago. I would recommend you to have your home directory in a separate partition, in that way you don't have to back up your personal data (remember not to format this partition, and use the manual option). Also, before reinstalling, I moved my files to a new directory in the home (not MY $HOME), because I had some small troubles when 8.10 used the previous configuration files already existent in my $HOME created under 8.04.
Good luck.
Guillermo.
