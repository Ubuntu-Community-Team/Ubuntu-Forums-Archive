### Post by Sarah L on 2008-05-24
Check to make sure that you have the correct installation disc for your processor. If you have an Intel CPU, you'll want the i386 version. If you have another type, such as AMD, then you'll need a different disc. Also, if your CPU doesn't support 64-bit architecture, then you can't use a 64-bit version of Ubuntu.
