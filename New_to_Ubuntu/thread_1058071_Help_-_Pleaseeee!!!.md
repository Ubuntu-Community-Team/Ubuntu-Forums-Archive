### Post by oldsoundguy on 2009-02-02
Once again, you do not need any of the things that McAffee offers or Norton or whatever to use any form of Linux.  Because of the nature of the install procedure utilized by the system, even installing something that may have been written to TRY and do something on Linux, is virtually impossible (nothing is impossible .. but thus far no one has been able to write a program that will SLIP BY and install.)
The only item suggested is that if you are running uncovered (not behind a network firewall), that the resident firewall program in Linux be activated.

As to Evolution .. you can remove that program and install the Mozilla program(s) such as Firefox browser and Thunderbird E mail as they are a lot more "look and feel" of the Windows counterparts.  Not saying better or worse, just easier for someone coming from Windows to grasp as an initial experience.
