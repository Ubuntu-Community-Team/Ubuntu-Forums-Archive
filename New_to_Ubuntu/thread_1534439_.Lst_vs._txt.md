### Post by SlidingHorn on 2010-07-19
when it comes to linux based systems, text files and scripts have a lot of different extensions.  .lst and .conf are usually configuration files of some sort.  They can easily be edited by any text editor like gedit, leafpad, mousepad, abiword, etc. (so be very careful when working with them!)  You won't find any .txt files on an Ubuntu system unless you put one there yourself.  

It's one of the many differences between Windows & Linux.  Plain text files in linux don't even really need an extension (ex: .txt).  If you plan on having Windows users view the text file, however, you would want to save it as a .txt
