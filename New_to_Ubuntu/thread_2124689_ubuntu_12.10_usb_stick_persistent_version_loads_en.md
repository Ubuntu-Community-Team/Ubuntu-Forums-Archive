### Post by I2k4 on 2013-03-16
Somebody might have a technical suggestion.  I'd say create another stick with another version of Ubuntu or Mint and start over - possibly the LTS version 12.04.  I've had several Ubuntu and spin-off versions that simply failed to find the hard drive on my machines consistently.

I've been using Persistent Live USB for a couple of years on my main machine, and my experience is they always go wonky in some way after a couple of months if used regularly and simply have to be reinstalled periodically.  I haven't had problems installing third party software.  My wisdom about making them work a while longer is never do any system updates, never install more than one thing (with dependencies) at a time.  The USB is slower than a hard drive and too much demand overloads the memory.  I've also found system cleaning with Bleachbit helps unclog the USB.
