### Post by moswell on 2010-05-13
Hi all.  I am, as implied by my posting this in the "absolute beginner" forum, a newbie to Linux and Ubuntu.  I just bought a new little Asus 1201T netbook, which came with no OS, and thought I'd give Ubuntu a try for fun.  I've been learning quite a lot about the system and like most of it so far, but I'm a little disappointed because I can't get the visual effects to work, so I haven't been able to try out multiple desktops or moving screens around, etc.

Like I said, I've got an Asus 1201T netbook, with 2GB of RAM, and an ATI Radeon HD 3200 graphics card. I installed Ubuntu 10.04 (the netbook version). The reading I've done suggests that the ATI card is troublesome with Ubuntu, and none of the solutions I've read are working for me.

I managed to get the "visual effects" screen to let me choose between none, normal, extra, and custom, but unfortunately if I choose anything other than "none" my screen goes all wonky and nothing works.

I've tried using both (what I think) are the open source drivers and the ATI driver that the system recommends installing.  Neither changes the screen problems I get when I try to use visual effects.  In fact, I think the graphics response time is pretty bad when I use the ATI drivers.

It could be that this graphics card (or this graphics card plus the amount of memory I have) simply isn't enough to run the visual effects.  If this is the case, is there anything I can do to play around with window switching or any of these features I didn't have in windows?

Thanks a lot for your time.

Argh, never mind.  I figured it out myself - I just needed to lower the screen resolution and now I can play with all sorts of fancy things.  I'm sure I'll have other questions eventually since I want to install a virtual machine with windows!
