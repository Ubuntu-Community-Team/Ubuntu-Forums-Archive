### Post by col48 on 2014-02-08
Very surprising you couldn't run the Live CD - given it doesn't take much effort, I think it would be worth trying 12.04 on the computer, especially as 10.04 has run out of full support.

**With any major OS changes it would be wise to backup all your files first.**

If you have enough disk space to dual-boot, (triple-boot if you want to keep XP) you could copy over all your files from the 10.04 partition to the 12.04 one with no problem.  I think with settings, you'd usually be better advised to do from scratch.

If you don't dual-boot, you have two options to get 12.04 given 10.04; either install over 10.04, which would wipe the partition, or upgrade 10.04 to 12.04, which would not.  In the second case, your files *should* be intact, settings maybe not; experience with upgrades is very varied - some people have no problems at all but others (perhaps those with more complicated/more unusual setups??) have all kinds of issues - lots of forum threads tell the story.
