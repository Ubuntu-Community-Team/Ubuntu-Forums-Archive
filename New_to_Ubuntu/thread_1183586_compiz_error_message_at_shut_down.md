### Post by lovinglinux on 2009-06-10
If you are using a dock applet like awn/cairo or screelets, then this alert is probably one that says you need compositing to proper render these applications. This happens because compiz is killed before them, so the alert kicks in during the shut down process. Nothing to worry about.
