### Post by conversationjames on 2014-01-02
Haven't tested but read about an Ubuntu Dictionary lens...
- Open terminal (CTRL+ALT+T)
- copy the line "sudo add-apt-repository ppa:scopes-packagers/ppa" (without 's), right click in terminal, paste, press ENTER
- copy the line "sudo apt-get update && sudo apt-get install -y unity-dictionary-lens" (without 's) right click PASTE, press ENTER
- type Exit and press ENTER or just close with X

Restart session and it should work!
