### Post by thatguruguy on 2010-01-27
you don't need ccleaner (since there is no registry) or defraggler (since ext3 or ext4 drives don't fragment the way that ntfs drives do).

Spyware and viruses written for windows won't run on linux boxes. For the most part, the only way to get malware on ubuntu is to install it yourself.
