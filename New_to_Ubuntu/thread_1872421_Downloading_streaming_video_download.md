### Post by ajgreeny on 2011-10-30
What videos, and from where?  You might be able to right click on the url and click on "save link as", but it depends on the stream properties.  For some flash and other format videos and animations you can use the firefox add-on DownloadHelper, but I don't use chromium much so don't know about doing the same using that browser.
