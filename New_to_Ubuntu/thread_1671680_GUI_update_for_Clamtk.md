### Post by judedawson on 2011-01-20
Hi there,
I'm sure there are several ways. Here's how I do it. 

First, after checking from the ClamTK Virus Scanner app that it needs an update, I close the ClamTK Virus Scanner app. 

I go to the ClamTK website ([http://clamtk.sf.net](http://clamtk.sf.net)).

In the website, I select 'Downloads'. This takes me to a Sourceforge website with the option to download the latest Debian/Ubuntu version of ClamTK.

I click on 'Download now' and select to save the file to the desktop (or download folder). Once the download is completed, I double-click on the 'package' icon and follow the instructions. 

After the update, I can open the ClamTK Virus Scanner app again.

Instructions on alternate methods are welcome.
