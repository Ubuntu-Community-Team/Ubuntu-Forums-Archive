### Post by techno-mole on 2008-04-28
1. Where can i find an easy to follow tutorial on how to make programs? Even if it's just a hello world program.

- i cant help you with this, but i would look for tutorials using python, some one else may anser this with more info.


2. Where can i find an easy to follow tutorial on how to change the system with basic changes that i might want to make?

- it depends on what you want to change, but for the most part you can do things in a similar way as you would with windows, for - example, you can go to - settings - preferences - screensaver to select a screen saver, and the time the system has to be idle before it activates, you can also set the power saving options as well.
have a look through the menu, there are options for setting system sounds, login windows, all sorts (and you dont have to input lines of code, most things have a gui set up)

3. Is there an easier option? I really don't want to have to memorize code which i have to type into the console when i want to open a file and when i want to install stuff.


- if you have just come from using a windows system, then you are going to have to learn a new way of doing things (as you would with most things) as for installing stuff and the general usage of the system its easy ( i very rarely use the command lines for day to day stuff, its main when ive broken something or when im experimenting :-))

there are a couple of ways to install stuff, you can go to synaptic and search for the application/s or you can go to add/remove programs and install what you need.

for other things there are usually packages you can download and install, if you find a file that ends in .deb its as simple as clicking it and of it goes (like an .exe in windows) for things that are a little more complicated there are ways of converting them to be less complicated ( i refer to using alien to convert certain archives to .debs)

my advice is to read the wikis for ubuntu, and if you get stuck ask, ive spent a lot of time using forums on various sites, but this one is by far the best.
