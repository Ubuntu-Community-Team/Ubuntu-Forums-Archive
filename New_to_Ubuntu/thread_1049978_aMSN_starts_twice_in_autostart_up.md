### Post by sylvainrb on 2009-01-25
When you log off uncheck the 'Save session for future logins' check box.

The reason it does start twice is when the session is saved, I assume that you have aMSN already running. The sessions is then saved with it running and when logging in again the autostart doesn't seem to notice it running already and starts a new one.
