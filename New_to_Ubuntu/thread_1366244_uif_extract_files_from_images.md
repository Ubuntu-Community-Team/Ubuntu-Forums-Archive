### Post by django0909 on 2009-12-28
My suggestions are:

Use the trial version of MagicISO, (which should run in wine), to extract the stuff you need if the uif is less than 300mb

If it's more than 300mb, you could use a program called uif2iso, to convert the file to an iso. This iso file can then be mounted and the files read - use the following link for instructions on this:

[http://www.ubuntugeek.com/mount-and-unmout-iso-images-without-burning-them.html](http://www.ubuntugeek.com/mount-and-unmout-iso-images-without-burning-them.html)

The problem with uif is that it is a propietary(sp?) format - so it can't be opened with any program, it needs to be something specific. You might have more luck converting to iso regardless.
