### Post by da burger on 2010-09-05
You need a CPU in a physical machine (like the one your probably sitting in front of right now). A virtual machine is a piece of software that you can install on a physical machine. It allow you to have a kind of second computer. So you would boot up your physical machine and be able to use that. And then once the physical machine is booted up you could boot up your virtual machine which would have it's own operating system.
