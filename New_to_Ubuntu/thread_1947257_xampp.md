### Post by Lars Noodén on 2012-03-26
You'll find that Ubuntu has a built-in ftp client, [ftp](http://manpages.ubuntu.com/manpages/oneiric/en/man3/ftp.3tcl.html), available from the terminal.  You can also add a more advanced client like ncftp or a graphical client like FileZilla. You can find ncftp and FileZilla in the software repository which you can get to via the Software Center, Synaptic or apt-get.  

In general it is highly recommended to use the software repository and avoid downloading programs from random web sites.  The latter *will* eventually make a mess of your system.  The managed packages, however, will keep the system tidy, secure and well-running.
