### Post by Dedoimedo on 2008-11-17
Hello,

You can manually add sections to the file.

If you've already used Ubuntu on the machine, if you have a copy of the xorg.conf file, you can use it.

Manually:

Backup the file first!

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf-backup

Open the file in text editor as sudo (vi, gedit, what you like).

Then add to the Screen section:

Section "Screen"
 Identifier "Default Screen"
 Monitor "Configured Monitor"
 Device "Configured Video Device"
 Defaultdepth 24
 Modes "1152x864 1024x768"
EndSection

Make sure you list all modes you want ...

Make sure the identifier, monitor and device are the ones you want. In my example above, the terms are generic.

Save, restart the session (ctrl alt backspace). Change resolution, if it already has not been changed.

It might not work if your monitor / graphic card do not support the selected resolution.

Hope this helps.

Dedoimedo
