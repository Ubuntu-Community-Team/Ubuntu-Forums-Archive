### Post by grahammechanical on 2015-12-07
> I'm just a regular teenager

I have news for you. Even when you are my age (67 years) you will notice that it is still possible to learn new things. It does not take long for a different way of doing things to become the usual way of doing things. Experience of life has taught me that.

Programs written for the Microsoft operating system will not run on the Linux operating system. Fact of life. There is something called Wine that will make it possible for some Windows programs to run on Linux but it has a very variable success rate and that is because both Windows & the programs are proprietary software. The Wine developers do not have access to the code of the program.

[https://www.winehq.org/](https://www.winehq.org/)

All computer operating systems do the same things. They have different ways of accessing the functions. That is all. The default Office suite in Ubuntu (Libreoffice) does the same things as any other office suite. And we can click icons, select options from drop down menus or use keyboard short cuts. Where is the difference?

FOSS = Free & Open Source Software. The fundamental  philosophy of Linux and Linux distributions.
Canonical = The commercial company that supports the development of Ubuntu.
Ubuntu Community = Anyone who contributes in any way with Ubuntu. It includes volunteers as well as Canonical employees and covers a wide area of usefulness.

[http://community.ubuntu.com/contribute/find-a-task/#!/toplevel/advocate](http://community.ubuntu.com/contribute/find-a-task/#!/toplevel/advocate)

UOS = Ubuntu Online Summit which anyone can attend and discuss the future development of Ubuntu

[http://summit.ubuntu.com/uos-1511/](http://summit.ubuntu.com/uos-1511/)

Ubuntu On Air = Where we can access video Question & Answer sessions as well as tutorials.

[https://www.youtube.com/user/UbuntuOnAir](https://www.youtube.com/user/UbuntuOnAir)
