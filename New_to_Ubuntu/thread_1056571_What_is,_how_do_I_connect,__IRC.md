### Post by Rolcol on 2009-01-31
There are different clients for this.  If you don't want to download a separate client, you can use Mibbit.  [https://www.mibbit.com/](https://www.mibbit.com/) .

If you want a desktop application, there is Xchat written in GTK and Konversation for KDE.

An IRC server allows people to join channels prefixed with a hash (#) symbol.  Like a lot of open source projects, Ubuntu is on the irc.freenode.net server.  Their help channel is #ubuntu.

### Post by Dr Small on 2009-01-31
You'll probably want to use xchat. Intstall that, connect to the server "Freenode" (irc.freenode.net) and (if I recall correctly) on the first time in xchat, you'll be given a dialog to join a channel. Enter a channel name there, such as #ubuntuforums-beginners

IRC is heavily based around simple and easy to learn commands. To join a channel, you would natuarally type:
```
/join #channel
```

It's basically that simple, and that should get you started ;)

### Post by -kg- on 2009-01-31
Pidgin is included with Ubuntu, and besides enabling you to communicate using the various IM networks (like AIM, etc.) it will also enable you to connect to the IRC (Internet Relay Chat) network.

Like Dr Small, I prefer X-Chat myself.  It is similar to mIRC, which I have used for years.  I found Pidgin rather awkward to use for IRC.

There is a great deal of information on the IRC networks available from the [mIRC help page.]("http://www.mirc.com/help.html") The page has a great amount of information on the IRC networks and the commands one uses to navigate and communicate.
