### Post by Rob Maddison on 2009-09-22
Hi,

I have the L300-29W and mine works fine with the Cheese app. If you've not tried it then go to Applications>Add/remove applications and search for Cheese then install.

I've not added an icon to my bar but I guess you can. I run mine through 'Run an application'.

Hope this helps.
