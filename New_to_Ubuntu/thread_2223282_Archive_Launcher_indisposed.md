### Post by grahammechanical on 2014-05-10
You also need to understand that there is a fundamental difference between the Windows operating system and the Linux operating system. Programs written for Microsoft Windows will not run in Linux and programs written for Linux will not run on Windows. You will need to use Linux versions of the programs you want to use or equivalent Linux programs.

The developers of Malwarebytes do not produce a Linux version as far as I can tell. So, the problem is the fault of the developers of Malwarebytes and not the Ubuntu software Centre.

The same is true of Avast Antivirus.

### Post by buzzingrobot on 2014-05-10
> **Edwin_Reffell said:**
> I like Ubuntu but cannot do what I want to do. Here are the issues I need help with. 1. I must install Malwarebytes Anti-Malware and Avast Antivirus without delay but Ubuntu's Archive Launcher is indisposed and totally prevents this.

If those are Windows archives, the Ubuntu archiver may or may not expand them.  But, first, as mentioned, Windows programs only run on Windows.  Second, Ubuntu programs are seldom installed by simply expanding a compressed archive.  They're collected in archives -- 'packages' -- that also contain the instructions the Software Center used to install them correctly. 

If you want to run anti-malware/antivirus software on Ubuntu, the most common anti-viral program is probably ClamAV. Be aware that the primary reason to run anti-viral software on Linux is probably to prevent passing infected Windows email along to any Windows users who may be on your local net. 

 >  ...I want to remove it completely from my computer but without losing my email and Viber and Skype. How do I do this?

This is doable and not particularly complex.  But, I'd wait until you have Ubuntu running to your liking and you have Viber, Skype, etc. running on Ubuntu. Then, come back and post another question about removing Windows and allowing Ubuntu to use the entire drive.

 >  3.Until now I have used Google Chrome as my browser. Should I change to Firefox?

Your call. Install both and see which you prefer.  Google does not make Chrome available for distribution by Ubuntu and other Linux distributions.  It's at google.com/chrome.

 > 4.I am transposing the 1891 census for Streatham for FreeCen using Excel. Is that possible with Ubuntu?

Test LibreOffice's Calc with the Excel files, as Gifford mentioned.  See if everything works as it should.

As a Windows program, Excel can't run directly on Linux.  A suite of software called Wine exists for Linux that, in effect, convinces Windows programs they are running on Windows. It may or may not work to your satisfaction.  Again, a question for later.

 > 5. Finally I want to be able to read my gmail when I press the M icon on the left hand side and not review my account...

Don't know what you are currently using to read Gmail.  But, there are ways to bind a keystroke to take some action, like launching a mail program.  Of course, if it's bound to the "M" key, you can't use that key to enter uppercase M's. Better to use a combined keystroke like Ctrl-M, or whatever.

I know GMail's web interface offers some keyboard commands. I haven't used it in a long time, so I don't know if they work in Linux.
