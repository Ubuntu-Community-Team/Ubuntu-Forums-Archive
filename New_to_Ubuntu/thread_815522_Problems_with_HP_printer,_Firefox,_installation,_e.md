If you are running 6.10, then you should update to a newer version of Ubuntu because support has been discontinued.  It lasted for 18 months, which expired in April 08.  If you don't want to update every 18 months, then install Hardy Heron, which has desktop support for 3 years.

To find out what version of Ubuntu you are running do this:

System > About Ubuntu.  In the second paragraph, it will tell you your version.
