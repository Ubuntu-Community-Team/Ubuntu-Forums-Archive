### Post by gmargo on 2010-06-20
How to find the netbook .torrent file:

Starting on [http://www.ubuntu.com/](http://www.ubuntu.com/), the big Download button takes you to this page:
[http://www.ubuntu.com/desktop/get-ubuntu/download](http://www.ubuntu.com/desktop/get-ubuntu/download)
Click on "Alternative Downloads", which takes you to this page:
[http://www.ubuntu.com/desktop/get-ubuntu/alternative-download](http://www.ubuntu.com/desktop/get-ubuntu/alternative-download)
Now scroll down to the "Complete Download Mirror List", and select any mirror site (as you'll only download the .torrent file from the mirror site, the location really does not matter.)
I'll pick OSU Open Source Labs, whose link is:
[http://ubuntu.osuosl.org/releases/](http://ubuntu.osuosl.org/releases/)
Then click on the "lucid" link, which takes you here:
[http://ubuntu.osuosl.org/releases/lucid/](http://ubuntu.osuosl.org/releases/lucid/)

And on that page you will find the .torrent links for the main versions, including this one for the netbook torrent:
[http://ubuntu.osuosl.org/releases/lucid/ubuntu-10.04-netbook-i386.iso.torrent](http://ubuntu.osuosl.org/releases/lucid/ubuntu-10.04-netbook-i386.iso.torrent)
