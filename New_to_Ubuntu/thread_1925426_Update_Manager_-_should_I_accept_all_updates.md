### Post by KiwiPete on 2012-02-14
I'm using Ubuntu 10.04 on a Netbook and have to say it is a brilliant set up. Have used this for over a year now and it works flawlessly.

I have a question about the numerous software updates that are routinely offered to me by Update Manager ( I have it set to automatically check and offer the updates). I usually just accept them all and do the downloads, but I have wondered whether I am slowly cluttering up my machine - in a similar way to what happens with Windows systems which gradually get slower and slower due to clutter. I have not seen any evidence of this with Ubuntu but thought I would ask. 

I know that I can go through the offered updates one by one and read about them to decide whether to update them or not, but to be honest that takes too much time - and sometimes even when I read the brief description of each update I am uncertain whether I need it or not.  Is there a simp;ler way - or am I OK to keep on accepting all the updates? Do other users have any useful "rules of thumb" that they find useful for deciding what updates to accept and what ones to leave out?

Thanks for any help offered.:D

KiwiPete

### Post by winh8r on 2012-02-14
My general rule of thumb is to remove whatever packages I either no longer need or never use. That way they will not require updates and consequently there are less updates required.

You must remember that some programs have interlinked dependencies, so make sure you know exactly what you are removing though!

You will always be prompted either in Synaptic package manager or in the terminal as to what is going to be removed and what it will affect.

### Post by Scott Baker on 2012-02-17
I just wanted to toss this in for consideration. Go to synaptic (or the software center in later versions), and get your hands on the app "remove orphaned packages". This app is not overly aggresive, when used properly. Stay with the default setting "remove orphaned packages" and just let the software work it's magic. This will help keep your machine clean. Finally, consider a "fresh" install of Ubuntu every now and then. There's nothing quite like a fresh install to help blow out the cobwebs, and get everything behaving good as new.  \\:D/
