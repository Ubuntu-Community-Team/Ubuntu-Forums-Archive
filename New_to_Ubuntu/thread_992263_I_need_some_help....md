### Post by Titan8990 on 2008-11-24
You have to burn the image to a CD.


I use CDBurnerXP: [http://cdburnerxp.se/en/home](http://cdburnerxp.se/en/home)


Make sure you select the burn ISO option.


Once you have created the CD, allow your computer to boot to it by placing it in the CD drive and rebooting.


From there you can select "Start or Install Ubuntu". From there you can preview what Ubuntu is like without it effecting your system. However, the live CD does not work with all computers and will be much slower than an actual installation of Ubuntu.
