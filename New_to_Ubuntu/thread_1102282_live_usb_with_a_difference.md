### Post by Temposs on 2009-03-21
You can run Ubuntu in Windows in two ways.

1) Virtual Machine.  Programs like Sun's Virtual Box, or the program called VMWare, would be able to run Ubuntu pretty well under Windows, I think. Personally, I do the opposite of this, since I prefer using Ubuntu rather than Windows. I have Ubuntu installed as my OS and I run Windows using VirtualBox for the few times I need it(MSSQL and .NET stuff that doesn't work under Mono).

2) There's a program that the Ubuntu makers developed called Wubi, which essentially lets you install Ubuntu as a program on Windows. I've never tried this before, but it should work as well.
