### Post by eriktheblu on 2009-09-01
Many pdfs (scanned pages especially) have text stored as an image rather than as text. If this is the case with your document, you won't be able to select it.

Assuming you're using Acrobat or the default Ubuntu document viewer, you should have a text selection cursor available (I've never had a problem selecting text with either program). It should look like a capital "I" (normally default for pdfs with text) as opposed to the "+" (normally default for image only pdfs)
