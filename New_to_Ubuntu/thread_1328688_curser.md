System > Preferences/Settings > Appearance

When you have a theme selected, click the customize button, then in the new dialog, go to the pointer tab, select the cursor you want to make bigger, then choose a size.

However, the only one that I know that does this are the default non-Human pointers. None of the gnome-look.org pointers can change size if I recall.
