### Post by Frogs Hair on 2011-09-29
Alternate installations are text based as far as I understand . Minimal installations include only the basic packages required for the installation. This allows the installation of only the software the user really wants or needs and can reduce the size of the ubuntu installation . 

[https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)
