### Post by ajgreeny on 2013-05-27
10.04 has now lost support so you will get no more updates of any of the GUI parts of the system or gnome 2 applications, so you should update if possible to a later supported version.

However, to answer your question, you may have removed either the notification-area or the indicator-applet from your panel, so try adding back those to see if the network icon re-appears, and also check that you have nm-applet (I think that's what it is called) installed
