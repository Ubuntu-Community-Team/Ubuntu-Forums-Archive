### Post by GreenN00b on 2008-08-06
This are listed because for example pidgin is installed. If you are sure you wont use it open synaptic package manager from system/administration menu and remove it, same for all applications that you think you will not use.

Be carefully not to remove a meta package like "ubuntu-desktop" when removing this or you may not receive all security updates.
Also before uninstalling an application you should make sure that is not needed by the system or the desktop environment.

After you removed applications check for updates again.
