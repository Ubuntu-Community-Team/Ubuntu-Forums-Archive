### Post by lisati on 2009-05-20
Installing the updates is optional - you might like to think of it as a form of "quality control". I find it reassuring that there's someone out there looking for better ways of doing things, and responding when something comes up that might need attention.

To manually install the updates (and to choose which updates to install), go to the System->Administration->Update manager
