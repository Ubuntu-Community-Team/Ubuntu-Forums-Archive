### Post by Monicker on 2008-05-06
It would be a lot easier to use the graphical package manager, which is Synaptic.

System -> Administration -> Synaptic Package Manager

Just search for Thunderbird, after it shows in the list (mozilla-thunderbird) just right click and mark for installation.  Click Apply.

### Post by WalmartSniperLX on 2008-05-06
What he is showing you to use is the Ubuntu Software Repository. It's a collection of installable software that's made easy for users to install just by a few clicks in System-> Administration -> Synaptic Package Manager.

Basically in Ubuntu, you dont have to go looking for software on the internet to download because most of the software you will need is available in the Ubuntu Repository. This makes it very easy. Just type in the name of the program in the search, if it shows up, click the checkbox next to it and select "install package". When you are ready to install it, click the big green Check mark near the top. When it is finished, all you have to do is find the new program under Applications and launch it like you would in win or mac.
