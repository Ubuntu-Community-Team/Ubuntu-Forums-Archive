### Post by zombifier25 on 2012-08-31
Torbutton can still be installed manually from Tor's website, however (to quote my own blog):
> [...] starting from the latest version (as of speaking), Torbutton is longer supposed to be used with normal Firefox, since the on/off toggle of Torbutton is now removed, and Torbuton is set to always on. If you want to use Tor for browsing, USE THE TOR BROWSER BUNDLE.[...]
So download the bundle.
