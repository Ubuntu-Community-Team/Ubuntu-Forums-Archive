### Post by monkeybrain20122 on 2014-03-23
Install ubuntu-restricted-extras from the Software Centre or Synaptic (if you have xubuntu or lubuntu rather than Ubuntul, as you said this is an old machine), that would install flash along with other multimedia codecs. In Ubuntu you install most things from the repo rather than downloading from webpages. Also Yum is a Redhat/Fedora/Centos (another family of Linux distros different from Ubuntu) thing that doesn't work on Debian/Ubuntu systems.
