### Post by 3rdalbum on 2009-12-25
System > Administration > Synaptic Package Manager. Type "abcde" in the Quick Search box. Right-click the "abcde" package, choose Mark For Removal. Click the Apply button. Done.

### Post by kermiac on 2009-12-25
> **soryu said:**
> how do i remove this program?

go to a terminal and type
```
sudo apt-get remove abcde
```

or for a graphical solution, goto System menu --> Administration --> Synaptic Package Manager. Then search for abcde. Now click on the "green box" next to this "abcde" package & select "Mark for Removal". Now hit apply.
