### Post by JeQhdMD on 2015-01-07
This is the kind of notebook you might be (much) better off installing one of the lighter Ubuntu desktops . . . Xubuntu 14.04 or Lubuntu 14.04 are better OS for a machine as light as the Samsung N210 (atom proc, 1>2 GiB ram, etc.).   Another possibility is Linux Mine Mate - whatever the latest version is (I think it is 17.1).    On all these the speed will be much better, and Xubuntu and Mint Mate will provide some tools to make any hardware adjustments needed.   IF your wireless isn't activated right away, it can be fixed by using an ethernet cable so you can goto the systems settings and install the appropriate drivers.
