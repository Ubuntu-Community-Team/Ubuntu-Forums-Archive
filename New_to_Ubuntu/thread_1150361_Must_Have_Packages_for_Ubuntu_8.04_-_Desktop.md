### Post by Zeedok on 2009-05-06
In my experience, finding info about the "best" programs and games in Ubuntu starts with google, or you could check out a few threads here on the forum - there's one called "[Cool applications you use that others might not know of]("http://ubuntuforums.org/showthread.php?t=382137&highlight=cool+programs")" which is a bit over whelming, but pretty awesome.  If you need something specific trying posting something like: "What's the best video editor in linux" and you get heaps of opinions.

You could also check websites like "[Linux Today]("http://www.linuxtoday.com/")", which always seems to have articles called "The ten best blah programs in linux" or check out [UbuntuGeek]("http://www.ubuntugeek.com/") or [HowtoForge]("http://www.howtoforge.com/").  You'll get heaps of suggestions and tips.  The best thing is the software is **FREE** so you can download it, try it out and if it doesn't suit uninstall it!

By the way, don't worry about the dependencies - just search for and install programs using Synaptic (System > Administration > Synaptic Package Manager) it will install all the necessary depedencies for you.

Good luck.
