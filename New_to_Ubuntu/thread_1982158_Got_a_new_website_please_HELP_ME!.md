### Post by sadaruwan12 on 2012-05-18
For HTML you can use Geeny, Scribes or BlueFish. I'm a web guy and I personally use Scribes and BlueFish. But if you want to do content management then I like to suggest a CMS environment like wordpress or joomla.

Wordpress is actually build for blogging but we can turn it to be a CMS environment it's very easy to learn and very beginner friendly.

Joomla on the other hand is machine cut as a CMS but one needs to be having a better understanding of the HTML environment and it's workings.
