### Post by Toshibawarrior on 2008-12-10
Simply download **gnome-bluetooth** from Synaptic and when done open it through "*Applications->Accessories->Bluetooth File Sharing*" and then send the file from your cellphone to the computer (making sure you computer is discoverable by checking the settings on the right-click menu of the bluetooth icon on the notification area on the panel).

When you browse the device via bluetooth most cellphones will show nothing somehow, but you can select the files on your cellphone and send them directly to the computer using the method above.

I've used this method with a Motorola RAZR V3r and an LG Shine CU720 and both work great!

Hope this helps! :)!
