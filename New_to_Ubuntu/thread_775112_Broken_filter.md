### Post by saskatchewan on 2008-04-29
When I tried to do an install and I was told that I had a broken packet and to use the broken filter to fix this.  What is the broken filter and how do I use it?

---

### Post by aheckler on 2008-04-29
Go into Synaptic Package Manager (in System > Administration). In the lower left, select "Custom Filters" then in the left pane select "Broken". Check all the packages that appear on the right and remove them.
