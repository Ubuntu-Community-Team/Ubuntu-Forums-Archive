### Post by stephanvaningen on 2008-11-06
Put a small Ubuntu-sticker on that awkward 'Windows-flag-covered' key between your left-ctrl and left-alt key and .. voila!: there's your Ubuntu-key! :-) (Often called the Super-key)

It is used by some Compiz functions for GUI-shortcuts like desktop zoom, enhanced task-switching, ..

that depends on what you want an "ubuntu key" to do :). if you want it to open the menu (like on windows) do the following:

System -> Preferences -> Keyboard Shortcuts

then scroll down to the 'desktop' section and find the entry that says "show the panel menu". then click it and press the windows key. close and it should now open the menu!

### Post by nhasian on 2008-11-06
actually we use the windows key too, but we call it the Super key instead.
some uses for the Super Key include:

Super-e shows all desktops
Super-m switches to high contrast

i setup super-z to make my screen go to sleep in System/Preferences/CompizCofig
