### Post by KlytusLord on 2011-01-18
An easy thing to do is to download Xubuntu and boot directly from the USB drive. You will be able to test if any of the hardware on the machine is not detected properly or does not work properly.

The only issues I have experienced with Acer laptops are related to the touchpad. In some models, the touchpad is detected as a generic PS/2 mouse, which means you cannot configure properly and the gestures/scrolling can act oddly, though normal mouse operations (cursor movement and tapping) still work.

If everything in Xubuntu works as expected when running from the USB drive, then Ubuntu will be fine once fully installed.
