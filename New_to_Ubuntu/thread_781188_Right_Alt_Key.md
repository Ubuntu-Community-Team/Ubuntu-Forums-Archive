### Post by undertakingyou on 2008-05-04
> **Bubba64 said:**
> system-preferences-keyboard allows some key assignment. For example my IBM think pad didn't have a windows key to trigger gnome do so I assigned the left alt key this function.

Bubba64,  Thanks, that was the right direction.  Going to SYSTEM=>PREFERENCES=>KEYBOARD then going to the 'layouts' tab and going to 'Layout Options'.  Once there I went to 'Alt/Win key behavior' and checking the 'Alt and Meta keys are the Alt keys'.  Then going to 'Third Level choosers' and checking only the 'Right ALT never chooses third level' and voila it works!
I appreciate it.
