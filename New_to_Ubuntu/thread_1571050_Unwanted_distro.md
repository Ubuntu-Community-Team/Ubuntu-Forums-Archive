If you don't have any valuable files, you can download an ubuntu from the ubuntu main site and burn the .iso file on a CD, then put the CD in your pc and reboot. It will boot from the disk and you can install ubuntu on the PC.
Then when it asks about the memory placements, you should select that it uses the whole memory/drive so no Madriva will be hanging around.
