### Post by avtolle on 2008-07-05
You may need to use Synaptic to see whether there is a help file. Use the "search" feature of Synaptic to find Qcad, and if there is a separate help file, mark it for installation and apply. You might also be running into a feature of "free" vs. "supported" programs I've seen a few times in the past; no help, w/o the "Professional" version.

EDIT: Just checked Synaptic on my system, and find a qcad.doc file that might be installed; HTH.
