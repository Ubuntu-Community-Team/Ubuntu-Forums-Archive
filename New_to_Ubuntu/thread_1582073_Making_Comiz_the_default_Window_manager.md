### Post by JohnHeikkila on 2010-09-26
If you installed 'fusion-icon' you can run it by pressing ALT+F2, then typing 'fusion-icon'. The icon will appear in the notification area and you will be able to Right-Click it and 'Select Window Manager'.
This is useful if you want to quickly change to Openbox for faster gaming experience and then back into Compiz for visual candy.
