### Post by Bob Appleby on 2010-06-10
Got a second hand Dell netbook on ebay and installed 10.04, with no problems. The f keys stop with f10. Does anyone know how to access the f11 (full screen) function. Using the Chrome browser tool I can go to full screen, but returning is a bit difficult.

---

### Post by snowpine on 2010-06-10
If you have the latest BIOS update installed from Dell, you can use Fn+Z and Fn+X as F11 and F12.
