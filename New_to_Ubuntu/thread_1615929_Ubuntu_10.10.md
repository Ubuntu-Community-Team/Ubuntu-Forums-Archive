### Post by na5h on 2010-11-07
Since you are already running Ubuntu 10.04, you can simply update to the latest version by using the update manager. Easiest way is to open a terminal (Applications -> Accessories -> Terminal) and type "update manager -d" (without quotes). The update manager should pop up and tell you that there is a new version available and ask you if you wish to update. 

Note: If everything is working fine in 10.04, there is really no need to upgrade...an upgrade can cause problems that you have not had before. 10.04 is the current long-term-support release (it will get updates for 3 years, 10.10 will only get updates for 1). So, if it is stability you want, then you might wanna stick with 10.04!

You can also download 10.10 from Ubuntu.com, and burn a LiveCD. By doing that, you can try it out on your computer before you install and see if everything is working the way it should. :)
