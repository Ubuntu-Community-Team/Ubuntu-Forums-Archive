### Post by bobbyb219 on 2008-09-18
I am brand new using Linux and finally figured out how to install the 9600 GT video card.  I started using compiz fusion and I guess I got a little carried away and set a mouse button to use a feature.  Now, I cannot use the mouse button to select programs, menus or anything else.  Is there a way to reset the mouse configuration?  Any suggestions would be appreciated!

---

### Post by Rhubarb on 2008-09-18
Press Alt + F2, a box will come up, type in:
```
metacity --replace
```

You should now be able to use your mouse to enter menus.
I suggest you change the key binding in your compiz setup to more sane defaults (you can press the broom icon to revert to defaults in the key binding menus in the compiz config settings manager).

To enter the Applications / Places / System menu, Press Alt + F1.
