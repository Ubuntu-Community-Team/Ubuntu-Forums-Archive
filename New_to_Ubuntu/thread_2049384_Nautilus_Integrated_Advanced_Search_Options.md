Does anyone know if there is a way to have advanced search options integrated into Nautilus?

Kinda like on a mac.

"Search This Folder"
"Search Computer"
"Search File Contents"

I have found external programs that allow advanced searching, and they work fine.

But there's gotta be a way to have advanced options build into Nautilus, right?

---

### Post by cbanakis on 2012-08-28
Found a viable solution.

Using gnome-search-tool combined with nautilus-actions.

Not exactly what I was looking for, but now I can right click on a folder and do an advanced search.

Available instructions for nautilus-actions were pretty outdated, but eventually I got it working with the info posted at [http://ubuntuforums.org/showthread.php?t=1376682&page=2](http://ubuntuforums.org/showthread.php?t=1376682&page=2)

Thanks for listening.
