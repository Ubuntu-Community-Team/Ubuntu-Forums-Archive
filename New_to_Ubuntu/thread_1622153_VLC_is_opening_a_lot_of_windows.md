### Post by Verbeck on 2010-11-15

### Post by halitech on 2010-11-15
Go to  Tools - Preferences and check "Allow only one instance" as Verbeck says. That should do it. You can also select Enque files when in one instance mood so it won't stop playing one song when you select another. For me it was right on the first page that opened.
