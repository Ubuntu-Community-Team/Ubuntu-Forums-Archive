### Post by HiImTye on 2012-10-01
make sure your 'F' mode is on (if you are using a keyboard with an F-mode toggle, such as a laptop) and if it still doesn't work, then check the 'shortcuts' tab in 'keyboard' and make sure that 'Show the run command prompt' in 'System' is set to alt+f2
