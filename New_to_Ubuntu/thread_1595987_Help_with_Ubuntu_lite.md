### Post by mikewhatever on 2010-10-13
Synaptic does both downloading and installing. If, for some reason, you need the downloaded packages, they are in /var/cache/apt/archives. Dev or -dev usually stands for 'development'. You probably don't need the dev related packages.
Wine is a compatibility layer for Windows programs. Is it supposed to be seen?
Here is a nice walkthrough -> [http://psychocats.net/ubuntu/wine](http://psychocats.net/ubuntu/wine).
