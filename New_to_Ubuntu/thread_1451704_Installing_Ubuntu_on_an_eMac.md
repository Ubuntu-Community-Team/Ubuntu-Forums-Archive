### Post by marshmallow1304 on 2010-04-10
If it has an Intel processor, the standard version from the Ubuntu download page will suffice.  If it has a PowerPC processor, you'll need to get the PPC version from [this download page]("http://cdimage.ubuntu.com/ports/releases/9.10/release/").


Edit after seeing the last post:
There's no reason to install 8.10.  I'd either install 9.10 or wait for 10.04 to come out and install that.
