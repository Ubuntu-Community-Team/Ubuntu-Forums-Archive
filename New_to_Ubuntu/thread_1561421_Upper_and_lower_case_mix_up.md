Ubuntu and Mint are different distributions although Mint is based on Ubuntu so help given for Ubuntu will help for Mint the layout of the desktop is however quite different.

For you problem I would first try System > Preferences > Keyboard and in the layouts tab return to defaults.

In Mint you can find this 

Menu > Control Center > hardware > Keyboard again reset the defaults in the layouts tab
