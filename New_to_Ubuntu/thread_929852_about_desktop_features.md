### Post by Hendrixski on 2008-09-25
There's tons of different desktop environments. I like GNOME the best, but Enlightenment is definitely pretty cool. KDE4 is going to be pretty awesome when it actually works ... and it will be able to work on Windows as well :-)

There's IceWM and things like that.  Also, if you hate the mouse, use RatPoison. It's basically all key-command based, and very very lightweight. I have a friend who uses it... but to be fair, he uses Emacs as his operating system, so using RatPoison kind of works out.
