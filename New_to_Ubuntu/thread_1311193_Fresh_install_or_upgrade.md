### Post by SlickRick on 2009-11-02
I find it's good practice to have your /home area in a separate partition, so that you could have a fresh install and keep all of your files, you would only need to re-download your applications. You would need to format 9.04 when installing 9.10 unless you install into a separate partition.

### Post by PartisanEntity on 2009-11-02
You don't need to delete a previous installation of Ubuntu, the upgrade will remove obsolete packages, add new ones and replace changed ones.

If you want to do a complete fresh install, you can select to format the Ubuntu partition during the installation process.
