### Post by beew on 2011-03-06
In the meantime if you are affected you can use the flashvideoreplacer plugin for firefox. It generally performs better than flash, uses a lot less cpu. I  actually use it instead of flash for all the supported sites except for one machine which has a vdpau capable Nvidia card since flash 1.02 uses vdpau and somehow mplayer when used as a backend to flashvideoreplacer in FF4 beta doesn't use vdpau even though it does in FF3.14 and as a stand alone player.

 Another option is minitube, install version 1.4 from a PPA, the version in the Ubuntu repo is broken.
