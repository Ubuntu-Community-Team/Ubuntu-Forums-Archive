### Post by sgosnell on 2009-03-05
The easy way to install an HP all-in-one is to connect it to the router, go to System/Administration/Printing and click on New.  Follow the wizard, and tell it to install a network printer.  It will find the HP, allow you to verify that's the one you want, and get the software and install it.  No need for a manual install, although you can do that if you want.
