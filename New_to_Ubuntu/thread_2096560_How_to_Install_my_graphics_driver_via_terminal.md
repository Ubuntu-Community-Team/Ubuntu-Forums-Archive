### Post by sudodus on 2012-12-20
If you can use the drivers suggested by the system:

***jockey-gtk*** will do the job in graphics mode and

***jockey-text*** will do the same job in a text screen.

Otherwise refer to for example this link:
[[COLOR="Red"]http://ubuntuforums.org/showthread.php?t=1948324[/COLOR]]("http://ubuntuforums.org/showthread.php?t=1948324")
