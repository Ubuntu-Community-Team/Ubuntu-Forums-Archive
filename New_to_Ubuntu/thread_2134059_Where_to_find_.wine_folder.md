### Post by ppjdee on 2013-04-10
I suggest you check out [WineHQ/AppDB]("http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true") for the game you are wanting to play, it may save you from a headache. If I am not mistaken, wine will install the game into the ".wine/whateverthepathis" for you.
Anyways to acces the .wine folder (a dot indicates hidden folder) click on the "Home" folder and press ctrl + h.
