### Post by iaculallad on 2008-06-02
Create a backup of your current xorg.conf file first then copy and paste the content of the edited xorg.conf file below to your xorg.conf file (overwrite it). Save and restart. Hope this could work in your case.

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.bak
sudo gedit /etc/X11/xorg.conf

```

# xorg.conf (xorg X Window System server configuration file)
