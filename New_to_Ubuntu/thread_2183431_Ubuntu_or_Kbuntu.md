### Post by mastablasta on 2013-10-25
it seems it depends on hardware mix and also version.
i am using Kubuntu 64bit on netbook, effects turned on, kicker menu. it's very fast. much faster than windows 7 starter. 1.5 GB ram available (0.5GB reserved for GPU). i've tried lubuntu and xubuntu first and they ran with about the same speed. i plan to upgrade this to at least 6 GB ram when i have some more money.

however there is also netbook plasma in Kubuntu (netbook interface). I would say that one uses less resoruces and is made for netbook screens.

i see that this 64bit uses a bit more RAM - a little over 400 MB, while the 32bit i have on desktop uses a little over 200 MB ram. I htink this can be further reduced with special kubuntu low-fat package.

battery last about the same as in windows 7 perhaps only slightly less.

as mentioned other options are good too. and they all have Ubunut capability. the lightest of the mentioned ones is Lubuntu.

you might also give Bohdi linux a try (based on ubuntu but with elementary interface). if you want something super light then install icewm or similar windows manager. it will look like windows 95 with modern applications :-D
