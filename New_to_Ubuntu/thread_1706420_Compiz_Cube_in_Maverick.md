### Post by fooman on 2011-03-13
desktop or laptop?  i only ask because this is a lot easier using a wheel mouse. 

not sure if its set by default....but try pressing and holding ctrl+alt+middle button on your mouse (wheel).  then move your mouse around while holding those 3 keys/buttons.

if that's not set by default,   you can set it (as well as a ton of other things) by using the compiz configuration settings manager (ccsm).  its not installed by default,  but you can get in synaptic package manager or in the ubuntu software center (look under "themes and tweaks" > advance desktop effects settings).

after it installs,  find it under > system > preferences > compizconfig settings manager.  have fun!  ):P
