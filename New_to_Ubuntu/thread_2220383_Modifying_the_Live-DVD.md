### Post by sudodus on 2014-04-28
Yes it is possible, but most people would use a persistent USB boot drive for that purpose. It is much easier to make. See this link

[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)

But if you do not intend to use it yourself, but to distribute it, then it is a different matter, and it might be a good idea to tweak the system and re-spin the iso file.
