### Post by Rex Bouwense on 2012-05-24
Did you copy the download or burn the ISO to the CD?  If you merely copied it, it will not be bootable.  If you are not sure open the CD.  If there is only one file then you probably just copied it.  If there are many files then you probably burned it.  If you burned it, your may have gotten a bad burn.  Try reburning at the slowest possible speed.
