### Post by Wootchismo on 2010-03-04
I've experienced exactly what you're describing with Hulu.  I previously used 32-bit flash on my 64-bit system but had troubles manipulating the controls (pause and scroll, specifically) so I upgraded to 64-bit flash which allowed me to pause and scroll but then lost me access to Hulu streaming.  Researching it online I found that many people using 64-bit were unable to watch Hulu anymore.  Other than having both .so files in one directory and creating a bash file to cp the one you want to your plugins directory, though, I don't know any easy way to have one browser use 32-bit flash while the other uses 64.
