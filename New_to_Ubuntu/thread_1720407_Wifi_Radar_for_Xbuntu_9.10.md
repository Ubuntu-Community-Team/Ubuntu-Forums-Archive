### Post by LowSky on 2011-04-03
Hi welcome to Linux and these forums.

I would like to say please upgrade to a newer version of Xubuntu. Xubuntu 9.10 is old and soon to be not supported. If you need a stable and long term supported version please use 10.04. Otherwise the newest currently is 10.10, and by the end of the month to be replaced with 11.04.

Side note Wifi should work fine on Xubuntu.

Simply open a terminal and type

```
sudo apt-get install wifi-radar
```
