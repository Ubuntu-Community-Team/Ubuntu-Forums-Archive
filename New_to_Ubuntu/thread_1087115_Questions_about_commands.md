### Post by Temposs on 2009-03-04
nautilus is the graphical file browser that Ubuntu uses under GNOME. So, if you have the standard Ubuntu and click the Places menu and choose "Home Folder", that will open nautilus to your home folder.

sudo gives you temporary admin priveleges. This is a huge part of Ubuntu's security model.
