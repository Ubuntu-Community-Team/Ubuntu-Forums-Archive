### Post by HermanAB on 2015-08-15
Well, one major reason why Linux is generally more secure and stable than Windows, is because it is difficult for users to install random cruft off the web.

It is best to stick with stuff that is part of the distribution and installed with the apt-get tools from the official repositories.

If you really have to install other stuff, then read the readme and install files that come with it.  Those will tell you how.
