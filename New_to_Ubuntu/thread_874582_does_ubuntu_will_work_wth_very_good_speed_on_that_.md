### Post by Growbag on 2008-07-31
Make sure the hard disk is a SATA2, not a SATA1, there is a difference.  120 gig is rather small by todays standards, and is possibly an old SATA1.  Look for 200 gig minimum, 300 is a good size and should be about the same price.

The mainboard also makes a difference in overall system speed, a slow (cheap) chipset is a lot slower than a fast one.

Also RAM speed, and CPU FSB (Front Side Bus) speed.  A faster CPU clock speed DOES NOT mean it is actually faster, if the FSB speed is lower, the chip will be slower.

Example:  a 2000mhz CPU with a 667mhz FSB will be a LOT slower than a 2000mhz CPU with an 800mhz FSB.  It is a bit of a con from the CPU manufacturers.

The bigger number is always the "external" speed, the smaller number is how fast the chip runs with the rest of the mainboard and peripherals.

Takes me back to the 486 days, when Intel first came out with this "clock doubling" nonsense.  A 486DX50 was lightning fast compared to a 486DX2/66, the DX50 ran at 50mhz whereas the DX2/66 ran at only 33mhz, but they got away with calling it a 66mhz chip because internally it ran at 66mhz, even though it could only "talk" to the rest of the system at 33mhz.
