### Post by cariboo on 2010-02-03
Moved to ABT, as you tacked your post on to the end of a 2 year old thread.

The file you found is the man page for xorg.conf. To read the contents, open an Applications->Accessories->Terminal and type:

```
man xorg.conf
```

the xorg.conf file is located in /etc/X11.
