### Post by lykwydchykyn on 2012-11-09
You're going to need to re-do the disc.  Putting the ISO file on disc is different from just copying it over like a normal file; you have to do it a special way so that the disc is bootable.

See if these help:

[https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)


Another option is to use a USB drive to install:

[http://www.ubuntu.com/download/help/create-a-usb-stick-on-windows](http://www.ubuntu.com/download/help/create-a-usb-stick-on-windows)
