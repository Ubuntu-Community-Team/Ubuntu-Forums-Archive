### Post by oldos2er on 2011-04-15
You can check [http://appdb.winehq.org/](http://appdb.winehq.org/) to see if there's a chance your games will run.

To install Wine, run ```
sudo apt-get install wine
``` in a terminal, or use the Software Center.
