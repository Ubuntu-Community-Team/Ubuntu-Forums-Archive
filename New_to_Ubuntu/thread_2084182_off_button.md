### Post by Hadaka on 2012-11-14
Hi, to add a system shutdown...turn off computer "button", google
"how to add icon to launcher <your ubuntu version> " have the desktop
configuration for it point to a path to root. In root build a simple bash script
with this command.

```
shutdown -h now
```this halts all applications and turns off the power.

to test before building..
```
sudo -i
shutdown -h now
```
here is a link for 16,599 images of different switches for your icon
[http://www.canstockphoto.com/images-photos/switches.html](http://www.canstockphoto.com/images-photos/switches.html)
have fun.
