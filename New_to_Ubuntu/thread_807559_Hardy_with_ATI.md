### Post by Tatty on 2008-05-25
> **Jshaw said:**
> I was just wondering how well Hardy Heron works with Ati graphics. I've just come into possession of a computer with Ati, instead of Intel, and I was just wondering if there would be any problems that I would have to face. I don't know much about it yet.

It depends on the card.  Generally if it is a Radeon 9500 or above it should be supported by the official proprietry driver.  The restricted drivers manager should install it for you, and if not then EnvyNG will probably do it.

If not then you can still use it with the open-source driver (which is loaded by default), but it is generally considered to be not as good as the proprietry driver, particularly with 3D.
