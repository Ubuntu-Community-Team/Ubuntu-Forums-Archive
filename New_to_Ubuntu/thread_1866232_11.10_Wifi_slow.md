Some names to search for include 'chilli555', 'wildmanne39', 'praseodym' and 'anewguy'. There are others of course, those mentioned are ones that post a lot. Once you have the correct drivers installed and working things should be flying, if not the only thing I can suggest is to plug a cable in and 'Factory reset' the router and set everything up again using manual configuration for the Wireless set up. Accepting the default settings for each stage usually works well enough however some things (wireless channel, level of security) might need to be changed to improve performance. Do not hide the signal, there is no advantage to you with that option. Even Microsoft says it is slightly more dangerous as your pc is broadcasting its own ID to log in to a network. I have gone slightly off topic, how naughty am I.
