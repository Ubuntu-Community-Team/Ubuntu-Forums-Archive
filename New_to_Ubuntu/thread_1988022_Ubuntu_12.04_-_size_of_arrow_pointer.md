### Post by fantab on 2012-05-27
> **Apanta said:**
> Hi.
I need to resize my  default arrow pointer only.


Install dconf-tools (installing this pkg will remove dconf pkg, its normal)

Then org-gnome-desktop-interface-cursor-size (default is 24, you can change it to suit you).
