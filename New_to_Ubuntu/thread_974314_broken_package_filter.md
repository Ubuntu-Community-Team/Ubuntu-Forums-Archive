### Post by drs305 on 2008-11-07
Open synaptic (System, Administration, Synaptic Package Manager), On the left side, you can select Custom Filters, Broken and see what package(s) are broken. From the top menu, you can select Edit, Fix Broken Packages to see if that corrects things.
