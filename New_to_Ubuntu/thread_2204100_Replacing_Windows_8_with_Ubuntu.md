### Post by oldfred on 2014-02-06
Some have done that on some systems.
But a few systems only boot Windows and it may be best to start with dual booting first.

Be sure to turn off fast boot and usually better to turn off secure boot.
Even if not wanting Windows I would still make a full backup. If later you want to sell system you may have to restore Windows to find a buyer.
One user posted that he just removes Windows hard drive and buys a new SSD. So Ubuntu is fast and later he sells system with a hard drive and brand new Windows install with out any work.
