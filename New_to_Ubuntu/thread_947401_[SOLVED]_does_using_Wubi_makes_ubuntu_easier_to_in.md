### Post by raydeen on 2008-10-14
Wubi so far as I've seen is extremely easy to use. It basically installs Ubuntu into a large file rather than a partition (so you don't have to go messing with your HD in any way). It modifies the boot loader so that you can choose Ubuntu or Windows at startup. To uninstall it, just boot into Windows and go to Add/Remove programs and remove it. Reboot and it's gone. The only downside is that I don't think you'll be able to easily transfer files between your Ubuntu and Windows installs other than using an external HD or thumbdrive.

### Post by bumanie on 2008-10-14
Generally the wubi install works well and for a first time user, it is easier to install/uninstall. However if you like ubuntu and want to keep it and have it operating at its full potential (it is somewhat slower when installed inside windows), there is a process that allows you to change ubuntu to its own partition. You can then uninstall it from windows via add/remove.
