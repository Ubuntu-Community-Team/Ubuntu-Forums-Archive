### Post by yetimon_64 on 2016-06-06
> **CantankRus said:**
> The colour can be changed using ccsm, in the **grid** plugin.

This is it OP. Just a note, I don't think ccsm is installed by default and you will likely have to install the package "compizconfig-settings-manager" from the software center _*or*_ with the next terminal command ...
```
sudo apt-get install compizconfig-settings-manager
```
Once installed you can search for and activate it from dash or with the terminal command "ccsm" as CantankRus notes.
