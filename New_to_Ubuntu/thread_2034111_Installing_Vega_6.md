### Post by evilsoup on 2012-07-27
Directory = folder.

The installation directory is simply the folder in which a program is installed. In this case, the directory you created when you extracted the tarball.

I just downloaded it to take a look, and when I double-click on one of the executables (Vega6, it has an icon that looks a bit like a cog), the program opens up perfectly well. No need for anything fancy.
