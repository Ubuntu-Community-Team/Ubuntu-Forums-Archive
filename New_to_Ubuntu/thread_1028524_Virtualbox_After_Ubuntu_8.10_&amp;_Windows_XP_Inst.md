### Post by ajgreeny on 2009-01-02
Not quite sure exactly what you mean, but if you want to run windows as a virtual machine in a ubuntu host, you can do it quite easily, and similarly you can run ubuntu in a windows host, but either way you will need to start the virtualisation process by creating a virtual machine, ie "installing" windows/ubuntu in the virtual environment (Virtualbox).  I don't think there is a way to run or port an existing install of windows in a virtualbox environment without doing this, though I could be wrong, and I believe there may be other virtualisation applications which can do it that way.  Wait for more answers before you do anything drastic.
