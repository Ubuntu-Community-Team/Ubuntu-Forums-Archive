### Post by Mark Phelps on 2012-05-11
Unless you did an image backup of your older version before attempting the upgrade (and if you had, you'd probably mention that), there is no way to get your old version back.

Canonical still has not bothered to provide the ability to roll-back from one Ubuntu version to a previous Ubuntu version.  You have to do a complete reinstall instead.

Sorry.
