### Post by rburkartjo on 2011-03-25
note this worked for me and i got all my old extension to work in ff4 even though firefox said they were not compatible

first install this extension night tester tools
[https://addons.mozilla.org/en-US/firefox/addon/nightly-tester-tools/](https://addons.mozilla.org/en-US/firefox/addon/nightly-tester-tools/)

once you install and restart nav to tools/nightly tester tools/then click
force addon compatibility

restart

however if there is a problem you can alway type in terminal
firefox --safe-mode
