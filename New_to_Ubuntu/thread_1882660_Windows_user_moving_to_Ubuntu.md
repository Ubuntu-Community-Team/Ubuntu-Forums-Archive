### Post by I2k4 on 2011-11-22
> **jamos316 said:**
> Hello everybody,
My situation is that my current Windows 7 is bogged down with crap

Pretty hard to make useful sense of that.  W7 works to run software millions of users need for work or school, and if the operating system is gummed up to prevent using important software you use professionally, then fix operating system or the software.  There are lots of un-crappifying utilities for W7.

On the other hand, if your hardware is too old or netbook weak to run W7 very well, or if you can freely choose software to achieve what you want to achieve and want to get into a supportive "movement" of Windows alternatives, Ubuntu and related distributions like Mint have a lot going.  If W7 is still running on your box, and you're in experimental mode, I suggest using Live USB with Persistence to permit testing various versions over a few weeks without wiping or partitioning or otherwise risking your hard drive.  Live USB runs reasonably well, saves your settings and software preferences, and gives a good feel for the version.  The versions are so different, it's worth testing them out extensively for look and feel.

### Post by cortman on 2011-11-22
Windows 7 can be made super fast and slick (for windows) by formatting the HD and reinstalling the OS. Be careful and research what drivers you'll need first, and how available they are.
I currently dual boot between Ubuntu 11.10 and Windows 7 on my laptop. Dual booting has been terrific, and I can't understand why anyone would be against it. It's nice to know I still have a functioning Windows OS somewhere back there too, though, in case I need to run an oddball program or something.
