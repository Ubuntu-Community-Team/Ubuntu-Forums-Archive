### Post by Bölvaður on 2011-04-13
Right click on the title bar of the window and select "Only on this workspace".
If you continue having this problem then it might be because you are using shortcuts to move windows around with you to another workspace.. does this happen with all windows or just this one pdf?

To switch to another workspace you use ctrl+alt and an arrow key (default settings)
but to switch to another workspace AND move the focused windows along with you to you use ctrl+alt+shift and an arrow key (default settings)

You can check the keyboard shotrtcuts in System &#8594; Preferences &#8594; Keyboard shortcuts. Switching to another workspace is on the lower half of that dialog.
