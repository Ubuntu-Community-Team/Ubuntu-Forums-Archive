### Post by lonestar32 on 2008-05-12
I've had a similar problem quite recently.  I think it has to do with the keyboard preferences.  What I did to fix it was: System -- Preferences -- Keyboard.  Then from there go to the "Mouse Keys" tab, and then deselect the "Allow to contol the pointer using the keyboard".

I think that should solve your problem.
