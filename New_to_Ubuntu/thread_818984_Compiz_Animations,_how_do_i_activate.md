### Post by Inxsible on 2008-06-04
have you installed the compizconfig settings manager?

type this in a terminal :```
sudo aptitude install compizconfig-settings-manager
```You can then access it under System>>Preferences>>Advanced Desktop Effects. Play around with the plugins there and select what you want.

### Post by Victormd on 2008-06-04
Assuming that you have compizconfig manager installed (if not, follow previous post), then simply enabling the Animations (marking the check box next to it) won't give you what you want, you have to click on the Animations button and setup all your animations there.

In your case, the minimizing effect on the video is called Magic Lamp and the Closing effect is called burn, and you can set them up under the respective tabs. Furthermore, you can customize each effect under the Effect Settings tab (the burn effect is set up as fire - the video uses the Randomly Colored Fire option)...
