### Post by cariboo on 2009-01-10
All Debian based distributions archive the packages you download. The way to clear up some space, is to open a Applications-->Accessories-->Terminal and type:

```
sudo apt-get clean
```

This will remove the archived packages, to remove any no longer needed dependencies in the same terminal type:

```
sudo apt-get autoremove
```

Jim

sudo nano /etc/hosts
```to edit the first line which should read "127.0.0.1 localhost" (w/out the quotes, of course).then, after pressing the enter key, write a new line "127.0.1.1 (name of the computer as identified on your network)", again without the quotes.
