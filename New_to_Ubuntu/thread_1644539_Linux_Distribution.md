### Post by lykwydchykyn on 2010-12-13
There are any number of distros that will run well on that hardware, the operative questions are how you want it to look, what do you want it to do, and how hard do you want to work to get it.

 - ease of use/configuration/maintenance
 - speed
 - available applications
 - aesthetics
 - age of software
