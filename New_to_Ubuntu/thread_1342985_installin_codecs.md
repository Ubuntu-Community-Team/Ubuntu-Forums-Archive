### Post by SteelCore on 2009-12-01
If you are on 9.10 go to Applications>Ubuntu Software Center, in the search box type "Ubuntu Restricted Extras", once found install it; the process is pretty forward.
This package includes a number of codecs and other software that will help you play several types of media on Ubuntu.
