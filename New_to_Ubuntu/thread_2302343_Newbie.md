### Post by grahammechanical on 2015-11-09
The version of Ubuntu? And the hardware specification of the machine?

If this is a version of Ubuntu that is out of life then you would be better off doing a fresh install of Ubuntu. Then you can set your own password. Or rather, your girl friend can choose a password.

If she does not have any important data on the machine, then a fresh install would solve this problem. It seems that it is still possible to load the OS and use it, So, download an ISO image and burn it to a DVD disk or USB stick and run a live session and then install Ubuntu from the live session.

[http://www.ubuntu.com/download/desktop](http://www.ubuntu.com/download/desktop)

[http://www.ubuntu.com/download/desktop/burn-a-dvd-on-ubuntu](http://www.ubuntu.com/download/desktop/burn-a-dvd-on-ubuntu)
