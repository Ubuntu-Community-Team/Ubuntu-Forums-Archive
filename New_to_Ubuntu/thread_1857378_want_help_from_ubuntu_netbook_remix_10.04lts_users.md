### Post by anewguy on 2011-10-10
And......11.04 is ubuntu with the unity desktop by default - the same as you see in the netbook remix.  You may want to consider trying 11.04 first, then deciding.  You get the same desktop with the option to use Gnome, etc., with or without effects, if you want to at login.  You'd be all set for the next LTS when it comes out, and if you have any problems we can hopefully get those ironed out.

It's really just your choice - use the latest LTS (10.04) remix, or go with it's universal replacement, 11.04 and be ready for the next LTS (12.04 - April of 2012).
