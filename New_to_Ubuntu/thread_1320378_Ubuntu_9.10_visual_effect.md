### Post by stoogiebuncho on 2009-11-14
Go to the Ubuntu Software Store and search for Gnome-Do.  Install it, and start it from the applications menu.

You will need to go to the options menu and set it to start up when your computer starts, and you want to set it to the "Docky" theme.

You could also try Avant Window Manager.  That's another dock that's in the software store.
