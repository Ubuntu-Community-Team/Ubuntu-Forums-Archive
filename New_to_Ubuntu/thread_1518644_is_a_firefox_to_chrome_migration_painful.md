As for google docs.. I'm not really sure what you can do in Chrome that you can't with Firefox..

### Post by Spiritof76 on 2010-06-26
Google docs is getting better, its fairly compatable with Openoffice. Open office is great for collaboration because it is easy to keep everyone on the same version. and it is consistant cross platform. 

 I like Chrome because it handles memory a little better. but its not a huge difference in performance.. You can still use firefox as a backup. Chrome might have fewer extentions but they seem to work well I use adblock. 

Google has the collaboration thing pretty well mastered. You can form a group register a domain name. maintain a site, do online conferencing using Google wave. and produce documents and reports all using google docs.

I would still back up everything putting all your eggs in one basket might be risky if you're not  real careful.
