### Post by ptn107 on 2013-02-27
If you can plug the macbook into a wired internet connection and do this [from a terminal], it would be ideal:
```
sudo apt-get install b43-fwcutter
```

otherwise...

You need [_b43-fwcutter_]("https://www.dropbox.com/s/5k714wkv7n7tzp0/b43-fwcutter.tar.bz2").  It includes the driver and binary needed to install it.  Download it with another internet capable machine and transfer it to the macbook.

Copy the file into your home folder.  Then open a terminal and do the following:
```

tar xvjf b43-fwcutter.tar.bz2
sudo ./b43-fwcutter -w /lib/firmware wl_apsta.o

```

I have an older legacy driver that may work if this one doesn't.

You may need to restart too.
