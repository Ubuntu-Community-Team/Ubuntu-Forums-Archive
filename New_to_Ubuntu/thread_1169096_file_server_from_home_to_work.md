### Post by inclusivedisjunction on 2009-05-24
If you have a router,you will need to forward port 80 to the computer running the server. Routers have slightly different configuration menus, so I can't explain exactly how to do this, but [PortForward.com]("http://portforward.com/") should have instructions for your router. You will also need your external IP address, such as 71.63.145.213. PortForward will show you this on their main page, as well as some other sites. You would just need to type the IP address into your URL bar to access the server.
