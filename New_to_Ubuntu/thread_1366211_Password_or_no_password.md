### Post by resander on 2009-12-28
This is for a dual-boot PC with Ubuntu 9.10 and Windows. I did not do the installation so don't know how it was done. I observe the following:

1. I don't need to give an Ubuntu password on startup (I just select Ubuntu as OS). Ubuntu does not prompt for a password.

2. When trying to fetch and install the 'Ubuntu restricted extras' to get Flash etc I am asked for a password. I leave the input field blank and just press Enter/Return (from 1 above I assumed there is no password), but then get an authorization error.

What do I need to do?

---

### Post by Bachstelze on 2009-12-28
You can either ask the person who installed Ubuntu on this machine for the password, or boot Ubuntu in recovery mode to reset it. The fact that Ubuntu doesn't prompt you for a password at startup doesn't necessarily mean there isn't one.

---

### Post by mörgæs on 2009-12-28
Your Ubuntu is set up to remember your password by boot, allowing you to enter the system as a regular user. 

If you want to change system settings or do 'dangerous' tasks, then you need to type the password again.
