I agree. The title and the text don't seem to match.........

Anyway. I use a multitude of CDs for "recovery". It depends on the problem. For instance I have a Live CD that boots into FreeDOS for low level disk recovery, I have a custom Ubuntu CD for data recovery and network tools. 

So yes you can download any live CD or any recovery CD. But there is a time and place for each one as there are benefits and drawbacks of each! 

If you are unsure of any of these tools then don't use them, unless you want to play on a machine where the hardware and software don't matter. I warn you, some "recovery" tools can irreversibly change your hardware. Partitioning is one example of this.
