### Post by grahammechanical on 2014-10-12
If you have installed Stellarium, then it would have put an icon in the Dash. Click on the icon and away you go. As far as I can make out, dummy packages are not the actual application but they can be used by the updater utility to help the transition of a package (application) to a new version, especially if it has a name change.

If you look at what that Software Centre is saying about that dummy package you will see that there is already a tick in the box to install Stellarium as an Optional Add-On. So, if you had installed the dummy package it would have installed Stellarium and afterwards you would be able to remove the dummy package if you want to.

Regards.
