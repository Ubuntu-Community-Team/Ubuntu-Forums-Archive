### Post by VCoolio on 2009-03-31
In system > preferences > keyboard shortcuts you can find a list and change them. ctrl+alt+d is default for showing desktop I think. You can also add a show desktop button to your panel (right mouse button on panel > add to panel > show desktop). 
You can google for more shortcuts, but most users will change them to their likings. Also compiz uses lots of shortcuts that can be changed. A 'giant list' would therefore not be very helpful, just search for default shortcuts in ubuntu / gnome. There is a tool called xkeybindings that lets you set shortcuts to whatever command you want.

### Post by rajhanschinmay on 2011-08-02
In newer versions, (ctl + alt + D) may not work.
It may be changed to (Mod4 + D).
Mod4 is the key which opens start menu. It is a key next to ctl and alt on the left hand side in most of the keyboards.

Click (Mod4 + D) to get back the same windows in front.
