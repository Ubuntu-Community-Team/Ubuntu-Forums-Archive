### Post by kanikilu on 2009-04-24
Did you chown and chmod the file itself, or just the home directory?

Should be:
```
sudo chown username /home/username/.dmrc
chmod 644 /home/username/.dmrc
```

More information: [https://help.ubuntu.com/community/dmrcErrors](https://help.ubuntu.com/community/dmrcErrors)
