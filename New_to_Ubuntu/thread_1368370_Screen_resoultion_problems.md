### Post by ST3ALTHPSYCH0 on 2009-12-31
If you have an xorg.conf (which I would think you should after running the dpkg-reconfigure command) you can edit it with this command
```

gksudo gedit /etc/X11/xorg.conf

```

Resolution issues have been a huge thorn in the side for a couple of releases now due to changes in the Xserver. If you have an extra CD or a thumb drive laying around, you could try my lazy man's way from my sig.
