### Post by lisati on 2011-02-14
Welcome to the forum.

Keep in mind that Ubuntu is an operating system, i.e. it can be used to help run your computer. Many of us here use it for our day-to-day stuff as a replacement for MS Windows. Whether or not it can be considered a game is a matter of opinion :)

If you decide to install Ubuntu, don't forget to make a backup of your important data. If you haven't already done so, make you have a set of recovery and restore disks for your existing OS available - many newer computers come preloaded with software that can help you make a set. As much as we like to think that the team has put together a safe package, things can occasionally go wrong, and it's useful to have the tools available to put things right.

Feel free to continue with questions.
