The easiest way is to just type "AdBlock Plus" into the search bar in that window, and click Install for the plugin listed (has an icon of a red stop sign with ABP in it). Same goes for NoScript.

Get Ubuntu Extensions downloads the extensions from the Ubuntu repositories, but you will not receive immediate updates and only a few extensions are available at this time.

Browse all Add-ons takes you to the Firefox Add-ons website to download extensions and themes. It's basically the same thing as typing something into the search bar in the Add-ons window.

### Post by zephyrcat on 2008-09-27
It is really your choice. I am not aware of any reason to use one over the other, except that the Ubuntu ones do not include everything and Firefox's include almomst everything. I did find that Addblock and Noscript are both included in the Ubuntu ones.

The deal is that Ubuntu has added some extensions to the set of software (repository) that you can easily add from Add/Remove in the applications menu.

Basically, take your pick, but be aware that you might not find something in the Ubuntu ones.
