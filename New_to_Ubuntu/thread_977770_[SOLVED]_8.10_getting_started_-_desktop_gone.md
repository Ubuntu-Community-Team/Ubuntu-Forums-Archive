Update:
I think it was a 'folder view' - a particular widget (a functional item), and to re create it I can add a widget to the desktop area.

right click>add widget 
then choose 
'Folder view' (widget)
this puts up the Home folder view for my PC.

I think the original (default) folder view was called Desktop (not sure) and the Home view can be changed to be the Desktop view by clicking on the spanner icon in the sidebar, when it appears that is - or right click on the folder view and choose settings - and then select the folder
'Desktop'.

### Post by Farmer of Bricks on 2009-01-03
In the upper-right hand corner of your screen there should be a faded-out Plasma icon. Hover over this and watch it colorize. Ckick it. Click on "Unlock Widg...". Click upon "Add Widgets". In the new window, click on "Folder View". Close the "Add Widgets" Window. Manipulate the Folder View as you so desire.
