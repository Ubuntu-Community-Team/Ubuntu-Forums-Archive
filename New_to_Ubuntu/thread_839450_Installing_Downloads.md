### Post by DGortze380 on 2008-06-24
In ubuntu, the closest thing to a .exe file (like you would download and double click to install in windows) is a .deb ... a more common format is .tar (tarball), this is source code that you compile for your system.

Before you download and install, you should see if the package/program is available through synaptic.
