### Post by LGCarmona on 2009-03-28
I've just installed Ubuntu on my laptop (Rev 8.10, Intrepid Ibex), on installation, I mistakenly named my laptop "desktop2" instead of "laptop2". Question: How do I change the computer's name from "desktop2" to "laptop2" for there is another computer in my house network named "desktop2" ? Thanks

---

### Post by cariboo on 2009-03-28
To change the name of your computer is really simple, open an Applications-->Accessories-->Terminal and type:

```
sudo hostname laptop2
```

Reboot and the name will be changed.
