### Post by wormser on 2008-06-19
That is a cool site.  The video I clicked on The Israeli Lobby had the youtube logo on it.  I am not sure if youtube hosts all of them.  There is command line tool called youtube-dl.  It looks like you will have to search youtube for it.  Then in the shell use the command in the directory you want it downloaded into.  Example:

```
youtube-dl http://youtube.com/watch?v=rQvrEJ1wWfM&feature=related
```

If you find a way to download directly from the site let me know.
