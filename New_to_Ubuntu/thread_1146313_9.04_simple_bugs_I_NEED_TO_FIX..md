### Post by cariboo on 2009-05-02
It sounds like you didn't complete the update. Open a terminal and type:

```
sudo apt-get -f install
```

then

```
sudo apt-get update && sudo apt-get dist-upgrade
```

As for your firefox problem, you probably need to create a new profile. To do this open a terminal and type:

```
mv .mozilla .mozilla.bak
```

then restart Firefox. You will need to copy your bookmarks file from .mozilla.bak to .mozilla.
