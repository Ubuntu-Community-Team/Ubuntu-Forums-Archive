### Post by SunnyRabbiera on 2008-08-29
Installing java should be simple, technically if you use the latest ubuntu java is already on your system but its not on par with the full java.
to get it go up to system> administration > synaptic
type in your password
after that go to "settings" then to "repositories" and check off all the repositories in both the first and second tabs except the source code ones.
hit refresh
then search for java runtime and install the sun java 6 package, but also remove the open jdk packages.
using synaptic should be easy to figure out, the boxes next to the app name are clickable and it should be easy enough for any newbie to figure out.
