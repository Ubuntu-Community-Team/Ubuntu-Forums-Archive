### Post by pompel9 on 2012-10-17
If you are using 12.04, then I would recommend using 2D instead of 3D. You can change this on the log in page, just above where you input your password.

2D is not as heavy as 3D. Or you can install Ubuntu Tweak, MyUnity and/or unsettings (I think it is spelled that way). You can tweak your system with these apps. I would recommend to install CCSM as well.
