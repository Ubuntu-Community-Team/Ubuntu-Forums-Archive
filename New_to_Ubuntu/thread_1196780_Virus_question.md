I know partitioning hard drive and doing a full install of Ubuntu would be better, but I don't want to get involved with trying to do this with a corrupt Windows system, can forsee numerous headaches!

### Post by Anzan on 2009-06-25
If you're running Wubi on Windows your system, but not Wubi, can still get a variety of viruses. Better to partition. And disable network access for the Windows side.

### Post by aeiah on 2009-06-25
a windows virus could potentially corrupt a wubi installed ubuntu, i think, or at least corrupt the uninstall option available under windows. this is very unlikely though since there's not much motivation for the virus authors.

for stability, you're best installing to seperate partitions as was mentioned.

if all the kids do is browse the net, download stuff and use msn messenger then they could easily do it in ubuntu. pidgin has most msn messenger functionality i believe, and can operate on the msn network.
