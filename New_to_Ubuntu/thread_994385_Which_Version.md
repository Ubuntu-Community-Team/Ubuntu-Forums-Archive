### Post by Coreigh on 2008-11-26
64 is out because you twin PIII's are 32 bit.

I like the LTS releases because they are supported longer, that is 8.04 at this time. Some will suggest 8.04.1 and I don't see why not.

Finally are you wanting a graphical interface (desktop with apps) or is this a file / web / mail server type machine?

For server then yes go with the server install, you can still add a small desktop if you like the GUI. I use Fluxbox, it is fairly small but still "pretty."

But if you want to use this as a web surfing emailing wordprocessing etc. computer then just go with Ubuntu desktop.

If a little of both, then go with the desktop and add the desired server applications via synaptic and/or apt.
