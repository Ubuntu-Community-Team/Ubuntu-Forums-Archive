### Post by grahammechanical on 2013-11-03
I do not understand what the problem is. Every time I log in to Ubuntu it automatically makes a WiFi connection to my router. It does that if I have a set up a wireless connection, of course. I have several installs of Ubuntu and I do not always set them up with a WiFi connection.

Go to System Settings and load the Network utility. Make sure that Wireless is set to on. I am assuming that you are using Ubuntu 13.10. It helps if you tell us the version of Ubuntu that you are using. We can give you more accurate directions.

With Wireless set to ON you should see in the right panel a list of wireless access points. Select your wireless access point and click the little arrow on the right. Now click Settings and under the WiFi Security tab you can put in the pass phrase or wireless key for your router/hub. Under the general tab you can set Ubuntu to automatically connect to that access point. That will do it.

Unless the problem is something else that you do not mention.
