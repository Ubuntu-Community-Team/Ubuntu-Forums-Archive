### Post by Paqman on 2008-05-27
Yep, a Wubi-installed copy of Ubuntu works just like a traditional install. The only difference is that it lives on a virtual partition instead of an actual one.

You can migrate a Wubi-installed copy over to a partition of it's own using [LVPM](http://lubi.sourceforge.net/lvpm.html), btw.

PS: Same goes for dual-boot, too.
