### Post by Double.J on 2014-04-13
Great post texuswriter!

Thought I'd just weigh in with a bit of anecdotal. I've been using the samsung NC10 since 2008 with ubuntu. It has an N270, gma950 graphics and 2gb of ram(I upgraded). I'm adding my experience because I've installed write a few of the netbooks. Now as gor will unity work? Well mine is an early example with hardware that almost any 2008/9 netbook had and I can run unity. However, I do not.

it is laggy at times, and compiz effects aren't a great idea but you can get by performance wise. The real issue is actually unity. The netbook editions of ubuntu back pre 11.x were good for the tiny screens, however i find unity to take up too much space. You have to set the launched to hide and use compiz to shrink it. Also a lot of the menus don't correctly fit the small screens - a lot of alt+click to drag them around. 

I personally prefer lubuntu/xubuntu, but it is worth noting that kubuntu and other kde distros do run on it, though usually in a lower settings mode

Jj
