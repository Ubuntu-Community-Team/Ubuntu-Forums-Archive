### Post by Paqman on 2008-09-10
You don't need to install from the Adobe site, you can get Flash from the repos. The package is called flashplugin-nonfree.

For future reference the package ubuntu-restricted-extras will install Flash, Java, MS fonts and a lot of other useful things on your machine. Very handy for a newly installed system.
