> Hey i was wondering if there is  way to have realplayer 11 be set as the default media player for 8.04. I tried doing it with mplayer, but I messed it up, I dont really even know what I did, and I actually dont like it anyway, the two windows bother me.

I installed realplayer through the .bin file from the website


1. Open a folder containing MP3 or any audio or video file

2. right click on that file

3. select properties

4.  select open with

5. select Real player on the list.


Now if you double click that file, it will be opened by realplayer.

Note: You have to do these steps for all file types that you want to be opened uysing real player
