### Post by TheFu on 2020-09-12
tar.gz apps (basically just a ZIP file) aren't really "installed." They are just copied, if even that.  A non-hacked-together installer would include how to uninstall.  The ./install script was custom made, so you'll need for 
a) find a README or INSTALL file and hope one of those explains how to uninstall.
b) the company who made it to tell you how to remove stuff
c) read the script yourself and figure out what it did; usually it isn't too hard for a properly written script.

Convince a friend to do one of those things for you.

Best to use Canonical supported packages when possible and avoid custom, 1-off, packing methods like ZIP or tgz or tar.gz files.
