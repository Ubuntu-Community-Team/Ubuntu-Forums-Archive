Hi,

I had a problem with entering an editor with gFTP. The problem has been [raised in a previous post but not solved]("http://ubuntuforums.org/showthread.php?t=635906"). I thought I'd post the solution here.

Problem: when entering an editor (typically "gedit") in the Editor field of gFTP > Preferences, you may get error messages when trying to edit a file such as: "Edit: .. is a directory. Cannot edit it."

Solution: you probably entered a trailing space after "gedit" (or the name of your editor). Type "gedit" in the field and make sure no space follows.
