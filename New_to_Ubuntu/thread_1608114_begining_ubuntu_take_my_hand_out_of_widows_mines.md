### Post by Mark Phelps on 2010-11-08
Don't mean to be unduly critical ... but ... every time you post, it's something different from the previous post.  This jumping around from topic to topic will not get you help.

Some folks browse these forums looking for post titles including problems they know how to solve.  Filling post after post with different stuff is only hurting your chances of getting any help -- as you have already seen by the lack of responses to this thread.

If you have a specific problem to address, then post on THAT. 

As to your question about .exe files -- asking about running those contradicts the very title of this thread!  You're not taking your hand "out of windows mines" if you're still concerned about running MS Windows apps in Ubuntu -- you're only making matters worse.  Why?  Because you'll have to install an emulation packages (Wine or something similar) and then you'll have to spend a lot of time tweaking and configuring that package to work -- for every single individual MS Windows app you want to run.  For someone who's made a point of mentioning how little time you have, trying to get MS Windows apps to run in Wine is NOT a good idea.
