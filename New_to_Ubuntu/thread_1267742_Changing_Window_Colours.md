### Post by stinkeye on 2009-09-16
> **MattDobson said:**
> This might be a simple thing but I can't for the life of me work out how to change the colours to something other than standard grey.
You can download themes here
[http://www.gnome-look.org/]("http://www.gnome-look.org/")
or you can use gnome-color-chooser available through synaptic package manager, which will allow you to change a lot of colors.
To install themes just drag and drop the compressed file into the appearances window.
Sometimes there are other themes in the file so you have to unpack first to get to the gtk compressed file.
