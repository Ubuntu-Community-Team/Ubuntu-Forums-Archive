### Post by The Cog on 2009-08-20
It is responsible for finding wireless networks and allowing you to connect to them. It places a small icon in the system tray that you can click for the list of networks.

It probably also deals with a wired network if connected, but I'm not certain about that.

---

### Post by 3rdalbum on 2009-08-20
Network Manager handles the discovery and connection of wired and wireless networks, can "roam" between devices as connectivity changes; can dial 3G modems, can connect to VPNs and can even dial regular modems.

It also informs client programs of the network status.

Better to keep Network Manager running in case any programs need it. If you only use a wired connection, Network Manager will not get in your way.
