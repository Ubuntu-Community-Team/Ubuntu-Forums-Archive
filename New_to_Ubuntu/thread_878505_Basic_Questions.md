### Post by tamoneya on 2008-08-03
1. Yes use wine.
2. Yes.  It has progressed a lot recently and I think both of your games should run very well
3.  No you dont do it that way.  You dont install drivers in wine.  You install linux drivers in Ubuntu and then wine interfaces through linux with all of your hardware.  It is actually easier this way provided you dont have really obscure hardware.
