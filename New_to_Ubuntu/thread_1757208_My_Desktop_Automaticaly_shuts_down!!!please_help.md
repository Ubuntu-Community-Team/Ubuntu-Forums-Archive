### Post by jtarin on 2011-05-13
Start the application in a terminal and watch for errors. If it shuts down again notice the time and then on reboot go to the Control Panel and click on "Log File Viewer" under the "System" category and look by date and time for anything innocuous. You might need to do it several times.
