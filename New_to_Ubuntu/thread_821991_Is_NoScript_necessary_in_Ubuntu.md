### Post by ibuclaw on 2008-06-07
In theory, yes and no...

No, Ubuntu isn't **as** vulnerable as Windows, (the fact that there are **NO Viruses/Spyware** in Linux accounts 99% of this truth).

But it is always good to be cautious when using cross-platform applications, such as Adobe Flash or Java. As, in theory a vulnerability in Windows could work in Linux too.
Albeit, not as "well" due to privilege restrictions, but it could "try" to run all the same.

And besides, safe from Viruses/Spyware does not mean "Safe from explicit content" (such as images).

It is an optional feature, but if you feel safer using it. There is no harm in that.

### Post by aysiu on 2008-06-07
NoScript isn't really *necessary* unless you're visiting shady sites on a regular basis.

And if you use it on Windows, I think it's a good idea to use on Ubuntu as well. Javascript exploits (before they're patched by Mozilla) can sometimes allow arbitrary code to be run.

For the most part, though, Mozilla and Ubuntu are pretty quick about patching flaws when they appear.
