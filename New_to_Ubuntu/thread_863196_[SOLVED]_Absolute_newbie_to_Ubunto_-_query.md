### Post by bhadotia on 2008-07-18
I think you can run ubuntu 8.04 with 256MB RAM but it would be better to run Xbuntu.
If you are using the live CD then you might not be able to install ubuntu from the live environment because the minimum requirement for live CD is 384 MB of RAM (I think).
So if you want to install ubuntu from the live cd use 'install ubuntu' option instead of the ' try ubuntu without any change to your computer' option .


Also , If you decide to install xubuntu download the alternate CD instead of the live CD (which I think you have for ubuntu at present). The alternete CD won't give a beautiful installation experiince but will work better than ubuntu Live CD.

You can also try other lighter linux distributions out there suited to your hardware.
