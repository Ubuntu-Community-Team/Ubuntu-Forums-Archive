### Post by snowpine on 2008-07-30
> **jwriter said:**
> 1) Is there a small distribution option?  In other words, is there a live CD option that loads quickly and runs in RAM?


There are distros such as DSL that are designed to run entirely in RAM, and Ubuntu is not one of them (by design)--it is simply impossible to cache a 700mb iso into RAM without driving up the minimum ram requirement astronomically (people already complain about 384mb). The Live CD runs very well and does not install anything to your hard drive, but you will notice when you open a new application that it loads from the CD.

There is indeed a minimal version of Ubuntu, though it is not a live cd. It is a 10mb iso that installs the command line interface by pulling packages off the internet, then you can build it up with whatever other packages/applications you like. You can read about it here: [https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD) and here: [http://www.psychocats.net/ubuntu/minimal](http://www.psychocats.net/ubuntu/minimal)

Some might also argue that Xubuntu is the "small version" of Ubuntu as it's hardware requirements are about half.
