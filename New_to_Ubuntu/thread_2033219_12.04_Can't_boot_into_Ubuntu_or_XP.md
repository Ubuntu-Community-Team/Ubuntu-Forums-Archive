### Post by drs305 on 2012-07-25
auglove3rd,

Welcome to the Ubuntu Forums.

If you hold down the SHIFT key during the first part of the boot does the GRUB menu appear?

If so, press 'e' to edit the first item. Cursor to the end of the line starting with 'linux'.

Use the backspace/del key to remove the line back to "ro". Add *nomodeset* and the CTRL-x or F10 and see if it boots to the Ubuntu Desktop. Let us know what happens.

I'm also a big fan of Boot Repair.
