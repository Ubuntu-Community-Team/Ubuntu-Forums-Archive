### Post by magdelaine on 2012-06-22
I installed Ubuntu as a dual boot system with Windows over a year ago on my Dell Mini 10 netbook. Although I preferred the Ubuntu environment to Windows, I gradually stopped using it because of two consistent problems. First is that it would get hot, and I mean HOT when it was running Ubuntu. Second is that it didn't seem to suspend properly as I would shut the lid and come back later to find the thing cooking in its own juices. I was confounded when I searched for solutions to this issue as digging into Linux can be quite a project, so I just stopped using the Netbook in Ubuntu.

I've decided now, however, that since it's probably a software rather than a hardware issue, it is a big enough improvement over Windows to make tracking down a solution worth it....I just don't know where the best place to start is. 

I'm currently upgrading the Mini to the latest version of Ubuntu and I should be ready to dig into the problem shortly. I'm also open to trying a different distro if it would work better on a Dell netbook. Any ideas for me?

Thanks!
