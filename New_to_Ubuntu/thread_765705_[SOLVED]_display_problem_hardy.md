### Post by tjwoosta on 2008-04-25
well im certainly no ubuntu expert but i was having problems getting my display set correctly when i was using gutsy.

i think it should be the same way in hardy

but anyway this is how i fixed it

first thing you will want to do is make a copy of your existing xorg.conf like this (you may need to use root for this, im not sure)
```
cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
```
now if anything goes wrong with your configuration all you have to do is get to a command line and do this
```
sudo mv /etc/X11/xorg.conf.backup /etc/X11/xorg.conf
```
ok now your ready to edit your config, so first thing you gotta do is stop Gnome Display Manager 
(this should stop you desktop and take you to a command line, so dont be afraid if you desktop dissapears)   
```
sudo /etc/init.d/gdm stop
```
then when it stops you will reconfigure X
```
sudo dpkg-reconfigure xserver-xorg
```
then once you have manually configured everything you do this to start gnome again
```
sudo /etc/init.d/gdm start
```
all should be well now.. hopefully

I actually had to reconfigure 3 times before i got mine set up right, and i still had to manually add the ("TVOUT" ignore) part that i  mentioned in the first post to get stuff alligned right

but if anything goes wrong just remember to get to a command line and restore your backup xorg.conf like i described above and all will go back to the way it was
