### Post by Impavidus on 2014-01-02
There is no standard way of installing tarballs. It depends on what kind of stuff you want to install and how the provider has packaged it. In this case it's a theme. You can install it system-wide with```
cd /usr/share/themes
sudo tar -zxvf ~/Downloads/74813-Overglossed.tar.gz
```You can also move your ~/Downloads/Overgloss directory there, but then you must recursively change ownership to root afterwards. Alternatively, to install just for yourself, move it to your own .themes directory, which you may have to create:```
mkdir ~/.themes
mv ~/Downloads/Overgloss ~/.themes/
```Next select the theme in the settings of your window manager. It has to be compatible with the window manager though.

Other software distributed in tarballs may require totally different ways of installation. Usually an INSTALL or README file is provided.
