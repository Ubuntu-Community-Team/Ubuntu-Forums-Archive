### Post by Ben Crisford on 2009-03-19
gnome-look.org is the place to go really...

Or you may want to try using KDE instead.  It is very easy to install, just type this into terminal:

```
sudo apt-apt get install kubuntu-desktop
```

Then to switch between gnome and kde:

[B][I]Log Out> Options> Change Session> KDE
Log Out> Options> Change Session> KDE[/I][/B]
