### Post by Terl on 2009-11-30
Using synaptic package manager, search for ubuntu-restricted-extras.  Install it.  That will get you flash, some fonts, codecs, and java runtime as well.

### Post by almufadado on 2009-11-30
Goto System -> administration -> Synaptic package manager 

 Enter root password for Synaptic package manager be opened

In the field "quick search" write  flashplugin

select this package :

flashplugin-installer
Adobe Flash Player plugin installer

And you are on ...
