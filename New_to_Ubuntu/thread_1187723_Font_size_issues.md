### Post by Therion on 2009-06-14
For starters, in Firefox, go to Edit/Preferences "Content" tab.  Under the Fonts and Colors section click on the "Advanced" button and then the "Minimum font size button" to set the fonts to whatever size you want.  Then clear the checkbox that allows web pages to select their own fonts.

Does that help?
