### Post by grahammechanical on 2016-01-10
Ubuntu will install into about 7 GB of disk space but that is a default install. Once we start adding applications and collecting data a Ubuntu partition that size will very quickly fill up.

Linux will put all program files & libraries into the root folder ( / ) and user configuration files and user created documents into the home folder.

Some of us use a separate partition for the /home folder. We think it is safer because we can re-install Ubuntu and so long as we do not format the /home partition we will not lose our data or configuration files. Some of us keep our data on its very own partition. That means there is even less chance of formatting our data by accident when we do a re-install.

I can only guess, but I think that, in your case, a 50 - 60 GB Ubuntu partition even with separate home & data partitions will soon fill up. We can always resize the partition. 

The software centre gives us a download size and an install size for application.
