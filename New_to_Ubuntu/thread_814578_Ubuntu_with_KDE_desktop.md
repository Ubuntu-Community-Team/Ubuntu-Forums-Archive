### Post by jordanmthomas on 2008-05-31
KDE4 is definitely not stable, and is only just becoming usable even.
All that happens if you install KDE4 is that you now have KDE4 installed as well as Gnome.  It may set up kdm instead of gdm for your login, but everything you have will still be there.

### Post by FuturePilot on 2008-05-31
When you install KDE you get an option which desktop you want to log into at the login screen. It will be under the Session menu. So no, it doesn't really turn Ubuntu into Kubuntu. You can choose which you want to log into. But regardless of which you log into you can use Gnome apps in KDE and KDE apps in Gnome.

Technically according to the version numbers KDE 4 is stable. But I have found it still quite buggy and other's opinions may vary.
