### Post by NikoC on 2008-06-14
System > Preferences > Advanced Desktop Effects Settings:
General options: Desktop size, Horizontal Desktop size --> make sure this is set to 3 or higher
Desktop: enable cube (here you can also set transparancy) and enable rotate cube

Have fun playing around with (endless combination) the different options.

### Post by Sealbhach on 2008-06-14
If you can't find the Advanced Desktop Effects Settings in the menu then you need to install it.
```

sudo apt-get install compizconfig-settings-manager
```

Then you'll find it in the menu.

Once you open in it, click on Desktop, then tick the Cube. Then double-click on the Cube and select the tab "Appearance". The background is called the Skydome - you can choose whatever image you want it to be.

The go the the main menu on the left, select General, then Desktop Size, and make sure the Horizontal Desktop Size = 4.

NB: You will need to have the Extra Desktop Effects enable in the Appearance Menu.
