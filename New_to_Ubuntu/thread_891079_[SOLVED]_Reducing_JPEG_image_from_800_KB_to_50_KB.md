### Post by coolbrook on 2008-08-15
In GIMP open your file and then **save as**.. choose a new file name and then save it (as a jpeg under All Images.)  A window will pop up that will allow you to reduce (and preview) the file size/quality without compromising much of a difference in the appearance of the image.
