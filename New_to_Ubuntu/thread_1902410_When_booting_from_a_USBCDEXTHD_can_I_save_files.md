### Post by Henkdroid on 2011-12-31
If you save the file onto the hard drive of your computer, or a different pen drive (if you are booting from one) then the files will remain. If you just save them then they will be saved to RAM and deleted when closed. The computer will run a bit slower because the read/write speeds will be lower than a hard disk in your computer.
