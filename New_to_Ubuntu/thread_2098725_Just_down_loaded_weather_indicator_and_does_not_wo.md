### Post by superDave972 on 2012-12-27
I had the same problems.

Apparently, Google stopped supporting the application. Switch the source from Google to Yahoo. I think Yahoo still supports the application in some aspects.

If that doesn't work, another member suggested to install the following packages: python3, python3-gi, python3-dbus and python3-cairo.

To install, open a terminal and enter 'sudo apt-get install python3'
Obviously, change "python3" for each package named above.

Check these threads for more info:
[Here]("http://ubuntuforums.org/showthread.php?t=2096510&highlight=weather+indicator")
[and here]("http://ubuntuforums.org/showthread.php?t=2093325&highlight=weather+indicator")
