### Post by Roger Allott on 2009-11-02
I'm running Xubuntu 9.10 karmic on my second PC.

I want to share some files from my main Ubuntu system, which ordinarily runs on my first PC by booting from a USB HDD.

So, I boot up Xubuntu and connect my USB HDD via a powered hub.

The behaviour I've seen in Ubuntu is that the drive appears in the list of 'Places', you click on it, it mounts and one can the browse around the disk as normal.

But that's not happening in Xubuntu, it seems. The USB HDD doesn't seem to be there!

Is this normal behaviour for Xubuntu? If so, how does one mount a USB HDD so it's contents can be seen?

---

### Post by Roger Allott on 2009-11-02
I've realised what the problem was. The USB cable seems to have been too long. Once I used a shorter cable, Xubuntu sees my USB HDDs quite happily.
