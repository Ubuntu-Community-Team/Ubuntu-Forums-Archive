### Post by yancek on 2017-11-30
If you are going to be writing to the disk from windows you will need to use a windows filesystem format as a default windows is incapable of reading or writing to a Linux filesystem whereas most any Linux system will be able to read as well as write to a windows FAT32 or ntfs filesystem.
