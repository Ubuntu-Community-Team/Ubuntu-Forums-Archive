### Post by mocoloco on 2009-03-21
It's not actually a separate panel applet but is running in the system tray, which is confusing since it's named "network manager applet".  I know you can go to System -> Preferences -> Session and disable the network manager applet there.  As long as the regular network manager daemon runs I think your network will still connect fine.  Let me know because I'm curious.
