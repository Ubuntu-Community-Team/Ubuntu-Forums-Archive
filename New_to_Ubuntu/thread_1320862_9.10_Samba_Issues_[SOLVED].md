### Post by Ucase on 2009-11-09
I think I had the same issues as most people when it came to SAMBA and the latest release.  I could not see any SMB shares from my Macs, PC's, or any live media I booted to with my laptop.  Everything checked out fine as far as settings go and following the great suggestions on this board yielded no positive results.

I decided to install SWAT to see if that would give me some relief.

Within SWAT, I set the Interfaces to "DEFAULT" which is blank and once I rebooted, all was well, I could see/browse all my SMB shares from any other OS and permissions seemed to be accurate.

I don't know if it was the SWAT install or the subsequent xinetd install which finally corrected the problem but things are running relatively smooth.  One thing I did notice is my ProFTPd server is no longer working, I will have to troubleshoot that a bit to see what is happening.

Thank you to all the people with their great posts and suggestions, while they didn't work, they did give me a greater understanding of how to troubleshoot this in the future.
