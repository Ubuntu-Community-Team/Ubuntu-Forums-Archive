### Post by sgosnell on 2009-02-25
For the panel problem:

Right-click on the top panel, and select Add to Panel.  Scroll to the bottom and select either of the Windows selections.  One gives one icon, and clicking on it gives a list of open windows to select from.  The other gives individual icons for each open window.  You can put these on either the top or bottom panel, or both.  I did what you're trying, removed the bottom panel entirely and put everything on the top.  You could also put everything on the bottom if you wanted, in fact you can just drag the top panel to the bottom.  It's all very configurable.  There isn't room on my EEE screen for two panels, so I have only the top panel, set to autohide.
