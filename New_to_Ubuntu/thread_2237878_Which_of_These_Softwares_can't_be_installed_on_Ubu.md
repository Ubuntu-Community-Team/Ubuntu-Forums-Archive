### Post by buzzingrobot on 2014-08-04
Wine is a batch of software components that translates calls to Windows libraries made by Windows software into calls to Linux libraries. It is not a magic bullet that allows any and all Windows software to run transparently on Linux.

Check here -- [https://appdb.winehq.org/](https://appdb.winehq.org/) -- to get an indication of how well Windows-only programs run in Wine before making a commitment.

### Post by Bucky Ball on 2014-08-04
You are much better to look for native alternatives to Windows issues. If you want to use a lot of Windows programs, though, possibly best to stick with Windows or dual-boot. 

Gimp is an open-source replacement for Photoshop. FTP software is available (you're not specific what you want to use it for, but I use Filezilla for FTP).

Please be aware that Linux/Ubuntu is not a drop in replacement for Windows. Most Windows apps are not going to run natively in Ubuntu. Some may work in Wine, with varying degrees of success, but by no means all. Perhaps have a read of this:

[http://linux.oneandoneis2.org/LNW.htm](http://linux.oneandoneis2.org/LNW.htm)

You can boot from a Live disk/USB and give it a try to see what you think without installing or altering your hard drive or current setup. Good luck.

### Post by Rob Sayer on 2014-08-05
On running Photoshop in linux using Wine:

[http://appdb.winehq.org/appview.php?appId=17](http://appdb.winehq.org/appview.php?appId=17)

which isn't encouraging.  With Wine anything rated less than Gold it's almost certainly going to give you problems, and note how up and down the rating goes with new releases.  Wine isn't reliable.  Best to keep at least one Windows partition in a dual boot setup or use a VM.

GIMP is a perfectly acceptable substitute for photoshop if it's for casual/hobby use.  Not if you want to publish professionally.  It uses a proprietary file format used by publishers, and it doesn't have Pantone color support which is also proprietary.  Ie. not open source.  I wouldn't hold my breath waiting for linux support there.

I use Chrome as my default browser in ubuntu now because I hate the new firefox releases, and the flash support in Chrome is better in linux.
