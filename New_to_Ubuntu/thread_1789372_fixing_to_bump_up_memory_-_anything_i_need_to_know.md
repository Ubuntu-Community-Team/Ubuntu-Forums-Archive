### Post by CharlesA on 2011-06-23
Might have to bump up the swap partition if you use suspend/hibernate, since the swap partition needs to be as large as the amount of memory in order for that feature to work.

---

### Post by LukeMorrison on 2011-06-23
There's nothing extra you need to know as far as running with extra RAM.  The one thing you may want to think about is the size of your swap partition, but with that amount of RAM, it's probably not an issue.
