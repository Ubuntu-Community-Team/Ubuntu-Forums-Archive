### Post by howefield on 2012-03-13
I use..

```
sudo shutdown -r now
```

Alternatively, typing

```
exit
```

will take you back into the program, cancel the next few screens to get back to the numeric options.

The Num Lock key isn't activated by default when booting with the Clonezilla Live CD, so you have to remember to press it before selecting your poweroff option or use the numbers along the top of your keyboard.
