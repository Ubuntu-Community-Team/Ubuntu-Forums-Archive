### Post by s1baker on 2012-08-15
If you installed your ubuntu off of a CD, you can boot off that CD and run the "try Ubuntu" selection to get Ubuntu running independent of your hard drive. After ubuntu loads off of the CD, bring up a terminal ( probably Control , Alternate, T ), or you can find it in a menu,
and type gparted in the terminal. It is a simple graphics programs with a menu selection at the top. Select "Partition & check", if it finds errors, let it try to fix them. But you have to run gparted from a CD so that it can be independent from the HD to look at and fix your partitions if they need it.

If you don't have a CD, you can go to the ubuntu website and download ubuntu and burn it to a CD.
[http://www.ubuntu.com/ubuntu]("http://www.ubuntu.com/ubuntu")
