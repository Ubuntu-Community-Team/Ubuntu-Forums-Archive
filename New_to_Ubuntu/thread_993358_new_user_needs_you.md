### Post by doctorbighands on 2008-11-25
Personally, I'm of the "if it ain't broke, don't fix it" mentality. I also know what it's like to have new things to play with, though.

If memory serves, there isn't TOO much difference that the average user will notice between 7.04 and 7.10. If you want to upgrade, I don't imagine it would hurt anything. You could start the upgrade process right before bed, and it ought to be done when you wake up. An alternative would be to go to ubuntu.com and order a free installation CD of version 8.10. That way, you'd be as up-to-date as possible, and you won't have to worry about waiting for a download.

Welcome to the world of Linux! It's beautiful here. :)
