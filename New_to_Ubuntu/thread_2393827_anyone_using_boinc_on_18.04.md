### Post by oldos2er on 2018-06-08
I rarely use any GUI package manager anymore, just apt in terminal: ```
sudo apt update && sudo apt install boinc boincmgr
``` *boincmgr* is the command to run the graphical Boinc Manager, where you can set all your preferences, and see the tasks it's running, etc.
