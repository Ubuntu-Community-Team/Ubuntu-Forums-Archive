### Post by forrestcupp on 2012-05-11
Try out the LiveCD for the latest version of Lubuntu. It's a lot lighter weight, and you'll have a better chance of it working on an old computer. If you boot into the LiveCD before installing it, you'll know for sure if it works. At worst, you're out some time and a blank disc.

But you definitely shouldn't be reinstalling an unsupported version that's that old.
