### Post by stoogiebuncho on 2009-11-07
Installing fonts is pretty easy in Karmic.  Just download the font file (usually a .ttf file), and double click it.  There will be a button that says "Install".

If you're looking for a place to find free fonts, there's a bunch at [http://www.dafont.com]("http://www.dafont.com").
