### Post by RedLion2074 on 2009-09-09
Hello [MattDobson]("http://art.ubuntuforums.org/member.php?u=743998").
About this issue, do you often use the (great) "block content" function of opera? If yes, browse the opera menu going to:

[COLOR=Teal]**Tools -> Advanced -> Blocked content.**
[/COLOR]
Then check in the blocked content list if you don't have this entry:

**[COLOR=Red]" [http://*](http://*) "[/COLOR].**

If yes, remove it. Of course, you have guessed this entry blocks all possible sites you could want to visit. It worked for me. To finish, this issue can also occurs for windows users. Hope it will help. Bye.:)
