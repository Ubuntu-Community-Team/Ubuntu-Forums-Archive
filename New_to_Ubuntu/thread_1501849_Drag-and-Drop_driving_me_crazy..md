### Post by diablo75 on 2010-06-05
I trained myself to copy and paste using keyboard shortcuts to drag and drop when it's possible. You might have already known these shortcuts but they work in a lot of different programs.

CTRL-C = Copy
CTRL-X = Cut
CTRL-V = Paste

CTRL-LeftClick = Select/Unselect individual files to create a multi-file selection to be copied/cut.

Shift=LeftClick = Select a range of files in a row instead of individually.

And if you ever find yourself trying to paste a command into a Terminal window, be sure to add the shift key to the above combination (Ctrl-Shift-C to copy, Ctrl-Shift-V to paste, etc.)
