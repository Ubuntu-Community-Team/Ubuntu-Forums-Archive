### Post by JamButty on 2010-09-20
All the above is right and also one of the more essential uses is for programs like Gimp that open up multiple windows for one task. It is a real pain to deal with 3 or 4 other programs' windows in the same workspace in an environment like that. Workspaces lets you isolate windows like that (or any group of windows you feel are related, like a media player and a file browser for example).

One neat feature is that the workspaces function as one 4x wide screen divided up - if a window is a bit off to one side of the workspace, it also appears in the adjoining workspace.

### Post by JamButty on 2010-09-20
Sure. It is not meant as a way of fixing all program usage in stone, just to organize it as you want to as you use it. If you don't need it, all windows accumulate in the first workspace. If you need it, you either click on that workspace and open a new program, or you right click on an currently window tab and choose another workspace to move that window to.
