In gpodder click on the  Podcast > Additional components there is a list of extra  helper programs gpodder doesn't install by default.  You can install them from synaptic by finding the gpodder program right clicking on it and looking at recommended and suggested for installation options. There are two that are not listed vorbis-tools and lame. I also had to setup my ipod for the first time on ubuntu using gtkpod.
