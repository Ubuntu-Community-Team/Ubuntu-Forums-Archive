### Post by powel212 on 2009-06-11
I run Jaunty and I sysnc my ipod in iTunes in a virtualbox running XP.

Sun has a new Virtual box that runs so well and is so easy to set up. Just install Sun Virtualbox, then install xp or another windows flavor inside it and then itunes inside that. No rebooting, no trying to get your playlists exported and imported again, no messing around in general.

I tried a bunch of different solutions for this ipod sync problem but none of them were satisfactory until I tried the virtualbox. It saved my life.
