### Post by Lisiano on 2011-10-16
1) Do you mean My Computer, Home, Network Servers and Trash? These can be added using [Gnome Tweak Tool (Click to open Ubuntu Software Center)](apt://gnome-tweak-tool)
2) The themes need to be installed, I don't know from where but USC and Synaptic should have some themes. The fonts and icons are tweaked in [Gnome Tweak Tool (Click to open Ubuntu Software Center)](apt://gnome-tweak-tool)
3) Probably not possible since in Unity, if you maximize the window, the title bar merges with the taskbar/menu. Probably should be possible in Gnome Fallback Though.
