I have XPAbiword is not a program for scanning documents. It is a word processor and is used for creating (writing) and editing text documents.

To scan a document, you require a scanner and, because you mention you only use Windows XP, you will need to make sure that your scanner has the correct drivers. 

Also, the scanner must have a program that allows scanned documents be saved as 'OCR'. Otherwise, you will not be able to do much with your scanned document.

I hope that made sense.
