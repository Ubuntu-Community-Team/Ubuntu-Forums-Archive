### Post by audiomick on 2013-01-16
One of those Fn keys may be a switch for the camera. Both of the machines I have that have a webcam have such a switch.

If you can locate such a switch, do
```
lsusb
```

in a terminal.

See if you can see something in that list that might be your camera.
If it is not there, switch the switch once, and do lsusb again. See if the camera is there now.

If it is, try cheese again.

Even if you can't see a switch, do the lsusb and see if you can see the camera.
