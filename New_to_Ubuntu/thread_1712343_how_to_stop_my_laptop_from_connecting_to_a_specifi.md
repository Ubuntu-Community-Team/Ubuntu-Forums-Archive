### Post by sallam on 2011-03-22
Greetings

Sometime ago, my connection was down, so I connected to a neighbor's wireless connection.
The problem is that, ever since I did that, each time I start my laptop it connects to that neighbor's connection automatically.
How can I stop this please?

---

### Post by Quackers on 2011-03-22
Uncheck the "connect automatically" box in Edit Connections for that particular network.

---

### Post by sallam on 2011-03-22
Got it:
System > Administations > Network Tools > Network Device: wireless interface > configure > wireless > (select the connection then: edit) > untick: connect automatically.
