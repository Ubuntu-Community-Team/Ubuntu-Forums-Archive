### Post by Chesamo on 2010-05-21
Wubi is different from the standard Ubuntu install, in that it installs Ubuntu into your OS (as ubunterooster mentioned). The data may still be there on your D: partition, but it's inaccessible. If you want to save Ubuntu and still be able to reformat Windows on occasion, you'll have to set up a dual-boot system, which sets up two *separate* Ubuntu and Windows installs.

[https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)
