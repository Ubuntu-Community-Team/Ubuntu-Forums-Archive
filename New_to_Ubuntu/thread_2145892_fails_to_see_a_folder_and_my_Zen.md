### Post by Merrattic on 2013-05-17
Your folder ".:OBEJRZANE:." becomes hidden on linux file system because of the "." at the beginning.

Either remove the "." or press CTRL+h to see hidden files. You should be able to tick a box in your file manager to always be able to see hidden files, sounds like this is already done in XP ;)
