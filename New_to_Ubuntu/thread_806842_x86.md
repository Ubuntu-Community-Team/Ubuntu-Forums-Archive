### Post by Joeb454 on 2008-05-25
x86 is 32 bit. Most systems will be 32 bit systems.

If you downloaded the desktop edition then I'm 99.99% sure that you will have an x86 system :)

If you really want to check, open a terminal and run ```
uname -m
``` that will tell you whether you have x86 (32 bit) or x86_64 (64 bit)
