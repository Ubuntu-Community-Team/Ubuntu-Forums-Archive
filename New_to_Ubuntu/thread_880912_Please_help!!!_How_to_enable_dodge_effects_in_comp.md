### Post by gerben1 on 2008-08-05
By default the the Fade option is selected in:

system > preferences > advanced Desktops effects settings > animation (under effects) > Focus animation 

Select the line that has Fade as focus effect and click "edit" in the dialog you can choose which effect you want choose dodge, and that's it.

** edit **
While in that edit-dialog, you probably also want to make the duration a little longer, something like 300 or 400 ms looks a lot nicer than the very fast default.

### Post by gerben1 on 2008-08-10
> **SandyM said:**
> If u have seen dodge effect video at you tube u can see that open windows can be inverted. How can it be done

I assume you mean the part where a window flips around and you see another window on the backside.

This is part of the "Group and Tab windows" functions. You can just put a check mark in the box in front of "Group and Tab windows"in CompizConfig settings manager. The feature you are asking about is already on by default.

Here's an example of how it works:

- first you select a window window and add it to your "selection" by pressing: 
<supper> + s
- then select a second window and also add this one to your selection of windows, again by selecting the window and pressing:
<super> + s 
- add a few more windows to your selection if you wish, in the same way
- now turn this selection of windows into a group by pressing
<super> + g
- now "tab" the windows together by pressing:
<super> + t

Now you will have a stack of the windows that you selected and grouped. 

You can cycle through the windows  by clicking in the title bar and selecting one of the small window-thumbs to select another window from the group. If you select another window in this way, you will see the effect you were looking for.

see here:
[http://wiki.compiz-fusion.org/Plugins/Group](http://wiki.compiz-fusion.org/Plugins/Group)
