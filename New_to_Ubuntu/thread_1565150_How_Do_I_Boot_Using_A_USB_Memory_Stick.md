### Post by Pough913 on 2010-08-31
In side your bios set it to boot USB diskette key first. then restart with the flash drive plugged in. 
And please refer to [https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick) for more information on how to install Ubuntu to a flash drive/ memory stick.

### Post by C.S.Cameron on 2010-08-31
Boot the Live CD, plug in the USB stick and run Startup Disk Creator.
If you want to save settings, install programs, etc choose Stored in reserved extra space.
