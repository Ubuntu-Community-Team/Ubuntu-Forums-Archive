### Post by wdaniels on 2008-07-02
Yes, this works fine. Just install the relevant package(s) in the usual way. The only difference is that KDE apps use a different set of widgets than Gnome apps, so they may look a little out of place. Also it tends to consume slightly more resources because you are loading sets of libraries from two different desktops at the same time. But many people do mix Gnome and KDE apps...you shouldn't have any trouble.

### Post by sayakb on 2008-07-02
It should work out well except that you may need to install a big chunk of KDE libraries. Also if the KDE program crashes, only the KDE apps will be closed w/o altering any Gnome component. As a personal experience, Ive seen that KDE apps crash more often when used with Gnome, but thats not a point to worry, really ;)
