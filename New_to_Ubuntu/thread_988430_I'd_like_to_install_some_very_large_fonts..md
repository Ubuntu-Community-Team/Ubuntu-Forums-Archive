### Post by Kellemora on 2008-11-20
Hi John

Assuming you have msttcorefonts installed and not using rasterized fonts.

To get font sizes larger than shown in the drop down list, you just simply TYPE the size you want into the window that displays the font size.

EG:  Put a ZERO behind the 12 shown there and you have size 120.

That's all there is to it!

It's best in the larger sizes to keep your font size in multiples of 12

EG 84, 96, 108, 120, 132, etc. to prevent ragged edges when printing.
