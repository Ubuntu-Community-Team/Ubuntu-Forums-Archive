### Post by C.S.Cameron on 2010-07-20
Sounds like a loaded question.

Ubuntu should be fine as a full install or as a Live install with a big casper-rw partition on the 8GB.

Puppy is probably more suitable on the 4GB as it's small size allows for more persistence. It also runs in RAM and should be faster from USB drive.

Vuppy and NimbleX come with Vbox so if you put it on the 8GB you could also install XP.
I understand that you might run into some Windows only programs at college.
