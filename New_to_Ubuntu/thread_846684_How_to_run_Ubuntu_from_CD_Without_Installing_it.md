### Post by avtolle on 2008-07-01
The iso you downloaded and burnt as an image to CD was the Live (Desktop) one, correct? If so, put the CD into the drive, and restart your system. First, though, be sure your BIOS is set to boot from the CD drive first. 

If you have the alternate install iso or the server iso, these are for installation only. You need the Live CD.
