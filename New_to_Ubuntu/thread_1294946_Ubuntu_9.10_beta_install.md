### Post by cljbville on 2009-10-18
I had a little trouble with my first 9.10 install too.  I did a clean install on a desktop PC and also had partition trouble after reboot.  So what I did was use the CD to do another  install.  Then once it brought me to the desktop, the first thing I did was go to System | Administration | Update Manager.  It had plenty to update.  For some reason it complained it only wanted to do a partial install, so I let it do the partial and have had no trouble with it since.  I just got a brand new laptop, a HP Pavilion dv6t, and I did the same process on it and it too is working great!
