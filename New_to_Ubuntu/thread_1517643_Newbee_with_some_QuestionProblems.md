### Post by mk1w86 on 2010-06-25
I can answer a couple of your questions.  I assume you have Kubuntu installed and are using Gnome and KDE.

> 1) What Is a difference of PURE GNOME and UBUNTU 10.4, How I can install UBUNTU 10.4 on PURE GNOME (As I do not see option to check social networking Facebook/Twitter/Orkut)


If you want stock Gnome you will probably have to compile it from source unless anyone knows any other way.

> 3) I have downloaded one icon theme and trying to install it. After hitting apply, I do not see any changes happning


You might have to enable it in System > Preferences > Appearance > Theme tab (if it is a Gnome theme)

> 4) How to change the change system sound for various actions. I would like to add MAC sounds as per Mac4lin theme. 


Go to System > Preferences > Sound > Sound Effects tab

> 5) How to add/remove applications. I do not see any list coming in ADD/REMOVE panel in system settings. 


You can use the Ubuntu Software Centre (recommended if you're new to Linux) which is under the Applications menu or if you want to install individual packages use Synaptic under System > Administration.
