### Post by hermes0710 on 2008-05-12
Follow these only if you use ATI or Nvidia g.card

Go to "System>Administration>Hardware Drivers" and see if there is an entry for your g.card. If yes then tick the checkbox in order to have the drivers automatically **downloaded** and installed. You will be asked to reboot. Do so and I hope you'll be able to enable the effects after that. (Internet connection is required)
