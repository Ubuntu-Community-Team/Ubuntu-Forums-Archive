Welcome to the forums.

That dialogue box is letting you know that the software you are installing is not actually in the repositories.
If you trust the source of the software you have to click Okay. If you do not trust the source click Cancel.

Trusted sources are the repositories and sourceforge.net and the software authors own website. EG [www.Ubuntu.com](www.Ubuntu.com)

Installing from untrusted sources can allow malware to run on your system.
