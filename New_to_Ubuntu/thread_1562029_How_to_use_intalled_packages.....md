### Post by Peter09 on 2010-08-27
OK,
in Linux some task will not have a GUI interface, these are run straight from the terminal by typing the task name. Often these tasks are sub-tasks of a larger lump of software - so don't assume all the tasks you see can be executed straight without going through the GUI.
 
As for downloaded tasks, these will come in a few flavours, the best are those with an extension of .deb. These will be more-or-less be automatically installed when you click on the file. 
 
Files such as .tar are compressed files or archives of several files, the contents of which may vary and may not be easy to install.
 
The best advice is - if possible- use the tasks in the repositories through the Software Centre.
