### Post by tjwoosta on 2008-11-18
im not 100% sure which window manager that is but i know for sure that it is not gnome


with linux there are many different window managers and/or desktop environments to choose from

the two most popular are GNOME and KDE   
(ubuntu comes with GNOME installed, kubuntu comes with KDE installed)


but there are also many other, not so well known, window managers available

some examples are 

fluxbox, openbox, windowmaker, jwm, icewm, blackbox, enlightenment, xfce, sawfish, and many many more

each of those window managers has a unique look and feel, as well as having different ways of doing the same things

every window manager has its pros and cons, it maninly a matter of opinion

it is possible to install any window manager you want into any linux distrobution, but it is usually a very complicated task, and not something i would reccomend to any beginners
