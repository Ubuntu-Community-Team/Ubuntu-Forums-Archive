### Post by sideffects on 2009-06-22
First of all, when you are booting up, there should be a point where the computer says "Press any key to boot from CD/DVD..." If that doesn't show up, you still don't necessarily need to go into BIOS. If you can figure out the shortcut key (I think it's F2 on my Compaq) for the boot menu, then you can select to boot from the CD.
