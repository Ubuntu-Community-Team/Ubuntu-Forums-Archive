### Post by yeats on 2009-01-23
Sure.  Both VMWare and VirtualBox have Linux versions.  I prefer VirtualBox, because it is truly set up for a single PC installation, but VMWare is more widely used (there are hacks to make VBox read VMWare work).  If you're a stickler for Free/Open Source Versions of software, a version of VirtualBox OSE (Open Source Edition) is in the repositories, but I've found that installing the VirtualBox deb file for the "non-free" version is easy and there are MANY more features, including ease with networking (a new feature to VirtualBox).

Go here:
[URL="http://www.virtualbox.org/wiki/Linux_Downloads"]
http://www.virtualbox.org/wiki/Linux_Downloads[/URL]
