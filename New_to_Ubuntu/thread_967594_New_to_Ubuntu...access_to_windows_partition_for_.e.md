### Post by WRDN on 2008-11-02
Click Places > Computer and the windows partition should be listed there. Just double click on it to mount it.
To run executable files under Ubuntu, you will need WINE (stands for Wine Is Not an Emulator), and luck. If possible, you should install Office again using WINE, as you will probably have more success than if you run the executable from the installed application.
