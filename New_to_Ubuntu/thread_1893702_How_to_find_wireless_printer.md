### Post by papibe on 2011-12-10
Check your router's admin page in order to know both the printer's hostname and IP number.

Make sure you selected the 'Wired LAN connection' in the network setup of your printer.

Also, check this [thread]("http://ubuntuforums.org/showthread.php?t=1475336") about drivers.

Hope it helps.
Regards.
