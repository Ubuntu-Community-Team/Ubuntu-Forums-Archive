### Post by newbie-user on 2012-04-28
You can do remote desktop. Enable remote access on the Win7 machine, then use Terminal Server Client on Ubuntu to access it.

Alternatively, you can use third-party programs like logmein or teamviewer.

### Post by Zukaro on 2012-04-28
Google recently put out their own cross platform remote desktop tool.
What you need is Google Chrome, then look for the remote desktop tool (still beta but I hear it works fine).  You need Google Chrome on both computers, then you choose "ask for help" or something like that on the computer you need help on, it then gives a number which on your other computer you choose "offer help" or something like that and put that number in.  Then you can remote desktop in; however, you need to keep Chrome open on both computers.
