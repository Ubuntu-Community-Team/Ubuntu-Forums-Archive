### Post by Brain-free on 2008-11-15
No worries, it's really easy.

Go to System -> Preferences -> Appearance and in the window that appears in your desktop, drag n' drop the .tar.gz - or whatever - you downloaded.

### Post by Adamantus on 2008-12-23
What usually works for me is downloading them to a folder (I created and "Ubuntu Themes" folder in my user folder) and then just using "install" from the System>Preferences>Appearance box. That has always worked for me.

Also, make sure they're not a .zip file. I sometimes get a zip file while downloading from gnome-look, which won't install correctly.

### Post by BGFG on 2008-12-23
Be sure to check what type of theme it is or rather, which window manager it works with. On gnome look  the are sorted as 'Metacity' or 'compiz' and whatnot. The shiny glassy looking ones are usually compiz. (i haven't been able to get the compiz ones to work yet :))
So check which window manager you are using, then download the appropriate theme. here are some nice ones:
