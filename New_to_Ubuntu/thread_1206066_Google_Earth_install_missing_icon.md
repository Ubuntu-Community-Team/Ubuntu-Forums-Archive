### Post by taurus on 2009-07-06
Open up synaptic again and in the **Quick search**, type googleearth and read the info about that package.  Basically, you only installed the utility to build your own personal version of Google Earth, not the Google Earth itself.  

Therefore, you can either download the GoogleEarthLinux.bin and install it by hand or install the one from Medibuntu.

[https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)
