### Post by Jaxite on 2008-07-16
Boot windows normally, then insert the Ubuntu disk and go to your cd drive, so it opens the ubuntu files. Then there should be something called Wubi; run that, follow onscreen instructions, and voila. You can now boot windows and ubuntu.

### Post by halitech on 2008-07-16
before you start installing anything, reboot into windows and run defrag at least twice to get all files moved to the beginning of the drive. Then, when you bot back into Ubuntu, there should be an install icon on the desktop, just click that. when it asks about the partitions, do it manually and select where to install
