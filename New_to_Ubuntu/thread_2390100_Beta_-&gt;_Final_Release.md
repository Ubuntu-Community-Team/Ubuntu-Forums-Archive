### Post by deadflowr on 2018-04-25
Most who reinstall a newer iso from something like a beta release to a final release candidate do so for QA testing.
(It's important for the devs to know if a particular build is broken or not, or has other issues)

But in reality once you install any version of a particular release, be it 18.04 or 16.04. It'll be that release regardless of which iso it was installed from.
So long as you keep it updated.
