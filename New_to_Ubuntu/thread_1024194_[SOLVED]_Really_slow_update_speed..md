### Post by DOS4dinner on 2008-12-28
I've got a wired connection to a fiberoptic 1.2 megabytes/second internet. On Normal files and websites, I get that speed. However, having just re-installed ubuntu 8.04, the updates are coming in at 10-30 Kilobytes a second. Are the servers just overloaded, or did I do something wrong?

---

### Post by nhasian on 2008-12-28
you can change the updates so they come from a different server.  go to System->Administrator->Software Sources then change the Download From Other, then select best server

---

### Post by taurus on 2008-12-28
You can go into System -> Administration -> Synaptic Package Manager -> Settings -> Repositories and try another server.  Use the Main Server if you wish or have it check to see which is the best server for your location.
