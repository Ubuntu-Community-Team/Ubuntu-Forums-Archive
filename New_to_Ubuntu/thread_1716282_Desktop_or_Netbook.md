### Post by migs73 on 2011-03-29
Why the server edition?

Just use the standard desktop 64 bit (AMD64) Ubuntu with gnome and either install unity using the software centre or type into a terminal;

```
sudo apt-get install unity
```

This will work with the server edition too, but I don't know what the differences are between it and the desktop version.

Do not be fooled by the desktops AMD64 tag it is not limited to AMD processors.

BTW if you can wait until Natty Narwhal (11.04) comes out next month, it will have unity as the default GUI.
