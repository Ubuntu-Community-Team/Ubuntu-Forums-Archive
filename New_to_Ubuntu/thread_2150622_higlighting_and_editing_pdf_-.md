### Post by Impavidus on 2013-06-02
First of all, pdf has been designed as a printer language for the cheap consumers printers of the 1990s. It's a simplification of ps, which was designed for the expensive professional printers of the 1980s. Later some handy features for on-screen viewing of printable pages, like hyperlinks, were added. Still, pdf only contains a graphical description of the document, not a logical one. This makes it very hard to edit pdfs in any way other than simply adding or removing elements. For reflowing text et cetera we invented other file formats. We try to avoid editing pdfs whenever possible, editing the source from which it was generated instead. This is easier and will always give better results. People asking us to edit pdfs haven't understood the whole concept behind this file format.

Nonetheless, sometimes one cannot avoid editing pdf files. One of the better programs of doing so in Linux is Inkscape. It allows you to add and remove elements, move things, scale things etc. You can get it from the software centre.
