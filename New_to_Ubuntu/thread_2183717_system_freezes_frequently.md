### Post by mastablasta on 2013-10-28
certain atoms are not supported well in linux because the GPU part of the chip was actually not made by intel but by a thrd party for intel. and they locked the drivers and didn't make any for linux. so windows work well on these atoms, but not linux.

to install xubuntu you need to find and install xubuntu desktop package from software center then you select xubuntu-desktop on login.

note after such install you may notice duplicate apps doing same things (e.g. two network manager, two file managers etc. this can be relatively easy removed later on.
