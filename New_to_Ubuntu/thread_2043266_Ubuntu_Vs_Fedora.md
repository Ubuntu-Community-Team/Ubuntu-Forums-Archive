### Post by Carborundum on 2012-08-16
Of course. Both operating systems are Linux based, so any application that runs on one of them can easily be made to run on the other. Platform specific pre-compiled binaries, such as .deb files for Ubuntu and .rpm files for Fedora, won't just work on the other platform though. If you absolutely must run an .rpm file on Ubuntu, it is possible to do via an application called alien, but it's not recommended. If possible, you should try to find a .deb file instead, or compile the application yourself.
