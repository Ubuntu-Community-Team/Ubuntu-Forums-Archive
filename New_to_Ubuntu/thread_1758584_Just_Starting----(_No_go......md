### Post by Autodave on 2011-05-14
Put the CD in the tray and reboot the machine.  You have to make sure that your boot sequence is lookinh at the CD rom drive first.  You can check the boot sequence (and change it if necessary) right after the machine powers up: watch you screen for the command for boot sequence which will be something like ESC or F12, etc.  

Once it boots, you will have a screen that gives youy the option to TRY or INSTALL.  Choose the TRY option and look around a bit.  Keep in mind that it will be painfully slow running from the CD.

Funny. No one mentioned Wubi to the guy.

You have a perfectly good solution of "Wubi".

It's a program that helps you install Ubuntu within Windows. You get a completely hassle free Ubuntu installation and still, very much under Windows 7.

Maybe you already know it. But, well, here it goes:
1. Keep the Ubuntu ISO ([http://www.ubuntu.com/download/ubuntu/download]("http://www.ubuntu.com/download/ubuntu/download")) and the latest Wubi ([http://www.ubuntu.com/download/ubuntu/windows-installer]("http://www.ubuntu.com/download/ubuntu/windows-installer")) in the same folder in Windows.

2. Right click Wubi.exe and select "Run as Administrator". Then go on with the installation, install in the drive you wish and with the size that you wish.

3. PC restarts and you get a bootloader option in the Windows bootloader to go to Ubuntu or Windows (Windows by default). Select Ubuntu and the installation will continue and get finished at one point of time.

Then you can boot into Ubuntu from the Windows bootloader and use it.

Later, if you think you don't want to use it anymore, you can uninstall from within Windows 7.
