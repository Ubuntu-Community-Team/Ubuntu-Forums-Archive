### Post by DaveyG on 2011-12-17
Synaptic; as of 11.10 isn't installed by default anymore. Click the Ubuntu icon on the top left-hand corner, type Terminal. In Terminal type: > sudo apt-get install synaptic

That *should* install Synaptic, you could also install via the USC (Ubuntu Software Centre). It's usually somewhere in the list, on the left hand side

### Post by dFlyer on 2011-12-17
11.10 does take some getting use to. To open a terminal just click on the dash icon top left and type terminal or just press ctl-alt t will also open a terminal. Synaptic is not installed any more by default so you can install it via the Ubuntu Software Center or from the terminal with the following command:

sudo apt-get install synaptic

The best way to learn Unity is to take a few hours and explore. I've seen son links posted on this forum and also google search pulls up so also. A lot of people have complained about Unity, but I like it. It just took some getting use to. Now that I am I find it easier to use than classic gnome, or gnome shell. Anyway just go slow and have some fun learning the new desktop environment.
