### Post by Mark Phelps on 2012-02-25
System info is unreliable -- do NOT use it.

If Linux were really NOT recognizing your video chipset, instead of a graphical desktop, you would be seeing only a command line interface.

Since you didn't mention having only a command line interface, then Linux IS recognizing your video chipset and has already installed a working driver.
