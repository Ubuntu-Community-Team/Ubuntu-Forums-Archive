### Post by Jordan_Cameron on 2014-04-22
Having used ubuntu on a toshiba from 2006 and a toshiba from 20010, i've experienced no problems.  But as stated above, always try the Live environment first, as it gives you an idea of what's working and what's not.  Also, maybe try 12.04.4 instead of 14.04, as 12.04.4 has had a couple of years to mature, whereas 14.04 is still fresh of the rack.  They are both LTS releases.  Maybe even try Lubuntu for a lighter system.

Have fun!

### Post by Topsiho on 2014-04-22
I would advise a lighter version such as Lubuntu or Xubuntu. I have a Toshiba laptop from 2005 (A50) that, due to processor not supporting pae, now runs Xubuntu (12.04 I think), and does this well. Some other older computers have become to slow to run Ubuntu, and run Lubuntu quite well.

You should try a LiveDVD first, to see how your computer runs with (X)(L)Ubuntu. When installed it'll run faster than from a DVD (or USB-stick), but my experience with Ubuntu is that it only runs fast enough on my main desktop computer that is two years old.
