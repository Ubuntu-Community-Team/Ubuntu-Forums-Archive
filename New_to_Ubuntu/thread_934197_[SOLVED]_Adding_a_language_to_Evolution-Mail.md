### Post by germanix on 2008-09-30
I am using Evolution-Mail as my standard mail program. Currently the spelling checker only checks the English language. I would like to add the German language as well. How do I get the German dictionary and how do I then install it into Evolution?? Alternatively (and much better for me) is it possible to set up Evolution in such a way that it allows me to automatically write my mails in open-office-write which already has a German dictionary and spelling checker installed. (something like when Windows auto. opens Word when you write a mail in Outlook)
I thank you all for any help.

---

### Post by germanix on 2008-09-30
Problem solved. synaptic-package-manager, search aspellXX where XX is the country (DE for germany) Install this package which is then automatically integrated into Evolution.
