### Post by sadaruwan12 on 2010-11-06
Using UNETbootin is very easy do this.

[LIST=1]
[*]download the UNE (Ubuntu Netbook Edition) ISO from the web site and save it on you PC
[*]Download UNETbootin .exe from their website and save it on to your PC
[*]If you're using Win7 right click on UNEtbootin from the context menu click on run as administrator
[*]Now you'll be asked for permission click on allow or yes to give the program administrator rights and run
[*]After that select the ISO option click on browse and give the location of the downloaded UNE iso file.
[*]Check your USB is selected correctly if not select the correct drive letter
[*]Now click on OK wait till it finishers after finishing click exit.
[*]Now reboot and boot from the USB and install UNE on to your netbook.
[/LIST]

Thats it.
