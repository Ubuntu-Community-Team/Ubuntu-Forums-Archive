Printing to file is on by default in Ubuntu. Hit ctrl+p and select 'Print to file' as the printer.

### Post by HermanAB on 2011-12-02
Howdy,

It is the cups-pdf service that creates PDF files.  Doing a Print to File usually creates a Postscript file which can be converted to PDF using ps2pdf.
