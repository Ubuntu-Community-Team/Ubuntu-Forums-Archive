### Post by tonymoloney on 2009-06-07
Whenever I burn a CD using K3b, after I remove the disk and try to close the application, I get the following warning "DataCD0 has unsaved data - Save/Discard/Cancel". I have to pick one of the options to be able to close the application and I have found that Discard works without seeming to cause any problems.
My question, then, is what does this message mean? The burn process has already closed, so it would be impossible to write any unsaved data to the CD and the other options have no effect, so what's going on?

---

### Post by amingv on 2009-06-07
It's just asking if you want to save your burn project (to burn it again later, for example).
It's perfectly normal and most (all?) burning software will do it when you close.
It's perfectly OK to choose "discard" if you don't want/need to save it.
