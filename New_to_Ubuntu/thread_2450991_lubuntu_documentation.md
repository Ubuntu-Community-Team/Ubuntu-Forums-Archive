### Post by guiverc on 2020-09-24
If you want an official Ubuntu site, don't ask google or a search engine where it is. Go to a site that is reliable, for Ubuntu that is ubuntu.com.  You'll end up at [https://ubuntu.com/download/flavours](https://ubuntu.com/download/flavours) which will send you to the official sites for any Ubuntu *flavor*.

Lubuntu manual - [https://manual.lubuntu.me/stable/](https://manual.lubuntu.me/stable/)
Lubuntu links - [https://lubuntu.me/links/](https://lubuntu.me/links/)
Lubuntu wiki - [https://phab.lubuntu.me/w/](https://phab.lubuntu.me/w/)  (*this is more technical*)

And if you go to the official Ubuntu wiki, eg. [https://wiki.ubuntu.com/Lubuntu](https://wiki.ubuntu.com/Lubuntu) , you'll see messages like the following on most Lubuntu pages

> This wiki should be considered unmaintained and deprecated, though some information may still be relevant. Please instead refer to [Lubuntu.me]("https://lubuntu.me")

(*If detail was historic in nature and unlikely to be a problem for users, I didn't add the warning, but there will be pages I missed*.  *Ubuntu or Lubuntu cannot control unaffiliated or fan sites.*)
