Every now and then my numeric keypad stops working.

I discovered that the option "Allow to control the pointer using the keyboard" (System -> Prefences -> Keyboard -> Mouse Keys) is on whenever this happens.

Do you know why this turns on apparently by itself every now and then? Perhaps I'm accidentally pressing some keyboard combination?
