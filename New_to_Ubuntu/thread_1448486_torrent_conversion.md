### Post by lovinglinux on 2010-04-06
A torrent file only contains information (metadata) about the real ISO file you want to download. As already explained, you need to open the torrent file with a BitTorrent client like Transmission. It will contact Ubuntu torrent tracker server and connect you with other users sharing the same file. The program will then start to download the ISO to the folder specified on your torrent client. As soon as you have a couple of pieces of the ISO file, the client will also start ti upload to other users.

See [BitTorrent optimization and troubleshooting guide](http://ubuntuforums.org/showthread.php?t=1259923) for a more detailed explanation of how BitTorrent works and how to setup your system to use it.
