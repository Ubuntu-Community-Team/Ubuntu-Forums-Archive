### Post by pricetech on 2010-10-08
Your "recovery disk D" is a restore partition.  Unless you received install media for Windows 7, that's the only way you can fix or reinstall windows.

Does your OS have a recovery media creator ??  It may be called something different, but what it's for is to create a disk or a set of disks used to restore windows in the event you need to.

One man's opinion, but I'd never buy a computer that didn't come with install media.

How big is your recovery partition ?? Most aren't that large.

### Post by Yougo on 2010-10-08
^the D:\ drive can contain some recovery tools, but if it is accessible and is large (bigger than 5GB) and with lots of empty space, it's just a partition. if your din't get a Windows CD with your system, there should be a third, hidden partition.

the "installation disk" usually is a "hidden" partition that normally doesn't show up in Windows Explorer. i believe it should be between 5 and 10GB and crammed with data. you want to steer clear of this one if your plan of keeping your windows. (ordering a cd costs serious bucks <_< )

### Post by andrewthomas on 2010-10-08
From windows, I would burn some recovery disks, if you can, then use disk management to shrink the windows partition in order to make some space for ubuntu.  

Then, install ubuntu using the side-by-side installation option.
