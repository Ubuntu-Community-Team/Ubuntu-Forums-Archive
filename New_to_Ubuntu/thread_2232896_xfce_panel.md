### Post by Dennis N on 2014-07-05
In **Settings > Personal Section > Panel > Appearance Tab**:

In Background section, set Style to "Solid Color". Choose color by clicking on the color box. Set Alpha (transparency) to whatever you like. This fixes the problem.
