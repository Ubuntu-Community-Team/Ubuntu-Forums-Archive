### Post by cortman on 2012-02-20
Somebody SERIOUSLY needs to make a "Beginning programming tips/advice" sticky thread somewhere...
Welcome to the forums, and to Ubuntu and GNU/Linux in general! If you're interested in learning programming, I would recommend you start by learning the command line and shell scripting. We've been compiling a thread with resources on that [here]("http://ubuntuforums.org/showthread.php?t=1909108"). Along with that learn Regular Expressions, sed, awk, and at least one classic editor (emacs, vi, or nano). With that knowledge tucked under your belt, look no further than Python. You can use Python to write programs varying from scripts to very classy GUI desktop applications.
It's a lot of fun, I'm just starting myself as well. Cheers!
