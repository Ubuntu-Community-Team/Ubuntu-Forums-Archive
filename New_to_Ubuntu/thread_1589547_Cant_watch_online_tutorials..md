### Post by bobbob1016 on 2010-10-06
I don't know what site that is, but I wouldn't use it.  The files are windows program files, that is what the .exe thing is.  Personally, I wouldn't trust a site that uses an EXE for videos, since it can also cause/install viruses/adware.

If you must use that site, you might be able to run the exe in wine.  If you have Ubuntu 10.04 or greater, just go to applications -> software center, and search for and install wine.  It also might help to know the site in question.
