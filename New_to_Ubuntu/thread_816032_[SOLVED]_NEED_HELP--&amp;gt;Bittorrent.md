### Post by Paqman on 2008-06-02
You don't need to install anything. Ubuntu includes Transmission by default.

Just download the .torrent file that you need. Open up Transmission and start a new download by pointing it to the .torrent file and telling it where to download to. Transmission is pretty simple to use, you shouldn't have any trouble.
