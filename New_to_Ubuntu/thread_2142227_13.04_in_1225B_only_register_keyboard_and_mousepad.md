### Post by Toz on 2013-05-05
Try using the **i8042.reset** and **i8042.nomux** kernel parameters. As root, edit /etc/default/grub and edit the GRUB_CMDLINE_LINUX_DEFAULT line to look like this:
```
GRUB_CMDLINE_LINUX_DEFAULT="quiet splash i8042.reset i8042.nomux"
```
...save the file then run:
```
sudo update-grub
```

Reference: [https://bugs.launchpad.net/ubuntu/+bug/1014240]("https://bugs.launchpad.net/ubuntu/+bug/1014240")
