### Post by 3rdalbum on 2009-03-15
Why do you want to install a program onto another disk? 

Linux programs pretty much depend on themselves being installed into your root filesystem. I believe, if you compile a program yourself you can specify a different directory to be placed in, but this is a lot of work for dubious benefit.

### Post by ALIENDUDE5300 on 2009-03-15
It is possible although a lot of work. You can mount the other drive as a directory like /mnt/hdd2/ and then install it to that directory, although you'd be better off having all programs installed to your main drive.
