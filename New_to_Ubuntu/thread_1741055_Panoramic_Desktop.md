### Post by rosencrantz on 2011-04-28
Those are all desktop switching animations for Compiz. To get closer to the Android experience, you might want to fake a continuous background image, i.e. cut a panoramic image into 4 pieces and set them as backgrounds. I know it's possible to use different icons/widgets for each desktop in KDE - not so sure about Gnome/Compiz.
