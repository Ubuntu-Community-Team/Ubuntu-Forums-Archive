### Post by Nytram on 2012-04-08
If you installed to the home folder then these commands should start it, open a terminal and type:

> 
cd ~/bbtether
chmod +x bbtether.py
./bbtether.py


You'll also need wxpython installed to use the GUI:

> 
sudo apt-get install python-wxgtk2.8


Let us know if this works for you, and if not could you post any error messages.
