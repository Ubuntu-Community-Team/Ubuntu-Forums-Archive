If you are comfortable using the terminal try this command:
```
sudo dpkg-reconfigure -phigh xserver-xorg
```

Also, what monitor are you using?  Make sure it is capable of resolutions higher than 800x600....some of the older CRT types supplied with those systems are not capable of anything higher.

### Post by w4ett on 2009-01-01
Ok...let's try manually editing the file to include the monitor refresh rates. In the terminal backup your current settings:
```
cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup 
```

Then we will get down to business:

```
sudo gedit /etc/X11/xorg.conf
```

This will open the editor.  Replace this section:

> Section "Monitor"
Identifier "Configured Monitor"
EndSection

With this:

> Section "Monitor"
        Identifier      "Viewsonic GS771"
        HorizSync       30-70
        VertRefresh     50-160
EndSection

Save the file, and restart X. Ctrl>Alt>Backspace

Then login and go to System>Preferences>Screen Resolution and try to enable the desired resolution.
