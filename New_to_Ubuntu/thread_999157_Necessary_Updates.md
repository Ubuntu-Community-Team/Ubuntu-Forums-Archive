### Post by Idefix82 on 2008-12-01
You will never be offered updates for stuff you don't have installed.
I would recommend you to always install at least the important security updates. They usually won't be too large.
If you open the Synaptic Package Manager (Under System->Administration) you can specify that you only want to be notified of the important security updates (you do that in the 'Repositories' dialog).

### Post by ibuclaw on 2008-12-01
Offtopic - I wish ISP's were /that/ cheap in britain :(
Most ISP providers charge £20 up ($30-40).

Ontopic - You can set your sources so that only security updates are taken into account.

Open Synaptic and go to **Settings**->**Repositories**
Click on the **Updates** tab and uncheck **Recommended Updates (hardy-updates)**

You should get less updates through the update manager now, but still get the security updates as they get rolled out.
