### Post by Tatty on 2008-04-26
Use the cd integrity checker on the boot menu to check the CD.  If it fails the integrity test then do the following:

1. check the md5sum of the .iso [https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

2. burn the CD again on a much slower speed.

3. check the integrity again.
