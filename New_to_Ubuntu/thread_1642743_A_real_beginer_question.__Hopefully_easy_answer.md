### Post by beew on 2010-12-10
How did you install Ubuntu in a USB? if it is just a live usb with persistence (say you made it with a live usb creator and the drive is formatted as FAT rather than ext3 or ext4) I suggest that you be careful with the update because any system level update (say a  kernel patch) will break your system.

On the other hand if it is a full installation then just run update manager or open Synaptic, mark all updates and then click apply.
