### Post by spacecheck on 2012-01-11
> **hectorash said:**
> Where do I find that icon??
Not sure the icon will appear, if Wifi and/or Network connections are disabled.

I'm sure you are aware, that for most (all?) laptops there is also a hot  key combination to enable/disable wireless transmission (WiFi &  blutooth). On an IBM this is the blue FN+F5 . Try your hotkey combo first, to see if the icon suddenly appears.

IF using a docking station, make sure properly seated and try the LAN cable in each (laptop & docking station) socket.

If ok there, you should go to Settings, Network Connections and for each access type (wired and wifi) select the connection and click "Edit",  then make sure there is a check in the box for "Automatically Connect".

If the Icon for wired connections is visible, you may also be able to click that and select "Enable Networking" or "Enable Wireless".

If it works, be sure to use the thread tools to mark the thread Solved.

If not, report back for further input.

Good luck!
