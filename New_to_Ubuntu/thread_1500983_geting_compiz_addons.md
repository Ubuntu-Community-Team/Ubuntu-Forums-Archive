### Post by buntudawg on 2010-06-03
Assuming you have the main package and mean just the extras package:

```
sudo apt-get install compiz-fusion-plugins-extra
```

### Post by ChadMMc on 2010-06-04
To add faces to the 'cube' (in which it will no longer be a cube then of course), just goto the compiz general settings -> desktop size tab and change horizontal virtual size from 4 to whatever you want.
