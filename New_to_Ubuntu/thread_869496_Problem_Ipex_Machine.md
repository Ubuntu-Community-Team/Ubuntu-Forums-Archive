### Post by jimrz on 2008-07-24
> **ryco120842 said:**
> Have onboard video, 256mb of ram
Since the "live" install cd needs at least 256 Mb ram and your onboard vid is certainly using some of your 256, I would suggest that you download the "alternate" install .iso, burn it "as an image" and use that disk for your install. Also with only 256 Mb ram, you might consider using Xubuntu rather than the standard Gnome install or possibly going with an even lighter window manager (openbox, fluxbox or ice). Any of those options will give you a system that is less resource intensive and therefore will run faster on your equipment.
