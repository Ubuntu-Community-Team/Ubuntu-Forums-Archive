### Post by grahammechanical on 2013-06-23
I have not tested it but I am sure that a mouse sold under the Microsoft name is not so very different from a mouse sold under some other name. I think that the same applies to keyboards. With a Microsoft OS we may need to install a driver. But with Linux/Ubuntui drivers are built into the kernel (core part) of the OS. So, I do not understand how it is that you say you cannot use the mouse. It might be a setting in the BIOS causing this.

A clean install is an expression that some people use. On Ubuntu we can upgrade from one version to another but for various reasons the upgrade does not always work as it should. This is why some of us recommend installing a later version of Ubuntu afresh (a clean or fresh install).

You can install 12.04 in the same way as you installed 10.04. Set the machine's BIOS to boot from the DVD drive. Put the 12.04 disk in the drive and switch the machine on. At the desktop click the Install Ubuntu icon. You will be given an option to tick Install Third Party Software. Tick that box. That will bring in various proprietary codices that will make it possible to play sound files and view videos.

The installer will notice that 10.04 is on the machine. It may ask if you want to upgrade. You would be better if you ticked to install using the entire disk. Then it will be a fresh/clean install.

Then once the OS is installed and you are at a desktop you can open the Ubuntu Software Centre and search for Adobe Flash Player. The software centre will download and install it for you.
