### Post by Bölvaður on 2009-05-15
There is not that much difference really.
Edubuntu is mainly aimed at school children hence the name.

The only way to install edubuntu is first install ubuntu and then... well then you might have to download the edubuntu to get all the programs and such that are meant to be for the kids. But what I belief that happened is that you downloaded the package edubuntu-desktop and perhaps other edubuntu packages.. could that be?

Look at the upper left corner:
System &#8594; Administration &#8594; Synaptic package manager
Filter "edubuntu"
if there are any green boxes next to any of the edubuntu stuff, take it away by clicking the box and mark to completely remove.
Then look for ubuntu-desktop and make sure that has a green box next to it. Then click apply and log out (no need to shut down the computer) and then log back in.

If that doesnt work.. give us a screen shot :D
