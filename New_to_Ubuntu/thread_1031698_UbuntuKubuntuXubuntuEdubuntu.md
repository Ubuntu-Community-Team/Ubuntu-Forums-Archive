### Post by donkyhotay on 2009-01-05
Ubuntu uses gnome, kubuntu uses KDE, xubuntu uses xfce. Basically there isn't a lot of difference between them just personal preference. Generally most windows users feel more comfortable with kubuntu since KDE is more windows-ish. Personally I much prefer the look and feel of gnome and so use ubuntu itself. xfce is for lower-end systems and so you would use xubuntu if you have an older computer. Edubuntu is focused on education utilities and is designed for schools. Honestly though there is less difference between all the ubuntu variants then there is between XP and vista so it really doesn't matter. Be aware linux is all about choice, although sometimes it's hard to figure out what choices to make (especially when starting) since the software is free your more then welcome to try them all out until you find what works for you.

### Post by Titan8990 on 2009-01-05
It is the types of desktops and software they have on it. Ubuntu is the way to go because it gets the most love from the Ubuntu developers.

Ubuntu = [GNOME](http://en.wikipedia.org/wiki/GNOME)

Kubuntu = [KDE](http://en.wikipedia.org/wiki/Kde)

Xubuntu = [XFCE](http://en.wikipedia.org/wiki/Xfce)

Edubuntu = Gnome + education packages



It doesn't matter which one you get. If you were to get Ubuntu (gnome) you can also install KDE, XFCE, fluxbox, jwm, or any other window manager/desktop of your choosing. That is the flexibility of Linux. 

However IMO Gnome is the only way to go for Ubuntu. Many will disagree but if I wanted a KDE system I would use something more like openSUSE because many other distros focus more on KDE. Xubuntu is a horrible attempt at making Ubuntu run with less memory. Compared to the XFCE that other distros such as Gentoo or Slackware use, Xubuntu is bloated.
