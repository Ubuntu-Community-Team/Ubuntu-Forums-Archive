### Post by Jive Turkey on 2009-05-18
If it makes you feel any better, you've already thrown the money away whether you use word or not.  It's what they call a "sunk cost."  The question one should ask before abandoning an asset (your license to use word) is: "will the replacement asset provide more benefit?" It is irrational to argue that some asset cost $X so I shouldn't do anything else.  The rational argument is that some alternative (word or OO writer) gives whatever additional benefit over the other. 

 I'm not telling you which to chose, only a more effictive decision making method.

I read that word can work under Wine, if that's not correct, there is another emulation application that is not free which does work to let you run MSOffice under linux, the name eludes me at the moment.
