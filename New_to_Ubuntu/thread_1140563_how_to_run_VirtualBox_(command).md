### Post by Locke_99GS on 2009-04-27
In the Gnome menu, do *Applications > System Tools > Sun xVM VirtualBox*.

-OR-

In a shell, run: ```
VirtualBox
``` 

It is located at */usr/bin/VirtualBox*
