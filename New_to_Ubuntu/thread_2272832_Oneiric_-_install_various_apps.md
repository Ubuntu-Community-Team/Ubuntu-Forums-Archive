### Post by newb85 on 2015-04-09
Indeed, 11.10 won't update because the repositories for 11.10 have been taken down.  (It hit End-of-Life April of 2013.)  Software updates (including security fixes) compatible with 11.10 haven't been released since then.

If your hands are tied and you absolutely can't change from 11.10, there is an archive that can be used, but without more recent security updates, it's not really a good idea.
