### Post by Joeb454 on 2008-04-26
Kubuntu uses the KDE Desktop Environment, as opposed to Ubuntu which uses Gnome.

The default app's are different - Kubuntu uses KDE apps, Ubuntu uses Gnome app's. Though you can install KDE app's on normal ubuntu etc. :)

Hope this helps, other than that there is no real difference

### Post by TeoBigusGeekus on 2008-04-26
They are the same thing. They just differ on their graphical user interface (GUI): ubuntu uses gnome whereas kubuntu uses kde.
They come with different applications, but nobody can stop you from using gnome applications in kubuntu or kde applications in ubuntu. 
Honestly, it's just a matter of taste...
Image Google for Kubuntu and Ubuntu screenshots and see what you like better... or better still, download both live cds and see for yourself.

### Post by Kinst on 2008-04-26
Ubuntu is the regular distribution that uses GNOME for the graphic user interface.

Kubuntu is just ubuntu with GNOME removed and KDE used instead.

You can make kubuntu yourself by installing kde and uninstalling gnome.
