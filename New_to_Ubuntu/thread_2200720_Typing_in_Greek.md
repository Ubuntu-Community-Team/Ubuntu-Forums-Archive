### Post by grahammechanical on 2014-01-21
What version of Ubuntu are you using? The methods of setting this up have changed over the years. And so there are different instructions. Basically, you need to add a Greek keyboard layout. And then an iocn will appear in the top panel which will let you switch and the click of a mouse between keyboard layouts.

In Ubuntu 13.10 (with the Unity Interface) we go to System Settings (power/cog icon) and Keyboard and click the Layout Settings link at the bottom left. Then we click plus ( + ) and then select the language layout from the list and click "Add." That is all there is to it. Older versions of Ubuntu will be similar but might have a different name for this utility.
