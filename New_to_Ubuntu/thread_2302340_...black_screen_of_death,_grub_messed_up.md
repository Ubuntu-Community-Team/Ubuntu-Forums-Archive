### Post by ajgreeny on 2015-11-09
The Windows OS is installed in UEFI mode, so I'm afraid you can not run Ubuntu in legacy mode in dual boot with Windows at the moment.  Both OSs must be in the same boot mode.

You should read all the info at [http://ubuntuforums.org/showthread.php?t=2147295](http://ubuntuforums.org/showthread.php?t=2147295) and [https://help.ubuntu.com/community/UEFI](https://help.ubuntu.com/community/UEFI) which may help you.

You must boot the USB install medium in UEFI mode to be able to install Ubuntu in that same mode.  If you still have problems after reading those two links come back again.
