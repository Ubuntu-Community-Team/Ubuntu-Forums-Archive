### Post by kjohansen on 2008-09-05
i think the problem is you have set multiple home pages.

go to Edit->Preferences->Main

and set you home page to a single page or hit restore to default.

### Post by anotherdisciple on 2008-09-05
Under (Edit--> Preferences--> Main) there is a dropdown under 
"startup"... change it from "show my windows and tabs from last time" to "show my homepage" and set your homepage to whatever you want it to be.
