### Post by chiliman on 2010-07-08
According to this ubuntu documentation page, [https://help.ubuntu.com/community/KeyboardShortcuts](https://help.ubuntu.com/community/KeyboardShortcuts)
> To find (and change) system-level keyboard shortcuts, one can (at least in later, GNOME-based distros, e.g. ubuntu karmic) run the **Keyboard Shortcuts** applet, e.g. via System>Preferences>Keyboard Shortcuts on the main menu
 
and one the link on the bottom of that page to this [http://en.wikipedia.org/wiki/Table_of_keyboard_shortcuts](http://en.wikipedia.org/wiki/Table_of_keyboard_shortcuts)
says that alt + Space default is for "pop up window menu"
 
I hope thats what your are looking for.
