### Post by baruch60610 on 2010-01-21
Using the menu, go to Settings, then select Window Manager.  You should get a dialog box that has four tabs.  One of them should say "Advanced".  Click on that one, and uncheck the boxes that talk about "snapping".  Those are the first two boxes, if memory serves.
