### Post by wpshooter on 2009-06-14
IF, you want to keep M/S, then my suggestion would be to have it on a completely separate computer.

There are some reasons for having it.  For instance, if you wanted to run software applications like Quickbooks or Peachtree accounting software (and I am sure that there are many others), then you will need to have Windows because you are going to find that many of these M/S compatible applications have NO EQUALS in substitute applications that will run natively on a Ubuntu computer.  You might try to run them under something like WINE but IMO, you are likely to be disappointed with the results and if you need such applications then you would be much better in having a separate M/S computer to run them on natively.  The fact that applications like the aforementioned M/S compatible applications have not been coverted to run natively on Ubuntu systems is why Ubuntu/Linux has unfortunately not (and probably, never will) supersede the M/S O/S.:(
