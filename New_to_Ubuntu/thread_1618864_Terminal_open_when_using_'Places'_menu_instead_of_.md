### Post by plucky on 2010-11-11
> **mr_luksom said:**
> I have no idea what I have done, but on Maverick Ubuntu (GNOME) install, when I click on an item in the 'Places' menu it opens up a terminal instead of a nautilus window.

Does anyone know how to remedy this?

ALT+F2 and type in **nautilus** Then right click on any folder and select "Open with Other Application".
From the window that opens select "File Browser" and select "Open.



Good Luck

---

### Post by mr_luksom on 2010-11-12
That worked.

For anyone else's reference, I guess the reverse of this is what caused the problem, ie I must have opened a folder using this option, specifying the gnome-terminal.

Cheers.
