Sure can....navigate to the web site of your choice. Hold the alt key and hit the print screen key -- in a couple of seconds Ubuntu will take a screenshot of your active screen -- save it to your home folder/Pictures. ### Post by Mark Phelps on 2009-02-27
Basically, AFAIK, -- no.  

Linux is not Windows. 

In Linux, the desktop is managed differently, by default through Gnome, but also through other desktop managers such as KDE or XCFE. There are ways to create widgets that sit on the desktop, but they use other approaches.
