### Post by I8NY on 2010-03-20
If you can get to terminal then try running "startx" and that should bring up the desktop.  Now i had a similar problem and i fixed by getting to the desktop by startx then reinstall the display driver.  After that a kernel panic error would happen for me but that was fixed after a recovery option to repair packages was selected.  So it worked for me but i don't know what startx does completely so use it at your own risk but i hope it works.  It is worth a shot.
