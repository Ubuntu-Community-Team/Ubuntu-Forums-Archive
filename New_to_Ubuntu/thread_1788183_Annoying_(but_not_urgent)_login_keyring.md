### Post by CaptainJamie on 2011-06-22
Ok, I think that has solved it. For anyone else reading this thread:
The password is blanked using "passwords and encryption keys" (a program pre-installed)
right click on the passwords folder, go to change password, type in the current password and select OK, ignoring the warning. Done.
