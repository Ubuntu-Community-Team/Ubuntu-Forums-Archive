### Post by QIII on 2014-07-14
Yeah.  I think that is an exception thrown by Flash or Adobe Air.  If the site expects an updated version of Flash (or Adobe Air at all) it will likely not work with Linux.

Flash for Linux is no longer under development by Adobe.  So we have a very outdated version which is only receiving security updates for the next three years.

I'd say to stick with the site where you can view the videos.

Another option would be to try Chrome, since it comes with its own Flash version which is up to date.
