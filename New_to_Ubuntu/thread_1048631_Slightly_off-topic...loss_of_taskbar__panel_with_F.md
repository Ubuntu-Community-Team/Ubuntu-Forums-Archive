### Post by loobyloo on 2009-01-23
Apologies in advance for the slightly off-topic nature of this post but I wonder whether anyone knew anything about this.

I've noticed that since I've installed Zotero it causes the loss of my taskbar / Panel when I open FF. A couple of clicks of F11 solves it but it's annoying. If I open FF in safe mode it doesn't have this behaviour.

I gather this is a known problem with some extensions on FF but I was wondering if anyone had found a way to prevent this happening.

I've already tried unticking the permission in FF which allows sites to resize windows, and I've also made sure that the command that's being used to open FF is simply "firefox" as some people said the %u or whatever it is that sometimes follows it can cause this problem. But it's definitely something to do with Zotero. 

As I say I realise I'm not necessarily in the correct site here but if anyone had any ideas I'd be most grateful.

### Post by fooman on 2009-01-23
this should fix it....for good:

open firefox....press f11 twice to get to a normal screen.

next click the unmaximize button.

close firefox.

reopen firefox.

maximize firefox.

hope that helps. :)
