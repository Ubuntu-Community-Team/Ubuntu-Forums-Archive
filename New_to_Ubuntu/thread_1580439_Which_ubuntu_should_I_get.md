### Post by mcduck on 2010-09-23
Yes, if you just install all the available updates you'll end with the final version the day it's released (or actually day or two before that, most likely).

But you should be aware that it might not be very stable at the moment, Ubuntu's beta releases aren't quite like Gmail Beta and such, they really *are* unstable (the development cycle is pretty short, so one month to release day is pretty long time). If you are new to Linux, and aren't planning on doing actual testing, I'd recommend going for the stable release instead..
