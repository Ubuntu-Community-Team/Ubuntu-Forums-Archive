### Post by migrate from windows on 2011-03-16
HI Vivek, take care when you install windows after ubuntu . Your grub may be overwritten (still your data will be fine, only ubuntu will not be in your boot up list) but you can restore grub using live cd and other ways.

[https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot) the link gives you info about insalling ubuntu after windows and vice versa...Good luck.
