### Post by spikoley on 2011-05-07
When you are in a shell the cursor will not move when typing in the password.  As a trouble shooting step try updating the password.  When grub loads select the recovery option.  To change the password use the command below.

```
passwd <user name>
```

### Post by speedrcj on 2011-05-08
hmmm, that is very strange.  Whatever login you provide for the GUI should work there as well.  While in the terminal application type "cat /etc/passwd" and look for your login.  That will tell you all the logins available to your system.  You'll see a few you don't recognize most likely since Ubuntu will create some, but try looking for one that you might have accidentally created.  If you find one try logging into it in command line using "su <user>" to make sure you have the password.
