### Post by mikewhatever on 2010-03-25
Cody, you need to uncomment the following line in /etc/default/grub, and then run <sudo update-grub>:

```
#GRUB_HIDDEN_TIMEOUT=3

```

According to grub2 docs:

> If a user with only Ubuntu wishes to display the menu, place a # symbol at the start of this line to disable the hidden menu feature.
