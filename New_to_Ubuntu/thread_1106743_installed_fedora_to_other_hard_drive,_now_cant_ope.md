### Post by louieb on 2009-03-26
To Fedora's menu add a configfile entry and see if you can load ubuntus menu.lst

```
title load Ubuntu menu
configfile (hd1,0)/boot/grub/menu.lst   
```

Take the 2 map lines off the windows entry in the Fedora menu.lst. 
That should fix windows.
