### Post by jbrown96 on 2008-07-21
It's going to be a pain to download the .debs. They are convenient if you have an internet connection and need an app that isn't in the repos. The problem is that .debs resolve dependencies, meaning that if you try to install something like Amarok from its .deb file, you need many other files to satisfy the dependencies. AFAIK there are two ways to solve this. First, internet connection, which is a problem. Second, download all the .debs and the .debs for all the dependencies, and start installing the ones that have no dependencies and work your way from there.

Good luck
