### Post by Therion on 2009-03-20
I've run an 8800GTS for the past couple years with both 64 and 32 bit distros of all flavors.  

The Restricted Driver Manager has never once failed to recognize the card properly or too install the correct driver for it.

It's a little different in Ubuntu in that you don't have to go "hunting" for device drivers.  

Relax, do your install; you'll be prompted to install the correct Restricted Driver for your video card.

### Post by Ms_Angel_D on 2009-03-20
The easy way for a beginner is to install EnvyNG

[CLICK HERE]("apt://envyng-gtk") to install it then go to Applications->System Tools to launch it. Also you want to install Nvidia Settings [CLICK HERE]("apt://nvidia-settings") for that.

If you want the absolutely latest drivers you can download them from Nvidia's website and then follow [THIS]("http://ubuntuforums.org/showthread.php?t=990978") tutorial.
