### Post by mastablasta on 2016-11-07
Ubutnu and Kubuntu both have 3D accelerated desktop. this is a resource that is not necessary if you used for example Xubuntu, Lubuntu or simply just a widnows manager. Windows also has 3D accelerated desktop since i think Windows 7. so anything that runs 7 well runs Ubutnu well or even better.

next that is "resource hungry" in reviews is RAM. again Ubuntu, Kubuntu and Gnome all have various some apps that take up ram and generally they use more than the lighter versions of desktops. so for exmapel Kubuntu live Will take up arround 540 MB RAM on boot, while something using just a windows manager (jwm, icewm) will take up 80-100 MB or even less. again PC's with win7 usually have 2 GB ram or more. but many PC running widnows XP will run happier with Xubutnu or Lubuntu particulary if they have less than 1 GB ram available.

so while for some 540 MB ram on boot is "heavy" others would just say "meh... windows might need 2 GB".
