### Post by Cheesemill on 2013-05-02
To install the OpenJDK run one of the following commands, depending on which version you want....

```
sudo apt-get install openjdk-6-jre
```
or
```
sudo apt-get install openjdk-7-jre
```

Instructions on how to install the different versions of Java can be found on the wiki page...
[https://help.ubuntu.com/community/Java](https://help.ubuntu.com/community/Java)
