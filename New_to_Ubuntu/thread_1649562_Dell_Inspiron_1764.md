### Post by janpol on 2010-12-20
You can try out Ubuntu without installing it using a livecd (just remember it might be a little slow, since it reads from a CD instead a HD) or install it as a Windows app using Wubi ([http://www.ubuntu.com/desktop/get-ubuntu/windows-installer]("http://www.ubuntu.com/desktop/get-ubuntu/windows-installer")).

Either way, you won't mess up with your current Windows installation, and you'll be able to see if your hardware works fine under Ubuntu ;)

Good Luck!
