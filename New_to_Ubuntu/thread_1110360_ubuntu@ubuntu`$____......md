### Post by spcwingo on 2009-03-29
Try:

```
startx
```

More than likely this will not work, though.  The reason I say this is, if it was going to work it would have started X itself.  You might want to check the CD for defects from it's boot menu.  If it says that it's okay try booting into safe graphics mode from the CD menu.  If none of the above work you might want to give the alternate installer a shot.
