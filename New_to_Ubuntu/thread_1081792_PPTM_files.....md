### Post by zaidaus on 2009-12-03
When trying to open a PPTM file in OpenOffice.org Impress 2.4, it prompts me to select the file format type in the filter selection window. I googled for PPTM and it is apparently a "Power Point Microsoft Office Open XML Format Presentation with Macros Enabled." So I chose "Office Open XML Presentation Document" from the filter selection prompt and the file opened with some success. The main problem I had with viewing the presentation was that some images weren't being displayed and were replaced with icons instead.
