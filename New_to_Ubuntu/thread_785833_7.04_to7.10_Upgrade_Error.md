### Post by forestpixie on 2008-05-07
If you haven't upgraded yet - open you sources.list and disable all 3rd part repos, not usre what your using but for ubuntu

```
gksudo gedit /etc/apt/sources.list
```

then update

```
sudo apt-get update
```

for kubuntu 

kdesu kate

for xubuntu

gksudo mousepad
