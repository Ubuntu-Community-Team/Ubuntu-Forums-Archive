### Post by iaculallad on 2009-01-16
There are hundreds of terminal commands available for use in Ubuntu. Forum member's will try list some commands and others will try to post you a link to follow for those terminal commands. One of the most important command I could tell you is: man (the interface command for online command reference manual).

Example:

-To know more about sudo/gksudo, issue the command below on your terminal:

```
man sudo
```

or

```
man gksudo
```

This will give you enough information on how the command operates.

Syntax:

```
man the_command
```

HTH.

### Post by ibutho on 2009-01-16
Take a look at [linuxcommand.org]("http://www.linuxcommand.org"). Its a very good website for those getting started with the command line. If you have some money to spare get a copy of a book called UNIX (Visual Quickstart Guide) by Deborah S. Ray and Eric J. Ray. It will teach you a lot of basics and optional advanced stuff. Its written in a very friendly and laid back style, rather than like a text book and the command learnt can be used on Linux, BSD, Solaris etc.
