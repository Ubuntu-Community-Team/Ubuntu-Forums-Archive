### Post by Bashing-om on 2013-02-19
Project ICE; Hi !
Try and boot up the install CD to test that ubuntu runs well.

Bios set to boot the cd as 1st boot priority,
at the purple splash screen (stick figure and keyboard emblems at bottom of screen) hit any key -> language selection screen - hit escape to accept the default-, -> boot option screen -> f6 key for preset boot options, try the "nomodeset" option first (effective 85% of the time on difficult boots) ->f10 key to continue the boot process. IF not booting try the other options.
What version are you booting ?
12.04: desktop -> launcher(left side of screen) -> System Setting -> Additional Drivers: install the recommended driver.
12.10: Desktop -> launcher -> Ubuntu Software Center ->Software Sources(taskbar menu) -> Additional Drivers (tab): install the recommended driver.

Play with Ubuntu, ensuring all functions well.

Then install to hard disk :: if there is a problem after the install a different method is to be employed for the graphics to work. Advise pending.
