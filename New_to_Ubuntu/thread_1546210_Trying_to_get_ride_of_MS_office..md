### Post by Paddy Landau on 2010-08-05
For future note, I recommend that you use [PlayOnLinux]("http://playonlinux.com/"). It's a front-end to Wine, and makes it easy to install, run and uninstall Wine programs.

PlayOnLinux has other advantages, such as separating Windows programs so that they don't interfere with each other. It also has specialist knowledge of a number of Windows programs (such as Office), making them even easier to install and run.
