### Post by qyot27 on 2010-10-01
The only way that will work is if you install KDE for Windows in its entirety first using their installer, as the only two ways to get binaries for KDE apps in Windows are to do it through said installer, or compile them yourself after having done that installer.  Look here for stuff on that:
[http://windows.kde.org/news.php](http://windows.kde.org/news.php)

Personally, I think that would be a monumental waste, both of space and time.  Just use ImgBurn ([http://www.imgburn.com/](http://www.imgburn.com/)) to do it.
