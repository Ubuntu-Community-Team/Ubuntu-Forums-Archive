### Post by Utkarsh Ray on 2012-05-12
By default every network connection except ethernet is disabled. I've seen the need to
Enable Wireless
on all the systems with fresh install I've seen.
When I installed 12.04, I also had to do
Enable Networking
and then enable the option to use a mobile device as modem to use internet.
