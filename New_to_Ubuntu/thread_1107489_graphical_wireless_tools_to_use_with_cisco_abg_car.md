### Post by wildman4god on 2009-03-26
Ubuntu already has some graphical network tools:

system => administration => network tools
system => prefrences => network configuration

also you can get to network configuration by right-clicking the network manager icon in system tray and select "edit connections" from the drop down menu.

If you don't like those you can download another wireless network manager called wicd:

open a terminal and type:

sudo apt-get install wicd

or if you don't like terminal you can goto:

system => administration => synaptic

and then click search and type wicd in the search box

holler if you need help.
