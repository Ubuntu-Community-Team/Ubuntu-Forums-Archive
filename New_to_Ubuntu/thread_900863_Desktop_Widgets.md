### Post by txcrackers on 2008-08-26
For KDE 3.5.x, you want to use the program *superkaramba*. You may need to install it. To get "widgets" for your desktop, start superkaramba. One of the options in the main window is "New Stuff" - click on that and you can see just about every widget available.
