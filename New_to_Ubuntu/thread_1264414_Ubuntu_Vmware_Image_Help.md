### Post by yeats on 2009-09-12
> But i need to know how to view all the drvies in windows XP in Ubuntu through VMplayer

This is not going to be possible in the way you're expecting it to work.  When you create a VM, it acts as if it is a completely separate system on a separate machine.  Even though it is hosted on your XP machine, both Ubuntu and XP do not "see" each other they way they would if you were dual-booting or using a live CD/USB.
