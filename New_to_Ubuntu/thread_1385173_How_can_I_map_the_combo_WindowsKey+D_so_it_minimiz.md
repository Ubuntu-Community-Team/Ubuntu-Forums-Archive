### Post by sisco311 on 2010-01-19

> Or is there a similar key command in Linux? :P

Ctrl+Alt+d is the default. You can change it under System -> Preferences -> Keyboard Shortcuts -> Window Management -> Hide all normal windows and set focus to the desktop
