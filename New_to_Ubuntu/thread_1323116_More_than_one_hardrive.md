### Post by undecim on 2009-11-11
They are both internal drives? This should be easy to do. Both the Ubuntu and Windows installers will allow you to choose which hard drive you install to. No need to reformat the drives with a livecd, because that will be taken care of when you install.

Just make sure you install Ubuntu second, so that it will see your windows hard drive and add it to the grub menu
