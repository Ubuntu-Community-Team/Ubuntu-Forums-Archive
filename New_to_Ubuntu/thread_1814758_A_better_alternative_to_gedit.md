### Post by kaldor on 2011-07-30
> **abhishes said:**
> I am editing a HTML file which is very large. I edit it with gedit and it hangs while scrolling and is very very slow.

Bluefish (*sudo apt-get install bluefish*) is a decent IDE. Aptana (not in repos) is a web development IDE with loads of features.

However, I assume you want something simple. Try some of the other simple editors available in the repos such as Kate, Leafpad, etc. 

Also, you can try cleaning up your code. Separate your css into a different file (styles.css) and go through your code to see if you are using any useless elements.
