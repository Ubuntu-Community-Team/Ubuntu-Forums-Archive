### Post by stumbleUpon on 2009-08-11
In nautilus, just go to one level above the directory you want to add as a shortcut. Left click the directory you want to add and DRAG IT into cairo dock.
 At the prompt whether you want to monitor the contents of the directory say 

(1) no if you want to click on the icon to open the directory in nautilus

(2) yes if you want the contents of the directory to appear as a sub-dock to the launcher you added (note that the contents will NOT be monitored recursively)
