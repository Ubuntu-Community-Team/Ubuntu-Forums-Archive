Sadly some of the flash-enabled sites do not work with Linux due to buggy Windows-only code (sorry NEW FEATURES) that cause different issues.  For example, I use Facebook's poker game and it works just fine.  HOWEVER, I cannot select what room or casino I wish to join, I can only use the PLAY NOW feature.

In order to fix this issue, I had to run Virtualbox, install Windows on a virtual drive, open it, load up Internet Explorer and then play it.

Kinda sad to have to jump through hoops because of bad coders (Sorry, I mean undocumented feature writers!) focused on only one OS.

Anyhow, it sounds to me like you only need Windows for a couple minor things, you might want to consider ditching it in favor of a virtual windows machine that you can fire up from inside Linux.
