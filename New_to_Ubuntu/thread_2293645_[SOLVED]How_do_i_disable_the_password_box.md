### Post by coffeecat on 2015-09-06
Little cogwheel icon top right of screen -> System Settings -> Brightness & Lock -> switch lock to off, or choose a longer period before it is locked.

### Post by Dennis N on 2015-09-06
Menu > System > Control Center > Screensaver 

Note the two check boxes at the bottom. Uncheck the box "Lock screen when screensaver is active".
