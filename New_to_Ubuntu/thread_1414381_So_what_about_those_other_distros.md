### Post by QIII on 2010-02-23
I have ATI graphics and use ATI/AMD's Catalyst 9.10.  It was specifically made available for Karmic by ATI/AMD (although it is now freely available to all).

If you go to the ATI/AMD website, you will notice that 9.10 is not the latest version.  I have not tried the newer versions because I don't want to fix what's not broken.  But I would install it from the Karmic repos rather than from the ATI/AMD website anyway.

I can tell you, however, that ATI/AMD has been having some difficulty getting a driver out for the version of X in Lucid.  I expect that to happen shortly before the final Lucid release.

The difficulty with snowpine's suggestion is that you may not get a full feel for driver performance in VirtualBox.  It uses its own virtual drivers, so you wouldn't, for instance, be using the ATI/AMD driver necessarily.  You might not be able to apply Visual Effects, etc.
