### Post by mikodo on 2011-04-05
I want to be to change the focus of a terminal window to another window that is on the desktop and vice-versa, by key strokes and not loose the other window from sight.

Ctrl -> Alt -> Tab will change the focus of windows but minimizes the first window to the task bar. I would like to learn a keystroke that changes focus to the other window, but keeps the other window in view, and be able to change the focus back to the first focused window, without having to take my hands off the keyboard and have to click the mouse. 

Is there such?

Thanks.

### Post by mikodo on 2011-04-06
OK! 

I found if neither of the two windows are maximized, then Ctrl + Alt + TAb, will change the focus to the other window and return and still keep both on the desktop for viewing. An example, is being on Ubuntu forums writing this in a less than maximized window and having a less than maximum terminal window on the desktop also and switching focus with Crtl + Alt + Tab. To return focus to the other window later, use the same key combinations.

It just does not work with a minimized window (like a terminal window) on another fully maximized page.
