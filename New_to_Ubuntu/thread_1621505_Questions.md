### Post by sikander3786 on 2010-11-14
> **velzevool said:**
> So i'm using ubuntu and i have some questions.Can i install windows or should i have windows and then install ubuntu so i can have both?Also if i have both installed will it affect my pc performance?I want windows only to run 2 or 3 games.
You can go either way. Install Ubuntu first or second. But if you install Windows later, you'll need to reinstall Grub, the Ubuntu bootloader in order to dual boot. So, it is always preferred specially for newbies that they first install Windows and then Ubuntu as it saves them lots of hassle and hard work.

Having 2 operating systems should not decrease your system performance at all. Each operating system runs apparently regardless of the other, without even knowing about the other one present there. The only thing that matters is HDD space. If you've got that, having 2 or even more OS should not slow down your computer.

And once dual booting, you can do whatever you want to do with Windows without just sticking to 2, 3 games :P

And some informative links about dual booting have been posted above, please go through them.
