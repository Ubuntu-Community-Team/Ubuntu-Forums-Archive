### Post by sanderd17 on 2010-06-05
it all depends on what scripts you want to make.

If it's only for linux, you could use bash. This is just command line rules in a file. [http://gd.tuwien.ac.at/linuxcommand.org/learning_the_shell.php]("http://gd.tuwien.ac.at/linuxcommand.org/learning_the_shell.php")

If you want to write scripts that act more like programs, you could use python. [http://www.python.org/]("http://www.python.org/")

If you want to make real programs, you could use Java. [http://en.wikibooks.org/wiki/Java_Programming]("http://en.wikibooks.org/wiki/Java_Programming")

If you want to write real programs with a big interaction with the hardware itself, you could learn C or C++. 

The languages are sorted from easy to hard but the sorting is rather subjective. The first 2 are scrip languages, the last 2 are programming languages.
