Live cd load times all depend on the spec of the computer.
The whole OS is loading into memory so if you dont have too much of the stuff it can take a while.
If you haven`t already you should hit "enter" after the cd spits out to complete the shutdown.
I`ve had machines/Distro`s myself that have done a similar thing in the past when it came to shutting down, with full installations at that.They`d go through the entire shutdown process but then hang at the very last moment.I`m sure there were fixes but just holding the power button for a moment should do you for a Live CD Environment, if hitting "enter" is`nt enough of course.
