### Post by DavidMorton on 2011-03-09
That's normal.  Chances are, the formatting won't be lost.  Save a copy in the standard format first, so you have a copy, then go ahead and save the .doc file.  Open the .doc file in Microsoft Word (or have a friend do it on their computer) to make sure it looks pretty much the same.  

Anytime you convert between any file type, you're going to lose some information.  That's normal, but more than likely, the formatting you're going to lose isn't enough to make the document look wrong.
