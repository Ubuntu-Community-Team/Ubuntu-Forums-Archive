### Post by deadflowr on 2021-10-22
Fuse is a file system in userspace, so check the File manager for it.
If using Ubuntu look in Files (scroll down to Other Locations and check if the device is listed in there.)
If it is, then click on it then navigate it to find the files and a simple copy and then paste will work, most times.
Any issues would be most likely permission issues.
