### Post by 3rdalbum on 2011-07-02
Wow. That's a lot of hate over a simple mouse pointer.

All open-source software depends on bug reports and triaging. Bug reports are when users or developers notice unintented behaviour or problems, and then report them (in Ubuntu's case, on Launchpad). Triaging is where people come along later and say "Right, this bug is critical/ this one is a security flaw/ this one is not serious/ this one is not a bug/ this one has already been reported".

The cursors issue is sufficiently obscure for the bug to be introduced by accident. It's also sufficiently minor to be triaged as such and not put on high priority (or perhaps, ANY priority) to be fixed.

It was fixed in 10.04.3, I think; but was accidentally reintroduced in 11.04. It's one of those things that nobody really cares about, and that new users are unlikely to find as the cursors option is hidden away these days.

I've reported fairly obscure bugs before to Ubuntu-specific programs and had them addressed fairly quickly, but it's a lot easier to fix bugs in your own software than in someone else's, and those bugs were seemingly quite easy to fix too.
