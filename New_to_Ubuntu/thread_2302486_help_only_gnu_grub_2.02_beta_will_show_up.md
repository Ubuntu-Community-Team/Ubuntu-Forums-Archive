### Post by sandyd on 2015-11-10
Most computers have text at the bottom-right corner of the screen when starting up (It flashes past very quickly for some) that indicate what buttons you should be pressing to get to the boot menu or bios to set which device to boot from first. If it does not boot when using the boot menu, or changing the boot order to boot from the CD/DVD first, then check your CD/DVD in another computer to see if it is working.

Without the computer model, it is not possible for me to tell you directly which key to press. Probably try F12 or some of the function keys if you can't see which key should be pressed.
