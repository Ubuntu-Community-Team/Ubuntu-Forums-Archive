### Post by GMachine_24 on 2009-10-25
Hi:

I have an desktop which pretty much does nothing but run Squeezecenter software which allows me to play music from that computer on any other computer, etc.

Squeezecenter is today known as Squeezebox Server.... I think... anyway... it started as a very compact piece of software years ago called Slimserver.....it was fast.....as I said, compact, and very stable. Then Logitech took it over. At times there were issues running the software on various Ubuntu versions, which is why I started running it on a comp running Fedora.. but that was many years ago.

Anyway, it allows you to search through your music by artist, CD, genre etc. and.... yes.......folders. So maybe this would be worth a try.

Mine runs on a computer with Ubuntu 8.04. Don't bother trying to get the Squeeze software player - which would reside on the computers you want to use to play the music - as I had nothing but problems with that. But you can use amarok, vlc media player, songbird........ any software that will play from a streaming source over a network. It's really very simple.

[http://www.logitechsqueezebox.com/support/download-squeezebox-server.html](http://www.logitechsqueezebox.com/support/download-squeezebox-server.html)

I do not run the most up-to-date version .... mine runs fine and I have no need to update it.

Hope this helps.
