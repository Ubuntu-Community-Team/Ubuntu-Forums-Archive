### Post by Crafty Kisses on 2008-12-18
Try running the following command:
```
alsamixer
```
Check and see if all the volumes are turned up.

---

### Post by fooman on 2008-12-18
if you double-click on the speaker/volume icon in the top, right panel...you will see some sliders in the alsa-mixer.  try turning those all up (pcm ?) and see if it helps.
