### Post by Chesamo on 2010-10-23
I've done IMAP with Thunderbird before, and it just picked up my IMAP folders... so it should be a server configuration issue.

Though I believe there's a way to "subscribe" to individual IMAP folders... switch to All Folders mode using the arrows above the side panel, right-click on the account name and hit "Subscribe...". You should be able to select your IMAP folders in there.
