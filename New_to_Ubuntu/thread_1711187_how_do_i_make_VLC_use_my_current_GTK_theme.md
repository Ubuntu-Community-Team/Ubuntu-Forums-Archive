i found a solution

to change VLC QT interface to GTK+
-------------------------------------------------------------------------------------------------------
go to Ubuntu Software Center

install QT 4 Settings

after finishing install, open Menu>Settings> QT 4 Settings

Appearance>GUI Style>Select GUI Style = GTK+

exit and save changes
