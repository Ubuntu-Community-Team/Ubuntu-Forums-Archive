It might be as simple as a "rogue" application eating up CPU or RAM.

System > Admin > System Monitor 

Look in the Processes Tab to see if anything is using a large chunk of RAM or CPU.

Slow performance might also be due to a dodgy graphics driver - have you enabled a graphics driver or installed a new one?
