### Post by eightmillion on 2008-08-08

### Post by tjwoosta on 2008-08-08

that way it would just make the arrow under the launcher disapear when i minimized windows rather then showing the minimized window

after that i offset the launcher arrows in the preferences to the point where they are no longer visible

now when i minimize windows it shows nothing on my dock (but you must have launchers for everything you minimize, if you minimize a program that has no launcher it will minimize to dock like normal)
