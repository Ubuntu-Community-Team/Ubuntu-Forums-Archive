### Post by HotShotDJ on 2009-01-03
> **keithpeter said:**
> I've already noticed that the 'Various Artistis' category is getting very large as abcde seems to tag any multiple artist CD into that category.CheersI've always found grip to be a great tool for ripping cd's.  It is in the repositories, so give it a spin!  As far as the "Various Artists" thing, I believe that is a function of the way that AmaroK sorts things.  I, too, have a very large collection of compilation CD's due to my previous occupation.  I've found that by using Grip, which allows you to code the artist field per track rather than per album, you'll be better able to get a usable sort from AmaroK (or in my case, Rhythm Box).
