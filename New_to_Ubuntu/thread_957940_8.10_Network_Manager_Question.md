### Post by igknighted on 2008-10-24
What is your wireless card?  NetworkManager doesn't "recognize" wireless card, it simply configures them.  The question is whether Ubuntu will automatically recognize which driver to use with your wifi card.  Type ```
lspci | grep network
``` and post the results.
