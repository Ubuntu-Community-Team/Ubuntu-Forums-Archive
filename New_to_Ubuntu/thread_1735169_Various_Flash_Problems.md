### Post by NecroLycan on 2011-04-21
Ah, the good ol' flash problems. 

I am not sure why you are getting this problem, because I have been using Linux systems (and more recently Ubuntu systems) for the past 5 years. I've had my fair share of flash frustrations, but as of now I am running on Ubuntu 10.10 (which is awesome, everything works perfectly). 
Perhaps make sure if everything is updated? 
In your browser (whether it being Google Chrome or Firefox) type in (in the address bar) "about:plugins". This should give you an overview on what you currently have (and are using). 
Note that Adobe also has a lot of problems (One thing is that they hate Macs). My flash works really well and I am able to engage and play games online like runescape and DeadFrontier. These require quite a lot of flash. 

When I type in "about:plugins" in my Chrome Browser (I recommend using this one, for it's friendlier user interface), I get: 

>  

Java 6 - Version: 1.6.0.24
The next generation Java plug-in for Mozilla browsers.

Shockwave Flash
Shockwave Flash 10.3 d162

Default Plug-in - Version: 1
Provides functionality for installing third-party plug-ins

I do not have any adobe flash player listed. But as far as I know, all youtube videos play perfectly, and I am able to access my mail and flash websites (like Picnik, Psykopaint, Runescape) etc. 



Hope this helps!
