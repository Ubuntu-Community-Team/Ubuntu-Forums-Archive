### Post by perce on 2009-09-08
I guess the problem comes from Samba, because CUPS usually doesn't ask for username and password. 

I don't know about Samba, but you can configure CUPS through a graphic interface, instead of messing with configuration files: go to System > Administration > Printing, and, on the printers configuration window,
go to Server > Settings.

If your Ubuntu computer always has the same IP address, you can set up the printer in Vista without Samba (i.e. without MS black magic) following [ thread. ]("http://ubuntuforums.org/showthread.php?t=1248007")
