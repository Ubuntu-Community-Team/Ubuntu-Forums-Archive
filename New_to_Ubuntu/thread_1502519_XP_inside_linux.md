### Post by John Bean on 2010-06-05
> **saakeman said:**
> It's just like a normal pc inside a pc

Correct. But for obvious reasons it performs like a slower PC than the real PC that hosts it.

Additionally the RAM allocated to a guest can never equal the real RAM available to the host, although it can come quite close if you don't need to run multiple VMs or memory-hungry apps in the host while you're running the VM.

All in all it works very well so long as the host system has good resources and performance to start with. No free lunches though.
