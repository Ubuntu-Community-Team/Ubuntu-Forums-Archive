### Post by snowpine on 2008-10-29
Hi Loafer,
First things first, the Hotmail website is lying to you... you do NOT need to upgrade your browser to use hotmail! I am using the latest Firefox and I still get that warning. You can ignore the warning and click ahead to read your email. I think Microsoft puts that warning there to scare people into using Internet Explorer. :(

If you install Ubuntu 8.04 Hardy Heron (the current stable version) or Ubuntu 8.10 Intrepid Ibex (the "testing" version), they will automatically include Firefox 3. I have never used Linpus so I can't help you there.

The only warning is that there's no guarantee Ubuntu will work "out of the box" with all features of the Aspire One. Definitely check out a user forum such as this one for tips and tricks: [http://www.aspireoneuser.com/forum/](http://www.aspireoneuser.com/forum/)

Good luck!
