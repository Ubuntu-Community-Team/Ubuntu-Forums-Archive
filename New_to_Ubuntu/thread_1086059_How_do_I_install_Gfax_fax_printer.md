I haven't actually sent a fax yet, because I need somebody to receive it.  

However, efax-gtk does not support a print-to-fax capability so I installed gfax.  I can launch gfax from the main menu, but it is also supposed to support a print-to-fax.  The problem is that I don't know how to create a "gfax printer".  gfax did not create one by default.

Any suggestions on how to create a "gfax printer" so I can easily fax right out of applications?

My solution to my gfax problem:
Instead of using gfax I switched to efax for the print-to-fax capability.  I wrote up a couple page paper on getting the internal "winmodem" and efax working on my Dell Latitude D610 laptop.  This is included as an attachment to this message (see below).  Note:  The attachment was written with OpenOffice Writer 3.0.  In order to open it I had to first save it to a file and then open that file.

Finally I uninstalled gfax with Synaptic.

The reason I say that the problem is only partially solved is that I can't make a long distance fax call using my MCI phone card.  See [http://ubuntuforums.org/showthread.php?p=6929191#post6929191](http://ubuntuforums.org/showthread.php?p=6929191#post6929191)
