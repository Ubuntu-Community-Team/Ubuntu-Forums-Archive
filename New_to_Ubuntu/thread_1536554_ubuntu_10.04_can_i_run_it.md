Other than much faster boot/shutdown with 10.04, I don't notice any performance difference from 9.04 or 9.10, possibly because ext4 filesystem is more efficient. Acer netbook, 1gb RAM, Atom processor, 32gb SSD. Visual effects **OFF**. 

20 second boot, 4 second shutdown, most apps (other than Gimp) load in under 4 seconds (SSD is responsible for most of this performance). With Chrome and several tabs, and terminal running, system monitor shows about 330 mb of RAM committed; at fresh start, about 150 mb. Swap almost never gets used.

I'm big on RTFM. Before installing, download and read "Getting Started with Ubuntu 10.04." 

Also read and print: [https://help.ubuntu.com/community/Grub2](https://help.ubuntu.com/community/Grub2)  Many of the 10.04 installation issues have involved the switch from grub legacy to grub2. If you read the grub help, you should be able to recover calmly from these pitfalls.

Clean install is preferable to upgrade.
