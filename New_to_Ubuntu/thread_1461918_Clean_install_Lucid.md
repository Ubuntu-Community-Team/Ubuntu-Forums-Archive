If you have been running Ubuntu and have been doing updates you have the same as a daily build which means you are at this point running the same as is downloaded  version. You can just install the Kubuntu Desktop and choose between the Ubuntu desktop you have now or the Kubuntu desktop by choosing either at the login gui. I would do that to see if Kubuntu is what works for you, personally I don't like it, but others do. 

As far as a complete reinstall you will not be leaning or cleaning your computer, it is Linux it cleans itself by not having all the temp files inherent in Windows.

If your set on reinstalling back up the other OS use the gparted partitioner on the install disc of Linux to remove the Ubuntu partitions and reinstall in that empty space.  

I am assuming here that when you say dual-boot that you have MS and Ubuntu on separate partitions and not a wubi install, which would have been installed from a running Windows setup. If your running wubi disregard all of my instructions as I am just not familiar with it and or it's limitations. So make sure in the future to identify the type of install, good luck and welcome to open source.
