### Post by tjwoosta on 2008-05-09
it is very easy if all you want to do is change colors

go to System-Preferences-Appearance

choose one of the themes then click customize you can then choose the colors to use for everything

you could also get whole new themes from gnome-look.org and add them by simply draging the file into the theme selector window

also if you are using compiz fusion you could go to gnome-look.org and use the beryl emerald themes 
(you would need to install emerald theme manager for this) it should be in synaptic package manager


linux is well known for the ability to customize everything and anything you desire
