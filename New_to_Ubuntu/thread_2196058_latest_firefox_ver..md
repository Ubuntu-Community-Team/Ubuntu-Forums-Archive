### Post by Impavidus on 2013-12-27
If you're running a supported Ubuntu, you should automatically get a notification of a new firefox within days of its release, as part of the normal update system. Just install the update. If not, you may have to change some settings. Open the software centre, click Edit&#8594;Software sources&#8594;Updates and change the settings as appropriate.

If you can't find it, the terminal always works.```
sudo apt-get update
sudo apt-get upgrade
```The first is to check for updates, the second to install them.
