How to stop fading screen after some times?
For example: After some time my screen light become darker)

---

### Post by mcduck on 2009-09-15
System/Preferences/Power Management, turn off the "Dim display when idle"-option. Also note that if you are using a laptop, there will be separate settings for both AC and battery power.
