Type *about:addons* in Firefox search bar to access the Add-ons Manager.

The Global Menu is part of Ubuntu, but to make it work with Firefox it needs to install an add-on on it. So, disabling the add-on is enough to prevent Firefox from using the global menu.

Keep in mind that when you First disable the global menu, you won't see the original Firefox menu bar, but the new Firefox button. You need to right-click the navigation toolbar, then then tick the "Menu Bar" option. When you do that, the Firefox button goes away and the Menu bar is displayed.

It's both!  It's the menu for Firefox, but it's shown in the Ubuntu top panel.  11.04 has a global menu (if you've used a Mac, you'll have seen it before) which means that most programs now have their menus in the top panel, but don't show them unless the mouse is on the top panel itself.  So you can use that if you want to save the web page.
