### Post by themusicalduck on 2010-03-05
If you install Ubuntu with wubi then while using Ubuntu it will be just as secure and virus resistant as a normal install.

Ubuntu is installed on a virtual disk which is stored as file on Windows. So it is still technically possible that something could happen to that file if a virus is installed on Windows while you are logged in to Windows. Also it is prone to fragmentation and any other filesystem problems Windows might have. Also, it won't run as fast as a full install.

However, if you can't boot up the disk to install it, it's unlikely a wubi install will work anyway.. Especially if your netbook has been described as unlikely to work with linux.

Unless you mean to install Ubuntu inside a virtual machine instead?
