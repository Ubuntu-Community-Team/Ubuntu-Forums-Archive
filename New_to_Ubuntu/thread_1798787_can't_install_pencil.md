### Post by ubunwhat on 2011-07-06
This means that the program requires libming0, which it is not finding.  Check in Synaptic to see if libming0 is installed.  If it is not installed, then install it and you are good to go.  If it is installed then you are having an issue with the application wanting a different version.  That's a different kettle of fish, and there is no point worrying about it up front if all you need to do is install libming0 in Synaptic.
