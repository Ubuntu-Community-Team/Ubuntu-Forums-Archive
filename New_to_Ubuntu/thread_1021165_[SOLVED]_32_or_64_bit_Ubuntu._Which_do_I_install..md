How much ram you can address is one reason to use a 64bit OS.  There are many more though.  Since you can pass more complex information to the CPU, many CPU intensive tasks can be performed much quicker (encoding audio/video, applying effects in GIMP, encrypting/decrypting, etc.).  Not to mention any kind of HD Video will almost certainly play better in 64bit.

Since there are very few 32bit only applications (skype is the only one coming to mind right now), and there are work-arounds to get even those apps going, there really isn't much of a reason not to go 64bit if your system is capable.
