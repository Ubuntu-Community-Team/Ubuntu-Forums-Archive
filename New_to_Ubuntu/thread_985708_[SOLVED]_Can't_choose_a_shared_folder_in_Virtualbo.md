### Post by FAMUgolfer on 2008-11-17
Wow. Of course as soon as I post my problem I solve it on my own in a matter of moments.

My solution:
Virtualbox automatically creates a name for your folder once you select it, and all you have to do is modify that folder (I guess so there is no confusion between the two, which I do not remember doing in hardy or intrepid RC).  Once this is done, the "OK" button can be selected.
