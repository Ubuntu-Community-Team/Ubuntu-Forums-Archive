### Post by daniel_szollosi-nagy on 2008-06-03
I'm sure someone else will give you a technical answer to how you can restrict download for user. Running Windows programs (including trojans and viruses) is not possible by default in Linux. You have to run Windows programs explicitly through a program called wine. Some programs work fine under wine (see [http://appdb.winehq.org](http://appdb.winehq.org) for an exhaustive listing on how well certain things work). Also see [http://ubuntuforums.org/showthread.php?t=72598](http://ubuntuforums.org/showthread.php?t=72598)
