### Post by pedja_portugalac on 2009-11-12
Maybe you'd like to install p7zip-full. p7zip is the Unix port of 7-Zip, a file archiver that archives with very high compression ratios.not only does it handle 7z but also ZIP, Zip64, CAB, RAR, ARJ,GZIP, BZIP2, TAR, CPIO, RPM, ISO and DEB archives. 7z compression is 30-50% better than ZIP compression. It work from terminal. You change to the directory where the file-s are stored and write: 7z e <name of the file or first file if separated> 

To install write in terminal:

sudo apt-get install p7zip-full
