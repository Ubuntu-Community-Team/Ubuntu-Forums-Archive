### Post by snowpine on 2009-11-10
Hi Eaglehacker, the question does not make sense... a "tarball" is a type of archive (like a .zip in Windows)... so your question is like asking "how do I install a .zip file in Windows?"

Anything could be inside the archive: source code, executables, photos from your vacation, etc.

Can you be more specific about where you got the tarball, what's inside it, and what's your end goal?

If you're trying to install ndiswrapper (can't tell from your question), typically you'd:

```
sudo apt-get install ndiswrapper
```

(while connected to the web with a wired connection)
