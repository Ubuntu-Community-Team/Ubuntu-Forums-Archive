### Post by Mk32 on 2014-04-08
It's not unusual at all. 

The .iso is issued at a certain time, and then afterwards updates trickle down the pipe. After some time, a large number of the install .iso packages are out of date. 

Typical on Mac OS X and especially Windows. I think my Windows 7 install has 10GiB worth of updates just for the OS. (It's 32-bit Ultimate.) Mac OS X -- not quite so much as it used to be, as computers would come with install CDs or DVDs (usually machine-specific), whereas now it's all done over the internet.
