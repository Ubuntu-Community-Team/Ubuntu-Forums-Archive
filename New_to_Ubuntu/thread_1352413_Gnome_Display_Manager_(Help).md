### Post by 4Orbs on 2009-12-11
Go to the System menu on your top panel (taskbar), click on Login Window (or Login Screen if that's what it is called). That should open the Login Screen manager which has a few tabs at the top, one of which is titled Local, another is titled Advanced, and a couple more which I don't recall the titles. Click on the Local tab and it should open the window showing pictures of the different Login screens you can choose from.

I'm still running 9.04 and I'm not sure if its changed however this is how I would do it.  Go to Administration --> Login Window.  Type in your password(assuming you're the admin) then in the window that opens go to the Local tab.
