### Post by mcduck on 2014-03-27
Holding down Alt key allows you to drag windows around from any point in the window (instead of just the title bar like usually). That should allow you to move the window higher so you can access the OK button.

However it sounds strange yu'd be running on such a small resolution these days that the dialog in question wouldn't fit on the screen. A screenshot might be a good idea so people can help you fix that...
