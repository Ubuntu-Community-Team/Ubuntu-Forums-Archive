### Post by logos34 on 2008-11-27
ogg is the default format.  you can change it 

if you tell it to 'create audio cd' it will convert the lossy format (ogg, mp3, whatever) so that the disc plays in any cd player.  But if you just burn the files to disc as data, it won't playback (unless they're mp3 tracks and you have a cd player that supports mp3). 

You might want to check[ this page]("https://help.ubuntu.com/8.04/musicvideophotos/C/index.html") out.
