### Post by Peter Frank on 2008-09-16
Possible solutions:
1. Run Windows in a virtual machine under Linux.
2. Install the Eclipse IDE in Synaptic as an alternative.
3. Install 'mingw32' in Synaptic, which is a cross-compiler that runs under Linux but generates .exe Windows executables.
