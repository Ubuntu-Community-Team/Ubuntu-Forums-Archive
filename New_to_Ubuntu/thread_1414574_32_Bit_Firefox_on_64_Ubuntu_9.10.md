### Post by Drone4four on 2010-02-23
Why would you want to install 32bit Firefox on 64bit Ubuntu?  If it's because of lack of 64bit Flash support, think again.  You can download a 64bit Flash library from Adobe's website.  Google it.  

I use Google Chrome instead of Firefox.  IMHO Chrome > Firefox.  64bit Chrome runs 64bit flash better than 64bit Firefox runs 64bit Flash.
