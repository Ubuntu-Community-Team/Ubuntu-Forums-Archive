### Post by Joe Curtis on 2011-11-17
In order to get AutoCAD working in the Ubuntu desktop environment, I would suggest using [COLOR=RoyalBlue]_**[Wine]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=86")**_[/COLOR], which lets you run Windows software on a Linux distro.  I highly suggest checking out [COLOR=RoyalBlue]_**[this thread]("https://answers.launchpad.net/ubuntu/+source/wine/+question/145910")**_[/COLOR] which helps with more info.  It appears that Auto CAD in particular has very test results for Wine support.

Do you already have saved work from AutoCAD version 14 that you want migrated over?  If you are just looking for an alternative program [COLOR=RoyalBlue]_**[this list]("https://help.ubuntu.com/community/UbuntuEngineering")**_[/COLOR] can h[COLOR=Black]el[/COLOR][COLOR=Black]p you get[/COLOR] started.

Hope that helps you find the right direction!
