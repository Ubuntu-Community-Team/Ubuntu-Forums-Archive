### Post by dje on 2008-05-01
Go to System >> Preferences >> Sessions and click on 'Add' 
1. For name put whatever you want 
2. Then in the command box put:
```
avant-window-navigator
```
3. You can then add a comment if you want and close the window, now when you login awn will start

hope that helps,
dje
