### Post by Mark Phelps on 2011-06-09
If you're going to buy a laptop with Win7 preinstalled, your choice can be affected by HOW you want to then use that with Ubuntu ...

If you want to dual boot Ubuntu and Win7, Dell has a nasty habit of preloading their Win7 laptops with the maximum number of four primary partitions.  This makes a lot of otherwise unnecessary work to then convert that into a true dual-boot machine.

Don't know about Toshiba, but don't see nearly as many complaints from folks on this forum about those.

If you're going to wipe the drive and install Ubuntu by itself, then it comes down to which has better driver support.  You should browse the Laptop forum to see if other folks have posted about either of these models -- to see if anyone has reported their experiences.
