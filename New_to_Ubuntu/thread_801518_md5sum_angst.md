Serious enough that you shouldn't use the iso to burn to a cd. I'd either download again (if you know how to use a torrent, that almost always gives the best results for file integrity) and run the md5sum check on the iso. If it passes, then burn it to a CD-R (my experience is that this works best) at a slow (4x) speed, and use it. Otherwise, you could use the 7.10 CD from Canonical and install from that (although 7.10 was not without its own peculiarities), and after fully updating, try a distribution upgrade through the upgrade manager (I've done this without issue, others have had some major problems). Best of luck.
