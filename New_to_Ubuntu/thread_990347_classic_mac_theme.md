### Post by macksgarage on 2008-12-17
Most people don't seem to grasp the idea of classic mac or platinum themes, as evidenced by the posts above. I have the same problem, I haven't been able to find an MacOS 8/9 theme.  However, I did find a theme called xfce-smooth that is very similar, feels almost like an update to the old mac theme, style wise. same gentle greys, clean simple design. it's provided by the gtk2-engines-xfce package, which does not depend on xfce, it's just the gtk themes that ship with it.

there's another hidden feature that's useful in this endevour. metacity can have the toolbar buttons rearranged, using gconf-editor. the key is apps/metacity/general/button_layout.  download the package gconf-editor, run program by the same name, which is like a windows registry editor for gnome, and you can flip the buttons to the other side. also works with compiz using the default gtk-window-decorator, which uses metacity themes and (some) configuration data.
