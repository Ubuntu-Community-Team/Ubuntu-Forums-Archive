### Post by Sir Jasper on 2009-07-30
Hi,

AVG is an excellent program but, from memory, it is quite complicated to install; also, all the av programs only detect Windows malware (Linux malware is as good as unheard of) so if anyone is only using a Linux Operating System (eg Ubuntu) they do not need a Windows av. 

The Debian (on demand) version of Avast is ideal for Ubuntu especially if the user has broadband (as the daily updates are full as opposed to incremental). It can be downloaded to the desktop, then double click and it will be automatically installed under Accessories.

It takes me about six minutes a day to update Avast and do a thorough scan on my (smallish) Xubuntu Home directory.

Eliminating Windows malware from a Linux system may help to protect friends with Windows sytems (eg with clean outgoing emails) but, as stated above, unless your friend uses Windows as well Ubuntu, it will be of no direct help to them.

My regards
