### Post by duanedesign on 2009-04-19
the easiest way to access settings for Compiz is the CompizConfig Settings Manager. 

Open Terminal (Applications>Accessories>Terminal)  and type:

```
sudo apt-get install compizconfig-settings-manager
```

Once installed You can access all the effects settings.
