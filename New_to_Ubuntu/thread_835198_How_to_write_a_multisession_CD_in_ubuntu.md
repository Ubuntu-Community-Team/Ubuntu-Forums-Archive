### Post by sayakb on 2008-06-20
If you are using Ubuntu Hardy 8.04, you should have Brasero by default. Once you press *Burn* button in Brasero, you may **check** *Leave the disc open to add other files later* checkbox to burn a multisession disc.

### Post by Christmas on 2008-06-20
Or using K3b: after clicking on *Burn*, go to the *Misc* tab and select *Start Multisession* from the *Multisession Mode* dropdown list.
