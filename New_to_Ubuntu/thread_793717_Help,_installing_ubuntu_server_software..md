### Post by Xiong Chiamiov on 2008-05-15
If you've never used Linux before, not only would I recommend using an install that comes with a GUI (Ubuntu server does not), but I wouldn't suggest that you should be running a server open to the wide world.  If you don't know at least the basics of how to handle a desktop system, you'll be in trouble trying to administrate a server.
