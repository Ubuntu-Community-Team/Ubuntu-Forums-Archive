### Post by Impavidus on 2014-05-22
To give the complete explanation: Although Intel developed a 64 bit processor of their own, it was AMD that developed a 64 bit processor that was backward compatible with 32 bit software. The instruction set used by this AMD processor is known as AMD 64. This proved to be an advantage, so Intel started using the same instruction set and Intel processors are therefore compatible with AMD 64. They arranged it via some sort of patent exchange, where not only Intel used AMD's 64 bit technology, but AMD also used some technology developed by Intel.

In short: the AMD in amd64 is there for historical reasons, it works on Intel too.
