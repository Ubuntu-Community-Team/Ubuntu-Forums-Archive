### Post by Therion on 2009-06-08
> **knuckle dragging monkey said:**
> well i have been trying to install some wallpapers but they are jpeg and when i drag and drop onto themes preferences. it tells me that it's an invalid file or something like that.

You don't drag-and-drop pictures to use them as wallpaper pics.  Here's what you can do:

Put the .jpg file you want to use as your wallpaper in a new folder in your existing /Home folder.
Open your Appearances menu.
Click on the Backgrounds tab.
Click on the "Add" button.
Navigate to the new folder you created in /Home that has the .jpg file you want to use and select the file to set it as your wallpaper.
It will be set as your new wallpaper and be in the selection menu (Backgrounds) from now on.
