Going back to the terminal prompt after a command has been carried out is quite normal and means that the command did exactly what was asked, and no errors were found, so the "clean" was done for you.  If there was a problem of any sort an error message would have appeared.

To start a new thread, open up the forum main page [http://ubuntuforums.org/index.php](http://ubuntuforums.org/index.php) and then click one of the sub-forum headings, eg General help, Desktop environments, etc etc from the listings.  That will take you to a new forum page for that sub-forum.

Read the stickies at the top of the list of posts, as they may answer your problems, but if not there is a dark gray button top left which says **New Thread**.  See screenshot for an example
