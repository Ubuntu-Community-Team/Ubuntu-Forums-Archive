### Post by t0p on 2008-09-11
There's a command-line utility, **youtube-dl**, that will save youtube vids.

```
sudo apt-get install youtube-dl
```

Unfortunately, youtube-dl hasn't worked for me in awhile. :(  But maybe you'll have better luck.

Another way to save a vid off youtube is to watch it online, then go to the /tmp directory. You'll find the vid file there, named something like Flashabcde.

I don't know of any way to save a streamed BBC show.  The BBC have done their best to stop us doing that!
