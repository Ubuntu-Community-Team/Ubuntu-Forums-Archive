### Post by martrn on 2009-03-12
If you install gnome (standard ubuntu install) and then

```
sudo apt-get install kubuntu-desktop
sudo apt-get install xubuntu-desktop
```

Then you will have a pretty simular set-up. (as installing kubuntu/xubuntu/etc), and one of the other desktops.  There are some minor art-work differences with each such as a boot up logo, but there are not any major noticeable differences, aside from a few installed packages.

Ubuntu/Kubuntu/Xubuntu all draw from the same repositories, so aside from whats on the installation CD there is no difference other than the destop will be gnome kde or xfce (or even enlightenment).

That is the difference, the desktop.

### Post by ajgreeny on 2009-03-12
With more than one DE you will probably end up with very much longer and more involved menus in your menu bar or gnome main menu, but apart from that it will make little difference and give you the best of two or three or even four DEs, if you want them.
