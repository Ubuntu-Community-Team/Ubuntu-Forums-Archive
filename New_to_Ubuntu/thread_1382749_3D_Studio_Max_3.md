### Post by hessiess on 2010-01-25
> **Slasher The Great! said:**
> well now the background doesnt bother me any more. what mainly intrests me is how to make guns apear in my hands by the way im only 11years old.

You would be better off using `fake' gun props. Adding a muzzle flare in post pro is relativity easy in comparison to adding the entire object in 3D. Doing so would involve a substantial amount of work with camera/object tracking, matching lighting, hyper-realistic texturing ec-cetera.

Anyway, this would not be possible with just 3DS Max because it does not have a compositor (needed to mix 3D/ live action footage).

Computer graphics as a whole is EXTREMELY complicated and getting good results requires years of practice. I would advise against trying anything this complicated to begin with because you almost certainly will not be able to produce the sort of result you are aiming for.

What you need to do is pick up a 3D graphics program, Blender is good for this because its free, then learn to model, unwrap, texture, rig, light, animate, render and composite. Then learn a 2D graphics program like Gimp, for texture painting. Then learn about the techniques involved with compositing 3D and live action, such as Green screen and camera tracking. Here things start to get expensive because you will need a half decent camera and lighting to be able to pull a decent matte.

If I haven't lost you yet, go to [http://www.blender.org/](http://www.blender.org/), download Blender and start following tutorials and generally playing around. Also sign up on [http://blenderartists.org/forum/](http://blenderartists.org/forum/).
