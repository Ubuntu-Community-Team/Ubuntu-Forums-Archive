### Post by snowpine on 2009-04-09
Old hardware is funny. I have heard of people using Puppy with only 96mb, but that does not mean it will necessarily work on your brother's specific hardware. Best way is just to try it. You can download and burn a Live CD to try it with no change to the computer.

[http://www.murga-linux.com/puppy/](http://www.murga-linux.com/puppy/)

If Puppy doesn't work, an even lighter suggestion is DSL (Damn Small Linux).

### Post by HereInOz on 2009-04-09
Puppy needs very few system resources.  A 266 Pentium with 64MB RAM will run it OK - with 128MB it will be very happy.

You can actually boot it and run it from the CD.  It is an excellent OS for an old machine.  

Download it, burn the CD, and boot from it.  See what you think.
