### Post by avtolle on 2011-06-03
fdupes runs from the command line. I suspect it is installed at /usr/bin/fdupes, but you can check by typing ```
which fdupes
```from the terminal (System>Accessories>Terminal). BTW, as it doesn't have a GUI, as I recall, it won't show up in the menu.
