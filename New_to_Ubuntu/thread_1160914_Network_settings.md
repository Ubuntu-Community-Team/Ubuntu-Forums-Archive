### Post by cariboo on 2009-05-16
To make any changes to any files not in your home directory you have to be root. In your case, click the unlock button and enter the password you created when you installed Ubuntu.

Linux is a lot different from what you are used to as a Windows user, I would suggest you download and read the [Ubuntu Pocket Guide]("http://www.ubuntupocketguide.com/index_main.html"), this will explain many of the question you have.
