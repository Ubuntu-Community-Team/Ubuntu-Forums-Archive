### Post by taurus on 2008-11-01
Just boot off the Ubuntu CD and follow the instruction on the screen.  When you get to the part where to install Ubuntu, pick the option to use the whole harddrive.  It will wipe out Windows and format your harddrive to ext3 for you.  No need to format it first if you don't want to.

### Post by Joeb454 on 2008-11-01
You can either change settings in the BIOS (change the boot order).

Often nowadays - PC's will have a "Boot From" option - i.e. mine is F12 - which allows you to choose what to boot from without editing the BIOS settings :)
