### Post by Xiong Chiamiov on 2008-05-10
Yes, wine is a great project that allows you to run windows programs on Linux.  Some work very well, others... not very well at all.  There are native Linux programs for dealing with iPods.

Will Linux recognize your hard drive?  Most likely.  Will it work the same way as in Windows?  Not really.  In Linux, you can "mount" a hard drive wherever you want on the system (instead of being restrained to C, D, E, etc.), so in that sense it'll be different, but you should be able to browse the folders just like in Windows.

The default CD download is what we refer to as a Live CD, meaning that you can run Ubuntu entirely off of the CD without installing it on your hard drive.  You can then run the installer once you are booted into the CD.

### Post by Sef on 2008-05-10
> the 'live cd' ? I just quickly looked at install methods, I was wondering, what exactly is this... is this just a cd boot (ie i wipe computer and start from cd just like installing xp), or do I not have to unistall xp? Is it like a test or something?


The Live CD is an operating system that runs off of the cd.  Once you have it up, you can install it to your hard disk.  If you want you can either install it to the whole disk or make a dual boot.  A dual boot is where you have two operating systems on one hard drive.  To read more about Ubuntu in general, including installing, read [Psychocat's blog]("http://www.psychocats.net/ubuntu/").
