### Post by wojox on 2012-11-06
Hey Norm, press Ctrl + Alt + T to open a Terminal window. Type the following command at the prompt and press Enter.
```
sudo apt-get update; sudo apt-get install gnome-session-fallback
```
After installation log-out of Ubuntu. While logging in back, click the gear button on the Password box and select GNOME Classic from the drop-down menu.
