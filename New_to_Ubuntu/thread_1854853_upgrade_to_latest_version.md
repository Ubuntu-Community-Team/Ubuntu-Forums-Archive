### Post by Linux_junkie on 2011-10-05
You cannot upgrade from 10.04 to 11.04 using update manager but you can do a fresh install of 11.04 which will overwrite 10.04.

If I were you I'd wait another week or so and "upgrade" to 11.10 (fresh install) if you really want to upgrade otherwise keep 10.04 which will still be supported until April 2013.

---

### Post by howefield on 2011-10-05
Probably worth adding that it is generally possible to upgrade directly from one LTS (Long Term Support) version of Ubuntu to the next eg, 8.04 to 10.04 (and to 12.04 when it arrives), but not the intervening editions where you have to move sequentially through them to get to the current edition.
