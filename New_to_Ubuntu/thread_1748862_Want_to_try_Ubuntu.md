### Post by 5149.5 on 2011-05-04
Log out, click your user name, look at the lower panel, click Ubuntu, select Classic, put in your password and continue the login. Now, Unity is gone and you have the classic Gnome desktop. Repeat the procedure but select Ubuntu and you have the Unity desktop back.

### Post by mastablasta on 2011-05-04
ok to add to this a very similar desktop to gnome is XFCE (Xubuntu) in terms of looks not in terms of GUI setup options and other things. XFCE is also a bit lighter than GNome in term of system resources use.
 
Then there is the KDE (Kubuntu) - if you are coming from windows you can right click the K and select classic menu -- then it looks a lot like windows. It can be made even more like windows with a few additional settings. Again unlike gnome KDE has different GUI settings and the way it does things. There are more options to choose from (for some too many) and all of them need to be "applied" before any changed are made (sort of like in windows)
