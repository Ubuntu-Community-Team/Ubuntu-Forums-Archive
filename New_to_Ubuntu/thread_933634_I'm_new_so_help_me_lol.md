### Post by Sef on 2008-09-29
> Printer access, how do I install the printer? Or is it working automatically?

What printer do you have?  Some like HP will almost always work once you plug it in and turn it on.  Epsons often work too.   Lexmarks almost never work.

> CD Programs, like my sister's mp3 player software that comes with it? How can she download that?

MP3 programs are provided by default.  If you don't like what is provided by default, then check out the others in Add/Remove.

> Audio - this is a big one! How do I install the SoundMAX, the speaker functions, etc. On XP I used a CD to install everything, so is it the same?

Post your problem in separate thread.

> Adobe Flash Player, Reader, Shockwave, Java, and other Internet downloads!

From the Ubuntu repositories.  Most of those are in Add/Remove (under applications.) Search for restricted - will locate ubuntu restricted extras.

> Give me installation/downloading 101 for the future

Read [Psychocats Ubuntu]("http://psychocats.net/ubuntu").

> Compiz, and the 3-D scope properties

From Add/Remove, download CCSM.  (Type that in search.)

> The resolution isn't quite that great, it's a little too small. Anyone recommend any solutions or what did you guys do?

Start a separate thread on that problem.

> Windows Live Messenger - My sister wants this! Anything I can do about it!

Either use pidgin - the default im.  It supports many ims.  Or download either amsn or emesene for msn clone only.  Again they are in Add/Remove - search by their name.

> Security - Do I need an anti-virus/anti-spyware?

Only if you use an email client, so you don't infect your Window using friends.  Those don't affect GNU/Linux.

> Fonts - It doesn't look like there's any familiar fonts, how do I fix this?

Again in Add/Remove, type in Core Fonts.

> Anything else you guys can help with

Always ask if you have a question.
