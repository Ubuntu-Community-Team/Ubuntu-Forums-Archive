### Post by ajcham on 2008-12-11
First, open the Windows menu.

Then try 'Recently closed docks' - the combined Layers, Channels, Paths dialog might be there.

If not the next option is Dockable dialogs, from which you can select the Layers dialog (alternatively, pressing Ctrl + L will also bring up the Layers window).
