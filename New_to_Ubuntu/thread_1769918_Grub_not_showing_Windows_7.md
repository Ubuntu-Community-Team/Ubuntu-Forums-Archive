Was your windows installed to an older (XP?) windows partition. Windows starting with Vista starts its partition at sector 2048, XP started at 63. Older installer used to move the windows partition from 2048 to 63 which then  required some windows repairs. 

I would try fixboot command from a Windows repair console. If you do not have a Windows repair CD see my post #9. Otherwise I copied a bunch of commands in post #7. You may need chkdsk and fixboot.
