### Post by Lars Noodén on 2012-01-03
It's an open secret that any SIP client will talk to any other SIP client if you know the other person's "phone number"   So that gives you a choice of many options, none of which are mutually exclusive.  They can all talk to each other.

Blink, Ekiga, Kphone, Linphone and Twinkle are four to get started with.  Lately, I have been using Blink a lot.

What Skype has, besides dreadful security problems, is very good word of mouth marketing.  It's something that the SIP tools need to overcome.
