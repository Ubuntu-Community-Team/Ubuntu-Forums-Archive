### Post by stalkingwolf on 2009-02-15
you can try rebooting.  When Grub starts hit ESC.  this will bring up the boot menu.  Select recovery mode ( should be the second ubuntu choice),
hit enter.  when it finishes a window will open offering several choices.
select " fix broken packages" hit enter. when it finishes reboot.

Hope that helps.
