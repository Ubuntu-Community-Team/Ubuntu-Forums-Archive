Both of you are incorrect.  You can sometimes recover lost partitions and files.  

For recovering lost partitions: get [Test Disk]("http://www.cgsecurity.org/wiki/TestDisk")

For recovering lost files: get [photorec]("http://www.cgsecurity.org/wiki/PhotoRec").

Also run the Live CD and do not use the hard disk.  You have a better chance of recovering lost files if you do not use the hard disk.
