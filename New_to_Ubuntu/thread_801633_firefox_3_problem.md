### Post by tamoneya on 2008-05-20
you can revert to firefox 2 if you like.  Type the following into terminal```
sudo apt-get update
sudo apt-get remove firefox-3.0
sudo apt-get install firefox-2
```That should even keep all your book marks and such but I would back them up just in case.
