### Post by Marshal0505 on 2008-09-03
I used to have the same problem and fixed it by editing my xorg.conf to a resolution appropriate for my monitor.So in my case i added the line;

metamodes" "1600x1200_85 +0+0" 
to the "screen" section, after doing ```
sudo gedit /etc/X11/xorg.conf
```
You might want to adjust 1600x1200 to personal preference.

This made my entire login screen visible again.
