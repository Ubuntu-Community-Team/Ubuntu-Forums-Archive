### Post by lovinglinux on 2010-07-02
Is probably an extension like AIOS interfering with the sidebar behavior. Start Firefox in safe mode to check if this is the problem.

To start Firefox in safe mode, close any Firefox instance, then open a terminal and type:

```
firefox -safe-mode
```

Also see Firefox [[COLOR="Sienna"]**General Troubleshooting Steps**[/COLOR]](http://firefox-tutorials.blogspot.com/2010/05/general-troubleshooting-steps.html) tutorial.
