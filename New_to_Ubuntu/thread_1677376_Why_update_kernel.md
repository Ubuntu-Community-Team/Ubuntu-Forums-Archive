### Post by cariboo on 2011-01-28
Most updates after a version is released are for security reasons, If you feel you don't need then, just set update manager to never notify you of updates.

I'm running Natty here, I've lost count of the kernel updates we've had so far, but there hasn't been a problem with any of them, there have been lots of problems with other packages though.

I hate to tell you this, but this is not limited to kernel updates and in fact kernel updates are the easiest to fix, simply boot an old kernel, lol.

Many new users update twice a day, as you get more experience you learn to update less often and to check the forms and launchpad for bug reports first.

Personally I update once a week or so.

I generally review the package list, some packages can be more problematic then others and if the system breaks I want to know exactly what was updated.
