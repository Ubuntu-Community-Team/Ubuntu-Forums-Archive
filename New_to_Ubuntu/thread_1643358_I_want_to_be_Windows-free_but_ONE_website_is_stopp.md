### Post by TCHebb on 2010-12-11
If the site uses some kind of ActiveX or IE-only feature, you might be out of luck (you can try running IE in wine, but that's not guaranteed to work), but if actually works with Firefox and is blocking it for no reason, you can try the [User Agent Switcher]("https://addons.mozilla.org/en-US/firefox/addon/59/?id=59") add-on to identify yourself to the site as using IE. Ultipro itself seems to support Firefox, so this solution is likely to work.
