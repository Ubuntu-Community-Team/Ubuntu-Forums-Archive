### Post by webwiller on 2009-10-17
SORTED OUT AS FOLLOWS:

- Insert iso dvd to be copied
- Right click on disk icon on desktop
- Choose "copy disk"
- In the shell that appears choose on the bottom line "make a copy of the image disk" on /[user]/home/Brasero/[file].iso
- After copying has finished open Brasero and burn an image disk choosing the file from the path above set.

- Done!
