### Post by ajgreeny on 2010-11-16
It has to be all icons, and all other actions as set in the nautilus preferences dialog.  It will also affect nautilus itself, so a single click will not select, but will cause the file or folder to open.

It may seem strange to you at the start, but you quickly get used to it.  It is the way I have had my system set up for about 5 years, and I now find it very odd to have to use a double click for actions to occur.
