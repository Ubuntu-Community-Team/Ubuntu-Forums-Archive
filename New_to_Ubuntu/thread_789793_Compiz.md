### Post by cgarre on 2008-05-10
1. Make sure your panel has workspace switcher. (Right click on gnome panel - thats the one which is equivalent to taskbar of windows) then select Add to Panel..., and search for "Workspace Switcher" . 

2. Once you have that right click on that and select "Preferences". Make sure you have Rows X Columns = 4. So if you put Columns as 4 , your rows should be 1 (I use that settings). 

3. Now in compiz settings , make sure you have "Desktop Cube" and "Rotate Cube" both enabled. If Rotate Cube is not enabled, cube will not rotate, and also though it shows 4 workspaces you will not be able to go to any other workspace. Make sure desktop wall etc are not enabled.
