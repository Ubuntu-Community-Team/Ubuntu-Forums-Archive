### Post by issih on 2008-05-27
Hmmmn, almost everything is configurable, so this is a tricky question....From memory, the defaults I remember are:

Copy, Cut, Paste:                Ctrl C,X,V
Show Desktop:                    Ctrl Alt D
Move to Workspace left/Right:    Ctrl Alt Left/Right Arrow
Move window to Workspace l/r:    Ctrl Alt Shift Left/Right Arrow
Run Dialog:                      Alt F2
Main Menu:                       Alt F1
Close Window:                    Alt F4

I think those are right, but it is from memory, you can get a better idea by opening System->Preferences->Keyboard Shortcuts. There you can see what is currently set up and change them to your own liking, I always bind a terminal window to ctrl alt T as I find it useful, but its your box, do what you want :)
