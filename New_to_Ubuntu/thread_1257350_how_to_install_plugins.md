The user interface is a lot like Windows, in many ways, it's just a little unfamiliar, that's all. 

A quick way to install many common programs in to use the Add/Remove feature.

Click on the menu Apllications and at the bottom you'll see Add/Remove. 

There's another similar but more finely tuned install manager called Synaptic Package Manager, you'll find it in the menu under System/Administration.
