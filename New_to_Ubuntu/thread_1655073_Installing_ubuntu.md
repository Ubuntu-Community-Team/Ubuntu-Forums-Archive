### Post by rockerphil on 2010-12-29
first and foremost, welcome to Ubuntu and the Ubuntu community. secondly, i believe you should definitely start with some basic reading material to get yourself acquainted with Ubuntu, so here are a couple of links:

[http://www.ubuntupocketguide.com/index_main.html](http://www.ubuntupocketguide.com/index_main.html)
[http://www.psychocats.net/ubuntu/index.php](http://www.psychocats.net/ubuntu/index.php)

these are some good starting points, and google can come up with a lot more reading material. now as to your question about the laptop being resurrected without having to reinstall Windows, it's very possible so long as you have an internet connected computer in which you can download and burn an install CD. the file can be downloaded here:

 [http://www.ubuntu.com/desktop/get-ubuntu/download](http://www.ubuntu.com/desktop/get-ubuntu/download)

if the computer is a bit older you can use Xubuntu, which uses the lighter weight XFCE desktop environment, or if you prefer something more similar to Windows you can use Kubuntu which uses the KDE desktop environment, both of which are fully supported here on the ubuntu forums. i hope this info helps, and if you have any more questions feel free to ask,

Phil
