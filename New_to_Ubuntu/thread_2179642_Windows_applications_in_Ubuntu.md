### Post by sudodus on 2013-10-09
I think it depends on the applications. Wine is faster, but it works (well) for a small fraction of the Windows programs. See this link

[http://appdb.winehq.org/](http://appdb.winehq.org/)

Windows in VirtualBox is much more likely to work (most programs work), but is slower due to the virtualization. It also needs more RAM to work well. So if you let us know what application programs you want to run, you can get more precise advice.

There is also a third alternative: Use a native program in linux, that can do what you intended to do with the Windows application program.
