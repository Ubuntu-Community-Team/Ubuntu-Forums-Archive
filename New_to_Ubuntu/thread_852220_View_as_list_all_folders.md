### Post by SIStem_shock on 2008-07-07
How can I set all folders to "view as list" instead of icons? I'm having to set it on each folder individually right now, and it doesn't always seem to keep the setting, either. Is there a way to set it to always view all folders in list view?

Also, I accidentally set a picture as the background of a folder view and don't know how to remove it.

---

### Post by tuxneo on 2008-07-07
In nautilus, go to **Edit -> Preferences** and select "list view", instead of "icon view" (in the "default view" section) ;)

Edit : for the background, you just have to go to **Edit -> Background and Emblems**, select the "Reset" item and drag it to your background (in the right folder)

And drag into window
