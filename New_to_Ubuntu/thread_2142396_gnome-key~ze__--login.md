### Post by Mark_in_Hollywood on 2013-05-05
There is no question to be answered here. This post is for Admins and Developers, mostly.

On power up, the hard drive made more noise than usual. Opening iotop showed this:

gnome-key~ze --login

as running like crazy, using about 1300 kb a second. (That's a lot, even though I have a sata 3.0 gig drive)

I used Alt-SysRq-R-E-I-S-U-B-K to warm re-boot. That brought up my XUbuntu in a typical manner. But there is nothing on the web with gnome-key* and I post this in case this is part of the Apache2 server attack: [Linux/Cdorked.A]("http://arstechnica.com/security/2013/04/admin-beware-attack-hitting-apache-websites-is-invisible-to-the-naked-eye/")
