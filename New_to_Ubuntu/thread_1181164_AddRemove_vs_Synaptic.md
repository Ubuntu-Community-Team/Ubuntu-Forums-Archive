### Post by Miljet on 2009-06-07
Add/Remove is much simpler to use, especially for noobs. The main difference is that Add/Remove only displays program (package) names. If you need to install specific library files, you have to use synaptic.

Actually, both are just GUI front-ends for the same dpkg installer.
