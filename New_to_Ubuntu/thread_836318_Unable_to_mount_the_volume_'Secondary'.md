### Post by hyper_ch on 2008-06-22
Assuming this (/dev/sdb2 25497 30739 42114397+ 83 Linux) is the partition that you want to mount do:

(1) create a mountpoint somewhere
```

sudo mkdir /media/sdb2

```
or on the desktop
```

mkdir ~/Desktop/sdb2

```
.... I will use /media/sdb2 for the rest

(2) open fstab
```

sudo nano /etc/fstab

```

(3) add this:
```

/dev/sdb2  /media/sdb2     ext3    defaults        0       2

```

(4) Repeat the same for sdb3

(5) reboot or remount everything
remount:
```

sudo mount -a

```
