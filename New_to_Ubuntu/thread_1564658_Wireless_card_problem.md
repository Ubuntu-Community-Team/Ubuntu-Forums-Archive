### Post by coffeecat on 2010-08-31
You have a Broadcom wireless card. There are licensing issues with the driver and/or firmware - I can't remember which - and it can't be included in the default install. Do what SpockVulcan suggests and you should be OK. If not, post back and we can take it from there.

**Edit:** to use the Hardware Drivers utility you need to be able to connect to your router with an ethernet cable. If you can't do that, post back.
