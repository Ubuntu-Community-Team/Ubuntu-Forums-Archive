### Post by fatality_uk on 2009-05-16
Hi elkiemo, welcome to Ubuntu Forums.

Firstly it has to be said that Ubuntu on the NC10 does have a few problems. There is a page dedicated to the NC10 [https://help.ubuntu.com/community/NC10](https://help.ubuntu.com/community/NC10) which has a lot of help if you do find any issues.

I have an NC10 and the main issue I have is with screen brightness when off mains charge. There are ways around this so don't worry.

Can I suggest that if you require Windows for updates only, then setup your NC10 with a dual boot using Wubi. You can run Wubi from the CD/USB stick to install Ubuntu. If you have already installed Ubuntu, then you can use something like [http://www.tatanka.com.br/ies4linux/page/Main_Page](http://www.tatanka.com.br/ies4linux/page/Main_Page) IE4Linux to install Internet Explorer within Ubuntu.

If you need any thing else, just shout.
