All three run the same basic version of Ubuntu. The difference between the three is the desktop environment, and then in turn some of the defaults apps they use. Default Ubuntu uses the GNOME desktop environment, Xubuntu runs a "lighter" desktop (called XFCE) and hence is better suited for older pc's.

You don't need a special firewall program. Linux already runs a type of firewall known as IP Tables. You can install a gui frontend for this called Firestarter if you wish. 

Some Windows apps (but by no means all) will run under a "compatability layer" known as Wine. You'll need to install this.
