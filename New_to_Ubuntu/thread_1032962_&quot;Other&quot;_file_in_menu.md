### Post by JoshuaRL on 2009-01-06
Yardsale?  Or maybe a charity would like the donation.

Seriously, look for Menu Editor (or maybe Alacarte) in Add/Remove Applications.  If you can't find it, you can install it from the terminal:
```
sudo apt-get install alacarte
```
Then you launch it either in the Applications menu, or so:
```

alacarte
```
Then, it's spring cleaning time.
