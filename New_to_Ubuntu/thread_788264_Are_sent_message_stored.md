### Post by ACJarrett on 2008-05-09
What program is your co worker using to send messages?  Some programs log any message sent out, others make it optional.

there are ways to log sent messages using the >> command in terminal.  This will take the output of the command before it and send it to the command after.  For instance if i typed

```
ls >> samplepage.txt
```

all the output from ls would be written down into samplepage.txt
