### Post by heyho on 2008-05-18
You will need to activate advanced desktop settings via system>preferences>appearance>visual effects and check the custom settings box.

Then open the compiz-fusion settings manager, and enable whatever you want.

### Post by roe_polak on 2008-05-18
If you have Compiz Fusion installed and working, the effect you are looking for is called Burn. It is actually the same effect as the flames but you need to check the option "Randomly Colored Fire" under the effect settings in ccsm (Compiz manager).

Hope this was of any help.

### Post by Tom--d on 2008-05-18
Thats easy :D

Install ccsm (Add/Remove or the repos) 

You must have compiz running. 
In terminal, type:
```
compiz --replace
```


Then in ccsm (Advanced Desktop Effects Settings - (System - Pref))

And it comes under animations :D 

You will see. There are many many lovely animations to fiddle with in there ;)
