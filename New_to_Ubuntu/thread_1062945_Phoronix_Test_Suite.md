### Post by jgoguen on 2009-02-07
This looks like a command-line application.  Open Terminal (Applications -> Accessories -> Terminal) and enter the command **phoronix-test-suite**.  If that doesn't give you what you expected, or to see more command options, go to [the Phoronix usage page](http://www.phoronix-test-suite.com/?k=usage) and see what they have to say.  Since you already have it installed, you can probably ignore everything before the first screenshot on that page.**
