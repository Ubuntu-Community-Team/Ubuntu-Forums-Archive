winff should be able to do that.

to install it type these commands.

to add the gpg key.

```

wget --quiet --output-document=- "http://winff.org/ubuntu/AAFE086A.gpg" | sudo apt-key add -

```

to add repository.

```

echo "deb http://winff.org/ubuntu jaunty universe" | sudo tee /etc/apt/sources.list.d/winff.list

```

to refresh the repositories.

```

sudo apt-get update

```

to install winff.

```

sudo apt-get install winff

```

### Post by Silent Warrior on 2009-10-12
(... Does anyone know why anyone would want to convert m4a to mp3, of all formats we can choose from? Anyway.)

To append them, I recently discovered a fairly nice way.
1. Import the files you want to bring together in Audacity.
2. Copy/Cut and Paste them in sequence at the end of the main file (press 'k' to move the cursor to the end)
3. Close all the instances of the files you've pasted into the bigger one.
4. Export as whatever you like. (The first prompt will be to set up the tagging, the next will deal with the actual format, filename, and save-location.)

I once tried mp3wrap/mp3split, but only once, and never really had the need until I found this way.
Audacity might be a bit big in resource-usage as compared to command-line tools, but it gets the job done neat as you like.

### Post by Jose Catre-Vandis on 2009-10-12
```
ffmpeg -i input.m4a -acodec mp3 -ac 2 -ab 128 output.mp3
```

and batch:
```
for i in *.m4a; do \
ffmpeg -i "$i" -acodec mp3 -ac 2 -ab 128 "${i%m4a}mp3"; \
done
```

If ffmpeg complains about bitrate, put a "k" after the number, e.g. from 128 to 128k
