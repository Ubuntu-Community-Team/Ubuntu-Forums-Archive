### Post by Troll_the_Great on 2009-12-13
> **rosie'sdad said:**
> Thanks quinnten83, would you mind elaborating on the live cd bit

A live CD lets you run an entire operating system from a CD, without the need to install it. Just download what version Ubuntu you want (the latest version is 9.10 Karmic Koala), burn the .ISO to a CD and boot from it. It will give you the option to try the operating system without installing it. Just keep in mind that the system will run slower compared to a normal installation and any changes will be gone upon reboot.
This feature allows you to see if your hardware is compatible with Ubuntu or not.
Cheers!

I have a friend who uses Ubuntu 8.04.3 on a vaio VGN-SZ series model. The only problem so far has been that there is a keyboard slider switch on this particular model (near to the screen) which gives a choice between a label of speed or stamina mode. This seems to switch between the nvidia (speed) graphics chip and an Intel graphics chip. The install went perfectly in speed setting, without realising anything. A month or so later, the switch was used in stamina mode and although there is some confusion in the story, the laptop did not subsequently work except in low res graphics mode with Ubuntu when the switch was set again to speed (nvidia, speed, as installed).

A boot up and choose Ubuntu rescue mode and fix X graphics cured this quickly, but it was an unforseen issue.

No other problems I know of, and the guy is very pleased with Ubuntu.
