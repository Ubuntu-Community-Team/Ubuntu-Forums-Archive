### Post by _DD_ on 2008-06-07
If you use OpenOffice (the one which comes with Ubuntu) then you must make sure you saved documents in .doc format. The default format it saves in cannot be opened by Microsoft Word.

If you want to then you can run Microsoft Word under Wine (a Windows API layer for Linux). I run Microsoft Office 2007 on Ubuntu and it loads faster than on Windows :)
