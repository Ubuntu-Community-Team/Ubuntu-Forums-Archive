### Post by mikechant on 2011-05-20
It's probably best to stick to the x86 (32 bit) install unless you have 4GB or more RAM. There are still a few issues sometimes with the 64 bit install (not nearly as many as there used to be though).

Also, you might do better using a Flash drive (USB stick) to install from rather than a CD, then you don't have to worry about burn speeds etc.
