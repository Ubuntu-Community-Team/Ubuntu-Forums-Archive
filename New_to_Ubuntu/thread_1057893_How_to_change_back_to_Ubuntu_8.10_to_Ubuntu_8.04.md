### Post by snowpine on 2009-02-02
1. Back up all important data
2. Install Ubuntu 8.04 over your current 8.10 install

Unfortunately there is no way to "downgrade" and keep your current install.

---

### Post by lazyart on 2009-02-02
Instead, put in a Live CD with 8.04 and see if they effects can be enabled there.  There shouldn't be a restricted driver for S3, so if it works, it should work out of the box.  If it does work for you, then go ahead and backup and install.

Just note that you'd have to stay in 8.04 indefinitely... just for the effects.

### Post by oldos2er on 2009-02-02
Can't say for certain, but it's likely your S3 card will use the same drivers in 8.04 as in 8.10. If you want to run compiz-fusion, you probably should buy a better video card.
