### Post by BDNiner on 2008-10-05
Correction it is in:
System -> Preferences -> Screens Action

To run it from the terminal just open a terminal from the accesories menu and type "brightside". alternately you can press ALT+F2 to bring up the run dialogue box and type "brightside" there.
