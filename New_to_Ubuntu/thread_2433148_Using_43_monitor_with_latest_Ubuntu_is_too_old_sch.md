### Post by oldfred on 2019-12-14
How much RAM? What video?

I have an old Toshiba with 1.5GB of RAM. Originally XP (supposedly would work with Vista). 
Ran Ubuntu but used fallback/gnome-panel which was like the old Ubuntu with top & bottom panels, not side panel. My main desktop is 4:3 and I use fallback on it as icons on side too too much valuable space. I retired it in 2014 when I go new desktop and it was running 12.04. Installed 16.04 just to see if it worked still and was a bit supprised that Unity had improved enough that it was almost usable. Still needed fallback to work well. Also learned to only load one larger app like Firefox and smaller app like terminal or it would go to swap and become so slow as to be unusable.
You could try Lubuntu or other lightweight version to see how it works.

       Light weight flavors
Lubuntu, xubuntu, Ubuntu MATE, Budgie
[https://wiki.ubuntu.com/UbuntuFlavors](https://wiki.ubuntu.com/UbuntuFlavors)

If memory really low, you may only have a system that runs well in terminal mode.
