### Post by Miljet on 2010-02-27
Although you cannot run the install wizard from the CD because it is a Windows .exe file, there should be a PDF file that you can copy to your hard drive that gives instructions for setting up the router. I think the file is called User's Guide.lnk.pdf. Just insert the CD and browse for the Guide.
