### Post by dirtlamb5 on 2010-02-02
I'm getting rather annoyed with the notification bubbles in Karmic. I just started using Gwibber, which usually gives me a lot of notifications. Including that and the notifications from Empathy, Evolution, and Rhythmbox, I'm getting a near-constant stream of notifications. However, the notifications can only be displayed one at a time, so it takes a long time to show all of them. In addition, since the notifications come from many different programs, it would be nice to have the ability to click the notifications and have the program that sent the notification open. Is there any way to have either of these things in Ubuntu Karmic? And will any of these things be implemented into Lucid?

### Post by nhasian on 2010-02-03
you can disable the empathy notifications in Empathy by going to Edit->Preferences from the menu bar, In the Preferences window select the Notifications tab.  Untick the option “Enable Bubble Notifications”

You are correct though, in the upcoming Ubuntu 10.04 Lucid Lynx there will be changes made to the pop up notification to address the exact issue you are describing.  there will be different urgency levels for the popup notifications.
