### Post by prshah on 2008-12-09
> **Jin127 said:**
>  but when i go to system>preferences>sessions, the window pops up, with only two tabs in it.  the startup programs tab looks fine, but the options tab only has a mark for "automatically remembering applications when logging out" and a button for "remember current running applications".

No there's no problem; from Intrepid (Hardy?) onwards, there are only two tabs.

Programs active in the current session have always been viewable from System Monitor (System-Administration-System Monitor-Processes), and also making this available in the "current" tab was duplication of effort, and thus (so I guess) removed.
