Xubuntu is now my choice of DE for any computer, old, (up to a point), or new.  It is what I am using on my desktop with i5-3570K, 8GB ram, and with the integrated Intel HD4000 graphic chip, and it flies.  Unity works very well on it but I still prefer the classic desktop style of xfce, so therefore Xubuntu is the #1 for me.

If you meant your question more about the choice of 32 vs 64 bit, either should run OK, but with 2GB ram it is really up to you; there are no problems of missing applications or drivers for most things any more that could once have dogged the 64 bit system, and while 64bit generally runs a bit faster than 32bit, it does also use more memory, so with 2Gb it is your choice.  If you can afford, and have the chance to add another 2GB ram chip, that would definitely push you to the 64bit system, but even with just 2Gb, 64bit would still be my choice.
