### Post by dmizer on 2009-03-02
Welcome to the forums.

The "Tutorials and Tips" section of the forum is for people to post tutorials and tips, not to request them. I've moved your post to the Absolute Beginners section of this forum.

There are a huge number of reasons why you should not install software directly from source (tar.gz or tar.bz2), not the least of which is that it's untrusted and unknown code. Sure, it's open source, but unless you take the time to review it yourself, you have no idea what's in there.

Here's a good link about how to install software, including how to install software from source: [https://help.ubuntu.com/community/InstallingSoftware](https://help.ubuntu.com/community/InstallingSoftware)
