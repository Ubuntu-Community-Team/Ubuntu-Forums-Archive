### Post by Frogs Hair on 2012-03-01
When the dash icon is selected at the very top you can search for applications by typing the first few letters of the application name. 

when you select the more apps the category filter will appear on the right side of dash . You can also add and remove applications to the panel so your most used application are easily found.

The application name is displayed when the cursor arrow is hovered over the icon . Unity 3D allows the the panel icon size to be changed so more applications can be easily seen without over lapping . 3D also has auto hide and transparency settings.

### Post by Double.J on 2012-03-01
Hi there suomalainen,

don't worry, you're not missing anything it does take a bit of getting used to! I do hope you stick with it though, as I have now come to like unity!

A couple of things I wish I'd known on day 1;

-Install CCSM from the software center, not just for effects like wobbly windows, but also to do things like resize the icons and set transparency

-if you botch compiz settings and are presented with a blank desktop with no icons, open a terminal and use```
unity --reset
``` to undo

-holding down the super key brings up numbers next to the icons in the launcher - just then press that number to launch the program; can be quite useful!

You can also now move the launcher panel to the bottom with tools from third party repos!

Hope it helps!

True.
What helped me most was working out how the search thing worked. I am currently trying to train myself to use the "super" key to open the dash. (The one that microsoft likes to call the "windows" key...)

To find an application, hit that, type in a couple of letters of the applications name (often also works with the english name although my desktop is set to German), then down arrow and left/ right arrow to get to the icon and then enter. Sounds clumsy when you describe it, but it is actually very quick.

Also, you might like to look at this
[https://launchpad.net/myunity](https://launchpad.net/myunity)

There is a PPA for it
[https://launchpad.net/~myunity/+archive/ppa]("https://launchpad.net/%7Emyunity/+archive/ppa")

I don't know, but expect that this tool will be developed with time to be able to change more settings. I have the strong impression that the emphasis is to make Unity work first, and worry about being able to change settings later.

Oh, yeah. When you are using the Compiz Settings mangaer, don't turn the cube rotating function on. I broke my unity doing that, and ended up doing a new install because I couldn't fix it to my satisfaction... ;)
