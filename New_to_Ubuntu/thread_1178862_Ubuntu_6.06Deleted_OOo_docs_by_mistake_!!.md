### Post by QIII on 2009-06-05
If you deleted the entire partition, TestDisk might help.

If just the files are gone, you might be able to recover them with PhotoRec.

The latter is really intended for image files, but I think it might be able to recover other file types.

Unfortunately, neither may work.  It's a crap shoot.

If neither of those work, recheck the thread frequently.  Someone may have some other ideas.


BTW -- unless you have LTS, Dapper is no longer supported.  Not even sure if the LTS is even in effect.
