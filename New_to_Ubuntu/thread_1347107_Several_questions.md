### Post by akelsall on 2009-12-05
As far as I know, Ubuntu will automatically detect your hardware every time you boot (unless the hardware is so new that drivers haven't been developed for it yet). 

Once you load Ubuntu onto a hard drive, you can install that drive into any machine and it will boot. There's no need to install Ubuntu each it. In fact, anything you have on that drive will be available on that new computer (just as if you had never moved it to another computer).

Andy

1. I'm assuming you're talking about non-plug&play hardware (such as PCI cards, video cards, etc).  Every time you boot Ubuntu will detect any newly installed hardware and should automatically get any drivers needed.

2. I've never done it so I could be wrong, but I don't believe you can install the Ubuntu OS to a hard drive, then remove that drive and put it in another machine and run it just fine.  When the OS installs it gets configured based on all the internal components on the machine (chipset, motherboard, etc) and you will likely crash if trying it somewhere else.

(I know for a fact that Windows installs will crash but maybe Ubuntu is different)
