### Post by limasierra on 2008-08-04
You can't use an alt-code to insert the square root symbol, but you can go to "Main Menu > Accessories > Character Map". Perform a search for the term "Square Root". Double click the symbol and select the copy button.

### Post by Bachstelze on 2008-08-04
In all GTK applications, you can use Ctrl+Shift+Uxxxx, where xxxx is the Unicode code of the character you want to insert (in hex, not in decimal like in Windows).

Therefore, the square root symbol is Ctrl+Shift+U221A : &#8730;
