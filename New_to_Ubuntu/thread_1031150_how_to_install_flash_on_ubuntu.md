### Post by Sef on 2009-01-05
Easiest way is with Add/Remove.   Applications > Add/Remove > Show: All Available Applications > Search: Adobe > click on box next to Macromedia Flash Plugin > click on apply changes > click apply > close.

### Post by 3rdalbum on 2009-01-05
What version of Flash do you have? Some versions work with Wine - check out [www.winehq.com](www.winehq.com) for a list of programs and versions that are compatible.

If all you want to do is create 2D animations, then there are plenty of Linux programs that can do that. Synfig is one of them. Using Flash for non-interactive animation is like buying the most expensive wine to mix with your orange juice.
