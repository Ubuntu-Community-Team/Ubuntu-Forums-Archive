### Post by TBABill on 2010-01-28
Nygel, I agree with the comment above. The .iso file you downloaded contains the entire Ubuntu system (meaning that it is compressed into one file). You will need to burn that file to a CD as an image, not data, and the program you use to do so will write the files correctly to the CD so it will be bootable. The .iso alone will do nothing on the CD. In addition to burning it, you first need to make sure the file downloaded without errors. There are lots of threads on burning an iso to a CD so a search of the forums would benefit you greatly if the link above does not provide the information you need.
