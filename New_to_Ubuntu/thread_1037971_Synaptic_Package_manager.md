### Post by jimmy the saint on 2009-01-12
Ubuntu does not support non-free items.  That means that proprietary software (like many drivers) are not supported and are called restricted.  Another package you will likely want to install is the ubuntu-restricted-extras package, which contains many things that are closed source, but desirable.  This includes flash (owned by adobe) java (owned by sun) and mp3 support.  If Ubuntu included them (and supported them) they would have to pay fees to the companies, so they must be installed by the user and Ubuntu has nothing to do with them.
