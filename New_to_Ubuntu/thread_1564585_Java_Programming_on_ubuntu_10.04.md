### Post by BlueCanary9999 on 2010-08-30
Go to Synaptic and download openjdk-6-jdk and openjdk-6-jre.  You can then use "javac FILENAME.java" in the terminal to compile and "java FILENAME" to run it.

You might also want an IDE, depending on your style.  Many use Eclipse and Netbeans.  I prefer a simple text editor like Emacs.  I think all of these can be downloaded from the Ubuntu Software Center.

Good luck!
