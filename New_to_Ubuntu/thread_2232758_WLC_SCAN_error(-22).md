### Post by Vladlenin5000 on 2014-07-07
> **siva7 said:**
> I reinstalled Ubuntu 12.04 LTS using a CD. Interestingly, the CD with a single iso file did not work, but the one in which I could the individual folders and files worked for installation.

Interestingly? No, no really. Things working as they should aren't that interesting. An ISO file (any ISO file) or better yet any "ISO image" was never intended to be burned as is. It's a "container" intended to be interpreted by the CD/DVD/BD burning software in order to produce an exact copy of the original disk the ISO was taken from.

[http://en.wikipedia.org/wiki/ISO_image](http://en.wikipedia.org/wiki/ISO_image)
