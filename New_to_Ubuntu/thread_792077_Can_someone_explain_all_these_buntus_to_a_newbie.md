### Post by Taras on 2008-05-12
The main difference between the distributions is that they use different desktop environment, but they all share the same security and packages.  
KDE desktop environment is a little harder to use and is slower than gnome, but provides a bit more of extra graphical user interface control. 

Xubuntu uses lightweight desktop environment, and therefore is intended to be used by older machines, if you use xubuntu on a new one it will run very fast, but only gtk application can be run on it, in another words you will not be able to play linux 3d games.

### Post by PeterJS on 2008-05-12
> **Taras said:**
> 
Xubuntu uses lightweight desktop environment, and therefore is intended to be used by older machines, if you use xubuntu on a new one it will run very fast, but only gtk application can be run on it, in another words you will not be able to play linux 3d games.

Gnome also uses GTK, there's no reason SDL or any other gaming libraries couldn't be installed in an XFCE desktop. You can also mix and match desktop environments. If you install XFCE because it's light weight, and then install heavier Gnome or KDE applications on top of that you're going to slow your system down, to what degree will depend on your hardware.

I ran a very mixed desktop for a long time, KDE applications with the XFCE desktop elements, with compiz for 3d effects. Also there is no "best" desktop, just the one that you like most and works well for you.
