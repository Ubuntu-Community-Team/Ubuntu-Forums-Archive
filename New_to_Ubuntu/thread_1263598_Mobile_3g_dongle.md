### Post by HereInOz on 2009-09-11
It should work out of the box.  Just plug it in and Ubuntu ought to recognise it and allow you to connect.  You need to be using the Gnome Network Manager to make it easy for you.  

I use a Virgin Mobile dongle in Australia and that needed a couple of configuration changes, but apart from that, it worked fine.

The first thing to do is to plug it in, and then see if it appears in the Network Manager, then see if it will connect and authenticate.  If it doesn't, search these forums and you may find your answer.
