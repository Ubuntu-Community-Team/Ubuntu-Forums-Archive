### Post by Joeb454 on 2008-06-11
If you don't get an error when running ```
compiz --replace
```Then it should be running correctly :)

Also, open a terminal and enter ```
sudo apt-get install compizconfig-settings-manager
```Then go to System > Preferences > Appearance and set "Desktop Effects" to normal.

This will make compiz start, if it doesn't return an error, then go to System > Preferences > Advanced Desktop Effects and tweak away at the compiz settings :D

### Post by Victormd on 2008-06-11
> **hazman said:**
> what is compiz and emerald ? and how do you use it downloaded them but can not find the to use

Compiz is a window manager that adds effects and transitions to your desktop and emerald is a "window decorator" that allows you to add some pretty nice themes to your desktop windows.

Emerald is dependent on Compiz so you need to be sure that Compiz is working before you can try emerald.

To use Compiz, you will need to have a 3D graphics card with the 3D capabilities enabled (most commonly done through the restricted drivers - SYSTEM>ADMINISTRATION>HARDWARE DRIVERS). What graphics card do you have? When you go to HARDWARE DRIVERS, your card should be listed, otherwise, chances are you don't have a 3D graphics card recognized in Ubuntu.

If you do have a 3D graphics card, then you can install compiz. Actually, compiz is installed by default, but you don't have direct access to it (through the GUI) until you install compizconfig-settings-manager (you can install this through Synaptic).
You will then be able to change the settings under SYSTEM>PREFERENCES>ADVANCED DESKTOP EFFECTS SETTINGS.

Once you get that going, then you can use Emerald...

### Post by Victormd on 2008-06-11
Just did a quick google search and it doesn't look like compiz will work with your card. There is a workaround though... but not sure how well it will work...

backup your xorg.conf file:
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
```

Then install xcompmgr
```
sudo apt-get install xcompmgr
```
from the terminal or search for it in synaptic.

Then edit your xorg.conf:
```
sudo gedit /etc/X11/xorg.conf
```
and add or replace the following line:
> 
Section "Extensions"
	Option 	"Composite" "Enable"
EndSection

Let me know if this works...
