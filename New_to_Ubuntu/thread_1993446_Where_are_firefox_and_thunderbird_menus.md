### Post by ajgreeny on 2012-06-02
This is the problem with global menus, in my opinion;  they are not totally consistent on all applications.

If you prefer the menus where they should be, ie, in the window titlebar, remove the firefox and thunderbird globalmenu packages.  If you want the same behaviour in all applications remove the appmenu packages as well, however many users eventually get used to global menus and you may also do so.
