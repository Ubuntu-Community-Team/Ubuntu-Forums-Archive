### Post by monkeybrain2012 on 2012-12-06
Install ubuntu-restricted-extra and add the medibuntu ppa and install w64 codecs (or w32 if you have 32 bit machine) and non-free-codecs

Instructions to add the medibuntu ppa.
[http://www.unixmen.com/how-to-add-medibuntu-repository-in-ubuntu-via-terminal-and-gui/](http://www.unixmen.com/how-to-add-medibuntu-repository-in-ubuntu-via-terminal-and-gui/)

BTW, there is no Ubuntu 12.1, you either have 12.04 or 12.10
