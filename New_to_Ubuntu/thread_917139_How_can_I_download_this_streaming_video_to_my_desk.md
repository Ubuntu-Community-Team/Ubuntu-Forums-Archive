### Post by Znupi on 2008-09-11
As Vivaldi Gloria said, I can barely play it, too. It usually just gives errors. But sometimes it starts. Here's how to do it:

1) Open VLC
2) File -> Open Network Stream
3) Select HTTP/HTTPS/FTP/MMS and in the text box near it input the URL
4) Check Stream / Save and click on Settings
5) Check Play Locally (if you want to also view the stream) and File. Click on Browse... to tell VLC where to save it. Specify a .wmv file.
6) Check both "Video Codec" and "Audio Codec"
7) Click Ok, twice ;).

It should start playing and saving to the file you specified now.
