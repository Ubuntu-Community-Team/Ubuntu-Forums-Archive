### Post by C.S.Cameron on 2008-10-20
Unetbootin is probably the fastest and most reliable method to install non-persistent Ubuntu 8.04.1 to flash drive from windows.
Once downloaded to windows, (or ubuntu), it will build a USB flash drive from the Live CD or from the downloaded Ubuntu iso file.
Liveusb is also good but takes a couple hours to run.
Usb-creator is fast and simple if using 8.10.

### Post by mgmiller on 2008-10-20
> installed unetbootin in windows distribution= ubuntu 8.04 live---disk image= iso, what do i put in the following line? I will be using usb drive.

Those are 2 different areas of the program.  Specifying on the distribution line will download and install directly off the web.  The image=iso line wants you to browse to an iso that you have already downloaded.  Use one method or the other.
