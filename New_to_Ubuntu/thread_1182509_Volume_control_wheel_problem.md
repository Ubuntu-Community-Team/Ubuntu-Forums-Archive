### Post by wubsieonline on 2009-07-24
Hey everyone!! I had the same problem. When i scrolled the volume wheel the slider would go completely up, or completely down. I also couldn't write anything, and no menu-items would pop up when i clicked on them.

I was able to reverse this to first use CTRL + ALT + BACKSPACE and after that use CTRL + ALT + F8 and then back to CTRL + ALT + F7.

Still that is pretty annoying when you are used to scrolling the wheel every time you want the volume to change.

Thanks to your posts i was able to make a temporary solution. In the *Keyboard Shortcuts* screen i changed the key combinations to a different set of keys. And now the scroll wheel doesnt work anymore, but the combination i use does.

I will use this solution, until a fix is released in another update.

Thx guys!
