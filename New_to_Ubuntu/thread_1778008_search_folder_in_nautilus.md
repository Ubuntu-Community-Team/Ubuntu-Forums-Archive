### Post by Mr. Shannon on 2011-06-08
In Nautilus go to **Edit->Prefrences** and then choose the **List Columns** tab and then check **Location**.  Now close the settings window and change the view to **List View**.  Now you can see the location of files and folders.  To see which are in the top folder click the heading labeled **Location** this will sort the search results by location and you will end up with all of the files and folders in the top folder listed first.
