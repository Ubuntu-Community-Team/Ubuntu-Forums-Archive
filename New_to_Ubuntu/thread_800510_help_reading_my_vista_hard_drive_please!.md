### Post by wadelewis4 on 2008-05-20
Did you shut windows vista down properly? If not the drive can't be mounted until windows does shut down properly and "lets go" of the drive. If you're unsure, just boot into windows and restart from the start > turn off computer > restart. Now boot into Ubuntu. Then just double-click on that hard drive in Places >"Computer". 

Just a suggestion.
