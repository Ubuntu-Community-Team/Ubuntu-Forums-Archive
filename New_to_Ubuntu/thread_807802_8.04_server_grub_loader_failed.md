### Post by Xiong Chiamiov on 2008-05-26
GRUB is what tells your computer where to look for the OS.  If you didn't install it, you can do so with [SuperGrubDisk]("http://supergrubdisk.org").

EDIT: Apparently you're fine booting up.  The Server edition doesn't come with a GUI, so don't be expecting one.  If you ever need multiple operating systems, you'll want GRUB.
