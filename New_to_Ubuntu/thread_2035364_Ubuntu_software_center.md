### Post by richpri on 2012-07-30
The Ubuntu software center is a new offering from the same people who distribute Ubuntu [Canonical]. I hear that it is intended to replace the Synaptic Package Manager which is still available in 12.04.  And I believe it works much the same [new front end for apt?].  

The way Synaptic works is that you can get software from a number of different sources that are listed in in a "Repositories" file.  As long as you stick with the default repositories you should be safe enough.
