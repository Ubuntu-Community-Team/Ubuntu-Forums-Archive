### Post by Frogs Hair on 2012-12-31
Some of the Compiz Extra and Experimental Plugins are no longer maintained due to developer workload but I can't give an exact list. The compiz-plugins-extra package is a dummy package in 12.10 according to the description in synaptic.

12.10 uses only Gnome 3.6 GTK themes and there is persistent  fix for custom cursors but you have to search the forum . Install the Gnome Tweak Tool also if you haven't already.

### Post by grahammechanical on 2012-12-31
CompizConfig Settings Manager (CCSM) is still in the Ubuntu Software Centre. As already noted, some of the plugins have been removed in 12.10.

There are some Unity settings in the Ubuntu Unity plugin. You can make the top panel and the launcher transparent and reduce the size of the Launcher icons (also in System Settings>Appearance). There is other stuff that I have not messed with. You click on the plugin label to get access to the settings.
