### Post by grahammechanical on 2016-02-26
There is no trade-off between security and performance.

The Linux part of the distribution is the same whether we are running a distribution with a user interface that requires hardware accelerated 3D rendering (Ubuntu or Kubuntu or Ubuntu Gnome) or a distribution that can run very well on a machine with a lower hardware specification (Xubuntu or Lubuntu or Ubuntu Mate).
