### Post by deadflowr on 2014-08-22
try
```
sudo apt-get update && sudo apt-get upgrade
```
post the output.

The update command updates the package lists available to you. Typically when a package no longer exists it is because a newer version has replaced it in the archives and as such the old one is gone.

The upgrade command will upgrade the existing packages on your system to whatever is currently the most up-to-date in the repos.

Do not worry, the upgrade command will only update the packages per the version of Ubuntu you are on, it doe not upgrade the version of Ubuntu or anything like that. (example, the command will not make change ubuntu from 14.04 to 14.10, or something like that)
