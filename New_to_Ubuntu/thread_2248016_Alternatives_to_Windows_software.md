### Post by Impavidus on 2014-10-12
Concerning browsers: Firefox is installed by default. Several other browsers are available from the software centre. Just have a look there. If you want Google Chrome, you need to download the .deb package from Google directly. For license reasons it cannot be included in the software centre.

There are various graphics programs available from the software centre. The most advanced is gimp, which is more or less as advanced as Photoshop. Most people, including me, never use it to its full potential.

I don't really know about the other software you mentioned. On the few occasions I uploaded using ftp, I used command line tools. On the rare occasions I write html, I write source code in an ordinary text editor. But I'm sure more advanced tools exist.

It's best to stick to the official repositories (i.e., those programs available from the software centre) as much as possible until you know your system. People tend to break their systems if they install software from others sources without knowing what they do.

### Post by whitesmith on 2014-10-12
My 2¢: Resist the temptation to be drawn into HTML editors like Dreamweaver or the one previously recommended. Dreamweaver, for example, works on Linux only under Wine--how well I cannot say. But that isn't the real point. Because it allows you to build a webpage by entering data or graphics into dialogs, drag-and-drop borders, etc. you have only an illusion of control over your final work. If something goes wrong--as it will, believe me!--and the required change can't be made from the GUI, you're in a real pickle. The alternative is to learn HTML, CSS and JScript. Working knowledge of those tools (by which I mean grasping 20% of their entirety) allows construction of websites without worry. I gave up on man-behind-the-curtain thingies like FrontPage years before they began to fall fall of support and never looked back.

### Post by nerdtron on 2014-10-12
Don't forget [Kompozer ]("https://help.ubuntu.com/community/InstallKompozer")for Linux. Although no longer on the repository and not very much updated, still an HTML Editor.

Basically, if you want to switch completely to Linux
1. List down all your software that you are using now on windows
2. See if those program have Linux versions, like Firefox, Filezilla, OpenOffice, Chrome, Dropbox, Skype, all have Linux version.
3. If there are no linux version, try alternatives programs that have Linux versions.
4. Continue using these open source programs/alternatives on Windows.
5. When you are comfortable on your new software list, moving to Linux will be a breeze since you are all aware of the programs and just need to adjust to the OS.
