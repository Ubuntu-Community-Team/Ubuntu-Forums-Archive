### Post by donkyhotay on 2009-02-05
[http://appdb.winehq.org/objectManager.php?sClass=application&iId=17](http://appdb.winehq.org/objectManager.php?sClass=application&iId=17)

Always check appdb.winehq.org for wine compatibility questions. My experience is that cs3 & cs4 work pretty good but do require some tweaking first (never used cs2). There is a bug that occurs with cs4 with menu's not vanishing properly that has been fixed but (as of right now) you have to patch and then compile wine yourself in order to use it. Information on this bug can be found here:
[http://bugs.winehq.org/show_bug.cgi?id=9615](http://bugs.winehq.org/show_bug.cgi?id=9615)

The various distro's are a matter of personal preference. Ubuntu is the dominant distro using debian packages while Red Hat is the dominant distro using YUM packages. If you are thinking about migrating from windows (which it sounds like based off your post) I would strongly recommend installing ubuntu with wubi first so you can safely try it out and make certain it works the way you need to before committing to it. Linux is *not* windows, all of the advantages you often hear about linux being more stable and secure mean that it has a completely different way of doing things (especially with program installation) in order to remain stable and secure.

### Post by squeabs on 2009-02-05
I've tried Photoshop in Wine, but it was slow and bogged down. I wouldn't recommend it, but you may be able to get it to work.
The best thing to do when switching to Linux is to look for the alternatives available to you. Linux isn't Windows, so many Windows apps won't run quite the same, even when emulated. Even then, you're defeating the purpose of switching to Linux.
As for Photoshop, [Pixel 32]("http://www.kanzelsberger.com/pixel/?page_id=12") is probably your best bet. It's not free, and I've never tried it. It may get you away from trying to run Photoshop in Ubuntu.
