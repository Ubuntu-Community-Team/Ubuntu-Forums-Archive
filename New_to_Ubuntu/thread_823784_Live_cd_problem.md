### Post by Dazed_75 on 2008-06-09
It sounds like there is an issue with the graphics mode.  There is another option in that menu called "Safe Mode" or something like that.  Try that option first.  If that does not work, repost with some information about your system.  Be sure to list your graphics card, system memory as well as your processor and its speed.

BTW, 7.04 (2007 April) is 2 releases back.  If you have enough memory, you might have better luck with 8.04.  However, it requires more memory than 7.04 so if you are low (e.g. only 256 MB memory) don't try 8.04 quite yet.
