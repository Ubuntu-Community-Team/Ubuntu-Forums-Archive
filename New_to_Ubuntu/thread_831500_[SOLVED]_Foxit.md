### Post by rraj.be on 2008-06-16
Evince is a good choice but you can use some more

like pdfcube which shows even 3D in PDf.

```
sudo apt-get install pdfcube
```

Further you can also use ghostscript viewer

```
sudo apt-get install ghostsciprt
```
