### Post by Psumi on 2010-03-30
I wanted to post this how-to right away, because I hate waiting for moderation to approve of the topics.

Since a lot of people are probably going to complain when yahoo takes over ubuntu's firefox. You cannot change the address bar search engine from the Quick Search NOR the manage search engines window. This will show you how to change the search engine for the address bar. This will show you how to change the address bar from the current google search to the yahoo search.

1. In the address bar type **about:config** and press enter.

2. In the filter box type **keyword.URL** and wait for the results.
2b. If keyword.URL does not exist, create a new string with that name.

3. change the value to **[http://search.yahoo.com/search?p=](http://search.yahoo.com/search?p=)** (For google English, it's **[http://www.google.com/#hl=en&q=](http://www.google.com/#hl=en&q=)**)

Now you can search yahoo via the address bar.
