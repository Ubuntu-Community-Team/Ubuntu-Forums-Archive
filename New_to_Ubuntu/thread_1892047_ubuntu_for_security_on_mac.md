In short **NO**.  **Ubuntu/Linux is not more secure than any other operating system**.  The advantage you would have with Linux/Ubuntu is the lack of virus infection risk due to no viruses being in the wild currently that effect linux.

It is as easy to compromise a Linux system remtotely as it is another OS, depending on patches, updates, firewall rules etc etc.

It is upto you to make your system secure within the realms of functionality and ease of use

However Viruses are an extremely small percentage of possible attack vectors, payloads are often a nuisance and not meant for compromise.

As far as online fraud goes it is more about the browser for the most part.

Make sure you run NOScript if you are using Firefox or NotScript if using Chrome/Chromium.

Have tight firewall rules both inbound and outbound and restrict applications with Apparmor.

See here for more information on those things [https://wiki.ubuntu.com/BasicSecurity](https://wiki.ubuntu.com/BasicSecurity)

Where we have started to put together a wiki covering basic security concepts.

Enjoy
