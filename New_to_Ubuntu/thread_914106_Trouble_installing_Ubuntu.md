### Post by OldTimeTech on 2008-09-08
It could be the cd you are using that is causing the setup problems.  The iso download needs to be burned at the very slowest your cd burner can burn at.

And if the desktop cd doesn't work, then you might need to you the alternate cd, which is text based instead of gui based.

Also, Windows (whatever flavor) needs to be loaded first, defragged before starting the setup for Ubuntu.

Just a few possibilities.
