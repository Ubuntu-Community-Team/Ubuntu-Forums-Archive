### Post by UbuNoob001 on 2010-10-26
Hey don't be frustrated! or at least know that LOTS of people, newb and non-newb alike, get frustrated with computers and all OSs. Luckily the Linux community makes learning and getting "un-frustrated" lots of fun.  As per the suggestions above KISS and be methodical:

1. Decide what program you want. 
2. Check Applications>>Software Center (if its there this is very easy click 'install'...done)
3. Read about Synaptic if you'd like a bit more control [HTML]https://help.ubuntu.com/community/SynapticHowto[/HTML]4. If you still have no luck, go to the website of the program you wish to download. See if they have a .deb package which is super easy to install (download the file, then simply "open with .deb package manager" or something like that)
5. also, if you know something is in the repositories [HTML]https://help.ubuntu.com/community/Repositories/Ubuntu[/HTML] then you can always open terminal (Applications>>Accessories>>Terminal) then type ```
sudo apt-get install <program name>
``` which can be fun if you want to try the command line interface for something simple. 

also, once you have something specific in mind, if you cant find a solution, just feel free to post back here and someone can help you with your specific need. 

Welcome!
