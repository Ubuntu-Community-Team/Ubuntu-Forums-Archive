### Post by Patb on 2008-05-18
Black, you will need to install Secondlife from a terminal.  Ubuntu (and other debian based distributions) uses packages in *.deb files which can generally be installed using Synaptic (or apt-get in a terminal).  Programs in *tar.gz or *tar.bz files cannot be installed using Synaptic. They have to be installed differently.  

See instructions in this link and in the link provided by Herman in this thread.  
[http://ubuntuforums.org/showthread.php?t=81800](http://ubuntuforums.org/showthread.php?t=81800)
