### Post by dniMretsaM on 2011-04-05
Ok, I'm probably missing something totally basic here, but how can I delete a shortcut to Trash from my desktop? I've tried manually moving it to the trash, but that doesn't work, and it isn't in the Desktop folder either. So, any help?

---

### Post by TeoBigusGeekus on 2011-04-05
Applications>System Tools>Gconf Editor
Navigate to Apps>Nautilus>Desktop
Uncheck the Show Trash on Desktop (or something like that) button.
