### Post by majiciannz on 2009-09-15
If I understand you correctly you want to download and save video from a web site.
For this I use ant.com toolbar.

In firefox open tools>Add-ons>Get Add-ons and in the search box type ant.com.
When it's found, install it.

The tool bar has a download button so if you are watching a utube video and want to save it you hit the download button.

Hope this helps
