### Post by appi2012 on 2008-12-27
You could try out FreeCiv, a open source game developed to be an alternative to the commercial Civilization. It seems pretty similar to Age of Empires, but I haven't actually played it. You can install it like most other ubuntu software, by going to Applications -> Add/Remove, and then searching for FreeCiv. 

Their website is:
[http://freeciv.wikia.com/wiki/Main_Page](http://freeciv.wikia.com/wiki/Main_Page)
