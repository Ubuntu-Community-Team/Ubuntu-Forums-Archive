### Post by Cheesemill on 2013-03-07
You're having to download so much because there are a years worth of updates that have been released since 12.04.

If you're going to be installing on more than one machine then it would be worth your while downloading the 12.04.2 iso, as then you will only have 3 weeks worth of upgrades to do.
However the initial download of the new disk would be around 700MB so this would only be worth your while if you were going to be doing multiple installations.

Also the only difference between the CD and the DVD is that more languages get installed with the DVD, if you only need to install in English then there is absolutely no reason to be using the DVD in the first place.

Edit - I'm not sure why you need a gig of updates. I've just tested with a VM by installing 12.04 from the original disc, straight after installation I only need 316MB of updates to bring the system up to date. It must be all of the extra language packs that need updating because you installed with the DVD instead of the CD.
