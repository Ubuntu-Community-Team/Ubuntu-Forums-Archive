### Post by Jerry N on 2009-12-05
I just got through using Clonezilla to clone a Win XP installation on a 160gb drive to a 500 gb drive.  Clonezilla created a 160gb partition on the 500 gb drive and left the rest as unallocated space (anything on the drive, including formatting, is wiped out).  At that point, gparted can change the size of the created partition and format the unallocated space as desired.  The following site was very helpful:  [http://www.mindworkshop.info/windows/windows-adjustments-back-up-your-windows-installation/](http://www.mindworkshop.info/windows/windows-adjustments-back-up-your-windows-installation/)
I think cloning Linux works exactly the same way.
