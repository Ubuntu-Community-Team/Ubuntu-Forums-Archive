You can load a thumb drive with Lucid with the Ubuntu disc loader if it's on the computer in Hardy or unetbootin. Unetbootin may be in synaptic if not download it from here.
[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)
Load the thumb then try it out live and make sure it works for you, if the Ubuntu thumb loader is used it has a persistence that will allow you to save stuff so you can run it until your sure it works and your ready to install. Unetbootin does not save anything.

Once you decide to install just hit the install icon on the desktop and choose install to the whole HD. You could if you wanted to remove the hardy os with gparted on the thumb or a live cd works as well. I have suggested the thumb as this computer seems to have no cd reader so all the instruction about the thumb are the same for a live cd.

So I have assumed that Hardy is the only OS on the computer as you have described it being a dell-centric setup. If you have any other OS post that information.
