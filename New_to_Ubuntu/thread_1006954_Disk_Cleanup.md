How can I perform maintenance on my system, using ubuntu 8.10 such as removing install packages like one would do in windows with Disk Cleanup?

---

### Post by Tatty on 2008-12-09
```
sudo apt-get autoremove
``` will get rid of all installed packages which you no longer need.

Everything else pretty much takes care of itself.
