### Post by Rex Bouwense on 2011-09-25
Welcome multiwings.  There are several things that could be causing that.  First, you may have a bad download.  Check the downloaded ISO.  In other words run an md5sum.  See here for instructions if you need them.
[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)
Second, you might have a bad burn.  If the md5sum checks out OK, then reburn the ISO at the slowest possible speed.  That should fix you up.
