### Post by germanix on 2008-10-06
I have purchased a 4GiB USB stick on which I wish to back-up my home files. Checking with GParted I saw that it was currently formatted as Fat32. Is it advisable and would it make more sense to re-format it as ext3?
Ubuntu is my sole OS.

---

### Post by leg on 2008-10-06
I would say that there is no need to do this. Especially if you ever intend to use the files on a Windows computer.

---

### Post by alynx on 2008-10-06
Im using my USB stick with fat32 on a regular basis and that works out well , so I would also say no you don't have to format it.

---

### Post by halitech on 2008-10-06
short answer - no. Long answer - maybe.

if there is ever the remote chance that you may need to get a file off the drive on a windows or mac computer, leave it as fat32 or format it as NTFS (works fine as you are aware). If you don't think there will ever be a chance, then format it ext3
