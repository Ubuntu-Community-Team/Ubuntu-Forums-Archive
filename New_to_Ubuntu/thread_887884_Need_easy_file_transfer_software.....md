### Post by nicedude on 2008-08-12
If you want a GUI based one program solution then you could check out Filezilla ( versions for Linux & Windows ).


Heres the info from synaptic package manager on its wonderous features.

FileZilla includes the following features:
  * Supports FTP, FTP over SSL/TLS (FTPS) and SSH File Transfer Protocol (SFTP)
  * Available in many languages
    (translations included in the filezilla-locales package)
  * Supports resume and transfer of large files >4GB
  * Easy to use Site Manager and transfer queue
  * Drag & drop support
  * Speed limits
  * Filename filters
  * Network configuration wizard
  * Remote file editing
  * Keep-alive
  * FTP-Proxy support
