### Post by newb85 on 2012-09-22
I'm not sure I understand your question, but maybe this will help.

If you're using Ubuntu 12.04, there should be a networking menu in your system tray (the top right corner of the screen).  There are a lot of options there, and two are "Enable Networking" and "Enable Wireless", which act as toggle switches.  If you un-check the first, all networking (ethernet and wireless) will be disabled.  If you un-check only the second, only wireless will be disabled.

Note: Sometimes machines have hardware switches for disabling one or both.  If networking or wireless are disabled, there will be a line at the top of that menu to tell you so.
