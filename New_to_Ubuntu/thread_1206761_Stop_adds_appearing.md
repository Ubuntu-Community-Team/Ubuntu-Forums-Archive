### Post by CatKiller on 2009-07-07
My favoured solution is to use a custom hosts file to redirect the requests to a whole bunch of ad servers to the localhost instead. A search for "ad block hosts file" should be sufficient to get you the instructions you need.

Alternatively, you can use AdBlock Plus as has been suggested. It's quite good and is in the Universe repository. You can install it in Synaptic.
