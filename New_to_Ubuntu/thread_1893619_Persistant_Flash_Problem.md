### Post by coldraven on 2011-12-10
1) Run Firefox
2) Go here and install Flash-Aid 
    [https://addons.mozilla.org/en-US/firefox/addon/flash-aid/](https://addons.mozilla.org/en-US/firefox/addon/flash-aid/)
3) You should now have a little red icon with an "f" inside it on the left of the address bar.
4) Click it and run Flash-Aid
5) Go with the default choice (64 bit beta)
5) Enter your password when the terminal screen asks for it..
6) Relax :)
