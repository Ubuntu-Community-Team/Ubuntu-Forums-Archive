### Post by Sarai the Geek on 2009-08-10
Click on the applications menu, then accessories, then "terminal"
Type "sudo apt-get install ubuntu-restricted-extras" push enter, key in your password and push enter again. It will download and install the software automatically. :)

That package (piece of software) contains the flash plugin along with mp3 support and a variety of other goodies you will want.
