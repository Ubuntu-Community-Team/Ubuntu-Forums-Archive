### Post by undecim on 2009-11-22
Putting the .iso file on the disc and writing it to the disk as an image are two different things. You probably just burned the file to the disk and if you open it from windows, you will see the .iso file there.

In order to write the file as an image, you need a program like [InfraRecorder]("http://infrarecorder.org/").

To make sure that you burned the CD correctly, open it from "My Computer" and make sure you see several files, instead of a single .iso file
