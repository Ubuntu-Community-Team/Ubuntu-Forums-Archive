### Post by Petro Dawg on 2012-08-15
Personally I like Code::Blocks as a C++ IDE.  I've made some simple programs with rudimentary graphics and sounds using it.  There are tutorials on-line that teach you how to make executable programs using codeblocks.  

I do most of my useful programming using VBA in Excel as most of what I do is solve engineering problems (it's just about the only thing that keeps me from getting rid of windows altogether).   I believe Libre Office Calc has  ability to program in Basic to write programs/macros with in it, but I haven't done anything but briefly look at it, so I'm not sure how easy it is to work with.  

Honestly, if I were to learn a new language I would start with Java (using netbeans) to write software that will work across all platforms.  Writing android apps would be kinda of interesting and rather useful. 

Good luck and have fun.
