### Post by Impavidus on 2013-10-30
It looks like an acute accent, which looks similar to the apostrophe. Maybe the key has been defined as a [dead key](http://en.wikipedia.org/wiki/Dead_key). If you hit it once followed by a letter that can have an acute accent it will produce an accented letter, if you press it twice you may get the bare accent. In some cases the same key may also produce an apostrophe.
