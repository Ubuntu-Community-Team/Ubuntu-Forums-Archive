### Post by canadastitan on 2008-05-06
Hi everyone..When i switched from windows xp to ubuntu on my acer travelmate 2303LCi laptop,my wireless card would detect but not work..
What my wireless lan that ubuntu detected is..:
Broadcom Corporation BCM4401 100Base-T (rev 01)
Linksys, A Division of Cisco Systems [AirConn] INPROCOMM IPN 2220 Wireless LAN Adapter (rev 01)
To install the correct driver for it i open'd terminal and installed ndiswrapper utils 1.9 , Which created a windows wireless driver utility in system administration as example in this link..

http://lxer.com/module/newswire/view/46385/

..I previously went to Acer global and got the windows xp drivers for the wlan which is called 

Foxconn_ID_Acer_Dri2_22_UI_3_20_50110_WHQL_Pass_Install UI in WinXP.zip 

..I extracted the files to my desktop then used the Windows Wireless Drivers utility that is now in my system administration folder to select the ""neti220.inf"" file from the directory it was created in...
And now my wireless card is working fine so far..
..Hope this helps others with wireless card driver problems..
