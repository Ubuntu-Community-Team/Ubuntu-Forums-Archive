### Post by urukrama on 2008-06-07
Yes that is Fluxbox. It is in the repositories. You can install it with:> sudo aptitude install fluxbox

Log out, and in the login window choose Fluxbox in the session menu (F10 > Sessions > Fluxbox).

See [this thread]("http://ubuntuforums.org/showthread.php?t=371144") on these forums for more help.

I'm not sure what the style is that is used in the screenshot. The fonts are an artwiz font (snap?).
