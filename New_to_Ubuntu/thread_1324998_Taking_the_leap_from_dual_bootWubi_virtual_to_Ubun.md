### Post by undecim on 2009-11-13
I can't help you migrate wubi (never used it before), but I would like to mention something about running windows software on Ubuntu.

For running windows programs on Ubuntu, you have 3 options:

1: Wine lets some windows programs run, but not all of them work perfectly.  Look at appdb.wine.org to see if you application works.

2: Crossover is the commercial version of wine. It costs just under $40, but works better than wine.[http://www.codeweavers.com/store/](http://www.codeweavers.com/store/)

3: The virtual desktop you're talking about is a program called VirtualBox. It runs a virtual machine, and can run any software that the guest OS can. The drawback to this is that you need to have a license and install disk for the OS you want. (maybe you can set this up when those XP discs arrive?)
