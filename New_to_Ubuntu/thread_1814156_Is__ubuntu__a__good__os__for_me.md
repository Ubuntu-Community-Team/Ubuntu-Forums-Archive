### Post by Ms_Angel_D on 2011-07-29
Hi Krishna,

first You'll notice I edited your title and moved your thread..this will hopefully get you some better responses since the cafe is generally for off topic discussions. 

Secondly when you perform an upgrade all you are doing is upgrading the system, your applications will remain in place so they were not re-installed as they were actually never removed.

To get a fresh start you'll want to wipe your hard drive and then install the operating system again. Doing so will remove all your installed applications along with all your personal files so **MAKE SURE YOU BACK THEM UP FIRST**.

### Post by sadaruwan12 on 2011-07-29
Hi and welcome,

Ok, you've upgraded from maverick to natty online but what we recommend a use or newbie to do if he/she want to install a new version of ubuntu don't do a online upgrade unless he/she have a very good internet connection other wise things tends to get corrupted.

So for you friend the best thing is to do download new ISO or CD image from the ubuntu website and burn it to a CD and do a full system installation.

But first backup your personal stuff in the /home folder also when you're redoing your partition table remember to create a separate /home partition that way you can keep your data safe even if you have re-install the operating system.

Hope this helps and yes ubuntu is a good OS but if you breaks it'll break so please know what you're doing and then do it.

But don't worry if some thing goes wrong 'cos we're here to help you from what we know.
