Normally, when media is inserted, the OS looks for recognizable files (pictures, movies, music), that's why it spins the CD and reads its TOC.

The backup partition is normally just so big to get the OS OEM CD/DVD, drivers and software you get with the computer on. Simply said, it is already full, mostly hidden and represents some 5% or less of the disk.

You can install Ubuntu on the first partition alongside Windows or just try it out.
