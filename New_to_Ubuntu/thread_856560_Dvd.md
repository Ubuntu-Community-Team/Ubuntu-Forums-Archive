### Post by eragon100 on 2008-07-11
Go to add/remove and check if you have the program "ubuntu-restricted-extras" installed (you have to select "all available applications" first at the "show" box on the upper side of the screen).

If you do, also install "vlc media player".

after installation insert dvd you want to watch.

Start VLC media player (applications -> sound and video -> VLC media player)

Click on "File", then "Open disc", and in the menu that appears, don't change anything, simply click ok! :wink:
