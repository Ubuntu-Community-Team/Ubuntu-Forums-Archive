### Post by Irihapeti on 2009-04-15
System -> Administration -> System Monitor. Click on the Resources tab and it's there in graphical form, along with some other interesting stuff.

That's for Ubuntu - no doubt Kubuntu and Xubuntu have their equivalent utilities.
