Stability, as far as I'm concerned is a measure of whether the system can be relied upon to work the same way down the road, implementing new features without breaking old ones.

If old kernel modules aren't guaranteed to work with other releases within it's version (i.e. modules for 1.0 not working with 1.3 or some such), that's sloppy and unstable. All the other big POSIX systems maintain this stability, and so does Windows.

The only reason I use Linux instead of BSD or Solaris for that reason is because Linux supports my laptop hardware better at the moment, and being that it's more popular, Linux has more software available.
