### Post by caljohnsmith on 2009-01-22
It sounds like you just need to add your Ubuntu's Grub menu as an option in your Fedora's Grub menu. In your Fedora /boot/grub/grub.conf file (I think Fedora uses grub.conf rather than menu.lst), try adding an entry like:
```
title Ubuntu Grub Menu
configfile (hdX,Y)/boot/grub/menu.lst
```
And replace (hdX,Y) with the Ubuntu partition. How about giving that a shot; if it doesn't work for some reason, let me know what problems you run into and we can work from there if you want.

```

```
That will create a "RESULTS.txt" file in the same directory from where the script is run, namely your desktop; please copy/paste the contents of that file to your next post, highlight the copied text, and click the pound sign # graphic in the Ubuntu forum message box so that the text will get "code" tags put around it. ### Post by Ben Page on 2009-01-22
Fedora 10 should automatically recognize any other system in installation procedure, you can there set up the entries in grub, but by default it will enter all available OSes in the grub.
