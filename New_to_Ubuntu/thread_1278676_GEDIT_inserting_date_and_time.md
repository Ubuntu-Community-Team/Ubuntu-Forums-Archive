### Post by WorldTripping on 2009-09-29
Make sure that you have the plugin enabled.

In gedit it's:

'Edit | Preferences | Plugins | Insert Date / Time'

Then in your document choose:

'Edit | Insert Date and Time'

You'll get a raft of options / styles or you can use a custom format.

(Actually you can configure the plugin to use a preset style if you prefer.)

### Post by WorldTripping on 2009-09-29
eg For Custom Formatting:

%a = 3 letter day eg 'Wed'
%h = 3 letter month eg 'Sep'
%c = Text Day / Month / Year / Time String eg 'Wed 30 Sep 2009 01:45:46 BST'

There are probably others too.
