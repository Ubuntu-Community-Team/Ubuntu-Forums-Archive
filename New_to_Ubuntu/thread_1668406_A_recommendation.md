This is I think an example of so called 'eye candy', and nothing wrong with that at all if you like it. Desktop effects in Ubuntu (I think 'compiz') can certainly offer some amazing stuff (note), and did you know that you can install kubuntu-desktop also into your existing Ubuntu and choose between them each time when you see the log in screen (choose which session). I believe that Kubuntu is very much more into impressive eye candy.
Note that after both ubuntu and kubuntu sessions are installed, your menus wil include a merged set of items covering both. It is also not as easy to remove kubuntu as it was to install the package, but it can be done. A Kubuntu live CD is a way to check it out easily of course.

(My own preference is for a lack of active eye candy and graphical responses, I 'see' a drain on PC resources and a more sluggish PC).

note:
system>preferences>appearance>visual effects
and if that is enabled then using Ubuntu software centre, install the package
OpenGL Window and compositing manager (Compiz) and also  Advanced Desktop Settings (ccsm)
then see your menu
system>preferences>compiz settings manager
