### Post by germanix on 2010-05-28
1. Ive a pretty good CPU. Quad Q8200 , 2GB DDR2 RAM and a NVIDIA 9400 GT. Can you please sugesst me the cleanest, coolest and the slickest version of Ubuntu ?
Ubuntu 10.04 is the latest. What is cool and slick depends on your personal tastes. I like this latest version.

2. What is the main difference between the different versions of Ubuntu ?
Some versions are LTS meaning Long Term Support. These versions are supported for 3 years. The other versions are supported for 18 months. Some versions like Ubuntu uses Gnome as the desktop manager others like Kubuntu uses KDE. Again depends on personal taste. The Ubuntu with Gnome are more popular but KDE also have many fans.

3. Why do newer versions keep coming out ? Why isnt the existing version just updated by fixes ?
In Ubuntu we dont have to fix things. The OS is constantly improved and so new versions come out with these improvements. You do not have to install the new versions if you dont want to. I have just installed 10.04 LTS and aim to keep it for at least the 3 years.

4. If I have a particular version installed and a newer version comes , then will I have to reformat the installed version and install all the software again ?
No! you can install the new version if you feel you must always have the newest version. You can either Update which means you just update your system to the newest version or you can do a clean install. In this case you either save your Home files on another drive or you can partition your drive in such a way as to have your Home partition separate from the OS so you never loose any files when installing new.

5. Are all the softwares for free ?
Mostly. All the software  you can directly install from the Ubuntu and partner depositories are free. (over 30,000) Some open source developers may ask for  contributions to help support them but you can decide if you want to or not. 

6. Would the functionality of OS (not in terms of stability but softwares and other features) be on par with Windows ?
Most anything you can do with windows you can do better with Linux. There are however some exceptions. Linux also have a lot of games but not all games written for windows will run on Linux. There are however software you can install on Linux that will allow you to run Windows programs on Linux (example Wine)

7. Is Ubuntu virus prone ?
Not at all. Never used a anti-virus program, don`t need it, never had problems. Some people use anti-virus not to keep Linux safe but to make sure they do not send Windows viruses to friends.

8. KDE and GNOME - two desktops. How can we have them on the same Ubuntu ? Do all the versions of Ubuntu give the option of installing both the desktops ?
Not sure here. I never used any thing else but Gnome but I believe you can switch between them.

### Post by orlox on 2010-05-30
Some few windows games can have better performace when they are runned in ubuntu, using wine (someone gave the link before for the application database), but if you really spend a lot of time gaming, keep your windows partition (or as many here say, use what fits your needs better, which in the case of gaming is (sadly) windows).

If you are trying ubuntu for any of the other million use cases where it excels, then dual-boot.

Also, what mike555 means by a rolling release distro, is a linux distribution that is kept always very up-to-date, by adding frequent updates to many applications. If you're not familiar with linux, I just recommend you to stay with ubuntu.

The update model in ubuntu, consists in fixed versions for most programs in each release. Meaning this, that when the last version of ubuntu is out, the versions of each program are mostly fixed, and the updates are mostly bug fixes and security fixes. Even when a new version of ubuntu is out, you still have the same version of the applications you had before, and you can choose to update your complete system to the new release, or stay with the current one.
