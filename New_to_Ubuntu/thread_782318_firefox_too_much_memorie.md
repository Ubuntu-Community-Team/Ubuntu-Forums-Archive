### Post by Sinkingships7 on 2008-05-04
> **lunaluna said:**
> firefox is taking about 150 mbs of ram!
is this normal?

How much RAM do you have?

If you have a sufficient amount (say, 1 GB or more), then yes, 150 MB is okay if you've been running it for a while. If it takes that much just to start it up, then no. That would be a problem.

If taking RAM from the memory heap isn't an issue, then firefox (and any app for that matter) will take as much as it can so that it can run comfortably. That is to say, it never *needs* that much RAM to run, but why not if it has that much available?
