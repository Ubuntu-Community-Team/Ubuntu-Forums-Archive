### Post by krendar on 2008-10-11
You can use Unetbootin:

[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

Select network install of Ubuntu 8.04. Remember to select to install the the harddrive. The program can be downloaded in a version for windows xp as well. Then just reboot and you will be given the option to either boot XP or boot Unetbootin. Select "unetbootin" to start install of Ubuntu. Depending on your internet connection it can take a while as it is installing over the internet.

When you install Ubuntu you are given the choice to shrink the XP partition if I remember correctly (never did dual boot before). The boot loader of ubuntu will have an entry for Windows XP as well.
