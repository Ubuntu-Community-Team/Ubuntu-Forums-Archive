### Post by Marflus on 2009-09-25
For the tutorial, nhasian has given you a good link.

As for the background, that's simple. First you need compizconfig-settings-manager, which since you have desktop cube working I assume you have installed, but if you don't, then type this in a terminal:

```

sudo apt-get compizconfig-settings-manager

```

Then open it from System -> Preferences -> CompizConfig Settings Manager.

For the background picture to the cube, you want to click on "Desktop Cube" from the "Desktop" Category of the settings manager. Select the appearance tab, and then go down to "Skydome". Check the checkbox at the top of the section (labelled skydome) and in the field below select which image you want there. Easy!

As for getting rid of the cube gears, go back to the main CompizConfig Settings Manager screen. Under effects, simply uncheck the checkbox next to "Cube Gears".
