### Post by Thingymebob on 2008-12-12
If you have compiz (desktop effects) enabled and either the magnifier or enhanced desktop zoom plugin enabled (look in system preferences>compizconfig settings manager - Accessibility) you can use super + scroll up/down to zoom the desktop (super is normally the key with the windows logo on it)

You may have to install compiz settings manager,

* System>administration>Synaptic Package Manager
* type compizconfig in the search box
* right click the package and mark for installation
click Apply
