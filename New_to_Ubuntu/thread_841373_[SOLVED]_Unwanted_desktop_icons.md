### Post by sydbat on 2008-06-26
As long as they are only links, I don't see why you couldn't put them in the trash. Try one, see if you can still access it via Applications > whatever section > whatever program. If it works fine, you can empty the trash. If not, go into trash and restore it to the desktop.
