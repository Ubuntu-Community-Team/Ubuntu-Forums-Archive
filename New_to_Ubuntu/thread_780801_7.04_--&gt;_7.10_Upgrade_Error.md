### Post by mrzaius on 2008-05-03
That one's cake. You're referring to an apt source in your /etc/apt/sources.list file (also in the aptitude configuration) that can be removed with a simple text editor. Just strike the line that references that URL, and you'll be 90% back up and running. You just won't have access to the packages that you got from that 3rd party source until you fix the entry. Check your version info on that line.

Just out of curiosity, can you please specify how you performed your downgrade/uninstall?

### Post by RequinB4 on 2008-05-03
> **Spoken said:**
> lol first off, how do i remove the line from the text editor (im still a noob in some areas of ubuntu)

and i didnt downgrade / uninstall.  luckily dell has a restore to factory default option in the grub menu lol.

First, Backup
```

sudo cp /etc/apt/sources.list /etc/apt/sources.list.backup

```

Now open the file
```

sudo gedit /etc/apt/sources.list

```

Now delete the line that has "hendrik.kaju.pri.ee" in it.
