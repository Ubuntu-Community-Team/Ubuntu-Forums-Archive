### Post by QIII on 2013-08-01
Hello!

Adobe stopped development of Flash for Linux some time ago and is only releasing security updates.

I believe 11.2 is the latest version for Linux, so if your game requires 11.5 you may have to bite the bullet and use a Windows machine.  There is no 11.5 for Linux.

Chrome comes with Pepper Flash built in, and I don't know how that keeps up with Flash.  It might be a step behind what your game wants.

Best wishes.
