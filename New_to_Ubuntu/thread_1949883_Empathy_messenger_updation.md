Hi,

Usually Empathy is installed from the official repository for your release, and that's the recommended version to use, unless you absolutely must have a feature found only in later versions, or your setup doesn't work for some reason.

If you really need the latest version, try installing from this PPA:
[https://launchpad.net/~telepathy/+archive/ppa]("https://launchpad.net/%7Etelepathy/+archive/ppa")

Instructions for adding the PPA are given on that page itself.
