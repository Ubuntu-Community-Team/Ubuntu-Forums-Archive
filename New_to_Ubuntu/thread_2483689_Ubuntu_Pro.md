### Post by ian-weisser on 2023-02-06
Ubuntu Pro is a proprietary service.

While some folks might have user experience like yours, nobody here has special insight or knowledge or training. We have not seen the code. Forum help is limited.

Most folks' problems are self-inflicted: Typoes, didn't follow [the instructions]("https://ubuntu.com/pro/tutorial"), tried to use a non-LTS release, generally didn't understand how Ubuntu sources work, etc. We can help you with most self-inflicted problems, because most of the help consists of us saying "start over, and this time follow the instructions."

If you are absolutely sure that your problem is not self-inflicted, then re-read the instructions. It tells you exactly whom to contact for additional help with Ubuntu Pro services, and how to contact them.

### Post by MAFoElffen on 2023-03-14
Quoted from Canonical Blog:
> 
Canonical's Ubuntu Advantage for Infrastructure subscription is now rebranded to Ubuntu Pro (Infra-only) with no price or scope changes.

I did the testing for Ubuntu Pro subscriptions for "Ubuntu Members", getting over 5 machine-subscriptions (50 machines).

When I asked their Ubuntu Pro Team where to direct people from the Forum to live help, it is their team that you can contact at this page:
[https://ubuntu.com/pricing/pro](https://ubuntu.com/pricing/pro)
Using the Chat icon in the lower right of the page...

Note that when that Icon say "Leave a Message", that means that Tier 1 support is offline. So, it looks like that tier is no longer 24/7 support. But they do give phone numbers there to call.

EDIT: I sent some emails to the wiki admins for the Ubuntu Member pages, recommending changes to update the "Ubuntu Member" pages on update instructions to clarify the hows for us... I couldn't edit those myself because of the permissions on those pages.
