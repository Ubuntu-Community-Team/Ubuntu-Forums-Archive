### Post by LazarUbus on 2008-04-22
1. Install startup manager:

sudo apt-get install startupmanager

2. Go to System—>Administration—>Startup-Manager

3. In Appearence Tab you'll find what you want.

Details here: [http://www.ubuntugeek.com/startup-manager-change-settings-in-grub-grub2-and-usplash.html]("http://www.ubuntugeek.com/startup-manager-change-settings-in-grub-grub2-and-usplash.html")
