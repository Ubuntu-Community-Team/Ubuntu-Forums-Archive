Just have two quick questions:

1. 2.  This is potentially an obvious one, but when running a Kubuntu session will the update manager (or equivalent) function properly and update the Kubuntu desktop with the latest fixes to KDE? (I've read it's moving pretty fast, and I think version 4.2 is out or close to being so)
