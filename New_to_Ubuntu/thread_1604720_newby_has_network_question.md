[http://www.econowics.com/internet-guide/147/install-netgear-wgr614-in-linux-ubuntu-804-hardy-heron/](http://www.econowics.com/internet-guide/147/install-netgear-wgr614-in-linux-ubuntu-804-hardy-heron/)

The reason many home routers don't mention Linux is that they haven't written instructions for getting on-line with Linux and they haven't included software that makes the process simpler.  So once you get it started and find the admin website that CharlesA mentions, you will need to know a little about networking to get it going.   

As soon as you get the website up and running, remember to change the admin password for the website and reboot the router.  Otherwise you risk someone else accessing your router and from there, who knows.
