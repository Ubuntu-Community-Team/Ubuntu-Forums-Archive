### Post by ZankerH on 2009-10-19
For the record, I find the easiest way to learn bash is by example - eg, pick a task you want to do, do it in GUI, than search for a way to do it in bash, and do it. Getting and retaining knowledge through hands-on learning like this is a lot easier than memorising what looks to you like a seemingly random list of commands, switches and argument syntax.
