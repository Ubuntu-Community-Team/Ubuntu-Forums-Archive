### Post by coffeecat on 2011-06-18
And I have never understood why Firefox is setup by default to use the backspace key to go back a page in Windows and MacOS, but not in Linux. It irritates me enormously that I have to re-configure Firefox every time I install a Linux OS.

@computerandu, in Firefox type "about:config" in the address bar. Click through the childish here be dragons message. Type "back" in the filter. About 5 down you'll find "browser.backspace_action". Double-click on it. Change the 2 to a zero. End of annoying problem.

### Post by pjaysnowden on 2011-07-23
If you wanna go forward using the backspace key....just press and hold Shift while you press backspace. This also works with the Tab key. Pressing Tab moves to next link...Shift-Tab moves to the previous link. I hope you find this helpful.
