### Post by slooksterpsv on 2011-04-28
You can, would it be recommended? No.

Cause you won't get the latest patches, security fixes, etc. which if your server was hacked, there could be exploits there due to not having the latest updates. You may think it'll never happen, but it can.

### Post by Paqman on 2011-04-28
If you want to avoid doing upgrades all the time, you should use the LTS version. LTS releases of Ubuntu server are supported for five years.

The current LTS release is 10.04, it will be supported until April 2015. So it would be perfectly safe to keep using it until then. If you're running 10.10 then support will run out April next year.
