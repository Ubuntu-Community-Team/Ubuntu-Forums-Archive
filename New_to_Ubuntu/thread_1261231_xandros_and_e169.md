### Post by cariboo on 2009-09-10
Xandros is(was?) based on debian, the instructions you find were should work for you. THe first thing to do is to open a terminal and type:

```
lsusb > lsusb.txt
```

the above command creates a text file called lsusb.txt, copy the file to your computer with a network connection and paste it in your next post.
