### Post by Rex Bouwense on 2015-03-03
That geyish looking icon is probably a generic web browser.  If you want the Firefox icon
1.  Right click any open space on the LXPanel.
2.  Choose Add/Remove Panel Items.
3.  Select Application Launch Bar (should be the second item in the list).
4.  Select Edit.
5.  On the right side of the window choose Internet.
6.  Select Firefox Web Browser.
7.  Select Add and then the icon will appear on the left side.  You can move it where you want it by using the up and down buttons.
8.  Close the window and the Firefox icon will be on your LXPanel.
