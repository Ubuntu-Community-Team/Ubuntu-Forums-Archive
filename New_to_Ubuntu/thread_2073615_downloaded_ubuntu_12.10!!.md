Hi!

If you have a 64 bit system, you should be able to download the 64 bit version of Ubuntu and be OK. Whether it will work completely out-of-the-box with your hardware depends on what hardware you are using.

Thankfully, there's a lot of resources that list easy, compatible hardware from graphics cards to keyboards. And, the community here is pretty great, so if you do run into any major issues, you should be able to get assistance.

One thing tho: I've booted up a Live CD before and didn't have any issues. Once I installed, I had to tinker with a few different drivers, which was easy, but one thing I learned is my experience with the Live CD doesn't always translate to how my install goes.

Enjoy, and welcome to Ubuntu! :)
