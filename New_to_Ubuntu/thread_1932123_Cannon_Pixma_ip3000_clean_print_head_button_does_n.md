### Post by Krytarik on 2012-02-27
> **Paper Pusher said:**
> When I get into the properties dialog the test page button works but the clean print head option button is not turned on, so I can't deep clean
Out of my own experience, driver support for Canon's IP3000/IP4000 series in that regard under Linux is really poor; quite a while back, I've tried very hard to get some better drivers working to particularly support the maintenance features you can access under Windows very easy.

There is a manual way to initiate a print head *cleaning*:
> 
[LIST]
[*]Ensure that the Inner Cover is closed.
[/LIST]

[LIST]
[*]The Print Head can also be cleaned from the printer itself. With the printer on, hold down the RESUME/CANCEL button until the POWER lamp flashes once.
[/LIST]
Source (choose "Quick Start Guide" there; PDF):

[http://www.usa.canon.com/cusa/support/consumer/printers_multifunction/pixma_ip_series/pixma_ip3000?selectedName=BrochuresAndManuals](http://www.usa.canon.com/cusa/support/consumer/printers_multifunction/pixma_ip_series/pixma_ip3000?selectedName=BrochuresAndManuals)

But it seems like for initiating a print head *deep cleaning*, you got to resort to either Windows or Mac OS X.
