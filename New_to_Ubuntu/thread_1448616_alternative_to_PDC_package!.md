### Post by 3rdalbum on 2010-04-07
> **MBG1987 said:**
> is there an software for ubuntu 9.10 to view an PDC files?

No. PDC is a format of Digital Restrictions Management. It locks the files to a specific computer or ebook reader running a specific operating system with specific software running on it. If the format was readable by other programs, it would defeat the DRM.

LockLizard doesn't make a version of the software for Linux.

In future, don't download "copy protected" or "DRM-protected" files, they are simply not usable on Linux, and even on Windows they can be a bit of a nightmare.
