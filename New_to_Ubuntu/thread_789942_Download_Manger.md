### Post by danpos on 2008-05-11
> **Nikunj93 said:**
> i want for normal torrent downloads
also i need a scheduler for linux

Torrent client: **[qBittorrent]("http://qbittorrent.sourceforge.net/")**.

Download Manager: **d4x**.

```
sudo apt-get install d4x
```

It works great with **[flashgot extension]("https://addons.mozilla.org/pt-BR/firefox/addon/220")**, where you configure this to catch links up to **d4x**.

Scheduler: **[cron]("https://help.ubuntu.com/community/CronHowto")**.

Danpos.
