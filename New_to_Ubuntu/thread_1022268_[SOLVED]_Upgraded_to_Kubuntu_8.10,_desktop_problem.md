### Post by judytownshend on 2008-12-27
I don't understand enough about computers to know exactly why this happened, but one of my friends told me to do this and it worked.

System -> System Settings -> Advanced -> Service Manager -> Detecting RANDR (monitor) changes - untick the 'use' box to disable this, and the screen stops flickering.

Note that if you plug in another monitor this may not be automatically recognised.
