### Post by northern lights on 2008-08-05
The picture shows a modified GnomeDo.

With all repositories but 'proposed' and 'backports' enabled, run```
sudo apt-get update && sudo apt-get install gnome-do gnome-do-plugins
```

There are extra plugins for amarok and rhythmbox, if you use either, install also...
