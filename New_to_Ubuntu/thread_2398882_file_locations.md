### Post by oldfred on 2018-08-19
Back when I had XP and started dual booting Ubuntu, I moved the profiles to a shared NTFS data partition. I then used same profile in XP and Ubuntu. Profiles now are in shared ext4 partition, and shared among various Ubuntu installs.
