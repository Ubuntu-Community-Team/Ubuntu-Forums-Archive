### Post by Mark Phelps on 2011-08-20
The only way you will find out for sure is to boot into Ubuntu using a LiveCD.  If there is no CD drive, or you can't boot from one, then Google for unetbootin.  It will allow you to create a bootable USB stick/drive -- which you can then boot from to see how well Ubuntu detects the touchpad hardware and installs the proper drivers.

But, don't be surprised if the touch function does not work -- which will pretty well rule out Ubuntu if that happens.
