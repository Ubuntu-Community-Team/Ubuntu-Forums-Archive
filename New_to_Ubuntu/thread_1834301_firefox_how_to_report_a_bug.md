### Post by kaldor on 2011-08-27
Use the _[Mozilla Bugzilla website]("https://bugzilla.mozilla.org/")_ to report all bugs for Firefox. However, you are using a quite outdated version of Firefox and it may no longer exist. I recommend the _[Firefox PPA]("https://launchpad.net/~mozillateam/+archive/firefox-stable")_ to upgrade to the latest stable version.
