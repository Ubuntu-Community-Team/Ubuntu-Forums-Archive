### Post by e4uforums on 2010-04-21
IE will run under Linux, just not real well.  You can use PlayOnLinux ([http://www.playonlinux.com]("https://addons.mozilla.org/en-US/firefox/addon/11327")) to configure WINE and install IE 6 or 7.  The functionality is not perfect.  I use it on my install but only to test website compatibility.  I'm not sure how well it would work for actually getting things done online.

There's an add-on for firefox that will let you change your http header information (user-agent) and present yourself as a different browser ([https://addons.mozilla.org/en-US/firefox/addon/11327](https://addons.mozilla.org/en-US/firefox/addon/11327)).  This may yeild some results.  It depends on what exactly is preventing browsers other than IE form working.   If the site needs a plugin like Silverlight, you may be out of luck.  If they just prefer IE, you may be in business.
