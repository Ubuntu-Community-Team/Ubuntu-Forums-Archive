### Post by jbrown96 on 2009-03-21
I've had problems with a few themes. usually they need a "theme engine", but I've never figured out how to get them to work.

The generic way to install a theme is to go to gnome-look.org. Choose the gtk-2.x section. Themes come in a "tar-ball". Download the file to your desktop (makes it easy to find it). Open the appearance section (System===>Preferences===>appearance) then go to themes. Just drag the downloaded file into the window.
