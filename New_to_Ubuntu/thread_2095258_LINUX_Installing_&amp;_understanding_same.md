### Post by coldraven on 2012-12-16
Before you do anything else make a backup of your files. If you install any OS mistakes can happen but if your data is safe then it won't be a problem.
Most Linux distributions have what's called a Live CD which lets you boot up and try it out without installing anything. Even easier is to create a bootable USB stick using "Startup Disk Creator" or "Unetbootin". It saves wasting a CD if you decide that you don't like that version of Linux.
Either way you will need to change the boot order in your BIOS to allow booting from CD or USB before the hard drive. It can be changed back later.
