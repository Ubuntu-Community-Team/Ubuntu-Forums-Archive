### Post by john newbuntu on 2010-07-12
Yes you can run linux straight on it.  You don't necessarily have to partition or format it.  The installer does it automatically.  Just pop in the CD, install it and give it the entire disk to work on.
[http://www.psychocats.net/ubuntu/installing](http://www.psychocats.net/ubuntu/installing)

If you are familiar with partitioning and have the time, plan on whether you need separate partitions for /home, an NTFS partition for storing data or more partitions for other installs in future .  In this case, you would partition upfront and/or then choose manual partitioning while "preparing disk space".
