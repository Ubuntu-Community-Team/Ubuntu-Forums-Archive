### Post by hansolo4949 on 2011-02-06
If you are posting on this forum and not getting answers, then perhaps you should post the thread in a different place, such as the absolute beginners talk forum subsection. What is your problem that you posted that no one has answered? Perhaps it will be answered on this thread. If the thread is old and buried, you can give it new life by posting a "bump" on it, which should bring it back up, and get you responses. This forum moves quickly, so if it hasn't been answered in a week, you probably won't get any more responses.
