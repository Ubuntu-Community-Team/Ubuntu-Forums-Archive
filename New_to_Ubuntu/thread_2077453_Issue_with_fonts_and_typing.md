### Post by grahammechanical on 2012-10-29
When we install Ubuntu we get a dialog that gives us the option to install third party software. That software includes Microsoft's core fonts. We are asked to accept the end user license. If we do not accept those terms and conditions then the Microsoft core fonts will not be installed.

This no hardship as there are very good equivalent open source fonts available as part of the default installation.

If you need those fonts then search the Ubuntu Software Centre for mscorefonts and you can install a utility that will automatically download each of those fonts and install them for you. You will need to accept the End User license otherwise the download will stop.

As regards the second issue, that is to do with your keyboard layout. Go to System Settings>Keyboard layouts and add a layout that is a better fit for your keyboard. You did get as part of the Ubuntu installation process an opportunity to select and test a keyboard layout.

Regards.
