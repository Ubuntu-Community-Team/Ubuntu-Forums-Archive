### Post by etnlIcarus on 2008-09-10
Go to Tools>Addons>Plugins.

Right-click on anything relating to Flash or Adobe and Disable it.

Restart Firefox.

If Firefox doesn't crash as often, it's because of the Flash player plugin. There are various fixes around the forums, including [downloading Flash Player plugin 10](http://labs.adobe.com/technologies/flashplayer10/). or doing the following in a terminal:

~ sudo apt-get install libflashsupport
