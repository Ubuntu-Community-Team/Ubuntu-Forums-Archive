### Post by mprince on 2009-05-06
Open up Synaptic Package Manager and search for 'ubuntulooks'. (Use the search button)

[INDENT]System> Administration> Synaptic Package manager[/INDENT]


Right-click on the 'gtk2-engines-ubuntulooks' entry and select 'Mark for installation'. Then hit the 'Apply' button.
