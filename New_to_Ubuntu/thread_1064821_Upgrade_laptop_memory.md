> My manual warns me that I should only use DDR333 SDRAM.

DDR2 Will not fit into a DDR1 slot, they have a different amount of Pins. DDR2 is so cheap because it is the most purchased and the companies have to be very competitive. DDR1 is not used on any new machine and only for older machine that need updating, so the companies making the stuff can charge more.
