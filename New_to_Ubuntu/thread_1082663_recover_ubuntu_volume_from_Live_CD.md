### Post by prshah on 2009-02-28
> **duke max said:**
> Can the Live CD recover the volume without losing the data stored on it? 

The live CD can be used to recover your data and files from the HDD provided there is no physical damage.

If you want to "recover" the installation, you can use the "alternate" CD instead, and choose the bootup option "rescue a broken system."
