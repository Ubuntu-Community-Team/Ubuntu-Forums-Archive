### Post by Darkade on 2008-07-09
Frets on fire is in the repos, you could install it from a synaptic
just go System>Administration>synaptic package manager
and do a search for Frets on fire, select it and install it.

from a console i *think* the package is called fof

so try 
```
sudo apt-get install fof
```
