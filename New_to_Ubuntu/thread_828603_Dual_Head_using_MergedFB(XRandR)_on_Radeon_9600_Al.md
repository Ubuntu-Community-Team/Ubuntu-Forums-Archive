### Post by tonyman on 2009-12-18
Hello, 

I'm no expert at this, but I also have an  ATI Radeon 9600 AIW  and I had the same issue with xrandr saying my other monitor is disconnected and not seeming to recognize it. I finally managed to get the dual head (big desktop) display working. I'm using Ubuntu Karmic Kaola 9.10

Here's my xorg.conf I basically started out with the one you posted and modified it by adding a few lines and commenting/deleting some others. The stuff that I added is in **bold**. 

```

Section "Device"
    Identifier    "ATI Technologies Inc RV350 AP [Radeon 9600]"
    Driver        "ati"
    BusID        "PCI:3:0:0"
    BoardName       "RV350 AP [Radeon 9600]"
#    Option        "XAANoOffscreenPixmaps" 
[B]    Option        "monitor-VGA-1"    "Flatron1"
    Option        "monitor-VGA-0"    "Flatron2[/B]"
EndSection

Section "Monitor"
    Identifier    "Flatron1"
    VendorName    "LG"
    Option        "DPMS"
    Option         "PreferredMode" "1280x1024_60.00"

```You can restart X and try out the new xorg.conf with right-alt + Prt Scr + k. 

For the next step, I just followed the directions from here [http://ubuntuforums.org/showthread.php?t=1112186](http://ubuntuforums.org/showthread.php?t=1112186) , to create and add the 1280x1024_60.00 mode to the monitor that xrandr -q listed as disconnected (VGA-0). 

1. use gtf to create a mode line
```
ubuntu@ubuntu:~$ gtf 1280 1024 60.00

  # 1280x1024 @ 60.00 Hz (GTF) hsync: 63.60 kHz; pclk: 108.88 MHz
  Modeline "1280x1024_60.00"  108.88  1280 1360 1496 1712  1024 1025 1028 1060  -HSync +Vsync
```2. add the new mode using xrandr
```
ubuntu@ubuntu:~$ xrandr --newmode "1280x1024_60.00"  108.88  1280 1360 1496 1712  1024 1025 1028 1060  -HSync +Vsync
```3. add this new mode to the output (monitor VGA-0)
```
ubuntu@ubuntu:~$ xrandr --addmode VGA-0 1280x1024_60.00
```4. finally, set the new output and mode to the right of the main output
```
ubuntu@ubuntu:~$ xrandr --output VGA-0 --mode 1280x1024_60.00 --right-of VGA-1
```Here is what my xrandr -q looks like now```
