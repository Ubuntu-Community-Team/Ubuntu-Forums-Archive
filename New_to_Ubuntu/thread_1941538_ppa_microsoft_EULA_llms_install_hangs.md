I found the answer on another forum.

terminal freezes at Microsoft EULA

Go down the page in the terminal with pgdown (the text will scroll down)
When you see 'OK' then hit right arrow to highlight it with the cursor.
Then hit ENTER.
Then use right and left arrows to select yes or no.
Hit ENTER again.
