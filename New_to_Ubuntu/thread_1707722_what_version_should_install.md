With Ubuntu the LTS (Lont Term Support) release is 10.04 it is supported for 3 years from 04/10. The latest release is 10.10, supported for 18mths from 10/10(the version is the date it was released). 

Going to an older version of Ubuntu will not necessarily help you. Going to a lighter version of Ubuntu will. That would be Xubuntu. 

The releases are the same(10.04 - 10.10) etc. Since you're new, you'll probably want to go with 10.04 LTS.

Ubuntu may or may not work a little better on your laptop. One good thing about Linux & open source is that you can try it for free. If Ubuntu doesn't work well, Xubuntu should. If Xubuntu doesn't, Lubuntu will.

imho, with 512MB of RAM, Xubuntu 10.04 Lucid Lynx is your best bet.
