### Post by Puck7 on 2009-12-21
If you are trying to install conky would be better to install it through the terminal, or Add/Remove (Ubuntu Software Center). The command to install it through the terminal:
```
sudo apt-get install conky
```

To install it through the Add/Remove or Ubuntu Software Center, depends what kind of Ubuntu version you have, just open the software, and write conky in the search box.
