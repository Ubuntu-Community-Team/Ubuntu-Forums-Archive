### Post by Kasami on 2011-01-09
First of all, welcome to Ubuntu and the forums.

Next getting themes working:

Go to System > Preferences > Appearance

Then click a different them and see if that works. Did you want to install another theme? If so go to a site for downloading them, I suggest gnome-look.org click gtk 2.x on the left hand sidebar and look for one you like, download it and open Appearance. Click "Install..." and look for the file you just downloaded (Default is "Downloads" then you should be able to use it.
