When i go to Google, the query box is black, and i don't really want that. This happened because i am using a different theme, i was wondering if there was a way to revert to the old colored query boxes without changing my theme.

---

### Post by kyalee on 2008-11-03
System--> Preferences --> Appearance 

Click "customize", go to the tab that says "colors" and change the color of the background for "input boxes".
