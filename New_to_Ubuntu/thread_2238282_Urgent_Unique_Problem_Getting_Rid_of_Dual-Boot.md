### Post by TheFu on 2014-08-07
I don't think you can do what you ask without administrative rights.  Sure, you can delete partitions by booting off a liveCD, but that won't change the size of the Windows partition - that should only be performed inside windows to avoid data loss.

Changing partitions is risk and should only be done after a full, complete, know-you-can-restore backup is made. Be careful.

If you are willing to take some risks, **fsarchive** may be able to help, but you'll need to improve your skills from end-user to something closer to IT administrator to use that tool.
