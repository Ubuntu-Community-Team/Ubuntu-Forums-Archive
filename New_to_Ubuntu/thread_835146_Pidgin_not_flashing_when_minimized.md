### Post by Dynaflow on 2008-06-20
If you go into Tools -> Plugins, you can enable the Message Notification plugin.  That will give you a bigger set of notification options.  Enabling the "Window Manager 'URGENT' hint" might make the window pulse on the taskbar.  Play around with the settings for the plugin, and see what they all do.
