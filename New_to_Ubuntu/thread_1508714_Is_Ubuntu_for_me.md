### Post by TeancumTBH on 2010-06-13
Hi everyone,

I had an old HP Pavilion PC running Windows XP, and I had very few problems.  Until the motherboard died.  So I built a new PC, relevant specs are in my sig.

Rather than purchase and install Windows 7, I decided to try a Linux distro.  I found the newest version of Ubuntu (10.04 32-bit I think?) and installed it from a CD.  I am NOT familiar with Linux or how to use it, and I've been struggling to get some basic things set up and working.

I got Skype and Firefox working with a wireless network adaptor, so I can at least see and talk to my family while I'm on a summer internship for the next couple of months.  But the PC was supposed to be my basis of entertainment, and it isn't doing the job.  Games and video (playing DVD's) are my biggest problems.

I found Wine, which (allegedly) allows me to play windows-based games in Ubuntu.  But I've had problems with each game that I've tried that make the games unplayable.  I also have installed nearly every DVD application that I can find, including VLC and Kaffiene, but still can't play DVD's on this machine.

When I search for answers, all I find are posts that are at least 3 years old and involve a lot of what seems to be code writing.  I'm willing to give things a fair shot, but if doing simple tasks like playing Starcraft or watching a DVD in Ubuntu are going to require me to have a knowledge of coding, then I think I may be better off with Windows.  Sometimes the devil you know is better than the devil you don't.

So what do you guys think?  Would I be better served by moving back to Windows, or should I continue to give this a shot?

### Post by howefield on 2010-06-13
> **TeancumTBH said:**
> But the PC was supposed to be my basis of entertainment, and it isn't doing the job.  Games and video (playing DVD's) are my biggest problems.

You likely only need to install the relevant packages to play your DVDs.

[https://help.ubuntu.com/community/RestrictedFormats/](https://help.ubuntu.com/community/RestrictedFormats/)

> I found Wine, which (allegedly) allows me to play windows-based games in Ubuntu.

Wine isn't a panacea that allows all windows applications to run, check with the application database and the wine forum here for more info.

[http://appdb.winehq.org/](http://appdb.winehq.org/) 

As for the coding comment, I assume that you are talking about the command line, most operating systems have one, Ubuntu is no different, I'd be surprised if you ever had to resort to the command line if you didn't want to, but like everything else, a bit of perseverance would go a long way.

Also, bear in mind that Linux is not Windows, it has a learning curve just as there was when you started using your previous operating system.

Do you have the patience, inclination and time to learn how to operate your new system, No ? Then use what you know.
