### Post by deadflowr on 2013-10-23
When was the last time you installed any updates?

Updates come to supported releases throughout the cycle(now 9 months for non-LTS).
They are unrelated (for the most part) with the updates coming for a new release.

I think in the software settings there should be a section that has a notify for version.
Normally it is set for either Any new version, LTS versions only, or Never.

if you set this to never, then you can be sure that the updates are for the existing version, without reservations.
Of course I can't remember what Kubuntu calls this section.

Edit: yeah looking at Kubuntu you open the software updates > then go to more and click open configure muon update manager > then uncheck distibution upgrade > then apply.
Then recheck for updates.
