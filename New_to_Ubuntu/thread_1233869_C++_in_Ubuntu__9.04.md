### Post by furuno on 2009-08-07
Well altough I'm a Java programmer, maybe this'll help :

Programming in any OS is I think about the same, it just depend of you IDE (Integrated Development Environment) of choice, especially when you're only doing single-tier application (not client-server and such). If you're into C++ then here's some of nice IDE that you can choose :
[NetBeans for C/C++]("http://www.netbeans.org/downloads/index.html")
[Eclipse for C/C++]("http://www.eclipse.org/downloads/")

If you're programming a Linux apps then usually you'll need to choose you're GUI platform, it can be GTK (Gnome) or Qt (KDE), it's only a matter of preference tough...

For some C++ programming tutorial I found on the net :
[C++ Programming]("http://en.wikibooks.org/wiki/C%2B%2B_Programming")
[Understanding C++]("http://en.wikibooks.org/wiki/Understanding_C%2B%2B")

### Post by braddcadd on 2009-08-07
If you are learning I would start with geany and learn how to compile with gcc/g++.  Then use kdebug because it is so smooth.

[http://www.geany.org/](http://www.geany.org/) (you can get it from Applications->Ad/Remove)
[http://www.cprogramming.com/g++.html](http://www.cprogramming.com/g++.html) (gcc/g++ usually comes installed in Ubuntu)
[http://www.kdbg.org/screenshot.php](http://www.kdbg.org/screenshot.php) (also in add/remove)

If you want an all in one tool, try Qt Creator.  It has  lot of tutorials and is an up-and-coming app.  Although, I find it a little too bloated for my small apps.

Good luck!
