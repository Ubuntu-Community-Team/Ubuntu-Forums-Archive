As per previos posted this has been asked but I have found an alternative solution to the problem.
So this is mearly an update to help others.
To uninstall Zoiper simply open terminal and
run the following commands:

cd /usr/share/Zoiper  ---(default directory specified on install)
sudo ./Uninstall

follow the promps

and you are done
