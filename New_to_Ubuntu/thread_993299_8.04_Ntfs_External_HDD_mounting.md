### Post by ByteJuggler on 2008-11-27
Try the following:
1.) Open a terminal (Applications->Accessories->Terminal);
2.) Enter the following commands in the terminal (or better yet, copy and paste them), one after the other, pressing enter after each:

```
sudo mkdir /media/500GB-External
sudo cp /etc/fstab /etc/fstab.backup
sudo echo "/dev/sdh1 /media/500GB-External ntfs-3g defaults,locale=en_IE.UTF-8 0 0" >>/etc/fstab
sudo mount -a
```

3.) After the last command, the disk should've been mounted, and should now continue to be auto mounted on boot. **Make sure to enter the commans *exactly* as posted**, or you could cause trouble.  (As I say, best to copy and paste from the above.)

Note/Warning: The above is probably not ideal for an external disk since the above assumes the disk will always be available at boot time.  If this is not always going to be the case, then we should probably try to find you another solution.
