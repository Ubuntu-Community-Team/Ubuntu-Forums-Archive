### Post by yancek on 2024-10-25
Are these empty Tor directories in the /home/Downloads directory of your user?  What happens when you navigate to your user Downloads directory and run: rmdir  Tor?
(You would need the actual/exact name of the Tor directory).  Did you try left clicking the directory icon, then hitting Shift + Delete?  You should see a window asking if you want to delete the named directory/file.  This is a default Ubuntu so if you are using some other related OS/distribution or Desktop environment, the second method may not work but probably will.

Have you done an online search for removing apps from the panel.  If you are using a current Ubuntu and are referring to the standard panel on the left of the screen you should be able to right click on the icon and get options including Remove.
