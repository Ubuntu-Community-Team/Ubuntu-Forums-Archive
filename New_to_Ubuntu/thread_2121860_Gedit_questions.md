### Post by Frogs Hair on 2013-03-03
If you navigate to the document location from File > Open  only one instance of Gedit will open and  a new tab for each document will open. If you select  a document from its stored location a new instance of Gedit will be opened displaying only that file. This is the behavior I see on 12.10.

---

### Post by audiomick on 2013-03-03
Further to that, if you find yourself in the situation that you have multiple instances of gedit open and want to have it all in one instance on tabs, you can click and drag the tabs in to the one instance, same as you can in firefox, for instance.

Conversely, you can drag a tab out of the gedit window and let it go to have that tab appear in it's own (new) instance.


That is to say, I just tried that on this 10.04 Netbook remix install. I assume the behaviour is consistent in other versions.
