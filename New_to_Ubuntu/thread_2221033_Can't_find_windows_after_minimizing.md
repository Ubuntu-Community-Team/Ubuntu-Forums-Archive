### Post by deadflowr on 2014-04-30
> **thek2 said:**
> , I can't use keyboard shortcuts like alt+tab because it will cycle through by windows applications rather than the ubuntu ones.  Can anyone help me?

Is this through a VM?

I ask because if it's a normal Ubuntu(non-VM), you would only be accessing Ubuntu programs.

If so, then it's probably a screen resolution problem.
Having Applications and Places means you should be running gnome classic, rather than gnome-shell.
Gnome-Shell lists Activities.

If it is gnome classic, then the task manager is probably on the bottom panel.

(A panel is the bar across the top of the screen, gnome classic should have two, one at top and one at the bottom.)

If there is no bottom panel,place mouse cursor on top panel and try windows key + alt + right click > add to panel, look for task manager.
