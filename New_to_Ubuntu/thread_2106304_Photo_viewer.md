### Post by ajgreeny on 2013-01-20
You can use either desktop for any application; it really doesn't matter if it is a qt or gtk app.

If you want to compare them use ```
sudo apt-get install kubuntu-desktop
```or if you prefer just the kde desktop without all the kubuntu extras and artwork etc etc, there is a smaller kdebase (I think) package.  Then just choose your session at the login screen.
