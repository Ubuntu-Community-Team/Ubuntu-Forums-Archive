### Post by talsemgeest on 2008-12-18
Just to make sure I am understanding your question right: You have 20 pcs which, when ubuntu is installed, has incompatible hardware causing the USB, LAN and Graphics cards to not work.

OK, if you can install ubuntu onto one of the pcs, then get the hardware working (restricted drivers for graphics etc...), you can use a program called remastersys to create your own version of ubuntu to be installed on the other pcs.

You can install it with ```
sudo apt-get install remastersys
```

Hope this helps :)
