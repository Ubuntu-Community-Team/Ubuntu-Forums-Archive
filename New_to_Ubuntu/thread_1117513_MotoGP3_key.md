### Post by ibuclaw on 2009-04-07
Have you set the d8input.dll as "**Native**" ?

To do this, select the motogp3 installed program in PlayonLinux, then select "**Tools->Configure Wine**". This opens up the winecfg program. Go to the "**Libraries**" tab then add **dinput8.dll** as **native**.

Iain
