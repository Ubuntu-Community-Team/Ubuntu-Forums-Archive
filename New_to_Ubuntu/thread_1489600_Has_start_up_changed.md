### Post by KdotJ on 2010-05-21
Lucid is known for it's fast boot times. It now uses the 'upstart' process to boot up instead of the older 'init' process. This means scripts can be executed and loaded asyncrinously during the booting process, rather than one after the other. 
Enjoy it!
