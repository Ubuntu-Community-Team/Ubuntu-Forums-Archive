### Post by Didius Falco on 2010-04-20
You can probably get everything you'll need through the repositories, particularly for a minimal version.

You can install the mini.iso from here:

[http://archive.ubuntu.com/ubuntu/dists/lucid/main/installer-i386/current/images/netboot/](http://archive.ubuntu.com/ubuntu/dists/lucid/main/installer-i386/current/images/netboot/)

This is truly a basic install - it's a simple command line interface, but it gives you access to the internet, and thus the repositories. You could then use a custom script to install just what you want - which leads me to this second link - a thread devoted to minimal Ubuntu installs:

[http://ubuntuforums.org/showthread.php?t=1155961](http://ubuntuforums.org/showthread.php?t=1155961)

You can get ideas and scripts already written that you can edit yourself to add or remove particular packages you want in your own install.

I hope this helps!

Regards,

Didius
