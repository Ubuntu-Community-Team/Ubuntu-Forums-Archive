### Post by audiosnow on 2011-05-20
I've got two separate hard drives in my laptop, one running Windows 7 and the other Ubuntu 11.04.

Personally, I prefer to use [EasyBCD]("http://neosmart.net/dl.php?id=1") to control the dual-boot menu, because the interface is a bit cleaner than GRUB. It was a very simple process to install GRUB to the Ubuntu partition and configure EasyBCD to accept both Windows' MBR and Ubuntu's GRUB.

Best of luck.
