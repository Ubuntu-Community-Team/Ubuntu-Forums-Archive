### Post by grahammechanical on 2011-04-10
Canonical takes responsibility for Ubuntu and for the genuineness of applications in the repositories. It will approve updates to the Ubuntu operating system and allow updates to programs that are installed. Canonical cannot approve the updates to these applications because it did not produce them. They are coming through Canonical not from Canonical. This is what this message means.
