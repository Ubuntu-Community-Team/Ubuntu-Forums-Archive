### Post by candtalan on 2008-07-29
> **grumpyhillbilly said:**
> I'm trying to install xubuntu on my kids computer in the living room. It's an older compaq originally with ME on it. It will load up till you see the pheonix on the screen and then sticks. Any idea how to get around this?

Almost for sure this is because you are using a live CD (the so called desktop CD) and the RAM needed to run it all without use of the hard drive is a problem.

With such a machine it would be more usual to just install from a different CD, a so called Alternate Cd. Very similar install sequence although it avoids gui (in ram) and uses text lists for information.

Can be downloaded here for example
[http://mirror.anl.gov/pub/ubuntu-iso/CDs-Xubuntu/8.04/release/](http://mirror.anl.gov/pub/ubuntu-iso/CDs-Xubuntu/8.04/release/)
choose the alternate install cd.

good luck
