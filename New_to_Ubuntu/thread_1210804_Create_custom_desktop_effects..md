if you want to set up all the 3d effects and wobbly windows and whatnot:

> sudo apt-get install compizconfig-settings-manager

### Post by nhasian on 2009-07-11
compiz is installed by default in ubuntu, but you need to install the compiz configuration settings manager to make any changes.  with ccsm you can make windows burn up with fire when you close them or fold up into paper airplanes and fly down to the taskbar.  you can make and rotate a desktop cube/sphere/cylinder, and loads of other nifty eyecandy.  also i recommend the new openGL enabled Cairo-Dock
