### Post by mikewhatever on 2010-05-30
System->Admin->Software Source, in the drop down menu select 'Other...', then click 'Select the best server'. Reload when prompted. If you need Medibuntu, go [-->here<--]("https://help.ubuntu.com/community/Medibuntu"). Drapes is in Universe, so just use the software center.
