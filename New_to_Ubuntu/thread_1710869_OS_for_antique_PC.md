### Post by MooPi on 2011-03-20
Don't want you to forget about just Ubuntu, as it will run on those spec's. Install a command line install from the Alternate install CD. At prompt after the install you can add whatever additions with regard to memory limitations. Try this ```
sudo apt-get install openbox obconf pcmanfm leafpad xorg xdm feh
```This should give you a basic desktop with notepad and file manger for starts. Feh can handle your desktop background. That should give you a starting point that is lean and fast on minimal specs. Be sure to keep it lean with additional applications for a complete setup. In addition I'd give yourself ample swap space on such minimal memory.

+1 for Lubuntu.

Please note that with the low RAM, you must install using the "**mini-iso**" CD.  

No magic though - that is a very minimal CPU even for Lubuntu.  

I definitely agree with installing more RAM.  Your system can handle up to 512Mb (256Mb in each slot).  

If you can't max it out at this time, I would strongly suggest you populate *at least one slot* with a 256Mb module and keep the other slot with whatever is in there now.  You can do that yourself for about $25 using brand-new, high quality RAM and for much less using  pulled/budget stuff.
