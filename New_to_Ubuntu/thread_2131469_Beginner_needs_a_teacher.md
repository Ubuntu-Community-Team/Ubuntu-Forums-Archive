There are many ways that you can customize Ubuntu to your liking. You can change the desktop background by right clicking and select change desktop background. If you like to work in the termminal and you dont like the standard purple you can change the layout by right clicking click edit profiles go to colors and there should be an option that's checked by default it says something along the lines of use standard colors or something like that uncheck that and then select your txt color and background color. Also if you find that you dont like Unity there are older version of Ubuntu that you can install that have Gnome insted also there are spinoff distros such as Kubuntu which features KDE instead of unity. Just as some general advice if you want to play movies in Ubuntu you have to install the decryption package open the terminal and type   sudo apt-get install libdvdread 4 then once thats finished type sudo/usr/share/doc/libdvdread4/install-css.sh  also to get important updates in ubuntu open the terminal and type sudo apt-get update, you can also do sudo apt-get upgrade. In order to watch youtube on ubuntu you need flash. To do this open up the software center and type in flash plugin and then download the flash plugin. There are many many more things that you can do in linux and there are many ways to customize it and make it your own i hope i've been helpful.

### Post by Impavidus on 2013-04-02
Welcome to Linux & the Ubuntu fora.
Pick your favorite desktop environment and choose your favorite theme. You can also download extra themes and even design your own. This way you can give your box the look you like. For installing software it's best to stick to the official ubuntu repositories until you get more familiar with linux. This is the easy way of installing software and will prevent you from breaking your system. The windows way of installing software is less save. Most software designed for windows doesn't run on ubuntu – but plenty of alternatives available: [http://www.linuxalt.com](http://www.linuxalt.com). You may want to install the **ubuntu-restricted-extras** package. It contains a collection of popular software (flash player, dvd codecs, microsoft fonts) that cannot be distributed on the installation disk for legal reasons. You can also install separate parts of it individually.
