### Post by C.S.Cameron on 2008-09-14
2G is a bit small for full Ubuntu.
Try ubuntu-eee
[http://www.ubuntu-eee.com/index.php5?title=Main_Page](http://www.ubuntu-eee.com/index.php5?title=Main_Page)
I boot my wife's eee from a pendrive to get the full version.
You can install to pendrive from the Live CD same as to HDD.
Just tell the eee,s bios that you want to boot from the PD first.
Ubuntu takes a lot longer to boot than Xandros eee.
