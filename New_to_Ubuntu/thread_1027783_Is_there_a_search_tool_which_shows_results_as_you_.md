### Post by 2hot6ft2 on 2009-01-01
Not exactly what you're looking for but here's something along the same lines but for the shell.

> fish (friendly interactive shell) and I must say I really like it. The subcommand tab completion is very cool.

You can type sudo apt-get install and then the very first couple of letters of the package and hit tab and fish will show you all possible packages that match so far.

Fish is a shell geared towards interactive use.  Its features are focused on
user friendliness and discoverability.  The language syntax is simple but
incompatible with other shell languages.
