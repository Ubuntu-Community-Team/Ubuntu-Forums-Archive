### Post by Xiong Chiamiov on 2008-06-05
Kubuntu uses KDE, while Ubuntu uses GNOME.  Edubuntu is aimed at schools, and Xubuntu at lightweight systems.

KDE and GNOME differ in their philosophy in that GNOME tends to be more difficult to customize, while KDE is very easy to customize, but has an abundance of menus, which can be quite confusing sometimes.

You can install any of the various options together, by installing the kubuntu-desktop, edubuntu-desktop or xubuntu-desktop packages once you've got ubuntu installed.
If you just like the way it looks, you can install just the desktop environment without all the extra packages (KDE from Kubuntu and XFCE from Xubuntu).
