### Post by y-lee on 2008-06-07
I don't think you can turn it off (somebody correct me if I am wrong) but you can chose or install or create a new cursor theme.

in your menu under preferences choose cursor selection (or Apperance-->customize themes depending on ubuntu version) and see what you have available. You can also look around online for [new mouse cursor themes]("http://www.gnome-look.org/index.php?xcontentmode=36"). See [How to install cursor themes]("http://tombuntu.com/index.php/2008/05/19/how-to-install-cursor-themes/") for more info.
