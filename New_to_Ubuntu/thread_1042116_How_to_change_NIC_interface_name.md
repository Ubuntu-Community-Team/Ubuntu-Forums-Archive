### Post by tristangrimaux on 2010-06-09
Tried this:

```
SUBSYSTEM=="net", ACTION=="add", DRIVERS=="?*", ATTR{address}=="your-mac", ATTR{type}=="1", KERNEL=="**eth***", NAME="**lan0**"

```

And it works in Hardy and in Lucid.

The kernel is going to identify the eth part, so keep it there. The naming is for your own handling.

After changed the names I've issued a 

```
config -a

```

to see if ALL the interfaces and there it is! Probably you'll have to change also settings in /etc/network/interfaces and in many places too.
[B]
RESTART FOR THE CHANGES TO TAKE EFFECT!!![/B]
