As far as I know you are correct.  They are configured with a certain packages by default.  They all use the same repository and kernel I believe.  I know the repositories are because I have installed some kubuntu packages on my ubuntu desktop.

Ubuntu uses Gnome desktop and apps.  Kubuntu uses the K Desktop and apps.  Both of those are aimed toward modern computers.  I would say if the computer is 3 years old or so it should run fine.
xubuntu is running the xfce desktop.  It is a less memory and cpu intensive option.  Xubuntu is aimed at older computers.  Mostly those that have 1 gb of memory or less.

Linux its self can run on very little memory.  The issue basically is running a graphical desktop like Gnome, KDE, XFCE, etc.

Hope this helps.

### Post by Legendary_Bibo on 2010-07-19
Yep that's the only difference. In order of hardware resources needed from heaviest to lightest

Kubuntu -> KDE Desktop Environment
Ubuntu -> Gnome Dekstop Environment
Xubuntu -> XFCE Desktop Environment
Lubuntu -> LXDE Destop Environment

applications run from any Desktop Environment just fine.

Gnome and XFCE are similar in layout but different in memory footprint and options for customization. A lot of people like how KDE looks, it looks really nice. Lubuntu is currently having issues at least for me but it's the lightest.

However if you've got 2gb of RAM whichever desktop environment you choose shouldn't be a problem. Also there's more than just these ones.
