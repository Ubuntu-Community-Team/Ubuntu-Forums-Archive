### Post by sanderd17 on 2011-03-06
You don't have to install it via a .tar.gz (which is just a kind of zip file),

The screenlets are availiable in the repos.

Just go in the software center and search for "screenlets" and you should be able to install them.

EDIT

Sorry, this is how to install the screenlets app. if you want to add a screenlet, you should start the screenlets app (press ALT+F2 and type "screenlets" or search it in one of the menus) and then choose to install a screenlet.
