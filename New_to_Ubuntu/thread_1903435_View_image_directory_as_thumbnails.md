### Post by audiomick on 2012-01-02
I assume you are using standard Ubuntu, in which case the file manager is Nautilus. On my 11.04 install, and I am sure it is the same on the other installs I have, there is a drop-down menu in the task bar at the top of the Nautilus window. If you choose "symbol", a folder full of pictures will show up as thumbnails.
