### Post by zeroseven0183 on 2009-10-14

Icons in the desktop are aligned by grid.

### Post by zeroseven0183 on 2009-10-19
> **rhuea said:**
> is there anyway to change them all evenly? and can i adjust the grid so it becomes much tighter?

I believe, until now, the option to adjust the desktop icon grid is still in-process. Check this out: [Idea #3272: Change the desktop icon grid size]("http://brainstorm.ubuntu.com/idea/3272/")

To adjust the desktop icon size
1. open **Nautilus**
2. click **Edit**
3. select **Preferences**
4. change the Zoom Level under **Icon View Defaults** and **List View Defaults**

### Post by CatKiller on 2009-10-25
> **Don Bowen said:**
> "Open Nautilus"  ... ??? where is that?

Nautilus is your file manager. You'll find it wherever you manage your files. Your Home folder, anywhere from the Places menu or by selecting Applications &#8594; System Tools &#8594; File Browser, or by pressing Alt-F2 and typing *nautilus*, or by opening a Terminal and typing *nautilus*, and probably a bunch of other places that I haven't thought of at the moment. If you're interacting with something in a window that isn't specifically another application, it's probably Nautilus.
