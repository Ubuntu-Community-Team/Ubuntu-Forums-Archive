### Post by AndyCooll on 2008-09-06
If you are using Ubuntu there is no need to install Digikam, the GNOME equivalent is F-Spot and that's installed by default. Or, as mentioned in the previous post, alternatively you could use Gthumb (which is also installed by default).

F-Spot has a great big "Import" button on the main window.
