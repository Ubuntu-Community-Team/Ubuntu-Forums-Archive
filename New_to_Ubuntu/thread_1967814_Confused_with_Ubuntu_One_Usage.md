### Post by jroa on 2012-04-28
When you first open Ubuntu One, it asks which folders you want to automatically sync.  When you select your shared folder, it will start to upload because it is in the process of syncing.  If you do not want to share everything in the share folder, create another folder to use just for Ubuntu One.
