Install vnc4server, on the computer, then on the windows 7 laptop, download and run a vnc client such as tightvnc.

Then, you can connect to the hostname of the (K)(L)(X)(E)ubuntu computer to view the Destkop

### Post by lytithwyn on 2010-03-18
You might also look at using FreeNX, which should be faster and give you better video quality.  Check out this page for instructions: [https://help.ubuntu.com/community/FreeNX]("https://help.ubuntu.com/community/FreeNX")

This uses a protocol similar in function to Microsoft's remote desktop protocol, and has similar performance.

I use this myself, and it was easy to set up.
