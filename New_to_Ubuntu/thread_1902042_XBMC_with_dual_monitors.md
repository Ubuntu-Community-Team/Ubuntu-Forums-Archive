### Post by papibe on 2012-04-07
If you have and Nvidia card and the proprietary driver installed, you can set 'Separate X Screens' on the 'Nvidia X Server Settings' (read [here]("http://dorkythorpy.blogspot.com/2012/03/guide-to-setting-up-dual-monitors-in.html")).

If you don't, I only know how to do it manually editing the X configuration file.
