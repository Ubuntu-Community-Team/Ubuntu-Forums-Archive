### Post by walt.smith1960 on 2010-03-01
I'm not 100% sure what you mean by "ubu" but if you're referring to an Ubuntu installation, I'm using 9.10 NBR and it works as expected. A common problem is Wireless networking (big surprise there :rolleyes:) My wired network adapter is Atheros AR8132, the wireless is Atheros AR9285. These were both problems in 9.04NBR. 9.10NBR supports them natively. The only button that doesn't function is the top left silver "disable the touchpad" button. I haven't needed it so haven't fussed with it. I find the Eee1005HAB much more responsive running Ubuntu NBR than running WinXP home that came installed. I hope this is what you were asking.

P.S. I found the touchpad difficult to use and purchased a notebook mouse. I've had Thinkpads and was happy with the pointer stick, but no Netbook manufacturer offers the pointer stick that I'm aware of.

### Post by RedOctober on 2010-03-01
I have several EEE PC "boxes" in the clinic that I manage.  Win XP was crashing badly on all of them, not at first, but, slowly, over time, I'd get "out of memory" errors and the menus system would "scramble" and mostly dissappear and when I tried to reboot it would tell me that I did not have access rights to reboot.  I installed Norton AntiVirus the day I got them, and after the trouble started happening I removed the Asus installed stuff like EEE Manager.   There was no app software installed on these things, because the staff was just using them as a dum terminal to remote desktop into our Win 2003 Server.  They were crashing only because of XP Home Edition. 

I installed Ubuntu 9.10 for desktop, on every one of them, as a complete replacement of the OS.  (They no long have XP on them).
I installed WINE and set up a rdesktop icon.

No problems so far, and it's been about 2 months.  No crashes, no freeze ups... Ubuntu just seems to work.
