### Post by Cypher on 2009-01-30
BY default Ubuntu chooses to make 2 partitions, one which holds all of Ubuntu and the second a SWAP partition. Since you chose the installation option, it removed the previous partitions and created this new scheme.

There are a couple of data recovery programs like Photorec that might be able to get data back for you.

Check out [this thread]("http://ubuntuforums.org/showthread.php?t=417761") for pointers and try those out.

If all of that fails the only option might be a data recovery place which can get very expensive..
