### Post by ddecator on 2010-04-22
I don't know if it's quite what you're looking for, but there are a couple of possibilities. 

On Ubuntu 10.04, Rhythmbox uses an icon on the top panel that you can click and use to control playback. The final version of this release is due soon, so you can wait and upgrade.

Docky (not GNOME-Do Docky, but the standalone version) has a Docklet that allows for the same functionality as the icon Rhythmbox places in the top panel, but it is located in the dock instead. This should work in every version of Ubuntu that Docky can be installed on.

As far as I know, there is no way to have it show the buttons in the panel, there are just the options to click the icon and select the option from there.

Hope this helps!
