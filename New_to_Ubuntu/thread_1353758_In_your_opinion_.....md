### Post by TBABill on 2009-12-13
Volcom, I have a slightly different perspective than those already presented. Since Linux versions are able to be run via Live CD, why not just download various versions and burn them to CD so they are bootable? That way you should be able to identify any compatibility problems before you disrupt your system with an installation. It's a very easy method of confirming you can run the system before committing to a partition, boot change, etc.

Just a suggestion. Besides, if it is a version you will be keeping you will need it on CD (or flash drive) anyway so you'll really only be out the download and CD time and cost of the CD.

### Post by t0p on 2009-12-13
I use 8.04 on one computer, 9.04 on another.  The one big improvement in 9.04 is the network manager.  Hardy's network manager stinks.

The main differences between the two is which version of which applications is installed.  Jaunty has newer software than Hardy.  But unless you are after a newer version of an app for some specific functionality, there isn't much point.  Wait a little longer and the next LTS will be upon us.  You can upgrade then, and you won't need to do it again for ages.
