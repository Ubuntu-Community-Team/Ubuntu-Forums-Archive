Sounds like it could handle it.

I would suggest trying the LiveCD. Burn the .iso from the ubuntu website and select 'try without making changes' when you boot to it.

Note that the LiveCD will run slightly slower than a regular install would, so dont make any speed judgements from the livecd.

If regular Ubuntu is too meaty for you, I would suggest using Xubuntu, which uses a lighter desktop manager that works much better on older systems.


Have fun
