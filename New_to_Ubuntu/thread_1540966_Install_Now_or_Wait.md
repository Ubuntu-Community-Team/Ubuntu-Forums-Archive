### Post by tophatandy on 2010-07-28
It all depends on if you like being on the cutting edge of ubuntu. If so, then just upgrade now and you can upgrade again when 10.10 comes out (which as previously stated will be in october. this is what the second number in the version number stands for: the month it was released and the first one being the year)

You don't have to do a fresh install every time you decide to upgrade either. You can actually just upgrade from command line and keep all of your files (this is easiest if you have an internet connection available for the machine, which i am assuming you do).
