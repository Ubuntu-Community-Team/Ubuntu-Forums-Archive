### Post by halitech on 2008-09-01
if you have either avi or mpg files, you can use ManDVD or Devede to create a burnable iso file. If you have mulitple clips, I would use ManDVD so you can create a custom  menu and put background sound on the menu. if its a single file, use devede and it will give you an iso file you can burn with brasero or k3b
