Hi! The dash (Ubuntu button menu) is showing the wrong browser. It is showing chromium instead of Firefox. Firefox is set as the default browser, I've checked the dconf key to confirm it. What needs to be changed to fix this?

Thanks in advance!

---

### Post by Mystro256 on 2011-05-04
Open Preferred Applications and make sure Firefox is set to the default in there. If there's two Firefox's, then choose the one with the icon. You can also try changing it to chromium via this menu, closing, testing via the dash, then open again and change it to Firefox. It seems to be a bug, I had this issue with the vice versa of what you have setup.
