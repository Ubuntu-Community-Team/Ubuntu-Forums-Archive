> **bob brazie said:**

Here's what you do:
1. Download the .Torrent file from here.
[http://releases.ubuntu.com/10.04/ubuntu-10.04-desktop-i386.iso.torrent]("http://releases.ubuntu.com/10.04/ubuntu-10.04-desktop-i386.iso.torrent")
[http://releases.ubuntu.com/10.04/ubuntu-10.04-desktop-amd64.iso.torrent]("http://releases.ubuntu.com/10.04/ubuntu-10.04-desktop-amd64.iso.torrent")
2. Run the .Torrent file using your preffered Torrent client (I would use Deluge)
3. While the file is download, install k3b image burner.
4. Run k3b and select the 'Burn ISO Image' option and select the .ISO you just downloaded.
5. Burn it **at the lowest possible speed** to avoid errors.

That's about it. :)
