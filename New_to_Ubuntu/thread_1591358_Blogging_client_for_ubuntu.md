### Post by theanalyst on 2010-10-09
There are a lot of Blogging Clients, which can post to blogger. gnome-blog is a rather minimalistic blogging client in the Gnome Desktop environment (the default desktop environment if you are using ubuntu). You can also try other blogging clients like blogtk. I personally use a kde blogging client called bilbo(now called blogilo) which has a nice preview feature for blogger blogs. All of them are listed in the synaptic package manager, just type something like blog in the search field. Have fun blogging:)
