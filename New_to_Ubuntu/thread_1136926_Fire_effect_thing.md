### Post by NightwishFan on 2009-04-25
Do this in a terminal window.

```

sudo apt-get install compizconfig-settings-manager

```

Then below appearance in the preferences menu is the advanced compiz effects configuration.

Look for the "animation" section, and it should be under there, just edit the current "close" animation to use fire. If burn is not under there, perhaps activate the animations addon plugin. I am sorry I can not give good help, I am painfully using Windows Xp right now.
