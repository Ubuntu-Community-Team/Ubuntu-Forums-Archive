### Post by concentricpuddle on 2012-03-04
Ah so it's already installed. Don't worry then, everything's probably already installed. Are you able to run it?

If so chances are the MusicBrainz library's been installed already.

puddletag's Tag Sources work's differently to Mp3tag's. Start puddletag then, in the Windows menu open "Tag Sources". In the Tag Sources list, check if "MusicBrainz" is there. If not, install it using the Software center. Start it up using this [this guide]("http://www.psychocats.net/ubuntu/installingsoftware"). Search for **python-musicbrainz2** then install it.

Start puddletag again after the installation, then check out [this]("http://puddletag.sourceforge.net/source/tut4.html").
