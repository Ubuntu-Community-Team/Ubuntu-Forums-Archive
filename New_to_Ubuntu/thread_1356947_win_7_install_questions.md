### Post by baddog144 on 2009-12-16
NTFS is the best choice for a Windows 7 install, and if you install Ubuntu using Wubi, it won't care what filesystem Windows 7 is using. Driverwise, if you use Wubi as opposed to native Ubuntu, there are no differences. And if you decide to install Ubuntu over Windows 7, that's perfectly possible too. 

Hope this helps :)
