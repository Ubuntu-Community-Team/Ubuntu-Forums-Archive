### Post by palmgrower on 2013-11-07
My nVidia GT 240 card is giving me trouble. I cannot install 13.10 because the screen freezes while CD is being read. Screen gets full of colorful speckles. Is there a way around? Thanks.

EDIT: Solved the problem by installing MinimalCD (non-graphic installation) and installed nvidia-current from command line (sudo apt-get install nvidia-current). Then installed ubuntu desktop (sudo apt-get install ubuntu-desktop). Next boot started with the usual ubuntu screen, not command line.
