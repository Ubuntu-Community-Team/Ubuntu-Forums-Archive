### Post by coffeecat on 2014-09-03
Well, of course you won't get the same as in Windows with a maximised Chromium window if you are using the Unity desktop. The top bar contains the application menus - that's why it's called the menu bar - and all the indicators on the right. There's no room for the Chromium tabs as well. That's the way the menu bar works. Windows has no equivalent. MacOS does and it would be interesting to see what Chromium looks like on a Mac desktop.
