### Post by jordanmthomas on 2008-06-28
Generally, metacity and gtk themes can be installed by simply moving the extracted folder **themename** to ~/.themes.  Inside the folder **themename** should be subfolders named metacity or gtk-2.0.
Icon themes can be installed by moving the **themename** folder to ~/.icons

This is how I install all my themes and I've never had any trouble.

Hope this helped some
