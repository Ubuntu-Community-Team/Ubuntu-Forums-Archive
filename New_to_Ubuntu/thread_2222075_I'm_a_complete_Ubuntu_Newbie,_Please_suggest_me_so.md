### Post by Impavidus on 2014-05-05
If you have only 7 years of Windows experience not all is lost. I had about 10 years of Windows experience before switching and I still managed. First forget everything you thought you knew about computer systems and then invest some time in getting to know Ubuntu. ### Post by sammykur on 2014-05-05
I am fairly new to Ubuntu and there are a few things I can suggest

1. Install synaptic package manager

2. Install lucky backup (It does take up quite a bit of room for the backup,but I have A large amount of storage)

3.Keep a record of the software you install  since a lot of it is free I seemed to try out programs for uses that were way too cost prohibitive in windows. I use a free cad program to design furniture I build , but Autocad for windows is way too expensive to justify buying it is one example.

4. I wish I would have found this guide for bash scripting sooner

[https://help.ubuntu.com/community/Beginners/BashScripting](https://help.ubuntu.com/community/Beginners/BashScripting)

5. when asking for help make sure to furnish all details that might be relevant like the hardware you are using or if you are trying to use a command in terminal what you are actually trying to do,as someone may have a better way of meeting the objective using a different approach than what you are trying . 

I think you will find everyone’s experience  with Ubuntu to be quite different ,from users who just browse online and store photo ect to those that highly customize it and utilize it in for things I don’t think windows is even capable of doing.

Have fun and  Welcome
