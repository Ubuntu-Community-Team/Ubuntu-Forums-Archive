### Post by NightwishFan on 2010-04-01
Saving all your settings and files simply means copying everything from your home folder to an external backup. (DVD, CD, External HD, or USB FLASH.. etc). Ubuntu will overwrite the root drive on a fresh reinstall.

Personally I advise you create a separate home partition on the new install so you never have to backup data on a crash or reinstall. (Backup anyway, but less urgent need to).

I advise you go with 64bit. I have never found a downside to doing so. I have used 64-bit with no issues since 2007. Also not to mention the potential performance gains. Any questions feel free to ask I will clarify.

### Post by 3rdalbum on 2010-04-03
If you want to wait, then don't let me stop you.

Some people forget that "open-source" also often means "open-community", and they think that, like in the proprietary software world, they'll only be able to use the software when it is released. With pretty much all Linux distributions, you can get development versions (alpha, beta, release candidate) and often even "daily builds" that represent the current state of development.

So if you'd prefer to wait until the official release, then wait. Or if you want to wait until Beta 2 (coming shortly) or Release Candidate (a few days before release) then that's fine as well. But you do have the option of jumping in right now, and then just using Update Manager every day; by the time Ubuntu 10.04 is officially released, you'll already be using the release version.
