I would recommend downloading and burning a CD or creating a LIVE USB from another computer of the newer version of Ubuntu that you want to run. Make sure to backup all your files off the old Ubuntu install and perform a clean install.

I think the clean install would be a better option than having to manually upgrade to each release until you hit the desired release. 

Once you have the system installed and running smoothly I would recommend looking at the [RestrictedFormats]("https://help.ubuntu.com/community/RestrictedFormats") page so you can setup mp3, flash, and java playback on the system.
