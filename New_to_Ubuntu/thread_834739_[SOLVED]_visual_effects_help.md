### Post by Delever on 2008-06-19
If you want to use proprietary driver, EnvyNG can help installing it. It basically does the same - downloads appropriate proprietary driver for your video card and installs it. It may also do it better. To get it, type

sudo apt-get install envyng-gtk

Then go to Applications -> System Tools, EnvyNG should be there.

Use it to install your driver and say if it succeeds/not.
