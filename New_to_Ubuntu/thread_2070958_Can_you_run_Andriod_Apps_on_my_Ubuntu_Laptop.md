### Post by ajgreeny on 2012-10-14
Not exactly an answer to your specific question, but you can get an iso of the android full OS which will run as a live system, and can be installed if you wish, from [Android-x86 - iso files.]("http://code.google.com/p/android-x86/downloads/list")

I have tried the **asus RC2** version on a netbook (MSI) which I use but only as a live system, where it seems to run very fast, though it does have a few glitches with some of the default apps which suddenly turn to landscape aspect, ie, on their side even though the mouse movements are still in portrait.  That makes mouse control extremely difficult.

I have no idea if it's possible to download and add extra apps to the system, live or installed, but thought you might like to know about this just in case it is any use to you.
