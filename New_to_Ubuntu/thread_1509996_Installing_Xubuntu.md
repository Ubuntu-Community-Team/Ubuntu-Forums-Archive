### Post by warfacegod on 2010-06-15
No need for a HDD wipe. If you can create a Live USB of Xubuntu (or wahatever) with either the USB Startup Disc Creator or unetbootin you can mark your drive or partition to be formatted, after selecting Manual partitioning during the install process.

If you go this route, you might consider creating a separate partition and marking it /home during install.

Don't forget to make a backup of your files.
