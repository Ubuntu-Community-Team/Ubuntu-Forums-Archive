### Post by westcoast01 on 2012-12-22
Hello Muk, more than one way but try this. Swing your mouse all the way to the left side of the desktop .. the top unity icon will be the "dash home" .. right click the dash home and select "Applications" .. you are looking for "Additional Drivers" so if you do not see it click "show more" .. in additional drivers select the Nvidia driver that you want then select activate. I would suggest using the driver that says "Recommended" If this solves the problem please mark the post as "Solved". Best of luck.
