### Post by prshah on 2008-11-13
> **patilkaustubhv said:**
> i am using core2quad processor.. 2gb ram....
which virtual box version do i use? i386 or amd64?


The core 2 quad processor can use either i386 and/or amd64; however, which version of virtualbox you use depends on which Ubuntu you have installed: i396 or amd64. 

If you have installed any version of Ubuntu, type i386, then you can use only virtualbox i386.

If you have installed any version of Ubuntu, type amd64, then you can use only virtualbox amd64.

I recommend you use i386 for both initially, until you are more comfortable with Ubuntu, at which point you can switch to the amd64 (64 bit) edition if you want.
