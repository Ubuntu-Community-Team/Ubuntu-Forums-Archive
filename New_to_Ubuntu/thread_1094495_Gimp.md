### Post by martrn on 2009-03-12
Launch the Gimp, eg '**gimp-2.4**' and select 

help-->help and this will launch help.

Scroll down until you see the section "3. First Steps With Wilber" and then read about the subsection "1. Basic Concepts"  subsection "2.1 The Main Toolbox".

There is a pointer on some of the tools when running the gimp, but usually it will also have an icon underneath the pointer when editing a graphic.  The Gimp is a full blown image editing aplication and the pointer with nothing other than a pointer would have no meaning would it ?

If you need a selction tool select the rectangle selection tool or the elipse selection tool or some other selection tool, but these have a '+' symbol for the cursor not a pointer.

The gimp is very difficult to get a grasp of at first but it is a very powerful image manipulation package once you get use to it.  Its not intuitive though and takes some getting used to.

> **Georgia boy said:**
> Was reading something in a Gimp manual PDF. It mentions about hitting the reset opbtion button in the toobox and if I read it right it wll set back to default. Hitting the reset botton in the toolbox resets the values of the current tool you are using to the defualt gimp settings simular to that when you open the gimp the first time you use it.  Eg, If I selected the fuzzy select too and changed the threshold tools the reset button will reset the threshold to the standard values for the tool I am using (eg fuzzy select.)  Gimp may remember the last tool you used.  Select the tool box on gimp and select File-->Preferences-->Tool_Options and deselect the 'Save Tool Options On Exit'

You can do this in the gimp without much problems but the interface for gimp takes some getting used to.  There are three windows for a start when working on and editing a file, and it takes, a fair amount of understanding if you have come from a previous application which has only one main menu for all its windows.  Its just diffrent not easier.

You can select using a variety of methods from the tool menu including selecting rectangles ellipses, free-hand selection and magic wandy type of hope selection.  You can also select a 'selection' from the menu on the image you are working on by colour, or make changes to the selection such as invert the selection etc.  It takes a while to get used to.

The gimp takes a while to get used to but worth the persistence.
