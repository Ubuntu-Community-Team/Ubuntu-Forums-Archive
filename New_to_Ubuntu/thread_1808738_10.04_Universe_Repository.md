### Post by CatKiller on 2011-07-21
> **TommyACD said:**
> b) my original /etc/apt/sources.list only had the line;

deb [http://archive.ubuntu.com/ubuntu](http://archive.ubuntu.com/ubuntu) lucid main

That's weird, but I've not done a server install, so I don't know for sure what's normal for that.

You can have a look at the repositories in a browser to see what's available, but in terms of Lucid releases you have a choice of *lucid*, *lucid-updates*, *lucid-security* and *lucid-backports*. Some or all of them might have *main*, *universe*, *multiverse* and *restricted* as components. You can specify multiple components per line, such as

```
deb [http://archive.ubuntu.com/ubuntu](http://archive.ubuntu.com/ubuntu) lucid-security main universe multiverse restricted
```Or you can have each component on a different line, for ease of commenting.

You're right that update will refresh the list of available packages.
