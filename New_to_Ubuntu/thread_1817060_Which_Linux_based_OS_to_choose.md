### Post by Sotanaht on 2011-08-02
Minecraft and Skype will run natively on pretty much any Linux distro, but idk about Hamachi and Teamspeak.  Try running the windows installers in Wine if they don't run natively on Linux.
As for choosing a distro to go with, don't just go by what people tell you on a forum, download the ISOs and make a liveUSB to try them yourself.  See which one has the interface you like the most.  I use Ubuntu 11.04 on my netbook/tablet and Ubuntu 10.04LTS on my desktop.  Since you're already experienced with 10.04LTS, don't bother testing it on your desktop unless you want to install it.
I'd recommend trying the following:
Ubuntu 11.04 (if you don't like the Unity UI, you can switch back to Gnome by choosing "Ubuntu Classic" though I'm not sure if you can do that on a liveUSB.
Linux Mint
Debian
Fedora
Red Hat

If you have a big enough flash drive, you can install multiple distros to one flash drive using [YUMI]("http://www.pendrivelinux.com/yumi-multiboot-usb-creator/")
