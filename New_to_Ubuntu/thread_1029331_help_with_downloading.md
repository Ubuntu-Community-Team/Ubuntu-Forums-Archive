### Post by MenZa on 2009-01-04
As for ARToolkit, I have no idea.

You can download Blender by running the following command in your terminal (accessible from Applications &#8594; Accessories &#8594; Terminal):

```
sudo apt-get update && sudo apt-get install blender
```

Mind you, there are about a million different minor forks of Blender. The one included in the Ubuntu repositories might not hold the capabilities you seek.
