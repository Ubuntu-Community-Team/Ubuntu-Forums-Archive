### Post by Mike_Walsh on 2014-10-13
Hallo, Colin, and welcome to the forums.

Are you by any chance trying to install it from the Adobe website? Because that will give you all sorts of problems, and it won't work. Besides which, there's no need.

If you go into the Software Centre (which should be on your launcher, to the left side of your screen), and use the search box, top right hand corner of the page, to search for 'flashplugin-installer', that should give you what you need.

Alternatively, search for 'ubuntu-restricted-extras', and install this. It will install some other media codecs, etc., in addition to the FlashPlayer, which for copyright reasons, cannot be included with the OS as released.

You can also install Chromium browser from the Software Centre. It never hurts to have a second browser, anyway, if only for troubleshooting purposes.

Hope this helps.

Regards,

Mike.

### Post by Mike_Walsh on 2014-10-13
Glad it worked out.

With regard to your soundcard, the drivers SHOULD be included in the kernel (the bulk of which, by the way, is nothing BUT drivers). However, if you search in the Dash (the icon at the top of the launcher), and type in 'Software and Updates'; then click on the 'Other Drivers' tab on the far right, if you need a proprietary driver for anything (doubtful for a sound card), and let this run for a few minutes; if there is anything to be found, the system will hunt it down, and it will come up, along with instructions as to what to do.

If this doesn't work, then please start a new thread, in the 'Multimedia' forum.....hopefully, you'll get a reply there.

Please mark this thread as 'Solved' if your posted question has been answered. You'll find the means to do this in the 'Thread Tools' at the top of the page.

Mike.
