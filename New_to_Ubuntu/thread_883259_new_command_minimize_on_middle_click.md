### Post by glennric on 2008-08-07
If you are using compiz with emerald you can do this by opening System->Preferences->Emerald Theme Manager, selecting the "Emerald Settings" tab, and setting the "Titlebar Double-Click Action" to Minimize.

---

### Post by hvac3901 on 2008-08-07
and if you like

system > Preferences > KeyboardShortcuts ; scroll down to windows management and find "minimize active window" if you dont like that key combination you can change it if the combo isn't being used for another function, thats the fastest way i know to do it, even though as i read it, its not what you wanted to do.

Normal run of the mill Ubuntu desktop 8.04; system > preferences > windows : "set titlebar action to what you desire." but it requires two clicks :)
