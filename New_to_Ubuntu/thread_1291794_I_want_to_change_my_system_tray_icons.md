### Post by hellblazer on 2009-10-15
As far as I can tell the system tray icons are part of the icon theme that you have installed.

Right click on the desktop, select *Change Desktop Background*
Select the *Theme* tab
Your current theme will be highlighted, click the *Customize* button
Choose the *Icons* tab
You'll be able to choose a icon theme from those installed.

This is not really what your after, cause this will change all your icons not only the system tray ones. You might want to search the forums for pixmaps.
