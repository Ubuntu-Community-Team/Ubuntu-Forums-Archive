I installed Adobe Acrobat and then uninstalled it.  Now two minor problems arose.
1) Evince is no longer in my "Applications" list, although it's still installed.
2) The default application for opening .pdfs is Gimp.

So how do I get Evince onto the Applications list again, and how do I change the default opening application?

Thank you.

### Post by Schendje on 2009-11-06
Evince is disabled by default to show in the Application list. Right-click on Applications, then choose "Edit Menu" and then you can enable it. (It's under Graphics.)

To open files with a different program, either right-click on a .pdf file and choose "Open With..." or, if that doesn't work, right-click and choose "Properties", then navigate to the "Open With" tab.

Hope that helps!
