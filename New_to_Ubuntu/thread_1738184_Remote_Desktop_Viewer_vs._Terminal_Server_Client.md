---

### Post by elzy89 on 2011-04-24
It quite simply in the name. RDV is a viewer with limited options and mainly centred around basic VNC access.

Whereas, TSC has many more options for the RDP protocol and is a proper client in terms of the way that it ports the sound and locally attached devices from the host machine to the client machine. There are many more advantages but I try not to confuse.
