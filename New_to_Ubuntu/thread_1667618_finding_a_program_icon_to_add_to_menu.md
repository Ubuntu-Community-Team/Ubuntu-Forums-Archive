### Post by flanagan on 2011-01-15
This seems too simple, but since you can see the icon on the program's launcher in the folder it was installed to, instead of double-clicking to launch, can you right click the launcher and bring up Properties? In the resulting Properties dialog the program's icon should be in the upper left corner. Click it and it should take you to where the icon resides.
