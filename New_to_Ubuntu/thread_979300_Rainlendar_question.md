### Post by MikeBrown on 2008-11-11
I use a program called Rainlendar in Windows Vista, and was wondering if there was any possible way to use an exported .ics (saved calendar file) with the default calendar/clock in Ubuntu. 

If that is not possible, is there an alternative way to keep my saved .ics calendar and use it in Ubuntu (besides using Rainlendar in WINE)?

Thanks! 

-Mike

### Post by wtruong on 2008-11-16
It's very simple:  first follow the instructions to install rainlendar on linux from the website.  Then right click on the rainlendar calendar and select options.  Click the calendar tab and then click add.  Then supply the path to your ics file.
