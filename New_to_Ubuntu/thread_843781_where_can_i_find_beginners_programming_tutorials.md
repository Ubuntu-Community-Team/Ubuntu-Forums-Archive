### Post by RKCole on 2008-06-28
Hello, kilrush.

This is not necessarily about C++ tutorials, per se, but you may be in search of an Integrated Development Environment (IDE) when you get into C++ programming.  If this is the case, I would certainly suggest [Code::Blocks]("http://www.codeblocks.org/"); it is a wonderful open-source product.  I have used it for years, although I still do not know entirely enough about C++ programming to provide significant contributions to open-source projects.  

In any case, there are Code::Blocks packages for Ubuntu 8.04 (64-bit) at [GetDeb]("http://getdeb.net/app/Code%3A%3ABlocks").

All the best in your journey into the programming world. :)

Take care.
