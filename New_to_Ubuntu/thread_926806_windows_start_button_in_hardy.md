### Post by prshah on 2008-09-22
> **Meow27 said:**
> 
is there a way to make the windows start button work for ubuntu to open the desktop enviroment's start menu?

> **meindian523 said:**
> Not the Start Button,but the Main Menu would do what you want.

I think he means the "Windows" / Super key on the keyboard.

Click System-Preferences-Keyboard Shortcuts; scroll down to "Show the Panel Menu" (Alt+F1) and click the "Alt+F1"; now press the Windows/Super key. From now on, the panel menu will open everytime you press the Super / Windows logo key on the keyboard.
