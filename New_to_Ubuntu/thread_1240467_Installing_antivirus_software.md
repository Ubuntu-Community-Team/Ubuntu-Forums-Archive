### Post by ad_267 on 2009-08-14
Like others have said, you don't need antivirus software.

Ubuntu isn't Windows, so you can't expect it to behave the same way. If you're after a free clone of Windows you've come to the wrong place. If you want a free, stable, secure operating system and you're willing to learn a few new things then stick with Ubuntu and use Google and these forums for help.
