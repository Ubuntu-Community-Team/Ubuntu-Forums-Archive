> **Can I add a "Graphic User Interface", [GUI] To a Server?**

 Yes you can, depending on what window manager you wish to use you can install the xserver and the window manager via apt-get. For details see the **[ServerGUI]("https://help.ubuntu.com/community/ServerGUI")** page. start there ;)

### Post by PriceChild on 2008-12-04
Just a note...

The "Ubuntu Server" is not really that different to the "Ubuntu Desktop". They both contain the same underlying packages, and take software from the same places. You can install server applications on the Desktop installation, and Desktop applications on the Server installation.
