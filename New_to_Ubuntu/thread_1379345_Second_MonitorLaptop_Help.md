### Post by bryceowen on 2010-01-22
In my experience with using an external monitor under Ubuntu, I found I had to modify the power settings to 'do nothing' when I close the laptop and set the display for the laptop to 'off' under display properties. That was the only way I was able to both deactivate the laptop display and close the laptop without the system shutting down.
