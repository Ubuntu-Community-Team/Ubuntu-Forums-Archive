### Post by LordEager on 2012-07-20
Supposing it's a only usb printer (not wireless) you should plug the printer to the pc first, then  download drivers from here [http://it.software.canon-europe.com/software/0028619.asp?model=](http://it.software.canon-europe.com/software/0028619.asp?model=)

It's a deb package so it should autoinstall.
Achieved this you should go to the system settings and check the printer status, it should show your printer and you should be able to print a test page.

EDIT: the link gives you italian drivers, you should go back in modules and complete it with your nationality info.
