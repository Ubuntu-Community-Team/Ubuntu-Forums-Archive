### Post by taurus on 2009-01-12
Can you boot your machine with the Ubuntu LiveCD?  You can use gparted in System -> Administration to delete all the partitions on your harddrive and format it to either fat32/vfat or ntfs.  Then, windows installer should be able to defect the harddrive/partition.

Otherwise, check the BIOS to make sure the CD is the first bootable device.
