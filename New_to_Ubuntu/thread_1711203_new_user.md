### Post by Hedgehog1 on 2011-03-21
Sorry to hear your SSD died.

I think you will need to install Ubuntu to get the driver loaded.  Do you have a USB stick you could install Ubuntu onto?  A faster one is better.  A 4 gig USB stick would work, but an 8 Gig will give you some working room.

Plug the USB stick in and click on the 'Install Ubuntu'.  Ubuntu will happily install on your USB stick if there is no hard drive.  Then you can run the updates (getting your wireless network driver in the process) and be able to save documents.
