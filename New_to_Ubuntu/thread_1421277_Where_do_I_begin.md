### Post by TBABill on 2010-03-05
Please identify your specific issues with connecting to the internet so users can better understand your problem and help you troubleshoot it. 
 
1. Broadband or dial-up?
2. Type of wireless adapter or dial-up modem?
3. Alternative connections possible - such as plugging into the router directly and attempting updates, then trying wireless again?
4. Software you are trying to install - hopefully not Windows software unless you are running Wine or another layer to process/understand the software. Ubuntu does not run Win software directly.
5. How you want to get this software? The repositories are the safest way, but there are others.
 
There are many other questions so any specifics will speed up the ability to get you answers quickly and get your machine up and running.
