### Post by Uluru on 2009-11-23
As I understand it, Wubi is a program which installs on Windows a Program called Ubuntu. Just like any other program under Windows. I've not done such but that is what Wubi does.

I have used VirtualBox to try different Linux Distros, it installs onto a HOST OS, you then install the GUEST OS into it.

Therefore, to answer; "im gonna try some other linux distros" Wubi would be of no use to you, as it installs Ubuntu on Windows, no other Distro.
VirtualBox will enable you to install any Operating System for testing.I suggest start reading  [http://www.virtualbox.org/](http://www.virtualbox.org/)
