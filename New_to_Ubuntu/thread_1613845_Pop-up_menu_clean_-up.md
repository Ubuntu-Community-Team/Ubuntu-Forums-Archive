### Post by itsols on 2010-11-04
I thought I'll share this with the community...

I used to have Quanta Plus and I uninstalled it. But when I right-click on a php file, to edit, I still see the pop-up menu with the words "open with Quanta Plus". Although GEdit is my default editor, nothing would happen if I double-clicked the php file since Quanta Plus is no more.

This is how I got rid of it.
Step 1. Right-click on the php file to get the pop-up menu.
Step 2. Select "Properties" and select the Tab that reads "Open with"
Step 3. Select the program to take off the menu. Eg: Quanta Plus.
Step 4. Click Remove.
Step 5. Click on the program name you want to use as default. Eg: GEdit
Step 6. Click 'Close' and you're done.

The same applies to all files that you would want to open with any program.

Happy Computing!
