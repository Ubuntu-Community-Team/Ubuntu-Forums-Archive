### Post by Noour on 2009-01-27
Just insert your XP pro CD and boot [In the BIOS, first boot must be from CD]. Windows XP Installation CD will ask you if he can erase everything so it can install. 

After you erase the partitions, it will ask you if you want to create NTFS or FAT32 partition. Better choose NTFS.

(did it many times)
