### Post by Frogs Hair on 2012-01-11
First . find out if the computer is Unity 3D compatible with the following terminal command .```
/usr/lib/nux/unity_support_test -p
``` 
Next , install the CCSM from the software center . The Ubuntu Unity Plug-in inside of the CCSM has some configuration options .

If you are content with Unity 2D the application at the link will offer some of the same options. [http://www.addictivetips.com/ubuntu-linux-tips/myunity-comprehensive-unity-tweak-for-ubuntu-11-04-and-11-10/](http://www.addictivetips.com/ubuntu-linux-tips/myunity-comprehensive-unity-tweak-for-ubuntu-11-04-and-11-10/)
