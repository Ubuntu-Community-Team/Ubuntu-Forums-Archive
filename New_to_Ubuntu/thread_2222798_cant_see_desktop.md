### Post by kc1di on 2014-05-08
Hi Farhan_Zafar and welcome to Ubuntu Forums,

You need to boot with the nomodeset parameter then when you get to the desktop you'll have to install a driver for your video card.
You can boot with nomodeset by holding down the left shift key just as ubuntu starts to boot,  on the screen that comes up there will be several f key options at the bottom of the screen , select F6 then from the list presented select nomodset hit the esc key and enter. 
good Luck
