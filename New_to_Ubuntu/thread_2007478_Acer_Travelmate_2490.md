I do not know the specs of a travelmate 2490, but I have a travelmate 2200 and would not consider Ubuntu 12.04 on that; it is not powerful enough, with a slow celeron cpu and only 512MB ram, nowhere near enough for Unity to run properly, and hardly enough for the classic gnome desktop.

Give either Xubuntu or Lubuntu a try instead; both need lower resource machines, but both have full access to everything in the ubuntu repos.  Of the two, Xubuntu is probably easiest to use as it has many more GUI configuration options that Lubuntu, which requires more manual editing of config files.  Both are good OSs with a much more classic desktop layout when compared with Unity/Ubuntu
