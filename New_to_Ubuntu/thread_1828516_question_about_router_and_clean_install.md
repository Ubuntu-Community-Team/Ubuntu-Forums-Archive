### Post by mikewhatever on 2011-08-19
It most certainly should. Routers usually don't require installations, they are plug&play devices, besides, whatever is installed in Vista doesn't affect Ubuntu in any way. 
So if a wubi installation works with the router, a proper installation should as well.
