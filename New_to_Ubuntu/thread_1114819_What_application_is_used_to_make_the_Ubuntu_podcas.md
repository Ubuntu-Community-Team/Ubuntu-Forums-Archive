### Post by balaknair on 2009-04-04
Try **Avidemux** or **Kine**, they're both pretty good video editing apps.
[http://avidemux.berlios.de](http://avidemux.berlios.de)
[http://www.kinodv.org/](http://www.kinodv.org/)

They've probably used one of these two, it's certainly possible to create stunning videos with them. Or maybe they used a Mac :P (supposed to be far better than Windows apps for Multimedia).
