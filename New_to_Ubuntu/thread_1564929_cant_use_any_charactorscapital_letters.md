### Post by coffeecat on 2010-08-31
Has this been like this since you installed, or has it just happened? It sounds as though you have the wrong keyboard layout enabled.

Go to System > Preferences > Keyboard > Layouts tab. Make sure you have the right country in the top field. Now see what is set by "keyboard model". What does it say? If this is obviously wrong, change it to something that works.

**Edit:** wrong keyboard would cause this in everything, text editor, Open Office, as well as Facebook and Yahoo mail. Is the problem just in Facebook and Yahoo mail, or everything?
