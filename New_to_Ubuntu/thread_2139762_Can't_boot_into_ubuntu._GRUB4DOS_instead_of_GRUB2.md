### Post by oldfred on 2013-04-27
Do not know about EasyBCD.

Some BIOS do have issues booting anything beyond a certain point on the newer large hard drives. Boot-Repair often suggests a separate /boot partition fully inside the first 100GB of a drive. But that requires you to move Windows NTFS partitions around. And Windows does not like to be moved. 
I prefer smaller system partitions for both Windows & Linux and then have data in separate partitions.

Do you really have a 2TB drive formatted as FAT32?
Unless you have to have it for an Xbox or other reason FAT32 should only be on smaller partitions, as chkdsk on a 2TB drive may take days since it has no journal.

[http://technet.microsoft.com/en-us/library/cc938932.aspx](http://technet.microsoft.com/en-us/library/cc938932.aspx)
