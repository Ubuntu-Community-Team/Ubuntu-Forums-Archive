### Post by stonecoldjha on 2008-11-22
well i disagree.neither does ubuntu look beautiful or professional by default,nor is this thread supposed to be closed.now asfar as making things beautiful goes,i assume that you are using intrepid.install compiz configuration settings manager,open it(system>preferences>compiz configuration settings manager) and modify and use compiz, and try out all of the effects.second,install emerald,and download some beryl themes from gnome-look.org,install these themes from emerald(open emerald,click on import and direct to the location where you have downloaded the beryl theme),install a dock( preferably the cairo-dock,and make use of its features like stack etc,),install screenlets manager,and download some exotic screenlets from gnome-look.org(now open screenlet manager through system>preferences>screenlets and click on install then directing it to the loaction of the downloaded screenlet.now download some GTK theme from the same website,and then go to system>preferences>appearence,open themes tab,and drag the GTK theme into it.similarly,an icon theme like hydroxygen can be dragged and installed.


presently i am using the GTK theme named kde4 oxygen,cairo-dock,emerald theme nimbus,hydroxygen iconset.as you can see in the screenshot a well configured ubuntu does look good.
