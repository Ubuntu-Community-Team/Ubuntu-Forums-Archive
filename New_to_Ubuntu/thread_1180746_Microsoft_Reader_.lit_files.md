### Post by MaxVK on 2009-06-07
I had a browse about my old CD collection and found a whole bunch of .lit files for the MS Reader. They are not usable in Ubuntu by default so I installed the MS Reader via WINE and found that the old files worked perfectly when dragged into the reader window.

I just thought Id post in case anyone else had a bunch of these files. For information:

Standard Jaunty Install
Standard WINE install
Downloaded Reader from MS site ([Here]("http://www.microsoft.com/reader/uk/downloads/pc.mspx"))
Installed Via Wine
Did NOT activate (Don't know what it is but I doubt that it would work)
Opened the reader from the WINE menu and that was it.
