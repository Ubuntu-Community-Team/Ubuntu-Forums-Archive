### Post by Paqman on 2008-11-05
First of all: do you want to upgrade your current system, or do you want to reinstall from scratch?

To upgrade: go to Update Manager and look for a button at the top telling you that version 8.10 is available.

To reinstall: boot from the LiveCD and go through the install process like you did the first time. If you have a seperate /home you'll be able to keep all your settings. If not, everything will get wiped.

### Post by Sef on 2008-11-06
> There is no listing on the Update Manager to install 8.10 and I have been trying to find any kind of previous post on installing it... I know there has to be some listing on 'How To...' Any ideas

System > Administration > Software Sources > Update > Change to 'Normal Releases' > Close

then you will be able to upgrade.

Be sure to** back up** your **data** before upgrading or clean installing.   There are not 100% guarantees that all will go well.
