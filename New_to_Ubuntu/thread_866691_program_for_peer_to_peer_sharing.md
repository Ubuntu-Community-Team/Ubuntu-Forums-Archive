### Post by ad_267 on 2008-07-23
Transmission is a bittorrent client, it isn't the same sort of thing as frostwire, which is more what you are after I think. You need to download a .torrent file from a website in order to download a file with transmission.

See [http://en.wikipedia.org/wiki/BitTorrent_(protocol](http://en.wikipedia.org/wiki/BitTorrent_(protocol))

rtorrent is a command line application. You can run it by typing rtorrent into a terminal.
