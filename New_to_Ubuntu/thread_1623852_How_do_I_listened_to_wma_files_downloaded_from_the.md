### Post by Blutkoete on 2010-11-17
Hello!

With the restricted extras installed (which you also need for MP3s), the problem might be that the file is DRM protected. 

Try download and play a file from e.g. here ([http://www.sound-emotion.com/sound-emotioncom/1024index.shtml](http://www.sound-emotion.com/sound-emotioncom/1024index.shtml)). If that works, the problem is not WMA itself, but the copy protection (DRM).

If that's the problem, I can't help you, but maybe others can.

Best regards,

Blutkoete

### Post by coffeecat on 2010-11-17
Which players did you try? Do you have the package ubuntu-restricted-extras installed? If you have ubuntu-restricted-extras you should be able to play unencrypted WMA files.

You could also try VLC - it's in the repositories - which is my favourite Swiss Army Knife of multimedia players.
