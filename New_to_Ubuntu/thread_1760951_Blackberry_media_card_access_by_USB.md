### Post by crispy_420 on 2011-06-15
I have a Blackberry Storm (v1) and it just works for me by asking if I want to switch to mass storage mode. If you dig through the "Options" menu there you can select "Memory". In that menu you can enable/disable "Mass Storage Mode Support" and configure the prompt to show or not.

Hope that helps.
