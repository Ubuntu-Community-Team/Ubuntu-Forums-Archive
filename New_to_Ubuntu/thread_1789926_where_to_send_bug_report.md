### Post by cariboo on 2011-06-25
You have to use ubuntu-bug to create a bug report. Press Alt-F2 and type:

```
ubuntu-bug <package_name> 
```

Where <package_name> = the name of the package you are having a problem with. I'd suggest you check [bugs.launchpad.net]("https://bugs.launchpad.net") to see if the bug has already been reported. 

**Note**: You need to have a launchpad account before you can report bugs.
