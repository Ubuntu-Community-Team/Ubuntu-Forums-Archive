### Post by rohedin on 2008-08-18
Beryl is outdated now, use compiz instead. You can enable it by going to System &#8594; Preferences &#8594; Appearance &#8594; Visual Effects and turning it on.

For wine, go to Applications&#8594;Add/Remove and then search for 'Wine' and install it. For a complete list of what is supported by wine then check out [WineHQ]("http://appdb.winehq.org").


To install anything you can just go to Applications&#8594;Add/Remove and search and select what you want to install.
