### Post by sikander3786 on 2010-10-25
Update Manager is a vital part of Ubuntu, how can it bring malware to your computer?

Actually it might. If you want to be sure, don't add PPAs i.e, extra repositories to your Software Sources. Only add the one's that you believe are secure. Launchpad PPAs are the safest one's. Another one's I trust are from webupd8.com.

If you are not sure, go to Software Center > Edit > Software Sources and see what you have added on the Other Software Tab.

Linux doesn't do anything without your permission. If you allow the malware to enter you PC, Linux won't stop it either.
