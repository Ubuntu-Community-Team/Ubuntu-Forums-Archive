How did you try to do that?  You first need to download the Ubuntu iso file, then either burn it as an image to a DVD or put it on a flash drive then boot the computer with the DVD or flash drive set to first boot priority and you can then try Ubuntu without installing or being the install.  You will have some choices so it makes a difference if Ubuntu will be the only operating system or if you will be keeping another operating system.  Take a look at the site below which will give you some information on the process.
