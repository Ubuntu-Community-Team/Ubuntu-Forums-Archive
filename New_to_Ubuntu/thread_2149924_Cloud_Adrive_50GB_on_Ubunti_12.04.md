### Post by ShadowVegan on 2013-05-30
Open a terminal (ctrl+alt+T) then type these commands:

```
sudo apt-get install icedtea-7-plugin
```

```
sudo apt-get install openjdk-7-jre
```

Type your password when prompted, and after doing both of those, try opening Firefox.
