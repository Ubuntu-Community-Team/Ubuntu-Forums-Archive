There have been updates to both relatively recently.

Try this Firefox add-on to sort out your flash installation.

[https://addons.mozilla.org/en-US/firefox/addon/161939/](https://addons.mozilla.org/en-US/firefox/addon/161939/)

> It will remove conflicting flash plugins from Ubuntu Linux systems and install the appropriate version according to system architecture.
