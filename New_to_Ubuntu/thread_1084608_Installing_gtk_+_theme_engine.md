### Post by bolucpap on 2009-03-02
Try using synaptic to look for the engine quoted and install it as you would any other application. For some theme engines it will be necessary to get a .deb package from another party if they are not included in the list of packages synaptic will let you install. Be aware that those packages will not update themselves, you will have to manually recheck the sources.
