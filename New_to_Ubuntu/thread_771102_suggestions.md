### Post by riven0 on 2008-04-27
Beryl and Compiz is going to run like a hog on those specs. I don't recommend it, at all. Also, you may want to try either Xubuntu or Fluxbuntu, which work better on older hardware. Otherwise, you can download the normal Ubuntu and install the Xfce desktop to use instead of Gnome, which will also be very slow on those specs.
