### Post by Hospadar on 2009-03-29
I doubt you're going to find one.  One area where linux is lacking is really high quality cad-type software.

That said, there are some cad softwares out there for linux, qcad comes to mind of the top of my head, but to my knowlege it doesn't do all the things a professional package does.  You could try running your software in wine (although a quick search of the [wine appdb]("http://appdb.winehq.org/index.php") appears as though this software doesn't run well in wine).

You might try using a virtual machine (virtualbox from sun is my personal favorite, it's very easy to install and configure, i'd go to their website and get the non-free-as-in-speech but free-as-in-beer version, it has more features than the ones in the ubuntu repository.

There's a couple other free cad programs for linux, a search in System->Administration->synaptic  or add/remove programs should turn them up.  I think also there may be some commercial cad softwares that have linux versions, although I'm not sure about that.

In the end it may just be easier to have a little dual boot set up on your system for drafting.  While sometimes some things can be made to work, often times I find it nicer to just run them natively (video games for example)
