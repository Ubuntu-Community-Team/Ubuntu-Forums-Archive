### Post by Mark Phelps on 2010-04-28
> **flameproof said:**
> ... How can I scan the HDD. I know FSCK is a nice tool, but how I use is for a NTFS HDD?

You can't use FSCK for NTFS partitions.  There are Linux tools (ntfsfix) that you CAN use, but they are not Linux versions of CHKDSK; they will only fix SOME NTFS problems, not all.

The only way to really fix the NTFS problems for sure is to attach your drive to an MS Windows box and run chkdsk on it -- and even then, it might not be able to fix all the problems.
