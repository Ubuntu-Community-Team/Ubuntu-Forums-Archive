### Post by DrTebi on 2011-05-06
I recommend using rEFlt to handle the dual booting for you. It's a boot loader specifically for Intel Macs. I assume that Grub can do the same, but as far as I remember, using rEFlt was a lot easier:
[http://refit.sourceforge.net/](http://refit.sourceforge.net/)

Most hardware should work out without a problem. However, you may want to just run Ubuntu from a Live-CD to test it out first.
