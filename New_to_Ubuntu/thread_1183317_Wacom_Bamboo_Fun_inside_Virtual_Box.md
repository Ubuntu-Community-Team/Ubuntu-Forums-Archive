### Post by SoftwareExplorer on 2009-06-09
Right Click on the CD icon in the bottom right corner of the Virtual Machine Window after it is running. Select Select the option that says something like host drive blah blah...

--or, if the virtual machine isn't running--

Select the Virtual machine you want to use the CD with where they are listed near the Left side of the Window. Then click CD/DVD-ROM (to the right of the CD icon, it somewhat looks like a hyperlink) in the right half of the window. Make sure that mount CD/DVD drive is checked and click the dot next to host CD/DVD Drive.

Then the Virtual Machine should be able to use it.

Hope that helps
