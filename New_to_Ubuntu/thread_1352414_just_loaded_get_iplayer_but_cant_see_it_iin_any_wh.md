### Post by Greenwidth on 2009-12-12
You can alias the commands you use most often to cut down on typing into the primitive apparatus, for example:
Create a file in your home called .bash_aliases 
```
alias bbchd='get_iplayer --vmode=flashhd -o /home/username/videos -g'
```
Then to get show xxx in HD:
```
bbchd xxx
```
