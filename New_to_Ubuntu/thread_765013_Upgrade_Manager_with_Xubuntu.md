The Hardy release will be my first upgrade with Xubuntu. I was wondering how the Update Manager handles the upgrade.

How does it know to not install Gnome?

I've seen a few articles that says to use the Updatee Manager to upgrade Xubuntu to Hardy but wanted to be sure that would not give me an Ubuntu install with Gnome vs Xubuntu with XFCE.

### Post by fluteflute on 2008-04-24
Ubuntu will look at waht you already have installed. In particular whether you have the 'ubuntu-desktop' package installed or the 'xubuntu-desktop' package.

Don't worry, you will not get Gnome (unless you have installed it in the past in addition to XFCE).
