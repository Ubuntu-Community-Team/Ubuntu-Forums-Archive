### Post by randy78 on 2008-10-30
The alternate install image is a text based installer, therefore it doesn't have a graphical installer like a Live-CD.

If you aren't very experienced with partitioning and the sort, you should download a Live-CD and then search for Ubuntustudio within Synaptic once you get up and running.

Hope this clears things up for you :)

### Post by nynoah on 2008-10-30
Ubuntu Studio can be added to any standard Ubuntu OS.  Just open synaptic and download all the packages.  That is all Studio really is, is Ubuntu with more programs.  Before you do that, I suggest turning off your proprietary graphics drivers.  If you don't you will get the black screen.
