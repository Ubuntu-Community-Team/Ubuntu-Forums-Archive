### Post by CJ Master on 2009-06-08
Ubuntu is ubuntu, but it's not exactly the same.

Wubi creates "virtual hard drives" on your windows installation, and then installs Ubuntu on there, then uses a bit of magic to allow Ubuntu to boot from them when you start up. You can remove Ubuntu just like any other program with Add/Remove.

Disadvantages to wubi is that you can't suspend to Hard Drive and that it will be a bit slower.
