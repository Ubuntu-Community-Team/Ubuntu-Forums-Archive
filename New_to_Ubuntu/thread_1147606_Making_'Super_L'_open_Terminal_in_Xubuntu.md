### Post by lorderico on 2009-05-03
Not sure if this will work in Xubuntu, but it works in regular Ubuntu.

Go to System -> preferences -> keyboard shortcuts.  Click Add.  Name it whatever you want, but put in gnome-terminal for the command.  Then go over to the shortcut column, click on "Disabled" and type "super-l".

Eric
