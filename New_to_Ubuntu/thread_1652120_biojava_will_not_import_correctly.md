### Post by shamanou on 2010-12-24
i have found the sollution thanks to a friend of mine!

i used a bash script to put the jar files in bashrc but i forgot to export the CLASSPATH variable, this is the script:

CLASSPATH=/usr/share/java/
for jar in /usr/share/java/*.jar ; do
 CLASSPATH=${CLASSPATH}:$jar
done
export CLASSPATH
