### Post by robert shearer on 2008-10-28
Agreed,' soundconverter' is the one.

Applications=>Accessories=>terminal.

When it opens type this (or paste)....

```
sudo apt-get install soundconverter
```

Once it is installed Applications=>Sound & Video=>Soundconverter.

When it opens go to the 'Edit' tab and open 'Preferences' where you can set it up for the conversion you wish to do.
