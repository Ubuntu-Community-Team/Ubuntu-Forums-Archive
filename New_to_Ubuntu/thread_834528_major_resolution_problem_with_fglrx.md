### Post by markbuntu on 2008-06-21
You need to use

aticonfig

to configure the ati driver. It will ignore changes made to xorg.conf unless you run the proper aticonfig command. Try this thread about that:

[http://forum.compiz-fusion.org/showthread.php?t=6794](http://forum.compiz-fusion.org/showthread.php?t=6794)

You can also type aticonfig --help and get a list of options to use with aticonfig. There are a bunch pertaining to configuring a tv to work with the driver. Usually if you have the tv connected and turned on when you boot, it will be autodetected and you can use Catalyst Control Center to set it up if it is not working to your satisfaction. The aticonfig command has more options than CCC so you may need to use that for final tweaking.

good luck.

### Post by markbuntu on 2008-06-23
What you need to do is remove and reinstall the catalyst control center that sort of worked before and use that to change the resolution. 

changing xorg.conf will not make any difference with the ati driver and displayconfig does not work with the ATI drivers.

The best way is with CCC. It seems to move around  with every install. Mostly it is in Applications/Other which is sometimes turned off in the menus. Once it was in Applications/System and once I found it in Applications/Accesories. In any case, this is what you need to change the resolution. 

Another thing you can try. If you get high resolution in the login screen change your session to gnome safe mode. The high resolution should continue. If it does then logout and login and the resolution problem should be fixed.

If this works what happened is that the resolution in xfce and gnome has been somehow screwed up and just logging back into your previous session keeps it that way. Logging into gnome failsafe basically reboots your previous session setting to the failsafe settings, putting everything back in order.
