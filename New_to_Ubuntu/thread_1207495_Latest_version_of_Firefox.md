### Post by Tibuda on 2009-07-08
The update manager won't update your 3.0 to 3.5, because it could break something in your system. You have to install the [firefox-3.5]("firefox-3.5") package to install the latest version, and you'll have both versions (unless you remove the firefox package. The 3.5 from the repositories is named Shiretoko, so people with the two browser installed don't be confused about it. But don't worry, if you open help > about you'll see Shiretoko is Firefox 3.5. The next ubuntu version will have 3.5 branded as Firefox.

If you don't like the Shiretoko name, read this thread: [http://ubuntuforums.org/showthread.php?t=1193567](http://ubuntuforums.org/showthread.php?t=1193567), but I really recommend you to stick with the repository version.
