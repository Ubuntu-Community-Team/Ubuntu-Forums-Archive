### Post by bpowell2005 on 2008-08-23
Wine is a great (Free) option. How it works is detailed, but to sum up: it interprets Windows programs to Linux and back.

I'm not sure what the 850MB HP program is, but it'll most likely run under Wine. Another option is Crossover which is not free, but is said to support more windows programs (such as Office 2007, some games, etc.)
