### Post by kramer65 on 2008-08-11
Hello,

The thing I want is not so easy to explain, but I'll give it a go.

When I am editing photo's in the GIMP I often select a detail (an eye for example) make another layer out of it, and the change only that layer. However, when you select a layer in the layers list it always puts a yellow line around it. Since the layer is really small however, this totally limits my view on how the layer looks. 

So what I want is the following:
When I select a layer in the GIMP it should not put a yellow line around it.

Does anybody know how this is possible?

---

### Post by ad_267 on 2008-08-11
Click View
Uncheck "Show Layer Boundary"

---

### Post by kramer65 on 2008-08-11
Allright!!

I've been fighting with this for ages!

Thank you so much!

---

### Post by kramer65 on 2008-08-11
One more thing. It doesn't save this setting when I close the window. How can I make it save this setting?

---

### Post by ad_267 on 2008-08-11
I think if you go to File - Preferences - Image Windows - Appearance and Untick "Show layer boundary" there it will save it.
