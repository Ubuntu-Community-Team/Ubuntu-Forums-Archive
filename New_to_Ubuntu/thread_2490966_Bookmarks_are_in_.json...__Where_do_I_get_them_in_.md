### Post by Holger_Gehrke on 2023-09-21
Firefox can export to html (most - if not all - browsers can). Hamburger-Menu -> Bookmarks -> Manage Bookmarks (or the keyboard shortcut ctrl-shift-o), 'import and save' -> 'Export to HTML' (Options might be named differently, I'm re-translating from German ...). The actual bookmarks are stored inside a sqlite database - 'places.db' - in your firefox profile directory.

Holger
