### Post by Bodsda on 2008-06-09
The easiest way to download embedded videos is to use a FireFox plugin called 'downloadHelper'

[https://addons.mozilla.org/en-US/firefox/addon/3006]("https://addons.mozilla.org/en-US/firefox/addon/3006")

Just install the addon, restart firefox, then whenever theres something you want to download, there will be a small rotating image (looks like 3 balls) to the right hand side of the url bar

Hope this helps

### Post by worldy on 2008-06-09
Well, you can use programs that do the work for you.

But the truth is that once you watch a Youtube video, the .flv file is cached in your Cache directory.
See your 'Cache directory' in the 'Disk Cache Device' by typing about.cache
in the Navigation Bar.

the flash video files have much size. Use this attribute to locate them . many times they them self show up as video files. Clear the cache before visiting Youtube ,if you want to locate the file easily. You should move the video file to other directory and rename it appropriately.

flash movies can be viewed in totem,if you have the right plugins installed or use VLC-the universal media player.

Convert them to other formats if you want.
