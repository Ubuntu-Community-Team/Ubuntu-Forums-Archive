### Post by guiverc on 2023-07-18
I'm not sure what you're asking.

Ubuntu has many releases each year.

The main release we've had this year was [Ubuntu 23.04]("https://fridge.ubuntu.com/2023/04/20/ubuntu-23-04-lunar-lobster-released/"), or the 2023-April release using Ubuntu's *year.month* format for releases.  I'm currently using Ubuntu *mantic* which will be released as Ubuntu 23.10 in 2023-October.

The prior release in 2023 was 22.04.2 (see [https://fridge.ubuntu.com/2023/02/24/ubuntu-22-04-2-lts-released/](https://fridge.ubuntu.com/2023/02/24/ubuntu-22-04-2-lts-released/)) which was the *second respin* of the 2022-April release.

LTS or *long term support* releases occur in April of the *even *year, so the last was Ubuntu 22.04 LTS being the 2022-April release, and the next being Ubuntu 24.04 LTS expected in 2024-April. Each LTS gets a number of *respins*, the [first about 3-4 months]("https://fridge.ubuntu.com/2022/08/12/ubuntu-22-04-1-lts-released/") after release, subsequent to that it's every 6 months.

The first ever Ubuntu release was in 2004-October, thus it was Ubuntu 4.10.

There are also some *snap* only releases, of which there is only one per *even* year, eg. Ubuntu Core 22 which was the release in 2022.

### Post by ActionParsnip on 2023-07-19
Twice a year. One release in April (the 4th month) and one in October (the 10th month). The first part of each release number is the year of release, the second part is the month.
So, Ubuntu 20.04 was released in April 2020.
The LTS releases are every two years on the even numbers and always in April.
So, 18.04, 20.04, 22.04, 24.04 are all LTS and supported for 5 years.
