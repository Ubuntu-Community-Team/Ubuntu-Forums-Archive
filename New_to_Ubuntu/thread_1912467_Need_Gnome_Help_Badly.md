### Post by cmcanulty on 2012-01-21
try these steps in order. Install gnome fallback from synaptic, run update, turn off auto login for just this one time using system settings- users,log out, restart, log back in after using the gear icon choose gnome classic either with or without effects. Then you can reset your login to auto. Once in classic rt cl panels while holding alt to add items or customize. You can have several panels if you like, I have 3. The thing is unless you logout you won't see the classic choice. This only has to be done the first time, it will remember your choice
