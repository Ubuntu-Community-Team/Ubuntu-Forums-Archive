### Post by PinkFloyd102489 on 2008-07-20
Check System > Administration > Printing. There should be an option to share an attached printer over the net. Then go into Printing on the other machine and search for printers on the LAN.

### Post by dutch1918 on 2008-07-20

> Check System > Administration > Printing. There should be an option to share an attached printer over the net. Then go into Printing on the other machine and search for printers on the LAN.

Also make sure that on the computer that has the printer attached. do the following:

system > administration > printing click on Server Settings. Place a check mark on both Show printers shared and Share published printers.
