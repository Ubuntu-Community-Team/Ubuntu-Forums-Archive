### Post by Bluesan on 2011-08-24
> **rawbertb said:**
> REALLY stupid question--how do I tell if I have the 32-bit or 64-bit version of 11.04 installed?

Type at the terminal:

```
uname -a
```

If you see a "64" in the output, you've got a 64-bit install. If not, you don't.
