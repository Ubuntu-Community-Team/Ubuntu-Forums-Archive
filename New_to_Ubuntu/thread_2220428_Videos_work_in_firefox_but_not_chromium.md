### Post by deadflowr on 2014-04-27
From what I know, the chromium project have made using the normal flash-plugin unusable for chrome/chromium.
To get flash to work on chromuim, you need to install the pepperflash-plugin package.
It should be listed in the software center. It will be something like pepperflash-plugin, or pepperflash-pluginnonfree.
What this does is downloads google-chrome, but only extracts and configures the pepperflash-plugin, so that chromium can use it.

Hope that helps.
