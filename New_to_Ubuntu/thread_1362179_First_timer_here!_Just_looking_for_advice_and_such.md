### Post by SchizmWolf on 2009-12-23
Welcome! I was browsing around, thought I could offer my advice (what there is of it.)

There are a lot of really fun Linux games out there, as one of the other posters linked above. If you're into FPS and such and really, *really *need to have, say, Halo, then I'd recommend visiting [http://www.winehq.org]("http://www.winehq.org/") and downloading the version appropriate for your distro by following the instructions on the page. Wine is a program that allows you to run Windows apps on Linux. After that, update your system (Terminal ---> sudo apt-get update) and when that's done, execute "sudo apt-get install playonlinux" . This installs a front-end (called, appropriately, PlayOnLinux) for Wine that's specially suited for windows games. Works great for my Half-Life 2, without which I would be a very, very sad penguin.

That's all the advice I have to offer at the moment, but if I think of more that hasn't already been suggested, I'll let you know :)
