### Post by mali2297 on 2011-07-14
You can find filelists for all Ubuntu packages at [http://packages.ubuntu.com](http://packages.ubuntu.com).

In this case, see [http://packages.ubuntu.com/natty/i386/gfortran/filelist](http://packages.ubuntu.com/natty/i386/gfortran/filelist) or [http://packages.ubuntu.com/natty/amd64/gfortran/filelist](http://packages.ubuntu.com/natty/amd64/gfortran/filelist) (depending on architecture).

EDIT:
Perhaps you want to look at the file list for [gfortran-4.5](http://packages.ubuntu.com/natty/gfortran-4.5) too.
