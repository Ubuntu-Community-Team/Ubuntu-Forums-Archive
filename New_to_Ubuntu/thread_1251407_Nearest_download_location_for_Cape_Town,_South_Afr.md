### Post by drs305 on 2009-08-27
More likely it would just be slow but a slow connection exposes you for a longer period, of course. You should be more interested in transfer rates. If you are using Synaptic or Software Sources, go to Settings, Repositories (in Synaptic) Ubuntu Software tab. In the Download From section, select Other, then Choose Best Server. A test will be run checking the mirrors and a recommendation made. Be advised though that the 'best server' would change over time so you would want to occasionally run the test again or chose a mirror which you find consistentely provides acceptable results.
