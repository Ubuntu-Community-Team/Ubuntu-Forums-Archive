### Post by arochester on 2009-08-25
Install the Medibuntu Repository.

Instructions here: [https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)

Note that it is a 2 part installation 1) Install Repository 2) Get GPG key

Install the decoder: libdvdcss2
