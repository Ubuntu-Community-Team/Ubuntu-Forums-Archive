### Post by MidnightGrey on 2013-06-19
This is the procedure to upgrade Ubuntu by DVD. I can only assume it is simliar in Kubuntu.
```

1. Boot into your 13.04 disk
2. Select Install Ubuntu
Continue through the menus until you hit "Installation Type" and it asks you what you would like to do.
3. Select 'Upgrade Ubuntu 12.04 to Ubuntu 13.10
4. Select 'Install Now'

```

If your Kubuntu CD does not show an upgrade option then cancel the installation process.
