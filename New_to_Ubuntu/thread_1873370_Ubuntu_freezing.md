### Post by little_penguin on 2011-11-01
I had the same problem trying to run the standard desktop version of Ubuntu 11.04 or 11.10 on my netbook (Samsung N140). I think the specs are just too low to handle it, so I installed Ubuntu Netbook Remix, which is more geared to these devices. I don't see freezing now and if it does happen, it's usually due to overheating. Perhaps you could try installing the netbook edition.
