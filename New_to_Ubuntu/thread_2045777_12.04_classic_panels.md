### Post by Austin Texas on 2012-08-22
Unfortunately, there is no clear guide to the gtk.css files.
I have invested dozens of hours just in the last couple of weeks trying to track down all the necessary changes that I want, with somewhat incomplete success.
A couple of tips:
Start with the settings.ini file if there is one. It controls a lot.
The menu titles you are looking for would be "toolbars" and "menu item"
Any changes you make in  gtk-3.0/gdk.css have to be duplicated in the gtk-2.0 rc file, because some programs, like synaptic, don't use the gtk-3 settings.
