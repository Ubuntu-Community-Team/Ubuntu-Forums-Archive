### Post by Cheesemill on 2012-08-10
You need to use an actual CD or USB drive.

Daemon tools will only run after Windows has loaded, but booting the Live CD has to be done before this happens, once Windows starts loading you are to late in the boot sequence to be able to boot the Live CD.
