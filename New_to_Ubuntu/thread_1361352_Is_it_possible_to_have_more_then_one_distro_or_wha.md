### Post by perlluver on 2009-12-21
You can install Kubuntu alongside Ubuntu.  Namely you would want to install kubuntu-desktop.  Then when you log on, you can switch between Gnome and KDE.  You can find kubuntu-desktop in System>Administration>Synaptic Package Manager.  There are quite a few files, so it might take a little while to download them all.

Using the method perlluver, you can have Gnome & KDE Desktop Environments installed.
You can also install Xubuntu, the XFCE DE.

Note that you need to boot into them, by choosing which DE you want at log-in.

Also, the menus can get quite cluttered, with GTK, KDE and XFCE apps all being installed. But almost all apps will run in each other's DE.

I always install both Gnome & KDE DE's.

I have also installed Gnome & KDE separately, each in it's own folder system.

You can also run them in Virtual Machines, as others have stated.

Each method has it's advantages and dis-advantages.

Pick your poison, or choose several.

Choice... it's The Unix Way.
