### Post by Jon Monreal on 2010-05-01
As far as I'm aware, moving windows between workspaces like that is only available when visual effects are enabled. To (try to) enable them, go to System>Preferences>Appearance, go to the "Visual Effects" tab and select "Normal". You will need a sufficient video card and drivers for this to work.

As far as the problem with pinning windows, try right-clicking in them in the window list and selecting "Always on Visible Workspace".

See if your "Desk 1" problem fixes itself with this. If not, we can try to see what the problem is.
