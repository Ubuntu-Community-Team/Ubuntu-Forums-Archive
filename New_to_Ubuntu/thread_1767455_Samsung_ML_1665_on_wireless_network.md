### Post by ICEhockeyGOALTENDER on 2011-05-25
When you got your printer did it come with an install disk?

-If so then put the install disk in the computer(s) running Ubuntu v10.10 (I would recommended updating to v11.04) then if there is an option for "Linux" system then choose that and proceed with the steps it tells you 

-If not then click [HERE]("http://www.samsung.com/ie/consumer/print-solutions/print-solutions/mono-printers/ML-1665/XEU/index.idx?pagetype=prd_detail&tab=support") to go to the driver page. Under the downloads tab in the somewhat middle of you screen click drivers then find "Linux" and the most updated one(v3.00.65) all the way to the side of that row click on the icon int he last column defined "File". After that follow the steps that is given(click "Run" or whatever it says). 
-If this doesn't work please write back on this thread because i would like to help you to my possibilities. If I am completely off on what you are saying then could explain further so I could help you more?
