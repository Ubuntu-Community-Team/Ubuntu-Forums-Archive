### Post by dfarrell07 on 2011-09-14
First, to make sure you have Flash installed properly, search for 'Adobe Flash Plugin' in the Ubuntu Software Center. Install that. Sound should then be working for YouTube videos. If not, the first thing I would try is restarting your browser, then your computer if needed.

To test for sound, you can used the System Testing program provided by default. Search for it using the windows-key search thing. One of the first tests is for sound, you can just skip the others.
