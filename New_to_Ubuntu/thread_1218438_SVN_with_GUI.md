A bit of advice for the OP: when you want a new bit of software, it's a good idea to search the repos for it before you start downloading packages from websites, trying to compile source, etc etc.

The easiest way to search the repos is to use the Search function in **Synaptic** - **System > Administration > Synaptic Package Manager**.

Installing software through Synaptic, or by using the **apt-get** program like zerhacke suggested above, is a whole lot easier than downloading the package from somewhere then fooling around with source code n stuff.  Software repositories is the Ubuntu Way!!  Dig it!!
