### Post by j.bell730 on 2009-08-28
Yes, programming can be done with it.
Give this a try...
Open gedit and paste this in:
```
#!/bin/bash
echo Hello, world!
```
Then, save it as something.sh
Open a terminal (**Applications** > **Accessories** > **Terminal**), and type this:
```
sh something.sh
```
Replace "*something*" with the name of your file.

Congratulations, you've just programmed a bash script.
