### Post by Mark Phelps on 2012-03-29
Wubi is mean primarily as a way to try out Ubuntu -- with minimal changes to your existing environment.

If you are going to be using Ubuntu for the long haul, you need to read through the Guide about migrating it to its own partition -- and that can NOT be an existing NTFS partition.
