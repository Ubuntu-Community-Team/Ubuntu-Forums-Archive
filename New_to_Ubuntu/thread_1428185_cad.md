### Post by lkraemer on 2010-03-13
While lots of M$ Software runs in WINE, the only way to know for sure
is for you to try it.  Check the Application database on their website.
You can also try Crossover by Codeweavers as it is a somewhat advanced
version of WINE.  If you install in WINE, or Crossover, you can set the
Version of M$ Windows you want to try and emulate.  And you can always
install VirtualBox, then install XP or Win98 in VirtualBox.  That way you can
install your software in Windows.  Plus if you install Win98 in Virtualbox
read the FAQ and download the Video Drivers and install so the Video works
correctly.  I'd bet it will work in XP, or XP with Win98 Emulation enabled.

If you want to try VirtualBox you'll need at least 2 Gig of RAM.
~7xx Meg for Windows and the remainder for Video and Ubuntu.
