### Post by greyfox1 on 2010-10-20
Hi Lephosto, welcome to GNU/Linux! I hope you are enjoying your experience so far. Here are a few answers that I can provide:

1. You can start by checking out [http://help.ubuntu.com](http://help.ubuntu.com) (the official Ubuntu help site) and [http://wiki.ubuntu.com](http://wiki.ubuntu.com) (community documentation). Neither is a one-stop shop for everything you'll need, but they will help.

2. Everyone has a slightly different answer to this question, but [here is one guide]("http://blog.thesilentnumber.me/2009/09/top-things-to-do-after-installing.html") I think is very comprehensive and well done. It was written for 10.04 but most or all of it still applies.

3. [Here you go]("http://fosswire.com/post/2007/08/unixlinux-command-cheat-sheet/"). This cheat sheet was written by Jacob, who is a mod on these boards.

4. Officially, no. Supposedly this will be supported in the future. I have read about some experimental ways to make it work, but trying it runs the risk of ruining your Windows installation. I wouldn't try it unless you are prepared for that contingency.

5. I'll leave this one for someone else.

Good luck, and welcome to the forums!

### Post by Ichtyandr on 2010-10-20
1. You can download Ubuntu manual [http://ubuntu-manual.org/](http://ubuntu-manual.org/)
Also use google and search button at the forums, I found with time that all questions I could ever think of were already answered here.

2. You pretty much get a working system on default install. Typical recommendation is to install ubuntu-restricted-extras package to get flash and mp3, but afaik the 10.10 version installs restricted stuff during installation. Consider [http://www.osalt.com/](http://www.osalt.com/) to see typical alternatives to programs you use at Windows

3. Internet is full of CLI commands. Check out [http://linuxcommand.org/](http://linuxcommand.org/) for beginners, I started there. Later you will learn to use man command to bring out manuals and find your way through

4. You can run windows program through program called wine, that you have to install. Lots of things are hit and miss but lots of programs work. Check out the list at [http://www.winehq.org/](http://www.winehq.org/). Also you can user virtualbox or vmware or other virtualization programs to run windows within ubuntu. Moving existing windows installation can be a pain but again google is your friend, I never tried this but look here perhaps [http://www.virtualbox.org/wiki/Migrate_Windows](http://www.virtualbox.org/wiki/Migrate_Windows)

Also you probably know that good preliminary backup can always save you lots of time and stress

good luck
