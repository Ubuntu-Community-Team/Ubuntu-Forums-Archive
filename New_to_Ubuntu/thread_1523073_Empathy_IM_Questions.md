Yes, I use pidgin as my default IM client. Under the options, in pidgin, is where you can choose to always have a system tray icon or not and also what it logs. Like with every program I'd read over all the inner menus because programs often have a lot of cool little things "hidden" in plain sight.

You can uninstall Empathy if you'd like also.

As far as the icons go you'll probably have to change the icons where they're stored, which can be a hassle.
