### Post by Brandon Williams on 2009-10-30
File name matching may be enough, but you're better off verifying the md5sum of the file. If you look [here](https://help.ubuntu.com/community/BurningIsoHowto), in the section entitled "Verifying disk Integrity", there is information about checking the md5sum of an official ISO file. Using this method, you can be certain that you downloaded the official release ISO, regardless of whether or not the filename matches.
