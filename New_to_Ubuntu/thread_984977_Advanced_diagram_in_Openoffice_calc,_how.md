### Post by vanadium on 2008-11-17
In a spreadsheet, you will need to calculate data points for the fitted curve yourself using the equation of the line. Them, you can plot these values as an additional data series, and format it as a curve without data point markers. Just calculate datapoints within the entire range you want to see plotted (up to "5" in your case).
