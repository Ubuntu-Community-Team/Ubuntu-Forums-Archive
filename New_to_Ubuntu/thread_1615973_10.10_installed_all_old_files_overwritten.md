### Post by sikander3786 on 2010-11-07
First of all stop using the system. It would be better to run a Live CD at this point so your files (if any of them are still actually present) don't get over-written with something else. The files are there on your HDD unless you write something new on the same sectors.

You can see this page for a workaround.

[https://help.ubuntu.com/community/DataRecovery](https://help.ubuntu.com/community/DataRecovery)

Testdisk is the best option in my opinion.

[http://www.cgsecurity.org/wiki/TestDisk](http://www.cgsecurity.org/wiki/TestDisk)

Try with testdisk or some other data recovery software and hope you are able to recover them. If you can't, and if the files are that important, you can send your HDD for forensic recovery.

But don't use your 10.10 install at all. Use a Live CD/Live USB instead as already mentioned.
