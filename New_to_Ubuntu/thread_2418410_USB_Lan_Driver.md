### Post by Holger_Gehrke on 2019-05-06
Ubuntu (or rather the Linux kernel) comes with a lot of drivers included. Download the ISO, make a bootable medium (USB-Stick or DVD) from it, boot from it and select 'Try Ubuntu' from the menu. This will run Ubuntu from the Stick (or DVD) without installing it to hard disk. If the USB NIC works in this trial system, it will work in an installed system too.

A quick search for "Ubuntu USB LAN" shows that a lot of these adapters work out of the box. Without knowing what exact model you have, we can't tell you whether or not yours will work.

Holger
