### Post by Kris_Spencer on 2014-04-02
You don't really *have *to upgrade, but it's recommended. The updates give you security updates that fix security weaknesses, which are nice to get. But, it's not unheard of for people to be running Ubuntu passed its life cycle. If you don't want to upgrade the system on the six month cycle, you may want to use the LTS versions (Long Term Support). They release every two years; 14.04 will be LTS and is releasing *very* soon. 12.04 was the last LTS and is supported until April 2017.

### Post by oldrocker99 on 2014-04-02
BTW, if you do a fresh install, select "Something Else" at the disk page and specify ~128 GB for the / partition, and everything else for a separate /home partition. This makes upgrades and reinstalls a lot simpler, since your /home partition (and its contents) just needs to be mounted as /home, and ~128GB is LOADS of space for a / partition.
