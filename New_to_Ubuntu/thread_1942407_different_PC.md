### Post by IWantFroyo on 2012-03-17
They're different processor architectures.

64 bit: amd64, x86_64

32 bit: i386, x64

64 bit processors can do some tasks faster, but for this functionality you need an amd64 or x86_64 image. They can also run i386 and x64 images with no problem. They can also utilize over 2GB of your RAM.

32 bit processors can only run 32 bit architectures, and won't work with a 64 bit image. Also, they can only use 2GB of your RAM efficiently.
