### Post by Tristan_Williams on 2014-03-02
There are a few ways to have widgets.

1 - Cairo Dock. You can configure it without the panel, and put whatever you want on the desktop.
2 - Conky. This basically just gives you a system monitor. With [Conky Manager]("http://www.webupd8.org/2013/07/conky-manager-gui-for-managing-conky.html"), you can make it look REALLY good.

Cairo Dock is probably what you are looking for.

To install it, run the following commands:
```
[COLOR=#000000][FONT=UbuntuBeta Mono]
sudo apt-get install cairo-dock

```

Start Cairo Dock, and right click the panel. Under "Cairo Dock -> Configure" you will find all the options you need.[/FONT][/COLOR]
