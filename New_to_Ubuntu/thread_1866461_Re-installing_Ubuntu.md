### Post by coffeecat on 2011-10-21
> **dcrobinson said:**
> I can't even tell you what version of Ubuntu I am running.

Open a terminal and run this command:

```
cat /etc/lsb-release
```

> **dcrobinson said:**
> Note, I am having to use a Windows-based laptop to download the iso image, since I can't get Firefox to start

You can use Windows to burn the ISO. Have a look here:

[https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)

Windows 7 has ISO burning functionality built in, and if you are using an earlier version of Windows, the free application Infra Recorder works very well.
