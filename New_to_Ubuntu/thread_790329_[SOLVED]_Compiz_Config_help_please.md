### Post by Joeb454 on 2008-05-11
You need to venture into the "General" area of the CCSM.

From there look for Desktop Size, or Horizontal Desktop size. Now set that to 4 :)

Voila - you have a cube (hopefully ;)) I quite like changing this to values between 5 and 8 - just so it isn't a cube :p
