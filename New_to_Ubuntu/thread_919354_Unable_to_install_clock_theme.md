### Post by farsight on 2008-09-14
Hey hey

I have had the same problem before. The way I've got mine to work is to follow this:

System>Administration>Synaptics Package Manager

Click the search button and type in "screenlets"

Mark the screenlets package for installation and then click apply.

Following installation click: System>Preferences>Screenlets

On the right is an install button. Use this to navigate to where you saved the screenlet and then install. 

Hope this helps you, good luck buddy,

Regards Farsight
