Exactly how did you install Gnome? What packages?  From the command line or from Synaptic or Software Center?

Correctly installed, a new desktop environment like Gnome can be selected at login.  

If you have opted to log in automatically, you won't see a login prompt at reboot.  You need to manually logout and log back in to select a new environment.  Once done, the new DE will stay the default until you select another.
