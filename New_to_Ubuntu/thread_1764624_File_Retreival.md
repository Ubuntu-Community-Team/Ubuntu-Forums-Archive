### Post by Hedgehog1 on 2011-05-22
Assuming that the windows partitions are not damaged, you can usually use the 'Places' menu and open the partitions in the GUI (called nautilus).

You would normally connect a USB stick or external drive to the system and boot from the LiveCD/LiveUSB.

Then use nautilus to copy data from the Windows partitions to the USB Stick or external drive.

You can also visit the this forum using the LiveCD/LiveUSB and ask further questions if you get stuck.


***The Hedge***

:KS

p.s. If the windows partitions are damaged, there are a few tools on the LiveCD/LiveUSB to attempt repairs so you can get your data off the partitions.
