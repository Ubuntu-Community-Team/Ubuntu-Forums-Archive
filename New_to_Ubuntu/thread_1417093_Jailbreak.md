Now, back to your question.  Though Ubuntu strives to be open source, and tries to use open source drivers whenever possible, sometimes this cannot be avoided, as sometimes, the proprietary drivers are the best option rather than the open source one. It's a simple choice really. 

Since the latest jailbreaking apps are developed for Windows and Mac, you should simply go with that as a last resort if you cannot make any apps to work at all.

You could try Wine to get these apps to work at ubuntu, or try a virtualization software like VirtualBox (which would need some extra space in your harddrive 16GB for Win 7, and a key if you wanna install legally). 

Getting an app to work with Wine may need some tweaking and patience, and installing Windows (or OSX for that matter) in a virtual environment will need some patience and lots of hard drive space.

If you are already dual booting, why not just boot into OSX or Windows and use the jailbreaking app like redsn0w? Everyone has computers these days and if you don't have a machine with Windows or OS you can simply borrow a friend's computer instead. Less hassle and time.
