### Post by zeroseven0183 on 2009-08-12
Welcome to Ubuntu and to the Forums!

After downloading the .IMG file from [http://www.ubuntu.com/getubuntu/download-netbook](http://www.ubuntu.com/getubuntu/download-netbook), you will then need to transfer it using an application to your flash drive. It's better if you use first the Windows application, Win32 Disk Imager. The instruction is found here [https://help.ubuntu.com/community/Installation/FromImgFiles](https://help.ubuntu.com/community/Installation/FromImgFiles)

Although, the laptop model you have is not specifically stated in [https://wiki.ubuntu.com/HardwareSupport/Machines/Netbooks](https://wiki.ubuntu.com/HardwareSupport/Machines/Netbooks), it's worth the try.

Have fun!
