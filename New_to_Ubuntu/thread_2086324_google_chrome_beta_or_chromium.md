### Post by I2k4 on 2012-11-21
The main noticeable difference is Chrome includes Flash while Chromium requires a separate install. FWIW Chromium seems a little quicker. 

I run them side by side in Lubuntu with Chromium default because
a) there's less personal tracking in Chromium, and I want to minimize personal disclosure to Google, and 
b) Google seems to configure its own web services e.g. Docs, iGoogle, Maps, etc. to run better under Chrome, and since they're tracking anyway the privacy is already gone.  So I "silo"  Google services in Chrome, otherwise use Chromium.

I'm very pleased that both browsers are still current for my Lubuntu 10.10 install. I'm not going to upgrade the OS so it's great to have up to date browsers.
