### Post by coffeecat on 2014-01-24
I have a HP Mini 110, although it may not be identical to yours since there are many variants. It has a Broadcom BCM4313 wireless device which works out of the box with the default open source driver in all the versions of Ubuntu I have tried it with. However, there are several different Broadcom wireless devices, some of which work with the open source driver and for some you have to install one of the non-free drivers. Here is a useful wiki page:

[https://help.ubuntu.com/community/WifiDocs/Driver/bcm43xx](https://help.ubuntu.com/community/WifiDocs/Driver/bcm43xx)

The best thing would be to run Ubuntu from a live USB and see if the wireless works and, if not, use the instructions in that page to identify the wireless chip.

As far as "too much for it" - no. I've had several versions of Unity running on it, although I haven't tried 13.10 yet. You'll see some people saying that the Unity desktop is too heavy for a machine like the HP Mini, but I think they're being a tad pessimistic. In my experience it was a little slow but usable, and that was with only 1 GB RAM. Nevertheless, you might want to consider a lighter variant such as Xubuntu with Xfce or Lubuntu with Lxde.
