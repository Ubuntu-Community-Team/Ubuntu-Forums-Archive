### Post by Elfy on 2008-08-22
BIOS should be on mobo - so that should be ok - if you boot pc now without harddrive - esc, delete or maybe an Fn key combination will get you to setup - that is BIOS (more or less)

Before you get ubuntu make sure you will have sufficient memory for it, at lest 384Mb or you might need xubuntu instead.
