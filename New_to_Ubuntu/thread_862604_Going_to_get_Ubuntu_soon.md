### Post by seagullplayer77 on 2008-07-17
As for partitioning, the installer has a partition tool, although I would very strongly recommend using the Windows tool to do it. When I first installed Ubuntu, I made a mistake with the partitioner in the installer (it isn't the most intuitive thing ever), and I ended up wiping my hard drive. If you make the partition in Windows, it'll show up as free space in the partitioner and you can select it and not have to worry about royally messing something up.

Perhaps I'm not the wisest person on earth when it comes to this, but I don't have any security software installed on my Ubuntu machine--absolutely none. People don't really write viruses or worms for Linux because not that many people (relatively speaking, anyway) use it, so that's not a big deal. I imagine the same thing goes for spyware and adware and all that other nasty stuff. If you want to get some security software anyway, there's a tool called Synaptic that comes with Ubuntu that has a huge list of software packages you can install. You can just do a search for "firewall" or "anti-virus" in Synaptic and see what shows up. Good luck!

### Post by damis648 on 2008-07-17
Welcome to the Linux and Ubuntu world! For starters, the permissions system is very secure so even if you do get a virus, which would be EXTREMELY rare, it cannot really affect you (without a password, which would be obvious if a random app was prompting for your password). So, antivirus is quite unneeded. As for firewalling, you do not really need one. If you do not feel secure without one, or just relying on your router, install "Firestarter" after you install ubuntu and that will take care of it. When installing, it is really quite easy. Just follow the prompts. When it asks how to partition the disk, just drag the separator on the top to align the partitions to whatever sizes you would like.

Hope you have fun, and remember, if you have any problems, jupt post here. Good luck!
