### Post by Locke_99GS on 2010-11-15
Bluefish should install into the Applications/Programming menu. If it hasn't, use *alacarte* and make sure the Programming submenu is checked to display.

Failing that, the icon on the desktop or on a panel is called a "launcher". Right-click on the desktop or panel and choose "Create Launcher...", then just fill in the information to create a launcher for Blufish.
