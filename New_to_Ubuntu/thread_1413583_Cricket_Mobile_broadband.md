### Post by Gumbyy on 2010-03-01
> **wood144 said:**
> I recently install unbuntu 9.04 on an old compaq presario v5000 and was thinking about getting a mobile broadband card from cricket to use with it.  does anyone have any experience with this? does it work well? and could you recommend a model number of a card that will work out of the box?


   Yes it does work.I'm not too sure with 9.04,however I run linux 9.10 and I got the cricket broadband UM185C modem (wireless) and it runs just a bit under 1mbps.It works ok.Not the best Internet connection,but for the price it works good.If you were to watch youtube vids it takes a few moments to load it sometimes,but as I said not bad for price.

   Now in my case when I got mine to work I installed it on windows xp first activate it,then put it on Linux shortly after.I'll I did was just plugged it in first time worked liked a charm.I did end up unplugging it though,then when I went to plug it back in had a error message saying some stuff about it mounting wrong.That didn't make a  difference though at all.

   All you have to do to get it to work is plug it in,then go to Places - Computer - ( then it shows a picture of a CD ) that says Cricket underneath it.Just right click on it then go down to where it says eject and hit that.That is all I did to get my mine to work.I never once had to put any codes or anything in to get it to run.
 
  It never once has dropped out on me either so far.I had Verizon USB 760 and I would get dropped signals on it ALL the time.Every like 5 mins pissed me off so bad..Maybe I did something wrong,but whatever.Ran alot faster but cost way to much to keep it..So I just bagged it.

  So for your just regular Internet web browsing,checking emails stuff like that runs way good.If you go into IRC it tends to lag a lot more.I would get around 550 ping in IRC channels.Sometimes jumps to 1000.Still would get the messages though.All and all though I like it that is why I still use it today,cause its only 35 bucks a month for wireless Internet which isn't bad for what you get.

  Hope that helped you out.

O btw I got a Asus Eee PC and it runs Linux great.Even runs Compiz on it.
