### Post by newb85 on 2014-01-06
Welcome to Ubuntu and the forums!  &#12393;&#12358;&#12382;&#20837;&#12387;&#12390;&#65281;

Ubuntu has a great Romaji input method for Hiragana, Katakana, and Kanji.  (Actually, my Kanji knowledge is quite limited, so I don't know how great it is, but I know it works.)

First, open Language Support (I think you can search for it using Cinnamon's start-menu-equivalent) and make sure the Japanese language is installed.  Also, make sure the Keyboard Input Method system is set to IBus.
Then, open IBus (a.k.a. Keyboard Input Methods).  On the Input Method tab, check Customize active input methods.  Under Select an input method, choose Japanese>Anthy and hit Add.  Under the General tab, you can set up keyboard shortcuts to switch Anthy on and off.  IBus has an AppIndicator that provides a menu for mouse-controlled switching, too, but I don't know how it will work in Cinnamon.
