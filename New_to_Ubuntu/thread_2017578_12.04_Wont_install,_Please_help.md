### Post by MIJ-VI on 2012-07-05
OP, how are you trying to install Ubuntu 12.04 LTS, via a DVD you burnt from a downloaded .iso?

If so, did you check the .iso's md5sum before burning it to a DVD-R or a DVD+R? (DVD+R discs which worked fine for me under Ubuntu 10.04.x & 10.10 for creating Ubuntu install DVDs wouldn't boot under 12.04. However DVD-R discs work fine.)

BTW. I've found that burning DVDs at no more than 8x speed works best for creating an OS boot/install DVD.
