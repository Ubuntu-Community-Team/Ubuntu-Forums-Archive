### Post by r-senior on 2011-05-09
If a page had multiple images, how did you choose which one was saved?

You can save images using a shortcut key in Firefox, as long as the image is all that is shown in the window. For example, pick an avatar off a post, right click and "View Image". The image is all that's shown and you can press Ctrl-S to save it.

The only thing about this is that it's quicker just to right click aforesaid image and choose "Save Image As" from the menu that appears.
