### Post by grahammechanical on 2011-03-14
I am not sure what you want to do. Do you want to click on an icon on the desktop and have a file or document open in the program that the file/document was created in?

Go to the folder that the file/document is in. Select the document and right click and from the menu select Make Link. A link or icon will appear. Cut and paste that icon to the desktop. When you click this icon the document will load in the program that would normally load that document. If you think that the icon is too large you can re-size it with a right click.

If this is not what you want then I am sorry. I do not understand.
