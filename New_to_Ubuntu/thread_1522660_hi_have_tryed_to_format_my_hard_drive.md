The correct question would be:

Are you trying to "format the hard drive" from an installation on the same hard drive, or from a Live CD?  You should be able to format a hard drive from an installation, as long as you are either formatting a secondary hard drive (one that the installation isn't installed on) or on the same hard drive, as long as you're not trying to perform operations on a mounted partition.

A good starting point would be to read the information at the link in my signature block, "How To Partition", but certainly we need more information.
