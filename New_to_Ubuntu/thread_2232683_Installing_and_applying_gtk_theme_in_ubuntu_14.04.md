### Post by deadflowr on 2014-07-03
You should have a .themes folder in your home folder.
(It's hidden so you need to enable show hidden files/folders, in the archive manager it should be right-click to enable, in the file manager ctrl + h)
Put it there, first.
That way you can test it for the local user.
If after testing you want it available for all users, copy it into the system folder
/usr/share/themes/

If you don't seem to have a .themes folder in your home folder, make one.
Just make sure to name it with a dot in front, to make it hidden.
