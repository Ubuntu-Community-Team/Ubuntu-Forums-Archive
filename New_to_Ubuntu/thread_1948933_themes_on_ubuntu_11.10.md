### Post by solocommand on 2012-03-29
I had a horrible time getting gnome shell extensions installed. If it is installed correctly, you should be able to select the (None) [Folder Icon] button next to the theme drop down to select the theme package you want to install. If you can't click on this button (greyed out, warning icon), gnome-shell-extension-user-theme is not installed correctly.
