### Post by jbrown96 on 2009-04-14
> **thesuperjman said:**
> if i save an openoffice.org document on my usb memory stick, will i be able to open it at school (which runs windows xp)

I imagine that they have Office installed, so when you save the oo.org document change the type to MS office. I recommend the Office 97/2000/XP .doc format. If you just need to view/print it, you can also just export it as a PDF, which will ensure that no formatting gets messed up. 

There also a way in Tools--->Options--->Load/Save--->General to set it to save to this format by default.
