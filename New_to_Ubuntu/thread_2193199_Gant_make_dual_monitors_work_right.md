### Post by bug67 on 2014-03-04
Sorry I took so long getting back to this.  I've been using the Nouveau drivers all this time because I could not for the life of me get my dual monitors to work with the Nvidia propretory drivers.  The solution I found yesterday (after 3 months) was stupidly simple!  The problem I was having with the Nouveau drivers is that, the system would not wake from sleep.  x was crashing on wake.  I had to set it up so that the machine wouldn't go to sleep.

Turns out, after I install the proprietory drivers and set up the monitors how I want them, *I also need to go to "Displays" in system settings and ensure my monitors are set up identically to the twinview setting in Nvidia settings.*  Bam!  Now, not only am I running Nvidia's proprietory drivers but, the latest ones as well.  I'm so happy with my system, I could vomit!  :P

I hope this helps someone else out who might be having similar issues.
