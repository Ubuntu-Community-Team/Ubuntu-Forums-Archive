### Post by kansasnoob on 2008-09-10
Try gthumb for digital pictures. I like it.

```
sudo apt-get install gthumb
```

It'll show up in Graphics.

As far as video playback and editing vlc brings in good support for quicktime.

I go to synaptic and install:

vlc
vlc-nox
AND ALL vlc-plugin- EXCEPT vlc-plugin-alsa (I'm on pulse audio!)

You may need to run:

```
sudo apt-get -f install
```

To meet unmet dependencies.
