### Post by Cheesemill on 2013-04-02
It's not possible.

The CH3HNAS uses a ARM controller chip that Ubuntu stopped supporting several years ago (even then it may not have worked properly).

You ***may*** be able to install the ARM port of Debian on it, but it won't be easy even if it is possible.
