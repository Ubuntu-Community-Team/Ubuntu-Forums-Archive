### Post by Jerry N on 2009-09-27
Freespire is based on Ubuntu 7.04 with so-called "non free" items added and using KDE.  It is way out of date and as far as I can tell is not supported.  Add the "restricted extras" to Ubuntu (or Kubuntu) and you get basically the same thing.  Freespire was easier to used on a mixed Linux/Windows XP LAN than Ubuntu 7.04 but Ubuntu has marched on.  I played with Freespire a few years back and I would consider a current version of Ubuntu, with the "restricted extras", or LinuxMint to be much better choices.

Jerry

### Post by Paqman on 2009-09-27
Is Freespire even still active? According to their website their last release was based on Ubuntu 7.04. That's two and a half years ago.

Even an older version of Ubuntu like 8.04 will have a lot of features over it. Examples:

[LIST]
[*]Compiz Fusion desktop effects
[*]Full NTFS support
[*]Fast user switching
[*]Print to PDF
[*]Desktop search
[*]The Wubi installer
[*]Tons of updated packages
[/LIST]

And that's on 8.04, which is pretty old now. Install a recent version like 9.04 (or 9.10 when it's released next month) and you'll get a lot more on top of that.
