### Post by Rocket2DMn on 2008-06-07
You should try using EnvyNG to install the restricted drivers for you, the 173 drivers may not be compatible with your card.
[http://www.albertomilone.com/envyngfaq.html#A](http://www.albertomilone.com/envyngfaq.html#A)

You can remove the drivers you install manually with something like
```
sudo ./name_of_installer.bin  --uninstall
```
Then have X reconfigure to a generic xorg.conf file with
```
sudo dpkg-reconfigure xserver-xorg -phigh
```
You should reboot before installing the driver with EnvyNG.
