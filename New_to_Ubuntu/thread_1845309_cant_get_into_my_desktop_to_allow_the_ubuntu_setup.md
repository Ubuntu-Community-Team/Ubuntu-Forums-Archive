### Post by Rex Bouwense on 2011-09-16
Is there anything on the laptop such as documents that you need?  If the answer is no then you probably have an easy fix.  You will need a bootable CD or flash drive with Ubuntu on it.
How do you get that?  If you have access to another computer download the desktop ISO.  Check the hash (md5sum).  Burn the image (not copy) it to a CD or flash drive.  Place it in your laptop and set the BIOS to boot fist from the CD or flash drive before the hard drive.  That will give you the entry to Ubuntu to try (to make sure that it works on your machine) or install it.  Sit back and wait 20-30 minutes and you have Ubuntu.
