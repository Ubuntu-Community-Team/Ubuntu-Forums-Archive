### Post by Old_Grey_Wolf on 2014-02-14
You need the GPG key> Enter these commands in a terminal.

```
gpg --keyserver keyserver.ubuntu.com --recv A5D32F012649A5A9
gpg --export --armor A5D32F012649A5A9 | sudo apt-key add -
```
