### Post by yragrelluf on 2008-06-18
Make sure it is plugged in when you install from a liveCD and use the manual install feature and create an ext3 drive and the swap space. When you boot, you should change the boot order in the systems bios by pressing f9. It may be a different key on your computer though. Change it to boot from usb hard drive, and then you can run ubuntu like it's 2099!:lolflag:
