You used to be able to do this with a gui, system>prefs>sound. You cant as yet. Regression methinks. So.

Hint. /usr/share/sounds/ubuntu/stereo

Not difficult mod, just backup the default first then rename your sound to the default.

```
gksu nautilus
```
Then browse to the above folder. Careful with the root browser. ;)
