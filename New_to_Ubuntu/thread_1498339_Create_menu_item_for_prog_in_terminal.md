### Post by raysa on 2010-05-31
I have a program which runs in terminal - no graphical front end. I'd like to create a menu item so that I can start it like any other application without first opening a terminal and having to remember/type the executable name.

What would I need to put before the executable name in the 'command' field of the "New item" command in the "Main menu" facility, so that clicking on the menu item would open a terminal, load the executable and run it?

Thanks, Ray

---

### Post by arrange on 2010-05-31
In the *launcher* dialogue navigate to **Type** and choose **Application in Terminal**.
