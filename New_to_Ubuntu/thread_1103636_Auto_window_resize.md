### Post by mgmiller on 2009-06-25
When it's in full screen mode, what happens if you just move your mouse up to the top where the console should be.  It should slide down as if appearing from "auto hide mode".  Once you have the display as a full size window where the top and bottom panels are visible, try resizing the firefox window by grabbing the bottom right corner and making it smaller.  It also sounds like you have the window adjusted so that when it's in resizable mode, it's the same size as a maximized window.

A quick way to tell is if the button next to the "X" to close the window says "Maximize Window" when you mouse over it, and the window is already maximized, then you need to resize it by grabbing a corner.

When it's working correctly, there are 3 different size windows available.
1) A less then maximized window that you can resize to your preference.
2) A maximized window that fill all available space, but leaves the top and bottom panels visible.
3) A full screen mode that fills your monitor with the contents of the browser with nothing else visible unless you slide your mouse to the top, where a double  row navigation bar with forward, back, window controls and etc. and your open tabs is visible.  This top double row bar functions in "auto hide" mode.

This auto hide mode is the default, but it can be changed to keep the top tool bar visible when in full screen mode.  
Right click a blank area of the top of the top tool bar and uncheck "Hide Toolbars".  You will also find an option to exit full screen mode.

Note that the F11 key will toggle between full screen and whatever the last window size was, so if you had it maximized, it will toggle between full screen and maximized.  This sounds like what you are experiencing.  If you first resize Firefox to a smaller unmaximized window and then try the F11 key it should switch between full screen and the smaller window.

You have mentioned that this problem only happens when someone else is using your computer.  Is it possible they are putting it in full screen mode and/or also resizing the unmaximized window to be the same size as a maximized one?

If I go to Tools > Error Reports, I also have a huge screenful of errors listed.  I believe these are referring to html errors on the pages you visited, not errors in Firefox itself.  You will notice that most of them are informational and they build up as you visit sites that have errors on them.  You can click across the top to hit the clear button and then try browsing to google.com and see if anything shows up.  It's a nice simple page with no errors on it and now that the error cache is empty, nothing should appear.

I actually prefer to browse in full screen mode on monitors that are smaller and have lower resolution, like on netbooks or older laptops with 1024 x 768 screens.
