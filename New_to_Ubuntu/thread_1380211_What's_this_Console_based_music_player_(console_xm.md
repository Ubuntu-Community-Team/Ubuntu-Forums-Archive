### Post by fancypiper on 2010-01-13
It's hard to tell from the pix but it may be [Moovida](http://www.moovida.com/).

Add this to your repositories
ppa:moovida-packagers/ppa
Refresh by:
sudo apt-get update
Install with:
sudo apt-get install moovida

You can do all this through Synaptic.
