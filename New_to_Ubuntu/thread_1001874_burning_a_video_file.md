### Post by ohzopants on 2008-12-04
There is a program called "devede" that can convert most video files into a dvd image.

It's a pretty long process.

Lastly, but possibly most importantly, the last time I used it there was a bug in the program that resulted in garbled audio in the resulting dvd.  You may want to check their website to see if this is fixed.
