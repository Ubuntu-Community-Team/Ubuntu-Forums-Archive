Somehow the associations for other file types have been linked to the Ebook reader software.  To change it back right click on the type of file you want to open (eg a jpg image) then click "Open With.." and "Other Application".  When you get the dialogue box come up, click on the checkbox for "Remember this application for JPEG image files" and then click the application you want, eg image viewer.

Now, whenever you click on a jpg file it should open with the viewer.
