### Post by _sAm_ on 2008-09-26
Yes;
sudo apt-get install mozilla-plugin-vlc

And install this in Firefox: [https://addons.mozilla.org/en-US/firefox/search?q=mediaplayer&cat=all](https://addons.mozilla.org/en-US/firefox/search?q=mediaplayer&cat=all) 
Just run the wizard.
