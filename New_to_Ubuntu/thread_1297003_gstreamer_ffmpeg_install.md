### Post by fooman on 2009-10-21
where did you try to download it from?

it is available from the universe repos in synaptic.  

open synaptic (system > administration > synaptic package manager) and in the toolbar,  go to settings > repositories

when the software sources box opens,  look on the ubuntu software tab and check off the first 4 choices (main-universe-restricted-multiverse).  then close that box.

click the "reload" button.  then click on the search button and type "gstreamer" (no quotes) in the space.  click "search"

when the choices pop up....find "gstreamer0.10-ffmpeg" in the list and install it.

hope that helps.
