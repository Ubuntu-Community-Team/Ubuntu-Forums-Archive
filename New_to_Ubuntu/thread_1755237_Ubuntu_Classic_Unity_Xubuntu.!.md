### Post by dniMretsaM on 2011-05-11
By the time GNOME 2 (Ubuntu Classic) is no longer supported, Unity should be farther along. It'll be more user-friendly, less buggy, etc. so you may want to give it a try then. If you still don't like it, try out any other DE that you want.  KDE (Kubuntu), XFCE (Xubuntu), LXDE (Lubuntu), GNOME 3 with GNOME-shell (currently no Ubuntu derivative with this DE is available, but I'm hoping for a Gnubuntu when 11.10 comes out), or whatever. Or you could just stick with the unsupported GNOME 2 if you really wanted to. I personally recommend KDE. Extremely customizable and very clean.

### Post by blauendonau on 2011-05-11
I'd say the environment closest to GNOME 2 is Xfce. With a bit of tweaking you can make it look almost exactly like Gnome. Actually, Xfce is more configurable in many areas. KDE is good but can take up more resources if the bells and whistles are enabled. Seeing as you have 3GiB of RAM that shouldn't be a problem. I have no experience with LXDE or others.

Don't forget you can install multiple DEs on the same system - you're not restricted to just one! Run "apt-get install xubuntu-desktop" (or kubuntu-desktop) from a terminal and choose the DE you want at the login screen.
