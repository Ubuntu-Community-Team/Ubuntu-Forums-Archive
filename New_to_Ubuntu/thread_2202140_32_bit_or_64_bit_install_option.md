### Post by joey8 on 2014-01-27
I would stick with 32 bit install with less than 4gb of ram. 64 bit would not be usefull at all in this situation.  See link [http://howtoubuntu.org/how-to-decide-if-you-should-use-32bit-or-64bit-ubuntu#.Uua5pPgo6V4](http://howtoubuntu.org/how-to-decide-if-you-should-use-32bit-or-64bit-ubuntu#.Uua5pPgo6V4).

### Post by deadflowr on 2014-01-27
I'd go 64-bit, simply because you have an added bonus of being able to increase the ram you have.

If you eventually run across the needed ram, it's easier to plop the new ram in than to reinstall the system.
