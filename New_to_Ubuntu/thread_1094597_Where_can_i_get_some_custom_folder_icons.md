### Post by plucky on 2009-03-13
> **Ash R said:**
> Hi again,

I suspect I am being a pain but I am having a little trouble installing [this]("http://gnome-look.org/content/show.php/Black-Red?content=91098") theme.

Open Nautilus and navigate to where the file was downloaded.Right click on the file and select "Open with Archive manager".Another window will open and you can extract the file.Select where to extract the file in your /home folder.It will create another folder called "Black-Red" and inside that folder is a file called "black-red.tar.gz".

The go to **System > Preferences > Appearances** and select install.Find the file "black-red.tar.gz" and "open".This will copy the theme into the window.Select the theme and select "Customize" will install it.


Good Luck.
