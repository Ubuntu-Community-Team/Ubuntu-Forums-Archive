### Post by yancek on 2016-10-21
> will i lost my data in ubuntu if i use "boot-Repair" tool ? 				

No, not if you select the Create BootInfo Summary option on the boot repair, it will simply probe your system and not make any changes and will then give you a link to where you can access that information and you should post that link here so it  can be reviewed.  Without that information, any suggestions you get will be only guesses as there are too many potential problems.
