Firstly, thanks again to those who helped me out with the BIOS flashing on my machine that will supposedly make this possible.

Alright, my new shiny Core 2 Duo E8400 should be arriving soon, but in order to prep my machine for the CPU's arrival my motherboard's website states that I have to overclock the FSB.  

I will be going from a CPU that uses a 800mhz FSB to one that uses a 1333mhz FSB. 

The board supports "1066 FSB, 1333 by Overclocking."

Has anyone done this before?  I can't seem to find a description or detailed step by step for it.  How difficult will it be?  Do I need to run out and grab a degree in electrical engineering?  Do I overclock the FSB first and then replace the CPU, or do I replace the CPU and then overclock the FSB?

### Post by jasontu on 2008-11-14
I turned out to be quite simple actually.  There was an easy option with the BIOS to OC the FSB to 1333 for Core 2 DUO processors that required it, and that was it.

The E8400 is a speedy chip!
