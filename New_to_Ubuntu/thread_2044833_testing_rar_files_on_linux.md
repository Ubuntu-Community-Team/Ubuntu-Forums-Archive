### Post by marlow59 on 2012-08-20
You can install 7-zip, it will add extra features to the archive manager : one installed, righ-click the rar file and look for the 7-zip menu, there will be a test option or so. Another possibility is to you the md5sum command (read the manual entry ```
 man md5sum 
```) or use the following software : [https://help.ubuntu.com/community/FileIntegrityAIDE](https://help.ubuntu.com/community/FileIntegrityAIDE)
