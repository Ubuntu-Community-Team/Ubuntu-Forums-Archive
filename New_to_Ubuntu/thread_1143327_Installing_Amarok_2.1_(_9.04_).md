### Post by bswilson on 2009-04-30
Well, Amarok 2.1 is still a beta release, so you're not going to be able to install it via Synaptic Package Manager or apt-get.

However, you can add the package repository listed [here]("http://amarok.kde.org/wiki/Download:Kubuntu"), and from there you can perform a command-line or GUI installation.
