how bout looking this over see if it applies or have you tried it already ?

**5. CardBus and PCMCIA Devices**

 
[LIST=1]
[*]Startup your computer without the wireless adapter plugged in. 
[*]Login when/if prompted. 
[*]When the desktop loads, open a terminal window and type the following command: sudo tail -f /var/log/messages
[*]Insert your wireless adapter into an available slot. 
[*]Repeat the command above and note the difference between the two. 
[*]If the above does not work, see the next section. 
[/LIST]
 
**5.1. Manual Installation of a PCMCIA/CardBus Card**

  
**5.1.1. Non-recognized Card**

 Some devices are not recognized by the OS upon insertion. 
If card doesn't show up (in lshw) try these steps: 
Run the following command. *Hopefully you'll get some output about the device.* 
sudo pccardctl ident[IMG]https://help.ubuntu.com/community/IconsPage?action=AttachFile&do=get&target=IconExample48.png[/IMG] 
  Socket 0:     product info: "Atheros Communications, Inc.", "AR5001-0000-0000", "Wireless LAN Reference Card", "00"     manfid: 0x0271, 0x0012     function: 6 (network)
[LIST]
[*]**If you get no output then the memory on the card cannot be read.** 
[/LIST]
Now we need to open a configuration file and add this information to it: 

gksudo gedit /etc/pcmcia/config.optsthe information you add should look like this, with your own data substituted. 
[IMG]https://help.ubuntu.com/community/IconsPage?action=AttachFile&do=get&target=IconExample48.png[/IMG] 
card ""Atheros Communications, Inc.", "AR5001-0000-0000", "Wireless LAN"   manfid 0x0271, 0x0012   function: 6 (network)   bind "ath_pci"After making this change run this command: 

sudo kill -HUP `cat /var/run/cardmgr.pid`Now run [lshw]("https://help.ubuntu.com/community/WifiDocs/WirelessTroubleShootingGuide/Commands#lshw") to see if the card is recognized and loaded properly. 
[IMG]https://help.ubuntu.com/moin_static192/light/img/icon-info.png[/IMG]  Most cards requiring this step will be older devices. You should not  have to do this with any cards made in the past few years, but there can  be exceptions. 
[IMG]https://help.ubuntu.com/moin_static192/light/img/icon-info.png[/IMG]  Notice the bind line which states what driver should be allocated to  the device. You will need to find what driver your device uses. More on  that in the next step. 
 
**From: [https://help.ubuntu.com/community/WifiDocs/WirelessTroubleShootingGuide/Devices](https://help.ubuntu.com/community/WifiDocs/WirelessTroubleShootingGuide/Devices)**
