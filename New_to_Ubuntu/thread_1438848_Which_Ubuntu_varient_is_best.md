### Post by patchwork on 2010-03-25
Ubuntu uses the GNOME Desktop environment, Kubuntu uses the K Desktop environment, (both are fairly heavy, but user-friendly), Xubuntu uses a lighter-weight XFCE Desktop environment.

Factors in your decision include the hardware you're installing the system on (old hardware, go for lightweight, like XFCE), and your personal preference.  Try a live cd from each.

As long as your hardware can support them, it's simply a matter of personal preference.

No matter which you choose, you can install multiple desktop sessions or window managers, and choose between them at startup.  Installing the full desktop environments will require more downloads, as KDE, GNOME, and XFCE have their own library and dependency requirements.

### Post by uRock on 2010-03-25
The four *buntus use different desktop environments. 

Gnome(ubuntu) is the most popular and works really well out of the box. 

XFCE(xubuntu) is a bit lighter on resources to help it run on systems with older/slower hardware. 

KDE(kubuntu) is designed to look really good, but it is a resource hog and it has programs that are easier to break.

LXDE(lubuntu) is a really light version aimed at getting the most speed out of older hardware. (I like it)
