### Post by ajgreeny on 2017-01-04
See the thread at [https://ubuntuforums.org/showthread.php?t=2219403](https://ubuntuforums.org/showthread.php?t=2219403) which though for 14.04 still works for 16.04 which I have (as Lubuntu, admittedly) on an old Atom netbook I use sometimes for minor stuff, but which has a useless battery, hence hibernate is essential to me.

Bear in mind that hibernation in Ubuntu and the family can be a bit hit and miss, so do not be surprised if it does not work on your computer even after adding the text file shown in that link, and make sure you try the ```
sudo pm-hibernate
``` first as well to see if it is possible with your hardware.
