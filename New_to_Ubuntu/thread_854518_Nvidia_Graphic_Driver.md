If you have nvidia-settings installed you can open that-I forget where that is but you can type it in a terminal too. When it opens you will see what version of the driver you have.
If you do install the driver manually you will need to re-install with every kernel update.

### Post by sdennie on 2008-07-09
The drivers on the nvidia site are slightly newer but, I don't think you will see any difference in performance on a 7 series card.  If the default Ubuntu drivers work for you, I highly recommend staying with those drivers as it's much easier to maintain them.
