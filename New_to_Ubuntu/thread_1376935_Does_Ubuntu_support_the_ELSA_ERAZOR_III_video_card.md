### Post by ankspo71 on 2010-01-11
Hi,
The best way to find out is to boot into the ubuntu live cd, select "try Ubuntu without any changes to your computer" (or something to that effect), and see if that graphics card works correctly. What will happen is you will be able to run the actual Ubuntu operating system right from the cd drive, without having to touch your hard drive. 
You can also search [http://www.ubuntuhcl.org/index](http://www.ubuntuhcl.org/index) to see if any of your hardware has been reviewed.
Hope this helps,
