### Post by grahammechanical on 2014-08-10
The new user to Ubuntu should really start with Ubuntu Help. Type Help in the Dash or select it from the power/cog menu. As for fully understanding the system, that, in my opinion, is a high target.

You might need to separate understanding Ubuntu from understanding Linux. We can get by in Ubuntu without needing to understand much about Linux. 

[https://help.ubuntu.com/community](https://help.ubuntu.com/community)

[https://help.ubuntu.com/community/ExternalGuides](https://help.ubuntu.com/community/ExternalGuides)

Remember, Ubuntu development is a community project and not a commercial project. It takes time for volunteers to update manuals and guide books. This is especially true because Ubuntu has a 6 month release schedule. So, although much of the advice is applicable it seems out of date because the version name is the old version name.

Ubuntu 14.04 has the Unity 7 user interface and that first appeared in Ubuntu 13.10. So, not a lot of change there. The next release of Ubuntu will be 14.10 and that will use Unity 7 and it is very likely that 15.04 will use Unity 7. It is not intended for us to get Unity 8 until the release of 16.04. Even then the differences between 7 and 8 will not be so much in the look but in the underlying code.
