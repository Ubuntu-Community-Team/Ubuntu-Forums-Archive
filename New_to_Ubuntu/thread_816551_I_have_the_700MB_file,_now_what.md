### Post by laffinet on 2008-06-03
Right now you are in the Live CD environment, which means you haven't installed Ubuntu yet, you are running Ubuntu from the CD. That's the reason everything is so slow.

To install Ubuntu and see it run much faster double click on the "install" icon on the desktop.
