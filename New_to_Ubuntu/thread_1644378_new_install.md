### Post by migs73 on 2010-12-13
Welcome

Just go ahead and install ubuntu on your new HDD if you have no intention of putting windows back on it. If you want to have windows as well as a dual boot, it is always better to install windows first.

Ubuntu should have almost all of your drivers out of the box. The only ones usually required to be downloaded are for wireless and videocards. Make sure you have a wired connection available during the install and this shouldn't be too much of an issue.

Lots of tutorials on how to do both;

[https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)

[https://help.ubuntu.com/community/Installation](https://help.ubuntu.com/community/Installation)


that's just a couple of the 'official' ones.

All the best :)

> would ubuntu help speed up a computer ,,y im asking is i might install it on the kids very old pc ,,,for a little test run first

You'll probably see a decrease in boot times and shutdown times. Unlike windows, Linux doesn't slow down over time as it is better at its own housekeeping. That means when you do a clean install any performance will remain.

For an older PC Lubuntu is a good bet. (128mb RAM and a pentium P2 minimum req; [https://wiki.ubuntu.com/Lubuntu](https://wiki.ubuntu.com/Lubuntu))
