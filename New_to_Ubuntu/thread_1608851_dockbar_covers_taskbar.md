### Post by chaanakya_chiraag on 2010-10-29
If you *hide* it, it will be accessible by hovering the mouse near the top edge of the screen.  If you *delete* it (or just kill gnome-panel), you will not be able to access that stuff **through a menu**.  However, you should still be able to access all the shortcuts in there through Gnome-Do/Kupfer, which are quicklaunchers.
