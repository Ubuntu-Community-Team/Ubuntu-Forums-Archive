### Post by igknighted on 2008-11-13
Some apps have linux versions, and others can be run through wine (a compatability library to run windows apps), but despite the instability of windows, most solutions for running windows apps in linux result in even more instability.  So I would strongly recommend finding a linux application to complete your task (this is possible in almost all cases), or else you might really want to consider whether switching OS's is the right move.  In the end you need to use your computer to accomplish a task, and if you need tools in windows to complete them, why keep trying to fit a square peg in a round hole?
