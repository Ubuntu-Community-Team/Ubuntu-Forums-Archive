### Post by Paqman on 2008-08-01
They use different desktop environments, but are otherwise the same system.

Ubuntu = Gnome desktop envirnment
Kubuntu = KDE
Xubuntu = XFCE

Gnome and KDE are pretty much equivalent, it's just a matter of personal preference. XFCE is a special lightweight desktop that's good for systems with modest specs.

You can actually install any (or all!) of the DEs on any Ubuntu system. The packages ubuntu-desktop/kubuntu-desktop/xubuntu-desktop will install the appropriate DE, and you can choose which to use when you log in.

Psychocats has a nice article about [Gnome vs KDE](http://www.psychocats.net/ubuntu/kdegnome), and gives more info about install and removing the different DEs.

### Post by toucher5 on 2008-08-01
The differences are simple, they each come with different desktop environments. There is a derivative of Ubuntu (not official but pulls from the same repositories or update server) called [Linux Mint]("http://www.linuxmint.com/"). The desktop environment is the same as Ubuntu which is Gnome, but the difference is the creator has changed the menu system to be similar to Windows so that the transition is smoother. I like Ubuntu but Linux Mint has an easier to transition into menu system.

Also some sites that you maybe interested in since you are new to linux:
[Wine HQ]("http://www.winehq.org/") - program for installing most windows programs
[How-to Forge]("http://howtoforge.com/") - website with how-to's
[Gnome-look.org]("http://gnome-look.org/") - Site for customizing your desktop environment.
