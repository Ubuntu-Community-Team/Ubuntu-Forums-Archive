### Post by Psumi on 2010-03-28
First, backup your current icons in here: **/usr/share/pixmaps/pidgin/tray**

Then, rename the icons you want to be put in using the names found in said directory.

After that, open TWO root file managers from terminal: **gksudo <file_manager>** (Open a new terminal tab with CRTL+SHIFT+T)

copy over the new icons from their current directory into the pidgin directory (BOTH FILE MANAGER WINDOWS MUST BE OPENED AS ROOT/SUDO). Keep in mind they have to be the same size as indicated by the folders, otherwise you'll have issues with sizing of the panel.
