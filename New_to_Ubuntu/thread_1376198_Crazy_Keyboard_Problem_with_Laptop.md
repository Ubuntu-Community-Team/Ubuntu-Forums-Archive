### Post by blazemore on 2010-01-09
It might be an accessibility option you have enabled somewhere.
Some users unfortunately cannot hold down multiple keys, so there is an accessibility option to hold them down automatically.
If you go to Preferences -> Keyboard, are there any likely looking options in there?
I'm afraid I don't know much about this sort of thing, but that would be my first guess.

Also, have you tried with a totally different keyboard?
