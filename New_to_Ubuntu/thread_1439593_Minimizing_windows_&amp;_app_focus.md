### Post by Kulgan on 2010-03-26
Your window list may for whatever reason not be on your taskbar. 
Add it with the context menu. (Right-click or Ctrl-click or whatever you Mac folks do :P )

For window switching it is Alt-Tab on a Non-mac keyboard. It can be changed in gnome through gnome-keybinding-properties, and I assume there is a KDE equivalent. It may be that it's set up to use a key you don't have...

### Post by Kulgan on 2010-03-26
This is peculiar. It should be listed under K &#8594; Control Centre &#8594; Regional/Accessibility &#8594; Keyboard Shortcuts. 

If you use Compiz/desktop effects, you can always override the settings with compizconfig-settings-manager. You will have to enable the application shifter plugin and set the keybinding for it.
