### Post by rustutzman on 2009-10-07
Install Cheese and see if it works. Go to Administration and look for Synaptec Package Manager. Run that and click on the Search button (don't use Quick Search) and type in Cheese. when you see it in the list put a check by it and click the green checkmark at the top to apply the changes.  When it's done go to the Sound and Video menu and Cheese should be there.
