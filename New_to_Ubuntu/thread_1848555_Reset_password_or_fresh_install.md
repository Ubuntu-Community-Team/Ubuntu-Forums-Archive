If you had gone with the fresh install like everyone has suggested (including me), you would have been done already.  Instead, you're still contemplating the whole password thing, so you tell me, which is faster and less work? 
To me in the long rung, it is much, much easier to do a fresh install than to reset a password and every single day carry around all of the unnecessary baggage that was never mine to begin with.  Think about all of the little applications that will be running in the background from the minute that you start up the system everyday.  That alone will undoubtedly slow you down and can most definitely be avoided with a fresh install.
Like it was stated before, this isn't Windows, which means that whatever software you want, you can get whenever you want it from the Ubuntu Software Center.
Save time and do a fresh install.  Then you know exactly what you have and you can customize it as well.
