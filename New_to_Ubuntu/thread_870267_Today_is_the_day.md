### Post by tamoneya on 2008-07-25
Welcome!

There should be no specific drivers needed for your DVD drive however you may need to install some proprietary codecs if you want to play DVDs.  There are several tutorials and threads in the forum about this but we will cross that bridge when we come to it.  As for other drivers I cant be sure.  You may need to install the drivers for your wireless card or you ethernet.  It depends on which chipset your laptop uses.  Again though we will solve it once you get ubuntu installed and see what works.  The nice thing about the liveCD is that you can test the internet and everything with the liveCD so that you will know in advance whether you will need the drivers.

### Post by candtalan on 2008-07-25
If you are getting the desktop CD (a live CD) then boot from it first and just see if everything works ok. Then install. When you install, there is no need to format the drive first, particularly if you intend to wipe the PC. Just choose the install option 'take over the complete drive'.

The dvd cd drive should be recognised ok, drivers should all be in the cd, the live cd
