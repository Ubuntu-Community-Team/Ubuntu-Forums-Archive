### Post by Monicker on 2008-05-05
When I used to dual boot between Windows and Linux, I created a small FAT32 partition for files which I wanted to be accessible from either operating system.  Or you could just put the files on a usb flash drive, if you have a port for it.  Depends on how large the files are that you want to share, and how frequently they would be accessed.
