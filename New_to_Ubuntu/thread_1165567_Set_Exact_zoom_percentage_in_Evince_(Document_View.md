### Post by enoksrd on 2010-06-17
don't know how to set custom zoom in evince, but acroread allows custom zoom (just type a number in the zoom box).

even better in most cases: acroread has a "fit visible" view where it attempts to eliminate the margins.  combined with fullscreen this gives about the best possible view, assuming a one column document, without fiddling with manual zoom values.  you can access "fit visible" from the drop down menu for the zoom box, or with CTRL+3 (which is the only way to turn it on in fullscreen mode, AFAICT).
