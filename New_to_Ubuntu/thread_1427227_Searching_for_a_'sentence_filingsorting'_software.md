### Post by Sir Jasper on 2010-03-11
Hi,

For Windows the free version of Evernote could be ideal. It deals with, text, pictures and multiple tags. It is one long list, but it is readily searchable and the manual has only about 30 well illustrated pages. It can, if desired, also have on-line storage and multiple tags. There is also a portable version which can be synchronised.

There is also a Linux version of Evernote, but the latest version is reported to have problems (possibly only in conjunction with Firefox); however I am using an older version which works well through Wine.

Do you need pictures or just text and in ´buntu are you happy to use Wine?
