> I installed then updated a few things (now it was a dual boot) Ubuntu  10.10 x64 was on separate drive from windows 7 x64 and I have had major  problems with windows ever since I installed Ubuntu, so I uninstalled  but was liking it, so need to figure out how I can get both running on  the same system with out windows getting messed up! **I think I used wubi  to install or one of the installers that was on the Ubuntu site.**
Just for your info OP a wubi install of Ubuntu is actually installing Ubuntu within a file on the Windows NTFS filesystem. It is not a "normal" dual boot. Does your post mean that the original problematic install was actually a wubi install? Updating the grub packages in a wubi install is a major pain when it happens.

Determine what type of install you originally had and try a side by side partition install. This will give you the best of both worlds, gaming in Windows and general computing (stable and secure) in Ubuntu.

### Post by Paqman on 2011-04-17
> **Bradford1040 said:**
> what do most people use it for if not a replacement?

I do almost everything with Linux, but I do keep Windows for gaming. Windows is an excellent gaming platform, so if you do a bit of gaming, then you're probably going to want to keep Windows on your machine, even if you don't use it much.

You will be able to do some gaming on Ubuntu. Some games like World of Goo having Linux versions available, and some like Minecraft are cross-platform, so will run on anything. Wine really only works well for older games in my experience, and even then they'll probably still run better in Windows.
