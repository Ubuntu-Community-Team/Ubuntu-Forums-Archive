### Post by 73ckn797 on 2008-11-28
Ubuntu does not need to be defragmented. The utilities are basically under System on top panel. If you do not have System/Control Center you can go to Applications/Add/Remove and select "services". That will have all of the basic utilities you will need in one window.

### Post by Coreigh on 2008-11-28
Its all under System >> Administration, for the most part. If you don't find the utility you want there you can probably find it in the Add and Remove programs or Synaptic.

Linux doesn't need a defrag utility because the file system works differently from Windows and rarely becomes fragmented.
