### Post by srs5694 on 2011-04-26
It's possible; however, if you're trying to install OS X in a non-Apple PC, that topic is frowned upon by the moderators here, so I recommend you take the discussion to a "Hackintosh" forum.

If your Ubuntu box is a Mac, then you may have to re-install OS X on the disk you pulled from the eMac. I'm not positive, but I think that OS X customizes itself for each system, so that an install on one computer won't work on another. Using [rEFIt](http://refit.sourceforge.net/) can help with the dual booting once everything's set up.
