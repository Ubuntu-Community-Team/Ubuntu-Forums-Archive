### Post by tjwoosta on 2008-07-10
i changed my bootsplash with startup manager

```
sudo apt-get install startupmanager
```

then it will be in system-administration-startup manager

and for the gdm theme you should go to system-administration-login window
(you should be able to just drag and drop the .tar.gz right into the the other themes and it will install automatically)
