### Post by asuastrophysics on 2009-11-22
Hey everyone,

I can't help but feel a little disappointed about the available media players for Ubuntu. I've tried Amarok, Banshee, Rhythmbox, Songbird, and now Winamp.
(I'm not trying to hate on anybody who happens to be a fan of the above players! 8)  (I just happen to have a gigantic collection of music that's rather hard to manage)

Is there any media player that just "has it all"? ;)
By "having it all", I mean:

 - CD Ripping / burning support
 - Audio file converting support (EX. MP3 to AAC, WMA to FLAC, etc)
 - ID3 Tag editing support
 - Album art fetcher
 - "Auto tag" feature
 - Decent DAAP Support (Banshee will READ daap, but won't broadcast it. Winamp has no DAAP support whatsoever.)
 - No nagging / telling me to "Search for a suitable plugin" when the file in question is a *.png album art file. (ahem, Rhythmbox...) <- I have 85GB of music...It's a big library, that's why I get this message often with RB
 - Ipod support (Syncing, restoring, and updating) :D
 - Visualisation plugins (Don't need to have these, but they're nice!)
 - Equalizer support
 - Skinnable 
 - A player that doesnt freeze up / freak out because some instance of flash player is running somewhere in the background
 - Customisable columns in the music library (Rhythmbox is the only MP I know of where you absolutely cannot adjust or move the Artist / Title / Genre / Year columns.

Sorry this list is so long and again, **I'm really not trying to bash Linux Players, and I'm not trying to sound picky, even if it comes across that way. I'm open to any suggestions** :D I just have a _gigantic_ music library that's in need of a player that can handle all of this :P

Does anyone know of a player that would do all this? If so, that would be really awesome! :popcorn: Thanks for any suggestions in advance! :-)

BTW, if you know of any plugins I can add to any media player you see above to make it do these things, please share them with me :p
