### Post by 3rdalbum on 2010-06-30
The command:

```
eject -T
```

will open and close the tray.

If you are running Compiz, you can use the Commands plugin of Compizconfig Settings Manager to set a keybinding for that command.

If you're not using Compiz (and I suspect you might not be on a G4) you could set up a launcher on your panel for that command, but I don't think there's any way of setting a keybinding unfortunately.
