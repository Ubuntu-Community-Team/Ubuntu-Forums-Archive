### Post by ks07 on 2009-09-21
Use the location setting, it will go to pretty much any kind of address. For example, if you want to make a launcher that goes to Google, you would pick 'location' and type the location in as [http://www.google.com](http://www.google.com)

This will open your default web browser on the chosen page (usually Firefox).
