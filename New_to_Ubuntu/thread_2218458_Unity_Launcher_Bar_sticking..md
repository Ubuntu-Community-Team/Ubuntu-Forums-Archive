### Post by deadflowr on 2014-04-20
Maybe try reset.
right click an application and select unlock from launcher.
then open the dash menu and find the application you just unlocked and drag/drop it back to the launcher.
rinse and repeat if it worked for the first one.

You can navigate the applications in the dash more directly, by either running a quick Search, or by right-click the dash icon selecting applications OR, by keyboard, super(windows key) + A, which will open the application subb menu.

Simply a quick suggestion you might try, is all.
