### Post by SlickRick on 2010-03-30
Unetbootin can only install operating systems that have a live cd.
Since windows doesn't come in a live version, you won't be able to install it on your USB.
You could try installing windows to a USB as if it was a hard drive. By booting up the installation CD and selecting the USB.

...and the usb startup thing is for ubuntu only
