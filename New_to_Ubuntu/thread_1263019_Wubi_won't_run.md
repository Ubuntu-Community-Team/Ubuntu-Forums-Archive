### Post by Paqman on 2009-09-10
Did you burn the disk as images, or as data? I'd also make sure to burn them at the slowest speed possible.

Strictly speaking, you don't need to have a disk burnt to install using Wubi. Just put your disk image in the same folder as wubi.exe and run it. Wubi should spot your disk image, if it doesn't it'll just pull a new one down over the internet.
