### Post by SqRt7744 on 2009-01-31
get the driver with .deb extension (for Debian or Ubuntu),

Open a terminal window (Applications->accessories->terminal) and run the command: sudo /usr/local/lexmark/setup.lexprint

install the printer by clicking on applications->system tools->lexmark printer driver (or type lexprint into applications->accessories->terminal)

you might also have to update the software from within the lexmark printer driver program.
