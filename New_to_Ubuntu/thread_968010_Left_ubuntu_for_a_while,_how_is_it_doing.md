### Post by MichaelSwengel on 2008-11-02
It's AMAZING....

For me (Macbook Pro 4th gen) much more is supported out of the box. Many more drivers have been added to the default installation. The Hardware Drivers panel in System > Administration has been greatly improved. It detected my wireless hardware immediately when, before, I had to unrar an EXE of the Windows driver to get to a specific INF file.

Now Ubuntu just downloads the driver for me. SaWeeeeet.

I've noticed that I don't have to do nearly as much hardware configuration as I did in 7.04-8.04.

As far as Office....Open Office is fantastically improved.... iTunes, I believe, is outperformed by Amarok. But that's down to personal preferences.

However, I remember reading something about being able to emulate iTunes in Ubuntu if you really need/want it.

Try it out again. I think you'll like the changes.
