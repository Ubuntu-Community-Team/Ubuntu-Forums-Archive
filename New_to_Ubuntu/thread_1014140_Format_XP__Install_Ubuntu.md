### Post by jespdj on 2008-12-17
During the Ubuntu installation process, you can delete the Windows XP partition and create a partition for Ubuntu. You don't need any other special tools to remove Windows XP.

In fact, I think it's as easy as answering "Yes" to the question "Use the whole disk for Ubuntu".

### Post by ok_dr on 2008-12-17
I was in your same situation up to 10 hours ago. Now that I'm finished installing I can say that Ubuntu does it all for you, I inserted the cd, of the 3 options I chose the first (demo and install) and when it booted from the cd I chose install, after that ubuntu formatted the hard drive by itself, you just have to answer to some questions about the configuration during the process.
Anyway if you just want to have a feel of ubuntu without installing at all, you may not install it rightaway but boot from cd. Anyway keep in mind that when ubuntu runs from the cd it'll be very slow, so you can just evaluate the look and the included software not the performance, once you install it on the hard drive it's as fast as windows.
