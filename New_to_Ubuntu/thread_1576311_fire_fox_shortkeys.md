### Post by timmy303 on 2010-09-17
I always really enjoyed the backspace button as the back short-key for my web browser. i have been reading around and haven't seen anything that says i can change my Ubuntu web browser so that it acts in the same way. is it possible to change my short-keys in Ubuntu?

---

### Post by fly-by-night on 2010-09-17
Type **about:config** in the address bar.  Enter **backspace** in the Filter box.  Double click on the only listed item and set it to 0 (zero).

Edit: You might get a warning if it's your first time in about:config.  Confirm that you know what you're doing.  
Also: this work in Firefox only.

### Post by fly-by-night on 2010-09-17
about:config hold all Firefox settings.  Some of these can be changed from menu items and add-ons etc, but for many including the backspace setting you need to get under the hood.  The changes take immediate effect without the need to restart Firefox.

[http://kb.mozillazine.org/Editing_configuration](http://kb.mozillazine.org/Editing_configuration)
