### Post by Dennis N on 2018-11-08
You can have a switcher on the panel, but I don't know of one that shows the icons of the applications so you can see at a glance what is there. I added one just so I could quickly switch to a new workspace if I want to use it - see screenshot. I find features like the alt-tab switcher a faster way to go back and forth between applications, and with that you get a small preview. All these added features are gnome extensions.

The panel workspace switcher extension shown is called "Workspacebar". 
The alt-tab switcher extension that works across work spaces is called "AlternateTab".

Note: "Workspacebar" is partly broken, but it still does the basic switching in gnome shell 3.28 (used by Ubuntu).
