### Post by nerdtron on 2015-06-21
Take a look at this post for creating a shared folder in Ubuntu for windows computers. It uses samba for interoperability for windows machines. [http://www.howtogeek.com/116309/use-ubuntus-public-folder-to-easily-share-files-between-computers/](http://www.howtogeek.com/116309/use-ubuntus-public-folder-to-easily-share-files-between-computers/)

Also, installation of software on Ubuntu is not like windows where you "download this file, extract this file, then double click this file" etc. For installing software in Ubuntu, you should open the Software Center and search for the application you need. Click it then hit Install. The application you need will be download and installed for you.
