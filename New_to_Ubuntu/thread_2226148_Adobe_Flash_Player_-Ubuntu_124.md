### Post by Impavidus on 2014-05-25
+1 to flashplugin-installer

However, Adobe has indeed stopped development of flash for Linux. We are stuck at version 11.2 and only get security updates. Some sites, few but they do exist, insist on a later version of flash, so these won't work in Linux, even with flash installed.

If it doesn't work with the flash player, you can try Chrome. It has a flashplayer build-in.

### Post by monkeybrain20122 on 2014-05-25
Flash on Linux is stuck at 11.2 because Adobe has stopped providing feature updates since 2012 (but will provide security updates til 2017), 11.2 works for most websites but FB games require newer versions. So on Linux the only way to get it would be through Chrome (or Chromium + the pepper flash package for Ubuntu 14.04) because Google made a deal with Adobe to package the latest flash.
