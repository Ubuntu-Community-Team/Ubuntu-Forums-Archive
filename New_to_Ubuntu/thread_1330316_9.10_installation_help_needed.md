### Post by ukripper on 2009-11-18
>   * I can't download and install upgrades
Can you run this in terminal:
```
sudo apt-get update && sudo apt-get dist-upgrade
```
does that let you update packages?
 >    * I can't install programs from Ubuntu software centre
What program you are trying to install?
    > * I can't watch Utube vids and quite a few others
You need flash for that.
Run this command and install ubuntu extra packages including flash.
```
sudo apt-get install ubuntu-restricted-extras
```


hope it helps
