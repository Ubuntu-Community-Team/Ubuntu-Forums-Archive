### Post by NoNameWill on 2012-04-23
Where did you download the package too?

From the command line type

```
ls
``` This will give you a list of files and directories. Now change directories t the one you d/l'd it to.

```
cd path/to/directory
``` i.e cd Downloads 

Your prompt should change. to some like this 
```
mike@mikes-pc:~/Downloads$
``` ```
ls
``` again to make sure your file is there. 

Then run the command 
```
sudo dpkg -i linux-firmware-nonfree_1.11_all.deb
```You could also just do something like this.
```
sudo dpkg -i ~/Downloads/linux-firmware-nonfree_1.11_all.deb
```
