### Post by Mark Phelps on 2010-02-23
Also, the "upgrade" version will not do an in-place "upgrade" from XP to Win7; what it does is a clean install of Win7. I believe it provides the capability to copy over settings and data, but it does NOT provide the ability to copy over applications (programs). The "upgrade" part of the title means it is an upgrade license.

It will search for an existing MS Win OS on your machine as part of the install.  Not finding that, it will prompt for the CD Key of the original XP installation.

If you want to do an in-place upgrade from XP to Win7, you have to use the product from LapLink.  That's an additional cost -- but worth it if you have lots of apps installed and no longer have the install disks/CDs to reinstall them on Win7.

Correction:  A Win 7 Upgrade install does _NOT_ ask for the CD Key of the original XP installation.   However,  a complete install of Win 7 from the upgrade disk without the presence of an existing XP install *is possible*.   A Google search will reveal some clues as to how to do this, although there are no guarantees.   If you paid for the original installation of XP on the computer, and that copy of XP is not being used elsewhere, you will have met Microsoft's stated requirements for a Win 7 upgrade.
