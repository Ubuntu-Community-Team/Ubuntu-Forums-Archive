### Post by Nano Geek on 2010-07-13
It could be a number of things such as insuficient memory, bad RAM, malfunctioning motherboard, or a bad installation. 

For starters, I would do a memory check using the Live CD. If that comes up clean, then you might try keeping an eye on memory usage as your working to see if it's maxing out the available memory. If nothing seems wrong, you might just want to reinstall Ubuntu.
