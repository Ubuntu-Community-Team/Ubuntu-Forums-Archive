### Post by ramjet_1953 on 2009-02-21
Re Foxit Reader for Linux:

This is a great pdf reader and incredibly fast.

You don't need to go through the install procedure that is listed in the User Manual.

All you have to do is un-tar the archive and copy the folder into your /usr/share directory using Nautilus in sudo mode.

Then just use >Preferences>Main Menu to crate a launcher in the Office Category.

### Post by sgiessl on 2010-07-14
I highly recommend Konqueror with the embedded Okular component.

Okular is lightweight and Konqueror provides the shell with all tabbing features I need. What's especially nice about Konqueror is, in my opinion, the ability to have multiple views for one document (window splitting), automatic (in case of crashes or restart) and manual session management, allowing me to easily refer to a set of tabs in the future.
