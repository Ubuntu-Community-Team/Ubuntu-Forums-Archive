### Post by cariboo on 2009-01-27
Most of what you need is available in the repositories. Check out Applications-->Add/Remove programs, make sure All Available Packages is selected, or System-->Administration-->Synaptic Package Manager. The Last time I check there were over 30,000 packages available, not all of them are programs, but you should be able to find any thing you need using the search button.
