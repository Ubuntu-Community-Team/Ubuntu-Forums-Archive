Check it is not set to scan a Text Document,as that will come out as no colour.

In Simple Scan it is **Document > Scan** and make sure **Photo** is selected.

### Post by khelben1979 on 2012-05-12
How long have the scanner been working for you?
Have you done any major Ubuntu upgrades?
What software are you using for scanning documents? 

Also if the scanner is old, it could be that it needs to get replaced, so that's also good to know. Testing different scanner software might exclude some software bug, but more likely the scanner system libraries will affect the result and performance of all your scanner applications ([libsane]("http://packages.ubuntu.com/precise/libsane")).
