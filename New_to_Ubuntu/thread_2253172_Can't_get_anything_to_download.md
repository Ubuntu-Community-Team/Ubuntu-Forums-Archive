### Post by coldcritter64 on 2014-11-17
> Ubuntu 13.04 has gone EOL (end-of-life) and is no longer  supported - the  repositories have been taken offline (or at least,  archived)
OP, please note steeldriver's post again, this is why, the repository no longer exists where it previously was located, it has been archived. This will affect installation from the repositories by any method including the ubuntu software center.

You either need to change the repositories to old-releases.ubuntu.com OR _*preferably do an upgrade/reinstall (so your system can get security updates)*_.

> and have been search the internet throughout this process with no problemsYes, you still have internet and networking, Canonical (the company that produces Ubuntu) have moved the repositories you are trying to access as previously noted.

### Post by grahammechanical on 2014-11-17
The next version after Ubuntu 13.04 is 13,10 and that also has reached end of life. so, if you try to upgrade to 13.10 and then upgrade to 14.04 you will not be able to do it unless you follow the advice here.

[https://help.ubuntu.com/community/EOLUpgrades](https://help.ubuntu.com/community/EOLUpgrades)

Which is why it is being recommended that you fresh install Ubuntu 14.04 which will be supported up to about May 2019.
