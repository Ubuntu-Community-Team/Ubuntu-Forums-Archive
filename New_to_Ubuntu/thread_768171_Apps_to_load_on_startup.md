### Post by kneewax on 2008-04-26
Hi folks, another simple question.  Does anyone know a simple way to get apps like Skype and Pidgin to load on startup.  I had assumed this was application dependant but notice that none of these pieces of software offer the option to start on boot.

Thanks

---

### Post by Crazy Jesus on 2008-04-26
Yup, just goto System | Preferences | Sessions

Click add, and then type in the name of the command you'd use to run them in a terminal and give it a name so you know what it does when you look at the list next time, and maybe add a comment if you need it.

Any troubles, feel free to ask.

---

### Post by sdennie on 2008-04-26
If you go to System->Preferences->Session, you should be able to add applications to automatically start when you login.
