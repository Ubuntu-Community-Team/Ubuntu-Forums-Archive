### Post by krzysz00 on 2009-02-21
A virtual machine = slower than normal. Live boot (from USB) = a bit faster than boot from Live CD, which is darn slow. So both together = horribly slow. Plus, VMWare has (in my experience), huge overhead. Ask your (teacher/ Prof.) if you can use VirtualBox
