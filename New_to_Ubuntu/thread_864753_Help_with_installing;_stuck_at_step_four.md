### Post by dshipp28760 on 2008-07-20
If you are seeing sb a and sb b or some thing similar,  I might think you have a promise array or some other dual raid drive. I would suggest you unplug one of the drives then boot to  windows to see which one affects what! One may hold data, while the other holds the operating system. if you can't boot, try the other. If you still can't boot you may have a redundant set up. Also, Possibly , instead,  you need to read about setting up a dual boot system using Vista. I know Ubuntu wants to take over your whole drive (and system) unless you first tell windows you are setting up a dual boot and installing a second OS.
