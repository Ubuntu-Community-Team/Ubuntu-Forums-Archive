### Post by compiledkernel on 2009-01-19
Use a tag editor and see how the audio file is being tagged. Odds are its tagged incorrectly, and thats what Rbox is using to pull the album data from. 

I recommend CowBell for this (sudo aptitude install cowbell).

You can google image search the album cover and save the image, then drag it onto the picture.  I find that the amazon default album art is usually very small, and thus ends up blurry when it's resized in Rhythmbox.
