### Post by stoogiebuncho on 2008-10-21
Konversation is in the Add/Remove Programs place, so it would be a lot easier to install it from there, plus it would update automatically when new versions come out.

Go to Applications > Add/Remove and search for Konversation.

Generally in Ubuntu it's easier to install programs via Synaptic Package Manager or Add/Remove.  If you can find a .deb file on the internet, you can install that much the same way you would install an .exe file, but pretty much everything else involves working in the terminal and gets a little more complicated.

### Post by stoogiebuncho on 2008-10-21
Oh, and if you want to use the terminal, you can just do this:

```
sudo apt-get install konversation
```

That will download and install it from Synaptic.
