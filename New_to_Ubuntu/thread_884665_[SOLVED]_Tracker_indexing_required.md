### Post by AlexMono94 on 2008-08-10
System -> Preferences -> Search and Indexing

Then tick enable indexing, log out and log back in. You should see an orange magnifying glass and Tracker will begin to index your system. It is wise to enable watching and reduce the index delay to 3 seconds)

This is for Hardy so if you're on Gutsy I can't help you sorry.

People seem to hate Tracker but it works perfectly for me. =/

Hope this helps. =]
