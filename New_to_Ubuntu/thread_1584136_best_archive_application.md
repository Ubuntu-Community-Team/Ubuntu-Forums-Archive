### Post by beew on 2010-09-28
You can open rar files with p7zip by installing unrar in Synaptic (fyou have to get the non free version, as the free one doesn't work,--so I have no idea what it is good for)

You can also try PeaZip.  It opens .rar file automatically without having to install any other module. It is not in the repos so you have to download and install the .deb file yourself.
[URL="http://peazip.sourceforge.net/"]
http://peazip.sourceforge.net/[/URL]

One odd thing about PeaZip ( bug?) is that if you try to open an archive on the Desktop by just right clicking it on the desktop you will get an error message. But it opens fine if you instead go to Places > Desktop and do the right clicking within the windows that opens up (so in other words you have to work through the file manager)

P.S. I think file roller is garbage, got rid of it as soon as I installed PeaZip.
