### Post by TeoBigusGeekus on 2008-04-28
Ubuntu uses the gnome desktop environment whereas Kubuntu uses the kde one. Although they come with different applications, you are still able to install a kde app to ubuntu or a gnome app to kubuntu. The differences are only aesthetic.
Xubuntu uses the x desktop environment which is lighter and it is therefore recommended to older pcs.

It depends on the programs you want to use. There might be open source alternatives in Linux, try these pages

[http://www.linuxalt.com/]("http://www.linuxalt.com/")
[http://www.linuxrsp.ru/win-lin-soft/table-eng.html]("http://www.linuxrsp.ru/win-lin-soft/table-eng.html")


Ubuntu has wine, an application which lets you run some window$ programs on linux, but not all of them. Check this page for more information.

[http://appdb.winehq.org/]("http://appdb.winehq.org/")

If you still cannot get results, you can try virtualisation, using vmware or virtualbox.
And, finally, you can always dual boot...

### Post by sasquatch74 on 2008-04-28
Hello and welcome to Ubuntu.  The difference between the three versions of Ubuntu is the windows manager.  You have to think of linux as layers.  The underlying programming is the same for all three verions of Ubuntu.  The difference is how the user interfaces with the operating system.  Ubuntu uses Gnome as the interface.  Kubuntu uses KDE.  Xubuntu uses XCFE.  Each offers a particular style and tool-set to use your computer.  You can try out each and decide for yourself which you like best!  
As far as your windows programs... There is an application called WINE that will allow you to run some windows software.  Check their website to see if what you need is support by WINE.  There are other threads to help you out with intalling and working with WINE.  If your windows applications are not supported by WINE, you can dual-boot, by keeping windows in one section of your hard drive and ubuntu in another (which is what I have to do).  Again, there are threads to help you set this up.  
Get used to using the search feature on this fourm and you will be fine. 
Good luck!

### Post by kestrel1 on 2008-04-28
The difference between these three are basicalliy the desktop user interface.
Ubuntu uses the gnome interface, Kubuntu uses KDE & Xunbutu uses XFE. If you want to transfer from Windows to Ubuntu, you need to look at the programs you want to run from Windows. Some may run under Wine, you may be able to get an open source alternative or you could look at running a virtual machine such as VirtualBox where you could run Windows under a virtual environment & run the programs from there.
I have recently decided to ditch Windows XP for Ubuntu, but am running Windows XP under a virtual environment. It work very well, but can take a bit of setting up.
I would recommend that you install Ubuntu or Kubuntu as these may cause less problems than Xubuntu, unless your hardware is old.
What programs do you want to use from Windows?
