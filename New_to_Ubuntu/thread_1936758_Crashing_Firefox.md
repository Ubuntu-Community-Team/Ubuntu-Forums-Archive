Funny you mention that, Swagman, because RAM problems was a the very thing I was going to mention as I was scrolling down to the reply button on this page.

When you have a clean install of any OS, VictorGF, and right off the bat you are having problems, you should start suspecting possible hardware issues.

Go [here](http://www.memtest.org/#downiso) to download the latest version of the bootable Memtest86+ ISO, burn it to disc, reboot with it, and let it check your RAM. See what happens.

What you're complaining about should not normally be a software problem.
