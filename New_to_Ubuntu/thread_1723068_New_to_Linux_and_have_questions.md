### Post by 1clue on 2011-04-06
I agree with trying Open Office or similar, but I will touch on what nobody on this thread seems to have touched yet.

There is a commercial product called Crossover Office [http://www.codeweavers.com/](http://www.codeweavers.com/) which is wine but with some commercially licensed drivers in the mix too.  Codeweavers is the company which brings you the free wine package everyone is telling you about.

This is a completely licensed and legal-to-use product, even from Microsoft's point of view.  The license for the crossover product pays for your right to use the Microsoft DLLs they give you in the product.

It doesn't handle absolutely everything, but some time back I needed bulletproof Microsoft Office performance, and that gave it to me.  It was years ago, but I'm sure that mainstream products like that are still probably bulletproof.

### Post by Jerry N on 2011-04-06
Something not mentioned - Openoffice.org and libreoffice will both run in Windows 7.  Try downloading libreoffice, prepare a document in LibreOffice Writer, save it as a DOCX file, and see if it meets your class requirements.  Libreoffice isn't as fast as Microsoft office, whether in Windows or Linux.  If you need spreadsheets, Gnumeric is probably more compatible with Excel than Libreoffice and is a lot faster.  It too is available for Windows or Linux.  Neither Libreoffice nor Gnumeric do Microsoft macros.  

Ubuntu might start a little faster than Windows 7 but I have yet so see any performance benefit over Windows 7.

Your Windows 7 probably came with Norton antivirus.  Norton antivirus is pathetic.  If you want to stay with Windows (or dual boot with Linux) get rid of Norton and install a free antivirus program if the yearly cost is significant to you.
