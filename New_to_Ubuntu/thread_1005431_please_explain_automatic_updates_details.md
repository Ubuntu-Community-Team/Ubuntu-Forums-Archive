### Post by kramer65 on 2008-12-08
I set up a pc for my father (64 years old) to use for the internet only. I used number two and I unchecked the "recommended updates" above that.

It now does security updates fully in the background so my dad is not bothered with passwords etc. and it simply doesn't do the normal program updates. When I visit him once every couple months I simply do an update of all the programs. He's pretty happy with it, mainly because it doesn't bother him as much with "handy" notifications as MS windows did.

I hope that helps! Good luck!
