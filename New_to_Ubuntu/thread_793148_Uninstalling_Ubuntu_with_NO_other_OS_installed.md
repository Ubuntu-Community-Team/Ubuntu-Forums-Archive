### Post by LeoSolaris on 2008-05-13
If you have the Windows disc you wish to use to install, put it in your cd drive, then restart. As your system restarts, watch the BIOS screen for Boot Options, or Boot List, Different BIOS call it different things. It will bring up a choice of locations to boot from, pick the CD drive with Windows in it. That should start the installer program on the Windows CD.

If it reads the CD, and skips it to go to the hard drive, you either have an issue with the CD, or with the CD player. Most likely it's the CD. Clean it, and check it for scratches, just to be on the safe side.

To test your CD player, try putting the Ubuntu LiveCD in to see if it will load it. If it does not, then your CD player may be broken.

I highly doubt this is an Ubuntu issue, since Ubuntu has no control over your computer booting from a CD.

### Post by MrWES on 2008-05-13
I'm new to Ubuntu, but ran slackware for years. Anyhow, two recommendations going forward:

1. If you're running any Linux distro, burn yourself a Linux System Rescue CD -- great to have around when you need to repair something or run qtparted, etc.

You can download it from here: [http://www.sysresccd.org/Main_Page]("http://www.sysresccd.org/Main_Page")

If you're a Windows user, I STRONGLY recommend you invest in a PE disk -- Preinstalled Environment. I personnally use BartPE. I can't tell you how many computers I've saved at work (and now own :)) using BartPE. You can read more about BartPE and the howto here: [http://www.nu2.nu/pebuilder/]("http://www.nu2.nu/pebuilder/")

Also, I've spent many hours doing a fresh install of Windows XP, etc. and I would recommend DriveImageXML. What I did was take a fresh install of XP and use DriveImage to create an image of my Windows install -- great for when you want that annual fresh install of Windows, btw.. Microsoft actually recommends that! Sigh..

You can snag DriveImageXML here: [http://www.runtime.org/driveimage-xml.htm]("http://www.runtime.org/driveimage-xml.htm")

DriveImage and BartPE work very very well together. With these tools, or others like it, you are ready for about any failure. Remember, to learn you must get burned :)
