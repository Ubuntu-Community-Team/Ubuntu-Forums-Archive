### Post by derred on 2008-06-02
I've only had one problem with updates (in 06 i think) while installing it messed up my xorg.conf and I couldn't get graphics. It was easy to fix and the only bug I've encountered so far.
I usually apply updates every day and upgrade all programs (since they sometimes depend on one another).
If you want to make sure that you won't lose your settings just have a separate home partition. This way, all your settings, documents, music etc will be safe in case you have to reinstall.

Hope this helps

### Post by Paqman on 2008-06-02
> **cwmoser said:**
> Is it "safe" to click on the Software Updates Icon and Apply All updates?


Absolutely. In fact, it's safer to update than not, since updates often include security and stability fixes.

> 
Are "updates" something that we all should learn how to carefully review -- or are updates so reliable that we just "apply all" that we are offered?


I've never had any major problems applying updates. In the unlikely case that an update causes breakage you should submit a bug report. The maintainers of that package will most likely issue another update fixing the problem within a few days at most.
