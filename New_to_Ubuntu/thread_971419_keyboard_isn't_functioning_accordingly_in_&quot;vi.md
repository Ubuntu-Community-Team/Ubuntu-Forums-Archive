### Post by Sorivenul on 2008-11-05
Are you trying to learn Vi(m)?  Or are you just trying to edit a text file in a non-graphical editor?

If you are trying to learn Vi(m), congratulations and good luck.  It is a challenging but rewarding experience.  However, it is *not* a beginner's editor.  Graphical editors like "gedit", "kedit", and "mousepad" also exist and can make things easier for new users.  The keyboard "insert" key would not work as expected due to Vi(m)'s keybindings. Vi(m) works in modes, more specifically and an edit/insert mode, and a command mode.  

I have included a list of links at the end of my post to help you with Vi(m) specifically.

For something more simple, but still non graphical the program "nano" is included, and is much more straightforward.

### Post by Sorivenul on 2008-11-05
> **rocky463 said:**
> does this have any thing to do with the key board type that we select while installing UBUNTU..........
Where are you, and what keyboard type did you select during install?

Also, are you on a desktop or a laptop?  Some of the specific laptop keys need to be mapped by the user to function properly, as many of the keyboard types available don't include the specific functions available on laptop function keys, etc.  For example, my F2 key is supposed to also mute my volume, but this function was not available by default. If you are on a desktop, the odds are your Vi(m) problems have very little to do with your selected keyboard.

Hope this offers some insight for you.
