### Post by ankspo71 on 2010-07-23
Hi,
I think it would be helpful to her if you could create a new user account for her in your computer, and spend a day or two showing her how easy it is to do things you already know how to do. This would give her a little experience so that she isn't totally unprepared for using Linux. If she doesn't like using Linux, this would prevent installing Linux on the new computer.

Ubuntu and LinxMint I would recommend equally. PCLinuxOS is nice too, because it will install 3rd party graphics drivers (and possibly other drivers) automatically during the installation, which makes that step easier. The thing with PClinuxOS is though, is that it is not based on Ubuntu, so there will be differences between your's and her's computers, and she won't have access to the same large repositories as you do. (the main reason I choose to stick to *ubuntu)

Try to make sure she gets a computer with Linux compatible hardware too (if possible), if she plans on using Linux. You can see what people had to say in their reviews over at [http://ubuntuhcl.org/](http://ubuntuhcl.org/) They have reviews for Desktop and Notebook computers as well as individual hardware.
Hope this helps.

### Post by spydeyrch on 2010-07-23
I would recommend a few options:

-You can make a live cd/usb of both ubuntu & mint to see which of the two she likes more. My wife liked mint more than ubuntu because it seemed more ‘windows-ish’ than ubuntu. One of the major differences between ubuntu and mint is that mint comes pre-packaged with codecs for audio and video. So no need to get those installed. Plus mint looks more ‘crisp’ than ubuntu right from the beginning.


  -If you have to make the decision about which to go with, go with mint. Almost everything that runs on ubuntu will run on mint.


  Something that you should be aware of and make sure that your sister is aware of, windows programs won’t run on linux ……. unless you go through wine, which has its pluses and minuses. So, if she is dependent upon any windows programs (iTunes, etc), then she needs to realize that she won’t be able to run those natively in linux. But……. You have a few options other than wine.


  -You could do a dual boot. This would allow her to run both windows and linux and, hopefully, slowly help her transition over to linux completely.
  The only issue that I see with doing a dual boot is that you said that you didn’t want to have to help manage and maintain her windows install (i.e. viruses, etc.) Soooo…….


  -If you machine has the capable hardware, you could install linux as the only OS and run a windows VM in linux. This would solve two things: she would have access to windows and you could take frequent snapshots of the VM. This means that if it gets infected with a virus, you just revert back to the previous snapshot and viola! No more virus!!


  The last option, and probably the best in my opinion, that you have is teach her how to maintain and manage her own computer, whether it is windows or linux, so that she doesn’t have to rely upon others. Then just install windows if that is what she really needs/wants.
