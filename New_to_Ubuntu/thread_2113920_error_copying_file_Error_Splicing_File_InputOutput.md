### Post by 3rdalbum on 2013-02-10
> **chadwell said:**
> Any way to try and salvage this situation?

Yes. Buy a new hard disk as soon as possible. Copy the data from the old hard disk to the new hard disk, as soon as possible. Some files will be damaged already, probably; that's what the Input/Output error tells you, that spot of the disk is corrupted.

The old hard disk is dying and if you keep trying to use it, you'll kill it before you've salvaged your data.
