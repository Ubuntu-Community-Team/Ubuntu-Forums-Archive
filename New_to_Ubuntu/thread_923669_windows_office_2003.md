### Post by Therion on 2008-09-18
> **Jazzy_Jeff said:**
> Why do you need MS Office??
That's a question I try not to ask when I see someone who wants to use a MS app in Ubuntu, but I'm going to have question it in this particular instance as well. 

If you want to run apps like MS Office instead of using a native linux application instead, I might suggest that you will be better off actually *using* Windows; or dual booting Windows for your Windows apps if you're determined to use Ubuntu/Linux.  

There are so many excellent applications that are native to Linux that will allow you to interface with your MS Office using peers that I really don't see the point of installing a huge app like Office and then running it in an emulator.  And yes, I know what "WINE" stands for.  Don't go there.
