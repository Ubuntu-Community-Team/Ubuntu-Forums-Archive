### Post by pjasnos on 2011-03-07
You don't need to, just unplug.

You only really need to 'safely remove'/unmount stuff that stores data (USB drives, memory cards) and some advanced devices with moving parts that need to know they are going to be unplugged to put those parts in safe position. Most of the equipment an average person uses with the computer doesn't need any particular 'safe remova'.
