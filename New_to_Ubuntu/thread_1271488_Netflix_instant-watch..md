### Post by Temposs on 2009-09-21
The problem is that Netflix bases its movie player on Microsoft's Silverlight platform, which obviously only works in Windows.

As tom957 above mentions, there is a project called Moonlight that aims to implement Silverlight compatiblity in Linux. They've done good work but it is not up to speed with Silverlight yet.

The botoom line is, if you want to watch Netflix movies with their web-based player today, you have to run Windows.
