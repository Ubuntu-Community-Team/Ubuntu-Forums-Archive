### Post by Don Bowen on 2010-04-06
When you went through the install process, it asked you for your First and last name, and a password.  From that it created an account for you to use to login.  My name is Don Bowen, so it created an account named "don" and used the password I told it to use.

It might be a good idea to re-install and watch for the username/password moment.  Note what you type when it asks.
When it's done installing, it will force a reboot and then display the logo, the ubuntu name, then the user account it created for you.  Hit Enter to select that user, and then type in the password you made for it.

Please let us all know if this works, so that we can help more as needed.
cheers,

Before you reinstall, make sure that the CD has not corrupted during download / burning.

You need to check the "md5sum" and then also use the "Check CD for errors" when you boot.

If these things don't make sense - try Google or ask back here.  There have been lots of reports previously from people who got a corrupted download that asked for a password when booting from the CD (which should not happen).
