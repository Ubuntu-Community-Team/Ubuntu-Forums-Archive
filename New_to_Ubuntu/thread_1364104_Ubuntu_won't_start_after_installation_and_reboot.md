### Post by drs305 on 2009-12-25
Welcome to Ubuntu and the Ubuntu forums mikaratsel.

Your specs meet the *miminums* to run Ubuntu but the specifications site recommends installing via the alternate CD in these cases. The absolute minimum is 64MB but recommended minimum is 384MB.

There is another variant of Ubuntu called Xubuntu that doesn't require quite as much memory. Recommended minimum for Xubuntu is 256MB. You might try installing Xubuntu, but first visit this page:
[https://help.ubuntu.com/community/Installation/SystemRequirements]("https://help.ubuntu.com/community/Installation/SystemRequirements")

There are other Linux distributions that require even fewer resources, a quick search should reveal what is currently available. I haven't used any of the non-Ubuntu distros but others on here can probably tell you the names of some of the most popular ones and their experiences with them (such as Puppy and DamnSmallLinux)

### Post by Geoff918 on 2009-12-25
DSL was a bit too minimalist for my tastes. Puppy is pretty darn good, and still looks visually appealing.

You may want to try Masonux.

Masonux is Ubuntu with LXDE loaded. It's one man's project remastering the Ubuntu distro for himself. With the release of Ubuntu 10.04 is scheduled a light-weight release called Lubuntu which also runs the ultra-slim LXDE desktop. I personally look forward to it.

In the meantime, if you're set on Ubuntu--and it is one of the best supported distros as a result of a heavy user base, these forums, etc.--you may want to use Masonux.
