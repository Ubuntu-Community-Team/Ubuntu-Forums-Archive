### Post by drs305 on 2011-12-01
> **NJames99 said:**
> What I'd really like to have is a launcher/script that reboots into Windows, and going from Windows to Ubuntu is easy since Ubuntu is the default.

OK. I see what you want and your 'grub-reboot' idea probably would do what you want. Once you start using Grub 1.99 you are going to have to deal with submenus that may mess up the numbering of grub-reboot.

I should have mentioned this earlier - rather than a number you can use the exact title in the grub-reboot command. You use the entire title between quotations from the grub.cfg menuentry. That will protect your script from menu changes.

Example:
```
sudo grub-reboot "Windows 7 (loader) (on /dev/sda2)"
```
