### Post by suomalainen on 2010-11-01
# For icons within a Nautilus directory window, select the icon and choose Edit > Stretch Icon from the Nautilus menu bar.
# When you select Stretch Icon, four blue dots appear on the corners of the icon.
# Grab one of the blue dots (left mouse button) and drag it. This will stretch or shrink the icon.
