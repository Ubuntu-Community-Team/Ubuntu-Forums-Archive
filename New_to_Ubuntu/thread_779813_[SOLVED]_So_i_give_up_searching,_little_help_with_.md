### Post by shae on 2008-05-03
Do not try to use Envy as the 9250 is not supported by ATI's closed drivers.  However, it should be supported by the open source drivers.  In your xorg configuration file, replace where it says "vesa" with "radeon".  If that does not work, try "ati".
