### Post by ExileAmongYou on 2011-07-19
Ubuntu has shifted to using "global menus", similar to the way Mac OSX does it. When an app isn't fullscreen, the menu is still in the top panel (the same place it is when the app *is* fullscreen). If you move your mouse over the window's title in the panel at the top of the screen, you should see the menu appear.

### Post by beew on 2011-07-19
If you hate the global menu like I do you can get rid of it altogether by uninstalling the package indicator-appmenu from Synaptic. After that all the menus will return to the application windows.
