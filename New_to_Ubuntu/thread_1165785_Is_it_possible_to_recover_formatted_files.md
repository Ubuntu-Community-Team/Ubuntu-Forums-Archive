### Post by amingv on 2009-05-21
I have been able to recover files before using [Photorec]("http://www.cgsecurity.org/wiki/PhotoRec"), it's pretty easy to use, too, just:

-Don't recover to the same partition the lost files were in.
-The less you use the drive/partition you were in, the more chance to recover the files (and better the integrity they're likely to be recovered with).
-Notice the recovered files won't be named the way they used to be (but will likely keep the file format), so you may need to search for them amongst lots of other files.
