### Post by snowpine on 2010-10-09
I would use Netbook on a monitor less that 10", standard Ubuntu if it's bigger than 10". But I would encourage you to try both and decide for yourself. Some people like the Netbook interface on big screens. 

Both versions are equally well optimized for the Atom processor; it is unlikely you'll see a big performance difference between the two.

### Post by P4man on 2010-10-09
There used to be a netbook remix which used a different compiler or compiler flags (I think) to produce code optimally suited for atoms that would supposedly increase battery life. You had the regular x86 and AMD64 versions, and then also one called LPIA (low power intel architecture)

But that was more an intel sponsored publicity stunt than anything else. Its been shelved now, and the netbook remix uses the same code as the other x86 and AMD64 ubuntu versions. After all, Atom is just another x86/AMD64 CPU.
