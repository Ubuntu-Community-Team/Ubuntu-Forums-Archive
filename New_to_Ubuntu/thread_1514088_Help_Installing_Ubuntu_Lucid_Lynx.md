### Post by TBABill on 2010-06-20
CD vs. DVD should make no difference. I burn CD iso's to DVD regularly, depending on what I have on hand. The machine cannot tell the difference unless you have only a CD drive and not a DVD drive. 

You could have errors on the disk. Did you verify the download using md5sum in a terminal? Did you burn at the slowest possible speed? Does the CD/DVD work when you boot into live mode? If not the culprit is either the downloaded file or the disk itself.
