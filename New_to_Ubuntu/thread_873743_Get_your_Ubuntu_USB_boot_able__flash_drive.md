### Post by Egyptix on 2008-07-29
now you can get boot able flash stick as Mandriva but using Ubuntu
all you need is Ubuntu CD (in iso format) + 1 GB USB flash drive + Fedora Live USB-Creator([download here]("https://fedorahosted.org/releases/l/i/liveusb-creator/liveusb-creator-2.7.zip")) or Ubuntu Live USB-creator ([Page here]("http://klik.atekon.de/liveusb/"))

:KS
Note: Fedora creator is designed for Fedora 8 or 9 but it works will with Ubuntu,Xubuntu but it doesn't work with Knoppix or mandriva(I tried all of them).it's more stable than ubuntu live usb creator but the little bad thing :( is that it works under windows only.
Enjoy Live USB stick.:lolflag:
