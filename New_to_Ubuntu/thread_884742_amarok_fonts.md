### Post by jordanmthomas on 2008-08-09
You should be able to run either qtconfig or kcontrol (if it's installed) to select a new font for Qt programs like amarok.  Your gnome font preferences mean nothing to amarok.

A font with good multilanguage support you should already have installed is Deajvu {Sans, Serif, Monospace, etc}.

Finally, I would not suggest using amarok2 yet as it's still very much not done.
