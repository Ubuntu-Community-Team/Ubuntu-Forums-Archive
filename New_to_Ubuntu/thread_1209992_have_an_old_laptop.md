### Post by Martin Marshalek on 2009-07-11
I tried a similar attempt on my old Toshiba T4700CT with pretty similar stats but the issue was not even the ram so much as it was the hard drive space. It only had about 400mb of hard disk and with floppy only, I was severely limited. Also, it became difficult finding a distro that could be split easily, put on a floppy, and then recompiled in Windows 3.1. However, if your computer has Windows 95, you're in luck, because 95 supports usb, which could be used to add a flash memory or ever a CD drive.
