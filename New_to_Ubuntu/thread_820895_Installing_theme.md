### Post by sayakb on 2008-06-06
If its a GTK theme, goto System->Preferences->Appearance and drag n drop the theme on the Appearance Window to install it.
If it's an emerald theme, extract it, goto System->Preferences->Emerald theme manager, press Import and navigate and select the *theme.emerald* file.
