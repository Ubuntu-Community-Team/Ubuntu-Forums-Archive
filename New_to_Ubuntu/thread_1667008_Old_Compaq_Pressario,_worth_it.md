> The specs are, AMD Semperon 2800+ 1.59GB, 640MB Ram, 40gig hard drive. 

This machine would definitely run Ubuntu, but you might consider Xubuntu (which uses the lighter XFCE environment) or Lubuntu (which uses the even-lighter LXDE environment).

> 1: Will there be drivers avaliable on Ubuntu for this laptops systems (the wireless is a must)

Nothing to do but to test, though I would assume that Ubuntu/Xubuntu/Lubuntu would "just work" on your machine, including wireless.

> 2: Will it run well. Th eonly thing i need it for is word processing, PDF reading the occasional movie watching and playing NDS roms on desemu.

Word processing and PDF reading for sure.  Not so sure about the video capabilities.

> 3: will desemu (Nintendo DS Emulator) and Firefox run on it?

I'm not familiar with desemu, but I would not be surprised if it does.  You can also run many Windows programs using wine (an application layer that "fakes" a Windows environment).  Firefox comes standard.

> 4:what version of Ubuntu ( as there seems to be be many) should i use and where can i get it, links if possible please.

Personally, I would try Xubuntu ( [http://www.xubuntu.org/](http://www.xubuntu.org/) ) or Lubuntu ( [http://lubuntu.net/](http://lubuntu.net/) ).

Whatever you use, it will feel like a brand-new computer after running WinXP!
