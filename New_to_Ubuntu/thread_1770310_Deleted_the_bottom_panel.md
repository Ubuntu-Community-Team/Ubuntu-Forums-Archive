### Post by Macskeeball on 2011-05-29
[quote="[Gnome Library's user documentation](http://library.gnome.org/users/user-guide/stable/gospanel-10.html.en)"]To add a panel, right-click on a vacant space on any panel, then choose New Panel. The new panel is added to the GNOME Desktop. The new panel contains no objects. You can customize the new panel to suit your preferences.[/quote]

That will create an empty bottom panel, so you'll still need to add the trash (recycle bin), window list, etc. The steps for that are here: [http://library.gnome.org/users/user-guide/stable/panels-addobject.html.en](http://library.gnome.org/users/user-guide/stable/panels-addobject.html.en)
