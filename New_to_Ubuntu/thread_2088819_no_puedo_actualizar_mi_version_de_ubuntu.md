### Post by Cheesemill on 2012-11-27
From what I can gather you are having problems connecting to the 11.04 repositories.

This is because 11.04 has reached end-of-life and no longer receives any bug fixes or security updates. One of the results of this is that the repositories are taken off-line and moved to a different address.

For more information Google 'ubuntu eol', but you should be prepared to upgrade your OS to a supported version as running an unsupported OS could result in you easily being hacked due to the many vulnerabilitys that have been discovered since you stopped getting updates.
