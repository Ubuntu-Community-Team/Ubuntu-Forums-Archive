### Post by coffeecat on 2013-12-01
This should put you right:

[http://www.psychocats.net/ubuntu/resetpassword](http://www.psychocats.net/ubuntu/resetpassword)

However, you haven't said which version of Ubuntu you are running. That shouldn't make any difference to the psychocats instructions, but with a second hand machine you may have an obsolete version of Ubuntu. What version is it? If you don't know and you can open a terminal, the output of this command will tell you:

```
cat /etc/lsb-release 
```
