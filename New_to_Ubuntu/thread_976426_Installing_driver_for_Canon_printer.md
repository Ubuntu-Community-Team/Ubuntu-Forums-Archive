### Post by Shwefty on 2008-11-09
[http://www.openprinting.org/show_printer.cgi?recnum=Canon-LBP-660]("http://www.openprinting.org/show_printer.cgi?recnum=Canon-LBP-660")

CUPS is the Linux printing utility, and it is awesome.  It apparently has your printer driver in there, and it works.  I just go to System: Administration: Printing -> New

It should autoconnect, pull in drivers and such, but it's been a while since I've tried.  If doing that doesn't work, I'm sure theres a command line way to do it too.
