### Post by lobralleo on 2010-11-23
There is a number of good text editors and development interfaces for C and other programming languages available by default in Ubuntu.

If you are looking for a text editor with syntax highlighting, you can use Gedit (default in Ubuntu) or Kate (in Kubuntu); both programs have a lot of useful tools and shortcuts to make programming easier.
If you need a more complete development environment, you could try Eclipse.

There is much more in the repositories, at every possible level of complexity, but this is probably the easy way to start.
