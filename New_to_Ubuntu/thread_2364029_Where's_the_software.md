### Post by grahammechanical on 2017-06-17
I do believe that the Ubuntu Software Centre was replaced by something called Gnome Software. Which in turn was rebranded and called Ubuntu software. The launcher should have an icon for it. It is an orange shopping bag with a stylized letter A on the front. If it is not in the launcher, then search for "software" in the Dash and the Dash should show you Software Updater, Ubuntu Software, Software & Updates & Additional Drivers.

Launch Ubuntu Software & the Utility should load with an icon in the launcher. Right click the icon and select lock to launcher.

Regards.
