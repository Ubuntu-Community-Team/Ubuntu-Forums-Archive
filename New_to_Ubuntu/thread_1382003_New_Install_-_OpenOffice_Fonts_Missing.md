### Post by duanedesign on 2010-01-20
Bitstream Vera has been replaced with the package ttf-dejavu. DejaVu provides an expanded version of the Vera font family aiming for quality and broader Unicode coverage while retaining the original Vera style.

The comments of [this guide]("http://74.125.155.132/search?q=cache:QYQAWCDUAsQJ:maketecheasier.com/turn-your-ubuntu-intrepid-into-mac-osx-leopard/2009/01/08+ubuntu+mac+theme+Bitstream+Vera+Mono+Karmic&cd=1&hl=en&ct=clnk&gl=us&client=firefox-a") by Karmic users seem positive. If you read the comments (starting November 9 2009) they mention the missing fonts.
