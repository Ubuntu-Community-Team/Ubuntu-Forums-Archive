Vidalia is crossplatform so you can use it for linux. The last option on their download page for installing from source is your best bet. I have no idea what your software is, but it is likely that you will need to add some packages to get it to work. I always install the build-essential package because it has alot of the tools you need to compile things from source. 

Check here for direct instructions on Ubuntu linux! [http://trac.vidalia-project.net/wiki/InstallSource#UbuntuKubuntuXubuntutestedon6.067.04and8.10](http://trac.vidalia-project.net/wiki/InstallSource#UbuntuKubuntuXubuntutestedon6.067.04and8.10)
