### Post by freebeer on 2008-08-02
The first thing you want to check is that your copy of Ubuntu is a good one.  [https://help.ubuntu.com/community/HowToMD5SUM]("https://help.ubuntu.com/community/HowToMD5SUM")

If that checks out, you might want to try the Alternate CD instead.  Sometimes, with the graphical installer, it doesn't recognize your video card hardware properly... which would explain your problem.  (Or so it seems to me)  The Alternate CD is a text-based installer.  Once you're installed, you can then work out which video driver you need.
