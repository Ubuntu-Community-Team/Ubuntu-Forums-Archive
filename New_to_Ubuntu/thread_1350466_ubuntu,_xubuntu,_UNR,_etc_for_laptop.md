If it is a Dell 1100 (which I think you meant as I had one once and specs are same) I think it can go to a gig of ram.  There are 2 slots, you can check the site to be sure.  It has shared video so unless you put in a separate graphics card the ram you have is reduced for graphics.  

I would get at least 512 megs of ram and see about getting a pci graphics card (you can get a relatively newer version of an nVidia card relatively cheaply) as that would help out a lot, especially if you want some effects on the desktop.
