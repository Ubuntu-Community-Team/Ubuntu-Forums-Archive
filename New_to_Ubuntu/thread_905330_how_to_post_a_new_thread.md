### Post by drs305 on 2008-08-30
Welcome to the ubuntu forums. Here are some techniques:

When you make a post, please use a descriptive title (as you did this time). "Need Help" just doesn't help readers...

When your question or problem is answered/resolved, please mark the thread SOLVED using the "Thread Tools" link near the top right of the first post. You can always unmark it if things regress as well.

Most of the questions on the forum have been asked before. Don't be afraid to use the Search function and key words to see what's been written about your problem previously. It helps to add keyword "solved" if you are getting lots of hits in your search.
