### Post by grahammechanical on 2012-02-21
> I was not expecting those menus outside the window on top of the screen.

The purpose of this is to make the window frame smaller to allow more space inside the window frame for content. 

It does not take much to get used to this way of working. If you press F10 you will get access to all those menus and you can use the arrow keys to move along the panel.
