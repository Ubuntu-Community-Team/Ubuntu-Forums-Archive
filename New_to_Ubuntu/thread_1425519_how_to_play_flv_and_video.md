### Post by zeroseven0183 on 2010-03-09
Have you installed **ubuntu-restricted-extras** already? If not, type in a Terminal:
```
sudo apt-get install ubuntu-restricted-extras
```Note: You can also copy and paste the command

After that you should be able to view the FLV video file on your default player.
