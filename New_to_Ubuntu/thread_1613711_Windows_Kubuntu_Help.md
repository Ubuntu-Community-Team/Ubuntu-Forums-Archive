### Post by anewguy on 2010-11-04
+1 - I suspect you did the install and didn't manually partition the disk.  Taking the default means that the ENTIRE disk will be used for Ubuntu.  It also means it created 1 swap partition and 1 partition for everything else.  By default, the partition for everything else is formatted.  So what it means is yes, if you did an actual install and took the defaults, Windows is no longer on your disk.  If your disk had a recovery partition so you could recover your OS in case of failure, that will have been wiped out as well.  No way to get it back (well maybe, but we're talking a LOT of money through one of the companies that specialize in data recovery).

I would tease you about backups, but the truth of the matter is that a couple of releases ago I didn't backup before installing the new release (I always delete the old and then install the new, rather than just doing an update).  Sooooo......I can't tease you there (but somebody else might mention it ;) ).

If indeed this is what you did and you didn't mean to, I'm sorry that happened.
