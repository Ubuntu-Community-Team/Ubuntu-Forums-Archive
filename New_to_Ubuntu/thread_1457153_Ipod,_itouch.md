### Post by Paddy Landau on 2010-04-18
Sorry, iTunes was carefully designed to work only on Windows or Mac.

You can install [PlayOnLinux]("http://wiki.winehq.org/PlayOnLinux"), and install and older version of iTunes (version 7), but that's going to be your best bet on Linux and you may find it a little buggy.

If you don't want lock-in, then avoid Apple products.

Your alternatives are:

[LIST]
[*]Purchase and install Windows for dual-boot.
[*]Purchase and install Windows in Virtual Box (if your RAM is big enough, and your CPU fast enough, to cope).
[*]Purchase the cheapest Windows machine that you can find.
[*]Return your iPod and iTouch to the store where you bought it and purchase a different model, one that doesn't lock you in.
[/LIST]

```
gksudo update-manager -d
```

Then type your password. The update process takes anywhere from an hour or two depending on the speed of your computer and internet connection. 




Both methods will allow you to sync your iPod with a program called Rhythmbox, which is installed by default in Ubuntu.

### Post by geodanny on 2010-06-24
I have the latest generation iPod Touch (well, at least it was the latest generation when I got it in March). I do not recommend an iPod for anyone wishing to run Linux only. Apple hates and is threatened more by Linux than Windows. You need Windows or Apple to activate the device and anytime you wish to update the OS. If it craps out on you, such as if the database is corrupted, you will need to run to an Apple or Windows machine to fix it. That is, in my experience at least. 

I updated my iPod to OS4 today (used Windows). The upgrade broke my ability to use it with Ubuntu. That was easily resolved.

In a terminal, run:
```
sudo add-apt-repository ppa:pmcenery/ppa
sudo apt-get install libimobiledevice-0 python-imobiledevice
sudo apt-get update && sudo apt-get upgrade
```


ppa pmcerney/ppa is the repository for libimobiledevice. It will let you get the latest version of packages (the ubuntu version available in package manager is older)
See:
[http://www.libimobiledevice.org/]("http://www.libimobiledevice.org/")

You may need to run Synaptic Package Manager to install libimobiledevice-0 and python-imobiledevice  

Note: Do not install other packages from that repository. You only need and want the libimobiledevice ones. Update manager will give you the option to update gvfs, ifuse, and other packages from that repository. :-) 

You can see the packages in Synaptic Package Manager if you search for ipod.

Also, I have Lucid (10.04) running and use Rhythmbox (You may need to activate the ipod plugin manually).
There are options if you use earlier versions of Ubuntu. For example, if you use 9.10, see: [http://fatbuttlarry.blogspot.com/2010/01/ipod-touch-iphone-3g-ubuntu-910-in-5.html]("http://fatbuttlarry.blogspot.com/2010/01/ipod-touch-iphone-3g-ubuntu-910-in-5.html") There are others out there. They generally use the same packages though.
