### Post by PaulW2U on 2018-05-16
> **sathees24 said:**
> how to download and install WebEx network player. Any bdy Please give suggestion.
The Webex website clearly shows that only Windows and Mac OSX are supported. Is [WINE]("https://www.winehq.org/") of any use to you here? It's available in the Ubuntu Software Center and is able to run some Windows programs but to varying degrees of success. If you search their online database for Webex it does produce some very old results so you might want to study those to see how successful previous attempts to install the Webex player have been in the past.

If using Webex is so important to you I would also contact their support to see what they have to say. They'll probably say they don't support Linux but ask if they have any Linux users who have managed to work around that. Good luck.
