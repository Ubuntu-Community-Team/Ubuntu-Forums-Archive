### Post by 4Orbs on 2009-06-02
A suggestion, assuming your desktop effects are working correctly:
1. Install the CompizConfig Settings Mgr (it's in Synaptic)
2. Install the Fusion Icon (Synaptic)
3. Start the Fusion Icon and the icon appears on your panel.
4. Right click the icon and select Emerald window decorator and select Compiz window mgr.
5. Right click the icon and select Emerald Theme mgr and choose a theme
6. Download a new Emerald (Beryl) theme from gnome-look.org and save on the desktop. If it's packed in a zip file, extract it to the desktop.
7. Use the Emerald Theme mgr to import the new theme from your desktop.
8. Activate the new theme in the Emerald Theme mgr.
