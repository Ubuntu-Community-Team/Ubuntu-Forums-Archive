### Post by ajgreeny on 2016-09-01
As this was an installation on a USB 3.0 external disk I suspect that the freezes you see are a result of the I/O speeds of using a USB, and bottlenecks in moving the data needed to do whatever you are using Ubuntu for, particularly if that is a high resource job.

Is it a full USB installation, not a live persistent system that you are using?
Why are you running on an external disk and not a full install to SATA internal disk either as a dual boot or as a sole OS? It would, I believe, make the OS run much more smoothly than on a USB.
