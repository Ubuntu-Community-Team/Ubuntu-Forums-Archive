### Post by tjwoosta on 2008-05-19
after you have the theme in the emerald theme manager, all you have to do is this


press alt-f2  then type 
```
emerald --replace
```
and hit enter

to make this command run every time you logon you will have to go to System-Preferences-Sessions then click ADD and put the command i gave you above in the command box (whatever you want in the other two)
