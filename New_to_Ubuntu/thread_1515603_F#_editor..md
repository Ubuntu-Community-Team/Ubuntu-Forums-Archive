### Post by SKLP on 2010-06-22
There will probably not be very good F# tooling for linux unless it becomes open source (like MS has said they plan to do at some point).

Currently, Gedit can do syntax highlighting if you manually select the language in the menu bar (it cannot infer that it is F# from the .fs extension). Also there is a F# plugin for monodevelop in early stages, which I haven't tried.
