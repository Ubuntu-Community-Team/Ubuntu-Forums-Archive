### Post by grahammechanical on 2011-02-21
The top panel is that thin strip along the top that has on the right an Ubuntu icon and the words Application, Places, System and  a Firefox icon. Take the mouse over that strip of screen and right click. On the drop-down menu that appears you will see Add to Panel at the top of the list. Select it. Then look for Notification area.

You will need to reboot. If Network Manger is one of your Startup Applications, and it so be, then the network icon will appear. Right click the network icon and make sure that you have a tick mark against Enable Networking and Enable Wireless. These line act like on/off switches.

Is that more clearer?

Regards

P.S. If you want to do it through a menu do not go to system>Administration (Indstillinger, in Danish, I am guessing) but System>Preferences>Network Connections. The icon along side is two small computer screen with purple screens.

The Network Manager icon which appears on the top panel is two arrows in opposite directions when connected by ethernet and four conical arcs radiating upwards when connected by wireless. If you do not have any connection, either by cable of wireless, then there will be a red exclamation mark against the four conical arcs.
