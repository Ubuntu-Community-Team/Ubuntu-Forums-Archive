### Post by pmohankumar on 2013-04-16
Hi,

I just changed the default port in following files for xampp apache and mysql.

/opt/lampp/etc/my.cnf (mysql)
/opt/lampp/etc/php.ini (mysql)
/opt/lampp/etc/httpd.conf (apache)

The issue solved.
