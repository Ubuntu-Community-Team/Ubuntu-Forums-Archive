### Post by Paqman on 2010-01-16
> **RLSumner said:**
> What's a good media player? I download a lot of stuff, I want one player that will do it all. From every stupid little video format to my mp3s.

I like Banshee personally, but everybody has their own favourite.

>  What's the best firewall? Anti-virus, Anti-everything else? 

You already have the best security of any desktop OS built in. Security on Linux is part of the system, you don't need to cobble it on afterwards like you do on Windows. All the ports on your machine are secure by default, there's no need for a firewall. And if you stick to the repos for your software and stay patched up to date you don't need antivirus.
> 
I've been to a few websites and the download didn't work.

You don't need to go to websites to get software, it's all available on your desktop. Check out Applications > Software Centre, it''l automatically download and install software from Ubuntu's repositories (which are enormous), and it'll keep them all updated for you too. There's a more in-depth package manager at System > Admin > Synaptic.

> I don't really play a lot of games, if there's a way to play windows based games it would be nice.

The easiest thing to do is keep a Windows partition, and dual-boot. Windows games work best on a proper Windows system, although some can be played on Linux using a compatibility layer called Wine.

> 
Oh yea Yahoo Messenger, how can I make that work? 

The default IM client (called Empathy) can connect to just about any IM network, including Yahoo, MSN, Google Talk, etc.
