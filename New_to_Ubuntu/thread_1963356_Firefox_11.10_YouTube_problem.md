### Post by lovinglinux on 2012-04-23
If you are using dual monitor, then is a known issue of flash, that maximizes only to the size of the smaller monitor.

There are many threads about this, but I don't remember a definitive solution. Perhaps changing the monitor order might help.

> **markbuntu said:**
> By default flash full screen opens at screen location 0,0 which is the top left corner. If the left display is smaller than the right then full screen will be larger than the display. There has been bug reports on this for over a year now but adobe does not seem very interested in fixing it. The only thing you can do is put the larger screen to the left.
