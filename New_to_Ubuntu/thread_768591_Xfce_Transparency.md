### Post by Niniel on 2008-04-26
xfce and compiz?
I don't think you need compiz, xfce has its own built-in compositor that handles those things. 
You can set transparency from the xfce control panel, I think the item is called Window Manager Tweaks, go there and then to the Composit tab.
