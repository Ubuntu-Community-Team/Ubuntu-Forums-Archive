Just in case you use skype and have not noticed, there is a new version available.  It is labelled as a beta release and there are 32 and 64 bit versions available.

I have installed and tested the 64 bit version and my initial feeling is that this is a very good step forward.  No confusion with choosing sound devices to get it to work.....  works "out of the box".

Details at this link
[http://www.skype.com/intl/en/download/skype/linux/choose/](http://www.skype.com/intl/en/download/skype/linux/choose/)

The download is a .deb so that is easy but you do need to uninstall the existing version first.
