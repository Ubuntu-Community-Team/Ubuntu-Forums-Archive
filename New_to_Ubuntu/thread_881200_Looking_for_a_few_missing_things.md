### Post by spiderbatdad on 2008-08-05
You should be able to rename any file or directory in your /home directory with right click. You should not try to rename root owned files.

Not sure what you're looking for in the way of "screencaps." You can take screenshots with the tool in Applications>>Accessories. You can even add the tool to your panel. If you want to record your screen activity, there is gtk-recordmydesktop in synaptic package manager.

### Post by deathvalleyjoker on 2008-08-05
IF you have Compiz Simple settings Configurations app installed, there should be a utility in there called "screenshot" i believe. I enabled that, and went into the settings of it and gave it a keyboard shortcut of super+leftclick, so all i have to do is hold down super key and drag an outline box around the screenshot i want, then when i relise the mouse, it saves it right to the desktop as a jpeg i believe. 

As far as taking a ss in a full screenmode, i am not sure. But that screenshot utuility under compiz is def cool and i use that over the pre-installed app.
