### Post by namdung on 2009-01-01
This is where Ubuntu is different from distros like Fedora, which have several desktops pre installed.

Ubuntu has different versions viz. Kubuntu, Xubuntu,UbuntuStudio etc. keeping in the requirements of different user groups. However, Ubuntu allows users to install different versions.

Personally I don't recommend this as performance could be an issue. 
If u're unsure, first try out the different versions in Virtualization and then stick to one as there are subtle differences in each versions that takes time to learn.

However, if ur machine has the mojo, go for it.
