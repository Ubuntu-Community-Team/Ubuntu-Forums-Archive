### Post by spupy on 2008-07-21
Aurora with animation is pretty good. But I would recommend Murrine, because:
1. It has a graphical theme configurator
2. It is **twice** as fast as Aurora.

Yes, different theme engines have different speeds. And it is noticeable. Aurora looks good, but it is the slowest one. Clearlooks is also nice, because it looks good and is not pixmap based (means it's speedy).

### Post by MatthiasHeil on 2008-07-24
I like "Freezy Caramello" very much. Installation is a breeze thanks to Deb packages at [http://www.gnome-look.org/content/show.php/Freezy+themes+pack?content=80899](http://www.gnome-look.org/content/show.php/Freezy+themes+pack?content=80899) (includes engine) - if you cannot get used to the unusual window borders, combine "Freezy Caramello" with the more Vista-ish "Clearlooks Blend" at [http://art.gnome.org/themes/metacity/1346](http://art.gnome.org/themes/metacity/1346) - as for wallpapers, "Webilder" is a fascinating tool you might wish to check out: [http://www.getdeb.net/app/Webilder](http://www.getdeb.net/app/Webilder) - and, finally, the best system font I've found up to now is "Glisp" from [http://www.gnome-look.org/content/show.php/Glisp+TTF?content=80129](http://www.gnome-look.org/content/show.php/Glisp+TTF?content=80129) (which looks best at 9 pt size IMHO) - check the attached screenshot if you're interested!
