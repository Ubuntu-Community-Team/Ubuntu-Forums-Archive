### Post by themusicalduck on 2010-07-31
It's unlikely ubuntu will run well on 256mb ram. I tried installing it onto that much ram once, and the installer wouldn't even run. You might get it to install with the alternate installer, but it won't be fast.

You could look into using Lubuntu or another lightweight distro. (Apparently Puppy Linux is good).

For the wireless adapter, hopefully it'll work straight off, but you might need to download the drivers for it if it needs, so you'd need an ethernet cable connection for that.

BTW, you cant upgrade your RAM to 1GB, 512MB is as much as your motherboard chipset supports.
