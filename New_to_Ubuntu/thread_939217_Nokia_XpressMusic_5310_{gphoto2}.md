### Post by heroes_on_a_half_shell on 2009-08-03
I know this is an old thread, but I found it whilst having this problem before discovering the simple solution on my own.  As such, I feel the need to state it for others.

The fix is actually in the phone:

Settings-> Connectivity -> USB Data Cable -> Data Storage

Enjoy
