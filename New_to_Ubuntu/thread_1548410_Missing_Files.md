### Post by jtarin on 2010-08-09
Do you have system restore (in Windows) turned on for that drive? If yes I would turn it off. Check you disk sapce from Windows and Linux. I get those messages every now and again but I have plenty of space on that drive.....about 30%. If you have deleted any files on that drive from within Linux go into Windows and you will see a file or two labeled Trash.000 or similar. Just delete them as it only takes space....but if you have something valuable that got deleted that might be where it is. Be careful writing to that disk as you might overwrite your lost files before you have a chance to recover.
