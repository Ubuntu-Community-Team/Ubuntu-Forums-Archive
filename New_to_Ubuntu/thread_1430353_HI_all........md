Hello and welcome, 

The question "which distro should I use" can only really be answers by you. Sure there are many many different distros and some are voted better than others, but at the end of the day I believe the best distro is the one that works for you. 
You say you're using fedora 7 in your course. I used to use fedora before switching to ubuntu and it was a great distro and it worked well for me. My reason for switching is because I tried out ubuntu in a virtual machine and liked it, so I installed it and kept it. As for lastest releases, should you decide to go with fedora or ubuntu, will be ubuntu 9.10 or fedora 12. Both are stable.

### Post by Kevbert on 2010-03-15
I suggest you get a Windows laptop as many of the packages such as MS Office and Visual Studio are commonly used in the outside world. Once you've got this laptop you can then set it up to dual boot with Linux (I believe most servers even in Microsoft are Unix/Linux based). The advantages of linux are most software is free and so is support through forums such as this and in some cases (but not all) programs can be run via WINE in Linux.
My prefered flavour of Linux is Ubuntu as it's easier to configure. Fedora is not so easy to set up (I'm currently looking at V12 - Constantine).
The laptop should have a good size hard disk (say at least 300Mb), Nvidia graphics, Broadcom or Intel wireless.
