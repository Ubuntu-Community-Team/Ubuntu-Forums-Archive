### Post by d4m1r on 2012-02-12
Ideally, you should be installing Ubuntu from a boot medium (CD or USB) before Windows XP or any other OS is loaded.

If you do have access to remote desktop, maybe download Ubuntu to that PC and try to install it over remote desktop? I know remote desktop is pretty limited in terms of graphics and performance but Ubuntu does have a "install in Windows" option so try it.
