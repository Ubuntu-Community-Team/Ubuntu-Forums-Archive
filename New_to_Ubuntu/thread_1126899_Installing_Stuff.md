Be welcome to linux :-)

Could you please be a bit more precise about what "stuff" would you like to install? 

As for firefox 3.1, you have choose to have a hard start in linux. You have to dowload from athe tarball (a form of not ubuntu specific file which usually involves compilation, among other things) from here [http://www.mozilla.com/en-US/firefox/all-beta.html](http://www.mozilla.com/en-US/firefox/all-beta.html)

Let me give you a small piece of advice: FF 3.1 is still beta. To test beta software is not the most advisable thing for a newcomer. Also, as a long term windows user, you have probably the habit of picking and installing anything you find over there. One of the points that make linux far more secure than other more extended OSes is the good habits of the users. This will IMHO be more important in a near future in which linux will have a wider user base. 

Regarding our working machines, many of us only install software only from the official repositories, or from well known and trusted sources (once we know what we are doing). We do test beta software, sure, but in our secondary machines. Using a beta browser in your main machine exposes it unnecessarily to security threats or instabilities. I'd wait at least to the main release of FF 3.1 ...
