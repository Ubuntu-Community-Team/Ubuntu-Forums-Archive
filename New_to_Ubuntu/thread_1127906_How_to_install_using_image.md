### Post by freak42 on 2009-04-17
There are two methods to install

1) Install inside a file in windows with the ubuntu-wubi installer, this should work with a image mounter tool like daemon tools.

2) Install in a partition (wipe windows, or dual boot), this cannot be done from within windows and therefore an image mounter can't help you. You have to burn a disk (or use a usb stick?) to install.
