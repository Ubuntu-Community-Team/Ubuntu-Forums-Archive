### Post by jamesrfla on 2009-02-15
You can do a dual boot. So you can use Vista for itunes. That way you don't have to move all your data to a flash drive. Since you are a beginner I recommend Wubi for dual booting. Simple download it here [http://wubi-installer.org/](http://wubi-installer.org/) and put in all the information like username and password and hit next. It will download Ubuntu and install it so when you boot up your computer you can either boot up into Vista or Ubuntu.
