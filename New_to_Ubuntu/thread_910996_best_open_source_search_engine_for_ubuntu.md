### Post by ProgramErgoSum on 2008-09-05
Or, build your own. ;-)

You have [Lucene Java]("http://lucene.apache.org/") for basic indexing and searching, [Solr]("http://lucene.apache.org/solr/") for enterprise searching, [Nutch]("http://lucene.apache.org/nutch/") for Web based searches, [Hadoop]("http://hadoop.apache.org/") a distributed file system for MapReduce support.

And, then do a PageRank implementation (after coughing $$$ as patent royalites).
