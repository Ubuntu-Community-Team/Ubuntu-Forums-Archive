### Post by oldfred on 2014-05-19
Linux is not Windows:
[http://linux.oneandoneis2.org/LNW.htm](http://linux.oneandoneis2.org/LNW.htm)

   Ubuntu is not Windows
[http://www.psychocats.net/ubuntucat/a-home-users-successful-migration-strategy-from-windows-to-ubuntu/](http://www.psychocats.net/ubuntucat/a-home-users-successful-migration-strategy-from-windows-to-ubuntu/)
[https://help.ubuntu.com/community/SwitchingToUbuntu/FromWindows](https://help.ubuntu.com/community/SwitchingToUbuntu/FromWindows)

If very new you should only install software from the repository. Some Windows software will run in wine, but most does not fully run or run well. Better to find and use a Linux equivalent. In some cases the Linux software is better, and others not as good but very functional.

All the software in the repository is in the Software center and just just have to click on it to install. And if installed from repository it will automatically be updated if any updates are released. But not new versions.

      
 Linux alternatives to windows apps
[http://www.osalt.com/](http://www.osalt.com/)
[http://www.linuxappfinder.com/](http://www.linuxappfinder.com/)
[http://www.linuxalt.com/](http://www.linuxalt.com/)
