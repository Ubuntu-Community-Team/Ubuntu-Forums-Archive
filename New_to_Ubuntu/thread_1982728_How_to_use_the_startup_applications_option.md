### Post by Peter09 on 2012-05-19
In Unity,
click on the cogwheel shape in top right screen. Select Startup Applications.


[LIST]
[*]Click Add
[*]In the box call Command put transmission-gtk
[*]The other two boxes, enter what you want.
[*]Click Add
[/LIST]


Done
