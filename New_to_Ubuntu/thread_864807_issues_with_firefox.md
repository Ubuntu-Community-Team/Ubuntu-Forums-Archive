### Post by steveneddy on 2008-07-20
FF3 is now stable.

It most likely is an add on in Firefox.

Try disabling all of the add ons that you added and see if that helps.

If not, delete the .mozilla folder in your /home folder, reinstall Firefox from Synaptic and restart, again without the add ons enabled.

Turn each add on back on individually to see which one may be causing the problem.
