### Post by Borg9 on 2011-11-26
1. Activate the java web application, Scottrader in this instance.
2. Close firefox since it is spawning new windows.
3. Go to Downloads.
4. Right click on the .jnlp file.
5. Click on Properties.
6. Go to 'Open With' tab.
7. Select 'IcedTea Java 6 Web Start'.
8. Click on 'Set as default' button.
9. Delete the file.
10. Activate the java web application, Scottrader in this instance.
11. Enjoy a working java web application. ;)
