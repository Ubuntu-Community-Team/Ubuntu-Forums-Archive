### Post by Sirron on 2008-08-09
If you're looking for a multi-protocol client, pidgin is your best bet on the default Gnome desktop. However, if you only want it for MSN/Windows Live Messenger, try:

[LIST]
[*]aMSN, it's quite slow compared to the competition, but is by far the must fully functional in my opinion. It supports audio messaging, and webcam. Also, you'll probably be able to find a plugin that allows /me commands.
[*]Emesene, it's much faster but has no support for audio. The development version has limited support for webcam I think. As for /me, a plugin may be available, especially since fully implementing Messenger Plus! functionality is a goal of the project.
[*]Mercury Messenger, written in Java, very fully featured, but cluttered.
[/LIST]

[www.getdeb.net](www.getdeb.net) is down as I'm writing this, but you can get the latest versions of Emesene and aMSN there later. Mercury Messenger can be downloaded [here]("http://thebachman.info/images/stories/Mercury/Versions/mercury-messenger-1.9.deb").
