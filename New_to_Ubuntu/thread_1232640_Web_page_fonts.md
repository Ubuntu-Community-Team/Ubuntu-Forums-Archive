### Post by mcduck on 2009-08-06
> **CatKiller said:**
> Edit &#8594; Preferences &#8594; Content &#8594; Fonts & Colors &#8594; Advanced &#8594; uncheck "Allow pages to choose their own fonts, instead of my selections above."

I wouldn't recommend that, as it will definitely change the way how web pages look by forcing all pages to use the exact fonts you have configured..

Instead just set the minimum font size in the same place and you'll be fine.
