### Post by Apollo87 on 2010-02-26
Make sure you backup before doing any editing just in case:
```
sudo cp /etc/fstab /etc/fstab.backup
```

You said you formatted the drive in ext4 but it looks to still be trying to mount a ntfs drive:

> /dev/sda1 /media/Secondary\040Disk ntfs-3g defaults,locale=en_GB.UTF-8 0 0

You can try changing that 'ntfs-3g' to 'ext4'. I'm not sure it'll fix your problem but you might as well give it a shot.
