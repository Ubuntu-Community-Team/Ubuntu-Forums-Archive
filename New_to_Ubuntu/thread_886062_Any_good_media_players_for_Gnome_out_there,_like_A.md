### Post by rainwalker on 2008-08-10
For all-around functionality while fitting wtih GTK, Banshee is probably the way to go (I think the version in the repos is a little outdated, look up how to install version 1.0 or later). You can manage your music library as well as videos (I think); I know you can at least watch videos in it. The only thing I don't like about it is the top section with the progress bar, because it seems like it should be bigger. However, it's a great player with an otherwise great interface and definitely has tons of great features.
Other popular ones I've tried are Listen, Exaile, and Quod Libet. I'd recommend them in that order (for some reason I really like the way Listen's interface is laid out). Exaile is aimed at being somewhat of a Gnome equivalent of Amarok, but I didn't find it too similar or full-featured (and the interface doesn't seem as logical sometimes). Quod Libet is just weird IMHO, but it's **GREAT** at handling large libraries. Of course, there's always Rhythmbox, too, but I think it's more of a standard player without some of the features those others have.

### Post by MatthewPlanchard on 2008-08-11
> **OutOfReach said:**
> Banshee, all the way.

Yeah. Banshee 1.2 is out now, an it's amazing. Last.fm, video, play queue:  it's incredibly full featured. To install, go to Main Menu -> System -> Administration -> Software Sources and add

```
deb http://ppa.launchpad.net/banshee-team/ubuntu hardy main
```

To the Third Party Sources list.

Then open up a terminal and type

```
sudo apt-get install banshee-1
```

That'll get you the latest release.  The one in the default Ubuntu repositories is way outdated. Go figure.
