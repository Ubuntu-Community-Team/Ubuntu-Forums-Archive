### Post by Dr Small on 2008-10-17
Install Xchat, for the best preformance.
Then enter your nick, alternate nick, and select Freenode and connect. That's the simple way. Then you can join channels with:
```
/join #channelname
```

Or directly contact a nick with:
```
/query nick
```

Xchat makes things all the more simpler.

### Post by Keen101 on 2008-10-17
the rest of pidgin is hidden in the "task bar" (notification area) in the upper right corner. It looks like a squarish cartoon talk bubble with a green circle. click on it, it is next to your volume control.

you will need to create an account. this is how mine is set up. irc.ubuntu.com is a mirror of the freenode one, so it is the same.

you will then need to use the buddies tab to join a channel, and find your friend or teacher. I normally use the #ubuntu-us-co irc channel, because that is the LoCo channel in my area. this code should work to find him directly if he is logged in.

```
**/query fxn**
```
