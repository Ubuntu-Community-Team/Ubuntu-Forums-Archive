### Post by Simian Man on 2010-04-23
But why are you using vim without knowing how to use it?  If you just want to get some basic text editing done, use something like gedit (or nano for the command line).  If you really want to learn vim, enter vimtutor on the command line which will teach you how to use it.

As for your question, you can just delete the newline before the line you want.  Or is that not what you mean?

### Post by anewguy on 2010-04-23
100% +1 as per Simian Man.  The only time I have EVER used vim was when Ubuntu wouldn't boot to a GUI and so I was in text mode and needed to edit a file.  Even then there are better choices.

If you have X windows already running (i.e., your Ubuntu boots normally to a GUI, not to a text mode terminal), then use gedit.  If you need help with some file, and not sure how to do the edit and save the file as needed (perhaps the file is not owned by your userid), please post back and we'll be happy to try to help you through.
