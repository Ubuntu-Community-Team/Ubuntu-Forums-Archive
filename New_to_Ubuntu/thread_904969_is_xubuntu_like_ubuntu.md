### Post by BDNiner on 2008-08-29
yes, all those applications will run with no problems and can be installed with synaptic and from the command line using apt-get. xfce is much lighter than gnome so you will notice a difference. by the way you don't have to format your computer and install xubuntu just run
```

sudo apt-get install xubuntu-desktop

```

from a terminal and the xubuntu desktop will be installed along side the regular ubuntu desktop. you can then switch between them by clicking on 'session' on the login window and selecting xfce. that way you can switch back to gnome if you feel like it.
