[https://help.ubuntu.com/](https://help.ubuntu.com/)

[http://en.wikipedia.org/wiki/Ubuntu_(operating_system](http://en.wikipedia.org/wiki/Ubuntu_(operating_system))
[http://en.wikipedia.org/wiki/List_of_Ubuntu_releases](http://en.wikipedia.org/wiki/List_of_Ubuntu_releases)

but just clarifying your doubts:

Long Term Support (LTS): is a version of ubuntu released every two years, the last one being 10.04. It is supported (that is, repositories from where to download applications are maintained, and updates are available) for longer period than normal releases. For such reasons, it is usually more stable (less bugs)

GNOME: is a graphical user interface (GUI), i.e. all the software that lets you interact with your computer without using the command line (prompt). GNOME is just one example of GUI, there are others out there, like KDE, or xfce, that have to the untrained eye slight differences.

Unity: is the new desktop environment (basically, the way information is presented graphically to you on the screen) introduced in the latest release of ubuntu 11.04 (nicknamed Natty Narwhal). It is based on GNOME.

Libreoffice: is a productivity suite, like Microsoft Office.
