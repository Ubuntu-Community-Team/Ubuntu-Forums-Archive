### Post by ajgreeny on 2014-04-25
There is no downgrade pathway from 13.x to 12.04 so I'm afraid you are left only with the option to reinstall 12.04 from scratch.

Don't use wubi as that is now deprecated, and in any case it was never intended as a method for a long term dual boot system but only as a means of testing how your hardware would work with Ubuntu.

You could also consider getting a live DVD or USB of the new LTS version 14.04 and installing that; it seems to be excellent, and pretty bug free to me, though I agree that 12.04 is also a great OS; very stable and smooth and worth keeping if it still runs as you want it to.
