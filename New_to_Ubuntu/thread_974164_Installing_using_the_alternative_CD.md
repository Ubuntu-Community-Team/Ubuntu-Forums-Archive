### Post by cdtech on 2008-11-07
I would follow the basics keeping it simple. There are many configurations you could follow but as a beginner I would go basic and learn. You can select to have a separate /home and most important a /boot partition of at least 100M if you plan on doing an upgrade later. Kernel images are around 45M and to have more than one (which you will need with an upgrade) you will need more space within the /boot partition.

You can always play around with the install and modify it with "gparted" later if you need.

Good luck........

### Post by Therion on 2008-11-07
Are you wanting to do fresh install, meaning you do not have any version of Ubuntu installed currently, or are you upgrading to v8.10 from an earlier version of Ubuntu?

If you're upgrading using the Alternate CD you simply put the disk in the drive after you've booted your PC and you'll be prompted to upgrade.  Just follow the instructions as they're given.

If you're doing a fresh installation see the documentation here:

[https://help.ubuntu.com/community/Installation](https://help.ubuntu.com/community/Installation)

### Post by hictio on 2008-11-07
Yes, the Alternate CD is the same thing, in the end, than the regular one, bu, it uses far less resources and allows you to make a fine tune installing.
It is recommended, at least, IMHO, not only for boxes with a small amounts of RAM, but also for older hardware.
At least, using the Alternate was the only way I could install Hardy on my crappy desktop (Pentium II @ 350 MHz with 512 MB of RAM, and a GeForce2 MX/MX 400), because the regular CD gave me a lot of errors, described the install process [here](http://oesediez.blogspot.com/2008/08/some-orbis-tertius-ubuntu-install-notes.html).
