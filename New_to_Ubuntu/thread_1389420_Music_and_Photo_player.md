### Post by llawwehttam on 2010-01-24
Moovida, formerly Elisa, is a beautiful media center which is perfect for setting up a Home Theater PC (HTPC) or TVPC like the [Neuros Link]("http://www.neurostechnology.com/") and it uses the GStreamer multimedia framework to support playing almost any kind of file.

to install [click here]("apt:moovida") or run the following:
```
sudo apt-get install moovida
```

Personally I quite like it.
