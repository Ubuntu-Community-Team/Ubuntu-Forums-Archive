### Post by vikramaditya on 2008-06-29
I installed hardy on my Dad's box last week and everything was *splendid* until I today replaced the gnome network manager with wicd.  You may ask "Why did you do that, you stupid f**k?!".  Well, it's because I hate those frickin' error messages generated by network manager whenever shutting down or restarting.  To get rid of them, I installed wicd on my own rig several weeks ago and it works fine.  I had no reason to believe it wouldn't work on Dad's machine.  BTW, we both just use wired cable connections, so wireless is *not* an issue.

Anyway, network manager gets removed when you install wicd.  Since wicd won't connect, there's now no way to reinstall network manager.  Synaptic searches valiantly, but without a connection it's no go.  Is there any way to restore network manager that doesn't involve the dreaded tarballs or hair-raising sessions in terminal?

### Post by iaculallad on 2008-06-29
Try to insert your Ubuntu LiveCD then drop to your terminal:

```
sudo apt-cdrom add
sudo apt-get install network-manager-gnome
```
