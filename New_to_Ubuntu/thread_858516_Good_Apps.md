### Post by OttifantSir on 2008-07-13
Well, the obvious app is already installed: Add/Remove Programs. It's at the bottom of Programs in your (by default) upper-left corner.

It doesn't have the whole range of apps, and there are things it can't do that Synaptic does, but it's quite decent, and a little more beginner-friendly than Synaptic or the CLI (Command Line Interface found in Programs->Accessories->Terminal)

### Post by AnLGP on 2008-07-13
sudo apt-get install (program)

do that in your terminal.

ex:

sudo apt-get install jwm

that will install jwm or Joe's Window Manager.

sudo apt-get remove (program)

will remove programs.

### Post by Ahunt on 2008-07-13
Aside from what comes with Ubuntu, here is what we installed on our PCs:

    * Audacity - audio editor and recorder.
    * Celestia - space travel software
    * ClamTk - anti-virus software 
    * FileZilla - file transfer protocol (FTP) 
    * Stellarium - planetarium software 
    * Sunbird - free-standing calendar
    * Tesseract - optical character reader (OCR) 
    * VLC - media player Software for Windows

All are available on "add/remove" except Tesseract, which is on Synaptic.

### Post by rossjman1 on 2008-07-13
- Avant Window Manager - Dock app (like Mac's dock)
- NoteCase - Note taking app (good for students)
- Enemy Territory - Awesome free FPS for Windows, Linux, and Mac
- LinCity-NG - City building game
- XQF Game Server Browser - Good app for remembering good game servers (works with Enemy Territory)
- GNU Paint - Simple MS-Paint-like app
- Frostwire - Java Limewire client
- Amarok - Music player
- DeVeDe - Allows you to convert .avi to .iso for dvd burning
- k9copy - DVDShrink for Linux
- Compiz-Fusion Icon - allows you to quickly turn on/off Compiz
- Conky - Simple program that displays text (such as system stats) on your desktop
- Google Desktop - you know...

ALSO 

- Java
- Flash
