### Post by lolpenguin on 2011-12-08
Welcome to the Ubuntu community!
Nice job on the install...but where d'you find the disc image? Not on the Downloads pages of the Ubuntu site...
We all suggest you try a **supported** version of [Ubuntu]("https://wiki.ubuntu.com/Releases").
I'd suggest either 10.04 LTS "Lucid Lynx" if you want stability, or 11.10 "Oneiric Ocelot" if you want bleeding-edge.
Ubuntu's version numbers correspond to the date of release. Typical releases are released in April and October every year, eg 11.04 "Natty Narwhal" was released in April 2011. Typical releases are supported for 18 months, the End of Life of a release. After this date, the release is no longer supported, which will render it vulnerable to security threats and will not receive updates nor patches. You should upgrade before the End of Life. The version you are currently using, 6.06 "Dapper Drake" was released in June 2006.
You should download a disc image from [_http://www.ubuntu.com/download/ubuntu/download_]("http://www.ubuntu.com/download/ubuntu/download"). You can select either 11.10 or 10.04, 32-bit or 64-bit. Burn the download to a disc and install it. Feel free to explore after that.
Remember, if you need help on (almost) anything, search [_askubuntu.com_]("http://askubuntu.com/") and [_Ubuntu help_]("https://help.ubuntu.com/") if it is answered, if not you can ask it here in the Ubuntu forums.
We promise you'll love Ubuntu.

### Post by coffeecat on 2011-12-08
> **aliuntu said:**

Juts to add to what others have said. An iBook G3 would have a PPC processor, so if you want to try a supported version of Ubuntu, you would need to download the correct ISO image, not the one that most people use. You can get a PPC image for 10.04, here:

[http://cdimage.ubuntu.com/ports/releases/10.04/release/](http://cdimage.ubuntu.com/ports/releases/10.04/release/)

Unfortunately, that's for the original 10.04 release, not the latest 10.04-3 point release, for which I couldn't find a download source. If anyone knows of one, if one exists, that would be helpful.

Also - your iBook would have limited RAM, and I'm surprised even Dapper Drake is running on it. How much RAM do you have?

Lastly - if you want a basic guide to Ubuntu, a good place to start is with the Ubuntu Manual, here:

[http://ubuntu-manual.org/downloads](http://ubuntu-manual.org/downloads)

Make sure you download the 10.04 one if you decide to install 10.04.

Good luck!
