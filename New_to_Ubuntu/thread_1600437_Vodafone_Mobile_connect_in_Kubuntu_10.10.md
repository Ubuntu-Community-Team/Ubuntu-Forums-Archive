### Post by Phillip Priest on 2010-10-19
I am new to this, so if I am repeating known info I apologise.
I have noticed people have problems enabling Vodafone Mobile Connect in Kubuntu
I have got mine to work by doing the following: open Manage Connection, open Mobile Broadband and at Number type in *99#, then type in Your user name that you have used on the Vodafone web site. The password is the password you use for access to the Vodafone Web site.
At Advanced Type in for APN: vfprepaymbb if you are using prepay account, if not check either the details in Gnome desktop or ring Vodafone.
Then Type is 3G [UMTS/HSPA]
PIN is that which you use for Vodafone as is PUK.
At PPP leave unchecked Require 128-bit encryption and Stateful MPPE.
Press Ok, then Apply, then Ok then restart Computer {with usb mobile attached} and it should start.
Sometimes I found that it takes a while for Kubuntu to recognise the usb mobile if you put it in after the Desktop appears.
Again I apologise if this is old info, but I could not find anything on this problem when I "googled" it.
