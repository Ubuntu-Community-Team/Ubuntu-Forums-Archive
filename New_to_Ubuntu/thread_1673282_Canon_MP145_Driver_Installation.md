Dear All,
This post has been written to ease the pain of installing drivers for Canon MP145.
First download the drivers from-

[http://support-in.canon-asia.com/P/search?model=PIXMA+MP145&menu=download&filter=0&tagname=g_os&g_os=Linux](http://support-in.canon-asia.com/P/search?model=PIXMA+MP145&menu=download&filter=0&tagname=g_os&g_os=Linux)

Here deb files are available which can be installed by GDebi. Drivers files required are:-

1.[ ScanGear MP Ver. 1.10 for Linux(debian Package for MP140series)]("http://support-in.canon-asia.com/contents/IN/EN/0100084801.html")
2. [IJ Printer Driver Ver. 2.80 for Linux(debian Common package)]("http://support-in.canon-asia.com/contents/IN/EN/0100083403.html")
3. [ScanGear MP Ver. 1.10 for Linux(debian Common package)]("http://support-in.canon-asia.com/contents/IN/EN/0100084701.html")
4. [IJ Printer Driver Ver. 2.80 for Linux(debian Package for the MP140 **...**]("http://support-in.canon-asia.com/contents/IN/EN/0100083701.html")

First install common package then MP140 series package. You are thru.....
