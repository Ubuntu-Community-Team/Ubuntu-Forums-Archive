### Post by deadflowr on 2013-08-16
Audio Video.

Here's a manual to get you into the basics of Ubuntu
[http://ubuntu-manual.org/](http://ubuntu-manual.org/)

Here's another place to look for interesting stuffs
[https://help.ubuntu.com/community/NewDocs](https://help.ubuntu.com/community/NewDocs)

And thirdly, in the sub-forums section of this thread(Absolute Beginners, for now at least) is a sticky thread on linux command line learning.
It has lots of links to tutorials and such.

Linux and Ubuntu have a cool feature that you can use to learn about various commands,
It's called man
open a terminal and run
```
man <command>
```
It'll open up a manual on how that command can be used, sometimes with examples.
Try it out.

Added: Here's another resource
[http://psychocats.net/ubuntu/index.php](http://psychocats.net/ubuntu/index.php)
