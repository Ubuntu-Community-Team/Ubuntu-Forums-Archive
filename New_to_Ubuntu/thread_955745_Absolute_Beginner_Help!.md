### Post by Sub101 on 2008-10-22
So you have downloaded a file with the extension .iso?

Once you have this file you must use a program like Nero to burn it to a disc. (unless you want to run it in a virtual machine?) 

When you burn the disc be sure to select burn ISO not to copy the file to the disc. If you do not have Nero there are plenty of free ISO burners available to download.
