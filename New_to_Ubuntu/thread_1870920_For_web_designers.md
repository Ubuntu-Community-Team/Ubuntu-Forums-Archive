### Post by mcduck on 2011-10-28
You can also check at least basic layout and things like that using services like [http://browsershots.org/]("http://browsershots.org/").

I usually use browsershots for quick checking, and then just go to a friend or any other place where I can use IE to do final checks that everything is working. Luckily latest IE versions are fairly standards-compatible so there isn't that much that would need special attention. If you want to support IE6, that should run just fine with Wine.
