### Post by mcduck on 2009-01-20
Core 2 processors are 64-bit, which means that you can use which ever version you want , i386(32-bit) or AMD64(64-bit).

64-bit version might be slightly faster depending on what kind of programs you run, while 32-bit version has slightly more programs available.

If you have 4GB or more of RAM get the 64-bit version, as 32bit operating systems are not able to handle that much memory.

Yeah, as said since you have a 64-bit machine, the 32-bit version should work fine. However, there is a speed increase on 32 vs 64 (no way), but 32 has more applications.
