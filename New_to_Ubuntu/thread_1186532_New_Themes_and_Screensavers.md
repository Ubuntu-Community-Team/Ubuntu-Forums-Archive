### Post by QIII on 2009-06-13
[www.gnome-look.org](www.gnome-look.org) is a good place to start.

A lot of the themes come with installation instructions.

If you use Emerald, I've found that I can often "trick" Emerald into using other theme formats by simply changing the extension to .emerald.

Not always, but often.
