### Post by 3rdalbum on 2011-11-15
[http://alldroid.org/tabid/40/g/posts/t/1125/Mount-Internal-Storage-of-Xoom-in-Ubuntu.aspx](http://alldroid.org/tabid/40/g/posts/t/1125/Mount-Internal-Storage-of-Xoom-in-Ubuntu.aspx)

[http://forum.xda-developers.com/showthread.php?t=981774](http://forum.xda-developers.com/showthread.php?t=981774)

It's harder on 10.04 apparently because you need to recompile an MTP-related package. Any reason you're not using a more up-to-date Ubuntu?

It seems to be a royal pain even on later distros, and a lot of people are having trouble getting the Xoom's storage to mount on Ubuntu. You might prefer to install a file manager like File Expert on your Xoom, and run an SMB share on your computer that contains the files you want to transfer. Use File Expert to transfer them from the computer to the Xoom.

Or something like that - Bluetooth would work too.
