### Post by tamoneya on 2008-05-12
when you reinstall it will do a reformat however you tell it to.  Since you have XP as a dual boot you need to be careful.  Here is the best way to proceed. 
0. Backup all your data
1. Download and burn HH CD
2. Boot Hardy CD
3. Run "sudo gparted" and delete all of your linux partitons
4. Run the install to harddrive
5. Tell hardy to install into free space on harddrive (this is where your linux partitions were)
6. Hardy will reconfigure grub for you so you dont have to worry about your dual boot as it will be kept intact.
7. reboot and take out liveCD.
