### Post by Mark Phelps on 2017-10-07
Your best bet for testing the CD drive is to create boot media for Ubuntu, boot with that, and run it in Live Mode -- to see how well the Ubuntu version detects the hardware and installs working drivers.  Any hardware that then does not work will range from easy to impossible to fix -- depending on the ready availability of Linux drivers.

As to the "favorites" -- if these are for Internet Explorer (or even worse) Edge, then NO -- as there is no even recent version of IE that runs in Linux, and Edge certainly does not.  Your best bet there is to export the favorites to a bookmarks.html file and import those into whatever Browser you then decide to use in Ubuntu.

### Post by Impavidus on 2017-10-07
It's very unlikely a dvd drive wouldn't work. I never read about that on the forum. Sometimes some hardware doesn't work because there are no good drivers for it, in particular new hardware and in particular graphics and wifi, and usually this can be fixed. I'm typing this on a 6 year old HP laptop where everything worked out of the box.

Ubuntu will not pick up anything from Windows automatically, no favourites, no settings, no files. If you instruct the installer to replace Windows with Ubuntu, it will simply destroy whatever is on the hard drive. So copy files to some safe location.

If you happen to use Firefox on Windows, which is the default browser in Ubuntu, or Chrome or Chromium, which are available for Ubuntu, you can copy your profile.
