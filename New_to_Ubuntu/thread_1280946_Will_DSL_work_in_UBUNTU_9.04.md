### Post by donkyhotay on 2009-10-02
I've set up ubuntu with a number of DSL services over the years. I've never had a problem with ethernet, never tried the USB. Some companies will not want to help you if you have linux but generally so long as you can prove that your computer can communicate with the modem (ping commands, etc.) and that the problem is with the modem they'll help you since the solutions are the same. On the other hand if you need help configuring the ethernet on the computer itself you're on your own unless you can get onto these forums on another system.
