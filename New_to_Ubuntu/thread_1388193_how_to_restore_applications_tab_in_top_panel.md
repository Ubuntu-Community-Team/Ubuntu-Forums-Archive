### Post by pablolie on 2010-01-23
and there are no hints on how to reset the top panel in xubuntu to the original in these lists - none that work with 8.04 or later, it seems.

the only thing i have been able to do is...

1. create a new user 
2. copy all my old files to the new users (not the hidden folders, since that would mean you also copy over the messed up xfxe settings!)
3. delete old user folder (needs thunar so you can do the next step) 
4. create new user with old user name (which is now free)
5. reverse everything
6. you have original settings back.

Sounds silly? it is. there should be a way to protect those panels so that delting them is a 2 step process and can't be done by accident as easily (see how many messages on that there are in this forum for Xubuntu).

the reason Xubuntu is so vulnerable for it is because XFCE has a very poor standard setting for touchpads in portables, and double-click on the touchpad can not be easily disabled. there are also several threads on that.

### Post by nitstorm on 2010-01-23
i am not sure about xfce but this is how i do it on gnome. 

1.right-click on the panel and click "Add to Panel"
2.A new box comes up, scroll down to find "Menu Bar"
3.Click on it and click add
4. Voila! It should be there on your panel.

hope that helped, if not , excuse me ..
