### Post by tom.swartz07 on 2010-04-13
> **Faith95 said:**
> apparently, the other users on my laptop can't access the wireless network (which has a password and a keyring)
how do i make it so that others can use the same network without me having to log into the administrative account to grant permission?

All you have to do is log in under your profile, right click on the network icon, select 'edit connections', edit the wireless network in question, and select available to all users. Type your password and youre all set!
