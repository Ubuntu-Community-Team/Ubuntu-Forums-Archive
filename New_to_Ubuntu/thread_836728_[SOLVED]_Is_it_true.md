### Post by aysiu on 2008-06-21
If it's formatted as HFS+ (a Mac OS X format), then you may not be able to make changes to it.

If, however, it's formatted as NTFS, FAT32, or Ext3, Ubuntu should be able to mount it just fine.

If you do run into problems (it won't mount, it mounts read-only), post back here, and we'll help you get it sussed out.
