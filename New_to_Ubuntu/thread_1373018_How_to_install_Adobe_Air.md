### Post by warfacegod on 2010-01-05
Terminal code:

> sh ./name-of-bin.bin


That should work. If not type sudo in front of it. You may have to use the terminal to navigate to the folder beforehand.


Example:

```
cd /home
```

Will bring you to your home folder.
