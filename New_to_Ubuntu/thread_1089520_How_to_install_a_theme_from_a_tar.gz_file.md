### Post by Asterio on 2009-03-07
I found a theme I like from gnome-look, but I'm absolutely stumped on what to do with the tar.gz file that I downloaded. 

How do I install the theme?

---

### Post by t0p on 2009-03-07
Go to **System** > **Preferences** > **Appearance**, then drag the tar.gz file into the Themes window.

### Post by howefield on 2009-03-07
Sometimes you will find a collection of themes inside the initial downloaded tar.gz. In which case, extract to the desktop and then do with the themes inside as per tOps' post.
