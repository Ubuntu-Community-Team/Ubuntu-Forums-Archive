### Post by halitech on 2009-12-12
I don't see /dev/sda1 actually mounted and /dev/sda3 is your windows recovery partition so may not want to be doing anything with that

what I would do is this, decide where you want the drives mounted. ie ~/storage1

then make the folders
```
mkdir <foldername>
```
then edited fstab
```
gksudo gedit /etc/fstab
```
add the following
```
/dev/sda1 /home/<username>/<foldername1>	ext3	defaults	0	2
```
```
/dev/sdc1 /home/<username>/<foldername2>	ext3	defaults	0	2
```
Save the file and then in the terminal run
```
sudo mount -a
```
Hopefully this will get us 2/3 of the way there
