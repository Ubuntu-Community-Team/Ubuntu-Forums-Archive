### Post by abuelmagd on 2009-07-15
Just to keep this thread alive I'll describe my experience:
 
I'm running Ubuntu 9.04. I plugged the ML-1630 printer in and it worked right away. Things can't get any better than that.
 
The only extra thing I installed was the smart panel software that was on the cd. It tells me the amount of toner remaining in the cartridge. So I only run that application when I want to check.
 
I highly recommend this product for ubuntu users.
