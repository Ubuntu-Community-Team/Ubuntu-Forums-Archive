### Post by Mark Phelps on 2018-05-13
The main problem with running Linux on ANY laptop is the routine use of specialized hardware by laptop OEMs to keep the costs down -- hardware for which, very often, there are no Linux drivers.  Eventually, if a laptop model is popular enough, the Linux community may catch up with Linux drivers -- but not always.

I've gone through five different laptops over the years and only ONE was fully Linux compatible -- meaning, ALL the hadware worked -- and that was only true with a 14-year old version of Ubuntu.  Since then, NONE of them have been fully compatible.

The ONLY way to know if a specific make and model laptop is fully Linux compatible is for someone to actually test it -- so your request here, if that is the case, may work out for you.

Good Luck in this ...

### Post by monkeybrain20122 on 2018-05-13
I don't know how good is Ubuntu on touchscreens, never have one and hate people leaving fingerprints all over.  But other than that I think all but the first would work, the first is a bit tricky because of the convertible (not that it won't , only saying I don't know since this is not standard hardware)

I have ONLY laptops for years, very few people I know,--young professionals and students,-- still own desktops. With the high rent and all who has the space and who carries a desktop to a conference or on a trip? Moving with a desktop is also painful. It is not 2005 any more and Ubuntu support for laptops are pretty good. Don't let outdated info scare you (of course assuming the hardware is reasonably recent, not some junk that you can put a 14-year old Ubuntu version on.)

 I have gone through many models, ALL but one worked and the one that didn't work fully had some wifi driver problem, a dongle solved the problem. Dell is usually excellent except some old broadcom chips for wifi but they are probably retired by now, Lenovo/hp are good too in general but need some research as some models are not compatible, I have had others too.My first "real" Ubuntu laptop was a Samsung laptop which Samsung told me linux would never run on it, but everything worked out of the box beautifully.  right now the Toshiba netbook works beautifully for 16.04 and 18.04.

Usually if you can boot a live usb (meaning no uefi locked down and such) , then it is down to graphic card and wifi chip. Intel graphic works out of the box, Nvidia works too if install the proprietary driver (noveau should be able to at least get you the desktop if graphic card is not too new) I don't have an optimus laptop but from threads here they  do work,--maybe with some tweaks. I never used amdgpu, but that should work too.

In general avoid hardware which is too new since driver support takes time, but that is the same advice even for Windows (it might have driver but not fully work or buggy) Also avoid amd-intel hybrid cards.
