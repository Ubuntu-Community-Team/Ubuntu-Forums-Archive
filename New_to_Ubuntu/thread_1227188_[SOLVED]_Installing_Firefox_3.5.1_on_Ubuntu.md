### Post by arochester on 2009-07-30
You can get the "ordinary" Firefox 3.5.1 by using the script called Ubuntuzilla for here: [http://ubuntuzilla.wiki.sourceforge.net/](http://ubuntuzilla.wiki.sourceforge.net/)

Page down to "Installation". I have a 32 bit machine so the operations are - download the .deb, right click it and install, then run the command (in a Terminal):
> ubuntuzilla.py -a install -p firefox
