### Post by DuckHook on 2012-10-07
You may wish to reconsider whether you are making things needlessly complicated for yourself. I believe you have posted previously and a number of forum members have responded including the author of the Python script that you have quoted above. This author basically created a GUI front end for the Clam AV app in Python. Unless you have the proper Python modules and dependencies installed, you will not be able to run the script.

My question for you is: why do you feel that you need AV at all? In the Linux world, all AV apps are useful only as a public service to Windows users so that Windows viruses don't get passed on downstream to other Windows users. Linux itself is highly hardened against viruses, so AV will not add anything to your system except bloat and unnecessary overhead. Since switching to Linux, I have not had to deal with the problems and headaches of AV software at all.
