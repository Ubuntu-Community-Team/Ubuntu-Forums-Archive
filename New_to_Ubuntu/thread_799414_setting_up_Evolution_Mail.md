### Post by HotShotDJ on 2008-05-19
> **Tubby said:**
> On the Identity page it says " Full name.... "Put your name there.  Doesn't matter who owns the computer or the account for that field.  Make sure you fill out "Name" just above it too!
> **Tubby said:**
> Server- pop3......net.auGet rid of the space between "Server-" and "pop3"  No spaces in server names.  Be sure Security and Authentication type are correct.  Use the "Check for Supported Types" button to be sure.> **Tubby said:**
> On the Sending mail section he told me to put-
Server configuration - smtp.mel....net.auAgain, make sure you've got the Security and Authentication stuff set up correctly.  Unless your ISP has told you otherwise, the server probably does not require authentication.  If you end up with trouble sending mail, call them and find out.
