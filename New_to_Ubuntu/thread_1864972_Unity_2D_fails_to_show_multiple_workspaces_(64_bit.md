### Post by Potet on 2011-10-19
I have a HP Compaq 6735b, and I find regular Unity to be a bit slow on it. I would prefer to use Unity 2D, but it fails to show more than one single workspace. I can't switch using ctrl+alt+arrow, and the workspace overview shows only one desktop, the rest of the screen is black. 

I remember seeing all of the workspaces earlier. I don't know if it matters, but I have also installed the standard Gnome 3 desktop.

UPDATE: Never mind, I went into GNOME Classic, and saw that there was only one desktop there too. I changed the numbers of desktops there to four, and it's allright now. This probably happened because i set the number of desktops in Gnome Classic to six, since i didn't know it would screw up Unity 2D.
