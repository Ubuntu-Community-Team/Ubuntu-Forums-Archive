### Post by darthmob on 2009-04-20
a workaround would be to emulate a virtual desktop in wine. you can add it for your game in the wine configuration under graphics.

simply set it to use the same resolution as your desktop. it will be windowed but using the full screen and it should not effect your icons. :)
