### Post by taurus on 2008-11-18
Sounds to me like the ntfs filesystem is having a little technical difficulty!  What happens if you try to mount it by hand from a terminal to see what's wrong with it.

```
sudo mkdir /media/windows
sudo mount -t ntfs-3g /dev/sda1 /media/windows
```
Assuming that /dev/sda1 is where your ntfs partition is.
