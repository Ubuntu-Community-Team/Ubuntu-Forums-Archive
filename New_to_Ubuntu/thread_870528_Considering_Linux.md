### Post by ad_267 on 2008-07-25
1. Do I need to know any programming code to use Ubuntu?

No.

2. For a beginner, what is more recommended: Ubuntu or Kubuntu?  Which OS is closer in appearance and usage to the Windows OS?

I would recommend Ubuntu as it is more commonly used and if you need help from the forums people are more familiar with it. Some people seem to think that Kubuntu is more like Windows by default but I find Ubuntu is very easy to use and can be easily customized to look more like Windows just by dragging panels around and adding and removing things from them.

3. Are there compatibility issues for Ubuntu with Windows software?  Some of the software I use regularly are: Photoshop, Gom player, Veoh TV, and various PC games.  Will I be able to install and use these software with Ubuntu?

Yes, windows software is written for windows so unless the sofware developer provides a Linux version then you may have issues. There is wine which provides a compatibility layer and allows you to run some windows software on Linux, but the best option is to use the many free alternatives which many people find to be just as good or better. For Photoshop there's the GIMP. There are many good open source games and some work with wine. I'm not sure what the other applications you mention are. I suggest just sticking with software that works on Linux and only use wine if you really need to. You will probably find that most things you need are installed by default on Ubuntu. This link is useful [http://www.osalt.com/](http://www.osalt.com/)

4. Does Ubuntu have its own anti-virus/spyware programs or can I use programs such as AVG anti-virus, adaware, Spybot, etc. on Ubuntu?

You don't need this on Ubuntu. There are anti virus programs but they really only scan for Window viruses.

5. Does Ubuntu have right-click? =)

Yes! It also have lots of other useful stuff like if you highlight text then middle click it will paste the text.


Just try out the live CD. It won't actually install Ubuntu and you are free to play around and see if you like it without affecting your windows installation.
