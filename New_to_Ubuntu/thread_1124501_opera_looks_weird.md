### Post by SunnyRabbiera on 2009-04-13
Opera does sort of stick out a little when using Ubuntu, but thats mainly because Opera is QT based in Ubuntu.
QT is a widget toolkit, but by default Ubuntu uses GTK another widget toolkit.
In the world of linux there are multiple toolkits that are used, while in windows the toolkit is the same for all apps Linux offers choice.
But there is a downside, QT apps dont look as good in Ubuntu.
This is why I use the other/static package when installing Opera in Ubuntu.
I dont use the default installer package offered by opera
