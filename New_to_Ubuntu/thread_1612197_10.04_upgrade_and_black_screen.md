What is the problem? And how is it solvable? Thanks greatly for any help.Try:
```
1. During startup, keep "SHIFT" pressed until you're in the grub menu
2. Once in the grub menu, press "e"
3. Find the line with "quiet splash" and type "nomodeset" after it
4. Press "CTRL+x" to boot
5. This should start ubuntu (although with a sub-optimal graphics driver). 
6. Then install the proprietary NVIDIA driver and reboot normally.
```

```

```

when that completes you may have to run 
```

sudo nvidia-xconfig
```
