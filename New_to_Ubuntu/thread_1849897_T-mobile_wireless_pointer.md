### Post by 3rdalbum on 2011-10-09
> **Dermot Doyle said:**
> Hi Gandaran, yes I haven't had any problem recognising wireless networks on the computer here and in France.

I think I remember reading that some Wifi channels overlap a set of restricted frequencies in France, and as a result French wifi equipment can only deal with a smaller set of channels.

If your wifi card or computer came from France, then perhaps the T-Mobile device is set to one of those frequencies that are not allowed to be used in France.

Otherwise, perhaps the "pointer" (?) is set to create a Wireless N network only, and your equipment can only connect to G and B.
