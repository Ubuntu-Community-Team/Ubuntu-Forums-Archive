### Post by oldsoundguy on 2008-09-29
If it appears it will not P.O.S.T. and go into the initial screens, yes, you could have a video card issue, or there could be a problem with the processor, memory, on board chip, power supply, power rail capacitors or motherboard.  Remote control fixing of that particular issue is hard, as "you really got to be there".
If such has occurred, the real issue then becomes "has it done any MORE damage?"

The suggestion of "shut it down and let it be for a while" is a very valid one.  BUT, when you try to boot again .. take notes as to everything that happens and try and be relatively specific, as, if you have to take it in, being able to start the tech at point c or d vs point a will save you both time and MONEY.

Most shops will have what is called a POST TEST card that they can insert into any desktop and it will give them a readout of a lot of issues.  Laptops need to go to a specialist.
