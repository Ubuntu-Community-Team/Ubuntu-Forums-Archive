### Post by Zoot7 on 2009-10-22
Ubuntu Netbook Remix is just really the normal version of Ubuntu but with a nicer interface that's more set up for and easier to use on smaller screens.
Other than that there's no difference. You could run either on your netbook. :)

---

### Post by yeats on 2009-10-22
You should definitely try Ubuntu Netbook Remix.  It has the [clutter menu]("http://1.bp.blogspot.com/_zKmpLPXjevA/SgI2F69ddfI/AAAAAAAAAEQ/VWyMILXa5Ls/s1600/Untitled.png") which is ideal.  Otherwise, it *is* Jaunty, meaning that the underlying programs are the same between UNR & "regular" Jaunty.
