### Post by grahammechanical on 2011-02-27
The maximize, minimize and exit buttons are a function of the desktop and not the application that you are using. Go to System>Preferences>Keyboard shortcuts. In there you will see keyboard key combinations that will do what you want. Could it be that you are somehow going into full screen mode? You might want to think about choosing a different theme in Appearance.
