### Post by Falcorian on 2008-06-07
Killing firefox will probably get rid of it.

```
killall firefox
```

You can use ctrl+alt+F1 to get to a terminal, and then ctrl+alt+F7 to get back to the gui after.


However, you should probably run [NoScript](http://noscript.net/).
