### Post by grahammechanical on 2011-11-10
Have you tried opening Update Manager or the Ubuntu Software Center and clicking on Settings? Under the Ubuntu Software tab you should have box with the heading Installable from CD-ROM/DVD and the option to put a check mark against CDROM with Ubuntu 11.10 'Oneiric Ocelot.'

If this option is not there you can go to the Other Software tab and click Add Volume. If you have the CD in the drive it should be detected.
