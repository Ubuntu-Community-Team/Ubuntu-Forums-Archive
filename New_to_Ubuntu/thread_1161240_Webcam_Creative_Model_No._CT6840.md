### Post by Marlonsm on 2009-08-11
I have used a Creative once and it worked out of the box.

If it doesn't work, try this:
-Go to System > Preferences > Multimidia systems selector (if it's not there, press Alt+F2 and run this: gstreamer-properties)
-In the Video Tab, change the default input to other values, maybe one of them will work (mine works with "Video for Linux 2" and "Default")
