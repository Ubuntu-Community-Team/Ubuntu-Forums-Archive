### Post by Vunutus on 2009-04-26
I assume you're using Jaunty and this is something related to the new pidgin->user switcher functionality. Assuming that is correct, try right clicking an open panel space, going to "Add to Panel", and re-adding whatever it is you removed (probably the user switcher)

### Post by nandemonai on 2009-04-26
> **Messyhair42 said:**
> yes i'm using Jaunty, and when i try add to panel pidgin doesn't appear in the list. i've also got pidgin as a quick launch icon

If you want the 'new' way of displaying back, add the 'indicator applet'. If you want the old way back (imo better) where pidgin has it's own icon, go to tools - prefs (in pidgin) and set 'show system tray icon' to always.
