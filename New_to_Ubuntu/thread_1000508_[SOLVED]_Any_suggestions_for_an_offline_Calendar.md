### Post by mbsullivan on 2008-12-03
> Thunderbird with the Lightning extention is also great. 

Sunbird (which you don't like) is basically the same as Thunderbird with the Lightning extension. 

Note, however, that the version of Sunbird that is in the repositories is **perpetually** outdated, because it is being actively developed. So... if you don't like it because of peculiar behavior or bugs, perhaps consider installing it from [source]("http://www.mozilla.org/projects/calendar/sunbird/download.html") and see if it's any better.

Mike

PS: Once installing from source, to make an entry into the Application menu, etc, see [here]("http://ubuntuforums.org/showthread.php?t=278206").
