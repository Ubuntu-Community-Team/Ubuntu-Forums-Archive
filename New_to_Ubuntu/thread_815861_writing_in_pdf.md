### Post by HotShotDJ on 2008-06-02
> **i.mehrzad said:**
> I went to export but they dont have export as pdf option.If you've got Ubuntu 8.04, you can use the PDF printer that is installed by default (pretend that you are going to print the file, and then choose the PDF printer).  In previous versions, you'll need to add the PDF printer manually using System --> Administration --> Printing.

Also, as noted by Maestro23, you could use OpenOffice which has an export to PDF function built in.
