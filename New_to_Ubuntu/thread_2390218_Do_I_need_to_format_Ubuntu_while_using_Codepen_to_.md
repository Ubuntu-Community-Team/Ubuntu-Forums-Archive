### Post by sevendogsbsd on 2018-04-28
Not too many other options except to try opera, or other Linux browsers. Problem is the alternative browsers are not widely used. You can write code that senses what browser the end user has and display appropriately, or you can google the particular tag that is causing the problem to see if there are any workarounds for specific browsers. Not all browsers display HTML or CSS code the same, or to HTML standards. This has always been a problem since the invention of HTML. Since codepen is a collaborative site, might want to reach out to other devs with the issues you are having to see if there are workarounds.
