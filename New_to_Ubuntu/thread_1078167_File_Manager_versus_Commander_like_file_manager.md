### Post by Mark Phelps on 2009-02-23
> **longtom said:**
> Hi,
Anything out there which might be satisfying my possibly unreasonable demands?


I think the problem is that your demands mix file managers with file handling by certain apps.

I've used Nautilus and Gnome-Commander and while they both allow browsing file systems on networked drives, viewing files (pictures, videos) or playing files (audio) both require the mounting of those file systems.

I think you'll have to experiment with different apps to see which ones will allow you to view and/or run a networked file without mounting the filesystem.
