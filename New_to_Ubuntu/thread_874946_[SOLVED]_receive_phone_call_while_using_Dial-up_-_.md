### Post by northern lights on 2008-07-30
A regular landline call while using dial-up is only possible with two lines.

If you want to make phone-calls while on a dial-up connection with only one line, VoIP is your best bet. Ubuntu comes with 'Ekiga' preinstalled. Skype is another option.
