### Post by migs73 on 2011-02-08
Loren

In your last post I did mention that a re-install 'should' not be a problem. If you have only ever done it in windows before, you'll be surprised how easy it is.

**Make sure you have an internet connection**..... then go to Applications > Ubuntu Software Centre.

This then presents you with lots of different applications in categories. All of these (bar one) are free to download and will install automatically if selected.

Go to the search bar in the upper right hand corner and type in Open office,

There will be a green tick on the icon as you already have it installed. Click on it and look to the right, it'll say Remove.

Click on remove and type in your password when prompted.

Wait for it to do its stuff (might take a couple of minutes)

Once it is done the remove button will now say Install.

Click on that and wait a few minutes till it is complete.

Done....Uninstall and reinstall. No searching the web, no manually deleting anything, the Software centre does it all for you.


AGAIN YOU MUST BE CONNECTED TO THE INTERNET TO DO THIS.

If you are not confident to try this, go to the games category and select a game you might like, click on instll and see if it works! If you don't like it click remove.

There are thousands of programs available, even if OO still causes problems, try another WP from the Office category!!!
