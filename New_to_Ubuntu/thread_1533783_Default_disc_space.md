### Post by darolu on 2010-07-18
After you have chosen the file you want to transform to DVD and after you've set the menu options, you click on Next, a Window will pop up asking where to save the DVD Image, there you have to click little arrows so a select menu rolls down, select "OTHER" to choose a different directory; needless to say you have to mount the hard drive/partition you want to use before you can use it.

An "iso" is a file that contains your DVD 'image', this is the bit-by-bit data that will be stored in your DVD (the actual disk), iso refers to the file extension .iso that depicts a standard: the ISO9660 standard.
