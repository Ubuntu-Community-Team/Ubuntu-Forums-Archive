### Post by andrewthomas on 2010-08-30
edit /etc/default/grub

GRUB_CMDLINE_LINUX_DEFAULT="quiet splash"
to 
GRUB_CMDLINE_LINUX_DEFAULT="single"

and then sudo update-grub

### Post by michel. on 2010-10-19
Just edit /etc/init/rc-sysinit.conf and change the following line to the runlevel you would like:
```
env DEFAULT_RUNLEVEL=2
```Alternative is to manually create /etc/inittab containing the following:
```
:id:2:initdefault:
```
