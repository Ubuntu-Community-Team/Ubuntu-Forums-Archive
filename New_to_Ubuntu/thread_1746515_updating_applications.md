### Post by Dr. Nick on 2011-05-01
Short of compiling it from source you just have to wait for it to get added to the repos then click update in synaptic of run "sudo apt-get update" from a terminal. If you try to update software from the 11.04 repos you will eventually screw up something when you start mixing versions and dependencies. 

You can try [this site]("http://www.webupd8.org/2011/04/install-pinta-08-in-ubuntu-now.html") as it has a newer version and instructions how to add the repo and install it. I dont use the pinta so I am can't be much help beyond that
