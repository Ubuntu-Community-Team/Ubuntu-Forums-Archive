### Post by rampageoberon on 2009-06-20
You can find you internal (If you are using a router) IP address using ifconfig or you can right click on the network manager icon and select connection information.

```
ifconfig <INTERFACE>
```

To check your external ip (if using a router) go to [http://checkip.dyndns.org](http://checkip.dyndns.org)

Hope that helps
