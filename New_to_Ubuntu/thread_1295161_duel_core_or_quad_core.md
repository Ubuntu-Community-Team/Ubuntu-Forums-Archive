### Post by K.Mandla on 2009-10-19
> **weebit said:**
> Is it necessary to have a quad core using Ubuntu?
Absolutely not. Provided you have adequate memory and adequate patience, you can use Ubuntu on machines as old as the Pentium III generation.
> **weebit said:**
> But someone told me that Ubuntu doesn't need all of that processor, and I would be happy saving my money and just getting the duel core. Is this true? Or not?
It's true. And depending on how much you want to save, you can probably find a high-end Pentium 4 system with more than enough memory for around US$50, and have as much success with it.

### Post by 3rdalbum on 2009-10-19
There are Linux programs that can take advantage of multiple cores, so you can definitely use it. There will be more such programs in the future too.

If you're choosing whether to get a good dual-core or a good quad-core, then I'd say in this day and age, get the good quad. If the choice is between a good dual and a cheap quad, then go the dual.

### Post by freak42 on 2009-10-19
get the quad if (and only if) you do cpu intensive tasks often.

this may include:

-cutting/converting high end video
-scientific computing
-gaming this and next generation games (windows-games that is)

if you 'only' use it for
2d graphics (gimp/photoshop)
web,email,wordprocessing, casual games 
and you don't mind waiting a bit if you do convert a dvd
or something every once in a while and money is your primary concern
then get the dualcore
