### Post by NikTh on 2012-11-13
Hi , 
is there a specific reason you want to install python 3.3.0 ? 

Ubuntu has python 2.7 pre-installed , also you can install python 3.2 with a command in terminal , no need to download files from anywhere. 

```
sudo apt-get install python3.2-minimal
```You can see the current python installed version with this command in terminal 
```
python -V
```or 
```
python3 -V **#**see what python 3 version is installed**#**
```Thanks
