### Post by sudodus on 2013-01-07
If you get your father's in law system running, check the version with the terminal command

```
lsb_release -a
```

If it is any of

Ubuntu 10.04 LTS, eol April 2013 (quite soon)
Ubuntu 11.10, eol April 2013 (quite soon)
Ubuntu 12.04 LTS, eol April 2017 (5 years support)
Ubuntu 12.10, eol April 2014 (18 months support)

it has support.

Otherwise it has passed end of life (eol), and you will receive no security updates.
--
If you won't get the installed system running, with that hardware, I would suggest that you ***try Xubuntu 12.04 LTS***, eol April 2015 (3 years support)

I think [vanilla] Ubuntu 12.04 LTS needs more horsepower to run well.
