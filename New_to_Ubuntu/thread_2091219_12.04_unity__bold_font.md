### Post by COMECON on 2012-12-04
As you're using Ubuntu 12.04, you can download and install **MyUnity** from the software center. Once installed, navigate to the **Fonts** tab and change the various fonts as you want. I think the one you're looking to is the **Desktop Font**. Just change it to, if you're using the Ubuntu font, Ubuntu Bold.
I hope it helps you.
