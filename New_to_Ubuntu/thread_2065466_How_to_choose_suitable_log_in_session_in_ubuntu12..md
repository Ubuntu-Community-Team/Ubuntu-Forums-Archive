### Post by arroy_0209 on 2012-10-02
The log in session in ubuntu12.04 has four options: recovery console, ubuntu, ubuntu2D, user defined session. How to decide the most suitable option from this list? Also what precise is meant by "user defined session"? In case I install lxde, will there be one more option corresponding to lxde?

---

### Post by 2F4U on 2012-10-02
The most suitable option would probably be "ubuntu", because this is the default desktop. "ubuntu2D" is the environment for less capable graphics cards, that do not support the default desktop. The "recovery console" is, well, what it says. For an explanation what "user defined session" is, look at post #6 of this forum thread:

[http://ubuntuforums.org/showthread.php?t=2041180](http://ubuntuforums.org/showthread.php?t=2041180)
