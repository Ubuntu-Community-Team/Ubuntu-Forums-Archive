### Post by Scunnered on 2009-11-18
If you follow the symbols along the top right of the screen you will find the wireless network connection.  If you click on this it should show you all available wireless connections. Select yours and if asked for a password enter it.

Password required in my case was that provided to my ISP (Virgin Media - UK) and on entering that everything worked smoothly.

If you can't connect based on that please provide more details. If you can advise modem, ISP etc this will assist in providing answers.
