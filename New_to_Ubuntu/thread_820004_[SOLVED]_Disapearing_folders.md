### Post by iaculallad on 2008-06-05
> **Valthinos said:**
> Hey, every time I name a folder starting with "." ( a period), the folder disappears, but when I try to make a new one with the same name, I get a message saying the original is still there. How do I access it?

A . before a folder would make it invisible (hidden attribute). In order for you to see it, press Ctrl+H in the folder where you created that hidden folder. Or click on "View" menu on your nautilus and select "Show Hidden Files"
