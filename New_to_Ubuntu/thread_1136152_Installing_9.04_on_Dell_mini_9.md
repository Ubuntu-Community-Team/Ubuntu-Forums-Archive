### Post by MontelEdwards on 2009-05-07
The best way is to but the live CD on the flash drive.
It is a lot faster, and runs better in my opinion. I like Ubuntu's own USB start up disk maker, and I use that when I can. But if I can I use Unetbootin, which supports Windows and Linux.
Just move the ISO to your computer, delete it off the flash drive, and then run Unetbootin setup. You will have to select the path of the ISO, like is it on your desktop, etc.
You can get Unetbootin from [here:KS]("http://unetbootin.sourceforge.net/")
