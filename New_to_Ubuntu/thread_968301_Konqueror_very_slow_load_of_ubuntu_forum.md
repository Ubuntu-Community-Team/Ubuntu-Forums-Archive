### Post by igknighted on 2008-11-02
There are probably tweaks you can use for konqueror, but for some reason I don't think Ubuntu builds konqueror with the webkit engine, so I wouldn't invest too much effort into it.  Konqueror w/ webkit is probably the best browesr available for linux.

PS... if you are using Firefox on Kubuntu, I would recommend installing either swiftfox or firefox from the firefox website.  If you install the version from the repo's there are a ton of gnome dependencies that get brought in that you really don't need.
