### Post by cariboo on 2013-02-15
What I normally do when I install a new testing version, is just copy the ~/.thunderbird directory to my new home directory. Open Nautilus, and press Ctrl-h to reveal the hidden files and directories, then you can copy and paste the file to however you are going to transfer the directory to the new computer. 

Once copied the directory to your new system, start thunderbird normally, and all your settings including the address book and emails should be there should be there.

If you use enigmail, you also need to copy and paste the ~/.gnupg directory.

### Post by howefield on 2013-02-17
Nautilus is the file manager, open the dash by pressing the Super (Windows) key and start typing nautilus, after a few characters you will see the icon appear, alternatively click on the second from top icon on the launcher, the one that looks like a folder.

Then Ctrl + H to show the hidden files/folders of which .thunderbird is one.
