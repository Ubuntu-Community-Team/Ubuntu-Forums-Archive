### Post by avtolle on 2008-06-13
> **Graahl said:**
> Hi!

Is there any programs for burning CDs and DVDs in Ubuntu Gnome?
Brasero is the default application for this purpose; I've used it quite a bit, with no problems. K3b is another good application, and yes, it is not a Gnome application, but can be installed (along with some other minimal Kde apps and libs) in Gnome, and has also worked well for me. If you want to install K3b,  do it through either the Add/Remove Program utility under Applications, or use Synaptic.
