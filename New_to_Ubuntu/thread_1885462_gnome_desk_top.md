### Post by jjex22 on 2011-11-23
Were you trying to get gnome 2 or 3? 3 is called gnome shell and is available from the software centre. If for old style Gnome. did you follow;

[CharlesA's classic Gnome sticky]("http://ubuntuforums.org/showthread.php?t=1859960") This is for the kind of gnome used in 10.04 -to select, log out, click the cog next to the user login box and select Gnome Classic.

If you installed gnome shell from the software centre (this is the new Gnome 3 without Unity) you just need to log out and select 'Gnome' from the cog next to the user login box.
