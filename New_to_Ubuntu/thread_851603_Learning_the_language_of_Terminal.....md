### Post by iaculallad on 2008-07-06
I'd say, use the **man** terminal command. This command comes with the certain command's synopsis, description, and options.

i.e:

For fcsk command:

```
man fsck
```

For sudo, gksudo, gksu:

```
man sudo
man gksudo
man gksu
```

And so forth...

If you come across with new commands suggested by forum members, it's good to man it first before executing so you know what it does to your system.
