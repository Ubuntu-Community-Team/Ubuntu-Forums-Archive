### Post by mgranet on 2009-04-23
Can't say I've ever had a problem like that. Are you by chance starting a empty session on login, or resuming? In System Settings, under advanced tab, there should be a Session Manager. In there, select the "Restore Previous Session" radiobutton, if it's not already. Not too sure that will help, but it's all I can think of that would cause that.
EDIT: Also, let's say you have your desired wallpaper in your home directory, and you select the option to have the desktop background Slideshow, it will only slideshow the files in /usr/share/wallpapers. If you don't have the extra wallpapers installed from the repos, I think blue curl is the only paper in that directory, and so would be the only one that shows up on reboot.
