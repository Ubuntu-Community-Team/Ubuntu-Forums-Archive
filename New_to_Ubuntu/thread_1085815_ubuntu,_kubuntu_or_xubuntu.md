### Post by perspectoff on 2009-03-03
Be aware that the Gnome desktop (that is the default in Ubuntu) is not as customizable and does not have as much innovative development as does the KDE desktop of Kubuntu.

Many of the current advances in the Linux/Ubuntu world are currently occurring in KDE, in fact. The reviews of Jaunty Jackalope for Ubuntu are tepid so far, while Kubuntu Jaunty Jackalope, anchored by a new version of KDE, continues to rapidly advance and is quite exciting.

I used Ubuntu for two years before I discovered Kubuntu, and am now very frustrated whenever I use Ubuntu (it is too limiting because it does not have as many options as Kubuntu).

Kubuntu has a very similar appearance to MS Windows, so it is also an easier transition for former Windows users (at least Kubuntu Hardy Heron is). 

I personally install Kubuntu Hardy Heron (not Ubuntu) for all new users, especially those who are converting from MS Windows. Most users for whom I have installed Ubuntu went back to Windows XP, because they couldn't get used to the Ubuntu (Gnome) interface. That doesn't happen with Kubuntu.

The current release of Kubuntu, Intrepid Ibex, has a new, beautiful (KDE) interface that I absolutely love and I use it full-time (although I still use Windows for Netflix and TurboTax).

However, this new KDE4 interface of Intrepid Ibex has a bit of a learning curve, and initially had a lot of bugs (which now seem to be sorted out with KDE 4.2, which will be part of Kubuntu Jaunty Jackalope).

So for now, I still think Kubuntu Hardy Heron is the best for new users, especially since it has just had (this month) an overhaul and service pack update.

I always recommend these guides for new users:

Kubuntu guide ([http://kubuntuguide.org](http://kubuntuguide.org))

or

Ubuntu guide ([http://ubuntuguide.org](http://ubuntuguide.org))


PS Xubuntu just plain sucks. It is oriented towards crippled computers with limited RAM, and It doesn't do anything I want it to do. If you have an old computer with limited resources you want to use Linux on, I would use Puppy Linux (or Damn Small Linux) before I would use Xubuntu. Xubuntu doesn't have great documentation, either.
