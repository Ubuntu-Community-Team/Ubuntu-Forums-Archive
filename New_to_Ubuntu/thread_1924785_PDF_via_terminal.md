### Post by dcsoldschool53 on 2012-02-13
I am not sure if this is what you mean, but try the less command. This is what I use to read .pdf files in the terminal only. For example, ```
less enter_your_name.pdf
```displays a readable version of your pdf file in the terminal, and when you are finished reading the file, press the Q key to quit.
