### Post by Titan8990 on 2008-11-24
The following command will do it:


sudo apt-get remove --purge avant-window-navigator


Or your can use the synaptic package manager -> right click on avant-window-navigator -> Uninstall


I found it annoying too unless you set it to auto-hide.

### Post by bobnutfield on 2008-11-24
If you installed it from the Synaptic Package Manager, you can also select for removal the same way.  Just open the Package Manager, look for your app, and double click it, and it will offer to select it for removal.
