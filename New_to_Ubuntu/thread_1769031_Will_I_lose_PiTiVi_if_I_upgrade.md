### Post by Swagman on 2011-05-27
If you "Update" you wont lose it but the repo might be  hashed out.

It wouldn't really matter anyway as a simple

```
 sudo apt-get install pitivi
```

would reinstall it, or install via software centre.


Hashed out refers to the hash mark # that means "ignore this line" in the software sources. in the gui it's a simple tick box

ie: look down my list to see "disabled on upgrade to Maverick" checkbox

[img]http://img62.imageshack.us/img62/5760/ubuntu189.jpg[/img]

Oh.. Be warned... **Upgrades** quite often b0rk so make sure you have backed up important personal data **BEFORE** you begin to upgrade.

Best of luck

### Post by Swagman on 2011-05-27
The program itself wont disable on upgrade anyway... Only the updating mechanism for *THAT* program.

**YES** Pitivi will still be available in the software Centre or Synaptic.

If, by some unfortunate circumstance (I can't foresee everything) it's not in either of those places (and I really can't see that being removed) then google pitivi.. go to their website and follow their destructions for adding their PPA (Software source) and then click on the .deb to install it again.

Really though.. It'll still be in Synaptic (System/Adminstration)

[edit]

Just checked and Pitivi **IS** in both Software Centre and Synaptic.
