### Post by quixote on 2009-06-15
so long as it's booting, I wouldn't worry about the longer time it takes.  I've had that happen (no idea why it does that, and it is annoying) but it doesn't lead to anything worse.

Otoh, if it does lead to longer and longer boot times, or, even worse, messages about any kind of filesystem corruption or the like, then that's different and does need to be dealt with. If it's just a longer boot time, after an update maybe, it's nothing to worry about.

The new FF version should have come through your regular ubuntu updates.  Start Update Manager and tell it to check.  Maybe for some reason it missed that one.

The reason the option to update is grayed out is because firefox is installed system-wide, but you're usually running it as an ordinary user.
