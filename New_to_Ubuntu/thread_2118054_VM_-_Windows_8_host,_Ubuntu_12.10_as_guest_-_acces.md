I currently have decided to just run Ubuntu in a VM for a while on my Windows 8 laptop.  The networking will be by NAT, so Ubuntu will think it is a direct-connect to the Ethernet.  The laptop itself is wireless.  I have a wireless HP printer.  How can I connect to it from Ubuntu in a VM when Ubuntu will think itself directly cable connect to the Ethernet?

### Post by albandy on 2013-02-20
If it's a wireless printer it should have an IP. If Ubuntu can see this ip you can add this printer as a network printer.
