### Post by OutOfReach on 2008-09-13
The program is called file-roller, you can launch it by pressing Alt+F2 and typing in file-roller.
Or you can click on a zipped file and file-roller will open automatically.

### Post by acidsolution on 2008-09-13
You can use zip command 
if not installed install it by 
```
sudo apt-get install zip
```

and to zip filename.doc 
```
zip filename.zip filename.doc
```

To unzip you can use unzip command similarly
