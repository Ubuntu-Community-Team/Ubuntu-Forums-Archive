### Post by wildmanne39 on 2016-05-09
This is a direct quote from lubuntu website:
> We have done many tests and we've found out that Lubuntu can be installed on a Pentium II or Celeron system with 128 MB of RAM, but such a system would not perform well enough for daily use.

With 256MB - 384MB of RAM, the performance will be better and the system will be more usable.

With 512MB of RAM, you don't need to worry much. Your computer probably would not be able to run it and if it did it would not run it well.

You could install a minimum version of lubuntu that should run.
[https://wiki.ubuntu.com/Lubuntu#System_Requirements](https://wiki.ubuntu.com/Lubuntu#System_Requirements)
[https://help.ubuntu.com/community/Lubuntu/Documentation/MinimalInstall](https://help.ubuntu.com/community/Lubuntu/Documentation/MinimalInstall)
