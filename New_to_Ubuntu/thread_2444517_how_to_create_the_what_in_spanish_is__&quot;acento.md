### Post by Impavidus on 2020-05-31
It depends on your keyboard layout and some other settings. If your keyboard config has a "dead circumflex" key, hit that key, then type the letter on which you want to put the accent. Maybe there's a key for this directly, maybe you have to use a modifier key (like AltGr+^). Some layouts have dedicated keys for various accented letters. Maybe you need some modifier key to access them. There's the compose key (if you configured one). You can type compose, ^, some vowel to get the vowel with the accent.

If all else fails, you can use the Unicode code points in hexadecimal. The same applies to all other accents.
