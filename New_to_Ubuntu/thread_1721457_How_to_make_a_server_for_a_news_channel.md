### Post by Hedgehog1 on 2011-04-05
Ubuntu offers a server install that is a good choice for a web site.

If you have not used Ubuntu yet, might I suggest you start with a desktop version to get a feel for it (this same desktop version will likely be how you control and update the web server later).

You can get the desktop version of Ubuntu here:

[http://www.ubuntu.com/desktop/get-ubuntu/download]("http://www.ubuntu.com/desktop/get-ubuntu/download")

And the server version of Ubuntu here:

[http://www.ubuntu.com/desktop/get-ubuntu/alternative-download]("http://www.ubuntu.com/desktop/get-ubuntu/alternative-download")

Will your news channel site be mostly text and graphics?  Or will you also want to have videos available on the web site? What kind of network band width will be available for the web server?
