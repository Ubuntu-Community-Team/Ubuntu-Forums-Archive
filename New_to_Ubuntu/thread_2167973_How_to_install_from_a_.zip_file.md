### Post by Petro Dawg on 2013-08-15
Depends...

Usually Linux compressed files are in ".tar.gz" files.  If you have a .zip, it may be for Windows.
 
If the compressed archive has an executable in it, you may just have to "enable to run as a program" in the file options and then double click it; if you have a bunch of source code in the files you will have to compile and install using terminal (which can be very confusing for new Linux users).

What is the extension on the program?  If it is an ".exe" you will not be able to install it on Linux (without great effort and knowledge of Wine anyways).

Without knowing more about what's in your .zip file, there's not much anyone can help you with.  If there is a README in the archive, I suggest reading that first to get an idea of what you need to do next.
