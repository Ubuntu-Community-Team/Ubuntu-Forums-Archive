### Post by lkjoel on 2010-06-22
Did you buy Autocad?
If so, just type in a Terminal window (Applications->Accessories->Terminal)
```
sudo apt-get update
sudo apt-get upgrade
sudo apt-get install playonlinux
playonlinux
```
Click on install, then search for autocad.
