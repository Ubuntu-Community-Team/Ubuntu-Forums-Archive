### Post by t0p on 2010-12-20
How did you "burn" the .iso to the usb stick?  I don't believe you can do it the same as you would burn to a cd.  

The precise tool and procedure required depends on what computer you're using to "burn" the .iso to the stick.  I have Ubuntu on my computers, and when I want to write an .iso to a usb stick I use a program called [UNetbootin]("http://unetbootin.sourceforge.net/").  If you click on that link, you'll see that UNetbootin also comes in a Windows flavour, so hopefully it will sort your problem.
