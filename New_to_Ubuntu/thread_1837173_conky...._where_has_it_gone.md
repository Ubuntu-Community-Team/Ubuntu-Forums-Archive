### Post by Frogs Hair on 2011-09-01
If you have the installed the default Conky configuration from the software center you can start it by using Alt+F2 or the terminal , type the code , and select run or enter if using the terminal . ```
conky
```

To stop Conky : ```
killall conky
```

The default configuration is not much to look at , so you may want to find a nicer configuration . Here are some configurations and there are many more .
Read the installation instructions carefully .
