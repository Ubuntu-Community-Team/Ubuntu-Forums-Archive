### Post by nhasian on 2010-05-03
installing an operating system is a delicate process.  if it is interrupted by a power loss, i would recommend starting the installation all over again from scratch.

if you can boot from the Ubuntu 10.04 LiveCD or a bootable usb-thumbdrive, backup your /home.

then you can do a fresh installation of ubuntu and restore your data afterwards.

### Post by nhasian on 2010-05-03
on the post screen some computers allow you to choose a boot device.  you need to tell it to boot from the cdrom (or usb thumbdrive if thats what your using)  if your computer does not have this feature, then you will need to enter your bios and change the device boot order to allow it to boot from the cdrom.  be sure to save any changes you make to the bios settings before exiting!
