### Post by snaggletooth on 2009-07-01
You probably have it figured out by now, but...

One way to do it is to create a folder of shortcuts with custom names. For example, create a shortcut to C:\Program Files\Adobe\...\PhotoShop.exe and name it PS.

Same with firefox, then name it whatever (for me, it's 'ff').

Then, have launchy index that folder for *.ink files.

That's it!


Cheers,
