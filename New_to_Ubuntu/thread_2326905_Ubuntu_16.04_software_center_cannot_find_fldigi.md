### Post by CantankRus on 2016-06-05
It's there.
Install via terminal...
```
sudo apt install fldigi
```

Most who have been using ubuntu for a while bypass the Ubuntu Software Center and use **synaptic** or the terminal.
```
sudo apt install synaptic
```
