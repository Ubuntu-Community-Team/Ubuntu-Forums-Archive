### Post by yeats on 2009-10-28
> If anyone would help me with pingus , a lemmings like game , I would appreciate it. It seems to have downloaded all the necessary files but when I try to run it it tells me "no such job" .

I'm assuming this means you went out and downloaded files from the web?  Pingus is in the Ubuntu software repositories (which is the best way to install software on Ubuntu).  Go to the Synaptic Package Manager (under the System -> Administration menu) and do a search for pingus.  Then select it for installation and click Apply.  There are literally thousands of programs available at a click through Synaptic.
