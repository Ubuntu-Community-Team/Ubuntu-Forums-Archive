### Post by t0p on 2008-10-09
> **Chris Edgell said:**
> Will someone please tell me if there is a comprehensive tutorial about threads.  First,I need to learn how to delete the solved threads.....

There is no way for you to filter out threads you don't want to see.  We all see the same lists of threads.

The [Solved] threads are useful anyway.  If you have a particular issue, you can do a search on a key word, and if the search results include [Solved] threads you may find the solution to your problem there.  Though I would recommend you use Google rather than this site's search engine.  You can use Google to search this site by going to [www.google.com]("http://www.google.com") and typing into the search field:

```
"whatever you are looking for" site:ubuntuforums.org
```

Google will then confine its search to this site.
