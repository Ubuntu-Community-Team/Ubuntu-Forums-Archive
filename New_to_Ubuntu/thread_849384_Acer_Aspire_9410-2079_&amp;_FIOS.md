### Post by ZabiGG on 2008-07-05
Hi there ;

I may be wrong, but from what I've gathered, using Wubi is using Ubuntu as if it were a virtual machine under Windows. 

If that assumption is right, it means Ubuntu cannot talk to your hardware directly, which would explain your connection problems.

If you want to try Ubuntu while actually using your hardware, the best solution will be what is called a "dual-install," so that you may keep Windows and start exploring Ubuntu while enjoying it to the fullest.

There is a great guide on how to dual install here:

[http://users.bigpond.net.au/hermanzone/](http://users.bigpond.net.au/hermanzone/)

But before you leap, you might want more information. Just click on the "Look here" link in my signature below for links to great starter info.
