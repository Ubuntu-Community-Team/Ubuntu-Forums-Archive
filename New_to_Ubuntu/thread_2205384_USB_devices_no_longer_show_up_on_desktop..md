### Post by monkeybrain20122 on 2014-02-13
You can make it show up on the desktop. Install gnome-tweak-tool. Open it, go to Desktop and change "Show mounted volumes on the desktop" to "on"

Edited: this works with Unity and gnome-shell, I suppose it would work with classic gnome as well.
