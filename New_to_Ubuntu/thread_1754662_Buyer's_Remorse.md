### Post by blauendonau on 2011-05-12
If you don't like Unity, that's easy enough to change: just log out and, at the log in screen, select your username and find the menu at the bottom of the screen that allows you to change desktop environments. "Classic" should be one of the options, try logging in with it.

If you didn't like Unity, you probably won't like GNOME 3 either. I've seen demos of it, and it's a big change from "classic" gnome, it looks a bit like Unity from what I can tell.

Unfortunately, Classic GNOME will not be available starting from next Ubuntu version, 11.10 (actually the GNOME team is going to drop it altogether, which I find sad). If you want to continue using a desktop environment that looks like classic GNOME in the future, I recommend XFCE. It's highly configurable and you can make it imitate GNOME with a few tweaks.
