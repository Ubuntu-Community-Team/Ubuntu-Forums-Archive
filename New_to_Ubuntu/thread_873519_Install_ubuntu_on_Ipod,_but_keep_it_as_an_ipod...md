### Post by ilrudie on 2008-07-29
You can probably do this but it will wear out the ipod's hard drive faster.  The disk in an ipod is designed to be written to very rarely and read nearly sequentially.  Random reads and writes will put more strain on it and wear it out faster.  I'd recommend looking into other alternatives like putting puppy or dsl on a flash stick.
