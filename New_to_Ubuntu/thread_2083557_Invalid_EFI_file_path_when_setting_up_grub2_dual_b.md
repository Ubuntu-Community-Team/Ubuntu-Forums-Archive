### Post by Unewbeginner on 2012-11-12
Hi everyone, I installed my Ubuntu 12.04 successfully but I can't set up dual boot with windows 7 (64bit). The two systems are on two seperate hard drives. I searched online and found that boot-repair might fix the problem automatically. The windows entry generated by boot-repair as below:

```
### BEGIN /etc/grub.d/30_os-prober ###
menuentry "Windows 7 (loader) (on /dev/sdb1)" --class windows --class os {
	insmod part_msdos
	insmod ntfs
	set root='(hd1,msdos1)'
	search --no-floppy --fs-uuid --set=root F8AAB60DAAB5C880
	chainloader +1
}
### END /etc/grub.d/30_os-prober ###
```

but when I restarted I still can't get into windows system. The error is 

```
Invalid EFI file path
```

How can I fix it?

Thanks in advance

### Post by Gone fishing on 2012-11-13
As you are using 2 different drives why bother with EFI? Why not for Ubuntu use oldfashoned mbr with grub on the mbr of the Ubuntu drive and the EFI system with Windows on the other hard drive. 

Then you can boot into either system using BIOS just select the drive you want to boot from - boot from one drive Windows the other Ubuntu.
