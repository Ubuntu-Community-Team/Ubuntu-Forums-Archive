### Post by SunnyRabbiera on 2008-05-11
eh its going to be hit or miss.

for a vista-like theme I rather like this gtk theme:
[aero-clone]("http://www.gnome-look.org/content/show.php/Aero-clone?content=57352")
this icon theme:
[vista inspirate]("http://www.gnome-look.org/content/show.php/Vista-Inspirate?content=28352")
and this window decoration (needs emerald):
[vista basic]("http://www.gnome-look.org/content/show.php/Vista+Basic?content=44228")

your start menu request is a little harder, the one I like best is the mintmenu that you can get on this page:
[the repository]("http://www.linuxmint.com/repository/")
all packages there are ubuntu compatible.

for OSX thats a little harder...
