### Post by Sarah L on 2008-06-19
You'll have to set the file to executable before you can run it.

There are two ways of doing this:

1) Right click on the file, select Properties, and then under Permissions check the Is Executable box.

2) Open up a terminal window (usually you can just hit F4 in your file manager), and in the directory you downloaded the file to, type ```
chmod +x PlaneShift-v0.4.00-x86.bin
```
