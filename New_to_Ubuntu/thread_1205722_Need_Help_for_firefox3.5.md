### Post by enli on 2009-07-06
If installing from .tar.gz file seem difficult for you, you can do this from GUI way.

Go to System > Administration > Synaptic. Click Reload > Search for firefox and mark firefox-3.5 for installation and apply changes.

Or terminal way,
```

sudo aptitude update
sudo aptitude install firefox-3.5

```

Access it from Applications > Internet > Sheriteko Internet browser (if i remember correctly, not on ubuntu right now). But this approach will overwrite your old firefox profile. Nothing serious really, but if you dont want that you can use .tar.gz approach as suggested by someone above.
