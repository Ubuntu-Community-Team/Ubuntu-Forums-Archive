first, understand that you will not be able to save your programs.  Windows programs are not designed to run in linux.  
Second, you will be able to dual boot regardless of which install cd you use. 
As far as your files, back them up and put them into your new (improved) Xubuntu system, find programs that can read them (for example - office files - OpenOffice) and you will be fine.  BACK THE UP TO AN EXTERNAL HD!!

Live CDs will always run slower, as reading from a CD is always slower than a HD.

Finally, once you have installed the system, check out this thread:
[http://ubuntuforums.org/showthread.php?t=854557](http://ubuntuforums.org/showthread.php?t=854557)
and you should be good to go!

### Post by Bucky Ball on 2008-09-23
Back up anything you don't want to lose. Download the Xubuntu i386 alternate install cd. Boot from it. When you get to the partitioning section, delete all partitions then create three and make them EXT3 type partitions (the swap is EXT3 automatically so no need):

/
/home
/swap

It might look a bit confusing now, but when you play around in there you'll get what I mean. That should give you a machine with Xubuntu up and running. Remember with the install, you have your valuable data backed up so you are free to experiment and make mistakes (which means you learn as you go!) without fear of losing anything. \\:D/

Once you have Xubuntu up and running you can re-post with any new problems that arise (better than adding them to this thread). :)

### Post by Idefix82 on 2008-09-23
One more thing: you do need to specify the size of the partitions. Make the swap partition roughly double the size of your ram (maybe even slightly more, since you have so little ram) and the / partition gets the rest. Normally you would also want to create a /boot partition and a /home partition at least, maybe also some others.
