### Post by rainwalker on 2008-08-02
If you're talking about files I download, they usually fit into the Pictures, Movies, Music, or Documents folder. For other stuff that I'm not sure if I'll need again, or if I just need a temporary place to put something, I made a Stuff folder. When I know I'll need to come back to something, I save it to my Desktop so I'll see it (otherwise I try to keep everything off my desktop). I also have a Computer folder for anything related to software and stuff like that.
For GTK, GDM, and icon themes I've downloaded, I just delete the archives after I install them (no need to keep them around).

GTK stands for the GIMP toolkit
GDM stands for the Gnome Display Manager (the login screen)
The tilda (~) is an abbreviation for your Home Folder.
