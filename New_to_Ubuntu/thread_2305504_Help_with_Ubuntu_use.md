### Post by grahammechanical on 2015-12-06
On the menu that we use to shut down Ubuntu there is Ubuntu Help. That will load the Ubuntu Desktop Guide. It is a fine place to start for those new to Ubuntu.

Ubuntu is designed so that we can do most usual stuff without needing to use commands. If you browse the help sections of this forum you will see certain commands recommended. Make a note of those commands and work out what they do. This way you will learn some of the most useful commands.

There are web sites devoted to explaining Linux commands or the commands for the Bash shell which is what the command line really is.

[http://ss64.com/bash/](http://ss64.com/bash/)

[http://linuxconfig.org/linux-commands](http://linuxconfig.org/linux-commands)

[http://www.linux.org/categories/linux-tutorials.52/](http://www.linux.org/categories/linux-tutorials.52/)

As for books, see what is available in book stores as well as checking out Linux magazine. This magazine always has a section on the command line and back issues are available for download.

[http://fullcirclemagazine.org/](http://fullcirclemagazine.org/)

If you are going to experiment with commands, then I suggest that you do it in a spare installation of Ubuntu. Then when you break it you can simply re-install.

Regards.
