### Post by grahammechanical on 2014-01-05
We learn Ubuntu by learning how to use the user interface. We learn commands when we want to learn Linux which is the underlying operating system that the components of the Ubuntu distribution sit upon. There is a sticky thread right at the top of this section listing user's suggestions for Linux learning resources.

[http://ubuntuforums.org/showthread.php?t=2015424](http://ubuntuforums.org/showthread.php?t=2015424)

Another way to gain experience is to read the posts in the help section and note the advice and then carefully experiment using some of the commands. Especially those commands that produce information about the system. Write down useful commands.

Regards.
