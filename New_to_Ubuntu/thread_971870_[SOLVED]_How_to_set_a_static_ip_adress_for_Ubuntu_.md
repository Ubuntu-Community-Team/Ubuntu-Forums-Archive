### Post by platinumits on 2008-11-05
Go to System Menu, Administration, Network
Click the Unlock bottom
Type your password
Select your Network card and click the bottom properties
Disable the option: Enable Roaming Mode
Select Static IP Address from Configuration in the screen
Type the information at each, IP, Subnet and gateway address
Click Ok, Click Close bottoms....and Done.

### Post by reg4c on 2008-11-05
Hey, 

right click on the Network Manager icon in your notification area and click on Edit Connections.

Then go to the Wired tab and click Add.
Select system settings and connect automatically.
Select the IPv4 tab and choose Manual under the Method.
Click on Add.
Type in the Address, Mask and Gateway.

Click on close.
