### Post by taseedorf on 2010-01-17
Yes, the operating system is stable Hence, it was released.  Being that your screen is blank doesn't affect if the operating system is stable or not.  All problems one has with Ubuntu is because of hardware incompatibilities or incorrect settings.  Probably 90 percent of the existing home user hardware is capable of running it properly.  

OBVIOUSLY, yes, there are bugs.  Many major ones are correctly quickly after release, but some do take time.  It sounds like during the upgrade, the x.org file could have been changed.  Sometimes this can happen, especially if a different driver for your hardware is used with a new kernel.
