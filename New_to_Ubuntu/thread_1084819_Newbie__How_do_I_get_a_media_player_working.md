### Post by Ms_Angel_D on 2009-03-02
You want do download the .deb file for real player you can find it here. [http://www.real.com/linux](http://www.real.com/linux) (look under the Download Realplayer Button and select Deb Package)

As far as media is concerned open add/remove programs and make sure show all available applications is selected and then search for Ubuntu Restricted Extra's and install that package. It will install the proper media codecs.
