### Post by mikechant on 2009-03-16
The Atom processor is ideal for low power consumption - this is probably the processor you want if your priority is long battery life on a laptop/netbook. The Core 2 duo is well suited to desktop machines, or Laptops where processor power is more important than battery life (e.g. mostly run from mains).
Ubuntu (and all mainstream Linux distros) fully support multi processors and multi core processors, although indivdual applications may only use one core at a time.
