### Post by MAFoElffen on 2023-03-12
What you must have pressed was the icon to "Eject" or unmount.

Just go to "Other Locations" and select them again, so that the File Manager will mount them again.

If you really want them mount all the time (at boot up), then add them to your fstab file...
