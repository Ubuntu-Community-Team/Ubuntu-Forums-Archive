### Post by tom56 on 2009-04-28
Generally, no. By default there are no ports open in Ubuntu so there is no need for a firewall. Obviously, this is not the case if you have installed server software.

You can check how secure you machine is (in Windows or Ubuntu) here - [https://www.grc.com/x/ne.dll?bh0bkyd2](https://www.grc.com/x/ne.dll?bh0bkyd2)
