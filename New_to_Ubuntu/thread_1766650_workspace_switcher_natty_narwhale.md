I don't follow that. If you don't want to use the keyboard shortcuts that fritz269 posted (which work very well) and you want to use the workspace switcher with a mouse, this is what you do:

1 - Click on the workspace switcher icon. You see the default 4 virtual desktops in one screen.

2 - Double-click on the virtual desktop you want to use. It fills the whole screen.

### Post by dwhite on 2011-05-24
when you are in the new workspace and want to open up say a new firefox window, right click on firefox in the launcher and select "open a new window", a new firefox window will open on that workspace.

you should now be able to switch from one workspace to another and have a separate firefox window on each workspace.

i think this is what you wanted to do :)
