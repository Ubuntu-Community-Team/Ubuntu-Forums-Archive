### Post by NightwishFan on 2009-08-06
You should use the flashplugin-nonfree package if you do not mind accepting adobes licence. (Which is free just not open source).

You can use Add/Remove under the applications menu to find and install ubuntu software. Or a more advanced one under System -> Administrator -> Synaptic Package Manager

If you want to view local .swf files, you can try vlc player.

### Post by starcraft.man on 2009-08-06
Best is relative. The adobe flash player is their implementation. The swf and gnash players are attempts to make a free implementation so we won't need adobe's flash player. Flash in general doesn't give great performance. If you just want the regular flash from adobe, install the **flashplugin-installer**, that's what most people use.

You might also consider the **ubuntu-restricted-extras** package. It will give you lots of extras like codecs, java, things most people generally want.

If you need help installing, see sig or do as Nightwish noted.
