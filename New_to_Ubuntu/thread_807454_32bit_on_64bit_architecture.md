### Post by tamoneya on 2008-05-25
it wont run at its full potential but you should be fine.  Generally what I recommend is that you run 32 bit unless you have 4GB or more of RAM in which case you should try 64.  I think it just isnt worth the extra work unless you are hitting the 3.2GB limit in 32 bit OS with a standard kernel.
