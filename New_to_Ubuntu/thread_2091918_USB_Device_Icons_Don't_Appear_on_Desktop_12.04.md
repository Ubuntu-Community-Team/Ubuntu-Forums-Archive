### Post by monkeybrain2012 on 2012-12-06
You should see the device mounted on the Unity launcher. If you want it to show up on the desktop, install gnome-tweak-tool, then go to the dash and search for "Advanced Settings", go to "Desktop" and turn on "show mounted volumes on desktop".
