### Post by anewguy on 2012-07-14
If the driver is not included with Ubuntu, then first try the Additional Drivers (of course, this will require a network connection of some sort).  If that doesn't work or you don't have access to temporarily connect via hard wire, then you will probably be left with getting the driver (or keeping a copy) and installing it each time a new release is available.  This can also happen with kernel updates as well.
