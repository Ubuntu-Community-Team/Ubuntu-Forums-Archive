### Post by trikster_x on 2010-10-20
If your program has a cli (command line inteface), it likely has a man or help page built in as well.  Best place to start is to type
```
man genesis
```

or

```
genesis --help
```

These commands will generally give you a list of syntax and options you can use to get your program started
