### Post by iaculallad on 2008-05-26
Navigate to:

System->Administration->Synaptic Package Manager

Search for the "KDE Desktop" string in the Search Box.
When search completes, scroll down and locate "kde" Description will be "the K Desktop Environment Official modules"....

Right on the item and Click/Mark for Complete Removal.

Hit Apply button at the top to remove/uninstall.
