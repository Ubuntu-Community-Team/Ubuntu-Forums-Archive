### Post by Copernicus1234 on 2009-08-20
Depends on what package type it is. If its a deb package, you can double click it. If its a archived file, you need to unpack it and install it yourself somewhere.

For a beginner, its best to install things from the repositories using System -> Administration -> Synaptic Package Manager.
