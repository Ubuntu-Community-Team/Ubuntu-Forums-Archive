I would offer that I have used Virtual Box on Ubuntu with out issue.

The biggest challenge you face, is having enough ram to comfortably run your existing system (Ubuntu) and allocated ram (in Virtual Box) for the test system (in your case, XP).
Adjusting video ram in Virtual Box for your test system helps as well.  And setting the hardware parameters to match the hardware you are using may help.
Though, that is what is nice about Virtual Box, you can set the hardware parameters to what you like to test that configuration of hardware.  That said, it would be good to make sure the hardware parameters (in Virtual Box) meet the hardware requirements of XP.:)
