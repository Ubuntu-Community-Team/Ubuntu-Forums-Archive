### Post by undecim on 2010-03-31
Have you check System -> Administration -> Hardware Drivers (You need to be connected to the internet for this)

If you've already installed any video drivers there or don't have any to install, open a terminal and type
```
xrandr
```
And post the output here.

Also do
```
lspci | grep VGA
```
and post that output here as well.
