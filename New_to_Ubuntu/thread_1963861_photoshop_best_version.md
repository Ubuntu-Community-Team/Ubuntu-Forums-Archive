### Post by anewguy on 2012-04-23
If you want to be 100% Linux, then I would install Windows in a virtual machine in Ubuntu and install PhotoShop there.  Windows will just run as a process within Ubuntu - you can even have it automatically startup when you start Ubuntu.  That way Windows, and PhotoShop, is just a click away anytime.  I would recommend more memory though if possible.  I also know that, at least for virtualbox, the graphics have improved but I've read (I don't do gaming) that they still lack for running heavy-video games.  I don't think this would make any difference in PhotoShop though.

### Post by souravc83 on 2012-04-23
Are you running Gnome 2?
I haven't used 11.10, but if you install gnome-shell, you get gnome 3. 
To do this, go to your terminal and type:

[HTML]sudo apt-get install gnome-shell[/HTML]

Then, log out, and when you log back in, click on the gear icon to go to gnome.
Here is a pictorial description, if you like:
[http://techhamlet.com/2011/10/install-gnome-shell-in-ubuntu-11-10/](http://techhamlet.com/2011/10/install-gnome-shell-in-ubuntu-11-10/)

After you do so, please visit:
[https://extensions.gnome.org/](https://extensions.gnome.org/)

Click on whichever extension you like, and drag the slider to the left of the page from off to on. And there you go! you have the extension instantly on your computer. Remarakably easy, ain't it!
