### Post by ubuntu27 on 2012-05-07
It seems that the original poster did not install Ubuntu Linux yet.

HI jainexgroup. What you have to do first is to download the Ubuntu ISO file from [here]("http://www.ubuntu.com/download/desktop/alternative-downloads")

You want the file that says "desktop" not "alternate"

The Desktop CD let's you test the Operating System before installing by running it "live" from within the CD. This means that it won't touch anything that it is on the computer. It is safe.

If you see that the Desktop CD (a.k.a Live CD) it runs fine on your computer. then you can proceed in installing it on your hard drive.

If possible, try downloading it with bittorrent, since that will make sure that the file is downloaded without any errors or corruption.


[Ubuntu Linux Resources]("http://www.psychocats.net/ubuntu/") has excellent guides on how to get and install Ubuntu. I highly recommend it.

[How to get Ubuntu]("http://www.psychocats.net/ubuntu/getting")


PS: If you have a 64-bit microprocessor, then get the ISO file that says "ubuntu-12.04-desktop-**amd64**.iso.torrent"

### Post by morhin on 2012-05-08
I'd suggest using Ubuntu version 10.04 rather than 12.04.

10.04 has been out awhile and a lot of the bugs have been worked out or are at least manageable. 12.04 is new and it has a ways to go before I'll feel comfortable moving into it. Watching the forums I see things happening with 12.04 that would frustrate me.

As I've only been using Ubuntu for about 6 months and am a non-geek I can't even understand half of what these guys on the forums are talking about.

I found a book (Ubuntu for Non-Geeks by Rickford Grant with Phil Bull for version 10.04) that walked me through everything I needed to get me started with Ubuntu. If it hadn't been for this book I'd probably have gone back to windows.
