### Post by mapes12 on 2010-06-26
Ubuntu is internet dependent. How did you make your post if you can't access the internet? If you have another machine that can access the web there are ways to put packages on removable media which can then be installed on your Ubu machine - aptoncd. It's in Synaptic.

For codecs you will need ubuntu-restricted-extras from Synaptic and also install the packages recommended at [http://www.medibuntu.org/](http://www.medibuntu.org/)

### Post by mac9416 on 2010-06-26
> Ubuntu is internet dependent.

Not true. Ubuntu can be run offline. It is difficult to install software without an Internet connection, but tools such as [Keryx]("http://keryxproject.org") and [APTOnCD]("http://aptoncd.sourceforge.net") make it much easier.

ayeshrajans, you'll need to install the ubuntu-restricted-extras package to get multimedia codecs. Check out those tools and let me know if you have any questions.
