### Post by dearingj on 2009-09-19
1) Open a terminal (Applications->Accessories->Terminal)
2) type "sudo apt-get install djview"
3) Put in your password when asked (it won't show your password)
4) Close the terminal
5) Run djview (Applications->Graphics->DjView)
