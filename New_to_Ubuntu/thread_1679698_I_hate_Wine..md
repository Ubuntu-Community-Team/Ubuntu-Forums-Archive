Wine is a bit of a pin , it works well for relatively simple applications, but not so good for more complex applications.

Configuring wine often requires archaic knowledge of both Linux and Windows.

Configuring wine is often application and wine version specific.

Your best source of information is the wine web site, see the application database.

[http://appdb.winehq.org/](http://appdb.winehq.org/)

Complaining here, without posting any details, application, version of wine, error messages, description of the problem will not help you fix your problem and so I am going to close this thread.

Because of the difficulty with wine, most people dual boot or run virtualization (Virtualbox).

Linux is not a drop in replacement for windows and you should try to let go of your windows applications in favor of linux equivalents, although I understand this is sometimes not an option.

Best of luck to you.
