### Post by seawolf167 on 2011-04-09
Both ATI and Nvidia have pretty good graphics card support for linux now. I personally like their proprietary driver suites (available for download on their website). If you haven't tried that yet - I'd try that first.

As for possible graphics cards, I am running dual 5870's (in crossfire mode) with no problems. I don't have a single 2500x1500 monitor, but two 1920x1200s run perfectly (I did hook up 4 1920x1200 just for fun when I first installed the cards (worked perfectly), but I have nothing to use 4 monitors for lol)

### Post by seawolf167 on 2011-04-09
Crossfire/SLI is usually only supported for specific graphics cards, and then usually just for cards of the same type.

[Here]("http://game.amd.com/us-en/content/images/crossfirex/CF_combo_chart.jpg") is ATI's compatible crossfire hardware chart. 
[Here]("http://www.geforce.com/#/Hardware/Technologies/sli/supportedgpus") is Nvidia's compatible SLI hardware.

Basically, unless you have supported card(s), you'll only want to have one graphics card in your computer
