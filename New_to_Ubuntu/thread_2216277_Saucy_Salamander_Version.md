### Post by Deuce1912 on 2014-04-11
> **Kevin McCready said:**
> How can I know if my linux is latest Saucy Salamander? Thanks

In terminal type the command:

```
lsb_release -a
```

For example, mine says: 

```
Distributor ID:	Ubuntu
Description:	Ubuntu 12.04.4 LTS
Release:	12.04
Codename:	precise
```
