### Post by cariboo on 2008-07-02
It really depends on if that is the disk it boots from normally, otherwise yes it should work. Ubuntu is not like windows, it may complain about a module or two but it should run just the way it did in the old computer. The only real problem I can think of is if the video card is of a different manufacturer.
