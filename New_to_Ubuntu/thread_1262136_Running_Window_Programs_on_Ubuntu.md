### Post by ghostborg on 2009-09-09
Check out Ubuntu's Wubi installer.
[http://wubi-installer.org/]("http://wubi-installer.org/")

Wubi is an officially supported Ubuntu installer for Windows users that can bring you to the Linux world with a single click. Wubi allows you to install and uninstall Ubuntu as any other Windows application, in a simple and safe way. Are you curious about Linux and Ubuntu? Trying them out has never been easier!

I have been running this way for several months now.
You might get lucky with Wine but most likely you will need Virtualbox to run Windows within Ubuntu. You are probably better off using a dual option, Wubi installer doesn't frack you system if you decide to change.

Checkout [www.getdeb.net]("www.getdeb.net")
choose your platform and look for apps.
