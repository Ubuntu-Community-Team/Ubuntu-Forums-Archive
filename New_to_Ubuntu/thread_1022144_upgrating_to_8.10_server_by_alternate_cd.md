### Post by Zill on 2008-12-26
I believe the server editions are intended to be Long Term Support (LTS) versions.

8.10 is a "standard" release and therefore *not* LTS.  8.04 is the current LTS release and is supported until April 2013 (Server).

[https://wiki.ubuntu.com/Releases]("https://wiki.ubuntu.com/Releases")
[http://www.ubuntu.com/products/ubuntu/release-cycle]("http://www.ubuntu.com/products/ubuntu/release-cycle")
