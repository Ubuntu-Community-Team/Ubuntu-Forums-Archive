### Post by Zill on 2013-07-01
ajgreeny:  With Claws Mail 3.8.1 images appear inline in received messages if the following preference is set:
Configuration - Preferences - Message View - Image Viewer - Display images inline

With the "Fancy HTML Viewer" plug-in enabled, HTML graphics also appear inline.

Note:  Claws Mail is closely related to Sylpheed.
