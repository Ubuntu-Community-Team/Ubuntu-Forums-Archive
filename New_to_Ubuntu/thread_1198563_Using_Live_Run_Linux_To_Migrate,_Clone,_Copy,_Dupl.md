Some additional information:

Parted Magic is a live run utilities suite that techs and hobbyists will enjoy! It contains Gparted, Clonezilla, Firefox and a secure erase utility.
http://www.partedmagic.com/

Damn Small Linux is a live run and installable micro-distro at only 50MB! This distro will probably run on just about any computer you have, no matter how old!
http://www.damnsmalllinux.org/

Puppy Linux is a live run and installable mini-distro at only 100MB! This distro contains Gparted, SeaMonkey, and other delightful applications.
http://www.puppylinux.org/

Clonezilla is a live run stand alone cloning utility.
http://www.clonezilla.org/

Super Grub Disk is a multi function bootable repair and recovery utility application.
http://www.supergrubdisk.org/
