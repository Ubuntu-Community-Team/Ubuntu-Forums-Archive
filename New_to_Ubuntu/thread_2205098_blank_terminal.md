### Post by deadflowr on 2014-02-12
I, as always, assume that this is default Ubuntu.
If so, then if you push your mouse to the top of your screen, then the terminal menu should appear.
(try pushing the mouse toward the left side of that top panel, and the menu should appear.)
If this works like it should, you should see a menu an option for "Edit".
Click on Edit and in this dropdown go to "profile preferences".
In here should be a setting for "Colors".
The rest should be easy enough to figure out.
Set the colors however you want, and click on the close button to exit.
You should be able to see the changes as you do them.
