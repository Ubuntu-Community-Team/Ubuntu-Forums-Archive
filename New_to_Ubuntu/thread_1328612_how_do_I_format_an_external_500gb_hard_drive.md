### Post by ajgreeny on 2009-11-16
When and if you have deleted files and folders from it in the past, how have you done it, and how much have you deleted?  If you just hit the delete button, all those files/folders will still be taking up space in the external disks own trash folder.  Certainly formatting will get rid of everything, but that is perhaps overkill if you can just empty the trash from the disk.

### Post by corncob on 2009-11-16
The .Trash folder is hidden.  In the file browser open the view menu and click on "show hidden files".  You might not be able to delete the .Trash folder itself until after you delete the items in it.
