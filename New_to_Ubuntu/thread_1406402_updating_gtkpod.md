### Post by 2hot6ft2 on 2010-02-14
He must have had a version of ubuntu that was prior to hardy that is an old thread. To install gtkpod-aac just open a terminal
Applications > Accessories > Terminal
and use:
```
sudo apt-get install gtkpod-aac
```
Hit Enter
Enter your password which wont be displayed and hit Enter
Wait for it to finish and it will be in
Applications > Sound and Video > gtkpod iPod Manager
