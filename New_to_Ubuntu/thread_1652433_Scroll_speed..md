### Post by crichard on 2010-12-24
Preferences (Ctrl+F12) -> Advanced -> Shortcuts -> Middle-Click options

In select middle mouse click action

Make sure to pick start panning & Enable horizontal panning.

That's it. If you middle click the mouse on a web page, you can scroll through a page by moving the mouse up and down. You don't have to use the scroll wheel. :)

### Post by The Chef on 2011-05-13
OK SOLVED.

I found this fantastically simple solution worked for me:

[http://ubuntuforums.org/showthread.p...24#post8583324](http://ubuntuforums.org/showthread.p...24#post8583324)

Basically, unplug wireless USB mouse dongle while the PC is running, count to 3, plug it back in.  All fixed.  You still can't adjust the wheel sensitivity, but mine is now at a sensible scroll rate.
