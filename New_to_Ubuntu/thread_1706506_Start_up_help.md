### Post by Rubi1200 on 2011-03-14
Personally, I recommend not using Ubuntu Tweak. You can achieve the same results by going into Synaptic and searching for and removing older kernels.

However, it is advisable to keep at least one older kernel for troubleshooting purposes in the event an update/upgrade proves to be problematic.

To remove older kernels:
[http://www.howtogeek.com/howto/17787/clean-up-the-new-ubuntu-grub2-boot-menu/](http://www.howtogeek.com/howto/17787/clean-up-the-new-ubuntu-grub2-boot-menu/)

### Post by grahammechanical on 2011-03-14
I have found Grub Customizer to be very good. It does not remove those extra kernels. At least I have not tried to use it for that but it can hide menu entries and move them up the list. So, you might want to keep Memory Test just in case you might need to do it. But you might not want to see it. And you can move Windows 7 (loader) up near the top if that is more convenient for you. Grub Customizer will also make it easy to put a image behind the Grub menu.
