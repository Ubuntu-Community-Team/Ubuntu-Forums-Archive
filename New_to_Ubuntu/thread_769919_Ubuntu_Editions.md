### Post by LaRoza on 2008-04-27
> **confrontation said:**
> what Ubuntu Edition is right for me?

ubuntu
kubuntu
edubuntu
xubuntu
gobuntu
what are all the different ones for?

Ubuntu - Uses GNOME
Kubuntu - Uses KDE
Edubuntu - Uses GNOME, but is meant for classroom use
Xubuntu - Uses Xfce (sort of like GNOME)
Gobuntu - Uses GNOME but is strictly free software as defined by the FSF

They just have different defaults.

I suggest using Ubuntu. You can install the other packages on it if you want after.

If you want to see screenshots, check wikipedia or google for "GNOME", "KDE" and "Xfce".

### Post by SunnyRabbiera on 2008-04-27
Each one is targeted at a different audience.
Ubuntu is targeted at the general linux audience, and the newcomer as well.
Kubuntu is targeted at KDE users, you see in linux there are a multitude of user interfaces for one to play with unlike in windows and OSX where you are locked down to one interface. Ubuntu by default runs Gnome, but some people don't like gnome so Kubuntu was created.
Xubuntu same thing, only Xubuntu comes with XFCE another user interface
Gobuntu is for those free software only folk, some people feel that Ubuntu comes with too many proprietary drivers and such so Gobuntu was made for them.
Linux and Ubuntu are about diversity and finding what works for you, you can actually try each one to see what suits you best for free so no worries.
