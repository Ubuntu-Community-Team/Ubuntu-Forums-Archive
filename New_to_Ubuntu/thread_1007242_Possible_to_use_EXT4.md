### Post by mikechant on 2008-12-10
binbash said > You can use it at next release :

I would emphasize the *can* (as opposed to *should*)

Although by all accounts it is pretty stable, given how critical file system integrety is I would wait another 1-2 years unless:
a) You want to just try it out as a test partition
or
b) You really need some of the new features.

Maybe I'm over-cautious, but I certainly wouldn't use it immediately just because it's been labelled stable. The thread linked to above mentions 'hundreds' of testers; I'd rather wait till that was 'tens of thousands', (e.g. those brave pioneers running Fedora!)
