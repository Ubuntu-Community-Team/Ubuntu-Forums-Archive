### Post by achase79 on 2009-05-07
The permission structure in Linux makes it very difficult for a virus to be installed.  Viruses are OS specific and mainly target Windows.  Open Source makes viruses/malware/spyware very hard to distribute in linux as well because everyone has the opportunity to look at the code.

### Post by fatality_uk on 2009-05-07
I would suggest that if you regularly exchange files with Windows users, and you actually LIKE the people you exchange files with, you could install an AV app. This won't have much effect on your system, but will help prevent you transmitting a potentially infected file to a friend.

Where I use Linux, in a mixed office environment for instance, I will usually make sure I have an AV running.
