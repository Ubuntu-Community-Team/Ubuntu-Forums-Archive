### Post by Wim Sturkenboom on 2011-08-01
Last time I checked (it's a while ago), Ubuntu fitted in about 4 GB.

You need space for extra software, updates as well as documents. I would say that 10GB is the minimum, 20GB is nice and 30GB should be sufficient for as long as the install lasts.

By the time you need more, you're hardly using windows so it will be time for a dual boot or an Ubuntu only installation.

### Post by amjjawad on 2011-08-01
> **tmtorvalds said:**
> I am using Wubi to install Ubuntu, ubuntu recommends at least 10 GB for installation size, my windows computer that I am using along-side has 385 GB, and can't decide what installation size I should choose. I am thinking about setting it to 17 or 20GB. Someone said 30GB is good also.

If by using Wubi you want to try Ubuntu and nothing more, don't worry much about size as even 4GB would be enough.

If you are going to install apps, etc then 10GB might be good.

If you are thinking to use Ubuntu for a longer time then Wubi is definitely not the right choice. Dual-Booting is the best approach OR [Virtualbox]("http://www.virtualbox.org/").
