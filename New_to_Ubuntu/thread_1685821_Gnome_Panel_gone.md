### Post by guruoogway on 2012-09-29
In Cairo dock enable 'Application Menu'. Now from this menu start Terminal.
Type in terminal 'gconf-editor' , press enter. Gconf-editor will start.
Go to " /desktop/gnome/session/required_components/panel "
In front of field "panel" write 'gnome-panel'.
Restart your computer. Gnome panel is back but without any icons , add icons from 'Add to panel' option.

### Post by berserkus on 2013-05-23
I had the same problem, and found a quick and easy solution for it:

Once you run the gnome panel from terminal as you described go to: System > Preferences > Startup Applications

Click "Add"

In the "Command" field write: gnome-panel
Name it whatever you like and save.

Next time you reboot the gnome panel is up an running.
