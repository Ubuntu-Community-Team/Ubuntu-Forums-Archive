### Post by kanikilu on 2008-12-18
I have no idea what DE Linpus uses, but from the screenshots it looks similar to the default OS on the ASUS EEEPC, which is a custom tabbed interface with IceWM.

As for Ubuntu on a netbook, the standard version should work fine. The only limitation you'll likely run into is screensize. On my 7-inch ASUS, I end up having to ALT+Right Click alot to drag windows around so I can access buttons and other things that run off the screen. On a netbook with a larger screen, that shouldn't be as much of a problem.

There's also the [Netbook Remix]("https://launchpad.net/netbook-remix") that will make the UI a little more friendly to small screens that you can install after installing the full version of Ubuntu if you find things to be a little cramped.
