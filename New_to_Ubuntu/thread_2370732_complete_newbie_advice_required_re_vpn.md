### Post by veddox on 2017-09-06
If you're not afraid of the commandline, [Ubuntu Server 16.04]("https://www.ubuntu.com/server") is the one to go for. It has a long support life and comes preinstalled with many of the components a typical server is going to need. At the same time, it chucks out all unnecessary clutter (including graphics), so it's slim and doesn't need a lot of resources.

However, like I said, it's not for the fainthearted - if you have no previous Linux experience, you're going to have a *lot* of learning to do. Fortunately there are comprehensive manuals available, both online and in book form. It's just going to take a lot of patience. (But it's plenty of fun :D )

Also note that old desktop computers tend to use a lot of electricity, you're going to have to factor in those costs too. Perhaps renting a cheap virtual private server would be a good alternative?
