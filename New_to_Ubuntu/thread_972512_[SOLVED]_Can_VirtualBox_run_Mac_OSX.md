### Post by Cypher on 2008-11-05
VirtualBox will not run Mac OSX. You can, however, run Mac OSX inside VMWare. 

If you have a multi-core CPU, VMWare is only stable running Mac OSX when configured to use one core, so your CPU has to be very fast to run OSX at any reasonable speed to get anything out of it.
