### Post by keplerspeed on 2008-11-08
Firstly you need the driver for your wireless card. Assuming that it is avaliable, go to system>administration>hardware drivers and you should be prompted with the appropriate wireless driver. Install the driver, and report back with the result!

IF the driver is not available, you can install the ndiswrapper, of which can be used to run the windows driver for the wireless card. see here:

[https://help.ubuntu.com/community/WifiDocs/Driver/Ndiswrapper](https://help.ubuntu.com/community/WifiDocs/Driver/Ndiswrapper)
