### Post by jrothwell97 on 2009-03-31
Which particular password? Are you locked out of your account, or can you not unlock the keyring?

If it's the second, it's easy enough to get around. Open up a terminal, and run these commands:

```
cd .gnome2/keyrings
mv default.keyring default.old
```

Close the terminal (or type 'exit') and then log out and back in again.
