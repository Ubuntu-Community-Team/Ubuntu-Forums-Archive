### Post by Didius Falco on 2009-04-10
I'm on Jaunty, but I found the same behavior here. A little experimenting showed me what I hope is the fix for you.

On the Main Menu screen, click once on System Tools and look to the left side, at the Items section. Make sure at least one item there is ticked for display.

Then you should be able tick the System Tools folder for display - actually, on my system, it automatically ticked for display when I selected the program in the Item list.

I hope this helps.
