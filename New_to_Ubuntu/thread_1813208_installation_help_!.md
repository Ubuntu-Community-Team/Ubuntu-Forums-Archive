### Post by turtle153 on 2011-07-27
Are you using Ubuntu 11.04? If so then Firefox is already installed and you can open it by clicking in the top left hand corner of your screen and choosing "Browse the Web".

To install Thunderbird, click the + icon on the left-hand panel and search for Thunderbird. Which you click its icon the Software Centre will open and you can install it there. Run Thunderbird by clicking in the top-left and search for it again.

Bear in mind that Ubuntu comes with Evolution, another mail manager that you may like to use.

To use your Yahoo Messenger, look for your name in the top-right and click the x speech bubble next to it. Choose Chat Accounts to set up your Yahoo account on it.


You might not have realised it but when you install applications on Ubuntu they don't ask you where to install like Windows does.
