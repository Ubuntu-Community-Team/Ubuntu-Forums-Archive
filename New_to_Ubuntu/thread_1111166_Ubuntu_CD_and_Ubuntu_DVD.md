### Post by ajgreeny on 2009-03-30
To be honest, if you are OK with a text installer you could download the minimal install CD from here
[https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)
and then allow the system to install full ubuntu system from the net direct.  This means you will get the most recent packages for whichever version of ubuntu you choose, and not have to do the hundreds of MB of updates after installing the system from a normal CD.  Worth considering, I think.

You can even use the minimal CD to install a command line system and then add whatever you want.  I have just tried it with the LXDE desktop and just a few packages, to see how it works and it is very very fast with LXDE, though it may not be to everyone's taste.
