Doesn't look like the repositories have 1.6.2 yet.... they're still on 1.6.1.  Even the official download page ([http://www.emesene.org/download.html](http://www.emesene.org/download.html)) doesn't have 1.6.2, so you may need to wait.

Once updated, you can upgrade from System > Administration > Synaptic Package Manager or via terminal:

```
sudo apt-get upgrade emesene
```
