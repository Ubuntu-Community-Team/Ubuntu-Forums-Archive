### Post by Cheesemill on 2013-04-05
If you aren't using a proxy then just leaving it blank and hitting enter will continue on with the installation.

If this isn't happening the first thing I would do is check the integrity of your installation media as a corrupt CD will cause these sort of issues.

First of all have you checked the [MD5SUM]("https://help.ubuntu.com/community/HowToMD5SUM") of your download to make sure it's OK? If your MD5SUM checks out then the next step is to use the 'Check this disc for errors' option when you boot from the CD. If both of these tests pass then we can start looking at other possibilities, but most of the time installation errors such as yours are caused by a bad CD.
