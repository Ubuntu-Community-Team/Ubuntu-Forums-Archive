### Post by El Zoido on 2010-05-06
There are a number of possible solutions.

You can use Gimp, and while it may take some time to get used to it, you have complete control over the graphic.

A pretty simple solution would be to use OpenOffice Draw.
You can import graphics and while you could not directly edit them you can easily overlay shapes to higlight parts, add text, etc.
However, I'm not sure if you can simply import Pdf documents, so you should maybe scan them as jpg or png.
There should be a plugin available for OpenOffice though that makes it possible to import pdf with OpenOffice.
