### Post by jbrown96 on 2008-12-04
Advantages:
You don't have to mess with paritioning, and a bootloader.
Good way to see if you like Ubuntu before drastically changing your system.
Allows you to run a mixed environment where you can use Windows and Linux apps at the same time.

Disadvantages:
Since vmware creates "virtual hardware", you can't tell if a real install will work properly.
Poor performance unless you have a lot of RAM. I've found that the processor doesn't matter too much as long as you have at least 2GB of ram. I use Kubuntu and was trying to run XP as a VM (to use iTunes only), and it was pretty slow (although it may just be itunes) on my system with 2GB of ram.
Difficult to transition. If you are trying to completely switch to Linux, it will make it more difficult to make the jump. When I switched, I removed Windows and never put it back on. You will be forced to learn Linux, and while it may be frustrating at times, it really pushes you.
