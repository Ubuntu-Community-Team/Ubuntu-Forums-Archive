### Post by howefield on 2016-02-12
Or just press the Super + A keys to search apps or Super + M to search music, Super + F to search files ect, ect.

Pressing the Super key for a few seconds should bring up an overlay of unity shortcuts.. the one I like is Super + Tab which cycles down through the icons on the launcher.

Not sure how up to date it is, but the built in help docs should give you most of the info.
