### Post by Mark Phelps on 2011-10-18
First, and most important, despite your enthusiasm, you're probably NOT going to be able to "shift" to Ubuntu.  That implies leaving MS Windows entirely behind and totally moving to Ubuntu.

Why?

Because your .NET development work can not be done in Ubuntu.  Folks will tell you about Wine -- but that won't work for what you want to do.

Even if you DO install Ubuntu, you will have to retain MS Windows in a dual-boot setup (NOT Wubi) in order to continue your .NET development work.

So, if you're willing to "add" Ubuntu to your set of experiences, install in true dual-boot fashion, then let us know; but, you're not going to be able to leave MS Windows entirely behind.

### Post by vandamme on 2011-10-18
You sound like an intelligent lad, so check this out:
[http://apcmag.com/the_definitive_dualbooting_guide_linux_vista_and_xp_stepbystep.htm](http://apcmag.com/the_definitive_dualbooting_guide_linux_vista_and_xp_stepbystep.htm)
(Not something I'd suggest to the sub-power user.) 
If you need a reason to try Linux look at [http://www.whylinuxisbetter.net/](http://www.whylinuxisbetter.net/). 
You might also try LinuxMint in a couple flavors, sometimes it's easier than Ubuntu, but sometimes not. 
Download a few onto CDs and give them a spin to see which works, and which you like. 
Clean up your Windoze machine as much as you can: delete the crap & recycle bin, defrag, then reboot off the Linux disk. The hard part is figuring out how much and where to put the linux partition, and how to boot from it (which partition gets the "slash / " that makes it boot from there). 
You might want to have a grub rescue CD handy too ([http://www.supergrubdisk.org/super-grub2-disk/](http://www.supergrubdisk.org/super-grub2-disk/)) saved my butt a couple times. 
Remember how long it took to learn Windows? It won't take that long for Linux, but it takes getting used to.

### Post by Joreus on 2011-10-18
From my small understanding of Ubuntu and desktop environments... think of Gnome(Unity)/Xfce/KDE as different user interfaces. They present Ubuntu to you in different ways. The underlying OS is still the same and unless you're a power user the changes between each of the environments is mostly cosmetic (from my experience). Each of them also brings their own group of programs designed to work best in that particular environment. But, you can almost always use one environments program in another environment (albeit sometimes with a small performance loss).

You can install any of the desktop environments from the Ubuntu Software Center and try them out. If you don't like it, you can uninstall it just as easily. Ubuntu is fairly forgiving to newbies (like myself :D ). Try it all and above all have fun.

Of course, if you run VirtualBox Windows 7, you will need a fairly up to date computer because the two OS must share your computer's resources.  Also your processor must handle virtualization capability if you have 64-bit.
