### Post by HavocXphere on 2008-12-27
Right click on the K Menu button and select Menu editor. You can add/edit/delete a lot of things from there. You will need to know the command to launch the app to add it though.

To add a favourite, find it somewhere in the Kmenu right click-> add favourite. (Apparently on can't do that from the menu editor)
