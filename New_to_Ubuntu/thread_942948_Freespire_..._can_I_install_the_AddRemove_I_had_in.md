### Post by Spaceman9 on 2008-10-09
Take a look at this  Xandros buys Linspire – What does it mean for Linux? [http://www.itpro.co.uk/604461/xandros-buys-linspire-what-does-it-mean-for-linux](http://www.itpro.co.uk/604461/xandros-buys-linspire-what-does-it-mean-for-linux)

Xandros will post the new version of Linspire and Freespire in the third quarter of 2008. The new versions will be based on Debian.

I used Freespire earlier this year. It's not a bad distro, but Ubuntu is so much better. Linux Mint is also a much better distro. 

Debian will post it's new version 5 some time before the end of this year. Even with the KDE desktop there's not much ready out-of-the-box. You'd have to do a lot with the CLI and install a lot to make it like Ubuntu and Mint. 

Mandriva just posted Mandriva 2009. It's very good and easy to install. Most people would just have to install the software they want. Not much needs to be done with the CLI. It's a very good distro for anyone just starting with Linux. Personally, I like Ubuntu better.

Dreamlinux and Sidux are also based on Debian. Both are good, but better suited for experienced Linux users. Both have a lot more development to do to come up the the level of ease-of-use that Ubuntu and Mint have.

I'd recommend Ubuntu or Mint for a first time user. Mint has more ready out-of-the-box, but the Ubuntu Forum is much better when you need an answer quickly.
