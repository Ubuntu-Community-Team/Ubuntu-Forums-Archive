### Post by JBA2337 on 2008-08-18
I got the Debian menu installed on my computer, and in my case it lists about 126 applications that are on my computer. Many of the applications that are listed in Debian Menu are the same ones that I can run by clicking on the other sub menus in Applications. 

In many cases I notice that the Debian menu may list exactly the same application as in other sub menus, but the name of the app. in Debian Menu is completely different than the name that you find in the other sub menus. 

For example, the Disk Usage Analyzer is called Baobob by Debian Menu; Archive Manager is called File-Roller by Debian Menu;  CD Player is called Goobox by Debian Menu; and etc...

This is very confusing. Why doesn't the Debian Menu use the same names as the applications that I am used to. Is there any easy fix for this?

JBA2337

### Post by Anzan on 2008-08-18
Well, those are the real names of the applications. 

Ubuntu simplifies things by calling Gedit, Text Editor for example. But there are many different text editors available and each have different strengths. If they were all called Text Editor that would be confusing.

So there's nothing wrong, all is well.
