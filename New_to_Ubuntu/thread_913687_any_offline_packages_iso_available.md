### Post by Sorivenul on 2008-09-10
I can't think of any easy way around this, and I'm tired, but this is the best solution I can come up with ***if*** you still have a Windows machine around that can connect to the internet, or a friend nice enough to loan you theirs for a day. 

1. Use Wubi to install Ubuntu within Windows.  
2. Download [APTonCD]("http://aptoncd.sourceforge.net/") in the Ubuntu install via command-line, Synaptic, or Add/Remove.
3. Use APTonCD to make your package ISOs.
4. Burn your package ISOs.
5. Reinstall Ubuntu as you currently have.
6. Use your ISO discs to install packages.

Sorry this isn't more detailed.  I'll try to update more later.  Good luck in the meantime!
