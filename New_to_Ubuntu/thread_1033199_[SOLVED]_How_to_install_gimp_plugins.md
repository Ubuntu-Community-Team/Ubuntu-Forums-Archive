### Post by SteveDee on 2009-01-07
Gimp scripts should be added to your script directory (e.g. /home/steve/.gimp-2.6/scripts). Then restart Gimp and they should appear in a menu (which menu depends upon the function of the script).

Heres a useful link;
[http://gimp.org/docs/scheme_plugin/index.html](http://gimp.org/docs/scheme_plugin/index.html)
