### Post by eMJayy on 2009-06-05
> **CoCoKnots said:**
> Sorry, I'm a newbee. How do I get into the repositories?

Hi. Repositories are online servers that store the Linux software packages that Linux uses to install software and update itself. Here's a link from Wikipedia giving an outline of what they are
 
[http://en.wikipedia.org/wiki/Software_repository](http://en.wikipedia.org/wiki/Software_repository)

To access the repositories, Ubuntu has what is known as a Package manager called Synaptic [http://en.wikipedia.org/wiki/Synaptic_Package_Manager](http://en.wikipedia.org/wiki/Synaptic_Package_Manager) 


You can find it by clicking  [B]System >Administration > Synaptic Package Manager.
[/B]
It's already configured to access files for use in Ubuntu, so all you have to do is type in a search word and the program will find the file in the on-line Repository for you. The best part is that, once you find the package you searched for, you can install and uninstall the package right there in Synaptic by simply right-clicking the file you want to install and selecting *mark for installation*. To install the program, you then just need to click Apply.
