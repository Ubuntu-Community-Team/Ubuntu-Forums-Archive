### Post by cavalier911 on 2010-08-01
**Obkey Openbox Key Editor for Lubuntu**

 [http://code.google.com/p/obkey/](http://code.google.com/p/obkey/) 

```
tar -xvf obkey-dev-e63aa00.tar.gz
``````
cd obkey-dev-e63aa00/ 
``````
sudo python setup.py install
``````
obkey ~/.config/openbox/lubuntu-rc.xml 
```
