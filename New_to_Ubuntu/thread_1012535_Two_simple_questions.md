### Post by Shhnap on 2008-12-15
Its not so bad as you think.

1)
i) Applications -> Wine -> Configure Wine
ii) Drives tab
iii) Autodetect... button

This will give you a bunch of drives that both wine and ubuntu recognise. Then if you want to browse the drives that wine recognises go Applications -> Wine -> Browse C Drive

Then from here just click the .exe file in the wow cd and the install process should start. That's how I installed starcraft but there might be more to wow that you need to setup. Like wine and networking ect. ect.
