### Post by herbertportillo on 2009-12-21
You can download Firefox 3.5 by opening up a Terminal and typing
```
sudo apt-get install firefox-3.5
```

or you can go to [http://www.google.com/chrome]("http://www.google.com/chrome") and download Google Chrome. I personally use Chrome. It's crazy fast, even faster than Firefox 3.5, if that was even possible.

You can also download Epiphany, a browser that integrates well with your Ubuntu system by typing ```
sudo apt-get install epiphany-browser
```.

Cheers
