### Post by grahammechanical on 2011-10-12
There are four options for the Launcher (in 11.10 at least) Never, Autohide, Dodge Windows and Dodge Active Windows. On your system the Launcher is set to autohide.

You need to open CompizConfigSettingsManager and go to the Ubuntu Unity Plugin and under the Behaviour tab you will find a menu labelled Hide Launcher with the options I have mentioned.

You may need to install the CompizConfigSettingsManager if you have not already done so.

Regards.
