### Post by umoreno_89 on 2010-02-08
do u mean the pictures as icons, when you see them in the file browser.

If so, from the file browser:
[LEFT]Edit->Preferences->Icon view defaults
from here, you can change the zoom level, aka size, of the pictures.

Hope this helps.
[/LEFT]
