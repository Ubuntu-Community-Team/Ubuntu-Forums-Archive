### Post by nhasian on 2008-06-30
Hello Smiling Jak,

I've played around with running Fedora and Yellowdog Linux on my PS3.  Its painfully slow since Linux only has access to 256Megs of ram (sony locked us out of the graphics card's 3d and additional 256M from there).  If you were going to install ubuntu on the PS3 I'd recommend xubuntu.

you can do simple things like firefox, openoffice, mame/snes emulators, mp3 or standard definition video playback.  Forget about High Definition playback.  I tried to play Elephant's Dream (world’s first open movie, made entirely with open source graphics software) and it was so choppy it was unwatchable.

unless Sony opens up their video card in the PS3 to the linux community, i'd recomment just buying a used 2 year old 3 Ghz desktop off of craig's list for about $200.

### Post by mcduck on 2008-06-30
> **Smiling Jak said:**

I had just heard that, well, you can indeed play some Windows games on Linux through various means. So, I figured if the PS3 is running Linux, you might be able to play PC games as a result of that.

Sorry,but no. You won't be able to play any Windows games on PS3. The PS3 has Power-PC-based processor, while Windows (and all it's programs) are only available for x86 processors.

Wine won't help her, it has the same problem.

Only way to run Windows programs on PS3 would be some kind of hardware emulator (which would be painfully slow, even for simple programs, not to mention games).
