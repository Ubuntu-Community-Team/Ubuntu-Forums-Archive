### Post by oldfred on 2012-05-03
Not many computers directly boot Compactflash cards. Many newer one's boot USB flash devices. It may work if plugged into an USB adapter, but I have not tried that.

Full install of Ubuntu needs 4.5GB now, but Lubuntu may fit. But USB port is slower than hard drive and flash is slower (particularly writes) than hard drive. 

I have an install on USB flash in an 8GB partition and it runs ok, just not speedy. 

Installs to 4GB flash, newer versions require 4.4GB as minimum
