At one time, Pascal was a very popular language (if for no other reason than it was used to teach beginning programming), and a lot of older shareware programs were written in it (thanks to the easy availability of Borland's Turbo Pascal compiler).  It's still in use outside of educational circles (Embarcadero Delphi is the descendant of Borland's Turbo Pascal), but has mainly been displaced by C++ for most applications.

It *is* an excellent language for learning basics (far better than BASIC), but I wouldn't count on ever actually using it once you complete that course.  You *might* run into it later, but it's not all that likely.

The core of Linux (i.e. the kernel) is written primarily in C and assembly, as is most every modern operating system.  However, technically "Linux" is only the underlying part of the operating system that interacts with the hardware.  The part of that most people see and interact with is the "GNU" part.  The graphical user interface technically isn't called "Linux".  The applications that people use everyday are made with nearly any language you can think of, such as C, C++, Python, etc.
Of course, one of the beautiful things about Linux is that the source code is available and free. You can download the source code and browse through it to see what language is used and how they did it (even modify it if you want).

I learned Pascal in college too (you don't happen to go to college in Florida do you?).  "Delphi" uses Pascal to create GUI applications for MS Windows, so that might be worth using or experimenting with while in school.  There is also something called Free Pascal which is the open source version of Pascal extremely similar to Borland's Pascal, but text-based.  They have a something called "Lazarus" for GUI's, which is similar to Delphi.  You should probably download that and become familiar with it. Lazarus is cross-platform, so you can program Linux or Windows.

I can tell you from experience that you better learn another language  while you are in school.  'C' is still a good language to learn since it  is the basis of nearly every language out there.  I also recommend learning about web programming, including HTML and Javascript as well as a bit about web servers, since so many things are web based now.
If you really want to program Windows, learn Visual Basic.
