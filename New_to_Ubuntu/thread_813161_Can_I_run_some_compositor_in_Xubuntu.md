### Post by Inxsible on 2008-05-30
> **ubernuber said:**
> Which one is better and how do I get it?

Compiz has the most effects..but is resource intensive. If running AWN is your primary concern, then you should simply enable the xfwm4 by going to Applications>>Settings>>Settings Manager>>Window Manager Tweaks and enabling it on the Compositor tab.
