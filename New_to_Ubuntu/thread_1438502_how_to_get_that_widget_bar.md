### Post by pj_kare on 2010-03-25
I looked at SneekyLinux's  youtube video and it looks like it may be **cairo-dock**, this is a bar you can have across your screen with various icons for various programs, with animated effects as you mouse over them. You can add/remove and also change icons for different programs. I think (If I remember rightly as I have it, just not installed) it also uses widget type things for weather etc.

 Open Synaptic Package Manager and do a search for cairo-dock. You need to install it and all it's associated files.

 It may also be Gnome do's "Docky" which looks basically the same as cairo-dock. [http://do.davebsd.com/wiki/Docky](http://do.davebsd.com/wiki/Docky) 
 If it is Docky, then search for gnome-do in Synaptic Package Manager
