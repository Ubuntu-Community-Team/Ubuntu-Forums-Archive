I'm having this same issue.  I've done some custom work on my mediatomb database as well...and every time it 'loses' the database I have to re-do the work.  It's tedious.  I was avoiding rebooting my computer...now I just back up the .db file in /home/*user/[I].mediatomb*[/I] and if mediatomb loses it's mind for whatever reason, I just rename the backup to mediatomb.db.  It's annoying, but at least I don't lose my work.

Funny thing is, I didn't have this problem with Gutsy.
