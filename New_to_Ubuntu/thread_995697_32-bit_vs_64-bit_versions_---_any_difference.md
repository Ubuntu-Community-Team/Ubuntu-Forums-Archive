### Post by 3rdalbum on 2008-11-28
Call me a philistine, but I do not notice any difference in performance between 32-bit and 64-bit. The 32-bit edition can only use 3.3GiB of RAM on my machine, and somewhere around 3 GiB if I had a better graphics card; whereas the 64-bit edition can use more RAM.

I don't believe running 64-bit code on a 64-bit processor actually makes things faster. If you have 4 GiB of RAM or more, the extra cache that the processor can address may increase performance.

Unless you've got 4GiB of RAM, there's no need for you to change to 64-bit. It requires a fresh installation.
