### Post by NightwishFan on 2009-04-30
Hi, and welcome to Ubuntu forums.

Installing with Wubi is fine. I am not sure if it can upgrade, as I have never tried it, however I think it would work. Please wait for someone who has actually tried it to confirm, though.

If you want to reinstall, simply copy all the files "you want to keep!" from the Wubi Ubuntu to a folder in Windows, then use add/remove in Windows to uninstall WUBI. After you do that, simply reinstall Wubi using the new 9.04 Jaunty Jackalope Install CD. Then copy all of your data back into WUBI. For all file transfers to and from WUBI I think you need to be using Ubuntu to do so.
