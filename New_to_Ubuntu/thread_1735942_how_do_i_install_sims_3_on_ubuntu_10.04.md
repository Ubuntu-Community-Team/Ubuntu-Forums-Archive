### Post by rosencrantz on 2011-04-22
General Wine instructions.
First, check whether it works at all on Wine:
[http://appdb.winehq.org/objectManager.php?sClass=version&iId=16664](http://appdb.winehq.org/objectManager.php?sClass=version&iId=16664)
Second, open a terminal (Accessories->Terminal), navigate to where the Windows installer is (terminal howto: [http://ubuntuforums.org/showthread.php?t=73885](http://ubuntuforums.org/showthread.php?t=73885))
and type wine InstallerFileName.exe (replace with appropriate file).
If everything goes well, you'll find a link somewhere in the start menu afterwards that launches your game.
