### Post by yancek on 2021-01-30
Not allowing Desktop icons is a deliberate choice of the developers, see the link below and many similar if you do an online search.

[https://www.omgubuntu.co.uk/2018/01/gnome-desktop-icons-removed-3-28](https://www.omgubuntu.co.uk/2018/01/gnome-desktop-icons-removed-3-28)

The link below gives 2 options to change this and allow Desktop icons, ubuntu-tweaks and  gnome-shell-extension-desktop-icons.  I just used the latter which can be installed with the command:

>   sudo apt install gnome-shell-extension-desktop-icons



[https://www.linuxquestions.org/questions/ubuntu-63/ubuntu-20-10-~-desktop-files-don't-appear-on-desktop-4175688885/](https://www.linuxquestions.org/questions/ubuntu-63/ubuntu-20-10-~-desktop-files-don't-appear-on-desktop-4175688885/)
