### Post by grahammechanical on 2015-06-24
There is something that you need to be aware of.

If we install Wine or Ubuntu Restricted Extras we also get Microsoft truetype core fonts and we need to accept the Microsoft End User License Agreement (EULA). If we do not click to accept that agreement or click to reject it the install process will pause and wait for our input and if that never comes then the OS has a problem that it cannot solve.

Why would we not click to accept or not click to reject? Because we cannot see the dialog that is asking us to confirm our acceptance of the EULA. It is hidden behind the Software Centre Application Window. So, when installing Wine or Play on Linux or Ubuntu Restricted Extras move the Software Centre to one side and wait until that EULA confirmation dialog appears.

I know what to expect and yet the other month when I installed Wine. I got caught out for several minutes. Your problems might be caused by these software installations not being completed.
