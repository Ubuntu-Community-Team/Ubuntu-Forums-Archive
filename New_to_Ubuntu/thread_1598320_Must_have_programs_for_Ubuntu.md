### Post by chaanakya_chiraag on 2010-10-16
Depends on your needs:
VLC for hassle-free media playback
Audacity for sound recording
OpenShot Video Editor for awesome video editing
Inkscape for vector graphics editing
Rosegarden for MIDI stuff as well as score writing
Musik Open Source Music Creator for text-based music creation (check my signature for the link)
TexLive for LaTeX Document preparation system
Cairo Dock for a dock better than Mac OS X's
Ubuntu Tweak for easy adding of repositories and easy editing of system settings that are not usually available
Eclipse for any programming needs
youtube-dl to download YouTube movies (command-line program)
xpenguins for a lotta fun
Kupfer for quick searching/launching applications/etc. (kinda like Quicksilver for Mac, but way better!)
Dropbox for easy sharing and backup of files (sign up using the link in my signature)
