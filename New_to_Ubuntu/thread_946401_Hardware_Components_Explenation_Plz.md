### Post by Bölvaður on 2008-10-13
I wish it would be that simple you could see how good something is by the price.. but it is very foolish and you will probably end up with too expensive computer compared to cost.
What you need to do is write down eveything you are going to be doing with the computer and figure out how you achieve those goals and estimate the useage of ram, cpu and graphicalcard.

With that in mind you have to remember that because of drivers the performance of some hardware differs from windows and linux. And there are even hardware that doesn't support linux and no linux developer wants to support, so there might be nonworking hardware, so check that out also.

Comiz-fusion does not require expensive graphical card at all. GF7xxx might do well.

If you have old box with cd player and such I would recommend reusing it because that is a BIG part of the price which you would definitely avoid.
