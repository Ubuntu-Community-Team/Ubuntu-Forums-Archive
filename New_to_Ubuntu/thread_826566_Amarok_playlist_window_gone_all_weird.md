### Post by Xiong Chiamiov on 2008-06-13
Hmm.  I'm assuming you can't drag the collection bar to the left and get it to show?
Try right-click the name column and choosing "hide name".  If that doesn't work, I'd recommend purging amarok and reinstalling, since a normal reinstall doesn't get rid of settings.
```

sudo aptitude purge amarok; sudo aptitude install amarok

```
