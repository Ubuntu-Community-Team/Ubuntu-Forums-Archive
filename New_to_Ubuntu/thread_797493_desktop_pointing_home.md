### Post by TeoBigusGeekus on 2008-05-17
Applications->System Tools->Configuration Editor
Apps->Nautilus->Preferences on the left side of the dialog box.
Uncheck Desktop is Home Directory on the right side of the dialog box.

If you can't see Configuration Editor under system tools, right click Applications menu->Edit menus, go to system tools and select configuration editor.
