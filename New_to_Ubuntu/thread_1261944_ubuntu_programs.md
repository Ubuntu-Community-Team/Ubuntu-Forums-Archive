### Post by NightwishFan on 2009-09-09
This book is excellent, and is a free download in PDF. You can also buy it for cheap if that is what you want.

[http://www.ubuntupocketguide.com/download_main.html](http://www.ubuntupocketguide.com/download_main.html)

Good Programs:

Celestia - Space Simulation
Stellarium - Virtual Planetarium
GIMP - Image Editor (In Ubuntu by default)
Frozen Bubble - Fun 3d Game
Wine - Run windows programs/games in Ubuntu

To install programs, go to Add/Remove at the bottom of the applications menu, and change the drop down menu to Show All Available Software.

### Post by NoaHall on 2009-09-09
First - don't install via Wubi. If your Windows fails(which is likely) then you'll not be able to access your Ubuntu.

Download and install flash, maybe Opera too, get your graphics drivers installed (System -> Admin -> Hardware Drivers). Then get compiz fusion, set that up and maybe then get AWN ( Sudo apt-get install avant-window-navigator)
