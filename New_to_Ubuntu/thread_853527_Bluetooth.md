### Post by LowSky on 2008-07-08
First off there is no way to send a folder via bluetooth or email. You need to send it as a compressed folder. Otherwise you will need to send each item seperatly. 
Try opening the folder selecting each file (Ctrl+A) and the right clicking and using Send to that way. Each file will go over in a batch.
