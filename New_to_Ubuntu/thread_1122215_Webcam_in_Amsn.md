### Post by jmmL on 2009-04-13
Do you know the make of the webcam?

Have you tried in any other programs?

I find cheese to be fairly good at deciding whether a webcam is supported in linux. You can install it by typing 
```
 sudo apt-get install cheese 
``` in a terminal.

You could also try ```
lsusb
``` to list USB devices connected to your system to find out the manufacturer of your camera.
