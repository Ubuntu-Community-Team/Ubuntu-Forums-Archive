### Post by Yuki_Nagato on 2008-08-17
You cannot, "use it" per se.

You can boot to it however, and try it.  Start up, pop open the CD tray, insert, close, and then restart.  When the BIOS pops up (the first thing you see) find the command to enter the boot-sequence.  Select the CD-ROM drive then, and you should be good to go.

Kubuntu is also just Ubuntu with the KDE windows environment instead of the GNOME windows environment.  And Xubutnu is the XFCE.  Gobuntu is completely different.
