### Post by donkyhotay on 2009-02-11
Not counting games (c;

Firefox
thunderbird
yakuake
gedit
make

Firefox and thunderbird are pretty obvious. Yakuake is a quake-style CLI program that is really useful. Just hit f12, type a few commands, then do something in the GUI while it processes my command in the background. I use gedit way more then open office by me because I generally want unformatted text when writing a quick note or need it when editing config files on my system or reading/tweaking source code. Make of course is for compiling programs from source which I do with certain programs (games) that I want the latest version. Either that or occasionally I'll find a bug I think I may be able to fix and to recompile to see how my changes work.
