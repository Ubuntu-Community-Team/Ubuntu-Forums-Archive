### Post by dolphin194 on 2012-10-22
If you chose to install Ubuntu alongside Windows, you can simply use the Boot Repair to recover it.

Here's how to do it if you still have Windows on your hard drive:
1. Start up Ubuntu
2. Install Boot Repair from the Ubuntu Software Center
3. Run Boot Repair
4. Click "Recommended Repair" and everything should be good.


*HOWEVER*, if you removed Windows XP, you will need to re-install it from a CD.


Of course, you can stick with Ubuntu and install the Lubuntu or Xubuntu desktop environments, however they may not work for you.
