> **reaper0686 said:**
> and once it restarts this error comes up "ERROR: No boot disk has been detected or the disk has failed."

Seems like grub-bootloader didn't installed in right place. Have you installed Ubuntu from a USB-stick ? 

Try this tool , all you need is internet connection - live image on USB or/ DVD and follow the instructions 
[https://help.ubuntu.com/community/Boot-Repair#A2nd_option_:_install_Boot-Repair_in_Ubuntu](https://help.ubuntu.com/community/Boot-Repair#A2nd_option_:_install_Boot-Repair_in_Ubuntu)
