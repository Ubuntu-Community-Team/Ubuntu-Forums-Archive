### Post by Paqman on 2008-05-01
AFAIK it should be basically the same as going from the s-video to the TV itself. If you haven't got it already you want to install nvidia-settings.

[Click here to install nvidia-settings](apt:nvidia-settings)

The go to System > Admin > Nvidia Settings
