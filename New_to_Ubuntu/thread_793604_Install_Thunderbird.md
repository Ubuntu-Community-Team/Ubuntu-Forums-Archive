### Post by Blue Snapper on 2008-05-15
Being new to Ubuntu 8.04 I did it the easy way:

1. In Applications select "Add/Remove Applications";
2. In the Show option make sure "All available applications" is selected and in the search field enter "mozilla" (without my quotes);
3. Select "Mozilla Tunderbird Mail/News" when it show up in the list and then "Apply Changes" & just "follow the screen". 

It all went very fast (with broadband connection) and was really as easy as 1, 2, 3.

Good luck.
