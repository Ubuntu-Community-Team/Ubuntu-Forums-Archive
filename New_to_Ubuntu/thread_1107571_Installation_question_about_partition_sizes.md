### Post by 123456789123456789123456 on 2009-03-26
the main ubuntu installation does not require more than 4gb hard disk space.  30 gb was plenty big to handle a ton of updates, and storage room.  The reason you only see 28gb, in theory is that ubuntu sets up a swap file/partiton by default, it is possible that it is a 2 gb partition.  The swap partition would not be viewable under windows.
