### Post by Lars Noodén on 2012-11-20
Well, the lightest official Ubuntu distro is Lubuntu.  That uses fewer resources because it is built around the LXDE desktop.  
[http://lubuntu.net/](http://lubuntu.net/)

If there are applications missing, you can add the ones you want from the repository.

If you want lighter, you can skip the desktop environment completely and just run a plain window manager.  Choices include FVWM, FVWM-crystal, Openbox, Fluxbox and Enlightenment.
