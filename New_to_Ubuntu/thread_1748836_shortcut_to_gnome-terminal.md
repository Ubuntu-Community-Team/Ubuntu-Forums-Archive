### Post by jamesjenner on 2011-05-04
Bonjour JRev,

Could you explain what you mean and also what version your running?

I have a vanilla version of Ubuntu 11.04 running and Control + Shift + T works perfectly. Note that I'm using a US keyboard set to the US layout. I'm not certain what impact the French layout could have to the use of control and alt (though I expect none).

If you have 11.04 then you should have a short cut on the side panel for the terminal already. If you have 10.xx then you should be able to right click on the existing terminal under utilities (in the menu) and add to the desktop. 

Otherwise you can add a launcher and tell it to run gnome-terminal as the command.

Hope that helps.
