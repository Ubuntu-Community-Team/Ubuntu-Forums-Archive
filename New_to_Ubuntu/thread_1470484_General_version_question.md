### Post by lavinog on 2010-05-03
> **bgast1 said:**
> Is it possible that computers like older revisions of ubuntu verses the newer? I just upgraded and it seems my machine liked older revisions.

Generally newer versions should have better support for hardware.  Occasionally support for really old hardware is removed, but this should not affect many people.  What might happen is that new hardware is grouped with old hardware, even thought they can be drastically different.  Sometimes implementing a new feature for new hardware can cause a problem with old hardware.  It is important to report issues in bug reports so that the problems can be resolved before the next major release.

Depending on the problem, you may be able to continue using the latest release.  The problem might be addressable with a configuration file or a PPA.
