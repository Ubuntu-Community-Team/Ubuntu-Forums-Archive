### Post by KdotJ on 2010-05-18
I personally run the normal desktop version on my netbook, but have the choice to boot into the UNR if I fancy it. All that's different is the interface and the default apps. Underneath its the same... 
If I had to choose one, I'd pick desktop. I feel that the UNR interface feels a little restrictive, and having the big menu in the background sometimes a little annoying.

And to give you feedback, I have no trouble with overheating or fan speeds but then I had a different netbook to you so I can't speak for everything. 

If you really can't decide, you can have both like me. This can be done by installing the UNR from within ubuntu desktop version or vice versa.
