### Post by pelle.k on 2011-12-24
If you're running unity, and if you set a window to be 75% (correct me if i'm wrong here) or larger, unity automatically launches it "maximized" the next time you launch that application.
You can "turn it off" by installing ccsm, open the unity plugin, and set this to 100%, meaning, it'll only maximize windows that already cover the full desktop width and height.
