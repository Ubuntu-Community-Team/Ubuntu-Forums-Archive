### Post by freeman2000 on 2009-05-27
First I would go to System --> Administration --> Hardware Drivers, and see if Ubuntu can round up the necessary drivers for you.  
Just "activate" whatever drivers it "recommends".  If that doesn't work, then:

for the nVidia driver, go here....

   [http://www.nvidia.com/object/unix.html](http://www.nvidia.com/object/unix.html)


for the SoundBlaster, go here....

   [http://support.creative.com](http://support.creative.com)


Good luck.
