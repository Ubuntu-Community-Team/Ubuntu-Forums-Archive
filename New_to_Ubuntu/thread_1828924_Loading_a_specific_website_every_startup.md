### Post by JC Cheloven on 2011-08-19
As an addition to Rex' advice, you can include firefox among autostarted applications. This way you'll see firefox open at that web page when you start, with no intervention on your side.

I also remember of a desktop widget able to display a web page as a background, but I can't remember what was tha name of that. Perhaps anyone else?

### Post by IWantFroyo on 2011-08-19
I don't remember how to add a command to startup programs in XFCE, but once you find it, just tell it to run this, replacing Firefox and the Ubuntu Forums with your web browser and website:
```
firefox www.ubuntuforums.org
```

Now, when you log in, it should open the webpage inside your browser.
