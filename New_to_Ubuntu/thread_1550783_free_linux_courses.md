### Post by QIII on 2010-08-11
Writing programs is not really a "Linux" thing.  Except for cases such as .NET for Microsoft platforms,  programming is largely OS independent.  (As an aside, .NET can be used and run on Linux by virtue of the MonoDevelop development platform.)

What might be a better question, and I will allow you to rephrase it if you wish, would be "Are there any free courses in learning to program in the X language?", where X is whatever language you are interested in learning.

With regard to learning to use Ubuntu, try giving this a read:
