### Post by slooksterpsv on 2013-02-05
Open a Terminal from Dash (click on the Dash icon and type Terminal). Then type: 

sudo apt-get update 

Type in your password when it prompts for it it won't show the password as you type it but it is accepting it.

Then type in:

sudo apt-get install bison

It should install bison
