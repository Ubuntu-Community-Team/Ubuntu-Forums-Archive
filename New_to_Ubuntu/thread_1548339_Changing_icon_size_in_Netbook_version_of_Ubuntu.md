### Post by beew on 2010-08-09
For the Desktop version  this is what you do to change the size of the icons..Open any folder (or file browser, as it is called), go to Edit. From the drop down menu choose preference, then a window will open, from the "view" tab you would be able to choose the icon size as well as configure other things.

Have you tried something like that on the netbook edition?(I wouldn't know if it would work as I have not seen one)
