### Post by KenUBF on 2017-04-09
Hi dusty1980,

If you're new to Ubuntu the easiest way to download games might be to go to the Software Center, search for the games you want, and hit install, and they will be installed for you. Go up to your menu bar and choose System --> Administration --> Ubuntu Software Center. In the top right corner is a search field where you can type what you're looking for. Once you find something you want to try, highlight it by clicking on it and you'll see an Install button. Click that and it will be installed for you. You will find the program listed in your Applications menu. I hope this helps!

### Post by amanchesterman on 2017-04-11
In Mate there's a 'software boutique' (so-called), described as a 'specially curated' selection of software designed to work well with the Mate desktop. In the 'games' section there's the app mentioned by gordintoronto which says it's a collection of over 1000 (!) solitaire games. You just click on the button for that to install it, Mate does the rest.
You'll find the software boutique if you go to the Mate main menu and select System > Administration > Software Boutique. Personally I find it very easy to use and I've installed several apps from it. I'm puzzled that you find it difficult? There's no need to go looking through the general Ubuntu software center when the Mate developers have provided their own, unless you really can't find what you want there.
