### Post by ibutho on 2008-05-26
AFAIK, once Ubuntu releases a specific version of its distro, no new features are added to keep the release as stable as possible. They do issue issue security updates, but don't update many packages to their latest versions. This means that they keep the same kernel version for the entire lifetime of the release and patch it to fix bugs etc.
