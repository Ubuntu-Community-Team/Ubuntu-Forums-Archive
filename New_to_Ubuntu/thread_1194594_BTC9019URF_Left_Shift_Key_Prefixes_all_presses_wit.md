### Post by kendoori on 2009-06-22
I just set up a machine with 9.04 and am trying to use a BTC 9019URF wireless keyboard. All seems fine except for the fact that whenever I press the left shift key, it produces a < character. This is not true of right shift key.

I've tried multiple keyboard layouts to no avail.

Is there a way to remap this key only to ensure that it is just shift?

---

### Post by kendoori on 2009-06-22
Apparently these keyboards ship with the British default for the left shift key, which cause a "\" character under Windows, and a "<" under Ubuntu. To fix this, one must physically remove they key and remove one of the "switches."

This was driving me crazy, but found the solution here:
