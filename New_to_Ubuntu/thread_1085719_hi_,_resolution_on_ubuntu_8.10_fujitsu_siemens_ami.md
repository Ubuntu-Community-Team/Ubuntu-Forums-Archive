### Post by benit on 2009-03-04
Hi, I just had that same problem with the Amilo V2055 and Ubuntu 8.10, check out the solution given by kansasnoob here: [http://ubuntuforums.org/showthread.php?t=1041680&highlight=Amilo+v2055+resolution]("http://ubuntuforums.org/showthread.php?t=1041680&highlight=Amilo+v2055+resolution")

Now the screen resolution is perfect this should fix your problem, good luck!! :)

Read the two pages, the solution is the one in post number 19, just enter the following codes:

Save a copy of your original xorg.conf in case something goes wrong you can restore it:
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
``` 

This is to restore it in case something goes wrong:
```
sudo cp /etc/X11/xorg.conf.backup /etc/X11/xorg.conf
```

to modify the xorg.conf use:
```
gksudo gedit /etc/X11/xorg.conf
```

Copy the code given in the post number 19 by kansasnoob and paste it in the xorg.conf edit window, (replacing the one that you have). Now just one final change as your screen is a 1280x800 you need to make a small change on the code that you just pasted, in the Section "Screen" the line virtual change from 1280x768 to 1280x800. Then save it, close, and reboot, it should be working then.
