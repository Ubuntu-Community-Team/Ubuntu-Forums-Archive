### Post by mikewhatever on 2011-02-27
Yes, you can set the minimal font size under Edit->Preferences, Content->Advanced.

---

### Post by Vaphell on 2011-02-27
and if you want dynamically resize text while preserving original resolution of images, in view->zoom (or whatever it's called, i don't have EN version) enable 'zoom text only'
