### Post by Sleven7 on 2011-01-24
The very first splash screen you see on booting is your BIOS splash.

There should be a boot option, F2, F10, F12, Esc  one of the f-keys or esc should let you chose where to boot from.

Alternatively you can go into the BIOS set up, from the same splash screen, and change the boot order.
