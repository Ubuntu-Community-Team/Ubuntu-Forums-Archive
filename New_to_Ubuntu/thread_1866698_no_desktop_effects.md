### Post by grahammechanical on 2011-10-21
I know what you mean. When I had Ubuntu 10.10 I had the effect of wobbly windows. But with 11.04 I could not have that effect because when I used Compiz Config settings Manager the change took away the top panel.

There is good news. I am using 11.10 right now I have enabled wobbly windows. I cannot say if other effects also work.

You need CompizConfigSettingsManager and its Ubuntu Unity Plugin to make adjustments to Unity 3D.

If you try to enable an effect and something goes wrong then press the super or Windows key (also called meta key) that will open the Dash and the Launcher. If you type compiz in the Dash search panel you will get the Compiz Config Settings Manager icon. click on that and the utility will load and you can undo what you have done and not lose the desktop.

By the way, I am using an Intel core 2 duo computer. I am running Ubuntu 64bit. I am sure that you can also. The 32bit version is recommended because it will run on 32bit and 64bit CPUs. The 64bit will only run on a 64bit CPU. If someone does not know what they are doing they might download the wrong version and wonder why it will not install. The 32bit cannot be the wrong version in that situation.
