### Post by VCoolio on 2012-10-29
Why not use US (international) layout so you can do 'e to get é and `e for è etc.? That's called dead keys and I'd say it's much easier to remember than Ctrl+Alt+D and what not.

### Post by twipley on 2012-10-29
And if you ever want to use multiple layouts like me, it can be quite simple to change from one to the other. In the layout settings, go to options > key to change layouts > alt+caps.

Changing to french, typing accent, then back to English layout can take as few as a single second once used to.

Just a tip. ;)
