Only one thing, and its trivial.

Install windows 7 first.
Then use the windows 7 partition manager (I think its at control panel -> administrative tools -> computer management -> disk management) to resize the partition and create free space. You can only create free space to the RIGHT of a partition btw.

dont use the ubuntu installer to resize the windows partition cause windows is gonna scream and tell you to stick the recovery discs in.
