### Post by CatKiller on 2009-04-11
> **luvmyMac said:**
> I'm a bit confused; I thought that Linux and OS X were the same or similar?

Not really. OS X is based on NeXTStep, which in turn used the Mach kernel and was based on one of the BSDs (can't remember which one) - a form of UNIX.

Linux is a Free kernel that isn't based on anything else, but a lot of the tools that work with Linux are UNIX-like in their implementation.

Some things, like X, can be run on either, but OS X has many applications that won't work on Linux, and GNU/Linux distributions come with many applications that won't work on OS X.

They are both UNIX-like, but quite dissimilar to each other. They are more alike than either are to Windows, though, which is probably where your confusion came from.

It is possible to install a GNU/Linux distribution, like Ubuntu, on your Mac, though.
