### Post by 2hot6ft2 on 2010-04-22
Applications > System Tools > Sysinfo
If you don't have it you can install it in 9.04 or newer by clicking this
[Sysinfo]("apt:sysinfo")
Or
Open a terminal
Applications > Accessories > Terminal
and run
```
sudo apt-get install sysinfo
```
