### Post by Neoncamouflage on 2011-06-27
Right now the only way I have to play Minecraft is go into Downloads, Minecraft, right click the .jar file, and choose run with Java 6. I tried creating a desktop launcher but because it just tries to run the file and not with Java, it fails. Is there any way to specify "with java" when telling it to run with the launcher?

---

### Post by ubudog on 2011-06-27
Yep, just change the command.  Put the word "java" in front of the command. 

Example command for the launcher:
```
java $HOME/Downloads/minecraft.jar
```
