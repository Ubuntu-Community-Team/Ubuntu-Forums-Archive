### Post by powel212 on 2009-10-07
> Our household has 4 ipod nanos 4th gen and they all work perfectly.

I don't believe this for one second.

I sync my ipod by running iTunes in a win7 virtualbox.

You already own vista so it is no problem to run it in virtualbox and if you just use it to run iTunes then you won't likely get any viruses in there.

Another nice trick is once you install and update Vista in the virtualbox you can backup the vista.vdi file and then if the thing does get a virus you can just delete the original .vdi file and use the one you backed up previously. Easy and time saving.

I hope that helps.

My experience is that iPods and other mac products will only sync with apple made software.
