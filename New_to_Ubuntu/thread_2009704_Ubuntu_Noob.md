### Post by Cheesemill on 2012-06-24
The online course will have been written for older versions of Ubuntu when it still had an Applications menu. The Add/Remove Applications application is now called Ubuntu Software Center and you can open it by hitting the Super (Windows) button on your keyboard and then start typing  'software' till the icon appears.

You probably want to read this as well:
[https://help.ubuntu.com/12.04/ubuntu-help/index.html](https://help.ubuntu.com/12.04/ubuntu-help/index.html)
