### Post by sgosnell on 2010-12-26
LastPass will work, as will KeePassX.  KeePassX doesn't sync, but you can use the same database across different OSs.  I sync it by putting the password database in my Dropbox folder, so that it's synced across all my Dropbox folders.  That way, I know that no unencrypted password files are stored anywhere other than on my machines.
