### Post by Vunutus on 2009-06-20
Pretty much the only difference between ubuntu desktop and ubuntu server is the packages they come with. Ubuntu Server comes without a GUI or any of the "fluff" utilities like media players, image tools, etc. It's meant to run network services.

That isn't to say that you couldn't install X on the server edition and run it like a desktop version, it would just require more configuration to get stuff installed and working. On the flip side, you could just as easily have your standard desktop installation act as a file/print/database/whatever server and still use it like a normal workstation.
