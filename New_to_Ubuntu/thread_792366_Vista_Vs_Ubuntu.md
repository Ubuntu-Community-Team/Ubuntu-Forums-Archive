You can watch wmv and dvd's with many options of front ends.  However, you'll have to install some third party software (trust me, very easy with lots of step by step tutorials out there) because ubuntu cannot legally ship software (codecs) that are not open source (liscenced under the GPL), even if the software is monetarily free.

Your best bet is to dual boot.  You can run both vista and ubuntu on the same machine.  I did that on my toshiba, and for a while i've been at the point i never need to boot windows.
