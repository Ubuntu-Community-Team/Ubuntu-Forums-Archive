### Post by christopher-lees on 2013-11-05
It wouldn't matter if you had an 8-core Snapdragon CPU with top-of-the-range GPU and 4 GB of RAM in that tablet. If Ubuntu has not been ported to that specific tablet, then you can't run it.

Unbranded devices from China usually won't have a port of Ubuntu available. Stick to more popular devices and you'll have more luck.
