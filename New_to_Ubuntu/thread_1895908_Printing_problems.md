### Post by seawolf167 on 2011-12-15
> **bytescribe said:**
> Without installing the hplip-gui/toolbox thing...

You don't need the hplip-gui to print anything, all that package does is provide a GUI for the HP toolbox. See [here]("http://packages.debian.org/squeeze/hplip-gui") for more information regarding that package. The place "with the printer name with the checkmark" is where the printer driver is actually being assigned and used (as a basic function print driver)
