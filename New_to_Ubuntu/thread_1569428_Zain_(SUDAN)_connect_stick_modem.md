### Post by t0p on 2010-09-06
I take it that when you insert the stick it doesn't (eventually) appear as a modem in the network-manager?

I have a Huawei E160X (I think) that uses the Vodafone (UK) mobile network.  I seem to recall that before I could use the modem, I had to insert it into a machine running Windows (I used XP) so it could register or configure itself or some such thing.  After that, I could access it through network-manager no problem.  I still use it sometimes.  It's a very nice modem, except it's locked to Vodafone and their extortionate prices.  

If you still have problems, the bottom link in my sig [[_How to use a cellphone as a modem]_]("http://tinyurl.com/dfrwwa") might be of some use.  The program **wvdial** may be particularly helpful.  There's also a link there to further info about using mobile broadband.

Good luck.
