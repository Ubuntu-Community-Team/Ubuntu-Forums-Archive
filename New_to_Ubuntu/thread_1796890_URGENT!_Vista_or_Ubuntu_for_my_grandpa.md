### Post by wep940 on 2011-07-05
As far as the original question goes, I can relate to you my similar experience.

My parents at least have used Windows a little for several years - reading email and playing simple games most of the time.  The problem also has been that they also never keep their anti-virus and malware software up to date, let alone run it.  So I have been on the phone (they live 2000 miles away) trying to straighten things out for them many times.  The last time they just gave up.

So, I sent them an older laptop to use, but I did something a little different:

[LIST]
[*]I did set it up for dual boot - Windows XP for their games and other apps they are used to (similar to what your grandfather would probably need to know if he ever tried to use another computer).
[*]I set up Ubuntu with the idea they would use it to get on the internet due to the lack of viruses and malware for Linux.
[*]I first made sure ubuntu was up to date, then I turned off updates so they would never have that come up and then question it.
[*]I edited the grub menu file to only show  the Ubuntu boot line and the Windows boot line.  Since no updates will be run, I don't have to worry about the menu changing.
[*]I changed the text title for the Ubuntu  boot line to "Get On The Internet"
[*]I set up the wireless to auto-connect
[*]I put firefox &u in the startup entries
[/LIST]
The result is the boot page only has 2 options:

Get On The Internet
Windows 

If they select Windows that's what boots.  If they select Get On The Internet, Ubuntu boots, then the wireless auto-connects and firefox is started automatically.  

This has worked out well for them.  They are both actually older than your grandfather.

I have no idea if this would work in your situation, but it worked well for my elderly parents and I don't get calls any more about problems on the computer.  They are quite happy, and so am I.
