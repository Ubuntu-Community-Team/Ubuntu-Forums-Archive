### Post by C.S.Cameron on 2008-10-23
Thanks to theozzlives I have found that the Live CD can be made persistent using a flash drive.
The flash drive is divided into two partitions.
The first partition is labeled "casper-rw", the second is labled "home-rw".
Typing "F6" and then "(space)persistent" at log in, will save settings and installed programs from the session to the USB drive.

Also noticed that when using a bootable flash drive created using usb-creator, that persistence defaults to this second drive.

Not sure what practical use this is, besides for separating home to a second drive, but it is sorta cool.
