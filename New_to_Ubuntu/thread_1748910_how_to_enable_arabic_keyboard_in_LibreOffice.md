### Post by grahammechanical on 2011-05-04
You do not do it in LibreOffice or OpenOffice. You set up an alternative keyboard from System Settings. The instructions are slightly different if you are using the Unity interface or the standard Ubuntu interface.

With Unity you go to System Settings by clicking on the shutdown icon and selecting the last line on the menu. Then look for Keyboard on the page that appears. Not Keyboard shortcuts, but Keyboard. When the utility opens select the Layout tab and click Add. You can then make a selection either by country or Language.

When you have finished an icon of a keyboard will appear in the top panel. And you can click on it to select the language you want. Do this when you have opened a LibrOffice document and the keyboard will type in the language you have selected. You can change languages and type in different languages in the same document. 

This keyboard utility will let you set LibreOffice up so that you can type English in one document and then switch to another document and the keyboard layout will change to Arabic. To do that put a tick mark against the box called Separate language for each window. If you click on Options to can choose a key to switch back and forth between keyboard language layouts.

Be careful of this one thing: If your login password is in English and you change the keyboard layout to Arabic and then you are asked to authentic an operating system action, such as updating, your password will be typed in Arabic not English and will not be accepted as the correct password.

With the standard Ubuntu interface try System>Preferences>Keyboard.

Regards.
