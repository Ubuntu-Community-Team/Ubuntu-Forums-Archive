### Post by scott_g on 2012-10-03
I've recently upgraded to Ubuntu 12.04, and am having an odd issue with my keyboard.  When pressing the same key more than once in a row causes an audible beep, and the key stroke is not recorded.  For example, typing 'press' misses the second 's', unless I wait ~2 seconds and press 's' again.

The key press does not have to be sequential either - Ubuntu comes out Ubntu when typing at a normal speed.

This gets quite annoying when using the backspace and arrow keys...

I've tried two separate keyboards across multiple desktop environments.

Any ideas?

Scott

---

### Post by scott_g on 2012-10-03
I stumbled across the solution...  some how, 'bounce keys' were enabled in All Settings > Universal Access.  No idea how that happened, although I hope this helps someone else if they come across the same issue.

Scott
