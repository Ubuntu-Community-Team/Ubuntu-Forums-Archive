### Post by Lars Noodén on 2011-11-12
Also it helps to think of it as the shell or as shell scripting because then you think about learning on-demand or just-in-time instead of just-in-case.  There is no way to learn all the 'commands' but one can learn how to look up what programs are available and how to hook them together in order to solve any given task.  Each program does one thing and does it well.

It's fun once you get started.
