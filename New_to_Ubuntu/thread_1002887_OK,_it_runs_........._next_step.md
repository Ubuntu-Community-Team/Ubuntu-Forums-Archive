### Post by CatKiller on 2008-12-05
The [Ubuntu Studio project]("http://ubuntustudio.org/") probably has more information specifically on running audio applications well on Ubuntu, including Rosegarden and JACK.

The Ubuntu Studio distribution runs with the "realtime" kernel, which is in the repositories, but it isn't needed for running either JACK or Rosegarden.
