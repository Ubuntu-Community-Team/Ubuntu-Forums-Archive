### Post by ajgreeny on 2009-09-21
use synaptic to install **brother-cups-wrapper-laser**, and you should be good to go, or to get used to the terminal, type ```
sudo apt-get install brother-cups-wrapper-laser
```Both ways will also install a few extra packages needed for the system to use that printer.
