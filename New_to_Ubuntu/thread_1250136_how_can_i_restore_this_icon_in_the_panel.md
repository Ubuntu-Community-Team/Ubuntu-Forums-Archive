### Post by kpkeerthi on 2009-08-26
If you are missing the System Tray, you need to add 'Notification Area' to the panel. Panel -> Right-click -> Add to Panel.

If you are missing the network manager applet alone, you need to add **nm-applet** to System -> Preferences -> Startup Programs.
Type **nm-applet** at your command prompt to start in the current session.
