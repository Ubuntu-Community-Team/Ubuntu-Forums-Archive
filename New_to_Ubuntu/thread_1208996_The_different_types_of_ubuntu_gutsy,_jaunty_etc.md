> **FoxFireX said:**
> Also whats the difference of the ubuntu releases that have a different letter before the u?
The big difference is the desktop environment.  To a lesser extent what applications are installed by default. Ubuntu uses Synaptic as it's package manager but Kubuntu uses the K Desktop Environment (KDE) and Adept.  Ubuntu uses Gedit by default while Kubuntu uses Kate.  Ubuntu uses Firefox as the default browser while Kubuntu uses Konqueror.  I'm not familiar enough with Xfce to know what default applications it uses.  Google Image Search would show you the grapchical differences between the three desktop GUI's really well though.  Try GIS'ing KDE, Gnome and Xfce and see what you like best.

There's also Edbuntu, Gobuntu, Lubuntu, Mythbuntu aaaaaand Ubuntu Studio if you're interested.
