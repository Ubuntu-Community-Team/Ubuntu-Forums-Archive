### Post by bumanie on 2009-01-17
There is a program called WINE that is a set of windows-like API's that can run some native .exe programs. Documentation indicates that photoshop CS4 may work, but lightroom seemed to be unconfigurable. Go [here]("http://www.winehq.org/") to read about wine and search for programs you want to run. Probably a dual boot or a virtual environment may suit you better - at least initially.

### Post by spcwingo on 2009-01-17
Check in the wine app database.  In case you don't know, wine is a compatibility layer that allows you to run windows apps natively under Linux.  Search for those apps here:

[http://appdb.winehq.org/]("http://appdb.winehq.org/")

I think the latest version of Photoshop supported is cs2.  I have been trying to get my wife (who is also a professional photographer) to make the switch for a while now.  The only reason she hasn't yet is I can't get Portrait Professional to run under wine yet.  :roll:

Also, if you have any problems there are numerous forums for photoshop/Linux users, including the Wine website.  I'm not entirely sure if the Mac version of photoshop will work with wine, though (I'm assuming the G5 in your name means that's what you've got).  It may only work with the Windows version, but that is just a thought and should be easy to look up.  Another thing is, I believe Adobe released a version of photoshop for Solaris, so there may be a way to run it without a program like Wine...
