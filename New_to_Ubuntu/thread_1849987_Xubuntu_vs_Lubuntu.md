How to disable "System Program Problem Detected" errors in Ubuntu (Xubuntu) 12.04 LTS? 

It used to happen very rarely when I first installed Ubuntu (Xubuntu) 12.04 LTS. But after several upgrades and new software installations, these errors have become more common place and sometimes annoying even. Disabling Apport permanently helped me solve this issue. This should work for you as well. Follow the steps. 

sudo nano /etc/default/apport

Change value of "enabled" from 1 to 0 (instructions are provided in the text file itself). Save and exit. Do a system reboot. Everything should work fine from now on. And Xubuntu is still the best distro around.
