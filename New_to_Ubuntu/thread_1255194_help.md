### Post by 3rdalbum on 2009-09-01
[http://packages.ubuntu.com](http://packages.ubuntu.com)

If a package is listed as having "dependencies", and you don't have those dependencies installed on your computer, you need to install those as well before you can install the main package. If you had an internet connection on your Ubuntu computer, the package manager would automatically install those for you.
