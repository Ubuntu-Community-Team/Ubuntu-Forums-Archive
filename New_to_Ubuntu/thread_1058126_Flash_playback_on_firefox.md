### Post by jbrown96 on 2009-02-02
You have the open-source flash installed. If you go to Synaptic (System--->Administration--->Synaptic), it's called swfdec (I believe or something very similar) or Gnash. Remove it and then search for flash. Install the package called flashplugin-nonfree. Do all of this with firefox closed.
