### Post by deadflowr on 2013-10-30
You probably haven't done anything wrong.

First, let's determine whether or not you have the saucy kernel series installed.
run
```
ls /boot
```
are any the listings for 3.11.(It'll be the top entry for the various file types, abi, config, initrd, vmlinuz, System.map)
If so, then it might just be that grub wasn't updated properly.
Try
```
sudo update-grub
```

If you don't have it, then something must have happen during the upgrade to cause it to not install.
Would seem odd though.

### Post by fantab on 2013-10-30
First try:
```
sudo apt-get update && sudo apt-get dist-upgrade
```

If the above doesn't install latest kernel, then:
```
sudo apt-get install linux
```
