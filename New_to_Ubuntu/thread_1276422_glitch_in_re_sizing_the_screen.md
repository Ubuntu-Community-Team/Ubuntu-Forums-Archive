That's a rather strange resolution, but if your display really uses that then you should be able to set it up by adding the resolution and suitable vertical/horizontal sync ranges into xorg.conf manualy.

But anyway, the simplest solution for your problem with Evolution is to simply move the window up a bit so that you can reach the buttons. Just hold down the Alt key and you'll be able to move the window from any point inside the window instead of just the tile bar.
