### Post by zombifier25 on 2012-05-14
Simply install lubuntu-desktop:
```
sudo apt-get install lubuntu-desktop
```
if you only want LXDE and not all the apps that comes with Lubuntu:
```
sudo apt-get install lxde-core
```
Then log out and choose LXDE at the session chooser (the Ubuntu logo at the right most corner of your username)
