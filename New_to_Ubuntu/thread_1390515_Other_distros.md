### Post by Techsnap on 2010-01-26
> another distro you might like to play with is suse. it has a similar package manager as apt, but builds everything from source for you. 
 
OpenSUSE/SLED are binary distros with binary packages, it doesn't build packages from source. It does however use delta rpms for downloading updates to reduce filesize. OpenSUSE will install programs like Gnome and OpenOffice in more or less the same time as any binary package manager apart from zypper may be a little quicker on install time.
 
It won't take hours though, you've been mislead into thinking that OpenSUSE is a source distro. Don't post things in Help forums if you know nothing about it them.
