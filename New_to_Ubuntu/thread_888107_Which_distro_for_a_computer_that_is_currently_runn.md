### Post by snowpine on 2008-08-12
I would definitely recommend Crunchbang. In addition to being very fast, media like Youtube and Java works out-of-the-box. Crunchbang has kind of a minimalistic, zen philosophy--do you think your friend would like that? A slightly slower alternative is Xubuntu, which is about 90% the same as Ubuntu but works better on old hardware. You would need to install 'ubuntu-restricted-extras' to get media working on Xubuntu.
