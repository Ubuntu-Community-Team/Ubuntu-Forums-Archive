### Post by Autodave on 2010-05-30
> **camn said:**
> Okay, last extremely nooby question... How do you install Java?

Type this into a terminal window:

sudo apt-get install ubuntu-restricted-extras


After that installs, go to Sytem > Administration > Synaptic package Manager.

In the search block type

jre

See it it is installed.  If not, you can then check it and install it.
