### Post by forrestcupp on 2012-04-26
|{urse's link is a good one to follow, but it won't help you if you can't find the terminal.

First of all, what you want is called Gnome Shell, not Gnome. Unity is based on Gnome, but Gnome's default desktop environment is called Gnome Shell. You can install it from the Ubuntu Software Center; just do a search in the Software Center for Gnome Shell.

But in case you need the terminal, click on the Dash button at the top of those buttons on the left in Unity and start typing **terminal**, and it will come up.

No matter how you install it, follow the rest of |{urse's link to see how to log into it.

### Post by cryptotheslow on 2012-04-28
Press and hold the Super key (or Windows key) - a useful display of shortcuts will appear.

For Terminal: Ctrl-Alt-T

Or if you want a Terminal clicky in the launcher bar - just find terminal in the dashboard and drag it over to the launcher bar.

If the bar is too big click System Settings in the bar, then Appearance - at the bottom reduce the launcher icon size to 32.

If you would like it to go away - click on the Behaviour tab and put the slider for Auto-Hide to "On" - while in there set the Reveal sensitivity as high as it will go.

Then move on with life :)
