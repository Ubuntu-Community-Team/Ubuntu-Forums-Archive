### Post by Boondoklife on 2009-05-26
Well In my theme it is called gnome-main-menu.png. But I have found that some of these change from time to time with package upgrades.

Easiest thing to do is to look into /usr/share/icons/human and find the images that match the icon you want to replace and then name the image file appropriately in your theme.
