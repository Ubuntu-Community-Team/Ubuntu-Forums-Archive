### Post by Kevin McCready on 2012-11-08
clementine for recording off internet stream
jack for cd ripping 
libreoffice-writer (closer to Microsoft word functionality and much better than abiword or openoffice)
Dolphin as File Manager (file manager (PCManFM 0.9.9) is awful and slow (sorry guys)) 
smplayer (need to install PulseAudio Volume Control and then select Sound Recorder)
leafpad (brilliant little text editor for HUGE text files)
gnumeric (spreadsheet)
ibus-pinyin (for chinese - it needed tweaking)  not ipin sw
clipit (so that stuff cut to clipboard buffer is available after I quit the application)
fbreader for epub files

for my ctrl-alt-? keybindings I tweaked 
~/.config/openbox/lubuntu-rc.xml

### Post by I2k4 on 2012-11-09
Good page above.  I found this pretty helpful:

[http://www.linuxalt.com/](http://www.linuxalt.com/)

It's well worth your while to read closely about the preinstalled software on different distros and simply create some Live USBs on thumb drives to try them out.  I spent a lot of time testing, installing and removing stuff in Lubuntu before finally settling on what I like.  Might have gone with Mint or something else just for convenience, even though I love the lightweight OS performance.

Also, I'd pay attention to apps that have Windows or Mac versions or compatibility - things like TrueCrypt and DropBox are seamless working across platforms and can save grief dealing with somebody who's not on Linux.
