### Post by manishtech on 2008-06-27
If you are on windows you can use Nero for burning it. Just find "**Burn Image to Disk**" option, select this ISO and start burning.
Then make sure that CD-ROM is above HDD in BIOS boot order

Useful software in Linux to burn CD's
* Brasero
* GNOME CD/DVD Burner (bundled by default in ubuntu)
* **K3B** (best till so far,even kicks *** of Nero :D )
