### Post by snowpine on 2008-07-02
If you don't have enough RAM (I think it is either 384 or 512mb) to run the Live CD, you can download the Alternate Install CD from the same source. It will let you install Ubuntu to your hard drive (though not run it from the CD) with, I think, 256mb RAM. 128-256mb, you are best off with Xubuntu, and less that 128 you will not be able to use any of the "official" 'buntus, better off with Fluxbuntu, Puppy, DSL, etc.
