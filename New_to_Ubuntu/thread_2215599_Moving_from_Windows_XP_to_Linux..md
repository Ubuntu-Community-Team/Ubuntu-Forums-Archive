No, these programs are made for Windows only and are not available on Linux.

Some Windows programs are about to run with some special software called Wine, but those three programs are reported to not work at all or not be usable.

There are, however, Linux alternatives to these programs. You might have more luck asking in the Multimedia And Video or even Ubuntu Studio forums.

(Ubuntu Studio is an edition of Ubuntu that is designed for this kind of work, you should check it out).

Ubuntu Studio  comes with Muse Score for music notation.
Note sure what Cakewalk does but if it is kind of like Fruity Loops, then you can use LMMS (Linux MultiMedia Studio).

There is no real alternative to Band-in-a-Box. 
But a combination of MMA(Musical Midi Accompaniment, it is run through terminal) & Linuxband/LMMS which gives a GUI to MMA gives you certain options.

You can read up this link to understand the array of programs that come default with Ubuntu Studio : [https://wiki.ubuntu.com/UbuntuStudio/PackageList](https://wiki.ubuntu.com/UbuntuStudio/PackageList)
May as well visit: [http://ubuntustudio.org/](http://ubuntustudio.org/)

As has been pointed out earlier, none of the softwares you listed will run on Wine.
