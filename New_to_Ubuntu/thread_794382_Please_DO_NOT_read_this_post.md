### Post by Bodsda on 2008-05-15
I'm going to put acronyms and abbreviations and usefull things to know.


GUI -- Graphical User Interface. Which means anything that uses windows and boxes and colour as opposed to command line.

CLI -- Command Line Interface. Which means anything that uses command line as opposed to windows and boxes and colour.

Term -- Terminal

BASH -- Bourne again shell

KDE -- Kubuntu desktop

Gnome -- not an acronym but the default desktop for Ubuntu

Gedit -- A simple but useful text editor. Includes highlighting for programming

Com Docs -- Community Documentation [https://help.ubuntu.com/community/UserDocumentation]("https://help.ubuntu.com/community/UserDocumentation")

NM -- Netwrok Manager

ccsm -- compizconfig-settings-manager   A utility for managing compiz plugins

Hope this helps

### Post by y-lee on 2008-05-15
many people come to linux from windows without realizing there is a philosophy behind linux. Two terms one hears in  linux that may be unknow to windows users or confusing are GNU and FOSS.

[Foss]("http://en.wikipedia.org/wiki/FOSS") means Free and Open source software.

While one doesn't have to embrace these philosophies and some linux users do not to enjoy Ubuntu it does help to realize the motivation behind the wonderful programmers that bring us the tools we use in this great operating system :)

Hey all ;)

While we wait for an answer about the GRUB utility tools, here's something else worth exploring.

VM, VMWare and Virtual Box

VM, for virtual machine (virtualization) allows us to create a simulated computer inside our computer. 

VMWare and Virtual Box are tools to achieve this. 

Essentially, they allow us to install other operating systems or distributions (distros) independently in our machine so we can try them out without affecting our current setup.

I'm currently exploring the differences between Ubuntu and Kubuntu and having a field day. My next project is virtualizing XP securely to be able to switch fully to my ubuntu computer with all my essential working tools and stop fussing and switching constantly between 4 external and internal hard drives.
