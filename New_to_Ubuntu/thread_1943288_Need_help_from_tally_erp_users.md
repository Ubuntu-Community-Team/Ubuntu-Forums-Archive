### Post by dolphin194 on 2012-03-20
Tally.ERP is not compatible with Wine, however I would actually believe that it would run in a virtual machine. Download a virtual machining software, such as Oracle VirtualBox, and install Windows in it. That way, you can keep Windows from messing with your Ubuntu installation, and still run Ubuntu while you are using Windows.
