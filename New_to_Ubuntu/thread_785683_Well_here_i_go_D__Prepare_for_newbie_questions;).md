### Post by Delever on 2008-05-07
Some basics:

You install wine (as any other program) using Programs->Add/Remove, then search for wine, and install by checking checkbox near it.

Linux programs do not have any extension like .exe. File without extension means it is program in Linux. To run windows program, you execute

wine <program>

in terminal. To do that from graphical interface, right click on .exe file, then choose "Open with another application". If wine is in list, select it, otherwise expand "use a custom command", type wine and click open. The downside of doing it this way is that you won't see any error messages if application suddenly terminates.

To do it from command line, try following guides which were posted by others.

### Post by dublued on 2008-05-07
> **nubnubjoeyg said:**
> ok so major genius here, spent 8 hours trying to set up wireless on ubuntu 8.04 atheros wireless card.... garbage... but now its up and running, and i cant figure out how to use wine to install my world of warcraft... plus i want to put beryl on and have no idea how... anyone got instructions for both... need very precise details... please type slowly :P

You may not need Beryl.  Compiz already comes installed with Ubuntu 8.04.

All you have to do is install the manager.  It's very simple. 

Go to System>Administration>Synaptic Package Manager
Put in your password when prompted.
If this is the first time you have started Synaptic Pacakge Manager, then click on the Reload button from the top toolbar.
Once that is done, click on the Search button.
Search for compizconfig-settings-manager
Once it comes up in the search results, right click it, and select Mark for Installation.  Click ok on the window that comes up.
Now simply click on Apply and watch it download and install.

After that is done.  Just go over to Sysmtem>Preferences>Advanced Desktop Effects Settings

that should give you access to all the effects settings you need.  go wild.

an even simpler method though is to do the following:
Start terminal by going to Applications>Accessories>Terminal
Once terminal has started put in the following command:
```
sudo apt-get install compizconfig-settings-manager
```
It will prompt for your password, enter that and watch it install.
When it's done you can access the config manager as described above.
