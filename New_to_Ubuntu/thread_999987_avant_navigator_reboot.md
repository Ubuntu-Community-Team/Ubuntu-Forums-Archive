### Post by adult swim on 2008-12-02
you need to go to system>preferences>sessions and then select add.  in the box that pops up, write in the following
Name: Avant Window Navigator
Command: avant-window-navigator
Comment: dock
then click add and it will start the next time  you reboot.

### Post by Joeb454 on 2008-12-06
If you drag an icon onto the dock, this creates a launcher. The first instance of a program will be assigned to this launcher. **However** if you open another instance of that program, another icon will appear in your dock.

Personally I prefer this to the method OS X uses, because I can then select individual windows :)

I hope I made that clear, it is a little confusing
