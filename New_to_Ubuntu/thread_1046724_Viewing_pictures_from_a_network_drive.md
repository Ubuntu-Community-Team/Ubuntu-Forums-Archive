### Post by moody_mark on 2009-01-21
Im running ubuntu 8.04 and using the default file viewer (Nautilus 2.22.5.1). I have a network drive which I connected to with Nautilus and its running a windows share so its mapping it as smb:///192.168.2.2.

How can I view pictures as thumbnails on this drive using Nautilus rather then having to copy them to my local machine first? I can only see the filenames and a genric icon instead of thumnails of the pictures

---

### Post by Delvien on 2009-01-21
In Nautilus click Edit> Preferences> Preview Tab> Show thumbnails> Always
