playonlinux is just a wine frontend to help you install some windows programs not run them, and yes I think (not very sure) DVD Decryptor can work in wine but theres no need to install playonlinux, install wine only then install DVD Decryptor and see if it will work.
but you should try some native linux dvd rippers, some of them are just as good as the windows ones, search for them in synaptic (type dvd in search box) there are about a dozen of them to choose from, my favorite ones are k9copy and dvd95.
