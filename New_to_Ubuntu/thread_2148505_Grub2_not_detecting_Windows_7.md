### Post by Mark Phelps on 2013-05-26
There are none of the Win7 boot loader files on "sdb " -- so GRUB doesn't see the OS.

You might be able to repair this with Boot-Repair:  [http://ubuntuforums.org/showthread.php?t=1769482]("http://ubuntuforums.org/showthread.php?t=1769482")
