---

### Post by keplerspeed on 2009-06-24
You can make another partition, so you have a total of 3, and triple boot.

So it depends, do you want to get rid of windows, and just install ubuntu onto that partition? This way you just need to go to manual configuration in the partition editor section of the install, and select the windows partition as ext3, format and mount as /.
