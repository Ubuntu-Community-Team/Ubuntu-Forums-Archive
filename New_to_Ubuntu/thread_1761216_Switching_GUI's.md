### Post by compmodder26 on 2011-05-17
As far as desktop environments for official ubuntu distributions, you have Gnome/Unity, KDE, LXDE, and XFCE.

You can install from either apt-get on the command line or through the software center and/or synaptic.

To get KDE, you will want to install kubuntu-desktop.  For XFCE, install xubuntu-desktop, and for lxde install lubuntu-desktop.
