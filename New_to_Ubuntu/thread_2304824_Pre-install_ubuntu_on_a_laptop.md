### Post by Lars Noodén on 2015-11-30
Exciting.  

I would recommend always staying with LTS.  So for now that means 14.04 LTS.  Later, when 16.04 LTS is ready, you can look and see if you should stay with 14.04 a while longer or not.  But 14.04 LTS is good through April 2019.

As far as installation goes, if you want to set the system part way and let the users fill in the details, then there is (or was) an OEM mode for installation.  I don't know if it can be done in bulk, though.

### Post by grahammechanical on 2015-11-30
If the user will not have internet access because of location then a main hazard to computer security will have been removed. An important part of support for any version of Ubuntu is security fixes. That machine will not get that support without internet access. But, then again, the machine will not be at risk if it is not connected to the internet.
