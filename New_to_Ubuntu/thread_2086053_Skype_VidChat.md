### Post by newb85 on 2012-11-19
I assume you're using Skype 4.0.0.8.  If that's true and there aren't issues with Ubuntu integrating with your webcam, the following should apply.

Once you start a call with a contact, you should have an option to turn on video.  Also, if you go to options (Ctrl+O from the main Skype window), under "Video Devices" there's an option to "Start my video automatically when I'm in a call".
