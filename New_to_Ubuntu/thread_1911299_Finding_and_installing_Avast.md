### Post by BC59 on 2012-01-18
The security issue was covered by @migs73. I agree -don't use an antivirus- it's useless.
About installing a program, for you now the easier way is through the Ubuntu Software Center. In Ubuntu latest versions you can find it, pushing the wheel at the upper left corner -the dash- and wright in the search box, the first letters, like Ubu... and you can see it. Open it and check which programs you like to install. Push Install, put your password and that's it.
