### Post by michaelzap on 2010-10-17
I know that it's possible to run Photoshop in Ubuntu using Wine, and I know that lots of people do it, but I really don't understand why. Running Windows apps in Wine is always slower and more problematic than running them in Windows. I'd say that if you really can't live without Photoshop, you could dual boot Windows or run it in a virtual OS and save yourself the struggle with Wine.

I dual boot Windows and Ubuntu just in case I need Photoshop, Flash, Powerpoint, or AutoCAD. But in practice I pretty much never need Windows software anymore (except occasionally to walk other people through fixing their Windows problems over the phone). I know that the Gimp doesn't please everyone, but it's plenty good enough for me (and far faster than Photoshop could ever be).

I'd suggest that you keep Windows around just in case, but keep it and all its software separate from Ubuntu. Try to use the native Linux apps to do what you need to do, and invest a little time and effort into learning them. Like me you may find that you no longer need Windows at all and can't imagine ever switching back.
