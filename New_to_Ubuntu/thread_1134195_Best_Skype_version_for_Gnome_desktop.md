### Post by luvinit on 2009-04-23
The difference is basically that the static one comes with the extra dependencies that it needs included in the package. I find that the static one looks better because it comes with the extra files needed to make the fonts smoother. Non native gnome applications have a tendency to look fuzzy. As far as I can see, the sound problem that existed as of Hardy (sound in one speaker only) seems to have gone with Jaunty also.
