### Post by cariboo on 2009-11-26
It's pretty easy to reset the win2000 password. Ubuntu can resize the partition, but I really wouldn't suggest doing it before running running defrag at least twice.

You can reset the windows password using Ubuntu. Have a look at this [howto]("http://www.psychocats.net/ubuntucat/resetwindowspassword/comment-page-1/").

### Post by Buuntu on 2009-11-26
Yes, if you install Ubuntu you should be able to recover files from your Windows partition.  However, if that's the only reason you're installing Ubuntu then I suggest running it off of the live CD and avoiding installing the whole operating system.

### Post by jrrader on 2009-11-26
If you boot in from a live CD you can get to all your old files just by clicking on the hard drive in Nautilus.  Windows 2000 password only stops people attempting to come in the front door.
