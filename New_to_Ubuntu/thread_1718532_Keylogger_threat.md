### Post by Anthorn on 2011-03-31
It's true that any malware designed for Windows will simply not install on a Linux system and even if it does it will be locked into the home directory with no exec permissions if you are logged in as a user which everyone is on an Ubuntu system. What is most definitely not true is that the same threats seen in Windows are not seen on Ubuntu because there is rather a lot of malware that is browser based and platform independent in the same way as, for example, Firefox entensions. So the same care needs to be exercised when accepting an install in Linux as in Windows.

Are there browser based, platform independent keyloggers? Probably not - yet!
