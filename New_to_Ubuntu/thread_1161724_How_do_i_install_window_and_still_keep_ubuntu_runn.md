### Post by diablo75 on 2009-05-17
I once wrote a blog about installing Windows after Ubuntu and it exceeded the maximum number of allowed images when I tried to post a copy of it in the tutorials forums here.  In other words DON'T BOTHER trying to install Windows after Ubuntu.  It's better to backup whatever you need to an external hard drive, install Windows on a formated hard drive, then reinstall Ubuntu after.  It is by far much safer and easier to do by comparison.

Alternatively (as mentioned above) you could just install Windows inside of a virtual machine.  It would actually be the quickest way to get that OS running and use it for productivity purposes, but if you're goal of installing Windows is to run games or 3D graphics based software, you'd be better off with installing Windows natively on your HD after removing Ubuntu, and then reinstall Ubuntu.
