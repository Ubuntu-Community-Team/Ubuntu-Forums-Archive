### Post by Old Jimma on 2010-12-28
Hi Ubuntu Community:

I found that Totem Movie Player crashed on starting a DVD.... although, it has always worked nicely for playing music and streaming audio.

I did a little checking on the forums and found a thread that sort of proposed a solution.

[COLOR="Red"]**See: [http://ubuntuforums.org/showthread.php?t=1332495]("http://ubuntuforums.org/showthread.php?t=1332495")**[/COLOR]

The solution that worked for me that was suggested there was to install [COLOR="Red"]**libdvdcss2**[/COLOR]

The thread suggests that you need to have the medibuntu respository enabled. 

If you don't know if you have enabled medibuntu read: [COLOR="Red"]**[https://help.ubuntu.com/community/Medibuntu]("https://help.ubuntu.com/community/Medibuntu")**[/COLOR] and pay close attention to the bash code listed under the section **Adding a Repository**. I.e., open a terminal window, and cut that code from the web page to your terminal window. Enter your password, and hit return. 

Once you have that done, use synaptic to install [COLOR="Red"]**libdvdcss2**[/COLOR].

It worked for me! :P
