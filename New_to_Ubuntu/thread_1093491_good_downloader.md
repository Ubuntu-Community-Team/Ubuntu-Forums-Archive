### Post by jjacobs2 on 2009-03-11
Darin, I would suggest jdownloader.  It's an open source java program.  It's designed for downloading from megaupload style sites but it also works fine with youtube and gets updated frequently.  All you have to do to use it is open the program and copy the link to clipboard.  It should be added to the program for download and it'll automatically download as many links as you want.
