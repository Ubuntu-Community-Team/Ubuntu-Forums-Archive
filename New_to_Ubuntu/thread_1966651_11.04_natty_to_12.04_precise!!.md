### Post by zombifier25 on 2012-04-27
You should backup your entire home folder (including hidden files) so that your personal files, configurations, tweaks, etc. is preserved. For installed applications, run this in the terminal:
```
dpkg --get-selections > somefile.txt
```
And all of your packages will be listed in the file somefile.txt
I don't recommend feeding this file to dpkg to reinstall all of your packages, because between Ubuntu releases things will change, and doing so may break your system. Instead, use that file for reference and reinstall your softwares manually.

### Post by 3rdalbum on 2012-04-27
This is actually a lot easier than these other people are suggesting.

You don't need to repartition or even back up your /home (although the latter is always a good idea anyway).

Just run the Ubuntu 12.04 CD and run the installer. When you get to the part where it asks you if you want to erase a disk, resize a partition etc just tell it you want to do an Upgrade. The Upgrade will do a fresh install but preserve the contents of your /home folder, and also remember what software you have installed and then reinstall that software once 12.04 is installed.

Note that unfortunately the Upgrade option doesn't seem to work if your hard disk is formatted as "btfs", but it would only be formatted this way if you had purposely chosen this option when you installed 11.04. A quick look at:

```
sudo fdisk -l
```

will tell you if there are any btfs partitions, which is unlikely anyway.
