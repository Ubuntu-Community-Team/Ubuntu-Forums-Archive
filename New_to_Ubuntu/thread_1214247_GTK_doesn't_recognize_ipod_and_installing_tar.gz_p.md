### Post by ptn107 on 2009-07-15
Your iPod *may* work with Songbird after you tell it to install the iPod addon.  Getting Songbird working is easy:

1) Download Songbird 1.2 for _[32-bit]("http://download.songbirdnest.com/installer/linux/i686/Songbird_1.2.0-1146_linux-i686.tar.gz")_ or _[64-bit]("http://download.songbirdnest.com/installer/linux/x86_64/Songbird_1.2.0-1146_linux-x86_64.tar.gz")_ Save the file wherever.  (You may need the latest nightly build release for iPod touch support: _[32-bit]("http://developer.songbirdnest.com/builds/trunk/latest/Songbird_1.3.0a-1179_linux-i686.tar.gz")_, _[64-bit]("http://developer.songbirdnest.com/builds/trunk/latest/Songbird_1.3.0a-1179_linux-x86_64.tar.gz")_)

2) Double-click the tar.gz Songbird archive you downloaded and it will open up in file-roller.

3) Drag the 'Songbird' folder from that archive to a place like your desktop or home folder.

4) Run Songbird by going into the 'Songbird' folder you extracted and double-click the file called 'songbird'.  If it asks you to run or display it's contents, click run.
