### Post by egalvan on 2009-10-28
On a related note, if you want to install more than one Desktop Environmet, you can do this via Synaptic, or the command line.

In Synaptic, search for kubuntu-desktop, xubuntu-desktop, edubuntu-desktop, ubuntustudio-desktop, among others.

Install them, one or all, and when you boot, you will be able to choose which environment you want.

Don't forget the restricted-extras, and medibuntu.

I have at least three computers with multiple environments.

It does lead to crowded menus.
