### Post by keplerspeed on 2008-10-14
I have sucessfully partitioned the HD that I was booted from, using Gparted. gparted is a very simple partition editor just like the partition editor when installing ubuntu (though, these are probably the same thing). Further you can burn gparted to a live cd, boot from it, and to edit the partitions, are create a new one, just drag the size of the partition in the line graph that will appear.

I would not recommend editing partitions while booted, so getting the live gparted cd is best. Just download the gparted live iso and burn it to cd. remember, partition editing can take a long time. Dont assume something has gone wrong, be patient.

Good luck, but with gparted, your task will be easy.
