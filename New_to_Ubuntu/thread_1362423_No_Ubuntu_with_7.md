### Post by george.varghese40 on 2009-12-23
Ok I think i can help you with that , because i had just installed ubuntu on windows 7.
So I need You to do this
1. Boot into windows 7. Ok. Click on My Computer. (right click). go to manage. In manage make navigate to disk management. 
2. Make a new partition from the primary partition ( shrink volume). To what ever size You want the partition of UBUNTU to be on. 
3. Remember that the partition should be made in NTFS format. Ok . And yes do format it. ( to NTFS file system)
4. Restart the computer and install from mannual partition . There u will find the shrinked volume . You can select it from there and then edit the partition my making the mount point /.

That is it . You are done.. Now You can see ubuntu installing on your system
