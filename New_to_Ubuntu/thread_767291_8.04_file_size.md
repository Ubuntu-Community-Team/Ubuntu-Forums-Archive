### Post by nhandler on 2008-04-25
Instead of using the filesize to try and determine if your iso is corrupted or not, use the md5sum. This will let you know if anything in the iso is even slightly different (corrupted) than it should be. To learn how to check the md5sum, read this page: [https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)
