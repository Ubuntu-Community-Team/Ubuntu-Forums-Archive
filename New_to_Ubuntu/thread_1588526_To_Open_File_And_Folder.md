### Post by brujoquizz on 2010-10-05
I don't think so... the idea is that spaces are the commands separators.
The easiest way to open folders that have long or separated names is to type the first characters of the folder (for example, #cd Th ) and then press the Tab button (the key with the two arrows on the left of the keyboard). The full name of the folder should appear automatically.
