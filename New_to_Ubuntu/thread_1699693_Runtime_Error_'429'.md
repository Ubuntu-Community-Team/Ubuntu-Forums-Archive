### Post by Paddy Landau on 2011-03-04
I don't know too much about running Windows programs on Linux.

However, the error is saying that it cannot load an ActiveX object, which is a Windows type of object. It is possible it will not run on Linux.

Be aware that Wine has many shortcomings and is not guaranteed to run many Windows programs, especially when the Windows programs do not adhere strictly to the correct standards.

I would recommend that you uninstall your Thibus program; install [PlayOnLinux]("http://www.playonlinux.com/"); then attempt to reinstall Thibus through PlayOnLinux. It is no guarantee, but PlayOnLinux does aid the process of installing and uninstalling Wine programs.
