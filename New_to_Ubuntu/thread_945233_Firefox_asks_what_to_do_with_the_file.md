### Post by seshomaru samma on 2008-10-12
there are many text editors for linux, ubuntu comes with gedit as default
you should have that in your 'open with ' menu
if not you can find it in /usr/bin/gedit

EDIT: just realised you are using XFCE. forget gedit , look for mousepad
