### Post by QIII on 2014-02-25
What I am saying is that it is not likely that everything will work like it might on a generic PC or laptop (and even then, a Linux user's experience may vary) and it is likely that there will be some work involved to get to a "least frustrating" condition.

Right now there is no upstream support for the touchpad issue, so a "micro-mouse" might come in handy.  As far as I know, there is no virtualization software for Chromebooks, so even virtualizing is not an option.

Remember that you are using chroot on a device that was purpose-built specifically for one OS.  So what you are encountering is not the typical experience for Linux users.

Phoronix.com has some articles on Ubuntu and Debian on Chromebooks.  There may be some comments on those articles giving some information from users who have figured out work-arounds.  You might want to see if you can find something helpful there.
