### Post by gashcr on 2009-05-18
sure, not hard at all, here's how

open the terminal
go to edit-> profile preferences
under the color tab, unckeck the "use system colors" option, which is enabled by default
then, you can select one of the schemes (green over black I think is the one you're looking for...) or make your own combination of text and background color.

Also, if you are using effects, you can go to the background tab and enable transparency to make it look slicker ;)

have fun
