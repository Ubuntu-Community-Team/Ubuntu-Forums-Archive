### Post by newb85 on 2013-10-06
Solid edge - advanced 2D and 3D CAD system.

Short answer: you can't.  

Siemens released Solid Edge for Windows only, so it doesn't natively run on any Linux distribution (of which Ubuntu is one).  Many Windows programs can be run on WINE, which is an abstraction layer for running Windows programs on Linux.  (It's commonly called an emulator, although WINE's maintainers maintain that Wine Is Not an Emulator.)  According to WineHQ, however, the ratings for Solid Edge on WINE are all garbage.
