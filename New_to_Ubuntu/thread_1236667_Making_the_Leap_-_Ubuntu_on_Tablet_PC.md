### Post by Mark Phelps on 2009-08-12
> **Schendje said:**
> I don't know much about touchscreen tablets, but I think the best way to try if it works would be to simply install Ubuntu.

Strongly disagree!! I have a tablet PC and while it was somewhat difficult to get it to work in 8.04, versions of Ubuntu since that do not work hardly at all.  Tablet PCs are especially difficult to get working because of the need for handling touchscreens/writeable screens and tablet rotation -- options that ordinary laptops do not have.

The "best way to try it" would be to burn various Ubuntu LiveCDs, boot from each one, and see which version best provides support for the tablet functions.

As to On-screen Keyboard (OSK), I used Cellwriter and agree it's one of the best.

If you want an equivalent of the MS Windows Journal app, try Xournal.  It allows you to take notes by writing on the screen with a stylus and saving the results in files.

Unlike with MS Windows, there is no specific "Tablet PC" version of Ubuntu, and apart from the OSK and Xournal, there's not much tablet-specific software, either.

Running the LiveCD will tell you what hardware is detected. It should find the USB port without any problems.
