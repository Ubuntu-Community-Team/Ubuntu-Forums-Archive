### Post by Chronon on 2011-12-01
Rockbox is not Linux-based.  It was written from scratch as a replacement firmware for some old Archos mp3 players and has been ported to an array of different hardware.  

More recently, Rockbox has been adapted to run as an application on Android.  Given that the iPod Touch has bluetooth and WiFi hardware it would probably make more sense to use Rockbox in this way, as an application within iOS.  

IMO, Rockbox would not make a viable firmware replacement for iOS devices -- at least not without a significant amount of work to enable support for bluetooth, WiFi, TCP/IP, hardware drivers, etc.  Rockbox is a volunteer project, so in order to make either of these possibilities happen it will take people with skill and interest (and access to the device) to put in the work.
