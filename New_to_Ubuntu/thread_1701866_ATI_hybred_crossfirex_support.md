This all depends on ATI/AMD. Although there are open source drivers for ATI cards, you have to download the proprietary ATI/AMD drivers to get full support. I remember talking to a guy that managed to set up four monitor setup with ATI surround and hybrid cards, so it works, but the setup can be tricky, you'll have to dive in xorg.conf file and edit it manually if somethings doesn't work OTB.
