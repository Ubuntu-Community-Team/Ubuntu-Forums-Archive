### Post by jimmy the saint on 2009-04-27
If you are in Ubuntu it should be a matter of simply dragging and dropping (or copy pasting).  You will have a hard time doing it from within windows because windows does not read ext3 (a linux filesystem) by default.  Ubuntu on the other hand, has no issues reading windows drives.
