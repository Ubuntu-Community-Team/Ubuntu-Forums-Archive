### Post by jimmy the saint on 2009-01-20
Most (with few exceptions) apps are available for 64bit in Ubuntu repos.  The ones that aren't can be installed from source.  You will see a performance boost with 64 bit.  As far as memory goes, 2 Gigs is more than enough either way you go.  I understand (someone can correct me if I'm wrong) that X ammount of memory will go slightly less far under 64 bit than 32 because it is divided up into larger chucks, but you will not likely notice a difference.  

The only question is if you are willing to download it!  Personally, I use 64 bit with no issues and I like it better.  If I already had a 32 bit disk and would have to spend a day downloading the 64 but, I might be tempted to stick with 32 bit.
