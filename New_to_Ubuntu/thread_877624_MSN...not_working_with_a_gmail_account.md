### Post by RuleMaker on 2008-08-02
This is what you need to do:
1) Open up pidgin (Applications->Internet->Pidgin Internet Messenger), when asked, click add account, choose MSN protocol from the dropdown list and enter your gmail as screen name because you use it as your live ID, enter your password, it will work.
2) If you want others to see what you are listening to, I'd highly recommend you to use "exaile" player.
In exaile go to Edit->Plugins->Available Plugins, find and install IM status.
Hint: you can install exaile trough synaptic.
