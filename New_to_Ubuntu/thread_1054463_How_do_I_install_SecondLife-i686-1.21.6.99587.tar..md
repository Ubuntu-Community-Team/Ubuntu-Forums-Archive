### Post by sandyd on 2009-01-29
ok, lets get started. Firstly, you need to know a big thing that not many people comming to linux from windows can relate. 

Windows : .exe installer
Ubuntu: .deb

so when you want to install a third party app, firstly search appname + deb on google
from here, we have two options.

we can either use the file you downloaded, or we can use the deb, which is kind of the equivalent of an installer. 

1. Using the file you just downloaded
assuming that you downloaded it to the desktop, double click on it and extract all the files to the desktop. 

Then open up the folder that just appeared in the desktop, and double click on the file named "secondlife"

select run in the window that appears...

2. using the easy way (for newbies at least)

[http://www.getdeb.net/release/3125](http://www.getdeb.net/release/3125)

click download and double click on the file once its finished downloading
