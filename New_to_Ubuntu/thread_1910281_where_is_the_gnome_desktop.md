### Post by grahammechanical on 2012-01-16
This change began with 11.04 and that is when the moans started. Then we got them again with 11.10. And I expect that those going from 10.04 LTS to 12.04 LTS in a few months will also start moaning. So, please understand that some of us (me in particular) are fed up with it.

Gnome organisation has stopped developing Gnome 2 and has gone over to Gnome 3. So, Canonical had no choice. Gnome.org also released as a user interface something called Gnome shell. It looks a bit like what you see now. And Ubuntu could have looked like that but Canonical wanted to do things a bit differently and asked if certain Canonical coded libraries could be integrated into Gnome 3 and the Gnome people refused. At this point Canonical came out with Unity. It is what you see now.

The intention is for Unity to be a unified user interface for having Ubuntu on all kinds of devices. Not only tablets and netbooks but even smart Television sets. Do a search for Ubuntu TV.

[http://www.ubuntu.com/tv]("http://www.ubuntu.com/tv")

Canonical wants to do deals with the manufacturers to get a slice of the profits. Which will be useful to keep Ubuntu being developed.

By the way are you running Unity 3D? Did you run Additional drivers? There is Ubuntu (Unity 3D) with what used to be called enhanced effects for those with the hardware capable of doing it. And then for those with less powerful hardware there is Ubuntu 2D (Unity 2D) which is what we get from a fresh install.

Ubuntu 3D runs on compiz. To modify Unity 3D install Compiz Configuration Settings Manager and look for Ubuntu Unity Plugin.

Ubuntu 2D runs on metacity. So, modifiactions made in CCSM will not work in 2D.

Oh, one more thing. This new look is not going to go away.

Regards.

### Post by MARP1961 on 2012-01-17
You can stay with Ubuntu 11.10 which works well but it doesn't have to stay looking like what you see in front of you. The default desktop environment is the new Unity Shell. Before you lose patience and downgrade take a deep breath and look at options:

1. Get used to Unity. It works well but looks asymmetric and ugly.
2. Go to Software Centre and install 'Gnome Shell'. Gnome Shell is an interesting alternative to Unity, it is similar but looks nicer.
3. Install Xubuntu Desktop. This will give you back an experience similar to what you remember from Gnome 2 in earlier Ubuntu versions.

I would advise you to visit YouTube and look at reviews for Ubuntu 11.10, Gnome Shell 11.10 and Xubuntu 11.10. Try them all out. As for another Linux distro, Mint 12 is good (although you can have Ubuntu 11.10 looking the same in Gnome Shell).
