### Post by Holger_Gehrke on 2014-12-29
Many.
WUBI is old, deprecated and isn't updated any more.
Do you want to install Ubuntu or only see whether it's of any interest to you ?
If you only want to try it, get a live-CD, boot from it and go.
if you want to install, either resize your Windows-partitions and install a dual-boot system or install a Hypervisor (VMWare, Virtual Box ...) and set Ubuntu up as a guest inside Windows.
