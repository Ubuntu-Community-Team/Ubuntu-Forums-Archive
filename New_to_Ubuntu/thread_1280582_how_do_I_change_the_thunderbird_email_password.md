### Post by LewRockwell on 2009-10-02
In Thunderbird running on Ubuntu:

Select "Edit" drop-down menu

Select "Preferences" from the drop-down menu

In the "Thunderbird Preferences" window, select "Privacy"

In the "Privacy" area, select the "Passwords" tab

Select "Edit Saved Passwords"

Edit Passwords

.
