### Post by ajgreeny on 2019-07-14
That laptop, if it has default specs of:-

AMD Dual-Core Processor C-50 (1.0 GHz)
2 GB Memory 250 GB HDD.
AMD Radeon HD 6250.
1366 x 768.

is probably not really of high enough specification for Ubuntu with the default gnome desktop to run very well.

I suggest you try a lighter desktop environment (DE) version such as Xubuntu, Ubuntu-Mate or even Lubuntu, and I also suggest you use the 18.04 version of whichever DE you choose as that will have a longer supported lifespan.

### Post by again? on 2019-07-15
I believe ubuntu wants to consolidate the different software apps. 
The application to manage sources/additional drivers and update frequency is **Software & Updates**.
Via terminal....
```
software-properties-gtk --open-tab 2
```
