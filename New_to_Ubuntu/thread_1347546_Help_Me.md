### Post by iamgeniusrnti on 2009-12-06
> **qu33ndris said:**
> Unfornatly I am unable to dual boot because I had lost the Ubuntu CD. So... If you know any other ways without using the ubuntu cd ... that would be help ful

Firstly, I feel your pain.  Folks on here are proud they don't boot into Windows--they think of it like kicking a nicotine habit.  I play rpgs and would rather not dink with Wine so I keep a small partition around just for gaming and the like.

There are a 1000 ways to install ubuntu without a CD.  Personally, I dont mess with the CDs at all.  Install a LEGIT copy of Windows.  Windows is simply too pissy to be borrowing copies these days and if its IP stack was as secure as it's copy protection, we wouldn't need Linux at all.

Then get "unetbootin" installed.  Download Ubuntu iso files from [http://www.ubuntu.com/GetUbuntu/download](http://www.ubuntu.com/GetUbuntu/download).  Then get you a thumbdrive, run up unetbootin.  Tell it you want to install Ubuntu, browse to the ISO and it will format the thumbdrive.

Boot into the thumbdrive, run gparted.  Shrink the Windows partition and leave the space after free.  Install Ubuntu and tell it to auto partition using the most free space (or something like that).   Badda bing, badda boom, your're done.
