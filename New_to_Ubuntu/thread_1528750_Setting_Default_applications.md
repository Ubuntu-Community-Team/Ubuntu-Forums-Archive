How do I set the default program for opening a certain kind of file?

### Post by mikewhatever on 2010-07-11
You could right click a file (or alt+enter), select Properties, select the 'Open with' tab, and chose the applications you want. The topmost one will open that type of files by default, the rest will show in the right click menu.
