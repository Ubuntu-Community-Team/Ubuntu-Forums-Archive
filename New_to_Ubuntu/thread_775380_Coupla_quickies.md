### Post by alexandremrj on 2008-05-21
You can add to the right click menu in nautilus with the package nautilus-actions

Search for it in Synaptic or use: sudo apt-get install nautilus-actions


Then you will see it in System -> Preferences -> Nautilus Actions Configuration

You can then add several commands to the right click
