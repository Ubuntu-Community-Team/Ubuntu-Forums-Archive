### Post by irrdev on 2008-09-18
> **PocketDog said:**
> Kubuntu uses Adept package manager
Synaptic works perfectly fine on Kubuntu, too. I personally far prefer it over Adebt! Go to the desktop, and hold down ALT+F2. A dialog will show up, and starting typing "synaptic" in the texbox. As you start typing, a list of matching programs should show up below, including Synaptic. Double click on the Synaptic icon, and you are off!

NOTE: If for some reason Synaptic doesn't show up in the application runner, you can always run it from the command-line. Just type this command in konsole or the terminal and click enter:
> gksudo synaptic
