### Post by NightwishFan on 2009-06-20
Press alt+f2 and type:
```
konqueror
```
Then push enter.

Is there a menu editor such as alucarte (called menu editor in settings probably) on Xubuntu?


If it does not show up in the menu editor, try this command in a terminal:
```
sudo apt-get update && sudo apt-get install konqueror
```
