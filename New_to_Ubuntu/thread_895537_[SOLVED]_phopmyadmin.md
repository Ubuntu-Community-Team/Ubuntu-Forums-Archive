### Post by OffAxis on 2008-08-20
you'll also want to check your folder (**/var/www/** by default) for a phpmyadmin link to **/usr/share/phpmyadmin/**

if it's not there a
```
cd /var/www/
sudo ln -s /usr/share/phpmyadmin/ phpmyadmin

```

can add the link.
If you have any directives in the .htaccess or .conf file that interfere with the linked connection you'll need to amend those as well.
