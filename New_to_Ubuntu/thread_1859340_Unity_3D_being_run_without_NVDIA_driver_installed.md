### Post by beew on 2011-10-13
You can run Unity with just the open source noveau driver even in 11.04, but in 11.04 you have to upate your software first so in a fresh install you boot into the classic desktop the first time. But if you update and reboot you will get Unity even without the Nvidia driver. So I supose the nouveau driver that comes with 11.10 is already more up to date than anything in Natty so you don't need to drop down to Unity 2d to do an update in order to get unity3d.
