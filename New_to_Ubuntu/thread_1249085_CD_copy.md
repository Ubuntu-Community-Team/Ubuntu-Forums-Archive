### Post by 3rdalbum on 2009-08-25
If it's a video DVD that uses encryption (i.e. a commercial DVD) then you can't simply copy it; you need to decrypt it first, which requires a program like K9Copy.

If it's not a video DVD, have you tried just right-clicking the icon on the desktop and choosing the "Copy Disc" function?
