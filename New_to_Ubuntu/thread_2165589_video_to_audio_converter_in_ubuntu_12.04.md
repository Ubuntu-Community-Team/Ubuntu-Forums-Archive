### Post by deadflowr on 2013-08-05
As long as the audio file is ffmpeg compatible and you have the ffmpeg goodies installed on your system.Then audacity will work.
I usually do this with audacity over winff or cli because the editor options are easy to understand(such as cutting the size down and adding effects)
and easy to use.
I only learned it was possible because I wanted to edited a sound file but was in my Video folder by accident.
I had a video file with the same name sans the extension, so a picked the file to open, or at least thought it was the file, and it started the import/convert program, which normally didn't run.
Of course, I also looked in my video folder to make sure the file didn't get overwritten somehow(never know!)
