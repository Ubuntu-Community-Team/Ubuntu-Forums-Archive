### Post by wigout on 2011-05-07
Some old versions itunes worked with with wine- itunes 7 maybe- but the only solution that seems to make sense is to run a virtual machine (like virtualbox), install some old version of windows you've got hanging around (windows xp, whatever) and let that be your windows source for running programs that require windows.

I know that sucks. But after hunting around for alternative approaches, that's what I've settled on.

Good luck,
