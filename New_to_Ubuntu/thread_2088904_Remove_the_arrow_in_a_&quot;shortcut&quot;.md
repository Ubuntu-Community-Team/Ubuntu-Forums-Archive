which desktop environment are you using, and what are the links for?
 
(I assume that you are talking about linked files, not launchers, since only links get the arrow at least on Gnome desktop. And there is no direct equivalent of a windows "shortcut"...)

If you are using Gnome, and they are links to files or directories, then the only way would be to edit your icon theme and replace the arrow emblem with empty transparent image.

And if they are links to applications, the solution is to create a launcher for starting the program instead of making a link to the executable file.
