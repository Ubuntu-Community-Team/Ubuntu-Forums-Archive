### Post by Kirboosy on 2011-01-27
Open up Synaptic (System-->Admin-->Synaptic Package Manager)

in the quick search box type "startupmanager"

check the package that appears and install it by clicking the apply button.


Open up System-->Admin-->Startup Manager.

From here you can change your Grub time and the default boot option. :D
