### Post by rockerphil on 2008-06-22
ok, an antivirus is not needed seeing as with Ubuntu all your software comes from a central repository, so viruses are verry uncommen in Linux.so there's no need in slowing down your machine with one. as far as a firewall goes Linux comes default with it's own known as "iptables" but requires more work than the n00b wants to do, so for a good GUI firewall there's a few good ones. gnome-lokkit is a good "set it and forget it" app and can be installed either through Add/Remove, Synaptic or the terminal with aptitude or apt-get. i personally like Firestarter, which can be installed the same way. or if you're looking for a bit more customization from your firewall you could go with guarddog, but it takes a bit more tweaking to get it working properly. hope this information helps

### Post by WitchCraft on 2008-06-22
You never need to, but you always can.

The advantage of Linux/Unix is that Viruses don't work very well, because you would need root privileges and you would have to recompile them for every major system...

However, I would recommend you to install a rootkit checker/hunter, to periodically check for rootkits, since rootkits are more common on Linux than on Windows.
