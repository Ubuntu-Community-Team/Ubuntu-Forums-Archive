### Post by NightwishFan on 2011-09-21
Linux will usually use 100% of the available cpu power when an application requests it. If you are using less, it means an application does not need it.

[http://en.wikipedia.org/wiki/Completely_Fair_Scheduler](http://en.wikipedia.org/wiki/Completely_Fair_Scheduler)
