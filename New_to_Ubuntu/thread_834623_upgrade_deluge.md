### Post by subfactors on 2008-06-19
I am using Deluge 0.5.8.9. Every time I open the program, it remains me that there is a newer version. When I click the link, it takes me to download the new version which is in deb format. 

I heard that ubuntu would automatically update all of my softwares. If that's true, how come it didn't update deluge?

If I install newer version of deluge via deb, can I uninstall it later? If so, can anyone tell me how please?

p.s. i am using ubuntu 8.04

---

### Post by bumanie on 2008-06-19
Deluge is third party software. Some third party software is maintained in repositories, but some is not. Deluge is one third party program that is not kept in repositories, thus you have to download updates yourself.  Just go to the deluge site, download the .deb and install. You do not need to uninstall the previous version first.

---

### Post by mivo on 2008-06-19
You can install the new version simply by double-clicking the .deb file you downloaded. Note that in case of Deluge you will need to uninstall the previous version first (including deluge-commons). This can be done via Synaptic (in your System menu).

### Post by subfactors on 2008-06-19
If I install deluge using the deb package, can I uninstall it using Synaptic later?

---

### Post by mivo on 2008-06-19
Yes, you can uninstall it via Synaptic later even if you manually install the .deb file. By double-clicking the .deb file you actually do install it through the same system.
