### Post by Temposs on 2009-10-06
If you're worried about being able to read web pages, you can leave your monitor's default resolution how it is, and then set the zoom property of your web browser. Assuming you're using Firefox as your web browser, you can go to View->Zoom and increase the zoom to the level you'd like. Then you can read your web pages better and still keep the best resolution for your screen.

---

### Post by Temposs on 2009-10-06
Also, if you want your various other text in Ubuntu to be larger, please go to System->Preferences->Appearance. Click the Fonts tab. Notice that all the font sizes there are at size 10(well, they are for me). Now change the sizes of the different fonts there to something bigger, until you are satisfied with the size.
