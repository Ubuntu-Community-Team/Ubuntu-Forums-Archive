### Post by Arijit_Kundu on 2011-03-23
Mount the partition containing the Ubuntu installation. I think it is sda3. run

sudo mount /dev/sda3 /mnt

if it runs OK then we can reinstall the grub. If sda3 is not the root partition then unmount it (sudo umount /mnt) and mount sda5. root partition contains folders like /etc, /boot so you can recognise.
