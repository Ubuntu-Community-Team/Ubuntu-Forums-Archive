### Post by 3rdalbum on 2012-04-06
> **expatCM said:**
> 
I am trying to find out things like
* How to add / remove icons

Three ways:

1. Find the program in the Dash, drag its icon to the Launcher.*
2. Run the program, right-click its icon in the Launcher and choose "Keep in Launcher".
3. When installing the program in Ubuntu Software Center, you will be asked at the bottom of the window if you want to add the program to the Launcher.

If it's something that is not already in the Dash (for instance, a custom script, or Xkill, or any other program that doesn't have a menu item) you can create a shortcut for it using the Main Menu program. The shortcut will appear in the Dash and then you can drag that to the Launcher.
