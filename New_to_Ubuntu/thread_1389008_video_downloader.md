### Post by PenguinInside on 2010-01-24
I use the youtube-dl script

[http://bitbucket.org/rg3/youtube-dl/wiki/Home](http://bitbucket.org/rg3/youtube-dl/wiki/Home)

In addition to one-off downloads, you can specify a file in which you have a whole list of YouTube URLs. It'll then download each of them. You can specify that the downloaded files be renamed to include the video code and the video title.

It also supports downloading all videos of a playlist.
