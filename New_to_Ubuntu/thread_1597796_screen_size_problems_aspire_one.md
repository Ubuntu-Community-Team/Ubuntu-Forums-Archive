### Post by DogMatix on 2010-10-23
I also have an Acer One and the way I get around this problem is to use the **alt+click drag** key combination to shift the dialogue box around.

Hold down the **alt** key **click** anywhere in the dialogue box in which you can't see the buttons (the cursor changes to a clenched hand) and whilst still holding the keys down simply **drag** the box up a bit until you can see the buttons.
