### Post by rrnbtter on 2017-07-10
Greetings,
Are they secure, Yes. Can they keep you from doing something stupid, No. 
However from a technical point of view the browser can be divided into two parts. The engine and the desktop. It is common place to have updates to the engine which you can't see but no updates to the desktop which you see and use. Hence since Opera for instance uses the Chrome engine but supplies it's own desktop it may have updates to the desktop but no updates to the engine and vice versa. You the user have no way to know what is being updated so you cannot say that the updates are out of sync. If you use the Chrome browser you are using the Chrome engine and the Chrome desktop but you still don't know what is being updated. 
The important part is that neither the engine nor the desktop updates will keep you from doing something stupid. The horror stories are endless.
