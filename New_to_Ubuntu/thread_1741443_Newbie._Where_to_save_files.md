### Post by jtarin on 2011-04-28
Use the Nautilus file manager to graphically make your way around the system until you get used to it. Whenever you login in your environment is your "/home/usename" folder. This is where your desktop exist. If you can't find the Nautilus file manager through the menu open a terminal and type "nautilus". I normally have a "download " folder in my home directory that I use. If you use Firefox to download you can direct it to download to that folder. There's hundreds of ways to do it.....[this should get you started]("http://live.gnome.org/Nautilus").

_**USE SOFTWARE CENTER FOR SOFTWARE THAT IS TO BE INSTALLED**_.

### Post by mcduck on 2011-04-28
I'd say the first step is to stop downloading applications from web sites and trying to install them manually like you'd have to do in Windows. That's not the way programs are usually installed in Linux. :)

Just open the Ubuntu Software Center and let the application management system download and install the applications for you automatically. Or give the Synaptic Package Manager a try if you want a more powerful tool for managing the installed applications and packages.

edit: This might be helpful: [http://www.psychocats.net/ubuntu/installingsoftware]("http://www.psychocats.net/ubuntu/installingsoftware")

### Post by grahammechanical on 2011-04-28
Click on Places in the top panel and that will load the file manager and you will see four folders already in place. You can move files from the desktop to these folders. Have you noticed that you can right click the desktop and select Create folder to do just that. Whether you leave things on the desktop or not depends on whether you like a clear or cluttered desktop look.

The 11.04 release of Ubuntu is about to become available. I suggest that you do not upgrade to 11.04 for a fews weeks until you are more comfortable with what you are doing. There are changes to the way things are done in 11.04. When you do upgrade use Ubuntu classic as the desktop for a while. Unity 2D has a very different look from one you are seeing now.
