### Post by buzzingrobot on 2015-05-02
You first need to enable a setting in System Settings->Appearance->Behavior to have menus appear in the app window.

Then, you need to launch dconf-editor (install it if necessary). You'll see a column of entries on the left.  They're arranged hierarchically. Navigate along "com -> canonical -> unity" and enable "always show menus".


Also, just in case, the images from the daily-live section are not the release image.  They are images created every day that reflect the status of the current development release, which is 15.10.  At this time, there will be little if any difference compared to the 15.04 release image, but soon the difference will be greater.

However, only 15.04 server images seem to be available at cdimages.ubuntu.com, which is odd and counter to previous practice. I'd go to ubuntu.com and grab an image.

### Post by fbio7 on 2015-05-02
Hi buzzingrobot.

Thanks a lot for the tips shared and for the answer. 

I also found a command line in the link ([http://www.omgubuntu.co.uk/2015/04/things-to-do-after-installing-ubuntu-15-04](http://www.omgubuntu.co.uk/2015/04/things-to-do-after-installing-ubuntu-15-04)). 

The command line for enable menu all time is: **gsettings set com.canonical.Unity always-show-menus true**
and to turn-off and returno to default is:** gsettings set com.canonical.Unity always-show-menus false **
