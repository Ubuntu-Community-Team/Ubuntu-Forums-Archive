### Post by grahammechanical on 2013-10-13
Application windows can be resized. Just grab the window edge or corner. As regards the amount of space an email client gives to the number of emails listed in comparison to the amount of space allowed to reading each email that can also be adjusted. Applications usually have a View menu that lets us change font size. And in the Universal Access utility we can adjust the font size of system characters.

If you go to System Settings>Appearance you will find an option to reduce the size of the icons in the Launcher (left panel). It is a long time since I used 12.04, so if 12,04 does not have this feature you need to install Compiz Configuration Settings Manager through the Software Centre and somewhere in there is the means to reduce the Launcher size.

It is better to give consideration to those who need things to be large and provide an option to resize things for those who do not need things large than the other way around. Universal Accessibility is important.
