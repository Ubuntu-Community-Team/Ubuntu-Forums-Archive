### Post by nothingspecial on 2010-02-13
You do not have to install the whole of kubuntu to try kde.

Similarly you do not have to install the whole of xubuntu to try xfce.

```
sudo apt-get install xfce4
```

```
sudo apt-get install kde-minimal
```

and optionally ```
sudo apt-get install kdeplasma-addons
```
