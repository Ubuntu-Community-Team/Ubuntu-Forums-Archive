### Post by dudude on 2008-06-10
You have to install the drivers for your graphics card, then download compizconfig-settings-manager from synaptic.

You can then edit what effects are used in System->Preferences->Advanced Desktop Effects

### Post by gr4nf on 2008-06-10
beryl is the old compiz. You should already have compiz, it's a standard now. Do you have hardy?

try:
```
sudo apt-get install compizconfig-settings-manager
```

when that finishes, go to:
system->preferences->Advanced Desktop Effects Setteings
