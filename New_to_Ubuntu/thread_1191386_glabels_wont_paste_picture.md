### Post by Michael Dooley on 2009-06-20
OK.

I downloaded Glabels and gave it a try. It is pretty unintuitive as to getting an image (for instance) on a CD or DVD label. However, I did manage to actually do what I think the OP wanted. 

After selecting a template (or creating your own), then select the Objects, Create, Image command and use your mouse cursor to describe a box. The box is filled with a checkerboard pattern. You can resize this box, move it and change its horizontal to vertical proportions. 

In the "Image object properties" dialog box that displays to the right, you can browse for the desired image. Select the image file, click open and it appears in the image box.

Its a little obscure as far as usage and documentation, but you [COLOR="Red"]can[/COLOR] make it work. I printed a test DVD label and it printed out OK. If I continue to use this app, I'll make my own templates though.

Good luck stinger30au.
