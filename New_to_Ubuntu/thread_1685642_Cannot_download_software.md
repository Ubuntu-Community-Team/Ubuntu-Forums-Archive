### Post by sikander3786 on 2011-02-11
Welcome to the forums :-)

GIMP is not on the install disc and for most of the software, you need to install them from internet/repositories.

So, go to Software Center > Edit > Software Sources and _disable_ CD-ROM. Then click Close > Reload.
