### Post by HotShotDJ on 2008-12-06
Check out webfs ( [http://linux.bytesex.org/misc/webfs.html](http://linux.bytesex.org/misc/webfs.html) )
It is in the Ubuntu repositories, so you can install it easily using Synaptic or from a terminal with```
sudo apt-get install webfs
```
