### Post by 3rdalbum on 2013-10-05
Ubuntu Desktop is simply Ubuntu Server, plus a desktop environment.

Or, if you prefer, Ubuntu Server is just Ubuntu Desktop but without a desktop environment.

If you want Ubuntu Server plus a desktop, then *you're already running it.* If you don't want the desktop, then just remove it and presto - you've got Ubuntu Server.

I see no reason why the Ubuntu Server disc isn't loading, assuming you've burnt it and tried to boot it the same way as you did Ubuntu Desktop. Maybe the burn was bad. Maybe the download was bad. If you really want to, you should check if it works on any other computers, and check that your BIOS is set to try booting from CD before looking at the hard disk. However, you don't need to install Ubuntu Server because you've already got it.
