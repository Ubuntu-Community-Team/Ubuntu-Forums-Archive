### Post by lazyart on 2008-05-14
You'll have power to spare.

You have a 64 bit processor-- go ahead and get the 64 bit OS.  Someone is bound to say "you don't have over 4 gigs of memory so you won't see any change and 64 bit has it's issues..."

I run 32 bit on my laptop (it's a Pentium M running 1.6 and 1 gb ram).  I run 64 bit on the desktop (Opteron 165 and 4 gb ram).  They both work fine, no driver issues, flash & java are happy.

One day you will go 64 bit and you'll have to wipe clean because there is no upgrade path from one to the other.  Save yourself the headache and do it when you don't have to worry about it.  DO create a separate /home partition.  One day hopefully you can get a second HD and RAID 'em for redundancy.

I have a Microfly case.  It runs my Windows Server.  Neat little box-- just built it a few weeks ago as my old server was, well old.
