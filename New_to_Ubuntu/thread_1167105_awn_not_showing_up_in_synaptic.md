### Post by raymondh on 2009-05-22
> **marenum said:**
> That doesn't work either.

Try reloading your syanptic and then look for  avant-window-navigator or awn-manager

I know you asked for AWN using synaptic.  Have you tried the terminal with

```
sudo apt-get install avant-window-navigator
```

Before doing that, make sure you have desktop effects enabled as you will need compositing.  If for some reason you cannot enable desktop effects, AWN has a ppa that can run without 3D rendering.  It'll be labeled TRUNK. 

Here are some links for your reference

[http://wiki.awn-project.org/FAQ](http://wiki.awn-project.org/FAQ)
[http://wiki.awn-project.org/DistributionGuides](http://wiki.awn-project.org/DistributionGuides)

Good luck ... let us know how it goes.
