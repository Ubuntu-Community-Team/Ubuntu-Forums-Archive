Hello,

Still learning Ubuntu, Netbook Remix 9.04 version, and ran into a small problem with user administration.

I added my son, as a user, with both a username and password via the "Users and Groups" function (in administration). While doing some other messing around with the system, I later deleted that username via same users and groups function as it was established.

Now that I've added more hard drive space to support more users, when I try to add my son back with the same username that was used (and deleted) before, it tells me that it already exists...yet it doesn't show under the users on the "Users and Groups" function.

How do I fix this so I can add him again under the same username and pass word as before?

### Post by nomadluap on 2009-09-19
You also have to delete the group named after your son's old username. 

in Users and Groups, unlock the window, then click the Manage Groups button. Find the entry named after your son's old username, and delete it.
