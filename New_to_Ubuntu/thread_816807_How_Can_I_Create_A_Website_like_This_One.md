### Post by Samhain13 on 2008-06-03
What is MySQL and PHP? **A database. A scripting langage that helps you create web pages dynamically (among other things).**

Can we use MySQL and PHP in on Ubuntu? **Yes.**

Do I have to use Ubuntu server edition If I will create a website? **Not necessarily. But you will have to install additional services like the Apache webserver if you plan on creating dynamic websites. Otherwise (if you plan on making only static websites), you don't have to install any additional stuff in your Ubuntu desktop.**

If ever I have finished a website... How can I put it in the internet so people can see and visit the site, and post their questions? **You'll need a hosting service (web host) and a domain name service (dns) provider.**

### Post by Paddy Landau on 2008-06-03
If you're a newbie, I strongly suggest that you consider having a company host your website (look around; they're inexpensive these days). You don't want to worry about trying to host it from your own computer.

You'll also want a domain name (e.g. [www.mac143_01.com]("http://www.mac143_01.com")). There are many places that sell domain names; one that I've found cheap and reliable is [GoDaddy]("http://www.godaddy.com/") (they also do hosting). Or, you may find that your chosen host will give you the domain as part of its package.

If you have little or no knowledge of HTML, Javascript, PHP, MySQL and so forth, then you may find it easier in the long run to get someone else to write the website for you.

An alternative is to use a CMS (Content Management System). One free example is [Joomla!]("http://www.joomla.org/"). Once set up on your host, you can modify your website yourself. Again, you can hire someone else to set it up for you. If you go this route, ensure that your host provides PHP and MySQL.

Unless you get someone else to do everything, be prepared for a steep learning curve. Very steep.
