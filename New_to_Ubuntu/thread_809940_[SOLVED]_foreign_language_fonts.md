### Post by ZabiGG on 2008-05-27
You're not thick ;) I've been (and am still) in your shoes! lol.

I found using a dual keyboard layout (French-English) with the SCIM keyboard tool was the best option for me. It allows me to switch languages easily with a key combination and to decide which language is the default.

Go in System > Preferences > Keyboard and click on the Layouts tab

From there, you'll be able to set your languages and set a default if you want, in addition to defining other layout options that you will need.

Once you've done this, if the problem persists, please reply with the steps you've taken so I can see where the problem might lie.

Cheers,

Z.
