Actually you can softmod a stock xbox to run Xbox Media Center (xbmc) which is a very, very nice media center.  Very easy to do in fact. I have 3 running in my house right now, and said media center does have local ftp access by default, although if you want to do much storage, you'd want to upgrade to a larger, speedier hard drive.
