### Post by Geezer60 on 2012-04-04
I had a similar problem.  I found a help file that had me type "localhost:631" in the browser url block which took me to the printer server settings panel.  I'm running ubuntu 11.10.  My other computer is running 10.04 and there I had to check the box on the system/administration/printers settings to "publish all printers"  Both computers will now print to my wireless Brother MFC9325CW.  Hope this helps.
