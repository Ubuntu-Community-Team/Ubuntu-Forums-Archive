### Post by yeats on 2010-03-14
If you mean the newest version as released by OO.org, you can remove the installed version of OpenOffice, then download the .deb for the newest version from here:

[http://download.openoffice.org/other.html](http://download.openoffice.org/other.html)

pick a "DEB" that suits your computer's architecture, then double click on it once it downloads.

This is fine to do with applications like OpenOffice, but it's generally recommended to stick with the Ubuntu repository versions of programs - they are built specifically for your system and won't break anything.
