### Post by 3rdalbum on 2012-01-31
> **trolol said:**
> Just decided that lubuntu (lubuntu is the lightest one right?? please correct me if im wrong) is my best bet but I was wondering will it work common programs such as Skype, MSN, media players, and Chrome?

Any software that runs on regular Ubuntu will also run on Lubuntu. The only difference between Ubuntu and Lubuntu is the desktop environment (the GUI).

Skype and Chrome are available for Linux/Ubuntu. I'd suggest using Chromium instead - it's a little more up-to-date plus a little easier to install, and it's basically Chrome without the branding. MSN is a Windows-only program so it won't work, but you can install Empathy or Pidgin in Lubuntu which are both multi-protocol instant messengers. Or you could use the Meebo website, which is like Pidgin but web-based.

There are also plenty of media players for Ubuntu/Lubuntu. I'm sure Lubuntu comes with one, but in either case you can install VLC; that's the most popular video player on Ubuntu/Lubuntu.

> And maybe microsoft office or something similar?

Libreoffice is preinstalled on Ubuntu. I'm not sure if it is on Lubuntu, but it's easy to install. Microsoft Office is Windows software, so you can't use it.

> And will it sync with my phone so I can download music and transfer it to my phone? (nokia, running on s60)

You can transfer music to a Nokia phone just by dragging and dropping it into the Music folder, like it was a USB flash drive. That works pretty well.

> Also does it work special effects like the real ubuntu (downloading themes, 3d effects, transision effects when opening  and closing windows for example etc), and is it just as customizable or are there limitations?

Lubuntu is designed for low-performance systems, so I don't believe this will be preinstalled by default. However, you can install Compiz which is the system that provides the effects on Ubuntu. It might need a bit of tweaking to start off with.

LXDE (the Lubuntu desktop) has been in development for longer than Ubuntu's Unity desktop, so it can actually be tweaked and customized more than Unity.
