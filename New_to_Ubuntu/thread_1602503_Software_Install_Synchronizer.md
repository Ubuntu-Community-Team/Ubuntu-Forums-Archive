### Post by trikster_x on 2010-10-21
Short answer:  no 

Though there is the ubuntuone cloud program for syncing specific files to computers remotely, the ability to sync programs over the internet would be a massive security risk for home or business computers.  The reason it works so well for Chrome is because all those extensions are stored on a remote server.  Could you imagine someone managing to gain access to one remote computer and be able to effect three or four more?  Or if you made an accidental change to a program that locked you out and the other comps updated and locked you out of their program as well?   Although it is a fantastic idea, it would have the ability to cause some major problems.
