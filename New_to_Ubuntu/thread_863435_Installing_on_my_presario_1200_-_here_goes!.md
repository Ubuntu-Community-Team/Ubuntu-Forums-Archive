### Post by avtolle on 2008-07-18
If you in fact have just 188 mb RAM on that laptop, I don't know whether you're going to be able to install Ubuntu and have it run; the minimum system requirements for Ubuntu with the Gnome desktop editor include 256 mb RAM. Perhaps consideration of Xubuntu (which has a lighter-weight desktop, xfce) from the alternate install disk would be better. There are some other distros you might consider (which are Linux, not *buntu): these include DSL (damn small linux), Puppy Linux. Good luck.

### Post by seagullplayer77 on 2008-07-18
I'm going to have to agree with avtolle 100% on this one. Ubuntu is excellent at giving old and slow systems new life, but even it has limits. Ubuntu will doubtlessly *work* on your machine, but it probably won't be the speed demon you're expecting. I would definitely think about some other, lighter versions of Linux. Xubuntu is a good one, as are the other ones that avtolle listed.

Ultimately, it's up to you which Linux distro to use, but I would recommend against anything that uses KDE or Gnome for a desktop, as they're both fairly memory intensive. Xfce would probably be better suited to your machine. That said, good luck with your install!
