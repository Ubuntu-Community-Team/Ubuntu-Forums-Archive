### Post by Sef on 2009-04-27
There are two options in this case:

1) Buy a printer that will work with GNU/Linux.

2) Buy an old XP computer and set it up as a print server.   That means that the printer will print from the XP computer.   Your Ubuntu computer will be connected to the XP computer for printing purposes.

The old computer can be a P3 or P2.  Get one for a thrift store, ebay, or other place that sells used computers.  They are fairly cheap.
