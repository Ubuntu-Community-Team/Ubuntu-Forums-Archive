### Post by jrothwell97 on 2009-08-23
Welcome to the Ubuntu forums!

Of course you can install Ubuntu on your hard drive. Judging what you've said, I presume it's blank and has no data on it, in which case you can tell Ubuntu to occupy the entire hard disk during installation. Just tell it to use guided partitioning and use the entire disk when it gets to the "partitioning" stage, and Ubuntu will automatically determine the sizes of the / partition and the swap partition.

### Post by rg_stephens on 2009-08-23
Right. Installing Ubuntu is much easier than Windows. Go to Ubuntu.com and download the desktop version of Ubuntu, (64 or 32 bit, whichever you have), and burn the image to a disc. Boot from the disc you just created, and you when it starts up and you run Ubuntu, you'll have an "Install" icon on the desktop. The default options are usually ok. Make sure you have your data backed up before you do an install, because it will re-format your drive. You can do this when you're running the CD. 

When you boot the CD, it also has options to check the memory for errors (a good idea to do this if you're having problems) and check the CD. 

Let us know if you have any trouble.
