### Post by sikander3786 on 2011-01-13
Welcome to the forums :-)

How was the USB Drive prepared. Which software did you use? Are you sure the USB drive is in good health?

First of all, I would recommend to check the integrity of your downloaded image to make sure it didn't got corrupt while it was being transferred to your PC.

[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

If the sums match, try using Unetbootin.

[http://unetbootin.sourceforge.net](http://unetbootin.sourceforge.net)

Keep us posted ;-)
