### Post by Xiong Chiamiov on 2008-06-01
You can see right here:
```

Section "Device"
Identifier "Configured Video Device"
Driver "vesa"
EndSection

```
that you're using the Vesa driver, which is a failsafe driver.  I unfortunately don't have an xorg onhand that uses an intel driver, but I think you just need to change "vesa" to "intel"?  You might be better off doing a 
```

sudo dpkg-reconfigure xserver-xorg

```

### Post by J A Smith on 2008-06-01
Gentlemen (and ladies?) it's fixed.

Indeed changing 'vesa' to 'intel' worked a treat.

I used 'sudo nautilus' to run at root to manually change the xorg.conf file.
