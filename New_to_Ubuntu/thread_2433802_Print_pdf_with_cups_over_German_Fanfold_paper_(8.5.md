### Post by Holger_Gehrke on 2019-12-28
If I understand the manual I found online correctly, this printer defaults to 11" page length. It has a menu system to make various settings including page length. You activate this by holding the button labeled 'font' while turning the printer on. You need to have paper in the printer at that time because it prints both instructions on how to use the system and the menus themselves. If printer and software disagree on the page length the results are usually ... interesting ...
