### Post by grahammechanical on 2014-11-16
As to your first question. Ubuntu 64bit amd64 will run on 64 bit CPUs made by either Intel or AMD. It does not matter if another OS on the machine is 32 bit. No, not at all. I have several installations of Ubuntu on my two hard disks they are all 64 bit except a 32 bit install of Ubuntu Mate and a 32 bit install of Windows 8 preview edition. They do not interfere with each other because I only use one OS at a time and they each have their own partition.
