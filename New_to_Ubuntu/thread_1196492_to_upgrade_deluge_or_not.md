### Post by lovinglinux on 2009-06-25
BTW...

> **krisalis said:**
> is it worth updating?- I am happy with the version I have got so is there a good reason to update?

I'm not sure about the 0.5.x series, but I guess it doesn't have all the remote options of the 1.1.x. For instance, you can connect to a server with cli, remote gtk and a very nice webui. As far as I remember, webui was a plugin in the 0.5.x and I don't remember anything about a remote gtk.

Anyway, if you use Deluge only locally, these features won't make any difference for you. If you are happy with it, stay with it. Like we say about football, "You don't mess with a team that is winning".
