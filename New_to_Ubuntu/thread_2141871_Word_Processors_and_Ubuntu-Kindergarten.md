### Post by Impavidus on 2013-05-04
If you want compatibility with MS Word, LibreOffice is as good as it gets. There's also AbiWord, the default in Xubuntu, but I never tried it. And consider exporting to pdf. That's the most portable way of sending your files to someone else, as everybody can read pdfs and have them displayed in exacly the same way on any computer (or even e-reader), something not even possible using different versions of MS Word. The files won't be editable afterwards.

I never liked any Word-like applications, so I was quite happy when people wanted me to move over to [LaTeX]("http://en.wikipedia.org/wiki/LaTeX") several years ago, which is about the only format accepted in physics (or at least, here it is). It can do everything you want (title page, footnotes, endnotes (if you wish as "book: chapter, verse: note"), hanging indents, per paragraph setting of line spacing – although this is quite complicated) by typing commands (no menus or even mouse involved) and then compiling, but it's quite a learning curve. I've never used anything else since, with all editing of the source done in [Vim]("http://en.wikipedia.org/wiki/Vim_%28text_editor%29"). That's quite a learning curve too. That's how I wrote reports, my thesis, my parts of articles in science journals, even fiction (with some characters occasionally discussing math problems). No GUI, but some graphical front-ends (like LyX you mentioned) are available.

But if you're in a hurry the learning curve and incompatibility with Word may be too much.

### Post by Zill on 2013-05-04
> **Shansie said:**
> Will this still work if the single spaced area has several different font sizes in it (as with my makeshift title page), or will I have to make multiple styles for the title page fonts?
The spacing is a paragraph attribute and so this will be a constant for the paragraph irrespective of font.  However, you can also have a font defined in a paragraph style if you wish.  The best thing is to make the "default" style for the majority of the document have your chosen fonts, spacings, indents etc and then create different styles for any changes to your default style.  Just play around with the various paragraph and font style dialog boxes and see how it all works.
> **Shansie said:**
> How do you set up templates in Libre Office? This seems like it would be extremely helpful for my lab reports, research papers, and essays that have to be formatted a certain way, so that I only have to go through and get it to work correctly once. I'm going to look into this.[/COLOR]
You simply save a document containing your desired layout and styles etc as a "ODF Text Document Template (.ott)".  This is a pull-down list option in the "File" "Save" dialog box.  If you then click on this saved file it will open a brand-new (unnamed) document based on the template.  This can also be done via the "Templates and Documents" manager within Libre/Open Office itself.
