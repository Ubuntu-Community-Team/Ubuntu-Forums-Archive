### Post by kaldor on 2011-12-05
Just a note of advice.

WINE isn't perfect. For me, I'm lucky in that the only Windows program I use (a game) happens to work perfectly in WINE. Aside from that it's very hit and miss.

Your best option is to just use open source alternatives to your Windows software. This will make your experience with Ubuntu much smoother than trying to get Windows software running on Linux. Keep in mind that Linux is not Windows at all.

I never expect WINE to run stuff. If it does, it's a bonus- if not, then that's to be expected. The fact that Linux can even flawlessly run some Windows programs is pretty great in itself.

If you need ideas on Linux alternatives to some of the Windows programs you need, then post back and the community will probably be full of suggestions. 

Good luck :)
