### Post by Kellemora on 2008-11-04
Hi Drake

I'm using Eudora 7.1, have been using Eudora since 1995 and still have all valid e-mails that were not trash or junk since that time.

Unless you have certain checkboxes selected, all the GARBAGE that comes with the SPAM is stored in places like the Attach folder and the Embed folder.  EVEN IF you delete those e-mails, or they scroll off the Junk folder, all those Garbage files will remain behind FOREVER.  And they are what usually take up all your hard drive space.

I've tried Thunderbird's Eudora copycat, it's still Thunderbird with a Eudora face, not what I liked.
Gave Evolution a good month trial, had none of the features I needed.

So, I tried installing Eudora in Wine and it works perfectly, so I moved ALL of my old files over to it.

Since making sure the checkmarks for empty Attachments and empty Embedded files when emptying from trash were checked.  None of the Spam attachments are saved, only those attachments that belong to valid files are retained.

The only thing hard about using Eudora in Wine is that Wine is a hidden folder, so keeping your Eudora Data file backed up can sometimes be tricky.  Also, you cannot set it up manually else it will install on the z/ drive which is the /root drive.  You have to let it set itself up in the files locations on Drive C that IT Chooses, such as Program Files.

I've not found a way to show HIDDEN files while in a programs INSTALL mode!  So just bit the dust since it really isn't Windoze anyhow, just Wine, hi hi..........

Eudora is FREE so you might want to check it out!

Oh, forgot to say, eon's ago, before I learned about the checkbox to delete attachments and the like from inbound Spam.  I went all the way back to 1995 and opened each e-mail to see if it had an attachment, if so, I would copy the attachment to a folder I called keepers (temporarily) so I could delete ALL the gazillion files that didn't belong to valid e-mails.  That was about 99% of my wasted storage space.
It's not that hard to scroll down the lists of good e-mails as they put a pair of documents icon in the header line to show if that file has attachments or not.  Embedded files are a different story, have to go through each of those one by one to see if they have any embedded files.

But once you do that, it should be the last time you ever have to do that again!  If you click the checkboxes to delete Attachments, etc.  It DON'T delete attachments UNTIL you empty trash, and it don't delete the saved e-mails attachments.  So it's safe to have those checkboxes checked.

As for address book, you could always go through and export to both vcard and csv/other formats (TB uses ldif) so you will not have a problem.
