### Post by beew on 2011-06-21
Instead of installing in a usb flash drive I recommend instead installing to an external hard drive, the procedure is the same.

I have currently 2 usb installations (Xubuntu and Ubuntu, each on a 8 g flash drive) which I lend to people to try out, the problem is that it can be quite slow and tend to freeze up for multitasking aside from the size limitation so you don't get a true picture of Linux performance. An installation in an external hard drive on the other hand gives you the same performance as an internal install, except the boot time may be significantly slower depending on your BIOS.

Wubi is buggy as hell as it introduces unnecessary complications, it has to work through Windows thus is dependent on it, it is not a real Linux install IMO. I think the bugginess and instability actually give Linux a bad name.

Also, while WUBI is promoted as a safe way to try out Linux for the timid souls who need the Windows safety blanket, it has more chance to mess up Windows than installation in an external drive because you have to install WUBI in Windows! I read a thread the other day where some guy messed up his registry after uninstalling WUBI.
