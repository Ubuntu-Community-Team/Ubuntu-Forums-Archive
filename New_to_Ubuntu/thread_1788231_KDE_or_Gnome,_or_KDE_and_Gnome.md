### Post by dFlyer on 2011-06-22
It's okay to run both kde and gnome if you have the disk space or any other desktop environment. The nice thing about Linux is the choice is yours. Other distros in the past use to install both gnome and kde in there standard install. Under kde if a gnome lib is required to run a program it's pulled in automatically and the same applies to gnome if a kde lib is required to run a program under gnome it's lib is auto installed.
