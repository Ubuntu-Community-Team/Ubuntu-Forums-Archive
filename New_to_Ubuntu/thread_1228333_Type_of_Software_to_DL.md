### Post by powel212 on 2009-07-31
+1 Synaptic and +1 add/remove

I think you are asking what the file extension is for Ubuntu install programs? The answer to this question is .deb

.rpm is for red-hat, suse, Mandriva, and fedora and other versions of Linux may use other extensions for install programs.

Most everything most people need can be found in add/remove and so it is not recommended to download .deb files from the internet and install them unless you are really sure about where they are coming from. And when you download .deb files from the internet you will also often need to download an authentication "key" which may or may not be an easy thing for a new user. 

I hope that helps.
