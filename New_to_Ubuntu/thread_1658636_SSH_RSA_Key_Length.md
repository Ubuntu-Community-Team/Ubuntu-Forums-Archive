### Post by perspectoff on 2011-01-10
Occam's razor.

Why make it more complex if it's not necessary?

It takes supercomputers working to break a 2048-bit encrypted key. Although it has been done (to show it can be done), who's going to do it routinely?

You can use 4096-bit keys, but why?
