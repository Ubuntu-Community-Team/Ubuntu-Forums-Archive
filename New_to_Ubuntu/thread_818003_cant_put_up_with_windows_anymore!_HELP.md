### Post by barnex on 2008-06-04
Try logging in and type 'startx' to try to get into graphical mode. 
Installing from inside windows is ok to play a bit with Ubuntu, but if you are going to use it as your primary/only OS, it's better to do a normal install. I suggest you try to borrow a USB cd drive somewhere, or have a look at these alternate install methods (from USB stick, network):
[http://news.softpedia.com/news/Alternative-Installation-Methods-for-Hardy-86977.shtml](http://news.softpedia.com/news/Alternative-Installation-Methods-for-Hardy-86977.shtml)
