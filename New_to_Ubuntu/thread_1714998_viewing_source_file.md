### Post by hakermania on 2011-03-26
apt-get source program-name (root not needed)
Yes, you can compile them and run them to windows, as long as they don't need something ubuntu/linux specific, and they are simple programs that do a simple job (like a simple calculator :P). But, still they may need to create files at paths that, in windows don't exist.
So, only very simple programs would compile and run well...

As for contribution, see REVU (revu.ubuntuwire.com), and, happy packaging if you try this way!
