### Post by rosencrantz on 2011-04-24
Are you looking for a presenter console with an integrated slide browser, to be used on a two-monitor setup? 
I haven't found anything like that for PDF yet - the existing PDF presenter console app has only a two-slide preview and might choke on a 100 page document anyway. 
You can import page images into OpenOffice using a python script ([script]("http://gbbaux.googlepages.com/pdf2odp.py"), [blog post]("http://eumenidae.blogspot.com/2011/01/keynotifying-pdf-presentations.html")). Additionally, you'll need  the odfpy module and Openoffice's/Libreoffice's Presenter Console.
If your page doesn't fit 1024x768, you might have to adapt the script a bit.
