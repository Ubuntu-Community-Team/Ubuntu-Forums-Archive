### Post by ramjet_1953 on 2010-01-13
If you use Synaptic Package Manager to install these packages, configuring and using Compiz is much easier.

1. [COLOR="Blue"]compiz-config-settings-manager[/COLOR]

2. [COLOR="Blue"]fusion-icon[/COLOR]

These packages provide a much more detailed user interface for setting-up compiz.

You will need to add fusion-icon to your startup applications, if you want this icon to automatically load on boot.

ie. [COLOR="Blue"]System>Preferences>Startup Applications[/COLOR].
