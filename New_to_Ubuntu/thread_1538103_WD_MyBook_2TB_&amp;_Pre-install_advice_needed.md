### Post by RJ12 on 2010-07-24
Personally I think the hard drive will work in Linux. Is there any software for that drive, if so they mean the software will work only in Windows and Mac OS. If you really want to see if it works, then you need to burn a Ubuntu CD using the iso from the download page, then boot from it, and choose the Try without changes option. Then when you plug in the drive, it should appear on the desktop (And most likely automatically open). I have never heard of a hard drive that doesn't work on Linux. If the mybook has password encryption on it, then maybe that would be the reason they say it won't work.

### Post by howefield on 2010-07-24
> **Etain said:**
> Yes, the drive comes with pre-loaded software that automatically installs itself on your computer the first time you plug it in.

It won't install itself in a linux machine, but the hard drive should work fine like any other external hard drive as long as you have NTFS support, which the drive will be formatted with unless you change it by formatting to another file system.

There may be some minor side effects of not running the software, eg, the capacity indicators may not work without the software installed.
