### Post by grahammechanical on 2014-03-29
> [COLOR=#000000]But the cpu is a 64bit cpu.[/COLOR]

Why do you think that is a problem? We do have a 64bit version of Ubuntu. It is called AMD64. It works just as well on 64 bit Intel CPUs as on 64bit AMD CPUs.

I only have 1GB RAM in my Intel Core 2 Duo machine but I also have a graphic adapter with 1GB of its own memory. I think that the power and memory size of the graphic adapter is just as important as the power of the CPU and the amount of RAM.

If you find that Ubuntu + Unity is not giving you a good user experience then we also have Xubuntu and Lubuntu which use different desktop environments that may give you a better experience.

The thing to do is to download a Ubuntu ISO and burn it to DVD or USB memory stick and then run a live session. The system will not work as fast as a hard disk installation but you can test out if all the hardware works. Even WiFi connections to the router.

I have not had any problems with USB memory sticks and Linux/Ubuntu.

Regards.
