### Post by agnes on 2010-04-29
afaik that' s not possible with the Gnome (taskbar-)panel... but you could do it with the tint (taskbar-)panel, which is more configurable, and has a clear config file.
If you remove the default bottom panel and replace it with the tint panel, you can control the maximum width of a task and the outlining. The icon size will still depend on the panel size, but you could make the panel bigger and the text smaller, same effect.  
See [here]("http://code.google.com/p/tint2/wiki/Install") for general instructions, and how to configure.
