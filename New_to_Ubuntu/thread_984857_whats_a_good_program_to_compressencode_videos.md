### Post by bumanie on 2008-11-17
Devede encodes nearly any codec to dvd output or avi - from having used it, I also thinks it provides some compression, but this is not its main purpose - its more a side effect of the 'behind the scenes' tools it uses. I could be wrong on the last point, but it what I've noticed, the stated size of the input file is always reduced post encoding.

PS: It depends a bit on what you are wanting to compress - eg., there are tools for shrinking dvd's
