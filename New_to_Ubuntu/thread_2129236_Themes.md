### Post by ManamiVixen on 2013-03-25
So it's Ubuntu then. Go to gnome-look.org. They have lots of themes for Ubuntu. The themes are in the GTK3 section. **BUT** **THEY HAVE TO STATE GTK 3.4 COMPATIBLE. **Then to apply the theme, you will have to install myunity "sudo apt-get install myunity". That program lets you tweak Unity and the theme.
