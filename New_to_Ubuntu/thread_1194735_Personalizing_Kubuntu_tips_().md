### Post by mgranet on 2009-06-23
KDE 4.2 has a nice compositor, which supplies all the eyecandy natively. No need for compiz. You can find it's control system in System Settings-->Desktop. Make sure you have the proprietary drivers for your video card installed if you have an ATI or Nvidia card. If you are running a newer Intel graphics card, should already be installed.
