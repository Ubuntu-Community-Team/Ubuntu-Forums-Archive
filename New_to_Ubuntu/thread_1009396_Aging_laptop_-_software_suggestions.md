### Post by ibuclaw on 2008-12-12
Window Manager: [LXDE]("http://lxde.org/") or [XFCE]("http://xubuntu.com/")

Web Browser: Firefox isn't that much of a beast, although I put my mozilla cache folder on a tmpfs to prevent it from running fsync() on the hard drive every time I change pages.
As for light alternatives... I have some friends who say good things about google chrome, maybe you could try that [as soon as it gets released.]("http://www.google.com/chrome/intl/en/linux.html")

Chat Client: There is a bug in pidgin, it has some serious memory leak problems, so if you leave it running for too long, it can quite easily fill a 512MB RAM slot. :o

What clients do you use to chat? If you are an IRC fan, then xchat will do just fine.

Music Player: VLC
Not Library Orientated, but it serves my needs.

You may be interested in trying out some CLI alternatives too:
[Everything can be done in CLI (mostly)]("http://spreadsheets.google.com/pub?key=pnJEQb-WFHaNvBnn9Mfdubg")

Regards
Iain
