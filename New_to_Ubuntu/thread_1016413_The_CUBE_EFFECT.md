### Post by jerome1232 on 2008-12-19
Okay, you'll want to install compizconfig-settings-manager:

First enable universe; System-Administration-Software Sources; check the box that says "universe" in it. Close, reload.

Then install the package in synaptic

System-Administration-Synaptic Package Manager
Search for compizconfig-settings-manager
Mark it for installation
Log out, then log back in.
System-Preferences-CompizConfig Settings Manager, and play to your hearts content.
