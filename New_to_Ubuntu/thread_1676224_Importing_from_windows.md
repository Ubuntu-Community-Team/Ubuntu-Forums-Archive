### Post by peculiar penguin on 2011-01-26
If you're keeping Windows, you should install the [Sync extension]("https://addons.mozilla.org/firefox/addon/firefox-sync/") (both in Windows' and Ubuntu's Firefox), this will keep your bookmarks, settings, etc. synchronized between both installs.

To do it manually go to Bookmarks > Manage Bookmarks > Import and Backup.
