> **vagelism said:**
> Hello to all!
I when I go to system settings of my pc it says that the drivers for the video card are unknown so it works in compatibility mode.

Ignore it. That particular program doesn't seem to work as it should and as far as I know, it reports all drivers as being "unknown".

The Additional Drivers program in Ubuntu, however, will tell you if there are better graphics drivers available that you're not using. If there aren't, then it will say "No proprietary drivers available".

### Post by Mark Phelps on 2011-11-30
According to Wiki, the Acer Travelmate 4070 uses the Intel 915 chipset -- which means there aren't any proprietary drivers available for it.  The only working drivers are installed with Ubuntu automatically.
