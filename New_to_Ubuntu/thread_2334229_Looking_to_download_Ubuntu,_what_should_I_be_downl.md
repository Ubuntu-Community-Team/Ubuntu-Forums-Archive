### Post by grahammechanical on 2016-08-17
You left one off the list. It is the most useful edition for ordinary computer users. Ubuntu Desktop.  Ubuntu Server does not have a Graphical User Interface (GUI). All work is done through the command line and all the default applications are useful for people who want a server OS.

Ubuntu Desktop comes with a GUI. It is called Unity. It is what used to be called a WIMP interface (Windows, Icons, Mouse, Pointer). Sound familiar to what you already use? Ubuntu Desktop comes with the usual default applications that a desktop user would find useful. The Ubuntu flavours differ from Ubuntu in having a different GUI and having default applications most suited to the particular desktop environment (DE) that they have.

Ubuntu.Com is for Ubuntu. The flavours have their own download web sites.

[http://xubuntu.org/](http://xubuntu.org/)

[http://lubuntu.net/](http://lubuntu.net/)

[http://www.kubuntu.org/](http://www.kubuntu.org/)

[https://ubuntugnome.org/](https://ubuntugnome.org/)

[https://ubuntu-mate.org/](https://ubuntu-mate.org/)[URL="https://ubuntu-mate.org/"]
[/URL]
[https://ubuntustudio.org/](https://ubuntustudio.org/)

[http://www.mythbuntu.org/](http://www.mythbuntu.org/)

[http://edubuntu.org/](http://edubuntu.org/)

Every two years at the end of April Ubuntu developers release a version of Ubuntu with 5 years support. They are called Long Term Support Releases (LTS). In between there are releases every six months but these versions only have 9 months support. The developers of the flavours follow the pattern but because of only having a small team of developers the LTS versions of the flavours are only supported for 3 years.

The LTS versions are 12.04, 14.04 & 16.04. Your hardware is compatible with any version of Ubuntu and the flavours. So, download a 16.04 version and test it out in a live session. All the default applications are present in a live session.

Regards.
