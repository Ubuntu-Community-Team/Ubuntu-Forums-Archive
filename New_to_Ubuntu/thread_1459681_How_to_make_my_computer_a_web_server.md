### Post by _0R10N on 2010-04-21
You have to do 2 things.

First: install apache and deploy your folder under /var/www (this location may vary)
Second: open an account in no-ip or in an similar site.

Note that if you're behind a router, you'll have to forward the apache port (tipically 80) to your PC. This is a simple router configuration...
