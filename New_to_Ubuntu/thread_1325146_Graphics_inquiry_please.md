### Post by dvlchd3 on 2009-11-13
A couple of good links for you:

[http://www.howtoforge.org/the-perfect-desktop-ubuntu-9.04](http://www.howtoforge.org/the-perfect-desktop-ubuntu-9.04)

This link is for 9.04, so a few modifications are necessary:
Step 5: 5 Configure Additional Repositories

Instead of:
> 
sudo wget [http://www.medibuntu.org/sources.list.d/jaunty.list](http://www.medibuntu.org/sources.list.d/jaunty.list) --output-document=/etc/apt/sources.list.d/medibuntu.list
Use this:
```
sudo wget http://www.medibuntu.org/sources.list.d/karmic.list --output-document=/etc/apt/sources.list.d/medibuntu.list
```
Everything else should be ok.  This gives you a lot of applications most people would use, as well as some restricted codecs to allow you to listen to all audio/video files.

Another project you may want to look at is Ubuntu Studio.  This project aims to please the Graphics/Picture/Video/Audio creating/authoring/editing/etc community.  They have some great tools which can all be installed after following the guide above.
[http://ubuntustudio.org/](http://ubuntustudio.org/)

Also, Ubuntu provides a GREAT help site. ([https://help.ubuntu.com](https://help.ubuntu.com))  Here you will find some great documentation on tweaking Ubuntu to your needs.

You may want to start here if you are migrating from Windows:
[https://help.ubuntu.com/community/SwitchingToUbuntu/FromWindows](https://help.ubuntu.com/community/SwitchingToUbuntu/FromWindows)

Otherwise there are some great guides here:
Troubleshooting:
[https://help.ubuntu.com/community/TroubleShootingGuide](https://help.ubuntu.com/community/TroubleShootingGuide)

Eye Candy:
[https://help.ubuntu.com/community/UbuntuEyeCandy](https://help.ubuntu.com/community/UbuntuEyeCandy)

Also, Ubuntu Geek has some great guides on getting some cool software to work in Ubuntu:
[http://www.ubuntugeek.com/](http://www.ubuntugeek.com/)

And of course, the forums are a great place to learn things as well.

Hope all this helps!
