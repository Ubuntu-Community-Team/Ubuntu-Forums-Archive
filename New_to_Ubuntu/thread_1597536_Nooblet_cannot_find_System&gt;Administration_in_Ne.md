### Post by empty_spaces on 2010-10-15
The directions in that post refer to 

System->Administration->Hardware Drivers
System>Administration>Synaptic Package Manager

So what you ultimately need are the "Hardware Drivers" and "Synaptic Package Manager" options.

In Meerkat, I believe "Hardware Drivers" is now called "additional Drivers" and "Synaptic Package Manager" should be the same.
I don't use netbook edition, but you should be able to search for both in the search bar on top.
