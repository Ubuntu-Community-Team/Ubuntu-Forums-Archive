Perhaps, the problem is you didn't install Ubuntu into the flashdrive properly.  Assuming you're on a Windows machine, try the following steps.

0. Format your USB.
1. Select your downloaded disk image. 
2. Install it into your USB.
3. Boot the USB. 
4. Select either Install Ubuntu or Try Ubuntu.
