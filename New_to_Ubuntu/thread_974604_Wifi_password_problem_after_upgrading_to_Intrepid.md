### Post by Mark_in_Hollywood on 2008-11-08
You can select the type of pass word from the list. Right click the network icon on the panel. Select Edit, then OK the access as you like, then highlight your modem/router and select edit. On the right of the word Security, you can pull down a menu. Select yours: WEP 40/128 or WEP 128 passphrase, or whatever you enter in Windows (I assume the same password/code/phrase here)

Mark the thread solved if this does it for you, please.
