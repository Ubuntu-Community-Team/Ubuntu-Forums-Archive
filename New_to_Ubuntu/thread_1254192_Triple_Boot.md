### Post by Liolikas on 2009-08-31
>is it possible to triple boot Kubuntu 9.04, Ubuntu 9.04 and Windows Vista.
Yes. You can continue this list as much as you want and until you have free space. 

Maybe minor problems will appear, but you can solve everything quickly if you know how to edit /boot/grub/menu.lst file or ask help here. If you provide good info about your partitions and what is inside them ppl here will write good menu.lst for you.

You can always run Kubuntu and Ubuntu off of the same install.  From an Kubuntu install go to the terminal and type sudo apt-get install ubuntu-desktop, from Ubuntu sudo apt get install kubuntu-kde4-desktop(or kubuntu-desktop if you want kde 3.5).  The only nuisance is that you will have some crossed over menu items which can be fixed rather easily.
