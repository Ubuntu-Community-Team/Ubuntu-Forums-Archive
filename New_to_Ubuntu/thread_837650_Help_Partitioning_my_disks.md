### Post by freebeer on 2008-06-22
My guess is that the hard drive is mounted.  You'll have to unmount it first before the partitioner will allow changes to it. (safety feature)

Had to run off to check/verify something.  On a previous screen you'll see a graphical bar-like representation of the hard drive and any partitions that it has (if you only have one partition, you'll only see one bar).  Right clicking on the bar will bring up a sub-menu with an option to unmount.  If you unmount it, you should then be able to access the options in that other screen.
