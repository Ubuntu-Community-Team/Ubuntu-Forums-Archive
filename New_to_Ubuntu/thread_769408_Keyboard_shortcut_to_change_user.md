### Post by Big Luke on 2009-05-26
Yes, it does work, I tried it.  All you need to do is hit the "Add" button and then enter what you want for the name and then enter put ```
gdmflexiserver
``` in for "Command", hit "Apply" and the option is created.  BUT you are not done yet, you still need to designate a keyboard shortcut for it, directly accross from the newly created option, click once on "Disabled" and then push the keys you want to be the shortcut.
