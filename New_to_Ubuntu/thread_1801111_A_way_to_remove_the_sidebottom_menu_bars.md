Thanks!Hi, no they can not be removed, you can set the left launcher to stay hidden, where it will only open with the super key or moving your cursor all the way to the top left of the screen. Here is how.
In the Unity plugin, set Reveal to Auto hide and the Reveal edge to None or Disabled. screenshot shows that the edges are disabled so that it will not open ,you can tell because all around the little window it is in red, if it was set to open when the mouse was moved to the edge it would be in green. You do have to have ccsm installed to change unity plugin settings.

Right click on the desktop, choose "Change Desktop Background", choose the "Behavior" tab, then set "Autohide Launcher" to On. The left Launcher will hide itself, and you need to push the left side to reveal it (which is a step up from 11.04, in which the Launcher would reveal if you simply touch the left side, which is an annoyance)
