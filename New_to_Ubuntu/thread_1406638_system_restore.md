### Post by skymera on 2010-02-14
I reinstalled Ubuntu no long ago.

I simply copied my /Home folder and pasted it on my memory stick.
Reinstalled Ubuntu and replaced my /Home folder with the one i copied 

Most of my settings werre restored.

So i suggest this if you need to reinstall this, others will suggest differently like having your /Home on a seperate partition entirely.
