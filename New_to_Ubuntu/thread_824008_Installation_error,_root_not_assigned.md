### Post by wormser on 2008-06-09
Is this happening when you are trying to install? Try the alternate Cd.  During the partition set up you have to select the / (root) partition.  It has been a while since I installed and cannot tell you exactly where to click.  During the manual partitioning you need to click a button.  It might be called advanced.  Then use the drop down to select / (root) on the partition you are installing.  I recommend formatting the drive as ext3.
