### Post by thane1 on 2011-07-12
Since you have 1TB of storage on the machine, one thing you might consider is, whether you may possibly want to also have windows on that machine in future.  If you have a copy of windows laying around, there's one snag with regards doing a dual boot (having the choice to boot either windows or linux at startup.)  Windows needs to be installed first prior to installing a linux sytem, for there to be a smooth startup choice.  If this will definitely only be a linux machine using a single boot procedure, go wild.  I've never looked back and only use windows as an absolute necessity, for a few windows only programs and a couple of websites, which only accept internet explorer (although with virtualbox I can get around that).  Cheers and welcome to the free world - enjoy!
