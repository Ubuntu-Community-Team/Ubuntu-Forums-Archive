### Post by mcduck on 2011-05-10
Dash won't find it because there isn't any menu entry for it by default.

Instead of using the Dash, press Alt-F2 to enter the Run dialog, and type "gconf-editor" in there, it should appear. :)

Edit: If you want Dash to find it, start "Main Menu" from the Dash (or from System Settings), then enable the "Configuration Editor" menu item under Applications/System Tools and log out & back again.
