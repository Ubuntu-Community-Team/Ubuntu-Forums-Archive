### Post by ankspo71 on 2009-09-19
Hi,
Have you tried this?

```
compiz --replace
```

I think that command will also work in the start up applications list (which I think is named differently in Kubuntu).

If KDE uses metacity (I'm not even sure if it does) you can type this to return it to normal.

```
metacity --replace
```

If one day you would like to use an emerald theme, you could try this:

```
emerald --replace
```

Those are the only commands I know for 3D effects/themes.
hope it helps,
James
