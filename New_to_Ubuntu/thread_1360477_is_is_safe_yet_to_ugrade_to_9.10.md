### Post by t0p on 2009-12-21
A question like that, "is it safe to upgrade?" is going to get such varied answers I can't really see the point of asking.  Some members will have had no issues and will tell you so; others will have had a pig of a time with Karmic and will not be shy telling you about that either.

I advise that you run 9.10 from the live cd for a bit, do the updates (even though they won't "stick" - that isn't important) then see if those updates have fixed your network problems.

If the problem's solved, then good - you can upgrade or reinstall as preferred.  If you still have no internet, post the problem in a new thread, explaining exactly what's wrong and what hardware this is affecting.  Then someone may be able to help.

Just asking "is it safe?" will do you as much good as it did the dentist in *Marathon Man* - if we don't fully understand the question, we can't give you a meaningful answer.

**EDIT** If the OP updates the network manager on a live cd, will he then be able to use the updated network manager?  Or would it need a "restart", which obviously is impossible in a live session?
