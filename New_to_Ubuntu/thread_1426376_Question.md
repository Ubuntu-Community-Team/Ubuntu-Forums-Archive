### Post by audiomick on 2010-03-10
I think that is a file that a program is using at the moment, and that they are only visible if you have selected "view hidden files". If you, for instance open an existing open office file you will see one appear that has the same name preceded by .~lock.
This file then disappears again when you stop working on the file.

### Post by ajgreeny on 2010-03-10
In that case it means "read only" and is in use by an application.  The tilde is also used as a shortcut indication of your home folder, as in ~/Documents, to save a lot of typing.

### Post by lisati on 2010-03-11
