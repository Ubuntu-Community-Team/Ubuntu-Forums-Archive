### Post by Frogs Hair on 2015-07-31
With some github projects there are contact links, but not all. If the project is also hosted on launch pad there is usually contact information and in the cases where there is none you have no choice but to seek help. Any .deb package should open in the software center. Packages you have to build sometimes include a read-me page with instructions.

### Post by steeldriver on 2015-07-31
FWIW I disagree that forum threads requesting software support are useless or bothersome
 
[LIST]
[*]If a community member is able to offer help, then not only do you benefit, but potentially others who search for the same issue do as well
