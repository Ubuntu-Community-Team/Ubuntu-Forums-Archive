### Post by Lisa Y on 2009-03-29
For it to show up in your "applications menu" go to

System> Pref> Main Menu

From there you can either place a check mark next to it...OR add it as a new item

### Post by ugm6hr on 2009-03-29
They generally appear in /usr/bin, although there is also /bin and /usr/sbin

What CD are you using to install?  Most .deb files will update the menu automatically.

If not, you can add applications manually in System -> Prefs -> Main Menu
