### Post by rosencrantz on 2011-03-13
Are you worried about performance issues? I'd think the bottleneck would still be the USB connection, so there'd be no big difference. From my experience, dumping a live image to USB JustWorks, and for everything beyond OS tryouts and rescue operations, I've always gone for a full HD install anyway.
