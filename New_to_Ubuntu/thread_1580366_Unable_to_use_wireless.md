### Post by owners4life5 on 2010-09-23
hello,

good to see another person move to the linux community (:

anyhow, connect to your friend's wired network, and do a software update

1)on top-left, click system
2)administration
3)update manager.

check for updates, then install all of them.

reboot if necessary

after possibly rebooting, (or after you install, if you didn.t HAVE to reboot)
1)system
2)administration
3)hardware drivers

if there are any needed, enable them. if there is more than one driver for each piece of hardware, select (enable) the one that has been tested by the dev.s...

let me know how things go.
