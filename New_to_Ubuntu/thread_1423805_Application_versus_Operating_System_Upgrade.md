### Post by ubu-fresh on 2010-03-07
I started with Xubuntu 8.04 LTS. It supported my hardware (Thinkpad T22) very well.
However, some software packages were not usable, e.g. Abiword marks words black (so you do not see the black text anymore). I tried Xubuntu 9.10. The new version of Abiword works well now, Evolution is better (allowing to produce a backup file) and Open Office has some advantages.

However,hardware support  for the T22 is much worse in 9.10: e.g. poor support of the trackpoint, memory management (swap) markedly worse).
And a windows legacy software (InfoSelect) for which I have not found a good replacement yet under Linux, has some new problems under 9.10.

Synaptic and Add/Remove Applications does not allow to install newer versions of Abiword, Evolution or Open Office. This strict link of Operating System and other software appears quite unbelievable to me: I would very much like to continue with 8.04 LTS as long as possible, since it supports my hardware much better, yet would like to install some newer versions of the software available.

**My question: **
How can I get at the versions of Abiword and Evolution contained in the packages for 9.10 and install them on 8.04 LTS? (PS: Please consider that I am a Linux beginner.)

### Post by relay_man on 2010-03-07
I've only been able to install and use an updated version of a particular package once.  It was a rather small application, and I use only the most simple parts of it, but it seems to work fine.

I'm not at all familiar with AbiWord but you can sure try it. 
You can get the most up to date version here:
[http://www.abisource.com/](http://www.abisource.com/)

And there is some good info and help here:
[http://abisource.com/wiki/Install_on_Ubuntu](http://abisource.com/wiki/Install_on_Ubuntu)


I believe you can add the repos from 9.10 to your list of software sources for 8.04 so that you could just use synaptic package manager to load the version that is available in 9.10. HOWEVER, I've personally never done this so if you haven't either, I suggest you search the forums for an explanation on exactly how this is accomplished.

If you decide to try the update, remember to back up any critical information or documents you may have before you attempt install.

Keep in mind that 10.04 LTS (Lucid Lynx) will be released soon and a more up to date version of AbiWord will likely be available.
