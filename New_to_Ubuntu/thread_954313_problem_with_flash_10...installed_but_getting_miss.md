### Post by eragon100 on 2008-10-20
in a terminal, do: sudo apt-get install flashplugin-nonfree

This will install flash player 9, but that version is still good enough for all websites. When ubuntu 8.10 "intrepid ibex" comes out 10 days from now, that command will install flash player 10 for you :wink:
