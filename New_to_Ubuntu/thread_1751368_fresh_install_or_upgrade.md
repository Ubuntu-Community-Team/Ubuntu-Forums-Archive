### Post by fballem on 2011-05-06
I always backup my data (home directory, minus the desktop folder), do a backup of my evolution files (using the options in evolution) and backup my FireFox bookmarks.

I then do a fresh install. I know that upgrades are supposed to be painless, but they rarely are. The only advantage that I can see with doing an upgrade is that your settings will be preserved. Some programs are sufficiently changed that the old settings might not work as well.

I've been using ubuntu for nearly three years now, and I have used each version as it's come out. This version, 11.04, includes Unity, which has engendered strong opinions on both sides - a lot of people like it, a lot of people hate it. Before you decide, you may want to peruse some of the posts on Unity to get a feel for it.

Hope this helps,
