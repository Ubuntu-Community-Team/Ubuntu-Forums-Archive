### Post by jeromey on 2008-11-19
to configure compiz fusion you will need to install CCSM (compiz config settings manager)

go to "applications-add/remove" then set it to search for "all available applications"

search for ccsm

after installing it should be in the menu at system-preferences-compizconfig settings manager
