### Post by Wim Sturkenboom on 2011-08-09
I'm using OOo 3.2.0 (default in 10.04) and it just works with <ctrl><enter>; tried in a fresh workbook

> 
Writing Multi-line Text 
1.Pressing the **Ctrl+Enter** keys inserts a manual line break. [COLOR="Red"]This shortcut only works directly in the cell, not in the input line[/COLOR].
If you want the text to automatically break at the right border of the cell, proceed as follows:
1.Select all the cells where you want the text to break at the right border.
2.In Format - Cells - Alignment, mark the Wrap text automatically option and click OK.
