### Post by mr_luksom on 2011-05-11
When you say you installed GNOME, what package did you install?

If you just run:

```

sudo apt-get remove <insert package name here>

```

It should go back to normal.

If you want to use GNOME, you can select 'Classic GNOME' from the login menu.
