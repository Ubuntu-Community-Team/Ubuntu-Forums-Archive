### Post by rafac24 on 2009-03-08
Ubuntu has a 'Adjust Date & Time' when you right click the clock.

Have you tried:

```

```

See if that lets you.

### Post by rafac24 on 2009-03-08
Or:

You can start Time Administration Tool in the following ways:
System menu
    Choose Administration &#9656; Time and Date.
	   
Context menu in the clock applet
    Choose Adjust Date and Time.
