### Post by mcduck on 2010-10-26
If you want a dedicated tool for the task, EasyTAG is a great application for managing the tags in audio files, and also handles file naming and moving files around based on tags.

Takes a bit to figure out, but after that it's probably the best tool around for this kind of purposes.

[http://easytag.sourceforge.net/]("http://easytag.sourceforge.net/")
(it's in the repositories, the link is just for more info :))
