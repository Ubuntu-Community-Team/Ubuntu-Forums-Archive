### Post by Joebewan on 2011-08-10
I think previous posts have covered pretty much all your questions in one form or another, so I'm going to just give you an example of what I would do in your situation;

Start with the latest LTS release, which is 10.04.3 (don't quote me on the ".3" part, but I think that's right)  LTS means Long Term Support and you might feel more comfortable with it, like I do.

Boot to the live CD and make sure everything works for you.  Wireless is a frequent annoyance (based on posts I've read) so it may take some tweaking.  My experience has been that pretty much everything else will just work.  I rarely run into hardware issues.  Maybe I'm just lucky ???

If you decide you want to take the plunge, clean up your Vista install, defrag it, and shrink the partition using gparted on the Ubuntu live cd.  After you shrink, you'll need to boot back into Windows and let it run chkdisk, maybe more than once.  Your disk hasn't been harmed, it's just that you've changed it and Windows flags it as "dirty" because of the change.

Install Ubuntu on the freed space and enjoy.  Dual boot lets you drop back to Windows in the event you want to or need to and you can ultimately revert back to Windows exclusively, or move forward to Ubuntu exclusively, depending on what works best for you.

Someone mentioned Virtual Machines, and I want to second that.  Download virtualbox from virtualbox.org and install it.  It's actually very intuitive and easy to set up.  Once it's installed, you can set up Vista in a VM or any other Windows you have a license for.  That may get you closer to moving to Ubuntu as your main OS.

Above all, have fun.  If you have problems along the way, you'll probably find a solution here or elsewhere with just a little effort and your favorite search engine.
