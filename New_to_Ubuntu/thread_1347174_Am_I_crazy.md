### Post by The_Pirate_King on 2009-12-05
DUAL BOOTING UBUNTU: AN EPIC SAGA OF INSANITY

I started using Ubuntu a couple months ago and jumped into it with both feet... which may have been a mistake. It has been what most sane computer users would describe as a total nightmare. 

1: installed WUBI under the guidance of some guy I met online. 
1.5: Break firefox profiles by [attempting to mount it to the RAM.  ]("http://crunchbanglinux.org/forums/topic/3629/mount-firefox-profile-in-ram/")Spend hours trying to fix it.  College GPA suffers.  Allow drunken hacker friend to SSH into my system and try to fix it.  System goes down for restart and desktop has been changed to 4chan face [[this fellow]]("http://i26.photobucket.com/albums/c110/themadblimper/smileyfaec.jpg"). 
2: upgraded from Vista to 7 [WUBI still boots somehow, don't ask me how]
3: decided to make a partition and install Ubuntu permanently.  can't remove WUBI because it is not in the list of add/remove applications. 
3.5: Spend hours trying to find a way to get all my fragmented files and system files to the beginning of the disk so that I can create a partition.  GPA suffers
4:  Windows no longer boots; some sort of conflict with GRUB, WUBI bootloader and Windows bootloader. 
5: Reinstalled windows.  Can no longer boot Ubuntu.  
6: Don't have a live CD because I installed via flash drive.  Flash drive no longer works.  Spend hours trying to find a program to help me reinstall GRUB.  GPA suffers. 
7: Give up and decide to reinstall Ubuntu.  Accidentally overwrite Windows boot partition with Linux Swap.  Can no longer boot windows. 
8: Give up on windows for a while.  Experiment with Ubuntu.  
9: Spend hours over Thanksgiving break attempting to crack into my neighbor's wireless network.  Was never successful but made me feel like a hacker. 
9.5: Create a live CD because I am fed up with trying to use a flash drive. 
10: Break X.org attempting to install a new graphics driver. 
11: Cousin tries to help me fix X.org.  Suggests reinstall.  
12: Reinstall Ubuntu to find that the mouse doesn't work.  Spend a few minutes trying to fix the mouse using only the keyboard and eventually give up.  
13: Reinstall again. 
14: Break X.org trying to install a graphics driver again.  Decide to try to fix it this time.  
15: Install finch to communicate from terminal.  Manage to fix Xorg and find my screen resolution is down.  Discover that my graphics card is no longer supported for linux distributions. 
16: Reinstall windows.  
17. Use live CD to install GRUB and boot Ubuntu.  
18: Configure GRUB so that it can boot Windows. 
19: Actually have a working dual boot of Windows 7 and Ubuntu Karmic.  Rejoice. 
20: Still have not found working driver for graphics card on Ubuntu.  Pleased nonetheless. 
21: Post this thread [and another detailing my graphics card problem]

So, all in all, I have done eight operating system installations in the past few months. 

Now, I don't know much about normal people, and I'm certainly not one of them, but I would think that a normal person, at this point, would throw their live CD out the window, delete their Ubuntu partition and never touch a linux-based OS again.  And yet, I have not done this.  In fact, I am planning to reinstall Ubuntu to fix my graphics driver issues and possible install a "real linux" OS such as Slackware in yet another partition. 
When it comes right down to it, for some reason, I still really like Ubuntu [or at least linux].  Even though I have NEVER had these problems with my windows install, NEVER had to reinstall it once. 

So, my question is, am I crazy? 
Can anybody relate?



UPDATE: THE FINAL CHAPTER IN THE EPIC SAGA OF UBUNTU ON A TOSHI SATELLITE!!! 
I finally seemed to have everything configured the way I wanted it.  I had a working dual boot, I found some really nice programs for both Windows and Ubuntu, and although I might have needed to reinstalled Ubuntu in order to get my graphics card working properly again, I was experiencing very few issues. 
When all of the sudden, out of nowhere...
```
No bootable device detected.  Insert boot CD and press any key to continue.
```

Hard drive failure.  Disc making a noise like a dying frog.  Things are looking pretty grim.  Might have to get a new laptop or at least a new HDD...
