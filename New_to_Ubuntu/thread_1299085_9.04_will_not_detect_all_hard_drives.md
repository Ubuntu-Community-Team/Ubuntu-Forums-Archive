### Post by swerdna on 2009-10-24
You can test that partition like this:
[LIST=1]
[*]make a folder to test it in:```
sudo mkdir /mnt/sda9
```
[*]test mount it:```
ntfs-3g /dev/sda9 /mnt/sda9
```
[/LIST]
There might be an error about the dirty bit being set. Were there any error messages?
