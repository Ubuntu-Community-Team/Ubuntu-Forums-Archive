### Post by wingkitf on 2010-12-05
There is a linux command [COLOR="DarkRed"]**dbus-launch**[/COLOR] which might be helpful to sort this out.

I use the followings:

**$[COLOR="DarkRed"] gksu dbus-launch nautilus[/COLOR]**

In Nautilus window opened by the command above, type in the address bar the SMB path with the actual IP address of Windows share like 

**smb://x.x.x.x**

You'll be asked to enter username, domain and password for authentication. 

It works perfectly for me!
