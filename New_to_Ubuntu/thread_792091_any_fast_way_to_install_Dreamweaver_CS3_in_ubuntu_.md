### Post by nixeagle on 2008-05-12
I doubt any of those packages will be in the default ubuntu. The reason being is those packages are not free. You have to pay money for a dreamweaver license I believe. That is not compatible with Ubuntu and the free software world's ideals.

So while its possible to run it in wine, as part of wine's attempt to make windows programs work on linux, I doubt there will be a linux specific version until and if macromedia chooses to release software specifically for linux. 

If you want easier installs, direct requests to the maker of dreamweaver, adobe. Here is how you contact them: [http://www.adobe.com/aboutadobe/contact.html?ogn=EN_US-gntray_comp_contact_adobe](http://www.adobe.com/aboutadobe/contact.html?ogn=EN_US-gntray_comp_contact_adobe)

Tell them that you would like to see a linux version of their software, (dreamweaver, photoshop, flash, whatever it is they make that you want). Let them know there is demand for it.
