---

### Post by Frogs Hair on 2021-03-13
Open the file manager, right click on the folder you would like to change. From the the context menu select properties and click folder icon. From there navigate to the directory where new folder icon you would like to use is stored, select the icon and select open. To revert go through the same process and use the revert option on the dialog box. The process may differ slightly depending on what file manager and flavor is in use.
