What package manager are you using?

First go to system -> administration -> software sources.

Make sure universe and multiverse repo's are enabled.

In the third party software tab, make sure the partner repo is enabled.

--

Reload (you'll be prompted for it)

--

In applications -> add/remove, search for Thunderbird, tag the box and press apply

(make sure all availabe applications are enabled in the top center drop down menu)

--

I don't know if vmware is in the repos, you can use virtualbox instead.
