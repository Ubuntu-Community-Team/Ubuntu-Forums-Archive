### Post by grahammechanical on 2013-10-10
You have a very, very old version of Ubuntu that is no longer supported. Linux has moved on from kernel 2.6 up to kernel 3.0 and beyond. I have no idea what version of Ubuntu you are using. You do not tell us. Nor, do you tell us the specifications of your hardware. This would be useful information.

Each Ubuntu version has a list of internet addresses that it uses to find the software packages. These addresses are called Software Sources or Repositories. Each version of Ubuntu has its own repositories which are longer maintained when that version of Ubuntu reaches End of Life. This version of Ubuntu could be trying to access server locations that are no longer existing.

Does this machine have a network to connect to? Is it connected to a router/hub which in turn is connected to an Internet Service Provider?

You need to give more information about your system and what you want to do.
