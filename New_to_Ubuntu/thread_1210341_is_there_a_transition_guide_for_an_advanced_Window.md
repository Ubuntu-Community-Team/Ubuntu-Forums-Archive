### Post by prvteprts on 2009-07-11
You could try reading Beginning Ubuntu Linux: from novice to professional by Keir Thomas. I checked it out recently and it seems to be an easy to read guide for a transitioning user. ### Post by kwacka on 2009-07-19
One more 'under the hood' book is 'Rute User's Tutorial and Exposition', its in the repositories so just ```
sudo apt-get install rutebook
```

Alternatively, its online at [http://rute.2038bug.com/index.html.gz](http://rute.2038bug.com/index.html.gz)

An analogy - a few years ago I took a car driving lessons (I'd been riding a bike for over 30 years) and found that it was totally different in just about every way. My instructor had problems because I was trying to drive a car like an experienced road user, not as a learner.

Its the same with Windows & Linux; Linux is not a different version of Windows, its a whole new game. There are applications that run in linux that do the same (or a simialr) job as apps that run under Windows but, for example, there's no equivalent to 'Drive C:'
