### Post by souravc83 on 2012-05-10
You don't have to worry about the wine stuff if you install "PlayonLinux"
Wine is a great program, but is always very difficult to deal with, even for pros. Playonlinux makes things simpler by installing wine, other dependencies (some .Net thing or whatever), automatically for each program. Its far more user friendly.
However, Windows programs running on linux will always have issues.
Firefox through wine/playonlinux does not make sense, unless there is something specific you want which only the windows version will do.
Run firefox natively, it will be much much better.

### Post by souravc83 on 2012-05-10
Virtualbox is a software which will let you run an operating system inside it, just like a program.
So, you can run windows in virtuabox, just like you are running a software, like firefox or google chrome.
Virtualbox is in the Ubuntu Software center.
Open up the software center, type "virtualbox" in the search bar, and you will find it.
You can install an Windows system there, if you want.

### Post by Baldrick_NZ on 2012-05-10
Hi James New, and welcome to the forums...

May I suggest you play with and familiarise yourself with Ubuntu and it's software centre before delving into windows-based programmes on Ubuntu. 

Best to use a native Windows OS for games (and other proprietary Windows-based software like iTunes) in the meantime.

I've heard it said that Linux in general is quite a learning curve when you've been using Windows operating systems. There is some truth to that. After you've found your way around for a while and feel confident, then would be the time to consider wine, play on linux or virtualbox. 

Have fun!
