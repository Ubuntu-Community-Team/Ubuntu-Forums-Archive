### Post by sanderd17 on 2011-05-16
Why would you edit those with libreoffice?

XML files are data-files. So you can use them in a database, or edit details in a text editor or use it in a program that supports a specific kind of XML file. Like you can have JOSM to edit OSM files, which are XML files with geographical data in a certain format. But there is no good reason to edit them with openoffice.
