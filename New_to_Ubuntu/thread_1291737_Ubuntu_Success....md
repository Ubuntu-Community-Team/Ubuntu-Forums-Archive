### Post by msimon1960 on 2009-10-15
Bravo!

I've loaded Jaunty on two machines now -- both are up and running without too much effort.  Most of the basic hardware compatibility issues I've encountered in versions since Dapper appear to be resolved.

I can access my home server and the internet.  Most of the functionality of Microsoft Office is available in OpenOffice.

- Samba and networking in general still needs work.  WAY to much tweaking needed to get basic networking functioning.

- Needs a decent database program.  Opens Office Base ain't quite there yet as a Rapid Application Development product.  So close I can almost taste it...think Foxpro/dBase frontend on a MySql engine.

I'm running primarily on Linux now -- Windows on dual boot for those few things that Ubuntu/apps can't quite do yet.  Looking forward to buying some quality linux apps.  In my opinion the OS should be open source but I'm happy to pay for quality apps.

Credit where credit is due...

Signed an increasingly satisfied customer,
