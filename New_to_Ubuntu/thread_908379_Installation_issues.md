### Post by mashimaro on 2008-09-02
If it stops at the point you specified it probably is a cd burning problem. Try burning the live cd at 4x and that may solve the problem. It stopped maybe because your cd drive can't read the burn if it's burned at higher speeds. If that doesn't, you can use the alternate text installer cd located at [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)
and check the following:
> Check here if you need the alternate desktop CD. This CD does not include the Live CD, instead it uses a text-based installer.
