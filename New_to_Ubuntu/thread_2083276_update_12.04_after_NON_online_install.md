### Post by deadflowr on 2012-11-12
If you understand the layout of Ubuntu, open up the dash and type update manager, click enter.
When the update manager opens click the check option first, this will fresh the repositories, then if all is well click install.

If you don't understand the layout, the dash is Ubuntu's menu system, and to open it either press the windows(aka super) key, or click on the top most icon in the leftside panel(known as the launcher), this is the dash.

### Post by Bartender on 2012-11-12
Just a quick comment - although the installer seems to encourage you to get online before installing, IMO there's absolutely nothing wrong with installing offline.  

Some folks (myself included) have had an install fail when connected, only to try again offline and install successfully.  Nowadays my default is to install offline, then get updates, install restricted-extras, etc.

If you install offline, then simply use the PC for a day or two, I guarantee Update Manager will pop up and bug you to update before long...

### Post by grahammechanical on 2012-11-12
It should also be noted that in 12.04 there is an option in the power/cog menu called Software Updater. That will also run the updates for you.

In 12.10 updating is part of the Details page of About this Computer. There is a panel that says either Checking for Updates or System Up to Date. Then we may get the option to Install Updates.
