### Post by erginemr on 2008-07-10
If importing from a "*.wab" file is your main concern, then I can suggest exporting from inside OExpress to another format such as "*.csv" or "*.txt".

Address books are nowadays mostly used in conjunction with e-mail readers (evolution, Thunderbird, Kontact, Claws Mail, etc.), so stand-alone address book software are few and far between.

There is an entry in UbuntuGeek for such an application however:
[http://www.ubuntugeek.com/fast-address-book-for-your-desktop.html](http://www.ubuntugeek.com/fast-address-book-for-your-desktop.html)

which, provided that it is in the repositories, should be installed easily with:
```
sudo apt-get install gfa
```
