### Post by Old_Grey_Wolf on 2011-01-20
You could install Real VNC **Free** on you XP machine. [http://www.realvnc.com/products/free/4.1/index.html](http://www.realvnc.com/products/free/4.1/index.html)

There is also a **Free** UltraVNC that you can install. [http://www.uvnc.com/](http://www.uvnc.com/)

Then enable remote desktop on your Ubuntu machine via "System > Preferences > Remote Desktop".
