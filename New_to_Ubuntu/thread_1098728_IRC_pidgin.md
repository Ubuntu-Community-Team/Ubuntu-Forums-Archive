### Post by saisai on 2009-03-17
I just learnt that you don't need an IRC account to participate in the chats. All you need is a nick name, and you can start chatting. So to create a nickname go to:
Manage Account -> IRC -> and enter a nickname but leave the password field blank, then you are ready to start chatting. To use the Ubuntu chatroom go to Buddies->Join a chat-> under channel type #ubuntu
