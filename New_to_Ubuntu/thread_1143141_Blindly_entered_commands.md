### Post by Niniel on 2009-04-30
There is no need to use GPartEd beforehand.
You will have ample opportunity to partition during the installation (for maximum control, use the "alternate" installation CD).
If a VM installation of WinXP is enough depends on what programs you want to run. Some don't do so well in VMs - e.g. if they need direct access to hardware or better hardware than the VM is able to emulate.
