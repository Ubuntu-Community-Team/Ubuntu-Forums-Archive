### Post by toddward on 2009-01-21
Since it's USB, it should be recognized when the system is running without issue.  

I'd suggest starting up the computer, then press and hold the power button until the computer turns off (around 4 seconds).  Then, before powering up, disconnect the USB keyboard.  Restart.  Once you're back up into an viewable environment, plug the USB keyboard back in.
