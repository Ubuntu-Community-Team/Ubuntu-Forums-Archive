### Post by bwang on 2008-11-21
1. 512 MB will work, but doesn't hurt to have more. I've used 800 MB out of 2 GB before
2.Yes. You have enough space to dual boot
3.The existing cards will work. If you buy a card, get an nVidia. I've seen numerous posts about ATI issues.
4.If you're using windows right now and want to dual boot, the windows apps will not be overwritten; however, you will not be able to use them under Linux.
