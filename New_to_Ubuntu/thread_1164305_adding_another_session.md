### Post by Didius Falco on 2009-05-20
Hi,

To add a new user, go to **System/Administration/Users and Groups**. You'll need to unlock it with your admin password then you can set up the new user account and privileges.

KDE and XFCE are both alternate desktops. The best way to find out about them is to go to the source and read up on them:

[http://www.kde.org/](http://www.kde.org/)

[http://www.xfce.org/](http://www.xfce.org/)

Here is a guide to installing each of them - they are available through Synaptic. Scroll down to the "Playing Around" section and you'll find a lot of useful information about KDE, XFCE and a comparison betwen Gnome and KDE. This is a good site to bookmark...

[http://www.psychocats.net/ubuntu/index](http://www.psychocats.net/ubuntu/index)

I hope this helps...

Regards,

Didius
