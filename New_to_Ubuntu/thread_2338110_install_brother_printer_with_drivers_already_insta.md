### Post by ajgreeny on 2016-09-24
If you are using Ubuntu with the unity desktop click on the top icon in the launcher (the dash) and type "printing".
An icon, or icons, should show so click on the Printing icon and follow the dialogue to add a printer.  If you already have installed the drivers it ought to be easy.
