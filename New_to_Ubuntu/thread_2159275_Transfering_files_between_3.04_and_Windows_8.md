### Post by wc711 on 2013-07-02
I am having great success with 13.04 installed in a separate partition.  Everything works fine, but I have learned a few things I thought I would pass on.  All operating systems appear on the launch side bar making it convenient to check files in my Windows 8, but I found that transferring files, especially music files directly from Windows 8 to Ubuntu can create problems.  I did this and ended up with video files becoming mp3 files and having Carbonite informing me that the backup has been turned off because of some "errors".  They didn't say what they were, but I'm fairly sure it had to do with the transfer issues.  Carbonite seems to have overcome the problem on their end now.  The files that were changed to mp3 are apparently permenant so I will reboot in Windows 8 and transfer the  copies of the originals to my SD card then reboot and install them in the appropriate Ubuntu folders.  Being a newbie, I'm not sure if "mounting" the Windows 8 drive would accomplish the same thing as transfering to and from my SD card, but until I find out, I will take the extra time to make the transfers.  I have learned that there are things you can do with  Windows 8 with media files that you can't do with Ubuntu like iTunes, but I found the Guayedeque music software in Ubuntu that does a pretty good job with my music files.  The interesting thing is, the reason I'm able to transfer files from an SD card is because Ubuntu recognizes it; Windows 8 doesn't read the SD card in the SD slot.  I had to buy a usb card reader to gain access to the card.  I have some great home made videos from my travels that Window 8 won't read, but my Ubuntu has no problem using Movie Player.  It's great.  Over all, There are more pluses on the Ubuntu side than The Windows 8 side. Using the Wine program loader software I was able to load some of my old great games like Flight Sim 98 with success.  It turns out though, the my Nvidia graphics card just doesn't cut it with these games.  It's still progress because Windows 8 didn't comes close. With a better graphics card, I can load games I've had for years and enjoy them with Ubuntu.
