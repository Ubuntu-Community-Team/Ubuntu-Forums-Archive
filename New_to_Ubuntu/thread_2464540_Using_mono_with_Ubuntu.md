### Post by tea for one on 2021-07-04
Ubuntu 16.04 has reached end of life and is unsupported unless [https://ubuntu.com/blog/ubuntu-16-04-lts-transitions-to-extended-security-maintenance-esm](https://ubuntu.com/blog/ubuntu-16-04-lts-transitions-to-extended-security-maintenance-esm)

The best advice is:-

Back up your data
Download 20.04 LTS ISO
Make a USB bootable drive
Try Ubuntu 20.04
Install if you are happy
Restore back-up
