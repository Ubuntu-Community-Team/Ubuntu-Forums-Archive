### Post by lykeion on 2010-09-16
I haven't heard of any CLI tools for this.
You could write your own web scraping bash script to fetch lyrics from a site using wget, grep, gawk and sed commands.
See this: _[Linux.com - Extract data from the Internet with Web scraping]("http://www.linux.com/archive/feed/52837")_

Amaroks seems to be using a JavaScript plugin to fetch the lyrics.
Some info here:
_[Amarok scripts available for download]("http://amarok.kde.org/wiki/Scripts")_ 
_[Ultimate Lyrics Amarok 2.0 Script]("http://kde-apps.org/content/show.php/Ultimate+Lyrics?content=108967")_
