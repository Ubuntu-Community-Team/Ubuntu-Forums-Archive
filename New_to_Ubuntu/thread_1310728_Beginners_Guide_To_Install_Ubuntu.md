### Post by coldzero24 on 2009-11-02
i like to share with you my steps to install ubuntu, I'm a windows user and i didn't know anything about Linux until a few months a go when i needed an operating system for old PC, and what a suprise when i found about linux and specially Tinyme distro it was very fast on 128 MB ram i just couldn't belive it.

Now i have both ubuntu and windows on my new pc, i use ubuntu all the time :D, and because it was a problem for a beginner to install linux without errors i learned my way the hard way.

here i like to share resources and tools i used and still use to fix any problem:

1- ultimate boot cd - UBCD [http://www.ultimatebootcd.com/](http://www.ultimatebootcd.com/)
 you sould download and burn it to CD, keep it in safe place becasue you will need it, it helped and saved me from disasters.

2 MBRfix.exe - [http://www.sysint.no/nedlasting/mbrfix.htm](http://www.sysint.no/nedlasting/mbrfix.htm)
this small exe file will restore your hardidsk MBR if you made any error 
works on windows , You need administrative privileges to make it work!

> 
MBR=master boot record
read about it here [http://en.wikipedia.org/wiki/Master_boot_record](http://en.wikipedia.org/wiki/Master_boot_record)
3- Unetbootin - [http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)
this is very good tool use to create bootable Live USB drives or( install to harddisk ) for a variety of Linux distributions from Windows or Linux, without requiring you to burn a  CD

4- Wubi, [http://wubi-installer.org/](http://wubi-installer.org/)
the best and most safest way to install ubuntu :), you can install ubuntu inside windows as easy as setup normal windows software, no partintion needed and you harddisk is safe. i highly recommend you use this first before you try to install to harddisk.

**Just a few tips what you do if you face a problem:**

1- if you have a problem with grub, insert the ultimate boot cd restart your pc and goto filesystem tools > boot managers > super grub disk

2- if you want to remove grub and restore MBR you can use MBRtool but you must already have created backup before. otherwise you can use super grub disk and boot to windows and use MBRfix.exe
