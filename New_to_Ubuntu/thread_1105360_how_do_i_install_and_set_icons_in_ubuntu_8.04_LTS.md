### Post by UbuntuNerd on 2009-03-24
if you want to install icons go to the System > preferences > appearance, when the menu opens make sure theme is selected and drag the icon theme to the box it should install itself.
To change icons in the Appearance preference box make sure "theme" is selected and click on customize > icons and choose one
