### Post by easybake on 2009-05-03
Some solid options would be to use ftp(sftp) or ssh if you want to connect through a network to another computer. 

Ssh can be used to execute commands and such on another computer. While sftp can be used to grab files from another computer. 

Ubuntu comes loaded with both ssh and sftp but windows requires openssh to be installed. You can read a tutorial on how to set it up [here]("http://www.bmonday.com/articles/653.aspx")
