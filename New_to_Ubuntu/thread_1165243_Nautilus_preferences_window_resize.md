### Post by mcduck on 2009-05-20
I assume you wish to make it smaller to fit in the screen, not make it bigger to use _all_ screen space? In that case the answer is no, it only scales to minimum size that's able to contain all the widgets.

If you have problems accessing some settings just hold the Alt key down and you can move the window from any point inside the window (not just titlebar like usually).

Alt+left button moves window
Alt+middle button resizes window
