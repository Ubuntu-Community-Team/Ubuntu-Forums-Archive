### Post by jmszr on 2009-11-05
chris a,

The only real reasons to run an AV in Ubuntu Desktop are if you are in a LAN with Windows machines, if you are downloading and sending files to a Windows machine, or if you are using WINE (and even then any infection should be limited to your WINE installation). 

You should find this a useful primer on Ubuntu Security: [http://ubuntuforums.org/showthread.php?t=510812](http://ubuntuforums.org/showthread.php?t=510812)

### Post by Dude-PWB- on 2009-11-05
I have KlamAV installed, but like other have mentioned, it's not for my Ubuntu/Linux installs.

I use it only for scanning e-mails, so that if someone happens to send me a virus, I don't forward it on to someone else. Or, if someone has a virus infected windows install, I can hook the drive up to my PC and scan it from Linux.

For normal everyday use of Ubuntu, a virus scanner is not required.
