### Post by roger_1960 on 2011-11-21
Hi

To change the icon order, clickand hold and drag the icon to the right (out of the launcher), move it up or down and place it back in the launcher where you want it and release.

Not sure about the second point, the dash "button" has been put in the launcher in 11.10.  I use the windows key - its really worth looking at the keyboard shortcuts in Unity.  Have a look at this site (its linked in a sticky in the general help forum)
 [http://castrojo.tumblr.com/post/4795149014/the-power-users-guide-to-unity](http://castrojo.tumblr.com/post/4795149014/the-power-users-guide-to-unity)

To set the launcher to reveal with the mouse at top left:

Install compizconfig-settings-manager (or CCSM), run it and select "Unity Plugin".  Then set the "reveal mode" to top left.  Personally I use bottom left so that it doesn't conflict with the back arrow in Firefox.
