### Post by Yeti can't ski on 2008-09-10
As I first switched to Hardy Heron, I was a having a very similar (perhaps the same) problem. 

Downloads and video streaming were both fine, but I was having a 15 seconds delay in accessing any pages not previously stored in cache. 

Disabling ipv6 in Firefox solved the problem for me. I believe the issue is that my wirelles modem/router is still not compatible with ipv6. 

I suggest that you update your router's firmware or disable ipv6. 

[http://ubuntuforums.org/showthread.php?t=6841](http://ubuntuforums.org/showthread.php?t=6841)  

One other thing, if you are try to set up you network configuration don't really trust the graphic user interface. This seems to be one the last weak points of Ubuntu's GUI. It is harder but better to use command lines. 

Good luck and let us know if it works.
