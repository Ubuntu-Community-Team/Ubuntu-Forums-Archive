### Post by 23dornot23d on 2012-02-09
can always use ksnapshot and put them where you want ....

do this in a terminal .... or look in the software center ......
[B]
sudo apt-get install ksnapshot[/B]

when you save the screenshot .... it asks where to put them and what type you 
would like to save them as ....... jpg ( is small and good for the web )


use imageshack to save them online so your friend can get them .... its free and you can easily browse your computer for them .....

[http://imageshack.us/content_round.php?page=done&l=img815/8062/snapshot12v.jpg](http://imageshack.us/content_round.php?page=done&l=img815/8062/snapshot12v.jpg)

### Post by bluexrider on 2012-02-09
Did you name the file that you tried to save? As in if the default extension is .jpg the portion before the (.) needs to have something XXXX.jpg or else it will become a .HIDDEN file.

You can check that in nautilus by opening your default folder and using the Ctrl+h key to revel the hidden file.
