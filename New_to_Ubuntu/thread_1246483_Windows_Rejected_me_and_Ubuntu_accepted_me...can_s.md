### Post by donkyhotay on 2009-08-21
An install of windows on one computer won't work on another because of the licensing involved with microsoft. Ubuntu license doesn't care so it's a non-issue and transfers. If it's only windows with the problem then it's a licensing issue and there isn't anything you can really do about that, if it's both windows & ubuntu on one specific computer (which is what I think the problem is) then it's with the HW on the computer having the problem and it's time to get that computer replaced.
