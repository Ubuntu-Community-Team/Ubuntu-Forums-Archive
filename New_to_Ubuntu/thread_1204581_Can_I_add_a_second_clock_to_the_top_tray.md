### Post by diablo75 on 2009-07-04
There's a better way to do what you're wanting.

LEFT-click on the clock you already have.  It will expand out.  You'll see a Locations > thing that can be expanded with an edit button next to it.  Click on the edit button and use the dialog to Add a new clock to the current pop-out.
