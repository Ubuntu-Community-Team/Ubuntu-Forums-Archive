### Post by laidback on 2009-12-20
You probably have your system set to boot straight into the default user. Goto System>Admin>Login Window  ..You'll have to give your password. Then choose the Security tab where you should find the first option..Enable Automatic login ... is ticked. Take the tick off, reboot. Now you should get the login screen.

This is how it works with v9.04, you may need to search to find the equivalent in 9.10.
