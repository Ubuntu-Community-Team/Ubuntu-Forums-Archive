You go to the software center, find the software you want removed and click the remove button.

or


### Post by papibe on 2011-11-19
Hi Mikesla.

The easiest way is to use the 'Software Center', search for the program, selected and press remove.

If you are familiar with the terminal you can also remove a package like this:
```
sudo apt-get remove package_name
```
Hope it helps.
