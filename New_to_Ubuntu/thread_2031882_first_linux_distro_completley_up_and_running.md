### Post by techvish81 on 2012-07-23
+1 for ubuntu restricted extras it will cater most of your media playback requirements. 
you can go to ubuntu software centre and browse thousands of softwares to install what you like. 

if there is any problem, don't hesitate to ask here, the community here is the best in linux world. one more thing to keep in mind while using any of the linux distro is that most of the distro and software packages are open source and developed by volunteers,  so we should have some respect for their enormous efforts. if something breaks in your OS,  you get a chance to learn new things. if you prefer stability over leading edge software technology,  keep the 12.04 LTS and don't upgrade to newer versions every six months.  LTS is now supported for 5 years,  so you will get security and other updates till expiry.
