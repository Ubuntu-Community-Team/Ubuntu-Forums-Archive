### Post by Andrew_Hynes on 2014-01-20
I have zorin installed and a dualboot OS on my Windows XP system. I want to replace zorion with xubuntu. How do I go about this? Can I over write zorin with xubuntu in the existing partition?

---

### Post by robin7 on 2014-01-20
Yes you can!  When the installer brings up the disk partitioner, choose "Something Else" and install Xubuntu to the existing Zorin partition (choose "format this partition").  If you have a separate /home or /data partition, don't format it.

Best to do a backup of everything first just in case!
