### Post by dearingj on 2009-10-16
A couple things you'll want to try:
1) Check the md5sum hash of the iso to make sure it downloaded properly. If you do have to redownload it, I'd recommend using BitTorrent because that includes automatic hash checking. See [https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM) and [http://www.ubuntu.com/getubuntu/downloadmirrors#bt](http://www.ubuntu.com/getubuntu/downloadmirrors#bt)
2) Try burning the disk at a low speed; high speeds sometimes cause errors.
