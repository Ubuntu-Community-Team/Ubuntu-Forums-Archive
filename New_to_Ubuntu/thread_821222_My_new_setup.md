The memory should not make any difference. And even with windows, 64 bit processors have not been full or even mostly supported. However there are users that use 64 bit processors. Also the network card should be fine. 

I would also recommend to run the live version. If you like it, but don't want to take the full plunge into linux, i would recommend installing linux within windows. This installs linux in the same partition as windows. This allows Ubuntu to run almost the same as it would if it had its own partition, but this configuration takes away alot of the confusion of the dual boot process.

### Post by markbuntu on 2008-06-08
The ATI drivers are much improved lately. You should have no problems with them unless you have a HD 3870x2 which is not supported yet. Older ATI cards (pre 9000) are fully supported by the open source drivers and ATI is doing a good job lately with its restricted drivers for newer cards.

I am running the restricted drivers on my Radeon HD 3650 with a 3800+ sinlge core processor and have no problems at all. Check this out for the latest on setting up an ATI card for best performance:

[http://forum.compiz-fusion.org/showthread.php?t=6794](http://forum.compiz-fusion.org/showthread.php?t=6794)
