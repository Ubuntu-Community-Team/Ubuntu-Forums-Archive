### Post by ugm6hr on 2009-02-01
> **Kellemora said:**
> I will probably run 8.04 until it expires, which I thought was in 2011, but I've now seen mention of 2013 also in unofficial places?

The reason I asked is that I wasn't sure if the 64 bit was different enough to cause me problems with available software.
Or mainly, will 32 bit software still run on the 64 bit version?

All the standard repositories have both 32 and 64 bit versions of everything.

If you *need* non-standard apps, then they *may* be 32-bit only.  You *probably* will be able to run them on a 64-bit OS, but if this is a major concern, I'd recommend just sticking with 32-bit.

8.04LTS has desktop support to 04/2011, and server support to 2013.  There will be a new LTS every 2 years (i.e. 10.04, 12.04 etc).

If you are unsure - just stick to 32-bit to keep all machines the same.
