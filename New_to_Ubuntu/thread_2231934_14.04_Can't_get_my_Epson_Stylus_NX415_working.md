### Post by cariboo on 2014-06-29
The driver you are looking for is cups-driver-gutenprint (5.2.10~pre2-0ubuntu2), open a terminal and type:

```
sudo apt-get install cups-driver-gutenprint
```

the above command should install the driver and it's depends.

BTW, a great way to find packages if you aren't sure of the exact name is to search on [http://packages.ubuntu.com/](http://packages.ubuntu.com/)
