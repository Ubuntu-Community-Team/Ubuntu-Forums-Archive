### Post by drs305 on 2008-09-25
When you set up the machine in VB, did you select audio. You can check under "Settings" and see what value you selected. I recently installed Hardy and Ibex in VB and used the PulseAudio option.

Inside VB, you could go to System, Preferences, Sound and try the various test options. If you selected PulseAudio in the VB settings, that should also be set up inside ubuntu's sound preferences. If you selected some other sound system in VB it should match that selection.
