### Post by Chesamo on 2010-10-22
Right-click on the network manager icon. Click "Create New Wireless Network". Have the other machines join the wireless network you create.

To create a network share you'll probably need Samba. [http://doc.ubuntu.com/ubuntu/serverguide/C/configuring-samba.html](http://doc.ubuntu.com/ubuntu/serverguide/C/configuring-samba.html)
