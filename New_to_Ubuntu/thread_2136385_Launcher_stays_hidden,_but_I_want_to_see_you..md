In Settings under Appearance on the Behaviour tab is some settings for the launcher being hidden.
If you have already messed with this then try turning the Reveal Sensitivity down?

Unfortunately I'm not sure about screens impingeing on the launcher.

EDIT: I haven't had the launcher hidden in a long time, but I just turned it back on and had a hard time also, but I think I've figured it out. It seems it doesn't like to be just tapped with the mouse. You have to move the mouse to the left side of the screen and drag up or down while dragging left more.
