### Post by mcduck on 2009-04-16
Have you installed the Microsoft Core Fonts package? You are probably more used to seeing fonts like Arial, Courier etc. on web pages..

Just install the "msttcorefonts" package.

You might also want to check what fonts & font sizes Firefox is configured to use.

(Web pages are only able to suggest your browser what fonts it should use to render the page, but the actual fonts must exist on your computer before they can be used. If the suggested font doesn't exist, the browser uses what ever it's configured to use by default.)
