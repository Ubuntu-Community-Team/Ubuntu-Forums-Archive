### Post by grahammechanical on 2015-09-18
Some of us on this forum think it is a good idea when buying a computer with Windows pre-installed to replace the original hard disk with another one. Then, when passing the computer on to someone else (may be selling it) they can put the original hard disk back in and pass on a working machine without too much fuss.

Some of us also think that when dual booting Windows and Ubuntu it is good to have 2 hard disks - one for Windows and one for Ubuntu. This is not a requirement but it is something to think about.

There are definitely less problems with installing Ubuntu onto its own hard disk and if it is the only OS on the machine. But just so you are fully informed here is the link to the information about installing Ubuntu on a Windows 8 machine. Most of it will apply to a Windows 10 machine. And you never know, you just might one day need to put the Windows hard disk back into that machine and run the 2 OS side by side.

[https://help.ubuntu.com/community/UEFI](https://help.ubuntu.com/community/UEFI)

### Post by Sweet_Baby_Jamie on 2015-09-18
I found a Dell laptop at a flea market for $50.  I brought a bootable Ubuntu disk with me so I could try it "Live" without making any changes to the computer.  If it works off a Live DVD, it'll most likely (not guaranteed, but *likely*) work when installed to the HDD.  Kind of a "try before you buy" thing.  The one I found did work from the Live DVD so we bought it and everything works!  You could try that - but be sure to get permission from the sales person to boot from a DVD or USB.

### Post by Geoffrey_Arndt on 2015-09-18
Also, it much simplifies the question of what hardware (model) to buy if you remember to look first for an "all Intel" based pc.    In other words, Intel graphics and Intel wireless.   Not needing to modify boot parameters (to fix the infamous "black screen" after install) or needing to get broadcom drivers (while being connected via ethernet) greatly improves the install and maintenance process.   Unless one is a gamer, this has been a good solution for many.  Also, just remember to also try for a pc with at least 2 GiB of ram if you're wanting to run Ubuntu proper (with Unity desktop)
