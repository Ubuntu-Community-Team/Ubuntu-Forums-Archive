### Post by oldfred on 2014-04-30
How much RAM and why 32 bit?
There is only one open source Intel driver.
Just about any system that needs 32 bit should be using Lubuntu as it is not new enough for full Ubuntu with Unity.

But my old 2006 laptop with CoreDuo Intel and 1.5GB of RAM and internal Intel video has 64 bit Ubuntu, but Unity was so slow that I had to install fallback/flash back which is a lighter weight gui with the old gnome2 menus style.
Many install Kubuntu or Lubuntu for systems that do not have the power for full Ubuntu. With 1.5GB of RAM I also find I can only load one larger app like Firefox and a couple of smaller apps. If I load more screen turns grey for a couple seconds while it uses swap and is slow. But my normal use works fine.
