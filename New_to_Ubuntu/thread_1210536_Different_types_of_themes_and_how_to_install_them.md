### Post by oldos2er on 2009-07-11
There are install instructions on the page you gave for the 'FinalCut' theme. Usually you just drag-n-drop the tar.gz file to the 'Theme' tab in System, Preferences, Appearance.

 Here's some more info: [https://help.ubuntu.com/community/UbuntuEyeCandy](https://help.ubuntu.com/community/UbuntuEyeCandy)
