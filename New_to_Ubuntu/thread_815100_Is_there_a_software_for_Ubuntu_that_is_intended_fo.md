### Post by SneakyWho_am_i on 2008-06-01
I'm a fan of Quanta or Kate. I've never tried a WYSIWYG editor outside of Windows. They're too depressing, what you see is never ever ever actually what you get. What you get instead is normally problems for colourblind/international/proper_browser_using people.

It can be hard to get started doing it "properly" if you rush it and try to learn it - so don't. But do try doing it by just writing the code. Google in one hand, editor in the other. If you're just making minor tweaks to your web page, you should get on track that way within about a week. If you're looking at making a bigger website, it should start working out after maybe 72 hours.
I speak from experience here. WYSIWYG editors are your enemy. If you're only writing one line of code, then by all means, use a WYSIWYG editor. If you come back after less than a fortnight and write a second line of code, then it would be far more useful to do it with a text editor like Kate.

If it helps, you **can** see what you're doing in a non-WYSIWYG-editor --- just keep the page you're working on open in a tab in your browser while you work, lol.

And of course (but you know this, right?) if it's just html and/or javascript and/or css, then there's no reason why you can't just save the file to your disk and work on it there. No web server required.

Food for thought, probably not useful to you but at least I tried :P
