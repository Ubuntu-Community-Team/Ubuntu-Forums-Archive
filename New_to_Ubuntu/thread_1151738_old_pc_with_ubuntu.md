### Post by halitech on 2009-05-07
The alt install cd is a text based installer. You will end up with the same desktop as if you had used the Live cd to install but has lower requirements to run the install since it doesn't boot a desktop first.

I would also suggest going with Xubuntu as it is slightly lighter on resources and should run better. You can download it from here:

[http://www.ubuntu.com/getubuntu/downloadmirrors#alternate](http://www.ubuntu.com/getubuntu/downloadmirrors#alternate)
