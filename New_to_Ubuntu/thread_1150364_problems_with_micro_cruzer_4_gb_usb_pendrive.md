I think that is going to be tough.  Password protecting a USB drive will always rely on software installed on the software - which most vendors probably won't produce for linux.  If it is for specific things like passwords or credit cards you could try KeePass, which you can get for windows and as a [portableapp]("http://portableapps.com/apps/utilities/keepass_portable") and I think there is a Linux version, but I prefer Revelation (in the repositories).
