### Post by dannyboy79 on 2009-03-04
I can tell you that if you want a good book specifically for Ubuntu, than go with one Authored by a Pro, Mark Sobell. 
[http://www.amazon.com/Practical-Guide-Ubuntu-Linux-R/dp/product-description/013236039X](http://www.amazon.com/Practical-Guide-Ubuntu-Linux-R/dp/product-description/013236039X)

I actually was a Technical Reviewer for the book and it is a very well written book for all Linux users whether you're a Linux Pro already or a Novice. It has screen shots, commands, and the whole nine yards. It's written for Gutsy Gibbon.
