### Post by lisati on 2009-04-21
[http://geocities.com/rjv_soft/ssc](http://geocities.com/rjv_soft/ssc) has a couple of offerings that clean out the current Windows user's temporary files.

For the other stuff, the "add/remove software" option on the control panel and the All Programs->Accesories->Disk cleanup options are good for doing a cleanup.

Don't forget to defrag your Windows partition a few times before shrinking it.
