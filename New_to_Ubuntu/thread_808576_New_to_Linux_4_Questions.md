### Post by bobblehat on 2008-05-26
Hi,

I'll kick this one off and maybe others will fill in the gaps...

> **JimmyJim said:**
> 1. I read that Creative has poor support for Linux. Should I install their BETA drivers or should I just use my motherboard for sound?
Just personal opinion - I'd get going with the motherboard to give yourself some confidence then fiddle with the more advanced and BETA stuff later.

> **JimmyJim said:**
> 2. On Windows I used to use WAMPserver for practicing web development. Is there a Linux equivalent that installs a similar package?
Don't know about a direct equivalent but all the standard LAMP stuff is available via Synaptic and there's a starter on dev tools here -> [https://help.ubuntu.com/8.04/programming/C/ar01s05.html](https://help.ubuntu.com/8.04/programming/C/ar01s05.html)

I tend to use Bluefish most these days.

> **JimmyJim said:**
> 3. I want to make sure that all of my drivers are properly installed and up to date. How can I make sure?
Just accept all the automatic updates to keep your system up to date. That helps with the next one too.

> **JimmyJim said:**
> 4. Is there anything I should know in regards to security?
There's a starter here -> [https://help.ubuntu.com/8.04/keeping-safe/C/index.html](https://help.ubuntu.com/8.04/keeping-safe/C/index.html) and then if you have any specific questions I'm sure if you post them here people will help out.
