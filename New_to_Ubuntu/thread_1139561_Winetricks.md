### Post by blazemore on 2009-04-27
The idea of winetricks is you install what you need, when you need it.
On the Wine AppDB, to get specific applications working, there might be one or two things you need to grab with Winetricks.
The comments will say what these are, so you can install them if you need them, or copy them from your own Windows installation.
