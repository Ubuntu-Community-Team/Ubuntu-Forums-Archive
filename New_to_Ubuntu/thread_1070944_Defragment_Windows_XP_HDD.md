### Post by kansasnoob on 2009-02-15
Just use XP's own utilities but clean up first!

In fact do this:

(1) Create a new restore point!
(2) Clean up - go for the gusto! Get ris of all the older restore points,etc.
(3) Defrag! If you haven't done it in a long time just start it at bedtime!

But, always back-up anything you can't lose before resizing! The one time you don't do so is the time you'll wish you had!

### Post by TwilightDawn on 2009-02-16
IMHO, use a cleanly formatted and partitioned drive if you want a dual boot system. It's the safest way. Sometimes, there may be system files that cannot be moved within windows and losing those files may cause problems down the line. You cannot also rely on the drive map of any defragger to determine exactly whether all the files have been moved.

Another option is to use a dedicated partitioning program, but I have not used any of those.

As for the defragger, I use Diskeeper 2009 Pro for my XP. It's automatic, thorough and works brilliantly. Unfortunately, it's not free, but you can get free limited time (but fully functional) trial versions from [http://www.diskeeper.com](http://www.diskeeper.com)
