### Post by sydbat on 2008-06-27
Go to System > Preferences > Main Menu. When the window pops up, go to System Tools and check Device Manager...this will place it under Applications > System Tools. I find it is similar to the Windows version of device manager, but has WAY more info.

---

### Post by nick_h on 2008-06-27
Install the package gnome-device-manager.  Then select Applications->System Tools->Device Manager.
