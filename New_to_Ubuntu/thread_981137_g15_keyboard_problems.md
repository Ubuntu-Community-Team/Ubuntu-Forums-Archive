### Post by quirks on 2008-11-13
I don't have a Logitech G15 myself, so everything I say is much like a shot in the dark, but:

What keyboard model have you selected? Is it the right one for yours? You can change it by going to System -> Preferences -> Layout tab -> Keyboard model button. Then select Logitech as a vendor and Logitech G15 extra keys with G15Daemon. (This is yours, right?)

You might need to log out and back in again or reboot even. Not sure, as I've never changed my keyboard model.
