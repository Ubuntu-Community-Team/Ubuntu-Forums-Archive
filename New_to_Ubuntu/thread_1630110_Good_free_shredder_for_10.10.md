### Post by Andrew.Z on 2010-11-27
[BleachBit has a GUI shredder](http://bleachbit.sourceforge.net[/url) for those who don't like the shred command line interface.  In the BleachBit file menu, there is an option to shred any file (such as an OpenOffice.org document) anywhere on your hard drive.  The web site has the most recent version (soon 0.8.3), and Ubuntu 10.10 has 0.8.1 or 0.8.0 available through Synaptic/APT.
