### Post by Sorivenul on 2009-03-06
As long as you've got web access from your pendrive install you can save to the web/cloud services. I don't use any specific client on my pendrive install, just my browser, as I don't see the point in installing more software when a website can do it for me. 

Anymore I just use [DropBoks]("http://www.dropboks.com/"), but if you are looking for more than 1 free Gb, it may not be for you. I've also used a "Basic" [ADrive]("http://www.adrive.com/") account in the past.
