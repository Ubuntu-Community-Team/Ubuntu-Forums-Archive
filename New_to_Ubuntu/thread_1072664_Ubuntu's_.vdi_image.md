### Post by Fire_Chief on 2009-02-17
You can create a VDI file which will be used for the VM environment disk. You then install Ubuntu from within the VM environment using the ISO file.

Run the New Virtual Machine Wizard from the Virtualbox console and you'll see what I mean.
