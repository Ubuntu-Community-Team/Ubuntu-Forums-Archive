### Post by canthus13 on 2008-04-22
Run the test CD option when you boot it.  If it fails, reburn at the slowest speed your CD burner allows.  If it doesn't fail, it could be that you need to adjust the boot options a bit.

Test the CD, reburn if necessary, and if it's still a problem, search for your make/model of computer/motherboard and see if anyone else has similar issues.  It's quite likely that you'll find the fix somewhere.  If not, post the computer info here and we'll see what we can do for you.
