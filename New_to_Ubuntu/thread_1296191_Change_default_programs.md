### Post by timmyhiggy on 2009-10-20
Hi

I recently installed xpdf and i much prefer it to document viewer, but whenever i double click on a pdf file it opens it in document viewer. Is there a way of changing this so that it will use xpdf by default or am i destined to forever use right click?

---

### Post by sisco311 on 2009-10-20
Right click on the .pdf file -> Properties -> Open With tab and select xpdf.

---

### Post by timmyhiggy on 2009-10-20
so this does mean that im forever destined to right click and can't just set a different default program for pdfs?

### Post by mocoloco on 2009-10-20
I'm not familiar with UNR, but in Gnome the properties dialog for open with will say "Open This file and all files of type 'pdf' with:" and then you select the app.  So it becomes permanent.
