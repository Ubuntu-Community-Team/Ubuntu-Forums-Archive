### Post by Paqman on 2009-11-17
> **winjeel said:**
> 
1. Why do they still sell this computer with 8.04?


8.04 is a Long Term Support version. It's just easier for Dell to support one version for several years than to be shifting to the new release every six months.

> 2. If I did upgrade it, what problems would I expect?

You should be fine, but search these forums for the Mini 10V (same name in the West, btw)

> 3. Is it better to do a clean install with 9.10 (or 9.04) than the supplied 8.04? Or install 8.04 then upgrade to 9.10?

The upgrade path from 8.04 would go 8.04 > 8.10 > 9.04 > 9.10, so it would be waaaay easier to just reinstall with 9.10. I'd highly recommend using Ubuntu Netbook Remix on your machine. Alternatively, you could wait six months for the next LTS release. You'd be able to upgrade straight from 8.04 to 10.04.

> 4. I'm in Japan, but my Japanese isn't that good. Is it possible that the 8.04 that's coming with this Dell will be multi-lingual? (Assuming that English would be one of the other languages included).

Ubuntu supports dozens of languages. You should have no problem switching it to English.
