### Post by mc581 on 2009-11-20
Dont dual boot try running ubuntu in a virtal machine. I have had very good results with VmWare player v3 in vista. I also run another virtual machine with xp . . . all on a vista laptop! It is great check it out. Also I had to re configure and format the hdd when ubuntu accidently tried to install a dual boot. ### Post by Dr. Oddbio on 2009-11-20
well I am currently dual booting on a desktop computer with win7 and ubuntu.

If you want to "choose your OS" at startup don't use wubi. Instead get the live CD and do a real install, it should be pretty straight forward to set up dual booting.
However, some people report having problems.
But as long as you have windows 7 installed first, free up some disk space and make sure you defrag your hard drive, then I don't see why you would run into any trouble.
