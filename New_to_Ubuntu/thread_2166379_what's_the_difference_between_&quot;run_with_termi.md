### Post by TheFu on 2013-08-09
Do you need/want to see output that the program sends to stderr or stdout?  If so, run in terminal is what you want.
There is ZERO impact on security running either way ... unless your terminal program has been replaced by something dangerous to the system. The real program could be just as dangerous.

Have you tried both ways and seen the difference?
