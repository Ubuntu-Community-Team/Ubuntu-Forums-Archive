### Post by chaanakya_chiraag on 2009-12-18
There's something called PDF Mod that will do exactly what you want.  Search for it in the repos or install it using ```
sudo aptitude install pdfmod
``` (enter that into the Terminal: Applications->Accessories->Terminal)  That should do the trick :)
- Chiraag
