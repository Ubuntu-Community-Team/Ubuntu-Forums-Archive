### Post by wilsong on 2008-09-30
You can install the applications that come with ubuntu studio while using ubuntu regular 8.04 or whatever brand of buntu you use. It is nice, it has a lot of free audio software. 

open Synaptic Package Manager and type in Studio in the quick search 

Install ubuntustudio-video ubuntustudio-audio ubuntustudio-desktop 

and whatever else you see fit.  I didnt install the ubuntustudio-screensaver, but i guess thats personal choice.. 

you can do this through terminal too 

```
apt-get install ubuntustudio-desktop
```

Cool stuff tho! google should find some forms or groups
