I didn't find how to do what you want, but if you go into Pidgins preferences and choose "Show tray icon Always", you will be able to see the status properly.

Also, I swapped to the "Main Menu" option on the panel instead of Custom Menu (the default), as this contains the shutdown/restart options within it - Then I removed the User Switcher completely as I don't need to switch users in this manner.

Hope that helps

### Post by PureLoneWolf on 2009-06-14
If you right click the panel and choose "Add to panel" then select "Main Menu" from the list that appears, you will get a small Ubuntu icon.  This is a full menu that includes Applications/Places and System consolidated.  You can then right click the existing menu and choose "Remove from panel" and do the same for the User Switcher (your username on the panel)
