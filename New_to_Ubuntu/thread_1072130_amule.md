### Post by 4Orbs on 2009-02-17
In your file manager, go to "View" and tick the option to "Show Hidden Files". Then you can find .amule in your home folder.

### Post by dzark on 2009-02-17
Amule -> Preferences -> Directories should list your incomming folder.

Mine by default is home/.aMule/Incoming

So to see that in your file browser you'll need to go to your home directory and press Ctrl-H to view hidden ( .whatever ) files and folders
