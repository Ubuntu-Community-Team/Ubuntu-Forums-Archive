### Post by Pirkiz on 2011-03-24
Hey folks

The other day, I installed Ubuntu 10.10 Netbook Edition on a Samsung N145 netbook. Everything's working fine, except one thing: the screen brightness is suddenly very low. It doesn't matter if the computer is running on battery or AC power. Adjusting the screen brightness using the keyboard or the brightness applet for the panel doesn't have any effect whatsoever. This computer was previously running Windows 7, and then there was never any problem with the screen brightness.
Any ideas?
Thanks!

---

### Post by Peter09 on 2011-03-24
As a work around you can use the keyboard function keys for brightness successfully when you are in the 'select O/S' screen during boot.

---

### Post by Pirkiz on 2011-03-24
That works perfectly! Thanks!
