### Post by Mark Phelps on 2015-04-10
The MS drivers won't work in Linux but you shouldn't need any mouse drivers.  I've used numerous wireless mice in different Linux distros and never needed any special drivers.  You will get only basic functions, but that's typical of using Windows hardware in Linux. 

Since the mouse, as you claim, also does not work in Win7, then there's clearly a hardware issue. 

I would first check the batteries and put in new ones if they aren't new.
