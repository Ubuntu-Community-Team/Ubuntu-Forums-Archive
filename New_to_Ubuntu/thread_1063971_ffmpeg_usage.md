### Post by paul.gevers on 2009-02-10
In principle all coding that can be done with ffmpeg can be done with winff. However, it really depends on which version of ffmpeg you have installed. Please see [https://winff.org/winffwiki/UbuntuInstallation](https://winff.org/winffwiki/UbuntuInstallation) for a complete installation procedure.

Winff does batch conversion of multiple files to one format. So depending on your exact question, yes.
