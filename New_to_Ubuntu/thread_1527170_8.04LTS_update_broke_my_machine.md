### Post by jivabill on 2010-07-09
Found some solutions, most requiring a lot of command line magic.  Decided to go for one of two easy fixes: 1) fresh install on a HD that fits the BIOS for this machine or 2) Use another Distro that is not so fat that it gets the kernel into this problem with the BIOS.  

For now I will just take it from the Error 18 message and select the previous kernel to boot.  When I fix it I will put in a small HD 40Gb and do a whole new install.  Cumbersome but it works.  Certainly feels a bit like Windows, these problems...
