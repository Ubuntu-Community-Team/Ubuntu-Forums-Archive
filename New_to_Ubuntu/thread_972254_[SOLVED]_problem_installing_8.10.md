### Post by eolson on 2008-11-05
Go to 

system
administration
software sources

and all the way at the bottom make sure the cd rom is UNchecked.

### Post by johnn1949 on 2008-11-05
try going to Administration--> software sources and see if the boxes for cdrom are checked at the bottom.  if so uncheck them.

It looks like you are trying to read off the disk which isn't there.
