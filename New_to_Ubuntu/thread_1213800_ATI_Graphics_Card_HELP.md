### Post by carml on 2009-07-15
First of all try to right click on the desktop,select Set Desktop Background->Visual Effects->Optional (also Normal should work) .
If pops up a message claiming that those effects can't be enabled,your card need some proprietary driver if you want to use advanced 3-D effects or advanced videogames.
If the last option is true go to System->Administration->Hardware Driver and look if there is the voice Accelerated Ati Video Drivers,left click on the square and reboot et voilà it's all done.
If none between 3D effect or advanced videogame is your aim don't worry about drivers.
Post here if you have any doubts or problems and someone will answer. :)
