### Post by sisco311 on 2010-07-18
For Empathy and other Gnome apps you need the aspell-pt package. OO, chromium & other apps use hunspell.

In OO go to Tools -> Language Settings -> More Dictionaries Online...

In chromium right click on an input text box (like the Quick/New Reply box) select Spell-checker Options -> Language Settings
