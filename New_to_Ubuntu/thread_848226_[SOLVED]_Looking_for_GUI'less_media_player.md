### Post by Inxsible on 2008-07-03
Cool. Infact there are many console music players.

1) aplay
2) splay
3)mpd - music player daemon - This is great because you can set it up on a server and then connect via a client like sonata or ncmpc. It is also CLI based, so you can have it on your localhost too and use settings in conky to play music as well.
4) Xmms2 - is also a console based player - which has gone the mpd way -as in you can connect to it via different clients.
