What have you tried - you mentioned downloads or mirrors not working? It's been a long time (5+ years) since I've tried Linux on a PPC platform, but it only took a simple Google search to find the URL to the ISO for the latest Long Term Support release (12.04):

[http://cdimage.ubuntu.com/releases/precise/release/ubuntu-12.04-desktop-powerpc.iso](http://cdimage.ubuntu.com/releases/precise/release/ubuntu-12.04-desktop-powerpc.iso)

Another page, with links to the PPC FAQ and Known Issues:
