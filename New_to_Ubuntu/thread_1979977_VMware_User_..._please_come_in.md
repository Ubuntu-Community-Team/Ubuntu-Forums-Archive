### Post by searchfgold6789 on 2012-05-14
It depends on which you use more frequently. If you use Ubuntu more frequently, make it the host for a Windows VM, and vice versa. In either case you won't gain many advantages.

### Post by zandman on 2012-05-14
And there are tools available from VMWare that can convert a physical system to a virtual machine that you can then easily import to the virtual machine manager.
Running Ubuntu 11.10 on this machine, 12.4 on another one. In both a WinXP VM runs like a charm with Virtualbox.

As mentioned earlier: decide which system you&#341;e going to use most and make that the host.

One thing: don't hold back on ram. The main machine that i use has 8GB.

Big advantage of going virtual is that you don't have to shut down one to go to the other.
