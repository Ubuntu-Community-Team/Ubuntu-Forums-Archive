### Post by varunendra on 2013-03-17
Welcome to the forums DLeBeau206! :)

> **DLeBeau206 said:**
> If I have Ubuntu on USB and I boot from it is it not going to use my hard drive at all?

Not unless you want to. The local hard disk partitions will show up in the file browser but will remain unmounted until you click on them.

Once mounted, you can read or write to the partitions as you wish.

### Post by varunendra on 2013-03-17
Yes, and that too if the usb is 'Persistent' (all popular live usb creation tools make it persistent by default).

Unless you explicitly choose to save something to the local drives, the downloads will go to the usb as well (in the casper-rw file/partition on the usb), and will be accessible only when you boot from it.
