### Post by Alwimo on 2011-07-13
Launchpad is a website that can be used by developers to help them collaborate on projects, and people can tell them about bugs they found. [https://launchpad.net/](https://launchpad.net/)

Ubuntu One automatically saves files to a server that is somewhere else, over the internet. Files that are saved in your Ubuntu One folder automatically get synchronised with that folder. This lets you access the files from other computers or devices, when you log into Ubuntu One. You can put up to 2gb for free, but you can pay for more. [https://one.ubuntu.com/](https://one.ubuntu.com/)

PPAs are Personal Package Archives. Ubuntu comes with lists of programs that it is familiar with and you can install them through the software centre, synaptic or the terminal. PPAs are a way to let you make Ubuntu aware of more programs, or different versions of programs (such as a later release). Be careful, though, since they can be security risks so only install ones you trust.
