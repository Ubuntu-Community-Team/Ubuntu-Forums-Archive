Hello 4-Ever-Euphemistic (great name btw)

   There is some software/firefox plugins to do this but for me, I prefer the simplest solution which is:

  Navigate to >Places >Computer >Filesystem >tmp

When the youtube video you have in your browser is fully downloaded, you can simply drag and drop from tmp onto your desktop, or anywhere else you might like to put it.

### Post by Fingers &amp; Thumbs on 2008-07-03
No probs mate, I perhaps should have been clearer in my explanation.

  Heres an example,

Using Firefox go to youtube and select a video to view, lets say for example:

[This Video]("http://www.youtube.com/watch?v=2rlIpwN5stg")

As I'm sure you are aware, sometimes the videos are slow to download, but the progress/play bar will progress until full.

  What this means is that the video is actually being downloaded to your computer, it has to do this in order to play it, since it is not a stream. It stores this file temporarily in tmp, but this file will dissapear when you navigate to a different page in firefox or whichever web browser you use.

  But be sure to wait until the file has completely buffered to your tmp folder before dragging n dropping.

Hope this help, report back if not, or please mark this thread has solved.

Best wishes.
