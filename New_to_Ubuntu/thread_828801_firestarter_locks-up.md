### Post by gn2 on 2008-06-14
You do not need Firestarter running, in fact leaving it running has been argued to be a security vulnerability.

Firestarter is just a GUI tool for adjusting the settings of the default installed Firewall which is iptables.

Use Firestarter to adjust the settings (if required) then close it.

Reading this might prove useful: [http://ubuntuforums.org/showthread.php?t=510812](http://ubuntuforums.org/showthread.php?t=510812)

### Post by ajgreeny on 2008-06-14
As gn2 says, the only reason you might need to even install firestarter is if you have a server running and attached to the internet and your network.  For a standalone desktop or laptop it is superfluous and seems to cause more problems than it solves.
Try to forget your windows experiences and let linux, and Ubuntu take you to a new and safer place.
PS, you don't really need to worry about virus checkers either.  When I first started with Ubuntu about 3 years ago, I had both firestarter and clamav on my machine, now I don't bother with either.
