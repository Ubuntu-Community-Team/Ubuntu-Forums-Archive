### Post by mikeyphi on 2009-10-09
You can confirm Free space - when you plug in the USB drive it should automount, bottom left of the screen gives Free Space.

You will probably find that it's not enough!

I suggest you install Partition Editor
When you start Partition Editor you will be able to see ALL of the USB drive (it will usually be the last drive shown) and be able to reformat, if you choose, giving maximum available storage.
