### Post by diablo75 on 2010-09-02
One thing for you to do is use Gimp to change the canvas size of an image.  If you size the width down to 1024 the height is probably going to land on 768 and you want 600.  If you don't keep the aspect ratio, the image is going to look squished.  So after getting the width down, use the change canvas size tool to crop off exactly 168 pixels from the top, or bottom or both.

### Post by Vaphell on 2010-09-02
in gimp you can pick the crop tool (shift+c) and in its properties (bottom part of the toolbar) you can enter your resolution and set the tool to preserve fixed proportions. Click-drag on image to select the relevant part, you can move and scale the selection box. Once you set the green rectangle properly press enter and you'll get new image with only that interesting part. Save it and your wallpaper is ready.
You can additionally resize whole picture to fit your resolution but it's not necessary as the wallpaper in 'stretch to fit' mode will do that for you.
