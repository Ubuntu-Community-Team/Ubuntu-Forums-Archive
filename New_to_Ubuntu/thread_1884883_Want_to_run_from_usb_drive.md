### Post by audiomick on 2011-11-22
I think the key word is "alternate". The alternate images don't offer a live environment, only the installer, which is also a text based installer and not the gui based one that is on the live cd.

I can't say for sure what has hung up, but one thing that occurs to me is that you might have got stuck at the bit where you can test to see if your keyboard configuration is correct. Can't remember how to get past that, but it might involve cursor arrows or something... :-k

If all else fails ( I would have another go at it... ) you could try this
[https://help.ubuntu.com/community/Installation/FromKnoppix](https://help.ubuntu.com/community/Installation/FromKnoppix)
