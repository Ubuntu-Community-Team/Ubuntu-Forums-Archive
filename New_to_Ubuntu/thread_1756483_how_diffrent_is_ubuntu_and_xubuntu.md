### Post by Paqman on 2011-05-12
> **neil_1 said:**
> how different is ubuntu and xubuntu ?


They have a slightly different interface, and many of the default applications that come preinstalled (eg: file manager, movie player) are different.

> are the terminal stuff same ?

Yep

> which ones amd64 would be faster ?

Depends which type of speed you're talking about. If it's something like opening an app that's not already running, then that's mostly governed by the size of the app and your hard drive speed. The same app will open at the same speed on both, but Xubuntu includes a lot of lighter apps that might open quicker than bigger apps you might have on a Gnome system.

They both use exactly the same repositories, so you have all the same apps available on either. So there's nothing stopping you from installing the lightweight word processor from Xubuntu (Abiword) on Ubuntu, for example.

If it's speed that's governed by your processor, then they'll both be exactly the same.

### Post by mcduck on 2011-05-12
> **neil_1 said:**
> and are most of the runtime stuff also same ?
like the gstreamer plugin ,etc ?

Every Ubuntu version is actually one and the same OS. The different versions just give you different things (like desktop environment and selection of programs) straight from the install.

So apart from the DE used by default (or complete lack of one in the Server version) there is absolutely no difference between Ubuntu versions. They are one and the same Ubuntu.
