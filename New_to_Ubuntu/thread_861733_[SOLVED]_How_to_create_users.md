### Post by phidia on 2008-07-16
From the top panel menu System>Administration>Users and Groups you should be able to use that to create a users with just those privledges.

### Post by RomeReactor on 2008-07-16
Hi. Go to 'System->Administration->Users and Groups'. To restrict access to the account, use the 'User Privileges' tab.
