### Post by anewguy on 2009-04-25
There are many posts here already about this, so the first thing I would do is use the search function in the forum and search for acer aspire one wireless and see what comes up.  I think you need the Atheros driver, but for some reason I thought it was included but I must be wrong.  It may be that you need the madwifi drivers.  Since you are using Wubi, can you "see" your Windows files while running Ubuntu?  If so, go in to Windows and get on the net, search for madwifi, go to the site, then download the driver and the instructions.  Go back to Linux and try to install the driver you downloaded.  I'm not sure what you're going to do though if it wants any dependancies, perhaps something like build-essential.  That's why I suggest you use the search function in this forum and read all the posts.  That's what we do here to try to learn, and it's what some of us have to do in order to try to help someone also.
