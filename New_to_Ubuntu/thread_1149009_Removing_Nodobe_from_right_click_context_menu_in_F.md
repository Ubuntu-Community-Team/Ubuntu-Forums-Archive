### Post by pastalavista on 2009-05-04
Type 'about:config' in the address bar of Firefox, find the line(s) about Nodobe and either delete or edit it (change true to false)

btw.. what is Nodobe? If it's an add-on, extension or plugin, you may be able to disable it in the tools menu.
