### Post by apienk01 on 2008-04-25
I've been using Ubuntu amd64 for 2 years now, and I must say that if I ever decide to do a clean install, I would install 386. The 64-bit system works fine, but there are problems with all the proprietary software one can hardly live without nowadays, like Adobe Flash in Firefox, RealMedia formats, or Skype. And for more obscure Linux software, not included in standard repositories, you'll often find only 386 packages, leaving you with no other choice but to compile yourself. For Flash, I have finally settled with Gnash, an open source Flash implementation. It is far from finished. The rest I have finally made to work, but this required quite an amount of effort and digging into the system.

My bottom line: use 386, not amd64. Life gets easier.
