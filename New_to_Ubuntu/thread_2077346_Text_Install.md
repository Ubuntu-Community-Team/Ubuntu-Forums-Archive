### Post by Lars Noodén on 2012-10-28
Ubuntu no longer has an alternate install CD but the other versions still do.  You can get Lubuntu's alternate CD and the install the package ubuntu-desktop to get Ubuntu.  You can grab the torrent for Lubuntu's alternate CD here:

[http://cdimage.ubuntu.com/lubuntu/releases/12.10/release/](http://cdimage.ubuntu.com/lubuntu/releases/12.10/release/)

That will work.

Another option might be to try doing the same thing with the server image.
