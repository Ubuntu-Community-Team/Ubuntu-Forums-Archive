### Post by forrestcupp on 2012-05-10
Well, you could make a shell script to shut it down. Then you could create a launcher to run your shell script. If you don't want it even asking for your sudo password, you could hardcode it into the script, but that's not a smart thing to do if someone you don't trust has access to your computer. But if you just really want to do that, you would open Gedit, enter the following code, save it somewhere, and make that file executable. When you open the script, choose to Run from the Terminal.

```
#!/bin/bash
echo "*password*" | sudo -S shutdown -h now
```Where "password" is your sudo password, leaving it in the quotes. Then you can create a launcher to that script, and it will shut your computer down immediately. If you want to be safer and you don't mind entering your password each time, you could change the 2nd line to 
```
sudo shutdown -h now
```There are ways to edit your sudoers to run a script in root without requiring a password, but that's beyond me.
