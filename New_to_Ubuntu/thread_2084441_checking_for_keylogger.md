### Post by TenPlus1 on 2012-11-15
Keyloggers need root or super-user permissions to run and monitor your key presses so I doubt it will be that...  Linux in itself is quite difficult to infect or run spyware unless you personally give the virus your password...

I would recommend upgrading yout system to 12.04 so you are on an LTS release and installing a program called BleachBit which will not only delete your browser cache but many other things including logs that could be slowing your system down...

### Post by Myoldmopar on 2012-11-15
There should be 32-bit and 64-bit versions of all recent releases of the Ubuntu-family of OS's.  Just make sure you get the 32-bit and you're all set.
