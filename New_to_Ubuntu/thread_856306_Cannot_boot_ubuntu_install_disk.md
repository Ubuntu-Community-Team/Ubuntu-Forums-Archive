### Post by bumanie on 2008-07-11

### Post by Victormd on 2008-07-11
Check your boot order again. Some comps ask you to hit a key to boot from the CD (even though it's set to boot from the CD first).
If this is not your case, then it seems as if the ISO file you downloaded may have gotten corrupted during download and that's why the system can't boot from it. Try another download and check the CD after burning (there is usually an option to verify data on CD once burn is complete).
