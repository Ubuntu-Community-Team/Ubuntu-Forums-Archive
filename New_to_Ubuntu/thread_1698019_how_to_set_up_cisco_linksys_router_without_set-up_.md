### Post by uRock on 2011-03-01
In a browser, enter **192.168.1.1**, you'll get a pop-up asking for a username and password. The default password is **admin** on linksys routers, just leave the username line blank. You'll be greeted with a set-up page with multiple tabs.

If you have any more questions on the setup process, then let us know.

Cheers,
uRock

### Post by eriktheblu on 2011-03-01
In your browser address bar, enter [http://192.168.1.1/]("http://192.168.1.1/")

This is the default address for Linksys routers. You should be able to modify router settings there.
