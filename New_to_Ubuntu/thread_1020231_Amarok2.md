### Post by Progressive on 2009-01-16
Amarok2 is integrated with KDE4 so you can go into KDE's system settings to change the appearance.

The new 2.0.1.1 version has queuing and a few other nice features. Some of the other little features that I miss are already on the agenda.

I agree we need better portable music player support. We also need smart playlists, visualizations, and grouping-by-letter in the collection.

While most of this stuff is on the agenda, Amarok is now much more scriptable, so it should be fairly easy for interested 3rd parties to implement new things.

Amarok2 is faster and better integrated into KDE4. It is quite stable. Most of the old good features have already been implemented. The new scripts provide some awesome functionality. I just wish they came standard so more people could know about them. PACPL should also come standard.

Seriously, there is more good than bad here, and it is getting better all the time. The old Amarok received tons of polish, so it is understandable that we have high expectations, but to call Amarok2 "rubbish" is unjustified.
