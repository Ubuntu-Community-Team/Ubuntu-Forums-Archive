### Post by NightwishFan on 2010-12-05
There is no supported method to downgrade. Back up important data to a portable drive or DVD, and do a fresh install of 10.04. Ensure the graphics problem is a 10.10 issue by running a live cd of 10.04 first.

### Post by karthick87 on 2010-12-05
It's **possible** at the level of the packaging tools(apt).But **the resulting system may not be equivalent to doing a reinstall, and you may get errors along the way**. This is because many packages contain specific support for upgrades.
