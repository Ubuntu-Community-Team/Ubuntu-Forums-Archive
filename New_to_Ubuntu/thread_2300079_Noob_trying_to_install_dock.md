### Post by Sweet_Baby_Jamie on 2015-10-23
> **Shunsaddi said:**
> 
Going to look at those, and see if I can find out how to install.

Open software center and click "install!"  My preference is Synaptic Package Manager.  Open, click "_R_eload" to get all the latest info, then choose Docky or Cairo Dock, whichever you want to try and put a check in the box next to it.  Then _A_pply.  The selected software is automatically downloaded, unzipped, and installed.  It should then appear in the menu to be opened!
