### Post by TeoBigusGeekus on 2011-01-07
Open terminal, launch nautilus with
```
nautilus
```
right click on a folder/file, go to properties and choose to always open it with nautilus.

### Post by gordintoronto on 2011-01-07
(Thanks to Frogs Hair in the Ubuntu Forums) Right click the desktop and select create folder. Right click the folder and select open with another application. Select file browser from the menu and make sure "remember this application" is checked. Remove the folder when finished.
