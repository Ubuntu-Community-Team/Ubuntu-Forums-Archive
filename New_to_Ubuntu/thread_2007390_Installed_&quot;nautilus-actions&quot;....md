### Post by kanikilu on 2012-06-20
The dash was introduced in 11.04, I think...

[https://help.ubuntu.com/11.04/ubuntu-help/unity-dash-intro.html](https://help.ubuntu.com/11.04/ubuntu-help/unity-dash-intro.html)

As for nautilus-actions, I'm not sure what menu (if any) it will show up in for 10.04, so you might just want to run it from a terminal. Go to Applications > Accessories > Terminal, and then type the following and press enter:

```
nautilus-actions-config-tool &
```
