### Post by 3rdalbum on 2010-02-17
Linux rule of thumb: Don't use old distros.

The newest version of regular Ubuntu that will run on 128mb of RAM is Ubuntu 5.10, which is four years old. The software repositories for Ubuntu 5.10 were deactivated in 2007, leaving you with no way to install software if you decided to install it.

The only thing you'd be able to install would be rootkits (malware), because 5.10 has a number of security issues that have since been fixed in later distros. At least one of those makes it easy for any user-owned process to gain your password.

In short: Use the latest Puppy Linux or something similar. DO NOT use ANY distribution that has reached End Of Life.
