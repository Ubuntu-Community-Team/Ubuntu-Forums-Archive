### Post by mcduck on 2013-01-27
Right-click on a sound file of the type you want to assign to VLC, then select properties, on on the open with-tab set the program you want. Repeat for all the other audio/video file types you want to open with VLC instead of the current default application.

### Post by GT1 on 2013-01-27
I would also suggest you change your default music player program like this as well as following the steps above

System settings > Detail > Default programs, then select your desired music player from the drop down box next to music

Also works for setting default programs such as web browsing, mail,
calender, video and photo's
