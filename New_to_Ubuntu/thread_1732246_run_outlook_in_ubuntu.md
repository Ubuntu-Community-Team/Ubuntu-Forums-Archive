### Post by Rasa1111 on 2011-04-18
You cannot just install outlook onto a Linux system. 
.exe files are for windows only. 

You probably could run outlook through WINE, but in my opinion it wouldnt even be worth it.

 You'd be better off installing Thunderbird or something instead. 

 windows programs/.exe's are not made for Linux, so they will not just work for Linux.

 Thunderbird is awesome though!
