You should be fine from there, ff3 should import the bookmarks just fine

### Post by Unstuck on 2008-06-17
Is it too late to install [Foxmarks]("https://addons.mozilla.org/en-US/firefox/addon/2410") on FF2?  If not, install the add-on, upload your bookmarks, load it into FF3, and sync the files.  It's easy and handy.
