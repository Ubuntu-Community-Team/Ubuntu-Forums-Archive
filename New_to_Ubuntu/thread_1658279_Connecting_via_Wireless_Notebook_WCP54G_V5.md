### Post by perspectoff on 2011-01-02
Are you using Network Manager?

If so, join the thousands of people with similar problems. The solution often is not to install wireless drivers manually (a long and thankless task fraught with problems), but to get rid of Network Manager.

I had the same problem for months as well (but with a different wireless card), then uninstalled Network Manager and installed Wicd... 

Everything worked after that.
