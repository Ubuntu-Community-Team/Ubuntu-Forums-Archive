### Post by AusIV4 on 2009-05-25
I run Xmarks on several different computers, and I've only been asked for my password when I set it up on each computer. I believe it uses Firefox's password manager for storing your username and password though, so if you're clearing that it may be losing its password.

### Post by sgosnell on 2009-05-25
I'm also only asked for a password at setup.  On the Xmarks settings window, are you checking the "Remember this password between sessions" box?  If not, you will be asked every time.

---

### Post by powel212 on 2009-05-25
I encountered the same problem once because I had forgotten that I had changed my cookie permissions. If you are using a cookie manager it may be that "Foxmarks.com" was allowed to save cookies but "Xmarks.com" maybe isn't allowed cookies.

If you are not using a cookie manager just check the cookie global settings.

Powel
