### Post by avtolle on 2009-03-06
Evolution is installed by default on Ubuntu, and normally in the top panel next to the Firefox icon there is an envelope icon by which to launch Evolution in the default Gnome install. You could check to see whether Evolution is installed in Synaptic (type evolution in the search box and see if it indicates it is installed by the box next to the name being a solid color). You could also choose to install Thunderbird as the email client, and add the icon to the panel by right-clicking the panel, select add to panel, then application launcher (IIRC), go to internet, open that, and select Thunderbird. To add the Evolution icon to the panel, if that's what you want to use, proceed in the same way (again IIRC).
