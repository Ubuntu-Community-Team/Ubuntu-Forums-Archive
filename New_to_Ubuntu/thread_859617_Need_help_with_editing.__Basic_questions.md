### Post by Hospadar on 2008-07-14
Kino's nice for cutting, if you want a nice advanced graphical transcoder (if you're uncomfortable with CLI transcoders like mencoder and transcode) you may want to look into avidemux.  I use it along with my command-line encoders.  There are some things it does really well.

If you need more advanced video editing, you'll probably have to look into commercial software.  Blender could do some animation, but it's not easy to use, you'd need a book and a whole lot of time at least.

### Post by fenian on 2008-07-14
> 1. what is the best free software i can get for home made editing?

Kino
Kdenlive
Open Movie editor
Cinelerra (higher learning curve but more powerful.)

> 2. I wanted to make a movie but, I only have a couple people that can help me out. is there any way i can edit thier faces to make it look like a diferent person?

Pretty advanced CGI here I don't think you will find a free solution,maybe with Blender.You can also [Rotoscope]("http://en.wikipedia.org/wiki/Rotoscoping") using gimp,very tedious and time consuming.

> 3.how does green screen editing work?

Good basic info [here]("http://en.wikipedia.org/wiki/Bluescreen")
Can be done in both Open Movie and Cinelerra.

> 4. how do i put in action efects?... Like gun shots ect.

For video effects this is again more advanced CGI you can use rotoscoping or a [matte technique]("http://en.wikipedia.org/wiki/Matte_(filmmaking)").
For the audio you would just create a seperate track for audio effects and put the clips in the appropriatplaces.

Any advanced video editing/effects programs on any platform are pretty expenive ($2,000 +) as well as dificult to learn.I would recommend  trying to use as little extra effects as possible (not just on linux but any platform).
