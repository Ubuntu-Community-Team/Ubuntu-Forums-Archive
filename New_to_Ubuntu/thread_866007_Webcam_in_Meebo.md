### Post by Sef on 2008-07-23
To download Amsn, (an msn clone and has webcam support):

Applications > Add/Remove > Show: change to 'All Available Repositories' > Search: Amsn > click on box next to Amsn > click apply changes > apply > close

### Post by imon9 on 2008-07-23
meebo has 2 webcam chat servie provider. One of them is [www.mebeam.com](www.mebeam.com) (which i use for over 2 year now)

go to [www.mebeam.com](www.mebeam.com) and create a random room name. (no registration needed at all)

it uses flash to do video-chat, and flash in linux only support v4l webcam, if u have a newer webcam, u might need to tweak it a bit by installing flashcam ([http://www.swift-tools.net/Flashcam/](http://www.swift-tools.net/Flashcam/))

### Post by ajhtiredwolf on 2009-05-15
You need to allow meebo access from the settings manager, go here 
[http://www.macromedia.com/support/documentation/en/flashplayer/help/settings_manager06.html](http://www.macromedia.com/support/documentation/en/flashplayer/help/settings_manager06.html) Look for mee.tokbox.com set to always allow. This works on a few other websites too. The problem is when you set it to allow while in the website for whatever reason it doesn't actually do it. 

Also after setting it to allow restart firefox
