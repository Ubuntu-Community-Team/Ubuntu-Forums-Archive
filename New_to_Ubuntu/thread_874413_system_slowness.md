### Post by Famicommander on 2008-07-29
512 MB is the lower end of the spectrum for running Ubuntu. Ubuntu uses GNOME by default, which is fairly resource-intensive.

You might try installing Xubuntu, which is Ubuntu with Xfce. It's the same basic operating system, but Xubuntu does better with computers that have lower specs.
```
sudo apt-get install xubuntu-desktop
```

Then restart. When you get to the splash screen, you'll be able to choose which to boot by clicking "Sessions". You can keep Ubuntu around for if you ever get more RAM, or remove it and use Xubuntu exclusively.

All of your settings should transfer over to Xubuntu painlessly.
