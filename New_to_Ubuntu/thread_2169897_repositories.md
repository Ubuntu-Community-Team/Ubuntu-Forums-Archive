### Post by grahammechanical on 2013-08-24
The Software Centre does all the work for us. It will install and it will remove. It is best to use the Software Centre and the Update Manager to install/remove software and update the system. Those utilities are there for people like you and me.

There is another advantage to installing through the Software Centre, the program's code has been assessed to make sure the program will not do anything nasty to the operating system.
