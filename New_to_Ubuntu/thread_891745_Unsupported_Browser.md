### Post by starcannon on 2008-08-16
If the website in question is using activeX, then you have a few options.

You can run a small windows install in a Virtual Box, this is the most robust and trouble free solution. Next you can run a dual boot set up, annoying because you have to reboot the computer to switch operating systems. And finally you can do as suggested and run IE4Linux, it "works" but not for everything or everyone, you may want to try it, and if the website you need works fine with it, then your solved, if not then one of the 2 other approaches would be your best bet.

Personally, I think VirtualBox is a great program, and for those rare occasions where it has to be IE or nothing, its very useful.

You can get VirtualBox here:[[COLOR="DarkOrange"][SIZE="3"]Sun VirtualBox[/SIZE][/COLOR]]("https://cds.sun.com/is-bin/INTERSHOP.enfinity/WFS/CDS-CDS_SMI-Site/en_US/-/USD/ViewProductDetail-Start?ProductRef=innotek-1.6-G-F@CDS-CDS_SMI"). If you get stuck getting it setup be sure to post back here, I or someone else should be able to help you get it working perfectly. Once the VirtualBox is working, then you just install windows onto it like you would an Actual PC, only in VirtualBox it goes much faster, smoother, and once installed Windows runs on VBox very fast and smooth as well.
