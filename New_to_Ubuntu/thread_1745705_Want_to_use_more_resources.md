### Post by yetiman64 on 2011-05-01
With the figure you quote of 19 % of RAM, I assume you found that in System Monitor. Is that correct? 

My figure of RAM use with 8 GB RAM usually shows at about 10 or 11 %. But this figure is actually somewhat misleading (it is the figure for program use only).

If you use the "free" command in terminal or Install System monitor in a panel drawer (and configure it to show as I have) you will see more RAM in use (used as cache) when the system is working on a task. See the attached screenshot of my system. 

It shows as 11 % in System Monitor, whereas at the moment (probably because Transmission is running) my real figure of RAM use is closer to 56 % (at times I can get it up to 98 or 99 % when the System is working really hard).

The "free" command used in terminal will give you a more accurate picture of how your RAM is used in Linux. I suspect you are already using more resources at times but just aren't aware of it.
