### Post by vanadium on 2019-04-04
You can't, unless you edit the source code. Alternatively, your best bet is to file this as an issue ("bug") to the developers of the Gnome Shell extension "Dash to Dock". The Ubuntu dock is based on that extension. I fully agree with you that the previews should, just like the docked icons themselves, be static (in order of opening the program), and not in "recently used" order.

### Post by vanadium on 2019-04-06
Yes, there are many. Alternatively, you can configure a bit. You could disable the Ubuntu Dock and either live without any dash on the desktop, or install an alternative Dock like plank, or install the Gnome Shell extension "Dash to Panel", which I already mentioned. The latter provides a layout resemblant to that of Windows, with everything in a bottom bar. With that extension, window previews remain in the same order, as you are looking for (and as they should, also in my opinion). Still alternatively, you could try a different desktop than Gnome to see if that suits your taste better. On Ubuntu, you can install the xfce desktop, the KDE Plasma desktop, the Budgie desktop, ... Separate editions of Ubuntu exist that come by default with another desktop, e.g. Xubuntu with xfce, Kubuntu with KDE, Ubuntu Budgie, ...
