Consider switching to **Lubuntu 10.10**.  It is a Ubuntu derivative and was actually developed to be used on "elderly" hardware.

I am currently using it on what I consider an elderly system, and it is working fine.  

It uses the same repositories as Ubuntu and there is a "*lubuntu-restricted-extras*" package to give you Flash and a lot of other good stuff.
