**STEP 1 – Delete the Partition**
a. Make sure you have the "Widows Reinstallation CD" or "Live CD" – we will need it later.
b. Go to Control Panel and open Administrative Tools.
c. Under Storage, choose Disk Management.
d. Right-Click the Partition that has Ubuntu on it.
e. Press "delete partition".
[B]
STEP 2 – Create new MBR (translation: be able to work with windows)[/B]
a. Reboot using the "Windows Reinstallation CD" or "Live CD".
b. Wait… (I know…)
c. The blue screen will ask you what you want to do:
    Type “R” for Repair
    (Don’t Panic! Nothing’s wrong, we just need that menu)
5. The blue screen will ask:
    “Which Windows installation would you like to log onto?”
    type “1”
6. The blue screen will ask for Administrator’s Password:
    just click Enter (unless you have one)
7. Type the magic word: “fixmbr”
6. Type “exit”
Windows will now restart automatically.
