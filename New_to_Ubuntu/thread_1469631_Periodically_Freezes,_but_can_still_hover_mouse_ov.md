### Post by HermanAB on 2010-05-02
Hmmm, umpteen reasons...

It can be a bad power supply, bad keyboard, blown capacitors on the motherboard, bad memory, inadequate cooling due to a seized fan or dirt in the CPU heat sink, a bad interrupt controller...

Try to boot with something else (eg BartPE or Knoppix) and see whether it works properly - if so then the problem is not hardware.

Open the case and look for distended capacitors - if there are, buy a new mother board.  If you have a spare PSU in your junk box, try it. If the case is dirty, use a vacuum cleaner.  If some fan doesn't work, get a new one...
