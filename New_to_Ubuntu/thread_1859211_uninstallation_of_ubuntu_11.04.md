### Post by cavh on 2011-10-13
I doubt very much you can do a downgrade to an earlier version - you'll have to get a copy of 10.10 and reinstall from scratch - if your home folder is on a separate partition then don't format that partition - and make sure you take a full backup of your home folder regardless.

By the way, if the reason for wanting to downgrade is because you don't like the Unity desktop, you can always choose 'Ubuntu classic' on the login screen to get the classic style Gnome desktop - that's what I do. That way you get all the goodness of 11.04 but accessible in the way you're used to.

### Post by kolinab on 2011-10-13
Hello again angelgog,

Unfortunately if you don't want to spend time waiting to download version 10.10, then I don't have good news for you. 

Do you have a fast internet connection? If so, it shouldn't take too long to download a copy of version 10.10 from Ubuntu.com. If you do that, it should be able to download all the relevant updates it needs while it is installing. 

Now there *might* be a way to downgrade it, but 1) I don't know what it is and 2) I'm pretty sure the experienced folks here wouldn't advise it. I think by downgrading (if possible) you may cause yourself more time and trouble and frustration than it is worth.

If I was in your situation and wanted to stick with 10.10 for now, I would download a fresh .iso of 10.10 and install it from scratch. If your connection at home is slow then maybe you could download it somewhere else you have a fast connection and put the .iso image on a USB stick using the 'startup disk creator.' There are instructions on how to do that on [http://www.ubuntu.com/download/ubuntu/download](http://www.ubuntu.com/download/ubuntu/download)

I'll go see where to download 10.10 (because it doesn't show up on the main ubuntu.com download page) and post it here when I get the link.

[edit]You can download 10.10 here: [http://releases.ubuntu.com/10.10/](http://releases.ubuntu.com/10.10/)  Remember that it is only officially supported until April 2012, but that should still give you plenty of time to decide if you want to move on to one of the newer supported versions.
