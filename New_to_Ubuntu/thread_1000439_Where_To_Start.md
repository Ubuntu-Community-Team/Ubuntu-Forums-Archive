### Post by Silare Alvreon VI on 2008-12-03
On a normal scale, I'd say "Congratulations on making the switch!" but the fact that you're being forced to convert isn't cool of the people who forced it on you. =/

Either way, congratulations on taking up the challenge. Not many would. Although still, being forced to use only one thing sucks (although at my school's case, it's Windows that the district is forcing us to use, even if the school is sorta' half defying it).

To install things, you have two choices:

The more familiar GUI-based way with clicking on buttons and the like can be done by Synaptic. At the top left, click on the System menu, click on Administration, and then click on Synaptic Package Manager. Since this involves installing things/removing things, you need to put in your password. It should start up.

Synaptic to put in the best Windows analogy I can think of, is a very very extensive Windows Update. You can install more than just what's on your computer, since it'll have 'packages' (which are the equivalence of Windows .exe files, though by no means do they run the same way) that you can download and install (ranging from Firefox to OpenOffice to Banshee Music Player [think iTunes-style], etc.). It's extensive. You won't be running around downloading .exe files from the Net as much, since Synaptic will do most of your work for you (although you may download .deb files here and there that are packages to install). Generally though, you will have 'repositories' (places that just house a bunch of packages) to download packages from.

To work Synaptic, you can click on the Search button at the top, type in a term like 'firefox' and it'll show you a list of things. If you're interested in installing them, read the description, click the box to the left of the name, and click to say you want to install it. It may ask if you want to install additional packages with that - do so, since they are dependencies that the package you selected will need to run. When you're done choosing what you want, hit Apply at the top and it'll download and install it all for you.

The Terminal way, which while is faster and later on at times potentially preferred, is a lot harder to get used to at first when you're new to Linux from Windows. If you're interested in learning this instead, make sure you have an idea of how all this packaging works.

Hope this helps.
