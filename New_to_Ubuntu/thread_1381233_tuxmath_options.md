### Post by s.fox on 2010-01-15
Hello,

RE:  /home/yourname/.tuxmath/options

.'s indicate hidden directories.

The .tuxmath is a hidden directory :D

In order to show it you will need to do  the following:

1)  Navigate to yourname

2) Then click View on the menu at the top.  You should see an option to show hidden files.

.tuxmath should now be shown and you will be able to open the directory and place options folder in the desired place.

Hope this helps,
