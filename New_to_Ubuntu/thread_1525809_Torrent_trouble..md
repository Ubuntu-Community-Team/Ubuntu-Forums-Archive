### Post by mikewhatever on 2010-07-07
The reason I thought space wasn't the issue is because had it been the issue you would have gotten a warning from Ktorrent. While I am on the subject, torrent clients don't cache files, when set to, they allocated disk space required to store the file, however, that doesn't double the space needed for downloading.
You seem to be convinced that there must be peers for that torrent simply because there were peers before, which obviously isn't the case.
