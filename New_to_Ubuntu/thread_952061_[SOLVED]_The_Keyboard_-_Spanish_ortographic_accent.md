Andá a System - Preferences - Keyboard, hacé click en la solapa "Distributions", elegí "Add" y seleccioná la opción correspondiente. Creo que es así. 

La Ayuda de Ubuntu dice:

Preferencias de distribución del teclado

Use la solapa Distribuciones para establecer el idioma de su teclado, y también para la marca y modelo del teclado que está usando.

Esto permitirá a GNOME usar las teclas multimedia especiales en su teclado, y mostrar los caracteres correctos para el idioma de su teclado.

Modelo de teclado

    Use el botón de visualización (etiquetado con una elipsis, ... para elegir otra marca de teclado y modelo.
Distribuciones seleccionadas

    Puede cambiar entre las distribuciones de teclas seleccionadas para cambiar los caracteres que produce su teclado cuando teclea. Para añadir una distribución, pulse en Añadir. Puede tener hasta cuatro distribuciones de teclas. Para quitar una distribución, selecciónela y pulse Quitar.

    	        

    Para cambiar entre distribuciones de teclas, use la miniaplicación del panel Indicador de teclado.

    	        
Separate layout for each window

    Cuando esta opción está seleccionada, cada ventana tiene su propia distribución de teclado. Cambiar a una distribución diferente sólo afecta a la ventana actual.

    	        

    Esto permite teclear con un teclado en ruso una palabra en un procesador de textos, después cambiar al navegador web y teclear con un teclado en inglés, por ejemplo.

Pulse Restaurar predeterminados para restablecer los ajustes de distribución de teclado a su estado inicial para su sistema y configuración regional.

### Post by fballem on 2008-10-20
With apologies to the other posters, in case they have answered the question. I don't speak Spanish.

I work in three languages, English, French, and Portuguese. When I'm working in French or Portuguese, I use the US International Keyboard with dead keys. I have found this is the simplest solution for a North American standard 105 key keyboard. My solution has two parts:

1. Set up an alternate keyboard. To start, select System > Preferences > Keyboard from the menu. You will get a screen similar to the attached Screenshot. Select the keyboard tab, then the + sign to add a new keyboard layout. I use the US International with dead keys that allows for proper accenting for most Western European languages. The remaining screens are self-explanatory. When you are done, your screen should look like the sample. Don't forget to designate one of the layouts as the default before you leave the dialog.

2. Add the keyboard indicator to your panel. To do this, select one of the panels (I use the top panel) and right-click on the panel. This will bring up a menu, from which you select 'Add to Panel'. This will give you a list of items that you can add to the panel. If you scroll down, you will find the keyboard indicator (see the attached screenshot). Highlight the keyboard indicator, then select the Add Button. You will then need to close the screen by selecting OK. This will add the keyboard indicator to the top (mine shows as USA). If you click on it, it will switch to the next layout (mine shows as USA2). Click again, and it will switch back.

When I'm typing, I normally prefer the standard US keyboard. When I'm working in French or Portuguese, then I will switch.

If you are familiar with the Microsoft US International Keyboard, this is similar, but not identical. For example to type é, you type the apostrophe key (') followed by the 'e' key. If you actually want the apostrophe, type the apostrophe followed by the spacebar. In the Microsoft keyboard, you would type ç by typing the apostrophe, followed by the 'c'. With this keyboard, you type ALT+comma. That's the only major difference I have found for the languages that I use.

There is an option to print the layout of the keyboard. I would recommend it until you're used to it.

Hope this helps,

Hello,

Have you tried:

1. press Alt
2. press Ctrl
3. press accent mark you want
4. Now press letter

So for example to get the € sign on a U.S. keyboard you would press:

1. press Alt
2. press Ctrl
3. press =
4. Now press letter e

Hope this helps
