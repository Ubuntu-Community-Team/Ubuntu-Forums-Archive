### Post by beew on 2010-08-01
Hi, 

I wonder if anyone knows of a good PDF reader in Linux that has the tab feature and probably can remember and restore the last session.

Here are some options I have tried and none are very satisfactory.

Acroread is out of the question, it is just a bloatware, I don't use it even in  windows. It is ridiculous that you have to install over 100 M just to read a pdf file.

Evince is slow as a slug in loading and doesn't have many features. Pros: it is an all in one reader that can read many file types : pdf ps djvu.. Cons: It is bad in  all of them (I have djview4 to read djvu files) 
It seems to have gotten faster since the latest update. For this reason alone that I am not getting rid of it, just to see if it makes progress in the next little while, but I am not using it as my default reader either and am hiding it from the context menu.

Xpdf is ugly and lacks feature, doesn't integrate well with firefox for inline viewing. Pros: kind of fast if the document has no picture. Cons: really basic.

I heard good things about Okular. But it is rather slow in  loading (like Evince) and still doesn't have tab reading. Moreover I don't want to install a big chunk of the KDE desktop just to read pdfs (though I can make it worthwhile by installing other KDE programs. :))

In the meantime I am using the Foxit Reader (linux version) as my default PDF reader. It is lighting fast, renders pictures well and integrate seamlessly into Firefox using after some tweaking with Mozplugger. I also like its interface.

The cons: It is a poor cousin to the Windows version, doesn't have most of the features of the windows version. It doesn't have tab reading feature and it cannot remember last session. It has some bugs in that it renders certain fonts incorrectly. The biggest blunder is that it cannot render its own manual properly, open it in Foxit Linux and all you see is gibberish. Finally, there has been no upgrade for a while even though version 1.2 was promised back in Sept 2009. Maybe the project is dead, like so many Linux spinoff commercial softwares. (Oh, some people may say it is not Free as in freedom, but I don't care)

I have tried using the Windows version of Foxit and Pdf Xchange viewer in WINE in order to use the tab and remember last session  features. It went ok except  a bit slow in starting because of WINE, but still perform better than EVINCE. The cons are obvious, I would prefer a native Linux application rather than using WINE,especially since viewing PDF is such a basic task, there should be good Linuix solution without having to use faked Window$.
