### Post by shai3421 on 2010-05-25
This is a way to get a program to automatically request root password when run from a GUI:

System -> Preferences -> Main Menu -> navigate to the program you want ->
choose Properties -> in the Command field, change it from "some program" to gksudo "some program"

This may be helpful to beginning users, so I decided to share this tip.
