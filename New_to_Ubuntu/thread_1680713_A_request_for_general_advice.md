### Post by Sealbhach on 2011-02-03
OK, well look through this thread here:

[http://ubuntuforums.org/showthread.php?t=210401](http://ubuntuforums.org/showthread.php?t=210401)

Might give you some ideas. Also, a fun thing to do is install tdfsb and see a visual representation of the files on your system. Try it out, it's amazing.

```
sudo apt-get install tdfsb
```

[Omgubuntu]("http://www.omgubuntu.co.uk/") and [Lifehacker Linux]("http://lifehacker.com/tag/linux/") also have some good and interesting stuff. Also, you need to have some familiarity with [this]("http://tldp.org/LDP/Linux-Filesystem-Hierarchy/html/index.html").
