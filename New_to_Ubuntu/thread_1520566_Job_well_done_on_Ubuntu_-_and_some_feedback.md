### Post by zrednaz on 2010-06-29
Hi Ubuntu community,

I am currently on day 2 after my switch to Ubuntu 10.04 and I want to congratulate the Ubuntu team on a job well done. For a person who had no prior experience with anything but Windows, I must say that the transition has been mush easier than expected. The documentation is excellent and all drivers (including the built-in webcam) for my Thinkpad T500 seem to work like a breeze so far.

Even so, I have a few suggestions to make the switch even easier for future users.

-Make absolutely sure that new users read the excellent Ubuntu documentation  [https://help.ubuntu.com/10.04/index.html]("https://help.ubuntu.com/10.04/index.html"), specifically "New to Ubuntu?". I say this because my initial instinct was to search the web for needed programs while I clearly should have resorted to the built-in Software Center instead. :p

-In Nautilis, I would suggest making it easier to have it default to the "Location bar" instead of buttons for navigation. Buttons instead of a simple address bar is one of the very things that made me change from Windows 7 in the first place. I've found a solution for this, but it wasn't very straight forward, and it should be added to the Nautilis options panel (like it apparently used to be).

-Resizing windows in Ubuntu is a pain. Grabbing the edge of a window with the mouse requires too much precision. In Windows it is much easier to grab the edge.

-The desktop icons are very big by default, and the fact that the default size of icons is changed from within Nautilis makes no sense. This should be moved to System -> Prefs -> Appearance.

Apart from these things, I am pleasantly surprised with how well the system is working so far and how simple it is to accomplish a wide range of tasks, such as configuring the firewall and compressing a set of files. Keep up the good work.
