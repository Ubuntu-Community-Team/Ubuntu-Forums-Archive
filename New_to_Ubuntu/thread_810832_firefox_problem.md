### Post by quelx on 2008-05-28
Open firefox and type **about:config** and promise to be careful.

In the filter type **warnOn** and make sure the options are set to **true** if not double click to change them.

close firefox (all of them) and try again

If this doesn't work it may be related to this bug which was recently fixed but probably not make it's way into Ubuntu yet

[https://bugzilla.mozilla.org/show_bug.cgi?id=419009](https://bugzilla.mozilla.org/show_bug.cgi?id=419009)
