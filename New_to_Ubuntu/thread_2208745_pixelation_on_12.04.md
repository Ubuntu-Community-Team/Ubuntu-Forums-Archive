### Post by Impavidus on 2014-03-02
Sounds like a graphics driver issue. Depending on the exact graphics card/chipset (which isn't the same for all HP omni 200s), you may be able to install proprietary drivers. However, doing so on the live usb won't survive a reboot, unless it has persistence, in which case it will complicate matters when you want to use the live usb on different hardware some other day.
