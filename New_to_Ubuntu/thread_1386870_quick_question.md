The easiest way is via Synaptic.

System --> Administration --> Synaptic Package Manager.
Then Settings --> Other Software

That will list what you have & allow you add / remove. Do be careful, tho' !!

Quite often for specific debian and ubuntu packages, they will include the repositry and give instructions as to how to add from from the command line.

There's a good write up over here --> [https://help.ubuntu.com/community/Repositories/CommandLine](https://help.ubuntu.com/community/Repositories/CommandLine)

### Post by 3rdalbum on 2010-01-21
System > Administration > Software Sources.

This is the safest way to add to your sources.list.

If you really want to you can edit /etc/apt/sources.list manually in a text editor, but if you make a mistake your package manager will stop working until you fix the mistake. Better off just to use the Software Sources program.
