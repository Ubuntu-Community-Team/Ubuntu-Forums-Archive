### Post by herbster on 2008-05-15
I'm guessing you're using nvidia-settings? Just run nvidia-settings as root:

```
sudo nvidia-settings
```

and then hit the Save to X Configuration File button, save the changes and they will actually be written to your /etc/X11/xorg.conf.
