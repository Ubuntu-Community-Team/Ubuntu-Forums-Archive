### Post by quinnten83 on 2008-06-15
> **satipera said:**
> Hello, my friend wishes to send me a large sound file 120mb which will not come through on email. What programmes do we have to run and how do we set it up?

I think the very first question would have to be:
- Are you and your friend on the same network, or are you on seperate locations and can only use the internet to communicate?

For the former, samba will work, for the latter, you need to setup an ftp server.
Your friend can set it up on his computer, and you can use an ftp client like filezilla to download the file.
google for setting ftp server in windows and let your windows friend do the setting up.
