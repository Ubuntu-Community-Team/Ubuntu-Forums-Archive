### Post by lisati on 2010-01-27
First of all, welcome to the forums.

I can't offer any specifics on getting LOTRO to work with Ubuntu, short of suggesting that you try it with "Wine" which can let you run *some* Windows programs on Ubuntu - see [https://help.ubuntu.com/community/Wine](https://help.ubuntu.com/community/Wine)

Another option is to consider using "Dual Boot" or "Wubi". This lets you run Windows when you want to, and Ubuntu when you want to. More information can be found at [https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot) and [https://help.ubuntu.com/community/Wubi](https://help.ubuntu.com/community/Wubi)

### Post by benmoran on 2010-01-27
Welcome to the forums. I second the dual-boot idea. A lot of us started out that way. You can take the time you need to adjust to the different way you do things in Ubuntu, while still being able to fall back on Vista if you need it. As time goes on you'll probably end up wanting to use Ubuntu 100% of the time, as a lot of us have. Those that do end up keeping Windows around, usually only do it because of one or two pieces of software that don't yet work in Wine (or those wanting to play the latest games). 


Most importantly, whatever you do, ask for help if you get stuck. I think you'll find that most Linux users are more than happy to help others, and will do so if they are able.

### Post by cpeake on 2010-01-27
I'm currently in the middle of a transitional process from Win XP and Win Vista, and currently I'm trying out 9.04 and 9.10.

I was a total Linux noobie when I first got involved, but luckily my Dad's quite an expert on on Ubuntu and Linux in general. We did the initial install together, but after that I've been pretty much on my own.

Here's some stuff I've noticed about Ubuntu:

+ Ubuntu in general is *a lot easier* than I thought it would be!
+ A big bonus for me has simply been that Ubuntu looks better than Win XP, but also better than Vista in my opinion.
+ Customisability is top class in Ubuntu, make the system look like you by customising themes and get all the programs that interest you.
+ Moving over from Windows I've had to look for program 'counter parts' only to notice, that the Linux version does everything a little better.
+ WINE has given me Spotify and other Windows -only programs to the Ubuntu side. Using WINE is simple: instead of double-clicking to open a program, just right-click and say "Open with WINE". Setup was automatic. Although I'm only exploring into using WINE at this moment.
+ A lot of programs integrated into the OS seamlessly, making their use easy and you get to start out with a very well set-up OS. Unlike Windows, where you have to get *everything* seperately.
+ If there's any field of software that you use to a deeper extent, say audio related programs like I do, you'll find that a lot of programs top those commercial programs on the Windows side, but they are *Free!*

There's a lot of other things I like about Ubuntu, but yes, some games seem to require that they be played on the windows side, and there isn't much that can be done about that.

Therefore I think I will be sticking with a dual-boot. At the moment I'm keeping Windows around only to play those games I haven't (yet) figured out how to play in Ubuntu.
