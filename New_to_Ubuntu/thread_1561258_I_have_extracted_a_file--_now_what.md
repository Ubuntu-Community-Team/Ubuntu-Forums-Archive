### Post by DavidOfLondon on 2010-08-26
Take my advice - download the Ubuntu Manual and read it from start to finish.

Every question you are going to ask over the next few weeks is either answered in the manual (use the Find option to skip to relevant part) or the manual with help you to google a more specified question.

What you have to get used to is the fact that Linux programs don't always have an interface (the pop-up screen with buttons for you to click) but rather the Terminal (Applications>Accessories>Terminal) is where MANY programs are booted from or used. 

You need to very quickly get used to not thinking in Windows terms. Every user of Linux has to, as did I. It looks daunting but once you've figured out how files are saved, how the Terminal operates, what the package manager is for etc life becomes much easier.

Linux OSs are free and that means a bit more homework to get it going.

But once you've read that manual you will smile and get on with it - it's a fascinating world, is Linux.

28 seconds to load up a beautiful, functioning, peronalised operating system?! 

I'll take that over a constantly crashing, insecure, slow, virus-riddled Microsoft any day.

Read That Manual :)

David.
