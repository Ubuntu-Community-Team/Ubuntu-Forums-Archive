### Post by Futant on 2017-08-15
fsvs isn't what you're looking for, the package description is[COLOR=#000000]:[/COLOR]>  FSVS is a backup/restore/versioning/deployment tool for whole directory
 trees or filesystems, with a subversion repository as the backend.
 It can do overlays of multiple repositories, to achieve some content
 separation (base install, local modifications, etc.)

The File System Virtualizer you mention is very old (2001), you could try it but it probably won't work.
