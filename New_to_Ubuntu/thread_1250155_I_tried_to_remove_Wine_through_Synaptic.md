### Post by NoaHall on 2009-08-26
Synaptic is the same as Add/Remove, just better for more advance people.

Open a terminal and type "sudo apt-get remove --purge wine" - that should get rid of anything left, if you did it wrong.

iTunes doesn't work on Wine - except maybe a old version. Try songbird or aTunes instead.

### Post by embiggened.badger on 2009-08-26
Indeed, iTunes support for Linux is virtually non-existent. I've always found that to be rather ironic, since Mac OS X is built around the same Unix essentials as pretty much any other 'flavor' of Linux, but I digress.

I would recommend [**Amarok**]("http://amarok.kde.org/"). It's well-known among Linux users for its ease of use, and it offers a lot of support for iPod file transfers, proprietary audio codecs (e.g. AAC used by iTunes), and a ton of features. I installed it myself using the Add/Remove feature and haven't had any complaints.
