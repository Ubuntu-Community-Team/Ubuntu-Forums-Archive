### Post by Old_Man_Unix on 2011-04-01
LibreOffice is a fork from OOo and I would be very surprised if they  messed with the file formats - they would have absolutely no reason to.

In Open Office, if  you use a MS font and you save your document as a . DOC file, it is essentially the same thing as a file saved from MS Word with the same font.  No program can tell the difference.

I have used files from both OOo and MS Word  for years because at different times I have had access to both of them.  I have to say that at this point, there is no way for me to say *exactly where* some of my files originated.  No problems. 

I have downloaded templates from the MS Office website and used them in Open Office, and vice versa. 

I also have saved .DOC files from OOo and  submitted them  to various organizations  - some of which are*** very*** particular about file formats - and have never had any problem about the format.  

So I think that something else is going on here that has nothing to do with file formats or Linux. 

A .PDF file is another thing entirely.  It IS different from a .DOC file.  Many organizations do not like PDF files and will not accept them. 

As far as hard copies and OCR is concerned, once a file is printed to a hard copy,  there is no record in the hard copy as to what format the original file was in.  A hard copy is just marks on a piece of paper - what you see is what you have.

CrossOver has nothing to do with file formats as such.
