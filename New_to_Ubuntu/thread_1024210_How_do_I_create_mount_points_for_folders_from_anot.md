### Post by AClark on 2008-12-28
I used this guide on setting up NFS for networking between my Ubuntu boxes.

[http://ubuntuforums.org/showthread.php?t=249889](http://ubuntuforums.org/showthread.php?t=249889)

A mount point is just a folder that you create and then designate as the mount point when you mount the folder from the remote machine.


FYI if you create the mount point in /media an icon will appear on your desktop when you mount a network share to that mount point.

Hope this helps.
