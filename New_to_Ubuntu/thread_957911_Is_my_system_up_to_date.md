### Post by Nxion on 2008-10-24
What is it that you are wanting to update?

Lets see what kernel you have, type this in the terminal:

```
uname -r
```

Also this:

```
lspci
```

And this so we can see if you have your video adapter configured.

```
cat /etc/X11/xorg.conf
```

Can you please post the output of the above commands? Also what version of Ubuntu are you using?

### Post by bodhi.zazen on 2008-11-09
You should have an applet on your panel that alerts you if updates are available.

If no you can either open synaptic, reload, mark all upgrades -> apply

Or open a terminal and enter

```
sudo apt-get update
sudo apt-get upgrade
```

If any updates are available they will be installed.
