### Post by starcannon on 2009-07-30
Kompozer is what you likely want; its in the Synaptic Package Manager.

System>Administration>Synaptic Package Manager: Search: kompozer

Or open a terminal:
Applications>Accessories>Terminal
and put in this command:
```

sudo apt-get install kompozer

```
Either way will install the same program.

 Enjoy
