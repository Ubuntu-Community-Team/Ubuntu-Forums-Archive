### Post by grahammechanical on 2016-08-19
Was Ubuntu installed with the machine connected to the monitor by HDMI/VGA connection? Is the monitor a digital monitor or a CRT monitor. 

System Settings>Screen Display has an option "Detect displays." That is for when we are running on an open source video driver. If we have installed a proprietary video driver then there should be a settings utility that will also have the function "Detect Displays."

### Post by HereInOz on 2016-08-20
This looks a bit like the HDMI to VGA adaptor is not sending the necessary information for the screen to bring itself live, so it is staying in sleep mode.  By going to TV first, then switching to VGA, you may find that this is bringing the monitor up and then leaving it up so that the VGA signal is visible.

Just a guess, but if this is the case, then the adaptor is probably at fault - see above post re crap, garbage and platinum.
