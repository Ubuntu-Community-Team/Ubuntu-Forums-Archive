### Post by bludgard on 2010-12-31
Ok.
 
Open ImgBurn.
 
Select "Create image file from files/folders".
 
On the next screen under "Source",select the second icon-"Search for a file...." and navigate to your .rar file.
 
Under "Destination",for convenience sake,choose to save to desktop.Click the "File to .ISO" icon at the bottom.Done.
 
You should be able to burn the image to disk with either the native Windows image burner or ImgBurn.
 
You could burn the .rar file directly to disk,however,if something goes wrong,you have a wasted disc.I think it safer to burn from image file.
 
Hope this helps.If you need further assist,don't be shy.
