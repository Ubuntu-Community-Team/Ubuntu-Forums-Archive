### Post by grahammechanical on 2011-06-24
At the log in screen click your user name and then look at the bottom panel and you will see three menu boxes. In the right hand box there will be a list with Ubuntu classic (without Unity) listed. If Ubuntu is listed, then that is Ubuntu with Unity. It will only be in that list if you activate the additional drivers for your system. There is also an option of using Unity 2D. It is installed from the software centre. It gives the Unity method but without the 3D look. It is for systems that really do not have the hardware to run Unity 3D. Once installed it will appear in the menu list at log in time. At the moment the installation defaults to loading Ubuntu Classic. Which is what has happened to you and also to me and all of us.

Regards.

P.S. I have just looked at the screenshot. That is Firefox. I thought you were talking about the Ubuntu desktop. You have set the google site as your home page. To change that page or any home page you will need to edit the page in a web page editor and then set it as your home page. In other words, create your own home page.

Also I see from the screenshot that there is an icon in the top panel inviting you to activate additional drivers. Click on it and activate the drivers.
