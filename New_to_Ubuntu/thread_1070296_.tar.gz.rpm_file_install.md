### Post by Temposs on 2009-02-15
Alien is very easy to use.

To install:

```
sudo apt-get install alien
```

To use:

```
alien printerdriver.rpm
```

This will produce a file called printerdriver.deb, which you can double click on to install.

Alien has generally worked well for me, though some warn that since this is doing a conversion, you're not guaranteed a perfect install.

### Post by x33a on 2009-02-15
bradtek is right .rpm are for red hat based systems.

all these commands should be typed in a terminal

to install alien:
```
sudo aptitude install alien
```

to extract from a .tar.gz, 
```
cd into the directory containing the file

type "tar xzvf filename.tar.gz"

then type "ls -l", this will post the contents of the directory.

cd into the extracted directory.

type "sudo alien -k filename.rpm", it'll be converted into a deb.

then type "sudo dpkg -i filename.deb", this will install the driver.
```
