There is no way to revert back; instead, you have to do a clean reinstall of 12.04.

Like I said before, unless you're having problems with bugs and/or want to use Ubuntu for a while without having to upgrade, you should stick with 12.10. It has some useful features that the previous release doesn't. 

Like Mark Phelps said, there is no way to revert back. That's (in a way) like reverting from Windows 8 back to 7. It's a big change that would require a lot of unnecessary work to make it possible.
