### Post by rrashkin on 2012-04-24
It looks to me like you file-copied the .iso file to CD rather than using the iso recorder method.  Is that true?  If so, it won't work as a liveCD.  Otherwise, just put the CD in and re-boot.  When you get a chance to specify the boot device (you might need to edit the BIOS or hit like F12 or ESC to get to a menu of boot options), select CD-ROM.
