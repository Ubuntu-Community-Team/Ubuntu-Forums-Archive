### Post by Morbius1 on 2017-12-31
It's true that the kernel in 17.10 changed the default smb dialect from 1 to 3 so if that is why you got the original error message this is the wrong syntax:
> //openwrt/sda1  /media/sda1  cifs  guest,uid=1000,iocharset=utf8  0  0  vers="1.0"
Try this instead:
```
//openwrt/sda1  /media/sda1  cifs  guest,uid=1000,iocharset=utf8,vers=1.0  0  0
```
No quotes. Not after that last "0 0".
