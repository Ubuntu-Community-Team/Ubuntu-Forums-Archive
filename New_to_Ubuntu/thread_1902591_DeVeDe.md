### Post by coffeecat on 2011-12-31
> **Eastwood1 said:**
> Clicking 'Add' has no effect.

Which add button did you click on? The left one under Titles adds a title. Click the right add button under "Files" and that should open a "File Properties" window. Click on the File bar near the top and you will be able to browse to your video files and add one.

> **Eastwood1 said:**
>  Clicking on' file 'and 'open' lets you browse but no way can I add the files!

File -> open is to open a *.devede disc structure file that you've already created and saved.

### Post by crazy bird on 2011-12-31
> **3rdalbum said:**
> A lot of people install DeVeDe, not realising that the default CD burner, Brasero, can make basic video DVDs too. Give it a try and see.

Not a good hint by the way, advising Brasero since Brasero is very well known for all sorts of problems and issues. Check this forum and you see what i mean. 

With Devede you can create an .iso file containing the video, which then can be burned with a program such as K3B or Xfburn. Xfburn is a lightweight burning tool where K3B is a burning tool with more options and configuration options, more heavy weighted compared to Xfburn.
