### Post by rjonnal on 2011-06-29
There are *not* 'a ton'. There are a few, though.

[http://www.matthewjmiller.net/howtos/create-a-curriculum-vitae-using-latex/](http://www.matthewjmiller.net/howtos/create-a-curriculum-vitae-using-latex/)
[http://phaselockfel.blogspot.com/2009/03/prepare-resume-using-latex-and-bibtex.html](http://phaselockfel.blogspot.com/2009/03/prepare-resume-using-latex-and-bibtex.html)

The first one uses the multibib package. The second one is straight BibTeX, and allows only one section of references in the CV.

I developed my own technique, which is detailed here:

[http://pointsofsail.org/wikka.php?wakka=LatexCV](http://pointsofsail.org/wikka.php?wakka=LatexCV)
