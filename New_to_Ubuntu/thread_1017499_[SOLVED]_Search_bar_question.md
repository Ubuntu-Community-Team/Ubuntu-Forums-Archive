### Post by ahood on 2008-12-21
Good question.

After a bit of searching google, I found this article [**[COLOR="Sienna"]Hey Firefox - Let Us Pick Our Own Search Engine![/COLOR]**](https://addons.mozilla.org/en-US/firefox/browse/type:4), which leds me to believe that, technically, this is a Firefox 3 issue, rather than specific to the Ubuntu OS.

You probably discovered that adding additional search engines can be done via the [**[color="Sienna"]Firefox Add-ons[/color]**](https://addons.mozilla.org/en-US/firefox/browse/type:4) website. 

To add a specific search engine not available in the add-ons, it seems the  [**[COLOR="Sienna"]Groowe[/COLOR]**](https://addons.mozilla.org/en-US/firefox/addon/534) add-on will enable this feature. I haven't used this add-on, so I don't know how well it works.

I hope this helps.
