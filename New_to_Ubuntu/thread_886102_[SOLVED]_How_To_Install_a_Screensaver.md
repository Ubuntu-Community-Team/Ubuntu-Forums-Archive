### Post by sujoy on 2008-08-11

Description:
Alternative lock dialog for Gnome Screensaver. Unpack and copy each file (4) to /usr/share/gnome-screensaver.

Set key "lock-dialog-theme" to "turbolence" on gconf, /apps/gnome-screensaver


so try that :)
