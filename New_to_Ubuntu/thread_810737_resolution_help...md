### Post by SteveNorman on 2008-05-28
If you copy your original conf file to a file named xorg_backup.conf:

```
cd /etc/x11
cp xorg.conf xorg_backup.conf
```
to get it back after you messed with it just do
```
cp xorg.conf_backup xorg.conf 
```

to copy it back, Or print your xorg.conf file and return it back to normal afterwords. I always comment what I did so I can fix it later.

Comment like this:

# this is a comment
# this is another comment

the # tells the computer to ignore the text after it on that line.
In my example I added the comment:
 #fa mod
to tell me what I did to the file so I can find it later
