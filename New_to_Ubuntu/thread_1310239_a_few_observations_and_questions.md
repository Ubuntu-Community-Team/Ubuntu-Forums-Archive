### Post by paul_in_london on 2009-11-01
You don't need to use the command line all the time or even often if you don't want to.

It's just that it's usually a lot easier to try and help a poster by giving a command (or set of commands) that they can copy/paste than attempt to explain how to do the same thing by point and click.

The terminal also tends to give feedback if a command didn't work, which GUIs sometimes don't.

Have a look at this:

[http://www.ubuntupocketguide.com/](http://www.ubuntupocketguide.com/)

The pdf is free to download.

HTH

(Oh and there are various desktop environments, window managers, themes, backgrounds etc. to choose from. Everything's very customisable, nothing's hidden, you're in control).
