### Post by f37u5g0d on 2008-06-10
Amarok is a great alternative to iTunes.  It has a crossfade feature but I haven't tested it with .m4a  Althought I do know that amarok plays .m4a.  It also has excellent playlist handeling.  As for real "mixing" as you call it you probably already found the best software out there (mixxx).  Since you're considering linux you should probably consider switching your music collection too.  .flac is lossless, completely unrestricted (unlike .m4a), and has a pretty good compression ratio.

### Post by markbuntu on 2008-06-20
Try djplay for djing. 

The gmusicbrowser has support for huge collections:

Uses gstreamer, mpg321/ogg123/flac123 or mplayer for playback. It has easy access to related songs (same artist/album/title), supports multiple genres per song, ratings, and customizable labels. The windows layouts are completely cusomizable, it uses dynamic filters with unlimited nesting of conditions, has a powerful mass-tagging dialog ...

You should really just try add/remove in the menus. If you are going to use you computer for live djing I reccomend you get Ubuntu Studio or at least the rt (real time) kernel. It will put cpu priority on the real time needs of the sound playing/mixing etc apps.
