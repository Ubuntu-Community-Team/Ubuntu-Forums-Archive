### Post by Ricx94 on 2012-01-08
since that processor supports 32-bit or 64-bit, it comes down to the amount of RAM the computer has (or will have). If you want more than 4GB of RAM, then 64-bit. if 4GB or less, then either will work. i believe Ubuntu still recommends 32-bit, so maybe go with 32 if you will have 4GB or less RAM

### Post by KdotJ on 2012-01-08
People in the past have always bad mouthed the 64bit versions (especially as the website has recommended the 32bit version) but I've never had any problems with any 64bit version. As mentioned its only worth it if you have (or plan to have) more than 4gb of RAM. 

If you opt for the 64bit version, some apps will require some 32bit libraries to be installed but this isn't an issue, you can install them easily via the repos.
