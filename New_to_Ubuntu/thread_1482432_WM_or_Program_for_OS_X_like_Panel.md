Yes there is:
[http://code.google.com/p/gnome2-globalmenu/](http://code.google.com/p/gnome2-globalmenu/)

It puts the menu bar for GTK applications in the panel.
It doesn't work with Firefox or OpenOffice, but "true" GTK applications work fine.
