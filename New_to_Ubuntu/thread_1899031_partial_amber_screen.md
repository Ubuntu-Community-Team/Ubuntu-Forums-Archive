### Post by r-senior on 2011-12-23
Hmm. Half amber screen sounds like the preview you get with the Grid plugin. The idea is that if you want a window to resize to a preset size, e.g. the right half of the the screen, you drag the titlebar to the edge of the screen. You get an amber-coloured preview of the size and when you let go it assumes that size.

Do you get this all the time? Or just when dragging windows around?

Perhaps you could try disabling the Grid plugin? You'll need CompizConfig Settings Manager installed, which you can find in the Software Centre. The Grid plugin is right down at the bottom.
