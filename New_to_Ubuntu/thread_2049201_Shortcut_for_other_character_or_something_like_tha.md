### Post by LewisTM on 2012-08-28
You can add the Esperanto keyboard layout in Xubuntu.

If you haven't done so, install the xfce4-xkb-plugin package and add the Keyboard Layouts plugin to once of your panels.

Right-click it, choose Properties, add a new layout and pick Esperanto.

You can switch to Esperanto with a keyboard shortcut or by clicking on the plugin. You might have to find the corresponding keys on your keyboard. On a QWERTY keyboard &#349; is q, &#285; is w, &#265; is x, &#309; is [, &#293; is ], etc.

If you want to keep the US keyboard you can get by with the [compose key]("http://www.hermit.org/Linux/ComposeKeys.html"). Set the compose key in the xkb plugin. To make a &#285;, press the compose key (don't hold it) then the circumflex ^ (Shift-6) then g. Same goes for other letters.

Cheers!
