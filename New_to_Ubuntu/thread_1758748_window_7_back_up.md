### Post by Mark Phelps on 2011-08-29
To do it the Win7 way ... enter Backup in the start area.  You should then see an entry for Backup and Restore.  When you select that, you'll see a Control Panel page, with Create System Image on the left -- select that.

OR, you could download the FREE version of Macrium Reflect from their website, install it, make a backup using that, use the option to create and burn a Linux Boot CD.  Then, you have a backup you can restore later.
