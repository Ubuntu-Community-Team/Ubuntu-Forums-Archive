### Post by mtgmutton on 2008-08-21
[http://www.thelinuxsociety.org.uk/content/reinstall-grub-using-live-cd](http://www.thelinuxsociety.org.uk/content/reinstall-grub-using-live-cd)

This person describes it well. better than me at least :D

As for the XP error, it really depends on the error itself... Oftentimes (if you can't boot xp) you can fix it with a windows disc. You could go into the install dialogue on the disc (it won't actually install unless you say so) and highlight your partition, then press "r" and it will repair, which for windows means reinstall. Then you can do a system restore to go back to before the error. I suggest backing things up regularly, daily if possible. If your files are backed up you can just reinstall, copy your files and be done with it :)

If you can boot (or if you boot into linux), try googling the error. GOOGLE KNOWS ALL :D Plus if you post the error here, someone will most likely be able to help.
