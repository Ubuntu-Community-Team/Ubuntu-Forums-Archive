### Post by ad_267 on 2009-08-28
When you download it you can extract the CoverGloobus directory to anywhere you like. Then to make it start automatically when you log in go to System - Preferences - Sessions, click Add, then for the command enter the full path to the CoverGloobus.py file (this is case sensitive), eg:
```
/home/yourname/somedirectory/CoverGloobus/CoverGloobus.py
```

You will also need to edit the CoverGloobus.cfg file to set your media player and the theme you want to use.
