### Post by Paqman on 2010-06-27
There a few ways:

[LIST=1]
[*]Navigate to that folder and hit Ctrl-D to bookmark it
[*]Drag the folder into the bookmarks list on the left side of the file browser
[*]Go to Bookmarks > Add bookmarks in the file browser
[/LIST]
