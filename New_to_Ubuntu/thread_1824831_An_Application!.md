### Post by Frogs Hair on 2011-08-14
If you are using Unity the window list and the bottom panel functions have been removed and replaced by the indicator marks on the launcher panel . The trash and workspace switcher are on the launcher also .

If you are using classic gnome the window list is still included . If it doesn't appear right click the bottom panel , select add to panel and from the menu add the window list .
