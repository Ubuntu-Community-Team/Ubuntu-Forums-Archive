### Post by civillian on 2008-09-09
I had this issue with floola when I started using it, and to the best of my reccolection, the way I fixed it was to sync it to a PC with iTunes, then unmount it from there, and then open it again using floola.

If this doesnt work/you can't do it for lack of a pc that can run iTunes, you might also try the support forums at the floola website [http://www.floola.com/modules/newbb/](http://www.floola.com/modules/newbb/)

Once you get past this hurdle it should work fine (if I remember correctly) I still use it now with my 80GB Classic, and its superb, so persevere :D
