### Post by Mark Phelps on 2011-11-16
Don't mean to be discouraging ... but to be honest, the MORE you need Ubuntu to look (or behave) like MS Windows, the more disappointed that you are going to become.

Why?

Because Linux is not a free VERSION of MS Windows; it is a free ALTERNATIVE to MS Windows.

While true, if you use some of the same apps in Ubuntu that you did in Windows (good example being Firefox or Chrome web browsers), you can get an experience that is very close to what you have in Windows. But even then -- it's not identical.

When you then start trying to replicate Windows-only stuff (examples being Internet Explorer v9 or MS Office 2010) you very quickly hit a brick wall where this is simply not possible.

Happines in the Linux community comes from a willingness to learn new ways of doing things. Folks that are SET in their existing Windows ways of doing things are setting themselves up for frustration and disappointment.

You already said your wife finds "any changes frustrating" (I know how this is -- really!). She's going to have to make SOME changes to use Linux.  How frustrated she's going to become waits to be seen.  Just don't be surprised if she gives up early on and wants Windows back!
