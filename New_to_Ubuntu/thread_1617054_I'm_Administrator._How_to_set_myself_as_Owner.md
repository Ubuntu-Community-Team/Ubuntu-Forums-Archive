### Post by Zzl1xndd on 2010-11-08
For Files labeled as root as the owner you, shouldn't need to alter them, however if you do, I normally use gksudo nautilus. This will launch a Window as the with root privileges and allow you to access them

If you are using the command line you can add sudo to the beginning of the command to get root access.

It is possible to run a root, however it is not recommended.
