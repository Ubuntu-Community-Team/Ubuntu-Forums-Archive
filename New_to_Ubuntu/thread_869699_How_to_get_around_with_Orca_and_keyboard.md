### Post by pauper on 2008-07-26
I'm not sure if this any help, because I don't quite understand from your post
whether that person needs keyboard shortcuts to panels under Gnome or a way to
configure shortcuts for the same purpose in Orca.
[http://live.gnome.org/Orca](http://live.gnome.org/Orca)

Here is how it's set up in Gnome.

[color=blue]System--Preferences--Keyboard (Shortcuts)[/color]    

**Desktop tab:**

"Show the panel menu" -- *access top panel.*         
Alt+F1 or "Super L"*, then press arrow keys.
*--that's the key with Windows logo, next to left Alt key.

**Window Management tab:**

"Move between windows with pop up" -- *access window tabs on bottom panel.*
Hold down Alt key and tap Tab.

"Move between panels and desktop with pop up" -- [i]access top, bottom, desktop
panels (not necessarily in that particular order).[/i]                                           
Hold down Ctrl+Alt tap Tab key once--desktop, twice--top, thrice--bottom; then
press the arrow keys (but arrow keys only work for top panel).
       
"Move between panels and desktop immediately" -- (see above), [i]without pop up
window.[/i]
Hold down Ctrl+Alt tap Esc key; then press arrow keys.
