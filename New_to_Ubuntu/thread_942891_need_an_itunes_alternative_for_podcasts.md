So perhaps look in Itunes somewhere for the URL of that podcast's RSS feed and try importing it into Rythmbox.

### Post by ChildOfMana on 2008-10-09
Banshee supports podcasts. Go [here](http://banshee-project.org/download/) to download the latest version of banshee - full installation instructions can be found on the site.

In Banshee, right click on *Podcasts* in the left hand pane and select *Subscribe to Podcast*. Now just paste the URL for the RSS feed in the box, select the frequency for checking for new versions and et voila!
