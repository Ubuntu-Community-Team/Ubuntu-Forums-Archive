### Post by coolgamer512 on 2008-12-27
This tutorial is meant for people that are thinking about downloading Ubuntu Studio. New users to the regular Ubuntu should also read this. If you already have the regular Ubuntu, find this (--/INSTALLED\--) by using crtl-F and copying that into the text box that appears at the bottom of the screen.

Ubuntu Studio is a variant of Ubuntu that is aimed towards multimedia creation. It's a really cool version of Ubuntu. However, it does not include some of the basic programs that the regular version of Ubuntu includes, and it does not receive as many updates. I found a way to get the advantages of Ubuntu while still having the programs that Ubuntu has to offer. On top of that, you just need to burn a CD, not a DVD.

First thing you want to do is download the regular version of Ubuntu off of ubuntu.com. You may be wondering how this is going to get you Ubuntu Studio, but if you read the rest of the tutorial, you see how. After the iso downloads, use an iso burner (such as MagicISO or PowerISO, the free trials of both should work) to burn the iso onto a CD. A DVD isn't necessary. After the disc is done burning, keep it in your disc drive and reboot your computer. When it boots up, go into the BIOS and make sure that the disc drive(s) has the highest priority out of all of the boot devices. This will make sure that your hard drive can't boot up the already installed OS before Ubuntu can come up. When the Ubuntu screen appears, go into live CD mode (top of the menu), because it's much easier to install that way. After the desktop appears, get used to the OS a little bit, then when you're ready, double-click the install icon on the desktop. A window will come up. Just follow the instructions, and use the slider set the partition space for Ubuntu rather low compared to your current OS. For my 500 GB hard drive, I set Ubuntu to have a 100 GB partition. You can set it higher if you find yourself using Ubuntu more than your other OS. It will take a very long time for the partitioning to work, so be paitent. Once that's done, follow the rest of the instructions, and Ubuntu will be installed into the new empty partition, then Ubuntu will ask you to restart your computer. Restart the computer when prompted, and then you will boot into Ubuntu by default. Don't worry, your other OS isn't gone, you can just use the arrow keys to select it in the GRUB bootmenu (seen upon booting up) and press enter to go into that OS. But Ubuntu will be the default. 

Now that Ubuntu is installed, let's let the people that already had Ubuntu installed fin this spot.

(--/INSTALLED\--)

Now you have a regular Ubuntu OS. To get all of the features that Ubuntu Studio includes, but keep the advantages of having the regular Ubuntu, go into the system menu at the top of the screen, mouse over administration, and then select Synaptic Package Manager. Type in your password, then when Synaptic comes up, type ubnutustudio in the quick search. You should see about 16 items on the list. Check them all of, and Ubuntu will also add a few other stuff for you. When everything on the list is checked off (make use you scrolled down to see everything), click apply. It can take take anywhere from 30 minutes to a few hours, so find something to keep yourself occupied. Remember, you can use everything else in Ubuntu during the process. After everything is downloaded, everything will be installed, and then you will have all of the programs that Ubuntu Studio has to offer. But if you want your Ubuntu setup to look and sound like Ubuntu Studio, there's a few more steps. First, go to the system menu, mouse over preferences, and go into appearance. The themes tab should be the first one you see. Go down the list and select UbuntuStudio. Next, go back into the system menu, mouse over preferences again, and go into sounds. Then select Ubuntu Studio as the sound theme. Finally, go into the system menu yet again, but this time mouse over administration, and go into Login Window.go into the local tab then scroll down and select UbuntuStudio. Now, at long last, you have the best of both worlds.

So there you have it. It's Ubuntu Studio with more frequent updates and all of the features in the regular Ubuntu.
