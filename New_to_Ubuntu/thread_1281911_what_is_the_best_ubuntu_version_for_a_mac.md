### Post by Marlonsm on 2009-10-03
The newest (non test version) one, just like for any other computer. Right now it's Ubuntu 9.04. And soon we'll get 9.10.

But if you need a very stable system, I'd recommend 8.04, as it's the long-term-suport version.

Deciding between Ubuntu, Kubuntu or Xubuntu is up to you, as it's a matter of personal preference.

BTW, i've never installed Ubuntu in a Mac, so I'm not sure how will it work.



Also, you can also consider installing Linux Mint, that is based on Ubuntu, can do pretty much everything Ubuntu does but is even easier to install and set up.

### Post by 3rdalbum on 2009-10-03
Always use the latest stable version unless you have particular reasons not to.

Also I wouldn't recommend Linux Mint. Upgrading from one LM release to the next is risky and until recently it couldn't even be attempted. Ubuntu can upgrade from one release to the next quite reliably, so this will save you time in the long run, especially as it only takes an hour to get an Ubuntu system set up 100%.
