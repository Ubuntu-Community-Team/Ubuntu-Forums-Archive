### Post by pricetech on 2009-10-06
Use Terminal Server Client.  Point it to the IP they give for MAC users.  I connected, just didn't have any credentials.

The KB.... file they want winders users to download is an update to the RDP client in winders.

The file for MAC users is undoubtedly an add on for OSx.

If Terminal Server Client isn't already installed it can be installed using Add / Remove under Applications, but I think it's already installed by default.  terminal Server Client appears under Applications - Internet.

I'm using Hardy, but I don't think it moved in Jaunty.
