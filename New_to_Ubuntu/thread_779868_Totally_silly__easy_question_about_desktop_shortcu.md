### Post by Thingymebob on 2008-05-03
for a trash can on your desktop,
right click the desktop:
select create launcher
in the command Type:
nautilus trash:///
give it a name and choose your icon.

---

### Post by Paqman on 2008-05-03
What Windows calls a shortcut, Ubuntu calls a launcher. Right click on the desktop and "create launcher".

---

### Post by RiceMonster on 2008-05-03
For your home folder, do the same thing but instead of "nautilus trash:///", type "nautilus /home/yourusername"

Making desktop shortcuts for applications is a lot easier. Go to the applications menu, right click on the application you want, and select "add to desktop".
