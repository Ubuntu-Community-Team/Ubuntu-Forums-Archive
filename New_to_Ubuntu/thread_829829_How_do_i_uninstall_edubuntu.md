### Post by Stefanie on 2008-06-15
you mean you want to upgrade from 7.10 to 8.04? you can do that with the update manager, but it's impossible to "downgrade" afterwards. 
Maybe you can make a copy of your current partition with partimage? If things go wrong with your 8.04 upgrade you can restore the partition afterwards. 
to use partimage you should boot the live cd. detailed instructions can be found here: [http://www.psychocats.net/ubuntu/partimage](http://www.psychocats.net/ubuntu/partimage)

### Post by Eisenwinter on 2008-06-15
Unfortunately, you can't simply uninstall it - you will have to install another operating system on it.

Or, you can create 2 partitions on the disk, install 2 operating systems, and just format the Edubuntu partition. This will cause permanent loss of ALL data on Edubuntu though, so back up important thing before you do it.
