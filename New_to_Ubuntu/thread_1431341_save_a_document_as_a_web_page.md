### Post by mcduck on 2010-03-16
Yes, you can. Just set the output format to "HTML Document (.html)" in the Save File-dialog.

edit: If you want XHTML then use the "Export"-option from the File-menu and set the output format to "XHTML (.html; .xhtml)"
