### Post by ajcham on 2009-01-17
You still have the top panel, yes? Right-click it, and select New Panel.  Put your new panel in position, then right-click and select 'Add to Panel'.  You want the Window List applet for the taskbar.  If I remember correctly, the default panel also includes the Show Desktop button and the Workspace Switcher, so you will want to add these too.
