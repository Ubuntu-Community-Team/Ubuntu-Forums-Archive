### Post by Brian0312 on 2011-06-06
I'd go to the software center and search on the word restricted.
 
Then install the Ubuntu Restricted extras.  It'll add all of those codecs you need to play DVD's, flash, etc.
 
Ubuntu generally won't play DVD's out of the box unless you install it.

### Post by Catharsis on 2011-06-08
You should open up your DVD in your file manager (Nautilus) and check its contents. One DVD I have is only usable with Windows -- there is a specific program on the DVD that auto-runs and plays the DVD, and that program is for Windows (only, naturally).

If any of that doesn't make sense let me know and I'll check my DVD for some specifics.

Edit: To echo the other posters, you definitely also need to make sure you have the necessary codecs installed too; installing 'ubuntu-restricted-extras' is the easiest way.
```
sudo apt-get install ubuntu-restricted-extras
```
