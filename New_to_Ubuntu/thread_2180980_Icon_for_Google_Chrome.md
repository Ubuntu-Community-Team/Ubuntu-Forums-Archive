### Post by deadflowr on 2013-10-15
You're on Ubuntu, right?
Then click on the dash button(in the leftside panel;called the launcher, it looks like the Ubuntu logo, it is the top most icon)
and type chrome.
Does google chrome show up?

Google-Chrome is like any other app, such as firefox.
Though it can be started through the terminal, it should have a launcher(icon) like any other.

If, after searching for it in the dash, it is listed, click on it and chrome should load.
When it loads, a chrome icon will be placed in the launcher Right click it and select "lock to launcher", which will keep it there forever.

If it isn't there, something would be wrong. You might need to try and remove the program and reinstall it.
And if it something is wrong, you could help us help you by telling us where you got the deb from.
