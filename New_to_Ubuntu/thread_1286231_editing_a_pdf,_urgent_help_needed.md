### Post by c0ldfuse on 2009-10-08
Ugly but quick method:
1) Get the page you need into some image form (ie .jpg)
2) Use image editor to paste in sig
3) Print page as .pdf or keep as .jpg (your choice)
4) Replace page in current document with "new" page through combine pdfs or some similar method.
