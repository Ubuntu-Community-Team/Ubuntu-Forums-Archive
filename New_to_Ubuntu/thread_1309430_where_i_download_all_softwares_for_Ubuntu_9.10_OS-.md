Since you're new, one of the most valuable things to learn about Ubuntu is this -- Linux is NOT Windows!!

Your references to "softwares" sounds like a litany of MS Windows apps, if I take your brief list to mean the following MS Windows apps:
1) Windows Live Messenger
2) Windows Media Player
3) Internet Explorer
4) Photoshop
5) Pagemaker

There is a product known as Wine that runs SOME, let me repeat SOME, MS Windows apps.  The link below takes you to their application compatibility site where you can search for the apps you want to use and see how well they "rate".  Anything not listed, or rating less than Gold is going to be basically a waste of time:

[http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true]("http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true")

If you're planning on staying with Linux for a while, you would do better learning Linux alternatives to the MS Windows apps.
