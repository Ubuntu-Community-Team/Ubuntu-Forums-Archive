### Post by Kevbert on 2009-06-06
SonyEricsson phones work best via Bluetooth. In saying that you should still be able to connect via USB. When you connect the phone, via USB, you may get a menu appear on the phone asking you to set the phone into file transfer mode or phone mode. Set to file transfer mode.
Initially you may need to go to Places-Computer to see the phone (and memory card) and it will be displayed as if it's a disk drive. If you go straight to Places-Removable media instead, initially, you may not see the phone.
