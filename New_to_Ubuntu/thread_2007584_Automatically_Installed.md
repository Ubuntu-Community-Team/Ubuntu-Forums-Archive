### Post by spikoley on 2012-06-21
Ubuntu has a Try Ubuntu option where it does not install.  When you boot into the LiveCD it has the option to Try Ubuntu or Install Ubuntu.  I believe it defaults to the Try Ubuntu after a set amount of time.  I think this is what happened.  In order to install you would have to create a user and select a few other options.

Your data should be fine.  Look in the upper right hand corner for the user name.  I think the LiveCD user name is Ubuntu.  If this is the case just reboot.
