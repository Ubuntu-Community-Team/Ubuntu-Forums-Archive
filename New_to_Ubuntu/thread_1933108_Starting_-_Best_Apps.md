And I will only add: learn how to use the command line terminal.
You start full screen text terminals with
***ctrl + alt + F1 ... F6***
and return to the graphical desktop environment with
***ctrl + alt + F7*** (or sometimes ***ctrl + alt + F8***)
but the most convenient way is to run terminal windows in the graphical desktop environment with
[COLOR="Red"]***ctrl + alt + t***[/COLOR]

The command line interpreter is usually ***bash***. Browse the internet for ***bash tutorial*** to find a suitable tutorial.
