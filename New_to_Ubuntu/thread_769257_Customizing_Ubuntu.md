### Post by elmer_42 on 2008-04-26
If you are interested in customizing Ubuntu, you want to browse [this]("http://www.gnome-look.org/") site and find either an emerald theme or a GTK theme. GTK themes are much easier to use, because all you have to do is drag the file into System -> Preferences -> Appearance. Emerald themes require you have Emerald, which can be installed with this command:
```
sudo apt-get install emerald
```
Also, some GTK themes come with icons. Emerald themes are only for the window borders, as far as I know. You can also install icon themes from [here]("http://www.gnome-look.org/index.php?xcontentmode=121") by just dragging them into System -> Preferences -> Appearance. Hope everything works for you!
