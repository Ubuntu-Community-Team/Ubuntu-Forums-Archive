### Post by kpatz on 2015-01-08
Quick instructions:  Download Lubuntu desktop ISO, burn to DVD, boot DVD and "try Lubuntu" (live mode), insert USB drive then in the System menu click "Startup Disk Creator".   From there you can select the source (DVD) and destination (your USB), erase the USB, and install and make it bootable.  Once done you can shutdown the live CD and remove both, and the USB should boot up anywhere you can insert it and boot from USB.

If you can boot the school computer from a DVD you could do it there.  Otherwise you might have to borrow a friend's computer.

If your laptop has a DVD drive you won't need to make a USB.  You'll just need to download the ISO and burn it to a DVD (maybe from school computer?)
