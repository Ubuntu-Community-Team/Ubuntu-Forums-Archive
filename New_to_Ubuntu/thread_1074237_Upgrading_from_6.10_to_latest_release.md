### Post by jimmy the saint on 2009-02-19
My understanding is that you need to upgrade one addition at a time.  In order to be current, you would have to do four or five upgrades.  At that point, it would be far simpler to just make a backup, wipe the partition and install intrepid (or Hardy if you prefer LTRs).  That's what I would do anyway.

As for the page, Without seeing it, I don't know how complex it is.  Much of the time, you will find that linux/ubuntu tutorials are CLI instructions.  This is mainly because the CLI is mostly the same no matter which distro you have, but also because it is simpler to throw a few CLI commands on the page as opposed to screenshots and descriptions etc. For the user, it is (hopefully) basically cut and paste, making sure to follow instructions carefuly, operation.
