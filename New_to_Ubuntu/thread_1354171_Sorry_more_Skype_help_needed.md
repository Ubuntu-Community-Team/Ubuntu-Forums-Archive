### Post by gradinaruvasile on 2009-12-13
First: describe your problem. What does not work as it should?
Then:
Is the sound working properly in other applications?
What version of Skype do you have? 

Important: make sure that "Allow Skype to adjust my sound levels" are unchecked.

If PulseAudio is loaded properly, the only devices you have are "pulseaudio sound server (local)".

I use the Skype 2.1 .deb version (the 2.0 also worked perfectly) from their site ([http://www.skype.com/download/skype/linux/choose/]("http://www.skype.com/download/skype/linux/choose/")). It works on every Ubuntu computer i installed it (6 different models, 4 of them Dell, Ubuntu versions 8.10, 9.04, 9.10).
