### Post by kellemes on 2009-12-30
> **ryanaw2 said:**
> [*]Could not get my printer Lexmark X6150 to work.  I downloaded the drivers but did not know how to install them.  Not possible to use scanner.


The scanner will indeed not work with Linux, it's just not compatible.
**Never buy Lexmark again!**
And at least check compatibility before buying hardware in the future..

Assuming the driver's name ends with *sh* (example: lexmarkdriver-.deb.sh) you can execute the installer like so..
From a terminal-window *cd* to the folder where the driver is..
```
cd <foldername>
```
Make the file executable..
```
chmod +x lexmarkdriver-.deb.sh
```
Execute the installer..
```
./lexmarkdriver-.deb.sh
```
