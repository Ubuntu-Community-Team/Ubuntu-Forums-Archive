### Post by cwsnyder on 2008-08-24
From my experiences with other networks, you can probably copy the first 3 entries from any other computer on the network, but you will need an identity and password assigned by the network administrator, _unless you can substitute for one of your host's computers_.  The Private Key entries will be needed only if you wish to transfer files to the network server or to other computers on the network.  I think the CA Certificate File and Client Certificate File are also used for the same purpose as the Private key, but as a Public Key.

The simplest solution on a temporary basis is to substitute for one of the existing computers with the same settings on your laptop.

cwsnyder
