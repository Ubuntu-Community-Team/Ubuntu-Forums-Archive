### Post by Frogs Hair on 2011-06-26
Some themes support color change . Open Appearance Preferences > Customize > Controls . Select the control in use from the list and select colors. If you change the color in one area you may have to change the text color and it takes some experimenting depending on the theme. I like dark themes , but they don't stay in use long because this problem . If I use Open Office I switch lighter theme because I find it hard to work on documents with a dark theme .
