### Post by chiefmanyrabbitguteat on 2008-06-06
I've never tried using Gthumb to make a CD.  Maybe submit a bug report?

In the mean time, try Brasero.  It comes standard with Hardy (8.04).  If you don't have it, you can get it through Add/Remove (in the Main Menu) or by entering
```
sudo aptitude install brasero
```

inside the Terminal.

Once installed, launch it, click "New Data Project," drag and drop your files into the Brasero window, and hit "Burn"
