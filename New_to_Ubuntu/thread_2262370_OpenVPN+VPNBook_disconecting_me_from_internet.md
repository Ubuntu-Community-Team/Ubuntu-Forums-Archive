### Post by nerdtron on 2015-01-25
Seems to have correct syntax.
For the verbosity, you can increase it to 4 or 5.

On the default openVPN config we have
```

# Debug level (default 1)
verb 5

# VPN logfile location
# If you don't specify a location here, logging will be done through
# syslogd and write to /var/log/messages
log-append /var/log/openvpn.log

```

The first several logs upon starting openvpn are the most important ones when we are debugging.
