### Post by EkuquoL on 2011-08-02
I find it funny when people keep using terminal commands to do small status checks. When the Unity and any linux GUI environment was meant to allow you to not have to remember all of those ambiguous commands.

If you do not see the configuration editor listed in your application menu... Then UBUNTU musta hid it like many other things.

What you can do is -  in terminal type:

   gconf-editor

That will open up a small window that will have a bunch of applications and stuff to configure, find the Audacity folder.. Then see if the plugins are enabled.
