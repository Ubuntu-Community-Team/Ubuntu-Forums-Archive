### Post by gruffyddd on 2010-11-07
I'm looking to buy a new desktop hopefully just for Ubuntu 10.04, though I may dual boot with Windows XP. My main use will be for programming - I'm a Java developer, and it's beyond a joke that I am still trying to use my 10 year old computer (AMD Athlon XP 2400+, 256 MB RAM) with modern memory hungry IDEs like Eclipse. I want my new computer to be capable for software development and running 'mainstream' Ubuntu software + youtube, iplayer etc. However I am not currently a gamer.

**Motherboard:** ASUS® M4A87TD/USB3: DUAL DDR3,SATA 6.0GB/s, USB 3.0
**Processor**: AMD PHENOM II X2 560 (3.30GHz/6MB CACHE/AM3) - BLACK EDITION
**Graphics Card:** 512MB NVIDIA GEFORCE 210 PCI EXPRESS
[INDENT]**Total:** £373[/INDENT]
