### Post by Paqman on 2009-03-28
> **soham_1207 said:**
> Thanks i am going to reinstall it.

Thank you.

You probably don't need to reinstall. Just make sure that you've got the applet packages in addition to the base AWN package.

Specifically:
avant-window-navigator = main AWN package
awn-manager = AWN manager (used to change what applets you're using)
awn-applets-c-core = Some basic applets
awn-applets-c-extras = Lots more applets

You should see all this in Synaptic. Just install everything you like the look of.
