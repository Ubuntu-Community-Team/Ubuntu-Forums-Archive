### Post by Dennis N on 2018-07-11
> In Ubuntu Mate 18.04 LTS version, does any one know where the repository tab collector thing is hiding as I would like to install gimp and Stellium

I assume you meant 'stellarium' for the second application? You can use the **Software Boutique**, or **System Tools > Software** or **System Tools > Software Install**. Search for 'gimp' or 'stellarium' in each. 

I found both in **System Tools > Software Install**. (I don't have the Software Boutique anymore, as it disappeared when I upgraded from MATE 17.10. The Boutique is a snap package). 'Software Install' and  'Software' run the same application.

You can also use **Synaptic Package Manager** to install anything in the sources you have enabled except the snap packages.
