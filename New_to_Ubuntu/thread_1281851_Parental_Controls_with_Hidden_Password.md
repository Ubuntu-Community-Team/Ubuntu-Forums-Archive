### Post by diesch on 2009-10-03
Ubuntu CE uses [DansGuardian]("https://help.ubuntu.com/community/DansGuardian") for web content parental control. You can install it on any other Ubuntu flavour, too.

But of course if **you** have root access, you can control it.

### Post by diesch on 2009-10-03
root can control anything, so as long as you have root access you can modify or bypass any parental control running on that system. So yes, it's either you not having root access, or the parental control running on another system.
