### Post by mcduck on 2010-08-10
To copy&paste in a terminal you need to use Ctrl-Shift-c and Ctrl-Shift-v, or the right-click menu if the terminal you are using has one.

OR you can use the primary selection, which is gives you the same result even easier. All you need to do is highligh what you want to copy and then middle-click where you want to paste it. It works on pretty much every Unix/Linux program, including terminals.
