### Post by Vaphell on 2010-10-17
not likely - clicking on stuff should bring the window up, no questions asked, that's why all these icons are there :)
in general terminal is invaluable in such situations, but you need to know the command that runs the app. You can right click panel/desktop launchers to find out, menu items can be copied as launchers. Type the command in terminal and pray it spits anything out. Error message + google and there is a chance you'll find an answer.
