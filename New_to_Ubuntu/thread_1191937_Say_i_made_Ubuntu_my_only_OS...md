### Post by Chemical Imbalance on 2009-06-19
If you just overwrite the Vista bootloader, then the partition will still be there to access from Ubuntu.  
I assume you have Ubuntu installed alongside Vista.

If you choose to reinstall Vista, just make sure you specifically point the installer to only format the old Vista partition, not your Ubuntu partition or whole drive.

The only problem is if you reinstall Vista, it will overwrite GRUB and you will have to boot a liveCD and reinstall GRUB again.

You should be fine otherwise.  Linux can read NTFS partitions with the ntfs-3g drivers (Ubuntu can by default).

Youd have to wipe it and reinstall the OS.

It all depends on what you do and what type of files you have. If you are a video editor or graphic designer using tools like premiere, photoshop, etc those files will be useless to you inside Linux. But if youre just talking about general files like office docs, images, music, video etc, then you can easily copy them over to your ubuntu drive and continue to use them.

My advice is just get a second harddrive, and put your windows drive to the side. Install Ubuntu in on that second drive and use that drive as your boot drive. You can then either add the windows drive and mount it as an ntfs drive where you can still access your files, or put it a external case so you can use it as an external drive while you copy the files into your linux drive.

next basically set your windows drive aside incase it turns out you need to go back to windows. (then u can just switch the drives and be back to business as usual.).

I been using Ubuntu for about two years. And use it as my primary OS in my laptop. It works fines as a genral use destop, but has few drawbacks when it comes to certain things like being able to watch netflix movies and stuff like that which only work in systems that support Silverlight, or if you use a 64bit system, you cant do stuff like watch youtube videos cause theres no 64bit flash plugin for linux. 

Anyhee, its a big leap so I would recommend putting it on a separate drive and living with it for a few weeks see how it feels to you.
