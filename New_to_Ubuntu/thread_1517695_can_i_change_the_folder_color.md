### Post by duanedesign on 2010-06-27
Seems there was a bug filed on Gnome for this.

> I'm unconvinced that this feature is necessary. There are already a plethora of
ways that a user can customise the appearance of their folders, including
emblems and custom icons. We don't need any more. Setting to NOTABUG.
[SIZE="1"][https://bugzilla.gnome.org/show_bug.cgi?id=523171]("https://bugzilla.gnome.org/show_bug.cgi?id=523171")[/SIZE]

For changing folder appearance system wide you will need to change the icon theme. Icon themes that can be downloaded from [www.gnome-look.org](www.gnome-look.org) or you can use the Themes Manager in System--->Preferences--->Appearance--->Theme  From there you can select your theme and click Customize. This will bring up another tabbed dialog box. Their will be an Icon Tab where you can select different icon themes that you have installed.

The icons will change the look/feel/color of the folders.
