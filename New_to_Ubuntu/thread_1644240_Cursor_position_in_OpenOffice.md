### Post by philip on 2010-12-12
I hope this is the right place to post. I was having a problem restoring cursor position in OpenOffice 3.2.0 under Ubuntu 10.4 that I had not had before under my previous installation of Ubuntu.  I came upon the discussion on page [http://ubuntuforums.org/showthread.php?t=526949](http://ubuntuforums.org/showthread.php?t=526949) but cannot reply there. Apparently, this issue comes up recurrently.

My problem is that I want to close a writer file (saved as *.odt) and shut down and then find the cursor at the same position when I open the document again.  Silly me, I had set the option under:
>Tools>Options>Openoffice.org>Security>Options>Remove_personal_information_on_saving

That options will prevent cursor position form being restored in the reopened document.  I hope this saves someone else some time.
