### Post by yeats on 2009-02-01
so do you have Ubuntu installed already?  If you do, you can open a terminal window (Applications -> Accessories -> Terminal) and type

```
python
```

This will put you in an interactive Python terminal, giving you instant feedback from whatever you put in.

If you don't have Ubuntu yet, you can still use Python on Windows.  Download from here:

[http://www.python.org/download/]("http://www.python.org/download/")

IDLE is the name of the Python interactive program.
