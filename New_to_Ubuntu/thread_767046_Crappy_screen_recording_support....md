### Post by realgt on 2008-12-01
i did my research on this too a couple months back and settled on gtkRecordMyDesktop. it has mouse-follow, outputs an ogg file that can be directly uploaded to youtube, and supports compositing, so you can see all the fancy compiz fusion effects...still it'd be nice to have an interface on it more similar to Jing Project than camstudio.

    **sudo apt-get install recordmydesktop gtk-recordmydesktop**

its good enough for [my screencasts]("http://www.realgt.com")
