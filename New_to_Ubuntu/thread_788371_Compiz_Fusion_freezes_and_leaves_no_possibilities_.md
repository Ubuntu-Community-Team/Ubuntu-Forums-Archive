### Post by tjwoosta on 2008-05-09
there is no need for a clean install!!

go to the login screen, in the bottom left corner there is a menu

choose start in safe-mode (this will start ubuntu in safe graphics mode)

in safe mode compiz wont even activate (should allow you to change things back to the way they were before, then just logout and login in normal mode)

using safe mode is a life saver  this has saved me many times from having to reinstall
