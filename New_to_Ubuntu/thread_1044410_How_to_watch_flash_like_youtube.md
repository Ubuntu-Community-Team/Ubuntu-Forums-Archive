### Post by billgoldberg on 2009-01-19

Search for gnash and swfdec and completely remove then.

Then search for "ubuntu-restricted-extras" and install it.

Besides flash-non-free, it includes java, audio and video codecs, ...

If you need dvd codecs, ... see link in my signature.
