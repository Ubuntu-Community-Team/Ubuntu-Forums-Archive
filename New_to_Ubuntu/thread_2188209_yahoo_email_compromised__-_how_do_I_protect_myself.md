### Post by Irihapeti on 2013-11-16
Yahoo has a history of security breaches, and what you describe sounds like a repeat of what happened earlier this year.

I've been using Thunderbird for several years and not had any problems of that kind (even when I received a few dodgy emails from Yahoo account holders). I understand that Thunderbird doesn't run javascript or java by default. Nor does it automatically download and display images.

However, you still run the risk of receiving malicious attachments, and clicking on links can open them in your browser. If the link is questionable, the end result will still be the same. But you probably know that, anyway.

If I get an email with an attachment from someone I don't know, I just delete the thing. If I'm not sure of a message, I'll use ctrl-u to view the source code, which has a lot of info about where it was sent from. It's kind of amusing seeing a message claiming to be from USPS or my local bank, but originating in Brazil! Obviously, anything like that also goes into the trash.

As for root-kit scanners, I've seen lots of people running those and getting freaked out by what are nearly always false positives. As I see it, there's some point to it if you're operating a server and know what to look for. Otherwise, I'd be inclined not to bother. But check the stickies in the Security sub-forum as well.
