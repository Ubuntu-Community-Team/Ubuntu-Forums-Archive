### Post by MidnightGrey on 2013-06-18
A quick search of minicom shows that it is located in universe repo. Do you have this repository turned on in Software Sources?

You can check by going into Ubuntu Software Centre > Edit > Software Sources > Ubuntu Software.
Is **Community maintained free and open-source software (universe)** enabled?
