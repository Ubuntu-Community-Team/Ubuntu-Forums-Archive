### Post by domeboy on 2009-04-19
Has anybody heard of a way to assign a Function key (F1,F2,...) to a workspace using the Workspace Switcher?

---

### Post by pytheas22 on 2009-04-19
Open System>Preferences>Keyboard Shortcuts.  Towards the end of the list you should see entries that begin, "Switch to workspace ..."  You can assign F1, F2, etc. as the hotkeys for these bindings.

Just be careful, as a lot of the FX keys are already bound to other functions in individual applications, which will take precedence--F5, for example, will cause the page to refresh in Firefox rather than switching to workspace 5, unless you explicitly disable that feature in Firefox.

I set the key bindings so that control-alt-1 switches to workspace 1, etc.
