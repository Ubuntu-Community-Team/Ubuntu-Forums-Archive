### Post by TheFu on 2014-07-08
startx is useful only for servers that have X/Windows installed with a window manager or DE and actually have a graphics card and graphics drivers loaded. It isn't needed on desktop installs.  Also, it only works on the local X/Server, not a remote X/Client.  Does that make sense?  If not, read up on X/Windows C/S architecture at wikipedia.

Please copy/paste the exact output from your startx if none of the above is an issue.
