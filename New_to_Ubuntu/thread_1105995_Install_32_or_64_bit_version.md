### Post by jespdj on 2009-03-25
Either the 32-bit or 64-bit version will work fine.

The 64-bit version historically has had some problems, like Flash and Java being a little harder to get working, but those problems have really almost been solved nowadays.

If you have 4 GB of RAM or more, then certainly go for the 64-bit version, because the 32-bit version is limited to around 3.2 - 3.5 GB RAM (depending on your hardware).

With regard to software compatibility: Almost all software for Ubuntu is available as 64-bit software. And for those very few programs that aren't, you can still run 32-bit programs on 64-bit Ubuntu.

Wine on 64-bit Ubuntu runs the same as on 32-bit Ubuntu (and it emulates 32-bit Windows - not 64-bit Windows).
