### Post by mikewhatever on 2010-06-10
You should edit /etc/default/grub file to get rid of the second boot menu. Use the following command to open the file for editing:
```
gksudo gedit /etc/default/grub
```

the first four lines need to look as follows
```
GRUB_DEFAULT=0
GRUB_HIDDEN_TIMEOUT=0
GRUB_HIDDEN_TIMEOUT_QUIET=true
GRUB_TIMEOUT=0

```
save and exit when done, then update grub with
```
sudo update-grub
```
