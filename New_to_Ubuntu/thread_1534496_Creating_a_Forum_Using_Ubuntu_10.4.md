### Post by Linux000 on 2010-07-19
Yes, you can run a web server on Desktop edition, but you shouldn't really use a server as a computer if your hosting a website as it could crash, or freeze, and bring the website down.(It's not Ubuntu, just the fact that using it loads down a computer)Plus server edition doesn't need new hardware, just reinstalling Server Edition on the computer, but be warned, Server Comes with no Graphical User Interface, just command line.

### Post by WRDN on 2010-07-19
Other than a very limited number of kernel configuration changes, the desktop and server additions are the same. You can remove stuff from the desktop to make it seem like the server edition, or add stuff to the server edition so its usable as a desktop.

As Linux000 said, I wouldn't use a server as a day to day machine, it should be a dedicated machine.

Also, you don't need good hardware at all, I'm running Ubuntu 9.04 server edition (with a LOT of services) on an IBM low profile rather old and clunky machine (with a whopping 20GB HDD) - it runs fine and fast.
