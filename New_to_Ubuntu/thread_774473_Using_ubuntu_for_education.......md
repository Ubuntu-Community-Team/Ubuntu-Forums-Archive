### Post by Golem XIV on 2008-04-29
My not-yet-4 daughter enjoys GCompris.  It's in the standard repositories.  Make sure you also install the language speech pack so the child has speech cues.

This package features many educational applets and games, including memory games, number and letter recognition, music, geography, logic puzzles, etc.  The applets are graded for suitable ages and it is possible to configure it so the advanced ones do not show as an option so that the child doesn't get confused and frustrated with stuff over her age.

### Post by willylindsay@yahoo.co.uk on 2008-05-14
Hi Techno - that circuit building thing needs gnucap to run.
Type at terminal:

        sudo apt-get install gnucap


It should work then.  The easiest thing to do is to go to Applications--Add/Remove--Education.  

Things you might find useful are tuxtyping and tuxmath.  I am running a primary school on Ubuntu so if you find anything good please post for me too.
