### Post by da burger on 2010-04-04
.tar.bz2 is a compressed folder. For programs they normally contain source code. If they do you need to decompress them t```
ar -xjvf file.tar.bz2
``` (or open with archive manager and click extract.)

Then it is normally these three commands: ```
./configure
make
sudo make install
```

However you can use checkinstall reather than the last one but I don't know much about that.

NB It is well worth checking that you can't get the program from the repos or a .deb file instead as they are much easier to use.
