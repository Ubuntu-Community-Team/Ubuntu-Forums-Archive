### Post by R3fr4cti0n on 2009-10-16
i found this in the arcives but i have no Opacity Settings in my General Options anyone know how to do this with 9.04?  I HAVE ALREADY DISABLED TOOLTIPS in gedit-config it wont work if you are using compiz


1. Open up CopmizConfig Setting Manager.
2. Go to General Options
3. Go to Opacity Settings
4. Add this value: (name=gnome-panel & type=tooltip) and for Opacity Value: 0
5. Enjoy! The tooltips are gone (well, not really gone, they're just invisible).
