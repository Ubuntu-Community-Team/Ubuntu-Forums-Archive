### Post by 73ckn797 on 2010-01-03
Go to System/Administration/Hardware Drivers and see if there is a recommended driver for your video card.

If your wireless does not work you could go to System/Synaptic Package Manager and install "ndisgtk". You could then install the driver from the wireless CD. Likely from the Windows XP directory. If you do have a driver CD.
