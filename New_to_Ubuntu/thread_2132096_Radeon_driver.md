### Post by 3rdalbum on 2013-04-03
On Linux, we don't usually download software from websites and install from there. It is usually the worst way to install software.

Instead, we use a package manager, which is a program that knows where to get software and how to install it. Ubuntu Software Center is our package manager.

If you go to Software Center, then to the Edit menu and come down to Software Sources, you can install the ATI driver from the last tab.
