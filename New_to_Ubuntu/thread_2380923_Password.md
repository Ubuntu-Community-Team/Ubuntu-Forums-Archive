### Post by deadflowr on 2017-12-23
You can set autologin, which will allow you to login without the need to enter a password.
(It will still require a password to run administrator tasks, such as updating or installing new software.)
The setting is in user Accounts, which should have a toggle to switch Autologin on or off.

Unless you want something else...
