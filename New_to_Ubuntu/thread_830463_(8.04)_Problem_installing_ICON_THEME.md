### Post by powerpleb on 2008-06-15
Best way to install icon themes I find is to extract the folder, then in a terminal:
```
sudo mv /icontheme /usr/share/icons
```

When you open up appearances it will be there.

Alternatively, you could open up the filemanager with superuser privileges by hitting ALT + F2, then entering 'gksudo nautilus'

Then you can extract the theme archive onto your desktop then drag it into the /usr/share/icons/ directory.

PS. Same method works for theme, the folder is /usr/share/themes though
