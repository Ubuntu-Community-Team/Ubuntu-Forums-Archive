### Post by cybrsaylr on 2008-08-17
I have a dual boot system and use my ext HDD for both OSs with no problems.
When using the Ext HDD in Windows you have to click on the 'safely remove hardware' button on the taskbar next to the clock on Windows when done when going from windows to linux. If you don't do that you get the message you got ("Cannout Mount Volume") when trying to use it in linux.
