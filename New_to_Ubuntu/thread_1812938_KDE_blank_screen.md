### Post by StephanG on 2011-07-27
As a Kubuntu user that's tried to install Gnome a few times, I've realized you have to be careful which packages you choose.  It isn't always the ones you think you need.

In this case, I think you either want kde-plasma-desktop (not plasma-desktop) or otherwise kubuntu-desktop.  But kubuntu-desktop will try to install all of Kubuntu's programs as well.  So you'll probably end up with A LOT of extra apps (for example both Transmission and KTorrent and they basically do the same thing.)

I hope that helps.
