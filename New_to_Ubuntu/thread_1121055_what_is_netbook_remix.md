### Post by OutOfReach on 2009-04-09
Well pretty much the only difference between regular Ubuntu and netbook remix is the user interface. The netbook remix includes a launcher that takes up the desktop area and a few other tools (for example, it also forces all (most, actually) apps to run maximized.
