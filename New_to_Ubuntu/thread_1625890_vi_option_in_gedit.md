### Post by newport_j on 2010-11-19
The vi editor has an option that if you put the cursor on a "{" or "}" and then press % on the keyboard you get the matching "{" or "}". This is very convenient when rearranging code or removing code. Now is there something like that on the gedit editor? This saves a lot of headaches; it would be nice to do this is gedit.
 
Newport_j

---

### Post by TiBaal89 on 2010-11-19
Yes there is!  Go to Edit -> Preferences and on the View tab click 'Highlight matching bracket.'
