### Post by marshmallow1304 on 2009-10-14
> **random turnip said:**

That means that whoever made the theme didn't (properly) make it so that Ubuntu can auto-install.  Not all the theme makers use Ubuntu :).

> **random turnip said:**
> I basically want a really quick crash course on how i go about installing a theme.
also, will this lose my compiz fusion settings?

If drag-and-drop doesn't work, look for a readme file or instructions on the gnome-look page.  Themes (at least those installed through the theme manager) will not affect compiz settings.
