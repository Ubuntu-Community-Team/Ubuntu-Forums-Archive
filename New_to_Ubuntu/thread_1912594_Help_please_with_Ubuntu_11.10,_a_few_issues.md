### Post by Double.J on 2012-01-20
Hi there, sorry I was doing some benchmark tests on my hardware for comparision. XFCE definately did less resources, but my personal fav is LXDE, so it's personal preference really:

Xubuntu - ram 12% cpu 4%
Lubuntu - ram 19% cpu 5%
unity - ram 13% cpu 8%

I just switched DE on my system and loaded the same task manager up 2 min after log in.

As you can see they look pretty similar, but it's worth noting that this is a netbook with limited graphics and the first 2 load much faster than unity. I can't really choose performance between L&X I think it just comes down to preference - maybe check out some screen shots, or live cds...
