### Post by abodsalas on 2008-08-12
Hi.
I have a notebook Acer Aspire 4720-6518, with Core 2 Duo, T5250, 160 GB, 2 GB DDR, Mobile Intel Graphic Media Accelerator X3100, 802.11a/b/g WLAN, DVD Super Multi DL.

I installed Ubuntu 8.04 and erased all trace of Vista.

WLAN OK
Screen and Graphics Acceleration OK
Keyboard OK
Hared Disk OK
Sleep Button OK
Sound, Earphone Jack OK

It all works great except for the microphone. I had to add to Volume Control the: Input Source, Digital and Mic Boost Options and activate them.

On input source I select Mic, and it activates. Sometimes it deactivates so I change it, and then change it back to Mic.

Thats all so far.
