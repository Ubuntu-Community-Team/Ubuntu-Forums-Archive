### Post by SeijiSensei on 2013-06-23
If it uses the same version of Android (4.1.2) as my Galaxy S3, you need to upgrade the Ubuntu machine to 13.04.  That's the first Ubuntu version with solid support for the MTP file transfer protocol which is the standard on recent Android releases.
