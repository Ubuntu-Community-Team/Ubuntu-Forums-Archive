### Post by ndefontenay on 2009-04-26
Hi.

I just want to give a feedback of my installation as a witness to those unsure of doing the move.

I've installed ubuntu 9.04 Jaunty Jackalop on a Vaio laptop.

I've tried performing the upgrade but i'm in thailand and despite a thai mirror, it was marking 2 days average to download the packages.

I decided to go with a fresh install. This is, in my opinion the best solution in that case because in ubuntu 8.10, I had to use ndiswrapper with a windows driver to get the wireless working.

The installation result:

1) I got the wireless card detected and working. Connecting to wireless has never been so easy.
2) being on a laptop, when booting the screen was showing weird lines for a short time before going to desktop. This is solved.
3) It's fast to boot so JJ stands to its promise.
4) I've listened to a few mp3s. It asked me if I wanted to download the encoders. It worked too. That feature existed before but wasn't working well in my opinion.

Skype: Skype is not released yet for ubuntu 9.04 (and the download featured on [www.skype.com](www.skype.com) is for 8.10 but that one works fine with 9.04.

I am yet to test a CD burning which failed to work previously in 8.10.

I'm always surprised and happy with the newest release of ubuntu :)
