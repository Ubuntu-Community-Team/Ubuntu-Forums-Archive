### Post by mister_pink on 2008-04-30
Put in an obviously wrong password - does it do the same thing?
If you press Ctrl + Alt and F1 you will get to a command line - note, its usually Ctrl Atl F7 to get back - try other F keys if that doesnt work. Can you log in here? Note that nothing appears as you type your password on the command line, but it is working. Type "logout" once you've shown you have the right username password. (Bare in mind its all case sensitive.
Finally, if this shows youve got the right username/password, then you can try selecting "Sessions" and the Failsafe option, and try again.

If on the other hand the password still isnt working, you'll need to go into recovery mode, I'll post instructions if that turns out to be the case.
