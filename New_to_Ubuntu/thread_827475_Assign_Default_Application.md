How do you assign a filetype to be opened to a particular application, in my case I installed Zend Studio for Linux but somehow the *.php files does not open with ZDE automatically.

---

### Post by iaculallad on 2008-06-12
Right click your .php file, click "Properties" and select the "Open With" tab. From there you could select the Zend Studio application by clicking on the "Add" button to select it from the list.
