### Post by mcduck on 2012-02-25
So you actually have a third-party repository that provides up-to-date packages for aMule?

Software versions in Ubuntu's official repositories are generally kept the same for the full lifetime of that release, and updates are only made for security reasons and to fix any serious bugs, never just to give you the latest available version or new features. (the reason for this is stability, the original versions were all tested to work together without problems, and doing a detailed testing for all the updates with all the possible software and hardware combinations wouldn't be possible).

So if you want to always have the latest version of some program, you'll need to find a third-party repository that provides such updates, or install the latest versions manually.
