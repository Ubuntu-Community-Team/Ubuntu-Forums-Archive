### Post by kanikilu on 2010-07-07
Not that I'm aware of. I think you used to be able to download the updates manually, but AFAIK, you still needed iTunes to apply/install the updates.

Last time I needed to use iTunes on my Ubuntu machine, I used it in a virtual Windows 7 installation in VirtualBox. At the time, the latest iTunes version did not work, and I had to get an older one (9.0.2, I think?) from oldapps.com and then it worked fine. Granted, I don't think I did a firmware update via VBox, just sync'ing, so YMMV.
