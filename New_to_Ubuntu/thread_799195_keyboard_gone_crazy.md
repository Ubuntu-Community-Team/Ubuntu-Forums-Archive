### Post by sdennie on 2008-05-18
I would check System->Preferences->Keyboard->Layouts.  It sounds like you are using a UK or European layout and not a US layout.  Also, Ctrl-C/X/V should work like in windows except in the terminal where you'll need to use Shift-Ctrl-C/X/V because Ctrl-whatever has special meaning in a terminal.
