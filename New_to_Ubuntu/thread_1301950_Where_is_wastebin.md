### Post by drs305 on 2009-10-26
That link is a bit old. In new Ubuntu versions the user trash is in ~/.local/share/Trash. There are two folders, info and files. The former holds the file info, the latter the deleted files.

Root's trash is in /root/.local/share/Trash.

I wrote a guide about finding and deleting Trash a while back:
[ How To: Disk Full? - Check Your Trash  ]("http://ubuntuforums.org/showthread.php?t=898573")
