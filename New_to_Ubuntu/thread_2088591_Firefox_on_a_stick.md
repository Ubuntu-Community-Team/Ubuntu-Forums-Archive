### Post by zombifier25 on 2012-11-28
So, if I read correctly, you want to remove the Firefox installed on your Linux system, and you want to create a launcher to launch Firefox.

My solution is to not remove the built-in Firefox so that applications can still associate certain web protocols with it, but to replace the firefox executable with a shell script that launches the Wine Firefox. First off, launch your favorite text editor and fill it with these lines:
```
#!/bin/sh
wine /path/to/firefox.exe $@
```
Modify /path/to to the full path to the Firefox .exe file. Then save the file as "firefox" in your home folder . Then run these commands:
```
cd ~
sudo chmod +x firefox
sudo mv /usr/bin/firefox /usr/bin/firefox.bak
sudo cp firefox /usr/bin/
```

If you want to revert the changes, simply run this:
```
sudo mv /usr/bin/firefox.bak /usr/bin/firefox
```

I haven't tested this, because I'm not really in the mood of replacing Linux Firefox with Wine Firefox, but if it doesn't work, just use the command above to revert the changes.
