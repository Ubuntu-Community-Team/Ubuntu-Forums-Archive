### Post by Autodave on 2017-10-31
That is interesting. I don't know if this will help you or not, but whenever I get a laptop that is doing something weird, I always try this first. About 50 % of the time, it works.

  	 	 	 	   1. Power down and disconnect power cord.
2. Unplug all peripherals.
3. Remove battery.
4. Wait 10 minutes.
5. Reinstall battery.
6. Connect power cord.
7. Boot computer and test to see if condition is gone.
8. If good so far, now you can reconnect peripherals.
