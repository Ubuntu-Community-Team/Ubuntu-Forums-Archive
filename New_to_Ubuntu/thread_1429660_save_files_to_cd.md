### Post by mcduck on 2010-03-14
You need to use a CD burning application to write your files to CDs. Ubuntu doesn't come with any package writing software so you can't just drop files to disks. (the normal CD filesystem is read-only by standard, even if you use CD-R or CD-RW disks)
