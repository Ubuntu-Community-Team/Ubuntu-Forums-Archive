### Post by localhost8080 on 2013-11-24
probably the best way if you use unity is to press the windows key (this will bring up the menu) and type calc
if you have open office installed, then 'open office calc' will appear in the list along with gnome calculator

openoffice calc is like excel, so thats not the one you want, but you learn pretty quickly to type:
win, calc, down-arrow,enter
