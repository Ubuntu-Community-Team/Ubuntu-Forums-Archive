### Post by Bob Bill on 2011-02-26
I have found a solution which means you don't have to install extra software.
I'll go through everything I did to make sure it can be followed.
Sorry if its unnecessary and boring for some. i am new to Linux and have Ubuntu 10.10 recently installed.

I had put the birthdays under personal details in the contacts list. 
To see the birthdays I click on the calendar in Evolution and make sure the Birthdays and Calendars is checked.
Then right click on the birthday and click Copy to calendar ( or Move presumably) and click personal calendar.

Then the alarms can be set in the usual way.
I wonder if this can be done in reverse. i.e set up the birthday in the Personal calendar first, move it to the Birthday Calendar and then see if your preferences for alarms still work.

There seems to be a default of an alarm 10 days before the birthday, when the birthday was set up under contacts.
