### Post by mattbutsko on 2009-07-08
right click the network manager applet on the panel (if you're knew, that's the gnome equivalent to the taskbar), and select edit connections. Choose the connection, usually auto eth0, and select edit.

Head over to the IPv4 settings tab. Select "manual" instead of automatic for Method settings, or change it to whatever works best for you.

Hope that helps.
