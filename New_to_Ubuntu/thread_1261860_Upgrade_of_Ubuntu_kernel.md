### Post by bumanie on 2009-09-09
I think that is the latest kernel for 9.04. If a new/updated kernel is released, it will come via update manager. They are tested for stability prior to being released. Occasionally a new/updated kernel may cause problems with a few computers (due to some idiosyncratic issue with a particular computers configuration), but most of the time, they work fine. It is always a good idea to ensure you have at least one older kernel that works, so that if an upgraded kernel does cause problems, you can choose the older kernel from the grub screen and still continue to be able to use ubuntu.

### Post by 3rdalbum on 2009-09-09

Nothing is bug-free, the same way that nothing is future-proof.
