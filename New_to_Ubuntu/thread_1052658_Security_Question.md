### Post by emarkd on 2009-01-28
Viruses, trojans, spyware, it's all just software.  It has to be written (or at least compiled) to target the host operating system.  Windows viruses don't run on Linux any better than Windows software does.

There have been only a few known Linux viruses and none are currently known to be "in the wild".  You're very safe browsing with Ubuntu, as long as you don't copy those suspicious files over to your Windows box.
