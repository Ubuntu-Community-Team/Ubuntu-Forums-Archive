### Post by Aquatic Fist on 2010-04-02
1. Press alt + f2
2. Write "gnome-volume-control-applet" and press run.

Is this what you are looking for?

If you want it to start on boot:

1. Goto System> Preferences > Startup Applications.
2. Press Add and Write "gnome-volume-control-applet" after command.

Hope this helps! :)
