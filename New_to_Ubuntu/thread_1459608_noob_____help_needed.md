### Post by undecim on 2010-04-21
Use the "check" option from the menu where you choose install to make sure that the image burned well.

If there is an error, check the MD5 of the .iso (the CheckIt Firefox plugin is great for that. [http://lovinglinux.megabyet.net/?page_id=662](http://lovinglinux.megabyet.net/?page_id=662)) check it against [http://releases.ubuntu.com/karmic/MD5SUMS](http://releases.ubuntu.com/karmic/MD5SUMS)

If that works out fine, then burn the CD again at the slowest speed you can.
