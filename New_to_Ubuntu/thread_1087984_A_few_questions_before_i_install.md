1. Ubuntu is fairly hassle-free. It will boot live (from the CD without affecting your hard disk) and even the default installation minimizes its effect by setting up a dual boot. Windows offers neither of these options.

2. There has not yet been a single widespread Linux malware threat of the type that Microsoft Windows software currently faces.

3. Both documents created in Microsoft Word and Microsoft Word itself (via WINE) can be used in Ubuntu.

4. Ubuntu will read from FAT32 and NTFS filesystems so yes you can read data you put on a memory stick with Windows.

5. In a dual boot Ubuntu lets you access Windows partitions (FAT32, NTFS), Windows including Vista generally cannot read Linux partitions (ext2, ext3). So yes, you can (and I have) written a document in Ubuntu and saved it to a Windows partition (for use by both).

6. Ubuntu will hold your hand. Since v8.10 Ubuntu's default method of installation is a dual-boot even where you already have seperate extended partitions.
