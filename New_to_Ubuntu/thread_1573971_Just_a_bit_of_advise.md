### Post by Jazzy_Jeff on 2010-09-13
I would recommend downloading the Live CD file and putting it on a thumbdrive. Boot from the thumbdrive and give it a try before installing. That should show you if everything is working. You can use a program unetbootin to put the Live CD on a pendrive. Good luck. If you have any questions just come back and ask.

Ubuntu should run faster than Windows on that machine. You can either use regular Ubuntu or the Netbook remix version.
