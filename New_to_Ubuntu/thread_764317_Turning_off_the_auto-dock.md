### Post by rouge568 on 2008-04-23
Are you using compiz or metacity?
If compiz, first install compizconfig-setting-manager. Go to System>Preferences>Advanced_Desktop_Effects_Settings. Then click on "Wobbly Windows" or "Snapping Windows", whichever is enabled (probably wobbly windows). From there, you can adjust if you want snapping, and if so, how much.
I'm not sure what to do if you're using Metacity; Google is a better friend than I.
Let me know how it works out!
