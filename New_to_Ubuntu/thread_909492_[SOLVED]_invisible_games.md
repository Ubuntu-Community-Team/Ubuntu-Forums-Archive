### Post by Vivaldi Gloria on 2008-09-03
I installed castle-combat and it didn't make a menu item. I guess the package maker didn't add it. 

You can start such games in a terminal by writing their name (castle-combat) or press ALT+F2. 

You can also make a menu item yourself. Right click on the ubuntu icon and choose edit menus.

Also note that some apps are supposed to be run in a terminal, e.g. bsdgames. These never get menu icons.
