```
top
```
shows the processes which consume the most memory. However, to show the running programs, go to the dash (assuming you have plain ubuntu):

[IMG]http://www.omgubuntu.co.uk/wp-content/uploads/2012/02/new-dash-buttons.png[/IMG]
Search for "System monitor"

You should get something similar to the attachment.

Normally, the processes with icons here are running programs.
