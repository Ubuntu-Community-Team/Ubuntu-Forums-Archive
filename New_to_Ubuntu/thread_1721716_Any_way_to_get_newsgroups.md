### Post by alphacrucis2 on 2011-04-04
> **Learning Linux 2011 said:**
> Is there any way to get newgroups using Ubuntu 10.10?

My internet service provider doesn't offer newgroups.

There are few newsgroup readers available such as Pan, or you could use Mozilla Thunderbird which supports newsgroups. I think what you are really after is a usenet/newsgroup provider. Many ISP's dropped providing usenet servers as part of their standard offerings but there are still a few open servers around (try googling 'free usenet server' or you may have to opt for a paid service like giganews. If you only want text newgroups then they can be accessed by the google groups portal (if you are really really stuck).


Edit: I notice the version of Pan in the Ubuntu repos is fairly old. The latest release 0.134 is available in this launchpad ppa.


[https://launchpad.net/~klaus-vormweg/+archive/ppa]("https://launchpad.net/~klaus-vormweg/+archive/ppa")
