### Post by WanderingAlbatross on 2011-10-19
You should be able to install with:
```
sudo apt-get install kubuntu-desktop
```

Then you must log out and change desktop environments.  To do this there should be able to find a icon or box that you can click on that will show your available environments.  Click on Kubuntu and you are off!
