### Post by bumanie on 2008-10-02
Impossible to say until you try it. You are the only one who can decide that. Initially you could either install via wubi or dual boot with windows until you learn more about linux. It in most cases is not harder to use, in fact many features are easier, but one needs to develop a different mindset. For me and many others, windows is a relic from the past - try ubuntu, keep an open mind and give yourself time to learn and you will probably see the ubuntu/linux advantages. Nothing ventured, nothing gained - You can always return to windows if linux does not suit you.

### Post by Thelasko on 2008-10-02
Keep in mind it is better to use native Linux software than to run programs in Wine.  See [this page]("http://linuxappfinder.com/alternatives") for a list of Windows programs that have Linux alternatives.  I recommend using the search function, as the list is many pages long.  Keep in mind that this is not a complete list.

Also, make sure you are familiar with [installing software in Ubuntu]("http://www.psychocats.net/ubuntu/installingsoftware"). It's much different than Windows.
