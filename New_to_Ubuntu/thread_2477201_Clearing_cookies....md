### Post by deadflowr on 2022-07-18
> To capture the screen image; now found ! is the "print screen" key, which does not ask where to save the image or where it is and gets lost in limbo if I want it on a folder or desktop.
It saves to Pictures.
Might also save to a sub-folders inside Pictures, perhaps named Screenshots.
I don't remember or know if this can be changed anymore.

That said, I doubt you'll find anything useful about clearing cookies in The Ubuntu Settings program.
Most cookie data is set inside the specific apps.

Also, Ubuntu Help is not too useful for help with apps themselves.
It's only really helpful for navigating the desktop itself.

In terms of Firefox, the 3 horizontal line icon in the top right corner is still where settings can be found.
Or
You can always put
```
about:preferences
```
in the address bar to open the preferences directly.
Cookie settings are in the Privacy and Security section.
That can also be accessed directly with
```
about:preferences#privacy
```
