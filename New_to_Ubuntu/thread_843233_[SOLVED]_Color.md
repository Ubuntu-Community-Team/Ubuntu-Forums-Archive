### Post by iaculallad on 2008-06-28
For the color depth in your xorg.conf, it should be similar to the (Bold texts) Code below:

```
Section "Screen"
        Identifier      "Default Screen"
        Device          "Intel Corporation 82865G Integrated Graphics Controller"
        Monitor         "771S"
**     DefaultDepth 24**
EndSection

```

### Post by RomeReactor on 2008-06-28
Hi. The new X server doesn't seem to reference the xorg.conf file fro the display settings, so on fresh installs the file won't have the necessary information. You can change the color depth in nvidia-settings, as iaculallad said.

EDIT: I think the nvidia-settings program isn't installed by default; to install it:
```
sudo aptitude install nvidia-settings
```
