### Post by t0p on 2009-03-28
Windows programs don't run natively in Linux. *Some* Windows programs can run in Linux if you use an "emulator" called Wine.  But generally, you should look for a Linux alternative for the app you used to use on Windows.

For example: you said you want to use Yahoo messenger.  Well, if you've just installed Ubuntu, you will have an app called **Pidgin** which is an IM client for Yahoo, GoogleTalk, AIM and more.  It should be in the Applications menu under Internet.

Also, I believe there's a little browser-based app for Yahoo Messenger at Yahoo's email site.  I think it works okay with Firefox.
