### Post by SeijiSensei on 2012-11-30
Try a more lightweight Ubuntu derivative like Lubuntu or Xubuntu.

Did you create a swap partition at boot?  If not, try adding a swap file:

```

sudo dd if=/dev/zero of=/swapfile1 bs=1024 count=1048576
sudo mkswap /swapfile1
sudo chmod 0600 /swapfile
sudo swapon /swapfile1

```

That will add a 1 GB swap file to the system.  To make this active every time you boot, you would need to edit /etc/fstab (as root with sudo) and add the line:

```

/swapfile1 swap swap defaults 0 0

```

to the bottom of that file.

Have you tried installing the proprietary driver for your NVIDIA card?  Run the Additional Drivers applet and see what it shows.

### Post by 3rdalbum on 2012-12-01
Ubuntu 12.10 uses software rendering when 3D acceleration is not available or is incomplete. Unfortunately, software rendering is slow. That's why, when there's no 3D acceleration at all, Ubuntu makes the Dash and Launcher opaque to speed things up a bit.

This isn't perfect, however - if you have any kind of 3D support, no matter how slow or incomplete, the Dash and Launcher will be transparent, which causes massive slowdown.

Hopefully the Ubuntu developers will fix this problem, but until they do, you can get a speedup by following this guide: [http://www.ubuntuvibes.com/2012/10/make-unity-more-responsive-in-ubuntu.html](http://www.ubuntuvibes.com/2012/10/make-unity-more-responsive-in-ubuntu.html)
