### Post by guitarafro on 2014-06-24
You need to be very careful with BleachBit. It can and will delete necessary files to keep on your system if you click through the settings blindly. If you are new to Ubuntu/Linux, I wouldn't even use it.

### Post by sameermw on 2014-06-26
If you want to clear and clean your pc just use **ubuntu tweak janitor, **After you understand about Ubuntu and its behaviors read Bleachbit documentation and just follow [monkeybrain20122]("http://ubuntuforums.org/member.php?u=1843403") words.[COLOR=#000000]

[/COLOR]Install Ubuntu tweak tools

```
[TABLE]
[TR]
[TD]sudo add-apt-repository ppa:tualatrix/ppa
[/TD]
[/TR]
[TR]
[TD]sudo apt-get update
[/TD]
[/TR]
[TR]
[TD]sudo apt-get install ubuntu-tweak
[/TD]
[/TR]
[/TABLE]

```

Deb file can be downloaded here [Ubuntu-Tweak_0.8.7.deb]("https://launchpad.net/~tualatrix/+archive/ppa/+files/ubuntu-tweak_0.8.7-1~precise1_all.deb")
