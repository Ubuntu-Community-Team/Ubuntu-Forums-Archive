### Post by DaveM753 on 2008-04-28
You might want to start with the application nvidia-settings (I believe that's the name).  It allows much more control of your nvidia card than the built-in "Screen Resolution", including many different resolution settings.

If that doesn't help, there are many different forum discussions here about nvidia problems.  Just do a search for:

nvidia screen resolution 8.04

(assuming you're running Ubuntu 8.04)
