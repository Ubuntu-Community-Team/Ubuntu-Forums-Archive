### Post by sandyd on 2013-10-21
> **mcsheffrey said:**
> I installed "additional drivers" to configure third-party and  proprietary drivers.

If you install nvidia drivers, typing 'nvidia' in the dash (ubuntu icon on top-left) should give you nvidia settings
If you install ati drivers, typing 'amd" or "catalyst" in the dash (ubuntu icon on top-left) should give you ati settings
