I have been working on my first GTK program (with a VERY rusty C programming mind).  This program will find, open, unlock (you need a key generated from another program), close, download all pictures/video, show the camera key (for use in the external key generation program) and clear the memory of hacked CVS one-time use digital camcorders and digital still cameras.  The "how-to's" for hacking these cameras to be reusable are on camerahacking.com, and this software works with them.  The program is built from code of 2 existing programs(1 worked on camcorders, the other on digital still cameras).

I have attached some screen shots, taken while running in Ubuntu Hardy.

If someone would be in need of this software it is available free as I found the original programs on sourceforge.net and they are GPL'd.
