### Post by Jenks141 on 2013-04-28
Hi bluesman501
 
I mentioned that I had a problem with the Yahoo webapp on another thread on my experience of upgrading to 13.04 and Tanker Bob suggested the following:

*I disabled all webapps in Unity Tweak Tool and then Yahoo mail worked fine in Firefox. I then went into Synaptic and **uninstalled ***unity-webapps-yahoomail***. I re**-**enabled webapp prompts and now all is right with that world.*

Our problem was with the yahoo mail webapp so if you have this installed this may work for you.

You will need to install these applications from the Ubuntu Software Centre if you do not have them installed, Unity Tweak Tool and Synaptic Package Manager.

Hope this works for you.
