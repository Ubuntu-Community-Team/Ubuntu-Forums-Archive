### Post by Didius Falco on 2009-05-26
BTW, a belated welcome to the Ubuntu forums!

In case you aren't sure where to find Synaptic, it's in **System/Administration/Synaptic Package Manager**.

You can change the usplash yourself by going to the **Fast User Switcher** (where you log out, shutdown, etc. - it defaults to the top right corner of the screen) and Right-clicking it. Then choose **Setup Login Screen**, enter your password and go to the **Local** tab.

Or, you can use the package recommended by Clonne4crw, if you prefer.

Regards,

Didius
