### Post by tarps87 on 2009-04-16
It depends on what game they play. I still use windows for some windows games (Guild war and c&c) because they don't run smoothly under wine. There are plenty of games for GNU/Linux. If there are a windows gamer there they should stick to windows as software will always run best on the platform it was designed for. To answer your question, yes and no, I use windows for my existing game that won't run on GNU/Linux but I now no longer buy and windows games.
Also although wubi is a good introduction to Ubuntu the performance will be lower than if it install directly onto the hard drive, this is due to the reason you like it, as it is running on a ntfs formatted hard drive rather than a ext format which it is designed for
