[http://people.ubuntu.com/~dpm/ubuntu-10.10-translation-stats.html](http://people.ubuntu.com/~dpm/ubuntu-10.10-translation-stats.html)
If you want to get involved, start here:
[https://wiki.ubuntu.com/Translations](https://wiki.ubuntu.com/Translations)
