### Post by Giant Speck on 2009-08-31
Is this problem happening on the desktop?

If so, try restarting Nautilus.  Open up the run dialog (Alt+F2) and enter "killall nautilus."  The run dialog will disappear, along with the icons on the desktop.  Then open the run dialog again and enter "nautilus."  The icons should return to the desktop and a window should open.

Also, you mention that someone could have messed with your computer.  This may sound stupid, but have you tried seeing what is set as your wallpaper?  Someone could have pulled a prank on you and uploaded a screenshot of your desktop as a wallpaper.
