### Post by wilee-nilee on 2012-05-10
The only app I have found to effectively load XP to a thumb, as a single load mind you is run from windows, works every time. You must have a legit XP though.
 
[http://wintoflash.com/home/en/](http://wintoflash.com/home/en/)

This multiloader probably can, but since I know one that works I have not really investigated it. Needs additional software on the thumb for download though.

[http://www.pendrivelinux.com/multiboot-create-a-multiboot-usb-from-linux/](http://www.pendrivelinux.com/multiboot-create-a-multiboot-usb-from-linux/)

I use this second one primarily, it can load W7 with no problems. W7 though will load to a thumb with a NTFS and boot flag with a extraction in linux and boot just fine anyway. XP was not designed to run on a usb, and has a different boot setup.
