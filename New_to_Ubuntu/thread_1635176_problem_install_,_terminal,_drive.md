### Post by cariboo on 2010-12-01
First off, if you are having trouble reading the forum at the default font size, press Ctrl++ to zoom the page. (I have to do it myself on occasion)

If the system is asking for the CD, you can disable it by going to System->Administration->Software Sources and remove the check mark beside CD-Rom.

I'm not sure what you mean by seeing a disconnected usb drive, if you open Nautilus (Places->Home Folder) and click File System you should see all the directories in / in the right pane.
