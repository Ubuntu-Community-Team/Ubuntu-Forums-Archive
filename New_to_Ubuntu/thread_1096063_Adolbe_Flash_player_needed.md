### Post by linux6994 on 2009-03-14
Welcome to Ubuntu

To install the extras that you need please do the followinng:

1. Applications > Accessories > Terminal 
2. paste in this command line sudo apt-get install ubuntu-restricted-extras 
3. enter and answer y for yes

This will install the packeges that you need.

also the following terminal commands are used for updating your system:

sudo apt-get update 
sudo apt-get upgrade
