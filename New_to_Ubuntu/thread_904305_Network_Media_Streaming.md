### Post by billgoldberg on 2008-08-29
1 answer: Openssh.

SSH works best between linux machines, but with some extra software you can use it in Mac OSX and Windows XP/Vista.

[http://linuxowns.wordpress.com/2008/06/08/share-files-between-2-ubuntu-computers/](http://linuxowns.wordpress.com/2008/06/08/share-files-between-2-ubuntu-computers/)

--

You could also use a upnp server, but that's a bit more tricky to set up. But this will also allow you to stream media (videos, pictures, music) to your PS3 or Xbox360.

[http://linuxowns.wordpress.com/2008/04/28/stream-media-from-ubuntu-to-your-ps3/](http://linuxowns.wordpress.com/2008/04/28/stream-media-from-ubuntu-to-your-ps3/)
