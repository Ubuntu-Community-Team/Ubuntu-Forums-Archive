### Post by Monicker on 2008-05-11
Firefox?

Edit -> Preferences -> Privacy -> Settings, make sure Download History is checked.

Then you can use the Clear Now button in Privacy, or Tools -> Clear Private Data

### Post by nynoah on 2008-05-11
A cool thing to add is called "download status bar" in the firefox addons.  It makes it so there is a small bar on the bottom of your browser instead of launching another window that you have to click on to get out of your visual way.  It also has an icon for just hitting clear.

Go to "tools-addons-get addons-browse for addons" and search for it.  There are a lot of cool features that you can add to Firefox that way.
