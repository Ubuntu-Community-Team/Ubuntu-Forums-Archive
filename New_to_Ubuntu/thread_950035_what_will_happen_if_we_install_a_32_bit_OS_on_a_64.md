### Post by OutOfReach on 2008-10-16
Installing a 32bit OS on a 64 bit machine absolutely alright, alot of users do that.
There are some software issues involving 32bit software on 64 bit machines (take flash for example).
Yes, 64bit is significantly faster than 32bit, but really not enough to see a real difference.
As far as I know you need to compile PAE into the kernel to use more than 3gb on a 32bit OS.

### Post by PocketDog on 2008-10-16
[QUOTE=legolas_w]
I am worried  because I always seen windows application have problems with 64 bit hardware and It make me think that Linux applications have similar problems.[/QUOTE]
No, Windows applications (mostly drivers) have/had problems with the 64bit OS, not the hardware.

[QUOTE=legolas_w]If I install a 64 bit OS on the T400, can I install 32 bit applications on it?[/QUOTE]
Yes. 

[QUOTE=legolas_w]Is 64 bit Ubuntu faster than a 32 bit one?[/QUOTE]
If you have more than 3GB or RAM, certainly. Maybe not by much but that depends on the application you're running, not the OS. 

[QUOTE=legolas_w]Does 32 bit Ubuntu can use 4GB of ram?[/QUOTE]
No, all 32bit OSs have a 3GB ceiling.
