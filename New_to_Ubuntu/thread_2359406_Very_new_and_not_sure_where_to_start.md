### Post by Bucky Ball on 2017-04-23
Welcome. The Lenovo will probably work, and so will the Toshiba! Lenovo is known to be Linux friendly while some Toshiba models are difficult to install on while others easy. I'm typing to you now on a Toshiba Satellite Pro L510. Works faultlessly, installs a piece of cake, everything works, but this is an older model and does not use UEFI.

The main issue with getting Ubuntu on a machine is often not the machine hardware so much as its BIOS. UEFI can make things tricky. There are standards which prevent manufacturers from tweaking the UEFI to make installing other operating systems difficult. Sometimes manufacturers ignore them which makes installing Linux hard, but not impossible. 

People tend to read or be told one bad thing about one particular model and it puts them off the whole brand. Someone 'told you Toshiba were bad.' Maybe they had a bad experience and now, in their humble opinion, all Toshibas are 'bad'. They're not. Some Toshibas (most) are fine.

Good luck.
