### Post by Paqman on 2011-10-29
> **Harley36 said:**
> Is there any problem (security,etc) when installing 11.10 as a Windows program instead of in its own partition.

There are a couple of limitations:
[LIST]
[*]You can't use hibernation
[*]Any loss of power can severely corrupt the filesystem
[*]The size of your Ubuntu system is limited by what you choose at install time (it can be changed later, but it's not straightforward)
[*]You can't use RAID or disk encryption
[*]You can't install behind a proxy
[/LIST]
