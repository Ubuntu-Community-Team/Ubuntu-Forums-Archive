### Post by slickymaster on 2015-02-10
First of all, keeping your computer's software up to date is the single most important task for protecting your system. By default, you'll be notified daily for security updates and weekly for non-security updates.

You can change the frequency of the check and the way in which updates are handled, but you must always have present the fact that **Important security updates**, which are the ones that fix critical security flaws and the **Recommended updates**,the ones that fix serious software problems should always be enabled by default.

### Post by grahammechanical on 2015-02-10
If you are using a proprietary video driver then you may have a problem after accepting a kernel upgrade. We can get around that by using Recovery mode>Resume to get to a desktop and then using System Settings>Software and Updates>Additional Drivers tab to change video drivers.

At some time in the future even the users of an LTS release (such as 14.04) will need to upgrade to the next LTS release (16.04). Before doing that it is always best for the OS to be fully updated. System Settings>Software and Updates>Updates tab does give us some control over what we get. For myself, I accept the lot.

Regards.

### Post by ian-weisser on 2015-02-10
> **kingdominsight said:**
> have steered clear of updates involving applications i don't use
Uninstall applications you don't use. Then you won't get prompted to update them anymore.

> **kingdominsight said:**
> i have mostly avoided system updates because the OS is performing very nice now resource wise and in stability.
That's a great way to keep your system vulnerable to newly-discovered exploits.

Your Windows experience with updates is NOT how Ubuntu works.
After release, most Ubuntu updates are critical bugfixes and security patches. Do them..
Most new software versions are snapshotted in the next 6-month release of Ubuntu, not in the daily updates.
