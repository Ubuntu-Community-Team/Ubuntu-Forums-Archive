What you need to be able to do is create a bootable disk or USB -- you will need to then boot from that to do the install.  Copying the file to your hard drive is not going to do this.
 
If you go to pendrivelinux.com, they have an app you can do to make a bootable USB from an ISO file.  You can point it to the Mint file (presuming it's an ISO file) and use that.
