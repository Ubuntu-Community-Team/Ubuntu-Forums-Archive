### Post by Lazy-buntu on 2009-04-09
If you're just looking to reformat the drives, move/resize partitions, etc. pop in a live cd with gparted. Or, you could get a gparted live cd and just use that. The gparted live cd does have a GUI so you don't have to worry about a command line or anything.

If you're familiar with gparted, it should be a breeze.
