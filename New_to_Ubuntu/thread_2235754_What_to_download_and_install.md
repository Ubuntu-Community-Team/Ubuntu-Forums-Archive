### Post by Rob Sayer on 2014-07-23
For old/slow hardware I wouldn't touch the Unity desktop (the "default" one) with a 10 foot pole.  It needs as much power as Windows 7 or 8.

Kubuntu has the KDE desktop, which is also considered bloated but you can configure it to speed it up considerably, which Unity won't do.  But I wouldn't use it in less than 2Gb minimum myself.

For slow machines with less than 2Gb Xubuntu or Lubuntu is better.

The best thing would be to dl all 4 ubuntu flavors, burn them, and try them.  Burn to a USB stick and boot from that if possible.  Optical discs are a bit too slow to really get a feel for how the DE works.

### Post by mastablasta on 2014-07-25
main difference is that Unity uses 3D hardware acceleration to draw windows so besides from RAM a descent GPU is needed. This kind of rendering can be tunred off in Kubuntu, while Xubuntu and Lubuntu do not use it I believe.

ofcourse there are varaiations to this. e.g gnome flashback will not be doing 3D, MATE the fork of old Gnome is also does not need 3D acceleration, Cinnamon (the one Minth uses) needs 3D....

by installing Xubutnu or Lubuntu you free your GPU for other things (watching movies etc.) and they both use much less ram than Unity or KDE. 

KDE can work well with 1GB ram, but you need a descent GPU (something with 1GB ram, & GPU with at least 128 MB ram and descent processor on it would be minimum to get fast & no desktop effects desktop running smoothly more is of course desirable). Kubuntu has (or at least it had) low fat package (for lower spec systems) as well as it has netbook plasma for smaller screens (an interface with larger icons, letters etc.).

For Atoms and such it might be better either to use some netbook interface or use Xubuntu, Lubuntu or just a a WM (iceWM, JWM, Enlightenment... there is plenty of them to choose from).
