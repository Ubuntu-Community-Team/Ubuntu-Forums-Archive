### Post by joetait on 2012-02-22
When I had an ipod I found it would work intermittently with Rhythmbox and Banshee, and eventually I worked out that it wouldn't delete things properly, and then when it was full (though I couldn't see any files) it would stop. Anyway, I ended up downloading gtkpod ipod manager and that worked fine, though the interface is perhaps a bit odd. So I would recommend trying that.
