### Post by MAFoElffen on 2023-02-07
Most likely, you will be able to use bluetooth if installed on metal. A VM is isolated from hardware, unless you do a pass-through.

Doing either an install beside Windows or in place of Windows is a personal decision, left up to you. I have a "Beside" dual-boot, and have for about 18 years. I rarely use Windows these days. But if needed, it is there. It doesn't cost me anything to sit there unused.
