### Post by Paqman on 2010-03-27
> **vivek40 said:**
> Is there I can have all those plasma widgets of KDE on my Gnome desktop without actually having to install KDE desktop and all.


Yes and no.

You could install all the packages you needed to get plasma widgets to work, but in doing so you'd have to install so many KDE dependencies that you'd be installing most of KDE anyway.

There's no reason you can't run KDE apps on Gnome. The problem is that you'll have to load a lot of KDE stuff into memory when you do. That will eat up quite a bit of RAM and make the first KDE app you try to launch open a bit slowly.
