### Post by deadflowr on 2013-02-13
Ok, so Regular Ubuntu, and not KDE variant Kubuntu?

In that case, you'll need to open the dash and search for it.

At the left-side of the screen is a line of icons. This strip is called the launcher.
At the top of the launcher is an icon that has the Ubuntu logo in it.
Click that icon and a sub screen will appear. This is the dash.

Now the top area of the dash has a search bar, type the name of the program you installed.
A list of applications and files will appear, find the one that matches the program you installed and click on it.
It should automatically open, or at least start to open(depending on the system, it might take a minute or two, but once you click on the app in the dash it should start loading it with the program needed to use it.)
