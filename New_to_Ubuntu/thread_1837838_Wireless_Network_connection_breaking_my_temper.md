### Post by Schtenne on 2011-09-04
Ok, I was finally able to solve this problem on my computer (Asus K53S, Ubuntu 11.04) by updating the Linux kernel from version **2.6.38-8-generic x86_64** to **2.6.39-020639rc4-generic x86_64** through following link:

[http://kernel.ubuntu.com/~kernel-ppa....39-rc4-natty/]("http://kernel.ubuntu.com/%7Ekernel-ppa....39-rc4-natty/")

I downloaded and installed **linux-headers-2.6.39-020639rc4_2.6.39-020639rc4.201104191410_all.deb** and **linux-image-2.6.39-020639rc4-generic_2.6.39-020639rc4.201104191410_amd64.deb** and after the reboot, it has worked smoothly since yesterday.

I have noticed that the Atheros AR9285-chipset doesn't work at all for some people. To solve this issue I can recommend this thread: [http://ubuntuforums.org/showthread.php?t=1791389&highlight=atheros+AR9285](http://ubuntuforums.org/showthread.php?t=1791389&highlight=atheros+AR9285) or searching the fora for "Atheros AR9285" and related stuff.

But note that my wireless network card (Atheros AR9285) did work partly before the fix, it was not completely out of order.

Since this method worked for me, this thread will be marked as SOLVED :guitar:
