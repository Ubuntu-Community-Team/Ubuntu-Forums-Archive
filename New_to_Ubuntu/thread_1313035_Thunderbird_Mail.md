### Post by arochester on 2009-11-03
Connect to the Internet, open a Terminal and input the command: > sudo apt-get install thunderbird

TIP:How to copy and paste into your Terminal automatically if you have a wheel (or 3 button) mouse and avoid spelling mistakes, typos and other errors.

1) Highlight the command.
2) Put your cursor anywhere in your Terminal
3) Press the wheel (middle button)

Job done. No typing!

### Post by whitefort on 2009-11-03
Wow!  A question I know how to answer!

From the menu, choose SYSTEM/Administration/Synaptic Package Manager.

Then type Thunderbird in the search box.

Thunderbird will be listed.  Right-click it, and select install, then click 'apply.'

### Post by mlentink on 2009-11-04
You do not really have to use terminal to install (most) software. You certainly do not have to use terminal to install Thunderbird
Go to Applications>Ubuntu Software Center> and click on internet, and search for Thunderbird.
You can also type in Thunderbird in the search box.

Easy, huh?
