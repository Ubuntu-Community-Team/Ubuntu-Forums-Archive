### Post by original_jamingrit on 2009-08-05
Have you installed any software from un-official repositories lately?

In the meantime, if you're using firefox, disable cookies (disabling third-party cookies should be enough), use the NoScript addon for firefox to selectively disable Javascript (Or just turn off Javascript altogether).  If you want to be extra cautious, download a TOR (The Onion Router) addon for firefox or use search sites like [http://www.yauba.com](http://www.yauba.com) .

This is not quite tin-foil-hat level security, but it's pretty close if you're a newbie feeling paranoid.  If you already know your way around a computer well enough, also try watching your outgoing connections for spyware phoning home (if your answer to my first question is yes).

### Post by LewRockwell on 2009-08-05
forbid all cookies

disable java
(had to enable javascript for flashblock)

block all pop-ups

use:

ghostery(notifies you of trackers and gives the ability to block them)

ad-block plus(make sure you subscribe to at least one filter)

flashblock(blocks flash cookies(local shared objects)

firebug(easily see the exact source of each and every webpage as desired)

noscript(self-explanatory)


you'd be surprised how much your browsing will be hampered by the above

it should also make you more cautious and selective in the sites you visit
