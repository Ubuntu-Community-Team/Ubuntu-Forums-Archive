### Post by ace007 on 2008-07-29
NOTE: Copying an xorg.conf identically from a different computer may not work for you.

you'll have to do the copying and pasting via the command line to access root privileges.  

So save the new xorg.conf file to your desktop then open a terminal and:
```

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.BACKUP
sudo mv ~/Desktop/xorg.conf /etc/X11/xorg.conf

```

And that will make a backup of your current xorg.conf and then move the one from your desktop to the location of the old xorg.conf.

With a modern graphics card like yours, you may want to look into installing a better driver than the restricted ones that ubuntu installs by default.

### Post by CatKiller on 2008-07-30
I've not had any problems with the standard restricted driver with my 6200.

If the graphical configuration tool doesn't work for you, you can edit the file manually with ```
gksudo gedit /etc/X11/xorg.conf
``` It's generally worthwhile to make a backup first, as ace007 suggests, since it's quite easy to restore your backup from the command line in case your graphical interface doesn't start.

A good place to look for things to help get the resolution you want is in the **Monitor** section. Many monitor/graphics card combinations do not pass the correct EDID information on, which would enable the resolution to be configured automatically. So in this section you can specify what the refresh ranges should be. In your case, you could put ```
HorizSync    31-81
VertRefresh    56.3-75
``` (numbers taken from the Acer website)

You can also tell your video card not to bother passing on any EDID numbers in case they conflict with the ones you've just manually entered by putting ```
    Option    "UseEDID"    "False"
``` in the **Device** section.

Another thing that you can try is to specify the resolutions that you want to use, in the **Screen** section. For example, part of my Screen section has the following: ```
        SubSection "Display"
                Depth           24
                Modes           "1600x1200" "1280x1024" "1024x768" "800x600" "640x480"
        EndSubSection
``` There is one of these for each of the bit-depths 1, 4, 8, 15, 16 & 24. Put in whichever resolutions your monitor can do.

**Ctrl-Alt-Backspace** will kill and (hopefully) restart X for your changes to take effect.

EDIT: The Driver line in the **Device** section allows you to choose between the restricted driver (nvidia), if it's installed, and the open-source driver (nv). If you get a refresh rate in the 50s, it's probably because you're using Compiz (for all the Desktop Effects). For some reason, it reports the refresh rate incorrectly, so it will say that it's a refresh rate in the 50s when it's actually much higher.

EDIT 2: If you want to know more about permissions and the *root* user, read this page: [https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo)
