### Post by Keiran230 on 2011-02-18
There's a chance it's an .iso file already and since you have WinRAR installed, WinRAR has associated itself with the file and changed the icon.

You can unhide extensions by opening a random folder, tapping ALT to get the file menu, going to tools > folder options, view tab, then uncheck "hide extensions for known file types."

If you're running XP, ImgBurn is a decent "free" (but not open source) image burner.
