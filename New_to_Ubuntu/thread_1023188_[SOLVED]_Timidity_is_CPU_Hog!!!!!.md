### Post by Kellemora on 2008-12-27
Hi Gang

For the past couple of days I've noticed my CPU was running at a constant 100% duty cycle.

Absolutely nothing showed up in System Monitor that totaled over 18% and that was only at peaks.

After heating my office up to 81 degrees via the CPU I thought I had better dig a little deeper.

Ran TOP and found Timidity was using 98% of the CPU and that was with NO OTHER PROGRAMS RUNNING!!!!!

Restarted X since that often fixes things like this, no affect!

A cold reboot will fix it for a few days, but something causes Timidity to start hogging the CPU again.  Have no idea what?

But I cured the problem by uninstalling Timidity and making a note in my notepad of the problems it caused!

I think it was installed to try to fix another problem that has gone unresolved now for over 3 months.

Trying to play a Midi file from a web site produces 0.46% cache then it stops downloading.  The same file when selected manually from nearly any player will play just fine!

Don't need a response here, just posting the heads up in case others are finding their CPU heating the room up!
