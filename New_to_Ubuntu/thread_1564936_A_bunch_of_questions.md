### Post by Thryn on 2010-08-31
> **migs73 said:**
> Click on Places--> Search for files.. then type in the name of the file you are looking for and where you want to look for it in the dropdown menu.
And if you want to use this often, you can right click on any panel (by default at the top and bottom of your screen) and use the menu to add a search icon.

As for dictionaries I use Stardict, which is available through the software center (applications menu) or Synaptic. Checking [their website]("http://stardict.sourceforge.net/Dictionaries_misc.php") I see that there are both Greek-English and English-Greek dictionaries available.
