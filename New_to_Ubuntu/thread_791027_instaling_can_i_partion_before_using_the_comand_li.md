### Post by Aearenda on 2008-05-11
Sounds like you really should seek help from a local Linux user group. 

If you can't do that I would:
1. Check that the computer's BIOS recognises the new hard drive
(I can't tell you how exactly as I don't have a Thinkpad)
2. On another computer, download and burn the Alternate installation CD
3. Boot from that CD
4. Choose your language then make it verify the CD
5. Run the memory test for a few hours
6. If no errors, reboot from the CD and begin installation
7. Take the options through the partitioning steps so that the setup uses the entire drive and formats it for you.

If that does not work, I would find a Windows XP installation CD and see if that can be installed (does not need to be activated just for this test). If this also does not work, I strongly suspect you have a hardware problem and you should seek help locally.
