### Post by NightwishFan on 2010-12-26
Well Mr. Dictator, welcome to Ubuntu! Here is some documentation regarding a dual boot between Windows and Ubuntu:
[https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)

Ubuntu will be a fully separate entity from Windows and will not share files or applications. (Though you can browse Windows from Ubuntu, not the other way around).

The biggest snag you will hit is of course partitioning. I suggest you use a Windows tool to do this, shrink space before hand using the Windows tool, and install Ubuntu in this **continuous** free space. Ubuntu needs only 2 partitions, so if you only have 2, Ubuntu will create the other 2 in the free space on it's own.

The reason I say use a Windows tool is Windows can be picky, and might not want to boot if another tool modifies it. You can find a tool if you right click My Computer, and hit "Manage". There should be a disk/partition utility there. I think it does not have the ability to modify a partition in use though, so you might have to unmount the second partition first. Someone who knows Windows better than I can help you with that. I will possibly be around to answer any specific questions for you. :)

### Post by candtalan on 2010-12-26
> **dave williams said:**
> computer active Linux for home users with a disc inside. With instructions on how to install etc

Welcome!
I have a friend who also tried to begin by using the Computer Active approach, and unfortunately it did not go smoothly, perhaps because he did not quite understand some of the instructions, I do not know. However, I then tried to help him after the event, and there was a significant problem. The problem was that the CD and instructions he used were nothing like what I was familiar with. The Computer Active CD is not an ordinary Live CD, so once you have begun you will be unlikely to find easy help from anyone here in these excellent forums.
The Computer Active suggestion of creating a partition in preparation for (Ubuntu) also suggests that you will be expected to use the 'Advanced' Method of installation, not the semi automatic and assisted 'resizing' of Windows.

My advice to my friend, and finally what he actually did, was to start again, with using only a well known Ubuntu Live CD. 

You are nervous about loosing your Windows, and that is sensible. The worst that can happen is that if things go wrong you will loose everything and have to re install Windows from scratch, copy in your windows data from your backups, and again install Ubuntu. That is a lot of work. And if you have never yet installed Windows you will find it not as easy as installing Ubuntu.

So please consider the situation. 

Do you have good backups of your Windows data?
Do you have an easy way or reinstalling Windows such as a partition image? Or a full re installation CD?

Perhaps you have an older and spare PC that you can get started with?

Where are you located? Maybe there is a local group who can help?
