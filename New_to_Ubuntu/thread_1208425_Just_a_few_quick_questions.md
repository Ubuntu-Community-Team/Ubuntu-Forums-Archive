### Post by random turnip on 2009-07-09
Unless your very capable with Linux, i strongly advise not logging in as a root user unless you don't mind reinstalling if/when you break something.
Sounds scary, but in a normal user you have more than enough power to do everything you will need for basic computing.  Normal user is very safe, no viruses :)

To change res go to System --> Preferences/Admin (can't remember which one off by heart) --> Screen size (something like that) and it will have a list of available sizes.
You can do it manually in the terminal, but i don't know the commands.

You won't be in root to start with unless you specifically chose to be.
You can creat an account by going System --> Admin --> Accounts (again, i think it's called accounts, you might have to click a few options that look like a logical place for it to be, you can usually guess)
And there will be an option there somewhere for sure.

Like i say, things like this are usually guessable once you have gotten used to Ubuntu a bit.

### Post by The Cog on 2009-07-09
For the font size, rather than changing the screen resolution, I prefer to tinker with System->Preferences->Appearance, Fonts, Details, and change the dots-per-inch resolution. The more dots-per-inch, the smaller it thinks the screen is and the larger it makes the fonts to compensate.
