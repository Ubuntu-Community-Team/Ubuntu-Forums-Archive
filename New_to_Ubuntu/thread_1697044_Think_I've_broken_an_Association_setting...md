### Post by Frogs Hair on 2011-02-28
Hi and Welcome.
 
1. right click desktop
2. select create folder
3. right click on the folder
4. select open with another application
5. select file browser from the list
6. select open
7. test the places menu
This will reset the defualt applicaion for opening folders , you can remove the folder when finished.
