### Post by twisted_steel on 2009-11-12
Welcome to the forums :)

There is a driver for Windows called [Ext2Fsd]("http://www.ext2fsd.com/"), but it does not yet work with ext4, the default file system in 9.10 (assuming you did a clean install).

It is possible to stream to the 360 from Ubuntu.  I have not done it recently, and just did a fresh install on my two computers, so I have not been able to test it.  However, I do remember seeing a few programs out there in the past that are supposed to work.  Look at [uShare]("http://ushare.geexbox.org/"), [FUPPES]("http://fuppes.ulrich-voelkel.de/"), and [TwonkyMedia]("http://twonkymedia.com/").  Twonky is not free, but they do have a trial.  I have used the first two in the past with some success.
