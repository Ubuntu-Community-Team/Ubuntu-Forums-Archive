### Post by Jest3r on 2010-07-01
The only n64 emulator that i know of with an online function is [project 64 (http://www.pj64-emu.com/)]("http://www.pj64-emu.com/").

However this is a windows application, so you would need to run it under [wine (https://help.ubuntu.com/community/Wine)]("https://help.ubuntu.com/community/Wine"), which may impact the performance.

If you want to play the n64 games without linux, then check up [mupen64 (http://mupen64.emulation64.com/)]("http://mupen64.emulation64.com/").
