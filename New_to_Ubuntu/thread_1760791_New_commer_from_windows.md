### Post by compmodder26 on 2011-05-17
Best bet for using 100% of windows programs is running a virtual machine.  You can try to install Wine from the software center which will allow you to run your windows programs as if they were native linux programs, but not all windows programs behave with Wine.

Of course you can try to find linux alternatives to the windows programs you need.  Perhaps you could list out your mission critical windows programs and we could try to provide alternatives for you.

### Post by Prince Valiant on 2011-05-17
There's a program called Wine ([http://www.winehq.org/](http://www.winehq.org/)) which can run a large number of windows programs.  Some versions of wine work better than others for specific programs, so you might want to look up which works best for your particular software.
