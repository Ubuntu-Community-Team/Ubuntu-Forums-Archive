### Post by I8NY on 2011-01-25
Type "top" in the terminal and that will show you what is using some of the cpu. (It is better than the system monitor)  Hopefully the trouble program will be at the top and depending on what it does you can uninstall it or fix it.  For the speakers make sure the sound settings are set to the correct number of speakers.  Another thing to try is install ALSA to replace pulseaudio.  There are some guides to install it.  Hope this helps.
