### Post by greenblob on 2011-12-22
There are applications made to help transfer your applications from one Ubuntu install to another,but none of them seem to work very well.

It's possible to just copy over your home folder or whatever folder contains the settings between different installs as far as I know, as I've never had any trouble doing this.

If you make a booteable Ubuntu USB drive that's at least 1GB in size, you can reserve the rest of the space in the drive (size of your drive minus install size - roughly 700MB) to save settings and applications between sessions. A 2GB drive is more than enough to copy settings and a large sum of applications.

hope this helps ;)

### Post by mcduck on 2011-12-22
> **BNZBKK said:**
> Can I save my 'settings' to a USB and after a re-install have exactly the same Desktop, Folders etc, etc

..and if so, how ?

All the user-specific settings are stored in hidden files & directories inside your home. So if you just back up everything from your home directory to the USB drive you'll only need to worry about installing the applications themselves back after the re-install.
