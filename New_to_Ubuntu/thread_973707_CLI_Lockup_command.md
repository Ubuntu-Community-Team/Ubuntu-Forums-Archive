### Post by hictio on 2008-11-06
On a console, you can type 'Ctrl + S' to lock the screen, unlock it with 'Ctrl + Q'; but there is no password asked.

### Post by hictio on 2008-11-06
More options:

[vlock](http://freshmeat.net/projects/vlock/)
[lockvc](https://launchpad.net/ubuntu/+source/lockvc/4.0.5-7)

You can install those thru aptitude, on your (unlocked) terminal :D
