I can tell you from experience that it CAN happen yes.

My experience in this occurring was with OS X, not Ubuntu but I'm sure it applies all the same. I was using my webcam to make a video when my toddler unplugged the computer from the wall. From that time forward the application I was using to film the video would not work properly. It would lock up constantly and i was having to force quit it all the time.

Techs from Apple ran diagnostics and discovered that I did have some file corruption. My only choices were to migrate to another user or re-install my entire OS if I wanted everything working properly. I ended up choosing the latter.
