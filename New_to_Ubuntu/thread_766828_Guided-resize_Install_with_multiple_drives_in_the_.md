### Post by VanCardboardbox on 2008-04-25
I took another look at manual partitioning in the install. Yes, you can select a drive to edit and create a partition. But the app's display, unlike the guided resize, simply shows a new partition size and indicates nothing about what will happed to the *existing* partition - it did not instill confidence that my XP partition would remain intact.

So I did it the hard(ware) way - opened the box, temporarily disabled all but my XP drive and used the guided-resize. Ubuntu installed exactly where I wanted it, the dual boot is working as it should, and my new OS is running fine.
