### Post by ronparent on 2010-06-29
A start is to left click anywhere on empty desktop and from the popup box select 'Change Desktop Backgrounds'. From the Appearance Preferences' window select the 'Fonts' tab. At the end of each font listed you will see the font size (probably 10). Select each one in turn and from the dropdown list opened select the size you want, The text will be previewed in a box at the bottom of this window that you opened. When you have everything how you like it just close the window.

Individual applications may over ride the default settings and you may have to adjust fonts within that application. Also how each text message displays is often based on the original defaults and you could get jumbled text messages in sume cases.
