### Post by Giant Speck on 2009-04-12
The only thing I can think of is that the extension of the image you are trying to upload is in all caps.

Does the image's filename look like this in your file browser?

example.JPEG
example.PNG
example.GIF

If so, change the file extension to lowercase letters:

example.jpeg
example.png
example.gif

And then try re-uploading the images.
