### Post by TBABill on 2011-02-10
I think you are probably wanting the Flash Plugin. If you want that and a few other goodies to help with multimedia you can open terminal and ```
sudo apt-get install ubuntu-restricted-extras
``` or you can do the same thing by searching for it in either Synaptic package manager and marking it for installation, then clicking apply, or even easier, just go into Applications>>Ubuntu Software Center, search for it and click it, then click install. Either way is fine.
