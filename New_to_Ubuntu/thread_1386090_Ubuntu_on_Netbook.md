### Post by BT1 on 2010-01-20
> **ex-wirecutter said:**
> I am about to buy a netbook with Windows XP installed , but before I do I would like to know if I can install Ubuntu as dual boot , and if so how , as the netbook doesn't have a CD drive . I have memory sticks/flash drives but not sure if you can alter the boot sequence on a netbook.

Google "Unetbootin" which lets you write disk images to thumb drives. You can then tell your laptop to boot up through a hard drive to run the LiveCD/Installer.

Ubuntu has its own netbook flavor (which I'm using right now in class) that runs very fast. Also Kubuntu has its own netbook flavor too. (google both for download links) You can get away with installing regular Ubuntu Desktop, but some functions won't work right. Like program windows will sometimes be too big to be displayed correctly (like Disk Utility).

You can tell the installer to set up a dual boot automatically, or if you know enough about partitioning, you can do it yourself. If you install ubuntu second, it will make a new boot loader that will let you pick between starting up windows xp or ubuntu. This is usually the easier option.

Have fun :)

### Post by spiderbatdad on 2010-01-20
It would be helpful to know the brand of netbook you are interested in. User experience and ease of installation vary from model to model. I can vouch for the acer aspire one as being extremely Ubuntu friendly...or vice versa.
As to installation. What works best for me is using a live Ubuntu cd of 8.10 not 9.10, and using the usb image writer from the live cd to write a saved 9.10 iso (saved on external drive) to write to the usb stick. Others use unetbootin to write to the usb from windows. I have not used unetbootin.
