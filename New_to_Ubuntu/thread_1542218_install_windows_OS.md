### Post by cdated on 2010-07-30
I hate to be evasive in response to a straight forward question, but @Paqman has a point.  Most Windows functionality can be duplicated on Ubuntu, you just have to find alternative applications.  If you intend on using strictly internet services like email, youtube, instant messenger, etc then we can help you cope; however, using Microsoft products (ie MS Office) will not be possible without a bit of effort and possibly extra money to buy software licenses from Microsoft.

Frankly, if you have a windows install disc and license key you can buy/borrow a USB cdrom drive install it without any trouble.  You will lose your Ubuntu in the process.  Please note that MS Windows is not a program that you can simply add with the Ubuntu Software Center or Add/Remove Programs dialog, it is what runs your entire machine like Ubuntu is doing now.  Best of Luck.
