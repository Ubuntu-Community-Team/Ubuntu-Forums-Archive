### Post by grahammechanical on 2015-08-08
You are misunderstanding something very important here.

The Ubuntu Software Centre has many, many free applications but it also allows developers to sell their applications. Are you going to work for nothing? What is wrong with a programmer wanting to sell an application?

I open the Ubuntu Software Centre and I search for "weather" and I see 4 weather applications and they are all free. I open the dash and I search for "weather.london" and the weather scope opens with the weather for where I live. If I click "10 days" then the browser will open at the weather channel.

Nobody is being scammed. Linux is free but it is free as in "free speech" not as in "free beer." There is a lot of "free beer" in Linux but there is nothing in the philosophy of Free and Open Source Software (FOSS) that says a person is not allowed to earn a living. It is what you want to do is it not? Earn a living?

### Post by sandyd on 2015-08-08
As v3.xx mentioned, you can use synaptic instead of the Ubuntu Software Center. You don't even need to open it.

Think of it as Ubuntu's "Marketplace", similar to the Windows Store, which has both mixed free and paid content. The choice to choose another GUI for managing packages is out there, and you do not need to use Ubuntu Software Center, which is the only app that has sells paid software simply because it is part of UbuntuOne.

### Post by mystics on 2015-08-09
> **jerry_k said:**
> with Kali I got a full penetration suite for free

I've used both Kali and Ubuntu for similar tasks, and as far as I know, there isn't anything pre-installed in Kali that you can't install for completely free on Ubuntu. So if there are things in Kali that you would like to have in Ubuntu, just install them in Ubuntu. It won't cost you anything.

Also, no one is forcing you to buy any of the apps on the Ubuntu Software Center, especially since there are often plenty of free alternatives. But if you are that put off from using anything that offers paid options to people, then you could follow other people's advice and download the Synaptic Package Manager. Just type

```
sudo apt-get install synaptic
```

if it isn't already installed. It isn't as user-friendly as the Ubuntu Software Center, but it is all free of cost.
