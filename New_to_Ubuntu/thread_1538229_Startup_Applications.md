### Post by Anuovis on 2010-07-24
> **selittl said:**
>   What directory are application files usually found?
Frankly, I am not sure myself. I think it is */usr/bin*.

But you do not need the location if the program appears in the menu. Just right click on that big menu *Applications | Places | System* and pick *Edit menus*. Then find an app you want to add and click *Properties*. You will be able to see the command to launch it (see the sreenshot).

Another way is to press *Alt+F2*. An application launcher will open. If you start typing the name in the box it usually completes it automatically or you could also scroll the list and find it there.

More often than not it is just a name of an app but sometimes a more quirky command must be used, it really depends.
