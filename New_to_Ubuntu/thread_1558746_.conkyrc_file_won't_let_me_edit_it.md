### Post by stoogiebuncho on 2010-08-22
Your .conkyrc file should be saved in your /home/username/ folder.  Is that where yours is saved?

Assuming it is, you can give yourself permission to edit it so that you don't have to put in your password every time you want to make a change.

Press Alt+F2, and type: gksudo nautilus

This will open up a file manager with root access.  Find your .conkyrc file and right-click on it, and go to Properties.  Go to the permissions tag and give yourself permission to read and write.  After that you should be able to edit it without using your password.

(Note: A lot of system files will not work if you change their permissions, so don't go doing this to every file you find.  But your .conkyrc file is supposed to be owned by you, so you can change the permissions without worrying.)
