### Post by bodhi.zazen on 2009-04-15
First install build essential.

```
sudo apt-get install build-essential
```

Then install the dependencies, then build the drivers.

See : [http://intellinuxgraphics.org/install.html](http://intellinuxgraphics.org/install.html)
