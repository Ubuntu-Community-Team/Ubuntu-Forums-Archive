cat router.log > router_'%Y%m%M'_'%H%R'.log - rename

### Post by Monicker on 2008-05-16
Use the date command to create your timestamp string

```
#!/bin/bash

tstamp1=`date +%Y%m%M`
tstamp2=`date +%H%R`

echo "router_${tstamp1}_${tstamp2}.log"
```
