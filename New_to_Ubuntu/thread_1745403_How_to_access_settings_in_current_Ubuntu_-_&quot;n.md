### Post by Hedgehog1 on 2011-05-01
It is 'Natty Narwhal', by the way...

Basic settings are under the 'power' button menu now:

[IMG]http://img820.imageshack.us/img820/7027/nattysystemsetting.png[/IMG]

You cannot run ***all*** the Compiz items in Unity, but you can in the 'Classic' desktop: [**_Natty Info: Your UI options_**]("http://ubuntuforums.org/showthread.php?t=1741293")

To make changes to Compiz, please run the Ubuntu Software Center and search for 'ccsm' - install this to make changes.

There is a new item for Unity in ccsm:

[IMG]http://img51.imageshack.us/img51/558/ntchangeiconssize.png[/IMG]

This allows you to change the icon size in the Unity launcher and make the top panel transparent, too.
