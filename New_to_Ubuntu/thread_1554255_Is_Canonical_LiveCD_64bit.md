### Post by ubudog on 2010-08-16
It probably depends on which iso you downloaded.  You can find out by typing this in a terminal on the livecd (while running it of course):
```
uname -a
```

If it says anything about i386, (or maybe i686) then it is 32bit.
If it says anything else in that section, then it is probably 64bit.
