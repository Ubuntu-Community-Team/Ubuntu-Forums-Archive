### Post by grahammechanical on 2011-04-30
Whenever there is a kernel update, the latest kernel is always put on the top of the list. The upgrade to 11.04 also updated the kernel.

I use a utility called Grub Customizer (search for grub-customizer). That lets you mark the order the operating systems appear in the Grub2 menu. It does other good things as well.
