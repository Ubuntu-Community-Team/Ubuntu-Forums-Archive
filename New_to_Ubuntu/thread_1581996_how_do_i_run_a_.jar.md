### Post by Nytram on 2010-09-25
I don't know if it's the same on Xubuntu, but here on Gnome it's just a case of right-clicking a .jar file and choosing open with Java. You'll need the JRE installed if you haven't already (Java Runtime Environment.)

### Post by skyjacker on 2010-09-25
Right click on the file and choose Properties, Permissions tab, and check the box that reads "allow executing file as a program".
