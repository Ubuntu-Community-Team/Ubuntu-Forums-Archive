### Post by elperrillo on 2011-03-08
It is very easy, you just have to make sure not to use the fake "remote desktop" that won't connect to any Windows machines because it is really VNC and it is not compatible with the real Windows Desktop, to use the real Windows desktop you have to use "rdesktop" which comes with Ubuntu but you won't find it on any menu.

Read this:

[http://geekyprojects.com/tutorials/how-to-use-the-real-windows-remote-destop-in-ubuntu/](http://geekyprojects.com/tutorials/how-to-use-the-real-windows-remote-destop-in-ubuntu/)
