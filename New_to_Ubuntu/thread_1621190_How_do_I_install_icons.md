That's not going to be easy, since it's not an icpn set made for Gnome desktop but for Windows.

If I remember right Gnome should support .ico files, but you still need to create a theme out of them first.

The easiest way would probably be making a copy of some existing theme, and then starting to replace the old images with the ones you want.

(Icons sets made for Gnome can be easily installed as described in the above post, or by manually extracting the icon package into ~/.icons or /usr/share/icons.)
