### Post by searchfgold6789 on 2012-12-22
> **ralphmiddleton said:**
> How can I update ubuntu 10.04 to 11.04 ?
If you NEED 11.04, rather than the latest version of Ubuntu, you can use the Live CD for 11.04 to do the upgrade.

Otherwise, upgrade to 12.04 as ibjsp4 suggested. There are simple instructions for doing so here; all you need to do is boot into Ubuntu 10.04 and have a network connection: [https://help.ubuntu.com/community/PreciseUpgrades#Network_Upgrade_for_Ubuntu_Desktops_10.04_LTS_to_12.04_LTS_.28Recommended.29](https://help.ubuntu.com/community/PreciseUpgrades#Network_Upgrade_for_Ubuntu_Desktops_10.04_LTS_to_12.04_LTS_.28Recommended.29)

### Post by Wim Sturkenboom on 2012-12-23
I've done upgrades via the web with mixed results. 6.06 to 8.04 worked fine. 8.04 to 10.04 and it said at the end that I could possibly end with a instable system. They lied :) I had a very stable busybox after reboot :(

If you want to go to the, no longer supported, 11.04 you have to go via the no longer supported 10.10. And next you have to goto 11.10 which is the oldest supported non-LTS version (till april 2013).

From both above, I would do a backup of your documents and do a fresh install of 12.04.
