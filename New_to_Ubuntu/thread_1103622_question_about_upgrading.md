### Post by anchorschmidt on 2009-05-31
I've used ext3 and ext4 and the latter gives you much faster performance. I did an upgrade to Jaunty and it gave basically the same performance as Intrepid. However, after doing a clean installation, I can notice the performance changes. So my advice would be backing up your data and doing a clean installation. If you don't want to, you can always use Intrepid till 2010.
