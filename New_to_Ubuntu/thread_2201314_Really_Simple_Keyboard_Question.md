### Post by Keith_Beef on 2014-03-27
Open the System Settings window; in the Personal section select Keyboard Layout.

Select the Layouts tab.

If there are several layout, for each layout in the list, select the layout and then click on the - button to remove it until there is only one left.

Now add the English UK layout: click on the + button to get a list of layouts, and from the list select English (UK) then click on the Add button.

Now go to the keyboard layout selector applet and select the UK layout (since I added this to the US layout, it is labelled en[sub]2[/sub]) and then also in that applet select Show Layout Chart.
