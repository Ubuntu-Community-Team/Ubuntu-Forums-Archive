### Post by Mark Phelps on 2010-08-02
There are no ATI proprietary drives that will work with your card and current versions of Ubuntu. Period.

Trying to force the install of ATI drivers is only going to corrupt your display.

If you didn't actually install the ATI drivers, then you have the only available drivers already installed -- the open source drivers.
