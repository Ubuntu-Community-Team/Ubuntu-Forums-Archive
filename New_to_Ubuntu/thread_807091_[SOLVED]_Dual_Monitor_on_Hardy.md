### Post by nacho-wan on 2008-06-08
> **ingrid_ozikem said:**
> You are lucky.. with Nvidia, you should be able to get it working (AND have compiz turned on) by following one of these links,

[http://ubuntuforums.org/showthread.php?t=221174](http://ubuntuforums.org/showthread.php?t=221174)
[http://gentoo-wiki.com/HOWTO_Dual_Monitors](http://gentoo-wiki.com/HOWTO_Dual_Monitors)

 It is just that with an Intel card (like many laptops), Hardy seems to not have an Driver mentioned in the Device section. There seems to be an unspoken assumption on what to use somehow and that's annoying when you want to modify it to set up a dual head. 

Does anyone know how to configure an Intel graphics card through xorg.conf in Hardy?

Well I manage to did it last night. You have to open your xorg.conf on gedit: go to terminal and type:
```
sudo gedit /etc/X11/xorg.conf
```

Then look for the Device stanza. You should add a line with the following information Driver "Intel". The stanza should end as this:
```
Section "Device"
	Identifier	"Configured Video Device"
	Driver		"intel"
EndSection
```
This tells X to use the intel driver instead of guessing as it does by default.

Now change your screen stanza to look like this:
```
Section "Screen"
	Identifier	"Default Screen"
	Monitor		"Configured Monitor"
	Device		"Configured Video Device"
	SubSection "Display"
		Modes		"1280x1024" "1024x768" "832x624" "800x600" "720x400" "640x480"
		Virtual	2560 1024 
	EndSubSection
EndSection
```
The virtual line tells ubuntu the maximum size of the virtual desktop, so you choose a size that covers the resolution of both monitors.

Now you can use xrandr to create a dual head setup. In my case I sue the following command:
```
xrandr --output VGA --auto --right-of LVDS
```
This tells ubuntu to place the external screen at the right of you laptop screen. And before somebody asks, by using this solution you can keep compiz working. The only thing that I'm having problems is with the desktop, because the wallpaper (even set to zoom) gets cut at the right leaving a blank area. :( However you can put a window on top and forget about it:)... still I would be grateful if somebody tells me how to correct that!
