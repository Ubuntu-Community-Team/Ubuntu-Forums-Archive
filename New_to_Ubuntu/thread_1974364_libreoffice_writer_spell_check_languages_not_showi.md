### Post by frytek on 2012-05-19
I got the same problem. Only my environmental spelling works, but none of other languages. Seems like Libre Office can't see any spelling engines on 11.04 except the main system language. 

I tried to install all spelling engines (hunspell, myspell, aspell) etc. but this did not work. 


Then I went here: 
[http://extensions.libreoffice.org/extension-center?getCategories=Dictionary](http://extensions.libreoffice.org/extension-center?getCategories=Dictionary)

and downloaded spell checker as oxt and installed it from the Tools / Extensions manager menu. 

And it works. This could be a solution for you.
