Please define "anything".  Whenever possible, you should install software from the Ubuntu repositories; installation is very easy, and updates are automatic.

As for Urban Terror, the download web site has this advice for installing on Mac or Linux:

> Just unzip the zip and make a shortcut to the ioUrbanTerror executable.
Mac executable: ioUrbanTerror.app
Linux 32bits executable: ioUrbanTerror.i386
Linux 64bits executable: ioUrbanTerror.x86_64
In linux, you need to make it executable first: right click it, properties and tick the 'allow execution' box.


To unzip the file, find it in nautilus (the Gnome file manager) and double-click it.  It will open in Archive Manager (file-roller) - just hit Extract and select the directory you want to install to (somewhere in your home directory, and you should probably create a new directory for it beforehand).

The web site's instructions on making the file executable need a little modification:  right-click, select Properties, select the Permissions tab, then check the box labeled 'Allow executing file as program'.  You should then be able to double-click ioUrbanTerror.i386 to run the game.
