### Post by adam_kimber on 2009-01-26
As Ubuntu is based on the Debian system they have copied Debain's nice web front end to the packages available for a specific release. This can be found at [http://packages.ubuntu.com/](http://packages.ubuntu.com/) 

Then you just search for the package in question. [http://packages.ubuntu.com/search?keywords=glib&searchon=names&suite=intrepid&section=all](http://packages.ubuntu.com/search?keywords=glib&searchon=names&suite=intrepid&section=all)

Then click on the distribution below the package for more information. I believe this is the one you are looking for [http://packages.ubuntu.com/intrepid/libglib2.0-data](http://packages.ubuntu.com/intrepid/libglib2.0-data)
