### Post by papuccino1 on 2009-09-09
How can I stop Ubuntu from changing my screen brightness whenever it feels like it? 

I know that I'm running on battery, don't change my brightness!!!:guitar:

---

### Post by filifunk on 2009-09-09
System --> Preferences --> Power Management---> on battery power

uncheck "reduce blacklight brightness"

Hope this helps!

---

### Post by papuccino1 on 2009-09-09
Yep worked like a charm. Thanks a bunch.
