How do install a new GDM Theme, I think thats the login screen theme. Andf how do install a screenlet and desklet? I'm using GNOME Desktop Enviornment.

---

### Post by unutbu on 2008-05-29
To choose a new GDM Theme: Click on System>Administration>Login Window

To install screenlets:
Open a terminal (Applications>Accessories>Terminal) and type

```
sudo apt-get install screenlets
```

To install desklets:

```
sudo apt-get install gdesklets
```
