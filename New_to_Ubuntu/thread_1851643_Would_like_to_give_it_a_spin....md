### Post by Paddy Landau on 2011-09-28
> **hednik4am said:**
> 1. I can't get into setup to boot from a CD or USB thumb drive. How would I do a clean install ?
Unfortunately, that is the only way. However, if you clean your XP, you can install WUBI, which will give you a good experience of Ubuntu. Sadly, WUBI is unstable and therefore is not suitable as a long-term solution.

> **hednik4am said:**
> 2. Should I tr to do a dual boot or is this possible ? A corrupted version of XP might stopping this ?
Windows, corrupted or otherwise, cannot prevent you from installing Ubuntu. But you need to be able to boot from a CD or USB.

> **hednik4am said:**
> 3. What's the learning curve like ?
Steep, but fun. And these forums are friendly and patient, so you'll get plenty of help.

> **hednik4am said:**
> 4. In short, can installing Ubuntu (which looks awesome and many people seem to love it) help me save this laptop ?
Yes, if the hardware is supported.
