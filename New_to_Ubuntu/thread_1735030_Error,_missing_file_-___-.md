### Post by dwasifar on 2011-04-20
A LiveCD is an actual CD that you burn and boot from.  You may be thinking of Wubi, wherein you install Ubuntu to run inside Windows.

Your problems certainly could be related to using the beta.  I've had installer problems with the beta myself.  This particular release seems to have more problems than most.  I wouldn't recommend you using any Linux beta until you're more familiar with Linux; it's likely to throw you problems you're not equipped to solve yet.

If you're just wanting to install Ubuntu in place of Windows, or as a dual boot system (where you choose whether to boot to Windows or Ubuntu when you start the machine), then I would recommend using a CD.  It's the most reliable, straightforward, and trouble-free way to install.  Download the .iso of the current stable release (10.10), burn it to a CD, and boot from that.
