### Post by avtolle on 2008-06-07
If you don't want to partition, you could install 8.04 using Wubi. This creates a folder inside Windows which runs Ubuntu "natively", it is not a virtual machine in other words. You may also dual boot with Vista, but this will require partitioning your drive and installation of Ubuntu on the HDD. 

As for your apps, there are Windows apps that run under Ubuntu using Wine. There are, for example, linux apps that do the same thing, or nearly the same thing, as some Windows apps, such as Open Office for MS Office. There's a lot more, I suggest you search the forums for ideas. Good luck.

Basically here is what happens when you try to install Ubuntu.

1. Insert LiveCD
2. It boots up and you are in the desktop.
3. You can use desktop as a NORMAL computer (but it will run from livecd) or you can click on the install icon.
4. There are some general options. 
5. Then you will get a choice (something along the lines of)
a) install ubuntu by erasing entire hard drive DO NOT USE THIS
b) install ubuntu using the largest free space DO THIS
c) manual edit 

If you choose b) ubuntu installer should do all the partition management for you!

2. If you use the recovery disc USUALLY it wipes out everything on your hard drive and restores to original factory settings, so its just a case of popping it in and using it.

3. Don't worry the beauty of software is that as long as your data is backed up and safe you can erase and re-install as many times as you want without any major damage to your computer.

### Post by Sealbhach on 2008-06-07
> **oregoncrete said:**
> 

1) Dual Boot interests me, seems hard, how can I easily do it, and will I F*** up my computer if I don't know how?


No, it's not at all difficult.

First, I would advise you to get an installation CD (also called Live CD). When you boot up from that, you are already running Ubuntu, but from the CD drive, it doesn't touch your hard drive.

You'll see on the desktop an Install icon, but don't click it if you want to leave your system unchanged.

It's a good way of seeing how well Ubuntu detects your hardware. When you finsh your session and remove the CD, it's as if it never happened.

If you decide to install, I recommend a dual-boot if you're new. Once you're more comfortable with Linux, you can always re-install Ubuntu only.

This walk-through [here (with screenshots)]("http://www.psychocats.net/ubuntu/installing") shows you exactly how to do it. It really is very easy, but of course, be sure to make a back-up copy of anything you don't want to lose.

If anything does go wrong you will find help on the forum here and very quickly get it solved - that's my experience.

All the best!
