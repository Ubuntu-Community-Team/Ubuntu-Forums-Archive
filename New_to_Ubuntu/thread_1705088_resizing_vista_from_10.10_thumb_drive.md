### Post by guilleme on 2011-03-11
You can resize it with many tools. For example:
Gparted, on command line or graphical mode.
The ubuntu installer, which you can use to resize things.
A lot of other command line tools.
It is not only possible, but not that hard. It should be possible to resize it from the Ubuntu installation on your hard drive, but at least more difficult.
