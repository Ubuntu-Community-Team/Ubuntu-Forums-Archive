### Post by 3rdalbum on 2012-09-15
They might be able to have a look at one in store, and ask the salesperson if they can boot an Ubuntu live CD to test if it works.

Otherwise they can try and find out the model of touchscreen hardware that's used in the computer, and do a search for the model number plus the word "Ubuntu".

For example, if the actual touchscreen is a TouchEx T-103, they could look up "T-103 Ubuntu 12.04" and see whether anyone has mentioned that it works.
