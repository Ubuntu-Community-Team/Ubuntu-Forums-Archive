### Post by pythonscript on 2009-09-29
I started out using Evolution because it was built into gnome, but I ended up switching to Thunderbird because the e-mail functionality seemed more stable (Evolution crashed on me when accessing my gmail account with 19000+ conversations) but Thunderbird never had this problem. ### Post by bigboy_pdb on 2009-09-29
With respect to functionality, they both seem good.

I started using off using Evolution. There's a few reasons why I switched to Thunderbird. Every so often Evolution would get stuck in a state where it can't connect to the server. Also, Thunderbird seems to be moving in a direction where it will become one of the more widely used free e-mail clients. Furthermore, I've installed Thunderbird on my wife's computer (and it's easier to know one e-mail client well than two).

My problem with Thunderbird is that the contacts don't hold anywhere near as much information as Evolution's contacts do. Also, there seems to be a bug where only the last custom reminder shows up.

Both will fix their bugs at some point and both have reasons to use them. For example, Thunderbird will probably get a larger user base in the end and will probably get more plugins (and the calendar will be built into it in the future), whereas (the Ubuntu default installation of) Evolution is well integrated with the GNOME desktop and is a full functioning e-mail/calendar program.
