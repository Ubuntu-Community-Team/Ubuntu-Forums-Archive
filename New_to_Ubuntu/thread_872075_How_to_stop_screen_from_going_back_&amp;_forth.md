### Post by Shazaam on 2008-07-27
Sounds like you have desktop effects running. Go to System>Preferences>Appearance>Visual Effects and see if it is set to "None".
If you want effects left on, right-click the workspace box and go to Preference>Workspaces and set it to 4.
To get more info go here.....
[http://ubuntuforums.org/showthread.php?t=809695](http://ubuntuforums.org/showthread.php?t=809695)

### Post by silkstone on 2008-07-27
By default the mouse wheel switches desktops if desktop effects are enabled. You can probably disable that in the Advanced Desktop Effects Settings, but unless you really want the eye candy you can turn off desktop effects altogether as suggested.
