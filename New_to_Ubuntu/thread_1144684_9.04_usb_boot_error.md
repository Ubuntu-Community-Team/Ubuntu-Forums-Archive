### Post by anjilslaire on 2009-05-12
The official 9.04 ISO works fine from usb. It all depends on the usb stick being written to, and the tool used to write it. 
I recommend using th built-in "Create a USB Startup disk" included with 8.10 and newer.

If you're on 8.04, you can install it via
```
 sudo apt-get install usb-creator
```
