### Post by drs305 on 2009-10-08
Are you sure the BIOS is now set up to boot the CD drive first? Do you hear the CD-ROM spin up during the initial part of the boot process?

If the BIOS is set to boot from the CD first, it sounds like it is not recognizing the disk before it times out and moves on to the next option (hard drive). 

Have you tried putting the Windows CD in another computer to see if it will boot on a different system?
