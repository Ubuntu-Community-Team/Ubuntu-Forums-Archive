### Post by e79 on 2010-09-14
>  ...I would like to change the item in the "Places" drop down menu to "Downloads" and point it back to my now only Downloads folder in /home/username.  Please help.Thanks in advance.Simply open your Home Folder (or any folder using Nautilus for that matter), and go to :

Bookmarks --> Edit Bookmarks

You should be able to Rename your folder to Downloads and change the path accordingly to your username (see Edit Bookmarks.png).

Hope this Helps
