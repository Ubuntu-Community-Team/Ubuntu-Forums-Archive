### Post by grahammechanical on 2011-04-09
Ubuntu does things differently. When you install and put in a user name and password. That person becomes both the administrator and normal user on the same account. You only become the administrator when you try to do something that needs administrator access and you happen to know the correct password.

Someone using the computer while you are away will not know the password and so cannot make changes to the system (such as changing their access level). But they can use it as a normal user. This is thought to be more secure than the method where you log in as administrator because all the time you are logged in as administrator anyone or any program (think virus or trojan) can access and change system settings.

With the Ubuntu way once you have completed the administrative task you stop being the administrator and become a normal user again. With Ubuntu you can also give someone a guess session. Or, set someone up as a separate user with their own home folder but you can limit their access to programs etc.
