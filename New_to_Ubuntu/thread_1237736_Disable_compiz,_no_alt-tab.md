I disabled any special desktop effects (Appearance > None) to speed up my computer, and now I've lost my ability to alt-tab between programs.

Is there a way I can get this back?

### Post by pteri498 on 2009-08-11
Ahh, as it turns out, this was already solved:

[http://ubuntuforums.org/showpost.php?p=4402298&postcount=6](http://ubuntuforums.org/showpost.php?p=4402298&postcount=6)

Just go to Preferences > Keyboard Shortcuts, scroll down a decent in the Window Management section amount until you see "Move Between windows, using a popup window". Set the key binding to alt-tab, and there you go.
