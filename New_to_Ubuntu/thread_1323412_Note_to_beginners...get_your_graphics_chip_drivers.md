### Post by austinb82 on 2009-11-11
I have been using computers for years in fact I work for an internet company however, I just recently installed Ubuntu which is the first time I have used Linux.

I have a very underpowered netbook because I was looking for something ultra-portable to compliment my desktop.  I was very disappointed with the performance of the system resource hog windows vista on it and it was extremely slow just running basic system tasks like opening folders.  I installed windows 7 on it which runs a little better but still relatively slow.  I decided to try installing ubuntu on a separate partition to see if it was any better.

At first I was very disappointed because it seemed to be running as slow as Vista and even windows 7 was running better than Ubuntu.  While looking for drivers to get my wifi working I decided to see if I could get drivers for my GMA 500 graphics chip as well because it wouldn't display in the screen in the native 1366x768 resolution.  I did end up finding a script that automatically installed the drivers and I am happy I did.  After installing the graphics drivers Ubuntu runs great now, before just navigating around the operating environment was slow and choppy, now everything loads up quick and snappy.

I even tried a couple open source games that I could not run before installing the drivers and just assumed my netbook was underpowered.  Now the games run flawlessly.

I just thought I would post this so any newbies to ubuntu like myself don't get discouraged after installing it thinking it is slower than it should be.  After I installed my GFX drivers it runs flawlessly even on a netbook and I am thinking I may start using ubuntu more than windows.
