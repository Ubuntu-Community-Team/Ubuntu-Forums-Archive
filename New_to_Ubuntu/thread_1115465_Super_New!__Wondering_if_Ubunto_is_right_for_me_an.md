> **Vick said:**
>  If I start trying to install it and I mess things up how hard will it be to get back to windows without having the original install discs?


You can use CloneZilla to make a disk clone image first.
CloneZilla has a little learning curve, but I like it for making disk-images and save them on an usb-disk. [http://www.clonezilla.org/](http://www.clonezilla.org/)

And you can go for Wubi in case you're a bit worried about messing up bootloaders and things like that.
[http://wubi-installer.org/](http://wubi-installer.org/)
