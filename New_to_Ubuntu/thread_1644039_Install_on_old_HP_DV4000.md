### Post by rockerphil on 2010-12-12
Linux is known for working best on older hardware, and i've had other Linux distros running on machines with much more modest specs than what you've described, namely an old Dell Dimension L500cx running a 500 MHz Intel Celeron processor with as little as 128 MB of PC100 SDRAM and experienced little to no lag when properly configured. so Ubuntu should run just fine on that laptop. about the only thing you might run in to problems with is the Wifi card, but i've personally successfully configured a couple of Broadcom wireless cards, so it's not impossible. i can't remember exactly how, but some Google searches should do the trick. hope this helps,

Phil

edit: you might try running in a Live CD for a while before actually installing to the hard drive to make sure that your various devices work, i.e. sound and video card and things of that nature

### Post by Chesamo on 2010-12-12
The only issue you're going to run into is that the video doesn't display properly on first boot. After you fix the resolution the first time, it should be fine.

I installed this on my friend's laptop and it ran fine, provided I was able to activate the Broadcom wireless card via a wired connection. (You'll need that.)
