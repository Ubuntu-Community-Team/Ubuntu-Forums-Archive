### Post by beew on 2011-11-20
Don't know, but if you want a fully functional and fast gui to manage your packages use synaptic instead of the software center, which is slow,bloated and lacking in features.
 
Synaptic used to be included but not anymore because they want people to use the Canonical developed junk that is the Software Center (looks good but not very useful, catering solely to the lowest common denominator)

```
sudo apt-get install synaptic
```Also install gdebi for left click install of .deb files install gdebi, it is stripped down and  a lot faster than the Software Center. I find it ridiculous that you have to fire up a big gui with tons of bells and whistles just to install a .deb file
```

sudo apt-get install gdebi
```
