### Post by mikechant on 2011-05-20
Looks like you're wasting your time trying to install V3 under wine, appears to be just about useless even if installed successfully.
If you really want this package working under Wine maybe you can get a discounted upgrade to a later version which may work acceptably.
Wine does a difficult job as well as can be expected but some windows programs are just too difficult to get working properly outside of windows.

An alternative is to try Ubuntu Studio which is stuffed full of the best of the free music making packages, but I won't pretend that they are necessarily as good as some commercial packages.
Some people are quite happy with this however.

Also note that if you do get any success with Wine + Reason, running them with Ubuntu studio should improve their performance due to the low-latency kernel.
