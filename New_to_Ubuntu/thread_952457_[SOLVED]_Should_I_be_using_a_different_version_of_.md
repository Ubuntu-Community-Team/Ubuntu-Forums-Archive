### Post by Paqman on 2008-10-19
When you first boot up the disk, one of the options should be to check it for errors. I'd say definitely give it a run through.

As for 64-bit vs 32-bit. You will get a slight speed increase with 64-bit. This comes at the small price of a marginally bigger size on the hard drive. I'd recommend using 64-bit on any chip which is capable of it though.
