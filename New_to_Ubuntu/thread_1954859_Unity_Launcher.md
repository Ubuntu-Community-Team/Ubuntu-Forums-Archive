### Post by dwhite on 2012-04-08
> **acimi66 said:**
> Is the launcher simply like a desktop short cut?

Any thoughts


they work as shortcuts but also provide much more information and more functionality;

[LIST]
[*]only one click to launch program, or open folder (as opposed to two clicks typically for desktop short cut)
[*]in the unity launcher there are indicators telling you how many active windows are associated with a particular program (# of arrows on left hand side of program icon)
[*]also if the arrows on the left hand side of icon are not filled then that indicates the program is open on another workspace
[*]the arrow that shows on the right side of the program icon indicates that the currently active window is associated with that program
[*]if you click on an icon for an open program you will navigate to that program (if for example that program is on another workspace you will go to that workspace)
[*]if two or more windows are open in a particular program if you click on the left hand arrows you will be shown mini versions of all the open windows for that program and can then choose which you would like
[/LIST]
 
for example; the attached screenshot indicates that three programs are open
rhythbox, libreoffice writer, and firefox

libreoffice is on the current workspace and the others are on other workspaces

and libreoffice has the currently active window

if i wished to navigate to the open firefox window i would click on the icon and i would be whisked away to the other workspace
