If you're setting up a network that only involves friends and family you can make a [COLOR="DarkRed"]**free**[/COLOR] private network using ning.com. It's very easy. I recently made a network there for Linux users that you can reference. [Tux Space]("http://tuxspace.ning.com")

With ning you can very easily upload your own videos and pictures. If you only want something as simple as private file sharing you can upload files to mediafire.com and secure them with a password, that of course, only people you shared it with would know.
