### Post by Kellemora on 2009-02-17
Hi LD

Something else to keep in mind also!

A blank space between two words in GNU/Linux OFTEN means TO

EG:  copy dog cat

Will copy the file named dog to a file named cat.

The trailing / makes it a directory or folder.

So if you would type Practice Folder/ might have generated a new folder named Folder with a file named Practice inside, if Practice existed.
