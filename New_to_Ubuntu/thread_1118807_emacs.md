### Post by unutbu on 2009-04-07
emacs is not installed by default, but you can get it by installing a package from the official repository.

If you are using Hardy or Intrepid, I'd recommend installing the 
"emacs-snapshot" package.

If you are using a pre-Hardy version, I'd recommend using these instructions: [http://peadrop.com/blog/2007/09/17/pretty-emacs-reloaded/](http://peadrop.com/blog/2007/09/17/pretty-emacs-reloaded/) to install emacs-snapshot.

The difference between emacs-snapshot and the "normal" emacs package is that the emacs-snapshot version can display beautiful TrueType fonts.

### Post by duanedesign on 2009-04-07
The easiest way to see what is or is not installed is the Synaptic Package Manager. 
At the top of your screen click on the System menu. 
Go down to Administration. 
Navigate the list that comes up and go down to Synaptic Package Manager. 
If prompted for your password enter it. 
When the Package Manager comes up you will see a Quick Search at the top. 
Type in emacs (or whatever package you are looking for)
If the package is installed the box to the left of the package name will be green. To install a package click on the box and select "mark for installation"
After you have selected the package you want to install/uninstall select Apply in the menu at the top.
