### Post by QLee on 2009-11-26
I certainly agree with the other posters, you don't need a BIOS update if it doesn't fix anything with your system.

The place where you found the BIOS update would also have some document to explain what the update "fixed", it should be possible from that to see if any of it applied to your system. (or to anything that you might attach to your system in the future)

Similar to the way one one would read the release notes of a new version of Ubuntu before upgrading to it, one should read the BIOS notes before risking an update. It is possible to destroy the ability of your system to boot while flashing the BIOS.
