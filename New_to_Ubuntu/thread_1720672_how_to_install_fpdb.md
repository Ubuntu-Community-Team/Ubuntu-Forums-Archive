### Post by rosencrantz on 2011-04-03
If you don't insist on bleeding edge versions, go for the deb: python-fpdb_0.22-1_all.deb, found here: [http://sourceforge.net/projects/fpdb/files/fpdb/](http://sourceforge.net/projects/fpdb/files/fpdb/)
install with sudo dpkg -i python-fpdb_0.22-1_all.deb or by just clicking on the file after downloading. Dpkg should handle all your dependency issues.
