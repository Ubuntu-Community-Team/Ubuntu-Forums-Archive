When editing files that don't allow you to save them, make sure you do this:

```
gksudo gedit /path/to/file.extension
```

In terminal. In order to copy files/folders between folders that need root access:

```
gksudo nautilus
```

In terminal, then open another terminal window and run the same command so you have two root nautilus windows running. DO NOT close the terminal windows until you're done, otherwise the windows you opened as root will close too.

### Post by Favux on 2009-09-04
Hi Gnovak99,

Like coldReactive says to edit xorg.conf you need to enter in a terminal:
```
gksudo gedit /etc/X11/xorg.conf
```
Both sudo and gksudo make you root/super user.  You use gksudo with graphical programs like gedit (gnome editor/Text Editor).
