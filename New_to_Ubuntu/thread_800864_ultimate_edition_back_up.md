### Post by niteshifter on 2008-05-20
Hi,

Yes, it's possible. It's encouraged even :)

What you want is to make what's called an "image backup". What I use to do this is a Live CD: [SystemRescueCD]("http://www.sysresccd.org/Main_Page"). This has a program "partimage" that will make image backups. It also has other tools that are quite handy to have when the system breaks.

What you should do is spend some time with the above site and get familar with using partimage (there's a link to partimage's site at the above URL). Figuring it out as you go at the first time is not recommended. Another thing that should be done is test your backup by verifying it immediately after making it, while still "in" the SystemRescueCD.
