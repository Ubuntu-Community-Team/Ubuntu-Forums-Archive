### Post by DayLite on 2010-09-25
I was reading the Ubuntu Manual. You can [download it]("http://ubuntu-manual.org/") . The answer is found on page 103.
 >  **Changing printer options**

They can be changed by right clicking a printer and choosing **Properties.**
This specifies how much ink is used when printing. **Fast Draft** using the least ink and **High-Resolution Photo** using the most ink
