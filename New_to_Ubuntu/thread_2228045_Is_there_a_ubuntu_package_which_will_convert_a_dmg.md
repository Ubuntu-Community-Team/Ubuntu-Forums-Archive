### Post by monkeybrain20122 on 2014-06-05
What do you want to do with it? You can mount an .img with the command line in Ubuntu without conversion

If you don't need sudo (say to install program from the .img) you can use a gui tool to mount it
[https://apps.ubuntu.com/cat/applications/acetoneiso/](https://apps.ubuntu.com/cat/applications/acetoneiso/)
or
[https://apps.ubuntu.com/cat/applications/furiusisomount/](https://apps.ubuntu.com/cat/applications/furiusisomount/)

I think acetoneiso also converts .dmg directly to .iso.
