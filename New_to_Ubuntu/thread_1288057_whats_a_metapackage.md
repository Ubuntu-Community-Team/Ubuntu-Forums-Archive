### Post by Bachstelze on 2009-10-10
A metapackage is a package that doesn't contain anything by itself, but is used to install a collection of other packages using dpkg dependencies.

GTK 2.0 is a widget toolkit, a piece of software responsible of drawing graphical elements ("widgets") of an aplication, like buttons, checkboxes, etc.. By changing the theme it uses, you change the way widgets look.

Metacity is a window manager, it is responsible for drawing and managing windows. By changng the theme it uses, you change the way windows look (mostly their title bar and borders).
