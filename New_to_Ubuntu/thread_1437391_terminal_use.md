### Post by admiralspark on 2010-03-23

### Post by 3rdalbum on 2010-03-23
The included Empathy instant messenging program does support Yahoo and AIM.

If you really want Pidgin, you don't have to go to the terminal. The terminal command that you've been given ('sudo apt-get install pidgin') is actually the equivilant command for something you can do in the GUI.

Go to Applications > Ubuntu Software Center and look under the "Internet" category; Pidgin is there, and you can install it with just some mouse clicks.
