### Post by dodle on 2011-02-28
If you are looking for a map editor there is Tiled: [http://www.mapeditor.org/](http://www.mapeditor.org/). I'm pretty sure it is in the Ubuntu repository. 

```
sudo apt-get install tiled
```

If it's not it has an Ubuntu PPA repository available.

If you want to create the graphics yourself, I would probably just use the GIMP. You can also try Tile Studio, I think it only works on Win32 so you will need WINE: [http://sourceforge.net/projects/tilestudio/](http://sourceforge.net/projects/tilestudio/).
