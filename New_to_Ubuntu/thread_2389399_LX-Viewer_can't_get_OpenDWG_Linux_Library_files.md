### Post by Holger_Gehrke on 2018-04-16
You're talking about a program that hasn't had an update since 2005, according to sourceforge. Even if you could get those libraries, you'd have to basically rewrite the program to work on a modern system. opendwg.org is a parked domain. From what I can see in the snapshots of that site on archive org, those libraries where only available to members of the Open DWG alliance under a non-free license. The current site of the group that created those libraries is opendesign.com. They offer a closed source freeware viewer and commercial libraries for working with AutoCAD-files.
