### Post by 3rdalbum on 2010-05-10
> **TriBlox6432 said:**
> but for how snappy 10.04 is compared to XP, I would expect it to have less SysRQ.

That's very logical, but it's not correct. Windows XP is virtually hardcoded to run best on computers from its era; it doesn't really make good use of resources over and above its "recommended" system requirements. If you start with 256 MiB of RAM and upgrade it to 2 GiB, you probably won't see a speed improvement on Windows because Microsoft's engineers never imagined that any desktop user would have that much memory.

Ubuntu has more modern assumptions about your hardware, and the Linux kernel itself scales up better than the old Windows kernel.
