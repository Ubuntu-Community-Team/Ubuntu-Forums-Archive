### Post by halitech on 2009-01-05
with only 256 meg of ram, you won't be able to use the live cd to install as it requires 384 to run properly. I've run both Ubuntu and Xubuntu on the same machine and to be honest, Xubuntu does not run that much better because it uses alot of gnome apps to get things working. If you want something lighter, I would go with Debian with XFCE if you want a lighter distro. 

[http://forums.debian.net/viewtopic.php?t=26566](http://forums.debian.net/viewtopic.php?t=26566)

Also, installing Open Office on a lightweight distro is counter productive, you would be better going with lighter alternatives like Gnumeric and AbiWord.
