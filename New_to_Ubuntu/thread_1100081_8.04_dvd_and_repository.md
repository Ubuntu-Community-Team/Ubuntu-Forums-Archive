### Post by cariboo on 2009-03-18
The package you are looking for is apt-cacher. 

> Apt-cacher performs caching of .deb and source packages which have been
downloaded by local users. It is most useful for local area networks with slow
internet uplink.

Apt-cacher is in the repositories, use Synaptic Package Manager to install it.
