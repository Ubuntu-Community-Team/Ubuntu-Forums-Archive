### Post by ActionParsnip on 2022-01-24
Ubuntu isn't a program like a web browser or media player. It is an operating system so no it doesn't work with Windows 11. It doesn't work with Windows because it is an operating system. You can dual boot Ubuntu on the same system and choose the operating system that you want to boot using GRUB. After that either Windows will load or Ubuntu will load.

### Post by nginmu on 2022-01-27
I have Lubuntu 21.10 on one HDD partition, and Windows 11 on another as a dual boot setup. It was easy to set up using a USB-stick-based Lubuntu installer at a time when the laptop (HP Elitebook 850 G3) only had Windows 11 installed on it.

The Lubuntu installer automatically set up the booting system so you get a simple choice when you turn on the machine: Do you want to use Windows or Linux?

I use a USB FAT32 external hard drive to ease the sharing of files between the two different operating systems. Both operating systems can also use an NTFS-formatted disk.

On my system the two happily coexist, I find one is better for some things, and the other better at others.

These days I definitely spend a lot more time using Linux than Windows though I think.

Don't worry about getting stuck, the help available in here is amazing.
