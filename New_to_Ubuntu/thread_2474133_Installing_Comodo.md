FYI: Not really needed.

On Linux, when antivirus software is installed, it is usually looking for window viruses in files that are passing through the system (like email and file servers).  It helps to  protect our Windows using friends from files that we process for them.
Don't log in as root.  Back up your user data.   Don't click on links in emails you don't know.
Enjoy :)
If your worried about security linux has some useful tools found Here: [https://wiki.archlinux.org/title/List_of_applications/Security#Threat_and_vulnerability_detection](https://wiki.archlinux.org/title/List_of_applications/Security#Threat_and_vulnerability_detection)
