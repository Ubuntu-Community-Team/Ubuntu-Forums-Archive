### Post by faraco2 on 2017-12-07
Hi, I been using Atom editor and Visual Studio Code to program my web applications. They are both cross platform, has vast web centric plugins and free (in price) also open sourced.

Visual Studio Code:
[https://code.visualstudio.com/](https://code.visualstudio.com/)

Atom editor:
[https://atom.io/](https://atom.io/)

PS: Do note, both of these editors do use a significant amount of RAM (typically similar to other major web browsers memory usage). If you want a lightweight alternative, consider Geany or Bluefish or you can give Sublime Text 3 (not free) a try.
