### Post by Paqman on 2008-10-03
Add/Remove is supposed to only contain GUI apps that will install themselves nicely and create a launcher in your menus. So you won't find all the terminal apps in there. So it's just a more user-friendly front end for the package management system than Synaptic is.

I've found myself using it more and more lately, since it's a bit snappier than Synaptic.
