You can still install Ubuntu and still have XP.  The newest version of Ubuntu will provide you an option to install Ubuntu in Windows which allows you to use Ubuntu without having to partition your hard drive and easier to clean up if you don't like it.  It does lose some performance though but it is a great way to try and see if you like it or not.

Another option is to do what most people do--dual boot.  This would require you to shrink your XP partition and create a new partition for Ubuntu.  

To start off, I would try out the Live CD first.  It will tell you what will work out of the box.  It will allow you to play around with Ubuntu without having to install.  If you think you like it, you can then try the option to install in Windows.  Then if you find it wonderful, dual-boot.  Once you are done with Windows, wipe it out!!!

The hardware that you listed looks fine.  Hope you enjoy the experience!

### Post by kk0sse54 on 2008-05-16
Wubi brings about mixed feelings with most people. Some people use it with absolutely no problems then there are others that have experienced a lot of problems. I suggest trying Ubuntu out through VMware for a few weeks as a virtual desktop (note that any OS running through a virtual machine will run slower and not all of the functions available for Ubuntu will be able to run such as 3D effects) and then setting up your computer to dual boot so that you can keep your windows OS and create a separate partition on your hard drive for Ubuntu which when using GRUB will allow you to pick which OS you want to run on the start up of your machine. As far as your specs go they should work but just to make sure burn the Ubuntu iso to a cd create a liveCD and boot from that in order to test out Ubuntu on your computer with all of it's functions enabled without making any changes to your computer.
