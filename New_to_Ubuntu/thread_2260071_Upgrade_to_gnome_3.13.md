### Post by buzzingrobot on 2015-01-09
If you upgrade, you want to go to 3.14.  The Gnome project uses odd-numbered releases, like 3.13., to denote development releases not intended for users, and not generally made available in the same channels. 

If you are running Ubuntu Gnome, a quick search here or via Google will turn up info on upgrading.
