### Post by balaknair on 2008-12-17
I've got a Creative WebCam Vista, works out of the box with Intrepid(Ubuntu 8.10) 64 bit, no issues so far with Ekiga and Skype. Current price I think is about £6-8.
The Creative Live Cam Vista(£5.39 on Amazon) also works, I have a friend using it on 
Ubuntu 8.10 32-bit, but I'm not sure if it worked out of the box.

I suggest you read through the following before buying a new webcam


For a list of webcams(various manufacturers) and support in Linux
[http://linux-uvc.berlios.de/](http://linux-uvc.berlios.de/)

[http://www.ubuntuhcl.org/browse/search+webcams?category=33](http://www.ubuntuhcl.org/browse/search+webcams?category=33)

[https://wiki.ubuntu.com/HardwareSupportComponentsMultimediaWebCameras](https://wiki.ubuntu.com/HardwareSupportComponentsMultimediaWebCameras)


For the Logitech Quickcam on Ubuntu 8.10,
[https://answers.launchpad.net/ubuntu/+question/49739](https://answers.launchpad.net/ubuntu/+question/49739)

For an easy way to install webcam drivers
[https://help.ubuntu.com/community/EasyCam](https://help.ubuntu.com/community/EasyCam)
