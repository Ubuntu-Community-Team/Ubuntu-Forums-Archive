### Post by Roger Allott on 2008-12-18
Today I got the recent security update for Firefox, as I suppose everyone else has or will do shortly.

I'm not sure whether I've ever looked at the 'Ubuntu-it' drop-down options before, but I just did so and was mildly amused to find that many of the second level options there are written in what looks to me like Italian.

So, when I click 'Ubuntu-it', I get a list of:
Ubuntu-it
Homepages
Forum
etc.

all nicely in English.

When I click on 'Forum', I get a list of:
Forum - Homepage
Search
Forum Menu
User Menu
Available Forums
etc.

all nicely in English.

When I click on 'Available Forums', I get a list of:
Area di benvenuto
Notizie
Ubuntu-it
Supporto Tecnico
etc.

all rather oddly in Italian!!

All links beyond that level are also in Italian.

I've never knowingly installed anything in the Italian language onto my PC, so where has this Italian stuff come from?

I might be way off on this, but might there be a problem with the Ubuntu-it plug-in where the Update procedure sees the '-it' suffix as indicative of being an Italian language installation?

Has anyone else encountered this problem yet?
