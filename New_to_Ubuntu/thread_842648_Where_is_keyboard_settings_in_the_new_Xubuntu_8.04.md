### Post by whoscheesemine on 2008-06-27
Normally, when I install Xubuntu 7.10 on a computer I always like to go into keyboard settings and set the tilda button to run the command:

```
xfce4-terminal
```

that way I always have a quick way to bring up the terminal to run commands as its oh so much faster to do things this way, and my computer doesn't run on mouse clicks (although it would be nifty if someone came up with a program that measured how much "energy" could have been created from mouse clicks as a laugh) However, there is no keyboard settings..... nevermind, while typing this I answered my own question, I figure I might as well continue though, just in case someone else is having the same problem. If you click applications -> Settings -> Settings Manager and then click on keyboard. I would put this as solved, however, another question comes to mind... what would be the terminal command for keyboard preferences?
