### Post by jacksinn on 2009-05-07
First step is to install a plugin that allows you to display flash content. You can either get the proprietary package from the Adobe site or from the Synaptic Package manager. There are also foss plugins available. Fire up your package manager and search fro 'flash'. If for some reason FireFox isn't working, don't forget you can use Epiphany too.
