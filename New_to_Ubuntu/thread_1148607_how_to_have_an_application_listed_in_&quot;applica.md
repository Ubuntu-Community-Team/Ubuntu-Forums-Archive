### Post by decoherence on 2009-05-04
right-click on the desktop and select "Create Launcher..."

Give it a name like "Flyback"

For the command put

```
python /home/andrea/flyback/flyback.py
```

And put in any comments you like. I don't know Flyback... if it isn't a GUI program, you might have to change "Type" from "Application" to "Application in Terminal"
