### Post by tarps87 on 2010-07-16
That exact clock is part of KDE which is the default desktop environment of Kubuntu.
I believe there is a similar screenlet clock, part of the screenlet package, this is assuming you are using gnome not KDE.

To install:
```
sudo apt-get install screenlets
```

It will appear in on of the menus somewhere (I don't remember which one, it's easy enough to find)
