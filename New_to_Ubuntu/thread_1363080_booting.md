### Post by unmole on 2009-12-25
First verify the integrity of the image file.
Download [this]("http://www.pc-tools.net/win32/md5sums/") for Windows and see if the output matches these values

fb3d93499e71104f40a8388397d796a3 *ubuntustudio-9.10-alternate-amd64.iso
89a9b8f2f3911fe74990a417e23ee27c *ubuntustudio-9.10-alternate-i386.iso

If the Hash vales are correct, instead of wasting another DVD (or CD) you can make a bootable pendrive with [this]("http://unetbootin.sourceforge.net/")
