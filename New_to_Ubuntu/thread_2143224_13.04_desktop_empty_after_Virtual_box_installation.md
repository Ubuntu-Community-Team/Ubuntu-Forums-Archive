### Post by ibjsb4 on 2013-05-08
There are several ways to update.  In this case, updating in terminal will also give you an error report if anything is wrong. So ..

[Open a terminal]("https://help.ubuntu.com/community/UsingTheTerminal#Starting_a_Terminal") and enter:

```
sudo apt-get update
```

If that works without error, enter:

```
sudo apt-get upgrade
```

[https://help.ubuntu.com/community/AptGet/Howto#Maintenance_commands](https://help.ubuntu.com/community/AptGet/Howto#Maintenance_commands)

After you get your desktop back in working order we can get vBox working :)
