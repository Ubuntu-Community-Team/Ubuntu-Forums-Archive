### Post by dp1 on 2011-01-03
Apparently there is no 'good' solution for the problem.

The possible solutions are:
- choosing a lower screen resolution, but ubuntu won't look smooth anymore
- increasing the font size in firefox, but the layout of the webpages will change
- zooming in, but all images will become blurry

I have done the latter as the least 'bad' solution and I use NoSquint to keep the zoom levels constant. NoSquint adjusted automatically the right zoom level for me.
