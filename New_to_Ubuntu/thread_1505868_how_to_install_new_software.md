### Post by Mark Phelps on 2010-06-09
> **juntjoo said:**
> i'm pretty new to linux ...

Don't take this as a criticism, but being new, one of the LAST things you should be messing with installing stuff you download from the Internet!

In most cases, you're going to get some source code -- which will then require you to download lots of packages to build what is essentially an application development environment on your PC.  Then, you'll have to struggle with config files, make files, and other scripts.

Which is an awful LOT to take on when new to Linux.

A much easier approach is to use the Software Center to install applications.  Of, if you prefer, use Synaptic.  In those cases, nearly all the work is already done for you.
