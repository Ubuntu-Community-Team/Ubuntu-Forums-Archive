If you still have Firefox installed, install  [Flash-Aid](http://www.webgapps.org/add-ons/flash-aid) and run the extension wizard. It will detect installed flash plugins, remove them and install the best option according to your system architecture and version. Additionally, it will apply some tweaks that should improve performance. 

Chromium will detect plugins installed by Flash-Aid.
