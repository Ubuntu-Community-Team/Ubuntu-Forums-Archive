### Post by drs305 on 2009-02-21
> **DarinB said:**
> thanks its a snap now i have to wait for the sender to see if it works 

is it automatic or do i have to use tools run filters on folders????????

If set up properly it is automatic. You can manually run the filter after you set it up to see if it works. To test it, you could get an old email from the sender, put it in your Inbox, then manually run the filter to see if it puts it where you want it to. When a new email arrives, it will do it automatically.
