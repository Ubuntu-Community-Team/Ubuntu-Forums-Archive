### Post by uRock on 2011-01-31
You'll have to create one / partition which will contain the OS and your data and one Swap partition which will be used in the same manner as the Windows swap file.

The Swap partition needs to be 1.5 times the amount of RAM your system has.

The screenshots should be helpful with seeing what to do when creating the partitions.
