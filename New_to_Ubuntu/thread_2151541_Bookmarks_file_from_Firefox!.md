### Post by MidnightGrey on 2013-06-04
Firefox bookmarks are stored here:
$HOME/.mozilla/firefox/<8digitcode>.default/bookmarkbackups
to see hidden folders in nautilus press "Ctrl-H"

alternatively you can backup bookmarks via the cloud in Firefox > Edit > Preferences > Sync.
