### Post by speedwell68 on 2011-05-01
My suggestion would be to forget the BBC's iPlayer website and try get-iplayer and the gIplayer interface.

First install get-iplayer in the terminal...

```
sudo apt-get install get-iplayer
```

Then install the GUI interface by downloading this .deb file from here...

[http://sourceforge.net/projects/giplayer/files/](http://sourceforge.net/projects/giplayer/files/)

With this installed, not only can you download the programme you want to watch, you get to keep it forever too.
