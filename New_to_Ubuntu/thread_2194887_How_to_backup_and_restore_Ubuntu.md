### Post by grahammechanical on 2013-12-21
Will this machine boot from a USB stick with a Ubuntu Live Session? If the live session was created with persistence it should be possible to use a USB Ubuntu live session to access the data and copy it onto the USB memory stick. It all depends on how much data there is and how much free space there is on the memory stick.

It may also be possible to create another partition and format it as Ext4 and then using the version of Ubuntu that you can boot into or a live session copy the data onto the new partition. I have done that myself.
