You can just launch from there if you want.

+1

Either press super + A to navigate directly to the applications section of the dash.

Or, simply right-click on the dash icon and select the application quicklist link.

The application menu in the dash has three sections. Recently used, installed and downloadable.

In the applications menu in the dash, the installed applications are listed alphabetically. 

Also when in the dash, on the right side is a filter option, allowing you to modify which app-types you want to browse through.
