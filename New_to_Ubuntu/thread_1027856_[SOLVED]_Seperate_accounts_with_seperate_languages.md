-When I log in I want it go be in English
-When my dad logs in I want it to be in Spanish

### Post by mkvnmtr on 2009-01-01
I use both those languages on my single account.My main language is english and my keyboard is in english. You can install an applet to the panel to switch languages. there are also many support packages for both in the package manager. Search under language and dictionary. But for your question. I believe when you get to the log on page you can set the language. If you set it one time on his account I think every time he logs on it will be the same. I am going to check that now. You might also want to look for fonts. Everything you need might not be installed by default on an english install.

Yes indeed that is the way it works. Sit the language once and then it boots up in last language. Be sure to check the fonts and get the keyboard to switch. I believe there is also a package that will make the terminal multi lingual.
