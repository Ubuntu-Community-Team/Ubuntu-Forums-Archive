### Post by mlentink on 2012-05-25
Welcome to the free world!

My first instinct when I want a printer is to click on the dash (the ubuntu logo on top of the launcher bar to the left) and type printer and then click on the printer management icon.
Then click on 'Add' and search for your printer. Is it connectd locally or is it a network printer?

### Post by Brian_new_to_linux on 2012-05-25
Generally Ubuntu 12.04 will find most print drivers as suggested, but if it doesn't work you can download the drivers here:
[http://download.ebz.epson.net/dsc/search/01/search/searchModuleFromResult](http://download.ebz.epson.net/dsc/search/01/search/searchModuleFromResult)

PS: make sure you download and install all the scan driver files and install the data file first -the scanner will not work at all if you don't!
