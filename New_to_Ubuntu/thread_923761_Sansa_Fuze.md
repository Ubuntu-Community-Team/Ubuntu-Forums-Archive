### Post by rain4ever on 2008-09-18
ok i looked around and some were saying to switch usb modes. The problem was that it was in msc mode and it couldnt find the files. Simply switching to mtp fixed it. There were a couple others on here that had the same problem. So if your files cannot be read from a mass storage device switch the usb mode to mtp
