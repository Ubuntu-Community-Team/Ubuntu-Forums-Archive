### Post by Bucky Ball on 2014-07-17
Welcome. Personally, I'd tackle why you can't get your dual-boot working rather than go wasting money on a new machine. Post a thread in Installations & Upgrades about what is going wrong with as much detail as possible and a descriptive thread title. Might work out a lot cheaper.

Any member of the Ubuntu family will run Armory fine, or at least should install it, as there are Ubuntu/Debian packages ready to go on their site (they install with a couple of clicks).

> I have made several failed attempts to run Ubuntu alongside windows on one of my machines. So, I thought it might just be better to buy a refurbished machine off of Ebay ... like a Lenovo Thinkpad without an operating system at all. Then just run Ubuntu off a flash drive.

* Ask for help with your dual-boot;
* Buying a new machine is overkill without attempting to fix the dual-boot problem (we are here to help and how can you expect to know how to fix it when you just got here?);
* Running off a flash drive is slower and definitely not the ideal way to go about this.

To me, your plan of action is expensive and over-complicating things (and in the end not as efficient as a hard-drive install). If you ask for help with installing your problems might be fixed in a jiffy.

* PS: How do you know all the software works? Armory probably will, but if you are intending on using any Windows software, you will be jumping through hoops to get that to work, if you ever can, because it won't work natively on Linux.
