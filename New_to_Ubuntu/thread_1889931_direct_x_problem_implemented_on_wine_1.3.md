### Post by Lisiano on 2011-12-02
Please tell us the game name. Usually you can find a way to make a game work by going to the [Wine AppDB]("http://appdb.winehq.org/"). For example [EVE Online]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=24887"). As you can see there are instructions there on how to make the game work.

PS. All Windows libraries, including DX are not recommended by the Wine Wiki. If you use them, you are not allowed to send error reports against certain features but you are certainly allowed to use them.
PSS. You can just install a few certain DX9 libraries instead of the whole DX9 suite (General overkill)
