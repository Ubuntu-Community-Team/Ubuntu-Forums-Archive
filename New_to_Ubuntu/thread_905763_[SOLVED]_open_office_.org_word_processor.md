### Post by kansasnoob on 2008-08-30
Glabel is good, also I don't bother with Open Office. I use the gnome office suite which includes abiword.

You needn't remove Open Office to try it (in fact I recommend not doing so due to messing with other dependencies) just go to synaptic and install either gnome-office, or if you only want a simpler word program install abiword-gnome if you're using Ubuntu. If you're using Kubuntu or Xubuntu you should just use plain old abiword.
