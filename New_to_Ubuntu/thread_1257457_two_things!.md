### Post by aeiah on 2009-09-04
for the freezing, open a terminal and run dmesg once your computer is back up after a reboot and see if you can find any odd messages.

menu > applications > accessories > terminal

type:
```

dmesg

```

it'll be a long list of logs but near the bottom (but before the boot up messages obviously) there should be some error reporting

as for the beats: check out ubuntu studio. its an ubuntu distro for multimedia production and anything included in there is available for you to install in the normal ubuntu too. the site should give you a description of various audio programs.
