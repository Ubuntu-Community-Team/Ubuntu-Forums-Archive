### Post by Frogs Hair on 2013-10-11
With an upgrade you won't loose data , but you would still want backup in case of an upgrade problem. Non official applications from software center are often updated for new Ubuntu versions and when that occurs depends on the software developer. Some applications are not immediately compatible with new Ubuntu versions. If you are using any ppa software you will have to disable the software sources before upgrading or you will most likely end up with a broken package system.  

If you preform a clean installation you will start with a clean slate and unless you have your home folder on its own partition you will loose your data if not backed up. You will have to install your preferred applications on the new installation. It is actually faster to backup your data and perform a clean installation than upgrade.
