### Post by wvonstockhausen on 2013-06-27
Having noticed that this thread was closed way back in 2008 I realized Ubuntu and Audacity have advanced to the latest version, because I could not find any good solution, even in Windows environment. So I had to figure it out myself and like to share it in this community. I assume you know how to install Audacity in Ubuntu.
Start having an internet radio station on so you have continous audio coming in. Open Audacity. In the drop down menu window right from the Mic icon select "systemdefault:mix:0" note do not select "default:mix:0". Be sure the next window right is on "2(stereo)..." Mouse over the RL level meters. Click on the right (input) and you should see the level readings. Adjust for correct levels by mousing over the " input volume slider " on the right and adjust the slider. Go to your selected Youtube video, start recording on Audacity and start Youtube. Minimize both windows so you can almost can start simultaneously.
