### Post by ian49 on 2014-08-07
Having been using 14.04 for a month or two when my mouse suddenly stopped working.

I was using a cordless mouse connected to a USB port so tried changing the port but that made no difference. Have since tried using a corded PS/2 mouse but still didn't work. I've since tried booting onto 14.04 on a memory stick as well as booting to a Live CD version of 10.10 but still the mouse doesn't work. So, am now baffled. Does this suggest it's a hardware failure of the motherboard of some kind?

Update: problem solved. Turns out my cordless mouse has become faulty and for whatever reason none of the versions of Ubuntu recognised the PS/2 mouse. I finally tried a third, corded USB mouse and that restored mouse functionality.
