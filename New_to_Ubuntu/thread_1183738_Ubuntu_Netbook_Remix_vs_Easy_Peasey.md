### Post by pauna on 2009-06-11
> **listerdl said:**
> Hi can someone kindly confirm that Ubuntu Netbook Remix is better than the Easey Peasey?


I have never used Easy Peasy on my 901, I have UNR (with classic GUI). I value the fact that I'm using a standard big-name distro with regular update and upgrade paths. With these smaller specialized distros you never know how long they are supported and how fast problems are fixed. I used to have a specialized EEE kernel in my 901 and now I'm very glad the stock kernel works almost for all hardware capabilities.

I haven't found Firefox to be extremely slow in my 901 + UNR, but I don't use it for heavy lifting.
