### Post by anewguy on 2009-10-31
+1 and to expand on XCan comments:

There are several shells available in *nix/*nux, each with different command structures, etc..  The scripting abilities of these shells is very powerful, much more so than the old DOS/Windows command line.  Besides learning the commands available in the terminal window/command line, also try to learn as much as possible on the scripting available for the shell you are using for the terminal window/command line - even if you just read about it - and you'll see how powerful it can be.
