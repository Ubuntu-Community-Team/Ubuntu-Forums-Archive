### Post by Spice Weasel on 2010-08-14
You can play AoE in WINE fine. Just hit this in to the terminal, wait for it to install and then you can run most Windows programs:

```
sudo apt-get install wine
```It's against the rules to talk about piracy here, but you can find the older AoE games from retailers and online shops very, very cheaply. :)

All you would need to do is put in the CD, and it would automatically open in the file manager if you are using GNOME. Right click on the .exe, and select run with WINE.
