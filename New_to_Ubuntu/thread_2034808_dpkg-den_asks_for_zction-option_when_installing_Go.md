### Post by Curtis6767 on 2012-07-29
You don't say which version of Ubuntu you're using, but I have found the easiest way to install Google Earth on 11.10 & 12.04 is to first install Gdebi from the software center. With Gdebi installed, find the .deb file for Google Earth wherever you downloaded it to, right click the Google Earth file and select Gdebi from the drop down window. Let Gdebi do the rest.

I don't know if this is going to solve your problem, but if you still have the Google Earth .deb file, you could try this and see what happens.
