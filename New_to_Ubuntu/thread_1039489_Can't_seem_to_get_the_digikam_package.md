### Post by gn2 on 2009-01-14
digiKam is in the "Universe" repository. ([explanation here]("http://www.ubuntu.com/community/ubuntustory/components"))

Here's [how to enable Universe in Kubuntu 6.06]("https://help.ubuntu.com/6.06/kubuntu/desktopguide/C/extra-repositories.html").

After you enable Universe, Adept should be able to find, download and install digiKam, which is an excellent application. ;)
