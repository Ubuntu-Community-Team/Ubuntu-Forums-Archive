### Post by jakupl on 2009-03-24
Are you dual booting or is ubuntu currently the only operating system?
If you are dual booting, than I would download and burn the [gparted live cd]("http://gparted.sourceforge.net/livecd.php"), also download and burn [supergrub]("http://www.supergrubdisk.org/index.php?pid=5").
Use the gparted live cd to delete the ubuntu and swap partition, then use the supergrub cd to restore the windows boot.
