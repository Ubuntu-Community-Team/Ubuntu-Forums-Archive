### Post by glennric on 2008-08-09
Do you have your mic working so that you can record from it?  If so then just open volume control and unmute the mic.  If that isn't loud enough check the mic boost under the switches tab.  If these options don't show up edit the preferences by clicking on the Edit menu and selecting Preferences so that they do.
