### Post by grazed on 2008-04-27
I would certainly say go for it. You get to use a superior file system (ext3) and have a much faster experience due to ubuntu having it's own partition and not running off of large file contained in the windows partition.

Most likely you'll see around a 30% speed increase, at least I did. :roll:

good luck!

### Post by grazed on 2008-04-27
yup, uninstall wubi.

one thing to note that i forgot to mention before. While going through the install process, you'll get prompted with a partition manager asking where you want to install to.

make sure to make a 2gb-ish partition, and assign it to be used as swap space. might sound confusing... but once you're in the installation it's all presented in a very self explanitory way.

good luck^
