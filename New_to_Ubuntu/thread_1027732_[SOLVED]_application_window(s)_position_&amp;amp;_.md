Myabe this is a minor issue but I wonder if there is a way that any open window at re-opening will go back to the former size and position.

### Post by nhasian on 2009-01-01
you can use the *Place Windows* option under *Windows Management* in the Compiz Config Settings Manager.  if you dont already have the ccsm installed you can install it with:

```
sudo apt-get install compizconfig-settings-manager
```
