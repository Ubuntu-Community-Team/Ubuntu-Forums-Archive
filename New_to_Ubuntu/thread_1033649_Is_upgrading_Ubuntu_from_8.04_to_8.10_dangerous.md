### Post by Guille Damke on 2009-01-07
Hi,

I have not tried upgrading, I prefer a clear install. I have my home directory in a separate partition, so I do the install and format the '/' directory, but no the /home (choose the "manual" option, and set the partition, mounting point and format or not). Also, right before restarting to boot from CD and install, I move all my $HOME directory to a different one to avoid problems when the new install creates my new home with new configuration files. Then I just move my documents, music, etc. to my just created $HOME (since I use the same user name).
I have the feeling that 8.10 works better than 8.04, for example the Network Manager works a lot better and pulse audio does too. You can try the liveCD to check if your hardware is supported, then just click on the install icon in the Desktop.
Good luck.
