### Post by Rhubarb on 2008-09-08
You could try out gthumb.
```
sudo aptitude install gthumb
```

[LIST]
[*]Open up a picture in gthumb
[*]Image --> Resize ...
[*]Enter in your resize preference
[*]Then press the save button
[/LIST]

Please note this will overwrite the original image you open up, so if you need to, make a backup of the original image before proceeding.
