### Post by The Cog on 2008-06-22
The two commands I can think of are **sudo lshw** and **glxinfo**. Both print a lot of stuff so you might want to pipe the output to a text file like this:
[B]sudo lshw > lshw.txt
glxinfo > glxinfo.txt[/B]
