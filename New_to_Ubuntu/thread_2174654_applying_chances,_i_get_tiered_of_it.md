### Post by CeejRab on 2013-09-15
Hello Kaj_rumpff,

I believe youre referring to "applying chan***_g_***es" which occurs whenever ubuntu installs a new package or software updates.

Make sure your ubuntu is up to date with the update manager. Then reboot your PC and open terminal. Type the following without quotes "sudo apt-get update"

Then type "exit" when it is done and press enter.

When installing a package, be patient, if you have a slow system it may take a while. 

"applying changes" is a normal part of installing packages and updates in ubuntu, nothing to get jumpy about ;)
