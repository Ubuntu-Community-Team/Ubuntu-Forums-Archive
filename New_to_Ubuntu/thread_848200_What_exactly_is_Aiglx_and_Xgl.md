### Post by S1xp4ck on 2008-07-04
[i] "Accelerated Indirect GLX ("AIGLX") is an open source project founded by Red Hat and the Fedora Linux community to allow accelerated indirect GLX rendering capabilities to X.Org and DRI drivers."   [http://en.wikipedia.org/wiki/AIGLX](http://en.wikipedia.org/wiki/AIGLX)


" Xgl is an X server architecture designed to take advantage of modern graphics cards via their OpenGL drivers, layered on top of OpenGL via glitz. It supports hardware acceleration of all X, OpenGL and XVideo applications and graphical effects by a compositing window manager such as Compiz or Beryl. The project was started by David Reveman of Novell and first released on January 2, 2006. "  [http://en.wikipedia.org/wiki/Xgl](http://en.wikipedia.org/wiki/Xgl) [/i]
