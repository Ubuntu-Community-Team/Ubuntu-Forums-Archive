### Post by madmax75 on 2011-03-25
> **weezyrider said:**
> FF has just updated, and I think Chrome might have. I don't see any options for these.

You may have to add PPA (Personal Package Archives = external software repositories) to get the very latest updates.

Here's advice on how to get Firefox 4 in 10.04 and 10.10:

[http://www.omgubuntu.co.uk/2011/03/firefox-4-ppa-for-ubuntu-10-04-and-10-10-users/](http://www.omgubuntu.co.uk/2011/03/firefox-4-ppa-for-ubuntu-10-04-and-10-10-users/)

You may want to check out Ubuntu Tweak. It's a great way to add extra software repositories and then some. I have used it for a couple of years now and so far no problems with it:

[http://ubuntu-tweak.com/](http://ubuntu-tweak.com/)

> All I get is Adobe Reader which I'm not interested in. What's a good native PDF reader? I usually prefer Foxit.

Evince (which comes preinstalled on your system) is a good one. Starts much faster than Adobe Reader. Applications -> Graphics -> Document Viewer (= Evince)

Foxit is a Windows program, so it's not a good option (you can probably rig it through Wine, but why bother?)
