### Post by tuxxy on 2008-08-26
Did you install nvidia-setttings package?

This allows you to edit resolutions, brightness, image qaulity, gamma correction, anti aliasing levels etc

Search for nvidia-settings in synaptic or type in terminal

```
sudo apt-get install nvidia-settings
```

To run the application once installed type

```
sudo nvidia-settings
```
