### Post by mc4man on 2015-04-17
For some reason *some *users/installs get this issue where some changes revert back to the defaults. Edge flipping, (edge flip move),  seems to be one that's affected.

A 'workaround' - 
open ccsm, set up as you want.
then, >   Preferences > Plugin List > disable Automatic plugin sorting.
in the right side column highlight rotate then click on Down until it's below unityshell
close out ccsm

If wanting to make ccsm changes down the road,  1st re-enable Automatic plugin sorting, make changes, then again disable, move rotate, ect.

I'm rarely affected here, maybe once in last several months (though am using 0.9.12.0+ in trusty...
