### Post by jbennett on 2009-01-30
First thing to do is make sure you've properly burned the ISO file to the CD.  This isn't functionality that's built into Windows, you need to have downloaded a third party tool.

Also, make sure your download of the Ubuntu ISO was successful.  You can do this by finding the MD5 checksum of the ISO file from the mirror you downloaded from (should be available from the Ubuntu download page).  Generate an MD5 checksum for the ISO you downloaded and make sure it matches the one provided by the mirror.
