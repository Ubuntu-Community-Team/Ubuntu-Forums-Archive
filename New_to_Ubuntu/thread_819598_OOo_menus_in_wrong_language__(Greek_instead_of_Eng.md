### Post by ru7hl3ss on 2008-06-05
This happened to me, how I fixed it was by changing the font the system uses (system->preferences->appearance->fonts tab). Unfortunately, I do not recall if it was application font, document font, etc... so I suggest trying to change them all. I am currently using DejaVu Sans Condensed and do not have that issue with OOo. Hope this solves the problem, good luck.
