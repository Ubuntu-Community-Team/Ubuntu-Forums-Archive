### Post by cjminton on 2010-02-03
If you are planning on running games made for windows using wine I would not recommend the 64 bit version of Ubuntu. Get the 32 bit version if you are a heavy PC gamer. However if you have no intention of running windows games under wine then I would get the 64 bit version. This is just my opinion, I've tried both versions and have had less hassle getting windows games to run perfectly with the 32 bit version. To be honest I couldn't tell the difference between 64 and 32 bit performance wise except that the 64 bit was more of a pain in the *** to get to work perfectly.
