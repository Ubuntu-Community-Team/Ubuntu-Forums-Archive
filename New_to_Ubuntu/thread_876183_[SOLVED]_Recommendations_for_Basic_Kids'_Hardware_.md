### Post by degroof on 2008-08-22
Thanks to everyone for the info. I used pricewatch.com to locate a cheap no-OS machine. It pointed me at the AscendTech Athlon 64 3200+, which I then customized to my specs. Here's the config I ended up with (scraped from the AscendTech site): 

Atx Black Midtower Case W/ 400watt Ps
Amd Athlon 64 Motherboard (Fic C51g S.754 Sata Pci-E Ddr400 2000mhz)
2gb Ddr400 Pc3200 
250gb Serial Ata Sata 7200rpm Hard Drive   
Ati Radeon 256mb X600 Pro 
Dvd Rom / Cd-Rw Combo Drive All In One 
REALTEK ALC655 5.1 CHANNEL STEREO SOUND CARD
10/100mbps Ethernet Adapter
Generic 104-Key Ps/2 Keyboard
3-Button Ps-2 Scroll Mouse Beige
Standard Computer System Speakers 
Dell 17" Tft Lcd Recertified Monitor

I *had* a wireless G PCI card in there too but it turned out to be useless. Swapped it out for a D-Link WUA-1340 USB dongle.

I installed Ubuntu 8.04 and added: 
- Edubuntu 
- Lokkit
- a secondary account for the kids, *without* root access
- Adblock Plus
- Procon Latte
- Java JRE
- Art of Illusion
- libdvdcss2

All told, it came to about $400, plus a few night's work (most of which was trying to get that $^##*ing WIFI card to work).
