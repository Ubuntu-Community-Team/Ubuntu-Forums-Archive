### Post by Victormd on 2008-06-02
I don't know why that's happening but I can give you some suggestions...
You can press the ALT+left mouse button to move the windows (just click anywhere in the window to move), without having to move the task bar...
More so, you can press SHIFT (I'm pretty sure it's shift, someone please correct me if I'm wrong) while closing the window to force ubuntu to remember the window position/size...

Another thought, do you have compiz installed? If so, you can use the "Place Windows" plugin and try different "placement modes" (i.e., Smart, Centered) to see if it fixes your problem...
