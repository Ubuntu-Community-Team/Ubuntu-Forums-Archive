### Post by Orange_and_Green on 2008-10-14
Hello :)

Go to System->Preferences->SCIM Setup. A window will open, expand the "Panel" entry, you will find an entry called "GTK". Higlight it. At the right of the window there should be a button named "Font" that allows you to change font type and size.

I have always been using SCIM and I'm very happy with it, so unfortunately I don't know of any alternatives to recommend...

Good luck :KS
