### Post by Ozymandias_117 on 2011-06-29
Are you talking about the Folder View widget?

If so, simply right click on the Desktop, select "Add Widgets" (You may have to click "Unlock Widgets" first if you've locked them) Then add Folder View.  You can go into it's settings to decide what folder contents it should show, etc.
