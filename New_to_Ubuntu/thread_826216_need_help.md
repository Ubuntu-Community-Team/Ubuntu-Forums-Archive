### Post by Lord Xeb on 2008-06-11
Plug it in, use ndiswrapper, then go in and type 192.168.1.1 into the adress bar of your internet browser. From there, use [this]("http://www.governmentsecurity.org/articles/DefaultLoginsandPasswordsforNetworkedDevices.php") to find your default login for your router, then once your in, configure it the way you need it. there are plenty of tutorials on how to configure a router. Linksys is not system specific and you should be able to set it up using linux.
