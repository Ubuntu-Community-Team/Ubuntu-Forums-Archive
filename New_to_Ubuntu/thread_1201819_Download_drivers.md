### Post by ramnarayan on 2009-07-01
Hi

Ok if you do have access to another machine which runs ubuntu and is connected to the internet then its simple. Or even if you can run a Live Ubuntu Session on a fast internet machine for a short while.


Make sure the Ubuntu versions are the same. 

On this Ubuntu (and internet machine) install apton
 Then run all the video and audio files you want to play and allow the Ubuntu to search for and install all the codecs etc. 

After you do this (you could install all the software you want) run apton and create an iso image. You can choose to write it to a disk or have it on your HD ( i prefer the latter).

Copy this iso image back to your Uninterneted computer and open the apton iso with archive manager and find the apton.deb file and install that.

This will install apton with will be available in System Administration or Preferences. 

Direct apton to the location of your aptonxxx.iso from where it will make available all the packages to you through Synaptic. (System --Administration-- Synaptic) and then you  can comfortably choose to install.

What apton does it that apart from the base install it collates all the additional software installed into one neat package which can be carried around to install in other same version Ubuntu's. It also serves as a good back up incase one needs to reinstall and does not want to go through the rigmarole of updating and reinstalling again. 
****

A second option is to find some one / or agency who have access to the Ubuntu Repository, about 5 DVD's worth. Try and get hold of those and you will never need the internet to have a fully loaded Ubuntu.

In India these DVD's are available for as low as US$ 5 - which is basically the cost of the media and postage. Its what i like to do to make sure internet access does not affect my Ubuntu experience.

ram
