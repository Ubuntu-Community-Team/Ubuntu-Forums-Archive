### Post by superzorro on 2009-09-23
Perhaps there is an easiest way, but this is the way I have managed to do it.

If you only like to change the default template in OpenOffice (as in MS Word normal.dot):

In OO Writer go to Format -> Page and change to your desired margins.

After that go to: File -> Templates -> Save  and give it a name e.g. Newnormal

Now go to: File -> Templates -> Organize and look for your new template, it should be in the folder "My templates" after you select it (e.g."Newnormal") go to the right and you will find a button named "Commands" select "Set as Default Template" and your done.

Now all the preferences that were selected (font, margins, etc) when you saved the Template will be part of the new default template
