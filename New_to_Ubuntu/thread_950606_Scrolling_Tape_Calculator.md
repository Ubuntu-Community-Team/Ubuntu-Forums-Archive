### Post by SirWeazel on 2010-02-28
Hard to find this type of app on almost any platform, then i found superbcalc. It will work on multiple os's, so its great because i use it almost everywhere. Its a small java app that you don't even have to install.

Note, you will have to have java installed. Also, you will have to right click and choose "open with java" to use, otherwise i think by default archive manager will open it.  You can also right click and choose "open with other application" and make java the "default" app so you don't have to right click all the time.

Just do Internet search for "superbcalc". Or, here is the web adress.

[http://www.mariottini.net/roberto/superbcalc/](http://www.mariottini.net/roberto/superbcalc/)

Hope this helps.

### Post by SirWeazel on 2010-12-05
JonPi, Please reread my post. The file you donwload will be "superbcalc.jar". You DO NOT want to extract this file. You want it to run as the java file it self so you need to have java installed. I'm not sure if you are running kubuntu or ubuntu, but in ubuntu you can just right click on the file and "open with java runtime."

Think of the file kind of like a windows .exe file. You want to "run" the actual file, and you want java to run it.

You could also open up the terminal and wherever you have the file at you could type Ex: if file is located in home directory "java -jar superbcalc.jar"

1. find where you donwloaded the file.
2. (in ubuntu) right click on this file, and choose open with java run time
OR you can right click on the file, choose properties, choose open with, choose java runtime.
3. It should work.
4. if it doesn't work, post back with any other info. (version of unbuntu, gnome/kde, ect.)

I hope you can get it to work. Please post back if you got it to work, or clarify any other instructions you had trouble with.
