### Post by kc1di on 2014-03-31
That will happen if grub can not find a Linux Kernel for some reason.
You may want to download and run boot-repair that may fix the problem

you can get boot-repair from [here]("https://help.ubuntu.com/community/Boot-Repair").

follow the instructions on the page. 

Also a little more info may be of help to us if boot-repair does not do the trick. 
Boot-repair save a log file online that can be posted here which may also point to the problem. 
beyond that please give us the following information, is this computer UEFI for instance. Number of partions/ drives.
