### Post by mcduck on 2011-10-19
If you are trying to resize a Windows 7 partition, you should do that using the tools in Windows itself, not with the Ubuntu install disc.

Other than that, just make sure the image you downloaded, and the burned CD, are free of defects. There's an option to check the disc in the first menu when you boot with a Ubuntu CD.

(If the partitioner is still scanning, and hasn't actually started editing any partitions yet, it should be safe to just power down the machine)
