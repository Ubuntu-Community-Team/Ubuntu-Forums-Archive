### Post by deadflowr on 2014-02-07
Firefox and Thunderbird updates will come when they come; no need to do anything special for those two, but the newer kernel series is something you'll need to install yourself.
The kernel should be part of the lts-saucy series.
more on kernel and graphics stack upgrade here
[https://wiki.ubuntu.com/Kernel/LTSEnablementStack](https://wiki.ubuntu.com/Kernel/LTSEnablementStack)

For anyone who downloads the latest lts from Ubuntu.com now, they will have that kernel series by default.

No.

Then those packages will keep getting whatever updates come for them.

I only mentioned that the newest version of precise on the website will be with those already by default.
