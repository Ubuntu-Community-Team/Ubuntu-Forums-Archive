### Post by mikeyphi on 2009-09-28
Lots of info, including wiki and forum here:
[http://www.eeeuser.com/](http://www.eeeuser.com/)
Start with the wiki....

However the short answer is that you should be able to either download the iso and burn to CD or (your choice) download the img for a USB drive.

Installing Ubuntu is almost automatic but there are plenty of us willing to offer help if needed! Please indicate which eeepc you have and if you're using only the inbuilt memory or a flash card.

Remember, installing a new OS will overwrite all the data on the eeepc, so you will lose xp along with the virus.

Also look here for more info re eeepc OS:
[http://www.array.org/ubuntu/index.html](http://www.array.org/ubuntu/index.html)

Good luck - but please ask if you need.
