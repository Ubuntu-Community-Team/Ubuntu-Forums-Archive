### Post by JC Cheloven on 2009-05-26
Hey, I assume you have tried the obvious things (right-click on network manager icon to see if everything is activated, left-click to choose a network to connect to, and so on).

Further, you can try system -> admin. -> hardware controllers
Here you can see if a proprietary controller is installed and in use for your wireless card or other stuff (video, etc). If some is in use (for wireless card), toggle it to not use, or vice-versa.
