### Post by drugon on 2009-01-25
Go to System -> Administration -> Hardware Drivers.

Here you will be able to see if your video drivers are installed or not. if not then go the manufacturer's website and you will be able to download it from there. 

You can also do System -> Administration -> Hardware testing to see if all of your hardware components are working great.
