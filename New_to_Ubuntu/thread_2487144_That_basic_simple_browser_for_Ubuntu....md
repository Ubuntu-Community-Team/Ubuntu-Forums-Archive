### Post by TheFu on 2023-05-24
"Web" is a generic term.  There are 50 different web browsers.  You haven't said enough for people to differentiate between them yet.
I use lynx, dillo, firefox, brave, chromium and a few website download tools like curl, wget, and custom perl scripts.  Which gets used depends on the job at hand.
For example, I use firefox for the 10 websites I trust and visit daily.  

Before 20.04, I used chromium in a specifically constrained container for financial websites and inside a new private container for sites like airlines, hotels, and untrusted places.  In 20.04 and later, I moved to Brave.  It has some issues, but I expect to figure those out.  I never use brave or (previously) chromium without a set of constraints to protect my system from dangerous websites.  I don't consider snap package constraints to be trustworthy. I want control and use firejail for these needs.

For sites I don't trust, I use either the text browser lynx or dillo.  Both don't support javascript, which is a good security practice. Hard for a website to hack you if your browser refuses to run javascript.
