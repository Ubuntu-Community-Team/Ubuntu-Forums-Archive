### Post by QLee on 2009-11-03
[QUOTE=Utahreddirt]I have 3 PCs on my home network. Two are running Vista and one is running XP. The XP machine is used as a file server and print server. I want to change the XP machine into a Ubuntu PC.

Well, you could use either version, depending somewhat on how you want the network to function. Is the server dedicated to the task?

For a "server" I would use a server LTS (that stands for long term support) version and not run X on it, but your needs may differ.

Reading this link will probably help you learn about samba (what you will probably use for file sharing)
[https://help.ubuntu.com/9.10/serverguide/C/index.html](https://help.ubuntu.com/9.10/serverguide/C/index.html)

Edit: I see that I gave the link for 9.10 which is the newest version of Ubuntu, there are other guides specific to whichever version you choose.
For example, [https://help.ubuntu.com/8.04/serverguide/C/index.html](https://help.ubuntu.com/8.04/serverguide/C/index.html)

These are good too for general Ubuntu help.
