### Post by halitech on 2008-05-09
You can go into Synaptic (easiest way) and just search for desktop and install xubuntu-desktop and kubuntu-desktop and then you can pick from the log in screen which you want to start on login.

If you are feel adventurous, you can do the same from the terminal by typing
```

sudo apt-get install xubuntu-desktop
or
sudo apt-get install kubuntu-desktop

```
and it will install them for you
