Here is a command that may help you to install all the media libs for ubuntu 

sudo apt-get -y install gstreamer*

If you want to play DVD then go to Applications > Add / Remove.. and type **DVD Movie** in the search text box and check all the applications and Click on apply Changes button.

You can get most of the video and audio files played in ubuntu 

If you feel your audio is not loud enough then double click on the volume icon on the task bar you will get the volume control window, make the changes there as per your need.
