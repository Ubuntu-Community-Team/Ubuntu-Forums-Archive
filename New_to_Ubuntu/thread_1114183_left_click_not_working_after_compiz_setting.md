### Post by xpod on 2009-04-02
Whatever you changed in the Compiz settings manager can easily be set back to default by clicking the reset button that each setting has next to it.
Remembering exactly which section of ccsm you were messing about in would probably help narrow it down of course:wink:

If you dont have a menu entry for ccsm you can start it via ALT-F2 with exactly that
```
ccsm
```
