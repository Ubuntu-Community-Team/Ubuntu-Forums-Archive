### Post by cmnorton on 2009-02-14
I would query the manufacturers, especially the more well-known ones like Logitech, Kensington, and so on. Ask them if their "stuff" works with Linux, specifically Ubuntu.

So far, I have found most USB peripherals -- plugged into Linux and Ubuntu Linux systems -- to work well with KVM switches, keyboards, and so on. An older Acer 630 Travelmate did not behave well in a KVM switch. It took a few seconds for the mouse to become responsive. I no longer have that problem with a Thinkpad T61p.
