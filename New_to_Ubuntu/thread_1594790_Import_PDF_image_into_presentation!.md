### Post by ajcham on 2010-10-12
It's easy to extract the image files from a PDF:

Open the PDF in Evince (Ubuntu's default PDF reader - you should be able to just double-click on the document), then right-click on the image you want to use and select "Save Image as..".  Save it somewhere useful (~/Pictures, for example) and you can then use it in your presentation program.
