### Post by trikster_x on 2010-10-31
> .if m running xp...can i still install this prog?
Yes, though I would suggest looking at a few guides for partitioning first.

> what does it run on
runs on most pc's with little or no modifications.  You should find out the specifics of the hardware your computer uses so you'll know if everything is compatible, and if not what you have to do to make it work.  Graphics card, wifi adapter, sound cards, all that stuff will be important to the difficulty you have installing and using ubuntu.

>  lets see if i dont happen to like it..its ez to reinstall it without losing files n stuff?
If you are running windows XP, I would suggest you try [wubi]("https://wiki.ubuntu.com/WubiGuide") before partitioning your hard drive and doing a full install of Ubuntu.  Then if you don't like it, you can just remove it like any other windows application.

> also let me know what do i need to install ubuntu...separate drive?
You can actually use the same hard drive you have windows on, but you will need to partition it first.

> what is ubuntu n kubuntu?
Ubuntu is the main flavor.  Kubuntu uses the K Desktop Environment (KDE).  I would suggest you start with Ubuntu and learn the basics, then install the Kubuntu packages and play with those on the side.  Kubuntu is a bit more fun to use, but Ubuntu is more user friendly, IMO.

> i wanted to install ubuntu desktop...whats that?
That would be another name for the main flavor of Ubuntu.

### Post by kaldor on 2010-10-31
Ubuntu is a Free operating system that can be used as an alternative to Windows or Mac OS X. 

It is built on the same general foundations that the Mac operating system is built on. This means it's efficient, stable and mostly malware-free. 

It's not a program. It's an Operating system for your computer.
