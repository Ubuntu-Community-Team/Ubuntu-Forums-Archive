### Post by Elfy on 2008-07-26
Firstly welcome to the forum, secondly have a good look at these 2 sites, the first (psychocats) I used myself when I turned up here. The second is a site which gets a good few references on the forum.

I would really recommend that you dual boot to start with, it is quite painless. The reason I say to dualboot is while I like and enjoy using ubuntu when I first installed it there were still a few things I needed to do with windows and of course trying to get stuff done with a new os took some time.

When you have read and are ready to go if you have any questions the place to ask them will be the [Absolute Beginners Forum]("http://ubuntuforums.org/forumdisplay.php?f=326") not here.

It is also possible now to install ubuntu within windows - this will give you the chance to get to know the os before you throw win away completely - it's not quite the same as a full blown installation but you can later transfer the installation to it's own partition.

When you do start to ask question in the forum  please, please use sensible titles for yourthreads - you are more likely to get sensible and concise answers to your problems and obviously telling people you really need help puts you in exactly the same place as everyone else - because evryone needs help when they have posted a thread :D

Good luck and enjoy the experience, walk away for ten minutes if you have to - it will still be there when youcome back.
