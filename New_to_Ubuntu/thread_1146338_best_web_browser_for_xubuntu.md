### Post by rmockler on 2009-05-02
I'm amazed that I don't see more reference on the the forum to the Seamonkey browser. I always install it immediately with every new release of Ubuntu that I install. Then when Firefox decides to act funky, I just switch over to Seamonkey and life is back to normal. Seamonkey doesn't do all that Firefox does, but what it does is done right. It also includes a great pop3 email client as part of the package. Bookmarks are compatible with Firefox, so just export them from Firefox and import them into Seamonkey and away you go. Seamonkey has a small set of extensions and some neat themes. I change between SeaFox, SeaGnome and SeaHawke 3D. You can find those themes by Googling.
Give it a try, you'll hardly see any difference from Firefox except it works. You can find Seamonkey in Synaptic.

### Post by Kixtosh on 2010-08-30
I don't know if I've tried Epiphany, but I solved my speed issues since my last post above by the following:

- Having installed Xubuntu 10.04, ...
- Use Ubuntu Software Center to add LXDE desktop environment,
- Change browser to Chromium.

In this configuration, I've got startup times on this old Portégé down to under one minute (about 55 seconds), and browser speeds are as fast as I've been able to achieve:

- The first time the browser is used, about 8 seconds to load.
- Subsequent sessions take about 2 or 3 seconds only.

So, if speed is a concern, and one is using older equipment with limited resources, that should work. In fact, it's as fast as Puppy in my tests, and significantly faster than Puppy to boot up too. Xubuntu with LXDE is also a more "grown up" interface in my opinion, so I much prefer it.

I suppose it depends on what you mean by "best web browser". For me, in this instance, it meant "which one will work at reasonable speed, with ten year old technical specifications, look good, and remain very stable ... even with a dozen tabs open in two or three simultaneous sessions combined". My usage is limited to regular browsing though, (nothing fancy).
