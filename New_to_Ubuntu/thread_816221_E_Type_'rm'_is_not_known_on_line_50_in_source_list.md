### Post by Sef on 2008-06-02
> How do I do that, I am COMPLETELY in the dark about how all of this works. 

Applications > Accessories > Terminal

> To get an uneditable copy
Quote:
cat /etc/apt/sources.list

To get an editable copy
Quote:
sudo gedit /etc/apt/sources.list 

Note: There is a mistake in the code for 'To Get an Editable Copy'

It should read ```
gksudo gedit /etc/apt/sources.list
```

You type in the code or easier just copy and paste the code in the terminal.  Just use ```
 cat /ect/apt/sources.list 
``` for until you need to make any changes, if needed.  No password needed for cat.
