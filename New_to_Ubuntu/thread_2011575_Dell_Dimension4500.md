### Post by Cheesemill on 2012-06-27
I wouldn't recommend using 8.04 on a desktop any more as it has reached end-of-life and no longer receives any support or security updates.

As it's an old machine I would recommend using either [Xubuntu]("http://xubuntu.org/") 12.04 or [Lubuntu]("http://www.lubuntu.net/") 12.04 as these will work much better on old hardware.

### Post by mapes12 on 2012-06-27
Most Linux distro's can be run on a live CD so if you wanted to try Ubuntu 12.04 then the live CD will give you an idea if the laptop can handle it. Keep in mind that running Ubu off a CD will always be much slower than from the HDD. You can then make sure that the hardware can run it, particularly any wifi adaptor you want to check out.

My Dad has an old Desktop with only 40GB HDD and 512k RAM and it runs Ubu perfectly. I did a fresh install of 12.04 on it the other w/e and it's running fine.
