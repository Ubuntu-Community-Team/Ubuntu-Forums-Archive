### Post by DuckHook on 2015-12-26
The best way to see if any of the 'buntu flavours work is to simply run a LiveDVD/USB session. All of the main flavours are built to boot as a live session from the DVD/USB.

Once you've driven them around for a few days, you can determine which run fastest and/or look best. The same LiveDVD/USB will then allow you to install onto your HDD.

In your case, I would highly recommend speed and stability over looks. My recommendation is Lubuntu.

Also, stick with 14.04, which is the latest Long Term Support version (until 16.04 comes out in April).
