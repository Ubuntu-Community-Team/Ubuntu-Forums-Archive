Just finished an Ubuntu installation (my first in several years), and despite the nvidia drivers (and the resulting headaches), all's well, except for one REALLY strange problem.

I've Googled 'til my fingers hurt, and can find anything related, so I thought I'd toss it out here.

I have two Epson Workforce all-in-ones, both set-up as network (WiFi) shared printers.

Even though nothing has been done (yet) to configure the Ubuntu box to utilize them, every single time the machine is booted (or, whenever a USB device is connected, or disconnected), both Epsons "come to life," and initialize themselves!?

Not only is it annoying (one is in my wife's office, and it scares the you-know-what out of her every time it suddenly "jumps to life"), it's also (unnecessarily) adding wear-and-tear to the Epsons.

Now what's odd is that again, these Epson Workforce machines are not in any way physically connected to anything (save for the AC), so this "wake up signal," or whatever it is they are receiving is coming via the WiFi, and this is 100% repeatable... I can sit here plugging-in (and unplugging) a USB memory stick, and every time I do, both Epsons come-out of hibernation  and initialize.

I even went so far as to completely re-install Ubuntu from scratch, to no avail... Same bug.

I'm at my wits end, and would sure appreciate any advice, suggestions, workarounds.....?

I should add that there are three Windows (7) machines on this same network, as well as a Macbook Air, and two Androids (a phone and a tablet), and none have any problem "playing nice" with the Epsons... This is truly a Ubuntu "quirk."
