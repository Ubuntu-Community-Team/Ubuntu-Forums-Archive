### Post by sdsurfer on 2019-12-23
This is not an Ubuntu issue, it is a FireFox issue and I've experienced it off and on for years in Windows and Ubuntu. There is not a definitive answer that I know of, but I still use FireFox as my main browser for both development and general use. FireFox is well known as a resource hog, mostly memory and CPU. That combined with the idea that developers have no issues using as many of the client's resources as they like. Things like React are one of the culprits. It's seldom malicious.

Hardware is definitely an issue. On one of my machines if I have too many tabs open, FF will choke. On more powerful machines it seems to make no difference.

One thing I do notice since moving to Ubuntu is if a software update includes FF it will choke very quickly after the install is complete and you have to restart the browser. There are some things web searches will reveal that may help, but I've never found an answer that makes it go away entirely. FF restarts are an acceptable compromise. :-D
