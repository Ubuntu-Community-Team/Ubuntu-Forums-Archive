### Post by Svensk023 on 2009-02-24
With the Kubuntu 8.10 live cd comes KDE 4.0 which had many many problems. But since it's release alot of them have been corrected, and if im not mistaken the current release of KDE is 4.2 which has been said to be alot more stable. Be in mind that KDE 4.0+ is not using konqueror as its file manager anymore, they have integrated Dolphin instead, which alot of people who made the change from 3.5+ to 4.0+ have voiced dissaproval.

By default GNOME is alot more mature at the moment so I currently use it as well. 

and as to answer your question, Ubuntu 8.10 is more stable out of the box than Kubuntu 8.10 but if you update KDE it may work just fine for you. But version 8.04 of either Ubuntu or Kubuntu is a long-term release, meaning its meant to be alot more stable than normal releases, so its meant to be supported for a longer period of time.

In my personal experience, i have only one problem with Ubuntu 8.10 and that was the wait period I had to endure for the new proprietary drivers to be released, other than that it works very well.

-Good Luck
