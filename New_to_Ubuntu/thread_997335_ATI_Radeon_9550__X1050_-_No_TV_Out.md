### Post by Mark Phelps on 2009-06-21
If you're talking about using Ubuntu 9.04, the reason you folks aren't getting an answer is because, there isn't one ... at least ... one that supports other than basic graphics.

AMD/ATI recently dropped support for all but the newer HD graphics cards in their restricted drivers, and the older restricted drivers (Cataluys 9.3 and prior) won't work with Ubuntu 9.04.

So, basically, you're stuck using the open source drivers -- which provide only basic features.
