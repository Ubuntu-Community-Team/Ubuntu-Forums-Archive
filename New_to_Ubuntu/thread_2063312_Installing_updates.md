### Post by jerrrys on 2012-09-28
You really don't have a choice about updating a fresh install.  Things just won't work or not work right.  An example of this would be the software center; no update after the install = no software center.

You have limited ram, maybe [swappiness]("https://help.ubuntu.com/community/SwapFaq#What_is_swappiness_and_how_do_I_change_it.3F") can be adjusted or maybe not.  Its a trial and error thing.  Really be better sticking some more ram in it.

As for this crashing.  One thing to look at is your video driver.  Are you using the best one out there?

I run an older system and found [gnome-classic]("http://www.ubuntugeek.com/how-to-install-classic-gnome-desktop-in-ubuntu-12-04-precise.html") runs better than anything else.  It can be added to your current install.

And last, some general reading [here]("http://www.googlubuntu.com/results/?cx=006238239194895611142%3Au-ocqbntw_o&cof=FORID%3A9&ie=UTF-8&q=12.04+performance&as_qdr=all&sa=Google+Search&lang=en&siteurl=http%3A%2F%2Fwww.googlubuntu.com%2F")

good luck
