### Post by Mark Phelps on 2010-06-10
IF you're not comfortable hand-editing the Win7 BCD (and I don't blame you if you are), then go to the Neosmart Technology forums, download and install EasyBCD (make sure you get build 100 or newer).

That provides a GUI for making changes to the BCD.  ALso allows you to backup and restore the existing one before you make any changes.
