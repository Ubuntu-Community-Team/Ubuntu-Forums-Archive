### Post by tomek_wap on 2011-11-05
something weird happened. due to messed up ATI drivers, had to reinstall Ubuntu from scratch ...

decided to install Windows 7 that was shipped with my notebook, to see if Bluetooth is there and if it works under Windows. And it did :)

I've switched on the Bluetooth in Windows (software switch), and left it like that.

Then installed Ubuntu replacing all data on HDD ... and then I found that Bluetooth is working under Ubuntu :)

What a surprise. Seems to me that the software switch had soemthing to do with this.

Anyways, problem solved :)
And maybe it's a solution for others to try out.
