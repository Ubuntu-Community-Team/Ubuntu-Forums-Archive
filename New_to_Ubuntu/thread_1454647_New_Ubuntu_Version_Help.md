### Post by lavinog on 2010-04-14
Clean installs can offer better performance.
There are some things that do not get upgraded when you upgrade such as grub or the filesystem.
Ext3 can be converted to ext4, but it wont have the full benefits.

It is also good to format a partition every now and then to reduce fragmentation (it happens over time...just not as fast as windows)

You also want to do a clean install if you have been tinkering with programs, and config files.
