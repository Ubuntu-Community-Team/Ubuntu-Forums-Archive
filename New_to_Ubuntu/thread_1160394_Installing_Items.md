### Post by Marlonsm on 2009-05-15
Linux does not use .exe, it's exclusive for Windows (although there is a way to run some of them).

To install on Linux, it has it's own format (like .deb, for example) BUT, the best way is to use Synaptic, or Add/Remove (this one actually adds programs, unlike windows' add/remove).
In Ubuntu, go to Applications > Add/Remove..., type whatever you want, and Ubuntu will find programs for you, click to install and it will do the rest of the work.


To run .exe you'll need a program called Wine (you can find it on Add/Remove), install, and right click any .exe you want to open. But remember, it's not perfect yet, some programs don't work 100%, so only use it if you can't find the program you want(or some good alternative) for Linux.

About .rar, install 7zip (if not installed yet), it will open them for you.


And if you have any doubts, just use the forums. :)

It's a pretty easy question because it has a simple answer: you can't expect exe files to run very well. You can install wine and run some Windows (.exe) programs with it, but you're better off installing software that's natively built for Linux (sounding complex? Understanding Linux isn't always easy). In Ubuntu, the most common file type for software packages is .deb, .deb files can be installed by downloading and executing it (like an exe) or you can use the built in software to connect to the software repositories for Ubuntu (which holds over 20,000 packages that you can install).

Hope this helps.
