### Post by Dennis N on 2014-07-25
Open Directory from File Manager: Right click on directory, select open with: select (or browse if needed to other applications) to VLC. (creates new playlist)

From VLC's Windows:

(Main Menu) To play directory: File > Open Directory 
Add files: (from Playlist Window Menu) > Manage > Simple Add File > (browse and select files to be added to playlist - you can add multiple files at once.)
Add directory to current playlist: (from Playlist Window Menu) > Manage > Add Directory (adds entire directory to current playlist)

My version of VLC could be older than yours and slightly different.

### Post by Dennis N on 2014-07-25
I don't use Gnome or Unity (I mostly use Xubuntu and Lubuntu, but do know the basics of Ubuntu Unity). The file manager in Gnome is just called "Files" - I think you have found it at the Filing Cabinet icon. I have read here in the forums that when right-clicking on a folder, the Open With > (choose application) has been removed in recent versions of Ubuntu's file manager. So, you would have to forget about doing that and open music from the VLC menus.

(the option in the file manager to open folder > (choose application) still exists in Lubuntu (all versions) and also Ubuntu 12.04)

### Post by Dennis N on 2014-07-25
You can try this from the file manager - I checked (in Xubuntu) and it works as an alternate method:

Open the music folder you want to play, and select all files (Ctrl + A will do this quickly, or use Edit > Select all) then right click on the selected files, and Open with > (choose Application) is available. They will load into a new list and play.
