### Post by roelforg on 2012-05-24
Ubuntu has package repos.
They contain packages compiled and uploaded by both canonical and the community + the packages from debian (cause ubuntu is based off debian).
The debian repos are filled in the same way.
Upstream (in the case of ubuntu) means either the debian repos or the source code of the app (it can take a while before the new version shows up  in the repo)
