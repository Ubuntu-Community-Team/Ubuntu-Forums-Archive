### Post by ibutho on 2009-01-29
Xubuntu is Ubuntu with XFCE as the default DE. Gobuntu is the same as Ubuntu, but without any proprietary stuff. Mythbuntu is the same as Ubuntu, but has been optimised for use with mythtv which is digital video recorder.

### Post by Sirron on 2009-01-29
There's also [_Ubuntu Studio_]("http://ubuntustudio.org/") which is for multimedia work, and countless other 'unofficial' variants.

In case you're trying to find a solution that covers all ubuntu software, it's worth noting that the differences are usually just that they look different and include different packages in the default installation.

For example, you can easily install KDE on ubuntu, and any of the other Kubuntu packages, Kubuntu just has KDE installed by default.
