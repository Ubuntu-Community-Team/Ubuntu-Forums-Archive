### Post by themusicalduck on 2010-11-06
Beagle indexes files for quick searching. There's probably a search program somewhere in your menus that allow you to search through indexed, I use Tracker instead, so I'm not sure how to use Beagle but, I expect it's very similar.

You could probably disable it by uninstalling Beagle from Synaptic or perhaps by disabling it in Startup Applications if it's listed there. To be honest, even though it says that it's using up your CPU, it should run at a very low priority and allow other applications to use the CPU when it's needed. I never notice it when tracker is indexing things, I just let it get on with it in the background and it seems to do it very efficiently.
