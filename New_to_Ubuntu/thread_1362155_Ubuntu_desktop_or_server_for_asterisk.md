### Post by medley on 2009-12-22
Asterisk will run fine on an Ubuntu desktop, but I have to ask what your intentions are in running *. If you intend to use it as a fulltime PBX, I would suggest a dedicated system just for it. I have about 3 years experience with Asterisk (it provides my sole telephone access for my household). When you run it on a desktop, you will be interrupting service from time to time when you reboot, etc.

To answer your question, yes it will run on a desktop/no you don't need to run it on a server. Part B, though, is that it probably doesn't make sense to do so.
