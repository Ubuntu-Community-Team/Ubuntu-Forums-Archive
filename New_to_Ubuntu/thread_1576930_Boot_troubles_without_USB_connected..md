### Post by wilee-nilee on 2010-09-18
No boot loader is installed in the MBR of /dev/sda

For some reason when you installed with the thumb it didn't put grub in sda.

So here is a link to the commands I will give you.
[https://help.ubuntu.com/community/Grub2#Reinstalling%20from%20LiveCD](https://help.ubuntu.com/community/Grub2#Reinstalling%20from%20LiveCD)

So boot the thumb if it will get you to a live cli or a live cd.
```
sudo mount /dev/sda1 /mnt
```
then
```
sudo grub-install --root-directory=/mnt/ /dev/sda
```

Reboot and run in the OS without the thumb in and run
```
sudo update-grub
```

The only thing I left out was that what ever live cd or thumb you use for this has to be a Karmic or beyond version.
