### Post by mlentink on 2009-04-02
For all practical purposes, *all* programs *always* run in RAM. A program might be programmed such that it uses additional pieces out of a library of (sub)programs, but when it does, those pieces are loaded into RAM.

A LiveCD cannot store data (it is a 'closed' cd).

BTW: Ubuntu, or windows, for that matter, is itself a 'program'
