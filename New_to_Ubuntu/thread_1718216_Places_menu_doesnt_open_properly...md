### Post by plucky on 2011-03-31
> **monicka said:**
> Hii everybody,
                   when i click places menu and open desktop means,
it is opening with media player and shows some error message.
But computer item is opening from the places .
Please help me to fix it .I dont know how to update nautilus libraries also.

Type Alt + F2 and in the window that opens type **nautilus**

Then right click on any folder and select **Open with Other Application**.In the window that opens, select **File Browser**.

This will reset your folders to be opened by the file browser rather than the media player.

Good Luck
