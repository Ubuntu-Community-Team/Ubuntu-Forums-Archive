### Post by mrmaceurope on 2013-04-20
I have found a working workaround for the Skype Video Webcam blue flashing issue. It is a workaround but it fixes the issue.

1. Remove current version of Skype (4.xxx) -- sudo apt-get remove skype
2. Download and install the old version of Skype - (I found this link/version) [http://download.skype.com/linux/skype-ubuntu_2.2.0.35-1_i386.deb](http://download.skype.com/linux/skype-ubuntu_2.2.0.35-1_i386.deb)

Problem kind of solved --- until someone are able to fix the video issue in the new version of Skype.
