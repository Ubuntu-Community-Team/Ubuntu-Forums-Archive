### Post by Blasphemist on 2011-09-30
The easiest way to fix this is to run your live cd/usb and use boot-repair, unless there was a serious error during installation. [https://help.ubuntu.com/community/Boot-Repair](https://help.ubuntu.com/community/Boot-Repair)

This also uses the boot summary that the earlier posted script would produce. Doing this from the live cd will get past your password issue and automatically post the results of the summary on the web for us to review. Just pass us the url. You can create the summary for us to look at before you have the program do anything else it you'd like.
