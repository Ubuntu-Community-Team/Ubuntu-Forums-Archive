### Post by Stan% on 2011-01-10
> **Chris Edgell said:**
> Somewhere in the wayback time I vaguely remember what it means if the computer just beeps when I turn it on.  (It is a second computer and I hadn't turned it on in quite some time.)  

Seems like there are coded beeps for the Dells, but this is my old Antec.  Is there an easy answer.

If it's a single beep, that's good. System is okay. If it's a series of beeps there is a problem. You first need to know the BIOS manufacturer. 

On the first screen or two do you see the letters/words AMI, IBM, Award, or Phoenix? One of those should be the BIOS manufacturer. 

Check the following web site for beep code interpretations......[http://www.pchell.com/hardware/beepcodes.shtml](http://www.pchell.com/hardware/beepcodes.shtml)
