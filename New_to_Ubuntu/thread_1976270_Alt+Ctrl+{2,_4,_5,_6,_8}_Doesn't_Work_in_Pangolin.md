### Post by grahammechanical on 2012-05-08
First, hold down the Super and study the keyboard shortcut overlay and see what key combinations are bound to what action.

Second, install CompizConfiguration Settings Manager (if not already installed) and use the Ubuntu Unity Plugin and see if you can set things up that way.

There is also System Settings>Keyboard Layout>Options

Regards.
