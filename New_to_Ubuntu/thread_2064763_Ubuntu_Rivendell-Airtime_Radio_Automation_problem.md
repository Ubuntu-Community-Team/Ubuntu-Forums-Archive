### Post by newb85 on 2012-09-30
I think the .mp3 issue will be the easiest to overcome.  For legal reasons, Ubuntu doesn't ship with codecs for .mp3, .wma, or other proprietary media formats.  Many of them can be installed by installing "Ubuntu Restricted Extras".  (You'd probably find it easiest to install it through the Software Center.)

### Post by grahammechanical on 2012-09-30
Hi and welcome to the forums.

Most of what you are talking about is over my head but I can offer this:

1) I do not think that the XP install can interfere in any way with Ubuntu once Ubuntu has loaded. Although it does take up disk space.

2) Those 2 computers may not have video cards but they must be using some kind of video adapter built into the motherboard. It could be that these adapters are using system RAM to substitute for the lack of memory on a discrete video card adapter. It could also be that the allocation of system RAM to video use is different on each machine and that may be the reason for different results.

3) Using system RAM as video memory is not so much an issue if the machines are going to be used 'headless,' without a monitor. Just make sure that system can get hold of as much memory as it needs.

4) Do you not have a spare video card that you could put into these machines specifically to set up the systems and which you will then remove?

5) Could you not think of Remote Login to access these 2 machines when needing to make adjustments in the future? Ubuntu 12.10 will soon be released and that comes with Remote Login as an option on the Ubuntu login screen. I do not know how it works. I do not have that kind of setup but it seems to me as if it might be useful to you in future.

Regards.

I have just thought of something else:

You can use an alternative desktop that is not so video intensive. In Ubuntu 12.04 you can install Unity 2D but that option will not be available from 12.10 onwards. I am testing 12.10 and I have installed fvwm-crystal as a lightweight alternative for when the standard Ubuntu Unity 3D desktop breaks. It will let me get to a desktop and do repairs such as updating.

You select an alternatively installed desktop from the login screen by clicking the Ubuntu icon on the password panel. The last used desktop becomes the default desktop.
