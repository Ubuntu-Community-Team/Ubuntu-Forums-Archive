### Post by Mark Phelps on 2010-02-18
> **narutarded said:**
> ...i want a software that i can buy from a store like best buy to let microsoft programs work on linux.
Wine (and its variants including Crossover and PlayOnLinux) are the general ways of doing this.

Before you do a lot of work, check the WineHQ Application Compatibility site for the MS apps and versions you want to run:

[http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true](http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true)

Anything that doesn't show up on a search, or returns a rating of "Garbage" or "Bronze" is either not going to work at all, or run so badly that it will be a waste of time installing it.

And, before you ask, Wine can NOT be used to install MS Windows drivers.

>  if possible, the software also needs to be able to let anything by apple be install on it so an ipod/ipod touch can be used.

Wine only supports MS Windows apps, not Apple.

There are Linux apps that will allow you to attach iPods, but don't plan on trying to use iTunes.

The simple answer is VirtualBox.
