### Post by GiuHei3u on 2011-05-19
Downloaded a doc file to my /download directory and it opened with Libre Office 3. The doc had links for me and when I click on them they don't open. So I right clicked on the mouse and copied it and then went to my FireFox browser to do a paste of the address but the paste was not highlighted.

What must I do to be able to do one of the two things just mentioned?

thanks.

---

### Post by r-senior on 2011-05-19
By default LibreOffice only invokes a hyperlink if you hold down the control key while you click. You can turn off this behaviour in Tools -> Options -> Security -> Options.

The paste should work OK. Try that one again.
