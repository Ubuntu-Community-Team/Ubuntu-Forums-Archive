### Post by y-lee on 2008-08-27
Open office has the ability to make pdf files. 

[LIST=1]
[*]Open or create the document  you need to create a PDF out of, using OpenOffice Writer.
[*]File -> Export As PDF…
[*]Browse to the folder you want the PDF to be created in, type in the name, and hit the Save button.
[/LIST]

I however would recommend learning how to use [LaTex]("http://www.latex-project.org/") and [ps2pdf]("http://www.linux.com/articles/35022").

As for greek fonts I don't know, you may want to see [Greek font help ]("http://www.perseus.tufts.edu/Help/fonthelp.html")or try google.
