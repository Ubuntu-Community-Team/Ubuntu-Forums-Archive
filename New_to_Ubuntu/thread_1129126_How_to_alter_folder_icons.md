### Post by shabazzk on 2009-04-18
Click Places > Home.

Then select Edit menu and choose preferences, this will take you to the File Manager Preferences, where you can change the icon size.

On the "Views" tab, change the icon size to less and what it is.

This should be in System > Preferences, but for me its not. But I guess I can figure out how to put it there.

Change color of folders in GNOME is not supported now, Gnome uses the emblems instead. You could try KDE, which I think does support it. Or you could vote for them to add it in Gnome and wait ([http://brainstorm.ubuntu.com/idea/4828/]("http://brainstorm.ubuntu.com/idea/4828/")).

As for the arrows, not sure if you can.

### Post by akelsall on 2009-04-18
Well, I can at least help you out with resizing your icons. All you need to do is right-click over the icon and click "Stretch icon...". The icon will change appearance, and you'll see four small squares in each corner. Just click and click on one of those squares. If you want to reset the icon to it's normal size (after you've re-sized it), right-click over the icon and select "Restore icon's original size."

I'm not sure if you'll be able to get rid of the arrow on the shortcuts. This might be something embedded in the operating system so that it reminds you that it is a shortcut. Might try poking around Google.
