### Post by grahammechanical on 2015-05-16
Do you wish that Portuguese is the default language? Do you wish Portuguese to be the language used during the installation? Both are possible. See the first and fifth images in this link

[https://help.ubuntu.com/community/GraphicalInstall](https://help.ubuntu.com/community/GraphicalInstall)

After installation we can go to System Settings>Language Support and install other languages and the Keyboard utility will allow us to install other language keyboard layouts. This makes it possible to switch between languages.

If we wish to run the Ubuntu live session in a language other than English then at the first purple screen press Enter and select the language. At the next screen we can select Try Ubuntu or Install Ubuntu.
