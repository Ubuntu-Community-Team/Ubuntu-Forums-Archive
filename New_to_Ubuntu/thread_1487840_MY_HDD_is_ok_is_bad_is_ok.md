### Post by cariboo on 2010-05-19
It may be that the hard drive has an intermittent problem. Go [here]("http://www.samsung.com/global/business/hdd/support/utilities/Support_HUTIL.html") download and use the diagnostic utility on your hard drive. If there is a problem, you'll get an error code, it will tell you if you need to return it.
