### Post by neurostu on 2008-10-02
I'm not sure what you're trying to do here, but if you want to get connect to freenode's irc servers do the following:
Install a chat client (if you haven't already)
```

sudo apt-get install xchat

```
Launch your irc chat client then connect a server with:
```

/server irc.freenode.net

```
then join a chat room with:
```

/join #ubuntu

```
just replace ubuntu with the name of the room you want, leaving the # sign.
