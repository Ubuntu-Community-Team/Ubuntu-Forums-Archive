### Post by 3rdalbum on 2011-02-08
Welcome.

You don't need to be scared of compatibility issues; partly because most things are compatible, and mostly because Ubuntu has a "live mode" that allows you to test Ubuntu and its compatibility BEFORE installing it.

Boot up your computer from the Ubuntu CD and when asked, choose "Try Ubuntu". After a delay, the desktop will load. Plug in your 3 mobile broadband dongle and wait a couple of seconds (nothing visible will happen, just give it a few seconds to recognise). Now go to the top panel on your screen, towards the right - one of the little icons there will be the Network Manager, and it will probably look like some sound or radio waves going upwards, with a little red cross.

Right-click it and go to Edit Connections. Go to the Mobile Broadband tab. Click Add, follow the onscreen directions, and it should set up your dongle for use within the live session.

When that's all done and you're back at the desktop, left-click the Network Manager and choose your 3 broadband connection. When it connects, use Firefox and see if you can get some sites to come up. If you have trouble at this point, take a look at the advice I've printed in this article: [www.chrislees.info/ubuntu/opendns.shtml](www.chrislees.info/ubuntu/opendns.shtml)

When you can confirm that your broadband dongle is working in Ubuntu, then you don't need to be scared anymore. Back up all your data (as is prudent whenever you're making invasive changes to your computer) and run the Ubuntu installer. Note that you'll need to set up your dongle again for use with the installed system, but this is done the same way.
