### Post by panorack on 2008-08-19
I have found 'Unix in a Nutshell' published by O'Reilly invaluable. Although it is not aimed exclusively for Linux it has sections on commands that are specific to the various form of Linux as well as the basic core of Unix, on which Linux is based.

A new book focusing on Ubuntu Linux is 'Ubuntu Linux Toolbox' published by Wiley. Haven't had chance to study it yet but it is reviewed in this month's Linux Format and gets a rating of 10/10. 

Both of these can be obtained through Amazon at less than cover price - and they qualify for free delivery.

Hope this helps.
