### Post by monkeybrain20122 on 2014-11-08
You can use Libreoffice draw, okular can also fill pdf forms. You can also try Master PDF Editor [http://code-industry.net/free-pdf-editor.php](http://code-industry.net/free-pdf-editor.php), or run pdf-xchange viewer in WINE. There are many ways, I wouldn't bother with Adobe's unmaintained bloatware if I can avoid it.
