### Post by nhasian on 2008-11-04
first make sure you have the latest firmware for your router.  you may need to download an update directly from linksys' website.  sometimes router's have tools in their setup page to see if there is a newer firmware, but that doesnt always work...

anyway, for dyndns to work all you need to do is enter your hostname, username and password that you setup on dyndns' website.  You dont need to worry about any ip addresses.  it will update the ip address automatically.
