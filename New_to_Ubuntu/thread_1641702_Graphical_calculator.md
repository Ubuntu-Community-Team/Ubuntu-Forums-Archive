### Post by ender4 on 2010-12-09
In my opinion the best program that can do all of that is Qalculate.

However for there are several programs that are specialized in a certain kind of task.
  For solving equations symbolically (or any other symbolic work for that matter) I use maxima, which has a GUI interface called WxMaxima.  For drawing graphs there are several programs, personally I prefer KmPlot for simple functions, although as bhishan mentioned Lybniz is pretty good, and doesn't require KDE (I am a KDE user myself).  For matrices, there are Octave (QtOctave is one GUI frontend for it, there may be others), Scilab, and Freemat, which are all essentially MatLab clones, and all the GUI's have a way to create matrices using a spreadsheet. 

Most of these programs (except the plotting programs KmPlot and Lybniz) can do everything you want, as long as you want to spend to time to figure them out, and a lot of them are just glorified consoles.

Hope that helps.
