### Post by robin7 on 2014-05-07
Are you using 14.04? If you are entering that command in the Terminal, that's *not* the update manager! And you do not want to install "untrusted" packages as a general rule. Use the menu to find Update Manager and use it instead of the Terminal for updating, at least while you are an "absolute beginner" (which I am assuming because you posted your question here in the Absolute Beginner section).
