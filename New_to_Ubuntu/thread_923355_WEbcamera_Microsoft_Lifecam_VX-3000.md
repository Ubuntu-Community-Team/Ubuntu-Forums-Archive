Hi, apparently that camera uses the same driver mine does (gspca).

Step 1: go to Synaptic Package Manager under system-administration.

Step 2: find "gspca" package and mark for installation

Step 3: Click the checkmark at the top that says "apply" or something and it will install the gspca driver

Step 4: Go to add/remove programs and get "cheese" and try it out, its pretty much photobooth for ubuntu.

Lemme know if it works for you!
