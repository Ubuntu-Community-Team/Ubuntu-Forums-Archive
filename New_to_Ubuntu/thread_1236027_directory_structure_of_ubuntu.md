### Post by matthewbpt on 2009-08-09
In Linux (and all Unix based OS's) a file or folder whose name starts with a period is hidden, ussually these are files or folders with configuration files. In ubuntu if you are in the file browser, click on the view menu at the top of the window and tick "show hidden files and folders," you should now see a many new folders in your home folder, including the one you're trying to find '.config' which contains configuration files for many programs in ubuntu. Hope this helps =)

### Post by Possum Films on 2009-08-09
Any file or directory that begins with a dot (such as .config) is hidden. You can see hidden files by clicking on View > Show hidden files in the file browser.
