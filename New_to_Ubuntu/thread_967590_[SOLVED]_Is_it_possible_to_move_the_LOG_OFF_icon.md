### Post by nothingspecial on 2008-11-02
right click select move, then move it. If it wont move then select remove from panel, then right click panel again add log off to panel then try moving it again

### Post by kahping on 2008-11-02
If the "Lock to Panel" option on the right click menu is enabled, you need to disable it first before you can move it
