From their forums

Linux Installation:

> Linux
1. Extract rainslickep1_linux_r1.tgz into a folder anywhere you'd like to have it located.
2. Ensure you have execute permission on RainslickEp1 and RainSlickEp1_bin
3. To run the game type ./RainSlickEp1 from the command line or double click RainSlickEp1 from within a graphical file manager such as Konqueror (KDE) to start your new adventure.
