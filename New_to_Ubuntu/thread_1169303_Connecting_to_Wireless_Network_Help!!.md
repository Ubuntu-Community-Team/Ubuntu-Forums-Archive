### Post by l-x-l on 2009-05-25
I'm guessing here but maybe you can try to right click the panel, then "Add to Panel". Then look for the Network Manager applet. If that doesn't work, right click the panel, "Add to Panel" then select Application Launcher & look for Network Manager. If you can't find it, right click the panel, "Add to Panel" then select Custom Application Launcher & under command type "network-manager".

Hope this helps.
