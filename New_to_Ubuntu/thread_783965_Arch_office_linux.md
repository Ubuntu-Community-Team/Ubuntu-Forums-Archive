It seems pretty much dead to me.

From the website...
> 
This project started: 31-July-2006
Last update: 17-December-2006 
Kernel version 2.6.18. 
Uses LILO


The current kernel version in arch is 2.6.24.
I saw the installation wizard screenshots at the website. I can't see any difference from Arch's standard installer (except that KDE + additional packages pre-bundled). But you can always install what you want with pacman -S <package>. See [Beginners Guide]("http://wiki.archlinux.org/index.php/Beginners_Guide").

I would say don't use it.
