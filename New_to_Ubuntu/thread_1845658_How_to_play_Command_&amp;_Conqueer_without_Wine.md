### Post by anewguy on 2011-09-17
One way to learn - install dosbox and try it.  If that doesn't work, try dosemu and try the game.  That's pretty much what I would have to do to be able to tell you, but I don't own the game.

Remember - it can't hurt to try in this instance.  By installing dosbox and trying your program I don't see how you could ever harm the rest of your ubuntu installation.

Dave ;)

EDIT:  Just found a download, originally posted in 1990's, on downloads.com for CnC Red Flag.  It states on there that it requires Windows 95 or Windows 98, and that it won't run in Windows XP.  Check the requirements for your game - if it requires some older version of Windows you would be best served with a VM as mentioned by Lisiano.  VirtualBox is good, and you could install Windows 95 or Windows 98 to it (if you're in the U.S., and if you have a floppy drive, I could send my old Windows 98SE to you if you need it).
