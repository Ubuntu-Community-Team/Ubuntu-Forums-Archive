### Post by fantab on 2013-04-01
Ubuntu 12.04 LTS will be a better bet. If you can wait till the end of this month then 13.04 will be released. 
Making it portable to be able to run from ANY computer is, AFAIK, not possible (with different Hardware on different computers) with a complete install as opposed to running a Ubuntu LiveUSB with persistence. However, LiveUSB will be slower than the proper Ubuntu install. 

So, if you want a portable Ubuntu that can boot on ANY computer you have to use [LiveUSB with persistence]("http://www.howtogeek.com/howto/14912/create-a-persistent-bootable-ubuntu-usb-flash-drive/"). [More Info]("https://help.ubuntu.com/community/LiveCD/Persistence").
