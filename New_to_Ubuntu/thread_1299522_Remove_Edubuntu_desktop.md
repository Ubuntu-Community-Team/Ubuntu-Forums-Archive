from synaptic:
This package depends on the complete set of Edubuntu-supported packages. It
provides an educational desktop on top of Ubuntu. It includes:
 * Edubuntu artwork and documentation
 * Preschool application bundle
 * Primary level application bundle
 * Secondary level application bundle
 * Tertiary level application bundle
 * Miscellaneous applications useful in educational settings

It is safe to remove this package if some of the desktop system packages are
not desired.
