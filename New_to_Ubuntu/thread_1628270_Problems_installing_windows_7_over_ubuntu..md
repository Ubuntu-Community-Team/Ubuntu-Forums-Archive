### Post by khelben1979 on 2010-11-22
A quick fix would be to change the harddrive and installing Windows 7 on that.

Otherwise I think the problem can be related to [MBR]("http://en.wikipedia.org/wiki/Master_boot_record") and it's definitely located on the harddrive, so wiping that clean should solve the problem he is experiencing.
