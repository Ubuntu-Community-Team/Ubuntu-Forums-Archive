### Post by mardybear on 2013-07-11
hi Smallwheels.

I run older versions of Ubuntu (not version 13) and most of my previous fresh installs required at least some tweaking to run proper, usually related to the screen resolution, sound and/or video card. My suggestion is **try a liveCD and see how everything works before making an install decision**, including video reslution, sound, graphic performance (not will run slower in live mode), flash video, internet, etc. Looks like you have a decent system, i'm running older Ubuntu on 10 year old hardware quite well and prefer to tweak my systems lean.

When i was first learning linux i set up a dual boot (WindowsXP and Ubuntu), so i could fall back to Windows if experiencing problems or wanted to use my old software.

In my experience the Ubuntu forums are pretty good since there is a large user base. Many Ubuntu users are happier with variants, such as Xubuntu, which tends to run leaner and has a more traditional interface.

If your genuinely interested and motivated, my advice is take the time to learn linux. I've been at it for ~7 years and am still learning but everything is more familiar now. Once you get a decent running system tuned to your preference, it is unlikely you will every want to return to an Apple or Microsoft product.

take care,

When installed, you'll be able to watch commercial DVDs in VLC, although probably still not in Totem Media Player.

The reason it doesn't install along with the Flash/MP3/etc support is because unlicensed decryption libraries are illegal in some parts of the world; yes, it's a crazy world. Some operations will be faster. You can still use a 32-bit OS on a 64-bit processor of course (the reverse is not true) but most people here have been using 64-bit operating systems for years without issue.

There is certainly a speed benefit with using a USB stick compared to a DVD.  However, a system installed to the HDD generally runs much faster than both these options.  Live DVD/USBs are really only intended to trial a release before installing it properly.
