### Post by Paqman on 2009-10-18
> **CrimsonBizarre said:**
> So far, I have only used ubuntu 9.04 32-bit on a Live CD. When I was using it, there was no sound. I think this was because it could not find my sound driver.


You probably don't need a driver as such. If you let us know what kind of computer you have, someone might be able to help troubleshoot your sound problem.

> If I were to install ubuntu on an external hard drive, and re-install the sound driver, would it have any effect on the sound driver working on my local operating system? (Windows XP).

It wouldn't touch Windows at all.

> 
Also, would ubuntu be a lot slower running ubuntu from an external hard drive? 


Yes. 

Speed across the USB bus is a lot slower than you'd get from an internal hard drive. I'd recommend installing Ubuntu onto your internal drive, not an external one. There's a very easy Windows based installer included on the disk. Just boot up into Windows, insert the Ubuntu LiveCD and it'll help you get it installed onto your hard drive really easily.
