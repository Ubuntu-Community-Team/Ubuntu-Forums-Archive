When I double click an archive in Nautilus, File Roller opens the archive. Can I set the default action to open it in Nautilus instead of opening it with File Roller?

---

### Post by fabricator4 on 2011-02-18
Right click on the tar.
Select "open with other application"
Select "archive manager", tick the box for "remember this application"

Good to go
