Noticed several old posts that I could not update.

To be able to open in Firefox .MHT files, simply install the latest version of the MAF add-on directly
[https://addons.mozilla.org/en-US/firefox/addon/212](https://addons.mozilla.org/en-US/firefox/addon/212)

MAF = Mozilla Archive Format
Save complete web pages in a single file. Supports MHTML (MHT) files from Internet Explorer and MAFF multi-page archives (based on ZIP)

[http://maf.mozdev.org/](http://maf.mozdev.org/)
