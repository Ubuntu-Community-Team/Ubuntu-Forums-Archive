### Post by Mr_JMM on 2008-11-22
If they are just updates to hardy then no reason not to install them. IMO it is always best to install the updates. They may improve things! Also, just because everything works now, doesn't mean that future additions will.

Some of them will be security updates which should definitely be installed.

Upgrades are a different matter. If Hardy is working for you and you're happy I would wait a bit before upgrading to Intrepid.

Bottom line is yes, install them.
