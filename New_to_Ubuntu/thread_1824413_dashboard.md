### Post by kherring7383 on 2011-08-13
Currently the new version of Compiz Fusion for Natty has a setting for Unity to re-size  the icons in the launcher(Dashboard) As for the application icons, there is no fix at this time to re-size them since the icons pixels don't re-size very well.  As a fix, you can add the Gnome 2 Menu to Unity which will appear at the top of the screen. Check out this link which explains how to add it.

[http://askubuntu.com/questions/39069/how-do-i-use-the-gnome-panel-in-unity]("http://askubuntu.com/questions/39069/how-do-i-use-the-gnome-panel-in-unity")
