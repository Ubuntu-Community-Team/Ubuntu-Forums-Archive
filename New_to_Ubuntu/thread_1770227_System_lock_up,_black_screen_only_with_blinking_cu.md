### Post by candtalan on 2011-05-29
Certainly consider to use a live CD to hopefully run the machine from CD drive and in RAM memory without calling on the hard drive installed system. Also use the live CD initial menu to check your PC RAM memory.

I do not know what the error message means. However, when running the live CD you can use the 'disc utility' to indicate what your hard drive status is, including probably its own reports.

If disc utility says the hard drive is OK then it is often possible to recover data using the live session. If your hd is failing, then it may (or may not) fail fast, so recovery of data may need thought. Then the more you use the hd then the more it may fail fast.
