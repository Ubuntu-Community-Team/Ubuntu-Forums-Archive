### Post by jerrylamos on 2008-11-08
I'm running fine on an IBM Thinkpad R31 which has 1 gHz Celeron but does have 512 mb memory.  I'd suggest 384 mb which is a bit marginal for Ubuntu (Gnome desktop medium eye candy) & Kubuntu (KDE desktop lots of eye candy).  Xubuntu should be a bit better using Xfce desktop, a bit plain but most function except for local lan access to other computers in my house.

I'd suggest turning off compiz the splashy eye candy & animation by selecting System Appearances Visual Effects None.

Try CD Live first.  If it doesn't work, the install probably won't.

Jerry
