### Post by bodhi.zazen on 2009-06-30
I [s]have never[/s] rarely used the cables, just the short adapters.

[IMG]http://tbn1.google.com/images?q=tbn:wRlBqheQa0n9aM:http://img.tomshardware.com/us/2005/11/23/pc_interfaces_101/ps2_usb.jpg[/IMG]

I have never had a problem with any usb / ps2 adapter I have used. My most frequent use is with old computers lacking usb ports or where the usb port is broken. It is sad that it is hard to find anything but a USB mouse these days.

The other use I have is using a wireless mouse / keyboard with a KVM. Most older KVM do not have a usb port so use a usb wireless keyboard / mouse w/ a splitter to connect to the KVM switch.
