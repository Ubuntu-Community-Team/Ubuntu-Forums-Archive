### Post by 3rdalbum on 2010-02-22
Gnome 2.30 is the basis of Gnome 3. Gnome Shell is just the most visible change to Gnome.

Gnome Shell is interesting, but definitely not feature-complete. It's close to unusable for an everyday desktop - you really need to be running some sort of dock as well so you can actually get things done. You can try it right now, there are plenty of Gnome Shell PPAs and it's a doddle to compile from source.

No compatibility issues that I've seen, but you need 3D graphics support.
