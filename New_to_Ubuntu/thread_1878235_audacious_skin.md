### Post by checkit on 2011-11-10
> **Mark Phelps said:**
> From what I can tell, Audacious has removed the skin selection capability.  I used the WinAMP skin back in 11.04, but when I installed 11.10, and went to do the same, I noticed the skin selection no longer appeared.  Now, it looks like you have to choose between two "default" skins.

This is only true if you are using the GTK Interface.  If you change to the Winamp Classic Interface, then you change the skin.

In order select a skin in Audacious 2.5.3, do the following:
[LIST=1]
[*]select the Winamp Classic Interface (View --> Interface --> Winamp Classic Interface)
[*]go to Preferences --> Skinned Interface (this option does not appear in GTK interface mode)
[*]choose your skin
[/LIST]

Enjoy!

In Audacious 3.2.4, to change the skin you need to switch to Winamp classic interface, then go >>View>>Interface Preferences and choose the preferred skin ( I like Audacious grey theme from [HERE]("http://pakeklinux.wordpress.com/2010/11/17/the-best-audacious-skin-that-i-know/") ).
