### Post by kestrel1 on 2009-01-07
Not sure if this will help, but I have had problems with some text missing in menu's before & doing the following sorted it out:
Go to System - Preferences - Appearance then go to the Fonts tab & select Subpixel Smoothing (LCD's).
If that doesn't help go to the same appearances menu & the use the Visual Effects Tab & turn off all visual effects.
Let us know what happens.
