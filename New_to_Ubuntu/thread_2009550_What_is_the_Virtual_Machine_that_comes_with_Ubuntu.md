### Post by Curtis6767 on 2012-06-24
KVM, but I couldn't get it to work. I've been using Oracle Virtual Box. Nice thing is it comes as a .deb file so you can open with Gdebi or the Software Center to install it.

[https://www.virtualbox.org/wiki/Linux_Downloads](https://www.virtualbox.org/wiki/Linux_Downloads)

P.S. When using VB, you do not need to burn a CD/DVD to install and run an OS, such as Windows. You can set up VB to run and install off the ISO that you will/should have in your downloads folder. In fact, I've found it easier to go that route than to run OS's from disk.
