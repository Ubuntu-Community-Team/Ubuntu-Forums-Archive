### Post by tbzep on 2011-01-21
Photobucket:

Ignore the big green "select photos and videos" button and go underneath it for the "bulk uploader".  It's in small print, so look for it.

When I try the normal uploading method (big green button), it locks up Firefox.  When I use the bulk uploader, it causes some temporary slowdowns, but it uploads all my pics and then gets back to normal.
