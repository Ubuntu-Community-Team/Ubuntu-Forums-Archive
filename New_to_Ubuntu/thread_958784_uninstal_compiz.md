### Post by Pro-reason on 2008-10-25
Simply use the menus to get to your desktop settings, and turn Desktop Effects off.

You can also just uninstall it if you're not going to use it any more.  Do this the same way you would with any other software (i.e. you can use “Add & Remove”, or Synaptic, or you can type “sudo apt-get remove compiz” on the command line).
