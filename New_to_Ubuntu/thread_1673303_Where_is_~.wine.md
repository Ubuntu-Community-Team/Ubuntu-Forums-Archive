### Post by howefield on 2011-01-22
Open your home folder (Places > Home) and press the ctrl + H keys to show the hidden folders.

The "." in the file path indicates that it is a hidden folder.

~ means /home

---

### Post by Verbeck on 2011-01-22
~ means the current user's home directory
files/folders preceding with a dot are usually hidden by default, so press ctrl+H to see .wine in your home folder
