### Post by BT1 on 2009-11-14
I am currently running my 9.10 x86 install off of a thumb drive (full system install), but still mount my hard drives when I need to (because Vista is installed on them). I have to transfer 105 Gigs to a USB Hard Drive and in Windows Vista the speed was 18.2/Mbps but using ubuntu, the transfer speed is currently hanging around 24.6/Mbps.

On a side note, I'm still blown away about the transfer speeds when it comes to Hard Drive to Hard Drive using Ubuntu+Ext4 vs Windows Vista+NTFS. When using NTFS even with the correct drivers, I was getting transfer speeds around 30/Mbps. When using Ubuntu+Ext4, the transfer speeds are upwards of 50/Mbps.

So, it's actually faster with Ubuntu for me (and many others). There are some people who have USB issues but there's usually quick fixes when more brains get involved. This is a an issue that has come up for other people that I have seen in these forums, so perhaps do a quick forum search to see how others fixed it.

Hope it works out well!
