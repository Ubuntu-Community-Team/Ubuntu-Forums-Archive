### Post by ajgreeny on 2011-03-07
Did you just copy the .iso file to the usb stick or did you "burn" it as an image; if you just copied it. the USB will never do what you want; it is just a data stick.

Use [unetbootin]("http://unetbootin.sourceforge.net/") to turn the .iso file into a bootable USB, or burn it as an image to CD; there are plenty of applications, free and paid for, available for windows to do that.
