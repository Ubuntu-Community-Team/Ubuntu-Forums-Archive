### Post by anewguy on 2010-04-20
I use a Netgear router that has a little tiny push button in the back to reset it, so I assume you are talking about something similar?

On my router, reset sets it back to factory defaults - this includes wiping out any encryption and/or MAC filtering, essentially leaving me with an open network.

But it's only temporary - once you get into the router with the default admin userid and password you can then reset all of your encryption, MAC filtering, etc..
