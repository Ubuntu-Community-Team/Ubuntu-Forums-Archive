### Post by ppv on 2010-12-10
Ubuntu can install in fairly small amounts of disk space but it depends on what packages would you have. So usually something between 10-25 GB would also be fine for Ubuntu. You would have to set it up as dual boot to access windows. After installing it as dual boot you can also access windows partitions from within ubuntu. You may also want to backup data before doing all of this.

### Post by IndyGunFreak on 2010-12-10
my "Main"... Ubuntu partition, is 20gigs.  I have about 10 free.  I keep all my music, videos, etc.. on a separate partition.  I also have a 6gig virtual disk for XP on my main Ubuntu partition... so really, with "just Ubuntu", I only have about 4gigs in use.

As far as programs installed, I have a lot (lots of multimedia stuff, 3-4 web browsers, couple IRC clients, various IM clients, remote desktop stuff, Frostwire, Google Earth, you get the picture).  Personally, I think the average user who is keeping their music, etc.. on another partition, can get by quite easily w/ a 15gig partition.  I think the "base" install is around 2-3gigs.
