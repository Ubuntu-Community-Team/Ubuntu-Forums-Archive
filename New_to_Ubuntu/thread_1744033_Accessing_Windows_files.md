### Post by bcbc on 2011-04-30
Assuming you are running Unity (the dock on the left). Click on the Home folder at the top, then File System, then on "host". That's the windows host partition.

If it's a different windows partition you want it should be listed in the left panel (under Network) by partition label or size.
