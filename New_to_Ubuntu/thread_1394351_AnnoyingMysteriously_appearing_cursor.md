### Post by TBABill on 2010-01-30
Easy fix. Go to Firefox ---> Edit ---> Preferences. Under Advanced ---> General tab, select "use autoscrolling". You may then have to restart the browser you are using for it to take effect, not sure. I had the same problem and this fixed it. Un-checking smooth scrolling may speed up scrolling web pages as well.
