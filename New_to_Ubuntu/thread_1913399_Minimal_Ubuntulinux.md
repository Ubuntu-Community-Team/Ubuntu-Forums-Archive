### Post by Lars Noodén on 2012-01-22
Of the pre-packaged Ubuntus, Lubuntu is the lightest.  It uses the LXDE desktop environment and installs lightweight apps.  If you want to keep using kWord you can always add it after the rest gets installed. 

If you want even lighter, you could install only a window manager and skip the desktop environment.  Some of the choices there are Openbox, Fluxbox and FVWM-crystal (or plain FVWM).
