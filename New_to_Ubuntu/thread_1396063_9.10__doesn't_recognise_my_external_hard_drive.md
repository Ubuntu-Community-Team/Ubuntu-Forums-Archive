### Post by nhasian on 2010-02-01
for some reason it doesn't seem to be able to read/mount the fat32 partition on the hard disk.  you can try deleting the partition in gparted and creating a new one.  

If you are only going to use this hard disk on Linux computers, choose EXT4 as the filesystem for the new partition.

if you are going to to use it Windows as well as Linux Computers, choose NTFS as the filesystem.

finally if you need to be able to use the hard disk on a console like an xbox360 or PS3 (or dvd player with usb input), then you will need a FAT32 partition.

[COLOR="Red"]WARNING[/COLOR] deleting the partition on the drive will destroy any data you have on it.  so make sure you select the correct drive using the drop down menu in gparted before deleting the partition.
