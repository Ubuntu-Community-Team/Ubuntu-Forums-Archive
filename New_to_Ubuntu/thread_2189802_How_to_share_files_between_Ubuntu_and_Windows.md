### Post by jenswagner20 on 2013-11-24
Right click on the file in windows, properties and set correct permissions. 

To share files between windows and Linux, create a Windows partition with FAT Filesystem and name it Gallery or something. It will be visible in Windows ('My PC') as another drive and in Linux too.

### Post by fantab on 2013-11-24
Ubuntu can read and write to NTFS partition. FAT is has its limitations. If I were you I would rather share NTFS partitions between Windows and Linux.
