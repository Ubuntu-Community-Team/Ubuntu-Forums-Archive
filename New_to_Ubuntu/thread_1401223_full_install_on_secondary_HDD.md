### Post by mechro on 2010-02-08
The instructions in my post apply if you are putting the HDD back in the laptop i.e. there's no point in doing a normal install while the HDD is in your desktop as the "wrong" hardware will be configured. Unetbootin is just an install method when you don't have a working CD drive, USB drive or even a floppy drive.

If the HDD is going to stay in your desktop then you can do a standard install from a LiveCD; choose manual partitioning and select/configure the HDD for installation.
