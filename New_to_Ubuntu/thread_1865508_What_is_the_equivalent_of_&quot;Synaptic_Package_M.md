### Post by Paqman on 2011-10-20
Synaptic isn't pre-installed in 11.10, but it's still in the repos. I tend to put it on all my machines, it's a good tool. Either search for it in Ubuntu Software Centre or open a terminal and enter:
```
sudo apt-get install synaptic
```

### Post by mcduck on 2011-10-20
By the way, all the packages *are* available through Ubuntu Software Center, it's just that it hides most of them unless you either search for them using the exact name of the package, or click the "Show *n* technical items" at the bottom left corner of the window. ;)

That being said, I still prefer having Synaptic around for all those more complicated package tasks.
