### Post by kellemes on 2010-01-15
Kubuntu..
Right-click caption bar -> To Desktop -> Select the virtual desktop you want the application to be.

You can control windows up to every little detail (including in what virtual desktop you want them to open) from System Settings -> Window Behavior.

### Post by IsstanarLW on 2010-01-15
You can run different plasmoids on each virtual desktop.
First left click the little icon in the upper right hand corner and select zoom out. Then, click configure plasma check the box for different activity for each desktop.
