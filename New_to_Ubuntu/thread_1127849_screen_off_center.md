### Post by jbrown96 on 2009-04-16
I've found that Ubuntu does this some times, but it can usually be fixed by using the "auto adjust" tool on the monitor. If that doesn't work, it would help to know some more about your system: graphics card, graphics driver, Ubuntu version, monitor type (LCD, CRT, TV, etc.), and screen resolution.
