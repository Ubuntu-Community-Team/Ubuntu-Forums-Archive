### Post by howefield on 2008-10-04
2 versions of virtualbox, you have the OSE (Open Source Edition) edition which is in the Ubuntu repository and can be installed via synaptic package manager. The other version is the PUEL (Personal Use & Evaluation Licence) which has a few more features like USB support, Serial ATA controller and other goodies, this version is installable via a .deb file downloadable via virtualbox.org.
