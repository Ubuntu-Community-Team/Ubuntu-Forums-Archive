### Post by cariboo on 2008-11-26
There is no root terminal in Ubuntu, use sudo to run commands as the root user, in a terminal type:

```
sudo synaptic
```

with out opening a terminal and running graphical app, press Alt-F2 and in the run box type:

```
gksu nautilus
```
