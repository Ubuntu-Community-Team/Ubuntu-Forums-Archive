### Post by cyfur01 on 2009-02-26
If you installed it from the .deb file on the page you linked to, you should be able to use synaptic to remove it.

Go to System->Administration->Synaptic Package Manager
Once that loads, click on search, type in floola, and it should show it. Right-click it and select "Mark for Complete Removal" if you wish to remove all traces of it.



Also, it should have installed a menu entry: Applications->Sound&Video->Floola
