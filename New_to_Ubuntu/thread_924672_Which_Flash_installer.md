### Post by adult swim on 2008-09-19
robti i think everytime ive ever tried to install a plugin from firefox's suggestions, it has never worked.  if this is the case for you, it is easy to install flash from a terminal, just type in ```
sudo apt-get install flashplugin-nonfree
``` restart firefox and you should be set.
