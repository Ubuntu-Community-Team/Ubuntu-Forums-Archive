### Post by I2k4 on 2012-04-10
If you are creating Live USB from Windows, you should follow this step by step exactly - it's foolproof:

[http://www.ubuntu.com/download/ubuntu/download](http://www.ubuntu.com/download/ubuntu/download)

The one extra step not shown at Step 2 is to create Persistence on the PenDrive Installer - use all space above 1.5GB on the USB  drive for Persistence.

This will create a very good testing installation on the USB and save your settings between boots (that's Persistence.)  However, these Live USBs go wonky after several weeks, so should be treated as temporary trials to learn about Linux and experiment with software, and should be relied on as permanent.

It's fun, enjoy it.
