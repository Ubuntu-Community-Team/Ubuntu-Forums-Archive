The lightest Ubuntu flavour is ***Lubuntu*** 13.10 and 14.04 LTS. If you get problems with the driver for (old) graphics, you might instead get a ***re-spin of Ubuntu 12.04 LTS***.

If it is not important to make the computer safe against unwanted 'tweaks', you can simply make it ***auto-login*** and ***auto-start a slideshow with the program feh***.
