### Post by cjhabs on 2013-02-22
An easy way, if you are not too familiar with the command line is to navigate to the folder using Nautilus. Open a terminal and type "cd " there is a space after "cd" -  then drag the folder from nautilus to the terminal and hit return.

### Post by androideka31 on 2013-02-22
ls /media    - your usb will be in this folder
cd /media/(usb)
you'll be in the usb directory. use ls (list contents) and cd (change directory) to navigate the folder. cd .. brings you up one  folder ie cd .. while you are in /media/usb will take you to /media.
