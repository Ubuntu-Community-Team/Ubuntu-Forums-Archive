### Post by billgoldberg on 2008-10-16
> **J03y said:**
> Just wondering, whats the difference between Ubuntu 8.04 LTS Desktop Edition CD and Ubuntu 8.04 LTS Server Edition CD?

The desktop edition comes with a GUI *(gnome)* and lots of apps installed that most people will need.

The server version is meant for servers. You can install lots of server apps when installing. It doesn't have a GUI, it's text only *(you can install a GUI if you want, there isn't one by default)*.

--

If you are thinking about what one you want to install, use the desktop edition.
