### Post by Cheesehead on 2012-06-11
Yes, it's possible. How you do it depends on which word processor you are using.

For example, in LibreOffice press F11 to bring up the Styles and Formatting window,
then click on the Page Styles icon,
then select Left Page or Right Page.

You will need to actually edit the Left Page and Right Page style formats to match your desired margins, font, font size, line spacing, etc.

I'm using Apache OpenOffice 3.4 under Windows but I imagine this will be the same in OO 3.2 under Ubuntu.

Select Help from the menu, select the Index tab and then type in "page styles". (Without the quote marks.)  One of the options offered will be "left and right pages".  It says it all, far better than I can.
