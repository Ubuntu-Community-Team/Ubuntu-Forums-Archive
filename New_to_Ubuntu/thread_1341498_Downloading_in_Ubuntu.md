Gnome themes are normally not in the .deb format that ubuntu uses, Ubuntu tends to use one theme per user but gives the user options to install themes independently.
Also for most software its best to stick to the repositories, most linux software is in the Ubuntu repos.
Here is some help installing programs in ubuntu:
