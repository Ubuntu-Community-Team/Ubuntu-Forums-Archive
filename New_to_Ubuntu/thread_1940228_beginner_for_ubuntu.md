### Post by winh8r on 2012-03-13
First make sure that software centre is closed, also synaptic package manager is closed if you have it installed.

Open a terminal

NOTE*** you should not run as root, use the sudo command instead.

enter these commands in this order:

```
sudo apt-get clean
```
```
cd /var/lib/apt
```
```
sudo mv lists lists.old
```
```
sudo mkdir -p lists/partial
```
```
sudo apt-get clean
```
```
sudo apt-get update
```

### Post by Frogs Hair on 2012-03-13
