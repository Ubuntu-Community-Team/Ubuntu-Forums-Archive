### Post by earthpigg on 2009-07-27
the more complex a solution is, the more can go wrong :D

my suggestion is to simply save/restore program settings for everything.

places -> home folder -> view hidden files -> copy everything to a backup medium


copy it back on fresh install.


BAM. firefox bookmarks/settings, pidgin stuff, etc, all the same.
