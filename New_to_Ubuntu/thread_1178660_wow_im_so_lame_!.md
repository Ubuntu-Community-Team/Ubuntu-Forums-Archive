### Post by Yvan300 on 2009-06-04
Ok lets see what you need to do.
1. Firstly, i suggest you back up all your important information. This is not necessary but if you have the time and storage media, back it all up.

2. You need to download an iso burning program. If you are running windows then there are many free programs that can do the job. Examples are at [http://www.ntfs.com/iso_burner_free.htm](http://www.ntfs.com/iso_burner_free.htm) or you could install the ashampoo burning suite. 
P.S When burning your ISO, if the option is available, choose the slowest burning speed that you can, if you have the time. This prevents errors when burning.

3. After you have burned your ISO, you have 2 options, you can either install ubuntu inside windows, through an installer called wubi or by booting from the CD as a live session. To install inside windows, you just pop in the cd while in windows and install. If you want to install it from the Live session, then you must enter the bios on startup and change the boot order so that the pc will boot from the cd. Once you've done that, the rest is pretty strait forward.
