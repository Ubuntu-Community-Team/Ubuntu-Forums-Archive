### Post by Autodave on 2017-07-29
Installing Ubuntu over (deleting) Windows is easy. D-L the one you want (16.04 is a long term service release and what I would recommend). You will obviously need access to another computer to d-l the file and then you need to install it as an .iso file onto a DVD or USB stick. Boot your machine with the disk or USB. Follow the prompts. When you get to the one that will seem confusing, you want to pick "Erase Windows and install Ubuntu".

### Post by Mark Phelps on 2017-07-29
Failure of a previously working Windows PC to boot up and/or login tends to indicate either (1) a filesystem problem with Windows that resulted in corruption of the boot loader files, or (2) a hardware problem with the hard drive that resulted in filesystem corruption.

If (a) you really want to get back into Windows, then tell us the version you are running (i.e., Win7, Win8x, Win10) and we may be able to provide links to stuff you can download from another PC and use to repair Windows to get it working again.

If (b) you don't want to bother with fixing Windows and (2) is the case, then Linux is not going to work any better because it won't install to a hard drive that has suffered hardware failure.

### Post by Impavidus on 2017-07-31
Your Windows no longer boots, but you still have a valid Windows license. You should be able to reinstall Windows without buying it again. If you don't have an install disk (or just a repair disk, which should be all you really need), it may be a bit more involved, but still a Windows problem. Keep in mind that Ubuntu is not a drop-in replacement for Windows. But you are free to try Ubuntu.

Note that to install Ubuntu alongside Windows the Ubuntu installer has to recognise the Windows system, which may not be the case if your Windows is damaged.
