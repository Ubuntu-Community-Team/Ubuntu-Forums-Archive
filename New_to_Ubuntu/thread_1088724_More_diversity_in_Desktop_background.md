### Post by swoody on 2009-03-06
Well you can certainly download new backgrounds. There are a lot of websites available, but a couple of my favorites are:
[www.gnome-look.org](www.gnome-look.org)
[www.interfacelift.com](www.interfacelift.com)

Then you can either right-click an image and "set as wallpaper" or you can save the file to your computer, and add it by going to System>Preferences>Appearance and clicking on the 'Background' tab. Then click "Add" and you can find where you downloaded your new wallpapers to.

You can also install the wallpapers from Ubuntu Studio by searching for their wallpaper packages in Synaptic package manager (System>Administration>Synaptic Package Manager) and using the search bar at the top to find the package:
ubuntustudio-wallpapers
Once installed, they will automatically show up in the Appearance Settings described above.

### Post by MockY on 2009-03-06
GNOME comes with a set of pretty nice wall papers.
```
sudo apt-get install gnome-backgrounds
```
And if you would want to be able to right-click on any picture, anywhere you are (nautilus, firefox and so on) install the following:
```
sudo apt-get install nautilus-wallpaper
```
