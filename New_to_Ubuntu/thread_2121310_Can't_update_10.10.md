### Post by ptn107 on 2013-03-01
You need to make a change to your software sources.  When an Ubuntu release hits end-of-life the repository for that release is moved from the main site to another, which makes future upgrading a hassle if you stick around on an end-of-life release.

Pop open your sources list (from a terminal type...):
```
sudo gedit /etc/apt/sources.list
```

Replace all instances of
*[http://us.archive.ubuntu.com/ubuntu/](http://us.archive.ubuntu.com/ubuntu/)*

with
*[http://old-releases.ubuntu.com/ubuntu/](http://old-releases.ubuntu.com/ubuntu/)*

and rerun
```
sudo apt-get update
```

You can then upgrade to 11.04 (natty, also end-of-life), then to 11.10 (oneiric, support ends in apr 2013).  So once your on 11.10 change your software sources back to *[http://us.archive.ubuntu.com/ubuntu/](http://us.archive.ubuntu.com/ubuntu/)* because that release is still supported.

From here I would upgrade to 12.04 LTS if I were you.

Update: I second GreatDanton's post.  If you backup your stuff and do a clean install of 12.04.2 LTS or 12.10 things will go much faster and smoother.
