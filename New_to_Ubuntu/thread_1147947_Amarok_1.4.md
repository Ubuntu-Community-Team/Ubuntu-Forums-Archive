### Post by sandyd on 2009-05-03
Add the correct resipository( for your version of ubuntu)

[http://apt.pearsoncomputing.net/](http://apt.pearsoncomputing.net/)

then
```

sudo apt-get update&&sudo apt-get install amarok-kde3
```

If you want kde 3 too
```

sudo apt-get install kubuntu-desktop-kde3
```
