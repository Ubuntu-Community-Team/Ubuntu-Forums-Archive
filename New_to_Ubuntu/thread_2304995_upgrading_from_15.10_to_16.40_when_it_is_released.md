### Post by scoutchorton on 2015-12-01
Ubuntu comes with an update manager (go to Dash and type Software Updater). When 16.04 (16.04 represents the release date, April 2016 BTW) comes out, Software Updater should run and give you the option to upgrade to 16.04. All you need is some Ubuntu, internet, and time. That's about it.
