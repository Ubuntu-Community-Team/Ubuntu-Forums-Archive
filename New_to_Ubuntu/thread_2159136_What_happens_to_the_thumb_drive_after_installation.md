### Post by GregA on 2013-07-02
It might be a good idea to keep the USB flash stick with Ubuntu on it as it could be a "rescue device" if something were to go wrong in your new install.   

IF the usb is large enough, I would suggest creating a data partition (for storing data) on it using gparted.  If using with win and linux, format it ntfs or fat32, if using Linux only, format the partition with ext4 or similar linux file system.

### Post by grahammechanical on 2013-07-02
To answer your questions. You can remove applications just like we can with a hard disk install. You can send data to the rubbish bin and then empty the rubbish bin. You add data until the USB disk is full, just like with a hard disk and You can format it and store data on it just like a hard disk.

It is a memory stick and you can do what you want with it.
