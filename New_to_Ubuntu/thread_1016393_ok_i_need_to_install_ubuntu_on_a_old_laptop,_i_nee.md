### Post by logos34 on 2008-12-19
I'd try unetbootin:

[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)
(hard disk or usb)
[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)

If that doesn't work for some reason try the **netboot** approach:

[https://help.ubuntu.com/community/Installation/FromWindows](https://help.ubuntu.com/community/Installation/FromWindows)
(-->'Windows NT/2000/XP (using Grub)')  *but you'll need the **alternate-install** iso

You can even install it on the usb flash drive:

[http://www.pendrivelinux.com/2008/04/14/ubuntu-804-usb-hard-drive-install/](http://www.pendrivelinux.com/2008/04/14/ubuntu-804-usb-hard-drive-install/)
