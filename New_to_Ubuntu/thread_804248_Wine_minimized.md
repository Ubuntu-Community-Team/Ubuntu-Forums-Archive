### Post by Bodsda on 2008-05-23
I dont have the answer but i know some people that probably do.

Go to -- [http://forum.winehq.org/viewforum.php?f=2]("http://forum.winehq.org/viewforum.php?f=2")

Thats the official wine forum. Make a new thread their, although you may have to register.

or install an irc program such as xchat

```
 sudo apt-get install xchat 
```

and connect to the 'Ubuntu Servers' (irc.freenode.net) and join 

#winehq

to join a channel with xchat, once you have selected the appropriate server and its logged you on type

```
 /j #winehq <press-enter> 
```

(dont actually type <press-enter>)

anyway, hope it helps ;~)
