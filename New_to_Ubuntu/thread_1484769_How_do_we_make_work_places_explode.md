### Post by nhasian on 2010-05-16
you need to enable compiz by going to System-> Preferences-> Appearance and changing the visual effects from None to Normal or Extra.  Also you must install the compizconfig settings manager

```
sudo apt-get install compizconfig-settings-manager

```
You may also need to install the extra plugins:

```
sudo apt-get install compiz-fusion-plugins-extra
```

then you will be able to customer the effects of minimizing, maximizing, closing windows and a whole ton of other stuff.  it will take you hours to figure it all out :lolflag:
