### Post by Paqman on 2008-11-02
The CD doesn't have all the available applications, just the core ones. If you have an active internet connection try enabling the Universe and Multiverse repos (in System > Admin > Software sources) and refresh Synaptic.

If you need Universe and Multiverse on a disk, you'll need to download the [LiveDVD](http://cdimage.ubuntu.com/dvd/current/).
