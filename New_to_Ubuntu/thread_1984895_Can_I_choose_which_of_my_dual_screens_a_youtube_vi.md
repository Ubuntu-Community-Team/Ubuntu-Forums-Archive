### Post by Houmie on 2012-05-23
I found the solution. Install Chrome Extension [Window Expander for Youtube]("https://chrome.google.com/webstore/search/window%20expander%20for%20youtube?_ac=0"). 

It fixed the original problem and also added a full new feature. :)

### Post by Houmie on 2012-05-23
Unfortunately this solution works only with Youtube.

Hence I pocked further around and found the real solution.

Install Synaptic Package Manager

Open it and mark Chrome (not to be mixed up with Chromium) for a **reinstallation**. Then **Apply**.

 Now it works as expected. Must have been some bad settings or so.
