### Post by ad_267 on 2009-06-13
You can open a terminal from Applications - Accessories - Terminal. Then run "mysql -u root -p" and enter the root MySQL password to begin entering MySQL commands.

You can also install something like phpMyAdmin to work with a MySQL database.
