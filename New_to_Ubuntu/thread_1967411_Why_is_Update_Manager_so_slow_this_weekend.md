### Post by 3rdalbum on 2012-04-28
You could try choosing a different mirror. Rather than the main server for the UK, maybe one of the smaller UK mirrors dotted around the place might work better. The Software Sources program has a small list of them you could try. Even better, check if your ISP has their own mirror for customers - mine does.

The servers are likely slow because of the recent 12.04 update, but things should start getting back to normal within a few days I expect.
