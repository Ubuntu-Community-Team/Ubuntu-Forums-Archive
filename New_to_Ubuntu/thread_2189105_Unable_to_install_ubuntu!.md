### Post by ccmiller12 on 2013-11-20
Perhaps the .iso file is corrupted. Try re-downloading the .iso file, and then burn onto your CD again (make sure to format CD first), then boot from Live CD again, and go from there.

Follow these instructions to install: [http://www.ubuntu.com/download/desktop/install-desktop-latest](http://www.ubuntu.com/download/desktop/install-desktop-latest)

You can use checksums to know wheather your downloaded .iso is corrupted. You go to the site from which you downloaded the iso there will be a checksum given. Now calculate the same checksum for your .iso. Now if they are same then your download is not corrupted. The problem will be mostly with your distribution or tool which you used to make bootable thumb.
