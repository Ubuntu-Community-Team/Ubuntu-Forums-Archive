### Post by grahammechanical on 2016-03-12
The thing I like about this Ubuntu users forum is that we do not mind people asking the same questions that other people have already asked. In my case it means that I already know the answer. :)

It is possible to browse the forum and get an education but it is also true that we may have similar but slightly different problems. And so individual attention is important.

There is a family of Ubuntu Linux distributions. All members of the family will run fine on later hardware. Some family members do not run so fine on older hardware. Whereas, other family members seem to be tailor made for older hardware. So, knowing the hardware specification is important. And do not forget to include the specs of the video adapter. A lot depends on the power of the Graphical Processing Unit (GPU) and the amount of its own memory that it has.

If Ubuntu is the only OS to go on the machine then installation is very simple and there is not much preparation that we need to do. But if we wish to dual boot with another OS, then it matters what that OS is and if the machine has a BIOS or UEFI boot system. Advice can only be based upon the information provided.

LTS = Long Term Support = 5 years of security fixes counting from when the Ubuntu version was released. Ubuntu versions are released every six months at the end of April and the end of October and are numbered accordingly. Ubuntu 14.04 = released April 2014.

LTS versions are released every 2 years. So, the next Ubuntu LTS will be released end of April 2016. Those who are using Ubuntu 14.04 will get a direct upgrade path to Ubuntu 16.04.

In between the 2 yearly LTS releases there are 3 interim releases that come out at six monthly intervals. They only have 9 months support. So, Ubuntu 14.10 & Ubuntu 15.04 have already reached end of life. And Ubuntu 15.10 will reach end of life July 2016. There is a direct upgrade path from 15.10 to 16.04.

Welcome to the Ubuntu Forum!!! It's always good see new people interested in Linux and Ubuntu.

The version you provided is the current LTS, which is a great option for a permanent setup. Here are my thoughts about what I think you should know in terms of Ubuntu options:

Latest Ubuntu release - 15.10: Good because it has the latest Ubuntu experience and hardware support for the installation. It has the latest features Ubuntu wants to bring to the table, both in server and desktop, so it's great for testing Ubuntu. The bad: support is for 9 months, since the focus here is in releasing features.

LTS versions - 14.04: Good because of support, it is the current LTS (Long Term support) version and will be supported up to 2019. It's great for setups that will stay like that you don't want to change. The bad: It might not work for hardware released after 2014, but linux usually have generic drivers that works well.

Note: The next LTS release, 16.04, goes out next month.

[Other variations]("http://www.ubuntu.com/download/ubuntu-flavours"): Ubuntu has some specific target releases, they package all you need to have a full functioning environment of something. Instead of you having to bring the pieces together installing and uninstalling software, these projects bring it already working. So, if you want to try XFCE or to have an Ubuntu installation with it, the Xubuntu project it's the way to go for it.

Installing process: I recommend you to try a live session before installing, to see if your hardware works well. For the media, I recommend you to [Install from USB]("https://help.ubuntu.com/community/Installation/FromUSBStick") instead of [Burning isos]("https://help.ubuntu.com/community/Installation"), but if you feel the need to do so, the option is there.


Hope to have helped. If you have any other objectives for Linux or Ubuntu, let us know. 

Peace.
