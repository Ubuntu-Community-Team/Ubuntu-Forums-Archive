### Post by monkeybrain2012 on 2013-05-10
Open the terminal and type
```
sudo apt-get install lubuntu-restricted-extras
```

Or you can install lubuntu-restricted-extras from synaptic

Then add the medibuntu ppa and install w64(or w32) codecs 

see instructions here to add the medibuntu ppa. [https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)
