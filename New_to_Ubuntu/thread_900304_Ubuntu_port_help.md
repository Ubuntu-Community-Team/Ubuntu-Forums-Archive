if a program (like a torrent client) tries to open a port of ubuntu, it will be allowed. there is no need to 'open' a port as in windows. are you using the transmission torrent client? if so, i think you have to select the port you want to use in one of the settings menus.

EDIT:
if after selecting the port, the program is still not getting through, your problem is not ubuntu. your ports are being filtered elsewhere (maybe at the router, or even by your ISP)
