### Post by unutbu on 2008-06-26
Press 'b' to select the bezier path tool.
Click three points on the canvas. To close the path and thus create a triangle,
while holding down the Ctrl key, click the first point again. Thus it takes 4 clicks to make a triangle.

[[IMG]http://ubuntuforums.org/attachment.php?attachmentid=75443&stc=1&thumb=1&d=1214533214[/IMG]]("http://ubuntuforums.org/attachment.php?attachmentid=75443&stc=1&d=1214533217")

Select the color you wish the triangle to be using the usual color selector.

Go to the menu bar and select Edit>Stroke Path. A new dialog box will pop up.

[[IMG]http://ubuntuforums.org/attachment.php?attachmentid=75444&stc=1&thumb=1&d=1214533214[/IMG]]("http://ubuntuforums.org/attachment.php?attachmentid=75444&stc=1&d=1214533217")

 You can change the line style and line thickness here. You can also draw ("stroke") the path using the paint brush if you wish. This will allow you to use a variety of brushes, and all the toolbox options that the paint brush allows. 

Press Enter when you have made your choices.
This is the result with default choices:

[[IMG]http://ubuntuforums.org/attachment.php?attachmentid=75445&stc=1&thumb=1&d=1214533214[/IMG]]("http://ubuntuforums.org/attachment.php?attachmentid=75445&stc=1&d=1214533217")

Press 'r' to select the rectangle tool and make the green highlighting go away.
[[IMG]http://ubuntuforums.org/attachment.php?attachmentid=75446&stc=1&thumb=1&d=1214533214[/IMG]]("http://ubuntuforums.org/attachment.php?attachmentid=75446&stc=1&d=1214533217")
