### Post by fantab on 2013-02-07
First of all, create a Ubuntu CD or USB and boot from it. When appropriate screen is presented choose "Try Ubuntu"... You will get a fully functional Ubuntu to test without installing anything. If you are satisfied with it, meaning check whether all your hardware is recognized and working, see if your internet is working etc...

Then you can (if you so choose) remove Windows and install Ubuntu. There are many guides on how to do this, just google. And Don't forget to BACK UP any important DATA that you may have saved in Windows. You cannot UNINSTALL Windows, you will just have to format the Windows Partition(s) clean. thats all.

If you have any more doubts, feel free to ask.


Good Luck...

### Post by Mark Phelps on 2013-02-07
IF, when you're done testing Ubuntu running from the disk, you want to replace Vista with it, all you have to do is reboot from the disk and select the option to use the entire drive.

But ... that will completely erase all of Vista on your PC, including any files, settings, and data.

You're not uninstalling Vista, as much as you are removing it.
