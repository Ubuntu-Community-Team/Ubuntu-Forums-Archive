### Post by Legendary_Bibo on 2010-05-07
This is just to get the transparent bar, not the blur effect. I don't know how to do that but I was messing around in the Gimp and decided to make new panel bars while learning photo editing.
Personally I don't use it, I still use the bars I made from my first attempt at making blue glossy menu bars that are semi transparent that were supposed to have cloud streams but I got too trigger happy with the blur tool, but I still prefer them.

Step 1. Open The Gimp.

If you don't have it you can do this

Applications -> Ubuntu Software Center -> Graphics -> The Gimp -> Install

Step 2. Start pleasing the gimp by starting him...

Applications -> Graphics -> GIMP

Step 3. Open a new image and give it the horizontal resolution of your screen and the vertical set to whatever your panel resolution is.

Step 5. On the right hand side (assuming you didn't close the side menus that open with gimp) you'll see a slider bar with the word 'Opacity' next to it. Slide it to '10' or you can type it in.

Step 6. Underneath the box that shows your 'layers' there's an icon with a green plus sign on it. Click it to add a new layer. Set the horizontal resolution to what you had in the original, and set the vertical resolution to 2.

Step 7.1. Click on that same button again to add a new layer but make the vertical resolution 1. Now go to the Toolbar menu that should be on your left and find the icon with the paint can. Set the color to Black and paint that layer in. Now save it with a .png extension

Step 7.2. Now select the whole image and go under Image -> Transform -> Rotate 180 degrees. Save with a different name (This will be your Top bar)

Step 8. Now right click on your panels, select properties -> background -> click on the background image button then click on the folder icon next to it and find your images and Voila! You've got Windows 7 looking panel bars.

Now I could've just posted them, but by having the actual image file you can change the color however you want.
