### Post by Lolpanda on 2011-04-28
> 
1.What is Unity actually? Where does it refer to? The Desktop remains GNOME (as I see ~/.gnome2 dirs and some other commands containing 'gnome') and the distribution is called Natty, so what is Unity?


Gnome really is just a set of libraries and a toolkit. Gnome-Shell (Gnome 3) is just a GUI that lays ontop of that toolkit and library, which Unity is the same. Its just another face to Gnome.


> 2.What is 'Zeitgeist Datahub' (I show it in the start-up applications and I don't know what is does)

Zeitgeist if I remember correctly is the new Indexing / Tracking system for Gnome and KDE. It remembers when you access certain files so that a user can (eventually) do more accurate and specific searches.


> 3.How do I minimize an application? Only from the minimize button? When it is open and I click it again from the (brand-new-cool-amazing) bar on my left it does nothing but focus on it (even if it is already focused) and it doesn't minimize.

I'm willing to give them the benefit of a doubt only because this is essentially Unity 1.0, which for any software -- the 1.0 launch is always tricky.

> 10.Desktop Cube/Wobbly Windows? These two were told to be the most beautiful features of ubuntu, and now they are missing/i cannot enable them.  Or can I?

search synaptic for "ccsm" you're looking for either "simple-ccsm" or "Compiz Control Screen Manager" or something like that. simple-ccsm should install the other as a dependency, through that you should be to enable Cube / Wobbly Windows unless Unity doesn't allow them. 

OR... since Compiz's ONLY use right now is essentially for Unity, Ubuntu may have told the Compiz devs to take those effects out. 

Gnome shell uses Clutter, (or is it mutter?) and doesn't allow anything else. 

KDE uses Kwin (a fork of compiz or atleast a different branch of it) 

XCFE can still use compiz if you tell it to. 

LXDE I don't THINK supports Compiz. 

And E17 has their own compositor -- which by the way is amazing.
