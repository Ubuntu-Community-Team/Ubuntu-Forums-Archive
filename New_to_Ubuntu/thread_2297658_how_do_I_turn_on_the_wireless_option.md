### Post by grahammechanical on 2015-10-05
Do you have a wireless/WiFi adapter? Is it switched on? If this machine is a laptop then the wireless adapter may be switched to save battery power. You may need to activate a combination of keys to turn the adapter on. If you have Windows installed on that machine and wireless is switched off in Windows to save battery power then it will not be switched on when Ubuntu is loaded.

You may need a driver to work the wireless adapter. The Ubuntu live session may not have a driver for your wireless adapter. Or then again it may have but if the wireless adapter is switched then the live session will not detect a wireless adapter to load a wireless driver module for.

With a working wireless adapter we click the Network Manager icon and in the menu should be a list of wireless access points in range. Select the one we want to use and a dialog will appear that will allow us to enter the WiFi passphrase for that access point.
