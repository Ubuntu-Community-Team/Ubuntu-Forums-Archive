### Post by rtimai on 2010-06-19
On the Ubuntu desktop toolbar click on System, Help & Support, which takes you to the Ubuntu Help Center over your default web browser. This is an online user manual, not a lot of detail, but it will get you started with brief descriptions of what you can do with the applications that are installed by default. 

Don't be afraid to explore your Applications menu!

After you install the restricted extras you should look at this post it is a very detailed walk through with pictures.It will show you the basics as well as some more advanced eye candy and games.
