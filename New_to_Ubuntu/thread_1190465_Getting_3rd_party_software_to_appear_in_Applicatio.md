I just installed an open source statistical computing package called R on my jaunty box.  I used the Synaptic Package Manager to do this. ### Post by Hobgoblin on 2009-06-17
System > Preferences > Main menu

select the submenu where you want the app.

Click New Item

Type a name for it

next to command enter the command you would use at the command line (R)

Click the icon if you want to change it.

Click OK

(you may also need to change Type to Application in terminal)
