### Post by Dennis N on 2015-08-16
The settings dialog you refer to did not appear in a Xubuntu release until 15.04. That's why you don't find it in Synaptic Package Manager in 14.04.

You may have to edit the configuration file instead to make changes: **/etc/lightdm/lightdm-gtk-greeter.conf**
