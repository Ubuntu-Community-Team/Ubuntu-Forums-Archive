### Post by tom.swartz07 on 2009-12-29
Hi,

Sorry to hear that you're dumping Ubuntu.

Unfortunately, Its rather hard to get Windows on a netbook unless you have a CD drive.

If you do, then its just a matter of popping in the Windows disk and away you go.

If you dont, youre limited to USB install. Windows doesnt play very nice with USB installers, especially if you dont have a Windows environment to create a bootable drive.


HOWEVER: there is a Windows 7 LiveUSB maker (it might work with other versions, but ive never tried it) You could download it from here: [http://wudt.codeplex.com/](http://wudt.codeplex.com/)
Granted you have a working .iso file for Windows, and a Windows computer, it works without a hitch.

If you want to restore your Ubuntu, simply create a LiveUSB using a guide on [www.pendrivelinux.com](www.pendrivelinux.com) and install it that way.


Hope this helps!
