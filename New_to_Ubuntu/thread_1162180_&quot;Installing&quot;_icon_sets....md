### Post by 67GTA on 2009-05-17
If you download an icon package from gnome-look.org or similar site, just open the "Appearances" window and drag/drop the package onto the window. There is a bug that causes the appearances window to crash sometimes with big icon packages. If this happens, just extract the package into /home/username/.icons and then reopen appearances and select them.
