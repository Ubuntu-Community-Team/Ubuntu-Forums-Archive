### Post by sikander3786 on 2010-09-30
No need to install antivirus/firewall inside Ubuntu. Viruses will never damage Ubuntu and Ubuntu will never let them damage your Windows install. If your Windows install is protected by an antivirus, you are pretty safe.

Ubuntu comes with an uncomplicated firewall, UFW. See here.

[https://help.ubuntu.com/community/UFW](https://help.ubuntu.com/community/UFW)
