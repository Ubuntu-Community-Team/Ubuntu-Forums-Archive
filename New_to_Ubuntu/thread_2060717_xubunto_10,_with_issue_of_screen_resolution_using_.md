### Post by Toz on 2012-09-20
Hello and welcome to the forums.

Can you provide more information about your system? Make/model?

Also, open a terminal window, type in the following commands and post back the results:
```
cat /proc/cmdline
sudo lspci -vnn | grep -A10 VGA
```

And finally, can you post back the contents of the following files:
- /var/log/Xorg.0.log
- /var/log/dmesg

The /etc/X11/xorg.conf file has been deprecated and isn't normally needed (although you can create one if needed). Most of the necessary info can be found at /usr/share/X11/xorg.conf.d. More information located [here]("https://wiki.ubuntu.com/X/Config").
