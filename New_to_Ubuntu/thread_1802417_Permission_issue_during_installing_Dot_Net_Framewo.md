### Post by Mark Phelps on 2011-07-12
You're wasting your time.  None of the .net versions newer than 2006 will work in Wine.

BEFORE you install anything in Wine, you need to go to the WineHS site and look up the app and version -- to see its rating.

The following is the page for .net framework and, as you can see, the recent versions are rated Garbage -- meaning, they will NOT install or work, no matter what you do:

[http://appdb.winehq.org/objectManager.php?sClass=application&iId=2586]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=2586")
