> 
Do you know whether there is the same kind of guide but with more details and practical lesson?

The Ubuntu Server Guide covers the absolute basics and the details in which the configuration of the various servers on Ubuntu differs from the standard. For details that go beyond this guide, the documentation of the programs on their various home pages is probably the authoritative source, for example for apache [http://httpd.apache.org/docs/](http://httpd.apache.org/docs/) or [http://www.git-scm.com/doc](http://www.git-scm.com/doc) for git (both for client and server).
