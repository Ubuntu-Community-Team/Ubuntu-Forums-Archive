### Post by nothingspecial on 2009-11-02
Open system > administration > software sources and add these 2 lines -```


deb http://packages.medibuntu.org/ karmic free non-free
deb-src http://packages.medibuntu.org/ karmic free non-free
```

Then type ```
wget -q http://packages.medibuntu.org/medibuntu-key.gpg -O- | sudo apt-key add - && sudo apt-get update
```

sudo apt-get install sun-java6-fonts sun-java6-jre sun-java6-plugin
