### Post by sepo on 2012-12-21
In short: when you install Ubuntu it will partition your HD (to give itself some space) and it will install a boot loader (to allow you to choose which operating system to launch at startup).

You'll never partition your RAM (?); only one Operating System will run at a time (Win or Ubuntu) and it will use the whole RAM.
