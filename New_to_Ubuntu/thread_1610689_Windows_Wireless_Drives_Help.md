### Post by garvinrick4 on 2010-11-01
All you should have to do is set up a Windows Workgroup in Vista.
With a login password in Windows and share whatever you want.
Will show up in Places to Network to Windows Network to Windows Workgroup
and all that is shared will be accessable. Every install I do it shows up the same
way with no additional installations of packages needed. I assume you have a 
home network running a router.
