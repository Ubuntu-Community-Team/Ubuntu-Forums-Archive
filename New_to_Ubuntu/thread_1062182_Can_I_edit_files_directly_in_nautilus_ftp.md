### Post by blueridgedog on 2009-02-06
I don't think the secure ftp mechanism supports editing.  You may be able to ssh directly to the server, getting access to your home directory and then edit the files via command line tools.

### Post by bgates on 2009-02-06
I think you can do it in the gui too.  Places > connect to server > choose ssh as the service type.  Then connect and it should be the file browser where you can open files and edit and then save them.
