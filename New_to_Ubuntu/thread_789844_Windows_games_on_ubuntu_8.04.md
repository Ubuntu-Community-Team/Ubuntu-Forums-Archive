### Post by warbread on 2008-05-10
Being a Microsoft published game, I wouldn't be surprised if Halo didn't work at all.  The [WineAppDB]("http://appdb.winehq.org/") has information on what games work on what distributions and versions of Wine.  If you really care about your gaming, though, I suggest dual booting Windows and Ubuntu.  Wine is a pain and can really screw up your OS if something goes wrong (I used to reliably be able to bork my whole Linux partition by trying to install Diablo II: LoD - a game that is supposed to work!).  Trying to run a Windows game in Linux doesn't make any more sense to me than someone trying to run a PS2 game on an Xbox.

Try all you want, though.  I hear tell that forcing the Wine version to .46 solves a lot of issues regarding frames per second in Steam games.  Maybe that'll help you, too?
