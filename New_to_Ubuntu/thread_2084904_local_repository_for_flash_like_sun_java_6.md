### Post by QIII on 2012-11-16
If your brother is addicted to a game that requires Java 6, I highly recommend breaking the addiction.

Sun (now Oracle) Java 6 has many security vulnerabilities and reaches End of Life this month.  So does OpenJDK 6.  (Edit:  Looks like EOL was extended to February 2013)

I recommend you try to install OpenJDK 7 from the repository to see if it will work.  If not, try installing Oracle Java 7 using the wiki link in my signature under "Oracle Java 7", "Command line methods", "Using webupd8.org's strikingly simple method."
