### Post by x33a on 2009-02-22
in a terminal type:
```
gksudo gedit /etc/modprobe.d/alsa-base.save
```

put a # sign before the first line, this one:
> "options snd-intel8x0 ac97_quirk=3"# autoloader aliases

save the file. and see if you get the error on next boot.
