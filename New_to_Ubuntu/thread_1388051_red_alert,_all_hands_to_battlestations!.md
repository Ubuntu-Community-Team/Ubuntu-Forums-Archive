### Post by lykwydchykyn on 2010-01-22
> **jahangir99 said:**
> but canone explain to be the basic structure of beyrl coz im confused to wat it is as i am a noob 
:)

beryl ... which is now compiz... is a **Window Manager**, which is the piece of software that enables you to, well, manage windows on the screen.  Without a window manager you couldn't move, resize, minimize, or maximize windows, among other things.

Compiz is a **Compositing Window Manager** which means, to gloss over the gory details, that it can do window management with a lot of snazzy visual effects.

If your video hardware is up to the task, Ubuntu enables compiz by default.  The settings are quite conservative by default, though, which is why you want to install the settings manager package I suggested before.  This allows you to go in and enable optional effects and tweak all the settings to your heart's content.
