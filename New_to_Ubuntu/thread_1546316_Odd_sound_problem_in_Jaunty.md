### Post by MaxVK on 2010-08-05
Hi. My son (SonOfMaxVK!) had a **Creative Live!** soundcard running under Jaunty and Windows. It was old and a bit flaky and last night while plugging in the Mic in Windows the machine screamed for a bit and then rebooted itself.

But it didn't start. Didn't even POST.

We removed the sound-card and we could boot into Ubuntu straight away, so... the sound-card had died.

We had a spare card - this time a **Creative Live! Drive** that came from my old machine and is known to work.

Problem is when we started with the card in place both Ubuntu and Windows start okay, but there is no sound.

Iv had a hunt around the forums and followed a few things, but nothing has worked - we are yet to get a peep out of the card.

The card is found correctly by the system (Ubuntu) and the driver is an **EMU10k1-Audigy**, but thats as much as we know. Iv tried removing the entire sound system and reinstalling it (From a troubleshooting thread on here) but that didn't work.

Everything appears to be fine but there is no sound and if we play a video it will play without sound for a minute or so and then the whole system locks up, keyboard included, and we have to hit the power button.

We have the same problem in Windows, where everything appears to be fine but there is just no sound. We've not gone far enough to see if it locks up.

Does anyone have any ideas? I would have expected the two cards to use the same drivers but from what Iv been reading this might not be the case. Iv looked around for the EMU10k1_Audigy driver, but Iv not actually been able to find much that seems helpful.

Any help would be much appreciated.

Cheers

Max

### Post by MaxVK on 2010-08-06
A conclusion of sorts:

In the end nothing would make the sound card work so we swapped out the motherboard and used an on-board sound card, which works fine.

Incidentally the sound card the would not work has been put back into the old machine and is running on Gutsy quite happily after its ordeal!
