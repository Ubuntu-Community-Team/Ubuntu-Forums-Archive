### Post by Jayce71 on 2011-09-14
I think I got it after multiple trials...
 
The "master password", the keyring, is always stored. I wasn't asking how to create or change it, nor what it is used for. But what is the function of the Lock / Unlock option in the right-click menu...
 
My understanding:
The Lock / Unlock function of the right-click menu allows to change the status of the keyring for the logged in user.
If an auto login was performed, the keyring was not unlocked. It can be unlocked using this right-click menu.
If a manual login was performed, the keyring was unlocked and this menu option allows to re-lock it if necessary for whatever reason.
