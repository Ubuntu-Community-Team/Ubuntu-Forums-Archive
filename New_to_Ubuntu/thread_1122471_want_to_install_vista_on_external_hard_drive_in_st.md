### Post by keplerspeed on 2009-04-11
You want to dual boot? In this case you can install vista either on a partition on your internal HDD or on the usb HDD. Just make sure your mobo can boot from a usb HDD, (may need to enable this in BIOS) and boot up the live cd installer and install on the USB HDD!

### Post by Nixie Pixel on 2009-04-11
After doing this you may need help in setting up GRUB to correctly handle the dual-boot.  If you install Windows after you install Linux, that is.  If you do it Windows first, Linux second, GRUB will handle everything for you.
