### Post by Wobblybob on 2010-05-11
it's not a directory mate it's an action you are configuring software, you will have downloaded the tar package and then extracted it's contents to a folder say temp in the path /home/yourname/temp you then need to open a Terminal and cd to that folder with cd /home/yourname/temp then still in the Terminal type ./configure and hit enter, you may need to use sudo ./configure

edit looking at the link the command you need to type is ./configure --enable-dynamic-minors
