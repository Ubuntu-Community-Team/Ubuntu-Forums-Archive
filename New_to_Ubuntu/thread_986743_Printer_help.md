I would start with HPLIP, which is HP's own project to create drivers for their printers.

[http://hplipopensource.com/hplip-web/models/laserjet/hp_laserjet_p1005.html](http://hplipopensource.com/hplip-web/models/laserjet/hp_laserjet_p1005.html)

It is already install on Ubuntu (I think), so you just need to set it up:

[http://hplipopensource.com/hplip-web/install/manual/hp_setup.html](http://hplipopensource.com/hplip-web/install/manual/hp_setup.html)

Good luck!
