---

### Post by TeoBigusGeekus on 2011-10-16
Of course it's possible.
Go to Places, find the disk on which windows is installed and click it.
Once open, go to Users>YourWindowsName>Documents.

### Post by benjetta96 on 2011-10-16
Which OS of Windows are you using? If XP you should be able to go into the Windows partition of your HD and go through:

Documents and Settings/[Your Username]/My Documents

### Post by d4m1r on 2011-10-16
Open your Home folder and in the top part there will be an icon of a computer that is equal to "My Computer" in Windows where all your HDDs should show up. You can view/copy/paste whaever files you want from there including Windows ones.
