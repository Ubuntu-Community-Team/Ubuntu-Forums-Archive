### Post by Jguy on 2009-11-06
Stick the CD you receive into the machine and reboot windows. Ensure that the CD Drive is set to first boot, and then choose the option 'try Ubuntu without making changes' and Ubuntu will simply run from the CD ROM Drive. You can then take it for a test drive.

Keep in mind that if it seems to run slow, that is because you're using it from a 56X or so CD Rom Drive, the OS can only load into memory equally as fast as your CD Drive can transfer that data to memory.
