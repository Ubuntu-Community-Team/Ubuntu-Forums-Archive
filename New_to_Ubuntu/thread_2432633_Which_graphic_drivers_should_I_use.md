### Post by meiselstein on 2019-12-05
Hello everyone,

I have finally found a computer that I can use for Ubuntu: everything has installed correctly and there are no strange streaks like previous attempts (my new computer is an Asus Vivobook 15). 

One question I have is I don't know which graphics drivers I should be using for general use. My computer has two graphics cards (Intel UHD and nVidia GeForce MX250). My options are the official nvidia drivers or the open source drivers. My gut tells me that using the official nVidia drivers is ideal, but the default setting is the open source driver.

Thanks in advance for your help.

### Post by Autodave on 2019-12-05
Do NOT use the drivers from nVidia's site!!!!  Use the one that is recommended under Additional Drivers in your setting menu.  You may have to disable the Intel card to let the nVidia one work.  This is sometimes possible to do in the BIOS.

---

### Post by SeijiSensei on 2019-12-05
If you want to use only the NVIDIA adapter, then disable it either in BIOS or via whatever other method your manufacturer offers. (My Lenovo laptop has a physical switch.) Then use the Driver Manager in System Settings to install the NVIDIA driver.
