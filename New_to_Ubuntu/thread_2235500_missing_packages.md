### Post by Mark Phelps on 2014-07-21
> How can I find out which packages are missing? 

Unfortunately, without advanced preparations (as in, creating a list of installed packages, saving that off, and browsing that to see what you had installed previously), there is no way to determing what did NOT get installed other than by looking through what you have now and discovering what is not there. Each new Ubuntu release installs a default small set of packages; any you installed beyond that, you will have to reinstall yourself.
