### Post by spiderbatdad on 2010-02-23
Quite easily. Go to System>>Preferences>>Main Menu. In the left pane select programming, then on the right click new item. A dialog will open. Name the application and in the command section use the program name that launches the application, probably komodo.

If you want to change the icon...use Main Menu again and select the item...then click properties. When the dialog opens, click the existing icon in the upper left corner of the properties window...icon file will open, or use browse button to locate other icon.
