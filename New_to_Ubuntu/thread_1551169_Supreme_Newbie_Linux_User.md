> When I partition my hard drive will 10 gigs be enough because i still would like to run xpNot really; unless you plan on not using Ubuntu to save data etc., you need at least 15-20GB. The default install with all programs is about 3GB, but if you save photos, music etc. you could quickly run out of space with only 10GB.

### Post by psyberpnk on 2010-08-12
> **Zrgaloin said:**
> Hi
When I partition my hard drive will 10 gigs be enough because i still would like to run xp


You could try xubuntu.  I am using a 7-year-old HP pavilion laptop with a 20gb hdd split between XP (15 gig) and xubuntu 5 (gig).  With only 384 MB ram it takes a while to boot but it works for playing around on the internet, word processing, and tinkering w/ shell scripts.

XP also runs slow, but I can still do things like write code using netbeans or visual studio (both are resource hogs) and test webpages running XAMPP lite.

Just need to be patient and allocate resources where they will be needed the most.

Also, you can uninstall many of the pre-installed xubuntu applications (e.g. gimp, games, etc) to slim down the system.

Hope this helps
