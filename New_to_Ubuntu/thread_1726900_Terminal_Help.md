### Post by mads65 on 2011-04-11
View Directories: **ls** (lists files in a directory)
Create Directories: **mkdir** (create a directory you specify)
Change Directories: **cd** (will change from your current directory to any directory you specify)
Copy Files: **cp** (will copy any files you specify)
Copy Directories: **cp -r** (will copy any directories you specify)
Rename/Move Files/Directories: **mv**
Remove Files: **rm**
Remove Directories: **rm -r**
Find Files/Directories: **locate** (will search your computer for any filename you specify)

These are some to start with. But, the best way as other users have already mentioned, is to use the links you've got in the 2nd post.
