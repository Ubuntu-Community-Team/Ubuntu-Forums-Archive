### Post by Cold-Gin on 2009-05-21
Just wanted to post an FYI that might be helpful to somebody. If you install java 5 via the command line like so:

sudo apt-get install sun-java5-jdk

You will get a confirmation screen during the installation. If you hit enter nothing happens!

[B]
You need to hit the TAB key to highlight the OK option before hitting enter!!!
[/B]

I bailed out of the terminal window the first time and had to rectify the situation by uninstalling through synaptic package manager (the uninstall didn't go well the first time either. I wound up having to reboot). 

Luckily, somebody that I work with had ran into this issue prior to me, so I am passing the info along.
