### Post by OrangeCrate on 2014-03-13
Hi Jody,

We used to be able to just drag and drop programs from Dash to the desktop. As far as I know, right now, we can't. It's a bug, that has not been addressed in 13.10, but, may be addressed in 14.04, which is currently under development. You can read more about this issue here:

[https://bugs.launchpad.net/unity/+bug/1241972](https://bugs.launchpad.net/unity/+bug/1241972)

You don't say which flavor of Ubuntu you're using, so, I'll assume it's the current release, Ubuntu 13.10. As far as what's installed, open Dash and look at the installed applications list (the "A" with the horizontal box through it, next to the house will get you to that list). If you need a description of what a program is, or what it does, a quick search with the browser of your choice, should answer any questions you might have.

Linux is not hard, it's just not intuitive. Which means, as with all things new, there is a learning curve. Here are a couple of resources that will start you on the path of learning your new operating system:

[https://sites.google.com/site/easylinuxtipsproject/Home](https://sites.google.com/site/easylinuxtipsproject/Home)

[http://ubuntuguide.org/wiki/Ubuntu_Saucy](http://ubuntuguide.org/wiki/Ubuntu_Saucy)

Many, many more resources available on the web...

Welcome to the forums.

### Post by grahammechanical on 2014-03-13
If you hold down what we called the Super key, it is the key with the flying windows logo, an overlay will appear on the screen and you can find out about keyboard shorts cuts.So, super+A will open the Dash Applications lens and under See nnn more installed we will see all the icons for all the applications installed. With Unity we do not need to clutter the workspace with icons because we have this fold-out Dash that will reveal all the application icons. We can set the Dash to full screen.

If you are going to put icons on the workspace then perhaps you should go to System Settings>Appearance>Behaviour and tick "Add to show desktop icon on the Launcher." Then you will be quickly able to reveal the desktop from under all the open application windows.

One of the principles that drives the design of the Ubuntu user interface is the principle of as much screen real estate for the user as possible. To put icons of the desktop, even of 14.04, you may need to install Unity Tweak Tool. It is in the Ubuntu Software Centre but I do not know if it will do what you want.

Do not forget that in Linux we have more than one workspace. We can put different applications different workspaces. The default is 4. Again look in System Settings>Appearance>Behaviour to put the workspace switcher icon on the Launcher.

Regards.
