### Post by tmNihonsuki on 2012-12-25
For the wallpapers on ununtu, 

```
sudo apt-get install nitrogen
nitrogen &
```

then set up your background directories in preferences - ubuntu defaults are at,

```
/usr/share/backgrounds
```

follow by choosing either to span both monitors with a single image or set an image for each monitor individually.
