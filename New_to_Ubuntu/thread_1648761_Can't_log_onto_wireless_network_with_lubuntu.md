### Post by vangop on 2010-12-19
Hi,
check the connection preferences->wireless security tab (at least on ubuntu).
The point - check the settings for the security (wep/wpa). I have the same behavior on my local wifi spot. It is an adhoc network and if I select wep 128bit key it will repop me the password window. If I select wep 40/128 key - it works.
Try to play with these settings.
