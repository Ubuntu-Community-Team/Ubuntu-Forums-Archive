### Post by oldfred on 2012-09-21
Gparted is not part of standard install as you cannot edit the partitions you are in. But you can download from command line or synaptic.

sudo apt-get install synaptic
sudo apt-get install gparted

I may use gparted from my install but that is just to review partitions, layout sizes etc. You can use gparted from your install on other drives or USB drives.
