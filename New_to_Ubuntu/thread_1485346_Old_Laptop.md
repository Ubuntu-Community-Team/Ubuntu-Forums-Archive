### Post by snowpine on 2010-05-16
The system requirements for Ubuntu are well-documented: [https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements)

With only 382mb of RAM, I would recommend the lightweight Xubuntu or Lubuntu flavors. Either will have the same core system and repository of applications, but feature a different user interface (Xfce or LXDE respectively) that uses less resources than the Ubuntu default (Gnome).
