### Post by Cliff_Simonds on 2014-08-23
Just a recommendation: try dual booting first until you find linux equivalents for your favorite games & programs(if any), and get used to using them. I personally have left my system dual boot since I started using ubuntu 4 years ago. If you only have the one computer and an os "dies"(like some windows computers after the august 12th updates) you still have internet access.
