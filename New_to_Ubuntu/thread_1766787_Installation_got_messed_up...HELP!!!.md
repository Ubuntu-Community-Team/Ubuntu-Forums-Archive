### Post by Wim Sturkenboom on 2011-05-24
Windows has some tools on the install CD to repair the MBR. That will allow you to get the Windows part going. Things have changed during the last iterations of windows but a search for [windows 7 fix mbr](http://www.google.co.za/#hl=en&source=hp&q=windows+7+fix+mbr&oq=windows+7+fix+mbr&aq=0&aqi=g8g-v2&aql=&gs_sm=e&gs_upl=1507l7985l0l19l16l1l2l3l0l364l2781l2.3.5.3&fp=bf531b198141390a&biw=1104&bih=758) should get you going.
