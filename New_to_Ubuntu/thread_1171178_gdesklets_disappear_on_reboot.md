### Post by allenbradley on 2009-05-27
Is gdesklets on your startup applications list?
Go To
```
System->Preferences->Startup Application
```
Press Add 
Name : <anything>
Command : gdesklets (or) the command you use to launch.
Press OK, close and reboot.

Does that help?
