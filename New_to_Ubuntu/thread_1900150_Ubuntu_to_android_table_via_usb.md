### Post by gandaran on 2011-12-25
> Spent some time looking around here but didn't find anything to help out. Was wondering if anybody knows of a good reference source in setting them up via USB for transferring info, etc, between the two
almost all android devices have an option to mount the SD card as USB storage, some use MTP instead (install **mtp-tools** from software center) and on some you can disable MTP so it can use USB storage mount instead, checkup your tablet specifications on how to connect to computer via USB
I would recommend an app from the android market to connect via wireless like **airdroid**, with airdroid you have full control of the android device on your PC.
