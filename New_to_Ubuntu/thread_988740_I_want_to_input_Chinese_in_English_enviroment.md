### Post by mrboojive on 2008-11-20
You need to go to System > Administration > Language Support. Check the box for Chinese. Then check the option at the bottom of the window called "Enable support to enter complex characters." Click OK and it will download some packages. You'll need to log out and log back in, but after that you'll be able to select an IME by clicking on the SCIM icon in the system tray (the one that looks like a little keyboard).
