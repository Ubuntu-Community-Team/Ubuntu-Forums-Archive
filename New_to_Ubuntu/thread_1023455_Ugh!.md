### Post by teh_yodeler on 2008-12-27
Wow.  If you paid that much for Ubuntu, they should have installed it for you.

Try to get your money back, if you can.  Ubuntu is free to download, you can burn the disc yourself and then boot from the CD and install it.

If you don't know how to download an *iso, burn it to a disc, and boot from it, Ubuntu will probably be a long hard road for you.  Not to say that it wouldn't be worth it, but I would never *sell* someone Ubuntu and send them home to install it themselves.

If you do decide to continue installing Ubuntu, basically you have to figure out how to boot from the CD.  Put the CD in your drive, and restart your computer.  If it doesn't load automatically, you will need to enter your BIOS and change the settings so that the CD drive has top priority.

Once you do get it booted, choose "try Ubuntu without any changes to my computer."  The last thing you want to do is accidentally install it and overwrite EVERYTHING on your windows partition, especially if you don't have your windows installation media.


This place will help you out a lot, for sure.  Everyone is willing to help and there is lots of information available.  It may seem overwhelming for a while but you will get through it.


What kind of computer do you have?  Brand/make/model etc.  Laptop/desktop?  Do you know the hard drive size, how much RAM there is?  Is it 32 bit or 64 bit?

The more we know about your computer, the better we can help you, so don't be afraid to post too much.

Come back here for help, people will answer any questions you have
