Hi Cthulu2156.

When you press 'Extract' a window opens. There, you can select where you want to extract your files. By default it opens on the current directory (where the compressed file is).

If you don't navigate to another directory and press 'Extract' at the bottom left of that screen, the files will be extracted on the same directory that the tar/zip file is.
