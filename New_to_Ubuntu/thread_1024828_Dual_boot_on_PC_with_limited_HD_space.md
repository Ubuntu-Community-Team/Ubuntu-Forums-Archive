### Post by namdung on 2008-12-29
Installing Ubuntu in external drive with dual boot is possible. It is possible to install even on a USB pen drive. But again pls use internal for performance if possible.

Xubuntu is recommended for PCs with lower specs. Also Hardy Heron LTS 8.04.2 is out on 1/1/2009 which is a much stable release than Intrepid.

[https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements) 

But if ur system has enough mojo to support virtualization, consider installing Windows as virtual PC.

### Post by crf on 2008-12-29
Not all computers allow booting from a USB hard drive.

If yours can, there often will be an option in the computer's bios (which you can usually configure by pressing f8 or f10 while the computer starts up).
