### Post by yeats on 2009-12-27
Plug-ins are developed and maintained separately from the main Firefox product, so if they are not working with a beta product, that would not be surprising.  

Have you considered [Ubuntuzilla]("http://sourceforge.net/apps/mediawiki/ubuntuzilla/index.php?title=Main_Page")?  It will keep you on Mozilla's current versions even if the Ubuntu release you're on does not include them.  It's not as cutting edge as beta releases, but plug-ins are more likely to work with it
