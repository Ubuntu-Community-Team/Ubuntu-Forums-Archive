### Post by LiamOS on 2012-12-11
[www.tldp.org](www.tldp.org) - The Linux Documentation Project is a good resource.

Linux from Scratch is great in many ways, but introduces you to a much lower level of linux than you might be prepared for, or even want at this point. :P
A good place to start might be installing Gentoo Linux in a virtualbox. Installing Gentoo is a good learning experience which will teach you a lot of things about what is on your system, how everything interacts, and how to set up things the way you like. You can also ignore the Gentoo recommendation to use their liveCD and use Ubuntu's, making things easier and generally more accessible.
[Arch is similar, but 'easier'; I didn't take to it, but many have.]
If you get some ways with Gentoo and enjoy your experience, Linux from Scratch is almost mandatory to try; LFS will leave you - in a manner of speaking - at the point where you begin on a Gentoo install.

### Post by DuckHook on 2012-12-11
> **MikeLinux99 said:**
> ...do you know how to learn linux from "Linux From Scratch"

If you have not been exposed to Linux before, you might be jumping in at the deep end. This is fine, provided you really understand what you are getting into.

Do you understand the command line, Linux files structures, basic utilities like *ps*, *cat*, *ls*, *cd*, *mv*, *rm*, *mount*, etc? If not, then what you are proposing is a little like trying to high dive before learning how to swim. I may be totally off-base here, but I don't know your level of expertise.

My suggestion for people trying to learn Linux basics is to install a minimal install in a VM first. If you get to the point where you can do everything from the command line (and I mean *everything*), then building a Linux system from source is a fun and manageable adventure. Otherwise, it's biting off more than you can chew. Maybe those are just the thoughts of an old and too-cautious geezer (it *is* kinda refreshing to see enthusiasm and ambition). The minimal install CD can be found [here]("https://help.ubuntu.com/community/Installation/MinimalCD").
