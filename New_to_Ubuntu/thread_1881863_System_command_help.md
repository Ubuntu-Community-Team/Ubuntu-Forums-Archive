### Post by mikechant on 2011-11-16
The instruction to use
System -> Administration -> Samba
would be for the older 'classic' menu interface.

From memory (I'm not at my Ubuntu PC and I mostly use classic/10.4) I think you probably want to open the 'application lens' (the button with the magnifying glass with a + in it). Then start typing 'samba' in the text box and the samba button should appear, click to start it. 

If you want it to be easily accessible in future you can right click on its 'launcher' button when it's running and pin it to the launcher.

Much easier to do than describe! That's one of the disadvantages of the new interface...
