### Post by Mark Phelps on 2009-05-05
If you're going to be trying to run MS Windows apps in Wine, you want to check out the CodeWeavers link below and search for the specific version of each app you want to run.  If it's not shown as supported, or if it gets a Silver or lower rating, you're wasting your time installing Wine:

[http://www.codeweavers.com/compatibility/browse/name]("http://www.codeweavers.com/compatibility/browse/name")

If you've installed Ubuntu only to run MS Windows apps, I can guarantee that the experience will be disappointing.

### Post by albinootje on 2009-05-05
> **coope said:**
> Have just changed from windows to ubuntu. can anyone advise where i can download wine for jaunty

Wine is in the Ubuntu repositories, see here how to use those :
[https://help.ubuntu.com/community/Repositories/Ubuntu](https://help.ubuntu.com/community/Repositories/Ubuntu)
[http://www.psychocats.net/ubuntu/installingsoftware](http://www.psychocats.net/ubuntu/installingsoftware)

You should realise that Wine itself cannot promise you to run Windows applications smoothly and stable all the time.

See here for alternatives for well-known MS-Windows software :
[http://www.osalt.com/](http://www.osalt.com/)

And if you want to run something like MS-Office on Ubuntu, consider buying CrossOverLinux, which is based on Wine.
With buying that you get support for that software, and you will be supporting the development of Wine indirectly.
