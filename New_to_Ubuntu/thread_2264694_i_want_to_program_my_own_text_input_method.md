Sorry, but your choice of words is about as bad as it gets. An "Input Method" is a technical term to user and developers using the X Window system. The suite of programs used to map multiple keys to one Chinese character is an "Input Method". Writing things like that is very technical and complicated. What you are actually looking for is a "keyboard layout", still rather technical but **a lot** simpler .

If I were you I'd first try the various layouts that are available (there probably is way more than one - there's something like 7 or 8 for German, so there should be more than one for Turkish). [This]("http://www.wikihow.com/Change-Keyboard-Layout-in-Ubuntu") article describes how to set them up and switch between them. If none of them are to your liking, you can create your own. A quick Google search for "Ubuntu create keyboard layout" gives several hundred thousands of results.
