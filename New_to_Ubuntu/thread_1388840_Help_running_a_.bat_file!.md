### Post by mikechant on 2010-01-25
Firstly, acording to my googling, hl2.exe is 'half-life', not 'halo'.

Secondly, according to this:
[http://appdb.winehq.org/objectManager.php?sClass=application&iId=2095](http://appdb.winehq.org/objectManager.php?sClass=application&iId=2095)

It looks as if you should be able to get it to run under Linux by using the 'wine' Windows compatability layer.

If you have used Ubuntu much then you'll know what I mean when I say 'install wine from the repositories', if not ask.

Once you've installed wine you may need to perform various steps to get hl2 installed and working.
There appears to be a how-to here:
[http://lhl.linuxgames.com/howto/half-life-HOWTO-0.5.html](http://lhl.linuxgames.com/howto/half-life-HOWTO-0.5.html)
but I can't look at it because games sites are blocked from my work.
I just googled for 'wine half-life' and this was the first result.

It isn't always easy getting windows programs to run under wine and you may need some patience and to ask more questions, but as per the first link above, it looks as if it should be possible.
