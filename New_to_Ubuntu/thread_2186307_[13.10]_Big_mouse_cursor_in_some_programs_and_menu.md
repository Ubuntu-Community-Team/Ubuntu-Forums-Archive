### Post by fantab on 2013-11-06
Open **dconf**, if don't have then you can install 'dconf-editor' from software center.

Navigate yourself to* org-> gnome-> desktop-> interface*
Look for '*cursor-size*', default =**24**, select it and at the bottom click on '*Set to Default*' or you can change it manually and hit enter key to apply.

That should take care of "huge cursor in some programs".

If you want to use the command line then run:
```
gsettings set org.gnome.desktop.interface cursor-size [COLOR=#EE82EE]**24
```**[/COLOR]
