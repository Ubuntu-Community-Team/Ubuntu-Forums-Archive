### Post by Paqman on 2011-07-31
> **Nikhil Kenvetil said:**
> i have reserved anohter 100 GB partition(NTFS) for ubuntu

Ubuntu can't install onto an NTFS partition, that's a Windows filesystem.

What you could do is either:
[LIST=1]
[*]Quit the installer, boot back into Windows and delete that partition. Then run the Ubuntu installer and install it into the free space you just made.
[*]OR use "advanced partitioning", and format tthe partition as Ext4 (mount point / ) and a small swap partion no bigger than the size of your RAM. Be very careful doing this though. If you get the wrong partition you could accidentally overwrite Windows.
[/LIST]

### Post by Rex Bouwense on 2011-07-31
> **Nikhil Kenvetil said:**
> hey..
got a question for you.. so if i go ahead and install 10.04, can i UPGRADE to 11.04 directlty..? or do i have to go to 10.5,.6,.7.. thru all that? or how does that work?

If you install 10.04 you can update to 10.10 and then to 11.04.  However 10.04 is a long term support version and is supported until April 2013.  You can use it to get comfortable with Ubuntu before you upgrade because it is a very stable release as is 10.10 but support for it ends April 2012.  Something else to consider, you could not install 11.04 on your machine so it may be difficult to upgrade to it.
