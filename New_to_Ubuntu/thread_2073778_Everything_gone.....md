### Post by cmcanulty on 2012-10-20
I think there is an easy way to lose a separate home partition on  upgrades. You have to remember to choose "something else" at the partition step  set your old /Home as /Home and the file system type. Otherwise it defaults to "do not use this partition", it caught me one time as it is easy to think if you have a separate home it will stay as home. I really think they need to put a caution note at that step in th install process. This doesnt'happen if you choose upgrade but it does if you do a new install or upgrade from a DVD or USB.
