### Post by tom.swartz07 on 2010-10-07
Hey.

There have almost always been problems reported with the WUBI install.

Have you tried rebooting and actually booting from the USB key, rather than installing using WUBI?


To do this, you need to follow these steps:
Plug in your USB key- after it has been formatted for the LiveUSB- you should be good, as it recognized it as a WUBI installer in Windows.

When you first hit the power, IMMEDIATELY hit the ESC key. You might have to tap it a few times, but this will bring up the 'bootloader' menu. You should be able to select from USB or HDD from this menu. IF THIS DOES NOT WORK, and it boots to windows, Hold the power button and try again. You cannot harm the existing Windows install if you catch it before it gets halfway booted up. IF YOU STILL cannot get it, stop and I will type out the longer method to force it to boot from the USB key. 


If you do succeed, then just select "Try Ubuntu" and install once you have verified that the drivers work. Ive outlined the proper method for this in my post ^^ above.

Let us know how it turns out!
