### Post by dawynn on 2009-01-21
There are a couple menu options that are available to you.

1) Add Software -- its in the first Gnome pull down menu, at the bottom.  This tool has software grouped by categories, and there should be one for Internet, Networking, or Web.  Note that there is an option at the top of the tool for the repositories you want to show options from.  You'll probably want the option to show from all repositories.

2) Synaptic -- This can also group things by category, but I forget the option name (try the buttons in the bottom-left corner).  It will have a few options for Internet, Networking, and Web applications try them all.

You'll find things like Firefox, and Iceweasel (closely related to Firefox, but less proprietary), along with minimal-resource browsers, even console web browsers (may seem silly at first thought, but very helpful for blind people with machines that can read console text).  Take a few for a spin.  Some have better compatability with the web than others.  *Basically* Firefox is well respected and extremely compatable while also being resource hungry as browsers go.  *Generally* lighter browsers sacrifice web compatability, some to unusable extremes.

### Post by dawynn on 2009-01-21
> **sisco311 said:**
> Dillo
Epiphany
Konqueror
SeaMonkey
Lynx/Links
Iceweasel
Opera
Swiftweasel

I've heard good things about Opera.  Konqueror works fairly well, especially if you use KDE.  Konqueror doubles as a file management tool (much like M$ Explorer).  Browsers based off of Firefox seem to work pretty well in my experience, but I would investigate more why you're moving away from Firefox before choosing one of those (Iceweasel, SeaMonkey, Swiftweasel).  I believe that Swiftfox (or is it Swiftweasel now?) was supposed to be compiled to work better with newer machines, others (Iceweasel) try to remove all the proprietary pieces out of Firefox.

For the lighter browsers: I had great experience with Epiphany compatability, and its fairly light.  Kazehakase works well in most areas, but I couldn't figure out how to set up a home page -- kinda weird that way.  Dillo is lightweight to the point of unusable.  Of all the browsers I've tried, Dillo seemed the most incompatible with the web.

Lynx is a console based browser.  It has its place, but unless you're blind, or just want to have a backup browser for when your GUI crashes, this is probably not for you.

In the end, its your machine, and welcome to the world of Linux.  You now have choice.  Install some and see what you like.

As long as you're using one of the main supported GUI interfaces (Gnome, KDE, XFCE), installing these packages should create menu options for the various browsers.
