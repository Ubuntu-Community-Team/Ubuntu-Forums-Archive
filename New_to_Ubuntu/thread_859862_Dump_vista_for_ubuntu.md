### Post by dhughes on 2008-07-15
No Windows games unless you can install them using the Windows Emulator (WINE) application, but you probably wouldn't do that as a beginner.

 There are lots of pros; stable, low cost, tons of support, tons of apps, highly configurable

 Cons: it's not Windows (if you're used to Windows), more restrictive when administering the system, different applications for similar purpose i.e. Pidgin instead of MSN Messenger, AIM

 The gist of it is Linux is not Windows. It's an Operating System and these days with IM and Web browsing it doesn't really what you use as long as it's stable and you are comfortable using it.
