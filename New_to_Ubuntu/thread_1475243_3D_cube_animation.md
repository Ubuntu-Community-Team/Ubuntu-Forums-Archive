### Post by quadproc on 2010-05-06
> **strictly86 said:**
> Hi how do I do the 3D cube animation? I have set the Visual Effects to extra and can't see the options to enable it anywhere? I'm using 10.04 thanks!
You need to set up several things.

In system > preferences > compiz, set desktop cube and rotate cube.  Also click on _rotate cube_; this will bring up a screen of settings.  Set the zoom to something around 0.3; you won't get the effect at 0.  Reclick on rotate cube if it went unselected.

Make sure that you have a 4 column 1 row set of workspaces.  They are located in the bottom row, next to the trash can.  Right click on one of them, select preferences, and set your rows and columns.

Now, hold down ctrl alt and press the right or left arrow key.  Or, holding ctrl and alt, hold down the left mouse button and move the mouse.

If you are running version 10.04, all bets are off.  Many people are having severe display bugs with that release.

Enjoy!
