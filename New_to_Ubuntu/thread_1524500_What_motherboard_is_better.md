giga-byte

biostar-usa


Five of one, half-dozen of the other...

these boards use the sane chip-set ( # VIA VT8601A (PLE133)
# VIA VT82C686B )
so they are basically the same.

The Giga-Byte has a **slight** edge... 
it seems to have a better audio codec, and on-board network.

RAM will be the issue, as SDRAM (PC133) can be hard to find in larger sizes. You only get two slots, so finding a pair of 256M sticks is (IN MY OPINION, LET THE FLAMES BEGIN) a minimum.

Definitly try for a Tualatin core, cache size DOES matter, reject Celeron if at at possible.
Tualatins can be found in cores over 1GHZ, with 1MB cache.

Maxing the RAM will give you the most Bang-For-Buck.
