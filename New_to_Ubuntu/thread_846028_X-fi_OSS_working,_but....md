### Post by Mornedhel on 2008-07-01
OSS is kinda outdated, but still supported since some applications depend on it (it's guaranteed that whatever environment you use, as long as it's Linux, if it supports sound at all it will at least support OSS). But OSS was never really meant to mix sounds from various sources, neither will it in the future. If there are ALSA drivers around, use them.
