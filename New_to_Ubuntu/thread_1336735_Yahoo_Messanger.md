### Post by 3rdalbum on 2009-11-24
The preinstalled Empathy program can connect to the Yahoo IM network.

If you don't like Empathy: the Ubuntu repositories contain Kopete and Pidgin, and there are many web-based IM clients that can connect to Yahoo. My favourite is Meebo.

### Post by zeroseven0183 on 2009-11-24
Click on **Applications** > **Internet** then select **Empathy IM Client**. There you can setup your Yahoo! account and other networks as well.

If you don't like it, as 3rdalbum mentioned, you can try Pidgin or Kopete. You install them from the **Ubuntu Software Center**.
