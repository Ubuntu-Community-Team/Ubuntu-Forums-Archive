### Post by mooreted on 2014-07-20
To answer the question as to why you would choose 64-bit over 32-bit: 

Processing:
--Modern processors are all 64-bit processors so you are utilizing the capabilities of modern processors. 

Expandability:
--You can add more RAM later if needed, you aren't limited to ~4Gb.

UEFI:
--Modern motherboards now come with UEFI which requires Ubuntu 64-bit to install on these boards.

Graphics:
--Graphics processors are also 64-bit so you tend to get better performance from 64-bit operating systems.


There are lots and lots of discussions about it these days, but unless you have some ancient hardware, there is no reason not to use a 64-bit operating system.
