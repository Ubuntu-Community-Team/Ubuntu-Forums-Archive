### Post by Poyntz on 2008-11-06
or from the ubuntu menu, you could click on on **System** -> **Administration** -> **Synaptic Package Manager** and an installer with an interface will pop up. Click where you see a list of files (under **Package**) then type firefox. A search box will pop up where it will match your search query to a result and highlight it. Where you see firefox-3.0, left-click on the box there (a menu will pop up), select "Mark for installation" next to firefox-3.0, then hit the big **Apply** button (it has a tick icon). That will install firefox-3.0
