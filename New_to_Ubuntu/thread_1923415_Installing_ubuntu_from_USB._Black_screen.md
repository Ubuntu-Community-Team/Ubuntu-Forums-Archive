### Post by Scott Baker on 2012-02-10
A couple of questions here. First, how big is the USB drive that you are trying to set up. I've not had much luck installing 11.04 or 11.10 (especially 11.10) on anything less than 4 Gig. It will install on the drive, but booting and use is strangely, hit or miss. For the later versions of Ubuntu, 4+ Gig seems to be the ticket. Next, did you start with a "clean" drive? If it wasn't clean, this could be part of your issue. If you could answer these questions, it would help greatly. Thanks.  :-k
