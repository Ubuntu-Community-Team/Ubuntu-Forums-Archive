### Post by grahammechanical on 2014-02-28
This is what I see when I do what you did,

1) Try Ubuntu without Installing
2) Install Ubuntu
3) Check disc for defects
4) Test memory
5) Boot from first hard disk.

Why did you choose to boot from the first hard disk? You should have chosen Try Ubuntu without Installing. Then you will be able to test Ubuntu before installing it to see if there are any problems running Ubuntu on your machine and whether you want to install Ubuntu.

An alternative is to not do anything and just let the Ubuntu live session load to the desktop. 

What operating system do you have on that machine? Is it working?
