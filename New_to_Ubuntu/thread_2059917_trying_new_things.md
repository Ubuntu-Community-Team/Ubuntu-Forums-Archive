### Post by newb85 on 2012-09-19
If you're looking for programs to add, you should do some browsing around at:

[omgubuntu.co.uk]("omgubuntu.co.uk")
[webupd8.net]("webupd8.net")

They frequently post articles about neat programs to add, and usually give easy-to-follow instructions for installing.  If you have something specific in mind, their search functionality is pretty good, too.

If you want to go further than that, you could install different Desktop Environments.  The ones in the repositories would be a great place to start.

XFCE (used in Xubuntu) - less flashy than Unity, less demanding on graphics card.
```
sudo apt-get install --no-install-recommends xfce4
```

LXDE (used in Lubuntu) - again less flashy, less demanding all the way around.
```
sudo apt-get install --no-install-recommends lxde
```

Gnome - still flashy, still demanding, just different.
```
sudo apt-get install --no-install-recommends gnome
```

Cairo-Dock - can be added to another Desktop Environment or run as its own DE.
```
sudo apt-get install cairo-dock
```

If you install another DE, you can get to it by clicking the Ubuntu icon next to your name on the login screen and selecting the new DE.

While I'm not just going to whinge about your posting style (hint guys!), it is a fact that better-written posts get better answers. Just saying, feel free to ignore me. Onto the questions.

"and will try or add anything that will make it better". We try to make it as good as possible by default. ;)

Seriously, some things to try:

1. Check out the stuff in the software centre. There's a rating system to follow, and some hidden gems in there that don't always make the front page.

2. One of Linux's great talents compared to Microsoft or Apple operating systems is that you can change the default interface. The one you're using (in Ubuntu, by default) is called Unity. There is tens, probably hundreds of different interfaces. 

Some are very minimal and include only software to manage the windowing system, examples including DWM, openbox, etc. These tend not to be for people new to Ubuntu, but are very powerful once you get used to them.

Some, on the other hand, are all-inclusive, attractive, and correspondingly a little more user-friendly. Examples include KDE, Gnome-Shell, and Unity.

There's also a big mix in the middle. Some highlights include XFCE, LXDE, older versions of GNOME, and the various GNOME 2 clones.

3. Ubuntu is still basically a graphical shell around a command-line environment. There is a good reason for this. Linux's command-line is amazing. It really is worth learning, for various reasons. It depends how much you want to explore. To start with, it will feel clunky, but after a while you'll learn to use it in a different way.

Here's what I learnt the basics from: [http://linuxcommand.org/](http://linuxcommand.org/)


Yes, you can just reinstall ubuntu, should you bork it that badly. However, one of Ubuntu's strengths is that it recovers very nicely. Often it's a case of how much effort is required to reinstall versus how much effort is required to recover. :)

Playing with the OS is very easy. You can understand much of how Ubuntu works via point 3 on the above list, but if you want  to REALLY understand how a piece of software works, the source code is freely available from the Ubuntu repositories.
