### Post by Fast_Wyvern on 2009-12-22
I'm using ubuntu on my Alienware M5500i-r3, only problem was getting the mic to work, but once the settings were correct in alsamixer and the model set correctly as F1734, a rebranded Fujitsu apparently everthing is working fine

Info on snd-intel-hda modification can be found[ here<<<]("https://help.ubuntu.com/community/HdaIntelSoundHowto")

need to change the ```

options snd-hda-intel model=F1734
```

Restarted, then open alsamixer  (Type alsamixer in terminal) and unmute the mic
