### Post by stoogiebuncho on 2011-03-06
I've done this before and it's not the most user friendly thing.  I think I just went through that tutorial and copy-pasted all of the commands into a terminal one by one.  I don't know that you'll get better instructions somewhere else.

I've heard that Ubuntu Christian Edition has Dansguardian installed by default so you don't need to mess with anything - don't know if that's enough of a reason to switch to Ubuntu Christian Edition, but now you know.

Another option to consider is filtering through DNS.  OpenDNS offers different levels of content filtering, and it's a bit easier to set up than Dansguardian.  A lot of schools use these type of filtering systems.  

OpenDNS is free, you just have to change your DNS settings and if you have a dynamic IP address you'll have to set up ddclient.
