### Post by gandaran on 2009-03-29
install lottanzb, get the ubuntu .deb package [here]("http://www.lottanzb.org/downloads/")
the set up is easy, in preferences » newsbin tab, enable newsbin, enter your newsbin username login account and password.
you also need a news-server so find one or use the same one you have on News Leecher
