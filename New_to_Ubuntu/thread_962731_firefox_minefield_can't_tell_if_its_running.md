### Post by bangmalley on 2008-10-31
you can install minefield from mozillateam repo
add this to third party in Software Sources, change hardy to intrepid if you're using intrepid
```

deb http://ppa.launchpad.net/mozillateam/ubuntu hardy main

```

or you can read this [guide]("http://www.debianadmin.com/install-firefox30a1-in-ubuntu-and-list-of-addons.html") for manually install firefox. The guide is for 3.0a1, but can be a useful reference.
