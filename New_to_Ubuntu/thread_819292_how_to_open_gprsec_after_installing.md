### Post by Sealbhach on 2008-06-05
You can add items to your menu bar:

System/Preferences/Main Menu

Select the Menu group you want the menu item to be in, e.g. Office, Sound,Other.

Click New item

Change Type to "Application in Terminal"

Put in a description and for the command (in this case) make it:

gprsec

Close. 

When you click the menu item now it should start the program running from the terminal.

You can add any new program to the menu in this way - all you're doing really is saving a record of a terminal command.  

Incidentally, you can start any program by typing its name in the terminal e.g. type "firefox" in the terminal and it starts up Firefox. If you close the terminal it closes Firefox. Forgive me if you already know this stuff....
