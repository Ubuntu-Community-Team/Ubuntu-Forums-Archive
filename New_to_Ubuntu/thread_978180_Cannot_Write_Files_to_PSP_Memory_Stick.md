 Write Protection Switch
	 
Some memory cards are having a write protection switch integrated on the side or on the back of the memory card for preventing the content from being erased or overwritten by accident, just like floppy disks. Both SD Memory Card and xD-Picture Card have their write protection switch on the side of the memory card in contrast to the Memory Stick PRO and some Memory Stick Duo that have their switch on the back.
