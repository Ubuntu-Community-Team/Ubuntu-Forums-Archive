### Post by RichardLinx on 2009-04-24
You need to set the BIOS to boot from the flash drive. On startup the computer should tell you how to enter the BIOS, if for some reason it doesn't that's not a problem. It's usually one of the following keys: "F2" "Delete" "Escape" or another F* button.
