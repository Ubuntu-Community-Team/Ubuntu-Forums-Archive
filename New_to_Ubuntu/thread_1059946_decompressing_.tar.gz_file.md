### Post by balagosa on 2009-02-04
```
tar xvzf foo.tar.gz
```
this command decompresses the target *.tar.gz to the current folder it is in.

As for installing a file, we need more info. Maybe you can do an **"ls"**. Usually the installers come in *.sh format or makefile.
