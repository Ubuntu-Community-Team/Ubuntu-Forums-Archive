### Post by lwaaleboer on 2011-10-15
I am one of the many new converts from windows to Linux/Ubuntu.
I love Ubuntu but still run windows for a few programs which are not available or up to scratch in Ubuntu. So far I have not had many problems, I learned to use the terminal and to find my way around the files and folders, which is not all that much different then windows. So yesterday i upgraded and everything went pear shaped from there.The upgrade went through it motions until the restart. Got the *waiting for network connection * and then *waiting 60 seconds more for network connection *messages*. *That was it, no more Ubuntu, it died on the job. I got on my vista machine to search for answers. Soon found out I was not the lone ranger. I followed the instructions in [https://bugs.launchpad.net/ubuntu/+s...us/+bug/811441]("https://bugs.launchpad.net/ubuntu/+source/dbus/+bug/811441") post 24. That worked , got Ubuntu up and running but not much was working right. No network, graphics were up the creek, could not copy files onto my USB backup drive. I got the problem licked by reinstalling Natty ,did a restore from backup files and had it running again as before the upgrade attempt. Then  I copied all my data files , videos and pictures on the backup disk , just in case the backup program would not work in 11.10 or mucked it up. A clean install did the job, it's running good (for now).
learned something new, don't upgrade from the update manager but do a clean install. (much like windows):wink:
