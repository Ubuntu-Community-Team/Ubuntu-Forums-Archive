### Post by HuckleSmothered on 2009-06-29
While sending an email does require it to go through an email server, you could always setup your own mail server. It would send the email message pretty close to directly to the target computer if it is internal. Otherwise, it would send it out to the cloud if external. Granted, that is probably a bit too much work unless you have many computers which all want to do a lot of internal communicating.
