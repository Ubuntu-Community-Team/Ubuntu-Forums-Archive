### Post by yc2 on 2008-07-04
How do you enter "safe graphics mode" in Hardy Heron Live-CD?

In older versions of Ubuntu this option could be chosen as soon as the Live-CD booted, but when booting Hardy Live-CD it is not there?

---

### Post by sharks on 2008-07-04
try again from the Hardy live CD. When you get the initial startup menu, hit F4 while your cursor is on "Try Ubuntu without changing anything..." and select the "Safe graphics mode". Hit return to select that mode and start Ubuntu.
