### Post by Temposs on 2010-05-02
The method of notifying about updates has changed starting in Ubuntu 9.04. Ubuntu checks for updates automatically as usual. If there are updates, there is no longer any icon regarding them. Instead, Ubuntu pops up the Update Manager for you automatically.

If there are Security/Critical updates, Ubuntu will open Update Manager for you at most once per day. If there are no Security/Critical updates but only Recommended updates, Ubuntu will only open Update Manager once per week.

The idea is that you don't want to be always bothered with updates whenever they are detected, which could be multiple times per day, but instead having the Update Manager come up for you on a regular basis with the accumulated updates.
