### Post by routed on 2011-12-24
A couple suggestions..
Not being able to boot from cd could be several things. Have you successfully been able to boot from a different cd? That would be a surefire way to tell if your cd rom is working correctly. You could try it with a different Linux distribution, just to see if it works.

I am assuming that your BIOS is configured correctly to boot from cd. 

If you want to install on top of Windows, maybe something like Wubi would help the installation go better then extracting the files to a drive. 

Has the image you used to burn ubuntu worked in the past? It could of possibly been corrupted.
