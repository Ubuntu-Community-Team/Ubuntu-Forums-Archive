### Post by Paqman on 2013-01-19
Wubi will automatically download the correct version of Ubuntu for your machine. If it's downloading the 64-bit version of Ubuntu then your machine must be 64-bit capable. There's no reason to use a 32-bit OS on a machine which can run 64-bit.
