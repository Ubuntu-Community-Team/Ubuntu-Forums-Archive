An easy fix, it seems 10.04 doesn't come with the fortunes package by default, and can be easily installed thusly:
```
sudo aptitude install fortunes
```

There are more quotes that you can download as well, you may like different language ones, or offensive ones.
- Just do a search in the synaptic package manager for "fortunes" then select which ones you want to install.
