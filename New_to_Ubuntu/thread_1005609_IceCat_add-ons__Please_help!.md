### Post by dsterry on 2009-08-05
The point of GNU Icecat is to remove the temptation to install non-free addons. There are many addons in Mozilla's repository that are not free software so GNU disabled the addon finder. If you want to find free software addons, there's a great list at: [http://www.gnu.org/software/gnuzilla/addons.html](http://www.gnu.org/software/gnuzilla/addons.html)

The other issue you mentioned was flash. The plugin you may be talking about is Gnash which has frequent updates. You may want to get it straight from the source at [http://www.getgnash.org/packages/snapshots/ubuntu/](http://www.getgnash.org/packages/snapshots/ubuntu/)
