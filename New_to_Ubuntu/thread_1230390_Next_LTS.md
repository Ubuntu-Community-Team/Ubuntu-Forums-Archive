### Post by Sef on 2009-08-03
It could be either 10.4 or 10.10.

From an [interview]("http://www.twistedlogic.co.cc/2009/07/13/shuttleworth-about-gnome-3-0-whats-good-whats-missing-what-needs-work/") with Mark Shuttlework by [Twisted Logic]("http://www.twistedlogic.co.cc/"):

> The LTS will be either 10.04 or 10.10 &#8211; based on the conversation that is going on right now between Debian and Ubuntu. So we&#8217;ve given up some control when our LTS will be, Debian&#8217;s giving up some control but the shared result will be quite powerful, we&#8217;ll be giving upstreams a reason to say &#8220;Let&#8217;s not land all those crazy changes right now, if we want the Debian and Ubuntu major release go out well&#8221;.
