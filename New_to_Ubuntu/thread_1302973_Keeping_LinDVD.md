### Post by niteshifter on 2009-10-27
Hi,

If you do clean install of 9.04, then yes, LinDVD goes away. If you upgrade from 8.04 to 9.04 then LinDVD should persist. It did when I moved from Dell-supplied 7.10 to 8.04.

At any rate you don't need to pay for LinDVD again. You need to contact Canonical - see [this post]("http://en.community.dell.com/forums/p/18963399/19086385.aspx#19086385") (community.dell.com).
