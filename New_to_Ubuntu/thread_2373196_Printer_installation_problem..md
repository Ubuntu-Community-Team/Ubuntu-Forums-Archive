### Post by gurdenite on 2017-10-04
Looking at Epson's download site it looks like you need to install LSB first then download and install the required .deb package depending which version of Ubuntu you are using, either i386 or amd64

If you go to [this site](http://download.ebz.epson.net/dsc/du/02/DriverDownloadInfo.do?LG2=EN&CN2=&DSCMI=64679&DSCCHK=25b7edf9937bf2d2293b6d67ed40a6dc8bc8279c) then scroll to the bottom, click the Accept button the instructions and download links will appear.
