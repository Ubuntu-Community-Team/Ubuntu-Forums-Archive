### Post by snowpine on 2010-02-16
No reason to install UNR unless you like its special "launcher" interface. "Regular" Ubuntu is a better choice if you want the Gnome desktop. 

I recommend researching your specific netbook model on these forums, to identify any possible hardware issues. (Most netbooks will work fine, but some made need a little "tweaking" to get wireless, video, etc. working correctly.)

Yes, you should be able to "dual boot" with your existing OS, here is one tutorial: [https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)
