### Post by Frogs Hair on 2011-11-12
You can try to restart the shell , but you will have to enable the command prompt which is disabled by default .

1 . Open the 'Keyboard' application .
2.  Proceed to shortcuts > system > " Show The Run Command Prompt " 
3 . Follow the instructions on the bottom of the dialog box and use Alt + F2 as the shortcut .
4.  Close all programs and open the command prompt .
5.  Run the following command/s.
```
r
```OR
```
restart
```
