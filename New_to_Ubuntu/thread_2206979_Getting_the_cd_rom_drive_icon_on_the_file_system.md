### Post by grahammechanical on 2014-02-21
When you put a CD in the drive it should be detected by the File Manager and an icon (circular disk image) for it should appear in the Launcher. Well, this is what happens with me. Right clicking that icon should give an option to eject. These are the option I get from a right click on that icon

Open
Audio Disk
Unlock from Launcher
Eject.

It should be at the bottom of the Launcher. If the Launcher is full just move the pointer to the Launcher and move it down to the bottom and the Launcher will slide up and reveal what was off screen.

File Manager should also show an icon for the CD and a right click will reveal a menu with the option to Eject. So, two methods.

### Post by grahammechanical on 2014-02-21
Ah!

There is a Disks utility in the Dash. That lists the Devices/drives on the system. Select the CD/DVD Drive and in the Window at the top right there will be top square panels. The left panel (with an upward facing triangle) has a mouse-over that reads Eject the medium in the drive. It opens the drive whether or not there is disk in the drive. I had just tested this.
