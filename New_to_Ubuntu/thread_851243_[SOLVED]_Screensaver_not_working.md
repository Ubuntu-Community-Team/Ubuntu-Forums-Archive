### Post by wpshooter on 2008-07-06
Make sure you have the timing slider for the screen saver kick-in set to less than that of the blanking of the display.  For instance set the screen saver at 2 minutes and then set the screen/display blanker at 4 minutes and then restart the computer and see if each function does not kick in at the proper time.
