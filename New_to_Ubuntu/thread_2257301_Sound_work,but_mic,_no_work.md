### Post by ajgreeny on 2014-12-18
Run alsamixer again, use cursor keys to move to the columns for Mic (there may be two columns, one for front and another for back) and use M key to unmute if there are two Ms at the bottom of the columns, then cursor keys again to raise the levels, if you need to.  Press Esc to save settings.
