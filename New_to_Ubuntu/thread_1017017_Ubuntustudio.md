### Post by eagle416 on 2008-12-20
What kind of stuff do you want to be able to do? Special effects, or just plain editing?

I understand Ubuntu Studio comes with PiTiVi, a fairly simple multitrack video editor. I don't know much about it. I heard it's good for fairly basic stuff.

What I use is a program called Cinelerra, which is really powerful and I would recommend it for stuff like Chroma Key. Installing it can be a pain though, and there are some funny businesses with how it handles file formats.

This Cinelerra Install walkthrough is for Ubuntu Hardy, but I'm sure you can tweak the instructions for Intrepid (assuming that's what you're using).
[http://http://www.ubuntu-unleashed.com/2008/05/howto-install-cinerella-video-editor.html]("http://http://www.ubuntu-unleashed.com/2008/05/howto-install-cinerella-video-editor.html")

As for system requirements, I'm not certain, but if you're not doing spectacular graphical stuff (i.e. complex Blender), your computer should work.

Hope that helps.
