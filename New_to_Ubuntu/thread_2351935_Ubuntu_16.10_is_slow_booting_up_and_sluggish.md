### Post by ajgreeny on 2017-02-08
Your 965GM graphics card may well be the bottleneck in the system if you are using the unity desktop as that needs very good 3D capabilities.

I suggest you have a look at one or more of the other desktop environments as in Xubuntu, Lubuntu, Ubuntu-Mate, for example all of which should run faster on that hardware.  If you are having to reinstall I would also suggest you use 16.04 as it is an LTS version, supported until either 2021 for Ubuntu or 2019 for the other versions; 16.10 will have to be upgraded to 17.04 in July this year.

Even though my hardware can run the unity desktop well, I have chosen Xubuntu now for many years; it has a Vista-like layout which you may also prefer.

### Post by ajgreeny on 2017-02-08
Ubuntu is the only OS that uses the unity desktop; Xubuntu uses XFCE, Lubuntu uses LXDE, and Ubuntu Mate uses Mate desktop.  All of them are lighter than unity, requiring fewer resources so try them all if you can.
