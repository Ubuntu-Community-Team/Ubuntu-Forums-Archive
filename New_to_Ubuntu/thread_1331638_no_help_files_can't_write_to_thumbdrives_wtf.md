### Post by ankspo71 on 2009-11-19
Hi,
aptoncd is another thing worth looking into.  If you are satisfied with the packages that Ubuntu has to offer by default, then I don't think Internet is that much of a requirement, but it is if you want to take advantage of all of the packages that can be downloaded through it's repositories. see [http://aptoncd.sourceforge.net](http://aptoncd.sourceforge.net) for a description of aptoncd.

An alternative to Ubuntu is Debian which ((I believe)) provides a way for a person to download all software on cds (currently 31 of them). One difference between Ubuntu and Debian is that Debian does not provide software for non Gnu/opensource packages. I believe aptoncd would be better than going this route though.

Help documentation seems to be scarce in some areas because some package providers (etc) did not implement the help files (yet). I agree this area could use some work, especially for the non Internet users.

Also there are a few free PDF guides for Ubuntu (some have been linked in this forum, check the sticky posts near the top - that might be where I seen them last). There are also some books available in stores.

I hope this helps,
James.
