### Post by Chrissss on 2009-06-13
Drag&Drop works with Nautlius the same way like inside the windows explorer

 * Drag&Drop moves a file (or makes a copy when you move the file to a another drive)
 * Drag&Drop+Ctrl copies the selection
 * Drag&Drop+Ctrl+Shift creates a link

### Post by niteshifter on 2009-06-13
Hi,

You've got options ... and here's one more:

Click and drag to wherever - but at the destination press and hold the Alt key then release the mouse. A menu will appear giving you options to:
Move here
Copy here
Link here
Set as background (greyed out if the object is not an image)
Cancel
