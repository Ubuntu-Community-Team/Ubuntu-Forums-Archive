### Post by Waqsione on 2008-07-10
Hello. I'm thinking of dual booting Windows and Ubuntu, and have some questions regarding the matter.

1. If I were to install Ubuntu on a separate hard disk, do I just follow the ordinary installation procedures?

2. Any security concerns I should watch out for?

3. Are there be any notable side effects, and if so, what can I do to prevent it?

4. Will my computer wear faster, not so much, or not at all?

Any help will be appreciated. (Also, please tell me if there is anything else that you think I should know)

---

### Post by echo314 on 2008-07-10
1) If you are using a different drive, you should be able to select that drive in the partitioning setup menus, but have it automatically set up the drive for you. You can also create your own root and home partitions with info from a quick search 'separate home partition'

2) No security concerns that I am aware of, I don't think you have anything to worry about. Other then, I guess, anyone else with access to the comp has access to both operating systems potentially. 

3) On bootup, Ubuntu will be top of the list (over your already-installed OS). This is easily fixed if you desire. 

4)Your computer will not wear any faster then normal (no comp wears 'not at all.' Its only a different operating system, which wont effect (for all our purposes) how long your computer lasts.

If you need specific instructions, or get confused, please ask. Its better to be safe then sorry when dealing with partitioning. Don't get me wrong, its not something you should be fearful of, but there is the potential to wipe all the date on your current HD if you get something wrong. Just be careful and ask questions.
