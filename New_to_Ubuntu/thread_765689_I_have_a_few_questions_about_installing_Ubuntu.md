### Post by kk0sse54 on 2008-04-24
1. you can back up your data on a seperate hard drive or on DVD's and is usually recommended before you do any partitioning
2.Dual Boot consists of making two new partitions: your ubuntu partition which is the actual OS and the swap partition about the size of your RAM which helps with memory managment. As long as you have extra space on your hard drive you can select how you want ubuntu to make those partitions. Before you install try using the liveCD as you can screw with it without messing up your system and it lets you try out ubuntu to see if you like it
3.Yes you can install the beta version but be awared that there will probably be hundreds of updates that may take a few hours.
