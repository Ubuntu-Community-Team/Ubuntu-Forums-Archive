### Post by hubie on 2008-09-23
If that is the really old format I think it is, you can try converting them (open, then "Save As") with [ImageJ]("http://rsbweb.nih.gov/ij/") using the [InstantImager Reader]("http://rsb.info.nih.gov/ij/plugins/iireader.html") plugin.

If you go this route, it can be super easy to convert a whole bunch of images at once (it is for the natively-supported image formats; otherwise it depends on how the image opener plugin is written).
