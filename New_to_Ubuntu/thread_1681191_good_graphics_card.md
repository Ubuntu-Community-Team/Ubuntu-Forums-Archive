### Post by scheuri on 2011-02-04
I would also advise you to buy nvidia. Never had problems so far. Depending on what you need you might go for a passive cooled card (less noise and no fan that can get clogged up by dust).
The only drawback is, that you (still) should use the driver from nvidia. That is relatively easy to install, but still needs to be done. The free drivers start to get better and better, but may lack some features you might need.
