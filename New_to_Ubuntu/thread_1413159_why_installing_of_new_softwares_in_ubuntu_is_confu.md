### Post by mcduck on 2010-02-22
It's confusing and difficult because it works differently from what you are used to, not because it would actually be any more complicated or require any more work.

(actually it's a lot *simpler* and requires less work than downloading, installing and updating applications in Windows does).

It's quite easy to forget that once you were a beginner in Windows as well, and had to learn how to find the software you want and how to install it. Probably took more than few days to learn all that... ;) Just give it some time and you'll learn how it works. The main point is to understand that most of the things you have learned about using computers wasn't actually about computers, it was about using *Windows*...

### Post by nerdtron on 2010-02-22
Installation is a lot easy and secure in ubuntu. The open source programs are in one central location. The repositories are constantly updated so you have to reload the software list. It contains stable releases of programs. Just check the one you need and hit apply they will be downloaded and installed automatically. Note that in this also install the dependencies of the program you want to install. And you don't have to restart the whole PC after the installation. BTW you can also downlad the latest software release from their website and install it manually. Its very similar to XP, just double click the package, enter your password and installation will follow.
