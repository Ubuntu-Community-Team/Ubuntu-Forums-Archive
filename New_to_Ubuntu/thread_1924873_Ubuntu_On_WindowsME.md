### Post by Mark Phelps on 2012-02-13
While all this "bashing" of WindowsME is, I'm sure, amusing to many -- the ACTUAL answer to your question is -- probably NOT.

Linux distros are all a little different (and some are VERY different) from each other, so they all have different minimum machine requirements.

But to give you the impression that a current Ubuntu OS will run on a PC built BEFORE the Windows XP days -- that is entirely wrong.

See the link below for Ubuntu 11.10:

[https://help.ubuntu.com/community/Installation/SystemRequirements]("https://help.ubuntu.com/community/Installation/SystemRequirements")

That said, there are other lightweight distros that MIGHT be able to run on an WindowsME-vintage PC.  To check them out, go to distrowatch.com and look for info on Lubuntu, Puppy Linux.

Maybe others, who have used vintage equipment, can provide more specific examples ...
