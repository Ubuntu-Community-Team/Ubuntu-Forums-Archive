### Post by drs305 on 2008-07-05
You can right click and select 'Uncheck All'. There isn't a gconf-editor setting to 'load unchecked' but the click isn't too hard if you have a mouse.  ;-)

### Post by sdennie on 2008-07-05
Why would you not want to install all the updates?  Ubuntu is not Windows and so updates aren't going to limit your control over the system or anything devious like that.

Still, if you are really adamant about not getting updates, you could probably go to System->Administration->Software Sources and disable almost all the sources but the security updates.  That may or may not work.
