### Post by 3rdalbum on 2010-02-24
> **MechanicalDaddy said:**
> Hello all, I have been trying to install Google Earth on Ubuntu 8.04 LTS  I followed the instruction given in this thread, and effectively, nothing happened.  Is there a secondary step that I am missing?  Thanks for your help!

Here is what happens in the Terminal:

steve@Stewie:~$ 
steve@Stewie:~$ 
steve@Stewie:~$ chmod a+x /home/steve/GoogleEarthLinux.bin
steve@Stewie:~$ 

...and nothing.

Steve

chmod a+x makes the file ABLE to be executed (by all users). You need to run the file now. In fact, it's an installer that will want to put files into locations outside your home directory; so you need to run it as root.

```
sudo /home/steve/GoogleEarthLinux.bin
```

That's assuming that you can't find a repository or a Debian package for Google Earth, which would surprise me.
