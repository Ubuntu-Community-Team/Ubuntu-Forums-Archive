### Post by chrisod on 2008-11-19
Have you tried opening it in Open Office? If that does not work, you'll probably have to get access to a Windows box and see if Office will open it. Or, you can download the free trial of Wordperfect and re-save it as a PDF or PPT file that Open Office can work with. Word Perfect used to be available for Linux, but I think that was discontinued a couple of years ago.

### Post by bennachie on 2008-11-19
You might also be able to run an evaluation copy of WordPerfect under WINE. If you can get access to a Mac on which MacLinkPlus has been installed, you could directly convert the file(s) from SHW to PPT.

Finally, I have never attempted the exercise, having never had any backlog of WordPerfect files to convert, but there is just a small chance that you might be able to persuade WordPerfect 8 to run under Ubuntu. See:
