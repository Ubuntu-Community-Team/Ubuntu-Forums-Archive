### Post by tuskpc on 2008-05-09
Does anyone know how to permanently change the size - or in my case the width - of a terminal?  I would like to make it 207 x 24.

---

### Post by spiderbatdad on 2008-05-09
You can right click on the icon and select properties. Add --geometry 207x24 to the end of the command line.

---

### Post by noynac on 2008-05-09
> **tuskpc said:**

Gnome-terminal has a --geometry switch that will do what you want.
