### Post by quixote on 2010-10-09
I'm not sure, but I seem to remember that, for Windows, all grub really does is hand control to the Windows bootloader.  What's happened is probably the Windows bootloader stopped working and needs to be reinstalled.  To do that in WinXP (not Win7, if I remember right) you just need to boot with bootable Windows media, and run "fixmbr" (without quotes) at the command line.  Then you'll probably need to reinstall grub, because Windows bootloader ignores the existence of all other OSes and will overwrite it.
