### Post by aysiu on 2010-04-16
I think you'd be better off trying to install the .rpm than the .tar.gz.

Try pasting these commands into [the terminal](http://www.psychocats.net/ubuntu/terminal): ```
wget -c http://www.jetico.com/linux/BCWipe-1.9-4.i386.rpm
sudo apt-get update
sudo apt-get install alien
sudo alien -i BCWipe-1.9-4.i386.rpm
``` Have you thought about using *shred* or DBan to wipe instead?
