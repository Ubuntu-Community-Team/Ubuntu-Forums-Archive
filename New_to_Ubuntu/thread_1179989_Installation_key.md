### Post by lovinglinux on 2009-06-06
Before installing I recommend reading the tutorials at [http://www.psychocats.net/ubuntu/index](http://www.psychocats.net/ubuntu/index)

While Ubuntu installation is not complicated, reading some of those tutorials will allow you to better plan how you want to install it and will improve your experience in the long-run. For example you can install Ubuntu [without removing your current OS](http://www.psychocats.net/ubuntu/wubi). 

One tutorial that is very nice is about [creating a separate partition for home](http://www.psychocats.net/ubuntu/separatehome). "Home" is where Ubuntu save all your personal stuff and personal system settings. A separate partition for home allows you to keep your settings and personal files after installing a new version of Ubuntu or re-installing the version you have. So, if you mess with the system files, just re-install and you don't loose your files.
