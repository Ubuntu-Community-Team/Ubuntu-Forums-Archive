### Post by KL_72_TR on 2011-05-20
System > Administration > Printing 
Printer Configuration > New Printer
'Searching for Printers' might take some time.
'New Printer' select printer.
You will see printers found by Ubuntu in 'Devices', select than 'Forward'. 
A message 'Searching for Drivers' will pop-up.
Select manufacturer or leave the default. Than Forward.
For the name leave the default.
