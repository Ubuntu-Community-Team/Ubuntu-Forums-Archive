### Post by sandyd on 2009-12-06
```

gksudo gedit /etc/dhcp3/dhclient.conf

```
make sure there is a "#" in front of the line that says "prepend domain-name-servers"
on a new blank line after that one, add
prepend domain-name-servers 208.67.222.222, 208.67.220.220;

and save.
restart.
