### Post by ACJarrett on 2008-04-30
The best way to get the cube working is to goto your Add/Remove Programs under Apps, then search for Compiz and install the Advanced Desktop Manager.  Install That

Now under Preferences you should have the advanced desktop manager available, use that to set up the cube to your liking.
