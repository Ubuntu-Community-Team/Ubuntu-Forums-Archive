### Post by Neural oD on 2008-10-27
hi - make sure that your bluetooth hid's are set to discoverable - this is different for each device. Then once you have made sure of that - scan for them - u can use graphic tools for this - try Blueman - is really nifty. Do a "sudo apt-get install blueman" to install. 
U can then scan with this - and hopefully things should work out! If this doesn't work, then some hardware needs specific drivers - but in linux there are many workarounds :)
