1. Yes, it is very friendly. Much friendlier than Windows. Be sure to install Windows first and Ubuntu will automatically set it up so you can choose between the two at bootup.

2. It is much simpler to set up with one partition, but if you put .home on a separate partition, it will easier to reinstall if you need.

3. Swap memory is disk space that you set aside (usually in a partition) as a kind of RAM overflow. (Over-simplified example) If you load a 3GB video and you only have 2GB of RAM, it will put the rest in your swap area for later use.

4. The current version is 10.04.1, also known as Lucid Lynx.

### Post by Thryn on 2010-08-31
Swap is also where the data from RAM is saved when you hibernate, so that it can reboot with all the same things running, right where you left off. Quite handy. This is I believe part of why it's good to have at least as much swap as you have RAM.
