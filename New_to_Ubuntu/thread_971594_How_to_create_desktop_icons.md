### Post by pointyblue on 2008-11-05
I've decided to give Xubuntu a try on my old computer and it seems to run smoothly.

However: I can't figure out how to create icons for programs on the desktop. In Ubuntu/Gnome I just right click the program in the applications menu and create a shortcut for the desktop. But that doesn't seem to work in Xubuntu/Xfce...

Can someone please tell me how to do this?

Thanks! :)

---

### Post by Zill on 2008-11-05
1)  Right-click on Desktop and select Create Launcher
2)  In Name box, start to enter the name of function required
3)  A list of all possible apps with this name appears
4)  Select required application from list
5)  Comment, Command and Icons should automatically appear
6)  Change Options as required
7)  Click on Create button
8)  New Icon for application appears on Desktop
9)  That's it!
