### Post by TheFu on 2017-04-23
> **mjhaider said:**
> Yes, I know that, but as I am new to Ubuntu it will take some time to learn things.

There are hundreds of guides on youtube, if you learn that way.   Most decisions aren't that important except the storage layout choices.  If you weren't completely new to Linux, I'd suggest using LVM.  However, LVM is an advanced topic and often confusing for normal end-users just getting their feet wet. LVM is extremely flexible, but not mandatory.

Also, if your machines are fast enough, using a virtualization hypervisor to build/install a few test VMs can't hurt.  As with anything, the more you practice, the better you become.  An install takes me about 15 minutes these days - start to finish.  It isn't a major commitment like with Windows.  If you don't get something correct, you can fix it later or just reinstall. It isn't like you are burning a license key. ;)

If you do go with a VM install, would be worth your time to watch a few how-to guides for the hypervisor you've decided to use. Some choices for the virtual hardware given to the client OS are very important for performance of the entire machine, not just the VMs.
