### Post by pdlethbridge on 2010-03-22
When I deleted a couple of files tonight I had problems accessing files like home, computer, etc. Going into synaptic again, I had used it to remove the files, I found something I hadn't seen before: a history of downloads and files removed by date
This was found under files / history. A window will open and show a date and arrow on the left. Clicking the arrow will show several dates where programs have been added or removed. On the right will be the programs that were worked on. 
So I highlighted a program and used the search button to check it off for reinstalling. Problems were solved and it was easy. Thought I'd share this with everyone as I don't think this is a well known feature as I have not read about it here in the Ubuntu forums. If everybody knows about it, well, my bad!:D:D:D:D
