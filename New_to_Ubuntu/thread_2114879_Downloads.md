### Post by grahammechanical on 2013-02-11
What downloads do you mean?

Ubuntu updates are downloaded and then installed into their relevant folders. The same thing happens to applications we install through the Software Centre.

Downloads of files (such as PDF) over the Internet will go into the folder that we direct the web browser to download them to. The default folder is Downloads but if you have sent that folder to the Rubbish bin then I guess the download cannot proceed without you specifying a new location.

With Firefox, if we go to Edit>Preferences, we will see what the default downloads folder is. Open the General tab. It has the heading Save Files To.

Regards.
