### Post by Dennis N on 2020-09-11
You can also move to a new workspace with Super + PageDown. Super + PageUp will return to the previous workspace. Any application window you open while in the new workspace stays there until you close it or move it. This segregation is useful for many users' work habits. It's the same as the desktop concept in XFCE, MATE, or Lubuntu - you can even have a switcher on the panel to move between workspaces with a mouse click. If you think will you have no use for more than one workspace, don't use them.
