### Post by sirgogo on 2011-07-28
Hahaha. Yeah, you're right. When you install grub via the liveCD (or liveUSB in this case), you are essentially installing it to RAM, so it won't live past this boot cycle.

You should, however, still be able to run grub commands on the hard drive. Check these links:
[http://www.webtechquery.com/index.php/2010/04/install-grub2-from-live-cd/](http://www.webtechquery.com/index.php/2010/04/install-grub2-from-live-cd/)

I've done this loads of times, particularly when I've installed Windows on top of Linux. Honestly, I'm at work so I may not be providing you with the best links right now, but I'm sure you can find it on the internet. Re-installing grub is a pretty common task, so no worries. Your life will be a whole lot better when your compy starts booting properly :D
