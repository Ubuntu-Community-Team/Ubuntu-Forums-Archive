### Post by mkoehler on 2008-05-14
In order to diagnose what's wrong, we'll need more information.  However, you should always try to avoid doing a hard reset.  Instead, the next time it freezes, try doing this:  Hold ALT & PRINT SCREEN, then slowly type R-E-I-S-U-B.  That's "busier," just typed backwards.  That should perform a soft reset, if all else fails.
