### Post by deadflowr on 2014-05-22
Ubuntu 11.04(codename natty narwhal) is dead and the repos have been moved to the old-release repositories.
You have two choices, either install a supported release, or change you repo list to the old-release repos.
To change the repos to old try this
First backup the old source.list file
```
sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak
```
The run this
```
sudo sed -i -e 's/*archive.ubuntu.com\|security.ubuntu.com/old-releases.ubuntu.com/g' /etc/apt/sources.list
```
This will change all those entries to the old release versions.
Another thing is you need to disable the extras repositories, so try this
```
gksu gedit /etc/apt/sources.list
```
and find the two lines for the extras repos and change from this
```
deb http://extras.ubuntu.com/ubuntu natty main
deb-src http://extras.ubuntu.com/ubuntu natty main
```
to this
```
#deb http://extras.ubuntu.com/ubuntu natty main
#deb-src http://extras.ubuntu.com/ubuntu natty main
```
by simply adding the # symbol, the line will disabled.
The reason to do this is because those repos don't get moved.
After doing that, run
```
sudo apt-get update
```
If you get any errors, perhaps post them here.

Even though I have pointed you in a direction of how to change the repos, I would strongly suggest installing a fresh version of a supported version of Ubuntu.
