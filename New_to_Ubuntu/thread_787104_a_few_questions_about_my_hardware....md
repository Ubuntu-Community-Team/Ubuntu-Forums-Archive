### Post by Cypher on 2008-05-08
Yes, you'll want to use the 32-bit version of Linux. THe specs don't list the amount of video memory you have, but it should probably be sufficient to run Compiz.

You need not worry about matching memory sizes, the only thing you have to worry about is the memory speed. The memory bus will go only as fast as the slowest memory chip. So if your 2GB stick is faster than the one already installed, you're not reaping the full benefits..
