### Post by Cypher on 2009-06-02
Aparantly Evolution can't handle the .pst file natively (yet). So you will need to go through Thunderbird before heading to Evolution (unless you're perfectly happy using Thunderbird)..

The blog post [http://www.cyberciti.biz/tips/moving-outlook-email-data-to-linux-evolution-mail-client.html](http://www.cyberciti.biz/tips/moving-outlook-email-data-to-linux-evolution-mail-client.html) details the work. Here Thunderbird on XP is used to import from Outlook and then to Evolution. There should be no reason why you can't use both Thunderbird Evolution in Linux to do the same thing..
