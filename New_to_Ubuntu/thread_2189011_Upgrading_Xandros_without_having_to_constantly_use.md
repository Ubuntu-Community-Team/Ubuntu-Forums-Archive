### Post by buzzingrobot on 2013-11-20
You will need to replace Xandros with another Linux distribution rather than 'upgrade'.  Xandros has been defunct for a long time.

Yes, you can and should install a new Linux to the hard drive.  Most Linux distributions, however, allow their installation images to be run in 'live' mode from a USB stick or DVD.  This is intended to allow you to evaluate a distribution before commiting to the actual installation.

### Post by Topsiho on 2013-11-20
xandros is a different version of Linux, so "upgrading" is not possible, in the sense of downloading a bunch of updates and installing them.
As already said you have to do a reinstall of Ubuntu, and as your netbook is probably not capable to run Ubuntu, you have to go for a light version, Lubuntu or Xubuntu, if you want an Ubuntu derivative.

After downloading usually you have to burn an IMAGE of the iso file on a CD (creating the liveCD), and so on, but I think your netbook does not have a CD-player to run the CD.
The (a?) way out is to make a bootable USB-stick with the iso-file on it, and install from there, same way as from a liveCD.
In the ubuntu's creating such a USB-stick is very easy, but in Xandros and Windows, I don't know how to proceed.
Maybe you know someone living near to you who uses Ubuntu and can help you.

The USB-stick should be 2 GB or more, the harddisk in your netbook probably more than 8 GB. If the harddisk is that small you probably can let the installer do the partitioning.

Succes,

Topsiho
