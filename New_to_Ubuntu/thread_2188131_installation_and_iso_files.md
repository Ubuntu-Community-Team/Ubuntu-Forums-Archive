### Post by chris_coulson2 on 2013-11-15
It sounds like you were attempting to run an Ubuntu installation from within an active Windows environment?

If you're burning a disc, you shouldn't be extracting anything at all. The download mirrors from Ubuntu ARE the .iso files and should be approximately 700MB in size, depending upon your application. From within a Windows environment, yes, you would download the .iso file (~700MB), and follow the steps you mentioned. You'd then REBOOT the computer, enter the boot menu, and boot from the disc you just created.

### Post by bcbc on 2013-11-15
You have the .ISO extension registered with your WinZip software (or similar) so the Icon makes it look like a zip file, but it's not (right-click and select Properties and it will show you the full name and extension). Don't extract it and you can burn it.
