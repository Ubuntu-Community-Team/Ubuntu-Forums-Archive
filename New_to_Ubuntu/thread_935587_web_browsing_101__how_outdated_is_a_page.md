### Post by niteshifter on 2008-10-02
Hi,

I believe you mean:
```
link:somedomain.com/path/to/page
```

```
link://abcdefg
```
returns pages that contain the pattern "link abcdefg", link:abcdefg returns a list of sites that link to the site abcdefg. 

Google [cheat sheet]("http://www.google.com/help/cheatsheet.html").

### Post by DrMega on 2008-10-02
You can right-click on the page to bring up the context menu, and in there you will find either 'Properties' or 'Page properties' (Can't remember and not at my Ubuntu machine). That will bring up a dialog with such info as when the page was last updated. It's far from fool proof, but it is one tool at your disposal.
