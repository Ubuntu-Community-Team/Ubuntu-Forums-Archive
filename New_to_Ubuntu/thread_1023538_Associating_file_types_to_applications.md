### Post by Ms_Angel_D on 2008-12-28
I personally am Not familiar with that file type or the application, however I would think it would be as simple as discovering the command used to run the application

For example right click on the applications menu -> select edit menus and find the program you want to associate, highlight it and click properties, the command should be listed there, copy it. or even leave it open..

Then go to the file you wish to associate, right click on it and select properties -> Open With -> Add

At the bottom select Use custom command and paste the command there.

That should do the trick.
