It's actually really easy.

On the K800i go into: Settings -> Connectivity -> USB -> USB Internet -> Turn On.

Make sure to choose which connection you wish to use.

This has worked perfectly fine for me in windows and ubuntu 7.04, 7.10 & 8.04 with the USB cable.
