### Post by newbiecan on 2011-10-16
Here's a little account of my issue if it can help anyone. I'm a newbie.  I installed Xubuntu 11.10. I also installed with synaptics the xfce print gui, but I don't know if that altered any of the subsystem or not.  I plug in my USB Ml-1640 (low cost samsung laser). It's recognized and the driver is installed. I change only one option which is set toner save for all jobs. Then I go to Firefox and try printing. It prints flawlessly. Then I go to a second page and print. The job appears in the print queue, the led blinks on my printer as if it's receiving data, but nothing comes out and the blinking stops. The queue indicates that the job has printed.  I realized that if I shut off then turn back on the printer, the printing will work again, once. So I go to the printer settings and DISABLE the power saving feature. Now it seems I can print many jobs in a row. Weird heh? Anyways, I haven't tested much but I think that did it.  Thank you.
