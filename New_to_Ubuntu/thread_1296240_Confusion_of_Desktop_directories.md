### Post by mcduck on 2009-10-20
> **running_rabbit07 said:**
> I just threw an idea into Ubuntu Brainstorm on making customizable folder icons such as you can do in Windows.

You can replace any file's, directory's or program's icon with any image file you want already... Just right-click the item, select "properties" and either drag&drop the image you want to use to the icon shown in the top left corner, or click the icon and browse to find the image you want to use.

(This even works for removable devices, for example I've replaced my iPod's icon with image of correct iPod model & color. Gnome will automatically use that icon for my iPod, while all others get the icon specified in the icon theme.)
