### Post by Windows Nerd on 2010-04-25
You *can* sync music to an iPhone, it just depends what firmware version you have and if you are willing to jailbreak your iPhone. iPod nanos work fine with many media applications such as Amarok, Banshee, Gtkpod, ect. I use a iPod nano 3rd gen with Banshee and it works fine. 

To sync your iPod Touch/iPhone (with firmware versions 1.x or 2.x) using Amarok or Gtkpod, go [here]("https://help.ubuntu.com/community/PortableDevices/iPhone")

Another, simpler-looking guide (though slightly outdated, and may work depending on which version of Ubuntu you are using) [here]("http://lifehacker.com/388785/sync-your-iphone-wirelessly-in-linux")

A much simpler yet guide using Karmic [here]("http://www.webupd8.org/2010/01/easy-way-to-sync-your-iphone-with.html"), and provides a workaround for firmware 3.x.

Note that these guides use Terminal ALOT, so you want to be completely comfortable with it and using a Command Line Interface. All of them require jailbreaking your iPhone/iTouch, and knowledge of how to use all of the features of one. I found all of these with a quick Google search, so, in the future, try that first before asking here. 

Hope it works for you!

Scott
