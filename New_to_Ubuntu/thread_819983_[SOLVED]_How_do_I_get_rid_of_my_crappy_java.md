### Post by iaculallad on 2008-06-05
Open up your firefox and on the address bar, type this address-> **about:plugins** (that would be a colon (about.colon.plugins), not smiley) and press enter. Locate the crappy java applet and try visiting this [site]("http://plugindoc.mozdev.org/faqs/uninstall.html") on how to manually remove it.
