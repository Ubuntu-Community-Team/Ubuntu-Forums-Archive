### Post by uberlube on 2008-07-28
1.You can set up a dual boot with ubuntu and windows but make sure that you derfragment your HD before attempting to resize it to make room for ubuntu. Once you have this done you will be given the option to boot into either OS at startup.
2. You can choose between partitions yes, but dual hard drives might be a little trickier.
3.You can run some windows programs with Wine ( a windows compatibility layer) Their home site has a app database which shows you which programs work with it and how well.
4.Yes that is quite sad, lol. But yes you can install a dial up client from the repos.
