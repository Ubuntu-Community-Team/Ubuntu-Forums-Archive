I guess Chris is not very well acquainted with desktop environment. Maybe he has been working on console or if he has been using a GUI, he didn't get to use all it's features.

@Chris: Operating systems used to come with CLI (Command Line Interface) way back in time, where you type commands on a black/blue screen and they run in text mode. But that's history. Now operating systems (well most of them if not all) use a GUI (graphical user interface) where you can use mouse and graphical actions to run different commands. Windows has its own desktop environment, Mac has it's own. But when it comes to Linus, there are several different desktop environments available. Ubuntu has its derivative Ubuntu, Kubuntu and Xubuntu which feature GNOME, KDE and Xfce respectively.
To have a detailed study on "Desktop" visit this
[http://en.wikipedia.org/wiki/Desktop_environment](http://en.wikipedia.org/wiki/Desktop_environment)
