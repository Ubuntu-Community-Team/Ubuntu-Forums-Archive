### Post by ddecator on 2010-04-24
I use Compix and setup Grid (within Compiz) so that I can use hotkeys for this feature. For example, if I use Super(the Windows key)+Left, then the current window covers the left half of the screen. I also set some up to cover the top/bottom halves, and the four corners. Doing it this way is faster than dragging the window, and easier in my opinion.
