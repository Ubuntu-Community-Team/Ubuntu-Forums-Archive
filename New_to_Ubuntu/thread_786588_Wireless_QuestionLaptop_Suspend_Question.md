### Post by luisromangz on 2008-05-08
1) If you connect using KNetworkManager, KWallet (KDE's password manager) should ask you if you allow KNetworkManager to use the wallet to store the password.

2) If you click in the battery icon in the tray area, a dialog appear in which you can choose the action launched when the laptop's lid is closed.
