### Post by buzzingrobot on 2014-10-19
The Nouveau driver is the open source Nvidia driver, which is installed and used by default when Ubuntu detects the Nvidia card.. It's generally considered good enough for everyday use, but not sufficient for serious gaming.  For that, you need one of Nvidia's proprietary drivers, which are distributed as (in)famous binary blobs of code without source. They handle games better, but if something goes wrong everyone waits for Nvidia to fix it.
