### Post by Landngroove on 2013-03-03
I am new to Ubuntu, so please bear with me. Setting up Thunderbird Ubuntu 12.04., When I send an email, I get message: "The message could not be sent using SMTP server mail.myfairpoint.net for an unknown reason. Please verify that your SMTP server settings are correct and try again, or contact your network administrator." I am running Ubuntu alongside Windows XP. I compared the outgoing server settings, and the only difference I see is that in Connection Security in Ubuntu Thunderbird it is STARTTLS. In Windows XP Connection Security the setting is none. So, how can I change this STARTTLS in Ubuntu Thunderbird ?  Thanks.

---

### Post by steeldriver on 2013-03-03
It should be the same as in the Windows version of Thunderbird - go to Account Settings, right at the bottom you should see Outgoing Server (SMTP) - select the appropriate server and then 'Edit', you should get a 'Connection Security' dropdown there
