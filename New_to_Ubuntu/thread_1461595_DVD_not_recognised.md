### Post by ttshivers on 2010-04-24
Ah.  I think the problem is that you do not have the necessary codecs installed to play the DVD.  To install the codecs, first open the synaptic Package Manager.  To open it, click on system, administration, then synaptic package manager.  When you have done that, click on settings and then on repositories.  Next, check all the boxes under the text "Downloadable from the Internet".  Then, close all those windows and open the Update Manager.  To open the update manager, click system, administration, and update manager.  When that is open, click on check for updates.  Install all the updates that appear and it should work.

If that doesnt work, then install a program called vlc.  To do this, open the ubuntu software center by clicking on applications and then ubuntu software center.  Next type vlc in the search box.  A program called VLC media player should come up.  Click on the arrow next to it.  Now, click the button called intstall.  

This thread might also help you:  [http://ubuntuforums.org/showthread.php?t=627993](http://ubuntuforums.org/showthread.php?t=627993)
