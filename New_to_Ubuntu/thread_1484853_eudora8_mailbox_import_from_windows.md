Therefore I was keen to try out Eudora8 (Thunderbird with  new skin). Having installed it OK (I know it is a bit flakey in places still) I wanted to import all the email from my win version that is on another PC. Unfortunately I can find no instructions for doing this. Considering doing it manually poses two problems. One is that the Eudora8 mailboxes appear to have no extension so I am not sure if they are in the same format as the win '.mbx' files and I guess the Eudora8 '.msf' files are the equivalent of the Win '.toc' files but again a different format.

The other issue is that in Eudora8 mailboxes are associated with a specific personality(account). They are not in Eudora for windows.

Has anyone faced these issues and worked out how to resolve them please?

OK I have worked it out...doh..... sorry to have troubled people but I will explain in case it helps others:

It seems that one can just copy the win .mbx files into the /home/user/.thunderbird/[random nos]default/mail/local folders/[folder name]  

This seems to resolve both issues - a .msf file is created automatically and Local Folders are not associated with any single user. Hope this may help others, but remember I'm a newbie at this!
