### Post by jimmy the saint on 2009-02-26
Deluge for two reasons. 
1) Blacklist support.  This is useful since there is no peerguardian for linux.  We have moblocker, but it can be a headache if you make a mistake configuring it.

2) Seperate gui and application.  This means you can install it on two computers and control the torrent downloading box with another.  For example, I have my home server download all my torrent, but I control it with my main box.  It isn't a cheesy web interface, the actual gui connects to the applicaton on the other box and you can't tell the difference.  That means that my main box doesn't have a performance hit because it isn't managing hundreds of connections.

Deluge all the way.  You have to use the current versions from their repos though, not the one in the Ubuntu repos if you want these features.
