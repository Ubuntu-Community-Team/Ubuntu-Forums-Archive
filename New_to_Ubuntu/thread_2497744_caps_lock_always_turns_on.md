### Post by TheFu on 2024-05-16
The CAPS LOCK initial setting  is in the computer hardware BIOS.  That's controlled BEFORE any OS is booted.

I think there are tiny applications that can toggle the state, but I've never needed to use them after I found the setting in the BIOS on any computer in the last 30 yrs.
