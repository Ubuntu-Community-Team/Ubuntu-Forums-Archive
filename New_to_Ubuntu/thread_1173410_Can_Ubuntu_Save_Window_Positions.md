Someone mentioned that compiz can store window positions, this is absolutely true, but how do you find the positions of the windows you ask?

xwininfo, how do you use it?

```

seotechd@EdenII:~/Downloads/apathy/Apathy$ xwininfo

xwininfo: Please select the window about which you
          would like information by clicking the
          mouse in that window.

xwininfo: Window id: 0x6600067 "Buddy List"

  Absolute upper-left X:  960
  Absolute upper-left Y:  55
  Relative upper-left X:  960
  Relative upper-left Y:  55
  Width: 314
  Height: 915
  Depth: 24
  Visual: 0x23
  Visual Class: TrueColor
  Border width: 0
  Class: InputOutput
  Colormap: 0x20 (installed)
  Bit Gravity State: NorthWestGravity

  -geometry 314x915-6-54

seotechd@EdenII:~/Downloads/apathy/Apathy$ 

```just type it then click the window when you have it where you want it.


once you have the info, hit alt + f2 then type CCSM go down to your window rules and find place windows.  In the place windows plugin, click fixed window placement, input your newly discovered window information and VIOLA, window will always appear there, you can even specify which workspace you want it on.
