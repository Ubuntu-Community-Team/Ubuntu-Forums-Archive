### Post by Wharf Rat on 2008-04-27
I upgraded to 8.04 on my Thinkpad T-61.  It was pretty painless.
However,  I found I had a really bad internet connection at home (<10K B/S that is 9600 baud - yeah Comcast). The online upgrade indicated 29 hrs! to download.  Instead I went to work and downloaded the Alt-CD via bittorrent.

Even following instructions it took a couple of hours to get upgraded.  It appears even though you have the Alt-CD, 8.04 still needs to pull quite a bit of files from the net.   In fact while I waited for my upgrade to complete (at work), I had time to use the 8.04 desktop CD to install on a desktop PC from scratch in less time.

But, I am VERY pleased and impressed with 8.04.

Issues I encountered:
1. Wireless drivers would not connect using WPA2 security.  I had to remove and reinstall the drivers. Yes, they are proprietary for an Atheros card.  They worked fine as long as no security measures were used.  A re-install fixed it.

2. Sound is off a bit.  Software controls don't totally control my sound. Neither does my wireless keyboard.  But, the buttons on the laptop work just fine. I have been Jammin with Exaile all day today.
 

3. Firefox 3 is a bit flakey. It works but.....

4. I had to install Java from the repository so Moneydance would work.

Not many other problems that I can note.

Pros:
My enhanced graphics now work!  No cube, but I can flip between desktops and the widows look cool.

My hat is off to the development team that can roll out something this complex and make it so nice, simple and solid.  Thanks guys.
