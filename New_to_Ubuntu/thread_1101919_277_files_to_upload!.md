### Post by leonardo_neo on 2009-03-20
Well you should do all the updates provided.

The fresh install doesn't have updates, so over the period of time they get accumulated and become huge.

If you wish to save some of your downloading time, then you can avoid installing old kernel version, as there has be 2 updates of kernel ( as far as I know) since it was released. You can avoid downloading and installing the kernel version 2.6.27-7.

Rest all you should download and install.
