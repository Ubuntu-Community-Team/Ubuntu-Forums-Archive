### Post by blackpointer on 2020-06-18
Hi!

Ary you sure you really need an antivirus? Antivirus programs aren't generally needed on Linux, the exception being if you often interact with Windows users/machines, and want to make sure your files are scanned for windows virus before sharing, or if you're running a mail server.

The most often recommended free antivirus for Linux is Clamav. It is in the repos and probably in the gnome software centre.
