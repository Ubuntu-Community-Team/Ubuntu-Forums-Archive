### Post by Paqman on 2011-02-07
> **D-Train said:**
> if I go under System>Administration there is no Software Sources listed there like it should be

On recent versions of Ubuntu it's been taken out of the menu. You can find it at System > Admin > Synaptic > Settings > Repositories.
