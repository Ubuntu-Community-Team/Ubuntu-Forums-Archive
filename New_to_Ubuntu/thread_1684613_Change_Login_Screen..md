### Post by skymera on 2011-02-09
Hi, the simplest and easiest way for you to understand is to use Ubuntu Tweak:

[https://launchpad.net/~tualatrix/+archive/ppa](https://launchpad.net/~tualatrix/+archive/ppa)

Install that and run Ubuntu Tweak, there is a whole bunch of settings for the Login Screen, one of which allows you to change background image.
