### Post by mörgæs on 2010-05-18
You should not copy the ISO itself to the CD.

The ISO is an image of a CD. Think of it as a zip-file, the content of which should go to the CD.

Don't worry about the details. The CD burner program should be able to recognise an ISO and suggest "create CD from image" or something similar.


EDIT: The file ubuntu-8.04.4-desktop-i386.iso is 700 MB according to [http://releases.ubuntu.com/hardy/](http://releases.ubuntu.com/hardy/)
