### Post by trinitydan on 2010-12-19
I can offer a little advice.  For starters, the Windows games.  Some of them may work in Wine better than others, you can check the Wine software database and experiment around with that.  A common source of frustration for new users however is trying to run too many Windows applications.  I did it myself when I first started using Linux.  I spent a great deal of time trying to figure out what Windows programs I could get to run in Linux that would have been better spent familiarizing myself with open source software alternatives.  There are some good Linux native first person shooter games you can play on line.  Check out Urban Terror and Alien Arena.  For Windows games you can't do without that don't work in Wine, you may want to try a Windows installation in VirtualBox, but this will require a Windows installation disk.  This is also (from what I've heard) a good method of getting iTunes to run in Linux to sync iPods and other necessary tasks.  

What type of .mp3 players are you trying to sync?
