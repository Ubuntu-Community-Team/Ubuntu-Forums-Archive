### Post by diablo75 on 2009-03-02
Welcome!

There is a firewall already included with Ubuntu called iptables, and by default it does not have any open ports listening for inbound connections.  If you'd like a GUI interface to manually open or block ports for your own purposes, you can install Firestarter (Applications>Add/Remove>search for firestarter).

You don't need any antivirus software for Linux, but you can install it if you wish to do your Windows using friends the courtesy of filtering emails you might forward on to them.  For example, you might get an email containing a Windows virus, be completely unaffected by it, but forward it to someone else who is then a victim.  You might also be using your computer as a serve among Windows machines and it could be used to scan those networked PCs, but that kind of situation is not common.

A popular antivirus program available in Applications>Add/Remove is called ClamAV.  Change your search filter to "all available applications" and do a search for Clam and you should be able to find it, check it off, and apply it to your system.
