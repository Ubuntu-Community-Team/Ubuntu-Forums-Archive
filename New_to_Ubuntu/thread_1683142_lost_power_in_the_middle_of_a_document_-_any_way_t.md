### Post by ankspo71 on 2011-02-07
Hi,
Not sure if this is going to help any, but... 
Some editors have an automatic backup feature already enabled so the editor automatically creates a backup document for you. You can try looking for a hidden document with the same filename in the same directory your document was in. Hopefully there will be a backup document there.

I just checked my OpenOffice Writer options and the automatic document backup feature is there, but it is disabled by default on my Kubuntu 10.10. But I have seen these features enabled by default in Gedit and Kate once before though. Edit: Sorry, I just installed Gedit and I can see the feature is not enabled by default anymore.

Hope this helps.
