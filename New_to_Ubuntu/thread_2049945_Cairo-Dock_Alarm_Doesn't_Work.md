### Post by fabounet on 2012-08-30
Actually there is a new way to add an alarm in Cairo-dock:
simply click on the clock to display the calendar.
then double-click on the day you want to set the task

You can set the task to repeat each week/month/year.
The applet will warn you 15mn before, and if it's a birthday, 1 day before too. :-)
