### Post by NewJack on 2008-11-24
Sorry for seeing this so late, but....

So what you are saying is this:

1- You installed Ubuntu 8.04.1
2- After installing Hardy, you put the Edubuntu Add-On CD in you cd tray and used the installer to add the Edubuntu Packages

If you did those 2 things, you should have restarted and it should have booted into the Edubuntu Desktop after Logging In.  You should find an Applications Menu entry of "Education", which is where you will find stuff such as GComprix, Tux Typing, etc.

If you go to System>Preferences>Appearance, you will find the Edubuntu/Ubuntu Wallpapers, Themes, etc.

Hope that helped.
