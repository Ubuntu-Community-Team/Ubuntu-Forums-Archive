### Post by Defrector on 2010-06-11
For running script from keyboard shortcut

K-Menu->System Settings -> Input Actions.
In Input Actions left pane, near the bottom:
Edit->New->Global Shortcut->Command/URL
Use the tabs to fill it out
Hit apply BEFORE doing anything else (otherwise it will clear it)
