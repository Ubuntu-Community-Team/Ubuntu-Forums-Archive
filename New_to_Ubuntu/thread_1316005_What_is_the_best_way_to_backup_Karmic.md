### Post by Schendje on 2009-11-05
"Back in Time" is a back-up application.

Ubuntu One has 2 gigabyte (more if you go paid). If you remove your Ubuntu install, the files in the Ubuntu One folder will still be available. If you delete the contents in the folder while Ubuntu is still on though, they'll be removed from U1. (In other words, delete Ubuntu or the Ubuntu One folder while you are *disconnected* from Ubuntu One and there won't be any changes to the contents.)

Also, are you sure you have to reinstall? Maybe we can still help out with the problem.
