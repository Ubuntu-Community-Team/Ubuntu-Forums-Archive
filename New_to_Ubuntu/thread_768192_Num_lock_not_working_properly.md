### Post by bluefrog on 2008-05-13

uncheck the box.

numpad will work again with figures and will not move the mouse

### Post by bigsuccess on 2010-02-22
Thanks for this thread guys. I had random non-working numpad for a few days, couldn't figure it out at all.

After finding this thread I did (ubuntu) System > Assistive Technologies > Keyboard Accessibility > Mouse Keys (tab) > un-ticked "Pointer can be controlled using the keypad"

I know I didn't turn that on so I then went to the Accessibility (tab) > and un-ticked "Accessibility features can be toggled with keyboard shortcuts"

just in case.. anyway that did the trick for me - think it's the same as previously mentioned but a different way of getting there.

Cheers guys/gals!
