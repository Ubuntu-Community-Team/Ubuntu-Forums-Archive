### Post by jawilljr2 on 2013-09-28
IMO the best way to watch Netflix is to use [PipeLight.]("http://fds-team.de/cms/articles/2013-08/pipelight-using-silverlight-in-linux-browsers.html")

What Pipelight does is adds SilverLight as an addon to a Linux browser. So there is no need to use Windows' version of FireFox.  You can also use Linuxes version of Chrome if you want.

[Installation instructions.]("http://www.webupd8.org/2013/08/pipelight-use-silverlight-in-your-linux.html")

Just make sure that you close all browsers before installing PipeLight.

[PipeLight FAQ's]("https://answers.launchpad.net/pipelight/+faqs")
