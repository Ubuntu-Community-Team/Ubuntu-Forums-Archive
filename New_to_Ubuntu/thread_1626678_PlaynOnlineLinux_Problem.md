### Post by WienerWuerstel on 2010-11-20
Hi, 
it's PlayOnLinux and not "PlayOnlineLinux". How do you download the Game? PlayOnLinux is a Wine Frontend that helps you install various Windows Games. You insert the CD of the Game you want to install, select the Game Install Profil on PlayOnLinux and follow the Instructions. If the Game is not supported you just use Wine.
