### Post by stchman on 2009-04-03
> **frogdude said:**
> Is there a way to open .rar with an ubuntu applcation? Or do I have to manually go and download winrar and then, will WINE run that application?

Yes, File Roller just needs some packages and it will be able to create and unpack .rar and .7z archives.

```

sudo apt-get -y install rar unrar p7zip p7zip-full

```

If you wish to rar and unrar archives within WINE you will probably need to install WinRar in WINE.   Since Ubuntu supports them, there is no need.
