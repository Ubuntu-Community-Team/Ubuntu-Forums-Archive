### Post by bubba_169 on 2008-07-03
The way I did it was to edit xorg.conf...

run this in a terminal:
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.bak
gksu gedit /etc/X11/xorg.conf
```

In the gedit window that comes up should be your xserver configuration file. Look for a section titled "Screen" then add this line just under all the other lines in this section before endsection...

```
DefaultDepth    16
```

Save this, log out then in again and you should be in 16bit...
To reverse change the 16 to a 24 or run this command:

```
sudo cp /etc/X11/xorg.bak /etc/X11/xorg.conf
```

Another way you'd go about it with an Intel card would be to use 915resolution
```
sudo apt-get install 915resolution
```
Then find which mod your using with
```
sudo 915resolution -l
```
Then to change it use the following arguments <mod_number resolution depth>
(example):
```
sudo 915resolution **34** **1024x768** **16**
```

To keep it permenant I think you'll have to add it to your startup scripts.
