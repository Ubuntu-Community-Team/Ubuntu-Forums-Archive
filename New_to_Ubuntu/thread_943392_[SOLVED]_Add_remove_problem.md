### Post by Orange_and_Green on 2008-10-10
There is no package called "attantik" in the repositories. Are you sure you didn't mean "atlantik" (with an "L") instead? (It is described as a KDE-based client for Monopoly-like online games). If that is what you want, try
```
sudo apt-get install atlantik
```

NOTE: if you are running Kubuntu, everything is OK. If you are running the "regular" (GNOME-based) Ubuntu, please note that you will have to install KDE support as well. If you run the command above, you will be prompted for this.
