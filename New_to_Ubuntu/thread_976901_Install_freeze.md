### Post by Bölvaður on 2008-11-09
the alternate disk does not change the functionality at all. The only difference is _how_ it is installed. Instead of gui (graphical user interface with windows and mouse) you will get a command line installer which is the most solid and least ram consuming installer for linux (I belief).

So you might want to get the alternate disk for ubuntu. Xubuntu is very nice but as you are asking for ubuntu, I have to recommend the alternate disk of ubuntu: [http://www.ubuntu.com/getubuntu/downloadmirrors#alternate]("http://www.ubuntu.com/getubuntu/downloadmirrors#alternate")

I would think the freeze could be because of 3 different problems.
[LIST=0]
[*]Too little ram
[*]Error on cd
[*]Too slow cd drive
[/LIST]

You can bypass the second and third one by burning the cd on low speed (like 4x &#8594; 8x). And plug in faster cd drive during the installation (in my experience this is mostly only problem on very old computers, probably rare problem).
Oh and the freeze could actually be just some slow down time. I have seen a guy having to wait for few hours for the install to return from freeze (thread on this forum).
