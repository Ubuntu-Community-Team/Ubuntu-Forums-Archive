### Post by Mark Phelps on 2011-11-21
I ran into a similarly confusing problem when I first started using my latest PC, which dual-booted Windows 7 and Ubuntu 11.04.

When I booted into Windows, the local network was OK and internet access was OK.

When I booted into Ubuntu, it complained of no network connection.

Turned out there was a problem with the onboard Realtek network chip which was being automatically disabled when Windows shut down.

I had to find a run a Windows-based Realtek diagnostic utility -- which allowed me to change the default behavior.

After that, the network chip no longer shutdown when I rebooted.
