### Post by blueridgedog on 2009-05-23

### Post by mike555 on 2009-05-23
If your talking about the text on a webpage in Firefox there is a great add-on called;  "Plain text to link" , that lets you highlight text and automatically save it to a text file ,along with the date and URL ........... 
  [https://addons.mozilla.org/en-US/firefox/addon/623](https://addons.mozilla.org/en-US/firefox/addon/623)
