### Post by NightCrawler03X on 2008-06-27
There is no need to use ePSXe under wine. There's a native linux port of ePSXe.

If the reason for you using epsxe under wine is that it's too hard to configure under linux, give my installer a try:

[http://ubuntuforums.org/showthread.php?t=550304](http://ubuntuforums.org/showthread.php?t=550304)

It downloads ePSXe and all of it's plugins, before automatically sorting everything into the correct folders. It makes installing ePSXe so much easier, but do note that it does not configure any of the plugins (it just puts everything in the right place). Once you've installed everything correctly, configuring the plugins is a breeze, so that's not a problem.

It doesn't download any bios images (for obvious reasons), but those are pretty easy to find anyway.
