### Post by whitesmith on 2013-10-31
To add to **newb85**'s comment, hibernate not only doesn't work with all hardware, it sometimes screws up in Really Ugly Ways on some hardware. I would stick with suspend, particularly if you use an SSD. In that case the speed delta between hibernate and suspend isn't significant--and I have never known suspend to cause damage. Cheers!
