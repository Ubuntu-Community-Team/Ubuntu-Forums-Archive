### Post by desperado665 on 2009-11-06
Be advised that Kmymoney is an application for the KDE (kubuntu) desktop. It should run on GNOME, but you will have to install a bunch of KDE libs to get it to work. I try to avoid running cross platform applications. Usually any program that starts with k is a KDE version. 

You can get GNUcash from [www.getdeb.net](www.getdeb.net)
