### Post by ExileAmongYou on 2011-05-17
I'm not all that familiar with KDE, but I don't think you should be disabling Dolphin. Nautilus, the file manager for Gnome, is also used to manage the desktop, so if for some reason you uninstalled it, your desktop would disappear (and it's quite likely you'd get other problems). I suspect that similar things might happen if you were to get rid of Dolphin.

When you have to find a file at the moment, how are you doing it? I suspect you might be using Dolphin without realizing it every time you open a folder.
