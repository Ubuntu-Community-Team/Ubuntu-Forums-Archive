I attempted to download Gnucash from the internet onto my new Ubuntu 8.4 and was advised by the computer that the software was available through the software channels.  So I went to Add/Remove and downloaded the Gnucash application with no problem.  It seems to be working fine but, everytime I press I click "Help" I get told that Gnucash.docs, the manual is not available.

### Post by tonyn999 on 2008-07-03
Felicity, The documentation you are looking for comes in a separate package.  

The easiest way to install it is from the command line:
apt-get gnucash-docs 

This should do it, or go to
System -> Administration -> Synaptic Package Manager
and search for gnucash

You will see the docs package, install it.

Cheers!
