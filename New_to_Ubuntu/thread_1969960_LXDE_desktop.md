### Post by Peripheral Visionary on 2012-04-30
Or if it's just the [LXDE]("http://lxde.org") desktop environment you want, rather than the whole Lubuntu software bundle, just

```
sudo apt-get install LXDE
```

Then log off and choose LXDE (or Lubuntu if you installed "lubuntu-desktop") from the *Session* menu.
