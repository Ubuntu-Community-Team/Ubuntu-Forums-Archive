### Post by puntigamer on 2013-01-09
I like dual panel file managers more, but the default file manager "Nautilus" is pretty good for everyday tasks. You will see all your drives, can attach drives on servers via SSH or FTP. The 3.6 version of Nautilus is still a bit buggy in Ubuntu 12.10, but the default 3.4 is quite stable ;)

### Post by Lars Noodén on 2013-01-09
I'll second the recommendation to try out a live CD.  While you've got it running, you can also install some of the other file managers and see which one you like best.  The main ones are Nautilus, Thunar, Dolphin and PCManFM.  You can install them in your live CD session and then they all go away when you've ended the live session.
