### Post by jeremychristopher on 2011-05-22
Gnome is one of the 2 major desktop environments in Linux. The counterpart of it is KDE. The operating systems Ubuntu, Debian, and BSD use the Gnome Desktop Environment. The operating systems Mandriva, Kubuntu, and PCLinuxOS use the KDE Desktop environment. Most Gnome environments have 2 taskbars, one at the top and one at the bottom. While the KDE is similar to the Windows Desktop environment. One taskbar located at the bottom most of the time. 

You can find out more about Gnome from here: [http://en.wikipedia.org/wiki/GNOME](http://en.wikipedia.org/wiki/GNOME)

### Post by dFlyer on 2011-05-22
A few nice to know commands:

cp is copy
mv is move and also used to rename a file
free -tom will display memory
df -H will display disc space
rm -rf file.name will remove a file (should never be run as superuser sudo, unless you know what your doing as you can wipe a drive clean)
sh can start a scrip
ls -ahl will list all file is a folder
locate file.name will find a file

Under linux Name is different from name as compared to windows where name Name NAME is all the same.

### Post by kaldor on 2011-05-22
You use OS X. Linux and OS X are both UNIX-like OS's, so many things under the hood will be familiar. If you're familiar with OS X's terminal, you will find that most terminal commands in Linux are identical.

A desktop environment (DE) is, in a nutshell, a library full of applications that make up what you use. The "UI" software such as the panels, icons, artwork, desktop, make up GNOME. The default GNOME applications (Gedit, Nautilus, etc) make up the integrated experience. You get a consistent desktop.
