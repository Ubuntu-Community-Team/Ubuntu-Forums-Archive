Hi, you got an answer for this yesterday on [this]("http://ubuntuforums.org/showthread.php?t=1534124") thread.

You don't "open" TinyOS. It's a highly specialised operating system for running wireless sensor networks. If you want a graphical tool to start writing applications for TinyOS, then there's plugins available for the Eclipse integrated development environment.
