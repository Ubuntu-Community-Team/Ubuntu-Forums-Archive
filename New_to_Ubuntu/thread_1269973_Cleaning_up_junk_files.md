### Post by jocko on 2009-09-19
> **humphreybc said:**
> 1) When I do a fresh install, obviously I won't have all my original software. When I copy over my old home folder, does it have a list of software that I had installed and will Ubuntu download all the packages of the old software? Does it do this after you restart or run apt-get update?
No. You'll have to install everything manually. But you could use this command:
```
dpkg --get-selections > ~/selections
```to give you a list of all installed packages (in the file "selections" in your home directory).
After the reinstall you can use this to install everything that was installed:
```
sudo apt-get update
sudo dpkg --set-selections < ~/selections
sudo apt-get -u dselect-upgrade
```You can also save a list of installed packages from synaptic, and have synaptic to install everything from that list (I don't know what these options are called in english, but the straight translation from the swedish version is "save markings" and "read markings". I guess it is "save selections" and "read selections" in the english version)> **humphreybc said:**
> 2) Will my panel, background, font, appearance settings all be restored?Yes, all of that is stored in your home directory.

> **humphreybc said:**
> 3) What about my proxy settings?
I don't know, but I guess proxy settings are stored in the network settings, which are system-wide. That means they are stored somewhere outside of the home directory (probably in /etc). So they will not be restored by copying your home.

> **humphreybc said:**
> 4) What will happen to programs that I didn't install through the repositories, such as Google Earth, Skype, Picasa, Google Chrome etc...
You will need to download and install them yourself. If you got them from 3rd-party repos, just add those repos to your software sources before you run the "dpkg --set-selections" command or corresponding synaptic procedure...

> **humphreybc said:**
> 5) My wine and wine-doors set up is very complicated and messy, but it works... what will happen to this? Will I have to re-install wine and Office 2007 and Photoshop/Dreamweaver etc?All of that is in your home directory too, so it should still be there. You will of course need to install wine and wine-doors (but my answer to your first question should have taken care of that).

> **humphreybc said:**
> 6) What about my virtualbox Windows XP computer?virtualbox stores the virtual machines in your home directory (unless you specify another path), so it should work fine once virtualbox is reinstalled.

> **humphreybc said:**
> 7) Are there any real benefits associated with doing a complete fresh install when each release comes out, instead of just upgrading?
It shouldn't really matter, but you may miss a few new things if you just upgrade compared to a complete reinstall. As an example: if you installed jaunty with an ext3 file system (which is the default choice for jaunty and all earlier ubuntu versions), an upgrade to karmic when it's released will still use the ext3 file system, while a fresh install of karmic will give you an ext4 file system (which has some nice performance improvements over ext3).
Karmic will also have grub2 as default boot loader, but I'm not sure if an upgrade from jaunty will replace grub1.
