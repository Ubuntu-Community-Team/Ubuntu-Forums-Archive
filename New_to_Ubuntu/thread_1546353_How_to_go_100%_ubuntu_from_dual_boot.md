### Post by TBABill on 2010-08-05
No benefit to deleting Windows since it does not interact with ubuntu on a dual boot. You just gain hard drive space, but if you have enough already there is no advantage. Keeping it could come in handy in the future if your Ubuntu is ever borked and you need something working immediately, or if you need access to something Windows specific. 
 
Well, I guess there is one real gain. You won't have to keep all your security stuff up to date in Windows for just the few events you might need or want to use it. Having both is not a bad thing...has more to do with your ideals. And if you don't have Windows on a backup CD/DVD I would recommend keeping it (or making those disks now!).

### Post by papibe on 2010-08-05
It depends on what kind of work you do, or going to do. It may exist a killer app that is essential to the field you are interested in. For example: Pro Tools for sound recording engineering, Photoshop for graphic designing, or the windows DirectX platform for a game developer.

I know, I know... all are VERY arguable examples, but I'm trying to get a higher vantage point here.

If you're new to Ubuntu, I would recommend to wait a little bit so you get used to it. This way you'll be more prepared for any eventuality.

That being said, I also would suggest to commit to Ubuntu, and:

[LIST=1]
[*]Shrink your windows partition to minimal functional size (backup first!).
[*]Learn how of use wine if in case you need to use a simple Window software (e.g. utorrent).
[*]Optional & Advance: install VirtualBox and create a virtal Windows machine.
[/LIST]

There's a disadvantage to the dual boot option: you have to boot Windows every once in a while to update it, so when you REALLY need to go there you don't have to apply dozens of Windows/Firewall/Anti-virus updates.
