### Post by SunnyRabbiera on 2008-05-14
I dont think what you are asking for is possible, yet.
The desktop cube is still young and so is compiz.
But virtual desktops in general dont usually work this way, virtual desktops are not really desktops of their own, no they are on the same single desktop but you can put different things in each one.
The main purpose of virtual desktops is simple: to save room in the taskbar, in windows it is so easy to clutter the taskbar so that is why virtual desktops come in handy.
what you are asking is to make the virtual desktop behave like you actually have four different desktops, but this is not the actual case.
I think some reading is ahead of you on how virtual desktops work:
[http://en.wikipedia.org/wiki/Virtual_desktop]("http://en.wikipedia.org/wiki/Virtual_desktop")
