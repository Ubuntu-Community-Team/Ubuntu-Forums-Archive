### Post by Elfy on 2010-09-30
I'd say it was this 

/gwibber-_dailey_/ppa/ubuntu/dists/lucid/main/binary-i386/Packages.gz 

Go to software sources - in the sys - admin menu

Other Software tab - find the line for this ppa and edit it to read _daily_

Reload - you might get an error about a key. Go to wherever you got the ppa and get the key information

Then run update manager again - check - see what it says now

You could also delete the line then re-add it - [https://launchpad.net/~gwibber-daily/+archive/ppa](https://launchpad.net/~gwibber-daily/+archive/ppa)
