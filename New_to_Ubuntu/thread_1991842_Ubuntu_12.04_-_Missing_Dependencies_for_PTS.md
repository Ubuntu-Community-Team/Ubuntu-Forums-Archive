### Post by lykeion on 2012-06-24
Better late than never, here's my answer:
phoronix-test-suite is available in the ubuntu repositories so you can easily install it using Ubuntu Software Center, Synaptic, or apt-get from command-line.
If you wish to try the latest version of PTS you can download and install a Debian/Ubuntu package from the official site of PTS: [http://www.phoronix-test-suite.com/?k=downloads](http://www.phoronix-test-suite.com/?k=downloads)
