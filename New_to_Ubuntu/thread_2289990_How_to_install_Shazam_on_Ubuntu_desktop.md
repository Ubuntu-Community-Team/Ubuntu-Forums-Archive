### Post by grahammechanical on 2015-08-08
You should really be asking about Android apps on an Android forum.

Keep in mind that not all apps are created equal and that applies to every computer platform. Apps written to run on a phone or tablet form factor device do not scale well on a desktop monitor unless the developer has done some work to make the app give a good user experience on different form factors. How many of these hypothetical 100 apps have been written to use touch input and have no facility for mouse or keyboard input?

I will give an two examples of what I am taking about.

Ubuntu Web Browser (Browser) is the web browser on the Ubuntu phone. It is also part of the default installation on Ubuntu since the release of 14.04. The first versions did not have scroll bars and the mouse had to be used to do swipe gestures. The latest versions have scroll bars and tabbed browsing which the early version also did not have.

Music app is also a Ubuntu phone app that will be installed on a Ubuntu desktop if we install unity8 session. The Music app is, in my opinion, a great little music app but it will play audio files that are stored in the Ubuntu partition but not in some other partition. It does not have a feature for importing music from another partition.

Developers have to write apps that can convert to different form factors and input methods depending on the device they are running on.
