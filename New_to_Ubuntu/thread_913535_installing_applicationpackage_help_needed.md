### Post by Xiong Chiamiov on 2008-09-07
More specifically, you'll probably want to be able to close the terminal window afterwards, meaning you should either run
```

xtrkcad&

```
or use the run dialog (alt+f2?).

There is an excellent guide to installing things on Ubuntu [here]("http://209.85.141.104/search?q=cache:IfbspYjsGWIJ:www.monkeyblog.org/ubuntu/installing.html+how+to+install+anything+in+ubuntu&hl=en&ct=clnk&cd=1&gl=us").
