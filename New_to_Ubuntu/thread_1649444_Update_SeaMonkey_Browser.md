### Post by JBAlaska on 2010-12-20
SeaMonkey is a all-in-one Internet application suite, Containing an Internet browser, email & newsgroup client with an included web feed reader, HTML editor, IRC chat and web development tools.

If you just want a faster lighter browser, you can try chromium as cariboo907 mentioned, although for some reason I can't put my finger on, chromium just leaves me...cold for lack of a better explanation.

If you want to try a lightweight browser that's more erm conventional, give Midori a try, it's also on the repo's. I've used it on a computer with very low memory (256meg) and it works very well.
