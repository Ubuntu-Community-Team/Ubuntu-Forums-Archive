### Post by mikeyphi on 2010-02-21

> I would like to install Ubuntu on my system using a dual boot up. I have partioned my hard drive and would like to know if when I load Ubuntu, can I direct it to load into this new partion?

Yes
During the Install process you will reach the Partitioner section and get asked where you want to install - do not use the default option, select 'manual' and, on the following screens, select where you want to install. *CAREFUL* if you make a mistake you can overwrite data!
