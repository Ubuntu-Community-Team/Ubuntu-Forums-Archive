### Post by Impavidus on 2016-10-17
When you install Ubuntu, it comes with a number of themes and icon packs. But you can install more. It also comes with some applications, but you can install more from the repositories and third-party applications from other sources. I think the IT guy at your workplace added some applications he thought useful and some themes/icon packs he liked.

32 bit or 64 bit makes little difference for the available software. 32 bits software runs on a 64 bit system too and there isn't much software available in 64 bit only. But Google Chrome is one, so you can't run Chrome on a 32 bit Ubuntu system.
