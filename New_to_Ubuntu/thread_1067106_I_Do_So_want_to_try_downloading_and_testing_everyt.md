### Post by dca on 2009-02-11
Hmmm, *nix enviroments (Unix, Linux, BSD) all started off at the command line.  Start by learning basic commands in terminal (bash shell)
 
cd = change directory
ls = list directory
mv = move something
mkdir = create directory
 
...then move on to what's applied on top: x-server (graphics), gnome (desktop environment), package management (Synaptic), etc, etc
