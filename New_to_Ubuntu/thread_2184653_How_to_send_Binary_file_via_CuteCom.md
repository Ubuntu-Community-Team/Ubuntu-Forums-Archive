### Post by Zill on 2013-10-30
K.F.Lee:  I can't give any definitive advice on this one as it depends on what you are communicating with but would suggest you experiment with the various options.  Hopefully, you know what baud rate, data and stop bits to use, along with the appropriate parity and handshaking.

I would try with the plain file option and hex input set, but you may be able to use, for example, XModem if you have problems with end-of-line characters.  Just try various options and see what works. ;-)
