### Post by cariboo on 2008-10-25
You may be better off installing Ubuntu on it's own partition, you will probably get better performance, than running inside XP. You specs are a little on the low end for Ubuntu, you may see better performance, by installing a lighter weight desktop, either search System-->Administration-->Synaptic Package Manager for xubuntu-desktop or you can install it from the command line:

```
sudo apt-get install xubuntu-desktop
```

To select a different desktop, click on the sessions icon in the lower left corner of the Gnome log in screen.
