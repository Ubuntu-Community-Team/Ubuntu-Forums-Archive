### Post by Holger_Gehrke on 2023-07-20
It should. Any program trying to control media playback will go through MPRIS (Media Player Remote Interface Specification). sayonara is MPRIS compatible according to [this]("https://www.linuxlinks.com/sayonara-player-small-clear-fast-audio-player/7/"). But Media Player Indicator isn't at the address listed in the article anymore. Search for it on the Gnome Extensions site (or find a newer extension that does this, there are quite a few).
