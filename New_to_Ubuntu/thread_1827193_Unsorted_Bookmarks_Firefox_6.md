### Post by lovinglinux on 2011-08-17
The Unsorted Bookmarks exists since Firefox 3.0. It hasn't change on Firefox 6.

It is the place where Firefox saves bookmarks, if you do not select a specific folder. This happens when you just click the star icon in the address bar once. If you click the star icon twice, you can select where to save the bookmark.

If you want to hide it, use [Sidebar Bookmark Selector]("https://addons.mozilla.org/en-US/firefox/addon/sidebar-bookmark-selector/") extension.
