Since you say, "just did a bunch of package installations including firefox extentions", my first thought is to go to Tools > Add-ons > Extensions and try one by one disabling each of the new extensions to see if one or more is the culprit!

You'll have to restart Firefox after each change by going to File > Quit and then restart Firefox.

If you have no luck with that please include a list of packages you just installed.
