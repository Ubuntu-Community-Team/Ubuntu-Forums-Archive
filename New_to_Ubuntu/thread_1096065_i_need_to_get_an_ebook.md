### Post by mvalviar on 2009-03-14
You may use fbreader as well. On the terminal (Application>Accesories>Terminal) type 
```
sudo apt-get install fbreader
```
You will be propted for you password. Type it in and wait for the installation to finish. To use it launch it from Application>Office>E-book reader
