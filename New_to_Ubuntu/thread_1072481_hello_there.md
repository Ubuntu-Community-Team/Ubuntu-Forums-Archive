### Post by taurus on 2009-02-17
I believe jdownloader requires you to have a working java (sun version) on your machine before you can run it.  So, have you installed sun-java6-jre on your machine?

Applications -> Accessories -> Terminal
```
sudo apt-get update
sudo apt-get install sun-java6-bin sun-java6-jre sun-java6-plugin
```
When you get to the license screen, press the Tab key to highlight the <OK> and Return to accept it.
