### Post by Znupi on 2008-09-12
My 2 cents:

You don't need a book. Really. I didn't use a book and I can pretty much say I know my way around Ubuntu. Do you feel like reading hundreds of pages or experimenting with your new OS? Thought so :). Plus.. you have Google ;)
If I were you, I'd do a reinstall of Ubuntu (format everything). Get a fresh start. I would hate finding leftovers from the previous owners. Also, you'd get the latest version of Ubuntu, with the most support, drivers, features, etc etc.
Some advice:
When you want to do something you don't know how, always check [the docs](http://help.ubuntu.com), especially the [community docs](https://help.ubuntu.com/community/). They offer a great deal of help. Only after checking those out, ask your question here.
Installing / Removing applications is usually done through **Applications -> Add/Remove...**, you can find tons of applications there. Also note that Ubuntu, by default, comes with lots of great applications, for office, IM, e-mail, music, videos, etc.
To get codecs for most "un-free" music/video formats (like mp3, etc), windows fonts (Arial and the like), DVD codecs and all such things, go to **Applications -> Add/Remove...** and install **Ubuntu restricted extras**. You might have to choose **All available applications** under **Show:**.
Usually it's a good idea to install restricted drivers for your graphics card (for 3D acceleration and stuff, especially if you want desktop effects -- the cube, wobbly windows, etc. Although I keep those off because they slow down my system when I start opening lots of applications).
Themeing. I always use [gnome-look.org](http://gnome-look.org/) for new themes, and [deviantart.com](http://deviantart.com) for wallpapers. The gnome-look.org site can be a bit confusing (it has a lot of sections!) -- just go to the [GTK 2.x](http://gnome-look.org/index.php?xcontentmode=100) section and download some themes. Don't unpack them! Leave them archived and go to **System -> Preferences -> Appearance**, click on **Install...** and point it to the downloaded archive.
I think that just about covers it up.

/My 2 cents

Glad you chose Linux! Maybe you'll convince that wife of yours to use it, too :)

### Post by AnLGP on 2008-09-12
use the terminal.

it might seem daunting at first but it is the linux power house.  if you know the terminal in one version of linux then it's pretty safe to say it will be similar in other areas.

the most important thing you can do is if you have a question, ask.

the second most important thing you can do is be patient.

the third most important thing to do is learn the "man" command.

to do this open a terminal and type in man <whatever> where whatever is what you want a manual on.  

so for example i'll use 

```
man cmus
```

and hit ENTER.

A summary of what cmus is and the commands and flags (example: -v) are will pop up.  It's a wonderful thing, the terminal.

Here's some pages that you might find useful:

Given that you've never used Ubuntu before, only Windows:
[http://doc.ubuntu.com/ubuntu/switching/go01.html](http://doc.ubuntu.com/ubuntu/switching/go01.html)

And since you're buying a used Computer, the default software might be different, 
but here's some applications that will probably be installed (and their windows' equivalents):
[http://doc.ubuntu.com/ubuntu/switching/applications-equivalents.html](http://doc.ubuntu.com/ubuntu/switching/applications-equivalents.html)

It's also very easy to install new programs, though it's rather different, like this:
[https://help.ubuntu.com/8.04/add-applications/C/gnome-app-install.html](https://help.ubuntu.com/8.04/add-applications/C/gnome-app-install.html)

When dealing with System modifications, you need super user privileges, so check this out:
[https://help.ubuntu.com/8.04/administrative/C/](https://help.ubuntu.com/8.04/administrative/C/)

One last thing, 
Ubuntu is much more secure than Windows (ie. no exploits in the wild that target Linux), so antivirus software isn't really needed; though updates are of course a good idea:
[https://help.ubuntu.com/8.04/keeping-safe/C/updates.html](https://help.ubuntu.com/8.04/keeping-safe/C/updates.html)

Edit: You mentioned watching videos etc., so this might be of value as well:
[https://help.ubuntu.com/8.04/musicvideophotos/C/codecs.html](https://help.ubuntu.com/8.04/musicvideophotos/C/codecs.html)
