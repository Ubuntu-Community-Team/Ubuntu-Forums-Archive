### Post by JerichoKru on 2010-02-16
[http://ubuntu-tweak.com/source/ubuntu-tweak-stable/](http://ubuntu-tweak.com/source/ubuntu-tweak-stable/)

Add ubuntu tweak to your software sources and install it.

Then open it, go to Source Center, unlock it, and enable the "Firefox Stable Channel Packages".

Refresh it, and it should say there are updates for firefox.  Install them and you now have 3.6.

**Edit:**

Or this more simple way:

[http://ubuntu-tweak.com/source/mozillateam-firefox-stable/](http://ubuntu-tweak.com/source/mozillateam-firefox-stable/)
