### Post by deadflowr on 2012-07-03
By default Ubuntu does not install "non-free" codecs. You'll need to install them yourself. I believe the package is called Ubuntu-restricted-Extras, be sure to say yes to any EULA statements that come up, as saying no can hamper the installation. This package installs "non-free" codecs, such as mp3, and flash.

---

### Post by Merrattic on 2012-07-03
By cli:
```
sudo apt-get install ubuntu-restricted-extras
```
