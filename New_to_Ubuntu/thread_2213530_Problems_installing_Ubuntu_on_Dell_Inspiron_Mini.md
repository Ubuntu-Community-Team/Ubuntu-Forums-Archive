### Post by kc1di on 2014-03-27
There must be a bios setting on that machine that allows you to boot form USB drive.  I'm not sure how you access it on that machine but you should be able to hit an f key or esc or something similar to take you to the bios setup page. once there make sure it's set to boot usb drives and make sure usb drive is first to boot.  

Ok did a little research and it seems either the fn+f2 key or the fn+f12 key should be able to get you to the bios setup.  when the machine first boots hit the keys until you see the bios set up screen.  once there you should see a tab or menu entry for boot order.  Good luck.
