### Post by forger on 2008-07-07
Sure, 32-bit ubuntu works on 64-bit, though not with the same speed.
If you have more than 4gb of ram and want to use all of it efficiently, I suggest to use the 64-bit version.

The 32-bit also has its advantages, e.g. better support for java and flash which are 32-bit :)
