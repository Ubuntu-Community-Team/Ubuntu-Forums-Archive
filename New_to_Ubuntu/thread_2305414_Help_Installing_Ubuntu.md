### Post by Vladlenin5000 on 2015-12-05
You probably downloaded the correct file already (just in case check again in the website above). However, if you did it from Windows -and- you have WinRAR installed, then the icon is the same because of the file type association created by that software. It's still an ISO file though and you DON'T just extract it. You're supposed to use a CD/DVD burning tool - there are hundreds, free and commercial - to burn a DVD directly from the ISO file by using the proper option. The nomenclature varies depending on the software you're using.

Even better and faster, use a USB flash drive: [http://www.ubuntu.com/download/desktop/create-a-usb-stick-on-windows](http://www.ubuntu.com/download/desktop/create-a-usb-stick-on-windows) and also [https://rufus.akeo.ie/](https://rufus.akeo.ie/)
