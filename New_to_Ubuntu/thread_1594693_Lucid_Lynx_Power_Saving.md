### Post by admiralspark on 2010-10-12
This is probably related to some bugs that've popped up concerning the power management settings. I found that the only way I could disable it is to go into Screensaver --> un-tick the box next to "Enable screensaver when inactive". In power options, make sure after all your settigns are changed that you click the "make default" button as well. And if this doesn't work, it may need to be run as root (I don't remember the command ATM).
