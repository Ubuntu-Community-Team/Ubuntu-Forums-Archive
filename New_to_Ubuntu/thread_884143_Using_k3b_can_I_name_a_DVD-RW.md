### Post by bobnutfield on 2008-08-08
That is how K3b has always saved the filename for me as well.  However, for backups to DVD, I create a folder with the name of the back-up data (i.e. backup08_08_08), then I put the data I want to back-up into that folder.  The written disc is now named the title of the folder I dragged into k3b.  The data can be anythng you want it to be, including other folders.
