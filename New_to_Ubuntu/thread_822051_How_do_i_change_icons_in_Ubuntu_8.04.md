### Post by Golem XIV on 2008-06-07
Open System -> Preferences -> Appearance, drag the .tar.gz icon package over it.  It will ask you if you want to use the new icon set; confirm and you're set.

### Post by ibuclaw on 2008-06-07
Right-Click on the Desktop and select "**Change Desktop Background**"

In the new window, enter the "Theme" tab and click "Install..."

Locate the gunzipped icons file (ie: icons.tar.gz) then open it and apply it.

Lastly, Run the command ```
killall gnome-panel
``` to show all changes.
