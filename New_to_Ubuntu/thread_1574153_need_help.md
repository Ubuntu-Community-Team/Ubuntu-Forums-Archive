### Post by Rasa1111 on 2010-09-13
You need to turn on metacity compositing manager,

You can do this by using alt+F2, and using gconf-editor,
or you can use Ubuntu tweak, if you have it installed. 

If you need help using gconf-editor, let me know.

 Compiz (visual effects) are not needed for AWN.

Setting visual effects to 'none' is one of the first things i make sure is done on a new install. No need.
