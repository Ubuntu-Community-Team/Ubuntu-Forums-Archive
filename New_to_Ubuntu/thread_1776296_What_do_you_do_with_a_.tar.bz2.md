### Post by alphacrucis2 on 2011-06-05
> **TAspr said:**
> I need to addition to the software I have been using.  What is a .tar.bz2 and how do you install it?

It's a compressed archive like a zip or rar file. You extract the contents to a folder and then you should find a readme file that specifies how to install it. It may require compiling if it contains source files but the readme file should explain that. If you need to compile you will probably need to install the build-essential package from the Ubuntu repos first. BTW you should be careful that you obtained this archive from a source you can trust.
