### Post by Schendje on 2009-10-31
Hi, and welcome to Linux! :D

Personally I use Bluefish, regular old Gedit and gFTP for my web development. Another tip is to use the awesome amazingly wonderful Firefox extensions Firebug (but you will probably also know that one, since it's also available on Windows!).

However, Bluefish isn't a WYSIWYG editor, so you might find some useful info here: [http://www.micahcarrick.com/09-28-2007/web-development-linux.html](http://www.micahcarrick.com/09-28-2007/web-development-linux.html)

Also, Adobe and Macromedia are both companies. I think you mean Adobe Dreamweaver, which is an application made by Macromedia, until Adobe bought them. :)

### Post by ken22 on 2009-10-31
Personally I use a text editor like vim for small projects. For large enterprise software I use the eclipse ide with it's web tools. 

I like eclipse because it's support for Java/J2EE which I use in my web applications is superb. If you're looking to maintain flash based sites on Linux, I don't know what to tell you.

For basic html/php/script pages, a newer web authoring program known as bluefish looks promising. I think that'd be worth looking at.
