### Post by TBABill on 2011-04-17
You don't really install "from" another distro. You install from LiveCD, LiveDVD or LiveUSB. If you cannot boot into the live session then you likely have a bad download (errors) or a bad burn to whatever media you are using. You need to verify your iso downloaded properly by confirming the md5sum matches and then burn to DVD or CD as slowly as possible to prevent errors.

If you already did those things, please clarify the problem further.

### Post by XubuRoxMySox on 2011-04-18
If you like what you have and just want to "test drive" another distro, there's no need to install anything!

When you burn the Ubuntu iso to a CD, make sure you have burned it as an *image*, not as a data disk. Linux Mint uses Brasero I think, and it has the option in the menu to burn a file as an image.

Now reboot your 'puter with the new LiveCD in the tray (door closed), and during bootup hit whatever key lets you select boot options. You'll want to select "boot from CD-ROM."

It will boot into Ubuntu and you can explore Ubuntu without having to install anything. Running from a LiveCD is slower than running from an installed system, so don't judge the speed too harshly if you're running from a LiveCD.

Kick the tires, drive it around, see if everything works. No need to install anything unless you like it and want to retain any changes you make.

Enjoy!
