### Post by Oropher8598 on 2009-10-31
Maybe you're using a slow mirror? The 'Software Sources' app can choose a faster one for you.
[list]
[*]Open 'Software Sources' (*System -> Administration -> Software Sources*)
[*]On the 'Ubuntu Software' tab, open the 'Download From' dropdown and select 'Other...'
[*]In the 'Choose a Download Server' window that appears, click 'Select Best Server'. It'll ping a load of mirrors to find the fastest.
[/list]
