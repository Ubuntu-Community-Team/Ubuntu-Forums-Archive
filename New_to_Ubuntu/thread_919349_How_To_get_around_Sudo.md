### Post by p_quarles on 2008-09-13
This topic is frowned upon here. The guiding principle here is that changing the way that root privileges are handled opens up big problems with security and stability, so should not be supported here.

Furthermore, there is plenty of information on this subject to be found via an easy Google search. While we do not normally tell people to "Google it" in order to find the answer to their questions, this is an exception. Going against the security model chosen by the Ubuntu developers should only be done by those willing to do the research and risk the consequences.

Here's a thorough description of our policy:
[http://ubuntuforums.org/showthread.php?t=765414](http://ubuntuforums.org/showthread.php?t=765414)
