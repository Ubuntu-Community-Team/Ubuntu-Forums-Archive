### Post by ayenack on 2009-09-01
If I'm understanding you want Open Office to open the files. If this is the case then right click on the file/s then select **Properties** and the **Open with** and select Open Office. It's that simple.

BTW. Once you've associated a certain type of file let's say .doc all other .doc files will open with Open Office. So you don't have to click on every individual .doc file to associate them. Same applies for other file types to.
