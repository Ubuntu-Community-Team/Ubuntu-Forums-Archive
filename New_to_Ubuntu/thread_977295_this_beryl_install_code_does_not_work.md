### Post by Paqman on 2008-11-10
Compiz Fusion has replaced Beryl, and is installed by default on all versions of Ubuntu from 7.04 (Feisty Fawn). So if you're using that version or later, then you already have Compiz installed.

To enable desktop effects go to System > Prefs > Appearance > Visual Effects and select what level of effects you want.
