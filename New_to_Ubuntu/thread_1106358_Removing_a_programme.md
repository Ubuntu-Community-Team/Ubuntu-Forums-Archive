"Everybody learns Linux in a hard way" :)

Please don't be depressed, after some time you will understand this OS, and then you will start liking it. When I started using Linux, I had to format my hard disk 7-8 times because of various problems in installations......but gradually I learned this OS, and now I like it like anything.

So have patience....we are here to help you.

First, why do you want to remove SCIM?? If it is not bothering you, then why don't you leave it on it's own? Remember, never uninstall something which you don't know what is it used for! 

*So I strongly recommend you please do not uninstall this.*

Still, if you wish to remove this, then open "Synaptic package manager" (System--->Administration----->Synaptic package manager)

On top of the page, you will find a "quick search" box. Type scim in that box.

You will then find some package list starting with scim. The one which are with 'gray' colored box, are the packages installed on your system. Find the scim package with gray box, and then right click on that and then select "mark for removal"......Then press the 'apply' button on top. And this package will be uninstalled. 

Hope this helps.

P.S. Again, I recommend, do not remove this package.
