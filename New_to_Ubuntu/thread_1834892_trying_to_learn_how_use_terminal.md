### Post by TeoBigusGeekus on 2011-08-28
```
cd -
```
is the equivalent of pressing Back in Windows Explorer: it will get you to your previously visited folder. If you haven't visited any, then it will give you this message.

For a good introduction to the command line:
[http://sourceforge.net/projects/linuxcommand/files/TLCL/09.12/TLCL-09.12.pdf/download]("http://sourceforge.net/projects/linuxcommand/files/TLCL/09.12/TLCL-09.12.pdf/download")
