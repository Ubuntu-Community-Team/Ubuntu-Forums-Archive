### Post by Nepherte on 2008-11-13
You also need the java browser plugin. If you're using ubuntu 32bit:
```
sudo apt-get install sun-java6-plugin
```

If you're using 64bit:
```
sudo apt-get install icedtea-gcjwebplugin
```
