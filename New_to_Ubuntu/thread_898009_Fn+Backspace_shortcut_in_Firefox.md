### Post by kaibob on 2008-08-22
> **Sii said:**
> Hi again,

On my Windows laptop, I frequently use the Fn+Backspace keyboard shortcut to go back pages in Firefox.  This is particularly handy when navigating 'Unread posts' in forums.  Is there a way to recreate this shortcut in Ubuntu?

On a PC running Windows and Firefox, the backspace key takes you back a page in history. You can configure Firefox on a Linux box to do the same thing by changing the value of the following about:config entry to 0 (thats a zero):

browser.backspace_action

I assume this would also work on a laptop with the Fn-Backspace key combination, although I don't have a laptop to try it out.
