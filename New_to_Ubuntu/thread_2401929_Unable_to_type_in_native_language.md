### Post by idkwhatimdoing1.0 on 2018-09-24
Hi there! I see you have done the correct steps from my knowledge. I would suggest opening up the terminal writing 
> sudo apt update
sudo apt upgrade 
to see is the language pack has to be updated or installed. Do make sure that in the input sources box the hebrew language is there. the top selection is the default and the other under are the other possible keyboard languages. To switch the language simply press Ctrl+WIndow Key+SPACE and then select the Hebrew language.
