### Post by twright on 2008-04-27
if you are running ubuntu 8.04 you can use Brasero. just open it and click disk copy. select to write to a file image and then you can burn that to another dvd

### Post by PhoenixP3K on 2008-04-27
Have a look at **k9copy**. It's able to take the content of a DVD, a resize it to an image file that can fit on a 4.7GB DVD, you can then burn that image using Brasero or the default burning software that starts up when you put a blank DVD inside Ubuntu.
