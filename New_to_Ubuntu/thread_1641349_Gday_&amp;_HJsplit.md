if you don't mind command line stuff, theirs lxsplit.

if you want graphical, do this.
```

sudo apt-get install wine
```

go to where you downloaded hjsplit for windows, right click on hjsplit (make sure its the standalone version), and select properties.

In the permissions tab, choose "is executable" (or something similar, im using KDE, not gnome).

click "ok" and double click on the program.
