### Post by feelshift on 2009-04-15
Basically you need two partition for installing Ubuntu: one to install it and one for swap. You can resize your current windows partition if you have no empty space on your hard disk, then make a partition ext3 and set it's mount point "/" and a small one (mine is 1GB) swap and set it's mount point to "swap" (all this can be done from Ubuntu's installer). ATTENTION NOT TO DELETE YOU WINDOWS PARTITION AND MAKE SURE TO BACKUP ALL YOUR IMPORTANT DATA BEFORE YOU TRY THIS. Then proceed installing Ubuntu. If all works out nicely, you should have GRUB installed when you restart you computer after the installation is complete.

If you have any question, please ask ;)
