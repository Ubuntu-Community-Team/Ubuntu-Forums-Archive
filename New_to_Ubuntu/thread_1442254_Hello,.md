### Post by Rayve on 2010-03-30
Grant121,

You can either install Ubuntu "inside" Windows using Wubi, but this can sometimes be unstable.  It's a good way to test drive Ubuntu, but I suggest test-driving with just the LiveCD and choosing "Try Ubuntu but make no changes" at the starting screen.  

If you like Ubuntu and have the room, you can then do a true dual-boot, which means installing Ubuntu and Windows side-by-side.

Make sure you back up any files, just in case, and then defrag in Windows.  Then, once you have the LiveCD working, you can choose the "install Ubuntu" option.

There are several posts on this forum to help you with installing Ubuntu beside Windows [not inside, but beside] so be sure to do some reading before you get started.  :)  Here are some links: [http://ubuntuforums.org/showthread.php?p=5004671#post5004671](http://ubuntuforums.org/showthread.php?p=5004671#post5004671)

Good luck, and welcome to the community!

Edited to add: You cannot have both Windows and Ubuntu running at the same time unless you do a virtual terminal, but you can *install* them on the same hard drive.  You'll just need to restart if you want to boot out of one and into the other.

### Post by TriBlox6432 on 2010-03-30
mkay,

So get some DVD's, Flash Drives, or Online Storage space and back up all your data, just in case.

Go to Start> Search Box> Type: Disk> Click on disk Defragmentor> Run it.

Then make sure you have your CD/DVD/Flash Drive that you are going to boot Ubuntu off of.  Inset it.  Restart your computer, enter your BIOS, and make sure it is set to boot from either CD/DVD or Flash Drive first.  Save Changes and quit.  Restart.  Click "Try Ubuntu without Changing my Computer".  Play around with it, and make sure you really want it.  Then on the desktop, double click the icon that says "Install Ubuntu".  Follow the steps and when you get to partitioning, drag the slider to where you want it.  How big you want your Ubuntu, and Windows partitions.  Then enter all other applicable data like your name, etc. and click install.

When it's done, it will tell you to remove your CD/DVD or Flash Drive and reboot.  Then you should be able to chose between Win7 and Ubuntu at start up.  ;)
