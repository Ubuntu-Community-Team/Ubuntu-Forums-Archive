### Post by RequinB4 on 2008-06-11
The default GNOME Image viewer will show only the first frame.  Pretty much any other program you view it with will show you what you want.  If you just want to look at it, you can open it with the gthumb image viewer.  If you want to edit the gif, GIMP will do fine (view playback in the last tab on the top toolbar --> animation) to check how you're doing so far.
