### Post by CatKiller on 2019-11-07
To help you find what you're looking for, what you are trying to do is set the default device.

Other desktop environments allow you to do this from the normal audio settings, but Gnome is rather limited. I understand that in Gnome you'll need to install one of the pulseaudio helper utilities - I can't remember which one - to set the default device.
