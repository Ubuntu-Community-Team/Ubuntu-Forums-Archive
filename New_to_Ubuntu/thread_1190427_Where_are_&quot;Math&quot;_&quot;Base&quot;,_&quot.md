### Post by Therion on 2009-06-17
You have to add them separately but they're in Add/Remove.  You can install the OpenOffice.org Office Suite if you want to get everything in one fell swoop.  Just use the "Office" filter in Add/Remove.

If you want just the separate packages from Open Office, "Math" is now "Formula", "Base" is called "Database" and "Cash" I'm not familiar with.  In more ways than one!

### Post by Cheesemill on 2009-06-17
You can edit your menus by going to System > Preferences > Main Menu

### Post by Ragnar Bocephus on 2009-06-17
For whatever reason the menu item for OpenOffice Formula is not enabled by default.  Right-click on the Ubuntu icon in the upper left corner of the top panel, go to Edit Menus, then click on Office in the left pane.  You should be able to check the box for it in the right pane and get it to show up.
