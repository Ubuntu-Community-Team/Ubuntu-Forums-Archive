### Post by JRV on 2012-04-30
You need to install ubuntu-tweak:
```

sudo add-apt-repository ppa:tualatrix/ppa
sudo apt-get update
sudo apt-get install ubuntu-tweak

```

Run the program.
Click Tweaks=>Window
It's right on top.
