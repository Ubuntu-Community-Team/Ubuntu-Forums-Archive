### Post by seventhsamurai on 2010-07-20
I think the best way to upgrade is through Synaptic Package Manager. Search for Ubuntu Studio and mark all upgrades and install.

If you have an installation that is working correctly right now, this is the method I would suggest.

I had Ubuntu 10.04 Lucid running on my machine. I reformatted and a did a fresh install of Ubuntu Studio. To my shock, a lot of stuff wasn't working properly. It wouldn't detect my network, there were issues with my sound card, some USB ports werent working.

So I reformatted again, and installed Lucid, then upgraded to Ubuntu Studio through synaptic package manager. It works flawlessly now.
