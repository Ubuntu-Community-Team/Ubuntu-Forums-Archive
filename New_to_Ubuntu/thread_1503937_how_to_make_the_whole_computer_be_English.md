### Post by germanix on 2010-06-08
Do what "Irony" in the post above already told you. 
Go to-  System > Admin > Language support 
There you will see the window showing which languages are already installed on your computer. 
You should see the language Japanese as you already installed this and you should also see the language English. If the English language is not there, you must install it first by clicking the button "add a language" then choosing the language from the list and install it. (american english or british english as you wish)
If the English language is already installed or after you have installed it, move the english language on your list in the window to the top position (by click and drag) then click the button "use systemwide"
Then click on the tab "text" in the same window and change the country for the date/money you want to use and also click the button to use this system wide.
Now re start your computer and the language for your computer and all else will be in english.
