### Post by oregonbob on 2011-04-02
I use Linksys netcams. Each one is a server with it's own IP address. I haven't experimented with high priced stuff. The Linksys WVC80N wireless is a good low cost at around $100 unit with built-in motion detection or you can use something like zoneminder.

I have three Linksys and for the price they do a good job. If you want to use webcams using USB Logitech makes several good models under $100.
