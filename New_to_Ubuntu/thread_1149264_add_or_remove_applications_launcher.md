### Post by coffeeaddict22 on 2009-05-05
It's not that newb.  NOT playing with the system is newb :).

If you right click on the Applications menu, select "Edit Menus", then add an item to the Applications menu.  the Type: is an application; Name: is Add/Remove...; Command: is /usr/bin/gnome-app-install, and FWIW, Comment: is Install and remove applications. 
That should do the trick OK.
