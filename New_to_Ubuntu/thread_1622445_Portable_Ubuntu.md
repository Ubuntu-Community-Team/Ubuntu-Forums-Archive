### Post by sikander3786 on 2010-11-15
And if you need a persistent install, you can use the Startup Disk Creator in Ubuntu under System > Administration. A persistent USB will save your settings, changes, documents and all that stuff even after a reboot.

You can also install Ubuntu to your USB exactly as you install on an internal HDD. Just unplug your internal HDDs and install it to your USB.

### Post by beew on 2010-11-15
Well a live usb with persistence is not really installing into a usb drive. You cannot upgrade kernel or install graphic cards, for example.

Yes, you can have a full installation in a usb drive, but it is slow so if you want a workable portable system you should install in an external hard disk instead. I have done both, an installation on a 8g flash drive was really sluggish but my other installation on an ancient  4 g external hard disk is lightning fast (live usb with persistence is much faster than installation in the same usb because it runs from ram, but it has the limitations I mentioned above: doesn't support system level updates)

Also, pendrivelinux is basically for Windows users. All (or most) of its tutorials only work in Windows. If you already running Ubuntu you can create a live usb with persistence easily by using System > Administration > Start Up Disk Creator
