### Post by emarkd on 2009-01-19
If you're using Compiz it can handle the shortcuts for you.  Run the CompizConfig Settings Manager from System -> Preferences and click on the 'General Options' button.  Select the 'Commands' tab.  Under the 'Commands section, enter the command you want to execute into a field, then under the 'Key Bindings' section, select the key combination you want to use to execute the command.

If you don't have the CCSM already, you may have to install it from Synaptic.

If you're not running Compiz, I got nothing. :)
