### Post by grahammechanical on 2016-04-21
Documentation is mostly written by volunteers. This is especially true of Community documentation. This kind of documentation is usually accurate at the time it was written and is usually useful even if it is out of date in regard to the versions of Ubuntu that are referenced. Ubuntu 16.04 has only been released by a couple of hours and here you are expecting the documentation to have been updated already.

If you download the mini ISO from that Lubuntu link you may find that it is the Ubuntu 16.04 mini ISO. The same core software is used for Ubuntu & all its flavours. It is during installation that we get to choose whether what we are installing becomes Ubuntu, Xubuntu, Lubuntu or one of the others.

This was my experience when I downloaded Xubuntu Core (15.10) the other month. It was actually the Ubuntu mini ISO & I was able to install Xubuntu core which had a very minimal Xfce user interface or Xubuntu desktop which had more applications.

So, well done Lubuntu documentation team for being so quick to update that page.

The promised support given is usually appropriate for the kernel. As you can see from this wiki page it is Linux kernel 4.4 that is in 16.04 that is receiving support for 5 years. 

[https://wiki.ubuntu.com/Kernel/LTSEnablementStack](https://wiki.ubuntu.com/Kernel/LTSEnablementStack)
