### Post by grahammechanical on 2012-11-12
To know if there are any packages that can be updated the Update Manager will download the latest sources lists. This can be something like 120 or more scripts. Every time we check for updates the sources lists are updated. I think that this is what you are seeing and thinking that the updates are being repeated.

On Ubuntu 12.04 and even 12.10 there will not be updates every day. I am running the 13.04 development release. It is actually 12.10 being slowly, bit by bit, changed into 13.04. So, every day, sometimes more than once a day, we can check for updates and find that there is actually stuff to be downloaded. But it is not like that on versions that have already been released.
