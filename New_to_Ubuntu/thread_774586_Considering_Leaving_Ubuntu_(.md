### Post by jimv on 2008-04-29
About the dual monitor issue...it really doesn't work like Windows.  I've been messing with it, and you pretty much either have one big desktop or two separate desktops.  Kinda sad there.


About the xbindkeys thing...you can add either a launcher to the desktop or to your menu.  To add the program to your menu, right click on the menu and click Edit Menus.  Then select the folder you want to add the xbindkeys to and click Add Item.  Name it and then put the command to launch xbindkeys in the Command box.  (Should just be the program name).


I can right-click and paste in my terminal...but I think you want the right-click to be the paste button.  I think you can set that in Edit>Keyboard shortcuts.


About the Allnetic Time Tracker...have you tried using it with Wine?
Just 'sudo apt-get install wine' in a terminal, and then try opening the Windows installer for Time Tracker (just double click the exe file you download).  Actually, I just tried it and it seems to work just fine...I have Time Tracker running in Ubuntu now...including the tray icon.  So give it a shot.


I hope this has been of some help.

Ctrl-C is used to interrupt processes in terminal. Changing it to something else is not a good idea.

You can use Ctrl+Shift+C and Ctrl+Shift+V respectively to copy and paste in the terminal.
