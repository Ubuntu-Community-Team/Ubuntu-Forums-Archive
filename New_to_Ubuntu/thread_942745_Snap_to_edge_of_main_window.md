### Post by tjwoosta on 2008-10-09
first, if you dont already have it, you will have to install compiz config settings manager (CCSM)

```
sudo apt-get install compizconfig-settings-manager
```

then open CCSM by going to (System-Preferences-Advanced Desktop Efects Settings) 

the setting for snapping windows could be in a couple of different places depending on which version of compiz you have


sometimes it is under the wobbly windows settings

and sometimes there is a seperate setting for snapping windows 
(like the screenshot)
