I did the same upgrade on a Acer TM244LC which had been working well for several months. The upgrade went smoothly if slowly, but after the reboot the machine died after a Ubuntu splash screen. I'm guessing that the video drivers or x configuration file were wrong. I tried various boot options without success and in the end did a clean install back to 9.04, as I wanted the machine working again soon. 

Moral - if you do want to do the upgrade, only do it after you are sure that your  machine will run fine from a live CD or equivalent, so that you know that at least all the drivers are OK.

Thanks to the folk who are developing and working on correcting the release, and to Canonical who make it so easily available.
