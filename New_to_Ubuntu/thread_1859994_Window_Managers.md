GDM is not a window manager, it is a display manager; a similar but different job.

You are correct, however, to say that you only run one WM at any one time, so if you only want metacity that is all you need to have.

aewm, amiwm, blackbox, compiz, ctwm, dwm, fluxbox, flwm, icewm, are all window managers, most of which I have never seen, and there many more available in the repos, but if you like what metacity does for you and don't want or need compiz, then fine; get rid of compiz and use just metacity.

### Post by mcduck on 2011-10-15
> **mdavis1231 said:**
> Thanks for the response, you guys.  Although I think it looks wicked, I personally don't have Compiz installed.  OK, what about "Byobu Window Manager"?  I'm not even sure what it's useful for, and if I'm using Metacity, do I need this one also?

Unless you have yourself removed it, then yes, you do have Compiz installed. It's part of the default Ubuntu install.

Byobu isn't actually a window manager, it's used in terminal as an enhancement to *screen* (which is best described as *terminal multiplexer*, allowing you to better multitask with command-line interfaces). [http://unixlab.blogspot.com/2009/11/enhancing-terminals-with-byobu-on.html]("http://unixlab.blogspot.com/2009/11/enhancing-terminals-with-byobu-on.html")
[http://en.wikipedia.org/wiki/GNU_Screen]("http://en.wikipedia.org/wiki/GNU_Screen")

Like 23dornot23d and ajgreeny said, you only use one window manager at a time, so you only need one. However uninstalling the ones that came with Ubuntu isn't going to benefit you much, they really don't take any noticeable amount of disc space.
