### Post by nhasian on 2008-11-26
must have:

[Boxee]("http://www.boxee.tv/") is a free and open source cross-platform media center and entertainment hub with social networking features.
[uShare]("http://sourceforge.net/projects/ushare/") is a UPnP (TM) A/V Media Server to stream audio and videos to Xbox360, PS3, etc 
[VirtualBox]("http://www.virtualbox.org/") is a general-purpose full virtualizer for x86 hardware.  Basically you can install multiple Operating Systems and run them in a window while Ubuntu is running.
[Wine]("http://www.winehq.org/") is a compatibility layer for running Windows programs. 
[DeVeDe]("http://www.rastersoft.com/programas/devede.html/") create video DVDs and CDs (VCD, sVCD or CVD), suitables for home players, from any number of video files.
[Songbird]("http://getsongbird.com/") is a desktop Web player, a digital jukebox and Web browser mash-up.
[Firefox]("http://www.mozilla.com/en-US/firefox/") free and open source web browser currently has 20% marketshare!
[Thunderbird]("http://www.mozilla.com/en-US/thunderbird/") is a free, open source, cross-platform e-mail and news client.
[HellaNZB]("http://www.hellanzb.com/") retrieves nzb files and fully processes them. The goal being to make getting files from Usenet (e.g.: Giganews Newsgroups) as hands-free as possible. Once fully installed, all thats required is moving an nzb file to the queue directory. The rest; fetching, par-checking, un-raring, etc. is taken care of by hellanzb. 
[LottaNZB]("http://www.hellanzb.com/") is a GTK front-end for hellanzb.
[Pads]("http://pads.piesweb.co.uk/") is a TV Show downloader which uses MYEpisodes/Newzbin for use with hellanzb.

Basically once you've setup hellanzb, pads, and told it which tv shows you want to keep up with, it will auto-download the episodes as they become available every week. its like Tivo for your computer but without having to have a tv tuner.  Once you have the videos you can watch them on your big screen tv by streaming them with ushare to your xbox360 or ps3.  Or if your pc is connected to your tv, you can use Boxee.

### Post by scorp123 on 2008-11-26
**gpm** - mouse driver for plain old text console ... can't live without it and it makes using "vi" in pure text environments sooooo much easier. With "gpm" installed you can switch to another terminal, highlight text, switch back to your "vi" session and then hit the right mouse button ... it will copy and paste stuff just as it would in GUI.

**links2** - a text-mode WWW browser.

**vim-full** - the default "vi" version that ships with Ubuntu just plain sucks. **vim-full** adds all the bells and whistles (syntax highlighting! **:syn on** command in "vi") 

Together these packages are quite powerful ... and you need never again be afraid to hose your system, because even if you completely screwed up, with **links2** you'd still be able to surf the web and search for help (just plugin a cable into your Ethernet port, voila), with **gpm** copy & paste between text terminals (Alt+F2, Alt+F3, etc.) is easy (useful if you have to copy code snippets around), and **vim** is more fun to use when the "full" package is installed ....

### Post by jamesrl on 2008-11-27
This has more of a science/programming type bent to it but:

emacs (specifically emacs-snapshot for better font rendering).  This handles all your text editing, especially good for programming/scripting as well as many other features.

kile - a wonderful LaTeX environment.  If you ever want to make latex presentations you also need latex-beamer.

octave.  similar syntax to matlab but open source.  

scipy/pylab/matplotlib - for scientific python.  (Array data structures for quick math, and plotting utilities).

[Note I deliberately skipped ROOT, which I would recommend not using as SciPy is so much better for so many more reasons.]

mathematica (costs money, but is the best (symbolic) math software out there).  wxmaxima is a free alternative, and sagemath is something that tries incorporating octave + maxima + scipy.

And then there's stuff that you have installed that can be tremendously useful you might not know about:

screen - if you ever ssh into another machine, and worry about getting disconnected and losing your session, you need screen, so you can reattach to it on your next login and thats just the beginning of the screen manager (type man screen).

at - Let's you run a command at a specific time (without setting up cron jobs to do it periodically).  E.g., you want to go to sleep at 2am today, you can stop your music player (say exaile) by typing into the terminal: 
```

at 2:00am
exaile --stop
```

python.  This is such a useful language.  Python scripts are easy to read and write, and get to do more complicated useful stuff (while seeming simple).  So much easier to script/program in that Bash/Csh/C++/C

### Post by Xiong Chiamiov on 2008-11-27
My list of "must have" software is probably different than yours, since I do different things than you, but just for kicks:

* Firefox - I recommend Opera for most people, but I can't live without [all my freakin' extensions](http://delicious.com/xiong.chiamiov/addons)

* amarok - best *nix music player, if you prefer features over lightweightness.  Also the only one I've seen that will group my massive compilation collections under "Various Artists" while preserving correct tags.

* dolphin - the kde filemanager.  Tabs, split screen, attached terminal.  The version I had in Kubuntu was buggy as hell, but I don't know how it is now; the latest stable release is quite slick.

* SMPlayer - seems to be the video player that gives me the least troubles.  Enables subs if they exist, and picks up where you last played a file.

* Komodo Edit - not particularly popular, but it has tabs, projects, split-screen, easy code folding, and the ability to edit over FTP.

* [Awesome]("http://awesome.naquadah.org/") - it took a little while to get used to a tiling window manager, but now my workflow is much improved.  Unfortunately, it looks like version 3 isn't in any of the Ubuntu repos yet, even Jaunty.

* GNU screen - allows you to switch between multiple terminal windows in just one window, as well as being able to detach windows, and pick them up later.

* rtorrent - *very* lightweight torrent client, that I keep running on my server all the time

EDIT:
Oh yes, I almost forgot:

* ipython - if you do any work in python, it makes your life so much easier.  Seriously.

I suppose I could also add yaourt, but since it's a wrapper for the package manager for another distro, I don't think that it qualifies...

### Post by capriccio on 2008-12-12
good thread - i'll bite.

gui apps
--------
gnome-do - in my opinion, better than Launchy for Win or Quicksilver for Mac.
firefox - though my enthusiasm for firefox has been waning, I haven't found a better alternative yet.  Opera is zippy but doesn't comes close to firefox with its meager collection of widgets.
inkscape - impressive vector graphics editor; not up to par with Illustrator but for being free, it's damn impressive.
amarok - my favorite music player on any platform followed closely by Winamp 5.3 (anything later is too badly bloated) and then iTunes.
vlc - simple media player.  I only use this with Linux; on Win, I use K Media Player.
keepassx - very useful password manager utility.
truecrypt - I use this to encrypt all my removable media: thumb drives and external HDDs.  Simple to use, minimal overhead, rock solid AES encryption.  Good stuff.
ktorrent - simply the nicest bittorrent client around.  If you can get over the trauma of having to install kde libs--I did, quickly--then this is a no-brainer.  Only slightly more resource-intensive than transmission or deluge and far less so than Vuze/Azureus, this gets my win.  uTorrent is still my choice on Win.  I refuse to run it through Wine however.
blender - very promsing 3d modeling tool which I'm only just starting to get acquainted with.  The controls are complex and the learning curve seems pretty steep, but so far, I do like it.

console apps
------------
emacs - been using this to code with since, forever.  Of course, it's one of the first things I install.
imagemagick - good for quick conversions of images from one format to another using the 'convert' program.
mutt - best console mail program period.  Also useful is its ability to send emails with attachments from the command-line.
sshfs - useful utility which allows you to locally mount a directory on a remote host over ssh (in userspace with FUSE).
lftp - best command-line ftp/sftp/ftps client there is.  Supports command completion with GNU readline.
octave - as another user pointed out, useful if you're a math/sci/engineering guy.  It's basically an OSS version of Matlab.nmap - versatile port scanner; use this to quickly test the security of YOUR remote system.  I shouldn't have to explain why you want to be a white hat hacker. :) 
inadyn - zippy dns updater client for use with free dynamic ip and dns services.
doxygen - documentation generator utility that works on C/C++/Java/PHP code.
links - text-based web browser.  Great if you're in a terminal and just want to quickly browse through some html docs.
