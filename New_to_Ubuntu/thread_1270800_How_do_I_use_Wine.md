### Post by tahitiwibble on 2009-09-20
The first time I ever installed Wine, I found by some miracle that if I inserted a Windows cd into the reader it asked me if I wanted to install the program (just like in Windows), I asked the installer to leave a program icon on the desktop, which it did, and it worked without any further todo.

Since then I have found that if you type "winefile" in the terminal you get a very Windows-like gui that's easy to figure out.

Oh, in the beginning you might need to do 'winecfg" in the terminal.
