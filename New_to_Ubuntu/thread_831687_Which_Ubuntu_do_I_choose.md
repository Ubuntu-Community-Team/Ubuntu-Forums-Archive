### Post by Kevbert on 2008-06-17
Xubuntu is a cut down version of Ubuntu, hence the speed increase, but has all the necessary programs, browser, word processor etc.  There's nothing wrong with the interface.  
With your spec machine Ubuntu should work fine, but if you go for Hardy Heron 8.04 you may want to uninstall compiz-fusion, which gives fancy graphics effects, as it will eat up resources.
If you have the room on your hard disk why not try both.  You'll need at least 4Gb for Xubuntu, 8Gb for Ubuntu and 1Gb for swap (temporary memory storage/paging file).

### Post by mivo on 2008-06-17
Xubuntu is fairly full-featured, and it's not ugly. ;) I actually like some aspects of xfce (the Window Manager) better than of Gnome, and the included software covers my needs better (i.e. I prefer Abiword over the OpenOffice Writer), though that's not very important since apps can be installed easily, and disk space is usually not a problem these days.

Like in all Ubuntu flavours, the restricted multimedia codecs need to be installed manually, so for listening to MP3, watching AVIs or playing enrcypted DVDs, you need to add the Medibuntu repository and install some packages. This is very easy and fast, though. Instructions can be found here: [https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)  Alternatively, I also recommend VLC as your video/DVD player. It's just better than the default media player (Totem) and comes with the codecs, too.

DreamLinux already comes with the codecs out of the box and it's arguably the prettiest xfce-based light distro I have seen so far. It's quite impressive, eye-candy-wise. I still use Ubuntu (desktop) and Xubuntu (laptop) because the community is huge and there are help pages and answers for just about any problem you could ever run into. And of course you can change the appearance. But if I show someone Linux, I usually boot up the DreamLinux (light), Mint or Sabayon Live CDs. :) (Note that running a Live CD is always slower than a real installation since everything has to be loaded from the slow optical medium.) First impressions matter and it won't be before October that Ubuntu gets a make-over for the default looks.
