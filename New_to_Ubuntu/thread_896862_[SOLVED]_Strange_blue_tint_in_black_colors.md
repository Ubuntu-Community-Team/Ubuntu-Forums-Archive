### Post by bawilson2 on 2008-08-21
I'm running 64 bit Hardy with the 2.6.26.2 kernel.  I'm using a Nvidia 8500 video card running the 173.14.12 drivers from the nvidia website.  The problem that I'm seeing is that occasionally the darkest parts of the screen will turn blue instead of black.  If I open the nvidia-settings manager the problem is corrected for a while until it reappears.  I can't figure out what is going on.  Anybody have any suggestions?  I'm at a loss here.

---

### Post by bawilson2 on 2008-09-02
I was able to fix this by installing the 177.68 beta drivers from the nvidia website.  I had tried other drivers in the 173.14.x series but all had the same problem.
