### Post by Impavidus on 2016-10-02
MD5 sums can be found via this site: [https://help.ubuntu.com/community/UbuntuHashes](https://help.ubuntu.com/community/UbuntuHashes)
Specifically, you want to check one from this list: [http://releases.ubuntu.com/16.04.1/MD5SUMS](http://releases.ubuntu.com/16.04.1/MD5SUMS)

You can also try the torrent download. It wil do the check automatically and redownload the corrupted piece, so there's no need to download the entire .iso file again if you have a corrupted download.
