### Post by Eldera on 2009-07-02
Welcome! And prepare for a new adventure. I am what they call a Newbie, since I have only been using Ubuntu since Feb. Two very basic things that I found different from Windows: 

One: things like CDs,DVDs and printers with camera chips in them "mount" themselves when connected and need to be unmounted before being disconnected. They will show an icon on the desktop. Right click the icon and select unmount or eject from the menu that appears.
Printers connected by USB (without camera chips) can just be unplugged.

Two: the command line. This is very useful for a lot of things and there are oodles of info in the "help" sections of Ubuntu. BUT PLEASE READ the information in the blue banner at the top of each sub-forum in this forum about malicious commands before you start playing around with the command line. Access the information in the banner by clicking the little brown box in the right end.

Go for it! Eldera

PS Read the posts in this forum marked "Sticky" They are lots of help.

although compiz comes installed by default, the configuration manager isnt!  from a terminal type:

```
sudo apt-get install compizconfig-settings-manager
```

now you can enable that awesome desktop cube that everyone raves about.  also i highly recommend [Gnome-Do]("http://do.davebsd.com/").  When you enable the "Docky" theme of Gnome do, it puts a nice application dock on the bottom of the screen like MacOS has.

Oh one more excellent eye candy app i've discovered is [webilder]("http://www.webilder.org/").

Webilder is a wallpaper switcher that you can set to change your desktop background every X number of minutes.  the beauty of this program is that you can set it to download photos directly from Flickr with the tags you specify.
