In those settings, you can make the entire thing transparent, or just some menus, but not just the window.

In System Settings > Desktop > Desktop Effects > All Effects, there is a plugin called transparency. In it's options you can change the transparency for just the window decoration.

This will only work if you're not using emerald. It's a bit basic, and doesn't look as good as the windows aero effect.
