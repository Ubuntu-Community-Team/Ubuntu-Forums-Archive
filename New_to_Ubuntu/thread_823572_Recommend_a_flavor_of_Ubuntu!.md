### Post by avtolle on 2008-06-09
> **jasontu said:**
> Alternate install CD?  I'm unfamiliar with the concept and why it would be used, can you explain what it is?

Thanks
The alternate install CD (or, more correctly IMO, the alternate install iso which is then burned to a CD) uses a text based installer rather than the graphical installer employed on the Live (or Desktop) CD. The text based installer uses fewer resources to run (in fact, for Ubuntu 8.04, it is stated that 384 mb RAM is a minimum to use the Live (Desktop) CD), and permits installation of Ubuntu (or other "flavors") on a lower resource computer. While it is stated that Ubuntu 8.04 will run, with the Gnome (or KDE, for Kubuntu) desktop on 256 mb Ram, you need more than that to install from the Live (Desktop) iso (384 mb, as stated before). 

Use of the alternate install, then, allows installation of the O/S on computers where it won't install otherwise using the Desktop CD. HTH.
