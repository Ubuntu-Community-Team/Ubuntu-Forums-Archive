### Post by jeremyswalker on 2009-10-13
I assume you are referring to the use of a virtual machine. XP would start up as if it were on its own physical machine. Check out [Virtualbox]("http://www.virtualbox.org/"). Here is a link to a tutorial on the subject --> [http://www.freesoftwaremagazine.com/articles/using_virtualbox_to_run_ubuntu]("http://www.freesoftwaremagazine.com/articles/using_virtualbox_to_run_ubuntu"). It is a couple years old, but it should remain fairly accurate. You could also search the [Virtualbox]("http://www.virtualbox.org") site and read the [user guide]("http://download.virtualbox.org/virtualbox/3.0.8/UserManual.pdf").
