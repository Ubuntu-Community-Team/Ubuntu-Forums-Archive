Deep breath ... and ...

Have you tried right clicking on the music icon on the webpage and downloading the file that way??? You need to fix up your media codecs anyhow so:

System->Administration->Synaptic Package Manager

Search for and install 'ubuntu-restricted-extras'

Add the Medibuntu repositories (to get to a terminal: Applications->Accessories->Terminal): 
[https://help.ubuntu.com/community/Medibuntu#Adding%20the%20Repository]("https://help.ubuntu.com/community/Medibuntu#Adding%20the%20Repository")

Check this out: 
[http://www.psychocats.net/ubuntu/nonfree](http://www.psychocats.net/ubuntu/nonfree)

That should make it work when you click the icons on the webpage.

If you are good with Mac and Windows you will probably enjoy this learning curve. Hope that helps and welcome.
