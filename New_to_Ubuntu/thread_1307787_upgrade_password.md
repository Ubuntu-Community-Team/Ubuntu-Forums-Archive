### Post by Xomm on 2009-11-02
If you've done a clean install and made sure your pass is typed right, you might have a broken disk. Try doing the "Check CD/disk for Defects."

If it says there's an error, check the md5 of the .iso

If it's right, then just re-burn the disk (remember to check your cd's max speed, burn-proof and finalize).

If it's wrong, re-download the .iso and burn it.


Hope this helps.
