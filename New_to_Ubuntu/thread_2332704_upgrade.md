### Post by howefield on 2016-08-03
> **Primus1 said:**
> Hi, I have Ubuntu 14.04 LTS, I am getting a message when I boot my computer saying "a new version of Ubuntu is available would you like to upgrade?" and it mentions Ubuntu 16.04.1 LTS. I thought 14.04 was to last for 5 years?

Yes, 14.04.1 and 14.04.5 are supported till 2019 so there is no need to upgrade, at least not because of support status.

> Should I upgrade?

Only you can answer that one, have a read at the [release notes]("https://wiki.ubuntu.com/XenialXerus/ReleaseNotes") and [change summary]("https://wiki.ubuntu.com/XenialXerus/ReleaseNotes/ChangeSummary/16.04.1") to see if there is any compelling reason to take it. If you do upgrade, remember all the usual prior preparation, backing up data, removing proprietary drivers, ect.

### Post by grahammechanical on 2016-08-03
When we install an LTS release (5 years support from release day) the system is set to notify us of the next LTS release. LTS releases come out every two years.

Ubuntu 14.04 may not have driver modules for hardware released to market in the 2 years since 14.04 was released. So, we have new versions to keep Ubuntu up to date with newer hardware.
