### Post by Zill on 2013-04-18
nerdyme:  Welcome to Ubuntu Forums and your offer of help is greatly appreciated.

I suggest that your starting point is to install and use Ubuntu on various systems for a while to see how everything works.  As you hit problems (which we all do from time to time!) just search the forums first to see if others have experienced the same problem and, if you can't find anything, post details on Ubuntu Forums and someone will help you out - we are a friendly bunch here. :-)

As your knowledge of Ubuntu specifically, and Linux in general, develops you can then pass this on and help other users via the forums.

With time, you will find various areas for improvement and, at this stage, you should be able to get involved more deeply with Ubuntu development.  However, I do suggest that, unless you are already an experienced programmer, viewing source code will be of little help to you.

Ubuntu does offer many options for helping with development and the [ContributeToUbuntu]("https://wiki.ubuntu.com/ContributeToUbuntu") wiki entry gives lots of details.

### Post by Impavidus on 2013-04-18
Yes, first familiarise yourself with the software and only start writing code yourself when you understand how the software works.

On the other hand, projects need more than just coders. If you wish to contribute without coding yourself, you may be able to provide translations or artwork, hunt bugs or work on the documentation.
