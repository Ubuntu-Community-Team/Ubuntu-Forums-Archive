### Post by DuckHook on 2013-02-17
Hate to put it so bluntly, but your whole box is way underpowered for gaming, from CPU to GPU to system RAM. And if I'm not mistaken, the Intel 3600 steals system memory for video RAM which whittles your system RAM down even further. Your system specs are barely sufficient to run Ubuntu itself without any application load. Sorry for the bad news.
