Anyone out there have this Netbook?  Were you able to get Ubuntu Netbook installed?  How did you do it?

Thanks!

### Post by wilee-nilee on 2010-12-28
Power on the computer immediately peck on the esc key to get to a per-session boot menu and try booting the USB from there.

There may be a on off for the esc key prompt in the bios for this post out of the bios menu, look in the bios for this if this doesn't work right away.
