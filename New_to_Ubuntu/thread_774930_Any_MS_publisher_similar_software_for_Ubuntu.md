Well, that's the problem.  The beauty of Publisher is that it comes with hundreds of awesome templates.  Programs like Scribus have the functionality of a stripped down Publisher, but without all of the content.  There are some templates for it, but you will never find anywhere near what you are used to with Publisher.  In the Free Software world, programming an app is easy, while creating content is not.  That's one reason that MS Office costs so much.

If you don't have the very latest version of Publisher, you can probably run it with Wine.  You may be better off trying that.
