### Post by nothingspecial on 2011-05-16
In the partitioning bit (where it asks you how you want to install Ubuntu) choose "Something Else" (I think it says that now)

Choose your empty partition (make absolutely sure it is the correct one)

Choose to use it as an ext4 jouranaling file system

tick/check the format box

Where it asks you for a mountpoint choose /

Again, make absolutely sure you choose the correct partition.
