### Post by Copernicus1234 on 2009-08-22
Most of your settings for your programs are in your home directory. For example, settings for Evolution are in .evolution, settings for Firefox are in .mozilla and so on.

So My Documents is basically /home/<your user name> and if you want, you can create folders in this directory for e-books, music, movies etc.
