### Post by SunnyRabbiera on 2008-09-13
Photoshop CS2 I know works via wine, I am not so sure on Dreamweaver 8 however I know it can get installed.
For installing Photoshop, you dont need any coding to use wine, if you have a .exe and you click it, the ihnstallation should start up automatically.
Wine is very easy to use these days, no coding or anything.
But you may want to try wineconfig first, in your menu under applications in your wine submenu there is a app called "configure wine"
with this if the default is windows 2000 there is a pulldown menu to change it to XP.
Another helpful tool is wine doors that i know can be used to install dreamweaver 8:
[http://www.getdeb.net/app/Wine-Doors](http://www.getdeb.net/app/Wine-Doors)

to install it just download the application and when its donload it install it like you would a windows .exe by double clicking on the package.
The installation should be simplistic.

as for bodog and the like, I have no idea if it works under linux, most of those kind of sites like windows/ IE only
