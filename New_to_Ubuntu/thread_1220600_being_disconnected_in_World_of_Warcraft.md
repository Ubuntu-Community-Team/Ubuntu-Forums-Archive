### Post by apjone on 2009-07-31
To list all hardware you can run the following from a terminal and output the info to your home directory in a text file. It could be either Video can not handle the amount of stuff to render ( happend in Eternal Lands to me ) or it could be your network connection to server dropping.


```

lshw > ~/MyHardware.txt

```
