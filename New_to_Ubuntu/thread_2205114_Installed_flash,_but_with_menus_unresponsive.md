It is a known problem, either use the desktop based settings manager (if it is available to you, depends on how you installed flash; search for 'flash' in the dash to find it), or the web based one (google for 'flash settings'; you might need to unblock it in Firefox.)

### Post by monkeybrain20122 on 2014-02-12
This is a bug that affects only Unity apparently. The Ubuntu devs claim that it is a flash bug and adobe is not updating flash so that's that. But somehow it works on other DEs. The workaround in Unity would be to go to adobe's site and change the settings there.
[http://www.macromedia.com/support/documentation/en/flashplayer/help/settings_manager06.html](http://www.macromedia.com/support/documentation/en/flashplayer/help/settings_manager06.html)
