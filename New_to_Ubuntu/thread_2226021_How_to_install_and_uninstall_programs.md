### Post by buzzingrobot on 2014-05-24
Ubuntu maintains tens of thousands of programs on servers called "repositories". These programs are maintained as archives that contain instructions for automatically installing all the bits and pieces in the correct locations. The "Software Center" is an application that allows a user to search those repositories and easily install anything he or she finds. Just findwhat you want and click on install.
