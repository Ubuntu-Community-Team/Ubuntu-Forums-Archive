### Post by norm7446 on 2009-11-12
If you have the NVIDIA driver installed, try booting with a live CD to see if that works. If it does then it's the driver if not then it could just be a loose connection, if so unplug the VGA /DVI lead from the monitor and the base unit and plug it all back in ensuring that all the thumb screw's are tightened.
