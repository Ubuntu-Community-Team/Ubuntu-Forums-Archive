you can try them before you install them. download the *buntu of your choice (Ubuntu, Kubuntu, Xubuntu. Mate, Gnome, Lubuntu...) "burn" them to a USB (unebootin or linuxliveusb) or DVD and then boot from them. select "try it" and see which ones performs best on your PC. newer versions have newer kernel (which is where most of the drivers are in Linux). you can however upgrade just the kernel in LTS (see hardware enablement). so the choice is yours. try it see what you like how well it works etc and then use that.

### Post by grahammechanical on 2015-01-08
We do not have versions of Ubuntu for specific machines. Ubuntu is a modern operating system it requires a minimum hardware specification. The requirements are not as great as the minimum requirements for Windows 7. So, I think that it is safe to say that Ubuntu and its flavours will work on that machine. But the key areas are WiFi adapters and graphic adapters. Are there drivers for WiFi and video?

If Ubuntu was pre-installed on that machine by the manufacturer then everything would work. But with Linux we get a one size fits all models solution and sometimes it is not an exact fit. This is especially true of the newest hardware. It is less true for hardware that is not the very latest because the Linux developers have had time to catch up.

The best advice, as already given, is to burn an ISO image to DVD or USB memory stick and Try Ubuntu.
