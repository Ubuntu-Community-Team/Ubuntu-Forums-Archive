### Post by mcduck on 2012-06-03
It's not possiblr to use CD's (or any othe roptical media) as an extension of hard drive space. Apart from the obvious option of moving any personal files there for long-term storage.

The filesystems used on optical media are different from the ones you'd use on hard drives since the discs are alyed out as tracks instead of sectors like on hard drives, and in general don't allow adding, editing or removing of individual files, instead requiring the whole filesystem to be written at once.

And even with sector-based media like DVD-RAM the speed would be way too slow to serve as a direct extenson of hard-drive space.

I'd recommend just getting an actual hard drive instead. Some old drive with more space than what you'd have on any CD/DVD would probably cost less than a can of coke, and if you buy a new one the cost of hard drve space is really low.

### Post by na5h on 2012-06-03
As far as I know, CD:s and hard-drives use different technologies. CD:s are optical media (which make use of lasers and optical sensors for reading/writing data) while hard-drives are magnetic systems.

I think you can burn a CD in a way which leaves it "open" and allows you to add more data to it later on...but I'm afraid I can't give much help on how to do that.

Getting another hard-drive or using a cloud storage (like Ubuntu One or Dropbox) is probably a better option.

### Post by Mark Phelps on 2012-06-03
Back when storage was a LOT more expensive than it is now, various optical drives came with drivers (Windows, of course) that would allow you to simulate a file system on the rewriteable CD.  It never worked all that well, but it was useful because, back then, 500MB disk drives were enormously expensive (hundreds of dollars!)

Now, with 64GB USB sticks a reality, the real-time performance of using a CD, when compared with a USB stick, makes this impractical -- even if you could find the needed drivers.
