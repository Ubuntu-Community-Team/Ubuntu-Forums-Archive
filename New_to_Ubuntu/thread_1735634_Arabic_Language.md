### Post by Raan03 on 2011-04-22
Allow me to help you.

To access the Synaptic Packet Manager, go to System / administration / Synaptics Packet Manager. (or alternativley you can open a terminal and type "gksudo synaptic")

In Synaptics Packet Manager, go to edit / Search (or alternatively press CTRL + F) and type "TTF Arabic" (without the " ") and mark them all.

Afterwards, press on the Apply button and let him download and install everything.
