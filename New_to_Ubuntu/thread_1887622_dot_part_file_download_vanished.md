### Post by gdonwallace on 2011-11-27
That is the downside to Firefox downloads.  It will start with the .part download and then when its finished, it will rename it to what the file should be.  However; if there is a problem with the download, or it gets interrupted, the Firefox will remove the .part file.  

I have had that happen a few times, and it really sucks.  Especially when it is a big download.  You might try a download accelerator, that seems to help me out when I have that problem.  I have not been able to find any other way around this.
