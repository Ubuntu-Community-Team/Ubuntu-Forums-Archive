### Post by jlhaslip on 2009-11-30
start evolution

Edit > Preferences
select Mail Preferences from the left sidebar
untick the box re: use the same fonts as other applications
specify your choices
close the selection

enjoy

### Post by jacobs444 on 2009-11-30
When creating an e-mail, look under the subject line and to the left, change the button that says plain text to HTML, then you will have all the normal tools for changing text size.
