### Post by graben3 on 2008-08-27
OK... 

1. Download the Zip file to your Desktop

2. Right click on it and Extract Here

3. Open a terminal window and paste this :

sudo cp -r $HOME/Desktop/Tango\ Inspirat /usr/share/themes

This should make it appear in :
System -> Appearance

in the theme section.

Just click on it after this.


Hope it helps... tell me if you experience problems.
