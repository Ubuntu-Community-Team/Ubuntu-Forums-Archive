### Post by nothingspecial on 2010-12-13
Boot a ubuntu cd

Choose to install

When you get to the partitioning bit, choose manual

Choose the partition that has kubuntu on it

Use it as an ext4 journaling file system

Give it a mount point of /

Choose to format it.

Make a brew

Now you have ubuntu.
