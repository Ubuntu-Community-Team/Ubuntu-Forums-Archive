### Post by ExileAmongYou on 2011-07-20
Why do you specifically need this kind of shortcut? Unless you have some particular reason for needing a read-only shortcut, I suspect it might be easier to create a launcher the normal way: right-click on the Desktop background, choose "Create launcher", and enter the command for the application you want (e.g. firefox, libreoffice) in the Command box. If you've done it right, the icon should be set automatically, and you just have to enter a name.
