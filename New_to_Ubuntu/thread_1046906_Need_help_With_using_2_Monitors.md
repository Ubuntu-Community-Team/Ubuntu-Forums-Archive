### Post by sowelie on 2009-01-21
Yes, although with ATI it's not all that easy.  The easiest way however is to install the non free drivers through System -> Administration -> Hardware Drivers.  Once that is done, use the ATI Catalyst Control center to configure your monitor.  See my other posts on how I eventually got this working properly: [http://ubuntuforums.org/showthread.php?t=1000758]("http://ubuntuforums.org/showthread.php?t=1000758").
