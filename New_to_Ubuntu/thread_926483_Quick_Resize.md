### Post by Polish Rifle on 2008-09-22
I used to resize my pictures very quickly in Windows XP.  I downloaded an addon that included the option to resize my pictures by adding the command to the right-click menu.

It was always fast because all I needed to do was right click on an image file.  Is there such a shortcut in Ubuntu?

---

### Post by Tatty on 2008-09-22
You can get this by installing an extension for nautilus:

```
sudo apt-get install nautilus-image-converter
```
