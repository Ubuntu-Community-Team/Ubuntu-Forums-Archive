### Post by moody_mark on 2010-08-20
Hi, to search through for any particular package in ubuntu you can use this page as a reference;

[http://packages.ubuntu.com/](http://packages.ubuntu.com/)

For example information on the "meld" diff viewer package is here:

[http://packages.ubuntu.com/lucid/meld](http://packages.ubuntu.com/lucid/meld)

This is more advanced though and if your just starting and you want to find out more about a program, just click on the "Applications" menu and then go to the "Ubuntu Software Centre" each program has some basic information and usually a link to a website with more info.

Hope this helps
