Hi

What can I say? I am not sure what you want. I suggest that you tap the super (windows) key and type help, and open the Ubuntu Desktop guide and learn how to use 12.10.

You can also hold down the super key and see some keyboard shortcuts. They are a useful thing to learn.

Also if you tap the Super key and type terminal and press enter the Terminal will open. What do you need to use a terminal for? In 12.10 most things can be done without using a terminal.

The last Ubuntu to be installed will put its version of Grub into the MBR (Master Boot Record) on the hard disk that it is being installed on to. So, the Grub menu that you see is the 12.10 Grub menu.

From the Grub configuration file I see this

Ubuntu 12.10 is installed on the second hard disk and in the seventh partition. Look for hd2,msdos7. That is the same as sdc partition 7.

Windows 7 is installed on the first hard disk and in the first partition. Look for hd0, msdos1. That is the same as sda partition 1.

Ubuntu 8.04 is installed on the first hard disk and in the fifth partition. Look for hdo,msdos5. That is the same as sda partition 5

Ubuntu 10.04 is installed on the second hard disk and in partition 2. Look for hd1, msdos2. That is the same as sdb partition 2.

There are lots of entries for older Linux kernels.

Grub has not detected Ubuntu 11.04. What did you do when you "surpressed" that partition to make room for Ubuntu 12.10? Did you remove/delete it?
