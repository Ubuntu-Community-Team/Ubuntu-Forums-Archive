### Post by viipu on 2012-03-22
Looks like Chromium can't handle RSS feeds out of the box (that's what that is, have a look at it in Firefox or some other browser if you can, Live Bookmarks or similar take care of feeds automatically). 

You'll need an extension for feeds, there's some [here]("https://chrome.google.com/webstore/search/rss%20feed") but can't really recommend any, sorry.

Bear in mind Chromium Project is kind of in constant beta, a development project, the actual product is e.g. Google Chrome which is Chromium with Google branding and some added features (not that Chrome could handle feeds out of the box either it seems).

Hope this helps!
