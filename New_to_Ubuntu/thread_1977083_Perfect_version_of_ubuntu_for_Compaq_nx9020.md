### Post by audiomick on 2012-05-10
> **Stortkop said:**
> What happens if the support period runs out?

No more updates are available. The system will, of course, continue to work. 

No updates means that you don't get newer versions of the programs you are using. If your system works fine, then this isn't a problem as such. It just means that if some program acquires a natty new feature, you wont get it.

More importantly, any security problems that might be discovered in the course of time wont be rectified. It is up to you to decide if this is a problem or not. I am personally not that concerned about this, but it is a consideration.
