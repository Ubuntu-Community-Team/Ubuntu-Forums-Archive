### Post by howefield on 2010-09-07
> **grobar87 said:**
> Can someone explain to me what is the differences between sun java and open java...and why sun java is better choice?

Said who ? 

Maybe it is a better choice, maybe it isn't ;)

You can have both installed side by side, some applications will have dependancies for one or the other. Depending on what applications you install, you may end up with both.

Setting the default can be done with..

```
sudo update-alternatives --config java
```
