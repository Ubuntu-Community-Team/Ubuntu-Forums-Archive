### Post by Impavidus on 2014-06-15
Vim is a bit complicated. It takes some time to learn it. It's modal, you have to press i to get into insert mode and escape to get out again, and then learn all the other commands. Search the web for the vimbook, read the first (and maybe second) chapter and you're good to go.

Edit: here it is: [ftp://ftp.vim.org/pub/vim/doc/book/vimbook-OPL.pdf](ftp://ftp.vim.org/pub/vim/doc/book/vimbook-OPL.pdf)
