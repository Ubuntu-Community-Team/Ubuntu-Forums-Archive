### Post by gn2 on 2008-12-28
It's not required to have Firestarter running.

Firestarter is just a GUI front end for adjusting the firewall (iptables) settings and it has been argued that leaving Firestarter running is actually a security vulnerability.

Reading this might be worthwhile: [http://ubuntuforums.org/showthread.php?t=510812](http://ubuntuforums.org/showthread.php?t=510812)
