### Post by themusicalduck on 2010-06-03
Any application installed from the Software Centre or from a .deb goes into your root partition (as far as I know).

If you install a program manually (ie, copy the file to wherever) then you can put it where you want to.

In my experience, applications hardly take any space on Ubuntu so you probably want a bigger home partition if you're going to use it to store everything else.

At the moment my root is 27.5gb total with 12gb of it free and my home is 293gb.
