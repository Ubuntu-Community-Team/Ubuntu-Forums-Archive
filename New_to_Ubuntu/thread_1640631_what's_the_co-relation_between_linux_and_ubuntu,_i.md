### Post by bodhi.zazen on 2010-12-08
Kernel == Linux

Add the GNU stuff (gcc and such) == GNU/Linux == the core operating system.

All "linux" distros, Ubuntu included, use a Linux kernel + some, if not all GNU software.

GNU/Linux + X + Gnome = a "basic" distro.

GNU/Linux + Gnome + a few customizations + some verdor specific applications (Ubuntu One) = Ubuntu

GNU/Linux + Gnome + other additions = Fedora, Debian, etc ...

See also : [http://en.wikipedia.org/wiki/GNU/Linux_naming_controversy](http://en.wikipedia.org/wiki/GNU/Linux_naming_controversy)
