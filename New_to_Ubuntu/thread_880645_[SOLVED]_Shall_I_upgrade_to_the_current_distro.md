### Post by ajgreeny on 2008-08-05
If you look on the Mint website, they suggest not doing a distribution upgrade, but better to clean install.  Some posters on the forum, however, say there have been no major problems to upgrading, so the choice is yours.

I suggest you make sure you are backed up first, then try the upgrade and if that doesn't work to your liking, go for a clean install.  I run both Ubuntu 8.04 and Mint Elysa though I still prefer Ubuntu, but my wife uses Mint and loves it.

I can see no enormous differences between the two except for the cosmetic differences, and of course the codecs etc already in Mint, but that takes me about 5 mins in Ubuntu to put right now I know how.  I do think, however, that Mint has some good odeas such as the right click in menu items to uninstall programs, though I still like synaptic, which I use almost to the exclusion of other methods of installing and uninstalling.

### Post by OldTimeTech on 2008-08-05
I've done the upgrade to Hardy Heron both with the clean install and with the distro upgrade...I had one machine that I lost everything on because I didn't check the hardware out (dumb, dumb...should of known better).
