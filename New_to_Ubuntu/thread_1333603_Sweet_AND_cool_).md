### Post by drakkoss on 2009-11-21
I am so impressed with Ubuntu!

I created a partition on my Gateway laptop and installed Ubuntu 9.10 (pre-release at that time) a few weeks ago to give it a try.  The first thing that I noticed is that the fans in my computer run much slower when I use Ubuntu than they do when I am using my Windows 7 partition on the same computer.  I don't know what that's about but I have to believe that anything that makes a computer run cooler is a good thing.   Less current sucked up from the wall socket  leads to a little less pollution in the environment.  

Next thing I noticed is that I could receive all my favourite online radio stations with no problem.  I have previously had issues with stuff like this using Linux to play music from online music streams.  

Additionally, Skype works great with Ubuntu right from the install, even using one of these less popular $100 dollar Logitech web-cams.  I talked via Skype with video for about four hours this morning with a friend in Perm, Russia  with no problem what-so-ever.  

I'm using Google docs rather than Openoffice so I can't speak to Openoffice as a substitute for MS Office.  

Now, back to this issue of Ubuntu being cooler than Windows 7.  I performed a clean install of Windows 7 last weekend thinking that some program was running in the background on Windows Vista which caused the computer to consume so much power.  The computer is still running hot under Windows 7 ... so much so that I can FEEL the difference on the keyboard ... and it is a significant difference.  Clearly, the CPU and the associated video processor are the reason for this.  The CPU is just not working as hard under Ubuntu for some reason.  Icing on the cake as far as I'm concerned.  Congrats to the Linux programmers.

In summary, Ubuntu is sweet AND cool :)
