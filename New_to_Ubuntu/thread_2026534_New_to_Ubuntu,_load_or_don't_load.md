### Post by cybergalvez on 2012-07-15
I took the plung a few years ago. In general ubuntu is very easy to use, quite stable and there is very little that you actually can't find a suitable alternative for. I set my Mother up with it a few years back after I got tired to constantly removing viruses from her computer. She is using it without difficutly. 

What I would recomend
1) backup up all your files - if something goes wrong you will be very glad that you did, and even if it all goes as planned it will make the trasition simplier. Also don't use a windows program to back up your files, for this you are better off just copying everything to an external HD.
2) download the live ISO and burn it. 
3) check out the live disk, this will give you a general feel for Ubuntu before you instal anyting, although note it will seem a little slow, but that is because its not really installed
4) When you are ready, install and have it format the entire disk. This will destory your windows install and most likly all your data too (which is why you backed it up  in the first place)
5) Copy your data back and you are ready to rock and roll.
6) If you still have your windows disks you can install virtualbox and install windows as a virtual machine, this is what did so that I can watch netflix on my desktop. 

Have fun

### Post by xedi on 2012-07-15
> **Elizabethza said:**
>  However, what I really want to know is must I uninstall all Microsoft stuff and will it override Windows 7, is it easy to learn because I'm too old to learn new tricks. 


No, you can leave windows installed and install ubuntu alongside of it. There are two ways to do it, dedicating a partition (a section of your hard drive) to it or use the WUBI installer/windows installer. I normally recommend the first method, but because you are unsure whether you would at all like it and about the installation process, I recommend WUBI.

[http://www.ubuntu.com/download/desktop/windows-installer](http://www.ubuntu.com/download/desktop/windows-installer)

With WUBI you can install Ubuntu like normal software and also uninstall it as if it were some other windows program. If you happen to like it, you can then make it into a "normal" ubuntu installation and if not, you can simply remove it.

If you choose to go the full route from the beginning, during the installation process you can choose to install alongside windows. While this leaves Windows in tact, I suggest making a backup, in case something goes wrong.

Ubuntu is easy to learn, I find many things easier than in windows but it is different so you have to be willing to learn new things, Ubuntu is not a windows clone.
