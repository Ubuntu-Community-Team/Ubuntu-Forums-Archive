### Post by newb85 on 2012-09-01
Whether you've installed it or you're running from a LiveCD or USB, Ubuntu should support virtually any hardware for connecting through ethernet or wifi.  Ethernet should work automatically if you connect it.  If it's wireless, you'll have to select the network in network manager, which you can access through the system tray in the top right corner.

F.Y.I., the default browser is Firefox, but you can install scores of others.  The first place to look for alternatives (if you're dissatisfied or curious) is the Software Center--also in the launcher.
