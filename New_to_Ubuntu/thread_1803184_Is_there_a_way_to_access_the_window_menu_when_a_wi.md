### Post by mahad on 2011-07-12
You can use the following:

ALT + F5    Minimize
ALT + F10  Maximize
ALT + F7    Move       - Use the Cursor keys/Arrows keys  to move it 
ALT + F8    Resize      - Use the Cursor keys/Arrows keys  to move it 
ALT + F9    Hide

I hope it help, take care :D
