### Post by lukeiamyourfather on 2010-01-16
> **Vignesh S said:**
> Don't worry, I found out that I moved it to "My Pictures" inside of "Documents" instead of Pictures. omg, I really can be somewhat of a noob sometime ;-). But if this were the case, how would I recover deleted files?

That all depends. For example if something is written to the disk before the recovery is attempted it might overwrite those blocks (since they're marked as free) and a recovery would be impossible. People often think there's some magical tool to "undelete" files but its not that simple in most cases. There's a lot of information about the topic if you search.

[http://www.google.com/search?hl=en&rlz=1C1GGLS_enUS337US339&q=recover+deleted+files+linux&aq=f&oq=&aqi=g3g-m2](http://www.google.com/search?hl=en&rlz=1C1GGLS_enUS337US339&q=recover+deleted+files+linux&aq=f&oq=&aqi=g3g-m2)

Consider it a lesson about why you should backup your files. Hard disk failure isn't the only way data is lost, and if you had actually deleted them the backup would have it covered and there'd be no need to ask about how to recover deleted files. Cheers!
