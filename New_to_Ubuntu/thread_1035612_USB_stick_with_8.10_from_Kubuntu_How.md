### Post by evilkastel on 2009-01-09
GNOME MENU does not exist in kubuntu. is especific of Ubuntu. Kubuntu uses K Desktop enviromen, while ubuntu uses GNOME desktop enviroment.}
So find Terminal (or Konsole) in the Application menu and run it.
with the pendive inserted, 
In the terminal window write ```
liveusb
``` 
The app will recognize all.
In about 10 minutes you'll have a liveusb.
