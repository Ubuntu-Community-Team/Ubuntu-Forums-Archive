### Post by OneRingShort on 2008-10-23
I'm trying to find a way to change which program opens when a file of a specific file type is double clicked in nautilus or opened from firefox.  I know you can "Open with other application" but there is no option to make it the default program, and sometimes the program I want to use isn't listed.  I've also searched the system preferences and found the "preferred applications" option, but it only includes a few different types of programs.

A few of the main things I want to change office documents (.doc, .xls, etc...) to open with Open Office 3 instead of Open Office 2, torrent files to open with Ktorrent instead of Transmission, music to open with Songbird instead of mPlayer, and movies to open with VLC.

Thanks for the help

---

### Post by Zill on 2008-10-23
Right-click on the file in Nautilus or on your desktop.
Select Properties and the Open With tab.
Select the radio button for your preferred application.
If it isn't in the list then click on Add and select the application.

Your chosen app should now be the default :-)
