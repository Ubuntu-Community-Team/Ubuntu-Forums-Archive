### Post by jmszr on 2011-07-14
homebre973,

The Firefox add-on Flash-Aid will "Remove conflicting flash plugins from Ubuntu/Debian Linux systems, install the appropriate version according to system architecture and apply some tweaks to improve performance and fix common issues." You can find it here: [https://addons.mozilla.org/en-US/firefox/addon/flash-aid/](https://addons.mozilla.org/en-US/firefox/addon/flash-aid/).

If you still have problems after running Flash-Aid this thread may be of help: [http://ubuntuforums.org/showthread.php?t=1799217](http://ubuntuforums.org/showthread.php?t=1799217) .

Hope that helps.
