### Post by JKyleOKC on 2011-04-22
If you're still shopping, look especially for machines with dual or quad core processors, and at least 3 GB of RAM. This will make it much easier to use virtualization software should you later find a need to have Windows, or try other systems without harming your primary installation. The additional processor power won't hurt in everyday use, either... While I personally prefer AMD processors, Intel units are almost as good:D.

Your original post indicates you plan to reflash the BIOS on the machine. Why? There's no need to do so; I've never had to reflash BIOS on any of my machines although some of them are running with 15-year-old versions. The boot-time actions are handled by code that's stored on the hard drive in predefined locations (usually reached through the drive's Master Boot Record) and the only BIOS change required is to set the boot sequence to try the CD or USB first when powering up...
