### Post by ajgreeny on 2012-09-21
For chromium it's just ```
sudo apt-get install chromium-browser
```
For Google-Chrome you start by downloading the latest version from google [https://www.google.com/intl/en_uk/chrome/browser/?hl=en-GB&brand=CHMI](https://www.google.com/intl/en_uk/chrome/browser/?hl=en-GB&brand=CHMI) and when that installs (by double clicking) it will automatically add the chrome ppa repository to your software-sources to make sure it is updated when new versions appear.
