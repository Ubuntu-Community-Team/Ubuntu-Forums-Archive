### Post by BandD on 2010-05-18

### Post by musicman10489 on 2010-05-19
I really enjoy AWN for a dock. It seems to have become loads more stable than I remember it ;]
As for effects, under the Keyboard Shortcuts menu I assigned 'Initiate Window Picker' to my menu key. This basically brings all open (non-minimized) windows into a clean view. Much like Mac's expose feature :]
