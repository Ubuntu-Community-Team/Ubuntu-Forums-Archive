### Post by Mark Phelps on 2011-08-29
If you're going to use MS Office via Wine, then you need to become familiar with the WineHQ application compatibility database ratings -- because test results there can tell you what hasn't worked for others and what they did about it.

MS Office, generally, does NOT work well in Wine -- the linked page for PowerPoint shows mostly Garbage ratings:

[http://appdb.winehq.org/objectManager.php?sClass=version&iId=12813]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=12813")
