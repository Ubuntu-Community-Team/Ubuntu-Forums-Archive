### Post by 3rdalbum on 2009-10-10
If you have a real dual-boot that you installed by booting the Ubuntu CD, then your Windows partition should be listed in the Places menu and you should be able to mount it like that.

If you installed Ubuntu from within Windows (known as "Wubi"), then click the "Filesystem" button in the left pane of a file browser. Then double-click on the "host" directory.
