### Post by Mr_Speedy on 2010-07-24
I have not used the application myself, but according to the description I think it is just what you are looking for.
Click on Applications > Ubuntu Software Center   and in the top right hand corner type in "caps lock". Install the "keyboard lock keys applet for the GNOME2 panel". There are other matches to that search which might also be fine.

If you are not getting any matches, it is possible that you need to allow an extra repository to download software from. 
System > Administration > Software Sources    then click "Other Software" and check the box next to "http://archive.canonical.com/ubuntu lucid partner"

Hope this helps  :)
