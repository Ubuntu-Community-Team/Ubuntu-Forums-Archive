### Post by michael-piziak on 2013-12-21
I have a libreoffice file I want to duplicate and place the duplicate in the cloud.
How do I duplicate a file?  I can't see the option anywhere when I right click the file.

---

### Post by coffeecat on 2013-12-21
Right-click on the file and select copy. Then right-click on a blank space in the file browser and select paste. A duplicate of your file will be created with "(copy)" appended to the filename. You can then rename it if you wish.
