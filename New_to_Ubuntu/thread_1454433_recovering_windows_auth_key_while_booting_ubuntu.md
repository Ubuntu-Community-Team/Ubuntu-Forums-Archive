### Post by Jon Monreal on 2010-04-16
I'm not aware of any way to get this from Ubuntu.

I think your best bet would be to get all of the files - including the  \Windows directory - off the hard drive and then using [this open-source Windows  program]("http://magicaljellybean.com/keyfinder/") to get the key.

Or you could convince him to convert to Ubuntu :P
