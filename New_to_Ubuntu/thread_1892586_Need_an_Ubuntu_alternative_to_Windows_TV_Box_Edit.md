### Post by 3rdalbum on 2011-12-11
> **osprey2000 said:**
> Someone suggested downloading Wine to use the program then Uninstalling Wine if I decide not to use it BUT I thought I'd read here that using Wine without AV protection leaves you open to viruses ???

Using Wine probably won't work for what you want it to, but you can give it a try.

Wine is only a program that can run Windows programs. Realistically, you can only get viruses through Wine if you purposely run them; and even then, the virus would probably fail to run. It certainly wouldn't infect the host Linux system.
