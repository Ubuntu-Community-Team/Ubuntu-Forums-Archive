### Post by Alterax on 2009-05-07
For what it's worth, I'm getting a pretty thorough workover with the following books:

Problem Solving with C++ (Savitch, 5th edition)  
  (Which I had to get for University)
GNU/Linux Application Programming (Jones, 2nd Edition)
  (Which I got a gently used copy for $7.00 US online)
And the Debian developers' documentation (available on their site, free).
