### Post by Anzan on 2008-06-15
Without reading it's hard to get anywhere.

Do a search for bash. There are many manuals available.

There's a tutorial you can follow called Unix For The Beginning Mage that is available as a free pdf. The metaphor is that commands are like spells. Silly, but it's a good tutorial.

As well, while you are playing around in the CLI you can use the man command to bring up documentation on commands and programs, even on the Perl and Python languages.
