### Post by Kosimo on 2009-04-13
Hello guys.
I would like to understand if there is any easy way to go back to a specific version of program after an upgrade using a 3rd party PPA.

For example, now I've upgraded my Intrepid Compiz to 0.8.2 using a launchpad PPA. So, if I ever want to go back to the previous official version is not enough to remove the PPA as the installed version will be always newer than the one in the official PPA. Compiz has lots of dependencies and things that have been upgraded by this third part source.

So there is any way to do it quickly?

Thank you

---

### Post by jmmL on 2009-04-13
Select the required package in compiz and go Package > Force version.
