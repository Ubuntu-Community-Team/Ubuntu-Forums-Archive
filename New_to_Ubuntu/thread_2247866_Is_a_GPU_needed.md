### Post by Vaphell on 2014-10-10
it's the gpu that provides monitor compatible sockets, either d-sub, dvi or hdmi. If the mobo doesn't have onboard gfx then you need a discrete card.

### Post by mcduck on 2014-10-11
The options are a separate graphics card, GPU integrated on the motherboard, or, quite commonly these days, a GPU integrated in your processor. Either way, you do need a GPU...

So, if you don't plan on using a discrete graphics card, make sure that the CPU you have has a GPU included. Although most of them do these days...)
