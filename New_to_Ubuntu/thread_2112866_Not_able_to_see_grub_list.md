### Post by Impavidus on 2013-02-06
First of all, 10.10 has been unsupported for almost a year. You can better install a supported version (12.04 or 12.10; use xubuntu or lubuntu if your hardware can't handle the new user interface). It may fix your boot problem at once.

Otherwise, you can boot from a cd and use boot repair. You can generate a BootInfo summary and post the link here. It may tell us what's wrong with your system. See [http://ubuntuforums.org/showthread.php?t=1769482](http://ubuntuforums.org/showthread.php?t=1769482)
