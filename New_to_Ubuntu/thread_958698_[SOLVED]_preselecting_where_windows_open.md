### Post by jpoRS on 2008-10-25
I know that Compiz has the option to map where windows go, but I don't know if you are using/are willing to use Compiz.

If you don't have Compiz and want to try it, try [HERE]("http://ubuntuguide.org/wiki/Ubuntu:Hardy#How_to_enable_Compiz_Fusion_in_Ubuntu")

It is a LITTLE on the advanced side, but I am only saying that because this is posted in Absolute Beginner.  Once you have it installed just go to System>Preferences>Advanced Desktop Effects Settings and then look for "Place Windows" under Window Management.  I haven't used this feature personally, but it may be what you are looking for.
