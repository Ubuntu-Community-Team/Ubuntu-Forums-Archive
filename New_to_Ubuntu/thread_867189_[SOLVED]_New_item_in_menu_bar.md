I've just installed Xubuntu 8.0.4.1 on my laptop.  How do I add a new item onto my top menu bar ?  I want to add the terminal but can't just drag and drop the icon onto this bar unlike Ubuntu or Kubuntu and it's not one of the dozen or so programs listed that can be put there.

### Post by rbc on 2008-07-22
Right click on the top panel, and choose "Add to Panel". The first selection you should be given is Custom Application Launcher. Select that, then in the Command field, type xfce4-terminal

### Post by jimmy the saint on 2008-07-22
If you are wanting to add a launcer, right-click on the panel, select "add new item" and select "Launcher.  This will opon a dialoge. You can select a name, an icon to use, then type in the command to open the program.  For example, to open the program "Mousepad" the command is "mousepad". The new launcher will show up on the bar.
