### Post by kc1di on 2012-05-26
you would do it the same way you use to . You need to make a bootable dos floppy if that machine has a floppy drive, if not make a boot able dos cd rom and install it from there 
Dell instructions here:
[http://www.dell.com/support/drivers/us/en/19/DriverDetails/DriverFileFormats?c=us&l=en&s=dhs&cs=19&DriverId=R109329#1]("http://www.dell.com/support/drivers/us/en/19/DriverDetails/DriverFileFormats?c=us&l=en&s=dhs&cs=19&DriverId=R109329#1")

follow the ones for Dos install.
Here is an article that tells you how to make the dos disk from linux.[http://tuxtweaks.com/2009/05/create-a-bootable-usb-drive-ubuntu-freedos/]("http://tuxtweaks.com/2009/05/create-a-bootable-usb-drive-ubuntu-freedos/") it uses free dos but think it will work ok.
