### Post by soxs on 2008-11-03
tipp10

it is really nice to learn with, it has statistics and you can even learn typeng C/C++ code with 10 finger system.

Note: In case you are a 64 bit user you will have to download the package here from the forums (search for "getlibs") ```
sudo apt-get install libpng3; sudo getlibs -p libpng3
```
And then install the pending debian package from [www.getdeb.net](www.getdeb.net)

It's the way I learned typing. Doing a everyday lesson of 30mins seems to be enough to learn the 10 finger system for the standard office requirements (at least this pimped my skills to a sufficient level)
