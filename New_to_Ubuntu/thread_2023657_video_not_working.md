### Post by Paqman on 2012-07-12
On Ubuntu the first place to get software is the Ubuntu Software Centre. Open it up and search for "flash" then install the package it suggests (which should be called [flashplugin-installer]("apt:flashplugin-installer")).

You could also install [ubuntu-restricted-extras]("apt:ubuntu-restricted-extras"), which contains Flash, Java, extra fonts, etc, etc.
