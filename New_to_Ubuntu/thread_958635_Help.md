Afterthought! Don't *install* anything at first!

Pick out a live distro!

Ubuntu, Kubuntu, Xubuntu, Linux Mint, Open SUSE, Simply Mepis, Fedora, etc. and just run the live CD! Then run the Live CD some more!

When you find a distro you seem to like then consider something like a dual boot! The only distros I personally recommend for a nOOb like me are Ubuntu based distros like Ubuntu, Xubuntu, Kubuntu, and Linux Mint! 

Simply Mepis is a decent alternative to Kubuntu if you like KDE.

I've found Open SUSE to be stable but harder to install and slower once installed compared to Ubuntu.

I think Fedora is probably fine, but I still find it much harder to understand than Ubuntu, and I find the Ubuntu Forum community to be much more friendly to noobs like me.

If I were personally to recommend a distro that's free and just works with the least tweaking it would be Linux Mint. But Mint is Ubuntu with a few tweaks and a face-lift!

The beauty of Linux is that we "Linuxers" all agree on one thing: Linux is great but we all have our own "favorites"!
