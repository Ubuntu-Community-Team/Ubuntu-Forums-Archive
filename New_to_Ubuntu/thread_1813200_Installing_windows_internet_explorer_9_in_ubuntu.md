### Post by Paddy Landau on 2011-07-27
If you absolutely must have Internet Explorer, you have three options.

[LIST=1]
[*]Dual-boot Windows.
[*]Install Windows in a Virtual Machine (if your computer is strong enough to handle it).
[*]Install [PlayOnLinux]("apt:playonlinux") (it is a front-end for Wine) and follow the instructions to install IE 7 (yes, that's 7, not 8 or 9).
[/LIST]
EDIT: A fourth option is to write to the website owners and complain, though that is unlikely to make any difference.
