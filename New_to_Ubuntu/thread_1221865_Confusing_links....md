### Post by moody_mark on 2009-07-24
> **NOTAGEEK said:**
> what is this silverlight stuff ??? is this not microsoft windows stuff ??? 

Silverlight is the Microsoft version of Adobe Flash and used on some sites (ITV.com is one I can think of) instead of regular flash. For us linux users it means that we cant really guarantee to be able to display the content of these sites. I have seen some posts on this forum regarding how to get such sites working but im not sure how successful people have been.

Regarding you radio station feed generally for a listen link you need the URL and a port number, most stations dont expose this instead they have a file that you can download and open with a player. Rhythmbox should be able to play a few of these I think looks like KSLX FM dont have this info on their site, you could drop them an email and ask, you never know :-)
