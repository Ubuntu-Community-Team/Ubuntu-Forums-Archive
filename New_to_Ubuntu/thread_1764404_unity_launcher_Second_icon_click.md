### Post by mc4man on 2011-05-21
> before that - when I let say opend up firefox window,and pressed the launcher icon again - the window would go into something like scale mode.
A single or sometimes double l. click on app icon will open the scale window picker on window group only if you have 2 or more windows open
(1 or 2 clicks depends on where you are in relation to the app's windows and what focus is on - works very logically.

You would normally find that binding hard to unset because you can't set any binding to 'Window Picker for Window Group' that will work or interfere.

So open 2 firefox windows and ck., if no good then  make sure scale is still enabled
