### Post by mcduck on 2011-02-02
It will most likely appear in the right-click menu when clicking on a image file (*directly* in the menu, not in the applications to open the file). And you'll also have to restart Nautilus for such program to appear after installing. (Hit Alt-F2 and run "killall nautilus" or log out and back again)

edit: I'm now assuming that it's the *Nautilus Image Converter* you are talking about, a simple Nautilus extension used to resize images and convert them from one file type to other. It's not actually image *editor*, or even a standalone application you could open
