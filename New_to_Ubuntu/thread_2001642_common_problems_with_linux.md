### Post by Curtis6767 on 2012-06-11
There seem to be a couple of recurring problems when people move from another OS to Ubuntu.

The first has to do with wireless connections and the second has to do with video drivers.

If Novatech tests your system ahead of time, then they'll be able to make changes so you won't have to deal with these issues, yourself.

Next get familiar with the install process ahead of time: [http://www.ubuntu.com/download/help/install-ubuntu-desktop](http://www.ubuntu.com/download/help/install-ubuntu-desktop)

Before you do the install, make sure your install media is not corrupted: [https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

### Post by Mark Phelps on 2012-06-11
Unfortunately, it's common these days for educational institutions to REQUIRE their students to use Microsoft products for class assignments, which is why they often provide such products at very low prices (known as Academic versions).

So BEFORE you jump into any Linux distro, you need to check with your school to see what they require.  MS Office does not run well in Linux, and some of its components (Outlook, Access) do not run at all.

Also, programming coursed often require installing and running Windows apps -- so the same problem can happen.

IF you have to end up using MS Windows products for class work, there is no free solution for that; you will have to buy the software -- and you will need a Windows OS to install it.

### Post by kurt18947 on 2012-06-11
All good answers.  If I were looking at a notebook, I'd steer clear of hybrid graphics if possible.  Some people have gotten them to work but the process seems far from easy or certain.  Wireless *can* be a problem but there are nano USB adapters that work very well and are inconspicuous if the internal WiFi adapter cannot be made to work.   The last issue is can you do everything you need to do with Linux software?  

 An example:  I wanted to update the firmware on a multi-function machine.  The update was only available as an .exe file.  .Exe files don't run on Linux.  And no, it wouldn't run on an OS X -only Mac either.  There are devices that require database updates.  The installer is Windows only.  If you don't have these issues,  go for it!  I spend 99+% of my time in Linux but the first time I didn't have a rusty Windows install available I'd be up the proverbial creek without it.
