### Post by Frogs Hair on 2011-09-16
The most _recent_  version on the Adobe site is 9.42 and the version in the software center doesn't display the full number . If you have to install Adobe for some reason the software center is the easiest method . You can download a Linux .deb from the site and install it with Gdebi package installer from the software center . After installing Gdebi right click the .deb and select Gdebi from the context menu .    [http://get.adobe.com/reader/otherversions/](http://get.adobe.com/reader/otherversions/)

Be sure to select the ..deb it is the easiest to install .

### Post by varunendra on 2011-09-17
My favourite is [Foxit Reader]("http://www.foxitsoftware.com/downloads/") - light (3.6 MB), fast and capable. I kicked away adobe acrobat years ago.

Just select "Linux Desktop" in the linked page, then "deb" package. Oh if the installation fails to work (it happened to me once), just download the bz2 package > extract it to a folder and create a symbolic link to the executive file, which you can then use to directly open the application.


**_PS_:**
If you choose it, do not forget to install the add-ons from the same page. They extend its capabilities and are installed from within the reader.
