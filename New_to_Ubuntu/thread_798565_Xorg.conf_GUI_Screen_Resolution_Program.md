### Post by mbarak on 2008-05-18
The Xserver that comes with ubuntu tries to be very automatic so that you don't have to configure it anymore
Have you tried to simply restart the X server by presseng ctrl + alt + backspace ? (sometimes my screen is stuck at 600x480 resolution and that fixes it)
also - what Xorg gui did you download? there is a gui to configure the Xserver that comes with ubuntu that you can try to use
run this command either in the terminal or by pressing alt+f2
```
gksu displayconfig-gtk
```
