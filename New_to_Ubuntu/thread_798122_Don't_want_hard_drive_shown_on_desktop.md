### Post by z0mbie on 2008-05-17
Launch the Gnome config editor by pressing **Alt+F2**, then type in:

```
gconf-editor
```

Navigate to **/apps/nautilus/desktop**

Uncheck **volumes_visible**.

### Post by Colin82 on 2008-06-20
Ok, well this should work for you no problem.

To change the mount point of your HDD, you will need to edit the fstab file.

In a terminal type
```
sudo umount /media/Media
```
to unmount the drive.

Then make a new directory to mount the drive in
```
sudo mkdir /mnt/[COLOR="Red"]shared[/COLOR]
```
Replace the "shared" that is in red with whatever folder name you want for this HDD volume.  You can use Media like it was before if you want.

Now we'll make a copy of fstab for a backup
```
sudo cp /etc/fstab /etc/fstab_backup
```

Now open the fstab file
```
sudo gedit /etc/fstab
```
and change the line referring to sdb1 to read like this
```
/dev/hdb1 /mnt/[COLOR="Red"]shared[/COLOR] ntfs nls=utf8,umask=0222 0 0
```
where the "shared" in red matches whatever directory you used above.

Then you will just need to mount the drive, and you'll be good to go.
```
sudo mount -a
```
