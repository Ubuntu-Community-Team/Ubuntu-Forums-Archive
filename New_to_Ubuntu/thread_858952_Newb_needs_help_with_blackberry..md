### Post by ChameleonDave on 2008-07-14
> **gbrad901 said:**
> 
Another thing, is there a way to set Ubuntu back to the original installation? I don't know if i mucked things up in the last couple days and should start fresh.

As nothingspecial says, a fresh installation is the way to do this.  There is no "system restore" enabled by default in Ubuntu.  If you want to go back to a previous state, you have to make a backup.  That's something to consider for the future.

However, since it is so easy to back up just your /home directories and reinstall the rest from the live CD, there is maybe no need to make a full system backup.
