### Post by adobrakic on 2009-02-18
To install Compiz go into terminal and do:

```

sudo apt-get install compizconfig-settings-manager

```

As for installing WinRAR, have you tried right clicking on .rar files and going to 'Extract Here'? That should extract your zipped files just as WinRAR would. 

But to run .exe files, you can install Wine with

```

sudo apt-get install wine

```

and run them through there. Some programs work well with Wine, and some don't. You'll need to check the AppDB at Wine's website to find out which programs work and how well they work.
