### Post by mcduck on 2014-04-24
It's more than likely that the password protection is based on software that came with the drive, which probably isn't available for Linux.

There are some cross-platform tools for encrypting data on portable drives, but this far I've never seen any commercial drives shipped with those instead of the drive manufacturer's proprietary software. I'd say the best approach is to copy the data from the drive na Window computer with the provided software, and then either remove the password protection/encryption from the drive or redo it with some  cross-platform compatible tool.
