### Post by Bloch on 2008-10-28
You probably installed cinelerra video editing software from the Akirad Repository
[http://akiradproject.net/repository](http://akiradproject.net/repository)

I think some of the codecs it wants to install are more recent versions of codecs installed with other packages.

If you are not a keen cinelerra user then my advice would be to ignore this, and untick the update.

There surely exists some way of forcing the over-write, but it might not be a good idea - perhaps refer to the Akirad forums for specialist advice.
