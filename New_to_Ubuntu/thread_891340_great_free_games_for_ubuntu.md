### Post by RomeReactor on 2009-07-28
> **dgrilli said:**
> /home/dave/.setup3712: error while loading shared libraries: libgtk-1.2.so.0: cannot open shared object file: No such file or directory

Install libgtk1.2, and in 9.04 you may need libglib1.2ldbl as well:
```
sudo aptitude install libgtk1.2 libglib1.2ldbl
```

If you're running Ubuntu 64-bit, you need to install [additional packages]("http://gwos.org/doku.php/guides:64bit:et_wolfenstein?s=enemy") (the page doesn't mention libglib1.2ldbl, so if you get the error mentioning it, run the following also):
```
getlibs -p libglib1.2ldbl
```
