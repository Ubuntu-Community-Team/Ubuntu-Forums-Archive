### Post by svk on 2008-06-28
Flash is known to have some problems in Linux.  I haven't experienced specifically the type of crashes you mentioned, but I did make my problems with Flash go away by updating to the beta version of Flash ([download link]("http://labs.adobe.com/downloads/flashplayer10.html")).

Click the .tar.gz version for Linux, save it somewhere, and extract the archive.  Then run the install program as root and follow the directions it gives you.

I'm not sure, but maybe you should first uninstall the old version of Flash (via your package manager).

Good luck and post back if you run into problems.
