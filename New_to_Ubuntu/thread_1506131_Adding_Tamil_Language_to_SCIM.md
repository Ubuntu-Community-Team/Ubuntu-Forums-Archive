### Post by expatCM on 2010-06-10
Scim was basically retired starting with 10.04. It was replaced with iBus which you will find is already installed for you.

I think what you need to do is to 

1.  Remove scim
2.  Install the relevant language support and enable ibus on those screens (system / administration / language support)
3.  Configure your iBus preferences (system / preferences / iBus)
