### Post by cariboo on 2010-02-17
You can install LAMP and OpenSSH-server during the installation if you are using the server installation CD. If for some reason you're using the desktop edition, you can install the server packages by going to System-> Administration->Synaptic Package Manager->Edit->Mark packages by task. You'll have to install Webmin separately as it isn't included in the repositories.
