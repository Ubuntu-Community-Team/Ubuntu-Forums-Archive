### Post by audiomick on 2010-07-10
Hallo.
I don't know that Xubuntu is significantly faster than Ubuntu. I have read conflicting opinions on that subject. I suspect that Ubuntu would run faster than XP, although that is just a guess.

What you should be able to confidently expect, I think, is a faster boot from an Ubuntu only system. Also, I read a thread just the other day where someone made the point that the access is faster at the front of the drive than at the end. The XP system will be at the front, so if you get rid of that and install the Ubuntu in the first partition, you should theoretically see a slight speed improvement. The opinions expressed in the thread, however, were at odds about how significant this would be.

edit: what Ubuntu wont do, at least not nearly to the same extent as XP, is get slower as time goes by.
If you can possibly get more RAM into the machine, you should. As far as I know, that is what is most likely to gain you some speed.

### Post by Jerry N on 2010-07-10
In my (limited) experience, I have not found Xubuntu to be faster than Ubuntu.  Also, I have not experienced the great speed improvement of Ubuntu over Win XP of which others on these pages speak so enthusiastically.  If your Win XP is not working anyway, you can't lose anything by installing Ubuntu (Or Mint 9) to see how well it works, although 512mb RAM is rather low for these distributions.  Puppy Linux 5.0 should work very well on your computer and is based (somewhat) on Lucid Lynx. I wasn't able to get the wireless working with the Puppy live CD on my 4 year old HP laptop but I think I would have had more options with a hard drive install.

BTW, I got wireless working with the Lucid Lynx live CD on the old laptop where earlier Ubuntu versions did not work.
