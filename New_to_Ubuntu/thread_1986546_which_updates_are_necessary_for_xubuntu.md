### Post by rai4shu2 on 2012-05-25
Xubuntu is Xfce, but it also includes a lot of Gnome apps, as well. For example, if you use file-roller (Archive Manager), then you still need "nautilus-data" for that to work properly. You don't really need any Unity or Ubuntu One stuff (unless you use that).
