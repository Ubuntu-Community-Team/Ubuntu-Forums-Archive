### Post by Mark76 on 2009-04-16
Mail notification is one of the things that makes Linux easier to live with than Windows as far as I'm concerned.  With Windows and Outlook you either have to keep the application open in the task bar (no minimise to systray/notification area last time I looked) or check manually every so often to see if mail's arrived. In the home environment this isn't such a big deal. But in the workplace where you might receive internal and external emails regularly during the course of the day it's a major inconvenience. Especially if said emails need to be acted on urgently.

### Post by Alekz_ on 2009-04-16
Thunderbird do what you want!

You just need to install an Addon and you'll have a system tray icon in your notification area (while Thunderbird is on execution) and the same icon will change to a letter when a new message arrives!

To install thunderbird, just type:
apt-get install thunderbird

and here is the URL for the addon:
[http://moztraybiff.mozdev.org/releases.html](http://moztraybiff.mozdev.org/releases.html)

PS: It works like MS Outlook! Easy to use! :)
