### Post by fantab on 2013-07-05
If you are using 9.04 then let me remind you that it has reached its 'End Of Life'. NOT SUPPORTED ANYMORE.
You Must upgrade to a latest supported Ubuntu version, like 12.04, 12.10 or 13.04.

However you might find this useful: [http://ubuntuguide.org/wiki/Ubuntu:Jaunty](http://ubuntuguide.org/wiki/Ubuntu:Jaunty)
For the latest Ubuntu try this: [http://ubuntu-manual.org/](http://ubuntu-manual.org/)
Official Documentation is [HERE]("https://help.ubuntu.com/"). And you can find Community Documentation [HERE]("https://help.ubuntu.com/community").
