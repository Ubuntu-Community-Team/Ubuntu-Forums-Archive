### Post by Ceannfaolaidh on 2012-07-07
If you're looking for tools to make your photos better (but not necessarily do content-heavy edits), here are a few tools you should definitely take a look at.

[RawTherapee]("http://rawtherapee.com/blog/features") is a fantastic tool for working with and converting RAW format images. It's got great support and a helpful community. [Install RawTherapee]("apt:rawtherapee")

[Hugin]("http://hugin.sourceforge.net") is a panorama photo stitcher with some great point tracking and perspective views. [Install Hugin]("apt:hugin")

[Luminance]("qtpfsgui.sourceforge.net") is an HDR photo compositor. It has some quirks that may take getting used to, but it does a great job. Unfortunately the Ubuntu Software Center version is fairly out of date, so if you want the most recent iteration you'll need to build it yourself.

Umm...no. Photos are bitmaps, where every pixel is a certain color, also known as raster graphics.
Vector graphics is not really graphics. A .svg file is actually **_plain text_**. Seriously. Open any vector in a text editor. You get something in this format:
```
<?xml?>
<svg>
<g>
<path (a bunch of maths)/>
</g>
</svg>
```

### Post by AlbertJB on 2012-07-09
Regarding Photo management, I will recommend Shotwell for its simplicity, if you want to go more pro get Digikam.

Regarding quick editing screenshots or pictures (copy&paste, select area, cut, scale/resize, etc.), I would choose KolourPaint. It's the only simple image editing app that have all the features I need to send quickly modified screenshot or image to a friend, for instance..

Of course, the king of photo editors continues to be GIMP, if you are a PRO and need all the resources and options it has.
