### Post by mali2297 on 2008-08-03
> **im new here said:**
> i dont get it what is the difference between free of charge and open source i dont want to use somthing illeagle

It is a philosophical issue. Realplayer is the official player from RealNetworks and perfectly legal. The point is that Ubuntu and the community around it prefer programs whose source code is available for anyone to view and modify. See [http://www.ubuntu.com/community/ubuntustory/philosophy](http://www.ubuntu.com/community/ubuntustory/philosophy).
