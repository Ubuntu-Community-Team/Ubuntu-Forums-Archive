### Post by Apv507 on 2010-05-10
This is a possible solution, first go to my computer in Windows 7 (as if you had a choice yet right?).  Right click and go properties. You should see a window with your Experience rating and other system info. Now click advanced System settings on the left hand side, it should be the 4th down. A window called "System Properties" will open, and it should be on the advanced tab my default, if not go there. Now click settings under the startup and recovery section. Another window opens and It allows you to select a default operating system and set the time to display list of OSes and some other options. You can either set the default to Ubuntu and then use grub to choose between Ubuntu and Win7 or you can click the time to display list of OSes and set it to about 10 seconds (or a time you prefer). I'm not 100% sure that this is your problem but it could very well be. Let me know how it goes!

If my explanation didn't make sense try this: [http://www.howtogeek.com/howto/windows-vista/easily-set-default-os-in-a-windows-vista-and-xp-dual-boot-setup/](http://www.howtogeek.com/howto/windows-vista/easily-set-default-os-in-a-windows-vista-and-xp-dual-boot-setup/)
Its a link to set the default but it will tell you how to get where you need to go and has pictures, which are always helpful!
