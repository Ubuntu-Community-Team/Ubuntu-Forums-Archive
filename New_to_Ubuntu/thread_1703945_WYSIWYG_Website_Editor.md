### Post by deconstrained on 2011-03-10
WYSIWYG is obsolete for website editing.

The new web paradigm (actually it's been around for a while now) is to create a CMS template with well-written HTML/CSS/PHP and never having to touch it once you have it set up and working, and then doing all the page/site editing through the CMS.
