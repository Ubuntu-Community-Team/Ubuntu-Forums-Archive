### Post by nzjethro on 2011-05-31
I have dual booted Ubuntu 10.10 alongside Windows 7 for a couple of weeks now. I really enjoy Ubuntu, and haven't used Windows since. I don't game much, so I've found that Ubuntu meets all my software needs (LibreOffice instead of MS Office, GIMP instead of Photoshop, and Transmission for torrents). Also, the look and customisability of Ubuntu is fantastic, much better than Windows.
Also, I did an introductory Java programming course at uni, and have since dabbled in Python (de facto language of Ubuntu) which really helped. Open up Geany and a YouTube Python tutorial and see what happens. It's pretty useful learning to think like a programmer, it helps a lot with using a terminal or solving some of the more difficult troubleshooting problems I've come up against.
Most importantly though, remember that there is a lot of help around. It seems like whatever problem I encounter has been encountered and solved by someone else. :P
