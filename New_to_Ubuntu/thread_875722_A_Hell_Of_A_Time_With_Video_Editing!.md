### Post by purplepaint on 2008-07-31
If you want a video editor that runs well on 2GB of RAM and is fairly advanced, try LiVES.  I've been using it for a while and, while it's still at the beta stage of development and quite prone to crashing unless you save frequently and you're careful, you can get quite a lot of professional-looking editing out of it.  The website was listed by KIAaze ([http://lives.sourceforge.net/](http://lives.sourceforge.net/)), where it links to [http://www.getdeb.net/app.php?name=lives](http://www.getdeb.net/app.php?name=lives) of you don't want to compile it from source (which I didn't).

### Post by raptor2552 on 2008-07-31
Sadly, movie editing software in Linux is still in it's infancy, I was displeased with the many I tried. My solution is to run Win XP as a virtual machine under VirtualBox and then Sony Vegas, which I must say runs really well and possibly better than under native Windows. Vegas is the only reason I keep XP.

You might want to consider this as an option with whatever editor you used to use. As far as virtual machines go there are several options, Sun's VirtualBox suited me best and if you want to try it out Google virtualbox do not use the OSE (Open Source Edition) found in the repositories.
