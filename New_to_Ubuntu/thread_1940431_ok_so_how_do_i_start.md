### Post by audiomick on 2012-03-13
If your Vista install is beyond repair, there is not much of a reason to not just install Ubuntu over it. You _***must***_ back up anything on there you want to keep first. Unless you have a separate data partition that you retain, everything will be gone. If you then decide to go back to Vista after all, you have not lost anything. It needed to be re-installed anyway.

Otherwise, idotprone's advice is fine.
