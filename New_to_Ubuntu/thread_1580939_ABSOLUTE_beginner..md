### Post by mcduck on 2010-09-24
What comes to MSN, Pidgin should wrk just fine, but you could try Empathy (which is installed by default in any recent Ubuntu version) or some other IM application instead.

There is no Linux version of iTunes, and I really doubt Apple would ever make one.

OpenOffice is capable of both opening and saving using Microsoft's document formats, and is also able to export to PDF documents, so you should have no problems there.

And no, Ubuntu doesn't require you to run any code to make manually-downloaded things to work (or it might, it depends what you actually downloaded. But since youa re beginner it's pretty safe to say that you shouldn't be downloading anything at all). It's just that the installation of software in Linux is very different from Windows or OS X, and you generally *don't* download stuff manually. You don't need to, when most of the programs are easily available by the package management tools, which will handle downloading, installing, updating and all the other such tasks for you very easily.

Just keep in mind that all programs are made for certain operating system, and in general they will not work on others. Windows programs don't work on OS X or Linux, OS X programs don't work on Linux or Windows and Linux programs don't work on Windows or OS X. (there are ways around this, like Wine for running Windows apps on other operating systems, and Linux apps, being open-source, can quite easily be ported to OS X.)

### Post by irishbreakfast on 2010-09-24
Welcome to Ubuntu!
And congratulations on finding your way here.

1) I wish I could tell you how to get MSN working. I've done it but is was a while ago. I am quite confident that it was with Empathy.
2) Our family abandoned iTunes and it has made my life easier. It is likely you'll need to add some 'stuff' to Rhythmbox. Fortunately, this is easy, have a look at [https://help.ubuntu.com/community/Medibuntu]("https://help.ubuntu.com/community/Medibuntu") when you are ready. 
3) OpenOffice will read and write MSOffice files (except for MSAccess). I've done some 60 page documents this way and have found that it is not the size of the document but what features it has can cause quirks. Tables usually but not always convert just fine, but that improves all the time. 
You may want to look at [PortableApps]("http://portableapps.com/"). Portable apps lets you put portable versions of programs (OpenOffice and others) on a USB so you can take it anywhere. This allow me to keep my files in the format I want and still use them on machines running Windows (like your roommate's)
4) The Software Centre is wonderful. Easy to install and remove programs as you see fit (and no need to reboot afterwards).
5) My teenagers use ubuntu with growing confidence and never use the command line or open a terminal, it is all point and click for them.
