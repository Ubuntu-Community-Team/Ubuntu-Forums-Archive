### Post by aljames2 on 2020-04-13
It depends on you.  If you need the gui environment to run things, or you want to potentially do other gui stuff, and IF your hardware can handle the Desktop version including Plex and still give you the performance you want then go with Desktop.  On the other hand, if you are ok with the command line for management and prefer the light-weight speed as Server uses less resources then I'd go Server all the way.  But it really depends on your experience and comfort with the 2 options.

### Post by DuckHook on 2020-04-13
Since you are going headless, it makes little sense to install a DE.

Here's the excellent rule-of-thumb that for so many years has kept Linux just a tad more secure than those other OSes out there&#8230; /sarc

Always strive to present the smallest attack surface to the bad guys. GUIs have far larger attack surfaces than CLIs. So in the Linux-sphere, servers are practically always sans-GUI.
