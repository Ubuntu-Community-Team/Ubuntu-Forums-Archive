### Post by aloshbennett on 2008-09-04
Not a fine solution, but you could say 'Open with -> Open with other application' and select vlc or mplayer.
Gnome would remember your choice and next time onwards it would show up in the list.

### Post by ntbutler on 2008-09-04
As ashlonbennett said, do the Open With option and choose your program that you want (such as mplayer). Your option should show in the open with window next time you look. It should, but may not be the one checked (with the radio button) in the list. Check it, and it should then remain as the default, but if you do it on an mpg file, it will only be saved for that file type, so you may have to change it for other file types such as avi and divx.
(I am not sure about this because i am not at my machine at the moment, but i think in System->Preferences there should be a preffered applications option. Have a look in there, you may be able to set a default media player, but again, i am not sure about that one at the moment)

### Post by mcduck on 2008-09-04
Don't use the right-click -> open with, but instead of right-click -> properties -> open with. That will allow you to add new programs & change the default.
