### Post by bedpotato on 2012-04-08
I don't know if you are using the same version as me, so all the buttons and things might look slightly different, but it might help someone else reading this thread.

I'm a newbie and I had the same problem as you. Here's how I find things after installing them. I do it by clicking on "Dash Home" and then a search box will come up, into which I type the name of the program I've installed. Then an icon for that program will come up, and I click on it and it opens. 

Here's how I managed to create a permanent visible shortcut for Chrome (without having to keep typing it into a search). 

While the program is open and I'm using it, a shortcut icon to it will temporarily be installed on the sidebar on the left. If you want that shortcut icon to remain there even when you've closed Chrome, right click on it and select "keep in launcher."

I don't know if "shortcut icon" is the right terminology, but I hope this helps some fellow newbie. :)
