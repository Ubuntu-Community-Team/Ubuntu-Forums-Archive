### Post by grahammechanical on 2015-11-06
You are posting in the wrong section of the forum. It should be in the Linux, Ubuntu and OS chat section or even the Cafe.

A Linux distribution is made up of hundreds and hundred of packages which are binary files, libraries and scripts. I would be surprised if there was one person who understood how every thing fitted together. Does a developer need to know how everything fits together? Or just how their project fits in with the projects around it?

A person could spend hours studying and then discover that progress has passed him by. So, a person finds out how X fits in to everything else and then X is replaced by compositors using the Wayland protocol or if it is Ubuntu were are talking about, then by Mir. You study the folder structure of Ubuntu and you install Ubuntu Personal and the file structure looks familiar but different somehow.

The question you have not provided the answer to is: Why do you have a need to know?
