### Post by dBuster on 2010-02-02
Could mean the drivers need to be looked at is why you were asked about your graphics card.

As far as running it in a virtual window within windows, you are using quite a lot of computer resources to run windows alone and then add on top of it the virtual window eating up more resources...  

Better off NOT running it like that and run a dual boot if you must retain windows.  I have ditched windows a good 4 years ago now and only use it at work, no choice there.

Might want to look at how much RAM you have in your bog Dell machine as well as if the video memory is shared memory.  Could be something as simple as that with the way you are running it.  

Not sure if this might be an issue, but do you have 64bit or 32bit windows?  Are you running the same Ubuntu?  Meaning 32bit windows and 32bit Virtual Ubuntu...  Might want to look at that.

By pulling the plug your windows installation may be corrupt or be in need of some repair and is causing your issues.  

Just some food for thought....
