### Post by Flimm on 2008-10-06
Which version of Ubuntu are you using? 947 is a lot of updates, methinks.
If you click on an update, you can then click on the description tab to find out what the package is.
My handy tip: if you right click on one of the updates, you can untick all the updates, allowing to only update the packages you want.
You might want to uninstall some packages that come with Ubuntu if you know that you're never going to use them. That way, you won't have to download updates for them.
Oh, here's another tip: if you click System, Administration, Software Sources, and then Updates, you can only select security updates.

### Post by drs305 on 2008-10-06
As a minimum, go to System >Admin >Synaptic >Settings >Repositories >Updates (or System, Admin, Software Sources) and make sure "important security updates" is checked.

For the most conservative user, you could have only the security updates  checked. If you start having a problem with a particular package, you could to a forum search for the solution or perhaps check the "Recommended updates" for that particular application. You could update only the problem program and then untick it again. 

That being said, "Recommended updates" should be ok to use on a regualar basis (that's why they are recommended) but it's up to the individual user to decide.
