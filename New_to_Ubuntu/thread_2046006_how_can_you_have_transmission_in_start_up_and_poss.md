### Post by sepo on 2012-08-22
You can add the Transmission client to 
[B]System > Preferences > Startup Applications
[/B]to have it automatically loaded on startup.

To have it in a specific workspace I think you can right click on the program (while running) then select "only in this workspace" (it will always runs in that workspace).

An alternative can be to turn on the "automatically remember applications when logging out" (System > Preferences > Startup Applications >  Options) to save the application position.
