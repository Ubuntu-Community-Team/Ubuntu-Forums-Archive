### Post by grahammechanical on 2016-01-30
As a general rule, the newer the hardware the newer the release of Ubuntu. And this is because the 6 monthly Ubuntu release schedule is designed to keep Ubuntu up to date with progress made by the Linux kernel developers. And it is the LInux kernel that responsibility rests for hardware drivers and not with Ubuntu.

Every 2 years we get a release of Ubuntu that has 5 years support for security fixes. We call these LTS releases. The last LTS was 14.04 released in April (.04) 2014 (14.). To stop these LTS releases becoming out of date with the latest hardware the LTS release will get what is called a point release. So, we have had 14.04 and 14.04.1 and then 14.04.2 and then 14.04.3. 

On 14th February 2016 we will get 14.04.4. So, if you decide to install 14.04 because it has 5 years support starting from April 2014, then install 14.04.4. It will come with the same Linux kernels in Ubuntu 15.10 along with appropriate driver modules.

In between LTS releases we get Interim releases that only have 9 months support. So, if you install Ubuntu 15.10 you will have to upgrade to 16.04 LTS between May & July this year. We get notifications of new releases and upgrading is a one click operation. You will also be able to upgrade from 14.04. to 16.04 by a one click operation.

In the past it was said of Ubuntu that it was Linux for humans. The intention was to produce a distribution of Linux that did not force the user to use the command line. I would say that the developers have succeeded in that. If we go the System Settings>Software & Updates>Additional Drivers tab there is a utility that will find additional video and WiFi drivers for us. Select & click to install. We need to be connected to the internet. 

We can test all this out by running a live session from DVD or USB memory stick.
