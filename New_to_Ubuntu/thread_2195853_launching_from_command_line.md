### Post by kajoky on 2013-12-26
You can click the top icon in the launcher, Dash Home
 On the next screen, click the second icon from the left at the bottom.  The first one looks like a house.
Now, there should be an item on the screen that says: Installed see x more results >
Click: see x more results >
You should now see all the icons for the programs installed.  Click one to start that program.
Once it starts, it's icon will be in the launcher.
 Right click on it's icon in the launcher and a menu will appear with one option being: Lock to Launcher
If you click that option, that program's icon will stay in the launcher after you close the program to make it easier to launch next time.

Repeat each step above for each program you want to stay in the launcher.
Once a program is locked, you can following these steps and choose Unlock should you want it removed from the launcher.
