### Post by cariboo on 2010-12-28
My younger brother copied a bunch of files to one of  our systems, I'm not sure if he did it on purpose, but all the file names started with a dot and a space. The files showed up as read only on windows, but were hidden when I copied them to my server. I didn't look at the file names, and was surprised that they didn't seem to have been copied to the server. After copying the directory with the same results, I looked at the file names, by pressing Ctrl-H in nautilus. Check the file names, to make sure they don't start with a period.
