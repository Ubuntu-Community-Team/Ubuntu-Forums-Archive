### Post by matt_symes on 2012-06-05
Hi

That's because the repositories are for Jaunty. This has reached EOL.

What version of Ubuntu are you using ?

If you have just installed Jaunty then you will want to upgrade to 10.04 LTS, 11.10 or the new 12.04.

Open a terminal and type

```
cat /etc/lsb-release
```

Copy and paste the results back here by highlighting the text->right click->copy and paste it into your next post.

Also post the results back of the command

```
cat /etc/apt/sources.list
```

Kind regards
