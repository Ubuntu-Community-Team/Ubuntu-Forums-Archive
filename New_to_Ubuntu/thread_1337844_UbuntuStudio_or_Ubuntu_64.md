AMD 3800+ Athlon 64 processor

Now, he's interested in the Studio type apps in Ubuntu Studio. Should he run Ubuntu Studio 9.10 or install Ubuntu 9.10?

If he wants to run the programs in Ubuntu Studio, then it will be easier for him to install Ubuntu Studio. It's possible to use ordinary Ubuntu and install the Studio programs on it, but why not just start with them already installed.

> And should he use the 32 bit version or 64 bit version?

1 GiB of RAM... probably the 32-bit version. The 64-bit version would be fine, but it takes up a little more RAM. If you double the RAM in the computer, you'll probably see a slight improvement with the 64-bit.
