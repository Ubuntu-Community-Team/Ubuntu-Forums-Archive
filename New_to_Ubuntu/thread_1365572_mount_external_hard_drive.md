### Post by Lateralis on 2009-12-27
All drives will be listed under /dev if they are connected.  They are only mounted to /media (or to /mnt, or to some other arbitrary location) if that is where the operating system is told to mount the filesystem.  

To the OP... Firstly, sda* will be partitions on the internal hard drive, not the external one.   Secondly, under the /dev/sda3 entry, it says that the file system type is Ext'd.  If it is an external hard drive I'd expect it to be FAT 32 (or less commonly NTFS) unless you have specifically reformatted it otherwise.  Thus you are getting errors as you're trying to force it to recognise it as a Window's format when it isn't.  

Now, if you did the fdisk whilst the device was connected and it isn't listed, then there's the chance the hard drive isn't working or it isn't recognised.  In which case, can you tell us what external hard drive you are using (make and model etc) and can you open a terminal, type in the following commands and provide us with the output: 

```

sudo lsusb
blkid

```


Thirdly, external hard drives should automatically mount when connected to the operating system.  Which version of Ubuntu are you using?
