I, like many, have a keyboard with those "extra" buttons at the top above the function keys. You know, the ones that let you change volume, play/pause music, print, or start a program. In which these keys really only work with the computer the keyboard came with because the software is installed on that computer.

### Post by mprince on 2009-04-12
You might want to take a look at:

System>Preferences>Keyboard>Layout tab

Then click on the 'Keyboard model' button and see if one of the choices matches your keyboard.



You Might want to try here also. I think you can configure just about any of them.
[B]
System>Preferences>Keyboard Shortcuts[/B]
