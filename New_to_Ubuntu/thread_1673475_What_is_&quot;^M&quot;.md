### Post by SeijiSensei on 2011-01-22
If you're routinely moving files from DOS/Windows to Unix, install the package "[tofrodos]("http://packages.ubuntu.com/maverick/tofrodos")" from the repositories.  You'll then have two utilities (todos and fromdos) that will add or strip the additional linefeed character from files.

### Post by JKyleOKC on 2011-01-22
To answer your original question, that's the character entered by the keyboard when you press the "enter" key. Many other systems store two characters, return and linefeed, at the end of each text line. All *nix systems, though, store only linefeed (and make the translation automatically when you press Enter).

The other responses show you a number of ways to get rid of these unwanted "return" characters.
