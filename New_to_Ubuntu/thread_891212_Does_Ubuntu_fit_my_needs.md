### Post by 3rdalbum on 2008-08-17
Definitions time:

Photoshop is not a vector graphics editor, it is a bitmap (raster) graphics editor.

A raster graphic consists of a mesh of pixels, with each pixel defined as a colour. A photograph can only be stored as a raster graphic. A raster graphic says "Pixel 1 is yellow, pixel 2 is red" etc. Raster images require a lot of disk space because every pixel is defined.

A vector graphic consists of calculations that define to the computer how to draw lines, squares, fill in objects with colours. The computer draws these using mathematics. A vector graphic just says "Draw a red line from here to here" or "Make a circle this big and put it here". Vector graphics are used for drawings, but cannot define photographs.

If you want a raster graphics editor on Linux, there is The Gimp, Krita, or Pixel (which is commercial software); or you can run earlier versions of Photoshop in Wine. If you want a vector graphics editor, Inkscape is the most popular one for Linux.
