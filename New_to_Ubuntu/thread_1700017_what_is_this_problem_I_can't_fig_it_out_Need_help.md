### Post by maqtanim on 2011-03-04
Ubuntu has three ways to install softwares. 
1. Ubuntu Software Center
2. Synaptic Package manager
3. Terminal

From the above three, at a time you can use just one. So if the the other two are running then it will cause problem.

To solve this problem, close all above three programs. Then start/run only one of them to install software and install the software using that program.
