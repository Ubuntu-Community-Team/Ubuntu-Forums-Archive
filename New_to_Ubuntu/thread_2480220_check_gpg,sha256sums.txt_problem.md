How about looking at the output of 'sha256sum' without filtering ? That should tell you more about what's wrong:
```

sha256sum -c SHA256SUMS.txt

```
This should print one line for each file listed in SHA256SUMS.txt and the state of the check, either 'OK' or 'FAILED'. The filter you had after sha256sum removes all lines that don't contain 'OK'.
