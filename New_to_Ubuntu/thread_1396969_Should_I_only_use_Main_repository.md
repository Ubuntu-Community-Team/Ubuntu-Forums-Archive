### Post by mcduck on 2010-02-03
> **Missing_Number said:**
> Should I only use the main repository?  How long does it take for "Universe" packages to become "Main"?  

If I do use any "Universe" packages, should I only use the ones that are out of beta?

Packages in Universe will never become "Main". Or at least unlesss the package becomes so popular that Canonical starts maintaining and supporting it.

Universe contains community-maintained software, Multiverse software that is resricted with copyrights, patent issues etc. So neither repository has anything to do with beta testing or anything like that.

(The "Proposed"-repository contains testing versions of packages, when a package is updated it's available in Proposed from couple of days to couple of weeks before it's moved into normal repositories.)
