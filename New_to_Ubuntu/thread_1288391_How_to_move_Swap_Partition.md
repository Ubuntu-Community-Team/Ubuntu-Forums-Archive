### Post by Boondoklife on 2009-10-11
There is a resize/move option you can use (if available due to partitions that exist already), that you can use to do just that. Your statement about the swap being towords the first sectors is best, but if you have a fair amount of ram (2GB here) then you really wont touch it often so it wont matter.

It does take a fair amount of time to move that data around so if you do it make sure you have a good game or some other stuff to do in the meantime.

### Post by louieb on 2009-10-11
