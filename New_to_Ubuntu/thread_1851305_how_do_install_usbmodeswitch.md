### Post by 3rdalbum on 2011-09-28
usbmodeswitch is preinstalled, I believe.

You don't necessarily have to "unmount" the virtual CD.

I mean, did you try just setting up the connection in the Network Manager applet? Click on the network applet on the top panel and come down to Edit Connections... and then go to the Mobile Broadband tab and click Add.

You might need to "eject" the virtual CD, but then you might not.

Don't "Safely Remove" the virtual CD because this will remove power to the USB device.
