### Post by Paddy Landau on 2010-06-12
> **dcn427 said:**
> ... installing Adobe flash player...
As far as possible, install and uninstall *always* through Synaptic (Ubuntu Software Centre, Synaptic Package Manager or command-line apt-get; they're all the same).

[LIST=1]
[*]Start Ubuntu Software Centre.
[*]In the search bar, type *ubuntu restricted extras*.
[*]Click on *Ubuntu restricted extras*, and click Install.
[*]Wait for it to finish.
[/LIST]
This should install your Adobe Flash plug-in for you. If not:

[LIST=1]
[*]Return to Ubuntu Software Centre.
[*]In the search bar, type *adobe flash*.
[*]Find *Adobe Flash plug-in* and click Install.
[*]Wait for it to finish.
[/LIST]
