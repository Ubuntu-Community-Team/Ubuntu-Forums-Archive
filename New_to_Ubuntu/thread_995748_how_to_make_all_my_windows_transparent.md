### Post by 3rdalbum on 2008-11-28
If you want background windows to become transparents, you can use the ADD plugin in Compiz Config. If you want them to become a particular transparency based on how long since you've used a particular window, then you want the Trailfocus plugin. Both these plugins should be preinstalled on Ubuntu, as long as you've got Compiz Config Settings Manager.
