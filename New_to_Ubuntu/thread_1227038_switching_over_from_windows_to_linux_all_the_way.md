### Post by Calmor on 2009-07-30
> **lestatius said:**
> I'm just wondering for those only running on ubuntu. How long did it take you to switch all the way to ubuntu from windows?

I consider myself as someone who "mainly" runs linux.  I can't say "only" because there are a few programs which I need - Quicken, for one, and Visual Studio, as I do a bit of coding (as a hobby) for Windows projects.  

That said, it took probably less than two months to switch my main computer to linux.  It does have a VirtualBox Windows XP on it, though, for those things I have to have.  

It took a little longer to switch my HTPC over from Windows/BeyondTV to Mythbuntu.  I'd say the experience has been both gratifying and frustrating, as I still fight the occasional demon, but overall am much happier with the outcome.  That box also runs Wine so that I can play Portal and HL2 (which don't run quite as well as they did natively, but they're acceptable).

>  I actually like to game alot and I've noticed that it's not possible to run all windows games on linux; ie: Arma2. 

Yeah, there are a few linux games, but most of the big/popular ones are Windows and won't run well in a virtualized environment or in Wine, though that is changing with every Wine release.

>  I also need windows cuz I'm doing a windows course for work while linux is just a hobby, but I figured I can use vmware, so that won't be much of a problem. 

That's how I do it for school.  The only time I get stuck is when I need direct hardware access - I was taking an assembly language programming class that required this.

>  Right now I'll just stick to dual boot, until I'm familiar enough with linux and how to get around linux and how I can get things done right or figured out right.

That's probably the best plan - find out if it's right for you.  When you find yourself rarely if ever returning to the Windows partition, you'll know you're ready.  Then again, there's no reason why you can't keep the Windows partition around, unless disk space is a serious issue.  

On my laptop, I actually gave Windows the biggest chunk of the hard drive, and just store all of my media (school documents, pics, music, videos, even VirtualBox hard drive files) on the NTFS partition.  That way, regardless of which way I boot up, I can access all of my information.
