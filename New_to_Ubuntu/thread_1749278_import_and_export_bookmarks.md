### Post by coffeecat on 2011-05-05
> **saradisn said:**
> I can't see the Import and Backup in Show All Bookmarks. Are there hidden somwhere?

If you are running in the Unity/Natty desktop the Bookmarks manager menu is in the panel - it's a global menu too.

I'm intrigued that a couple of posters can't find Bookmarks > Show all Bookmarks in Firefox 4.0.1. Just to repeat in more detail, this is what works for me:

Running Firefox 4.0.1 in Natty/Unity with the firefox menu in the panel (global menu). Bookmarks > Show All Bookmarks, **or** ctrl-shift-O, the "Library" window opens. Then, the menu for the library is in the panel - the usual 'Organise', 'Views' and 'Import and Backup' menus.

To those having difficulties: is the firefox menu in the panel? You need to hover the mouse over the left end of the panel for the global menu to appear.

### Post by weedwacker on 2011-05-06
I found it.

You have to enlarge-expand the Bookmarks Library window to full page by clicking on the square symbol at the top and then when it is full screen you can get to the Import and Backup by hovering in the very top bar.

Phew!
