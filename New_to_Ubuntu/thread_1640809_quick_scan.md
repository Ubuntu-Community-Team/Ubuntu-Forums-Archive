### Post by QLee on 2010-12-08
> ... known intrusions that are a problem for Ubu these days..??? if there are any....

The simple answer is there aren't any. If there were known ones, there would be a multitude of posts about them on the forum.

I expect you're thinking about something like chkrootkit or rkhunter. However, those really should be run from somewhere other than a suspected compromised system where you couldn't trust the binaries, because on a compromised system the bad guy(program) has probably already covered the tracks. Most of what you would see would probably be worthless to anyone except a security knowledgeable user anyway and just leave you with questions.
