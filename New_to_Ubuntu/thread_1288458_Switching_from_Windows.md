### Post by Marlonsm on 2009-10-11
Switching is not easy, it'll take some time for you to feel comfortable in Ubuntu, so you might want to keep both Windows and Ubuntu in dualboot, this way you can choose which one to use when you turn the computer on.


Ubuntu will be able to share files with Windows and to use them.

But your programs from Windows might not work, you'll need to find Linux alternatives or to use them under Wine (a program for Linux that runs Windows programs, it's not perfect yet, tho).

### Post by wubiboy on 2009-10-11
MsWriter

A really safe way of trying Ubuntu on your PC and still leave Windoze on it is to install Ubuntu using Wubi.

Wubi creates a Veritual partition on your hard drive where you can do a full install of Ubuntu and then your machine will dual boot to either Windows or Ubuntu.

If, after you try Ubuntu, and you do don't like it for what ever reason, you can simply uninstall Wubi through your Add Remove Programs in Windows and the Install will be gone.

For More information on Wubi you may go here> [http://wubi-installer.org/](http://wubi-installer.org/)  It's actually quite easy to get up and running.  I use wubi on my laptops in the trucks so my technicians can boot to either Windows or Ubuntu while on a service call.

Hope this helps.
