### Post by fantab on 2013-04-10
Welcome!

First of all, Why do you want Ubuntu-32bit? I would recommend that you use 64bit.
I think you are trying to install Ubuntu with WUBI inside Windows. I don't recommend that. Instead you use Ubuntu LiveDVD/USB to 'Try Ubuntu' without installing it first. The "Try Ubuntu" option will load a working ubuntu desktop without installing it. Check the performance and confirm that everything work fine. Then you can proceed with Installing Ubuntu as DUAL BOOT with Windows.

Good Luck...
