### Post by clive littlewood on 2009-06-12
Hi

Virtualbox creates a virtual drive on your Ubuntu box to install a different OS within Ubuntu.

You have to install XP or any windows OS then run your windows program from within this virtual drive.

If you will need USB connection on your virtual OS then install the free version from the sun site as the OSE version in the repos does not support USB.

Hope this helps
