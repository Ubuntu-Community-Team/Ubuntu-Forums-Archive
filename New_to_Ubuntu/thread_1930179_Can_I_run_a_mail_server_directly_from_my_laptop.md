### Post by HermanAB on 2012-02-23
Yes you can run a mailserver on a laptop machine.  I have run Citadel on an EeePC ([http://www.aeronetworks.ca/howtos/eeepc-mdv-howto.html](http://www.aeronetworks.ca/howtos/eeepc-mdv-howto.html)) - towards the end of that.

However, you would need to forward mail through your ISP mail server, because the laptop IP address will change each time you plug it into the internet.  Citadel can do that too of course.  Citadel is very old, can do everything and supports all mail protocols ever invented.
