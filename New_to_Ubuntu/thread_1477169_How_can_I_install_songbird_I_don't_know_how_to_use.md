### Post by jbrown96 on 2010-05-08
Songbird doesn't really install. You just need to extract it to a folder, then run songbird executable inside. You will probably want to create a shortcut or something to this file.

Last month, Songbird announced that they are not going to support Linux in subsequent releases. You might not want to get too tied to it.
