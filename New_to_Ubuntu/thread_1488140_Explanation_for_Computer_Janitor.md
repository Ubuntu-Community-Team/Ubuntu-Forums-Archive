### Post by howefield on 2010-05-19
I'd leave the linux headers alone, if the printer driver packages it wants to delete are simply the downloaded install files, then you are safe enough to go ahead.

To be honest, I don't have much faith in the Computer Janitor, it too often wants to remove useful stuff, (or at least, it used to) you have to be careful with it.

There are probably better ways of keeping your system "clean"

Bleachbit for example is a useful tool. It's in the repository, take a look at it.
