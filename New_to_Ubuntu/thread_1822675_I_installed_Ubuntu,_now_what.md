### Post by TBABill on 2011-08-11
Not really a "fun" thing, but it'll remove some headaches for you. Go into Synaptic or Ubuntu Software Center and install "ubuntu-restricted-extras". That'll get you a bunch of codecs including Adobe Flash. Then go into Software Sources and enable the Ubuntu partner repos, then update, then search for Sun Java and install the Sun Java Plugin. Open Java gets installed by restricted extras, but Sun Java is far superior. Then you can go enjoy your web experience much more pleasantly.
 
In Software Center you can find a lot of games, browsers and other apps that may be fun to practice learning to install software, remove it, etc. 
 
Speed....just enjoy how fast Linux is.
 
Most things you can do with a GUI, but to learn more you can search for Linux commands to do things via command line, which is normally faster if you are experienced with it. First things to learn are how to update, how to install software, how to look at your hardware and other basic OS tasks.
