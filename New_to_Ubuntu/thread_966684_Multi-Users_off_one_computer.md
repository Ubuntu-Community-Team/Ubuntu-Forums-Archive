### Post by Nepherte on 2008-11-01
1) Linux is designed from the start with multi-users in mind. So yes, you can use several users simultaneously on one computer. In fact, several important services run with their own user account as well. I'm not familiar with Userful so I can't compare with it
2) A normal user is never able to delete/edit system files as you need root privileges for it. Just make sure they don't have it.

It sounds like you might be interested in the so-called kde kiosk mode. Just do a search on the forums here or on google.
