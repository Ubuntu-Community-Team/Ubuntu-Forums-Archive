Use synaptic to install wine and its dependencies.

Download OperaTor! :

[http://archetwist.com/opera/operator](http://archetwist.com/opera/operator)

Navigate to the unpacked folder, right click operator.exe, properties, set to  open with Wine.

Double click operator.exe and you should be surfing!
