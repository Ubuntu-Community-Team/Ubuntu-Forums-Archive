### Post by squakie on 2012-11-02
I know there is a post somewhere about what programs will work or won't work and possible Ubuntu alternatives, but I don't know where it is at.

In the mean time, if you have relatively new hardware, you may want to consider running Linux as your primary OS, installing VirtualBox, then install Windows XP in a virtual machine in VirtualBox.  That way XP will be just another process to Linux, yet you can install your Windows programs there and use them.  Hopefully while doing that you can try some of the native tools side-by-side to see the differences and how to make something work.
