### Post by Jose Catre-Vandis on 2011-04-23
Gnome Planner will import MS Project 2002 and above if the files are exported from MS Project as xml. You will lose some stuff along the way though, as planner is not as fully featured as MS Project. Good for the basics of project management though :)

Search for **planner** in the repos
