### Post by Boyntonstu on 2012-05-11
Making some progress in booting 11.10 on a USB flash.

I needed to know what was the fatal error.

I learned the following using Google search:

When booting, hold shift and you can choose 'recovery mode'.

Follow to boot and you will have 11.10 running in terminal mode.

Log in.

Change directory:

cd /var/log

enter ls to see the files.

enter nano Xorg.0.log 

This micro editor (nano) will allow you to view the boot log file and you will see where the boot stopped.

From 1 noobe to others.
