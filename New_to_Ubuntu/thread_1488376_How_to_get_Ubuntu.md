### Post by Chlorhydrikk on 2010-05-20
Using a torrent would spare time to you and money to Canonical.
Downloading directly from Ubuntu will be the easiest way.
Ordering a live CD from shipit.ubuntu.com or using an old one and then using the Update Manager to get the final code.

To install it, you can either burn it to a CD or make a bootable usb device. I have heard that you can install it inside Windows (Wubi installation) with a CD drive emulator (like Daemon Tools, Alcohol 52/120%...)
