### Post by lolpenguin on 2012-02-02
Ubuntu is a GNU/Linux distribution, an *free* (as in freedom) Operating System that uses GNU as a base and the Linux kernel. There are many many other distros out there, to name a few: Debian, Fedora, OpenSUSE, Arch.
GNOME stands for GNU Network Object Model Environment, but the name is out of date, so don't worry. Like others have said, GNOME is a Desktop Environment. Other desktop environments have different features, different levels of customisability...etc...> Gnome is one of the desktop environments (what you see, when you log in), which is used by Ubuntu version 10.04 LTS. Do you have that version?Ubuntu has always used GNOME, and still does, but now with their own shell, Unity.

It's not _hard_ to use GNU/Linux, but as always, there is a learning curve. Many applications you have used in Windows are actually *free*: Firefox, Chrome (as Chromium), VLC, etc.

Since GNU/Linux is a Unix-like OS, and not DOS like Windows, you can't run Windows applications. Most have a equivalent, although you can run a growing number of them using Wine, a open-source re-implementation of the Windows API. It isn't complete, so don't expect everything to work, but some stuff does.

### Post by 3rdalbum on 2012-02-02
That computer is probably running Xandros Linux, not Ubuntu; those computers came with Xandros.

Ubuntu is not difficult to use, but it is different to Windows. As you've discovered, it doesn't run Windows programs any more than a Japanese person can speak Latin.

If you can't play videos and music on this computer, then you don't have the correct codecs installed. A codec is a little piece of software that tells the computer how to understand the sound or video data. If you can confirm that it's Ubuntu running on this computer, then we can help install the codecs you want - it's pretty easy, but once again very different to how Windows does it.
