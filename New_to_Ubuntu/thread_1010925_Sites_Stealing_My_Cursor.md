### Post by sillyxone on 2008-12-14
Basically, this happens when Javascript hides a textbox with cursor currently in it. The website programmer just need to be careful to move the focus somewhere else before hiding the active textbox, or the browser could be smarter to detect that and move the cursor automatically.

Yeah, Noscript would solve the problem. Or in some cases, you could manually put the cursor in another box that will stay on the page before performing the action will kill the cursor.
