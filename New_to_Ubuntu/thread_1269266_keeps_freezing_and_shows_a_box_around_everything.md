### Post by MaxIBoy on 2009-09-18
As far as I know, the screenshot button on the keyboard doesn't work if a menu is being displayed, but there is a workaround.

Just pull up a terminal "applications > accessories > terminal" and type in the command "gnome-screenshot -d 5." Don't capitalize anything, it needs to be lowercase. Or you can hit alt-f2 and type in the same command, that works too.

Gnome-screenshot is the computer program you see when you hit the screenshot button in Ubuntu.

"-d 5" is an 'argument' or 'parameter,' in this case telling the program to wait five seconds so you have enough time to setup the screen how you want it to take the screenshot. (d stands for a delay.)
See:
[http://en.wikipedia.org/wiki/Command-line_argument](http://en.wikipedia.org/wiki/Command-line_argument)
[http://en.wikipedia.org/wiki/Parameter_%28computer_science%29]("http://en.wikipedia.org/wiki/Parameter_%28computer_science%29")

If you wanted it to wait 10 seconds you would type "gnome-screenshot -d 10" instead.

Save the screenshot anywhere you like, then use a flash drive to move it to another computer.
