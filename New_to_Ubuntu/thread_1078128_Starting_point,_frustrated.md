### Post by joey-elijah on 2009-02-23
I don't mean to be rude but you're stuck with a handful of packages if you're living the mantra of  'Its on the CD or it doesn't exist.'

 If you're expecting a gazzilion packages/software to be fitted on a cd that also includes the OS then you're going to be disappointed! That's what the installation DVD's are for - they come with many, many packages on them.

Firefox is not the default browser in KDE -  Konquerer is. Firefox comes pre-installed with Ubuntu, however.

Secondly - if the machines are going to be online then what is wrong with using add/remove to install any extra programmes? 

I would have chosen Ubuntu instead of Kubuntu because it's generally lighter on requirements than KDE. It also comes with openoffice, firefox, f-spot photo manager and easier add/remove options than Kubuntu.

 You don't need Gnash to play flash - there is a flash plugin which can be installed by installing the multimedia package.

The best thing I can suggest for you is to try using Super Ubuntu.  It's the full version of Ubuntu (with updates) and comes preloaded with a lot of non-free codecs (for .mp3 support for just one example) as well as Flash, Java et cetera. 

[http://hacktolive.org/wiki/Super_Ubuntu](http://hacktolive.org/wiki/Super_Ubuntu)   Download link at the bottom of the page.

And what's on the CD, frankly, isn't going to be that much, I don't care what distro you're looking at.  A CD is a CD is a CD.  The installer and packages on the LiveCD are meant to provide a BASE-install, not a full blown, fully tricked out distro with every available bell and whistle.
