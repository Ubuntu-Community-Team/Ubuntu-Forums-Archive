### Post by sisco311 on 2008-09-27
> **chris062689 said:**
> It didn't have that option when I booted from the Ubuntu 8.04.1 Alternative Install CD.

It's not that I want to install a CLI because of my system specs, I want to be able to choose which GUI I have, either KDE 4.1, XFCE, GNOME, etc.
