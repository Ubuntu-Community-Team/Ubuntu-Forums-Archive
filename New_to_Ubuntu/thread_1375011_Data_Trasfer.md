### Post by Black Sun on 2010-01-07
I have a laptop with ubuntu installed and a desktop pc with windows xp installed, both of these systems are connected to internet through a router. My problem is whenever I have to transfer some data from one system to another I have to use a pen drive which is a real pain as the usb ports of the desktop are at the back side and not easily accessible, now since there is actually a link between the 2 systems through the router I was thinking is there any possible way to transfer data from here to there or there to here? 

My laptop is connected through a usb cable and the desktop is connected through ethernet cable.

---

### Post by bswilson on 2010-01-08
If as you describe, both these machines are on the same Ethernet network, then you can use many different tools or methods of transferring files between the two.

An easy way might be for you to download and install [FileZilla]("http://filezilla-project.org/") on your Windows PC, then use that to connect to the Ubuntu system.  Be sure to use SFTP/SSH instead of regular FTP.  You should be able to easily connect and be dropped right into your home directory.
