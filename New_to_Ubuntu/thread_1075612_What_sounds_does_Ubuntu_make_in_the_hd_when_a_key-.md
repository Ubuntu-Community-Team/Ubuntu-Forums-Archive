Since there are no Linux viruses running wild, you won't find any antivirus software for detecting them either (the virus must exist before antivirus software can search for it). All antivirus programs made for Linux are made for searching for Windows viruses, to be used on mail servers etc.

Same applies to spyware/malware scanners. The way Linux works makes it rather impossible for any nasty application to install itself on your computer without you manually installing it and giving it required permissions and so on.

If you want to run a firewall, Linux itself includes a firewall, you only need a tool to configure it. Ubuntu includes ufw by default, but since you probably want a graphical tool then install Gufw. (not that running one would do anything useful when there are no running services listening for incoming connections.)

### Post by walmis on 2009-02-21
You can watch all block writes by using these commands:

```
sudo -i
echo 1 > /proc/sys/vm/block_dump
exit
tail -f /var/log/kern.log
```

Also, in linux i/o is done asynchronously, so if the keylogger were to write data, unless it explicitly issues a fsync() the data would go to internal write buffers and synced to disk later, and given that keyloggers output very little data, there shouldn't be any noticeable disk activity.

Those sounds you were hearing was probably firefox updating it's cache.
