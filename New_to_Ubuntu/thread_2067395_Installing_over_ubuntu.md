### Post by ajgreeny on 2012-10-06
Choose "**Something Else**" at the disk preparation stage of the installation and you will be able to see all the current partitions, and then will be able to choose the correct ones to reuse, ie overwrite with the new version of Kubuntu.

Make sure you backup all important files first.
