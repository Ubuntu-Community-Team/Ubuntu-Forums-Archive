### Post by tarps87 on 2009-01-12
Look in package manager for a package called virtual box, once this is installed click new and follow the wizard and it should be fairly easy. It will guide you through setting up a virtual hard drive and then you boot/ install media.
You will need to make a few changes to get usb support, there is a guide in the FAQ:
[http://www.virtualbox.org/wiki/User_FAQ](http://www.virtualbox.org/wiki/User_FAQ)

Hope this helps
