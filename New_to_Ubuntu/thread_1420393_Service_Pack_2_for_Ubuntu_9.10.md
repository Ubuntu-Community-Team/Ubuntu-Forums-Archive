### Post by Mark Phelps on 2010-03-03
You would do better NOT installing Wine -- and trying to get MS Windows programs to run -- as one of your initial forays into the Linux world.

Getting programs to work under Wine can be a LOT of work, and when you're done, you're basically STILL dependent upon MS Windows!

As others have suggested, you would be better off in the long run learning to use Linux alternative apps.  That way, you can wean yourself away from MS Windows and benefit from a huge collection of  free applications.
