1. if I make Ubuntu my sole operating system on my computer and use a VM to run W7, can I still use my Windows 7 product key, or do I need a new one in order to run on a VM.

### Post by Verbeck on 2010-12-30
1. [s]you'll need a new one[/s] - re-read the license (part 3:d)
2.in the installation process when it comes to partitioning, choose 'manually partition' and select the partition on with you want to install (i.e. old ubuntu partition)
