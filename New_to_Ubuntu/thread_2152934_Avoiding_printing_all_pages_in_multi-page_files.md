### Post by Creakey on 2013-06-09
Usually, if I have a file from a web-site or e-mail which has multiple pages, it is easy to pick out one chosen page for printing.
Sometimes, however, the file seems to have been rigged to over-ride my selection in the "print" box and insist on printing
all the pages. Can anyone please tell me how this is done and whether there is a way of overcoming this nuisance.
Apologies in advance if this is a very elementary matter !

---

### Post by tgalati4 on 2013-06-09
Typically I just put a "1,2" in the "pages" dialog box to only print the first 2 pages of a file.  Alternatively, you can "Print Preview" to determine which pages you really want to print and then from the Print Preview dialog hit print and put in the page numbers.
