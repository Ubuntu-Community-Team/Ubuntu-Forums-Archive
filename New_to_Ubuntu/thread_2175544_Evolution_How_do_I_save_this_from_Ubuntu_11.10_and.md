### Post by lukeiamyourfather on 2013-09-19
Applications with user specific preferences are stored in the home folder for that user. Like ~/.mozilla for Firefox preferences. Exporting preferences from the application is an option if it exists, otherwise just copy the contents of the home folder (including hidden files).
