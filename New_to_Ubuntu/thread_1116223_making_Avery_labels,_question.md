### Post by Zimmer on 2009-05-01
Well, I have had a look at the label process generally in OO and cannot find what you seem to want. However, I do create address labels every year for my Xmas card list. 

To do this I created a label template and a data table in OO database and I import the fields to my labels on a sheet by sheet basis. (ie I select 21 adresses from the data base and print that sheet, discard and select the next 21 etc.)
 If you are creating labels with exactly the same text in each then you use the synchronize option and just type your data into the first label and it will copy that to the rest on the sheet. Then, I guess , you would print as many sheets as you required. 
I am not sure under what special circumstances you would want to be manually creating individual labels in quantity without somehow using a data table and linking to fields in the label template... but... if you do, then, at present,  the only way I can see is the way you are already doing it.
