### Post by Celauran on 2009-05-28
Open gconf-editor, go to apps -> metacity -> general, find button_layout, and reverse the order.

### Post by xaviermerino on 2009-05-28

### Post by CatKiller on 2009-05-28
For that matter, if you're using Emerald as your Window Decorator (because you're using Compiz and shun Metacity) you can go to System -> Preferences -> Emerald Theme Manager -> Edit Themes -> Titlebar and change the layout options.
