### Post by xedi on 2012-04-22
Short answer: No

Long answer: Maybe

As far as I know, the developers never made a native Linux version, so there is no straight forward way to play them on Ubuntu.

However, you could try Wine or PlayOnLinux (which uses Wine but is specialized for games) to try to run the Windows versions.

According to the [Wine database]("http://appdb.winehq.org/objectManager.php?bIsQueue=false&bIsRejected=false&sClass=application&sTitle=Browse+Applications&iItemsPerPage=25&iPage=1&sOrderBy=appName&bAscending=true") people have mixed results depending which total war game they try and other factors.

I personally don't have any experience with the series, maybe someone else can comment on that, but generally I would advise you to just try it out with Wine/PlayOnLinux and see whether it works.
