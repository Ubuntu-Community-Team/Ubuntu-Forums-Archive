### Post by grahammechanical on 2011-10-02
It is my experience that Libreoffice Writer will let you export a Writer document to PDF format, thus creating a PDF document. But Libreoffice cannot edit PDF doucuments. We get the blank page as you have described.

Some how it seems that Libreoffice Writer has become associated with PDF files as the program to open them with.

Load the file manager - select a PDF document and right click and select Open With and choose Document Viewer to open the file and then Document Viewer will become associated with PDF files as the program to open a PDF file when you click on it and not Libreoffice Writer.

If we are not careful we could associate any program with folders. So that a click on the folder will run the program instead of opening that folder in File Manager. Some have had this experience. The solution is the same as outlined above. I think that this is what happened to you but with Libreoffice Writer being the program to open PDF files.
