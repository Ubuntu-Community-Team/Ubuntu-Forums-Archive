### Post by zjagannatha on 2009-03-31
> **dragonborn2780 said:**
> hello,i am a newby i would like to know if i can have windows and ubuntu os on same harddive.

It's certainly possible. You would need to partition your hard drive with something like gparted. Just boot up your Ubuntu LiveCD and go to *System -> Administration -> Partition Editor*.
Resize your Windows Vista partition to be a bit smaller. The size depends on how much space you want for each operating system. I wouldn't give Windows less than 30gb. Ubuntu should run well on 10gb, but if you are going to be using it as a primary like me, then you probably want to have a bit more. (I know I've surpassed that in Music alone.)
