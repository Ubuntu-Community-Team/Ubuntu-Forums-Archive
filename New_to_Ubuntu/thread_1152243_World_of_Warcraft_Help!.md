### Post by acimmarusti on 2009-05-08
Yes, there is a free, in the sense its source is open, alternative to Cross over. It's called Wine. In fact, Crossover is a polished version of wine. Wine isn't extraordinary (sometimes it takes a lot of trickery to get an app working with it), but it helps a lot. Warcraft 3 and the frozen throne expansion work pretty well on it. The latests versions also support WoW, but it might require some tweaking.

Wine website: [http://www.winehq.org/](http://www.winehq.org/)
Wine application database (WoW on gold list): [http://appdb.winehq.org/](http://appdb.winehq.org/)

Installation Instructions for latest wine in Ubuntu: [http://www.winehq.org/download/deb](http://www.winehq.org/download/deb)

Info about WoW on Wine in Ubuntu 9.04: [http://appdb.winehq.org/objectManager.php?sClass=version&iId=14154](http://appdb.winehq.org/objectManager.php?sClass=version&iId=14154)
(follow these tweaks to get it working)

Good luck
