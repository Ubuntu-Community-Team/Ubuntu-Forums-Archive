### Post by billgoldberg on 2008-05-05
If I understand you correctly you want to copy two movie files from two cd's and put them on 1 dvd.

First you will need to rip the cd using "k9copy" *(or another program)*.

Then, if it is one movie cut in two, you could merge the two movie into one movie using "avidemux".

Then you could either burn the dvd as data or as video *(to watch on a dvd player)* using the standard installed "brasero" or "k3b".

If you really want to use nero *(I don't see a reason why you would)* there is a linux version of nero available [here]("http://www.nero.com/eng/linux3.html").
