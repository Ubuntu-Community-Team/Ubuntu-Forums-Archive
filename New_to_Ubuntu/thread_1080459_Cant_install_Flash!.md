### Post by jaminux on 2009-02-25
Open the Synaptic Package Manager, search for ubuntu-restricted-extras and install it. That will install a variety of packages including flash.

If you are using 64 bit Ubuntu there are known problems with flash though.

### Post by marshall.robert on 2009-02-25
Go to Add/Remove... in Applications and then type Ubuntu Restricted into the search box. Tick the item labeled "Ubuntu Restricted Extras" and then Apply and you will have all the needed internet and media apps and codecs. Including flash.

@jaminux - I like to keep directions away from the terminal and synaptic unless its really needed since its more user friendly. and thats what we want :)
