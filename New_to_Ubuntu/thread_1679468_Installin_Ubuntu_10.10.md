The Alternate CD might be useful.  It has a text based installer which both takes up less memory and is less likely to have graphics problems. The alternate CD boots into the installer however, it doesn't have a "try" option to run it first.

Your graphics problems are most likely to be the graphics card, or possibly the monitor is not reporting it's type so Linux is just selecting the lowest common denominator, probably 800x600 or even less.  It's quite likely that the problems can be overcome but we'll need to know what graphics card/chipset you are using, and what the monitor is. It would also be helpful to know a bit more about the computer, such as what memory and hard drives are in it.

For the first time user I think I'd recommend 10.04 LTS because it's a relatively mature release and is very stable.  LTS means long term support.
