### Post by Jaybyrrd on 2011-10-31
Depends on what you mean by getting viruses. In order to dual boot you first partition your hard drive or use two separate hard drives. Then you install the OS on whichever hard drive does NOT have Linux. From there Windows can still get viruses, however anything in your Linux install will not be affected. 

So yes, but only in Windows, (and I suppose you could get hacked in Linux if you give someone access to root, highly unlikely though).

### Post by mcduck on 2011-10-31
> **LukaHr said:**
> I have question. If you install win XP on linux virtual machine, is there posibility of geting viruses?

The windows install running in the virtual machine behaves exactly like one running on a real machine would, so yes, it is vulnerable to viruses and malware. However the viruses will not cause any danger to the Ubuntu system serving as host for the virtual machine.
