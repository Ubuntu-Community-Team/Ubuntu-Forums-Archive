### Post by jjex22 on 2011-11-12
are you able to boot into vista? if so, I would recommend backing up to cd using vista's backup tools - if you have a home version you can only back up files to disks or hdds, but if you have a higher version such as ultimate you can create an entire system image onto the dvd's. This is done from the back-up centre. 

The reason I'd recommend using vista to do this if possible, is that that way when you restore these files from an install cd, windows will intuitively know where to put them back.
