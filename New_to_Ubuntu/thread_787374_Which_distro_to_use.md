### Post by lemming465 on 2008-05-08
The easiest way to see if you have sound support in a particular Ubuntu version is to boot the live desktop CD.  If it makes noise when you log in, it's got sound.

If you don't have sound in 8.04, you might not get it until 8.10 in October, unless you are willing to compile you own kernel this summer.  Creative is not as helpful as some companies with linux drivers for their hardware, alas.

You can install Kubuntu (the KDE desktop variant) directly.  Or if you have Ubuntu (Gnome desktop) installed, you can add the package *kubuntu-desktop* (or kubuntu-kde4-desktop) and have a choice of both.  Which is what I usually do.

Fedora and SuSE are both releasing new versions soon; if you aren't getting what you want from Ubuntu you might try one of them.
