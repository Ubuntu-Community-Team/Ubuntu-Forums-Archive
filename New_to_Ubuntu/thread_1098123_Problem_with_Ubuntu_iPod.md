dont "jailbreak" whatever that is. plug in ipod, look for it in nautilus or your file manager, browse the folder then open it, now open the music player and using amarok 2.0 as an example go to settings>configure amarok>configure collection, find the ipod folder(check the box) then click on rescan collection then OK, go back to the main amarok screen it should be listed there,
