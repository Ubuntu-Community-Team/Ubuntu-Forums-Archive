First download and burn the Ubuntu cd.

When you boot from the Ubuntu cd and choose install you will eventually come up to a point it asks where to install it. Choose "Use the whole disk" (or similar) and it will wipe out everything on it. This is assuming you want to get rid of everything on the hard drive and have nothing you want to save on it.

### Post by KaiO on 2010-10-27
Just download your preferred Ubuntu version.
Burn ISO image to cd.
Reboot with you cd in the player and you should get the choice to install Ubuntu.
(If this doesn't work you might have to change *boot order* in BIOS)

During the installation you can choose to overwrite other operating systems (i.e. Suse).

### Post by coffee412 on 2010-10-27
> **karin66 said:**
> thanks for the answer. how can i boot from cd?

During the boot up process and before the operating system starts, Along the bottom of your screen you should see some definitions for boot options.

In some of the HP line of computers:

F2 - Run setup
F12 - Boot from alternate device

These definitions change from manufactuer or model. You will only see these options briefly for a few seconds before the operating system starts. Press the appropriate key and choose your cdrom drive to boot from. Then follow the prompts from the Ubuntu cd and your on your way.
