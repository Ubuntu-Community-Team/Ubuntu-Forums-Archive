### Post by oldfred on 2014-01-20
You cannot boot Windows 7 from an external drive. 
It used to be you could not boot any Windows from an external drive, but I guess there now is a version of Windows 8 that will.

### Post by Mark Phelps on 2014-01-20
FYI ... the only version of Win8 that can boot from an external drive"legally" is Enterprise.  There are hacks (of course) that claim to allow other Win8 versions to boot from external drive, but folks who have actually tried those approaches claim they do not work.
