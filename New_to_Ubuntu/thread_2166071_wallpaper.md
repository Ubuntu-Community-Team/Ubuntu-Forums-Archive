### Post by linuxonbute on 2013-08-07
If you try to change your desktop background you would right click on the background and you get a window pop up
with the label Appearance. On the righthand side near the top you will see a dropdown box which says "Wallpapers".
Clicking on that box shows there are 3 options.
The one you have easy access to is "Pictures Folder"
If you launch nautilus it will launch into your home folder by default
You will see several folders one is Documents and one is Pictures.
You have full access to that folder.
If you have a camera and plug it in Shotwell may automatically start up and you can import all ( or as many photos as you select )
into a subfolder of Pictures. It will create a folder for Year/Month/Day and copy the photos there.
Now any photo you want to be able to use as a background needs to be copied or moved from these folders into Pictures itself.
Then when you select Pictures Folder from the dropdown box all of the pictures you have copied will be there and you can click on one to choose it.
