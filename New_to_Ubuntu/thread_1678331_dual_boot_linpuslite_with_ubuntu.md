### Post by mikewhatever on 2011-01-30
From the last remark, it looks like you've installed Ubuntu to an sd card, in which case, you'll have to boot from it to load up Ubuntu. Presumably, you must be booting from the device Linpus is installed on, and hence get only Linpus.
Edit: I should probably have mentioned that boot sequence is adjusted in the bios, which you can access by hitting del, f12, or whatever Acer assigned it to.
