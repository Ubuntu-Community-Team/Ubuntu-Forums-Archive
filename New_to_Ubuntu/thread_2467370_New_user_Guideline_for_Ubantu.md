There are two web sites that you should visit.

[https://ubuntu.com/](https://ubuntu.com/)

[https://canonical.com/](https://canonical.com/)

Canonical is the commercial organisation that publishes the Ubuntu distribution of Linux. That web site will given you information of the services offered by Canonical. Some of the services are paid for services.

The Ubuntu web site is hosted by Canonical and it provides information about the many things that can be done with Ubuntu and software built on Ubuntu. There is also information on paid for services.

Ubuntu is provided as free to download, install and use. This applies to the user who only has one machine and also to the user who has many machines. You should read Canonical's Intellectual Property Rights Policy.

[https://ubuntu.com/legal/intellectual-property-policy](https://ubuntu.com/legal/intellectual-property-policy)
