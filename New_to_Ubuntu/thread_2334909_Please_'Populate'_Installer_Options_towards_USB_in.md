### Post by ian-weisser on 2016-08-23
The Ubuntu installer is designed to cover the most common installation scenarios a new, unskilled user is likely to want. Adding additional options quickly becomes confusing.
Ubuntu is popular because it is easy and works for most users, not because it works in every possible scenario.

Most users who decide to install full desktop Ubuntu onto flash drives  are neither new nor unskilled...so they know how to look it up. It's not hard.

A "newbie" user trying to install to USB is rare.
Nevertheless, you are welcome to offer an improvement patch to the installer developers. They are nice people.

### Post by yancek on 2016-08-23
You can install directly to any usb or flash drive by simply using the manual "Something Else" option and selecting the drive and partition on which you want to install.

Trying to have options for every possible user scenario could get pretty complicated in a hurry.  If you have one internal drive and one usb drive, pretty simple.  How about if you have one internal drive and two external drives.  Which external?  How is the  installer supposed to determine that.  What if there are 4 or 5 or more external or flash drives attached.  In situations with multiple drives it will require some user input and there are already countless tutorials as well as documentation on how to do this.  Understanding the drive/partition naming conventions will take you a long way to success.
