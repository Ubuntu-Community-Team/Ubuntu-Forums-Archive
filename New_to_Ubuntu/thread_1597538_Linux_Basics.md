### Post by Old_Man_Unix on 2010-10-15
If you are just referring to the terminal itself, there is this:

[http://library.gnome.org/users/gnome-terminal/stable/index.html.en](http://library.gnome.org/users/gnome-terminal/stable/index.html.en)


All the commands, including the shell itself,  have *man* pages.  But if you really want to learn how to use the terminal, you should learn the BASH shell, since that is the default shell.

Here is a nice introductory website (it is written for Fedora but the shell is basically the same):

[http://linuxcommand.org/learning_the_shell.php](http://linuxcommand.org/learning_the_shell.php)

And if you want a more extensive review of the whole CLI, there is this excellent book:[SIZE=1]

[B]Linux Command Line and Shell Scripting Bible by Richard Blum

[/B][/SIZE]:) Good luck!
