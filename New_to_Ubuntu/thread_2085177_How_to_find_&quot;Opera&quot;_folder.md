### Post by arpanaut on 2012-11-17
While in your file browser Hit ctrl + h this will show you the hidden files, i.e. the "dot files"
./Opera,  Ubuntu hides the configuration files to remove clutter from the Home directory.
You can also go to view>show hidden files in the menu.
