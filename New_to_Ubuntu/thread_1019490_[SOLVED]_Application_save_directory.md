### Post by Sarmacid on 2008-12-23
It's probably hidden in your home folder. To view hidden folders, go to your home folder and hit ctrl + h.

If that doesn't work, you can always try with find```
find / -name "*snes*"
```
That will tell you where are all the files whose name contain snes.
