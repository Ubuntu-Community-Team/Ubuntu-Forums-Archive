### Post by Mighty_Joe on 2009-08-03
There is a topic [here]("http://ubuntuforums.org/showthread.php?t=1193680") that covers the various ways to install Ubuntu to a flash drive.  To run the Live CD image from a flash drive, you'll need a 1Gb drive.  To install Ubuntu directly to a drive, you'll need a 4Gb drive.  My preference is to [do a full install]("http://ubuntuforums.org/showpost.php?p=7497700&postcount=4")

### Post by wpshooter on 2009-08-03
You need at least a 4gb pen.  But when I get to the point of trying it again, I am going to use an 8gb.

I have tried it on a 2gb and it will do the initial O/S install but as soon as you try to apply the updates, you are out of space.

And the way that I recommend doing the install is from the ALTERNATE CD version of Ubuntu, with any and all hard drives in the system disconnected during the installation.
