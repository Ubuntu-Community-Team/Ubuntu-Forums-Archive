### Post by mc4man on 2009-11-01
Pretty simple

Picking one as an example 
From here [http://www.apple.com/trailers/wb/invictus/](http://www.apple.com/trailers/wb/invictus/)

I get this as the link

```
http://movies.apple.com/movies/wb/invictus/invictus-tlr1_720p.mov

```
So for command use this 
```
[COLOR="Blue"]wget -U QuickTime/7.6.2[/COLOR] http://movies.apple.com/movies/wb/invictus/invictus-tlr1_[COLOR="Red"]h[/COLOR]720p.mov
```

Blue is the wget command, added the red h
