### Post by Frogs Hair on 2011-04-18
No , that would require the installation of  a  program that Appears like the update manager and that is very unlikely to come from to official repositories.
 
Running across a site that offered a download for the ubuntu update manager would be a huge red flag becuse the update manager is installed by default and updates via the repository.
 
It is possible to down load the applications from an ISO from the index page , but that is a secure page and page origin can be checked with the internet browser.
