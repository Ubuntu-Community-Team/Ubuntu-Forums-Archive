**[COLOR="Red"]SOLVED![/COLOR]**I am running Ubuntu 10.04. Evince or OpenOffice use to open pdf files but after updating Gimp is now the default viewer. How can I change the default opener of all pdf files other than right click and select open with? :confused:

---

### Post by VCoolio on 2012-04-25
not rightclick > open with (this file this time only), but rightclick > properties > open with (this file type always)
