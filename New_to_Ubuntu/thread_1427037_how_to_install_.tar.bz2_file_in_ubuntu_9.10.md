### Post by lidex on 2010-03-11
Desktop is fine. When you use the terminal to install you'll just need to "cd" (change directory) into Desktop so it becomes your working directory (default is "~"; or your user directory). A simple command to accomplish this is:
```
cd ~/Desktop
```

Edit: Sorry I didn't see the first part. When you download file right-click and select "extract here". In the extracted files look for a "readme" or similar for info.
