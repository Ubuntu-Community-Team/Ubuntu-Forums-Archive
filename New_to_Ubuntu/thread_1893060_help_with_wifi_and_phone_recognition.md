Any particular reason why you're using 10.04? It's rather old now.

Ubuntu 11.10 is the latest stable version. It has newer hardware support, so it's more likely to be able to use your wireless hardware.

Ubuntu 10.10 and later can use a tethered Android phone as an internet connection, out-of-the-box. You just have to go to the Network Manager icon on the top panel and it will appear as an option, if the phone is plugged in and tethering is turned on in the phone. Ubuntu 11.10 can definitely do it still.

With the tethered internet connection, you will be able to open the Additional Drivers program if necessary to download firmware for your wireless card with one click. It's not always necessary to download, though, so if your wireless card "just works" without any additional downloads don't be surprised.
