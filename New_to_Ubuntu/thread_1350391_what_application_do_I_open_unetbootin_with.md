### Post by nokia9300 on 2009-12-09
1) For a linux distribution you would want to download "unetbootin-linux" file.

2) chmod u+x unetbootin-linux  (allowing execute permissions to the user)

3) ./unetbootin-linux  (execute the command and you will see the GUI)

Project website has lot of information if you get stuck or have questions

[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)
