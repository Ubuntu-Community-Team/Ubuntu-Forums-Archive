Purchased version of Win7 generally come with two DVDs -- one for 32-bit, and one for 64-bit.  They both use the same activation key -- so you can use either one, but not both at the same time.

As to advice, best course of action is usually to ask in the right forums -- in this case, sevenforums.com  (Win7 forum).
