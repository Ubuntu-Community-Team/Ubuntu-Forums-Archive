### Post by thegod_slayer on 2011-03-12
if you want to change name of the home folder then its easy.

go to the terminal and type sudo nautilus.
this will open the nautilus file browser and then you can go to the filesystem and rename the home folder to anything.

but if you want to rename the folder having your username inside home then you need to change the username.
