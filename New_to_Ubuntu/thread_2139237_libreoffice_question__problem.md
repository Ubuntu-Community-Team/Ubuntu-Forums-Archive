### Post by r.stiltskin on 2013-04-26
Equations written in MathType *are* graphics objects, as are equations written with MS Word's equation editor and equations written with Libre Office's formula editor.

I can think of no reason why LibreOffice running in 12.04 would be any less capable than LibreOffice running in 10.10.  But I see two possibilities: maybe the equations you previously were able to edit were written using the MS Word equation editor, which LibreOffice can edit.  Or maybe they were written with an old version of MathType (v. 3.1) which I believe LibreOffice can edit.  (But that would have to have been a *very* old version.)

According to MathType's website, if you are editing a document in LibreOffice *running in Windows*, and you click on a mathtype object in that document and it will be opened in MathType.  Of course, you must also have MathType installed.  Hurrah for proprietary software.
