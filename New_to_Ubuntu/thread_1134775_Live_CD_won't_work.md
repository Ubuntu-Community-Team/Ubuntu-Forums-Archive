### Post by lkraemer on 2009-04-24
You most likely have a bad burn or a bad download.

Did you verify the MD5SUM of the downloaded file to make sure it
matched with what was posted at the download site?
There is software for Windows that will VERIFY the MD5SUM.
GOOGLE for MD5SUM Windows Software

Did you burn the ISO with Nero or Easy CD Creator as an Image?
If you used K3B burn as Disk at Once (DAO).

Did you burn at 4x or 8x speed to prevent errors?
Did you use good Media?
Did you try booting your CD on several computers?

Another option is to try the Alternate Install CD, if the
regular LiveCD won't boot.
