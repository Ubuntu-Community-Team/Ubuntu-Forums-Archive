### Post by ahood on 2008-12-20
I am not aware that gmail is an open source application, but it is free to use and popular.

I do have a gmail account and use Evolution. I have set up Evolution to display gmail messages using IMAP. I like IMAP instead of POP3 because with IMAP, messages are not downloaded onto the local drive and those same messages deleted off the gmail server. 

I don't know the workings of IMAP, but I like to think of IMAP as using Evolution as a shell application that provides the interface for filtering and reading gmail messages. Because the messages stay on the gmail server when using IMAP, I see the same gmail messages and folders when I am away from my home machine and use the gmail web application. 

I also like to use Evolution because I have other web-based email accounts, and I can manage all of the email messages with this application.

I have used Thunderbird and it has a nice interface. I am pretty sure it will support IMAP as well as Evolution. I use Evolution simply because it has an integrated calendar functionality and comes as the default mail application with Ubuntu.
