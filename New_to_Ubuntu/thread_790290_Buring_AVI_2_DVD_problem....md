### Post by diablo75 on 2008-05-11
You're using a DVD player that has AVI playback capability.  And you're probably doing a standard data disc project with Brasero when you create a DVD.  Perhaps you could try burning the disc at a slightly slower speed than "max"?  Also, uncheck the "make more compatible with Windows" check box and see if that helps as well.  And the disc should probably be closed, and not left open for further burning sessions (so that check box should be checked off as well).

### Post by diablo75 on 2008-05-11
In brasero, when you click the Burn button in the lower right corner, a new window will appear that looks like this (mine is greyed out because I'm not actually at my desk and can't put a blank in, but you can still see it):
