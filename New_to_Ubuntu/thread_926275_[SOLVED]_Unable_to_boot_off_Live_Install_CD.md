### Post by pieoncar on 2008-09-21
I had a similar problem when I was installing - I wasted a couple days of work and about 6 CDs before I figured out that I was burning and reading from a "sketchy" CD drive.  I ended up burning the minimal CD ([https://help.ubuntu.com/community/Installation/MinimalCD]("https://help.ubuntu.com/community/Installation/MinimalCD")) because it is less than 10 MB.  Luckily I was able to get a clean burn and a clean read from it.

The minimal CD will download almost all its packages from the Ubuntu servers, so that might not be a good option if you have a poor network connection, or it's highly throttled, etc.

It sounds like you're having a problem with the CD more than a hardware issue to me, at least.
