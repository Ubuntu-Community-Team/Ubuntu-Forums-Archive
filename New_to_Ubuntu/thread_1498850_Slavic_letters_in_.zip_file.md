### Post by da burger on 2010-06-01
In a terminal go to the directory where you have the .zip archive and run ```
unzip filename.zip
``` (obviously substitute filename for the actual name)

Alternatively try right clicking on the archive and clicking extract here. I'm not sure if either of those will get around your problem but they are both worth a shot.
