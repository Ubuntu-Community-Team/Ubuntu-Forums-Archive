### Post by chamira on 2010-12-14
I have no clue about Macs, but what you need to do is associate different file types with the applications to open them. Obviously this is all set up in Windows (Explorer > Folder Options > File types) you need to find a similar process for your Mac.

Also, for some file types such as PDFs you may need the appropriate plugin installed on your browser.
