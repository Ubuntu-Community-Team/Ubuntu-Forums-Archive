### Post by OutOfReach on 2008-06-22
Your talking about CD Burning programs for Ubuntu right?
All you have to do is:
Insert a blank CD, a Blank CD-R Disc icon should show up on your desktop, double-click, it should lead you to a blank window with a blue bar saying: "CD/DVD Creator Folder", Now the Linux Mint download will be a .iso file, drag and drop it onto the Blank CD-R Disc window, click Write to Disc, there'll be some prompt about .iso images, Select the first button which I think is "Create from Image", Then there will be a prompt "Write to Disc" you can select your options that suit you there, or just press the "Write" button, Thats all it takes. :) Thats the easiest way to burn CDs that I can think of.

If your talking about Windows, or another Linux Distro, I cant help you there, sorry :(

Hope that helped.
