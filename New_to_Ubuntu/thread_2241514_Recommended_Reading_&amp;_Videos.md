### Post by yancek on 2014-08-26
Everything in Linux is not longer command line but that is a good way to learn.  Lots of things are easier and less cumbersome using a terminal than a GUI.

The tutorial below is good for installing Ubntu 14.04 but the first part has a lot of basic information on Linux and partitioning and also includes a number of links to other really good tutorials.

[http://www.dedoimedo.com/computers/ubuntu-14-04-install-guide.html](http://www.dedoimedo.com/computers/ubuntu-14-04-install-guide.html)

The version of Ubuntu coming out later is not a major change.  I expect some changes but I would not think the manual would be obsolete.
