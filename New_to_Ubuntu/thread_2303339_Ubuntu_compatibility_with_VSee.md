### Post by Bucky Ball on 2015-11-18
Welcome. VSee doesn't appear to be a Linux application so the only way you would be able to run it is with Wine, if that's possible at all. The rating's not good on [WineHQ]("https://appdb.winehq.org/objectManager.php?sClass=application&iId=14262") so I wouldn't go there personally. (This said, if you need to install a lot of Win only apps, best to install Win or run it as a virtual machine).

Skype works fine for a lot of people, me included, and is easy to install from the Software Centre. Sure others will chime in with other apps or you could beat them to it with some research on '[video conference ubuntu]("https://duckduckgo.com/?q=video+conference+ubuntu")' and see what you find, try out a few things.

If the user on the other end is using Windows, though, and you both need to be using the same app, there is another hurdle in some cases, so you may need to find something that is cross-platform. Like Skype.

The other option is, as mentioned earlier, to run Windows as a virtual machine and install VSee in there. You need a copy of Windows for this, of course.
