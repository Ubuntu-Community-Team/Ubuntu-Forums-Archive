can you post your etc/fstab file
```
cat /etc/fstab
```
and 
```
sudo fdisk -l
```

it may be that its mounting fine but you haven't set up the permissions properly on the drive. Anything outside your /home folder is owned by root so you will probably need to change the ownership of the drive.

### Post by halitech on 2009-09-18
I think you need to change this line
```
/dev/sda /media/sda ext3 defaults 0 2
```
to
```
/dev/sda1 /media/sda ext3 defaults 0 2
```
you need to mount the partition, not the drive itself.
