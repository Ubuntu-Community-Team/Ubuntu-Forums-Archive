### Post by I2k4 on 2011-07-10
It's running quite well off my Live USB installation of Lubuntu 10.10 - you don't mention what version you're running.  I'm not comfortable with the terminal, and have found that FireFox seems to permit automatically installing "deb" through its dialog box, while the Chrome browsers only download it and leave you to figure out what to do.  

Since you already have Earth installed, you might check the Synaptic Package Manager and try reinstall using that utility. Or else, completely delete what you've got using the Synaptic Package Manager and use FireFox to install the latest "deb" from the Google Earth site.  

Obviously there are much more knowledgeable users, but what's easy for them is often too much for me.
