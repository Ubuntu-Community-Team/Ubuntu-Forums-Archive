### Post by schauerlich on 2008-09-07
Well, a lot of those commands aren't really the "base" commands that all Linux systems come with, so you will be less likely to find a comprehensive list of those on the internet. The way many of us learn is simply by doing - when we come across the need for a command, we use google and try to find a solution. That's how I've discovered most of the commands I use regularly now. That said, there are reference books made especially for Ubuntu that may contain those sorts of commands, although I've never thumbed through them.
