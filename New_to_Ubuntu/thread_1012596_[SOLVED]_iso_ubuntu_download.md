### Post by Sef on 2008-12-16
> I downloaded the Ubuntu OS and then extracted using Winrar and made an .iso image file to burn to a cd.

1) It is an iso upon download.  You do not need to extract it. 

2) Check the md5sum upon download.

3) Burn it at 4x or less.
