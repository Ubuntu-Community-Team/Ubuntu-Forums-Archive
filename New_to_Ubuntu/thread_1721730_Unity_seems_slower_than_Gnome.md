### Post by rosencrantz on 2011-04-05
AFAIK, the standard Unity shell requires 3d acceleration and has more desktop effects than plain Gnome, which would explain slower performance. On my first VirtualBox Natty Beta tryout, I didn't get it to run at all. There seems to be an Unity 2d on the way, which might be easier on your netbook.
I think the main purpose of netboot remix distros is to provide a desktop interface suitable for small screens/touch devices (cf. your problems with the regular gnome layout) and a software selection customised for 'fun machines'. 3D eye candy has become so much of a standard that it's often a default option regardless of performance issues. 
As wolfen69 suggested, Lubuntu should have a netbook interface included in the desktop edition. If you want something less purist, you can try Kubuntu's netbook remix. The Plasma desktop is 3d by default, but you can switch all effects on and off via hotkey.
