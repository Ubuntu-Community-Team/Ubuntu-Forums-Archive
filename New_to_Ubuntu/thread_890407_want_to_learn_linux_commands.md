### Post by AndyCee on 2008-08-15
I've attached my uni notes/reference for common unix commands, as well as basic script concepts. They're hardly in-depth, but fairly easy to read.

If you ever have a task you wish to achieve, but don't know the  command, I use
```
$ man -k "something"
```
with "something" being a keyword (sort, user, boot, circle etc). Man pages are your friend, but unfortunately they aren't always friendly.

Start with transferring a single command (eg. $ ls -la) to a script, and go from there (Best *not* to start with
$ shutdown now).
[This]("http://www.faqs.org/docs/Linux-HOWTO/Bash-Prog-Intro-HOWTO.html") has helped me remember the basics too.

Hope this is some help.
