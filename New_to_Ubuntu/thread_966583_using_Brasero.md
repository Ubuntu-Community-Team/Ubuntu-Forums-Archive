when you select files to burn in Brasero it goes 1 file at a time in a directory. How do you select  all the files in the directory instead of 1 at a time.???

---

### Post by nhasian on 2008-11-01
you can press Control-A for all files.  

hold down the Control button+mouse click to select multiple files individually, or you can select the first file in a list, then hold shift+mouse click and select the last file and it will choose everything in between.
