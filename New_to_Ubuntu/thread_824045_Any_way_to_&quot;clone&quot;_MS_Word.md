I'd like to kick Windows altogether, but I'm just not sure I have the time or energy to retrain myself on this low-level stuff. Is there some way to "clone" Word, making OpenOffice (or Abiword, or whatever) mostly identical to Word?

### Post by s3r4phim on 2008-06-09
I've had problems with wine in a lot of situations, primarily those that involve an installer as opposed to just the executable.

You could try AbiWord, which i've heard is more similar to MS Word, but i have yet to try it.

If you are going to try to actually run MS Word on linux, i would suggest getting the apple version, then running that with MacOnLinux (MOL), which is more stable.

If you are willing to put money into it, though, the best option is probably to use CrossOver Linux, which is very reliable and supported, and would give you far more options than just word.

Before buying anything, though, try the free options, of course.

Good Luck!

### Post by Sef on 2008-06-09
> BTW...there is also a commercial version of Wine (forget what its called). It may run Word if Wine has trouble.

It's called [Crossover]("http://www.codeweavers.com/products/cxlinux/").

> You could try AbiWord, which i've heard is more similar to MS Word, but i have yet to try it.

Abiword would not work for the OP.  I don't know if OpenOffice would either.  The compatiblity with OpenOffice is very high, but at times, there can be problems. Most of the problems would be small formatting, fonts, and macros not working, but in legal situation, they could be extremely important.  I would recommend that the OP use either WINE or Crossover because 100% compatibility is needed, so MS Office with Crossover would be my choice.
