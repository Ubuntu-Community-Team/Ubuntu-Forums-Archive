### Post by 3rdalbum on 2013-08-13
Ubuntu has a program called Main Menu. You can use that to make an icon to a program or website, and then drag that icon from the Dash to the sidebar and it will stay there.

It's easy, but just a little different to 10.04. It is not necessary to install a whole new desktop.
