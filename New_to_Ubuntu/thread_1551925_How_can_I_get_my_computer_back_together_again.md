### Post by PumaMania on 2010-08-16
My problem was that I accidentally told my BIOS to try to detect a Floppy Drive, which my computer does not have. So it took a while and it would eventually say that it wasn't there. So it took longer trying to find that drive. I disabled that option in the BIOS so it no longer looks and the speed has gone back to normal. Thank you!
