### Post by Old_Grey_Wolf on 2009-07-07
I don't think there are very many genealogist on this forum. :)

I had about 400 years of family genealogy or roughly 17 generations of data in an application that only ran on Microsoft Windows. I exported the GEDCOM and imported it into GRAMPS ([http://www.gramps-project.org/wiki/index.php?title=Main_Page](http://www.gramps-project.org/wiki/index.php?title=Main_Page)) on my Linux box. GRAMPS is also available in the repositories. 

It has worked well for me; however, the GEDCOM format is flexible enough that not all programs will import the data the same way. That is also true with Microsoft compatible versions of Genealogy programs.

GRAMPS does have support for the newer XML format that is replacing GEDCOM.
