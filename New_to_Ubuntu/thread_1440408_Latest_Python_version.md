---

### Post by Tahakki on 2010-03-27
Python 3.1 is in the package 'python3.1'; the package 'python' installs 2.6.4, the most commonly used version (right now).

Just search Synaptic for it. :)

---

### Post by kgarbutt on 2010-03-27
Like  Tahakki said use Synaptic. Find it here: System:Administration:Synaptic Package Manager

---

### Post by oldfred on 2010-03-27
Whatever you do, do not uninstall 2.6. Most of Ubuntu relies on 2.6 and will not work with a newer version. Not all supporting programs in python have been converted yet to v3.
