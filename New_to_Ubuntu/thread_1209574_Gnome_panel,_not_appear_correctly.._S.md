### Post by LinuxFox on 2009-07-10
I just tried this out, and it probably the way you set it up.

First thing I did was right click properties, and unchecked expand.  Then I changed it to left or right and its in the middle.  Changing it back to top, got me the same result you have.

To fix it, check "expand" and it will return to the top.  If you want it on the left side, just set it to the left.  If you want it smaller, then uncheck expand after moving it to the left.

I hope this helps.  I'm just answering it after trying it myself, I never had this problem before.
