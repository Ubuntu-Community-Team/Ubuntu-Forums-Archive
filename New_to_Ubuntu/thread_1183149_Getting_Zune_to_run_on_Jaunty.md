### Post by glennric on 2009-06-09
I believe you can add music to the Zune using either Rhythmbox or Amarok.  I know rhythmbox has a click and drag interface that works with the Sansa View.

If you are interested in editing playlists you can use a little program I wrote called playedit.  Check out the website at [MTP Playlist Editor]("http://playedit.sourceforge.net/index.html")
Eventually (when I have time) I want to add the capability to add and remove music as well.
