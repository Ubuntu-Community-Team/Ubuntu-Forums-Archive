assuming you have Gnome not KDE:

create a launcher that has this command
```

gnome-screensaver-command --lock
```

how to create a launcher: [https://help.ubuntu.com/community/HowToAddaLauncher](https://help.ubuntu.com/community/HowToAddaLauncher)
