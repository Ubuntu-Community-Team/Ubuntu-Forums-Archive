### Post by cfr42 on 2011-04-17
The latest stable Ubuntu release (10.04) for PPC is at [http://cdimage.ubuntu.com/ports/releases/10.04/release/](http://cdimage.ubuntu.com/ports/releases/10.04/release/) but you might want to take a look in the Apple Users forum, too. It seems to mostly boot one G4 I tried but didn't work at all for the other. But I've seen comments from people having problems with 10.10 on G5 so if it were me, I think I'd try 10.04 to start with.

If you download the desktop image, you can burn a LiveCD and boot from it to see if it works before deciding whether to install. The file for that is ubuntu-10.04-desktop-powerpc.iso.
