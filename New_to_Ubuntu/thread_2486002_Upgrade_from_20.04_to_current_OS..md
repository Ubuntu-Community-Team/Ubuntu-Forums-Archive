### Post by deadflowr on 2023-04-17
Check in Software and Updates > Updates.
Look at the section marked as Notify me of new release.
Toggle it to either Long Term release or Any new version.
then run Software Updater.
If you have any updates you need to apply those first before you will get an upgrade option.
(Sometimes, when you have updates that need to be applied, you also need to reboot before the upgrade will show)
