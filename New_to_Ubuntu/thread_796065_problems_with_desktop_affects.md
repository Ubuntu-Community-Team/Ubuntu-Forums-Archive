### Post by nicedude on 2008-05-16
Run this command in a terminal window

sudo apt-get install compizconfig-settings-manager

then after its done go to top menu bar System -> Preferences -> Advanced Desktop Effects Settings

This is the program to manage your cool effects like 3D cube and I beleive you are asking about the EXPO effect where it zooms out and you can view all 4 desktops at once. Just remember for each effect you need to set your activation keys for how you want to activate the effect.

PM me if you can't get it working :-)
