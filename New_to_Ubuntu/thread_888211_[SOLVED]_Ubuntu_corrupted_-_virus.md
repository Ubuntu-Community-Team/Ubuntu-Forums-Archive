### Post by appi2012 on 2008-08-13
Ubuntu does not use windows drivers. also, I do not think you actually installed ubuntu. When you started ubuntu, you just put it into the cd drive and rebooted, right? Did you ever get to an ubuntu desktop? 

Ubuntu comes on a live cd. That means when you boot from the cd, and choose the first option from the menu, you are able to view a normal ubuntu desktop. This desktop is running from your system's RAM, so it does nothing to your hard disk. However, if you double click the Install icon on the desktop, it will install ubuntu on your computer.

It is not working for you, because there is a problem with the disk that stops it from booting up into the Live Desktop. You need to download ubuntu, and burn it to a cd (SLOWLY - so that it doesn't get messed up)

This website will really help you get started:
[http://psychocats.net/ubuntu/index.php](http://psychocats.net/ubuntu/index.php)

If you have more than 512MB of RAM, there is no need to use the alternate disk.
