### Post by coffeecat on 2010-10-16
> **Amsanc said:**
> Apparently the company (Genius) that manufactured the keyboard didn’t release the driver for Ubuntu, but anyway, I reinstalled it on Windows and it doesn’t work either.

You don't need a driver for Ubuntu. If it's been mapped for you it's a simple task. If it hasn't - well, that's beyond me. As Joshwaa points out, if it doesn't work in Windows it may be faulty, but just in case it's OK and Windows is playing up, this is what you could try in Ubuntu.

System > Preferences > Keyboard. Click on Layouts tab. Check that you have the correct country as the default in the top field. If not, click on the Add button and add your country and set that as the default. Now click on  the 'keyboard model' button. In the new window that opens, choose 'Genius' in the Vendors field, and see if your exact model is listed in the models field. If not, try the others and see if those keys work for you.
