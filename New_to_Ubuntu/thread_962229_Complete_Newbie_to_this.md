### Post by ad_267 on 2008-10-29
Well I don't think Vista is really that bad with vulnerabilities. And I'm surprised you've got by that long without any kind of anti virus.

There aren't really any viruses for Linux based operating systems. Of course it's still possible, but the way that users only have access to their own home folder and many other design features of Linux make it far more secure than Windows. Most people don't bother with a virus scanner unless they are sharing a lot of files with Windows users.

As to bad computer mojo, I guess the worst out there are root kits, but again that's not usually a major worry and attacks on Ubuntu are pretty rare. There was a competition a while back where competitors had to try and crack into a Windows Vista, Mac OSX and Ubuntu laptop. Only the Ubuntu laptop was left standing after 3 days.

It would be best to read up on some of the disadvantages of Linux before you want to give it a go. I use Ubuntu for everything I do, and have no complaints. Some problems that come up are poor support from hardware manufacturers, things like printers can be a pain unless you have an Epson or a HP.

You can set up a dual boot with both Ubuntu and Windows if you want to give Ubuntu a go, then you always have Windows available as a backup if you don't know how to do something in Ubuntu.

### Post by bumanie on 2008-10-29
Agree with the above. Basically, unix like systems, of which  linux is one, were built with security in mind from the start. Some other OSes started off with usability as their main aim and ended tacking security on as an after thought. There are no known viruses in the wild that affect linux. If you download a windows virus, it won't affect linux at all. Even if you were unlucky enough to execute a linux virus, being a modular OS, it is very hard for the virus to infect your whole system and pass itself onto other machines. Any known linux viruses were made in labs as theoretical things and are all protected against at the kernel level. At this point in time, you have no worries with viruses when using linux.
