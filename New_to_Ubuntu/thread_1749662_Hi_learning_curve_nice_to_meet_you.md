### Post by Rasa1111 on 2011-05-04
Hi ben,
 You may want to install "ubuntu restricted extras, for music & video codecs, flash, java, etc.

 Open Dash, means to either 1: Press the 'Super/Windows' key on your keyboard, and the "dash" will come up.
or 2: Click the ubuntu logo in the top left corner. 

Then just start typing whatever it is youre looking for.
So, say you open Dash, and want terminal, Just start typing the word *terminal*
once it recognizes a couple/few letters, it gives you results matching what you typed. 
Then you'll see "Terminal" pop up in the dash below your search bar, and just click it to open a terminal.

Or, you can press Ctrl+Alt+T and it will also open a Terminal. 

To install things, like 'ubuntu restricted extras',
Just Open Dash, (or just look in the panel on the left) for Ubuntu software center, (or "software center" will work if searching in dash).  
When it opens, 
you can type in the name of something, which it will find, and you can install,
or you can just browse through the different apps. 

 (You can also install things with the terminal)
So, Say we wanted to use terminal instead of software center to install "ubuntu restricted extras" for example, 
We would open a terminal, type in 
```
sudo apt-get install ubuntu-restricted-extras
```
and this would do the same thing, only without the GUI. 

Hope something here can help.
If you need anything else, and I can answer, feel free!, ill be around. lol

Good luck,  have fun!

 Hi Ben,

 I forget exactly which key needs to be pressed to "accept the terms" when using the terminal to install it.  That info is around here somewhere, But I have no idea where at the moment, Im sorry. 

 An easier way to deal with that "accept the terms" at the end of the install is to use Synaptic package manager to install it. 

 When the time comes to "accept the terms", a simple graphical interface box will pop up, and all you have to do is "check" the little box that says "I accept", and click "Ok". 
And it will then finish the install. 

 Synaptic may tell you that "restricted extras" is already installed..
If thats the case, just click it and choose to "reinstall", then itll do its thing, and present you with the "accept terms" box when it's ready.
 

 One way you could check to see if it did install properly..
Is to open a youtube video in your browser, and if it plays, you should be set..
but if it tells you that youre missing a plugin, then the install did not fully finish. 

Yes that package does install numerous packages at once. 
Music file codecs, video codecs, flash, java, basically everything one needs to work with any form of multimedia.
