### Post by bumanie on 2008-11-01
I would:
1) Check the integrity of the disc with the check cd for errors tool.
2) Check that the download was clean by doing an [md5 checksum]("https://help.ubuntu.com/community/HowToMD5SUM") or,  if theose two things check out,
3) Download the alternate cd and install with that. It is text-based and usually installs if the live cd is giving problems.
