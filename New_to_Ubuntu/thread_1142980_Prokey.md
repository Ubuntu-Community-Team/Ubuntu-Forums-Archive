### Post by Fenris14 on 2009-04-29
A long time ago in an OS far far away...

Does anybody know of a macro keyboard program working like this (old) little tool (Prokey)?

Okay. It was long ago and on DOS... 

The fun with this tool was to be able to program on the fly some keys and reuse them at will later. 

It worked like this:
There was a shortcut to activate it, **ctrl + alt + =**, if I remember well. You then had to press the key you wanted to program, let's say **ctrl + shift + F12**, and then continued to use your keyboard normally. When you wanted to stop programming this shortcut, you just had to hit the initial shortcut, **ctrl + alt + =**. After that, the **ctrl + shift + F12** would repeat your programmed keys.

Among other features was the possibility of using some formating tools. For example, you could define a key to put in the parenthesis and hyphen in a phone number. In north america, the phones number are written (999) 999-9999. With this feature, we just had to hit the programed key and the numbers, all the other characters would be added automatically. 

This is basic macro stuff but it had the quality of working simply, easily and dynamically. All other macro programs I have seen so far have a lower usability. I either have to let go of the keyboard to use the mouse or it doesn't take window switching even inside a multi-window program. It lack switching capabilities from one program to another. It cannot take preprogramed format. It lack the capability of nesting programed keys. The original even had a recusive function. 

Ever since Windows 95, I've looked around and found no software doing this well enough for my taste. 

Anybody has any idea where I could find this?
