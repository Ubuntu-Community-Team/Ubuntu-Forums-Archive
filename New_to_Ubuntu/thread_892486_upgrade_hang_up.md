### Post by gobbles414 on 2008-08-17
I encountered the *Generating Locales* error while trying to install Spanish language support in Ubuntu 7.10. I don't know if this solution will work for your or not, but... I was able to prevent the error by going into GRUB immediately after turning on my computer (press Esc). I then selected the oldest version of the Linux Kernel from the list.

By the way, I've never had very good luck using the upgrade option. Maybe you'd prefer to just download the 8.04.1 LiveCD, backup all of your files, and reinstall Ubuntu. It takes about the same amount of time to download the upgrades as it does to download the latest LiveCD.
