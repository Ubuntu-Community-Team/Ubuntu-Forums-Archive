### Post by tomreid on 2009-07-16
Yes it will work fine on most windows capable (i.e. X86 chip sets) machines.

The only problem you may face are drivers for the hardware, every install of Ubuntu I've ever done, the PC has worked with the hardware drivers provided by default on the install disc, although some performance was improved by changing out the drivers. 

Wireless cards tend to be the most tricky, so if you are not sure, plug the ethernet cable in when you do the install. 

Ubuntu can easily run without Windows, in fact it is designed to be put on "Virgin" pcs. 

The only reason you see so much advice about installing Ubuntu on machines that already have Windows, is that some people like to experiement with Linux, whilst keeping Windows on their machune in case they need it. In fact the only thing I need Windows for ow is streaming Netflix movies.

Good luck.

### Post by raymondh on 2009-07-16
Yes you can.

Before you do,  kindly read the [release notes]("http://www.ubuntu.com/getubuntu/releasenotes") and some tutorials on installation.

[http://www.ubuntupocketguide.com/index_main.html](http://www.ubuntupocketguide.com/index_main.html)
[https://help.ubuntu.com/community/GraphicalInstall](https://help.ubuntu.com/community/GraphicalInstall)
[http://www.psychocats.net/ubuntu/index](http://www.psychocats.net/ubuntu/index)

Some tips:

1.  Download and burn the image at slowest speed possible
2.  Boot into (aka *set BIOS to boot cd drive first*) the liveCD and check for defects and [md5]("https://help.ubuntu.com/community/HowToMD5SUM")
3.  Try the liveCD in live session (TRY UBUNTU WITHOUT CHANGING) to see how your computer and specs' play/react to the ubuntu version.  Check wifi, sound, keyboard functions, etc)
4.  When happy and ready, go install.

Preparing yourself by reading/researching what you are about to partake will minimize hair-loss-from-frustrations later on.  Nevertheless, the forum is only a liveCD and browser away :)

Happy Ubuntuing.

### Post by fballem on 2009-07-16
Before you do anything else, please make sure that you have backed up your Documents, Music, Videos, and any other files that you will want to keep. These should be on an external disk.

Do not use a Windows backup utility to do this - make sure that you can copy the files intact, or in the worst case, in zip files.

Also please let us know what e-mail program you use and where your contacts are. Depending on how you do this, there are specific ways that these can be migrated to your ubuntu installation.

Ubuntu is not Windows. There will be some things that will be done differently than you are used to. Be patient with yourself, and ask. We've all been through the learning curve, and most of us are happy to help.

Enjoy!
