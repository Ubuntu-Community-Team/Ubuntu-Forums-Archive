### Post by Hobgoblin on 2009-03-28
It's a firefox extension.

> For a single link, right click and select "Mark link as visited" or "Mark link as unvisited" (depending on current state).

For multiple links, The View and Context menus contain the menu entry to "Mark Links Un/Visited", which provides the ability to mark all links, or those within a selected region, as visited or unvisited.

If the 'Always Mark Bookmarks Visited' preference is on, bookmarked links will not be marked unvisited.
