### Post by Paco62 on 2010-05-05
Skype for Ubuntu is buggy and only properly plays a few of the custom .wav files I have collected.  (I use them for custom notification sounds, the default sounds in Skype are too soft.)  But Skype also gives me the option to run a script on an event, so can someone tell me how to write a script to get VLC to run a .wav file located in my Music folder?  I am a complete newbie with respect to scripts.  Thanks in advance

---

### Post by marshmallow1304 on 2010-05-05
Just put this under "Execute the following script:"
```
cvlc "/home/<username>/Music/filetoplay.wav"
```

### Post by WorMzy on 2010-05-05
If you have gstreamer installed, you can use the following command:```
gst-launch-0.10 playbin uri="file:///path/to/the/wavfile.wav"
```
