### Post by undecim on 2009-10-04
Was this while you were installing Ubuntu?

Any operating system needs to format the drive to install. If you installed it to an external drive, it should have been formatting the external drive, or a partition it created on the external drive. If you didn't tell it to reinstall, then it likely created a new partition next to your windows install and was formatting that partition.

In other words, it should have been formatting its own EMPTY partition, not erasing files.
