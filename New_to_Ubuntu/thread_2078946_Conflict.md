### Post by deadflowr on 2012-11-01
> **Yezinki said:**
> Can one install Compiz, CompizConfig Settings Manager & cairo dock using terminal, if yes what are the commands?

Thanks.

If yer running Ubuntu then compiz is installed, so for compiz config settings manager it would be compizconfig-settings-manager(I think), and and cairo-dock.Sudo apt-get install program program.
```
sudo apt-get install compizconfig-settings-manager cairo-dock
```

In terms of conflicts and problems, I've run cairo-dock with compiz and never saw a big problem, however if conflicts and errors occur, I think looking over .xsession-errors would probably be good place to start to see if anything is wrong.
