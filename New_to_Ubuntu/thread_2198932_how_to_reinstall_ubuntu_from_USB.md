Is this W7 and Ubuntu side by side installation or ubuntu under W7?

If it is side by side installation, during installation steps it will tell you that - "you already have ubuntu installed on your system. would you like to overwrite it and install new version?" or something very similar to it. So just do it. It will wipe (EVERYTHING WILL BE DELETED) old installation and install new one. 

If it is inside W7 just uninstall it like a normal program and reinstall it.
Hope it helps. Good luck
