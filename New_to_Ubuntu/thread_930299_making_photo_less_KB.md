### Post by bigmonmulgrew on 2011-06-21
A method I used to use back on windows (using paint)
was to save the file using an older file type that supports less colours. In many cases to the human eye there is little to no difference with many files. Then you save it in a more modern file type which has a better compression ratio eg .png
A bit long winded I know but as long as you have several saving formats available you could do it with pretty much any image editor.
