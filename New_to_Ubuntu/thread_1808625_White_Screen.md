### Post by Terl on 2011-07-20
Check which screensaver you are using.  I have gotten freezes here and there if using an Open GL screensaver.  I switched to blank screen and have not had a freeze since.  This solution is also listed on Wildmanne39's link.  I would try the screensaver to blank screen idea first.  Most of these freezes are GPU related I think.
