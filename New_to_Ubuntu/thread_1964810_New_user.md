### Post by newbie-user on 2012-04-24
Yes, you can download the Ubuntu iso on another computer, create a bootable USB, and then install it on your computer. Here's what you'll need if you're using a Windows computer:

1. Unetbootin to create the bootable USB
[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

2. Your computer needs to be able to boot from USB

3. Free space on your hard drive to install Ubuntu. At the very least, 5GB for a full desktop installation.

4. Ubuntu ISO for use with unetbootin
