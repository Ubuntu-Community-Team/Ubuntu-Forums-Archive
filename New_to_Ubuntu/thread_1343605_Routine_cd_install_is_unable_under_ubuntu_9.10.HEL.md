### Post by jrrader on 2009-12-02
The good news is that you don't need that CD.  Generally wired routers work when you plug them in.  

If it's a wireless router you can configure it through your web browser.  Go to this address:
192.168.1.1   

username is admin and password is ... password

Netgear's setup has a pretty good set of instructions along the right side of the page.  Just read through it and you should be fine.

### Post by 3rdalbum on 2009-12-02
The router stores the internet configuration, not the computer. It will probably just work when you plug it in.

If you want to change the settings, you can access the router's web-based interface. See your router's instruction manual for more details.

My router uses the address:

[http://192.168.0.1](http://192.168.0.1)

Username: admin
Password: admin
