### Post by OutOfReach on 2008-11-28
Try going System>Preferences>Sessions
Then click 'New' type in the name as something like 'to-do list'
then in the command box type in:
```

gedit /Path/To/My/Todo/List

```

Then log out and then back in and it should work.
