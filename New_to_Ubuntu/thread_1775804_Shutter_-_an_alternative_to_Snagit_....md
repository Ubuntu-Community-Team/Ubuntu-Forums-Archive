### Post by cwmoser on 2011-06-05
I have longed for a Linux alternative to the windows-based Snagit.
Snagit is the best Screen Capture Tool I have ever used - but it runs only on Windows.

Now come Shutter - a very nice alternative open source screen capture that works very well and has all the features I typically use for screen capturing.

Below are some notes about Shutter I put together:

**_Shutter &#8211; Screen Capture like Snagit_**

*This is a very good facsimile of Snagit for Ubuntu.*

[http://shutter-project.org/](http://shutter-project.org/)

**_Install:_**
Use Synaptic or
$  sudo  apt-get  install  shutter

If when you right click and the **Edit** button is disable, do this:
$  sudo  apt-get  install  libgoo-canvas-perl 
Then restart Shutter.


_**Hints:**_
1. After initially running, you can start Shutter via the blue speed button in top status bar
2. Select **Simple selection tool** via the down arrow beside **Selection**
3. To use _highlighter_ and put in _arrows_, right-click on your captured image and select Edit.
4. Hold down the control key when using Highlighter for straight lines
5. Special effects:  **Screenshot** &#8594; **Run a plugin** for effects like torn page, etc.
