If your machine is an older one, 
and if it seems to be a little laggy or slow, it might be the
 "visual effects" (which are meaningless/useless) causing it. 

If that happens, 
Just go to the drop down menu,  **System**> then to **Preferences**>**Appearance**..
open the appearance window, click on the far right tab called 'Visual Effects", and set it to "none".

It sounds as if you wont be having that issue, but just in case, it's good to know, and a very simple "fix" for old machines that can't handle lots of fancy "eye candy." lol

 Oh, also, Once you are installed~
You should install what is called ubuntu-restricted-extras. 
You can do this from the Software center, from synaptic package manager, or from terminal. 

From terminal, you would simply type in 
```
sudo apt-get install ubuntu-restricted-extras
```

Or from software center or synaptic, Just type in the name, and it will appear, then just choose to "install". 

 This will give you just about every codec under the sun,
to play all kinds of music files, video files, flash,java,  etc. 

 Then you wont have to do them one by one, or have to search for the proper codecs for whatever you need. 

 Once that is installed, you should be ready to do anything you like.
