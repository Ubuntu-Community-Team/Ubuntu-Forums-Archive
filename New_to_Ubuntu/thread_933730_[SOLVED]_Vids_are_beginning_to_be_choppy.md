### Post by Rocket2DMn on 2008-09-29
Yeah, your xorg.conf is hardcoded with the vesa driver, which is the fallback safe mode driver - no 3d acceleration there!.  Let's reconfigure X and see if that helps.  First make a backup.
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.vesa
sudo dpkg-reconfigure xserver-xorg -phigh
```
then restart X with CTRL+ALT+BACKSPACE (this will kill all open programs, so save your work please).
