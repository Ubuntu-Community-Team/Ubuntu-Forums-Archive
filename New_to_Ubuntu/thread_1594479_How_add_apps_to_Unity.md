### Post by themusicalduck on 2010-10-12
To add them to the sidebar, open up the application so that the icon appears. Then right click the icon and select to keep in launcher.

To add a program as in install it, then either search for it in unity and install from there, or search in the Software Centre and then install it.

The 'reboot and try again' option worked fine for me with Pidgin. After playing with it for a couple of minutes, I discovered the bug.  After installation of new software, the unity interface will not let you add it to the launch bar until after you log out/log in.
