### Post by camper365 on 2009-05-26
Yes you will.
KDE itself was free from the beginning, but after it was based on Qt, which was not free at the time. After pressure from GNU and the FSF, Qt released its software under the QPL (Qt Public License) which GNU accepted as compatible with the GPL.
Usually if you download non-free software from the repositories in Synaptic, you will know because it presents you with a license agreement and other stuff like that. Any software that is non-free is either in restricted (which contains mostly hardware drivers) and multiverse (which is software like the Java JDK)
