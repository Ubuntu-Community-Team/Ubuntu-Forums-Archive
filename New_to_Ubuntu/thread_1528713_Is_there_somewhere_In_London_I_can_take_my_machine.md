### Post by new_tolinux on 2010-07-11
Ubuntu will probably _never_ work with Windows-programs. You'll need something like Wine to get them to work.

If it's only disk access, Ubuntu is capable of reading NTFS (Windows) partitions, but getting your Windows to read ext3/4 (the Ubuntu file-system) will be a lot harder. The easiest way is just save the files you'll need in both Windows and Ubuntu on your Windows-partition.

### Post by ov3rcl0ck on 2010-07-11
Sounds like there is a what dpkg calls a half installed package, or apt calls a broken package on your system. Basically something happened while a package or several packages were installing and they didn't install all the way and basically broke apt.

Luckily theres an easy fix, you can fix it in a single command, but I recommend running these three just to make sure. Try this:

```

sudo apt-get -f install
sudo apt-get autoremove
sudo apt-get autoclean

```

If that doesn't work maybe its an error with dpkg, but I doubt it since you weren't getting this error, or I didn't see anything about you getting a dpkg configuration error.
```

sudo dpkg --configure -a

```
