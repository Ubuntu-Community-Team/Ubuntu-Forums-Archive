### Post by Mark Phelps on 2013-08-01
If Win7 came preinstalled on your laptop, you may be able to obtain full-restore disk media from the laptop supplier for a nominal fee (usually, $20 or so) -- which is a LOT less than going out and buying an new copy of Win7. ### Post by Hexxus on 2013-08-01
Yes, there is the universal usb pen drive loader (unetbootin). You'll need a usb drive and an image of Windows 7 (whichever version your product key is for). 

Then once you get the image on there using the tool, boot your computer off of the flash-drive. During the install when Windows 7 askes where you would like to install it, If I remember correctly it will detect ubuntu and allow you to install side-by-side with it. I'm not 
100% sure on it but I seem to recall it.
