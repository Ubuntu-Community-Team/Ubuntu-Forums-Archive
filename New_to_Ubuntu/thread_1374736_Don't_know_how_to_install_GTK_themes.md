### Post by Methuselah on 2010-01-07
Right click desktop, select change desktop background, and select the themes tab.
I think an install button there will do the trick if you are having a theme file.

### Post by ajgreeny on 2010-01-07
You normally unpack the tar.gz with your archive manager, extract the theme file and then use the theme manager to install it.  It will, however, depend on what exactly the archive contains.

### Post by ajgreeny on 2010-01-07
Much more difficult other than to install another of the ubuntu family usplash packages, and then run ```
sudo dpkg-reconfigure usplash
```Changing the picture you see within a usplash is possible but frankly not worth the hassle, as it involves a lot of command line work and configuration.
