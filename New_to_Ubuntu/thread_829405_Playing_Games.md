### Post by Yuki_Nagato on 2008-06-14
I imagine this would be the best place to start:

[http://appdb.winehq.org/objectManager.php?sClass=version&iId=1717]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=1717")

This is called "WINE."  It allows some Windows programs to be run on a Linux machine.  It is, however, not an easy process, but with the more popular games, as World of Warcraft and Command and Conquer, there often are walkthroughs on setting up the game.
