### Post by Dennis N on 2015-10-24
Looking at the newest release, Ubuntu MATE 15.10 that you ask about:

When you login, you see a **Welcome Screen**. On that is a big green **Software Button** takes you to the **Software Page**. Click on the categories to see suggested software - this is currently being expanded as it is a new feature! Click the install button on any item to install it.

_If you don't see what you want_, click on the Green "**More Apps**" category on this same Software Page.

This option shows you three more tools to find and install software. Each has a description of what it is:

1) App Grid
2) Ubuntu Software Center
3) Synaptic Package Manager

Click the Install button on the right to install any of these!

To visit the Welcome Screen later, go to System > Welcome in the main menu.

Enjoy your Ubuntu MATE 15.10!
