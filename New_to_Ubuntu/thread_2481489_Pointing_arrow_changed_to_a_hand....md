### Post by arieleoar on 2022-12-01
Hi!!

It only happens in Firefox? Have you tested anything else that has some link (e.g. chrome, an hyperlink in a word document, etc)?

If that icon appears everytime you hover over a link looks like the cursor theme changed in the desktop environment, so if you are using Gnome for example you have the *gnome-tweaks* tool (that doesn't come pre installed, you have to install it manually) that allows you to change your cursor theme.
For KDE it has its own preinstalled configuration to manage the cursor themes (as i recall you only has to go to settings).
