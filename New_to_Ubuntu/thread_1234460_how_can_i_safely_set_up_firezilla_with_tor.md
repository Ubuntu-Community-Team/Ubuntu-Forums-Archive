### Post by Paddy Landau on 2009-08-10
Firefox comes with a simple add-on that makes turning Tor on and off easy.

[LIST]
[*]In Firefox, go Tools > Add-ons
[*]Select Get Add-ons > [Browse All Add-ons]("https://addons.mozilla.org/en-US/firefox")
[*]Search for [URL="https://addons.mozilla.org/en-US/firefox/addon/2275"]Torbutton
[/URL]
[*]Read the instructions, and install
[/LIST]
You will find Tor incredibly slow; it's real use is for people living in countries with oppressive regimes, so please use it sparingly.
