### Post by jakslev on 2010-11-08
Hi Noob :o), 

Sure - you can have a dual boot setup with Windows and Ubuntu. I'm not sure of the exact size of the installation - suppose it depends on a lot of factors; but would probably make sense to do 50-50? 

As for choosing what version of Ubuntu to use; you definitely want the 32-bit one if you only have 1 GB RAM. No real advantage with running a 64-bit system on less than 4 GB RAM. In fact the 64-bit systems tend to be more buggy both on Windowze and Linux. 

Hope that gave you some help :)
