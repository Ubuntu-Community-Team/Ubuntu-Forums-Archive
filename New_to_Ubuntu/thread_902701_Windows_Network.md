### Post by bobnutfield on 2008-08-27
Ubuntu includes the Samba client program by default.  This icon is simply a shortcut to opening Windows shares if there are any on your network.  The icon is there to create a search for Windows shares if they exist.  If they did, then the next icon you would have seen is the name of the Windows workgroup used by the Windows machine (which is either the default of "mygroup" or "workgroup".  

Doesn't affect anything.  Wouldn't worry about it.
