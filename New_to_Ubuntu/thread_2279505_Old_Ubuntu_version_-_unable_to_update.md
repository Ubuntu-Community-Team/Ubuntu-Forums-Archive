### Post by Geoffrey_Arndt on 2015-05-24
Probably best to run Xubuntu 14.02.2 on that HP.   Go to the Xubuntu web site [http://xubuntu.org/](http://xubuntu.org/)

Download the 32 bit version iso file, and then use PendriveLinux (a windows program), to burn the iso as an "image" file to a usb flash-stick.  [http://www.pendrivelinux.com/universal-usb-installer-easy-as-1-2-3/](http://www.pendrivelinux.com/universal-usb-installer-easy-as-1-2-3/)

Boot up the HP using the flash-stick (you may have to setup the HP to boot from usb first, hard drive second).   That's done using the HP's BIOS setup program.   

Once you can boot up, the rest is easy if you're just going to let the installer program use the "whole disk" - - let it use the installer defaults and it should install pretty smoothly.   Let us know if additional questions.
