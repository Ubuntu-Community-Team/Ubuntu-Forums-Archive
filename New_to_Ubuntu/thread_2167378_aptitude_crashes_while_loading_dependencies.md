### Post by tgalati4 on 2013-08-21
So it looks like you might have an error in your dependency tree at about 50% point.  You might need to purge it and recreate it.  A normal apt-get check would look like:

tgalati4@Mint14-Extensa ~ $ sudo apt-get check
[sudo] password for tgalati4: 
Reading package lists... Done
Building dependency tree       
Reading state information... Done

Comment out any added PPA lines in /etc/apt/sources.list.   Then try:

```
sudo apt-get clean
sudo apt-get check
```
