### Post by Sylos on 2010-08-26
Hi there.
Nautilus is the file browsing application used in Ubuntu to view your files and folders.

If you click on the home icon under places this will launch nautilus to show you the contents of the /Home directory.

If you want to launch it independently then you can do it from the termianl by simply typing 

```
nautilus
```

Hope that helps.
