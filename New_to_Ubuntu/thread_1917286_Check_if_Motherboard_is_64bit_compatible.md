### Post by oldfred on 2012-01-29
Usually on laptops you cannot change anything. Some make it difficult even to add memory or change hard drives.

Some have all the chips soldered in to save money, not like a desktop that has the way to change things. Also too much more than the original designed chip(s) may cause heating issues as laptops are not well designed for cooling.

Adding memory may be the only option. I only have 1.5GB RAM and run the 64bit system, but that is so I have the same system as my Desktop system. Even then I find I rarely use swap on my laptop, but do not run many or large applications.
