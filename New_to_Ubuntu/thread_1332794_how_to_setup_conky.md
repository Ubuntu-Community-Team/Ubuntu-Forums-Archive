### Post by Trevster on 2009-11-20
It starts with the command "conky" which can be added to the startup programs.

There is a .conkyrc file in the home folder that can be edited to suit your needs. [http://ubuntuforums.org/showthread.php?t=281865&highlight=conky]("http://ubuntuforums.org/showthread.php?t=281865&highlight=conky")

### Post by slumbergod on 2009-11-20
basic conky setups are easy. you only need to copy the file (as sudo) into /etc/conky/ then start it (as regular user).

It gets more complicated when you start to desire fancy features that are not built into the conky app itself - weather, email inbox checking etc.

The HUGE conky thread in this forum has lots of code examples to try.
