### Post by solitaire on 2008-04-24
You should burn it to a disk, although if you run the "Update Manager" it should prompt you if you wish to update via the web. (This will take a few hours to d/l the Updates and install).

Best to backup your "home" folders before you upgrade.

The CD will be quicker.

People say a wipe and clean install is best others say the upgrade works fine.

As long as you have a backup, pick which ever you are comfortable with and take the plunge :D
