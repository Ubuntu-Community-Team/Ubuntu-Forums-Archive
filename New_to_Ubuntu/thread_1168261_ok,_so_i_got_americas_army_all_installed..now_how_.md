### Post by 3rdalbum on 2009-05-23
If you chose not to install a link into your main menu, then that's why it isn't appearing in your main menu.

I don't have the game so I don't know exactly where it would have installed. Try going into a terminal and start typing "amer" (the first four letters of the name). Press the Tab key and hopefully it should auto-complete into the full path, and then you can press the Enter key and it will run the game.

Once you know exactly what the program name is, you can add that name as a command to your menu.
