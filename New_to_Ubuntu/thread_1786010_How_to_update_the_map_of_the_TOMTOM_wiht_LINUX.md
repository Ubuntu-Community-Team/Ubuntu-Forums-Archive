TomTom have stated that they will not be supporting Linux at this time.  This is a bit weak, considering the TomTom runs Linux...

The only solution that works that I've found is to run XP in VirtualBox.  You'll need the PUEL version of VirtualBox from Oracle, as the OSE (Open source edition) in the repositories does not support USB ports. I guess the USB code is proprietary.

Also get the extension pack as it upgrades the USB ports from 1.1 to 2.0 compatibility.

It's easy to install, and then all you need is to install XP in the "virtual" machine you create.  You can then run windows programs on your Ubuntu desktop.

VirtualBox is also good for other things: trying out an Ubuntu Beta, or another distro, or just playing with a virtual machine.  A friend of mine uses it for testing windows malware and finding ways of fixing them

I've recently updated my TomTom this way: got the new V9 navcore and new maps.

The only downside is that you need a bit of extra RAM to run VirtualBox - You need RAM for the host, and RAM for the guest OS (XP in this case)  I've done it on this 1GB machine no problems.

Chris.
