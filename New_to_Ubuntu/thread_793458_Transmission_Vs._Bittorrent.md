### Post by macaholic on 2008-05-13
I much prefer Transmission to the old gnome one. I like transmission over Azureus because IMO, I want a bittorent downloader to be like a noraml download, I don't need all those bloated features. Just let me select the port I want to use, fiddle with some priorities and I'm happy.

Also I love that Transmission is a truly cross-platform app. Since the program basically works by accessing a universal library file, individual GUIs can be written for each os, for example, transmission has a full Cocoa GUI on OSX.
