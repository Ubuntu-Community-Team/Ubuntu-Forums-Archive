### Post by sudodus on 2013-06-04
Are you using Chrome or Chromium-browser? My experience is that today Firefox uses less memory and works better than before and better than Chromium-browser. I have not used Chrome in linux.

XFCE is a desktop environment of its own, not a flavour of Gnome (although the desktop can be tweaked to look like a Gnome desktop). If you want something lighter, try Lubuntu with LXDE, and if you want to run really light, at the log in screen, select the Openbox window manager (not even a desktop environment). Right-click on the grey background and get a terminal window. From there you can do 'everything', and really without eye-candy that makes the system slow down.
