### Post by bennachie on 2009-06-14
Netbook Remix comes as a .IMG file, not a .ISO. Full details of how to transfer such a file to a USB stick are here:

[https://help.ubuntu.com/community/Installation/FromImgFiles](https://help.ubuntu.com/community/Installation/FromImgFiles)

Most of the previous contributors have focused on .ISO images, since these are much more common (it is possible to generate hybrid .ISO images, which can be transferred to a USB stick either by disk imaging or by unetbootin, but only Mandriva seem to have followed that path at the moment). You can use a 1GB stick, but 2GB sticks are so cheap now that I wouldn't bother.

Incidentally, the vanilla version of Ubuntu also runs perfectly on modern netbooks (indeed, it runs fine on my own eeePC 4G). If you really want to maximise screen real estate, you could have a look at Linux Mint 7, which is closely based on Ubuntu but adopts a slightly different version of the Gnome desktop, and uses some very attractive artwork. Both should work pretty much out of the box.

> there is.

you can go get a usb drive (or other removable media, their pretty cheap)
and use the info here to install from that media

[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

That will allow you to choose a whole bunch of different distributions and versions to put on a flash drive. The benefit to the flash drive is that your changes are saved even if you don't install. If you install your system, and after a while you make it customized, you can use the USB Startup Disk Creator (System -> Administration -> USB Startup Disk Creator) you can create a customized Ubuntu on a flash drive for when you need to reinstall. If you do that you keep all your settings.
