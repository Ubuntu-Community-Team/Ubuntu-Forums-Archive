### Post by bcbc on 2011-07-29
Sometimes, if the 'time to display operating systems' is set to zero, Windows won't give you a choice.
You can check this by right clicking on My Computer, Properties, Advanced, Startup & recovery settings. Check and modify the timeout to 15 or 30 if zero.

The ISO is still present if you install with Wubi... it saves a hidden copy but it sounds like you should have the original you downloaded as well. Hit F3 and the whole computer for *.iso
