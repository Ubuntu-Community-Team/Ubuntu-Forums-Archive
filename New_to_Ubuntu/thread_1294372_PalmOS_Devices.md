### Post by sgosnell on 2009-10-18
PalmOS devices are PDAs and smartphones which run PalmOS.  Support is built in, and you can't delete anything to provide more space.  Python is a programming language and runtime environment, and you need it installed to run many programs.  The bug report tool reports bugs encountered while running programs.

Don't worry about them, there isn't anything you can realistically do to gain more space by trying to delete these.  You can remove packages you don't need, and you may be able to gain space through this.
