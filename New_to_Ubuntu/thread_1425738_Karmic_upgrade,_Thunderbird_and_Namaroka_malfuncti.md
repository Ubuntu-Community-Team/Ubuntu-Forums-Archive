### Post by Michael Dooley on 2010-03-09
I think what you now have installed is Shredder, a pre-release form of Thunderbird. This surprised me too.

At the dialog box this morning (clicking on an html link in an e-mail), I navigated to File System, usr, bin, and then selected firefox. I then saw a message that explained how/where I could change this setting.

Note it down if you want to know how to change this setting.
