### Post by 3rdalbum on 2009-09-02
Benefits:

1. You can address more RAM (a full 4 gigabytes and more)
2. Some mathematical operations run quicker, which adds up when you're doing a lot of them (video encoding, for instance)

Disadvantages:

1. Slightly more RAM gets used
2. Using 32-bit Debian packages requires an extra step or two (often using Getlibs); not really an issue these days as almost everything is 64-bit.

BTW Wine works perfectly well on a 64-bit host.

If you have a 64-bit processor, then just install the 64-bit edition, and you'll be set for life.
