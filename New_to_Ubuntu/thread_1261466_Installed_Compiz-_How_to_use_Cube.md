### Post by SoftwareExplorer on 2009-09-08
Go to Applications->Add/Remove. Search for 'ccsm'. Check the box next to 'Advanced Desktop Effects Settings'. Hit apply. Hit apply again in the window that comes up. It will ask for your password. It will download and install it. Go ahead and close all of the windows we opened when it's done. Go to System->Preferences->CompizConfig Setting Manager. Scroll down to the desktop section. Check the box by destop cube. It will complain about conflicts, disable the desktop wall and enable the cube when it asks. You will probably also want to check Rotate Cube.

### Post by Aeroblade on 2009-09-08
You also might want to set the Number of Desktops to 4 in General Options. (Otherwise it's  more of a piece of paper than a cube).  You also might have to install your video driver (under System => Administration => Hardware Drivers) if you have a discrete video card.
