### Post by dat311 on 2008-05-18
My guess is that you don't have enough RAM for Ubuntu.  I tried Ubuntu's live CD on my system, but it wouldn't even run on 128Mb, so I cannibalised some other old PC's and got it up to 256Mb.  The live CD worked, but was horribly slow - Open Office was just crawling.  I've now installed Xubuntu instead, and am happily posting from that machine now. :)

If you're not sure of your system specs, go to the Start button, then Programs -> Accessories -> System tools -> System information

This will give more detailed information about your specs than the "system" icon in Control Panel (though that would give you enough info, I guess)
