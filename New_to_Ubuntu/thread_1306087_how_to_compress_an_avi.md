### Post by 3rdalbum on 2009-10-30
The software that you use to convert the DVD should have an option for "bitrate". Turn the bitrate down, and the file size will come down too. You should re-rip your DVD in order to get the lower file size.

You'll probably never really get a video file down to "e-mailable" size - you could use a service like Rapidshare or Megaupload to send the file.
