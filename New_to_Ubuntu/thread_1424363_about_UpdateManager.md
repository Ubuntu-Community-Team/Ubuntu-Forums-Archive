### Post by egalvan on 2010-03-07
I've never had Update Manager propose "updates" to non-installed software.
For instance, I don't have Apache installed on my work machine,
and have never gotten an up-date for it...

Update Manager uses the list of installed software to check for needed up-dates...

It's possible that software you think is not installed,
is actually installed...

This can be so true with libraries and such types of shared software...


Another way to check on installed/not-installed is to use Synaptic Package manager...

Click on the "Mark All Upgrades" tab and then browse them to see if any are really not installed...


And Synaptic has a notification window at the bottom that tells you how many packages are available, and how many are installed...
you may be surprised at the numbers...:)
