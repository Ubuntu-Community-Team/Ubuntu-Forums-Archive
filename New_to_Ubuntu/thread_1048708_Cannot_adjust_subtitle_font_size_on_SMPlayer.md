### Post by rvm4000 on 2009-01-23
Those options in the preferences dialog set the **default** size for subtitles. That's it, the size that will be used when playing **new** videos.

For videos you already played, you can change the size using the options "Size+" and "Size-" in the Subtitles menu.

Note that the size of subtitles can only be changed for text subtitles, it won't work with DVD subtitles or idx/sub files.
