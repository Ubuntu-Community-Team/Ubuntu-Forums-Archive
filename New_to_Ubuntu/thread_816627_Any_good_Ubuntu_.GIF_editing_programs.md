### Post by EXCiD3 on 2008-06-04
When you edit an animated gif with gimp each frame is a separate layer. The menu option *Filters - Animation* has some tools for animation. You may need to install GAP the GIMP Animation Package in order to get everything you need. GAP is available for Gimp 2.4 here: [ftp://ftp.gimp.org/pub/gimp/plug-ins/v2.4/gap/](ftp://ftp.gimp.org/pub/gimp/plug-ins/v2.4/gap/) I'm not sure whether or not that will work with your currently installed version of gimp or not, but its worth a shot.
