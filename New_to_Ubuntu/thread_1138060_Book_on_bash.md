### Post by hw-tph on 2009-04-26
If you want to do bash scripting, look no further than the excellent [Advanced Bash Scripting Guide](http://tldp.org/guides.html) by Mendel Cooper.

If you want to learn how to do things by the command line instead of relying on graphical user interfaces, [the Rute](http://rute.2038bug.com/index.html.gz) is usually a good start. While it is quite dated by now and doesn't cover Ubuntu (which wasn't anywhere around at the time the document was written) it is still very helpful in understanding the general workings of Linux systems.
