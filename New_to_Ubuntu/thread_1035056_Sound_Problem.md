### Post by Terl on 2009-01-09
Ubuntu is using Pulse Audio and not alsa.  For more info on pulse audio [go here]("http://www.pulseaudio.org/").

You could uninstall the pulse audio and install alsa.  Some good info here:  [https://help.ubuntu.com/community/SoundTroubleshooting](https://help.ubuntu.com/community/SoundTroubleshooting)
