### Post by chris_0076 on 2009-12-24
I did the same sort of thing with mine installing it on a flash drive.

All you have to do is do an installation just like normal. Just make sure that you pick the external hard drive to install on. So in step 3 (I think) it is you pick use all of the space and then pick your external hard drive. Then when you get to the last step (I think step 7) you go to the advanced button and make sure to install the boot loader on the external hard drive.

See this website for more specific instructions.
[http://www.pendrivelinux.com/ubuntu-804-usb-hard-drive-install/](http://www.pendrivelinux.com/ubuntu-804-usb-hard-drive-install/)

Hope this helps
