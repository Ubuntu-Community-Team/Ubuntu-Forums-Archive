### Post by Larkspur on 2011-05-03
If you've installed cssm, you should be able to: go to Desktop > Viewport Switcher > Desktop-Based Viewport Switching and define the mouse buttons for Move Next and Move Prev (buttons 5 and 4?) and you should be able to use your scroll wheel if the desktop has focus.

### Post by malspa on 2011-07-27
Another nice option for changing workspaces with the mouse (sorry, this doesn't involve the mouse wheel, but it does work when windows are maximized) is to go to CompizConfig Settings Manager > Desktop section > Expo > Bindings tab and set up a desktop edge to bring up the workspace switcher. 

I have mine set to the bottom right corner.

Then, when the workspace switcher comes up, simply **right-click** on the workspace you want to go to. With this set-up, you can change workspaces with one click -- closer to how we did it in GNOME2 -- although it's a right-click instead of a left-click.

Anyway, it's another approach for changing workspaces in Unity.
