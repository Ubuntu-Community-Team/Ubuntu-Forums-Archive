### Post by diablo75 on 2010-06-21
You should be able to download flash as a Deb file from Adobe's website.

Alternatively, I think it's included with Ubuntu Restricted Extras, which you can search for and install with the Ubuntu Software Center (formerly Add/Remove at the bottom of the Applications menu).

I understand the 64-bit version works fine these days.
