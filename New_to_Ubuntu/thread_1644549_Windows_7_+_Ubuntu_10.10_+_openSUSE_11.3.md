### Post by Rubi1200 on 2010-12-13
I agree with redbook4574 as far as the order of installation. Ubuntu is pretty good at picking up other operating systems and will happily add the right items to its menu, allowing you to boot all 3 systems.

However, make sure you prepare the partition in Windows 7 using the built-in Disk Management tools and run chkdsk a few times to make sure Windows is not complaining about anything.

Couple of other things:
Backup all data first

Last time I checked the latest OpenSuse installer has some odd options (difficult really I suppose) so I suggest you also ask on their forums if there is anything you need to be aware of before you start. What I mean is that if you are not careful at the partitioning stage you could end up wiping everything out!

Hope this helps.
