man ls

but not pointed to another door to the answer.

Colors in ls's result set when executed in terminal are defined in a database and can be accessed and controled by, according to 'man ls', 'dircolors' command. To see all the color definitions, according to 'dircolors --help', run `dircolors --print-database'.  ):P
