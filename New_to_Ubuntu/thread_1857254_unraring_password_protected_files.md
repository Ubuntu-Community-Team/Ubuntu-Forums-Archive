### Post by papibe on 2011-10-10
The Ubuntu archiver or File Roller uses command line tools to actually compress and uncompress data.

Some of the tools, like rar and others, are not installed by default. Install unrar (non-free) from the software center, and then try to unpack your file.

I hope it helps,
Regards.
