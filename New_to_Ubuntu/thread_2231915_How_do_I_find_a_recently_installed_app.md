### Post by buzzingrobot on 2014-06-28
Icons appear in the Launcher (the dock) only when you choose to put them there. Icons will temporarily appear in the dock after an app is launched.  Right click on that icon to add it permanently.

Properly packaged and installed applications should be find-able in the Dash:  Tap the Windows key, begin to type the name of the application, and its icon should be displayed in the Dash.

### Post by grahammechanical on 2014-06-28
If you click the power/cog icon you will see a link to the Ubuntu Desktop guide. It is called Ubuntu Help. It explains how we work in Ubuntu. Once you have opened the guide click on the heading Desktop, applications & windows and then click find apps, files, music, and more with the Dash. Using the Desktop guide is quicker than waiting for a response from a forum.

Regards.
