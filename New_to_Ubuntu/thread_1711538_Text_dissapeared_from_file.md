### Post by Telengard C64 on 2011-03-21
Some editors (Emacs is one example) can automatically backup the previous version of a newly edited file. The backup copy will have the same name as the original, but usually followed by one or more extra characters such as **~**. Check for such files.

It may be too late to recover your file (or maybe not), but to prevent future disasters you need the backup religion.

[list]
[*][https://help.ubuntu.com/10.04/keeping-safe/C/backup-files.html](https://help.ubuntu.com/10.04/keeping-safe/C/backup-files.html)
[*][https://help.ubuntu.com/community/CategoryBackupRecovery](https://help.ubuntu.com/community/CategoryBackupRecovery)
[*]For more ideas perform an [advanced search](http://ubuntuforums.org/search.php) of these forums for threads with the word *backup* in the title.
[/list]

I wish you success.
