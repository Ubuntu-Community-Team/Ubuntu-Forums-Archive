### Post by sikander3786 on 2011-01-15
Do you mean you can't find it or do you mean it is not working?

You can find it under CompizConfig Settings Manager > Effects > Paint Fire on Screen.

If it is enabled and still not working, try a reboot.

### Post by sikander3786 on 2011-01-15
You need to install the compiz-fusion-plugins-extra package.

Applications > Accessories > Terminal:

```
sudo apt-get install compiz-fusion-plugins-extra
```
