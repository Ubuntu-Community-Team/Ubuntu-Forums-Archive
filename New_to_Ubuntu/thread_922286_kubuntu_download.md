### Post by t0p on 2008-09-17
> **Louis de Broglie said:**
> What is torrent anyway ? :lolflag:

This is when you use a peer-to-peer network to download files.  You use a program like transmission, bittorrent, ktorrent.  It downloads the file in little bits, from different people all over the internet.  You can stop and restart the download at any time.  You can read all about the protocol [here]("http://en.wikipedia.org/wiki/BitTorrent_(protocol)").

Install a p2p app, like bittorrent.  Go to a "torrent tracker" site - my faves are [thepiratebay.org/]("http://thepiratebay.org/") and [www.mininova.org/]("http://www.mininova.org/"), though others might be better for ubuntu isos.  Search for the name of the file you want and click on the link... it should start automatically.

1. Make firefox your active window and type CTRL + y

2. Double click on the file you have downloaded on the firefox download manager

3. Click the add button in the "transmission" window

4. If you want to see your progress, on the upper right side of your gnome panel, that is, before the network icon and the clock, you will see an icon with two little red things , double click it, and you will see your progress
