### Post by iaculallad on 2008-06-03
I edited you're original xorg.conf file posted above. Create a backup of your current /etc/X11/xorg.conf file and copy/paste the xorg.conf file below to overwrite your original xorg file.

Create xorg.conf backup:

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.bak
```

Open file for editing:

```
sudo gedit /etc/X11/xorg.conf
```
