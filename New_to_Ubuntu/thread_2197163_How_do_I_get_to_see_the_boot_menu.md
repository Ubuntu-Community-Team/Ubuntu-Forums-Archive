### Post by ajgreeny on 2014-01-02
For the boot priority device menu there will probably be a key-press needed; which key to press is normally shown on the bottom of the first boot screen with manufacturer's logo.  This may also get you to the BIOS or UEFI setup but there may be another key for that also shown on first boot screen.

The grub menu will only show when you have more than one OS on a machine, so if you have just a single OS installation you will not see it unless you hold (or keep tapping) shift, for some machines, but Esc for others, including mine, with UEFI rather than legacy BIOS.
