### Post by howefield on 2011-09-18
You might find that the deb file in the partner repository is more up to date.

Enable the partner repository by going into Synaptic Package Manager > Settings > Repositories > Other Software tab and place a check against Canonical Partners.

Close and press the reload button, then install.
