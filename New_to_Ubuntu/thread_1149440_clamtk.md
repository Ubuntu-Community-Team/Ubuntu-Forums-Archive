### Post by Sir Jasper on 2009-05-05
Hi,

Since you deleted it you would not expect to see it in quarantine.

Do you use and Windows programs, e.g. via wine? If so, it would usually be best to investigate any infection before quarantining or deleling - especially if it is in a Windows System folder.

Avast (for Debian) has a significanly higher detection rate than Clam and it scans really fast (but does not offer a right click context menu scan as does Clam). It is also best for those with broadband as the daily update is full (i.e. not incremental).

I use "Wine" and I am considering Avira Antivir which has an even better record and can provide active protection - but is quite complicated to install.

If your email and browser are still working it is unlikely you have any worries and it might, in any case, have been a False Positive.

My regards

### Post by Sir Jasper on 2009-05-06
Hi,

Since you do not use Windows I am 99.99% certain (from my reading) that no Windows virus can do your own machine any harm at all.

However, I think it is considerate not to risk passing on Windows viruses (e.g. via emails)to friends who use Windows.

If you google "avast debian" then download it and then click on it - it should, from memory, install in your Accessories Folder.

You could then use either Avast or Clam.

As regards your own protection I use a firewall (with outgoing protection) though many users do not. Enter "firewall" in the search box above if you want to learn more.
