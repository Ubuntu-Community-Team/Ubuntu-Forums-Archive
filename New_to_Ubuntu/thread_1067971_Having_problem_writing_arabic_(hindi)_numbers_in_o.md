### Post by Ripose on 2009-02-25
Other than changing the language settings you also have to be using a font that includes the Hindi numerals.

If that does not help you might have to switch to an Arabic Keyboard Layout, but you will still need fonts that contain numbers & letters.

### Post by Majiq on 2009-02-25
There are actually specific fonts for writing like that. I would have thought that that one option would have triggered everything, but for lack of that, use one of the "digits" versions of the keyboard (i.e. azerty/digits or qwerty/digits). That should fix your problem for writing. I don't know about reading, though.
