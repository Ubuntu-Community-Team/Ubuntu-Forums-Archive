### Post by iaculallad on 2008-07-22
> **redfox1160 said:**
> I wanted to know how to uninstall wine-doors. I'm not sure how to uninstall it because I didn't install it through synaptic. Any help is appreciated. Thanks.

Try the commands below on your terminal:

```
sudo apt-get --purge remove wine-doors
sudo apt-get autoremove

```
