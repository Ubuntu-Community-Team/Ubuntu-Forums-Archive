### Post by Pougnet on 2011-01-22
Xubuntu can be installed with one of 2 CDs, both of which require at  least 1.5 GB of hard drive space. Installing with the Desktop CD  requires 192 MB of RAM, while the Alternate CD, which uses a text-based  installer, requires 64 MB of RAM, and also allows access to additional  options for the installation.[]("http://en.wikipedia.org/wiki/Xubuntu#cite_note-XubuntuRequire-21")[]("http://en.wikipedia.org/wiki/Xubuntu#cite_note-UbuntuRequire-22") Once installed, Xubuntu can run with 192 MB RAM, but 256 MB RAM is  strongly recommended.
Straight of wikipedia.

### Post by epp on 2011-01-22
Ubuntu 10.10 will not run systems containing i586 and below CPU's as well as i686 CPU's that do not have CMOV support, beginning with that version.

Ubuntu 10.04.1 LTS was the last version that will run on these CPU's and it will be supported until sometime in 2012, I believe.

*FWIW, the LiveCD of 10.10 will _not_ boot on a PC with an AMD Athlon 32-bit CPU (i686) that _has_ CMOV support...*
