### Post by UltimateCat on 2012-07-22
Anything that you are able to download from off of the internet should find it's way into your downloads folder.

Places>Downloads folder

Depending on which version of the kernel of Ubuntu you installed:

Wine should be at the end of your Applications Menu above Ubuntu Software Center.  And the Wine doors also should be there.

Windows associates it's files with .exe
Ubuntu associates with .deb and .run 

There is community documentation that can help you.
[http://help.ubuntu.com/community](http://help.ubuntu.com/community)
