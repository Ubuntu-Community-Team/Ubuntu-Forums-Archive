### Post by na5h on 2013-03-05
The specs are more than sufficient. Before you install, you should boot Ubuntu from a LiveCD to make sure that everything is working the way it should (wifi, etc). You can also do a dual-boot install and run Ubuntu and Windows side-by-side.

edit: mastablasta beat me to it. Here's some more information about the UEFI and Secure Boot thingy: [https://help.ubuntu.com/community/UEFI](https://help.ubuntu.com/community/UEFI)
