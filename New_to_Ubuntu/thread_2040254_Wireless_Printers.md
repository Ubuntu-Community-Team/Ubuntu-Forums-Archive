### Post by ajgreeny on 2012-08-10
Go with HP and you'll not usually have any problems.  I have a HP Photosmart b110 which is actually my wife's printer, and it has worked faultlessly by wireless connection with all versions of *ubuntu from 10.04 to 12.04.  It needed a newer version of hplip when using 10.04 than was in the repos, but that is no problem either as you can get it and easily from [http://hplipopensource.com/hplip-web/index.html](http://hplipopensource.com/hplip-web/index.html) and then follow the instructions to install it at [http://hplipopensource.com/hplip-web/install/manual/distros/ubuntu.html](http://hplipopensource.com/hplip-web/install/manual/distros/ubuntu.html)

I don't know if it is necessary, but I have given the printer a static IP using the router setup pages so I know which address it is at always.

### Post by wyliecoyoteuk on 2012-08-10
Always a good idea to use a static IP for a printer.
Wireless printers are just network connected printers that use a wirelss interface. As long as you are using a wireless router or access point, and not a direct connection, the only worry is that the printer driver works with 12.04
