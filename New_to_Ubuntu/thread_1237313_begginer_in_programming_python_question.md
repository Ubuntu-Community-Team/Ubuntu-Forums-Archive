### Post by MrJoeyUK on 2009-08-11
> **heyyy said:**
> speaking as a beginner what is IDLE?

If you haven't already, go to applications > add/remove programs and search python, and install the IDLE that shows up. It's pretty much a text editor which you can write python code in, and save it.. it has syntax highlighting and stuff so it's pretty cool.

Although you could always just write python code in a simple text editor(I'll second Geany), but not all of them have syntax highlighting, and with IDLE you can run it within the IDLE shell without having to run it in the terminal.
