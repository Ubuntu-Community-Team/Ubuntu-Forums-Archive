### Post by bcbc on 2011-07-10
Check the "time do display operating systems". You can get there by right clicking on Computer, select Properties, Advanced system settings, Startup & recovery settings. If it's zero set it to 15.

Also while you're there check the default OS drop down box. It should have an entry for both Ubuntu and Windows (don't change this from the default of Windows).
