### Post by HiImTye on 2012-06-23
please remove the bit about what each type of file is. the forum rules dont allow for piracy.

that said, if you want to convert video files, you can use a graphical editor such as Avidemux, OpenShot, or PiTiVi (I have used all of them and they each are good for different purposes)
alternatively, if you are concerned with only speed then there are two very useful command line tools:
ffmpeg and mencoder
each of them have extensive features, documentation and between the two of them, short of actually editing your videos they can do pretty much anything (including capturing your desktop and streaming video files or live captures to the internet)
