### Post by Neoncamouflage on 2011-06-23
Are you 100% set on learning java? If this is your first programming language, and you don't specifically need it for something or just have some really strong urge to learn it, it's probably best to start with Python. Far simpler and clearer than Java in my opinion, and you can use it to easily write scripts for daily use. I've spent some time learning both, and I have to say, I like Python more just in terms of the general learning experience.

As for Java, if you really want to learn it, I suggest Java For Dummies. Very nice book and comes with a useful program for writing/compiling/running Java programs. May have to use Wine for it though.

### Post by bleutyler on 2011-06-23
It is never too late.

Try installing Eclipse, it is a IDE for java development.  You can find it on the Ubuntu Software Center.  I have used it in a professional environment, and it is open source so there is even a chance your course uses it as well.

Then from there, for a quick start you can type in "intro to java and eclipse" into the  Youtube search bar.
