### Post by snowpine on 2009-07-13
If you already have a working Ubuntu install, you can easily convert it to Ubuntu Studio.

Open the Synaptic Package Manager (gksu synaptic) and install all packages starting with 'ubuntustudio-' as well as the package 'linux-rt' (the Linux real-time kernel).

The real-time kernel and the suite of multimedia applications are the two main differences between Ubuntu and Ubuntu Studio.

Hope that clears things up for you.
