### Post by MIJ-VI on 2012-07-03
Re dual-booting Windows 7 with Ubuntu 10.04.x - 12.04 LTS from their respective partitions on a single hard drive:

Some Windows updates will overwrite the hard drive's MBR with a fresh copy of Windows' bootloader (which only recognizes Windows) thus deleting **GRUB2** ( IIRC, the version of Ubuntu's bootloader as of Ubuntu 9.10 or later) which also recognizes Windows.

This ***will*** happen and it may also leave one's machine incapable of booting into either OS (which happened to a new Windows 7 Home Premium-equipped Acer Aspire 5517 a while back causing me to dump Windows for good). ):P

Ubuntu newcomers who wish to dual-boot with Windows would do well to review and print out these now somewhat dated **GRUB2** recovery instructions...

RecoveringUbuntuAfterInstallingWindows
[https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows](https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows)

...or consider installing Ubuntu on a Windows hard drive via WUBI until they're comfy with their new OS:

[http://www.ubuntu.com/download/desktop/windows-installer](http://www.ubuntu.com/download/desktop/windows-installer)
