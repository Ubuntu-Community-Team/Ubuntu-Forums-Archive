### Post by 3rdalbum on 2013-09-07
12.04 predates the AMD Radeon 7670m, which means you probably can't use the automatic driver installation feature in Ubuntu. You would if you were using 13.04.

A quick Google search yielded these two pages:

[http://wiki.cchtml.com/index.php/Ubuntu_Precise_Installation_Guide#Installing_Catalyst_Manually_.28from_AMD.2FATI.27s_site.29](http://wiki.cchtml.com/index.php/Ubuntu_Precise_Installation_Guide#Installing_Catalyst_Manually_.28from_AMD.2FATI.27s_site.29)

[https://help.ubuntu.com/community/BinaryDriverHowto/ATI#Installing_upstream_drivers_directly_from_AMD.27s_website](https://help.ubuntu.com/community/BinaryDriverHowto/ATI#Installing_upstream_drivers_directly_from_AMD.27s_website)

In this case you want to install the driver straight from the AMD website.

If you have any problems installing this driver, let us know where you got stuck and we'll try and help.
