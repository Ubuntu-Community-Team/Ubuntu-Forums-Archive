### Post by Joeb454 on 2008-05-10
I'm not sure about the last 3, but for the first one.

Yes, before the install actually happens (the final screen before you tell it to install the system). In the bottom right hand corner, there is an button that says advanced.

Click this - it will show you some options for grub, if I remember correctly, one is a checkbox, and unticking this will tell it to NOT install grub.

All you would have to do then, is add the system to your current grub.
