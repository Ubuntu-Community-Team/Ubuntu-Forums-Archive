### Post by ugm6hr on 2008-05-04
> **weeman7007 said:**
> i need something that is fairly easy to use as its not just me who will be using it, my step mum will use it more if it doesn't lag. how easy is puppy to use? i like the name :)

Puppy is very easy.

The problem with inexperienced users and Puppy is that you always log in as root (i.e. full administration rights without passwords).

So it is just as easy to make a mess of the installation.

As I found out when I tried it on my mum (she has since been happy with Ubuntu).

With 64MB, a minimal install Debian or Ubuntu might be possible.  IceBuntu might fit (although I haven't tried it).

Nevertheless, Puppy is worth a try to start with.
