### Post by sadaruwan12 on 2012-05-14
These kind of things tends to happen when you do a upgrade some times files get dropped with out notice and it gives issues afterward.

So my advice to is if you can get a Ubuntu cd image then create a live CD or a USB stick then run a live session to see if the same issue occurs if not then back up your home folder any other file you have and do a clean install of your system.

If that problems persists on the live session as well then let us know will see what we can do.
