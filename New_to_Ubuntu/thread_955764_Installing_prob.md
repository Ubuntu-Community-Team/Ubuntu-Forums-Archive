### Post by Crandom on 2008-10-22
Put your cd in the cd drive, then reboot your computer. It should load up on the cd, with a menu saying Ubuntu above it. If it doesn't restart, press the button that your computer says is "Multi-boot selection" or "Boot device selection" and select your cd drive.

On the menu from the livecd, select "Check cd for defects" and let that run. It should tell you if there is any problem. If there is, reply.
