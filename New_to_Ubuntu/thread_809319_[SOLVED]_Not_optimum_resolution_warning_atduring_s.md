### Post by wpshooter on 2008-05-27
Yesterday I setup an old IBM Netvista desktop for my sister.

I installed Gutsy on it and that went fine.

However, she is using one of the new Samsung (model 906BX) 19" widescreen LCDs which apparently has a maximum resolution of 1440 X 900.

The display seems fine once I get in to X, the desktop/gnome displays fine and the resolution is showing as 1440 x 900.

However, during bootup and restart/shutdown, I am getting a little diagonal (which I assume is being generated by the LCD) saying that the resolution is NOT at optimum of 1440 x 900.  The diagonal basically covers up the Ubuntu splash logo.

Is there anyway to edit perhaps the splash configuration file in Ubuntu so that I do not get this optimum resolution warning on the splash screen ?

I was sort of under the impression that the splash screen could not be displayed properly at that high of a resolution, is that correct ?

Thanks.

### Post by sayakb on 2008-05-27
```
sudo apt-get install startupmanager
```
Run startup manager (system->administration->startupmanager) and adjust the boot resolution.
