### Post by lavinog on 2009-10-07
Welcome to ubuntu.
You will be able to get your questions answered much quicker if you use descriptive titles for your threads.  A title such as "Putting Ubuntu on thumb drive" would be better.  Not only will it help you, but it will help others looking for the same information.

How did you get the ubuntu cd? did you download the iso file and burn it to cd?
The iso file can be used to install the live cd onto a flash drive.
Go to:
System > Administration > USB Startup Disk Creator

If you don't have this option you can add it by clicking here: [apt:usb-creator](apt:usb-creator)
