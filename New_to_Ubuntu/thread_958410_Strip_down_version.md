### Post by Dr Small on 2008-10-25
> **Idefix82 said:**
> There is Xubuntu which is the lightweight version of Ubuntu but mainly because it uses the xfce desktop environment which is lighter than GNOME and KDE. 
Xubuntu now uses GNOME libraries, so it is not very lightweight anymore, just a different 'flavor' of Ubuntu again. If you want to go lightweight, install Ubuntu from the Alternate CD, and install the essentials.

Then install Xorg, your favorite *box or WM and whatever other applications you need. It will be much more lightweight that the default Ubuntu installation.
