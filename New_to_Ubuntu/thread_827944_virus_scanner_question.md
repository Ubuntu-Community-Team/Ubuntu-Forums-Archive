### Post by Rocket2DMn on 2008-06-13
Indeed.  Essentially, because of the nature of open source software, viruses don't really exist in linux.  This is partly because every distribution of linux is a little different, but also because it is more secure out of the box, and if some problem where to be recognized that a hacker or virus could take advantage of, the problem would be fixed so quickly that the virus would not have a chance to spread.  There are so many eyes on the open source code, that problems are recognized early and somebody/anybody can fix them.  
Some people claim that linux just isn't a target for viruses, but this is not true - there are a fair number of desktop users, and linux owns a very large portion jof the server market - the chances are if you are visiting a webpage, it is hosted on a linux machine.
There aren't currently any viruses "in the wild" for linux.

That being said, remember that security doesn't come in a box - it's not "install and forget" - it is an active process for everybody.  Keeping your computer safe is your job, so have a good look at that thread linked in the previous post, it is full of valuable information.

### Post by hyper_ch on 2008-06-13
there is very little need for a virus scanner on linux...

(1) if you run a mailserver you may want to delete virus ridden emails
(2) if you run a fileserver and have windows clients
(3) if you yourself use windows and exchange files/apps
(4) if you want to protect others from passing virus ridden documents to them

However, to protect yourself you don't need it. Linux virus scanner search for windows viruses.
