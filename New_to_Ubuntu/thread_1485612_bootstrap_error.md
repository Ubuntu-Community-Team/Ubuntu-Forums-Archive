### Post by mikechant on 2010-05-17
You need a minimum of 256Mb RAM to install from the standard installation discs, so that would explain them not working.

You should be able to install the standard version of Ubuntu (from the alternate installer that you already have) instead of installing the command line version.

However, you might well find it far too slow with 192MB RAM and do better with a more light-weight version of Linux for this PC.

xubuntu (see [http://www.xubuntu.org/getubuntu](http://www.xubuntu.org/getubuntu) ) is basically ubuntu with a light-weight interface, and should just about run OK on 192Mb (but they still recommend 256Mb).

Lighter still is Lubuntu (see [https://wiki.ubuntu.com/Lubuntu](https://wiki.ubuntu.com/Lubuntu)). It looks like this is very much aimed at a machine with your specs. However, note the current version 10.04 is still a beta version unlike Ubuntu/Xubuntu where 10.04 is a final release.

Leaving the Ubuntu family, you could look at versions of Linux aimed at even lower spec machines such as
Puppy Linux - [http://www.puppylinux.com/](http://www.puppylinux.com/)
DSL (Damn Small Linux) - [http://www.damnsmalllinux.org/](http://www.damnsmalllinux.org/)

DSL has an option to run entirely in RAM if you have at least 128Mb - this will give a big speed boost.
