### Post by Cypher on 2009-04-01
What kind of a network do you already have? If you have Windows machines in your network that you want to share with, you'll want to use SAMBA to share folders with those Windows machines and access their exported directories.

If you are using all Linux machines, then you can native NFS to share folders..but you can't browse NFS folders like you can Samba/Windows shared folders..
