### Post by grahammechanical on 2015-12-26
When we install Ubuntu we are asked to set a user name & password. That user name name is the name we log into Ubuntu with and that password authenticates logging in with that user name.

Once we login to Ubuntu we can do most tasks without needing to authenticate the action. But there are certain tasks that only we are allowed to perform and for those tasks we are asked to authenticate the action. We enter the password that we created when we installed Ubuntu.

We are also asked to authenticate the first time we set up a wireless connection to a WiFi access point. In this case we use the password for that WiFi access point & not our user password.
