### Post by sampathz on 2009-04-02
I am also having same problem, unable to use proxy in pidgin. I set the proxy info for the specific account in pidign, but it didn't work.
But the below works.

> Go to System-Preferences-Network Proxy and set your proxy there, so that all applications uses this system proxy. 

> Bu the bigger question is what if I have two accounts in jabber where one requires proxy and another doesn't require proxy.
