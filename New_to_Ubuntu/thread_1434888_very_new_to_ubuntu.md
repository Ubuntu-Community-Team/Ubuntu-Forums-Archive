### Post by themusicalduck on 2010-03-20
To be honest I'd just experiment a little. Look through the Applications menu and see what is installed already. Click on Ubuntu Software Centre in the Applications menu (or it might be add/remove depending on what version you have) and have a look, or search through what applications are available (e.g. for an image editing program, just search for Image Editing or something smimilar). Click install to install a program. (Or check a checkbox and click Apply if it is like that)

If there's something specifically you'd like to do, then post what it is and we can tell you how to do it.

If you're familiar with Windows, a lot of the Ubuntu interface is quite similar, so it shouldn't be too hard to get used to the slight differences.

Also, you can find any of your old threads when you have logged in, by clicking on your username in the top right next to "Welcome, ", the click on Statistics, and Find all threads by *username*.

Good luck.

### Post by 2hot6ft2 on 2010-03-23
You really don't need to reply to each persons post. As long as you're reading and learning what you both need and want to know is enough. We know it can be overwhelming at first, we have all been there. No one was born knowing everything.

You'll get the hang of things, just take your time and don't be intimidated. You didn't know how to use windows until you learned how and the same will be true for Ubuntu.

To fix things (if you need to) you will often be told to open a terminal and run some commands. While there may be a way to do it thru the GUI (Graphical User Interface) the terminal is usually much faster both to explain and to do. Plus there are many GUI's but the terminal remains the same for each so it's more universal.

Don't be afraid of the terminal. It's both very powerful and very handy. Use copy and paste so you don't have typos when you can, and if someone says to run anything that starts with sudo, gksu or gksudo wait a few minutes and if there is anything malicious, dangerous or risky about the command someone will say something about it. Also when running any of those you will be prompted for your password (the one you used to log in with) when you type it in it wont be shown (security in case someone is watching your screen, they wont see it) just type it in and hit Enter.

You'll get used to seeing some of the common commands like:
To update the repositories
```
sudo apt-get update
```
To install one or more applications from the repository **(don't actually run these** there is no application that I know of with the name of applications-name, **they are just examples)**
```
sudo apt-get install applications-name applications-name
```
To un-install one or more applications
```
sudo apt-get remove applications-name applications-name
```
To un-install an application along with all of its configuration files purge is used like this
```
sudo apt-get purge applications-name
```
To remove packages that are no longer needed (installed by Synaptic, apt-get or updates, but the original package remains on the computer taking up space)
```
sudo apt-get autoremove
```

Many if not all applications have manuals info or help files which you can access in a terminal like this:
To open the Applications manual
```
man applications-name
```
Info about the Application
```
info applications-name
```
Help file for the Application
```
applications-name -h
```
or
```
applications-name --help
```
To exit the manual, info or help file just hit the letter "Q" on the keyboard the case doesn't matter.

If you have any other questions or need help with something just ask. Someone will help. It may take a little while sometimes depending on the topic but as long as you use a thread topic that gives some clue as to the question or issue it will be answered 99.9% of the time.
