Hello,

I have used both KDE and Gnome. In Gnome, when I add a user, I can tick off boxes for what I want the user to be able to do - administer the system, access this, that, and the other, etc.

But in KUser, there seems to be no such option. Everything is done by adding the user to certain groups. I don't know what group(s) to make a new user a member of if I want him to be able to sudo, use 3D acceleration, connect to wireless networks, etc.

To be more precise, I want to add a user that would be equal to having every box ticked off in Gnome's user manager. What groups does this user need to be a part of?

### Post by anjilslaire on 2009-04-08
look at your own regular user account in KUser, and replicate that user's groups as appropriate, meaning admin, 3D, networking, etc.
