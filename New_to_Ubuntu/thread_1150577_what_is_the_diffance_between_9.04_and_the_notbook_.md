### Post by blazemore on 2009-05-06
The advantages are that a lot of the window management features have been condensed into one panel at the top, which is better suited to small screens.
There is also a quicm point and click interface with large icons, which is easy to use on a small screen.

There's no disadvantages, as it's almost exactly the same system underneath. If you're running a desktop or regular-sized laptop, there's no need to run the netbook remix, unless you particularly like the interface.

There's a wealth of information about the interface, and how you can add it to a regular installation at the Ubuntu Wiki: [https://wiki.ubuntu.com/UNR](https://wiki.ubuntu.com/UNR)
