You can install nautilus-actions and add an action to start pyrenamer on a right-click of one or more files.

However, I am not sure if it will work the way you like. The problem seems to be, that pyrenamer only accepts a directory as a starting point, so you would not be able to select files and send just those to pyrenamer.

You may be better off using a different renaming tool. On the nautilus-actions page you can find some actions that are ready to use for some of the other renaming tools.
