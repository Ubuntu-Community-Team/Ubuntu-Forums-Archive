### Post by jwbrase on 2010-03-02
The answer is a qualified yes.

*.exe is a DOS/Windows executable format, so it doesn't work straight out of the box on Linux. However, you can get many (but not all) DOS and Windows programs to work on Linux, Windows programs with Wine, DOS programs with programs like DOSBox.

### Post by bumanie on 2010-03-02
What the two above have said is correct. Being new to linux, it is important to remember that linux is not windows - linux uses a different files system and has different file extensions for native programs. A suggestion would be to look [here]("http://wiki.linuxquestions.org/wiki/Linux_software_equivalent_to_Windows_software") and [here]("http://www.linuxalt.com/") for alternative 'equivalent' programs that run natively in linux. If you really need/want windows programs, consider a dual boot system or consider running windows in virtualbox. WINE does however run a number of .exe programs very well, but most (apart from games) have great open source alternatives.

### Post by mikechant on 2010-03-02
It's always a good idea to ask if the are good Linux equivalent programs before running windows .exe programs with Wine. Although the Wine people make heroic efforts, it's such a difficult task getting windows programs fully working under Linux that the results are rather hit-and-miss. 

Native Linux programs are generally available through the repository system and so they will be automatically security patched etc., where as you will have to update windows programs manually and new versions of them may not work with Wine at all even if the old version works well.

If you've got enough RAM etc. you can always run windows in a Virtual Machine under Linux instead of using Wine, and this will guarantee that most windows applications will run perfectly.

However, don't let me put you off trying Wine, I use it sucessfully for a couple of odd programs (with no exact Linux equivalents) myself.
