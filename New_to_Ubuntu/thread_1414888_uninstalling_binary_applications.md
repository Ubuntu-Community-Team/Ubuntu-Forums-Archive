### Post by dvlchd3 on 2010-02-24
While in the directory with the make file (where you executed make && make install)
```

make clean

```

This may vary depending on the software, but seems to be pretty standard and consistent through most applications I compile from source.

### Post by mcduck on 2010-02-24
..and, for future reference, you may want to use Checkinstall when installing anything from source code. It creates a .deb package from the program and then installs it through your package management, which makes the program very easy to uninstall.

So, install the checkinstall package, and then instead of doing

```
./configure
make
sudo make install
```

do this:
```
./configure
make
sudo checkinstall
```

..and your program can be removed with apt-get/Synaptic just like any other package.
