### Post by RichardLinx on 2009-10-15
The two best choices are konversation and xchat. (IMO). I personally use konversation (even in GNOME environment) because it's a little more capable than xchat (more options). But xchat is fantastic too.
```
sudo apt-get install konversation
```
```
sudo apt-get install xchat
```

Give them both a try and decide for your self!
