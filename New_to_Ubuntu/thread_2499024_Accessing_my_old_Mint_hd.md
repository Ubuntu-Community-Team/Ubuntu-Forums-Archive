### Post by yancek on 2024-07-08
> How can I mount a hd that already has data on it - with the intention of accessing the data? 

I don't know what you've been reading but the process is the same in either case.  If it is a new drive, it is not likely formatted with a Linux FS so you won't want that step if you already have data on a partition. In the case where there is no data, you just won't see anything.  You create a mount point or use a current directory (the /mnt directory is often used) or /media then you mount the partition as root (use sudo) by device name.  The parted command doesn't show device names but the fdisk -l command will list them.  So guessing from your output, the 480GB drive is your old Mint.  I see you have a bios_boot as well as an EFI partition on that drive which you don't need, generally only use one or the other.  I also see that you are using LVM so if that is the old Mint drive, you need to post that information here so someone can explain how to access and LVM partition.  I don't use LVM myself so you might do an online search while you are waiting for a response.
