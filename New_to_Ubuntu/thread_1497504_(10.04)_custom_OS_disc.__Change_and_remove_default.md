### Post by dv3500ea on 2010-05-30
To change the available wallpapers press alt-f2 and enter
```
gksudo nautilus /usr/share/backgrounds
```
To edit the examples folder do the same but eith
```
gksudo nautilus /etc/skel/Examples
```
To edit default user settings
```
gksudo nautilus /etc/skel
```
