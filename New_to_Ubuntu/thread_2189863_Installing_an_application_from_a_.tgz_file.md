I know nothing of the game you downloaded, but in general either within the extracted files, or on the games web page should be instructions on how to install it.
In the extract files it might show up as readme, or a simple text file like install.txt.

### Post by heir4c on 2013-11-24
If you want to install that downloaded tgz file you can open your filemanager and a terminal.
You drag the tgz file into the terminal and click Enter. 
Now it start the installation.
