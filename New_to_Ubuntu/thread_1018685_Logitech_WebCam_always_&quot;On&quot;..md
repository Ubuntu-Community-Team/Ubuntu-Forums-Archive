### Post by kostkon on 2008-12-22
If you want to know when your camera is working then install the nice [*Camera Monitor*]("http://www.infinicode.org/code/cameramonitor/") utility. It will put an icon in the taskbar informing you when the camera is working. You can install it using *Synaptic*.
