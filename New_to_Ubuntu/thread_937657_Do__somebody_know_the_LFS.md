### Post by Vivaldi Gloria on 2008-10-04
Some people prefer making their own cars from scratch because:
1. They enjoy it.
2. They learn a lot building it.

LFS is similar. You build & configure everything yourself. If you're new to linux then I suggest you don't try LFS. Just get used to ubuntu. If you insist then try [[COLOR="Sienna"]arch[/COLOR]]("http://www.archlinux.org/") which is LFS based but more usable and popular. There are a lot of packages for arch. See the installation guides in the arch wiki.
