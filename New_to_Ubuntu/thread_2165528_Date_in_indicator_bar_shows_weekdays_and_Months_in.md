### Post by siddharth007 on 2013-08-06
Hi.It seems like you need to set English language as system-wide setting.To do so,click on 'System Settings' option on the Unity Dash and click on 'Language Support' icon.You can select your preferred language there.I assume you are using Ubuntu 12.04

So here is how i solved it:
