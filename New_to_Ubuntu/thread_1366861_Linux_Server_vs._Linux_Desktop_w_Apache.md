### Post by lykwydchykyn on 2009-12-28
In the case of Ubuntu, the difference is mainly the selection of preinstalled packages, and the configuration of the kernel.  The server kernel is configured with options more appropriate for server hardware and applications.  And the selection of packages is stripped down on the server version of Ubuntu, for example there is no desktop or GUI.

But in terms of "Linux" there is just one kernel, though it can be tweaked in various ways as I mentioned.

I assume other distros do much the same thing, though some (debian, for instance) don't differentiate between server and desktop versions.
