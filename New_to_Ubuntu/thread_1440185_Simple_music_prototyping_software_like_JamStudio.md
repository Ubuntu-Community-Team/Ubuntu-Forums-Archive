### Post by b33tfr33kr on 2010-03-27
Ubuntu Studio includes several programs that may suit your needs.  LMMS the Linux MultiMedia Studio claims to be "easy music production for everyone", Ardour is a multi-track studio package, there are drum machines, loopers, BEAST is goot for music composition and modular synthesis, midi synthesizers, meters...  i guess my short answer would be to get ubuntu studio.  you can find it in the software center (or synaptic package manager). the trick is figuring out the JACK audio connection kit. JACK allows you to reroute audio in/out and midi in/out from one program to another in any combination.  you can use one program for live looping, and feed it through an effects program while leaving your drum playback from yet another program effect-free, for example.
