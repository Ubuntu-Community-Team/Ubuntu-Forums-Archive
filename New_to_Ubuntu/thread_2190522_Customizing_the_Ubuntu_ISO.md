### Post by deadflowr on 2013-11-28
> **rackit said:**
> Here is the official documentation: [http://is.gd/Bod7kH](http://is.gd/Bod7kH)

TinyURLs are frowned upon here, because people don't know if what they're opening is okay or not.
Here's the link for those people
[https://help.ubuntu.com/community/LiveCDCustomization](https://help.ubuntu.com/community/LiveCDCustomization)

In general that guide is pretty good, more or less.
Quick note though, aptitude isn't installed by default.
But genisoimage is.
And if you decide to update the kernel, take extra care on what you do.

Also note that you need to run on the arch type that the iso is.
So if 32-bit, it must be done on a 32-bit system.
same for 64-bit.
