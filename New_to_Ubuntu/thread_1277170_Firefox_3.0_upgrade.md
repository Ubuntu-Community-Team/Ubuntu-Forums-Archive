### Post by 3rdalbum on 2009-09-28
> **shankhs said:**
> I have manually upgraded my FF to 3.5.But the update manager keeps popping out to install FF 3.0 and this is very annoying.
Please suggest what to do?
Should I degrade to 3.0?
If not then whats the method to disallow update manager to pop anytime it wants?

If you manually installed Firefox 3.5, then it should be side-by-side with Firefox 3.0. If this is the case (you can launch either 3 or 3.5) then you can safely upgrade 3.0 without losing access to 3.5.

If your Firefox 3.5 has overwritten 3.0, then you should use Synaptic Package Manager to "lock version" on the firefox-3.0 package. This will stop Firefox 3 from being updated.
