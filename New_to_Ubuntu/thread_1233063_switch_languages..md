### Post by mcduck on 2009-08-06
I don't think there's any easy way to change language for the whole desktop while you are logged in (you can of course change the input language, as in your keyboard layout). It's possible to define language when starting a program from command-line but that doesn't really count as "easy", and only applies to that program and not the whole desktop.

The normal way to change language is to select it at the login screen.

### Post by Bölvaður on 2009-08-06
> **eric489 said:**
> Hope developers could use xp's simple language switching  button.

There is no language button in xp :-k you must be talking about something completely different.
Perhaps you are talking about switching keyboard layouts. How it is done in gnome is much better than in xp if I remember correctly.
Simple way to do what I think you want:

Right click on the panel
Click +Add to Panel..
Find Keyboard Inicator

And from there you can add as many layouts you want and switch between them with any button you want... my favourite is the right control button. You can also have gnome follow which layout for which window to a certain extend.. but using a key to switch manually is better imo.
