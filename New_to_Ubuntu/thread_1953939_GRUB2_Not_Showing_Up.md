### Post by ksprasad on 2012-04-07
Hi,

Did you do any changes before this issue?

Boot with ubuntu live cd. 

mount the partition onto /mnt

sudo mount /dev/sda5 /mnt

Run this command:

sudo grub-install
If the above command gives error try this:
sudo grub-install --root-directory=/mnt /dev/sda

unmount:
sudo umount /mnt

reboot

Regards,
Ksprasad
