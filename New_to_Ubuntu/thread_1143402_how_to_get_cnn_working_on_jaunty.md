### Post by secular on 2009-04-29
I just did a new install of Jaunty, and one of the first things I did was install ***Ubuntu restricted Extras***. After doing this, I can see videos on Hulu, CNN, etc. 

If you haven't done this yet, click on ***Applications, Add/Remove***. From there, click on ***Show, All Available Applications*** and search for ***restricted***. When you do this, you'll see ***Ubuntu Restricted Extras***. Just select it and install it.
