### Post by superbenny on 2008-06-04
I use Songbird. The bugs are managable, and it has the ability to import iTunes libraries from other partitions. Amarok is amazing, but I found that it leaked a lot, and caused extreme memory over usage, although I haven't heard much about other people having this issue, so I would give it a go. If you want to stick with an iTunes-like interface, Songbird is your best bet, followed by Banshee. I also played around with xmms for a while...its a Winamp-like media player. I prefer Songbird, only because I like the interface. So my reccomendations, in order:
Songbird
Amarok
Banshee
xmms
Also, to get rid of iTunes you can always just delete the folder in the pseudo-C:\ Drive. It wont get rid of the registry entries, but you would probably get the same results in Windoze anyway :)
