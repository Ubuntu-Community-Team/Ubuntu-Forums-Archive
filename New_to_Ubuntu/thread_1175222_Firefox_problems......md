The terminal is the command line interface found in any version of Linux, in Windows this is also known as the command prompt or cmd.exe if that helps - the terminal in Ubuntu is an awesomely amazing tool that you can do so much with and you should do yourself a favour and get to know it ;)
 
It can be found in Gnome under Applications/Accesories/Terminal
 
You can use it to install software using the apt-get comman from the repositires (instead of using Synaptic) you can use it to install a .deb package with the command dpkg -i - as I said its a VERY powerful tool and very much a part of Ubuntu.
 
A lot of the tutorials and HOWTO's in both these forums and out on the internet will have instructions written for the command line
