### Post by billgoldberg on 2008-08-31
Don't use isohunt.

It's probably the worst torrent site in the world with a record amount of fake/malware torrents.

If you need to download linux .iso's use [http://thepiratebay.org/](http://thepiratebay.org/) .

Read the comments before downloading them to see if the file is legit!

--

Are the problems related to flash?

Try switching to ALSA in the "system -> preferences -> sound" menu.

You could also try disabling some add-ons. Some can cause crashes.

You can also use the add-on "flash-block". It will prevent flash animations to start by themselfs. You need to allow them to play, so it helps stability and speed.

--

And if you can't get firefox to work properly, I suggest you use the excellent open-source gnome browser calld "Epiphany".

Be sure to install the plugins or extras package from synaptic, it includes things like adblock and mouse gestures.

Opera is also a good browser, but it is not open-source.
