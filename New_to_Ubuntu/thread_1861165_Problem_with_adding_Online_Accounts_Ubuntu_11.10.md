### Post by Mr. Picklesworth on 2011-10-15
For the Online Accounts panel, that's because it only has support for Google accounts right now. (Lots of others, like Twitter and Facebook, require developers to have API keys and &#8216;keep them secret,&#8217; so there are some politics to be done before they start working). For chat accounts, that's a little strange. Try installing telepathy-idle, telepathy-haze, telepathy-gabble and telepathy-butterfly.
