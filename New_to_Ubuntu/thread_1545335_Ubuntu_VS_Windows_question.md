### Post by eriktheblu on 2010-08-03
I find Ubuntu to be easier to use, easier to customize, and less prone to malicious code than Windows. Problems are typically user created, and easily solved.

GIMP is typically used in place of Photoshop, but it may not suit your needs. The alternative software available for Ubuntu in many cases is significantly different (requiring relearning) or just plain inferior to the pro grade stuff you can get for MS/Apple OSs. 

If you absolutely must use Dreamweaver and Photoshop, I would recommend Windows or Mac.

If you're willing to try differently functional programs, Ubuntu is great and doesn't cost much.

### Post by pricetech on 2010-08-03
I find Ubuntu, and Linux in general, to be much more customizable.  I can also do "geek things" better in Ubuntu.  For example, believe it or not, Remote Desktop to a windows workstation works better in Ubuntu than it does in windows and I find the VNC support to be excellent as well.

I also can set up remote access to my home network with a lot less worry about security with SSH.

But a lot of it is personal choice.  Use whatever suits you best.
