### Post by Hospadar on 2008-04-28
I'm not super sure about this, because I don't know all that much about certificates and linux networking in regard to them, but I believe:

You'll have to install the certificate differently for different applications, some applications may not support it (although your network may block the ports those other applications use anyways, so the point might be moot)

At least for firefox, I think you can go to Edit->Preferences->Advanced->Encryption->View Certificates->Import to manually install a new certificate.

Try that and see if that works.  Do they post any other info about connecting to their network?  If they have some how-to page you could post a copy of (even if it's for windows) that would be helpful for us here to figure out what you may need to do.
