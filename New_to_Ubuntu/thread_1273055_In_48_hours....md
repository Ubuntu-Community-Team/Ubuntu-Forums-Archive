### Post by uninventiveheart on 2009-09-22
I've managed to break my Ubuntu Jaunty Jackalope installation at least three times and had to reinstall.
 
I figured I would post it here instead of advanced topics, since I am very much a beginner at Linux, but all the same, I need help.
 
So far, I've come up with the following 'lessons' on my own as a beginner--
 
-- Persistent Mode Linux on a USB Stick needs room to breathe before installing software updates... ran out of room, programs in memory failed, kernel fault... if you're not happy with the default install (shy of three - four programs on a 2GB USB stick) get a bigger stick, or just let it go.
-- Never switch window managers without knowing how to restore X11 in bash (which I don't know how to do yet... and famously enough, "metacity --replace" I've learned is a horrible idea for a beginner to explore.)
-- If you lose your wireless connnection while downloading software packages and the download isn't complete, don't bother installing the packages you've already downloaded until you get the rest first. (Compiz ate Gnome alive, got garbled screen output, couldn't fix.)
 
I'll keep trying, hell, I ruined my first computer many times over (Leading Edge 8088 XT, DOS 3.22, 20MB Hard Drive) before finally learning enough of the operating system not to do dumb things.
