### Post by alexandremrj on 2008-05-19
Hi, 

sorry for this late reply. If you are still watching this thread i can tell you that Swfdec is an opensource Macromedia/flash player.

Because of the closed code in the Flash Player (until recently) not all sites work with the open source alternatives of flash.

You can wait for an update or in synaptic uninstall the open source library and install the adobe one.

All the best
