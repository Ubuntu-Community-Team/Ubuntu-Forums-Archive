### Post by deadflowr on 2012-12-03
I'm not fully familiar with installing from usb, but I think, if it is anything like installing from a disk, you need to remove the usb stick and restart the system.

You can see if anything actually installed on the hard drive by opening up the home folder and looking at the panel on the left side. You should see a listing for your hard drive under devices, click on it to open the file system and look to see if anything is on it. If Ubuntu installed you will see folder names like bin, boot, etc, usr, home, root, var and others.
