### Post by angelsguitar on 2008-09-18
> **Tsukasa Buddha said:**
> Hi, I just installed Ubuntu, and I would really like to be able to practice piano without having to switch back to Windows.

You could install Jack, QJack (a graphical interface for jack), Qsynth and use piano soundfonts with Qsynth.  You'll also need to download the realtime kernel for better latency.  And, of course, a good low latency audio interface for better performance.

> Is there a Session/Garageband type program for Linux?

As far as I know, LMMS.  Not exactly GarageBand but similar in many ways.
