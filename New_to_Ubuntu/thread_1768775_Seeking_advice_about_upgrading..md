### Post by frankbooth on 2011-05-27
Doing an upgrade should work the same as on a "single boot system", yes.

Doing a clean install won't differ much either, just don't be careless when partitioning (Read: *Don't select 'Use entire disk'*).

**Personal advice:**
Do a clean install, it's usually the easiest way to avoid problems.
Don't forget to backup your Home folder if you want to keep any settings.

### Post by grahammechanical on 2011-05-28

I think that it is possible to download a DVD image and upgrade from that but I have never tried it. The DVD download servers were faster I found. I gave up trying to download a CD ISO. Slower than my old dial up modem.

I upgraded to 11.04 without problems but I think that I will do a fresh install for 11.10. It clears out unnecessary configuration files that might be causing conflicts.
