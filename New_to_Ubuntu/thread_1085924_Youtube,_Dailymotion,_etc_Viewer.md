### Post by ubuntu27 on 2009-03-14
I know a software that does precisely that and more!

It is called Miro. It allows you to download videos from youtube in high quality and it also acts as a Channel Guide. It has bit-torrent capabilities as well

Here:
[http://www.getmiro.com/](http://www.getmiro.com/)


It is in the repositories. 

```
sudo apt-get install miro
```

But, I will recommend you to install the new version from their website. The one in the repository is too old.

[http://www.getmiro.com/download/for-ubuntu/](http://www.getmiro.com/download/for-ubuntu/)
