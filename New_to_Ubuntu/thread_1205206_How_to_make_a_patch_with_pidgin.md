### Post by ohzopants on 2009-07-08
In order to apply a patch you will need to download the source code for pidgin and apply the patch, then compile it manually.

From the linked page it appears that the patch has been integrated into version 1.54, so if you download the source code for 1.54 and compile that, the bug should be resolved.
