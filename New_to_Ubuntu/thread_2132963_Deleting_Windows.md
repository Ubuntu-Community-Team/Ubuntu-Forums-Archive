### Post by r-senior on 2013-04-06
You don't need to delete it as such. When you install a Linux distribution you just let it use the whole disk rather than installing alongside Windows. The computer then becomes a Linux-only machine. If you dual boot Windows and Linux, *then* decide to remove Windows, you can reformat the Windows partitions and use them as extra storage space.

It's almost exactly four years ago that I "actually deleted my Windows OS". ;)

Lubuntu is a fine distribution.

### Post by DuckHook on 2013-04-06
Deleting is very easy. It's installing that is usually where you run into problems. Just a matter of deleting the partition(s) on which Windows resides and then running ```
sudo update-grub
```I would not recommend it, though. People often underestimate their need for Windows. It's nice having a dual-boot environment which allows you choice of either.

If you want to get rid of Windows, I recommend simply performing a fresh install of Ubuntu on the computer so that it is the only OS. Just follow the wizard that Ubuntu takes you through when installing and indicate that Ubuntu will be the only OS. 

I only used windows prior to Ubuntu, so moving to ubuntu was a bit of a shock at first. Once I got used to it and found suitable applications to replace everything I was doing on Windows, I axed Windows and haven't had any problems. Just don't be too quick to axe Windows. I knew I was ready when I hadn't booted into Windows for anything in a few months :0
