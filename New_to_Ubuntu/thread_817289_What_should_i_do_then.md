### Post by theDaveTheRave on 2008-06-03
Hi

you seem to have a number of what on the face of it should be reasonably easily resolved issues.

If you want to run KDE as opposed to Gnome then obviously you want KUbuntu (as it is "designed" with KDE in mind). If you are so inclined you could install Ubuntu (with it "default" Gnome desktop) and then add KDE via the respitories, it would then give you the option at login as to which desktop environment you desire.

I did this for a while, experimenting with KDE / Gnome / and Xfce, but now I mostly just drop into Gnome.

From memory (and it has been a while since I used KDE!), the menu bars had a load of options for customisation, I'm surprised that you can't reduce the size without causing problems for your clock?

As for installing pidgin (it isn't something I have ever used), I'm surprised that you can't simply select it in the list of options from the install manager! The same goes for Firefox - although if you want the all new singing dancing versions you may have to install from source. I'm sure that there are instruction on this on the Firefox website, or in these forums.

However, and this may be the crunch, if you are running the system through wubi (again something I have never done!) it may be that there are issues in the underlying Windows system that is causing your grief.

Why not try booting into a live CD and seeing if you can adjust the settings that you are having issues with, if so then I would suspect it is a Wubi issue and not a KUbuntu issue, also you can try the different ubuntu flavours out this way (if you can stand having the wait to download the 3 different forms... ubuntu, KUbuntu and XUbuntu... and then burn them to CD), try each from the live CD and see what you think of them.

have a look at this [[COLOR="Blue"]link[/COLOR]]("http:////ubuntuforums.org/showthread.php?t=814182&highlight=reinstall+KDE") this guy has removed KDE4 and backdated to KDE3, it should help with any issues you have in doing this.

Also it seems that KDE problems are something you are not alone in experiencing, have a look [[COLOR="Blue"][COLOR="Blue"]here[/COLOR][/COLOR]]("http://ubuntuforums.org/showthread.php?t=787492&highlight=install+KDE3"), and you'll see what I mean. It seems that they may endorce my idea of installing Ubuntu(Gnome) and then installing KDE over the top! Instructions  are [[COLOR="Blue"]here[/COLOR]]("http://ubuntuforums.org/showthread.php?t=761557&highlight=remove+kubuntu-desktop+hardy") I think? Remember to install a desktop environment first of course, or you'll be very bothered when you re-start later on!


Hope this helps, good luck in resolving your issues, and maybe give dual booting a try, it really isn't scary.
