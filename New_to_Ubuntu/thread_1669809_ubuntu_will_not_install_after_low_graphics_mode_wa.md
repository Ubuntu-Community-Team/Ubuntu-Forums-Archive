### Post by smurphy_it on 2011-01-18
Put Windows CD in drive, boot PC format, and re-install.  If it fails.  Boot a linux live-cd (if you have one), run gparted and attempt to format the drive.

If it fails, it's usually pointing to a hard drive problem (and in rare cases a memory issue).  I'd suggest getting a hardware diagnostic CD and testing RAM and hard drive at the least.

If all comes back fine, you can always download the low level format software, and low level format your hard drive.
