### Post by hobo89 on 2009-02-24
If it's the same problem I have, press F11 (this full screens it) then press it again (to un-full screen it). The title bar etc. should have reappeared. I also then un-maximise it, resize the window a bit and maximise again. Apparently this prevents it from doing the same thing when it's next started. I'm not sure if it's true, it's just a habit I got into after reading the solution. This isn't a permanent fix though, it still happens occasionally, but not every time you start Fx.

### Post by Therion on 2009-02-24
If you're running Compiz-Fusion go to:

System/Preferences/CompizConfig Settings Manager.
Scroll down to the Utility section.  
Open the Workarounds plugin and UN-check the tic-box for **Legacy Fullscreen Support**.
