### Post by donkyhotay on 2011-12-12
Both of the previous posts are correct, 7zip is CLI (command line interface) only program however it does integrate with many other GUI based archive utilities. Depending on your version of ubuntu it'll probably be called archive manager or file roller. Just try double-clicking on a 7zip file and it'll probably pull everything up automatically.
