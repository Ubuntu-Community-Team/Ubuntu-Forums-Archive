### Post by thunk77 on 2008-09-22
Hello,

All you have to do is go to System - Preferences - Appearance - Interface Tab and uncheck the box "Show Icons In Menus".
Please note that this will disable all icons in your main menu. However the menu you posted as an example seems to have that tweak.

Best Regards,
