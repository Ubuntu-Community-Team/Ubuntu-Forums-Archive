### Post by drs305 on 2008-07-25
This should open it. You can make a launcher or edit your menus to include it:

```
gksu bum
```

Added:

To access it via menu, it is located in System, Adminstration, BootUp-Manager. Once you find it there you can drag the icon to a panel or desktop to create a shortcut.
