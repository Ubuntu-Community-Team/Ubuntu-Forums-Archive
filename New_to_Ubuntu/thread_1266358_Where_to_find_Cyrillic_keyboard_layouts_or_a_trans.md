### Post by Rumtscho on 2009-09-14
I have a German keyboard with a Qwertz layout and I can touch-type on the Qwertz layout. This is sufficient for most languages with a Latin alphabet, but I use Bulgarian, too. Bulgarian is officially written with a Cyrillic alphabet only; writing the Latin transliteration looks ugly and some people cannot read it at all, including my mom with whom I want to chat. (If you want to know how bad it is, imagine writing English with a Greek alphabet) 

I tried installing a Bulgarian layout from System -> Preferences -> Languages, but under Bulgarian I only had the options for Standard layout and Traditional phonetic layout. I cannot type the standard one, it would be like changing from Qwerty to Dvorak. And the phonetic one is intended for people who know the American Qwerty layout, not the German Qwertz layout. I could live with the fact that "Y" and "Z" are switched, but the problem is that every single non-alphanumerical character has a different position under Qwerty. Not only is touch-typing not an option then, when I look at the keyboard I cannot see where all the chars are, because my key labels are Qwertz. 

Is there a place where I can find a layout which maps a German keyboard to Cyrillic characters? It doesn't have to be a "real" system-level layout; a resident application which intercepts keystrokes and inserts Cyrillic chars would be okay too - just like the Transliterator Firefox add-on, but it should function in other applications too, including Skype and maybe Open Office. 

I am running 9.04.

### Post by zvacet on 2009-09-16
From system>admin>language support download Bulgarian locale.Then in system>preferences>keyboard>select by country.
