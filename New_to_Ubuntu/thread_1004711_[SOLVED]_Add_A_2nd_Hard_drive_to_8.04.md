### Post by taurus on 2008-12-07
Edit /etc/fstab

```
gksudo gedit /etc/fstab
```
and add these two lines to the end of it, one for each partition on /dev/sdb.

```

UUID=81b19c36-f439-4cf4-829c-cb7d3aa46b6b   /media/sdb1   ext3   defaults   0   2
UUID=1fff22c8-19d8-4efa-8509-5eb401a8c5f1   /media/sdb2   ext3   defaults   0   2

```
Save the file and exit gedit.  Then, create those two new mount points and mount them.

```
sudo mkdir /media/sdb1 /media/sdb2
sudo mount -a
df -h
```
Now, if you want to change the ownership of both /media/sdb1 & /media/sdb2 from root to your login name, tom, you can run the chown command.

```
sudo chown -R tom:tom /media/sdb1
sudo chown -R tom:tom /media/sdb2
ls -la /media
```
