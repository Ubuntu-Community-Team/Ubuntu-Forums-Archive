### Post by alzie on 2008-05-11
Its #2,  It imports rather than links to files.

[http://www.psychocats.net/ubuntu/mountwindows](http://www.psychocats.net/ubuntu/mountwindows) is a tutorial to allow ubuntu to access your windows.

What I did was to set up a partition (in windows originally) to store my music and pictures on.  Its fat32 which ubuntu can read.  Both windows and ubuntu can access this drive so if I rip music in either OS I can access it from the other.

(I'd bookmark psychocats' site,  there is a wealth of information there)

I hope this helps
