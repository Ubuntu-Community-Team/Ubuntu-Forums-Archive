### Post by bobnutfield on 2008-10-17
Hello, welcome to Ubuntu.

You can run Ubuntu on almost all the newer computers with Intel or AMD processors.  I would probably suggest an Nvidia graphics card over an ATI as users seem to have better luck configurirng them.  If you do not plan on having Windows at all, you can just format the entire hard drive and install Ubuntu only.  The installation process will let you do this.  As a matter of fact it is the default unless you choose another option.  But keep in mind that you can easily set up a dual boot system to have both OS's on your new computer.  

You can upgrade to Intrepid when it is released, but it is being released on October 31, so you can just as easily install the Beta now as it is almost at release candidate status and is pretty stable.  But as a new user, I would recommend using Hardy. It is stable and it is a long term support release.  You will have better luck with that.

You will find plenty of help here when you need it

Good Luck!.

### Post by cardinals_fan on 2008-10-17
Do not buy a Vaio!  This is both because of troublesome Linux support and countless complaints I've heard about Sony's quality (or lack thereof).

EDIT: If you are sure that you're done with Windows, then buy a machine with Ubuntu preinstalled (or build one yourself).  However, if you have really never tried Ubuntu before, buy a machine with Windows that also has decent Linux support.  Why?  Simply put, Linux is free and Windows isn't.  Buying Windows through an OEM costs maybe $50; buying it yourself costs maybe $150.  Also, I have always found Ubuntu/Linux-in-general easier to deal with than Windows, so I'd rather fix hardware problems with Ubuntu than with Windows.   Visit [this site]("http://www.ubuntuhcl.org/") for more tips on compatability.

I get about 700fps in glxgears with the fglrx driver.
