### Post by leonardo_neo on 2009-03-22
As a matter of fact I started using Linux first from Mint 18 months ago. It was some version called 'Cassandra'.

I learned that it was Ubuntu based, and uses the Ubuntu kernel. They tried some new applications on this kernel. They also give you ready made pre-installed media codecs (which of course you can install in Ubuntu as well from medibuntu).

At that time I was experiencing too much difficulty with Mint, because it had some update issues. There were some updates for Mint, some for Ubuntu kernel and because of that some conflict. So I was tired of Mint distro and then I switched to Ubuntu. Since then never had any update trouble. 

So my personal opinion is that you should use Ubuntu. Mint fans can differ from my views and I respect that.
