### Post by Temposs on 2009-02-17
hi twitch

Why don't you try out Avant Window Navigator. It will create a bottom panel sort of like the one in Mac OSX, but more customizable, I think. You can make it look really pretty, and it's definitely different from what Windows offers. 

If you're running Ubuntu 8.10, it's in the repositories, so you can go to Applications->Add/Remove and search the keyword "avant". It will take the place of your bottom panel. 

If you're using a version of Ubuntu before 8.10, you'll need to get Avant manually. Here's the website:

[http://wiki.awn-project.org/](http://wiki.awn-project.org/)

If you want flashier window decorations you could replace the default window decorator with Emerald. Just install the package emerald through the package manager. To switch enter the command:
```

emerald --replace

```
then go to Gnome-look.org and find yourself a nice Emerald theme to use.
