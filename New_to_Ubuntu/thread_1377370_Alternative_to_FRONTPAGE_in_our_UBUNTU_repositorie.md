### Post by Paqman on 2010-01-10
Try Kompozer, it's a lot better than Frontpage. It's sort of similar to Dreamweaver if you've ever used that.

You might find that sites made with Frontpage don't work properly with other HTML editors. That's because Frontpage writes terrible HTML that doesn't comply with the web standards. To be honest, the best thing to do with a site made with Frontpage is to scrap the whole thing and start again using nice clean standards-compliant code. Frontpage code is almost impossible to maintain.
