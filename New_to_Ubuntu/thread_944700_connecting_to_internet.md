### Post by PhoenixP3K on 2008-10-11
Bottom to top. 
More and more peripherals are supported are being supported under Ubuntu, and the upcoming 8.10 (October 31, 2008) will be no exception. 
What is suggest would be to install using the Live CD, you put the disc in and run the Live CD on your machine. From there, attempt to connect to your local network; it's it's a wireless you'll be able to pick it and input the code to access it.

This way you know your hardware is supported, then you start the install from the Live CD, it will do it's thing and at one point should check online if updates are available. This way you'll have a fresh and updated install.
