### Post by Mark Phelps on 2011-10-03
From your initial description and the line in the script output that references wubildr, it's apparent that you originally installed using Wubi -- which does NOT use GRUB!

So, installing GRUB not only did not fix your problem, it actually made it worse.

Wubi uses the Windows boot loader to gain initial access and boot Ubuntu.  It looks like you corrupted this when you forced the installation of GRUB.

If you reinstall the Win7 boot loader, you might then be able to boot into Ubuntu.

If you have a Win7 INSTALL  DVD handy, boot into that and run Startup Repair at least three times.

If you don't have that, or a Win7 Repair CD, go to the link below, download and burn the appropriate CD image and boot from that:

[http://neosmart.net/blog/2009/windows-7-system-repair-discs/]("http://neosmart.net/blog/2009/windows-7-system-repair-discs/")
