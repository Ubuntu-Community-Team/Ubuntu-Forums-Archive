### Post by 2F4U on 2012-10-03
If you are looking for Todo lists, that can be used on any other OS, I would suggest to use Emacs with org-mode.

[http://orgmode.org/](http://orgmode.org/)

Basically, these are text files, which can be used on any operating system (unless it can edit text files) with a special kind of syntax. I am using this myself and am very impressed by the functionality. Of course, it takes some time to get used to it, but it pays off.
