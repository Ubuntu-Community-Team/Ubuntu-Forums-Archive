### Post by uRock on 2012-01-21
I use Handbrake for ripping DVDs to my home server. To install Handbrake, copy and paste the following commands into a terminal,```
sudo add-apt-repository ppa:stebbins/handbrake-snapshots
```^^^Adds Handbrake's repo to your system's Software Sources, so that it can be installed and updates run.```
sudo apt-get update && sudo apt-get install handbrake-gtk ffmpeg
```^^^Will update packages within the package manager, which will add the packages that are in the newly added software source, then install Handbrake and ffmpeg, which is great for changing the file format when wanted/needed.
