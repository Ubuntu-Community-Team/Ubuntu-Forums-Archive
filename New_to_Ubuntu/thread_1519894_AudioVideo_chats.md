### Post by YannBuntu on 2010-07-03
Hi,
as Microsoft changed their MSN protocol in March 2010, it is now impossible to do audio+video calls with this protocol.

If you want to do audio+video calls with a Windows user, I recommand :

- Ekiga ([http://wiki.ekiga.org/index.php/Main_Page](http://wiki.ekiga.org/index.php/Main_Page)), which exists also on Windows
- Empathy with Telepathy PPA, the ubuntu-restricted-codecs and a jabber account, to do video+audio calls with Windows Gmail users
- Skype, which exists also on Windows, but is not open-source
