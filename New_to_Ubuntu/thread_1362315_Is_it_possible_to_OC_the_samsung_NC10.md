### Post by seventeener on 2010-06-10
+1, same question here. For now I have only found how to lock the frequency on the maximum supported 1.6 GHz. Simply add the CPU Frequency Scaling applet to the panel. By default Ubuntu uses the on-demand profile which means 800 MHz and speed up when needed. But I can say that with the 1.6 always on, Gnome feels snappier, as you don't waste those milliseconds while the cpufreq-selector daemon realizes it's time to hurry.
