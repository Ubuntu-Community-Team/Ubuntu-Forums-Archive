### Post by dizee on 2008-05-29
By default the fire plugin is Shift+Super(windows key)+left-click drag. It has to be enabled first though, install compizconfig-settings-manager and open "Advanced Desktop Effects Settings" from your menu, go to "Paint Fire on the Screen" under Effects and make sure the box is ticked.

To change the close window animation to burning, go to "Animations" in the settings manager. Edit the first Close Effect entry so that it reads "Burn" instead of the default "Glide 2".
