### Post by kroq-gar78 on 2011-02-27
from [http://forums.dropbox.com/topic.php?id=26301](http://forums.dropbox.com/topic.php?id=26301)

run in a Terminal (Applications -> Accessories -> Terminal)
```
sudo apt-get --purge remove nautilus-dropbox
```
