### Post by ajgreeny on 2012-09-13
It will depend on the actual PDF file whether or not you can select specific blocks of text, as some PDFs are simply images, and don't have embedded text.

If the PDF will allow selection of text, you can do it the same way as in any other document by using the mouse and dragging across the required text block.  If it is an image you will not be able to select text with evince, though other PDF display applications may let you do so;  give some other applications a  try.

### Post by Zill on 2012-09-13
czgirb:  AFAIK highlighting of text is not possible with Evince.

However, highlighting of text *is* possible with [PDFedit]("http://pdfedit.cz/en/user_doc.html#gui_onclickFunctions_highlightingText").  Depending on the pdf file you are editing, it may be necessary to use PDFedit to "delinearize" the file first.

It's in the [repos]("http://packages.ubuntu.com/lucid/pdfedit").
