### Post by ubudog on 2011-07-21
Hmm, that's a hard one.  There really aren't any good alternatives for Linux (yet).  However, you may want to try Blender, or maybe Animata:
[http://animata.kibu.hu/](http://animata.kibu.hu/)

If you really want to, you could use Adobe Flash Professional in Wine, which allows you to run (some) Windows programs inside Linux.  Also, you could install a virtual machine, install Windows on it, then run AFP on it.  

Hope that helps.  :-)
