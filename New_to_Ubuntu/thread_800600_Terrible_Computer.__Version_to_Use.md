### Post by jtibau on 2008-05-20
Do you really want to use it as a desktop??? I say this because I administer (sortof) a small webserver with similar specs... It's been working non-stop for at least a year, I think that's actually some sort of miracle lol...

My point is that you might want to consider using it as a local file server, web server, "anything" server... It might do you much more good, than try to use it as a desktop computer.

If, however, you need a desktop, I would recommend you install the server edition, and then install a very light window manager... This is certainly a little harder than installing xubuntu :S.
Another option could be to try fluxbuntu: [http://fluxbuntu.org/js.html](http://fluxbuntu.org/js.html)
It's based on ubuntu of course, but it's not an official derivative, I think. Regardless, it is fast, and resource efficient. Plus, you practically have everything you will need out of the box.

### Post by ugm6hr on 2008-05-21
Xubuntu 7.10 will install from the AlternateCD, and will likely run acceptably (albeit a bit slow).  I have just given away a K6 128MB computer with Xubuntu 7.10 - and it surfed web etc OK.

Did you install Xubuntu 8.04 with the AlternateCD?  I suspect that it should work with 128MB RAM, since it seems that is is lighter than 7.10.  If you didn't - I would try that first.

If that doesn't work, and you want a machine just to experiment with, perhaps Ubuntulite might be an option ([http://ubuntulite.tuxfamily.org/](http://ubuntulite.tuxfamily.org/)).  The 8.04 derivative is in testing - so hopefully should be easily installable soon.  As mentioned, a "minimal installation" would be a good option, but is not something for a beginner to be trying.

Be aware that from a usability / graphical impressiveness viewpoint, these "low end" versions will be less than impressive.
