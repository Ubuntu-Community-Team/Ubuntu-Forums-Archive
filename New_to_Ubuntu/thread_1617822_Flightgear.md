### Post by lkraemer on 2010-11-10
You could try starting the program from a Terminal Window (Console)
Once you have these environment variables set up, simply start FlightGear by running fgfs --option1 --option2&#8230;. Command-line options are described in Chapter 3.5. 

You could try starting the program from a Terminal Window (Console)
```

fgfs --option1 --option2

```

REF:
[http://www.flightgear.org/Docs/getstart/getstartch3.html](http://www.flightgear.org/Docs/getstart/getstartch3.html)
