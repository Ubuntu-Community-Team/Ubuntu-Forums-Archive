### Post by nicedude on 2008-08-22
Yes the other guy is 100% right as the disclaimer about you needing windows can mean you need windows to flash the BIOS with the latest version in order to get the newest features which usually means more of the newer CPU's will be supported. So I would recommend before you pick a CPU you might want to go to foxcon website or google it and see what CPU's are supported by what BIOS yours came with and then at least you wil know what you will have to do if you want a certain CPU. Worst case though is if you buy a certain CPU and it requires a BIOS update to support the CPU you buy then you will need a working CPU ( that is supported by the default stock BIOS ) to plug in along with RAM and at least floppy or CD drive used with a boot disk to flash the BIOS to a newer version at which point you could then reboot, turn-off & then remove that older CPU and now install your new CPU. But that is allot of work and takes an older CPU laying around of the same soclet type, it might be easier to see what is supported first by the stock BIOS ( unless if to you PC tech stuff is easy and you have the extra CPU ect ) and just get the fastest CPU it supports with stock BIOS ( like if you don't want any hassles with updating the BIOS and just want it to work easily.


Good Luck with your new PC
