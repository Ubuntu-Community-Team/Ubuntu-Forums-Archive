### Post by starcannon on 2008-11-13
Press and hold the Alt key + Mouse drag on the window that you can't see entirely, this will allow you to drag the window around and move the off screen parts onscreen.

If this started out as an upgrade, or if it started out with /home on its own partition, either way I have found that I need to delete all the user specific gnome config files from /home/username/
A list of the folders that should be deleted (don't worry gnome will make new default ones):

[LIST]
[*].config
[*].gconf
[*].gconfd
[*].gnome2
[*].gnome2_private
[*].metacity
[*].nautilus
[/LIST]
You should also be sure to run this command in terminal:
Alt+F2:gnome-terminal
```
gconftool-2 --recursive-unset /apps/panel
```
reboot
That should solve you if it was just some previous gnome settings causing issues.

If you can not use alt+f2 to get a terminal you can always ctrl+alt+f1 to get to a full command line interface, then simply use the rm -r command to remove the directories and sub directories and files from your /home/username folder.

GL and have fun
