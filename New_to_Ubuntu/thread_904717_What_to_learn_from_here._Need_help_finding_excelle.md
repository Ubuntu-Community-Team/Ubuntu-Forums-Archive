### Post by tuxulin on 2008-08-29
yes i understand your situation
like most i too left windows of years being a bill gates boy with quite a few MCP's and years of hands experience. 

when i moved to linux the mistake i done was try to learn it with the mindset of windows. within a short time i learned that it was impeding my linux learning curve than helping. and so i started to learn again this time with a open mind.

linux is completely different than windows and its structure is so wide there is not a main route to take when learning this GREAT product.
what i did was to learn Gnome as in where was things and how to do things.
next where was my config files and how to configure them. i believe once
you can do this effectively then applying linux to real world situation 
does come more easier since the underlaying technology is more or less the same as windows i.e TCP/IP, DNS, WINS, Web Servers and the likes.

the apparent struggle that new users face is the basics like how to install a program and where is my installed program, how to create a shortcut and more.

once you learn the basic core concept and apply a little time in using Linux you will be hooked and most other situation will become so easy (over time that is) :)

be open minded, give a little time try and read as much as you can, pretty soon it will come together.       

Tuxulin
