When installing or removing software, is there any difference between using terminal and the Software Center?  Is one more efficient or thorough than the other?

---

### Post by Zemblan on 2010-05-28
For basic operations: installing and removing software, browsing the repos etc, the graphical frontend can be more convenient in some ways; certainly a bit easier on the eye. However it is just a frontend to the command-line package management tools. 
For more complex and flexible package management using those tools directly gives a lot more control over what's happening, allows options that aren't available in the gui, and gives more detailed feedback, but it's more complicated and has a steeper learning curve. 
Which you use depends on personal choice and what you are trying to achieve.
