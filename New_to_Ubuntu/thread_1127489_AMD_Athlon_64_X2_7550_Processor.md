### Post by sdennie on 2009-04-20
As said above, I'm sure it will work fine.  The installation of 32 or 64 bit is another matter.  If you have less than 4G of memory, you can install either easily.  But, you will find more opinions than facts by asking the question.  Some will claim 32bit has better support and others will claim 64bit is faster.  Both are probably right.  If it's something you are really worried about, install one or the other and be certain to create a separate /home partition.  After using one for a while, switch to the other.  You've lost no personal data and can compare the two.
