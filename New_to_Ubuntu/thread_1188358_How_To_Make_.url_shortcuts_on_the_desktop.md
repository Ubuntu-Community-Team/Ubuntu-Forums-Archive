### Post by dnairb on 2009-06-15
You can make shortcuts on the desktop that point to webpages:

Right-click on the desktop and select "Create Launcher"
Give it a suitable name
In the Command field, type **firefox [http://www.ubuntuforums.org](http://www.ubuntuforums.org)** (for example)
Click OK
