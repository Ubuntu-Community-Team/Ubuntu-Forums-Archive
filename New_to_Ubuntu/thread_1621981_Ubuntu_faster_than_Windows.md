### Post by lykwydchykyn on 2010-11-14
This is pretty commonly observed by people comparing the two, but I think it depends on a few things; for example, which version of Windows?  What hardware (poor graphics drivers for some hardware gives a perception of slowness)?  How is Ubuntu configured?  What features are enabled/disabled on each OS?

But comparing the default installs of the latest Windows vs the latest ubuntu, I don't think too many people around here would disagree with the statement that ubuntu is faster.
