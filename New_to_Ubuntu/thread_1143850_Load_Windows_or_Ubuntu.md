### Post by Niniel on 2009-04-30
Yes, just make sure to install the Grub boot manager.
Also, you can change the default OS it loads. There's a neat program called KGrubEditor (you can get it from System/Administration/Synaptic Package Manager) that offers a GUI to administer Grub. Very neat. It needs some KDE libraries, so don't be surprised if a few packages have to be installed just for this one program.
