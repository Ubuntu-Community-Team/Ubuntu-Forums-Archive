### Post by arvevans on 2009-12-19
Seems that there may be a misunderstanding regarding what Ubuntu is and how it relates to Linux in general.  In Linux, most applications are written to run on all versions of Linux.  An application being Ubuntu-specific would probably be ridiculed by the Linux community and destined for the trash bin.  

Open-Source software (most Linux software is Open-Source licensed), is available in source code format as well as in executable binaries.  The Open-Source application code is readily available and can be compiled to run on almost any OS, including most all variations and derivatives of Linux and UNIX.

Some searches on various Linux software repositories could help find a great number of "recording" hits, but it does depend on what you want to record, and how you want to do it, that will determine which application is right for you.
