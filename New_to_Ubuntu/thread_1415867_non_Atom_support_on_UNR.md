It's all a matter of how you use the notebook in question. I installed UNR on my core2 11.6" notebook (1366x768) and prefer it to "plain" Ubuntu. On my desktop, with a 22" monitor, I'm running the plain version, of course ;)

The questions you have to ask yourself are the following:

- Will you regularly use unmaximized windows? (UNR automatically maximizes (almost) all windows, although you can UNmaximize them).
- Do you want to place icons all over your desktop? (You can still do it in UNR, but you're forced to be a bit more tidy).
- Do you need the extra vertical space UNR provides by 1) having only one desktop panel (along the top by default) and 2) having window titles (and associated bars) displayed in the top panel?

For me, the answer was obvious, but I did tweak one vital (for me) thing: I increased the number of virtual desktops (workspaces) -- and set UNR to only show icons of windows in the current workspace (as opposed to the default of showing them all). To me, this gave me the best of both worlds (plain Ubuntu and UNR).

I'm unsure if there are any power usage tweaks in UNR that are not present in regular Ubuntu... I haven't been able to find a comprehensive "differences" list...

### Post by snowpine on 2010-02-25
Ubuntu and Netbook Remix have different user interfaces (Gnome desktop vs. netbook launcher) but they are the same core system underneath. All of the Atom optimizations in UNR are also present in regular Ubuntu. UNR does not require an Atom and should run fine on your Centrino (so long as it meets the minimum hardware requirements).
