### Post by LowSky on 2009-05-12
Hi welcome to the Forums

Regretablly windows.exe never ever had that power. Windows.exe (is the command to launch Windows from DOS, when you could still do that, circa 1992.

You seem to be confusing you computer lingo and thats OK but can confise popele so here a quick lesson, a site is commonly shorthand for a website, files or folders are stored locally on the computer and the correct name for asking where to find something, ie where is such file.. For that you use the word program and that program is nautilus. 
If you want to see your files and folders use my documents which is under the top panel bar called Places, or if you want to use the command line, open a termnial (under Applications>Accessories> Terminal) and type
```

nautilus
```
