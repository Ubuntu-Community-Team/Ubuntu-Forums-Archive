### Post by 3rdalbum on 2009-08-20
Printers is so easy you've probably already worked it out - go to System > Administration > Printers, create a New Printer and it should be detected and set up.

On Ubuntu you don't "map" network drives, but you do "mount" them. Gnome is capable of doing this; go to Places > Network and double-click the drive when it appears.

It will mount into the hidden folder ".gvfs" in your home directory, so you might want to add ".gvfs" into your bookmarks so you can get at it from within programs.
