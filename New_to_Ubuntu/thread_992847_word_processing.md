### Post by AndyCooll on 2008-11-25
The default for Ubuntu is OpenOffice Writer as mentioned in the above post, and of the three it is the most complete feature-wise. In fact OpenOffice is a full suite of applications containing spreadsheet, presentation, database apps etc too. And there is a version for Windows too.

OpenOffice has extensive multi-language support. It contains a number of language versions as part of the default install, and as part of the default install it is likely to have installed your native language. If there are other language packs you want to add you can try Synaptic Package Manager (System > Administration > Synaptic Package Manager) do a search for Open Office and the language you wish to add (or just do a search for Open Office and browse the list), 

Abiword is the GNOME desktop environments word processing application, and KWord is the KDE equivalent. Both of these are lighter applications and fine for most basic aspects of word processing.
