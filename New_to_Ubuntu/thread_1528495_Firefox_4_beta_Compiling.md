### Post by lovinglinux on 2010-07-11
Just because a program is distributed as tar.gz, it doesn't mean it's source code that needs to be compiled. Firefox falls into this category. All you need to do is to extract it and click the Firefox file inside it to run. Mozilla also distribute Firefox source code as tar.gz, but those have the "source" word in the file name.

You can also use my extension [FoxTester](https://addons.mozilla.org/en-US/firefox/addon/141505/). It allows to install any version from the context menu, by simply downloading them to a common folder. [See video demo here](http://foxtester-extension.blogspot.com/2010/05/foxtester-102-released.html).

Also see the [[COLOR="Sienna"]**Installing Other Versions**[/COLOR]](http://firefox-tutorials.blogspot.com/2010/05/installing-other-versions.html) tutorial.
