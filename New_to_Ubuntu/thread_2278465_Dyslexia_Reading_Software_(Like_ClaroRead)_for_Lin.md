### Post by serafini-c on 2015-05-16
I have re-used an old mini Laptop (Acer Aspire One) and installed Lubuntu onto the system. I am using this notebook for work, as it easy to transport to customers sites. I only use the Terminal, Wireshark, KeePass, Thunderbird, Libre Office and Teamviewer.  

 

 This system works great for my basic needs but there is one peace of software that I am really missing within Windows compared to Linux. I was diagnosed with dyslexia and because of this the United Kingdom (UK) government provided funding for software to help with my learning. I was given ClaroRead which is not available for Linux.  

 

 ClaroRead has many features but the main features I used was “Reading Microsoft Word, Adobe Reader, OpenOffice/LibreOffice and Internet Explorer with speech and highlighting so you can follow the text as it is read out by [h]("http://www.clarosoftware.com/voices")[igh-quality human-sounding speech.]("http://www.clarosoftware.com/voices") Reads text in email, webpages, and any other program you want by selecting with the mouse, pointing with the mouse, or capturing from the screen.”
 

 I have found software like ClaroRead is essential for me to review important emails and documents before they are given to clients. I like ClaroRead so much I tried to get the software working with Wine HQ but I could not got it working.  
 

 Is their an Open Source Software equivalent to ClaroRead that works within Linux (Lubuntu)?  I would be interested in supporting an open source project that provides this service and could really help children with learning difficulties. 

 

 Also I got a few teacher friends that are interested in Open Source Reading Software for all the operating systems for the children.

ClaroRead URL: [http://www.clarosoftware.com/claroread](http://www.clarosoftware.com/claroread) 

ClaroRead Overview Video: [https://www.youtube.com/watch?v=7gFy7o9vqMM&list=PL1A31CA479D1F8699](https://www.youtube.com/watch?v=7gFy7o9vqMM&list=PL1A31CA479D1F8699)
