### Post by sojakob on 2011-01-11
Other wise. Since you are running 9.10 you could take a look at [this howto]("http://ubuntuforums.org/showthread.php?t=202605&highlight=windows+share+samba+howto") on setting up Samba. 
By using Samba you can access the Ubuntu machine from the windows machine as well.
In later versions of Ubuntu, the Samba service has been integrated in a very user friendly way.
Good luck.
