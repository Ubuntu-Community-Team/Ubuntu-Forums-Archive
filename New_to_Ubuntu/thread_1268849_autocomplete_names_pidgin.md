### Post by Francewhoa on 2011-04-26
The tab works fine in Pidgin. Ensure you type in the person Nick. Not the person Alias. Sometime they are different.

If unsure then in the chat window right click on the person Alias and select "Info". This will display the Nick.

If above is not working then in Buddy List window remove the Alias and keep only the Nick
