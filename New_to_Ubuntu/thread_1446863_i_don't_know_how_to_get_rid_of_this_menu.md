### Post by PhilGil on 2010-04-04
I don't think there's much you can do about the launcher - the netbook remix is not as configurable as a standard Ubuntu install.  I'm not aware of any way to hide it.

My advice would be to leave it as-is for a few days and see if you get used to the look  If you still don't like it you can always install the normal version of Ubuntu and have far more options to customize the appearance and functionality.  You can remove the netbook remix features but (IMHO) on a new install it's easier to start fresh.

On my netbook, I went back and forth several times between the standard Ubuntu and the netbook remix.  There are advantages and disadvantages to both, but I finally settled on the remix because the launcher was more convenient for me to use.
