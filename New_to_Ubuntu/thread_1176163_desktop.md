### Post by lovinglinux on 2009-06-02
You can disable them too, using the "Configuration Editor". I don't remember exactly where to find it in the menus, because I moved mine but it should be somewhere like "Accessories", "Other", "System Tools" or "System >> Preferences".

After opening it, expand "apps/nautilus" tree and search for the options related to desktop icons.
