### Post by Rubi1200 on 2011-06-03
For future reference, if you have a problem with a Wubi install see here:
[http://ubuntuforums.org/showthread.php?t=1639198](http://ubuntuforums.org/showthread.php?t=1639198)

There is also a small utility which can be used on Windows which could have saved the data:
you can recover your data from the Wubi install from _within_ Windows. In this case, you can use [ext2read]("http://ext2read.blogspot.com/") to recover any important data from the Ubuntu root.disk.

And, welcome to the forums :-)

By the way, if you actually meant that Ubuntu was installed separately on its own partition, then wolfen69's comments apply.
