### Post by phidia on 2008-07-25
Grub error 18 is about selected hard drive cylinder size exceeding maxium cylinder size reported by the bios. There's an old thread [here]("http://ubuntuforums.org/archive/index.php/t-6448.html") that covers some of that including some ideas on how to fix that, but you might just want to use the live or alternate cd to reinstall grub.
