### Post by Lars Noodén on 2012-01-20
LibreOffice will get you the closest to a clean import though.  docx is not a very useful nor well-documented format.  The short answer is that you will probably have to do some of the template by hand.  How much depends on which features are used.  Once you get a good Open Document Format template, there are several applications which can use it without problem.
