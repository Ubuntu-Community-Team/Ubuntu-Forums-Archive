### Post by Kosimo on 2009-04-28
What happens is that the home folder has hidden folders. 
If you want to see them open your home folder and hit: CONTROL + H

Then you'll see many folders starting with a dot .xxx 

This folders are typically used by ubuntu and applications to save settings and specific user data, like pidgin's accounts, games settings etc.
