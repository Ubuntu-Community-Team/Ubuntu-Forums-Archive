ok, first, drive letters are a windows thing, and basically arbitrary. You have a particular drive you want ubuntu studio installed on. Download the Ubuntu Studio CD/DVD (not the regular ubuntu cd, as this will not include the same software as studio). Install the drive in the computer you will be using ubuntu on, boot from the live cd/dvd and install there. You're more likely to get all the appropriate drivers etc. this way and have a shorter setup time.
