### Post by julio_cortez on 2010-12-09
> **wt8008 said:**
> I just wanted to clarify that .conkyrc is a hidden file, not folder.+1.
.conkyrc is a file, and this explains why you're able to edit it with gedit but not "browse" its content with Nautilus.

Of course, if you don't see the file in Nautilus even triggering the hidden files view with Ctrl+H, it's a different problem.
In that case try to make a```
cd
cp .conkyrc conkyrc.nothidden
```and see if the conkyrc.nothidden could be seen among your "normal" files (since it's lacking the point at the beginning it shouldn't be hidden anymore).
