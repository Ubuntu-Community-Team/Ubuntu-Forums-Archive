### Post by mc4man on 2013-03-07
The Dash is affected by the opacity setting in "Background Color" option of unity.
If you're using 12.10 you can use black (#000000) if a color doesn't suit, don't know if 12.04 was ever fixed for black

Just raise the opacity till you're happy with result 
(on 12.10 if using a very high opacity then you may as well also switch to the "no blur" option

Whether these are in your helper apps don't know, otherwise install compizconfig-settings-manager & you'll find in the unity plugin settings
