Well, I don't think  there is a need for anti-virus. Ubuntu is less of a target for malware, as well as it has a sophisticated security system and runs applications in a different way than Windows so malware isn't as big of an issue. You CAN try Klam AV (I think it's called) or Avast antivirus if you are still concerned, though. 

As for updating, you have two options: You can download an ISO and burn to a CD or other media and when inserted, it will detect the updated packages and will be able to update (i think, I havn't actually tried this). You can also do a base install through booting the CD/other media, or you can start update manager after 11.04 is released and update the package cache, then it wil say something like "release ubuntu 11.04 available" and will allow you to update the system without removing any files.

To me, the pros compared to Windows are that it's not as much as a malware target, most software and hardware works out of the box without the need of additional software/hardware installation, programs can be installed from the internet by downloadable .deb or .tar.gz (compressed file, like .zip or .rar) packages, or through the use of the repositories through terminal or Software Center (which through the repos is quicker and easier in my opinion). And, compared to Windows, Linux doesn't run slow over time.

As for cons, the only thing I would say would be the lack of acknowledgement from larger companies so there are fewer programs and drivers released for Linux because it is a very little used operating system compared to Windows. But still, just about anything you would need, there's an app for that.

2.  When newer versions of Ubuntu come out (soon to be 11.04 I believe), do you just do an update or due I have to install the entire new system?

It depends, most users upgrade, a lot of people prefer reinstalling, some people separate their home directory into a separate partition altogether, so if they need to reinstall they don't need to backup all their documents etc

I personally advise everyone to get a $5 USB drive, use Unetbootin or USB Startup disk creator to make the USB a bootable ubuntu drive, then use the desktop live cd as a test, if everything is cool, then upgrade, if not, back off and wait till the next release.

Main reason being 11.04 has some large changes incoming.

### Post by Legeril on 2011-04-17
Welcome to the community, there is a lot of help here and masses of knowledge. But try to ask intelligent questions and give as much info as you can when asking questions, this helps us help you.  I prefer to stick to LTS releases as they are generally smoother and because I don't see the need to upgrade every 6 months.
