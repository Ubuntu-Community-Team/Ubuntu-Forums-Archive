### Post by sudodus on 2013-03-10
If you can download an iso file reasonably easily, I suggest that you download the lubuntu 12.04 alternate iso file and make a CD or USB boot drive. The computer should boot from that, and Lubuntu can be installed. Then we can help you with commands to evaluate the system and find out if it is suitable or if another version or flavour of Ubuntu would be better.

Maybe the computer has horsepower enough to run a desktop environment with more eye-candy, but Lubuntu alternate is a good way to get started.
