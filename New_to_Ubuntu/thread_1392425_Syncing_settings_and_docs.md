### Post by Temposs on 2010-01-28
With Ubuntu One(in the current version), it will sync any files you place in the Ubuntu One folder. This will change in Ubuntu 10.04, but that's a limitation for now.

So, if you are able and willing to keep the files you want to sync inside the Ubuntu One folder, then it'll work perfectly well. Unfortunately, that means right now you can't really sync something like settings for a program unless you copy it to the Ubuntu One folder whenever you want to sync to your other computers and then drag the settings out on the other computers.

---

### Post by Temposs on 2010-01-28
But, I think Ubuntu One is where the future is at in terms of syncing files on Ubuntu, so I recommend trying to use it. It's going to be expanding a lot in functionality in Ubuntu 10.04, and it's already a pretty nifty thing to have included by default in Ubuntu.
