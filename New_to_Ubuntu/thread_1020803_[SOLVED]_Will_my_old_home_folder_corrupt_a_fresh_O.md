### Post by RealPSL on 2008-12-24
When I installed 7.10 I did not like it for whatever reason I can not remember now. The beauty of having a separate /home partition is you can do a clean install of an older release and chances are everything will still work. Even if something is broken chances are removing the hidden configuration files in your home directory will fix it. I do not however recommend going back to an unsupported release. I will admit that 7.04 has a special place with me as well but going with older unpatched releases is not something I recommend. 

8.10 has been pretty good for me so maybe you should try going forward. I did not like 8.04 much for some reason either.

Note that although I have a separate /home partition I still take a backup before doing an install. Stuff happens mostly caused by me so I take one just to be sure.
