### Post by louieb on 2010-03-23
You can set the level of update to Security updates only - or apply all available updates.

System>administration>Software sources. 

Its not unheard of that an update can cause weird things happen. I don't worry about it. I have my system set to apply all updates. And I just let it update.

Are you using GRUB legacy or GRUB2?  No cookie cutter answer for this one. Check the [Tutorials & Tips]("http://ubuntuforums.org/forumdisplay.php?f=100") section.  Just depends on how you set things up.
