### Post by ajgreeny on 2014-12-11
If you like to see what's going on, as you put it, install synaptic and use that instead of software-centre.
Synaptic allows you to choose to "Apply the changes in a terminal window" in its preferences dialogue, but also gives you a lot more detailed information about, for example, dependencies required for a package, and its search facility is very good.

In my opinion it gives you the best of both worlds, GUI and CLI.
