### Post by winwiz on 2014-05-06
I upgraded my Desktop from Ubuntu 12.04 LTS to Ubuntu 14.04 LTS and the installer told me that I may have to re-install some of my apps. In-order to make it faster and easier, I wrote a post install script to do the work for me.

I call it **init_my_ubuntu**

It maybe useful for some of you. Just customize the script to your needs and run it after a new install or upgrade. Please feel free to contribute to make it better and useful for even more users.

I have done some stuff like:
- Added android development related stuff. That is just for me :)
- Automatic shutdown when done.
