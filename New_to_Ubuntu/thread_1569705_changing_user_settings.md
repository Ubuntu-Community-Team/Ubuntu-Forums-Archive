### Post by mcduck on 2010-09-07
> **mike101 said:**
> I've changed my user setting to adminstrator but want to change back to custom. When I try to do this graphically the custom option is grayed out. Does anyone know how I cna do it?

There is no single "custom" setting, it simply means anything else than one of the preset settings. Just click the Advanced Settings -button and set the permissions as you wish, and if they don't match any of the pre-made account types it will become "custom".
