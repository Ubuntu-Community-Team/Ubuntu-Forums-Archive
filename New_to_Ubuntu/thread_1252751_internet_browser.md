### Post by bodhi.zazen on 2009-08-29
There are several thing you can do that will make a (significant) difference.

First is upgrade to firefox 3.4.2 or icecat

Do this by downloading the binary and put it in /usr/local/firefox (or /usr/local/icecat)

Second there are a few performance tweaks that may help . Yes there are endless tweaks, but I found a few that made a difference and posted them here :

[http://blog.bodhizazen.net/linux/netbook-optimization/](http://blog.bodhizazen.net/linux/netbook-optimization/)

using tmpfs and mouting your cache in /tmp/cache (as well as the other ff tweaks) should help.

If that does not work, try an an alternate browser. Personally I like dillo.
