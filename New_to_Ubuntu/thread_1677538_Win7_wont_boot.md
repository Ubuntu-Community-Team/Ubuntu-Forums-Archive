### Post by Quackers on 2011-01-29
In Windows your file system has been changed to SFS. This often means that at some time you have exceeded the maximum number of primary partitions on one disc (4). 
When this condition is reached Windows changes its partitions to dynamic disks. This is normally bad news for installing other operating systems, as the drive becomes unrecognisable to systems other than Windows.
However, you have Ubuntu installed, so I'm thinking that the partition error may have occurred since you installed Ubuntu.
Anyway, since you can boot Ubuntu, I'm hopeful that you can use both systems (but not alter any Ubuntu partitions until Windows is re-installed).
If you boot the machine from the Windows installation disk and enter the recovery console, you could try running a chkdsk with the following command entered in to the command prompt.
```
chkdsk C: /r
```
If that runs a chkdsk and reports that errors were found and fixed, please run the same command again, until no errors are reported.
Then try booting Windows again and cross your fingers :-)
