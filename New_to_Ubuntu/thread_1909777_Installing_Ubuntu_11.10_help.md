### Post by mastablasta on 2012-01-16
if it skips the disk and boots to windows then it means something is wrong with the disk you made or with the downloaded image file. check the md5sum of the image file and attempt another burn on the disk or use a USB stick instead. 
 
[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)
