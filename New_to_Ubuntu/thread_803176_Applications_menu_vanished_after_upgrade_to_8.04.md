### Post by oshilig on 2008-05-22
SOLVED - went here:
/usr/share/applications 

chose Menu Editor

selected Defaults.

Solved.

The problem before was that none of the options to display on the Applications menu were checked, which is why there was no menu.

I don't know how this became like this.

### Post by Francewhoa on 2009-02-15
[FONT=Arial]Easy solution with Ubuntu 8.04 Desktop
1. Right-click on panel bar. Make sure you click on the panel bar NOT an icon or a menu item.

2. Select '+ADD TO PANEL...'

3. Find 'MAIN MENU' item. You might need to scroll down.

4. Click on 'ADD' button

5. 'Applications Places System' menus will be added to your panel :). If you want to change the location right-click on the 'MAIN MENU' then select 'MOVE'. Drag and drop to new location.

------------------------

If above doesn't work press ALT-F1

An extended menu will appear. That won't bring the menu back to its original place but at least you can access it.

(For absolute beginners press 'ALT-F1' means press and hold ALT key on your keyboard then press F1 key.)
[/FONT]
