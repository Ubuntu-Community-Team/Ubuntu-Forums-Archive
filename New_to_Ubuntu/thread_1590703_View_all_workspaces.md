### Post by warfacegod on 2010-10-08
You have a few options. You could use the Expo feature. If I recall, the shortcut for that is: Superbutton (Windows Logo)+E. By Left/Right cursor, I assume you mean the arrow buttons. Have you tried using the mouse instead? You can set the Cube as transparent. That way you can "see" the other desktops.
