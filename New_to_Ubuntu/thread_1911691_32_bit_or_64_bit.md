### Post by Fernhill Linux Project on 2012-01-19

64bit will only work 64bit machines, and is recommended if your machine is 64bit.

### Post by Double.J on 2012-01-19
Yup! all 64bit systems can run 32bit :) Most people with new machines and 32bit OS are running 32 on 64 hardware.

One of the reasons for this is that 64bit uses a few more resources. The general rule of thumb over the last few years has been 64bit is better, but only worth it if you have 4GB of RAM or more.

32bit can't read more than 3.5GB of RAM, so you aren't getting all you paid for! however if you have 3GB or less, 60% or more could be taken up by running a 64bit OS.

There have also been previous stability issues with linux and 64bit. I don't forsee this being much of an issue going forward - 12.04 is going to be a default 64bit OS (it'll also come as a live dvd not cd :) ) so  it's probably a good idea - you can't upgrade 32bit to 64bit to tie in with your other thread ;)

Hope it helps!
