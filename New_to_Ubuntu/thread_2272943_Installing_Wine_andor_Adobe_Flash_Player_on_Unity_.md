### Post by Holger_Gehrke on 2015-04-09
Wine is not an emulator, it's a Windows compatibility layer. The machine you're running it on needs to have an x86 processor. Adobe Flash for Linux on Arm ? I don't think something like that even exists ... You might have some luck trying gnash, an open source flash player from the FSF, but that is even further behind in features compared to flash on windows than Adobe Flash for Linux.
