### Post by doctorbighands on 2008-10-01
Try:

System > Administration > Software Sources
check all the boxes in the "Ubuntu Software" tab, then click close.
You'll probably be prompted to update, so go ahead and do that.

There's a way to do this at the CLI, but this way is easier and less confusing.
