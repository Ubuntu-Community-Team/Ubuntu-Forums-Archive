### Post by ajgreeny on 2010-05-31
It sounds as if you are using the ubuntu netbook edition.

You can still add a launcher.but not by the right click method in that ubuntu version.  You will need to start up the menu-editor, called Main Menu in System->Preferences from your left hand groups, and in that click on new item, having selected the main group where you want it to be (graphics).  Fill in the boxes with the name, command and description, click on the default icon shown and choose another if you want to and you should then have a new icon/menu item available.
