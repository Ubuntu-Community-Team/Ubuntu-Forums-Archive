### Post by edd07 on 2008-09-25
XULRunner is a Mozilla technology that takes XML-like syntax, parses it, and shows it as a UI. It can use Javascript also, to control behavior. It is used by Firefox, Thunderbird, and other non-Mozilla OSS proyects, such as [Songbird]("http://songbirdsnest.com")

Hope this helps

Oh, and by the way, XUL stands for XML User-Interface Language
