### Post by andrewmv on 2009-07-30
I don't know if this was the original cause of the problem, but as of version 0.25, the TweetDeck developers have removed System Tray functionality from the application, due to that feature causing crashes on Windows based machines

Because it's an Adobe Air application, they don't have different releases for different operating systems, so the system tray is gone from all versions for now.

They're planning on adding it back in a as-of-yet unspecified future version.  More information is here:

[http://tweetdeck.posterous.com/minor-update-to-tweetdeck-v025](http://tweetdeck.posterous.com/minor-update-to-tweetdeck-v025)
