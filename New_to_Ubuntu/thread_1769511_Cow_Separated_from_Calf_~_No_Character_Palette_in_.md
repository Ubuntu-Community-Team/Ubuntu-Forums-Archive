### Post by iansane on 2011-06-02
Just posted this in another post. Make a launcher by right click on the desktop, leave it as "application", put the name character map or what ever you want, put in the command "gucharmap" or as someone else posted in the other post, "charmap" works.

Or you can right click on the panel and choose add to panel>custom launcher and use the same command.

The biggest problem to me is Unity which I quickly removed after a couple of days trying to get used to it. Well I didn't actually remove it but I set my default login to use classic ubuntu.

Go to applications and type in the search for "login screen", open it and set Ubuntu classic as the default session.

Now everythings back to normal. I don't even notice I'm using natty. Of course I haven't run into the bugs everyones talking about yet.
