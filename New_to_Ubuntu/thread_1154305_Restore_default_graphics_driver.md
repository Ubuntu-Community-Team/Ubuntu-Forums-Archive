### Post by humbug01 on 2009-05-09
I installed Jaunty Ubuntu on my son's computer. It all went well until I allowed it to install the proprietary ATI graphics driver through the Hardware Drivers program. When I rebooted it sent me to a CLI and would not start X.

How do I restore the original, default graphics driver so that I can get back to the GUI? (It was working--though with no special windows effects--before I installed the restricted driver.)

His system is: Asus P5Q Pro motherboard with two ATI Radeon 3000 series (3650?) graphics boards in Crossfire arrangement.

Thank you in advance....

---

### Post by ibuclaw on 2009-05-09
Reboot the computer, and in the boot menu, select the "**Recovery Mode**" option for Ubuntu.

Then select the menu option named "**Fix X**", once it's finished running, you can continue with normal boot.
