### Post by cariboo on 2009-11-08
THe server install is command line only, there is no gui provided, so an onboard graphics adapter is good enough. Because you aren't running a gui, a 4 core cpu and lots of ram aren't needed unless you intend on doing virtualization. A good quality low to midrange motherboard is good enough. If you intend on setting up a raid array get hard drives optimized for raid. Depending on what you plan on using the server for, a case with redundant power supplies would be a good idea, as well as a good quality ups.
