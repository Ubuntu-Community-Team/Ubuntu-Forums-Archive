### Post by ajgreeny on 2016-12-11
Go to [https://www.ubuntu.com/download/alternative-downloads](https://www.ubuntu.com/download/alternative-downloads) and in the Torrent section download one of the torrent files.  They are very small so it will not matter too much if you don't want to use it to download an .iso image of Ubuntu.

Right click on the downloaded torrent file and choose Properties from which you should be able to choose the default to open them, though exactly how depends on the DE and file manager you are using.
