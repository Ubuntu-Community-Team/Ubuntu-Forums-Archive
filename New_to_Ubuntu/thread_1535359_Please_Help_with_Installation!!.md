### Post by themusicalduck on 2010-07-20
You need to select to save the file, rather than use 'Open With'. You might be able to just copy it to your desktop or something from that the temp folder. Otherwise just download it again, but don't select to open it with anything.

Alternatively, I think Unetbootin has a feature that downloads the iso automatically before putting it onto the usb stick. You could try just running Unetbootin and then selecting Ubuntu 10.04 from the drop down menu.

I'd take a look at this too - [http://ubuntu-manual.org/](http://ubuntu-manual.org/) there's a section on installing Ubuntu which might be useful.

### Post by Vaphell on 2010-07-20
1. can't you right-click that iso file in firefox download window and select 'open containing folder'? if it's not there then it's not there. if that is the case i think it's because you selected 'open with' instead of 'save to disk'. 'Open with' saves file temporarily so the app can open it, but after closing it can be deleted. 'Save to disk' = 'yes, i want to store that file permanently'
