### Post by sav2005 on 2008-05-24
I use LaTeX a lot. For the hard core user you can't go past Emacs with the Auctex package (or Vim-latex-suite), but ... most of us aren't hard core!

My number one recommendation is Kile. Kile is a very user friendly application that looks a bit like Quanta or the code view in Dreamweaver. Lots of wizards and alt+space bar autocompletion of tags and such. It will install some kde libraries but not the whole desktop. 

Number two on my list is Winefish - a Bluefish offspring that is a bit smaller and quicker than Kile but not so good for setting up multidocument projects or creating a preamle.

KDissert is a mindmapping tool that exports an outline into a Latex article or beamer document.

If you've come from Windows then Kile looks a lot like TeXNicCenter or Winedit but I think it is much better than either. Winefish looks like the Gnome Text Editor with nice plugins.

You can apt-get or Synaptic install all of these.
