### Post by oldrocker99 on 2018-08-02
And definitely check out the **BEST distro I have *ever* used,** Ubuntu MATE, which, by the way, allows the interface to look like Windows (Redmond), Mac (Cupertino), or, if you really want to, Unity (Mutiny). It's lightweight, especially compared to Ubuntu GNOME or Kubuntu. At boot, in idle, it uses <500MB of RAM. There are glowing reviews of it on YouTube.

I use it as it installs. In 2008, I installed Ubuntu 8.04, which came with GNOME 2.3. I fell in love with the interface, which took me all of 10 minutes. Then, the GNOME developers decided to fix what wasn't broken, and created the greatly-disliked (by me) GNOME 3, and then Canonical devs created Unity, another desktop I hated. Then, an Argentinian guy decided to use the original GNOME 2.3 source code, updated it and released it to Mint. Then, another set of developers started Ubuntu MATE, which is an official Ubuntu flavor. I love it for a number of reasons, especially because I despise big program icons. It's fast, and suitable for newbies and experienced users. 

MATE, by the way, is not pronounced "Mayt," but "MAH-tay." It's named after the caffeinated Argentinian herb, yerba mate.

I have tried quite a few distros over the years, and UM is, hands down, the very best distro ever, IMHO.
