### Post by Rhubarb on 2008-10-29
Try soundconverter from the Ubuntu repositories:
```
sudo aptitude install soundconverter
```
Then you'll find it in Applications --> Sound & Video --> Sound Converter.

It's a nice little program that does what it says on the tin.  :)
