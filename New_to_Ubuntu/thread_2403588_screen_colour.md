### Post by Frogs Hair on 2018-10-13
Can you link the theme you are trying to use ? To change the desktop background right click the screen or use a picture of your own. To use 3rd party themes you need to install gnome-tweaks, gnome-shell-extensions and enable the user themes extension in gnome tweaks.You can install those packages easily from the terminal.  

```
sudo apt install gnome-tweaks 
``````
sudo apt install gnome-shell-extensions 
```
