### Post by Lars Noodén on 2013-10-21
Are you trying to extract the attribute "platform" from the XML element "attributes"?

```

#!/usr/bin/perl                                                                 

use strict;
use warnings;
use XML::TokeParser;     # from package libxml-tokeparser-perl                                           

my $file = shift || '-';        # name file or use stdin                        

my $p = XML::TokeParser->new($file);    # parse from file or stdin                      

# keep reading 'attributes' elements                                            
while ( my $token = $p->get_tag( 'attributes' ) ) {

    # extract and print the attribute 'platform'                                
    print $token->attr->{platform},qq(\n);

}

```

That will print them all out.
