### Post by supererki on 2011-01-29
If I were you - I would move grub to sdb device. 

So reboot using your liveCD.
Mount the linux device :

$ sudo mount /dev/sdb1 /mnt

Mount all other devices :

$ sudo mount --bind /dev /mnt/dev

Mount proc

$ mount -t proc proc /mnt/proc

The change the root :

$ sudo chroot /mnt

And install grub

$ update-grub
$ grub-install /dev/sdb
