### Post by Hippytaff on 2011-02-08
try Logging into a terminal (rescue/recovery mode) and uninstall Xorg-edgers, and reinstall the standard X.org

```

sudo apt-get remove (whatever xorg-edger is called) && sudo apt-get remove xorg

```
```

sudo apt get-install xorg

```

If you feel comfortable doing that :-)

Well, xorg is your graphical interface (login screen, desktop, everything you see), so uninstalling it will almost certainly cause problems.

If you want to put it back on, you can press ctrl-alt-F1 to get to a terminal and login there.

From there it'll probably be a simple:
```
sudo apt-get install xorg
```
and a restart.
