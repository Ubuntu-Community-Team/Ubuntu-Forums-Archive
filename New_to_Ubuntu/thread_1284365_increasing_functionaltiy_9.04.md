### Post by lovinglinux on 2009-10-06
You can install a dock like [awn](apt:avant-window-navigator) or [cairo-dock](apt:cairo-dock) then free some screen space by removing a panel. There are also several cool [screenlets](apt:screenlets) available.

 I also like [easystroke](apt:easystroke), since it allows me to easily access things I regularly use, with simple mouse gestures.

If you are not familiar with BitTorrent, take a look at [BitTorrent optimization and troubleshooting guide](http://ubuntuforums.org/showthread.php?t=1259923). Downloading new releases of Ubuntu with BitTorrent is usually faster, specially in the week of release.

You can also try different distros inside a virtual machine with [VirtualBox](apt:virtualbox-ose).

If you want to try KDE, you can install [kde-core](apt:kde-core) and [plasma](apt:kdebase-plasma). The run the command below to launch plasma over Gnome:

```
plasma
```

You can also learn to use command-line, since you can do a lot of stuff from the terminal, which is usually faster than clicking through several menus to accomplish the same task.

### Post by lovinglinux on 2009-10-06
> **halfstrike said:**
> Also is it possible to run windows in a  virtual box if you already have it on its own partition?

Yes, you can. Actually, you can run pretty much any OS, although some of them do not play nice with the VM environment.

> **halfstrike said:**
> What is KDE?

[KDE](http://en.wikipedia.org/wiki/KDE) is another [Desktop Environment](http://en.wikipedia.org/wiki/Desktop_environment), like [GNOME](http://en.wikipedia.org/wiki/GNOME). Is the default DE of [Kubuntu](http://en.wikipedia.org/wiki/Kubuntu). I don't really like it, but it has a really nice interface.

It's weird for most users switching from Windows to understand the concepts of a Desktop Environment, because on Windows you have to use the only DE available and lot's of other things are not customizable or replaceable. But Linux provides you with freedom to choose almost anything.

### Post by cwsnyder on 2009-10-06
Something else you may want to look into which is available under GNOME, and M$ Windows is GNOME-Do.  This adds the capability to type in a few letters which begin a program name, and then either select the program from a list of suggestions or simply press enter if there is only one selection to run the program.

I don't use this myself, but I have heard others rave about it.
