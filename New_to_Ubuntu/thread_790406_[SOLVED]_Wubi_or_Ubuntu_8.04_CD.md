### Post by shifty_powers on 2008-05-11
> **UnknownFear said:**
> So Wubi is on the new 8.04 LiveCD? How would I get into Wubi once I boot into the LiveCD to make sure everything works OK? I don't want to burn the CD and find out half-way through the install process that my disk is damaged because of the burn process.

1). boot from the livecd, (i.e. choose the option to work from the cd rather than install), to see if the drivers for your hardware work ok.

2). boot from the livecd again, but when you get to the initial menu, there should be an option to verify the disk.

3). Boot into windows and right click on the disk and select autoplay, this will give you an option to install wubi within windows ;) If the disk is damaged it won't make a difference if you're installing wubi anyway. In fact iirc, wubi will download an image from the ubuntu server anyway...
