### Post by dapps2 on 2014-02-18
I am using Lubuntu 13.10 and would like to know how to stop the screen from going black every 5/10 minutes or so?

---

### Post by Dennis N on 2014-02-18
Turn power manager off:

**Preferences > Default Applications for LX Session**

Set 'Power Manager" command to NO  and press "Apply" button.

Next boot, it will not start automatically and cause this.
