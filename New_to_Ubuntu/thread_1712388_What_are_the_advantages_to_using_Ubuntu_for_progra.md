### Post by Telengard C64 on 2011-03-22
For the most part the code you need to interface your program with the rest of the system (called APIs some places) is well documented and available without restrictions. Other systems have so called secret APIs which are forbidden to access without license.

For the most part every aspect of Linux systems is well documented with source code available. Instead of coding against an unknown black box and depending on unpredictable mis-features one can know exactly how various parts of the system work. Again this is without having to seek license to do so.
