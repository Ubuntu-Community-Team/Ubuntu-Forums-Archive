### Post by nhasian on 2008-11-01
whoa slow down there cowboy.  if your going to upgrade, dont format anything!  :)

if you want to keep all your settings, programs, and data, then boot into the OS first then you can do 1 of two things:

run the upgrade from the update manager*, or insert the Alternate 8.10 disc into the computer and you will be prompted if you would like to upgrade from the disc.

* You may need to go to System/Administration/Software Sources, and choose the Updates Tab and under the Release upgrade line change it from Long Term Support Releases to Normal releases.  then next time you run the Update Manager it will notify you a new version of the OS is available.
