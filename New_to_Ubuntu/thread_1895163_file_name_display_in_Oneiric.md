### Post by lechien73 on 2011-12-14
In a Nautilus window, pressing Ctrl+H allows you to see hidden files and folders.

Pressing Ctrl+2 gives you a list view, rather than icons, and Ctrl+3 gives you a compact name-only view.

If you bring your mouse up to the top panel and select **Edit -> Preferences** (or Alt+E then N, if you prefer), you can set it so that Nautilus *always* shows hidden files and folders, and you can set the default directory view.
