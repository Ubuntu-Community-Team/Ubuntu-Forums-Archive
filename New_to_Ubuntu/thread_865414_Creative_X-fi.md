### Post by Dapilot1 on 2008-07-20
It is a compressed file, extract it somewhere. Then open up the terminal and type cd (with a space after it) and then drag the folder you just extracted in to the terminal. This will change the directory to the one with the installer in it. To run the installer, copy and paste this.
```
sudo ./installer
```
It should work. That is what the readme says at least.
