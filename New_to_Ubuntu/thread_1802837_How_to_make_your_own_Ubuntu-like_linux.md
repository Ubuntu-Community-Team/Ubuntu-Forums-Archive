Tiny Core Linux:
[http://distro.ibiblio.org/tinycorelinux/welcome.html](http://distro.ibiblio.org/tinycorelinux/welcome.html)

Super light, still has a GUI.

Hard to get smaller than that, it's 10MB :)

Now, if you want a lot more options and niceties, you can go with others such as Puppy (~100MB), or Knoppix or others.
