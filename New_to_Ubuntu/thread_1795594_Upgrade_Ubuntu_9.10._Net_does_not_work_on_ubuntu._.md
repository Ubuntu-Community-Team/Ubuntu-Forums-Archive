### Post by yeats on 2011-07-02
You'll need to provide more details for a useful answer, but I would suggest the following:

1) using Windows (since the net connection works) download an Ubuntu installation disk for a supported version (10.04 LTS, 10.10, or 11.04).  9.10 has been out of support since April.

2) back up the files you want to keep in Ubuntu to external media.

3) burn the Ubuntu disk and use it to reinstall.

If you're running an older version than 9.10, it will be easier to reinstall than to try and upgrade through unsupported versions.

Unfortunately upgrading from the LiveCD is not supported until the 10.10 release. 

To upgrade online you'd have to go ->10.04 ->10.10 ->11.04

You might want to stick with 10.04 as it is the LTS release, supported until April 2013.  Maybe just download the version you want to use and install it.  Backup ALL your data first!


[quoteWhen i download ubuntu from windows , its a very large file. When while downloading net gets disconnected , i needs to install from beginning.[/quote]
