### Post by Dragonbite on 2011-02-17
Kubuntu is the core Ubuntu system (the bits you usually don't see) but instead of using the Gnome desktop environment that Ubuntu does, it uses the other popular desktop environment KDE.  You can see some of the difference from these [[COLOR="Blue"]_Kubuntu screenshots_[/COLOR]]("http://www.thecodingstudio.com/opensource/linux/screenshots/index.php?linux_distribution_sm=Kubuntu%2010.10").

Kubuntu includes some different applications than Ubuntu, these being developed more for their specific desktop environment. Such as 
[LIST]
[*]Kopete instead of Empathy (Instant Messenger)
[*]Dolphin instead of Nautilus (File Manager)
[*]Amarok instead of Rythmbox (Music player)
[*]Digikam instead of Shotwell (Photo Manager)
[*]etc.
[/LIST]

It doesn't mean the Gnome-based applications included in Ubuntu will not work in Kubuntu (or vice-verse), but they may have some dependencies which need to be installed.  The webcam program Cheese, for example, requires a number of Gnome-based libraries.  In Ubuntu this isn't an issue because these libraries already exist, as they are used by the Gnome desktop environment.  In Kubuntu these libraries are not used by something else installed and so needs to be installed when you install Cheese.

This all works in vice-verse. You can install Digikam in Ubuntu, it will just bring its KDE-orientated dependencies along with it.

Sorry, I'm not familiar with Wubi and its limitations, or how to tether a phone to the system.

### Post by TBABill on 2011-02-17
Kubuntu runs the KDE desktop environment. It's different than the environment you see using Gnome and operates a bit differently. You can install KDE, xfce, LXDE all on your single install of Ubuntu and when you enter your name and password before going to the desktop, just pause before you hit enter. Instead find the item on the screen to select your environment and choose the one you want to use for that session. It's a great way to try them all out without commiting to just one.
