### Post by Celauran on 2009-06-11
Installing Ubuntu from scratch is actually painless, especially if you don't have any dual-booting concerns. Boot from CD, follow seven (I think it's seven, maybe nine) simple steps, and you're up and running in about 20 minutes. You'll have most of what you need (browser, email client, OpenOffice, image editor, etc) installed 'out of the box'.

The GNOME UI is pretty intuitive, so I don't expect it will take you long to become accustomed. Really, the biggest difference will be when it comes to installing software. Rather than tracking down software application by application, downloading from the developers site, and then running the executables, Ubuntu houses all its available software in a repository. You just fire up the software installer (there are a few installed by default, they all do the same thing), browse through the list to find what you want (it's broken into logical subsections and has search functionality, so you won't be looking for a needle in a haystack), mark for installation and, once you've marked everything you want installed, click Apply and it will download and install everything for you. It's like RSS for software, if you will.

Can't help you with the tablet, though. I'm sure there's some flavour of Linux that will work, but I have zero experience with tablet PCs.
