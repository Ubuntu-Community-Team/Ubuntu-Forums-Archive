### Post by zemega on 2013-04-24
Introducing WPS Office, A China clone of Microsoft Office, featured in OMG Ubuntu. Its still in beta, but it might interest you.
About the formatting, you might want to save in doc format eventhough you're using Libre, that might help you a bit, since you juggle between Libre Office and Microsoft Office. If your Asus laptop  comes with Windows 7, I'm sure there is a sticker underneath there with the key, so you can install it again whenever you want. 

13.04 is coming out, and it has faster Unity compared to 12.04 if you are interested. Ubuntu Tweak and MyUnity might be of interest to you. You might wnat to check whether your sound plays correctly when using a headphone, there are cases that when you play sound with headphone plugged in, both headphone and laptop speaker 'sound'? together.

Which model are you using? the low low end or perhaps a medium low end like 1225B (I used this one, it can run Win 7 and Ubuntu 12.04 just fine and fast). If you are willing to splurge, but keep the laptop, a change from HDD to SSD will work wonders for Windows perfomance, get the new SSD though, the olds one are buggy.
