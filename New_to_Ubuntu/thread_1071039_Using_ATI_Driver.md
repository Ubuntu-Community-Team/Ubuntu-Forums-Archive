### Post by Ptero-4 on 2009-02-16
You probably need to use the radeon or radeonhd driver instead of ati (which is for older ATI cards). Also you need to be able to access X11 to run displayconfig-gtk (and probably will need to install it). You can use the "recovery" boot option and use the "Fix X11 (or something like that)" in the menu that appears at the end. This gets you an usable X11. Then install xorg-dev and try first using the radeon/radeonhd drivers and if they fail, try the ".run" file you tried early.
