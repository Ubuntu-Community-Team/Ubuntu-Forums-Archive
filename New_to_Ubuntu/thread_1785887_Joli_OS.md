### Post by dFlyer on 2011-06-18
md5sum is a checksum that insures you have a good download. If your using an linux distro enter the following in a terminal where your file is downloaded:

md5sum "Name of file.iso" with out the quote marks.

Your should be able to get the md5sum from the website you use to download your ubuntu file. Also as states above, I'd recommended 10.04-2 or 11.04. The 10.04-2 is the LTS verson, but the 11.04 version comes with unity as default desktop.
