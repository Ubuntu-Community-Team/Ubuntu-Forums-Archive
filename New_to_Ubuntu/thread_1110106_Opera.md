### Post by benerivo on 2009-03-29
If you create a launcher, i think there are a couple of commands you might try adding. -nomail disables opera's email function, and -notrayicon removes the icon from the notification area of the panel.

Use them with this command for the launcher...
```
opera -notrayicon -nomail
```
