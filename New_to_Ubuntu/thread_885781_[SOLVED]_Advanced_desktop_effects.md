### Post by superbenny on 2008-08-10
> **peterjakey said:**
> Hi, I was just wondering how you can set different desktops to have different wallpapers. Under 'configure desktops' there is only the option to set the wallpaper for all desktops. Also, how can I add my own background to the top of the cube. =)

To add your own desktop to the top of the cube, you need to enable the "Cube Caps" plugin (and then configure it). As to the multiple desktop thing, I've been trying to figure that out myself. I had it setup fine in Kubuntu, but when I switched, I couldn't figure it out for the life of me.
In Kubuntu, you can go to the Settings Panel, and set your multiple desktops under Desktop which is under the "Look & Feel" category. You can set up to 32 desktops I think. As long as your # of sides corresponds to the # of desktops you have set here, you'll be fine.
