### Post by lisati on 2008-06-04
If it's a "Live CD" you'll be able to give Ubuntu a test drive before you actually install it. This will give you a chance to figure out which hardware you'll need to get drivers for, if any. If all goes well, you won't need to go hunting around.

If you discover that you need specific drivers, you can always ask in these forums. Just remember to be as specific as you can.

EDIT: p.s. presumably you're checking out version 8.04 - the number after the decimal point will matter later in the year.
