### Post by frank604 on 2011-04-24
I agree with above reply. Download the i686 or 86_64 image. Burn to cd/dvd/or usb. Set bios to boot from cd or usb before hdd. Boot from the image and it will present you with two options. First is to try and second is to install. Select try and it will boot up ubuntu off cd. The os will be slower than usual since its booting off cd but you can see if your devices work. Don't be discouraged is something doesn't work as you can easily configure it after installation. As far as I know ubuntu can easily communicate over the network with windows and shared devices. Best is to try it out and if you come across any problems there are tons of wikis to help and the forums.
