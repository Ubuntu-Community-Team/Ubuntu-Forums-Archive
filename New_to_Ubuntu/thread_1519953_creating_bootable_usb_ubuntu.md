### Post by C.S.Cameron on 2010-06-29
You can do a Full install which is almost like having Ubuntu installed to internal hard drive, but 4GB is just about minimum for that.
Startup Disk Creator from the Live CD will make a persistent disk image install.
Universal USB Installer will make a persistent disk image install from Windows.
UNetbootin and LinuxLive will make non persistent disk image installs that can be made persistent using persistence partitions.

Startup Disk Creator is probably the easiest if you have already burnt a Live CD.
