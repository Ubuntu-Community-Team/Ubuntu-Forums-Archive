### Post by sayakb on 2008-05-24
I'm not sure about the extra keys you have but you might have a key combination to start OpenOffice. Open Advanced desktop effects settings (i.e ccsm, or if you dont have it, type: sudo apt-get install compizconfig-settings-manager)
Goto General Options -> Commands
There, inflate command and add ooffice at Command line 0 and add a key binding by inflating the Key bindings area.
