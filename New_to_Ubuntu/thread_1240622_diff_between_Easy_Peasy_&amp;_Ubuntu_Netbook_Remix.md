### Post by aysiu on 2009-08-14
Ubuntu Netbook Remix is Ubuntu but with the special netbook interface (the fullscreen one with the huge icons) preinstalled.

Easy Peasy also uses the same interface, but it also contains tweaks especially for the Asus Eee PC netbooks (to get wireless working, to get f-keys working, etc.).
