### Post by Vaedrah on 2009-04-11
I am considering upgrading from a Acer Aspire 5920 with 4 GB RAM (yes it only see's 3.0 GB) running Ubuntu 8.04 and VirtualBox for XP.

At the moment I see a few sales but the notebook technology is still dominated by 1~2 GB RAM, 32 bit, 320MB drive machines. Some are dual core up to 2.4 GHz or so and all have Vista but I wonder about what we may expect next year. Any crystal ball gazers please reply!!!!

I can't see any real benefit in buying a new notebook but I would like to see more RAM especially when virtualizing as RAM gets shared between host and guest OS's.

I don't like gaming (ok if others do of course) but I like number crunching and direct math speed is useful. So I am looking to see if quad cores or octals etc will emerge in the near future. I can't see much movement beyond 2.5 GHz per core as some fundamental technical issues exist here. However, multiple cores seem to present a pragmatic solution but is this communal resource efficiently managed?

If a quad, octal or higher core uses Windows XP, Vista, Windows 7 ?, Ubuntu, Fedora, Debian etc, will the core management be equivalent between each OS?

Further, the 3.0 GB limit seems to be related (in a silly way) to the address space of 32 bits (2^32 = 4.294967296 MB) but perhaps some advertising sleight of hand with 4kB = 1,000 B rather than 4,096B is used somehow? In any case, although computers can access other memory much larger than this, the standard computer just limits out at 3 MB for whatever reason/excuse. So I guess 64 Bits is the next obvious step forward.

I have seen one 64 bit notebook so far but it is massive. I prefer a standard size notebook (certainly not a PC!) Will 64 bit notebooks become the norm in a year of two? 

If I virtualize another OS on a 64 bit host can this be a 32 bit guest OS? Or will 64 bit hosts run 32 and 64 bit applications?

Perhaps not ubiquitously - I don't mind some limitations as long as there are temporary work arounds. So I guess I am looking for advances in the following wrt notebook available in the near future

1. 64 Bit OS
2. RAM NOT limited to 3.0 GB - 32 GB expandable would be cool.
3. Quad core as a minimum - Octal and 16+ Preferred
4. 2.5 GHz core clock speed or higher
5. 500 GB hard drive OK (1 TB preferred)

So if anyone have good predictive skills, just as users of the old XT and AT computers would not have foreseen even low range computers of today, will the machine I would prefer be commonplace in future and if so, how soon will this future arrive?

### Post by LowSky on 2009-04-11
1. 64 bit laptops have been out for years. Every computer made in the last 4 years is capable of using 64bit, well not all, but certainly most

2. laptops are not desktops, they have limitations, and beside no operating system uses more than 2-3GB of RAM even doing some of the most intense stuff,

3. Quadcore is the most current multi-core chip made (dual for mobile). Intel and AMD will be releasng 8+ cores in a year or more. Not really useful in a notebook or even a desktop, but in a server running VM's, it will be awesome. AMD was supposed to release fusion a processor and a graphic chip all in one, it was supposed to make computations in ridiculous time, an take up less space as the graphics were built in, but the project has fallen through, really sad.

4. Clock speed means nothing these days, its all about chip cache and FSB and core utilization. anything over 2Ghz is more than enough.

5. seriously Laptops can be had with 500GB now, not worth it as the larger the hard drive the more common failures occur. especially in the 1TB range. The reason is because of the new lead free designs that have appears, lead free solder it not as good as the old stuff when it ages, it has a tendency to crystallize and spread like water becoming ice.
