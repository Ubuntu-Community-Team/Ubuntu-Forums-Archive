### Post by ankspo71 on 2009-11-19
Hi,

I recently found out that Linux (and Mac) support many more characters in filenames than windows does. So when making data cds, and you want them to be usable in windows, try to keep filenames simple like:
a-z
0-9
"space"  
"-"   
"_"

Any other characters might cause problems between different OS'es.
see 'Naming Conventions' here
[http://msdn.microsoft.com/en-us/library/aa365247%28VS.85%29.aspx](http://msdn.microsoft.com/en-us/library/aa365247%28VS.85%29.aspx)
