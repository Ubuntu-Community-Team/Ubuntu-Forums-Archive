### Post by snowpine on 2009-04-27
Hi Marcus, welcome to the forums!

If you are new to Ubuntu, I suggest setting up a "dual boot". This will keep both Ubuntu and Windows on your machine. Each time you boot, you can choose from a menu which OS you want. I find this is very helpful while you are learning; you can always fall back on Windows if you don't know how to do something in Ubuntu.

There are two ways to set up a dual boot:

1) Use the Ubuntu Live CD to install. When you get to the "partitioner" step of the installer, choose the option to resize your existing partition and install Ubuntu in the free space. (If you don't see this choice, or don't understand the options, post back and we will help.)

2) Use a program called "wubi" to install Ubuntu inside of Windows. A lot of beginners find this is the safest method, because it does not make any permanent changes to your partitions. 

If you want to get rid of Windows and have only Ubuntu, use the Live CD like I suggested above, but when you get to the partitioner, choose "guided install, entire disk."

Good luck!
