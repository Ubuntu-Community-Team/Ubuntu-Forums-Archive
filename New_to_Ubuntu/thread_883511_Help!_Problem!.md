### Post by iaculallad on 2008-08-08
Try: System->Administration->Login Window
In the first tab "Local" of "Login Window Preferences", check if "Menu Bar:
Show Actions menu" is selected. If no, check it and close the form then do Ctrl+Alt+Backspace and re-login.
