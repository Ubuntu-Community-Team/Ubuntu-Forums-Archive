I have a wifi hotspot in ubunut 14.04 and I have forgot my password to login to wifi. How can I retrieve my password?

---

### Post by cariboo on 2015-08-08
If you are using the desktop version, right-click the network icon in the top panel and select:

```
Edit connections
```

Next select your network in the Network Connections window, then click the Edit button.  In the Editing window click the WiFi Security tab, then click Show password.
