### Post by JLeon85 on 2013-03-29
You should be fine with the hardware, but you may need to go to the "additional drivers" area in the settings (under "software & updates") to get the graphics card to work. 

To get those games working you will need to use Wine, which is a program that allows you to use many Windows applications. For example, Starcraft II currently has a platinum rating on Wine's wiki, so it will apparently run very well: [http://appdb.winehq.org/objectManager.php?sClass=application&iId=11123](http://appdb.winehq.org/objectManager.php?sClass=application&iId=11123)

Steam is now available on Linux and it has a growing selection of quality games. You can also install Chrome and run most of the games available on their webstore.
