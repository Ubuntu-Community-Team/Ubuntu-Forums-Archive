### Post by colsandurz on 2009-08-19
try this 

```

sudo mount -t vfat /dev/sdXN /media/mountPoint

```

Where /dev/sdXN is the device and partition number, like /dev/sdb1.  I won't be /dev/sda because that's probably going to be the main disk.

Also, you have to do
```

mkdir /media/mountPoint

```
