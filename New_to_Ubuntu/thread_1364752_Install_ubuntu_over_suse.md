### Post by Pastor_JW on 2009-12-26
Did you want to replace SUSE with Ubuntu or were you wanting to dual boot?  Replacing is easy, you just install using the whole disk.  The dual boot requires setting up partitions and installing on to the correct partition.  The install CD can do this alright but you may want someone who knows a bit about Ubuntu to sit alongside you while you work.
