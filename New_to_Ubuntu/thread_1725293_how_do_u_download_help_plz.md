### Post by Enigmapond on 2011-04-09
What you are trying to download and install is a windows file. EXE will not run. If you want 7zip to install in linux, Open Synaptic and search "zip" and then install p7zip-full. Or open the terminal and do:

```
sudo apt-get install p7zip-full
```

However, it will all be handled through the Archive Manager.
