### Post by jrothwell97 on 2008-09-30
Welcome to the Ubuntu Forums!

Trackpad scrolling is available in Ubuntu. Go into System/Preferences/Mouse. Under the Touchpad tab, enable vertical and/or horizontal scrolling. I'm not sure if pointing stick scrolling will work, but try Googling it and see if you can find anything.

Ubuntu might be incorrectly reporting your battery life, so try fully charging the machine and then leaving it and seeing how long it will take to discarge fully (there are tools, I believe, which will automate this so you don't have to watch the machine for eight hours.
