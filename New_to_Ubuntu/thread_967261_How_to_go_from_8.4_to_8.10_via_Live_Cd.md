### Post by vinay.v on 2008-11-01
You cannot use a live CD to upgrade your existing installation. You can only install afresh using the live CD (by booting from it).

If you want to upgrade your existing installation, download the alternate installation disc. You can just put it in the CD-ROM drive and Ubuntu will automatically detect and ask you to upgrade. (You must have applied all security updates before upgrading).

Another simpler way is to upgrade is to use the Update Manager. 
See more information here: [http://www.ubuntu.com/getubuntu/upgrading]("http://www.ubuntu.com/getubuntu/upgrading")
