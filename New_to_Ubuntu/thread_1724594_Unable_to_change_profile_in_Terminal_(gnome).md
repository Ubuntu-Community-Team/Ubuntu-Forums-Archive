### Post by AlphaLexman on 2011-04-08
The menu item 'Terminal' has an option 'Change Profile' if you have not setup an additional profile besides the 'default' it will be greyed out because there is nothing to switch to.

You can go to Edit -> Profiles to create a new profile.

You can go to Edit -> Profile Preferences to edit the default profile.

That option is greyed out as you most probably only have 1 profile setup and thus there are no other possible options to change to.

To change this go to: 

Edit>Profiles

Then select the profile you wish to edit, or if you choose you can create a new profile. 

Once you have chosen edit, you can make the appropriate changes - blackbackground -yellow text etc.
