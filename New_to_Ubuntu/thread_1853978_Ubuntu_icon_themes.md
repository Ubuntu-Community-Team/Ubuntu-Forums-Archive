### Post by Frogs Hair on 2011-10-03
You can replace an icon , but if the icon was not included with the theme to begin with placing the image in the correct folder won't work . 

[http://gnome-look.org/content/show.php/Pidgin+Oxygen+Style?content=84627](http://gnome-look.org/content/show.php/Pidgin+Oxygen+Style?content=84627)

You will have to look in the Oxygen icon folder locate and replace the icon with the correct size and file type . By file type I mean , .svg , .png and so on . Renaming the image to the usually works .
