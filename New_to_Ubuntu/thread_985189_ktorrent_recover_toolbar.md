### Post by aeonsablaze on 2009-11-13
You can restore the ktorrent toolbar like so:

1. Make sure ktorrent is closed

2. Open "/home/[your user name]/.kde/share/config/ktorrentrc" in your favorite text editor

3. Scroll down to the [Main Window] section

4. Change menubar_hidden to false

5. Celebrate with the happy dance of your choice :P.

note: I discovered this solution running KTorrent 3.2.5 on Fedora 12. I assume that these instructions apply to Ubuntu and whatever version of Ktorrent they currently have in their Repos.

P.S. Sorry to reawaken an old thread but I didn't know if either of you had solved the problem and I thought it would be good to have the solution posted for others that find this page.
