### Post by matt_symes on 2012-06-11
Hi

To add a new repository you can use the command

```
sudo add-apt-repository <ppa_name>
```

where ppa is the name of the repository to add. 

To do this you need to open a terminal and type the above command. Enter your password. It will not be echoed to the screen.

When asked hit Y to import the keys.

It sounds like you may have other problems though. To see if you have problems with your repositories, open a terminal and type

```
sudo apt-get update
```

Enter your password.
