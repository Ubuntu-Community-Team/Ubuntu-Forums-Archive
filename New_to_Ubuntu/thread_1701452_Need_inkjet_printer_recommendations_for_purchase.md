### Post by s1baker on 2011-03-06
Hi
I need recommendations on purchasing an inkjet 3-in-1 printer that is well supported ( and will be well supported in the future ) for Ubuntu.
I want one that has scan/fax/print, 8-1/2 x 11, and also one that uses the refillable ink cartridges. I do 90% of my printing in black only, and I find that I mostly only need the color cartridges with ink in them so that I can print in black ink. I think all of HP's printers now use the sealed vacuum ball type of cartridge that is difficult to refill, so HP is out. I would not be adverse at looking at Brother printers as they seem to be priced competitively I had a Brother printer that was over 4 years old when I started using Ubuntu and found that they did not support it for Ubuntu 10.10 ( which I am using ), so if I buy a Brother now, I would want that they would provide support for it.

To summarize, I need:
1. A 3-in-1 inkjet printer, 8-1/2 x 11 paper
2. Can refill the cartridges ( with a needle, not vacuum ball type refill )
3. Good price
4. Will provide good Ubuntu support for drivers in the future.

Thanks

### Post by s1baker on 2011-03-06
I have a large size HP printer that I use to print 24x36 and 18x24 prints for work. I recently bought a HP 7000 wide format printer as a back up to my large size HP printer. The HP 7000 can print 8-1/2 x 11 up to 13x19 prints. I can print 2 13x19 prints and it is usable in case ever my large size HP printer goes down. The problem I had with the HP 7000 was this. It has the vacuum ball type cartridges and after only using it a few times I let it sit for a couple of months. When I went back to use it the cartridges ( mainly the colour cartridge ) were nearly dry. I bought some new cartridges and again used it a few times, and this time I completely shut the printer off ( thinking that the printer was periodically cleaning the cartridges, thus the cartridges were emptied the first time because I left the machine on all the time, in the "ready" state ). Again, after a few months when I went to turn on the printer, the cartridges showed to be empty. You cannot print in black unless you have colour cartridges. I will use this printer as and emergency printer to my large size HP printer, but the next printer I buy will not have these type of cartridges.
( also, my large size HP printer has this vacuum ball type cartridge, only a lot larger in size, and I have never had this type of problem with it's cartridges )
