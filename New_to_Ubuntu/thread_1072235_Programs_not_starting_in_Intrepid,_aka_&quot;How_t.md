### Post by dzark on 2009-02-17
First step would be running the program from a terminal, that way when it dies, you'll have some output - usually quite simple, not necessarily a 'bug' but maybe a missing file or so

Next would be to install the bug reporting package 'apport' and setting yourself up a launchpad account (you can do that once the first bug has been reported)
