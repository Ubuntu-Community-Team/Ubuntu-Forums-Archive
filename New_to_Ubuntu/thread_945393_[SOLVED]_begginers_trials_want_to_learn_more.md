### Post by Idefix82 on 2008-10-12
Welcome!
First things first: you open a terminal through your usual menu by going to Applications->Accessories->Terminal.

When you ask specific questions and get instructions for the terminal, just ask what exactly they do. People only give the instructions first because most newbies don't want to learn, they want to get the job done. If you ask, people will be delighted to explain everything to you.

To learn some commands yourself, the key word you are looking for is bash. This is the interpreter that sits in the background. The class of all such unix/linux interpreters is called shell. So if you just google for something like "bash commands" or "shell howto" and so, you will find plenty of resources.

P.S.: Don't worry about being a burden. Many people who frequent these forums, including myself, fooman and others are looking for "burdens" like you to help them get started with this wonderful OS.

### Post by dhtseany on 2008-10-12
Welcome aboard! I'll tell you this much: The learning curve these days isn't nearly as frightening as it was 5 years+ ago. My house (yesterday actually) for the first time went 100% Windows free (5 computers) and while yes, I'm an IT guy by professional, I was at first just as intimidated as you are now. My computer illiterate sister picked up how to use it without **any** help from me and she hasn't needed my help yet!

Things to remember:
1. Linux is not Windows. You can use a program called Wine to run *some* Windows programs. Go to [www.winehq.org](www.winehq.org) and browse the AppDB. There you will see reviews on how programs run through Wine and if the apps you desire work good. You can install Wine by typing the following code in the terminal:
```
sudo apt-get install wine
```

Remember that even though you can get some Windows apps to run, there are plenty of better apps available that are 100% free to use!

2. The forums are here 24/7 to answer your questions. Just remember to be as specific as possible with your questions.

3. (And this may be silly to some but) Firefox is how you browse the internet. :)

If you have any questions please ask before just giving up.

Peace
