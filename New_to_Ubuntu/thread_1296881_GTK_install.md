### Post by yeats on 2009-10-21
> I just installed Ubuntu today and I would like to install GTK+

You can go to System -> Administration -> Synaptic Package Manager and do a search for gtk.  If you're looking for development files, look for packages that end in -dev.  GNOME is implemented in GTK+, so a lot of them should be installed already.
