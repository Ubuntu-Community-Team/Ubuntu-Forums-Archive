1) Open Contacts
2) Open Mylist
3) Select a contact and right click
4) Select "New Contact List"
5) Give it a unique name
6) Select where you want to put the list
7) Press Select
8) Add members to the list

When finished select Close.

You will now have a list with all the contacts you want to send to with a unique name.
When you send a new mail,you select the list and send. as normal.

Good Luck
