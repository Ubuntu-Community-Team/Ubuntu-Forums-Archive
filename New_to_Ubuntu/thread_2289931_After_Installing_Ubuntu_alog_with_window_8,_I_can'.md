### Post by grahammechanical on 2015-08-08
You are not going to like this.

There is only one computer operating system on that machine and it is Ubuntu. The Ubuntu installer gives us these options:

Install alongside      [that is other operating systems if they exist]
Erase disk and install Ubuntu
Encrypt the new Ubuntu installation for security
Use LVM with the new Ubuntu installation
Something Else    [this allows us to create and resize partitions ourself or choose multiple partitions for Ubuntu]

Which option did you choose?
