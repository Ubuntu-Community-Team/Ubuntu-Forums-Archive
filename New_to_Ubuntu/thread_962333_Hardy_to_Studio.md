### Post by stephanvaningen on 2008-10-29
Click Applications, Accessories, Terminal
there copy/paste (Paste = Ctrl+Shift+V) the command

It will ask for your password and then it will start installing.

--
Suggestion: I would really advise to download an .iso of the Ubuntu Studio first and burn it to CD or make an USB startup disk (in Intrepid Ibex under System -> Administration -> Create a USB startup disk) and boot from that first. This makes sure you see the system before you actually do changes which can not easilly be undone.
