### Post by buzzingrobot on 2016-05-12
No, it isn't difficult.  The steps to do it are outlined in the release notes for 16.04:  [https://wiki.ubuntu.com/XenialXerus/ReleaseNotes](https://wiki.ubuntu.com/XenialXerus/ReleaseNotes)

If 14.04 is working well for you, there's certainly no rush to upgrade.  14.04 support continues for a few more years.  The release notes explain what's new/different in 16.04.  It's an evolutionary release on an established platform.

Fully update your current system before going for the release upgrade.

It's always smart to be prepared for things that aren't supposed to happen but do happen.  It wouldn't hurt to back up any files you absolutely cannot afford to lose.  Just in case.

Also, if you have added software from PPA's or other unofficial sources, it's a good precaution to remove all that before the update, and add it back afterwards.  You want to try get the system back to its base state before updating. When you add a PPA, you add software from it and you also add that PPA as a software source.  The updater will see it, just as it sees official Ubuntu sources.  There's at least some  potential risk that you might pull in incompatible packages from a PPA.

(If you aren't sure what all that last paragraph is talking about, relax and don't be in a hurry to upgrade. You'll pick things up as you use Ubuntu and ask questions here.)

### Post by DuckHook on 2016-05-12
It won't be any easier installing 2nd life into Xenial than into Trusty. So you are no further ahead there. There are very few cosmetic changes from Trusty to Xenial; they look almost the same. The biggest changes are all under the hood. Different start system, different tools, an alternative app installation process called snappy, lots more, but nothing that changes the fundamental user experience.

If you are curious, I would suggest that you:

1. Wait until most of the bugs have been worked out of Xenial, usually by the first point release (16.04.1)
2. Read up on dual boot so that you can have access to both OSes. That way, if one runs into trouble, you can always fall back on the other.

BTW, before you even think about upgrading, read the warning in my signature.

PS for help installing 2nd life, post a separate thread in the gaming forum.
