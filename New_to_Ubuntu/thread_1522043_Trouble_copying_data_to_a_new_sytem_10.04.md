### Post by lovinglinux on 2010-07-01
If you are running Firefox 3.6.6, try to delete the file **secmod.db **from your profile. Some users reported they couldn't start Firefox after updating to 3.6.6 and that deleting that file fixed the problem.

If that is not the case, then see [Fixing a problematic or corrupted profile](http://firefox-tutorials.blogspot.com/2010/05/fixing-problematic-or-corrupted-profile.html)
