### Post by otrojake on 2008-05-05
It depends on which part of the menu you're looking at.  Some of the italicized words aren't applications, but categories for applications.  For example, the Education section underneath Applications will only activate when there's an education program such as TuxPaint installed.  My guess is that's probably what you're clicking on that isn't accepting the select option, since they only work if there's an application installed for that category.
