### Post by varunendra on 2013-03-10
Welcome to the forums Mahesh ! :)

> I have **4gb of RAM** and about 320 GB of hard drive.Can i install the 64 bit of Ubuntu ?
The processor is Intel(R) Core(TM)2 Duo Processor **E8400** (3 GHz, 1333FSB, 6MB
Yes you can.

As for partitioning, if the system is going to be Ubuntu only, you can leave it to the installer.

However, if it is going to be a dual boot with windows, you need to be careful. Boot from the live cd, open a terminal (Ctrl + Alt + T) and post output of following command to get a precise help on that -
```
sudo fdisk -l
```

Also,
> Overall there is probably not much benefit to using 64-bit over 32-bit, unless you have a large amount of RAM that you wish to utilise.
I tend to disagree. There is enough advantage to prefer 64 bit if you can.
