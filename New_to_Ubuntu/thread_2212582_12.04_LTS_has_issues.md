### Post by Impavidus on 2014-03-22
Let's look at some further diagnostics.

Please run```
sudo apt-get update
sudo apt-get -f install
```in a terminal and post the output. One possibility is that installation of linux-headers-3.2.0-60 was aborted because the package manager was killed or the computer shutdown, in which case these commands should fix it. Another possibility is that you ran out of disk space or inodes, so to run these commands```
df -h
df -i
```and post the output to check this.
