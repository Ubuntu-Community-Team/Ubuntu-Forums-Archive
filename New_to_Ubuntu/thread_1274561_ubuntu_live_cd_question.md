### Post by QIII on 2009-09-24
Not likely -- _***IF***_ you have your BIOS set to boot from CD, you actually boot from CD and you don't put the CD in while you have Windows running.

When you run the LiveCD from boot, you are using that entirely independently of what might be on your hard drive.  The OS loads into memory directly from the CD and uses the CD much as it would use the hard drive (albiet much more SLOWLY than it would run from an HDD.)

### Post by 3rdalbum on 2009-09-29
> **stoneofshadow said:**
> if i use the version of ubuntu running from a cd on a windows computer with a lot of viruses will any of those viruses be able to detect anything i am doing on the ubuntu live cd portion?

Viruses are not magical, they are just computer programs. In your case, all the viruses are just Windows programs sitting on your hard disk, and like any other Windows programs they stop running when the computer turns off, and will not run unless the infected Windows is running.

The live CD works only with your computer's RAM, and doesn't save anything to hard disk (unless you install Ubuntu), so no trace of its use is left when you turn the computer off.

So, no. Nothing that you have installed in Windows will run on the Ubuntu live CD, unless you explicitly install it.
