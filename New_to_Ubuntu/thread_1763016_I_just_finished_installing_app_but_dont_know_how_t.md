### Post by garvinrick4 on 2011-05-19
Hit Windows key lower left of keyboard and the letter "a" at same time.
You can get to any "application" from there. If you know the name just type
in the first few letters and click on it. And in upper right of that box is a menu
item choosing from. Lots of things start with Window key, like with,a, s,d,w and such.
Really nice if you get used to using the key commands with left hand. Windows key 
and when you see the launcher with number on launcher opens that package.
Right click on anything new in launcher and you can keep it in launcher.  Play around
with it a while and read what you can. Here is a link to start you.
[http://ubuntuguide.org/wiki/Ubuntu:Natty](http://ubuntuguide.org/wiki/Ubuntu:Natty)
