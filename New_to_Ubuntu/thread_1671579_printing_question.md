### Post by audiomick on 2011-01-20
Go to the System menu, to 
System> administration> printer
you should get a little window with an icon for your printer in it. Right click on the printer icon, choose properties. There, if anywhere, will be a tag for printer options.

### Post by ajgreeny on 2011-01-20
Install the hplip-gui package which will give you a toolbox.  In there you can set the printer to use just the black cartridge.

However, if it is like my HP Deskjet F2180, it will work fine with only the black cartridge installed; no colour cartridge, so have a look in your documentation or the PDF file Manual, which is probably on the windows driver disk to see if yours is the same.

If it needs the colour cartridge(s) installed to work at all, you will possibly need to set the print mode in each application, if the hplip toolbox way does not work.
