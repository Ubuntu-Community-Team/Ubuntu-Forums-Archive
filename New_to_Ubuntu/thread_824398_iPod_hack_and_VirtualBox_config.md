### Post by jbrown96 on 2008-06-10
I have an iphone and have tried to connect it using Virtualbox. I know that the iphone uses a different usb sync format than is standard, and linux doesn't support it. I'm not sure about the nano, though. Why don't you try some of the great linux software to sync ipods. I personally love Amarok, but there's also Rythmbox (installed by default), GTKpod (just manages ipods), and Songbird (looks/behaves very similarly to itunes but also has firefox built-in). My suggestions are Amarok ```
sudo apt-get install amarok
``` and Songbird (download via their website)
