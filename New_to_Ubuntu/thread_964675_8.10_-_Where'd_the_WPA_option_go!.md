### Post by tarsius4 on 2008-11-01
Are you absolutely positive your wireless router is secured with wpa?  I ask this because I'm currently having great difficulty connecting to my wpa (not wpa2) network after a 8.4->8.10 upgrade.  When I attempt to select my connection from the network manager icon, after some time I am presented with a dialog box requesting Wireless Security and Password.  The Wireless Security box only contains the option "WPA and WPA2 Personal", so I expect that whatever option you receive may match the setting of your router.  This is different from what I remember: this particular message box used to give me several options.

In my case, I still cannot connect to the router, but when I right-click on the NM icon and select Edit Connections, I can manually change my network's profile to whatever security settings I desire.  The full list of security schemes available to me is: None, WEP 40/128-bit Key, WEP 128-bit Passphrase, LEAP, Dynamic WEP (802.1x), WPA & WPA2 Personal, WPA & WPA2 Enterprise.  Just for the sake of troubleshooting, I set the encryption to WEP 128-bit Passphrase and used my WPA passphrase... and when I next tried to connect, Ubuntu actually created a new profile with the same network name configured for WPA & WPA2 Personal security.  My suggestion is that you first check the "Edit Connections" dialog to see if you can manually configure a WPA connection there (just to prove that you have WPA "installed"), but next go to the router settings to double check its configuration.

Good luck,
Steve
