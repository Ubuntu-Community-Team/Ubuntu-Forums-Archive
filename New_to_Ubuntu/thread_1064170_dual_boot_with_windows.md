### Post by halitech on 2009-02-08
Ideally its better to put XP (or any Windows OS) first but if you don't want to reinstall Ubuntu, you can install windows and then use SuperGrub to restore Grub.  I've also heard (but never tried) of people using the windows loader to boot Linux as well

### Post by dougalkerr on 2009-02-08
Google your request to dual boot linux with windows. There is a great tutorial out there that will guide you through all you need to know and get grub back. Can't remember who does it but, I have used it and it works for Vista, XP and any Linux. You can also download and burn the Supergrub image to a small cd in case you loose any bootloaders. Supergrub will reinstall a bootloader from a choice of setups...
Good luck and have fun...
