### Post by Chesamo on 2011-01-18
Why do you need an ICQ number? The only things I've experienced coming from having an ICQ number is the million-and-one Russian spam bots. It's an aging service that will probably be discontinued in the next few years.

AIM is compatible and linked to the ICQ network, so if you want a number you can register an account [here](https://new.aol.com/productsweb/Controller.jpf?promocode=825028&ncid=txtlnkuswebr00000002) and make it a number.
