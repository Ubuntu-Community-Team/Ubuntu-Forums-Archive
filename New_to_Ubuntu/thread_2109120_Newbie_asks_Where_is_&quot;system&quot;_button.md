### Post by 3rdalbum on 2013-01-27
Don't follow instructions for old versions of Ubuntu - if you use 12.04, use instructions for 12.04 not for 11.10 or earlier.

Ubuntu changes very quickly and it no longer has a System menu. The equivalent is now the System Settings item in the "Cog Menu" at the top right corner of the screen.
