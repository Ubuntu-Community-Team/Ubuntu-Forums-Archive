### Post by seawolf167 on 2011-05-11
The server version is definitely stable. Though for more rock-solid stability you could always use the 10.04 LTS version (or Debian comes to mind... but this is a Ubuntu forums). I would do the following:

[LIST]
[*]redownload the .iso file
[*]check the MD5SUM to ensure the download is uncorrupted
[*]burn the iso to a dvd/cd on the *slowest possible speed setting available*
[*]boot off it, and you should be good to go
[/LIST]
