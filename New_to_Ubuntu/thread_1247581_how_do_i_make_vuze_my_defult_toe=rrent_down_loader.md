### Post by kelvin.illa on 2009-08-23
if what you meant was that you want vuze to be the default torrent client when opening a torrent file: 

just right click on a torrent file, choose 'properties,' find the tab wherein you can set the preferred 'open with' application (then choose vuze).

oh yeah--but if what you meant was the default torrent app after downloading a torrent file from (i would assume) firefox:

in firefox, go to edit, then preferences, then the apps tab. scroll though the file types, if you see a torrent file with a setting already, change it to your preferred app (vuze).
