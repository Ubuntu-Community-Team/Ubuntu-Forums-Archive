### Post by dcsoldschool53 on 2011-09-14
To read your web pages off-line, you can try adding your cup-pdf from synaptic package manager to print your web pages into PDF form. After that, you will need to create a PDF directory in your home user directory. This will send all pdfs that you will create to that folder. This will save you a lot of space.
