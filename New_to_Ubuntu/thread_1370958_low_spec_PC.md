### Post by gsmanners on 2010-01-02
Andy, your RAM is probably the bottleneck for your system. You should try using the "alternate" install disc (which can handle all the way down to 128 MB systems). Your installed system will still run slow, however, unless you use a desktop designed for low-end systems (like Xfce or LXDE). I would strongly suggest you try out the Xubuntu alternate CD.

Celeron is a really low spec CPU. It rates somewhere between a Pentium 2 and a P3. Now, a bare minimum requirement is for 486 and newer, but you really should have P4 or better.

### Post by QIII on 2010-01-02
Kernel panic is odd.  I think on a low-spec machine you'd just get nothing much to happen with the normal Ubuntu install disk.  Did you get a good burn?

I would suggest the Xubuntu alternate install disk.  But don't expect even Xubuntu to be particularly spunky.  It'll run, but you won't be smoking tires.

If you want to, you can also try crunchbang, Mint (as above), DSL (Damn Small Linux) among others.  DSL takes a bit of a learning curve, since you generally have to do things the old way by hand with DSL.

Take a swing by distrowatch.com and read up a bit so you can shop around and make an informed decision.
