I have the same card and it was almost painless setting it up. It was so easy I couldn't believe I did it untill I was on line. 
1. Click on the small antenna in the upper right.
2. select "New Mobile Broadband Connection"
then "Forward" => Select your country
then "Forward" => Select your provider (Verizon)
then "Forward" => Apply
You're done, and connected. Once set up this will start up when ever you start your system with the card installed.
