So now you have Ubuntu up with Gnome.

There are other DEs also to choose from such as KDE, XFCE and others. If you would like to try them, you could either install them from the software center. And once installed select the one you want to use from the login screen. Other way to try it would be create a live USB or CD of the distro with that DE built into it, like Kubuntu has KDE by default, Xubuntu has XFCE by default and so on.
