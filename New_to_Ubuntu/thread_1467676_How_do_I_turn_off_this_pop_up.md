### Post by Didius Falco on 2010-05-01
Keeping in mind the above warning about problems running scripts, you can set it to simply open by opening Nautilus and going to **Edit/Preferences** then go to the **Behavior** tab.  

In the middle of that tab there is a section labeled **Executable Text Files** with three radio buttons under it. Select the button next to **View executable text files when they are opened **and then close the menu.

After that, it will simply open the file in an editor instead of running it.
