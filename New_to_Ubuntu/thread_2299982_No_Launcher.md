### Post by deadflowr on 2015-10-22
Is it only the launcher, or is the top panel also missing?

If only the launcher is missing, could it possibly be auto-hiding?
You can test to see by slamming the mouse cursor against that side and hold it there for a few seconds.
if it shows, then you can reset(or set) it to always show in System Settings >>> Appearance / Behavior and toggle the auto-hide to off.

If no panel either then try the terminal command
```
unity --reset
```
