### Post by gradinaruvasile on 2009-10-10
Seamonkey 2.0 beta2 / pre. It is faster and snappier than firefox, and more stable , it has the same core as firefox 3.5.3. Has built-in mail-client (It is essentially Thunderbird bundled), IRC client. In the most part, supports the firefox extensions (not every one though).
 Recommended extensions: NoScript, AdBlocker Plus, SeaTab X(This one puts close tab buttons on every tab) and EnigMail (OpenPGP encryption support for the mailer component) - version 0.97a supports Seamonkey 2.0 series.
