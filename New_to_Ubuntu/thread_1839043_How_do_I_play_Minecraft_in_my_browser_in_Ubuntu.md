### Post by gdonwallace on 2011-09-07
I played in FireFox this last weekend.  Like your self I'm not a hard core player, I like to get on and just mess around with things for a few hours.

What I did was completely uninstall anything that was java related through synaptic.  Check the about:plugins in firefox to make sure there was nothing left that was the Linux version on java.

Then from the software center I installed java, and it worked fine.

There is a conflict between java and the Linux equivalent, they don't play well together.  

Doing the above seems to have worked for me.
