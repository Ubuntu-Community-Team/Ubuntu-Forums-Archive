### Post by sm_here on 2010-02-08
Hi Ken,

A .odt file can be opened by OpenOffice.org word processor only. If you have to send a document created using OpenOffice.org to a person who will use MS Office to view it, save the document in .doc format. 
So for the file that you have already created and want to send, open the file using OpenOffice.org word processor, click on Save-As and in the dialog box that opens up select Microsoft Word 97/2000/XP under the option File Type. You will now have a copy of the same file in .doc format. You can mail this now and the person will be able to view it.
