### Post by o770 on 2011-02-01
Hi there. I'm about to install [Handbrake]("https://launchpad.net/%7Estebbins/+archive/handbrake-releases"). Is there any important difference between either adding the PPA as a software source or just downloading and running the .deb package?

If no difference, may I consider best practice to download and run the .deb package when available instead of adding many ppa sources?
Thanks.

---

### Post by mcduck on 2011-02-01
Adding a software source will not only give you the current version of the package, but also allows it to be automatically updated through Ubuntu's package management, just lie the system itself and everything coming from the official repositories.

Installing the .deb package will just give you that exact version of the package, no updates.
