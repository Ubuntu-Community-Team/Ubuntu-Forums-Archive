### Post by LastDino on 2014-05-22
You can perform every single action mentioned there in GUI.
1.Right mouse click on what you downloaded, there should be extract option somewhere if you've extractor installed or your filemanager supports it by default.
2.Simply right click on the script and there should be dialogue box>go into properties>permissions>select you username>enable read and write> tick mark the option which says ''run as program''>close
3.Now double click on that script or right click and select execute.
