### Post by NightwishFan on 2008-05-09
If you have a 64bit processor, attempt to use the amd64, as almost all compatibility issues are fixed, and it is much faster. There would be no downside in using 32bit other than not having the benefits of the 64. If you have something like wireless you aren't sure will work, generally if it works in 32bit it will work in 64. If you are again unsure, just do some research before hand, but I strongly recommend Ubuntu 64.

### Post by goiggles on 2008-05-09
I also use the 32 bit version on an amd 64 bit processor. from my experience you don't gain a whole lot with the 64 bit version(a little speed in certain apps) but you also get issues with drivers and other annoyances it just doesn't seem worth it right now. but if you could afford to play around with both go ahead and try em both out.
