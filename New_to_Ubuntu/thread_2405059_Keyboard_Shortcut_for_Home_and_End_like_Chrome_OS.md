### Post by Geoffrey_Arndt on 2018-10-31
If using Unity DE . . . System Settings>Keyboard>Shortcuts Tab (may need to create "custom" keybinding if yours not available).   If using new Ubuntu (18.04/18.10), Gnome DE has similar settings program and there should be a keyboard customization tool.
