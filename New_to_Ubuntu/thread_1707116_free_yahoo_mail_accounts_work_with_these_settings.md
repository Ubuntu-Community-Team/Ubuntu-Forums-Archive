### Post by sean neilan on 2011-03-14
This was given to me so that I could get my evolution mail to work with my free yahoo mail account and it works great with these settings

server type= imap
server= imap.mail.yahoo.com
user= yahoo ID without @ yahoo.com
security= ssl encryption
Auth= password check remember password
receiving options tab= check for new messages in all folders

sending=
server type= smtp
server= smtp.mail.yahoo.com
check mark= server requires Auth
security= SSL encryption
type= plain
user name= yahoo ID without @yahoo.com
checkmark=remember password

now once you input these re-start your computer and it should work fine does for me with these settings
good luck and always have fun.
