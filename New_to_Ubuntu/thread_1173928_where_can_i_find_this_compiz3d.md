It's in **System/Administration/Synaptic Package Manager**. Just do a search for Compiz and you'll see a whole list of things. 

From the command line, if you want everything, just type ```
sudo apt-get install compiz*  simple-ccsm fusion-icon libdecoration0 emerald libemeraldengine0 python-compizconfig
```

That will get you Compiz, Compiz-config, all the plugins, Emerald Window Decorator and Compiz-icon - an icon you can put in your system tray to have easy access to configure Compiz and switch between Compiz and Metacity.

Have fun!
