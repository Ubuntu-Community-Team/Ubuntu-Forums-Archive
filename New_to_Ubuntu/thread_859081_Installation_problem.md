### Post by talsemgeest on 2008-07-14
You can try clicking on the advanced settings when you are about to start installation (after you have done all of the username and partitioning etc...).

Once there you can change the disk the grub gets installed to. Normally the default of (hd0) is fine, but you might want to try something else. If all else fails you can try the super grub disk: [http://www.supergrubdisk.org/](http://www.supergrubdisk.org/)

Hope this helps you out :)
