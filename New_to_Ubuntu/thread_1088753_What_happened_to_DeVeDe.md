### Post by Xiong Chiamiov on 2009-03-07
Have you tried using the same file as you've used before, to make sure that it's not a difference in files?

VLC will only pick up subtitles that are named the same as the video track by default.  You should really use a proper container, like .mkv, if you're going to be including softsubs with a video.  That, and use an mplayer frontend, because VLC tends to be really crappy with subs, in my experience.

You can view .isos with VLC before burning them to a DVD, if you're worried about going through discs while trying to figure things out.
