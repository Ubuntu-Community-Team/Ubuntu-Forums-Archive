### Post by spydeyrch on 2009-12-14
So if I understand you correctly, and forgive me if I don't, you can't see the icons in your gnome menu for docs, pics, vids, right?
 
I believe that there is a simple fix for that. I am not at my ubuntu machine right now so I am trying to go off of memory, forgive me if what I say isn't exactly the same as what you'll see on the screen.
 
Right click on your desktop -> go to the settings option for your desktop (where you can change the back ground, theme, desktop effects, etc.) -> on one of the tabs, I believe it is the interface tab (could be wrong on that name for the tab) there is an option to show menu icons. It is either a radio button or a check mark box. Make sure it is marked and see if that helps. :D
