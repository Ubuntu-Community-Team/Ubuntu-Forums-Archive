### Post by cariboo on 2010-10-08
My system plays m4a files (I just tried it) with no problems, the only thing I did was install the ubuntu-restricted-extras meta package.

You can install it from the command line:

```
sudo apt-get install ubuntu-restricted-extras
```

or either the Software Center or Synaptic Package manager
