### Post by Luggy on 2009-05-07
And I take it you are trying to install this on your Linux box too.

Well the answer is that Microsoft office will not 'natively' install on a Linux machine.

You can however get it working by using Wine. Check out the Wine subforum for more details.
[http://ubuntuforums.org/forumdisplay.php?f=313](http://ubuntuforums.org/forumdisplay.php?f=313)

There are applications that can work with Microsoft file formats (dunno about docx...). OpenOffice can read and write MS Office files and is very similar in use.

If OpenOffice doesn't work or you NEED to use MS Office, try to get it working in Wine.
