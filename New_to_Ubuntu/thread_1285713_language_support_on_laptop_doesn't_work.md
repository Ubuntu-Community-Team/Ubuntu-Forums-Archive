### Post by Kapitän Rotbart on 2009-10-08
Try System>Admin>Language Support and click on Install/Remove Languages...

If you install language support from here, you get system-wide language support (you can even make Ubuntu entirely in Greek, minus what hasn't been translated, on your next login if you want to). You'll get that language support for OOo, spellcheck everywhere including Evolution mail and Empathy (but not Pidgin, didn't get spellcheck for other languages working in Pidgin since Feisty). It's really the easiest way to install language packages via the Language Support. Enjoy!
