### Post by 123456789123456789123456 on 2009-06-18
facinating, as far as I know, it should automatically attempt to connect as soon as the desktop loads, if not, click on the network indicator, choose your network you want to connect to, and select auto connect.  if you are going to have a different network all the time, you should be able to set the network prefferences to auto connect to the first network it finds, however if it locates more than one network, and does not have a list of networks to connect to, it will probably hang, and ask you for which network to connect to.
