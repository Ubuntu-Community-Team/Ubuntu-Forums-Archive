### Post by Havacore on 2011-01-31
I'm a fairly new user of Ubuntu and when I first installed it I thought that having cairo-dock looked pretty cool.  Today I decided that it was just mostly getting in the way so I decided to uninstall it through the package manager.  When I first installed cairo-dock I got rid of my bottom panel and now I want it back so that when i minimize a window it'll be there.  Getting the panel back was easy enough, but now when I minimize, the window just goes into oblivion, never to be seen again!

So basically my question is, how can I have minimized windows  go to my bottom panel again?

---

### Post by mikewhatever on 2011-01-31
You need to add the Window List applet to the panel. Right click the panel, select Add and look for the applet near the bottom of the list.

---

### Post by nick_goodfate on 2011-01-31
But first you must add a new panel. Right click on top panel -> new panel. Then do what mikewhatever said.
