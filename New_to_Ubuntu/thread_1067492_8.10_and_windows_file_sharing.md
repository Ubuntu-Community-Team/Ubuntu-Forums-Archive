### Post by cariboo on 2009-02-11
The one thing to check is to see if all the computers are in the same workgroup. Depending on the XP version, Home uses MSHOME and XP Pro uses WORKGROUP by default. Ubuntu also uses workgroup by default. If you use a different workgroup name then make sure it is the same on all computers.

In Ubuntu just change the name of the workgroup in /etc/samba/smb.conf and restart samba. I'm sure you know how to do the same in Windows.

Jim
