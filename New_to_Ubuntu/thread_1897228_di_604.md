### Post by Buntumatic on 2011-12-18
Alright, then we need to know if the driver is loaded for your NIC.
First open a terminal and do ```
sudo -i
``` to gain root rights.
Then run ```
lspci -k
``` to see if the driver is loaded.
If it is, run ```
ifconfig -a
``` to see if your NIC has an IP address. If you are using DHCP it should be obtained automatically.
If it has an acceptable IP address on your subnet check if DNS is assigned by running ```
cat /etc/resolv.conf
```

### Post by MartijnNL on 2011-12-19
Looks like your network card doesn't have an IP address. Was your computer connected to your router (assuming you use a router) when you installed?

Are you using your router as a DHCP server (does it give IP addresses to your computer(s)). (This is a common case.) Or do you have to set your network details manually.

There should be a connection named 'Auto eth0'. If there isn't, can you try adding one? (Click on Add, name it 'Auto eth0', make sure 'Connect automatically' is checked and press apply. You might want to check 'Available to all users as well' if more than one user uses your computer.
