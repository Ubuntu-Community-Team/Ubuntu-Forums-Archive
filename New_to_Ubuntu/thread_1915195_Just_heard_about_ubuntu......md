### Post by amauk on 2012-01-25
Just bear in mind that Linux is not Windows

You cannot run a Playstation game on something that isn't a Playstation (not without emulation)
Same goes for Windows software on non-Windows systems

Linux has a wide array of audio production applications
Look in the software centre for what you require

but if you want to use your old windows software and have deadlines to meet, you may be happier sticking with what you know
