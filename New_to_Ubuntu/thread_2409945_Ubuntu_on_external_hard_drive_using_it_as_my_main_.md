### Post by Impavidus on 2019-01-09
Normally there should be no problem running an external hard drive for a long time and every active device produces some heat. The temperature should stabilise after a while, say 15 minutes or so.

If it gets excessively hot (maybe 20&#8304;C above the environment; hard drives shouldn't get very hot), this may be an indication that your external hard drive is failing. Not using it may make it last longer, but it will be useless.

Just make sure you've got good backups of all your valuable documents on different physical media.
