### Post by mcduck on 2009-06-02
Type "about:config" in the address bar, find the key "general.autoScroll", right-click it and select "Toggle" to set the key's value to "true".

---

### Post by cariboo on 2009-06-02
In firefox got to Edit-->Preferences-->Advanced and enable **use autoscrolling**
