### Post by coldraven on 2012-03-23
I only tried this in Ubuntu 10.10 but it should be similar in 11.10.
Right click on the desktop and select "Create Launcher"
Edit the launcher window
Title: My Homepage
Command: firefox "http://www.myhomepage.com"

Change myhomepage to whatever your address is. You can also change the browser and the icon if you have made one..

### Post by Bucky Ball on 2012-03-23
