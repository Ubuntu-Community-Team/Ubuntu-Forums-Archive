### Post by tommpogg on 2011-09-01
Are you sure you downloaded the right file?
You can download it from the official site, [here]("http://www.ubuntu.com/download/ubuntu/download").

Are you  using some program (like winrar) that let you open an iso like a normal zipped archive? In this case you actually have an iso file. Check the extension.

### Post by coffeecat on 2011-09-01
> **dannylortz said:**
> where is the iso the file i got was a zipper folder full of all sorts?

Is this "zipper folder" about 680-700MB? Then it's the ISO. Your Windows system is probably configured to see an ISO as a zipped archive. As tommpogg says, check the extension which is hidden by default in Windows.
