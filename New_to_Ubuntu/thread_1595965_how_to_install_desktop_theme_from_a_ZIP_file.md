### Post by SoulSmasher on 2010-10-13
firstly thanks for letting me know about this theme, awesome!

First of all, click on this link:
[http://dbgthekafu.deviantart.com/art/black-white-2-Vista-73277142](http://dbgthekafu.deviantart.com/art/black-white-2-Vista-73277142)
download the iconset,
right click -> extract here
right click on desktop -> change desktop background
go to themes tab, click install
go to the folder where you extracted. there is a readme file and and a tar.gz file, select the tar.gz file

Now go to this url:
[http://dbgthekafu.deviantart.com/art/black-white-2-Gloss-73274930](http://dbgthekafu.deviantart.com/art/black-white-2-Gloss-73274930)
Same procedure..

Now click on what you downloaded,
right click -> extract here

there are 2 tar.gz files and some emerald files

If you want to install the themes over emerald theme manager, first you must install emerald from application center.
then go to system->preferences->emerald theme manager
Click install and select emerald themes.
They won't be shown at first.
Run this on terminal:
```
emerald --replace
```
You'll see shiny new look

OR you should install metacity theme (which on this case i'd prefer)
To do that:
Right click on desktop -> change desktop background
go to themes tab, click install, and selct 2 tar.gz files which were in arcihve next to .emerald files

Enjoy!
