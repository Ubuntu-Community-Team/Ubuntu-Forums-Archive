### Post by Gone fishing on 2010-07-02
It sounds like you should be using IMAP rather than POP. 

IMAP leaves your messages on the server. On my system users can use a mail client like Thunderbird or use Webmail, for this I use an IMAP server as it allows them to see the same inbox where ever they are.

Outgoing mail is SMTP?

Obviously if you are to use IMAP on your evolution mail client, the server will need to support IMAP.
