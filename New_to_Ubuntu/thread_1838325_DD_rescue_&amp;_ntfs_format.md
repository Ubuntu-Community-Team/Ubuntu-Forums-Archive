### Post by Leshrac on 2011-09-03
Your best bet is probably to format the disk, make a clean install of whatever operating system you want to use on it and then copy the data you have saved to it.

Keep in mind that ddrescue won't be able to recover ever single file on your old drive if it is broken.

On installing windows without having the disks, I am not a lawyer but I don't think Microsoft cares about what disk you use to install their software as long as you have a license to use it, so you could probably download a "pirate" copy from a torrent site and use your license key to activate. As far as I know, that should be legal.
