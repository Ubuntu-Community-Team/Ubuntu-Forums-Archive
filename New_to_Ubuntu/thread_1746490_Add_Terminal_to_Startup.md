### Post by Dr. Nick on 2011-05-01
If you are using gnome it is "gnome-terminal" without the quotes. Just type that in where it says command and it should work

### Post by kaldor on 2011-05-01
Also, for reference, the path to programs in general is usually /usr/bin :)

Example, */usr/bin/gnome-terminal*
