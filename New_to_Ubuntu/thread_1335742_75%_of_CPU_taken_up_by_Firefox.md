### Post by sldavid on 2009-11-23
I have a suggestion that may help: install the NoScript Extension in Firefox.

I'm using an old Compaq Evo N1000c laptop (P4-M 1.8 GHz, 768 MB RAM) and the NoScript extension made a big difference for me. Originally, I installed it as a security precaution but I immediately noticed things ran much soother and quicker in Firefox. I suppose this is because all those CPU hungry scripts aren't running.
