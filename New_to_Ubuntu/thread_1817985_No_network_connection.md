### Post by grahammechanical on 2011-08-04
The wireless icon of four radiating arcs is used when Network Manager is searching for a connection. It only changes to the two arrows icon when the ethernet connection is made.

What messages do you get?

> Network disconnected you are now off line means that you have not enabled networking. Click the icon and click Enable Networking.

> Wired network disconnected you are now off line means (a) the ethernet cable is not plugged into the router or (b) the router is switched off. (c) Enable Networking is not ticked.

Also, you might confirm that networking is enabled in the BIOS.
