### Post by grahammechanical on 2015-09-09
If the manufacturer does not provide a method for updating the BIOS from Linux then the only sure and safe way to do it is to follow the manufacturers instructions and do it from Windows.

I have visited the Lenovo web site and for some Lenovo products there is a program that can be downloaded and run from a CD to update the BIOS. There is a BIOS update bootable CD ISO image for the Thinkpad. But I do not see any reference for the same kind of ISO image for the Yoga 3 Pro.
