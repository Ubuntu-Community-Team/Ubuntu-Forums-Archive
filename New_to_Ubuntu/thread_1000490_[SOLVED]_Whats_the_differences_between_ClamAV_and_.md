### Post by niteshifter on 2008-12-04
> **shiningkenmonster said:**
> ah i see. so basically they're they same but one has a GUI and the other is a command line.

Not same. The program which does the anti-virus work is clamav. One directs and controls it via typing on the command line.

clamtk allows one to control clamav via clicking and typing into a graphical interface, in essence clamtk does the typing for you. The program clamtk does to anti-virus protection whatsoever.
