### Post by lkraemer on 2009-11-29
Just open Synaptic Package Manager, and install HTMLDOC.

Then you will need to create an ICON in APPLICATIONS -> OFFICE
to be able to execute it via the ICON.

APPLICATIONS -> Right Click on menu to EDIT -> OFFICE
Add HTMLDOC

NAME: HTMLDOC
COMMAND: htmldoc
