### Post by grantk86 on 2009-01-14
For the most part R is command-line based only in Linux.  There is a GUI program called "rkward" that is still in development, if you'd like to try that.  To install it, open up a terminal in Ubuntu and enter the following:
```
sudo apt-get install rkward
```
I personally haven't used this program, so I can't tell you how much use it will actually be.
