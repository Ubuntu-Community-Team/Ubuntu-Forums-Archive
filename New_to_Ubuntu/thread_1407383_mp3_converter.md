### Post by Rhubarb on 2010-02-15
I think you are wanting a general sound format conversion application.
If this is so, try out soundconverter from the Ubuntu repositories - it's free.

```
sudo aptitude install soundconverter
```

I'm not sure if it uses lame or gstreamer backends, so you may also need to install those as well.
