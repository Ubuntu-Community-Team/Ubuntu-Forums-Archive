### Post by TheFu on 2019-07-19
Many of the infrastructure things that 10.04 used, while it was supported, are gone now.  For example, rc.d/ isn't used anymore.  Networking is entirely different, all the GUIs for desktops are different, DNS and name resolution are all different.

And nobody in the world has used 10.04 in over 4 yrs.

Basically, sounds like the exercises would be more helpful to an archeologist, not anyone trying to use computers today.

Personally, I miss 10.04 and consider it to be THE BEST Linux ever made.  It behaved like a Linux should before all the Windows programmers started migrating over to Linux and screwed up the infrastructure tools.  Most non-GUI changes since then have been to solve issues that I never had and to make things more complex that didn't need to be changed.



IMHO.
