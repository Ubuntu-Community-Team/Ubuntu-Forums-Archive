### Post by Bashing-om on 2013-10-26
wkinne; Hi ! Welcome to the forum .

In direct response as to (re-)installing, if that is the path you want to take ->
for instance, installing from the liveDVD;
Set in bios to boot the DVD as 1st boot priority;
Boot the liveDVD -> install ubuntu;
Choose "erase disk and install ubuntu" if this install is to be the sole operating system on the hard disk.
The install wizard will erase that hard disk, and set up a standard install with minimum input from you; Just enough to set up for your locale and set up a user account and related password. 
Done !
