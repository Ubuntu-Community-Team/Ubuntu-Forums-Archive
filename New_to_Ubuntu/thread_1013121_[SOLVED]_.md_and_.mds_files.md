### Post by Hospadar on 2008-12-16
You can also convert mdf files to iso files with the "mdf2iso" utility in System->Administration->Synaptic, then you can just extract the files with the standard archive manager (double click the iso, extract the files).

I assume you'll be using wine to actually install and run the game?  You may need a no-cd crack, emulating disks to the level required by copy-protection schemes is basically impossible in linux.
This may help:
[http://appdb.winehq.org/objectManager.php?sClass=version&iId=43](http://appdb.winehq.org/objectManager.php?sClass=version&iId=43)
