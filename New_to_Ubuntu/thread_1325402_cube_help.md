### Post by fargle on 2009-11-13
If you only have two desktops configured, you'll just see an "animated flip" effect, not a cube.  If you go to General Options and the Desktop Size tab and set Horizontal Virtual Size to 4 desktops, then you can use CTRL and ALT while dragging to see an actual cube.
