### Post by mc4man on 2014-03-29
Some dvds contain structure protection of varying sorts. In most cases Sp does not interfere with playback but occasionally some do.
In those cases maybe try vlc > Media > Disc >  No disc menus

If that doesn't work then you'll need to find the correct title # of the main movie & have vlc start there (Starting position > Title
(- easiest way to find correct title # is play dvd in a hardware dvd player, once movie starts check on title # thru whatever button on remote displays such info
