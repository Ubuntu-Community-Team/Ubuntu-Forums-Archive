### Post by perce on 2009-02-21
Does sound work with other applications? for exemple, try to play a CD or some audio file (you should have some in the directory Examples). If sound works with other application, on skype there is a menu to configure sound devices: click on the light blue S at the bottom left corner of skype's main windows, and choose options, it will open skype configuration window, then click on "sound devices". Do some tests, and good luck.

If you don't have sound with other applications either, or the above doesn't solve your problem, please open a terminal, type the command 
```
lspci
```
and post output, and the model of your computer.
