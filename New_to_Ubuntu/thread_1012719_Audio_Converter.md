### Post by Rhubarb on 2008-12-16
There's an excellent program called sound converter.

You can grab the package from Synaptic:  **soundconvert**

Or open up a terminal, and type this in:
```
sudo aptitude install soundconvert
```

You'll then find it in your Applications --> Sound & Video --> Sound converter
