### Post by Autodave on 2021-01-28
Sounds like a hardware problem, but do you know what kernel you are using?  This desktop would not connect to the internet when it was running any version of the 5.8 kernel.  If you are using the 5.8, try chosing a 5.4 kernel at boot and see if you can connect then.

One easy thing to try is this....and it works quite often......disconnect every cable from PC.  Remove battery and leave out for 10 minutes.  Reinstall battery and try.
