### Post by audiomick on 2012-04-22
The best advice I can offer is to organise a USB stick with an installer on it and use it for the "try without installing" option. This will give you a fair idea if 12.04 will run nicely on the machine, and also give the user a chance to look at Unity befor installing it.

Unity is different. You have to be prepared to learn a few things, but I got used to it fairly quickly. I think whether your potential user will come to terms with it or not will mostly depend on how he or she thinks. It suits some people very well, and others not at all.

### Post by Paqman on 2012-04-22
If your graphics card is a bit puny I would strongly suggest using Unity 2D instead of the default 3D version. Unity 2D runs well on very modest graphics cards, and is virtually identical to the 3D.

You can choose to log into Unity 2D by clicking on the wee cog by your username when you log in.
