### Post by mglennpooh on 2011-10-22
Go to Software Center and type adobe - you should get two options for adobe flash plugin - one is the older version 10, the other is the Installer for the new version you downloaded earlier. If you happen to have your browser engaged, I recommend closing it before you install or else it will take three times as long to complete, then, click install - done!

You can do the same thing for Adobe Reader except they don't update for Linux as often as they do for Windows so don't be surprised when you discover their version for Linux is 9 vs 10 (or 11).

Google Chrome & Chromium already have Flash & Update built in - no need to do anything here.

You can do the same for other Canonical Partners but not Independents (Non-Partners) - you'll have to use the Download Folder and the Command Terminal - William Shotts has a free pdf download for commands via his website linuxcommand.org, and many others. I recommend Mark Sobell's Practical Guide to Ubuntu (10.04) or Fedora(15). Linux based Amazon.com has better prices than most, plus a 100% free app "Kindle for PC" or "Kindle for Mac" for mobi-based ebooks.
