### Post by oldfred on 2011-01-02
Some of the discussion above is about grub legacy and you have grub2.

You have installed windows in sdb5 and sdb1 is the extended partition. You should always install windows in primary NTFS partitions with the boot flag. You must have had another install of windows or else windows would not let you install to a logical. It then puts its boot files in the primary, or you copied the partition from a primary to a logical. Because the boot sector says the partition starts at sector 63, I assume you copied the windows partition. 

Windows officially will not boot from a logical partition. You can have another windows partition with the boot files and then the install in the primary will boot the second install.

Since you do not have any other partitions on sdb, I would just reinstall to a primary partition. I do have links to ole-timers here who can change logical to primary partitions by directly editing partition table, but it is not for those who do not want to risk corruption. I also have links for making XP boot from a logical, but you have to use grub or lilo and update the partition boot sector. Since you have available primary partitions that would be the best choice.
