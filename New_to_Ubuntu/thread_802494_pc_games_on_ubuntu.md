### Post by NullHead on 2008-05-21
You can use Wine witch is a free windows compatibility layer for linux :D You can install it by issuing this command in a terminal window: ```
sudo apt-get install wine
``` Then you can configure wine from the menu and then you should be able to run diablo 2 ... it depends on if it works or not with this program. 

Check here to see if diablo2 runs in wine [http://appdb.winehq.org/](http://appdb.winehq.org/)

### Post by dorath on 2008-05-21
WINE can also be installed through Add/Remove on the Applications menu.  Just search for "wine" (without the quotes), check the box for 'Wine Windows Emulator', and click Apply Changes.

There is an Ubuntu forum dedicated to WINE, it can be found [here]("http://ubuntuforums.org/forumdisplay.php?f=313").
