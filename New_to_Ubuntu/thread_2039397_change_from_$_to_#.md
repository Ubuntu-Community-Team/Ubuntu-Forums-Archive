### Post by sisco311 on 2012-08-08
When executing interactively, the shell displays the primary prompt when it is ready to read a command. 

Traditionally, a shell prompt ends with $, % or #. 

$ indicates a shell which is compatible with the Bourne shell, like a POSIX shell or Korn shell or BASH (the default user shell in Ubuntu). 

% indicates a C shell (csh or tcsh). 

If the prompt ends with a #, this indicates that the shell is running as the superuser (root).
