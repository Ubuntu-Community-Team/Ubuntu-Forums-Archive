### Post by JBAlaska on 2010-10-07
You will want to use archive manager (file roller) to un-rar those movies (avi) and subtitles (srt). It should work by default, just double click on the correct rar. This is where it can get a bit confusing, Usually it's the last file in the set (with the .rar extension) but if the person uses the "new" method to rar the set, it's the first one in the set (with the .r00 extension).

Once you double click on the correct file in the set, simply drag the file(s) shown in archive manager to a location that you want them extracted in.
