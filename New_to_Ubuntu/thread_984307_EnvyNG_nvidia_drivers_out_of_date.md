### Post by inigomontoya on 2008-11-16
I'm not sure why anyone even uses Envy.  For NVidia cards the install is really easy.  The latest driver is 180.06 beta. The latest stable driver is 177.82.

More info is available in this forum. [http://www.nvnews.net/vbulletin/forumdisplay.php?f=14](http://www.nvnews.net/vbulletin/forumdisplay.php?f=14).  The NVidia linux driver developers frequently post there.  

I'm currently using the 180.06 betas.  They are very fast and trouble free.
