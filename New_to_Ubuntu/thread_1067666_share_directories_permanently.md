### Post by Metallion on 2009-02-12
Are you using gnome? Then you should just be able to add your shared by right clicking on them and choose sharing options in nautilus.

If you did it like that before, it would make sence that the shares are not in smb.conf. Nautilus places its shares in /var/lib/samba/usershares/ and creates a file for each share.

You could try adding these to smb.conf manually. Like if you're trying to share a directory called Documents, there will be a file /var/lib/samba/usershares/Documents

Open smb.conf and add the line [Documents]. Now under that, add the contents of the file mentioned above and see if that works.
