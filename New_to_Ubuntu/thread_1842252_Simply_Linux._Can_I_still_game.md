### Post by waynefoutz on 2011-09-11
I've found PlayonLinux to be a great tool, it's easy and it's free. You're probably not going to play cutting edge Windows games, but Ubuntu will run a surprising amount of Windows games. What works and what won't just depends on a lot of different factors.

Try this person's Youtube channel:
[http://www.youtube.com/user/Jakejw93]("http://www.youtube.com/user/Jakejw93")
He has videos of hundreds of games, most of them Windows games, but some are native, as well as notes explaining how to install them so they run properly.
