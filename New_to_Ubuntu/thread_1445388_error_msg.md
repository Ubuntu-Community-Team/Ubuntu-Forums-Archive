### Post by Shazaam on 2010-04-02
If you are using a non-Ubuntu (third party) repository in Synaptic to download software you probably haven't added their gpg key. Synaptic will print out an error message but you can still get the software. Most reputable web sites will have instructions on adding their key.
Read this...
[https://help.ubuntu.com/9.10/add-applications/C/adding-repos.html](https://help.ubuntu.com/9.10/add-applications/C/adding-repos.html)
