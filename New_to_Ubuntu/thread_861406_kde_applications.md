### Post by snowpine on 2008-07-16
> **azery said:**
> I'm using ubuntu.Can I run KDE applications, such as digikam under gnome? Or do I have to switch to kubuntu?

I am not familiar with digikam, but generally speaking, yes. When you install an application (using one of several methods), the system automatically installs any other packages on which that application depends.

If it's your first KDE application, there might be quite a few packages installed; if you've used other KDE apps before, the list may be shorter.

Personally, I love this system! :)
