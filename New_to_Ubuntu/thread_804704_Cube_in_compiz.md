### Post by perillux on 2008-05-23
well you should still be able to rotate it even if there are just 2 sides.  So assuming that you are in fact able to see it rotate in 3D with the 2 sides, then all you should have to do is right click on the "screen changing applet" at the bottom right panel, right in front of the trash.
So right click it and go to preferences and increase the number of workspaces.

OR, if that is gone for any reason then just open up the CompizConfig settings manager:  System > preferences > advanced desktop effects settings.

Under the General category click general options, then click the desktop size tab.

### Post by sisco311 on 2008-05-23
Open compizconfig-settings-manager and go to General Options, Desktop Size and increase the Horizontal Virtual Size.

To install ccsm from the terminal:
```
sudo apt-get install compizconfig-settings-manager
```run it:
Applications -> Settings -> ??? Desktop Effects ??? or from the terminal:
```
ccsm
```
