### Post by AnPar on 2010-09-14
In Skype you go to Options > General and see if "Start Skype minimized in the system tray" is unchecked.
If you want to know more about Startup Applications have a look at [https://help.ubuntu.com/community/AddingProgramToSessionStartup](https://help.ubuntu.com/community/AddingProgramToSessionStartup)
Hope this helps!
