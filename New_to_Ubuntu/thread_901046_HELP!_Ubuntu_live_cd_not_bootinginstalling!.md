### Post by thumper13 on 2008-08-26
My first suspect is that your disc image is corrupt. Did you check the MD5 tag (I think that's what they're called, a big list of letters and numbers that you match to ensure the download is perfect) to ensure the disc image is perfect? Thats the advanced technical answer. The short version is try downloading the .iso a second time, or else you could get a copy from Canonical in the mail (for free). These lovely people provide the OS for free, on discs already, so you know they're perfect.

Apart from that, burn the image as slowly as your burner goes. You ARE burning an operating system, not a backup or anything. Also, if you're burning in Windows, most software comes with an option to verify written data, which would be the same as comparing MD5 tags.

I'd definitely try re-downloading and re-burning the disc first though, save the headaches. Or if time is not an issue, get the disc from Canonical. It takes a few weeks to ship, but I always get them if I'm installing it as my sole OS, to make sure it's perfect.

Hope it helps.
