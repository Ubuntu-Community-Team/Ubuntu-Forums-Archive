### Post by Miljet on 2011-12-08
All operating systems require some type of boot loader to direct the machine to the files required for starting the computer. Ubuntu uses GRUB (GRand Unified Boot loader).  If you have been using Ubuntu since summer, you have been using GRUB since summer.
