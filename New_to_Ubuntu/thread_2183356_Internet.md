I'm running Ubuntu Unity 12.04LTS amd64 - *64bit*, on a Toshiba Satellite c655-s5503.

this will speed up your system response, and Firefox has a history - search function of it's own, so it's kinda reduntant to duplicate that funtion through Dash.
