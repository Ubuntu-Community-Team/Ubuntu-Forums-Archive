sudo apt-get install os-prober
``` then ```
sudo update -grub
``` and windows, or other linux OSs will suddenly appear in the menu.

### Post by oldsoundguy on 2011-04-09
Start with your burned ISO of Ubuntu (you DID burn it at the slowest speed possible, closed it out and verified the burn, did you not?)
