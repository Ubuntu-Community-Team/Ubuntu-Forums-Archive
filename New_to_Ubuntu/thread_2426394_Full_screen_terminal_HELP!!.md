### Post by CatKiller on 2019-09-08
How you configure each terminal application depends on which one you're using: gnome-terminal vs konsole vs whatever. Most of them have a profile setting, though.

If you want a fullscreen command line, though, you can just access one of the virtual TTYs with Ctrl-Alt-F2 to -F7.
