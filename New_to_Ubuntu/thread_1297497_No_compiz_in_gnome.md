### Post by martrn on 2009-10-21
I don't belive your graphics driver is set up properly.  You need to install the graphics drivers (kernel modules), as described by your video card manufacture and install these graphics drivers the non-recommended way in the ubutnu documentation.

This is not recommended because during such process you can be left with malfunctioning graphical device drivers, a corrupted X11.org file, and/or several annoyances with failsafeX.

See : [https://help.ubuntu.com/community/Video]("https://help.ubuntu.com/community/Video")

Good luck.
