### Post by dca on 2008-08-01
If you're using Ubuntu w/ Gnome desktop, go to the 'Add/Remove Software' option under your 'Applications' menu from main menu bar.  In search, type in 'messenger' and make sure you have it set to 'All Available Applications' in the drop-down dealie...  After the search is complete you should see quite a few options.  Before installing them, jot down their names and research using Google finding their sites to see if that's what you need...
