### Post by Kellemora on 2008-12-26
Hi SMJ

If you KNOW the particular font set they are using, you could download and install that font in /usr/share/fonts/truetype if they are truetype fonts, else in a font folder for that type of font.

Also, it could be the font association file is pointing to a different font set than the one requested by the browser.
