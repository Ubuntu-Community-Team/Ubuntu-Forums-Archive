### Post by Paqman on 2011-06-05
Yep, just install the regular version. However, if your machine isn't capable of running the new Unity desktop (due to having no 3D graphics) then the fallback alternative is the classic Ubuntu desktop, which isn't great on a small screen. You can solve this by installing the package unity-2D from the Software Centre, log out, then select "Unity 2D" from the session menu at the bottom of the screen when you go to log in.
