### Post by azurepancake on 2008-06-16
This guide helped me a lot for setting up Samba shares on our offices Ubuntu server [http://samba.netfirms.com/](http://samba.netfirms.com/)

Ubuntu Server edition is nice because during installation, you can choose "presets" such as "LAMP server", "Samba server", "OpenSSH" server and a few others. These presets will install required packages and services needed to run your server.

It also lacks a GUI for optimization. If you want hands on shell experience, this is the way to go!

Most importantly, have fun with it.
