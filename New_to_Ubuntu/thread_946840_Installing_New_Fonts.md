### Post by SunnyRabbiera on 2008-10-13
There are no dumb questions.
Anyhow the best way I know to get new fonts in Ubuntu is to create a .fonts folder by going into your home directory and right clicking and selecting "create new folder"
and name your new directory named .fonts.
The new folder will be made hidden so to unhide it hit control-h and then add the fonts you need to it.
