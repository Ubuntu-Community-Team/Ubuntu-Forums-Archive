### Post by yancek on 2016-02-14
If your only operating system is windows 7, you can get software designed specifically for this purpose.  The two most commonly used which will run on windows are "unetbootin" and "pendrivelinux".  Just enter either in the search engine of your  browser and you should get the site.  Make sure if you get unetbootin, you select the version which will run on windows.  Follow the instructions and after the bootable flash drive is created, reboot the computer and access the BIOS setup.  This varies so watch the screen on boot for a message to enter setup and hit that key.  You should get a window with several tabs and you need the one which says Boot.  Look for boot priority there and set it to whatever the flash drive is, save and reboot.  Take a look at the link below which has detailed instructions on dual booting beginning about half way down the page.  If you have never done this before, it would be a good idea to read the entire page before starting.
