### Post by patchwork on 2010-02-09
I'm assuming you restarted the X server after reverting your xorg.conf?   

Have you checked your Xorg.0.log for any clues (i.e. what resolution is being attempted, and errors)?  FWIW, adding the line 

```
Option "ModeDebug" "True"
```
 to the Device section of your xorg.conf will create a more verbose log file after attempting to restart X

If you have the program hwinfo installed, can you please post the output of 

```
hwinfo | grep fglrx
```It should appear similar to this (though I am using nvidia)
```
