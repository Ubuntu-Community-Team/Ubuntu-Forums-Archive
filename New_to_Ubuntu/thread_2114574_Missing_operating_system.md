### Post by Cheesemill on 2013-02-10
You need to use an application to correctly write the ISO image to a USB stick, you can't just copy it over.

[http://www.ubuntu.com/download/help/create-a-usb-stick-on-windows](http://www.ubuntu.com/download/help/create-a-usb-stick-on-windows)

You also need to change the settings in your BIOS so that the machine boots from the USB stick first.
