### Post by grahammechanical on 2021-11-10
There was a period of transition where we had a Ubuntu Software (Gnome Software) to install deb packaged applications and also a Snap Store to install snap packaged applications. The two "stores" have now been merged.  I not sure if Ubuntu Software contains all the snap packaged applications available. It does have both deb and snap versions of Firefox and Thunderbird. Some applications are only available as a snap. Chromium web browser and Zoom-client are two that I know about.
