### Post by jerenept on 2010-08-01
[http://cdimage.ubuntu.com/ports/releases/lucid/release/]("http://cdimage.ubuntu.com/ports/releases/lucid/release/")

select "PowerPC Mac" or something to that effect. You need to burn this cd image onto a CD using Infra Recorder[http://infrarecorder.org/]("http://infrarecorder.org/") on Windows, Brasero on ubuntu or Disk Utility on OSX.

You then install Ubuntu from the CD.
Insert the CD into the drive, and, when the computer is booting up, press "C" on the keyboard. This should get you booting up from the Ubuntu CD and ready to install.

This will also be of great assistance: [https://help.ubuntu.com/community/Installation/]("https://help.ubuntu.com/community/Installation/")
