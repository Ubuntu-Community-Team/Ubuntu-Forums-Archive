### Post by earthpigg on 2010-08-18
hi,

option one is to use firefox addons.

option two is to manually copy the videos from firefox's temporary folder before firefox deletes them.

for option two:

1) start firefox, go to the video you want to download, and let it load completely. leave firefox open.

2) places -> home folder

3) view -> show hidden files

4) firefox -> random-letters-and-numbers.default -> cache

5) view -> list

6) click 'date modified' to sort by date modified, with most recent at top

7) copy the flash videos out of FF's temp folder
