### Post by arochester on 2009-05-03
You can only upgrade from one version to the next - for example, 7.04 to 7.10, 7.10 to 8.04 - and not from 7.04 to say 9.04. Trying to upgrade across several versions will most likely end with problems.

A new, clean installation is best - even though you will lose your settings.

### Post by drs305 on 2009-05-03
wiilliis,

Welcome to the ubuntu forums. 

If you want to save all your configuration settings you must either do a "network" upgrade online or for an install from a CD have a separate /home partition which you designate during the partitioning set up and *do not format*.

For the network upgrade, as was mentioned, you can only advance one release at a time.
