### Post by Bölvaður on 2009-09-13
no question is a stupid question if you dont know the answer :KS


Here is something that is useful to know about making launchers.

if you select the (binary) file you want to execute, then copying the file and pasting it into the box next to 'command' will make... :D well just try it.

For icons you can download a random image and paste into the image path, the same way as with the command.


You can put more than just paths there, you can put any command in there, so if you want to open firefox on a special webpage then "firefox www.xkcd.com" or to mount a remote (nfs) harddisk... any command at all (note that you can not see the output as you dont have a terminal open)


hope this helped anyone.
