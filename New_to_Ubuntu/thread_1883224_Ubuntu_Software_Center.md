Please note that LSC is STILL under development hence there is no final release yet. It's not even yet in the official repositories. 

Did you follow these instriuctions on [http://ubuntuforums.org/showthread.php?t=1880394](http://ubuntuforums.org/showthread.php?t=1880394)

To Install Lubuntu Software Center:

From LXTerminal, please run these commands:

Code:

sudo apt-add-repository ppa:lubuntu-desktop/ppa

Code:

sudo apt-get update

Code:

sudo apt-get install lubuntu-software-center
