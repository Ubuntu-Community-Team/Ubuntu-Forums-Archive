### Post by mastablasta on 2011-01-25
as they mentione there is a good MSN linux alternative called aMSN
 
sticking with LTS you will get mostly security and system updates but likely not software updates and upgrades. Software updates can still be done (but in a different way). if you go with later editions things can work fine on start but then there could be issues later on as they implement new features to the OS and are sort of testing system response, users response and all... :-)
For example the next version commin out in april will be totally different as it will use new desktop "manager" (whatever) called Unity. At the moment it is still being developed and netbook users had quite a bit of problems with it. so i see plenty new issues for users on release. luckilly they will include the classic desktop as well. but that one might have new issues as well. while LTS will stay as it is. 
 
i using a linux computer for work as well (actually my wife does) and i want it to be as stable as possible. so i run LTS edition on both of her computers. I will see later if i need to upgrade system (new software feature, some new solutions, imporvement in graphics drivers...) but so far it seems all is working fine and upgrade is not really necessary.
