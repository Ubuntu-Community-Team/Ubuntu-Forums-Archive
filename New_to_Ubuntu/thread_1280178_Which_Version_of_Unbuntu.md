### Post by yknivag on 2009-10-01
I guess it depends what you mean by version.

If you're referring to the version number, then 9.04 is the latest stable version.  If you're referring to the DE type (ie Ubuntu, Xubuntu, Kubuntu etc) then I'd stick with standard Ubuntu and Gnome in the first instance (this seems to be what Sopcast is written for).  It's also where the most support can be found (here and elsewhere).

EDIT: Mind you, for that kind of hardware, you might be better off with a much lighter distribution. Something like "Damn Small Linux" or "Puppy".

### Post by snowpine on 2009-10-01
Hi Galwayguy,

Here are the recommended minimum requirements for Ubuntu:

> Recommended minimum requirements

Ubuntu should run reasonably well on a computer with the following minimum hardware specification. However, features such as visual effects may not run smoothly.

    * 700 MHz x86 processor
    * 384 MB of system memory (RAM)
    * 8 GB of disk space
    * Graphics card capable of 1024x768 resolution 

Based on your system specs, I would not recommend Ubuntu. There are other Linux distributions that are better suited for older hardware; Puppy Linux is probably the most popular in that category. If you can get ahold of a newer computer (or even a ram upgrade on your existing hardware), you will have a better candidate for Ubuntu. Good luck!

### Post by Sef on 2009-10-01
Xubuntu System Requirements:

> You need 192 MB RAM to run the Live CD or 128 MB RAM to install. The Alternate Install CD only requires you to have 64 MB RAM at install time. To install Xubuntu, you need 1.5 GB of free space on your hard disk. Once installed, Xubuntu can run with starting from 192 (or even just 128) MB RAM, but it is strongly recommended to have at least 256 MB RAM.

So I would either get more ram, or as erilidon said in post 2, get DSL.  Also check out [Puppy Linux]("http://puppylinux.org") and [DeliLinux]("http://delilinux.org").
