There is a C/C++ plugin that should be available in the Eclipse plugin manager (or whatever it is called - I don't have Eclipse installed at the moment).

However, if you are beginning with C, I'd recommend Geany instead. Eclipse is a great development environment for Java with lots of rich features but you'll find your way around Geany much more easily. I've used Eclipse a lot but if I'm writing C, I use Geany.
