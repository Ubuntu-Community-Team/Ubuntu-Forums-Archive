### Post by Paddy Landau on 2011-10-21
You should have downloaded the .deb file. If not, you have downloaded the wrong file. Go to [the download screen]("http://www.avast.com/linux-home-edition#tab4") and download "avast! Linux Edition (DEB package)".

Go to your download folder and double-click the downloaded .deb file. It should then install. (You will need enough RAM.)

Once installed, press the Super key (that's your Windows key) and start typing *avast*. An Avast icon should appear; click it once, and Avast should open.
