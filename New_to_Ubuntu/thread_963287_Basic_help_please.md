### Post by Tony Flury on 2008-10-30
Use Windows to burn the iso file onto a CD-ROM (esentially an iso file is a CD-ROM image), and then boot your PC with that CD-ROM.

One of the options when you boot will be to install Unbuntu.

### Post by hellion0 on 2008-10-30
The ISO file is (normally) meant to be burned to a CD. Windows Help is not involved at all.

Go [here](https://help.ubuntu.com/community/BurningIsoHowto) for instructions on burning the ISO file to a disk.

Then you need to put the CD in the tray and restart your computer. It should start from the CD and put you into a working Ubuntu system.

### Post by random turnip on 2008-10-30
If you're just looking to try Ubuntu without installing, when the disk loads press enter when "Try Ubuntu with no change to my computer" is highlighted, this will then load up Ubuntu to how it will be when installed so you cann have a quick look round.  Bu this will run much slower than Ubuntu will run after a full installation, and things like the screen res may not have the correct settings.

However, before doing this i would wait for the release of 8.10, which (in theory) should be in about half an hour, as there would be no point in installing something that is going to go out of dat eon the same day of installation XD.
