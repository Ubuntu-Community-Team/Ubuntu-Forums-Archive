### Post by UnknownFear on 2010-05-11
It comes installed, meaning it's already installed and ready to go.

All you have to do is open a Terminal and type python, like so:

```
unknownfear@ubuntu:~$ python
Python 2.6.5 (r265:79063, Apr 16 2010, 13:09:56) 
[GCC 4.4.3] on linux2
Type "help", "copyright", "credits" or "license" for more information.
>>>
```

If you want help with Python, take a look at the [Programming Talk]("http://ubuntuforums.org/forumdisplay.php?f=39") section of the forums.

And, as always, have fun :)
