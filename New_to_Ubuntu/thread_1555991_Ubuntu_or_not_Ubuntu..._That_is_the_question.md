### Post by Baba-Juju on 2010-08-18
Ubuntu 10.04 worked out of the box regarding hardware with my Dell Inspiron 15. I just needed to install some software drivers. This goes automaticly: For example, I browse a website with flash. Ubuntu provides me the freedom to choose to download and install an open source or proprietary driver in order for Ubuntu to be able to play flash.

I still own a compaq presario 900 and the only issue I had with Ubuntu 8.04 was finding a driver for my wireless PCI card. And with a little help of my Ubuntu friends (forums/wikis/google) found a solution.

All in it's worth to try it. I did not regret it. You can test drive it first with live cd/usb or make a dual boot. You choose!
