### Post by Michael Dooley on 2013-01-09
I assume that you are talking about adding "Force Quit" to the upper panel. To do that, one must press the Alt key and then right-click on the panel. A dialog box should appear with "Add to Panel..." as the first option. Click on "Add to Panel..." and a subsequent dialog box opens with Force Quit as one of the selections. Please report back if this doesn't behave as I described.
