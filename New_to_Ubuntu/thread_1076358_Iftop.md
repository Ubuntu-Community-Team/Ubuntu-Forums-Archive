### Post by yeats on 2009-02-21
Looks like this runs in the terminal.  Try opening a terminal and typing

```
sudo iftop
```

If you're after general system monitors, plain old "top" is the best.  Just type

```
top
```

in your terminal.  "q" exits out - fyi.
