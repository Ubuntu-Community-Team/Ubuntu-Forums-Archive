### Post by DFlame on 2009-10-23
1. First of all, you don't need to install it to try it. The OS will run from the CD for as long as you need to make the decision whether to install or not.

Installing a [dualboot](https://help.ubuntu.com/community/WindowsDualBoot) will allow you to swap between Windows and Ubuntu as much as you like. You will be given a choice of which one to use when the computer starts.

2. The partition containing Windows and your documents etc will be available from within Ubuntu. However, this will not work in reverse.

### Post by halitech on 2009-10-23
WUBI doesn't do a "true" install so you won't have any issues unless your windows install dies. WUBI uses a loopback file system inside a file to give a quasi-filesystem that works like a filesystem. A true dualboot means you actually have 2 seperate filesystems on 1 drive (normally) that can run each system independant of the other so it 1 dies, the other stays working.
