### Post by Sarah L on 2008-09-01
Open up Pidgin's buddy list.
Go to Tools > Preferences > Smiley Themes > Add.

However, before you can do this, you need a Smiley Theme first.
These exist on the Internet.
If you can't find the one you want, you can make one according to the directions at [http://developer.pidgin.im/wiki/SmileyThemes]("http://developer.pidgin.im/wiki/SmileyThemes")

Hope this helps,
Sarah

### Post by bloodchilde on 2009-01-14
Well, what imgkg means is that he/she is not able to save custom emoticons from others via right click -> save. That is one of the reasons Pidgin still sucks after all these years, as an MSN client. Who uses custom smiley themes anyway? The only good thing about Pidgin is that it's GTK based and it supports multiple protocols. But forget about custom emoticons (animated or not), mouse-written messages, MSN nudges (winks etc.), audio/video chat, and games. If you want a REAL client (slow and buggy though), go for the aMSN. If you want a step in the right direction, use Emesene.
