### Post by HereInOz on 2011-05-06
Can be one of four things:

A bad CD 
A bad CD drive
A bad HDD on the computer
Bad hardware (RAM, for instance) on the computer.

Try the CD in another computer if you can, and if you are actually going to install Ubuntu and not just try it, download the alternate install and run that.

Was the CD burnt in the same drive from which you are attempting the install?  If not, there could be tolerance incompatibilities between the drive which created the CD and the one trying to read it.  You may find that it works fine in another machine, and a CD burnt on a different computer could work fine in the one which is giving trouble.

Sorry for the lack of specifics.  Best I can do.
