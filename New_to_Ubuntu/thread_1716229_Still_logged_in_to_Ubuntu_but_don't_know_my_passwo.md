### Post by nothingspecial on 2011-03-28
If you don't know your password you won't be able to change it.
Restart and hold down shift.

When the menu comes up choose recovery mode/root shell (or whatever they call it at the moment)

Type

```
passwd ryan
```

Change ryan for ypour real username (ryan was a guess)

Then type your new password.

You can leave all tose telephone number etc bits blank.

Type it again then type

exit
