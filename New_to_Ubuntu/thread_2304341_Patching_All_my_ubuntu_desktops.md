### Post by mastablasta on 2015-11-26
probably they mean updates managed from central server. if that is the case then there is Landscape: [http://www.ubuntu.com/management/landscape-features](http://www.ubuntu.com/management/landscape-features)

or if you do not need a server you can set up automatic updates on all machines: [https://help.ubuntu.com/lts/serverguide/automatic-updates.html](https://help.ubuntu.com/lts/serverguide/automatic-updates.html)

similar to landscape but free is [http://theforeman.org/](http://theforeman.org/)

as for setting up to use same repositories: [https://help.ubuntu.com/community/Repositories/Ubuntu](https://help.ubuntu.com/community/Repositories/Ubuntu)
