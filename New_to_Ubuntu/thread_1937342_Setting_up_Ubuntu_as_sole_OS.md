### Post by ubudog on 2012-03-07
Once you get the .iso file downloaded in Windows, use a program like Windows Disc Image Burner to burn the .iso onto the CD.  (Not Copy Pasting the .iso onto the CD).

Once that is done, and you have all your stuff backed up, reboot and tell your BIOS to boot off of the CD.  (Normally F12 or ESC on start)

Select "Install Ubuntu" using the arrow keys.

Then, let it boot up and follow the on screen instructions on how to install.  If you want to wipe out Windows and have Ubuntu as the SOLE operating system, be sure to select "Entire Disk" in the Partitioning screen of the installer.

Note: This will remove all your data on the computer, so be sure to have backups of important files you'd like to keep.
