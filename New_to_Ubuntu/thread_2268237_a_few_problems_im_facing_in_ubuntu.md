### Post by Bill Tetzeli on 2015-03-07
Re. Question 2, don't forget to download the initial updates after you install before going into the software center.  Most of the software choices won't be available for download before you do that, which is when you get the "Use This Source" option.  You should get a prompt to update shortly after you log in for the first time.  If you don't see it, click the button at the top left corner (called the Dash button), then in the search bar type "update" and click on "Software Updater" when it pops up.
