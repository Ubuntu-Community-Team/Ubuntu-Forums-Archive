### Post by monkeybrain20122 on 2013-10-22
Just install on top is fine. When you are installing, choose manual install (advanced?) and "something else", then install in the 32 bit partition (choose the partition, click change and choose file system and check format partition) It will erase the 32 bit version and install in the vacant partition.
