### Post by snowpine on 2010-10-05
You're in the right place! Any question you can think of, somebody else has already asked it on Ubuntu Forums. In particular, pay attention to the "sticky" threads at the top of each sub-forum.

Another great resource is the official Ubuntu documentation at:

[http://help.ubuntu.com](http://help.ubuntu.com) and
[http://wiki.ubuntu.com](http://wiki.ubuntu.com)

In general, I advise you to be skeptical of unofficial (not from ubuntu.com) help documentation you randomly find on the web. It may be difficult for a new user to tell whether a tutorial is outdated or unreliable. However, one very good unofficial resource that I can recommend is:

[http://www.psychocats.net/ubuntu/](http://www.psychocats.net/ubuntu/)

(the author is a moderator on these forums)

### Post by MattBD on 2010-10-05
I found the best way to learn more and get confident with Linux is to install Virtualbox and use that to mess around with a Linux distro or two - if you have an install in Virtualbox that makes it a lot less hassle if something goes wrong as you don't need to worry about messing up the main operating system on your computer, so you can play around with it to your heart's content, trying different desktops and things like that.

Also, try INX ([http://inx.maincontent.net/](http://inx.maincontent.net/)) - it's a distro based on Ubuntu that's designed to teach you how to use the command line properly, and it taught me a lot.
