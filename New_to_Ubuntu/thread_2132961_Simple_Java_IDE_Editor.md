### Post by r-senior on 2013-04-06
The best is a tough one to answer because it depends on you. The most popular are Eclipse, Netbeans and Intellij IDEA. I'd say Netbeans is the simplest for Java programming but I'd suggest trying all three because each have their devotees.

At a pinch you could use Geany and set up the build commands but most Java programmers benefit from the refactorings offered by the mainstream IDEs as they start to write more complex code.
