### Post by wizard10000 on 2009-07-30
You can add a second panel by just right-clicking the panel at the top and adding another one from the menu that comes up.  Then right-click on the new bottom panel and you can add a window list from there.

If you really want to change to the standard Ubuntu desktop all you have to do is install ubuntu-desktop from within synaptic or from a terminal window like this -

sudo apt-get install ubuntu-desktop

Hope this helps -
