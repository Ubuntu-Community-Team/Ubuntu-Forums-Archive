### Post by OffAxis on 2008-05-22
Is your 'Numlock' on?

If it's off 'Enter' will still work, but other keys on the numpad won't.

### Post by OffAxis on 2008-05-22
Have you tried a different keyboard?
Could be a hardware issue with it.

If you don't have a spare you could boot from a liveCD and see if it functions correctly there (this would tell you if it was a hardware problem or a software configuration problem).

You also might want to check the keyboard layout in System preferences and see that it's set correctly.

### Post by bluefrog on 2008-05-22
ctrl shift numlock

Your numpad is making your mouse move, you can also go to system/preferences/keyboard,[mouse keys] tab and uncheck the box.
