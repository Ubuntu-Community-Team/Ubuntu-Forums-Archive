### Post by bumanie on 2008-11-25
Bluetooth is included in the kernel as a default application - you should not need a installation disc of any kind. If you have an installation disc, it is likely windows-based and won't work in ubuntu. You should be able to see a bluetooth icon under System --> Preferences. Click on the icon to open a window to configure a device.
