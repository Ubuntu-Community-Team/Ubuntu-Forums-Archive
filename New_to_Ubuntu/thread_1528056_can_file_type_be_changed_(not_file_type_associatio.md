### Post by kfrancist on 2010-07-10
i have a firefox extension that saves web pages as maff files--which is some kind of derivative of a zip file. 

firefox can open maff files as perfect reproductions of the original webpage.  so i'd like to associate maff files with firefox, but ubuntu apparently makes no distinction between maff files and zip files--nautilus labels them as "zip archives" in the type column.  

i'm wondering if i can change the file type for maff files to "maff file" rather than "zip archive" so i can have firefox open them automatically without also trying to open actual zip files all the time.

is this possible?

### Post by audiomick on 2010-07-10
Hallo.
Have you tried this? : Right click on one of the files in question, choose "open with", choose firefox from the  list and make sure the box is checked for "remember this application for this file type".  The wording might be slightly different, as I have to translate it from my German speaking computer... :)
