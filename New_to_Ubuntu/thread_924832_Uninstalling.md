### Post by Ciju on 2008-09-20
How do I uninstall a program that I downloaded with a command line? Take Scribus for example.

---

### Post by wraith1986 on 2008-09-20
Go to terminal type "sudo apt-get update". Let it update, open synaptic and type in the name of the program you want to uninstall. If it is installed on your system it will have updated it not in the reps but on your software listing under installed. Simply remove it then as you would any program from synaptic.
