### Post by mastablasta on 2020-03-13
yes, very likely.

it's a different desktop environment that is based on QT libraries and to run it on gnome (which uses primarily GTK libraries) you need it's libraries. there is also KDE4Win which makes KDE software run on windows for example.

in any case storage these days is cheap and these files are not big and will not affect your machine performance.

i use KDE (found in Kubuntu), however i also use some Gnome apps, which means when i installed these apps they pulled in GTK libraries. this is not an issue as long as they work as intended.
