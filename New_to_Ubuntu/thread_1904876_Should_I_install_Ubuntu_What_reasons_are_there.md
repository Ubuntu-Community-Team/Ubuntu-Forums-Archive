### Post by carranty on 2012-01-05
I recommend installing virtualbox, this is a program which you can use to run other operating systems within your own.  You can download it here

[https://www.virtualbox.org/wiki/Downloads](https://www.virtualbox.org/wiki/Downloads)

It'll run a little slower than a straight install, and will likely have issues with 3D acceleration but other than that its great.  I use it all the time.

If you find you like ubuntu, then go ahead and install it alongside windows, you don't have to choose one or the other, you can dual boot both quite happily - I think most of us linux gamers do.

### Post by Mark Phelps on 2012-01-05
Since your focus appears to be on Gaming, especially with Windows games, then I recommend NOT installing Virtualbox.  You won't be able to do decent gaming with it.

If you want to "experiment" with Ubuntu, just to see how different it is from Windows, then use Wubi to install it.  This causes the least risk to your machine as it does not require messing around with the existing partitions.

If it turns out that you like using Ubuntu, then remove it from Windows, come back here, and we'll address how you can reinstall it using dual-boot approach.

As to programming, if you just want to program for PCs, then Python is as good a language as any (and better than some) and a Linux distro would be a good place to experiment with that.

But, if your programming is going to be for the Windows environment, that is very different, and you should then stay with Windows -- as their development tools install and run there, not in Linux.
