### Post by HermanAB on 2010-06-06
Note that the Skype audio codec runs at about 10 kilobit per second.  Therefore even a modem connection is plenty fast enough.

Your Windows machine is likely weighed down by a zoo of viruses and spambots.  If you install Linux, then it should work OK and keep working.  Get an Ubuntu live CD and try it.
