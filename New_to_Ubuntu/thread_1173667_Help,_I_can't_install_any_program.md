### Post by Sef on 2009-05-30
1) What software do you want to download?

2) Most software can be downloaded from the Repositories.  For some software you may need to open either** All Open Source Appllications** or **All Available Applications**.

3) **Applications > Add/Remove** is the easiest way to install.  Note: you may need to set **Show** to one of the above mentioned in #2.

4) Read [How to Install Anyting in Ubuntu]("http://amitech.50webs.com/installing/index.php.html").
