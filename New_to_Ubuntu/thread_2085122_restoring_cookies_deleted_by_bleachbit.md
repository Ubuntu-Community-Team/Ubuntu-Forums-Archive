### Post by Frogs Hair on 2012-11-17
Cookies are normally restored when visiting a site and the user saves account or page settings again. 

Some web browsers allow the user to make exceptions for cookies so it is possible delete only unwanted cookies.

Many customized personal pages require cookies to be enabled while others like email accounts store settings online.
