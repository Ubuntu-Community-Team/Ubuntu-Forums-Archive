### Post by uaPtah on 2011-05-22
Try Wally. It is nice app to set up wallpaper slide-show. Can be added different folders or files from different sources. 
Can be installed through Syneptic. After install you may need to add it to Startup Applications in order for this program to start with system boot. Go to System - Preferences - Startup Applications - Add - type "wally" in the field for command.

P/S
It is a shame that Ubuntu doesn't have similar app by default.
