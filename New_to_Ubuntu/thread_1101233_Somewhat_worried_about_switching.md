### Post by MaindotC on 2009-03-20
Dell sells machines with Ubuntu on it.  Is there any reason their customer service can't work something out with you?  Hell, if anything Ubuntu puts the machine in a lot better condition than some of the virus-infected, spyware-filled machines they have to repair anyway!

Steam works in wine and you can see a list of windows applications that work successfully in wine at [http://appdb.winehq.org](http://appdb.winehq.org)  Look when you install Ubuntu, all your doing is placing something on the hard drive.  You remove the hard drive, and you have the same machine that came from Dell.  If you're really worried, just get a small hard drive to plug into your Dell (not sure if it's a desktop or laptop) and install Ubuntu and load up your applications.  Keep the other hard drive that came w/ Windoze on it.  I dunno if Dell put a sticker on the hard drive to keep you from ripping it out.

Still worried - then use the live CD to form a partition for Ubuntu and load it up that way.  You don't like it then use the live CD to remove the Ubuntu partition and combine it with the windoze partition and no one will know the difference.
