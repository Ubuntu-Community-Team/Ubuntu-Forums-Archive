### Post by Frogs Hair on 2012-01-08
I'm not sure about your music files , but I made R.Box default by doing the following .

1. Removed Banshee 
2. Removed Banshee residual configuration from the synaptic package manager
3. Installed Rhythmbox 
4. Opened System Info > Default Applications 
5. Selected Rhythmbox from the music drop-down menu 

After logging out and in Banshee no longer appeared in the panel volume applet and was replaced by Rhythmbox.
