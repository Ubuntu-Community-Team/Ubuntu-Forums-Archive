### Post by Hedgehog1 on 2011-02-28
Welcome to the forums and the world of Ubuntu!

If you install Google Chromium (the Linux version of Chrome) you can have the Google tool bar.

If you menu to Applications >> Ubuntu Software Center, you will see a huge repository free programs to use (including Chromium).

You already have a CD/DVD burner installed (Brasero), and an MP3 player installed (Rhythm Box), and you will find more options for these and other programs.

I have found I prefer Banshee as my MP3 player, but Rhythm Box is very good too.

Explore the Ubuntu Software Center - there is a lot to choose from.

:KS

p.s. Gimp is a powerful photo editing tool; you can decide if it or a simpler one is the best fit for you...
