### Post by Bucky Ball on 2014-09-16
Welcome. Do you have Pulseaudio Volume Control installed? If not, install it and start an audio stream in Firefox/iPlayer (play some music), check the 'Output' tab in PAVControl and at the bottom there should be the Firefox/iPlayer stream. You can set where it's going in the drop-down menu to the right. Is it set to the correct place?
