### Post by Crafty Kisses on 2009-12-01
Sadly there is no support for iTunes in Linux, but I have heard some stories that iTunes will work in [Wine.]("http://www.winehq.org/") Now depending on your iPod you can easily add and remove music just like iTunes, if you have an older iPod the chances of it working are probably higher I had way back a white 30gig iPod and it worked perfectly with gtkPod. You can install gtkPod by running:
```
sudo apt-get install gtkpod
```
There's also tons of alternatives, like Songbird, Rhythmbox, Amarok, etc. Hope this helps.
