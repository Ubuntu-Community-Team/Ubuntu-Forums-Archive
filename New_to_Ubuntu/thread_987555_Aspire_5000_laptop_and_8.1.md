### Post by bennachie on 2008-11-19
I can't be absolutely sure, since I have never seen the "5000" model, but 8.10 works fine on an Acer Aspire 5315. There was quite a bit of discussion a couple of months ago about wireless problems with the Aspire 4900 series, but these seem to have been resolved by the release of 8.10.

The LiveCD should indeed be your friend. However, if the laptop already has a Windows OS in place, you may find it worthwhile to do an "embedded" install (just boot into Windows, insert the LiveCD after Windows is up and running, and follow the instructions). 

The advantage is that you can easily uninstall Ubuntu later (just as if it was an ordinary Windows programme) if you run into any problems. 

Disk access is apparently a little slower than when Ubuntu has been installed directly, but the OS still runs very well, and feels a good deal more responsive than Windows. By contrast, just running Ubuntu directly from the LiveCD makes everything seem pretty sluggish.
