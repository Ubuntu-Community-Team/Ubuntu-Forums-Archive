### Post by ugm6hr on 2008-11-10

### Post by aysiu on 2008-11-10
There is no administrative password.

There is your user password, and that's it.

If your user is the first user created during installation or a new user you explicitly made an administrator, then you can use your own user password to gain temporary administrative privileges.
