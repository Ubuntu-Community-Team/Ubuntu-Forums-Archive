### Post by Dennis N on 2015-04-19
I don't know who "bd users" are, but the optimal server for Ubuntu upgrades can by found for your computer's location from **Software Updater > Settings > Ubuntu Software Tab**;

From there, use the dropdown box "Download from:" and select "Other" then use the button "Select Best Server" then "Choose Server"
