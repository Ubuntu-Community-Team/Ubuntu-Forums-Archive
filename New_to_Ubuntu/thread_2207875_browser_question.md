### Post by deadflowr on 2014-02-25
Yep, you can install google chrome quite easy.
Just go to the chrome website.
It should automatically point you to the one you need.
Download the file and inside the Downloads Folder, double click on it.
This should launch the Software Center and allow you to begin the installation.

Then open the Ubuntu menu(dash) and type chrome.
Then launch it and chrome should start.

Whether Chrome or Firefox is better than the other, is something to each his own.

### Post by John_Swing on 2014-02-25
Or you can install Chromium, the open source project from which Google Chrome draws it's source code (it's the same browser without the proprietary stuff). It's present in Ubuntu repositories and you can install it using Ubuntu Software Center or typing sudo apt-get install chromium-browser in a terminal.

### Post by SeijiSensei on 2014-02-25
One trick you can try is installing the [User Agent Switcher]("https://addons.mozilla.org/en-US/firefox/addon/user-agent-switcher/") add-on for Firefox.  Then you can tell the browser to send a Windows user-agent string during negotiation with the site.  Often the only restriction against using Firefox or Linux is that the application looks for an IE user agent and refuses anything else.
