### Post by bogan on 2012-12-15
Hi!,** pete1977x**,

Install a program called "Gnome Color Chooser", run it and select the 'Specific' Tab; in the middle on the left [ I forget what it is called] reset the value from 15 to 24, and Click on 'Apply' & Close.

You can find & install it in Ubuntu Software Center, but make sure you get the right one, there are several with similar names.

Edit: It used to work, but when I tried to run it to checkthe name of the option, it crashed [ running 12.10 ]. 

I removed and re-installed, but it still would not allow me to change Tabs, and would not respond. However, I still have 24 pixel scroll bars!

Hope you have better luck. Edit2: on the third attempt i got it to run; surprisingly, the option is called: "Scrollbar Width!
