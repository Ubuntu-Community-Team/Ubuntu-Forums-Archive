### Post by chang47 on 2009-05-17
Hi,  I guess not too many people using Sea Monkey.  I happened to try it out and discovered as you did that we should use 1.1.16 for security update.  

I assume that you already downloaded the installer from Sea Monkey web site. The Readme file asked you to delete 1.1.15 so that you could install the new version in the same directory.  Use Synaptic Package Manager to delete the old version.  Once completed just run "./seamonkey-installer" as instructed in the Readme file.

I am up and running, good luck.
