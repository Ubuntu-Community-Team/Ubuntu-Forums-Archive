### Post by RC1139 on 2009-10-30
Your desktop looks normal, they changed the theme in Karmic.  You can change or customize it through System > Preferences > Appearance.  Also you can install the Compiz configuration settings manager either through Synaptic or using 'sudo apt-get install compizconfig-settings-manager'.  This will let you add some neat visual effects.
As for the sound I'm not at all a expert so try this link:
[https://help.ubuntu.com/community/Sound](https://help.ubuntu.com/community/Sound)
