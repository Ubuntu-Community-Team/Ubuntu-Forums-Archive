Making a dual boot machine with ubuntu and XP is really easy. Is is done in the installation process and works without having to configure anything. However one disadvantage : you have to reboot and get to the other machine all the time. If you use win & ubuntu frequently you may want to know there are other options :

-Virtual box in seamless mode (makes your xp start menu appear at the bottom while your gnome panels are at the top). It is a virtual machine software that runs pretty fast.

- Wine : runs windows programs without anything from windows. They basically rewrite all functions in all DLL coming with win xp (& other versions) to make a linux equivalent so win programs runs natively on ubuntu

### Post by gareth_005 on 2008-05-05
Welcome!

You should try to test XP in VirtualBox on a clean drive if you can, I initially installed a dual boot setup, then I found VirtualBox and never booted into the XP partition again. It was a waste of space and you also don't want to reformat as you will have setup Ubuntu nicely.

I use the VirtualBox open source edition, it does not integrate into the desktop.
The free for private use VirtualBox edition is the one you want for desktop integration.

p.s. You can't play games in VirtualBox.
