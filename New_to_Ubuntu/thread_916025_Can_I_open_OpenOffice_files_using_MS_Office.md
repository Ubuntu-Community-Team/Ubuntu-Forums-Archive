### Post by dioltas on 2008-09-11
If you save it in Microsoft office format, .doc, it can be opened in Microsoft work no problem. I work on stuff at home using Open Office and at work using microsoft office. Never had any problems with format or anything. 
Just in case you are worried about the formatting getting messed up when you swap over, you can print your open office file to a pdf.
I think this comes built in with ubuntu, just click file, print and select print to pdf. 
Of course, this is only useful if the person you are sending the file to doesn't have to edit it.

On a related note, open office calc seems to handle Excel formatting fairly good as well.
