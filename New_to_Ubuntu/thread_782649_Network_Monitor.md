### Post by _Narcisse_ on 2008-05-05
Hi FredJones,

If you want a real time network monitor on your panel, jusr right click on it, "Add to panel" -> "System Monitor" -> button Add, then right click on the new monitor that shows processor activity, check "Network" and uncheck Processor if you want. Theres some widgets for compix that does it too if you like eye-candy.
