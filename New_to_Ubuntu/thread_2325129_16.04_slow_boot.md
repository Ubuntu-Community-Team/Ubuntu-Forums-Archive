### Post by workshop99 on 2016-05-19
I found this [http://askubuntu.com/questions/639559/very-slow-boot-with-ubuntu-15-04](http://askubuntu.com/questions/639559/very-slow-boot-with-ubuntu-15-04)
The problem was that I'd moved the swap partition using Disks but the original entry in /etc/fstab remained unchanged so at boot it was looking for a partition that wasn't there.
