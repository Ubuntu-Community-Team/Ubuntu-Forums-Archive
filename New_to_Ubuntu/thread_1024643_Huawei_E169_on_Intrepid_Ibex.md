### Post by Sealbhach on 2008-12-29
There will be a Network Manager already installed when you install Intrepid. I had a Huawei E220 which I used on Hardy (8.04) and I used to have to get a deb package called Gnome-PPP to manage the connection.

However, as I understand it, these Huawei dongles are now plug-and-play in Intrepid 8.10 but it would be best to get someone who has actually used one in 8.10 to verify this for you.

This guy says it works

[http://cviorel.easyblog.ro/2008/12/04/huawei-e169-usb-stick/](http://cviorel.easyblog.ro/2008/12/04/huawei-e169-usb-stick/)

So does someone else here:

[https://answers.launchpad.net/ubuntu/+question/39633](https://answers.launchpad.net/ubuntu/+question/39633)

so I think you're good to go.
