### Post by DCO on 2015-02-20
Note, I'm not an expert of any kind so I could be mistaken on some points but as I understand it all:

It is an operating system. but not like Windows. Linux is Unix based and has an ingrained user and permission system (unlike Active Directory which seemed added after the fact as I studied it.) This creates a more secure system because root is it's own user (the system itself,) where in Windows the administrator is the root user. An administrator, through their password, (proving they are an administrator,) can gain root privileges temporarily if used as intended. 

It is possible to create a root user but that weakens the security quite a bit. 

Linux servers make up a lot of the backbone of the internet and is used by entities like the New York Stock Exchange and other high security areas. 

That's how I understand it in a nutshell. I hope that helps.
