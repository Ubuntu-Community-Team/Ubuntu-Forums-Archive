### Post by Mark Phelps on 2012-09-30
Any time you install an OS, or otherwise mess around with partitions, you run the risk of data loss.  To be safe, you should do the following BEFORE you attend installing Ubuntu:
1) Back off the Win7 install to an external drive.  Download and install the free edition of Macrium Reflect, use the option to create and burn a Linux Boot CD, and then back off the current install to a drive.
2) Use the Win7 Backup feature to create and burn a Win7 Repair CD.  This will allow you to repair or restore the Win7 boot loader, should anything happen to it during your dual-boot installation.
