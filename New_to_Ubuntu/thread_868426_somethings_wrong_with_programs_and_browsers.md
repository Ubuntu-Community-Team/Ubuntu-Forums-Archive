### Post by uberlube on 2008-07-23
either disable compiz to regain your normal desktop settings or if you want to keep compiz active and choose cooler looking borders make sure emerald is installed. Once you have emerald and a theme installed in it go into the advanced desktop effects manager and go to the window border area. where it says command type in "emerald --replace" and then you should be good.
