### Post by mastablasta on 2014-08-11
```
wine setup.exe
``` 
or
```
wine install.exe
```

should get you started then to launch

```
wine *gamename*.exe
```
where gamename is the game file used to launch the game.
for more on specific game check here: [https://appdb.winehq.org/](https://appdb.winehq.org/)

### Post by cj13579 on 2014-08-11
Wine has probably come on leaps and bounds since I last used it but you could also look at installing Steam or CrossOver as a way of playing games on that machine. I use a mix of both of the above and have never had any problems. I use Steam for most games and am using CrossOver for older ones. While CrossOver isn't free, I have found that with the games I have played it has been smooth, fast and stable.
