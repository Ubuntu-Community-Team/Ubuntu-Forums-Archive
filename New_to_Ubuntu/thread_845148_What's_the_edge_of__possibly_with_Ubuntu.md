### Post by seuzy13 on 2008-06-30
For Photoshop, I have been able to get various versions working at various times, but it was hit or miss whether they would actually run and cooperate. The Wine AppDB gives CS3 a bronze for the latest version of wine. [Here.](http://appdb.winehq.org/objectManager.php?sClass=version&iId=6584)

I would check around the wine appdb to see what is available if you are curious. Wine is a very powerful and useful piece of software, and once you install it, you'll probably forget it's there. 

It is also possible to boot Windows from *within* Linux with virtualbox or VMware if you want to go that route. In this case, most everything would work, but you would be working with limited resources, since your RAM, etc. would be divided b/t two OSs. In my experience, everything runs very well, though.

Basically, there will be some sacrifices while Microsoft still has a hold on the home computer OS world. The main thing to worry about is your hardware though. Graphics cards and sound cards are usually what'll get ya.

3ds Max no, Maya yes! My 3D using friends all say Maya is more of the industry standards these days so that's not a bad thing! I still have the CS3 master collection on my windows boot just in case...flash seems to be the real kicker and not work well. [http://www.linuxlinks.com/article/20080530054213402/CommercialGames.html](http://www.linuxlinks.com/article/20080530054213402/CommercialGames.html)

[http://www.winehq.org/](http://www.winehq.org/)
