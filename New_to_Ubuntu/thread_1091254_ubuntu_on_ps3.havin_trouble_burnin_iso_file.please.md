### Post by 3rdalbum on 2009-03-09
Checklist:

1. Are you burning the ISO for the PowerPC processor? I think last time I checked there was a PS3-specific ISO; maybe not anymore, but you should be using the PPC ISO and not the regular i386 one.

2. Are you burning using the "burn image to disc" function of your software? Just dragging the file to a blank disc will not do the trick. "Extracting" the ISO and dragging the contents to a blank disc will also not work. And don't use the "Create bootable disc" function of your software; this does something completely different.

3. Try burning at a lower speed, and/or using CD-Rs rather than DVDs.

### Post by emshains on 2009-03-09
First, get the CD image from here, take the ps3 one. 
Please note that the server one will leave you with no gui at all, so take the alternative installer, and you can't have wpa wi-fi encryption while installing, that means, if you are using wpa, you will have to plug your ps3 via ethernet to the internet.
[http://cdimage.ubuntu.com/ports/releases/8.10/release/]("http://cdimage.ubuntu.com/ports/releases/8.10/release/")


After that, burn the image to the disk, don't just copy the image as a file, you should use Nero. 

I am pretty sure that my ps3 reads cd's as well as DVD's and blurays, but if yours doesn't read CD's then stick with dvd's.
