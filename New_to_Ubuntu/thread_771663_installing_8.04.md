### Post by dark_harmonics on 2008-04-27
Did you try the playing with the VGA graphics mode in the first screen of the live cd? Your graphics card may require a restricted driver to work properly. 

On some installations (like my macbook pro) you need to install from the alternate cd which has a text-based installer. Then after the OS loads, you install the graphics package from the command line and reboot into the full GUI. This is mostly caused by graphic-card manufacturers not having open source drivers. Shame on them!
