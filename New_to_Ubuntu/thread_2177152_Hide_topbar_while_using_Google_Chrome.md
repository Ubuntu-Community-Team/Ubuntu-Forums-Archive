### Post by newb85 on 2013-09-28
I see.  The problem is that Unity's top panel doesn't have a hide or autohide feature, and Chrome's fullscreen mode doesn't show tabs.  Given the direction Canonical has taken Unity, the former is unlikely to change.  

You could try a different desktop environment, like Cairo-Dock, that doesn't require any permanent real estate.  Or, you could try a different browser.  Firefox, for example, will show the tabs and address bar in fullscreen mode if you mouse to the top of the screen.
