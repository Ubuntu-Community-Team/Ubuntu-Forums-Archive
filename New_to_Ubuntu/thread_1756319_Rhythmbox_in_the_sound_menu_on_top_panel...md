### Post by frankbooth on 2011-05-12
It has to do with 'indicator-sound' which was updated in Natty.
I guess it should be possible to use it in 10.04, but you'll need to compile it yourself since I don't think there are any packages for older versions of Ubuntu (unless the Natty package works for 10.04).

[https://launchpad.net/ubuntu/natty/+source/indicator-sound/0.6.6.1-0ubuntu3](https://launchpad.net/ubuntu/natty/+source/indicator-sound/0.6.6.1-0ubuntu3)

EDIT: As user below pointed out, it was already introduced in 10.10.
But the above should still be true for 10.10 packages.
