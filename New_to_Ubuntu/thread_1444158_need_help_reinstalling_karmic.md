### Post by mikewhatever on 2010-03-31
The installation is exactly the same, the only difference being the need to manually specify the partition. At the partitioning stage, click 'Advanced', right click on the Ubuntu partition and select 'edit', specify the file system (ext3 or ext4) and the mount point (/), check the format box. Needless to say, you'll have to backup all important files before doing that.
