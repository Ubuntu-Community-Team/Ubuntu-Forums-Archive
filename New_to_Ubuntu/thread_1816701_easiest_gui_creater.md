### Post by LemursDontExist on 2011-08-02
If you're looking to throw together a few quick scripts with gui, I'd recommend Zenity.

For slightly more complex projects I'd recommend Python + Glade.  Here's a [tutorial]("http://www.overclock.net/application-programming/342279-tutorial-using-python-glade-create-simple.html"), there are many more around.

Actually, I'd recommend against GUI programming until you're comfortable programming generally.  GUI programming is messy and complicated in very uninteresting ways, but if you're set on it, glade will deal with a lot of the complications for you.
