### Post by argedion on 2010-12-02
I have dual booted with beta versions and the final release of Windows 7 and have had no problems. As far a performance issues go you should not have any with Windows 7 as it wont recognize your Linux system. If there are concerns you could always just run your Ubuntu off of a disc or even a removable drive. (Running from a USB Drive will be much faster and better than an optical drive)

> I ask this because this is only our second system, the previous one is faulty NOTHING to do with ubuntu 

If you have another system and are willing to experiment then try wiping that system out and just installing ubuntu on it and leave the current one with windows 7. If you have a failing hard drive on the other system just replace it.
