2. swap    -a memory swap-partition

if you want to keep it easy:
- first install XP (let it use the whole disk)
- then install Ubuntu and let Ubuntu make the side-by-side installation automatically. Ubuntu will repartition the disk and divide the space equally.

if you decide to partition "by hand" (choosing "something else" in the installation options), then give the SWAP partition about the same as your ram (in this case 1 gb)

Installing Ubuntu and *then* XP makes it kind of complicated, because then XP will take over the mbr, not letting you choose between XP and Ubuntu on startup. This can be repaired, but I would stick with the straight forward installation for now.
