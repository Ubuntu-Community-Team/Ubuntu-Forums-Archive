### Post by beew on 2011-06-11
I wouldn't recommend WUBI. If you want to get a true feel of Ubuntu without partitioning your hard drive you can install Ubuntu into an external harddrive and boot from it. It is in every sense a real install and it is master of its own domain unlike WUBI , it uses real Linux file systems and you can do system updates and install proprietary graphic cards and it is fast too so it is not like  a live usb. The only thing that you may not get a true idea of with this method is boot time, depending on the BIOS it may take a lot longer to boot from an external drive (or a usb) than an internal install, but once it is up and running everything is the same.

See my post here for instructions

[http://ubuntuforums.org/showthread.php?p=10914503#post10914503](http://ubuntuforums.org/showthread.php?p=10914503#post10914503)
