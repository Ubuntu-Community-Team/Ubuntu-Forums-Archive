### Post by endotherm on 2010-10-28
as i understand it, gmplayer is a very old very unmaintained project, that was never fully integrated into ubuntu. 
I recommend smplayer as a frontend for mplayer, and it does save my settings when asked.
there are several PPAs from which up-to-date versions of both the cli mplayer package and the smplayer frontend can be downloaded, which are far more recent than the versions in the repos.

[http://smplayer.sourceforge.net/downloads.php](http://smplayer.sourceforge.net/downloads.php)
(check the "packages for ubuntu" section)
