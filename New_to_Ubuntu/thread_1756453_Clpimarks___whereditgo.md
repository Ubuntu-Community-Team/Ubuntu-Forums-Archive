### Post by lovinglinux on 2011-05-12
Use [Amplify]("https://addons.mozilla.org/en-US/firefox/addon/amplify/").

> Check out the new Firefox add-on from the makers of Clipmarks...
Please note that we recently introduced a new service called Amplify that has most of the features of Clipmarks as well the ability to share URLs and sync your clips with services such as Twitter, Facebook, Tumblr, Posterous, Friendfeed, delicious, diigo, Plurk and Clipmarks.   To use Amplify to clip & share interesting things you find on the web, please check out our Amplify Firefox add-on. 

However, if you still want to use Clipmarks, you can install [Add-on Compatibility Reporter]("https://addons.mozilla.org/en-US/firefox/addon/15003/"), in order to disable compatibility check, so you can install Clipmarks. I have not tested it throughly, but it seems to be working without any errors.
