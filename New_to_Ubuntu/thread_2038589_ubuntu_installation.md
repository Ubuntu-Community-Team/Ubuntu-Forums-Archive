### Post by mastablasta on 2012-08-07
first make sure that everyhting runs ok by booting into live media (CD/USB) and select try it.
 
then if you don't need C drive you can simply point to it during install process (select something else to switch to manual install).
 
if you are unsure how to do that you can also boot into live, run gparted delete C: drive (as windows labels the partition) and leave it unformated. during install select to install to largest unformated disk space.
 
here is how the install process looks like: [http://www.psychocats.net/ubuntu/installing/](http://www.psychocats.net/ubuntu/installing/)
 
more on installation: [https://help.ubuntu.com/community/Installation](https://help.ubuntu.com/community/Installation)
 
you can also check the community made manual for more info.
