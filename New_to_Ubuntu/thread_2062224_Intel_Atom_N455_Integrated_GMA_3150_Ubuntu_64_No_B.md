### Post by albandy on 2012-09-24
I'm using Ubuntu 12.04LTS 64 bit in my atom media center.

Probably a grub error during installation caused that you can't boot from HD.

Simply download the alternate install CD, burn it like it was the standard cd, and boot it. 

Then, a text menu will appear and there is an option to restore grub.
