### Post by cwsnyder on 2009-02-16
If you have problems connecting when running from the first option on your Ubuntu install CD (Try Ubuntu without installing), then you may have problems with the full install.  On the other hand, if you have no problems with the 'Live' session, you should have no problems after you install with at least that portion of the installation.
