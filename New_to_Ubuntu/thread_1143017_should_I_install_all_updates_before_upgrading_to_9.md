### Post by Didius Falco on 2009-04-29
Canonical recommends that you apply all the updates to 8.10 before upgrading to 9.04 - I'd go with their advice.

On Edit:If 3D is important to you, you should check at the nVidia site and make sure that card will work with 9.04 in 3D. It might be one of the cards that has been moved to legacy support, in which case you won't get 3D under 9.04.

If that's not important to you, there is an Open Source 2D driver that will work just fine.

I use an ATI card, so I'm not up-to-speed on which nVidia cards are unsupported for 9.04. A forum search might help you find out.

I'd hate to see you upgrade and then be unhappy....

### Post by markbuntu on 2009-04-29
It is very important that you remove any proprietary nvidia or ati drivers from your system before upgrading or you will have problems.

It is also important to be fully updated because many of the recent updates concern the migration path to jaunty so update your system before upgrading.
