### Post by pe7er on 2012-07-14
When I encounter problems or questions, I mostly use Google with some keywords: the linux distribution / version + the software that causes a problem + error message.

At the moment I am reading "Ubuntu Linux Toolbook - 1000+ commands for Ubuntu and Debian Power Users" which I find the best book regarding learning to use the Command Line for Ubuntu/Debian.

I also have "Ubuntu Unleashed 2008" which nowadays seems a bit outdated. I think that you'd better buy a book that isn't written for a particular version.
