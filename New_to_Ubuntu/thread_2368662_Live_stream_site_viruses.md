Basically, Linux will be a protection layer against binary viruses that are intended for Windows. That is, Windows viruses will not run in Linux. Even if you download a strange "install.exe" file, it will not run in Linux. You can just delete it, and that's it.

Except if you actually run a virus through Wine, for example.

However, web browsers are designed to run things like Java applets, Javascript, and Flash, which interact with you, and are operating system independent. That is, these things run in any supported web browser, independent of the device, either Windows PC, Apple notebook, Android phone, etc. So, you should always be careful when visiting shady, dodgy, obscure, spam ridden, potentially malicious websites.
