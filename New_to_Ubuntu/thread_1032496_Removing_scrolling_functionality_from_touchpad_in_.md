### Post by computer_freak_8 on 2009-02-10
I was so excited that got my touchpad working like I wanted it to, I just had to share with someone else. You seemed like the perfect candidate, so, here goes something:

First of all, the package(s) that you will need can be installed by issuing this command:
```
sudo apt-get install tpconfig synaptics
```

Secondly, I have my "/etc/X11/xorg.conf" file modified; with the following section added:
```
Section "InputDevice"
        Identifier      "Synaptics Touchpad"
        Driver          "synaptics"
        Option          "SendCoreEvents"        "true"
        Option          "Device"                "/dev/psaux"
        Option          "Protocol"              "auto-dev"
        Option          "HorizScrollDelta"      "0"
        Option          "SHMConfig"             "on"
EndSection

```
However, I do not believe this particular configuration file modification is necessary.

Next, I found [this page]("https://help.ubuntu.com/community/SynapticsTouchpad#shmconfig"), but I thought I did something wrong. Apparently, it is simply not worded clearly. Here's what I did next:
[LIST=1]
[*]Open a terminal and type in
```
gksu mousepad /etc/hal/fdi/policy/shmconfig.fdi
```
[*]Then, enter this code (copy and paste would probably be best) into the new document's window:
```
<?xml version="1.0" encoding="ISO-8859-1"?>
<deviceinfo version="0.2">
 <device>
  <match key="input.x11_driver" string="synaptics">
   <merge key="input.x11_options.SHMConfig" type="string">True</merge>
  </match>
 </device>
</deviceinfo>
```
[*]Reboot
```
sudo init 6
```
[*]From the terminal, use "synclient" to modify settings. To disable both horizontal and vertical scrolling, issue:
```
synclient VertEdgeScroll=0
synclient HorizEdgeScroll=0
```
[/LIST]
****Edit: The above code for the "/etc/hal/fdi/policy/shmconfig.fdi" file  will not change it permanently. You will have to use "synclient" each time you reboot **unless** you set it in the "/etc/hal/fdi/policy/shmconfig.fdi" file. For example:
```
<deviceinfo version="0.2">
 <device>
  <match key="input.x11_driver" string="synaptics">
   <merge key="input.x11_options.SHMConfig" type="string">True</merge>
   <merge key="input.x11_options.VertEdgeScroll" type="string">0</merge>
   <merge key="input.x11_options.HorizEdgeScroll" type="string">0</merge>
  </match>
 </device>
</deviceinfo>
```****
Unfortunately, I don't know how to get the GUI tool working, but I'll try my best to remember to let you know when/if I get it figured out.

Be sure to post back to let us know what happens after you try the above steps. Good luck!!
