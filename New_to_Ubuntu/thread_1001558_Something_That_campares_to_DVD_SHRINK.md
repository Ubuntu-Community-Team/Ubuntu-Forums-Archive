### Post by Kobalt on 2008-12-04
Handbrake is a cross plateform software that will rip your DVD off and output to a designated file and format, using several presets (iPod, PSP, film, etc.). 
It has a native linux version, and is present in the Ubuntu repos. It's really easy to use and very efficient (it now even support .avi or .mpg files as a source).
