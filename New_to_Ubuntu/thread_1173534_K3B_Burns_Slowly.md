### Post by djbushido on 2009-05-29
There should either be an option under preferences or when you burn to select 32x. I personally don't worry too much.
Also, if you are doing an audio cd, try converting the file to wav ahead of time. Since the conversion is taking place on disk, it should take less time. If you still need to get the time down, I use brasero, which allows me to set the speed at burn time.
