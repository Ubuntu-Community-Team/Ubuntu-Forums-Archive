### Post by SEisch on 2011-03-09
If you Want Evolution to load images automatically you have to select Preferences from the Edit menu.  In the Preferences there is a tab for HTML messages.  In that tab there should be a few different options for loading images automatically.  You can either select it to only add images if the mail came from one of your contacts; or to load all images regardless of the sender.
Hope this helps
