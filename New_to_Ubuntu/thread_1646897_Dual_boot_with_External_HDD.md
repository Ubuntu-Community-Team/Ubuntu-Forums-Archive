Do you have a XP install cd if not you can install the lilo boot loader to the sda hd=XP.
 
For the MS bootloader from a XP cd.
[http://helpdeskgeek.com/how-to/fix-mbr-xp-vista/](http://helpdeskgeek.com/how-to/fix-mbr-xp-vista/)

If you don't have a XP disc do this with a Live booted 10.10.
```
sudo apt-get install lilo
```
```
sudo lilo -M  /dev/sda mbr
```
Lilo is a MS friendly bootloader and should have XP booting in on the computer.

You are also missing a bootloader for the external lets get XP going first. Getting Ubuntu booting with a bootloader is quite easy as well.

"Excellent" Mr Burns

Okay so to load the grub bootloder to the external boot the live 10.10 cd with the external plugged in and run these two commands.

```
sudo mount /dev/sdb1 /mnt
```
then
```
sudo grub-install --root-directory=/mnt/ /dev/sdb
```

Then reboot making sure the external HD is read first, from adjusting the bios or a per-session boot menu prompt.

Once you're in the Ubuntu install run in the terminal.
```
sudo update-grub.
```

So you will notice that you will be able to boot XP without the external, but XP should show in the grub menu and be bootable from there as well.
