### Post by gregphil on 2008-09-14
I finally figured out the new KDE4 interface.

To change the icon assigned to a shortcut:

Right click on the shortcut
Select --> Icon settings
In dialog box that opens up click on the ICON (no menu item listed)
Select an icon picture from the list (not all are shown....but better than nothing)

The move existing shortcuts out of the menu Lost and Found:

RIGHT click on the menu launcher (the gear)
Select menu editor
Expand the "lost and Found" section of the tree
Drag and Drop lost shortcut to another part of the tree (say "utilities")
click "Save"

This isn't so bad once you find the (somewhat hidden) interface.   But....how did the shortcuts end up in "Lost and Found" in the first place?
