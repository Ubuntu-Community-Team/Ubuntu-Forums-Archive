Hello all,

I am wondering if it is possible to set a resolution in BURG which is higher than the list provided by vbeinfo.

I've read the following pages without being able to find the answer to my question:
[https://help.ubuntu.com/community/Burg#Define%20hot%20keys](https://help.ubuntu.com/community/Burg#Define%20hot%20keys)
[http://groups.google.com/group/burg-devel](http://groups.google.com/group/burg-devel)

The list of VBE supported resolutions is displayed by entering c to go to the terminal followed by: ```
vbeinfo
```The list generated by vbeinfo shows the same resolutions as the ones that are displayed when I press the default key for 'change resolution' in GUI mode (F2 if I recall correctly). My display supports 1920x1080, but the highest resolution listed by vbeinfo is 1400x900. I have opted for 1152x864 (maximum resolution devided by 5 and multiplied by 4, to preserve the correct width:height ratio).

Are vbe supported resolutions dependant on hardware, or is there perhaps a way to set a higher resolution using software?

### Post by b0b138 on 2011-05-20
i believe you can set it in the /etc/default/burg file

There is a line, GRUB_GFXMODE=saved, that you could change.

You can try changing it, but even if it works, you'll probably be better off with a lower res as the themes don't seem to be made for real high resolutions. And, the higher it gets, I've experienced it being real laggy.
