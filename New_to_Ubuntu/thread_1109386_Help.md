### Post by fly-by-night on 2009-03-28
[http://www.ubuntugeek.com/blueman-bluetooth-manager-for-ubuntu.html](http://www.ubuntugeek.com/blueman-bluetooth-manager-for-ubuntu.html)

This have instructions on how to add the repository for blueman to Synaptic.

Add it in Settings/Repositories/3rd Party...
In the Apt line paste the line mentioned at ubuntugeek.com.

Then once you're done remember to click Reload, to reload your repositories.

I just tested it, and it was added to my repositories.  I didn't test the keys as I'm not planning to install it.
