### Post by jbrown96 on 2008-11-17
try doing the following in a terminal. ```
sudo apt-get update && sudo apt-get upgrade
``` If this gives errors, then try ```
sudo apt-get -f install
```
If you're still getting errors, you might try removing the packages and reinstalling them. You should try the apt-get method first.
```
sudo apt-get remove python-gobject python-gtkhtml gnome-games-data ptyhon-libxml2
```
If that doesn't work use ```
sudo dpkg -r python-gobject python-gtkhtml gnome-games-data ptyhon-libxml2
``` After that run the update and upgrade commands from earlier. You shouldn't get any errors. You can then try to reinstall the packages.

# Parse the version format from .version files
class version_list:                           
  def __init__(self,version_info):            
    self.verlist=[]
    dummylist=[]
    for item in version_info.rstrip('\n').split(','):
      v=[i.strip() for i in item.split('-')]
      if len(v)>1:
        if v[0]:
          dummylist.append(v[0])
        else:
          v[0] = py_supported_short[0]

    self.verlist.sort(version_cmp)
