### Post by Melk79 on 2008-05-16
Ubuntu 8.04 LTS (Hardy Heron) will run Compiz Fusion well, as long as you have a capable video card.  Once installed, you can enable desktop effects and have the fancy effects like the YouTube video.  You may need to run:
sudo apt-get install compizconfig-settings-manager

In order to have the Advanced Desktop Effects Settings choice under System>Preferences.  From this menu, you'll be able to turn on the 3D cube, wobbly windows, etc.
