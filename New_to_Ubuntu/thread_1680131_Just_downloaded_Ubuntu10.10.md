### Post by Sealbhach on 2011-02-02
The iso file you have downloaded is a CD image. You need to burn that image on to a CD, then boot up from the CD. You could use something like [Nero]("http://www.nero.com/eng/") to burn the CD - make sure to choose the option "Burn CD image" rather than "Data CD".

If you have a USB stick, it would be quicker and faster to make a bootable USB drive using [Unetbootin]("http://sourceforge.net/projects/unetbootin/").

You will also have to make sure that the boot order in your BIOS is set so that it will boot from the CD or USB drive before booting from the hard drive.
