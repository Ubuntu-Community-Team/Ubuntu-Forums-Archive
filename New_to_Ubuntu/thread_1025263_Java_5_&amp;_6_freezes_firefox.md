### Post by taurus on 2008-12-29
Why not install the Sun version?

Applications -> Accessories -> Terminal
```
sudo apt-get update
sudo apt-get install sun-java6-bin sun-java6-jre sun-java6-plugin
```

Use the Tab key to highlight the <OK> and Return to accept the licensing agreement.
