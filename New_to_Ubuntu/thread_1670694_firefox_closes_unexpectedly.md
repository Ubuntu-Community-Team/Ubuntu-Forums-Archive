@gracehudson, Firefox is installed by default in Ubuntu.

@pmohankumar, I've had that same problem happen in the past, it ended up being caused by one of the extensions I'd installed. (have you installed any new ones lately?) 

I don't remember which extension it was that caused the problem.  However, the way I found it was to disable all of my extensions, and run Firefox to see if the behavior continued. 

This took some time to do because I'd barely have time to open the "Add-ons" window and disable an extension before Firefox would disappear again.  But eventually I disabled all of them and Firefox worked just fine.

If disabling your extensions allows Firefox to work normally, then begin to re-enable the extensions one at a time, trying Firefox after each one, until the culprit is found.

I hope I explained this right, and hopefully it will work for you.  It's worth a try. :)
