### Post by CyberPrime on 2008-11-20
Well, there are a lot of options when deciding on what kind of PC to build, if you don't really need to do a lot of graphics intensive things I would suggest getting a motherboard with a build in graphics and sound card, make sure the graphics card is about 64MB, 128 is good. I would suggest an Nvidia motherboard, if not then stay away from "No-Name" parts (meaning brands or companies you've never heard of). If you put together a list I would be more than glad to look it over for you quickly and tell you if it looks good.

Also, I would suggest buying your parts from newegg.com It isn't a UK based site, but it is probably the best there is.

Good luck, HAND.

EDIT: As well, if you don't get a motherboard from a company that nobody has heard of (unlikely) then you probably won't have any trouble with drivers.

### Post by tgalati4 on 2008-11-20
For a laptop, get a used Thinkpad.  Lots of those floating around.  They run Ubuntu well. 

For a desktop, get a used gamer's machine.  Gamer's like to upgrade often and the older machines (with lots of memory, getter graphics, and fast harddisk) will also run Ubuntu well.  You can run the Live CD to checkout the initial compatibility.

There are several low-power VIA and Intel Atom-based machines out there, but construction quality is dubious.  If you go that route, it pays to spend the $12 US to have the pc vendor actually power up the motherboard before sending it out.  Out-of-the box failures are common on these low-end machines.
