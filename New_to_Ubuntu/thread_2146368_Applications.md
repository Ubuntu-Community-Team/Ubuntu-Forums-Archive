### Post by r-senior on 2013-05-18
Just to clarify, I assume you are running Ubuntu because you refer to Unity? The post is tagged as Lubuntu for some reason.

You don't have to know the exact name of an application to search for it. The search is more intelligent than that and works on various keywords for each application. For example, if you hit the Super key and type "spread" as the search, LibreOffice Calc comes up because that is the spreadsheet program.

If you want to browse, hit Super-a to get the applications lens of the Dash. You can use "See xx more results" to see a longer list, or you can browse categories using the filter at the top right.

Your most common applications will be locked to the launcher. When an application is running an icon appears in the launcher bar down the left side. You can right click and choose "Lock to Launcher" to keep it there even after it is closed.

Minimized applications have icons in the launcher down the left side of the screen.

Give Unity time -- at least a couple of weeks. It's a new way of working. Don't abandon it until you have mastered it and can make an informed choice.

Anywho, If you want to run Ubuntu with Unity but don't really like the Dash Menu you can bring back the classic menu:

 ```
sudo add-apt-repository ppa:diesch/testing
sudo apt-get update
sudo apt-get install classicmenu-indicator
```
[COLOR=#0000cd]***EDIT**: You will need to logout and log back in for the icon to appear on the panel.*[/COLOR]

It put's an icon in the tray of the upper panel ( if it's still called an panel in Unity ) and when selected looks just like the Menu that you remember.

### Post by 3rdalbum on 2013-05-19
Your open programs appear with one or more little triangles to the left, in the Launcher (that's the little bar on the left of the screen). The number of triangles corresponds with the number of windows open. If you click the icon once, you go to the last-used window of the program. Click it again, and all the windows of that program will display on the screen, and you can click the one you want.
