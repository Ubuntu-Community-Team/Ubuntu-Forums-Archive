### Post by Rob Sayer on 2015-12-06
I wouldn't install any themes that are 4 years old, as are the comments.  In fact I wouldn't install any themes that do not come from the repos.

There are many threads here from newbies whose system doesn't work properly after installing 3rd party (i.e. not supported) themes.  I don't even have any app software installed from ppa's or tar files on 2 linux installs.  There's no way I'd do it for DE stuff.

### Post by grahammechanical on 2015-12-06
Ubuntu is built on the Gnome 3 desktop environment. So, if Ubuntu no longer supports GTK2 themes it is because Gnome 3 DE no longer supports GTK2 themes.

Best to stick with the GTK3 themes.
