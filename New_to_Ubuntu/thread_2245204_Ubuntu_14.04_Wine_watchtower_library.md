### Post by grahammechanical on 2015-02-24
I never notice this thread back in September otherwise I would have posted in it sooner.

When Wine is installed it puts three utilities in the Dash. Search for wine and load Uninstall Wine Software. That will load a Windows type Add/Remove dialog with an Install button. Click that and a Windows type dialog will open that will let you browse to the CD/DVD drive with the Watchtower library CD in. Click setup.exe and the usual Windows type install program will run and it will put a WT Library icon on the desktop to use to launch Watchtower Library.

It is not helpful that with wine the installer is called an uninstaller.
