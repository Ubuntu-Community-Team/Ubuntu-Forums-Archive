### Post by dorkdork777 on 2009-02-28
AFAIK there's no such artificial limitation, but it could be that your machine doesn't have enough RAM to support the LiveCD interface. In other words, it can't load enough from the CD at once for you to use it, so it crashes partway through, leaving you with that error message.

However, a quick search shows that it should have 1GB of RAM, which seems fine, so that shouldn't be a problem. Does yours have the full 1GB?

Which install disc are you using in particular - alternate or LiveCD? Which architecture? Which version?

And could you possibly copy the error message into a reply here so we can see it?
