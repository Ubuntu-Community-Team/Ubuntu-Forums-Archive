### Post by drs305 on 2008-12-16
If you have the Workspace Switcher in the bottom right of the desktop (to add, rt click, Add to Panel, Workspace Switcher), right click, select Properties, and change number of workspaces from 2 to 4.

In compiz (System, Preferences, Compiz Config Settings - if installed), General, Desktop Size, Horizontal Virtual Size, 4.

### Post by BastardNamban on 2008-12-16
Drs is right. Set the virtual desktops in compiz settings manager to 4 (under general options- desktop size). Alternately, you can set that number to 3 for a triangle box, 5 for a pentagon box, and up, and so on.
