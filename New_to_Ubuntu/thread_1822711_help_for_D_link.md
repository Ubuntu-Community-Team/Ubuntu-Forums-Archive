### Post by anewguy on 2011-08-12
Not to muddy the waters, but you can indeed do router setup wirelessly.  I do that quite often - you just have to know if you change certain settings that the router is going to reset and you'll need to connect again.  If you are setting up something like MAC address filtering, and you are the first to set it up, be wise and use a hard-wired connection.  If you set up some method such as WEP, WPA, WPA2, etc., you'll probably get disconnected from the router after the change, but you can just connect again.
