### Post by cj.surrusco on 2010-06-30
You would probably want more than 1 GB of RAM for gaming, but it differs per game. I would check the system requirements that games post. 

As far as the HD video, you would probably want at least 3 GB of RAM and definitely a decent graphics card. 

Crucial is definitely a dependable manufacturer of RAM and Gskill is also pretty good. If I had to choose one, I would choose Crucial.

Also, you need a 64-bit system to use more than *4* GB of RAM.

EDIT: The link for the Crucial RAM that you posted is for NOTEBOOKS only. For desktops you need 240pin. Also, don't forget to check that your motherboard supports more than 1 GB in each slot. I made that mistake when upgrading RAM once.

### Post by 3rdalbum on 2010-06-30
Just ordinary DDR2 will work; it's the longer sticks, not the short stubby sticks (those are for laptops).

Don't buy Kingston memory, it WILL fail.

1080p video decoding is CPU-intensive. If you want to be able to decode 1080p video then you should buy an Nvidia graphics card (they can handle most of the decoding workload), but you might need a better power supply in order to run it. An Nvidia graphics card will allow you to play games too.

4 GiB of RAM is a lot. A real lot. You can do all you want to do with 2 GiB of RAM.
