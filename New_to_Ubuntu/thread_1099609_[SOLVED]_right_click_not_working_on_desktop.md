### Post by lukjad on 2009-03-19
Hmmm... well this post seems to have an answer:
[http://ubuntuforums.org/showpost.php?p=1435048&postcount=5](http://ubuntuforums.org/showpost.php?p=1435048&postcount=5)

```
sudo apt-get install gtweakui
```

Then go and click on *Preferences->gTweakUI - Nautilus*.

Under the **Desktop** heading, make place a check mark next to *Use Nautilus to draw desktop*. 

Hope this helps! :)
