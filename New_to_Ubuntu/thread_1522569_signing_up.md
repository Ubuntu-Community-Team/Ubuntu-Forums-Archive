### Post by MaracKoMarac on 2010-07-02
First, download it from [here]("http://earth.google.com/thanks.html#os=linux").
Then, right click on it, and check the "Allow executing file as program" box on Properties-Permissions. After that, run the terminal, type ```
cd /home/username/wherever_the_file_is
``` and then type ```
sudo ./GoogleEarthLinux.bin
```EDIT: I'm sorry, didn't see the link above :oops:
