### Post by Cheesehead on 2011-10-09
One good place to start is [https://help.ubuntu.com/community/UsingTheTerminal](https://help.ubuntu.com/community/UsingTheTerminal)

If there is an [Ubuntu LoCo Team]("http://loco.ubuntu.com/") or other Linux User Group near you, dropping in on a meeting is a great way to have a friend at your shoulder if you get stuck or have fast questions.

IRC is a good place to find immediate help, too. Both in your LoCo's channel or the normal #ubuntu help channel. Or here, of course.

If you have any particular goals in mind, like to play music using the command line, or learn how to make scripts, or create complicated database queries, just ask.

### Post by ktat on 2011-10-10
I'm realatively new to the commandline and have found that "A Practical Guide to Linux: Commands, Editors, and Shell Programming" is an extremely good introduction to using the terminal and understanding how linux is put together.  Also it is a useful reference for many of the utilities available at the commandline, so it's always handy to have close by.
