### Post by HermanAB on 2013-04-15
Yes, there is.

Install Virtualbox, then make a virtual machine and install Windows in that.  This way, you can run Windows in a window.  For most purposes, this is the preferred method to run both Linux and Windows on one machine, since you can run both at the same time.

There is a forum topic dedicated to this.
