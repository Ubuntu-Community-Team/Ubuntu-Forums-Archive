### Post by ugriffin on 2009-04-23
Synaptic and KPackage (or Adept) automatically check for updates when you're connected to the internet. Should you want to update manually, the command is:

```

sudo aptitude update
sudo aptitude upgrade

```

Have fun with ubuntu :)
