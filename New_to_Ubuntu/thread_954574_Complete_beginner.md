### Post by Idefix82 on 2008-10-21
I find that Ubuntu is a very painless way to learn the command line. You get by without it but you soon realise that many things can be done more efficiently with the command line. And when I say many, I include things like sending emails, simple image processing and of course data management.
The best way I have found so far to learn the command line is to take the jobs you are doing anyway and try to port them to the CLI one by one. Asking questions here is a good way of learning the relevant commands.

Here is an example: if you have a website with a basic picture gallery then you might have to do the repetitive task of resizing pictures to get thumbnails and saving them in some directory many times. Once you have done that three times you will start wondering whether you can script such a thing. And if course the answer is yes. This goes for many other tasks as well.

### Post by H.Callahan on 2008-10-21
> **halitech said:**
> just keep in mind, Linux is not microsoft so the way of doing things in windows doesn't apply here.
Just to expand on this a bit...

It is not so much that the things that you might want to do are in a different location than in Windows (and they are), but that some of the actual underlying concepts are different.  Software distribution/installation is one big one.  File system structure is another ... and there are many more.  Just don't fall into the trap of thinking that if I can do something on Windows, I can do the exact same thing in Linux.  Usually there is a way to achieve a similar result, but it will likely not be exactly the same thing.

Many of the changes are for the better.  A few things ARE better in Windows ([img]http://www.clipart.co.uk/clipart/mazeguy/animated/duck.gif[/img]).  Fortunately, there is (IMHO) more good than bad.

Just remember to keep in touch here when you have questions and problems (and you WILL have questions and problems).  The very best thing about Ubuntu over Windows and all the other Linux distributions is this community that is willingness to help everyone from the rankest beginner to the seasoned veteran with equal attention.
