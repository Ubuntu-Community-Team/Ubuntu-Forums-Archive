All are compatible. Ubuntu only have Chromium in SC because Chromium is open source while Chrome isn't.

### Post by deadflowr on 2012-12-25
Yes do as Pjotr123 says.
When you get the .deb for google-chrome, right click or double click and the Ubuntu Software Center will open and install it like any other software.

The difference is chromium is open-source and chrome builds on that with added google stuff.(Chrome has flash built in but chromium does not).
