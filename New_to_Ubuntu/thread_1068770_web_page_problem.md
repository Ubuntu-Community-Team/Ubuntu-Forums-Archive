### Post by redseventyseven on 2009-02-13
I don't think this is a Ubuntu problem specifically, but I think this will help.

What button are you clicking on to close the window - the one in the top right corner, or the one just above where the page is displayed?

The link may be opening in a new *tab*, rather than a window. See [http://www.mozilla.com/en-US/firefox/tabs.html](http://www.mozilla.com/en-US/firefox/tabs.html) for details.

Alternatively, try using the browser's "Back" button instead of closing the window completely.
