### Post by PPPilot on 2011-05-11
I don't exactly know what you mean by going to sleep but your problem could be the new Unity Desktop that comes with 11.04. Logout and select "Ubuntu Classic" from the session menu before you log back in. This will get you the regular Gnome Desktop which should work. If you can not logout you may have to reboot in order to get to a your login screen. There seems to be lots of issues with the Unity Desktop. I have been running 11.04 since beta and my desk top machine still will not run it. My laptop will run it but I do not like it at all. Hope this helps....
