### Post by Impavidus on 2018-02-05
I suggest a lighter flavour of Ubuntu, like Xubuntu or Lubuntu. Basically the same system, but with a lighter GUI and some lighter default applications, using the same repositories, so the same applications are all available. The LTS releases of the light flavours are only supported for 3 years, so you have to take either 16.04 LTS (supported until April 2019) or 17.10 (supported until July 2018). Move to 18.04 LTS later this year.

### Post by TheFu on 2018-02-05
> **vinothmathew said:**
> thank you impavidus. so if upgrade 14.04 to 16.14 LTS i'll get the better process?

No.  For more speed, you need to change the Desktop Environment, DE.  
You don't need to do a fresh install either, but moving from 14.04 to 16.04 would be smart, since the 14.04 desktop applications are much older versions.  Mate, XFCE, LXDE are all lighter DEs.  Mate is pretty popular and includes most "control panel" GUI management tools.

Steps:
* sudo apt update
* sudo apt upgrade
* sudo apt install ubuntu-mate-desktop
* logout, before logging in, choose the "gear" and select MATE from the choices.
* login and see the new GUI.
* run for a week or so with MATE to verify that you like it or not.  If so, purge unity-* from the system.

After all that is done, you might want to update to 16.04.  
* sudo apt update
* sudo apt upgrade
* **make a know-you-can-restore backup; this is critical.  ** If anything bad happens in the next step, a backup will be highly, highly, highly, desirable.
* sudo do-release-upgrade

I moved 3 systems from 14.04 to 16.04 this week.  2 servers worked fine. 1 desktop had/has some issues still.  The middle mouse button clicks on the desktop stopped working. The middle mouse works everywhere else and the scroll wheel works still too.  Perl was broken, which seems to put the system into a very slow shutdown cycle.  12 seconds after I'd get a desktop, systemd would tell it to shutdown without my request.  Trying to fix an issue like that with just 12 seconds before a shutdown isn't easy. ;)

Definitely have a great backup. In my experience, you won't need it 2/3rds of the time, but for that other 1/3rds, you'll wish you had it.
