### Post by guitar player on 2009-11-07
Just make sure you shrink the vista partition with vista's disk management utility I would think it could sort stuff out on its own.  To get to it right click on my computer, click on manage, then click disk management, click on the partition you want shrink then right click and then click on shrink volume. Then when you install kubuntu have it use empty space.   Edit: also I forgot to remind you to back up important files before you mess with partitions.

### Post by seenthelite on 2009-11-07
There are many ways of doing this but a safe way is to go to disk management in Vista, select the partition that Vista is on and shrink it. Then when you install Ubuntu select install in largest free space. This will result in a dual boot if this is what you want.
