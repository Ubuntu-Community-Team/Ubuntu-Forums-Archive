### Post by Rex Bouwense on 2011-08-31
Are you talking about the different Ubuntu kernels that you can boot from and the Memory Test?  For each Linux kernel that is installed you will have a recovery boot option.  These other options can be eliminated but then you might be in a fix if you could not boot from your current kernel or you wanted to test your memory.  Just like in Microsoft Windows without a Safe Mode.
