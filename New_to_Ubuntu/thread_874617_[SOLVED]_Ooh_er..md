### Post by powerpleb on 2008-07-30
I have that exact laptop. Everything works like a charm out of the box except wireless, which is automatically detected incorrectly by Hardy. To get it working you need to compile a patched version of madwifi... These are the instructions that I followed:
[http://www.ubuntugeek.com/atheros-5007eg-with-madwifi-on-i386-platform-2.html](http://www.ubuntugeek.com/atheros-5007eg-with-madwifi-on-i386-platform-2.html)
[http://ubuntuforums.org/showthread.php?t=816780&highlight=acer+5315](http://ubuntuforums.org/showthread.php?t=816780&highlight=acer+5315)

I need to recompile the driver every time the kernel is updated, but apart from that everything is working fine.

BTW... I told Acer that I put Linux on it when I asked them to repair a dodgy headphone jack. They had no issue with completing the repair because it was a hardware issue and not a software one.
