### Post by fakstory on 2018-08-24
To untar it, run in a terminal:

tar -zxvf name_of_your.tar.gz

if you see a .deb in there, simply run:
sudo apt install your.deb

(but I doubt, you will probably see a script file)

cd to the "install" directory and then run:
./setup /net

Also, have a look at this link:
[https://www.openoffice.org/dev_docs/setup-linux.html](https://www.openoffice.org/dev_docs/setup-linux.html)
