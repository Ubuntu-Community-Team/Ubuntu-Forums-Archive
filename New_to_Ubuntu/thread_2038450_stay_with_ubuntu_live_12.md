### Post by cryptotheslow on 2012-08-06
Have a look at Startup Disk Creator - you'll find it in the launcher menu - just search for startup and it will appear.

If you use that to write your ISO to the flash drive it provides an option at the bottom part of the screen:
[b]When starting up from this disk, documents and settings will be:
"Stored in reserved extra space"[/b]

That will allow you to save your docs etc. on the "live" disk between bootups - just make sure you set a suitable amount of space when creating the flash drive.

This may also be a useful read:
[https://help.ubuntu.com/community/LiveCD/Persistence](https://help.ubuntu.com/community/LiveCD/Persistence)
