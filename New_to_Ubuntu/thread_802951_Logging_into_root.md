### Post by cprofitt on 2008-05-21
1.  the root account by default in Ubuntu does not have a password. You can achieve the running as 'root', typically, by using sudo for terminal programs and gksu (gnome) for graphical programs.

2.  updating the bios would likely help, but I am not sure if you will be able to find a bios for an old computer -- it will depend on the manufacturer.
