In the terminal, how can I cd to a directory with a space in the name?

---

### Post by kaibob on 2009-12-05
> **Gyrotwister said:**
> In the terminal, how can I cd to a directory with a space in the name?

Put the directory in quotes. For example,

```
cd "home/kaibob/financial data"
```

An alternative,

```
cd home/kaibob/financial\ data
```
