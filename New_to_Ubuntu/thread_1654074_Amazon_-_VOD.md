### Post by lovinglinux on 2010-12-28
Your problem is possibly due to the fact that flash in Hardy is actually version 9 instead of 10, even considering the package in the repositories says it is flash 10.

Get [Flash-Aid]("http://www.webgapps.org/addons/flash-aid") extension, but instead of installing flash when prompted, use the **Remove** option from the extension menu. This will make sure any conflicting plugins get removed.

Then grab the latest flash from Adobe:

[http://get.adobe.com/flashplayer/](http://get.adobe.com/flashplayer/)
