### Post by jtarin on 2011-07-09
When you get to the login screen switch to a virtual terminal by one of the key combinations "Ctrl + Alt + F1-F6". Then login there as the user your having problems with and at the command prompt after successful login issue the command "startx" without quotes and note and report any errors. To get back to a GUI use "Ctrl + Alt + F7".
