### Post by avtolle on 2008-07-10
Since it is older hardware, you might want to download the alternate iso install (from the same place as you d/l the Live CD iso), burn it slowly (4x) to a CD-R and give it a try. The alternate iso has a text based rather than a graphical installer, which seems to work better on older hardware.
