### Post by e4uforums on 2010-04-15
Reinstalling the 32 version of Ubuntu on the same partition as the 64 bit version will effectively erase the 64 bit version.

Although if you're reinstalling anyways, it's not gonna hurt to do a full format.

### Post by srs5694 on 2010-04-15
Note that if you installed without a separate /home partition, you're likely to lose all your user data files when you re-install, so you should be sure to back up anything important first. I recommend re-installing with a separate /home partition, if possible.
