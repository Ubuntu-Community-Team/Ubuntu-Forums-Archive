Quick Question:

If you keep doing the occasional updates via the Repositories, is it necessary to do an update in April when the next version is released?

Having just installed 8.10, should I plan on doing an install of 9.04 when it comes around? or will 8.10 plus the Updates amount to the same configuration?

---

### Post by dorkdork777 on 2009-02-28
No, you will need to do a large upgrade when 9.04 rolls round.

This is because it introduces new features and configurations, and to keep the OS constant, routine upgrades for any version don't add functionality, only fix bugs. Only "backports" upgrades (from the optional backports repository) and distro upgrades (e.g. going from 8.10 to 9.04) actually add new features.
