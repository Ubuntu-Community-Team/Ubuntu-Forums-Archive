Some of those old computers can boot to a CD by using SBM  (smart boot manager) it goes on a floppy disk and the files are on the Ubuntu live CD.  

When I 1st started using Linux - had to use it - put this web page up - lol have not looked lately - [Nuts on Smart Boot Manager]("http://louboldt.com/ilinuxsbm.htm") - also the Ubuntu wiki has a page on it also.
