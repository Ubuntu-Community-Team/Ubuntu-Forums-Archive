### Post by forger on 2008-08-30
I suppose you are connected on the server using  Places > Connect to server ?
If so, head to menu File > Save as
include .txt in the filename.

But be warned, in windows they have to read it with either MS Office Word or Wordpad, since Notepad will show everything in a single line.

### Post by Cheesehead on 2008-08-30
If you saved the file as 'foo',
then rename it (or 'save as') 'foo.txt' or whatever extension it needs to make the other end happy.

Linux (mostly) doesn't use extensions to determine file type.
