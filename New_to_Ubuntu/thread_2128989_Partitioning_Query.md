### Post by oldfred on 2013-03-25
My wife does not use iTunes, but has never connected iPhone nor iPad to computer. I have backed up photos to Ubuntu with direct connection, but have seen a lot of users trying and having issue with music on iPhone and Linux.

Generally virus' are not an issue, as they are designed for modifying Windows and Windows software. Windows does not even see the Linux partitions. But a very nasty virus that erased entire drive, partition table or MBR could of course cause problems. 
In Linux you can get a virus but it has to spoof you into thinking it is something you want and you giving it root access. It cannot just run a program in the background as it cannot get past sudo for file ownership and permissions.
