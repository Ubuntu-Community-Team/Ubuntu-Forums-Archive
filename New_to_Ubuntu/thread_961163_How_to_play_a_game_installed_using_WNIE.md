### Post by Xiong Chiamiov on 2008-10-28
That should be all you have to do.  If it doesn't launch, you can launch it through the terminal.  You'll have to navigate to that directory (know how to cd?), then run it like so:
```

wine ./FM.exe

```
If it doesn't work, you should at least get some useful output.
