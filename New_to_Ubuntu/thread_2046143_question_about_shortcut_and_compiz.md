### Post by Lisiano on 2012-08-22
A shortcut is just that, a shortcut. It is just a pointer to a file, so if you copy it anywhere, it will just point to a non-existant file. You need to copy the actual file if you wish to use it. 

As for Compiz, I don't think you can use it without 3D acceleration. Try to install Vmware Tools, maybe they will provide the driver.
