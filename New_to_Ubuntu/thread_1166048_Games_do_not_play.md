Hey matt, I'll try to answer your issues

*At first they worked but not too long after they stopped working. Specifically what is happening is they won't even start up when i try to play them.*
I assume those are the windows games you've played through wine? Do you remember any changes that could've caused them to stop working?

If you could run them before without the CD, then you propably don't need to put the CD in now either.

*Most of these games start up automatically but when they try to it comes up and gives and auto-run error.*

can you elaborate this error, what does it say exactly?

*I have been told that older windows games should run very easy on unbuntu but so far no of the old games that I have want to work and some of them i can't even install.*

Generally, the older a game is the more likely it will run in wine. Old games like black & white barely run properly while spore works like a charm. There is a database on how well games and applications work in wine: [http://appdb.winehq.org/](http://appdb.winehq.org/)

However, keep in mind that running games in wine is not an easy task. often you need to make detailed configurations and still not get the full performance or functionality. I recommend you to consult an experienced linux user to get games working on wine.

Oh, and having KDE along with gnome does not make ubuntu grumpy ;-)
