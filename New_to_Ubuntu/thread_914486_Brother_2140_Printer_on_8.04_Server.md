### Post by graben3 on 2008-09-08
You probably tried this already but lets ask in case your missed this :

Go into Administration -> Printing

Then click new printer

Select Brother as the manufacturer

Then as this link says :
[http://www.linuxprinting.org/show_printer.cgi?recnum=Brother-HL-2140](http://www.linuxprinting.org/show_printer.cgi?recnum=Brother-HL-2140)

Use the HL 2060 driver for this printer
So select this printer model and all should work OK
