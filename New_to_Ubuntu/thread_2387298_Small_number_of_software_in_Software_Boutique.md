### Post by cruzer001 on 2018-03-17
No, there is nothing wrong.  The Software Boutique is not intended to cover all packages, but packages selected by  the Mate community.

For a complete package list install synaptic package manager.  I believe its in the  Software Boutique.  or

```
sudo apt install synaptic
```

[https://www.google.com/search?client=ubuntu&channel=fs&q=synaptic+package+manager&ie=utf-8&oe=utf-8](https://www.google.com/search?client=ubuntu&channel=fs&q=synaptic+package+manager&ie=utf-8&oe=utf-8)
