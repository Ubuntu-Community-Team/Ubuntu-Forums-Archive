### Post by drs305 on 2008-11-27
And we might as well address this topic, as you will come upon it eventually ;-)

If you are going to copy text *from a terminal* (a command for example) instead of CTRL-C it is CTRL-SHIFT-C. To paste a command *into the terminal*, you use CTRL-SHIFT-V (or middle click mouse). 

So if you are inside a terminal window, use the SHIFT key in combination with what you are used to. If you are outside a terminal window, use the standard copy/paste commands.
