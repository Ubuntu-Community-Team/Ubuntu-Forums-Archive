### Post by uRock on 2010-11-13
Try installing firmware. Open you Applications menu, then click Ubuntu Software Center, then enter **linux firmware nonfree** in the search box, then install the package. This has woked for a me a few times when my wireless wasn't working.

To install the package via the command line, open a terminal via Applications> Accessories> Terminal in the menu, then copy and paste this command, which will install the needed package. ```
sudo apt-get install linux-firmware-nonfree
```
