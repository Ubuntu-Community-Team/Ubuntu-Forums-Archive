### Post by issih on 2009-03-13
As far as compilers go, it obviously depends on what you are planning to code in, but a good start is to install the "build-essential" package. That contains C and C++ compilers, its really intended so that people can compile programs released as source code, but its still what you need. I think a version of java comes installed, and you can get hold of sun's implementation from the repositories. Finally interpreters for python and various shell scripting languages are built right in.

I'd go poke around the programming sub forum here if that is what you are interested in, there must be a million threads on getting started in programming in there, and just as many opinions on how to do it :)

Hope that helps
