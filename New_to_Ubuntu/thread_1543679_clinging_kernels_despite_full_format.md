### Post by pbrane on 2010-08-03
I'm not familiar with a wubi install, I don't use windows, but you should try booting to Ubuntu 10.04 and look at your grub config file. Assuming wubi installs use grub boot loader.
In a terminal:
```

gedit /boot/grub/grub.cfg

```
look for *menuentry* items and see if any match Ubuntu 9.10. I think they are left over from you 'clearing' your disk with vista. Maybe. Note that you will not be able to edit/change the file, with this command.

Are you sure that there are no ubuntu partitions or installations other than 10.04? If that is the case then you may be able to fix this by typing this in a terminal:
```

sudo update-grub

```
It will ask for your password to continue.

### Post by JKyleOKC on 2010-08-03
It looks as if the 9.10 versions are on your 250 GB drive, which is probably D: in Windows. If it's an external drive, you can unplug it when you run "sudo update-grub" to prevent the updater from seeing them.
