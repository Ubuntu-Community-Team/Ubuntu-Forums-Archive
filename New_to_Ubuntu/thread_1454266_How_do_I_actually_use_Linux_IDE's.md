### Post by doas777 on 2010-04-14
In general I find visual studio to be much better than either netbeans or eclipse, but both are decent IDEs that i have used extensively. you should be able to get them going without too much issue. just remember, both of these started as java IDEs so you will need to install the plugins for C/C++ to make use of those features. 
sometimes other langagues in eclipse don't get the same attention that java does. I occasionally use pydev in eclipse when i need step through debugging, but it is buggy at best so i usually do most of the work in geany, and just pull out eclipse when I'm really scratching my head.
