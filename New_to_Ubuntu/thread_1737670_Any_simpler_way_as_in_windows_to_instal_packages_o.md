### Post by Locke_99GS on 2011-04-23
Linux is not Windows. 

Scouring the web and installing softwares as you do in Windows is not the recommended method in Linux, and is very very seldom required.

**Applications -> Ubuntu Software Centre** if you're using a newer version of Ubuntu. Most of the things you'll want will be available through that. Simply select what you wish to install, and click "Install". Couldn't be any easier.

### Post by Vaphell on 2011-04-23
in ubuntu you can install hundreds if not thousands of programs from a single place - can't get easier than that. In windows you have to visit 30 different websites to download bunch of setup.exes to have a functional system - i don't see how anybody can call that convenient (unless years and years of using windows made you sorta brainwashed and resistant to change)
On top of that the package manager takes care of apps being up to date, while in windows world everybody and his dog has an ambition to put his own updater into the system tray (not to mention that people don't really bother to update apps that don't update themselves so after some time there are dozens of apps to exploit)

if you really want setup.exe route - look for .deb files
as an inexperienced user you should look for software in order:
- official repositories
- 3rd party repositories (so called PPAs)
- deb files
- source code

### Post by BertN45 on 2011-04-23
The easiest ways to install programs is to use the Ubuntu Software Center and their are 35.000 programs available for download. One central spot where you can find almost all programs you could need. If you download from there, you will receive also updates of the programs. It is is superior to what Microsoft offers. If you download a programs from elsewhere, make sure you download the deb file. ### Post by Zimmer on 2011-04-23
[https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)

Is that what you were specifically looking for?
If yes then Ubuntu Restricted Extras is the search to make in Software Centre.

Most of what used to be 'difficult' is now found in the Software Centre with a simple search...
If it is not there then you may not have the required Repository added (Edit Software Sources) .

If it is not in any of the Official Repositories then you want to be looking for a .deb file to download ( rare event , the only 'normal' one I can think of at this point is Opera ) Java can be found in the Software Centre (Multiverse Repository must be enabled).

Hope this has helped..
