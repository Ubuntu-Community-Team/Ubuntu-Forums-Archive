### Post by beelzebufo on 2014-02-20
I have had the same problem it is VERY difficult to customize your ubuntu install now for some reason, I hope they are working on it.  There are several tutorials that you can use to get things to work, most of the themes that you can find on gnomelook.org will work if you follow the instructions, make sure you install unity-tweak-tool and gnome-tweak-tool (especially the unity one) Which themes are you trying to install?

usually all you have to do is download the package (which is usually a tarball) unpack it into your /usr/share/themes folder, then use unity-tweak-tool or gnome-tweak-tool to switch it up.
