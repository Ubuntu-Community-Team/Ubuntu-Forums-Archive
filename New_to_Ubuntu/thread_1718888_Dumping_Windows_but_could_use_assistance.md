### Post by Zintha on 2011-04-01
My Suggestion to you?
Duel-Boot.

I play a fair few games, mostly Steam stuff and maybe WoW.
While, after a week, I got wow working perfect in ubuntu 9.10 (was a while back) I still kept windows on duel-boot for other games, including my iPad. It so much easier to just flick across to windows to do stuff I don't want problems with, stuff I know that just works there.

I know there are alternatives in Ubuntu but i've yet to get one working -just- how I like it.
However, don't force yourself away from windows if you don't have to or don't want to.

Keep the duel boot, if for nothing but needing it in the future.

EDIT: OR! If you have more than one computer, like me, put it on one, leave windows on another. (or duel-boot). I have my M11x laptop on Windows 7. Desktop on 10.10.
