### Post by jerome1232 on 2008-10-20
Oh boy, you are going about it the hard way :)!

It's in the repos!
```
sudo apt-get install wine
```

To use it you pretty much double click .exe files and it starts up.

Getting it to ***work*** with certain programs is the fun part, usually winehq's database of apps will have the workarounds posted though.
