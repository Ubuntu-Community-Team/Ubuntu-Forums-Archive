### Post by jtarin on 2010-10-10
[Get to work.]("http://ubuntuforums.org/showthread.php?t=812552")...no short cuts for you.
I'ts possible to do it easier in 10.04 and I have done it on my machine at home, but I'm not there now so can't guide you through that one, but Go to the language preferences under System | Administration | Language Support and firstly drag from the list displayed, the Japanese language to the top of the list using normal mouse pick and drag and drop, so that Japanese is at the top of the list.
Next select Install / Remove Languages and remove the English components from the list if you do not want them..
