### Post by sisco311 on 2008-08-13
add this to the fstab:
> 
UUID=489E-E1C6 /media/**insert-name-here(80GB)** vfat defaults,umask=0007,gid=46 0 0
UUID=1617BD3D5ECF9453 /media/**insert-name-here(250GB)** ntfs defaults,umask=0007,gid=46 0 0
create the mount points:
```
sudo mkdir /media/**insert-name-here(80GB)**
sudo mkdir /media/**insert-name-here(250GB)**
```mount the partitions:
```
sudo mount -a
```
