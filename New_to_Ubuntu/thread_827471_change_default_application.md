### Post by iaculallad on 2008-06-12
Goto System->Preferences->Preferred Application, select the "Multimedia" tab, and change/select the default application listed in there.

---

### Post by aktiwers on 2008-06-12
Or right-click a movie file and pick "properties"

Then find the "open with" tab and change the app there. After that this app will be default for all movies in this format.
