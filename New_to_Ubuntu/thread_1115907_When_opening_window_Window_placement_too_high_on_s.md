### Post by conryf on 2009-04-04

### Post by Peter09 on 2009-04-04
In Compiz-Settings-Manager (which should be under the Preferences Menu) there is an option for Window Management and in there there is an option for place windows, make sure its set to 'smart'.

Note if you do not have Compiz-Settings-Manager installed, its in the repos.

---

### Post by WakkiTabakki on 2009-04-04
Not really a solution, rather a tip:
Holding down the Alt-key and left-clicking anywhere in the window, allows you to move it (you don't need to click the title bar).

I _think_ this is a default setting in both metacity and compiz...
