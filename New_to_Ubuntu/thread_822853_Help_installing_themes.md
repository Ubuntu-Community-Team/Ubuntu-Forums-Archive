### Post by crashsystems on 2008-06-08
Ubuntu uses GTK 2.

Yes, I would say that this is the easiest way to install themes. You simply download a theme file from art.gnome.org (or gnome-look.org) and save it to your desktop. With the appearance app open on the Theme tab, click "Install" on the bottom right section of that window. The changes should take effect immidiately. Then with that theme selected, you can click "Customize" to tweak the settings. If you get everything exactly how you want, click "Save As" so that you can always revert to it in the future.
