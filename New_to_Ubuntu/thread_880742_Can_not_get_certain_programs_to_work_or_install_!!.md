### Post by decoherence on 2008-08-05
> **oDEIMOSo said:**
> What am I doing wrong . There is got to be a way to run these programs !!!!!

No there doesn't. WINE does not provide compatibility with all Windows software. It provides an open, Windows-like API that many Windows programs can run on.

Getting a program to run on WINE is generally beyond the scope of these forums, unless there is someone here who already has those programs running. You may have better luck getting help or advice from folks at WINEHQ 

[http://www.winehq.org/](http://www.winehq.org/)

they have a database of applications that are known to run. both zbrush and mudbox appear on the list so perhaps try their user forums?

good luck
