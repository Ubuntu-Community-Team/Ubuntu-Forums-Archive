### Post by 123456789123456789123456 on 2009-03-29
sounds like the driver Ubuntu is using is not 100% compatable with the modem.  Since you are running xubuntu 7.10 this could be the case.  expecially if the modem is one that was not included for drivers for ubuntu when 7.10 came out.  If the reason for using 7.10 is hard drive space, and memory.  Try upgrading to 8.04.  It uses the same memory requirements as 7.10, and no extra space required.  It may actually have the correct driver for the modem.  If you don't want to try, search the web for ubuntu driver for that model modem.  Installing a driver that is more compatable should take care of the problem.
