### Post by mali2297 on 2008-08-12
Thunar can search files in the current directory.

For a global search, you can try [catfish]("http://software.twotoasts.de/index.php?/pages/catfish_summary.html"). It is a GUI frontend to *locate*, *find* and other search tools. It is available in the repositories.
