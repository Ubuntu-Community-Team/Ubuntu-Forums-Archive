### Post by philinux on 2009-07-25
Open synaptic search firefox or

sudo apt-get install firefox-3.5

It will end up under apps> internet.

It copies you 3.0 profile and will be called shiretoko for now. You'll have both on your system this way.

Others will chime in with different ways but thats my system set up for now.
