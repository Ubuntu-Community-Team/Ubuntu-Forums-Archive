### Post by lkraemer on 2009-01-16
I am using Ubuntu 8.04 LTS with Kernel 2.6.24-23 Generic and xsane
.995 included with Ubuntu.

Here is the BUG I have found when scanning with my HP 3970 Scanner.
Scan any full page of text at 50 DPI in Color.  It will be displayed
in a nice window, and allows the window to be Maximized, Minimized
and closed.  I didn't try the ALT F10 & ALT F11 on this window.

Now scan the same page at 300 DPI and there is no window.  You can't
do anything to the window.  But, you can attempt to close the file
viewer, but that doesn't work either.  By using ALT TAB once you can
get the small Close File window to come up, and now you can close the window.
I should be able to use ALT F10 and ALT F11 to change the window size
before closing the file!  It works in the Beta version of PCLinuxos
N1PTT-TR5.ISO with the same xsane .995 version.
