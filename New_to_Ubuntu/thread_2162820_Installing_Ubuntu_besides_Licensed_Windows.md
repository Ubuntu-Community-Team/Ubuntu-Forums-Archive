### Post by Mark Phelps on 2013-07-16
To be safe, BEFORE you go the dual-boot route, do yourself a favor and use the Win8 option to create a "recovery Drive" (actually, a disk).  This will come in handy later should the boot loader become corrupted from the dual-boot installation.  And do NOT, repeat NOT, use the Ubuntu installer to shrink down Win8.  Doing so can lead to filesystem corruption in Win8.  Use only the Win8 Disk Management utility to do the shrinkage.
