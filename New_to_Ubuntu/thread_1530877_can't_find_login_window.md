There's a new version of Ubuntu every six months. The pace of development is very fast, and things change very quickly. As a result, you can't use tutorials from old versions of Ubuntu on newer versions.

In Ubuntu 9.10 there was a new version of the login screen that was a full rewrite, for various reasons. They did not add much of a theming capability to the new login screen. So you can't use the old GDM themes. You can, however, use Ubuntu Tweak to change the background picture and controls (GTK) theme for the login window.

So yeah, remember in future that if a particular tutorial is for a previous version of Ubuntu, take it with a grain of salt; things may have changed and it won't necessarily work. Of course, this is the same for other operating systems such as Windows, but with Ubuntu it only takes six months for a new version to come out.
