### Post by dizee on 2008-12-03
You don't need to go to a website.

Select & install the one you want from the Synaptic Package Manager (in the System>Administration menu). For example for KDE you would install the package "kubuntu-desktop".

Then when it is installed, logout and click on the Sessions button on the login screen before entering your username and password. This will give you the option of choosing a desktop environment.

It's as simple as that.

If you need any info about the different desktop environments and window managers just ask.
