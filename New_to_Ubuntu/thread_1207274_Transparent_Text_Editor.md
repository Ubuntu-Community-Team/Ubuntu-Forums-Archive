### Post by nhasian on 2009-07-08
open ccsm by going to System->Preferences->CompizConfig Settings Manager.  tick the box to enable *Opacity, Brightness and Saturation*  Now you can adjust the opacity by holding down the ALT button on the keyboard and using the scrollwheel on your mouse.  alternatively you can set specific opacities for different windows in the ccsm.
