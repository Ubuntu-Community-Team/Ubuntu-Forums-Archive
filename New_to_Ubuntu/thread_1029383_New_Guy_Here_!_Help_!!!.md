### Post by 2hot6ft2 on 2009-01-03
For the graphics cards drivers go to System>Administration>Hardware Drivers and install the one it recommends.

It will download and install the driver. When it finishes it will require a reboot. After the reboot go back to System>Administration>Hardware Drivers and make sure it's active. If not activate it. If it is you can go to System>Preferences>Screen Resolution and set it.

All done with the graphics driver.
