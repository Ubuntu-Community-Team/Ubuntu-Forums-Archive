### Post by whitethorn on 2009-06-04
All you need to do is download the GoogleEarthLinux.bin once you have it on your desktop. 

```

cd Desktop
sh GoogleEarthLinux.bin

```

Afterwards it should ask you where you want to install googleearth.  Just install it into your home directory.  To start it just type

```

googleearth

```
or double click on the link on your desktop.
