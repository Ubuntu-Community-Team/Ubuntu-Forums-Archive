### Post by J697 on 2011-02-20
On the terminal type:
```

sudo apt-get install irssi 


```

Then type your password. After that finishes installing type in the terminal "irssi", that will start irssi, an irc client. Now type:

```

/connect irc.freenode.net


/join #winehq


```

That should get you the most help.
