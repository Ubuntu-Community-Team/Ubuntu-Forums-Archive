> **lorddeath said:**
> how do i install a java script on ubuntu so i can play games on yahoo games

I think your a bit confused, javascript is a language used by browsers. You don't need to install it, it comes standard with any browser including firefox. If you've installed noscript, allow the scripts on the page. The game is built with flash I believe, then please install the restricted extras package to get both flash and java installed, among other extras that many people find useful. To do so, please open a terminal (applications > accessories > Terminal) and then type the following and push enter (or copy paste):

```
sudo apt-get install ubuntu-restricted-extras
```

Then wait for it to finish installing, restart firefox and try again. To understand more about installing, see my sigs guide please. There is also a graphical means if ya go to the synaptic section.
