### Post by Sorivenul on 2008-12-14
The same way you install Java in the other Ubuntu versions, more or less.  To install Java with the rest of the restricted extras (Flash, etc.) open a terminal and do:
```
sudo apt-get install xubuntu-restricted-extras
```

If you want to do it graphically, it should be available in Add/Remove and Synaptic as well.
