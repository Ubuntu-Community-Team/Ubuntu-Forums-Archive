### Post by xedi on 2012-05-22
Security updates are quite important, because as the name suggests, they improve the security of Ubuntu and the software you have installed. If the notifications are bothering you, you can change the setting and configure how often you want to be notified of updates. In the case of security updates, you can even set it to download and install automatically.
