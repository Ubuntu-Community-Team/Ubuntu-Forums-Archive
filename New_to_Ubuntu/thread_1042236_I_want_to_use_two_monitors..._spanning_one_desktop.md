### Post by Garyhans on 2009-01-17
I have a NVidia 6600GT. I installed NVidia-Settings attached one Monitor to the DVI Out and the other the VGA out. I set mine up as x-server and have two individual desktops with cube of course. There is also an extended desktop option. I did this without running sudo nvidia-settings to experiment as to exactly what I wanted. Then sudo to set  my desired settings.
