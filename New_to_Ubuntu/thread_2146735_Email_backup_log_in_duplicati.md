### Post by Cheesemill on 2013-05-19
This is a Windows question rather than a Ubuntu one as the email functionality needs to be in the backup software, not the SSH server.

Currently Duplicati doesn't have this feature but it should be available shortly, you can read about it [here]("http://code.google.com/p/duplicati/issues/detail?id=29").

I'm afraid I can't advise you personally on any software that will do this as I don't back up my Windows machines, but searching for a free Windows backup program with email functionality shouldn't be hard - I'm sure someone else here will have some recommendations.
