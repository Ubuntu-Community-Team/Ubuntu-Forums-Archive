### Post by pratiks19 on 2010-01-13
While scrolling any webpage using keyboard arrow keys, firefox 3.5.7 shows the blinking keyboard cursor (as in a text editor). I've attached a screenshot of the same.
The scrolling really gets irritating when the webpage contains images.

I'm using Ubuntu 9.10 and this problem started after updating firefox.

Any solutions ?

---

### Post by Spooky5 on 2010-01-13
In Firefox, go to options, Advanced and then click on general tab. Uncheck the  "always use the cursor keys to navigate..." box. That should get rid of it for you.
