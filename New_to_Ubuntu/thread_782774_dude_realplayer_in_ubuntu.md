you can download it here:
[http://www.real.com/linux](http://www.real.com/linux)

Then from Terminal do this to install it:
(if you downloaded it to your desktop, go there first)
```
cd Desktop
```

Give yourself rights to execute it:
```
sudo chmod +x RealPlayer11GOLD.bin
```

and then run it:
```
sudo ./RealPlayer11GOLD.bin
```
