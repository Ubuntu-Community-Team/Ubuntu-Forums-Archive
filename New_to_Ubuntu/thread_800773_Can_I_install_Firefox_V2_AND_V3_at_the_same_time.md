### Post by Joeb454 on 2008-05-20
You can have both installed at the same time, but before you install FF2, try and download the User Agent Switcher, a Firefox addon.

From the tools menu, you can change what User Agent is sent to web pages (which some use to Identify the browser. Change this to IE7, and see if your bank lets you use it then ;)
