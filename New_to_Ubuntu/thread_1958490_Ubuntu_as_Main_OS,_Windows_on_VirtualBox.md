### Post by NadirPoint on 2012-04-14
I've been running XP in a Vbox session for years, just for iTunes.  Only complaints I ever heard from this setup was some games not working right.  That's not surprising, considering the video processing requirements some impose.  Adobe apps should pose no problem.  Allocate enough RAM and video to the virtualbox and it will run great.
