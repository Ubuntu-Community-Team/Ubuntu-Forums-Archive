### Post by NetworkGuy on 2008-07-02
Start with System / Administration / Hardware Drivers.

It will tell you if you are using the ATI driver or not.  If not, check the box, let it download the driver and restart Ubuntu.

---

### Post by markbuntu on 2008-07-02
Do not use the ATI fglrx driver. It does not support the 9000 card. Use the open source Radeon Driver from the Ubuntu repository.
