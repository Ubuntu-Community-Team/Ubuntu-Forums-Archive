### Post by The Cog on 2011-11-06
I haven't seen the books that crjackson suggests, but I suspect that he knows what he's talking about.

Don't be afraid to ask beginner questions here. It's the beginner forum, after all. And I guess that for every beginner question posted, there are a hundred other beginners who will benefit from reading the answers.

Gnome2, Gnome3, KDE, XFCE, LXDE and Unity are all GUI managers. It's a concept that windows won't have thought about because windows only has one, it's all just part of windows. But on Linux, lots of people have written different looking GUI managers. They all tend to have the same basic ideas - programs run in windows that you can (generally) re-size, drag around the screen, put overlapping with one in front and one behind etc. Most of them have something like the windows "Start" button from where you can launch programs, change settings etc. I think all of them put window "decorations" round the windows - a title bar, maximise, minimise, close buttons and maybe more. Unity (the default one in the latest Ubuntu versions) doesn't have a Start button, but has a set of buttons down the left hand side instead.

They all have a slightly different look and feel about them. It's rather like the different versions of windows, there MS add features, remove features, change the look, put things in different places with every new version. The differences between these managers go a little deeper than the differences between windows versions, although the underlying operating system is unchanged of course. 

They all tend to have all their own utility programs too, although I should point out that any program will actually work in any of the desktops. Each has its own file manager for instance (the "windows explorer" that shows all your files and folders and lets you copy, move, delete, open files). Gnome uses a program called nautilus, xfce uses thunar, kde uses dolphin, Unity uses nautilus, don't know about lxde. Again, these utilities are very similar so once you can use one you can use any of them, but they may look slightly different, and may have different menu options. As an example, when you drag and drop a file to another folder in dolphin, it always pops up a small menu asking if you want to copy the file, move it, or create a link. In nautilus, it sometimes copies the file and sometimes moves the file but never asks which you want (use cut/copy/paste to be sure). Ubuntu used to use Gnome2 but this year started using Unity by default instead. 

Both Gnome and KDE have released new versions that look and feel a bit different recently - Gnome3 and KDE4 are the new ones. The old versions will presumably eventually become unused, although I read that some KDE users dislike the new KDE4 so much that they intend to maintain KDE3. People can get very attached to a particular desktop appearance. 
Around 6 years ago I changed from KDE to Gnome2, and around 2 years ago changed to XFCE. But it's all personal taste. 

The best way to get a feel for the differences between these is to burn the live CDs for Ubuntu, Xubuntu, Kubuntu, Lubuntu and try them out for a few minutes each. The difference between these versions is which desktop system they install by default. Most folk will settle on one that they feel most comfortable with, and stick with that. 

For reference:
Ubuntu (pre 2011) uses Gnome2
Ubuntu (2011) uses Unity
Kubuntu uses KDE4 (KDE3 before maybe 2009)
Xubuntu uses XFCE
Lubuntu uses LXDE

P.S. You might find this reference useful:
[https://help.ubuntu.com/](https://help.ubuntu.com/)
