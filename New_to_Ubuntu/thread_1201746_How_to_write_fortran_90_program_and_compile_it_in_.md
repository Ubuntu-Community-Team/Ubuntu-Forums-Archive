### Post by ohzopants on 2009-07-08
You can create the fortran file in any text editor that you like, different editors will provide different features such as syntax highlighting and indenting (indenting being particularly important in fortran from what I remember).

Then you just need to compile it on the command line using gfortran (provided by the "fortran-compiler" package).
