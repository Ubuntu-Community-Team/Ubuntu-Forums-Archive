### Post by falconindy on 2009-09-03
I don't believe Wine uses its own temp directory. The Windows file structure resides in your home directory under '$HOME/.wine/drive_c'. It shouldn't require cleanup the way a normal Windows system does.

Personally, I wouldn't go near a Windows based cleaning program on Linux, but that's just my opinion.
