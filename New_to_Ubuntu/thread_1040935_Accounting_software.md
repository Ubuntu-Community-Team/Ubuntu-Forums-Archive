### Post by daaussie on 2009-05-11
I have been successfully running MYOB using the WINE emulator with 8.04. There are some issues with 9.04, but you can still use MYOB in that too.

How to get WINE?
Go to Add/Remove programs, then look for all open source applications, install WINE. 
Then you simply find WINE in your Applications list, click it, 
go to browse C: and then copy the MYOB install files in there. Then double click the MYOB setup file, it installs fine.
I copy a desktop shortcut to the panel and use that also.

There is a solution to every problem!
