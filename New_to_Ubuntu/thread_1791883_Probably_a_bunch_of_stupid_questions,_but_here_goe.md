### Post by sanderd17 on 2011-06-27
The best thing you can do is first of all burn a live CD, this way you can try a linux distro without installing it. Ubuntu is perfectly fitted for a first time you use Linux.

Just go to [http://ubuntu.com](http://ubuntu.com) and download the iso, then you can burn it with a CD burner application like the free infra recorder. Don't burn it with windows explorer, that won't work.

Once you burned it, put in the CD and reboot. Normally Ubuntu will show up. The boot will be a lot slower than normal (because a CD is a lot slower than a HDD and the OS is compressed to fit on a CD), but you should be able to try some things.

Some things you want to try:
[LIST]
[*]play music
[*]surf the web
[*]see if your media keys work
[*]see if the screen works well and if you can adapt the brightness
[*]check if suspend works
[*]try out the apps (libreoffice, firefox ...)
[*]install something from the software center (comparable to the OSX app store, but Linux has it for a decade already)
[/LIST]

When you have tried this things, you can click on the install icon and follow the guided install. There is an option to install Ubuntu next to Windows.

### Post by Beacon11 on 2011-06-27
Well, welcome! The first thing I would do is download Ubuntu and create a LiveCD, or put it on a flash drive. The instructions for this are located on the Ubuntu website-- go to [http://www.ubuntu.com/download](http://www.ubuntu.com/download) and select "Try it from a CD or USB stick." This will allow you to test Ubuntu out without touching Windows, to make sure that you like it.

If you want to completely switch from Windows to Ubuntu, the only thing you need to do is make sure you save all important files (e.g. documents, pictures, music, internet favorites, etc.) to some sort of backup. Then you can install Ubuntu over the top of Windows using the LiveCD/USB you created in the above step, and restore your files from your backup.

As for interoperability concerns, I'm not sure how much you know about Linux (particularly Ubuntu), so forgive me if I state the obvious. The biggest hurdle I think you'll run into is the lack of Microsoft Office. You won't have Word, Excel, Powerpoint, etc. You will instead be using an office suite called LibreOffice, which CAN use MS Office's formats, but it doesn't always work perfectly (e.g. formatting might be slightly off). If you'll be transferring documents around between operating systems, I would do my best to stick to PDFs if I were you. If you need to be able to edit documents on both Windows and Linux, stick with Microsoft formats, or install LibreOffice on Windows (the latter may not be an option).

I hope this helps get your thinking in the right direction anyway. Let me know if I can be of more assistance.
