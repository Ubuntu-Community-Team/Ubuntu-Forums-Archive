### Post by SunnyRabbiera on 2008-07-18
It depends on what you need.
You can start off with the bases by opening up synaptic located under system> administration
there is a way to check off extra repositories in it by going up to "settings" then to "repositories"
here you can enable extra repositories by checking off all the boxes except the source code ones. the first two tabs will have listed repositories for you to use.
If you need multimedia, try medibuntu:
[https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)

it has plugins and such you may need.

### Post by Bachstelze on 2008-07-18
You really should add none (yes, not even medibuntu, or actually *especially not* medibuntu). Using third-party, unofficial repositories is potentially harmful, and will give you a lot of headaches when you'll want to upgrade your Ubuntu.

### Post by HDave on 2008-07-18
I have using Ubuntu for over a year, and I have never needed to go outside the normal repositories.  Just be sure you have the universe and multiverse ones enabled.

There are very few programs I have installed that couldn't come from the repositories provided by Ubuntu:

Truecrypt
Google Earth
Picasa

and a few others.  I really wouldn't add any of these repositories unless you REALLY have a compelling need.  It could mess up your updates facility and waste your time.

Realize this -- you can install any programs from wine, medibuntu, getdeb, etc. and never add them to your repositories list.
