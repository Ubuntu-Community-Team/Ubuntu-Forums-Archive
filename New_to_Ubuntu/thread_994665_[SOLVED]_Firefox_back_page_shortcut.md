### Post by Yuki_Nagato on 2008-11-29
Go to the language bar in Firefox and type: "about:config"

It is the same as "about:blank" which brings up a blank page.

### Post by Xiong Chiamiov on 2008-11-29
It's actually the address bar, and if you type
```

about:config

```
then it will not be blank.  You can toggle values by right-clicking them.

I'd also suggest getting a mouse gestures extension.  There are several around, and, while you can do many things with them, the forward and back functionality alone will save you an incredible amount of time.  I have a hard time using other people's computers when they don't have one installed.
