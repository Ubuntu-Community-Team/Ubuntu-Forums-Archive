### Post by jerome1232 on 2012-08-30
If you press "Super + A" (super is the windows key on most keyboards) it will open the dash in Application Filter mode, which is the "pen, ruler, and paintbrush" icon.

If you hold down the super key, you will get a nice list of all the keyboard shortcuts for Unity.

Also, that is the dash, Heads Up Display or HUD, is what you see if you tap the ALT button while working in an application, it allows you to search the menu's for commands.
