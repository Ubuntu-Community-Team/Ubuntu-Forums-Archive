### Post by waynefoutz on 2010-09-27
Give Playonlinux a try before purchasing a commercial product. You'll find it in the Software Center. It gets the job done 90% of the time, and it's real easy to use. It will set games up for you, even to the point of downloading and installing the version of WINE that works best for the particular game you want to install.

Also, don't rule out native Linux games. Admittedly there are a lot of lame ones out there, but there are a few gems, some of them rivaling commercial Windows games in quality. A great place to look is this guy's youtube channel: 

[http://www.youtube.com/user/Jakejw93]("http://www.youtube.com/user/Jakejw93")
 I don't know him, but he shows off some of the better games, and it's worth a look.
