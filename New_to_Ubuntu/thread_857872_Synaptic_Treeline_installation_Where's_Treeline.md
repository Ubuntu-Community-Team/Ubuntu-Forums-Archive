### Post by Tim Sharitt on 2008-07-13
Apparently a launcher isn't installed in the menu by default, but it is installed. To add it to the accessories menu: left-click on the main menu and select "Edit Menus". Select "Accessories" from the pane on the left. Now, on the click the button labeled "New Item" on the far right. You should now have box asking for the applications name and a command to launch it. Enter whatever you want(probably Treeline) for the name and enter treeline for the command.Click ok, then click close on the edit menus screen. Treeline should now appear under the accessories menu as the name you entered.
