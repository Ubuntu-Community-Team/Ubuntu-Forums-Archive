### Post by Mark Phelps on 2013-06-29
Don't take this as being critical, but the fact that the programming language you mention, Visual Basic, is most closely associated with MS Windows implies that you have little or no familiarity with programming for other environments.

Your best bet would probably be to visit the programming forum here and see what tools and languages the folks are using.

Also, UI programming is a LOT more complicated than cranking out some VB scripts.  You will need to hunt down specific guides on UI programming -- ones oriented to doing that in a Linux environment, not Windows.

I've provided a link to information on Unity development -- so you can get an idea of what you're getting into.  You should read through it:  [http://unity.ubuntu.com/getinvolved/development/unity/]("http://unity.ubuntu.com/getinvolved/development/unity/")
