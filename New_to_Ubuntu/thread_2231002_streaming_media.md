### Post by Vladlenin5000 on 2014-06-23
The suggestion was to install the Linux version (via PPA) in your Ubuntu. You can also - eventually - install the Android version in the Google TV but it isn't mandatory.
Once you have the media center installed in a desktop (or notebook) in the same network and properly configured to stream its library (UPnP) any other capable app anywhere else in the network should be able to pick it up (Android, Ubuntu, Windiows, Mac, etc.).
