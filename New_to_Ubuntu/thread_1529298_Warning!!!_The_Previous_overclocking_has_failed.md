### Post by cryptotheslow on 2012-02-05
It may well be due to badly written firmware. Check your motherboard manufacturer's website for BIOS firmware and see if there are any updated versions for your **specific motherboard**. If there are **read** the release notes for each version higher than yours to see if it addresses such an issue.

If you find suitable updated firmware you can update your BIOS with it - follow the manufacturers instructions **absolutely to the letter** - proceed with caution, an incorrectly flashed BIOS **can** leave you with a PC shaped door-stop with no hope of recovery - so only attempt it if you have found a BIOS update that specifically addresses your issue and you are confident to proceed.

If the issue only crops up every now and then and you already have a method to resolve it then it may be sensible just to live with it if it is not too annoying - hence avoid the risk.
