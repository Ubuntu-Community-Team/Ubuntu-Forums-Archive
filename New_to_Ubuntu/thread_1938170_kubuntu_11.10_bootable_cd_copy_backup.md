### Post by Ghost_Mazal on 2012-03-09
Remastersys will do that for you. I frequently use it do build my own "distributable" cd that I install on other pc's that doesn't have internet. It makes an .iso file and you burn that and you have a boot-able , install-able cd/dvd that can be installed on any other pc. (or on your own pc if disaster strikes). That disc will include EVERYTHING , software updates , everything. Just the /home folders are not included (or any other extra data folders you have)
