### Post by Scrineym on 2011-03-17
I had the same problem with amsn before, what i had to do was go into my router and enable port forwarding (port 6901) is msn's default port for webcam and file transfers, i'm afraid i cannot help you with enabling port forwarding in your router as it is specific to the router brand on how to do so, once this is done in amsn press ctrl+ p for preferences, then go to connection and under file transfer peer-to-peer NAT settings enter 6901 as the port number and press test port, if a message in green appears saying your ports are well configured you will be able to use the webcam, otherwise you still need to set up port forwarding.

(Note if the person is using msn 2011 the camera wont work at all regardless of these settings, i'm guessing this is a problem with amsn development being behind msn development :P)

Let me know if this helps
