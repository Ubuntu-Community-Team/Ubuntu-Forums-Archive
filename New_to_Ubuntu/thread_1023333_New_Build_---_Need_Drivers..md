### Post by abyssius on 2008-12-27
You don't need drivers for your Audigy. Try double-clicking the speaker icon to activate the mixer. Click on the switches tab. Clear the check mark in the [Audigy Analog/Digital Output jack] check box. This should give you sound. 

The easiest way to install a driver for your video card is to install Envy. Go to [System/Administration/Synaptic Package Manager] and search for the Envy package, then run the program.
