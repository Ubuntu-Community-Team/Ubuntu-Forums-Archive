### Post by howefield on 2010-03-16
> **chrispche said:**
> Should I download the 64bit version of Ubuntu Linux? I currently have the 32bit version installed. What's the advantages?

The question is why not ?

One reason might be that if you have a working system and don't have more than 3 or 4 gigs of ram, and do little video encoding you probably wouldn't see a huge difference in performance.

Given that you would need to do a fresh install, it might not be worth the hassle.

If it were me, I'd wait till 10.04 is released, then do a fresh install with 64 bit. There are very few issues with 64 bit these days, witness the fact that there is no longer a 64 bit forum here, it is treated as business as usual.

### Post by KIAaze on 2010-03-16
The main problem I have with 64-bit is when I want to use some proprietary applications (skype, drivers, games), which only come as 32-bit binaries most of the time (even some repository packages are only available for 32-bit!).
But [getlibs]("http://ubuntuforums.org/showthread.php?t=474790") helps a lot with that.
Still, I wouldn't recommend it to someone new to GNU/Linux.
