"Jaunty" and "Karmic" refer to specific versions of Ubuntu: [http://en.wikipedia.org/wiki/List_of_Ubuntu_releases](http://en.wikipedia.org/wiki/List_of_Ubuntu_releases)
"Kubuntu" is a variant of Ubuntu with a different desktop environment (KDE instead of Gnome): [http://www.psychocats.net/ubuntu/kdegnome](http://www.psychocats.net/ubuntu/kdegnome)
The "minimal install" CD is a CD which only contains the minimum software. The rest is downloaded from the net as necessary: [https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)
"remix" is a lighter version for netbooks: [http://www.canonical.com/projects/ubuntu/unr](http://www.canonical.com/projects/ubuntu/unr)

Karmic Koala (v9.10) is currently the latest release.

Ubuntu releases a new version every 6 months or so. The first number corresponds to the current year, then the decimal corresponds to the month it was released. In addition to that, it gives a two part name to the release for easier reference, with the latter part being an animal. 

For example 9.04 I believe is Jaunty Jackalope. This release was official in April 2009. Some people leave out the latter name and just say Jaunty instead. 

Karmic Koala is 9.10, so when was it released? October of 2009. Some people just say Karmic.

The names are synonymous to saying "9.04, 9.10 or 10.04."

The latest STABLE release as mentioned is 9.10 (Karmic Koala). I would recommend downloading this as 10.04 (Lucid) is still not stable (have tried it myself and still a few bugs in it).

As for the differece between Ubuntu, Kubuntu, Edubuntu, Xubuntu, etc. It's just a matter of the look and feel of the desktop (think of it like themes in windows, but the windows and menus would behave differently in other versions) and the software installed inititally (which you could always get yourself for free later).

I really don't recommend the netbook remix edition if you have a screen of 10" or larger. Just go with one, like Ubuntu, and if you don't like it, you can get the kubuntu desktop easily later without doing a whole system reinstall.

Try burning the live cd's (or create a live usb by using the tool from pendrivelinux.com) and just try using the desktop environments of each and see which one you like before installing.

### Post by quadproc on 2010-02-26
A suggestion regarding which Ubuntu version to install: I chose to go with 9.04 at the present time because of some problems in 9.10.  In particular, the release notes for 9.10 indicate that the default file system (ext4) sometimes corrupts a disk.  That would be very bad news for me so I chose to stay back one step until the serious bugs are rooted out.

Regarding which of the "flavors" such as xubuntu, kubuntu, ubuntu, etc., I would recommend going with the most popular version provided that it has the capabilities that you need.  The popular version will have its bugs discovered sooner.

It is likely that 1 GB of RAM is good for any of the flavors; I find that my memory usage in Ubuntu version 9.04 is usually between 500 MiB and 1 GiB.  So far, I haven't seen any usage of swap so I don't know what kind of impact it has; probably depends a lot on your swap disk.
