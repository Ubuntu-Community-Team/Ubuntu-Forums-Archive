### Post by lechien73 on 2012-01-12
According to the directory listing at: [http://ppa.launchpad.net/team-xbmc/ppa/ubuntu/dists/](http://ppa.launchpad.net/team-xbmc/ppa/ubuntu/dists/) the oneiric distribution isn't in the repositories.

Do you have xmbc installed? The easiest thing would probably be to disable this repository in Software Sources. To do this, you can open the Ubuntu Software Centre, then go to the **Edit** menu, choose **Software Sources** and click the **Other Software** tab. Then deselect the repository.
