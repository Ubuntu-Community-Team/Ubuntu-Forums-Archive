### Post by ubudog on 2012-01-25
Hi there!

With the "Create Launcher" window, you can create a launcher for a specific application.  For example, lets say I want to create a launcher for Thunderbird (assuming it didn't make one upon install).  

Type: Application
Name: Thunderbird
Command: thunderbird
Comment: An email client.

Then you can add an icon by clicking on the spring in the top left corner.  :) 

Hope this helps,
ubudog

### Post by ubudog on 2012-01-25
You don't have to do that for every program.  It's designed for programs with no launcher.  In Ubuntu 10.10 and below, you can drag the program from the Applications menu onto the desktop or panel.  In 11.04 and above, you can search for the program and drag it into the launcher bar.
