> I have 11.04 in my laptop, and want menus.  Where is Software Centre?  If I can't find it, it is too hard to find.  Do I have to type the name in somewhere to find what I'm looking for, like every single thing is a search?
If you're using 11.04, then you can use Classic mode, which is GNOME 2 that previous versions used by default.  Software Centre should still be on the left-hand dock in Unity, though.  The very last button, if I remember correctly.

For versions 11.10 and higher, in order to get the same sort of interface as GNOME 2, you have to use GNOME 3 Fallback, Maté (which is a fork of GNOME 2), or can experiment with Cinnamon.  All of these need to be installed after-the-fact, though (Fallback through the official repositories/Software Centre, Maté and Cinnamon through PPAs).  Or you could install xfce, LXDE, or other desktop environments.  In any case, after installing one of those other options, you can just log into those when you start up instead of Unity.  You're not tethered to the default interface at all.

Funny enough, you don't need to remember any of the program names.

First, you can add any programs you sue often into the launcher panel on the left side of the screen.

Second, you can browse through the programs, they are conveniently grouped into categories based on what the program is for so finding one shouldn't be any problem.

And third, even if you end using the search to find the program, you don't need to know the program's name. Searching for *what you want to do* will show appropriate programs. So search for "text" will show you both Text Editor (Gedit) and OpenOffice Writer, search for "photo" shows both Shotwell Photo Manager and GIMP Image Editor and so on...


...and in the end, this is not a question of what Linux distribution you use. It depends on what desktop environment you choose to use. In case you didn't know, you can run many different desktop environments on Ubuntu so if the Unity user interface isn't to your liking, just use some other one. ;)
