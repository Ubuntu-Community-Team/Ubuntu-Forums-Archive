### Post by marshmallow1304 on 2010-09-04
There's an [unofficial Windows port]("http://ankitjain.org/blog/giver-on-windows/") of Giver, but I don't know how stable it is.

I think you could also install [Pidgin]("http://www.pidgin.im/download/windows/") and [Bonjour]("http://support.apple.com/kb/DL999"), then create a local Bonjour account in Pidgin and use that for sharing files.
