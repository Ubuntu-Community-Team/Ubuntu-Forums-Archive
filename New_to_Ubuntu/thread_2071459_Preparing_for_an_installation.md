### Post by TheFu on 2012-10-15
That is a tough question.  The answer is "it depends."

It depends on many things:
* whether the laptop is compatible with Linux
* whether the laptop has any unsupported hardware
* whether the HDD is older or newer
* whether the laptop chipset is supported or not
* whether the available video card drivers can output to the external port
* how the external video port is connected - DVI, HDMI, VGA, something else?
* whether the wireless network chips used are supported

While everything might be fine, there isn't any way to know until you try.  You can create a 10G partition and install there with a test install. If it works, great. If not, you can work on cleaning up your Windows install from all the bloat.  Linux installs can become bloated too, if you are constantly installing new software and not removing that which is not used.

Does a broken laptop screen matter? I don't know, but if you see the BIOS before boot up, then I wouldn't worry too much about that.

Until you try, you cannot know.
