What software did you download? Did you download the .iso file (the only file you need). You need the 32 bit PC version, if you have a Windows based laptop.

If you did download the .iso, did you burn it to a disk as you were supposed to do? or did you just extract it into a hard drive?

The disk you burn from the .iso file is exactly the same as the disk that will be mailed to you. 
Unless the burned copy is defective, getting a mailed copy won't help if there is something else wrong with your system.
You can save yourself a lot of time by burning it yourself.
