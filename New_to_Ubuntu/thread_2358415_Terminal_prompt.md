### Post by Impavidus on 2017-04-13
The prompt usually looks like ```
user@host:directory$
```So in one of your cases the current directory is ~, which is your home directory, and in the other case it's /, which is the root directory. The home directory is where all your personal files are stored. Every user has his own home directory. The root directory is the base of the entire file system.
