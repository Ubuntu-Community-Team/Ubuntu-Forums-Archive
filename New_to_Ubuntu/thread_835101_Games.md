### Post by fatality_uk on 2008-06-20
There is a possibility. You need to install an application called WINE to run these Windows applications. 
[http://www.winehq.org/](http://www.winehq.org/)
Check the App DB above and see if the programs you want to run are listed. If so, go to the Synaptec Package Manager and do a search for WINE and install it.
