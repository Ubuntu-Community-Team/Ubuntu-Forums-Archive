### Post by Aearenda on 2009-11-03
Use Netbook Remix, change the panel height to about 20 pixels, change the dpi to 86. I also set maximus so it doesn't automatically maximise everything - that's a gconf setting under apps->maximus.

Oh, and I use an external screen and keyboard at my desk - then the Netbook is a true laptop replacement (almost).

### Post by NickJones on 2009-11-03
Dropping down the DPI will not get more on the screen, the higher DPI (Dots Per Inch) the more info can be displayed. However increasing the resolution from say 600x800 to 1200x700 will give you way more stuff onscreen.
