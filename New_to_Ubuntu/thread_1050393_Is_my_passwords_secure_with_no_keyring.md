### Post by Sam Lars on 2009-01-25
I'm thinking that each program has to get its own authorization to access the keyring.  Only main Gnome things are stored to the keyring... Firefox, etc. will each have its own management for that.
If you don't have a keyring password, that just means that those applications can access their passwords automatically.  Of course, it would be better to set a password on it, but for most people I think it would be more trouble than it's worth.
Personally, I don't have one set.
