### Post by bigmonmulgrew on 2010-12-07
Never mind I started googling the boot options I didnt understand in a hope one of them was related to graphics.

Pressing F6 followed by escape brings up a line to edit boot options.
One of the options was VGA=788

I changed this to VGA=768 and its working ok now
