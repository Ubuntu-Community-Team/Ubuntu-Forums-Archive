### Post by MidnightGrey on 2013-06-19
Hi UbuntuRaptor,

Unfortunately LibreOffice handles templates a bit differently than MSoffice.

LibreOffice has a built in Template manager. You can locate it under File > Template.
You can save a file as a template by using File > Template > Save.
You can also open to edit the template by using File > Template > Edit.
If you want to create a new file using an existing template you can use File > New > Templates and Documents (in Libre Writer for example).

Hope that helps.
