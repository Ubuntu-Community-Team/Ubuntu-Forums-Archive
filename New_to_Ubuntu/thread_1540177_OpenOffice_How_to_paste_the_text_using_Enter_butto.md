I am really used to Microsoft Office Excel 2003. So to copy/paste the text in Excel I do:
1. mark cells with up-down-left-right keys 
2. <ctrl>+c to copy cells
3. move to cells with up-down-left-right keys I would like to paste to
4. <Enter> - this action pastes the cells.

For step 4) I can also use <ctrl>+v in Excel just like in OpenOffice, but using <ctrl>+v I need to look at the keyboard to press this key combination. To just press <Enter> I don't need to look at the keyboard, I know where Enter is at the end of the keyboard.

Is there any way I can set shortcut in OpenOffice to paste the cells using <Enter> key?
