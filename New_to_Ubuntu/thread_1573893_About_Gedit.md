### Post by Chesamo on 2010-09-13
Sometimes it takes a moment for GEdit to read the file and determine its type (for syntax highlighting). GEdit comes with syntax highlighting built-in, so it probably auto-detected some kind of markup or programming language.

I'm not 100% on this answer though.

### Post by Nanix on 2010-09-13
If you look at the gedit status bar you will see that it is set to highlight the syntax of VHDL. You might have accidentally selected that at some point or as Chesamo said, gedit might have auto-detected that language. 

Whatever happened its nothing to be concerned about. Just click the menu on the status bar and select "Plain Text".
