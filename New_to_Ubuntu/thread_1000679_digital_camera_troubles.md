### Post by Michael.Godawski on 2008-12-03
> **Import photos from a digital camera**


If Ubuntu does not ask you if it can import the photos:
                                    
[LIST=1]
[*]                       Open Applications &#8594; Graphics &#8594; F-Spot Photo Manager and choose File &#8594; Import....
[*]                       Select your camera as the *Import Source* and then press **Import** to begin importing your photos.
[/LIST]
                 
                 If your camera does not appear as an import source, press Places &#8594; Computer and check if your camera is visible there as a removable storage device. If it is, you can then copy the photos manually using the **File Browser**.
Try this and please report back.
