### Post by gsmanners on 2009-12-28
I don't know about "web apps" (overrated junk IMO), but there are lots of real app editors for Linux.

Lots of people swear by Eclipse, although Code::Blocks and Geany are good editors. You may not find all the features you want, but it's a good start IMHO.

For designing a user interface, Glade is a good way to go for GTK. I'm not sure what the developer tools are for the latest Qt.
