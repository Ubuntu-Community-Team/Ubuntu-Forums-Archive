### Post by omegamike3 on 2008-12-05
You'll want to make sure that the cd was burner properly. Meaning, does the cd just contain the .iso image file or was the .iso used as the source in the burning to create a copy of the image? You should be able to tell this in windoze by just looking at the contents of the disk in explorer. If the disk was burned properly, make sure you're bios is setup to boot off the cd/dvd drive or just use the option it gives you at boot (usually a function key like F12) to choose a medium to boot off of other than the hard drive. If it's trying to boot off the cd but can't, try burning another copy using the slowest speed setting. It also doesn't hurt to run an integrity test after the burn, which almost all burning software nowadays offers to do after a completed burn. :)

### Post by tomtom1982 on 2008-12-12
How did you burn the CD?

If you only had taken the ISO-file and burned in on CD, it won't be bootable.

So take a burning-program which is able to open and burn ISO-Files like Nero (exempted LE or SE versions) or ImgBurn.

There you choose open an ISO-file and burn it on CD, not faster then 24x and in Disk-At-Once Mode (DAO).
