### Post by Delever on 2008-05-16
If something does not work, and Ubuntu does not suggest to download and install drivers automatically, and you can not find them in System->Administration->Restricted drivers, or you have some older ATI card, come here and search this forum, someone quite possibly already asked about such problem.

Basically, kernel already contains most support for hardware, you can see that in 2.6.24 kernel changelog: [http://kernelnewbies.org/Linux_2_6_24](http://kernelnewbies.org/Linux_2_6_24)
