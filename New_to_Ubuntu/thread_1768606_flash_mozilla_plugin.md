### Post by lovinglinux on 2011-05-27
Unfortunately, there isn't.

There is Gnash as already mentioned and [lightspark]("http://sourceforge.net/apps/trac/lightspark"), but they never worked well for my needs. I haven't tested them recently tho.

If you are experiencing problems with flash, then install my  [Flash-Aid](http://www.webgapps.org/add-ons/flash-aid) and run the extension wizard. It will detect installed flash plugins, remove them and install the best option according to your system architecture and version. Additionally, it will apply some tweaks that should improve performance and fix common issues.

If you use YouTube frequently, get my [FlashVideoReplacer]("https://addons.mozilla.org/en-US/firefox/addon/flashvideoreplacer/") extension. It replaces the flash player with a better media plugin, if you have one installed. I recommend installing gecko-mediaplayer to be used with FVR. More info on my site [http://www.webgapps.org/add-ons/flashvideoreplacer](http://www.webgapps.org/add-ons/flashvideoreplacer)
