### Post by winh8r on 2012-02-14
As far as installation goes on the 40gb drive, when you boot up the machine with the Live CD and select install, this will start the disk partitioning software which will tell you that the HDD has windows XP on it. It will ask you how you want to install Ubuntu, and you can select to "use entire disk" which will overwrite the Windows installation and reformat you disk to the required filesystem for a Ubuntu installation.

With regard to the use of CC Cleaner and other windows based tools, these are designed to be used on Windows filesysytems and are not compatible with the Linux/Ubuntu file system so are not used.


There are various anti virus tools available for Linux but these serve more as a way of ensuring that you are not spreading infected files to Windows based machines.

There are virus codes and malware that can affect Ubuntu, but in general it is not a major problem for most users. The best security tool on any computer is the operator. 

Make sure you do not download from untrusted sources and never run any commands you are given from an untrusted source unless you know what they do.

for more about security in general.


Any questions you have , just ask.

### Post by redsariku on 2012-02-14
It looks to me as if you do not have xp already installed on the computer. If you don't then let me suggest that you get used to using Ubuntu first. You may find that you like it better than xp. I want to suggest that you use Synaptic Package Manager (it is in the Administration part of System on the top menu bar or in the System on the side bar depending if you are using Ubuntu 11.04 or greater, the Unity desktop. I use the Gnome desktop so for me it is on the top.) When it pulls up you may be asked to put in your password. Then You can do a quick search for Metapackages. This will give you a lot of codecs that will help with your music and videos. You need to right click your mouse and click "mark for installation". It will tell you that many other packages need to be included in order to use this package. Click on "mark" The other package you will need is ubuntu-restricted-extras. It will also ask you to mark a number of other packages. After you have these packages your computer should work very well with facebook and youtube and all other web based things. You will not be able to play a lot of games with this machine but most of the games today are going over to Windows 7 anyway so if it is an old machine then just use it as best you can. Check out the WineHq website to see what Windows games can be played on Ubuntu if you install Wine and follow directions there for installing games. Some games do work with Wine. If you want to install Xp you should do that first making two drives a C and D drive. Then when you reinstall Ubuntu you can use the D drive to install it there. On a 40 gig hard drive I would recommend just one system though Ubuntu can be installed on a small space of 15 gigs recommended but I have installed 10.04 on 6 gigs. 
Welcome to Ubuntu and lots of luck. Take some time to get to know Ubuntu. It will be worth the time.
