### Post by diablo75 on 2010-05-30
Only if said products are available for Linux.

In the case of Google Earth, you can download it from here:

[http://earth.google.com/download-earth.html](http://earth.google.com/download-earth.html)

You will download a file that ends in .bin

If you were to save it to your desktop you would need to then open a terminal window and type in the following:

```
cd Desktop

sh ./GoogleEarthLinux.bin
```
Commands in the terminal are Case Sensitive.


See the link in my signature (10 things to do...) for a more detailed instruction.

Google Chrome is available as a deb file.  You just download the deb file, as if it were a setup.exe file in Windows, double-click on it to start the installer, click "Install" and you're done.  Simple as that.

I don't know anything about Google Sketch but you might just check their website to see if a Linux/Ubuntu/Debian file is available.
