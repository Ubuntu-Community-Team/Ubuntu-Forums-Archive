### Post by Nixie Pixel on 2009-02-20
> **WorrFighter said:**
> 1. Is Office Ultimate '07 compatible with Linux?
Natively in Linux, no.  However, you have two choices - you can use OpenOffice.org's office suite, which as noted above is installed with Ubuntu or Kubuntu.  You can also use a program called WINE to run many Windows applications within Linux; looking at the [WINE AppDB]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=31") it appears that Office 2007 is on the "Platinum" list, meaning it works very well within WINE.  WINE, however, may or may not require some configuration to work properly for you.  

The easy answer is to try OpenOffice and see if it works for you.  Luckily, it is free and there is also a Windows installer so you can try before you even install Linux - just head to openoffice.org and download it, you can install it alongside Microsoft Office if you like.

> **WorrFighter said:**
> 2. I will not be using the computer for any serious computing. Only for school and web browsing. So, which Linux distribution would you recommend?
In terms of which distributions can handle what you want to do, there would be very little difference.  The big difference for you will be how easy the distribution is to learn coming from Microsoft Windows.  One of the best for that is Ubuntu, or Kubuntu (for a desktop that looks and feels more like Windows Explorer), and since you are already here and see the support available to you, why not give it a try?

By the way, [check out this post that I wrote which should help explain the differences between Ubuntu and Kubuntu for a Windows user.]("http://nixiepixel.com/blog/blog7.php/nixie-s-ubuntu-linux-a-travel-guide-for-")

> **WorrFighter said:**
> 3. Are Epson printer drivers available for Linux?
I can't answer this for sure, but using a Live CD can tell you rather quickly if you can do what you want to do - see my answer below.

> **WorrFighter said:**
> 4. I have considered a duel boot set up. But, how can I tell if my system is a duel boot system? And, is a duel boot the way to go?

Ubuntu and Kubuntu can be run on a Live CD, meaning you can try the operating system without making any changes to your computer at all.  All you need is the ability to download an .iso and burn the image to a CD (or DVD), and you can reboot your computer and run the operating system directly from the CD.  This will tell you what will and won't run on your PC right out-of-the-box, without configuration, and without any worry about damaging your Windows install or data.

Dual-booting is also a great way to become familiar with Linux without too much hassle.  I suggest using Windows' disk manager to shrink the disk partition before creating a second partition and installing Linux; it is the safest way to do it, though usually it is Vista that can be broken by using an external program to shrink its partition.

Good luck!

### Post by redseventyseven on 2009-02-20
1. As already mentioned, try using OpenOffice and see if it suits your needs. It's similar to the Microsoft Office suite but not identical - some of the differences might take a bit of getting used to. If, after a few weeks, you really really don't like it, then I'd suggest giving the run-Microsoft-Office-on-WINE idea a try.

2. I use Linux Mint and it's very good - mostly because it's largely based on Ubuntu but with a few extra touches on it that, well, don't add much, but make it a little easier to get going. Don't be afraid to try several though - once you've got the hang of it, re-installing a fresh distribution doesn't take too long (unlike a Windows re-install!)

3. Epson has a very good track record for making their printer drivers available on Linux. However, have a look at [http://www.openprinting.org/printer_list.cgi](http://www.openprinting.org/printer_list.cgi) for more information - unless your printer is a brand new model this should say whether it's likely to work or not.

4. Your system is a dual boot system - if you set it up that way! It's a little tricky but other users have already suggested good websites giving guidance. I've never used Wubi so I can't comment on it.

Before you start, though, make sure all your important work is backed up. Buy an external hard-drive (these are fairly inexpensive these days - it's a worthwhile investment) and spend a bit of time saving all your important documents onto it. Then unplug it from the computer and keep it in a safe place until you've got Linux working as you want it. That way, you can happily play around with Linux to your heart's content, breaking it and re-installing, safe in the knowledge that you're not going to lose shedloads of data.

### Post by jerome1232 on 2009-02-20
X.org is an Open Source implementation of the X Window System, it's the building blocks for the fancy gui you interact with.

I would try booting into Ubuntu again, and if you get the same thing, just hit Ctrl+Alt+F1 to get a virtual terminal, you should have a text based login prompt, just go ahead and login then issue this command to get xorg back to where it was.

```
sudo dpkg-reconfigure -phigh xserver-xorg
sudo apt-get remove nvidia-glx-*
sudo reboot
```

That should at least get your gui back and it'll remove the drivers your installed.

Next I would update your computer before reattempting installing drivers.
Just use update manger located at System->Administration->Update Manager, update your install and reboot if necessary.

Then you can try installing the drivers like you did before, or try envyng, I see alot of success with envy.

To use envy

```
sudo apt-get install envyng-gtk
sudo envyng -t
```

Select the driver you want and reboot when done, everything should be working.
