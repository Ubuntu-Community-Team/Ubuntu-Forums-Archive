Hi there,

I'm a pretty newbie linux user, installed 12.10 from a live cd and everything seemed to go smoothly. My only problem is that every so often the launcher stops responding to mouse input - the icons don't highlight at all on a mouseover, which I seem to remember them doing previously, and nothing happens when I click them. The top bar doesn't change either; whereas normally it would display the close/minimise buttons, 'file' menu etc. now there's no alteration whatsoever and it continues to display only the title of the window. 

I should note that this is an intermittent occurrence and seems sometimes to be fixed by such actions as repeatedly attempting to 'drag' the window. If there's any direction people could provide towards a more permanent and stable solution, I'd be very grateful. Thanks! :)

Edit: Windows now exhibit the same behaviour, but it's always either one or the other - if the launcher responds, the windows don't (and vice versa).

Edit 2: Experiencing same problem on a fresh install from live cd of 13.04. Please help, it seems to be getting worse!

Found a solution: I'm using a R.A.T.3 mouse from Cyborg, which has known issues with Xorg. I used a workaround from here: 
[http://techspheria.com/2011/02/how-to-fix-saitek-cyborg-rat-series-mice-in-ubuntu/ ]("http://techspheria.com/2011/02/how-to-fix-saitek-cyborg-rat-series-mice-in-ubuntu/")
Which completely fixed the issue.
