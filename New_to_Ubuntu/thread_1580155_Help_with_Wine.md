### Post by Paddy Landau on 2010-09-23
A tip for the future.

Install PlayOnLinux.

It's a front-end to Wine, and helps you to install, run and uninstall programs on Wine.

I wouldn't depend on iTunes working, though. Knowing Apple, version 8 will become obsolete and there's no guarantee that Wine will manage to get later versions working.
