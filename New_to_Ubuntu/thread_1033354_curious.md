### Post by decoherence on 2009-01-07
The first thing to do is check out the Add/Remove Programs menu item under the Applications menu. Find some programs that look interesting to you, install them and learn them!

Other than that... what do you want to do? Ubuntu is a general-purpose system with pretty much the same capabilities as Windows. If you installed it and it boots, chances are everything is fine. Would you like to learn about Linux? Setting up a server? 3D animation? The sky is the limit.
