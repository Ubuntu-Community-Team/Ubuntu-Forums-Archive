### Post by jeremyswalker on 2009-10-10
Picture files should not be a problem, at all. As far as video and audio codecs, installing the package "ubuntu-restricted-extras" should get you most everything you need. Might I also recommend VLC media player. You can install both of these with the following:
```
sudo apt-get install ubuntu-restricted-extras vlc
```
