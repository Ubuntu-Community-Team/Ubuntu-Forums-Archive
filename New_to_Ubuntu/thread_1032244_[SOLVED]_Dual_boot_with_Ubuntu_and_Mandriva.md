### Post by Sef on 2009-01-06
> Do i need to partition my drive from a live CD?

Not necessarily.  You can download [GParted]("http://gparted.sourceforge.net") and make the partitions ahead of time.


> If so do i re-partition 50/50 Ubuntu/Mandriva?

No.  Just make sure you have enough room for Ubuntu (8-10 GB.)


> When i install Mandriva,at the partition stage,do i use existing partions?

If you already made the partition, then just use it.  If you didn't make it, then you would have to make it then.
