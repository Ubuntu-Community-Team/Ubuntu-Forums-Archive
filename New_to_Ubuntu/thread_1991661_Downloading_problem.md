### Post by drawkcab on 2012-05-30
Get unetbootin from here:

[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

Download the 12.04 .iso file of your choice to your computer and then let unetbootin load it onto your usb drive so that you can then boot into a live session.

From the live session, run the installer.  The installer should provide you with the option to wipe windows entirely or set up a dual boot system.
