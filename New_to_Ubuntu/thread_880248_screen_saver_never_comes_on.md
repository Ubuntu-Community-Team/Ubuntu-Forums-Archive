### Post by alecz20 on 2009-08-13
This was very very strange.

I had the problem with the screen-saver not coming up. The screen would dim, but then just turn back to a normal desktop. After pressing Ctrl+Alt+L the screen was locked with the selected screen-saver.

I returned back to the normal desktop and waited for the dimming to happen. And surprised the screen-saver worked this time.

This is really weird, because it seems the Ctrl+Alt+L command "fixes" the screen-saver not coming up, since it comes up afterwards if you wait for it.
