### Post by d3ngar on 2011-01-12
Hi Pedro,

Noble effort on your website there...

I think where you would want to start is not necessarily with ubuntu, but maybe a section that compares operating systems, what their strength and weaknesses are. Then maybe a section for ubuntu as a case study. - Just a suggestion.

I think that it is a very common misconception that up-to-date Linux versions are more complicated to use than Windows or MAC OS these days. 

I was never a fan of Windows and have been using Linux since many years now as my sole OS, even using it at work in a Virtual environment. Back in the days Linux set-up was tedious and sound and video problems were part of the experience. Today everything seems to work out of the box (nearly) and in most cases much, much smoother than in Windows where you'd have to install yourself through hundreds of drivers :)...

As you might know, MAX OS and Linux are both based on Unix, so they nowadays seem very similar (albeit I hardly ever used Macs :D). Linux doesn't come with the huge price-tag of buying a MAC though and is probably more suitable for the hands-on computer user. I wouldn't want a MAC, not because I don't think they are great machines and would not do anything I need them too, but I'm more familiar with GNOME now.

As far as Windows is concerned, I think it's been down-hill since 3.11 :). MS Office is a great product and I run it on my Linux OS without any problems through wine.

Well, I don't wanna write an essay here, it's already so much!
