### Post by Sarah L on 2008-05-27
Using wine adds a "middleman" between your software and your operating system that doesn't exist when you're using plain Windows. There's a bunch of extra procedures that each network packet has to go through when you're running a Windows game in Linux than when you're running a Windows game in Windows.

If you're really concerned about performance, then you should stick with Windows when playing counterstrike. Alternatively, if you don't want to sacrifice either performance OR linux, then try out some open-source shooters, such as [Tremulous]("http://tremulous.net").
