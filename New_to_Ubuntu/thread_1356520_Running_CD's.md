### Post by 3rdalbum on 2009-12-16
Windows programs do not really run on Ubuntu. You can run SOME of them using some software called Wine, but your best bet is to stick to native Linux programs wherever possible.

Programs are not interchangable between operating systems.

Ubuntu contains a lot of drivers already for devices. Your webcam might already work, but you just haven't loaded any programs that take advantage of it.
