### Post by coldReactive on 2009-10-29
1. I would like to switch between desktops using the CRTL+ALT+Arrow Keys like GNOME, how do I do this?
*To do this: KDE Menu > Computer > System Settings > Keyboard & Mouse > Global Keyboard Shortcuts and choose KWin from the KDE Component dropdown. Change Switch One Desktop Down, Right, Left, Up as needed.*

2. Flash 64-bit from adobe labs has no sound in Firefox, how do I fix this?
*Run alsamixer from Konsole, and set PCM volume all the way up.*

3. How do I get rid of the envelope icon (right to the left of the time) in the Kubuntu panel? It's annoying.
*remove the plasma-widget-indicatordisplay package.*

4. I disabled popup notifications for incoming messages in Kopete, but they still appear!
*seems I had to disable various other notifications such as MSN Mail, etc.*

5. How can I change it so that the icon on the very right of the kubuntu panel shows the desktop rather than the panel settings/etc.?

6. How can i make it so when I try to drag a window off screen, it switches to that desktop while dragging? Like GNOME.
*Go to KDE Menu > Computer > System Settings > Desktop > Screen Edges, and select from the Switch desktop on edge dropdown, Only when moving windows.*
