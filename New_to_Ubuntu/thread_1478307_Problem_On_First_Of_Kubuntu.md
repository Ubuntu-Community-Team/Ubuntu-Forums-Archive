### Post by ankspo71 on 2010-05-09
4. Why are windows title bars not transparent like widgets?
That option can be set at:
System settings > Desktop > Desktop Effects > All Effects > Translucency (then click on the little wrench button to adjust those settings)

5. Where is software center?
They left this part out, but I believe it can be installed easily. 
You can go to System Settings > Add and Remove Software, then search for "Software Center" (but search using the "Find by Description" from the drop down list). "Synaptic Package Manager" is available too, which is what I use.

10. How to set time manually?
Right click on the clock, Select "Adjust Date and Time", then set the time under the animated picture of the clock. A reboot may be necessary for the clock to display the correct time, because it was in my case. Your password will be required too.

Kubuntu has two file managers, Dolphin and Konquerer. Konquerer doubles as a web browser and a file browser.

Keyboard shortcuts can be found at: 
System Settings > Keyboard and Mouse > Standard Keyboard Shortcuts AND Global Keyboard Shortcuts

Hope this helps.
