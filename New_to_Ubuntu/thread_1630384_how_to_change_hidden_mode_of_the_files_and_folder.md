### Post by KeLa on 2010-11-25
Windows and linux handles hidden files differently.
In windows files and folders are hide using hidden attribute and
in linux files and folders can be hide with adding dot as a first character
in the name.

So what are hidden in windows will be shown in linux and what are
hidden in linux are visible in windows.
