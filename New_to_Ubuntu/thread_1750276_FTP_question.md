### Post by SavageWolf on 2011-05-05
Nautulus has this by default, use the Places/File -> Connect to Server, and enter info. You can use this like any other folder.

There is an annoying bug (with my machine anyway) that bookmarked FTP links open in Firefox from the Places menu, but the bookmarks sidebar in natulus works as expected.
