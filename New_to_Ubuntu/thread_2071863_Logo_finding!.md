### Post by Cheesemill on 2012-10-16
Usually icons are in the /usr/share/icons directory. You'll find several sub-folders depending on what icon theme sets you have installed on your system.

### Post by Frogs Hair on 2012-10-16
Nautilus > file system > usr > share > unity > 5 or 6 depending on version > launcher bfb.png. You will need root permissions to change/remove items from this folder witch will require the terminal. This is for the dash home icon. Other icons can be changed via the main menu if alacarte is installed.  ```
gksudo nautilus
```
