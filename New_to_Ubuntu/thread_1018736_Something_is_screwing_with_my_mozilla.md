### Post by s.fox on 2008-12-22
**Adding or Removing Software**

[LIST]
[*]**To remove or 'completely remove' a package**
[LIST]
[*]Right-click on the package and choose **Mark for Removal** from the context menu
[*]Once you're satisfied with your choices, click on **Apply** in the toolbar or press **Ctrl + P**.
[*]A dialog box appears with a summary of changes that will be made. Confirm the changes by clicking on **Apply**.
[/LIST]
[IMG]https://help.ubuntu.com/community/IconsPage?action=AttachFile&do=get&target=eyes.png[/IMG] The **Mark for Complete Removal** option instructs Synaptic to remove any configuration files associated with the package as well.
[/LIST]



[LIST]
[*]**To install a package**
[LIST]
[*]Click **Reload** or press **Ctrl + R** to make Synaptic aware of the latest updates.
[*]Right-click on the package and choose **Mark for Installation** from the context menu, or press **Ctrl + I**. If the package requires the installation of another package, a dialog box appears: 
[IMG]https://help.ubuntu.com/community/SynapticHowto?action=AttachFile&do=get&target=synaptic-confirm-mk.png[/IMG] 
[IMG]https://help.ubuntu.com/community/IconsPage?action=AttachFile&do=get&target=eyes.png[/IMG] Synaptic will warn you if your choices conflict with packages that are already installed on your system. If this is the case, the dialog box will show you any packages that need to be removed. If you are **not sure** you don't need the package(s) that will be removed, make sure you look up its function and use before you apply the changes you've made.
[*]Click **Mark** to allow the installation of the additional package(s).
[*]Once you're satisfied with your choices, click on **Apply** in the toolbar or press **Ctrl + P**.
[*]A dialog box appears with a summary of changes that will be made. [IMG]https://help.ubuntu.com/community/SynapticHowto?action=AttachFile&do=get&target=synaptic-confirm-install.png[/IMG]
[*]Confirm the changes by clicking on **Apply**.
[/LIST]
 
[/LIST]
