### Post by fenrice on 2016-10-24
Sounds like you are using a virtual machine. The iso should boot no problem, if it doesn't you should check the md5sum. I would suggest running through the wizard again and keeping the default settings except maybe for the size, give yourself at least 20gigs if you wanna get anything substantial done.
