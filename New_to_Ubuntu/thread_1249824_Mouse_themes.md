### Post by fooman on 2009-08-25
try this...open your home folder (places > home folder).  in the toolbar, click view and put a check next to "show hidden files"

scroll down and find/open the .icons folder

look in there and see if there is a folder with the same name as the mouse theme you installed/uninstalled.  if it is there,  right click on it and "move to trash".

then go back try to reinstall the theme.
