### Post by lovinglinux on 2009-03-26
Welcome to Ubuntu.

I'm not a musician, but I guess Ubuntu Studio will make you very happy. 

I don't know what you mean by 3D. If you are talking about games, then I recommend installing Ubuntu in [dual-boot]("http://en.wikipedia.org/wiki/Dual-boot"), thus keeping your Windows installation just for games. Unfortunately, most commercial game companies do not provide support for Linux. You can run some games with Wine, which is a Windows emulator, but is not perfect. If you are an avid gamer, dual-boot is the best solution.

If you men 3D design, then you will be happy using [Blender]("http://www.blender.org/"), which is a powerful application for 3D modeling and animation. It is in the repositories, which means it can be easily installed using the built-in Ubuntu software manager.
