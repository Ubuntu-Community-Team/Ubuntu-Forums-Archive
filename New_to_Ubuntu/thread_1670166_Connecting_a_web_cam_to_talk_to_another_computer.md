### Post by hansheijmans on 2011-01-18
> **bob brazie said:**
> Thanks, the description only talks about "talking" nothing about video. Does skype handle video too.
Yes it does! With Skype you're able to talk, use video and even chat. You can test video by opening the Options menu and click on Video Devices (you will find a test button). To make sure audio is working you can make a test call (Options - Audio Devices).
