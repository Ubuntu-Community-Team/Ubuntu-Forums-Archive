### Post by Kegusa on 2010-01-25
Tried the compiz plugin Enhanced Desktop Zoom? It comes in default with compiz and to use it just press "Super-mousewheel" (Super = win key)

Also some other visualisation options you can configure.

If you dont have the compizconfig settings manager found in "System - Preferences - CompizConfig Settings manager" you can install the package "compizconfig-settings-manager" in synaptic ("system - administration - Synaptic package manager") 

OR just use the command line: sudo apt-get install compizconfig-settings-manager
