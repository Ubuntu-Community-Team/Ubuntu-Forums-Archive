I have an Aspire One and it works great with Intrepid.  To get the wireless working you need to install the backport kernel modules for Intrepid.

```

sudo apt-get install linux-backports-modules-intrepid-generic 
```


These modules offer better support for the newer Atheros cards.  You will need to add ath5k to your /etc/modules file to make sure the ath5k module is loaded at bootup.  Unfortunately the little LEDs don't blink with the ath5k driver.  They will blink with the ath_pci driver from madwifi.

Here is now to get the SD card readers to automount.

[http://ubuntuforums.org/showthread.php?t=1019922](http://ubuntuforums.org/showthread.php?t=1019922)

I also recommend installing netbook remix as it works great on these small laptops with limited screen area.
