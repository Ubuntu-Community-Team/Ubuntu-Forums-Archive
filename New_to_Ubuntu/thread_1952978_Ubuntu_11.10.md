Welcome to the forums.

1) You can easily install Chromium, which is the open source version of Chrome, using the Software Center. If you want Chrome instead, you will have to download a deb file from Google, which can be installed using the Software Center as well.

3) The Linux version of flash has many issues, specially in regard to performance. The latest version caused the SMURF effect on those computers running nVidia cards. Since Flash is closed source, there isn't much to be done. We depend on Adobe, which recently killed Flash for Linux and won't be developing it anymore. Instead, they gave the task to Google, which will implement Flash with the new Chrome Pepper API, which is not accepted by other browser vendors and won't be implemented in Firefox or Opera. 

The current workaround for the Flash problems is to disable hrdware acceleration in flash settings.
