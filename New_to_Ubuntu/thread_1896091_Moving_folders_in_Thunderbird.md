### Post by rewyllys on 2011-12-16
> **Auroraei said:**
> I created the folder by right clicking and clicking "New folder", and it won't let me drag it around. How do I make a subfolder in the inbox?
I suggest the following:
Right-click on Inbox, and select New Folder.  Name the new one "Important".
Go to your original Important folder.  Use Ctrl-A to select all the messages in it. Then right-click on the highlighted messages and use Thunderbird's "Move to" command to move them to the newly created Important folder.
