### Post by peculiar penguin on 2010-10-24
If you allow Firefox to store passwords, it'll do so for the Sync account password and encryption key, and it will sync automatically without you having to click on the icon or anything.

If you're paranoid about Sync not working, create a new empty Firefox profile (firefox -P), install the Sync add-on, and tell it to replace your local data with the stuff on the server. If your bookmarks start showing up, it's obviously working.

You should still export a local backup copy of your bookmarks every now and then just to be safe.
