### Post by Elmer Fudd on 2010-08-08
I changed my 87 year young mom from XP to Ubuntu on an old Acer laptop for the same reason. Viruses. 

Follow the previous recommendations. Test all her normal sites to make sure that you have all the codecs/Java/Flash installed properly and give her dual boot with BURG instead of GRUB2.

My mom hasn't used her Win partition since the first two weeks while I was tweaking things.

Set up remote desktop so you can support her- at least for non "lost the internet" issues.
