### Post by RKCole on 2008-07-16
Hello, mrfallreevfl.

I had a similar issue a long time ago on an older Ubuntu distro--it was because I was messing around and I accidentally deleted something from my bottom panel.

**If you are using GNOME and if your issue is similar to the issue I had in the past**, the following steps **MAY** solve your problem.  What I did a few versions ago was I accidentally removed the Window LIst from my bottom panel; I was not sure how to restore it, but eventually with the help of those here at the forums, I was able to solve this slight dilemma.  Here si what you could try on the GNOME Desktop (I ma unfamiliar with KDE/Xfce):

[LIST=1]
[*]**Right-click** on an empty part of your bottom panel and click on **Add to Panel**
[*]When the **Add to Panel dialog box** appears, search for and left-click on an entry labeled **Window List**
[*]Click on the **Add** button
[*]Close the Add to Panel dialog box
[/LIST]

If all went well, and if this was the issue, then you should now be able to select minimized windows on the bottom panel.

Take care.
