### Post by grahammechanical on 2013-04-11
If you search the Ubuntu Software Centre for pulseaudio you will see that we already have PulseAudio Volume Control installed. That is the standard volume control that we have as an app indicator.

You can also install PulseAudio Sound Server. That will give you some more options. And then there is PulseAudio Manager to manage the PulseAudio sound server. You may want that one.

Regards.
