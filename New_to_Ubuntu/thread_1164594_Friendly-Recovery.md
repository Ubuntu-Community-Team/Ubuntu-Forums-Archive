### Post by winh8r on 2012-02-12
> **johnnylong said:**
> Thanks, that's helpful information, but since we're on the subject is there any software in the repository that will preform data recovery on an external drive, and if so, what's the name?


You can use Testdisk/Photorec on any drive whether it is an internal hdd or external usb although it will work best on internal drives.

It has no GUI and is run from the terminal and navigation is done using the arrow keys on your keyboard.

There is a guide to step by step recovery using Testdisk/Photorec here:

[http://www.cgsecurity.org/wiki/PhotoRec_Step_By_Step]("http://www.cgsecurity.org/wiki/PhotoRec_Step_By_Step")


hope this is useful to you.
