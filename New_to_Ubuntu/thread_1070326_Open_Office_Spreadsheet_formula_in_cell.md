### Post by Tamlynmac on 2009-02-15
In OO I believe the "esc" key is basically the cancel function and can be added to any toolbar.

Open Spreadsheet Sheet and select  view/toolbars/customize/toolbars tab

Customize Popup window should appear
Under the heading of OpenOffice.org Calc Toolbars click on the drop down menu to the right of the word Toolbar. Select "Standard" from the list.
Left click on the "ADD" button on right side of the customize window.

Another popup window will open called Add Commands
In the "Category" list select "Edit"
In the "Commands" list select "Cancel"
Left click on the "Add" button
Left click on the "Close" button of the Command Popup Window

This should have added the cancel command to your "Standard"  "Toolbar Content Commands". Find it in the listing and if not checked - check it. 

Select the OK button at the bottom of the customize popup window.

The new cancel command button should now show in your "standard toolbar". Left clicking on the new "Cancel" command should have the same effect as the esc key.

Should you wish to change the text of your new command to an icon:
view/toolbars/customize/toolbars tab
toolbar - Select Standard
Command - Select Cancel
Left click on "modify" and select change icon
Select an icon and selcet OK bottom of "change icon window" and select OK in the customise window.

I think that should do it.
Hope this helps.
