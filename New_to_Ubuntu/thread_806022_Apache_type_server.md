### Post by tamoneya on 2008-05-24
dont use add/remove.  Either use command line like i showed above or use synaptic(system -> administration -> synaptic package manager).  This is because things like apache and other servers and libraries and such which arent supposed to show up in the applications menu arent in add/remove.  That is why you couldnt find it there.
