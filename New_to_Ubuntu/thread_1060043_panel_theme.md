### Post by mcduck on 2009-02-04
It doesn't really need any theme, you can simply use any picture as panel background. This can be set in panel's configuration or simply by drag&dropping any picture to the panel.

You'll probably finds some pre-made background pics in [http://gnome-look.org]("http://gnome-look.org").

(for best looking results create a picture that's exactly the same size as your panel is. Also, transparency in PNG and SVG images works fine and allows pretty neat looking panels.)
