### Post by dracayr on 2008-10-25
If you use the multiple desktops of compiz (the effect manager), then you probably can't set different backgrounds. At least with Gnome using compiz, this is the case (and I know, that with gnome+compiz, compiz manages the desktops, so I guess that with KDE+compiz, compiz manages the desktops also.) It is still of advantage to have multiple desktops though, because you can have your windows on different desktops, and won't have a taskbar with 20 windows on it...
