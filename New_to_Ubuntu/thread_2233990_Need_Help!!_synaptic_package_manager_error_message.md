### Post by grahammechanical on 2014-07-11
During the installation of ttf-mscorefonts-installer we are asked to agree to Microsoft's End User License Agreement. did you do that? If we do not agree, then the installation will fail.

In Synaptic Package Manager select the package ttf-mscorefonts-installer and right click it and select to re-install. See if that fixes things.

Regards.
