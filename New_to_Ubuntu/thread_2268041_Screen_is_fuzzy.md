### Post by grahammechanical on 2015-03-05
Ubuntu has not had a drop-down menu structure with System>Administration for years. Have you installed Ubuntu or one of the flavours of Ubuntu? Or an alternative desktop? Or Gnome flashback? It helps to know these things when giving instructions.

Assuming that you are using Ubuntu with the default user interface of Unity - go to System Settings. There should be an icon for it in the Launcher or select it from the power/cog menu and then open Software and Updates>Additional Drivers tab. Allow the utility a little time to do its work and you will find out what video driver is being used.

If it is an open source video driver then in System Settings under Screen Displays you might find an option that might help. If it is a proprietary video driver then there will be a settings manager for that driver and you can find that by searching for it in the Dash.

What monitor do you have? What cable is being used to connect to the monitor? VGA or DVI? That can make a difference, I have noticed. DVI is preferable to VGA.

Regards.
