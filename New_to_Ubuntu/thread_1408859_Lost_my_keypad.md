### Post by theaaronc on 2010-02-24
After a little searching it looks like having mouse keys enabled may cause this problem. Go to: 

System -> Preferences -> Keyboard -> Mouse Keys Tab

Make sure that "Pointer can be controlled using keyboard" is unchecked.
