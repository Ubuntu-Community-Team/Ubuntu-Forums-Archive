### Post by unmole on 2010-02-09
Answer 1: It is not necessary to install updates but it is advisable to keep your install up to date.

Answer 2: Why are you trying to install AVG anti-virus ? Unless this machine is going to be a server of some kind, taking connections from Windows machines, there is NO NEED whatsoever to install an anti-virus software.

### Post by MooPi on 2010-02-09
There is little need of anti-virus software to protect a Linux computer. Anti-virus on Linux is used to detect viruses so you don't pass a virus along to your Windows friends. It is also used on servers for the same reason. You can probably use a Linux program called>  clamav. Clamav should be sufficient to detect viruses and avoid the problem of your computer hanging.
