### Post by kircher on 2010-04-30
If you're using gnome with ubuntu, click system in the top left, go to administration, click Software Sources, tick multiverse. Then close. In the top left again click applications, then click Ubuntu Software Centre down the bottom, and then type ubuntu-restricted-extras into the search bar in the top-right. Install that package and the videos should play properly.

Edit: ignore, I have the same issue, and I installed flash by the above method

Try this [http://ubuntuforums.org/showthread.php?t=1414595](http://ubuntuforums.org/showthread.php?t=1414595)
