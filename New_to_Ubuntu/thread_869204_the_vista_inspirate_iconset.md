### Post by stalkier on 2008-07-24
> **stonecoldjha said:**
> how do i install the vista inspirate iconset?

Right-click the desktop
Goto Change Background image
Goto Theme
Install
CLick on the location of your iconset
install the iconset

It will say installed correctly and then it asks if you want to apply it now. The same goes with installing X11 cursors etc. You might have to restart for it to show correctly.
