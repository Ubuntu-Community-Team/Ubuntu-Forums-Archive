### Post by blazemore on 2013-07-30
You don't need to install the xubuntu touchpad setting dialog in Ubuntu.

To enable reverse scrolling in Ubuntu 13.04 Raring Ringtail, simply check &#8220;content sticks to fingers&#8221; in the mouse & touchpad configuration screen of Control Center
[IMG]http://jabstaboops.andthesethygifts.com/wp-content/uploads/2013/02/kineticscrolling1.png[/IMG]
