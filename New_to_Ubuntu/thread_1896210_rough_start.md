### Post by 2F4U on 2011-12-16
Why buy a new computer if it is just one component that fails (except maybe that the machine is a laptop). If I would experience such lockups I would first run memtest from the liveCD and if that reveals a problem, I would simply change the RAM. If the harddisk is the problem, it would most likely show up in the system log file and then only the harddisk needs to be changed.
