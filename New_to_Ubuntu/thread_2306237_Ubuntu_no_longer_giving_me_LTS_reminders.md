### Post by grahammechanical on 2015-12-13
Ubuntu 15.10 is not a Long Term Support (LTS) version. It only has 9 months support counting from the end of October. Ubuntu LTS versions are set to notify us of the next LTS release. Interim (9 months support) versions are set to notify us of the next release whatever it is. We can change this setting in Software & Updates.

To install Chrome we install a goggle repository. I have Chrome installed. It is at version 47 and from time to time it gets updates. Where have you heard that Google is no longer going to support Chrome? A web search comes up with information that

Google chrome no longer supports Java; Microsoft Silverlight; Chrome for Windows XP; Certain older versions of operating systems. I am also informed that Chrome now blocks Adobe Flash.

### Post by Bucky Ball on 2015-12-13
If you are running 14.04 it won't let you know of upgrading to an LTS. You are already running the latest. It will start informing you there is an LTS upgrade available sometime in the latter half of next year once 16.04 LTS is released.

### Post by ian-weisser on 2015-12-19
Chrome *32-bit* will be dropped by Google.
Chrome *64-bit* will continue to be supported.

Since most hardware today is fully 64-bit compatible, I suggest downloading a 64-bit version of Ubuntu 14.04.3 from [http://www.ubuntu.com/download](http://www.ubuntu.com/download) .
Test it on your hardware.
If it works, back up your data and install the 64-bit version.
Then install the fully-supported 64-bit version of Chrome.
