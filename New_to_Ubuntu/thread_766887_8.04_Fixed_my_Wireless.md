### Post by imakeart on 2008-04-25
In case anyone else finds this useful this is what I did to fix my wireless after upgrading to 8.04.

First off I'm running Ubuntu 8.04 on an Acer Aspire 3680 with the Atheros Wireless Card. I wasn't able to see why the wireless wasn't working - everything seemed to be present, but this is what fixed it.

1. You need to either install these from Synaptic or since I couldn't get anything off Synaptic with my internet down - I downloaded them from my Windows PC at [http://packages.ubuntu.com](http://packages.ubuntu.com) and installed them in the order below.

ndiswrapper-common_1.50-1ubuntu1_all.deb
ndiswrapper-utils-1.9_1.50-1ubuntu1_i386.deb
ndisgtk_0.8.3-1_i386.deb


2. You need to download the Atheros Driver:

[http://blakecmartin.googlepages.com/ar5007eg-64-0.2.tar.gz]("http://blakecmartin.googlepages.com/ar5007eg-64-0.2.tar.gz")

3. Extract the Atheros Driver (I put it in my home folder)

4. Open your Terminal and run these commands:

```
sudo su
<password>
ndisgtk
```

3. Wait a few minutes if necessary but the ndisgtk program will open. Click on 'Install new Driver' and find the net5211.inf. Follow the prompts from there and close the program.

I didn't even need to reboot - just clicked on my network icon at the top in gnome and it detected the network I normally connect to.

Hope this helps someone out there. I'm soooo happy to have my internet back and can start enjoying Hardy Heron now!
