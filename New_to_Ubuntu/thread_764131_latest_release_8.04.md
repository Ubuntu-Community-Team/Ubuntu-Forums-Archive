### Post by Tatty on 2008-04-23
> **hulio40 said:**
> well when i installed ubuntu i made a 5gb partition for it to test it out for some time but now i rly like it and have to repartition the hard drives and i feel its kinda dangerous and it would take me hundred of cd's and dvd's to back up  my stuff

The problem with having a small partition is that you are more likely to experience fragmentation of files - linux filesystems are very resilient against fragmentation, but if they get too full (over 80%ish) then they will start to fragment.

when i installed gutsy on my laptop it took about 2.1Gb, so i imagine hardy would take about the same.  But add to that any extra software you have installed, and then any files you have (music and videos in particular can take up a lot) and you can see how the space can add up.

It will probably install fine on a 5Gb partition, but you will have to be very cautious about not filling up the remaining space.
