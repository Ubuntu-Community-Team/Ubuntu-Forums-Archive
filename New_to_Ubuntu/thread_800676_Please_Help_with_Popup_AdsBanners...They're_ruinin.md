### Post by zxscooby on 2008-05-20
Make sure you have pop-up blocking enabled. Use the firefox menu Tools/Options/Web-features/Block popup windows. 
Get the extension 'Adblock' and import the 'G.Filterset', to suppress advertising pop-ups the blocker considers unwanted
