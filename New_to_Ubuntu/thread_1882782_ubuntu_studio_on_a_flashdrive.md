### Post by jjex22 on 2011-11-18
Sorry i think this got a bit distracted. basically your install options are:

1)Install ubuntu studio to the pc's internal ssd

2)install ubuntu studio to an 8gb usb key - you can plug it in  either pc, but it will only boot on the one you installed it on, for the other it'll still act as a usb key basically. (I personally don't recommend this, it is prone to bugs)

3)install Ubuntu to the pc and add the programs and codecs you will be using - more work, more chance of bugs setting up, but will use less of the 16 GB

4) 'install' ubuntu live cd to a 4GB or higher usb key, then add the programs and codecs - virtually the same as the above option but it will boot on most any pc made in the last 5 years (supports booting from usb)
