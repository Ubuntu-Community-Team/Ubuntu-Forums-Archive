That is, unfortunately, a known bug that hasn't been squashed yet. It was originally reported on Launchpad, here:

[https://bugs.launchpad.net/ubuntu/+source/nautilus/+bug/399974](https://bugs.launchpad.net/ubuntu/+source/nautilus/+bug/399974)

The Ubuntu developers determined it was a problem in Gnome, so kicked the bug upstream to them. It can be tracked here:

[https://bugzilla.gnome.org/show_bug.cgi?id=589156](https://bugzilla.gnome.org/show_bug.cgi?id=589156)

It might be worth a read through those two, but if you are going to add anything to the bug, it be be the bugzilla one.

Regards,

Didius
