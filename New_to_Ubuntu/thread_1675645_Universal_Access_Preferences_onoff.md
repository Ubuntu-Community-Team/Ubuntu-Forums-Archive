### Post by pme 72 on 2011-01-26
I had the same issue and tried disabling it in System>Preferences>Startup Applications but that did not work. An Internet search produced several answers and one seems to have worked:

System>Preferences>keyboard>Accessibility, uncheck "Accessibility features can be toggled with keyboard shortcuts". The icon is gone. 

I was using Typing Break from System>Preferences>Keyboard and I wondered if that caused it to appear. Now that you mention it, I had been playing with backgrounds and themes so perhaps that was it. 

A couple of days have passed and I have yet to notice any ill effects associated with the change.
