### Post by Paqman on 2011-09-27
The newer versions of Ubuntu use a different menu from those in your instructions. Instead, hit the Windows button and start typing the name you want (Users & Groups), it will pop up as one of the options and you can click on it.

### Post by dFlyer on 2011-09-27
Welcome to Ubuntu. Are you trying to install window or linux program? If your trying to install windows programs you will need wine or a similar program. If your trying to install linux programs use ubuntu software center or synaptic. If your using 11.10 you will have to install synaptic from the software center or command line with:

sudo apt-get install synaptic
