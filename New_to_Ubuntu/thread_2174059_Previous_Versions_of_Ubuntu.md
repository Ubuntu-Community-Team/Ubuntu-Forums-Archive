### Post by Jonathan Precise on 2013-09-12
As a bottom-line statement, I recommend Zorin OS 5 (based on ubuntu) if you want it to look like windows in general, and have the option for a traditional Linux look {7, XP, Linux (GNOME)}.
You mentioned you would not like windows 7. Unfortunately, the default look is windows 7. This can be changed to XP with Zorin Look Changer (Requires you to log-out and log-in again. FYI: The username is "live" (no quotes) and there is no password. If the login screen asks for a password, just press enter).
