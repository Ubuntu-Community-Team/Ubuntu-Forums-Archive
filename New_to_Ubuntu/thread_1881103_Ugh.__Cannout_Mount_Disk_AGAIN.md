### Post by oldfred on 2011-11-14
Since it is a NTFS partition, you need to have a Windows repair CD so you can run chkdsk. Just like Ubuntu runs fsck on its partitions every 40 boots or so, you need to manually run chkdsk on NTFS partitions periodically.

If you have another computer with Windows 7 or know anyone, then create this:

Make your own Windows recoveryCD/repair:
[http://windows.microsoft.com/en-GB/windows7/Create-a-system-repair-disc](http://windows.microsoft.com/en-GB/windows7/Create-a-system-repair-disc)
