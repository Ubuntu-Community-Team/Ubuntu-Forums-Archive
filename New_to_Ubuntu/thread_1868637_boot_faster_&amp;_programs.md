### Post by jnorthr on 2011-10-25
welcome aboard - pleased you were able to get broadcom working as this is a touchy issue for many people. You can find out more about what hardware you have in your system by using system information that is printed to a terminal screen. Open a console terminal session by clicking menus>accessories>terminal to start a session. then you can type various system commands to learn more. For example, if you type the command ifconfig you will see information about your network. Improve your understanding of ubuntu commands by reading our tutorials and reading the forum pages. Start reading here: [https://help.ubuntu.com/10.04/serverguide/C/network-configuration.html](https://help.ubuntu.com/10.04/serverguide/C/network-configuration.html) to gain more ideas. Good luck.
