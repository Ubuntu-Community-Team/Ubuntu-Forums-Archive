### Post by mivo on 2008-07-15
Internet will work if it is a wired connection or wireless that is very well supported. If you need restricted drivers or need to do anything that requires a reboot, it won't work. The same goes for restricted video drivers. Also keep in mind that running Ubuntu from the Live CD is much slower than running it from the hard drive. So if things seem slow and sluggish, that is because you run it from the CD and don't have a proper installation. It's only good for a general idea what it is like and to see if your basic hardware is supported.

Wireless -> maybe.

The live cd won't touch your hard drive. So nothing you do will show on your windows installation.

----

Notes: you won't be able to install anything on the live cd. So certain things like flash, ... won't work.
