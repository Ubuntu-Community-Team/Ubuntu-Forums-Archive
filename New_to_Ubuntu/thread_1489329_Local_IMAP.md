### Post by HermanAB on 2010-05-21
You can install a Citadel mail server in about 30 minutes.

Citadel can do 'fetchmail' to get mail from another server over POP and then save it locally for you.  In the outgoing direction, it can deliver mail to a 'smarthost', your ISP mail server for delivery.

[http://citadel.org](http://citadel.org)
