### Post by Mark Phelps on 2011-11-11
As I've tried to point out again and again here, reizing Vista or Win7 OS partitions, either with GParted, or with the Ubuntu installer, runs the risk of corrupting them and rendering Windows unbootable as a result.  Some have gone so far as to claim this can't happen -- maybe you should tell them what happened to you.

At this point, you will need to go to the link below, download and burn the proper ISO image to CD, boot from that CD, and run Startup Repair three times -- three times, not once:

[http://neosmart.net/blog/2009/windows-7-system-repair-discs/]("http://neosmart.net/blog/2009/windows-7-system-repair-discs/")

Please note that you have to PAY for these images, now.
