### Post by ofb on 2011-05-04
You don't have enough RAM for Ubuntu.
[https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements)

You have the bare minimum for Xubuntu.
[http://www.xubuntu.org/get](http://www.xubuntu.org/get)

"You need 256 MB RAM to run the Live CD or 256 MB RAM to install." BUT "Once installed, Xubuntu can run with starting from 256 (or even just 192) MB RAM, but it is strongly recommended to have at least 512 MB RAM."

You really want to find some more RAM. It'll help a lot.

By "which version", you mean 10.04 LTS or 11.04? Either, really. Read this post for an explanation. It applies the same to Xubuntu.
[http://ubuntuforums.org/showthread.php?p=10037618#post10037618](http://ubuntuforums.org/showthread.php?p=10037618#post10037618)

IMHO Lubuntu is still just a developing project. It may be more adventure than you want right now. (But it does look promising and you should keep an eye on it.)
[http://lubuntu.net/](http://lubuntu.net/)

### Post by frigginjoe on 2011-05-07
I have Ubuntu on a laptop with 256 megs of ram and I wouldn't cal it very useable.

512 megs running slow on Xubuntu doesn't sound right at all.  Look into seeing if integrated video is taking up a lot of that memory.
