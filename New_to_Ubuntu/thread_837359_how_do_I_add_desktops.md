### Post by drs305 on 2008-06-22
For ubuntu, do you have a 'workspace switcher' in the bottom panel (rectangular which allows you to select the workspace)? If so, right click, Preferences, and select the number of workspaces you want.

If you don't have it, right click the panel, Add to Panel, Workspace Switcher. Then do as above.
