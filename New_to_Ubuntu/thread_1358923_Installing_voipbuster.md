### Post by Temposs on 2009-12-19
Voipbuster is a program that can only be run on Windows. You cannot run it on Linux.

If you want to try to run a Windows program on Linux, you may download the **WINE** program, which you can get from Add/Remove or the Ubuntu Software Center. After you install that, you can try running Voipbuster, but there is no guarantee it will work at all.

If you want to use VOIP on Linux, probably the best way to do that is through the program called **Ekiga**. You can install that through Add/Remove or Ubuntu Software Center. I think it does the same thing as Voipbuster does.
