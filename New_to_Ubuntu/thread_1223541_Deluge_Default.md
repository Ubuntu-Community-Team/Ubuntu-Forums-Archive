### Post by Paddy Landau on 2009-07-26
For future note, this is how you do it through nautilus in the GUI.

[LIST]
[*]Right-click on a file with the right extension (e.g. *.torrent*).
[*]Select *Properties*.
[*]Go the tab *Open With*.
[*]Select *deluge*. If it isn't listed, select *Add* and find deluge.
[/LIST]
