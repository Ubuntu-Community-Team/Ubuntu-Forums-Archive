### Post by ankspo71 on 2009-09-29
Hi,

"Splash screens" are the first and last graphics you see when you boot and shutdown your computer. They are the ones with the progress bars.
"Gdm themes" are the graphics you see when you log in and out. 
"Metacity themes" are generally the window borders and or the 'controls' that you see when you click on the 'customize' button to the gnome theme manager. 
"GTK themes" (1 and 2) (2 is what we normally prefer in the latest gnome), is usually the closest to a complete theme you can get. It usually a includes metacity theme, a menu theme, and sometimes a panel theme too. 
"Emerald themes" are another window border (or window decoration as they call it) and can only be used with Compiz 3d effects. The package called [ emerald ] can be found in synaptic package manager, and it us used to install emerald themes. Note that this program requires that you use the code ```
emerald --replace
``` in your start up applications list (to runautomatically at start up), or in a terminal window to test the theme out.

Also, most themes (metacity, gtk, and even icon packs) can be installed by drag and drop in the theme manager window of gnome if they are packaged correctly. Sometimes you might have to manually extract them and place them in your home/username/.themes folder, or /user/share/themes. Icons have there own folders btw.
Hope this helps.
James
