### Post by Paqman on 2008-12-31
> **luke0927 said:**
> OK i'll try the VLC just to check it out...when you say repository is that where i can go to the synaptic package manager and get things?  or just do the apt-get command

The repositories are the online software libraries that Ubuntu's package manager gets the packages from. Synaptic, Add/Remove, Update Manager and apt-get or aptitude commands are all different ways of telling the package manager to go to these repositories to get stuff.

All the software in the repos is safe, tested and adware/spyware free. Having these centralised libraries of software is what helps keep Linux free of malware.

In short, repos rock.
