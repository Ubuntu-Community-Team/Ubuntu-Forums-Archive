### Post by SoftPops on 2009-09-29
I had the same problem with Evolution after changing my login password. 
 
Try going into "[COLOR=red]System", "Preferences", "Encryption and Keyrings[/COLOR]"
My Mistake: should be "[COLOR=seagreen]Applications, Accessories, Passwords and Encryption Keys[/COLOR]"
 
There should be a tab called "Passwords" which gives me one entry "Passwords: Login", if I right click this there is a "Change Password" option.
 
In here I have changed the password using my previous login password as the old password and my current one as the new password. 
 
This has removed the "keyring password" screen whenever I open Evolution.
 
Hope this helps.
