### Post by corncob on 2010-06-16
I right click on the picture, then on Open With.....  From the drop down menu I choose Gimp Image Editor.  When the picture opens I select Image then, from that drop down menu, I select Scale Image.  I'm not completely sure but it seems that Gimp didn't come with Lucid and I had to download it.  I know it came with previous Ubuntu versions but I guess they're trying to keep the size of it down so it will fit on a CD.

### Post by oldsoundguy on 2010-06-16
Gimp did not come with Lucid as a default in the install. That was done to keep the size of the live disk down to CD size, as not everyone has a DVD burner.

You will have to go to the software center or Synaptic and install Gimp if you want it at your fingertips.

### Post by dranockcir on 2010-06-16
Start Synaptic Package Manager, search for nautilus-image-converter, mark it for installation and click apply. :-)

While you are in there though you may want to also check

nautilus-wallpaper
nautilus-gksu
nautilus-open-terminal

adding those will give you right-click options of

Open as administrator

Open in terminal

Set as wallpaper

All this, of course, if you are using Gnome and Nautilus. :-)
