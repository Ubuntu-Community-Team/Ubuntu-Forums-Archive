### Post by mgmiller on 2009-08-22
There are a lot of things you can try listed in this thread:
[http://ubuntuforums.org/showthread.php?t=170862](http://ubuntuforums.org/showthread.php?t=170862)

Personally,  I use the nautilus-image-converter.  Just ctrl-click to select all the images you want and then right click and select resize image.

Unlike in the thread, which is old, you will find the package nautilus-image-converter in the repos now.

I also like gthumb which  can do a batch convert (scale images)

There is also a terminal command in the thread, you can try that.

Good luck.
