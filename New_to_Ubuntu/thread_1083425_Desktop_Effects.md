### Post by Hospadar on 2009-02-28
Kwin (the kde window manager) does most of the main things that comiz does.  To my knowledge it doesn't have all the options and plugins that compiz has, but It works great and I don't find anything missing that I want that compiz has.

I'm using kde 4.2 in the jaunty alpha and it works great.

If you really want compiz i would suggest just going with gnome.  A lot of the features of compiz really only work well when integrated into gnome and I think you'd be wrestiling a lot with the system when kwin or compiz in gnome would just be better
