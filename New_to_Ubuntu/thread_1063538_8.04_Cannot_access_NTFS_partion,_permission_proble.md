### Post by taurus on 2009-02-08
Try using the ntfs-3g driver.

```
sudo umount /dev/sdc1
sudo mount -t ntfs-3g /dev/sdc1 /mnt/hostc
ls -la /mnt
```

### Post by Rokurosv on 2009-02-08
Try this:
```

/dev/sda1  /mnt/windows  ntfs-3g users,noauto,uid=1000,gid=100,fmask=0113,dmask=0002  0 0
```

This makes your partition mountable and unmountable by any user and that the user and group owns everything on that partition. You could also try to remove the noauto part and replace it with auto so that it gets mounted everytime you start your pc.
