### Post by beew on 2011-06-27
If you asked the question a month ago I would say 10.10 is the best. 

But 11.04 seems to have come quite a long way by now and I have not experienced major problems after numerous updates. It is very fast once booted (boot up time is slower than 10.10 because it takes longer to load Unity, I suppose) It is actually faster than 10.10 on my machine, but then I use many ppas and my kernel is 2.6.39 rather than 2.6.38 so it may not the same if you get a vanilla install.

But there are still some bugs, mostly Unity related. For examples,  there is no consistent way to put launchers on the Unity bar,--sometimes you drag and drop from the dash, sometimes you start the app and click "keep in launcher" and sometimes you have to find the actual desktop file and drag it to the Unity bar; buttons and tray icon problems; problems that sometimes happen when maximizing and minimizing windows etc.

If you want to get used to Unity I would recommend 11.04 instead of using an old Unity version with 10.10 because it is likely to be even more buggy.
