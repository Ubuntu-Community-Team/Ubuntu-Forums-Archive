### Post by yeats on 2009-11-01
Quick forums tip:  ask questions in separate posts and keep them short.  This is definitely a "[tl;dr]("http://en.wiktionary.org/wiki/TLDR")" community :-)

> 1. Partition Size

I really don't know what size to allocate for the Ubuntu partition. I'm not sure what all I will end up installing and using as far as software is concerned. 

Maybe a minimum of 20GB?

> I'm hoping to find a great Video editor

Try [Cinelerra]("http://cinelerra.org/") or [Kino]("http://www.kinodv.org/"), which are available through the Ubuntu Software Center once you install.  Lumierra, which is a Cinelerra fork is under development and looks promising.

> a good alternative to Photoshop

[GIMP]("http://www.gimp.org/") is the closest you're going to get and works well, depending on your needs.  You can use it on Windows too.



> and possibly mess around with various DAWs

Try [Ardour]("http://ardour.org/").

> Do I need a lot of space to save files, or can I save and access them on/from my Windows partition?  

That is really what I don't understand: the relationship betwixt the two partitions. In my mind it would be great if I could run the programs in Ubuntu, but save and access using my Windows partition, or one of my external drives (both WD; both Fat32, right?). Is that how it works if I want it to? If so, then I may not need more than the standard 10GB (as I have seen suggested many times) of partition space.

I think, given your interests in a/v production, that you *do* need a lot of disc space.  I would use one of your external drives between Windows/Ubuntu.  If the external drive is already in a Windows-readable format (NTFS/FAT32), Ubuntu can read it fine - no need to include it in installation (which is to say, keep it unplugged while installing).

> 2. Reading various Windows files

Will I be able to read .txt and .doc files in Ubuntu? Really, Windows Notepad is one of my most often used programs; its my brain. I have that thing backed-up a dozen different ways, because if I don't have Notepad I wouldn't know which way is up!

Yes, you will be able to open .txt (using Gedit, the default text editor) and .doc (using OpenOffice.org, which is a lot like MS Office and is format-compatible as far as .doc -> OO.org goes).

Also, try out Tomboy, the note-taking program.  It's pretty handy.
