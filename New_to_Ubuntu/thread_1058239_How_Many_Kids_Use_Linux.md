### Post by cfree220 on 2009-02-02
Don't know if I still count as a kid. I'm a Freshman in college and didn't have my own computer until just this year (before I had to share a desktop with the rest of my family). My laptop came with Vista Home Premium, which I immediately upgraded to Vista Ultimate. Overall, I really liked Windows, and will probably use it again (in dual boot) when Windows 7 comes out. After having more than my share of problems with Vista, I finally gave up on it. Yesterday I installed Ubuntu 8.10 64-bit on my Dell XPS M1530. I'm still learning my way around, but it's fun to learn about how Linux works (it really is completely different from Windows).

After having been forced to reinstall Windows 3 times (actually, only two of them were necessary, the third was to make the configuration more maintenance friendly), I'm pretty comfortable with OS installs. In the past, I've installed Ubuntu to flash drives and external HDD, so I was familiar with the installer by the time I was ready to make it my only operating system.

When I run into problems with Ubuntu, I usually turn to google or ubuntuforums.org. The people here are so much more friendly and helpful than Microsoft tech support, which makes troubleshooting a lot less painful.

I think that I like Ubuntu and Windows for different reasons. I'm still acclimating to Linux, so I'd definitely have to say that I find Windows more user friendly. However, I've gotta say that I'm loving this free, open source thing. With different types of programs made so easily accessible, I'm able to learn a lot more about computing than I could in Windows.

Although I'd prefer to use Ubuntu as my only operating system, it's starting to look like I'll have to set up my computer in dual boot with Vista. Some of my classes require us to install software, which is usually only available for Windows and Mac OSX. I've not been able to get some of these to work under Wine.
