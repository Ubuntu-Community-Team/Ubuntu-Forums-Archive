Anyways  how to install through WINE ...

Insert the CD or ISO (Windows software you wish to install) ...  in the computer...

Afterwards...  wait till it appears in your Desktop  ...   Open the CD or ISO  ..  and   once inside  *Select all the Files .. and  Drop them into your Desktop  

Now  simply ...  on your Desktop .. *RIGHT-CLICK*  the  Setup.exe  file  and you will see  "Install with Wine"


After you install it.. you should be able to .. head in  *APPLICATIONS*... and in WINE.. you should be abel to see the Windows Application you just installed
