### Post by nothingspecial on 2010-07-27
Answer .........

To install it on your netbook you need to make the usb stick bootable.

If you already have {U,K,X,L}buntu then install usb-creator from the software center.

If you only have windows then you will need to use unetbootin

[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

First though, copy the iso from your stick to your computer because using eithyer method will over write any data on the stick, including the iso...... if you see what I mean
