### Post by ikt on 2011-03-16
> **BretMan said:**
> How exactly is it great for business 

If you mean business server wise: for example imagine you buying a server, and need to install windows server 2008 + exchange + other licensing for other applications, anti-virus etc.

You can build a server which offers similar programs and functions for much less, with a higher level of security for reasons mentioned below.

If you're talking about business desktop wise, if your document needs are relatively simple and you aren't doing complicated macros, scripts, or require features that *only* microsoft office has, then you can certainly (again) roll out desktops for a significantly cheaper price and find less security issues overall.

With that said I only recommend a linux server for admins who have been somewhat trained in the area, many Microsoft certified professionals come in expecting it to be easy, not realising they need to immediately dump all the knowledge they have gained supporting windows.

And I don't generally recommend linux on the desktop unless there is a permanent support person available to guide and assist users as they get used to the new desktop, a better option is to introduce users to open source applications on windows, firefox, libre office, etc and then once they are familiar with the applications switch over to ubuntu, that way you only have to deal with the problems and issues with the apps one at a time, instead of training users to use firefox, libre office, ubuntu etc all at once.




> **BretMan said:**
> and since security is a big issue for me, how also is it better in that respect?  Thank you.

Linux is designed to be secure from the ground up, Windows was built poorly and definitely not with security in mind, windows vista and 7 have patched in security but it's still not as secure as any linux os that has sane defaults.
