### Post by Djalmahal on 2010-10-04
The difference is that running in it VirtualBox means it runs on top of Windows, therefore you tax your system more by running 2 OS at the same time, boot time is similar to booting Ubuntu alone, but you an switch to Windows in no time. You can also have Ubuntu have access to your Windows hard drive (optional), so that's easy for Music, Documents etc.

Dual-Booting is going to take you the same time to set up (plus a bit), but you have lower system load. Ubuntu should boot in 10-15 secs on a gaming computer, so switching should take you about 30-45 secs.

One last word on Hardware, if your hardware is not supported out of the Box by Ubuntu (certain wireless etc) running it in VirtualBox is going to give you a better experience (out of the box) as VirtualBOx just passes the internet on. Again, most of these things are easy to fix, and 10.10 now supports my Atheros Wireless Card out of the box.

Hope that helps, you can always try out UBuntu in VirtualBox easy for 2 weeks, and if you like it enough you probably anyway want to install it as stand-alone.
