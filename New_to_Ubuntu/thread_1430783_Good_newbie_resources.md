### Post by ndefontenay on 2010-03-15
It's a rather large subject to handle in such a broad way.

Do you want to learn a bit of scripting language? You should Google for bash tutorial.

Try gawk also. It's pretty fun.

If you want to create themes, it's a whole different world too.

Or if you want to contribute to linux development, there's a variety of way to do so.

We can't decide that's good for you. You should try a bunch of things and decide what you like most.

Here's a nice homework for you:

Write a script to backup your files on an external hard disk and schedule it daily.

Requirements are as follow:
no backup are deleted until disk is full.
When disk is full, delete the oldest backup. If you're still lacking space, delete the 2nd oldest.

Some command you might want to look into: tar, rsync, cp

Nico
