### Post by dearingj on 2010-03-30
A few that I find very useful:

To install a program (replace package with whatever you're trying to install)
```
sudo apt-get install package
```

To find out more about how to use a program and what that program can do (replace command with the name of a command):
```
man command
```
For example, to find out more about apt-get, you would enter
```
man apt-get
```

To move into a directory (replace /path/to/directory):
```
cd /path/to/directory
```
