### Post by 1915flyer on 2010-03-10
> **crlang13 said:**
> Yep, then right click on the keyring you want to change and select "change password."

To be a little more clear about this, I had already changed my login password using the terminal / passwd command but it did not change my keyring password so it never occurred to me I had to change my login password again to change the keyring password. 

By going to Applications/Accessories/Passwords & Encryption Keys and right-click on Passwords:login, then selecting Change Password, I was able to change it using my original login password. This was not intuitive as I had already changed my login password.
