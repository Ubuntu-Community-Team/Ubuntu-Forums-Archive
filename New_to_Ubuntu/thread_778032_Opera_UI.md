### Post by scragar on 2008-05-02
> **iSplicer said:**
> Like what?

few things I love about opera, that firefox has never done:

[list][*]Personal bar(forget about adding things to the status bar, opera gives you a whole extra one to play with) and the whole set of customization with them.
[*]removing item from bookmarks toolbar does not delete bookmark.
[*]same form of quick searches as konqueror offers, "g term" does a google search for term(in konqueror I think it's "gg-term" or something similar, either way both let you change the whole behaviour of it).
[*]enable/disable java, flash and javascript on the fly, easy.
[*]opera console for errors is, in some ways superior to firebug(although line numbering in opera plain sucks error messages make furebugs look like enigma's by comparison).
[*]a really fast rendering engine, opera 9.2 is faster than firefox3, and a tiny fraction behind konqueror, and story goes that 9.5 is even faster than that.
[*]notification icon can be used to make opera disappear and reapear, perfect for when your waiting for a download or 2.
[*]restart option in menu, even if you need to enable that option first(dunno why, but it's cool).
[*]an RSS feeds system that feels like an RSS feeds system, forgive me, but live bookmarks are a terrible replacement.
[*]best java/flash support, in that flash in opera never crashed for me, yet in firefox it does. Java appears to run faster in opera, I could be mistaken though.
[*]ctrl+enter auto complete, firefox's falls short for some sites that try to help with their own javascript powered ones.
[*]more - I'm afraid of putting a complete list, since there's some pretty ambiguous claims, but the above is a fair point I think.
[/list]

Some thing I dislike:
[list][*]closed source
[*]lack of close buttons on tabs
[*]no clear private info option
[*]excessive ram usage after around 2or3 hours of use
[*]zoom should not be a replacement for text size, it should be alongside.
[*]menu's look like something from windows 95
[*]no 64 bit version.
[/list]

however, in the end I don't use opera now because of the last failing, nothing above that is annoying enough to counter act it's possitives.
