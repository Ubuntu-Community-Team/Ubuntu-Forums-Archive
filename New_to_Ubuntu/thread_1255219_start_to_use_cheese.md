### Post by bug67 on 2009-09-01
1.  Upper left hand of screen click on "Applications"
2.  Click on Add/Remove...
3.  In the search field type "cheese"
4.  check the check box
5.  click "Apply Changes"
6.  Cheese away!

**_Alternate method_**:

1.  Click on "System"
2.  "Administration"
3.  "Synaptic Package Manager"
4.  Type in "cheese" in the quick search field
5.  Click on the check box and click "mark for installation"
6.  Click "apply"
7.  Cheese away!

**_Alternate method_**:

1.  Click on "Applications"
2.  Click on "Accessories"
3.  Click on "Terminal"
4.  In the terminal type the following:

```
apt-get install cheese
```

5.  To launch cheese, from the terminal type:

```
cheese
```

6.  Cheese away!
