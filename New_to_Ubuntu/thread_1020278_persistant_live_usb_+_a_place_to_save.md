### Post by talsemgeest on 2008-12-24
As for the firefox downloads, all you have to do is go, in firefox: Tools>Options. Under the downloads section, choose where you want your downloads to go. But as for applications, as far as I know, there is no way to change the install directory. Since the installed applications are of no use in other Operating systems, I can not see this being a problem. However, if space is your problem you can always resize your partitions using Gparted.

Hope this helps :)
