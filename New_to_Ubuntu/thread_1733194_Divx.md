### Post by marl30 on 2011-04-19
> **chasin3883 said:**
> How do I play a divx movie on Unbuntu?  I am using Chromium and am very new to this any help would be greatly appreciated.

If you don't have Ubuntu-restricted-extras installed, go right to the software center and install it. It will add the Totem plugin that supports divx. You can also try the gecko-media plugin, it handles Divx streams better than Totem's plugin does.
