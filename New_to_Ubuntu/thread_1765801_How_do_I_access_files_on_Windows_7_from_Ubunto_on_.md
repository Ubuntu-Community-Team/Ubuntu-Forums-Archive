### Post by ppv on 2011-05-23

### Post by athenroy on 2011-05-23
I used Wubi for my dual boot install.  You can click on Places, go to computer, file system, then the host folder, users, your user name and you should have all your files available to you.  At least that is the way mine is set up! :)

### Post by ZeddCypher on 2011-12-27
With a dual boot build I decided to use a 80GB Solid State for Windows 7 and Ubuntu 11.10.

Leaving the OSs where they were I had a seperate 1 TB spinning disk drive mounted separately.  Windows and Ubuntu both recognized the other hard drive and I have just stored all data/music/movies/documents etc... and have not had a problem going back and forth between the operating system and sharing files between them.

Seems to be the easiest way with the assumption that you are using a desktop.  May be a bit tougher with a laptop.
