### Post by ukripper on 2008-06-10
> **ghostier said:**
> Hi. After installing KDE4 my login screen and start up pictures became like Kubuntu instead of ubuntu. When I click the login manager it says that the display manager is KDE4 instead of GNOME, my question is how would I change the display manager back to gnome?

Install startup-manager and you can change your splash screen and bootloader screen using that.

```
sudo apt-get install startupmanager
```
