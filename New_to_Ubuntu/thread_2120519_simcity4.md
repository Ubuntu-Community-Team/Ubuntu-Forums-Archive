### Post by sandyd on 2013-02-26
Unfortunately, software for Macs will only work on Macs, and not on Ubuntu

However, if you want to run Windows software on Linux, an app called WINE (WINE is not an Emulator) runs some windows applications on Linux. You can check the compatability list at [http://appdb.winehq.org/](http://appdb.winehq.org/)
