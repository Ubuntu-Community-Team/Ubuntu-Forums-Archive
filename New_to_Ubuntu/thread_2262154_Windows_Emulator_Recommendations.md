### Post by Holger_Gehrke on 2015-01-23
The AppDB at winehq.org rates wine's ability to run Familty Tree Maker 2012 as 'garbage' (ratings go from 'garbage' for won't run, no matter what you try to 'platinum' for it runs in wine as if it was written for it). It also gives a 'platinum' rating for the 2005 Edition and a 'silver' for 2006. And it mentions two programs for Linux as alternatives: Gramps and GeneWeb, both of which should be in the repositories (they are for 12.04, don't know what you are running).

AFAIK there's no other emulator besides wine, playonlinux which you might hear about is actually just wine with some fixes for specific games built in. The only way I can think of to run this program would be to keep running it on window, either in a dual boot environment or in virtual machine.
