### Post by walt.smith1960 on 2010-09-06
If G speed is okay, I've had excellent luck with TrendNet's UB424. The version I have has been plug & play on several machines.  This is a case where version matters, however. Mine is H/W version 3.1.  Hardware version 2.X used a different chip that was not plug & play--it needed NDISwrapper, I believe.  Version 3.1 has been around for a long time so you should be okay.  I can't help with wireless N products, sorry.
