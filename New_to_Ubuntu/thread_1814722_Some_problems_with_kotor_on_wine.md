### Post by Mark Phelps on 2011-07-30
Since you're new, you probably didn't know ... but in future, BEFORE you install something in Wine, save yourself some trouble and go to the WineHQ database site to check on the Windows app rating.

Had you done that for kotor, you would have seen the following page:

[http://appdb.winehq.org/objectManager.php?sClass=application&iId=9107]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=9107")

That Garbage rating means it's NOT going to work -- no matter what you do.
