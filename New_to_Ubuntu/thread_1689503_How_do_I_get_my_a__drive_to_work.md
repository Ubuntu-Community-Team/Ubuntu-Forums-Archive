### Post by JKyleOKC on 2011-02-16
Linux doesn't use drive letters. However you should have an icon labelled "Floppy" on your file manager, **IF** you had a (readable but not bootable) disk in the drive at boot time. Otherwise, try putting a disk in the drive. After a few seconds, it should show up in the file manager due to automatic mounting.

The same rules apply to the CD or DVD drive; with no disk inserted, they are simply ignored, but inserting a disk causes automounting.
