### Post by 3rdalbum on 2013-04-02
I think you've just seen some old screenshots of Gnome 2, which is a dead desktop environment. The Gnome 3 and Unity desktops don't support these frills for a number of reasons.

You can get CPU usage in your top panel by using System Monitor Indicator (indicator-sysmonitor package), and there are a few other indicators available, but nothing with eyes or other kinds of graphics.

### Post by Skald9 on 2013-04-02
The Gnome 2 desktop is still alive as a fork, called MATE. It's not and official Ubuntu desktop, but it can easily be installed on Ubuntu.
