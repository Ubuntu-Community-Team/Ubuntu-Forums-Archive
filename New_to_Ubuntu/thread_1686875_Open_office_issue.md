### Post by Vaphell on 2011-02-13
the sad truth is that only MSOffice can be 100% compatible with MSOffice, and even that not necessarily between different versions of MSOffice.

you have 4 choices
- stay with dual boot
- dump xp and live with the fact that OO is not perfect
- try installing MSOffice in wine (check [http://appdb.winehq.org/](http://appdb.winehq.org/) to evaluate your chances of success)
- install winxp inside a virtual machine so MSO can live in a bubble of its native environment
