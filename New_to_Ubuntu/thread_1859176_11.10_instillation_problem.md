### Post by kolinab on 2011-10-13
I hope we can help.

Did the installation finish 'gracefully' with no errors?

What kind of system did you install on? 

Before you installed, did you run a live version from CD or USB for a few minutes to see how your hardware worked? What happens if you boot into a live session again, does everything work?

I might:

1 - check that the iso file you are installing with is not corrupt: [https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM). You can find the hash files here at the top of the list: [http://releases.ubuntu.com/oneiric/](http://releases.ubuntu.com/oneiric/)

2 - play a bit in a live session to see that your hardware works

3 - Maybe just try to reinstall from the same media, once you have verified your iso is not corrupt.

Did you install Ubuntu as standalone, alongside a windows installation in a dual boot, or what configuration? If these suggestions above don't help, or you don't know how to do them, post back with your problems and I'm sure we can help you out!
