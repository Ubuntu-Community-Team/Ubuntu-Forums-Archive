### Post by newb85 on 2012-11-11
If you're new to Ubuntu and Linux, you would be well-advised to install alongside Windows, rather than over the top of it.  It will cost you some HDD space, but that's it.  Down the road, if you've been Using Ubuntu for a while and determined that you don't need Win7 for *anything*, then you can still remove Windows and give the rest of your HDD to Ubuntu.
