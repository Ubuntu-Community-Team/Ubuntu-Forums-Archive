### Post by arpanaut on 2014-01-28
11.04 is no longer supported, repositories are now off line.
You may find some relief through: [https://help.ubuntu.com/community/EOLUpgrades](https://help.ubuntu.com/community/EOLUpgrades)

### Post by Impavidus on 2014-01-29
11.04 is unsupported. The repositories have been archived, still reachable, but no longer updated. There is no way to update a 11.04 system beyond the moment it reached end-of-live 15 months ago other than manual installation. (In theory there could be some PPAs left, but I think most of them have been taken off-line when 11.04 reached end-of-life.)

If you can't get used to the Unity interface, give Xubuntu a try. It has a more traditional look.

### Post by Bucky Ball on 2014-01-29
+1 to Xubuntu if you don't like Unity. Many have swapped. If you want the Ubuntu apps and not Unity, Ubuntu has gnome-fallback or you can install Xubuntu's desktop environment by installing xfce4, log out, choose it from 'Sessions', log in.

11.04 is a dead horse. I wouldn't waste too much time on it. Go for the stable 12.04 LTS or 13.10, both directly upgradeable to 14.04 LTS released in April which has five years support (three for Xubuntu).
