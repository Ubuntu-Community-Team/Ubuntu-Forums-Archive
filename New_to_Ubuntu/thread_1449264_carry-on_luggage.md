### Post by scottuss on 2010-04-07
If you're talking about Ubuntu (or any other Linux distro) then, yes, you can have all of your media files (music, pictures, video)

Firefox is Firefox, no matter which Operating system you use, and so your bookmarks can be easily imported.

McAfee is not compatible, you don't need anti-virus software on Linux (my opinion, this is generally agreed upon by most Linux users, although some will argue otherwise)

### Post by oldsoundguy on 2010-04-07
Think about it this way:

IF the file is post-able and readable by others on the internet .. regardless of their browser or system (in other words UNIVERSAL FORMATS), they can migrate between systems.

That means .doc, jpg, bmp, mp3, mp4, mpg and so on.

As to McAfee, first off that piece of junk shouldn't be on ANY computer in the first place.
But echo on the "no need for an AV in Linux"  I have been running various builds of Linux for 10 years and have NEVER had a virus .. even when I click on a suspicious link!
AV is only REALLY needed if you are going to set up a mail server where Windows machines will have access to same. OR, if you do the dumb thing of FORWARDING eMail instead of copy and paste and send.

### Post by Steven_S on 2010-04-08
For Firefox bookmarks, you can use Xmarks (which is available as an add-on to the browser, or see [http://www.xmarks.com/](http://www.xmarks.com/)). This will basically put all your bookmarks online, and lets you synchronise them between computers and browsers. 

I have been using that for a while now, and find it extremely handy.
