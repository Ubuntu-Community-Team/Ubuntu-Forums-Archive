Seems like it would run fine, the only issue I see is this:
[COLOR=#800000]512 MB AMD Radeon HD 8650G (Integrated) and 2 GB ATI Sunpro HD 8570 Graphics (Dedicated)[/COLOR]
[COLOR=#800000][/COLOR]
The reason I say it is an issue, is that Graphics Card switching with Intel and AMD / Intel and nVidia seem to not work well in open source environments. This, however, is an AMD / AMD switch. Because of this, I can't say it will actually be an issue, but I would keep it in mind (for potentially getting a refund or otherwise).

A workaround might be to disable the onboard graphics in BIOS settings if it exists, assuming that it does actually produce a problem.
