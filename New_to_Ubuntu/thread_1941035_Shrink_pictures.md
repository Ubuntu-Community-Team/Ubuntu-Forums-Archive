### Post by lile001 on 2012-03-14

---

### Post by yetiman64 on 2012-03-14
```
sudo apt-get install nautilus-image-converter
```will give you the resize and rotate functionality in the context menus in 11.10. I have it installed and use it here. Can be used with a multiple file selection. Cheers.
