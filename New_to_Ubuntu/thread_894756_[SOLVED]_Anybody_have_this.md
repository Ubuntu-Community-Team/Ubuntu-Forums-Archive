just grab all the parts seperatly

Gtk theme - MinimaLista with a green color theme by default
[http://gnome-look.org/content/show.php/MinimaLista?content=62545](http://gnome-look.org/content/show.php/MinimaLista?content=62545)

Window manager - System G(C) 2005 Andy Morum. Modified to have a simple

[http://gnome-look.org/content/show.php/SystemG?content=19800](http://gnome-look.org/content/show.php/SystemG?content=19800)

Icons - OSX theme [http://gnome-look.org/content/show.php/OSX?content=31618](http://gnome-look.org/content/show.php/OSX?content=31618)
