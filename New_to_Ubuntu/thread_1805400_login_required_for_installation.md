### Post by paker on 2011-07-16

### Post by 3rdalbum on 2011-07-16
The disc or the ISO file you downloaded has an error on it. Try checking the MD5 sum of the ISO file according to the instructions on the Ubuntu website. When you know you have a copy of the Ubuntu IAO that matches the MD5 sum, burn at 8x.

The login screen should not appear on the Ubuntu CD.

---

### Post by paker on 2011-07-16
thank you.

EDit: I got a new download, checked md5sum, burnt @ lowest speed (16x). Installation went well. Thanks again.
