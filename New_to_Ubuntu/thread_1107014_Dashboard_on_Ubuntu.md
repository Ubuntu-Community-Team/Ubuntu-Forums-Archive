### Post by damis648 on 2009-03-26
If you're using compiz (aka Desktop Effects) you can have a dashboard-like thing. You just need to install screenlets:
```
sudo apt-get install screenlets
```
and then enable widget layer in ccsm (if you don't have it, do:
```
sudo apt-get install compizconfig-settings-manager
```
)
and set all the screenlets to be widgets (Right Click > Window > Widget):popcorn:
