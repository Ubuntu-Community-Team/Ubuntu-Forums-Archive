### Post by chaanakya_chiraag on 2009-11-21
First off, 1.3.9 is still BETA, so I would be careful.  However, the Ubuntu devs believe 1.3.9 is stable enough to be in the repos.  However, I have noticed a very annoying bug - FFMpeg import DOES NOT WORK.  However, I used 1.2.6 so long ago that I'm not sure if FFMpeg import is even **in** 1.2.6.  Bottom line, if you can try out both, try them both out and see which one suits you more.  I personally would still recommend 1.3.9 over 1.2.6 mainly because it's come a LONG way.
