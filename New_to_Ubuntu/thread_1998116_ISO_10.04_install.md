### Post by zombifier25 on 2012-06-06
If you have installed 12.04 before, I wonder why you can't do this one. If 12.04 was installed by someone else, then
Burn the .iso to a CD. If you don't have a CD burner, use a tool like Ubuntu's built-in USB creator or Unetbootin. Then reboot and boot of the CD or USB drive, choose Install Ubuntu and choose to replace the current Ubuntu installation. You may want to preserve your personal files, so back up your home folder.
