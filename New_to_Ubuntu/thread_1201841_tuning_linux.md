### Post by gjoellee on 2009-07-01
This is how you install themes (including, GTK + themes, icon themes an metacity themes).

> click on computer&#8594;desktop preferences. there you will find a theme manager. in there, all your themes are stored and changed.
you can add more themes by downloading them at e.g. [www.gnome-look.org](www.gnome-look.org). there you will get the tar.gz files for the themes. simply drag and drop them into the theme selector window and they get automatically installed.

have fun :)
this post was form "arctic" in 2005.

Websites where you download themes for GNOME:
[http://gnome-look.org/](http://gnome-look.org/) (the most used website)
[http://art.gnome.org/](http://art.gnome.org/)
[http://www.linuxmint-art.org/](http://www.linuxmint-art.org/)

You may find more website on Google!

Are you referring to gnome-do (the alternative to quiksilver)? You can install it via the terminal:

```
sudo apt-get install gnome-do
```

If you're looking for gadgets/widgets, try screenlets:

```
sudo apt-get install screenlets
```
