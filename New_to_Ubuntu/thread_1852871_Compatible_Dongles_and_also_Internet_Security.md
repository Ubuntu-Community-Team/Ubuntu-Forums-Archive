### Post by ppv on 2011-10-01
You can try Ubuntu without actually installing it by creating a live USB stick. This way you can also check whether your existing dongle will work with Ubuntu or no. Download Unetbootin to create a live USB stick.

Also, in recent versions of Ubuntu, there isn't a separate version for netbook aka UNR.

There are already security features built in to Ubuntu and you do not need to have a anti-virus program ar anything else like that. Ubuntu already has a firewall built in and if you want to customize that firewall then there are many tuts already out there.
