To use it as a webcam, the computer must first see it as a camera.  For many (most?) camcorders, the computer sees them as hard drives.  I have a Canon DSLR which works as a webcam, but both my camcorders do not.

Besides, you can pick up a webcam for $10 these days.  I have a grotty Z-Star Microelectronics Corp. ZC0303 WebCam which works fine with Skype or Cheese.
