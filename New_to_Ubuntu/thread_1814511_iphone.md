### Post by guilleme on 2011-07-29
Hi. It should by default. The answer depends on what you want to do. 
From the base up, there is a library called libimobiledevice ([http://www.libimobiledevice.org/](http://www.libimobiledevice.org/)) that enables linux to access the ios device via usb. I think it comes preloaded in ubuntu 11.04, and it certainly does in 10.10. With that, you can access you'r iPhone's files and folders. If you can't access the iphone at all, look for this library on synaptic. 
If you want something more itunes-like (for syncing and moving music/photos around), you can use rhythm box or one of many programs for managing iphones found on the ubu. software center. 
Hope it helped :).
