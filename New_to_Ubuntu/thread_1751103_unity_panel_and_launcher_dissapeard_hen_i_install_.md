### Post by bwallum on 2011-05-06
At the Desktop the Unity panel is visible by default. When an application is loaded full screen the Unity panel disappears. It can be made visible by touching the left side of the screen with the mouse pointer. Failsafe is to put the mouse pointer top left because it is possible to configure Unity to show the panel either by touching the left side or by touching the top left of the screen with the mouse pointer.

Apologies if you have discovered this already.
