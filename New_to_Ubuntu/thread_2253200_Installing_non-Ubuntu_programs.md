### Post by claracc on 2014-11-18
To install .exe (windows) files in ubuntu, you first need to install wine or play on linux, and then, if the exe program you want to install is supported ( not all windows programs run in wine) you will install it inside wine.

See:

[https://help.ubuntu.com/community/Wine](https://help.ubuntu.com/community/Wine)
[https://www.winehq.org/](https://www.winehq.org/)
