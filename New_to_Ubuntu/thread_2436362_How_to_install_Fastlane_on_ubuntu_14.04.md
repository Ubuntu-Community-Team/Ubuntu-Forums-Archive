### Post by guiverc on 2020-02-05
Ubuntu 14.04 LTS was the 2014-April (releases are *year.month* in format) which had 5 years of supported life. It reached the end of it's five year life in 2019-April ([http://fridge.ubuntu.com/2019/05/02/ubuntu-14-04-trusty-tahr-reached-end-of-life-on-april-25-2019-esm-available/](http://fridge.ubuntu.com/2019/05/02/ubuntu-14-04-trusty-tahr-reached-end-of-life-on-april-25-2019-esm-available/)) and as such is now EOL (*end-of-life*)

Ubuntu 14.04 LTS had the option of extended support, ie. converting the release to Ubuntu 14.04 ESM which is supported by Canonical through Ubuntu Advantage ([https://ubuntu.com/blog/ubuntu-14-04-esm-support](https://ubuntu.com/blog/ubuntu-14-04-esm-support))

If you're using Ubuntu 14.04 LTS, given it's now been *out of life* for approaching a year, and hasn't had security updates for that long, I'd recommend you re-install or *release-upgrade* to a supported release of Ubuntu.
