### Post by grahammechanical on 2012-01-24
What particular hardware install problems did you have? Could you run the live CD? Could you install and then you found that some hardware was not supported?

For example, the Ubuntu download ISO does not come with proprietary video drivers. These need to be activated (installed) after Ubuntu is installed. Because of this some find that even getting the live CD to run is difficult. They need to make changes to the live CD boot options just to get Ubuntu installed. Then they can activate the video drivers.

This situation will remain with each release because the Ubuntu ISO does not include closed source drivers for ethical reasons and it would be impractical include every driver needed to work with every possible hardware configuration. The ISO image is a one size doesn't fit all sizes product. Likewise, it is not practical to produce ISO images for different types of hardware.
