### Post by markjensen on 2008-08-19
Is it possible, I don't know.   If the app is listed in winehq's database, it might be a good start to see if it will work.

Alternatively, I would recommending doing a test install of Ubuntu with wine installed.  Then run the .exe installer for your app, and cross your fingers!   If it is a well-behaved billing app, it should be quite simple, with a basic UI and some file read/writes.   Easy for wine to do.

If, for whatever reason, it does not work, another option would be to use that app in a virtual machine.  It will be a bit slower, but for data read/write, it is not going to be as relevant as a game (for example).
