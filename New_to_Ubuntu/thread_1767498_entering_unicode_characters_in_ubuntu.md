### Post by soylentcola on 2011-05-25
Simply put, how does one enter unicode characters in Ubuntu? I'm running Natty 11.04 and I have a numpad on my laptop and all that jazz, but the regular old ALT+ Unicode Character isn't working, is there some program I should install?

---

### Post by duke.tim on 2011-05-25
You need to type a U before the unicode code

ctrl+shift+U +code + space
