### Post by MichealH on 2010-02-19
Compiz is composition but when you cant run it (Like me) I use the metacity composition It is still basic but It still can be pleasing. You get some shadows and some transparency especially on metacity themes :D

ALT+F2

```
gconf-editor
```

Goto apps then metacity then click genral

Now tick "compositing_manager" the screen will go blank then come back on in some cases

You now have metacity compositing

This is meta city compositing+notification - WoW! :D
