### Post by shane_faulkinbury2 on 2015-11-05
This is how you install Chrome using Firefox!

1.  In Firefox search for Google Chrome and click download Chrome.

2.  Don't forget to chose either 32 or 64 bit!

4.  Open the Download folder and double click the .deb folder you just downloaded.

5.  This should open the Ubuntu Software Center.

6.  Just install and your good to go!
