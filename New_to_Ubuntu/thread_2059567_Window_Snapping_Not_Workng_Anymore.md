this seems to happen to me randomly.. is rather annoying.. 

anyway my keyboard shortcuts still work.. so its not a total pain.

to maximise press:

```
ctrl super upArrow
```


to snap left:

```
ctrl super leftArrow
```


and to snap right:

```
ctrl super rightArrow
```


***note** the super key is almost always the key with the windows logo on it.  (some keyboards don't have the windows logo)

hope it helps..

to see more keyboard shortcuts hold down the super key (windows logo key)..
