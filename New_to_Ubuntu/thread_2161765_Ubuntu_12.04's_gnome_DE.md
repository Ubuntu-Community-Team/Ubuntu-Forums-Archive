### Post by Jaraxle on 2013-07-11
Hi, Gnome Classic would run faster than Unity but it sounds like your machine could handle Unity so I think it's more a question of personal preference. Personally, I don't care for Unity so I completely removed it from Ubuntu 12.10 (before that 12.04) and use Gnome 2 Classic instead.

Do you have a Ubuntu Live CD? If so, you can boot with the CD and see how it performs. I would consider installing Ubuntu 12.04 on a seperate partition and dual-boot with Lubuntu 13.04 until you decide which one you prefer before you change the desktop enviornment too much. Just in case something messes up.
