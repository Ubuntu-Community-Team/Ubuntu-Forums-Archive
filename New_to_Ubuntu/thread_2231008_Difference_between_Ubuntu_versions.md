### Post by grahammechanical on 2014-06-22
In terms of code Ubuntu 13.10 is much closer to Ubuntu 14.04 than it is to 12.04. Each Ubuntu release is built upon the code of the previous release. There is a development period of 26 weeks. Ubuntu 13.10 has the Unity 7 user interface which is also in 14.04 and will not be upgraded though out the support period of 14.04. I have also noticed that 14.04 uses less memory than 12.04 and this improvement was also seen in 13.10. There are also upgrades to the default applications and to the Linux kernel.

You can compare the release notes. I have included the 14.04 release notes for good measure.

[https://wiki.ubuntu.com/PrecisePangolin/ReleaseNotes/UbuntuDesktop/UbuntuDesktop-12.04](https://wiki.ubuntu.com/PrecisePangolin/ReleaseNotes/UbuntuDesktop/UbuntuDesktop-12.04)

[https://wiki.ubuntu.com/SaucySalamander/ReleaseNotes](https://wiki.ubuntu.com/SaucySalamander/ReleaseNotes)

[https://wiki.ubuntu.com/TrustyTahr/ReleaseNotes](https://wiki.ubuntu.com/TrustyTahr/ReleaseNotes)

Each release represents work by the developers to keep up with the improvements made by the developers of projects that Ubuntu is built upon. And there is also an aspect of the Ubuntu developers moving forward on their aims for Ubuntu. To get an idea of where Ubuntu is being taken review some of the sessions from the recent Ubuntu Online Summit.

[http://summit.ubuntu.com/uos-1406/](http://summit.ubuntu.com/uos-1406/)
