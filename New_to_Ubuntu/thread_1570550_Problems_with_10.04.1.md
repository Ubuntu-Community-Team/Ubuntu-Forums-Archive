### Post by Phil Binner on 2010-09-08
A little while ago I had a working system using Ubuntu Lucid. I have been trying to set-up MythTV, so I set up a dual boot with MythBuntu. After some pain I got this working. When I was in Ubuntu the wired network worked fine, but in Mythbuntu it didn't. Same machine, just a dual boot.

At this point I decided I didn't want the dual boot, so I downloaded Ubuntu 10.04.1, intending to load MythTV inside it once it was going. Bad decision.

The new system, once loaded, would not connect to either wired or wireless networks. The wired network says it's connected, but it won't do anything. That's not the only problem though, USB support is unreliable, and the system feels under stress (i.e. delays when you're typing or moving the mouse; windows not opening smoothly). If it were a windows machine I'd think there was a virus grabbing all the resource.

I have tried to get out of this by downloading the live CD twice, and repeating the set-up numerous times, but no improvement. I even did an aggressive format on the hard drive using an old windows CD, because one of the symptoms of the set-up was that previous versions did not get completely removed. There were errors on the set-up that looked as if they may be disc errors, but the disc is fine in windows.

The last thing I did to this particular machine was to load an old version of XP. I have no intention of keeping this, but I wanted to see if it would connect to the internet. It did, immediately.

I have posted lots of information about the networking problems. If I repeat it I think it will make it difficult to answer, but if I were connected I could probably get Ubuntu to repair itself by getting all the latest upgrades. The information is here [http://ubuntuforums.org/showthread.php?p=9808568#post9808568](http://ubuntuforums.org/showthread.php?p=9808568#post9808568)

As an extra to this, I also have an old computer cobbled together from the remnants of old upgrades, which I sometimes use for experimentation. I did all the same things on this machine, but although it runs from the live CD it just hangs half way through the set-up if I try to get it onto the hard drive (500gb - one of the newer bits). When it is running on the CD it does connect to the wired network, but not the wireless network. It is also clunky to run, and the mouse and keyboard connections kept stopping when they were in the USB slots.

I have just loaded last year's Kubunto onto the old box of bits, and it went on fine. It's connected to the wired network.

The upshot of this is, does anyone know if the 10.04.1 download actually works, because I don't see how this can just be me. The 2 versions I downloaded came from the same site, but via different windows machines (XP and 7).
