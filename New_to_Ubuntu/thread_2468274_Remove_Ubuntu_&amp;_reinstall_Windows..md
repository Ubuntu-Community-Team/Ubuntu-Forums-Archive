### Post by TheFu on 2021-10-23
Have you considered loading Windows into a virtual machine?  This way you keep Linux, but can run Windows for those 1-2 programs where it is still needed.
There are some caveats with using VMs, but most of those go away on any Core2 Duo CPUs with 6G of RAM or more from 2008 or later.  This assumes you aren't planning to do heavy graphics or video editing or gaming, but for nearly any other purpose, using a VM is fine.  I know a guy that runs AutoCAD in a VM on his travel laptop, for example.

But there are good reasons to run Windows directly on hardware too, sometimes.
