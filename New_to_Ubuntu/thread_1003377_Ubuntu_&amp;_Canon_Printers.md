im sure if you spent some time with cups youcould find a generic driver

or you can buy linux turbo print, i did, and it works well my my i865

[http://www.turboprint.info/](http://www.turboprint.info/)


the other way around it would be, and only in theory, i have not tried, but it should work

if you connect your printer to your pc via usb and do not set it up in linux and install virtual box and install windows xp with the drivers for the canon printer

virtual box will not allow usb devices to be shared at the same time between linux and xp at the same time

so if you set it up in xp ou can have the head clean function as well, you dont get this with the cups drivers, but do with turbo print

you can set it up in xp and share the printer and then point cups back it that driver in xp shared with a generic driver and print your documents away till the cows come home

but if you want to print, xp will have to be running in virtualbox everytime or no print jobs will happen
