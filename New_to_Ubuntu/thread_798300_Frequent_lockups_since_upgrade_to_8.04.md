### Post by brujoand on 2008-05-18
The thing is, that the upgrade is awfull. Has _never_ worked for me.
So i think you should just do a clean install instead. 
  But a tip, is to create two partitions, on "archive" for your music, video etc.. using f.x ext3 since it handles large files well. 
  And a boot partition using reiserfs since it handles small files really fast(read: os files) this way your system is more responsive and you can do a clean install every now and then without having to backup stuff. 
Best solution i can think of anyways :)
