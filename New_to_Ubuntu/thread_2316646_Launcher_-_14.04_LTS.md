### Post by grahammechanical on 2016-03-09
Not in Ubuntu. Not with the Unity user interface. It is all part of a cunning plan. On Ubuntu phone the launcher is on the left and stuff happens when we swipe from the bottom and right screen edges. It is the same for the Ubuntu tablet.

And when convergence finally comes to Ubuntu desktop and we connect to a touch screen monitor we will find that we can use keyboard, mouse, touch & swipe gestures and stuff will happen when we swipe from the bottom and right screen edges.

The launcher is fixed to the left screen side by design. A key selling point of convergence is familiar user interface across different device form factors.

There are flavours of Ubuntu where we can install a different kind of launcher called a dock and that can be set to the other screen edges.
