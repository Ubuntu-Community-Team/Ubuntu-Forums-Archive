### Post by mistypotato on 2009-02-11
[SIZE="5"]OMG !!![/SIZE]  :guitar:

I was expecting a difficult, hard to use, unreliable piece of an effort....

I was WRONG !

I got this VirtualBox installed, configured and running as if I knew what I was doing in minutes.

It performs INSIDE ubuntu EXACTLY as though I was on a separate PC with WinXP installed which blows me away.

It recognizes and allows me to run USB devices (even legacy devices) with utter simplicity.  I needed to use an old Visioneer scanner and it worked perfectly!  Even ran the Scansoft image manipulation software with NO problem (!!!!!!!!!!!!!).

In a nutshell, if you have to do something Windows related in Linux, look no further than VirtualBox.  (I have no connection with them whatsoever except I am highly impressed).

How much is a license and where do I pay?  This thing is worth it.
