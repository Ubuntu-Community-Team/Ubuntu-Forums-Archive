### Post by NM5TF on 2018-05-21
@swampwiz....

to clarify wildmanne39s answer.....since you are posting on the Ubuntu forums we can assume that you are using Ubuntu...
Ubuntu is based on Debian Linux, so downloads will be ".deb" files.....same holds true for Mint Linux, etc....

RPM stands for Red Hat Package Manager and is used for distros based on Red Hat Linux such as Fedora.....downloads will be 
".rpm" files......

other major distros may have their own unique package managers.......Arch Linux uses "pacman".....

welcome to Linux....once you get comfortable with it you will never look back....

### Post by yancek on 2018-05-21
Unlike windows which requires you to go online and search for different software which doesn't come with a basic install, Linux systems each have their own repositories on their own servers and package managers to download and install within the system.  You should not need to go online and search for software as you do in windows.for software which is not included in a base install.  There are some situations in which, if you want the latest software, it may not be available from repositories as the emphasis is on stability not bleeding edge untested software.
