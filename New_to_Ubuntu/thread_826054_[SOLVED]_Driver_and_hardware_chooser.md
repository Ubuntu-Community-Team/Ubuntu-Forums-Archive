### Post by twodogsdad on 2008-06-11
When I was trying to get my screen resolution up I followed the procedure of editing the xorg.conf file. When I rebooted my laptop I got an error that said Ubuntu was going to run in low graphics mode. A window then popped up that allowed me to configure the display by choosing a driver from a list under a "drivers" tab and my hardware from a list under a "hardware" tab. After running this configuration software and rebooting my computer (power down and up, not Ctl-Alt-Bksp) my display works great. 

My question. Does anyone know what this app was and how to run it without messing up the graphics mode by editing xorg.conf?

Thanks.:popcorn:

### Post by sdennie on 2008-06-11
You should be able to set the resolution with System->Preferences->Screen Resolution.

---

### Post by avtolle on 2008-06-11
The menu name is Screens and Graphics. Install by editing the menus under Applications, select Other, and from the list in the right box, click on Screens and Graphics to have it in a menu.
