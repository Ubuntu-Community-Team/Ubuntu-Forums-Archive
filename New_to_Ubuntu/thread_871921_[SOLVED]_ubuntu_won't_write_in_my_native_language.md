### Post by UbuntuNerd on 2008-07-27
To install additional languages in Ubuntu , there were special, easy language installer program given in Ubuntu Linux version 6. To start Language Installer program, go to Ubuntu Gnome desktop menu bar, then point to System > Administration > Language Support. A small language will appear where available supported languages were listed. Select the languages you want to install and configure in your Ubuntu, and click Apply button. Before hitting OK button, ensure that your Internet connection is up and running.

### Post by Rolcol on 2008-07-28
> **eightmillion said:**
> Have you tried running gnome-keyboard-properties and changing the layout to Israel?

That is found in System > Preferences > Keyboard.

To change to your language go to the Layout tab and click add in the middle of the screen.  Next, look for you area in the layouts drop-down box and change the Variant to suit your needs.  Click Add and check the box next to default and close out.

### Post by nir2000 on 2008-07-28
Thank you everybody problem is SOLVED thanks for your help.
The complete solution is as follows: system>preferences>keyboard and then selecting the layouts tab >add button selecting the desired language after that pressing the layout options button and looking out for layout switching choosing the "right Alt key changes layout" and that's it. when you want to toggle between languages pressing on right Alt key
