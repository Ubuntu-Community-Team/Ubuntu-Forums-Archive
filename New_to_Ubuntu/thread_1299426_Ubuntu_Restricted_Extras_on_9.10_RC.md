### Post by SteelCore on 2009-10-23
I just made a fresh install of 9.10 RC.  I normally use Add/Remove to install the much needed Ubuntu Restricted Extras but it has been completely removed. The new 'Ubuntu Software Center' can find the Restricted Extras but there is no button to install it. How can I get it? Thanks
ps I'm having the same problem with Virtualbox

---

### Post by Ewingo401 on 2009-10-23
Just open a terminal from the applications > accessories then type in "sudo apt-get install ubuntu-restricted-extras" (without the quotes).  That should do it for you.

---

### Post by SteelCore on 2009-10-23
Thanks for the help. The problem actually got solved after I updated via System>Administration>Update Manager
