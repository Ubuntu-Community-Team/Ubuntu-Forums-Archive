### Post by arpanaut on 2013-03-16
In 12.10 the command to reset unity is 
```
setsid unity
```

If that does not work hit the top icon in the launcher and type "files", click on the icon that comes up.
then go to the launcher and right click on the files icon, select "lock to launcher", then you can move the icon back to where it is supposed to be under the white icon at the top of the launcher.

You are plenty smart enough, it just takes some time to get used to things in a new OS,
Hang In There!
