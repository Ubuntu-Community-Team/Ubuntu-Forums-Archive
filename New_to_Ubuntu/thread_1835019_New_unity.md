The problem is that Unity in 10.10, 11.04 and 11.10 is/are three different things. The Unity in Maverick was based on mutter, whereas in 11.04 and 11.10 it is based on compiz. The Maverick version of Unity lacked a lot of the functionality that you find in 11.04's version. The 11.04 Unity runs on gnome2, whereas Unity 11.10 runs on gnome3 - very different foundations.

It is possible to install gnome3 shell in Natty through the use of a ppa, so I guess in theory you could install the gnome3 Unity of 11.10 in 11.04, but I haven't heard of anyone doing it. Both 11.10 and its version of Unity are still (just) in alpha so there is a lot of bug-fixing yet to do.

I doubt whether it would be straightforward. Better to simply upgrade to Oneiric when it is released.
