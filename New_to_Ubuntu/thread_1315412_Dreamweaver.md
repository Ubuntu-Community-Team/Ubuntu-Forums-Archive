### Post by Swerve1000 on 2009-11-05
You may want to take a look at Virtualisation which will allow you to run an instance of Windows within your Ubuntu setup.

You would then be able to run Dreamweaver, Photoshop and other windows only type programs.

A free virtualisation program is called VirtualBox.

I'm unsure whether DWeaver will run in Wine, which is a less complex solution.
