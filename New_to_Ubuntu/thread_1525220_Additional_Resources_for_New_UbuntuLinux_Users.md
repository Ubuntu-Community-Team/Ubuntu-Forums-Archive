### Post by kb3mf on 2010-07-06
I'm a new Ubuntu user.  I recently install 8.04 on an old Presario 2200 laptop.  In an effort to understand and fix my ERROR: Firmware file "b43/ucode5.fw" not found or load failed message I searched the ubuntu forums and after remembering that the local public library system had remote access to Safari Tech Books did a search there.  Wow!!  What a resource!!

Titles include "Ubuntu Linux Bible: (various flavors)", "Ubuntu Linux Secrets",  "Ubuntu for Non-Geeks, (various editions)",  "The official Ubuntu Book" and many, many more.  The content can be viewed and copied for future use and reference.  There seems to be everything that one needs to know to effectively use this operating system.

I solved my problem by following the instructions at "http://www.omattos.com/node/6" referenced in a forum response.  I have an understanding of what was going on from some of the references above.

I'm very please with Ubuntu.  Thanks for all the work!!

Check out your local library system.  You too may have access to some great resources!!
