### Post by MagicalMelon on 2011-01-27
You can install it under wine (I'm using 1.2.2) However, you will be missing MSVBVM60.dll which you can download right here: 
[http://www.dll-files.com/dllindex/dll-files.shtml?msvbvm60](http://www.dll-files.com/dllindex/dll-files.shtml?msvbvm60)

installing is as easy as dropping it in your ~/.wine/drive_c/windows/system32/ folder

Now, this gets the game up and running, however there may be other issues that occur, as I just figured this out myself. If you launch the program through the terminal you should be able to see what specifically causes any errors to give you a place to start. Hope this works for you as well as it does for me!
