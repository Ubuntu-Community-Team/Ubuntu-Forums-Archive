### Post by urukrama on 2008-05-30
Openbox is light and flexible. It is a minimalistic window manager and does little more than manage windows (no taskbar, no desktop icons), but you can use other applications to do all those other things. It doesn't come with status indicators, like icewm does, but you could use a panel that has such things in Openbox. 

If you decide to walk the Openbox path, have a look at the guide in my signature for help installing it and configuring it. You could also try LXDE, which uses Openbox as a window manager. There are a few threads about how to install LXDE on these forums.

About your original question: are you looking for something to manage your desktop (wallpaper and icons)? If so, try idesk (for icons) and Feh (for the wallpaper). You could also use a file manager that draws the desktop, like PCManFM (light) or Nautilus (heavier).
