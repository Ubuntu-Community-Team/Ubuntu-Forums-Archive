### Post by eltama on 2009-11-19
Transmission is so simple that there is not much to explain.
You first have to get the torrent file of what you want to download. There are several sites to search for torrents (of mostly illegal content) like [http://www.mininova.org/]("http://www.mininova.org/").

Suppose that you want to download Ubuntu 9.10. You first have to get the .torrent of the particular version you want from [http://releases.ubuntu.com/karmic/]("http://releases.ubuntu.com/karmic/"). Once you have the .torrent file just double click it and it will open Transmission. Transmission will ask you for the destination folder, select the one you want and click add. That's it! The download will begin and you can see a progress bar, the speed at which you are downloading and uploading, and the ratio.

There are several guides online just, google bittorrent help. I advise you to investigate a bit about the consequences of downloading illegal material in your country. For instance, in some European countries downloading is not illegal, but uploading is. In USA both are illegal.
