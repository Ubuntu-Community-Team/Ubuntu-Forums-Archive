### Post by Mark Phelps on 2011-08-23
Also, you need to be aware that recent PC Games use DirectX 10.  If your video drivers don't support that, even though you may be able to install and run the game, you won't get decent frame rates to play it.  The playback will be choppy and all but unusable.

Frankly speaking, Ubuntu is NOT a good platform for playing recent Windows PC games.

### Post by 3rdalbum on 2011-08-23
> **channman40 said:**
> I'm building a pc and have heard great things about this system and wanted to try it out. My main concern is if I will be able to run games made for Windows operating system?,

Some will work using some free software called Wine, most won't work at all. If you want to play the latest PC games, you need to have Windows installed as well as Ubuntu, as a dual-boot. This means that, when you start up, you have the choice to boot into Windows or Ubuntu. If you're in Ubuntu and you want to play some PC games, you'd need to reboot and choose Windows.

"PC games" are actually Windows games. Software is not transferable between operating systems, which is why you can't play Xbox 360 games on the PS3 despite them having compatible processors :-)
