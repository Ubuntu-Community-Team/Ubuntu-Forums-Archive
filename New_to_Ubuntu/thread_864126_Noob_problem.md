### Post by billgoldberg on 2008-07-20
With that kind of ram I wouldn't even use xubuntu.

I would suggest [puppy]("http://www.puppylinux.org/") or [dsl]("http://www.damnsmalllinux.org/").

That way your pc would be lightening fast.

But those distro's aren't exactly the most user friendly.

What you could try and would only take 1 minute is installing fluxbox. I'm using it right now. It will be MUCH faster than kubuntu.

In a terminal (konsole) type:

sudo apt-get install fluxbox.

Then log out (or do the old ctrl+alt+backspace) and select "fluxbox".

It will be a lot faster.

You start programs by right-clicking the desktop.

If you decide to use it, these websites might come in handy:

[https://help.ubuntu.com/community/Fluxbox](https://help.ubuntu.com/community/Fluxbox)

[http://customize.org/fluxbox](http://customize.org/fluxbox)


--

512mb or 1gb ram shouldn't cost you more than $30, &#8364;20 and you will really feel the difference.

--

Also, you might want to ditch firefox and use a lightweight browser like [Kazehakase]("apt://kazehakase")
