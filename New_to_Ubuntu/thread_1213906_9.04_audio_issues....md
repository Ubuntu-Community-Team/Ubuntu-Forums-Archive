### Post by swappa on 2009-07-15
I am afraid I don't have a clear answer or fix for you. But, I have spent a lot of time trying to make my sound settingswork (basically, sound in Linux is a mess...) and have found that while trying different stuff, you enable/disable/turn off/turn on a lot of other stuff as well. So it could be a muted check box somewhere or maybe a slider set to the lowest level someplace else. Have you tried removing Pulseaudio?

This did the trick for me; 

* killall pulseaudio
* sudo apt-get remove pulseaudio
* sudo apt-get install esound
* sudo rm /etc/X11/Xsession.d/70pulseaudi

### Post by Malta paul on 2009-07-16
To install VLC which has lots of useful codec.
Just go to Applications>Add/remove> then ensure that 'show' is set to 'all available applications' then Search VLC when it shows up, tick the box and then press 'apply changes' 
The program will now install and you will find it in Applications>Sound & video. 
Give it a try! :)
