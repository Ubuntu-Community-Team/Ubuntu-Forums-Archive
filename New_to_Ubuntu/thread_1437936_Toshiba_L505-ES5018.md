### Post by QIII on 2010-03-24
One caveat off the top of my head:

Toshiba defers fan control to the OS, rather than the BIOS.

That works fine if you run Windows, since Toshiba is pretty sure that you will be in 95%+ of cases.

Fan control with Toshibas has been a problem for some users on the forum.  But I believe some users have found software to deal with the problem.
