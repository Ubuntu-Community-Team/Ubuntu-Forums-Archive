### Post by snowpine on 2009-07-06
Wubi is for Ubuntu only.

If you want to try other linux distros (fedora, debian, etc) without changing your existing Windows install, you have two options:

1. Burn a Live CD and use that (note however your changes won't be saved when you power down)
2. Try a "virtualization" application like Virtualbox to run a linux guest session within your windows host.
