### Post by sanderd17 on 2009-08-21
I suppose the netbook remix is the most appropriate for you. The best would be to install it from an usb-stick. You can download the image from [here]("http://www.ubuntu.com/GetUbuntu/download-netbook"). While you're dowloading you can read the instructions you find at the download page. I would recommand to prepare the usb-stick with a pc and not on you netbook. If you find problems, please post them here.
You can read about the preparation of your usb-stick [here]("https://help.ubuntu.com/community/Installation/FromImgFiles").
To install ubuntu you should go in your BIOS (press F1, F2, Esc or F12 depending from computer to computer) and there select to boot from usb.

You don't have enough RAM to run Ubuntu. It requires 256 megabytes of RAM to run, and you'd only be able to install it through the "Alternate CD" installer.

Puppy Linux will run on that machine, but I don't think any *buntu will. You could also try Vector Linux or Zenwalk, or upgrade your RAM.
