### Post by meltindar on 2008-05-15
I'm also using alsa and had a similar problem after upgrade. I checked the sound configuration (System > Preferences > Sound) and all sound settings had changed from "USB Audio" to "USB Audio (Not Supported)" (I was using headphones ATM). So I just changed them back to previous value and it started working. If you didn't already checked the sound preferences, try that.
HTH
