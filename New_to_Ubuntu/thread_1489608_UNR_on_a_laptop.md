### Post by Paddy Landau on 2010-05-21
Previous versions of UNR were optimised for the chipsets used inside netbooks, so you would have been worse off installing onto a normal laptop.

However, since UNR 10.04, the only difference (I'm told) is the desktop.

As Ubuntu 10.04 is LTS (long-term support), and UNR 10.04 is not, I recommend that you install Ubuntu 10.04.

There was a lot of confusion on the subject of optimization of the UNR, probably because the hardware requirements paragraph used to erroneously state Intel Atom as one. In fact, UNR was always the standard i386. There used to be an experimental port of lpia architecture, but never was it the default choice.
UNR has always been just a customized interface, optimized for small netbook screens.
As for LTS, I really don't get it. Given UNR's components are essentially the same as Ubuntu-Desktop's, how can UNR not be an LTS?
