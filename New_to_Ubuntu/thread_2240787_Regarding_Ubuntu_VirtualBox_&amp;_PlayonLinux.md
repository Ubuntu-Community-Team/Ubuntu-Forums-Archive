### Post by QIII on 2014-08-22
Hello!

If by "original Windows" you mean a licensed copy that is not installed on any other machine, then yes.

For instance, it would be a violation of Microsoft's EULA to use the same disk with the same product code to install Windows on both a physical machine and in a virtual machine unless the product code is a multi-seat one.  That is unlikely with a retail version.

If you do not have Windows installed with that product code anywhere else, you are fine. 

Your second question is a different matter entirely.  Please create a new post for it in the Wine subforum of Gaming & Leisure.
