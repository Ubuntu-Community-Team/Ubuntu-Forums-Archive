### Post by hutsky on 2011-10-12
Thanks for the clue.  You can disregard my question since something I had read earlier came to me and applying it solved the problem.  It is a strange procedure but here are the steps:

-  Logout
-  Right click the user box to login
-  Select the Ubuntu button at the very bottom of the screen
-  Select "Classic"
-  Return and click the User's name to login

The Classic view returned.
