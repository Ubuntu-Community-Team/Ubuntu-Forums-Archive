### Post by Frogs Hair on 2011-03-09
If it is a launcher and not an applet , open dock preferences > task manager , select the item to be removed and select remove from the controls .

If it is an applet , click on the item to be removed from the list below the applets menu and select the up arrow.

Another way to remove launchers is to right click the dock icon and select remove launcher.
