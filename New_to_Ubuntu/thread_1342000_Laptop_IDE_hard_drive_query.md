### Post by mikechant on 2009-11-30
> Any 2.5" would do... 

I did read a article a while ago which said that some laptops were very tight on space so some drives would fit and others wouldn't (slightly too 'fat')! So if the price is OK you might want to get a higher capacity version of the same brand as the existing drive - the case dimensions are more likely to be the same. Or make sure the supplier will take it back if it doesn't fit - local shop which sells bare (unboxed) drives might be a more likely to do this.
