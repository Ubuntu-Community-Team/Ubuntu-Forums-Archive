### Post by FAMUgolfer on 2009-02-25
I have a dual-boot acer aspire 4530 laptop with nvidia 9100m graphics and AMD athlon processor.  Ive tried to tinker with this card (nvidia-settings manager) to try and control the thermal temperatures.  Currently it runs at 70C constantly, causing the fan to never shut off.  This happens in XP and Intrepid.  I believe this is a cpu frequency problem since this is happening in both operating systems.  I've been able to set the cpu to "ondemand" which is 1000Mhz and the only other setting is the max at 1900Mhz, which has brought down the temps to 55-65C.  The fan does shut off now, only when no programs are running and the comp is at idle, but shortly thereafter the thermal temperatures in nvidia-settings quickly increases and thus the fan comes on at full speed.

My question is, is there a way to have BIOS control the cpu frequency since this is occurring in both XP and 8.10?  If so, can someone show me with noob directions?
