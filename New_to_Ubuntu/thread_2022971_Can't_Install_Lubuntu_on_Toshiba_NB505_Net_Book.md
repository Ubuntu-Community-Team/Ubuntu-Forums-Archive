### Post by Rex Bouwense on 2012-07-11
Ah yes.  First check the md5sum on the download.  If you need assistance, see here:
[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

The hash or md5sums are here:
[https://help.ubuntu.com/community/UbuntuHashes#A12.04_LTS](https://help.ubuntu.com/community/UbuntuHashes#A12.04_LTS)

If you have a good  download, then you can proceed to burning the ISO to the flash drive.  Instructions are here.
[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)

You can use this method with Lubuntu or Ubuntu.  Enjoy.
