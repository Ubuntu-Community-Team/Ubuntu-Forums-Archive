### Post by grahammechanical on 2012-10-28
First of all it is better to install Ubuntu with all the hardware connected than to install Ubuntu and then plug in hardware. During the install process Ubuntu detects the hardware and installs modules appropriate to the hardware.

Second, have you checked the Libreoffice printer settings to see if it is using the correct one for your printer?

I am also assuming that you have tried to set up the printer using the Printer setup utility in System Settings.

Regards.
