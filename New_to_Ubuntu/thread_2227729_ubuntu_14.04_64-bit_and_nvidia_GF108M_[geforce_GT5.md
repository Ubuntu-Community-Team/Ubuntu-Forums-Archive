### Post by Vladlenin5000 on 2014-06-03
Hi, welcome.

There's no need to install the nvidia drivers from their website unless you have a brand new card better supported by newer drivers. Otherwise just go to System Settings > Software & Updates, open the last tab (additional drivers) and select and install the recommend driver version.
