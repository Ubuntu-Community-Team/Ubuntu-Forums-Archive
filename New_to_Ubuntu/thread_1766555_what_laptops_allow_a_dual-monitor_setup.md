### Post by audiomick on 2011-05-24
Should just work. I even had the experience that my netbook (Lenovo Ideapad) would only do a clone (same thing on both monitors) in Win7 starter, but would do an extended desktop in Ubuntu 10.04 . It has Intel Graphics, which are reckoned to be one of the lesser options for Linux. My Laptop has nVidia. It doesn't have a problem with two monitors, and I don't think anything half way up-to-date will.

If you are using the nVidia proprietary graphics drivers, the extended desktop thing is called Twin View, an nVidia proprietary solution that does pretty much the same thing as the Xinerama that Sandy mentioned.
