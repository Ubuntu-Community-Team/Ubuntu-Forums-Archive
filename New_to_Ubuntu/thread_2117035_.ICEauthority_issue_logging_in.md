### Post by schragge on 2013-02-17
First, you most probably still can log in on a text console. Press **Ctrl**+**Alt**+**F1** and log in as normal user (*jonny*).
Then try
```

sudo chown $USER:$USER ~/.ICEauthority
sudo chmod 600 ~/.ICEauthority

```
If that doesn't help, try
```

sudo chown -R $USER:$USER ~

```
You'll probably get a message that you cannot change the ownership of one folder (*.gvfs*). Ignore it.

[highlight]Edit.[/highlight]
To change back to the graphical login screen, press **Ctrl**+**Alt**+**F7**. To log out from the text console, press **Ctrl**+**D**.
