Two questions regarding Banshee:

1.  I downloaded the Banshee 1.6 .ppa and added it to my source list and have it up and running, but it does not have the "banshee" icon ... Is there a simple way to get this added? 

2.  I tried the version in the repositories and did not like it as much, but is there a way in linux that if you have an older version that you download from the repositories that you can update it?  (e.g. If I get the 1.4 version from the repositories, does "sudo apt-get update" work to upgrade the version from 1.4 to 1.6?)  Or does the version stay as 1.4 until the repositories update to 1.6 and then it would update?

### Post by Jon Monreal on 2010-04-24
1. Not really an easy way. If you want an icon on your desktop (and you're using Gnome), right-click on the desktop and select "Create Launcher...". You can then enter the name you want the icon to have, and select the icon image by clicking on the image to the left. Under command you can put "banshee" minus the quotes.

If you want it under the main menu, right-click on the menu and select "Edit Menu". On the left side, you will see the various menus available, select "Sound & Video". Then, click "New Item" on the right side. Then just follow the instructions above.

2. You cannot update to a version newer than the ones in the repositories unless you're getting the software from a .ppa or another source. When the repositories update to 1.6, you can update to 1.6.
