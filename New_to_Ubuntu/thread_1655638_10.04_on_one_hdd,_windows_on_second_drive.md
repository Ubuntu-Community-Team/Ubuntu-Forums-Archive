### Post by Quackers on 2010-12-29
If Ubuntu is on the first hard drive, install Windows on the second. It's actually less problemsome to have them the other way around, if possible.
Then make the Ubuntu drive the default drive in bios and when Ubuntu is booted up run sudo update-grub in the terminal and it should pick up the XP installation. An entry for XP will appear in a shiny new Grub menu :-)
