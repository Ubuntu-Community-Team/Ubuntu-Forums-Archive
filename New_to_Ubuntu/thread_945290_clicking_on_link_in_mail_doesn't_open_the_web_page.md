### Post by hellion0 on 2008-10-12
Make sure you have a default browser set in the first place. Just because Firefox is installed doesn't always mean it's the default, and by the behavior you're seeing, another browser like Konqueror isn't set default in its place.

Within Firefox, you can check by navigating to Edit -> Preferences. Click the "Advanced" tab, make sure the "General" tab there is highlighted, then you should see a button under "System Defaults" that says "Check Now" - click that button.

If that doesn't work, let us know.
