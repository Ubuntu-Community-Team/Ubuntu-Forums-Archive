### Post by bettyhills on 2008-12-18
Please tell me how to add a quick link to the left menu on the PLACES page.  

Right now it shows at the bottom of the left menu:  Documents, Music, Pictures, Video.  

I wanted to add a link there to a new custom directory called Personal.

Thanks.

---

### Post by jgoguen on 2008-12-18
Just go to the folder in Nautilus, then in the menus choose **Bookmarks -> Add Bookmark**.  It will then be on the left side of Nautilus and in the Places menu.

---

### Post by Paqman on 2008-12-18
Open up the file browser and click on Bookmarks, you should be able to add a bookmark pointing to your folder.

---

### Post by kanikilu on 2008-12-18
Go to Places and open one of the links already there (Computer, for example). Then in the file browser, go to the folder that you want to bookmark, and then go to the Bookmarks menu and choose Add bookmark (or press CTRL+D).
