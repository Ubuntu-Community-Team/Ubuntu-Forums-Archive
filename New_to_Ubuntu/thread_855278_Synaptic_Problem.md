### Post by jrasmussen0 on 2008-07-10
Ubuntu would not put anything directly illegal on their servers.  The multiverse repository, which Tuxguitar is part of, is software that doesn't meet strict Debian and/or Ubuntu licensing standards.

Some open source software might be listed here if they use technologies with patent issue (Windows Media Codecs).  The software might be free but in some countries people could be sued because of patent infringement.  Although, to my knowledge, nobody has been caught.

Another issue is with copyrighted or trademarked images.  There was a big stink about a year ago when Firefox's license had language in it that alluded to a possible copyright issue when using Firefox logos.  There was talk about renaming and forking the project.

> ["multiverse" component]("http://www.ubuntu.com/community/ubuntustory/components")

The "multiverse" component contains software that is "not free", which means the licensing requirements of this software do not meet the Ubuntu "main" Component Licence Policy.

The onus is on you to verify your rights to use this software and comply with the licensing terms of the copyright holder.

This software is not supported and usually cannot be fixed or updated. Use it at your own risk.
