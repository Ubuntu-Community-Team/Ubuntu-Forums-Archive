### Post by kai60 on 2008-04-26
It depends what you are trying to achieve. If you want two different versions of Ubuntu, let's say 8.04 and 7.10 at the same time and to be able to access them both from the same desktop, as running versions, I would recommend a Virtual Machine such as vmware.

If you are trying to run both of separate hard drives at the same time I am unaware of any way to do this. You can however access both systems and make changes to the one that you are not working in, just going through nautilus.

On the virtual machine issue, you could install a virtual disk image on your usb drive and as long as the computer where you intend to run the other os has a virtual machine that can read it, you can use it.

Hope that helps.
