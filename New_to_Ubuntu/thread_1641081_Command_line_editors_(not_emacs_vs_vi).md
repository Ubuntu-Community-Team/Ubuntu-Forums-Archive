### Post by bodhi.zazen on 2010-12-08
nano is a nice "basic" editor, but honestly, vim is superior in many ways.

The "trick" is to use and learn vim. Add some themes, color, what not.

google search vimrc for some very nice default settings.

Here is mine :

[http://bodhizazen.net/adblock/vimrc](http://bodhizazen.net/adblock/vimrc)

It is a bit "basic" but it works well for general editing. If you have a specific task you may want to alter or add customizations or key bindings.

If you want to do something, such as copy-paste google search

vim copy paste.

Also vim comes with a number of graphical interfaces, I like cream (it is in the repos) more then gvim.
