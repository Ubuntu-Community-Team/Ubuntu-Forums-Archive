### Post by Mark Phelps on 2012-02-09
There is something known as a "cold boot attack" which allows the viewing of encryption keys from memory shortly after a machine has been shut down -- but that won't likely do you any good.

So basically, the answer is NO -- if it was Yes, then Bitlocker wouldn't be of any use, would it?
