### Post by Kellemora on 2008-11-25
Hi Doug

I have 4 computers running through a KVM switch.
Handiest thing since the Shirt Pocket, hi hi.........

My KVM emulates a USB mouse and keyboard, but I'm a using PS2 keyboard and mouse with no problems.

Some KVM's don't have PS2 support, only USB and vice versa.
So you might check your manual to see which they are compatible with.

And I know this will sound dumb, but I've made worse mistakes!
Do you have them plugged into the RIGHT sockets on your KVM?
You're NOT using an adapter for a keyboard for the mouse or vice versa.
It's all these little things that drive me nuts sometimes!

### Post by mapes12 on 2008-11-26
...........and before you boot up each machine make sure the KVM switch is toggled to the machine you are booting from otherwise your system will not detect the input devices at boot. Once one machine has booted then toggle the KVM to the other machine and boot that one. Once both machines are up and running you should then be able to toggle between the two at will.

I use a KVM switch between my two boxes as well.

EDIT: Also, try the T40 without the docking station to see if that works.
