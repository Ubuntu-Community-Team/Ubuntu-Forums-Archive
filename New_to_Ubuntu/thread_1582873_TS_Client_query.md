### Post by fly-by-night on 2010-09-27
It provide a shortcut for Terminal Server Client that is on the menu under Internet.

It's not important and can be deleted.  

Bug report:
[https://bugs.launchpad.net/ubuntu/+source/tsclient/+bug/399436](https://bugs.launchpad.net/ubuntu/+source/tsclient/+bug/399436)

You can install a newer version of tsclient if you need it on the panel (link on the bug report page).
