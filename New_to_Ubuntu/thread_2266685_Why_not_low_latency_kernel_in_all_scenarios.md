### Post by tgalati4 on 2015-08-23
Windows tends to be a high-latency operating system.  You always have instant control over the mouse and clicking on things.  The Windows system can be slow to react to those clicks and at times, appears to be "locked up".  With low-latency, you have more CPU cycles devoted to computing and less to interrupts such as the mouse and keyboard.  So you would have a laggy mouse and keyboard, but it would crunch through tasks pretty quickly.

If you are running UbuntuStudio, open up LibreOffice and create a document and type really fast (90 words per minute) and see if the screen keeps up with your typing.
