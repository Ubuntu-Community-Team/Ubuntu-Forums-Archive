That was an awfully long and difficult path you
just took. Well done for persevering. I'm
seriously impressed.

PM'ing me is fine. No probs.

Kino can do this easily.

Here's a quick 'How to'

Check out the images I've sent with this e-mail.

Open Kino and click on the "FX" tab on the right
hand side of the window.

Your screen should now look like image 1.

Now, check out image 2.

Then click on the tap "Create" on the left hand
side of the screen and below that tab select "From
File".

Click on the file icon and select the still image
you wish to use. Then select the number of frames
you wish to use

Please note that at the bottom left of your screen
there is an option to change 'Time' from 'Frames'
to 'Seconds' etc. I generally use Frames, but you
can change it whenever you wish.

So, choose the image file (as stated above) and
then just above that you'll see (in image 2) you
can choose the number of Frames or seconds etc you
wish to make the still image last for. There are
25 frames per second (fps) in PAL (and 29 in NTSC,
I think)

Make your selection and then hit 'Render' on the
bottom right hand side. Done!

You can add audio by using the selections on the
same page at the top right. Select the Tab 'Audio
Filter' and off you go. You'd want 'Dub' to add a
new sound.

Hope this helps.

Get back to me if you're still stuck.

Oh, I nearly forgot. To export, select the
'Export' icon Tab at the bottom right hand side of
window. Then choose the place you want to export
the video to (e.g.
/home/<your/name>/Desktop/my_new_video) and the
type of video format you wish to export to.

Let me know how you go.

### Post by ikisham on 2009-02-28
To install Kino go to Applications>Add/Remove...then search Kino, tick it and install. It'll appear in Applications>Sound & Video.
Then go to that link I posted, to the thread number(#)12. Try to follow it. It is well explained, it has some screenshots and if you don't see the options that the guy says fiddle here and there in Kino and you'll find. 
Basically you must choose a picture and then ask Kino to render a movie from it (for example, if you want it to be a 60s movie, at 25 frames per second (FPS), it must be 60*25=1500 frames). So Kino will make this 60s movie. When it is done you must choose it and add a soundtrack to it (that will be the music you want to upload so you will have to create the movie not with 60s but with the duration of the music), and thats explained in the how-to.
If you understand, then you can see that just as you can make a movie with one still pic (say, 60s), you could also make it a slideshow with several pics (one pic 10s + other pic 30s + other pic 20s, for example). Just you have to fiddle around the options in Kino and see how to do it, but I guess it's simple. I did only that time so I can't tell you, do this and do that but it is really straightforward.
If you decide to try and if you can't really make it work, then ask and I'll try it here too.
