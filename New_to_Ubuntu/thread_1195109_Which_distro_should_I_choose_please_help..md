### Post by cptr13 on 2009-06-23
You have to work reverse on this and start with the most limiting factors first.

-No internet: This means you need either the right software preinstalled or better yet, the entire repository available on disc to install from. Does he have a cd-rom or dvd-rom?  The Debian repository is pretty big but worth it if no internet available.
-preinstalled codecs: I know linux mint has these and PCLinuxOS dooes as well.  I'm not sure of others off the top of my head though.  I know there's more.  Go through distrowatch.com if you want to see which others there are.  
-look and feel: you can adjust that on any distro.  Download the stuff before hand and take it with you
