### Post by BT1 on 2010-05-08
> **carlee said:**
> install ubuntu-restricted-extras or flashplugin-installer

To further elaborate if you're a newer linux user, go to your "Ubuntu Software Center" and in the search box on the top right, start typing "ubuntu" and you should see the "ubuntu-restricted-extras" appear at some point in the list. Just install (with firefox closed) and it will auto install the newest flash. Really easy, and one of the first things I install when I install Ubuntu on any machine, this is because Ubuntu doesn't support proprietary stuff out of the box, and flash is at some point or another.
