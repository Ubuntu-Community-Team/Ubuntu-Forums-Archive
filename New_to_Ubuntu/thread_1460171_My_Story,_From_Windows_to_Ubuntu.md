### Post by OneLine on 2010-04-22
Hey, so I'm pleased to be here, this is my story, I used Windows for 13 years this March, I was really tired of having to re-install windows every  3 to 4 months because something would go wrong and Windows just wouldn't load. Today was the day I moved over to Ubuntu, so what made me move? well.... 9:05am this morning, Windows 7 updated, restarted, and was in a continues cycle, restart... restart... restart..., I left it for half an hour and finally got sick of watching it restart, so I logged onto my laptop (which is currently now as I write this post downloading Ubuntu Netbook Remix) and downloaded Ubuntu 9.10, I knew of Ubuntu since late last year, but never really had much spare time, I poped the CD in, didn't know what to expect, and had the choice to try Ubuntu before Install, that was a big surprise as Windows never had that option, so pressed Enter and sat back, waited about 2 minutes for it to fully load and was lost for words, I'm very impressed, within 5 minutes I knew Ubuntu would be the future for me. Done the install, all updates, there is no way I'm going to look back. The only thing I do miss is Spider Solitaire as I was used to that as mentioned in my first thread, but I'll get use to the change with Aisle Riot Solitaire. Over the past 10 minutes I've been getting use to the Ubuntu Software Center, removed some applications that was of no need to me, the amount of applications available for free is amazing.
