I'm not aware of this problem, don't stream Divx though, poor format imo. Try **vlc** with the **mozilla-plugin-vlc** package. VLC never failed me with formats, maybe it's a quirk with way stream set up? I don't know. It's available from repos, please see my guide in sig for instructions on install.

You might also want to install the mediaplayerconnectivity addon for firefox (only works for 32 I believe), it is excellent at piping streaming formats to the player of your choice.
