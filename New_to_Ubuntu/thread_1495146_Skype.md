### Post by lisati on 2010-05-28
Skype is in the repositries. On System->Admin->Software sources, click on the "Other software" tab, and make sure that the partner check boxes are checked. Close the dialog with the "close" button, and from the command line (Applications->accessories->Terminal) type in:
```
sudo apt-get update && sudo apt-get install skype
```
