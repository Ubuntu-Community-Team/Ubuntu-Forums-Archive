### Post by Vladlenin5000 on 2014-05-06
Indeed most here do suggest Lubuntu right away for everything from netbooks like the Aspire One to other just a few years old machines...

I've installed many of those  by request - ACER Aspire One and similar contemporary wannabe notebooks - with only 1GB RAM and always used the standard Ubuntu, with Unity, except in a couple where I installed the Lubuntu based LXLE, also by explicit request. No doubt the lighter desktop is slightly more responsive than Unity but Unity runs (almost always) fine with everything one expects to do with such machines. At the expense of a little bit of snappiness the user gains a lot in ergonomics, hence productivity. 

Personally I find Lubuntu boring, featureless, inconsistent at times and on top of all this an outdated UI design. On the other hand LXLE was a nice surprise. They did a nice job tweaking the desktop and showed that it is possible to have something bearable out of LXLE while keeping the memory footprint small.

### Post by DogMatix on 2014-05-06
You can install Ubuntu using a USB pendrive 

Link: [Installation from USB stick](https://help.ubuntu.com/community/Installation/FromUSBStick)

I have an Acer One Netbook (circa. 2007) that originally had Windows XP installed. (CPU: Atom 1.6Ghz RAM: 1GB). It will run Ubuntu with Unity Desktop to a satisfactory degree. I personally have opted for Lubuntu as it runs considerably quicker. My default browser is Chromium, so Google's Chrome would run fine.
