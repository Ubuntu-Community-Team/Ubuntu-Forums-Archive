Hi,

In Microsoft Excel you can obtain the probability of the chi squared value for a data set with a certain number of degrees of freedom using:
```

=CHIDIST(chi squared value,  degrees of freedom)

```This is what it does:

> Returns the one-tailed probability of the chi-squared distribution. The &#967;2 distribution is associated with a &#967;2  test. Use the &#967;2 test to compare observed and expected values. For example, a genetic experiment might hypothesize that the next generation of plants will exhibit a certain set of colors. By comparing the observed results with the expected ones, you can decide whether your original hypothesis is valid.
**Syntax**
**CHIDIST**(**x**,**degrees_freedom**)
**X**   is the value at which you want to evaluate the distribution.
**Degrees_freedom**   is the number of degrees of freedom.

This function doesn't exist in Open Office Calc and I need an equivalent of it.   

Can anybody help?

Thanks in advance,

Ash R

EDIT:  Solved it.  Information found [here]("http://wiki.services.openoffice.org/wiki/Documentation/How_Tos/Calc:_CHIDIST_function")
