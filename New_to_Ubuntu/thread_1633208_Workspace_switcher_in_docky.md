### Post by cariboo on 2010-11-29
In order to get the workplace switcher, you need to install the development version fo docky from this ppa:

```
ppa:docky-core/ppa 
``` 

Just open Synaptic Package Manager and go to Settings->Repositories->Other Software and click the Add button. Once you have clicked the reload button, click the status button in the lower left, and it should show up as upgradeable.
