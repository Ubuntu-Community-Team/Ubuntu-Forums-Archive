### Post by radiocognition on 2009-12-11
Most of the time, web browsers will dump your downloads into your $HOME/Downloads directory.  You should be able to check where your firefox stores downloads by going to Edit > Preferences and selecting the Main tab.  Your downloads settings are configured here.

I have some bad news for you: 7zip is a windows utility and can only be run in linux using Wine or some virtualization layer.  Check out the alternative command line version released by the same group p7zip.  If I were you I would stick to the default extractor.
