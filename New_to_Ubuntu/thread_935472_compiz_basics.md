### Post by Idefix82 on 2008-10-01
It's quite simple: you activate the "Desktop Cube" and the "Rotate Cube" options in the "Desktop" section, then you go to "General Options" and in the tab "Desktop Size" you set the "Horizontal Virtual Size" to 4. You can now press Ctrl+Alt+left mouse button and start rotating your cube by moving the mouse.

To get 3D windows you have to activate the "3D Windows" option under "Effects". That's it. I encourage you to play around with the settings, since there is really quite a lot that you can adjust.

### Post by RedRat on 2008-10-01
> **Matt26 said:**
> i'm trying to get the compiz desktop cube working on my ubuntu hardy system, but i'm not sure what options i need to set in the Advanced Desktop Effects Settings screen to get this setup- basically i want to have the rotating cube and figure out how to customize it plus the 3d effects for windows.  is there some kind of how-to for this anywhere?  thanks.

Go to System>Preferences>Advanced Desktop Settings, look down under Desktop and make sure that the Desktop Cube is checked and 3D Windows under Effects.

Then to get the cube, hold down the Ctrl and Alt keys simultaneously and click the left button on your mouse. You should see the Desktop shrink a bit and the cube ought to be there.
