### Post by eragon100 on 2009-02-05
Is it ubuntu 8.10?

Back when 8.10 came out, the Nvidia legacy drivers for your old card didn't support xserver 1.6 yet, only the drivers for newer cards did.
If you install ubuntu 8.10, you get the open-source NV driver wich doesn't support hardware acceleration (so no effects / games), but does work with all cards. If you have a newer card you can simply install the Nvidia driver with a single mouse click (you'll be prompted to do so by a popup) and it will work. For the older cards this is not possible with ubuntu 8.10.

However, Nvidia has since updated their legacy drivers and they now support Xserver 1.6, so just go to [www.nvidia.com](www.nvidia.com) and download the latest drivers for your card there, then install them manually. Not only will the desktop effects work, but you should also be able to play 3d games then! :wink:
