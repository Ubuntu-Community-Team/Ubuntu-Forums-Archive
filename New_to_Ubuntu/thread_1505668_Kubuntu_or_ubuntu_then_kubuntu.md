### Post by li(fe)sp on 2010-07-26
I am also a newbie,  I am using kubuntu and ubuntu...wanna share what I learned.

Kubuntu-desktop installs with the KDE desktop environment + a set of tools that you use in KDE environment most of them starts with 'K'. GNOME ( the default DE ) comes with its own set of tools..

1. This K tools and Gnome tools are kind of redundant most of the times..so you will have a bloated menu

2. lets say you load a 'K' application when you are in GNOME, it will work but there will be a small price to play as it will (correct me if I am wrong ) load all necessary KDE libraries for it to work, vice versa for Gnome tools on KDE.
