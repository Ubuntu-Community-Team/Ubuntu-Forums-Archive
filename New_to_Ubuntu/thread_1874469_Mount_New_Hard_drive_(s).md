Hi I have been running UBUNTU 04.11 successfully until yesterday, when I decided to upgrade my Hard Drives and do a fresh install with 10.11.

I get through the installation procedure fine, letting Ubuntu automatically format [ext4] and partitionn the hard drive without dual boot, but when I go to boot, the system doesn't boot! All I get is a blinking cursor

When I boot from CD and go into disk utility, I noticed the new hard drive isn't mounted. Even if I click mount, I get the same problem at boot. And when I boot from CD again, disk utility says it isn't mounted again! 

I tried reinstalling but I keep on getting the same problem on boot: a blinking cursor

I suspect the problem is that I  am not able to make the new hard drive mount automatically at boot

Can someone help me with some up to date and simple advice?

UPDATE: I unplugged the second Hard Drive and now Ubuntu 10.11 Boots! now I have to figure out how to add the second drive woithout goimg back to not being able to boot.

Solved.

The problems was I was trying to start on two new drives. I had to start on one first. then add the second new drive.
