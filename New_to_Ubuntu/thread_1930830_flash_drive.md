### Post by bogan on 2012-02-24
Hi!, **oldfolk**s,

Edit: You Posted:> I have Ubuntu 11.10 and when I insert the flash drive into the usb port it does not show up in the left column. Do you mean in the Launcher side-bar, or in the sidebar of the file manager display, when you Click on 'Home' or 'Files'?

If you install 'My Unity', it has Launcher options to display devices in the Left side Launcher, either when mounted or when attached.

Similarly, under Desktop there is an on/off Switch to show device Icons on the Desktop.

In addition, as my 11.10 is set up a Nautilus file manager Window opens and displays the contents, but I am not sure if that is a default setting, or something I configured myself.

To get My Unity enter in a Terminal, each line separately and press 'enter'```
 sudo add-apt-repository ppa:myunity/ppa
sudo apt-get update 
 sudo apt-get install myunity
```Then you will find it in Dash.

Chao!, **bogan**.
