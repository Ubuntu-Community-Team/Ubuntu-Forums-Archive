### Post by grahammechanical on 2011-05-10
I am not familiar with Lubuntu. I can only explain things as done in Ubuntu. You can change the language of the operating system in a utility called Language support. I think that you then can choose which language to use when you log in.

The second way is to use a utility called Keyboard. It will let you choose different language keyboard layouts. It will put an icon of a keyboard in the top panel and you can click on it and change the language. Then whatever you type in any document will be in the characters of that language. I use this to type in English and Greek in the same document. I am now switching to Greek &#945;&#946;&#968;&#948;&#949;&#966;&#947;&#951;&#953;&#958;&#954;&#955;&#956;&#957;&#959;&#960;;&#961;&#963;&#964;&#952;&#969;&#962;&#967;&#965;&#950; that is the English alphabet when typed out using the Greek keyboard layout.

Find the Keyboard utility, select the Layout tab and click Add and choose Arabic both as a language and as a country.

Remember, if you login password is recorded in English and you change the keyboard layout then the next time you type your password to authenticate some action. It will be typed in Arabic and it will be rejected. 

I am sorry that I cannot explain where to find these utilities in Lubuntu.

Regards.
