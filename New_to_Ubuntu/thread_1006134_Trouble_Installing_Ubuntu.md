### Post by Tatty on 2008-12-09
You appear to have burned the CD incorrectly.  A .iso is an image of a CD and your CD burning software needs to be told this.

There will be an option somwhere to "burn image to disc", or something like that, choose that, then select the .iso file you downloaded.
