### Post by Bodsda on 2008-05-17
it means that you need to add/enable other repositories

go to System--> Administration--> Software Sources

and enable all in the first tab, anything with 'wine' in the name on the second tab.

If there is nothing with 'wine' in the second tab then it is likely you havent added the repo's

follow instructions here if that is so
[http://www.winehq.org/site/download-deb]("http://www.winehq.org/site/download-deb")
