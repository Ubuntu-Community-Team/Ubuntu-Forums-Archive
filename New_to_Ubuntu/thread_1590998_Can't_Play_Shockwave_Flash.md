### Post by lykeion on 2010-10-08
Well, I use ubuntu and can view the page in epiphany and firefox.
Though I have flash plugin installed. I guess you haven't installed it?
You could try the open-source variant first:
[LIST=1]
[*]Applications > Ubuntu Software Center > Search for "gnash"
[*]Install the "Plugin for Mozilla and derivatives"
[*]Restart browser and try to load the page
[/LIST]
If that doesn't work you can try Adobe flash plugin
[LIST=1]
[*]Applications > Ubuntu Software Center > Search for "adobe"
[*]Install the "Adobe Flash Plugin"
[*]Restart browser and try to load the page
[/LIST]

EDIT
So you've no problem with flash in browser, but you'd like to play a SWF file on computer?
Try VLC like above poster suggests, or install standalone Gnash SWF Viewer.
