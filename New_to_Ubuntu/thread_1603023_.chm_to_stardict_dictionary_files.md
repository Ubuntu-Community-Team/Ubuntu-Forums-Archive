### Post by simon_w on 2010-10-28
stardict files are not completely trivial to create yourself.

The simplest solution is to create a tab-delimited file of the format:

```

headword	description
headword2	description2

```

 - that is: the headword, followed by a tab, followed by the description, followed by a newline.

Then you can create a stardict package by using the tabfile program in the stardict-tools package, as described here:
[http://stardict.sourceforge.net/HowToCreateDictionary]("http://stardict.sourceforge.net/HowToCreateDictionary")

you will need to:

```
sudo aptitude install stardict-tools dictzip
```

If you want to do anything more complicated, you need to write a program to perform the conversion.  The specification is here:
[http://stardict.sourceforge.net/StarDictFileFormat]("http://stardict.sourceforge.net/StarDictFileFormat")

but it's a little unclear in places.  However, I've created a dozen or so stardict dictionaries now (mostly for Asian Buddhist dictionaries) and I think I've got most of it sorted, so I'll try to help if I can.

simon
