Just make sure that your active windows is Firefox , then open Hud (with Alt key) and write **add** , you will see all your add-ons and preferences . 
For example , i can see my flash-aid's preferences and wizard mode.. or you can specify your add-on by write Stylish . If i write flash-aid on hud i can see all the menu .. Preferences - Wizard mode - check update - documentation ..
