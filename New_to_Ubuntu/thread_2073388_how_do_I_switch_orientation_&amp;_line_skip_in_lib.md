### Post by StinkySQL on 2012-10-19
Not sure about orientation but your line spaces are set in paragraphs.

Go to Format, Paragraph. In the first tab "Indents and Spacing" you will see an option to change Line spacing to single, double and a whole lot more.

### Post by howefield on 2012-10-19
Format > Page > Page Tab > Portrait or Landscape.

and

Format > Paragraph > Idents & Spacing > Line Spacing

### Post by grahammechanical on 2012-10-19

Go to the Format menu and select paragraph. There you can set line spacing to single, 1.5 lines or double. It is under Indents & Spacing. Is that what you mean?
