### Post by dbbolton on 2009-03-04
Try running this command:

```

nm-applet
```

It should bring up Gnome's network manager applet in the KDE system tray.

Otherwise, you might have to browse through the KDE menu. I'm not sure what its network manager is called.

---

### Post by yther on 2009-03-04
It's called (what else?) KNetworkManager!  :KS

Applications > Internet > KNetworkManager, or just put "knet" in the search field and it'll find it for you.
