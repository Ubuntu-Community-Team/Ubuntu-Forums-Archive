### Post by Little Girl on 2009-10-28
Since you've had bad experiences before, it's understandable that you're hesitant to upgrade the kernel. The good thing about Ubuntu is that it adds the new kernel to GRUB without getting rid of the old one. The default kernel will be the most recent one, but the older ones are still available if you need them.

So if you go ahead and do the update and it doesn't work, you can reboot the computer and press the **Esc** key when you see the GRUB boot loader. That will put you into the GRUB menu, and from there you can use the arrow keys to manually select an older kernel to boot with. :p
