### Post by Mark Phelps on 2011-04-07
Some comments ...

Before you start raving about Wine, you should go to the WineHQ site, look up the apps and versions you want to run and see the compatibility ratings.  Don't be surprised if (1) the app isn't listed (meaning it probably will NOT work), or (2) the app has a low rating (Bronze).  A rating of Garbage means the app will not run -- regardless of what you do.

For example, Corel Draw, versions 10 & 11 are Bronze, version 12 is Garbage.  So if any of those are your versions, you're not going to be able to use it with Wine.

Regarding dual disks -- I do much the same and have found that installing each OS and apps to its own physical disk is the best way to go.  Prevents problems with MBRs being overwritten and simplifies any boot loader repairing activities. To do this, simply do the following:
1) Have only your planned Ubuntu drive connected
2) Install Ubuntu to that drive
3) Reconnect your Windows drive, but set the BIOS to continue to boot from the Ubuntu drive
4) Boot into Ubuntu, open a terminal and type "sudo update-grub".  This will generate a new GRUB boot menu that will include an entry for Windows.
5) Reboot your PC.  You should then see a boot menu with various selections.
