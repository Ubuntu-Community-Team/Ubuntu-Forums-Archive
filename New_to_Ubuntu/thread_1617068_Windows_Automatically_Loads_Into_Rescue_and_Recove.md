### Post by samacaster on 2010-11-09
Can you boot to Safe mode? If you can, do so with network access. Google EasyBCD. You'll see it when it comes up. DL and install. Run the program. The first time you run the program it will look for any windows BCD. It will find it, if you didn't erase it. It will be displayed in a small window. You can now add Ubuntu with the add button. Click the linux tab and choose the bootloader -(GRUB, GRUB2, LEGACY). Move the mouse down over the button labeled "write" and click. When you reboot you'll see: 

Windows Vista
Linux/GNU (or whatever you named it)

If you can do Safe mode, but no network, different thing. Open a command prompt and type C: to get to root directory. You'll see C:\

now type  cd\windows\system32

next type  bootrec.exe /fixboot

then          bootrec.exe /fixmbr

then type   bootrec.exe /RebuildBcd    (capitals do matter with this)

This will rebuild you Boot sector, then your MasterBootRecord, then write the new BCD to the new boot sector. When you reboot, you will not see Ubuntu, you'll go straight to windows. You'll need EasyBCD to add Ubuntu. 

Of course you could just go ahead and recover, but you'll still need to rewrite the BCD to include Ubuntu. Or load the LiveCD and do a bunch of SUDO this and SUDO that to reload the Grub bootloader which was got you into this mess in the first place.
