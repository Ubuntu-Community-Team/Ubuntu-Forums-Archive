### Post by HotShotDJ on 2009-02-08
> **Crescent0428 said:**
> How can I know whether my computer can support a static IP or a dynamic one? I can access the internet- does this mean that it supports a static IP? Also even before, when I was using Network Manager, I cannot access other computers in my local network.The question is not if your computer can support a static IP.  Your COMPUTER can certainly support it. The question is this: does the NETWORK you are using support it.  Most networks use dynamic IP assignment.  When you set up your computer to use a static IP on a dynamic IP network, you will run into problems if the network has assigned the IP address that you have set to another device. You will need to contact your campus IT support desk to find out if you can use a static IP.  If you can, they will have to assign you the static IP address.

I don't know exactly what you mean by "accessing" other people's computers on the network.  Nor am I sure what BorgChat is or does.  I'm not much of an "expert" on networking.  But there are others who may be able to help.  But they will need to know exactly what it is you are trying to do.
