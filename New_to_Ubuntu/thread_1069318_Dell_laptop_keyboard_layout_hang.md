### Post by Mark Phelps on 2009-02-14
Doubt that Ubuntu will install on a machine with so little RAM, but even if it does, you're NOT going to happy with the pokey-s..l..o..w re
sults.  You need at least 256MB of available RAM for decent response, 512MB is better.

You would do better with an alternative distro such as Damn Small Linux (DSL), or Puppy Linux, or Mini-me.  For the latter, see the distrowatch site for a list of Linux distros and their locations.

at any rate, you may want to google "ubuntu LaptopTestingTeam" and look up your latitude to see if there is a common keyboard problem to your model.
