### Post by jim_deadlock on 2016-03-12
I would say this is likely to be a problem with your graphics driver. If you run 'Additional Drivers' it will scan your hardware and (hopefully) give you a choice of drivers which you can apply. For example if you have nVidia then you should get a list of proprietary nVidia drivers and you can select the newest stable one.
