### Post by ST3ALTHPSYCH0 on 2009-11-17
If you're still running Hardy, then the tool to which you are referring is "Screens and Graphics", and has, sadly, been removed. Karmic will still use an xorg.conf if it's there. I'll go look for the posts that explain how to edit the file.

Read post #38 [here](http://ubuntuforums.org/showthread.php?t=1319491&page=4;%20postcount=38) to see how to properly edit your xorg.conf file manually. Post #34 contains a link to how I did it automatically with an 8.04LTS live CD.

It should go without saying but backup your xorg.conf file before editing. I realize that it doesn't give you the resolution you want right now, but you at least have a GUI, you may break that attempting to get the resolution you desire (I did several times).
