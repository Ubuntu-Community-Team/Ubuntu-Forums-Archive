### Post by jmszr on 2009-06-03
crboomer,

I don't believe that 7.10 is supported any longer. If you are running a desktop I would suggest 8.04 LTS (Long Term Support) [http://releases.ubuntu.com/hardy/](http://releases.ubuntu.com/hardy/) , as it is quite stable now. If you are using wireless, however, I'd suggest 9.04. Download the .iso at :[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download) and burn as image to a CD [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto) and then try it on your computer as a Live CD to make sure there are no problems with your hardware. Install as per psychocats and you'll be on your way.
