### Post by p_quarles on 2008-04-26
There are several different php5 packages. The one you most likely installed is the Apache 2 module (libapache2-mod-php5). If you want a standalone php interpreter, you should get php5-cli, which you can run in the terminal.

As for where it is, it's in /usr/lib/apache2/modules and in /etc/php5. There is no standalone executable in the Apache module.
