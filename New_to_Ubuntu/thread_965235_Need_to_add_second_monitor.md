### Post by BKonkle on 2008-10-31
Before you buy one, make sure your video card supports multiple monitors.  I'm using an Nvidia card that has one VGA output and one DVI output.  I'm using a DVI to VGA converter to hook up the second monitor.  In Ubuntu, I'm using the nvidia-glx-177 package and I use the Nvidia Settings tool in the Preferences menu to set up TwinView for the second monitor.  Let us know if you need more help!
