### Post by Xiong Chiamiov on 2009-03-01
Yes it can, and in fact, the easiest way is to just install the xubuntu-desktop package.  You'll then be able to choose whether you want to login to gnome or xfce when you login.

XFCE in general is faster than Gnome, but back when I used Xubuntu, it was so loaded down with Gnome daemons that there wasn't much of a difference.  I didn't like XFCE at all until I tried it in another distro.  This was also back with Hardy (or maybe Gutsy), so don't take this too seriously.

### Post by stoogiebuncho on 2009-03-01
I started out running Ubuntu on my laptop, but switched to Xubuntu after a while to get a bit of a speed boost.  I did this the way Xiong suggested - by installing the xubuntu-desktop package.

I do find it to be faster, but it's not a huge difference.  I find myself doing more command line stuff in Xubuntu also, because there are less options built into the GUI.  So if you don't like command line that might be something to think about.

But it's quite easy to try for yourself.  Just install the xubuntu-desktop package and you'll be able to choose your desktop manager when you log in.
