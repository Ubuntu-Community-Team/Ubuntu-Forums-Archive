### Post by lovinglinux on 2009-05-04
> **HiddenDev said:**
> I Was Updating a Flash webtemplate and it said use flash 5 and i was just looking for flash 5 haha i didnt know there was even flash 10. I was using it to edit the .fla files

You should update your stuff regularly, since they not only introduce new features, but also add security updates. Running old software might increase security risks.

Ubuntu comes with a nice update tool. Most of the time, you don't have too hunt for updates like on Windows, except when you want updates that are not security related. For that you  can for example use deb files from [GetDeb](http://www.getdeb.net/) or add additional repositories like PPA's from specific application developers on Launchpad.

More about installation on Ubuntu at [https://help.ubuntu.com/8.04/add-applications/C/index.html](https://help.ubuntu.com/8.04/add-applications/C/index.html)
