I should also mention that there are some updates in Intrepid Proposed! Alsa-utils and the pulse audio stuff is OK but some of the other updates still break things so if you decide to enable Intrepid Proposed in Software Sources be sure to "de-select" anything you don't trust! Then be sure to change Software Sources back again!

You break it, you own it :)

The proposed updates have not been fully tested yet! The same is true of backports! And sometimes there is no going backwards!
