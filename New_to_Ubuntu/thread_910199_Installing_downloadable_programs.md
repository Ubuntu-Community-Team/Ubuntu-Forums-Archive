### Post by baruch60610 on 2008-09-04
The problem is that many Websites assume you're running Windows.  They don't take into consideration that you might be using Linux.  So they present you with a program that only runs on Windows, and when you try to use it things get very confusing.

Apparently there isn't a Yahoo! Messenger version for Linux.  There *is* a Web-based version that would allow you to get many of the same features.  You don't have to install that version, you just use it like a Web page.

If you still want to use something on your own computer, there is a program called pidgin Internet Messenger, probably installed by default.  If you already have a Yahoo! account, you can use this instead of Yahoo! Messenger.  If you don't have a Yahoo! account, get on by visiting Yahoo! and creating one (just using the Web page).

Pidgin is pretty simple to set up.  You just have to tell it your username and password, and then it will connect to Yahoo! (or several other IM services, if you've got accounts).

Good luck.

### Post by nothingspecial on 2008-09-04

Have a look in your menus, look for add remove. If you want a media player, type media in the search box. If you like the look of the program tick the box next to it. Click apply, enter your password and about 10 seconds later the software will be installed and ready to use.
