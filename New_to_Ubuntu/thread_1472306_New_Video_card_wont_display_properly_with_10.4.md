### Post by lidex on 2010-05-06
Looks like you don't have a driver specified in xorg.conf. But why have an xorg.conf if you're switching video cards? Go back into failsafe graphics and into desktop. Open hardware drivers and make sure nvidia-current is enabled, if it's anything else, remove it.
Run this command and reboot:
```
sudo mv /etc/X11/xorg.conf xorg.conf.old
```

If nvidia-current is enabled do this instead:
```
sudo nvidia-xconfig
```
Reboot.
Use only one card through out.
