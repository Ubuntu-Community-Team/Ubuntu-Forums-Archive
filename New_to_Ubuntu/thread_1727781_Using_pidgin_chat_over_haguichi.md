### Post by ztefn on 2011-04-13
> **Farley said:**
> 
I was wondering if there is a guide on how to get this set up properly or if someone could help me out with it.
In Pidgin go to Accounts &#8594; Manage Accounts &#8594; Add... &#8594; Protocol: Bonjour, Username: Farley &#8594; Save. That's it! :D

> **Farley said:**
> Does the other user also have to have Pidgin installed?
Yes, Pidgin, or any other chat client that supports the Bonjour protocol. Though it seems that on Windows you need to install some extra stuff: [http://developer.pidgin.im/wiki/Protocol%20Specific%20Questions#CanIuseWindowsPidginforBonjour](http://developer.pidgin.im/wiki/Protocol%20Specific%20Questions#CanIuseWindowsPidginforBonjour)
