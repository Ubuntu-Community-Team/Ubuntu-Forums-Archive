### Post by theaaronc on 2010-02-24
If you are creating the desktop launchers manually, when you set it up you can click on the icon image in the upper left hand corner of the launcher creation window. This should open a window of icon images that you can browse through and assign.

If the launcher is already on the desktop, right click on the icon and click the image in the upper left hand corner of the properties window. If no icons are shown to select then navigate to:

/usr/share/icons/hicolor/scalable/apps/

This should get you a list of icon images you can use.


Someone else may be able to give you a more efficient way to do this, but I think this may at least get you some options.
