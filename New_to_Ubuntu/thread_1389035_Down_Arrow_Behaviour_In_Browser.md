When browsing, if i hit the down arrow key, it goes to the very bottom of the page.. and not just a page down. How/where is this behavior controlled?

### Post by gnometorule on 2010-01-23
Go to

Edit->Preferences->Advanced->Accessibility, 

and uncheck "Always use the cursor keys to navigate within pages.'

This should bring you back to incremental scrolling with the down arrow key.
