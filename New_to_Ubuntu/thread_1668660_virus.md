### Post by pbpersson on 2011-01-16
Anyone can create a web page that says "We see you have a virus, click here to scan your machine" and in most cases when you click the link to "fix" your machine, it installs a virus.

I always laugh when it wants to install an .EXE file on my machine since we know that is a Windows executable file.

These sites are written to infect Windows machines, they would have no idea how to deal with Linux.  :D

There are utilities to scan for viruses, worms, and trojans on Linux machines if you are worried.  You can Google articles on them but 99.5% of the malware out there now is written for Windows.
