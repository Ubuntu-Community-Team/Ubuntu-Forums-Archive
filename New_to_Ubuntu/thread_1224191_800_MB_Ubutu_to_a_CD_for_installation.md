### Post by 3rdalbum on 2009-08-13
The Ubuntu disc images are all under 700 megabytes in size. If it's 800 megabytes, then you've either downloaded the wrong thing or are otherwise doing something incorrectly.

Go to [www.ubuntu.com](www.ubuntu.com) and download the disc image as a ".iso" file. DO NOT double-click this file after downloading it. Just burn it straight to disc, using the "Burn ISO image" or "Burn Image to Disc" function of your burning software.

### Post by Mark Phelps on 2009-08-14
> **Arof said:**
> ... I bought the two 800 MHz CDs from Kjell & Co in Sweden.

It IS possible to burn more than 700MB onto a CD.  It's known as overburning. I used to do this myself with Plextor drives before DVDs came out.

But ... while this CAN be done, it only works on some media, with some make & model CD burners and, generally, can only be read by the same make & model CD drives as the burners. 

So, basically, they sold you CDs that you're most likely NOT going to be able to read in a CD drive.

If they are really CDs (not DVDs), then you should demand your money back.
