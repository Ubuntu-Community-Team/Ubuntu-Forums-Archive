### Post by dcsoldschool53 on 2011-07-25
I have Mozilla Firefox and if you are looking for the bookmarks, start looking in nautilus in the .Mozilla/firefox folder. It is located in your home folder. Or in a terminal type locate bookmarks, this will give the exact name of the folders and they locations too

### Post by diesch on 2011-07-25
Bookmarks are stored in the file *places.sqlite* in your firefox profile folder.  To find the profile folder open *about:support* in Firefox, then you'll get a page that has button to open the profile folder.
