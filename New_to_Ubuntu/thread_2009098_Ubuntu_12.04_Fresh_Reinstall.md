### Post by nipunshakya on 2012-06-23
Before you install Ubuntu, you should be sure that the disk has no errors and run it in LiveCd mode. When booting from LiveCD, select 'check disk for errors' and make sure that the disk has no errors(it will show if u have errors). After that try to run in Live CD and check if it works.. Otherwise, you need to re-download the .iso image for ubuntu(its better u look at torrent download of iso as it offers a good check)
