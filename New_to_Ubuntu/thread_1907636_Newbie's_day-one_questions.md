### Post by chenxinghao on 2012-01-11
I have just installed Ubuntu 11.10 on my desktop PC, an IBM ThinkCentre A50P. I have the following questions:
 
(1) Where in the Ubuntu desktop screen to access Help?
 
(2) How to open a terminal window?
 
(3) About the "workspace": It seems that I can not open the same application in multiple workspace; for example, if I open a Firefox window in workspace A, switch to workspace B and try to open another Firefox window, the screen would jump back to workspace A. The same goes with LibreOffice applications.
 
What I need to do in order to be able to open multiple Firefox sessions in multiple workspaces?
 
(4) I installed VLC from Ubuntu Software Center. Where do I go to find it?
 
(5) I inserted a music CD into the CD-RW drive and it pops up a application selection menu and I selected VLC to play the CD; then how do I switch to a different CD player application?
 
(6) How to access the list of installed applications from the Ubuntu desktop?
 
(7) I installed Ubuntu 11.10 with a dual-boot option. In the which-to-boot-from screen, however, the top selection line reads "Ubuntu Linux 3.0.0 ..." something. Why?

### Post by IWantFroyo on 2012-01-11
1) Hit Super (windows key) and type "Help"
2) Ctrl-Alt-T, or Super and type "Terminal"
3) Right click and select "new window"
4) Super, and type "VLC"
5) Open the other application, and play the CD in it. If you have a music file, you can right click on it and select to "Open with" and choose the app.
6) Super, click on Installed Applications
7) That's the kernel version. The Linux kernel is commonly updated while you're on one system, too, so be prepared to see a lot of those entries.

I apologize for any errors I made in advance - I rarely use Unity. I mainly use Gnome2 and Openbox.
