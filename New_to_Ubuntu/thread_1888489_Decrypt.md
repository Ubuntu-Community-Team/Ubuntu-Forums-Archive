### Post by DapperMe17 on 2011-11-29
Check the settings of Gdecrypt. It might be that the default "encryption type" has been changed in the newer version. 

If you were using it as a GUI for Truecrypt...there are many encryption types available in Truecrypt. 

Because it looks like Gdecrypt is outdated and no longer maintained, you might like to know about an easy little program called "Cryptkeeper".
