### Post by howefield on 2017-04-20
You'd usually find log files in /var/log/ eg, if they are crash reports you might find info in /var/log/crash/ or /var/log/apport.log.

Have a look for the log viewer application, I'm not sure what that would look like xubuntu but in the Ubuntu Gnome flavour it is called "Logs"

You'd have to be a bit more accurate with the message description for anyone to know for sure what it is.
