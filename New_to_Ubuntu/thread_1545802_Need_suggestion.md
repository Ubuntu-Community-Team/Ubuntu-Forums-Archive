### Post by TBABill on 2010-08-04
If this happens on all sites you probably have no Internet connection on that computer. Has it been rebooted to see if it comes back up? Without having too much knowledge about configuration, type of connection (wireless or wired), firewall, etc., a reboot (if it was working properly before) may be the easiest first step. More specific information about what works and what doesn't (like other Internet apps) would be needed to go much further. Also, there is an icon in the lower right that shows network status. You can click on this to connect, disconnect, troubleshoot, etc.
 
Hopefully that helps. If you need to go more in-depth you are likely to get help here but often with a bit of rudeness since the system you are seeking support for is Windows and not Ubuntu. If you can ignore some of those comments you may get to the answer that solves your problem. There are also Windows forums where you can also find assistance if you are unable to get quick answers here.

On a more serious note, open a command prompt and type "ipconfig /all" then post what you get. If the IP address is 169.XXX.XXX.XXX then its likely not getting an IP address from the DHCP server or router, or if it says media disconnected then there's a problem with the physical connection (cut wire, bad connector, etc.).
