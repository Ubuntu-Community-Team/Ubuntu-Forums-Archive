### Post by hyper_ch on 2008-06-17
The kernel is constantly developped and linux will not just overwrite the old one... it can be that some thing won't work... so it downloads each kernel as seperate file and makes an entry in the boot manager. So if you're unhappy with the newest kernel because something is not working, you can go back to an older one and use that until bugs in the new one are solved (of course if you find things that don't work you should also submit a bug report).
