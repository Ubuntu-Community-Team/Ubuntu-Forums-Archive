### Post by Verbeck on 2010-10-25
try using remastersys to create a backup disk. I'v tried it to create a backup livecd of the current installation (with all the programs) and it works perfectly. There's an option to backup the user files too..

.deb download:
[http://www.geekconnection.org/remastersys/repository/karmic/remastersys_2.0.17-1_all.deb](http://www.geekconnection.org/remastersys/repository/karmic/remastersys_2.0.17-1_all.deb)
