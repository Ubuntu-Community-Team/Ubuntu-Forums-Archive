### Post by Sable683 on 2009-05-03
I have been downloading google videos with clive (an awesome CLI program)... and wanted to convert the files to AVI as easily as possible and this is what I have found:

Revertex posted the link to Downtube w/ GUI in this post:

[http://ubuntuforums.org/showthread.php?t=467297]("http://ubuntuforums.org/showthread.php?t=467297")

Which will download a youtube or google video and convert it to an avi file for you. The problem that I ran into is that any video longer than 15 minutes, the program will stop. 

I have figured out that if you download the FLV file with clive (or any other program you may use), instead of entering the URL for the video in DownTube, you enter the full path such as /home/john/Seinfeld.flv into the video URL box. Then enter the name you want for the file and click the box for make download and convert video file from Youtube and just wait. Apparently, this program will convert the file in a short matter of time. 

** Special note - this will cause you to loose the FLV file. You will be left with only the AVI file.

Hope this helps someone...
