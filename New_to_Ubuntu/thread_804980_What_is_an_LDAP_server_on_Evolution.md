### Post by decoherence on 2008-05-23
Not sure what you mean. LDAP is the lightweight directory access protocol. It's like a very sophisticated phone book that can work with many different applications, including Evolution. Evolution would use it to manage your contacts and perhaps during your initial authentication, but there are many things it is used for.

You only need to care about it if you have an LDAP server on your network. You'd likely only have an LDAP server if you worked at a fair-sized company with a large directory of users they need to manage.
