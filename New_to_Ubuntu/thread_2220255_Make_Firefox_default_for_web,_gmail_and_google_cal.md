### Post by Frogs Hair on 2014-04-27
If you have been accessing mail and calendar via a web browser you can continue to do so with any browser, but you won't find the integration extensions available for Chrome. Thunderbird being a mail client is different than a web browser and offers Lightning as a calendar extension and from what I read allows for synchronization with Google Calendar.I don't use Google Calendar and can.t offer any setup information.  


[https://addons.mozilla.org/en-us/thunderbird/addon/lightning/](https://addons.mozilla.org/en-us/thunderbird/addon/lightning/)

### Post by lolwhites on 2014-07-06
For email, install Gnome-Gmail from the software centre, then set it as the default email application. Clicking on email links will now open Gmail in your preferred browser.
