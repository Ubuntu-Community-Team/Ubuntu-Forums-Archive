### Post by grahammechanical on 2014-05-02
Do you have Ubuntu One installed? There should be an icon in the Launcher and even a cloud icon as an app indicator in the top panel. If you click on the cloud icon you will have several options, including go to the Ubuntu One web site. That web page will give you an option to download all your files as one .zip file.

The Ubuntu One service is closing down. So, you need to remove all your files or they will be deleted on 1st June 2014.

If you open Ubuntu One from the Launcher of Dash you can sign in with your existing Ubuntu One Single Sign-On password and then you can Sync folders. There should be a Ubuntu One folder in your /home folder that the files will sync to.

Regards.
