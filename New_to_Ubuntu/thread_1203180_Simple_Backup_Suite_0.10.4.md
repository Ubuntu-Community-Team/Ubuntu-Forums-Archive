### Post by Sir Jasper on 2009-07-03
Hi,

I use Sbackup and am pleased with it.

You can close all screens.

Even if it is running it is not obvious and you will not know when it has finished.
However, if you install htop you will then be able to see what is running and when processes finish.

My Sbackup run takes about 5 minutes running in the background but that's for about 1 GB since my Home directory is not large. If you have a large home directory it may take an hour or more.

As regards its not working - it could help if you provide more information about your settings and chosen method.
