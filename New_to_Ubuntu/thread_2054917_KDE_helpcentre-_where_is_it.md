### Post by vasilbelarus on 2012-09-08
I have never use this app. But  KDE is another alternative DE for Unity(Gnome). The program you use is the part of KDE project, and help will work only if packages for kde installed(by default they are not).
Another way to find help is to see /usr/share/doc/ folder on your system and find nesessary  folder for your app it should be named like your program name(Skrooge).
