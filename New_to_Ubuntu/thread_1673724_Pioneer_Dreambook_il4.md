### Post by Spr0k3t on 2011-01-22
I've actually worked with this one before and kept running into the exact same issue.  The only way I was able to get it working was to use the 8.04 alternate install CD and load from ground up.  Keep the install completely clean and just update the distro one by one.  I'm not sure why, but I always had problems with the 1024x600 display and the chipset.  The IL4 isn't the only one with those problems either... I've also seen it in one of the early no-name netbooks which also used the C7 processor and the 1024x600 display.  The key is to use the alternate install more than anything.
