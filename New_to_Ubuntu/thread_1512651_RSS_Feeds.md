### Post by Naggobot on 2010-06-18
When you go to a web page with Firefox there may be a orangish yellow symbol on the right end of the browser address bar. This symbol has a dot and two concentric quarter circles around that dot. It looks a bit like the radio (wlan) symbol in the Gnome applet bar. 

Move your mouse pointer over that symbol and wait a while so that tool-tip is displayed. It should read "Subscribe to this page."

By clicking this symbol you can subscribe to the feed which is a bit like bookmark that updates automatically. To get more out of rss feeds you can use some reader that displays more information about the feed. I guess Google reader is common but I found it too difficult (or too easy do not know which) so I was unable to use it. I ended up using [bloglines]("http://www.bloglines.com") which I have found good for my purposes. Bloglines also has a button you can drop to firefox bar for easier subscription.
