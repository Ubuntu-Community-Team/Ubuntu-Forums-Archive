### Post by fallenashes on 2010-04-20
Solved: Used a Program called Talika, which emulates the Win7 taskbar pretty darn well (Icons only, Pin to Taskbar, and Aero Peek/Window Previews), and then moved all the info at the top to the bottom (Right Click, Uncheck Lock to Panel if Possible, and then Move to the bottom by dragging).  Will write this as a tutorial for newbs like me later.
