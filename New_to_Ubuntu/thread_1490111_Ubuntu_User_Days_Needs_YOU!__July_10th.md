### Post by elizabeth on 2010-07-02
Hi everyone!

Sorry for the confusion, this is an event on IRC (not the forums).

If you have an IRC client, see: [https://wiki.ubuntu.com/UserDays/JoiningIn](https://wiki.ubuntu.com/UserDays/JoiningIn)

If you'd rather use "Lernid" the application developed for events such as this to make IRC easier, check it out here: [https://wiki.ubuntu.com/Lernid](https://wiki.ubuntu.com/Lernid) (it can be installed with the "lernid" package).

How it works is an instructor will join the channels (chat rooms) #ubuntu-classroom (where they'll teach) and #ubuntu-classroom-chat (where users like you can chat and ask questions). These are 1 hour sessions, each on a topic, so unlike the forums you get a real time lesson and the opportunity to ask the expert instructor direct questions.

To see logs from the last User Days to get some idea of how it works, check out [https://wiki.ubuntu.com/UserDays/Logs/January2010](https://wiki.ubuntu.com/UserDays/Logs/January2010)
