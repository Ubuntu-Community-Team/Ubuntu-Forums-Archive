### Post by Jon Monreal on 2009-11-16
Try going to System -> Preferences -> Sound and going to the Output tab. Check to see if there are any options under "Choose a device for sound output:" that says anything about USB Audio.

If this doesn't work, check under the Hardware tab to see if there is anything there.
