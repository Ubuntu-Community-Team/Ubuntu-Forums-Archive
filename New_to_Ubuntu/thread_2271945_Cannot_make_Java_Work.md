### Post by Vladlenin5000 on 2015-04-02
1. [COLOR=#000000]Dell Latitude 3340 is a relatively new machine. It benefits from newer, not older, Ubuntu releases. If you intend to use a LTS realease then go fo 14.04, not 12.04.[/COLOR]

[COLOR=#000000]2. Installing Java isn't (shouldn't be) related to the age of the release. First of all, choose what Java you want... If it's the open source one then simply install [/COLOR]*ubuntu-restricted-extras*[COLOR=#000000] (includes Java) or the Java alone, both available at USC for your convenience. If, on the other hand, you want Oracle Java then follow this simple instructions: [http://www.webupd8.org/2012/09/install-oracle-java-8-in-ubuntu-via-ppa.html](http://www.webupd8.org/2012/09/install-oracle-java-8-in-ubuntu-via-ppa.html) [/COLOR]
