### Post by tcolar on 2011-06-07
If you are using Natty, you can try this (let me know if works):

A user reported that some changes in pppd are preventing it to work with relative path for thechat script.

So do this:
- Edit your conf file  (example conf/vodafone )
- Find te line that reads something like *connect /usr/sbin/chat -f conf/vodafone-3g-chat* (should be the last line) 
- replace the chat file path by a "full"(absolute) path, so for example you would have something like:
connect */usr/sbin/chat -f /home/me/berry4all/conf/vodafone-3g-chat*

Then try berry4all and see if that helps.
