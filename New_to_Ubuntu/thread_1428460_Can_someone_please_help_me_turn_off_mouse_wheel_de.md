### Post by eriktheblu on 2010-03-12
I believe this will be in your compiz-fusion settings. You may need to install a settings manager like CSSM.

---

### Post by zeroseven0183 on 2010-03-12
eriktheblu is right, you have to install CCSM (CompizConfig Settings Manager).

So when finished installing, access it through **System** > **Preferences** then uncheck the **Viewport Switcher** (under Desktop section)
