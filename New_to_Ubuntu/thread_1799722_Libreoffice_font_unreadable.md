### Post by jtarin on 2011-07-08
Go to the Gnome menu System>Preferences and change the fonts used for the applications. OO/Libre (Ubuntu version) has some problem with that setting sometimes.

Afterwards you can change it like this:
Go to Tools -- Options --> fonts
and use font replacement to replace "Andale Sans UI" with a font of your choice.

I do this to fix the menus sometimes and it sometimes works.
