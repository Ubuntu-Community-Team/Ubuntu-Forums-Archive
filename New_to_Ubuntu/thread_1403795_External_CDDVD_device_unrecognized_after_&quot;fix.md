### Post by AntonioPT on 2010-02-10
Medibuntu SURELY didn't cause that. About the DVD drive recognition, if only the current Ubuntu installation has a problem recognizing it, just install it again. Ubuntu doesn't need to recognize the drive, the BIOS does, in order to boot the Live CD. If you can't boot the Live CD, then it's a DVD drive problem, unrelated to Ubuntu.
