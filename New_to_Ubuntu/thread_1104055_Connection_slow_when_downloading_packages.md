### Post by sydbat on 2009-03-23
You may need to change the server you get updates from. In System > Administration > Software Sources, you can change the "Download from" in the Ubuntu Software tab to wherever you want to download from. 

Although I am in Canada, and the original server was Canada (as defaulted during install), I changed it to "Main server" because the Canada server was way too slow.
