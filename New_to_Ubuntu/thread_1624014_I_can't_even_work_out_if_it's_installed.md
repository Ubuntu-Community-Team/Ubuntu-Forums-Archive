### Post by chamira on 2010-11-17
> **spleenboy2000 said:**
> So, just so I am sure here:
Is Ubuntu installed or not? I am looking at an ubuntu desktop, with an install ubuntu icon on it.
Do I need to run this, or is ubunutu already installed?

Many Thanks for all the help

No you haven't got Ubuntu installed. 

What you are doing is running the Ubuntu installation CD image (the .iso file) from the harddrive rather than burning it to a CD and booting up from it. 

This CD is for trying out Ubuntu and then installing it. You are limited to what you can do from running Ubuntu from the Live CD - for example I don't think you can install any new programmes &etc and any changes you make to the settings will not be saved.

It would be safer to burn the .iso image on to a CD and then run and install Ubuntu from the CD (Although I am not sure what would happen if you carried on and installed Ubuntu from where you are now.)
