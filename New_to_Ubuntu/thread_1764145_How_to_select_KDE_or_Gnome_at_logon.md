### Post by dniMretsaM on 2011-05-21
As far as I have figured out, you have to have it set to ask for a password to get asked what DE you want. The selection menu only appears on the screen where you enter a password. See [this thread](http://ubuntuforums.org/showthread.php?t=1743264) that I made a while back. I also made a bug report ([click here](https://bugs.launchpad.net/ubuntu/+bug/773573)) but I haven't gotten any replies.
