### Post by gettinoriginal on 2009-03-04
Compiz is installed by default in Ubuntu, and the legacy fullscreen support is a default setting.  You will need to install CompizConfig Settings Manager from Synaptic.  After install it will be in System, Preferences, CCSM, Utility, Workarounds, click on it and untick legacy fullscreen support.
