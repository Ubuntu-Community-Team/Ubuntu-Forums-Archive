### Post by mlentink on 2009-01-22
The easiest way to install packages is simply go to the Applications menu and selecting Add/Remove programs

Alternatively, use the System>administration menu and select Synaptic package manager. You will find a 'Sources' entry in that menu as well, allowing you to add additional repositories
