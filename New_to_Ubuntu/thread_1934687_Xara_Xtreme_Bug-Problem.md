And I've been having the exact same problem in Xara, no tool bar. I'm running Ubuntu 11.04. But I've found a way around this right now! Sort of... Make sure the main window of Xara is selected (not any of the bars) and press Alt-F. The file menu ought to come up, and now you should be able to access the other menus by moving your mouse cursor to the right where you see a thin grey bar just above the file menu box. Clear as mud?

If you're not able to follow my directions, I might post a screen shot. Good luck!
