### Post by Uubnewb on 2009-01-31
Where does Ubuntu 8.10 save the applications I download via the "Add and Remove" manager?

---

### Post by taurus on 2009-01-31
After installing it, a copy of it should be saved in your archive, /var/cache/apt/archives.

Applications -> Accessories -> Terminal
```
ls -la /var/cache/apt/archives
```
