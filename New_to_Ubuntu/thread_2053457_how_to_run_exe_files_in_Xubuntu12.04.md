### Post by mcduck on 2012-09-05
...on the other hand, it's very unlikely that you'd actually need to run any specific program to set up your router.

In almost every case the router setup can also be done simply by connecting the router to a computer with a network cable and then accessing the router's settings with your web browser. All you need to do is find out the default IP address and login name/password, both of which should be mentioned in the router's manual.
