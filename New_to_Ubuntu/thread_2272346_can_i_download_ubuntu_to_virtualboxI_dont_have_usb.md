### Post by plucky on 2015-04-06
> Id like to open unbuto from virtual box.How can i get it in there?Not being a .ovo file(it said)

You can install directly from the downloaded .iso file as virtualbox will mount the iso so it looks like a CD/DVD and the install will continue from there.

When you ADD a new machine to Virtual box and Start it,it will ask you to select the start-up disk.The default is the CD/DVD drive,but if you click on the icon next to the box you can then select the .iso file from the Downloads area.

The install will then use the .iso file.

Good Luck
