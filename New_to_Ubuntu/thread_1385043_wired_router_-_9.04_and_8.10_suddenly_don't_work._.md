---

### Post by flameproof on 2010-01-19
Sorry, found it. For unknown reasons the static router IP wasn't set in the routers DHCP setting. Strangly it effected only Ubuntu, but not Xp.

I set the IP back to 192.168.1.1 and everything was fine again.
