### Post by MasterNetra on 2010-09-06
Of course, when you got your computer the manufacture had already installed all the drivers for your hardware. Windows alone doesn't usually have drivers for all the hardware. You might be able to get some from update but for most you will need to visit your computer's manufacture's website and download the drivers you need/want from there. If there is no drivers for Windows 7, vista drivers usually work fine sense Windows 7 is mostly Vista fixed up.

If you had vista previously then you could try using the driver's from its cd... if you have a driver CD. Though I suppose these days you get stuck with only a Restore CD/DVD...

Also if you have installed Windows 7 after installing Ubuntu, you will have to reinstall grub.

### Post by MasterNetra on 2010-09-06
> **guthix0009 said:**
> i made the computer and bought all the parts and then got ubuntu and windows 7

Ah. Then you will have to nab the drivers from the parts manufactures, even more inconvenient. But if you haven't tried already, run update and check both the important and optional updates as MS may provide some of the drivers via update. Assuming you can connect to the internet in windows if not nab your ethernet/wireless drivers from the manufacture(s) while in Ubuntu and put the file(s) on the Windows Partition and install that/them first (All this assuming you can boot into windows period of course).

And of course if you installed Ubuntu first (big mistake) you will have reinstall/setup Grub(2) which I've already pointed you to a thread that links to some instructions on doing this from Live-CD.
