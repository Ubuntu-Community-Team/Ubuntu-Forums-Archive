### Post by halitech on 2009-09-19
the HD4200 is currently supported and if you are just using light apps and don't plan on using compiz then the 4200 will be fine. You can always start off using the onboard card and then in a few months when you have the cash you can get a better Nvidia card that will allow you to run compiz and give smoother graphics.

I currently have an X1200 onbaord and it runs fine for my uses but I don't like compiz and I'm not a gamer so for web browsing and movie transcoding it is fine.
