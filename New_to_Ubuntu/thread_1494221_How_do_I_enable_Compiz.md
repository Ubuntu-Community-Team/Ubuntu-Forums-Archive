### Post by anewguy on 2010-05-26
The first thing I would do is start Synaptic Package Manager, find the package "compizconfig-settings-manager", mark it for installation and then click apply.  After it is installed, right-click on a blank space on the desktop, select change background and go to the Visual Effects tab to enable them.  Then go to System/Preferences/CompizConfig-Settings-Manager to set things up how you want.
