No, there is no netbook version of 11.04.  Ubuntu Netbook 10.10 had the Unity interface but that is now the default in 11.04 so the netbook version is no longer required as such.

Unity is not very configurable in its current form however, so adjusting the size of side bars and window handles and so on is not trivial.  You might like to look at Xubuntu 11.04 as it's nicely configurable, and some of the included themes have larger scroll bars etc.  It also has a large application launcher at the bottom of the screen, so you might find that useful as well.

Chris.
