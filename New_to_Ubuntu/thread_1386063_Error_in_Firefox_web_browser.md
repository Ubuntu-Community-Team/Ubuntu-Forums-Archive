### Post by drascus on 2010-01-20
downthemall is not the problem I use it all the time. I don't know what you might have done to cause this but don't worry your using linux there is always a way to fix things. Try this go to your home folder use ctrl+h key combination. This will display your hidden configuration folders. now find the one that says .mozilla and delete it. This gets rid of your firefox configuration files. The default ones will appear when you start your next firefox session. This should fix your problem. Note: please back up any of your book marks and take note of your add ons and extensions as these will be deleted as well.

### Post by houseworkshy on 2010-01-20
Good advice above.  One can also go to System>Administration>Synaptic Package Manager  at which you will be asked to input your password.  It is always worth clicking the blue reload icon at the top left first just in case things have changed.  Then find the firefox package.  To make this easier click the "status" icon at the lower left and then choose "installed".  Right click on the package and choose reinstall.  Then click apply.  This can often mend that which is broken.
