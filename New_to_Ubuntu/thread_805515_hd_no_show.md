### Post by quelx on 2008-05-24
It looks like you need to change sdb1 to **sdb5** since it's an extended partition.

If you want to avoid future problems (with new drives and such) do this:

```
sudo vol_id /dev/sdb5
``` and copy the resulting UUID into your /etc/fstab

```
UUID=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxxx /home/nnjond/Desktop vfat auto,umask=0000 0 0
```

BTW are you *sure* you want to mount the drive at the Desktop?  You should mount it somewhere else, like a subdirectory of the Desktop folder.  so make a directory under the desktop

```
mkdir ~/Desktop/store
```

and mount the drive there

```
UUID=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxxx /home/nnjond/Desktop/store vfat auto,umask=0000 0 0
```
