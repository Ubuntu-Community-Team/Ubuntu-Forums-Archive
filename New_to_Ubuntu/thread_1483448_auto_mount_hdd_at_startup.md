### Post by -humanaut- on 2010-05-14

### Post by zeroseven0183 on 2010-05-14
**Storage Device Manager** is a good application to start with and it is in the Ubuntu Software Center.
You can launch it by typing pysdm in the Terminal after installation or select it from the Systems > Administration section.
