### Post by Zzl1xndd on 2010-08-26
The Lacie drives have always worked well for me on Windows, Linux and Mac.

I don't dual boot anymore but I never backed up the OS, just media that I didn't want to lose as its pretty easy to reinstall the OS. 

I find this makes it easy, no software, 1 partition and I can back up from either OS provided they can both right to the file system (I believe Ubuntu can now write to NTFS by default).
