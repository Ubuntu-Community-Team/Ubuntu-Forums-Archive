### Post by Cheesemill on 2012-05-06
You probably get encryption software that comes with the drive, and it's this that doesn't run on Linux.

Edit - From Amazon:
And to make sure all your important files are protected, SanDisk offers  the free SecureAccess software download, which adds a private,  password-protected folder to your drive.

It is this software that isn't supported.
