### Post by SuperSonic4 on 2010-02-18
0. Thank you for confirming my point that many people erroneously see ubuntu as GNU/Linux

Gnome and KDE are, among others, desktop environments - they consist of a Window Manager and Apps. Some distros install a lot of apps

Window Manager is what controls the window ssytems on your shiny GUI, they are lighter because they do not contain apps like a file manager or browser by default.

You don't need a GUI to run GNU/Linux though, it can run from a pure text interface.

Debian, Fedora, Slackware etc are distros, there are hundreds of distros but virtually all are based on an established distro - **mint** is based on **ubuntu** which in turn is based on **debian**. The reason why they are so many is because they are free to modify and distribute and because computer users are extremely varied.

I'd like to add Linux is written in 'C' and most of the bashie stuff is too (cp,ls,grep,chown), as opposed to maybe [MenuetOS]("http://www.menuetos.net/") written in 'machine' language... you've got the kernel+drivers (machine talking to 'C') then 'programs' (like these commands above) then ontop you've got the [X-Window System]("http://en.wikipedia.org/wiki/X_Window_System#User_interfaces") ...you'll understand the various stylings but maybe still wonder which compiz is?
