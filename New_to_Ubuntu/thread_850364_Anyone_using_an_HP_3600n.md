### Post by fheinsen on 2008-09-03
I have one, and it works great -- just make sure to use the "hpijs" driver instead of the default recommended "pxljr" driver, which has [a minor bug]("https://bugs.launchpad.net/ubuntu/+source/foomatic-db/+bug/228635") -- if you don't know how to change the driver, just follow the instructions I posted [here]("https://bugs.launchpad.net/ubuntu/+source/foomatic-db/+bug/228635/comments/3").
