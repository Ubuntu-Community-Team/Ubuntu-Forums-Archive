### Post by Baelus on 2009-11-19
You'll need the compiz config settings manager:

```
sudo apt-get install compizconfig-settings-manager
```

Once downloaded you'll find it under the system > preferences menu.

The plugin is called 'expo' and is found under the 'desktop' section.
