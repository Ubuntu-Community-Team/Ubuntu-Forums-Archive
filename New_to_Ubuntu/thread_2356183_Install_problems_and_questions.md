### Post by oldrocker99 on 2017-03-20
You're best off reinstalling. A / partition need be no bigger than 64GB, which is plenty of room. The rest of your SSD is then available for storage (I use my extra space on my SSD as a separate Steam partition).

When installing, select "Something Else,' which allows partition creation and resizing.
