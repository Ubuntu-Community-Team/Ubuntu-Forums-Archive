### Post by phrostbyte on 2010-03-17
Wine is able to run some Windows applications without problems, and many more with problems. It is not a complete solution for Linux-Windows interoperability at this time.

Check [http://appdb.winehq.org](http://appdb.winehq.org) for per-app compatibility.

If you require running Windows applications on Linux, look into virtualization. It is far more reliable.
