### Post by Michael.Godawski on 2009-01-10
hi, 
does the evince viewer show up in the start up section of System> Preferences > Session?
Also what exactly do you want to change? Turn evince off so to speak or just alter the subdirectory?

Besides... you have done a very large upgrade. It is a long way from dapper to hardy.
If everything else works then you are fine, but usually such an upgrade is causing some
minor/major issues.

Next time you install create a separate home partition so your data remains untouched during the next fresh install.
