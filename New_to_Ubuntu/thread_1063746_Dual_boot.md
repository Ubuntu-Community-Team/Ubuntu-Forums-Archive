### Post by houstonbofh on 2009-02-08
You will need to boot some things, so the easiest way to do this is with a USB CD-Rom.  But if you have a big thumb drive and a working 8.10 system, you can use an iso to create a boot thumb drive.  Use that (or the CD-Rom) to boot the Live CD, and shrink your XP partition in half, and install Ubuntu to the remaining half.  You may also want to find ext2fs for Windows so you can see your Linux files from Windows.
