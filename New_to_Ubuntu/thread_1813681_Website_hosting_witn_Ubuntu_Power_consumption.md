### Post by Paqman on 2011-07-28
No, in fact the latest Linux kernel in Ubuntu has some issues with excessive power consumption. That will probably be fixed shortly, but even when it does I would expect to see much difference.

You probably *could* optimise a LAMP server to use less power than a WAMP one, it depends how obsessive you want to be. Overall I'd say hardware, apps and load are going to make a much bigger difference than the OS.
