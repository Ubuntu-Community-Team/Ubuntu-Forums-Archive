### Post by Gone fishing on 2014-12-10
I would start in recovery mode, first check is whether you can access the GRUB2 boot loader. Boot your computer while holding the Shift key. You see a menu with a list of operating systems click on “Advanced options for Ubuntu” then boot in recovery mode. 

Will it boot to the menu? If it does will it continue to boot if you select resume? How about failsafe X? if it will not boot to the menu where does it freeze?
