### Post by Mentem on 2008-05-30
Yeah, there is a click-free way.
compiz config -> general options. On the "General" tab, there is "show desktop", click the box behind it, there you can mark which corner(s) will minimize the desktop(taking the mouse back to same corner will unminimize)

edit: there is also a "edge trigger delay" option on the same panel, this adjusts in milliseconds, how long you have to keep the cursor in the corner. I set it to 500 as I seemed to get a few unintended minimizations with the 0 setting.
