### Post by Iowan on 2009-01-28
[Internet/ConnectionSharing]("https://help.ubuntu.com/community/Internet/ConnectionSharing").
For file sharing, you have a few choices. 
[NFS]("https://help.ubuntu.com/community/SettingUpNFSHowTo") is the native Unix/Linux file sharing. 
[SSHFS]("https://help.ubuntu.com/community/SSHFS") is a secure method.
[Samba]("https://help.ubuntu.com/community/SettingUpSamba") lets you share files between not only Linux machines, but Windows as well.
