### Post by themusicalduck on 2010-09-02
Why are you trying to install directx9? If a game isn't running because of directx, then installing it probably won't help. Wine games generally use Wine's own implementation of directx (Direct3D or something).

You could try using Winetricks to install some directx9 dlls.

Open a terminal (Applications > Accessories > Terminal) and type in ```
wget http://www.kegel.com/wine/winetricks
``` then type ```
sh winetricks d3dx9
``` You can try to install directx9 by putting directx9 instead of 'd3dx9' but it isn't recommended.

Also, just type sh winetricks for a GUI list off all things winetricks can do.
