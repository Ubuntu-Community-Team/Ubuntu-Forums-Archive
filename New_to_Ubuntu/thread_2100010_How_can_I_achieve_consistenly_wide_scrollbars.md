### Post by bogan on 2012-12-31
Hi!, **T C**,

There is a program called Gnome Color Chooser that will, amongst many other things, allow you to alter scroll bar width. It is available on ubuntu Software Center.

Select the 'Specific' Tab and alter the Scroll Width from 15, [ I use 24] then 'Apply' and close.
