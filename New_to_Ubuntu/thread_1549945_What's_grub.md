### Post by Darkness Des on 2010-08-10
Grub is the Bootloader packaged with Ubuntu. 9.10 and 10.04 use Grub2, so it's already installed. If you want Korn, go install the *ksh* package in Synaptic. If you want this as your default shell, look into the *chsh* manpage and read up on it.
