### Post by Impavidus on 2013-05-20
I think LibreOffice is installed by default on Kubuntu, and else you can install it from the software centre with one mouse click and your password right after installation. LibreOffice also contains a spreadsheet program more or less compatible with MS Excel, but not everything will work perfectly. On the image processing, there is gimp, which to a large extend has the same features as Photoshop. It's pretty advanced. I sometimes use it, but I'm not a power user or how you would call it. Concerning paper/ink combinations, I don't really know, but this gets close to printer drivers. Not all printers have good driver support on Linux.
