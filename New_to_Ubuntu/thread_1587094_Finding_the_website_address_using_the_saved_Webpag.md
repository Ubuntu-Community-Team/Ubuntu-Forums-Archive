### Post by ankspo71 on 2010-10-03
Hi,
If it was from a long time ago, the website might have closed since then.

But...
Does it have a title? you can do a title search at google:
Example title search:
intitle:"Welcome to Ubuntu"

Does the site you saved contain any external links that you can search with?
Example Google search with 4 external links:
link:ubuntuforums.org link:google.com link:yahoo.com link:about.com
It's better if you use longer website addresses though.

Does it contain any images that you can perform a reverse image search with?
If so, check out [http://www.tineye.com/](http://www.tineye.com/)
You can upload the images there and it will search for sites that are still using it.

Also check the source code for any hints. Check the images for any hints in their exif data too. Like the author of the webpage and or images might be a good clue.
