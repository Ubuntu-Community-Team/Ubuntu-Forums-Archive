### Post by Toz on 2013-02-10
> The last part to report back my backlight interfaces. I do not understand how to do what you've told me to do there.
I was asking you to run the command string that I posted. It would reply with a listing of your backlight interfaces and there settings.

But...

I think you can fix your brightness issue by adding the **acpi_backlight=vendor** kernel parameters. Here is how:

1. Make a change to your grub config file. From a terminal:
```
gksu gedit /etc/default/grub
```
...enter password when requested.

2. Look for the line:
> GRUB_CMDLINE_LINUX=""
...and change it to read:
```
GRUB_CMDLINE_LINUX="acpi_backlight=vendor"
```

3. Save the file.

4. From a terminal window, run:
```
sudo update-grub
```

5. When the previous command completes, reboot.

This parameter tells the acpi subsystem not to take control of the backlight interface and to let the computer bios manage it.
