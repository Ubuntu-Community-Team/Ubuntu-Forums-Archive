### Post by deadflowr on 2012-12-24
12.10 works different from earlier releases, so to get the launchers back, open the dash type the name of the drive(partition) you want when it pops up in the list of stuff, simply drag it over to the dash.
Unlike earlier versions of Unity, all the icons in between the dash home icon and the trash icon can be moved.
