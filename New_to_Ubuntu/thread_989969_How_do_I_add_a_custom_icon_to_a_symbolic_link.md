Any format (even .ico), any size, just right-click on it and select *Properties*. You should see its default icon in the first tab, on the left. Click on the icon to select a new one.

Hope that helps.

Afterwards, to resize it to any size you want, just right-click on the symbolic link again and select *Stretch Icon*.
