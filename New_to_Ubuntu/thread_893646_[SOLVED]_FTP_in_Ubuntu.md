### Post by HalPomeranz on 2008-08-18
You can always do "Places -> Connect to Server...".  That will bring up a little dialog that will allow you to connect to a server via FTP (and lots of other protocols besides).  This will probably be easier for you than downloading a command-line FTP client and learning to use it.
