### Post by Vladlenin5000 on 2014-07-06
Assuming you download something like this - google-earth-stable_current_amd64.deb - just double-clicking (or right-clicking and selecting the option) will open it in Ubuntu Software Center. Otherwise you can install it with dpkg: cd into the directory/folder where you downloaded the file and ```
sudo dpkg -i google-earth-stable_current_amd64.deb
```
