### Post by snowpine on 2009-01-19
Same "core" system, different user interface, different pre-installed applications (but all have access to the same repository or library of applications).

Best way to experience it for yourself is to burn a Live CD of each and test them out.

PS this is a very frequently-asked question; do a little searching and you will find it has been answered many times. :)
