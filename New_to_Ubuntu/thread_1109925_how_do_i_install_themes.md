### Post by UbuntuNerd on 2009-04-17
right click on an empty space of your desktop scroll to the bottom and click on (Change Desktop Background you should see the themes,
 to install new themes in the same menu make sure you have (Theme) selected.
and just go to this link: [http://www.gnome-look.org/](http://www.gnome-look.org/) and download the themes to your desktop and drag the theme to the appearance Preference box to install it.

### Post by eeeandrew on 2009-04-17
Head for the Add/Remove button. Search for "Art Manager" and install it. Open it, click on art->desktop themes->application. This links to the art.gnome.org website. Also available through this program are wallpapers, splash screens, login screens etc.

Hope this is useful,
Andrew
