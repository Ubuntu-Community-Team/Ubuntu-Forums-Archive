### Post by Jesua on 2009-05-08
You can not install SAP B1 on Linux, maybe you could use Wine, but the installations requires a lots of MS SQL 2005 packages.

If you are trying to connect to a SAP server is more difficult, because you have to configure Wine to connect the Windows environment.

If you are trying to connect  or install the server and client, I think it could be more easy, but still you should work on the Wine and SQL 2005 parameters.
