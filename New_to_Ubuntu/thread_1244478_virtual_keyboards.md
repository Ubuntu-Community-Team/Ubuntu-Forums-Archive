### Post by TokyoYank on 2009-10-15
[LIST]
[*]"onboard" is the app that shows a clickable keyboard for Ubuntu 8.04 on up
[*]Double check for 8.04, but System > Preferences > Keyboard > Layout can map other countrys' layout over your keyboard
[/LIST]

... but if haven't yet started the class, I think you may find that it is preferred to type Chinese with the keyboard layout you already know.  (Inspecting briefly, seems Chinese keyboards only differ from US qwerty by function keys in the number pad)  Typing east asian languages can be tricky--though for me, once you get used to it, it's often easier than writing with pen and paper.  I know this more Japanese than Chinese, but there are thousands of Chinese characters, each has a sound, you type the sound using roman letters/phonetics (probably with a system called pinyin) and then press the space bar to confirm the character or character string.  

Check out [https://help.ubuntu.com/community/SCIM/Chinese](https://help.ubuntu.com/community/SCIM/Chinese)

Have fun
