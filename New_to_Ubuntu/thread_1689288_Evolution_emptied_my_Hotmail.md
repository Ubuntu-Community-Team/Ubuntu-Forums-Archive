### Post by seawolf167 on 2011-02-18
For future reference, make sure that when you set up a client email program (Evolution, Outlook, etc.), you go through the settings ***before ***your first connect and ensure that the program is correctly configured to "leave messages on server", else it downloads them and removes them from the server (i.e. deleting from your hotmail inbox)
