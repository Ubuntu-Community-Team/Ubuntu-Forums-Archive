### Post by 3rdalbum on 2011-12-01
How low a spec are we talking about?

The latest version of Ubuntu, 11.10, works on a PC with 3 GiB of hard disk space (absolute minimum - you'll need more) and 512 MiB of RAM.

If you have less than that, you will want to look at Lubuntu; it can run on less RAM.

If you're talking about a very low RAM computer such as one with 128 MiB or less, then I don't think any Ubuntu variation will work. You would need to look at lightweight distributions of Linux.

Ubuntu 10.04 is an old version of Ubuntu. I highly recommend using the latest version of Ubuntu/Lubuntu; if this is not possible, use the latest version of a different distribution that supports your computer. Don't use an earlier version of Ubuntu if you can avoid it.
