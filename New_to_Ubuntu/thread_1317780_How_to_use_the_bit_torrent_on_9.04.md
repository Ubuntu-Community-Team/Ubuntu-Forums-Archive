### Post by jasonsjunk on 2009-11-07
Everytime you find a torrent file you wish to download a prompt will open up and ask you if you want to download the torrent file or open it with transmission.  If you select open it with transmission it will start the bit torrent download process.  

To find torrent files you need to go to a torrent tracker.  I am not going to list torrent trackers here but you can easily find torrent files by typing (filename) torrent  into Google.  

Here is a good tutorial:

[http://www.ghacks.net/2009/01/12/downloading-torrents-with-transmission/](http://www.ghacks.net/2009/01/12/downloading-torrents-with-transmission/)
