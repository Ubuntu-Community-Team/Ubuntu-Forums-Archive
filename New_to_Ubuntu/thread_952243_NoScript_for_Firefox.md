### Post by OrangeCrate on 2008-10-18
> **dizzy1kenobi said:**
> **does anybody use NoScript for Firefox?**  Does it get past the annoying in text popups?  If not what is it useful for?

Like the other posters, yes I do.

I also use Adblock Plus, with the three EasyList filters. You can find the filters here:

[http://easylist.adblockplus.org/](http://easylist.adblockplus.org/)

(See the instructions in the left panel?)

Then I suggest that you install CustomizeGoogle:

[https://addons.mozilla.org/en-US/firefox/addon/743](https://addons.mozilla.org/en-US/firefox/addon/743)

Once installed, you'll find the interface under "Tools".

As you work your way through the options, pay particular attention to the "Privacy" settings.

Checking those two options stops Google from building a profile on you, and it also stops Google from sending any cookies to Google Analytics.

If you use those three extensions, your as safe as you will probably ever be on the Web, assuming that you're not running around barefoot amongst questionable sites.

### Post by OrangeCrate on 2008-10-19
> **dizzy1kenobi said:**
> I diddn't find the interface under tools is it somewhere else?  Also which 2 options were you referring to?

**After **you've installed the CustomizeGoogle extension, go to Tools at the top of  Firefox > "CustomizeGoogle Options". It should be the next entry under "Add-Ons".

If for some reason it's not there, you can go to "Add-Ons", highlight CustomizeGoogle, and access the options from "Preferences".

In the log list of options, starting with "Web", you'll see "Privacy". In there, you'll see two items with check boxes.

The first says:

"Anonymize the Google cookie UID"

The second one says:

"Don't send any cookies to Google Analytics"

Check both boxes.

### Post by ugm6hr on 2008-10-19
I use NoScript for the reasons already mentioned: internet security; speed of loading; ad and video blocking.

One thing to consider is that it also sometimes messes up online transactions.  You need to manually allow javascript on individual pages, so online surveys etc can get confused.  For example, buying rail / plane tickets online often use javascript to search for times and prices, which will not work until you allow it.  Additionally, some companies use external credit card services, which also need to be allowed.

Once you get used to it, it's dead easy.  But the initial confusion of having some sites not work straight away is why it's not default for everyone.
