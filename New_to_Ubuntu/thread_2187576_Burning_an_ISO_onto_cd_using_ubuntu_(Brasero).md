### Post by alphacrucis2 on 2013-11-12
If you are on windows you can burn the CD using things like imgBurn or Nero. If you are on Linux then you can use Brasero (as mentioned in your thread title) or k3b. On linux you can also do it from the command line using mkisofs for example.

Edit. If you are running Windows 7 or later I recall that it has a built in image burn program

### Post by Dennis N on 2013-11-12
If you have the regular Ubuntu download, it won't fit on a CD. You need a DVD. Brasero should have an option to burn the downloaded iso to the DVD. It might say burn a disk image, or words to that effect. Once burned, you can boot it.
