### Post by deadflowr on 2017-03-24
You can create a new archive in archive manager.
There should be a plus + option next to the Open button in the archive manager toolbar.
That should allow you to create a new archive.
Then just toggle the archive type to 7-zip and select the files to archive.
also ctrl + N should set up the create archive feature as well.

File roller will also launch within the Files program directly, so just right click on any file or folder should show both extract and compress options.
