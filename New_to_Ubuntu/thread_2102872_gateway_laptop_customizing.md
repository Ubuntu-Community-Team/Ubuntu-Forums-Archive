### Post by Frogs Hair on 2013-01-08
The chip socket must be compatible to the CPU,so first you have find out what mother board it is and weather it supports the CPU you want to use. The same is true with the hard-drive as far as sata connector compatibility is concerned.

Gateway like other manufacturers uses different vendors for mother boards. The other _important_ question is weather the bios supports the hardware.

Gateway tends to stop updating the bios and drivers after a period of time so the new hardware would be useless on that particular computer without bios support.
