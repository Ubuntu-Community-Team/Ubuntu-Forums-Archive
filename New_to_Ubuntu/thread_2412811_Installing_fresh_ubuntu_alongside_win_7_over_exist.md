### Post by yancek on 2019-02-17
It is certainly possible and basically what you would do is install Ubuntu to the same partition(s) you currently have.  You would need to format the Ubuntu partitions which can be done either before installing or during the install.  You will be better off using the manual installation method (Something Else).  There are many tutorials available online such as the one at the link below. 

[https://www.lifewire.com/ultimate-windows-7-ubuntu-linux-dual-boot-guide-2200653](https://www.lifewire.com/ultimate-windows-7-ubuntu-linux-dual-boot-guide-2200653)

One thing you should check is whether you have an EFI install which is unlikely and definitely not the default with windows 7.
