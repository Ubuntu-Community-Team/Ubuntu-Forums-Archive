### Post by Azdour on 2012-02-17
Hi,

I've never used the Software Centre to install Lightning. I usually install Lightning by downloading the correct version of Lightning for my Thunderbird from:

[https://addons.mozilla.org/en-US/thunderbird/addon/lightning/](https://addons.mozilla.org/en-US/thunderbird/addon/lightning/)

If you have Thunderbird 9 then I believe you need Lightning version 1.1.1

This will download a xpi file. Then in Thunderbird I go to Tools | Add-ons and use the Install... button (bottom left in the dialog). Browse to where you downloaded the file and select it. Thunderbird should do the rest including restarting Thunderbird so that the Lightning becomes enabled.

Hope that helps.
