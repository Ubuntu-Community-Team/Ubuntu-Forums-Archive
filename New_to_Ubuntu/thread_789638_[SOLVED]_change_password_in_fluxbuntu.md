### Post by dizee on 2008-05-10
Type ```
passwd
``` in a terminal. You will be prompted for your current password and then asked to type a new one.

Desktop icons are not provided by Fluxbox automatically, however there are lots of programs that can be run to give you them. One of these is Rox filer.
