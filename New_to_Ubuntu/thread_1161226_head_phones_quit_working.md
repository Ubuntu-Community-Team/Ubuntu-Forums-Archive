### Post by MontelEdwards on 2009-05-21
Well for your missing plugins problem you should install
```
 sudo apt-get install ubuntu-restricted-extras 
```
You enter this command in a Terminal which can be found by going to 
Applications>Accessories>Terminal
Just copy and paste that.


As for your headphones issue, you should try to open the volume panel and click prefences, and select Headphones and make sure that it's turned up.
