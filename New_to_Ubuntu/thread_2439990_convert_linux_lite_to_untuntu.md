### Post by Artim on 2020-04-05
If you liked Linux Lite and want to move "up" from there, I suggest Xubuntu because it has a similar desktop (Xfce), but *do a fresh installation by all means* instead of trying to "convert" Linux Lite into something else.  You'll lose all the cool tools and stuff, but if you no longer want them or need them, do a fresh installation of the distro you want with the desktop you want: Ubu, Kubu, Lubu, Xubu, etc.
