### Post by aysiu on 2008-07-28
I don't know if aMSN is in the standard repositories or not.

Go to GetDeb.net and download the appropriate Ubuntu .deb file:
[http://www.getdeb.net/app/aMSN](http://www.getdeb.net/app/aMSN)

Then double-click the .deb file to install it.

Pidgin is an instant messenger client. You should definitely give that a shot, too. See which you like better.
