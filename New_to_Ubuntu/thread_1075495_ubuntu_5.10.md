### Post by Joeb454 on 2009-02-20
You can only upgrade to the next release, so you couldn't download Ubuntu 8.10 and then upgrade to that from 5.10.

The only exception is 6.06 to 8.04 as they are both LTS releases, though I've never upgraded from one to the other :)

### Post by mcduck on 2009-02-20
Yes, but you can't jump over versions so you'd have to go from 5.10 to 6.06, then 6.10, 7.04, 7.10, 8.04 and finally 8.10.

(actually upgrading from LTS version to next LTS should work, so you should be able to only do 5.10->6.06->8.04->8.10)

Anyway it would be easier to simply download the latest version and make a fresh install with that. If you backup everything in your home directory, including all hidden files, your personal stuff and settings will stay.
