You can use the built-in tools in Ubuntu with the command line to create backups. Try these on for starters:
[http://ubuntuforums.org/showthread.php?t=35087](http://ubuntuforums.org/showthread.php?t=35087)

If they don't work, or if you're looking for something different, post back and let me know :)
