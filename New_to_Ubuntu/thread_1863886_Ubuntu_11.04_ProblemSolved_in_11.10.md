### Post by grahammechanical on 2011-10-18
I had the same problem in 11.04 when I tried to activate Wobbly windows. I have been using 11.10 on a separate partition for weeks but I have been too nervous to try to set wobbly windows again. Until now. I have just used Compiz Configuration Settings Manager and I ticked wobbly windows.

I got a bit of a fright. Things froze. The top panel went to a dark colour. I have it transparent. The browser window disappeared. The app-indicators in the top panel disappeared.

But then everything came back as before but the windows now wobble as they used to in 10.10. Things seem to work. The app-indicator menus work the Launcher appears and loads applications.

So far so good. I have not yet tried re-booting. Should work OK. Here's hoping.
