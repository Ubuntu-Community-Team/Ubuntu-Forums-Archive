I've got a startup script which gives options at startup, launch browser etc, what command or set of commands can be used to open dvd, select all and launch player?

---

### Post by cchhrriiss121212 on 2010-10-23
```
xine dvd://
```
You can replace xine with whatever your preferred player is, but make sure you specify what the drive is in the preferences menu. For VLC it is: 
Tools>Preferences>Input & Codecs>Default Optical drive
