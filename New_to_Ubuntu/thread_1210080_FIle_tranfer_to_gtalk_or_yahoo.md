### Post by Temposs on 2009-07-11
I don't believe gtalk supports file transfer. It's just not something that's done, because with Google stuff they expect you to trade files by email or using Google Docs. You won't be able to transfer files with gtalk no matter what program you're using on whatever OS.

In Yahoo I'm not sure what the issue is.

In any case, instead of using IM file transfer, as a general rule I think you should use some remote storage system and IM a url link to the file you've uploaded. This is much more reliable, whether you're using Windows or Linux or whatever.

Some services you could try for uploading your files are:
Ubuntu One
Box.net
Dropbox
Google Docs/Gmail
