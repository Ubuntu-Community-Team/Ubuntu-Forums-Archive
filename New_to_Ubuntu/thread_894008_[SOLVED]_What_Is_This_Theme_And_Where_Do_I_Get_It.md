### Post by jualin on 2008-08-18
I am having trouble finding that theme but you can always customize the one that you have since is only a color change. To do this go to System, Preferences, Appearances, Theme, select the "Glossy" theme and then press customize. Go to the "Colors" tab and select the "Windows" Background color. Press the "Pick color" icon and using the image that you submitted click on it to get the same color.
