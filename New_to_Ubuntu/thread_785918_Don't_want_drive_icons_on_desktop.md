I've used the NTFS configuration utility to add mount points so that my 2 NTFS partitions are mounted on startup. ### Post by bodhi.zazen on 2008-05-07
Applications->System Tools->Configuration Editor
        Go to apps->nautilus->desktop->volumes_visible flag
