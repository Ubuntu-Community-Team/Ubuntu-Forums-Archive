### Post by Stunt Double on 2009-05-21
I'm running 8.04 on 4 separate computers with CRT monitors. I find the best color accuracy and clarity is with totem movie player and totem-xine. Unfortunately, totem movie player has a problem with menus. That's when I use totem-xine which you can get from Synaptic Package Manager. It won't install in Applications-->Sound & Video, so use the Terminal. The command is: totem-xine.
   With VLC and gxine, I often have a slight yellow tinge that I can't get rid of no matter how I adjust the settings. Both Totems work perfectly on the defaut settings.
   When you insert the DVD, it will probably automatically start with totem movie player. Turn it off and start totem-xine.
   The more options you have, the better, so I always install VLC and gxine as well.
   Other than the occasional yellow tinge, VLC is a great player.
