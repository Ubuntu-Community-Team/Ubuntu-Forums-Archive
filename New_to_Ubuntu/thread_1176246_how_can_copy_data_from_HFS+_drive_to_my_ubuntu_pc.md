### Post by labinnsw on 2009-06-02
See if you are able to do this as super user. 
Open a terminal (Applications >> Accessories >> Terminal)
Type: ```
sudo nautilus
```
Supply your administrative password
Now browse to the files/data you want to copy and try again.

It is not advisable for you to keep this open after you are finished.
