### Post by Dennis N on 2022-08-11
For software, there is "Ubuntu Software". That is the application name. It comes as a snap package, with package name "snap-store". To start it from the terminal, you use the package name. There is no "software-center" command in Ubuntu 22.04.

But, why not start it from the "Activities Overview", or the Dock? (I think it is there by default). Using the Overview, just start typing the application name in the search box at the top, and you will get an icon to open it. The search is very flexible. You only need part of the name to get the correct result.

The "A" stands for "Applications".
