### Post by jimmy the saint on 2009-05-17
open the terminal.
in the menu bar: edit>profiles.  Add a new profile.  select the profile and click "edit"

under the colors tab you can change the colors to match your tastes.  You can also make it transparent, change the font and all kinds of things about the way it looksa and operates.  

Juat make sure that you select the new profile in the dropdown box that says "Profile used when launching a new terminal"

### Post by charliehorse55 on 2009-05-17
There sure is. Open a new terminal window and go to 

Edit > Profile Preferences > "Colors" Tab

Then select the color preset "White on Black"
