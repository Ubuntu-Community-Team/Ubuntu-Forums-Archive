### Post by mlentink on 2008-08-12
Usually, when you start up your machine, it flashes a message on your screen about system setup and 'boot options'. The latter is what you need. It could be F10, but I've heard about Del as well.
If you press that button during boot, the machine will allow you to choose which device to boot from. Select the CD-ROM (Optical drive) with the Ubuntu Live CD in that drive and Ubuntu will start.

### Post by lukjad on 2008-08-12
Ummm... Ubuntu is not really a program, at least not in the way I think you are thinking. It is an operating system, with it's own programs. If you wish to dual boot or install Ubuntu instead of Windows, you need to leave the Ubuntu CD in the CD-ROM drive and reboot your computer. When you do so you can then choose to test it out with Wubi, or try it as a live CD without making any changes to your computer, or you can also install it on your computer, either as a dual boot or as the only operating system on your computer. If you choose to do install it, I suggest backing up your files first, to an external hard drive or burning them to disk to be safe.
