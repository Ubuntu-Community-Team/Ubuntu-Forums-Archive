Yep thats conky.

Edit -> To install just open a terminal and type in ```
sudo apt-get install conky
```
It won't look like it does in your picture however because it runs on user made scripts. You can look at some scripts others have made here. [http://ubuntuforums.org/showthread.php?t=281865&highlight=conky](http://ubuntuforums.org/showthread.php?t=281865&highlight=conky)
