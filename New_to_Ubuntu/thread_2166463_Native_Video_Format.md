### Post by SweetAurora on 2013-08-09
The default media formats with no proprietary codecs installed, would be the open source/expired patents ones. Like OGG Theora, WAV, MIDI, ect...

It is STRONGLY recommended that you install extra codecs to get a better functioning system. Here, just install this, but only if the laws of your country/area allow such things as the package sits in a grey area that varies from place to place.
```
sudo apt-get install lubuntu-restricted-extras
```
