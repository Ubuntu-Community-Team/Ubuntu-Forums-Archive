Transmission is a real lightweight torrent client.  By default it will run as your torrent client.  Under edit and preferences you can set your upload and download speed limits as well as setting a TCP port for it.  So as far as Torrent clients go, no real manual is necessary unless you are new to the world of torrents in general.

### Post by mcduck on 2008-06-27
> **johanus said:**
> I notice, too, that 8.04 installs version 1.06; the current version is 1.22.  Is that large a version lag unusual?
The basic rule is that after a certain pahse in the development of Ubuntu releases all software versions are locked, and they will not get any updates for other than security reasons and to fix serious bugs.

In some cases programs _may_ be updated, if they have little or no dependencies, and the update is simple to do & unlikely to cause any problems. Even in these cases it will take some time before new packages are built and tested. Typically after being built they are kept in the "proposed"-repository for at least couple of weeks before moving them to normal repositories.

This is to get a nice compromise between stable and reliable system and up-to-date programs.

I wouldn't expect any updates to Transmission. At least unless some secrity issue pops out..
