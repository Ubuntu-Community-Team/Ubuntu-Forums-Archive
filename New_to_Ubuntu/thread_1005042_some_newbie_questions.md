### Post by cobra741 on 2008-12-07
Did everything work for you when you booted up the live cd? 
If so it'll certainly work fine for you when you do a full install. 
make sure you check things like wifi, webcab, usb, display (does the live cd run in your laptops native resolution), etc
don't expect all of your Fn keys to work though :)
