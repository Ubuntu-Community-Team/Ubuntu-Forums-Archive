### Post by JustLikeYouImagined on 2010-09-11
I downloaded an icon theme tar ball. It said to "extract here" and then to run a script. I did all that. Now what do I need to activate the icon theme? Please help.

---

### Post by inameiname on 2010-09-11
If it installed correctly via the install script, just open Applications -> System -> Preferences -> Appearance and click 'Customize' and then click the tab 'Icons' and select your icon theme.
