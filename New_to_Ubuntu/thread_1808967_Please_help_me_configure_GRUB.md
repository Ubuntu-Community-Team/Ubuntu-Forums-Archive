### Post by Bucky Ball on 2011-07-21
Open a terminal (Applications>Accessories) and paste in these commands one at a time:

```
sudo add-apt-repository ppa:danielrichter2007/grub-customizer
sudo apt-get update
sudo apt-get install grub-customizer
```Wait for each command to do its thing. Once all done, go to *Applications > System Tools > Grub Customizer*.

From [THIS]("http://www.webupd8.org/2010/11/grub-customizer-20-can-change-default.html") page. Excellent grub tool it is ... ;)

### Post by Bucky Ball on 2011-07-21
Cool. The command, 'sudo apt-get update' is basically the same as running the update manager. It adds the contents of the repository '
ppa:danielrichter2007/grub-customizer' to your software sources (the only app in that repo being grub-customizer).

'sudo apt-get install grub-customizer' is just a terminal way of installing as opposed to opening Synaptic Package Manager, searching for grub-customizer and installing that way. ;)
