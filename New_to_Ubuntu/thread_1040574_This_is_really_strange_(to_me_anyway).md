### Post by HermanAB on 2009-01-15
WiFi requires the program for the little Arm processor in the WiFi modem to be downloaded and rebooted each time you turn the system on.  Finding that it suddenly starts to work after a system restart is not unheard of - it means that there is a mild hardware bug or a timing problem and sometimes the little Arm starts up properly and sometimes not.

The newer laptop evidently has hardware that is not properly supported by Ubuntu.  Do not despair, but try a different distribution: Fedora, Mandriva, Suse, Knoppix...  Chances are that one of the other Linux distributions will work.  (There are bazillions of different systems in the world and each distribution house has limited test facilities).

Cheers,

Herman
