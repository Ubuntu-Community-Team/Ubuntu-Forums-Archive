### Post by Dennis N on 2020-02-20
> weird bug (fade out fades to black but then briefly flashes the screen before becoming completely black)
I haven't noticed that. Did you use the correct command in the startup applications? Otherwise, it may briefly flash or show a spash image when it comes on - that might be what you are seeing. The command should be:
```
xscreensaver -no-splash
```
A updated version most likely wouldn't appear in the Ubuntu repository until the next Ubuntu release - 20.04 - or maybe not. Most applications don't get updated mid-release unless it's a security issue. Linux kernel is frequently updated for this reason. But there are alternate ways to get newest versions of some packages: PPAs, Snap packages, and Flatpak packages. There is a PPA at [Xscreensaver PPA]("https://launchpad.net/~savoury1/+archive/ubuntu/xscreensaver") but I don't see Snap or Flatpak releases. If you are willing to leave Ubuntuland, some Linux distributions will continuously update packages with newer versions - these are the rolling releases.
