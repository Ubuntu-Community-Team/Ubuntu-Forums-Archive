### Post by jespdj on 2009-04-26
Yes, that is possible, and it is even the default way Adobe Flash works on 64-bit Ubuntu (at the moment).

If you have 64-bit Ubuntu and you install the package **flashplugin-nonfree**, you'll get the 32-bit version of Flash installed with an adapter (nspluginwrapper) to make it work in 64-bit Firefox.

You can also install the native 64-bit Flash player from Adobe, but it's a pre-release version at the moment.
