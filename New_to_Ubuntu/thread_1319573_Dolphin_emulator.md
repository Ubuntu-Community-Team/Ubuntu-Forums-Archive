### Post by glennric on 2010-01-14
I don't know if you are still interested, but I have created packages for the dolphin emulator.  They are in my ppa.  You can add the repository and install the emulator from a terminal with
```
sudo add-apt-repository ppa:glennric/dolphin-emu
sudo apt-get update
sudo apt-get install dolhpin-emu
```
