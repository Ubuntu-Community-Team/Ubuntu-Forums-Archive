### Post by mkvnmtr on 2009-03-19
I will start by saying I know little about working with videos. What I do know is that ubuntu studio should already have everything you need to do your work. Anything you might need will be in the Synaptic Package Manager. Any software you encounter on the web will probably be for windows and won't work for you. The problem you face is the programs have diferent name than you are used to. As you want to do something like download your video nfrom your camera just do a google search and include the name Ubuntu. You should get some ideas how to proceed from there. Then come to this forum with a question how to do that one thing. somebody will know and help you.
On the other front you might have windows installed onn your computer. when you start up a screen should come up with a list of systems. It will only be there for a short time. The first on the list will be Ubuntu. If you press the down key it should stop the boot and give your time to look at the list and see if you have a windows os. If you do you can use the up-down keys to highlight it and then press enter to boot it.

If you want to work with video you will find you made a good buy with Ubuntu studio and you will never have to pay for software.

### Post by Paqman on 2009-03-19
On Ubuntu you don't normally install software the same way you do on Windows.

Instead of downloading it and having to install it manually, you can go to System > Admin > Synaptic Package Manager. It will automatically install any software you want, including dvgrab. It gets this software from centralised repositories, so it's all tested and safe.
