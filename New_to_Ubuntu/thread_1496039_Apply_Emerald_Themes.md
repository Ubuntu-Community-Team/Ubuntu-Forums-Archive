I *think* to go back to defaults it would be metacity --replace.  If you want to go back with compiz (which I thought replaced emerald itself so an emerald --replace shouldn't work that I know of), it would be compiz --replace.

Be sure to install the compizconfig-settings-manager package from Synaptic Package Manager, then check there for things you may want to do.
