### Post by starcannon on 2008-11-18
The Wubi install will not harm Microsoft Windows performance at all.
While a regular install will be slightly faster when running Ubuntu, its not enough to be a deal breaker. Go for it, if you don't like the results it's as easy as running the Windows Add/Remove utility to make it go away.
