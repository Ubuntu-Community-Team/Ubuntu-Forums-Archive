### Post by Shazaam on 2008-08-29
Once you have it installed it will usually be put in the Applications menu. If not, you can add it manually as long as you know the command to run the program. Either right-click Applications and choose "Edit Menus" or go to System>Preferences>Main Menu.
As an example Firefox starts with this command....
```
firefox %u
```
or just...
```
firefox
```
