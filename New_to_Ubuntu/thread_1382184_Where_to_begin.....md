### Post by doublewitt on 2010-01-16
I recommend Kontact + KOffice Workspace + Konqueror... all 3 combined is unbeatable... 

Kontact offers an intersting email application (KMail), great calendaring, journal, rss feed reader, notes, contacts, todo management, etc.

KOffice Workspace offers a "window" for managing and launching applications such as: KWord, Spreadsheet, Drawing, Flowcharts, Graphics, Slide Presentations, Project Management, etc. all conveniantly brought together in a panel. 

Konqueror is a great browser and file manager that lets you setup your home page with the bookmark folders for quick and conveniant access - a nice way to organize your URL collection.

You can also include Knotes (stickies). You can put the Knote file in a "dropbox" or "ubuntu one" folder and turn Knotes into a powerfull on-line sticky notes program. This makes a nice addition to the complete office package.

Anyways, many different opinions you will get! 
Try them all and see what's best for you...
