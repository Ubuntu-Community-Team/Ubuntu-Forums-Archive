### Post by jamesrl on 2008-10-05
I don't believe totem supports bookmarks within the program totem.  

You can create a Launcher (shortcut) to it in the usual way (right click on desktop, use command
```
totem /home/jamesrl/Music/Contemporary\ \(Modern\)/Iron\ \&\ Wine/Our\ Endless\ Numbered\ Days/12\ Passing\ Afternoon.m4a
```
with the actual file location instead of something that's on my harddrive.

However that might not be ideal for you.  

You could create a bookmark to the file within firefox (or your default web browser) that when you go to it launches totem to play the media file.  Just type in the location of the file (minus file) right click on the file, bookmark this link.  You should have a prompt of how to open the file from firefox, and for me totem (Movie Player) is default for a wide range of files.
