### Post by Thingymebob on 2008-05-12
Can you post your .xsession-errors and  /var/log/gdm/:0.log
from immediately after a failed login and a successfull login.

To do this:
after a failed login, hit ctrl+alt+f1 to open a vt and login
type
```

cp .xsession-errors ~/Desktop/xsession-errors.failed
sudo cp /var/log/gdm/:0.log ~/Desktop/gdm0:log.failed
```
do the same after a successful login (just replace failed wih success)
```

cp .xsession-errors ~/Desktop/xsession-errors.success
sudo cp /var/log/gdm/:0.log ~/Desktop/gdm0:log.success
```
