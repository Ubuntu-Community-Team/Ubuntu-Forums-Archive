### Post by Mark Phelps on 2010-12-27
From my experience the relative success of doing this is hit-and-miss, meaning, some packages will convert completely, some only partially, and some, not at all. It depends on the contents of the individual rpmm package and what libraries it relies upon.

So, relying to rpm packages in a debian-based package system environment is highly risky.

As others have said, you would better looking for a .deb package instead of using alien.
