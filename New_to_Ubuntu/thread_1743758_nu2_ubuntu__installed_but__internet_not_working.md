It would help if you told us what wireless chipset the system is using, If you don't know, open a terminal and type:

```
lspci > lspci.txt
```

the above command will create a text file called lspci.txt, you should be able to transfer the file to another computer with internet access via sneaker net, paste the results in your next post.
