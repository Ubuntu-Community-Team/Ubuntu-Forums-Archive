Also a Wubi install can go bad after a while. Wubi installs ubuntu 'inside' windows, and can be pretty unstable after an upgrade or any length of time. If you like ubuntu and want to use it long term it's best to do a 'real' dual boot setup. Lots of documentation and information on the forum about that :-)
