### Post by asmoore82 on 2009-05-23
Linux/X11 still doesn't have a "unified" global clipboard AFAIK,
you will be unable to "Paste" if you have subsequently closed the App you "Copied" from.

The reason for this is because in X11 you can "paste" with the middle mouse button - "clicking the wheel" on modern mice.
You can just highlight text in one place and middle-click to paste it anywhere else -
this method works globally for all X Apps automagically so
it's more consistent for the user and it's less work for developers 8-).
