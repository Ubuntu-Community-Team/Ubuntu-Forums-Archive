### Post by buzzingrobot on 2015-08-07
When you boot an Ubuntu installation image, you should see a display asking if you want to try Ubuntu or install Ubuntu.  If you choose the "Try Ubuntu" option, Ubuntu will run and you can, in fact, try it out.  Firefox is there and you can use it as long as the internet connection is active.  You can begin the install from there, too.  

Here a howto about the installation: [http://www.ubuntu.com/download/desktop/install-ubuntu-desktop](http://www.ubuntu.com/download/desktop/install-ubuntu-desktop).
