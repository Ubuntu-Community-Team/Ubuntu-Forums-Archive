### Post by dave-6 on 2014-01-26
Found answer.

This looks like the case where you're trying to run a 32 bit ELF binary, on a 64 bit install, without having the 32 bit libraries installed. If that's the case, installing the package named ia32-libs should let you run this adb binary.

ia32-libs has no candidate.  That answer here: [https://stackoverflow.com/questions/19523502/androids-aapt-not-running-on-64-bit-ubuntu-13-10-no-ia32-libs-how-can-i-fix](https://stackoverflow.com/questions/19523502/androids-aapt-not-running-on-64-bit-ubuntu-13-10-no-ia32-libs-how-can-i-fix)

sudo apt-get install libc6-i386 lib32stdc++6 lib32gcc1 lib32ncurses5
sudo apt-get install lib32z1
