### Post by TeoBigusGeekus on 2011-03-20
...or, if you prefer the command line, you can erase cdrws with wodim.
```
wodim -v dev=/dev/cdrw -blank=fast
```
Replace /dev/cdrw with whatever applies to your system (/dev/dvd, /dev/scd0, /dev/sr0, etc).
