### Post by Joeb454 on 2008-05-13
You can create them, open up System > Preferences > Keyboard Shortcuts.

Scroll down and there is an option for "Run a Terminal" :)

### Post by Cypher on 2008-05-13
You would use ALT-F7 to return to the GUI, if you are using Kubuntu, then you can hit ALT-F2 and then type in 'konsole' to get a terminal. I forget what the Ubuntu console application is, but that's what you'd enter as the command.

The other option is to just click on the Top panel (near the Firefox logo) and choose to add an application/applet and go to the Applications->Accessories and choose Terminal and get a button for it..
