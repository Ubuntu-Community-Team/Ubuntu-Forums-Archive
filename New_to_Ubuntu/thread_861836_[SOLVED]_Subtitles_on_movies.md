### Post by RomeReactor on 2008-07-17
Hi. Name the subtitle file the same as the video file and put them in the same folder; and in Totem, go to 'Edit->Preferences' and on the 'General' tab check **Automatically load subtitle files when movie is loaded**. Totem should pick it up automatically then. Example of both files in the same folder:
```
my_movie.avi
my_movie.srt
```

### Post by Dani Filth on 2008-07-17
You can google : 
> the_movie_name + subtitle
Sometimes when there are several groups release their movie on the Internet, you may want to add the group name as well - normally, this goes together with the .avi filename in [] or (). For instance : 
> movie_name + subtitle + group_name

Because different groups release different versions of movies, e.g : 1 CD (700MB) or 2 CDs (1.4GB)
