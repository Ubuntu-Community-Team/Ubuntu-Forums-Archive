### Post by gandaran on 2011-06-15
just look for flash in software center and install it, or terminal install
```
sudo apt-get install flashplugin-installer
```
if you are using 64-bits Ubuntu don't install this package, install the mozilla [flash-aid]("https://addons.mozilla.org/en-US/firefox/addon/flash-aid/") addon to help you install 64-bits flash.
