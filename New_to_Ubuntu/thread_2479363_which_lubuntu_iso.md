### Post by sudodus on 2022-09-23
I created the software at [https://phillw.net/isos/linux-tools/9w/](https://phillw.net/isos/linux-tools/9w/) several years ago. You can probably use software from it in an old Pentium 3 computer, but please be aware that you might get malware via the internet, because there are no updates, not even security updates.

It is probably a better idea to try **Tiny Core** or **Puppy Linux**, which are *current* ultra-light Linux distros. Then you should be able to connect to the internet, but be aware that many current web pages are too heavy for a computer with only 384MB RAM.

[hr][/hr]
The current Lubuntu iso files are available via [https://lubuntu.me](https://lubuntu.me).

All current Ubuntu family iso files are available via [https://releases.ubuntu.com/](https://releases.ubuntu.com/)
