### Post by Methuselah on 2008-06-24
There is always a certain danger to flashing the BIOS.
Be warned that your computer could be bricked.
Unless, there's a specific feature in the updated BIOS that you need, maybe you shouldn't bother. 
You know, an "it aint broke don't fix it", type of thing.

Anyway, since the actual BIOS flashing will be taking place outside of either linux or windows, it shouldn't matter if you build the bootable media on windows. It's akin to burning an Ubuntu disc in windows to boot from and install ubuntu. It doesn't really affect the content of the disk.
