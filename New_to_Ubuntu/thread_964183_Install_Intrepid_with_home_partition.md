### Post by hellion0 on 2008-10-30
On paper, that looks like it will work.

Make absolutely sure when you copy your old /home folder that all hidden files and folders are copied as well. Most of these hidden folders and files govern the configurations of the programs you use, and without them the programs will end up in a default state.
