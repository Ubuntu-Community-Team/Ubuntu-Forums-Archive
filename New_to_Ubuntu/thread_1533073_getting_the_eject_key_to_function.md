### Post by Chris1274 on 2010-07-17
I'm using a Dell Inspiron 1440 notebook. It has a key to open the CD drive, but I'm not sure how to get it to work in Ubuntu. It works when I boot up win7 because I have the appropriate driver installed; is there a way I can install the driver so it will work when I have Ubuntu running? It's not crucial because I can open it using the button on the tray, but the key is a bit more convenient.

---

### Post by sisco311 on 2010-07-17
Create a new keyboard shortcut (System -> Preferences -> Keyboard Shortcuts). In the command field type:
```
eject -T
```
