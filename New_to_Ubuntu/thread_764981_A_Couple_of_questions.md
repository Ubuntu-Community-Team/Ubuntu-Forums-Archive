### Post by swoll1980 on 2008-04-24
If you use a virtual machine you can switch fast but because it runs over top of vista you wont get full performence out of the ubuntu install. Ubuntu will automaticly read and mount the installed hard drive if installed normaly as a operating system.
Another thing to add as well if you install ubuntu in a virtual machine you wont get any of the 3d effects
