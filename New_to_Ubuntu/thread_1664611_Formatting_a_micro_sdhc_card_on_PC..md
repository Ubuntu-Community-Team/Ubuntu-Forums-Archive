### Post by C.S.Cameron on 2011-01-11
Welcome to the forums.

Go to Applications/Ubuntu Software Center.
Type gparted in the search box.
Install Gnome Partition Editor.
Plug in your 4g micro sdhc card.
Start System/Administration/Gparted.
Select the sdhc card at top right.
Right click the unallocated space.
Select New
Select FAT32
Click green arrow at top
