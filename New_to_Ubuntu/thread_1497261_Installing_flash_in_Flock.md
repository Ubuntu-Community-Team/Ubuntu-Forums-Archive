### Post by gandaran on 2010-05-30
do have firefox and flash installed? flock usually picks up the flash plugin installed in firefox, no need to do anything but if it wont work then you can download the tar.gz package flash plugin (or copy the libflashplayer.so file from the already installed flash) from adobe.com and just throw the plugin (libflashplayer.so file) in flock's plugin folder either in the flock root directory or user.
