### Post by taseedorf on 2009-02-26
Giving Ubuntu the whole drive means that the Operating System (Ubuntu) has access to the whole hard drive instead of just a portion (a partition).   This is good if you plan to only run Ubuntu, however, if you wanted to run Windows too, you'd want Ubuntu to only use part of the drive.

You should NOT run ANY games off of an external hard drive, atleast, not any games such as COD4 or any 3D games.....  If you have a very small hard drive, just buy a large internal (they are so cheap now)  and also an external.  

A GOOD wifi card for linux will be an atheros one, since they are well supported.  I'd suggest NOT getting a Broadcomm card.  However, I have one and have gotten it to work.

As far as video cards go.... most nvidia cards are well supported..same with ati...newer nvidia or ati are probably.  i know pinnacle tuner's though, don't like linux.

Also, instead of using "ubuntu" try Xubuntu (it is a lot more lightweight, and uses less resources) .....it's, for the most part, the same as Ubuntu with out Gnome and some apps ( you can install gnome too and all the apps if you want)

Backend, Frontend, debian, GNOME, deb, wubi

Backend is the workhorse computer........front end is your control station......... Debian is not backed by a company like other Linux distros...however, it is a basis for many of the company backed distros, like Ubuntu....

Gnome is a desktop enviroment that changes the way your computer looks (icons, wallpaper, widgets, etc)

deb is short for a debian package thatis basically a package used on debian based systems that will install a program.   think of it as an installer for a program

wubi is a special installer for booting into linux from windows
