### Post by Ocxic on 2009-05-04
No.

If you want to install to the blank HD the best, easiest and fastest way to do that is to burn the ISO to CD and install that way(or you could use a USB stick).
If you try to use virtualbox, the install will setup and use the virtualized hardware provided by vbox and not your computer, not to mention its a little difficult to get virtualbox to use a hard drive natively, and the boot loader may not get installed correctly
