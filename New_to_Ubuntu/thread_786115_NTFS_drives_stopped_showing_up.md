### Post by daimaru on 2008-05-07
not sure if this is your problem, but if windows does not shut down clean or if it goes into hibernation, suspension the ntfs drives will not mount on ubuntu. so if you had to do a alt+ctrl+del or hit the reset button last time you were in windows, or if you're on a laptop and it switched of via hybernation, then that is the problem. jus boot into windows again shutdown cleanly and the drives should show up again in ubuntu. 

hope that helps.
