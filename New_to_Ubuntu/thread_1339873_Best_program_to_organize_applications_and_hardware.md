### Post by japhyr on 2009-11-28
> As well as when opening specific files is there a program I can tell to universally open extension "x" with application "y" automatically?

If you right click on a file and choose "Open With", it just opens in that program this one time.  If you right click on a file and choose Properties, then go to the "Open With" tab, the program you select will always be used to open files of that type.
