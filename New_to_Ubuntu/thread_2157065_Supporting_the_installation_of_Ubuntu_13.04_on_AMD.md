### Post by grahammechanical on 2013-06-24
I have been running Ubuntu on a dual core processor since Ubuntu 7.04 and I have run every version since then including 13.04. If there is any issue with Ubuntu not being able to make full use of a processor's capabilities, then the responsibility lies with the developers of the Linux kernel and the issue would affect all Linux distributions.

What we do not have in Linux is utilities provided by the motherboard manufacturers. So, some utilities that are available in some other OS are not available in Linux. For example, the motherboard came with a Windows utility to update the BIOS from within Windows. But I did not get a Linux version of this utility. So, I cannot update the BIOS from within Ubuntu. This is a different matter entirely.
