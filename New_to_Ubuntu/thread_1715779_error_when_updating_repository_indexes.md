### Post by fabricator4 on 2011-03-27
> **bob brazie said:**
> Could you tell me where/how to find a mirror and how to have software sources see and retrieve the site/information?

If you go to "Software Sources", down the bottom there's a selection for "download from ..."

If you go to "other" you can select a server closest to you, or select "find the best server" (or words to that effect).  The system will try to find the server with the best response times for you.

If it finds something unusual, like a distant server then you might want to revisit this when the network is not so congested, or when whatever problem is causing the issues has been fixed.  Usually, the "best" server will be the one that is closest to you.

Chris.
