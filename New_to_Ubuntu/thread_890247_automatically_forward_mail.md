### Post by halitech on 2008-08-14
I believe this will work

Go to tools - message filters

create a new rule

select match all messages

perform these actions - change to Forward Message to

enter the email address you want to forward it to

click okay

I believe this will only work if Thunderbird is running. Only other way I know of would be to configure the mail server to do it
