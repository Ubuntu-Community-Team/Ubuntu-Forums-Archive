### Post by cavh on 2011-07-20
1. Download the custom installer package from [http://www.myeclipseide.com/index.php?name=Recommend_Us&req=Download&version=ME]("http://www.myeclipseide.com/index.php?name=Recommend_Us&req=Download&version=ME")

2. in a terminal window, cd to the directory where your downloaded package is. For example, if your download location is /home/<yourname>/downloads, type this in the terminal:

```
cd ~/downloads
```

(the ~ symbol is a shorthand way of saying 'the home folder of the person logged in')

3. make the installer file executable by typing this in the terminal window and hitting the return key

```
chmod u+x myeclipse-9.1-online-installer-linux.run
```

4. run the installer by typing this in the terminal and hitting the return key:

```
./myeclipse-9.1-online-installer-linux.run
```

Please note that the '.' and the '/' are important!

5. follow the installer prompts
