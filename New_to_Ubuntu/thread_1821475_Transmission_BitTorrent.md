### Post by Alwimo on 2011-08-09
Left-click on a link that has ".torrent" at the end. A window should appear asking you to either "open with" or "save" it. Open With Transmission should be an option. You then just have to confirm that you want to add it. If it's the first time you have opened transmission, you will have to say that you agree to take responsibility to use it legally.

Alternatively, download the .torrent file and save it, then and open that file using Transmission, and follow the instructions there.
