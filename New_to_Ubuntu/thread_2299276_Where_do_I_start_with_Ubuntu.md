### Post by grahammechanical on 2015-10-17
You say that you are new to Ubuntu. Does that mean that you have installed Ubuntu? If you have installed Ubuntu or if you are running Ubuntu from a DVD disk or a USB memory stick (Try Ubuntu session) then click on the power/cog icon that is at the far right of the top panel. And from the drop down menu select Ubuntu Help. And read about the Ubuntu user interface.

Learning Ubuntu and learning Linux are two different subjects. The purpose of Ubuntu is to allow us to use our computers without needing to learn Linux commands. I have learnt enough about Ubuntu to be able to use it to do all the computer tasks I want to. I have learnt only a little about Linux. I have learnt from experience how easy it is to break the operating system and how hard it is to fix it once it has been broken.

I have learnt to be ready to reinstall Ubuntu by having all my data backed up and on a separate partition. I have also learnt not to experiment with the only operating system I have on the machine. My advice is to have a second installation of Ubuntu to practice making modifications on or to practice running Linux Commands.

I have also learnt things by being a regular visitor to Ubuntu forums and reading about the problems that other people have and learning from the advice that is given by other forum members. Here are some other places I use as a source for learning Ubuntu.

### Post by Bucky Ball on 2015-10-17
Welcome. In an effort to help you along I have changed the title of this thread to reflect your support request rather than 'Please help me', which gives potential helpers no info about your support question. :)

Good luck and hope you enjoy the journey. I'd start by downloading the ISO, creating install media, booting from that and selecting 'Try Ubuntu'. This will let you run it from the install media without installing so you can have a look around and see what you think.

Who knows, perhaps a regular Ubuntu might not suit you. There are other flavours to try: Kubuntu, Xubuntu, Lubuntu, Ubuntu Mate, etc. :)

... and keep asking questions when you hit brick walls.

[Download Ubuntu.]("http://www.ubuntu.com/download/desktop")

14.04 LTS is the current long-term support release, supported until 2019, 15.04 is the current interim release, supported until January 2016. I'd suggest 14.04 LTS.
