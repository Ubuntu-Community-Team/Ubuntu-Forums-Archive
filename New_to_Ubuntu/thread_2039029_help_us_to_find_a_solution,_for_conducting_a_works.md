### Post by Dylan1473 on 2012-08-07
A VPN is not necessary. I'm pretty sure tools are included with Ubuntu for this type of thing. In fact, [here]("http://www.ehow.com/how_6802096_enable-remote-desktop-ubuntu.html") is a guide which you will hopefully find helpful. All those viewing will need some kind of viewing client. You shouldn't have any trouble finding one though and I think Ubuntu might include that as well. If not, I've used Remmina remote desktop client in the past and I think it offers that feature.

EDIT: [TightVNC]("http://www.tightvnc.com/"), a free and open source VNC viewer that will work with Windows.
