### Post by khelben1979 on 2011-01-03
Reinstall Ubuntu.

There is a possibility to have both Ubuntu and Windows on the same harddrive, but I would recommend to keep it on seperate harddrives to avoid any trouble which can occur which relates to the [MBR]("http://en.wikipedia.org/wiki/Master_boot_record").

If you're really serious in learning the Linux operating system, you can go to a local library and get yourself a manual to understand the basics of the system, otherwise there is lots of documentation on this forum: [Ubuntu Community Documentation]("https://help.ubuntu.com/community").
