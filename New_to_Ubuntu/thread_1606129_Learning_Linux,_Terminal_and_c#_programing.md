### Post by Jerry N on 2010-10-26
> **punkphotoman said:**
> The main appeal to me for Linux, is the speed increase, but I can't imagine shelving photoshop for GIMP.  

Don't assume that everything people tell you is correct.  I have experimented with a lot of Linux versions and have yet to note a significant speed increase over Windows XP or Windows 7.  Later versions of Ubuntu (and Mint) do boot faster than Win XP but I can't consider boot time to be a viable benchmark.  On older machines in the 256mb / 400mhz class, old Windows 2000 blows them all away.  

I am neither a GIMP expert or a Photoshop expert but if you are experienced in Photoshop you will find the GIMP user interface to be really weird and confusing.  If you are using Windows now, download GIMP for Windows and determine for yourself if it will do the job for you or not.  Will GIMP in Ubuntu feel faster to you than Photoshop in Windows?  Maybe.  Maybe not.

Jerry
