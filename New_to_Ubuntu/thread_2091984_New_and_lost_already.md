Software installation - in regard to just general user level stuff - might be the most drastic thing that's different in the Linux world compared to the Windows world.  In Windows, you just look for some software you want, download it, click install, and hope it's not infested with all kinds of internet goodies.  In the Ubuntu world (as well as a lot of the other distributions), package managers and repositories are used.  

In a nutshell, it works like this.  There are collections of Ubuntu specific programs (aka packages) that are maintained in secure repositories.  These packages are routinely updated and confirmed to work correctly with your system.  When you need to install a new piece of software (let's say you wanted to get something to edit photos, like the Gimp package), you would open the Ubuntu Software Center (one of many Package Managers), search for that piece of software, and then click install.  It will then pull that software down, install it, and make sure that every other package that Gimp requires is installed at the same time. This also registers in the package managers database and unlike the Windows world where you'll have various packages all looking for updates at the same time, sometimes presenting you with a host of pop up windows all asking to be updated, the package manager will take care of that for you.  When these packages are updated in the repository, the package manager will alert you, and you can choose to install the latest version from there. 

Here's the Community Documentation on package managers and installing software for more details.

All of this, of course, can be done from the command line rather than a GUI.  Many Linux users out of habit and (for me anyway) sake of convenience prefer to do it this way.  However, there is a GUI tool as a front end for just about any command you can run on the commandline.  So, start with understanding how things work, and then, if you're so inclined, you might take a trip in the command line to see how to do the same things and how much more control and information you get by doing so.

Finally, feel free to start new threads with questions as you go.  This is a really great group of very helpful and friendly people here who will definitely get you to where you need to be.
