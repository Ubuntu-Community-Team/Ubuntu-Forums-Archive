### Post by Cheesemill on 2012-07-14

> The first ubuntu I tried came up with a 'welcome to freeDOS' on the screen.

You haven't booted from the CD, it sounds like you have booted from the hard drive.
Servers that are sold with no OS often have FreeDOS installed on them.

Also it sounds like you haven't created the CD correctly, you don't just extract the files onto a CD you have to burn the .iso file as an image. For more information see the Ubuntu website.
[http://www.ubuntu.com/download/help/burn-a-cd-on-windows](http://www.ubuntu.com/download/help/burn-a-cd-on-windows)
