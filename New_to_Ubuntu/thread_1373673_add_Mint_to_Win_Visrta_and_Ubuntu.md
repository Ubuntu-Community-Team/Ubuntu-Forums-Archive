### Post by Peekie on 2010-01-06
Ubuntu 9.10 uses the new GRUB2. After installing Mint, I guess the Mint installer will have created a GRUB-menu which contains Vista and Mint. So you have to restore Ubuntu 9.10 in the GRUB2-menu.
This documentation is probably the first place to look: [https://help.ubuntu.com/community/Grub2]("https://help.ubuntu.com/community/Grub2")
