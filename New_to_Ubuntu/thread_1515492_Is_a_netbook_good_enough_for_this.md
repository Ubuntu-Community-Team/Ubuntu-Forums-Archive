### Post by Rubi1200 on 2010-06-22
The answer is yes and no.

Let me explain:

> 1.6ghz Intel Atom
1gb DDR2 RAM (Would it benefit to upgrade to 2 by the way)
160gb Hard Disk

This is fine and I think 1gig of RAM is okay for your needs, but 2 is always better!

> Intel Integrated Graphics

This could be a problem:

[https://wiki.ubuntu.com/X/Bugs/Lucidi8xxFreezes](https://wiki.ubuntu.com/X/Bugs/Lucidi8xxFreezes)

For the rest, take a look at this:

[https://wiki.ubuntu.com/UNR](https://wiki.ubuntu.com/UNR)

or this:

[http://www.geteasypeasy.com/](http://www.geteasypeasy.com/)

(I have seen the latter in action and it is cool!)

Hope this helps.

Using Wordpress.
> Twitter/Facebook/etc.
Gmail and Facebook Javascripts are always running in the background on Chromium.

My verdict is:
[LIST=1]
[*]Upgrade system memory to 2 or 4GB.
[*]If video streaming / playback is essential. Use a light WM.
[*]Skype could be an issue. But again, ensure that nothing is using precious CPU time when running it.
[/LIST]

Other than that, you are all in the clear.

### Post by admiralspark on 2010-06-22
Skype will work on those specs fine, as long as you're not trying to multitask while video chatting. 
If I could recommend a slight alternative to the 10.04 setup? Dual booting Puppy Linux 5.0 (Lucid-based puppy) and 10.04 Lucid Lynx will allow you to have a complete, powerful, feature-filled desktop available as 10.04, and then if you just want to quickly switch on and google something/do quick easy tasks you can boot to the superfast Puppy. Puppy 5.0 has the most popular programs from the Lucid repository available to it, and it loads completely into ram so it's literally near-instant load times on everything. 1GB ram is much more than you'd ever need to run puppy 5.0, and Lucid should run just fine with an Atom/1GB combo.
