### Post by 3rdalbum on 2009-12-16
These things get updated automatically if there are security fixes or major bug fixes. You won't get new major versions from the Ubuntu repositories for a number of good reasons.

If you absolutely need the latest versions of certain programs, you can often find a PPA (Personal Package Archive repository) containing the software you want. Add the PPA to your system, and every time a new version is packaged for your version of Ubuntu it will be added to Update Manager.
