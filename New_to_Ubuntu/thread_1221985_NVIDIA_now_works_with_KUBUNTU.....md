Now I am using Kubuntu with the proper screen resolution.

When I boot... it opens the Kubuntu session by default.

For some reason, the Nvidia Driver can be activated in Ubuntu and Xubuntu but not Kubuntu (KDE problem?)... so I had to do a workaround.
