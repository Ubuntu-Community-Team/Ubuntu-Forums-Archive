### Post by adult swim on 2009-02-12
you should be able to do it there if your network is up.  alternatively you could boot into recovery mode and drop to root terminal with network and update from there.  you could also boot a live cd, then mount the drive and chroot into it to update.  if the drive was /dev/sda1 you would do it like this
```
sudo mount /dev/sda1 /mnt
chroot /mnt
apt-get update
apt-get install nvidiadriverpackage
``` replacing nvidiadriverpackage with whatever the driver package is
