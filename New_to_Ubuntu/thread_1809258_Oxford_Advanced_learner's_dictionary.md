### Post by grahammechanical on 2011-07-21
It could mean that the CD contains a version of the dictionary that will run on the Redhat Linux distribution. If so, the main file or package will have .rpm as the extension.

This will not install on Ubuntu because Ubuntu is based upon Debian which uses the deb packaging method. You need the main file to have a .deb extension. I think that there are programs that can convert a rpm package to a deb package but how good they are I cannot say.
