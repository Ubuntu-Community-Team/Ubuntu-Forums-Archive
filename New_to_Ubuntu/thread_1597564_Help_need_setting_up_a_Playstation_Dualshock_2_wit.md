### Post by mister_playboy on 2010-10-19
The simplest issue with many USB controllers is to make sure it is plugged in and recognized before starting the emulator, since emulators check for a controller during startup and usually won't see one plugged in afterwards.

I would recommend using [bsnes](http://byuu.org/bsnes/) rather than zsnes if your computer meets the system requirements.  zsnes is theoretically still active, but definitely in development limbo right now.

Compiling bsnes is currently a bit of a pain, so I recommend using [this PPA](https://launchpad.net/~hunter-kaller/+archive/ppa/+packages) instead.
