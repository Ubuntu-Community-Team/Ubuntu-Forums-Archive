### Post by sisco311 on 2010-04-01
Go to [this]("http://xkcd.com/") site and type *man echo*. [/joke]




On a more serious note:


echo displays a line of text
 
-e tells echo to interpret backslash ('\') escapes 

"\0101\0160..." is the octal representation of the text.

The command is harmless, it only prints out a line of text.
