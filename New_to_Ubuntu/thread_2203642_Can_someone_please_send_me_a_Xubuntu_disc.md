### Post by Elfy on 2014-02-04
Get unetbootin - if you're using *buntu you can install it. If not find it on the net [http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

Once it's installed.

Plug in your USB - formatted to fat32

Start unetbootin.

Point it to the iso file

Press start/go or whatever it's called where you are and wait for it to finish.
