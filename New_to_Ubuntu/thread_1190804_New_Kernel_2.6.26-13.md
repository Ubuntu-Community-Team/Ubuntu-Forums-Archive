### Post by jgoguen on 2009-06-18
If you open Synaptic Package Manager (System -> Administration -> Synaptic Package Manager) you can view the changelog for any package with these two steps:

[list]
[*]Select the package you want the changelog for
[*]Under the Package menu, choose **Download Changelog**
[/list]

That will open a new window with the current changelog displayed.  This works for any package (except packages from a PPA), not only the kernel.
