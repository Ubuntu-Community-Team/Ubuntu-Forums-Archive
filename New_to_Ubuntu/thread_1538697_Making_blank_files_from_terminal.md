### Post by CaptainMark on 2010-07-25
Is it possible to use the terminal to make a blank file without using a text editor, what i mean is not using ```
gedit ./program.py
``` to make a .py file and open gedit to write it but the equvilent of right clicking in nautilus and clicking >create document>empty file and naming it. The reason is i often want to make a load of python scripts and name them all first and then write them later

---

### Post by Ozymandias_117 on 2010-07-25
```
touch prog_name.py
```

### Post by collinp on 2010-07-25
You can use the command "touch". In your case, the command would be
```
touch program.py
```
