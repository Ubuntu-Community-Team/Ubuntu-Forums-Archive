### Post by renzokuken on 2008-04-29
you can do it on a file type basis......for example to make it default for AVI files, right click on any AVI file and select Properties - Open With and choose VLC, click ok and all avi files should now open with VLC.

you can repeat this for other file types too, e.g. .mkv .mp3 .ogm etc etc

### Post by bryncoles on 2008-04-29
hi there peterkeyani. to make VLC your default application, rights click the file type you want it to open, select 'properties', 'open with' and choose 'VLC' from that list. if vlc isnt in the list, click add and choose vlc from there. it will now be the default program for that file type. 

hope that helps!
