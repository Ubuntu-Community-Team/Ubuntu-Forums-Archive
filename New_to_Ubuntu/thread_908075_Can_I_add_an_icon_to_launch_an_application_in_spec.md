### Post by Bakon Jarser on 2008-09-02
That's an interesting question.  When you right click on a launcher and click properties you see the command line.  That's what launches the program.  From there you can pass it parameters just like you would from terminal.  So if you open terminal and type "man gimp" it will open up the manual for gimp and tell you all the parameters that you can pass to it.  To exit the manual hit ctrl+z.  I hope you find the answer there somewhere.
