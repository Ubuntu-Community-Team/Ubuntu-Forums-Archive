### Post by Temposs on 2009-04-03
When you install things with Ubuntu, you rarely need to go to the application's website in order to install it. In your case, you don't need to go to Sun's Java website in order to download Java. Please read this how-to for installing apps in Ubuntu:

[http://www.psychocats.net/ubuntu/installingsoftware](http://www.psychocats.net/ubuntu/installingsoftware)

Basically open Applications->Add/Remove...

Search "Java" and then install the applications that come up. Very easy :-)
