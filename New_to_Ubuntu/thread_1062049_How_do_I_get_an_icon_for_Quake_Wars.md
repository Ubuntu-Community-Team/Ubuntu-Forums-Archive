### Post by donkyhotay on 2009-02-06
If you're using gnome then go to system > preferences > main menu then browse to the quake wars entry and double-click on it. Click on the icon on the left hand side, click browse on the top, browse to the ETQW directory and select open, you will then see the icon available to select and apply it to the menu item.
