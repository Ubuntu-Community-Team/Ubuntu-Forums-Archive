### Post by st33med on 2008-10-25
Compiz Fusion comes with Ubuntu 8.04, and it is better than Compiz because it is a merge of the Beryl and Compiz effects. You will, however, have to install ccsm to get that cube effect and other things.

```
sudo apt-get install compizconfig-settings-manager
```
