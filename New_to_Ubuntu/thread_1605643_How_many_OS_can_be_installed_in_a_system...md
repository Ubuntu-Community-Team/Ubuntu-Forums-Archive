### Post by rockerphil on 2010-10-25
to be honest as many as you can squeeze in there, you can probably save a lot of space by making a single /home partition for all of your OSes. a bare minimal install of Linux takes up less than 1 GB (that's without a graphical user interface though). i once squeezed 4 OSes in to a 30 GB hard drive. my only major advice if you're gonna install more than a dual-boot is 1)use the manual partitioning option so that you don't make more than 1 swap partition (totally unnecessary) and 2)don't install a boot loader (either GRUB or Lilo) until you install the last OS, it turns in to a big mess. hope this helps,

Phil
