### Post by HotShotDJ on 2009-02-01
> **mmcl said:**
> I'd like to be able to print some "no royalty" graphics which I find on the web.In Firefox, right-click on the graphic and then "save image as..."  Now open the image (Image Viewer, GIMP, F-Spot, whatever application you like).  Print it from there.

If you REALLY just want to print it from Firefox, right-click on the graphic and then "view image" then print it (you can get rid of any headers and footers using the "options" tab in the print dialog)
