### Post by squakie on 2012-10-31
First a little definition info:  SATA is an interface to a hard disk drive (HDD) or to a solid state drive (SSD),  so there is no difference there.  IDE is also an interface to a hhd's and ssd's - this is mostly seen in older drives now.  Technically, the SATA on todays motherboards will be faster that the old IDE especially if you pick a decent drive.  SSD's are normally faster since you aren't waiting for latency on a HDD (the disk spinning, moving the heads, etc.).

That being said, if you are doing that much formatting I personally would go with a hard disk.  I have a SSD, but I don't think I'd be formatting it every week or so.  That being said, why do you need to format every week or two?
