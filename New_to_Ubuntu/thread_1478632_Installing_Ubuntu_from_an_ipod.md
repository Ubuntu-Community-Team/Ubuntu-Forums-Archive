### Post by Ozymandias_117 on 2010-05-10
> Is it possible to install Ubuntu from my ipod as though it were an external usb hard drive?
I would advise against doing that unless someone else can confirm it works.
I'm afraid writing to the MBR of the iPod would stop it from being able to function as an iPod.

> Also, is there any specific advantage or disadvantage to choosing regular Ubuntu or Ubuntu Netbook Remix?

Netbook Remix simply has a different way of displaying the desktop that is supposed to make it easier on small screens. Other than that, the two are exactly the same.
