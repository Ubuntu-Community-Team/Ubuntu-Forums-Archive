### Post by XKCD64 on 2009-08-15
Hi, I have an 8 gig U.S.B. flash drive.  I was wondering if it is possible to use this as an external drive with Ubuntu on it.  So, if I'm ever using my moms computer, I just plug it in and boot onto the flash drive and into Ubuntu.  And have the whole Ubuntu desktop on it?  And save my files like I normally would on my computer?

---

### Post by mgmiller on 2009-08-15
Yes.  Click on System > Administration > USB Startup Disk Creator.  When you are setting up the drive, just enable the options for persistence so it saves all your stuff.  You'll  See the slider to set how much of the drive is available for that.
