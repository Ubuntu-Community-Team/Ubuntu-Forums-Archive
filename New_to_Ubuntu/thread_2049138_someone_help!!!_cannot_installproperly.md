### Post by coldraven on 2012-08-28
If you don't have a blank CDor USB stick then your options are:
Buy an installation disc, see 
[https://help.ubuntu.com/10.04/switching/C/installing-get.html](https://help.ubuntu.com/10.04/switching/C/installing-get.html)

or borrow a USB stick from a friend. You need at least a 1GB stick or SD card (if you have a card slot)
Get him/her to download a Linux distro. You can get various versions here:
[http://releases.ubuntu.com/](http://releases.ubuntu.com/)

Then download Unetbootin from here:
[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

Use Unetbootin to create a bootable USB stick, install Ubuntu and give the stick back to your pal. 
You will need to enter your laptops BIOS setup and change the boot-up order so that it boots from USB first. After the install has finished remove the stick and change the BIOS back to boot up from the hard drive first.
