### Post by JC Cheloven on 2010-11-22
> **Calais225 said:**
> Just received my Dell laptop today with Ubuntu 8.1 installed.  I really want to upgrade and wonder if I should do it by going to 10.? or 9.,? first.
Since ever, it seems to happen that a fresh install is always less prone to headaches that an upgrade. It's also true that your preinstalled OS may have some drivers or software that you may find useful (my dell mini came with a videochat program for example...). So I would suggest this:

- Download and burn a Lucid 10.04 disk, just in case. I suggest 10.04 because it is a Long-Term-Support release. Take apart the disk for eventual future use. 
- Try upgrading to 10.04 from the internet (System-Administration-UpdateManager, then use the upgrade link).
- If that was fine, you're done. If not, proceed with a fresh install using the CD you prepared formerly. 

(side note: LowSky, didn't we have the option to upgrade directly to the next LTS release when available? I kind of remember that...)

Cheers

### Post by Vi3GameHkr on 2010-11-22
> **Calais225 said:**
> My worry is that even though this is a new computer there might be some older hardware on it that might not work with newer versions.

There is relatively little chance of that.  If 8.10 works, there is little to no doubt that 10.04 works unless there is something specific to 8.10 that you work with (and in that case, you would probably know.)  I have an HP system from years back (I think we bought it in either 2003 or 2004) it can barely run Windows XP (any software like Microsoft Office 2007, or Photoshop or something won't work anymore) but it runs Xubuntu 10.10 well enough with GIMP, OpenOffice, and a browser watching YouTube at the same time.  If I used regular Ubuntu (with the GNOME interface) it would probably slow down because of the graphics, but if you are running a relatively new laptop, with a Celeron and 1 gig of RAM, I don't see any real issues.  I can't say this for certain, but I don't see any reason why hardware supported by 8.10 wouldn't work with 10.04.
