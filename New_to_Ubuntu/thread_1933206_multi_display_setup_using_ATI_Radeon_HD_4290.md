### Post by Mark Phelps on 2012-02-29
You don't have to install the catalyst drivers to have multiple displays.  I have a setup much like yours -- onboard AMD 4290 chipset, two monitors, one connected VGA, the other DVI, and it works just fine.

I have different resolutions for each monitor and I have the main monitor on the Left.

Unless you really NEED the restricted drivers (hardware acceleration for 3D graphics for gaming), you should consider uninstlling CCC and going with the open-source.
