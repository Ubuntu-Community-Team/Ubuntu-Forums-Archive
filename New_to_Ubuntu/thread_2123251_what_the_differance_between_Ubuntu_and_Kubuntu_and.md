### Post by TenPlus1 on 2013-03-07
The main difference is that each uses a different Desktop Manager...  Ubuntu uses Unity, Xubuntu has XFCE, Lubuntu works with LXDE, Kubuntu has KDE...  In the end it's down to personal taste or what your computer is happy working with, some lighter weight machines will happily use XFCE and LXDE where the higher spec computers will make use of KDE and Unity a lot more with the 3D effects...

### Post by ibjsb4 on 2013-03-07
If you are currently running the Ubuntu-desktop (Unity) you can easily install the different desktops and see for yourself.

To install Xubuntu-desktop:

```
sudo apt-get install xfce4 xfce4-goodies
```

To install Lubuntu-desktop:

```
sudo apt-get install lxde
```

And you didnt mention Gnome_Classic.  Its the old style desktop of 10.04 days (for those of us that know change is bad :P

```
sudo apt-get install gnome-session-fallback
```

You can choose which desktop to run at boot (login).

[https://help.ubuntu.com/community/UsingTheTerminal#Starting_a_Terminal](https://help.ubuntu.com/community/UsingTheTerminal#Starting_a_Terminal)

### Post by deadflowr on 2013-03-07
If you add various desktops to your Ubuntu install, the way to change between them is at the login screen.
In the login screen in the box where your name and password go, is a little cog/ubuntu logo or something, click on it and select the different desktop environment/window manager you'd like to use.
