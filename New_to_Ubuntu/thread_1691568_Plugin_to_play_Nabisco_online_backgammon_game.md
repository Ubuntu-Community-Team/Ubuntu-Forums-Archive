### Post by peculiar penguin on 2011-02-20
The game requires Adobe's Shockwave Player (MIME type "application/x-director"), unfortunately they haven't released a Linux version as far as I (and WIkipedia) know. If you really need that game, you may have to play around with Wine to run a Windows browser or run a fully virtualized Windows using VirtualBox or similar.
