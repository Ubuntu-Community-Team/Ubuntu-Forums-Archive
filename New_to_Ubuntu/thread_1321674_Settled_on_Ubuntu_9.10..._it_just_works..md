### Post by eltama on 2009-11-10
You can configure the icon sizes from nautilus (it also changes the size of the icons in the desktop). Go to Edit->Preferences->Views and adjust Icon View Defaults and List View Defaults to your needs.

For backup, I recommend you Simple Backup (the package name is sbackup). It's the one mentioned on the official ubuntu's help. A similar one is Back in Time.

For media player, if you don't like the default rhythmbox, try banshee. You can install it from the Ubuntu's repository or if you want the latest version, add ppa:banshee-team/ppa to your software sources.
Update: Another good music manager and player based on GTK+ is exaile: [http://www.exaile.org/](http://www.exaile.org/).
