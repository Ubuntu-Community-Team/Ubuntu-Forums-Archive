### Post by bcbc on 2010-11-13
I'm not sure what you are trying to do... install Ubuntu with the windows installer (wubi) or install direct to partition?

If it's wubi, then that metalink message means that it is trying to download the md5 hashes of the ubuntu disk images, but it failed. The only reason it should need to do this when you're running off a cd is if it didn't like the CD image. But it's useful to see the log file to pinpoint the problem.

If you don't want to use Wubi, then you have to figure out how to boot your computer from a CD. Either go into BIOS and set it to boot from CD/DVD before the internal hard drive, or look for a Boot menu key to override the select the boot device at startup and tell it to boot from the CD.

Actually there is a feature with the windows menu (not wubi) that runs when you put the CD in while running windows. You can click the first option (reboot and install or try Ubuntu), and then the 3rd option (something about boot CD from windows) - this will put an entry in the windows boot manager to allow you to boot from the CD ROM. As far as the differences between wubi and Ubuntu - there are none from a 'user experience' point of view (maybe a bit slower). Underneath the covers Ubuntu is actually installed to a virtual disk (a large file). It's great to try out Ubuntu - but over the long term it's vulnerable to updates (since it doesn't get as much testing coverage) and the virtual disk can corrupt if you're in the habit of hard power-offs. 
You also won't get as much help as most longterm users don't use it.

### Post by Rex Bouwense on 2010-11-16
I would have thought that this thread would have been solved by now.  You know that you have a good CD because you installed on another computer.  However, you cannot for some reason or another you cannot boot with that CD on your desktop which has Windows XP installed.  You also cannot get on the Internet with that computer.  Your aim is to install Ubuntu alongside of Windows on the desktop.
OK you need to change the boot order on your desktop.  Because we do not know what kind of computer you have you will have to read the screen very quickly.  Place the CD in the CD drive and turn your computer on.  The manufacturer splash will come on the screen and either on the very top or the very bottom, you will be told to press F2 or F12 or some other button on the keyboard to enter setup.  You must be quick because once the Windows splash comes up it is too late and you must reboot to try again.  I know it sounds frustrating.  Once you are in setup, you need to use the right arrow to move the cursor to the boot menu or boot order or boot something.  Hit enter and you will be in the boot menu.  Now you need to insure that the device that is the first boot priority is the CD ROM.  Make sure you save your changes or all of this work will be for nothing.  Exit and save changes.  It will boot from the CD ROM.  Hopefully.
