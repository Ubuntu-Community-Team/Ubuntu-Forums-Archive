### Post by u.b.u.n.t.u on 2009-11-30
Welcome to Ubuntu and to the community board here. Not too sure if there is a problem per se in what you have written. You say you are using Vista, tested Ubuntu and went back to Vista.

You may wish to download the following standalone software for Vista. It will tell you about your computer hardware. Please only download the portable version, so that your Vista registry isn't written to.

[http://www.piriform.com/speccy](http://www.piriform.com/speccy)

The following, at the bottom of the page, once extracted should run.

Download ZIP Portable (1.29MB): spsetup100.zip

This is still beta, but excellent. With this you can easily report what your hardware is.

You mention that your computer usage is mainly browsing, emails and the like, rather than more intensive usage.

Based on that criteria, Ubuntu is the better choice. It is much faster and has many excellent programs, and this community to assist.

The best option for using Ubuntu 9.10 is to install it via Windows, with the application wubi.exe on the Ubuntu 9.10 CD. This will install Ubuntu 9.10 onto the hard drive and Windows will treat it like any installed program. Yes Ubuntu 9.10 isn't as familiar as Windows but the effort in learning it is richly rewarded by what Ubuntu has to offer.
