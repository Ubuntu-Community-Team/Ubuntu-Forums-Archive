### Post by Pagh on 2008-11-09
When I reinstall my system (happens every 2 months or so when I break something - usually X) I always download Skype directly from the following url: [http://skype.com/intl/en/download/skype/linux/choose/](http://skype.com/intl/en/download/skype/linux/choose/) where I just choose the Ubuntu 7.4+. Then click save file and when my browser is done with the download I go to the directory where the downloaded package is located and double-click on it. This will start the gDebi package installer and the rest is easy as cake. This have always worked very well for me.

So my point is: don't bother with going through the repositories just do a direct download.

Good luck
