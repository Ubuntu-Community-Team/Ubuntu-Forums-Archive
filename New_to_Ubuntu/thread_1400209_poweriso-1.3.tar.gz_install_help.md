### Post by SuperSonic4 on 2010-02-06
Download the tarball and extract the executable

Then either run it locally or copy it to /usr/bin

```
sudo mv -v /home/user/Desktop/poweriso /usr/bin/poweriso
```

where user is your username and you extracted it to the desktop
