---

### Post by anewguy on 2010-04-18
From what I've found searching the net, that is Assistive Technology.  So if you have anything special enabled (like in desktop effects, etc.) to do things like talk, enlarge the screen, anything to help someone who is somehow "handicapped" from using things normally it would be there.  It appears that by default it is turned on.  If you don't need it, go to System/Preferences/Startup Applications and uncheck AT SPI Registry Wrapper and then I guess reboot.
