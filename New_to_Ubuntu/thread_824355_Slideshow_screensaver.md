### Post by Rhubarb on 2008-06-11
Right click on the directory that contains the pictures.
Select "Make Link".
Now cut the created link and paste it into your Pictures directory within your Home directory.

Make sure your screensaver is set to "Pictures folder" and all should work fine for you :D
