### Post by gettinoriginal on 2009-01-28
One of two ways, You can go to Synaptic Package Manager (System, Admin, Synaptic) and in the search box type "ubuntu-restricted-extras", without the quote marks, check it for install, it will ask for your password and then will install it.
Or, if you like the CL, go to Applications, Accessories, terminal, and type in the following:
```
sudo apt-get install ubuntu-restricted-extras
```  :p
