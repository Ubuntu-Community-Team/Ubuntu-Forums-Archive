### Post by mcduck on 2012-01-18
They mean nothing, Unity is just picking some color from the icon itself and uses that for the background, to make it easier to differentiate between the apps in the launcher.

If you change to a different icon theme, you'll see how the background color changes accordingly.

(If you haven't noticed, Unity actually does a similar kind of thing with the Launcher panel and Dash menu backgrounds, the shading colors change based on the wallpaper image you are using.)

### Post by migs73 on 2012-01-18
If you have CCSM and go into the Ubuntu Unity plugin settings> Experimental, you can set the background (or the backlight as it is called in the settings) to be off. This gives a nice clean bark grey/black background.
