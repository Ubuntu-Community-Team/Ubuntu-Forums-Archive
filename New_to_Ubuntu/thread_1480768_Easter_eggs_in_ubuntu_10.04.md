### Post by nothingspecial on 2010-05-14
> **ashora said:**
> how? and i love this thread!!

```
wget -r -A.tar.gz http://xpenguins.seul.org/contrib/
cd xpenguins.seul.org/contrib
wget http://www.linuxforblondes.com/applications/esheep/_files/xsheep.tar.gz
for i in *.tar.gz; do tar -xvf $i; done
rm -r *.gz
sudo mv ~/xpenguins.seul.org/contrib/* /usr/share/xpenguins/themes/
sudo mkdir /usr/share/xpenguins/themes/Sheep
sudo mv /usr/share/xpenguins/themes/XSheep/* /usr/share/xpenguins/themes/Sheep/


```

Then ```
xpenguins -t "Sheep"
```
Or if you want 200 turkeys```
 xpenguins -n 200 -t "Turkeys"
```

To see which themes you have available type```
 xpenguins -l
```

To start xpenguins with a random theme type ```
xpenguins --random-theme
```
