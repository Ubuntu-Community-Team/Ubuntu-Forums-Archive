### Post by heyandy889 on 2011-12-21
Hi, kiran.kashinat.shinde. Hmm, looks like quite a specialized program. This is some sort of accounting software for California? 

Have you used WINE before? It's my understanding that WINE provides implementations of Windows DLLs, basically libraries, which are needed by Windows programs. Really graphics-heavy programs, like games, are slow with WINE, but it *seems* like an accounting program should work.

Could always install Windows inside a VirtualBox and run the accounting software in there.

If you want to take a screenshot, it's pretty much the same in Ubuntu as in Windows. Just press the trustworthy Print Screen key on your keyboard, and bam! There should be a little pop-up that asks you what you want to name the picture (by default, something like Screenshot.png).

OK, rock on, dude. Glad you're giving Ubuntu a try, hope you can get over enough obstacles to have it work for you.
