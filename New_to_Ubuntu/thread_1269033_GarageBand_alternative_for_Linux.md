### Post by 3rdalbum on 2009-09-23
It depends what you want to use it for.

Most Mac users seem to use Garageband as a simple audio recorder, or (more rarely) as a multitrack audio editor. We use Audacity for this.

About 1% of Mac users actually use Garageband as a music composition tool. There are quite a few music composition tools for musicians on Linux, you might want to check out the packages that make up Ubuntu Studio.
