### Post by Glucklich on 2009-11-01
Some of the menu's might not be called this way since I'm using Firefox in my own language, but here it goes:

Edit / Preferences ... Privacy (tab)... at the bottom you'll find 'Private Data'... Everything you want is there. You can set more things to erase on the "Definitions" button.
