### Post by nicedude on 2008-08-29
Try looking in synaptic as I think there are hotkey type programs in there which would let you setup custom hotkeys to launch stuff. Also you might want to check out nautilus scripts manager which once installed and you put some scripts in its directory it gives you a new right click menu catagory where you can easily access various scripts to do all sorts of stuff. You can find scripts people have written all over the net or make your own, its my favorite as with a script I can do complicated things like configure my wifi for a certain mode etc with only a single click and password entry when normally I might need to run 5-10 commands to do it all.
