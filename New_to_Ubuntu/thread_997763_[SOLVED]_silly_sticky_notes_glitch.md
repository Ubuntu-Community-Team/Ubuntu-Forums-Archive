### Post by lovinglinux on 2008-11-30
Right click in the sticky notes icon in the panel and select delete notes. This should also reset the position of new notes.

BTW, I was able to reproduce this issue here. It seems that sticky note go crazy if you have the option "Put notes on all workspaces" enable, then try to move a note between workspaces. Disable that option and you should be fine.
