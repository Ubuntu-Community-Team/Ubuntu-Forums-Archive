As far as I know, the Nero 9 you have is for Windows only, not Linux.  The latest Linux version of Nero is Version 4 and it does not include any Lightscribe capability. Go to  [http://lightscribe.com/downloadSection/index.asp](http://lightscribe.com/downloadSection/index.asp)
and download and install the DEB version of Lightscribe for Linux.  I have created Lightscribe labels in Linux but it has been a couple of years and I don't even have it loaded anymore.  It has probably been nearly a year since I created a Lightscribe label in Windows although my CD/DVD writers are Lightscribe capable.  Lightscribe does work but I find it easier just to write on the disk with an ultra fine point Sharpie permanent marker.  

Jerry

PS I just checked and can't find any reference to Lightscribe in the Synaptic download manager.  I was thinking it used to be there but I might be wrong about that.

### Post by HotShotDJ on 2011-01-17
I have a CD/DVD burner with LightScribe.  It works perfectly.  Always has.  Follow the instructions in the Community Documentation, taking care to download the latest versions of the software. [https://help.ubuntu.com/community/LightScribe](https://help.ubuntu.com/community/LightScribe)

(Nero is not needed for burning in Linux.  I find K3b does the job and does it well.)

sudo apt-get install glabels

Using the Ubuntu Software Center just search glabels.
