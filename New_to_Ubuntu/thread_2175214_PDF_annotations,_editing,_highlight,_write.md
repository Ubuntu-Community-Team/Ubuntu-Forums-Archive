### Post by Impavidus on 2013-09-19
Pdf is a computer-readable description of human-readable pages. Therefore, both computers and humans have difficulty editing it. It was never really designed to be editable, as it was designed to be a printer language. So, if given the choice, avoid editing pdf and edit the source from which it was generated, for example a libreoffice document.

It seems that inkscape does a pretty good job at editing singe-page pdfs, as it can extract objects, insert objects, etc. As Adobe designed pdf their tools can be expected to be the best, but their Linux support isn't very good.
