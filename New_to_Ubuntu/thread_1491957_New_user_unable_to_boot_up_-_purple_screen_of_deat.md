### Post by rainmouse on 2010-05-24
OK after thinking about my own hardware spec I fixed the problem which turns out to be my own stupidity.  I had a dual screen setup on windows but my secondary monitor pretty much died so I disabled it etc on software but left it plugged in and turned off.  Seems Ubuntu liked it better for my primary monitor which is why my other screen was left blank and purple.

On a side note its interesting that a China bot has replied to my query within 5 minutes of the original post, impressive.
