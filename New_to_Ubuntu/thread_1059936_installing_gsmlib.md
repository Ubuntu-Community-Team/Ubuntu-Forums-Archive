### Post by petervk on 2009-02-04
It's been renamed to "gsm-utils" in ubuntu.

you can install it by launching a terminal and inputing the following:
```
sudo apt-get install gsm-utils
```

I found it by searching gsm on the ubuntu packages site. [http://packages.ubuntu.com/gsm](http://packages.ubuntu.com/gsm)
