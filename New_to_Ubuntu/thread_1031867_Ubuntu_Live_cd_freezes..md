### Post by pytheas22 on 2009-01-05
Try burning the CD again at a slower speed.  Sometimes you just get bad CDs.  You should also [verify that the ISO image]("http://www.psychocats.net/ubuntu/iso#checksum") that you used to create the CD is not corrupted.

Another option for installing Ubuntu, if you have Windows on the same computer, is to use [wubi]("http://wubi-installer.org/"), which doesn't involve a CD.

You should also be sure that your computer has at least 256 megabytes of memory.  If you have less, the live installer won't work and you should use the [alternate CD]("http://www.ubuntu.com/getubuntu/downloadmirrors#alternate") instead

If none of the above works, please tell us the exact model of your computer so we can see if other people have had issues installing Ubuntu on it.
