### Post by coffeecat on 2010-09-03
You couldn't do better than [the GIMP]("http://www.gimp.org/"). It's in the repositories so you can install it using Software Centre or Synaptic. No need to download it from that site.

Pre-Lucid it was included in a default install. With Lucid it was decided that most people didn't need as something as powerful as Photoshop so it was removed from the default install to make some needed room. But easily installed nonetheless.

It's very comprehensive. Have a look at the available plugins on that site. And if you have a camera that produces RAW images, there's an addon for that too. (In the Ubuntu repository. Package = gimp-ufraw.)
