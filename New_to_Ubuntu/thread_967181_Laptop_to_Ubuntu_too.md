### Post by ugm6hr on 2008-11-02
> **susanpenter said:**
> Is it better to have most things on the desktop and network the laptop to it or treat it like a separate system? I have a wireless broadband connection.

If you need it to be portable - it will need to be a separate system.  Also, if you wanted it to be a server-thin client type setup, the server (i.e. desktop) would need to be the network hub (not your router).

If the LiveCD ran, but wouldn't install, it is likely you don't have enough RAM.  The AlternateCD may work better.
