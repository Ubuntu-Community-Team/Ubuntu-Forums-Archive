### Post by bennachie on 2009-06-08
You should not need any special software to do this in Ubuntu 9.04 (or in Linux Mint 7, if you prefer that flavour). Just connect the Nokia phone to your computer via the USB data cable, and select "Nokia Mode" or "Modem" if so requested by the phone OS. 

Network Manager will detect the new mobile connection, and invite you to set it up, which is really only a matter of identifying the country and the carrier. This will work with 3G phones as well as 2G (eg GPRS) models. Just one of the advantages of open source over the dark side.
