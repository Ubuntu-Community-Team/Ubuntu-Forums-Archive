One option is to install Ubuntu and a Ubuntu virtual machine inside your Ubuntu installation.  If you wan't to try something out, just use the virtual machine and if it breaks then load a backup of the virtual machine and try again.

### Post by Anuovis on 2011-05-24
Games might be a problem on Linux. Another thing to consider is whether you need some sort of specific professional software like design, audio production, might even be image editing. There is a good chance it will run only on Windows/Mac and you might not find an alternative.

For average users that don't need specific work to be done (those who browse web, watch DVDs and write mails), Linux is more than enough and often way better than Windows. That's not to say Linux doesn't have powerful software or that you can't use Windows apps but that is always on a case-by-case basis.

There are tons of distributions and Ubuntu might not the best, however you define that. But it surely has one of the largest and most helpful communities. This means that if you run into some trouble, you can come here and get help. Large and friendly user base is really an awesome thing.

So go Ubuntu, if you ask me. At least for a start. Another thing to consider would be Gnome or KDE and well, that is mostly a religious debate. Go with default Gnome maybe, or try both and see for yourself.

Like others said, the best thing is to go and use it. Try dual booting or installing it on another PC and play around.

Have fun :)

### Post by amanchesterman on 2011-05-24
I'm a newbie like you, I have been using Linux on my laptop for the last two months and don't have any technical expertise. I'm using the standard 10.10 edition, Ubuntu Maverick with the Gnome desktop.

Before I installed it I tried out Mint and Kubuntu as well. I found Ubuntu the easiest to set up, everything worked 'out of the box', and I have been able to tweak the windows, menus etc. easily to my liking. The KDE desktop I found much harder to customise as a complete beginner: I believe you can tweak it endlessly, but the help menus were less clear to me. Mint I found harder to customise also, though maybe I missed something there.

Anyway, good luck and I hope you enjoy it. As others have said, the best way to learn is just to try it out -- but remember to keep all your important stuff backed up in case you have to re-install the system.
