Actually: no.

Ubuntu uses Linux applications. You cannot normally run a Windows app on Ubuntu. There is an application called Wine that you can use to try and get Windows apps to run on Ubuntu, but it's pretty hit-and-miss.

Generally speaking, there is an equivalent native Linux app for most Windows apps. Some apps are even available in both Linux and Windows versions.

### Post by 3rdalbum on 2009-11-11
I think you've all confused the original poster.

Windows doesn't run Linux programs, and Linux doesn't run Windows programs.

There is some free software for Linux called Wine, that makes it possible to run a *limited* variety of Windows software on Linux. There's also some paid software called Crossover that can run a slightly less limited variety of Windows software.

But in the end, the happiest Linux users are the ones who find alternative programs to use on Linux. Most people get along just fine with the included Openoffice.org software, instead of installing Microsoft Office. In fact, Linux has a massive amount of free software for most things you can possibly imagine.

Try the native Linux software first; if you absolutely need to you can try your Windows program on Wine, but expect it to not work, or to require some complicated hacking to get it to work.
