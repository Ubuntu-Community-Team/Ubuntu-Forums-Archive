### Post by eternalnewbee on 2008-10-23
As you are a new user, I would recommend starting with Ubuntu 8.04, not with 8.10 as it is still in the testing phase. If you don't have an install cd, download the LTS ( long term support ) version, burn it at a very low speed, preferably 4, and reinstall. and if you still have the same issues come back to the forums. Alternatively you could wait for advice from more experienced users.
Good luck & welcome to the Ubuntu Forums.
