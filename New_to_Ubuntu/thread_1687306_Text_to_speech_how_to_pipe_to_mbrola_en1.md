If  you  have not copied the MBROLA software yet, or if  you have
an  older  version, please consult the MBROLA   project  homepage
and get it.

Copy en1.zip into the mbrola directory and unzip it :

   unzip en1.zip (or pkunzip on PC/DOS)
   (don&#8217;t forget to delete the .zip file when this is done)

Try

   mbrola en1 euler.pho euler.wav
