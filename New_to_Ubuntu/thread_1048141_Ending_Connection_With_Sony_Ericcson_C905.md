Yes you're correct. The only way it would damage it is if the phone were disconnected while data was being written to it. Unmounting it would prevent any data from being written while you're unplugging it.

Just to be on the safe side though I always give anything like that a few seconds before unplugging or unmounting to make sure any reads or writes have time to finish up.
