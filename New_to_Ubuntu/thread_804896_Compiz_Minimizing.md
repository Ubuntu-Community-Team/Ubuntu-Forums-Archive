### Post by nzjethro on 2011-06-23
> **Psyket said:**
> What if I dont got all those options? Just a few, is there a command to install more effects?

Run

```
sudo apt-get install compizconfig-settings-manager compiz-fusion-plugins-extra compiz-fusion-plugins-main compiz-plugins
```

This will download CompizConfig Settings Manager, plus the extra plugins (like fire minimise).
