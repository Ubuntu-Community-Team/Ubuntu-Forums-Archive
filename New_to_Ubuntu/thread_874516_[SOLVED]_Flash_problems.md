### Post by gobbles414 on 2008-07-30
In Ubuntu, you should go *System => Administration => Synaptic Package Manager*. Search for *ubuntu-restricted-extras*, and install it. This will install Flash, Java, some Microsoft fonts, and a few other nice features. Restart your computer, just to be safe. You should now be able to access Flash content on the internet.

PS: A great place to find answers to questions like yours is in the documentation located at [https://help.ubuntu.com/community/]("https://help.ubuntu.com/community/").
