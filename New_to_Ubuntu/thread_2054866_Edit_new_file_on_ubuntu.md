### Post by taras.kuzyo on 2012-09-08
You can run 
```
touch filename
```
to create file called filename.
Then
```
nano filename
```
to edit it and press Ctrl+O to save it.

### Post by GreatDanton on 2012-09-08
You can also create it if you just type ```
nano <filename>
``` This will create file, so there is no reason for the first command.

I prefer GUI&#8594; ```
