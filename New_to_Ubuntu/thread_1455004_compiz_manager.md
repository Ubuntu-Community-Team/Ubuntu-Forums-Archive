### Post by mcduck on 2010-04-15
Could you specify which exact effects from that video you want to enable?

If it's the 3D window effects for minimizing etc. then make sure you have the "compiz-plugins-extra"-package installed, then enable the "Animations Add-Ons"-plugin in CompizConfig Settings Manager and those extra animations will become selectable in the Animations-plugin.
