### Post by jtarin on 2011-06-11
> **jrev said:**
> Hello,
how can I import the Firefox 4.0.1 bookmarks into the epiphany browser ?
:p
Thank you for your proper answer
```
In Firefox, go to "Bookmarks -> Organize Bookmarks 
(or Show All Bookmarks dependent on your version of FF)"
Then, in the Library window, go to "Import and Backup -> Export HTML"
Save the file where you can easily find it again.
In Epiphany, go to "Bookmarks -> Edit Bookmarks"
Then, in Bookmarks window, go to "File -> Import Bookmarks..."
Select "From File" Click "Import" Choose the html file exported from FF.
```
