### Post by qamelian on 2013-11-19
There have been a lot of changes since Gnome 3.2, which is the most recent version of Gnome, with which this theme is meant to work. Many themes no longer work on more recent versions of Gnome, and I'm betting this is one of them. Unless you're prepared to learn the ins and outs of creating themes, you may be out of luck.
