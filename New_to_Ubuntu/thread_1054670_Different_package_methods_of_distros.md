### Post by Raccoon1400 on 2009-01-29
One of the differences is the the tool used to install the packages.
They have different gui and or cli frontends, and different features
try some google searching. There's way too much to put in one post.

My favourite is the archlinux package manager, pacman.
It has only a CLI frontend, but it's really simple to do.
To update the package repos and update every package on the system, all I do is type:
pacman -Syu

List of package formats
.deb
.rpm
.pkg.tar.gz
.src
and many many more
