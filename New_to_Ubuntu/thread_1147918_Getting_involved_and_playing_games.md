### Post by inobe on 2009-05-03
welcome

ETQW edited link [http://community.enemyterritory.com/](http://community.enemyterritory.com/)

if you don't mind a little terminal work' this is how i install it.

get the game here [ftp://ftp.snt.utwente.nl/pub/games/etqw/demo/client/linux/ETQW-demo2-client-full.r1.x86.run](ftp://ftp.snt.utwente.nl/pub/games/etqw/demo/client/linux/ETQW-demo2-client-full.r1.x86.run)

save it to your desktop

click applications/ accessories/ terminal

in terminal type 

```
cd ~/Desktop and hit enter

```

now type 

```
chmod +x ETQW-demo2-client-full.r1.x86.run 

```

now do and hit enter

```
./ETQW-demo2-client-full.r1.x86.run
```

the installer will start within terminal, follow the on screen prompts and agree to the eul's

be sure to except the eul for punkbuster

you will reach one prompt that requests an installation path, at this point create a folder in your home directory, name this folder whatever' so as long i you remember it and it's location !

open the folder you created, just above on your file browser' click go then hit location, copy that path into the installer installation path, now click install, now that it's done installing you can close everything.

now to create a quick launch shortcut because for whatever reason i don't get one during the install.

right click on your desktop and select create launcher

in the dialog you will see four options

type: leave it set to application
name: i used ETQW
command: i used the directory the game was installed to /home/usernamehere/qw/etqw

comment: put whatever

at the top left corner of the dialog' click the square no icon button, when the icon dialog opens click browse and navigate back to the path where the game is installed, in there you will find the etqw.png icon

select the icon and hit ok, now close everything, you now have an icon that will start the game on your desktop.

you can drag the icon to a panel.

enjoy

### Post by Axis on 2009-05-04
I was in your very same situation once, and the best thing I can suggest is learn as much as you can. And spread the word to everyone you know. Its really easy to try and convert people to ubuntu when they are having problems with their windows PC, or they like the way vista looks or something of that nature, but don't want to pay for the upgrade. Sometimes people just want something new!

Learning as much as you can will help you educate them when they have questions for you, "Can I still play my games?" "I heard you have to use the command line a lot" "What about an anti-virus?" Learn as much as you can and break things a lot, I can solve a lot of my own problems now just beause I was exploring a little too much and had to learn to fix things :)

Good luck!
