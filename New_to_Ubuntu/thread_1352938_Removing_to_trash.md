### Post by jwh400 on 2009-12-12
It's safe to remove files from your Home directory although removing any hidden file will eliminate settings/preferences from whatever app directory you delete. 


Uninstall programs using the Synaptic Package Manager.
If you've downloaded a program to the desktop but haven't installed it, you can move it to Trash.
