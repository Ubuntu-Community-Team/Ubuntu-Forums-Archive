### Post by Ralph L on 2010-05-23
I am experimenting with LuckyBackup 0.4 (under Karmic) and so far am impressed with its snapshot capability .  However, I couldn't find a way to perform a restore of an older snapshot to any folder other than the folder that was the source of the backup.  

If I create a Restore task (Main menu>Create Restore Task) I can't find an option to select anything but the most recent backup dump to use for the restore.  There is no option (that I can find) to select an older snapshot.

If I use the Manage Backup (Main menu>Manage Backup) feature I can select an older snapshot, but I can find no way to change the folder to where I want it restored.

Maybe LuckyBackup doesn't have the capability for which I am looking, but it has so many other features that I am surprised this one is missing.

Any help appreciated.

Ralph

---

### Post by Ralph L on 2010-08-08
I found the answer to my own problem and thought I would post it.  If one brings up Manage Backup under the Task menu (with a backup task selected), then clicks Restore, the next window has an option to change the Restore path.  This does the trick.

Ralph
