### Post by Bucky Ball on 2016-04-21
Hasn't been officially released yet, but you can install it anyway if you like. [Install one of these ISOs]("http://cdimage.ubuntu.com/daily-live/current/") and do these commands regularly:

```
sudo apt update
sudo apt full-upgrade
```

You'll get to the officially released version.
