the best way to install wubi is to [download]("http://www.ubuntu.com/desktop/get-ubuntu/download") the ubuntu image yourself first. You can use [bittorrent download]("http://releases.ubuntu.com/10.04.1/ubuntu-10.04.1-desktop-i386.iso.torrent") - it's much quicker than the wubi installer, and it means that you could reinstall (if required) without downloading the whole image again. Also you can create a bootable cd/usb from the image.

Copy the .iso into the same folder as wubi.exe (get it [here]("http://wubi-installer.org/")) before running wubi.exe.

Note that the latest version of wubi.exe doesn't work with the netbook-edition, xubuntu or mythbuntu due to release compatibility issues (you need an image that is at release 10.04_.1_)

If you are comfortable partitioning - then installing Ubuntu direct is a good idea. If not, then wubi is ideal to try out Ubuntu with good performance and persistence. If you just want to see if Ubuntu will run on your computer, create a live CD or live USB and boot it (Select "try without installing").
