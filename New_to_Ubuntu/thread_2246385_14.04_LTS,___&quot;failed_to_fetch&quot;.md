```
sudo add-apt-repository --remove ppa:fyrmir/livewallpaper-stable
```

```

matthew-laptop:/home/matthew:1 % sudo add-apt-repository ppa:fyrmir/livewallpaper-stable

 More info: https://launchpad.net/~fyrmir/+archive/ubuntu/livewallpaper-stable

```

When you remove the ppa, it empties the sources file after first making a backup with the extension .save.

You can the remove the files using.

```
matthew-laptop:/home/matthew:1 % sudo rm /etc/apt/sources.list.d/fyrmir-livewallpaper-stable-trusty.list*
matthew-laptop:/home/matthew:1 % ls /etc/apt/sources.list.d 
matthew-laptop:/home/matthew:1 % 
```
