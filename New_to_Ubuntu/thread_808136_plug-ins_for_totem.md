The easiest way to install MP3 and AVI plugins is simply to open up Totem, navigate to the file you want to play, and attempt to play it. 

A wizard will open up saying it can't play the file do you want to install the necessary codecs and show you what you need to install. Place a tick against them and click install and away it goes.
