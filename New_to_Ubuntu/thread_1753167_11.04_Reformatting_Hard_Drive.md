### Post by 73ckn797 on 2011-05-08
You could get by with 10Gib for the system partition. Designated as "/" during the setup. Then create a second partition as "/home". Save a smaller, logical partition for a swap space. Typically 1.5 times the amount of RAM, ie... 2Gib RAM = 3Gib swap space. Later if you decided to install a newer release you could keep the /home partition and not lose documents or settings.

I believe the idea is that if you only wish to use open source software and keep your Ubuntu installation completely clean of anything that you can't read the source code of then you have the choice to not install that 3rd party software.
