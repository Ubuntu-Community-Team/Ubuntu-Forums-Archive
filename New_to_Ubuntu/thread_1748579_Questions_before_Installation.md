### Post by ClientAlive on 2011-05-03
I think Opera is a fantastic web browser! Absolutely love it. Some people like google chromium though too.

Oracle VM Virtual Machine is essential if you plan to be learning/ practicing code or experimenting. Even if it comes to running into a problem and wanting to try something that was suggested on one of the forums you can use it to do a practice run and be sure it's going to work and not hurt anything.

You can run an operating system inside the virtual machine and it isn't installed on your system or anything. It doesn't matter what happens to that operating system inside the virtual machine it won't hurt your actual system. So, for instance, you can run the same operating system you are using on you your system inside the virtual machine and practice any code you want to try, make mistakes, whatever.

Even if you totally mess that o/s up all kinds of ways - in the virtual machine - it won't touch your actual system. You can also use it to run different operating systems just to play around and see what they are like. Any o/s will run in it as far as I know - windows, linux, maybe even mac.

### Post by 3rdalbum on 2011-05-03
> **gumshoegrant said:**
> Which Internet browser?

Ubuntu comes with Firefox, but I prefer Chromium (which is the open-source, more often-updated version of Google Chrome.

> Email Client?

Ubuntu comes with Evolution, but some people use Thunderbird.

> pdf reader?

Don't even bother to download Adobe Reader, just stick with Ubuntu's default "Evince".

> photo album?

Stick with Ubuntu's default Shotwell.

> video viewer like what adobe is to windows?

Same answer as on Windows: Use VLC. Ubuntu comes with Totem, but VLC can play more.

> music player?

Well, there's about five million music players on Linux. Ubuntu comes with Banshee, but I still like Rhythmbox (the one it used to come with).

I'd also advise you to download the latest Ubuntu version, which is 11.04. Don't bother with 9.04 - it's out of support now and you can't download any software for it anymore. Keep up with the latest version.

There's so much software easily discoverable for Ubuntu with the Ubuntu Software Center program, so try out whatever software you want to; if you really want, you can have all five million music players installed and switch between them :-)
