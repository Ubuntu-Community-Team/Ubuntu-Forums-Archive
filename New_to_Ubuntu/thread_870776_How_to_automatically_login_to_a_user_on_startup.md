### Post by Krydahl on 2008-07-26
System -> Admin -> Login Window. Select security tab. Tick the enable automatic login box and specify the user.

### Post by Norfolk 'n' Good on 2008-07-26
Hope this helps

(1) Go to... System > Administration > Login Window

(2) A message box will appear prompting for your password... Do it.

(3) A new box will open called 'Login Window Preference'

(4) On the tabs at the top click on 'Security'

(5) Check the box 'Enable Automatic Login'

(6) Click on the drop-down arrow and search for your user name and select

That's it.
