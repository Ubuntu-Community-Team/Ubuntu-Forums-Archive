### Post by grahammechanical on 2016-03-18
When you extracted the package you should have ended up with a .deb file among other things. Did you? The next step would have been to install the program.

Open the file manager. Right click the .deb file and select to open with Ubuntu Software Centre. And the software centre will install the program. And somewhere in the KDE desktop there should be a menu item to launch the application. In Ubuntu + Unity we would look for an icon in the Dash.

Unless of course, this is a program that is loaded from a command line then we would have to open the terminal and run the respective command to launch the program. I cannot see any installation instructions on the web site.
