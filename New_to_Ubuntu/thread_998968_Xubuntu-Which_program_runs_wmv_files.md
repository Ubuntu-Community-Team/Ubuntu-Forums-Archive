### Post by Zalbor on 2008-12-01
First install the package called "xubuntu-restricted-extras". Also add the medibuntu repository (you can easily find out how by searching) and install "non-free-codecs". Now any media player should be able to play wmv files...

Other than that, I recommend VLC, which has its own codecs for everything and is a very good player as well.
