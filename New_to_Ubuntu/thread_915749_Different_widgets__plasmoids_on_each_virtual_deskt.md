### Post by HTML-insane on 2008-09-10
You might want to suggest that. However, at the moment at least, this isn't *exactly* possible (as Plasma draws the desktop AND the widgets as opposed to Superkaramba, which was like an application, it doesn't quite work like that). However, there is a function similar to the one you have just mentioned - the Zoom function in the toolbox. When zoomed out, the toolbox gives an additional option to, "add activity". Then, you can use the two different desktops for different plasmoids and/or wallpapers, then zoom and select between the two.

### Post by Ms_Angel_D on 2009-12-26
> **tarahmarie said:**
> I have compiz-fusion, KDE 4.1.1, and Kubuntu 8.04.  

How do I set it up so that there are only the widgets I choose on the 4 different cube faces?  Like the system widgets on the first desktop, the entertainment widgets on the second, and so on??


You don't actually need compiz for this as plasma & kwin handle it great on it's own Here's how:

click on the cashew icon on the top right of your desktop and click &#8220;zoom out&#8221;. Your desktop will be shrunk down, and a menu will appear on the left. Click &#8220;configure plasma&#8221; and check the box that says &#8220;different activity for each desktop&#8221;.

Zoom back in to your original desktop/activity, and you should never have to zoom out again. From this point forward, each of your virtual desktops will have separate plasma activities, meaning you can use different sets of kde widgets on each desktop. This will also allow you to have a different wallpaper on each virtual desktop..
