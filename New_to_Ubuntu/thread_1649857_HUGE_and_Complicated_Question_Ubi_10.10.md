### Post by Jerry N on 2010-12-20
Linux does not run Windows software.  Wine is available in the repositories and runs _**some**_Windows software, including some versions of Photoshop and Microsoft Word.  Gimp might satisfactorily substitute for Photoshop, depending on how much of Photoshop's capability you really use, and OpenOffice.org (installed with Ubuntu) will read and write Word files and looks much like Word 2003 (not Word 2007).  It is not compatible with Word macros and, if you have complex formatting, might mess up that formatting.

There are folks that will tell you that for any Windows program you can imagine there is a Linux program that will do the job much better.  That, of course, is only somewhat true.

Make specific queries about the things you need to do and there are a lot of folks on these forums that will try to help.

Jerry

### Post by 3rdalbum on 2010-12-20
Ubuntu is not Windows, and it doesn't natively run Windows programs. There is a program called Wine that can run some (<25%) Windows programs; there are lots of HOWTOs around about how to install and run Wine on Ubuntu 10.10, take a look at Google for more information about this.

Wine should really only be used as a last resort, if you REALLY need a particular program to be able to run. For things like image editors and music players, there are native Ubuntu programs to fulfil these tasks.

Yes most of those old programs are going to collect dust now.  There is a Linux version of Nero, which I prefer the free Linux CD/DVD writing software, but it will cost you about $40.  As has been said, Wine will run some Windows programs.  There is an advanced version called Crossover (about $40) that I have found to be much friendlier than Wine.  Microsoft Excel 97, Word 97, and Powerpoint 97 work very well under Crossover.  Old Photoshop Elements 2.0 installs under Crossover but only partially works.  If you are a Windows "Power User", then you are going to be frustrated when you try to convert to Linux application programs.  On the other hand, if you are not a power user (yet) and want to convert to Ubuntu Linux, then just abandon the old Windows programs and learn the Linux programs.  Be sure to back up your old files in a form that can be recovered by the Linux programs, though.
