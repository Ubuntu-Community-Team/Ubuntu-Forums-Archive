### Post by bennachie on 2008-11-09
If you burn a LiveCD from the iso, all you should have to do is to fire up Windows, then insert the LiveCD in the drive. You should be presented with an Autoplay dialogue offering you the option to have Ubuntu run within Windows (using Wubi, of course). The process worked fine for me, and the hit on disk speed seemed to be pretty marginal. It's definitely a better way to try out Ubuntu than booting from the LiveCD every time. That works, but is rather like wading through treacle. If you don't like Ubuntu (I do, but YMMV) all you have to do is uninstall it in the usual Windows way.
