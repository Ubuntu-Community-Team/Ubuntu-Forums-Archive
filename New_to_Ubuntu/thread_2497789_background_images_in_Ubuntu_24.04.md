### Post by ActionParsnip on 2024-05-17
They're in the ubuntu-wallpapers package
[https://packages.ubuntu.com/oracular/ubuntu-wallpapers](https://packages.ubuntu.com/oracular/ubuntu-wallpapers)

They take up very little space so I wouldn't suggest you go diving through the OS deleting the wallpapers.

You can download new ones from Google Images. Just set a high resolution and search. There are desktop wallpaper websites but I'd say these were searched using Google

### Post by BBQdave on 2024-05-19
+1

And I would not delete files in a new OS. Give yourself time to learn and explore Ubuntu.

As far as background images: click on upper right corner icons which brings up menu > select system icon (spoke wheel) > select APPEARANCE

You can add your own pics for background through APPEARANCE. I store my pics in the PICTURES folder and pull background pic from there.

### Post by coley9225 on 2024-05-20
In my case, I created a directory in my home directory, 'wallpapers'. If I see an image that I would use for a wallpaper, I download it to that directory. I use [https://pixabay.com/editors_choice/](https://pixabay.com/editors_choice/). They have a huge collection of photos, vector graphics, and more. I think you need to create an account(free), and the images have multiple resolutions, and are free for personal use. When I want to change my wallpaper, I go into my desktop settings, and enter the path to the image I want to use. You can also use pictures from anywhere in your computer, family, vacation pics, whatever. Another site you can check out is [https://unsplash.com/wallpapers/desktop](https://unsplash.com/wallpapers/desktop).
