### Post by markbuntu on 2009-06-19
Try QCaD. It is a very good CAD app. It can import and export turbocad and autocad DXF files. There are also a bunch of other cad programs but QCad is very professional quality. I use to use autcad and turbocad and switching to Qcad was very easy. I think it has some limitation about using the latest autocad formats but DXF works fine.
