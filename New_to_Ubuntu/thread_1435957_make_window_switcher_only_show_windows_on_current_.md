### Post by The Cog on 2010-03-22
Is it possible to make the KDE window switcher in the panel only show the window titles from the current page/desktop? By default it shows the window titles from all desktops, and as a result gets cluttered very quickly.

SOLVED. 
right-click some empty space in the taskbar or click the panel button on the right, then right-click the taskbar area to configure it
Thanks to chatzilla and thiago_home.
