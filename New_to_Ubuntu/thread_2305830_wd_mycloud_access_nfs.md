### Post by SeijiSensei on 2015-12-09
First, see if you can access the share from Dolphin.  In the address bar, type
```
smb://server/share
```
Replacing "server" with the hostname or IP of the server, and "share" with the shared directory you want to access.  If you have authentication enabled on the Windows side, you'll be asked for your Windows username and password just like you would accessing the share from another Windows machine.
