### Post by kostkon on 2008-09-14
You can install the [*pidgin-guifications*]("http://packages.ubuntu.com/hardy/pidgin-guifications") (for MSN style notifications) or the [*pidgin-libnotify*]("http://packages.ubuntu.com/hardy/pidgin-libnotify") (for getting bubbles) Pidgin plugin.

Don't forget to activate it in *Pidgin*, whichever you install, from *Tools -> Plug-ins*.
