### Post by thulefoth on 2010-02-05
You can also try to get a 'fully free' version, if you like to support the Open Source stuff.

Click Applications, and at the bottom click Ubuntu Software Center.  Go up to the little search-box and type in "mpeg2", no quotes.  There's some stuff.

Also go System > Administrator > Synaptic Package Manager.  Lots more robust search ability on this tool.

Things you get from these official Ubuntu repositories are not "restricted", like they are with the previous option.
