### Post by flyingsolo on 2011-09-07
So, in a way, I've fixed it by installing fglrx.
When I rebooted, there was an error box flashed for a short time that said something to the effect, "Your hardware isn't up to scratch for Unity; select Classic view for Ubuntu." and it carried on booting to my old-fashioned Ubuntu desktop which does now display properly (ie: I can see & read stuff!).
I didn't actually select classic view, it just came up with it.  So, bottom line is my Dell is too old for modern Ubuntu.  But then again I hear a lot of people grousing about Unity anyway and I'm Ok with what I've got.:D
Maybe this helps someone else trying to work with old computer guts!
