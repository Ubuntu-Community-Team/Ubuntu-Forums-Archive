### Post by Dedoimedo on 2008-11-27
Hello,

Open the terminal and then type:

```
which kdocker
```

This should tell you the path to the executable. So you can create a shortcut now.

Likewise, start typing kdock in the terminal and then hit tab twice. This will display all the possible choices for kdock* that exist in your path.

Lastly, look through the menus, try to find it.
