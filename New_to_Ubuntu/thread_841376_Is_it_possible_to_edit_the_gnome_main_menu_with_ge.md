### Post by jw5801 on 2008-06-26
Not easily no. The Gnome menu is made up of a bunch of .desktop files stored in /usr/share/applications/. These have a very specific structure and can often be difficult to get to work as you would like manually. As there is no one file that handles the entire menu, it'd be much more tedious to use a text editor than it would be to use Alacarte.
