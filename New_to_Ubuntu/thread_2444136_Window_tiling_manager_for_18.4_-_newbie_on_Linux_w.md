### Post by deadflowr on 2020-05-25
> Are there any window tiling managers available in this 'Ubuntu Software' place?
Ubuntu Software is for desktop applications more or less.
It doesn't do a good job of listing desktop environment/window managers, terminal applications, or any number of non-desktop application related tools.

To install or find packages like window managers it's best to install synaptic package manager,
or learn the apt package management command line system.
