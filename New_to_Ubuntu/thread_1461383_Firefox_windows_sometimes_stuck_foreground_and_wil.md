### Post by duanedesign on 2010-04-24
Might check that popups are blocked in Firefox. This will give you the option to open a pop up if you want, but block them otherwise.

Edit > Preferences > Content > Block Pop Up Windows

The keyboard shortcut to close a window is 

Ctrl + w
