Some answers for you:

Button3 is right click on the mouse. Button 6, I'm not sure.

The fold effect you're talking about might be the airplane.

The background folder is /usr/share/backgrounds. Any picture files you put there will show up in the appearance properties. However, you can choose backgrounds in other folders as well if you browse from the Appearance Properties.

You can put different backgrounds on each desktop with a little effort. You'll have to do a little googling though. To rotate the cube, use Ctrl+Alt+leftarrow and Ctrl+Alt+RightArrow.

### Post by mcduck on 2009-02-02
button 1 = left button
button 2 = middle button (wheel click)
button 3 = right button
button 4 = mouse wheel up
button 5 = mouse wheel down

after that the rest are all the extra mouse buttons.
