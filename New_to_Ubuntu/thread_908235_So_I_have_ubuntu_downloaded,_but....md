The only details you need to give for your Free CD from Canonical is name and address (and country). No bank account info or anything like that.

Alternatives.

If you have windows installed you can use Wubi to install Ubuntu from within Windows.

Not that this effects Ubuntu's performace a bit (not that much) and there might be some issues every now and than that wouldn't occur on a real install.

You can install it from a USB disk. If you computers can boot from usb (older ones can't).

You can run Ubuntu virtually using Virtualbox or something similar.

Or you can take out your HDD, take it to a friends house and install Ubuntu on your HDD using his pc.
[I]
Note: you are going to have to look long and hard to still find a linux distro you can install from a floppy.
