I used to use Vuze (not anymore), but you probably don't need to update vuze if you installed it from synaptic. I do remember that it was trying to update itself all the time, that is true. For what I remember It downloads its own update as a torrent so what I did to stop the problem was to right-click on the downloading torrent and stop it. This also stopped the endless request.

Just as an explanation, if you downloaded vuze via Synaptic this is not the last version (which is perfectly OK). If you do want the last version then fully remove it via Synaptic and download the last version from vuze website.
