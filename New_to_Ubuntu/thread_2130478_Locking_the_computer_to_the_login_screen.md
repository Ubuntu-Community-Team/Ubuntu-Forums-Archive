### Post by Cmiller2 on 2013-03-29
The lock screen looks a little prettier if you change you the desktop background. Juat right click the desktop, and click, "Change Desktop Background." When you lock the screen you'll still see the same desktop background that you have set for your User Account. For example, I have some clouds set up for my background. Its give a nice blue color to my screen, which is in fact my favorite color. :KS
