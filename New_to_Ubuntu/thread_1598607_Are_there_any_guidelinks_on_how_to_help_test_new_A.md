### Post by bennachie on 2010-10-16
Well, there's actually quite a lot that a relatively inexperienced user can do to help, especially when ISO testing time comes around. It's really all about trying out new versions and submitting useful bug reports when (as will inevitably be the case) something goes wrong. The process can be frustrating, but the rewards always outweigh the costs.

You might want to bookmark the newly opened Natty Narwhal forum ([http://ubuntuforums.org/forumdisplay.php?f=394](http://ubuntuforums.org/forumdisplay.php?f=394)) and begin by reading the excellent FAQ sticky from 23meg that appears right up front.
