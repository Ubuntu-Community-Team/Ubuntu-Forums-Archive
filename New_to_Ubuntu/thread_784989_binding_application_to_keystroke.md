### Post by tjwoosta on 2008-05-07
well i know a way to make some things open with a keystroke


if you go to System-Preferences-Keyboard Shortcuts

you scroll down and you will see one that you can set to run a terminal and theres one to open a web browser 
(theres probably other things you can set also)

those are the two i use, i have mine set so ctrl+t opens a terminal and ctrl+f opens firefox
