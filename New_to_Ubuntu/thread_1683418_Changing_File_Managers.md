### Post by TeoBigusGeekus on 2011-02-07
If your friend tries to uninstall nautilus, half the gnome desktop package will be uninstalled.
So, no... if he wants to use a different file manager, he'd better not touch nautilus.

All the above of course apply in the case where he sticks with gnome. If he wants to switch to xfce or lxde, then he can do whatever he wants with nautilus.

It depends.
The file manager for ubuntu is nautilus, which is a special case, as mentioned in my previous post.

Banshee with Amarok...
Network manager with Wicd...

...
the list is endless, but you get the idea.
In every case, you can keep the old program or simply delete it.

### Post by hansolo4949 on 2011-02-07
As far as I know, when you install a new file manager, Ubuntu asks you which one you want as the default, so your friend should be in good shape. If he ever wants to go back to another file manager, he can just uninstall the one he downloaded, and it should default back to nautilus.
