### Post by pi.boy.travis on 2008-11-30
If you are running from the Live CD, you can access your Windows partition.  Just go to the "Places" menu in the upper left hand corner, and your drive should be there.

You can install iTunes via  utility called Wine, but you will not get iPod functionality due to the fact that Wine does not support USB.  There are several ways you can use your iPod, however.  Check out: [https://help.ubuntu.com/community/PortableDevices/iPod](https://help.ubuntu.com/community/PortableDevices/iPod)

You can actually have both Windows and Ubuntu on your computer.  Check out: [https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)

Hope this helps!

### Post by sofasurfer on 2008-11-30
I would back up my Windbag files and then install them into your Ubuntu system once you have it installed.

However, most people keep Windbag when they install Linux for the first time. Us Linux users don't encourage it but it will make you feel safe for a little while as you learn how to make Linux do stuff. 

If you want to keep Windbag for a while just shrink your Windbag partition so that there is some free space on your hard drive. Then when you install Ubuntu you will be given the choice to "install on largest continuous free space". Choose this option and you should be good to go with both operating systems on your computer. And then you should be able to copy file from Windbag to Ubuntu.
