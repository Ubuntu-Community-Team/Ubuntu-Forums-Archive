### Post by hermes0710 on 2008-04-25
Select from the list of the partition manager the drive that is currently hosting your previous ubuntu installation, click edit, tick the "format" option and from the drop down menu select the "/" option.

This will tell to the installer that this specific drive is going to host your new os and since you are going to format it the previous one will be replaced by Hardy.

Good luck
