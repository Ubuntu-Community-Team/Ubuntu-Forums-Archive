### Post by Rubi1200 on 2010-08-23
Hi and welcome to the forums :-)

The easiest method is to open Applications > Ubuntu Software Center and search for "flash"

You have 2 options, either install just the Adobe Flashplayer or Ubuntu Restricted Extras which will also download and install codecs, fonts, and other things needed for normal computing usage.

The installation is done in the USC and is quite straightforward.

Good luck!
