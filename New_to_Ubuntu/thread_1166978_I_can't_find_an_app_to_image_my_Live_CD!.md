### Post by Hospadar on 2009-05-22
An important note, if you are using XP (or I think vista as well) and you don't have admin privileges, you won't be able to burn cd's (who knows why that is an important security decision)

You can use something like nero burnrights to give non-administrative users burn access if you need.

If indeed it is that your cd drive is for real bad, either
a) get a new one, they are so cheap these days
b) go to the library/anywhere they have public machines you could use that might have burners
c) find a mac or linux machine, they can all by default burn iso's without added software
d) order a free linux cd from ubuntu, or drop a couple bucks for one (I think the free ones can be slow to ship)
