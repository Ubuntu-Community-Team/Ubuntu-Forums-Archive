### Post by PointyWombat on 2008-04-22
Windows work pretty much the same way as you're probably used to so you most likely have a problem.

[LIST]
[*]Did you run a live CD before you installed to HD, and was it a problem there?

[*]Do your windows have buttons to let you minimize, maximize and close windows? (Is windows manager working?)

[*]Are you using any advanced desktop effects (compiz) ?

[*]Do all windows snap to the upper-left? Do new windows cover completely previous windows?

[*]If you double-click on the top of a window, does it maximize?
[/LIST]
