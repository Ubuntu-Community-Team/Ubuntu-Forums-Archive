### Post by Cheesehead on 2012-12-03
> **philpense said:**
> 
Windows Vista machine set to boot to the Lubuntu Boot CD

What does show up?
Nothing?
Error message?
Any message?
Dots?
Colors?

Does the CD work in a different machine?

The most common reason for the described failure (fails to boot from CD) is that the user didn't burn the ISO (extracted and copied the files instead), or the burn failed or errored.

Another common reason is if the BIOS is set to try HDD before CD during boot, and the user didn't override it.
