### Post by fiyer on 2010-08-25
It looks like I have at least partially, if not completely figured it out.

(This is all within the gui and not in the terminal.)

Part 1: Went into System, Administration, Language Support, let it install the language packs. Set keyboard input to "ibus". Restarted machine.

Part 2: System, Preferences, Ibus preferences, scrolled down "Select Input method," found language of choice, and chose "phonetic." 

However, it seems that not all indic languages have the option to choose "phonetic" as the input.  For some, you may need to learn a new keyboard layout.
