### Post by eightmillion on 2008-07-28
Did you check to see if your xserver-xorg-video-intel is the newest version? If you didn't please do that with the command:
> sudo apt-get install xserver-xorg-video-intel
Add this to your xorg.conf and restart X and run the 'glxinfo | grep direct' again.
```

Section "Module"
        Load "i2c"
        Load "bitmap"
        Load "ddc"
        Load "dri"
        Load "extmod"
        Load "freetype"
        Load "glx"
        Load "dbe"
        Load "int10"
        Load "vbe"
EndSection

```

---

### Post by eightmillion on 2008-07-28
Also, make sure the package, libgl1-mesa-dri is installed and that xserver-xgl is not installed:
```

sudo apt-get install libgl1-mesa-dri 
sudo apt-get remove xserver-xgl

```
