### Post by LowSky on 2011-11-05
you are not going to have any luck playing most of those games on Linux with only Intel hardware. Maybe Counter strike source IF, only IF you have newer Intel HD 3000. The rest is a push even on LOW settings.

Lets be fair, even on Windows those games couldn't run very well except maybe CS:S.

The first stop is making sure WINE and/or PlayOnLinux are installed. The next is looking up those game titles and seeing if and how people may have got them working WineHQ.org is a great place to check that.
