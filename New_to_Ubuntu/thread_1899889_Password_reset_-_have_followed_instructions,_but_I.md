### Post by GWBouge on 2011-12-24
You can tell it to format the drive if you want to, or you can just select the 'Use Entire Disk' option on the installer.  Either way, 8.04 will get overwritten.  (Edit:  If it's dual-booting (ie, if both 8.04 and Windows are installed), you'll want to do the Manual option, and specify and format the 8.04 partition.)

8.04 was only supported until April 2011, so even if you do get past the password, you won't receive any more security updates.  10.04 will be supported until April 2013, and non-LTS releases are supported for 18 months.  Starting with 12.04, Desktop LTS releases will be supported for 5 years.
