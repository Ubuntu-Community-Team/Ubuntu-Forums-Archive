### Post by Mark_in_Hollywood on 2011-02-10
At the top of your screen, you should see:

Applications

Places

System

Pull down System, pull down Keyboard. Select Keyboard.

On the window that opens, once Keyboard is selected, find at the top, Keyboard Preferences.

Select the Layouts tab.

Read the window and see if USA is listed. If not listed, click the Add radio button. Once that screen opens, select "Country". Pull that down to the Keyboard you want. I'm guessing USA. After the layout is selected, click Add and then select the new KB by moving it up.

Close out of the KB config. All should be well.
