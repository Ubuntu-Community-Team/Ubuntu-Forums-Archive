### Post by Paqman on 2009-11-30
You'll want the desktop version.

Server does what it sounds like: it's designed for installing onto a server, and can set up common server configurations (eg: headless LAMP server) during installation.

Alternate is a text-based installer that has some different options, mostly for advanced users or installing on low-RAM systems.

Netbook Remix is designed for netbooks, and has a different interface.

Depending on what processor your machine has you've probably got the option of either 32-bit (i386) or 64-bit (amd64). 32-bit is guaranteed to work on all machines, but if you've got a 64-bit chip then you might as well run a 64-bit OS.
