### Post by nhasian on 2008-12-20
theres a plugin for Compiz Fusion called *Place Windows* to have an application open in the viewport of your choice:

[http://wiki.compiz-fusion.org/Plugins/Place](http://wiki.compiz-fusion.org/Plugins/Place)

if you dont already have the compizconfig-settings-manager installed you can install it with:

```
sudo apt-get install compizconfig-settings-manager
```
