### Post by the8thstar on 2009-11-27
1. My graphic card is a NVIDIA GeForce 7600 GS. is there drivers for linux avalible for it? 

Yes, you can install the proprietary drivers for your nVidia gfx directly in Ubuntu.

2. Can i run World of warcraft fine in Wine, While i have skype & sometimes Ventrilo running?

You can run WoW in Wine. You can install the linux version of Skype. I don't know about Ventrilo.

3. Can i access my banks website & login fine. (My bank is a swedish bank called Swedbank)

I check my acount online in Ubuntu the same I used to with Windows.

4. can i install ubuntu from a USB memory?

Yes.

My best piece of advice for you is to burn a liveCD version and try on your computer just to see if all your hardware is detected and configured properly (webcam, printer, scanner, etc.). If most of it works (or better if all work) then you can safely install Ubuntu on your machine next to Windows.

### Post by Bartender on 2009-11-27
Check UbuntuHCL.org.  Here's the video card page -

[http://www.ubuntuhcl.org/browse/search?worksoutofthebox=on&category=6](http://www.ubuntuhcl.org/browse/search?worksoutofthebox=on&category=6)

+1 on dual-booting rather than kicking XP to the curb.  There are some things that just can't be done yet in Linux.  Like upgrading  Sansa Fuze firmware.  Little things like that.

If you were seriously thinking about tossing XP, and you have broadband at home, why not wipe the HDD clean, set aside a primary NTFS partition for XP, install it to that partition, update XP and run it for a coupla days to make sure it's all working, then install Ubuntu to the rest of the HDD?

XP will run a lot better with a fresh install, and if you use Linux for your web work XP will not degrade so quickly.
