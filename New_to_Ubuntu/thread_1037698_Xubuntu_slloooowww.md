### Post by randcoop on 2009-01-12
You will need to offer more explanation of the problem in order to get help, I think.  In addition to the hardware specifications, you should identify specific programs that are running that slow the system down.  If there are no specific programs, then is it when it boots?  Or is it slow when you are on the Internet?  

One possible place to start diagnosing the problem would be to run htop.  You'll find it in the Xubuntu menu under System.  Htop lists all the programs you're running and how much memory and processor they're using.  If you see anything that is using memory in the 80-95% range, it's likely that program is your problem.
