### Post by Geoff Blend on 2011-07-29
Yes you can still add the cub effect on your 11.04 installation. You have to add the compizconfig-settings-manager from the synaptic pacage manager. If you are using classic Ubuntu it will work perfect after the installation but if you are using unity then you will have to find a how to guide in here cause the cube effect doesen't work perfectly in unity after installation
