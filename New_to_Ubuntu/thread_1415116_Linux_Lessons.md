### Post by TBABill on 2010-02-24
+1 on the sticking to Windows during a class without assistance. Some proprietary items do not necessarily work well with Linux like they do with Windows when it comes to drivers and in an educational environment are you certain which hardware items you may need? (i.e, webcam, sound, printer, scanner, etc) Those items can be difficult depending on the overall computer configuration and if you get stuck with no way out you may be playing catch-up if they will then allow you to switch back to windows (assuming the instructor would allow it).
 
If it's just some basic work that you have researched enough to know you won't need to print, use peripherals you are unsure of or other issues, Mint is what I'd recommend to try first and only because it comes preinstalled with everything you'd need to get online, use online media (streaming video, etc.) and just operate without having to figure out as many things as you may need to in Ubuntu. Debian would be even more difficult for a new user. Ubuntu is pretty easy but if you are not familiar with it, learning it in the classroom could put time constraints on you that you cannot meet (like trying to get answers in the forums quickly). Of those you mentioned Mint is easiest, which is what it was designed to be and it's based on Ubuntu, which is based on Debian, so you get the best of all worlds that way.
 
Not sure I'd risk being a noob and using it as my classroom OS for an IT class all at the same time. Weigh the risks or check to see if dual-booting both would be ok till you are certain you have no problems using the version of Linux you choose.  Never hurts to ask.
