That makes no sense. A tar.gz file is simply a compressed archive, like a .zip file or a .rar file. Its contents could be anything.

The easy way to install programs is to use the repositories. That's what they're for.

### Post by oldrocker99 on 2019-01-28
Alien is only used to install .rpm packages, for Red Hat and Fedora and all its forks.

Tar.gz files are almost always compilable source code. It isn't all that hard to compile programs, and the program will produce a program that is built for your own system.

Look at this:[https://www.howtogeek.com/105413/how-to-compile-and-install-from-source-on-ubuntu/](https://www.howtogeek.com/105413/how-to-compile-and-install-from-source-on-ubuntu/).
