### Post by Master's Property on 2008-05-09
Hi Zooburner. Migration is pretty easy if you have a guide. I'd be happy to help anyway I can.

Linux is known for it stability, security, efficiency, and robust operation. But as with it's counterpart (windows) it has problems too. Thats the great thing about open source, you CAN fix it, and without paying a dime. Before you throw away the sound card, did you try to set the audio properties to use that type of device? Go to preferences, sound, and check each device in the drop down menu with the test button till you find the 1 you need. Also, I think Ubuntu can handle up to 3 gigs, but not a full 4. For the most part, Ubuntu is the perfect desktop. It does recognize most MB's resources, but to make sure you should use live disc for about a week to see if you run into any problems that might present themselves in everyday life.
