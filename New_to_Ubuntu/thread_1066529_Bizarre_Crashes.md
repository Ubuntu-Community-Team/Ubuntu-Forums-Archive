### Post by icyest on 2009-02-11
Yeah, despite popular belief (and might I add, very VERY popular belief), Linux tends to crash. I've heard that problem so many times, it almost feels like Windows Me. 

You might want to turn off any compiz if you have running in the background. Since you're not dual booting with Windows (as that has been known to cause god knows what problems), you should restart it several times. That somehow worked for me. Check your error log as well.
