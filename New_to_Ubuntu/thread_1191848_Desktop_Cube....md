### Post by evilkastel on 2009-06-19
Probably that was not a stupid question. Seems like you set only 2 desktops, for the cube you have to set 4. You can do that as easy as right clicking the workspace switcher panel applet, and selecting preferences. You have to set 1 row 4 columns. If you experience any further odd behavior, make sure your video card 3d driver is working properly.
