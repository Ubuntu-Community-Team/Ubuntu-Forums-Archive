### Post by Bucky Ball on 2014-09-16
Welcome. The problem with using an older release is that 13.04 anything has reached end-of-life and therefore receives no updates, including security updates, and is no longer supported by Canonical or these forums. So you can keep using it, but there won't be a lot of help available if anything goes wrong with the system, and the system will become more vunerable to security breaches as time goes on. 

Please [see here]("http://ubuntuforums.org/showthread.php?t=2229730") for further details.

But I'm not convinced you are actually using 13.04. Please confirm. Is '13.0.24' the number of the kernel you are choosing and is it correct?
