### Post by candtalan on 2008-07-10
> **sudeepk said:**
> HI iam new to linux and i dont know which distribution to use.. ubuntu or debian? whats the difference?

Welcome!

Ubuntu is based on Debian, so it uses the same programs (packages) and very similar philosophy and structures.
Ubuntu is intended to be easy to use, and the forums - such as here - can support newcomers with no experience ok.

Debian is often used by more experienced users, although newcomers can certainly use it ok, its just that beginners will often use ubuntu initially.

An easy way to start is to get a ubuntu Live CD and run it from the CD drive. It will not install unless you ask it to and will not otherwise change anything in your machine. Ubuntu Live CD will also run inside Windows. This way you can install into Windows (wubi facility) and can then choose to use either Windows or Ubuntu. Using this method (wubi) it is safe to install and safe and easy to uninstall too.

Good luck
