### Post by Hobgoblin on 2009-06-27
> **jrev said:**
> Hello all,

I wonder which Ubuntu I could install on my eeepc ?


Which eeePC?

If it's a 7xx I would use Ubuntu 8.10 with the array.org kernel and add on the UNR stuff.  This will give you everything working for minimal effort.

If it's a 900 then Ubuntu 9.04 without UNR and try some of the tweaks for better performance of the Intel graphics.  I found the .30 kernel broke CPU scaling so just did the safe configuration from [this thread](http://ubuntuforums.org/showthread.php?t=1130582&highlight=intel+graphics), you might want to also turn off desktop effects.

I also have a 900A which runs fine with completely stock 9.04

If it's a newer model then I'd try 9.04 and see how it goes.

It's personal preference but I don't think the netbook launcher is needed on the 8.9" (and above) screen.

### Post by t0p on 2009-06-27
I've got an EeePC 701 - the one with a 4 GB SSD.  I have got a 4 GB SD card permanently in the card reader.  That adds up to a total of 8 GB disk space.  Not a lot.

In the past I've installed regular 8.10 modified by RiceeeyTweak, and 9.04 netbook remix.  They both took up too much disk space.  Ubuntu is an awfully large distro!

My most recent installation was the latest eeebuntu base.  This install takes up very little room as it includes very few apps.  I'm still in the process of installing the apps I want on the machine.  But even when I've finished getting all the software I want, the system will still not be too big.  I tend to use not too many of the programs included with vanilla Ubuntu, so their absence on my eeebuntu system leaves a lot of room for the stuff I *do* use.

If disk space is at a premium, eeebuntu base is definitely the way to go!

I had ubuntu 9.04 netbook remix on my eee 701 for a while.  Seemed to me it was really just vanilla ubuntu with tweaked gui.  It might have also been teaked to make it more compatible with netbooks.  But when I ran RiceeeyTweaked Ubuntu 8.10 on my 701, that worked pretty okay too.  And remember: ubuntu netbook remix is a pretty big operating system.  It sure took up a lot of my disk space!

If you get an eee, I would advise the use of eeebuntu as it is tweaked to make it compatible with eee pcs in particular, whereas netbook remix has been remixed to work with netbooks in general.  The eeebuntu base install is cool if you have limited disk space.  If you have plenty of disk, the eeebuntu regular install comes with all the usual apps.
