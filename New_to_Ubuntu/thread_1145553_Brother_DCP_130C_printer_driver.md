### Post by TopEnder on 2009-05-02
John,
There should be printer drivers for the DCP-130C in Synaptic Package Manager, just do a search for DCP-130C, you shold find something like: [COLOR="Blue"]brother-cups-wrapper-bh7[/COLOR]  &  [COLOR="Blue"]brother-lpr-drivers-bh7[/COLOR] for the DCP-130C, install them and you shouls be up and printing in no time.  For the Scanner, I think you will need to go the Brother web site for the Scanner drivers unless they are now in Synaptic Package Manager no doubt I'll find out in a week or two when my Live CD 9.04 arrives.  there are many post that will help you with the scanner side.
