### Post by RedOctober on 2010-02-14
Is there a keyboard key combination that I can use to lock the Ubuntu 9.10 desktop?  My MDs need to move from one exam room to the other, and they don't want to leave the session open so the patients can snoop.

---

### Post by howefield on 2010-02-14
Ctrl + Alt + L should do it, although you can map other keys if desired via System > Preferences > Keyboard Shortcuts.

Or add an applet to the panel as explained here.

[https://help.ubuntu.com/7.04/keeping-safe/C/lock-screen.html](https://help.ubuntu.com/7.04/keeping-safe/C/lock-screen.html)

### Post by nick_goodfate on 2010-02-14
you can add a button in your panel . left click on the panel->add to panel, select "lock screen" and press add .
