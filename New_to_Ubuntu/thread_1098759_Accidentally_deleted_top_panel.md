### Post by leonardo_neo on 2009-03-17
I know what you want :D

Ok, first add a new panel as said above. I guess you want the panel which you got as default?

After you add panel, click right on the new panel---> Add to panel

Now you need to add these to have the default setting

1. Notification Area
2. Window List
3. Main menu
4. Volume control
5. Clock

Well this is not the 'exact' default setting, but with this you will have a functional task bar. After this play around with other add on, and see it for yourself what suits you best. That is the best part of Ubuntu :)
