### Post by anewguy on 2011-10-03
As per spiky001, post the model numbers and the specs of the laptop.

Also, google, yahoo, bing, etc., can be a big friend to you in this case:

put ubuntu 11.04 toshiba <your model here> in the search string and see what comes up.  Usually various threads will show up you can read through.  If there are problems, it doesn't mean you will have them, but sometimes it's best to read through them all so when you do install you can be a little more prepared.

If it's from 2002, it will probably work - the only thing will be things like video driver, wireless driver, etc., as these vary from laptop to laptop.
