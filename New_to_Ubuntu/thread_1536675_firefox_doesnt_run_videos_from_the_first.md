### Post by beew on 2010-07-22
> **mayur.shetye said:**
> Install appropriate version of adobe flash player from [http://get.adobe.com/flashplayer](http://get.adobe.com/flashplayer)


You can get the flashplayer pluggin using synaptic, just make sure you add "multiverse" to your repositories first:  start Synaptic, go to settings > repositories then  check the box that says "Multiverse" (or you may as well check all of them). Then click "Reload" to update your repositories. After that you can find the flashplayer pluggin by searching "flash" for example, in Synaptic's search box. Once you have found it, just click the box to download and install it.
