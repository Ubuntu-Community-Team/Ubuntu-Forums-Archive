### Post by wildman4god on 2009-10-05
check with your schools IT guys it sounds like your network blocks all ports except http and https port, which is what all web browsers use. It is standard security practice to block all ports except what is demmed nessicary, my school also blocks everything except web traffic, the only think to do is use web apps, like use meebo for im and your email provider web interface for email, assuming ofcourse that they don't block those sites.
