### Post by darolu on 2010-11-11
You can fix this by updating the *proposed* package **nautilus-share**.

Find how to install *maverick-proposed* packages here: [https://wiki.ubuntu.com/Testing/EnableProposed](https://wiki.ubuntu.com/Testing/EnableProposed)

You will find several updates in the new repository, install them if you like (some of them may change your Ubuntu system's aspect) but you only need the "**nautilus-share**" one.
