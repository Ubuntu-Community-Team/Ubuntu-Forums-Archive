### Post by jfloydb on 2011-10-25
I have a bootable lubuntu ISO on a USB Drive. It works/boots fine on any Windows XP machine. However, I tried to boot it on an HP Pavilion dm1 (piece of junk) machine that has Windows 7 Home Premium. I pressed f12 (and tried several other function keys) while starting, but none seem to give me the option of booting from a/the USB drive. What can I do in order to boot from the USB drive in Windows 7? Any Suggestions? Thanks...

---

### Post by Rex Bouwense on 2011-10-25
Check to see if the flash drive is recognized as another hard drive.  That happens on some computers.

---

### Post by wolfen69 on 2011-10-25
Press F2 or del to enter BIOS and see if it is listed as a boot device.

---

### Post by jfloydb on 2011-10-25
Thanks for the response. As it turns out, I had to press "escape" (which got me into the correct Bios) and then press "F9" in order to boot from the USB drive. I am writing now from the live lubuntu ISO. I'm not very enthusiastic about this notebook/netbook (HP Pavilion dm1, mainly because the touch pad is so flaky), but at least the wifi is working...
