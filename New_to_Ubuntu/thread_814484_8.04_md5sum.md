### Post by FuturePilot on 2008-05-31
MD5 can be found here
[http://releases.ubuntu.com/8.04/]("http://releases.ubuntu.com/8.04/")

To check it just do
```
md5sum path/to/ubuntu.iso
```
Compare to the output to the correct image name in the MD5SUMS file in the above link.

### Post by Rocket2DMn on 2008-05-31
That /path/to/filename is not actually what gets typed, he just means that you have to tell md5sum exactly where the file is.  Have a look here - [https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)
You can either "cd" to the directory you saved the .iso file to or just type out the full path.
