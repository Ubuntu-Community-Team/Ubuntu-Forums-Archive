### Post by vDub2000 on 2012-05-05
This may seem like an odd question. I installed Kubuntu 12.04 because I wanted to check out what KDE has to offer, but I don't really care for any of the applications that come pre-installed with Kubuntu. What is the easiest way to remove them and install the defaults that come with Ubuntu, but still keep the overall look and feel of Kubuntu itself?

---

### Post by SeijiSensei on 2012-05-05
Just install them as you would normally.  KDE has methods to display GTK+ applications like GIMP.  Try installing Firefox and Thunderbird and see how they look.  You can add them to the Favorites section of the launcher and delete the ones you don't want.  You can also determine which applications handle which tasks in System Settings > Default Applications.  For more fine-grained control, you can assign applications to MIME types in System Settings > File Associations. You can control how GTK+ apps appear in System Settings > Application Appearance.
