### Post by jimmy the saint on 2008-12-18
Compiz has the option, unfortunately, Nautilus (which manages the desktop) places its wallpaper on top of compiz' wallpaper.  There is a long, complex, buggy work around, but it is not fit for this forum.  One solution that does work is to get 4 wallpapers that you want, use gimp to make each one the correct resolution, then combine them all into one long image.  When you make it your background, make sure to select the "centered" option.  I have heard this works well, but I have never tried it myself.
