### Post by mc4man on 2012-03-15
It may not be included by default anymore, run this to see & install if need be
```
sudo apt-get install gnome-search-tool
```

If desired you can go to /usr/share/applications & drag on to the unity launcher, drag onto desktop, otherwise search in the apps dash or add to an existing quicklist - screen
