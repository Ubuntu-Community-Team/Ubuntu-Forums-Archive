### Post by Moop on 2008-05-09
There is no automatic backup with a default install of Ubuntu, so unless you made one there isn't anyway to return to where you started. There isn't a System Restore feature in Ubuntu like in windows. 

Having a separate partition for /home makes a reinstall fairly painless. Once you get it the way you want you can explore ways of making an image or backing up your setup.

---

### Post by kirios on 2008-05-10
As Moop pointed out, the only way to restore your Ubuntu system to its default state is to use an image of the default install. In other words, you need to have made a copy of the Ubuntu partition soon after installing it.If you haven't done this, you will have to reinstall.
