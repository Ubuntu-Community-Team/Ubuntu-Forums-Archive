Installing a theme is done by opening System>Preferences>Appearance clicking install and finding the theme package you downloaded.

### Post by steveneddy on 2008-06-25
If you are running compiz, then get Emerald

```
sudo apt-get install emerald
```

there are some cool theme's there, but if you go to gnome-look.org, you will find a bunch of compiz themes for emerald.

Emerald does the window borders for Compiz.

You have to add the extra icons, wallpaper and other custom touches.

Just read as much as you can, get a few emerald themes and try a few. Play around with the settings. Experiment.
