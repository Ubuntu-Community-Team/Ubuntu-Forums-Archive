### Post by kihjin on 2008-08-03
There are a few ways to get to a command line, ultimately they all perform the same function.

While logged into your desktop press ALT+F2. This will bring up a "Run program" window. Type in xfce4-terminal and press the ENTER key.

Another way, via the menu is to go to Accessories->Terminal

Finally, you can do CTRL+ALT+F1 - This will bring you to a full screen virtual terminal. In order to return back to you r desktop you need to press CTRL+ALT+F7, don't forget that!!
