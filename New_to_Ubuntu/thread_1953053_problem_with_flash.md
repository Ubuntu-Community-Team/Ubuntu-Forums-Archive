### Post by forrestcupp on 2012-04-05
If you can use Firefox, install the [Flash-Aid add-on](https://addons.mozilla.org/en-US/firefox/addon/flash-aid/). When you run the wizard, it will automatically remove all installed instances of Flash, install the latest Flash, and fix any possible problems. If it doesn't work with the 64-bit version, rerun the wizard and try the 32-bit stable version. Flash-Aid will set it up to run in a 64-bit OS.
