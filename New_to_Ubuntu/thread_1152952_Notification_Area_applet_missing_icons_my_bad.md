### Post by aeiah on 2009-05-08
can you right click on the panel and add a new panel item (the item being the notification panel?). if that doesn't work then the easiest thing to do may be to set up a new user. if the new user has the panel the way you want then you can copy that user's settings file to your default user's profile. remember to change the file's owner or just copy and paste config file contents else you may run into trouble.

i understand since you're new you may find those things daunting, but you should be able to find out how to do it by searching this forum or maybe someone else will chime in with instructions. right now im at work, running windows xp, so i cant click through it to tell you where things are.
