### Post by bishounen on 2008-05-11
Hunter,

The problem you are running into is that Bittorrent isn't designed to hide your IP.  Now, you CAN do some things to improve your security.

- First of all, make sure you are using a Bittorrent client that has some security functionality built in.  If you are using Ubuntu (which I assume you are, since you are posting here) then use either apt-get or the add/remove menu option to install the Azureus bittorrent client.

- Once that is installed, spend some time reading through the [FAQ]("http://azureus.sourceforge.net/faq.php") and familiarize yourself with how to set up Azureus.  It's not critically necessary to do this, but it will improve your experience with Azureus.

- Load up the [Safe Peer]("http://azureus.sourceforge.net/plugin_details.php?plugin=safepeer") java plugin for Azureus.  This will block known "bad guy" IP's from connecting to you and is regularly updated via the Peer Guardian block list.

that should go a long way to protecting you, but it's not perfect.  There really is no perfect privacy and security on the Internet, only levels of security and privacy.

Keep in mind too, that regardless of what materials you are downloading via Bittorrent, the chances of being sued because of something you downloaded are **less** than the chances of being struck by lightning.

Happy downloading!
