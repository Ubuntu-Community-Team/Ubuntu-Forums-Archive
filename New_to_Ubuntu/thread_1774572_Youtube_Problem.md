### Post by beew on 2011-06-03
> **joe0134 said:**
> Ok when I watch youtube videos in full screen it lags the video but not on small screen it never did this on windows 7 so i don't know what it could be.

Probably because Flash is not optimized for Linux? Install the FF plugin FlashVideoReplacer, or if you don't care about watching Youtube from the browser, install minitube which is a light, fast and smooth youtube viewer 

If you want to try minitube, install an up to date version from here

[https://launchpad.net/~nilarimogard/+archive/webupd8]("https://launchpad.net/%7Enilarimogard/+archive/webupd8")  The version is Ubuntu's repo is old and broken.

FVR allows you to watch Youtube "normally" through the browser, but the catch is it has to use the browser, which itself takes up a lot of resources. Minitube is a dedicated Youtube watcher so it is even more efficient on cpu.

There is now a third option, use UMplayer which is a frontend to mplayer and you can find it from the same ppa above.
