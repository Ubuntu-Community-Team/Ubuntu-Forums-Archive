1. You don't have to install anything off the CD.  Actually I doubt it has Linux drivers anyway.  But the Logitech's should already have a driver with ubuntu.  similarly, if it says Skype on the box that may only mean "Skype for Windows XP" or similar.  

2. if one app doesn't work try another.  webcam apps on linux can use different methods so even if, say Skype doesn't work, Cheese or Ekiga or something else might.
