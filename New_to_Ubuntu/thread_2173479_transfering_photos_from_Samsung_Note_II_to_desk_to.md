### Post by alphacrucis2 on 2013-09-09
With wifi connection you can use the Airdroid app which allows any web browser to connect to the Android device and transfer files etc. Airdroid is OS independant so you can use it from any OS.

1. Install Airdroid on the Note II via google play store
2. Run airdroid - it will tell you a URL to type into your web browser on the PC
3. Airdroid will prompt you when it sees the connection attempt - accept and the browser will display all sorts of goodies for accessing Note II or any android device.


You can also just plug in a USB cable and the Note II will mount under nautilus as a storage device. I think raring has the appropriate drivers for this.
