### Post by 3rdalbum on 2011-09-28
ClamAV is a command-line utility. As such, it doesn't install to the menu.

What you want is KlamAV, which is a GUI frontend to Clam. KlamAV will appear in your menu because it's a GUI.

However, you'll probably find that you don't need any antivirus at all. Linux doesn't get viruses. You only need an anti-virus if you want to scan USB drives and stuff for viruses before passing them onto Windows computers.
