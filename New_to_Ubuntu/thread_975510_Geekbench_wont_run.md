### Post by Vadi on 2008-11-11
Hi,

Try the following. Download [http://www.primatelabs.ca/download/Geekbench2-Linux.tar.gz](http://www.primatelabs.ca/download/Geekbench2-Linux.tar.gz) to your desktop. Right-click on it, and select *extract here*. Go inside the folder it make. Then got Applications - Accessories - Terminal, and drag the **geekbench** file from the folder into the terminal, and press enter.

It should run after.
