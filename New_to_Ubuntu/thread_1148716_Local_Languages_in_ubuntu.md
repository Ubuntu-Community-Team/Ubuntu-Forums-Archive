### Post by Didius Falco on 2009-05-04
> **ravi_buz said:**
> I have ubuntu 8.04 and installed in default language(english
) now how to change it to Tamil (it will be fun reading it in local language) and if its hard to understand how to change back safely

Just go to Synaptic and install the **language-pack-ta**
package.

To go back to English, you can either pick it at boot up at the login screen or go to **System/Administration/Language Support **and change it back. Doing it the second way may require you to log out and back in.

Cheers,

Didius

On Edit: According to Synaptic -

> Please note that you should install language-support-ta
to get full support for this language.
