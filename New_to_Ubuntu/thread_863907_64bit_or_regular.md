### Post by Cone on 2008-07-19
If you go with 32-bit you're less likely to have problems down the road as 32-bit is better supported in general.

If you have 4+ GB of RAM, not all of it will be used if you run 32-bit, whereas it will be if you use 64-bit.

In the end, it's a personal choice -- your processor can do both.  The benefits from 64-bit aren't that great and if you're not incredibly experienced with Linux I would recommend at least the first install being 32-bit.  (Wait for 2nd opinions before making a decision, of course.)
