### Post by jlaki on 2010-05-20
Ubuntu comes with the applications for everyday use installed by default, so you don't have to bother around anything.

If a webpage contains Java or Flash, Firefox should, like in Windows, ask you if you want to install required plugins, where it will simply guide you through the wizard.

On the other hand, there are video codecs that are a hell in Windows. Yet, Ubuntu asks you to install the missing codecs when you try to play a video or audio file that's not supported by default.

So, everything is rather simple. Just use it and if you run into problems, come back here.

If you need anything particular that you already don't have preinstalled, come back here.
