### Post by Zill on 2013-03-04
Old Jimma:  Your screenshot shows, I believe, an interactive [GRAMPS]("http://gramps-project.org/") report of your family tree, rather than the actual source data.  This is a [Narrated Web Site]("http://www.gramps-project.org/wiki/index.php?title=Gramps_3.3_Wiki_Manual_-_Reports_-_part_7") report and can be viewed via a standard web browser.  Just click on your "index.html" file to see the tree and then everything is a hyperlink so you can click around to see details of the selected person or place etc.

Unfortunately, this kind of GRAMPS report is of little use if you wish to directly import the data into an actual GRAMPS database.  You really need to have the actual source data in the original GRAMPS format.  This will have been in a hidden folder called ".gramps", probably in your home directory.  However, even if you can locate this data, formats have changed with different versions of GRAMPS and so, as yours goes back ten years, it may be very difficult to import this successfully into the latest version of GRAMPS.

You may need to resort to viewing your data in your Web report and then manually enter the data into a new GRAMPS database. :-(

The section entitled [Importing Data]("http://www.gramps-project.org/wiki/index.php?title=Gramps_3.3_Wiki_Manual_-_Manage_Family_Trees#Importing_data") in the GRAMPS wiki may be of assistance.  Good luck.
