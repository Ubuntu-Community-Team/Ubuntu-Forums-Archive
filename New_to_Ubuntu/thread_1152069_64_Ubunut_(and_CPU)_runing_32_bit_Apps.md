### Post by Hospadar on 2009-05-07
absolutely, there are some extra libraries needed to run 32 bit binaries, and for the few things that don't have 64 bit binaries (flash player namely) the 32 bid dependancies will get installed automatically.

Any 32 bit source code should compile just fine with nothing extra on a 64 bit machine.
