### Post by deadflowr on 2013-09-11
Ubuntu Gnome is a purely gnome based distro, where as Ubuntu, though based on gnome, does utilize non-gnome bits and pieces.
Ubuntu uses the Unity desktop interface, where as Ubuntu Gnome uses the gnome-shell and/or the fallback/classic interfaces.
