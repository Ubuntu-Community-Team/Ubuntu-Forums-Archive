### Post by Plueonic on 2011-05-15
For php, apache and mysql, search [http://packages.ubuntu.com/](http://packages.ubuntu.com/) (don't forget about the dependencies), or install them from the repositories and check in /var/cache/apt/archives/

I don't know whether there is a debian installer for joomla but you could just download the zip file from [here]("http://www.joomla.org/download.html") and extract it to your server root or a subdirectory.
