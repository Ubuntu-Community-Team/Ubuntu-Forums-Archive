From browsing the forum posts you will see that the easiest thing to do with Ubuntu is to break it.

### Post by Paqman on 2011-07-29
Hmm, well if you ask me, the first things you need to learn about are package management and file permissions. Synaptic is a good tool that exposes some of the core concepts of how packages work. After that you can start having a look at some of the important config files like /etc/fstab and learn how to mount local storage and network drives into your filesystem. That opens up lots of optins for felxibility in your system.

Learn how to use some of the more useful command line tools like rsync and wget, and write a couple of basic scripts (then drop them into anacron).

Try out some other distros and have a poke around under the hood to see how they differ from Ubuntu.

If you're feeling confident/adventurous, install Onearic on a spare partition and get involved in testing.
