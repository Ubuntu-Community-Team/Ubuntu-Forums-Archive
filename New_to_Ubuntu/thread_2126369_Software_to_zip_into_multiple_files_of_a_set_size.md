### Post by schragge on 2013-03-16
zip lets you create split archives. Search for *Split archives* in the [manual page]("http://manpages.ubuntu.com/manpages/precise/en/man1/zip.1.html"). And there's [rar]("http://manpages.ubuntu.com/manpages/precise/en/man1/rar.1.html") for Linux, too (albeit only command-line version). Besides, you can create a big monolitic archive, then split it into multiple parts with [split](http://manpages.ubuntu.com/manpages/precise/en/man1/split.1.html).
