### Post by deathadder on 2011-03-28
Coming from your other thread... [http://ubuntuforums.org/showthread.php?t=1715581](http://ubuntuforums.org/showthread.php?t=1715581)

Yes would would have to build it, the process seems to be nicely explained on [the projects site]("http://www.mono-project.com/Moonlight#Working_from_tarball_releases").

Maybe you could look at the nightly builds from here: [http://www.go-mono.com/moonlight/download.aspx](http://www.go-mono.com/moonlight/download.aspx)

Because it's beta software, you won't get it from a distro's normal repos' you'd need to be using the testing branch [Natty Narwhal]("http://www.ubuntu.com/testing/natty/alpha1") for example, not that it'll be included it's just an example. :)

There seems to be quite a bit of information on what you need to do to get it installed. For example: [http://www.novell.com/news/press/2011/2/moonlight-4-preview-now-available.html](http://www.novell.com/news/press/2011/2/moonlight-4-preview-now-available.html)

Bear in mind that this press release is just over a month old now for beta software, I would be surprised if any distro's had it in their testing repo's already. 

If you want the latest and greatest you'll need to compile from source...
