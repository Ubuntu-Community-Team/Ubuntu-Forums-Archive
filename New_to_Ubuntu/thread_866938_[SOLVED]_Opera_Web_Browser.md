Just FYI, a [user agent]("http://en.wikipedia.org/wiki/User_agent") is used to identify a client application. Through modification, it's possible to make one web browser appear as another.
 
I've never used Opera myself, but apparently Opera has a [built-in feature]("http://www.zytrax.com/tech/web/change-ua.html") to do just that.  Here's a useful [list of user agent strings](http://www.useragentstring.com/pages/useragentstring.php).
