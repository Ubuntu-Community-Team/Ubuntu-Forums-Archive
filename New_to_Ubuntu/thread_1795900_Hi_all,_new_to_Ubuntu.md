Welcome to the forums, and welcome to Ubuntu!

### Post by Swagman on 2011-07-03
It might help to understand what the version numbers represent.

If you are not already aware, a new version of Ubuntu is released every 6 months

eg: Where I boarded the Ubuntu train it was version ---


7:04 (pet name = Feisty Fawn) which = 2007 / April (4th month)
7:10 (pet name = Gutsy gibbon)which = 2007 / October (10 month)

A separation of 6 months

The "pet names" also go up in alphabetical order

So

v10:10 = 2010 / October

Hope that helps

btw .. I'd stick with 10:10 NBR if that were my netbook (just for now anyway)
