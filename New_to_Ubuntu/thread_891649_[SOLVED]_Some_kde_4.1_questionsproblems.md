### Post by benerivo on 2008-08-16
Your start button icon, may be related to the icon set that is chosen in system settings. Make sure you are using oxygen and log in/out. Or maybe removing the button and adding it back may work.

The next version of amarok will be qt4, and is very nearly ready. You can try a packaged development version [here]("http://amarok.kde.org/en/node/482"). Juk-kde4 is okay, but very basic. If you just want a player rather than a full on music manager, then use dragonplayer, or the latest packaged version of vlc [here]("http://nightlies.videolan.org/#debian").

What is wrong with the Autostart section of system settings? I haven't used it, but would be interested to know. I have always just used /home/john/.kde4/Autostart.

I don't think there is a transparency option in the panel settings, but you couyld try changing the plasma theme by right clicking on the desktop > Desktop Settings > New Theme. Some of the themes (try glass) have a transparent panel.
