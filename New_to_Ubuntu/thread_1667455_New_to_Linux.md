### Post by cgroza on 2011-01-14
You can do it the GUI way via Ubuntu Software Center or via a simple terminal command:

```
sudo apt-get install ubuntu-restricted-extras
```You will be prompted for your password. While you type it, it won't show any feedback on the screen, just type it and hit enter.
