### Post by Holger_Gehrke on 2017-12-04
> **itchyfatpants said:**
> I've downloaded the relevant .deb package from the site and installed it via Software install however when I run it I am not able to open .java or .class files from within the app.

Not really surprising, BlueJ - like a lot of IDEs - organizes all files belonging to one program into a project. You first have to create a project. Then you can either write new classes for this project or import Java files you've already written into it. The dialog of which you made a screenshot clearly says "Open Project" in the title bar, it wants to load a bluej project file (extension .bluej).
