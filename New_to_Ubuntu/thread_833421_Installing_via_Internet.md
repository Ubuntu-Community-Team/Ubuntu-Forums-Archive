### Post by 505 on 2008-06-18
You can't do a net based install with Ubuntu (I believe). However, you can do that with Debian. Ubuntu is based on Debian, and for a server Debian would usually be a better choice. For the net install you still need to burn a CD (about 40MB), the rest is downloaded over the internet as needed.

### Post by cariboo on 2008-06-18
You can use the minimal install cd that is available here:

[https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)

Try burning the image at the lowest speed possible and it should work out ok. This cd just installs the basics and downloads and installs what you need from the internet.
