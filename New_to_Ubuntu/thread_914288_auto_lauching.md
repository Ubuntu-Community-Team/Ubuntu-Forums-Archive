### Post by Sealbhach on 2008-09-08
Click in the menu - System/Preferences/Sessions. You can add startup programs from there.

You would need to know the command e.g. if you were to add Firefox as a startup program, it would be 

```
firefox
```

Just like if you open a terminal and type 

```
firefox
```

it starts up Firefox.
