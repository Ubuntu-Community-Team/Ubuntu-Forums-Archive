### Post by kansasnoob on 2008-12-03
OK, regardless of your answers to my first two questions, just do this:

Go to Places > Home Folder, then click on View > Show hidden files.

You'll see a file named .mozilla. Right click that file and select copy. Then click on Desktop and select paste. That way the original .mozilla is there to restore or extract from!

Now go back to Home > > .mozilla and right click again. This time send it to Trash!

Now in Firefox (you have it open because you're following my instructions) go to File > Quit!

Then restart Firefox! That should give you a whole new Mozilla! Then we'll have to restore what you want restored1
