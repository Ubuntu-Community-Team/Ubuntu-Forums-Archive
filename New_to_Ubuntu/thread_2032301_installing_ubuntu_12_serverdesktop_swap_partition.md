### Post by Bufeu on 2012-07-23
1. If you not choose "Something else", Ubuntu will automatically create a swap partition, I think 100 MB is default (I may be wrong).
2. I prefer the desktop version, but I doesn't matter at all which one you install.
3. No, Ubuntu Server does not come with a GUI as standard, but [you can install one]("https://help.ubuntu.com/community/ServerGUI").
