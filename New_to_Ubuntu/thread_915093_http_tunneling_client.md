### Post by elmoosecapitan on 2008-09-09
If you are talking about getting firefox to access a private, internal network, e.g., government, private business, that is not accessible to the general public, one LEGAL way of accessing such sites is using ssh and SOCKS.

If permissions are granted, is to ssh into a node through a well-defined port number. If you are running firefox 2.0+, then you must change your SOCKS settings, and once manual proxy settings are set you should have access to the private network.

I know how to do this in principle but I am still working out the details about the SSH options that are needed. Good luck.
