### Post by cariboo on 2013-09-06
Are you sure your devices aren't being detected? Open a terminal and type the following command shortly after you have plugged in a device:

```
dmseg | tail -n10
```

The output should look something similar to this:

```
dmesg | tail -n10
[33353.065800] sd 4:0:0:0: [sdc] No Caching mode page present
[33353.065807] sd 4:0:0:0: [sdc] Assuming drive cache: write through
[33353.068924] sd 4:0:0:0: [sdc] No Caching mode page present
[33353.068930] sd 4:0:0:0: [sdc] Assuming drive cache: write through
[33353.073688]  **sdc: sdc1**
[33353.076799] sd 4:0:0:0: [sdc] No Caching mode page present
[33353.076805] sd 4:0:0:0: [sdc] Assuming drive cache: write through
[33353.076808] sd 4:0:0:0: [sdc] Attached SCSI removable disk

```

As you can see my usb thumb drive has been detected and given /dev/sdc1 as the drive name.
