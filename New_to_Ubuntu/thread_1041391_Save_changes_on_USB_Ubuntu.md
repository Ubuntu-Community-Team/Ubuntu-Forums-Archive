### Post by snowpine on 2009-01-16
Unetbootin does not actually "install" Ubuntu to your USB drive. It creates a "Live USB" that is equivalent to running from a Live CD. 

Saving your changes from session to session is called "persistence" in Linux-talk. You can create a persistent USB in several different ways. The method Nhasian mentions is probably the easiest; other tutorials are on the pendrivelinux.com site.

Best performance (and full persistence) is achieved by installing Ubuntu to your computer's hard drive. If you want to keep your existing operating system, you can set up a "dual boot."

Good luck!
