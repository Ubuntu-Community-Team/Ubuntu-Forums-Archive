### Post by oldsoundguy on 2012-08-19
You will soon learn the reason why there is no need for an anti virus program FOR Linux.  That is because, as a newbie, you should only install programs from the repositories and the install procedure for ANY item requires more than one click.  PLUS, there are no virus programs that attack Linux itself running in the wild. There are virus programs that can and do attach to SERVERS .. but they attack WINDOWS computers that utilize the servers.  There is an AV program (several actually) that should be installed if you are going to use your machine as a server to protect those Windows machines.

Clean up is also not required often as the file system for Linux does not fragment and not much is left behind when you update (there is some).

Personally, I use a third party program for clean up (UbuntuTweak), but there are some command line programs available already within the system.  I'm sure somebody else will chime in and give you the code to cut and paste into Terminal . they work!!
