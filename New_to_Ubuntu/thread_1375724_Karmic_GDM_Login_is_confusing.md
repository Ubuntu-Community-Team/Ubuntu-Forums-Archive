### Post by llawwehttam on 2010-01-08
In karmic the GDM is very different and this gives it a much higher boot speed. If you want to have themes you must downgrade your GDM which I would not recommend.  The likelihood is that a new login screen manager will come with 10.04 LTS so if you can ...be patient!

### Post by Kazade on 2010-01-08
Karmic introduced GDM 2 which is not yet feature complete (it was introduced a little prematurely because some of the boot process upgrade work required it I think) - because of that you can't currently change the theme of the login screen via the GUI.

You can however hack it, there are some instructions here: [http://www.ubuntumini.com/2009/09/hack-karmics-gdm-login-screen.html](http://www.ubuntumini.com/2009/09/hack-karmics-gdm-login-screen.html)

Hopefully this will be fixed properly in Lucid (10.04).
