### Post by jbrown96 on 2009-02-11
It's installed by default. First, you need to enable your graphics drivers (unless it's Intel Graphics) in System===>Administration===>Hardware Drivers. Reboot. Compiz should work by default. There are two different levels available in System===>Preferences===>Appearance===>Visual Effects. If you really want to customize the effects, then install compizconfig-settings-manager in Synatpic (system===>Administration===>Synaptic)
