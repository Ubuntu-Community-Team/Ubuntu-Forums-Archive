See this link for information on [Upgrading Ubuntu](http://www.ubuntu.com/getubuntu/upgrading)

Basically You can upgrade over your Network connection from 6.06 LTS to 8.04 LTS or use the 8.04 LTS Alternate Install disk.**You cannot use any Live CD to Upgrade.**

Also upgrading from 6.06 requires you go through all the releases and their updates,so it would be better to either get the 8.04 LTS Alternate Install disk and use that to upgrade, or do a clean install of 7.10 (Gutsy).


Good Luck
