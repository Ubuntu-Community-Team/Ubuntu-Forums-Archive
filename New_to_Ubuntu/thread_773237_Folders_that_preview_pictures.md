### Post by PhilOSparta on 2008-04-28
Go into your Nautilus file manager and select Edit -> Preferences -> Preview -> Show Thumbnails -> Always or Local Files.

### Post by PhilOSparta on 2008-04-28
Good question.  I couldn't find it in the Applications menu structure, but since I have an old Ubuntu which has been upgraded any number of times, I can tell you how I got an icon on my task bar.

Right click on the task bar and select Add To Panel.  Then select Custom Application Launcher.  It will provide a box for you to type the command.  Enter   nautilus  and click the icon button.  I should go out and find the sea shell that is the Nautilus icon.  Click the Ok button and close the launcher.  You should now have an icon on your task bar / panel for Nautilus.
