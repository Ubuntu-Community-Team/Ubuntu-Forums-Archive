### Post by Crandom on 2008-10-24
Video ActiveX uses ActiveX, a proprietary windows format. There is no way you can use it on any *nix.

And, more importantly, I seem to remember a while ago getting a message to install video activex, and having a dialog pop up with "Install" and "Cancel". whatever you clicked it installed a **virus**. I would generally stay away from any kind of ActiveX and IE in particular. 

For the tl;dr people : it is a virus and doesn't work on linux.
