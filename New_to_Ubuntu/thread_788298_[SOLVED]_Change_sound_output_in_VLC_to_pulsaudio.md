### Post by ajgreeny on 2008-05-09
Open vlc Settings>>Preferences>>Audio>>Output Modules and make sure you have Advanced options in the bottom right corner enabled.  You should then have a drop down box to set the output to pulseaudio.  If that doesn't work, I'm afraid I can't think of anything else to suggest.
