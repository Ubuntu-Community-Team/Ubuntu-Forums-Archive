### Post by snowpine on 2008-10-15
Hi 22-250, welcome to the forums! I think your biggest problem is going to be that 18-month support for 7.04 ended earlier today, so unfortunately you won't be able to install anything from the repositories since they are now shut down.

I would recommend installing 8.04 Hardy Heron (the current long-term support version). Hardy includes Firefox 3, and you can easily install Flash from the command line with 'sudo apt-get install flashplugin-nonfree'. If for some reason using a recent version of Ubuntu is not an option, your best bet is to download the latest Firefox and Flash from the Mozilla and Adobe websites and follow their installation instructions to install using the command line.

Good luck!
