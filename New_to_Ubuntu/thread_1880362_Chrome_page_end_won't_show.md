### Post by LinuxFan999 on 2011-11-13
I don't think there is any way to manipulate it. If you have a Google account, you can report this bug to google code at: [http://code.google.com/p/chromium/issues/list](http://code.google.com/p/chromium/issues/list)
 
When you load the site, select the button near the top left that says "New Issue". There, you can report this bug. Make sure to provide as much detail as possible, and provide a screenshot if you can.
 
After filing the report, open the newly created issue, copy the link from the adress bar, and paste it here.
