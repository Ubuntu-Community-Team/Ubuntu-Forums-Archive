### Post by Chesamo on 2010-04-02
That's a very, very broad question. Widgets are essentially mini-programs that run underneath the plasmid system, meaning you need to know a programming language to create them. Alternately, web widgets (ie. Opera or Windows Sidebar widgets) have partial compatibility.

Your question basically asks "how do I make a program", to which the answer is "practice".
