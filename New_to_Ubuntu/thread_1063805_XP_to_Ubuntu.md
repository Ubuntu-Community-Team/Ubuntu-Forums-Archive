### Post by Therion on 2009-02-08
My first question to you would be "Why do you want to switch?"  My next question would be, how willing are you to roll up your sleeves and work to 1) get a new OS working on your machine and 2) learn a whole new way to do things?  

I'd strongly suggest you get your optical drive fixed so you can at the very least try out a LiveCD to see how well you like Ubuntu and how well it deals with your hardware out of the box.  

As to software, your Windows apps will not run under Ubuntu natively.  You can try using an emulator (Wine), or you can dual boot or use Virtual Box.  But if you're simply looking to run Windows applications without Windows you're setting yourself up for a heartache.  For the most part there are open source alternatives for just about every Windows application out there; but this goes back to question 2... How willing are you to learn a whole new way of doing things and so forth.

### Post by ugm6hr on 2009-02-08
Good reasons to switch.

Windows programs are designed for Windows.  Linux programs are designed for Linux. Mac programs are designed for Macs.  There are ways for these to overlap, but I would not recommend expecting that to be the case.  There are workarounds (e.g. Virtualisation of Windows in Ubuntu, WINE), but none are particularly elegant.

You can buy songs for your Ipod from non-Apple stores (Amazon sells non-DRM songs, I believe).

Wifi in Ubuntu generally works well if you have a PCI wireless card (i.e. internal), but less well for USB.  You can check with a LiveCD / USB.

Dual-booting: [http://www.psychocats.net/ubuntu/dualboot](http://www.psychocats.net/ubuntu/dualboot)
