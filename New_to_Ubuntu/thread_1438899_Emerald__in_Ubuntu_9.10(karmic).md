### Post by tad1073 on 2010-03-25
Alt+F2, then type:
```
emerad --replace
```

or
System>Preferences>Compiz Settings Manager>Window Decoration, in the box next to Command put in emerald --replace.

or
```
sudo apt-get install fusion-icon
```
