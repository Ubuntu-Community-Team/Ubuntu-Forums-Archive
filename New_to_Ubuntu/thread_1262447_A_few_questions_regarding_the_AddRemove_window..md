### Post by zephyrcat on 2009-09-09
Add/Remove will stay up to date as far as maintenence releases. It will not, however, add versions that introduce new features. THe only issue with VirtualBox (and this is an unusual problem) is that the open-source edition has a few less features. If you want USB support, you should probably download the version from the website.

With most applications, you can just install the version in Add/Remove programs.
