### Post by H3R3T1K on 2012-03-06
I installed Lubuntu in English but I write mails in both English and German. I checked the Sylpheed FAQ and it says in order to enable spell check you have to add "xterm -e ispell %s" as an external editor under preferences. I also read in a mail from a Debian mailing list that you can use aspell as well: "xterm -e aspell check %s". I installed xterm and aspell-de, selected "de" as the default input language and added "xterm -e aspell check %s" as external editor.
If I type German words, they are not underlined anymore, but if I click on the external editor button in the compose window to run the aspell spell check, I'm only offered English words.
Any ideas on how to fix this?
