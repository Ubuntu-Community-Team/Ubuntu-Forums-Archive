### Post by Buuntu on 2009-10-30
I had the same problem you had, and I solved it by running a dist-upgrade on the partition by chrooting in from the live CD.

1) Boot off the live CD.  Make sure you have internet, if not the next steps will not work.  It's better to connect directly to your router just to be safe.
2) Run the following command in the terminal:
(Change /dev/sda1 to whatever your partition with the failed upgrade is)
```
sudo mkdir /mnt
sudo mount /dev/sda1 /mnt
sudo chroot /mnt
apt-get update
apt-get dist-upgrade
```
And then tell me if you have any errors from that code.

Hope this works for you

### Post by Buuntu on 2009-10-30
I'm not sure if this will fix the problem, but just try to upgrade again now that you're in Karmic, and try all of the following commands.
```
sudo apt-get update
sudo apt-get upgrade
sudo apt-get install -f
sudo dpkg --configure -a
```Your problem might be because of broken packages and such, so it's worth a try.

In case that doesn't work, you could try running the following:
```
gksudo mv /etc/X11/xorg.conf /etc/X11/xorg.conf-old
```Karmic Koala by default doesn't start with a xorg.conf file, and for me doing this fixed all of my visual problems I was having.
