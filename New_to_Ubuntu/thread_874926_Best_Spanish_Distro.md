### Post by sdennie on 2008-07-30
Almost all the common things are localized to Spanish just fine in Ubuntu.  Some of the apps that you get out of synaptic may not be fully localized but, the core applications are very good in Spanish.  

If you just choose Spanish as the install language you should be fine except that not all of the language packs are included on the install CD so, if the installer isn't connected to the internet, you'll have to later go to Sistema->Administración->Soporte de Idiomas and make sure everything is downloaded.  (At least, this is how it's worked in previous versions when installing in Spanish).
