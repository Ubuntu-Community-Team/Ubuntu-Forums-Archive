### Post by OutOfReach on 2008-08-07
Oh a virtual desktop?
Goto Applications>WINE>Configure WINE.
or run ```
winecfg
``` in the terminal.
Then goto the Graphics tab.
And select the option 'Emulate a virtual desktop.' and press OK or Apply
