### Post by Impavidus on 2020-07-14
Regular Ubuntu may install on your computer, but you won't have a lot of memory left for web browsing, photo editing or whatever you want to do with it.

Xubuntu, Lubuntu, Kubuntu etc. are flavours of Ubuntu. They use the same software sources, so you can install exactly the same software from the official repositories in the same way. The difference is the set of applications that get installed by default. In the Linux world, the desktop environment is just another application and these flavours use different desktop environments. That's why they have very different looks and memory requirements, but under the skin they're basically the same.

Although you can install exactly the same software on all flavours, some combinations may work better than others as some combinations of software are better at sharing resources.

[https://ubuntu.com/download/flavours](https://ubuntu.com/download/flavours)

### Post by SeijiSensei on 2020-07-15
> **misternemo said:**
> does this mean in my case i should use these ? if yes why are there two ??? and is it really neccecary with my spec laptop
also what is the difrence what would i be missing out on if i use these instead of ubuntu or do they work the same look the same but simple or something?
Windows provides a single desktop with its OS.  Linux is structured very differently.  The operating system and the "desktop environment" are entirely separate.  The various flavors of Ubuntu use the same operating system but different desktops.  Some desktop systems are more graphically demanding and may need more memory to run well. Lubuntu and Xubuntu make fewer such demands.

No matter which flavor you choose you can still use the entire array of programs available for Linux like Firefox, Thunderbird, LibreOffice, and the like.  

I recommend this strategy.  Use [Rufus]("https://rufus.ie/") on Windows to create an installation USB with, say, [Lubuntu]("http://cdimage.ubuntu.com/lubuntu/focal/daily-live/current/") on it. Boot from it on the laptop and choose "Try Ubuntu" when prompted.  You can then run a Lubuntu session without making any changes to your hard drive.  Do the same with [Xubuntu]("http://cdimage.ubuntu.com/xubuntu/focal/daily-live/current/").  Pick the one you like best.
