### Post by DaleNewton on 2008-10-05
Hi,

Its  awhile back since you posted, but I have found a solution for some programs.

YOu can uninstall the program that is giving you the trouble using synaptic package manager, and then reinstall the program.

When you uninstall it, select 'completely remove program' including its configuration files.  THen reinstall it.

THis worked with GIMP image editor program for me, and IM going to try it on others that crash too.
