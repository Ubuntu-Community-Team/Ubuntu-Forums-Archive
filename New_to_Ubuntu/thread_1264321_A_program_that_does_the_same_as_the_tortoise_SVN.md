### Post by KIAaze on 2009-09-12
There are CVS and SVN plugins for Konqueror:
For CVS: [http://cervisia.kde.org/](http://cervisia.kde.org/)
For SVN: [http://kdesvn.alwins-world.de/](http://kdesvn.alwins-world.de/)

```
sudo apt-get install konqueror kdesvn
```
Of course, this will install several KDE libraries if you are not using KDE.

TortoiseSVN suggestions: [http://tortoisesvn.net/node/58](http://tortoisesvn.net/node/58)

Other cross-platform non-browser-integrated GUI solutions:
[http://www.syntevo.com/smartcvs/index.html](http://www.syntevo.com/smartcvs/index.html)
[http://www.syntevo.com/smartsvn/index.html](http://www.syntevo.com/smartsvn/index.html)
