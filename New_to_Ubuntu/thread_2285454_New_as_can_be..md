### Post by grahammechanical on 2015-07-06
Ubuntu Help

Click the power/cog icon and select Ubuntu Help. At the top of this section are what we call "stickies." They have links to learning resources. All operating systems do the same basic things. They look different and they do things things in different ways. But in reality they are all doing the same kind of things. 

I do not understand what you mean by run the system. Ubuntu will automatically check for updates and will inform you if they are available and the Software Updater utility will download the updates and install them. What else is there to do except use the OS to run applications? Ubuntu has a comprehensive set of applications installed by default. The Ubuntu Software Centre has a large selection of applications and the Software Centre will download, install and, if we want, also remove applications.

Ubuntu will allow us to make only limited modifications. Some people see that as a failing. I see it as being sensible. There are utilities that will give us greater power but they are not installed by default. The power to change things is also the power to break things. And that is why the help sections of this forum are far busier than the chat sections.

Regards.
