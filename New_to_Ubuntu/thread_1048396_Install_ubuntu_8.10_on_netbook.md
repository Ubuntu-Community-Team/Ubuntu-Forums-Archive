Take a deep breath Joseph.  Take two.

Okay, so you've got your netbook, and you've got an external drive of some kind, correct?  Is this an optical drive (CD/DVD) or is it a USB powered hard disk drive?  Your post isn't clear on that point.

You're going to need to create a bootable CD or DVD if you have an external optical drive, or if you have a USB thumb drive, we can use that and have you up and running pretty quick.  You're also going to need a copy of the .iso file for Ubuntu 8.10 as well.

The fastest, easiest way, IMO, is to use Unetbootin and a USB thumb drive.  If that appeals to you look at these instructions:

[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)

### Post by Therion on 2009-01-23
Okay, that's what I figured.

You need to change the settings on the Acer so that the CD/DVD drive is the first bootable device.  I know that on my Asus eee I can use the ESC button to get to menu that allows me to choose what drive I boot off of.  On your Acer, I think it's the F12 button that does that. 

So... Reboot the Acer with the 8.10 disk in the drive.
As soon as it starts to boot up, start slowly, repeatedly tapping the F12 key to see if you can change the boot drive to the CD/DVD drive.  If you can, obviously, select it and boot from the CD.  Install Ubuntu as usual from there.
