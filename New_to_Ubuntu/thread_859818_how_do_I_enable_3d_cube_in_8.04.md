Hi,

Go to Applications/Add-Remove 

Search for Advanced Desktop Effects Settings (make sure All Available applications is selected in the Show drop menu)

Install that which includes the compizconfig settings manager.

It can then be accessed from your System/Preferences Menu.

### Post by OutOfReach on 2008-07-14
Install compizconfig-settings-manager from Synaptic, or from the terminal. Then goto System>Preferences>Advanced Desktop Settings and enable the following plugins:
Desktop Cube
Rotate Cube
Cube Reflection (optional, but makes it look better in my opinion)

If it asks you to disable Desktop Wall, select Disable Desktop Wall.
Then press CTRL+ALT+Left Mouse Button and move the cube. :popcorn:
