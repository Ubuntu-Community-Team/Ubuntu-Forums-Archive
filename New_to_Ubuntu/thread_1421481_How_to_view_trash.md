### Post by L Campbell on 2010-03-04
When I want to Unmount a thumb drive I get a message asking if I want to 'empty the trash'.

Well, maybe I do and maybe I don't but I would like to be able to view the trash, so as to decide.

How does one do this?

TIA.

---

### Post by philinux on 2010-03-04
Before you unmount the drive, browse it and press ctrl h to show hidden files. You should see a .Trash folder on the drive.

---

### Post by LowSky on 2010-03-04
if you don't want it to put files into a trash folder use

Shift+Delete

that command will permanently delete the file without sending it to a trash folder.
