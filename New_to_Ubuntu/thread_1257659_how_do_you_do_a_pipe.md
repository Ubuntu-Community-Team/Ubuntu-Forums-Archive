### Post by tomski on 2009-09-04
the usage would be:

command | command 

or in real world here ia a simple command set:

locate resolv | less

this will tell linux to 'locate' a file with 'resolv' in the title name & output the results (PIPE) to the pager called 'less'

keystroke code:
0x7c
U+007C
