### Post by Impavidus on 2012-12-23
If it's an old machine i can be that it's simply not up to the task. Unity weighs in rather heavy on your graphics card (and memory). Try using unity 2D or install the package xubuntu-desktop, which will will give you the option of choosing a lighter desktop environment at login.

On the other hand, if you've got a recent machine, there might be a problem with the driver of your graphics card. If that's the case, could you post the exact make and model of your graphics card?
