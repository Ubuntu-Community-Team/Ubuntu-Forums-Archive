### Post by LowSky on 2010-06-28
You ight see as the computer is starting up...  Press F2 (or another key) to enter BIOS.

From BIOS you can do things like change the system time or the boot order of you hardware, like if you want to CDROM to boot before the hard drive. You can also change Power settings on some machine, and with that hopfully the battery use.

If not maybe try to find the application for doenload  and try to run it using a n applicaiton called WINE or an open source equivlent if one exists.

If none of thes eoptions will work. you will need to eiteher reinstall windows, or live with lower battery use.
