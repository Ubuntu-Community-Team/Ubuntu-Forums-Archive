### Post by =CrAzYG33K= on 2010-07-02
You can safely and easily install Ubuntu under Windows using WUBI.

Similarly, you can easily remove it as well. You have to go to Start -> Control Panel -> Add & Remove Programs -> Ubuntu and then 'Uninstall'

That's it! :p
No need for messing up your filesystem at all.
