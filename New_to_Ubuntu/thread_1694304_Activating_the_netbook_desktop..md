### Post by ankspo71 on 2011-02-25
Hi,
Yes, the Netbook version of Kubuntu is built in the desktop version. The netbook version has no problems running on large screens. The only difference (that I know of) between the netbook version and the desktop version is a 'search and launch containment' feature and panel design.

There are two ways to go about this:

Option one: 
The netbook version gives you the netbook's special panel (taskbar) and the netbook's "search and launch containment" desktop. Go to System Settings > Workspace Behavior > Workspace > Workspace Type: Choose Netbook instead of Desktop. Click Apply.

Option two:
If you end up not liking the netbook's panel, you are able to use the "search and launch containment" feature on the default Kubuntu desktop, which means you can keep the traditional KDE panel. If you are already using the 'desktop'-desktop (if not do the reverse of option one), right  click on the cashew shaped thing in the upper right part of the screen (unlock the widgets if not done already), choose 'Desktop View Settings' (or folder view settings etc), then when the settings window opens up, select the view category, then where it says "layout:", choose "Search and Launch Containment". Click Apply.

Hope this helps.
