To get help you need to use the title of the post to give a some indication of the problem. You also need to explain clearly what the problem is and tell us the hardware specification of the machine and also the version of Ubuntu or its flavours that you have installed. It is also useful to know if there is another operating system installed on that machine and what it is.

When we install Ubuntu we are asked to tick that there is an Internet connection. If you have an Internet connection (wired/ethernet) during installation then you should still have that same internet connection after installation when you loaded the operating system.

If you need advice on how to set up a wireless connection, then Ubuntu comes with Help documentation that can assist you with that. There is also an online version. Here is a link to it.

[https://help.ubuntu.com/14.04/ubuntu-help/index.html](https://help.ubuntu.com/14.04/ubuntu-help/index.html)

We need to know if the problem is due to you not knowing how to make a wireless connection to the router or if Ubuntu is unable to make a wireless connection because Ubuntu does not have a driver for the wireless adapter in the machine.

The problem could also be caused by the router not connecting to the Internet Service Provider (ISP). All of these things are a possible cause of the problem but you do not give enough information even to make a guess.
