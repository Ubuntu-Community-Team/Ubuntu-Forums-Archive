### Post by AlphaLexman on 2010-12-02
If you are running ubuntu (you're on the ubuntu forums) then are most likely using 'Gnome' the default Desktop Environment for ubuntu. 'KDE' is another Desktop Environment for Linux Distributions. 'Kubuntu' is the ubuntu version which includes 'KDE'. Unless you KNOW you installed Kubuntu, you don't have 'KDE'.

However, Dolphin is the default File Manager for KDE. 'Nautilus' is the default File Manager for 'Gnome'. If you are actually using Gnome and want Dolphin to be the default, it will require more system resources.
