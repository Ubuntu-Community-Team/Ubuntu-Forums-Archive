### Post by Slowspeed on 2008-10-03
Xubuntu is the distribution designed for older PC's. XFCE is the Desktop windows manager that when installed with the base configuration does not require a large amount of memory resources.

Gnome is the Ubuntu distribution default/standard windows manager. 

XFCE is "themed to Ubuntu" but the functionality is standard.

This is my understanding
