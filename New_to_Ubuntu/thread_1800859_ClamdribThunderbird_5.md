Since I got no replies to this inquiry, I did some research on my own.  I found a Firefox extension with instructions on how to modify it to work with the latest Firefox.  Same goes for Thunderbird.

Download the clamdrib extension.  Extract to desktop and open with gedit.  Look for **<em:maxVersion>version</em:maxVersion>** and where it says **>version<** modify it to whatever version of Thunderbird you're using.  Then it will work when installed.
