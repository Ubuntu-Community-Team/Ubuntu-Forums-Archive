Does anyone know how to add Cairo Dock to the startup list without it loading the GLX-dock (OpenGL) version?

Cosmic

### Post by Frogs Hair on 2011-01-11
After logging in , quit GLX DOCK Open GL and remove from the System > Preferences >  Startup Applications list .

Start the non Open Gl Dock and set to start at login from the dock settings . You can also use the remember currently running applications from the options tab in Startup Applications .
