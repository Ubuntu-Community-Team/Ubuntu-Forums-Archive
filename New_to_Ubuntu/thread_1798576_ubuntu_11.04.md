### Post by grahammechanical on 2011-07-06
The people here will need more information. Did you install Ubuntu inside Windows using Wubi? Or did you use the Ubuntu installer to create additional partitions that Ubuntu was installed in? In other words are you dual booting? It will make a difference to how you can be helped.

All drivers that come with any Windows operating system or Windows program will work with Windows but they will not work with Ubuntu. The drivers are not coded to work with any Linux operating system. What do you think you need these drivers for? On my computer Ubuntu works without any Windows drivers being needed. Everything I need to work the computer hardware is coded into the operating system.

What files are you trying to open? What program are you using? Open up the file manager. Click on Places. Click on computer. Look for an icon of a hard disk that represents the Windows partition that has those files. Click on the icon. Now, you have mounted that partition. An icon will appear on the desktop. Now, try opening those files.

Regards.

P.S. If you want you can copy those files into Ubuntu but remember, if you make changes to this files Windows will not be able to open the files on the Ubuntu partition. It may not even see them.
