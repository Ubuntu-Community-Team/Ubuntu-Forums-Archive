### Post by Bucky Ball on 2014-02-18
3/ The graphics card is the only thing that might be problematic in that list. You don't mention anything about wireless, but that might also need some tweaking. Do a search for 'ASUS 680GTX ubuntu' to find out about your graphics.

4/ Some Win software can be used with Wine, some, and it can run very well, not so good or not usable. If you want to use mostly Win software, dual-boot, run Ubuntu as a virtual machine inside Windows (or vice or versa) or stick with Win.

The answers to most of your questions probably lie in downloading a bunch of ISOs of your choice, burning them to disk or USB, booting into a Live session (Try Ubuntu) prior to installing and seeing what works, what doesn't, what you like and what you might like your eventual install to look and feel like. 

Don't forget that, depending on the flavour you use, Ubuntu can be very configurable so you are not stuck with what comes 'out of the box' with any install. 

I don't game at all so leaving that area to more experienced heads.

Good luck.
