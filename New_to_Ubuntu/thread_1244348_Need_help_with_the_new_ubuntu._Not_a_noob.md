### Post by Bucky Ball on 2009-08-19
Things have changed considerably. Welcome back. I don't think you are going to have too many problems with your graphics. If you want stability and long term support (production machine) I would go for 8.04 LTS still. 9.04 came out this April and most finding it stable, some not, but still tweaking (I installed it recently for a friend and it was seamless for the three days I had the machine - probably easiest setup ever - had Ubuntu and all Edubuntu add-on packs installed in under an hour, hardware up). 

I advise that once you have installed, whatever flavour, make sure you have an ethernet cable plugged in, boot up and wait for your updates. You will probably be advised of any restricted drivers that are available for your hardware. As previously noted, it's about that easy. :)

### Post by Bucky Ball on 2009-08-19
Nvidia should not be a problem. The wireless might be. I would install and get the machine up then see about the wireless (as long as you can still get online on another computer that is to research any problems and post here).

I would post again specifically about the USB wireless card and how to get it up prior to installing. Try and get as much detail as you can about it. 

9.04 continues to stabilise and as I say, works fine for some, not for others. For a production machine or something you need to be reliable stable, I would stick to 8.04. What you could do is try 9.04 from the Live CD (option on the install disk) and if that works and all is looking good (you might even be able to experiment with getting your USB wireless up that way before a complete install) then I'd say go for it (I stress again that I, personally, do not use it on machines I need to be stable).
