### Post by majiciannz on 2009-09-20
> **kpholmes said:**
> im using 9.04 currently and completely switched over a couple weeks ago, im liking it alot, but is there a download file, like a place where everything downloaded from firefox goes? i checked the home folder but couldnt find anything.
Open Firefox and click on Tools the Download.
That will open the download folder and show you all your downloads.

If you open Edit>Preferences>Main
Under downloads you can select where Firefox downloads to.
