### Post by Joreus on 2011-07-29
*.exe is an executable file that runs on Windows OS. If you would like to run Windows applications on Linux, you'll need to install a program called Wine. 

Before you do that though, do some searching on the Wine website about whether or not your program is even capable of working through Wine.

Wine website: [http://www.winehq.org/](http://www.winehq.org/)
You can download Wine through the Ubuntu Software Center. I believe it's called KpackageKit on Kubuntu.

That's about as much as I can tell you from my limited knowledge.
