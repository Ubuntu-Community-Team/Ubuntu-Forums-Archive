### Post by Bucky Ball on 2014-01-24
Welcome. Wubi is not the way to go. No longer being supported.

If you just want to try Ubuntu, download the ISO, burn to disk (or USB), boot from the disk (or USB) and 'Try Ubuntu'.

[http://www.ubuntu.com/download/desktop](http://www.ubuntu.com/download/desktop)

This won't change anything on your disk. The OS boots entirely from the install media.
