### Post by Hospadar on 2009-10-24
If you want a GUI, the very first thing I'd suggest is that you just do a fresh install with the desktop version.

The main difference between the desktop and server version is that the desktop version come pre-installed with a GUI.  There are also some subtle differences in the server kernel that can make some video hardware behave strangely (since the server version isn't really intended for GUI systems).  I suspect your problem may not be to hard to solve in the absolute sense of difficult linux problems, but as a newbie, I think you'd have a lot easier time learning about linux and navigating the system with a GUI that works.

So I'd say, step one, try just installing the regular version (and I'd install 9.10 as well, it's going to be out officially in a couple days anyways, and will not change really at all in the time between now and a couple days from now.
