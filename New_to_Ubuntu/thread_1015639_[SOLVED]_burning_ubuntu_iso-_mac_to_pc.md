### Post by PocketDog on 2008-12-19
Use the Mac's own Disk Utility to 'burn image to disc'. It might 
be called 'burn .iso to disc'. If you put the disc that you've burned into your machine and examine it, and it only has the .iso file on it, you're not burning the image correctly - you're just burning it as data

### Post by ugm6hr on 2008-12-19
Where you burn the CD (or whether you yoe "audio" CDs) is irrelevant.

See: [http://www.psychocats.net/ubuntu/maciso](http://www.psychocats.net/ubuntu/maciso)

I suggest CD-R (although CD-RW are generally OK).  Also - check the MD5sum.

When you look through the burned CD, what do you see?  There should be a lot of files and folders, not just one.

PS: You could create a LiveUSB with a 1-2GB USB memory stick using unetbootin: [https://help.ubuntu.com/community/Installation/FromUSBStick#Automatic%20Approaches](https://help.ubuntu.com/community/Installation/FromUSBStick#Automatic%20Approaches) and use that (if you want to save on CDs)
