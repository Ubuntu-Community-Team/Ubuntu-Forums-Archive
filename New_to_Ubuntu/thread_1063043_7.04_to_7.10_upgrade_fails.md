### Post by diablo75 on 2009-02-07
No Jargon.  You got it.

Click System>Administration>Software Sources.

You'll see this:

[IMG]http://buranen.info/wp-content/uploads/2007/10/screenshot-software-sources.png[/IMG]

Where it says "Download From:", click on that button just to the right of that.  It will fan open and allow you to select other servers.

Pick a different server.  Alternatively you'll find a button in the server listing that says something like Select best server, which will ping them all and determine the one that is closest to you.  Once it's selected, click Close, and then Reload.

Hope that helps.

### Post by Eax.exe on 2009-02-07
I found a solution :)

Comment everything except:

```
deb http://archive.ubuntu.com/ubuntu/ gutsy-security main restricted
deb-src http://archive.ubuntu.com/ubuntu/ gutsy-security main restricted
deb http://archive.ubuntu.com/ubuntu/ gutsy-security universe
deb-src http://archive.ubuntu.com/ubuntu/ gutsy-security universe
deb http://archive.ubuntu.com/ubuntu/ gutsy-security multiverse
deb-src http://archive.ubuntu.com/ubuntu/ gutsy-security multiverse
```
