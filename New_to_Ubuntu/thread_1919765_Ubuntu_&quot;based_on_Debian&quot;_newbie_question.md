### Post by snowpine on 2012-02-03
Ubuntu continues to get maybe 90% of its code from Debian Unstable.

There are a few applications that are unique to Ubuntu (and not found in Debian) such as the Software Center and Unity.

### Post by sudodus on 2012-02-03
Yes, ubuntu is based on debian. Ubuntu uses mainly ***apt-get*** to install programs (instead of directly using .deb packages). In turn, several other distros are based on ubuntu, for example linux mint.

This works because linux is free software, with the source code open for everybody to look at, use and develop.

### Post by QIII on 2012-02-03
@sudodus

apt was developed by Deb and Ian (Debian) or a developer they brought on.  Ubuntu does use .debs.

[https://help.ubuntu.com/community/AptGet/Howto](https://help.ubuntu.com/community/AptGet/Howto)

Take a look at */var/cache/apt/archives *and see what you find there.

@aykoola

Don't take this analogy too far.  Imagine a microorganism growing a bud that eventually splits off.  The individuals are free to go where they want. In this case, the "offspring" returns from time to time to see if the other has anything new and interesting to add to itself.  Other OSes based on another sometimes go off on their own and never look back.
