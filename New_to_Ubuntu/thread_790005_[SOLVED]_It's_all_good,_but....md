### Post by kirios on 2008-05-11
Most things are covered pretty well in the default install - office suite, browser, audio, video etc. - and you can find almost anything you need in the repositories BUT you may still miss certain programs like Photoshop. GIMP is _almost_ an adequate replacement for Photoshop and Wine is good for older versions of Photoshop. Windows games may be an issue. I'm still using Windows at work because I need to use a statistical program called SPSS, but I rarely boot into Windows at home.
