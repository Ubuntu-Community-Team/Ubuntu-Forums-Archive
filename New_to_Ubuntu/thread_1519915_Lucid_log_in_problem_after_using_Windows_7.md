### Post by wilee-nilee on 2010-06-28
So did you reload the windows bootloader or did you put grub on a install thumb, or inside of the Ubuntu partition, or choose not to load it? the mbr shows as being the MS bootloader but you have grub in Ubuntu. 

Edit: Actually I think you just need to load grub to the mbr use this link.
[https://help.ubuntu.com/community/Grub2#Reinstalling%20GRUB%202](https://help.ubuntu.com/community/Grub2#Reinstalling%20GRUB%202)

This will load grub to the mbr and mount Ubuntu. Reboot to Ubuntu and run in the terminal ```
sudo update-grub
```
