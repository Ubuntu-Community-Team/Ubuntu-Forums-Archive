### Post by Nepherte on 2008-06-26
navigating through directories: cd <directory

copy: cp <name and location of original file> <name and location of duplicate file>

cut/paste: mv <<name and location of original file> <name and location of new file>

installing: sudo apt-get install <application name>

Uninstalling: sudo apt-get remove <application name>

There should be a beginners command chart somewhere on the forum.

### Post by bryncoles on 2008-06-26
if you press tab twice in the terminal, it will give you all possible commands you could type in!

if you start typing a command and press tab twice, it'll give you all possible completions. 

if you type 'man command' (where command is w particular function) you'll get the instructions for that command. soimetimes they're even comprehensible!
