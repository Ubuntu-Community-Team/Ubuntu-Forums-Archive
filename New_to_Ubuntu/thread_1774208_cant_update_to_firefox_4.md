### Post by Mariane on 2011-06-03
Just reading this output I see an error: 
No such file or directory: u'/etc/apt/sources.list.d//mozillateam-firefox-stable-maverick.list'

There should not be 2 / between sources.list.d and mozillateam but only one. 

I would go to:
/usr/lib/python2.6/dist-packages

Then do 

```

grep -R '/usr/lib/python2.6/dist-packages' * 

```

To try and find where the error is in the python script.
