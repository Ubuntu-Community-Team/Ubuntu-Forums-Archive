### Post by overdrank on 2009-11-02
Hi and what is the model of the graphics card?
You may use the command in the terminal ```
lspci | grep VGA 
``` to identify your hardware.
If it is nvidia and you are using the nvidia drivers located under system, administration, hardware driver you may try and use  the command ```
gksu nvidia-settings 
```
