I've use Windows for many years, so I'll write the list of applications from the Windows that are most appropriate for me and their most important features (for me). **Help me find the best equivalents in Linux world.** (Ubuntu brings some pre-installed applications, I've tried it, but most of them are useless (for me).)

Subtitle editor: Subtitile Workshop (rich format support, easy to use)

Video player: Media player classic (fast, simple, HD support, GPU acceleration, ...)

Torrent: uTorrent (lightweight, excellent web interface)

If you have M$ Applications that are a "MUST HAVE" in Linux, you need to
check out that application at the following sites, to see if they can be
installed, and will work.

WINE & Application Database:
[http://www.winehq.org/](http://www.winehq.org/)
[http://appdb.winehq.org/](http://appdb.winehq.org/)

CROSSOVER & Application Database:
[http://www.codeweavers.com/](http://www.codeweavers.com/)
[http://www.codeweavers.com/compatibility/](http://www.codeweavers.com/compatibility/)

NOTE: 
If you have an older DOS Application, that doesn't use an install software package,
you most likely can just create some subdirectories, then run that application
in DOSBOX.  If the older DOS Application uses an install software package, you
can install it in the subdirectory .cxoffice that is created when Crossover installs
their first package, and run it from Crossover, as the environment is already setup.
Does his make sense?  It does work, I've done it for several packages.
