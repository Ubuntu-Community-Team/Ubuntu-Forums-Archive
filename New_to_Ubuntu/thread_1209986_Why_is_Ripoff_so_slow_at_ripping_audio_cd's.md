### Post by treschny on 2010-02-09
I've just discovered the best CD ripping program for Ubuntu.  It's actually CDex for Windows, it works perfectly with Wine.  I had previously tried Ripoff (was so slow) and Sound Juicer (buggy).  CDex is fast, awesome quality that's totally customizable with LAME 1.32 (Ripoff uses LAME 1.1), has multiple CDDB's, jitter correction (useful for CD's with scratches) a better default track naming convention.  It works very well.  I can actually hear my CD drive spinning faster than with Ripoff.

### Post by NightwishFan on 2010-02-09
Please note that many drives are slightly faulty and may rip poor audio quality or sound artifacts. You can test and prevent some errors by using software such as CDParanoia or similar. CDParanoia is installed in Ubuntu by default. Sometimes taking a bit longer to rip an audio CD is for the best. Speed = Errors

Also note that Sound Juicer (yes a bit buggy..) and Rhythmbox (works for me) use Cdparanoia to rip, and correct errors. I always have excellent and verifiable results, as I rip to FLAC.
