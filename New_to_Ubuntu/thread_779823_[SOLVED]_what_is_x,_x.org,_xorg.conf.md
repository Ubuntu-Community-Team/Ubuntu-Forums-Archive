### Post by iSplicer on 2008-05-03
Hi.

"X" is the system in Linux that is responsible for your rich graphical interface. It is the part where it detects your mouse, outputs video, etc.

xorg.conf is the configuration file that has all x's settings.

### Post by Exershio on 2008-05-03
X = Xorg
/etc/X11/xorg.conf = Xorg's configuration. (Where your input/video/etc settings are stored.

X is basically a window system that allows your computer to render graphics. (I really can't think of any way to explain it heh)

You need X to run a desktop environment (Gnome/KDE/Xfce/etc) so you aren't stuck in terminal 24/7
