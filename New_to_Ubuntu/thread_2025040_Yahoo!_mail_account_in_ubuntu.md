### Post by Frogs Hair on 2012-07-14
That is one more choice than the first version of the shell. I think that feature is being developed. Also, the Gnome shell is set up to work with the Evolution Mail client not Thunderbird as Ubuntu. You can add your account to either mail client.

When you set up the date on the top panel in the shell you will notice a calendar link that only works with Evolution. The Gnome shell is an additional choice not the default desktop so it doesn't integrate as if it were default.
