### Post by Paqman on 2008-06-11
The package manager in Ubuntu is very good. It uses Debians APT system. The primary GUI for this is Synaptic, while at the CLI we use the commands apt-get or aptitude.

Example:
```

sudo apt-get install foo
sudo apt-get remove foo

```

The packages are .deb files, which you can also install by clicking on them. We can even embed links into web pages that hook into the package manager, for example [click here to install Firefox](apt:firefox)
