### Post by damis648 on 2008-04-29
open terminal and try

```
sudo chmod a+x [path to RUN file]
```

then

```
sudo [path to run file]
```

DO NOT INCLUDE THE BRACKETS IN THE PATH

### Post by sam_delta on 2008-04-29
to run a file , you must specify the full path

```
 sudo ./name of file
```

the "./" represents the current path, so youll have to be in the correct working space, for example, to run a file in "Desktop" you must

```
cd Desktop
sudo ./file
```

or from where ever you like

```
 sudo /home/user/Desktop/file
```

sam
