### Post by Krytarik on 2011-09-27
It seems like you are currently presented with Unity. Please see this guide to get started:

[http://castrojo.tumblr.com/post/4795149014/the-power-users-guide-to-unity](http://castrojo.tumblr.com/post/4795149014/the-power-users-guide-to-unity)

If you want to use the classic Gnome interface instead, choose "Ubuntu Classic" as the "Session" option at the bottom of the login screen, after clicking on your username, but before entering your password.

If you have auto-login enabled, you obviously need to log out to do so, after being auto-logged in. If you have passwordless login enabled, you need to disable it before via "Users and Groups".
