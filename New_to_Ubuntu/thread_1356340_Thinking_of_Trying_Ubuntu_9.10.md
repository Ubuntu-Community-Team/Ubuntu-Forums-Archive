### Post by cenzorrll on 2009-12-16
I've had Ubuntu on my thinkpad x31 (wireless b, pentium M 1.4ghz, 512 ram) since i bought it and it works perfectly fine.  Actually, it almost works better than my hp tx2500 (amd turion, 3g ram, wireless n).  In my experience, the older the technology, the better linux will run.  This is due to the extensive testing and bug fixing as linux matures and the release of driver source code by manufacturers.

However, i wouldn't really suggest Ubuntu on something older than my x31, if you have older (read: slower) hardware, then i would suggest trying xubuntu (xfce), or even lubuntu (lxde).

I currently have debian running lxde on a fit-pc slim, which shouldn't even be considered a desktop (amd geode 500mhz), quite well.  one of the biggest drains on a computer is the desktop environment.  you don't need to try a completely new distribution in order to get better performance.  you can just install a different desktop environment and you'll see a huge difference.  the only problem with that is that some DE's need to be setup in order to function well.  which is why i suggest distributions that have already done that for you.

Hi Mike,

Ive been in your exact position; I tried Linux over summer break from school. I personally love it. 

INSTALLATION: personally, i have not had any major problems with this area at all. Looking at your spec sheet, i would definitely suggest (should you decide to install it) using the 64 bit version. This will allow you to take advantage of your RAM more effectively, and thus get a slight performance boost in some programs. IF you were to run into problems, it would be with your wifi drivers. Just make sure that you check that your bios is up to date (its a 30 second install, not counting a reboot). Ive cleared up many problems by updating my bios. dont forget that you could install ubuntu side-by-side with whatever your current os is too!

CUSTOMIZATION: When you first boot, your desktop looks rather plain. ### Post by 3rdalbum on 2009-12-16
The link doesn't give a lot of relevant information about your machine (for instance, sure it's got 10/100 Base T Ethernet - but what chipset?). Chances are that the unnamed components will be fine because the machine is a couple of years old.

With 256MiB of RAM I don't think you can run the live CD, but you can install Ubuntu and run it from hard disk from that CD.

Your graphics are not optimal for Ubuntu. ATI does not support the Xpress 200M anymore, and the last driver they made for it doesn't work on today's Linux systems. However, there is an open-source driver in development from the community that will be able to get you up and running with good 2D performance and some degree of 3D acceleration. This driver is installed and activated by default in Ubuntu.

So, basically, your system should work with Ubuntu from the information given. Try it and see :-)

### Post by 3rdalbum on 2010-03-31
Broadcom wireless.

Try this:

1. Connect your computer to your router with an Ethernet cable, so you have an internet connection.
2. Go to System > Administration > Synaptic Package Manager and hit the Reload button, so you have a list of what's in the repositories.
3. Close Synaptic and go to System > Administration > Hardware Drivers. It will briefly say "Searching for hardware drivers", and then give you the option to download and install a driver for your card.
