### Post by stchman on 2008-06-11
> **appoloin said:**
> Hi guys,

When i was using windows i always use winrar and winzip. Is there anything in ubuntu i can use that is similar to this?

You can create/unpack .rar and .7z archives with Ubuntu very easily.

Install the following packages.

```
sudo apt-get -y install rar unrar p7zip p7zip-full
```

You are now set.
