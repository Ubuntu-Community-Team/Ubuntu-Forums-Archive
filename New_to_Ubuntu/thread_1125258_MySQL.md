### Post by BDNiner on 2009-04-14
It is a database program comparable to Microsoft SQL. If you plan on setting up a database then this would be a great choice. There is also SQL Lite among other database programs. For example I use a MySQL database to organize my music in Amarok. It makes handling the large amount of data quick and painless.

### Post by achase79 on 2009-04-14
MySQL is a database server. These servers allow for accumulation and retrieval of lots of information.  The most common uses are for sites that include PHP or cgi for creating pages from the information in the databases.  Some examples of these sites are e-commerce, forum, news, etc.
