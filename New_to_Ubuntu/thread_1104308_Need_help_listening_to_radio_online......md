### Post by Sarai the Geek on 2009-03-23
According to their FAQ you should be able to listen using this plugin: [http://www.linux-sxs.org/multimedia/mplayermozplug.html](http://www.linux-sxs.org/multimedia/mplayermozplug.html).
Unfortunately it doesn't not look like they support linux beyond that, so if that doesn't work (or you can't figure out how to install it- you have to compile it) you may not be able to use the stream feature.

### Post by mocoloco on 2009-03-23
I decided to play with a few things.  First of all, there's no need to get the mplayer plugin from that link, it's in the ubuntu repositories.  However as you said it doesn't work.  But I installed the vlc browser plugin and it does work.  mozilla-plugin-vlc. Can't see the controls for the player though but you can hear the audio. So maybe play with that.  Not sure what other codecs are needed because I've got them all installed already.
