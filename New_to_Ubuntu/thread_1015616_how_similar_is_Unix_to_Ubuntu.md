### Post by bhaverkamp on 2008-12-19
Ubuntu is a Debian OS. Linux, OS/X, and Unix are all derived from the Unix of old as far as the user mode commands, command line, and file system experience are concerned. They differ in their kernels the basic code that handles the low level operations of the computer. Unix uses the kernel derived from the original ATT kernels. Ubuntu and all it's Linux breatheren use the Linux kernel which is derived from the first kernel handed down to us from Linus Torvalds. OS/X is I believe an offshoot of an MIT kernel that was customized for the NEXT computer OS that Jobs founded/joined during his 40 years in the wilderness. For the most part you cannot tell the difference from the command line or from the programs being run.
