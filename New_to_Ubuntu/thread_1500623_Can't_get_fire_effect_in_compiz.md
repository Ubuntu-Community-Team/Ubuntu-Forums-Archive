### Post by howefield on 2010-06-03
Are you using Ubuntu 10.04 & have compizconfig-settings-manager installed ?

If so, using Synaptic Package Manager, download compiz-fusion-plugins-extra.

This will give you a load of extra settings to configure in Compizconfig Settings Manager, one of which will be "Extra Animations" you'll find the fire plugin in there. Enable it and then go into Animations plugin and select the Close animation tab, highlight the first row in the Animation Selection window and press edit, change animation to fire.

It has been a while since I played with this, so hopefully I've given you enough to get it going.
