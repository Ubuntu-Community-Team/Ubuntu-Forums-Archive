### Post by jmszr on 2008-10-20
Ubuntu wants to make sure it is stable before putting it in the repos. If you want to run 2.48 go to getdeb.net > categories > Graphics and Design > Blender 2.48. Download the .deb file and install (easy). You will have to keep it updated yourself but that's not exactly difficult.

Edit: You will want to remove any existent version of Blender-either via Synaptic or Applications > Add/Remove before installing 2.48.
