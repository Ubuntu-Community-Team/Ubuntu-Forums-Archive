In Evolution Mail, I think, if you set up two accounts (with different smtp servers) then you would be able to send from either in the new message window, by selecting the desired account in the "from" section. Also, your problem of sorting by email address should be easily fixed with a message filter which sorts based on whom the recipient is:
In Evolution, this would be:
Edit -> Message Filters
Click "Add", then choose "Recipient(s)" "is"
Then, "Move to Folder" ...
