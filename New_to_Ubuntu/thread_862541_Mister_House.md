Take a look at [http://misterhouse.wikispaces.com/MisterhouseOnUbuntu](http://misterhouse.wikispaces.com/MisterhouseOnUbuntu) for how to set up on a slightly out-of-date version (6.06/Drake) but it covers most of the points.
 Mister House doesn't need compiling as it's a whole bunch of Perl scripts, you just need to set up the environment.
 I'm setting up a version on an eeePC with an SD card to hold all the scripts/configuration etc. It's looking pretty useful so far.
