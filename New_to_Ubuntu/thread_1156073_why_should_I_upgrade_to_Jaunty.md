### Post by Didius Falco on 2009-05-11
First, I'd do a little research. Read the release notes [http://www.ubuntu.com/getubuntu/releasenotes/904](http://www.ubuntu.com/getubuntu/releasenotes/904)
to see if there are any known conflicts with any of your hardware.

Pay particular attention to your video card - I can't count the number of people that upgraded only to find that their card doesn't have 3D support from the manufacturer under Jaunty.

Run the CD for a while and test it out. If you have room on your hard drive, install it as as second Ubuntu installation. That leaves your well-tuned Intrepid install ready as a fall-back without all the hassle of of an emergency backup.

Besides, I'm a big fan of the clean install. Upgrades always make me feel like I'm the second one to use the bathwater. <G>

Beyond that, it's basically up to you - do you enjoy getting "under the hood" and tinkering, if you should run into a problem? Is your current install "mission-critical" to you?

Regards,

Didius
