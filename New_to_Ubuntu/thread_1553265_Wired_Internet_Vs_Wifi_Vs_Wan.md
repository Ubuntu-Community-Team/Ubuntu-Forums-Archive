### Post by seanarickymurphy on 2010-08-14
Wi-fi is a service while Wlan is a application to connect to that service via through a router..

Edit - Alright, to put it simply Wi-fi is the service via hotspot, wlan is a application for that service to connect to it through lan or wired, its the same basics to put it simply. Wired is having a direct line to go into the service (Router) or wall. While wlan is pretty much just going wireless to connect to that wireless service which would be your router..

This is when people set up passwords and encrypted wlan's to stop people from doing just that. The owner's of routers can also set up access lists, which basically means that they can say which computers are allowed to connect. This is done by a MAC Address, which, if you like, is the unique signature of the hardware itself (network card) inside the computer that connects to the router. Each network card has its own unique MAC Address, and routers can be set up to only allow certain cards.

Also, there is a difference between WAN and WLAN...

WAN - Wide Area Network
WLAN - Wireless Local Area Network
