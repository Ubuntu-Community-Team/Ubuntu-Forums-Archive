### Post by Methuselah on 2008-08-13
Open firefox and choose help about.
What version does it say?

Firefox has been updated in the Ubuntu repositories.
If you have been applying automatic updates, you likely have 3.0.1 already.
If not you should have the update available.

There is no need to download it from the web for Ubuntu.
