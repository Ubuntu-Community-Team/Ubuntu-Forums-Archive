### Post by littlepeon on 2010-12-15
deluge torrent is the friendliest/easiest torrent client to use with ubuntu.
it includes the ability to use blocklist easy (if u don't know what blocklists are--google it)
it has a small memory footprint and can be used on older systems (i have a pentium 3--800mhz(hot stuff for 1999)system on my network running it 24/7)

there is also vuze (azureus) that most people from windows are used to, however that is java based and has a big memory footprint and can cause older systems to run out of memory quick (if ur going to leave it running for days on end)

there is a port of utorrent to linux/ubuntu and it is friendly/easy however i have issues with the owners so i can not in good faith promote that program (utoreent was sold to the original creator of the bit-torrent client brad cohen who is also in a partnership with the mpaa/riaa on promoting the protocol for movie use)

### Post by seawolf167 on 2010-12-16
The apps were covered above, as for the "how to", its seriously simply; just download the torrent file (from whatever tracker you like), then double click the .torrent file to open (by default in Ubuntu it should be associated with Transmission, which will then open the file). Just read the prompt that comes up (where to save, etc.), click ok and you're off!
