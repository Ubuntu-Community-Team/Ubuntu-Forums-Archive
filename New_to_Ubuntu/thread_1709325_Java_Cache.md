### Post by expatCM on 2011-03-18
Has anyone heard of clearing a Java cache?

I access Webmin through Firefox to access a server and the Webmin file manager has stopped working.  I get a page of html instead that says Failed to get language list.   This is not an unknown problem but google does not deliver any reliable solution.  I found one post that said clear the Java cache and the problem is gone. ---

### Post by Dutch70 on 2011-03-18
Try going to Tools in the Firefox menu bar. Select clear recent history. Make sure the box beside Cache is ticked, and click >clear now.
