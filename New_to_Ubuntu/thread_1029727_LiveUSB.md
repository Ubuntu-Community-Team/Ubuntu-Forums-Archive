### Post by BeastMode on 2009-01-03
Kinda wondering if maybe you shouldn't have asked this question on Fedora's forums.

Oh...yeah....because the Fedora community doesn't even compare to the Ubuntu one.

Gotcha.

But, yes...some laptops (Toshiba) don't like to boot from flash drives, as well as some desktops.  Check your bios or your boot sequence.  See if it's enabled there, or bootable from there.

To answer the format question, you can certainly do that.  I usually do it with cfdisk, but if you're not really sure what you're doing, don't go there.  Consider maybe GParted instead.
