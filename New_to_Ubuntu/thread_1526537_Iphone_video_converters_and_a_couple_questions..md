### Post by Paqman on 2010-07-08
Try Arista Transcoder, you can get it from Ubuntu Software Centre. It should have presets for the Iphone in there.

To install software form outside the repos, you've got a couple of options:

[LIST=1]
[*]Look for a PPA containing the package. This way you can install through your normal package manager and you get automatice updates just like any package you install from a repo.
[*]Look for a .deb package. These install just like a Windows .exe, download to your machine then click to install.
[/LIST]
