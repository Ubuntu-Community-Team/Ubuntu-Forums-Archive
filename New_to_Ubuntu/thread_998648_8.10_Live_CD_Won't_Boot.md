### Post by binbash on 2008-12-01
I have asus f3sr notebook , probably similar to yours, i got same issue.THe only way i could find : Download kubuntu alternate cd.install kubuntu.After that , ubuntu wont boot , you will get blank screen.

Now press alt +ctrl +f1 and drop to terminal.

sudo apt-get update
sudo apt-get upgrade

reboot now(YES reboot before giving the commands below).It wont boot again :) then drop to terminal again and :

sudo apt-get install build-essential xorg-driver-fglrx
sudo aticonfig --initial -f

reboot now.You will be ok
