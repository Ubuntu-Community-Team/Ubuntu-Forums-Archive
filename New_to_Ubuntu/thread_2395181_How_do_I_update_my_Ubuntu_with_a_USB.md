### Post by Xian on 2018-06-27
You need to tell the computer to boot from the USB. This is usually done by pressing the F12 key. 

See here for more info:

[https://tutorials.ubuntu.com/tutorial/tutorial-install-ubuntu-desktop#3](https://tutorials.ubuntu.com/tutorial/tutorial-install-ubuntu-desktop#3)

### Post by Geoffrey_Arndt on 2018-06-28
What do you really want to do?   Do you want to run 18.04 in "Live" (demo) mode?   Do you want to install 18.04 over 17.10?    Do you want to just try the "online" upgrade from 17.10 to 18.04?  (which doesn't even need the 18.04 usb stick at all)?     

Also,  did you check if your computer is "enabled" to boot from external usb instead of booting first from internal drive?    Further, there is a special key on most computers that will display the uefi boot menu (which is a boot menu that looks for all bootable devices (external and internal) and shows those devices on a menu for you to select from.    On many PC's, that key is F12 . . . which must be pressed as soon as you see the manufacturers logo in the PC start-up sequence.   On my Acer Travelmate, I press F12 as soon as I see the Acer Logo.
