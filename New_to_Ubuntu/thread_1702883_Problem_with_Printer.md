### Post by maqtanim on 2011-03-08
Does not it find the driver automatically from the internet? Did you try to add the printer from **System > Administration > Printing**? If you connect the printer and try to print something (open office as an example), Ubuntu should prompt you to install the printer from a list of printers (that's how my canon worked).
