### Post by gordintoronto on 2010-01-03
Two ways: one is to put the Ubuntu CD in the drive while Windows is running, and do a "WUBI" install.  Make sure you give Ubuntu more disk space than the default.  Second way, shrink your Windows partition so there is space for a Ubuntu partition, boot from the Ubuntu CD and install as a "dual boot."

Go to help.ubuntu.com and search for more information, there is a lot of it.
