Does the Audacity in the repositories not function properly for you? It has pretty much never given me problems, and I use it quite often. 

Ubuntu often uses beta versions, and if that version works well with Ubuntu, why use another one? The version in the repositories has been tested the most thoroughly for Ubuntu 8.10. The version 1.2.6 that you want has not been equally tested.
