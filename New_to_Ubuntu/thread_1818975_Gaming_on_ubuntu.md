### Post by akoskm on 2011-08-05
Some of them you can other ones not really. AFAIK the most widely used program for such purposes is wine (I'm using that one to play WoW).
They have a nice appdb: [http://appdb.winehq.org/]("http://appdb.winehq.org/"). This is a quite complete database with testing results from different operating systems, comments, workarounds etc.
If you are interested in specific game the most effective way is to use the search function of this forum, or Google.
Good luck!
