### Post by Thelasko on 2008-11-21
[I recommend this to all new Ubuntu users.]("http://www.psychocats.net/ubuntu/installingsoftware")  

P.S. I think this is what sets Ubuntu apart from most operating systems.  Installing software is different, but in the end, it's very easy.  In my opinion, it's easier to install most software in Ubuntu than in Windows.

P.P.S. WRT54GC is a [wireless router]("http://en.wikipedia.org/wiki/Wireless_router"), not a [wireless adapter]("http://en.wikipedia.org/wiki/Wireless_network_interface_card").  The wireless adapter is in your computer.  The wireless router shouldn't require any special driver, but the adapter should.
