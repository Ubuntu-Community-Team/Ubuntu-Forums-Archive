First thing you need to do is determine what version of Ubuntu you want to install: Dapper or Hardy.  I, like several others, would suggest you go with Hardy since it's a Long-Term Support (LTS) release, whereas Dapper was not.  Still, the choice is yours.

Now, it sounds like you have an install CD, and you've checked the BIOS to make sure the optical drive is primary in the boot chain; that's good.  Did you burn the CD as an *image* though?

### Post by Therion on 2008-12-31
I doubt the PC itself is trashed.  I would venture to say that 90+% of the time, failure to boot from a LiveCD can be traced back to one of a few things:

1.  The disc was burned too fast.  
Use 8x or slower for a CD, 4x or slower for a DVD.

2.  The download itself is corrupt.  
Did you check the MD5 Sum on the download?  If you didn't, try downloading via bittorrent.  Torrents auto-verify as the download is done ensuring a clean download.

3.  The burn is bad.  
Did you do a read-verify on the CD you burned?  Did you check the MD5 Sum of the newly burned CD?
