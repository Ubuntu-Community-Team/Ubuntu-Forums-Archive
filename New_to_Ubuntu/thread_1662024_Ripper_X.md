### Post by fleamour on 2011-01-09
[https://answers.launchpad.net/ubuntu/+question/42517](https://answers.launchpad.net/ubuntu/+question/42517)

1. Open Nautilus
      - click Places -> Home Folder

2. Choose the Edit, then Preferences menu item

3. Click the multimedia Tab.

4. All media handling can occur there, including multimedia players.

Hving used Ripper X, I would say it is work in progress.  A far more solid app altogether is Asunder.  

Also setting audio CD to auto run Asunder is counter productive as will open another instance alongside the original every time you load an audio CD.
