### Post by NexusZA on 2009-09-12
Most netbooks actually do come with recovery CD's that will reinstall XP but to be perfectly honest dual-boot with Ubuntu is such a doddle these days that you really should have no issue. I have just finished a dual boot install on the laptop I am typing from right now and that was with Windows 7 not XP and Ubuntu still accuratly detected the other OS and gave me all the options I needed to do an automated install with no problem.

The only issue you may have with a dual boot is to ensure you have enough drive space for two OS's so the 8GB Acer Aspire One SSD netbook for example would be no good for you.
