### Post by takisan on 2010-12-12
EDIT: I'm assuming you mean Quickbooks for Windows. so that's why I'm suggesting this.

I'm sure that there's probably a sufficient alternative in the Software Center, but for running Windows Programs on Linux, you can download and install WINE ( [http://www.winehq.org/](http://www.winehq.org/) ) and Windows programs can be installed if needed. Also, if WINE doesn't work, VirtualboxOSE can have a Windows VM installed and run. Best of luck.
