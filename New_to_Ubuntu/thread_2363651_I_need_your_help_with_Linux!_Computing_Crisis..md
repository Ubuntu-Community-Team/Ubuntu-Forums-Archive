This ThinkPad has a 7th gen i5, 940MX graphics, 1080p antiglare screen, and is all in all a nice machine. Windows cripples it. So if I install Ubuntu what do I use for the following needs: 

**Mail** - Thunderbird or Geary are two good options. If you really love Outlook, then Evolution is the nearest equivalent.
**Contacts** - depends on the flavor of linux. Some have this built in or as an add-on. Otherwise, this can be handled by your e-mail client.
**Calendar** - same as Contacts. Thunderbird now comes bundled with the Lightning calendar extension. I've used it extensively. Evolution has a built-in calendar.
**Music** - Rhythymbox is default in many installations, but Amarok and Clementine are also notable. (I think Rhythymbox is probably the most friendly to iPods - but you can't switch between MacOS and Linux for access - pick one)
**Photos (large collection)** - Shotwell is the one here. F-Spot was better in some respects but development stalled. Gnome is working on its own 'Photos' app though it is still not complete. Shotwell can download photos from your iPhone/iPad.
**Cloud Service** - Dropbox and others are available, some integrate well with the file manager. Not sure if iCloud works, you'll have to investigate that. I think at worst you can use the web interface for it.
**Backup to local HDD or online**  - I think Ubuntu still ships DejaDup by default but I prefer BackInTime as it is more like Time Machine. I'm sure there are some cloud options for this as well though I don't use any.
**Browser** - Firefox and Chromium(open-source base of Chrome) are the two leaders here. 'Web' (Epiphany) from Gnome is capable, but not as polished as the main two. There are plenty of other options but none of them can reliably render like FF and Chromium. You can also use Chrome proper if you like.
**Office documents for uni** - LibreOffice if you want 'free as in beer.' I've never had an issue with MS Office docs, but if you don't mind shelling out a few, check out CrossOver office. They guarantee compatibility with MS formats.
**Note taking** - lots of options here. It depends on the features you want, particularly in the syncing arena. Evernote clients exist if that's your speed.
***Does* Minecraft work on Linux or what?** - yep! I installed it 2 years ago for my cousin. Works just fine. (requires a java run-time I believe, was really easy to set up though)

As for which linux distro to use, try several as others pointed out. Here are some in the Ubuntu Family.

Lubuntu - light and agile, XP style desktop.
Xubuntu - also fairly light and more featured than Lubuntu, but more Mac Like in some respects.
Kubuntu - a little heavier than Xubuntu but not much. Very configurable - almost too much so. By default very similar in layout to Windows 10.
Ubuntu - A different approach all together using a desktop called "Unity". The 16.04 Long Term Support version will be around till April 2021. Starting with the 17.10 version this fall, they will switch to a stock Gnome desktop.
Ubuntu Gnome - designed with a focus on touch devices, though desktop use is getting some love lately in development. If you wait till October, just install the main Ubuntu version of 17.10 Artful Aardvark and you'll get a Gnome desktop. Otherwise, install Ubuntu Gnome 16.04 or 17.04. (17.04 is only supported for 9 months, as 17.10 will be. April releases every 2 years are supported for 5 years, 14.04, 16.04, 18.04 etc.)

A few caveats. ANY Linux distribution can probably be made to look and almost function like, any other. You can even clone Windows and MacOS in many respects. How difficult this is, or what setup you like out of the box is the only real deciding factor. (there's also default applications to consider, but those can easily be installed/removed on any of the above distributions)

Some are easier to change themes on than others. It all depends on how much you want to tinker to get it 'just right' and how much you want it to 'just work.'

There are plenty of other good distributions out there, but these are going to be the easiest out of the box to work with.

Some others that are based on the Ubuntu series but are usually not the cutting edge are:
Mint - based on latest Ubuntu LTS. Very user friendly. Uses a more traditional desktop layout like Windows <=7.
LXLE - based on latest Lubuntu LTS but faster and comes with options for 3 or so desktop layouts. One XP like, one Mac like, and a hybrid version.
Ubuntu Studio - based on latest Xubuntu LTS. Packaged with a set of apps designed mostly for artists, musicians and photographers. Certainly capable of other work, but you might need to install those apps yourself.

Based on your hardware, I'd probably opt for Ubuntu, Ubuntu Gnome, or Kubuntu depending on the desktop layout you like and how much you like to tinker. If that ThinkPad is touch capable, stick with those three as most other distros aren't quite there yet.

If Windows10 is already making your ThinkPad syrupy, I would not recommend running a virtual machine on it with Linux as a guest. Either dual boot, or convert the Win10 instance to a VM and run that with Linux as the host. Even better, if you have or can get a legit copy of Win7, run that in a VM for any necessary MS apps. That is a lighter and faster solution than Win10. Keep in mind, Win7 is already in extended support mode and eventually software makers will dump it though I think we're a few years away for that and it should get you through school. You can also research slimming down Win10 (or any version) and removing or turning off things you don't really need to speed it up.
