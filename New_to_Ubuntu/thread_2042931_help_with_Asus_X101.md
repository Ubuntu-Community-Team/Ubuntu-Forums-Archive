### Post by cariboo on 2012-08-17
You need to create a bootable USB thumb drive in order install ubuntu on your netbook. I'm pretty sure that meego doesn't have a utility to do so. If you have access to a computer running WIndow or Ubuntu, you can use [unetbootin ]("http://unetbootin.sourceforge.net/")to create a bootable device. It is also available in the Ubuntu repositories.
