Welcome to the forums!

Do you want to know how to launch the task manager in Linux?

In Ubuntu you can access an application called System Monitor by going to 'System -> Administration -> System Monitor' or (if you're using Unity) search for System Monitor in the dash.

It will look something like this:
[img]http://www.blog.highub.com/wp-content/uploads/2008/11/ubuntu-system-monitor.png[/img]

Another way would be to launch a terminal, and write:
```

top

```

This will launch a text based task manager.
