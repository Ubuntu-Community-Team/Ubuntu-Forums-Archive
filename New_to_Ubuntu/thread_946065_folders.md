### Post by meanburrito920 on 2008-10-13
If you install ubuntu through Wubi then I believe there is no way to access your windows drive. Only if you install it through dual boot is it possible. However, to double check, navigate to /media on your ubuntu install. if you see something like sdb or sda, etc., click on it and see if your windows stuff is there. that is where it will normally be because ubuntu sees it as being a separate disk.
