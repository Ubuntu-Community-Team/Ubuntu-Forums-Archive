### Post by Mark Phelps on 2012-10-03
MS Windows and Linux use different network hardware drivers. If you're accessing the same websites, using the same Browser and same version of that Browser, during the same timeframe, the "speed" difference is likely due to driver differences.

I ran into this long ago when I was using a Marvel network chipset.  The MS Window speed (using SpeedTest) was much faster than the Ubuntu speed -- both using the same version of Firefox.

I was able to download a compile a new version of the Linux network driver from the Marvel site -- and once I did that, the speed difference all but vanished.
