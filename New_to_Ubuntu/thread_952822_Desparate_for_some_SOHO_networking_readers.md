### Post by dash86no on 2008-10-19
This site got me started on my Ubuntu home server:

[http://linuxgazette.net/141/lazar.html](http://linuxgazette.net/141/lazar.html)

It will show you how to get the following up:

* Webmin for remote (GUI) management
* DNS/Web caching proxy/firewall

From that base I went on to add a CRM (sugarcrm), mail server (zimbra), CMS web site (Joomla) using various how tos on the web. 

Good look with the set up.
