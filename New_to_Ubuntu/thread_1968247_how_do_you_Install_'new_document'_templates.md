### Post by xTheAngrySockx on 2012-04-30
Solved my own issue, to anybody else who is wondering all you need to do is navigate to the "Templates" folder in the "Home" folder and drag and drop a file in it, or create a new document.
For example, go to "Templates " right-click, hit create a new document and name it "New Text File.txt" now whenever you right-click and hit create a new document, this will be listed in the availabe templates
