### Post by mastablasta on 2012-02-16
drivers in linux are often part of kernel so for latest drivers the easiest is to use latest version of the operating system (11.10). if latest version of Ubuntu works too slow then you can use Unity 2D or a completelly different desktop environment found in Xubuntu or Lubuntu to reduce the bling and get better performance.
 
if drivers for your wi-fi chip are already a part of kernel it could happen that wi-fi is immediatelly recognised when you boot live media (try it option). if there are proprrieatry drivers available for your wi-fi chip the easiest method of install is to plug in a cable and enable the drivers (this should download them and install them).
 
if drivers for linux do not exist then last chance is to use a wrapper (*ndiswrapper*) that "wraps itself" arround windows drivers.
 
if that doesn't work then i think that's about it - it is simply not compatibel with linux then.
