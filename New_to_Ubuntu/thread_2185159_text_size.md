### Post by grahammechanical on 2013-11-01
The text size in what? Google Chrome? That is nothing to do with the OS. In fact Google Chrome is not a default Ubuntu application. Chrome surely has a view menu where the text can be zoomed. It also surely has a settings dialog where the text size can be adjusted. I know that Chromium has this. Is this absent in Chrome?

Ubuntu has System Settings and a Universal Access utility where the system text size can be set to large text.

It seems that you do not understand the Free and Open Source philosophy of Community Development. The developers of Ubuntu have no intention of producing an all inclusive operating system. Space is allowed for community developers to produce utilities and they can do this because the source code is available to them.

I have just searched the Ubuntu Software Centre and I have found 3 tweak tools for download and installation. Unity Tweak tool, Tweak Tool for Gnome 3 settings and a Chinese Tweak Tool for Gnome 3 settings for use with Ubuntu Kylin, the Chinese Ubuntu edition. So, I do not see what you are complaining about.

Ubuntu is a community gift. It is free to download and install and use. There is some truth to the saying, "we get what we pay for." But I also believe in the saying that "beauty is in the eye of the beholder." And this is where you and I differ. You find Ubuntu to be cheap. I find Ubuntu to be priceless and beautiful because Ubuntu comes not from commercial greed but the good intentions and efforts of a community of developers, many of whom are volunteers.
