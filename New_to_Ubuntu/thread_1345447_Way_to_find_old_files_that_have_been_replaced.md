### Post by Old *ix Geek on 2009-12-04
A few things come to mind:

What program or editor were you using to edit the file? It may save a backup copy even if you never explicitly told it to do so.  BUT, it will likely only save ONE backup, so if you re-open the file now, the good backup will be overwritten with the bad file.

Speaking of backup copies... Yep, a backup would solve the problem! I think we're all guilty of getting lazy and not backing things up often enough, but one little incident like yours is a good reminder that backups are invaluable.
