### Post by askreet on 2008-05-27
A fresh installation of a Steam App should default to a friendly 640x480@60Hz 24-bit.  Very few LCDs don't support this resolution.

If I recall correctly, steam apps can have the flags -width XX -height YY added after the launch options.  In Steam, right click the application (Counter Strike) and find the game parameters options.  Set something like -width 1024 -height 768 (or whatever seems appropriate for your monitor).
