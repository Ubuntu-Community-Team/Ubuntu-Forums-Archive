### Post by terdon on 2010-08-19
Sorry, I do that using sed on the command line. I know that arachnophilia ([http://www.arachnoid.com/arachnophilia/index.php](http://www.arachnoid.com/arachnophilia/index.php)) which is in java and works on various platforms has this functionality. 

Now, if you really want to make the change to linux, I recommend using a more powerful editor such as emacs. Emacs can definitely do what you need (see [http://my.opera.com/AxxL/blog/2007/09/23/emacs-query-replace-in-open-buffers](http://my.opera.com/AxxL/blog/2007/09/23/emacs-query-replace-in-open-buffers)) but it is not the most user friendly of text editors. It is, however, IMHO the best. 

The other *NIX classic is vi, or vim. Even less user friendly than emacs though... You could also try nedit which is similar to gedit and quite good or pico.
