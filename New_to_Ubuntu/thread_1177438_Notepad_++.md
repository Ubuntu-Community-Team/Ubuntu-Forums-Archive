### Post by mcduck on 2009-06-03
> **Mornedhel said:**
> Gedit (the default text editor found in Applications > Accessories) has tabs and syntax highlighting, so it's pretty close to Notepad++.

There are a lot of other advanced editors available. For a time I used SciTE, now I'm an emacs fanatic.

I should add that while Gedit may seem pretty basic on the first sight, it's incredibly expansible through plugins.

So, if you feel like you are missing some feature first get the gedit-plugins package, and remmber also to take a look at other available plugins here: [http://live.gnome.org/Gedit/Plugins]("http://live.gnome.org/Gedit/Plugins")

At some point I had Gedit configured so I could use it to surf the net and play music. Not that it made any sense but I *had* to do that after a friend of mine compared Gedit to Windows Notepad.. :D
