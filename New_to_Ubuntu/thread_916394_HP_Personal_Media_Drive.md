### Post by zephyrcat on 2008-09-10
Assuming that it is just a USB hard drive, it should just work if you plug it in. Most likely, however, the software included with the drive will not work, though, so you will need to find some other backup software. Try going to Applications > Add/Remove... and searching for "backup."

Hope that works!
