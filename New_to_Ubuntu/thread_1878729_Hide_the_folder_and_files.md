I think he's talking about user permissions. Doensn't Ubuntu ship with users having permissions to view other users directories?

UPDATE:
From [https://help.ubuntu.com/8.04/serverguide/C/user-management.html](https://help.ubuntu.com/8.04/serverguide/C/user-management.html)

"By default, user home directories in Ubuntu are created with world read/execute permissions"

Under 'User Profile Security' on the above url will tell you how to change permissions on user accounts.
