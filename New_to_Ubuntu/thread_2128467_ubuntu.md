### Post by na5h on 2013-03-23
For a thorough answer, you should read the Wikipedia article that ibjsb4 posted or visit [Ubuntu's Official homepage]("http://www.ubuntu.com/ubuntu").

Somewhat simplified but hopefully easily understandable answer: It's a system where you point and click on things to open programs and perform everyday tasks, sort-of like Windows or OS X, except it's totally free. It's important to keep in mind, though, that Ubuntu is not a free version of Windows or OS X. It's a totally different system, so it also looks and behaves quite differently. The programs are different too, even though many well-known applications such as Firefox or Spotify have versions for Ubuntu as well.

### Post by SMOKE14 on 2013-03-23
Ubuntu is an operating system for PCs(personal computers). It's like Windows (which your PC probably has) or Mac OS X, except, it's free! Most software you need for day to day tasks is right there out of the box. Web browser, Office Suite like Microsoft Office (Word, Powerpoint, Excel) and music player. It will take some getting used to though, because it's a lot different than Windows/ OS X. I recommend using Wubi if you have Windows to keep Windows, but also try out Ubuntu.
