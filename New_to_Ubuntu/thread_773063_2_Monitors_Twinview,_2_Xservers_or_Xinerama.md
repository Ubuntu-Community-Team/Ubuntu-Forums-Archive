### Post by MrHippocampus on 2008-04-28
Replace the Options "MetaModes" line with this one (make sure to keep it all on one line):

```
Option         "metamodes" "nvidia-auto-select,1280x1024; nvidia-auto-select,null;"
```

Then once youve restarted X, start ETQW, the first time it should be across both screens, but in the options you should then have the option of the resolution of only your first screen, once you select that and restart ETQW it should work.

If that doesn't work for some reason, all you should have to do is press Ctrl+Alt++ (thats the plus key on the numpad) to cycle between one screen and two screens. So simply do that before you start ETQW and it should work :).
