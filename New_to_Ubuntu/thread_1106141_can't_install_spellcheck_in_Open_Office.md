### Post by meho_r on 2009-03-25
In 3.0 just find spelling dictionary you need on OOo web site and add it through Tools > Extension Manager. Almost everything now goes through Extension Manager. Much simpler than in 2.x

EDIT: BTW, you can use old spelling dictionary (for OOo 2.x) in case you can't find the one for v.3.0. Just zip the files and change the extension from .zip to .oxt and then add them through Extension Manager.
