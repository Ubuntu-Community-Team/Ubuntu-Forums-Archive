### Post by ankspo71 on 2011-05-04
Hi,
You can click on the "#" (the code tag) located above your post to insert blocks of text inside boxes.

Are you using this KDE removal command specifically for Ubutnu 10.10 (aka Maverick)?
[http://www.psychocats.net/ubuntu/puregnomemaverick](http://www.psychocats.net/ubuntu/puregnomemaverick)
Note that if you upgraded your KDE from 4.5.1 to a newer version (like  through an external repository), then the command provided from psychocats.net may not work.

Try using the apt-get install -f command to see if it will fix any broken packages:
```
sudo apt-get update
sudo apt-get install -f
```

Sometimes (on rarer occasions) packages also need to be updated to fix dependency problems (broken packages), so if the above command doesn't help, try preforming a system update:
```
sudo apt-get update
sudo apt-get dist-upgrade
```

Now try to use the long KDE removal command again.

Hope this helps.
