Some themes stick the theme archive into another archive, so try to extract it. If it leaves you with a bunch of folders, leave the archive as it is. If it contains another tar.gz archive, try installing that. Also [gnome look](www.gnome-look.org) is a very good site for themes, and most of theme work straight away.
