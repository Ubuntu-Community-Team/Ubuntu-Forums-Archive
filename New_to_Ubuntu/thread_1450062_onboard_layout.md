### Post by tgalati4 on 2010-04-08
If you are talking about multi-language keyboard support, then in gnome, keyboard preferences, just add another layout.  I have russian and english keyboard layouts and with a gnome-panel keyboard switcher, with one click, I can switch keyboards and bring up an on-screen layout so I can see the key mapping.

This is helpful when you are chatting in skype with multiple folks in different countries in different languages.
