### Post by Antarctica32 on 2011-01-26
Ok, so I think I figured it out. There are some serious bugs in the x11 cursor selector. Download a .gz cursor file, open it up, extract it to your .icons folder (hidden) in your user's home folder. Then it should work if you then go to System, preferences, appearance, the customize button at the bottom, then pointer on the upper right. That is so far the only way I know, and it only works for .gz files.
