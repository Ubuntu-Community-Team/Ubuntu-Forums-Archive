### Post by snowpine on 2010-08-11
Red Hat is an expensive Linux distribution or "distro" for advanced users, corporations, and system administrators.

Ubuntu is free of charge and is a popular and user-friendly distro for Linux beginners such as yourself.

I suggest keeping Windows while you are learning Ubuntu. You can set up a "dual boot" and choose between Ubuntu/Windows each time you turn on the computer.

In fact, you can burn a "Live CD" and test-drive Ubuntu with no change to your computer. This is what I would recommend to get started. :)

### Post by arashiko28 on 2010-08-11
Hello, first of all welcome and don't be afraid to ask, all of us started right where you are today.
Here you have a good guide.
> [http://www.ubuntupocketguide.com/index_main.html](http://www.ubuntupocketguide.com/index_main.html)
But it's pretty easy to do; 
1)first go to:
> [http://www.ubuntu.com/desktop/get-ubuntu/download](http://www.ubuntu.com/desktop/get-ubuntu/download)
2)Download the .iso file containing Ubuntu
3)Burn the .iso file as you would do normally. (I recommend the CD because you can keep it for further usage, installations and pass it on).
4)Make a backup of all of your important files. (Recommended in a sepparate HDD or burn it on DVD's)
5)After the backup is done, insert the Ubuntu CD and reboot your computer.
6)Choose start Ubuntu without making changes (Live CD) This will allow you to run, test and have a feel of Ubuntu before you install it, as well as a chance to check compatibility. If any problem, usually can be solved, there is a thread about that too. Once done that;
7)Double click on the install icon on your desktop while on live CD (this way, you can go online and solve any doubts while installing).
8)Just choose between using entire disk, installing side to side, or manually (If you want to delete Windows, use entire disk)
9)Create the swap partition and choose size (Same or double as your RAM cappacity)
10)Create your installation partition and choose /
11)Follow instructions, and in about 15 minutes, you'll have a complete functional Ubuntu installation.

If you have any doubts feel free to ask!
Good luck!
