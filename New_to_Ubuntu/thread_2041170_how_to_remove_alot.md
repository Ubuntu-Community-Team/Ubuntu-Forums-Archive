### Post by vexorian on 2012-08-12
I'd say this is a firefox thing, isn't it?

If you are just starting to use ubuntu, your best chance is to just delete your firefox profile: Go to your home or personal folder. Then use the view menu to make hidden files visible. Find the .mozilla folder and delete it. Then restart firefox.

Edit: If you enter the .mozilla folder and remove the "extensions" subfolder it should wipe only the extensions and not web history.

### Post by OM55 on 2012-08-12
It is a FireFox thing... Just click on the small great/red/blue/yellow icon which is to the left of the search box (top-right of Firefox window) and select the option at the bottom: "Manage Search Engines". There you can change back your default search engine to any of the others currently installed. Google is usually the first one on the list.

### Post by Shadius on 2012-08-12
You can also go to *Tools>Add-ons*, that'll open up the **Add-ons Manager** window, from there, with the *Extensions* category selected, find the appropriate add-on and remove/disable it.
