### Post by Technoviking on 2011-04-29
Here how to customize Unity.

1. Install CCSM
```
sudo apt-get install compizconfig-settings-manager
```

2. Hit *Alt-F2* and type *about:config*

3. Click the Unity icon and configurations for Unity should come.

I really like turning off autohide and shrinking my launcher to 32 pixels.

Cheers,
