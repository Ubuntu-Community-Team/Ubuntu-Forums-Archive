### Post by MEUtnubu on 2012-01-03
I'm also new to Ubuntu.
To manipulate the copy/paste in terminal, you need to install the "gpm" program. The General Purpose Mouse (gpm) provides an interface for that purpose.
To install, open the Ubuntu Software Center and on the top right-hand corner do a search for gpm. Once found, click Install to proceed.

The mouse features are great:
- to Copy, drag a text with left mouse
- to Paste, use the middle-mouse click

Have fun.
