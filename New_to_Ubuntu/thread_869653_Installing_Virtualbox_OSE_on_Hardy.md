### Post by sdennie on 2008-07-25
What specific problems are you having?

For the OSE version you should be able to install with:

```

sudo apt-get install virtualbox-ose virtualbox-ose-modules-generic

```

You can also download the more up to date and free (as in beer, not as in speech) version from the website and install it: [www.virtualbox.org](www.virtualbox.org).
