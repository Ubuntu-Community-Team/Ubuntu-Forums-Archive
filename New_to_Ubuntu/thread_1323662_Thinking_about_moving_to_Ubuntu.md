### Post by anewguy on 2009-11-12
You may want to search this forum, and the net in general (try "ubuntu 9.10 dual boot with vista") for dual-booting Vista and Ubuntu.  You already have an advantage over some - you already have Windows installed.  It is much easier to make a dual-boot system with Windows installed first.

*IF* you are familiar with what a partition is on your hard drive, basically you are going to make the Vista partition smaller, then add the swap and at least one more (maybe two if you want your home folder to be preserved if you reinstall) partitions for Ubuntu and install.  The LiveCD actually will walk you through the partitioning if you are careful, but as with anything - backup your important things first!.

If you go with dual-boot, it will let you boot to Vista when/if you need to, and also boot Ubuntu the rest of the time.

If you have an actual Windows Vista installation CD, you may want to back up everything you need and then let the installation from the LiveCD use the entire disk.  After you have Ubuntu installed and what ever extra drivers, etc., you may end up needing, you can always run a virtual machine (with virtualbox for example) in Ubuntu and actually install and run Vista in it while Ubuntu is still running.

I guess one thing I would ask is what are the specs of your DV 2000 - what processor, how much memory, video chipset, etc.?  that does make a difference in the install method you should follow.
