### Post by jgoguen on 2008-12-08
If you go to **System -> Preferences -> Sessions**, that will bring up the Session Preferences window.  In the [i]Startup Programs[i] tab, click the **Add** button.  Enter a name (any name you want, this is for you to identify it later) in the **Name** box, the full path to the program in the **Command** box, and optionally enter a comment in the **Comment** box (again, anything you want here, even nothing).  Press **Add** and that program will be started as soon as you log in.  Repeat for any other programs you want started automatically. Later on, if you decide you don't want them to start, go back to the Session Preferences window and uncheck the box beside the program you don't want to start.  If you're really sure, you can also select the item and press the **Remove** button.

### Post by shazbut on 2008-12-08
Or you could boot a clean session, open everything you want to start automatically, then go to system -> preferences -> sessions, and on the 3rd tab press the 'remember currently running applications' button.
