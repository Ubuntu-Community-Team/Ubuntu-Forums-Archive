### Post by Mark Phelps on 2011-08-14
First, make sure you use only the Win7 Disk Management tool to shrink Win7.  Do not use the Ubuntu installer because doing so risks corrupting the Win7 OS partition and rendering it unbootable.

Second, if you have an optical drive, then use the Win7 Backup feature to create and burn a Win7 Repair CD.  This will come in handy later, should you need to repair the Win7 boot loader.

And finally, if you already have four partitions on your drive, do NOT attempt to create a fifth. Doing so will force the conversion of your Win7 partitions into Dynamic Disks -- giving you a whole new set of problems to deal with.
