### Post by mewithafez on 2008-09-10
Right-click on the desktop and you will get something that says "Change System Background". That pulls up a tabbed preferences thing you can also find under system-preferences-appearance in the menu on the top panel. 

The tab farthest to the left is all of the themes you have, and it has a button that says "Install". You can install any theme like one from gnome-look.org or whatever that comes at as .tar.gz, so you could install Elegant Brit by selecting elegant-brit.tar.gz. Alternatively, you can just drag that file to the window with all of the themes and it will install it automatically.

Let me know how this works!
