### Post by 3rdalbum on 2009-03-13
It's generally better to install the latest version of Ubuntu and update before its end-of-life (usually when the next version comes out). This is because you get new features, new software, and more hardware compatibility.

Software in the Universe repository rarely or never gets security updates pushed out, so software in Intrepid Universe often has more recent security fixes than Hardy Universe.

If Intrepid isn't doing it for you, or if you need 3 years between dist-upgrades on a workstation or 5 years between dist-upgrades on a server, then Hardy is where you should go.

### Post by mcduck on 2009-03-13
For normal versions the support time is 18 months, LTS versions get 3 years of desktop support and 5 years server support.

For normal users I'd recommend using the latest available version, but LTS versions are important for business use where longer support times are needed as upgrading every computer every 18 months is not an option.

Apart from the support time there isn't any difference, although LTS versions are often a bit more stable and conservative when it comes to adding new features and things. (Non-LTS versions are in no way beta releases, but since they are less likely to be used on corporate environments they can be a bit more "experimental" than the LTS versions)

### Post by insane_alien on 2009-03-13
LTS versions are intended for use by business where they dont want to have to upgrade the software very often as that can cause all sorts of hassle from bugs to people not getting a new way of doing things. as such, these versions are kept afloat with a few years of security updates and bugfixes but no new features or updated versions of programs.

non-LTS versions are intended for more casual use, those who want the latest and greatest. this is not to say that they are beta, unstable or buggy as an ants nest. they are stable secure and the only difference is that they will not be getting the extended support and updates. after 18 months support and updates are terminated.

You can go with either the LTS or non-LTS it depends on what you value the most.

Do you want something that will sit on your computer for years without requiring any major upgrades or changes or do you want something with the latest and greatest?

If you choose the former, stick to LTS releases.
If you choose the latter, stick with the latest release regardless of whether it is LTS or not.
