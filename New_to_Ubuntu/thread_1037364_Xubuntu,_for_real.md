### Post by jimmy the saint on 2009-01-11
Ubuntu is the most polished of the *buntus.  If you have older hardware, Xubuntu is definitely a solid distro though.  You will miss certain functionality points like drag-and-drop desktop support.  As far as start up tones, if memory serves, Xubuntu has none.  It's startup is very basic, as is most everything in the distro.  If your hardware functions fast enough with Ubuntu, I would recommend sticking with it, especially if you are new to linux, as the configurations for most aspects of the OS do not require editing files manually.  Xubuntu is a great OS, but it still has a lot of situations where there is not a simple gui to perform configuration tasks (menus for example).

### Post by jimmy the saint on 2009-01-11
The minimal installation is one way to go.  The installed applications really don't take up that much room.  Ubuntu is not like an OEM windows installation with a bunch of crapware installed as a "courtesy." ### Post by Xiong Chiamiov on 2009-01-11
> **multiman said:**
> Xiong Chiamiov
Does the minimal CD image installs the themes and icons along with start-up sounds and etc. but just without any application?
Pardon, the minimal cd just downloads the packages from the net directly, rather than having them installed on the cd.  If you use the alternate install cd, then you will have the option of choosing which packages to install.

It's important to understand that nothing will be different about the system afterwards.  You can install any of the same packages as you could otherwise, so, yes, you can have the same sounds and color themes.
