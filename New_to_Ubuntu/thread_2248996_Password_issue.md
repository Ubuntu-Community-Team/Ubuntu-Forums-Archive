### Post by yancek on 2014-10-18
> Installed lubuntu on my sons laptop and now whenever try use the laptop and we enter the password which he thinks he registered

There is no "registering" of passwords, maybe this is just a question of terminology.  The standard method for root/administrative use is with the sudo command.  When you install Lubuntu, you are required to create one user and set a password for that user.  That user will then have root/admin privileges by using sudo.  If it is the case that the password has been forgotten or isn't working for some reason, you can follow the steps at the site below to change it:

[http://askubuntu.com/questions/24006/how-do-i-reset-a-lost-administrative-password](http://askubuntu.com/questions/24006/how-do-i-reset-a-lost-administrative-password)
