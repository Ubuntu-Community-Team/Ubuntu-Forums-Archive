### Post by ankspo71 on 2009-10-10
Hi, to make a shortcut to a drive, all you need to do it this.

Right click on the desktop and click on "create launcher".
Give the shortcut a name
type this command in the command box:```
 nautilus /media/(windows drive)
```
Select an icon if you want to.
Then click ok.
Note: I don't know if it will ask you to mount the drive or not, but that is how you create shortcuts to folders.... [ filemanager + /path/ ]
