### Post by roquefilipe on 2009-03-03
well, from the command line you can use the fowling option:

```
totem --no-existing-session 
```

---

### Post by powel212 on 2009-03-03
That works. Thanks.

I put the command into a launcher on the task-bar. One click and another player is open.
