Just keep in mind, Linux tends to be modular.

In windows you have monolithic applications that do everything, in Linux there tends to be smaller more specialized applications.

Assuming you have found an application you like more then K3b, go ahead and remove it. Just watch the "dependencies" and make sure you are not removing other applications along with it (unlikely with K3b on Ubuntu, but something to watch for with other applications).
