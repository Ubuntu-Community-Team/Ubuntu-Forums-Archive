### Post by NikTh on 2012-10-20
Hi , 
you can go to Software Sources and disable them from there. 

How to reach the software sources window ? 

Open the Ubuntu Software Center and then click Edit > Software Sources > Other software. 

Then click which PPA you want to remove and click [remove]. 

At the end you can close the software center and open the update-manager (so software sources will regenerated = updated). I'm not sure if USC updates the sources.
