### Post by darolu on 2011-03-23
Press Alt+F2, a window will pop up, type "crossfire" in, the program should auto-complete it for you, after that press enter and it should run.

You can fix the menu entry by right clicking the Applications bar and going to Edit, just navigate to the Games section, your game should be there, make sure it is enable (disabling and re-enabling it may work), if it is not there just create a launcher from there, I don't think you'd have trouble doing it, *if else*, just post again.
