If is Flash Chess the game you want to play , see if adobe-flashplayer is installed. I used Firefox too. 

Have you installed the restricted-extras package ? (flash player included). 

Open a terminal (CTRL+ALT+T) and give this command
```
sudo apt-get install ubuntu-restricted-extras
``` 
write your password carefully (because nothing will appear , like you don't write anything) and hit [Enter]. 
After installation finish , re-load Firefox and try again.

### Post by ImmigrantUS on 2012-12-20
For chess on Linux:

1) Free Internet Chess Server - [http://www.freechess.org](http://www.freechess.org)
Register there, download one of the graphical interfaces, or get Raptor chess interface - [https://code.google.com/p/raptor-chess-interface](https://code.google.com/p/raptor-chess-interface), learn how to use it and play;

2) Play on [http://www.chess.com](http://www.chess.com), no need to download anything;

3) Many other variations.

Yahoo chess is the last one to go to play chess, not good for any more or less serious play.
