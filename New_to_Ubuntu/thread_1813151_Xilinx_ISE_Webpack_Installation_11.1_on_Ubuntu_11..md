I have followed the tips as listed in a pdf tutorial from "Hardware Description Language" subject taught at Eastern Washington University. The tutorial is listed here: 

[COLOR=Red][http://web.ewu.edu/groups/technology/Claudio/ee36010/Cad/Steps_to_Install_Xilinx_11.1_on_Ubuntu_9.10.pdf](http://web.ewu.edu/groups/technology/Claudio/ee36010/Cad/Steps_to_Install_Xilinx_11.1_on_Ubuntu_9.10.pdf)[/COLOR]

A couple of minor issues I faced during the install: 
1. *License File*: At the end of the Xilinx Webpack installation, the installer popped up an error when trying to log on to Xilinx website to obtain the license file(.lic). I manually completed this step by generating a license file from Xilinx here: [COLOR=Red][www.xilinx.com/getlicense]("http://ubuntuforums.org/www.xilinx.com/getlicense")[/COLOR] . The license file is emailed and can be uploaded either at the end of xilinx installation or when all the steps have been completed in the above pdf tutorial.

2. *Failed to Compile C Code during "Behavioural Simulation":* An identical issue is discussed in the Xilinx forum link below: [COLOR=Red][http://forums.xilinx.com/t5/Synthesis/Failed-to-Compile-generated-C-code-Linux-64/td-p/49787](http://forums.xilinx.com/t5/Synthesis/Failed-to-Compile-generated-C-code-Linux-64/td-p/49787)[/COLOR]
