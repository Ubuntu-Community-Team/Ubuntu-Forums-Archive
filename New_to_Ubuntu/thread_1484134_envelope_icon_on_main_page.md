### Post by howefield on 2010-05-15
> **jackj said:**
> Also, I was talking about the little envelope icon on the top of the main menu, and I don't know what it's called.

In desktop versions of Ubuntu up to 9.10 there is a grey envelope icon next to the help icon (near the left hand side of the top panel).

In 10.04, it isn't there by default, but there is a monochrome envelope icon near the right hand side of the top panel, this is called the memenu.
