### Post by Lisiano on 2012-10-20
After a bit of thought, I remembered the following:
[LIST]
[*]OEM key (COA sticker) will work with a OEM disk
[*]OEM key (COA sticker) will _**not**_ work with a retail disk
[*]Retail key (CD/DVD sticker) will _**not**_ work with a OEM disk
[*]Retail key (CD/DVD sticker) will work with a retail disk
[/LIST]
The following is true at least for Windows 7, it might be also true for Windows XP and earlier releases. In short, if you only have a COA sticker, you might need to get a OEM version of the disk, those are not sold in stores.
