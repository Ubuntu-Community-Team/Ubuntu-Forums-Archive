### Post by steveydoteu on 2008-11-04
When booting, wait for it to get to the GRUB loading stage and press escape to show the menu. Select the second option, which should be recovery mode.

---

### Post by Peter09 on 2008-11-04
When you see the Grub prompt (which appears for a few seconds after the Bios text) hit the ESCAPE key. That will stop the boot process and show you a set uf boot options.
Go down (normally only one line) and select the recovery option. Hit enter and you will boot to recovery mode.
