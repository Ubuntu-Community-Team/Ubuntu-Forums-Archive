### Post by django0909 on 2008-08-26
The 'part' file is the partly downloaded ISO. It should disappear once the ISO is fully downloaded. The .iso is the one you want to be burning.

edit: By the sounds of it you tried to burn a partly downloaded iso to disk....Make sure the .iso is fully downloaded!

### Post by forger on 2008-08-26
Which operating system are you using to download it?

It means that it hasn't arrived correctly. I think mozilla browsers (firefox?) download a file as a .part file, then rename it properly when it's done. Yours probably hasn't finished downloading.

I'd suggest to check out Tools > Downloads and right-click > retry if there's anything there.
Otherwise, you could [download it as a torrent]("http://releases.ubuntu.com/hardy/ubuntu-8.04.1-desktop-amd64.iso.torrent") and use [utorrent]("http://www.utorrent.com") (windows) or transmission or deluge (if you have ubuntu or any other operating system) to download it correctly :)
Torrent programs like deluge and utorrent have hash checking, which can check if the files were downloaded successfully
