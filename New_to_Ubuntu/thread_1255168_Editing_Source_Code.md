### Post by ad_267 on 2009-09-01
Perhaps you could do a little bit more research into exactly what it is you want to do.

Do you want to edit the source code of the Linux kernel and compile your own kernel, or edit the source code of an individual application like Firefox or Nautilus?

Or is it that you want to create your own customised distribution with your branding?

You can use "apt-get source package_name", where package_name is the name of a package, to download the source code for that package.
