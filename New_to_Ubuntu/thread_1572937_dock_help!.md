### Post by Legendary_Bibo on 2010-09-12
Well you can try logging out then back in.

Restarting compiz by one of two ways, the first one being to right click on the compiz fusion system tray icon if you have it installed and clicking on Reload window manager.

If you don't then you can go into the terminal and enter this:
```
compiz --replace
```

Do you want the dock to be gone, or just auto hide?

### Post by Legendary_Bibo on 2010-09-12
> **thatguy93 said:**
> i want to use avant window manager, so i want to do whatever works better for it. you tell me lol

Oh well then you'll probably want to delete it, first make note of whatever applets are on the bottom then add them to the top panel so you don't forget, or make a note of them.

Now right click on the bottom panel and click "Delete this Panel"

Your top panel may be a bit overcrowded so I recommend Gnomenu, it's an applet that you can use to replace that "Applications, Places, System" Menu, but as an eye candy rich menu with a smaller button. You can find several themes of it like this [one](http://gnome-look.org/content/show.php/KDE+Gemstones?content=128264).

If you decide to use Gnomenu to fix the overcrowding issue, then I can help you with installing themes if you need.
