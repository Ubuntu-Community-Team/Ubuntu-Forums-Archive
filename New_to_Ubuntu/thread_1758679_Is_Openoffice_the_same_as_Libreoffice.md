### Post by dwasifar on 2011-05-14
LibreOffice is a "fork" of OpenOffice.org.  A fork means that someone (in this case, the Document Foundation) has started a new variant from the same source code as the original project.

As other poster has already mentioned, this happened because Oracle got control of the OpenOffice project.  Oracle was expected to behave contrary to the ideals of free and open source software, so the Document Foundation forked the project to LibreOffice and most major Linux distributions have switched to that.

LibreOffice is now one development cycle ahead of OpenOffice.org and has a couple of technical advantages, notably speed.  In the wake of the mass abandonment of OpenOffice.org, Oracle has conceded defeat on its plans for the project and has invited open source developer involvement, but it is probably too late for that to succeed now.  

If the point of your question is deciding whether to use LibreOffice or OpenOffice, you'll be better off with LibreOffice.
