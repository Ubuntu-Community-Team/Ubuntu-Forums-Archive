### Post by Paqman on 2009-12-06
> **smith_jc said:**
> Can anyone tell me if its possible to re-write the hard drive and install Ubuntu without being a programming expert.

Yep, during the install process it'll ask you if you want to install using the whole hard drive for Ubuntu. Say "yes", job done.

You'll have to set up a user account during the install. Don't forget to take note of the username and password of that account. By default any further accounts you create won't have admin rights (although it's easy to give them those rights if you're logged into the admin one)

The other option would be to use the "OEM install" method on the Alternate CD. You might want to try installing Ubuntu yourself and getting a little familiar with it before trying that one though.

### Post by ScottinSoCal on 2009-12-06
> **smith_jc said:**
> thanks but when i put the disk in it does not run at all.  whats wrong there?

You need to boot off the CD. If you're tried shutting down with the CD in the drive, and then starting back up and it doesn't work, there are three possibilities:

1) The CD was burned badly. Burn a new one, using the ISO image.
2) The BIOS in the laptop isn't set to boot off the CD before the hard drive. Go into setup and change it, then reboot again.
3) It's a very old laptop that doesn't support booting off the CD. That gets you into a whole new area of issues, and I'm not sure you can get there from here.
