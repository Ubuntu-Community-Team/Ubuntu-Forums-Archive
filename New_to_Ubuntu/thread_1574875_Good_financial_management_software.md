### Post by lkraemer on 2010-12-27
Just check out Crossover by Codeweavers.  After Installing (Purchase)
you can install Quicken in a Bottle (Environment) and then install Quicken.
You can also find a lot of other applications Crossover
supports.  Your next decision is STANDARD or PROFESSIONAL Version.

Then just save your Quicken file to a USB Flash Drive from Windows, and
import the file to your Linux machine with Quicken.  Zit Zot!  Easy.


Another nice package that imports the Quicken Data files is MoneyDance.
It requires Java Runtime Environment be installed (JRE).  It looks and
acts a lot like Quicken with a few nice features Quicken doesn't have.
