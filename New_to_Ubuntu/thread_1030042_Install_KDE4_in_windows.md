### Post by northern lights on 2009-01-04
> **retskrad said:**
> Also, how come you can have kde programs in windows?
You can't run KDE applications in Windows. There simply are some KDE native apps that have been transformed to work in Windows environments. It's not unmodified code, though.

> **retskrad said:**
> Is there a way to install KDE in windows`? I don't mean KDE programs, like amroak, I mean KDE in Kubuntu, like gNOME in Ubuntu.

KDE and gnome are both *nix desktop environments. The only difference between Ubuntu and Kubuntu is the desktop environment. You can run any Linux distribution with either. Windows, however, is a completely different operating system. If you want KDE, you have to boot *nix.
