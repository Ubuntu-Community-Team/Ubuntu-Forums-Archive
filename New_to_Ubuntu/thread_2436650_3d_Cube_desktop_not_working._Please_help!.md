### Post by CatKiller on 2020-02-10
Are you even using Compiz? It was part of Unity, but Ubuntu switched to Gnome instead of Unity for 18.04. You can still use Unity in 18.04, but it's something that you'd have to actually do and you don't mention having done so.

As an alternative, KDE has its own desktop effects, including the cube, without needing anything else, and I think it's a much nicer experience - especially for new users - than using Gnome.

### Post by deadflowr on 2020-02-10
Install a desktop that can utilize compiz.
Ubuntu's Gnome desktop cannot.

I'd suggest installing the unity desktop.
(Another desktop that works well is xfce (xubuntu))
**Edit:** Or Mate (ubuntu-mate) works very well.

Once the alternate desktop is installed, switch to it at login.

About unity and other desktops, see: [https://ubuntuforums.org/showthread.php?t=2415676&p=13848097#post13848097]("https://ubuntuforums.org/showthread.php?t=2415676&p=13848097#post13848097")
