### Post by sdennie on 2008-06-16
Maybe your BIOS has a default setting that Windows is overriding.  You should be able to adjust the brightness using the Fn key and the brightness up/down keys on your keyboard (it differs from laptop to laptop).

As for the free space, it's possible that Ubuntu is tell you the largest free contiguous space on the disk.  I would recommend doing a disk defrag from Windows before installing Ubuntu.
