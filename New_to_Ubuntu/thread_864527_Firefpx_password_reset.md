It would seem that Firefox has remembered a miss typed password for one of my frequently visited websites. No problem accessing the site as i know the pass but it is annoying. I looked through Firefox's preferences but they only had cookie deletion and a generel save password/ don't have password check box. Help guides for older versions of Firefox say to go to Tools> Options or Tools> Clear private data. ### Post by cpetercarter on 2008-07-19
Edit > Preferences > Security > Saved Passwords

Highlight the password you want to delete, click remove.
