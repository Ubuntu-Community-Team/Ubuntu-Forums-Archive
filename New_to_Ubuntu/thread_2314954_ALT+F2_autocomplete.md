### Post by deadflowr on 2016-02-26
alt+f2 is not the dash, and not even close.
alt+f2 opens what it called the run dialog, or runner (I think that's what unity calls it)

You cannot open alt+f2 with a mouse, since there is no icon launcher to open it with.
Just like you cannot open the hud with a mouse. 
Only the dash can be opened with a mouse click.

The dash, the hud, and the runner/run dialog are 3 different things that all use the same look, but all 3 differ in what they are
and how they open.

I do not know if you can add the tab function to the runner/run dialog, as tab already has a set keybinding that allows you to navigate selections.

I have not seen anywhere that you can change that outside of changing it in source.
