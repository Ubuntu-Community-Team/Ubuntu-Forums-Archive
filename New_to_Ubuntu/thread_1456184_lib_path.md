### Post by gmargo on 2010-04-16
/usr/local/lib should already be in your "library search path".  Where is this path stored?

When a program executes, it uses "**ld.so**" (see **ld.so( 8 )** man page) to find the shared libraries.  **ld.so** uses a pre-generated cache file **/etc/ld.so.cache** to find these libraries. The cache file is generated by the **"ldconfig"** (see **ldconfig( 8 )** man page) command, which is configured by the files **/etc/ld.so.conf** and **/etc/ld.so.conf.d/***.  Typically, the **/etc/ld.so.conf.d/libc.conf** file contains the line **/usr/local/lib**, which means that libraries in **/usr/local/lib** will be found without any further work on your part (assuming **ldconfig** was run after the library files were installed.)
