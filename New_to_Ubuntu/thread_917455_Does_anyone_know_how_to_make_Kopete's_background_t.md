### Post by StephanG on 2010-08-09
Hey :)

I'm not sure if this is what you're looking for, but I use Bespin. (See the pics for what my desktop looks like).

Its a widget style and window decoration for KDE.  You're going to have to compile the latest version yourself, but there should be several howto's scattered across the internet.

The blurring (highly recommended) needs KDE SC 4.5 or greater, so unless you're comfortable using a Release Candidate I would suggest waiting for 4.5 to be released.  (Its being released tomorrow, and kubuntu packages should follow soon after.)

You should also know that if you don't like Bespin, you can use Qtcurve instead of Bespin.  It also supports transparency since version 1.5.1.  I find Bespin more user friendly, but Qtcurve supports GTK apps.

In case you want to try it, I'm including the .deb packages for Qtcurve.  I can't quite remember where I dug them up.

Anyway, I hope that helps. ^_^

P.S.  If you want to use Qtcurve, I suggest looking for a good qtcurve theme.  Its a lot easier to load preconfigured settings than it is to set everything yourself.
