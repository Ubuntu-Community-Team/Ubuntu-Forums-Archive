### Post by dannyboy79 on 2009-02-11
I am still using Feisty so don't know if this applies but when you go under System>Preferences>Sessions, then you click on the application that is being started up automatically, you would click on edit button located down the right side of the Sessions dialog box, another dialog box appears with the command info. That's the special command that's being run upon startup. So you could edit your menu using that same command. Hope that helps ya.
