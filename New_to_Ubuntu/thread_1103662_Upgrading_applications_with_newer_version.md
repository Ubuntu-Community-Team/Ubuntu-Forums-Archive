### Post by Redache on 2009-03-22
It depends whether or not the new version has a feature you want. It's far easier to wait until Ubuntu catches up rather than compiling it yourself, which can cause problems for an inexperienced Linux user.

There are sometimes repositories that may have a newer version compiled explicitly for Ubuntu, do a quick Google search for "Banshee 1.4.3 Ubuntu Package" and see what comes up.

You could also check the Banshee project website for a .deb file, which will install in Ubuntu (as Ubuntu is a derivative of Debian and a .deb file is a Debian binary file). 

The first thing I recommend is reading the 1.4.3 release notes and see if any of the added functionality is something you actually want and whether there are any stability improvements you want. if there is no real world benefit for you then I'd say if it ain't broke, don't fix it.

possible, go to the synaptic packet program.
When it is through getting all sources, tell it to 'mark all updates'
It will then search, and see if there are any packets that you already have installed that have updates to them
It is possible that it will list Banshee
if so, tell it to apply.
tell it yes you are sure
it should download the updates, and install them for you.

if not, you may have to tell the program, to use all third party and open source repositories for software.
