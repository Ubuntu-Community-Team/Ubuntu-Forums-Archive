### Post by amanchesterman on 2015-05-03
It would help if you tell us which browser you are using to access the Admiral website.

My car insurance documents are provided online in PDF format, which is one of the most common formats for that kind of thing. If you are using Firefox (which is the default browser for Ubuntu) then the document **should** display in the built-in PDF reader. If it doesn't there is a page here about troubleshooting:
[https://support.mozilla.org/en-US/kb/view-pdf-files-firefox-without-downloading-them](https://support.mozilla.org/en-US/kb/view-pdf-files-firefox-without-downloading-them)
You will see near the bottom that the plugin will not work if Javascript is blocked in your browser. That might be worth investigating further, rather than Flash?
