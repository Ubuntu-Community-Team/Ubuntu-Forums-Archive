### Post by nhasian on 2009-05-15
i tried the user agent switcher on abc.com  unfortunately the default values didnt work so i imported the agent data from my firefox on windows xp which got me farther.  you can create a textfile with this xml data and then import it into the user agent switcher:

```
<useragent description="Firefox (Windows XP)" useragent="Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.0.10) Gecko/2009042316 Firefox/3.0.10 (.NET CLR 3.5.30729)" appcodename="Mozilla" appname="Netscape" appversion="5.0 (Windows; en-US)" platform="Win32" vendor="" vendorsub=""/>
```

With the user agent set to that i was able to download the firefox extention but received an error message of:

> "Move Media Player" could not be installed because it is not compatible with your Firefox build type (Linux_x86-gcc3). Please contact the author of this item about the problem.
