### Post by formaldehyde_spoon on 2010-09-30
I think the OP means he wants icons only, for open windows.

 @OP, install DockBarX: [http://gnome-look.org/content/show.php/DockbarX?content=101604](http://gnome-look.org/content/show.php/DockbarX?content=101604)
[IMG]http://gnome-look.org/CONTENT/content-pre2/101604-2.png[/IMG]  
   How to install:
- type in a terminal: **sudo add-apt-repository ppa:dockbar-main/ppa**
- also in terminal: **sudo apt-get update && sudo apt-get install dockbarx**
- Then right click on a panel, add to panel, dockbarX. (and delete the original window list).
