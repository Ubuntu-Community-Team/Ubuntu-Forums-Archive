### Post by yc2 on 2010-07-13
If I doubleclick an executable in Gnome desktop I get four choices, the rightmost is "run" and the leftmost is "run in terminal".

Please tell me the difference between these alternatives.

Thanks.

(My translation of the alternatives from the Swedish edition.)

---

### Post by Mr. Gnome on 2010-07-13
*Run in terminal* - open a new terminal window and execute the given application / script so that you could see the output.
*Run *- execute the given application / script in background.
