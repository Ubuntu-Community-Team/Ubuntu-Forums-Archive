### Post by 3rdalbum on 2010-05-17
By default, Gnome does not trust any '.desktop' file. It depends on you to tell it whether you trust this file.

So if you trust it, then tell Gnome that you want it marked as trusted.

This feature is to prevent malicious software that you accidentally download from being able to be executed without your approval.
