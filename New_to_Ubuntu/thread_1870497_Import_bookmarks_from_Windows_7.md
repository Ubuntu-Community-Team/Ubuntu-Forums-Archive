### Post by oldfred on 2011-10-27
With XP I just put the profile in a shared NTFS partition and in both XP and Ubuntu changed profile.ini to point to the new location. Then both systems are identical. Some minor issues when booting as some add ins are not compatible but never had an issue using system even when versions were not fully synced.

This is on Thunderbird but the setup of profile.ini is the same. I also move my entire profiles to my portable when traveling and copy back when I return without any issues.
