### Post by gerowen on 2013-04-04
If you ever exported your bookmarks to an html file with IE, then you can import that file in Firefox.  Just click "Bookmarks", then "Show All Bookmarks".  You'll then have a button that says, "Import and Backup" with several options to choose from.  The "Import Data from Another Browser" option only works with browsers currently installed in Ubuntu, so you need to access IE and use it to export your favorites to an html file.  I've attached some screenshots of the process of importing them in Firefox on Ubuntu to this post.  Just click them to see larger versions.

Here's some official Microsoft help on how to view, organize, export and import your favorites.  Just select your IE version using the drop down on the top right and it will customize the instructions to fit what you will see on the Windows laptop.
[http://windows.microsoft.com/en-us/internet-explorer/add-view-organize-favorites#ie=ie-10-win-7](http://windows.microsoft.com/en-us/internet-explorer/add-view-organize-favorites#ie=ie-10-win-7)
