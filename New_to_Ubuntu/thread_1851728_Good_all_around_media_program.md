### Post by SeijiSensei on 2011-09-29
One of the nice features of the KDE environment (in Kubuntu) is the way it handles CDs.  After the CD is loaded, KDE displays the disk contents as a set of folders with different encodings.  You can then drag-and-drop the items from the FLAC folder onto the hard drive to rip the files into that format.  Other options include Ogg and MP3 if you have the restricted-extras packages installed.  You also have the option of ripping individual tracks or the entire disc as a single file.
