### Post by Zimmer on 2009-01-28
Suggestion:
Use the application you currently have the addresses in to create a vcard (may be under an export function).
Use Evolution's 'Import' function to get the vcard file you have created...

But, re reading your post it appears you already have mails from people in Evo, but have not added them to your contact list.

The only immediate way I can suggest for that is to sort your messages by sender , open a message and use the Message >'Add Sender to Contacts' function for each sender you are missing in the Contacts file.
Check Evolution's Mail preferences to enable automatic Contact creation when you reply to a mail for future reference.
