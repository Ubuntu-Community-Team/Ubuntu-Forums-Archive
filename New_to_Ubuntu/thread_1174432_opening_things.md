### Post by 73ckn797 on 2009-05-31
If you right click, select properties/Open With you can change the default program.

Also: in Nautilus Preferences/Behavior check off "View executable files when they are opened" and see what that will do in addition to the first suggestion above.

### Post by anarchyinc on 2009-05-31
Why so many links to articles and how-tos?

Just right-click the file,
select properties,
then change the drop down menu option listed under "[SIZE=3]**opens with**[/SIZE]" to the new program you want to always open that file by default.

Once finished you might want to go ahead and reformat your computer, I'm not even going to tell you to go back to windows since they do it the **exact same way**.. unplug it and put it in a closet.
