edit: Now I think I figured out what's this about.. I suppose you found the command that allows you to set the theme and background for the login screen, but after rnnning it two accessibility icons appeared in the Notification Area. One disappeared simply by logging out and back again, but the other one is still there?

Easy fix. :) Go to System/Preferences/Keyboard, and on the Accessibility-tab disable "Accessibility features can be toggled with keybaord shortcuts". The icon will disappear.
