It's sometimes a good idea to have a play with the "Live CD" feature of Ubuntu first which lets you try Ubuntu without actually installing it, to get an idea if it works on your equipment and if you like it. It won't run quite a slick as a proper install, but it should give you at least some idea.

Once you're ready to install, you might want to start by considering whether you want to install Ubuntu "within" windows, or have Ubuntu in its own partition (known as a "dual boot"), each of which can be used to have both Windows and Ubuntu on your system. This can be a good thing while you're finding your way around, particularly if you're not sure if you want to get rid of Windows just yet or if something goes wrong (as much as we hate to admit it!) with Ubuntu.

"Within windows" (a.k.a. "wubi") has the advantage that you can use the Windows control panel's "add/remove programs" feature to remove Ubuntu if something goes wrong or, heaven forbid! you change your mind. The down side is that if something goes wrong with Windows, it can *sometimes* interfere with your ability to run Ubuntu.

The "own partition" partition approach has the advantage that if Windows "goes south" for some reason, you can, with a few minutes tinkering, keep Ubuntu working.


EDIT: **Very important:** before installing, be sure to make a backup of your important data. As much as we hate to admit it, things can sometimes go wrong.
