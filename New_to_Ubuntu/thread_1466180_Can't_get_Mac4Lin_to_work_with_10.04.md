### Post by Silent Warrior on 2010-04-30
I think it was recently updated. Make sure you have the latest version, and then extract the files manually to the .themes-directory if Appearance won't cooperate.

(Pay close attention to the file-extension. Appearance only handles .tar.gz, while uploads can be in tar.bz2, .zip, .7z, .rar...)
