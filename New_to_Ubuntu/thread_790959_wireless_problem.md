### Post by cprofitt on 2008-05-11
Gnome:

You should get the network manager icon in the upper right panel. If you click on it you should get the options to connect to any 'visible' SSIDs and the option to 'Connect to Other Wireless Network...' to join a hidden SSID. You can also create an ad-hoc by using the 'Create New Wireless Network...' option

If you right click you can edit SSIDs that were previously joined.

KDE:

The last time I used KDE I used network manager as well and the icon was in the lower right had panel (KDE usually does not have an upper right panel).

I hope that helps.
