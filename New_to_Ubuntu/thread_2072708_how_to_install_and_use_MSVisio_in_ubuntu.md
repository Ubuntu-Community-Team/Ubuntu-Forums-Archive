### Post by Cheesemill on 2012-10-18
You may have some luck trying to install Visio using Wine.

However according to the Wine website the only version worth attempting is Visio 2007 Professional with its Gold rating.
[http://appdb.winehq.org/objectManager.php?sClass=application&iId=119](http://appdb.winehq.org/objectManager.php?sClass=application&iId=119)

If you need Visio to work perfectly or need to run a later version then your only option is to run it in a Windows VM.

While cheesemill is correct about using Wine, if you really need to use Visio, in this case, trying to get it to work with Wine is largely a waste of time.  The bronze rating means it's all but worthless; the silver rating means one or more key functions don't work.

As cheesemill said, a VM installed with Windows is really the only way that Visio is going to be useful for you -- and in that case, you will need to install MS Windows from scratch to use it.
