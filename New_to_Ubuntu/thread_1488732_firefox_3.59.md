### Post by edgy360 on 2010-05-20
You would need to use an add-on to get the FLV (flash video) file using a add-on. Searching on [http://addons.mozilla.org/](http://addons.mozilla.org/) should find you one somewhere.

The .swf file is the movie player and does not have the video file to play.

### Post by mikewhatever on 2010-05-20
