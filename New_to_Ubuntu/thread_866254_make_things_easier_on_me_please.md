### Post by issih on 2008-07-21
Really not sure what you are trying to achieve...

When you install Hardy from the cd, all the program dependancies are already met for the OS and the programs installed along with it..

If you later install anything new via the package manager synaptic, then that automatically handles dependancies for you and downloads any that you are missing.

If you are trying to install a .deb file downloaded from a website, then you might have dependancy issues ou have to manage by hand. You really should use synaptic to see if there is a version in the repositories, or even see if there is a repository you could add to your sources list rather than just doing this however.

You may also have dependancy issues if compiling things from source, again there is no real way round this.

In the main, however, dependancies are handled automatically in ubuntu, provided you use the package management system.

Hope that helps
