### Post by CharmCityCrab on 2008-04-29
GRUB displays four Linux kernels to me that are related to Ubuntu 8.04.  Two are for recovery.  The two non-recovery boot options end in .16 and .14.  What's the difference?  Which one is generally preferable to use?

---

### Post by scragar on 2008-04-29
.16 is the latest, .14 is the version before it(from gutsy, 7.10) to be used if for some reason the .16 is corrupt(where you then reinstall the .16 kernel again :P)

Use the latest whenever you can, since it often offers support for more hardware and potential bug/security holes.
