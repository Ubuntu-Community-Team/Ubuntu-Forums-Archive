### Post by spacemonkey211 on 2010-06-02
andLinux works great.  I use it all the time to surf the web.  The only problem that I have is that the interface is a little slow at times since all the windows are being rendered through an artificial network driver.  The other thing is it is based on Ubuntu 9.04 which only has Firefox 3.0 installed.  The upside is no need to worry about viruses when you are surfing.

You can also install anything in the Ubuntu repositories or even compile anything.  It works just like having a native linux distro.
