When you set the system language to Belgium/French, Ubuntu notifies Firefox so that it receives by default pages in French, if available from the websites.
See in Firefox, at Edit/Preferences/Content/Languages/
You can edit manually the above list in put as first choice to receive pages in the English language.

If you want to change all your system to English, you need to use the Language Support setting mention above, then logout, select the English language and log in again.

However, if you only want to view Google search pages in English, I would suggest to simply visit with Firefox the page [www.google.com](www.google.com) and click on the link 'Google.com in English'. This will set a cookie so that you get the English Google when searching.
