### Post by ld114 on 2012-04-11
Hi

I have been trying out 12.04 on my netbook since the beta 2 was released. You can install "gnome-panel" from the Ubuntu Software Centre and this delivers extra login options including Gnome Classic (click on the Ubuntu logo in the login box). 

The 'Gnome Classic without effects' one is the best and delivers a classic gnome 2 desktop - with one or two variations. Amendments to the panel are done by pressing ALT-right click. 

Launcher icons are installed by ALT-right click/add to panel/Application Launcher/Forward - which provides a list of Apps to choose from. 

In Applications/System Tools/Preferences there is a Main Menu editing option too.

So it is pretty much there if you want it. I am getting used to Unity, even though I return to Gnome Classic every so often....
