### Post by Mornedhel on 2009-05-26
Yes, it's the "Scale" plugin, installed by default with Compiz.

I don't know if it's activated by default or what the keybindings are, as I usually customize mine.

Install compizconfig-settings-manager, click on System > Preferences > CompizConfig Settings Manager, scroll to the bottom, click on Scale.
