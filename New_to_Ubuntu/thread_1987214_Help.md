### Post by 3rdalbum on 2012-05-26
You haven't given precise information about exactly what components are in your computer. Some components don't give a very good experience in Linux - for example, older ATI graphics cards are not ideal, and VIA and SiS graphics are very poorly supported on Linux. Some wireless cards can be an issue as well.

However, most components you'll find in a computer WILL work. Ubuntu can even be run from the CD as a kind of trial run, so you can check on wireless, sound, graphics support etc*. Running Ubuntu from the CD will NOT touch your hard drive at all until you double-click the "Install Ubuntu" icon on the desktop, so **there's no risk** to downloading and running the Ubuntu CD.

*Note: Ubuntu does not come with many codecs for video playback, so don't be surprised if you can't play DVDs or watch AVIs from the Ubuntu CD. However, support for these is easy to add and can be done in a few minutes once you've installed Ubuntu.
