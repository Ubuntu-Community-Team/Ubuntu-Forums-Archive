### Post by arochester on 2009-06-18

### Post by elysianfields44 on 2009-06-18
> **arochester said:**
> Firefox>Preferences>Main>Downloads>Save files to (browse)

Yep. 

Also: Firefox > Edit > Preferences > Applications > PDF Document > Action (you can choose whether the default action is to display the PDF or to save it).
