### Post by Didius Falco on 2010-02-23
it isn't cross-platform, but you already have ms-paint in Windows - have a look at f-spot, which is available in the repositories, using the Ubuntu Software Center (or Add/Remove, if you are using a pre-9.10 version), in Synaptic or at the command line by opening a terminal and typing:

```
sudo apt-get install f-spot
```If you want to have a look at the website before installing it, go here: [http://f-spot.org/Main_Page](http://f-spot.org/Main_Page)

I hope this helps,
