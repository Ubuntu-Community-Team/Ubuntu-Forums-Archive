### Post by owenll on 2010-11-28
Use a terminal to install youtube-dl
```
sudo apt-get install youtube-dl
```

Then in terminal type:
> youtube-dl *urloffile*

for example

> youtube-dl [http://www.youtube.com/watch?v=BfIAKj3Gl1E](http://www.youtube.com/watch?v=BfIAKj3Gl1E)
