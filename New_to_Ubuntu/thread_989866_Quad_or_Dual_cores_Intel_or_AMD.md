### Post by mbsullivan on 2008-11-22
The best CPU will depend really on you, and what your needs for a computer are.

> will playing only some games(non-FPS) once in a while

FPSs really are the only games that stress a system to the max, and will benefit from a GPU upgrade more than an expensive CPU, anyway.

> using mainly the pc for music

Audio compression times scale almost linearly with number of cores, so if you do a lot of audio or video compression, this is one area that a quad core will excel.

> Microsoft Office, websurfing, watching movies, etc.

It sounds like you really don't need the added compute power offered by a quad core CPU. A dual core CPU will consume less power (saving a significant amount of money if you leave your computer on all the time).

Mike

### Post by HavocXphere on 2008-11-22
I'd go with an Intel. Maybe an E8400 or something along those lines if you want a dual-core.

Dual core --> More overclocking potential, less heat
Quad core --> Not fully utilized atm, but that will change

Usually a high-end dual will outperform a quad (ignoring i7) in game benchmarks.

Stay away from ddr3...its *very* expensive and the performance boost is not that big. Rather buy more ddr2.

If I had to buy now then I'd go for a Quad because I think thats the way everything is heading. Either that or delay till a low-end i7 is within reach.
