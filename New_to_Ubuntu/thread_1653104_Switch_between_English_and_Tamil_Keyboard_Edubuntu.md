### Post by shuzaimi on 2010-12-27
I found the perfect solution to this issue:-

Download & install the following packages.

ibus-im-switch
ibus-m17n 
m17n-db 
m17n-contrib

How to install? Open the terminal shell, type sudo apt-get install xxxx-xxx-xx
