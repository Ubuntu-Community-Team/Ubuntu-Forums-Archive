### Post by Rex Bouwense on 2011-03-23
You do a clean install when you download the ISO from a mirror, check the md5sum on the download, and then burn the ISO to a CD.  You then boot from that CD and it will ask you if you want to try Ubuntu or install it. If you choose install, it will ask you how you want to install it (on the whole disk or side by side with your current OS.).
