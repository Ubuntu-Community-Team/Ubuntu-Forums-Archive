When I right click on the desktop there's a create a document but inside is an entry called macchiato theme pack.

---

### Post by tgeer43 on 2009-08-30
This is probably a template. If so then you should have a 'Templates' folder in your home directory. Open it up and delete or move the template elsewhere and it should be removed from your right-click menu. You might have to restart X or reboot for the change to take effect.
