### Post by coffeecat on 2014-10-07
What works in a Windows hosts file would work in a Linux hosts file, and vice versa, would it not?

This is what I use:

[http://winhelp2002.mvps.org/hosts.htm](http://winhelp2002.mvps.org/hosts.htm)

Rather than replace my Ubuntu /etc/hosts with their version as they suggest, I edited in everything from the "# [Start of entries generated by MVPS HOSTS]" line to the end of my hosts file. That's mainly because I had a few lines for resolving devices on my home network. But use whatever works for you since the principle of a hosts file in the different operating systems is the same. Since you are new to Ubuntu, do you need help with gaining root privileges in a text editor in order to edit your /etc/hosts file?
