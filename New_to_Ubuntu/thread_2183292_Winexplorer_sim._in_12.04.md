### Post by jerome1232 on 2013-10-24
Are you wanting a list view instead of tile view? You can set that in preferences as the default view if you like. Edit -> preferences -> view tab -> at "view new folders using" drop down and select list view.

You can also modify what is shown in what order in the list columns tab.
