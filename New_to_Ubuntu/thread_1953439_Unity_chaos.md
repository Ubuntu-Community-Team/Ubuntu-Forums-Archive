### Post by 3rdalbum on 2012-04-07
System Monitor and Network Tools are in the applications list. You can click the Ubuntu button and then the More Apps button, then "See 210 more results". Or click Filter Results and click the System filter, then "See 20 more results".

So yes, it's quicker to type with Unity unless you install one of the "menu replacement" indicators that gives you an old-fashioned menu. Even so I still find it quicker to type what I want as I don't have to think 'Is this a program, or a document, or a video, or a song...", I just have to start typing its name.
