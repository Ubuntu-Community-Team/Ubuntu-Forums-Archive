### Post by grahammechanical on 2015-05-05
I find that automatic spell checking is usually on by default. Go to the Tools menu and see if there is a tick mark against Automatic Spell Checking. If not select it to activate automatic spell checking. Or press shift+F7

You can install any additional languages in Ubuntu, system wide, and that will pull the LO dictionaries required. Additional languages won't change your desktop menus or system messages unless you drag a different language to the top of the list.
This is quite easy from System Settings > Language... and it's probably the best way.
