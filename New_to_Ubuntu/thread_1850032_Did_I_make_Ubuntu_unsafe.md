### Post by grahammechanical on 2011-09-25
When you right click on a web page in firefox you get the option to View Page Source. If you click that you will see how that page was coded or formatted in Hyper Text Markup Language (HTML). This is the kind of coding, formatting or markup that web designers do so that we see web pages the way we do. To see the source code of a program you first need to download the source code and then open the file in an editor that displays program source code.
