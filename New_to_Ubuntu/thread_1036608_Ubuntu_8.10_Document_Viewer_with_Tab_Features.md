### Post by kansasnoob on 2009-01-11
The program is evince. It's a "hider"! You can't to my knowledge install the front-end to evince without "breaking" the Ubuntu desktop.

But go to terminal and type:

```
gksudo evince
```

I don't see a "tab" option but I haven't play with it much.
