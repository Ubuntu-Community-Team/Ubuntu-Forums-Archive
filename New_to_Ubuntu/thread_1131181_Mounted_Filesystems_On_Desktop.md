### Post by steve101101 on 2009-04-20
on the menu > applications > system tools > config. editor

then once in the configuration editor select apps > nautalis > desktop and click volumes visible.

you may have to edit the menu to see the configuration editor. this can be done by right clicking it and clicking edit menus. click on systems tools and check configuration editor.
