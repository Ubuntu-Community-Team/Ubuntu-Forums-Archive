While trying to install Ubuntu 8.04.1 LTS Desktop edition on one of my HD's, I had a failed install, and played around with things, trying to get the install done. Now, instead of being listed as a "SCSI drive" like my other drive, it's listed as "Filesystem." The problem being, Windoze no longer recognizes this as a drive, and since Ubuntu won't install on it, I want to install Windoze on that drive, and give Ubuntu my main HD as it's home.

How can I change my "Filesystem" back to a "SCSI Drive"?

---

### Post by Pro-reason on 2008-10-22
“Filesystem” is what GNOME calls the drive that Ubuntu is installed to.  It described other drives as best it can.  If they have a volume label, then it uses that; otherwise, goes by stuff like “SCSI drive”.
