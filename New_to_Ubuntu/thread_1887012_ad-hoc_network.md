### Post by roger_1960 on 2011-11-26
If they are Ubuntu laptops, try installing samba (do a search on it to learn about it!), then restart nautilus and the ad-hoc network should appear using places>network.  On each laptop some folders need to be shared for them to be visible to the other.  I think sharing folders may automatically install and configure samba.

### Post by Eiji Takanaka on 2011-11-26
I had a little mess about with this and yeah i think you can create an ad-hoc network fairly easily.  As i recall, i think you go to network manager, create new wireless network.  It essentially turns your computer into a wireless router.   Useful if for example you have only one broadband dongle, but want to enable multiple persons to go online. You can then also transfer files wirelessly, using smbclient.   Saves having to mess about with usb drives and the such. *You don't need to be "online" to do this.  ;)
