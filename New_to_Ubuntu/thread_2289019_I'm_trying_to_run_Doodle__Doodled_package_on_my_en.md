### Post by Rob Sayer on 2015-07-31
Well, one of the most common issues with linux novices is that they don't know how to find info yet.  For something like global searching I would not reccommend a command line tool for a beginner, especially one where you apparently have to update the index by CLI as well.

Disclaimer:  I personally don't use those things.  They speed things up at first for you but as the index grows in size it quite noticeably slows things down.  This isn't just a linux issue.  The Windows search index is worse if anything AFAIK, probably because the Linux file system doesn't have the fragmentation problems Windows has.

A quick search here has a suggestion by a very knowledgeable member here:

[http://ubuntuforums.org/showthread.php?t=2222612&p=13016564#post13016564](http://ubuntuforums.org/showthread.php?t=2222612&p=13016564#post13016564)

and it has a GUI.
