### Post by Rex Bouwense on 2011-11-16
I had a similar problem and it will occasionally pop up again.  What I did was to right click the panel and chose Properties.  I gradually increased the size of the panel until the icon reappeared and then I decreased the size to 24 which is the default I believe.  I know that it is just a work around but it has only happened 2 or 3 times and is only an annoyance so that is as far as I took it.

If you want to change the information, you need to go to User CP and then edit your information.
