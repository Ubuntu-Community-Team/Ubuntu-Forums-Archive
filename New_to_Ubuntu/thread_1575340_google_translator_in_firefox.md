### Post by beew on 2010-09-15
I went to a webpage in French, on top of the firefox screen there was a bar saying "Google translate view this page in <choose your language>" I chose English in the drop down menu and waited and it seemed that it was stuck at 0% forever and the page was still in French. I installed the google translator plugin but the result was the same, so I uninstalled it because it didn't do anything.

I navigated to the same page using FF in Windows and it translated right the way (There is no google translator plugin)

I don't think I need to install language support packages because it is done online and I can already read the French.

---

### Post by beew on 2010-09-15
Solved! Turns out the flash block addon to firefox was preventing the translator from working (probably because page has embedded flash) . Disable flash block and the page translates.
