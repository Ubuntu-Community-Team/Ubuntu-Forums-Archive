### Post by Derek Karpinski on 2012-06-05
Did you download google earth from the google site directly? Or did you install 'google-earth-package'?

If you downloaded and installed it from the google website, it should show up in the dash.

press super (windows) key -> type 'google' -> click on Google Earth.

or if you insist on running it from terminal:

```
google-earth
```

Will run it.

If you've installed 'google-earth-package', you'll have to follow the instructions here:

[https://help.ubuntu.com/community/GoogleEarth](https://help.ubuntu.com/community/GoogleEarth)
