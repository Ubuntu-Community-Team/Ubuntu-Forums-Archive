### Post by grahammechanical on 2014-01-28
I am not sure that I understand what you mean. A lot has changed in Gnome and Ubuntu since 9.04. You can go to the power/cog menu and open System Settings>Appearance>Behaviour tab and tick to enable workspaces and that will put a workplace switcher icon in the Launcher. The default number of workspaces is 4. Is that what you mean by tile?

Another thing you can do is to install CompizConfiguration Settings Manager (CCSM) from the Ubuntu Software Centre. Also install compiz-plugings. This brings in plugins that are additional to the default set. We can then use CCSM to enable certain special effects but take note of the warning message that we get when CCSM loads.

Another tool that might be useful is Unity Tweak Tool. If you want to assign your own images as wallpaper/backgrounds, then put the images in the Pictures folder and then open the appearance utility and change the drop down menu from Wallpapers to Pictures Folder and the images in the Pictures folder will be available to select as wallpapers.

Oh, incidently, hold down the super key and a keyboard shortcuts overlay will appear. You might find the keyboard shortcuts to be convenient.

Regards.
