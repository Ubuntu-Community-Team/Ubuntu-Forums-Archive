### Post by zemega on 2014-06-02
Regarding Ancestral Quest, seems like it is not avilabe in Linux. There is a program that can 'run' most Windows program inside Linux, called Wine. Ancestral Quest 12.1 is given Platinum Rating in Wine which means it works great in Linux. There are also other software for you to consider look at [http://www.gensoftreviews.com/index.php?sel=1&new=&lic=free&pla=unix&type=all&sort=](http://www.gensoftreviews.com/index.php?sel=1&new=&lic=free&pla=unix&type=all&sort=) . Its time you start looking at free genealogy program. Try Gramps in Windows, see if you like it and can open the files saved using Ancestral Quest. If it does and you like it, you should move to Gramps or any other program. It comes with learning how to use the program from scratch though.
Regarding your bank, its kind of hard to tell, but if you can access it with Chrome, good chance are you can access it using Chrome in Ubuntu as well. Gmail or Youtube is not a problem. And generally you don't really need to put antivirus in Ubuntu. Be sure to turn on the Firewall in Ubuntu though.
There is LibreOffice that is a free office suit, so you can edit your Word and Excel files as well. Hopefully not docx or xlsx. I use Kingsoft Office for Linux though.
You will need to install Chrome and any genealogy software that you wants.
Regarding Ubuntu choices, Xubuntu would be better choice to you than Ubuntu. XFCE (Desktop environment in Xubuntu) will feel similar to XP. Unity (Desktop Environment in Ubuntu) might  be difficult to understand if you;re not willing to learn it.
