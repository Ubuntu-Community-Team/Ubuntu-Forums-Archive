### Post by komputes on 2011-02-23
Make sure you update the system. If you see nothing when you boot, boot with the shift key down until you see the GRUB menu where you can boot into recovery mode, then select "netroot" to get an IP (Ethernet cable should be plugged in) then ctrl-d, then go to "dpkg", once that is done updating the packages go to "root" and type the command "reboot".

Does this fix the installation?

### Post by cariboo on 2011-02-23
I'm not sure about Ubuntustudio, but Synaptic Package Manager is installed by default. Look under System-administration. One nice feature that is available, if you have the correct packages installed is the Mark packages by task, under the edit menu. This groups packages into different activities. To enable the feature, you have to install tasksel, which can be done using synaptic See screen shot.
