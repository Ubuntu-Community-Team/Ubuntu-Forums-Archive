### Post by Shwefty on 2008-11-14
This *may* work.  It depends on if the Applications list has been updated.

Right click your Applications menu : Edit Menus

Then click on the Graphics tab on the left.  If you see your program in the list on the right, just select it and you're done!  If not, you can click "Add Item", but you'll have to know the command in Terminal to start it!  (possibly the command is ```
blender
```).  The command obviously goes in the command entry area.

### Post by rorkimaru on 2008-11-14
Thanks, I have it solved. 

For those interested I made a folder in the same place as the "Music", "Documents" etc folder called "other programs"

I then went system -> preferences -> main menu as described above.

I clicked on graphics, clicked add new and wrote the title (Blender)

Then I clicked browse and navigated to the blender file.
