### Post by lovabill on 2011-09-09
Well, imagine your disk as a table with two different board games on it being played simultaneously.

Now, imagine two tables with only the one board game on each. Better,right?

So, it's better to have separate partitions.

And, you can also check [this]("http://ubuntuforums.org/showthread.php?t=1075774").

I assure you that ext4 is way better. Try this: download a rar file with many files in it and unrar it both in Windows and in Ubuntu. Or try to copy a folder full of tini tiny files. You can feel the difference. Ubuntu can be many seconds faster.

### Post by Frogs Hair on 2011-09-09
If you are serious about using Ubuntu for extended periods , installing it on its own partition is a better option . Wubi installations can't hibernate and have a 30gb maximum size .

I started with Wubi and moved to a traditional dual boot .  Wubi installations are installed  inside a folder in the Windows partition  and if Windows fails so does Ubuntu . Read up on installing a dual boot before hand . A quick search will produce a lot of information .
