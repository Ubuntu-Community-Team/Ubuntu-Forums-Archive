### Post by clive littlewood on 2009-03-02
Hi

If you want to set up a different background image, do: 


Launch Applications->Settings->Settings Manager. 

Choose Desktop. 

In the section Image, click on the file icon. This will bring up up a open file dialog, where you can choose an alternative image.

Hope this helps          :P

Clive
