### Post by pgar23 on 2009-10-27
You did not install anything probably. Generally, when you install Ubuntu, it installs a "GRUB boot loader, something like the MBR (Master Boot Record) and the boot loader gives you options of which OS to boot into. If there is no GRUB boot loader then you probably did not install Ubuntu correctly.
