### Post by YannBuntu on 2010-07-05
As Microsoft changed its MSN protocol in March 2010, it is now impossible to do video&audio calls.

The best you can do with MSN protocol is using aMSN that can do, via some tricks, instant video (without audio), and voice messages, but no voice calls.
Emesene is not aimed at supporting webcam, aMSN is better for this.

If you really want to do voice&video calls with Windows interlocutors, I recommend :
- Ekiga (which is available on Windows too)
- Skype, which is available on Windows too, but is not open-source
- Empathy with Telepathy PPA, the ubuntu-restricted-extras package, and a Jabber account (e.g. a gmail adress), that can do voice&video calls with Windows Gtalk users.

Of course these 3 solutions don't use the MSN protocol, so they cannot do voice&video calls with MSN (WLM) users.

And for voice&video calls with 3 or more Linux users, you can use Muji (kind of Empathy extension).
