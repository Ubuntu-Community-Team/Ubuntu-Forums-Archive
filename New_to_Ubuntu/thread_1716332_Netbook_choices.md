### Post by nosehat on 2011-04-01
I just did a fresh install on a new netbook for my wife:

Netbook:  Went with a **Toshiba NB505**.  Chosen because it was cheap (less than $300), and had decent specs.  Also, we needed it that day, and it was available at the local big box store.

OS:  Went with **Ubuntu 10.04.1 Desktop** (not the "Netbook" version).  

Why I didn't go with Ubuntu Netbook Edition:  I installed the Ubuntu Netbook Edition first, thinking it would be good to have something optimized for netbooks.  I really didn't like it at all.  The desktop was gone, nautilus was gone.  I couldn't figure out how to do even simple file management except through the terminal.  The screen was crowded by a graphical application-launching bar on the left that I couldn't figure out how to customize, or get rid of.  It seems to me like it's an OS designed for touch screens and tablets, not netbooks.  Granted, I didn't give it much time to grow on me, but if you are looking for a seamless transition from using a desktop or laptop, the netbook edition probably isn't for you.

**The result:  Excellent.**  Everything worked fine with Ubuntu 10.04.1 right away, except the wireless card isn't supported.

Here's what you need to get the wireless up and running if you have this particular Toshiba Netbook:

```
sudo add-apt-repository ppa:lexical/hwe-wireless
sudo apt-get update
sudo apt-get install rtl8192ce-dkms  
``` Obviously, you need to be plugged into a wired network for that to work.  Once it's done, you can connect wirelessly no problem.

I also share the love of the **Asus eee** netbooks.  I'm posting this on my own, trusty **Asus eee 1000HE**, running a now very outdated eeebuntu (a good distro in its day, but no longer maintained).  Looking for a fresh new distro for my eee, which is what brought me to this thread in the first place.  :)
