### Post by 73ckn797 on 2010-11-14
I assume the wire is connected between the router and computer? Or, you can see the router as a wireless connection?

The default router address is 192.168.1.1. Enter that into the Firefox address bar to access the router. The default password is admin. The access is to set passwords and configure the router for wireless use and other settings.

The owners manual will give you all the info you need. If no manual you can download it from Linksys, that is if you have another internet connected computer.
