### Post by duke.tim on 2011-05-19
You might want to download from the software center CompizConfig.
or via terminal

```
sudo apt-get install compizconfig-settings-manager
```

It will contain a list of settings you can customize :)
It will look like this

[http://a2.sphotos.ak.fbcdn.net/hphotos-ak-snc6/230733_2093987633070_1346678943_2564363_4435011_n.jpg](http://a2.sphotos.ak.fbcdn.net/hphotos-ak-snc6/230733_2093987633070_1346678943_2564363_4435011_n.jpg)
