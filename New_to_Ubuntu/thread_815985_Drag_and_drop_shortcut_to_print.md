### Post by aseem on 2008-08-19
Just right-click on your desktop, and select "Create Launcher" for a create Launcher dialog.

In type field select "application i terminal"
In Name field, type any thing you want, ie. Printer
In Command field type:  lpr %1
Select an appropriate Icon


Now you have a printer icon that you can drag to, to print.
