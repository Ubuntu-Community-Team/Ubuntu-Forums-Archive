### Post by nhasian on 2010-06-19
i've never used the SGFXI script but you can upgrade to the latest nvidia driver 256.35 simply by using the [Ubuntu-X PPA]("https://launchpad.net/~ubuntu-x-swat/+archive/x-updates/")

You can update your system by adding **ppa:ubuntu-x-swat/x-updates** to your system's Software Sources.
