### Post by FoxFireX on 2009-07-10
I installed compiz desktop effects, but theres no settings manager in the system menu, or anywhere. The windows are wobbly and everything but, what do I do to get it so that I can go into the settings and enable the 3d cube and stuff?

---

### Post by kpkeerthi on 2009-07-10
Search for compizconfig-settings-manager in Applications -> Add/Remove and install it.

---

### Post by nothingspecial on 2009-07-10
```
sudo apt-get install compizconfig-settings-manager 
```

Then it will be in system > preferences in your menus.
