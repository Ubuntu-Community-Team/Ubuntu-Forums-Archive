Hi, I am no expert by any means but if you are trying to achieve a dual boot system, you need to use GParted, from a LiveCD/USB, to carve some free space on your drive first. There are many 'howto's on this forum - I know they helped me. I hope they help you. If it's any encouragement, I'm running a triple-boot system on a T43.

Also, look in the File System for the directory /media to see what Linux calls your Ultrabay drive. Alternatively, if your flashdrive is 4Gb or larger, you could run Ubuntu directly from it...

HTH
