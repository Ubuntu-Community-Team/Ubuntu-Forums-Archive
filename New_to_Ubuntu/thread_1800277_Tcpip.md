### Post by uRock on 2011-07-08
Using wireless or hardwire?

Either way, right click the network manager icon, then select Edit Connections, then select the proper tab, then click on the profile in use, then click the Edit button, then click the IPv4 tab, then from the drop down menu select Automatic DHCP, then click Apply and let network manager do its deed, which should get you connected.
