### Post by buzzingrobot on 2014-03-10
The most important difference is that Gnome 2 came to an end 3 years ago. The code has been picked up and adopted by the Mate project.  They are migrating it to be compatible with contemporary code while maintaining the Gnome 2 design and functionality.  Many of today's applications depend on resources unavailable in a pure Gnome 2 from 2011.
