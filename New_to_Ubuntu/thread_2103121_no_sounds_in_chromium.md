### Post by Bölvaður on 2013-01-09
Lets find out if Chromium is even trying to use sound.

First play some audio through Chromium.
Now click the audio icon (where you control it's volume) and click preferences). Then check if Chromium is listed under applications trying to use audio.

Is it there?

If not get a program named something like *pulse audio volume control* and do the same but with this program.

Is it listed?
