### Post by grahammechanical on 2014-10-06
Ubuntu has several flavours with different desktop environments (DE) and user interfaces (UI). Ubuntu uses the Gnome DE and the Unity UI. Xubuntu uses Xfce. Kubuntu uses KDE. Lubuntu uses LXDE. Ubuntu Gnome uses the Gnome DE but the Gnome 3 UI.

All are distinctive. During the loading process a splash screen should inform you what version is being loaded and so should the login screen.

[http://xubuntu.org/](http://xubuntu.org/)

[http://www.kubuntu.org/feature-tour](http://www.kubuntu.org/feature-tour)

[http://lubuntu.net/](http://lubuntu.net/)

[http://ubuntugnome.org/screenshots/](http://ubuntugnome.org/screenshots/)

I can say with some degree of certainty that the installation with the menu bar on the left (real name Launcher) is Ubuntu with Unity. Unless it is Ubuntu Gnome. What did you download and where from?

At the point where you login... Standard Ubuntu shows only your username and the small frame with the password field where, at the right, you find a button. LUbuntu has a different login manager and you find the available options for the DEs in "Session", near the dropdown menu to select language, keyboard, etc.
