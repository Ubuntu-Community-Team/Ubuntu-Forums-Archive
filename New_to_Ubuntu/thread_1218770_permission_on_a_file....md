### Post by mudcat on 2009-07-21
easiest way to change file permission (not command line) is to allow root user at login screen (System>Admin>Login) and log in as root user; then find the file you want to change the permission on and right click, se;ect properties and check the permissions tab; then you can adjust the permissions as you see fit. (Note: When your done, its probably a good idead to uncheck the root login capability)
