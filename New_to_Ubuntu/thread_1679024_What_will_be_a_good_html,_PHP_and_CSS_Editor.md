### Post by weedeater64 on 2011-01-31
*Gedit - nice straight forward text editor, with syntax highlighting for many languages, including those you listed.

*Geany - nice simple IDE with a few more options, including use of headers/templates that you can customize to your liking, and project management features.

*Bluefish - much like geany, but more for web-dev specifically, tag menu, dialog menu, project menu, auto elements, project management, popout help for tag completion...  

_***If some one knows how to get bluefish to use lowercase tags, instead of Caps, I'd be thankful.***_

Many, many other related applications to choose from. Spend some time browsing the available packages, learn how to search them , you'll be amazed at all the little specialized little know apps.
