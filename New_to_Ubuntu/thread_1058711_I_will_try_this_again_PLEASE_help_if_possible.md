### Post by dje on 2009-02-03
System >> Preferences >> Appearance. Select the theme you are using and click the 'Customise Button'. Next go to the 'Colours' tab and set the tooltip background colour to something other than black, alternatively click 'Reset to defaults'

hope that helps,
