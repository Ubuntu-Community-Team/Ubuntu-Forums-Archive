### Post by OlsenCNC on 2009-10-10
If you're trying to install Python perform the following:

Go to Add/Remove Applications from the Applications Menu.

Type "Python" in the Search box and press Enter.

Locate the Python Application you are looking for and Place a check to the left of it.

Click "Apply" at the bottom right of your screen and Enter your Password when prompted.

Then you should be able to Launch the program from the Applications Menu under Programming.
