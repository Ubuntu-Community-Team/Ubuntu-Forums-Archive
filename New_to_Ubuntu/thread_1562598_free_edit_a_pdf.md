### Post by stmiller on 2010-08-27
Openoffice can edit PDFs with this extension:

[http://extensions.services.openoffice.org/project/pdfimport](http://extensions.services.openoffice.org/project/pdfimport)

Tools > Extension manager - browse to install this extension.


Once that extension is installed, just click File > Open 

in openoffice and open your PDF.
