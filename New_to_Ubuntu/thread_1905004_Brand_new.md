### Post by Basher101 on 2012-01-05
hi and welcome to the vast and vibrant world of free open source software! make yourself at home.

Did you even install ubuntu or are you just using the Live CD sessions? (thus you boot from  the CD and click "try ubuntu") 

a terminal is where you can type commands. You open a terminal by either pressing ctrl + alt + t, or you can hit your windows key and type in the search bar "terminal" and click on the icon that appears. Once you obened your terminal, type the command ```
lsb_release -a
``` to see what version you use.

note: you can only paste into the terminal with ctrl + shift + v, unlike the default ctrl + v.
