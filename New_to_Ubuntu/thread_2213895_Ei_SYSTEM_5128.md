### Post by Impavidus on 2014-03-29
You're really low on memory. Try installing some extra ram if you have it lying around (probably not worth buying new ram for such an old computer). Also, make sure you install Lubuntu, which is the Ubuntu flavour designed for low-spec hardware. Graphics should be no problem.

Modern Ubuntu has hardware requirements similar to Windows 7. Lubuntu should run fine on most WinXP-era machines.

### Post by Rob Sayer on 2014-03-29
There seem to be a lot of linux newbie XP refugees lately who don't realize there are 4 different ubuntu desktops.

And, yes, as above, you can't really run unity on a 256Mb computer.  I'm in lubuntu (LXDE) on my netbook now and that's what you'd want.  It's by far the lightest andf fastest ubuntu desktop.

In fact 256Mb is the minimum ram recommended to be able to use lubuntu, and I think that's pushing it myself.  For example I'm in firefox in 32 bit lubuntu 13.10 right now.  FF is using almost 190Mb, and that's with one browser tab.  Which means in a 256Mb system firefox wouldn't be useable.  Another, lighter browser may though.

In this case I'd actually look into maxing out the ram in the machine ... can you even get ram for that hardware anymore? ... or going for a more lightweight, minimal linux distro.  It'd be more work but it would run better in that memory space.
