### Post by mastablasta on 2011-11-29
Kubuntu can be installed with a simple copy paste of command (or if you find it in software center), however you will have duplicated programmes (Ubuntu stuff will stay). Mint would have to be installed next to the other two using disk partitioning to create a triple boot.
 
however if you just want to try them it's much better if oyu use a liveUSB and install the live image to that and then boot from it. if your computer can't boot from USB then use easy to use PLOP boot manager.
