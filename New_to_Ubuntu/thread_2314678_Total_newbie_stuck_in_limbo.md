### Post by sandyd on 2016-02-23
Yep, you can download the latest version of Ubuntu from [http://www.ubuntu.com/desktop](http://www.ubuntu.com/desktop)

When installing, choose "Erase Entire Disk", which will remove all data from your disks.

I've also noticed that you have an older computer, which may have issues running Ubuntu depending on it's specifications.

If you feel Ubuntu is slow, try Xubuntu or Lubuntu, which may be faster and lighter on older hardware.

Ubuntu 14.04 is a LTS release that is supported until April 2019, while Ubuntu 15.10 is supported until July 2016.

### Post by buzzingrobot on 2016-02-24
You were not able to update because 10.04 is a very old release that is no longer supported.  As a result, updates for 10.04 not available.

As others suggest, you should install a supported release. Follow the link provided by sandyd. Look around that page, too, for guidance on how to do all this.

Several offical "flavors" of Ubuntu are available. Ubuntu is the primary version and uses the Unity interface.  Other flavors replace Unity with another interface, along with a set of applications.  The operating system remains the same, otherwise, and the same range of software is available from the same servers for all flavors.   An older machine like yours will likely benefit from using a version that places fewer demands on the hardware, especially the video card.  Those are Lubuntu and Xubuntu.  More here: [http://www.ubuntu.com/about/about-ubuntu/flavours](http://www.ubuntu.com/about/about-ubuntu/flavours)

When you boot an Ubuntu install image you'll be given the option to run it in "live" mode.  It won't work as fast as a real install but it is a good way to check out hardware compatibility.
