### Post by gdonwallace on 2009-10-25
In the settings for your virtual machine, there are options for the DVD/CD rom.  There should be a check box to enable support for DVD/CD and then another option to force the virtual machine to use the local mounted DVD/CD.  Make sure both of these have a check in the box and it should work.
