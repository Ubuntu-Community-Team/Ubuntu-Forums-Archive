### Post by Frogs Hair on 2019-02-16
I store some gedit themes in a folder gedit/documents and  navigate to theme using the fonts and colors path. Some themes download as xml files and need . placed in front of them.

Edit: I located and installed the theme as described above. Create a folder anywhere, name it, place the theme/s there, and navigate to folder using fonts and colors.
