### Post by ChameleonDave on 2008-05-03
> **explorenow said:**
> How do I uninstall Ubuntu 8.0.

Thanks for the help

There is no Ubuntu 8.0.  The most recent version is 8.04, meaning April 2008.

If you already have another operating system installed, boot into it and use its partition manager to delete the Ubuntu partitions.  The specifics will depend on your set-up.

If you don't have another operating system installed, then you will want to install one.  Insert its installation CD, and use it to remove the Ubuntu partitions and replace them with its own.
