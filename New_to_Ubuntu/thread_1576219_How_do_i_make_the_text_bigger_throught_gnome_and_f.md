### Post by David Andersson on 2010-09-16
> **bradmayne04 said:**
> the text is really small.  i know how to make it bigger in firefox but then it gets small again.  how can i make the text bigger and keep it that way?

Kerry's solution changing the DPI might work. If it does not work well enough, restore DPI to nominal value and change the font size in Gnome as you want it (*system > preferences > appearance > fonts*), then in Firefox install the extension [NoSquint]("http://addons.mozilla.org/en-US/firefox/addon/2592/"). Out-of-the box it zooms all pages to 120%. You can change zoom level, and wether to zoom text and images, or only text.
