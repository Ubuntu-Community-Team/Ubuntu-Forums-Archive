### Post by mcduck on 2014-03-18
In addition to those options, Nautilus will display more information even in the default icon view based on the zoom level. By default nothing is shown at default zoom level of 100%, but you change that (and what information is dispalyed on each zoom level) in the fiel manager's preferences.

I prefer having at least file/directory size enabled for default zoom level. Of course if you want o dispaly lots of informationa t the same time then using the list view would be better option.
