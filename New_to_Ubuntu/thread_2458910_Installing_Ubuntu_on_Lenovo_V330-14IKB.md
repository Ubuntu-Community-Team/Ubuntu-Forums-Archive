### Post by coley9225 on 2021-03-06
Hello francisco. I currently run a multiboot on a Lenovo 320-15 iap. I love it. I have windows(yuck),Lubuntu, and ubuntu(both 20.04). I spent a year+ getting linux distro to install, always hanging on grub install.I had to install without bootloader,,run boot repair, then after rebooting to linux os, purge and reinstall grub. Other than that, even for an under-powered machine I have had a wonderful experience with Lubuntu. It can be installed on Lenovos and run well. Whether or not Ubuntu will run as expected depend on hardware, but if it doesn't perform well, try other distros that are lighter on resources, like lubuntu. Best of luck and welcome to the freedom of linux.
