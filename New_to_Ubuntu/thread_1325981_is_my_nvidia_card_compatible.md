### Post by brianfactors on 2009-11-14
According to this page: [https://wiki.ubuntu.com/HardwareSupportComponentsVideoCardsNvidia](https://wiki.ubuntu.com/HardwareSupportComponentsVideoCardsNvidia)

The NVidia GeForce FX Series is supported by Ubuntu. The driver should automatically get configured, and you can go back to that page if you need to do any special configuration for the 5800.

Added: Oh, and Nvidia makes their own driver available for Linux, both 32 and 64 bit editions. [http://www.nvidia.com/Download/index.aspx?lang=en-us](http://www.nvidia.com/Download/index.aspx?lang=en-us) So, if all else fails, you can just install it by hand.

Good Luck
