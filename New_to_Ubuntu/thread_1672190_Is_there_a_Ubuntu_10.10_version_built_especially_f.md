### Post by Verbeck on 2011-01-21
he probably meant the 64 bit (also called AMD64) version which you may use if your processor is 64 bit capable - regardless of the manufacturer

edit: / btw, you could install 32bit on a 64bit capable processor but not the other way around

edit 2: link [http://www.ubuntu.com/desktop/get-ubuntu/download](http://www.ubuntu.com/desktop/get-ubuntu/download)
select 64bit from the drop down list and it'll download the ubuntu-10.XX-desktop-amd64.iso
