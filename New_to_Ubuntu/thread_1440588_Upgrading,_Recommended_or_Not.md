### Post by Rocket2DMn on 2010-03-27
I'm a little unclear about what version you are using now - you have 8.10 back on your system?  If so, I suggest backing up your data and doing a clean install of 10.04 when it is released.  Taking the upgrade path from 8.10->9.04->9.10->10.04 will be time consuming, and since you've had problems in the past, it is safer to skip that.  Lucid (10.04) is a LTS release so it will be supported on the desktop for 3 years, which means you won't need to upgrade if you don't want.

### Post by eeeandrew on 2010-03-27
I agree with the folk above. Best bet is to install a fresh version of 10.04 when it comes out. Have a look at this tutorial [http://www.psychocats.net/ubuntu/separatehome](http://www.psychocats.net/ubuntu/separatehome)
which will allow you to set up a seperate /home partition. This means that all your files, folders and settings are effectively on a different hard disk(although it may physically be the same one) which means you can overwrite the /root partition(the operating system) while keeping your documents and settings You will however, have to reinstall any programs that you installed yourself. This could include drivers.

Hope thats useful,
