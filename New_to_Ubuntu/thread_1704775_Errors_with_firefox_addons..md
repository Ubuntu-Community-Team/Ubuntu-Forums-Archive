During recent floods and cyclones I found I wasn't able to access certain things (like Ubuntuforums.org) and the update servers occasionally. At one point I was actually using the NZ servers for updating instead of the Australian ones.  After two or three days everything came good, until Optus had a major problem which took out half the accessible network again for half a day.  I stopped using the NZ servers because they were no longer reliable and started using the Australian ones again.

I found if I couldn't ping the server (network tools) then I certainly wasn't going to get any data from it.  This even happened with ubuntuforums.org for a while.

You could see if there's another source you can get the files from, try the Firefox site.

Internet routing should be fairly foolproof - that's the way the US military designed it to be.  The fact is though, if the problems are close to home and on the network service provider's equipment or main transmission lines, things seem to get a little hairy.
