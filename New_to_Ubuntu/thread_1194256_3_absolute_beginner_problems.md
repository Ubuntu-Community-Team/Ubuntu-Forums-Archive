### Post by LowSky on 2009-06-22
1. the lock up issue: [https://bugs.launchpad.net/dell-mini/+bug/300563](https://bugs.launchpad.net/dell-mini/+bug/300563), changing your wireless router settings will most likely stop the issue. Blame Dell for using sub par linux wifi parts

2. icons moving about: make sure that things are locked to the panel, right click on the object on chose lock, also using a second monitor might still mess things up if you dont lock them up.

3. Linux commands: please be a bit more specific. Learning the command line can be annoying, but the rewards are great. Commands from 7.04 should still work just fine, unless the applicaition in use was changed in newer releaes. And if a command says access denied, use the sudo command.
