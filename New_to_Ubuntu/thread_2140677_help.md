I would download this:

[https://launchpad.net/~gezakovacs/+archive/ppa/+packages](https://launchpad.net/~gezakovacs/+archive/ppa/+packages)

If you have Ubuntu 8.04 you'll want to click on:  unetbootin - 563-2~hardy1 and then download either of these 2 (depending on what cpu type 32 bit or 64bit you have installed):

unetbootin_563-2~hardy1_amd64.deb (299.8 KiB)
unetbootin_563-2~hardy1_i386.deb (302.6 KiB)

If you have Ubuntu 8.10:

unetbootin - 442-1~intrepid1

unetbootin_442-1~intrepid1_amd64.deb (243.2 KiB)
unetbootin_442-1~intrepid1_i386.deb (242.0 KiB)

Download and install the package, then use that to install the .iso image of linux mint or ubuntu.

or use a windows machine and download: [http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

Then download the .iso and use the above to write it to usb.
