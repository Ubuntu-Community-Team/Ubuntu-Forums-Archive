### Post by localhost8080 on 2012-10-30
im not sure you will get pc2100 any more, and 1gb sticks were rare when they were out (I used to be a manager at an independant custom pc building store round that time) - itd probably be easier to find new pc3200 sticks (ebay is probably the best place for this :)

most of the pc2100, pc2700 and pc3200 ram sticks work in motherboards that support one of those speeds - eg you can run pc2100 in a board that supports pc3200 and you can run pc2700 in a board that supports pc2100 :)

if you get the pc3200 then they will run as pc2700 in your machine, but you'll still notice a big improvement going from 384mb to 2gb!
