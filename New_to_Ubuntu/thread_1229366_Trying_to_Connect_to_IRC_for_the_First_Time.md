### Post by pedro3005 on 2009-08-02
The program XChat comes with ubuntu, and is better for IRC. Go to Applications, Internet, XChat IRC.
Select Freenode or type /server irc.freenode.org , then /nick to your registered nick. Then type '/msg NickServ identify password' , changing password to the password you chose. Now, /j #ubuntu-beginners . If that doesn't work, come back and post error message :)

I should point out that you shouldnt install the program xchat-gnome, you want the plain xchat one.
