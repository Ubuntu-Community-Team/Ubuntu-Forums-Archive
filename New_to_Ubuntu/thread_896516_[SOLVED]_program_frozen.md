### Post by anotherdisciple on 2008-08-21
Hit alt+F2 and type 
```
xkill
```

then click on the window that you want to close.

---

### Post by anotherdisciple on 2008-08-21
Oh... and if you want to do it much like windows go to your menu under...

System > Administration > System Monitor

that is much like the task manager in windows

or you can get to it by typing alt+F2 and typing:

```
gnome-system-monitor
```
