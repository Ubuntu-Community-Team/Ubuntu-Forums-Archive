### Post by drs305 on 2008-12-01
If you use Compiz, open it, go to Window Management, Place Windows, Fixed Window Placement, and expand 'Windows with fixed viewport'. Click Add, then identify the window by title/class/etc. and select the viewport.

### Post by ThrobbingBrain66 on 2008-12-01
Probably the easiest way is to use Compiz and it's Place Windows plugin.  Under the Fixed Windows Placement tab, use the 'windows with fixed viewport' section.

So to use your example, click the New button.  Then change the 'x viewport' property to 2.  After that, click the plus sign (+) next to 'viewport positioned windows'.  Use the 'grab' button to select the emesene window.  This will fill in the value text box.  Finally, click add and then close.  Now all your emesene windows should open in the second viewport.

Sure.
I just installed emesene and it opens with the title window 'emesene'.

Click 'New', Click the + symbol in the top right (Viewport positioned windows).

Click on the selection box and choose "Window title".
Type in "emesene" in the Value and enter.

Select which viewport you want in the X viewport position.
Click Close.

Try opening emesene and ensure it opens in the correct viewport.
