### Post by Lateralis on 2010-09-26
If you have Windows on one partition and everything else is unpartitioned then that is fine.  What I would *strongly* recommend doing is creating an NTFS partition which is to act as a shared data/media type partition.  This is to try to avoid reading/writing un-necessarily to the Win 7 OS partition.  How big you create this shared partition is up to you and will depend on how much data you want to share between Linux and Win 7 and how many other versions of Linux you want to have installed on the computer at any one time.  Keep in mind that Ubuntu (and Linux generally) do not really need that large a partition.  

As for needing a detailed tutorial, well, there are plenty around on the internet; have a google.  There's a pictorial installation link in my signature.  There are also Ubuntu wiki pages relating to installation, as far as I can recall.  

If you have any specific questions though, feel free and ask them.
