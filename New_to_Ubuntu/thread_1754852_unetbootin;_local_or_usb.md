### Post by beew on 2011-05-10
> **Matrix01 said:**
> somebody sggested me to use Unetbootin, and install distro.
 
how to choose where to install this: local drive or usb?

What?

Do you mean you want to use unetbootin to create a live usb and use it to install Ubuntu in hard drive, or do you mean  using unetbootin  to "install" Ubuntu in a usb drive so you use it as a portable? (it is not really "install" but more like copy so I use quotation around install) If it is the latter you should not use unetbootin because it doesn't support persistence. Instead use the startup disk creator if you are using Ubuntu or LiLi if you are in windows.[http://www.linuxliveusb.com/](http://www.linuxliveusb.com/) Both have the persistence option.
