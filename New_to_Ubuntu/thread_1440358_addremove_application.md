### Post by NickJones on 2010-03-27
Go:
System->Administration->Synaptic Package Manager
It may ask you for your password, if so type it in.
In the quick search box type what you want to find, for example, "2vcard"
Type it into the search box.
Click on the tick box to the left of the program name & description, then click "Apply"

For more info, read the guide:
[https://help.ubuntu.com/9.10/add-applications/C/index.html](https://help.ubuntu.com/9.10/add-applications/C/index.html)
