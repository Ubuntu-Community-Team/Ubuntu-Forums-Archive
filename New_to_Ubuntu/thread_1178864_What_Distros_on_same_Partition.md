### Post by Bucky Ball on 2009-06-05
Go right ahead. In fact, you can have one install of Ubuntu and install Gnome, KDE or Xfce or any other** desktop manager. _*These are not distros*_ ... **they are desktop environments.  When you boot your machine and are at the login screen, go to 'sessions' and choose whichever desktop environment you fancy (there is also Openbox and many others you can try).

And ... you can be running Ubuntu, go to Synaptics and install kubuntu-desktop. That will install all artwork and environment. Experiment and make some blends; all three of our machines are 'hybrids'. 

As for how many distros on the drive, go for your life! Just make sure you have them on seperate partitions and you have your grub tweaked correctly and you're good to go. (Ubuntu is built on Debian, incidentally).

Hope this is of some help. Have fun. :)

Something to remember: Kubuntu, Edubuntu, Xubuntu, Ubuntu are all built on Ubuntu, only the apps and interface are different. You are essentially using the same OS. You could not load Windows, for instance, on the same partition. You've got one or the other.
