### Post by BDNiner on 2008-10-05
The alternate cd is a text based install used for older computers who's video cards don't have enough memory to run the regular graphical installation. The desktop is the regular version. it boots to a working "live" version of the ubuntu desktop so you can try it out before you install it.

If you problems installing ubuntu with the desktop version it is a good idea to try installing it with the alternate CD.

### Post by shifty_powers on 2008-10-05
server is a stripped down version that is designed to be, yoiu guessed it, a server, be it file or internet.

desktop is your normal distribution that most use.

alternate is a text based, basic installer that is designed to solve a lot of installation issues.

tbh, i386, (ie 32 bit), desktop edition is usually the safest option...
