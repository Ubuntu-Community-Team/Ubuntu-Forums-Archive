### Post by 3rdalbum on 2010-03-28
Wine is a last resort; it's amazing that it works as well as it does, but there are massive chasms in compatibility. It's very difficult to reverse-engineer 25 years of Microsoft APIs.

Wine is more a "I really need this particular program, I'll try Wine" than a "I'll run all my favourite Windows programs on Linux".
