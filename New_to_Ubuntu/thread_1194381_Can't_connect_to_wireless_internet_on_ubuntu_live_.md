### Post by t0p on 2009-06-22
Have you got networking enabled?  Right-click on the network manager icon (in the top right corner of the screen) and make sure Enable Networking and Enable Wireless are both ticked.  Then left-click on the network manager icon to see if your wireless network is detected. If it is, click on it and it should connect.

If your wireless network isn't detected, right-click on the network manager icon, select Edit Connections, click on the Wireless tab and select to Add a new network.  Fill in the relevant details and hopefully your network will be enabled and connected.
