Hey Steff!

There is a lot in those few sentences.

I won't ask how you did it (yet), so:

1) the first goal is to save your data
2) the second is to get your system working again.

Right now you cannot shut down?  

If you can see the light on the external hard drive it is not flashing (no data being written), disconnect it from the PC.

This will separate your data from the messed-up install.

Then comes the hard part - figuring out how to save your Ubuntu install (or start over with a clean one).
