### Post by QIII on 2010-06-18
We are here to help, and we will.  But keep in mind that "laundry list" posts are often passed over by those who would otherwise be willing to help.

It is often best to start one thread per problem get that solved and move on to the next.  I recommend doing that.  Please include as much information as you can about each problem.  Also include the specs on your machine.

But briefly:

"First off, My firefox keeps closing just randomly."

Is it random, or can you tell us (in your new thread) what you are doing at the time, what other applications you have open?  What version of Firefox?

"I can't play any of the PC games that I buy:
Mostly because in WINE they lag like heck. Or you cant install them..?
I want to play GTA in WINE, but When I try to open it it just wont."

WINE is not part of Ubuntu.  It is an independently developed application that attempts to allow Windows applications to be run in what is known as a "loop device".  It is not a silver bullet and not every Windows application can be run on it.  They have a database of what people have been able to use, what they have not and everything in between.  You can start some research on what will work on it here:

[http://www.winehq.org/](http://www.winehq.org/)

"I have tried to dual boot with VMWare and Virtual Box. And have failed  twice."

Using a virtual machine is not dual booting.  In your new thread on this subject, please let us know which version of VirtualBox you are using, what you did to install it, how you attempted to create virtual machines and how you attempted to install the OSs.

"And now my VMWare that I spent 30$ on, won't even open."

Again, provide details, or we can't begin to help.

"Should I update to 10.04 or whatever?"

Perhaps.  But it would be best to solve the issues you have been having first.

"Or is there a way I can get my old windows xp back..?"

Not if you installed Ubuntu over it.  You would need to reinstall it.

"Kinda really hating this OS."

All of the things you have talked about are applications, not the OS itself.
