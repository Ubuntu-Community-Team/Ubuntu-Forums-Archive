### Post by mcduck on 2009-08-08
Sorry, you can't.

Well, if you download the source code for gnome-panel you might be able to edit it to get rid of the panel handles, but apart from that it's not possible.

Without the handles you wouldn't be able to access the panel's settings if the panel is set to not expand to whole screen width. There wouldn't be any place to click in the panel itself, you'd only end clicking panel applets instead of the panel itself..
