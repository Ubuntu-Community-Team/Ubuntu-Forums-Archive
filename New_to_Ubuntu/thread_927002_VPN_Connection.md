### Post by t0p on 2008-09-22
I use a java app called **Your Freedom**.  YF isn't available through the repos, but you can get it from [www.your-freedom.net]("http://www.your-freedom.net").

**This is what to do:**

**1.**  Go to Synaptic (**System > Administration > Synaptic**) and install **OpenVPN**.

**2.**  Now go to [www.your-freedom.net]("http://www.your-freedom.net") and download the java app **Your Freedom**.  Also download the YF Users Guide pdf.

The instructions for YF are self-explanatory.  But if you have any problems or questions, just ask.
