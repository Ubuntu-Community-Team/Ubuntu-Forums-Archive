### Post by finer recliner on 2008-06-26
there is a plug-in for compiz fusion that allows you to do something like this. its called maximumize. you need to install the advanced options for compiz (CCSM) and enable it from there. 

PS: be careful with it, i once accidentally applied it to one of my gnome panels, which made it maximize to take up the desktop area. i had a fun time figuring out to fix it...
