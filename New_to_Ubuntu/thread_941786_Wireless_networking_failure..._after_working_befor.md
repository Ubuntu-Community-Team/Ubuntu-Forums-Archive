### Post by SalishSea on 2008-10-08
I run 8.04 with a wireless card to a router in the basement of my house.
A few days ago i stopped being able to communicate with the router  and while I can get it going again, the network manager interface is "not like before". Here are some of the symptoms.

The network manager icon has shifted from a "signal strength" icon to an icon showing two monitors. I no longer can access VPN sites I had defined, and I am unable to save WPA passwords for my router.  Also, and this I find quite confusing, when I click on the network manager icon, I have two options "edit wireless networks" and "Manual network configuration". Both seem to want the WPA password, but neither seems to save it.   

I am not sure what changed, when it changed, was some unanticipated upgrade anomaly or did I (or the kids) do something unexpected?

Suggestions welcome.

### Post by talsemgeest on 2008-10-08
Try going to the network manager, selecting your wi-fi and clicking properties. Then make sure it is set on "roaming".
