### Post by bobnutfield on 2008-09-17
A quick google seems to indicate that you may need to use ndiswrapper with that dongle.  If you are not familiar with it, it is an app that will use Windows drivers to run a wireless adapter.  I saw no indication of native linux support, but did find other posts who had it running successfully with ndiswrapper.
