### Post by gn2 on 2008-10-20
> **Uchiha_madara said:**
> I understand from you there is no Live Cd With xubuntu>>>!!

There is a Live CD for Xubuntu.

The reason to use the Alternate (install only) CD to install from is that it requires much less RAM to run it than the conventional Live CD.

A Live CD has to run the Live session and the installer at the same time, to do this newer versions of Xubuntu, Ubuntu and Kubuntu require a minimum of 384mb of RAM installed.

There are Alternate Install CD's for Ubuntu and Kubuntu as well, but with only 256mb RAM fitted in your machine, Xubuntu is probably the the best option because performance should be better as Xubuntu's the lightest of the three.
