### Post by jrothwell97 on 2010-05-01
You should be just fine.

Ubuntu's package manager automatically calculates dependencies for each application in the OpenOffice.org suite, and so if you remove one program, you can be pretty certain the other programs (and their dependencies) will remain intact.

Good luck.
