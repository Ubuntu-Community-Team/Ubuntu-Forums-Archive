### Post by dFlyer on 2011-06-21
Before you do anything backup the data you want to keep, or make a clone of your drive, I prefer the clone. Should you decide otherwise that you don't like linux it makes restoring your windows system a lot easier. Before you install anything boot off the live cd to make sure everything works. If it does than install it along side windows in a separate partition for dual booting. Also remember that Linux is not windows, so don't expect it to behave or work the way windows does. Just take it slow and most of all have a lot of fun, if you have problem ask for help by providing as much information as possible. After a while should you decide you want to dump windows you can.
