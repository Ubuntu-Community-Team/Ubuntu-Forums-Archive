### Post by earthpigg on 2009-07-10
get a 32-bit CD :D

32 bit cd only installs 32 bit, 64 bit only installs 64 bit.

there are several ways you can get ubuntu:

[http://www.ubuntu.com/getubuntu](http://www.ubuntu.com/getubuntu)

make sure you select "32 bit" under "Computer Architecture."

follow the directions on the website for whatever method you choose to get ubuntu. downloading and burning a CD-R is probably the quickest way.

having them mail you one is probably the easist.

im not leaving you out in the cold by telling you to read the website's directions -- they really are _**that**_ good. :D


if you already know how to burn an .iso, then you have nothing to learn.
