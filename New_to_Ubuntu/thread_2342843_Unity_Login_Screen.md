### Post by tuxxllo on 2016-11-10
New to 16.10, I've noticed that Ubuntu is now utilizing the users home wallpaper for the login wallpaper. How can I go back to the default wallpaper.... JUST FOR THE LOGIN ONLY. I still want to use my custom wallpaper for my desktop. Is this even possible? Thanks in advance for your help.

LH

---

### Post by deadflowr on 2016-11-10
Change the permission setting for your Pictures folder to None for Others.
(right click on pictures, select properties, change to permissions tab, toggle Others from Access files to None)

Don't think you need click  *change permission for enclosed files*. As denying read access to the Top level/Pictures should suffice)
