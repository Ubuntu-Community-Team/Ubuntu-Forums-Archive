### Post by ajgreeny on 2015-04-24
Can you right-click in the tab-bar, or in the top of the screen and reselect the navigation toolbar, which is what appears to be missing.

If that does not work try renaming the configuration folder in your home called .mozilla as .mozillabak and then try again.  You can get all your bookmarks etc etc from the old .mozillabak folder if you need to which is why it is best to rename, not delete the folder.

Also, if you click the hamburger icon (three horizontal lines) choose customise and then bottom left is a toolbar menu used to add in or remove toolbar views.
