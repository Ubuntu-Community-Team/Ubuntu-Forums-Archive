### Post by dv3500ea on 2010-05-04
Install Flash Player:
Go to System -> Administration -> Synaptic Package Manager
search for adobe-flashplugin
click on the box next to it
click mark for installation
click apply (the green tick)

That particular game cannot work on ubuntu ([see here]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=11213"))
