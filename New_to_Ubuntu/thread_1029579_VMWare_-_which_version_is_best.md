### Post by scorp123 on 2009-01-03
I find VMware's current products very very buggy and their quality quite lacking  ...
[LIST]
[*]VMware Server 1.0.x has issues when used on 64-bit Linux (just search the forums here ...)
[*]VMware Server 2.x is slooooooow as a snail and the new web interface is something I can do without ... no thanks!
[*]VMware Workstation 6.x costs a lot of money ... and it's buggy too: the keyboard doesn't get properly recognised, you have to edit a config file or else e.g. the cursor keys won't work in your VM (... how ridiculous is that ???) ...
[*]VMware ESXi is free but also castrated and can't do much
[*]VMware ESX has very interesting features (e.g. VMotion) but it's $$$ expensive $$$ !!!! and only works on certain hardware ...
[/LIST]

I use VirtualBox 2.1 now and I am happy. It's not bug free either, no. But Sun lately produces so many releases with new features and lots of improvements lately that I simply feel better using VirtualBox: 2.0.2 was in October 2008, then in November 2008 came 2.0.6  ... and since December 2008 we have VirtualBox 2.10 ... That's one release each month.
