### Post by halitech on 2008-12-23
running 2 operating systems will not affect your speed in whichever you are running as only 1 can run at a time.

wubi will work but will not give you the full advantage of running in a native ext3 file system so you will notice some speed difference there.

You should always burn things at the slowest speed possible to prevent errors when burning. You shouldn't need to burn it to dvd unless you downloaded the dvd version.

If  you plan on only using xp seldomly then look at Virtual box or other virtual machine type software to run windows inside linux.
