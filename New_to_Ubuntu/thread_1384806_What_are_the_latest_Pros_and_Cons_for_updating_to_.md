### Post by -kg- on 2010-01-18
I personally have had no problems with the upgrade, but I haven't had many problems with any of the versions I've installed.  BTW, I always do a clean install.  I tried an upgrade with Karmic, but it didn't work too well, so I clean reinstalled it and it works perfectly.

All you need to do is back up the data that you want to save.  Of course, you'll need to reinstall the software, codecs, drivers, and all after you do the install (that is, if you want to upgrade).

Karmic works perfectly for me, but I realize that it won't work for everyone on every computer in every situation.  A good way to tell if Karmic will work is to run it from the Live CD.  If everything works as you wish, then (if you have room) you might install Karmic side by side with Jaunty and set it up the way you like.  If at that point you decide to go with the upgrade, you have it already installed and in operation and, with a little work, you can get rid of Jaunty and use Karmic.

Of course, as has been said, if you're perfectly satisfied with Jaunty and don't want to go through the hassle yet again with the next version (Lucid Lynx, which will be an LTS release and worth the hassle) then you can use Jaunty until Lucid's release time.

Karmic has some added features and capabilities, few of which I use.  GRUB2 is one I use, and it's pretty nice from what I've used so far.  But unless you're an experimenter, it's not absolutely necessary to upgrade at this time.  You *will* want to upgrade to Lucid, though.  Jaunty's support period will run out sometime just before the middle of next year ([according to this chart]("http://www.ubuntu.com/products/ubuntu/release-cycle")).  According to the same chart, Lucid's cycle will last until near the middle of 2013.

It's all up to you.  If you're the experimenting, "cutting edge" type, go for Karmic.  It seems to be a good release to me.  But if you're perfectly happy with what you have now and it works good for you.  Then continue with it until Lucid is released.  That will be the next LTS release and (assumably) should be a "stability" release that will last for the next several years, instead of the 18 month support cycle of the normal releases.
