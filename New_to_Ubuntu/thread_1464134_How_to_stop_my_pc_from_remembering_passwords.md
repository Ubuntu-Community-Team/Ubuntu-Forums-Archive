### Post by furoido on 2010-04-27
Everytime I open the yahoo mail page, my pc automatically fills in my secret password...how do I stop it from doing this without making it forget my other passwords for other sites?

### Post by 2hot6ft2 on 2010-04-27
Try clearing the password box and hitting enter like you're going to login.
It should ask if you want to change the password that is stored.
Click on the one that says yes you want it to.
Obviously you wont login but the password will now be set as a blank.
Then login again and use your password.
It should ask again. This time select Never for this site.
