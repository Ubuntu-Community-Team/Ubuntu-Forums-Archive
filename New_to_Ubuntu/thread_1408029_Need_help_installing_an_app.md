### Post by dmaxel on 2010-02-15
I don't know about you, but in my Karmic system I am able to see the package. Try enabling Universe, Restricted, and Multiverse repositories by going to **System --> Administration --> Software Sources**. Also, when looking through Synaptic Package Manager, don't forget the g in libgconf2.0-cil.
