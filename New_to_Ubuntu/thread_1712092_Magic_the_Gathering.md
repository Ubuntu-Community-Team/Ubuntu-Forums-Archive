### Post by An Sanct on 2011-03-22
Virtualbox is actually a hardware emulator, you will have to have  windows installed inside the virtual machine to run anything and even  so, virtual machines are not made for gamers (3d support is extremely  limited)

Try using wine for this...

If you already downloaded the helper, find the it, right click the file, select properties and make sure it has the executable bit set.

Now it should open in wine.
