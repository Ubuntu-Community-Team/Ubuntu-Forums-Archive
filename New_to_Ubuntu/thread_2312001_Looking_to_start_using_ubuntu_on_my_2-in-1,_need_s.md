### Post by Vladlenin5000 on 2016-02-01
Hi and welcome.

ASUS TP200SA isn't as much "Linux unfriendly" like some of its Intel "Bay Trail" CPU based predecessors therefore you should be able to install Ubuntu (dual boot or standalone) pretty much the same way you would do with any other x86_64 desktop or notebook. There are, however, two component that can be troublesome: Audio and WiFi. 
I suggest you boot a live session (no installation) and check what's working and what's not.

### Post by gordintoronto on 2016-02-02
One tip about Linux: we generally don't install drivers (or applications) the way you do in Windows. The system comes with a huge number of drivers. Some Wi-Fi adapters are a problem. We install a printer, and usually the driver arrives by magic.
