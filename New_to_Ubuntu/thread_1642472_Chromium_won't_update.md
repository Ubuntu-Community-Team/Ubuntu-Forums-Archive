### Post by Ahunt on 2010-12-10
If you are running Chromium 10.0.605.0 then you are signed up for daily builds, which is why you keep getting new ones every day. [Launchpad]("https://launchpad.net/ubuntu/+source/chromium-browser") shows the current stable version in the Ubuntu repositories, which is currently 8.0.552.215, the same as Google Chrome's current stable branch version. If you remove the daily builds version and install Chromium from the Ubuntu Software Centre you will get the stable version instead and a new version will only show up through the normal updates when the stable branch is updated, usually every four to six weeks.
