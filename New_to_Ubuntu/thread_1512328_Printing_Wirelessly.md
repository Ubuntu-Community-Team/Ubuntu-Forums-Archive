### Post by Windows Nerd on 2010-06-23
> **nhasian said:**
> make sure the printer is shared from the windows machine and then add the network printer from system-> administration-> printing
Then go to server>new>printer. It will then search for printers on your network. If it sees the printer on your network, it will show  up in the popup box on the left-hand side. Hit foward. Then it will search for availible drivers installed both on your system and elsewhere. If there is no drivers for your printer Make/Model, then you will have to search on the internet, and download + install the PPD file relative for your printer make/model. Otherwise, if it is listed, select it and hit foward, and your printer should be all set up!
