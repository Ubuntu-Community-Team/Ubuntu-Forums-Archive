### Post by buzzingrobot on 2015-08-03
The updater compares the version numbers of software packages installed on your system with the versions available in the software repositories your system uses.  If the corresponding packages in the repositories are newer, then they are downloaded and installed as updates, along with any dependent packages that may require updating.

What's updated on a system, then, depends on what is already installed on that system. Some updates may trigger the installation of a number of dependencies, and then those dependent packages may also have their own dependencies that need to be updated, and so on.
