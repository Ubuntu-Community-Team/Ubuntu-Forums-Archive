### Post by crispy_420 on 2011-06-01
The problem is this: P4 + 512MB. Sorry.

With that hardware you can expect a video to encode in double the time it takes to play. On my older box (P4 + 2048MB) it is close to real time so an 8 minute video finishes in just over 8 minutes. The problem is transcoding, or converting formats, that takes all that horse power. So to speed it up would not be cheap.

Maybe staying in the same format might speed it up, maybe?
