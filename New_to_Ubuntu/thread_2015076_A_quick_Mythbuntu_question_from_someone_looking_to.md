I am a long time mythtv user (~6 years) and just updated my maching to Mythbuntu 12.04. Over the years, the technical knowledge required to install and setup mythtv has declined. Once set up, technical knowledge is not required for daily use. 

Installation and setup is graphical, which lowers technical requirements significantly. Compatible hardware (i.e., hardware that works out of the box) will lower the technical knowledge required to set up a mythtv machine. 

Technical knowledge of varying levels is needed when: 
[LIST=1]
[*] A problem arises
[*] System information is wanted that the default software does not present graphically (e.g., want to know what processes are running in the background)
[*] A feature is wanted that the default configuration does not provide but can be accomplished by installing additional software through the Ubuntu software repository (e.g., import DVD that you own or use LVM instead of storage groups).
[/LIST] 

Note: I used to use LVM; however, storage groups (now the default in mythtv) is easier.

Minimum skills required:
[LIST]
[*] Willing to learn new technology
[*] Willing to troubleshoot/solve problems
[*] Willing to search the Internet for mythtv information
[*] Willing to read mythtv specific (and related) websites, forums, and documents
[*] Willing to seek help (i.e., post a message) after extensive searcing does not solve a problem
[*] Perseverance (i.e., your problem is unlikely unique and a solution is out there)
[*] Willing to copy and paste commands into a terminal if/when necessary
[/LIST]

Have fun!
