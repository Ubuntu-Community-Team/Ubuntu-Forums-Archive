### Post by binaryburnout on 2009-08-10
ChatZilla is a nice Firefox extension for IRC. I prefer [irssi]("http://www.irssi.org/") for my IRC needs. If you are looking to familiarise yourself with command line oriented applications, irssi will be nice. It is also very easy to install...
```
sudo apt-get install irssi
```
Or you can open up Synaptic and install it that way. The irssi site has plenty of tutorials on it for learning it and remember, Google is your best friend. I assure you if you run into problems, someone else has had it before. This is where Google becomes a very good friend. Also never be afraid to ask questions, After reading your other post about your recent misfortunes with linux, I assure you, you will quickly learn that the linux comminuty is always willing to help, teach and answer questions.
