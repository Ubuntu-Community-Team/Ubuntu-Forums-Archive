### Post by sikander3786 on 2011-06-10
As your profile tells us you are using Ubuntu 11.04. If using Unity, be careful while playing with CCSM or you might end with a broken system.

For the extra effects, you need the compiz-fusion-plugins-extra package.

Search Software Center for the package named above or from Terminal:

```
sudo apt-get install compiz-fusion-plugins-extra
```
