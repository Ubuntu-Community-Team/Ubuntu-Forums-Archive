### Post by yetiman64 on 2011-01-21
As mastablasta notes the card should show in Places > Removable media.

If f-spot no longer automatically opens for photos. Go to any folder (your home folder will do) > Edit > Preferences > Media tab, select the arrows on the right of the Photos setting and select the action required (Open F-Spot, for you).
