### Post by lswb on 2009-01-05
There are a few addons for firefox that may interest you. I use one called "Session Manager" It is configurable with various options and can reopen multiple tabs after a firefox restart. It also keeps track of open tabs while firefox is running and can restore them after a firefox crash and restart.

If you want to go the clipboard route, check the program "glipper" (or "klipper" for KDE)

### Post by -kg- on 2009-01-06
Another possible way around the issue:

Before you close Firefox, bookmark the page you were on.  Then you will be able to go into bookmarks and go right to the page you were on.  If you only require the bookmark to return where you were before you closed Firefox, then when you're done with it, just delete it.
