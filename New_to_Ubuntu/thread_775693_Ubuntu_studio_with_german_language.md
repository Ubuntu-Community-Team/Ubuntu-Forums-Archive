### Post by forrestcupp on 2008-04-30
I don't know if they have a German version of Ubuntu Studio, but you can install the English version then add German support.  To do that, you can open a terminal and type:
```
sudo apt-get install language-support-de language-pack-de
```and you will have German support for spell checkers, your desktop, and most of the apps.
