### Post by 3rdalbum on 2009-12-08
> **Thrasonic said:**
> Excellent info Phil!  It totally worked.  So a .deb file is like a .exe in Windows apparently.

No, nothing like it, actually. A ".exe" is a Windows program. Every program on your Windows system is a ".exe". A ".deb" is not a program - it's a package that the system can use to install a program.
