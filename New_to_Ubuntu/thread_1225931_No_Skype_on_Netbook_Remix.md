### Post by joris1977 on 2009-07-29
I am not sure you can install new software during a live session. 

If you decide you like ubuntu and give it a try. It is good to know that instead of installing software from a website it is good practice to take it from a trusted repositary. This is better because programs will be updated when there are security fixes.

Skype is not open source so it is not included in the ubuntu repositaries, but you can install it from the medibuntu repositaries. You probably want to use this repositary anyway, because it has some non-free codecs for playing media.
