### Post by AlphaLexman on 2011-01-03
Actually the problem is that there is nothing to do if the conditional is met:
```
if test $kpatchlevel=4; then
#alsaconf
fi
```Just having a comment after the 'then' statement won't work. For simplicity, just add an 'echo' before the comment like this:
```
if test $kpatchlevel=4; then
echo
#alsaconf
fi
```
...and save the file.

options snd-hda-intel model=specify_your_modeltype_here
