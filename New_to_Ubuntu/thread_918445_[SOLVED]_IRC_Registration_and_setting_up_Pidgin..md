### Post by t0p on 2008-09-13
As you want to use pidgin to access irc, here's a quick HOWTO.

1. Open pidgin.

2. **Accounts > Manage > Add**.  

3. Now select: **Protocol = IRC**; 
                **Screen name = whatever**; 
                **Server = irc.ubuntu.com**; 
                **Password = leave blank**; 
                **Local alias = whatever**;

4. Click **Save**

5. In the **Accounts** window, make sure your new irc account is marked Enabled, and Close the window.

6. In the **Buddies** window select **Buddies > Join a Chat**

7. Select your irc account.  You can click on Room List to get a list of channels, or you can type in a channel name if you know it.  Since we're using irc.ubuntu.net for this example, let's go to the channel called **#ubuntuforums** - type #ubuntuforums into the channel field. Click on Join.

8. A window called **#ubuntuforums** will appear - now you can chat!

There's a lot more to using irc - an excellent guide is available at [www.irchelp.org]("http://www.irchelp.org").  In particular, there are commands to learn, and other servers to explore.  Pidgin isn't the greatest irc program in the world.  I prefer **Xchat**.  But it'll do to get you going.
