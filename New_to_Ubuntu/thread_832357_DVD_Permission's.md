### Post by billgoldberg on 2008-06-17
What codecs do you have installed?

----

The easiest way to enable dvd playback is:

install medibuntu repo (check "media" link in my sig)

Then install the package "libdvdcss2" (this is also explained in the "media" link)

or install vlc then.

If you don't choose to use vlc, then you will also need to install some other packages.

The easiest way to get them is by install the package called "ubuntu-restricted-extras".
