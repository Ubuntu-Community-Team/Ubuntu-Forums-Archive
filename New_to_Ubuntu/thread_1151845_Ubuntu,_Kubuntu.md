### Post by mcduck on 2009-05-07
Ubuntu uses Gnome desktop (by default), Kubuntu uses KDE and Xubuntu uses XFCE.

Apart from the default desktop environment they are all the same OS, and different desktop environments can be installed on any of them (if you install more than one desktop environment you can select which one you want to use at the login screen).

Gnome: [http://www.gnome.org/]("http://www.gnome.org/")
KDE: [http://www.kde.org/]("http://www.kde.org/")
XFCE: [http://www.xfce.org/]("http://www.xfce.org/")

Desktop environment: [http://en.wikipedia.org/wiki/Desktop_environment]("http://en.wikipedia.org/wiki/Desktop_environment")

Pretty much a question of personal preference.
