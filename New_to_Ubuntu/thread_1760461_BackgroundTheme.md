### Post by compmodder26 on 2011-05-16
Yes, download the theme you want from gnome-look.

Then click on System -> Preferences -> Appearance, this will display the &#8220;Appearance Preference&#8221; window. On the "Themes" tab, click the "Install" button near the bottom.  Navigate to the theme you downloaded and open it.  It should then install and you can select it as your new theme.
