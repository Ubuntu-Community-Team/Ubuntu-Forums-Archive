### Post by Astnya on 2010-10-15
It ought to be as simple as exporting your contacts list to a file on your computer, then importing that file into Evolution.

Exporting the contact list is done differently according to each email service, but it should be pretty simple for each. Just look around for "Import" or "Export" options while browsing your contacts list. You ought to be able to export to a .csv file; if not, a Vcard might work, if that option is present.

Importing into Evolution is done by going into Evolution and hitting File > Import and choosing to import a single file.

### Post by bennachie on 2010-10-16
Nothing to do with email export and import is ever quite as easy as our responses may make it seem. What version of Thunderbird are you using? You'll find plenty of Ubuntu users who believe that Thunderbird 3 is a better email client than Evolution (and Thunderbird 3 would be the client of choice for Kubuntu users, given that KMail is not exactly a leading edge package at the moment). However, Evolution does provide a nicely integrated overall productivity package, where Thunderbird relies on add-ins to carry out the same tasks. You'll need to install a couple of those add-ins so that you can export mail folders and addresses from Thunderbird in a form that Evolution can import without any hassles, hence my question.
