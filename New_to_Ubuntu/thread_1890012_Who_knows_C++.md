### Post by anewguy on 2011-12-02
The very first thing you would need to do is install the package build-essential -> it contains all kinds of things needed for compiling on Ubuntu, including the compilers themselves.

The second thing -> be sure you have downloaded any extra header files specific to that developer.

The third thing -> think real hard if this is what you really want to do.  There are warnings in the code about how dangerous this code is.  Combine that with something that doesn't appear to have be maintained since 2004, and you have something that I personally wouldn't use.  If you need the functionality of that software, and it is very unique obviously, I would still search for another source for another supported program.

If you insist on continuing, be sure to read all the warnings in the source code.
