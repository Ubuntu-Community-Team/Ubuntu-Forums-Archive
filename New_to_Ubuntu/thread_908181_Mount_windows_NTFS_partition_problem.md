So...  as i can see, you have 3 NTFS partitions and only sda1 is not mounted... and probably there is not a file to be mounted on in folder media.. right? in terminal give ```
sudo mkdir /media/sda1

```  this will make the file where the partition will be mounted on.  then ```
gksudo gedit /etc/fstab

``` and add this line to the end of the file: /dev/sda1 /media/sda1 ntfs-3g defaults 0 0   

save and exit.  then give ```
mount -a
```
in terminal. hope this helps
