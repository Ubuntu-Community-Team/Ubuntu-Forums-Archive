### Post by NightwishFan on 2008-05-15
Yes you can easily do so with the command:
```
sudo aptitude install kubuntu-desktop
```
The downside is you will have menus cluttered with kde and gnome apps, but they work fine in either desktop environment, so it is of little consequence.
