I am confused that he is having trouble with this, and I think we need more information about what is going wrong. The usual procedure for this would be:


[LIST=1]
[*]Use the web browser to navigate to his email provider.
[*]Click on the "Compose" or "New Message" button to create an email.
[*]Enter a destination address, subject line, and whatever text he wants.
[*]Click the "Attach a file" button or link that should be somewhere on the screen.
[*]When the file selection dialog box pops up, navigate to and select the desired document and click "open".
[*]The document should magically upload to the email provider.
[*]Click "Send".
[/LIST]
This process is identical on linux, windows, and osx. Since it's all browser based, it should be pretty much the same for him as what he is used to from any other operating system.
