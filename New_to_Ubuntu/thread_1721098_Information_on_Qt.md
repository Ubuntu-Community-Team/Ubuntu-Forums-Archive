### Post by rosencrantz on 2011-04-08
Language bindings are modules connecting between certain libraries (in this case those for the Qt GUI framework) and programming languages. Qt has bindings for major languages like C/C++, Python, Java, Ruby etc.
You can use any kind of C libraries by referencing their header files in your source code and adding their include directories to your compiler commands.
Do you have a lot of programming experience, including GUIs? 
If not, do basic C tutorials before you tackle libraries and GUIs. No need to worry about language bindings yet.
