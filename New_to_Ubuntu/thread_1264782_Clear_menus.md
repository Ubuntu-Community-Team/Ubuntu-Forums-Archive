### Post by mcduck on 2009-09-12
What do you mean with "clear menus"?

edit:
If you want *transparent* menus, you need to add the following lines to the Opacity/Brightness and Saturation-plugin (andnthen set the opacity to whatever level you want):

```
type=Menu | type=PopupMenu | type=DropdownMenu
```
