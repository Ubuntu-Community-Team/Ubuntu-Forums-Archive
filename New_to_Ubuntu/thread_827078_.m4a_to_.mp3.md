### Post by DirtDawg on 2008-06-12
Hi. Run from the terminal, this script converts ALL the m4a's in a given directory into mp3's. I would recommend making backups just in case(I don't remember if the script makes copies or completely replaces the original files). You may need the faad and lame packages from synaptic.

```
for i in *.m4a
do
  faad -o - "$i" | lame -h -b 192 - "${i%m4a}mp3"
done
```

Also, I didn't write this, but it's been so long I can't remember who did. Kudos to them, though!

EDIT: You can change the quality by changing the number '192' to the desired bitrate (ie: 256).
