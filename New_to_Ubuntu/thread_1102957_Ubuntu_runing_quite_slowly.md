### Post by snowpine on 2009-03-22
A ram upgrade would help a lot (I always recommend 512mb minimum for Ubuntu) but the bottom line for an old laptop is that Ubuntu is one of the slowest choices. I personally use and recommend Crunchbang Linux, which is basically a minimal install of Ubuntu plus the Openbox windows manager, or SliTaz, which is very fast, but isn't based on Ubuntu at all.

If your heart is set on sticking with Ubuntu, then definitely disable anything you don't need: desktop effects, services, auto-started applications, etc.
