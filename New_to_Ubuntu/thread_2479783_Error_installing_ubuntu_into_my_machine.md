### Post by Impavidus on 2022-10-07
I hope the tutorial you followed (everybody is free to publish his own tutorials) told you to make backups of your important files. I'll assume you've got those backups.

Is there just a single hard drive in that laptop? Then the easy way to install Ubuntu is to tell the installer to wipe the drive and install Ubuntu. This will do exactly as it says. The result may not be ideal, but it's acceptable. If there are multiple hard drives, like one spinning drive and one ssd, you have to deal with it manually.

Ubuntu can run faster than Windows on the same hardware. If the laptop has Windows 8 or 10 and it's too slow, replacing it with Ubuntu will make the computer more usable. If this laptop still runs Windows 7, a light flavour of Ubuntu will be better.

Last but not least, remember that Ubuntu is not a drop-in replacement for Windows. Not by a long stretch. Most Windows applications won't run on Ubuntu.
