### Post by JillSwift on 2008-09-06
Hi!

To change the way Thunderbird displays your e-mail, have a lookie:

[LIST=1]
[*]**Edit --> Preferences** (It might be Tools --> Preferences if you didn't use the repositories to install Thunderbird)
[*]**"Display" icon, Formatting tab.**
[*]**Near the bottom of the dialog window, there's a button "Fonts...".**
[/LIST]
The window that button pops up will let you define the fonts used and their size.
Hope this helps!
