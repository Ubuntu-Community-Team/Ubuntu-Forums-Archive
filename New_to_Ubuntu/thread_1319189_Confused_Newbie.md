### Post by yeats on 2009-11-08
TIP:  The forums will work better for you if you address one problem per thread.  Loading up a thread with many questions is usually a good way to not get an answer :-). 

First of all, read this: [https://help.ubuntu.com/community/InstallingSoftware](https://help.ubuntu.com/community/InstallingSoftware)

> Q#1: how do i get SWF files to work through Mozilla? I have used the built in search and install plug-ins in firefox, and each of the 3 options failed through the installer. Reason being: Error: Dependency is not satisfiable: libnspr4-dev Now i have no idea what so ever that means.


When possible, you need to install software through the Synaptic Package Manager or Ubuntu Software Center (both do the same thing, basically).  Ubuntu software exists in web-based repositories and the software is organized so that when you download and install it, the dependencies are automatically included.  The Ubuntu package for the Adobe Flash Player is called flashplugin-nonfree.  


> I have read there are some scripts that can be used through terminal, but i am usure as to specifically what they are, and unfamiliar with terminal itself which i guess is linux version of MS-Dos or "RUN" but i still am unsure about the "lingo" etc.. so if anyone can reccomend a guide if one exists!

The most common way to install software from the command line is with a program called apt-get.  Use it this way:

```
sudo apt-get update #which updates the local cache of program names
sudo apt-get upgrade #which applies all available software upgrades (for individual programs)
sudo apt-get install *program* #installs an individual program (substituting the actual program name for *program*
sudo apt-get remove *program* #uninstalls an individual program
sudo apt-get purge *program* #uninstalls and removes all config files for an individual program
```
