### Post by Kid Charlemagne on 2009-03-24
I set a password to log on to my computer, which is running Ubuntu 8.04. It asks for the user name and password, but I noticed if I just wait a few seconds it logs on automatically. Is there some way to make it require the password before letting me log on?

---

### Post by kanikilu on 2009-03-24
Go to System > Administration > Login Window > Security (tab). Make sure "Enable Timed Login" is not checked.

---

### Post by Kid Charlemagne on 2009-03-25
Thanks. I unchecked "Enable Timed Login", and the problem was solved.
