### Post by Zilioum on 2010-02-21
I did the same thing than you just a month or so ago. I often found solutions by just googling my question. But here are some good guides I found:
[Ubuntu Server Guide]("https://help.ubuntu.com/9.10/serverguide/C/index.html")
There are a lot of good tutorials inside there, file sharing, printer sharing etc. Altough for file sharing I used another/better tutorial, but its in german (If you want to set up a samba share I can help you with that). Its a good idea in general to have a look at different tutorials, in some tutorials for example they have you do stuff you dont really need to do for your specific setup, so it helps to check.
If you need help with something in particular just ask.
