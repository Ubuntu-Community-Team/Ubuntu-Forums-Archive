### Post by C_M_L on 2011-03-06
I'm posting this for anyone who has had CD installation issues as I have had in the past day. I burned 4 CD's with the ISO image only to get errors at nearly the same point of installation. After the first two I thought I had a corrupted ISO file so I downloaded it again. To no avail, the next two crashed the installer at the same point with the same error message "I/O" issues and clean your CD. Well, the CD's were new and did not need cleaning. Back to square one. The Ubuntu site suggested a free CD/DVD burner program that I thought would be worth a shot. I had been using Nero to create a CD but this program allowed me to use a DVD and Nero did not. The install was flawless, faster, and no error messages.

So, if you are having installation issues with a CD, try this. And by the way, I have not used any Linux product for nearly 5 five years simply due to the lack of media compatibility. I am truly impressed with Ubuntu. It handles media files quite well and will do everything I need it to do. My only complaint so far is the Flash Player implementation..better than Mac but not quite as good as Windows. 

Just don't give up the ship simply do due a few bad installs. It is an OS well worth the time and energy to install.
