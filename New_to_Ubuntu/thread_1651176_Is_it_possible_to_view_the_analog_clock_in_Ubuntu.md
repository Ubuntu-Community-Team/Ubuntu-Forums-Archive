Hi Friend,

It is possible and works like Windows. Perhaps you have not yet set up your "home" location that's why it's not yet showing. Here's how it should look like. I've set up multiple locations so you can also see that it allows you to select your "home" (or current) location whenever you move.

[IMG]http://ubuntuforums.org/attachment.php?attachmentid=179178&stc=1&d=1293063703[/IMG]

To set it up, just right-click on the date/time and select "Preferences". You will see a window like this:

[IMG]http://ubuntuforums.org/attachment.php?attachmentid=179179&stc=1&d=1293063754[/IMG]

In the "Location" tab you can add locations. In the "General" tab you can change settings like 12-hour or 24-hour display, showing the second-hand, etc. In the "Weather" tab you can change whether you want to use Celsius or Fahrenheit. Hope this helps. :)
