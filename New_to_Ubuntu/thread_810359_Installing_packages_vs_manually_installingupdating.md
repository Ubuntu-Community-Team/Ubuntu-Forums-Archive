### Post by sayakb on 2008-05-28
Well, the packages in the repos are well tested and stable ones. Just for example, I downloaded a newer version of screenlets from getdeb.net, but it had a problem in the system monitor screenlet in showing the CPU4 graph. If you want to test new software, either check the backports/preleased repos, or download/install beta software. But do create a separate installation of ubuntu on a separate partition so that if in case it gets screwed, you could simply go for a clean install.
