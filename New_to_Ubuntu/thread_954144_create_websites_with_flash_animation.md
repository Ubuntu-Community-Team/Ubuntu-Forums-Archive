### Post by nhandler on 2008-10-26
Sadly, I do not think there is any program that you can run natively in Ubuntu which will allow you to create flash animations. I would suggest either using wine or a virtual machine in order to run some version of Adobe Flash. It appears that wine has good support for adobe flash, [http://appdb.winehq.org/objectManager.php?sClass=application&iId=23](http://appdb.winehq.org/objectManager.php?sClass=application&iId=23).
