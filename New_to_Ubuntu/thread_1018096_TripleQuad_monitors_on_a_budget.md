### Post by Kellemora on 2008-12-21
Hi LB

I don't really think you want to go with the Hub Route.

Right after Windows XP came out, some dude at a trade show was running 16 monitors from a small laptop, showing his advertising.
I do know that XP will support up to 32 monitors, somehow.

The following year, another dude had 12 monitors running, but in a different way, all the monitors combined were showing a single show that was spread out across all 12 monitors, more like Large Screen.  He too was using XP but with some type of Hub that cost like 5 grand.

Things are a little easier to do today though!

Nvidia makes QUAD pci video cards.
So, with two of those, plus the onboard Nvidia card, you CAN run 9 screens, no problem.  But to get all the screens to show a single thing spread out amongst them, you will probably need a software program like 9x our Ultramon, which are for the Doze.

I do know that in Linux there is a program called Xinerama, but I've never used it and don't know if it will allow spreading a single movie out over 9 monitors or not.
Xinerama is rated for multi-head video display, but that's the extent I know about it.

But at least you have a name now, to do some searches about it!
So I hope that helps give you some things to study about.

Good Luck
