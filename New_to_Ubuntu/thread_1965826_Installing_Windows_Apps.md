### Post by forrestcupp on 2012-04-26
First off you need to know that Ubuntu is Linux, and Linux doesn't natively run Windows software, just like Macs don't either.

But like philw said, Wine is your best bet. It's a Windows compatibility layer for Linux that allows you to run *some* supported Windows software, but definitely not all. ASA Prepware is not listed in their app database, which is a bad sign. Other people using older versions of Wine haven't gotten it to work at all. But it won't hurt anything to try installing it with the latest version of Wine.

If it doesn't work, you'll either have to dual boot with Windows, or install Windows in a virtual machine that you run from Ubuntu.

Good luck.
