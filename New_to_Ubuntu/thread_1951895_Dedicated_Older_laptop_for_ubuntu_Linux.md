### Post by QIII on 2012-04-03
"Main line" ATI driver support for that card is discontinued.  If you want to use an ATI driver, you would have to use the legacy driver, which I recommend against.  Just use the default driver.

That, of course, may hobble Unity so I would use Lubuntu or Xubuntu.  You'd get better performance out of one of them anyway.  Your memory is sufficient, but sufficient isn't always optimal.

Download and burn both Xubuntu and Lubuntu and run each in a live session.  They will run very slowly from CD (USB would be much better), but you will be able to check for compatability and your preferences.

I've installed Ubuntu and derivatives on machines with much lower specs, so I wouldn't worry too much.

### Post by sudodus on 2012-04-03
I would recommend that you download and try a few iso files and test them live (booted from CD or USB but without installing to the hard drive).

(Lubuntu and Xubuntu) (11.10 and 12.04 beta)

Since you have been programming before, I think that you can manage the new version, even if it is still in beta phase (it will be officially released at the end of this month). So test and find which flavour [LX]ubuntu and version 11.10 and 12.04 feels best and co-operates best with your hardware. And then you decide what to install to your hard drive.

The support for ATi graphics in linux has been rather poor, but it has improved during the last months. With this background I think that you might find it easier to get good graphics with 12.04.
