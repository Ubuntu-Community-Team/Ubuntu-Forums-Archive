---

### Post by Lars Noodén on 2012-02-24
Yes it can.  One way to do it is to install OS X then leave space on the disk for Ubuntu.  Then install Ubuntu in the free space.  I suppose it may be possible to resize an existing partition using the OS X installation CD, but I cannot remember.  Once Ubuntu is in place, install [rEFIt](http://refit.sourceforge.net/) to get the choice of systems when you start up.
