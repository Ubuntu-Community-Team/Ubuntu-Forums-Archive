### Post by SunnyRabbiera on 2009-06-04
Yeh the server edition does not come with a GUI by default, and with good reason.
GUI on servers can be troublesome, GUI slows down performance for the server so the server version is given a very spartan setup devoid of GUI.
But really why use the server edition if you are not running a server?
Besides the Desktop edition can be made to run a server but it will be laggy.
But one can get a GUI in the server edition too, but one has to learn a small bit of Ubuntu's commands such as apt-get.
