### Post by bodhi.zazen on 2009-03-11
The [UBT (Ubuntu  Beginners Team)]("https://wiki.ubuntu.com/BeginnersTeam") has been collaborating with [Ubuntu Classroom]("https://wiki.ubuntu.com/Classroom") and we would like to start offering educational opportunities to the community. 

We are proposing IRC sessions focused on specific topics of interest to the community with supporting documentation on the  Ubuntu Forums or Ubuntu Wiki. The goal of the IRC sessions is to enable a interactive teaching / learning format where the community can discuss issues and participants have an opportunity for Q&A.

We have some interesting topics we will be offering including packaging, bug reporting / triaging, using Launchpad, how to IRC, and Q&A sessions.

I am going to start with 2 IRC Q&A sessions on Security ( March 19th and March 26th , #ubuntu-classroom on Freenode or Ubuntu Servers , start 00:00 UTC)

For additional information on topics / sessions see :

[*][UFT Education Events]("https://wiki.ubuntu.com/BeginnersTeam/FocusGroups/Education/Events")

[*][Education Focus Group]("https://wiki.ubuntu.com/BeginnersTeam/FocusGroups/Education/")
[/LIST]

These sessions are open to all and, as this is new for the Beginners Team, we are open to feedback or suggestions.

### Post by ubuntu27 on 2009-08-02
> **bashirchacha said:**
> How can I admit myself in ubuntu education?

You have to use a [Internet Relay Chat]("http://en.wikipedia.org/wiki/Internet_Relay_Chat") (IRC).

There are several clients (software) that is used for IRC.

If you are using ubuntu (GNOME), you might want to use [X-Chat]("http://www.xchat.org/") or XChat-Gnome

To [install XChat-Gnome]("apt://xchat-gnome"):
```

sudo apt-get install [xchat-gnome]("apt://xchat-gnome")
```

To [install XChat]("apt://xchat"):

```
sudo apt-get install xchat
```


If you are using Kubuntu (KDE), then try "Konversation:
```

sudo apt-get install [konversation]("apt://konversation")
```.


Another option is to install an Add-on to Mozilla Firefox which will let you use IRC in Firefrox. I have not tried it, but I have read that it is user-friendly and ease to learn. Recommended for new-timers.

[https://addons.mozilla.org/en-US/firefox/addon/16](https://addons.mozilla.org/en-US/firefox/addon/16)


**NOTE**: xchat-gnome is a stripped version of the original XChat with less features and only the most common settings included (It follows the KISS principle).

A guide for using XChat can be [found here]("http://t0x.in/xchat.html")


Good Luck!
