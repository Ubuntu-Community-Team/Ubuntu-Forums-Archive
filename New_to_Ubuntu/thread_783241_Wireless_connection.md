### Post by Golem XIV on 2008-05-05
To change the default network that you're connecting to:

- Right-click on the Network Manager icon (on the notification area)
- Select Edit Wireless Networks
- Select the network you want to get rid of
- Click on Remove.

This will take it off the list and Network Manager will no longer try to connect automagically to it.
