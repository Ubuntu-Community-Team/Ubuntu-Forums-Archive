### Post by Neoncamouflage on 2011-07-01
What you want to do is open the terminal under Applications > Accessories. Then type in:
```
sudo apt-get install xchat
```

What this does is it tells Ubuntu to go to the repositories, great stores of software made specially for Ubuntu, download, and install XChat. After it does so, simply go under Applications > Internet > XChat IRC. It's then very simple setup, just type in your information and choose Ubuntu Server.

Wine allows you to download windows programs onto your computer and trick them into thinking they're running on Windows. It doesn't work for everything, but it's worth a shot. Download it the same way as XChat, but type Wine instead. If it wont' run your program, you may have to use a virtual machine, also very easy to set up using VirtualBox, so it's not a big deal if that's the case.
