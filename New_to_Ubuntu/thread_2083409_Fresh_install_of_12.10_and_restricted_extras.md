### Post by audiomick on 2012-11-12
I am not aware of any problems arising from the restricted extras, and so install them as a matter of course.

If you would rather not, why not just use the computer for a while and see what doesn't work (playing CD's, DVD's, Java, whatever...)? If you don't notice anything not working in your normal usage, then you can likely not bother with installing the restricted extras specifically.

I am afraid I don't know the difference between the option at install and the restricted extras package.

### Post by grahammechanical on 2012-11-12
It also installs proprietary video drivers from 12.04 onwards. Ticking that box used to pull in Microsoft core fonts but the user needs to accept the end user license to let the download proceed. People could not find the tick box because you need to scroll down the license page. I think that this caused some problems with the installation. 

I would not be surprised if that ticking that third party box only brings in a limited number of third party packages and that this is why it seems that Ubuntu Restricted Extras is not installed. The Software Centre includes MS core fonts in the package offered there.

Regards.
