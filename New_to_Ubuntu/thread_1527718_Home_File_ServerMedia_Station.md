### Post by cjtemple on 2010-07-09
Not to worry, you are not asking for too much. Check out Samba file sharing, this will allow you to share files with other linux systems, OS X, and Windows.

Here is a link to a how to on configuring Samba through the GUI (Graphic User Interface):
[https://help.ubuntu.com/community/SettingUpSamba#Samba%20Server%20Configuration%20-%20Graphical](https://help.ubuntu.com/community/SettingUpSamba#Samba%20Server%20Configuration%20-%20Graphical)

If you scroll down just a little more there are instructions for manual configuration.

Oh yea by the way, welcome to Ubuntu!!!
