### Post by Paqman on 2008-06-02
Update Manager brings you all the latest software in the repos.

If you want a more up to date version than something in the repos, you can try [GetDeb](http://www.getdeb.net/) where you can download .deb files for Ubuntu. These are not officially supported by Ubuntu, however, so haven't been properly tested to ensure compatibility with your system.

If you want a 100% stable system, stick to software in the repos.
