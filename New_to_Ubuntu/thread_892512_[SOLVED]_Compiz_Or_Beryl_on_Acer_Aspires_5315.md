### Post by tahina on 2008-08-17
The ~ in ~/.config/compiz/compiz-manager means your home directory. The following .config is hidden (as are all files or dirs that start with a dot.

You can choose to show hidden files in the file manager by choosing Show hidden files in the View-menu (or by pressing ctrl-h).

Maybe you all ready knew this, since I don't seem to have that file either, even if I show hidden files...

By the way, Beryl has merged back with Compiz. I don't know if there is any point in installing Beryl nowadays, since it's old and not developed anymore, as I understand it.
