### Post by bluefrog on 2008-04-23
open a terminal (apps/accessories/terminal) and enter the following command

sudo tasksel

you can try to run tasksel without sudo so you will see what it is without doing any harm to your machine.

Then you will be presented a choice of server programs to install. Select what you want with the space bar and up/arrow and press enter
