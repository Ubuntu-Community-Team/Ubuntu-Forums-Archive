### Post by Hospadar on 2009-06-03
It is possible to make a bootable usb from windows
[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)
you'll need to enter the scary world of windows command line, but that's not so bad really.

Scroll down to the "manual approach"
Looks like you'll need to get your hands on an ISO and syslinux for windows.
It also looks as though you'll need to get a cd drive emulator installed (like daemon tools would be a good choice, it's free) so you can actually copy the files off an ISO
