### Post by McMichael96 on 2010-05-05
You should have gave Ubuntu more memory. This "Memory" is Hard-drive space. Hard-drive space is what your computer uses to put app, pictures, videos, etc. Their is really no way to increase this hard drive space. I would uninstall wubi then reinstall and give Ubuntu around 20-35 GB of space if you have it.


Hope that helps. :D

### Post by 3rdalbum on 2010-05-06
If you install Ubuntu with Wubi, it is very easy to remove (you can remove it in Windows' Add/Remove Programs) and reclaim the space. So don't be shy about giving Ubuntu some room to move.

When you know you want to keep Ubuntu, then remove it from Windows and do a repartition and install using the live CD.
