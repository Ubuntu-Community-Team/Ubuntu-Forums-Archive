Its not a problem with Ubuntu but with the web browser. I presume you are using Firefox. Go to Edit->Preference->Content->Languages and add Bangla in the languages. 
Hope it solves the problem.

Alternatively, there is an Indian Browser Epic (based on Mozilla Firefox) which has native support for all the Indian languages.
