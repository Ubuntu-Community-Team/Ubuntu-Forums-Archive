### Post by vivekjustthink on 2012-09-11
Hi Nanur, 

I am also using ubuntu 12.04, 64 bit version. my problems is reverse to yours. i wanted to disable dwell click ("automatic click when mouse pointer is moved") since it was annoying me too much for over 2 hours. Finally i found the method to on/off Dwell-click feature. 

_To Enable dwell-click_
```
$ mousetweaks --dwell (put it in startup if u want it all the time)
```_To Disable dwell-click_
```
$ mousetweaks --shutdown
or
$ mousetweaks -s
```It works for me. If any other alternative or better solution, don't hesitate to put it here. i'll be happy to learn that :).
