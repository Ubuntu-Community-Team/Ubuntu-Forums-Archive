### Post by tvtech on 2008-04-26
WUBI  doesn't install linux to your hardrive as such,  it's kind of like vmware or parrallels it installs linux within windows  so you can use it but not to it's full capablity.  imho I'd just install cygwin and forgo using an entire OS to junk up my system if I was that stuck on windows.  it's just an easier friendlier way for people to get their feet wet with linux /bsd who can't afford a mac

### Post by Mazza558 on 2008-04-26
Rather than create a different partition for Ubuntu, Wubi installs Ubuntu as a program inside the windows filesystem, so it can be uninstalled just like any other program.
