### Post by yeats on 2009-02-14
The first thing to know is that Wine, while AWESOME, is not able to do everything you would want from Windows.  If you have a Windows box somewhere, or are dual booting, it will probably save you a great deal of time and trouble to just use Windows.

However, is it just the PDFs that you're looking to use?  You don't have to run the .exe files to access those, I'm sure.  See what you can open manually, and you may find that the files work well enough without needing to configure Wine.

And if this IS something you need to work with the .exe files, post back and perhaps some Wine wizard will respond with a smarter solution :-).
