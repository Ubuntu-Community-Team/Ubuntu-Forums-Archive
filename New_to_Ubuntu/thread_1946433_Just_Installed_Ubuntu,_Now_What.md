### Post by lovinglinux on 2012-03-31
Welcome to the forums. 

Wubi is safe to use for testing and you can easily uninstall it from Windows control panel, like you uninstall any application in Windows.

However, if you like Ubuntu and decide to keep it for long term use, then I would recommend that you remove Wubi and install Ubuntu as dual boot setup. You can get instructions on how to do that at [https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)

Wubi is fine for testing, but since it install the OS and all files on a disk image inside the Windows partition, it is more prone to corruption, which could lead to losing all data saved by Ubuntu.

Anyway, I am assuming you are able to get into the Ubuntu Desktop by now. 

Do you see this when you boot?

[IMG]http://i.zdnet.com/gallery/6352809-610-457.jpg[/IMG]

That is the Ubuntu Desktop when you first start it. You should see the Firefox logo in the Launcher on to the left.
