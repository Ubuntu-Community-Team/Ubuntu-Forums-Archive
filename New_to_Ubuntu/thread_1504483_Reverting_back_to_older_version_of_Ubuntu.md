### Post by Mark Phelps on 2010-06-08
> **KarinBee said:**
> ... Can someone tell me how he can revert back to the previous version he had installed.  Thanks.

Basically, you can't.  There is no rollback capability between Ubuntu versions.  You will have to save off your personal data and reinstall the older Ubuntu version, and then copy back your saved files.

### Post by PriceChild on 2010-06-16
Just back up everything you want to keep (like your office documents, photos and music) onto an external drive (or the ubuntu one service) then install Ubuntu again.

There's no need to "uninstall", the new install will overwrite it if you tell it to.
