### Post by Jose Catre-Vandis on 2011-08-26
Try it and see how far you get?

Am thinking the only issue will be about choosing the right connection method so both users get all the emails up on the server (you will have to tell the mail server to retain all emails and use IMAP as connection method so that both accounts sync) There may be a problem with each user seeing the emails they each of them has sent.

Would probably be easier to run two seperate addresses but to set up the mail server to forward everything to the other account if that is what you are after.
