The Gnome option is the Gnome 3 desktop environment (gnome.org) , Gnome classic is Gnome3 without alot of the eye candy and resembles Gnome2.x. Unity is ubuntu's desktop environment which in the new version 11.10 is built off of Gnome3, Unity 2D is Unity only without hardware intensive visual effects (ment for older hardware.). Xterm is a text based working environment (advanced), and Fail safe is like safe mode in windows (loosely anyway.) I hope this helps.
