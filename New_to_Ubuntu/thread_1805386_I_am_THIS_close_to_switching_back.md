### Post by SteveTheRed on 2011-07-16
You can have both, you know.

You can dual-boot Windows and Linux on the same machine, use Windows as a virtual machine inside of Linux with VirtualBox (free) or VMWare (not free) or use remote desktop to connect to another Windows machine.

I have a Windows XP virtual machine in VMWare because I thought that I would need it all the time. I only actually use it for a couple of hours a month.

My wife and I have similar laptop models with i5 processors that came preinstalled with Windows 7. I immediately switched to Kubuntu and she stayed with Windows. As time passes, her computer gets slower and buggier while mine is still lightning fast.

I really like how there is almost always an application or set of applications for what I need that are freely available and just an "apt-get" away. I like how Ubuntu keeps all of my apps up-to-date without rebooting whether I like it or not. I like a lot of things about Linux in general and Ubuntu in particular.

Don't let people tell you that you have to be some kind of super geek to use Linux. My Mom, who is 74 and has no technical skill at all, uses Xubuntu on her old computer to read her email and surf the web. I don't get calls about viruses or porn-filled windows popping up anymore, which is great because the last thing I want to talk to my Mom about is porn!
