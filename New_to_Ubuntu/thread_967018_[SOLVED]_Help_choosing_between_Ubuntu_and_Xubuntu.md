### Post by Mazza558 on 2008-11-01
Gnome should work quite well (especially if you use Ubuntu 8.10 Ibex), but if you find it's too slow, simply open a terminal and do 

> sudo apt-get install xubuntu-desktop

After it's complete, you can then logout and in the options, change the desktop to "XFCE" to get Xubuntu.
