### Post by gkestrel on 2008-10-29
It sound like when you had to reset your laptop that you might have corrupted the filesystem on your sd card. You could try installing testdisk to see if you can recover the filesytem on the sd card, also when you install testdisk you get a program called photorec, that can be used to try and recover any photos or any data you may have on the sd card.  Testdisk is available for install from the repositories, and is also available to download for windows if you prefer to try recovery from a windows machine.Try that first and see if it gets your sd card working again. Hope that helps you if not post back here with details of how you got on.
