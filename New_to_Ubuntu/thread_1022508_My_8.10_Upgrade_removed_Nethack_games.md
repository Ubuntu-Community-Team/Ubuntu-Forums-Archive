### Post by bikeboy on 2008-12-26
To search, it's actually *apt-cache* search <name>. Then to install, it's sudo apt-get install <name>.

I'm sure what you're after is still available in Add/Remove programs, it's just that apt-get is quicker and easier to describe over a forum.

### Post by bikeboy on 2008-12-26
Okay, an intermediate between Add/Remove and apt-get is the Synaptic Package Manager. System > Admin > Synaptic. You may like it better. Using the following search, it certainly looks like nethack-gnome no longer exists - [http://packages.ubuntu.com/search?keywords=nethack&searchon=names&suite=intrepid&section=all](http://packages.ubuntu.com/search?keywords=nethack&searchon=names&suite=intrepid&section=all)

Don't worry though, install the nethack-qt package and it will work perfectly fine under Gnome, the only difference being that some extra dependencies may need to be installed (automatically) and the appearance will be a little different.
