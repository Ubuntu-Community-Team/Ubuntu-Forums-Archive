### Post by dublued on 2008-05-13
are you using the ATI restricted drivers?  you can check by going to System>Aministration>Hardware Drivers, and see if it shows up there.

are you using Compiz?  sometimes that can give some problems.  try turning that off typing the following in terminal:
```
metacity --replace
```
to turn compiz back on type:
```
compiz --replace
```

good luck
