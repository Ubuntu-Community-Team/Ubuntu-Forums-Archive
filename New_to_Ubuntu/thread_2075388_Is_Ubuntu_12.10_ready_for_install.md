### Post by monkeybrain2012 on 2012-10-23
In my test installation most major things work and it seems reasonably good  but there are some quirks. For examples, trying to install "libav dev" files will remove the "extras" libraries installed via medibuntu (like removing libavcodecs-extra53 and installing libavcodecs53 etc) gecko-mediaplayer doesn't work, gimp's button seems not very responsive etc. On the other hand, some much waited bug fixes for compiz has finally been done.. I haven't tested other things yet, but I anticipate more bugs, such is the case with every new release.

I will wait for a month at least before I will use it as my productive system. BTW, the reason that I am even considering 12.10 is that some very annoying bugs in 12.04 seems to never get fixed but they are fixed in 12.10.
