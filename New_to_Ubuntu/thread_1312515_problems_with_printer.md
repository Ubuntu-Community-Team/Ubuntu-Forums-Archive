### Post by Melk79 on 2009-11-03
Install the HPLIP Toolbox from the Ubuntu Software Center.  Adds all the functionality of an HP printer on a windows box.  Cartridge levels, cleaning, etc.  Even adds the little HP logo to your panel for convenience minus all the annoying popups of the windows version.
