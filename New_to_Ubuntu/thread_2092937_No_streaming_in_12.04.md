### Post by SeijiSensei on 2012-12-09
[Clementine]("http://code.google.com/p/clementine-player/") does a good job handling radio streams.  It's a KDE application, though.  You can run it on regular Ubuntu, but it will need to install a bunch of dependencies.  Unless you're really short on disk space, this usually isn't a big issue.  Those of us who use GTK+ applications like Firefox on Kubuntu deal with the reverse situation all the time.
