### Post by Vladlenin5000 on 2015-05-28
You have selected the "dead keys" option for your keyboard layout.
I use it like this for a few US International keyboards I have because That option is also required  to be able to write ä / é / ç and many others typical of Latin languages.

You can change it to English (US) standard or some more appropriated for iDevices, just make sure you don't select any variant with "dead  keys".
