### Post by NightwishFan on 2011-02-21
If you can go somewhere with a net connection and download packages, you can make a cd of them using aptoncd.
[http://aptoncd.sourceforge.net/](http://aptoncd.sourceforge.net/)

An easier way might be to try to use something like keryx to do so:
[http://keryxproject.org/](http://keryxproject.org/)

You can also manually download packages here (and all their dependencies this way is complicated and time consuming :D)
[http://packages.ubuntu.com/](http://packages.ubuntu.com/)

Hope this is of help. It might be easier to try to get a dvd that has more packages, but the Ubuntu DVD does not include much.
