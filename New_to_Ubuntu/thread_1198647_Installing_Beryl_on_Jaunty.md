### Post by starcraft.man on 2009-06-27
Beryl is obsolete, it was folded back into compiz per my understanding.

Compiz is already installed. To get it working:

1) install graphics driver. System > Administration > Hardware Drivers. This automated installer works very well, usually does it fine for anything recent.

2) Go to System > Preferences > Appearances > Visual Effects tab and set it to normal. If you have a powerful card set it to the highest effects setting, you can go back to normal if it moves slow.

3) For more options to control. Install compizconfig-settings-manager. System > Admin > Synaptic Manager > Search for "compizconfig-settings-manager" then set the package to install and apply. Will appear in the preferences menu as compiz manager.

4) Enjoy!

### Post by sisco311 on 2009-06-27
Go to System -> Preferences -> Desktop Effects and click 'Enable Desktop Effects'

Open the Synaptic Package Manager (System > Administration > Synaptic Package Manager) and install compizconfig-settings-manager.

Go to System > Preferences > CompizConfig Settings Manager and configure Compiz Fusion to your likings.
