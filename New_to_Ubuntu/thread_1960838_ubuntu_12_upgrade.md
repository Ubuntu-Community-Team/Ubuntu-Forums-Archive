### Post by HiImTye on 2012-04-18
all modern distributions of Ubuntu come in either 32 or 64 bit flavors

you can use a 'GNOME2 type' feeling interface with the 'gnome-session-fallback' package (or whatever equivalent it is in the new version), however, some features might be missing that you had in GNOME2. it will not be exactly the same, as it is GNOME3 not GNOME2
