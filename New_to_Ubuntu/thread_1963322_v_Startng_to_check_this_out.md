### Post by Silent Sam on 2012-04-22
If you want to try Ubuntu before you commit, you can.
I wouldn't recommend using the Wubi installer. Ive seen lots of post about the Wubi installer failing and people being unable to boot up again.

Burn your Ubuntu .iso to a CD.
Restart your computer and hold down F2 (may be different on your computer).
If that doesn't work, keep rebooting and holding down different Function keys till you get the right one.
Change the boot order to boot from your CD (optical drive) first.
Insert your Boot CD.
Save your changes an restart again.
Let it load.
When it first starts up you will be given the choice to try Ubuntu or install.
If you chose install you will be given the option to install Ubuntu alongside Windows (dual boot) or erase everything and install Ubuntu by itself.

I'm sure you'll make the right choice;).

Come back and let us know if you have any problems with the install.
