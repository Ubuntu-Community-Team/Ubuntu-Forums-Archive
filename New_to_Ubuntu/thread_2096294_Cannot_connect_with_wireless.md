I am having trouble with Ubuntu 12.04 installed on a HP Pavilion dv2000. The machine have Windows Vista OS, and a Broadcom network card also. It can't connect via wireless. I checked the network card. This is the message received: 01:00.0 Network controller [0280]: Broadcom Corporation BCM4311 802.11b/g wlan [14e4:4311] (rev01). Is there any fix for this problem? Please help.

---

### Post by GordThompson on 2012-12-19
Connect to the Internet using the wired network connection (Ethernet cable). Then run the 
[B]
System Settings > Hardware > Additional Drivers
[/B]
applet. That should install the (proprietary) STA driver and get your wireless adapter working.
