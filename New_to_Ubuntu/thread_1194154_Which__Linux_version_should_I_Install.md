### Post by Andy06 on 2009-06-22
Frankly, upgrade your RAM. Otherwise if you cannot for some reason, look for speciality niche distros like Puppy, Crunchbang or DamnSmallLinux.

You could try Xubuntu but the Live CD did not run for me on 256 MB RAM (never mind the minimum stated on the website).

Ubuntu (with GNOME or KDE) will be painfully slow with your processor and RAM. I know Linux is supposed to run better than windows but it doesn't work that way. Coming from XP, you will find Ubuntu slower on your system with that much RAM. (compared to a clean uninfected and non weighed down XP install). 

Install at least 512 MB to get good performance from Ubuntu/Kubuntu but since memory is so cheap, I would say go crazy and install 1 or 2 GB.

Test the LiveCD and LiveUSB version of anything you might want to install and play with it for a while.

### Post by Andy06 on 2009-06-22
I might get flak for this but the OP is obviously a novice. (1st post and he is asking for which one to install and doesn't seem to have tried many so far) so I would advise all members to not steer him towards distros that are more suited to advanced users. It really turns off someone new to Linux. As it is shifting to Ubuntu and Mint (which are lightyears ahead of the others in terms of simplicity, usability and out of the box experience) is hard enough for them. The last thing you want is for them to install something special and not have half the hardware work and then suffer though installation, package management and general usability hell.

Mortos,
I guess the best thing to do is to work out how Ubuntu or Mint can be made to work. One other alternative is to use another Window manager. Fluxbuntu seems defunct but you can always install ubuntu and then remove GNOME and replace it with another lighter WM. IMHO, the best thing to do is upgrade RAM and stick with Ubuntu and Mint for a better than XP experience. I doubt you are going to feel comfortable with anything else and even Xubuntu is sort of basic. Download the images of all you are interested in and then try their LiveCDs out.
