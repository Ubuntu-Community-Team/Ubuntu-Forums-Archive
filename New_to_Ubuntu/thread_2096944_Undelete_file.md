### Post by Mark Phelps on 2012-12-21
You can't use Windows to recover a deleted file from an Ubuntu setup because Windows doesn't understand the Linux filesystem.

However, if you can boot into Ubuntu, you can use Testdisk or Photorec to recovery the file.
