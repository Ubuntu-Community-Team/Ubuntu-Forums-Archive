### Post by tilapia on 2011-01-15
I used this guide to [installing Ushare on Ubuntu]("http://techpad.co.uk/content.php?sid=156"). Basically you can just install Ushare from Synaptic or enter sudo apt-get install ushare. 

You'll then need to change some settings in the configuration file, so enter sudo gedit /etc/ushare.conf and change the settings to the ones mentioned on the blog above.

Works a treat for me.
