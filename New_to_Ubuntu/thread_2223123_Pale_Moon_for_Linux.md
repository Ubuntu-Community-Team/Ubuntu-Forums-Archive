### Post by grahammechanical on 2014-05-09
It is the responsibility of the project developers to submit their application for inclusion in the Ubuntu Software Centre. The application has to be code audited to determine if it contains any code that could be used to do damage to the operating system and the user's data. This audit will take time. You should make your request to the project's developers.

[http://developer.ubuntu.com/publish/apps/application-states/](http://developer.ubuntu.com/publish/apps/application-states/)

[https://wiki.ubuntu.com/AppDevUploadProcess](https://wiki.ubuntu.com/AppDevUploadProcess)

[https://myapps.developer.ubuntu.com/dev/apps/](https://myapps.developer.ubuntu.com/dev/apps/)

By the way, there is nothing stopping anyone from downloading the Linux version and installing on Linux. 

Regards.
