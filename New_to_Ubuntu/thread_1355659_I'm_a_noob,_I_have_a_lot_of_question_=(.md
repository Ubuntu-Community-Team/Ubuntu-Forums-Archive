### Post by howefield on 2009-12-15
> **dchelix89 said:**
> 1. What's the difference between Ubuntu, Kubuntu, and the other derivatives?

The desktop manager / environment. Ubuntu uses gnome, Kubuntu uses KDE, other derivatives use others like xfce. 

> 2. Is there anyway I can play my window's games (WoW, CS-Source, left4dead2, Crysis) on Ubuntu without having to use Wine? 

If there isn't a native linux port of the game, then no, it won't work without wine, you could virtualise for "some" games but if you are scared of wine, you'll be petrified of virtualising ;)

> 3. I use Winamp in Windows because I listen to a lot of shoutcast radio stations, will Winamp work in Ubuntu? If not, are there any alternatives for listening to Shoutcast on Ubuntu?

Winamp won't work in Ubuntu, but may through wine, however there are alternatives that you will be able to use. 

> 4. I use OpenOffice in windows and I love it- however, will my printer work with ubuntu even though the manufacturer hasn't released a Linux driver for it?

Most likely, but you'd need to tell us the make/model.

> 5. Should I be dual booting Ubuntu and Windows 7 on a gaming PC?

Absolutely. If you are a gamer, you will most likely want to keep windows available, dual booting gives you the best of both worlds.

> 6. Are there any benefits of using Linux on a gaming PC rather than Windows 7?

Err... windows for games, Ubuntu for everything else.

> 7. I'm worried Ubuntu will be a hassle for me just to get used to and to get games working on more than I'll enjoy it- Should I be this worried about compatibility issues?

Don't be under any illusions, it may well be a hassle, it might not be, but your questions seem to suggest you will not find everything sweet. However, by dual booting you get the opportunity of using what you are comfortable with whilst you need it, and at the same time being able to persue alternatives at your leisure.  

> 8. Does Ubuntu's media player able to play mp3's out-of-the-box?

Not out of the box, but the required codecs are a mere download away. Using the built in package manager, you will have mp3s playing in a couple of minutes. 

> 9. Will Ubuntu Server Edition work as a home server? I have an extra server from work that uses Windows Server (which is the devil), can I replace it with Ubuntu Server Edition and will it be able to interact with all the computers in my house running Windows or Ubuntu?

Short answer is yes, (long answer is also yes :)

Good Luck :)
