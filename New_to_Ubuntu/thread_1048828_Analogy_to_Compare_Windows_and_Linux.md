### Post by dasunst3r on 2009-01-23
My favorite analogy for Windows and Linux is point-and-shoot camera versus digital SLR.  While Windows is pretty straightforward, it doesn't offer much control.  Linux, on the other hand, give you a lot of control, should you want it.  The outcome of switching to manual varies depending on hardware, whether the person knows what he/she is doing, etc.  However, if one were to install a distro like Ubuntu, the user can stay in Auto and still enjoy high-quality pictures.
