### Post by r-senior on 2011-04-01
I've not tried it to a memory stick but I used to use the local directory sync to backup notes in my Dropbox directory.

If you go to Preferences -> Add-ins tab, you'll find an entry under Synchronization called Local Directory Sync Service Add-in.

My main doubt about this is how it will react when the path disappears as you unplug the drive. Tread carefully. I'd sync to a local directory first so at least you have a copy of your notes if it goes wrong.

The official way to sync/backup notes of course is using Ubuntu One. I'm finding that quite reliable now with Natty, although it's had its moments. The Dropbox method I used to use worked well.

Navigate to .local folder and click on it. Then open the share folder. Then open the Tomboy folder.
