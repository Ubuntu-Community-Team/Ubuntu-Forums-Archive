### Post by RobGoss on 2016-10-09
Hello and welcome, First you will need to tell us a little about what laptop you have and trying to install Ubuntu on starting with the specs Ram, Video card, so on

Is this a old laptop?

How did you create that live USB?

What version of Ubuntu are you trying to install?

Is your machine UEFI / Legacy Bios

When trying to install Ubuntu using a USB you will have to boot from that **USB** in most cases** F-21** will give you the option to choose which device you want to boot from. Please make sure you understand what's involved with the installation if not done correctly if can wipe out the entire system if there's another one on this machine. The more information you can provide us with the better we can advise
