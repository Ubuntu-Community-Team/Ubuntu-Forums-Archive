### Post by Lars Noodén on 2015-02-22
You can use cron for that via [crontab](http://manpages.ubuntu.com/manpages/trusty/en/man5/crontab.5.html) or gnome-schedule.  

[https://help.ubuntu.com/community/CronHowto](https://help.ubuntu.com/community/CronHowto)

You can then set it so that you user runs a script at a scheduled time.  

There's a third way, perhaps someone else here recalls it.
