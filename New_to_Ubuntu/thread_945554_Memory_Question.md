### Post by SunnyRabbiera on 2008-10-12
There is a basic app if you need it, look under system> administration> system monitor
In this application you will note there are a number of tabs.
The first tab shows your base system information
the second tab shows processes
the third shows memory and processor load and the last tab shows how much hard disk space you have left.
