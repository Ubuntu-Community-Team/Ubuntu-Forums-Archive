### Post by leth on 2011-11-15
FYI, you might want to also check out BasKet.  I have just started experimenting with BasKet and so far it looks very promising.  I looked at KeepNote but didn't like the layout (particularly the listview on the top) and couldn't tell if it allows for free-form placement of objects.  BasKet's layout seems to have a lot more in common with OneNote (every page/section is listed in a hierarchical list on the left) and it allows for free-form placement.
