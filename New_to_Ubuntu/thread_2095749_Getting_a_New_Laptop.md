### Post by fantab on 2012-12-18
> **Orochimaru777 said:**
> I realize you hear this a thousand times, but I am getting a new laptop: the Samsung NP300E5C-A0CUS. I am wondering if it can upload Linux Ubuntu on there. I know it's new and has Windows 8 (which I will mess with for a bit and throw on a CD), but if it can I will buy it.

Firstly, find out what GRAPHICS card it uses.. not just Intel but something like GMA4500 etc... There are certain Graphics cards that just don't work in Linux like PowerVR chips. Avoid these at all costs if you want to run Linux on your machine.

Secondly, find what sort of Wireless/WiFi cards it has.. then google about them with respect to linux and also search these forums...

Good Luck...

### Post by MoebusNet on 2012-12-19
I think that if I was going to buy a Windows notebook nowadays, I would buy a replacement HDD or SSD before I did anything else. I would remove the factory HDD to eliminate any possibility of screwing up the Windows install (thus preserving the resale value), install the replacement and experiment to my heart's content.

That way if you are not happy with things, just reinstall the original HDD and sell the notebook.
