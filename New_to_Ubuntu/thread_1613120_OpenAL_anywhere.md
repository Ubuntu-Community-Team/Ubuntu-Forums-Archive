### Post by Temposs on 2010-11-04
When you want a particular package in Ubuntu that isn't in the repos for your version, but you know it would be in another version, then you should check [http://packages.ubuntu.com](http://packages.ubuntu.com)

Here is a link to libopenal1, so you can just download and install the .deb for the 32-bit version: [http://packages.ubuntu.com/maverick/libopenal1](http://packages.ubuntu.com/maverick/libopenal1)

If you want to share ideas to add to Ubuntu, the best place to go is: [http://brainstorm.ubuntu.com](http://brainstorm.ubuntu.com)

Ubuntu developers have begun looking at brainstorm more closely recently, so it's a good time to make a submission to brainstorm :-)
