### Post by oregonbob on 2009-02-03
I have added many applications that have by default wound up in "Sound and Video" menu. I need to create two submenus, one with "Audio apps" and another titled "Video/CD Apps". Then I would like to MOVE the applications now all dumped in "Sound and Video" into their respective audio or video submenus.

I have used Preferences -> Main menu program to create the two new submenus. But there is no ability to MOVE applications from the standard Sound and Video menu into my new submenus. The only apparent way is to manually enter each application to the submenus by adding a new item, typing in all the properties of an application. Then I have to go delete the original application item from the Sound and Video section to clean it up.

Is there no drag and drop, or cut and paste way to accomplish this task? It seems very primitive and tedious as is.

### Post by Pollox on 2009-02-05
You can drag and drop an item from the items section on the right to a menu on the left.  This will add it to that menu, but you will still have to delete it from the original menu.  So there's no need to manually create a new item each time.  Still, if there's an easier way to do this (moving an item) I'd be interested in hearing it.
