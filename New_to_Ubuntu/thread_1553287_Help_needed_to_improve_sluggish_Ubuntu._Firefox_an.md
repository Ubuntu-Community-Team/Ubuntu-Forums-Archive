### Post by ron999 on 2010-08-14
> **scrypt said:**
> 

By the way. Out of pure curiosity. why is it sometimes a good idea to disable ipv6 ??

And.. What is the difference between ipv4 and ipv6. 

So far everybody's ip address is a bunch of 4 groups like this:-
xxxx.xx.xxx.xxx
So we only need to use ip4 which is quick for our computers to look up.
When we run out of numbers we will start seeing ip addresses with 6 groups.
That's when we will need to enable ip6 and it will be a bit slower.
