### Post by 3rdalbum on 2009-12-29
The screencast link is attached to all messages I post - it's part of my signature. That's Ubuntu from a few years ago, and yes it can still look like that (or better) if you theme it right. The demonstration in it is not related to your message.

Wubi definitely asks how much space you want. You can remove it from the Windows Add/Remove Programs, and then reinstall it. It asks about the amount of space on the same page as it asks what drive you want it on - you must have missed it.

This time, run the wubi.exe program from the burned CD of Ubuntu; it will save you having to download the CD image again. And give it 10 or 15 gigabytes if you can spare it.

There are disadvantages to using Wubi; so I do recommend you do a real dual-boot installation at some point down the track if you decide you want to keep Ubuntu.
