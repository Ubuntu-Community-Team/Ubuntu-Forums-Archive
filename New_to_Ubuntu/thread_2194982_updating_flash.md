### Post by Frogs Hair on 2013-12-21
Adobe has discontinued flash for Linux and the current available version will only get security updates. Google Chrome includes pepper flash which gets updated as needed with new Chrome versions.  

Old: [http://9to5google.com/2012/02/22/pepper-based-flash-player-coming-to-chrome-later-this-year-adobe-dropping-standalone-plug-in-download-on-linux/](http://9to5google.com/2012/02/22/pepper-based-flash-player-coming-to-chrome-later-this-year-adobe-dropping-standalone-plug-in-download-on-linux/)

### Post by craig10x on 2013-12-21
Yes...as Frogs Hair mentioned, the only way to get the latest flash on linux now is to use Google Chrome (not Chromium from the software center but Chrome from Google's website which you can get by downloading the deb file, right clicking and installing with the software center)...the built in pepperflash plug in that only Chrome has gets the latest flash...they have an agreement with adobe for that...

The old adobe flash on the system (that was installled with the ubuntu restricted extras) only gets security updates...no newer versions...and that version is over 2 years old already...
