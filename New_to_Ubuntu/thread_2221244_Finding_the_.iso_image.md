### Post by grahammechanical on 2014-05-01
Let us think this through. How did you install Ubuntu? Did download an ISO image from the Ubuntu website? Did you do this in Windows? Then, did you burn the ISO image to USB memory stick or DVD disk? Then did you use the USB/DVD to install Ubuntu into the VM running in Windows?

So, the original Ubuntu ISO image is in the Windows folder where it was downloaded to and which you accessed when you burned the ISO image to USB/DVD. I cannot inform you about Windows but on Ubuntu we have a Downloads folder and anything I download using the web browser goes into the Downloads folder. So, in my Downloads folder I have saucy-desktop-amd64.iso and trusty-desktop-amd64.iso. Two 64 bit Ubuntu ISO images, one for 13.10 (Saucy Salamander) and one for 14.04 (Trusty Tahr). You must have something similar in Windows. Unless you deleted the ISO image after burning it to USB/DVD.
