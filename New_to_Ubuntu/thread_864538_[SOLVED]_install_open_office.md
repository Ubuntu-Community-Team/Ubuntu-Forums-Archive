### Post by drs305 on 2008-07-19
As ozbolt mentioned, OpenOffice comes pre-installed. You can access the modules via System, Office or run "ooffice" in a terminal. 

If your attempt to install it messed up the default OpenOffice installation, you can try using synaptic, select "openoffice.org" if you want the complete suite, right click and select "mark for reinstallation", then Apply.
