### Post by Temposs on 2009-12-31
1) This question is not good English. I cannot understand it.

2) Sometimes you can get Windows-only programs to work on Linux using WINE([http://www.winehq.org](http://www.winehq.org)). But, you should first try the alternatives to Windows-only programs. Instead of Photoshop, see if GIMP works for you. Instead of Microsoft Office, see if OpenOffice works for you. Both of these are included in Ubuntu. Also, GIMP and OpenOffice both work on Windows as well, so you can try them.

3) Most file formats you use in Windows are supported in Linux.

4) If you are asking the difference between Linux and Unix, Unix is an operating system that was developed by a private company and is not free to use. Linux is a free operating system that uses the same standards as Unix, but was developed as an open source program, independent of Unix. Also, Malvrq is not an English word.
