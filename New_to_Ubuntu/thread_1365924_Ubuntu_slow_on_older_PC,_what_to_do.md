### Post by marshmallow1304 on 2009-12-27
Ubuntu is a full featured distribution; see the [System Requirements]("https://help.ubuntu.com/community/Installation/SystemRequirements").  You can start with the [Ubuntu Minimal Install]("https://help.ubuntu.com/community/Installation/MinimalCD").  From there, you will have a base command-line system from which you can install programs and a GUI if you decide you want one.  If you need assistance with installation, see [this guide]("http://www.psychocats.net/ubuntu/minimal").  I recommend substituting LXDE for GNOME but, as always, the choice is yours.

### Post by David Delony on 2009-12-27
If you do want to stay in the Ubuntu world, you might try using Xubuntu, which is a lightweight version intended for older computers. The Web site claims it'll run on 128 MB of RAM, though it might still be slow. Since you already have Ubuntu installed, you can just install the 'xubuntu-desktop' package.

There are a lot of distros optimized for older hardware. Damn Small Linux and Puppy Linux are also good choices, even if they're not based on Ubuntu. Good luck on your Linux adventure!
