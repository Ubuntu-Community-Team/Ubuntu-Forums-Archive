### Post by DezSP on 2008-07-06
In the edit menus dialogue, just click on System tools and tick whatever items you want from it. No menu is ever *not* there, but empty menus aren't displayed by GNOME.

### Post by wolfen69 on 2008-07-06
just right click on one of the menus on the taskbar and select "edit menus". then check off system tools.

### Post by jimbob on 2008-07-06
Right clicking on the main menu symbol on the taskbar will bring up a dropdown menu. Click Edit Menus then click Applications and  select System Tools.
