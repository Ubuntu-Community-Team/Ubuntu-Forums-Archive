### Post by manoriax on 2010-02-02
If you want to set up a multi-boot computer, it is possible. I'd say you install Vista first and Ubuntu afterwards, so the entire configuration is automated. In general it is also possible vice versa, but difficult to set up the boot-loader then.

Then 'in Vista' shrink the partition that vista is on and then when you come to install Ubuntu, just install it in the largest free space available (it will be the unallocated space that the shrinking of the Vista partition created)
