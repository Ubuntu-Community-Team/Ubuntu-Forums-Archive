### Post by Arlo012 on 2008-11-11
I've never heard of the monitor going into power saving mode mid-installation, but I can recommend one of two things:
1. Try the installation with a different monitor, if you have one available. That way you will know if it's a monitor-specific issue or something else.

or

2. Try using the alternate installation cd. It runs from a command line, and thus if video drivers or something similar are the problem it would be far less likely to create any issues.

Sorry I can't be much more specific, but those are two things you might try.
