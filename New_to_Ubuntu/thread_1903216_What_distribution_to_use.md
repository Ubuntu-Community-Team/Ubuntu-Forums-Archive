### Post by corrytonapple on 2012-01-02
Cool, we have the same names :D
Except mine is C-o-r-r-y.  :P  Anyway, back to topic.
Linux is not Windows.  It does not have a "Windows GUI".  It does have a GUI though.  It is what Ubuntu is based on, and much more stable then Ubuntu as well.  It uses way less memory to top the cake, you may say.
It uses the GNOME 2 GUI, just like < 10.10 (10.10 used G2 as well).
I would say GNOME 2 would work, but XFCE or LXDE would be a better choice. He won't be able to play online games, or those flash games if he's into that, due to memory restrictions.  Music and basic web would be fine.  Maybe even typing up documents too!
Debian is very user friendly, just like Ubuntu 10.04, but more stable.
Download a copy if you can, try it out.  It is very nice, I use it myself, but on higher specs and GNOME-SHELL (Too heavy for that machine).
