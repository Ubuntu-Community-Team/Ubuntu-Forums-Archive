### Post by silkstone on 2008-07-15
I assume you're using Lightning as the calendar add-on.

Make sure you have the latest TB for Ubuntu (2.0.0.14) and the latest version of Lightning.

[https://addons.mozilla.org/en-US/thunderbird/addon/2313](https://addons.mozilla.org/en-US/thunderbird/addon/2313)
