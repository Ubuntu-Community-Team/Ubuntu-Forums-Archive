### Post by Frogs Hair on 2013-02-12
System fonts are displayed in the Gnome Tweak Tool if installed. With mail it depends how you access mail. Thunderbird has a font setting preference under Edit > Preferences > General. Web browsers also have their own font settings. It also possible to change font when composing new when logged in via browser.
