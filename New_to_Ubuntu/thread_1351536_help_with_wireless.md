### Post by spydeyrch on 2009-12-10
do you have an ethernet cable that you could connect to your router/modem and to yoru computer? I usually do this after a fresh install of ubuntu. It allows me to download any updates for the new system and also, it should detect if it needs to install a driver for your wireless connection. If you go to the systems menu -> admin -> hardware, it should scan your computer for any hardware that needs updated drivers. It will give you a list of the different options. All you have to do is select the appropriate one and then install it. It will go online (make sure you are connected via that ethernet cable), download the driver, and install it for you. this usually works for my systems.
 
Another thing, prior to installing ubuntu, did you have internet connectivity? What are your system specs?

### Post by Some Penguin on 2009-12-10
You might want to explain *how* something does not work.

- Is the wireless hardware detected by the kernel as a device?  (see lshw, for instance)
- Is the hardware detected, but there simply isn't a driver set up for it?
- Is there a supposedly properly working driver already set up, but you can't see any networks?
- Are you seeing a network, but you just can't connect to it?

And of course,
- What hardware do you have?

And so forth.  There are gazillions of ways something might "not work", and if you don't specify any details at all, the only logical thing is to redirect you to the generic pages on how to set up your wireless network, which you could easily do for yourself with Google.
