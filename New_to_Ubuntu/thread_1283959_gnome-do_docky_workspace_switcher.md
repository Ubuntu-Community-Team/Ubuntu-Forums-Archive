### Post by tom.swartz07 on 2009-10-06
Hi, 

Unfortunately, it does not appear that gnome do has a workspace switcher. You could use control + alt + <arrow keys> to switch instead though. I find that method a lot faster anyway.

There are some options that you could get from right clicking on it and poking around, but there arent that many options.

If you are looking for a dock that you could easily customize, I would suggest using Avant Window Navigator, and changing gnome do to a popup window. AWN allows you to add custom launchers and also has a large library of applets (weather widget, volume, trash, OSX-style stacks, rss reader, etc)

Hope this helps!

### Post by Ryan_Singer on 2009-10-06
Docky has a workplace switcher in Karmic, but I prefer Ctr-Alt-Arrows.

Also, it has a new function called intellihide, which hides docky only when maximized windows are selected. When the selected window doesn't need the space, or you are looking at the desktop, Docky doesn't hide. When you have a maximized firefox window, Docky hides.

It's wonderful. Not even OSX does it.
