### Post by ajgreeny on 2011-08-01
Try the **flash-aid** add-on for firefox.  Even if you don't use FF the flash it installs for you will work OK with chromium.  It also checks for conflicting plugins which might cause problems and removes them.

### Post by EkuquoL on 2011-08-02
tl;dr

Go into the Synaptic packagemanager and uninstall every adobe flash there is.
Then install flash-plugin installer... Only.

Having more then one flash plugin installed causes "blackboxs" where media content should be
