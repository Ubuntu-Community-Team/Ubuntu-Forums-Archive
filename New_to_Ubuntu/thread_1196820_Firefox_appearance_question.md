### Post by Celauran on 2009-06-25
Edit -> Preferences -> Content, go down to Fonts and click the Advanced button. From there, increase the minimum font size. I'm currently using Bitstream Vera Sans size 16 with a minimum font size of 12 and every site I visit looks fine. This assumes 96 DPI.

### Post by Celauran on 2009-06-25
Dots per inch. If you take a look at System -> Preferences -> Appearance -> Fonts -> Details, you can set the font resolution. I think 96 DPI is the default, I just wanted to ensure you hadn't changed it.
