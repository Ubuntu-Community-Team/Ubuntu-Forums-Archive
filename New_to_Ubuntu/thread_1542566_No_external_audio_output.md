### Post by vacc73 on 2010-07-31
***Go into system, preferences, sound, and make sure everything is selected that you want. Click the hardware tab to make sure it sees your sound card and at the bottom of the window, make sure the sound preferences match your speakers(stereo, 5.1 surround, 7.1 etc) Check the input and the output tabs as well to make sure that what you want is selected. It still could also be, and I think it happens to all of us, that a mute button has been inadvertantly been checked! Do this with music paying so you can hear immediate results. If you have already done these things then hopefully someone else checking this thread will have an answer cheers!!:guitar:***
