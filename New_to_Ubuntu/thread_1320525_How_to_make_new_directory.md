### Post by Jguy on 2009-11-09
if you are in a directory in which you do not have permissions to make a new directory, you will have to open a terminal and type

$ sudo mkdir folder_name

obviously without the $. 

If you do have permissions, you can simply right click -> Create folder or on your terminal, the same command without sudo.
