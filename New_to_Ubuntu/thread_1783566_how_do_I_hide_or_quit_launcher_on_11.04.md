### Post by 3rdalbum on 2011-06-15
The launcher is part of the Ubuntu user interface. You can't just "quit it". It can be made to auto-hide if you use Compizconfig Settings Manager.

You can, however, change the user interface entirely. You can go back to old-skool Gnome 2 by selecting it in the "session" popup menu on the login screen. Or you can install KDE or XFCE or another user interface from Synaptic Package Manager.
