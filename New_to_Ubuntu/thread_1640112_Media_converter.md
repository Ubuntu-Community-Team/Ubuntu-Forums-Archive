### Post by owiknowi on 2010-12-07
First install several extra codecs from the ubuntu-restricted repositories.
You can activate them in Synaptic Package Manager trough the menu Settings/Repositories.

For sound I use: soundconverter and Audacity.
For other media: Brassero can do some converting as well.

More converters you'll find when you search in the SPM for 'convert' (MKV, AVI)
