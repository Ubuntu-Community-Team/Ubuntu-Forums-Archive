### Post by mac9416 on 2008-05-03
Do:

> sudo apt-get update
sudo apt-get install wine

And if your Linux computer isn't connected to the net (as mine is not) use NoNetDebs to download the correct files to install WINE. NNDs will provide you with instructions to install the software. 

Good luck!
