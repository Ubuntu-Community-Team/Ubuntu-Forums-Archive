### Post by krtica on 2008-11-25
Just go to Applications -> Add/Remove, and in a box "Search" type what you are looking for. In "Show" choose "All available applications". You'll get a list of software that can fulfill your needs.

For example, if you type "itunes", firs result is

**Rhytmbox Music player**,

with this description:

"**[I]music player and organizer for GNOME  **
Rhythmbox is a very easy to use music playing and management program which supports a wide range of audio formats (including mp3 and ogg). Originally inspired by Apple's iTunes, the current version also supports Internet Radio, [COLOR="DarkRed"]iPod integration[/COLOR] and generic portable audio player support, Audio CD burning, Audio CD playback, music sharing, and Podcasts.[/I]"

### Post by echo.whoami on 2008-11-25
Also for an alternative to itunes is banshee which you can simple plug-in the ipod to the computer, open banshee and ipod will be listed on the left. Then you can add a folder with tracks in banshee and with drag-drop on the device ipod, banshee will transfer the song to the ipod.

To install banshee write on a terminal:

```
sudo apt-get install banshee
```

It will ask you for a password which is the password you use for logging in ubuntu
