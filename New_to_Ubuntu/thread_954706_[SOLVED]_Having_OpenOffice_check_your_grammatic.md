### Post by ubuntu27 on 2008-10-21
First, I recommend you to upgrade to OpenOffice.org **3** as it has better support for language related issues.

and install the [LanguageTool]("http://extensions.services.openoffice.org/project/languagetool") extension to OpenOffice.org

Download the extension, and then open OO.o and then 
go to Tools menu--------> Extension Manager

> [LanguageTool]("http://extensions.services.openoffice.org/project/languagetool") is an Open Source language checker for English, German, Polish, Dutch, and other languages. It is rule-based, which means it will find errors for which a rule is defined in its XML configuration files.

You can find more extension for OpenOffice.org [here]("http://extensions.services.openoffice.org/")


PS:  [LanguageTool]("http://extensions.services.openoffice.org/project/languagetool") is still in alpha, it needs lots of improvements. But, something is something. It will be nice if more people contribute to it by adding rules and filling bugs.
