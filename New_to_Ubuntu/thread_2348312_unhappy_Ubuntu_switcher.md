### Post by Geoffrey_Arndt on 2017-01-02
No, you don't need to buy anything new to use Ubuntu or any other Linux.   Any OS (windows, mac_os's, unix, etc.) can be run inside the "host OS".   So, Linux can be run inside Mac, Windows, etc. as the guest OS, and the reverse, all OS's can be run inside Ubuntu - - with Ubuntu as the "host OS" and windows or MacOS being the guest OS.    

This is done via running software called a "Virtual Machine" (or VM for short).   Tens of thousands of companies do this every single day.   The most recommended VM software for Ubuntu is Virtual Box by Oracle.   It is 100% free (no trial ware etc.).    Search on Youtube for tutorials on how to set it up.   There is some detail, but it's not zactly "Rocket Science" by any stretch.

Interestingly, the lack of support for GPS products such as as Tom Tom, Garmin, Magellan, etc. kept Windows 7 on my home network for about 4 years or so.   Until I saw smartphone technology and Google GPS apps had progressed to the point a separate GPS device was no longer needed.   So, I nuked Windows from all my systems almost 3 years ago.   NO regrets.

Insofar as Music/Media, . . . . there are plenty of alternatives in the Linux world.   Some limited support for Apple  . . . but who needs it really?   Beyond streaming (web based apps, . . . spotify, pandora, etc.), there is music available via Amazon and Google up the proverbial _ _ _ _.   Why choose to be locked to I-Toones????   See [URL="http://alternativeto.net/"]http://alternativeto.net/
