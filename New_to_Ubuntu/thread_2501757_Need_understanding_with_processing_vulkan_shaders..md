### Post by nicolasdentremont on 2024-10-19
I've read that there has been a bug with Steam lately causing excessive processing times for the Vulkan shaders. Over an hour is pretty extreme. The games I've played so far take anywhere between a couple seconds to a minute.

I believe if you skip it, they should just continue processing while your game runs but may cause performance issues.

There is also a setting to have them run in the background by default.

The process is supposed to make games that use Direct X run smoother. I've run a bunch of of windows games through Steam that does this pre-processing and I've run a bunch more through Lutris. Lutris also uses DXVK but doesn't process anything before they launch. They both have similar performance.
