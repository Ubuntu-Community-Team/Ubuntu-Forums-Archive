### Post by herbertportillo on 2010-05-05
The default layout of GNOME, as we all know, has two toolbars: one on the top and one on the bottom of the screen. I like to have only one toolbar on the bottom with the main menu and the window list on it, like the default layout in Mint.

However, today I wanted to make the layout the normal default layout. So I right-clicked on the panel and clicked on New Panel to add the proper items, and I've noticed something. I can click on New Panel a ton of times, and a panel won't get added to the screen. Then suddenly, 12 panels get added all at once. The performance seems to be a bit sluggish. Karmic was much quicker. I guess it isn't really a problem, just a thought.
