### Post by jrothwell97 on 2009-08-25
It's called Conky. First, you need to install it using

```
sudo apt-get install conky
```

and then configure it by editing a file in your home directory called .conkyrc. More detailed instructions are available in [this thread]("http://ubuntuforums.org/showthread.php?p=6365702").
