You, sir, haven't cd'd to your download directory ;) 

It's on the desktop, right?

So either: ```
cd ~/Desktop
sh NVIDIA-Linux-x86-96.43.11-pkg1.run
``` or: ```
sh ~/Desktop/NVIDIA-Linux-x86-96.43.11-pkg1.run
``` 

Please note I believe you will have to run it as root, so add "sudo" in front of either one. :)
