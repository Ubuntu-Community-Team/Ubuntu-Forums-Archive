### Post by fooman on 2009-02-18
open a terminal and type or copy paste the following:

```
gksudo gedit /etc/apt/sources.list
```then press enter followed by your password.  when the file opens,  add the following 2 lines to the bottom of the file,  then save and close the file:

```
##launchpad kde4 repository##
deb http://ppa.launchpad.net/kubuntu-members-kde4/ubuntu intrepid main
```then in the terminal, run:

```
sudo apt-get update
```close the terminal.

next,  open synaptic package manager and search for "amarok-kde4".  it will most likely need a boatload of dependencies to go along with it...which should not be a problem.  synaptic should grab all of those for you as well.

please note:  *this will also uninstall the normal amarok version!*

to undo,  you can just uninstall the newer version and reinstall the older one.

hope that helps.
