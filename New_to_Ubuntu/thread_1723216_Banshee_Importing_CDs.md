### Post by LowSky on 2011-04-07
Media > Import Media > Pick folder where you have been saving the CD's to.

they should all be there. If they are not there you are not ripping the CD correctly.

if that doesn't work try sound-juicer (aka Audio CD Extractor in Ubuntu Software Center... gods know why they changed the name).Sound Juicer is even more simple and I think works a bit better than banshee as it give you more options for file extensions.
