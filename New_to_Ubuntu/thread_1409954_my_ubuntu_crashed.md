### Post by benny.kallstrom on 2010-02-18
As far as i know it it possible to save your data on the broken ubuntu installation.You will need an usb flash drive, and an cd or dvd with ubuntu on it. If you have an ubuntu cd that you have installed ubuntu with, you can put that cd in your computer and reboot. Then you will come to an menu, and at that menu press try ubuntu, and you will start the so called live-cd of ubuntu.

When the cd has booted up, and you can see the ubuntu desktop, you need to insert an usb flash dive in your computer, and save the data there. You should be able to acess your data by searching for it. When your done reboot your computer.


Hope this helps!;)
