### Post by Toshibawarrior on 2009-01-12
Go to Synaptic Package Manager and search for:

```
vlc

```

Or go to a Terminal and type:

```
sudo apt-get install vlc
```

Easy! It doesn't appear in Add/Remove...

VLC is awesome for DVDs, videos, music, etc...But I use it only for videos and DVDs...For my music needs I use ***Rhythmbox*** (doesn't get any easier and simpler than that) and my favorite ***Amarok***, but it's a KDE app so you'll need some KDE dependencies (which will be automatically installed).

Sadly this question is a matter of personal taste, you just need to experiment with different programs and see which one fits you best.
