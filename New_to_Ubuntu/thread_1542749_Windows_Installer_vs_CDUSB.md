### Post by mandazi on 2010-07-31
What are the differences, pros/cons of installing Ubuntu with the Windows installer vs. CD/USB boot.

---

### Post by 3rdalbum on 2010-07-31
For   a   permanent   installation   you   should   use   the   CD.   The   Windows-based   installer's   only   benefit   is   that   it   is   easy   to   remove   Ubuntu   and   you   don't   need   to   do   any   partitioning, but   if   your   Windows   stops   working   then   it  also   prevents   Ubuntu   from   booting.   This  doesn't   affect   a proper   Ubuntu  installed   from   the  CD.
