Yeah, this is a common problem that a lot of people face. How is the computer set up now? Is Vista installed? If so, it shouldn't normally be necessary to find a Vista CD at all -- if you want to set up a dual boot, you just shrink the partition that Vista is on, using the partition editor that's included on the Live CD.

If the problem is that you've wiped the hard drive, installed Ubuntu, and since have discovered that you need to install Vista as well, you've got a bigger problem -- I don't know any way around that except to completely back up your Ubuntu partition(s) onto an external hard drive, use the recovery CD, and then reinstall Ubuntu, recovering all of your own files from the backup you made.

If you don't have an external hard drive, it would be worth buying one anyway -- you can usually find a 1 terabyte external drive on sale for about $200 nowadays, and that's the easiest way to back up your computer, which everyone should be doing anyway.

Describe what is currently on your computer in more detail, and people will be able to give you better suggestions for how to do what you want.
