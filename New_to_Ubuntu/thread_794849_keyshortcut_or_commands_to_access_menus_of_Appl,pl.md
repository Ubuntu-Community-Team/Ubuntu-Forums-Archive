### Post by Xiong Chiamiov on 2008-05-15
Alt+f2 opens a run dialog.  It's the same as a terminal window, but you don't have to keep it open, and you don't get to peek at what the console output is.

For date/time, you can just enter
```

date

```
into a terminal window.

### Post by sdennie on 2008-05-15
Most commands are named pretty intuitively from the command line however, if you go to System->Preferences->Main Menu, you can navigate to the thing you want to run from the command line, right click on it and then select Properties.  That will tell you exactly what is being run when you click on the icon.
