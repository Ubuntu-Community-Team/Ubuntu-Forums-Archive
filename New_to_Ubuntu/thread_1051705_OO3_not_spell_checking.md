### Post by Rokurosv on 2009-01-27
Check if the dictionary for your language is installed. Go to Tools>Options>Language Configuration>Languages check the default language for the document, if there's a blue check then you have the spelling dictionary. If not go to [http://extensions.services.openoffice.org/](http://extensions.services.openoffice.org/) and look for a spelling dictionary for your language.
