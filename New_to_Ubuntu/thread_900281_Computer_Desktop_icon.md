### Post by pizzavortex on 2008-08-25
In gconf-editor:
apps>nautilus>desktop>computer_icon_visible must be checked.
If you don't want to do that, ubuntu tweak has the setting.
[http://ubuntu-tweak.com/downloads]("http://ubuntu-tweak.com/downloads")
Just use the deb file.
