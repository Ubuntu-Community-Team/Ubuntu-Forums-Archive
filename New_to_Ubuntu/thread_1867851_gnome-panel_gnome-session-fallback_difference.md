### Post by searchfgold6789 on 2011-10-23
See the sticky thread in this forum for instructions on how to get what you want - a huge amount of users are switching from unity to the old GNOME.
"gnome-panel" is the package that allows for a graphical menu to be placed on the screen to perform various tasks, such as window management, placing various instant-view utilities (i.e. Network Manager), and placing launchers.
"gnome-session-fallback" is the package that enables a user to login and compute without using the famously productivity-killing Unity interface.
