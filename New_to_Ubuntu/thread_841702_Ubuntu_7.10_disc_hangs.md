### Post by Rocket2DMn on 2008-06-26
It sounds like the LiveCD doesn't like your video card since it isn't loading X (the GUI base).  You can try booting the LiveCD into safe graphics mode (I think you hit F4 at the options menu), or you can try getting a Hardy Heron LiveCD since Gutsy is no longer the newest stable version of Ubuntu.
If you can live without seeing the OS first, you can install with an alternate install cd which uses a text based installer, but is still intuitive and very easy to use.  Your graphics should work fine after you install, the LiveCD just has limited support.
