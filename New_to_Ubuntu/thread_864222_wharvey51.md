### Post by baju on 2008-07-19
.bin files are executed with the console. But there is no need to install java from bin, you can install it as package with synaptic/aptitude/apt-get. There are different versions of java available. For example the packages for sun java 5 all start with sun-java5.

### Post by Sef on 2008-07-22
> How do I install Java in Ubuntu? It is a .bin file and I am new to linux so any assistance will be appreciated

Applications > Add/Remove > Show: Change to 'All Available Applications' > Search: Sun Java 6 > click top box (Sun Java 6 Runtime) > apply changes > apply > close
