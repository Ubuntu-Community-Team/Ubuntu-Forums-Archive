### Post by moboticdes on 2011-06-03
If I were you I would forget banshee gtkpod or amarok or rhythmbox as they all don't support syncing or only for a few ipod models, I would forget as well using wine to install iTunes and sync as wine doesn't support usb tethering and finally land to the only fully working solution:

1. install windows xp in virtualbox
2. activate usb detection
3. install iTunes and sync from there.

I know it sucks using an entire virtual machine for such a stupid job but again thank those people at Apple who dont want to make life easier but only to earn some big money with their proprietary software!
