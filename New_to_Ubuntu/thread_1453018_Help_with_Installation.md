### Post by J V on 2010-04-12
I think you just fried your ubuntu installation...

Wubi installs live on a file: They think that file is a harddrive, but it isn't really.

Once the "Hard drive" and boot system have been installed and ubuntu boots up it will format the "Hard drive" which is actually just a file on the windows partition.

I'd tell you to let it do its thing but now you will probably have to start all over...
