### Post by s1baker on 2011-06-17
just thought I would throw this in for what it is worth.
In Windows XP Google Earth, I had noticed that Google Earth uses the server that was set in Internet Explorer. Didn't matter whether or not you used Mozilla, Google Earth would look at the server used in IE to use as it's server. If a proxy was set up in IE, Google Earth would use that one. If it was a foreign proxy, Google Earth would use text pertaining to wherever that proxy was located.

Don't know how Google Earth for Linux gets it's server settings, but thought I would add this, might be something to consider.
