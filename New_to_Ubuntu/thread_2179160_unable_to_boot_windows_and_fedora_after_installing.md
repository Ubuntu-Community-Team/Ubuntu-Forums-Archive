### Post by fantab on 2013-10-07
Exactly, as deadflowr wonders, How did you install Ubuntu? The install method has deleted all the partitions on the Hard Disk.

If you wish to rescue the lost partitions then stop using the Hard Disk. Boot Ubuntu from DVD/USB and Read [HERE]("http://www.cgsecurity.org/wiki/TestDisk") to rescue Linux Partitions and [HERE]("http://www.easeus.com/data-recovery/partition-recovery-software/free-windows-7-partition-recovery.htm") for Windows Partitions...
Be warned that 100% recovery may not be probable. Since after deleting partitions Ubuntu was installed it will have overwritten data on that part of the Disk and it may not be recoverable.

Good Luck...
