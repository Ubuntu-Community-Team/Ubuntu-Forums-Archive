### Post by ajgreeny on 2010-05-30
You can download and install it manually direct from mozilla [http://www.mozilla.org/projects/calendar/sunbird/](http://www.mozilla.org/projects/calendar/sunbird/) or you can use the lightning add-on for thunderbird.

There is no lightning in the repos at the moment, but again you can get the add-on direct from mozilla and install it into thunderbird with no difficulty.  I have it running here and it works fine.

### Post by germanix on 2010-05-30
Sunbird will no longer be supported by Mozilla. They themselves suggest one updates to Thunderbird 3 and use Lightning 1.0 beta1 instead. However if you want to use Sunbird then go to the Mozilla page.
[http://www.mozilla.org/projects/calendars/sunbird/](http://www.mozilla.org/projects/calendars/sunbird/)
and download it from there.
