### Post by peculiar penguin on 2012-04-26
Open the Software Centre, Synaptic or whatever package manager you prefer and install the "flashplugin-installer" package.

If one specific website keeps complaining about the version while others work, that website is likely broken (hard-coded version numbers or similar) and needs fixing.
