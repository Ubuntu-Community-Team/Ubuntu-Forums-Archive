### Post by sffvba[e0rt on 2012-01-16
Hi,

Install the Gnome Tweak Tool (in terminal)

```
sudo apt-get install gnome-tweak-tool
```

Then run it (just use dash search and enter "tweak").  Under Desktop enable "Have file manager handle the desktop."
