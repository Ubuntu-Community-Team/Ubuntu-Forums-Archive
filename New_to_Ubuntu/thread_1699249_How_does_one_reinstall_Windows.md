### Post by tillum on 2011-03-03
You can download a win7 install disk image (iso) from torrent sites such as isohunt.com but you will need a torrent client. After you have downloaded the iso file, burn it on a disk (you will probably need a blank dvd to burn it on) then boot the computer off of it and install. 

I have to say, blindly downloading software like this isn't exactly safe, so caution is needed.

Note: If you are not able to burn it (no burner or something) there are ways to install from usb pendrives, just google around. 

Usually PC manufacturers provide drivers on their websites for download (look for a "support" link on their website). So for any hardware that isn't functioning correctly after you install windows, you will have to find the correct drivers for it.

The key on the bottom of her computer should work fine, although she will probably have to activate the license via the phone method.

If all else fails, google some more and/or contact M$ :)

Hope that helps.

### Post by wep940 on 2011-03-03
They put the key on the PC as it is a legal requirement of Microsoft.  If the PC comes with Windows it must have the key/proof of authenticity on the box.  Proof of one license.
 
As for downloading an image,  NEGATIVE!!  I tried that once just to see what would happen when I installed it on a junk PC, and believe me, it was full of malware.  I don't know the site you're talking about, but I wouldn't trust any.

### Post by Kixtosh on 2011-03-03
If your friend can get a copy of the install disk, it may come with whatever other software she may have bought from them at the time (I'm talking about you, Microsoft Office).

If you remove the drive and put it in an external enclosure, can you read anything? Perhaps you can discover the restore partition and just copy it, but you'll still need to find the manual to figure out how to invoke the recovery process.

The last laptop I bought with an actual recovery disk was in 2007 (and it was a clearance product, so it was "end of life" merchandise). Everything since has had a recovery partition and you are emphatically asked to make your own recovery DVD's (or CD's, it just takes more of them!) when you first start up. This should ideally be done before you do _anything_ else. Otherwise, I have used both restore disks from Toshiba, and the recovery partition, and have been able to get back to "out-of-box" state in both cases, including any packaged software or software trial versions. Frequently there's a key sequence to activate the recovery process from the restore partition.
