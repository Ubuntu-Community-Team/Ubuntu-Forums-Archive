### Post by spikoley on 2011-05-11
What type of video card do you have?  There is a [bug in jockey]("https://bugs.launchpad.net/ubuntu/+source/jockey/+bug/771788") that is causing issues like this.  Look under Additional Drivers for "The driver is activated but not currently in use".  If you see that message then it is the same issue.  Make sure you update the bug so they will know that it is widespread.
