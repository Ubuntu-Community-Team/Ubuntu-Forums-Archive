### Post by ajgreeny on 2013-06-23
I think you have two main options:-
[LIST=1]
[*]Run Irfanview in wine; it runs extremely well and using wine for the app instead of a Linux version is even mentioned on the Irfanview website.
[*]Try gthumb, which is just about the nearest you will find to Irfanview, I think.  There may be other similar small applications which I have not used, but gthumb is much simpler and smaller than the default picture managers of Ubuntu (shotwell) or Kubuntu (digikam), good though those may be.
[/LIST]

I used to use IV when I was on WinXP and liked it a lot and I did try it via wine for a short while, so I know how well it can work, but have now got so used to gthumb that I don't feel that I need it any more, so it has disappeared from my sights, for now at least.
