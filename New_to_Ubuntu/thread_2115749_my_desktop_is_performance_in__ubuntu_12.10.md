### Post by deadflowr on 2013-02-13
Let's go with 3rdalbum's suggestion and find out what graphics card you've got.

An easy way to find out is through a command line.

Open a terminal:   (Use ctrl + alt + T)

```
lspci -nn | grep VGA
```

you can copy it and paste it into the code tags (look for the # in the reply box toolbar).
