### Post by SeijiSensei on 2020-07-01
.aspx are the Windows -equivalent of PHP files.  It's possible the application expects to find software on a Windows machine that is not provided by Linux-based browsers.

You can try tricking the site into believing you're using the Windows version of Firefox by installing the User Agent Switcher add-on: [https://addons.mozilla.org/en-US/firefox/addon/uaswitcher/](https://addons.mozilla.org/en-US/firefox/addon/uaswitcher/)
