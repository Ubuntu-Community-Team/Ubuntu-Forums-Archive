### Post by wieman01 on 2008-12-30
From command line you'd do:
> sudo apt-get install sun-java6-jdk sun-java6-jre

_**EDIT:**_
The open source version of Java is:
> sudo apt-get install openjdk-6-jdk openjdk-6-jre
