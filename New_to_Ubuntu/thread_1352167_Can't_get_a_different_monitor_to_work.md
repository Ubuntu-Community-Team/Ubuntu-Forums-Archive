### Post by ST3ALTHPSYCH0 on 2009-12-11
If you're using basic Ubuntu, that's what you do when you log in.
The LCD settings aren't going to work on the CRT most likely. My advise would be to get rid of the xorg.conf
```
sudo mv /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
```

shutdown
Plug in the new monitor.
reboot.
Ubuntu should auto-detect the new montior.
