### Post by fredbird67 on 2011-12-21
Actually, there is too an E-mail client for Linux that can download and send E-mails on a Microsoft Exchange server.  It's called Spicebird, and it's basically a fork of Mozilla Thunderbird with support for Microsoft Exchange servers.  It's not in the repos, but you can add a special PPA repo for it.  The instructions are at [https://launchpad.net/~spicebird/+archive/ppa]("https://launchpad.net/~spicebird/+archive/ppa").
