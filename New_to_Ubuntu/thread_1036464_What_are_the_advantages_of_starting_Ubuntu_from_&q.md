### Post by bodhi.zazen on 2009-01-10
If you are one of the  people who feel the default installation installs too much and you prefer to pick and choose what is installed from window manager to browser, the minimal install is the way to go.

It is easier to start with a small base and add then try to remove from the default install (less breakage).

If you wish to keep your install light you should install say xfce rather then xubuntu-desktop or kde-core rather then kubuntu-desktop.

Personally I wish ubuntu gave more options in terms of package selection (or exclusion) in the install media, especially the alternate CD.

The minimal cd is also a "net install" meaning the ios is quite small and the packages are all up to date. No need to dl a 700 mb (8.04.1) cd and then turn around ad do another 700 Mb of updates.

I do a minimal install, not to save space on my hard drive, but because I now what apps I like and the minimal install gives me the option to mix and match as I prefer.
