### Post by QIII on 2011-12-26
The way to rule out a hardware issue, which is the first thing you should do, is to use a different keyboard (and a different set of fingers) that is known to work properly on another machine and attempt to recreated the behavior with it.

If that resolves the behavior, buy a new keyboard.

If it persists, you may need to dig deeper.

BUT:  Don't fall for the myth that Linux is unassailable.  The "You Have Won ...!" things are called "Social engineering" and Linux can fall prey to malware quite as easily as Windows if the engineer gets your mom to "click here to win".  Nefarious agents are not unaware of Linux, despite the myths to the contrary.  Although Linux "viruses" in a very narrow definition are very rare, malware can be installed if Mom is told to enter her login password.  And she is in the same danger with Linux as with Windows if she fills out a form with all of her personal information, SSN, passwords, bank account numbers and credit cards. 

Ignoring security just because you are running Linux is just as dangerous as when running Windows.
