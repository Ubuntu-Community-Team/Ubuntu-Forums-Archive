If it's not in the repository yet you'll have to do it the hard way.  a google search for the debian package takes you to this page: [http://packages.debian.org/wheezy/mpqc]("http://packages.debian.org/wheezy/mpqc")

You can down load the package (Warning it has quite a list of depenencies the download for your architecture will be found near the bottom of the page.  If any of the deps. are out of date on your machine you'll also need the newer version of those.

after down loading you can go to the folder you downloaded them to and right click on the .deb package and select open with software center. follow the instructions

or you can go to a terminal and cd to the proper folder and use the following command to install the package:
```
sudo dpkg -i <package name>
```

good luck
