Since no one has replied in a while, I will.  You don't want to hear this, but wine is not that great, and it's no replacement for applications you absolutely need to run on windows.  For some users, work or school requires keeping a windows partition, and occasionally booting over.

When programs get complicated, wine just won't handle them very well.

But, looking at your screenshot, it looks like that is a window error.  Maybe you can try to reproduce that error message while running the program in windowss, to trouble shoot the problem.

Finally, if you explain what it does, maybe someone can help you find an open source substitute.
