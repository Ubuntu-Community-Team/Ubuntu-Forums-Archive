### Post by RequinB4 on 2008-11-15
Remember that you can dual boot even AFTER you install ie have both operating systems on your hard disk at the same time.  This means you can run both your RTS games and Linux.  That said some games like WOW work(ed?) flawlessly in Wine (see [http://appdb.winehq.org/](http://appdb.winehq.org/)).

As for programming, I'd say hands down that an open source OS is essential to being a good programmer (or at least pretty close to it).  [http://www.catb.org/~esr/faqs/hacker-howto.html](http://www.catb.org/~esr/faqs/hacker-howto.html)

Antivirus - don't worry about it.

The ubuntu community is exceptional.  If you haven't noticed, the average thread in this forum gets an answer within a few minutes.  In the same way, security, bug, etc fixes are released within days, not months.  Add that to the fact that in linux, EVERY problem is fixable with the correct knowledge, and I'd say "learning" Linux is easier than "learning" windows or mac.  Most of the problems people experience are things mostly out of our control such as game developers who aren't smart enough to realize linux is a HUGE market of people that are extremely dedicated... but I digress :P

### Post by Mardoct909 on 2008-11-15
If there's some windows piece of software you CAN'T be without, you can try it in WINE (Not particularly reliable) or actually run Windows in a VirtualBox. If you want something as resource hogging as a 3d game or something, I'd advise dual-booting over virtualization. Both are easy.

As for programming, in a terminal you can just type in "python" and you have an interpreter right there for you. More notably, emacs! Probably the most featured text editor... ever. A lot of programmers use it for programming.
