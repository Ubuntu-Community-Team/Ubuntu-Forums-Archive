### Post by mcduck on 2008-07-15
GDM themes are installed through login creens configuration.

System/Administration/Login Window. Click the !Add"-button in the "local"tab and find your theme.

When installed, select it for use by marking the radio button next to the theme (simply clicking on the theme does not select it for use!).

### Post by zipperback on 2008-07-15
I wrote the following howto to help others who would like to change their GDM theme in Ubuntu.

"How to change the GDM theme in Ubuntu 8.04"

Step 1 ) Go to gnome-look.org and find a GDM theme that you want to use.

Step 2 ) Click the download link on the theme page. You will get a file that is named something like:  newtheme.tar.gz

Step 3 ) Click System -> Administration -> Login Window

Step 4 ) Enter your password when prompted.

Step 5 ) Click the "Local" tab.

Step 6 ) Click the "+Add" button.

Step 7 ) Select the them archive that you just downloaded. Example: newtheme.tar.gz

Step 8 ) Click Install.

Step 9 ) The new GDM theme will appear in the list with a thumbnail showing what it looks like. Select it by checking  the box next to the thumbnail.

Step 10 ) At the top of the window there is a selector labeled "Theme". Change it to say "Selected only". You can also have Ubuntu use a random selection of GDM themes for your login screen. Select more than one of the themes to use, and then select the "Random from selected" option instead of "Selected only".

Step 11 ) Click "Close", and then login out to see your new login screen.


I hope you all find this useful.
