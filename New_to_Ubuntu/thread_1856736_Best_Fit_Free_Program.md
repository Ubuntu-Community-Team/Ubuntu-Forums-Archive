### Post by anewguy on 2011-10-08
If you already have the images and just need to build up the pages, including the images, then OpenOffice.org writer or LibreOffice Writer would work great.  No big learning curve there.  If you are using a release prior to 11.04, OpenOffice.org writer should already be installed.  If you are using 11.04 (or a newer testing version) LibreOffice Writer should already be installed.
