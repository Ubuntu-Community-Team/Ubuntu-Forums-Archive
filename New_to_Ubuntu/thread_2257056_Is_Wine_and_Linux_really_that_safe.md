### Post by lukeiamyourfather on 2014-12-16
Linux is as safe as people make it out to be in my experience. There are malicious applications out there that will affect Linux but the permissions of the file system make it pretty difficult to slip through. For example you have to add the executable permission to something manually before it can be run, and by default it will have only normal user permissions which limit what files can be accessed or modified.

Wine is no more safe or unsafe than running anything else on Linux. Why do you think you need Wine? I'd suggest looking for alternatives before trying anything in Wine since only some applications work with Wine and some applications are time consuming to configure and troubleshoot with Wine.
