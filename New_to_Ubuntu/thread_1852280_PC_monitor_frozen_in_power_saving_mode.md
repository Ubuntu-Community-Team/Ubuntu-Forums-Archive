### Post by Birchle on 2011-09-29
Perhaps a silly question, but are you sure the computer itself started up (properly), or is that the root cause of the monitor staying black?  Or did the computer never shut down, only the monitor?

I ask because I've had several cases now where my laptop either hibernates or goes into standby, and when I try to start it back up, the fan turns on, but nothing else.  On a desktop, I'm not sure how easy/hard it may be to tell if the computer is running, or just the fan; I can only tell on the laptop because the hard drive light doesn't blink when I push the power button, when this problem occurs.  The only way I've been able to fix it is to hard-shut down, pull both the power cord and the battery, hold the power button for a few seconds, and then plug back in and turn it on again.  Just a hard shutdown and restart isn't enough.
