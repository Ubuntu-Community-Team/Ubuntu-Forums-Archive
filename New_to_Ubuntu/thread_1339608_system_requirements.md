### Post by staf0048 on 2009-11-27
Here's what's posted on the website.

" At least 256 MB of RAM is required to run the alternate install CD (384MB of RAM is required to use the live CD based installer). Install requires at least 4 GB of disk space."

According to that you should not be able to run the live session with what your computer has for RAM - so you might possibly run into troubles installing it from the live session.  I'd do a text base install (or buy some cheap ram).  Otherwise you should be golden.
