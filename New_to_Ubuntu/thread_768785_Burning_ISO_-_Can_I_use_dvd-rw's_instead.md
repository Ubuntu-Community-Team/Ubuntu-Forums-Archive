### Post by Alehbye on 2008-04-26
IMHO, RW's aren't the best to use because they are a bit more expensive and in most cases with ISO's in my experience, you have to finalize the disc so you can't write any more to it. So you're basically wasting an RW. Which is cool if you really don't mind.

If you have a DVD burner, it should burn RW's as well. Just make sure you're using the type your drive supports, ie. +RW | -RW. 

The media usually determines the speed of burning and most RW's, the ones I have next to me anyway, only burn at 1x - 2x. So the RW's burning at 1x is probably normal for your media. Regular DVD_+_R's should generally burn faster though. 

Hope this info helps.
