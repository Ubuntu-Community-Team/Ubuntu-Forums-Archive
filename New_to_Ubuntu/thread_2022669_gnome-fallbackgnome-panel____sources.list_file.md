### Post by mikewhatever on 2012-07-11
So, gnome-fallback is the session name, similar to ubuntu or gnome. It's just a name, and each session is defines to use certain components, like a window manager, panels, etc.

Gnome-panel is a package in the repositories. ([see details]("http://packages.ubuntu.com/precise/gnome-panel")). It's often used to install the components needed for gnome-fallback.

Comparing the two is sort of like comparing apples to tea.

A local server(same as mirror) will usually be faster. All in all, there are over 300 mirrors, most of them equal (not better or worse), so use the one you feel like using today.
