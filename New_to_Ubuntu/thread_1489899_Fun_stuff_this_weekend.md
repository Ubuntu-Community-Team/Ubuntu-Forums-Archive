### Post by K.Mandla on 2010-05-22
Moved to Absolute Beginner Talk. 

Depending on the hardware, yes it's possible to watch Internet video streams, but it may be a bit of work. I usually watch DVD rips against the framebuffer with a 550Mhz machine, and I've used 300Mhz machines to do the same from within X.

It will be a measure of how much power is in the machine, and how much effort you're willing to put into it.

### Post by oldsoundguy on 2010-05-22
The first thing I would do is copy Belarc Advisor to a floppy and run it on each machine to see what is actually IN there.  
Then Google the boards and see how much memory each supports and compare to the figures that Belarc gave you.
You MAY be able to find memory for them CHEAP on eBay.  Older stuff is sold there all the time for just a few bucks.  And you KNOW, it is always nice to have more memory.

Good luck and have fun, since this is an experiment .. do just that .. experiment.
