### Post by Rex Bouwense on 2010-04-30
For the most part a new release comes out approximately every six months.  A LTS version is supported for three years and generally overlaps the previous LTS version by about a year.  The normal releases are generally supported for only 18 months and overlap the previous releases by six and 12 months respectively.  Each new release has new and updated programs and it is generally considered to have the latest and greatest available in Linux.  You can use outdated versions but they are no longer supported with security patches or program updates.
