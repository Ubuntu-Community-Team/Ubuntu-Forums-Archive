### Post by grahammechanical on 2014-06-07
Things have changed a bit since 12.04. The 14.04 software centre has Gnome Session Flashback which will give two extra options at login, Gnome Flashback (Compiz) and Gnome Flashback (Metacity). The software centre also has Gnome classic. It is an extension to Gnome 3 shell. So, it installs Gnome 3 shell.
