### Post by kennedn on 2010-09-20
Okay I have found a way to convert .ciso to .iso.

First of all you will need wine to run the program as it is for windows but it does work

The file you will need is:
[]("http://www.4shared.com/get/222442337/5cbecaa6/WiiBackupManager035-beta1.html")

Extract it and save it into a folder somewhere

Then run the program (I assume you know how to do this) 

Then click add>files and find your .ciso file

and then make sure you have the little box to the left of the file clicked so there is an x in it

then click on transfer>ISO and select a place to save the .iso file and thats it, sit back and watch the magic lol.

after that you have an ISO file of the game and can use wiithon or what ever you use to put it onto your hdd

The program is prone to freezing but I managed to uncompress super paper mario with it.
