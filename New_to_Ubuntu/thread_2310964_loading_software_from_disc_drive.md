### Post by Petro Dawg on 2016-01-23
".exe" files are compiled for Windows only and is not something you will easily be able to install using Linux.  Ubuntu has built in printer drivers that work with most, but not all, printers.  Try installing your printer through the Ubuntu printer settings.  Can you find your printer when you go to System Settings  > Printers > Add?

If not, what is the model of you printer?

I just noticed you use Lubuntu, the path I gave above may not exactly match what you see in Lubuntu, but there should be a "printer settings" option somewhere where you can attempt to install the printer.
