### Post by v3.xx on 2015-03-08
Mate has the same access to the same software sources that ubuntu has.

There are two ways to install Mate.
1) A stand-a-lone install from the site
2) Added to a ubuntu install

The site install will give you Mate version 1.8 and adding it to ubuntu 14.04 will give you mate version 1.6
