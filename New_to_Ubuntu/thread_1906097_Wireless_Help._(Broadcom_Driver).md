### Post by MARP1961 on 2012-01-08
The latest Ubuntu (Oneiric Ocelot 11.10) has the new Unity Desktop with a side panel launcher. There are already a few icons on the launcher; but you might like to add your own or remove the ones you don't use.

Fortunately it is easy to find applications:

Click the top icon on the launcher (the one with the Ubuntu logo). This opens the Dash which will allow you to browse for your application or do a quick search. Simply start typing TERMINAL and it will find Terminal which you can add to the panel if you want.

Right click any of the icons on the panel to KEEP IN LAUNCHER if you want.

As regards your wifi problem, I think the following page might be of interest to you:
[http://www.computerandyou.net/2011/05/how-to-solve-no-wireless-networks-in-ubuntu-11-04/](http://www.computerandyou.net/2011/05/how-to-solve-no-wireless-networks-in-ubuntu-11-04/)

You may need a fresh install with an ethernet cable connected to allow you to get updates and then download your driver.
