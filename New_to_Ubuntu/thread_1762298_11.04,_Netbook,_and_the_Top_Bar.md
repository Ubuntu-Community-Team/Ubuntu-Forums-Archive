### Post by HermanAB on 2011-05-19
On my netbooks, I make a single bar.  Having two bars on a wide screen is really a dumb idea.  It was probably nice way back yoinks circa 1998, but it is not useful this century.

Create a new bar, unlock and drag the elements you need to the new bar, set it to the bottom and delete the old bars.
