And it just stops there.Suspect the part highlighted in red is the problem. Why did you change the boot sequence after installing? What you describe about the rest of the installation sounds correct. Also, just some advice, if you are planning to dual boot with xp when the disc arrives, it is much better for xp to installed prior to ubuntu - it is possible to do it the other way around, but it often cuases more problems than it is worth.
