### Post by tuxxy on 2009-10-04
You should install ubuntu-restricted-extras on a fresh system.  This package includes all your codecs, flash, java plugins.  Search for ubuntu-restricted-extras in Synaptic or enter in terminal;

```
sudo apt-get install ubuntu-restricted-extras
```

As for VLC if you like it then go for it also I think it includes some extra codecs which you may need one day :D

```
sudo apt-get install vlc
```
