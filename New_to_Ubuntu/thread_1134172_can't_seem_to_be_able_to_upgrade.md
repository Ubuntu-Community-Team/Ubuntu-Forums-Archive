In order to upgrade to 9.04 you must do it from Intrepid (version 8.10). You cannot upgrade from a previous version of Ubuntu (*other* than 8.10 - upgrading from Intrepid is easy). If you have 8.04 or earlier, you must install 9.04 from a live CD.
