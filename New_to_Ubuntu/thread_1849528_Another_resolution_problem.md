### Post by LowSky on 2011-09-25
1820x1020? is an odd resolution.

For best responsiveness use the native resolution. Technically using anything over the native resolution on a LCD monitor is going to cause interlacing and posible ripping if watching video, and the only way a 720p monitor can handle 1080 is by interlacing. because 1080i is [s]really[/s] kinda 540p.

Plus using such high resolution is tough on the eyes.
