if I were in your shoes (a new user) I would go for the deb file. You install it exactly as an exe file by double clicking on it. Most of the time will be installed without having to do anything else.

In case it does not (e.g. faced with unmet dependencies) then you have to install manually all the packages required. Try the deb first (especially try the deb corresponding to the version of ubuntu you are using) and see what you 'll get.

edit: If software center is not an option, you can apply the same process to other packages as well. Yet, software center should be your priority.
