### Post by undecim on 2010-04-02
I had an AA1 (I forget which model) on which Linux ran fine. I ran Arch Linux on it before giving it to my little brother, who installed Ubuntu on it and neither one of us has had compatibility problems. Of course, this is to be expected since the AA1 has the option to come with Linux preinstalled (which may save you some money, since you won't need to pay for a Window's license)

As for the modem, I found a couple pages on Google that might be helpful:
[http://en.wikipedia.org/wiki/Huawei_E220](http://en.wikipedia.org/wiki/Huawei_E220)
[https://help.ubuntu.com/community/DialupModemHowto/Huawei/E220](https://help.ubuntu.com/community/DialupModemHowto/Huawei/E220)

According to wikipedia, the modem should work with the Network Manager applet just fine. If not, the help.ubuntu.com link above has manual installation instructions.
