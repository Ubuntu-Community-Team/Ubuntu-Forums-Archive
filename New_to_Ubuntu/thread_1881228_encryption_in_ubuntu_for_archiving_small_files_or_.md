### Post by dave0109 on 2011-11-15
I guess one could say that PGP / GPG is the defacto file based encryption system anywhere.  The gpg command line is installed by default in Ubuntu. You may want to use a GUI as a front end, so look at something like GnuPG Shell or Seahorse (or others).

I don't know if you can create the Self-Decrypting Archives that you want, such that they'd work on Windows, but these work with a shared secret key/password, which then needs to be conveyed by some other means.

Password protected ZIP files might be OK for casual protection. I don't tend to use them and have never looked into their security.
