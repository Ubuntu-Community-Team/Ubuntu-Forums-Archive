### Post by egalvan on 2009-02-19
Hello and welcome.

First, as **overdrank** stated,

Check the md5sum of the file you downloaded.

If not good, download again. And check again.

If good, then boot from the LiveCD you just made.
One of the last options, near the bottom of the list,
is one called "check media for errors" or some small variant.

If good, then as **superprash2003 stated**,
check your system specs.

post your system info, and any errors you may encounter..
