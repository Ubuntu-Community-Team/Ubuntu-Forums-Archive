### Post by zero2xiii on 2013-06-19
Just to be clear,

When you restart the computer, does a "new" screen (the grub prompt) appear with windows 8 as the ONLY option?

May I suggest you go the route of Boot-repair:
[https://help.ubuntu.com/community/Boot-Repair](https://help.ubuntu.com/community/Boot-Repair)
This might be the easiest solution, given it is a easy problem and NOT a UEFI problem and such.

Please see this link (also GRUB2 in my signature) for a whole document on grub and installing, and fixing it to boot OS's and other related things:
[https://help.ubuntu.com/community/Grub2](https://help.ubuntu.com/community/Grub2)
[https://help.ubuntu.com/community/Grub2/Troubleshooting](https://help.ubuntu.com/community/Grub2/Troubleshooting)

Also see my "help us help you" link in my signature for more information.

 
Cheers and good luck
