### Post by Gone fishing on 2010-02-07
Linux has many Window managers, the two biggest are Gnome, which Ubuntu uses and KDE which Kubuntu or Opensuse use (although Opensuse has a Gnome as well). I personally prefer Gnome (but the latest Opensuse KDE has me thinking of changing anyone know what Mints latest KDE is like?)   

Clamav is an opensource Antivirus, Klamav is a front end for clamav for the KDE Desktop.   

However, this begs the question do you need an AV? To protect you Computer running Ubuntu (or any Linux or even BSD or similar) No, you system is very safe from viruses. However, if you share lots of file with Windows users you could install an AV to protect their systems from any viruses on shared files.
