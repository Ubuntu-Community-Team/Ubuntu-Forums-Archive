### Post by NightwishFan on 2008-12-31
Welcome!

Ubuntu is not too hard. Do you notice any problems with Ubuntu right when you install it? Do you get sound and internet?

There should be simple pop-ups for things like Updates and Restricted Drivers by the clock, when you install all of these, then you can work on installing the Wine Windows Emulator. Which should be under Add/Remove in the applications menu. Just hit check and apply.

When Wine is installed I believe you can just treat WOW like it is on Windows.

You can also install the Ubuntu-Restricted-Extras package to get codecs, etc.
