### Post by Bufeu on 2012-07-24
If you run a server with Windows-clients, you may need a antivirus program. Otherwise, follow these tips to prevent a infection:

[LIST]
[*]Don't have Flash or Java installed.
[*]Don't open files with the *sudo*-command if you don't know it's completely safe. The basic principle is to not use *sudo* for open files and documents.
[*]For daily work, don't use a user account who is a [sodoer]("https://help.ubuntu.com/community/Sudoers") (have rights to run commands with *sudo*).
[*]If you are a Firefox user, use [NoScript]("https://addons.mozilla.org/en-US/firefox/addon/noscript/").
[*]Use a firewall, for example *ufw*. If you prefer to use a front-end, a GUI-version for *ufw* is available, called *gufw*.
[*]Don't open unknown e-mails or strange attachments.
[*]Update your system and programs regularly.
[*]Regularly check your PPA's and don't add unknown or less known PPA's to your system.
[*]For unknown or less known programs/scripts, download the source and inspect it by yourself, before compiling and installing. If the own-compiled-version gets a different hashsum than the binary file, don't install it.
[*]Regularly check your auto-startups.
[*]And the most important one: use common sense!
[/LIST]
If you want a antivirus for your system, I recommand you to read [this]("http://www.makeuseof.com/tag/free-linux-antivirus-programs/").
