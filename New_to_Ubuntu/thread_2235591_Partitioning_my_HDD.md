### Post by yancek on 2014-07-21
There are countless tutorials on installing Linux and Ubuntu including the one below which is very detailed.  It also has several links to related tutorials including one on partitioning.  You are not going to see drive name by letter as you do in windows and of course you can do backups.  I'm not really sure what you meant by that.  Before beginning the installation, you should backup whatever is important to you on windows.  If you do not have a windows installation CD/DVD then you should at least have a Recovery CD for windows before beginning.  Simplest way for a new user is to just create one partition for the system and a second small partition for swap.  

Another important piece of informaiton you did not post is what version of windows you are using.  Windows 8 adds complicaations if you are using that.
