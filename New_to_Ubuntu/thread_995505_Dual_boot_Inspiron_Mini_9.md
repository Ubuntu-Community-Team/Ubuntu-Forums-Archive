### Post by Keen101 on 2008-11-28
umm... maybe. ubuntu really requires a minimum of 4gb to install, so you are kind of cutting it a little close there for both xp and ubuntu.

best option would probably be to backup the xp partition with something like clonezilla and then just do a full ubuntu install. Clonezilla is supposed to be an open source utility like norton ghost.

or better yet.... get a 8gb USB drive, and install ubuntu to that, and set the dell mini 9 to boot from USB first.
