### Post by m_duck on 2008-11-28
I'm not sure I completely understand the question, but you want to be typing the command after the details, for example:
```
user@host:~$ command goes here
e.g.
user@host:~$ sudo apt-get update
```Then press enter to execute the command. I hope thats what you were looking for.
