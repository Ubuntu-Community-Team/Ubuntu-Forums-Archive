When I first installed Ubuntu and put in a blank dvd to write, I was asked which app to use as the default. I chose Brasero. Now I would like to switch that to gnomebaker. How would I go about making that switch, so gnomebaker opens when I insert a blank dvd?

### Post by zeroseven0183 on 2009-09-23
Hi! I think here's how it is done:

 1. Open **Nautilus**
 2. click on **Edit**
 3. select **Preferences**
 4. under **Media** tab, define the **Action** (**Other Media** section) to do when a **blank DVD disc** is inserted
     * Instead of the default Brasero, choose **Open with other Application...**
     * Note that if *gnomebaker* is not in the list, you can type it in the **Use a custom command** box

Hope that helps.
