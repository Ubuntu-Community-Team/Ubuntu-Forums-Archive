### Post by Lars Noodén on 2014-01-26
Ideally you should  be able to find the program in the Software Center.  Check there first, it will save a lot of trouble and manage updates for you automatically.  Synaptic does the same thing so you can use that instead, if you have it, instead.  But if the program is not in the Software Center, you might check if there is a[PPA](http://www.omgubuntu.co.uk/how-to-add-a-ppa-to-software-sources-in-ubuntu) instead.  If not, then follow the steps pointed to in #2 above for working with a tar ball.
