### Post by sdowney717 on 2008-11-27
yes, it might be trying to start compiz. You can just get to a command prompt and type

sudo apt-get remove compiz-core

worked for me once when compiz was configured to start at boot and the system video could not handle compiz

stupidly, I had gone to preferences, appearance, and selected all desktop effects. Compiz started and gave me a white screen on every boot. So removing compiz then solved the issue. On that box, we have an x1300 AGP with an intel 875 chipset and compiz does not work because the accelerated video driver does not work.
