### Post by Nepherte on 2008-10-01
There's no guarantee you're "totally safe", it's just harder to do. The restricted user model makes sure nothing happens to important files that are required by the operating system to actually run the system, unless you give permission to it. You won't run into virusses (well you can, but they probably affect windows and not linux), nor do you have to fiddle with firewall settings in most cases. But things like firefox can still be vurnerable (using the noscript extension helps a lot already). Most of the time it's just the guy behind the computer that causes security risks.

### Post by Paqman on 2008-10-01
If you keep up to date with security updates, aren't running a server, and don't get tricked by social engineering attempts (such as this "scareware") then you shouldn't need to worry at all about malware.

The VAST majority of malware will only run properly on a Windows box, and the default Ubuntu install has no ports vulnerable to attack. So there's no need to raise the firewall or install anti-malware apps unless you've done something to reduce this built-in level of security.
