### Post by Didius Falco on 2010-04-06
I think you are coming at this problem from the wrong end.

So long as whatever package you are using in Ubuntu to play music can read the actual files, you don't need to use the Windows link files.

For example, I use Rhythmbox. I simply import all the music files into it and, provided the mp3 tags are filled in, it automatically provides a way to list them by various criteria; Album, Artist, Genre, etc.

What are you using to listen to the music? Then someone can help you with the best way to organize it in that program.

I hope this helps!
