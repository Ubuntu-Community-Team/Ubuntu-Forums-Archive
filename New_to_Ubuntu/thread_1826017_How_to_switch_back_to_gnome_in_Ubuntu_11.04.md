### Post by computerandu on 2011-08-16
Go to System Settings-> Login Screen .....unlock it and then Select Ubuntu Classic as default session...save it and reboot...you are back with gnome2....

if you want to try gnome 3: [http://linux4every1.blogspot.com/2011/07/install-gnome3-on-ubuntu-1104.html](http://linux4every1.blogspot.com/2011/07/install-gnome3-on-ubuntu-1104.html)
