I'm going to try and do a clean install later, and see if that makes a difference.

### Post by loveman on 2011-05-01
ok - got it sorted - open a terminal and paste this command in:

gsettings set com.canonical.Unity.Panel systray-whitelist "['all']"


then logout and log back in again and start vuze running.
The command lets vuze (or other apps) put their icon in the 'system tray' - the little blue frog is back in the bar at the top of the screen so I can click on froggy to get the vuze window to maximise. yay!):P
