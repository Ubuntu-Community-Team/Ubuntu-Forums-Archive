### Post by LowSky on 2011-08-11
I have the same card... very simple to change the audio to HDMI

open sound preference:

Go to output tab, switch audio to HDMI (mine is named GF104 High Definition Audio Controller Digital Stereo.

If it is not there go to hardware tab, make sure the GF104 is one and set to one of the outputs... mine uses Digital Stereo (HDMI) nr 2 Output for my TV. Try one till it works.
