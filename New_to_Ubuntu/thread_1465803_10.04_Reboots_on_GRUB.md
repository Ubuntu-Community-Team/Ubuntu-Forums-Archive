### Post by lreal on 2010-05-12
This is quite a strange problem.

I had already gave up solving that, so I installed Fedora (outside Windows, in a separate partition). But, for my surprise, after installing it, I tried to enter the Ubuntu inside Windows and it worked!

If you want to recover your data, you can mount your Windows partition and then

```

mount -o loop /mnt/windows/ubuntu/disks/root.disk /mnt/ubuntu

```

supposing you mounted the Windows partition inside /mnt/windows. This root.disk is the root of the Ubuntu you installed.
