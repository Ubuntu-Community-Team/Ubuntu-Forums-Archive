### Post by Cheesemill on 2012-10-15
That all depends on the file.

A .tar.gz file is like a .zip file, it can have absolutely anything inside it.
If it is software then there is usually a README file that gives you instructions.

However, as a beginner there should be no reason to use a .tar.gz file, this isn't the recommended way to install software in Ubuntu. For more details please read this page:

[http://www.psychocats.net/ubuntu/installingsoftware](http://www.psychocats.net/ubuntu/installingsoftware)

### Post by Lars Noodén on 2012-10-15
The best way, above all other ways, is to look for that application in the repositories using either the Software Center or Synaptic.  You'll save lots of trouble and extra work that way.  

Unlike Windows, package management in Linux is largely automated.  You can get an overview of your options here:
[https://help.ubuntu.com/community/InstallingSoftware](https://help.ubuntu.com/community/InstallingSoftware)

One of the big advantages is that if your chosen program requires additional helper programs, the package manager will get those for you, too, and install them automatically.

Should you still really want a .tar.gz or .tgz, then they almost always come with instructions.
