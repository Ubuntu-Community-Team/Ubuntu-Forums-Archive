### Post by The Cog on 2009-12-28
Whenever you see commands posted here, they are intended to be run in a terminal which you might start in Ubuntu by clicking Applications in the bottom toolbar, then clicking Accessories and then Terminal. But of course, that's not always the right way to start the terminal. In my Ubuntu, I click the terminal icon at the top-left of the screen, or the Ubuntu logo in the top-left corner then select Accessories/Terminal, because I have customised my GUI. If you have Kubuntu or Xubuntu it's probably different again, I'm not sure. 

In general, most of the commands you see posted in these forums have GUI equivalents. But as you see from the paragraph above, even telling someone how to start the terminal is fraught with ifs and buts. Some things that can be done quickly in the terminal would require an essay and flow-chart with diagrams to tell someone how to do it in a GUI. This is why it's easier to help someone by giving them a short series of commands that will work regardless of the GUI they are using. Also, GUIs often hide the real meaning of any errors. If there are problems, the messages produced from the command line are often more informative.

This **is** the absolute beginners forum. We do generally assume that people know how to start a command prompt though. If you're not sure of anything, ask for more clarification. It can be difficult judging someones level of experience sometimes, and most replies tend to take a middle line trying not to be too confusing or insultingly simplistic. If you ever ask for more explanation, you will not be regarded as being dumb, people will know the original answer was pitched at the wrong level, and you will normally get a whole series of explanations from others who want to be helpful.

As for your problem with not having access to USB or SD drives that you plug in, you certainly should have access to those. Whenever I plug in a USB stick or SD card, an icon appears on the desktop for it. Double-clicking the icon opens a file explorer where I can browse, copy, delete etc. as I want. If this doesn't happen for you then there is something wrong and it may take some digging to get to the cause of it. You will always need admin rights in order to format a drive though. 

It is probably best to pick a specific instance that you are having trouble with rather than a general "I always have trouble with this kind of thing". Creating several threads each with one specific problem is likely the best way of getting your different problems solved. 

So let's kick off with a simple question: Can you use your file explorer to navigate to Places / Computer and then plug in one of the USB sticks you are having trouble with. Does an extra item appear in your file explorer? If so, can you see the contents of the stick when you double-click it?
