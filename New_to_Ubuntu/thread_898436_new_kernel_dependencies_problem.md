You have the proposed updates repository enabled? Unless you are a developer or an experienced user willing to help QA as-yet untested in the field updates, it is a very bad idea to use this repository. Updates from proposed **will** cause problems on occasion.

You may wish to disable this repository, and rely on recommended and security updates only.
