To remove directory
```
rmdir <directory>
```To remove directory plus sub directories
```
rm -R <directory>
```

You're going to have to be more specific than that...  In a file browser, you can just double-click on it of course.  In a terminal you can output the contents of a file to the screen with:```
cat [filename]
```For longer files, instead of "cat" try "less", "nano", or... lots of other things.
