I have some message filters created (Edit - Message Filters) and that seems to work ok for new messages as they come in.

Problem is that all the options listed are greyed out.  So I can't create any rules.

I've searched for similar problems, and found some threads about the "send" button or other buttons being greyed out, but not for the rule options.

I did find a reference to where someone tried to report a bug for the same problem, but the user didn't provide enough info, so the bug was closed due to lack of info.

Anyone else have this issue?  And what should I do to fix it?
