### Post by spcwingo on 2009-09-12
Click on nm-applet in the tray.  Then select manual configuration.  Select unlock in the window that pops-up.  Now select you wireless interface and press properties.  Within that new window you will have an option to select WEP hex after you disable roaming mode.
