### Post by Rex Bouwense on 2013-04-12
Hi.  Have you installed Lubuntu restricted extras?  See here:

[https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)

According to the site, just enter in the lxterminal
```
sudo apt-get install lubuntu-restricted-extras
```
Of course it will ask you for your password which you will enter and hit enter.
