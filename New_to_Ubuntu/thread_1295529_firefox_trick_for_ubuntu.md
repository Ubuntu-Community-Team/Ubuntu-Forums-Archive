### Post by mcduck on 2009-10-19
Sounds like something that would make things rather slow. Having to read the data back from hard drive every time you unminimize a window would pretty much kill the desktop's responsiveness.. Pretty much the same as if you forced everything to swap instead of keeping things in RAM.
