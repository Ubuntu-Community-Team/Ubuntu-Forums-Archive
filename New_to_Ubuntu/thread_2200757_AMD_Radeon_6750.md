### Post by 3rdalbum on 2014-01-20
You are already using the open source driver. It does not support all features of your graphics card and you don't get full 3d acceleration, but it is an open source driver if that is important to you.

The Proprietary driver is made by AMD and supports more features of your graphics card. It has better 3D performance for games. In most cases I would recommend you use the proprietary driver (often called fglrx).
