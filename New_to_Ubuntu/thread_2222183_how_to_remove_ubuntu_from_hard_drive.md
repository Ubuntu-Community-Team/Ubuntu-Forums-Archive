### Post by Vladlenin5000 on 2014-05-05
Indeed, just running the Windows installer will most likely delete all the partitions it doesn't recognize. Errors in Windows or its installer media are completely outside of the scope of the help we can offer here. 
Removing a partition or more however doesn't require any installation media, Windows, Ubuntu, or otherwise. There are many free tools to do that. Almost all require a CD or USB flash memory in order to create the bootable media that'll run the tool. You can as well use the Ubuntu DVD: Boot from it, run GParted and do what you want.
