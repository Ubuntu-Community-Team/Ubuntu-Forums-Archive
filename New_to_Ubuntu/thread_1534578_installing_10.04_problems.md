### Post by CaptainMark on 2010-07-20
have you run a disk check, at the disk menu (which i found only appears now if you mash some keys when the little man appears on screen????) theres an option to check the cd for errors. The most common reason for install errors is a corrupt download or file, you can redownload it and check its integrity using a program called md5sum. Heres the link [https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM) or if you still have the original download you can run it on that.

Ps, this  page seems to have been added to since last time i looked, the first manual method will be the best for you as theres no install needed and you can physically check the hash is correct yourself
