### Post by PierreDeKat on 2008-11-01
If you're using a version of Ubuntu prior to Intrepid (8.10), there's a [wonderful application called Btnx]("http://ubuntuforums.org/showthread.php?t=455656") that will allow you to do anything with your mouse you can possible imagine.

If you're unfortunate enough to be using Intrepid, you're screwed, because [Intrepid broke Btnx]("http://www.ollisalonen.com/btnx/"). 

In that case, please file a new bug report or [go to this bug]("https://bugs.launchpad.net/ubuntu/+source/gnome-control-center/+bug/198313") and complain. Pitch a huge hissy fit. 

Because, if enough people with multiple button mice pitch enough of a fit, Ubuntu's developers may just give us a graphic user interface for configuring input devices in Jaunty Jackalope.

It's only fair, since they broke Btnx.
