### Post by Hospadar on 2009-04-06
Synaptic and Add/Remove do the exact same thing "behind the scenes"

Add/Remove programs is just designed to be a little nicer when you're looking for end-user applications (not like, for example, mystery CLI utility X and strange library headers Y)

So bottom line, they'll do the exact same thing.
