### Post by newb85 on 2012-09-02
If I remember correctly, enabling Desktop Cube disables Desktop Wall, which is the mode Ubuntu uses to switch workspaces by default.  Re-enabling Desktop Wall should return you to the previous behavior.

Also, if you enable both Desktop Cube and Rotate Cube, you should be able to switch workspaces with Ctrl+Alt+Left and Ctrl+Alt+Right.  Moreover, you should be able to change those keyboard bindings if you want under the settings for Rotate Cube, the Bindings tab.
