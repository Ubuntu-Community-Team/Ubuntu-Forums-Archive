### Post by mrbiggbrain on 2009-05-04
you cant do 2 2gb sticks your slot configuration supports a max of 1gb per slot. the slot config is equal to the maximum amount of memory divided by the number of slots... 4gb / 4 = 1gb per slot...

also, you wont see any deacrease as long as you use dual channel, as 4 slots would mean 2 sticks per channel. it might even inrease your memory channel through-put and give you better performance (as the 1gb sticks will run colder)
