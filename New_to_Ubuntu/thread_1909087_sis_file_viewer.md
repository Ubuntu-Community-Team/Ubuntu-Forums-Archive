### Post by charlie_b on 2012-01-14
I'm not familiar with .sls files but a quick search showed that they are simply a text file, showing a list of .jpg files to be played as a slideshow. If that is correct then the actual image files would presumably be stored in the same folder as the .sls file, so you could open them individually using any imaging program. So if you look in that folder, or subfolders called 'images' or the like, then double click on any files with a .jpg extension.
