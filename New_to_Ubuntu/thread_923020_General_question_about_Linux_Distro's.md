### Post by Paqman on 2008-09-17
> **misterhan said:**
> I was wondering if all Linux Distro's were all complicated to connect to the internet.


It's not the individual distros that cause problems, it's the hardware you're using. If you have hardware which supports Linux well, then it shouldn't matter which distro you use, it'll work equally well.

If your hardware isn't supported, then you're probably stuck. Although different distros might have different tools and workarounds available in their repos, you're always going to have a bit of a struggle.
