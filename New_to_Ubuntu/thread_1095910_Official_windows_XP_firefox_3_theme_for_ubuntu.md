### Post by lovinglinux on 2009-03-14
You could install the [Stylish]("https://addons.mozilla.org/en-US/firefox/addon/2108") extension and then load a Firefox style script from [http://userstyles.org/styles/search/Firefox](http://userstyles.org/styles/search/Firefox)

You can load any number of styles you wan and activate/deactivate them with a single click. You can even combine different styles to make "mixed" themes, by activating them at the same time. The order of activation can also produce different results.

Here are some examples:

[Firefox 3 - Vista Theme]("http://userstyles.org/styles/5060") ( I use this one )

[Blue Vista Theme for Firefox 3]("http://userstyles.org/styles/7057")

[Firefox 3 - Internet Explorer 8 (IE) Beta 2 Theme]("http://userstyles.org/styles/10994")

[Firefox 3 - Google Chrome Theme (Dark) Tabs on Top]("http://userstyles.org/styles/10558")

[Firefox 3 - Chrome Theme (Dark Gray) Tabs on Top]("http://userstyles.org/styles/11429")

[Firefox 3 - Google Chrome Theme (Lite) Tabs on Top]("http://userstyles.org/styles/10517")
