### Post by QIII on 2015-01-15
Threads merged.

If you are trying to start Ubuntu like a Windows program, it will not work.

Ubuntu is not a Windows program you run from Windows.  It is an entirely separate and distinct Linux-based operating system.

If you are trying to install it using the Windows installer called WUBI, be aware that WUBI is no longer under development and may no longer work.

### Post by grahammechanical on 2015-01-16
Was this the page that you downloaded Ubuntu from?

[http://www.ubuntu.com/download/desktop/thank-you?country=GB&version=14.04.1&architecture=amd64](http://www.ubuntu.com/download/desktop/thank-you?country=GB&version=14.04.1&architecture=amd64)

You will find the ISO image file in the folder that your browser downloads files to. The ISO image file will be called trusty-desktop-amd64.iso

You then need to "burn" the ISO image to a DVD disc

[http://www.ubuntu.com/download/desktop/burn-a-dvd-on-windows](http://www.ubuntu.com/download/desktop/burn-a-dvd-on-windows)

Or a USB memory stick

[http://www.ubuntu.com/download/desktop/create-a-usb-stick-on-windows](http://www.ubuntu.com/download/desktop/create-a-usb-stick-on-windows)

Then you enter the boot system of the machine called either BIOS or UEFI and change the boot priority to look for an operating system on the DVD drive or a USB port depending on whether you burned the ISO image to DVD disc or USB memory stick.

This is what you will see to install Ubuntu

[http://www.ubuntu.com/download/desktop/install-ubuntu-desktop](http://www.ubuntu.com/download/desktop/install-ubuntu-desktop)
