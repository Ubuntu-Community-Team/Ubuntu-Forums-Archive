### Post by RomeReactor on 2009-02-04
Hi. If you have desktop effects enabled, and you're using a mouse with a scroll wheel, then pointing the cursor on the desktop (_not_ on an application window) and scrolling  will change between desktops; another way is to press CTRL+ALT+RIGHT_ARROW or CTRL+ALT+LEFT_ARROW in the keyboard, and these latter methods work whether desktop effects are enabled or not.

### Post by RomeReactor on 2009-02-04
If you want to keep the desktop effects, install [compizconfig-settings-manager]("apt:compizconfig-settings-manager"); once you have it, go to 'System-Preferences->CompizConfig Settings Manager', open it and find the "Desktop Wall" plugin and uncheck it.

Or, right-click on the workspace switcher in the bottom panel, next to the trash can, right-click on it, select "Preferences", and change the number of rows and columns to one. This way, you'll only have one workspace.
