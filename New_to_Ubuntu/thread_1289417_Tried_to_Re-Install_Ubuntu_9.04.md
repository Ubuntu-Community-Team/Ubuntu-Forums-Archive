### Post by Merk42 on 2009-10-12
If it comes back with errors, first [check the MD5SUM](https://help.ubuntu.com/community/HowToMD5SUM) of the iso you burnt to make sure it's not corrupt.  If it's fine, you may want to try burning again, but this time at the lowest speed possible.
Failing that, you can [install via USB drive](https://help.ubuntu.com/community/Installation/FromUSBStick) if you have one.

### Post by Miljet on 2009-10-12
Very possibly a bad DVD/CD Rom drive introducing errors either during burn and/or read. I've had to replace two on my computers already. You can buy a new drive for little more than the price of a cleaning kit.

If you have access to another computer, you might try burning your install disk there, run CD check. If it shows good, try doing the same CD check on the computer you are having problems with. If the check is good on one computer but not on the other, it is a pretty good indicator that your drive is bad.
