### Post by lovinglinux on 2009-05-30
You need a genuine copy/license to run Windows on any type of system, otherwise you would be breaking the law. Technically speaking, the virtual machine doesn't care about the license or where you get the software, but you will still be susceptible to any anti-piracy measures Microsoft implements on it's softwares, like a regular Windows installation. The virtual machine doesn't magically remove that stuff.

If you don't want to buy a Windows license but need to run some Windows software, then I recommend using Wine. It doesn't require any license and it's not illegal. Other option would be using Windows 7 RC, which you can download for free and legally from Microsoft. It will work for about a year without restrictions.

### Post by keplerspeed on 2009-05-30
Virtualbox emulates hardware for a guest OS. For this guest OS, it thinks it is being installed on a normal computer natively. Pretty much, an OS that you can install natively on your computer can be installed within virtualbox. Only downfall is that virtualised hardware has limitations, such as vb has no SMP support, however it DOES have 64bit support for 64bit processors.

Therfore yes you can install linux, unix, win or any other OS that you can install on your normal, native hardware.
