Ok this is one thing I would also try, nvidia-glx-new is uninstalled right?

Try reverting to a default xorg, installing the driver, if it's still not working try nvidia-xconfig again.

```
sudo dpkg-reconfigure -phigh xserver-xorg     # reverts to a defualt xorg.conf
# restart x
sudo apt-get install nvidia-glx-new (or whatever is the appropriate one for your card)
# restart x, if it's fixed do a little dance if not try xconfig again
sudo nvidia-xconfig
```

I only suggest this because if you can get it to work with the restricted drivers in ubuntu your life will be better for it. 

If all of that doesn't work, maybe uninstall the driver, revert to a default xorg and install envyng, I've found it works quite well.
