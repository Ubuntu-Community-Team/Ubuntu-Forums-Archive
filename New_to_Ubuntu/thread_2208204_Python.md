### Post by lykwydchykyn on 2014-02-27
You can easily learn the core language from online sources; I did.  

As with any language, though, the core language isn't  terribly useful without knowing how to interface with other things, like a database, web service, GUI toolkit, etc.  Some of those things can be tricky to learn from online sources without some real discipline.  Sometimes it's worthwhile to break down and drop $40-$50 on a book to learn an API.

Of course, there are so many free resources these days to learn this stuff, even free online classes (edX, coursera, codeacademy, etc).
