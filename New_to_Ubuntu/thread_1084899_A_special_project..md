### Post by geirha on 2009-03-02
> **frbobs said:**
> Thanks for responding so quickly.

We want to install all Ubuntu on all of the computers and set up the network so that the machines would be regularly automatically be refreshed with new image. Are viruses a big problem for Ubuntu ?

You can set up ubuntu to download and install updates automatically, so they'll stay up-to-date as long as they are connected to the internet. 

As for viruses, they are not a big problem for Ubuntu. See here for a list of all known linux viruses:
[https://help.ubuntu.com/community/Linuxvirus](https://help.ubuntu.com/community/Linuxvirus)

> **frbobs said:**
> 
We have a couple of  network switches we can give them we would also have to provide for printing and sharing the internet connection which they get from a satellite.

Many printers lack drivers for linux, so it's best to make sure the model is properly supported in linux first. There's a database you can search at [http://www.linuxfoundation.org/en/OpenPrinting](http://www.linuxfoundation.org/en/OpenPrinting)

Internet connection sharing:
[https://help.ubuntu.com/community/Internet/ConnectionSharing](https://help.ubuntu.com/community/Internet/ConnectionSharing)

> **frbobs said:**
> 
We would want to simply install open office and skype and Firefox. Their needs are simple there. An internet connection would be a great benefit for everyone.


Open Office and Firefox are installed by default. Skype is proprietary, but packages for it are provided through the medibuntu repository
[https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)
