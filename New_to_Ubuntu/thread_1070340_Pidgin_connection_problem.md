### Post by Unanimated on 2009-02-15
AIM has a native Linux client (it's really old, though), or if you're only using MSN, aMSN is good. It's in the repos. If you absolutely need a lot of services at once, I would recommend [Meebo]("www.meebo.com"). You should also make an account with Meebo. It connects to MSN, AIM, Yahoo!, Facebook, Myspace, ICQ, Google Talk, Jabber, and a lot of other services. If you make an account, it'll connect to them all at once when you sign in. No download. It's what I use, and I have aMSN for when I need MSN file transfers (Meebo only does file transfers to other users using Meebo).
