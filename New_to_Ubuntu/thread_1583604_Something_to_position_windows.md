### Post by yetiman64 on 2010-09-28
It is possible to get the win7 aero snap effect (or very similar funtion) using compiz. Snaps an active window to half screen on the focused window to the left or right of the screen using the mouse moved to the left or right screen edge. Can be set up with maximizing the focused window on the mouse being moved to the top of the screen.

This is the guide I followed here [--LINK--]("http://www.omgubuntu.co.uk/2009/11/get-aero-snap-in-ubuntu/")
Got this working here, but had to change ` symbols copied from the site to ' symbols in the commands to have them work (note the comments also mention this problem with the instructions - pays to read the whole page before trying).

Sounds like what you're describing. Hope this helps.

BTW it's working like a charm here. I'm very impressed now with what compiz can do. (Apparently the info is originally from a tutorial on this site by UF user gotsanity : also may be worth searching out for you)
