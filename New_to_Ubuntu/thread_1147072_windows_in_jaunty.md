### Post by Mark Phelps on 2009-05-04
Before you install Wine, suggest you go to the CodeWeavers site linked below and do searches for the apps you want to install. If you don't see them in the supported lists, or if you do see them and they don't get at least a Gold rating, you're wasting your time installing Wine:

[http://www.codeweavers.com/compatibility/browse/name]("http://www.codeweavers.com/compatibility/browse/name")

If you installed Ubuntu just to run MS Windows apps, I can guarantee that you're going to be disappointed with Linux,.
