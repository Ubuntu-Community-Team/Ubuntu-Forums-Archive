### Post by nhasian on 2009-07-06
Transmission is the default bittorrent client in ubuntu.  you can access it by going to Applications->Internet->Transmission Bittorrent Client.

If you are familiar with utorrent, then you might like to try [Deluge]("http://deluge-torrent.org/") as the layout is very similar.
