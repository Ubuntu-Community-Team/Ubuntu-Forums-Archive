### Post by lovinglinux on 2011-08-26
Firefox 4 reached end-of-life with the release of FF 5, which in turn reached end-of-life with the release of FF 6. Is not recommended to use it, because it won't get any security patches, unless you upgrade to the latest version.

You can't install it via ppa. Only if you download from Mozilla and install it manually.

I would recommend trying Firefox 7.0b1. You can get it from the *firefox-next* ppa. See [Firefox 4, 5 & Beyond Mega Thread]("http://ubuntuforums.org/showthread.php?t=1712247").
