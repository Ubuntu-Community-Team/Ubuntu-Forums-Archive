### Post by androssofer on 2013-05-31
As MG&TL said. Unlike in windows where Ultimate gets your more (and costs more). In Linux you get it all anyway.. So ubuntu ultimate has more display customisation options (animations and window decorations etc)  and a few more things enabled by default.

Ubuntu 12.04 would be slightly less customisation options. And less enabled by default. but you can enable and install most things after its been installed. You don't get locked out of certain features because you didn't hand over enough cash. standard Ubuntu might be less slightly daunting for a new user, but both are awesome. Pick which ever you thing you'd prefer. You won't miss out on much picking one over the other.
