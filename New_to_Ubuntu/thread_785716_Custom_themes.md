### Post by NightwishFan on 2008-05-07
Emerald themes you install the .emerald file with the emerald manager "install" option.

Gtk2.x themes you install but dragging the .tar.gz into the Preferences -> Appearances main window. It will either show up in the main window or under customize.

To theme the panel drag and drop the image you want onto it.
