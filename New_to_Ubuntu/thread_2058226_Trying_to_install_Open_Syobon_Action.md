### Post by mikedafunk on 2012-09-22
Hello,

Thankx for the url of the game ! 

Procedure :
Download the archive and decompress it.  
If you run the game from terminal, you'll see dependency to libsdl-ttf2.0-0
So install libsdl-ttf2.0-0 from synaptic (or sudo apt-get install libsdl-ttf2.0-0)
And the game will normally run !  

Tested on Ubuntu 10.10
