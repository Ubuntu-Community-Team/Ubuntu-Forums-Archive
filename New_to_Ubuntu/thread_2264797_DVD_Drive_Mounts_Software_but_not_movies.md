### Post by monkeybrain20122 on 2015-02-10
Click the gear button on the upper right corner, from the drop down choose System Settings, then go to System > Details > removable media where you can set the default behaviour when a movie DVD is inserted.

Also to watch DVD you need libdvdcss2. [https://help.ubuntu.com/community/RestrictedFormats/PlayingDVDs](https://help.ubuntu.com/community/RestrictedFormats/PlayingDVDs)
