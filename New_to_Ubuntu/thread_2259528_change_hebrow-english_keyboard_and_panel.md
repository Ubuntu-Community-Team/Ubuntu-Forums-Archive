### Post by slickymaster on 2015-01-05
Hi pazadam, welcome to the forums.

First, download the Hebrew language support. You can do this from the **Install/Remove Languages** section of "Language Support" in System Settings. Then, and still in System Settings, double click **Text Entry** and select which language to use and the combination keys to switch between them.

### Post by grahammechanical on 2015-01-05
we are assuming that you are using Ubuntu and not one of the flavours of Ubuntu. If you are using Ubuntu 14.04 or later you should see an icon in the top panel showing an abbreviation of the language for the active keyboard language layout. We can click on it to select an installed keyboard layout or use Super+Space to change keyboard layouts. The super key is the key with a flying windows logo on it.
