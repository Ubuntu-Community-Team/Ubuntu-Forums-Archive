### Post by markjensen on 2008-07-11
In the hidden file .bash_history

There are nice tricks to using the history, like using the "!" exclamation point to repeat a specific command.  Or CTRL-R to 'reverse search' matching commands - hit CTRL-R and start typing something you may have typed before, and it auto-matches previous commands, starting with the most recent.
