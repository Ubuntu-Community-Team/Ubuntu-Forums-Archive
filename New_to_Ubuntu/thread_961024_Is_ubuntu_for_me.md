### Post by starcannon on 2008-10-29
I would recommend a wubi install.

Download an Ubuntu Live CD, burn it, open and close the tray, run the wubi installer located on the CD. This will set you up with an easily reversible dual boot setup. As close to a regular install as you can get; granted, the liveCD is pretty cool, but it lacks a lot of things you get from a Wubi or Regular install.

If you decide you like Ubuntu, there is an option to make your wubi install a regular install. If you decide you don't like Ubuntu, just go to Add/Remove programs in windows, and remove it.
