### Post by lkraemer on 2009-01-28
Have you verified the MD5SUM of the ISO for 8.04 LTS?
If not I'd try that first.  If it verifies, then you should
try running the LiveCD and see if it works correctly.
(You need to burn the ISO as DAO (Disk at Once) and at 4x or 8x.)
If you burnt the ISO with a Windows program, there are also free
MD5SUM programs that are available to verify the MD5SUM of the
image.

You might need to download the Alternate install CD for 8.04 LTS
and go that route.

lkraemer
