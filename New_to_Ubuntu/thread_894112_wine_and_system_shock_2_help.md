### Post by jualin on 2008-08-19
I was reading the link I gave you and it said that someone running the latest Wine was having freezing problems (was that you? :)). Anyways you can always downgrade wine and use an older version which worked for many people. To do this remove the "wine" package using Synaptic Package Manager or via terminal > sudo apt-get purge wine and then get an older release from [their website]("http://wine.budgetdedicated.com/archive/index.html"). Hope this helps!
