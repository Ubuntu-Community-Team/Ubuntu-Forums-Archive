### Post by MattBD on 2008-04-27
It's possible to create your own custom iso image, so you could always do that to create a CD with KDE3 and KDE4 on it. You'd almost certainly need to burn the finished image to a DVD, though, as it would be too big for a CD.

It's pretty daunting for a newbie however, so it's probably not a worthwhile proposition, but if you're interested check out [this article]("http://howtoforge.com/ubuntu-linux-mint-livecd-with-remastersys") at  for details on how to use Remastersys.

An easier option might be [Uck]("http://uck.sourceforge.net/"), but again if you're a newbie you might struggle. Maybe if someone somewhere creates one for their own purposes and posts it online, that might be handy.
