### Post by tangibleorange on 2008-08-14
1. To install more languages: System > Administration > Language Support and check the various boxes by the languages you want to install.

2. To change between them, at the login screen, you should look at the bottom left for 'options' and then select the language from there.

Good luck!

### Post by pansz on 2008-08-14
foreign language support has different level:

1. be able to input foreign language character
2. be able to display foreign language character
3. to have application UI translated into the foreign language.

if you want 1 and 2, you don't need to log-out and log-in at all.

if you want 3, you will have to log-out and log-in.
