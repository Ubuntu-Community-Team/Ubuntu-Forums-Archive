### Post by FM1 on 2009-03-07
Gimp can do that. File > Create > Screenshot. There are options to capture full screen, active window or a selected region, and also an option to set a delay (useful if you want to capture a menu or something that disappears if you hit Print Scrn/Alt+Print Scrn).
