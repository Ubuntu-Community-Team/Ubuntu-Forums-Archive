### Post by Elfy on 2008-08-28
Add it in then - not suire if position is importnat, but if in doubt - mine is below server layout at top of my xorg.conf

```
Section "Module"
    Load           "synaptics"
EndSection
```

Then add the other bits to you input device section, make sure you do a backup

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.2808
```

If it all goes wrong you can revert in recovery mode if necessary

```
cp /etc/X11/xorg.conf.2808 /etc/X11/xorg.conf
```
