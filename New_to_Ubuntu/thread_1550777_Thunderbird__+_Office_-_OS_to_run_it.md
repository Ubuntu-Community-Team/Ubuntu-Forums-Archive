### Post by LowSky on 2010-08-11
Arch Linux would be more suited to your needs.

itisbasi suggestion is great for people who want to preinstall aplicaitons, and not remove certain features.

Arch Linux will give you a very basic CLI based system to which you can install whatever Desktop enviroment you want and only install the bare minimum of applications you need, no extras if you dont want them. Arch also is a rolling release which means update shappen very quickly and easily, no waiting 6 months like Ubuntu.


Arch does have a learning curve but everything is well documented.
