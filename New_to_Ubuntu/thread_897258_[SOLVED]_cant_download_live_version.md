### Post by Sef on 2008-08-22
Try the alternate cd.  Sometimes that will work where the Live CD doesn't.  Just click the button just below the Start Download Arrow of the [download page]("http://www.ubuntu.com/getubuntu/download").  The alternate cd is text-based, but easy to install, especially if you are putting it on the whole disk.
