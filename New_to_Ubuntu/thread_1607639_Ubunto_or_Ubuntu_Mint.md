### Post by mastablasta on 2010-10-28
> **metalf8801 said:**
> The version of Linux Mint that is based on Ubuntu might be a little easier to use then Ubuntu but the main difference is that Mint comes with flash and media codecs pre installed


It's just one of the differences.

Mint is different in the following ways (from what i've noticed):

[LIST]
[*]different menu which also does internet search
[*]different "software center" - where there are user reviews
[*]is not upgraded but has mint backup applicaiton and is then supposed to be reinstalled fresh on each release
[*]different update manager, where a newby friendly mark will say how safe the update is and what are the chances it will ruin your system
[*]some things are preinstalled, but this could also make the basic system to need more resources to run
[*]various other small solutions made to make transition to linux smoother for newbies and some bugfixes that Ubuntu doesn't have
[/LIST]
I still found Ubuntu at the time more intuitive so i went with 9.10 and now i use 10.04 LTS (which is an upgrade).
