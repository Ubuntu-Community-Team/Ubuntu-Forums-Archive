### Post by wojox on 2009-08-24
Don't know how to do it from Ubuntu, but from XP, run regedit.
Then navigate to 
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion
The number should be listed under ProductId.

I used to love hacking around in my registry when I was using Microsoft. Let me know if that doesn't work.
