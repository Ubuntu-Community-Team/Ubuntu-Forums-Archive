### Post by old_dope on 2009-03-05
I was advised to install RealPlayer to play a certain type of file so as i couldn't find it either in Synaptic Package Manager or under Add/Remove i went to RealPlayer's home page and downloaded it from there. The problem is i nolonger want or need RealPlayer so could someone please tell me how i may uninstall it please. Thanks

---

### Post by howefield on 2009-03-05
Load up Synaptic pacakge manager and hit the Origin Button, then highlight the Local/main option. Realplayer will likely show up there and even though you didn't install via Synaptic you can remove it from there.

---

### Post by noobuntu7 on 2009-03-05
Open a terminal (Applications>Accessories>Terminal) and type:
```
sudo apt-get remove realplayer
```

Enter your password and voila.
