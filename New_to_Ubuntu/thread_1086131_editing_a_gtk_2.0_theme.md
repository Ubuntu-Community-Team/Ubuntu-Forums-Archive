### Post by easybake on 2009-03-03
> **libihero said:**
> when searching for themes, i often see in the previews an application that allows the theme to be edited.  what is it called and where can i download it?

I assume you are talking about the widget factory. It's not a program to edit themes, it's just to *display* the theme. 

I don't think that there is a really easy way to edit gtk themes other than getting your hands dirty in the config file. 

However there is [gnome-color-chooser]("https://launchpad.net/gnome-color-chooser") which may be what you are really looking to accomplish.
