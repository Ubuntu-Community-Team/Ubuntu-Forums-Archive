### Post by bobcatpost on 2010-08-13
I have decided to try to learn Linux. I have gotten what was called a "Live CD" of Ubuntu 10.04, where you can boot and run Ubuntu directly from the CD without installing it on your hard drive.
 
My question is: I am currently running XP on my computer. If I boot and run Ubuntu from the CD, can it write to my hard disk in any way?
 
Furthermore, if I connect to the internet, could my hard disk be affected by anything coming from the internet? I am concerned because by running from the Ubuntu CD, I will not have the firewall and anti-virus protection that I have installed in XP.
 
Thanks a lot for your assistance.

### Post by Bachstelze on 2010-08-13
Yes, you can write to your hard drive from the Live CD. However, if you decide to make your hard drive accessible from the live CD, then it is possible (at least in theory) that it will be affected from something you got off the Intrnet.

Basically, yes, it is possoble, but it's good practice to not do it anyway, unless you really need to. Even if malware is not involved, it is very possible if you're new to Ubuntu to make a mistake that could affect your data.

### Post by snowpine on 2010-08-13
> **bobcatpost said:**
> My question is: I am currently running XP on my computer. If I boot and run Ubuntu from the CD, can it write to my hard disk in any way?

Yes; absolutely! Your hard drive should appear in the Places menu, and you can read/write/move/delete files if you like. The Ubuntu Live CD is a great tool for example if your XP install crashes and you need to back up files to an external drive.

While it's true there are very few viruses and vulnerabilities in Linux compared with Windows, you still need to use common sense and browse responsibly. 

"Linux security" is a vast topic, too much for me to explain in a simple forum post. ;)
