### Post by sikander3786 on 2011-04-29
Press Ctrl + Alt + F1 from the Desktop. Can you see the tty1? If yes, login using your credentials and try,

```
sudo service gdm stop
sudo service gdm start
```

Click your username, don't enter the password yet. From the sessions menu at the bottom of your screen, choose 'Ubuntu Classic' and login. This might help if some graphics drivers are needed. Check under System > Administration > Additional Drivers.

Else, post the output of these commands.

```
free -m
lspci | grep VGA
```

### Post by sikander3786 on 2011-04-29
Try configuring a new xorg.conf file.

Backup the existing one.

```
sudo mv /etc/X11/xorg.conf /etc/X11/xorg.conf.old
```

```
sudo dpkg-reconfigure xserver-xorg
```

Reboot and see if it helps.

```
sudo reboot now
```
