### Post by Autodave on 2017-07-12
Sounds like a hardware problem, not a software issue. To confirm, you could try booting to a USB or DVD of Ubuntu. If same problem, then it is hardware.

If hardware, most likely is power supply first and memory stick(s) second. And then motherboard, CPU, etc.
