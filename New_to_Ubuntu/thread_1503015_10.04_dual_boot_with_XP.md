### Post by jnorthr on 2010-06-06
used partition manager to make enough room for / root ubuntu 10.04 partition along with my windoze XP partition. Live Cd installed fine. All worked great until i did a reboot and kept getting 'system error - mount system disk'.

Thinking i had blitzed the grub boot loader, i kept trying to use the live CD to repair it. But to no avail. Then had a cuppa and went back to find that i had left an unformatted floppy 3.5" disk in the floppy drive, took it out and booted up fine !

What a wally 8-[
