### Post by bobnutfield on 2008-12-08
If everything (hardware) is working smoothly for you in Hardy, there is little reason to upgrade.  There is greatly improved wireless support in the 2.6.27 kernel which is used by Intrepid, but there is not a huge enough difference to warrant a major upgrade if Hardy is working well for you.  The basics are still the same.
