### Post by cariboo on 2010-02-22
You can set some printer options in the cups web interface. Go to [http://localhost:631]("http://localhost:631"), select the printer tab, there may be a printer optiohns button on that page. You didn't say what version you are using, but if you are using Karmic (9.10) the interface has changed, so it may take a bit of digging. I have two printers set up on my system, I have to click on one, then click the Administration button to get to the printer options.

### Post by walt.smith1960 on 2010-02-23
I don't know about your model, but Brother MFC series are usually something like:

system>administration>printing>RIGHT click the desired printer>left click "properties">printer options (or something similar). There should be various print quality options.
