### Post by lykwydchykyn on 2011-09-01
There is none, and probably never will be.

About the only thing you can do is 
 - install Wine
 - install the Windows version of Firefox in Wine
 - install Shockwave to your Wine Firefox.

I've done that in the past, and it works OK-ish for some things.
