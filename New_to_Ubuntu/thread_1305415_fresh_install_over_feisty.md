### Post by werecatomega on 2009-10-29
you need to download the iso file at Ubuntu.com, (you should get 9.10) burn it in a certian way on the CD (it might be easier to put it on a usb flash drive (look online for a tutorial)), put it in your computer and restart, while it's loading up (Dell computers: Dell sign and blue bar, Mac computers: mac sign) and if dell, you should be able to press f12 during the loading screen and macs you hold alt. after that choose CD (or usb) and boot from that. after that, choose the install ubuntu and it should be self explanatory from there.
