> **Agusia said:**

A good way to test your hardware, is to boot up the Ubuntu Live-Install CD. With the Live Ubuntu CD, you can test your computer - try out Ubuntu before you install. If everything is working to your satisfaction with the Live CD trial, then select install :)

If your system is freezing, you may need to attempt a second install - first though, I would check out your hardware with a Live CD session.

If you have any data you would like to retrieve, you can use the Ubuntu Live CD - Retrieve your data, test your hardware, install.
