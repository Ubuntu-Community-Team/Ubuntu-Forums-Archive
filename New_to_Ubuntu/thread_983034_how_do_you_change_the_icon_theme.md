### Post by lswest on 2008-11-15
How are you trying to apply the theme?  Open the appearences window, go to the themes tab, drag the tar.gz file onto the window, it will install, then choose the theme you want to apply the icons to, choose "customize" and find the icons you installed under the icon tab in the new window.

### Post by fballem on 2008-11-15
> **Norfeldt said:**
> Whats the difference between all these things.. GNOME, KDE etc..

GNOME and KDE are two different Window Managers under Linux. There are several others, but these seem to be the most popular.

GNOME is the default Window Manager for Ubuntu. KDE is the default Window Manager for Kubuntu. If you want to start a flamewar (definition: a really enthusiastic, not always pleasant, discussion), then try saying that one desktop is better than another. At the end of the day, the desktop is a matter of choice in Linux.

Windows Managers can also define default programs that are designed to work with them.

A starting point for finding out about KDE is here: [http://en.wikipedia.org/wiki/KDE](http://en.wikipedia.org/wiki/KDE)

A starting point for finding out about GNOME is here: [http://en.wikipedia.org/wiki/GNOME](http://en.wikipedia.org/wiki/GNOME)
