### Post by fubbleskag on 2009-01-03
would typing "g nerf" be acceptable? if so:

- create a new bookmark called "google search"
- set the destination URL as: [http://www.google.com/search?q=%s](http://www.google.com/search?q=%s)
- set the keyword as: g

now, typing "g nerf" into your location bar will take you to: [http://www.google.com/search?q=nerf](http://www.google.com/search?q=nerf), displaying full search results.
