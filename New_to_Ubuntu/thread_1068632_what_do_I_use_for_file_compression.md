### Post by hellion0 on 2009-02-13
File-roller is there as a compression and extraction tool. Should be easy enough to find in the Applications menu.

As for winrar for Linux, there's a program called 'rar' that you can install via Synaptic to handle .rar files. It's commandline only, but you can also use file-roller to extract those .rar files once 'rar' is installed.

### Post by mcduck on 2009-02-13
If you are using Gnome, just select the files/directories you want to include in the archive, right-click and select "Create Archive" from the popup menu.

In case you need split rar packages you can use the same way, just install "rar" and "unrar" packages first. Then in the archive creation dialog set file type to "rar" and click the "other options"-arrow to show option for splitting the archive.

### Post by ddmdllt on 2009-02-13
A little note : you should use software related to the .rar" format only if you need to decompress data that you have only in this format.

Else, tar.bz2 archives may be useful and better for you (depending of your needs). (You can manage .tar.bz2 archives under windows too with the free software 7zip)
