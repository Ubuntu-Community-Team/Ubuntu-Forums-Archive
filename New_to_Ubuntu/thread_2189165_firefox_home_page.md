### Post by claracc on 2013-11-21
In firefox, go to settings, general tab. There select "to show my start page" when starting firefox, then in the start page write the url address of ubuntu forums: [http://ubuntuforums.org/index.php](http://ubuntuforums.org/index.php).
Then close and restart firefox.
