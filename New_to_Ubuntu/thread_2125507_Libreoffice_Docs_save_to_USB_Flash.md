### Post by fantab on 2013-03-14
Yes, it can be done.

Format the USB flash as FAT32 from windows or as FAT in ubuntu, which is FAT32. (Even a NTFS formatted USB will work on both).

While saving the Libreoffice created document you can navigate to the USB flash and save it.

### Post by Impavidus on 2013-03-14
Yes, that's possible. Make sure your USB flash drive works with both Linux and Windows (format as ntfs/fat32/fat16, but most usb flash drives are preformatted like that anyway). Then make sure your windows can handle libreoffice files. Microsoft office can do so, but sometimes seems to behave strangly, and you can also install libreoffice on windows.
