### Post by dniMretsaM on 2011-08-02
No, KDE is not sloppy. I use it and I love it (Kubuntu 11.04). The best way would be to run the applicable command on [this]("http://www.psychocats.net/ubuntu/purekde") page in terminal. It will remove your current UI and install KDE all in one shot. Also, you could install KDE next to GNOME if you wish (see [here](http://www.psychocats.net/ubuntu/kde)), but I've heard that this tends to cause problems. I have no substantial personal experience with running both though.
