### Post by r-senior on 2010-03-24
You can create lists of addresses in Thunderbird and use the list as a recipient.

Tools -> Address Book, then File -> New -> Mailing List

Add your addresses, then enter the list name in the "To" field when you create an email.

Open Office Writer has the ability for you to create a letter, to be sent as an email, and then send it to names and addresses contained in a spreadsheet or text file.  Sort of like a mail merge to email.
