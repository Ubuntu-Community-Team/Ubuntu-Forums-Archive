### Post by rosencrantz on 2011-07-05
Are you using the Unity interface (it's default in 11.04) and did you install a proprietary driver for your graphics card yet?
Unity 3d is very picky about rendering. Try logging in with Gnome classic (bottom panel on login screen) and install the proprietary driver with Main Menu->System->Administration->Additional Drivers.
