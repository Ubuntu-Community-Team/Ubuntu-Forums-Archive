### Post by bumanie on 2008-07-11
Unless you have an application that needs java5, I think it would be safe to remove that as java6 is the latest version. Do it via synaptic and choose remove not complete removal, that way if you find something doesn't work, it is easy to mark java5 for re installation. I know I only have java6 and everything works. Have contemplated trying JDK which is an open source version of java only released in the last month or so - but I haven't got around to it yet, so can't advise how well it works.
