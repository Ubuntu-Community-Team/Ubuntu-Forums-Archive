### Post by ChompTheMan on 2008-06-26
Either open up Adept Manager(k menu>>system>>adept manager) and install kbfx or open up  the konsole(k menu>>system>>konsole) and type in:

```
sudo apt-get install kbfx
```

You should now be able to add kbfx through add applet.
