Having multiple plugins installed for the same type of content will only cause conflicts and disable them won't help. See the tutorial above and make sure you have only one plugin for each type of media installed. For example, don't install vlc AND gecko-mediaplayer plugins at the same time. You have to choose one or the other. I recommend gecko-mediaplayer.
