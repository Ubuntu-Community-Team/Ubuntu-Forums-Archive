### Post by Alterax on 2009-05-11
Ideas:

Streaming media server with MythTV

SAMBA/NFS/CUPS print/fileserver

Application server (forwarding apps via ssh -X tunnels) for stuff you use occasionally

Virtualization server--install QEMU and use it to test out liveCDs

LAMP server--install apache2, mysql, and PHP and use it to make interactive database applications.

...lots of ideas.
