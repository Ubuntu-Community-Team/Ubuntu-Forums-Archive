### Post by aeiah on 2009-06-25
place blank versions of files you want as a basis for a template in there, then when you right-click and create document... you can select it as a template file. my template folder has:

document.odt
document.doc
python.py
webpage.htm



for example, my document ones are just blank documents made in open office. my webpage one is a text file with basic html tags already in place like:

```

<html>
<head>

<title>

</title>
</head>
<body>






</body>
</html>

```
