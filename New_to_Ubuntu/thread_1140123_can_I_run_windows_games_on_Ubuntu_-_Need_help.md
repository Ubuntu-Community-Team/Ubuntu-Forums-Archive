### Post by Didius Falco on 2009-04-27
Hi Mal,

I'm a bit of a gaming junkie as well. I'd recommend dual-booting with Windows. I use Windows only for the gaming these days.

Of course, you'll find some few of the major title games that will run natively in Linux, but those are few and far between.

You could also install Wine and use that, but I think it just adds another layer between the game and the pc, so I don't use it - YMMV.

Don't forget to take advantage of all the free, open source games available for Ubuntu. You'll find a bunch of them in the Add/Remove packages interface, and even more in the Repositories by using Synapatic.

I hope this helps...
