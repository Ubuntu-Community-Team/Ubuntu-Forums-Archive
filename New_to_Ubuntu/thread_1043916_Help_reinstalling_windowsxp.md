### Post by diablo75 on 2009-01-19
1.  Go to the website of the manufacture (Dell, IBM, HP, etc).
2.  Click Support.
3.  Look up the model number.
4.  Look for driver downloads.  They're usually self-installer *.exe packages that you can run in Windows.
5.  Save the files to USB thumb drive (even your cell phone might work for this if you have a MicroSD card in it).

6.  Reboot into windows and run the driver install packages one after another, starting with the network adapter.  If you can get internet access, you can do anything.
