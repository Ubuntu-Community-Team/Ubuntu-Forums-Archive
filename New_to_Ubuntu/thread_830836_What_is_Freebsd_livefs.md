### Post by niteshifter on 2008-06-16
Hi,

The 7.0-RELEASE-i386-livefs.iso just provides part of what's FreeBSD's version of a live CD. The other part is 7.0-RELEASE-i386-disc1.iso . It's a text-only Live CD-ish boot (from disc1) mostly for system recovery / maint. work.

Looking to install it? You need disc1, disc2 and disc3 CDs. Or to install via FTP you just need 7.0-RELEASE-i386-bootonly.iso
