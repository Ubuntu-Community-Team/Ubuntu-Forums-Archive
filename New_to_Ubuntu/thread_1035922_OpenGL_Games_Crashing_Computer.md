### Post by paydaydaddy on 2009-01-10
It would be hard to do more than make a few suggestions from the info you have provided. A few things to check:
1. Make sure that you are running the correct screen resolution and refresh rate for your monitor.
2. Be sure you are using the best drivers for your video card.
3. Make sure that your power supply unit provides enough power for your graphics card to work properly. A common problem for many newer computers is that the manufacturers install a PSU that provides the bare minimum voltage/amperage/wattage to power the computer. This is especially true if you have a PCI-e graphics. 

I'm sure that others will add more suggestions. You can help by listing complete system specs.
