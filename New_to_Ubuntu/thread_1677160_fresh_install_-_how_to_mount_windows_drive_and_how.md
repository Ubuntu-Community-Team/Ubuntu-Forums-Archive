### Post by Mark Phelps on 2011-01-28
Just realize that if you mount your MS Windows OS partition read-write, you risk data corrupting and rendering the OS subsequently unbootable.

IF you're going to share data between Windows and Ubuntu, a much safer approach is to move all that data to a shared NTFS data partition.  That way, if you get filesystem corruption of that partition, you can boot into Windows and run chkdsk to fix it.
