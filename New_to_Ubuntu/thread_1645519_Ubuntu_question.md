### Post by libssd on 2010-12-14
> **cvhashim said:**
> So I rebooted and started up Ubuntu but I couldn't find any of my files(Such as Adobe Reader, Java, Mcafee, etc..) from Vista. Is this normal? 

It was working well though.
Ubuntu is a version of Linux, which is a different operating system family from Windows. This classic essay may help clarify the differences: **[Linux is NOT Windows]("http://linux.oneandoneis2.org/LNW.htm")**

Adobe Acrobat reader is an application that is available, but not installed by default, because it is not open software. You can install Acrobat from the last choice on the "Applications" menu: **Ubuntu Software Center**. Java should be installed by default. McAfee (and virus protection in general) is not necessary with Linux. Open Office is the counterpart of Microsoft Office.

### Post by SoftwareExplorer on 2010-12-14
> **cvhashim said:**
> Haha right. 

So does Ubuntu already have these files, or should you download it to Ubuntu?

And should I download straight to my computer, or an empty cd/ or flashdrive?

If you're booted off the Live CD, then you should be able to open you files from the places menu. Just go to the Places menu and then try opening the various "<number> GB Filesystem" menu options. One if them should have your files. 
By the way, Adobe Reader, Java, Mcafee, etc are programs, not files. 

You can install programs in ubuntu under the Applications Menu>Ubuntu Software Center. If you are running the liveCD, installing programs won't stick; you have to install Ubuntu to your hard drive for program installations to stick between reboots.
