### Post by Mark Phelps on 2010-12-06
> **chidat said:**
> Would the recovery work, or would I have to install it from scratch? 
Depends critically on what is on the recovery "disk".  

If it's simply a copy of WinRE (Windows Recovery Environment), which then attempts to find a compressed Windows image on the hard drive somewhere, successful recovery will not happen unless it finds that image.  IF you wiped, erased, or damaged that image during your Ubuntu install, recovery will not be possible.

If, in contrast, the recovery "disk" is a DVD containing a full recovery image, then it most probably WILL be able to restore your PC to its original condition.

I'm betting it's the former, not the latter.

>  Is my product key still valid?
Yes -- but only to install the version you already had on the PC.

If you don't have gparted already in the terminal.
```
sudo apt-get install gparted
```

It will then be in menu-system-admin-gparted.

If it so happens that you are borrowing a W7 dvd that is the wrong distro you can edit a file on the thumb, to let you install the correct distro for your key.
