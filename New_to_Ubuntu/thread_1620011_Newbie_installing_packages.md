### Post by kesman on 2010-11-12
Well you can't do much without internet connection in the Ubuntu box :(
Most software is installed from packages (.deb -files) that can be downloaded separately or installed by synaptic or some other package manager. The problem of not having an internet connection is that packages depend on other packages, and other versions of packages require specific version of other packages. This makes it almost impossible to install these packages without an internet connection.

More info can be found here: [https://help.ubuntu.com/community/InstallingSoftware](https://help.ubuntu.com/community/InstallingSoftware)
