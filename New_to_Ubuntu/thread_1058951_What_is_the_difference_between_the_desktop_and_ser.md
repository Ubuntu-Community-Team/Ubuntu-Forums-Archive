### Post by mcduck on 2009-02-03
Yes it will.

The server version has kernel optimized for server use, and has no desktop by default. It's installer offers you options for installing typical server applications during the OS install.

Desktop version as kernel optimized for desktop use, and includes graphical desktop and collection of typical desktop applications by default.

Both kernels can be installed in both versions, and you can remove the desktop from desktop  version or install it in server version. In other words the only difference is what you get by default. Which version you should install depends on if you want a desktop machine or a server.
