### Post by Lars Noodén on 2012-01-25
If you're looking to set up a firewall, you can use [UFW](https://help.ubuntu.com/community/UFW), which comes pre-installed on your system.  If you want a graphical interface, then you can install GFUW, found in the Ubuntu Software Center.
