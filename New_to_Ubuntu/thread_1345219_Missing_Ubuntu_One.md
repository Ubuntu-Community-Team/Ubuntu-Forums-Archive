### Post by PPPilot on 2009-12-03
After some recent problems, I had to reinstall Karmic. Even though Ubuntu One shows as installed in Synaptic, it does not show up in Places Menu like it did on my previous install. Any Ideas???

Also I was getting error messages from Evolution and since I do not use it I un-installed it. No more errors but even though it has been removed from the Applications Internet Menu it still shows up in the Applications Office Menu. If you select it you get this error "Failed to execute child process "evolution" (No such file or directory)". Makes sense since it has been un-installed.  Any ideas how to get it out of the menu???

### Post by marshmallow1304 on 2009-12-03
I believe you have to run Ubuntu One from Applications->Internet before it shows up in Places.

For Evolution, right click the menu and click Edit menus.  Then you can navigate to the Office menu and either delete or uncheck the Evolution entry.
