### Post by mcduck on 2011-06-15
Yep, the 64-bit architecture is called AMD64 as AMD was the one that developed the original specification for it. They made it, so they got to name it. :)

The same architecture is often called x86-64 or x64 when people prefer a vendor-neutral name, but it's still one and the same specification so AMD64 works regardless of it the CPU is made by AMD, Intel or Via.

(Intel does have their own 64-bit architecture, IA-64, but that's only used on Itanium processors and isn't compatible with the x86 stuff.)
