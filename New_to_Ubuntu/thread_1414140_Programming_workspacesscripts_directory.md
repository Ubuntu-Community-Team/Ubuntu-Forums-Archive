### Post by themusicalduck on 2010-02-23
To add a folder to the places menu, open nautilus. The places are displayed to the left under partitions. Drag the folder to where you'd like it positioned and make sure there is a line displayed between the folders (so that you don't just copy the folder instead) then it should appear in the places menu as well.

### Post by nothingspecial on 2010-02-23
If you put scripts in ~/bin then they are in your path which means you can execute them from where ever you are in the file system without specifying the full path.

Alternatively you can add a whatever directory you wish to store your scripts in to your path.

Since ubuntu comes with ~/bin already in the path then I always put them there - although strangely there`s no bin directory in your home after a clean install.
