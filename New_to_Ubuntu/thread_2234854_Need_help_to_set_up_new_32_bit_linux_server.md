### Post by tgalati4 on 2014-07-17
Do you have any Window-specific software that you need to run on the server?  If not, then you can ditch Win2003 server completely and run your web-page in linux.  If you need .net or other Windows-only extensions, then you could install linux on the server and run the Windows2003 server as one virtual machine and Linux as another virtual machine.

[https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)
