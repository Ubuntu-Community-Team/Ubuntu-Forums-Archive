### Post by joey-elijah on 2009-02-09
This is more likely to be to unsupported hardware than a missing network manager. 

Try installing ndiswrapper and using the windows drivers. Most time this solves it.

Also you cna right click the notification icon for network manager. Make sure you UNLOCK it in order to do any changes!

### Post by houstonbofh on 2009-02-09
Network Manager is a icon in the top right of your screen that loock like 2 computer screens, or a signal strength meter.  But you may have undetected hardware.  Start with the Restricted Drivers Manager in the System --> Administration menu.  If you get no love there, you will have to start digging to find out what your hardware is.
