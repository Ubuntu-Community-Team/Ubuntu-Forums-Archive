### Post by GeordieJedi on 2008-11-13
Areet Scunnered.

I had a simalar problem while I was moving files on a DVD from XP to Ubuntu.

Firstly Don't worry. You should be fine to work with them but first of all, you need to unlock them.

1. Select the folder, right click on it. when the options come up choose properties.

2. When the properites box appears, select permissions.

3. Choose read, write, execute.

4. Now...when you go into the folder, you will see all the pictures in the folder have the lock symbol on them
 (Dont worry! this is normal).

Now select all the files in the folder by pressing "Ctrl+A" then just repeat steps 2-3,
 and Ubuntu should unlock all of the Pictures in that particular folder.

(This is the long hand way of doing it, I'm sure there's a quicker way to do it by using the terminal.
 But at least it works! Although it might take awhile if you have loads of pics on the disc)

You can now do as you please with them. (like re-burn them and try to put them back onto Doze)
