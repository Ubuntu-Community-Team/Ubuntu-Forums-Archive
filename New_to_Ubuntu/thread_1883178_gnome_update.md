### Post by snowpine on 2011-11-18
Let me explain. Work on 11.10 is finished. It was released last month, and the developers are already working on the next release (12.04 in April 2012). 11.10 will nevery get any new features or software versions (only minor updates like bug fixes and security patches).

In other words each Ubuntu is "frozen" at the date of its release.

There are ways to get the latest Gnome in any Ubuntu release but it is not necessarily a beginner project. Most Ubuntu users are happy to use the same Gnome for 6 months, then they update to the next Ubuntu release and use that Gnome version for 6 months, and so on. If it ain't broke, don't fix it!

Make sense? :)
