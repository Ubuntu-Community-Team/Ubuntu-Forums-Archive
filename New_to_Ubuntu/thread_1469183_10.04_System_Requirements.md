### Post by kmsalex on 2010-05-02
Now hold on a second you said you gave it a 10 gb parision right? how big is your hdd how much of it is windows on. boot into windows, run ccleaner (assuming you have) if you don't download and install it IMMEDIATELY, and do a defrage, i would recamend using defragler [http://www.piriform.com/defraggler/download](http://www.piriform.com/defraggler/download) from the same people that made ccleaner.
Then shrink your windows partision by right clicking on computer and select manage. Then 
Storage>Disk Management from there you can shrink you windows partition. now if you want to upgrade you will have to boot into the live cd and use gpart to increase the size of you 10gb partition, or you could just do a clean install and select the partition size from there.

### Post by sunk8 on 2010-05-02
Uninstall apps that u dont normally use.
Use lighter apps - like xpdf instead of evince, mplayer or vlc instead of having both totem n rhythmbox, etc.
Use the computer janitor to make more free space by removing dependencies.
Back up data that u rarely need  on some removable drives - CD/DVD.
Compress drives to save space.

Lubuntu i nice. And if you don't care about looks, go for a window manager instead, like fluxbox.
