### Post by sisco311 on 2009-03-22
Select the file/files and press Shift+Del or hold down Shift and select the Delete option from the Right Click menu.

### Post by lukjad on 2009-03-22
Well, I use pcmanfm for my browsing and deleting needs (*sudo apt-get install pcmanfm*). If you wish to delete something, simply hit the Delete Key. It will prompt you asking if you are sure and then delete it. I suggest pcmanfm because it is much faster and lighter than Nautilus.

### Post by kaibob on 2009-03-22
Nautilus preferences has an option to add a delete command to the right-click menu. The delete command bypasses trash and is in addition to the move-to-trash commmand. The delete command is the same one that sisco311 mentions except you don't have to hold down the shift key to see it.
