### Post by raysa on 2009-07-16
[QUOTE=FinalOnce you have everything set up properly, you'll have no reason to go back to xp[/QUOTE]
... unless you need to run some software that is Windows-dependent and won't work satisfactorily under Wine etc. Lots of Windows software will run on Linux using Wine, but some simply won't. 

I have a couple of big Adobe packages which I use for a client (so can't switch to an alternative) and although they load up in Wine they are unstable and have a lot of features missing. There's only so much time you can spend trying to get software working in the wrong environment - easier to boot up with xp just to get the job done and then revert to ubuntu for everything else.
