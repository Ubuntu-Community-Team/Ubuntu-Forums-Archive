### Post by apostate on 2008-07-09
You already have it. You only get the most basic options by default though, accessible by right clicking on the desktop and opting to change the background. The effects are in a tab called Visual Effects.

To get a nice GUI that will let you change all the effects, change the cube, etc. Use Synaptic and download a package called compizconfig-settings-manager.

This will add an item to your System > Preferences menu, called Advanced Desktop Settings. That is the good stuff. Enjoy!
