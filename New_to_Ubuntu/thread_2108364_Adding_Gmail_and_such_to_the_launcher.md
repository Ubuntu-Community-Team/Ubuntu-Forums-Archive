### Post by mrjava on 2013-01-24
When you go to [www.gmail.com](www.gmail.com), you should see a notice asking if you want to install the app version of Gmail. If you already clicked "dont remind again", you can get the Gmail app here:

[https://apps.ubuntu.com/cat/applications/quantal/unity-webapps-gmail/](https://apps.ubuntu.com/cat/applications/quantal/unity-webapps-gmail/)

If you want it permanently on the launcher, start the app through the Dash, then right-click on its' icon and click "lock to launcher".
