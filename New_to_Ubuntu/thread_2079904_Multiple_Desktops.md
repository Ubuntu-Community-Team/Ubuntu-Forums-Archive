### Post by SeijiSensei on 2012-11-03
Yes, KDE lets you maintain different desktops with unique backgrounds, icons, etc.  Right-click on the desktop background and choose the Activities option to set up the desktops.  Also you can configure the Panel to show only the applications running on each desktop if you like.
