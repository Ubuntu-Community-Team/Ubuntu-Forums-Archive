### Post by grahammechanical on 2024-06-09
Ubuntu Pro is not a computer operating system. It is a contract to provided an extension of maintenance support for the Ubuntu operating system. Read all about it.

[https://ubuntu.com/pro](https://ubuntu.com/pro)

> Anyone can use Ubuntu Pro for free on up to 5 machines

Ubuntu Long Term Support (LTS) versions are supported with security fixes for 5 years. With a Ubuntu Pro subscription a Ubuntu LTS version will get security support for an extra 5 years (10 years in total). I have Ubuntu 20.04 LTS. Support expires at the end of April 2025. I have obtained a free Ubuntu Pro subscription. And now my Ubuntu 20.04 install is supported until April 2030.

Ubuntu and the flavours built on Ubuntu, such as Xubuntu are as secure as an operating system can be for as long as that version is supported by the Ubuntu developers. The flavours, such as Xubuntu have a shorter support life. Xubuntu 20.04 was supported up to April 2023. This is why we are able to do an online upgrade from one version to the next. So, that we are always using a supported version.

Users of LTS versions can online upgrade from one LTS to the next LTS version. So, users of Xubuntu 20.04 LTS would by now have upgraded to Xubuntu 22.04 LTS and be planning to upgrade to Xubuntu 24.04 LTS.

Users do not choose a flavour of Ubuntu because it is more secure than Ubuntu but because they like its User Interface and it may be more suitable for their computer.

### Post by grahammechanical on 2024-06-10
> Does it mean that Ubuntu with out Pro [COLOR=#000000]subscription and[/COLOR]  any [COLOR=#000000] flavour of Ubuntu is as secure as [/COLOR][COLOR=#000000]Ubuntu with Pro [/COLOR][COLOR=#000000]subscription?[/COLOR]

Ubuntu and official flavours of Ubuntu are secure as long as they are in life support. This will explain the Ubuntu release cycle and the length of life support.

[https://ubuntu.com/about/release-cycle](https://ubuntu.com/about/release-cycle)

Because Ubuntu is Free and Open Source Software (F.O.S.S.) it is possible for a single developer or a group of developers to build their own Linux distribution using Ubuntu as the code base. This does not make these distributions Flavours of Ubuntu. Certain official conditions have to be met before the organisation that sponsors Ubuntu grants recognition as a Ubuntu Flavour. The support life of these derivatives and how often the distribution is upgraded is a matter for those particular developers. 

Check out the latest Ubuntu Weekly News letter. It lists the recent security updates as being applied to which version of Ubuntu that are still in life support. This is separate from having a Ubuntu Pro subscription which exten ds the security life for some Ubuntu LTS releases that have reached end of life support.

[https://ubuntuforums.org/showthread.php?t=2498362](https://ubuntuforums.org/showthread.php?t=2498362)

Ubuntu or a flavour of Ubuntu that is out of life support is definitely not as secure as Ubuntu or a flavour in life support. The Ubuntu Pro subscription gives Extended Security Maintenance (ESM) to Ubuntu LTS versions that are about to reach end of life.

Ubuntu LTS with a Ubuntu Pro subscription is as secure as a Ubuntu LTS that is still in life support.
