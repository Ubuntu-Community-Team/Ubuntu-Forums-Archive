### Post by sikander3786 on 2010-12-18
Welcome to Ubuntu and the forums :-)

If you've got a USB stick, you can use that for installation (if your Bios supports boot from USB devices).

[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)

[http://unetbootin.sourceforge.net](http://unetbootin.sourceforge.net)

Otherwise, you can install Ubuntu insdie Windows using the Wubi installer.

[http://www.ubuntu.com/desktop/get-ubuntu/windows-installer](http://www.ubuntu.com/desktop/get-ubuntu/windows-installer)

[https://wiki.ubuntu.com/WubiGuide](https://wiki.ubuntu.com/WubiGuide)

The wubi way is a bit slower and a bit less stable. Let us know which way you decide to go :-)
