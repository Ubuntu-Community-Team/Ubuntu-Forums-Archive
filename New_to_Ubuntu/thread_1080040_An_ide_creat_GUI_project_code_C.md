### Post by anewguy on 2009-02-25
If you are using C, I'll give you one of what may many suggestions:  use GTK (Gimp Tool Kit).  My C has been very rusty, and using GTK I was able to create an application (and far more than just hello world) that uses a GUI in Linux and uses the same code, the GTK for Windows package and the mwing and msys packages in Windows to compile the same source and run as a windowed application in Windows as well.

This is just one suggestion - more people will suggest more just as viable solutions.  There are simple how-to's on the net to get you started with GTK, then it's just a matter of experimenting (and in my case, asking questions!) to get what you want.  don't be intimidated by it - in the long run it really is quite simple!
