### Post by diablo75 on 2009-02-08
I mostly use Amarok for the included shoutcast playlists that have been pre-sorted by genre (over 285 different genres last I counted).  The catch about this is that if you installed Amarok in GNOME, you have to manually install libxine1-ffmpeg to get the streams to play (suggesting a fix for this fell mostly on deaf ears; being told it's not a "bug" but a "feature request", which I would think doesn't f-ing matter.  What matters is whether or not the damn thing works right).  This isn't an issue for KDE users who have installed the Kubuntu extras, but it is an issue for GNOME users, even if they have installed the Ubuntu Restricted Extras (libxine1-ffmpeg isn't included).

Are there any other programs out there besides Amarok that include so many radio stations in it already?  I'd love to see something like that.
