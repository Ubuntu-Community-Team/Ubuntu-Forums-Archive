### Post by Kellemora on 2009-03-20
Hi Ant

Sounds like the autosave feature is turned on.

In some cases it's OK, but in our case we leave it turned off, just for the same problems you are experiencing.

We draw a file from the file server and the auto save will try to save it back to the file server, which is set up as a no can do operation.
Rather than giving an error message, it just hangs, saying it's saving the file, but never does.

There are more than just a few bugs or aggravations in the OOoWriter programs File handling area.
