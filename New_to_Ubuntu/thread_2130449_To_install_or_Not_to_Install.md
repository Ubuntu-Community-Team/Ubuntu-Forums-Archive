### Post by Deaner13 on 2013-03-29
You could format the hard drive and install Lubuntu as suggested. Be sure to get all of your important data off the machine first as this will completely wipe the hard drive.

Typically Windows will start to slow down with age due to the amount of "crud" that ends up being installed with the programs you use, Windows update, and other sources. 

Generally speaking, if Ubuntu was able to run quickly on the machine, a fresh install of Windows would be able to as well. But Linux is better at not getting bogged down from general usage.

I have only a 120GB hard drive and run Xubuntu along side Windows (I need Windows for some Windows only programs that won't run with "Wine") I gave the Windows partition just 35GB.
