### Post by 3rdalbum on 2013-09-07
Somewhere between 95% and 100% of broadband routers have a web-based interface to configure them.  I.e. Plug it in, go to [HTTP://192.168.0.1](HTTP://192.168.0.1) in your web browser and put in the details given to you by your ISP. Obviously you should check that the new router has a web-based or browser-based configuration screen before you buy, but literally every router I have used can be set up this way and do not require Windows software.
