### Post by rhcm123 on 2008-11-02
It's not really dependent on your connection, but on other peoples. Obviously, it will make some sort of difference if you have a fast connection, but not a lot. I have a 1mpbs down-link connection (a 950 kbps uplink) but it still only downloads at about 50 kbps.

How torrents work is, instead of downloading from a central server like iTunes or Zune, it goes to lots of people's home computers, people with the file and seeding it, and then downloads bits and pieces of the file from them, taking those bits to make one whole file.

There is no real way to speed it up - you just have to wait.
