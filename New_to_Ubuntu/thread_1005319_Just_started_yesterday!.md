### Post by DaveyG on 2008-12-08
You probably mean File Permissions :)

To edit the Permissions for your Home folder, you can either CHMOD it ([heres a CHMOD Guide]("http://webmasterworkshop.com/guides/chmod_guide.shtml")), or right-click the file and select Permissions.

By default, you have read/write Permissions of your Home folder.

### Post by Bölvaður on 2008-12-08
#1. Make descriptive names of threads.
#2. Put tech support questions in the Absolute Beginners' Talk and put everything else in the Community Cafe
#3. alt+f2 , write "gksudo nautilus", that gives you admin rights while browsing the file system without touching the terminal. You change permissions with right clicking and press properties. WARNING having super user rights is not the wisest thing unless if you know that you aren't ruining anything ;)

Your home folder has all the rights you need, as you are the owner of it. If you are going to change rights on other places in the file system it is.... not the best way, try maintain the permissions as they come to you.
