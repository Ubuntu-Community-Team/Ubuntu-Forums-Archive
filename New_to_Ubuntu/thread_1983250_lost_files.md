### Post by kolinab on 2012-05-20
Hi!

These 'files' you are describing, are they the icons in the launcher that you can use to open a program? If that is what you are describing, then there's an easy way to get them back. Click on the ubuntu logo at the top of your unity panel. Then in the 'dash' type the name of the application that you want to replace in the launcher. When you see it displayed in the search results, open the program by clicking on its icon. Last, once the program is running, right click on the icon for the running application you want to keep in the launcher, and select 'Lock to Launcher.'

Post back if this helps or if you need more information!
