### Post by papibe on 2011-08-10
Unless you have a modem before the router, you have to connect to the router.

Open 'Network Connections', go to the Wired tab, select eth0, and press edit. In the new window check the box 'Connect automatically', and make sure that in the tab IPv4 the method is set to 'Automatic (DHCP)'.

Check if you are connected to the Internet, if not restart your computer and try again.

Good Luck!
