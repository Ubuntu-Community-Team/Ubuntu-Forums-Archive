### Post by Impavidus on 2019-12-17
You can search [https://help.ubuntu.com/](https://help.ubuntu.com/) and [https://help.ubuntu.com/community/Lubuntu/Documentation](https://help.ubuntu.com/community/Lubuntu/Documentation). Most things other than the desktop environment are the same in Ubuntu and Lubuntu (and all other flavours). Most applications have their own dedicated documentation sites.

Things in the Linux world change rapidly. A book would be outdated by the time it reaches the bookshop, although most command line stuff is still the same as 20 years ago.

The documentation you will find will tell you how to install applications or update the system. When you try that, you'll see that it fails. The reason is that Lubuntu 17.10 is no longer supported. So, I suggest you first install a supported release.

### Post by TheFu on 2019-12-17
Doing a fresh install should only need 15 minutes on current hardware.  18.04 Lubuntu has at least 3 yrs of support, perhaps 5 (I don't keep up with Lubuntu stuff).  18.04 Lubuntu will be supported (i.e. have patches available) until at least April 2021.  I wouldn't waste my time on anything else at this point.

17.10 only had 9 months of support, just like other non-LTS releases.  Nobody really makes documentation for those releases. After 18.04, Lubuntu completely changed from LXDE/GTK+ to LXQt/Qt so 18.10, 19.xx and 20.04 will all be Qt-based.  Everything will look different.

LTS releases seem to always have a manual and books are created to help people new to Linux.  I see an _Ubuntu Unleashed_ book, but I'd wait until the 20.04 version gets released, if I insisted on having a book.  Perhaps visit a used bookstore and find an older book or grab [http://linuxcommand.org/tlcl.php](http://linuxcommand.org/tlcl.php) for free.  TLCL will teach important concepts beyond what a point-n-click interface can show/access.  

LTS releases are easy to recognize - {even year}.{April}  Examples are 10.04, 12.04, 14.04, 16.04, 18.04 .... and the next LTS will be 20.04 in about 5 months.  Any Ubuntu release that doesn't follow that pattern is NOT an LTS.  LTS releases get 3-5 yrs of free support.  Non-LTS get 6-9 months.  *I only load LTS releases for any non-toy use.*

Whatever you do, please don't use 17.10 on any network. It isn't safe.
