### Post by unutbu on 2008-08-04
The font size for some applications can be set by going to System>Preferences>Appearance>Fonts (tab).

Some apps allow you to control the font through the app's own Preference menu item. (Gnome-terminal is one  such app). 

Google Earth's font can be set by editing

~/.googleearth/Registry/google/googleearthplus/User/render/guifontfamily
~/.googleearth/Registry/google/googleearthplus/User/render/guifontsize

If none of the above work, please tell us what app(s) have fonts you want changed.
