### Post by Durden on 2009-07-22
Depends. The one in the repos lacks some USB support so if you need to do anything super fancy the ones in the repo may not be ideal. I would install the repo one first and see if it meets your needs. If not uninstall from the repo and grab the one on their site. 

I ran the official one so I could get my magicjack working but otherwise the one from repos worked fine.

### Post by andrew.46 on 2009-07-24
Hi heyyy,

> **heyyy said:**
> hi id like to install virtualbox but there two versions: one in the repos and one on the official site(which is newer version also) which one should i install?

You have a choice of the Open Source version called OSE which is available in the Ubuntu Repository or from the official site or the closed source version called PUEL which is available from the official site only. Details of the differences can be seen here:

Open Source VirtualBox and other editions
[http://www.virtualbox.org/wiki/Editions](http://www.virtualbox.org/wiki/Editions)

For many people the deal-maker is the usb support available in the PUEL edition. For others, like myself, the Open Source version is always the better choice simply because it *is* Open Source :-).

Hope this helps,

Andrew
