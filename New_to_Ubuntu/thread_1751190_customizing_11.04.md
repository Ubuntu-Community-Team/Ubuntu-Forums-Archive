### Post by 5149.5 on 2011-05-06
It sounds like you are using the Unity desktop and reading threads for customizing the classic desktop. Go to system settings in the session menu which is under the power button on the far right of the top panel. Compiz Config Settings manager is where you set the visual effects. Go slow. Don't change too many things at once. You'll probably want the Unity plugin enabled but thats all of the advice I'll give. I still foul up my desktop every time I make changes.

### Post by r-senior on 2011-05-06
You need Compiz Config Settings Manager (not installed by default). Install it, then go to the Unity Plugin section. 

Alternatively hit Alt-F2 and type "about:config". This presumably still needs CCSM installed.

Threads that refer to the "Visual Effects" tab are for pre-11.04 versions.

Simplify the instructions.

To install Compiz Config Settings Manager

Hit alt-f2 on your keyboard

in the dash that will come up type ```
gksu apt-get install compizconfig-settings-manager
```

A terminal should appear and begin installing it for you, and it should close when it is done.
