### Post by Bucky Ball on 2011-06-30
You will generally not be able to get wireless up from a LiveCD session (depending on the card) as if you need to install drivers for it there is nowhere to install them. Nothing unusual. How easy it will be to get it up in a hard drive install depends on your wireless card. 

Do you know what you have? You should be able to find out through Device Manager in Windows.

### Post by Bucky Ball on 2011-06-30
Yes and no! There was a netbook edition up to 11.04. 11.04 is suitable for all computers as it uses the same desktop environment the netbook edition used (Unity). 10.04 LTS is the current long-term support release and there is a netbook edition of that.

Could you right click 'My Computer' in Windows and got to device manager. Look for Network Devices (or whatever it's called) and double click on the wireless device. That will tell you what it is then we will know if it is going to work ('out of the box' or with some massaging). ;)
