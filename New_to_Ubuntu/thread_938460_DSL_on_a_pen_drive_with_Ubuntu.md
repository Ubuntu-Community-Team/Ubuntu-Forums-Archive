You can manipulate the drive by using the GParted program on Ubuntu.

As for actually installing DSL (I presume you're talking about Damn Small Linux), that's something to learn on [http://www.damnsmalllinux.org/](http://www.damnsmalllinux.org/).  Generally, what you do is boot from a live CD, and then install to the USB flash drive from there.

### Post by C.S.Cameron on 2008-10-06
You should be able to install DSL to a pendrive the same as to hard drive.
Insert CD, insert pendrive and run install from CD.
You may wish to disable your hard drive just to be safe.
The official method is here:
[http://damnsmalllinux.org/wiki/index.php/Installing_to_a_USB_Flash_Drive](http://damnsmalllinux.org/wiki/index.php/Installing_to_a_USB_Flash_Drive)
