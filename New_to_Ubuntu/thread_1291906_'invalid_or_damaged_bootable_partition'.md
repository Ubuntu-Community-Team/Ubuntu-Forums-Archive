### Post by bumanie on 2009-10-16
Don't know much about wubi, but I think if you can still boot into vista, uninstall ubuntu via add/remove. If not able to get into vista, use a vista cd to rewrite the mbr and and then remove ubuntu if this makes vista bootable again. I would then try to reinstall ubuntu on its own partition. Sorry, this may not be of any help especially if you keep getting the I/O messages when you try to boot a live cd. Wubi is a bit of a hack and is  really there as an option for people to get an idea of how ubuntu works - it is generally best to install linux on a separate partition if you are looking at it being for long-term usage.
