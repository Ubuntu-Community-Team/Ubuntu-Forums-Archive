### Post by Golem XIV on 2008-06-24
Add the login screen via System -> Administration -> Login window, Local tab, Add button

Add themes via Preferences -> Appearance.  Just drag & drop the theme (as a tar.gz compressed file) over it.  The themes usually come in the tar.gz package format, if not, try to open the file in Archive Manager and look for the tar.gz package inside.

<edit> Note that sometimes the Login Window app takes a long time to activate.  Just be patient, it should pop up eventually. </edit>
