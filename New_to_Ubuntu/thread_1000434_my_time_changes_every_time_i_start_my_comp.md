### Post by tjwoosta on 2008-12-03
i think your problem is probably more likely to be that your hardware clock and your ubuntu clock are set to two different time systems


windows uses the local timezone to set your hardware clock while ubuntu, i think by default, uses UTC to set the hardware clock

check out this thread
[http://ubuntuforums.org/showthread.php?t=783839&highlight=clock]("http://ubuntuforums.org/showthread.php?t=783839&highlight=clock")

(this is an old thread i made a long time ago when i had issues with my clock in vista but i am almost certain that the solution i followed is the same one you will need to follow to get your clock working correctly)
