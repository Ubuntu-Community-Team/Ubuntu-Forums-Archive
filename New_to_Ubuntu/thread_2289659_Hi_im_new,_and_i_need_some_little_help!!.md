### Post by crusoe2 on 2015-08-07
Hi Cristian, wine can be problematic running certain windows applications. If you have good hardware specs I would recommend running a virtual machine or the like to create a windows system you can use for running all your favorite windows programs. You can install a Mac system as well which may open you up to a broader world of music making potential for your drum creations. I personally run a mac machine for all my music creations using garage band which suits my needs fine. Just a thought!

As ezdrummer has a VST version you can use it in Linux using a wrapper called dssi-vst. Here's a tutorial about installing Reaktor on Ubuntu:

[http://whiteelephantaudio.com/installing-reaktor-linuxubuntu/](http://whiteelephantaudio.com/installing-reaktor-linuxubuntu/)
