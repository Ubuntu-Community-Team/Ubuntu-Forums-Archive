### Post by EGamerHDK on 2011-05-05
I also had this problem, I'm not usuing Ubuntu at this moment, so I can't tell you the exact steps but,

I remember:
Going into the search (Hitting the Windows Key or also known as the Super Key)

Type "Power"

I believe "Power Options" or "Power Settings" is a choice.

It gives you a window with 3 tabs I believe.

1 is labeled "Plugged In"

The other labeled "On Battery"

When "On Battery" check the box that disables dimming the screen.

Hope that helps

**EDIT**: I'm on Ubuntu now. So, try these steps...

Hit the Windows key or Super Key

Type "Power"

Select "Power Management"

Click the "On Battery Power" tab

Uncheck "Reduce backlight brightness"

That should definitely work
