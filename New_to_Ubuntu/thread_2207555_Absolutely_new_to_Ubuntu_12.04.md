### Post by grahammechanical on 2014-02-24
This is a just for information note:

Windows applications do not run in Linux. So, if this is a Windows game, then you most certainly will have problems. You will need to check out this site but do not expect miracles.

[http://www.winehq.org/](http://www.winehq.org/)

Wine can also be installed through the Software Centre. But be warned. It also install some Microsoft fonts and we need to tick a check box to accept a Microsoft End User Licence Agreement. The dialog for this often gets hidden behind the Software Centre window. So, the download will seem to stall. Minimize the Software Centre windows once the download of wine begins.
