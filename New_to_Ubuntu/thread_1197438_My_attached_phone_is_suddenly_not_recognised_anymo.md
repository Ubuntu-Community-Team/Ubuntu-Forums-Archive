### Post by t0p on 2009-06-26
Okay, I've got a very clumsy solution to your immediate problem (getting the file off your phone).

You said that Ubuntu used to recognize your phone okay, but it just stopped doing so.  Was this after a software update?  I hope so, otherwise my following suggestion will not work.

Do you still have the live cd you used to install Ubuntu?  If so, go and get it.  If not, go [download the .iso]("http://www.ubuntu.com/getubuntu/download") and burn it to a disk.

Now, launch a live session.  As the live version running has not been updated, hopefully it will be able to recognize the phone.  So plug it in and see!

Hope this has helped...
