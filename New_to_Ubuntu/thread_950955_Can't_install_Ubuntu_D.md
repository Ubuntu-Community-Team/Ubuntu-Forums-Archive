### Post by bobnutfield on 2008-10-17
Hello and welcome to Ubuntu.  Before you try to install, have you run the live cd as a desktop to confirm it works with your hardware OK?  And, if you intend to dual boot, have you set up partition for Ubuntu, and, very important, if you will be resizing your windows partition, you MUST defrag it at least twice and cleanly shut it down.  Once you have done those things, you can install to your hard drive right from the live cd desktop.  Or have you not been able to boot to a desktop at all?

### Post by bobnutfield on 2008-10-17
When the Ubuntu selection screen comes up, select "Try Ubuntu without Installin", see if it bring you to a live session of the desktop.  If it does get you to a desktop, you will know that your hardware will *probably* work with Ubuntu.  If you have difficulty installing from the desktop of the live cd, you can download the "alternate" cd which will be a text based install, but just as easy.  People can usually install from that even if there are some small hardware conflicts.
