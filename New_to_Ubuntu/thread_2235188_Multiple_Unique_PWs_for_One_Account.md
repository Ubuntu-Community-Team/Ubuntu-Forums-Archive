### Post by Impavidus on 2014-07-19
I don't think so... Anyway, why? If they all share one user account, they can all modify each others files and do silly things to the desktop and application configurations, which they will do, even if just as a joke. That would make the account not more useful than the guest login. Giving them all a unique account seems more practical, with the added benefit that access to certain files can be restricted to, for example, certain groups of students. There are some tweaks possible to make such a large number of accounts more manageable.
