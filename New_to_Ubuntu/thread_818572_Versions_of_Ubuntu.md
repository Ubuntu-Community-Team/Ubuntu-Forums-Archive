Edgy obviously has newer packages than Dapper, as well as new features and improvements.

### Post by Cypher on 2008-06-04
Dapper and Edgy are both fairly old..Dapper was the last LTS (Long Term Support) version of Ubuntu..it is replaced with Hardy..while Edgy, Feisty and Gutsy as the regular versions of Ubuntu with shorter terms of support (18 months??)

The differences most notable are going to be Kernel versions and updated packages all the way around. A few new applications will be added. You can take a look at the release notes for each of those releases to see what's new..
