### Post by Brandon.Viking on 2009-03-05
> **union two levers said:**
> thanks for that, have you any suggestion as to how to get around my problem. i have tried fujifilm cd-r and tdk cd-r and tdk cd-rw's

There are CD-RW which can be written to, and re-written to full capacity if needed.
CD-R can only be written to once but what has been written can not be re-written 

Its not the types of disc you bought. CD-Rs  can be written to in such a way as to allow you to add to them later, its call "Multi-session" it takes an additional 15MB every time you add a session as it puts a new file table on that "session"" but it tis possible to add to a CD-R

Hope that helps explain the difference. Use software when burning that supports multi session. I have used a number under linux and I do not think you can use anything better than K3B. YMMV.

Brandon.
