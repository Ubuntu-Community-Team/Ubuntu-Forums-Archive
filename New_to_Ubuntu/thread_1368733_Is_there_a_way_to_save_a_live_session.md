### Post by zipperback on 2009-12-31
If you want to have an actual installation of Ubuntu along with Windows, but you are concerned about messing something up along the way with Dual boot, rest assured that creating a dual boot system is pretty easy and safe.

However if you really don't feel comfortable setting it up as dual boot, perhaps you might look into one of the many virtual machine systems that are available.

VirtualBox is free open source.
[http://www.virtualbox.org/wiki/Downloads](http://www.virtualbox.org/wiki/Downloads)

Vmware server is a commercial product, however is free to use.
[http://downloads.vmware.com/d/info/datacenter_downloads/vmware_server/2_0](http://downloads.vmware.com/d/info/datacenter_downloads/vmware_server/2_0)

There are others, but those two should give you a couple of really good options.

I hope this helps you.
