### Post by #&amp;thj^% on 2018-11-01
An experimental peer-to-peer Web browser [https://beakerbrowser.com/](https://beakerbrowser.com/)

In Linux (and in some cases macOS) you need libtool, m4, and automake:
```
sudo apt-get install libtool m4 make g++  # debian/ubuntu
```

There is a snap build if you prefer, found here: [https://github.com/beakerbrowser/beaker/releases](https://github.com/beakerbrowser/beaker/releases)

Or you can build from source: [https://beakerbrowser.com/install/](https://beakerbrowser.com/install/)

I have not used or tested it enough to add any input on this browser yet.
