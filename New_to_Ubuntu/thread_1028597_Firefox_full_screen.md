### Post by caro on 2009-01-02
Launch Firefox and if it is in full screen mode, press F11 twice to return it to "normal" with the top panel in Ubuntu showing.  Now click on the "maximize" button to unmaximize the window.  Click it again to maximize.  Close FF and relaunch.  The setting should be saved.  

If not, try the above again and this time resize the window before hitting the maximize button.
