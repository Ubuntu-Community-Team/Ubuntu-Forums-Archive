### Post by a0u on 2008-11-28
> **Beefdawg said:**
> I was wondering If I should get the 32 bit version or the 64 bit version.
Core 2 can run 64-bit.

In general, both should work fine, since virtually all software packages in the repositories are available in both 64-bit and 32-bit versions (unlike with Windows).  However, while you can run 32-bit applications in 64-bit, some software is especially stubborn (i.e., Flash - some users have made it work, some haven't).  Personally, I use 64-bit Ubuntu, and I find it quite stable.

[https://help.ubuntu.com/community/32bit_and_64bit](https://help.ubuntu.com/community/32bit_and_64bit)
