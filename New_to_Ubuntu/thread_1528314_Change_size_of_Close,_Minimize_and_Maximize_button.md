How do I change the size of the Close, Minimize and Maximize buttons for all windows? I was able to increase the scrollbar width using color-changer, but they don't have an option for what I need.

---

### Post by Matt__ on 2010-07-10
system/preferences/appearance
go to fonts tab and adjust window title font which will also change the button sizes.
