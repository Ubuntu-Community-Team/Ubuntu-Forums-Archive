### Post by slooksterpsv on 2011-02-02
Login to the desktop, press ALT+F2, and type in gconf-editor.

Navigate to: Apps -> Metacity -> General. While you have General highlighted, go to button layout and change the second column to: 

menu:maximize,minimize,close

Press enter after you've typed that in and it should be fixed =D
