### Post by ibjsb4 on 2013-04-29
Here's one more thing to consider.

13.04 is the latest release out and lots of good reports on it.  However it will only be supported for nine months.

Ubuntu puts out a version with long term support.  Which right now is 12.04 and supported till 2017.  Also 12.04 has a 2d mode for the new Unity desktop which may better suit your laptop.

If the new Unity desktop is not to your liking you can install Gnome-Classic which looks like the 9.x (gnome2) desktop.  This desktop can be installed on both 13.04 and 12.04.
