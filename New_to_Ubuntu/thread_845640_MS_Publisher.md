I didn't know the answer to your question, but I dug a little and found this: 

[https://answers.launchpad.net/ubuntu/+question/18546](https://answers.launchpad.net/ubuntu/+question/18546)

To install it, use your Synaptic Package Manager (Menu: System -> Administration -> Synaptic Package Mage).  Click Search and enter "scribus", you'll see the one you want to install, scribus, with an ubuntu logo next to it.
