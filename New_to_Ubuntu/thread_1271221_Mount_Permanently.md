### Post by ~sHyLoCk~ on 2009-09-23
> **randolf_ambrose said:**
> someone please help me...

i dont have any persmission at all on my other partition!!!

I have no idea about that software but just sugesting a "GUI solution" for everything isn't really a great idea. Do this:
```
sudo mkdir /windows && sudo gedit /etc/fstab 
``` 

And replace the entry which was generated by that app with:

```
/dev/sda3  /windows  vfat users,auto,noatime,rw 0 0

```

Then:

> sudo mount -a

Hope this helps.
