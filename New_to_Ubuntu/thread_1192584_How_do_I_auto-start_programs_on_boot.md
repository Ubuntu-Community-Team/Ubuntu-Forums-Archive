### Post by nothingspecial on 2009-06-20
You can start rhythmbox playing the radio just typing the url after the the command ie

```
rhythmbox http://www.bbc.co.uk/radio/listen/live/r6.asx
```

Will start rhythmbox playing BBC 6music. Amarok probably works the same way.

Try adding amarok [http://www.url/of/radio/station](http://www.url/of/radio/station) to start up apps.
