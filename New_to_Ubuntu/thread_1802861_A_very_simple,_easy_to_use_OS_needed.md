### Post by I2k4 on 2011-07-13
I'm new to Ubuntu, running Lubuntu 10.10 on an Atom netbook from a 2GB Persistent Live USB very well.  It's snappy and well designed.  The only real problem I had was "Chromium" instead of real Chrome was preinstalled along with much other software I had to delete and replace with what I want.  

I tried the new 11.04 Lubuntu version, and Natty slowed my little netty too much. Lubuntu 10.10 is here:

[http://lubuntu.net/blog/lubuntu-1010-released](http://lubuntu.net/blog/lubuntu-1010-released)

The foolproof "Show Me How" step by step installation using PenDrive is here:

[http://www.ubuntu.com/download/ubuntu/download](http://www.ubuntu.com/download/ubuntu/download)

I suggest at least 4GB USB Drive.  "Persistence" is not mentioned at "Show Me How" Step 2, but it's obvious on the installer - set it to leave at least 1.5GB for the OS, with the rest of the drive set for "Persistence".

Also, if you're running from the USB, you need to mount the HDD for various folder access requirements on your main hard drive - I just open up the File Manager and make sure the hard drive opens up. The user should simply avoid system updates and install software using Synaptic Package Manager one application at a time, so as not to overburden the USB.

Otherwise, quite snappy and running just as well as Windows on my netbook.
