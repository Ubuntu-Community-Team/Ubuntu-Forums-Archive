### Post by Kulgan on 2010-03-05
Ubuntu Studio has a different layout to the default Ubuntu installation. 

Since you have one panel, at least, you can right-click it and select "New Panel". You can then drag it to different parts of the screen by holding the Alt key while click-and-dragging it with the mouse. 

Once you have the panels where you want them, add what you want on them using right-click &#8594; Add to panel. Items you probably want are:
[LIST]
[*]Menu bar
[*]Notification area
[*]Log out..
[*]Show desktop
[*]Window list
[*]Workspace switcher
[*]User switcher
[*]Weather report
[*]Deleted items
[/LIST]
Some of these will probably be in the panel you already have, but adding those to the right places will have it looking like the default Ubuntu install ;)
