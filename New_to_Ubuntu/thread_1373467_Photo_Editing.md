### Post by gunksta on 2010-01-05
Linux is great for photo editing.

[Gimp]("http://www.gimp.org/") -- Installed by default
[Krita]("http://www.koffice.org/krita/") -- This is a KDE app and installs a bunch of extra stuff on Ubuntu, but it's worth it.
[Cinepaint]("http://www.cinepaint.org/") -- Kinda like the Gimp, but better for HDR
[Digikam]("http://www.digikam.org/") -- Professional Grade (or nearly so) Photo Management w/ SOME photo editing capacity (also for KDE and has several dependencies, but is worth it).
[F-Spot]("http://www.cinepaint.org/") - Default photo management tool in Ubuntu. Not my favorite option.
[Google Picasa]("http://picasa.google.com/linux/") - Not open source but is free, as in cost. I know it's running in a special version of wine but I've never used it myself.
Hugin
[qtpfsgui]("http://qtpfsgui.sourceforge.net/") -- Quite possibly, the worst name for a project ever - let's you do funky HDR stuff.

This is not a complete list but there are some very very nice tools here.

If you install Digikam, make you you install the kipi plugins too.
