### Post by davidmohammed on 2011-02-20
1. all packages will be bumped - i.e. will be reinstalled - thus you'll lose any specific changes you made - suggest try using a livecd to test compatibility with your printer
2. If you just use the upgrade manager to upgrade - no you will not affect windows.  But as always - backup your entire harddrive before you start - just in case.  I use clonezilla to backup to an external usb hard-drive.
3. Maverick is 10.10 - not sure what you mean by this.  Maverick works very well with windows in a dual boot situation.  Can you post some links to where you have heard otherwise?
4. software you have compiled yourself and installed will remain - you may have to recompile those again to work against the new libraries etc.  Any deb packages you've install via synaptic/software centre are likely to be upgraded.
