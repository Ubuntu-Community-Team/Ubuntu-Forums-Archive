### Post by ohzopants on 2008-12-19
Limewire can screw up a windows installation for at least 2 reasons:
1) spyware
2) malicious files

Frostwire does not have any spyware, but the potential for malicious files is still there.  Granted that most malicious files are designed to target windows system there still exists a risk for some clever mischief to be done to your system.
