### Post by Paqman on 2008-11-26
There's tons. Personally, I learned from a book ([this one]("http://www.amazon.co.uk/Integrated-HTML-CSS-Smarter-Faster/dp/0782143784/ref=sr_1_2?ie=UTF8&s=books&qid=1227731541&sr=8-2"), to be exact). Take note of the way that book teaches html and css simultaneously, which is the best thing to do. There's absolutely no point in learning html without also learning css.

[A List Apart](http://www.alistapart.com/) is really good for tips and tricks, and don't forget to check everything you write against [W3C's validation service]("http://validator.w3.org/") to make sure it's up to standard.
