### Post by peakpc on 2009-01-07
Adding windows networked printers is very easy, assuming you have samba (windows sharing) setup and can see windows computers in the file browser.  after setting up samba... just go to system,administration,printing then click the new button, then select "Windows printer via samba" and then browse for the printer. the rest is just like a local printer setup. hope this helps

I wouldn't try Ubuntu on that.  If you want a GUI, then your best bet is probably Puppy Linux, although beware that, while it's a very slick little distro, if you expect the GUI to work exactly like Windows or Ubuntu, it's a bit of a shock.  It's a great OS that works directly off a CD-ROM or USB-stick (you can install it on an HD too, through a utility on the LiveCD), and runs entirely in memory.
