### Post by TeoBigusGeekus on 2010-12-13
Navigate to file's folder:
```
cd /path/to/file
```

Make the file executable:
```
chmod +x xTuple-3.5.1-linux-installer.bin
```

Execute it:
```
./xTuple-3.5.1-linux-installer.bin
```
