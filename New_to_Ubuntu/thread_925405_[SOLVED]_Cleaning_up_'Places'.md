---

### Post by myidbe on 2008-09-20
Open the Gnome file browser-> The folders that you see under Places are also displayed there on the bottom left. In this area of the file browser, you can drag folders that you'd like to add to the menu as well as right click on a folder there to remove it. These changes should also affect the Places menu that you reach from your desktop.
