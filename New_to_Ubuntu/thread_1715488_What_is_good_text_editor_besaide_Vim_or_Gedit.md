### Post by mcduck on 2011-03-27
> **sadaruwan12 said:**

Gedit does that as well. And lots more. Have you checked the available Gedit plugins?

It really is a lot more than just a simple text editor, but most of the features are in separate plugins, keeping the main program simple and lightweight while allowing you to get exactly the features you want.

There's some plugins installed by default, bunch more available in the *gedit-plugins* -package, and plenty more here: [http://live.gnome.org/Gedit/Plugins]("http://live.gnome.org/Gedit/Plugins")
