### Post by RiceMonster on 2008-07-27
Try out TuxGuitar. I suggest going to [http://www.tuxguitar.com.ar](http://www.tuxguitar.com.ar) and downloading the Ubuntu installer, that way you'll get the latest version. You'll be able to open and make gp3, 4 and 5 files with it. While it doesn't have ALL of the Guitar Pro 5 features, I find it suits my needs perfectly. That will be much easier than fighting with wine.

### Post by diwas on 2008-08-14
Do this
```

sudo apt-get install wine

```
Alternatively, install it from synaptic...search for wine, check it and hit apply.
This would install wine.

Then goto the installation file of guitar pro...right click it and then there's option: Open with wine...or smthg like that. Then after you do that...its easy...install like u do in windows.
