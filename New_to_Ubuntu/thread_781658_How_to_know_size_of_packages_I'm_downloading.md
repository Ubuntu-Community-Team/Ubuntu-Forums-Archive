### Post by HornedBeast on 2008-05-12
If you install apps using the Synaptic Package Manager, it will tell you how big your download will be and how much more or less space will be used on your drive.

Add/Remove Applications is a simplified version of Synaptic Package Manager, but its worth learning how to use it.

There is a tonne of information on the Ubuntu Wiki on how to use the SPM.
