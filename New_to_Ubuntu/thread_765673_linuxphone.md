### Post by dyous87 on 2008-04-24
> **RadiationHazard said:**
> wait. is there anyway i can change my samsung t509 to run linux?? :)

Since samsung does not release the OS information about that phone we do not know what OS it actually uses. For all we know it may use an embedded linux distro already. 

However that aside if you wanted to install a custom linux yourself:
 
The short answer is probably not.

Longer answer is yes. Theoretically if you have the technical know how you can get a custom Linux build to install on anything. You'd have to know the hardware details of your phone. You'd have to compile the kernel to run on whatever processor the phone uses and then you would need to find, or more likely write custom device drivers for all the phone's features and hardware, including screen, keypad, volume, bluetooth etc. 

This would however be incredibly difficult, even for a team of experienced developers.

So in reality it would probably be virtually impossible to get linux working even close to 100% on such a closed, embedded device.

If you really want a Linux phone I would look forward to OpenMoko or Android. They will be great smart phone platforms once they are finally released. If you want a Linux smart phone now, you can get a Palm running the Palm OS which is Linux Based. 

I hope I answered your question.
