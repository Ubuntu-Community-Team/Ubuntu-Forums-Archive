### Post by WakkiTabakki on 2009-05-18
Yes, and there are short-cuts for moving, copying and creating links:
-- Left-click-move + Alt key = Pop-up menu
-- Left-click-move + Shift key = Force move
-- Left-click-move + Ctrl key = Force copy
-- Left-click-move + Ctrl and Shift key = Create link
