### Post by TBABill on 2010-11-23
I'm assuming you mean you are trying to install programs from the Internet? Windows programs? If so, that won't work in Linux. What you have to do is go to the Ubuntu Software Center (Under Applications menu) and search for the program  you want, click it, click install and then enjoy. Or you can search in Synaptic package manager for the same programs and install that way. But, what you cannot do is try to run a .exe of any kind in Linux unless you are installing through Wine, whose job is to allow you to run SOME Windows programs in Linux. However, the software center or Synaptic download programs from "the repositories", which is where all Ubuntu software is held. 
 
You can also install programs from source, but if you are brand new that's for much later. You may also install programs with file type .deb, which stands for Debian, the base system Ubuntu is built from. 
 
I'd suggest for now, until you learn a bit more about how Ubuntu works and how software is managed in Linux, install programs from the repositories and read up from the forums, the web and anywhere else you can find about how Linux distributions manager software applications. They're all free (well, mostly, but more of that later too) so find what works and install it from the sources suggested and you may be shocked just how much software is at your fingertips for a single click install.
