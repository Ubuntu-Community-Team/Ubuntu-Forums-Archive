### Post by yancek on 2016-01-14
Open the Software Manager which should have an icon in the left panel of the Desktop.  Move the mouse over the icons, it looks like a package.  Open that and in the Search box enter Skype and you should have an option to 'Remove'.  That only works if you used the package manager to install.  If you downloaded it directly rather than through the package manager, post back so someone can explain what to do.

### Post by ajgreeny on 2016-01-14
You may find that synaptic package-manager is a better way to do just that; ie, manage your packages for installing and uninstalling.  It is a lot more flexible and useful than the software-centre, but you will have to install it as it is no longer part of the default installation.

I never use the software-centre, in fact I have uninstalled it (using synaptic) from my Xubuntu 14.04.

As for your skype probem; you will probably find that the skype that you installed first time is now overwritten by the second installation, though without knowing in more detail exactly what you did, it is impossible to know.
