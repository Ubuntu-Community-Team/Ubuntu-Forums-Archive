### Post by Xiong Chiamiov on 2009-01-11
Create a video DVD, add a file, make sure that you set the video file to NTSC, rather than PAL (if you're in North America), and burn it to an iso.  Then burn the iso with GnomeBaker.  Before you burn the iso, you can play it in VLC to make sure it works.
