### Post by Pconfig on 2008-12-14
[http://aws.amazon.com/s3/](http://aws.amazon.com/s3/)

You might want to try this one. It's pretty cheap and fast. Some people wrote a fuse extension for it so you can actually mount this backup service in linux. Then you can just drag & drop your files into a folder which get automatically synced.

### Post by Kellemora on 2008-12-14
Hi kramer

I was using off-site on-line backup services myself for a long time.  Then I discovered something that was more beneficial to me, and a whole lot cheaper too!

The only drawback is they are now 530 miles away and not 2 blocks down the street anymore.  I moved south, they stayed put, hi hi.....

For roughly only 2 bucks more per month (25 bucks a year) I could increase my web page space to 500 gigs and use a hidden folder that's not accessible except by me as my off-site backup.  I've been with them for over a decade and they've never lost a single bit of data yet.  My folder is password protected as well, so not even they can see what I put into it.

Before I moved it didn't cost me anything, as my ISP said, drive space is cheap, just drop me off one of your drives and there will be no charge.  Although I've moved, I still maintain my e-mail and nearly everything else through them.  It's handy having an ISP I can connect to when cable is down here!

I also have local backup in a different building as well.  Triple redundancy, don't laugh, saved my tail lots of times already!  We had a backup set that was nothing but links back to the original drive.  Bug in the backup program, so I've done only Mirrors ever since.  Only way to go as far as I'm concerned!
