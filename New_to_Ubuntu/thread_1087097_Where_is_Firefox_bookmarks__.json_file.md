### Post by lykwydchykyn on 2009-03-04
Under ~/.mozilla/firefox you'll find a folder with a big weird name like "ywvp0w2d.default" (it's random and different for everyone).  This is your profile folder.  Inside it, there is a subdirectory "bookmarkbackups" where these files reside.
