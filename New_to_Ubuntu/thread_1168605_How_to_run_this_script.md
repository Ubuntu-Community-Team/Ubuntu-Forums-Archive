### Post by DizzyEwok on 2009-05-24
Ok, This is a python script, so...
Download the file to your desktop.
Then open up a terminal and type:
```
cd Desktop
```
then type:
```
sudo python pidgin-import-custom-smileys-from-emesene.py
```

Good Luck!
