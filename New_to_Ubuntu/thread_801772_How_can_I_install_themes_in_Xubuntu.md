---

### Post by morgengenuss on 2008-05-21
in case you use the .themes-folder in your home-dir, applications which you run as root (e.g. synaptic) will not have the theme applied. if you want a system-wide theme, extract the archive to /usr/share/themes (of course you'll need superuser-rights for that).
