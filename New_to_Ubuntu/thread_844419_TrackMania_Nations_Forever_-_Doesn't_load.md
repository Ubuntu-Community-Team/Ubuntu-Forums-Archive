### Post by sayakb on 2008-06-29
Press Alt+F2 and type in 
```
metacity --replace
```
And then try running the TrackMania
When you're done, press Alt+F2 again and type:
```
compiz --replace
```

See [here]("http://appdb.winehq.org") for apps supported with Wine.
