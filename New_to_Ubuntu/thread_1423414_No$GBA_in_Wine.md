### Post by mikeTherob on 2010-03-06
Hello there! I believe that wine can be installed through Synaptic Package Manager (System>Administration>Synaptic). Just search "wine" and it should be one of the first options, then just click the check box and then apply and let it do its magic.

That should install wine for you, but an alternative to running NO$GBA in wine (although running an emulator within an emulator is rather ironic) would be to find a Linux equivalent such as VisualBoyAdvance. VBA can also be installed from Synaptic in a similar fashion to wine.

Hope this helps!
