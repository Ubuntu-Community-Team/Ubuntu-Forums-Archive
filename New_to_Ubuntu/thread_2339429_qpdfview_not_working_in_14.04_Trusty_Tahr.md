### Post by #&amp;thj^% on 2016-10-08
Atril comes default in a Mate DE so I have not used Qpdfview for years now.
If you read a bit of epubs this might be of interest to you for firefox: [https://addons.mozilla.org/en-US/firefox/addon/epubreader/](https://addons.mozilla.org/en-US/firefox/addon/epubreader/)
a fairly good extension for the Firefox web browser that lets you read .epubs from the browser, especially good since on Ubuntu you are likely to be using Firefox and it is usually kept open when you are using your computer.
BTW this extension also works in Seamonkey Internet Suite.
There is also fbreader: [https://fbreader.org/](https://fbreader.org/)
That is installed by "sudo apt-get install fbreader"
Sorry not much help on your original query though.
Hope this is useful.
