### Post by Toz on 2011-07-07
If you are using xfce 4.8 (Xubuntu Natty 11.04 or greater),  right-click the panel (bottom tile), select "Panel" then "Panel Preferences". Go to the "Appearance" tab and change the value of the "Alpha" slider.
