### Post by SeijiSensei on 2019-11-22
You can't just copy the ISO onto a stick. You need to use an installer program to create a bootable USB device. If you have a working Linux installation it likely includes the tools required. If you have a Windows machine, use Rufus. 

See [https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)

### Post by TheFu on 2019-11-22
If you have a smaller flash storage device, perhaps a 4GB one, then I'd use that to avoid wiping the data on the big flash drive.

Having a dedicated Ubuntu boot-flash storage device is really handy.  There are times when fixing boot issues or disk issues or partition issues has to be handed by booting from flash media.  Any type of storage that can be booted is fine. SDHC, flash, USB, optical are all fine for emergency needs.
