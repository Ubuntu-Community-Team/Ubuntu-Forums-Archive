### Post by lovinglinux on 2011-06-23
Make sure Firefox is not set to delete NoSquint History on exit:

[[IMG]http://www1.picturepush.com/photo/a/5948409/640/5948409.png[/IMG]](http://picturepush.com/public/5948409)

I did some tests and when I delete the history with that option selected, the page immediately zoom out to the default zoom size. So I believe you have some setting or some extension forcing Firefox to delete NoSquint history.
