### Post by marshmallow1304 on 2010-04-24
I had a look at it.  As you probably already know, the download is available here:
[http://celtx.com/download.html](http://celtx.com/download.html)

After choosing the Linux/English option, I extracted the tar.bz2 archive, navigated to the resulting directory in a terminal, and ran
```
./celtx
```

You may also double-click the executable text file "celtx" and select "Run".
