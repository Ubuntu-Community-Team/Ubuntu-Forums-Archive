First off, the program. It might be difficult to determine- but you could definitely install the application afterwards. If you remember the name of the program(s), you could simply go to the software center and install them. 


If you do a reinstall, you have a few choices, you could simply overwrite your previous partition, you could create an other one, or you could even fine-tune how you want to install it; picking and choosing between various partitions for different data.
