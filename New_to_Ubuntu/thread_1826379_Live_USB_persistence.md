### Post by Rex Bouwense on 2011-08-16
Not sure if I understand your question.  You bought a live USB (flash drive I assume).  It is a live flash drive of some distro of Ubuntu (again assuming).  Nothing is staying?  (Any changes that you make to the OS do not remain upon reboot).  If all of those assumptions are correct, it is probably because there was not a persistent file set up when the flash drive was created.  It can used to boot and install the OS on a computer but without persistent files you will not be able to make changes to it.
