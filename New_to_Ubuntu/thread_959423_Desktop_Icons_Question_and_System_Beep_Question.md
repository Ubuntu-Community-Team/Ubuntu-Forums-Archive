### Post by LowSky on 2008-10-27
Figure I would post it here
[QUOTE=b0ng0]Right click on the Ubuntu Menu in the upper right and select "Edit Menus". Go to the system tools option on the left and then select Configuration Editor.

Now open up Config Editor from your menu. Then go to apps->nautilus->desktop and deselect the "volumes visible" option. Et voila.[/QUOTE]
