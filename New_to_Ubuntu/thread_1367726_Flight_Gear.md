### Post by Paqman on 2009-12-30
A good way gain permission to write to those folders is to install a plugin called [nautilus-gksu](apt:nautilus-gksu) (click to install). Then right click on the folder you want to copy the files into and "open as administrator". This will launch a file browser window that has permission to write to that folder. You should make sure to close this window after you're done copying the files.
