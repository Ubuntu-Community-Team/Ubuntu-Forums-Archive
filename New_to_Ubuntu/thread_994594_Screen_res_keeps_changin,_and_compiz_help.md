### Post by gettinoriginal on 2008-12-04
Install "CompizConfig Settings Manager" from synaptic.  After installation you will find it in System > Preferences.  Make all you settings from there.

Edit:
Also, if you plan on using animations, while you are in synaptic, install "Simple CompizConfig Settings Manager"
