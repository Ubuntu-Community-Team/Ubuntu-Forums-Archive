### Post by kanzan on 2011-03-24
I was keen to set my desktop up as a dual boot system, but for months couldn't figure out why my system kept going into sleep mode when using the Live CD. I read thread after thread, tried fiddling with the BIOS settings, and tried to find a way to get the nvidia driver installed (couldn't do it with a live cd, live usb, or persistent usb).
 
I have a custom desktop with a FoxConn 9800GTX graphics card and Acer AL1914 monitor, and don't have this problem when using Windows. When using the live cd, though, the computer would ignore anything I set using power management, would go to sleep, and wouldn't wake up no matter what I tried. For the life of me, I couldn't figure out what the issue was, and none of the solutions worked... until today.
 
The 9800 GTX has two DVI ports. I had my monitor plugged into one of them, and never thought to try the other one. Today on a whim I decided to switch the port I was using... When I plugged the monitor into the other port on the card, the computer would wake normally from sleep, and follow the settings I indicated using power management.
 
Problem solved... sometimes the solution is so simple, it's easy to overlook...:D
