### Post by jaddi27 on 2013-03-10
You should be able to use Ubuntu Server edition for that. I would suggest using the LTS version, so it does not need to be upgraded as often. Ubuntu Server 12.04 is the latest LTS version, and is supported for 5 years.

There are packages for apache httpd, php and other required web server programs. I suggest taking a look at [https://help.ubuntu.com/12.04/serverguide/httpd.html](https://help.ubuntu.com/12.04/serverguide/httpd.html) for more details.
