### Post by Paqman on 2013-03-06
> **carl4926 said:**
> 
WUBI is like linux with training pants on

Not really. Wubi is a perfectly good way of installing Ubuntu. Apart from the slightly different installer it functions almost exactly the same as a traditional install.

**Scooterdork07:**
Generally you'll find Windows games work best in Windows, so if you've got Windows available then that's the easiest thing to do. Wine can be a bit of a mission to get set up right.

Having said that, I'm told [WoW does work well in Wine]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=1922"), so you should be able to play it. I'd suggest installing Playonlinux (it's in the repositories, just open up Software Centre and install it). Use Playonlinux to install WoW, as it handles any little tweaks the game might need to run properly.
