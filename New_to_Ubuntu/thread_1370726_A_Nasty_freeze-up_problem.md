### Post by benerivo on 2010-01-03
Those videos are pretty jittery. The best performance i can get is by using the 64bit version of Google Chrome along with the 64bit version of flash (you'll need to be running 64bit Ubuntu for this to be a possible solution, which you can do as your machine has an AMD64 processor). In my opinion this is the best way to watch flash in linux, and those videos seem fine using this combination.

[Google Chrome 64bit]("http://www.google.com/chrome/eula.html")
[64bit flash]("http://labs.adobe.com/downloads/flashplayer10_64bit.html")

To get 64bit flash working, you'll have to follow [these instructions]("http://ubuntuforums.org/showthread.php?t=1358591").
