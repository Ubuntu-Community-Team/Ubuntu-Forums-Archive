### Post by John Bean on 2009-10-21
> **guriinii said:**
> I've got a massive problem!! I got grub error 17 and cannot access ANYTHING! So I'm downloading the 9.04 .iso file to install fresh. 
I have a Toshiba Satellite Pro A10 and am not sure which boot priority to use as it doesn't have USB-HDD option.

Please, can anybody help?!

You'll need to make a bootable USB drive from the ISO. Use either a flash device (1GB will be fine) or a hard disk. Use unetbootin ([http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)) or similar to make the bootable disk from the ISO.

Make sure it's plugged in before booting, then have a look at the  boot options offered by the bios - you should be able to select your USB drive from there. After that you're home and dry... hopefully ;-)
