### Post by Yuki_Nagato on 2008-12-06
Could be a bug.  Let us try an earlier version.

Remove the add-on from Thunderbird.  Got to Tools -> Addons and from there you should be able to uninstall it.

Then open up the Synaptic Package Manager.  Search for "Lightning."  It should show the "Calendar extension for Mozilla Thunderbird."  Check the latest version column.  Does it say 0.7, 0.8, or 0.9?  Take note, and then install through Synaptic.

If it still does not work, select the extension in Synaptic Package Manager, Go to Packages -> Force Version, and that should force the earlier, working Lightning.
