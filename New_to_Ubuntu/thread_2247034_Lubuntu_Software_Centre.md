### Post by Impavidus on 2014-10-05
I can think of two reasons why Lubuntu would work where Ubuntu doesn't. Either memory is unsufficient for Ubuntu – but 4GiB is more than you need – or the graphics card doesn't work with Ubuntu. Lubuntu doesn't rely on powerful graphics hardware for just drawing the desktop. You may need to install a proprietary graphics driver. Also, as you constructed this computer from several hardware components yourself and hadn't used it before, it's possible that not every component is properly connected. A loosely connected ram module can cause strange crashes.
