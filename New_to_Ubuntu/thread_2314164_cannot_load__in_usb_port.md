There might or might not be problems with the graphics card. We will find out later. But the first thing to do is to get a light-weight flavour of Ubuntu. With a Pentium 4 CPU and 1 GB RAM I would recommend the ultra-light ***Lubuntu*** or the medium light ***Xubuntu*** or ***Ubuntu MATE***.

I suggest that you try version 14.04.1 LTS, or if problems 14.04.4 (brand new point version), or 15.10.

[It will probably work to boot from a USB pendrive](https://help.ubuntu.com/community/Installation/FromUSBStick)

If not, you can start from a Lubuntu ***alternate*** iso file of version 14.04.1 LTS, which is small enough for a CD disk.

[Try Ubuntu (Kubuntu, Lubuntu, Xubuntu, ...) before installing it](http://ubuntuforums.org/showthread.php?t=2230389)

If problems with graphics, try with the boot option **nomodeset**.

[Boot options](http://ubuntuforums.org/showthread.php?t=2230389&p=13370808#post13370808)
