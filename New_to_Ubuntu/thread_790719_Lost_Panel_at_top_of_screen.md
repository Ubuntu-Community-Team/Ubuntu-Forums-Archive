You'll have to re-add each item though, which you can do by right clicking on your new and using "Add to panel".

### Post by sdowney717 on 2008-05-12
if you ever find yourself stuck with some weird situation.
Create a new user. The new user will have panels and menus. etc...

That happened to me once, could not load the gnome desktop, claimed it was a serious error. etc,,,
Everyone has their own user space. The file system is separate, so if the user space gets messed up, you can boot into a new user space and perhaps fix it or get the files out that you need.
To start up file manager with super user priv, just type at terminal prompt
sudo nautilus
