### Post by vasiliymeshko on 2008-04-29
With regular ISO you can boot from the CD and have a live session that will be identical to what your system will look like once installed (ideal for testing hardware compatibility).

The alternate disk, on the other hand, has a text-mode installer and no live session. This can be beneficial when installing on older hardware where, for example, most of the system memory would be eaten by the live session, leaving not enough to run the installer
