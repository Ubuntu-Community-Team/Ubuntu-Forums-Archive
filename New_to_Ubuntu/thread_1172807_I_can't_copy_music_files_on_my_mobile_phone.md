### Post by Zimmer on 2009-06-14
Mobile phones notorious for USB connectivity problems.

Mostly this is overcome by transferring files to the additional memory card, not the phone internal memory, in MASS DATA STORAGE mode.

Or, if you must, by accessing the internal phone memory using Bluetooth.

Not to forget to mention removing the microSD card and putting it in a card reader....
