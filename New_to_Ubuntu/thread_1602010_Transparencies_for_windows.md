### Post by bigsmitty64 on 2010-10-20
In compiz config settings manager, go to the settings for **Accessibility:** "opacity brightness and saturation" make sure its checked then you can press alt and scroll your mouse wheel down and up to control the opacity of any window! Or you can configure the buttons how you like em! Hope this helps.

### Post by trikster_x on 2010-10-20
You are looking for the opacity options.  I'm not familiar with the simple ccsm tool, but there should be a check box or something else to enable it, then you can play with the settings for how opaque you want the window to be.  It does not function exactly like the terminal's opacity feature does.  I'll include a screenshot of a calculator on top of my browser window with the opacity feature working.  I have mine set to 30% which makes it invisible enough to not be intrusive, but opaque enough to see any text you enter.  I would suggest you install CCSM (compiz config settings manager) for more fine control of addons.  If you play with some of them, you might find you really like the "extras", like scale, or window picker.

EDIT: there are actually two options for opacity in CCSM:  the option bigsmitty64 mentioned, which allows you to set opacity permanently to specific window types and allow full control over when windows opacify, and the "opacify" option, which allows you to set windows to automatically opacify when you move the mouse off of them and onto another window, and revert to normal when the mouse is back over them.  I prefer the latter, since it allows me to move back and forth quickly without extra keystrokes.
