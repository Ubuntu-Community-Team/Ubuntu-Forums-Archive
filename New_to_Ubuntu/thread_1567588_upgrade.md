### Post by kroq-gar78 on 2010-09-04

### Post by sikander3786 on 2010-09-04
That is possible. I don't know if directly it is or not, but indirectly, it definitely is. Install the package kubuntu-desktop on your existing 9.04 install, then log in to Kubuntu, and add Kubuntu 10.04 cd as a repository. 

[https://help.ubuntu.com/community/Repositories/Ubuntu](https://help.ubuntu.com/community/Repositories/Ubuntu)

You should be able to upgrade.
