### Post by RomeReactor on 2009-07-30
Hi. Open a media player and start playing an audio file; then right-click on the volume control applet in the top panel, select "Open Volume Control" and start playing with the sliders for the different channels. It's possible your master channel was changed to something else than the correct one. You can also try going to 'System->Preferences->Sound" and click the "Test" button on the "Music and Movies" section to see if it works. Are the settings there set to "Autodetect"?

As for the CPU usage issue, open the System Monitor in "System->Administration", go to "View" in the menubar and select "All Processes" and sort the fields by **% CPU ^**. You should then be able to see what is consuming all those CPU cycles.

Can you give us more details about your hardware?
