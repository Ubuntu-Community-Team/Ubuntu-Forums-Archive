So, what changed in 8.1? It used to be quite simple because it was like any other USB stick, ie, it used the standard mass storage device profile (so did [Android]("http://www.howtogeek.com/192732/android-usb-connections-explained-mtp-ptp-and-usb-mass-storage/") until recently). A quick google search brings some confusing info: On one hand, Microsoft support it telling MacOS users they should use a (proprietary) app for WP from their store *and* a special app for photos/videos, indicating a similar move to Android, but to a proprietary protocol; on the other hand, Microsoft is also pushing apps for newer Windows but at the same time saying that XP/Vista users can use the Windows Explorer... So, unless they quietly pushed an update containing the support for the new type of connections for XP and Vista, supposedly it should still work as a "USB mass storage device class".
