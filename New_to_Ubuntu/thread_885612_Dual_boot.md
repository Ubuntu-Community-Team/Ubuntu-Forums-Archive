### Post by chrisod on 2008-08-10
It's not clear from your post, but if the 32 bit software that you want to run is Windows software, you may end up disappointed. Linux can't run Windows software natively. You can run some Windows programs with Wine, but I'd recommend checking the[ Wine dB]("http://winehq.org") first to see if the specific applications that you want have any hope at all of running decently under Wine.
