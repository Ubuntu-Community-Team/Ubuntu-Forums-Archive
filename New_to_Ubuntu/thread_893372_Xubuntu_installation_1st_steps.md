### Post by Kevbert on 2008-08-18
It may be that your download was bad, that the burnt CD was bad or you don't have enough memory for the install.  If you've downloaded the standard version you need 384Mb of RAM to install.  The alternate version (text based install) needs a lot less.  Once you've downloaded the ISO file have you performed a [[COLOR="Red"]hash check[/COLOR]]("https://help.ubuntu.com/community/HowToMD5SUM") (checksum) ?  (Hashes are linked on that page you want the desktop i386 version by the look of it). Next you need to burn the CD at the slowest speed possible to minimize errors.  Finally you need to check the integrity of the CD (a menu item).  A memory check (via the boot menu) would also be a good idea.
