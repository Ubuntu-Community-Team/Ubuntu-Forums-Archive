### Post by victorgreen on 2009-02-06
Its certainly a screen resolution problem. If messing with the settings there didn't do anything, then its probably something up with the driver (or lack thereof) for whatever graphics chip your laptop has. 

If its ATI or Nvidia there are drivers available and thread upon thread on exactly how to make em work correctly. If its Intel onboard graphics then at this point it should pretty much work out of the box on the newest releases. If its some really old chip then it should work with older releases and you can probably hunt down the driver and make a newer release work. 

So to sum up
the model of laptop?
the graphics chip of it
how old is this thing
and finally what release have you been using so far on this thing?
