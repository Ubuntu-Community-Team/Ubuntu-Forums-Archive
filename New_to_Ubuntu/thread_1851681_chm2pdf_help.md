### Post by anewguy on 2011-09-28
You run it from the command line, and there is a man page for it.  So -

- open a terminal window

- type:

man chm2pdf <press enter>  for the manual page

---- or ------

chm2pdf --help <press enter> for similar information


This information will tell you what options to use to point to your input files, etc.. 

Post back with any questions.
