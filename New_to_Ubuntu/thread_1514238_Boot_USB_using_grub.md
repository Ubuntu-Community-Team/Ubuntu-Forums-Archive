### Post by bwallum on 2010-06-20
A really good solution is to use a USB external hard drive. Nice and compact and you can get up to 500GB using two USB ports. Brilliant for adding an additional OS, a partition that can be seen by all OS's (fat32 or NTFS) and still loads of room for backup (Back in Time).

I've done this for a number of people and would be pleased to assist. Look for 2.5" external USB hard drives for about $80/£60.
