### Post by bogan on 2012-07-23
Hi!,** czgirb**,

I have two 12.04 installations on the same computer, one is an updated version - originally 9.10 - the other a direct install using the pae kernal.

In respect of your query they behave slightly differently; though when logged-in to Gnome Classic, they both have the standard unity style Launcher set to 32 pixels, including the 'Dash'.

When logged-in to Gnome Classic  (No Effects), [ie session fall-back] the first has a Menu entry: Applications>Accessories>Unity 2d Launcher, which the PAE version does not have.

Both have a 'Unity-2d-shell' entry which , like 'unity 2d launcher' displays the unity style Launcher with 'Dash', but in the full size, and cannot be reduced to 32 Pixels.
[ Ubuntu-Tweak only works fully in 3D.]

If you do not have 'Main Menu', needed to display these menu items, you will need to install it [aka 'Alacarte'] from the Software Center.

I know this does not meet your wishes, but thought the info might help a bit to know what can be done with a standard set-up.

Chao!,** bogan**.
