### Post by ugm6hr on 2009-02-01
If you have an unstable internet connection, upgrade (or do a full reinstall) using the Alternate CD.

Download it using Torrent (which will allow integrity checks / restarts without losing already downloaded data).

Then, boot back into Ubuntu and then mount the iso as a CD.  You should be able to launch it from one of the icons.

If that doesn't work, you may need to burn the CD and put it in the drive.  That definitely offers to upgrade as soon as you put it in the drive.
