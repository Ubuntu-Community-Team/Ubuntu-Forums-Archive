### Post by ibuclaw on 2009-02-22
Add/Remove Programs lists program names that you can install/use.

Synaptic lists the same as above, but also includes many other packages such as libraries and developer files. Things that the average non-programming/compiling user may not want to be affiliated with.

Add/Remove Programs also provides an easier interface to find/choosing applications.

Synaptic is a little more advanced, and also includes some extraneous features, such as pinning/locking packages in their current version (very useful feature if you don't want an upgrade to break the package/packages).
