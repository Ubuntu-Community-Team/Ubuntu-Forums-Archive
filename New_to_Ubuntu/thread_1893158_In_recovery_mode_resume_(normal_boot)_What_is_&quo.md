### Post by jwbrase on 2011-12-09
It didn't display your username, it displayed the name you gave your computer when you installed Ubuntu. I believe that Ubuntu defaults to using [username of first account]-pc as the machine name if you don't specify something else (as it sets up your first user account at the same point in the installation process as it sets up the machine name).
