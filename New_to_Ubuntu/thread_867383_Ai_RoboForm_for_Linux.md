### Post by Rolcol on 2008-07-22
I don't know if this is what you're looking for but using ubuntu's package search brings up this: [http://packages.ubuntu.com/hardy/mypasswordsafe](http://packages.ubuntu.com/hardy/mypasswordsafe)

You should be able to install it by using this command:
```

sudo apt-get install mypasswordsafe
```
