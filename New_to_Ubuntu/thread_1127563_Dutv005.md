### Post by Elfy on 2009-04-16
This seems to be a download link [http://linuxtv.org/hg/~anttip/ce6230/archive/a93e31c75a28.tar.gz](http://linuxtv.org/hg/~anttip/ce6230/archive/a93e31c75a28.tar.gz) for it 

Once it is downloaded you can extract it

```
tar -xzvf ~/Desktop/ce6230-a93e31c75a28.tar.gz
```

You likely need to install build-essential

```
sudo apt-get install build-essential
```

Then change to the directory and make 

```
cd Desktop/ce6230-a93e31c75a28
make all
```

The install docs appear to not need configure. Further options are detailed in the install text file.
