### Post by sanderd17 on 2011-12-28
Since you say you are completely new, I will start with the beginnings.

First of all, oblivion is a Windows program, and Ubuntu is not Windows. So there is no reason why it should run on Ubuntu (just as there is no reason why it should run on your smartphone).

Now, some devs did create a compatibily layer called "wine". What you have to do is installing wine from the software center, and afterwards you can open the installer exe with wine. According to the wine DB, everything should work.

[http://appdb.winehq.org/objectManager.php?sClass=version&iId=7506](http://appdb.winehq.org/objectManager.php?sClass=version&iId=7506)
