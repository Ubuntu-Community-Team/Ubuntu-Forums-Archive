### Post by rsgangr on 2014-01-21
For a few years I used XSane but experienced some problems occasionally. Sometimes it would produce a partial image but the rest would be grey. About 18 months ago I changed to Simple Scan and it works really well. It allows you to save a scanned image as a *.pdf, *.jpg or *.png file. You can install it via the Software Center or via Synaptic.
