### Post by tacantara on 2009-10-10
I can't speculate on the possibility of a trojan/malware, although it is very uncommon (almost nonexistent) for that to occur in Linux.

As for the firewall question, Ubuntu has a firewall that is active by default when you boot your computer.  There are GUI based apps (i.e. Firestarter) available through the package manager that allow the user to customize the firewall settings.
