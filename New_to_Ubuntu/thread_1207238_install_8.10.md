### Post by anystupidname on 2009-07-08
If you mean, what is the hotkey to enter the BIOS, on Dells it is usually F2 or F1. Older ones may have been something else like CTRL-ALT-C or ESC. Check the boot sequence setting. It may not even support booting from CD in which case you'll need to use a floppy with a boot manager such as PloP
[http://www.plop.at/](http://www.plop.at/) which can boot the CD for you...

### Post by QIII on 2009-07-08
You may have to start and stop it several times to find out which key, but start with esc.

When you start the thing, just start tapping on the esc key until you get to the BIOS setup.

If you don't get there, try f1, f2, etc.

My wife has a brand new Gateway, and I usually solve similar problems on her machine with a three pound hammer...
