### Post by Troll_the_Great on 2008-07-15
Open nautilus as root.For that hit Alt+F2 and type 
```
gksu nautilus
```.
Navigate where you want the new folder and create it.Just be careful because as root you can do ANYTHING with your system.So don't mess it up :)
Cheers!
