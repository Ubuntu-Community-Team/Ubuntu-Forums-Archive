Ah. That's a wubi.exe installation. It's main advantage is that it's easy to uninstall if you find that you don't like Ubuntu (or Xubuntu in your case), but it has a few disadvantages, one being that it's a tad more delicate than a proper installation, and another that I've encountered is that it doesn't like being upgraded.
