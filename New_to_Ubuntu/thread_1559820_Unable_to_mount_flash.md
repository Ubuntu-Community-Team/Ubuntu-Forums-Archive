### Post by Temposs on 2010-08-24
This may happen because you put it into a windows machine and did not do the "Safely Remove" operation in Windows. You should plug it into a Windows machine again and "Safely Remove" it in the proper way, rather than just taking it out.

SanDisk actually has a way to remove the crappy U3 system from your flash drive, but you need to plug it into a Windows machine. If you can do this, then the flash drive will work better with Ubuntu: [http://kb.sandisk.com/app/answers/detail/a_id/2550](http://kb.sandisk.com/app/answers/detail/a_id/2550)
