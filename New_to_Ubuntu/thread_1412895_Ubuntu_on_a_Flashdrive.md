### Post by beckx020 on 2010-02-21
Partition the flash drive. I don't know if you have persistence on the drive or not.  But find out how much is used and make the rest a partition. If you need windows to read it too then format it fat32. If it's only for linux then format to ext2.

You can only do this in linux, so you will need to boot from another drive (or cd) and partition.

edit: I forgot that windows probably can't see that second partition, so you should be able to format it however you want.
