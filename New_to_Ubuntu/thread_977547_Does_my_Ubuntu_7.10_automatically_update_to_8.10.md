### Post by w4ett on 2008-11-10
Select System>Administration>Software Sources then select the Updates Tab and select Normal Updates...this will give you the distribution upgrade options in the Update Manager.

### Post by Soriano on 2008-11-10
To get ubuntu to update your going to have to make sure you tell the OS to update during every release.

Click System --> Administration --> Software Sources --> Updates 
At the bottome of the screen you will see 'Release Upgrade.'  

If you want it to upgrade to ALL new releases select Normal releases other wise if you only want LTS releases select Long Term Support releases only.

Click Close and now go to:

Click System --> Administration --> Update Manager

You should see a button on the top of the screen asking if you want to update to 8.10.

### Post by Paqman on 2008-11-10
To go from 7.10 to 8.10 you'll actually have to update twice:

7.10 > 8.04 > 8.10

Might just be quicker and easier to reinstall, especially if you have a separate /home partition.
