### Post by =CrAzYG33K= on 2010-07-04
Thread subscription. 
Just configure it so that it emails you all the activity on threads that you're subscribed to.

**_To subscribe to a thread:_**
From forum post, Thread Tools (one of the buttons at the top) > Subscribe to this thread.

**_Settings:_**
User CP -> Settings and Options -> Edit Options -> Messaging and Notification -> Default Thread Subscription Mode -> <Your choice of email notification>

Also, you can list the subscribed threads from User CP -> Subscribed Threads -> List Subscriptions
