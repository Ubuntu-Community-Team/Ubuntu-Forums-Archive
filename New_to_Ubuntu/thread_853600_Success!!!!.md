### Post by Rocket2DMn on 2008-07-08
build-essential is a metapackage that depends on other packages. See here - [http://packages.ubuntu.com/hardy/build-essential](http://packages.ubuntu.com/hardy/build-essential) - all those packages with a red icon are what it depends on, so you should grab those packages, too.

i386 refers to the Intel 386 aka 80386 (x86) line of processors which are 32 bit processors.  For more info see [http://en.wikipedia.org/wiki/I386](http://en.wikipedia.org/wiki/I386)
AMD64 (x86-64) is for 64 bit processors that contain all instructions for the i386 line as well as some new ones for the 64 bit architecture.  See [http://en.wikipedia.org/wiki/Amd64](http://en.wikipedia.org/wiki/Amd64)
