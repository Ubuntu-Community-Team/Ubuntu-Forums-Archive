### Post by Paqman on 2009-12-20
> **NoaHall said:**
> Ubuntu, don't use Wubi - it relies on your Windows install, so is slower. Go for the 64 bit version, too.

The speed difference between a Wubi install and a regular install is so small that you won't really notice it.

The actual drawbacks of Wubi are:
[LIST=1]
[*]You cannot hibernate. Suspend is still ok.
[*]You may need to boot up into Windows and defrag the drive occasionally
[*]The system is not tolerant of power interruptions. Obviously that's not an issue on laptops because they have batteries.
[/LIST]

[More info about Wubi](http://wubi-installer.org/faq.php)

Don't listen to the naysayers, Wubi is awesome. I'd fully recommend using it if you want a fast, simple way to get Ubuntu installed.

[*]You cannot hibernate. Suspend is still ok.

[/LIST]

[LIST=4]
[*]If Windows crashes, you can't get into Ubuntu until you boot up and shut down Windows again successfully...
[*]...which completely locks you out of Ubuntu if Windows won't start up at all.
[*]You're limited to 30 gigabytes of space for Ubuntu.
[*]There's no easy way to remove Windows and keep Ubuntu.
[/LIST]

Install Ubuntu normally.

I'd say start with regular Ubuntu, and 64-bit if your hardware supports it. The difference between the *buntus is that they have a different GUI and, as such, different preinstalled programs.

Ubuntu uses the Gnome desktop, Kubuntu uses KDE, and Xubuntu uses XFCE. You can install KDE and XFCE on top of Ubuntu, and vice versa, so you might as well just install regular Ubuntu and experiment on top of that.
