### Post by PhantomTurtle on 2012-04-06
The only folder that is shared by default is the Ubuntu One folder. If you want to sync other folder right click on the folder, move mouse over Ubuntu One --> Sync this folder. If there is nothing in the Ubuntu One shared folder, then nothing is synced.
