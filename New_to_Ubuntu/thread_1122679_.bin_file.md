### Post by pbpersson on 2009-04-11
Find and read the installation instructions on the web site where you got the files from.

The first step is to right-click on the file, go to properties, and set to executable.

However, the instructions will tell you to run it from a terminal prompt, they will tell you if you need to run it as root, and they will tell you all the steps (if any) you need to perform during and after the install.

### Post by taurus on 2009-04-11
Just install sun java from the repos unless you want to install it from .bin.

Applications -> Accessories -> Terminal
```
sudo apt-get update
sudo apt-get install sun-java6-bin sun-java6-jre sun-java6-plugin
```
