### Post by ManamiVixen on 2013-03-08
Install xcompmgr. It gives you basic graphics acceleration like window shadows and simple transparencies on your desktop and is lightweight. Archwiki has a good man page for setup.
[https://wiki.archlinux.org/index.php/Xcompmgr](https://wiki.archlinux.org/index.php/Xcompmgr)

Just do sudo apt-get install xcompmgr and use the man to configure! :)
