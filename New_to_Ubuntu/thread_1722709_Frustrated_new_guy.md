### Post by I2k4 on 2011-04-06
I'm a humanities major, enjoying Ubuntu/Lubuntu after years on Windows.  I found the Live USB installation here very well illustrated, step by step, foolproof:

[http://www.ubuntu.com/desktop/get-ubuntu/download](http://www.ubuntu.com/desktop/get-ubuntu/download)

I suggest: 
1) starting with Ubuntu 10.10 Desktop version, which preinstalls a lot of software and has a fairly easy user interface for Windows converts, 
2) adding the "Persistence" step on the Pendrive installer AFTER Step 2 of the "Show Me How" process; I'd suggest using a 4GB USB drive and setting 2GB of Persistence (leaving 2GB for the operating system.)  Without "persistence" you don't save your preferences and settings for Ubuntu or your installed software on the USB drive, which is a nuisance and doesn't give you a test of the operating system.
3) don't bother with upgrades until you're done with Live USB, and only try to install a few packages at a time while using it - it's slower than a hard drive install, but then again, you're not risking a major muck up of your hard drive.
4) keep running Live USB until you find a version of Ubuntu or Linux that you want to consider installing to hard drive.  Running from USB is a great test of OS efficiency and economy and a great way to compare surprisingly different versions.  
5) You can set programs that work better accessing the hard drive to do so, in my case FireFox cache and DropBox folder.  Because the OS is on USB, there's a delay in mounting the hard drive, so it's best to disable "automatic" loading of software that needs to read the hard drive immediately.

After three months, I'm still in my own personal "trial" mode but can run fully functional Lubuntu on my netbook and my desktop without any risk of fouling up the drive where Windows is installed.

Best wishes.

### Post by Javelin Dan on 2011-04-06
rkrider - 
    Don't give up! Everyone here does try to help. But in my case, I've found too many assume a certain level of knowledge about Linux that I just didn't have to start. I struggled with the same thing early on so I'll give you the basic info in one place that it took me many hours to glean from several sources. 
     You probably understand (but maybe not) that when you download an iso  file, you can not do a simple "save to disk" as you would when saving a document. You have to do a "data burn" using a CD burning software. First, grab a blank CD-R disk - I'm told you can have problems if you try to use CD-RW. Simply download the iso file from a mirror close to you, open your burn program and click on data burn, write to disk, or anything else it suggests for iso or data files. Next, a window should open  asking you about burn speed. Set it on 1x or whatever your slowest burn speed is and hit the go button. When  done, you should have a bootable CD. Insert it in your CD drive and restart the computer. It will only boot on start-up. From that point, it should be pretty self explanatory as the set-up wizard will guide you to all the normal choices made during installation. Remember, if you just want to take it for a test drive, you will be given a choice to not install it before anything else. If you do that, it will give you the feel of the desktop, but will run much slower than it would after installation. Hope this helps.
