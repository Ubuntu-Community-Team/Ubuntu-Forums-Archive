### Post by etali on 2009-11-13
Before you go installing it, have you tried the option to "Try Ubuntu without making changes to my computer" (I think that's the wording).  This will run Ubuntu from the CD.  It will be much slower that way than it will actually run once it's installed, so don't judge the speed based on running it from the CD, just use that as a way to make sure everything works.

If you have the right screen resolution / internet access / sound when using the live CD to try Ubuntu, then follow the link another poster gave earlier in the thread so that you can install Ubuntu without getting rid of Windows.

I hope you enjoy Ubuntu!
