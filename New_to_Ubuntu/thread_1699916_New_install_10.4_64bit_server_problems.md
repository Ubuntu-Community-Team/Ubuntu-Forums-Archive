### Post by jerome1232 on 2011-03-04
Server version is for servers. Desktop is for desktops.

Server doesn't include a gui but it includes server specific packages and a few server specific tweaks. It also includes encryption and lvm support right out of the cd.

So after a server install it's normal to get a black terminal screen, this is intended behavior.


You want the desktop version which contains the full blown Gnome Desktop Environment.
