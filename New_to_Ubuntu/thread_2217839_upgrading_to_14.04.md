### Post by grahammechanical on 2014-04-18
Open Update Manager and check for updates and it should then provide a button to upgrade from 12.04 to 14.04. Yes, we can go direct from one LTS to another. If you give the OS time it may present an invitation to upgrade.

My advice? Before you upgrade de-activate any proprietary video driver  and revert the user interface to default condition as much as possible.
