### Post by jrrader on 2010-07-16
I also have that Ethernet controller and have had 2 problems with it.  I did the replacing of the R8169 driver with the R8169B thing and it didn't seem to help.  

The first problem was that the controller was not powering on with the rest of the system.  If the device is turned off improperly it will not start again when you turn your computer back on.  To fix this problem disconnect the power supply from the source for a few seconds.  When you start your computer again it will work properly.  I think I got these instructions from the realtek website.

The second problem I had (and this is what I suspect is going on with you) is the 2.6.32-23 kernel.  Install an older version of the kernel.  I'm using 2.6.32-22 but others will work.  Reboot and select the older kernel from the grub menu, then uninstall 32-23.  Update will try to install it again so keep an eye out for it.  You might need a second Ethernet card to do this.
