### Post by quixote on 2010-05-08
> Ubuntu wants to be added to a group called "burning"
Not sure what you mean.  Or did you mean K3b where you wrote "Ubuntu"?

When a program wants to be added to a group, you can select one of your existing ones.  Somewhere near the "burning" option there should be a dropdown allowing you to select a group you actually have.  eg Accessories.

If you want the new group, go to System > Preferences > Main Menu.  Click on "New Item" to add a program, or "New Menu" to add a new group of programs (or "packages" in linux-ese).
