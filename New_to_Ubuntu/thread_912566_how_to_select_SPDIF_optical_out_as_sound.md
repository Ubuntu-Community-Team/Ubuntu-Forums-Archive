Hi,

i found this process overly hidden so i thought id post how to set the audio in unbuntu for optical out.

if you have a supported chipset like my iCE958 go to your volume control [speaker in top right corner] and go to edit -> preferences. select the ice958 checkbox and click ok. go to the "switches" tab and check the new ICE958 checkbox. BE SURE YOUR VOLUME IS DOWN! :)
