### Post by Sarah L on 2008-06-19
Whether you can rewrite a CD depends on whether it's a CD-R (one-time write) or a CD-RW (multiple rewrites). Generally what most CD burners do is write the data from the inside of the disk out, and then fill in the rest of the disc with blanks.

This means that if you have a CD-R, then you can't write to it anymore. A CD-RW will allow you to write on top of previous data.

### Post by cackles on 2008-06-19
A multisession disk is fine for keeping a hold of to update/add to but because it isnt fully written not all drives or systems will recognise it.

You need to write once for an installation disk because the computer wont recognise it. It wont have all the necessary information on it. You wont be able to add anything else to it later.

### Post by cackles on 2008-06-19
You can as long as the machine its going into, if its a different machine from the one with your DVD creator on it, understands what a multisession disk is.

If your just going to add little bits to it later and its for you only on the machine you made it with then fine, because it obviously has software that knows what a multi session is.

Dont put a little on it though then expect it to be used for a full image later because that wont work, even if the data you use is only 50MB and the image is 300MB.
