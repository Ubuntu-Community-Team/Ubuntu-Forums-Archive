### Post by Crunchy the Headcrab on 2009-08-03
For new users I recommend Ubuntu, Opensuse, or Fedora 11.  Ubuntu is good because it has a large community and since it is based on Debian it has a lot of software in the repositories (the easiest way of installing software on any distribution).  Opensuse is good because it seems to be very complete.  It has a lot of software loaded by default (some may not like this) and it can basically take care of itself, so it is fairly easy to use.  I like Fedora because even though it might be a little more complex than ubuntu it is still pretty easy for me, a new linux convert.  For example you can choose almost exactly what you want to install or leave out using the basic install disc as opposed to being forced to install a bunch of crap you don't use.  It also has some "newer" features than some of the other Linux distros.

If you are looking for ways to improve the look of Ubuntu try gnome-look.org.   It has a number GTK themes that you can go to System > Preferences > Apperance > Theme > Install and just drag and drop them in to get a totally new look for your system.  Shiki Colors is one of my favorite themes.

If you would like to change what icons are on your desktop search for information about gconf-editor.  It is a tool that allows you to do certain things like that.

If you are interested in a dock (think Mac OS), you might want to check out avant-window-navigator or cairo-dock.  They are both pretty popular.

Basics:  Don't worry about antivirus.  You probably won't need it on Linux.  Most of the antivirus programs just scan for windows viruses anyway, which don't affect Linux systems.

Finally if you want a system that acts like Win7, you should probably just get Win7 when it is released.  Linux doesn't try to be anything other than Linux.
