### Post by sanderd17 on 2011-08-20
Ubuntu does not work too great with brand new hardware because the Linux kernel devs need some time to write the drivers.

So if your touchpad is really brand new, it will probably not work in the current Ubuntu release (11.04). If it's a touchpad configuration that they reused from other laptops, than it will probably work.

Anyway, if it doesn't work with 11.04, try again with 11.10 which should appear in October (you don't need to install anything if you try it from the live CD).

I'm using simple multitouch (two-finger scrolling) and I don't know why I should need other multitouch things, but I can.
