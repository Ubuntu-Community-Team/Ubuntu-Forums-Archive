### Post by haqking on 2012-04-03
Linux is no more secure overall than any other OS generally speaking, it is as easily compromised as any other OS.

However it is less prone to Virus infection, though other malware can infect it.

The best security program for it is user education and good habits.

[Noscript]("https://wiki.ubuntu.com/BasicSecurity#NoScript") for browser security (as most exploits these days take place in a browser and are largely platform agnostic.

[Apparmor]("https://wiki.ubuntu.com/BasicSecurity#AppArmor") to control program activity

[Firewall]("http://ubuntuforums.org/showthread.php?t=1876124") to control traffic (common argument is it is not needed due to no listening services by default however people tend to forget about outbound traffic

[AV software]("https://help.ubuntu.com/community/Linuxvirus") is not generally needed unless you are sharing files with windows, no current viruses in wild that effect and if there was one current AV software wouldnt detect it anyways, and there are greater concerns than Viruses in the malware category.

Read here for a Good grounding [Ubuntu Basic Security Wiki]("https://wiki.ubuntu.com/BasicSecurity")

For clarification:

Distro is short for distribution, Linux is a kernel (heart of, control centre, core command processor etc etc) around that Kernel various companies, individuals and such build a usable system for the end user, there are many and they are referred to as a distribution or distro, Ubuntu is a distro of Linux from a company called Canonical, Red Hat is from Red hat, etc etc.

Gnome is a DE or desktop environment of which there are many and also Windows managers.

See here [http://en.wikipedia.org/wiki/Comparison_of_X_Window_System_desktop_environments](http://en.wikipedia.org/wiki/Comparison_of_X_Window_System_desktop_environments)

Gnome is the default in most distros or at least alot of, however there are many which are preferred over others such as KDE or Xfce etc.  You can use and install more than one and login to that environment getting a different look and feel for your system.

Some apps work in or better in a given DE.
