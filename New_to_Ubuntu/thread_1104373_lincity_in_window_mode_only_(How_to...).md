### Post by loyaleagle on 2009-03-23
Sorry, but found the answer to my own question....

Here it is:

lincity-ng
Command line overrides configfiles.
Known arguments are:
-v        --version         show version and exit
-h        --help            show this text and exit
-g        --gl              use OpenGL
-s        --sdl             use SDL
-S        --size [size]     specify screensize (eg. 1024x768)
[COLOR="Red"]-w        --window          run in window[/COLOR]
-f        --fullscreen      run fullscreen
-m        --mute            mute audio
