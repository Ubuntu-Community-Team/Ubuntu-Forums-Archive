### Post by dileepdinesh on 2008-11-27
[SIZE="3"]You did not mention the laptop,,anyway this is how it works with my dell laptop,,switch on the bluetooth on both laptop and mobile,if ubuntu has detected ur bluetooth hardware,(it would!!) it will show a icon on the desktop,,,
right click it and select 'browse device option',,ur mobile's name for bluetooth connection ( how it identifies) will appear on the list (if some hex values appear,just wait,the name will come),,then select ur mobile,,then u should accept the connection in ur mobile and give a passkey ( any number,i use 0) ,then give the same passkey in the laptop,,then both ur mobile and laptop gets connected and u will  be able to browse through the folders of mobile.
         I dont think u can watch movie which is on a laptop using ur mobile via bluetooth,,becoz i dont think bluetooth supports that much speed,,i am not sure.[/SIZE]
