### Post by Golem XIV on 2008-06-14
This is an old Dolphin bug dating from Gutsy Beta, as you can see here:
[https://bugs.launchpad.net/ubuntu/+source/dolphin/+bug/136695](https://bugs.launchpad.net/ubuntu/+source/dolphin/+bug/136695)
Apparently it won't ever be fixed in the current incarnation, because KDE is moving to version 4.
Recommendation:  Use drag & drop with Shift (for move) or Ctrl (for Copy) pressed.
