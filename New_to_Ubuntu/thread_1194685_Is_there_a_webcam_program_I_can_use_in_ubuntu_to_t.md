### Post by Dr Belka on 2009-06-22
you could try empathy for video chat.  It is going to replace pidgin in 9.10.  ```
sudo aptitude install empathy
```

You can try either cheese or camorama to take pictures and videos with your webcam.  ```
sudo aptitude install cheese cheesetracker
``````
sudo aptitude install camorama
```

### Post by Orion8 on 2009-06-22
Cheese is a real simple program to take pictures with a web cam, but oddly enough I just realized the other day that VLC Media Player (also in the repos) does this too!  Open up VLC and select Media -> Capture.  Hit the "Capture Device" tab and change capture mode to "Video for Linux 2" or some such.

I do not know about video chat except that Skype allows you to do this.
