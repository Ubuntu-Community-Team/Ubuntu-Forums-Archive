### Post by partloer on 2010-07-03
Hi rakibhasan,

The first thing that you need to do is install the language to do this go to System->Administration->Language Support then check the languages that you would like to support.  You will have to download and install the languages once this is complete you can change to another language.  

A few ways to change between language is System->Preferences->Keyboard->Layouts tab then select the language you want.  Additionally you can add keyboard indicator to the panel to select between different language.

Let me know if this doesn't help.
