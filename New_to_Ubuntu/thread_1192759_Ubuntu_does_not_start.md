### Post by wpshooter on 2009-06-20
1) BEFORE you are installing Ubuntu are you checking the integrity of the Ubuntu CD by running the verification function that is found on the initial Ubuntu boot screen menu ?

2) Do you have your computer on a surge suppressor or preferably on a UPS/battery backup (I am sort of assuming this may be a desktop computer instead of a laptop) to keep to computer from loosing power in case the AC power is lost due to storm, etc. ?

3) If you are installing from the LIVE CD version of Ubuntu have you tried installing using the safe grahics mode parameter ?

4) And if the safe graphics mode parameter on the LIVE CD version does not give you satisfactory results, the trying installing from the ALTERNATE (text based) CD version of Ubuntu instead of the Live CD version.
