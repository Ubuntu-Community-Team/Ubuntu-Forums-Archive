You should see Ubuntu listed. The following is my listing for Windows 7. Ubuntu 9.10 and Windows 7 reside on the same HDD, with Windows 7 being the default boot.
 

### BEGIN /etc/grub.d/30_os-prober ###
menuentry "Windows 7 (loader) (on /dev/sda1)" {
	insmod ntfs
	set root=(hd0,1)
	search --no-floppy --fs-uuid --set 3258695b58691f3b
	chainloader +1
}
### END /etc/grub.d/30_os-prober ###
