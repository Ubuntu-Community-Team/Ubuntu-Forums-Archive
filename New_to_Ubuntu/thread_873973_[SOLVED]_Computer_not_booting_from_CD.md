### Post by kspncr on 2008-07-29
You need to change the boot order on your computer's BIOS and set the boot order so that it boots from CD before it boots from the hard drive.

To get to your BIOS, you need to hit a key while booting up. Unfortunately, that key differs from motherboard to motherboard. On some it's delete (like on mine), on some it's escape, on some it's f12, etc. If you don't know what it is, you'll just have to try a few.
