### Post by dave131 on 2014-09-21
I'm by no means knowledgeable in this, but if you go back to printers/add then select Network Printer and wait a few seconds while it populates the list, then select your printer and click forward - should be self-explanatory from there.  This will add a new printer as wireless, so if you want to use it wirelessly you should change your default printer to point to it.  Since that is an all-in-one printer I don't know if other things will work on or not.  My HP wireless all-in-one also shows in simple scan so I can scan things in as well.
