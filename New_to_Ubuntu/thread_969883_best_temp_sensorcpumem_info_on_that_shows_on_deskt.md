### Post by kansasnoob on 2008-11-03
I use "computertemp" for cpu temp reading.

If you install:

```
sudo apt-get install computertemp
```

It'll then be available to install to either panel. Just right click the panel and choose add to panel.

Mine:

[ATTACH]91056[/ATTACH]

As far as full system info you can go to System > Administartion > System Monitor. Or if you'd like you could install sysinfo. It's in the repos so you could either just go to synaptic or:

```
sudo apt-get install sysinfo
```

It'll then show up in System Tools.
