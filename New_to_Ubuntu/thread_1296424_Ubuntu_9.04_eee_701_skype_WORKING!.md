### Post by Piddell on 2009-10-20
I finally got skype working on my eeepc 701 4G with good quality and sound.

steps:
. deinstalled pulseaudio (basically everything with "pulse" in it's name via synaptics)
. installed alsamixer
. set via sound preferences to alsa playback device
. went into skype and set all audio devices to "default"

Thanks to clues gained from the easy peasy support site!

The only issue I have now is I have to use an external headset to get loud enough sound and microphone, which I'm sure is a config issue somewhere.
