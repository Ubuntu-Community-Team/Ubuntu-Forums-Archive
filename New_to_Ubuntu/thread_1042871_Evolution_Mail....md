> **Leo Dragonheart said:**
> I would like to set up the Evolution Mail thingy, but I don't know how I connect to a server or whatever the info I need to set it up. Is there a way I can find this out or a program that will do it automatically for me?

Hi Leo,

Contact your ISP and ask them for the mail server and port, it will probably be POP/POP3 but it may also be IMAP. Then you will setup new identity and set your incoming mail server as pop3.isp.com for example and your outgoing mail server as smtp.mail.com. If you want to retrieve web mail from Gmail, Hotmail or Yahoo then check the help docs to get the server info.
