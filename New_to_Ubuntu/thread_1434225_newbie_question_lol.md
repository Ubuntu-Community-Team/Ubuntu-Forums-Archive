### Post by Ozymandias_117 on 2010-03-20
Yes, there is. It's still compiz.

```
sudo apt-get install compizconfig-settings-manager
```

Or if you want one that's easier to configure

```
sudo apt-get install simple-ccsm
```

The first will be modified by going to System -> Preferences -> CompizConfig settings manager

The second will be System -> Preferences -> Simple CompizConfig Settings Manager

Make sure you right click on your desktop go to change desktop background, Visual Effects and select "Extra" to enable Compiz.
