### Post by Jawshie on 2009-11-03
I have a MySQL server set up on an Ubuntu machine. It works great. The problem I have is that there are not many clients in the Ubuntu repositories for MySQL clients. I *can* use the command line for most of what I need, but having a GUI to manage everything makes my life a lot easier. 

Back in Windows land (like where I work), we use Microsoft SQL server and SQL Server Management Studio for most of our database work. It is a well-done software that allows for (mostly) quick access to everything I need. The thing I absolutely love about it, however, is the software suite Red Gate provides. Specifically, SQL Prompt ([http://www.red-gate.com/products/SQL_Prompt/video.htm](http://www.red-gate.com/products/SQL_Prompt/video.htm)). It is AWESOME and provides Intellisense-like functionality to SQL Management Studio. Are there any Linux products that provide similar features?

### Post by spikyjt on 2009-11-18
I personally use phpMyAdmin, which is in the repos and very easy to install manually if you prefer that method for web apps. It does everything most people need, but you may have more demanding requirements than me. If you prefer a native GUI, there is gmysqlcc or kmysqladmin, depending on your desktop choice. I've used kmysqladmin and found it good, but found using phpMyAdmin to be quicker to set up and more powerful. Sun do some commercial products for MySQL, but I haven't used them.
