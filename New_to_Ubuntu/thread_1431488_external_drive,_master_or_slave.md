This totally depends on the chip-set that the enclosure/interface uses.
Read the instructions that comer with the enclosure.

Note #1... for simplicity's sake, most all chip-sets use CS (cable select).

Note #2... the USB interface isolates the master/slave factor from the rest of the system, and renders it un-used.

Note #3... if it's a SATA, there is no master/slave selection at all.
