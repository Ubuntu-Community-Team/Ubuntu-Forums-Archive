### Post by hellomoto on 2008-05-12
Ok, I have solved this myself (so proud).

For anyone else looking at this thread because your having the same problem please do the following.

To mount your memory stick on your palm device (this is not to sync information I used gnome -pilot to do that).

1- Turn on your Palm OS device and navigate to data import on the main menu.

2- Make sure on the Data Import Screen that MS (memory Stick) is selected.

3- Connect your device via the USB cable to your PC.

4- Select connect via the palm device.

5- My Memory Stick now auto mounted with Hardy and I could navigate through the file browser.

NB: I'm not sure how well my palm device will cope with music files being put on it via the file browser because many devices now are bundled with multimedia syncing software so it may not recognize it... will keep you informed.
