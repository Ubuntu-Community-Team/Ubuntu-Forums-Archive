### Post by stoogiebuncho on 2010-01-27
> **pink87 said:**
> I don't really understand the reason behind dual booting. I am trying to get away from Windows.

Dual booting is good for people who want to get away from windows, but can't live without a few apps that only work in Windows.

It's important to understand that Ubuntu is different than Windows, and that if a program is written for Windows, it will not work in Ubuntu.  You can get some functionality from some programs using wine, but it's pretty hit or miss, and I wouldn't say that tweaking wine is a user-friendly enterprise.

That said, Ubuntu offers some significant advantages to Windows, and you may find perfectly adequate replacements for the programs that you feel you can't live without.

One of the best things about dual-booting is it lets you learn Ubuntu slowly.  I truly believe that Ubuntu is user-friendly and easy to use, but keep in mind that you've been using Windows your entire life (I assume), so Ubuntu will feel strange and confusing for a while.  If you dual boot, you can play around in Ubuntu for as long as you want, but if you get frustrated, you can go back to Windows for a while.
