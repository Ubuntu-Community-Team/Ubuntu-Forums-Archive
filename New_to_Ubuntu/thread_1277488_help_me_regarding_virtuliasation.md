### Post by beak02 on 2009-09-28
I'm not an expert in this field but as a general rule - wine doesn't work!  Some select few programs work on it - and many people have written programs to make programs work on wine (such as "Play on Linux") but windows programs don't usually go on without a fight.

Personally I use Sun Virtualbox which is similar to VMWare I believe.  I run windows in a window (or 'virtualbox') and install programs on that.  The downside of this is that sometimes linux doesn't tell windows about all of the hardware available and getting files on to and off of your virtual machine can be tricky.

Hope this helps!
