### Post by _h_ on 2010-03-15
Right click the ubuntu icon next to applications -> Edit Menus -> Internet -> Add new item

Type: Application
Name: Google Chrome
Command: /opt/google/chrome/google-chrome %U
Comment: Access the Internet.

And make sure to checkmark it to enable it. That should do it for you.
