### Post by cwsnyder on 2009-07-29
When you start up the Live CD, you are given the option to include something called 'boot options.'  These are there to allow you to 'fix' problems like a video system which is not being detected properly.  You can set a frame buffer or set to a known good video mode which works with both your graphics card/subsystem and your monitor.  Look at the bottom of the screen for function keys which will supply more information on setting those boot options.
