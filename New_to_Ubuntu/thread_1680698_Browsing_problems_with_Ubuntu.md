### Post by Sealbhach on 2011-02-03
Are you using Firefox? Just to experiment, maybe you should try a different broswer like Opera or Google Chrome to see if it's just a problem with Firefox.

If you're using Firefox, make sure to install the [Adblock Plus]("https://addons.mozilla.org/en-US/firefox/addon/adblock-plus/") add-on. This will eliminate most of the ads you would normally see. There is also a version of Adblock for Google Chrome if you prefer to use that browser.

Secondly, you could try changing your DNS servers. A good one to use is OpenDNS. This may speed up your page fetching times a lot. There is info here on how to do that, it's really easy, you just open a file and add a line of text to it:

[https://store.opendns.com/setup/device/ubuntu/](https://store.opendns.com/setup/device/ubuntu/)

Another suggestion is to install[ Ailurus]("http://code.google.com/p/ailurus/"). Then close Firefox. Then open up Ailurus from Applications/System Tools, select Configure Firefox, then click "Auto Tweak Firefox" and click "Save". This will make Firefox run better for you.
