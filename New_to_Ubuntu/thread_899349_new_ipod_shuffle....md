### Post by Sinkingships7 on 2008-08-25
It seems that you tried to just drag the music files into the iPod's base directory. This does not work (unless you're using an alternative firmware). To get songs onto the iPod, you must use an application that provides the same feature as iTunes: organizing your songs into the hidden iPod_Control folder. This is all done behind the scenes without you knowing. Gtkpod will allow you to do this with minimal fuss. If you're looking for an all-in-one music manager, then try Amarok or Songbird. I prefer the latter.
