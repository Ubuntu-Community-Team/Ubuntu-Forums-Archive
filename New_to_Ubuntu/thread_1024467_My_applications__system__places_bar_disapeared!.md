### Post by Michael.Godawski on 2008-12-29
If your whole panel disappeared, not only some application on it, you can just create a new one.

Right click on any other existing panel, for instance the bottom one, and select add new panel. You can drag it wherever you want.

Now you can add to panel the sections you want. Right click on the panel, add to panel.

There is no need to panic. In Linux/Ubuntu everything is customizable, so be curious but also cautious when you experiment.
