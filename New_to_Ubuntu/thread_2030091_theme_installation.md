### Post by Frogs Hair on 2012-07-20
You can install this theme via PPA and it will update automatically. Open the terminal and run the following. ```
sudo add-apt-repository ppa:webupd8team/themes
``````
sudo apt-get update
``````
sudo apt-get install adwaita-cupertino-gtk-theme
```

Search the forum for manual installation instructions because the question has been addressed many times.

You may like the Zukitwo Color themes from the link and they are part of the same PPA , but not all themes on this page have been updated for 12.04 yet .
