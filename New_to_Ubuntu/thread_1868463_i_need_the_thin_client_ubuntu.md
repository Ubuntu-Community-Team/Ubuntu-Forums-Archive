### Post by mastablasta on 2011-10-24
i would say the best way would be to download the ISO image and put it on USB (maybe even pesristent) and then boot from the USB . this will give you some idea if everything works or not. Persistent LiveUSB will keep your settings on the key in a file.
 
To put it on USB you can use:
[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)
or this is a good one if you are on windows:
[http://www.linuxliveusb.com/](http://www.linuxliveusb.com/)
 
All *buntu flavours (Xubunut, Kubuntu, Lubuntu Ubuntu) work with mozilla. If it all works then you can make a permanent install to USB if you need it. more on live and permanent install here: [https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)
