### Post by acelin on 2008-06-21
Mint has more codecs, so playback of media is easier. If you stuck with Ubuntu, and learned how to do things like adding codecs and such, it would be a great way to get started with adding programs.

If you want to add these codecs, just go to add/remove programs and make sure all programs are selected, even non open source programs, and search for codecs, or maybe even automatix.
