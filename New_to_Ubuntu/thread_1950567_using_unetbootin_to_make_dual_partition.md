### Post by kdane4 on 2012-04-01
Hi,

You can use Unetbootin to create a flash drive installer.. And then you need to boot from it to install Ubuntu. You can install Ubuntu on your created D: drive by choosing 'something else' from the installer and selecting your partition.

Another easy alternative would be [INSTALLING VIA WUBI]("http://psychocats.net/ubuntu/wubi"). You just need to select your D: drive from the install menu, choose size, input your username etc, and install.
