### Post by lswest on 2008-12-21
That's using the magic sysrq key combo, it exists for Ubuntu too.  It should be available in any linux distribution.

Just check the /etc/systctl.conf file to make sure the sysrq keys are enabled.

I think it is enabled by default.  Check [this site]("http://www.ubuntugeek.com/how-to-use-magic-system-request-keys-in-ubuntu-linux.html") for more ifno.
