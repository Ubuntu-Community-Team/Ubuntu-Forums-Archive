### Post by mlentink on 2012-11-22
You ***will*** install Ubuntu 'outside' of Windows, you will only be using the Windows install/remove programs thingamajig (whatever they call it) to install it, and it will be installed in a 'virtual disk' which is effectively one huge Windows(NTFS)-file. You will also be using the Windows-bootloader to start. 

But you will ***not*** be running Ubuntu and Windows at the same time. At startup, you will have to choose one or the other, you can't run both at the same time.
