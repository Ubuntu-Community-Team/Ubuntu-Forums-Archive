### Post by starcraft.man on 2009-06-28
I'm pretty sure it's set from Edit > Preferences > Music Tab > Preferred Music > Drop down should include AAC .m4a.

Personally, I'd just rip to MP3 with1 192 or 256, more than enough quality for my ears and MP3 support is more constant.

If you don't see the option, you probably don't have the codec installed, install the ubuntu-restricted-extras package via terminal or synaptic. Open synaptic (System > Admin > Synaptic) search for "ubuntu-restricted-extras" then right click and mark for install. Then apply and should all work.
