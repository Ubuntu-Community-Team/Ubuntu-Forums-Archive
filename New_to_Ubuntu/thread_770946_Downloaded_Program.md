### Post by ace007 on 2008-04-27
You can go to system>>preferences>> main menu and control what appears under applications.  You can choose where to place a new item under any menu.  All you need to know is what command launches it from the command line.  Sometimes the name of the package you installed from the repositories is the command that will launch the applications.
