### Post by Mark Phelps on 2013-06-01
Where did you get the audio manager? IF it came from a CD that came with a sound card, it is likely the Windows version -- and you can't install or use that in Linux.

The website linked has two different sets of Realtek audio software -- you need to see which one you need (HD Autio or AC'97), go to that page -- and see if there are Linux downloads.  If there aren't, you're Out of Luck:  [http://www.realtek.com.tw/downloads/]("http://www.realtek.com.tw/downloads/")
