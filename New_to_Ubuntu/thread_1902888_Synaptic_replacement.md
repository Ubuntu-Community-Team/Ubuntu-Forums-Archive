### Post by Mark Phelps on 2012-01-01
You can't really "replace" Software Center with Synaptic.  

While they can BOTH be used to obtain and install software,  Software Center has features that Synaptic does not, and vice-versa for Synaptic.  

So basically, you should ADD Synaptic and use it, if you're more comfortable using it than Software Center.
