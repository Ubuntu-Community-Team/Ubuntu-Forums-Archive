### Post by Paqman on 2008-08-20
> **Zekes said:**
> 
1. Is Ubuntu compatible with the Microsoft Zune


Could be a problem, as the Zune uses a strange protocol. Search these forums, i'm sure the question has been asked many times.

> 
Worldwide Telescope, DigitalPersona fingerprint reader, and Finale?


Unsure. Again, try searching these forums and see what pops up. If it looks like gobbledegook to you, someone will be happy to walk you through it.

If there aren't Linux alternatives to these apps, your options are:

1) Dual- boot
2) Use the Wine compatibility layer (very hit-and-miss)
3) Run Windows in a virtual machine (very hardware intensive)

> 
2. Is it really worth it in my case? I currently run windows vista however I've stripped it of almost every last windows component and all the trash they bloat it up with and have installed the (better) open source programs (Firefox, Thunderbird, pidgin, open office, etc...) 


Sounds like your transition would be very easy, since you'd be using many of the same apps. I'd say you have nothing to lose, and a lot to gain. Compared to Vista, Ubuntu is faster, nicer looking, cheaper, more secure, has no DRM, and doesn't belong to some big company.

> 
3. Is the conversion relatively simple IE simpler than the dreaded (1 billion hour) windows install?


Yes, Windows installs are extremely laborious compared to Linux. If you opt to not do any partitioning when you install (by using Wubi) then the install process is extremely simple.
