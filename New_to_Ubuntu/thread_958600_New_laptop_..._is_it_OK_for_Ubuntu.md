### Post by jmszr on 2008-10-25
Why don't you try Wubi. It wont take much room on your hard drive (I'd use at least 10GB, but even that much isn't necessary-you could get by with 7GB. That way you can test drive Ubuntu for a while and see how it works for you. One can always migrate it to a full install later if desired (a fresh full install is simpler). It will be a tad slower-but not noticeably so. I"d go with Wubi 8.04.1 as it seems quite stable now and though I've heard good things about 8.10 the first few weeks after release are bound to be a little hectic.
