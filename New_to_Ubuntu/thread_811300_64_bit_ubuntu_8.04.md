That means it is 32 bit. It will run on 64 bit processors as well. 

All PC's use this and you will be able to use the 32 bit version on new 64 bit processors as well.

(You might want to get the newest version of Ubuntu, Ubuntu 8.04 LTS)

x86 (or something similiar) is 32 bit, AMD64, x86_64, x64 are all the same. They are 64 bit processors that can run the 32 bit versions of programs as well.
