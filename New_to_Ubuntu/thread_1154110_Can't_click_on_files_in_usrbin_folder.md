### Post by cariboo on 2009-05-09
You have to be root to copy any files to any directory other then your home directory. If you are trying to install a new theme, right-click on the desktop and choose Change Desktop Background, then click the Theme Tab, then drag your them to the window.

---

### Post by durand on 2009-05-09
For cairo dock, I think you can install themes from it's configuration program. Right click on the dock and click on Settings. The theme stuff should be there somewhere. Otherwise, extract it in ~/.config/cairo-dock/themes with the file manager.

~ is a shortcut for your home.
