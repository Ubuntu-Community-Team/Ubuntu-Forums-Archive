### Post by patrickballeux on 2008-11-10
Hi,  You are looking for a software like Netmeeting on Windows?

There is nothing quite like it but here is some suggestion:

1 - You can share your desktop (using VNC).  It's already available on the Gnome desktop using "Remote Destop" in Preferences and using Terminal Server Client in App/Internet.

2- I've seen a software called Coccinella that should enable a white board sharing.  Don't know if it's good... [http://www.getdeb.net/app/Coccinella]("http://www.getdeb.net/app/Coccinella")

3 - Use a broadcast site like Stickam or UStream and use a virtual webcam to display your desktop using WebcamStudio for GNU/Linux.  Bonus: You will have sound also... and If you have a real webcam, you will see each other. [http://webcamstudio.wiki.sourceforge.net/]("http://webcamstudio.wiki.sourceforge.net/")

Good luck!
