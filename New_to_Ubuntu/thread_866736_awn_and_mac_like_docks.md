### Post by billgoldberg on 2008-07-22
> **hedrian said:**
> er im having troble downloding cairo..and what is synaptic (sorry but im really new to these stuff)

Go to [this page]("http://developer.berlios.de/project/showfiles.php?group_id=8724"), and download the latest &#8220;dock&#8221; and &#8220;plugin&#8221; .deb files. Double click them to install. After installation, you can open Cairo Dock using &#8220;applications -> system tools&#8221;.

Synaptic is a package manager, that will download and install software for you.

Most of the time you don't need to search on the internet for software, synaptic will have it.

But in the case of cairo-dock, this isn't the case.
