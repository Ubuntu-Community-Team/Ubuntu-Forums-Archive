### Post by mcduck on 2011-10-14
You are missing the Flash plugin (which is also what Youtube uses). Install it from the Ubuntu Software Center.

You can get just the Flash package, or the ubuntu-restricted-extras which also contains other commonly used things that can't be included on the default install for licensing and patent reasons.

### Post by mcduck on 2011-10-14
Just open the Ubuntu Software Center, type "flash" into the search bar in the top right corner to find Adobe Flash, and click the Install button. Flash will be downloaded and installed automatically for you.

Same goes for anything else you install using Ubuntu's own package management tools, they will download and install things for you, and also handle updates for whatever you install through them. You don't need to manually download stuff from web pages and then click through installers to get the programs working, not to even mention tracking and updating all your programs by hand, Linux distributions have long time ago figured that there's a better way to do such things... :)
