### Post by stoogiebuncho on 2008-12-08
A lot of people seem to like Cinelerra.  I just installed it myself and haven't had time to play around with it much yet.  It's not in the repos, you'll have to install it from their website.

I have also tried Kino, which seems like a good program except it's meant to work with .dv files and if you're trying to use any other file type it is a hassle to convert them.  

Cinelerra has been able to work with every type of movie file I've thrown at it so far.  It seems like it has a lot of options for editing, as well.
