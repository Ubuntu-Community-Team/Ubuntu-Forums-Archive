### Post by alphacrucis2 on 2011-10-13
> **ave2 said:**
> I usually perform a clean install, but this time have decided to upgrade to 11.10. I would thus assume some of the software I have installed over time will be out of date. what is the procedure for installing the latest versions through the software centre? will it pick up that my software is out of date and give me the option to upgrade?


The upgrade will automatically pull in newer versions of anything you installed as long as newer versions exist in the Oneiric repos. If you use PPA's you will find that the release upgrade will disable them so you will need to re-enable any that have newer software you need after the release upgrade is completed.
