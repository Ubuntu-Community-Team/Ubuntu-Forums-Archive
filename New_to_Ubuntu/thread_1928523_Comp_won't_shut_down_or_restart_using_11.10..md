### Post by HavarN on 2012-02-20
Do you use Gnome3? If so, you could try holding down the ALT key to see if the shutdown option appears in the drop down menu.

If you are using the standard Ubuntu desktop Unity, and the problem is that when you select "Shut Down..." from the drop down menu, it goes to the log in screen; you can just shut down from the log in screen.

From my experience, Ubuntu seems to behave this way if you have more than one user session running when you try to shut down.
