### Post by sisco311 on 2008-07-01
I guess sda9 is your home partition.
Mount it in the /mnt directory to check:
```
sudo mount -t ext3 /dev/sda9 /mnt
```If sda9 is your home add this line to the /etc/fstab:
> UUID=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx /home ext3 relatime,defaults 0 2where xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx is the uuid you get from
```
sudo vol_id --uuid /dev/sda9
```backup your current /home and reboot
```
sudo mv /home /home2
sudo reboot
```If everything works you can delete /home2.

To mount the fat partition add this line to the fstab:
> UUID=xxxx-xxxxxxxx-xxxx /media/MOUNTPOINT vfat defaults,umask=0002,gid=46 0 0```
sudo vol_id --uuid /dev/sdb1
```to get the uuid of the partition.
create the mount point:
```
sudo mkdir /media/MOUNTPOINT
```EDIT: use```
gksu mousepad /etc/fstab
```to edit the fstab.
