### Post by Mark Phelps on 2015-08-20
> My Ubuntu question is that if I can use Ubuntu (which is working fine) to set up the keyboard drivers in Win 10
Sorry - No.  There is no way to use Ubuntu to install drivers for Windows.

Windows 10, unlike its predecessors, does NOT allow you to disable any windows updates, and has been known to replace current drivers with older drivers -- sometimes resulting in problems like yours.
