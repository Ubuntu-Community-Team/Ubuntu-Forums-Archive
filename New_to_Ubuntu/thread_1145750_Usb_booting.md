### Post by PhilMize on 2009-05-01
Well, most computers less the 5 years old do support usb boot, or atleast should. Most bios's if you hit f12 while it boots it will bring up the option to select your boot device. You may have to enable the option in the bios. If that still doesn't work check the manufacturer website see if there's a bios update available that will give you the option. Also you can create any usb boot device if you already have 8.04 or later on ubuntu installed. Just go to System>Administration>Create a usb startup disk. Just make sure you have the iso image you wish to set up on the usb drive like on your desktop or something so you can easily find it.
