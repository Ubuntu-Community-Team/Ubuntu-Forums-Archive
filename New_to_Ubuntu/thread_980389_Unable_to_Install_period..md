### Post by phidia on 2008-11-12
Try the safe graphic mode and see if you can load the desktop with that.
If the live cd doesn't load now I think it's better to figure out what's going on before attempting an install.

When the system "hangs" try pressing Alt+Ctrl+F1 (sometimes F2 thru F7 in place of F1)
at the command line that opens from that keystroke combination type "dmesg" maybe you can see something in that output that provides a clue as to why it's hanging/stalling?
