### Post by vocx on 2017-07-18
> **bradybunchfan said:**
> I am new to all of this Ubuntu thing, and am curious to know if there is a standalone download that I can run alongside Windows 7.

Thank you.
I would say, no.

However, you can download a virtual machine, such as the free VirtualBox. Then you download the DVD or USB image file of Ubuntu, it's an ".iso" file. The you install that image inside the virtual machine. This is emulating a separate computer within your Windows 7. Then you can boot the virtual machine with the installed Ubuntu and test it as much as you want there. As it is a virtual machine, you can mess with it, install things, erase it, and it will not affect the outside Windows 7, and stay confined to its virtual machine.

Virtual machines work pretty well for most tasks that don't require much computing power such as reading and writing documents, browser the web, writing code (text files), etc. If you really need computing power for editing videos, playing games, performing long mathematical calculations, then installing Ubuntu as its own standalone operating system would be best.

### Post by vocx on 2017-07-18
> **bradybunchfan said:**
> What does a virtual machine do?
I can see you are a bit confused so I'm just going to clarify.

Ubuntu is not a "program". Ubuntu is an operating system. It is similar in this regards to Windows. If somebody asks you, "What is Windows?", how would you answer?

An operating system is a base environment that installs in a computer, for example, a desktop computer, a laptop, a tablet, or even a mobile phone. Inside that operating system you install other programs such as music players, office suites, drawing programs, etc. An operating system is like a house were you place things inside, the programs that you use daily is like the furniture.

You can have a parcel of land (computer), and in that parcel you can build a Windows house or an Ubuntu house. Or you can build two smaller houses, one Windows, and one Ubuntu, side by side. Normally, you cannot build a house within a house, can you?

Well, that's where the virtual machine comes handy. It precisely allows you to build an Ubuntu house inside the existing Windows 7 house. You basically simulate that you have a completely blank computer, and in this virtual computer, you install an Ubuntu operating system. You can create as many virtual computers (virtual machines) as you want and test different operating systems, or in my analogy, you can build many virtual houses inside your house, without going outside and actually buying new properties.
