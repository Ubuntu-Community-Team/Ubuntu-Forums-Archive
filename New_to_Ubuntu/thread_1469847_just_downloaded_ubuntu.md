### Post by the.dark.lord on 2010-05-02
For internet : On the top bar, go to System -> Preferences -> Network Connections.

For drivers: Drivers should be automatically set up. To see if you've any restricted hardware, for which you can download drivers from the internet- go to System -> Administration -> Hardware Drivers.
