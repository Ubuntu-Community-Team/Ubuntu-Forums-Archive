### Post by Mark Phelps on 2010-08-30
We like to say that Linux is a free "alternative" to MS Windows, not a free "version" of MS Windows. This means that, in some cases, stuff you're used to doing in MS Windows has to be done differently, and in the case of specific apps, some features are just plain NOT going to be available.

For example, SOME, and I emphasize SOME, MS Windows apps will run in Linux with an interface layer known as Wine.  But, I check the WineHQ site, and there is no listing for ePrompter -- not a good sign.

Not trying to discourage you ... but ... the more you are tied to the intricate details of a particular MS Windows app or feature, the unhappier you are likely to be using Ubuntu -- or any other Linux distro.

While Ubuntu DOES have screen savers (actually two different apps for doing it), they are not MS Windows screen savers, and while they perform much the same functions, I seriously doubt you will be able to integrate into them the same features as you have right now with Win7 and ePrompter.

In response to your question, no, the MS Windows screen saver will not run in Ubuntu, or in any other Linux distro.

That said, I will defer to others who have delved much deeper into screen savers to respond to you with specifics ...

### Post by tom.swartz07 on 2010-08-30
Not to say that there aren't other methods of letting you know about email when your system is idle...

I use a program called checkGmail. It can be easily configured (as in, copy/paste two command sentences) to light up the Scroll Lock light on your keyboard. 

If youre like me (or half of the world) you *probably* dont use your scroll lock. This will let you take advantage of that.


If you do in fact use your scroll lock key, then it can be just as easily changed to something else (caps lock?)
