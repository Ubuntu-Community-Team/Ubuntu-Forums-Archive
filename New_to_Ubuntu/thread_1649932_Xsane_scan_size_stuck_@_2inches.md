### Post by panthouse on 2010-12-29
It took me a long time to find out it myself in the end. 

Select WINDOW
Show Standard Options

a screen with GEOMETRY opens up. With the four slider you can select which part of the scanner size will be scanned. Select topleft-x 0, topleft-y 0, and select the maximum for bottomroght-x (5104) and bottomright-y (6848 on my printer). This should restore the scan size to the full size. 

Looking left of the scan button now a size of about 1276*1712*24 should be shown when using A4 paper size scanner (about letter) with 150 dpi setting. 

Good luck!!!!
