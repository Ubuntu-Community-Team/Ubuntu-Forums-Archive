### Post by chaanakya_chiraag on 2009-11-26
Well, you first want to make sure you have actually uninstalled Wine :D  Try clicking on the "Configure Wine" in the Wine menu.  If it loads, you haven't actually uninstalled Wine :D  Type ```
sudo aptitude purge wine
``` into a terminal (Applications->Accessories->Terminal) to fully remove Wine.
