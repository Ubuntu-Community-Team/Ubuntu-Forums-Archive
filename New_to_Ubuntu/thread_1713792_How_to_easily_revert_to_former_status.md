### Post by cariboo on 2011-03-24
Backup any configuration files before you edit them, and document what you do, so that you can undo what you've done if things don't work out.

### Post by synapsys on 2011-03-24
You can make an image of your hard drive. This is basically an exact copy of it that you can restore at any time you wish. The programs I've used, clonezilla, acronis, and norton, all work great.
