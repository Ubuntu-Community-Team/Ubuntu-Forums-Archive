### Post by kurt18947 on 2012-10-19
For printing I've had good luck with Brother's installer found here:

[http://welcome.solutions.brother.com/bsc/public_s/id/linux/en/faq_prn.html#f00104](http://welcome.solutions.brother.com/bsc/public_s/id/linux/en/faq_prn.html#f00104)

If you have difficulty downloading the script I've saved it here.  There are no nasties that I know of:

[http://ubuntuforums.org/showthread.php?t=1999216&highlight=brother](http://ubuntuforums.org/showthread.php?t=1999216&highlight=brother)   post #4

There are a number of wireless connection types available.  Personally, I've had the best luck with appsocket/HP JetDirect.  I assign a static i.p. address to the machine because the print driver 'talks' to a specific i.p. address if using appsocket.  Same with wireless scanning.
