### Post by Rhubarb on 2009-11-03
It's usually best to try the easiest approach first:-

For Ubuntu 9.04:
System --> Administration --> USB Startup Disk Creator

This allows you to write Karmic iso images to USB drives (so long as you have the karmic iso image of course).

If that doesn't work, let us know, if it does work, also let us know :)
