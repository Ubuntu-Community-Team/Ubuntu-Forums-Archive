### Post by coffeecat on 2010-03-19
Since it's a USB device, you would need to run...

```
lsusb
```

... not lspci.

Post the output. If it doesn't give the chipset (lsusb usually doesn't) googling the device ID (of the form xxxx:yyyy) usually provides useful information.
