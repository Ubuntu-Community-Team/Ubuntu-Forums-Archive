### Post by 73ckn797 on 2009-11-14
Did you enable software in System/Administration/Software Sources/Ubuntu Software and the /Other Software tab in Sources?

What Server are you trying to get software from within Ubuntu? That choice is in the Ubuntu Software part of the Sources menu.

See here:
[https://help.ubuntu.com/community/Applications?action=show&redirect=Software](https://help.ubuntu.com/community/Applications?action=show&redirect=Software)
