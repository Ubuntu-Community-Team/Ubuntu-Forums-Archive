### Post by NightwishFan on 2010-12-29
What is your final goal? To try Ubuntu? No, Ubuntu is not a virus, yes it is Linux.

If you just want to try Ubuntu on a flash drive, use this software to install it to one. You need the version for what OS you are using (Windows?). Then simply start the program and tell it to install the Ubuntu 10.10 Live CD to your flash drive.
[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

IF you want to install and use only Ubuntu on a computer, you have to download and install a Ubuntu live cd.

Download: [http://www.ubuntu.com/desktop/get-ubuntu/download](http://www.ubuntu.com/desktop/get-ubuntu/download)

How To Burn: [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)
