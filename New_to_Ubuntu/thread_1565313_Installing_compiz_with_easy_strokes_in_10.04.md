### Post by hhh on 2010-08-31
I would say that this is NOT for Absolute Beginners. You'll need a video card with 3D acceleration enabled (No 2 at this link)...
[http://www.omgubuntu.co.uk/2010/04/10-things-to-do-after-installing-ubuntu.html](http://www.omgubuntu.co.uk/2010/04/10-things-to-do-after-installing-ubuntu.html)

You'll probably need some extra Compiz plugins (this documentation looks wrong to me, you'd need to install compiz-fusion-plugins-extra in Ubuntu as well)...
[https://help.ubuntu.com/community/CompositeManager/CompizFusion](https://help.ubuntu.com/community/CompositeManager/CompizFusion)

And, of course, easystroke...
[http://sourceforge.net/apps/trac/easystroke/](http://sourceforge.net/apps/trac/easystroke/)

Then you'll need to configure it all to work seamlessly together, and I can't help you there since I've never tried easystroke...
[http://sourceforge.net/apps/trac/easystroke/wiki/Documentation](http://sourceforge.net/apps/trac/easystroke/wiki/Documentation)
[http://lifehacker.com/5049268/easystroke-makes-mouse-gestures-easy-in-linux](http://lifehacker.com/5049268/easystroke-makes-mouse-gestures-easy-in-linux)

Hope that helps.
