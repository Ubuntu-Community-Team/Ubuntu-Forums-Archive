### Post by driebel on 2010-03-12
I'm trying to configure my e-mail client to work with a new e-mail account.  To do so, I need to set the "IMAP server directory" to a specific folder (/home/driebel/mailtree).  How do I do this using Evolution?  I can not find a convenient setting for a server directory.

Yes, I know Thunderbird does this easily.  However, the fact that Thunderbird doesn't hide the deleted messages in my inbox is a fatal flaw.  It makes my account nearly unusable.  No, I do not want to move my message to a separate trash folder.  This feature alone makes Evolution a far superior program for my needs.

Anyone know where this setting is?  I'm running Hardy.

Dave

EDIT:  Got it.  To accomplish this, look under "Receiving Options" in the account settings.  Under "Folders" check both "Show only subscribed folders" and "Override server-supplied folder namespace."  Then enter the necessary folder in the Namespace: field.
