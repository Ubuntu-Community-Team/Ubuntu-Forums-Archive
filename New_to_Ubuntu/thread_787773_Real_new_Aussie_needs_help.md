sudo apt-get update

sudo apt-get install gnomebaker

Doesn't feel out of date! Works fine and I installed Gutsy last month in April. I don't think it was out of date then.

Download GnomeBaker if you get a chance. I've had better luck with it than with K9Copy and K3b.

Also, are these data DVDs? Or are they Movies that you're trying to copy?

You'll need something like DVDFab HD Decrypter to first rip the movie and then use Shrink to get it to a size to fit the typical blank DVD. I use this method and it works great. They both work fine in Wine.

I've also been using HandBrake to rip movies to fit on a CD. Really works well.

Make sure your system is up to date. That should take care of any program problems you might be having with Wine, etc.
