### Post by dawynn on 2009-04-29
Anecdotal experience:
I have a 1.4 Ghz Athlon classic with 1G of memory.  I've been running Ubuntu for some time -- since somewhere around the Breezy / Hoary era.

I've been using Linux Mint (XFCE version) and Crunchbang (#!) lately, although mostly LM (the lack of automatic menu updates in #! is my biggest hang-up there).

I was pleasantly surprised when I recently moved from Cable Internet (at a supposed 5Gb speed) to DSL (at a supposed 3Gb speed), that streaming seems to be going *smoother* now.  At this point, I don't know what to attribute it to, because logically, this just can't happen.

But I can now stream TV shows in full screen mode (had to do it windowed, if I wanted smooth transfer under cable).  I can listen to Pandora and still do other web searches and other activities -- again something I couldn't do with the cable modem.

Is it faster than the Windows ME system I can also boot into on my home PC?  Well, to be honest, if its any slower, I don't notice it.  But over and above everything, I can depend on Linux.  If I tell it to shutdown, I know it will.  Windows sometimes won't.  When Ubuntu shows me the desktop, I can use it.  There isn't that extra delay factor until Windows finally lets go and lets me work.  And if my Ubuntu system starts getting sluggish for some reason, at worst I just have to kill my session, and I'm back to work in moments.  Ubuntu lets me do full system installs and keep working without annoying messages every two minutes telling me I need to restart my PC.

I don't notice a whole lot of perfomance gain between Win ME and Linux, even with XFCE -- my PC is still an aging PC.  But my Linux system is very stable -- something I cannot say for my Windows experience.

[FONT=Arial]Are my expectations accurate in that I can expect better performance of my processor and memory in Ubuntu?[/FONT]


Overall performance is about on a par with Windows XP, which is impressive because Ubuntu is a lot more capable than XP.

Linux is also more modular than Windows. It's very easy to optimise performance by stripping out unneeded services, or even switching to a completely different desktop environment without reinstalling.

Your machine is fairly modest. Ubuntu is also not the lightest distro out there. If performance is not good enough with normal Ubuntu you could try switching to a lighter desktop environment such as XFCE or Openbox. If that's still not light enough you could try one of the lightweight distros like SliTaz.

> **dawynn said:**
> 
I was pleasantly surprised when I recently moved from Cable Internet (at a supposed 5Gb speed) to DSL (at a supposed 3Gb speed), that streaming seems to be going *smoother* now.

I assume you mean "meg" rather than "Gb".  Also, cable netowrking is like ethernet, meaning that the capacity is the capacity of the segment or node you are one and you share that with other users in the same node.  Typically you have more collissions or other sharing issues with those on your street.  DSL is more akin to a leased line and is not a shared bandwidth technology.  A 3 meg DSL connection shoud seem about the same speed as a 15 meg cable connection.
