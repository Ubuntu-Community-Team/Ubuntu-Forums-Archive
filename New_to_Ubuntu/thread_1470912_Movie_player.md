### Post by 3rdalbum on 2010-05-03
WMV is a heavily restricted codec, so no it is not meant to be playable out-of-the-box.

I'd advise you enable the Medibuntu repository (there are instructions at [www.medibuntu.org](www.medibuntu.org)), after which you will have access to a package in Synaptic Package Manager called "non-free-codecs". Installing this should give you the ability to play WMVs.
