### Post by Xero Xenith on 2009-03-29
Absolutely! Most popular Linux music programs integrate gtkpod's libs for iPod support, so iPod support is well orchestrated.

Ubuntu comes with Rhythmbox, which you can use to sync music (but doesn't always work for album art). Or you could take it a step further and install Banshee, which can sync music, videos and album art, or you could use gtkpod itself, which can sync everything iTunes can. However, gtkpod is notoriously difficult to use, so I'd advise sticking to Rhythmbox or Banshee.

The exception is iPod Touches and iPhones - support for those is few and far between. All other iPods work fine.

Enjoy :)
