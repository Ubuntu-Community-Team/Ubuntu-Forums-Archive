### Post by Vakman on 2009-07-11
Not sure if this has been posted before but I found a nice little application called App Runner. Very nice for new users who are unsure of how to run .sh or .run files for example or even for users who just don't want to every time. The link is [http://hacktolive.org/wiki/Runner](http://hacktolive.org/wiki/Runner) and the download is also there. It will let you run as root or user. Just right click and run script and it will run it automatically. Hope this helps :)
