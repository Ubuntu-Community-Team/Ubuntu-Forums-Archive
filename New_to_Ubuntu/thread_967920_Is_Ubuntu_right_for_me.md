### Post by diablo75 on 2008-11-02
1.  Ubuntu can play movies very easily (provided you install the video and audio decoders, and that is very easy... see my signature for more details).  Also, it is very easy to VNC into your own machine.  You can enable remote desktop via the system menu and place a password on your machine so that only you can log into it remotely.

2.  Installing Ubuntu is very easy.  It's actually easier to install on a computer than Windows, in my opinion.

3.  Odds are very good that all of your hardware will work right out of the box.  If your video card needs proprietary drivers, you can use the built in Hardware Drivers manager to enable them (it's as easy as checking off a box and restarting your PC).

If you're looking for a HTPC-setup, I'd suggest sticking with Hardy (supported for 3 years and very stable).  I've set up a similar HTPC (but with newer hardware: [http://www.buildyourown.org.uk/forums/topic.asp?TOPIC_ID=38069](http://www.buildyourown.org.uk/forums/topic.asp?TOPIC_ID=38069))

After installation, try VLC rather than the default Totem for video, and consider using the command "vlc -f" as default for video files (plays fullscreen).

### Post by snowpine on 2008-11-03
> **nimsu said:**
> already went with ubuntu.. any particular reason you would recommend xubuntu or fluxbuntu?

Ubuntu has the highest system requirements of the *buntu family.
Xubuntu is designed to be a little bit friendlier for older hardware. You can try it by opening a terminal and typing:

```
sudo aptitude install xubuntu-desktop
```

Log out, and from the login screen, you can choose Sessions then Xfce to try Xubuntu (or Gnome to get back to Ubuntu).

If you don't like Xubuntu, simply

```
sudo aptitude remove xubuntu-desktop
```

I would not recommend Fluxbuntu unless you are a real do-it-yourself type, because it is based on 7.10 and hasn't had any updates or bug fixes in over a year.
