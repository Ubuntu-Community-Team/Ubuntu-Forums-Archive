### Post by nerdtron on 2015-01-25
All flavors of the official Ubuntu family (from Canonical) like Ubuntu, Kubuntu and Xubuntu don't come with the restricted extras codecs. I think it's because the restricted extras has licensing issues (not sure, but i'm sure that these extras are not open source) so Ubuntu flavors can't include them by default.

Linux Mint (which is derived from ubuntu and software compatible) is not based from US and not governed by the licensing issues so they can include the codecs and other programs like VLC on the default installer.

I'm not sure of any other Ubuntu based distro that uses Unity by default, You can try the Ubuntu Ultimate Edition but it's a heavy download I guess.

Goodluck on your distro-hopping!
