### Post by p3tris on 2008-05-21
Hi i have a fresh install of XUBUNTU on a DELL INspiron 9300. I installed today: Tracker Search Tool. 
It returned an error "error reading data" when i plugged in my usb external hard disk (who used to mount automatically). 
I removed Tracker Search Tool. 
From that point on the disk does not mount automatically. I can see it in partition manager but cannot access it! And every time i restart i get this message: Failes to initialize HAL.

Can anyone help me?

EDIT:

Fixed :guitar:

Everything just fine :)
