### Post by athaki on 2008-11-12
you probably shouldn't install ubuntu onto a loaned computer. You can still use the live-cd but the performance will not be at its best. From my experience, I didn't need any windows boot disk to access windows; grub found my windows partition and was able to mount it perfectly.
