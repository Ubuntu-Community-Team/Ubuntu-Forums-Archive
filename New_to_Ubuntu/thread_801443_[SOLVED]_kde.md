### Post by Paqman on 2008-05-20
If you want to try out KDE, install the package kubuntu-desktop. Log out > Options > Select Session > KDE.

Packages designed for KDE will be slightly more responsive if you're using KDE, but Gnome aps might launch a bit more slowly.  Otherwise they're completely interchangeable. It really is just a matter of personal preference.

Psychocats has more info on trying [different desktop environments](http://www.psychocats.net/ubuntu/), including how to fully uninstall them. Not that uninstalling the meta packages (eg: kubuntu-desktop) often won't uninstall all the packages automatically.

### Post by SunnyRabbiera on 2008-05-21
KDE like others said is another desktop environment.
A desktop environment is the general graphical user interface of your operating system, a visual way to use your computer without typing in command line all the time like in the DOS days.
Now KDE is one of the fest DE's for linux, however when it first came out there was a fee to QT, the library of what KDE was built on.
As a result Gnome came about, as did another library called GTK.
Nowadays the fee of using QT has been dropped except for commercial use, and other desktop environments have come about as well.
But thats how it is on linux, its all about choice as opposed to Windows or OSX where you are locked into the user interface.
