### Post by bngo on 2009-12-04
Hi there,

Yes, for some reason PCMan X BBS does get installed with the language support for Chinese. To remove it (along with the language support, unfortunately) go to the Synaptic Package Manager in System> Administration>Synaptic Package Manager and right click on the the pcman package among the installed list and select it for removal.
