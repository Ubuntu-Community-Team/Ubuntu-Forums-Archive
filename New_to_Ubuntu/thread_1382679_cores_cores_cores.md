Ubuntu will work with all those (meaning any x86-compatible 32- and 64-bit processors with any number of cores and multi-CPU setups), and also with some stuff less common in PC computers like PowerPC architecture used in Playstation 3 consoles and older Macs.

Linux kernel itself of course runs on pretty much anything from super computer to a parking meter. :D

### Post by Paqman on 2010-01-16
> **DarinB said:**
> 
I know that 32bit will not show any benefit on a core 2 duo. which is what i have now...but the confusing part is when i go to system monitor it shows both cores have activity? therefore a 32bit system does use both cores?


Yes, 32-bit systems use multiple cores. Using multiple cores requires something called SMP, which all versions of the Linux kernel have had for years.

All the Core 2 Duos are 64-bit processors, so you can run the 64-bit version of Ubuntu. You could also run the 32-bit version if you wanted, but there's no point in doing that IMO.

> 
**My point is...is it a waste of money to get an expensive processor that will yield no improvement in performance?**

A better processor will always give better performance. The real question is: what level of performance do you require, and how big is your budget?
