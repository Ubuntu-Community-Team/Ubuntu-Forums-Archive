### Post by PaulW2U on 2012-04-26
When daily builds resume they will be for the next release, 12.10.

As 12.04 is an LTS several point releases will be released as per [https://wiki.ubuntu.com/PrecisePangolin/ReleaseSchedule](https://wiki.ubuntu.com/PrecisePangolin/ReleaseSchedule). The point releases will of course include all updates up to the time of the release so will minimise the downloading of updates well into the PP's life cycle.

Yes, you should be able to zsync between 12.04 and 12.04.1 etc.

### Post by grahammechanical on 2012-04-26
There is another place to look for daily builds. It is here:

[http://iso.qa.ubuntu.com/]("http://iso.qa.ubuntu.com/")

These are the ISO images that are used for QA testing. Make a note of the code number of the ISO and see if it changes.

The code number is actually the date of the image. So, the last image for Ubuntu Desktop amd64 is 20120425 (2012/04/25), whereas the code number for Ubuntu Desktop i386 is 20120423 (2012/04/23).
