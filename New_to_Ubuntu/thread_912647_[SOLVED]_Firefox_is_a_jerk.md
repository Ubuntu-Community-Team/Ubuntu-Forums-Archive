### Post by lazyart on 2008-09-06
Install the FF plugin [FlashBlock]("https://addons.mozilla.org/en-US/firefox/addon/433"). It will leave placeholders for Flash elements but won't run them unless you click on them.

As for opening the previous page-- if FF had a problem with a page and it crashed, what good would it be to reopen the same page (and presumably crash again?)
