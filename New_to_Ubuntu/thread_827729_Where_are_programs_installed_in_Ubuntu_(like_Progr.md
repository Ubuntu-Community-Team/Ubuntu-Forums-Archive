### Post by agillator on 2011-11-01
Looking for the complete answer, check the links. The short answer - wherever they or the user wants them to be. A file in any directory can be set to be executable. There are some standards as mentioned and as laid out in the links, but you can put a 'program' anywhere you have access to. Having said that, there is a whole lot more to it than just where you put the program, but that is the short answer to a seemingly simple question.
