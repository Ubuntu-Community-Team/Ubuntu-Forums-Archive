### Post by mcduck on 2009-08-17
This is related to people using Microsoft's Outlook, which defaults to sending mail attachments in it's own proprietary system instead of sing normal attachments.

The best solution is to simply ask for the sender to configure their mail client to use normal attachments, although there are some programs available that allow opening these attachments.

edit: [http://en.wikipedia.org/wiki/Transport_Neutral_Encapsulation_Format]("http://en.wikipedia.org/wiki/Transport_Neutral_Encapsulation_Format")
