### Post by Mark Phelps on 2012-03-28
There will be NO offerings under Additional Drivers because AMD dropped Linux driver support for that card YEARS ago ...

As to the drivers, System Info is unreliable -- wish they would simply get rid of it!

If you're seeing a desktop, not a text-based Command Line Interface, then you already HAVE the default Radeon drivers installed. There are no drivers you can download from AMD that will work with your card and any Ubuntu versions newer than 8.10.
