### Post by aesis05401 on 2009-03-13
As for the multimedia stuff, the instructions for adding the media repo are here: [https://help.ubuntu.com/community/Medibuntu]("https://help.ubuntu.com/community/Medibuntu")

As for not having net connection - there is a command from terminal called 'apt-cdrom' that you use to add a cdrom drive as a repository.  You can then use apt-get/update whatever to update or install from the cd-rom... so yeah, download at school, find instructions for making 'APTonCD', then take home and run apt-cdrom.. etc. 

Try 'man apt-cdrom' or google it for more info.

### Post by aesis05401 on 2009-03-13
Ubuntu is fine without the internet.  The poster suggesting Mint was trying to make your life easier.

If you want to run Ubuntu you can install everything you would normally get from a repository by burning the packages to a cd and using apt-cdrom to mount the cdrom as a local repository.  

The biggest issue with not having internet is that you are going to need to lookup instructions while you are performing these tasks.  Try this link for instructions on creating Cds and DVDs with all the packages you want:
[http://aptoncd.sourceforge.net/]("http://aptoncd.sourceforge.net/").;  I have not used this product, but it is one way for you to get started at home without an internet connection - provided you are able to download the packages and make your CD/DVD copies at school for transport to your home machine.
