### Post by kanikilu on 2009-03-23
Don't extract or "unzip" the ISO file, that *is* the image you use to create the CD. I haven't used the software you're using, but if you can't get it to work, something like [ImgBurn](http://www.imgburn.com) is a useful Windows utility and should get the job done.

### Post by ChildOfMana on 2009-03-23
unbuntu-8.10-desktop-i386-1.iso is the file you need to burn to disk - _without_ opening or extracting it first.

You also need to burn it as an image, not as a data disk. I've never used Free Easy Burner so can't direct you exactly but look for an option along the lines of *'burn image to disk'* or *'burn file to disk as image'* or something similar.

### Post by mlentink on 2009-03-23
Download your free copy of Infrarecorder. Then choose burn image to disk, anmd be sure to select as slow a speed as possible.

Again: the iso file already ***is*** the image
