### Post by zombifier25 on 2012-10-09
Make sure all firefox processes is killed by running this in the command line:
```
killall firefox
```
If that doesn't work, log out and log in.

### Post by Peripheral Visionary on 2012-10-09
You can MIDDLE-CLICK on the desktop and it will bring up a menu of minimized applications (in case you don't have a panel for minimized apps).  When you get that "already running" message, middle-click on the desktop and see if it appears in the list. Then left-click on Firefox if it's on the list.
