### Post by davidmohammed on 2011-02-18
have a look in the folder /var/lib/dpkg

there are various files ending with -old

try copying status-old over status

i.e.

cd /var/lib/dpkg

sudo cp status-old status

### Post by ianc1 on 2011-02-18
uniqueone-85 look in the top left of the screen after the Ubuntu logo you will see the Applications tab were all the programs are, then Places were are the folders etc are and then System where you change settings, install software etc.

Under Places choose Home that's where all your folders are.  The system ones are in File System accessed on the left pane when you've selected Home.

/var/lib/dpkg is a system file and is in the file system bit.

The code entered in the above post needs to be put in the terminal which can be found under Applicatons>Accessories.
