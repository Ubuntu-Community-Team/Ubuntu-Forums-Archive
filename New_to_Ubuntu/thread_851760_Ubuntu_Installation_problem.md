### Post by Rocket2DMn on 2008-07-07
Sounds like the LiveCD just doesn't like your video card (it has limited support), so you should install using the Alternate Install CD.  You can get it from [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download) by checking the box at the bottom for the alternate cd.  It uses a text based installer rather than a live session, but it is still intuitive and easy to use.
Once you boot the computer normally, it should run OK.
