### Post by zombifier25 on 2013-04-24
In simple terms, a bitmap image (like .jpg) is merely a collection of colored pixels. A vector image (like .dxf) is much more complex; it contains points, curves, lines, circles... which comes together to make an image. Converting a vector image into a bitmapped image is simple, but the other way around is much more difficult, since it would require the bitmapped image to be traced down to its very basics, which are points, lines, curves, etc. etc. so like what Lars said, it may not work well, especially if the image is complex.
