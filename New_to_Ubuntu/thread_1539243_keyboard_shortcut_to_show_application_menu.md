### Post by Zorgoth on 2010-07-26
The keyboard shortcut you are looking for is Alt-F1.  It will pop up the Applications menu.

Alt-F2 will let you enter the name of an application to run

But your applications menu should not disappear when using firefox unless you are running it fullscreen or something.
