### Post by Felix-the-Cat on 2009-03-14
There is a plugin for pidgin (i am currently using it) that works with the festival package for text-to-speech. As far as Speech-to-text you would have to use Dragon Naturally Speaking with Wine or find an old version of IBM's ViaVoice and get it working. You could run a windows machine underneath and VNC into ubuntu and thereby have speech rec. but that is a less than optimal solution in my opinion. I'm current trying to get ViaVoice working but have run into issues with my mic. but as soon as that is resolved I can get back to it. (I have it installed but I haven't set it up). I'm really interested to see what kind of success you have so let me know what you decide to do and how it goes.
