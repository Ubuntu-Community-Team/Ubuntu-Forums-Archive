### Post by El Tito on 2013-03-24
The quicker way is @arpanaut's. Also if you press alt+f2, and type in the box 'term', it should display the matches being one of them the terminal.
I use KDE, but if I'm not mistaken if you click in the ubuntu symbol (the dash) the first in the panel, a new box should appear that do the same function as alt+f2.
Also there is a tab to search for all your installed apps (I think above the search box is a menu of tabs).
