### Post by Tatty on 2008-05-05
Go to System->Administration->Software Sources
Then check the repos on the first tab.


That should do it :)

### Post by macaholic on 2008-05-05
Go into System > Administration > Synaptic Package Manager, select settings, and repositories. Here you will be able to check enabled repositories, and set a variety of other things.
