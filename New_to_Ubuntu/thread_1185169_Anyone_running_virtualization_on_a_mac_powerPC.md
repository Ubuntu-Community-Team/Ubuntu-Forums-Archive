### Post by &#1082;&#1086;&#1084;&#1084;&#1091;&#1085;&#1080;&#1089;&#1090;&#1080;&#1095;&#1077;&#1089;&#1082; on 2009-06-28
The only emulation possible on the PowerPC is either PearPC (which is not developed anymore) and QEMU or another hypervisor. Virtualbox can only virtualize on x86 and x86-64 computers. Qemu can emulate an Intel processor and PearPC can emulate a PowerPC processor. Another virtualizer is Mac4Lin which allows you to run Mac OS X on your PowerPC Mac.
