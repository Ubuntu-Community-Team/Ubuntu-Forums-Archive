### Post by ugm6hr on 2009-02-01
You can install a 32- or 64-bit OS on that.

64-bit programs will not run on a 32-bit OS.

32-bit programs can be made to run on a 64-bit OS (with a bit of work).

If you anticipate needing any programs that are not in the repositories, I would suggest using 32-bit, since 64-bit builds for non-standard programs are difficult to get.

The age-old argument re: Flash on 64-bit is much less of an issue now, although I still get intermittent Firefox crashes with some Flash pages on 64-bit.
