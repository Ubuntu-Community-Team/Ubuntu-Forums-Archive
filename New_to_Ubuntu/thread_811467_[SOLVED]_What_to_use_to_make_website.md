### Post by tigerplug on 2008-05-29
> **vlad1975 said:**
> Hi guys. i wanna attempt to create my own website. whats is the best easy to use tool that linux can provide?

If you looking for a "What you see is what you get" (WYSIWYG) editor then I recommend nVu for Ubuntu. 
[http://www.nvu.com/](http://www.nvu.com/)

```
sudo apt-get install nvu
```

This is not always the best way. Its better to understand your code. It really depends on what sort of site you want to build, funtionality needed and practicality.

Above all you will need HTML.
Here are some HTML learning resources. Its pretty easy to get the hang of and very worthwhile:

[http://www.w3schools.com/](http://www.w3schools.com/)
[http://www.webmonkey.com](http://www.webmonkey.com)
[http://www.html.net/](http://www.html.net/)

For styling it is recommended to use CSS (Cascading Style sheets) to conform with web standards -it does require an extra bit of effort to learn but it can really help speed things up and make them easier in the long run. You can also learn CSS at W3schools.


For dynamic/database sites you could use PHP/mySQL.

[http://www.w3schools.com/php/default.asp](http://www.w3schools.com/php/default.asp)
[http://www.mysql.com](http://www.mysql.com)
[http://dev.mysql.com/doc/refman/5.0/en/tutorial.html](http://dev.mysql.com/doc/refman/5.0/en/tutorial.html)
[http://www.php-mysql-tutorial.com/](http://www.php-mysql-tutorial.com/)


Another option is Ruby on rails which is basically the Ruby programming language using a web framework called rails. Its not too hard to learn at all and its  good place to start.

[http://www.rubyonrails.org/](http://www.rubyonrails.org/)
[http://www.onlamp.com/pub/a/onlamp/2005/01/20/rails.html](http://www.onlamp.com/pub/a/onlamp/2005/01/20/rails.html)
[http://www.digitalmediaminute.com/article/1816/top-ruby-on-rails-tutorials](http://www.digitalmediaminute.com/article/1816/top-ruby-on-rails-tutorials)

Hope these links can help you develop your understanding a bit more.


Cheers, 


Tigerplug :)
