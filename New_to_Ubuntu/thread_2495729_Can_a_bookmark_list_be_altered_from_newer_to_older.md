### Post by him610 on 2024-02-28
@Holger_Gehrke Thanks for the tip.

> Here's how to sort bookmarks in Google Chrome:
Open Chrome
Select the Side panel down arrow in the top right corner
Select Bookmarks
Select Organize
Choose a sort organization
Select Sort by newest, Sort by oldest, Sort by last opened, Sort by A to Z, or Sort by Z to A
