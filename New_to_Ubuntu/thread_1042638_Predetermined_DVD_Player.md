### Post by Cypher on 2009-01-17
If you are running Ubuntu with Gnome, then go to Places->Home Folder. Once in Nautilus, go to Edit->Preferences->Media. Here you will see options for CD Audio, DVD Video, and so on. If you have VLC installed, then you should be able to choose that from the drop-down box for DVD Video and upon DVD insertion, VLC should be automatically launched for you..
