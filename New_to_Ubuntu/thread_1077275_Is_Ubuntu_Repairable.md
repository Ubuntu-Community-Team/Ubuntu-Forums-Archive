### Post by stalkingwolf on 2009-02-22
You can try running the recovery mode.
when grub first starts hit ESC.  this will bring up the full boot menu.
highlight (click) recovery mode.  hit enter.

when that finishes a box will open giving you several choices,
select Fix Broken packages.  hit enter.
when that finishes select continue normal boot and hit enter.

Hope that works for you.
