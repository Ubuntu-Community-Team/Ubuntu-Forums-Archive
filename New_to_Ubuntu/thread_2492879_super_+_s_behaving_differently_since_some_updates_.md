### Post by Dennis N on 2023-11-26
All the shortcuts that users can change are found in Settings > Keyboard > Keyboard Shortcuts > View and customize shortcuts.
By default, to search for installed programs on the Overview uses 'Super' alone. This is a fixed part of Gnome and can't be changed.
Super+S is defined by default to open the Quick Settings menu, as you noticed.
There is an action named 'Search' under 'Launchers' section which is undefined by default. You can define a keyboard shortcut there.
