### Post by benerivo on 2009-03-11
So he formatted the partition his pics were on, and then he installed windows on it? If he's on windows he can try the ['recuva' program]("http://filehippo.com/download_recuva/") which can find file information that has been deleted, but hasn't been overwritten by another file. This will only work if he did a 'quick ntfs format', and if the files weren't physically on part of the disk that windows was then installed on to.
