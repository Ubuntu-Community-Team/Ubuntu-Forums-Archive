### Post by mcduck on 2009-03-13
Could be either one, your current Firefox theme or the GTK theme you are using.

As Firefox doesn't really use GTK but instead XUL with some hacks to make it look like native GTK it does some things a bit differently. This often creates small problems with colors, for example almost every dark GTK theme has problems with Firefox.
