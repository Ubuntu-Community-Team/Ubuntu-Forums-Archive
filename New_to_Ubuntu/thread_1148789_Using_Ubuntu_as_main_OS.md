### Post by skymera on 2009-05-04
Hello :D

Yes you can have uUbuntu as your main OS.
Burn the Ubuntu ISO to a CD and put it into your computer and reboot.

Enter the boot menu (This can be anything from F12, F8 etc) Yours may be different.

Choose "Install Ubuntu"
And follow the install procedure. 
I suggest using EXT4 filesystem as it is much faster than EXT3 :) It's not deafult so you will need to chose "Manual" when you get to the patrtition stage. gl.

### Post by mister_pink on 2009-05-04
If you've set up for a dual boot you can expand the ubuntu partition over vista. Boot the liveCD, then go to the partition editor (system, admin menu). Then delete the vista partition and expand the ubuntu one. Shouldn't take too long. Make sure you back up important data.

You may experience problems booting or get errors after you reboot. If you do, go back to the liveCD and post back in this thread for help. Hopefully you shouldn't have any difficulties.
