### Post by Azdour on 2011-11-02
Hi,

I'm using Ubuntu 10.04. When I view properties and see something like:

/media/New Volume/Downloads...

I use my mouse to select all the visible text (including the ...), then use copy and paste into something like gedit. I find this copies the entire path.

Maybe this will work for you? 

Edit: I've just found that if you are using the search in Nautilus you can add the Location column: View | Visible Columns
