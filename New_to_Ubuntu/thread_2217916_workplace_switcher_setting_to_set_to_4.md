### Post by deadflowr on 2014-04-18
For the first one, open system settings appearance( or simply open appearance)
go to behavior and check the Enable Workspaces.

The second action of setting it to move workspaces when hitting the right side seems like edge flipping.
You probably need to reset that with ccsm.
Should be in Desktop Wall.
