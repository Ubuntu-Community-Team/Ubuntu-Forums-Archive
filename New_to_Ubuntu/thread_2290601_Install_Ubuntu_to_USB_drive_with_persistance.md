### Post by tea for one on 2015-08-13
If you use the utility Start Up Disk Creator which is included in a standard Ubuntu iso, there is a feature in the lower part of the GUI which allows you to create a persistent storage area.

### Post by yancek on 2015-08-13
The link below has a very detailed explanation of how to do this.  Additionally, you could have used software like unetbootin to create a bootable flash drive with persistence.

[https://help.ubuntu.com/community/LiveCD/Persistence](https://help.ubuntu.com/community/LiveCD/Persistence)

How big is the flash drive?  If it is large enough, you could do a normal install to the flash drive.

### Post by HermanAB on 2015-08-14
Just do a 'normal' install to the USB disk and boot from it.  It doesn't matter whether it is SATA, USB or a tiny little SD card.  It must just have enough storage space.
