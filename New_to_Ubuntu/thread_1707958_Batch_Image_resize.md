Both are available in Ubuntu's software repositories.

### Post by Steeperton on 2011-10-28
```
sudo apt-get install nautilus-image-converter
```

Then all you have to do is right click on an image (or multiple), and select resize. Chose your options, then go. Much easier than anything else I've found.
