### Post by Paqman on 2010-06-23
You'll need a copy of Itunes if you want to manage your Iphones properly. There's a couple of ways you could do this:

[LIST=1]
[*]Set up an extra PC with Windows, as you mention
[*]Set up your current machine dual-boot. It's completely possible to have both Windows and Ubuntu on the same machine and pick which you want at boot time.
[*]*MY RECOMMENDATION*: Install [Virtualbox]("http://www.virtualbox.org/") and then install your copy of Windows (running Itunes) into it. This is a virtual Windows PC running as an application on Ubuntu.
[/LIST]

Check out "seamless mode" for Virtualbox. You should be able to get it so that Itunes looks like it's running as a native app on Ubuntu. Virtual machines chew through a lot of system resources, but if your machine can hack it, go for it.
