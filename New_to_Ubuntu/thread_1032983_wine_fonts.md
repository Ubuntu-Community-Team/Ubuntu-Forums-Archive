### Post by Darkdelusions on 2009-01-06
You will need the Microsoft core fonts.

```
sudo apt-get install msttcorefonts
```

or 

```
sudo apt-get ubuntu-restricted-extras
``` (this will install some other things like java flash dvd playback)



Type the following command into a terminal window and that should get you fixed up
