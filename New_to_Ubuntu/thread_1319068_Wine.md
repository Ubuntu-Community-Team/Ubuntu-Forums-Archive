### Post by Paqman on 2009-11-08
On Ubuntu, the best and easiest way to install new stuff is to use the package manager. Go to Applications > Ubuntu Software Centre and search for "wine". 

You can also instal things from System > Admin > Synaptic Package Manager if you want a more in-depth interface.
