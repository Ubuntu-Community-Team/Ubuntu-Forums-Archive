### Post by pedja_portugalac on 2008-12-20
I have an Acer Aspire 8920 laptop and my sound work good, OK integrated dolby home theater 5.1 speakers work only on 2 stereo speakers but it's better then nothing. Headphones work very good. By the way, I think in the next Ubuntu 9.04 new alsa driver will solve this issue. The only thing you have to do after fresh install and update of Ubuntu 8.10 is to put two extra lines inside config file.

This is what I have done:

Install fresh Ubuntu 8.10
Update system and reboot
(for computers with NVIDIA GC)Install proprietary graphics driver for NVIDIA cards (177) and reboot
Open terminal and type:

```
sudo gedit /etc/modprobe.d/alsa-base
```

Add this to the end:

```
# added by myself
alias snd-card-0 snd-hda-intel
options snd-hda-intel model=auto
```

Reboot and it then works.

**For the Crystal Eye integrated webcam**

Support is already built in the latest kernels. All I had to do is to install Chesse (program) and after that edit it's resolution preferences into 160 x 120 and it works. Ok, the image is bad quality but it's usable. I have tried to set up gstreamer-properties without success. If anybody knows something else welcome, we are waiting.
