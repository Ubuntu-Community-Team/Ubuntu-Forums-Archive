### Post by Cheesemill on 2009-11-27
It's all down to personal taste, I'm a Ubuntu guy myself. The main difference is in the DE (Desktop Environment), Ubuntu uses Gnome whereas Kubuntu uses KDE.

If you have a good internet connection then I would recommend downloading both the Ubuntu and Kubuntu desktop CD's.  These can both work in Live mode, meaning you can try them both out without actually installing them (performance will be slower though) and make your own choice about which GUI you prefer.

Also you can install KDE apps in Ubuntu and Gnome apps in Kubuntu so it's all down to which DE you prefer working with.

You also have the option of installing Ubuntu and then KDE as well, you will get a choice on the login screen of which environment you wish to boot into, all of your documents and settings will be shared between the two.
