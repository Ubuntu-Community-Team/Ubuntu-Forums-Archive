### Post by pawnrocket on 2009-05-18
The Linux File-system

In the beginning there was root (/) 
The symbol "/" denotes the root directory, like the root of a tree. In this case it is just one root. There can be only one. 

In the / Directory you will see other directories like 

/bin
/tmp
/sbin
/var
/home

and such. 

The /bin directory is filled binaries (programs) for the user

some of them are 
cd - change directory
ls - list directory
mkfsiso - a program for dealing with iso's

The /sbin is filed with binaries for the system (sbin = system binaries))

The /media directory is where external media (cdroms, usb drives, usb flash drives) is mounted (mounted = connected to the system)

The /tmp Directory is where temporary files are placed

The /var directory as far as I have noticed is where things like web servers are help /var/www and /var/ftp

Everything in Linux is a file
Harddrives are files - directories are files - external usb drives are files. 

And the last one for this list (I don't understand it all either) is the /home

The /home directory is where YOU live. Its your home, Its like Documents and Settings in windows. The desktop, personal documents, music folders, movie folders and such are all stored there. Desktop settings all of the stuff that makes your world on your system

I am still a n00b - this is what I have learned. 

If you don't like the theme I recommend going to gnome-look.org and checking out what they have there. Also nautilus is much more powerful then explorer (scripts and such) I recommend that you give it a couple months, after all how long have you been using windows? Give Ubuntu a tenth of that time you will feel more at ease.

### Post by issih on 2009-05-18
One more just to hammer it home :)

You will HAVE to spend some time to learn it. Unless it was a windows clone in every respect that is simply inevitable, and it isn't windows, it is linux.

Have a poke around here:

[https://help.ubuntu.com/9.04/index.html](https://help.ubuntu.com/9.04/index.html)

Most of all though, just play with it until you get the idea.

The file system is a bit confusing at first perhaps, but think of it as just being the top of the tree, assuming you have only one hard drive it will just be the contents of that drive (like c: in windows). The top of the tree is referred to as simply "/".

Inside that the only bits you'll care about under normal usage is your home folder which is like my documents, which is at 
"/home/yourusername" where the /s represent dropping down into the next directory.

In your home folder you have all your documents/pictures music etc, and also a desktop folder that represents the contents of your desktop.

Thats about all you need to know really, and other than that you just need to grasp that nautilus works more like finder in osx than explorer, the sidebar is basically a bunch of quick links rather than a file tree.

You just need to get used to it.

As for theming, first try is to just open up the appearance preferences and drag the theme files onto the dialog, but the themeing is a bit clunky and you would be well advised to just chose one of the installed defaults you can stand until you have found your feet.

Hope that helps
