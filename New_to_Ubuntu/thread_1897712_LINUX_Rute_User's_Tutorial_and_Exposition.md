### Post by seawolf167 on 2011-12-19
I don't know of that book, but for the basics and more (note that it was written for 8.04/8.10 so things like Grub2 are not covered; instead Grub legacy is covered) you can use the [Ubuntu Pocket Guide]("http://ubuntupocketguide.com/index_main.html").

Additionally, you can read up on man pages

```
man commandnamehere
```

[LinuxCommands]("http://linuxcommand.org/index.php"), or simply start looking through the [Sticky Post]("http://ubuntuforums.org/showthread.php?t=801404") :P
