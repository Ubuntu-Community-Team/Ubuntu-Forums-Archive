### Post by Bucky Ball on 2012-02-12
Welcome. Using the terminal to install HPlip is very easy and, as far as I know, the only way to do. All instructions are there in the documentation you will download with HPlip and probably in the links in the previous post. 

There is a learning curve involved with switching to Linux (as with switching or starting with any OS) and although most things can be done without the terminal (as with Windows and Mac OSs which both have a terminal available) I suggest you become familiar with it none the less so it 'holds no fear!' Linux is not Windows ... Enjoy the ride and the learning curve! There is a ton of help around when you need it. ;)
