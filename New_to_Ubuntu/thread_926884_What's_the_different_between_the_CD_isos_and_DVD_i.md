### Post by Therion on 2008-09-22
The CD contains only one version of Ubuntu (Ubuntu, Kubuntu, Xubuntu, Edubuntu) and is either the Desktop CD, the Alternate CD, or the Server CD. 

The DVD contains the entire Main and Restricted repositories, which means it has Ubuntu, Kubuntu, Xubuntu, and Edubuntu; allowing you to install any or all four versions without needing an internet connection.
