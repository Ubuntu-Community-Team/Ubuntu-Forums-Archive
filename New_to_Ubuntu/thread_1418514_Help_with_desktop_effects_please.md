Have you installed the settings managers?

$ sudo apt-get install simple-ccsm

The simple one is OK to start with. The complete one lets you fine-tune more.

### Post by r-senior on 2010-02-28
You can change it to four using either of the settings managers but the quickest way is to right-click on the panel component in the bottom right of your screen (unless you've moved or removed it), select "Preferences" and set the columns to four.

### Post by themusicalduck on 2010-02-28
You need to increase the number of desktop spaces you have.

Right click on the desktop switcher on the right of the bottom panel and click Preferences. Change the columns number to 4.
