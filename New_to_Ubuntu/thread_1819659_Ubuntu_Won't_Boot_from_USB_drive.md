### Post by ~!geek!~ on 2011-08-06
The instructions for creating usb thumb drive bootable are quite clear on the [www.ubuntu.com](www.ubuntu.com) So if you followed them, you should not have any problem with the bootable usb drive. I hope you pressed F11 to check the boot-order and select  usb thumb drive to boot from the list of media to boot from. If you are still not able to boot from usb drive, you may try to format the usb drive and repeat the iso burning process on the usb drive  and retry booting. On Dell machines it works like a charm, generally.
