### Post by Karlchen on 2011-11-22
Hello, cap10Ibraim.

As your report is not really overly detailled  it is hard to tell what you did exactly and why you received the error  message which you received.

Apart from this, on Ubuntu the  recommended way of getting and installing software is by getting them  from the official Ubuntu repositories.

Ubuntu LTS should be  Ubuntu 10.04 I suppose, it comes with Firefox pre-installed. So there  should be no need to install it a second time. The Firefox launcher  should be available in the Applications => Internet submenu and as an  icon in the menu bar at the top of the screen.

Thunderbird may not be installed. Yet, you can use the "Software Centre" in order to download and install it.

Canonical  may not offer the latest versions of Firefox and Thunderbird for Ubuntu  10.04. If you wish to use the current versions, Firefox 8 and Thunderbird  8, the most convenient way of getting and installing them is by using  the Ubuntuzilla repositories.

Find the instruction on how to add  the Ubuntu repositories to your list of repositories and on how to get  and install the latest Mozilla builds of Firefox and Thunderbird here,  please:  [**[B]Firefox / Thunderbird Installation**[/B]]("http://sourceforge.net/apps/mediawiki/ubuntuzilla/index.php?title=Main_Page#Installation").
