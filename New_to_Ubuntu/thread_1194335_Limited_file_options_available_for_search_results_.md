Select a file or folder in a users home directory, and then choose Edit in Nautilus' menu bar. The drop-down menu should display a range of options available for for acting on, or examining the properties of, the selected file or folder.
 

 Now commence a file search by selecting Places > File Search..., and then entering one or more key words in the  Name contains: text box. Right-click on a file listed in the Search results window, and what do you see? I see four options: Open, Open Folder, Move to Trash, and Save Results As.  
 

 The question is, why aren't more file options available, like Cut, Copy, Rename, Properties, etc., for the listed search results in Nautilus?
