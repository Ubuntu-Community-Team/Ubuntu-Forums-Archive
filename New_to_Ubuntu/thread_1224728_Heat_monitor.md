### Post by fooman on 2009-07-27
you can install the lm-sensors package and after installation run sudo sensors-detect.

when it is done,  you can add a panel applet which will monitor your cpu temps and more...

see here for instructions:

[http://www.connectedinternet.co.uk/2009/05/24/monitor-cpu-temperatures-in-ubuntu/](http://www.connectedinternet.co.uk/2009/05/24/monitor-cpu-temperatures-in-ubuntu/)

hope that helps.
