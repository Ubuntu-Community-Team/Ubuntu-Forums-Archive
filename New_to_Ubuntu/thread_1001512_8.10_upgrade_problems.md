### Post by Michael.Godawski on 2008-12-04
Try to download an alternate CD and update this way.
> 
**Upgrading Using the Alternate CD/DVD**

 Use this method if the system being upgraded is not connected to the Internet. 

[LIST=1]
[*]Download the **alternate** installation CD 
[*][Burn the ISO to a CD]("https://help.ubuntu.com/community/BurningIsoHowto#InUbuntu") and insert it into the CD-ROM drive of the computer to be upgraded. 

[LIST]
[*]If the ISO file is on the computer to be upgraded, you could avoid wasting a CD by [mounting the ISO as a drive]("https://help.ubuntu.com/community/ManageDiscImages#MountISOFiles") with a command like: 
```
sudo mount -o loop ~/Desktop/ubuntu-8.10-alternate-i386.iso /media/cdrom0
```
[/LIST]
[*]A dialog will be displayed offering you the opportunity to upgrade using that CD. 
[*]Follow the on-screen instructions. 
[/LIST]
If the upgrade dialog is not displayed for any reason, you may also run the following command using Alt+F2: 
```
gksu sh /cdrom/cdromupgrade
```Or in Kubuntu run the following command using Alt+F2:```

kdesudo sh /cdrom/cdromupgrade
```

### Post by mapes12 on 2008-12-04
> As far as installing it as Michael.Godawski posted, with the alternate cd, is everything on my /home drive going to be wiped? Would I need to back it up on a external hard drive or something?Yep, make sure you have either a) /home on a separate partition [http://www.psychocats.net/ubuntu/separatehome](http://www.psychocats.net/ubuntu/separatehome)  If you do this then when you do the new install you will get to a section of the install called "Partitioning". The default is " Guided-Use entire disk". You don't want this. Select "Manual". I think it's the last option. Then for each partition select the variables to tell the installer where you want it to install "/" /home and swap. Your /home data will then be preserved.

b) copy your /home directory to external media. Do the new install (without partitioning) then copy back your backup of /home to the new /home directory created during the install.
