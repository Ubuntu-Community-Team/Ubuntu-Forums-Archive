### Post by guiverc on 2019-09-13
Kubuntu 18.04 LTS is just Ubuntu 18.04 LTS, with the standard GNOME desktop (as found on Ubuntu 18.04 LTS desktop release) replaced with KDE Plasma - ie. KDE on Ubuntu 18.04 base. 

Lubuntu = Ubuntu with LXDE (up to 18.04) or LXQt (18.10 up)
Xubuntu = Ubuntu with XFCE 
Ubuntu-MATE is Ubuntu with MATE desktop
Ubuntu-GNOME was Ubuntu with GNOME when Unity was the standard desktop, it ceased after 17.04 as Ubuntu switched back to including GNOME by default
Ubuntu-Kylin is Ubuntu with Kylin desktop
Ubuntu-Budgie is Ubuntu with the Budgie desktop
Ubuntu-Studio is Ubuntu with XFCE setup for multimedia creation
etc.  ie. the same Ubuntu base - but with different DESKTOPs or GUI (graphical user interfaces)

Which is best for you - only you can decide.

Yes you can replace your Ubuntu with Kubuntu.  You can also have more than one desktop on the same machine (my Ubuntu install also has the XFCE, LXQt, & MATE desktops installed; I select which I want to use at login). There are costs to having more than a single desktop installed though - not ideal when you're starting in my opinion.

If you want to keep your data, use 'something-else' in installing-over, and don't format your partitions.  It will take note of your applications, erase system directories, install, then add-back your additional software you added - but not touch your data.  Of course you should still backup your data to be safe.

One additional note - *whilst Ubuntu 18.04 LTS comes with 5 years of supported life, official flavors of Ubuntu (such as Kubuntu, Lubuntu, Xubuntu, Ubuntu-MATE etc) only come with 3 years of supported life.  (*[http://fridge.ubuntu.com/2019/08/08/ubuntu-18-04-3-lts-released/](http://fridge.ubuntu.com/2019/08/08/ubuntu-18-04-3-lts-released/))
