### Post by steveneddy on 2008-06-01
Compiz Fusion is a compositing window manager that allows such desktop effects as Rotating Desktop Cube, Wobbly windows, Drop shadows on all windows on the desktop, eye popping effects on open/close/minimize/restore windows, with the help of Emerald will give you cool window borders.

Most of us see it as a toy to amaze your friends, some use it productively, but mainly it's just great eye candy.

Your icon shows the spinning cube. Did you know that was Compiz?

EDIT:

If that's your desktop in your sig, then you already know what Compiz can do.

```
sudo apt-get install compizconfig-settings-manager
```

Now goto System->Preferences->Advanced desktop effects settings to customize compiz..
