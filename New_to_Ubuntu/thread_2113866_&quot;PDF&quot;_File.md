### Post by sudodus on 2013-02-08
1. Browse to its directory with your file browser and double-click on the icon in the browser window.

2. In the terminal window you can use the command

```
evince /path/file.pdf
```

where you substitute the actual path and actual file name instead of 'path' and 'file'.
