### Post by ant2ne on 2008-10-29
Thumbing through my Ubuntu Bible book, I came across this section (page 826 in my book) that describes mapping a network printer, from a windows workstation, to a linux print server. Using the URL: [http://ServerNameorIP:631/printer/CupsPrinterName](http://ServerNameorIP:631/printer/CupsPrinterName) I grabbed the wifes laptops and tested it out. It works great. Better and faster printing than my current samba print service. And the wife can even navigate to the URL and manage the print jobs. (yet to be tested though) I was amazed. But then I got curious. Is windows to unix printing just something that is supported by default in CUPS or is this a samba thing behind the scenes.

Learn something new every day.
