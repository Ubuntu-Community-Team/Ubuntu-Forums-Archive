### Post by themusicalduck on 2010-11-22
If you're using the netbook remix edition then try this -

Open a terminal (click the Ubuntu logo at the top left and search for terminal).

Then type in nautilus and hit enter. The file browser should open, and you can right click on it on the icon that appears on the left and select to keep it in launcher.

I know this seems a bit of a silly process just to get to a file browser. You only need to do it the once though. Hopefully they'll make it a bit easier to do in 11.04.

### Post by HaggisSupper on 2010-11-22
1. click on "Applications" in the launcher
2. select "Main Menu" app
3. in the Menus: tree, select System Tools
4. enable "File Browser" in the Items: pane
5. reselect applications from the launcher and File Browser should be there.
6. once you have selected the "File browser" app and it appears on the launcher, right-click the launcher button to have the choice to pin it permanently to the launcher.

There are bound to be quicker/better ways of doing this.
