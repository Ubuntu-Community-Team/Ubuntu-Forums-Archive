### Post by mastablasta on 2012-03-20
what game particulary? if you mean the ones found in software center you can use Kpackagekit in Kubuntu or in latest versions of Kubuntu the "software center" is called Muon. so just open it select the game you want and click install and that's it.

if you want some other game that is not in the repositories then you need to either (1) add their respective repository (so it will pop up in the "software center"), (2) use the game site's install instructions or (3) compile it from the source (code).
