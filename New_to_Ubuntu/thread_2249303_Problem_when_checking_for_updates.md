That third party PPA repository don't provide packages for Trusty as you can confirm [here]("http://ppa.launchpad.net/rohityadav/vlmc/ubuntu/dists/"), and that's what's causing the '404 Not Found' error message you're getting.

You'll have to remove that PPA repository from your system by opening your software sources, click on the other software tab and find the repository that you posted ([http://ppa.launchpad.net/rohityadav/vlmc/ubuntu/dists/trusty/main/binary-i386/Packages](http://ppa.launchpad.net/rohityadav/vlmc/ubuntu/dists/trusty/main/binary-i386/Packages)) and remove it. Enter your password when prompt and after that start the update manager again and you will be fine.
