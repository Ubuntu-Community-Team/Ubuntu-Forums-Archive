### Post by renkinjutsu on 2009-10-29
Here's a good place to start learning Bash scripting (very useful)
[http://www.linuxcommand.org/]("http://www.linuxcommand.org/")
and here's documentation for a lot of common linux commands


also, for unpacking tar.bz2 or whatever archives, you'd use the tar command
you can type ```
man tar
``` into the terminal to read the manual for tar (press q to exit the manual)

and as for installing, most programs will generally come with a README file, so you should be covered
