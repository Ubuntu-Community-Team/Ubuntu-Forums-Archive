### Post by howefield on 2009-08-28
> **rkakkar said:**
> This should be possible. How do I go about it on the Windows machine?

Install portableapps launcher from portableapps.com, then openoffice [http://portableapps.com/apps/office/openoffice_portable](http://portableapps.com/apps/office/openoffice_portable) (and anything else you might want to run).

You can then run the apps launcher/programmes from the usb stick in any windows computer without leaving any files on the host computer.
