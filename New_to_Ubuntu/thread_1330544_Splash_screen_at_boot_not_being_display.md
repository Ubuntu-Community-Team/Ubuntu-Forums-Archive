### Post by ArielEnter on 2009-11-18
I had this problem yesterday and I didn't know what was going on. The Splash screen at boot (the first splash displayed while starting Ubuntu) was not being display, instead, a very awful splash was displayed, something like the TV at midnight with the rainbow color bars, and a lot of measurements that said something like 1:1 EG 220 210 etc.

It came out that the package USPLASH was uninstalled, so I just used the Synaptic Package Manager to install it back. I'm a new user of Ubuntu and I wanted to share this with the other new users since I couldn't find the solution easily in the forum.

Oh I also run USPLASH in the terminal, but don't think that made the splash come back. Anyway just in case.
