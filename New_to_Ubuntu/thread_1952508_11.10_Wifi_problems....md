### Post by wildmanne39 on 2012-04-04
Hi, run this command please:
```
sudo apt-get install --reinstall b43-fwcutter firmware-b43-installer
```
watch for errors then unplug wired connection and reboot if it does not come on post the output of:
```
lsmod
```
by clicking on new reply and click # and paste the information between the brackets.
Thank you

Make sure your wireless is on because it shows it is off and has a soft block but they may go away when you install the firmware.

This part made it work thx alot wildmanne39.

The thing was that the wifi when i booted up in Ubuntu was turned off and the hot keys for turning it again didn't work, so what i did was start up in vista and turn the option for controlling the wifi on/off via hot keys and then run the :)
```
sudo apt-get install --reinstall b43-fwcutter firmware-b43-installer
```

And now i am on my wifi and not mobile broadband :D

Once again thx a lot! :D
