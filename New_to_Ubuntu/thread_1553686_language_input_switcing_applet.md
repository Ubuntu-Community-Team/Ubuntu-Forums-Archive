### Post by superzorro on 2010-08-15
It is possible to have an "icon" and select it, however I find more useful to assign a key combination to change the layout.

First, I add the "new" layout in System -> Preferences -> Keyboard

I select the default I want, and in the "Layout Options" -> Key(s) to change layout

I choose the combination, and your done. Just press the combination and it changes, press it back and it will return.

After that if you also want to add the "applet", take the pointer to the panel, right mouse button and choose "Add to Panel.." and look for "Keyboard indicator" and add it.

This should create the applet in the panel, and then with the mouse you can select the layout.

I hope you find this useful.
