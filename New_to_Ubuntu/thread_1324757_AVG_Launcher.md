### Post by Simian Man on 2009-11-12
It will be just about as useful if you never launch it :).

But seriously, open up a terminal and type "avg" and then hit the tab button twice quickly.  If the application's name begins with "avg" then it should show you what the full command is.  You can also try "Avg".

Once you know the application's name, you can create a launcher yourself.
