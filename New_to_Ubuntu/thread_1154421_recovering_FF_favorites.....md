### Post by -kg- on 2009-05-09
If you have a bookmarks backup file with the extension, .json, that is an exported FF bookmark file.  Launch FF, and from the "Bookmarks" menu, select "Organize Bookmarks."

From the menu in that window, select, "Import and Backup," then on the "Restore" drop-down list, "Choose File."  Then just browse to where the ".json" file is, select it, and the backup file will be installed to your Bookmarks.

Of course, these files are dated as to when they were saved, so if they're *real* out of date, you might want to follow kaibob's method.  That way, you have your most recent Bookmark files.
