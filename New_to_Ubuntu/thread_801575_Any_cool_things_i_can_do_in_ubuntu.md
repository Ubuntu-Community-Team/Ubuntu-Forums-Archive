### Post by Paqman on 2008-05-20
Bad news: Beryl is discontinued. 
Good news: It's been replaced by Compiz Fusion, which is pre-installed on recent version of Ubuntu.

You'll need to install your graphics card drivers if you haven't already. System > Admin > Hardware Manager will sort that out. After that you'll need to enable desktop effects in System > Preferences > Appearance.

To really have some fun with Compiz, install compiz-config-settings-manager, which will give you a ludicrous amount of control over all the different animations and plugins that are available.
