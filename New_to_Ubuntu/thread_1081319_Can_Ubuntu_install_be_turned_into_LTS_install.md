### Post by Therion on 2009-02-26
Certain releases are designated LTS (Long Term Support).  8.10 is not an LTS release and 8.04 is.  If you want 8.04 on your system, and you have 8.10 installed now, you will have to do a fresh install of 8.04; there's no way around it.  

Truthfully though, a fresh install really is the best way to go.
