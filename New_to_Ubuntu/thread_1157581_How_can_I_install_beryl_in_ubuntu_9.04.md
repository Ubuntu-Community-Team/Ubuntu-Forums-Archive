### Post by swoll1980 on 2009-05-12
Beryl merged with Compiz, and became Compiz-Fusion then changed it's name back to Compiz. It's installed by default. If you want to configure it, install compizconfig-settings-manager
```
sudo apt-get install compizconfig-settings-manager
```
