### Post by Dennis N on 2016-07-01
When you login, the **Welcome Screen** that automatically pops up has the link to the community forum and other useful information. From it's main menu, click on "Community" for that link. The menu's "Software" link opens the **Software Boutique** for installing from the curated software collection.

If you disabled the Welcome Screen, it is still accessible from System > Welcome.
