### Post by ugm6hr on 2009-02-16
Mike

As mentioned, the AOl software is not available on Ubuntu, but the community most definitely is.

You can still see and interact with your AIM buddies using Ubuntu's built-in software **Pidgin Internet Messenger**.

I have never used the AOL gateway myself, but I am certain that you will not be the only Ubuntu user wanting to use it; given the AOL software was just Internet Explorer with AOL branding (and Netscape before that), I do not foresee any problems using it.  Consider asking on the AOL forums how to get the best out of AOL using FireFox; the fact that you use Ubuntu is largely irrelevant.

Just to clarify: you *did* have Windows on your computer before you installed Ubuntu (it was installed for you by the manufacturer).

There are some helpful links in the Start here thread linked below to help familiarise yourself.  As stated, you will probably need to install Flash etc, which is easily done.
