### Post by clairefun on 2009-05-20
I think it's to do with the new version being released and the repositories not being updated yet. What I did was downloaded the newest dock and plug-ins files from [here]("http://developer.berlios.de/project/showfiles.php?group_id=8724") and it worked perfectly. The same thing happened for me with my cairo-dock from the repositories. You can update the repositories using instructions [here ]("http://www.cairo-dock.org/ww_page.php?p=From%20the%20repository&lang=en#7-Ubuntu")- I think / hope that's all okay, I've only been using ubuntu for 2 weeks myself - but I do like my cairo-dock :)
