### Post by talsemgeest on 2009-01-09
You should have downloaded an ".iso" file, which can then be burned onto a disc. The files that you have would normally be contained inside the iso. The best thing to do would be to download one of [these.](http://www.ubuntu.com/getubuntu/download) If you for some reason can not download one of these, it may be possible to re-create one out of the files, but it would not be easy.

### Post by pastalavista on 2009-01-09
You need to burn those files to a CD and boot from it. You can't install from an active hard disk. If your E drive is a CD drive, then you need to re-start your computer and change the bios boot order setup to boot from CD rom drive first.. before the HD boots.
