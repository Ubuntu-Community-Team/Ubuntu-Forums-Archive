### Post by mcduck on 2009-06-25
Check the size of the image. Dimensions for skydome images should be powers of two, for example 1024x1024 works, 512x2048 works, but 1000x1000 doesn't work.

Also the maximum image size depends on your graphics card, so if you have problems even when the image size fills the above rule you should try with a smaller image.
