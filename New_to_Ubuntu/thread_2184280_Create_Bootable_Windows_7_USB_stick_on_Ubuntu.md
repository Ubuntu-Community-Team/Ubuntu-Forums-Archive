### Post by Mark Phelps on 2013-10-28
You can't run Win7 from a USB stick -- like you can Linux distros -- if that is what you're asking.

But, you can install Win7 from USB stick -- and the most reliable way of doing that is using the Windows 7 USB/DVD Download tool from MS.  You will need an ISO of Win7 and you can then use that tool to make a bootable USB Win7 installer.  But, you need to do that using MS Windows.
