### Post by unutbu on 2008-10-06
Right-click on the left-most icon in the notification area. There might be some faint double dotted vertical lines. Right-click on them if they are there. You should see "Lock to Panel" as one of the items in the pop-up menu. Make sure that it is not checked.

Then middle-click the date/time icon to move it into the notification area.
