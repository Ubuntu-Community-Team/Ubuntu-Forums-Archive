### Post by starcannon on 2008-11-19
First thing that comes to mind is Electric Sheep available in the Synaptic Package Manager.

System>Administration>Synaptic Package Manager
Search for "electric sheep"

Caveat, I can't remember for sure, I know this displays fractals from the Electric Sheep project, but I don't know if there is a way to specify for the Mandelbrot set.

GL and have fun.

P.S. here is a link to the electric sheep website: [http://electricsheep.org/](http://electricsheep.org/)
P.S.S. It takes a while to get your first sheep, after its going though, it works nicely, you can run it from the command line if you wish:
```
electricsheep
```P.S.S. Okies, likely my last amendment to this post lol, you got me playing with fractals again, so I thought I'd take time to let you know that Ubuntu has xfractint available in the synaptic package manager as well (linux port of the famous fractint fractal generator)
