### Post by xtremethegreat1 on 2011-02-13
You have the others in:

Applications >> Others

On the top left corner of the screen. If it is not there, then right click on applications and click edit menu. From there, you can select others on the left panel and check your avg icon on the right panel. Plz lemme know if that works...
