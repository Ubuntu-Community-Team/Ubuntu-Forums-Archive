### Post by Didius Falco on 2009-05-07
Go back to **System/Administration/Language Support**. When it opens up, you'll see the options to change the language for both menus and windows and to set it as the default language for everyone at start up and login.

After you change it, log out and back in and it will take effect.

Regards,

Didius

### Post by powel212 on 2009-05-07
I switch between Chinese and English and this is what I do. It should be almost the same.

open a text editor

type: CTRL+SPACE

This will start the SCIM module

You will now see a small icon that should look like a small keyboard on your upper taskbar. (see attached image)

Once you see the little icon you can right click the icon and select properties and then you can configure your input settings to your liking.

I hope that helps

Powel

### Post by neogaia on 2009-05-07
This thread is very helpful!

I searched through much more complicated threads but figured this method out on my own.

Usually clicking on the keyboard in the taskbar is what you have to do to change that you are typing.

And if you type F7 after a word that changes it to Katakana.

If you are just a Japanese language student/learner and not a Japanese native, I would recommend not downloading the basic translations, and extra translations if you don't want your computer menu to be in japanese, but just want to be able to type in it.

you just need input methods and fonts when you install. Thats what I did.
