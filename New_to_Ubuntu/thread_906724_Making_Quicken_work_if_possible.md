[Linux != Windows.]("http://linux.oneandoneis2.org/LNW.htm")

Read that first, it will clear up a lot of things.

I suggest you try "gnucash". *Read the bottom of this page if you don't know how to install software in Ubuntu.*

You can import your Quicken data into GNUcash:

[http://www.gnucash.org/docs/v1.6/C/t3948.html](http://www.gnucash.org/docs/v1.6/C/t3948.html)

Ubuntu can handle JPEG without any problem.

The default installed Image Viewer can read them, as can the default installed browser. The default installed Image Manipulator (Gimp) can save any picture as .jpeg of .jpg.
