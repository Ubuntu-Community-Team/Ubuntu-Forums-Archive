Hi,

You can easily create an archive (compressed or uncompressed) of the directories, and attach the archive file to your email. Using a graphical archive manager like File Roller, you can drag and drop any directory into it, and choose the type of archive you want when saving it. Then attach the newly created archive file to your email.
