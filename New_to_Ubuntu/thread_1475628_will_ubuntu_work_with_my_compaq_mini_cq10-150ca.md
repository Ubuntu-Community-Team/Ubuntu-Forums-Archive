### Post by warfacegod on 2010-05-07
I almost never use LiveCD's anymore. A LiveUSB, because the computer can read it faster than a CD, will give you a better idea as to how your computer with perform speed wise if you install Ubuntu to your HDD.

Mustard is correct. If everything runs fine in Live, then should run fine in an actual install. Its possible you'll have minor issues with your drivers at first but they should clear themselves up once you do an update.

If the only machine you have available to you is Windows then unetbootin will be your friend in creating a startup USB.
