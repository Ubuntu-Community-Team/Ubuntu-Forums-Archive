Also, what type of firewall do you use?
If it concerns you, the most generically safe firewall you should be using is ufw, as that is what comes provided with Ubuntu by default (it is turned off though, as a firewall is generally not needed on non-server platforms).

Uninstall whatever you currently have, then install **gufw**.
Go into 'System->Administration->Firewall Configuration' and
[LIST=1]
[*]Check **Firewall Enabled**
[*]Check **Deny incoming traffic**
[/LIST]
And you are all setup, you can close and leave it... forever. :)
