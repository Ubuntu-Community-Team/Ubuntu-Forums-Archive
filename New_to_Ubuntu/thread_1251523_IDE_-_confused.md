### Post by credobyte on 2009-08-27
> Abbreviated as *IDE*, a [programming]("http://www.webopedia.com/TERM/I/program.html") environment [integrated]("http://www.webopedia.com/TERM/I/integrated.html") into a software [application]("http://www.webopedia.com/TERM/I/application.html") that provides a [GUI]("http://www.webopedia.com/TERM/I/graphical_user_interface_GUI.html") builder, a text or code editor, a [compiler]("http://www.webopedia.com/TERM/I/compiler.html") and/or [interpreter]("http://www.webopedia.com/TERM/I/interpreter.html") and a [debugger]("http://www.webopedia.com/TERM/I/debugger.html"). Visual Studio, Delphi, JBuilder, FrontPage and DreamWeaver are all examples of IDEs.
Source: [http://www.webopedia.com/TERM/I/integrated_development_environment.html](http://www.webopedia.com/TERM/I/integrated_development_environment.html)
