Most of ubuntu is "open source" and therefore is covered by various open source licenses, GPL, etc.

There are some packages provided by various partner repositories that may contain non open source code, eg ubuntu-restricted-extras, libdvdcss2, etc etc, which are covered by proprietary licences, but there is no quick and simple way to show all of those.

Every program in ubuntu is licensed in its own way. Each programs license can can found somewhere in its documentation. If you'd like to see and read what the more common licenses are, go to /usr/share/common-licenses, where you'll find the documents for such licenses as Apache, Gpl,Lgpl, and so forth.
It makes for some interesting reading.
