### Post by mardybear on 2012-10-11
> **Pletched said:**
> Is the Firefox backspace button disabled?  Why doesn't pressing it make the browser go back to previous page? And what's its purpose now?

Hi...

I believe the backspace button is disabled by default.

To active the keyboard's backspace button so it allows you to return 'back' to the previous webpage:
open a new tab
enter about:config as the address
search for backspace
set the value to 0 - that's a zero

Now the backspace button should take you back to the previous page, might have to restart firefox (can't remember).

mardybear
