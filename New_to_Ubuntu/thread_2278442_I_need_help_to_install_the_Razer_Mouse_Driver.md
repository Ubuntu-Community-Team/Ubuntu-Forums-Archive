### Post by buzzingrobot on 2015-05-16
If it's the archive at this link ([http://bues.ch/razercfg/razercfg-0.32.tar.bz2](http://bues.ch/razercfg/razercfg-0.32.tar.bz2)), it contains only the source code from which you can compile (build) the driver.  After it is succccessfully built, then you can install it. (The readme contains those instructions.)

If you have no experience with building and compiling source code, my first suggestion is that you search for a driver that has already been compiled and packaged in a .deb format so you can easily install (and uninstall it) on Ubuntu.
