### Post by kc1di on 2013-02-15
Hello czgirb,

Here's what you need to do :

First go to Applications >System Tools> Preferences >Main Menu 

select System Tools and click on New Item. (It's the tab 2nd down on the left side.

when the dialogue box comes up just type in ubuntu-tweak in the command box and it should find the Icon for you.  then close the box.
Now while hitting the Alt key right click on the top panel 
the first choice in the list that comes up will be add to panel
select custom launcher again type ubuntu-tweak in the command box 
and it will come up select ok and your in business.
this method will add it to both the panel and your system menu.
good luck.

P.S. After writing this an alternative instead of using custom laucher select Application Laucher and you can select it from the dropdown menu- under system tools.
