### Post by Catalyst2Death on 2008-08-02
If you really want to dive into understanding how the operating system works, I would suggest trying Arch Linux.  In order to get the operating system running, you have to install and configure every aspect of how the system runs.  It has been very informative, however it is not generally suggested for absolute beginners.  I've been using ubuntu for almost a year, and I am by no means and expert, but I just got arch installed and working fine a couple days ago.  I really enjoy it, and if your willing to tinker much more than normal, I would definitely suggest giving it a look.

[www.archlinux.org](www.archlinux.org)
