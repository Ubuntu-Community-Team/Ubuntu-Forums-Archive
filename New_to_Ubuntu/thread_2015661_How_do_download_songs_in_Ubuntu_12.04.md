### Post by cbanakis on 2012-07-03
beemp3.com seems to work fine.

All you need to do is save the files, and put them in your music folder.

Then Rhythmbox will automatically see and index your music.

Pretty much the same as Windows.  Only better.

Better = free + very unlikely to get virus' and want not.


For the most part, everything that works in Windows works almost the same in Ubuntu.

The only exception is windows programs dont run in linux.
But youtube, netflix, beemp3.com, etc, etc...  are not programs, they are just websites, and most websites work fine.

You can even install most windows programs using "wine"

Its a program you can find in the Ubuntu software center that lets you run windows programs.

It works very well, but its not perfect, and does have its limits.

### Post by Dahmaster95 on 2012-07-03
When I installed Ubuntu, I transferred all my songs to a hard drive then back to the music folder. RythmBox automatically picked them up from there. If you want to import music to RythmBox, go up to the  toolbar all the way at the top click music. And a dropdown menu will appear with the first two choices as import file\folder.
