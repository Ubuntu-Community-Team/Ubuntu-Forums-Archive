### Post by RJ12 on 2011-04-10
If it is a theme / icon pack from gnome-look.org, you should just be able to drag the .tar.gz file on the Appearance Window (System > Preferences > Appearance) and it will install it for you. There is no exact way to install a .tar.gz file as it could be anything. It could be source code where you have to compile it your self or it could be a compressed binary, depending on what you are trying to install is how you are going to install it.

So basically if you could tell us what you're trying to install, we can help you better.
