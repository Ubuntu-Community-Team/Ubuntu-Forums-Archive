### Post by Alwimo on 2011-08-04
In the top right hand corner, there should be an icon that looks a bit like a power button. It's the one farthest to the right.
 
Click on it, and a few options appear under it. Choose "Shut Down", which is the second one from the bottom in 11.04.

### Post by tommpogg on 2011-08-04
> **Alwimo said:**
> In the top right hand corner, there should be an icon that looks a bit like a power button. It's the one farthest to the right.
 
Click on it, and a few options appear under it. Choose "Shut Down", which is the second one from the bottom in 11.04.

Indeed, this is the easiest way.

Two alternatives are:
1) Press Ctrl+Alt+Del. A window with the shut down option should appear.
2) This alternative is a little more advanced; open a terminal and type
```

sudo shutdown -h now

```
