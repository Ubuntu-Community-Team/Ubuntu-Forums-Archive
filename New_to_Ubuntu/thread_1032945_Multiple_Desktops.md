### Post by Zeroberry on 2009-01-06
Open Compiz Setting Manager from System -> Preferences and then check the box next to "Desktop Cube". If you don't have it try writing "ccsm" in the terminal, or in case you don't have it installed, "sudo apt-get install ccsm".
