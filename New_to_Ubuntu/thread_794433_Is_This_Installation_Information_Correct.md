### Post by ugm6hr on 2008-05-14
There are a number of issues that have been encountered by Hardy users, but that review doesn't really address any of them (except perhaps the nvidia issue - not sure about that one).

The problems with Update Manager and busy repos have settled, Compiz is just as easy to set-up (although obviously not if you can't find the Hardware Drivers menu item - which it is now renamed from Restricted Drivers), although CCSM is not default (which presumably is to keep things simple i.e. No effects, Normal and Advanced settings only).  I don't use the "cube" so not sure about that.  Broadcom wifi has always been a problem with all distros - that's not news to us.

The final comments about Mint and PCLOS sound like he/she has never tried them with his hardware.

If you are going to believe reviews, try something more impartial, or from a current Ubuntu user.

If you have specific concerns - ask about them.

If you are going to "upgrade" - I'd recommend a fresh install though.
