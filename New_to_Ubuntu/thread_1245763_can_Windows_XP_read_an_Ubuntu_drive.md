### Post by oldfred on 2009-08-21
If you are using both Windows and Ubuntu the best solution is to add an extra partition that is NTFS or FAT32 for shared data. Both windows and linux read those types without problems. Ubuntu reads windows but windows is not good about reading anything that is not windows.
