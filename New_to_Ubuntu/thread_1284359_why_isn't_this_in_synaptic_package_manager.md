### Post by QIII on 2009-10-06
So many applications, so few people and so little time.

As I understand it, applications for possible inclusion for the repositories are rigorously tested so that they can be determined to work properly with Ubuntu.  This would imply that either:  

1.  Some applications, however old, do not get considered.  Triage, you know.  

2.  Some applications take longer than others to get into the repositories.

If you feel strongly about it and don't want to install from anything but the repositories, you can recommend its inclusion.  But it won't happen tomorrow.

Sourceforge is generally trusted, but that is no guarantee that every individual application is above suspicion.  You might try googling the application and see if you can find other users to give you some guidance.
