### Post by IHeequ5i on 2011-07-25
Dell - if I'm not mistaken - sells laptops with Ubuntu already installed. No clue on features, prices, or anything like that.

I messed with various distributions (Mint, Fedora, RedHat, and others) before settling on Ubuntu as the one to stick with. This community in particular is VERY helpful to new folks, particularly those who want to learn their way around Linux. I strongly recommend getting one of the LTS builds if you decide to go with Ubuntu.  It sounds like this is a computer that needs to be stable.

Installing Ubuntu is reasonably easy, especially if you intend to blow away whatever OS is already installed on the computer and start "clean".  If you make a new thread when you get your laptop and post the specs, I'm sure the folks here will be more than happy to chime in with tons of advice on partitioning, etc.

One beautiful thing about Ubuntu is that it has a LiveCD. You can boot your computer off the CD and get to know your way around the OS. This may help negate some of the learning curve that you're going to hit.

The biggest difference I know of between Linux and Mac is that Linux folks use the Terminal Window to type in commands a lot, while Mac is almost exclusively GUI driven. You can do a lot with the GUI in Ubuntu, but some things REQUIRE the command line.

Figured you should know what you're getting into before you do it.

Best route to go is download an ISO of whatever distro you want to try. Burn it to CD as an ISO image file - this will make it bootable. Or you can create a bootable USB stick using whatever utility the Mac OS has or a program like 'unetbootin'.
Put the LiveCD in the tray or plug the usb stick in, change the bios settings to boot from the device of your choosing. When the menu screen appears select TRY. When lights have stopped flashing and the desktop appears you will have a system running in RAM only. It will not write anything to your hard drives unless you allow it to. Explore what is available to you.
If you do not like the first one try a different distro.
One thing to remember is it will be running off a CD or USB neither of which run as fast as an internal hard drive.
