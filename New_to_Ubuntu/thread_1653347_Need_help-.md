### Post by frustratednerd on 2010-12-27
Hello. To install the gnome-network-admin package, go to Terminal (which can be accessed by going to Applications -> Accessories -> Terminal), and type the following (or better yet, just copy&paste the following to prevent a potential typo):

```
sudo apt-get install gnome-network-admin
```

It will ask you for your 'sudo' password. Your sudo password is simply your log-in password.
