### Post by wgn on 2011-10-25
If Gnome is the only desktop environment installed (default) then there aren't any options to choose from.  To install some additional desktops

[https://help.ubuntu.com/10.04/config-desktop/C/other-desktops.html]("https://help.ubuntu.com/10.04/config-desktop/C/other-desktops.html")

Others are also available in the repositories.  Should give you choices at login after that.  Just search through the forums and the wiki for instructions when you're ready to change which desktop is the login default.
