### Post by hamofgrey on 2008-12-27
This is probably to do with windows management settings within a program called Compiz-Fusion. First perhaps you could try turning all the flashy windows animations off within System -> Preferences -> Appearance.

A second possible idea is to install a program called Advance Desktop Effects Settings from Applications -> Add or Remove Programs. Once you run this program you will see the many different options which you can change. These might help make this easier.
