### Post by monkeybrain2012 on 2013-03-26
Yeah. Install the CompizConfigSettingsManager . Then start the CCSM, find the Ubuntu Unity Plugin, and then in "Hide Launcher" change "never" to "autohide"

### Post by deadflowr on 2013-03-26
You can hide the launcher in the appearance section of the system settings.

Open the dash and type 'appearance'.

Then go to behavior and turn the auto-hide launcher on.

Set the reveal to where you'd prefer. 

And then set the sensitivity to whatever you'd like.
