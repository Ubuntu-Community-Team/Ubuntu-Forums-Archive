### Post by themusicalduck on 2009-09-10
If you click on Default next to the sound you want to change, it lets you select custom from a menu.

### Post by Tamlynmac on 2009-09-10
preferences/sound/sound tab

Left click on default/disabled (whichever one is displayed) and select custom from the pop up list. Then select the .wav file from the displayed list or find the one you wish to use in your file system.
