### Post by Rasa1111 on 2011-05-21
Hi Calihan.,

Gnome themes are actually very easy to install once ya know how.. 

Lets use this theme as an example.. 

[http://gnome-look.org/content/show.php/Ambiantastic?content=127730]("http://gnome-look.org/content/show.php/Paranoid?content=136003")

So at that link,
We would click the 'download" button..
(top one)

Choose to "save file".

Once download finishes..
Open your "Appearance" window, (in your "System>Preferences menu)

 make sure the "themes" tab is selected,
then you simply drag the file you just downloaded (probably in your downloads folder), into the themes window..
and if all goes as it should.. You should see a window pop up that says something like
"Theme installed correctly", and then you'll see it in the list of installed themes. 
Then just click it to use it, and your theme will change. 

If you need more help just ask.

Hope it helps
good luck

It just means that the creator of the theme thinks that the "any color you like" icons look best with the theme, but you can ignore it if you like it. 

That icon set is here~ [http://pobtott.deviantart.com/art/Any-Color-You-Like-175624910](http://pobtott.deviantart.com/art/Any-Color-You-Like-175624910)

But you don't need it if you don't want it. 

It is basically just a "recommendation" by the theme creator, Saying ("I think these icons look best with my theme"),  that is all.

### Post by Frogs Hair on 2011-05-21
Many themes and icon sets can be dragged and dropped directly from downloads to appearance preferences . Zip files will need to be extracted before doing this . if this does not work extract again because there is usually some thing in the archive that won't allow the theme to install an example would be an emerald theme .

You can also use the install button in appearance preferences > themes > install . You would then migrate to the folder where the theme is stored and select open . Extraction may also be required before hand .
