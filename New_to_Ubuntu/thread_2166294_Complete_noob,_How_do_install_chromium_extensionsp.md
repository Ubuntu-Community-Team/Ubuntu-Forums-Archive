### Post by ant2 on 2013-08-08
Fit to width can be added to Chromium as an [extension]("https://chrome.google.com/webstore/detail/fit-to-width/dfdmbpdbpocelpgahddghhkomnefchkl?hl=en").

Opera is available in the Lubuntu Software Centre

Menu > System Tools > Lubuntu Software Centre

### Post by ant2 on 2013-08-08
If you follow the link 'extension' link, posted above, you should find an 'Add to Chrome' button top right of the page that opens.

If you open Lubuntu Software Centre and type Opera into the 'search package' box Opera should show in the resulting list. It does for me here using Lubuntu 13.04.
