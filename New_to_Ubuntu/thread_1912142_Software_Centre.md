### Post by Dennis N on 2012-01-20
The lubuntu-restricted-extras package definitely exists. I am looking at it right now in the Synaptic package manager. Alternately, use the command line to install:

```
sudo apt-get install lubuntu-restricted-extras
```

You don't need to move any files manually. Chromium or Firefox will detect and use the flash plugin once it is installed.

The only flash plugin on this Lubintu 11.10 system is at:

/usr/lib/flashplugin-installer/libflashplayer.so

There is no libflashplayer.so in /usr/lib/chromium-browser/plugins

Good luck.
