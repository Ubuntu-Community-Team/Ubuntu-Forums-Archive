### Post by yamhill on 2010-09-24
I'm posting here because I new to ubuntu.  I'm trying to add usefulness to an old PC that currently runs xp sp3.

I get stuck during the 10.4 install because it doesn't recognize the liteon dvdrw drive that is running the install - go figure.

The dvdrw drive works fine in xp.  It is recognized by the bios as shw-160p6s.  It is slave on the first ide - master is a hdd, and I also have another hdd on the second ide.

The motherboard is ecs rc410l/800-m the cpu is Intel 805d 2.66 dual core.

The install program prompts me for a driver CD/floppy, but I don't have and can't find one.

---

### Post by yamhill on 2010-09-26
I searched more broadly and found some clues that pointed me to the jumper settings for the drive.  I changed the jumper on the drive from cable select to master - and all is well.
