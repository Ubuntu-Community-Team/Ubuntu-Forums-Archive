### Post by Karen Vincent on 2012-10-27
I kept getting this 'cannot download release notes' problem with the 'normal' upgrades offered so decided to wait until the LTS version became available only to find the same problem!

Searching the various online forums there was a huge array of complicated solution options usually about repositories which as a non geek I found really off-putting. One word kept appearing though -  ' proxy'.

I use a basic laptop with home broadband through a router etc for 99% of my online activity BUT on occasion I go into public networks some of which do require me to change the proxy server settings. Usually I do this on the browser and switch back at the end of the session. But I wondered 'what if' something gets switched over somewhere else, and sure enough when I looked in System - Preferences - Network Proxy there was my manual proxy public library setting. 

I deleted this proxy setting (even though it hasn't been used for some time and was not interfering with anything else), checked the default setting was 'no proxy', returned to the Upgrade Manager and this time it allowed me to proceed with the upgrade process.

Just thought I would post my solution on the most recent 'most similar to' thread as it may just help someone else!

Now to complete the upgrade ...

Karen
