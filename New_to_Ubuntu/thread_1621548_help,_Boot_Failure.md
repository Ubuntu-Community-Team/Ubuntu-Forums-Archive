### Post by lmarmisa on 2010-11-14
It seems that you deselected your hard drive as the boot device when you entered the BIOS menu.

You have to go back to that menu, select the hard disk as boot device and save the new BIOS configuration. 

Other possible options would be to select in the BIOS menu an option similar to "Load Optimal Defaults" or "Load Failsafe Defaults". Maybe these options are able to define your hard drive as  boot device.
