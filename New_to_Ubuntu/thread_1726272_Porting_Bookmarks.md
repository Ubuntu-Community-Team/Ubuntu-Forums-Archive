### Post by lmarmisa on 2011-04-10
Welcome to the forums.

First of all, verify that the CD is mounted. If the CD is not automounted, select Places -> Label of CD.

Open Firefox and select Bookmarks -> Organize Bookmarks. A new window is opened. 

Select Import and Backup -> Import HTML -> From an HTML file and then locate the HTML stored in your CD (look for the label of the CD below Search and Recently Used or visit the folder /media/LabelOfCD). Finally open the HTML file.

An interesting alternative is to use the add-on for Firefox named [Xmarks]("http://www.xmarks.com/").
