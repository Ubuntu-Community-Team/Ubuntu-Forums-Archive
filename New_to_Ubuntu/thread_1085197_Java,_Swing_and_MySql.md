### Post by Xiong Chiamiov on 2009-03-02
AFAIK, Python is as out-of-the-box cross-platform as Java; it's just more likely that a Windows user will have the JVM installed than Python.

MySQL works just fine on OSX, as you could see if you visited [their download page](http://dev.mysql.com/downloads/mysql/5.1.html#macosx-dmg).  I believe it's even installed by default (albeit an old version).
