### Post by Gazneth on 2009-11-17
Thunderbird makes it very easy to set up another account. Go to the menu entry for account settings and click the box for new account, enter your account details. Be sure if you want to use only one inbox for all accounts to use the global inbox setting. If you want a separate inbox just specify a different folder for your mail. Either way works fine it's really an appearance preference, you can always tell which account a message came to. When sending messages there is a drop down box in the sender area to pick which account you are sending from.
