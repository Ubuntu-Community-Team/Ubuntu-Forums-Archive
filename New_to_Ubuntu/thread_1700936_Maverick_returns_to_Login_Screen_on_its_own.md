### Post by Hedgehog1 on 2011-03-06
This is functionality you can turn on and off.

Menu to: System >> Preferences >> Screen Saver.

Un-click the 'lock when screen saver active', and this will stop happening.

***The Hedge***

:KS

p.s. It is meant for security for workstations that may be left unattended.
