### Post by igknighted on 2008-10-26
In windows you only get updates on the OS itself, and some select microsoft products as well (Office, for example).  Even there, they will give you a list of about 100 even more cryptic updates to install.

In Ubuntu (and really any linux distro), your system comes installed with a lot of software, and all those software pieces can get updates from ubuntu.  In Ubuntu, updates issued mid-release are only bug-fixes and security updates, so it is highly recommended to install them.  It is not impossible for problems to arise after updates, but it is rather rare (and you are more likely to run into bugs before the updates than problems afterwards.
