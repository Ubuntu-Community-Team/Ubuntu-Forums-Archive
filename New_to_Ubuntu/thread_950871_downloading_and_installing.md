### Post by ChildOfMana on 2008-10-17
Applications > Add/Remove Programs

Use the search function to find what you're looking for, tick the little box next to the program you want to install then click the *Apply Changes* button.

As far as installing a downloaded program goes it depends - do you have the program in a tar.gz file, or a .deb file?
