### Post by &amp;KyT$0P# on 2019-01-04
It could be related to Linux caching stuff in RAM.  The first time an app is launched, it loads from disk.  But on subsequent launches it loads from the in-RAM cache, which is much faster.

---

### Post by Impavidus on 2019-01-04
When an application is started, it must be loaded from a file. Furthermore, the application itself may load some files. In case of firefox, it will load a lot of browsing history and stuff like that. Assuming you've got plenty of RAM (as most computers have nowadays), all those files are cached and the next time you start the application it's much faster.
