### Post by blueridgedog on 2009-01-25
The 12/2008 issue of Linux Journal reviewed the Kindle and the fact that it was basically a Linux device and that the books were in mobipocket format with DRM added.  It is a good read and there is some interesting information in the article if you are interested in the format.  You should be able to get it from their web site.

The author is Daniel Bartholomew, daniel-bartholomew.com.

Perhaps he can suggest some resources for getting the format on Linux (assuming you do not have a DRM imparied file).
