Well, [here's the info I found for Palm desktop.](http://appdb.winehq.org/objectManager.php?sClass=application&iId=688)  And unfortunately I couldn't find information on abcEdit.  What does it do?  Do you think you would like us to try to find Linux alternatives for you?

As far as Palm Desktop, I'm not sure you have to use that Windows application.  If you'll go into Add/Remove Applications and search for "Palm" you'll find these applications which you might like:

PalmOS Devices (connects your Palm device with Ubuntu)
Evolution (synchronizes with Palm devices)
KPalmDoc (for KDE)
KOrganizer (calendar and scheduling)
J-Pilot (this one I think you'll like, a replacement for Palm Desktop)
MyVCard (edits contacts)

So just install those and give them a try.  You might like them better than what you originally had.  Or just be better able to work with them than through WINE.
