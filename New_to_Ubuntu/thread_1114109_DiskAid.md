### Post by cariboo on 2009-04-02
DiskAid will not work with Linux. You may be able to use it if you install VirtualBox. Get the package [here]("http://www.virtualbox.org/wiki/Linux_Downloads"), as the version in the repositories has problems with 64-bit network drivers. Once you have Windows install make sure to install the guest additions. 

Once you have Windows running in a vm click Devices and select guset addtions.
