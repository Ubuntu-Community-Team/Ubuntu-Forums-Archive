### Post by I2k4 on 2011-04-14

I'll be doing ordinary business using any software, and suddenly a random key press will trigger the Shutdown/Suspend etc. dialog box, as if I'd hit the Shutdown icon on the Panel.  There's no rhyme or reason, no particular key or event to warn - I've had twenty or thirty occurrences of this with no common element I can find.  If I happen to be typing too quickly, I'll not only trigger the dialog but the second key press will activate the Shutdown itself, so it actually shuts down.  

I don't have any unusual software installed - I've made the Panel retract to create more room on the screen, but that's not causing the problem on my other computer.  Appreciate any advice or experience, either on Acer netbooks or otherwise.

### Post by I2k4 on 2011-04-17
Thanks, pretty sure it's not the touchpad.  I can't replicate the problem with the touchpad or any keys intentionally.  The shutdown dialog just seems to "happen" after keystrokes, that are all over the keyboard.

UPDATE:  marked "Solved" by workaround - the random keyboard strikes apparently create the equivalent of a mouse-click in the extreme bottom right corner of this Acer netbook.  I moved the Shutdown icon away to the other side of the Clock, and inserted several Spacers to desensitize that area - seems to work.
