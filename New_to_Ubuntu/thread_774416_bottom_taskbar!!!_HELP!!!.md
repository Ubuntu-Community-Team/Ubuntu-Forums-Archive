### Post by anuban on 2008-04-29
> **iwallet said:**
> i dissabled the bottom bar in ubuntu where the windows minimize to!!! how do i get it back?!! please help!!!

Just right-click on the upper bar - Add new panel.
Select Bottom to make it a bottom bar.
Now right-click on this bar - Add to panel
There will be a launcher, drag and drop on this new bar something mentioned as Menu list or menu windows.  (Can't remember the exact name, sorry)
Now when you minimize the application windows will be visible there as before.
Hope that helps.:)
