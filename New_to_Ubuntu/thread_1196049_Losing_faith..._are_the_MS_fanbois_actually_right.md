### Post by executorvs on 2009-06-24
medibuntu is an additional software repository, not a peice of software.
you can add it to your list of repositories by typing in a terminal
"sudo wget [http://www.medibuntu.org/sources.list.d/jaunty.list](http://www.medibuntu.org/sources.list.d/jaunty.list) --output-document=/etc/apt/sources.list.d/medibuntu.list"
this will add the repositories location, next you need to add the security key for authenticating the packages from the repository type
"sudo apt-get update && sudo apt-get install medibuntu-keyring && sudo apt-get update"

once these are done you can reload synaptic and when you search for real player it should be listed.

another person mentioned adding the restricted extras to ensure you have all the proper plugins and codecs. this can be done by typing in terminal
"sudo apt-get install ubuntu-restricted-extras"
or you can select the package from within synaptic
