### Post by trag on 2014-07-08
[COLOR=rgba(0, 0, 0, 0.8)][FONT=Roboto Slab]1) Open Applications --> Accessories -->Password and Encryption Keys[/FONT][/COLOR]
[COLOR=rgba(0, 0, 0, 0.8)][FONT=Roboto Slab]2) Right-click on the "login" keyring[/FONT][/COLOR]
[COLOR=rgba(0, 0, 0, 0.8)][FONT=Roboto Slab]3) Select "Change Password"[/FONT][/COLOR]
[COLOR=rgba(0, 0, 0, 0.8)][FONT=Roboto Slab]4) Enter your old password and leave the new password blank[/FONT][/COLOR]
[COLOR=rgba(0, 0, 0, 0.8)][FONT=Roboto Slab]5) Press ok, read the security warning, think about it and if you still want to get rid of the Unlock Login Keyring dialog, choose "use unsafe storage"
goodv luck
trag[/FONT][/COLOR]
