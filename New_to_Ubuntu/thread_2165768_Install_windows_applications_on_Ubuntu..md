### Post by lego11 on 2013-08-06
Hi,
To run Windows applications and programs you need to install a compatibility layer known as Wine. You can get it from the Ubuntu Software centre (The shopping-bag icon with an A on it).
However, it is not 100% perfect so some applications will refuse to run, or only run partially.
For iTunes the best solution is to set up a Virtual Machine (i.e. running Windows in a program) such as Virtual Box and activate the phone from there. 
Or, if you could get a hand on a Windows CD, we could set up a dual boot (Ubuntu + Windows) system.
Let me know.

### Post by Mark Phelps on 2013-08-06
Your neighbor should have ASKED before installing Ubuntu -- because iTunes doesn't work well in Linux.  You have to install Wine, and then you get poor results.

Same is likely to be true of the other stuff you need.

You should check the WineHQ application compatibility database for the other things you want to use.  Anything not listed, or rated lower than Gold is likely not going to work well enough to be useful:  [http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true]("http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true")
