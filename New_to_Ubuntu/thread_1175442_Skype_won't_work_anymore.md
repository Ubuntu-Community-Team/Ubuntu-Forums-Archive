### Post by HereInOz on 2009-06-01
Try opening a terminal and running it from there.  Just open the terminal and type Skype and press Enter.

It will give you any error messages which occur and you may be able to work out why it is not opening from those error messages.  

Also, check whether the Skype service is still running by typing top in a terminal.  It will list the programs running in background and if you see Skype there, but it is not opening, then it is trying to run but is failing to bring up the gui.
