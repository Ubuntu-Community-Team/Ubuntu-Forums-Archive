### Post by shadow-of-sin on 2008-05-22
Execute the following commands:
```

sudo mkdir /media/win
sudo mkdir /media/lin

```

Open up /etc/fstab in a text editor (e.g gksudo gedit /etc/fstab) and add the following lines to it:
```

/dev/sdb2 /media/win vfat defaults 0 0
/dev/sdb1 /media/lin ext3 defaults 0 2

```

Then execute the following command:
```
sudo mount -a
```
