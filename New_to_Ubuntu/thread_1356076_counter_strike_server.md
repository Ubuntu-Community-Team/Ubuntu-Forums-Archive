### Post by teward on 2009-12-15
@KommaH:  There's a LOT to set up a server.

Setting up a server on ANYTHING means you need to clear it with your ISP, so they can assign a static IP address to your computer.  Having said this, you need to have the software configured for a port, firewalls configured, port forwarding configured, you need a lot of bandwidth...

oh, and experience with setting up servers.  I've set up and maintained Ventrilo servers before, so its actually harder than you think.
