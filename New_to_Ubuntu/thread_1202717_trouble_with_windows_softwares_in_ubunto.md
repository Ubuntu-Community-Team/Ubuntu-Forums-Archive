You have to install them in the actual virtualbox, not on your ubuntu install. So, if you're using a cd, make sure that in the virtualbox settings you have it set so the virtualbox software has access to your cd-rom drive. Then, insert the cd and mount the cd-rom drive- if your software has auto-run, it will prompt you to install it, otherwise you'll need to go to "run" in start menu to get it to start.

### Post by emeraldgirl08 on 2009-07-02
You can use recovery disks to restore your Windows. If you do not have the disks you will have to hunt around for them. I found restore disks on a forum for my particular brand of comp. If you have the original install disk of Windows then great!

It's best to wipe your drive first. I used Killdisk and then reinstalled my XP pro. If you are planning on dual-booting from the same drive you may want to type in Dual Booting in the "search" field and find ways to that suit your install needs. Some people triple boot and some people partition their HDD's before installing OS on their units.

Keep asking questions b/c that's the only way you'll get your sitch worked out :D
