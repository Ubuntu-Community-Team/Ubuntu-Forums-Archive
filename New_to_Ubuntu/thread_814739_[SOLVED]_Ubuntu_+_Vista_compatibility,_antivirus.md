### Post by tamoneya on 2008-06-01
ubuntu can handle windows documents pretty well.  Open Office by default can open .docs and all the other xp formats.  As for OOXML it has some trouble.  You can use online converters or an open office plugin if you have to use OOXML.  But personally I always request people to send me the doc files instead of docx.  

As for antivirus there are no linux virii in the wild so you have very few threats.  I still run clamAV to scan things like email attachements.  The primary purpose is protect my windows using friends from virii.  If someone sends me a infected file and then i send it to someone else they wont be happy.
