### Post by bodhi.zazen on 2008-09-04
Thread moved to a support forum.

There are many how-to's available :

[https://help.ubuntu.com/community/GraphicalInstall](https://help.ubuntu.com/community/GraphicalInstall)

In general :

1. download the apropriate ubuntu desktop iso.
2. burn it to a cd.
3. back up your data.
4. defragment you hard drive.
5. put cd in and reboot.
6. follow the directions on the installer, make sure you understand partitions :

[HowTo: Partitioning Basics - Ubuntu Forums]("http://ubuntuforums.org/showthread.php?&t=282018")
