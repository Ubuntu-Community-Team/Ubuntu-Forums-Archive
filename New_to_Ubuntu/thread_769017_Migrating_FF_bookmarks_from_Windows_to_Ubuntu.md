### Post by Michael.Godawski on 2008-04-26
Click in FF on Bookmarks > Organize Bookmarks then File > Export
and then save the bookmarks.htm file to you usb stick or something similar.

In Ubuntu click in FF on Bookmarks > Organize Bookmarks then File >Import >Import from File navigate to the bookmark.htm.
