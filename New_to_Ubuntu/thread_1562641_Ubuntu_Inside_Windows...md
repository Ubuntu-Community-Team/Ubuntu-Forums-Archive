Here is a wubi guide.
[https://wiki.ubuntu.com/WubiGuide](https://wiki.ubuntu.com/WubiGuide)

To be honest wubi is made to be a try out program to see if you like or can use Ubuntu. People have run it longer term and even upgraded but thats not the design. Your best to just try it if it works for you then dual boot it in the computer. So increasing the partition size is sort of moot unless you want to risk having Ubuntu inside of another OS or all by itself, along side W7.
