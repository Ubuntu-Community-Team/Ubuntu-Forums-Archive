### Post by Scott Deagan on 2012-01-30
I had the exact same problem on 2 desktop PCs and 3 laptops (that use a mix of nVidia and ATI graphics cards). I spent days experimenting and searching for a solution. In the end, I reverted to installing Ubuntu 10.10 (which is a pity, as I want to use the new and improved Ubuntu Software Centre).

Since each case is usually different, it may be worth your while exhausting all suggestions/workarounds others have mentioned on the Internet. If they fail, perhaps look at replacing Unity with Unity 2D or Gnome Shell (or one of the numerous desktops available for Linux). I haven't been able to find a solution to this problem, so am waiting to see if it is fixed in 12.04.
