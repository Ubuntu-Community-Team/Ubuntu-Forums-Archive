### Post by Brushstroke on 2011-09-25
If you were having problems with Ubuntu 11.04, I would recommend you try using 10.04. It's a Long Term Support (LTS) release which means it's not scheduled to be upgraded until April 2013 when 12.04, the next LTS release, comes out. Using this, you'll still have somewhat up-to-date software and you can keep your software up-to-date through [PPAs]("https://launchpad.net/ubuntu/+ppas"). 

Also, to stop your system from calling up distribution upgrades, you can go to System > Administration > Software Sources and in the third tab titled "Updates" the last option should say "Release Upgrade" and you can choose to disable distribution upgrades completely, keep them as normal, or set them to notify you only of LTS releases.
