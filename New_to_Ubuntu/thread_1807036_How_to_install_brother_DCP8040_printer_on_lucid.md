### Post by Cybie257 on 2011-07-18
You can start out here ([http://welcome.solutions.brother.com/bsc/public_s/id/linux/en/index.html](http://welcome.solutions.brother.com/bsc/public_s/id/linux/en/index.html)) for the official Brother method to install your printer in linux. 

I noticed that the driver is supported under Redhat and Debian, Debian noting 32-bit. I had a brother printer that I was able to install a few years back with their methods, but had to 'force' install the drivers into 64bit Ubuntu as they were 32-bit. Ubuntu is 'Debian' based, so use the Debian method. :)

Hope this helps,
