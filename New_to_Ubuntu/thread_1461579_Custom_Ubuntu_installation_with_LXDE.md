### Post by warfacegod on 2010-04-24
The SliTaz .iso is only 30 MB (yes Megabytes) so most of whatever size you get can go to your documents. I've never used an SD card for an install only USB jump drives. I've had plenty of luck with Gigaware and PNY's Attache. I usually run SliTaz from my 4 GB Gigaware.

You might want to get unetbootin for installing to a jump drive (don't know if it will work for an SD card). If you get it, remember to open its properties> permissions tab and mark it as executable.

[http://unetbootin.sourceforge.net/]("http://unetbootin.sourceforge.net/")
