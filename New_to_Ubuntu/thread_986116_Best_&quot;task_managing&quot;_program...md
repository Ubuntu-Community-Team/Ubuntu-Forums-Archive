### Post by jpoRS on 2008-11-18
I just went through this process.  Sunbird is a stand alone calender/task program.  Very intuitive, very good.  Had all of the features I liked about Evolution, PLUS most of the features I lamented Evolution not having.

Lightning, the plugin for Thunderbird, is essentially exactly the same as Sunbird.  If you are already running Thunderbird I suggest this option, if only to minimize running applications.
