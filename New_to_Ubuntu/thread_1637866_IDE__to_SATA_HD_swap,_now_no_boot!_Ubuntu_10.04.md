### Post by sikander3786 on 2010-12-05
The problem lies here as I see.

> 
=> Grub 2 is installed in the MBR of /dev/sda and looks at sector 2048 of the
same hard drive for core.img, but core.img can not be found at this
location.

We would first try to re-install Grub by using the simplest method.

Boot an Ubuntu Live CD and do these commands.

```
sudo mount /dev/sda2 /mnt
```

```
sudo grub-install --root-directory=/mnt/ /dev/sda
```

Note for the second one it is just sda without an integer and not sda1 ;-)

Reboot and try to boot from your hard disk.

And it would also be better if you please edit your above post and type [/code] at the end of your output and [code] at the beginning of it. It'd make it easier to read ;-)
