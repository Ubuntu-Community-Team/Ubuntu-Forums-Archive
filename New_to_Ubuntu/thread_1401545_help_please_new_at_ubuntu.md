### Post by howefield on 2010-02-08
> **juanjo836 said:**
> hi i'm sorry if this post isn't in the right place, i'm currently using windows xp and i want to install ubuntu 9.10, i have windows installed on a partition and al my music and documents in a diferent partition, how should i proceed to replace windows without loosing all my music and stuuf in the other partition??? thanks

First be certain that you want to replace windows, there's no shame in dual booting and keeping both :)

Then back up all the data you would not want to lose if the worst happened. Preferably to a separate media to your operating system disk.

Then boot with the live cd to the desktop, to help make sure that it works, then click the install button on the desktop to start the install.

When you get to the partitioning stage, you can elect to use the whole disk, or just the windows partition, or manually create your partitions. I'd do the the latter and create partitions for / /home and swap.

Before you do any of this though, it might help to watch a couple of the (short) installing videos at screencasts.ubuntu.com The two in particular, "Ubuntu Partitioning" and Ubuntu Dual Boot "Install"
