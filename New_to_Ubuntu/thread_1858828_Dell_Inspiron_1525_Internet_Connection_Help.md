Hi and welcome to the world of openness,

Ok no worries not every one using linux is a techy so as you've stated that you're using a LTS version but the issue is that some times the firmware or in windows terms drivers are little out dated.

Unlike windows one cant just download and install the drivers just like that. So a user who converting from win to lin it's better advised to install the newest version of ubuntu.

Go to your windows and download the newest version from the [Ubuntu]("http://www.ubuntu.com/") web site if you wait you can get the latest one 11.10 this contains mostly up to date firmware for newer hardware.

And also 10.04 do have some issues with wireless and some Ethernet chip sets mostly newer ones.

Hope this help and enjoy your visit in the Linux world.

Oh! and if you want more simpler solution get [Linux Mint]("http://www.linuxmint.com/") this is much easier for a user coming from a windows base after you master linux on this you can move to more advance once.
