### Post by niteshifter on 2008-04-24
Hi,

It depends. Images can show up in email as attachments or "in-lined" (as in in-line with HTML'd content).

First open the email.

If they are "proper" attachments, scroll down to the end of the email. You should see a right-pointing arrow, it's pointing to a button with a down-arrow symbol. Click the button and choose the application you wish to use to view / print the attached photo with.

No arrows and buttons? Then it's in-lined and in-lined can be a pain. If right-clicking on the image gets you no offer to open. save as or print then let's try this trick. With the email opened:
1) Click File (in the email display window, not Evolution's menu).
2) Click Print Preview. This will open a window displaying how the printed version will look.
3) In that (new) window, right click on the image. A drop list should appear and one of the selections should be "Save As".
4) Click Save As. A dialog appears, select a folder to save it in and give the image a name, but no extension.

Now the fun part: Open that folder you specified in step 4.
Right click on the file and click Properties. The file type will be displayed - and so far, for me in Evolution 2.12 it insists on converting the image to png type. This can mean a loss in resolution of the image.
After saving you can close the preview window.

Should that resolution shift be the case for you and it is not acceptable, the only fix I can think of is to email the sender and have them resend the messages(s) as plain text and attach the images, this would make them "proper" attachments. You  would then use the process for "proper" attachments I gave above.
