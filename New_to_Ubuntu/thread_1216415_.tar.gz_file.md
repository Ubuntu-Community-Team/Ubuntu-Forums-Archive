### Post by Abu_Dayya on 2009-07-18
a .tar.gz suffix menas its a compressed archive, similar to .zip or .rar in windows. To open it you need to right click it and select 'Extract Here'. 

But beware that most likely its the source code for whatever you are trying to install, not a prepared package. to install it you need to read the README file after extracting it to see how to install it from source

### Post by balaknair on 2009-07-18
tar.gz is an archive(like .rar, .zip etc) as Abu_Dayya noted.

Installation methods for tar.gz packages may vary, and are usually documented in a Readme file inside the archive.

For Easy Eclipse the instructions can be found here
[http://www.easyeclipse.org/site/help/install-distribution.html#linux](http://www.easyeclipse.org/site/help/install-distribution.html#linux)
```
To install on Linux, just download and extract the tar archive (extension 
.tar.gz) into any folder. The extracted folder contains an executable file 
that can be started from the command line, or by double-clicking on it in 
the Windows Manager.
```

So just extract the .tar.gz file into a folder, open the folder and double-click the executable file in the folder.
