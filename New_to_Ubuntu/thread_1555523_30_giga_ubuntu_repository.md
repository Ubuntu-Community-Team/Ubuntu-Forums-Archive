### Post by mcduck on 2010-08-18
> **Paul820 said:**
> If you absolutely need them, then download. They are only useful if you don't have an internet connection and you download them from a friends house or something. Do you need them?

I doubt *anybody* would need every single piece of software available in Ubuntu's repositories...

It would be pretty hard to even build a machine that would need every driver available, for example... ;)

Copying all that stuff would only make sense if you were about to set up your own repository mirror, for example for a small company/school or other organization with lots of different computers and stuff (and even then you'd probably be better off creating a repository with just the software that s actually needed). Or, of course, if have a decent server with a good Internet connection and want to set up a mirror repository open for public.

If you just want to get software for your own use then you'd be better of simply downloading what you need.
