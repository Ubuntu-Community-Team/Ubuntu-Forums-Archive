### Post by thulefoth on 2010-04-25
What I've done to find things that 'vanish' like that, is go into Synaptic and find the package name.  Right-click on the name to bring up a Dialog with several Tabs across the top.  Go over to the Tab that says Files and open it.  That usually lists a slew of all the files in the package, and shows you the actual directory-path for each of them.  It can kind of be a little 'overload', but it should show you where in the directory the package was installed.

Try that ...
