### Post by wog on 2011-02-22
Try using Synaptic to remove OpenJDK (which seems to be the default) and then install sun-java6-bin and sun-java6-plugin -- and then minecraft will run properly.

### Post by Paqman on 2011-02-24
Minecraft 1.13 actually runs fine with OpenJDK, although the devs do recommend you use the Sun Java. Make sure you've got Java files set to open with Java, not the archive manager (which is the default behaviour).
