### Post by Bucky Ball on 2009-03-28
Xubuntu! Lightweight and designed for lower spec machine. I use it on all my machines and they are not old. It is just fast and I love it. 

The difference (they all sit on Ubuntu kernel) is the desktop manager and look at is this way (version followed by desktop manager it uses):

**Xubuntu (Xfce)** Lightweight
Ubuntu (Gnome) Heavier
Kubuntu (KDE) Heaviest and wouldn't advise.

Once you have Xfce up with Xubuntu, you can add desktop managers and just see what your machine will handle, uninstall if things start cramping up. Incidentally, these are just a few of the available desktop managers (or desktop environments) available. :)

ps: I am suspecting pretty low specs so I would advise getting the 'Alternate' CD rather than the 'Desktop' version. That may crash your system and just not install as it needs some grunt to run its GUI and disco ball! Alternate is a text based installer and can be found with the link to other ISOs on the main download page.

Theoretically, you should be able to use any of these three versions (there are many more: puppy linux, damn small linux, the list goes on ...) with 512mb of ram but I would start with something light and work your way up once you are running. You can install the various desktop managers and choose whatever takes your fancy when you log in.

Let you know how once your installed and ready to roll. Good luck with it. :)
