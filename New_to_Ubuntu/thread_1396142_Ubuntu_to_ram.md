### Post by Tholley on 2010-02-03
You should be able to enter BIOS by gently tapping the Delete key, right as your computer is first booting up. 
 
Once you get into BIOS (it may say CMOS Setup also), find the Advanced BIOS Features, and change your First Boot Device to CDROM, you might change the Second boot Device to HDD-0, just so you won't lose that in case you put a new HDD in.
 
While you are there, go ahead and put the Live CD in the tray, press F10 to save and exit. It will then boot into the Live CD and you should be able to access you computer from and maybe even run a test on your hard drive.

### Post by Tholley on 2010-02-03
OK... revised for Gateway...
 
In order to access the boot options on this portable computer, press the ESC or F10 key while the Gateway screen is displayed during the startup process. This opens a Boot menu with the option to start the computer to any of the available drives or to enter the BIOS Setup Utility. Use the UP ARROW and DOWN ARROW keys to select a menu option, and then press the ENTER key to start to the selected device.
