### Post by Gnawnsense on 2012-10-20
Had the same issue. Try this:

> 
- Open up /etc/pulse/daemon.conf
- Locate the line that included "default-fragment-size-msec = 10"
- Change the 10 to 5
- Save file
- Run command: sudo /etc/init.d/pulseaudio --kill

If it doesn't work, it's easily reversible. But that should work; did for me for the same issue. I originally tried this solution because I didn't want to start purging/removing things from Ubuntu being a newbie. Thankfully it was pain-free and a fast solution.
