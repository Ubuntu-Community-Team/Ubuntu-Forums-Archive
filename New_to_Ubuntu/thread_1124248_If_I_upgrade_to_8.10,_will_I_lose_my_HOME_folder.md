### Post by drs305 on 2009-04-13
If you do an online update your home folder will be preserved. If you do a "clean" install from the cd and have a separate home  partition and you *identify but do do not format it during the install* it will be preserved. 

If you do not have a separate /home folder and do a clean install from CD it will be lost.

### Post by jrothwell97 on 2009-04-13
Short answer: no.

Long answer: no, as long as you upgrade using Update Manager. Don't forget to tell Software Sources to accept all distribution updates, not just long-term-support releases (hint: it's under System/Administration/Software Sources).
