### Post by patrick281981 on 2010-09-12
Hi to switch between X - Screens (Workspace) you need to use CTRL + F1, F2, F3, F4 shortcut (depends how many desktops you have).
Shortcut to switch between TTY (main terminals) CTRL+ALT + F2, F3, F4, F5, F6 or F7 (F7 is X window terminal).

Hope that helps.
