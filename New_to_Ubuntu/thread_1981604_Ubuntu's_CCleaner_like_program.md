### Post by wilee-nilee on 2012-05-17
Closest app would be bleachbit, runs in root and regular user. Be careful and look at what you are removing though it will remove passwords from FF and a email app like thunderbird if you tick the boxes to do so.

You can also run these two commands for a little cleaning

sudo apt-get autoremove

sudo apt-get autoclean

### Post by Ubun2to on 2012-05-17
Registry cleaning-Linux has no registry, so no need to clean it.
Cache, cookies, history, etc.-you can clear stuff like that from the browsers themselves.
Uninstalling programs-Ubuntu Software Center, or you could find some other program.
Cleaning up shortcuts, unused file extensions, and things of that nature-not sure if that is needed.

### Post by Frogs Hair on 2012-05-17
Open Ubuntu Tweak and select janitor . Check the box/s in the left pane and a selection of items to be cleaned will appear . Check the boxes on the right and select clean. You prompted for a password and you will be asked twice if you want to remove unused kernels.
