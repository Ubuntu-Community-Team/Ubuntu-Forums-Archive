### Post by 3rdalbum on 2009-08-08
If you absolutely need (?) the absolute latest point version of AWN, you should either look for a Personal Package Archive (PPA) that has this version, or compile it from source yourself.

A PPA is just a small repository - do a search for Avant Window Navigator PPA on Google and you might find one.

If there isn't one, then look on the AWN website for a Debian/Ubuntu package. If there isn't one, then you should download the source code and compile it yourself.

If compiling scares you or if you have trouble compiling it, then stick with the repository version.
