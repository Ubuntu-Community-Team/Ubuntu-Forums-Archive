### Post by Habeouscorpus on 2011-01-12
To access Windows files through *buntu:

Is there something called "File System" in the menu? If you click that, then go to "host," your files will be there.

To access *buntu files in the situation you described: 

Go to "File System" in the window, then click "home," then your username, and that's where all your stuff should be stored.

### Post by Habeouscorpus on 2011-01-13
You could create a launcher.

(I'm not sure if this is the same in XUbuntu, but:) 

Right Click on the Desktop, "create Launcher." Then select Location from the menu. Navigate to your Windows stuff, then select it. (Sometimes you have to select a file, then go back and take off the file in the file path, I've found.) Type a name, and create! Boom. Insta shortcut.
