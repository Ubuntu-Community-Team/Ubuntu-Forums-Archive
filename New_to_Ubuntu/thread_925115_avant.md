### Post by howefield on 2008-09-20
Go to System > Preferences > Sessions and click on the add button.

In the name field type Avant

In the Command field type avant-window-navigator

In the comment field type whatever you want, leave it blank if you prefer, it doesn't matter.

This should start avant dock when you log into your computer.

### Post by nothingspecial on 2008-09-20
To make a program start when you log in 
System > Preferences > Sessions
Click the startup programs tab
Click add
Put this in the middle box "Command"

avant-window-navigator

Put whatever you want in the other 2

This will work for any app just put the command to start the program in that middle box eg

pidgin 
or whatever you like
