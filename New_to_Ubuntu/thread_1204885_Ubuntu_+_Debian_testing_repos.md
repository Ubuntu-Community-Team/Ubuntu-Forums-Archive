### Post by snowpine on 2009-07-05
Debian Testing aka Squeeze is a wonderful rolling release distro, however, it currently uses OpenOffice 3.0.1, Pidgin 2.5.6, and doesn't use Firefox at all (Iceweasel 3.0.9). I do not think it is the solution you are looking for. (Nor do I think mixing Ubuntu and Debian repos is a good idea.)

If you want "Ubuntu with newer applications" I recommend giving Karmic Koala Alpha a try. Or, the closest distro to "rolling release Ubuntu" imho is sidux. Good luck!

### Post by shae on 2009-07-05
In my experience, mixing stable and testing is problematic enough in debian.  Mixing Ubuntu and debian testing is just a mess waiting to happen.  If you want to experience a true Rolling Release distribution, then the best choice is Arch Linux, but it comes at a price.  It does not feature all those "easy" ways of doing things, you have to manually set some things up, installing is more manual, etc.

But that is where you want to look if you want a Rolling Release Distribution.
