### Post by tomBorgia on 2009-10-19
Hi,

1 - Yes this looks like it will be the same as when you first got the computer.

2 - You'll have to change the boot device order in your BIOS to boot from CD/DVD drive first before HDD.

You'll need to backup all your data as either of these processes will wipe the hard disk.
