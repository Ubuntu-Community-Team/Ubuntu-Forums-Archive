### Post by iMattcotv on 2010-05-25
I had my screen resolution go from high res to something like 640x800.

It happened because I tried using Wine to run a game that wasnt set up using Wine, also it happened because I installed a "restricted driver" for ATI..

So what I did was go into System > Administration > Hardware Drivers and I deactivated the gfx driver that caused the problem, rebooted and everything went back to normal.
