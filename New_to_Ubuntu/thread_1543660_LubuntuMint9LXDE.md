### Post by Kixtosh on 2010-08-01
Well, on the ancient machines listed in my signature, less powerful even that what you have, I have tried Puppy 431 and 501 (LuPu), as well as Ubuntu and Xubuntu 9.10 or 10.04, and even Lubuntu 10.04. I also tried SliTaz.
 
**Puppy** is nice. It has everything basic that you will need and works without a full installation on your hard drive (working only from a bootable CD drive). I did also try a hard drive installation, but later removed it since it was actually slower to start up than other options. Puppy is an excellent choice, in my opinion, to try a simply Linux system without installing anything. It'll mean that you have a working system always, no matter what else might go wrong with any other installation you might try.
 
**Lubuntu** 10.04 did work, and had similar results to what is stated below.
 
**Xubuntu 10.04** is slightly better than a full Ubuntu installation on older machines with slower processors and less RAM (like mine) and I even had fewer issues with it. In fact, there really wasn't anything to fix. Best of all was when I added the LXDE desktop AND Chromium web browser instead of the default installation. I used the "Ubuntu Software Center" to do this (very easy to use for a new user) from the "Preferences" menu. These old computosaurs now start up in less than one minute, from pushing the power button to establishing an active internet connection on a clean desktop. Chromium takes less than ten seconds to load up ready for browsing. Shutting down is also very fast.
 
**SliTaz**, like I said, would not work properly, so I can't say how fast or slow, good or bad it might be.
 
So from my experience:
 
- Easiest to start using is Puppy.
- My preferred choice is Xubuntu with LXDE desktop and Chromium browser.
- Lubuntu is also a good option, with Chromium browser.
 
After start up, on a blank desktop, Xubuntu with LXDE is using between 65 and 75MB of RAM, and 0 swap space. There is no apparent advantage to using Puppy in my opinion, and this configuration actually boots up almost twice as fast as Puppy. The Firefox browser is far too heavy and slow for machines with limited resources such as mine.
 
Lubuntu uses only slightly less RAM, and again, like Puppy, there does not seem to be any speed of navigation advantage when compared to Xubuntu with LXDE. I don't use it simply because I prefer Xubuntu with LXDE.
 
In any case, any of these three options will work on very old, very slow computers (256MB of RAM, 400MHz CPU).
