### Post by LinuxFan999 on 2011-12-28
Type "chrome://plugins" in the address bar to access the list of plugins, then find the Flash plugin. After finding it, disable it, then close and reopen Chrome. Next, access the 
"chrome://plugins" page again and re-enable Flash. Then close and reopen Chrome again, and go to a site that uses Flash.
