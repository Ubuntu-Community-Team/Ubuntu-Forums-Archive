### Post by hogwartsnigel on 2008-07-25
As the other posts have said it really does depend on what you are looking at installing. Aysiu's comment was a little dismissive and technically correct but pointless, you choose to use Ubuntus options for downloading and installing.

By far the easiest way to install software is through synaptic or apt-get, either from the terminal or via the gui version that you can access from Administration-install software. Follow the posted guides at enabling repositories for Ubuntu's apt-get to search and list from to increase software option. Medibuntu is almost a must for most people but there are several others.

Some applications can be downloaded in installable packages from the internet, this is a little more hands on unless they option to be opned with synaptic. Often you have to follow accompanying guides to install the downloaded file from the command line. Again the guides will help.

What are you trying to install?
