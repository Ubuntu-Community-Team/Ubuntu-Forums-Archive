### Post by NightwishFan on 2009-06-13
Install compizconfig-settings-manager.

In CCSM, go to general options, and the last tab is desktops. Set it to 4 horizontal, 1 vertical.

Enjoy!

### Post by lswb on 2009-06-13
Just right-click on the icon in the bottom display, select Preferences, and change in the dialog that then pops up. Most panel icons have a right-click context menu where preferences and settings can be changed.
