### Post by aktiwers on 2008-07-12
Welcome!

If you wanna try out the 3d effects you could install ubuntu to your Hard drive. Ubuntu will make it a dualboot, letting you choose which OS to boot to when you turn on the computer.

Just remember NOT to pick "use entire disc" in the installation. Make sure you understand and know what you are doing during the partitioning part of the installation, and you should be fine. 

I have seen people who did not spend the 20 minutes it takes (max) about this part, and they lost there Windows install and files. This wont happen if you learn this step, which can be done fast using this link:
[http://www.psychocats.net/ubuntu/installing#partitioning](http://www.psychocats.net/ubuntu/installing#partitioning)

Actually, even though the rest of the installation is "common sense", this (almost same) link is good for beginners:
[http://www.psychocats.net/ubuntu/installing](http://www.psychocats.net/ubuntu/installing)

Welcome again! :)
