### Post by 2F4U on 2013-05-25
Why don't you try Lubuntu first as a liveCD/liveUSB and see if you like it. It is pretty different from Ubuntu beginning with the look and feel. If you like it, you can install either lubuntu-desktop on top of Ubuntu. This gives you everything that is in Lubuntu, but applications installed by Ubuntu remain intact. On the login screen you will be able to select which desktop to start. However, as you already said, this defeats the idea of a light desktop environment since Ubuntu is still there (and it is not recommended to remove it). But during the time when you are still deciding whether you like it or not, this may be a good way to go forward.

### Post by Impavidus on 2013-05-25
You can install lubuntu-desktop in your current ubuntu installation and select lubuntu at login. The heavy unity interface will still be installed, but when you don't run it, it will do little more than take up some disk space.Having two desktop environments generally works, but you may sometimes encounter compatibility issues. It's also possible to completely convert your installation to lubuntu: [http://www.psychocats.net/ubuntu/purelubuntu](http://www.psychocats.net/ubuntu/purelubuntu)
