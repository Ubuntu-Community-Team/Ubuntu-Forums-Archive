### Post by theharshone on 2008-12-13
try kopete. It has seperate windows for every new user, it notifies if you have a message, but I don't think that it notifies if a new user logs on. The downside to kopete, is taht you you have to install KDE4, which will take up a lot of space on your HD (there will be no downside if you like KDE4 and decide to use that instead :D) And, on my old 500 Mhz computer, I found Linux much much faster. (I used Kde3) I have concluded Linux is not for those people with closed minds and those who don't like to read (mainly errors, which pretty easy to fix IF you acutally read the errors. Linux errors are more descripitive than the usual windows errors, which usually say "Error 34!" without saying what error 34 is.) I recommend you use a Desktop envronment that does not use as many resources as Gnome or Kde. I always liked Enlightenment. I've heard good things about XFCE as well.
