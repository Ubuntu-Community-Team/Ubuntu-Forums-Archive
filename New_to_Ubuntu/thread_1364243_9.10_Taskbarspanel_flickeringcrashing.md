So it closes when it errors out before I can see that happens.  Is there some way I can open a terminal without using the panel? Then I would be able to la least show you whats going bad.

Is there a way to simply reset the panel to a default?

You could hit ALT + F2, it will open a launcher dialog. then type gnome-terminal.

You could create a launcher on the desktop with the same command.

try typing this in the terminal when you get it open. ```
rm -r ~/.gconf/apps/panel
``` 

This will delete any preferences for your panels, which will be rebuilt when you logout/login.

if that doesnt work, see if you could run the panel from the terminal like i said, and see if it says any errors
