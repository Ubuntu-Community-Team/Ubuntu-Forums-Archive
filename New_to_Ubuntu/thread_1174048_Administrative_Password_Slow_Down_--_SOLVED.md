### Post by insub2 on 2009-05-30
Whenever I'm asked for my administrative password (for Synaptic, updates, gksudo, etc.)  my computer seems to stop for a few minutes.

...while I was writing this up, it occurred to me that this started happening after I played with the Window Manager Tweaks.  So I unchecked the "Show shadows under pupup windows" and all is fine.

Thought I'd post anyway in case anyone else is experiencing this problem.  Or if you have a solution that will allow me those sexy shadows.
