### Post by waynefoutz on 2010-04-06
If you are having problems running do to lack of OpenGL support, which is the problem I suffer with, I've found that google maps is nothing more than a flash version of google earth. You can zoom out to the globe, and zoom in down to the streetview photos. It's just google earth with less features in your browser window.

[http://maps.google.com/]("http://maps.google.com/") Nothing to install here, so long as you have flash working.

If you are having problems installing google earth from the .BIN file from the website, here's how you do it:
Right click on the .bin file, choose, "open with other application"
in the command line area, type "sh"

Hmm?  Google Earth is extremely simple to install from [http://earth.google.com/intl/en/thanks.html#os=linux](http://earth.google.com/intl/en/thanks.html#os=linux)  

Download the file there and run it with

  **sh ./GoogleEarthLinux.bin  ** 

in terminal.  Installed!
