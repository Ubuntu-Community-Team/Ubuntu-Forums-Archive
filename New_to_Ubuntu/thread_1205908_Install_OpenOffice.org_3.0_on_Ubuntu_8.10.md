### Post by anand_chauhan on 2009-07-06
I guess everybody knows about open office suit. It’s a light and well functional free software that replaces other bulky counterparts. Well, I would like to explain about Open office
OpenOffice.org (OO.o or OOo), commonly known simply as OpenOffice, is an office application suite available for a number of different computer OS. It is distributed as freeware. It supports the ISO/IEC standard OpenDocument Format (ODF) for data interchange as its default file format, as well as Microsoft Office formats among others. As of March 2009, OpenOffice supports over 75+ languages. 



[COLOR=#3333FF]**STEP 1 - Add the OpenOffice.org 3 repositories** [/COLOR]
  
Navigate to  System -> Administration -> Software Sources...
  
[[IMG]http://lh4.ggpht.com/_NNCAqNFu7m0/SlJCgEoJs9I/AAAAAAAAASk/aHeAFkQjdn4/1_thumb.png?imgmax=800[/IMG]]("http://lh4.ggpht.com/_NNCAqNFu7m0/SlJCWUA0hRI/AAAAAAAAASg/0AOXKg8mgV4/s1600-h/1%5B2%5D.png")
Navigate to  the second tab, "Third-Party Software," click on the "Add" button, and paste the line below... 
  
deb *_**[http://ppa.launchpad.net/openoffice-pkgs/ubuntu](http://ppa.launchpad.net/openoffice-pkgs/ubuntu)**_** *intrepid main 
  
[[IMG]http://lh5.ggpht.com/_NNCAqNFu7m0/SlJCngzEGYI/AAAAAAAAASs/9at6DN6e4n8/2_thumb.png?imgmax=800[/IMG]]("http://lh3.ggpht.com/_NNCAqNFu7m0/SlJClRpTcEI/AAAAAAAAASo/wp6ZR5OBauA/s1600-h/2%5B2%5D.png") 
  
Get key files [**HERE**]("http://anandcatlysis.blogspot.com/2009/07/install-openofficeorg-30-on-ubuntu-810.html") (source) Navigate to the fourth tab, "Authentication", click the "Import Key File" button, navigate to to the location were you've just saved the key file (File System/home/YOURUSERNAME/Desktop) and double click it. You will immediately see a new entry called "247D1CFF 2009-01-21 Launchpad PPA for OpenOffice.org Scribblers". 
  
[[IMG]http://lh3.ggpht.com/_NNCAqNFu7m0/SlJCttI6f-I/AAAAAAAAAS0/0RLrLSzU65Y/3_thumb.png?imgmax=800[/IMG]]("http://lh6.ggpht.com/_NNCAqNFu7m0/SlJCqNUrzwI/AAAAAAAAASw/My9VLSkwB0w/s1600-h/3%5B2%5D.png") 
  
Now, click the “Close” button, then the “Reload” one and wait for the application to termnate! 
  
[[IMG]http://lh3.ggpht.com/_NNCAqNFu7m0/SlJC6LLoj0I/AAAAAAAAAS8/kz_kxt9FYEU/4_thumb.png?imgmax=800[/IMG]]("http://lh4.ggpht.com/_NNCAqNFu7m0/SlJC4oq_ziI/AAAAAAAAAS4/NZHdZFgJx1U/s1600-h/4%5B2%5D.png") 
  
[COLOR=#3333FF][B]
[/B][/COLOR]
[COLOR=#3333FF]**STEP 2 - Update OpenOffice to version 3.0.0 **[/COLOR]
  
When the Software Sources window will close itself, the update icon will appear in the system tray... 
  
[[IMG]http://lh3.ggpht.com/_NNCAqNFu7m0/SlJC-CRosjI/AAAAAAAAATE/KNu9WBUHUdU/5_thumb.png?imgmax=800[/IMG]]("http://lh5.ggpht.com/_NNCAqNFu7m0/SlJC8h7c_PI/AAAAAAAAATA/OnWhlklj-OA/s1600-h/5%5B2%5D.png") 
  
Click on it and update your system. Your open source office suite will be up-to-date from now on. Take a look below for some shots of OpenOffice.org 3.0 in Ubuntu 8.10 .
