### Post by aeiah on 2009-03-23
you'd probably be alright with ubuntu and not need to go with xubuntu, but some people prefer xubuntu and the desktop environment it ships with (xfce instead of gnome). both desktop environments are pretty similar to eachother though and applications tend to look the same in either.

commands you issue in a terminal will be the same for both, with the odd exception. ie, tutorials may tell you to edit a text file by issuing the command

gedit /some/textfile

wheras xubuntu's default text editor isnt gedit, it's mousepad. you get the gist.

### Post by halitech on 2009-03-23
I've used both and about the only thing you will need to change is gedit for mousepad unless you install gedit (easily done). Things that deal with gnome specific won't work (like gconf) but make sure you explain you are using XFCE when you post a problem to make sure people know.

Honestly, you probably won't see a lot of difference between Xubuntu and Ubuntu as Xubuntu uses a lot of libs and apps from Ubuntu. If you want to see a big difference, try Ubuntu then install Debian 5.0 with XFCE and you will see a big improvement.
