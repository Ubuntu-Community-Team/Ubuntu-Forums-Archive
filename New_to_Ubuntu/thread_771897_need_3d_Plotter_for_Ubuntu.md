### Post by Volt9000 on 2009-06-01
Since nobody else volunteered the information yet, to install KAlgebra go to a terminal and type

```

sudo apt-get install kalgebra

```

This will also install all the dependencies. Be warned that if you're running Gnome and don't have the QT libs installed this will be a big download.
