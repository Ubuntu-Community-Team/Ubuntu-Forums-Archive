### Post by Pinon on 2010-03-22

My problem is using Norwegian special letters through Wine. Then I am unable to type Norwegian letters, such as æ ø and å. (For example inside Microsoft Office, and all other applications running through Wine.) This is a problem only affecting Wine. When I use Linux programs, these letters works just fine.

How can I set up wine, so that it recognizes my Norwegian keyboard. I want it to recognize when I for example type the letter "ø", so that letter is inserted correctly into the program I use.

### Post by qyot27 on 2010-03-22
Maybe not the sort of solution you're looking for, but there's a panel applet for GNOME (if you use GNOME) that allows you to select the particular characters you want - click the letter, and then paste into the document.  It's how I use ö when I need them while using Ubuntu.  But of course I'm using a US keyboard, so I don't have any of those characters on the keyboard itself.

Right-click on the panel, choose 'Add to Panel', and the applet should be in the list.  Can't remember the specific name, but the word 'Character' is in it IIRC.

I am not a Norwegian speaker, but I often use the Norwegian keyboard layout because I like it more :)
