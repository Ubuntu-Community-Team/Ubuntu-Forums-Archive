### Post by Pro-reason on 2008-09-13
At some point, you are going to have to face up to the fact that you are running Ubuntu, not Windows.

KMPlayer for Windows is nothing particularly special.  Players such as Totem, VLC, MPlayer, Dragon, SMPlayer and KMPlayer for Linux are all perfectly good.  Just give them a chance.  You will become accustomed to them.

If you really want to run KMPlayer for Windows, then either stay on Windows or run the KMPlayer installer through Wine to see if it works.  There is a Wine forum here that you can go to for further advice.  (In fact, this thread should probably be moved there.)
