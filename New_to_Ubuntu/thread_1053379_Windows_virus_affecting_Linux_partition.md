### Post by emarkd on 2009-01-28
Viruses are just software, so they have to be written for the targeted OS.  Windows viruses don't run in Linux any better than Windows apps do.  So if you download the virus in Windows while the shared partition is mounted, it could effect the data on the drive if that's what its designed to do.  However, the virus wouldn't be able to run under Linux.

### Post by kk0sse54 on 2009-01-28
It wouldn't be able to "infect" the linux partition since there are very few viruses that can actually run in linux. As for your data as long as it isn't mounted it shouldn't effect it unless the virus has damaged the entire hard drive in some way.
