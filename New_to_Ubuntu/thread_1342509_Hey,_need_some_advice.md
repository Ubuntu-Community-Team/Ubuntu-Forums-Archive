### Post by jbrown96 on 2009-11-30
I think Python is a fantastic language to learn first. The syntax is very good, and Python encourages your to write readable code, which instills good practices for future work. 

C/C++ are more or less mandatory for programming. They are very popular, reasonably simple, and all programmers are expected to have an understanding of them. I wouldn't start with these, but you should look at familiarizing yourself with them. The C handbook/reference book is *The C Programming Language* by Kerningham and Ritchie. It's called K&R or sometimes "The White Bible." C is a simple language; you don't need a huge book. 

Java is another good language to learn. It's very widely used in commercial software. The syntax is similar to C++. I don't recommend learning this until you have some introductory programming experience (Python) and some intermediate work with a lower-level language (C or C++).

Ubuntu is a "Live" distribution. When you boot from the CD or flash drive, you boot into a fully-functional Ubuntu session. You can see if everything works properly before you install. There are some hardware that cannot be properly tested. Graphics cards and some wifi cards need to have special firmware installed, which needs a reboot. However, if you reboot the LiveCD, then you lose all the changes. You should be able to test almost everything.
