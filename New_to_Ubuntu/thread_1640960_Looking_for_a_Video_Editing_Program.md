### Post by Ahunt on 2010-12-09
That's how it works - crowd sourcing!

I have had good results with Avidemux and Kino for basic editing on a lower spec desktop (both are in the repositories). Cinelerra is definitely for advanced users. PiTiVi comes with Ubuntu by default and is very easy to use, but requires a high-spec hardware (dual-core processor, lots of RAM) and is prone to crashing so save often if you are using it.
