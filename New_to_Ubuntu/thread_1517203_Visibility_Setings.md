### Post by ibuclaw on 2010-06-24
You should see "Assistive Technologies" in one of the System menus. If not, Press **Alt+F2** and type in:
```
gnome-control-center
```
Once you spot it, open it, and enable it. Will require logout/login iirc.

For font resolution, go to System->Preferences->Appearance
Select the 'Font' tab, should be third across.
Select the 'Details' button, in the bottom right hand corner.
Increase the 'Dots per inch' number from 96 to 120, or greater.
