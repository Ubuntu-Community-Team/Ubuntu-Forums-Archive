### Post by ugm6hr on 2008-07-17
> As I have tried Ubuntu before and found it an awful experience, 

What are you expecting to happen this time?  Did you try a previous Ubuntu version before, or Hardy?

If you are uncertain about how to re-install with a revovery DVD, I suspect that trying to install an OS from a generic CD is going to be a bit tricky (and perhaps inadvisable).

I'm not try to warn you off Ubuntu, cos I think it is fantastic.  But I suspect that you may be frustrated again.

My suggestion - if you want to try again - try the LiveCD out (without installing) for a little while, rather than just diving in.  And do plenty of reading before installation.  Or get someone who knows what they are doing to install it for you.
