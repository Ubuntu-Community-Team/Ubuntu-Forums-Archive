### Post by stinkeye on 2013-12-16
In unity, install an indicator.

Install indicator-sysmonitor from [http://www.noobslab.com/2013/04/panel-indicators-collection-for-ubuntu.html](http://www.noobslab.com/2013/04/panel-indicators-collection-for-ubuntu.html)
Scroll down to "**14: System Monitor Indicator**"

In preferences, show in the customize output...
```
{net}
```
