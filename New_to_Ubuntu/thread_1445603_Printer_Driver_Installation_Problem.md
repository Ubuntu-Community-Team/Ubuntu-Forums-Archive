### Post by rinkydink on 2010-09-05
From what I can tell, your Dell AIO 922 printer isn't supported.

People have managed to print documents from a Virtual Machine ([VirtualBox]("http://www.virtualbox.org/")) using the Windows driver. This is quite a silly solution though, you have to install Windows in a virtual harddisk.

Epson printers are usually the safest bet. I'm sorry to be the bearer of bad news.

### Post by walt.smith1960 on 2010-09-06
Dell printers are often relabeled Lexmark printers.  My understanding is that "business class" Lexmark printers often have Linux support, "consumer class" (cheaper) printers usually don't. HP printers seem to be well supported.  Brother seems to care about Linux support. I have 2 Brothers and both work correctly with Linux.
