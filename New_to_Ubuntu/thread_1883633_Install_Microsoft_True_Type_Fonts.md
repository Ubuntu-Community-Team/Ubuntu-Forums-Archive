### Post by Jacobonbuntu on 2011-11-19
I don't know if you still have the fonts installed on a windows machine? If so, you can install fonts by creating a folder in your personal folder, name it ".fonts", and just copy the windows fonts from your windows machine (C>Windows>Fonts) to this folder.

*the folder ".fonts" is invisible from the moment you name it, press ctrl+h to make it visible...
