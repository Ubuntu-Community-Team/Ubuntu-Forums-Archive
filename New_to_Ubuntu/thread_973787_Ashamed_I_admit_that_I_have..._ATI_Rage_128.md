### Post by Peter09 on 2008-11-07
OK - here is a bit of experimentation ....

do the following command in a terminal

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.saved
```

then

```
gksudo gedit /etc/X11/xorg.conf
```

in the section called Device enter the line

```
Driver    "fglrx"
```

save the file and exit.

Now do a <CTRL><ALT><Backspace> which will take you back to login.
Try logging in a see if things are better.

If not try the same edit process but replace fglrx by r128

If at anytime you cannot get a GUI do the xfix procedure. IF that fails do the following command at the command prompt.

cp /etc/X11/xorg.saved /etc/X11/xorg.conf
