### Post by TheFu on 2018-06-06
Yes. It is a crappy old laptop. [https://www.cpubenchmark.net/cpu.php?cpu=Intel+Atom+N455+%40+1.66GHz&id=619](https://www.cpubenchmark.net/cpu.php?cpu=Intel+Atom+N455+%40+1.66GHz&id=619) 280 passmarks is pretty bad.
For any hope of a usable system, start with Lubuntu.  The default Ubuntu with Gnome or Unity will always be slow on that machine.  A used 7 yr old $75 laptop with a C2D CPU would be 5x-10x faster.  I have a $99 computer from 2012 (that was the price back then) with an AMD-E350 APU that is 2x faster.

It is fast enough to be a headless server, however.  That wouldn't have a GUI which would be hard for someone new to Unix stuff.
