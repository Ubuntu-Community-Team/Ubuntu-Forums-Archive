### Post by k3lt01 on 2009-01-10
Go to the OpenOffice website download the appropriate "tar.gz" file it will look something like "filename.deb.tar.gz". Uninstall your current OOo2.4 version. go to your terminal and cd into the OOo3 folder you have downloaded and unpacked. Then type.

```
sudo dpkg -i *.deb
```

after you have done that go into the Desktop Intergration folder and click on the .deb file in that. once all that is done you should be sweet.

[Here's a link]("http://ubuntuforums.org/showthread.php?t=594560") that may be of interest.
