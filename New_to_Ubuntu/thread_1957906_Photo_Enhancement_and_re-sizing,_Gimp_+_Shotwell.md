### Post by souravc83 on 2012-04-13
Use "Pinta". Its there in the Ubuntu Software Center.
For automatic enhancement, use Adjustments->Auto Level in Pinta. And sure it can do cropping and resizing. Its like Gimp or Photoshop, but much easier to use.

I don't use Shotwell. It doesn't have too many features I think. It lets you crop though, but I don't think it lets you resize.

### Post by tea for one on 2012-04-13
I've found Fotoxx to be a useful application for simple photo editing.

My photos generally need straightening and it includes this feature as well as the usual editing functions.

Also, Fotoxx will work with your existing folder system rather than trying to re-organise them on import.
