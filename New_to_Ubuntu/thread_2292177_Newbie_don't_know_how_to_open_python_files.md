### Post by Geoffrey_Arndt on 2015-08-25
What file?   What are you actually trying to do ("opening" doesn't really mean anything distinct & clear - - it could be "running" a program, "editing" a python script/text file, etc. etc.).  

If you installed a python program from the Ubuntu Software Center AND if it is a program with a "gui" interface, it will either show-up in your launcher or you can manually drag it there from the dash-search screen.

If the program does NOT have a gui interface, you have to enter some commands in the "Terminal" window (but you should check the download directory for a read-me file first).
