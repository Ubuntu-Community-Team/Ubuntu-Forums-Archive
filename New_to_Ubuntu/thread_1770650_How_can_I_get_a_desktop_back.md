### Post by keithunder on 2011-05-29
I fixed my own problem

I right clicked to create a folder on the desktop

I selected the new folder and right clicked.

I selected Ubuntu Software center to open it with

I am now back to Unity

Hooray now my computer is just buggy and awkward to use which I suppose is better than unusable. 

I probably won't use it every night now as it keeps waking me up when it freezes. I will update it regularly in the hope than sanity will prevail and this crappy unity gets binned!

Why not download 10.10 and install it alongside Windows as a dual boot and then delete the wubi installation in Windows? That way you can get back to sanity with 10.10 and the dual boot generally works better than the wubi which is inside windows.
