Ok, appologies if this seems like a silly question, but is there a way to have Xubuntu boot up with the num-lock key on by default?  I notice that my number keypad always defaults to the direction keys (off) on my keyboard in Xubuntu even though I switch it to number mode in each session, each time I boot.  

Thank you.

---

### Post by doas777 on 2009-01-18
I use numlockX to do this. you can install it with:
```

sudo apt-get install numlockx

```

then set it to load with the X like in [this]("https://help.ubuntu.com/community/NumLock") tutorial.

good luck,
franklin
