### Post by RomeReactor on 2008-06-10
Hi. Installing Compiz depends on which version of Ubuntu you have; to find out, go to 'System->About Ubuntu'.

Hardy Heron (8.04) is the current version; previous versions were Gutsy Gibbon (7.10), Feisty Fawn (7.04), and others. The first number in the version is the year of release, followed by the month. For example, Hardy Heron (8.04) was released this year (200**8**), in April (04).

If you have Hardy or Gutsy, Compiz is already installed. You enable it by going to 'System->Preferences->Appearance'. In Feisty, you can install it by going to 'System->Administration->Synaptic', and searching for **compiz** there. Once you find the package, double click on the checkbox to mark it for installation, and press 'Apply'.

### Post by RomeReactor on 2008-06-10
I don't think it would run; maybe just the icons were carried over to your Ubuntu desktop. You probably need to install Oblivion in Ubuntu ( I imagine BfME2 is a link to a Battlefield game). You can install Wine by going to 'System->Administration->Synaptic', then search for 'wine', double click the checkbox to mark it, and press 'Apply'. Once Wine is installed, you can right-click on the Setup.exe (or the correct name of the installer) and choose 'Open with wine', which will then proceed with the installation.

We have a section of the forums for running [windows games in Ubuntu through Wine]("http://ubuntuforums.org/forumdisplay.php?f=313").

As for the nVidia drivers, if you install Hardy you really don't need to edit the text file. You can choose the driver in a GUI application. I don't remember if this is available in Feisty, but try going to 'System->Administration->Restricted Drivers' and enable your nVidia card there.
