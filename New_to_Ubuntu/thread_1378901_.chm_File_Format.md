### Post by greenco on 2010-01-12
Here is a text file that I use to convert the "CHM" files.
*************************************************************************

CD to folder that contains the CHM file.
Remove all spaces from file names!!!!
In this example the chm file was in the Downloads folder.


coleman@coleman-desktop:~/Downloads$ chm2pdf --webpage DB_Design.chm DB_Design.pdf

NOTE 1 : In the above line, "DB_Design.chm" is the name of the file that you want to change into a PDF file. The "DB_Design.pdf" is the name you want to save the file as.

NOTE 2 : Either --book  or  --webpage must be used. One has to be present.
--wepage produces an unstructued format
--book produces a structured format

***********************************************************************

Hope it helps!
