### Post by collinp on 2008-07-20
Hmm, search in synaptic for alsa-base, if it is not installed, install it. If it is already installed, right click in the green box and click "Reinstall". If that does not work, there is another package called libesd-alsa0, check to make sure it is installed, and if it is not, install it. If it is, do as above to reinstall.
