### Post by lukeanders70 on 2011-05-02
you could always install a "dock" from the Ubuntu Software Center, I recommend one called "Docky" (it has an icon of an anchor). However, the unity launcher does specify what programs are open with an arrow and which is the active program with an arrow on the other side. You can also press the icon on the unity launcher to have all the windows of that program show on the screen (ex. if you have two Firefox windows open and you press the Firefox icon they will each fill up half of the screen). You can also make use of the workspaces by placing different windows in each and quickly switch by pressing "Ctrl + Alt + arrow key. I know this doesn't exactly answer your question but I hope it helps.
