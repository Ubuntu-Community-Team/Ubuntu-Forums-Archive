Just thought I would post something 'good'.

Windows 7 64 bit (Professional).
WUBI install Ubuntu 64 bit.
Beta install Adobe Flash Player 64-bit
Firefox 64-bit
Wireless home WIFI

Install went without a hitch.  Only been looking at it all for about an hour, but it looks 'great' and there are no problems so far.  Was slightly confusing on getting 'net connected but after a few 'wonder if this is it' clicks, got it working.  Just wish I could remember what the clicks actualy were.  Clicked on the magnifier button with the plus sign (Applications), managed to get it to SHOW ALL, found NETWORK CONNECTIONS.  Clicked on WIRELESS tab but had no idea how to fill in the blanks.  Noticed 'broadcast' icon at the top of the window .. very top.  Clicked on it and provided 'wep key'(?) and boom, connected.

Overall, went great .. and I like 11.04.  It is 'slick' .. and I mean that in a nice way.
