### Post by ibuclaw on 2008-06-07
What sort of file are you viewing?
With specific types of files, the colour patterns of them change (ie: A file ending with the suffix ".sh" will be automatically coloured in for easy reading. Whereas a ".txt" file will be in just plain black and white.

Open up "**Edit>Preferences**" in the tab menus and go into the "**Font & Colours**" tab.

The font should be set to "**Use the system fixed width font**" and the Colour Scheme set to "**Classic**", unless you changed it otherwise.
