### Post by Dennis N on 2018-11-30
You can make the cursor bigger and easily visible but you need a resizable cursor theme. I installed and used **DMZ-cursor-theme** package (not installed by default in Lubuntu 18.10). 

If you just install it and select it in the cursor theme dialog, you only get the small size (24 px). There is no option in any dialog to make it larger, but it can still be done. I changed it to use its 48 px size and it's definitely easily seen. 32 px is also a possible size should 48 px prove too big. 

Is this Lubuntu 18.04 or Lubuntu 18.10? Methods are entirely different in how you fix this.
