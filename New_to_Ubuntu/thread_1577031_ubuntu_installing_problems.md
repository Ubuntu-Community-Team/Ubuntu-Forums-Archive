### Post by Paqman on 2010-09-18
Check that your BIOS is set to boot from CD. How to do this depends on exactly what BIOS your machine has, but look out for options when it first starts to boot for choosing the boot device, or for getting into the BIOS settings. Usually you'll have to hit a button like Del, F12, F2, etc.

### Post by scorchgeek on 2010-09-18
Did you expand the ISO image onto the CD using an image-burning program, or did you just burn the file to the disk? Just burning an ISO file onto a data disk will not produce a usable CD.

And, like others have said, check your BIOS settings, it may not be trying to boot from the CD.
