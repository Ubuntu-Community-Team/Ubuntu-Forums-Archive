### Post by Elfy on 2011-01-18
Once installed, you need to create a symlink to your music 

```
sudo ln -s /path/tomusic/ /usr/share/musica/music
```

Then open a browser and

[http://localhost/musica/](http://localhost/musica/)

Your music should be there then.

After that I guess play about.

Not finding it very useful - some albums it plays, others it refuse to do anything, others it decides it wants to save the m3u file with.
