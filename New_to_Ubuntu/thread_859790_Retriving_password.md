### Post by mevets on 2008-07-14
If you use Firefox and think it should have stored your password then go to Edit > Preferences > Security > Saved Passwords... > Show Passwords

It will be there if you ever entered your password and afterwards told firefox to remember the password when it asked. If you never hit Remember there really isnt much you can do to recover the password.
