### Post by cave_man() on 2011-02-01
Hello Rami,

You have asked quite a large question. If you have not installed an operating system before you should do some more research before proceeding.

You can download 8.04 from here:
[http://releases.ubuntu.com/8.04.4/](http://releases.ubuntu.com/8.04.4/)

Then go to here:
[http://www.ubuntu.com/desktop/get-ubuntu/download](http://www.ubuntu.com/desktop/get-ubuntu/download)

and read step 2 (Burn your CD or create a USB drive). If you have a machine that supports booting from a USB device I would try that first. Select the radio button for the operating system that you are using and click "Show Me How" for some helpful reading.

Good luck.
