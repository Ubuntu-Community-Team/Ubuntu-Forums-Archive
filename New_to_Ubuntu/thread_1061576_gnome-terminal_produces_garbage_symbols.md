### Post by overlord.gaurav on 2009-02-05
You can't fix it. It's not an error.
If you want scroll up/down, use "CTRL + Shift + Page Up/Down" for doing the same.
If you wish to copy a text use "Ctrl+Shift+C".
If you wish to paste a text, use "Ctrl+Shift+V".

### Post by Robert.Cashman on 2009-02-06
