### Post by s1baker on 2011-01-09
hi,
 I'm just a noob, and I use the Gnome desktop( I think that is what it is called ), so my system might be different than yours, but I did a check of where on my system my wallpaper was located and I found it in two places. One place was in home folder and the other was under the home folder in .cache/wallpaper ( .cache is invisible and you have to set nautilus to see the invisible files.
 If you know the name of the wallpaper that originally worked, you may want to try to locate where it resides in your directories and see if the new wallpaper is also there. Just a noob suggestion.

 You can type -->   locate -i name of wallpaper
in a terminal and see where either wallpaper is residing and make sure the new one is where the old one is.
