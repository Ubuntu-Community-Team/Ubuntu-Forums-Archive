Haha, that was my fault. I was trying to be helpful by supplying a nice easy apturl link.

To let you know what happened: apturl is a way of a web page prompting your package manager (the system that installs, removes and updates software on your machine) to download and install a package. Clicking on an apturl is exactly the same process under the hood as going to Software Centre or Synaptic and installing the package from there. The package manager goes and gets the package from the Ubuntu repositories and installs it.

In your case Segofam, for some reason your browser doesn't know to pass apturl links over to the package manager. I'm guessing you're not using Firefox?
