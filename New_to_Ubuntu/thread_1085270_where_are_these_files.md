### Post by jpeddicord on 2009-03-02
.fr-xxxxxxx is a naming scheme used by the archive manager for temporary files. As soon as you close the archive manager, the directory and its files are deleted. If you would like to remove your document history, just go to Places > Recent Documents > Clear Recent Documents. To clear downloads in Firefox, go to Tools > Downloads and just hit the Clear button.

If I misunderstood your post and you are actually able to open that resolve.wav file, then in the file manager go to View > Show Hidden Files and the .fr-xxxxxxx folder should appear. Normally as I said these .fr-xxxxxxx folders are removed when the archive manager is closed, but it may still be there.
