### Post by t0p on 2009-07-31
You want a download manager like Flashget?  Why don't you try **Flashget**?

In Firefox, go to **Tools > Add-ons > Get Add-ons**.  Search for Flashget.  Install it.  Restart Firefox.  And there you are.

If there are any other Firefox add-ons that you'd like to use, do the same:
**Tools > Add-ons > Get Add-ons** and search for the add-on you want.
