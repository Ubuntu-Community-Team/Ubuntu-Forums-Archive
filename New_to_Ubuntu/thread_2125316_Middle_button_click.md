### Post by LewisTM on 2013-03-13
For Chrome/Chromium, the AutoScroll extension would do the trick.
Middle clicking in Xubuntu has other roles. It will open a link in a new tab. Middle-clicking a tab will close it. 
Same for folders in the latest Thunar 1.6.2.
It also pastes the latest selected text when pointing in an editable text area. 
On the desktop, it opens a window+workspace list.
