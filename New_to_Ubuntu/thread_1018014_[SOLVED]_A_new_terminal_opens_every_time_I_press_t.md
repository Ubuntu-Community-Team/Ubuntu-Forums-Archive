### Post by ant2ne on 2008-12-21
File from the menu bar. Open a terminal is somehow set to the s key. when I type ssh I open 2 additional terminals, and the s key never gets typed. How did this get set to the s key? More importantly, how do I change it back?

---

### Post by sisco311 on 2008-12-21
In the gnome-terminal go to Edit -> Keyboard Shortcuts... and set the New Window shortcut to Shift+Ctrl+N

If the menu is not present, right click on the terminal window and select the Show Menubar option.
