### Post by EXCiD3 on 2008-05-25
"unhackable" is a misunderstanding. Your system is only as secure as the software running on it. Recently a security hole was fixed in any debian based linux (including ubuntu) that had been around for 3 or 4 years and left secure connections essentially open yet nobody noticed this bug until just recently. That stated, linux is much more secure by default than any other os, partially because of customizability and a much lower user base leaving it last for hackers to work on when they want to steal information.

As for your laptop, you should take a look at the tutorial I wrote for HP and Compaq laptops, showing you basically everything you will need to install from start to finish. The link is in my signature.

Your best bet would probably be to dual boot with windows unless you feel that you can get by without windows on the computer. You can always setup a dual boot and then remove the windows partition later.
