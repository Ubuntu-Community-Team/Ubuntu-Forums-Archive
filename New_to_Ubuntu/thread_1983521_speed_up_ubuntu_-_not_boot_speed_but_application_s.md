### Post by 0011235813 on 2012-05-20
I would recommend you use lxde if your computer is slow. You can use:
```
sudo apt-get install lxde
```
from the terminal (Ctrl+Alt+T) or you can just search for lxde in the software center. If you want to install the applications that are default in Lubuntu as well, use:
```
sudo apt-get install lubuntu-desktop
```
