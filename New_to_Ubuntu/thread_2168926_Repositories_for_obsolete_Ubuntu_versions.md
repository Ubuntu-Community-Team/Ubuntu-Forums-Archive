### Post by DuckHook on 2013-08-20
You've probably heard this already, but unless you have absolutely no other choice, it is a very bad idea to run obsolete versions. The fact that it is no longer supported means that security holes go unplugged and the likelihood of getting owned increases with each passing day. Remember, it isn't just the OS itself--obsolete apps also lose support--which means that your security risk multiplies with each app you install.

If your objection to the latest releases is based just on preference, then explore some of the other flavours and even alternate distros. Start with [this]("http://linuxfreedom.com/Distros/") link. Forum members are quite willing to help with suggestions and alternatives.
