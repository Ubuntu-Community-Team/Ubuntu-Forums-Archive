### Post by fabricator4 on 2011-02-18
See if you can boot off it.  Some OEMs offer two levels of disk, a backup disk that can be used to restore everthing, and a recovery disk that can me used to install the OS and all of the OEM add-ins.

There's often also a recovery partition on the HDD.  Usually a particular key press at boot time will boot this recovery partition and give you the option of doing a fresh install.  You'll need the license number off the Windows label on the machine.  Most OEMs offer either the recovery disk shipped with the machine or the recovery partion on the HDD, not both.

### Post by Kixtosh on 2011-02-19
If you want to check how your DELL recovery disk works, you could try it before installing Ubuntu to see if it wipes the hard drive completely and restores the computer to its original "shipping" condition.

Certainly, this is how my last laptops (that still had restore disks, but no recovery partition on the hard drive) worked. You would have the option to use the restore disk to wipe the hard drive and restore everything, including whatever the original O.S. was, and whatever trial software (such as MS Office) was bundled. My newest laptops do not have a restore disk, but a restore partition on the hard drive only, although you are usually asked to create restore disks as well when you first start up. In this case, installing Ubuntu in a dual boot configuration did not wipe out the restore/recovery partition on the hard drive. 

If you do choose to use the entire disk when installing Ubuntu (not dual boot), then it is probable that any recovery partition that exists will be wiped as well.
