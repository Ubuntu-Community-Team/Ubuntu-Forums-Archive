### Post by mvalviar on 2008-12-24
The window switcher is (by default) activated by pressing super (win key) -tab. This is a fancier version of the application switcher which is alt-tab. Perhaps you are referring to the compiz-fusion plugin "Scale" which work like Mac's Expose. If you have compiz settings manager you can configure it to your liking.
