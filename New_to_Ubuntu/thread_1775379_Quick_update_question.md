### Post by dFlyer on 2011-06-04
> **Proto Blaze said:**
> Does "sudo apt-get update" update everything?

That only updates your archives.

sudo apt-get upgrade update the system

sudo apt-get dist-upgrade performs a distro upgrade

---

### Post by jaacko on 2011-06-05
So to get the latest versions of installed packages you need to run these lines:

```
sudo apt-get update
sudo apt-get upgrade
```
