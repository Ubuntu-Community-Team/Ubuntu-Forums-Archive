### Post by Paddy Landau on 2011-12-18
> **fantab said:**
> By the time 10.04 reaches end-of-life you will have 12.04 LTS already released (and I think it will be supported for 5 years).
That is correct. Support for 10.04 desktop ends 13.04, 10.04 server ends 15.04, and 12.04 desktop and server both end 17.04

However, bear in mind that only the first two years get hardware support updates. After that support for hardware updates ends.
[https://wiki.ubuntu.com/LTS](https://wiki.ubuntu.com/LTS)
[http://www.canonical.com/content/ubuntu-1204-feature-extended-support-period-desktop-users](http://www.canonical.com/content/ubuntu-1204-feature-extended-support-period-desktop-users)
