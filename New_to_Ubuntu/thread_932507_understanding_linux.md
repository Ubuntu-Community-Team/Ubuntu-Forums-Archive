Same reason Linux has a kernel. It is the software interface between the GUI and the hardware.

### Post by candtalan on 2008-09-28
Welcome!
Debian is the name of a well established distribution (a type of) gnu/linux operating system.

Ubuntu is based on a debian approach, particularly in terms of its use of the debian repositories, and hence the method of package management (installation of programs)

The kernel is the central core of operating code which has the rest of the operating system built around it. The kernel remains running when a user logs in and logs out.

I do not think windows has a kernel. It grew like topsy :-)

GNU is the operating system proper, it stands for Gnu is Not Unix.
