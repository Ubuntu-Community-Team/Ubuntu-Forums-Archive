### Post by Vaphell on 2011-01-14
imo you should go with a current release but install some lightweight environment on it, like lxde. Less eyecandy but more snappy which is what you'd want on an aged hardware with modest parameters. The only reason to use older versions i can think of is proprietary drivers (eg having support for 3d) which may not be compatible with new releases.

If Ubuntu is not suitable for your machine(s), then there are lots of other alternatives for you.
