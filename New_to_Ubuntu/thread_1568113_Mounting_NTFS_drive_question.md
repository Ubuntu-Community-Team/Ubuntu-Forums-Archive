### Post by NightwishFan on 2010-09-04
Ubuntu has read/write support for a Windows NTFS volume. It is stable enough to be included so I am sure it is safe however you CAN modify or delete files if you choose to do so. I would not mount a Windows drive in which Windows did not shut down cleanly.

So as for affecting Windows 7, as long as it shut down cleanly and you do not choose to delete a needed file, you can browse and open files on the Windows partition.
