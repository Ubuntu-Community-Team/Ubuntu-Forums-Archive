### Post by Mark Phelps on 2012-03-07
YES -- you should definitely confirm that Ubuntu can be INSTALLED and will WORK on the droid before you wipe it.

We've been getting threads from folks presuming that can be done, but what they link to is misleading because all their links really mention is running an Ubuntu-like desktop on their Android phones.

This is like swapping Unity for Gnome Classic on your Linux desktop PC -- it changes the desktop but the underlying OS remains the same.
