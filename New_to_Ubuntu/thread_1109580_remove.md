### Post by talsemgeest on 2009-03-28
You can disable a toolbar either by right-clicking next to the main toolbar (File, Edit etc...) and unchecking the toolbar you want to be disabled. However, if you want it to be removed completely, go to Tools> Add-ons and select "Uninstall" on the toolbar you want to be removed.
