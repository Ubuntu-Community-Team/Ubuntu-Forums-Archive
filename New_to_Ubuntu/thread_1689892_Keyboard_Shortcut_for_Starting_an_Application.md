### Post by garyjmellor on 2011-02-17
If you're into using shortcuts, I can't recommend Gnome-Do enough. You can learn about it here: [http://do.davebsd.com/](http://do.davebsd.com/). I installed this about a month ago and I love it. It really does make life easier. You can get it from Synaptic or open the Terminal (ALT+F2 as the previous poster pointed out ;)  ) and type sudo apt-get install gnome-do gnome-do-plugins.
