### Post by donkyhotay on 2008-12-18
It's a bash script then and you need to specify that to your system. The easiest way is by using the CLI you would type
```
cd %name-of-folder%
sh ./%name-of-file%.sh
```
otherwise if your using the GUI you would need to right-click on the file, select properties, click on permissions, check mark 'allow executing file as program' then try double-clicking on it (but the CLI is more certain).
