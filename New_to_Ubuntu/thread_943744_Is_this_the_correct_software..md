### Post by SunnyRabbiera on 2008-10-10
Well if needed you can install the desktop edition and then install what you need to run your server.
Please keep in mind that the server edition has no GUI, so if you need a graphical interface right off the bat then the server edition is not for you.
For the beginner, its easier just to install the desktop edition of ubuntu and then install what you need to run your server.
