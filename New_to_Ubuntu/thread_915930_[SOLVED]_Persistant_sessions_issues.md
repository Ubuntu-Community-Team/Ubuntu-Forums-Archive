### Post by lovinglinux on 2008-09-10
I'm having issues with sessions. No matter what I do, some applications still initialize on startup. For instance, I have uninstalled and purged Screenlets, but every time I start a new session there is a SlideshowScreenlet.py there. I have deleted it in the current session tab, deleted the .screenlets folder and saved the session without it, but it keeps coming back to haunt me. I know this probably doesn't hurt my system, but my OCD keeps complaining about it.

Any idea how I could solve this?

---

### Post by Seisen on 2008-09-10
Go to System>Preferences>Sessions and click on the Session Options tab. Make sure that the box that says "Automatically remember running applications when logging out" is not checked.
