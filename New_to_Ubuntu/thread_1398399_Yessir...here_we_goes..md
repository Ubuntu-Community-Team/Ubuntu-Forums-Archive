### Post by BT1 on 2010-02-04
Might wanna try Ubuntu first and not Kubuntu (even though I love Kubuntu's desktop). Ubuntu is the easiest Linux distribution to learn when migrating from windows to my knowledge and the knowledge of many others.

It's desktop interface is a lot like windows, possibly windows 98ish feel to some things in my opinion, but definitely has an XP feel in many others. You won't have to do command line stuff, and it's intuitive.

Time from install to surfin the net? From the moment the desktop first boots up you can be browsing through firefox, but you should do the update manager that makes sure everything is up to date. And it will tell you in the top right of the screen if you need to download a driver for your video card. Usually the whole process is very quick and painless.

For me I usually:

1.) Install Ubuntu
2.) Download the updates, download video card driver (it does both for you when you click on the icons with the "!" lol.
3.) Go to the Ubuntu Software Center and download "Ubuntu Restricted Extras" which gives you free mp3, mpg, avi, etc codecs.
4.) Now you're ready to browse the net, listen to most any music, watch most any video.
5.) The default "paint" program is more like Adobe Photoshop, it's GIMP. A high power graphical design program that comes included.

Pretty easy, and if you have questions about anything, we're here to help you!

Have fun!

I've found it to be consistent across KDE distros not just kubuntu. Let it be known that despite what many people here imply, ubuntu is not the only distro. I'd suggest Mandriva for a newbie or chakra if you're feeling adventurous

KDE is tightly integrated and yet largely modular, for example you only have to install a few parts of the desktop if you want (probably more advanced though)

If you want a KDE based distro I'd start off with [Chakra]("http://chakra-project.org/") or [Mandriva.]("http://www2.mandriva.com/"). The former is based on Arch Linux which does not come recommended for newbies but there is a live CD so the hassle of installing it is taken off. The advantages are that you can use the excellent Arch wiki and you'll have the latest and greatest releases. The downside is you may have to fix a complex problem with only google.
Mandriva is also based on KDE and includes non-free codecs pre-installed, the MCC is also very good because it puts everything in one place
