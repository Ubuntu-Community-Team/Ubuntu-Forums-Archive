### Post by yeats on 2009-02-05
Downloading the debs directly does work, FYI.  You can even remove the default OpenOffice and integrate OO3 into your menus with another deb package that lives in a folder called "desktop-integration" that will be in the "DEBS" folder.  If you try to install desktop integration with OO2.4 still installed, you'll get a dependency conflict error - it has to be one or the other.

The tutorial that Voland links to is different than what I'm mentioning here, FYI, and is probably the "safer" way to do this if you haven't done much with .deb package installation.
