### Post by philinux on 2008-05-29
The normal quit plus Hibernate hard locks the system up. As does suspend.

All I wanted was a power saving mode with a quick boot up.

```
sudo pm-hibernate
```

Does the trick, boots up to where I was in about 15 seconds from hitting the main power button. 

Some quick message comes up about activation failed.
