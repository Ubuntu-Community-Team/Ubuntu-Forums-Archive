### Post by life in color on 2011-06-23
Um I would reccomend simply getting another classic (preferrably 5g or older) I have heard that ipod Touches/iphones work with Linux after jailbreaking them but I can't say that I've tried. VirtualBox is HARD TO INSTALL!!!!!! I couldn't due it and as a result it sucked up some of my RAM temporarily, it's also very hard to find a windows iso file to use, I would look into VirtualBox and perhaps watch some tutorials on youtube, but I would tell you strictly not to mess around with it. Also you should definitely get VirtualBox running before buying the ipod Touch/iPhone. If you are dead set on getting one I would also suggest possibly getting an older version as jailbreaking becomes trickier with every new firmware. I would say 3.1.2 would be the best version to get, only because jailbreaking worked fine on my old iPod Touch (lost it long ago) with that version. WINE with iTunes is an option too though its probably even more trouble than VirtualBox, I installed iTunes through it probably 7 times and it never worked, if you mess up once with an installation your basically screwed too so if you try WINE definitely do it while watching an installation tutorial.
