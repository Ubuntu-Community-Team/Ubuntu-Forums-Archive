### Post by scottuss on 2009-12-02
Type this into a Terminal:

sudo apt-get install apache2

Or search for apache in Synaptic Package Manager

Edit: To get to Synaptic, if you don't know, you click System > Administration > Synaptic Package Manager,

You may want to also install PHP etc
