### Post by handy on 2009-02-25
I used to use Dreamweaver on both windows & OSX; but boy is it an over complicated piece of software imho.

For some years I didn't need to do any web work anymore; but then I decided I'd like to say little something, so I went looking for the simplest piece of software to do the job, that would run on Arch.

I couldn't find one that was satisfactory, so in desperation I thought I would see what was available on OSX, which is what I (rarely ever) dual boot my iMac with (as well as Arch, in case you missed it).

Very quickly I found Karelia software's [*_Sandvox_*]("http://www.karelia.com/sandvox/"), Sandvox is not perfect, but it is just so easy, & the problems incorporated in it are all but inconsequential to me, thus far.

So, unfortunately for me, who really doesn't like OSX, I have to go there occasionally to do my web work.
