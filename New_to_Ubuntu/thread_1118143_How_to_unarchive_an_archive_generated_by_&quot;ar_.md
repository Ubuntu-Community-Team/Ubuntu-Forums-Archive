### Post by joeashcraft on 2009-04-13
Is "ar cr x.a x1.o x2.0" a program? 
If you right click the file and select Properties you'll hopefully see the file type. If it's a compressed file try opening it in file-roller (you can press Alt+F2 and type in file-roller).
