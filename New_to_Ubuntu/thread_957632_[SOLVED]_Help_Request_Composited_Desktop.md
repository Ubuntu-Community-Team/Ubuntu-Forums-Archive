### Post by LowSky on 2008-10-24
Compiz lets you run a fancy desktop that can have cool effect like Cairo's dock, and cool things like Wobbly windows desktop cube and fire writing..etc

To use Compiz and programs designed to work with it you need a graphics card, and a program called CCSM

to install the program just open a terminal and copy and paste this command
```
sudo apt-get install compizconfig-settings-manager
```
