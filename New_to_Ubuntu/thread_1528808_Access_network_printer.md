### Post by Mark Phelps on 2010-07-11
Install Gutenprint.  Then, open a browser and enter "localhost:631".  This will opwn the Gutenprint printer management page.  You can add a printer from there and somewhere in the menus, it will search your network for attached printers, find them, and allow you to select what it finds.
