### Post by Zill on 2013-05-28
Acepony:  As a general rule, Ubuntu can be installed on as many systems as you wish.  Although there are different licenses used within Ubuntu, most of these, such as the [GPL]("http://www.gnu.org/licenses/quick-guide-gplv3.html"), are intended to *give* freedoms, rather than *restrict* them. ;-)

This [Licensing]("http://www.ubuntu.com/about/about-ubuntu/licensing") link gives more information.

Some software is provided with different licensing but these are generally proprietary packages, such as video drivers etc.

I have no knowledge of Ubuntu Studio but would expect it would have similar licensing to the "main" system.  However, if you have any particular concerns about a package just research the licensing for that specific package or post back here.
