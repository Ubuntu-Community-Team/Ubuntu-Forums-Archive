### Post by Sam Lars on 2008-11-14
You could install boinc-client and boinc-manager to run distributed computing at full processor.
You could run glxgears to max out graphics.
When you boot up, you may have to press Esc to get to the Grub boot menu, and you can boot into MemTest86, which will max out memory (and maybe processor, too).
