### Post by pastalavista on 2010-05-27
Ubuntu uses Gnome for almost everything. If you wanted LXDE, you should have installed Lubuntu. You can try ```
sudo apt-get remove ubuntu-desktop && sudo apt-get install lubuntu-desktop
```but it won't be supported as far as automatic updates are concerned.
