### Post by Crafty Kisses on 2009-05-13
When you say Compiz "isn't working" what do you mean by that? Try running the following command in the Terminal:
```
compiz --replace
```
Then see if that works, that will make Compiz the default window manager. If that doesn't work you might want to run the Compiz-Check script. Read this thread: [http://ubuntuforums.org/showthread.php?t=799070](http://ubuntuforums.org/showthread.php?t=799070).
