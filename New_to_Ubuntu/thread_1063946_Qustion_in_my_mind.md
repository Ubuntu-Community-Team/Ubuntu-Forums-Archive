### Post by ptn107 on 2009-02-08
You can install Ubuntu and run the following commands to install their respective desktop environments:
```
sudo apt-get install kubuntu-desktop
```

or

```
sudo apt-get install xubuntu-desktop
```

Understand, though, that in either case it will need to download all the necessary files.  It will be a smaller download than a Kubuntu or Xubuntu CD, but it will still be of significant size.  You're better off downloading the iso or purchasing it online if your connection is poor.
