### Post by oldfred on 2013-07-15
Did you install Ubuntu in UEFI mode or BIOS mode.
The BootInfo report from Boot-Repair will give us lots of details.

Best to shrink Windows from inside Windows and reboot as it has to run chkdsk after any size change.
Also Windows 8 is always hibernated and that needs to be turned off when dual booting.
