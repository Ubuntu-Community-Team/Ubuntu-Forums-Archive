### Post by nhandler on 2008-07-21
Have you seen [this]("https://help.ubuntu.com/community/LiveCDCustomization") wiki page? It outlines a way to modify the live cd iso without the use of any other programs. You can use this method to update Ubuntu and Java. This method does not have a GUI, it is all done in a terminal. If you are not comfortable with the terminal, you should try Reconstructor. It works well, and it will be able to do what you want.
