With my upgrade to 9.04 for some reason Brasero didn't update, and actually won't update/upgrade. Within my Update Manager it shows that an update/upgrade is available but it won't allow me to check it to be installed. Is this a bug or did something else happen? Anyway to fix this?

Nevermind, think I got this one too. Apparently there was some conflict between nautilus cd burner and brasero so I had to remove nautilus cd burner in order to update. Found this out by uninstalling brasero and trying to reinstall.
