### Post by wizrddreams on 2015-07-21
For the Universal USB installer ([http://www.pendrivelinux.com/universal-usb-installer-easy-as-1-2-3/](http://www.pendrivelinux.com/universal-usb-installer-easy-as-1-2-3/))
1. Select which Distro you are using (in your case Ubuntu)
2. Point the program to the .iso file
3. Point the program to your usb drive
(All drop down selections)
Click Create

If all goes well the program should run through and you will have a bootable usb stick. 

This method does not require a DVD at all. 

Step back from it for a day or two and come back to it!
