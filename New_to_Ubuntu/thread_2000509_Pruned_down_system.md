### Post by deadflowr on 2012-06-09
Unity is built on top of gnome.But yes, if you want to be able to log into either a kde(kubuntu-desktop) or unity, you can. But realize that if you log into a unity environment, you'll be using the gnome desktop enviroment with the unity-shell.
If you have the gnome options, such as gnome-shell, and gnome classic, I don't know what affect that would have on your system if you remove them.
But they are not needed to run, they are simply shells like unity.
