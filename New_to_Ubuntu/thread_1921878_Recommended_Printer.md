### Post by kurt18947 on 2012-02-07
Most Brother printers seem pretty Linux friendly.  I have 2 AIOs, an MFC-7820N Laser and MFC-6490CW 11X17 inkjet.  The 7820N is super simple; it's found and software is installed entirely over the network.  The 6490CW requires downloading 3 .deb files and some command line copy & paste. The only one I've seen have an issue is the HL-2270DW laser printer.  That required a patch in order to install & work properly.  Here are a list of Brother printers that have Linux drivers available for them:

[http://welcome.solutions.brother.com/bsc/public_s/id/linux/en/download_prn.html](http://welcome.solutions.brother.com/bsc/public_s/id/linux/en/download_prn.html)

A useful feature of some Brother printers is that LC61 & LC65 cartridges don't have any electronics built into them so they're easy to refill to help control ink costs.  The easiest installs are those printers that are listed at openprinting.org. The problem is that by the time printers are listed, they're often discontinued.
