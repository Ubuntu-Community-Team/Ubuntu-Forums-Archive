### Post by drs305 on 2012-08-11
Some linux versions continually update their packages/apps to provide the most up-to-date-apps, others do so infrequently in preference of stability. 

Ubuntu takes a middle ground, releasing a new version each 6 months and a LTS version every two years. Once an Ubuntu version is released, in general the packages are only updated with security patches and fixes. This allows the developers to concentrate on a new release and make sure the major bugs are eliminated during the testing period before the final version is released for that cycle.
