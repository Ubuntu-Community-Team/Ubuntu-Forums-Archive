### Post by cb951303 on 2008-05-18
there is nothing close to Dreamweaver in open source world. Sure kompozer, nvu, quanta etc... they are all good products but they lack a lot features that Dreamweaver users are used to have...

I personally use Eclipse + PDT + FTP Plugin for my needs. It's not WYSIWYG but if you are using PHP, I believe it's more productive than other open source alternatives. Sure I miss dreamweaver's WYSIWYG but after a while with Eclipse I got as fast as I'm in Dreamweaver. That's just what I need for a productive development environment...
