### Post by 3rdalbum on 2009-03-04
Synaptic Package Manager and the Add/Remove use the same underlying mechanism. You could install things from Add/Remove and remove them through Synaptic or vice-versa, if you wanted.

Add/Remove has a simple interface and shows only well-known Linux graphical programs. Synaptic Package Manager shows all available packages, including libraries, documentation, little-known software, command-line utilities, etc. Synaptic is more powerful.

If the interface for Synaptic doesn't scare you, I guess there's no reason to use Add/Remove. Just try not to break things, as it is possible to break your system by installing something dumb (or removing something essential) from Synaptic. Have fun.
