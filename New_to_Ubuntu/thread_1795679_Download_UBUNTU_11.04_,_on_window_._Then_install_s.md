### Post by nzjethro on 2011-07-03
If you are downloading from the Ubuntu page and you experience an internet crash, you will have to restart your download. However, with a torrent, if you lose internet, it will start downloading from the same point when your internet is back up and running. Thus, I'd recommend using a torrent ([here]("http://torrent.ubuntu.com/simple/natty/desktop/") is a link to the latest 11.04 torrent, and you can navigate to earlier releases from here).

Following this link will download a .torrent file to your computer (likely to your "Downloads" folder).ou need to open this file in a bittorrent programme (like uTorrent). If you already have a bittorrent programme installed, simply double click the .torrent file and it will open your bittorrent programme and begin to download the Ubuntu .iso file. When it finishes downloading, you must burn this to a CD, or create a Live USB using unetbootin, and then use this to install Ubuntu alongside Windows.

This link ([https://help.ubuntu.com/community/BitTorrent](https://help.ubuntu.com/community/BitTorrent)) will give you more information about using torrents to download Ubuntu. :)

### Post by Wim Sturkenboom on 2011-07-03
Adding to above:
You need to make space on your hard disk. If I'm not mistaken the Ubuntu installer will offer the option to shrink the windows partition. *_Before doing so, defrag your windows partition._*

Give Ubuntu at least 10 GB if you only want to try it out in a side-by-side install. Give it more if you know that it might become more serious.

PS Make backups of important data; things can always go wrong.
