### Post by Impavidus on 2013-05-10
Linux servers use antivirus software to protect surrounding windows machines. Other Linux machines don't really need it, as Linux tends to be* immune to viruses. Unless you disable the pretty strong security model of course. I don't run antivirus software. You have to remain careful though, especially with java or wine (both not installed by default). Don't install just any program you find on the internet and be careful following instructions you don't understand. Commands given here, on askubuntu.com or in the official documentation (help.ubuntu.com) are generally save, installing from the repositories via the software centre/synaptic/apt-get is save too. Don't enter your password if you don't know why. Have a look at the stickies here: [http://ubuntuforums.org/forumdisplay.php?f=338](http://ubuntuforums.org/forumdisplay.php?f=338)

*: Indeed, *tends to be*, not *is*, because viruses that attack Linux can exist. In practice the threat is small.

### Post by agillator on 2013-05-10
To expand on Impavidus remark, Linux is not 'immune' to viruses. However, it is not a real target for malware for a couple of reasons. First, Windows is a much more lucrative target because it is a far larger market and its security is so poor. Second, the security and system of permissions on Linux/Unix is such that even if malware got a foothold its chances of doing any damage beyond a single user's files, though not impossible, are quite limited. So without the user's knowing or unknowing cooperation there seems to be little point in wasting the time and effort in creating malware for Linux. That being said, I have been reading that malware on the Mac is increasing. It's operating system, I understand, is basically Apple's proprietary version of Linux or at least very similar. I do not know if its security is more lax or if it is just a bigger target. Bottom line: Linux users tend to not worry too much about viruses, as Impavidus said, because there are few if any, at this time, that affect them. However, this doesn't mean there won't be in the future and we can ignore the threat. It is just that we need not go out of our way to do anything more than be careful today. Tomorrow could be a different story.
