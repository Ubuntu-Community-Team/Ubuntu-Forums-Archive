### Post by quixote on 2009-08-11
The Desktop settings are in Nautilus preferences (Eg Start > Places > Home will open Nautilus, and prefs are under Edit).  There is an option there, I believe, to show only icons.  

But if you do have both icons and text selected, but nonetheless don't have text, there's one other possibility.  If the Default View for icons is set small, less than 50%, it won't display text either.  I assume that's a bug, and it's very annoying, but the only workaround I've found is to have that set to at least 50%
