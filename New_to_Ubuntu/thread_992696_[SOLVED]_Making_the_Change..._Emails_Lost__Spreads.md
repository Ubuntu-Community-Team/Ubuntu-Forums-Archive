### Post by bennachie on 2008-11-25
You're right, that's a flood of questions. Basically, there will be no problem with Microsoft Office documents unless you have been saving things in Office 2007 format, in which case you will have to update OpenOffice to version 3.0. Nor will you have any problem with Firefox or Thunderbird (so long, of course, as you save your emails and other data before wiping your old system). 

The simplest way to deal with emails is probably to install the ImportExportTools add-on (now at version 2.1), both in Windows and later in Ubuntu, then save your Inbox and Send folders in mbox format. The Linux version of Thunderbird, with the add-on installed, will read that without any trouble at all.

Running Windows programmes under Ubuntu is a bit chancier - you have to rely on software packages like Wine or Crossover Pro, both of which seem to handle "normal" programmes quite well, but are not guaranteed to deal with every possible set of circumstances.

Good luck!
