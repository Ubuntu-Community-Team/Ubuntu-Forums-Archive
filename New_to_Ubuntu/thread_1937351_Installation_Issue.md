### Post by grahammechanical on 2012-03-08
How much RAM does the machine have? I am wondering if it has 4GB or more then the install process might be wanting to convert the linux kernel on the alternate CD to a PAE kernel to make full use of that amount of RAM.

I suggest that you download the standard 32bit Ubuntu iso and burn it to another disk and when you are asked for Ubuntu 11.10 Oneiric Ocelot release i386 etc., you put this new disk in the drive.

The alternate install CD does not have all the stuff that the standard Ubuntu Live CD has on it. The install process is asking for the Oneiric disk because it needs to take some files from it.

In my opinion.
