### Post by whatthefunk on 2012-01-05
To get started, you should read up on bash commands.
[http://www.gnu.org/software/bash/manual/bashref.html](http://www.gnu.org/software/bash/manual/bashref.html)
[http://linuxreviews.org/beginner/Bash-Beginners-Guide/](http://linuxreviews.org/beginner/Bash-Beginners-Guide/)

You use bash in the terminal.

To download a program, you can either use a package manager, or the terminal by tyoping:
```
sudo apt-get install NAME_OF_PROGRAM
```
