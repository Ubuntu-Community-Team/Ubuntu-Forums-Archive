### Post by papibe on 2014-04-29
Hi retriever.

I'd recommend sticking to LTS releases like 12.04 and 14.04.

14.04 provide the functionality to send searches to 3rd parties, like reddit and amazon (very much like a web search), but ONLY if you enable it. By default if off so I wouldn't worry about it.

12.04 does NOT have that functionality.

Hope it helps. Let us know how it goes.
Regards.

### Post by deadflowr on 2014-04-29
Open the dash and type "details", open that and go to legal notice and read.

If you want to not have that happen to you then re-open the dash and type "privacy" and open it, go to search and toggle it to off.

Simple as that.

This does not affect Ubuntu 12.04.
