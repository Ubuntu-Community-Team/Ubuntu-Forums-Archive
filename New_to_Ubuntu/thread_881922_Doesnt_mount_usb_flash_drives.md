### Post by ilrudie on 2008-08-06
In vmware player the host os usually gets control of devices that are added and you have to tell vmware to let the guest os control the device.  Look around the player menu's and at the top of the window and see if you see any usb devices.  You should be able to click the device and allow the vmware guest to see it.  Some versions of vmware allow you to share folders between the guest and host and some allow drag and drop between the guest and host.  I'm not sure if player has either of these features.  Check out the vmware website.  I'm sure you can find info on shared folders as well as better info on how to mount usb devices.
