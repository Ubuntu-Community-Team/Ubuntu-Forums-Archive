### Post by sandyd on 2009-02-20
WARNING. these instructions may be overwhelming

```

sudo apt-get install git-core
mkdir dev
cd dev
mkdir git
cd git[FONT=monospace]
[/FONT]git clone git://source.winehq.org/git/wine.git ~/[FONT=monospace]dev/git/wine
[/FONT]cd ~/[FONT=monospace]dev/git/wine[/FONT]

```that should get your source code set up

now onto part 2
```

cd ~/[FONT=monospace]dev/git/wine[/FONT]
./configure
make
sudo make install

```Now try a few apps in WINE to make sure everything working... then go download the patch to here

~/dev/git/wine

apply it
then run this

```

./configure
make
make install

```P.S. this takes a LOOONNNNNNNNNGGGGGG time if your rig is slow. you might want to do the first part when you sleep.

### Post by sandyd on 2009-02-21
try this......
```

winecfg

```
go to the video settings section

select emulate virtual disktop

p.s. did you do this??
After that you may safely install and play fallout 3. In case you will have video problems, you may need to set some registry keys. Run regedit and browse to: Current_user/Software/Wine/Direct3D you may need to set these values:
"DirectDrawRenderer"="gdi" 
"OffscreenRenderingMode"="fbo" 
"PixelShaderMode"="enabled" 
"SoftwareEmulation"="enabled" 
"UseGLSL"="enable" 
"VertexShaderMode"="hardware" 
"VideoDescription"="NVIDIA GeForce 9600 GT" 
"VideoDriver"="nv4_disp.dll" 
"VideoMemorySize"="512"

set them with your own values of course
