### Post by coffeecat on 2015-02-26
If you choose "save to computer", the downloaded attachment will be saved to whichever folder you have specified in Firefox Edit -> Preferences -> General tab. The default is the Downloads folder. I haven't changed the default in my setup and I have just saved an attachment from my Yahoo account and it ended up in my Downloads folder.
