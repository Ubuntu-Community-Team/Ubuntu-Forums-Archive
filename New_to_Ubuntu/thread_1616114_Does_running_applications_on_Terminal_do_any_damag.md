### Post by diablo75 on 2010-11-07
No, you just get verbose output that is normally hidden from the user.  That "Goodbye" message happens on everyone's computers, you just never see it because nobody runs programs like Firefox from Terminal unless they have to for some odd reason.  This is typically done if you're trying to troubleshoot a problem or find subtle glitches.

### Post by janpol on 2010-11-07
Is exactly the same thing as running them from an icon in your desktop. Sometimes, when you are having problems with an app, its a good idea to launch it from a terminal, since you might get an error message that points out where the problem is.

But, besides getting messages, there is absolute no difference.
