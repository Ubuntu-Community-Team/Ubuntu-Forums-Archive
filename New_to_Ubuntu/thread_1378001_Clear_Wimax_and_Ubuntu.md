### Post by crescen7 on 2010-01-11
The main system, that is using the modem and connecting through an ethernet cable works great.  Download speeds consistently around 3MBs and upload around 1MBs.  Those are both better than what I was getting on Att dsl, but about half of what cable users are getting in this area.

Problem is that the mobile connection (usb modem) appears to only work on windows.  I just got that and I'm in the process of seeing if I can either find a linux connection method or set up a VM to run windows to get the connection working.
