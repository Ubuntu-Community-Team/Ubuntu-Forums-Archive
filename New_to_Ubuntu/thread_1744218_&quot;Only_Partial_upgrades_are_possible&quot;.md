I had this problem, I think it may be because you have installed additional desktop environments, so it can only do a partial upgrade to those DE's as they're only partially installed, 

[http://www.psychocats.net/ubuntu/puregnome](http://www.psychocats.net/ubuntu/puregnome)

there are scripts here that remove other desktop environments and will get you back to pure gnome, after you have done this run the computer janitor to remove any obsolete packages still installed then try updating again
