### Post by ajmorris on 2008-11-02
hi there, that is strange... i assume that the mount point exists? For mounting things via fstab, and mounting things manually, the mount point must exist, unlike the hotplugging method, that auto-mounting uses, and tools like gparted use. You also might need to add 'auto' to your options... but also, maybe not, i cant remember if auto, or noauto is the default... Also, can you please post the contents of your /var/log/syslog file (please either use [code] tags or upload the file itself, as this file is very long) This file will give us the output of any errors received when fstab tries to mount the partition    AJ
