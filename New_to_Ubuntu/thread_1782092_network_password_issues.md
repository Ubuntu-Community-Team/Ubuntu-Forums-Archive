### Post by Scunnered on 2011-06-14
Provided you are happy using what would be an insecure system it is possible to automatically login and have the wifi connect as well.

By selecting System > Administration > Login Screen you are able to set up the initial login.  By right clicking on the wireless network connection and selecting edit connections you can similarly apply the automatically login setup. Open the wireless connection and highlight your connection. Select edit and ensure that there is a tick at connect automatically & available to all users. 

I hope this assists.  I when I take the laptop out of the house reverse the process just in case it gets into the wrong hands.

Hope this assists

PS I am using Lucid (10.04) but I expect the processes will be the same for 11.04.
