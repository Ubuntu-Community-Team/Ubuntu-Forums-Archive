Well if you are going to do a fresh install as an update, then you would install 8.04 over the 7.04 partition and just use that again. You would have to make new partition and manually select that partition to be install on but not the old one.


My advice is just to _back up all personal data _ it is very important and makes your computer yours.

You can also manually backup config files from your home hiddenfiles. Just ask if you need help with that, but I would just recommend doing a completely fresh install
