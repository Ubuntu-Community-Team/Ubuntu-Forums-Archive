### Post by SeanHodges on 2008-07-05
[https://wiki.ubuntu.com/DevelopmentCodeNames](https://wiki.ubuntu.com/DevelopmentCodeNames)

Gutsy and Hardy were development code names (as were Whistler and Longhorn). The official names for these are:

Ubuntu 7.10
Ubuntu 8.04

However, partly because Ubuntu is a community effort, the code names usually leak out into general usage.
