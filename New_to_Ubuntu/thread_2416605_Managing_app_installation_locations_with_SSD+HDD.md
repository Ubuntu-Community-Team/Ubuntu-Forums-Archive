### Post by mosho00 on 2019-04-12
I am thinking about buying a laptop with 128GB SSD and 1TB HDD and want to install Ubuntu. I want to put root on the SSD and home on the HDD. In that case, where will the package installer install my apps: on SSD or HDD? Is there a way I can choose where it installs, because I don't want all my apps on SSD as it is not big enough, so could I install apps on HDD?

---

### Post by Dennis N on 2019-04-12
The package installer will not install applications to your home directory. They would go into certain other folders in your root partition. 128 gB of space is usually more than enough for any single Ubuntu installation. I only will give 25 to 30 gB space for the installation including the home folder. I don't use a separate home partition, but instead create a separate data partition for my own files (often on another disk).
