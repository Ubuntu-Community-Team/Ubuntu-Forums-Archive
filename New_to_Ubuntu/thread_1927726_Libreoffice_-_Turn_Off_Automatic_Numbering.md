### Post by ajgreeny on 2012-02-18
Sorry, but you've lost me totally!  Where does the number 21 come from; is it what you added with the pasted text or is it a bullet/number entry?

What I think you will need to do if you want to add paragraphs within a numbered paragraph situation is type the whole piece with all paragraphs.

Now highlight the whole piece and click on the numbers (or bullets) on/off icon in the toolbar.  If you don't already have those showing, add them as it makes life so much quicker.

This will add a sequential number to each paragraph, ie each additional time you used the return key.

Now for each paragraph that does not need a separate number, move the cursor to that paragraph and click on the toolbar numbering icon; it will turn of the number from that paragraph but leave all remaining numbers still in sequence, ie it will turn number 3 into number 2 if the paragraph between has its own number 2 removed.

Does that make sense?  I hope so, and sorry if that was not your problem.  Perhaps others will benefit even if you are asking about something else.
