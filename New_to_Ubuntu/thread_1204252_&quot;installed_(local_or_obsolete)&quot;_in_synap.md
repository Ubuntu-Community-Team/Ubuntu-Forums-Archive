### Post by CatKiller on 2009-07-04
That just means a package that is currently installed but isn't in the index of packages that's available from the repositories. It could be a package that was available but is no longer distributed (which is why it says obsolete) or it could be a package that you've installed from a .deb file (which is why it says local). Only you can say if you use those packages or not. I know I'd be annoyed if I had libflac7 removed and, to a lesser extent, Opera.
