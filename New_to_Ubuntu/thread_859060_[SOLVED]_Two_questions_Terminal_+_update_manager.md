There's one more way to copy&paste with terminal (also works with everything else in Linux): Just select the text you want to copy, and then middle-click where you want to paste it.

It's so easy I have troubles every time I have to use Windows or OS X as I always try that first couple of times ntil I realise that I need to press the shortcuts from keyboard to copypaste.. :D

To freeze the Flash version you can use Synaptic. Find the package, select it and you'll find the option to freeze the version in Synaptic's menus.
