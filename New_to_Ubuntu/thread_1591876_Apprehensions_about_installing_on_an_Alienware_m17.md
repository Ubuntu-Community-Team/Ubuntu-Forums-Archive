### Post by Softly Adverse on 2010-10-10

I would like to dual boot Ubuntu alongside the current Windows 7 installation. This would allow me to separate my programming environment from my gaming environment, hopefully increasing productivity a bit. Also, by all accounts, Ubuntu seems more suited to coding in general (school has me doing SML right now)

However, a search on these forums revealed some intimidating problems that other people have had with Alienware laptops. Specifically, [this post](http://ubuntuforums.org/showpost.php?p=9737561&postcount=9) talks about the fans not working correctly on an m15x laptop, and [this thread](http://ubuntuforums.org/showthread.php?t=1577092) mentions someone having issues with the 5870 Radeon Mobility graphics cards, which I'm running two of in crossfire.

So my question is: Is installing Ubuntu likely to be a good idea at all, given my situation? It seems likely that there will be many issues cropping up, mostly related to various drivers. Another thing I'm worried about is battery life. Currently I can get about 2 hours of battery out of the computer when I  turn off the built-in "Stealth Mode" which underclocks the graphics  cards and processor to allow for less power consumption. However, if this feature does not work on Ubuntu, I could end up with a programming environment that I can't use during lectures (I study computer science)

I suppose, I should mention that I have never run any type of Linux computer before. Thus, I do not feel all that well equipped to handle difficult problems that might come up (for instance, I wouldn't know how to write any custom drivers to handle these problems)
