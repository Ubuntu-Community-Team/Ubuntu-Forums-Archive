### Post by pythonscript on 2009-10-19
For video editing, I've used cinelerra for a long time. Google it and you'll find the instructions for how to install it (it's not in the traditional ubuntu repos) but it works really well. If you end up using KDE, cinelerra still works, but kdenlive is good and in the repos. You can use kdenlive with gnome, too, but you'll have to install all the base KDE libraries, which you might not want to do if you're primarily using gnome. It's not as powerful as cinelerra, but it's in the repos, which you might prefer.

EDIT:  Both, but especially kdenlive, are really easy to use if you're just interested in basic video editing.
