### Post by sameerds on 2008-08-28
> **lestatius said:**
> I was wondering how much knowledge of the commandline I need before I really understand how to install things on the command line, without just having to google everytime or use the ubuntu searching using a text search "how to install  application x on the command line" or other basic settings? You know what I mean?

You are giving too much credit to the command line. What you should be asking is "what does installing a package mean" and "what are dependencies". Once you get that cleared, you wouldn't need to worry about what is the right command to do it on the command line, and the right program to do it on the GUI. Both are just ways to talk to the system ... what you need to figure out is how the system works, and what you want it to do.
