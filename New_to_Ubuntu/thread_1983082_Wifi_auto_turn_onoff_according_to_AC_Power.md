### Post by blackbird34 on 2012-05-20
Hi there everyone, 
I haven't heard of anything like that either, but what i would do in that case would probably be to install [Jupiter]("http://shuffleos.com/3009/jupiter-a-light-weight-power-hardware-control-applet-for-linux/"): it is mainly an applet for saving battery, but there is also a drop down menu where you can decide how much power you want to use ("on demand", maximum power, minimum power), and change some hardware options like screen orientation and ... Turning your WIFI on and off! :D

(Other than that it might be helpful to install  Ubuntu 12.04 Precise Pangolin as the problem may be solved with the hardware updates it brings (perhaps on a spare partition first to try it out?). the problem might be solved, and if you're staying with Maverick Meerkat to keep GNOME 2 (i did for a while), you can install [MATE]("http://m.omgubuntu.co.uk/2011/08/gnome-2-forked/") on Precise).

Good luck!
