### Post by Blasphemist on 2011-04-28
To help you decide on doing the upgrade to natty, I recommend these.
[http://ubuntuforums.org/showthread.php?t=1741293](http://ubuntuforums.org/showthread.php?t=1741293)
[http://castrojo.tumblr.com/post/4795149014/the-power-users-guide-to-unity](http://castrojo.tumblr.com/post/4795149014/the-power-users-guide-to-unity)

I've been using natty and unity for a few months. It was rocky at first and when it was I'd just login to classic for a while. Now, I love natty and that is because I work more efficiently. It doesn't hurt that it looks cool too. :D

Generally, if you can run the iso from the cd boot, you will know if you will have any video issues. The kernel is updated as are xserver and gnome. Any of these could cause you to need to solve video issues and while you'll see in these forums those issues have come up, most users will not experience them. The same applies to wireless drivers. I have noticed that mainly only with broadcom wireless.

There is the don't fix it if it isn't broke thinking but I prefer to be using the best available and I think that is natty.
