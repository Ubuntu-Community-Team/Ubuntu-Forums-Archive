### Post by SkyNet2029 on 2011-08-05
VirtualBox is the way to go for virtualization support on windows. Excellent support for windows 7 64bit as well (i run OSX 10.6 in windows 7 X64).
While VmWare will do most virtualization tastks, it is rather resource hungry and the newer web interface is kinds of klunky.
VirtualBox sports additions, like mouse-focus and USB support (even USB 2.0!) while being light on the resources. Also, it has oodles of pre-built VM's for download.
It runs natively in either Linux or windows, 32 or 64bit.
Best of all,,, it is open-source to boot!
