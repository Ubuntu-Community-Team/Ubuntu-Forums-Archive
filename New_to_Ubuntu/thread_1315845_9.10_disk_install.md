### Post by QIII on 2009-11-05
Get your money back.  You've been had.  You paid for something that is free for the asking.

Download the .iso from ubuntu.com using a torrent client.

Burn the .iso to CD as slowly as your burner will allow.

Check the md5sum.

Check the disk for errors (an option on start up).

Run the CD in a live session (choose "Try Ubuntu without making changes to your computer" or something like that).

See if all is well before you install.
