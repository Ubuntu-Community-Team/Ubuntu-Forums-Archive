### Post by ibuclaw on 2009-03-11
Just a guess, but it may be because you are using a Development version of WINE, as opposed to the stable release (v1.0.1).

But, as a possible workaround for you, have you heard of x.game?
[http://ubuntuforums.org/showpost.php?p=4348105&postcount=1](http://ubuntuforums.org/showpost.php?p=4348105&postcount=1)
It is a script that essentially creates a new Xserver for your application to run in, so if you want to run it in isolation from the rest of your desktop, you can! without the need to have it in it's own emulated desktop.
