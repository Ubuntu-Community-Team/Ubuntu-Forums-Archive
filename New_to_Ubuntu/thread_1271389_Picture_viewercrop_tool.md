### Post by Omegaham on 2009-09-22
As long as you don't execute any attachments, nothing will get installed. Furthermore, with Linux you need to be root to install stuff. So, if a virus goes to install something, it will flash the "Install requires a password for administrative access" message.

Don't worry about it (unless you do all your stuff while logged in as root, in which case you should hit yourself and make your own user account).
