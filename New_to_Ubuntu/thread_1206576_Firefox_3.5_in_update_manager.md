### Post by Tibuda on 2009-07-07
The new version is available if you install firefox-3.5. It will keep the old version, and will not include the official Firefox branding. This means it will be named Shiretoko, but don't worry about it. If you really don't like the Shiretoko name, then try reading [http://www.psychocats.net/ubuntu/firefox](http://www.psychocats.net/ubuntu/firefox) , but I don't recommend it.
