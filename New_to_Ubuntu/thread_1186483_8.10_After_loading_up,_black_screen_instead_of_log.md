### Post by TJCIB on 2009-06-17
you could try to reconfigure xserver-xorg

Go into tty1 by pressing ctrl+alt+F1

login with your username and password

```
sudo /etc/init.d/gdm stop
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
sudo dpkg-reconfigure xserver-xorg
sudo reboot

```

1st code stops your gui
2nd code copies your current config just incase
3rd gets you into setting up your graphical settings
4th reboots (I like reboots, but you could just restart gdm)

When setting things up, if no drivers work, try the VESA driver

if you mess something up bad you can restore by
```
sudo cp /etc/X11/xorg.conf.backup /etc/X11/xorg.conf
```

I think this works =)
