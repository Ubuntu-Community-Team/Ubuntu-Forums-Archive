### Post by Gone fishing on 2011-11-19
You've simply reinstalled 11.10 that you were using before?

Make sure that you install all the same software, the copy all the files directories etc from the old home to the new home directory; for example /home/olduser/music to /home/newuser/music. To get all you old mail, settings etc you will need to also copy over the hidden files such as /home/olduser/.thunderbird to /home/newuser/.thunderbird
