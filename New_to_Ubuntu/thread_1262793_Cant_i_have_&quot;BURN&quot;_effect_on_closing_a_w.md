### Post by diablo75 on 2009-09-10
First, install Compiz Config Settings Manager (the "advanced" control panel for compiz):

```
sudo apt-get install compizconfig-settings-manager
```

Once installed, click System>Preferences>CompizConfig Settings Manager.

Look for the "Animation Add-Ons" to enable and configure the extra fancy animations.
