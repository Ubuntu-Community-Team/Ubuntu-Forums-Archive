### Post by Joeb454 on 2008-05-20
It's the Live CD, but it allows you to change settings, which will in turn be remembered. So you could - for example, change the background, and install Xchat.

Then the next time you boot the CD, you would still have the same background, and Xchat would still be installed :)

### Post by Golem XIV on 2008-05-20
> **The Jinx said:**
> will this allow me to setup the restricted drivers i need so i can enable desktop extra effects [the beryl/compiz goodness]

Probably, but I wouldn't recommend it.  The purpose of a Live distro is to be mobile, i.e. to be able to run on different hardware configurations.  If you "lock it in" to a specific driver, it may happen that it won't work with a computer that has a different hardware setup.
