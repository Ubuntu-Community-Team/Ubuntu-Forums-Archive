### Post by baruch60610 on 2008-09-04
Hi.  Things are a little different from Windows, so it can get confusing.

To install programs, it's best to use the "Add/Remove Programs" feature in your menu.  Doing it this way, you're letting your system keep track of everything that's installed, and can avoid problems from program conflicts.

Go to your Applications menu, and select the "Add/Remove Programs" option.  That should get you a screen where you see a list of programs to install or remove.  There is a search bar where you can type "superkaramba".  If you do that, you should get that program listed.

Next to that program's name is a checkbox.  Click the checkbox to choose it, then click on the button that says "Apply Changes".  When it's all through, you should have superkaramba in your system.
