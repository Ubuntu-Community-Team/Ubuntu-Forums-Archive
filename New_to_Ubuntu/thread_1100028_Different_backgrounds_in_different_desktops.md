### Post by jtsc on 2009-03-31
Just in case this thread comes up in searches -- Yes, I did get it to work. Here's how...

1. Go into the options for Nautilus and turn off the option for it to draw the desktop (I forget how this works.)
2. Go into CompizConfig Settings Manager, go to Desktop Cube, turn it on, and set it to the number of desktops you want.
3. Go back to the main CCSM window, go to Wallpapers (under utilities), turn it on, and load one image for each desktop.

My only problems... obviously this removes your desktop files, since nautilus draws those, which is much prettier but less useful. Also, the CCSM wallpaper plugin is somewhat finicky about what files it will open... for example, it does .jpg's but not .jpeg's or .JPG's, it will load some absolutely huge files, but not others, and so on. Good luck!
