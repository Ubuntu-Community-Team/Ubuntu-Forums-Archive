Welcome to Ubuntu and Happy Learning! :KS


* If you don't know what version of Ubuntu do you wanna install and use, try Ubuntu 14.04.1 LTS (Long-Term Support) to download .iso file format - [http://www.ubuntu.com/download/desktop](http://www.ubuntu.com/download/desktop)

*** Remember that Ubuntu package is **.deb  **format and not **.exe.** If you want to download and run some Windows file (exe) , Use WINE to get around this. You can download it by using this command after installing Ubuntu.[B][I] Caution - WINE is not an emulator and it will run some files that compatible with it. Please check here for what kind it will run -

[https://appdb.winehq.org/](https://appdb.winehq.org/)[/I][/B]

To install WINE :

Open terminal - Shorcut for CTRL+ALT+T

Then type -

```
 sudo apt-get install wine 
```

Remember that sudo (super user do) is a command to use root priviledges to your system. (Type the code carefully). ***Caution - sudo is a root command, if you tell terminal to delete your filesystem, it will do without asking and happily delete your files. Please take caution whenever you use sudo command.***

### Post by Topsiho on 2014-11-29
I don't think learning (only) Ubuntu is worth any serious effort. It is knowledge that is useful for only a very limited amount of time.
Instead you should learn Linux, e.g. using Linux Fundamentals that you mentioned. The command line.
Knowledge of the command line of Unix / Linux will be useful for decades, as it already has, and is useful on *** any*** Unix or Linux computer system.
Knowing the basics of Linux you can easily learn any graphical interface, be it Ubuntu 4.10 or Ubuntu 10.4, or Fedora, or, you name it. And any application with a graphical interface, for EACH of which you have to learn where the buttons are, and what is supported. And the changes in the next version.

Topsiho
