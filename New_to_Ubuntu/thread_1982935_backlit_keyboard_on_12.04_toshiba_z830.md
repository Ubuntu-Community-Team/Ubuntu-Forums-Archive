### Post by geonrd on 2012-06-04
I'm dual booting factory installed windows 7 64bit, with Ubuntu installed second. I had this problem too, and realized Gurranaki provided a hint.

In windows, I had set the keyboard light to off during the day. When booting into ubuntu I noticed the fn+z key showed an applet, but didn't change the backlight to on. I went into Windows and set the keyboard light to 'timer' then booted back into Ubuntu... now the light is on (timer). Haven't tried the BIOS thing, but I'm sure that works too.

But, the applet still appears in the top right, but doesn't change the keyboard backlight settings.
