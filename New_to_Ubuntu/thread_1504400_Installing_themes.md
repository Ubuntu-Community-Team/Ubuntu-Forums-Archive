### Post by Sealbhach on 2010-06-08
When you install Emerald, you should find an Emerald Theme Manager in your System/Preferences menu. Then when you download an Emerald theme, the file you need to use will be in the form of xxxxx.emerald.

I also install Compiz Fusion Icon, which makes it easy to switch between Emerald and Compiz.
