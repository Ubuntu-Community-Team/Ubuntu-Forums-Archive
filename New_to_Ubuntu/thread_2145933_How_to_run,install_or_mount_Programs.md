### Post by montag dp on 2013-05-16
From the website, it doesn't seem that this program is available for Linux.  There is this quote at the top:

> Brutus is one of the fastest, most flexible remote password crackers you can get your hands on - it's also free. It is available for Windows 9x, NT and 2000, there is no UN*X version available although it is a possibility at some point in the future.

As for why it doesn't work in Windows, well, it is quite old and appears to be available only for Windows 2000 and earlier.  It's probably not compatible with modern operating systems.

To answer the other part of your question, for most software you will need in Ubuntu, you can download it and automatically install it from the Ubuntu Software Center (or other utilities like Synaptic or apt-get from the command line).  There are other ways as well, and depending on the program you may need to unzip or untar it yourself and install it manually, or even compile it from source code.  Most programs will be supported through the Software Center, though.

IF your goal in switching to Linux involves reinstalling and running your favorite Windows apps, then you're setting yourself up for a major disappointment.  Instead, you should be looking into Linux alternatives to your Windows apps.
