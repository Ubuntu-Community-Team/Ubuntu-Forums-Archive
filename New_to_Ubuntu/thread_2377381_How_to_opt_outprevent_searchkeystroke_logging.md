### Post by TheFu on 2017-11-12
Ubuntu stopped sending queries externally. Also, it was only on the Unity Desktop release. 
[https://askubuntu.com/questions/192269/how-can-i-remove-amazon-search-results-from-the-dash-or-disable-the-feature](https://askubuntu.com/questions/192269/how-can-i-remove-amazon-search-results-from-the-dash-or-disable-the-feature)  is how to remove the packages, if you like.

The other DEs (xfce, lxde, gnome, etc) didn't send queries externally to my knowledge.  I wasn't running Unity, so it was never an issue. [https://askubuntu.com/questions/224549/will-xubuntu-also-have-amazon-ads](https://askubuntu.com/questions/224549/will-xubuntu-also-have-amazon-ads)

I haven't worried about this since 14.04.
