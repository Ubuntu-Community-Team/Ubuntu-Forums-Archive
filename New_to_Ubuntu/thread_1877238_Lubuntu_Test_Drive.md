-- This ^

Just to point out: you're not tied to the default LUbuntu set of applications. Almost everything works fine under lxde.

For example, the web browser Chromium is fine, but if you prefer Firefox, which is a bit heavier but not that much, you can install it and eventually uninstall chromium. Or if your system really needs something lighter, go for Midori. Also the gnome office suite (Abiword, Gnumeric, Osmo), or the email client Sylpheed, are nice, but you may like/need Libreoffice, or ThunderBird etc. 

Hope you enjoy LUbuntu. I do myself :)
