### Post by ruffEdgz on 2011-06-21
If you can, I would recommend backing up you system (if its Windows or Ubuntu).

This tutorial found in the **Tutorial & Tips** section of this forum isn't hard to follow: [http://ubuntuforums.org/showthread.php?t=35087](http://ubuntuforums.org/showthread.php?t=35087)

As for 're-installing' windows without losing anything, I know you can repair a windows machine by popping in the installer CD and instead of 'installing', you want to select 'repair' (this option will only show if the installer can see a Windows OS installed presently) and that will repair the base Windows OS only. Installation of any sorts will usually mean 'lose of data' in the Windows world :P

Also, you should also be able to upgrade 10.04 to 11.04 without removing any files by trying the 'upgrade' option found in:

System >> Administration >> Update Manager

Once that 'Update Manager' window opens, you should see something that says: "New Ubuntu Release '11.04' is available" with a button next to it. That should upgrade your system to 11.04.

If you don't take any of the advise above, just take one which is 'backup what you need on any operating system you choose!'. Period.

Cheers!

You don't need a computer science background to run Linux. I've been using linux since the 90's and have no computer science background. What a person needs to remember is that linux is not windows. My wife and I are both in our 60's and linux/Ubuntu is the only OS we run. Some advice in case you decide to give linux a try in the future. 

First, make a clone of your windows system, that makes it easier to restore should you decide to take linux off. You may find that it will take you 10 time longer to install windows with all the driver and programs you have compared to linux.

Second, make sure you download a live cd desktop version to test it to make sure all your hardware works correctly. Usually if everything works from a live cd it will work on your hd.

Third, make sure you read all the install instructions on the web site you downloaded your distro from. It usually provide all the information needed for a basic install.
