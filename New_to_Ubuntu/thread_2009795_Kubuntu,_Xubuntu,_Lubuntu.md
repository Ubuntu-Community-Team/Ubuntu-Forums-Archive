### Post by zombifier25 on 2012-06-24
Xubuntu uses Xfce, which is a lightweight desktop environment. Lubuntu uses an even lighter DE, LXDE. If you want to try them out, you can install them on Ubuntu. Install the package xfce4 (for Xfce), lxde-core (for lxde), log out and choose the respective DE at login.
