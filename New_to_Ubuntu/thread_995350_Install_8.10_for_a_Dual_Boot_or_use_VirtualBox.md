### Post by laurielegit on 2008-11-27
Virtulisation:

Pros: Easy to uninstall, gives experience.

Cons: Performance can be significantly reduced.

Duel Boot:

Pros: Easy to set up, maximum performance, gives a richer experience.

Cons: Not so easy to uninstall, requires a larger hard disk.

I suggest that you try ubuntu in virtual box and then if you like it, duel boot.

Good luck,
