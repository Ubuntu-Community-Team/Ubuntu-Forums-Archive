java Primes  10000
``` and it output a .txt file with all the primes in it called 10000primes.txt I realized that the file was rather large and tried to  compressed it with gzip. when I tried to do it, i got this message:

### Post by mbsullivan on 2010-08-24
It doesn't want to change the file because apparently it's a symbolic link (or has a link to it). Try to create a NEW file with the following:

```
gzip -c file > newfile.gz
```

Alternatively, you can compress using tar with the following:

```
tar -czf file file.tar.gz
```

For the one-file case, you can technically just gip it up with gzip. However, you typically see things distributed as tarballs (.tar.gz), which is alalogous to a zipfile under windows -- it can contain multiple files or directories.
