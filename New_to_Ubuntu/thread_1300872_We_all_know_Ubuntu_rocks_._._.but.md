### Post by markbuntu on 2009-10-26
Microphones etc are very hardware specific problems. The ALSA devs are working hard to accomodate the zillion configurations that hardware manufacturers use and every new alsa version has hundreds of fixes. This is especially true for laptops using hda-intel hardware. There is an extensive list of hardware fixes and how to apply them here:

[http://ubuntuforums.org/showthread.php?t=1043568](http://ubuntuforums.org/showthread.php?t=1043568)
