### Post by LarryJ2 on 2008-06-21
As installed,  the Places menu item (the one between Applications and System) has entries for Videos, Documents Pictures, and so on.  These are actually folder names and exist in your home directory.  For example, if you are logged in as user joe, then the Pictures folder/directory is located at /home/joe/Pictures.

1.  To remove a Places menu entry, click Places -> Home Folder.  In the Nautilus File Browser window, look on the left side under the heading Places.  You may have to click on "Places" if it isn't dropped down.  To remove one of these directories, high light the name with your cursor, right click and select remove.  Note that this only removes the entry under Places.  The directory this entry refers to will not deleted nor moved to trash.

2.  To add an item to the Places menu,  click Places -> Home Folder.  If you see your directory listed, just right click on it and drag it to the Places list on the left side of the file browser. (Under maybe Pictures, Documents Music, etc).   If your entry (directory) isn't shown, create it by doing right click then Create Folder.  That folder can be "dragged" into the Places menu.

3.  These references  to directories (Places)don't have to be in your "home" directory.   For example, you could add the handy folder "/usr/share/doc" 
as "Doumentation"  First use the File Browser to navigate to the/usr/share/doc directory.  Up at the top, right click on the "button" labeled "doc" then drag this down under your list of Places (under for example, Pictures, Documents, Videos etc).  Now the "place" doc will be shown.  Next, Right click on "doc"  then rename it "Documentation"  or some other appropriate neumonic.

Simple? Yes but this is the beginners area!
