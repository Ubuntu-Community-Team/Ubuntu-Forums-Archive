### Post by jetsam on 2010-04-25
You could just pick a letter from a hat filled with the letters d through i and use the corresponding letter of Ubuntu's boot disk to see if it works.  

To avoid randomness, you might want to try Hardy Heron.  I think it's still being supported which means it's receiving security updates.

Try the Xubuntu version of the corresponding distribution to save resources if you don't mind a slightly reduced desktop environment-- XFCE is sort of a lighter Gnome, but it's highly configurable as well if you like digging into those things.

I'd also recommend Puppy Linux if you want to look farther afield.  It's a great live cd to play with and a whole new world.  Also, I've gotten it to boot on almost everything I've thrown it at.

Even if you don't end up using Puppy full time, you should still check it out.  It can sit in a cd case for years and then be ready and happily functioning in the time it takes to find the cd and boot it.
