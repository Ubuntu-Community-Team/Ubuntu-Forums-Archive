### Post by deadflowr on 2014-06-08
Software Updater(the new name for update manager) should add an icon to the launcher, if one does not exist, whenever new updates are available.
If you add the launcher to the sidebar, then when new updates are available it will be marked as such, usually with an arrow on the left side of the icon.

A personal sidenote, from a force of habit, I always launch the updater, manually, whenever I first boot up.
The new updater, automatically runs a refresh to update the archives whenever you launch it.
(The old update manager would simply open and you *would've had to have*(¿) clicked on the "Check" button to get the freshest archive lists.)
