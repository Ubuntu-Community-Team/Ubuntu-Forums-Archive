### Post by yorobot on 2009-07-09
I bought an Eee 900HA with WinXP and installed ubuntu onto it after unpartitioning the disk.  When I installed, I selected the option to install ubuntu side-by-side to windows.

The installation was successful and soon I learned to switch between the PalmPilot GUI (netbook desktop) and the classic ubuntu desktop.

Soon I had a notification that updates were available and this is when problem started.  First, supposedly there was no disk space to download updates (I found then that the ubuntu partitioner had done 2.5G for main ubuntu partition and a bit less for the swap).  I addressed this issue by deselecting the updates I thought I didn't need.  Downloaded updates and all seemed OK.

Didn't use ubuntu in a while but used windows several times.  Next time I logged into ubuntu, the classic desktop I had chosen was there but instead of the red circle with the white horizontal line icon through it (to log off, power off, etc) with my name next to it there was now a little running green guy; also, the menu bar seemed slightly different.  My guess was that the updates did it.  I restarted the Eee Pc and logged back in and to my surprise the old GUI was back (with the red icon with the white line to turn off, etc). Surprising, but it gets worse.

Next time I logged in, after some windows usage, I found I didn't have the wireless networks icon.  Nevertheless, when I switched to the netbook desktop, it was there and I had internet connection (not with the classic desktop); so, I guess from now on bye bye classic.  It got worse.

Today I logged back in (after a couple of days of using windows, normal use, i.e docs,  email, etc) and found that the netbook desktop only displayed system administration menu and options menu: no Applications menu!  Switched to classic desktop and was even worse: no admin or options menus.  Only the clock and the ubuntu logo at the top left.

WHAT IS GOING ON?????

My only hypothesis is that the windows usage is eating away ubuntu hard drive and thus disabling ubuntu functionality chunk by chunk.  But this is just an uneducated guess.

What should I do?  Reinstall ubuntu?  Will it be safe for my windows partition this second time?  how do I know this self-destrutive behavior will not reoccur?
