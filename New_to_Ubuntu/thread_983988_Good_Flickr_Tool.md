### Post by Pezhead on 2008-11-16
I used to use Kflikr on kubuntu but now I am running Ubuntu.  Will a KDE program work in Gnome or is there another program I can use to upload my photos to Flickr?

---

### Post by llamakc on 2008-11-16
You can continue to use any KDE program inside of GNOME. If you have a clean GNOME system, installing the kflickr program will pull in the required packages for it to run without installing the entire KDE system on your machine.

If you search in Synaptic (or with the terminal and the syntax `apt-cache search flickr`) you'll find a few things that could be helpful.

Good luck!
