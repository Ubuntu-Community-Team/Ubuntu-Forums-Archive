### Post by tarps87 on 2008-09-15
You will need to use samba. There is a small gui available in the package manager called samba or a more complex one called gsamba. I suggest samba as it is simple, it should also install the smb protocol bit, you can use it to change the workgroup you are on and manage and samba shares on your Ubuntu system. Once samba is installed it should just be a case of opening the network browser from the places menu and browsing for you windows share
