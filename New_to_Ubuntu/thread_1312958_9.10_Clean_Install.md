### Post by ranch hand on 2009-11-04
If you are using ext4 on 9.04 you could just install (from the CD) on the same partitions but not have the home partition formatted (I would still back it up) and probably leave it in tact on your new install.

A clean install will go a lot smoother than an upgrade.  I upgraded about a dozen 9.04s during testing and most of them worked, but the ones that didn't were a bugger, and they still are.  There are a lot of changes inthe basic works of the system.

A lot of upgrades have trouble booting because of the /etc/X11/xorg.file.  9.10 does not have that file by default (as an example of system differences - pretty much at the base of it here).  That file is generated only if you need it.

Upgrades do leave you with grub-legacy.  It also hoses it about 5% of the time.

Do a clean install and read the link in my sig.
