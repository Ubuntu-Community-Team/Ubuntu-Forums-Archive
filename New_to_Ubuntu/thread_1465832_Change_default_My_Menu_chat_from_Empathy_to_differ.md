### Post by Jon Monreal on 2010-05-01
> **UnknownFear said:**

Yes, I use Pidgin. The only thing that doesn't work is changing status if you go "Offline". You have to open your buddy list and select a different status to come back online.

Otherwise, you'll see notifications of people logging in/out, new messages, be able to set your status, can close the buddy list with Pidgin still running, etc.

After you install Pidgin, I would recommend uninstalling Empathy.
