### Post by anotherdisciple on 2008-09-08
most of the codecs can be grabbed all at once by opening a terminal (Applications > Accessories > Terminal) and typing:

```
sudo aptitude install ubuntu-restricted-extras
```

Some countries have laws against this type of thing... you can read more about the restricted extras [[COLOR="Blue"]here[/COLOR]]("https://help.ubuntu.com/community/RestrictedFormats")
