### Post by BQAggie2006 on 2009-10-01
I used to work with GX240s during a previous job and they are definitely not 64-bit machines, so you should probably download the 32-bit ISO. Intel did produce some 64-bit Celerons and P4s, but it was the LGA775 and the GX240 is Socket 478 if I'm not mistaken.
 
You can also always try to install the 64-bit version and if it's not a compatible processor it's not going to install.
