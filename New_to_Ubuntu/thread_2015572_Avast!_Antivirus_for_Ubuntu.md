### Post by cipherboy_loc on 2012-07-03
Standard disclaimer about Linux not needing an AV unless you are moving files between various Windows systems aside...

Just go to the website ([http://www.avast.com/linux-home-edition](http://www.avast.com/linux-home-edition)) and download the .deb file. These days you can graphically install it, but if you want to do it from the terminal,
```

sudo dpkg -i /path/to/avast.deb

```
