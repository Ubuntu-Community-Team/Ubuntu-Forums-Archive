### Post by zkriesse on 2009-11-03
First it depends upon if you plan on running the flash drive like an actual hard drive. Second, 2 GB is hardly enough for that action. But yes, you would have a menu after hitting F12 during the booting process which would ask you if you would like to boot from the hard drive or the thumb/flash drive.

### Post by Matt__ on 2009-11-03
you could run both ubuntu & UNR as if they were liveCDs from a 2gb partition each, but a full install of either (using USB as an external HDD) would take most of the 4Gb for either OS and leave little room for additional data/program packages.
