### Post by bodhi.zazen on 2011-04-06
There is a bit of a learning curve.

What you have is an iso that needs to be "burned" to a CD.

If you do not have an application on Windows to burn iso, the iso, windows then reports the iso as an archive.

See:

[https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)

It does get easier, but installing an OS is a fairly major undertaking and you will almost certainly need to do some reading.

Gneeral information:

[https://help.ubuntu.com/community/Installation](https://help.ubuntu.com/community/Installation)

Graphical install:

[https://help.ubuntu.com/community/GraphicalInstall](https://help.ubuntu.com/community/GraphicalInstall)

Wubi:

[https://help.ubuntu.com/community/Wubi](https://help.ubuntu.com/community/Wubi)

wubi can be used as a program (wubi.exe) that then installs Ubuntu, see the wiki for details.

[http://www.ubuntu.com/desktop/get-ubuntu/windows-installer](http://www.ubuntu.com/desktop/get-ubuntu/windows-installer)
