### Post by vanadium on 2018-04-07
Important to note first is that you need the extension "User themes" installed before you can change the look of the shell with a custom shell theme. Secondly, Ubuntu dock and dash to dock are quite the same under the hood. Thus, both will respond the same with respect to shell theme changes. However, after installing Dash To Dock, you will have a lot of customisation options, including several options to determine the appearance including color, transparency, etc. So if theme changes do not work as expected, there is still a lot to tweak there manually so you can obtain your desired look better. These settings will be applicable to your newly installed "Dash to dock" extension *and* the Ubuntu one. Make sure one of the two extensions is switched off.
