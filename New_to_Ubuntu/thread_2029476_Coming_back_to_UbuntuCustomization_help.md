### Post by z3nhakr on 2012-07-19
if you want to customize ubuntu, try installing kde. you can do this with "sudo apt-get install kde" and then select it as your desktop manager on your login screen. from there you can customize almost anything you want and there are tones of apps to help you with it. check software center.
an alternative to that is gnome which is even more customizable (2 is. i dont know about 3) but it doesnt look as pretty
OR you can try to customize unity. as far as i know there are only 2 tools for this, one of which is the compiz settings manager.
