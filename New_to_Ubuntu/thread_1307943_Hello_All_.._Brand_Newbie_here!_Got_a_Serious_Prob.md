### Post by Merk42 on 2009-10-31
Did you install the recently released Karmic Koala?
That comes with GRUB2, if so, at the prompt, try running ```
update-grub
```
or ```
update-grub2
```

Also general help on GRUB2 is [here](http://ubuntuforums.org/showthread.php?t=1195275)

EDIT: Also check this thread.
You may need to ignore a lot of the solutions as they deal with menu.ls which is how the **old GRUB** did things. Not the way the **new GRUB 2** in Karmic does things.
