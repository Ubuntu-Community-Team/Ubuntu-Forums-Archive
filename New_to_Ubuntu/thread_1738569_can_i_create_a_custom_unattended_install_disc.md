### Post by earthpigg on 2011-04-24
Install Ubuntu 64 in virtualbox, configured exactly how you want it with the minecraft server stuff and everything.

use [remstersys]("http://geekconnection.org/remastersys/") to create a dvd .iso.

your buddy should then just be able to offer mostly/all default answers to the installer's questions and be good to go.
