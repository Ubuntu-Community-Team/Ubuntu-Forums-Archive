### Post by decoherence on 2011-06-15
Hello!

There are generally two ways to run software. Well, three ways I guess now that Ubuntu has introduced another interface.... aaannnyway.

1. If you are running classic gnome, most software should appear under the Applications menu. If you don't see an applications menu then you aren't running classic gnome and you can ignore 1.

2. If you are running Unity (you probably are if you have an aspire one) try pressing the windows key then typing the name of the software. Once the software is running, its icon should appear in the bar to the left the appears when you press the windows key. If you right-click that icon you should get an option to keep it in that bar for 'easier' access. (personally i find typing the name of the program easier but whatever floats your boat!)

3. Some software does not have a graphical interface. Software like this (and any other software for that matter) can be run from the command line. Press the windows key and type 'terminal' and run the Terminal program. In the window that comes up, type the name of the program you installed. Often you can type the first few letters of the program and then press tab and it will fill in the rest (or as much of it as it can.)

hope that helps,
