### Post by Temposs on 2010-01-20
If you're using a netbook, you might try the Ubuntu Netbook Remix version of Ubuntu, instead of Xubuntu. The interface is designed to work well on a small screen. It's pretty nice :-)

Ext4 is the current version of the ext filesystem. Some people say ext4 is still a bit unstable, but I haven't had trouble with it. Ext4 is also supposed to be faster. Ext3 is known to be stable and works well also.

Ubuntu comes with a torrent program called Transmission. It should be in the Internet section(although I don't know if it is in Xubuntu).

### Post by Temposs on 2010-01-20
Well, it should be:

```
sudo apt-get install ubuntu-netbook-remix
```

Back up your important files before you run that. If they haven't perfected the transition from Xubuntu to Ubuntu Netbook Remix, you don't want to be left with a broken GUI and having a difficult time doing anything.

Alternatively, you can download the Ubuntu Netbook Remix as an ISO and reinstall your system. Here are a couple download links:

[http://releases.ubuntu.com/9.10/ubuntu-9.10-netbook-remix-i386.iso](http://releases.ubuntu.com/9.10/ubuntu-9.10-netbook-remix-i386.iso)
[http://releases.ubuntu.com/9.10/ubuntu-9.10-netbook-remix-i386.iso.torrent](http://releases.ubuntu.com/9.10/ubuntu-9.10-netbook-remix-i386.iso.torrent)
