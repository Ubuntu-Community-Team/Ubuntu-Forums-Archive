### Post by crossedout on 2008-05-28
Usually, all you need to do is drag and drop the .tar file into the Preferences->Theme window.
If you aren't able to do that, than extract your theme archive, then navigate to your /home folder and do the following:
ctrl+h (to show hidden folders), navigate to .themes folder and drag/drop your theme folder (the one you extracted) into .themes.  Open up Preferences->Theme window again and it should be listed there.
