### Post by Frogs Hair on 2013-01-14
The Windows software won't work with Ubuntu. Open printing and select add and after Ubuntu locates the printer it will search for a driver and then follow the instructions in the dialog box.

HP is well supported but there may be additional steps for some Cannon and Brother printers. Search the forum because driver and installation questions have been addressed on other threads.

### Post by d4m1r on 2013-01-14
What kind of printer is it (manufacturer + model)? Go to the official website for that printer, and see if they offer linux drivers for it. They might not however. If it is a network printer (wireless for example), chances are it will work just fine and possibly without product specific drivers....

My 2008 Lexmark Wireless printer worked fine under linux without me having to install anything....It uses the built in CUPS interface.
