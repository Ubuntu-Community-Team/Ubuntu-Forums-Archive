### Post by EssexEagle on 2009-10-18
You could use the [Stylish add-on for Firefox]("https://addons.mozilla.org/en-US/firefox/addon/2108") and create a style for the websites you'd like to change (you need to know how to write CSS, or else check the [userstyles website]("http://userstyles.org/") to see whether anyone's already written a style which does what you want).
