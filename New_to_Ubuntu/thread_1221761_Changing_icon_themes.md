### Post by civillian on 2009-07-24
you should be able to just copy the icons in their folder (icontheme for the purposes of this post) to the directory ".icons"
You can do this in the GUI by opening up nautilus, pressing ctrl-h to show the hidden folders (folders with the "." prefix) and dragging icontheme folder into the .icon folder 

hope this helps :)
