### Post by wildmanne39 on 2012-04-28
Hi, try this please, if it works we will need to make it permanent so do not reboot:
```
sudo ifconfig wlan0 down
sudo rmmod -f iwlwifi
sudo modprobe iwlwifi 11n_disable=1
sudo ifconfig wlan0 up
```
Thanks

---

### Post by LorenzC on 2012-04-29
Super! This fixed it! Thank you very much! I googled the iwlwifi 11n_disable=1 command and did the following: ```
sudo touch /etc/modprobe.d/iwlwifi.conf
sudo gedit /etc/modprobe.d/iwlwifi.conf &
```Then I added the line:```
options iwlwifi 11n_disable=1
```After reboot internet still worked so I guess I will be all right. If this wasn't the correct way to do it of if there is a better one,  feel free to tell me.
