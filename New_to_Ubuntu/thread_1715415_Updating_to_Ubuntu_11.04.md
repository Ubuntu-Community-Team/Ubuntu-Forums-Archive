### Post by mathgeek2000 on 2011-03-26
> **Dark_Stang said:**
> While I applaud your enthusiasm for the upgrade process, it is very problematic.

As with all OS upgrades, backup anything important before hand and be prepared to reinstall if it fails.

As of now, if you type :

```
update-manager -d

```
At the bash shell (Command Prompt) you will get an extra option to upgrade. to alpha 3, I suppose. When 11.04 is released you'll get that option automatically. -- but it is optional 10.10 will still be supported for some time.

In the past, I've had trouble with in-place updates, for a few reasons, and I've gone with clean installs. -- but I take the time to setup what I need, youtube vids, commercial dvd video for example.

Meanwhile - for those who want to update some key apps:
LibreOffice 3.3.x & Firefox 4.0 are available for 10.10
