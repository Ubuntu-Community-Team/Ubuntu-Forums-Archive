### Post by jbrown96 on 2008-11-03
I think you have the opensource flash player installed. You need to get rid of it and install the proprietary flash player. Go into synaptic, make sure you have all of the restricted repositories enabled. search for flash. If there is something like gnash or swf... installed, remove them completely. I believe the package you want is called flashplugin-nonfree or something similar. Install that and restart all your firefox windows. You can also install ubuntu-restricted-codecs to get all your multimedia codecs at once (including java and flash plugins) but you should still uninstall gnash and swf... first
