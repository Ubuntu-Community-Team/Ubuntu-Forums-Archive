### Post by epic.mint on 2012-01-17
command line file moving is done with the "mv" command so move rfom desktop to home would be ```
mv /user/username/Desktop/filename /user/username/
```a shortcut for /user/username is ~ so /user/username/Desktop can be written as ~/Desktop

edit: C drive is called root and is directory wise just an / this is "where the directory tree begin"
