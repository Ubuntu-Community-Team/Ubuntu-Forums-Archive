### Post by sdennie on 2008-07-05
I believe Flash Cache is a disk cache that some Intel machines have.  I have the same laptop and the Flash Cache was configured just as you've described and I had no problems leaving it like that.  I've also had no problems since I've disabled it in BIOS.  So, if you don't have it, it doesn't seem to make a difference if you have it enabled or disabled.
