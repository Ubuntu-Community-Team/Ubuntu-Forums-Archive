### Post by Elfy on 2012-03-03
Edit grub file after backing it up and run the update grub command

```
sudo cp /etc/default/grub cp /etc/default/grub.bak
gksudo gedit /etc/default/grub
sudo update-grub
```

[https://help.ubuntu.com/community/Grub2#Configuring_GRUB_2](https://help.ubuntu.com/community/Grub2#Configuring_GRUB_2)

Mine shows with a 3 second timeout
> 
GRUB_DEFAULT=0
#GRUB_HIDDEN_TIMEOUT=0
GRUB_HIDDEN_TIMEOUT_QUIET=true
GRUB_TIMEOUT=3

Grub will only show by default if there are more than just one OS.
