### Post by whitethorn on 2008-08-12
I usually use Mplayer when I want to see a movie with subtitles.  To get subtitles to work, just copy the .sub/.idx into the same folder as the avi, and rename the file to be the same as the .avi  ex.  foobar.avi -> foobar.sub.  Start the movie and press j to switch between subs .  Don't know how to do this with vlc though. :)
