### Post by snowmobiler on 2014-09-29
I'm new to Ubuntu but I would check a couple things. First the Bios. I installed on a Dell Latitude which has some EXTRA wireless functions like bluetooth, wifi catcher, an wifi switch (for internal wifi not supported by ubuntu), etc. I had lots of code showing up on boot up and it woudn't shut down completely until I disabled all of those. Maybe all don't need to be disabled but I don't need them so it was better to do all than one at a time. Then I would check your power settings to see if there is something odd about when the laptop is on battery.
