### Post by petronell on 2008-09-02
Hello and welcome to Ubuntu.

Download the CD and burn it in Windows using Nero (or something similar).

Then insert it and boot the PC off the CD. This will allow you to use Ubuntu without doing anything to your hard disk, so there is no risk to your data. 

If you like what you see I would recommend building Ubuntu using Wubi at first as again this will not risk your Windows installation. For information on Wubi take a look on this link.

[http://wubi-installer.org/](http://wubi-installer.org/)

Once you are a bit more comfortable with things you can consider a dual boot system or even dropping windows all together.

It took me a couple of months before I saw the light and dropped Windows all together.

Good luck, and always post if you need a hand. It is a very friendly community and I have been helped on more than one occasion. People here believe no question is too stupid.

daveR
