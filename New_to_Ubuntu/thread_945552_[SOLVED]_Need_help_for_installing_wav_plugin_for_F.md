### Post by ajgreeny on 2008-10-12
You will not be able to install most plugins for firefox using the firefox plugin manager; that is OK for windows but not for Linux.  You need to activate the medibuntu repos [https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu) and then install with synaptic or apt-get.  Read all the link I sent and it will explain everything.
