### Post by tobeno on 2008-06-25
Is there a Linux equivalent for the key combinations using the Alt and the righthand placed numerical keys? Under that platform you can hold down the Alt-key and type in the underlying value/(ASCII-code?) for a character, up to 255. When releasing the Alt-key, the character pops up. I learned to use it as an MS-DOS user in the late 80's, and miss this simple feature.

---

### Post by Dr Small on 2008-06-25
Something similar is:
```
CTRL + SHIFT + u
```

After pressing those keys, then type in your number.
