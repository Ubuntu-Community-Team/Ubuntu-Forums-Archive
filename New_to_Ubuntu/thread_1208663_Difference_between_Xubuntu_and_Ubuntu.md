### Post by Wim Sturkenboom on 2009-07-09
From what perspective? Hardware requirements are a bit lower on Xubuntu and they come with different window manager (xfce versus gnome) and different applications for a number of tasks. What I can remember (Dapper Drake) is e.g. gnumeric and abiword versus openoffice and if I remember correctly thunderbird versus evolution.

FYI: you will be able to install e.g. openoffice and evolution if you need them.

### Post by Revolutionary101 on 2009-07-09
Xubuntu is not that much different than Ubuntu. Xubuntu uses the Xfce desktop enviroment. Ubuntu uses the Gnome desktop enviroment. Both look almost exactly the same.

Xubuntu is designed to be used on slower computers.

Xubuntu uses Thunar for a file manager instead of Nautilus.

Most if not all programs that work on Ubuntu work on Xubuntu.

Hope this info was useful.
