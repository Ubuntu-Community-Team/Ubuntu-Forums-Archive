### Post by PhantomTurtle on 2012-06-16
You can sync music using Clementine or any other music player that supports it. However syncing apps is something you cannot do without iTunes. You can try iTunes through Wine if you want to sync apps(not sure how well that would work though). Or you can use iTunes in a Windows virtual machine.
