### Post by coffeecat on 2009-12-15
The updates are mostly bug-fixes and security patches - just as in Windows and MacOS. When using Update Manager, highlight one of the updates and click the drop down for "Description of Update". That'll tell you what's going on.

The main difference between Ubuntu updates and Windows updates is that Windows updates are for the OS only. Ubuntu updates will include anything for your installed apps as well as the OS.

The libraries are vaguely equivalent to Windows .dll files. The main practical difference being that Linux libraries will be used by multiple applications. If a function is needed, chances are that a library exists to carry out the function so the app developer uses the library rather than reinventing the wheel. In the Windows world, most apps are proprietary so developers spend a lot of their time re-inventing the wheel. The libraries will also be updated if an update is made available.
