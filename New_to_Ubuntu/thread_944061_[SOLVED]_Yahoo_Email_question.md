### Post by jamesrl on 2008-10-12
Webmail is when you start your webbrowser, like firefox, and then go to [http://mail.yahoo.com](http://mail.yahoo.com) and login to check your email.  F11 should switch to full screen by this method (or by going to _V_iew menu and down to _F_ull Screen will select full screen).

An email client is a separate program where your start up a specific email application which downloads your mail from a web server.  Each email client will have different settings, and some (for example evolution) don't seem to have full-screen modes, though you can press Alt-F10 to maximize the window.  If there is a full screen mode, it would likely be accessible from the View menu.

### Post by jamesrl on 2008-10-12
Have you tried the new yahoo mail?  (I use gmail primarily, but have a yahoo account just for fantasy sports).  I just logged in and by default it sent me to the old version, I got a very small text box to type a new message into.  However, with the new version of yahoo mail, it is nearly full screen by default.  

So at the very upper right of your screen, you should see something like
"Hi (your user name) | Sign Out | All-New Mail | Help"
Click on All-new mail, and see if that's more to your liking.  You can always switch back to "Classic Mail"
(which could be selected in the upper left with options "Sign Out, My Account, Mail Classic").
This seems to be less of a linux vs windows question, but a yahoo mail problem (as firefox should interpret a web page the same way regardless of being in windows or linux).
