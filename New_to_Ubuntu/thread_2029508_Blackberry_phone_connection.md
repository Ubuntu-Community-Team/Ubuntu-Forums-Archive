### Post by levlaz on 2012-08-01
What are you trying to edit?

The blackberry software that you are probably used to using in Windows does not work in Ubuntu. 

A good alternative is to install a virtual machine with Windows inside of Ubuntu and use that when you need to work with your blackberry. 

If you are trying to access the files on the sd card this is definitely possible and we can help you with that. 

If you are trying to edit anything inside the blackberry software like contacts, calendar, etc.. the best thing to do would be to setup a google account to sync with the blackberry so that you can edit all of that information through the web, or if you have exchange on the blackberry use the Outlook Web App to edit all of this type of information.
