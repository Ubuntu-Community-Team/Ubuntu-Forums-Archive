### Post by switch10 on 2010-02-20
Put in the LiveCD, and choose install Ubuntu.  There is an option to use the entire disk, when you get to the part in the install when you are partitioning the disk.  If you use the entire disk, it will erase all windows partitions, and Ubuntu will install over it.

### Post by stoogiebuncho on 2010-02-20
> **summersource said:**
> thx switch but it's not giving me any options for some reason with win 7

On a lot of computers you have to press the F12 key or the F8 key right after you turn the computer on in order to get the boot menu.  

It should tell you during the boot up process.  Right after you start (or restart) the computer, look for something that says "Press X for boot options" or something like that.  You'll have to be fast, it's usually only up there for a second.

If you know how to get into your bios settings, you can probably also change your boot order there.  Set it to boot from the CD drive first.
