### Post by Didius Falco on 2009-05-04
Open the Synaptic Package Manager from **System/Administration/Synaptic Package Manager **and do a search for **Alien**, which is a program to do just that.

Here are some instructions on how to use it: 
[http://www.ubuntux.org/how-to-use-rpm-files-need-help-please](http://www.ubuntux.org/how-to-use-rpm-files-need-help-please)

While it might be necessary, occasionally, to convert an rpm file, generally you can find the same thing in a repository that you can add to your Synaptic, or even already in  the Synaptic file repositories.

Failing that, you can usually find a *.deb file (which stands for Debian - that's the branch of linux that Ubuntu is in) of the same package.

I hope this helps...

Regards,

Didius
