### Post by fuquaforlife on 2010-06-22
Is there a simple way to do a slideshow in a window (not full screen)?  I.e., I want to have a small window in the upper right corner that rotates through pictures.

I've looked in a lot of places, and can't find anything that isn't full screen or a widget.  I'd rather not install a whole widget framework for one thing. Thanks!

### Post by lkjoel on 2010-06-22
Try in a Terminal window (Applications->Accessories->Terminal)
```
sudo apt-get update
sudo apt-get upgrade
sudo apt-get install gwenview
gwenview
```
You can view pics with it, and you can do a slideshow.
