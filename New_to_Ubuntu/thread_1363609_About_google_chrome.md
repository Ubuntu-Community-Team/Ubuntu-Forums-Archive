### Post by lovinglinux on 2009-12-24
> **arochester said:**
> [https://chrome.google.com/extensions](https://chrome.google.com/extensions)

Extensions are not the same as plugins. 

The plugins for Chrome are the same as the plugins for Firefox. Just install [mozilla-vlc-plugin](apt:mozilla-vlc-plugin) and restart Chrome. It should detect the plugin. If you want to check it out, type the following in the address bar:

```
about:plugins
```

Anyway, I would recommend [gecko-mediaplayer](apt:gecko-mediaplayer) plugin isntead of vlc.
