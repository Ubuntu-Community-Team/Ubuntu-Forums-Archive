I'd suggest you start with an "inside windows" one, after which you can migrate it to a proper "partition" based install anytime.

The risk with this is you can lose all your ubuntu stuff if your Windows XP goes down.

Using the live CD first was a good, sensible move. As is your choice to try Ubuntu.

As for the overheating, it really shouldn't be an issue; I've been using Ubuntu since 7.10 on my Fujitsu laptop without problems. However, it can't hurt to be alert about temperature issues for a while.

### Post by Kevbert on 2009-06-11
> **pous said:**
> what difference is there between xubuntu and ubuntu??

actually i did normal ubuntu, and seems as fast or more so than xp, so no complaints on it, anyhow i'm soon adding a few things to my laptop and first on the list is +ram... thanks for all though!

Xubuntu is a cut down version of Ubuntu. The basic package uses less resource hungry programs and is based on XFCE as opposed to Gnome.  You can however install virtually anything in Xubuntu that you can in Ubuntu such as OpenOffice but that defeats the object.
Xubuntu will work quite fast with 512Mb RAM, Ubuntu less so, so if you stay with Ubuntu, the more memory the better. Compiz (the desktop cube) will slow the system down a lot, so keep desktop effects to a minimum, until you increase the memory.
