### Post by Sir Jasper on 2010-03-10
Hi,

I like the Firefox add-on Default Full Zoom Level. Link below:

[https://addons.mozilla.org/en-US/firefox/addon/6965](https://addons.mozilla.org/en-US/firefox/addon/6965)

It remembers the size you set for each page (which is easily adjustable).

If you do not like it then your text size settings can be altered.
