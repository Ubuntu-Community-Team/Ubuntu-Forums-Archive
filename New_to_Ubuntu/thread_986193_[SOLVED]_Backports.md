### Post by phidia on 2008-11-18
The Ubuntu wiki has [some reading]("https://help.ubuntu.com/community/UbuntuBackports") on backports.
You don't need to do anything unless the only way to get a package you want is by using backports and if that were the case you would need to enable backports. It's greyed out because presently you don't have backports enabled-see the link for more info.
