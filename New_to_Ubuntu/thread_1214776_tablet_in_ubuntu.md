### Post by Favux on 2009-07-16
Hi akimatsu123,

You've run into a "bug" with Jaunty's 10-wacom.fdi.  It doesn't parse the HAL names correctly into linuxwacom names.  To see this in a terminal type:
```
xsetwacom list
```
that should return stylus, eraser, and touch.  To see what HAL is calling them enter:
```
xinput --list
```

Read Jaunty Users near the top here:  [http://ubuntuforums.org/showthread.php?t=1038949](http://ubuntuforums.org/showthread.php?t=1038949)  You may want to try 3a.  I think rec also had a X200t.
