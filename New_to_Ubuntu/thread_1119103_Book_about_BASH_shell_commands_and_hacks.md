### Post by t0p on 2009-04-07
There's the **Advanced Bash Scripting Guide** available through the repos.  Summon the control line and type in:

```
sudo apt-get install abs-guide
```

The guide, in html format, will appear at /usr/share/doc/abs-guide.  I recommend it.
