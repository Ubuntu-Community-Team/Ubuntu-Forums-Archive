### Post by kroq-gar78 on 2010-08-20
did you use wubi to install ubuntu?

if you did, all the files on the windows C: drive will be in the /host folder.

If you installed using the live cd, then it would be something under the /media folder, where all external storages are stored. You must mount the hard drive first (at top-right, click on places => windows hard drive (whatever it's called. it might have a bunch of letters and numbers, like "SQ100562......" or something similar)). Then you can access all your windows stuff.
