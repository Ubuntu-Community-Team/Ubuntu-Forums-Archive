### Post by SamNSane on 2009-03-26
If you check mail in the account via the Web interface, all you have to do to make the message available for download to the POP3 client is to use the browser to mark it as unread.

If you check the mail in Thunderbird or Evolution, you can set the account so that mail is left on the server until you delete it.
