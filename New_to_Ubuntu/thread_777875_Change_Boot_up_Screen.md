### Post by por100pre1 on 2008-05-01
Install Startup Manager to change those screens:

```
sudo apt-get install startupmanager
```

Look here for some splash screens:

[http://www.gnome-look.org/index.php?xcontentmode=160](http://www.gnome-look.org/index.php?xcontentmode=160)

### Post by Sillywombat on 2008-05-02
Certain motherboards allow you to change start-up screen in the same way as they're bios. (for some reason if recall there are called ROMs, but i just cant fully recall)
HOWEVER, it isnt easy, the picture is rather bad, and it can be VERY DANGEROUS! I tried to change my start-up screen a while back and managed to completely destroy a motherboard at the same time.

SO, coming from me, if you want to try it, go for it, but keep in mind my principle philosophy - "if it ain't broke, dont fix it"
Good luck though.
