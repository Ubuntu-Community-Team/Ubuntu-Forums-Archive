### Post by I2k4 on 2011-05-08
My first gen Acer One has run every version I've tested on Live USB with immediate recognition of all the hardware, except for Xubuntu 10.10 failing to see the hard drive. I found the Unity interface of 11.04 and the 10.10 Netbook Remix very slow on my machine, though they worked.

You might also want to compare user forums for the two netbook models - the Acer forum has been very helpful to me both running Windows and Linux, and of course you also get any common beefs from any unhappy users there:

[http://www.aspireoneuser.com/forum/](http://www.aspireoneuser.com/forum/)

I recommend testing Ubuntu versions with Persistent LiveUSB for a while, since some are more new-user friendly with lots preinstalled (Ubuntu Desktop 10.04 or 10.10) others quicker and slicker (Lubuntu 10.10) on a weak netbook.
