### Post by lisati on 2009-05-15
Common default user names include something like "admin" or "administrator". Common default passwords include "password". Change the defaults a.s.a.p. after resetting!

You might want to look into setting up WPA security for the wireless portions of your system.

You might also want to look into setting up MAC address filtering to limit which machines are allowed to connect. Although it's not foolproof, it's an extra piece of information that someone will need to know to be able to connect.
