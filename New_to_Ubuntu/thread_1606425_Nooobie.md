### Post by AstroLlama on 2010-10-26
Phil3006 I've been using Ubuntu since 2004 and I also have minimal computer experience. The best thing to do is start by doing the basic install, then when something happens or you see something that you can't do, sarch for it on the forums or in a search engine and follow the best directions to solve your problems. Little by little you will learn more about commands, folders, and the real power behind linux.

Ubuntuguide.org is very helpful, as the previous poster mentioned.
also, [Full Circle Magazine]("http://www.fullcirclemagazine.org/")
and [Ubuntu Geek]("http://www.ubuntugeek.com/")

One tip, if this is going to be your main computer be sure to keep backups on a hard drive, because to be honest depending on the things you tinker with there is risk that you will have to reinstall. Most of the easy customizations in the beginning are not at all risky, however.

once you get the basics you can learn a lot from installing "eyecandy" ie. fancy effects, different icons, experimental panels, etc... 
the most important thing is: have fun!

### Post by Geoff918 on 2010-10-27
> **phill3006 said:**
> Also, is there any essential software that may not be included on Ubuntu?
I haven't had a problem opening any files yet, but I know I need to install some form of java.

Ubuntu is designed to have all the software a typical user would want already installed. However, some useful items are not included due to licensing / legal issues.

I would recommend the following:

Open a terminal window (Applications -> Accessories -> Terminal)

And do the following:

1) sudo aptitude update
(will prompt for your password)
2) sudo apt-get install ubuntu-restricted-extras
3) exit

You should be able to get useful things like flash from that. (Also some Microsoft fonts, etc.)

Apart from that, you'd have to let us know what sort of software you need that's beyond email, chat, web browsing, file sharing, image viewing/editing, office software, etc. that is installed by default.
