### Post by snowpine on 2008-10-31
Try Ubuntu, it should work fine with 512mb of ram. If you find it's too slow, you can add Xubuntu without reinstalling by opening a terminal and typing:  

```
sudo aptitude install xubuntu-desktop
```

Then log out and from the login screen choose options->sessions->Xfce to switch into the Xubuntu desktop.

### Post by JohnyN on 2008-10-31
I was working several months on Pentium II 433 MHz and 576 MB RAM with Ubuntu 7.10 Gutsy Gibbon. I think it was GNOME 2.20 and it was fine. But some applications were slower than on new machines.. eg. OpenOffice Impress with big presentations, Firefox on pages with too much of flash animations. So, I've used Flashblock addon for Firefox.
I think that in comparison with Win98, there wouldn't be any problems with power.

However,if it will be slower than you expect, than install xubuntu meta package, which will make from your ubuntu Xubuntu. Xubuntu uses XFCE, which is smaller than GNOME, but I think it can be compared to Windows XP power and GUI. Much better then Win98 GUI.
