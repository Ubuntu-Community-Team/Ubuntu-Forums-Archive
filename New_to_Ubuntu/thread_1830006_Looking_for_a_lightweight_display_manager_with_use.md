### Post by frankbooth on 2011-08-21
[LightDM]("http://www.freedesktop.org/wiki/Software/LightDM") might suit your need.

It has been chosen as default for Ubuntu & Xubuntu 11.10, pretty sure Lubuntu will follow. 
Lubuntu currently uses [LXDM]("http://wiki.lxde.org/en/LXDM") and is very lightweight as well.
