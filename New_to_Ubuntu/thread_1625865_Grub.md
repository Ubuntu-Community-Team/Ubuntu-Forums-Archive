### Post by ajgreeny on 2010-11-19
If you simply want or need to boot to the earlier kernel you can choose it from the grub menu.

If that menu does not appear at boot time either press **esc** at the start up if using legacy grub, or **shift** if using grub2.

With those two kernel options it should be grub2, hence the shift key, but if you updated from a previous version of ubuntu, eg 9.04, it could still be legacy grub.
