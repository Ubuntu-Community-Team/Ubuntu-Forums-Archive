### Post by DaveGalt on 2012-02-28
I know this post is a bit old, but in case this is helpful to anyone else, there is a very interesting solution to this problem at [www.readspeeder.com]("http://www.readspeeder.com").  The site is free, and it displays text in 'thought-units', in other words, in short meaningful phrases.  This is actually a speed reading course, but there is an older version which may still be available at [www.readspeeder.com/v1]("http://www.readspeeder.com/v1") which I think is closer to what you want; that is it allows you to set the speed of the text.  The best thing about this site, and why it's better than sites like spreeder, is that it does more than just show you a certain number of words at a time, but it intelligently displays meaning word-groups.
