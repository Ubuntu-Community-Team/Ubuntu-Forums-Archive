### Post by Temposs on 2009-12-24
It looks like there's a linux project for your micro controller:
[http://bstamp.sourceforge.net/install/](http://bstamp.sourceforge.net/install/)

There is a tutorial to install it:
[http://bstamp.sourceforge.net/install/](http://bstamp.sourceforge.net/install/)

But, those instructions aren't exactly how you should do it. You should read the Ubuntu tutorial on how to compile software. The instructions in this link below will also help you install robotools and simbad:
[https://help.ubuntu.com/community/CompilingSoftware](https://help.ubuntu.com/community/CompilingSoftware)

Note that there's no "configure" step in the case of the bstamp software.
