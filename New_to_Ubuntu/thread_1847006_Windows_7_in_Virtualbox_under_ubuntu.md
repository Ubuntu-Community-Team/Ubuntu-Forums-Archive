### Post by snowguy on 2011-09-20
Hi. I am thinking of getting a new laptop and of course I'll want to install Ubuntu on it. In the past when I have done that I have just wiped out the windows and do a clean ubuntu install. I love it...but inevitably I find there is some program I want to run under windows. And then the way I handle that is I put windows on virtualbox and run it under there. Works great for my needs. But to get there, when I did it a few years back, I had to get some (questionably legal) version of windows to load under virtual box because I had wiped out the windows version that came with my system. I justified this practice to myself by saying to myself "you are cool as you already paid for windows for this very machine." Made sense to me 

But, this time I want to do it right from the get go. And I still prefer to run everything under ubuntu vs dual-booting into windows & ubuntu. 

With that in mind, can anyone point me to a guide for how to take the windows 7 which comes with the machine I buy and put in in VirtualBox so I can run it that way under ubuntu? I figure this desire isn't unique but wasn't able to find anything after searching.

thanks in advance for any pointers/tips.

### Post by mikewhatever on 2011-09-20
Not sure that can be done. The copies of Windows bundled with laptops (obviously, they don't just come with them) are OEM copies, and according to the EULA, the user is not allowed to install and run them on anything other then that specific machine each is bundled with. A virtual computer is just another machine, as far as Windows is concerned, and so you have to buy a new Windows license.

I bet a lot of people make that mistake thinking "I bought Windows, so now I own a copy and can install it on whatever I want". Hate to spoil your fun, but that's not the case.
