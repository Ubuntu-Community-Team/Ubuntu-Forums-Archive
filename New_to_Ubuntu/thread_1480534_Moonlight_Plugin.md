### Post by Oropher8598 on 2010-05-11
AFAIK, Novell Moonlight currently only supports Silverlight features up to Silverlight 2. If the site you're trying to use it on has built on the more recent Silverlight 3 or 4, Moonlight may simply *not work* there until it's been updated. (You might try the [preview release](http://go-mono.com/moonlight/prerelease.aspx), although it's still alpha-quality.)

Does your Moonlight plugin work properly on other Silverlight pages, such as the [samples at go-mono.com](http://go-mono.com/moonlight/community.aspx)? If not, you could try installing it again - just visit [http://go-mono.com/moonlight/](http://go-mono.com/moonlight/), click the 'Install' button, and follow the instructions.
