### Post by heyandy889 on 2012-02-29
Hi, suomalainen. For 5), you might be able to access Google Calendar through Thunderbird by using "Provider" and "Lightning." [https://addons.mozilla.org/en-US/thunderbird/addon/provider-for-google-calendar/](https://addons.mozilla.org/en-US/thunderbird/addon/provider-for-google-calendar/)

For 2), I'm not sure how to align the stylus with the mouse cursor. However, a general trick that can help is checking for hardware drivers. I forget the precise command, but if you are in Ubuntu 11.10, just press the Super key to search applications, and start typing "hardware." An application for updating your hardware drivers should pop up. 

Beyond checking for hardware drivers, you could try a regular software update. Again, I don't remember the exact command, but press Super to do an application search, start typing "update," (I think it's called update manager), and run the update manager. Maybe there is a bug fix that you haven't downloaded yet. 

Beyond that, I think you're turning to Google - "toughbook stylus ubuntu 11.10."   Also, maybe an earlier version of Ubuntu, like 10.04, has smooth stylus support for the Toughbook. Happy troubleshooting!

edit: I wouldn't worry about the funky rainbow screen, ha ha. That happens to me sometimes and I've never had any persistent problems. Remember, it's not a bug - it's a feature!
