### Post by QIII on 2009-06-13
Since you have a very basic xorg.conf, you should be able to safely make a couple of simple changes.

If you have problems at next boot, shut back down and reboot into recovery mode.  In the terminal,
type

sudo dpkg-reconfigure -phigh xserver-xorg

Alternatively, use the down arrow to go to xfix (I think that's what it is labeled) which will take
you through a few questions and effectively do the same thing.

If you are really worried about it, make a backup copy of your /etc/X11/xorg.conf.

sudo cp /etc/X11/xorg.conf /etc/X11/xorgbak.conf

You can always restore it then.


So.  Here's how to change your xorg.conf:


sudo gedit /etc/X11/xorg.conf



In Section "Screen", between DefaultDepth 24 and EndSection, add


	Subsection "Display"
		Depth	24
		Modes	"1600X900"
	EndSubSection


It should look like this:


Section "Screen"
	Identifier "Default Screen"
	Monitor "Configured Monitor"
	Device "Configured Video Device"
	DefaultDepth 24
	Subsection "Display"
		Depth	24
		Modes	"1600X900"
	EndSubSection
EndSection


If this doesn't work, I'm sure you'll post back.

If it does work, please post back anyway.  This may help the next guy.
