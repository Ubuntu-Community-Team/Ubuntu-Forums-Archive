### Post by sikander3786 on 2010-09-13
Not possible. I mean there are a few alternatives to Synaptic but no one is built to act like a download accelerator.

If Synaptic is not picking up the speed it should, keeping your line speed in mind, try changing the Server from System >>> Administration >>> Software Sources. Click "Choose the best server". It might speed up the process a little bit.
