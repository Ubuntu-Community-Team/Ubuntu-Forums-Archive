### Post by Mornedhel on 2009-06-03
When you Save As a document in OO.o 3, you can check "Save with password".

If you want to encrypt other files, you will need gpg (installed by default) and your own secret key, which you can generate with gpg. I recommend reading on serious encryption if you are going this way.
