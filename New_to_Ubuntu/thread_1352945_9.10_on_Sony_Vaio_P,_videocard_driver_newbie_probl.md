### Post by NCLI on 2009-12-13
Ok, this is what you should do:

1. Open a terminal. You should be able to find one under Applications->Accesories->Terminal

2. Write this command, without the quotation marks: "gksudo gedit /etc/default/grub" then press enter.

3. Write your password and press enter again.

4. Find the line GRUB_CMDLINE_LINUX_DEFAULT="quiet splash"

5. Edit the line so that it now reads GRUB_CMDLINE_LINUX_DEFAULT="quiet splash mem=2000mb"

6. Save the file, then reboot.

If that doesn't work, I'd recommend either posting a new thread in [Hardware and Laptops]("http://ubuntuforums.org/forumdisplay.php?f=332"), or installing Ubuntu 9.04 and sticking with it for now, as the GMA500 should work correctly in that version.

after you edit and save "/etc/default/grub"
you have to actually update grub
just run "gksu update-grub"  after you edit the file
and this will update config file that grub reads on boot.
