### Post by ThePol1 on 2010-01-28
I installed Pacellite as a clipboard manager in Gnome (Ubuntu 9.10). My problem is that when I select a block of text, it automatically gets copied to the  clipboard.

This is really annoying  since I usually select another block of text to be replaced from the  clipboard contents only to find the Ctrl+v now pastes the unwanted text.

Any suggestions?

### Post by ajgreeny on 2010-01-28
Right click on the parcellite icon in your system tray and in Preferences you can set the Ctrl+C or Primary.  Choose Ctrl+C.

Done.

You can still use the very useful middle click to paste the last thing you selected, which I find I use a huge amount.

### Post by treehermit on 2011-05-05
you are probably using kupfer/Gnome Do, which is conflicting with your clipboard.

go to your kupfer/ gnome DO preferences and disable the clipboard plugin.

that's it. now selected text will be copied only when u press Ctrl-C or right click copy.
