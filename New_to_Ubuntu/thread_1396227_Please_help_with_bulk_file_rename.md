### Post by NightwishFan on 2010-02-01
The only software I have used to bulk rename audio is EasyTag, which is in the software repos, click to install:
[Install Easytag]("apt://easytag")

There are others such as pyrename, gprename, prefixsuffix, however I have never used any of them.

### Post by KenBW2 on 2010-02-01
If you download Thunar from Software Centre it has a really good file renaming tool. Just highlight the files you want to rename and select Rename from the context menu.

Then select your criteria to rename them.
