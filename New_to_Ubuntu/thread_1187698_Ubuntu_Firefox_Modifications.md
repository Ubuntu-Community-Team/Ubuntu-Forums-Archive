### Post by lovinglinux on 2009-06-15
Firefox [Peacekeeper](http://service.futuremark.com/peacekeeper/index.action) benchmark results:

Ubuntu Firefox Modifications extension disabled - 419 points

Ubuntu Firefox Modifications extension enabled - 410 points

The benchmark results differences are irrelevant, since only 9 points variation is expected as normal between two tests.

Just for comparison, I have  30+ extensions and when I disable all of them, the performance gain is about 30%.
