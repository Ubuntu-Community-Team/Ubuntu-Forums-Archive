### Post by mingle on 2008-07-25
Hi,

Ha Ha! I did the same thing when I was trying to make my system look more like XP!


You need to go to the panel, right-click then select 'add item to panel' (or similar - I don't have access to my Linux box at the mo!). Then, scroll down the list of options/items and check 'minimized task/programs' (again I can't recall the exact name), then click 'okay' (again, whatever to close the selection window) and you should be able to see the minimised windows again!
