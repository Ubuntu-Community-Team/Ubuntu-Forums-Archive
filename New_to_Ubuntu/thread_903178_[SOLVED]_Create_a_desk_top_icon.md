### Post by Gregmond on 2008-08-28
yes

Right Click on the desktop and select Create Launcher.
In the Name line, Type what you want to call it. In the application you want something like: 
firefox [http://www.google.com](http://www.google.com)
Change the address to whatever site you want.
Click the picture on the left hand side to select a different Icon.
Put a comment in if you wish and Click OK. You now have a desktop launcher to take you to your preferred website.
