### Post by trikster_x on 2008-10-16
If you are trying to run Ubuntu like a .exe file in windows that is your problem.  You actually need to boot with the usb drive or the dvd.  Just restart your computer and get into your BIOS.  Find your boot options menu, make sure that either it is set to boot a cd rom first or a usb device.  then put your dvd in IF you actually burned it as an image file.  If you did not burn it as an image file, then it will be no good.  Alternately, if your bios allows a usb boot option, enable that and stick your usb drive in.  Your computer should automatically boot into the liveCD program.  The easiest way to do it is with an image disk.

Edit: If you do get into the liveCD program, then it would probably be a good idea to search the forums for a how to on installation.  If you use a wireless adapter, look for the correct way to install drivers for that as well, as this is a source of irritation to most new Ubuntu users.  Belive me, it sucks when you spend 45 minutes on an install and then have to get right back into windows to figure out how to make everything work.  At least if you have a way to hook up the internet, you can use the forums for Ubuntu on Ubuntu, which saves a lot of time.
