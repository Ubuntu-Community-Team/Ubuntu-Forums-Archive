### Post by beastrace91 on 2009-10-13
You can install programs on Linux in a number of ways. The easiest way is to click on your Applications menu and select "Add/Remove Programs" it will let you easily install things :)

Winamp unfortunately will not install natively on Ubuntu. You could try running it under [Wine](http://www.winehq.org/) but before you do that I would recommend trying one of the many Linux alternatives. If you are looking for a good media player I would like to suggest "Banshee"
