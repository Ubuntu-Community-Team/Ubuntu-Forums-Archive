### Post by Brother Phil on 2010-05-26
Thanks for the pointer - my youngest managed to make this happen to me.

Having looked at the compiz panel, what seems to have happened is that window moving gets activated, with button 1 to activate it.

By using tab, you can get to this control, and change it to, say, Alt + button 1, and then it clears up.
