> 5. How do you " Try it out" first. I wasn't to sure about the instructions. I read somewhere once I've ordered a CD or downloaded and made a CD with the .Iso ( i've made those before) there is a way to just run Ubuntu off the CD just to try it out ( it wont be as functional as actually installing it on the hard drive though right?)

You need to download and burn the LiveCD onto a blank disk, then just pop it in the drive and boot up into it. This may require setting your BIOS to boot from the optical drive before the hard drive. The LiveCD is a fully functional Ubuntu desktop running completely off the CD, it won't touch your hard drive. As a result it will be slower, and you can't install things like graphics drivers in it, but otherwise it'll work just like the real thing. It'll allow you to confirm that your hardware will run Linux ok, and once you're satisfied you can start the installer straight away.

Even if you decide not to install Ubuntu, having a LiveCD around can be useful for rescuing dead or infected Windows systems.
