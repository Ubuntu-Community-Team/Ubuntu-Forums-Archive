### Post by jimmy the saint on 2008-12-10
The menu to the right of applications is "Places."  You will want to add the location there.  

To do so, open the file browser (places>[username])  Navigate to the folder you want to add the (Bookmarks>add bookmark)

that folder will show up in the places menu and the left pane in nautilus.
