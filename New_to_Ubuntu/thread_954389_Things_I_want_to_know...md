If you don't have internet connection, you may download individual packages from [http://packages.ubuntu.com](http://packages.ubuntu.com) and install the deb files with the command
```
$ dpkg -i filename.deb
```

Just make sure you download all the dependencies.

For videos, I would suggest using mplayer. For audio, you can use rhythmbox or even try exaile. If you are using the kubuntu flavour, try amarok. Installing amarok on ubuntu will pull in a lot of kde libraries...

### Post by fornix on 2008-10-21
For all kinds of MS office files -  doc, xls, ppt, etc, You just have to double click them and it will open in openoffice. It comes pre-installed with ubuntu.
Find open office in Applications -> Office ->

Openoffice.org Word Processor - Word
Openoffice.org Spreadsheet - Excel
Openoffice.org Presentation - Powerpoint

Similarly ubuntu comes preinstalled with document viewer called evince.
Just double click any pdf file and it'll open in evince.
For mp3, rhythmbox comes pre-installed. You can find it in Applications -> Sound & Video

Image viewing software called F-spot comes preinstalled too. If you want something like photoshop, there is GIMP. Both can be found under Applications -> Graphics

There are a lot of alternatives to these defaults. You have to try and decide for yourself which one you prefer.
I personally prefer amarok/exaile for mp3, mplayer for video, openoffice for doc, xls, Eye of Gnome viewer for image viewing. I find fspot a bit heavy coz its written in C#.

### Post by stalkingwolf on 2008-10-21
For a "Best" application YOU will have to experiment.  Remember Ubuntu and Linux are the embodiment of CHOICE.  What that means is, If you ask 10 users
you will likely get 10 different answers.  This is because each user uses
what does what THEY want to do best.

Remember Linux/Ubuntu is not windows. YOU have choices. If one application does not do what you want, try another. You can do this until you find exactly what you want.  If you dont find what you want you still have options.

You can use Wine, Cedega, Crossover, or modify an application to do what you want.

Its all about choices.
