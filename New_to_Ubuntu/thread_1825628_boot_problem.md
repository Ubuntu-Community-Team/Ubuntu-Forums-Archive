### Post by Mark Phelps on 2011-08-16
According the the script output, you have no Win7 boot loader files on the drive -- which is why it won't boot. If you copied a Win7 setup from a PC that came preinstalled, you failed to copy the boot partition as well -- and it contains the boot loader files.

You will need to use a Win7 installation DVD or a Win7 Repair CD, boot from it, and run Startup Repair three times -- to regenerate the boot loader folder and files.
