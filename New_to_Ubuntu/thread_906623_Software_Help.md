### Post by Daveski on 2008-08-31
If you are completely new to Linux or Ubuntu, then I would strongly recommend that you stick to exploring the 'approved' installations in the repositories. These have no-brainer installs, and will even keep themselves updated when additions are made to the repositories. Browse in Synaptic, or Add / Remove.

If you insist on downloading a tarball (rather like a ZIP file), then you need to read instructions provided on either the website where you located it, or more usually a README or INSTALL text file included in the tarball - however, the distributer of the game you are trying to install may not include all dependent libraries etc. which means you may have to hunt down and install other components - again stick to the repositories when looking for libraries and dependencies.
