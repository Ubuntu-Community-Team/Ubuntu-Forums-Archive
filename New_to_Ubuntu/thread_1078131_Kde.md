Well, if you're just looking to make your Ubuntu prettier, there's a TON you can do in Gnome!! You can install themes, icon sets, fonts, etc on your system. Check out:
[www.gnome-look.org](www.gnome-look.org)

You can also add all those cool desktop effects using Compiz. Check out how to do that here:
[https://help.ubuntu.com/community/CompositeManager/CompizFusion](https://help.ubuntu.com/community/CompositeManager/CompizFusion)
and here:
[http://tombuntu.com/index.php/2008/03/27/custom-compiz-effects-in-ubuntu-804/](http://tombuntu.com/index.php/2008/03/27/custom-compiz-effects-in-ubuntu-804/)

Check this out for a little Compiz Demo :D:
[http://www.youtube.com/watch?v=ZxfSwzhSn1c](http://www.youtube.com/watch?v=ZxfSwzhSn1c)

Also, since you're running an older rig, you might think about going to Xubuntu for a lighter environment, and a snappier interface - and then you can "pimp" it out from there with the same kind of effects, themes, icons, etc.
