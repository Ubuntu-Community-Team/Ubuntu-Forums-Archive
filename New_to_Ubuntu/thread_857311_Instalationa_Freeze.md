### Post by Xerp on 2008-07-12
There are only 6 steps?

[https://help.ubuntu.com/community/GraphicalInstall](https://help.ubuntu.com/community/GraphicalInstall)

In step 4 you enter your details - is there where your install freezes?

Does the LiveCD work OK otherwise?

### Post by avtolle on 2008-07-12
Are you trying to install 8.04 (Hardy)? If so, you should use the alternate installer. The Live CD's graphical installer needs 384 mb RAM to run correctly.

EDIT: If you need to get the alternate install iso, see [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download) where under the green arrow Download Now is a box to check to get the alternate install.
