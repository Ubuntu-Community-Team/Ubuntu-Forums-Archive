### Post by caro on 2009-01-01
If you are new to Ubuntu, burn a copy of the Live CD.  You can run Ubuntu from the CD without installing it.  That way you can check out the features and see if your hardware works with it.  

If you find you like it and you don't want to risk compromising your work computer, find an old computer and install Ubuntu on it.  Ubuntu works well on lots of old computers, even with minimal resources.  

Generally you can't install Windows program on Ubuntu, but if you get to the point where you want to dual boot or use WINE, you should be able to run whatever app you need.  There are lots of free apps for Linux that you may find you like.

Keep in mind Ubuntu is not Windows.  You will have to invest time learning how to do some things differently.  A lot can be accomplished using the GUI interface, but be willing to learn.

Good luck.
