### Post by ibutho on 2008-12-27
Some distros like Ubuntu, do not upgrade the version of packages in their repos unless its to fix bugs or other security issues. You can download the java bin file (don't download the rpm.bin file) from java.sun.com and install it yourself. Post back if you need more instructions.
