### Post by linesma on 2009-05-28
Just a question.  Did you check your start-up programs?  You can find a list of programs that start on boot at System-Preferences-Startup Programs.  It may be listed there.  If it is, and there is a tick mark in it, uncheck it, and the program will not run when you start-up.

I hope this helps.

linesma

### Post by tengteng on 2009-05-29
Thanks! It was not on the start-up programs.. 
But the weird thing was, I checked the box which says save session for future logins when I logged out and viola! epiphany does not auto start anymore!  Weird because it means that during startup with a new session, epiphany will autostart..
