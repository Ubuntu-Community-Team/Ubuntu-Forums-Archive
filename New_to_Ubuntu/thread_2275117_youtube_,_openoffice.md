youtube is primarily a website, so accessing it through a browser is the normal way to do it. Flash is not strictly necessary for this any more, because youtube is moving away from that and towards html5 video. 
There are programs to access youtube, but they usually stop working whenever youtube changes something major, which happens often. Since packages in the ubuntu repositories are frozen at release except for bug fixes and security updates, you'd need to get these programs from a PPA (Personal Package Archive) to get one that works.
