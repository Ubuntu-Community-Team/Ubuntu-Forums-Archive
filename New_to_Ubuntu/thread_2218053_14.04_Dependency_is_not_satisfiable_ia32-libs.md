### Post by mc4man on 2014-04-19
codeweavers has a Multi-Arch  deb that doesn't require ia32-libs, has since 12.10. If your are trying to install an older deb then see if you can get a more recent one.
Otherwise it's about 65 :i386 packages to install
The most likely way to install that .deb is with gdebi which is currently broken & will not install any dependencies. You can use it, after it fails then run - 
```
sudo apt-get install -f
```
or downgrade to gdebi 0.9.3
