### Post by sidzen on 2011-12-01
You may want to -- go to Tools => Add-ons Manager (or just "about:addons" in address bar) and check present versions of some Plugins like Shockwave Flash (*i.e.* is it 11.0 r1 or better?) ; update jre ; or use Opera.
