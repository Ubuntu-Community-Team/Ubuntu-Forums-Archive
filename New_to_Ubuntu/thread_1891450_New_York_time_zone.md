### Post by ratcheer on 2011-12-05
There is an option somewhere in the Ubuntu configuration where you tell it something like, "this computer uses UTC". When you do that, it then uses your time zone to display the time, correctly. But if you chose to set it to "local time" and also try to apply the time zone, it will be incorrect.

This should help you do what you need:

[https://help.ubuntu.com/community/UbuntuTime](https://help.ubuntu.com/community/UbuntuTime)
