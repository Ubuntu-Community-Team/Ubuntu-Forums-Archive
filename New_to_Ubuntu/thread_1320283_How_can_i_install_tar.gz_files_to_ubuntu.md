### Post by theozzlives on 2009-11-09
Tar is a command line archiver, gz is the compression, it comes with most flavors of Linux. To install the theme, just drag it to the themes window and drop it. Ubuntu will install it for you.

### Post by The Funkbomb on 2009-11-09
If you're coming from windows, tar.gz is sorta like a zip file.

You must unpackage it first.

The easiest way is like this:

```
tar xvfz package.tar.gz
```

It will then unpack the files into a folder.  Find the folder and proceed.

Oh yeah, like the others said, for themes you can just drag and drop.
