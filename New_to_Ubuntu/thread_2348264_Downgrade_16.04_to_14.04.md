### Post by Bucky Ball on 2017-01-02
Welcome. Nope. Clean install of 14.04 LTS is what you want. A better idea would be to start posting threads with descriptive titles in the appropriate forum areas about your issues with 16.04 LTS, one issue a thread.

For future reference, clean install upgrades are generally less problematic than net upgrades. Doing an upgrade via the net requires the machine to be as close to 'vanilla' as you can get it, no manually added PPAs or drivers. Even then things can blow up.

Just to clarify, you attempted this on a Ubuntu virtual private server running 14.04 LTS? This was a local server that you own and can physically get to? You should maybe give more details about the VPS. :-k

Good luck.
