### Post by alexandremrj on 2008-05-24
The best way to install software in ubuntu is first to search for it in the repositories (a set of different software for ubuntu).

You can use Synaptic, found in
System --> Administration --> Synaptic

There you can search for the software you want by name or description and then install it by clicking the left square near the name and selecting install. You then press apply for the ok.

Scilab exists in the repositories.

If you download from the web (very much less secure) you are better of with a .deb file (kind of the .exe for windows). A .jar file is a java archive.
