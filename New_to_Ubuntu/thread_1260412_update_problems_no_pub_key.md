### Post by drs305 on 2009-09-07
I checked and there is no binary-lpia/ in the hardy main section of that repository.

You can try changing servers to one that supports lpia or you can edit it out of your sources.list. I don't think the main ubuntu server carries the lpia packages.

If you don't have a dell-mini or equivalent, or don't know how to find the offending entries, post your sources.list or edit it yourself to remove the main ubuntu references.
```

gksudo gedit /etc/apt/sources.list

```


If you have a dell-mini and are using hardy, here is one sources.list you could use:
> 
deb [http://dell-mini.archive.canonical.com/ubuntu/](http://dell-mini.archive.canonical.com/ubuntu/) hardy main universe multiverse restricted
