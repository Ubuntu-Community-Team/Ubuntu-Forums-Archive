### Post by stoogiebuncho on 2009-03-19
> **finer recliner said:**
> i think rsync can do this

Yes it can.  But if you don't like command line, use [grsync]("apt:grsync").  It's the same thing with a nice little GUI.

It's set up more like a backup utility than Briefcase is, but you can use it for the same function.  It allows you to save profiles for the folders that you backup often so that you don't have to enter the information every time you want to run the backup.
