Even though Linux is said not to depend on file extensions, I find them useful. I suffix plain text files with *.txt*.

[LIST]
[*]Right-click on such a text file in your file manager. (I use PCManFM; you'll be having Thunar.)
[*]Choose *Open with*
[*]Again *Choose Open* with from the submenu
[*]Select the text editor from the window that appears. It may be under *Accessories* or *Utilities* or ...
[*]Tick "Set selected application as default action for this type of file". That's most important!
[/LIST]
