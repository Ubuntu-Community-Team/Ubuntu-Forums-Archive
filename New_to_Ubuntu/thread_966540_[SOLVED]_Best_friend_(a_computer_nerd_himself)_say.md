### Post by nhasian on 2008-11-01
Yes, they do make Nero for Linux but its not free :)
you can simply use Brasero that comes with ubuntu to create an audio disc.

Go to Applications, Sound and Video, then select Brasero Disc burning.

Once the program launches, just select the first option which is "Audio Project - Create a traditional audio CD"

There are different CD Media.  There are at least 3 that i know of 63 Minute, 74 Minute, and 80 Minute CDs.  In Brasero's settings it shows that it also supports 90 and 100 Minute CDs, but i've never seen those before.
