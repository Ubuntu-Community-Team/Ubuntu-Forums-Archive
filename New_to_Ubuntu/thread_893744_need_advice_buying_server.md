### Post by viciouslime on 2008-08-18
By your description I get the impression this is a sort of personal server that isn't going to see too much load (i.e. you don't intend to run a site like this one with thousands of users online at the same time) so pretty much ANYTHING you can find will suffice. Seriously, 2-300 euros is overkill. If all you want to do is stream media and run a basic webserver you could get something for £20/30euros on eBay that will do the job very nicely.

As for distros, if this is a personal project, I'd say you should try all the ones you list and see what you like best, but if you're not really up for experimenting and learning by doing, then I'd stick with ubuntu server edition because presumably you're already used to ubuntu on your desktop :)

### Post by FakeOutdoorsman on 2008-08-18
I use an old Pentium II at 300 MHz with 384 MB ram as a web development server for a video streaming site.  Secondary usage includes backing up data from the production server and running rtorrent.  It is quiet, small, and uses very little power: about 35 watts.  No need to buy a new computer.  You can find machines much better than mine for free.

I am running a minimal install of Debian Etch, but a command-line, server, or minimal install of Ubuntu would have been a good choice too.  I used Debian because it can be a leaner install than Ubuntu, but I recommend using whatever you're familiar with.

### Post by viciouslime on 2008-08-18
Yeh if you want that machine to actually do the watching of the DVDs you're going to want something a little more powerful or get something that has hardware mpeg2 decoding. I'm not familiar with the French TV system, but I presume you'll be wanting to use the French equivalent of Freeview (DVB - it's an EU-wide standard as far as I know). If so you might want to buy a TV card with hardware encoding as well.

Failing this you're probably looking at around a 1GHz machine, PIII maybe...
