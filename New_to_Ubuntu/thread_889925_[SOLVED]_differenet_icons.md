### Post by sayakb on 2008-08-14
If you use a custom icon theme, look into ~/.icons/Themename/size/mimetypes folder for file icon associations. If you use one of the default themes, look into /usr/share/icons/Themename/size/mimetypes
Note: .icons folder is a hidden folder in your Home folder. Press Ctrl+H to reveal it.
