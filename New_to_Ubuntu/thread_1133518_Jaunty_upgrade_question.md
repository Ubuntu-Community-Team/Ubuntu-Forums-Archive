### Post by Peter09 on 2009-04-23
If you need stability and you have no problems then there is no need to upgrade. 

When Jaunty is release the package manager will ask you if you wish to upgrade, if you do then tick 'YES' and the rest is automagic.

### Post by Sef on 2009-04-23
Before upgrading, if you do, **back up** your files.   A 100% guarantee that all will do well does **not **exist.  It is also advisable to have a copy of either Intrepid or Jaunty handy to use if your computer does not work correctly after upgrading.   That way you can post a thread here, in case there is a problem (or reinstall, if you choose to do that.)
