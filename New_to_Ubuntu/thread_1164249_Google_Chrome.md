### Post by newbuntuxx on 2009-05-19
download an install wine. 

```
sudo apt-get install wine
```

then add the following to your software sources:

Go to System - Administration - Software Sources
Click on third party tab
click on add, enter: "deb [http://deb.playonlinux.com/](http://deb.playonlinux.com/) jaunty main" without the quotes
Reload

then open terminal and type:

```
sudo apt-get install playonlinux
```

once installed, you will find it under: Applications - Games. Run it, and then click on install. You will get a list of categories. Select: Internet, and then click on Chrome.

You will be able to install chrome and even watch youtube videos. But it is extremely laggy and buggy. I mean, if you type fast, the letters will lag behind. So, I say stick to firefox, if you want my opinion.

Peace
