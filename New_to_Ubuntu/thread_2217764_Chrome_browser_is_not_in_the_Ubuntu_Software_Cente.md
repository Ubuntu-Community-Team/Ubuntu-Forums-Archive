### Post by su:bhatta on 2014-04-18
Press download in this page: [https://www.google.com/intl/en/chrome/browser/?hl=en&brand=CHMA&utm_campaign=en&utm_source=en-IN-ha-bk&utm_medium=ha&utm_term=%2Bchrome](https://www.google.com/intl/en/chrome/browser/?hl=en&brand=CHMA&utm_campaign=en&utm_source=en-IN-ha-bk&utm_medium=ha&utm_term=%2Bchrome)
You will have to choose 32bit or 64 bit & for Debian/Ubuntu.
Choose Accept & install.
It should by default open in Ubuntu Software center.

If you choose to download the file which will be named : google_chrome-stable-( Architecture here).deb

Once downnloaded, double click on the file & by default it should open with Ubuntu Software Center. 
Click install.

You will have Chrome installed. 
Search and select it from Dash, if you are using normal Ubuntu or select it from applications menu.
