### Post by BDNiner on 2009-05-05
All your settings should stay the same since none of the information in the configuration files is overwritten without your consent. The files in your home folder will be untouched. I noticed that some of my customizations of the config files like fstab file i was prompted on whether I wanted to keep the current file or change it to the default that is installed during the upgrade.
