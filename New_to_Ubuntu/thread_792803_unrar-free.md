### Post by gerben1 on 2008-05-13
The unrar package is in the multiverse repository, you probably have not added the multiverse repository to be checked for packages, you can do so as follows:

go to
System->Administartion->Software Sources
and check the checkbox in front of:
"software restricted by copyright of legal issues (multiverse)"

after that try to install unrar again (sudo apt-get install unrar), it should install fine now
