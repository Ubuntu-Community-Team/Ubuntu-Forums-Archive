### Post by newb85 on 2012-09-25
Welcome to the Forums and Ubuntu!

The specifics of the error message would probably be helpful, too.

By the way, don't spend too much time trying to make a Wubi install work.  Wubi was not intended for long-term use.  It's essentially just a self-contained virtual machine, and it has a tendency to break if you do too much with it.

Instead, I recommend installing using a Live USB.  Directions on creating the Live USB [here]("http://www.ubuntu.com/download/help/create-a-usb-stick-on-windows").  Directions on installing [here]("http://www.ubuntu.com/download/help/install-ubuntu-desktop").
