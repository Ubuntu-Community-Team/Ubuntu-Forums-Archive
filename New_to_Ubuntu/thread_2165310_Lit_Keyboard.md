### Post by GUZZLR on 2013-08-18
If anybody's interested...I have been using the new Razer Maurader keyboard for about a week now, with out issues while in Linux.  The only problem is it will not allow for changing colors while in Linux, simply becuause it uses Windows Color Spectrum.  So, you must set the keyboard colors in Windows; the color profile will remain when booting into linux.  
Also, if any gamers are looking for a well built gameing keyboard, in a compact size for better desk space, the Maurder keyboard is worth looking into, it is a compact keyboard, but with a full layout of nice feeling keys.
