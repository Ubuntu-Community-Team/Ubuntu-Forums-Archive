### Post by mkvnmtr on 2008-12-26

If yes - consider installing unetbootin on your Windows PC: it will download and write lots of different Linux distros (inc lightweight e.g. DSL / Puppy / Slitaz) on to your USB automatically for you to try out.

Out of the two, Puppy would probably be easier to configure but DSL will be faster. I had an old IBM thinkpad 380XD with 64MB of RAM running DSL very nicely.
