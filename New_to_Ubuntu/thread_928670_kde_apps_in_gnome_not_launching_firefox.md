### Post by NewJack on 2008-09-25
First off, make sure to check what your default application setup is for Ubuntu.  I don't remember where it is offhand (sorry, I'm at work and don't have Ubuntu installed here).  Make sure that Firefox is set as your default browser and as the app that opens HTML.  That should fix it.  I think by default KMail is set to use Konqueror to open links.  Also, I don't use KMail, but did you go through KMail's options yet?  Maybe there is a setting there that you need to adjust.

Also, I saw mention somewhere in another thread to install this package:  kdebase-kio-plugins

Here is that thread: [http://ubuntuforums.org/showthread.php?t=299133](http://ubuntuforums.org/showthread.php?t=299133)
