### Post by PGTips91 on 2010-04-29
> **OwNeD said:**
> so i am currently trying to burn the 10.04 iso file with PowerISO , but i get an error "data error cyclic redundancy check" ? i have tried burning other ISOs and they work fine so, must be something wrong  with the 10.04 ISO, dont you think?

You do not need to download the entire file again! Use zsync to correct any minor problems.

First download the .zsync file to the same directory as your current ISO file. Then open a Konsole window in that directory or navigate there, then execute this code : --

```
 mv ubuntu-10.04-dvd-i386.iso ubuntu-10.04-dvd-i386.iso.old
```

This simply renames the faulty file and allows it to be used as the input file for zsync.

```
zsync ubuntu-10.04-dvd-i386.iso.zsync -i  ubuntu-10.04-dvd-i386.iso.old -u http://cdimage.ubuntu.com/releases/lucid/release/ubuntu-10.04-dvd-i386.iso
```

If that is not the file you are downloading simply substitute the correct information.

The ' -i' option tells zsync which file to check for previously downloaded data.

The ' -u' option tells zsync the URL of the target ISO file.

You will have to wait for a few minutes while zsync checks the .old file for re-usable data, it will report the percentage so available, then go on-line and download any parts that contained errors the first time around. This could be a very small amount of data, but will generate an ISO that is verified as correct. Just wait until zsync finishes and reports that the file is OK. 

Zsync will read the old file, reuse any data that is correct and download the data that does not match the md5 signatures, so correcting the problem with the minimum of data needing to be downloaded.

Then burn again, using a slower burning speed if possible, and try again. You can even verify the DVD for its md5sum using the 'dd' command if necessary.

Here's an example from my system : --

> paul@paul-ArtistX07:/media/Elements/LinuxDownloads/DownloadedISOfiles/Ubuntu10.04$ date && time zsync ubuntu-10.04-dvd-i386.iso.zsync -i ubuntu-10.04-rc-desktop-i386.iso
Fri Apr 30 11:47:01 NZST 2010
reading seed file ubuntu-10.04-rc-desktop-i386.iso: **************************************************************************Read ubuntu-10.04-rc-desktop-i386.iso. Target -26.2% complete.

Also note that selecting a mirror closer to you will improve the download speeds.

Additionally, using aria2 will combine both the torrent and the mirrors, assisting in getting the ISO faster and also then sharing it with the torrents.

eg > paul@paul-ArtistX07:/media/Elements/LinuxDownloads/DownloadedISOfiles/Ubuntu10.04$ aria2c  ubuntu-10.04-dvd-i386.iso.torrent  [http://mirror.ihug.co.nz/ubuntu-releases/lucid/ubuntu-10.04-desktop-i386.iso](http://mirror.ihug.co.nz/ubuntu-releases/lucid/ubuntu-10.04-desktop-i386.iso)
