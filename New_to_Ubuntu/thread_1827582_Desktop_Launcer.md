### Post by 3rdalbum on 2011-08-18
The "old style" is no longer maintained by its developers, so it won't be available in Ubuntu 11.10.

You should try getting used to either the Unity interface (the default in 11.04) which will be improved in 11.10, or try a different desktop environment such as XFCE, LXDE, or KDE.

In Ubuntu 11.10 you'll also have the option of Gnome Shell from Gnome 3, which is also a really nice desktop environment. Unity is improving in 11.10 too.
