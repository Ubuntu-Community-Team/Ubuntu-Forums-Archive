### Post by sleepyjon on 2009-03-17
Linux doesn't run windows drivers or programs. The autorun program is probably a windows .exe.

In many cases the drivers needed are already installed.

If you can hear sound, connect to the network, and see 3d goodness you're probably all set to go.

To check 3d really easily open up a terminal (under gnome I think it's applicaions > accessories > terminal) and type:

```

glxgears

```

If you see the spinny gears, you have 3d goodness.
