### Post by vlamnire on 2010-02-21
EDIT: Before you do anything I say, try going into the Terminal which is located in Applications > Accessories > Terminal and type in 

sudo update-grub

Then type in your sudo password which is the password you log into with, you will not see any letters when typing, and press enter.  After it has updated, restart and see if you can get into Windows.

Ok, I think I know what your problem is... Your missing 2 lines of code in your GRUB menu.lst file.  I have GRUB 2 and you have GRUB Legacy which uses menu.lst and I use grub.cfg.  For your Vista entry in menu.lst you have this:

title Windows Vista
    rootnoverify (hd0,0)
    chainloader +1

I have: 

menuentry "Windows Vista" 
    insmod ntfs
    set root=(hd0,2)
    search --no-floppy --fs-uuid --set 5c0c85240c84fa72
    chainloader +1

Now you may think you could just edit your menu.lst and copy what I've got but the --set value doesn't work for every computer so you cannot do that.  The only way I can see you can fix this issue is to upgrade to GRUB 2 and it should automatically get a list of your operating systems and automatically set the arguments right.  To do this please follow this page (It's for Ubuntu 9.04 but it will work on Ubuntu 9.10):  

[http://www.howtoforge.com/how-to-install-grub-2-on-ubuntu-9.04](http://www.howtoforge.com/how-to-install-grub-2-on-ubuntu-9.04)

Either do all that or you can reinstall Ubuntu 9.10 from the CD again then once you boot after installation immediately go to System > Administration > Update Manager.  Check for all updates then restart after installing them.  You will get a list of several options in the boot loader, if you want to get rid of those just ask.

### Post by vlamnire on 2010-02-21
Try this command: sudo update-grub
Then reboot if that doesn't work try this command: sudo upgrade-from-grub-legacy
and if that command doesn't work try this one: sudo apt-get install grub2

The first one should update GRUB and apply all bootable options, second one should upgrade to GRUB 2 but I haven't seen that command before, the third command is just to install GRUB 2 normally.
