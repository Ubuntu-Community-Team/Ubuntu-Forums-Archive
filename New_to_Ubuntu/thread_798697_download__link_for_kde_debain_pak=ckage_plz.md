Solution 1
Download Kubuntu instead of Ubuntu. Burn to cd

Solution 2
use ```
sudo apt-get install kubuntu-desktop
```which will download packages to /var/cache/apt/archives and install from there.
If you have already installed kde check this folder to see if the packages are there.
