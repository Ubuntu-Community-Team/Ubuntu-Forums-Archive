### Post by Bennett2005 on 2008-06-12
Hello!!

An ISO file is a disc image.  You need to use a burning application like Nero or Infrarecorder (open source) to burn the image to a disc.  
Infrarecorder download site:

[http://www.downloadsquad.com/2007/10/02/infrarecorder-open-source-cd-dvd-burner/](http://www.downloadsquad.com/2007/10/02/infrarecorder-open-source-cd-dvd-burner/)

It's different than just copying the data files onto a cd.  You need a program that will give you an option to burn a disc image.  Then you select the .iso file and burn away.
