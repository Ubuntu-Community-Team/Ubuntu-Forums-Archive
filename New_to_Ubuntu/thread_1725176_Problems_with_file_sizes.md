### Post by CharlesA on 2011-04-09
You haven't really deleted the file, you "moved it to trash"

To get the freespace back, you'd have to empty the trash.

---

### Post by seawolf167 on 2011-04-11
Aye, just empty the trash as posted above. Note that a user's trash is different from root's trash, so if you send something to the trash as root, you'll need to empty that separately else you'll have "phantom missing space".
