### Post by Tatty on 2008-12-10
Go to applications->add/remove, then type in "divx" in the search box.  The first entry that appears should be the Gstreamer ffmped video plugin which should install a divx codec.

Alternatively you can install VLC which comes with support for pretty much everything out of the box - i dont know if this includes divx but i imagine it does.
