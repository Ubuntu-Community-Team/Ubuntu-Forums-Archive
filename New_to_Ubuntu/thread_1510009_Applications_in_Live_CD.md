### Post by ubuntu27 on 2010-06-16
> **warfacegod said:**
> I've never heard of a LiveCD being referred to as a Desktop CD.

Long time ago Canonical decided to call their LiveCD as DesktopCD to avoid the confusion in the objective or capabilities of the CD. 
[B]
Live CD[/B]---------> Desktop CD.- Live CD has been for a very long time just a tool to test the Operating System before installing. But, now that the current LiveCDs can also offer OS Installation, canonical decided to call it "Desktop CD"

**Install CD** -----> Alternate Install CD.- The text based install CD was a standard way to install a Linux OS. But since the LiveCD can now install the OS, it was decided to call it an "ALTERNATE (Alternative) install CD"

See this [Ubuntu Jaunty Download]("http://releases.ubuntu.com/9.04/") page:

Notice that is says: 


"**Desktop CD**

The desktop CD allows you to try Ubuntu without changing your computer at all, and at your option to install it permanently later. This type of CD is what most people will want to use. You will need at least 256MB of RAM to install from this CD."
