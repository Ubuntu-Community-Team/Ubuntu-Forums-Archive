### Post by mvalviar on 2009-03-14
Welcome to ubuntu!

1. I find the default apps with ubuntu quite useful so I don't bother changing them. I you really are customizing your desktop you might wanna use [ubuntu-tweak]("http://ubuntu-tweak.com/") and compiz config settings manager (ccsm). You can install ccsm using synaptic. Simply search for compiz and I should be there. Once installed you can find it under System>Preferences.

2.I'm using open office. But I believe its possible to use M$ office under the latest version of wine. Check it [here]("http://appdb.winehq.org/").

3. I use the default multimedia apps. But they say amarok is good and VLC can play everything. Both of them are available in synaptic.

4. No idea. If the app you mentioned is a windoze app. You can try to run it under wine.

5. [http://www.gnome-look.org]("http://www.gnome-look.org"). Look under Gtk 2.x. You will be given a tar.gz file. Go to System>Preferences>Appearance and drag and drop the file you downloaded to the Themes tab. If it doesn't work read the documentation that comes with the theme you got. 

I'm using ubuntu dust which is very nice. You might wanna try it. [https://wiki.ubuntu.com/Artwork/Incoming/DustTheme](https://wiki.ubuntu.com/Artwork/Incoming/DustTheme)

### Post by the8thstar on 2009-03-14
1. & 2. : 

Off the bat, I would recommend the following apps to you:

Sun xVM VirtualBox - run virtual machines
Avant Window Navigator - Mac-like dock
CrossOver - you'll have to pay $35 for this one but you can run Office 2003 and 2007 off the bat
Frozen-Bubble - game
Penguin Racer - game
Wallpaper Tray - auto toggle through your background images 
Skype
Brasero Disc Burner - like Nero disk burner
BleachBit - disk cleaner
Compiz Fusion icon - easy manager for Compiz
Preload - helps the system load apps more quickly
Prelink - same thing
Ubuntu Tweak - customize Ubuntu
gufw - graphical frontend for the built-in Ubuntu firewall

3. Movie Player, VLC and Rythmbox Music Player are my top 3 references. I use them for everything.

4. I don't know.

5. Download a theme on your desktop like **mvalviar **suggested. Then right click on the desktop, select the Theme tab and drag and drop your theme in that window. The install should be automatic and the system will offer to apply it immediately.

Just my two cents:

1. Useful apps... hmm Startup-Manager, Avant Window Navigator, Stellarium, Google Earth, Gmail notifier, screenlets, Compiz-Fusion icon, NTFS Configuration Tool, Compiz Settings Manager, Picasa and Wine.

2. Wine things are quite fiddly at the moment.. If I was you i'd just stick to OpenOffice. It is quite similar to Microsoft Office in many ways.

3. Well of course there is no "best" as it is each to their own opinion, but I personally use Exaile for music because it is nice and fast, responsive, reliable and has some neat features and then I use VLC Player for videos as it supports a huge range and is very configurable.

4. Have you looked at the compatibility notes at [www.winehq.org](www.winehq.org) to see if ChemDraw is supported?

5. Go to art.gnome.org and choose some things you'd like to install. Download them to your desktop, then go to System > Preferences > Appearance and drag and drop the package files one by one into the Appearance window and you should be able to install them to use.
