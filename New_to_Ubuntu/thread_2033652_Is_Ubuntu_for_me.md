### Post by nothingspecial on 2012-07-26
You'll just have to try it.

Maybe all your hardware will just work, maybe it won't.

Ubuntu is a full featured operating system and is unlikely to squeeze any more performance out of you hardware. A lighter version such as Lubuntu almost certainly will.

If you are a highly experienced windows user then there will likely be a steep learning curve. If you have better things to do with your time then maybe you ought to wait until you have some time free.

On the other hand, it you are curious then ............. it will be fun :)

### Post by Mark Phelps on 2012-07-26
The only way you will know for sure it to try it ...

Hardware drivers for Linux are hit-and-miss, especially with new PCs that were built to use special Windows-only drivers.  Running from a LiveCD or LiveUSB will show if you have any driver problems.

As to Apps, while it's true that there are thousands and (nearly all) of them are free -- they are not Windows apps, so some relearning is going to be needed to switch over to using different apps.

As to Wine, personally, I found it to be a waste of time -- but then, I get the latest Windows OS and SW through my company at a major discount and, in all too many cases, that stuff doesn't work in Wine.

As to a Virtual Environment to run Windows apps, while that can be interesting and certainly a learning experience, apart from that, in the case where you already have a Windows environment that is working well for you, it buys you little else.

But, by all means, give it a try and see what you think.
