### Post by newb85 on 2013-06-06
xfburn is the one installed in Lubuntu by default.  Brasero is the one in Ubuntu by default, and you could add it from the Lubuntu Software Center.

The Software Center should be the first place you look for programs to install.

You may need to install Ubuntu Restricted Extras for it to work.  (I'm not sure.)  If you're planning to play proprietary formats (like .mp3, .wma, etc.) you'll need to, anyways.  Again, the SC is where to go for this.
