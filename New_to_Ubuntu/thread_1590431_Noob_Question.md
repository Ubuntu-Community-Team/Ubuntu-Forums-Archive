### Post by lbrty on 2010-10-07
Do you mean when you download Ubuntu instead of install? In reference to the user named *Boring, *you must download the .iso from ubuntu.com, burn it to a CD or make a bootable USB flash drive from the .iso.

Are you planning on installing Ubuntu within Windows using the WUBI installer, dual-booting along side Windows or having Ubuntu as your only operating system?

I would recommend that you first test your hardware compatibility with the live CD or USB flash drive by clicking try Ubuntu instead of installing first.
