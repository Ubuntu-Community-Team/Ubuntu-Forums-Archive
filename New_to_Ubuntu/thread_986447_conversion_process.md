### Post by cwsnyder on 2008-11-18
The problem is there is a wealth of alternatives:  vim and emacs are the top of most lists for editing on Linux, with plug-ins to add specific functions for the languages/systems.  Phpedit or gphpedit are there specifically for css/html/php.  Gedit, which is included in your install as the default Gnome editor for Ubuntu, can add plug-ins also to extend the coverage for many languages.  Amaya is an editor which I have seen reviewed in some Linux magazines for that use.  See if any of these trip your trigger[:-)]
