### Post by ftmichael on 2008-05-06
The weather.com code for London, England is UKXX0085.

Just go to [http://weather.com/](http://weather.com/) and search for London.  Click on the link for London, United Kingdom.  The code is in the URL: [http://www.weather.com/outlook/travel/businesstraveler/local/UKXX0085](http://www.weather.com/outlook/travel/businesstraveler/local/UKXX0085)
