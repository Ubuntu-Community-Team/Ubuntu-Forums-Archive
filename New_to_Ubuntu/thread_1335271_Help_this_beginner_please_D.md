### Post by Anuovis on 2009-11-23
I am a beginner as well but your questions do not seem to be that hard.

Firefox is there.

MS Office has an alternative called Open Office. You can install it in Windows and see how it looks like. Be aware that although you can save in MS file formats the compatibility is still rather poor. Presentations get messed up, as well as tables. Might be other issues I have not run into yet.

Hardware is a tricky part. It may work it may not, or at least not to the extent as in Windows.

In any case, you never know until you try. Ubuntu requires some luck.
Play with Live CD, try dual-booting and see if you like it. I would not advice jumping into Ubuntu straight away. It has a learning curve and some spirits of random inside it.

### Post by snowpine on 2009-11-23
[Ubuntu is distributed as a "Live CD".]("https://help.ubuntu.com/community/LiveCD") You can try the Live CD with no change to your computer, see if you like Ubuntu with no risk. The Live CD is also the best way to tell if your hardware (wireless, keyboard, video, etc.) all work out-of-the-box.

Office is a Windows program; OpenOffice is the equivalent program in Linux (highly recommended).

[Ubuntu does not ship with DVD playback]("https://help.ubuntu.com/community/RestrictedFormats/PlayingDVDs") for legal reasons. 

I highly recommend setting up a "dual boot" so that you can still use Windows if you want. Linux is not Windows, and many users find it is useful to switch back and forth depending on the task.

Good luck!

### Post by Bartender on 2009-11-23
How much RAM that laptop got?  The processor speed isn't an issue, but RAM is.  If you've got more than 512MB you'll be fine.  1 GB of RAM would be better, but as long as you've got 512 or more you're in.

Do you have the original recovery disc or discs?  It sounds like you're willing to kick Windows to the curb, so I wanted to suggest an alternative.  Wipe the old Windows installation off and reinstall.  XP will definitely seem snappier with a reinstallation.

Then install Ubuntu (easiest would probably be to just let it auto-install to about half of the HDD).  That way you could still use Windows when you need to.

I've been slowly migrating away from Windows, but find there are some things that you just have to do on a Windows PC.  Like updating the firmware on my wife's Sansa Fuze.  Weird little things like that.

Then you could update Windows and leave it alone as much as possible.  Do all or most of your online stuff with Linux.  That way XP won't degrade as quickly.
