### Post by mikewhatever on 2011-04-09
Since there is no laptop edition, the desktop one is for both desktops and laptops. The netbook edition was intended for small screens, 1024x600 resolution.

To switch versions, select one at the login screen, under 'Session'.

### Post by DeviantGuy on 2011-04-09
It depends on what version you are running. If I was to guess. You are using 10.10, The Netbook edition uses "Unity" which has a launcher at the left. The Desktop edition uses "GNOME" And has a Applications menu in the top left corner. If you are running 11.04 It wouldn't really matter (Since the only real difference is the Desktop interface).
