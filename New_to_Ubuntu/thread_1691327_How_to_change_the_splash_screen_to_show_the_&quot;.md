### Post by RichieB07 on 2011-02-19
I recently installed Macbuntu via the .ISO image on my netbook, and when I start into it, it has the standard Ubuntu 10.10 splash screen (with the 4 dots underneath).  I was wondering how to change this to make it look like the Apple loading screen.

And honestly, I really don't care if you like Macbuntu or not, I just would like an answer to this.  It will help me learn how to change splash screens as well, so thanks for the help!!!

### Post by RichieB07 on 2011-02-19
I fixed this somehow.  I installed StartUp-Manager, and unclicked "Show Text" and it fixed it.  This changed it to the splash screen for Macbuntu.
