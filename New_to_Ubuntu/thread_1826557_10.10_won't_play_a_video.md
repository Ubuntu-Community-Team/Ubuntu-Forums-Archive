### Post by mikewhatever on 2011-08-16
What kind of video do you want to play?
The easy way to install most video/audio codecs is the 'ubuntu-restricted-extras' package. Installing it won't break other packages, and saves the trouble of hunting down individual gstreamer plugins.

### Post by seawolf167 on 2011-08-17
Running the below commands should get you up and running for any/all videos/dvds/etc.

```
sudo apt-get update
sudo apt-get install ubuntu-restricted-extras
sudo apt-get install libdvdread4
sudo /usr/share/doc/libdvdread4/install-css.sh
sudo apt-get install vlc
```

Then see if you can play the dvds/video/etc. files with mplayer or vlc
