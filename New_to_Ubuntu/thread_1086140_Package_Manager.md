### Post by swoody on 2009-03-03
Hi, and Welcome to Ubuntu, and the Forums!! ):P

For info on the Synaptic Package Manager and MANY other useful things for a beginner check out the link to the Free Ubuntu User's Guide in my signature! It's a great reference to have on hand while you're new to Ubuntu. I'd reccomend downloading it, and printing out a copy to have on hand nearby your computer for quick reference. The information on Synaptic starts on page 97 of the Guide, so start reading there!

It may be of great use to flip through the entire manual so you're familiar with the guide, and learn a bit more about Ubuntu before you dive in.

Again, welcome to the Forums, and let me know if you have any problems downloading the Guide :D
