### Post by niteshifter on 2009-06-15
Hi,

Click Applications -> Accessories - Terminal - this will open a terminal.

Type the command - in this case df -h

Select the output text in terminal by click and hold your mouse at the start, drag the mouse to the end and release the mouse.

Copy the selected text to the clipboard by pressing  Shift+Ctrl+C

Switch to the browser and in the Reply to Thread message box paste the clipboard by pressing Ctrl+V

Wrap the pasted text in "code tags" (this preserves formating, like tabs) by selecting it with the mouse and click on the # in the message box toolbar.
