### Post by grahammechanical on 2021-02-05
Windows is interacting directly with the motherboard hardware. Ubuntu is not set up to automatically override those motherboard settings. Something similar happens with WiFi. If WiFi is switched off in Windows it does not automatically get switched back on when we load Ubuntu.
