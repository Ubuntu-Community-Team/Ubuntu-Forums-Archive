### Post by Paqman on 2008-06-05
> **Omega Rhino said:**
> not sure how to check bios ... but if hard drive is fried, why does everything work off the CD?

Right after you machine boots it should flash up a message telling you what button to pres to get into BIOS. usually it's del, F2, or something like that.

As for the LiveCD, it'll work quite happily with a dead hard drive. It loads the whole system into RAM from the CD. Basically it's using the CD as a really slow read-only hard drive. Clever stuff.
