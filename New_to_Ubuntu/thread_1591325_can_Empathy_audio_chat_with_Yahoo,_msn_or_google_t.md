### Post by gradinaruvasile on 2010-10-09
Google has voice/video directly from the gmail web page and it works perfectly well on all platforms (Firefox/Chrome/Chromium/IE/Safari are the supported browsers).

Google talk works with Pidgin (tested it myself), did not test with Empathy but should prboably work. But the Gtalk web chat has better voice quality (like Skype).

Yahoo audio does not work, heck it doesnt even have file transfer. I wonder when it will...

Bottom line: If you want really reliable audio/voice chat go with Skype or the Gmail web plugin.
