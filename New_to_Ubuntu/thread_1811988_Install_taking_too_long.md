### Post by JC Cheloven on 2011-07-25
Unless you have a pc very short of ram or with some other disfunctionality, that long is not normal at all. The usual install methods are, in order of safety (safest first):

- Fresh install using the alternate disk (not the live cd).
- Fresh install using the live cd.
- Upgrading via web from a previous version. 

I personally like to fresh install, mainly because I shall have incured in a safety problem, this will restart everything for sure.

Well, if upgrade failed, you have to install. Fisrt put aside your sensible data (a live cd can help a lot with this), and then choose your fresh install method.
