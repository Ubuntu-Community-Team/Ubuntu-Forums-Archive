### Post by kroq-gar78 on 2010-12-20
First of all, welcome to the forums.

Darkwave Studio is, last I checked, a Windows program. Linux can't run windows programs unless you install a program called WINE (recursive acronym; Wine Is Not An Emulator, not WINdows Emulator ). Look into that to install it [http://winehq.org](http://winehq.org)

To install it, simply type in a terminal (in the top right, Applications -> Accessories -> Terminal):
```
sudo apt-get install wine
```

### Post by kroq-gar78 on 2010-12-20
WINE is a program to let some Windows programs run on Linux. Read more about it here [http://en.wikipedia.org/wiki/Wine_%28software%29]("http://en.wikipedia.org/wiki/Wine_%28software%29") and here [http://www.winehq.org/](http://www.winehq.org/)

Darkwave is a Windows application and you need to run it on Ubuntu (linux). WINE lets you do this.

To see how well Darkwave runs in WINE, go here [http://appdb.winehq.org/objectManager.php?sClass=version&iId=17809](http://appdb.winehq.org/objectManager.php?sClass=version&iId=17809)
