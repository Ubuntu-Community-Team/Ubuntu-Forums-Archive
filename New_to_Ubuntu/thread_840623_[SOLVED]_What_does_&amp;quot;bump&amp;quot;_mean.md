### Post by ndizazzo on 2008-06-25
bump simply means that a user is "bumping" the post to the top of the forum topic list.

The character you're speaking about is the pipe character: '|'. It is indeed on the button with '\'.

In (Un/Lin)nix, the pipe serves as a command line output redirect to the utility following the pipe, example:

echo "hello, I'm Mr_FixIT" | cat > my_file

would be a long way to write "hello, I'm Mr_FixIT" to a file. The output from echo is read by the cat utility, and then redirected to a file.

Hope this helps.
