### Post by freak42 on 2008-08-14
pidgin (as some other apps (gnome-apps?) allows you to set and/or override shortcuts in menus:
check your shortcut key behind the buddies->New Instant Message... entry?
Is it 't'?

If yes.. hover over the New Instant Message entry (go over it with your mouse but don't select/click it. Then press ctrl-m (which is the default shortcut for this entry.. it should override your other 't' - shortcut.

hth

### Post by freak42 on 2008-08-14
I think I got the wrong menu-entry as the pidgin main window shortcuts are not active while in a chat window.

But please check the shortcut key behind Conversation->New Instant Message... (in a chat window) and then proceed as above (check if it's shortcut is 't' and ''overwrite'' it)
