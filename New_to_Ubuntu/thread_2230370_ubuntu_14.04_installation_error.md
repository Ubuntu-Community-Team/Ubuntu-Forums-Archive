### Post by Topsiho on 2014-06-19
I am very sorry about you having to bury your friend right now.

You may be sure that on this forum everyone  wants to help you. But to do so full and correct information is needed.
It would be nice if you could find some "Ubuntugeek" living near you who can help you. He'll certainly be very glad to be able to do so.

Update: the installation of packages (programs) on the computer, some to remove security issues, and some to improve programs. In Windows speech also called patching.
Upgrade: to move a system from one version, say Ubuntu 12.04, to the next version, Ubuntu 14.04. You could consider this an enormous update, where all programs on the computer are updated in one run.

An upgrade is a very time consuming process, which can last for hours, and a large amount of data has to be downloaded, and installed, replacing the old programs with the new versions.

Ubuntu 11.10 is supported during 9 months only, from October 2013, so is not worth your trouble installing it.

12.04 will be supported until April 2017, Ubuntu 14.04 until April 2019, both being LTS (Long Term Service) versions.

It is unwise to use not supported versions, as you won't get updates ("patches" :) ) anymore.

Again, I am sorry, and suppose you might postpone this all to a later date.

Topsiho
