### Post by gngf123 on 2011-01-13
64bit version is only really needed with those people with 4gb ram and over, so the 32 bit version should be fine.

XFCE is more lightweight than KDE or Gnome, so will run slightly better, but also needs a tiny bit more work to get used to. I would definitely try it out on a live CD and see what you think.

### Post by akand074 on 2011-01-13
Ubuntu would work just fine with your specs, but I hear the most lightweight of all the desktop environments is LXDE, so [Lubuntu]("http://lubuntu.net/") would be the best choice in terms of resource usage. Also 64 bit not only allows you to use more RAM, but is also exponentially more efficient at using RAM. If your system supports 64 bit, then installing the OS in 64 bit may not be a bad idea, but it's not really mandatory, and I doubt you'll really notice a big difference.
