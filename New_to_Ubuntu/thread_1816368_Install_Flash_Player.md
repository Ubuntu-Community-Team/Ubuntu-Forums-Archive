### Post by kaldor on 2011-08-01
This isn't Windows; there's no need to install stuff from the web for most things.

In the Ubuntu Software Centre, install the "ubuntu-restricted-extras" package. This will install Flash, codecs, and everything you'd need in terms of essential media ability.
