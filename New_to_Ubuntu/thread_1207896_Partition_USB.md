### Post by starcraft.man on 2009-07-08
Just plug it in (the external drive) and partition. You can either partition while in Ubuntu or you can do it from a live CD if you also need to modify directories mounted during regular use. 

To do it while logged into Ubuntu, install gparted by your preferred means (see my guide in sig for instructions). The package is simply listed as gparted. Then you just run it with sudo power from the System > Amin > gparted.

To download a live CD to use, you can use the live [GParted cd.]("http://gparted.sourceforge.net/download.php") You'll also note that the gparted site has a documentation section, on the left menu. If your unfamiliar with it please read the documentation BEFORE doing anything your uncertain about. If you got questions ask.
