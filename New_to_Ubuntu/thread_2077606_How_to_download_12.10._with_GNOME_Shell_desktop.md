### Post by grahammechanical on 2012-10-29
Ubuntu had the opportunity to use Gnome shell as the default desktop more than a year ago but the developers chose instead to use their own invention of Unity.

So, why do you expect Ubuntu to provide you with a download image of Ubuntu with Gnome shell as the default and only desktop environment.

As ztac has pointed out you can install Gnome shell from the Ubuntu Software Centre but notice this comment:

> Canonical does not provide updates for Gnome shell. Some updates may be provided by the Ubuntu community.
