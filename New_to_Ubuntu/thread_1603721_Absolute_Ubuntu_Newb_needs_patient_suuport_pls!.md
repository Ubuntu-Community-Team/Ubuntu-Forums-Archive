### Post by coffeecat on 2010-10-23
For downloading an ISO of Ubuntu, click on the download link in the Ubuntu website:

[http://www.ubuntu.com/](http://www.ubuntu.com/)

Once you've got the ISO (make sure you get the 'desktop' version), you can use Windows, MacOS or Linux to create a bootable CD. Instructions here:

[https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)

Change your machine's BIOS to boot from the optical drive first and then start up with your CD. When you see a purple screen with two small icons at the bottom, tap the space bar. Choose your language from the selector and then choose the first item in the menu - try Ubuntu. (Don't worry if there's a delay of several seconds after choosing this.) You will be booted into a usable (but sluggish) live desktop where you can test all your hardware for compatibility. If you want to install, you simply double-click on the install icon on the desktop and follow the prompts. If you are sure you want to erase Windows, choose the 'use whole disc' option.

**Edit:**@Emily, just one thing to add before you commit yourself. I see you have backed up your data. Good! So long as you weren't running any unusual software, Ubuntu should have equivalents to read and use your files: OpenOffice for MS Office files, photo editors, music players and so on. But if you were using somethng out of the way, just post back and someone can help you.

Also, be aware that for licensing reasons Ubuntu comes with only a limited sub-set of multimedia codecs. If you need to play MP3s or watch DVDs, it's a simple enough task installing the requisites. Just ask!

### Post by trikster_x on 2010-10-24
Welcome to Ubuntu Emily.  A couple things to help you get started:

[https://help.ubuntu.com/community/Medibuntu]("https://help.ubuntu.com/community/Medibuntu")
Above is a link to the Medibuntu info page.  You'll need the info here to enable DVD playback of newer DVDs.  I'll give the important parts of it below.

You will also want to add the Ubuntu restricted extras to your computer to allow MP3 playback, flash capabilities, and some other audio and visual codecs that you need for media.  Just open a terminal by going to Applications>Accessories>Terminal and paste the following in and press enter:

```
sudo wget --output-document=/etc/apt/sources.list.d/medibuntu.list http://www.medibuntu.org/sources.list.d/$(lsb_release -cs).list && sudo apt-get --quiet update && sudo apt-get --yes --quiet --allow-unauthenticated install medibuntu-keyring && sudo apt-get --quiet update
```

This will get you the Medibuntu repository so you can download some of the neccassary packages.

```
sudo apt-get --yes install app-install-data-medibuntu apport-hooks-medibuntu
```

This adds the medibuntu program list to your package manager.

```
sudo apt-get install libdvdcss2 ubuntu-restricted-extras
```

This will get you going with the multimedia codecs you need for playback of most media files.

Also check out [http://gnome-look.org/]("http://gnome-look.org/") to get started on customizing the look of Ubuntu.

Hopefully you don't have any major problems, since at times Ubuntu can have a bit of a steep learning curve.  And never hesitate to ask a question on the forums, we love to help :D

Again, welcome and have fun!
