### Post by errolgreer on 2014-05-13
I have a multiboot set up on my PC. In order: Windows 7 32bit, Windows 7 64bit, Windows 8.1, Ubuntu 14.04.
I set this up using Easybcd 2.2. All work well, except that after shutting down Ubuntu, if I select Win 7 32 bit (The default), it starts booting, then hangs.
If I restart the PC, and boot into Win 8.1 or Win 7 64bit, then restart, it then boots into Win 7 32 bit correctly.

I dont have this problem after using Win 7 64bit or Win 8.1, only after Ubuntu.

Any ideas?

Thanks, errol

---

### Post by errolgreer on 2014-05-25
Found one of my hard drives faulty. After removing it, the problem went away!
