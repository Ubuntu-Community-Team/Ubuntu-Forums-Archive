### Post by Ms_Angel_D on 2010-08-28
Ubuntu comes with empathy by default which connects to multiple instant messaging networks. When you first run it you'll be presented with an account configuration window where you can set up your various accounts.

If you want something more MSN like you can look in the software center and try emesene or amsn.

There are also a two other instant messenger clients which support multiple networks  pidgin is gtk based and kopete which is qt based and is installed by default in Kubuntu.
