### Post by seawolf167 on 2011-07-06
A "torrent" is the file that says where to find the trackers and the file information for the specific item you are going to download. uTorrent, Transmission, Vuze, etc. are all programs that use the information from the torrent file (*.torrent) to actually transfer the file to/from your computer.

11.04 should come with Transmission installed by default to handle .torrent files. If not you can install your own BitTorrent program through the Software Center (I recommend either Transmission or Deluge for just getting started)

You can read more about it [here]("http://en.wikipedia.org/wiki/BitTorrent_%28protocol%29").
