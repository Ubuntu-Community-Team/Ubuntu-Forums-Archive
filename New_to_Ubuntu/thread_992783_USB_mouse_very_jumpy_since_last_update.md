### Post by Tomatz on 2008-11-25
Try lowering (or raising) the mouse polling frequency. This is the amount of times per second the computer recognises the laser/ball's position:

[http://www.linux-gamers.net/modules/wiwimod/index.php?page=HOWTO+USBPolling](http://www.linux-gamers.net/modules/wiwimod/index.php?page=HOWTO+USBPolling)

This can cause problems if set too high or too low.
