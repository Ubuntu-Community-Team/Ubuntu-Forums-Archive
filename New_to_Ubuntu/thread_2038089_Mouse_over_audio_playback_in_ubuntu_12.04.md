### Post by ld114 on 2012-08-10
Hi

This function has disappeared from Nautilus the file manager, but you can preview files (including audio and video files) using a package called gnome-sushi which is in the Ubuntu Software Centre. Once gnome-sushi is installed left click on the file you want to preview (to select it) and use the space bar to turn the preview on and off.
