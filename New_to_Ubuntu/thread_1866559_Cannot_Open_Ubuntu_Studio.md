### Post by grahammechanical on 2011-10-26
I may be wrong on all this but Ubuntu Studio is not a program that you run. It is supposed to be a variation of Ubuntu with programs related to audio and visual production.

I installed Ubuntu Studio 10.10 and 11.04 in a separate partition from the alternative CD. During the install you get the choice of choosing which sets of programs you wish to use - audio creation programs or visual creation programs and other sets of programs that I cannot now remember. Or you can choose to install the whole lot on offer.

When you install the Ubuntu studio desktop on top of an existing Ubuntu installation you only get the user interface. You do not get the specially selected programs that come with a Ubuntu Studio install. You can install all these programs from the Software Centre anyway. 

The idea of creating Ubuntu Studio is to have an operating system dedicated to using audio and visual creative programs in one package that music and video professionals would like.

To put it another way, Ubuntu studio is not a music editing/creation program. It is not Visual Studio for Ubuntu. It is a tailored operating system. 

Check out the web site.

[http://ubuntustudio.org/]("http://ubuntustudio.org/")

Regards.
