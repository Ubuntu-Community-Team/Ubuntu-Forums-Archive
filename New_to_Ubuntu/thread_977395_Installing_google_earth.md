### Post by jespdj on 2008-11-10
The easiest way to install Google Earth is via the [Medibuntu repository](http://medibuntu.org/). Follow the [instructions here](https://help.ubuntu.com/community/Medibuntu) to add the repository and then install it with:
```
sudo apt-get install googleearth
```
