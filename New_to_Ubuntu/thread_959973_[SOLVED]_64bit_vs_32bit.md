### Post by SunnyRabbiera on 2008-10-27
Well these days there isnt much of a difference between a 32bit processor and a 64bit one as most newer Intel's have 64bit capacity.
If you buy a intel dual core or a AMD64 it wont make too much of a difference though I would say the intel will be better as the latest intels blow anything from AMD out of the water.
As for Linux being able to use higher memory it can, it should have no issues.
However if you have a 32bit kernel it will be limited to 4GB of memory as that is the standard limit of a 32bit kernel unless its modded.
if you have like 6GB of memory then a 64bit kernel will come in handy.
its the kernel that matters in cases like this more then the processors.
