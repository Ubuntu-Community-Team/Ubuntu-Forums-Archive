### Post by audiomick on 2013-01-11
Hi.

Firstly, you can check the version with the system monitor. Look for it in the search box in the dash. The first tab will tell you which version it is. Having said that, if the manual that is installed is for 12.04, then that is most likely the correct version.

As far as the CDs go, "unknown" might simply mean that the creator of the CD didn't name it. You shouldn't have to give it a name to play it. 

I believe, though, that an un-named CD should come up as "Audio CD". I suspect that the system is not recognising it as an audio CD.

What you could try is to find the CD using the file manager and look at it to see what sort of files are on there. It is possible that it has been burnt as a data CD with mp3 files on it or something, or that it is simply a bad copy. If it is a correctly made Audio CD, the system should actually offer to open it with the appropriate player when you insert it.
