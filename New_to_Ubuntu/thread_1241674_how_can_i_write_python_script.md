### Post by Ranemills on 2009-08-16
All you need is a text editor. As to which one to use, I can't really give you much advice. I personally use SPE, but I haven't tried any others. [http://ubuntuforums.org/showthread.php?t=340028](http://ubuntuforums.org/showthread.php?t=340028) might be a good place to start for suggestions as to which program to use.

Then, when you've written the script just save it with the .py extension and you can run it! To run you can use the terminal with 
```
python file.py
```I know you can also run it inside SPE, once again I'm not sure about any others.

Hope this helps.
