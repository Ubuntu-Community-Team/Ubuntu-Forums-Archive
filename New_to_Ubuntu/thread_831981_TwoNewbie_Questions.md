### Post by kk0sse54 on 2008-06-17
> **J.sanchez said:**
> 1. I've installed Xubuntu and my MacBook Core 2 Duo, and sound and wireless are non-functioning as people have mentioned, even after trying to fix it following instructions given in the documentation; does ubuntu have the same problems?

2. My current partition size is a measly 10GB, should I reinstall Mac OS X so that I can resize the partition?

You don't need to reinstall MAC OS X in order to resize a partition just stick in a liveCD with Gparted and resize from there. As for 10 GB being too little that depends if you are storing data such as music, videos, and etc etc on that partition which in that case it would be a good idea to have a little bit more free space unless you have a separate home partition or data partition so that those 10 GB are only for system and program files.
