### Post by stoneage on 2012-07-21
> **x_lauren said:**
> ...can someone please tell me where the ARD folder is on Ubuntu? 
There isn't one.

If there are any config files the Ubuntu version can share from the Windows one they would be in the application's hidden folder in your Home directory.

View -> Show hidden files (CTRL+H) in your file manager. 

Please be aware linux is not like Windows and does not work like Windows.
