### Post by zemega on 2013-06-01
Some online games site like that might have Unity 3D games in the collection of the Flash games. Games that use Unity 3D does not work in Linux, as there is no Unity Web Player for Linux currently. I usually play at Kongregate. Some of the best games there requires Unity Web Player instead of Flash. Note that I'm not talking about Unity inside Ubuntu.
