### Post by Gen2ly on 2009-04-22
if you're talking about system-wide, i don't think this can be done.  A few releases ago i was told that the X server has a built-in scroll speed that can't be adjusted and i haven't heard anyting contrary since.  if you're talking about Firefox i has a built-in setting that can be adjusted.  Type about:config in address bar, then "mousewheel.withnokey" (without the quotes) and change "...sysnumlines" to false, and "...numlines" to the number of lines to scroll.  I prefer to use 7.
