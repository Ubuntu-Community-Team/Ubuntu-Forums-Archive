### Post by marco123 on 2009-12-10
Or install a command line only system using the Alternate CD, then follow the instructions here: [https://help.ubuntu.com/community/ServerGUI](https://help.ubuntu.com/community/ServerGUI)  for a basic setup I'd recommend installing "gnome-core" instead of the full desktop.

> sudo apt-get install xserver-xorg xserver-xorg-core

Installs X.

> sudo apt-get install gnome-core

Installs a very light Gnome desktop.
