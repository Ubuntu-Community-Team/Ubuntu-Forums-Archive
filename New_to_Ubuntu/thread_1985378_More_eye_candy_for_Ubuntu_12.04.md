### Post by black veils on 2012-05-23
you could try kde, it might be less confusing to learn!  

```
sudo apt-get install kubuntu-desktop
```

**or**

```
sudo apt-get install kde-standard
```


there are other choices also, for more minimal desktop, meaning it will bring less applications with it, but that can also mean missing functions.


remember, when you add another desktop environment to your current system, you will have the applications of both desktops in both, they will show in the menus. this can be useful, but it can also be confusing.


and after the install, you are probably best to restart the system, and at the login screen, click the icon at the right of username, to change your desktop to kde (or back to unity).

### Post by goaliedude3919 on 2012-05-23
There are a few programs you can download that give you some more customization options in Ubuntu: Advanced Settings, Ubuntu Tweak, My Unity, and CompicConfig Settings Manager. The last one you need to be very careful with though. When you open it, it will tell you to be careful because some changes you make could leave you with a desktop that doesn't work. So if you don't know what something does, make sure you either google it or ask someone here before you mess with it.

> **rgreener25 said:**

Huge +1 on this.  I just recently switched over to gnome shell and while I have actually been a huge fan of Unity, I am finding that I prefer gnome shell.  Out of the box I thought it was awful but after installing the following, I really like it much better than Unity:

1. gnome tweak tool (ok, I had this for Unity as well)
2. several gnome shell extensions (visit [http://extensions.gnome.org](http://extensions.gnome.org) to see the full list)
3. AWN -- a dock, which I moved to the left side so it functions and feels much like the Unity launcher


I also think that it performs much better than Unity (at least on my laptop with has an ATI card.  The nvidia card on my desktop at home performs great).

I highly recommend you install gnome shell and then just logout, click on the little gear icon, and change it gnome shell.  I think it's easy to customize, and in my opinion, "prettier" anyway.


Don't forget to install the Faenza icon set!  I actually use Faience-azul myself (same author...builds on top of and requires Faenza), because I like the blue folders.  :)


If you need any help let us know.
