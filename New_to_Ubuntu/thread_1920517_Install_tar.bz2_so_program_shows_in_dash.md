Hi, I downloaded a tar.bz2 file (Blender 2.61) and managed to unpack it. The program doesn't need to be compiled and runs when you click the executable. However, it doesn't show up if I search for it in the dash. Is there a way to "install" the program so that Ubuntu recognizes its existence? Thanks!

---

### Post by 3rdalbum on 2012-02-05
Use the "Main Menu" program (it's available in the Dash) to create a "menu entry" for it. Then the program will appear in the Dash, and can even be placed in the Launcher.
