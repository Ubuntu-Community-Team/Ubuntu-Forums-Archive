### Post by Copernicus1234 on 2009-01-05
If you are into networking, you should really install Wireshark to sniff packets on your network. It will really open your eyes to what is being sent back and forth.

The command is:

**sudo apt-get install wireshark** in the terminal.

Then it will show up in the Gnome menu under the Internet category. Cant be much easier than that. :)

You will find that the Linux network commands are generally like on Windows, but provides more information. Stuff like route, netstat, ping, traceroute etc are some examples.

Linux is great for learning how networks work.
