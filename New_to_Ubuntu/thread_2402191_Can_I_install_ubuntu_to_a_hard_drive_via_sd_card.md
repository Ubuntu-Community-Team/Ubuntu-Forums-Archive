### Post by Autodave on 2018-09-27
As long as you can boot the machine using the sd card, yes you can.  You need to down load the install of your choice, burn it as an .iso file to the sd card, and then boot your machine to it.  

That being said, here are a bunch of things to consider first:
1. What operating system is on there now?
2. Are you going to remove that operating system or do you want a dual boot setup?
3. Make and model of PC?
