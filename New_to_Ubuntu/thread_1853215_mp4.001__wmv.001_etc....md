### Post by papibe on 2011-10-01
Archive Manager usually takes advantage of all command line compression applications.

The first thing I would try is to open the File Manager, go to the directory where the files are, right click on one of the files and select 'Extract Here'.

If that doesn't work on a random file, look for the one with the extension .rar, .zip or alike.

Hope that helps,

### Post by ubuntu27 on 2011-10-02
In most Linux distribution, you can find [LxSplit]("http://lxsplit.sourceforge.net/") in the repository. It is used to slplit and merge files.

You can install it with your favorite package manager or with the terminal.

```
sudo apt-get install lxsplit
```


LXsplit is terminal based, so you have to run it inside the terminal.

Open the terminal and type 
```
man lxsplit
``` to read the guide or manual.
