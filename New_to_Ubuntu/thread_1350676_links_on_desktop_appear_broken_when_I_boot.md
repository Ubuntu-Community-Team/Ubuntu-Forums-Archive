### Post by ajgreeny on 2009-12-09
I'm not totally sure it will work, but try making launchers on your desktop by right clicking in empty desktop->Create Launcher and use Location in the top box, name the launcher anything appropriate, then enter the folder pathway, eg /home/username/music.  You can then change the icon you want to use by clicking it and navigating to the icon you want.  It should then stay exactly as you want, as the launcher will not be a link, but a .desktop, ie launcher file.
