### Post by lovinglinux on 2009-05-31
I recently noticed that if I setup compiz cube caps and other settings that use images with the wrong image path (missing file), compiz keeps using much more CPU than usual.

So check if any images used by compiz are missing.

Just wanted to let other people know about this, because it could be hard to troubleshoot CPU usage in this situation.
