### Post by arnab_das on 2010-04-08
> **asharpham said:**
> When I right click an application and select "Add this launcher to desktop" the icon that should show on the desktop is invisible. If I repeat the process it tells me the icon is already there. However it isn't visible. Anyone know what may be wrong?

press ALT + F2 from your keyboard. now type "gconf-editor" without the quotes. now search for apps > nautilus > desktop and select the appropriate options about showing icons. thats it.
