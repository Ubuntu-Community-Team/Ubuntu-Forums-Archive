Few tips here.

1 -- Don't post your password constructs, even if it's only one character.
2 -- Don't copy and paste passwords, depending on where you copy pasted it from the encoding on the charcters may get screwed up and not entered properl. For instance that ampersand could end up as any of the following after paste

```

&#38
%26
0x0026
x26
:26
%26
&amp;

```

Depending on where you copied it from. Type your password see if that works.

3 -- This is not OS dependent, as haqking said check out Facebook's help session, also consider changing your password.

Hope this helps.
