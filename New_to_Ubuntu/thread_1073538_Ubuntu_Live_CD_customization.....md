### Post by barney_1 on 2009-02-18
If you're making a very large number of copies of your live CD it's worth the work and a walk-through can be found here (written for 6.04 but should work for more current releases): [http://www.atworkonline.it/~bibe/ubuntu/custom-livecd.htm](http://www.atworkonline.it/~bibe/ubuntu/custom-livecd.htm)

If you just need one package or a few packages I would download them to a thumb drive and use that after the live-cd is mounted.  The apt-on-cd sounds like a good thing as well.

This might be of interest as well:
[https://wiki.ubuntu.com/LiveUsbPendrivePersistent](https://wiki.ubuntu.com/LiveUsbPendrivePersistent)
