### Post by beew on 2010-11-21
I suppose dual boot wouldn't work because OP's Windows is completely toasted. It just isn't worth it to go through the troubles to set up a dual boot and try to repair Windows from the Ubuntu partition. OP can try repairing Windows with a live usb if he/she still hopes to revive Windows.

Testing with a live usb _with persistence_ would be a good idea if OP's computer supports booting from usb (most newer ones do, by "newer" I mean 5-6 years old). I am sorry, forget about the live CD, it is clunky, slow and it cannot save settings and burning it requires relatively complicated tools (I have read people suggesting Nero, no kidding) and people have reported problems for burning with the wrong speed etc. Live CD is an archaic technology, you won't suggest people to use a floppy nowadays would you? :)

To create a live usb with persistence in Windows check out pendrivelinux.com (I suppose OP has to somehow get a hold of a working Windows machine anyway in order to download the iso and make a live CD)

### Post by apollothethird on 2010-11-21
Hi, Aesthetics.  I see you have lots of suggestions.  One of the best ones I see in this thread is for you to dual boot; the purpose being, from my perspective, is to preserve your current personal data from Windows.  If you don’t have any personal data (such as pictures, documents, notes, etc) then I wouldn’t see a need for a dual boot.

You can use Gparted (Gparted live cd) to resize your Windows Partition to a smaller size so that you’ll have more room for Ubuntu.  I also suggest that you install Ubuntu (from the Ubuntu Live CD) rather than spending time testing it.  Since you’re interested in it, you’d have a better experience with it installed on your Hard Drive rather than looking at it through other means which, while it looks good, are slower and have limitations that you shouldn’t have to deal with.  If on the unlikely event that you don’t like it, you can just remove that Ubuntu partition and use it for a different OS.

From Ubuntu you can copy any important data from your Windows partition and eventually remove the Windows partition, and either reformat that partition as a Ubuntu data drive or use Gparted to resize your Ubuntu partition to take up the slack.

By the way the many preview modes are a nice method for people who are still running Windows and you want to convince them to look at it.  But for someone who has made up their mind to really sample the OS, they’d best experience it in its fullest.  You should find just about everything and anything you need a click away in the Software Center via the Applications menu.

Welcome to Ubuntu.  As you see you have lots of help… lots of options.  Enjoy!
