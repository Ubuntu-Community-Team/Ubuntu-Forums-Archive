### Post by aeden.d on 2015-07-16
If Thunderbird is installed and is the default mail application for your system try this:

Start Thunderbird
Go to Tools->Add-ons->Extensions and disable Thunderbird extension "Messaging Menu and Unity Launcher integration". 
Close Thunderbird. 
Logout and Log back in to your system.

I'm not sure about Dropbox or Steam since I don't have those installed, but you might check the properties of each as you did in Thunderbird.
