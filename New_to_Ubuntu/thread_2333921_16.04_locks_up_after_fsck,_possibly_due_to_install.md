### Post by oldfred on 2016-09-11
Have you paid attention to all the issues of AMD and 16.04. 
You can only use the open source drivers. Kernel had to be updated and AMD's proprietary driver does not support those updates. They are in the middle of major changes to open source driver to make it work, but not released yet. There is a beta now.

       [URL="https://wiki.ubuntu.com/XenialXerus/ReleaseNotes#fglrx"]https://wiki.ubuntu.com/XenialXerus/ReleaseNotes#fglrx
[/URL]
 16.04 Release notes
When upgrading to Ubuntu 16.04 from a previous release, both the fglrx driver and the xorg.conf will be removed, so that the system is set to use either the amdgpu driver or the radeon driver (depending on the available hardware).

[http://ubuntuforums.org/showthread.php?t=2321963](http://ubuntuforums.org/showthread.php?t=2321963)

      [/URL]
