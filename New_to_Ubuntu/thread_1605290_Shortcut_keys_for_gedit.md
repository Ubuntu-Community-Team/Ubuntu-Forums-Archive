### Post by Jeanke on 2010-10-25
Hi John,

Not sure if this is exactly what you mean, but you can create your own shortcuts under System -> Preferences -> Keyboard Shortcuts 
There you will get a list of possible shortcuts and the possibility create new ones.
Choose option "Add", give it a name and a command.
For example: if you want to  start gedit with a key combination you can just enter command: gedit
Once you added the shortcut it will appear in the list and you can assign a key combination to it.

For shortcuts within the gedit itself just check the menus in the program, the possible shortcuts are listed there.
