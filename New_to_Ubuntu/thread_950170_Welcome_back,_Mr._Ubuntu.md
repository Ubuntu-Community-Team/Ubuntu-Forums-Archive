### Post by pastalavista on 2008-10-16
I downloaded [Unetbootin]("http://unetbootin.sourceforge.net/") and, with it, installed the latest Ibex beta on a 1GB flash drive. It works fantastic.. much faster than a live CD. Try it before you install it. I'll keep my Hardy Heron for a while, though. I totally wiped Windows Vista because it works so well. I'll probably upgrade to Intrepid Ibex when it has been out for a while to be sure all the bugs are worked out. But I love using it (the beta) from the USB stick. It's faster than my Vista was from the hard disk.

edit--
Dual booting is easier than ever and works better than ever. Be sure to defragment your Windows disk before you install Ubuntu to maximize your available disk/partition space.
