### Post by Orange_and_Green on 2008-10-17
Oh, now I see what you mean.

Well, in theory it is possible that authentication for incoming mail can be different from outgoing mail so it's actually normal that firefox asks them separately. If you don't want to type them in every time, you can either check the "remember password" checkbox or, if that's too insecure, you might want to go to Edit->Preferences->Security and use a master password.
