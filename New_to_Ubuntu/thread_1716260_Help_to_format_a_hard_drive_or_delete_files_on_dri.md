See how the second screenshot has a mountpoint of /

That means that is the drive with ubuntu on it, the actual system that is.

Fortunatly you can't format it while it's mounted and you can't unmount it without shutting down your system.

If you are doing this from Ubuntu itself, you can't go wrong. If you are doing it from a live cd, alot of care is needed.
