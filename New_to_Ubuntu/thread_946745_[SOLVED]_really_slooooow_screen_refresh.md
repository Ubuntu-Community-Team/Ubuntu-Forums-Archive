### Post by BMWDriver on 2008-10-16
LCDs don't actually have the same kind of technology as CRTs. So the screen refresh rate does not actually apply per se. We would actually be talking about 5 ms, which is the reaction time of a pixel to change color in microseconds. And lately, it's not been an issue in choosing a monitor since most screens are now apt at gaming. Gamers needed a minimum of a reaction time of 8ms, back when some screens did about 12ms or more.

So 60Hz is there just because there are still some CRTs out there, some of which are capable of higher resolutions than the best LCD can do. This simplifies things for adapting both of the technologies on the same computer - some sort of compromise if you will. Forcing a higher rate may actually not be ideal for the LCD.
