### Post by alejandro.mc on 2012-03-16
In case you don't understand the above or still have the problem that the birthdays are not showing up do this:

Press "Contacts" button, now you are looking at your contacts. Select "Personal", if not selected (or select whichever calendar you want to make default and have it's contacts birthday's shown). Go to Actions > Address Book Properties. Tick mark as default address book.
