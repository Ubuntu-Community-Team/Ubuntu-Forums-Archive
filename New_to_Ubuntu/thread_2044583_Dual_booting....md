Hi ! Sly14Cat, Welcome to the forum.

 ok... The advise is ---- Windows tools for windows problems.
First defrag windows from windows at least twice, use windows tools to shrink down the windows partitions (can only have 4 partitions to the drive) see how many partitions windows is using.
 Then, you can install ubuntu from the install CD in the "free" space created in the above. [I regret to say if no primary partition is available for ubuntu, others with more knowledge of windows will have to advise].
