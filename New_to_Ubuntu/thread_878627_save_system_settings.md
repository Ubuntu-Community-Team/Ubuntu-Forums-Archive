### Post by ajgreeny on 2008-08-03
Mostly this is all in your /home/username/.hiddenfoldername but there are also some settings in other file system folders, eg /etc, /usr, so it will not be too easy to do what you want.  Also, don't forget that a lot of the configurations will be specific to youir username so any transfer to other users may cause problems simply because they will have a different username.
