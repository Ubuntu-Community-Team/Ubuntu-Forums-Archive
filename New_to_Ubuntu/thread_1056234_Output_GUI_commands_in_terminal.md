### Post by joep-vd on 2010-03-15
What could help with some programs, is to start 'em from the terminal. Then, if this program is nicely made, it shows some of the stuff it does in the terminal. 

The problem here is that every time you give some command in the terminal, you are saying something along the line: "This program, these options, there." And within a GUI, there could be multiple options that are written in some configuration files of possibly multiple programs. 

I guess that the software that you are requesting could be made, but all the programs that you use should be changed so that it will output these things. So, that is quite a big task to get all the developers to implement your wishes... 

You could help forcing yourself to understand more of the terminal and working with text based configuration by using another flavor of ubuntu, crunchbang for example. 

Enjoy your learning curve!
