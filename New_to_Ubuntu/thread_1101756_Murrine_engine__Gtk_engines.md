### Post by mvalviar on 2009-03-20
First off, let me let you that I'm not an expert at this subject. I don't think your questions are easy at all. But I'll give it a try.

What you see on your screen is the GUI. At the very core of this is the X Window System that provides Gnome something to build upon. Gnome is the desktop environment in ubuntu. What provides some features to Gnome, like changing themes, is GTK.
GTK is the GIMP Tool Kit. In a nutshell it used to build a GUI. I don't think its correct to refer to gtk as an engine. Murrine is a gtk engine. Its makes the GUI look glassy. 

I think ubuntu has gtk running murrine by default. You need not to choose between them.
