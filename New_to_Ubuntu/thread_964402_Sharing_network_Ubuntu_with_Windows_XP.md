### Post by ad_267 on 2008-10-30
If you want to access shared directories on the windows machine you only have to install the samba client and smbfs. Then you can open up the file browser and go to File - Connect to server to access the windows shares. If you want to share stuff from your Ubuntu machine you need to install the samba package.
