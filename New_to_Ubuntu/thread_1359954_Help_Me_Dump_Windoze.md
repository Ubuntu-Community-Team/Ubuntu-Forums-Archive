There are other Linux distributions (like Puppy or DSL) that are more direct competitors for obsolete Windows versions like XP. Ubuntu is designed to compete with Windows 7, Mac OSX, etc.

Here are the system requirements for Ubuntu:

> Ubuntu should run reasonably well on a computer with the following minimum hardware specification. However, features such as visual effects may not run smoothly.

    * 700 MHz x86 processor
    * 384 MB of system memory (RAM)
    * 8 GB of disk space
    * Graphics card capable of 1024x768 resolution
    * Sound card
    * A network or Internet connection

As you can see, they are higher than Windows XP (but lower than Vista or Windows 7).

### Post by QLee on 2009-12-24
[QUOTE=Big_tone]
Lots of people love fedora but it lacks personality and there are issues with invoking root with the "sudo" or "su" commands.  You get a weird message; something like,"you are not on the sudoers list! this will be reported"... or something to that effect.[/QUOTE]

That is standard behaviour, not weird. You would get that in a Debian install too if you tried to sudo and were not in the sudoers list. The implication of that message is that root receives an email that the user tried to sudo. This probably makes more sense in a multi-user or/and "work" environment or maybe in a family with members who constantly test security of the system. The *buntus are configured differently with the sudo set up to make it easier for an admin user to use root credentials for a command and other changes so as not to confuse newer users too much, just a difference in default setup.
