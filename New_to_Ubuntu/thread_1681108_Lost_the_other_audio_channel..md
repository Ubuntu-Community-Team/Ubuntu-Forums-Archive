### Post by lukeiamyourfather on 2011-02-03
> **Metalmad said:**
> Umm might sound stupid but what is a live disc?

Insert an Ubuntu installer disc and then instead of the install option click the try option. It will boot into Ubuntu from the installer disc (or USB flash drive) without changing anything on the hard disks. This runs Ubuntu in the same configuration as a clean install.

The idea is if your sound works there without issue then the problem is in the software. If the Audio has the same issues there then likely the hardware (or failing hardware) is the root cause of the issue.
