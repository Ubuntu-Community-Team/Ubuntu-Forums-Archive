### Post by Mark Phelps on 2011-10-18
Sorry, unlike with MS Windows, there is no option to "downgrade" to a previous version of the OS.

If you want to go "back" to 10.10, you need to do the following:
1) Back off the files you want to save
2) Boot from the 10.10 CD
3) If it provides an option to "use entire drive", then select that and install it.  This will overwrite everything you have now.
4) Copy back the files you saved.
