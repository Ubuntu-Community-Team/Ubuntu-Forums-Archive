### Post by grahammechanical on 2011-05-23
The other year I purchased broadband from Plusnet and I got a free wireless router and the router came already setup to dial their connection. I did not see this as locking the router to this particular ISP. It just made setting up the wireless connection easier. You will need to do a couple of things.

1) Make a note of the ISP connection details. ISP username, password, telephone number.

2) Access the setup program in the router. Open the web browser and direct it to [http://192.168.1.64](http://192.168.1.64). This is the IP address of the router. (it is for my router. It should be for your router). This will access the router's set up program. You will need a username and password. These usually are admin and the serial number of the router (look for sticky labels on the router). Now, change the connection details to those for your ISP.

3) Create a new connection in Network Manager. You will need the wireless key, pass phrase also called WPA-PSK if you are using WPA encryption. You may also find this on a sticky label on the router.

Did the BT router come with a setup guide? That will be useful. Some ISPs have setup guides on their websites as a PDF file for download. Can you find out the maker of the router. Makers often have setup guides available for download.  This is the case for the Thomson Gateway router that I am using.

It is not impossible to do what you want to do. The information is out there. Do the preparation before you try it. I am basing my advice on what I have available for my system. You should be able to get something similar.

Regards.
