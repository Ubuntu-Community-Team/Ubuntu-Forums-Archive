### Post by cariboo on 2009-08-21
You need to be root in order to move,copy and delete files in any directory but your own. To run nautilus as root, press Alt-F2 and type:

```
gksu nautilus
```

For info on emerald, have a look at this [wikipedia]("http://en.wikipedia.org/wiki/Emerald_(window_decorator)") article. Emerald is installed by default.
