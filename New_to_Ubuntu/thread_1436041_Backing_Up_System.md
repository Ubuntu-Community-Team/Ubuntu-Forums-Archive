Thank in advance.What I did before going to beta1 was to create an image of my karmic 9.10 install so I could just go back if the system was too unstable. In xp I used macrium reflect (free edition) and can restore karmic 9.10 from the same program in xp any time.

I was then free to do an upgrade install: by pressing alt f2...  in the run box type    *update* manager -d  and hit the run button.  This will start the update and most all of your data files and configuration will be preserved and a part of your lucid install.

If you can't get a working or stable system you've lost nothing, provided you've created an image backup.
