### Post by LowSky on 2008-08-08
first Vista cannot natively see Linux files without help. You cant exactly sync the bookmarks but you can export your links between version with little hassle. Just look under hte bookmark tab for manage bookmarks. and save the file you create to a flash drive or somewhere ubuntu can see.

As for iTunes there is no Linux version, but there are apps that can read and play mp3 just fine including keeping liibraries, like amorok, banshee, rhythmbox, and exaile. Note they cant play itune DRMed music. If you really want to use itunes supposedly it can work with WINE, but I never tried it.

### Post by ready4thebreak on 2008-08-08
I used iTunes on Wine and had the biggest problem with it. Firstly, it depends on which version you install (I think 7.2.1 was the version you could use). Secondly, if you try to sync your iPod(I have a Classic 80GB) sometimes it hangs or just doesn't move.  The files don't show up in the browser once you close the program and reopen it(which I'm guessing is because of how Wine works and it makes the directory reset to Windows-like directory). You couldn't right-click and change the info to the song.  Luckily, you can import the iTunes library folder, but all the songs come up as not found because when installing Ubuntu Live CD it creates /host in front of the mp3 files original directory(I don't know if you're using Live CD or dual-boot) .

Now do not be discouraged though, because unfortunately with there being so many different ways to install Ubuntu and the different program options(like Compiz and all playing a role in the GUI experience) and everything, my results may not be consistent with others, so definitely give it a go and see how it works for you.
