### Post by grahammechanical on 2014-04-05
You have the medibuntu PPA enabled and Medibuntu is now longer supported by the maintainer and has not been supported for some months. Disable the medibuntu PPA and the problem will be solved.

[https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)

Regards.
