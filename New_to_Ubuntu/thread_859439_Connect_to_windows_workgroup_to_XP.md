### Post by bobnutfield on 2008-07-15
Make sure the folders you want to share on the network are "shared" in Windows.  Do this by right clicking on the folder and go to Properties>Sharing.  The same must be done for printers on the Windows machine, they must be set up as shared.  Also, you must have the correct printer driver set up on your Ubuntu machine.
