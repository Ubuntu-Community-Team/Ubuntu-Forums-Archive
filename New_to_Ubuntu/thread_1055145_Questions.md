### Post by smothpocket on 2009-01-30
Your windows won't access any of the files inside of the virtual disk that wubi uses. If you download a virus inside of ubuntu your windows install will be safe as long as you don't open the same file in linux.

Keep in mind that viruses made for windows generally won't affect a linux based OS and vice versa. So if you do download a windows virus while using linux it probably won't automatically mess with your windows folders. Anything can happen though.

It's still always a good practice to run a virus check against any download from an unverified source. The best thing you can do is use safe practices while browsing the web.
