### Post by Kellemora on 2009-03-01
Do you have a setting that says 2UP in the driver?
That places 2 pages on each side of the paper, for a total of 4 pages per sheet of paper printed on both sides.

When I know I'm going to be doing booklets and the like, I start out with with OOoWriter set up for 2 pages per sheet Landscape 2 Columns, that way you get a better rendition on the printed page due to margin settings and printer capabilities.

Converting full size pages to 1/2 size pages is something the OLDE Windows WRITE 3.11 program excelled at that no full blown word processor can do.
We have WRITE3.11 running in WINE and use it daily.  NOT the WRITE that came out with Doze95, it's totally different.

I've never seen another program (other than WRITE3.0 and WRITE3.11 that can do a global font increase or decrease while maintaining the original sizes reducing each equally in a global manner.

If you have some OLDE Doze 3.0 or 3.11 install disks laying around, dig it out, you'll be glad you did, if you do a lot of this type of work.
Also, grab the old original PAINT program while you are at it.  It comes in handy for working on Logo's and Icon's if you're into those too.
