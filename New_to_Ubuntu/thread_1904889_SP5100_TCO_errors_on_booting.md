Hi Guys,

I just found the solution for you if you get kernel messages indicating errors with the sp5100 module. 

- Solution:

Reinstall you ATI (or Nvidia not test) videocard drivers via the rescue-mode root shell. Reboot and all should be ok.

I hope this helps someone.
