### Post by DJWYMAN on 2013-06-08
Linux mint is based on Ubuntu either are good choices, I just depends on which you would prefer.   Mint uses an interface called cinnamon which has a traditional tool bar much like windows.  Ubuntu uses and interface called Unity which has a dock on the left side of your screen which has short cuts and shows running apps, it resembles the OSX dock in functionality but instead of being at the bottom it is on the left. ### Post by Rob Sayer on 2013-06-09
I've had mint running on my netbook.  Not the cinnamon desktop but mate & xfce.  It is based on ubuntu (most versions of it anyway) but it's an older version.  I don't use mint anymore because the tech support on their forums stinks compared to this one.  But mint with the mate desktop is pretty good.  I don't actually liker the ubuntu unity desktop and the mate cinnamon one uses similar technology.  I found it lacked performance on my i3 based laptop.  I use kubuntu.

I don't think the OP is really aware of the different desktops available.  Rather than download ubuntu v. mint I'd suggest downloading ubuntu (the unity desktop), kubuntu, and xubuntu.  Then boot each one from the live cd (actually I prefer to install that on a usb stick rather than burn to cd .... much faster) and try the different desktops for him/herself.

There's also lubuntu which uses the lxde desktop.  It's mostly recommended for old hardware.  I haven't tried it but from what I've seen from questions on this forum it looks harder to configure.  Sometimes light is too light.  I'm typing this on a kde desktop on a 1Gb netbook.  It runs just fine.

Having all these desktop interface options may seem confusing if you're used to windows.  Linux and unix have a much better basic structure.  It's more modular and the permission levels are *way* better, which is  largely why it's the most secure pc OS and windows the least secure.

It's easy to use a windowed interface like any linux GUI or windows and think the GUI *is* the computer OS.  In linux it's just another application program.
