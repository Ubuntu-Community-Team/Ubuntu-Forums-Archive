### Post by LinuxFox on 2011-03-12
Lucid, Jaunty, or Intrepid are names of different Ubuntu versions.  For example, Lucid is the name for 10.04.  To find out which version you're using, click System and choose either "About Ubuntu" or the System Monitor under the Administration menu.

If you choose the System Monitor option, be sure to click the "System" tab.  Hope this info is helpful. :)

### Post by qyot27 on 2011-03-12
System menu->Administration submenu->System Monitor

If I'm not mistaken, it will report the version number.  Lucid, Jaunty, Karmic...those are codenames.  Sort of like how OS X 10.6 is referred to as Snow Leopard.  The names ascend alphabetically, more or less.  Intrepid (8.10), Jaunty (9.04), Karmic (9.10), then Lucid (10.04), Maverick (10.10), and so on - in either direction, as releases have followed this scheme since 6.06 Dapper Drake.  Also see: [List of Ubuntu releases](http://en.wikipedia.org/wiki/List_of_Ubuntu_releases)
