### Post by dgray_from_dc on 2008-11-13
I looked around and figured out that there is a license installed onto the original machine to which the movie is copied.  Without this license, WMP cannot play the file.

  I looked around more and found a Windows-based app which (as long as you have a valid license installed) can strip away the DRM from the WMV movie.

  The only problem I have now is that AVIdemux can't convert it into another format without skewing the audio.  However, it does play just fine on any PC, XP or Linux, that's good enough.
