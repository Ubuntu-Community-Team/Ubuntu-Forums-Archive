### Post by Shazaam on 2009-03-20
One of the ways to do it (that I am familiar with) is to virtualize one or the other operating systems. Choose which one you want to dedicate as a "Host" and which one you want as a "Guest". You would boot the host then boot the guest "on top of" the host. Sun/Virtual box is one example.
Make sure you investigate which option you want go with as virtualizing some operating systems have drawbacks such as reduced 3d performance; slower disk access (i/o speed), etc.
