### Post by WRDN on 2010-09-03
It will be slower running an OS in VirtualBox as you are running one OS on top of the other.
Windows in VirtualBox will be able to run most of the native Windows programs with no problems, save for some DirectX applications (Direct3D in VirtualBox is experimental).

### Post by kaldor on 2010-09-03
Virtualization is more for some must have applications when you're in need of using an OS that can't be used as your main.

It is not meant to be used for Linux users who need games etc. 

If you require another for some mission critical things but can't migrate, then that's what you use it for.

> **JamesParnell said:**
> Ive just downloaded VB for linux, so i can run windows inside ubuntu, but i need to know what I can expect to happen, will nothing run anymore, or will it be roughly the same as it should be if windows was running normally?

What do you plan on running in the virtual machine? What version of Windows? What are your computer hardware specifications? 

You will always see a slight performance decrease when running an operating system within another operating system. It also depends on your system hardware of how big of decrease you will see and what you will be using the virtual machine for.

I have used VirtualBox and VMWare Player. In my experience they both run well, however some USB componets run better in VMWare Player than VirtualBox.
