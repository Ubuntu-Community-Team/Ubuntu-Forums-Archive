### Post by cway00 on 2012-10-03
> **Ulvdatter said:**
> Does anyone know???

1: In address bar type about:config
2: find the value: browser.urlbar.clickSelectsAll
3: Right-click select toggle
4: done , url bar gets fully highlighted on first click
