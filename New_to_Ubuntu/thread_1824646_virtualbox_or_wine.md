### Post by 3rdalbum on 2011-08-14
Virtualbox with Windows installed inside.

Wine probably won't run the latest versions of Photoshop, Dreamweaver and Flash (which I assume you're talking about).

Virtualbox just creates a virtual computer, and when you've installed Windows inside it it's got the same software compatibility as a physical Windows computer.
