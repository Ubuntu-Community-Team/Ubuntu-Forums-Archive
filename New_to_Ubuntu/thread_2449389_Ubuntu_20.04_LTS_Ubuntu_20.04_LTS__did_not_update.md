### Post by mastablasta on 2020-08-26
or you can use snap: [https://snapcraft.io/firefox](https://snapcraft.io/firefox)  then get the candidate which is 80. or beta which is 81 or extended support which is 68.

or use the PPA for untested version.

Ubuntu (cannonical +community) tests software before it updates it and messes something up in the system. windows doesn't do that. 

new software is not always necessarily better. stable means "it doesn't change, just gets patches" (firefox and few others are exceptions in Ubuntu).

to use latest software you would need something like Arch or Manjaro or Debian testing. but things then break more often on updates and are sometimes more difficult to fix. as it was shown by Linux Mint Debian when things went bad after testing update was pushed to desktop users without any checks. i don't have time to deal with OS once it is setup, so i leave hat to enthusiast and people with more time on their hands. i remember back when i was in highschool my "workday" ended often at noon,so i had a lot more time than now when it end in afternoon and then i have a lot of chores...
