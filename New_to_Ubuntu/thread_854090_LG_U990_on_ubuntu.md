### Post by skymera on 2009-04-28
If it has a memory card you can change the USB settings on the phone to be "Mass Storage"

Settings > Phone Settings > Connection

Plug the phone in with the USB cable and the memory card can be used like any other.

If you want to access the phones hidden files and modify the firmware, i have a program that i use (Windows only) but it may work in WINE.

@ I know the memory card works, i own the KU990 and run 9.04 :)
