### Post by Victormd on 2008-07-18
You'll need to check to see what proc. your motherboard takes. Just because it has a celeron 1.1GHz doesn't mean that it will even take the celeron 2.4GHz unless they both have the same number of pins. that goes for the P3.

PC133 RAM is different from DDR2 and won't fit. I just sold ~2GB PC133 on e-bay and you can probably get them pretty cheap.

As for the graphics card, you're probably going to have to get a PCI (check to see if it supports AGP but I doubt it). Do you have a reference number for the machine so we can look up the motherboard specs?

### Post by Victormd on 2008-07-18
Alright, so that tell's you all you need to know and look for:
> 
**Processor**
Intel Socket-370
370pin PPGA /FCPGA Socket
66MHZ/100/133MHz Host clock)
I think it's going to be a bit hard to find the proc. unless it's on ebay or something. Check [this link]("http://www.cpu-world.com/Sockets/Socket%20370%20(PGA370).html").
**

> **Memory**
Two 168-pin 3.3 V DIMM sockets
Supports up to 512 MB memory of 100/133 MHz Non-ECC
Unbuffered Synchronous DRAM (SDRAM)

These are easy to find on e-bay

> **Expansion Slots**
3 PCI

For the graphics card, you'll have to settle for a PCI graphics card (i.e., nvidia 5200 is a good and cheap card for this)

For everything else, you'll need to check the specs and see if they match
