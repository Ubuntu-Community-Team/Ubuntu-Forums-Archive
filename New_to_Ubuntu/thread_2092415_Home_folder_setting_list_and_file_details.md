### Post by MG&amp;TL on 2012-12-07
You don't say what version you're using, which we need to know as the file manager has changed substantially over the supported versions of ubuntu. :)

As for hidden files, just press Ctrl-H. Ubuntu doesn't hide system files by default (of which there are many: check out "File System" entry in the file manager), but it does hide folders beginning with a '.' - the reason for this will be apparent when you see the number of them. :)

On this install of 10.04, the file manager has a dropdown at the upper right to change the current view. In the menu edit> preferences you should find a default setting that you can change to whatever it is you want for a global setting. I have mine all set to list with details. I don't like icons at all. I can never find anything on a page full of icons.

>  I imagine it also hides system files by default.
Yes. Look in the View menu for the entry "view hidden files" or use ctrl+h to see them.
