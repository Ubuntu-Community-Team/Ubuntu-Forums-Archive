### Post by howefield on 2012-10-28
Generally speaking package versions numbers are fixed at the time of each Ubuntu release and only security updates are applied, eg in precise the version was 0.9.3 and in Ubuntu 12.10 the version number is 1.0.5

You can setup a PPA from the darktable website which will give you the latest release.

[http://www.darktable.org/install/#ubuntu](http://www.darktable.org/install/#ubuntu)
