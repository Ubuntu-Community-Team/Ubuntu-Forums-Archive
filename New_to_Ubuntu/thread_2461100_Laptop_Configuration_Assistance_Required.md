### Post by grahammechanical on 2021-04-24
This link will show how to install Ubuntu on a USB memory stick and then run a Ubuntu live session to test Ubuntu with your hardware. It will also show what you have to do to prepare the computer to install Ubuntu on it.

[https://www.wikihow.com/Dual-Boot-Windows-10-and-Ubuntu-16.04](https://www.wikihow.com/Dual-Boot-Windows-10-and-Ubuntu-16.04)

The link is for Windows 10 users. The version of Ubuntu is 16.04 but it applies also to Ubuntu 20.04 and any flavours built on Ubuntu 20.04. The web page also shows what you will see when you try to install Ubuntu. A section will tell you about creating a swap partition. You can ignore that as Ubuntu 20.04 does not use a swap partition but a swap file.
