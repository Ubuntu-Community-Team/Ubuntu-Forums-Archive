This has been answered many ways over the years. The best answer that I've every read is, Linux is not windows, you need to go slow and have fun. Most programs can be downloaded as a .deb file which are easy to install just by clicking on the .deb file. You also have the archives which  can be accessed by Ubuntu Software Center, Synaptic, the command line using apt-get and several others package installers.

Of course you can always download the source in .tar.gz format and compile your programs but it will take some reading and research. Most all source file programs come with ReadMe and Install instructions. Not all are the same so to answer you install questions will be awful difficult without knowing what program you are trying to compile or install on your system.

I've been using Linux since the mid 90's and have found it very stable for what I use a computer for. There is a learning curve and noticing that you just joined the forum 3 days ago, give it time and slow down. When you ask for help provide as much information about your problem, to include hardware, software and any error messages, so whoever jumps in to give you a hand doesn't have to guess whats causing your problem and can help you quicker.
