### Post by Cheesemill on 2012-04-08
These games are written for Windows, not for Linux.

You may be able to get them to work using Wine, an application which lets you run some Windows programs on Linux but it's not guaranteed.

[http://ubuntuforums.org/showthread.php?t=885111](http://ubuntuforums.org/showthread.php?t=885111)

As always the best advice is to use Windows for running Windows programs and Linux for running Linux programs.

### Post by forrestcupp on 2012-04-08
Also, I can't believe that nobody has told you to look in the Software Center. If you open up the Software Center, you can search through a lot of the games that are already in the repositories, and if you find one you like, it will automatically download and install it for you.

You'll save yourself a lot of headaches if you stop searching around the internet for games, which are likely for Windows, and start looking at the resources we have for games that are made for Linux. But then if you have a Windows game that you absolutely cannot live without, you can try it out with Wine and see if it works.
