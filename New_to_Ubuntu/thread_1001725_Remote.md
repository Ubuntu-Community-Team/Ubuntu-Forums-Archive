As the article says, it's a kernel issue.

Any distro that uses kernel 2.6.24 or below should work.

Ubuntu 8.10 is using kernel 2.6.27.

So unless you use an older Ubuntu distro (I think Feisty should have 2.6.24, but I'm not sure) I don't think there is much you can do.
