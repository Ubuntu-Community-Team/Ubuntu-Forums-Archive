### Post by bobnutfield on 2008-09-03
You should be able to use any recent live distro cd (any Ubuntu, Knoppix, etc.) to mount your windows partition.  You will probably have to use an external USB drive to copy the files to.  Should not be a difficult job since WIN98 is FAT file system which Linux handles just fine.  Since the system specs are low, I would try a live distro like Puppy linux which will do the job just as well as a heavy duty distro like Ubuntu

### Post by jasontu on 2008-09-03
If it can run Windows 98 it should be able to run Xubuntu just fine.  

Xubuntu is very similar to Ubuntu but with a less resource intensive interface.  

You can download it at
xubuntu.org

There are other even less resource intensive Linux flavors like PuppyLinux and DamnSmallLinux.  I have only used DSL myself.  I would recommend trying Xubuntu first, since you may feel a little more comfortable with the interface if you are used to Ubuntu.  

It's slightly different from Ubuntu though so things may not be precisely where you assume.  Keep us updated!
