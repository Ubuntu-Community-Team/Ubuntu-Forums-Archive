### Post by grahammechanical on 2014-11-06
When you installed Ubuntu did you have a network connection (wired - ethernet) to the router? We users would not want Ubuntu to put indicators in the top panel for stuff that we do not use. If we do not have an internet connection at the time we install it is possible that Ubuntu may not put a network indicator in the top panel.

You could try going to System Settings. That should be in the launcher. If not you will find it in the power/cog menu. Then open Network. That should list your wired and wireless connections with a slider/switch to activate the adapter.

Of course, the network adapters must not be disabled in the BIOS or switched off using some keyboard combination.
