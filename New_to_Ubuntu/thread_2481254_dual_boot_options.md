### Post by oldfred on 2022-11-23
There are boot managers & boot loaders.
Grub is both but does not use icons.

Your screen shot looks like it is from an UEFI system.
But different brands of UEFI are all different. 

If booting with UEFI you can add the rEFInd boot manager which uses icons.
Alternative efi boot manager for UEFI limited systems:
[http://www.rodsbooks.com/refind/](http://www.rodsbooks.com/refind/)
