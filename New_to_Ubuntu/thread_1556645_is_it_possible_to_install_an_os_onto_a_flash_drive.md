### Post by D-Dan on 2010-08-19
Alternatively, if you have access to Windows try [http://www.pendrivelinux.com/](http://www.pendrivelinux.com/)

If you have larger than 4Gb pendrive after installation follow the instructions at [http://www.pendrivelinux.com/create-a-larger-than-4gb-casper-partition/](http://www.pendrivelinux.com/create-a-larger-than-4gb-casper-partition/) to access all the remaining unused space.

I've got a fully functioning Mint installation on a pendrive, with Virtualbox installed and a virtual XP installation on a separate pendrive for those times when Windows is a must.

Steve

Edit: If you plan on taking the pendrive with you and have any personal info on it be sure to password protect both the default Mint user (root) and any user account you set up. There is no password set by default.
