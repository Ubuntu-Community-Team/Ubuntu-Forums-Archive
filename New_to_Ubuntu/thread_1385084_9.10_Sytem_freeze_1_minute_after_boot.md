### Post by spirou38 on 2010-01-19
I had a problem with an ASROCK G41M-GS mainboard and Ubuntu 9.10 : my system was "freezed" about 1 minute after boot :( .
 
That was due to the SATA disk that was turned off 1 minute after boot !!!
 
To make it work fine I had to modify the BIOS parameters and set "Load Performance Setup Default (IDE/SATA)" instead of the standard parameters !!!
Now my system works fine :P .
 
It's possible that people with the same ICH7 82801xx chipset have the same problem.
 
Note that I had not the problem with Widows XP, so I think it's because of the SATA driver.
