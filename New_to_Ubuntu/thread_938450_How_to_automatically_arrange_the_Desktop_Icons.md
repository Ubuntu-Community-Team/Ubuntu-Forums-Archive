### Post by ibuclaw on 2008-10-05
Right-Click on the Desktop and click "**Clean up by Name**". That will re-arrange icons in alphabetical order, and that is it in Gnome.
KDE (Kubuntu) will give you a few more options, if you wish to be specific.

Okay figured out how to edit the size of the thumbnail images 

Click "applications" goto "system Tools" then "Configuration Editor" 

In Config Editor goto "apps" then down to "Natulis" Then "Icon_View" there is an option there to set the size of the Thumbnails. Just right click it and select edit and change the number mine is 96 to a lower or higher number value depending on the size you want. This may take some expirmenting. To get the right value.

The downside is it effects all the thumbnails for images both on the desktop and in the Nautlis Folders.
