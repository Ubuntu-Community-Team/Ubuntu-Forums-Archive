### Post by Kareeser on 2008-12-21
You don't need to defrag the drive to install Ubuntu.

The traditional install will format your hard drive, which makes defragging pointless, since there's nothing left to defrag.

The Wubi install, you could format your hard drive, sure... if only to make Windows run faster, which will in turn affect the speed of the Wubi-installed Ubuntu.

If the "D" drive is going to be occupied by Ubuntu by it self the "C" drive does not matter.
