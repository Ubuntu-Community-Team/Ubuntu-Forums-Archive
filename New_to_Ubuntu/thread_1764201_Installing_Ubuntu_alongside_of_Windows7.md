Wubi installs are really meant as a way for a new ubuntu user to try out the system, without the dangers of partitioning. It is not meant for a long term install. 

A proper dual boot would be the better solution long term. I dont think you can increase the size either. Check out the wubi sites faq's.
[http://wubi.sourceforge.net/faq.php](http://wubi.sourceforge.net/faq.php)

and
