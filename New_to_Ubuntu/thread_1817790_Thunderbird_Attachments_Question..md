### Post by cariboo on 2011-08-04
I've been using Thunderbird since it was called Minotaur. I've never had a problem with attachments, when I click oh on an attachment that I haven't opened before a window pops up asking what program I want to use to to open it. If I click save, it ask me where I want to save the file.

The emails are stored in ~/.thunderbird/xxxxxxxx.default, where xxxxxxxx=a random string of letters and numbers, along with the attachments, so they are still available even if you didn't save the attachment. I use the email archive option in Thunderbird, in my case the archive goes back to 1999, I also backup my home directory daily using rsync to my file server, which also gets backed up daily to an external usb hard drive.
