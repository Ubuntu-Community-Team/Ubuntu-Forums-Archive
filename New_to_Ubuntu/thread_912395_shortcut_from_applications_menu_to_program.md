### Post by jba6511 on 2008-09-06
I want to create a shortcut from the applications menu to launch a few programs that are normally ran from bash. For example, I would like to create a shortcut that would launch tcpdump as root from the applications menu. Is this possible?

---

### Post by zephyrcat on 2008-09-06
Yes, it is very possible.

Go to System > Preferences > Main Menu.

Click on the category (at the right) you would like the shortcut to be under.

Click "New Item" at the right.

Change the type to "Application in Terminal".

Type in any name you want (it does not matter.)

Enter the command you want to run. For example, "sudo vim" or "sudo tcpdump".

You may enter a comment if you wish, or you may leave it blank.

Click OK.

Hope that works!
