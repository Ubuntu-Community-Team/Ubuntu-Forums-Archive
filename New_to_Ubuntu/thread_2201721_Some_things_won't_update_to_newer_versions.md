### Post by deadflowr on 2014-01-25
Download the latest clamtk from [here]("http://clamtk.sourceforge.net/"), look for the Ubuntu/Debian version.
When you download, open the file manager "Files" program and go to the downloads folder, then double click on the clamtk-something.deb file
It'll launch the software center and list it as upgrade.
Hit the upgrade button.
If I remember right all you'll be doing is upgrading the interface, the virus defintions should be updated regardless.

Wine is hit and miss, don't know what apps or which version of wine you have installed.

To get the latest flash on linux you need to install the google-chrome browser.
You'll have to download it from google.
When you finish downloading google-chrome, do the same thing of going to the download folder and double clicking to open the software center.

You have a firewall already in place.
If you want to do extra manipulations I suggest installing the program gufw from the software center.

### Post by whitesmith on 2014-01-25
ClamTK is a virus scanner for Linux. I'm not being snarky (really, sincerely) but Linux needs a virus scanner like Romney needs spare change. The design of all *nix systems makes creating a destructive virus unreasonably hard -- unless a user does something really stupid, like installing a corrupted Windows program in Wine as root. ### Post by claracc on 2014-01-26
Once you install an ubuntu release, you will receive security updates for the apps installed and perhaps also patches to fix some bugs, but the release will mantain the release of their apps, if you want the newer releases you will have to install the last ubuntu 13.10 or install them from ppas.

About wine, there is only some windows software which works with wine but no all of it, please read: [http://en.wikipedia.org/wiki/Wine_%28software%29](http://en.wikipedia.org/wiki/Wine_%28software%29)

About adobe flashplayer, the last one for linux is 11.2.202.335, and adobe won't develop a more advanced one for linux.
