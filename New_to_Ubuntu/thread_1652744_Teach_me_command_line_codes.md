### Post by lisati on 2010-12-25
Just a friendly aside here. For some of us, "entering codes" on the command line means something different to what might better be called "entering commands". Back in the days when personal computers weren't so common and people tended to use large mainframe computers instead, if you wanted to do something like change your screen resolution, you would type in a command that would cause a special code (think "control signal" if you like) to be sent to the screen or terminal that it would recognize and respond to.
