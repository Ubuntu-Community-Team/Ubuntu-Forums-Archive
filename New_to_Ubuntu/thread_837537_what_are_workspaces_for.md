### Post by geirha on 2008-06-23
A few shortcuts and the likes:

When you use several workspaces, you might notice that you can only switch between windows in the current workspace when you hit Alt+Tab. To cycle through all windows on all workspaces, you can use Ctrl+Alt+Tab.

Rolling the mouse-wheel while pointing at the desktop background, will switch workspace. At least if you are using visual effects (compiz fusion)

Right-clicking the workspace-chooser (to the far right of the bottom panel by default) and choosing preferences, allows you to alter the number of workspaces available.
