### Post by laurielegit on 2009-01-14
Have you tried different cds, or different distros? I recommend you try a kubuntu cd and a fedora cd, so as to find whether the problem is with the machine or with the cd. Another option might be a minimal install, which requires a fair amount of command line, but you can just copy the commands. To do a minimal install see the instructions on [*this*]("http://www.psychocats.net/ubuntu/minimal") page.
Also, be aware that using the live cd, things can take a *lot* longer to load. Make sure that you aren't just quitting the live cd as soon as it appears to freeze.

Best of luck,

Laurie
