### Post by Frogs Hair on 2012-12-29
Hello and Welcome

Try Keyboard > Shortcuts Custom Shortcuts . I don't know why you would want to add the key stroke unless it is just too hard to adjust to not using it.

If right click on the desktop is not working by its self install the Gnome Tweak Tool and make sure nautilus handle desktop is enabled.
