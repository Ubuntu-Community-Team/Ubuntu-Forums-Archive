### Post by OldTimeTech on 2008-09-01
System->synaptic package manager, type into search either libssl-dev or openssl-dev, highlight, go to top of page click apply.....that will find and install both of those libraries.

[www.adobe.com....look](www.adobe.com....look) for shockwave, gives a linux edition or Firefox add-on also give shockwave or Applications->add/remove, enable all available applications, type into search shockwave and when it comes up on menu click on it and apply.

For number 4, go to firefox add-ons and on left side of page it gives you add-on that say bookmarks.  Load into Vista first and synchronize, then go to firefox and add same bookmark application (make sure works in both windows and linux) when you run the application in firefox tell it you already have a account and synchronize again....should have all have all your bookmarks...etc.
