[LIST]
[*]Find the orange circle ("orb") at the very top left-hand corner of the screen and press it.
[*]You'll get a number of menus on the left-hand side of the screen.

### Post by The Cog on 2009-12-06
I have only had a 5-minute look at netbook remix, but from that quick look I think there are only 2 real changes:

1: The start menu has been changed so that rather than a pop-up menu that grows out of the Ubuntu icon (like the windows start button), the same menu items are spread out over the entire screen - there isn't really a Desktop any more.

2: Applications don't really live in windows that you can drag around or re-size any more. They are all effectively either maximised to the whole screen, or minimised so some other application is filling the screen. The bar at the top serves as an application switcher and a "system tray" area.

These changes are really only cosmetic tweaks to the screen layout. They make the PC feel more like a PDA, but they don't affect the applications in any way - they won't make them faster or slower, and they don't add or remove features.

### Post by Paddy Landau on 2009-12-07
> **The Cog said:**
> The only time I use two windows not maximised is when I want to drag a file from one nautius window to another.
I don't know the correct terminology, so I've explained best as I can below. Please ask if I've made myself unclear.


[LIST=1]
[*]Leave both windows maximised.
[*]Open your first nautilus window, and locate the icon (button) on the bottom task bar for your second nautilus window.
[*]Drag the file from the first window onto the icon for your second nautilus window; but don't let go of the mouse button. Keep your finger on the mouse button and pause.
[*]After a second or two, the second window will open, so you can complete the drag.
[/LIST]

If you use tabs instead of windows in nautilus, you can achieve the same result by dragging the mouse onto the tab that you want opened.
