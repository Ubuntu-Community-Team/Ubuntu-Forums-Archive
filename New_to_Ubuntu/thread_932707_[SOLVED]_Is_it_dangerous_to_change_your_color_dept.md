### Post by mike1234 on 2008-09-28
> **DOS4dinner said:**
> Just a quick little question. If I change the defaultdepth of my xorg.conf to 16 (It's currently 24), will that cause any problems? I wanted to test if doing that would be enough to allow an old 256 color program to run right.

 No. You'll just have 16 bit color (16.7 million) as opposed to 24 bit color depth. In other words, a slightly weaker color display. Although you won't probably notice any difference.
