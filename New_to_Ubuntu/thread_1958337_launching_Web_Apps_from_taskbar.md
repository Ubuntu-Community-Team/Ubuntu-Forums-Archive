### Post by MG&amp;TL on 2012-04-14
Most web browsers accept arguments, so I suggest you install the *alacarte* (main menu) application, and use it to make a launcher with the following command:

```
firefox http://google.com
```

replace firefox with the name of your browser and the address with the address of your web app.
