### Post by llamabr on 2009-12-03
For the most part, they're the same os, just with different packages installed by default.  The server comes without most gui programs installed, the desktop comes without apache installed.  of course, one can install either of those, from either installation.

Depending on the server load, you may not want to use your desktop as a web server.  Though if you only have one mail user, and a few web pages with a couple hits a day, it probably doesn't matter.
