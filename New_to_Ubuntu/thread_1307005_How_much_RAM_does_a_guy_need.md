### Post by Jakey_TheSnake on 2009-10-30
That depends. I use my netbook with 1GB of RAM and a 1.6GHz processor and it suits my needs comfortably. 

2GB would always be nice, and kind of 'future-proofs' your laptop a bit, but is by no means necessary. The minimum system reqs for 9.10 is 512MB of RAM, and you have double that ;) 

I would get 2GB if I had a choice, though 1GB is enough, and anything more is only necessary if you're a) gaming or b) planning to run Vista LoL.

### Post by Bruce H on 2009-10-31
I have 4 gigs of RAM, which I find to be plenty. However, I can still page to disk if I really stress things out. Like editing a really big image file or something. So, my advice is to get as much RAM as your machine can hold, or as much as you can afford, but keep your page file -- you might still need it occasionally.
