### Post by grahammechanical on 2015-03-01
Yes. You will find it in the Ubuntu Software Centre. And wine is very good but it cannot work miracles. Do not expect every Windows application to work under wine. And for those that work they work with different degrees of success. You will find a database on the Wine web site that lists and grades the programs that users say work with wine. You will also find a instructions on how to install beta versions of wine through a Ubuntu Personal Package Archive (PPA).

And this forum has a section especially for wine. It is a sub-section of the Ubuntu Specialised Support section.
