### Post by audiomick on 2010-02-08
Hallo and welcome.
I hope you enjoy exploring. As has been suggested, click around on all the menus and see what they do.

You should know about a thing called "sudo"
[http://en.wikipedia.org/wiki/Sudo](http://en.wikipedia.org/wiki/Sudo)
You will eventually want to use this yourself; until then you can be happy that the way it is implemented in Ubuntu gives you a timely warning that what you are wanting to do could be "dangerous".

If you click on anything in a menu that would allow you to mess up a setting that might break your system, you will be asked for your password. If you get this, think carefully about what you are doing, and if in doubt, cancel out of it.

Any settings that you can change without a password are user settings. It is possible to mess up your desktop with too many poorly chosen changes to user settings, but this is not fatal for the system as such.

The other thing is, don't be afraid of the terminal. It might be a while before you feel inclined to try and use it, or something might come up very soon. Either way, don't be daunted by it; it is not as hard as it seems. There is help available in the terminal with the command "man". If is  not immediately intuitive, perhaps, but it is very helpful in the long run.
Try typing this in a terminal (applications> accessories> terminal)
```
man man
```And then maybe this
```
man apt-get
```to have a first look at it.
