### Post by keplerspeed on 2008-10-27
Go to Applications/Wine/Uninstall Wine software and select the program you want to uninstall from the list.

---

### Post by MockY on 2008-10-27
You can also navigate to the Program Files in the wine folder and execute the uninstall file for the software you want to remove.
```
'/.wine/drive_c/Program Files'
```
