Ah, it's a Firefox issue, not an OpenOffice issue.  In Firefox go to Edit > Preferences > Applications.  docx should be in the list of file types.  Make sure it is set to "Use soffice" (you may have to go to "Use other..." and type in /usr/bin/soffice).  Then Firefox should be able to handle these files.
