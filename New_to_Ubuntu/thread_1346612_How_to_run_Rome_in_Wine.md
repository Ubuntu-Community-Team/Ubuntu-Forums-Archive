### Post by Arthur_D on 2009-12-05
You can try using an application called PlayOnLinux: [http://www.playonlinux.com/en/](http://www.playonlinux.com/en/)
It's using Wine versions and special configs for certain Windows applications, mainly games. I have had good luck installing other games with it that wouldn't install with "vanilla" Wine.

Good luck. :)

### Post by Ylon on 2009-12-05
And [here](http://appdb.winehq.org/objectManager.php?sClass=version&iId=4496), you get more detailed info.


Anyway, for gaming on Linux I'd suggest to check in order:
1. Native games
2. Wine Platinum
3. Game to try with Wine


the difference between point 2 and 3 is that while normally you just buy the game *then* try it on wine, you should first [check a game in wine's platinum database](http://appdb.winehq.org/objectManager.php?bIsQueue=false&bIsRejected=false&sClass=application&sTitle=Browse+Applications&iItemsPerPage=25&iPage=1&sOrderBy=appName&bAscending=true), _then_ buy and try :popcorn:
