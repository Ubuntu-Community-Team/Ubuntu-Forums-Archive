### Post by mcduck on 2011-03-06
> **xfalcon45 said:**
> Hi all, my question is, how do I know which updates I need and which I do not?

Thnxx, using ubuntu 10.10

All of them.

Ubuntu only gives you updates to provide security patches and to fix more serious bugs, and the updates are only offered for programs that you have installed on your system.

So, unless you have a very specific reason, you really should install all the updates that the Update Manager offers you.
