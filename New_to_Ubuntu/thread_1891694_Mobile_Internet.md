### Post by Paqman on 2011-12-06
In Ubuntu if you want to execute any command as root you simply prefix it with the word sudo.

New software can be found and installed from the Ubuntu Software Centre. Drivers for hardware can be found in the Additional Drivers tool.

Is there a specific piece of hardware you're trying to set up?
