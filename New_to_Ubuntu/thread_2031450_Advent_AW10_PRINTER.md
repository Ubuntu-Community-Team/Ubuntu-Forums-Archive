### Post by paulnewall on 2012-07-23
install the package c2esp for unbuntu 12.04, using for example synaptic package manager.
what version of c2esp do you get?
Provided you have version 2.2 (c2esp22) or later, you should be able to print by telling the printer manager that the print is a kodak hero 9.1
 
read the documentation for c2esp to get more details on installing
