### Post by Darkness Des on 2011-01-29
Well, there is gEdit (or Text Editor) which comes with Ubuntu and is very extensible with plugins and themes and all of those such things. Geany is a very powerful IDE but for some it's a bit too much bulk. Sublime Text in [WINE]("http://winehq.org") is also a good choice, or you could even try PFE in WINE. I've heard praises of NetBeans but personally, I have never used it. Same goes for Eclipse.
