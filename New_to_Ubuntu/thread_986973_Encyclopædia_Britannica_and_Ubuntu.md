### Post by Mark Phelps on 2011-05-31
If you have questions about stuff working in Wine, you need to go to the WineHQ site and search their app compatibility tool. The page for Britannica 2009 is shown below:

[http://appdb.winehq.org/objectManager.php?sClass=application&iId=10446]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=10446")

The Silver rating (which is only up one notch from Bronze -- the lowest working rating possible) indicates that SOME of the functionality works.  Check the details in the test results for more info.
