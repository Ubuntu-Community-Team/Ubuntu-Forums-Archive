### Post by LinuxFox on 2009-04-04
I would recommend choosing the "High Contrast Large Print Inversive" theme.  You could change it by right-clicking the desktop and choosing Change Desktop Background.  I noticed when I tried this, the icons were bigger in the Applications menu.

Not exactly desktop icons, but I think it might help with menu icons if you plan on keeping programs in the Applications menu as well.
