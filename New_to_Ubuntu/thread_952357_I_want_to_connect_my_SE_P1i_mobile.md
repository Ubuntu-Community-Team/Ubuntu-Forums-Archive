### Post by Kevbert on 2008-10-19
It depends on what you want to do with the phone and PC. If you just want to use the phone to connect to the web, have a look at [this]("http://www.arkko.com/linux/p1i.html").  If you want to transfer files just connect it to the PC via bluetooth or wired connection. If you want to use the comms suite your best bet is probably via Wine (in the repos, use Synaptic Package Manager). If you want to edit the phonebook you can use Wammu/Gammu (again via Synaptic).
Hope this is of use.
