### Post by madmax75 on 2011-04-19
**Ubuntu Tweak** - as mentioned already, a great configuration tool, also useful for adding some extra software package archives (like Medibuntu, LIbreoffice etc.)

**LibreOffice** - your fairly standard office package

**Shutter** - screenshot tool

**Gimp** - a versatile image and photo editor program

**Transmission** - a torrent client, useful for downloading Linux distribution .iso files, comes preinstalled

**Unetbootin** - a good tool for making bootable Linux distribution USB sticks

**hardinfo** - system profiler and benchmarking tool

**Celestia** - a nice 3D planetarium program, also Kstars, if you are into astronomy

**Audacious** - a nice little audio player

**Totem** - your standard media player (VLC is the best for watching DVD's)

**Gpodder** - a very nice podcast manager program

**Xfburn** - cd/dvd burner

**Soundconverter** - easy to use conversion tool for sound file types

**Some useful Firefox add-ons:**

Adblock Plus (adblocker)

SearchPreview (small screenshots of web pages in your Google search results)

HTTPS-Everywhere (available from the GNU project website, forces your browser to use more secure https connections if available)

**Games**

Billard-GL (3D billiards)
Freeciv (an excellent Civ2 clone)
Quadrapassel (your standard Tetris clone)
Smokin' Guns (a good western-theme FPS)
Sauerbraten (FPS)
Scorched 3D (3D tank battle)
Gunroar, Torus Trooper (stylish games from Kenta Cho)

Most of these can be found in your Ubuntu software repositories.

### Post by Fraoch on 2011-04-19
If you're doing image editing, obviously, GIMP is great but if you're used to the old Windows Corel programs, it's more like Photo-Paint - better for manipulating existing photos than creating drawings.  Try Inkscape for a true drawing program (like Corel Draw), it's in the repos.  There's also sK1, in rather slow development, which is available from their website.  Xara Xtreme is nice too - it's abandoned, but can be found if you search.

If you're into technical drawing, there's QCad which is pretty full-featured and uses the AutoCAD .dxf file format natively.  It's available from the universe repo.

And if you're one of the very very few model railroaders who use Ubuntu (there must be five of us!), XTrkCAD is available from the universe repo.  It won't do rotated text though, but the free GPL Windows version runs perfectly in Wine - which, by the way, is also a must-have for those very few Windows programs for which you can't find a replacement.  Wine is in the universe repo and it's more or less self-configuring at this stage of development.

With the advent of the Ubuntu Software Centre, you more or less just type what you want and it will come up with something.  I found a few useful programs through it like ConvertAll and gResistor for converting measurement units and figuring out resistor colour codes.
