### Post by mathgeek2000 on 2010-08-24
Couple of options for you :
    -- Install Virtual Box, from Sun / Oracle and
    -- Install legal Windows XP or greater, possibly several 'Virtual Machines'
     -- Each Windows XP Machine could have it's own Internet Explorer version.
  ** -- Advantage it's real Windows / IE. 

    -- Install Wine, and look for the various versions of IE 
        -- Up to version 7, last I checked, BUT IE 7 wasn't a perfect look & feel to
           IE 7 on Windows. 
        -- do a search for 'IE4sLinux'
          ex: [http://www.tatanka.com.br/ies4linux/page/Installation:Ubuntu](http://www.tatanka.com.br/ies4linux/page/Installation:Ubuntu)

            Supports Ver: 5, 5.5, and 6 officially, 7 in Beta last I checked.
