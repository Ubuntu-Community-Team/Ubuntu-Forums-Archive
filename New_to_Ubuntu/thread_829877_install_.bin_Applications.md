### Post by Joeb454 on 2008-06-15
If it's on your Desktop you could open a terminal and run ```
cd ~/Desktop
./RealPlayer11GOLD.bin
```

Though you may need to run it as sudo, in which case change the 2nd line to read ```
sudo ./RealPlayer11GOLD.bin
```
