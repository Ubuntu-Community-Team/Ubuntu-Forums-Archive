### Post by vasa1 on 2011-11-08
Instead of altering font size, you could try pressing the **control** key and the **=/+** key (the one to the left of backspace) to make things bigger and **control** and the **- (minus)** key to make things smaller.

Page designers often use fixed measurements for various things and if we just increase the font size, things could get messed up.

Using the route I described above, magnifies everything proportionately and so there shouldn't be a problem (I hope).
