### Post by Mark Phelps on 2013-02-16
What limited the allocation to 4GB?

You would need to be running GParted from something other than the memory stick in order to do this.  IF you check the GParted site, they do have ISO downloads, and you may be able to image one to another memory stick and boot from that.
