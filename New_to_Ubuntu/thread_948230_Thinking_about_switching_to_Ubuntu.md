### Post by alpage2 on 2008-10-15
Timing is everything, as the saying goes. If you can wait another couple of weeks, a brand new, improved version of ubuntu is due out on 30 October, although for the first few days the download servers are usually under pressure and rather slow.

On the other hand, if you want some installation practice, the current version is fine.

To add to all the commentary above, the other useful thing about linux is that it will run happily on older hardware that no longer has sufficient resources to run a supported (i.e. gets security updates) version of Windows. My old Win98 laptop now runs xubuntu, with a 400MHz CPU and 192MB RAM. Web browsing and Open Office applications all run fine. The main limitation is that it can't cope with streaming video. And there are super lightweight versions of linux that will run on much less than that!

Alan
