### Post by Lisiano on 2011-10-12
Wine1.2
If you want the one Ubuntu offers, just look for Wine 1.2 in Ubuntu Software Center, Synaptic or type this in a terminal.
```
sudo apt-get install wine1.2
```
To get the latest Wine (Both stable and beta)
Go here [http://www.winehq.org/download/ubuntu](http://www.winehq.org/download/ubuntu)
After that is done, just press here [Wine 1.2]("apt://wine1.2") or do the same thing as for the Wine that Ubuntu offers.
Note: If this works, I'm going to start doing apt links like the one above.
