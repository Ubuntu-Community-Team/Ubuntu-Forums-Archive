### Post by Mark Phelps on 2011-06-08
Wine is not a miracle cure that will allow the installation of ANY Windows app; it is very limited in terms of what apps it supports and how well it does that.

You should first check the WineHQ website and search their Application Compatibility database for the app and version you want to run.

I did that and it's not listed -- meaning that, most likely, it is NOT going to work.

Also, you could try posting your Wine questions in the Wine subforum.  Someone there might already have tried it.
