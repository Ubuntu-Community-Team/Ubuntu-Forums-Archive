### Post by Paqman on 2008-11-08
> **lucasl said:**
> 
All I ever hear is complains from the 64 end - so and so driver doesn't work, flash is terrible(r). Is there a way around these things?


Most (if not all) of the criticisms of 64-bit I see on these forums are from people who don't actually use it and are recycling old information. I've used 64-bit for several years now. There did use to be issues with Flash, etc, but they've been taken care of, and it's no longer an issue.

As for advantages, I find it slightly more snappy. It's especially noticeable in apps like Gimp. The RAM advantage isn't such an issue on Linux, as even 2GB is plenty, but it's nice to know that we're all sorted for when we need to push out past 4GB.

Basically, I use 64-bit because I don't see any reason to throttle a 64-bit chip down.
