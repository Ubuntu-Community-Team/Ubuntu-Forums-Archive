### Post by fooman on 2010-01-17
233 mhz may be the processor speed,  not the memory size.  memory would be measured in MBs on that machine.

the minimum memory required to run a basic 9.10 installation is 256 MBs:

[http://www.ubuntu.com/getubuntu/releasenotes/910](http://www.ubuntu.com/getubuntu/releasenotes/910)

you should probably look into a less demanding distro,  there are plenty of them out there, including many based on ubuntu.
