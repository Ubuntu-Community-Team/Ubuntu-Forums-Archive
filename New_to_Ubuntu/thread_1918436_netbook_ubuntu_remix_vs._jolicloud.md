### Post by Paqman on 2012-02-01
There is no specific netbook remix any more. You can just use regular Ubuntu running the Unity 2D desktop on your netbook. Some people like to use a lightweight version such as Xubuntu or Lubuntu on netbooks, but I've found it's not really necessary to do so and especially Lubuntu does lack a lot of features that you'll find in standard Ubuntu.

For better battery life install [Jupiter Applet]("http://www.jupiterapplet.org/"). It'll automatically tweak your system for battery life and switch between high and low power modes when you plug into the wall.

### Post by I2k4 on 2012-02-02
Currently dual booting Lubuntu 10.10 with W7 Starter on an Acer netbook (standard 1GB RAM, 1.6 CPU) and both run well, Lubuntu is faster.  I strongly suggest experimenting via "Live USB" with Persistence on a 4GB drive.  It will be slower than full install, but give you a good sense of resource demands of different distributions.  I found the 11.x versions all more demanding and slower than 10.10 (except for the horrible "Netbook Remix") with no added functionality - however, if you like the new Unity interface you'll want Ubuntu 11.10.  My main beef with Lubuntu is its choice of preinstalled software, as compared with Mint - such that I spent a lot of time uninstalling and replacing packages.  That's another good reason to test out Live USB with several different OSs.

(As for W7, it runs slower than XP did on my older netbook, but it does  work fine - if it has "slowed down" from previous acceptable performance there are any number of possible causes, some in Windows and others in software, e.g. antivirus, you might have installed.  A couple of suggestions:  get CCleaner and run both the file and registry cleanups, then use the Tools tab to disable unneeded start up processes.  Get rid of all the idiot "updaters" that every bloody program wants to keep running all the time. Then defrag the hard drive.  Then investigate the "Black Viper" recommended system settings - google it and confirm I'm not misleading you into a security hoax.  Finally, be aware there are some good tricks to overcome the crippling of "Starter" - e.g. renaming your LAN network "Workgroup" and setting it as "Enterprise", and installing the fine "Mr. dUsha patch" to permit the Aero personalization of desktop and setting your own background images.)
