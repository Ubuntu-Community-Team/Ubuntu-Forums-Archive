### Post by mikewhatever on 2008-08-01
Open System>Admin>Synaptic, search for dansguardian, right click and select 'Mark for installation'. 
If you want a shorter way, open Applications>Accessories>Terminal, type <sudo apt-get install dansguardian> and press enter. It looks like dansguardian is somewhat harder to setup, but very easy to install.
