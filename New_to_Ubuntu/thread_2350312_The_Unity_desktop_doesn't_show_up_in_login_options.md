### Post by ajgreeny on 2017-01-23
Yes, it might have been better, but whenever you add another DE to an existing system you add a lot of duplication of application types, eg text-editor, terminal-emulator, file-manager, etc etc, which can be a nuisance.

One way to be absolutely certain you get the unity desktop is to install the **ubuntu-desktop** package which will then definitely bring in a lot of extra applications/packages and may just add to your frustration.
