### Post by warbread on 2008-05-08
The [WineAppDB]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=8202") gives a good rating.  I suggest installing wine from Synaptic, running
```
:-$ sudo winecfg
```

and autodetecting the drives.  From there, just download and install the .exe from Ares' sourceforge page and you're good to go.   

If you're not already aware, though, there are lots of good P2P programs that natively run on Linux.  Why not give them a shot, too?  You have nothing to lose but time and hard drive space.
