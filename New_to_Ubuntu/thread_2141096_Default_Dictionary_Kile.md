### Post by cheatos on 2013-05-04
so, for all who may be interested, I have found the configuration today by accident when trying to install a new package ;)
It is in the "settings -> configure kile -> editor -> open/save -> Modes & Filetypes -> Variables" tab, there you can set the dictionary.
