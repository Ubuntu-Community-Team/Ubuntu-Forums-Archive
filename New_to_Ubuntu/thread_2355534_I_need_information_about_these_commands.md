### Post by sisco311 on 2017-03-14
They look like warnings generated by apt/apt-get. It would be more helpful if you could post the warnings in English. Just set the LANG environment variable to posix before you run the command:
```
export LANG=posix
sudo apt update
```

Which version of Ubuntu are you using:
```
lsb_release -a
```

And what is in your sources.list:
```
tail -n +1 /etc/apt/sources.list{,.d/*.list}
```
