### Post by Black_Tanto on 2010-09-21
If you just want all the same apps you installed to be replaced like you had them, you could try using AptonCD instead of remastersys. Try to use your remastered disk as a live session, mount a /tmp partition for more space, create a backup of your apps and try to download it somewhere besides ram, then take your AptonCD disk and install the whole thing to your clean install.
