### Post by Autodave on 2018-11-09
You probably have a sticking key.  My own Vaio has the same problem occasionally.  To make sure, plug an external keyboard in and see if it works OK. If it does, then here is what to do:

Turn laptop off.  If you have canned air, blow it over the keyboard in all different directions while aiming at about a 45 degree angle.  With laptop still off, work every key 15-20 tries.  Do one key at a time. blast with the air again and do the keys again.  Now power up and see how that works.

If still the same problem, the upgrade could have failed.  At that point, I would back up everything that you need to an external source and do a clean install of 18.04.
