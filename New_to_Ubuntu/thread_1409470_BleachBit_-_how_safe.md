### Post by wojox on 2010-02-17
I use Bleachbit and find it's pretty good at cleaning up junk.
You can choose what you want it to clean.
By default it cleans /home, /root directories. You can have it set to clean /log and /temp directories as well. 
Same with APT it can be chosen to clean, autoclean, and autoremove.
The only thing I don't have checked in System is Free Disk Space.
