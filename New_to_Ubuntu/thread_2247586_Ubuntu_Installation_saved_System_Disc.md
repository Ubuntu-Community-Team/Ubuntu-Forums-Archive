### Post by Norm_Stangl on 2014-10-09
I'm new to Ubuntu and wanted to report that it was successful in rescuing the system drive on my laptop. 

I had a hard drive fail on my Windows machine. I removed it and tried reading it using a USB Drive adapter on another computer, but I couldn't access any of the files on the disc. I had determined that it was the drive controller, so I sent the drive controller off to have it repaired. Upon return of the drive to me, the drive was still failing.  

I ran some diagnostics and the report came back that there were too many failed sectors. 

A friend then tried to rescue it by using a disc drive diagnostic tool but the same report of too many failed sectors showed up. I was informed that the drive had failed and there's not much I could do to save it except to spend about $300 to try to salvage the files with no guarantee of success. 

On a lark I thought I'd download Ubuntu 14.04 and try to install it onto my computer using the failed drive as my system drive. To my amazement, Ubuntu loaded onto the system without an issue. I elected to load Ubuntu along with the Windows OS. To may amazement, Ubuntu allowed me access to all of the files that were on the disc. This all using FREE software and providing a solution to what seemed to be a hopeless situation. 

I have now managed to back up all of my files and I'm happily exploring the Ubuntu world on my rescued system.
