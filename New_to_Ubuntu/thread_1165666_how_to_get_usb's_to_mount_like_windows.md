### Post by sgosnell on 2009-05-20
Yep, it makes no difference what OS you're using, if you remove the drive while data is being written to it, you'll get corruption.  You don't need to wait long, but you need to wait until the writing stops.  I seldom bother to unmount a USB drive, and have never had a corrupted file from removing one, because I make sure the light is off before pulling it out.
