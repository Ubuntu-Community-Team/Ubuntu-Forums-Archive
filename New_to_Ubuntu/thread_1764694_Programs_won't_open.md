### Post by Anuovis on 2011-05-22
Not sure if that's true for the newer releases but upgrades were generally considered a wild gamble a while back. A lot of the time they simply don't work and/or break down your system.

It might be possible to fix this but the easier solution would be to wipe out the hard disk and do a clean install once again. It is a guaranteed fix (provided your installation files are OK) and would probably take less time since you won't have to run around indicating numerous problems and then looking how to solve them.

I hope you backed-up your data. If not, make sure you do before doing anything else.

It's probably safer to avoid upgrades all together.
