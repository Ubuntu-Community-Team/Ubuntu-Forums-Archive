### Post by squakie on 2014-01-26
Since Visual Studio is a Windows product, it won't run in Linux.  I've used it in Windows, but never attemtped to install it in Wine inside of Linux.  I would think the best bet for this would be to run Windows in a virtual machine and just install Visual Studio there - if you have the hardware to support a virtual machine.  If not, I personally am not an OS "biggot" - I say use whatever works for you, and this may be a case where you are better of using Windows.
