### Post by Holger_Gehrke on 2018-11-25
If you're talking about [this Photoscape]("http://www.photoscape.org/ps/main/index.php"), then it's a Windows program you probably installed using wine. The '.lnk' file is a Windows shortcut and useless on Linux. You can delete it. The '.desktop' file is a basically the same thing as a '.lnk' file but for Linux. To make it useable you might have to make it executable, either though 'Properties' in the context-menu or from the command line with 'chmod'. If the installation worked right, it should also have created a starter in ~/.local/share/applications/ so you can run the program from the menu. If that's the case then the .desktop file on the desktop isn't strictly necessary either, and you can delete it unless you like to start programs from the desktop.
