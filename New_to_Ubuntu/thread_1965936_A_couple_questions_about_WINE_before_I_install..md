### Post by kimda on 2012-04-26
1. Some applications run better than others under wine. I would suggest lookig for your applications in the wine database. This database also has user suggestions on how to get your program working. [http://appdb.winehq.org/](http://appdb.winehq.org/)

2. No. I do not think you need to get a virusscanner. 

3. No. Ubuntu Linux cannot be infected by Windows virusses. But you could infect other windows users if you send infected windows documents to window users.

4. No because Wine is installed on another operating system and has nothing to do with existing Windows ssystems.

### Post by lisati on 2012-04-26
> **RadicaX said:**
> 
2. If I get WINE, will I need to get Windows like Anti-virus such as AVG?


Good question.

This is your choice, and opinions vary. Some will be quick to point out that [malware]("http://lisati.homelinux.com/wiki/index.php/Malware") designed for Windows is *unlikely* to hurt your Ubuntu installation. Others will point out that your choice of software to help you deal with the not-so-nice stuff is just one element of a process that begins with you being smart about what you put on your computer.

Some of the issues and opinions are discussed in this old thread: [http://ubuntuforums.org/showthread.php?t=788324](http://ubuntuforums.org/showthread.php?t=788324)

You might also want to take a look at [this sticky]("http://ubuntuforums.org/showthread.php?t=510812") in the "[security discussions]("http://ubuntuforums.org/forumdisplay.php?f=338")" section of the forum.
