### Post by forestpixie on 2008-04-26
Start it up and let it do it's thing - once it's opened, if you have been getting spam - check the junk inbox - if you have been deleting them they could still need to be expunged. 

Look in the left pane with junk selected - at the top does it say there are messages, but in the right hand pane it's empty?

If so, with junk selected do Ctrl+E to expunge the folder. See if that helps.
