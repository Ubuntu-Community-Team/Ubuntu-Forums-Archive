### Post by grahammechanical on 2014-06-25
MP3 is a restricted format covered by copyright an other licensing requirements and because of this the ability to play mp3 audio files is not enabled by default in Ubuntu but we can, if we are prepared to take the legal responsibility, install Ubuntu Restricted Extras which will install certain codecs that will allow media players to play certain restricted media formats.

> [COLOR=#333333][FONT=Ubuntu]Ubuntu&#8217;s commitment to only include completely free software by default means that proprietary media formats are not configured &#8220;out of the box&#8221;. [/FONT][/COLOR]

> [COLOR=#333333][FONT=Ubuntu]However, you can install the ability to play popular non-free media formats, including DVD, MP3, Quicktime, and Windows Media formats, by following the instructions below.[/FONT][/COLOR]

[https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)

Lots of utilities try to predict what the user wants them to do. Sometimes this is useful. Sometimes, not. It is the way things work.
