### Post by IRSAC on 2008-06-11
Some installation attempts For Ubuntu 8.04 LTS Desktop Edition on different machines:

Dell Dimension 8300: Live CD boots and runs Ubuntu 8.04 without problems.

HP Pavilion Slimline s7400n: Live CD boots and runs Ubuntu 8.04 without problems. Attempted hard disk installation under Windows XP Home Edition apparently completed to the "Restart Point". The machine refused to Restart and could not be shut down at all. The solution was to hold the front panel power button down until the machine shut down. I uninstalled Ubuntu 8.04 from the windows control panel. However, installation of Ubuntu 8.04 under Windows alters the boot.ini file so that a splash screen appears asking the user to select the operating system to boot each time the machine is started. To eliminate this splash screen one must edit the boot.ini file. In other words, this mode of installation does not leave the original windows installation "untouched"

Dell Inspiron 1100 (Laptop)
The LiveCD Ubuntu 8.04 boot fails to recognize the touchpad  and also fails to recognize a USB optical mouse, which makes it impossible to use Ubuntu 8.04 on this machine.
