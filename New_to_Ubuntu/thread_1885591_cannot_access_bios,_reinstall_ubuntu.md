### Post by roger_1960 on 2011-11-23
Hi

A few questions:

Have you backed up all your data to an external drive/disk/cloud?  If not, do so before upgrading.

Did you create the USB stick using Ubuntu (by right clicking the .iso file) or using Unetbootin?

Why install 11.04?  The current version is 11.10 and the current LTS is 10.04.

It sounds like you need to recreate the install USB so I would download Unetbootin and the iso you want (or at least check the md5sum of the one you have) and reburn the USB stick.

Is your laptop reasonably new? some old ones can't boot from USB devices.

### Post by Mark Phelps on 2011-11-23
If you recently updated Ubuntu online, then it was most likely to the 11.10 version -- for which we have beeng getting a lot of posts about failed updates.  So much so, that in the Installations forum, there is a sticky about upgrade errors.

You might want to read through that to see if your error is one of those reported, and if there is a fix for it.

Your problem sounds a lot like the situation where the Unity launcher and panels suddenly disappear, leaving you with a desktop background and mouse pointer.  If that is the case, that is another indication that you're likely running 11.10 -- and the fix for that problem is detailed in the link below:
