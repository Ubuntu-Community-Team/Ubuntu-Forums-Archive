### Post by NikTh on 2012-11-13
Hi , 
if you want to locate and see the boot messages , you can by giving this command in the terminal 
```
dmesg
``` 

The difficult thing is when you want to locate the shutdown messages :P 

As for the microcode amd ...etc message , yes is not a dangerous message and not creates problems , but if you want to resolve it , have a look here => [http://ubuntuforums.org/showthread.php?t=1945481](http://ubuntuforums.org/showthread.php?t=1945481)
