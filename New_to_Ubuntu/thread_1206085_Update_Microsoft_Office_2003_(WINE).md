### Post by Mark Phelps on 2009-07-07
If you really want to use Office 2K7, suggest you get the latest trial version of CrossOver.  They've done a lot of work to get it to use Office 2K7.  However, don't be surprised if IE7 doesn't work.  I installed the trial and was unable to get IE7 to even use favorites.  Email exchange with CodeWeavers indicated that they were are aware of a LOT of problems with IE7 and would be fixing them in the next update -- several MONTHS down the road!

Also, you need to realize that Wine working OK with one version of an MS product is no indication that it will work OK with a newer version, in fact, the opposite tends to be true.  Suggest you check out the CodeWeavers application compatibility site (link below) for the specific Office 2K7 apps you want to run:

[http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true]("http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true")
