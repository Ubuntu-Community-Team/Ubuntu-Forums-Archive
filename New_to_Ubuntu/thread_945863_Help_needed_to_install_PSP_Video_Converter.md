### Post by Victormd on 2008-10-12
I use Avidemux. Open a terminal and type the following to install:
```
sudo apt-get install avidemux
```
Check this [how to]("http://news.softpedia.com/news/Convert-Movies-with-subtitles-for-your-PSP-on-Ubuntu-67806.shtml").

### Post by Victormd on 2008-10-12
You can use Dvdrip to copy your dvds to your HD then convert the vob files to PSP format using avidemux. To install dvdrip, open a terminal and type:
```
sudo apt-get install dvdrip
```
Note: I am assuming that the DVDs you want to copy to your PSP are yours - do not use any of these programs to produce ilegal copies...
