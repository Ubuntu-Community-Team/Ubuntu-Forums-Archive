### Post by Locke_99GS on 2009-02-22
The standard 32bit version works best for most - I believe it is compiled for i386, and marked as such.

The 64bit version (AMD64, IA64) will work on 64bit processors only. I have no experience with it, but others suggest compatibility issues with applications compiled for 32bit CPU's.

You probably want the Desktop edition.

Intrepid works pretty well now, but Hardy is much more solid.
