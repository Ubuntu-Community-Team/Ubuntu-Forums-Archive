### Post by Jive Turkey on 2010-03-21
The word focus as I understand it refers to the application is selected input via mouse and keyboard from the user.  "Aero snap" is the latest windows feature that lets the user position and resize windows based on what screen edge the user has dragged the title bar.

Compiz has a similar function (more flexible).  I have ccsm installed and for me the related functions are under "window management">"Grid". It gives me 9 possible ways to position windows via key combinations.

Before I discovered this I thought maybe windows had a feature I liked that ubuntu didn't...wrong agian.
