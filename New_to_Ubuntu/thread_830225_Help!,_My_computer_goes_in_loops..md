### Post by jerome1232 on 2008-06-15
Sounds like either the ISO you downloaded was corrupt, the cd burnt incorrectly, the cd got scratched, or the cd-rom in this computer isn't reading the cd correctly.

Try to boot without changes on a different computer, if it does boot sounds like it's your cd-rom.

If it doesn't boot up on another computer, try re-downloading the ISO, burn it to CD at a slow speed. (1x being the most reliable) and try 'n boot again.

Are you trying to repair your windows OS? You may be able to recover important files from you old windows partition once you do get booted into ubuntu but I don't think you'll be able to repair the windows installation. Depending on what you choose during an install of Ubuntu, you can wipe everything off of your hard drive, so proceed with caution, check out some how to's on this forum.
