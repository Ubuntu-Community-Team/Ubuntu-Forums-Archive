### Post by mörgæs on 2014-01-25
Hi, welcome to the fora.

First, getting some additional memory will make a big difference. With 240 MB there's not much fun to do.

The problem with distorted screen could probably be solved using Vesa drivers, but you will have to accept a coarse resolution of, say 1024*768 pixels. Here's a [boot option]("http://ubuntuforums.org/showthread.php?t=361236&page=83&p=12292664#post12292664") you can add. 

If Lubuntu is still too heavy you could try Puppy Linux, selecting Vesa during installation.
