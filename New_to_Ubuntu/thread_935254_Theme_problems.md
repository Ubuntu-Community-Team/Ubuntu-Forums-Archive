### Post by fooman on 2008-10-01
in the top panel,  go to system > preferences > appearance

in the theme tab,  near the bottom,  click install

browse to the .tar.gz file,  highlight it and click open.

you should see a message saying that the theme installed correctly and it will then be listed with the other theme choices.
