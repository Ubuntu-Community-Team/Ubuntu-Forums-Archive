### Post by lisati on 2008-07-02
> **redfox1160 said:**
> So if I installed an os onto the HDD (Windows), then installed Ubuntu, It would automatically detect it, and I could dual-boot?
Yes. Ubuntu will give you a choice of how you want to partition your hard drive, and from there will work out the details of dual boot. Partitioning a HDD is kinda like letting you use one physical HDD as if it were more than one.

**IMPORTANT NOTE 1:** Make sure you make a backup of important data. A mistake can be a pain in the (wherever).
**IMPORTANT NOTE 2:** When the installation process asks you about how you want to partition your HDD, take your time to think about your response. A mistake can be a pain in the (wherever).

### Post by Sydius on 2008-07-02
> **redfox1160 said:**
> should I install Ubuntu or Windows First?

Installing Windows first is easier because Ubuntu will detect Windows already being there and handle the dual-boot setup automatically.  Windows, on the other hand, refuses to admit that Linux might exist on the same computer, and will not set up dual-booting automatically if installed after Linux.
