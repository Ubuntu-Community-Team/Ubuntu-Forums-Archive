### Post by y-lee on 2008-04-24
Alt-F2 will open a run dialog box. You can type gnome-terminal to open a terminal. I would try to reboot and see if that fixes your gnome panels problem.

```
sudo shutdown -h now
```

If rebooting doesn't help post back and we will help you get your panels back. Note Alt F2 and then typing firefox should open a web browser for ya:)
