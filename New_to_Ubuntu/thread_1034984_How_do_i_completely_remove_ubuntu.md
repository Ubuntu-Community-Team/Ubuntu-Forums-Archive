### Post by Facetious Falcon on 2009-01-09
Why would you want vista? It should work if the boot priority is truly CD. If that doesn't work then try disabling the HD entirely from BIOS temporarily. Grub is loaded onto the MBR of the HD you installed it on so if the boot priority is truly CD first then it should work.

If that doesn't work then just use a Ubuntu live CD to format it.
