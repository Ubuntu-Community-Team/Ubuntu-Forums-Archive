### Post by campb.andrew on 2010-12-09
Wow. Sorry to hear about that. You can try and import the drivers from another ubuntu setup some how. But i'm not sure how that would work. I'm new to ubuntu myself and had this problem just a few days ago. I solved it cause i had a wired connection. I'm experiencing low speeds when i use the wireless though for some reason. 

But yeah, like i said, try importing the drivers via a usb key or run the recovery mode at the boot screen when you restart the computer after selecting the ubuntu OS - im assuming you have another OS on the computer. (ubuntu -generic 2.3.35 [recovery mode]) <----Looks something like that.

A blue screen will come up, select restore broken packages and then after it does it thing, select resume boot. Enter ur username and password and then press the power button to shut down the computer. dont hold it down, just press it. 

You can restart after and everything should be ok....I know i'm long winded but i hope it helps.
