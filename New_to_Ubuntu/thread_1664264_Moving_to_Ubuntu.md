### Post by soapytheclown on 2011-01-10
Okay you can use the wubi installer however im sure most people would advise that you stick this on a usb thumb drive or CDr and try running a live session. This is probably in your best interest too as it will give you a full view of exactly how ubuntu will work on your machine. It will show you what does and what and doesnt work and give you a better feel if you like or dislike it. 

to answer your question though, if you decide to mount the image in something like daemon tools and use wubi its safe to use and remove using add remove programs, it uses the windows bootloader and should you remove it your machine will operate as if you never had it there at all.

### Post by akand074 on 2011-01-10
Do NOT use wubi. If I'm not mistaken, wubi installs Ubuntu within Windows and should only be used to see what Ubuntu is like. Thus, you would have to reinstall Ubuntu afterwards if you remove Windows. You should burn a CD or create a live USB and install Ubuntu on it's own partition. You can then choose from the boot menu which OS you want to boot into. You may then remove either OS without effecting the other.
