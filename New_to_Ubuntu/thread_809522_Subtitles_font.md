### Post by sayakb on 2008-05-27
For VLC: Settings->Preferences->Video->Subtitles/OSD->Text Renderer
Change Font from there..

### Post by tramir on 2008-05-27
Again, this is done by the player. The srt file is a simple text file - there's no info on size of subtitles etc. You might want to check the manual of your DVD player if it supports different sizes for subtitle fonts (I know I couldn't find this for mine, but maybe you're lucky).
