### Post by DuckHook on 2015-05-30
> **davidlubomirov said:**
> Nope I'm locking for a keyboard shortcut ?Your question is phrased rather vaguely, so it is difficult to understand what you are desiring to do.

I use multiple monitors as well&#8213;four of them, in fact&#8213;so find myself dragging windows among four monitors all the time. However, since X is a GUI environment, moving windows must be done manually. The OS has no way of guessing exactly where you want the window to be placed, nor which monitor you want the window in. In this sense, there is no keyboard shortcut that will automagically move a window to the next monitor.

It is possible to reposition windows without the mouse, if this is what you are asking. Keyboard shortcut is <Alt>+<F7> to go into repositioning mode, then arrow keys to position window, then <Enter> to leave repositioning mode and leave window at last position.

If you are actually running two monitors as two independent screens, then all bets are off. I've never done that and don't know how to manipulate windows between them.
