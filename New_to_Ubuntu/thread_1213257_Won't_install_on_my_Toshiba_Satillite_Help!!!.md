### Post by LewRockwell on 2009-07-14
let's review all the normal questions:

MD5 checksum was correct

CD-R was used to burn disk

CD-R was burnt at SLOWEST speed possible with your hardware

From first screen, have you selected and performed media CD integrity test?

Have you attempted at least six (6) cold boots with the disk?
(I know it sounds strange but I have had several disks work after 3 or 4 boots)

If all of these fail then you should burn another CD blank.
