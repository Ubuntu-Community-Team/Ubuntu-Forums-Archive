### Post by iaculallad on 2008-07-15
It's a must to enter both Username and Password when you're installing Ubuntu. Use that configured account to change your security setting if you opt to automatically login on your Ubuntu system.

To automatically login on your system: System->Administration->Login Window, "Security" tab, place a check mark on the setting w/c says "Enable Automatic Login" and select your configured username. Click on close button.

But this is not recommended.

### Post by Sef on 2008-07-15
You have to put them in; however, you can set Ubuntu to automatically boot.

System > Administration > Login Window > Security > click on 'Enable Automatic Login' > Set User to desired name > Close
