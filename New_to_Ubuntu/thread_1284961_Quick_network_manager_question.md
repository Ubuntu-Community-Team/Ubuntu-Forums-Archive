### Post by mikeyphi on 2009-10-07
As you said - right click on network, select Edit connections
then select your wireless connection (easiest if you are already connected)
and fill in all the required data.

If there is no connection named - you will have to 'add' a new connection name plus data. You won't cause a problem! if it all fails you will still be able to connect manually.
