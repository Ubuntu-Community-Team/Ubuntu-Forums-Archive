### Post by anim on 2008-12-08
Lillypad is an engine to convert text files into printed music. That being said, there really is no lilypad "program." You just use any old text editor to write the commands and pass them to lilypad to print out a nice pdf of whatever you told it to. 

A nice tutorial is [http://lilypond.org/doc/v2.10/Documentation/user/lilypond/Compiling-a-file#Compiling-a-file](http://lilypond.org/doc/v2.10/Documentation/user/lilypond/Compiling-a-file#Compiling-a-file)

Basically, you use any old text editor to write the music in, then use the command line to pass it that text file to lilypad. Lilypad then converts the text into a pretty looking PDF of your music. There is no graphical interface for the program. As far as you're concerned you will only be seeing programs that either 1) edit text or 2) display PDF's
