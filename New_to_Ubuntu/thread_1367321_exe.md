### Post by Bucky Ball on 2009-12-29
Correct. You can't use them in Ubuntu. But you can use Wine in Ubuntu to run Windows programs with varying degrees of success from brilliantly to not worth it. You can also run a virtual machine; Windows inside Ubuntu or vice versa.

What programs are you wanting to run? Have you looked for Linux alternatives? This is just a start:

[http://www.foogazi.com/2006/11/10/alternatives-to-windows-programs/](http://www.foogazi.com/2006/11/10/alternatives-to-windows-programs/)

Bottom line is if you want to run mostly Windows programs probably stick with Windows. :)

### Post by Temposs on 2009-12-29
In order to run exe files from Windows, you must install the WINE program, which you can get from the Ubuntu Software Center. Once that is installed, try running your Windows programs. Not all of them will work, but some may.

If you want to check how well a program works on Linux with WINE, go to [http://appdb.winehq.org](http://appdb.winehq.org) and search for your program in the top right search box.
