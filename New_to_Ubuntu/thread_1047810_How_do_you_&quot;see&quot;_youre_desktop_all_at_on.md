### Post by amauk on 2009-01-22
OSX's Expose (which is what I think you mean) is duplicated by Compiz's Scale plugin

if you haven't already, install the compizconfig-settings-manager
```
sudo apt-get install compizconfig-settings-manager
```
(or search synaptic for it)

Once installed, it's in preferences

Activate the scale plugin, and under the "bindings" tab, you can set your prefered key bindings

the key bindings have little icons beside then
Monitor = mouse moves to certain screen edge
Keyboard = key combo (plus modifier keys)
mouse = mouse button (plus modifier keys)
