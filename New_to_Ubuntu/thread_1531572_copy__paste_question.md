How do I select and copy output from a terminal to a text file.

---

### Post by Paddy Landau on 2010-07-15
[LIST=1]
[*]In the terminal, highlight (with your mouse) the text that you want to copy.
[*]Either right-click the text and select *copy*, or press Ctrl-C.
[*]Go to the text file where you want to paste.
[*]Either right-click and select *paste*, or press Ctrl-V.
[/LIST]

### Post by mcduck on 2010-07-15
..or simply highlight the text you want to copy and use middle click to paste it. :)

(Its called "primary selection" and it works for everything else as well, not just for terminals. So you don't even need to copy something to paste it...)

### Post by aeiah on 2010-07-15
you can pipe commands to text files as well, eg:

```

ls > filelist.txt

```

or append to an existing text file:
```

ls >> filelist.txt

```
