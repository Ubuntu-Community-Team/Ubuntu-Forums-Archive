### Post by pspsampsp on 2009-04-19
make sure you have compiz-config settings manager installed.

go into the compiz settingmanager (system -> preferences -> CompizCOnfig settings manager). now under accessability look for show mouse. make sure show muse is unticked.
hope that helped.
