### Post by sayakb on 2008-05-28
A GTK theme changes overall Window characteristics including decoration, buttons, scrollbars, menus etc.
An emerald theme changes the Window decoration (border)
You might use a GTK+Emerald theme together. I use a [Moomex theme]("http://www.gnome-look.org/content/download.php?content=57063&id=1&tan=18140697")(GTK)+[Xristal theme]("http://www.kde-look.org/content/show.php/Xristal?content=45690") (emerald)
Plus, I have kept the reflection plugin **on** for a better visual effect. 
Enable reflection by installing:
```
sudo apt-get install compizconfig-settings-manager
```Then goto* System->preferences->advanced desktop effects settings* to enable reflection.
