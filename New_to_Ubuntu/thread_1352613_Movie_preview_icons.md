Hi,

Initially I did not have the codecs installed and as a result, my avi files had the default "avi" icons (instead of a snapshot from the file). However now I have installed the codecs, so how can I "refresh" the icons of the file to now contain the snapshots from the avi file?

---

### Post by Physical Hook on 2009-12-11
```
rm ~/.thumbnails/*

```

Once you're done with this, open Nautilus and it'll regenerate them.
