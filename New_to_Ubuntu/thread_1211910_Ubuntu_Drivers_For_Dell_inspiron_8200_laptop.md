### Post by 3rdalbum on 2009-07-13
Windows doesn't ship with any drivers. Ubuntu ships with loads of them.

The best way to see if Ubuntu has all the drivers it needs, is to run the live CD. Anything that works on the live CD will work in the installed system. If the wireless networking doesn't work, then you can usually find a driver for it.
