### Post by Jacdeb6009 on 2009-04-12
Hi there,

I am no GIMP master and it is a bit difficult to understand exactly what you mean, however, I'll give it a bash...


It works like this.  The main window (i.e. where you can add and edit your image) only edits the currently active layer.  The active layer is selected in the "layers" window. Simply click on the layer you wish to have active.  Also, make sure that the active layer is visible, otherwise you will see nothing.

If the layer background is not transparent (i.e. if it has a solid colour, say white) then you will always only see the top layer, whether it is active or not.

I suspect that what you are trying to do will require the layer background to be set to transparent and then you animate by toggling the visibility of layer on and off.

Hope this helps.

There are some good site with tutorials available, one of these is  [http://gimp-tutorials.net/](http://gimp-tutorials.net/)

You should also download The Gimp's manual, it is also quite useful.

Patience helps... :)  The Gimp (like Adobe Photoshop) is very powerful, but requires patience to learn and it is (IMHO) every bit as good as Photoshop, you only need to learn to work with it.

Good luck!!
