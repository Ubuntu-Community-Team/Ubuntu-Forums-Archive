### Post by swoody on 2009-05-09
Hi fisheater, and Welcome to Ubuntu and the Forums ):P

After you reboot, please let us know if the freezing issue has gone away or not. Have you run all the updates available since you installed Ubuntu? If not, go to "System">"Administration">"Software Sources" and make sure Main, Mutliverse, Restricted, and Universe are all selected, and make sure "Installable From CDROM/DVD" is unselected. Then close that window. When you close it, it will ask you to reload your packages. Allow it to reload them. Then after that's done, go to "System">"Administration">"Update Manager" and click on the "Check" button on the bottom. It'll ask you for your password to continue, and then check for new updates. If there are any updates there, install them all, and then try to reboot your system again, and see if Evolution still freezes.

Also, if you'd be interested I have a link in my signature to a great Free Ubuntu Guide that you may find useful as you get to learn Ubuntu. You can either print it out and have it on hand (many pages, so beware :)) or you can simply download it to your computer, and look it over whenever you have an issue. Either way, don't be afraid to ask questions in the forums :D

```
sudo rm /home/username/.evolution
```
