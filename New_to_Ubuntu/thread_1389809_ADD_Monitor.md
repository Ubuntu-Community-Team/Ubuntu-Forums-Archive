### Post by rogue_0111 on 2010-01-24
Try **lxrandr**. It's so easy to use. I plug in my laptop to an external monitor, bring up a Terminal and type "lxrandr".  I get a panel that looks like this pop up:

[IMG]http://image.gihyo.co.jp/assets/images/admin/serial/01/ubuntu-recipe/0048/0048-08.png[/IMG]

To install bring up a terminal and type:

apt-get install [B]lxrandr
