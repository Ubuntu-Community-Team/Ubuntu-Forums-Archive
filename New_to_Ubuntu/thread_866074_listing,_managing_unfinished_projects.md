### Post by Potatoj316 on 2008-07-21
If your dealing only with packages (not self compilied stuff) and you use aptitude to install them unused packages are removed automatically.  I think they had to be installed with aptitude for that to happen though.  This same problem has happened to me, I advise that you use aptitude for dealing with packages (aptitude is in the terminal, you get a GUI like interface if you just type aptitude) You might also want to think about virtual machines for expirementing in the future.  You can get virtual box from the repos and install expiremental software in a virtual machine.

to install virtualbox ```
sudo aptitude install virtualbox
```
