### Post by wormyblackburny on 2011-08-29
Unless the BIOS is password protected you can install regardless of how "locked down" the computer is. You need to be able to get into the BIOS to tell it to boot from CD rom/USB BEFORE the HDD. If you can do that, you can install whatever (even reinstall Windows). 

Side Note: A good way to get in deep crap with your IT Dept is to do stuff like that, and believe me at some point they will have to turn the machine in....

### Post by Matt__ on 2011-08-30
Get her to create a persistent USB live disk from windows using a live cd and [UNetbootin]("http://unetbootin.sourceforge.net/").

Hit F12 to access boot order as PC starts.

She gets her Ubuntu to use anywhere she likes, the school maintains the sanctity of its machine...everyone wins.
