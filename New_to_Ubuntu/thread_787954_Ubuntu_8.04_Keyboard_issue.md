### Post by jpsanab on 2008-05-09
Hi, I've just upgraded from 7.10 to 8.04 on a Toshiba laptop M65, everything seems to be working good except my keyboard, when I press any of the keys on the numeric keyboard Ubuntu restarts, like if I had pressed CTRL-ALT-BKSPACE. I checked the keyboard layout and tried several others with the same result, any suggestion would be greatly appreciated, thanks!

Update: fixed, had to disable the Allow to control the pointer using the keyboard on the keyboard preferences menu, Thanks anyway!.
