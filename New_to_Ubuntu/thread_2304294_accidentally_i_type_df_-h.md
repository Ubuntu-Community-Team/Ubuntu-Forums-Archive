### Post by coffeecat on 2015-11-25
You haven't broken anything. All you get is an error message telling you the syntax was wrong, like this:

```
coffeecat@DesktopWily:~$ df -h/
df: invalid option -- '/'
Try 'df --help' for more information.
```

By the way, you don't need html BBCode tags. That's for formatting html code properly if you want help with your html code. Plain code tags will do fine, as I have used.
