### Post by Megaptera on 2011-01-11
Hi,
You could look at Vinux ... from their FAQ "Vinux is a remastered version of the Ubuntu Linux Distribution optimised for visually impaired users. It provides a screen-reader, full-screen magnification and support for Braille displays out of the box! It can be run from a Live CD without making any changes to your hard drive. If you like it you can install it to a USB pendrive or to your hard drive either alongside Windows, or as a complete replacement for it. There is also a virtual version available which can run Vinux as a guest operating system using VMWare Player on Windows."

[http://vinux.org.uk/faq.html](http://vinux.org.uk/faq.html)
