### Post by mcduck on 2012-02-16
There is no such unit as "gb", so I assume you are meaning either GB, or Gb (gigabytes or gigabits).

30 GB = 30000 MB

...and 30 Gb = 30000 Mb = 3750 MB

edit:
Or perhaps the units you were talking about should be *Gibibytes*, GiB, in which case the multiplier is 1024, and:
30 GiB = 30720 MiB = 28610.2 MB

### Post by Paqman on 2012-02-16
Prefixes in the metric system are usually multiples of a thousand. 

Mega = M = 1,000,000
Giga = G = 1,000,000,000

So a giga-something is a thousand mega-somethings. Computers try to confuse the issue by using a standard that defines units of 1024 instead of 1000, but they're close enough that you can normally ignore the difference and just use the nice easy SI units.

So 30GB = 30,000MB.
