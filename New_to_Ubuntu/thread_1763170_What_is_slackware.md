### Post by Paqman on 2011-05-20
It's a really, really conservative Linux distro that favours extreme stability over being up-to-date. It tends to be the absolute last distro to adopt any new technologies, which can be a good thing if you want a system using well-known and well-documented technology.

Personally I think using a distro that doesn't even have a decent package manager is torturing yourself, but horse for courses.
