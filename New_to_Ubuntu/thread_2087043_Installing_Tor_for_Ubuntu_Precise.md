### Post by DanR01 on 2012-11-22
The Tor browser bundle is probably the easiest and safest way to use Tor.

Just download it, unzip it and start the shell script.

It's available here:

[https://www.torproject.org/projects/torbrowser.html.en](https://www.torproject.org/projects/torbrowser.html.en)
