### Post by RobsterUK on 2009-02-03
In older versions of Pidgin the message window would flash when you got a new message. The current version doesn't seem to do that, and I can't find anywhere in the options to switch on this behaviour.

Any ideas?

---

### Post by cafeytv97 on 2009-02-03
Go to tools->plugins. then find the "Message Notification" plugin and mark the checkbox. Then click the configure button and mark the "Im windows" and "set window manager urgent hint" checkboxes.
