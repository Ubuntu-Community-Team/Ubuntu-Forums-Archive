### Post by frup on 2008-04-26
Using synaptic package manager in system > administration or the add remove programs tool can give you access to over 22000 programs. 

for newer versions getdeb.net is probably the best location but synaptic really wins.

If you are trying to install packages you need to make sure they end in .deb nor .rpm or .tar.gz 

.rpms are for fedora,red hat, mandrake and suse distributions 

.debs are for debian based systems such as ubuntu.

.tar.gz files are archives, usually source code for compiling. (with the exception of mozilla projects for example)
