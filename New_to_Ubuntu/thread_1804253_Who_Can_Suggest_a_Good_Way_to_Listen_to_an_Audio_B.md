### Post by owiknowi on 2011-07-14
a lot of audio books are sold as mp3, ogg or such, nowadays.
if you own(!) audio books on cd you can rip them to a hard disk and convert them (or the conversion can be done whilst ripping).

after that you can batch rename the files per cd like cd01_track01, cd02_track01 etc.
put all the files in one 'folder' and there you are.

if there are more than 99 tracks per cd name them cd01_track001. otherwise things will get mixed up.
