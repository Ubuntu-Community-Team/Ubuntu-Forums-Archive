### Post by tarps87 on 2009-06-23
Floppy or USB?
There are to ways I can think of.

1) The way I've done it
Run the iso in virtual box and use the Puppy Linux install app of the live cd.
For this you will need to download virtual box, if you are tring to make a live USB you will need the PULE version form [here]("http://www.virtualbox.org/wiki/Linux_Downloads") (The open source version does not have USB support)
Once installed just create a new machine add the iso image and boot from it, Puppy Linux will guide you through the rest

2) Try using [Unetbootin]("http://unetbootin.sourceforge.net/") it is in the repos

3) Well there is a third way, if I have miss read you post and you have a second computer with a cd drive, burn the iso, pop it in and click on the install Puppy Linux when it boots
