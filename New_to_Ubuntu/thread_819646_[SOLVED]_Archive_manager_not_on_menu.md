### Post by Oldsoldier2003 on 2008-06-05
you can add it to the menu manually, the command is ```
file-roller
```. you can access the menu editor graphically through the menus or you can launch it from a terminal using the command
```
alacarte
```

### Post by st33med on 2008-06-05
Go to System>>Preferences>>Main Menu. By default, Archive Manager is not checked in the Accessories submenu, so go and check that.

---

### Post by raymondvillain on 2008-06-05
Thanks, folks, for all the help.  System>Preferences>Main Menu has to have archive manager checked.  St33med knows all about it!
