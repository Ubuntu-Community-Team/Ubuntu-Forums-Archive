### Post by Ptero-4 on 2009-02-05
Select the "manual" option, select whatever partition happens to be something different from fat, ntfs or swap. Then click on the "edit" button, select "/" in mount point, in filesystem, pick the same filesystem it had b4 (you may pick others, but the installer´s default is usually good enough) and click on the "format" checkbox, then click "OK" and in the main window "next". That´s it you just dropped mint over the former Ubuntu´s partition.
