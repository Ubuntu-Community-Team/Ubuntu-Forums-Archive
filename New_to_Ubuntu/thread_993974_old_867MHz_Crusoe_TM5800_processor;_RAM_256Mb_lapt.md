### Post by snowpine on 2008-11-26
There are some great distros out there designed specifically for older hardware: Puppy, DSL, and my personal favorite, SliTaz. You can try any of these from a Live CD. In fact, many people find these distros run best from Live CD--a little slow to boot up, but very fast once started, since everything is loaded into ram. Any of these three should be noticably faster than Xubuntu.

If you are just looking to speed up your existing Xubuntu install, my recommendation is to install a lightweight windows manager such as Openbox or IceWM. LXDE is a very easy to use desktop environment based on Openbox. Also experiment with alternate software, for example using a lightweight browser (Links is my favorite) instead of Firefox. And of course, more ram.

At the end of the day, an old computer is an old computer, and multimedia tasks such as watching movies are hardware intensive. Good luck!
