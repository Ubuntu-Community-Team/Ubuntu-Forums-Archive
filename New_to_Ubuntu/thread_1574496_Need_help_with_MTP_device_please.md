### Post by Mark Stosberg on 2010-10-31
Some devices, like the Motorola ZN5, allow you to choose your USB connection mode. If you can access it as a USB hard disk instead of via MTP, you may be able to simply copy the audio files over if they are compatible with the device. 

One of the benefits of using Rhythmbox with MTP is that it will automatically transcode a file from OGG to MP3 if your files are stored in OGG on yoru computer for the device only plays MP3s.
