For an actual install, you will need the 4GB, and you only have 2GB. The nice way to proceed might be to create a bootable usb stick, and run the eeepc from the usb stick for Ubuntu in live sessions, perhaps using a usb stick created with 'persistence' so that some data is retained.

I believe also that the eeepc has a convenient sd card slot available. I use a 4GB sd card in my eeepc 900 and can install into it, using it at start up from the eepc boot selection menu.
