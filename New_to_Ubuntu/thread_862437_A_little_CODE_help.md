### Post by dracayr on 2008-07-17
every shell (or else called terminal, console, CLI, bash, etc...) has a working directory.
some VERY basic commands:

cd - change directory
pwd - print working directory
apt-get - program for installing/removing packages
cat - output a file
vim - text editor
ls - list files in directory
exit - exit the shell
rm - remove file
rm -r - remove directory and all files it contains
rmdir - remove empty directory
ln - create new link to a file

dracayr
