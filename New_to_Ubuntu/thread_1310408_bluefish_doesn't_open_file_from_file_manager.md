### Post by garyed on 2009-11-01
When I click on a file in Nautilus that is set to open in Bluefish it opens up Bluefish editor but not the file I clicked on. I don't remember having this problem in older versions of Ubuntu so I'm wondering if anyone else has the same problem. I'm running Hardy.

---

### Post by garyed on 2009-11-03
After searching it's seems there is disagreement whether this is a bug in Bluefish or Nautilus or neither but it is a common problem. 
I found that in Nautilus if I use "Open with" > "Open with other application" there are two entries for Bluefish Editor. If I choose the right one then it works fine & afterwards just clicking on the file will work from then on.
