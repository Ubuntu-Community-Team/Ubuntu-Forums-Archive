### Post by Mark Phelps on 2011-11-17
The problem you're likely to run into is one of the PC monitoring the keyboard whil it is asleep.  If it's not, it doesn't matter what key you press, it wouldn't see it.  You're probably going to have to live with pressing the power button to wake it up.

However, you should check in your BIOS settings, usually under Power options, to see what features it has for enabling wake on keypress and for what keys.
