### Post by yetiman64 on 2010-11-11
Sounds as though archive manager has taken the association from nautilus for opening folder locations. Instructions in the quote box should help you.

> Use ALT + F2, type in "nautilus" (without the quotes) then click "Run", right click any folder in the opened window and select "Open with Other Application", select "File Browser" and ensure "Remember this Application for "folder" files" is ticked, then click "Open". This will reassociate nautilus to opening folders permanently. 

If at any time you use any other program for opening folders ensure "Remember this Application for "folder" files" is deticked, this should avoid problems like this in the future.

Edit: the quote is from a text file I keep here to fix this issue when it occurs here. Instructions sourced from this forum (multiple posts)
