### Post by Irihapeti on 2013-10-12
Not a silly question at all.

You can reuse the config files so long as the programs in question haven't changed their config file layout. Finding out whether they have or not might be a bit tricky.

I'd suggest copying the config files/folders back one at a time and seeing how the program behaves. If it does, then fine, go on to the next one. If it doesn't, remove the config files/folder and let it create a new one.

Of course, if you can get definite information from, say, playonlinux, about whether or not the config files have changed, that would be helpful.
