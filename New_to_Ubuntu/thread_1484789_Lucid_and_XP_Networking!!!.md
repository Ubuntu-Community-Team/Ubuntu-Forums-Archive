### Post by 3rdalbum on 2010-05-16
> **randolf_ambrose said:**
> can anyone help me... i've setup both systems for nerworking... 

Have you followed the directions in the Ubuntu help file?

System > Help and Support

Internet And Networks

Share files and folders with other computers

This should set up a shared folder on the Ubuntu computer, that the Windows computer can also access.

And make sure you turn off any personal firewalls running on either computer.

Places > Network lets you connect to existing Samba shares, but not create new ones. The Howto in the Ubuntu help program, that I mentioned earlier, is an easy way to create a Samba share.
