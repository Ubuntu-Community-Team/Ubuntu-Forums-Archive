### Post by Mark Phelps on 2011-08-10
> **ALVI2284 said:**
> How to install windows base application in  ubuntu

The other folks have already given you great advice regarding installing Window apps.  Strongly suggest you go to the WineHQ site BEFORE you attempt to install any apps.  If the app version rating is low, you can save yourself a lot of time and work by NOT trying to install it.

Also, if you're asking about installing Windows in Ubuntu, the anser is NO -- that's can't be done.  Wine, and its companions, can only be used to install Applications. They also can not be used to install Windows drivers.
