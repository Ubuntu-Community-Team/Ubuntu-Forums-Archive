### Post by Little Bacon on 2012-06-16
If you're working with audio in Linux, I've learned that the best way to go is to take some time to learn how to operate [JACK]("jackaudio.org").

JACK can be complicated, but once you learn to use it you'll be glad you learned. JACK links together multiple programs that each have perfected one function to give you a compilation of developer's hard work.

Almost any option outside of JACK that isn't intensely limited would have to run through Wine, which can make just getting Reaper to install and run correctly a hassle.

If your heart is set on Sonar or Reaper, I would either Use a dual-boot and run it on a Windows partition or stick to Windows, depending on what else you plan on doing on your PC.
