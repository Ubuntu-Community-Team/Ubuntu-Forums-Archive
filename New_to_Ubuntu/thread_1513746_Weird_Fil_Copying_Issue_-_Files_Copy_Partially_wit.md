### Post by Ocxic on 2010-06-20
When Firefox downloads a file it will create 2 files (ex: file.ext and file.ext.part)
file.ext.part will disappear after the download has completed, and you will be left with just file.ext

Seems that you didn't wait until the file had finished downloading.
