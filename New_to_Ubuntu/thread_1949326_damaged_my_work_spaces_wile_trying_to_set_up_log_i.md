### Post by kazztan0325 on 2012-03-30
Hi koll,

I don't know why and how did you use "startup applications" to set up a different log in screen, and don't know what does *"now my work spaces only has one of the 4 squares"* mean.

Anyway, I would like to recommend **'Ubuntu Tweak'** to you if you would like to change your log in screen.


To install it, follow these steps:

1) Open Terminal, and type (or copy and paste) following command, and then hit Enter for adding 'Ubuntu Tweak' PPA to Software Sources.

```
sudo add-apt-repository ppa:tualatrix/ppa
```

2) Type (or copy and paste) following command, and then hit Enter for updating software list.

```
sudo apt-get update
```

3) Type (or copy and paste) following command, and then hit Enter for installing 'Ubuntu Tweak'.

```
sudo apt-get install ubuntu-tweak
```

4) Close Terminal and Open Dash and then type **ubuntu tweak**.
