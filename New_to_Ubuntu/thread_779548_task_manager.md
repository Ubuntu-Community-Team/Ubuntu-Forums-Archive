### Post by fatality_uk on 2008-05-02
Usually, try to close an app from the window list in the panel will bring about a "Force Quit" dialog and you can use that to single out the app in question.
However, if you need a Linux version of the task manager, type > gnome-system-monitor into a terminal and that will give you what you want.
