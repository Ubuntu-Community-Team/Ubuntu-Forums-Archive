### Post by finer recliner on 2009-03-19
dolphin is using the KDE4 theme, not GNOMEs. You won't find the option in Dolphin's settings. I think the best way to do this, is to log into a KDE session, and customize the KDE color scheme/theme colors (whatever they call it). This will change the colors for ALL KDE4 applications
