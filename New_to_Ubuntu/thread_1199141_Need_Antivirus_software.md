No, Ubuntu doesn' have one. But you don't need one either, as there aren't any Linux viruses such program would protect you from. ;)

There are some antivirus apps available for Linux, but their purpose is to scan for Windows viruses, for example on a corporate mail server to help protecting Windows computers.
