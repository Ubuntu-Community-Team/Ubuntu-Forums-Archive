### Post by deadflowr on 2013-06-10
How vanilla do you want it?
If you want a clear clean kubuntu/kde experience, then install kubuntu.
If it doesn't matter and you want the Ubuntu-Desktop packages on your system, then install the Kubuntu-Desktop packages.

If your system is a 64-bit system, then install 64-bit.
