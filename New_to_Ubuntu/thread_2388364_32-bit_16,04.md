### Post by deadflowr on 2018-04-01
Note that the same recommended system requirements laid out on the Ubuntu download page still apply to 32-bit.
[ATTACH=CONFIG]279150[/ATTACH]
While you can run Ubuntu on some old hardware, it'll be sluggish.

It's more commonly recommended to run a lighter flavor such as Lubuntu Ubuntu Mate or Xubuntu on old hardware.
(Most notably Lubuntu, as it's built with old hardware on it's mind. More so than other flavors.)

You can look at all the Ubuntu flavors by going to the top of this page and clicking on the Ubuntu Community dropdown menu.
We list the links for all flavors for easy access.

Hope it helps
