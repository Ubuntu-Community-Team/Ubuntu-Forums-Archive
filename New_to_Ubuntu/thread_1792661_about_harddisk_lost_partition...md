### Post by oldfred on 2011-06-28
Welcome to the forums.

Partitions do not just merge. If you think you may have overwritten something STOP using hard drive, only use liveCD as the more you use it, to more you overwrite. Rex Bouwense link to photorec & others only can find data not overwritten.

From Ubuntu if it is working or from Ubuntu liveCD.
