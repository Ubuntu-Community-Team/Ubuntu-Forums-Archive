### Post by running_rabbit07 on 2009-08-20
If you install VBox on Ubuntu, you will have to install MS in the Vbox to get the stuff you need, which will waste more space. Sadly if you want both worlds, you will have to keep restarting to use the other guy's programs, so to speak.

And, your MS install disk may not work on VBox unless it is a clean install disk from MS.
