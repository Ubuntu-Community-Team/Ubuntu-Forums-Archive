### Post by diablo75 on 2010-12-11
It's possible the CD you're using is flawed, or the drive you're using is dying (it's happened to me).  Try to check the disc for errors by hitting any key when you see the blank purple screen with the keyboard symbol at the bottom when you first boot from the disc.  It looks like this:

[IMG]http://pricklytech.files.wordpress.com/2010/07/ubuntu-boot-01.png[/IMG]

Then this will come up:

[IMG]http://www.hackourlives.com/wp-content/uploads/2010/10/Ubuntu-10.10_0011.jpeg[/IMG]

Do the "Check Disc For Defects".  If it fails, that doesn't mean the disc is bad.  To rule out the possibility of the drive being bad, take the disc to another computer and check it for defects one more time.  If it fails a second time, burn a fresh live CD at a slow speed (4x) and try again.
