Hi!, **ViliX64,**

In 12.10 Ubuntu The provided method does not provide Keyboard Preferences, only Keyboard Input Methods using Ibus; However, in System Settings>System>Universal Access>Pointing and Clicking, there is a Mouse Keys On/Off switch button. "Control the pointer using the keypad".

I have not tried it properly, 
You do not have to Log out & in again, for it to work, or to restore normal function, and the numberpad keys move the Mouse Cursor, but very slowly with their default settings.

I do not know if it is the same in Xubuntu.
