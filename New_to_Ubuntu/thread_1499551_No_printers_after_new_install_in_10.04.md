### Post by goinglinux on 2010-06-02
I had the same problem. All I had to do was re-install cups. "Presto!" The add printer button was now enabled.

To reinstall:
System > Administration > Synaptic Package Manager
Search for "cups"
Right-click on "cups" and choose "Mark for re-installation"
Click the check mark to apply the changes
