### Post by drs305 on 2009-01-28
I show it in the repositories, so you can do it either via command line or terminal.

Open a terminal (Applications, Accessories, Terminal, or ALT-F2):
```

sudo apt-get install gcc-4.2

```
You will be asked for your password. You won't see it as you type for security reasons. Once you have typed it, press ENTER.

Install via synaptic:
System, Administration, Synaptic Package Manager.
Click anywhere in the right pane of listed apps and then type "gcc". It should take you to entries near gcc-4.2. Highlight "gcc-4.2" and then right click, Mark for Installation, (if is says there are other packages to install, select "Mark" again), then from the top menu "Apply"
