### Post by cariboo on 2009-11-11
You still have Gutsy sources in your /etc/apt/sources.list, comment them out first, then try updating the package list again. You can edit the file by pressing Alt-F2 and typing:

```
gksudo gedit /etc/apt/sources.list
```

either delete the lines with Gutsy in them, or put a # at the start of the line.
