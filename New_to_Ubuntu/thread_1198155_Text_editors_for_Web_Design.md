I am in the process of moving my main applications from Windows to ubuntu, on my dual boot notebook. 

One of the things that I do is Web design, and I use a text editor on Windows called Notetab Pro.

Even as a paid registered user of NoteTab Pro, I want to tell the developers Fookes Software in Switzerland, to "stick it", and switch to a text editor that is made for web programming (and other programming), that is native to ubuntu/Linux. 

**I am interest in finding out some recommendations for any text editors that are designed for web developers. If they could work in Windows as well, great, if not, too bad.**

PLEASE let me know if you have any suggestions.
