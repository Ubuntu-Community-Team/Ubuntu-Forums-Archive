### Post by davidmohammed on 2011-04-06
try synaptic manager - right click each of the updates and choose the option to install.  On occasions to install some updates, it needs to deinstall something else.  Update Manager doesnt do this.

If it says something like "missing a dependency" - then its most probable that there are a series of updates coming from the repository but not all have arrived yet.  Give it 24 hrs.  They will usually arrive by then.
