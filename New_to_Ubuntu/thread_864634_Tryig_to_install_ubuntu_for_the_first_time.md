Well, lots of question here:
1. use the 32bit version
2. I would install using the manual partition option not the guided whole hard drive option. However if you have a restore partition for Vista this could create a small problem because max primary partitions are 4, so that means you have (1,Vista,1,Restore,1,Ubuntu,1,swap, then you cannot install any other partition if thats the way you want to go, but some computers will put a backup partition on there for your backup data and thats where the problems come in. Because now there are too many primary partitions. There are ways to work with Logical partitions etc. but you have to see if you have a backup partition first to be sure if it's reason for concern.
3. I would just install the standard Ubuntu first to get it all working, then you can add all the stuff and actually create a separate repository on a separate CD if you like, or DVD and install stuff from that. 
or just use the Add/Remove feature.


Anyhow it's nice to know it's not the computer and just something to do with the way the vid settings are acting. Once you get things installed and working good, then you can mess with the video resolution and setting to make it work on your big screen.

Thats what I recommend.
