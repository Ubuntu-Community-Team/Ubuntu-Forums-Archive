### Post by Bucky Ball on 2008-07-20
No problem. All updates will be waiting for you when you eventually get online and the system should automatically alert you to this. In the meantime, get aquainted with what is loaded from the install cd, have a look around and set up your machine a bit.

You might want to have a look at 'system/administration/Synaptic package manager' and in the drop down 'applications' menu 'add/remove'. You will need to be online to use these functions (and may get a message to this effect) but this is where you will get software later updates, packages/programs later.

Have fun ...
