### Post by tgyorgyi on 2008-09-17
Hi, as far as I know, none of the synchronisation softwares have come out with a stable version yet... [OpenSync]("http://www.opensync.org") is the most commonly known, if you prefer to use it through a GUI, you can go for Multisync. These are both available through Synaptic package manager.

On the Opensync website you can find a [device compatibility list]("http://www.opensync.org/wiki/DeviceCompatibilityList"), so you can check if your device is there, or if you are just about to purchase, you can see which way you should go.

If you get a device with Windows Mobile on it, you will need [SynCE]("http://www.synce.org/moin/SynceWithUbuntu").

### Post by tgalati4 on 2008-09-17
Skype works fine on the n800 for voice only.  There is no video support yet.  At 400 MHz for the n800, skype eats most of the cpu cycles so you can't do much else during a phone call.  For instance browsing a web page to find map directions will cause voice stuttering.  It's something that you need to pay attention to when using skype because the n800 is made for multi-tasking.  It just doesn't have the horsepower to make a smooth user experience.

### Post by oldsoundguy on 2008-09-17
The real issue here is what is the original OS and how much ram does the PDA itself have.  Standard distros just will NOT work most of the time.  You need a small distro such as DSL (Damn Small Linux) or Puppy or something similar or even SMALLER.
And then there is the fact that, unlike a computer, you do not LOAD the system, you FLASH the system onto the chip!!

There was an effort to create a system to rep[ace the Windows on a lot of the PDA's (called Familiar), but it seems that the developers have either lost interest of run out of money and time.  It was hit and miss for the iPaq models.  Sometimes worked, sometimes not and sometimes turning the unit into a chrome brick.
Some Japanese developers have come out with a really NICE Linux based substitute that DOES work .. only issue .. it is ONLY in Japanese.

When the US gets into PDA's as much as the UK and Japan, THEN you may see some development.

Currently Opera provides a legitimate browser for the Windows based units that bypasses a very crippled IE that works in TEXT ONLY .. no HTML on the earlier models.  (but you have to BUY the browser.) Chrome is supposed to be coming out with one also, but their Windows version, although nice, is weak on accessing some web sites.

All in all, PDA's are NICE.  Especially if they have 802.11 capabilities and Blue Tooth!  (I use mine to access the web and for GPS navigation mostly.)(but on most the sound sux, and a screen that small is really impractical for viewing!)

IF you want Linux on your PDA .. PALM!
