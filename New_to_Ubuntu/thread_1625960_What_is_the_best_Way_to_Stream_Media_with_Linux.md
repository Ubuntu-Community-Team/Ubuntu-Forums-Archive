### Post by LowSky on 2010-11-19
VLC is the best media player. For any OS.

Using MediaTomb would work, so would MythTV, both use UPnP, which will work on any computer (VLC can see UPnP networks) and on a PS3.

### Post by tgalati4 on 2010-11-19
It depends on where the media is and what your network capacity can handle.  Sounds like you might want to set up a mythtv backend on one machine (a media server) and put mythtv frontend on the machines that will play the content.

I have not played with mythtv in a while, so you would have to search the forums for the latest installation guides.

If you want something simpler, install Banshee on all the machines and enable DAAP sharing.  It works well with music, I haven't tried video.
