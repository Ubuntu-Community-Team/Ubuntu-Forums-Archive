### Post by drs305 on 2009-06-21
Connecting to yahoo via pidgin is a common problem. One thing that seems to work is to change the page server from the default to a specific IP address: 
Pidgin > Accounts > Manage Accounts > Modify > Advanced tab: 
Change the default entry in "Pager server" to: 
> 
66.163.181.170

See if it will connect now. If it doesn't, change it back to the default setting of: "scs.msg.yahoo.com".
