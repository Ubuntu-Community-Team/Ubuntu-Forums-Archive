### Post by Delirium_Trigger on 2009-05-12
Not everything is the same as others have suggested.

Yes, almost 100% of your apps will work, but you get new, KDE only apps as well.
Also, say goodbye to nautilus as dolphin in KDE 4 will be your new file manager. 

In theory, removing KDE if you aren't satisfied is relatively easy.
Some packages will have to be manually removed through Synaptic, however. 

Good luck.

### Post by XubuRoxMySox on 2009-05-12
I think there is more to it than just the desktop *appearance*. There are certain applications which, I'm told, "work better" with KDE (Kubuntu's default environment) and are _designed_ to integrate with it better than others. That's why Kubuntu comes with **K**onquerer (a combination web browser and file manager!) instead of Firefox, **K**mail instead of Evolution of Thunderbird, **K**word instead of Abiword (word processor), etc.

Certainly you can have the KDE desktop *environment* and still run those other applications, but the general consensus on most of the forums and FAQs I've read is that the apps that are *made* for KDE simply work better in that environment than the standard applications do.

So in that sense, the difference is arguably much more than mere eye candy.

-Robin
