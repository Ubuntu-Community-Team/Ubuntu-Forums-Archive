### Post by craig10x on 2014-08-19
Didn't he say he wants to download the iso on a mac but install on an old toshiba?  That's what i read...

If that is the case, download the iso for either 32 bit or 64 bit PC (depending on what your toshiba has) and burn it on the mac as an iso image...then set your toshiba to boot off a cd/dvd in the bios settings, and when you restart in will boot into the live session where you can see how it will work on your toshiba and if it is running well, then you can go ahead and install if you wish by clicking on the "install ubuntu" icon on the live desktop (either dual booting with windows or have it wipe off windows)...

If unity runs too sluggish, then you can consider putting on xubuntu or lubuntu instead...depends on the specs of your toshiba...
When you select the version you want on the ubuntu site, select either 64 bit or 32 bit from the drop down menu...don't select the mac one since you are not going to be using it on your mac but on a windows computer...
