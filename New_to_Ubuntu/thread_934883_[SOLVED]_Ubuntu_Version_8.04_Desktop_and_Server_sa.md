### Post by lisati on 2008-10-01
The "desktop" edition is good for everyday use, and comes with a graphical user interface (GUI). 
The server edition doesn't install a GUI by defulat (it's command line only but a GUI can be installed later) and is good for use as a "server" (e.g. hosting your own webpage, network management etc)
