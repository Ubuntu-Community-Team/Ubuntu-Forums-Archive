---

### Post by bumanie on 2009-12-08
Go to terminal and type > gksudo nautilusThis is super user graphical mode. You should then be able to navigate to the locked directories and delete them - but be careful, remember you are operating in graphical mode with root privileges, so anything you delete will be gone forever.
