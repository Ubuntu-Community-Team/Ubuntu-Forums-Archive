### Post by vanadium on 2013-12-26
Version 12.04 is a version that is being supported during a long time. It is mostly aimed at users preferring stability over the latest features such as more conservative users and companies. Version 13.10 is the newest version, contains the latest software and features, which sometimes comes at the expense more bugs here and there. If you are totally new to Linux, the 12.04 version, which has now been refined and debugged during several years, will probably be your best choice for now.

### Post by fantab on 2013-12-26
Yes, we get latest firefox.
And NO you don't have any anti-virus for Linux/Ubuntu. Linux doesn't work that way.
However, there are a few anti-virus applications which are mostly used to scan files if you are sharing those with Windows users and don't want to infect their machines.
To understand 'security' in Ubuntu: [https://help.ubuntu.com/community/Security](https://help.ubuntu.com/community/Security)

You can install firewall though, actually it is already installed you just need to enable and create some rules for it.
For more info on [Uncomplicated FireWall [UFW]]("https://help.ubuntu.com/community/UFW").
To configure UFW... [http://blog.bodhizazen.net/linux/firewall-ubuntu-desktops/](http://blog.bodhizazen.net/linux/firewall-ubuntu-desktops/)
