### Post by capnthommo on 2009-02-04
hi scunnered
what you can do is open places-home then 'create folder' from the File menu.
when you have this you can change the folder name to whatever you want and put your files in.
then close the file manager down again and go Places-Home foldern and grab the folder you want (Shift+left mouse) and drag-and-drop in the place you want on the left hand pane.
i just tried this out so i know it works, but be careful where yo let go or you might end up with the folder inside eg music.
what will probably also happen as you increase the number of items is that you will get a 'Bookmarks rtchevron' marker instead of the previous items but they are all in the 'drop down'
let me know if you have a problem with this.
cheers
nigel
