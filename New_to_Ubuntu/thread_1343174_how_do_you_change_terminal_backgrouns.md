### Post by teward on 2009-12-01
I have an idea, but it works in Jaunty and probably not much else (unless the terminals are the same).

FOLLOW THIS LIST!


[LIST=1]
[*]Open a Terminal Window.
[*]Click Edit > Profile Preferences
[*]Assuming you're using the Default profile (which you should be), go to the background tab.
[*]Select Background Image.
[*]Choose your image.
[*]If you want the image to scroll up and down with the terminal's scroll, select the "Background Image Scrolls" checkbox, otherwise don't select it.
[/LIST]
That should do it, and do it continually because its the default profile.
