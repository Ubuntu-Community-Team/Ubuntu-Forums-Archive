### Post by Frogs Hair on 2012-04-26
Unity the Gnome Shell and Gnome Fall-back session are all shells that run on top of Gnome 3 in Ubuntu 11.10 and 12.04. The Gnome Shell and Fall-back can be installed from the software center and Unity is the default DE. The Gnome Fall-back session resembles Gnome 2, but is missing many of its features. GTK 3.4 Themes from gnome look will work on 12.04 and GTK 3.2 themes will work on 11.10.   

Unity [http://unity.ubuntu.com/about/](http://unity.ubuntu.com/about/)

Gnome shell [http://www.gnome.org/gnome-3/](http://www.gnome.org/gnome-3/)

### Post by jerome1232 on 2012-04-26
Piggy backing on Frogs Hair a little bit, if you do like gnome-shell better than unity, it's very easy to install and use it on Ubuntu. (I did not follow the links so this may have been covered in them, I don't know)

```
sudo apt-get install gnome-shell
```

At the login screen, click the gear icon, select GNOME as your default session, and login.
