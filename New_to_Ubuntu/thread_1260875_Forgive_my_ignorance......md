### Post by ad_267 on 2009-09-08
A lot of Windows stuff is incompatible with Linux. [Ardour]("http://en.wikipedia.org/wiki/Ardour_(audio_processor)") is a DAW similar to ProTools, but it doesn't have any midi editing capabilities just yet (they're on the way I think). I think there are ways to get VST plugins working, but I haven't tried anything like that myself.

There are some commercial audio applications for Linux, eg Renoise: [http://www.renoise.com/](http://www.renoise.com/).

If you want you can dual boot Ubuntu, or install it using Wubi, so that you can uninstall it from Windows. You can also just run it from the CD to try it out. You can give it a go and see how it works while having Windows available too, but if you're a professional musician and sound engineer I doubt Linux will be able to be your full time operating system unless you put a large amount of work into learning new software and systems.

There's also issues with hardware compatibility, not all audio hardware will work with Linux.

### Post by eriktheblu on 2009-09-08
At least one user has reported Nuendo 3.2 working flawlessly under Ubuntu 8.04  and Wine.

If you're willing to do a bit of work, that may be an option for you.

It's always good regardless of your operating system to be proficient in starting over.

In my Windows days, I kept downloaded programs, plugins, drivers, etc., stored on a small file server (external hard drive works for this too).

If you can reinstall all your programs, the other issue is documents. Now by documents I also mean any files you have created, sound files, movies, spreadsheets, etc.

My recommendation:
1: back up all your pertinent data (copy and paste if need be)

2: Locate your Vista disks. This may be something you have to burn yourself, or contact the manufacturer to obtain. 

3: Go to [http://www.blackviper.com]("http://www.blackviper.com") and see if his advice for Vista helps you 

4: Failing that, download and install Ubuntu. Actually, depending on your patience, download and try out several Ubuntus (8.04, 9.04, Studio 8.04, Studio 9.04). Install Wine, try Nuendo. If it doesn't work, try a different version of wine (1.1.7 and 0.9.53 reportedly worked perfectly where 1.0.0 not so well). Try Ardour.

5: Failing that, try Windows 7 Beta. My wife uses this one and likes it quite a bit. All that I've heard is an improvement over Vista. Check your Vista to see if you have some kind of built in upgrade coupon (many of them sold recently do). Once it leaves beta, you might be able to keep using it for free (or reinstall it for free)

6: Failing that, reinstall Vista using your recovery disks. Your problems may fix themselves after a reinstall.

7: Failing that, you can still get XP.
