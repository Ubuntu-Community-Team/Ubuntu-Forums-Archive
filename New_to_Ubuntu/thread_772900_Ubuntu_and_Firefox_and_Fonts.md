### Post by Jammin80503 on 2008-04-28
I had this problem too, and here is what I did to fix it - 
to to appearance preferences>fonts and change the rendering to subpixel smoothing
if that doesnt work, open firefox, got to edit>preferences>content>advanced and mess with them a bit until it looks good.

EDIT: I'm not sure if the font changes happen immediately, so if you don't see any change, restart firefox.
