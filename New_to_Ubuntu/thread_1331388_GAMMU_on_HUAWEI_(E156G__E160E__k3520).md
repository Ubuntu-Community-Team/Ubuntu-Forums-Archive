### Post by Sven6210 on 2010-03-29
Another good choice is the 'Huawei E5830'. It is a WiFi- and 3G-router, so the device establishes the connection with the 3G network when you press a button on it. The laptop is connected through WiFi to the device. So the device does not need to be physically connected to the USB port of the laptop.

**The advantage**
It works with Linux and you do not need to bother about computability. Any machine with WiFi can connect to the device.

**The disadvantage**
You need a Windows machine to set-up the device (APN, user name, password etc.) and you can not send and receive SMS from Ubuntu.

You can get the device for app. 80 EUR at ebay UK or Hong Kong, the device is labelled with a '3' from the mobile network operator. The plain version without the operators logo is called 'Huawei E5'
