### Post by marshag63 on 2011-02-10
Hi CaptainMark! 

I have a USB install of INX 1.17 (based on Lucid).  You did not say which release you are trying to run.  INX relies on the framebuffer.  Its been a while since I booted an actual cd of INX - I think F5 of F6 will give you different screen size options - I had an issue with 1.17 when I first booted as it was set up for a much larger screen than my desktop had (everything scrolled off the screen - which sounds like your problem) but I got it figured out with the F5 or F6 key.

INX is amazing!  Lots of things I thought could not be done via a CLI environment are easy in INX.  Work through the tutorials, they will give you a very solid foundation to build from.

Here's a little tip - the "search for shoutcast streams" option does not work any more - shoutcast changed something, but if you use the "radio" option (it brings up a text web browser with shoutcast, lastfm and I think Jamendo) you can choose a link and it will fire up and play via playinx.  

Even wireless connections are supported via Ceni.

Be sure to stop by the IRC channel and say hi!

If you are still having problems, post back with what version you are trying to run and we will see if we can help sort this out.

MDG
