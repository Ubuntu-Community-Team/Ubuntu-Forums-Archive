### Post by Ripp_Steakface on 2014-05-14
If I was still running Windows XP, I'm sure my system would benefit from doubling the RAM from 1GB to 2GB, but now that I'm running Xubuntu 14.04, I rarely see even half of my system RAM being used. I monitor many things with conky, and I hover anywhere between 200MB and 500MB used, and I haven't seen it go higher than that yet. To be fair I typically just run Firefox (Ubuntu forums, social media, YouTube, Internet Archive, etc), Transmission, HexChat, File Manager, Terminal, and conky.

I'm using an EeePC 900HA, Intel Atom 1.6GHz CPU, 140GB hard drive. Currently has a single 1GB RAM module in it, and I was looking to throw this one in instead: [http://www.crucial.com/usa/en/eee-pc-900ha/CT909579](http://www.crucial.com/usa/en/eee-pc-900ha/CT909579)

My question is, still being new to Linux/Ubuntu/Xubuntu, is it even necessary to double the RAM? I may anyway because it's so cheap, but I'm interested to know because so far I'm amazed with how well the RAM is utilized. Any feedback is appreciated!
