### Post by ka1axy on 2009-05-13
I can't help you with TomTom, as I have a Garmin, but we may have the same issue: how to install firmware updates which require a Windows-only installer.

My solution was to install the non-free version of VMware server, install XP, and run the proprietary updater there. 

I know the above method isn't very "open", but it *does* work, I have updated both my Garmin handheld and my wife's Lowrance car nav this way.  I don't run Windows, but keep this XP virtual machine for these "gotta have Windows" moments.

VMware has gone to a HTML interface, which is too slow for me, so I'm now using the non-free VirtualBox, but haven't tried updating either GPS with it yet.  You need to load the "Tools" under XP to enable networking and USB. VirtualBox seems pretty good, but you do need the non-free version from the Sun website to get USB support.  

And FYI, Win7 RC1 does seem to run OK under VirtualBox.
