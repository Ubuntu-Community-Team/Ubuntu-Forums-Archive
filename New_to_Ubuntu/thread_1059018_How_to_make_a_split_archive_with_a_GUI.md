### Post by howefield on 2009-02-03
To clarify


1. From the file-roller menu select Archive > New
2. Name your archive.
3. Set the path to where you want it saved, (default is to your home folder)
4. Select the archive type.
5. Select other options and set the split you want and check the box.
6. Press create button.
7. Then press add files, and do that.
