### Post by aysiu on 2008-08-02
It actually depends on whether you're using Ubuntu 7.10 or Ubuntu 8.04

If you're using 7.10, the latest version is 2.0.0.16

If you're using 8.04, the latest version is 3.0.1

The best way to "add" it to the repositories is to just go with what you have. It's already in the Ubuntu repositories and security updates for Firefox should be included in your regular system updates.

If you use the Mozilla download of Firefox, you'll have to update it separately from the rest of your system.
