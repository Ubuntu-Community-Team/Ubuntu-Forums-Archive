### Post by cavh on 2011-06-24
What software are you trying to install via the Software Centre?

Please post the output of this command so we can see what repositories your system is trying to connect to:

```
cat /etc/apt/sources.list
```

In the meantime if there's a specific package you can try installing via the command line but may well run into the same issue until the repositories are confirmed as being correct:

```
sudo apt-get install [insert package name]
```

### Post by wildmanne39 on 2011-06-24
Hi give these commands a try
```
sudo rm /var/lib/apt/lists/* -vf
```
Please use this command exactly as it is written, and do not use with other files or you could wipe out your system. It will remove the damaged list and when you run the second command it will replace it with a new list. Its best to copy and paste the commands.
```
sudo apt-get update
```

### Post by Dry Lips on 2011-10-02
I've seen a lot of people (including myself) experiencing this.
Try running these commands from the terminal:
```

apt-get upgrade 
apt-get clean 
apt-get autoclean 
exit
```
