### Post by I2k4 on 2011-03-25
I tried 10.04 Desktop LTE early on - it really fits quite well on a 4GB USB Drive with Persistence - the OS and all the default packages come well under 1GB.  I suggest installing using the basic instructions here on a 4GB USB drive:

[http://www.ubuntu.com/desktop/get-ubuntu/download](http://www.ubuntu.com/desktop/get-ubuntu/download)

But at Step 2, after the Show me How steps for USB, be sure to set the installer for 2GB of Persistence. From then on, it will boot from the USB, run pretty well, and save your preferences and settings between boots.

I'm using the lighter weight, faster and more minimual Lubuntu now, and like it, but the interface is less helpful than full-featured Ubuntu Desktop.  If you really want Ubuntu 10.04, why not just try it for a while, then uninstall anything you don't want, using the package manager.  From USB, be sure to only uninstall one or two items at a time, since the slow drive gums up the process if you ask it to do too much.
