### Post by cymbaline42 on 2011-08-12
Open Terminal and type either 'realplayer' or 'realplay'.  If one of them works, you can check where it's installed by typing in Terminal: 'which realplay' or 'which realplayer' depending on which one works, and subsequently create a shortcut to that file.

For more information, see [https://help.ubuntu.com/community/RealPlayerInstallationMethods](https://help.ubuntu.com/community/RealPlayerInstallationMethods)
