### Post by MontelEdwards on 2009-11-15
Welcome to Ubuntu! 

And yes, there is a way you can get Evolution to work in the system tray.
1. Install AllTray
        This can be done by going into the "Software Center" if you are using Ubuntu Karmic (9.10) or in the Add/Remove section for older releases.

2. Put Evolution to tray.
        Go to Application>Accessories then click AllTray. It will ask you to click the application window that you want to be put to the system tray.

That is it
