### Post by cor2y on 2008-05-23
Make sure you have all the dependencies.
For the default mpeg2 output and i expect the other file formats, smilutils is required.
Also make sure the dependencies are the correct versions.
[B]QT4 >= 4.3
smilutils >= 0.3.2
netpbm >= 10.34
mjpegtools >= 1.9
sox >= 14.0
ImageMagick
mplayer / mencoder[/B]

Nothing listed for Imagemagick or mplayer/mencoder but i would still use a modern/recent version of each.
