### Post by patchwork on 2010-03-21
Absolutely you can.  I personally use icewm, fluxbox, and GNOME.  As for the slowing down, I haven't noticed anything of the sort (GNOME and KDE are both pretty heavy, so if you try XFCE you may even see a speed increase).

You can either install the entire desktop sessions (including all of the libraries and default programs) for kde and/or XFCE, or you can install only the bare window manager (you will need to customize to use your existing services)

For the full desktops```
sudo apt-get install kubunu-desktop
sudo apt-get install xfce4
```For the window managers```
sudo apt-get install kde-window-manager
sudo apt-get install xfwm4
```
