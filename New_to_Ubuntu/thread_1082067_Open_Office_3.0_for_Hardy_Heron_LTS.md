### Post by Ahunt on 2009-02-27
I do know that it can be downloaded and installed directly, but there is always the risk of untested incompatibility problems with that approach. 

A good example of this is ClamAV. Hardy is still running ClamAV 0.92.1, even though the current version is 0.94.2. 0.93 was tested as described on launchpad [https://bugs.launchpad.net/ubuntu/+source/clamav/+bug/228438](https://bugs.launchpad.net/ubuntu/+source/clamav/+bug/228438) and found to be incompatible with Hardy. 0.93 is available for Intrepid.

Downloading and installing your own apps is always going to be not as sure a solution as having the repository upgraded.

I just wondered, since Hardy is supported until April 2011, if OpenOffice would be upgraded the same way Firefox is?
