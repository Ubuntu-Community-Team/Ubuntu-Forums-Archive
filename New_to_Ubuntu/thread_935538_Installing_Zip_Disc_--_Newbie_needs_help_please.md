### Post by cariboo on 2008-10-02
If you have a usb zip drive, there is no setup needed, just plug it in and it works like any other usb device. If you have a parallel zip drive it should be detected as soon as you connect it. If you have an internal ide, it gets detected at boot just like a cdrom. I have all three, I just haven't used them for years. The only thing they are good for now is text files, 100MB is not enough for media files. :)
