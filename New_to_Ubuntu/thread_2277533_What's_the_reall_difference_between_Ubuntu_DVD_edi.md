### Post by sudodus on 2015-05-09
The only Ubuntu flavour that comes with a CD size iso file is Lubuntu. The other iso files are too big to fit on a CD disk (700 MibiBytes). The Ubuntu iso files (32-bit and 64-bit) do not contain the restricted extras. One reason is that it is possible to install Ubuntu in a computer and sell it. The restricted extras are OK for an end user to download and install, but it would be a violation of the EULA to include it in something that is sold.

If you buy a DVD disk with Ubuntu, I think you get the same content, as if you download the Ubuntu iso file and burn a DVD disk.

You might find some special 'remix' editions made by people in the Ubuntu community with a lot of extra packages included.

And of course, you have all the official Ubuntu flavours, Edubuntu, Kubuntu, Lubuntu, Mythbuntu, Ubuntu Gnome, Ubuntu Mate, Ubuntu Studio, Xubuntu. There are also many re-spins based on the Ubuntu repositories. Some of these flavours and re-spins come with more packages installed. They are tweaked in different ways, but most of them come with a minimum of application programs and leave to the end user to select which packages to select for installing.

Try Ubuntu (Kubuntu, Lubuntu, Xubuntu,  ...) before installing it[/URL]
