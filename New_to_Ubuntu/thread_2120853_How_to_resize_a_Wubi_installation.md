### Post by arpanaut on 2013-02-27
Wubi installations have a 30gig limit for the filesystem size, so you are stuck on that.
Have you considered a full install of Ubuntu on to it's own partition. Or a migration of the wubi to a true dual boot installation.

[https://wiki.ubuntu.com/WubiGuide](https://wiki.ubuntu.com/WubiGuide)
