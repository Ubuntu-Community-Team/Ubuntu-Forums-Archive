### Post by SkyNet2029 on 2009-10-18
Oh. haha. sorry about that. The links I provided are for beginning to advanced programming in c++. If you work through some of the downloadable source code, that should give you a good overview of the structure and logic used in creating a c++ program. Also, just google searching 'c++ tutorial' would garner alot of good, if not decent tutorials for some hands on experience. The key to any success at programming is getting your brain and hands dirty. Find an application that you understand, as to what it does and why, then get the source code and play around with it. See what makes it tick and don't be afraid of getting your hands on a good programmer's manual. Read it, then read it again. Main thing is to keep yourself active in programming so that all of those cool little projects you create stay fresh in your mind. Good luck!
