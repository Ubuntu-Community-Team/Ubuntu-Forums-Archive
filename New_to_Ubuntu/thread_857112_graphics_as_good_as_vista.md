### Post by marshall.robert on 2008-07-12
if you want the glassy look, go for emerald ([click to install]("apt:emerald?section=universe")), which is the fancy window borders, and i can make a vista like theme for you based on one i have on my older desktop.

i can never find a decent mac osx replica dock, so if you find one please let me know.

also you may want to install the Advanced Desktop Effects Settings, [click to install]("apt:compizconfig-settings-manager?section=universe"), and setup the effects to the way you want, you can create vista like effects if you tweak well.

i have a pretty much exact same hardware setup (though you didnt specify what chipset, but mine is an 8600m GT) and i have it running very smothly.

Hope this helps get what you want
