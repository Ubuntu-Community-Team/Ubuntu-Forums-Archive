### Post by Sef on 2008-09-07
GRUB is a boot loader.  It first awakes when the computer is turned on, then it wakes up the operating system.  In dual or more boots, it will give the option of which operating system you want to boot into.  GRUB = GRand Unified Bootloader.  For more information on GRUB, click [here]("http://www.gnu.org/software/grub/") (GNU GRUB) or [here]("http://en.wikipedia.org/wiki/GNU_GRUB") (Wikipedia.)
