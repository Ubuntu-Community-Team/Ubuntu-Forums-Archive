### Post by neoAnderson on 2008-07-02
> **Rim3nX said:**
> Can ubuntu have a button for switching input language, like WinXP has... I need to input Serbian, and also Engilish, and I hit the wall with this problem, I cant switch betwin languages... Help Please...:confused:

Actually, Ubuntu is more flexible in allowing you to switch between languages than Windows is... do the following:
(1) First, make sure the language support is installed (to make sure you can properly see all the special characters of your language like diacritics, accents, umlauts etc.) - System -> Administration -> Language Support, and then select the ones you want.
(2) Go to System -> Preferences -> Keyboard
(3) Here, click the Layouts tab
(4) Now, click the Layout Options button
(5) Out of the list, select Layout Switching, and therefore choose your preferred method of changing the input language
