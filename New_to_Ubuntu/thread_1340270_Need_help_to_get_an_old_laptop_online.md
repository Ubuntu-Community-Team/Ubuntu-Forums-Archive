### Post by lkraemer on 2009-11-29
If you have an Account, then you should have a loginname and password.

All you need to do is install wvdial, configure it, and dial in to the
local telephone number.

Once wvdial is working correctly, install Gnome-PPP and use it as it
runs from the GUI (Graphics User Interface).

Shouldn't take more than 30-40 minutes.

Guide located at Posting #4:....Just skip down to this Sentence:
If you don't have a serial port, or want to use a USB port check out
the following:

[http://ubuntuforums.org/showthread.php?t=1100310](http://ubuntuforums.org/showthread.php?t=1100310)
