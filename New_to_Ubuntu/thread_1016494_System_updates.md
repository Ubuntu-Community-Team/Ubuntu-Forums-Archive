### Post by xjcannonx on 2008-12-20
You set it when you installed ubunu but you can check by going to

 System>Administration>Software Sources click on the Updates tab

People usually set it to check every day and to only notify of changes...this is so you can review the changes that will be made before they actually happen, which can be really useful

You will get a notification in your gnome-panel when updates are available
