### Post by bwallum on 2010-05-10
Thanks for the screen shot, it confirms your interesting combination of releases. 

Lucid is brilliant at the moment. There is an issue around the splash screen and proptietory nvidia drivers fighting but the improved default video drivers are pretty good. However as a gamer you may have to forgo the default splash if you need full accelerated graphics that come with the proprietory drivers.

If you go to you home folder and have pictures, documents, music etc folders then back these up to any other media you might have. They will probably all fit on a good sized usb memory stick but you can check the size first.

Go to Places>Home Folder and look for the folders that you have stuff in. Right click on any folder and select Properties from the menu. That will tell you the size of the folder.

I would'nt back up any system files because of the current mix, I would do a fresh install.

Let's get a fresh download of Lucid. First you need to decide if you want 32bit (i386) or 64bit (AMD64) Ubuntu. This will be determined by your pc architecture. 

Then go to [http://releases.ubuntu.com/releases/10.04/]("http://releases.ubuntu.com/releases/10.04/") and download your choice. (Save the file)

When the file is downloaded use Brasero to burn it to a 700Mb cd (or larger if you only have a larger format).

When you have a nice new live cd, sswitch off and reboot with the cd in the optical drive. When asked if you want to use the entire drive, respond with yes and Ubuntu will then install afresh, completely removing all the past bits.

Let me know how you get on. It will take an hour or two so plan the install around that.

CAVEAT: I'm assuming that you don't have other operating systems on the hard drive that runs Ubuntu. Is that correct? Also do you dual boot Windows on the same computer? The above assumes that you don't.

Good luck

### Post by bwallum on 2010-05-10
> **blueghoti said:**
> ... I'm considering putting Windows 7 on my machine, but am just a bit anxious about whether that would work or not.

It will work ok if the hardware is up to it. I recommend a second hard drive. It's a lot cleaner when Windows subsequently screws up (with viruses or anti-virus software in my experience) Install Windows 7 on the primary drive (as per BIOS boot order), then install Ubuntu on the secondary drive. The boot loader grub will install on the master boot record of the primary drive and you will be presented with the grub menu that allows you to select the operating system to boot.

There are other ways to do it and if you want to go down that road try posting a new thread 'how do I dual boot Windows 7 and Lucid' or something similar. It's a well trod path and you are sure to get lots of alternatives to choose from.
