### Post by nothingspecial on 2012-03-02
If you are talking about 2 Ubuntu computers ......

Install openssh-server on the computer you want to connect to

Then in the file browser's menu on the other one go

file > connect to server

Input the ip address of the other computer along with the username and password.

Click connect :)
