### Post by gwoodruff on 2011-05-29
Are you using an ISO of the live cd or alt cd? You can press f6 to edit the boot parameters. Try adding a space and  –xforcevesa to the end of the boot sequence and this will force safe graphics mode. Some versions of the ISO's allow you to choose safe graphics with the f4 key (32bit versions I think).
