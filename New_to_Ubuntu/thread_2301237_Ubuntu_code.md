### Post by TheFu on 2015-10-31
There are 500+ different languages.  [http://rosettacode.org/wiki/Main_Page](http://rosettacode.org/wiki/Main_Page)
The .bashrc uses  "bash" as the language. The "Bourne Again SHell".
It is relate to other shell scripting languages, like ksh, Bourne shell, tcsh, csh. Bourne Shell = sh, but it is mostly gone on Linux systems.

C and C++ are compiled languages like java, Go, Dash. 
Perl, python, php, ruby, bash, awk and many other "scripting" languages are interpreted languages.

You've just opened a new door to a new world, especially if you came from the MS-Windows.  On Unix systems, it is very common to create little scripts to perform tasks. Whenever I do something  a 3rd time it is time to automate that thing - usually though a  small  script. Being lazy is a key attribute and skill for Unix power-user.
