Found these instructions from another user who uses Hebrew:

> System --> Administration --> Language Support
then in the next window:
--> Install/Remove Languages...

--> Hebrew (and here I added everything I could, Translations, Spellchecking and writing aids, Extra fonts)

### Post by grahammechanical on 2011-03-15
This is how I do it with 10.10:

system>preferences>keyboard>Layouts tab. click Add and add a hebrew layout When the Utiltiy closes you will see in the top panel a keyboard icon. Left click the icon and you will see a list of keyboard layouts installed. Select the one you want to use. Do this after you have opened a wordprocessor document and you will type in Hebrew. left click and select English and you will type in English. I do this all the time with English and Greek. To get accented characters I use the Insert>Special characters in Open Office.

One word of warning. If your keyboard layout is set to Hebrew and you need to enter your password then it will not work if the install language was English.

By the way, to get your menus back to English you may be able to do it through System>Language Support. Select English as the language and click Apply system-wide.

Regards.

P.S. You may not need a special Hebrew font. The fonts in Linux are unicode fonts. They have all the characters for many languages. It is just the keyboard layout that needs to be changed.

P. P.S. to install a font click on the font. The Font Viewer Utility should open and display the characters of the font. You will see a panel called Install font. Click it.
