### Post by PhoenixP3K on 2008-04-23
You might really need to install the ATI drivers for this to work. 
[http://ati.amd.com/support/drivers/linux/linux-radeon.html](http://ati.amd.com/support/drivers/linux/linux-radeon.html)

1. Launch the Terminal Application/Window and navigate to the ATI Proprietary Linux driver download. 
2. Enter the command ati-driver-installer-8-4-x86.x86_64.run to launch the ATI Proprietary Linux driver installer. The ATI Proprietary Linux Driver Setup dialog box is displayed.
3. Follow on screen instructions

Hope this can help too.
