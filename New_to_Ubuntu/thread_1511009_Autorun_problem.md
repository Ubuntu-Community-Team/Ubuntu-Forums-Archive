### Post by 3rdalbum on 2010-06-16
Go into your computer's BIOS setup (or Boot Order screen) and set it so that the computer tries to boot from CD-ROM before it looks at the hard disk. Then reboot with the Ubuntu disc in the drive and follow the instructions.
