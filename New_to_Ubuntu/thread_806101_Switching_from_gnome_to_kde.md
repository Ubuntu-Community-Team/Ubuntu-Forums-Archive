### Post by CloudFX on 2008-05-24
> sudo apt-get install kubuntu-desktop

will install it for you.  When you are at your Login Window, choose Options > Session > KDE.  Then when you log in, you will be given the option to change defaults. No data will be lost, and all the KDE applications you want will be available.

### Post by sayakb on 2008-05-24
1. To some extent, yes. Though if you miss out anything, just install it from the repos.
2. You might be able to upgrade to 8.04, but chances of success is not 100%, just, for example, update screwed up my entire installation.
