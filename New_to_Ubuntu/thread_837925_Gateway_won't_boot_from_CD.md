### Post by PoopyTheJ on 2008-06-23
According to [this]("http://www.michaelstevenstech.com/bios_manufacturer.htm") site to enter the BIOS you need to press F1. If you are getting a password prompt you'll probably need to disable the BIOS password, usually there is a jumper on the motherboard to enable/disable the BIOs password. try setting the jumper to disable the password and then try getting into the BIOS. You may need to find a manual for the PC if the jumper isn't easily marked, usually those can be found at gateway's website.
