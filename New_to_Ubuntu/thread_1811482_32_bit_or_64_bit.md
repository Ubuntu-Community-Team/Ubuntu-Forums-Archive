### Post by Joreus on 2011-07-25
General rule of thumb (in my experience): if you can run 64 bit systems, you should. You can run the 32 bit version, but it's a little slower than if you were to go the 64 bit route.

At least, that is what I've gathered from these forums.

### Post by Snowboi on 2011-07-25
32 bit is generally known to have better compatibility with applications, but 64 bit is generally faster. :KS

32 bit runs 32 bit applications
64 bit runs 64 bit applications

64 bit usually requires a stronger computer then 32 bit.

For speed use 64 bit, if your gonna use a lot of applications use 32 bit.
