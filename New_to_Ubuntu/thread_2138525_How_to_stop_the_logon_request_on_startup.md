### Post by ninjaondemand on 2013-04-24
Ubuntu keeps asking you to log back in (or "authenticate") so it can confirm that you* really* want to install what you are trying to install. This is a security feature, designed to make it so you can't install malicious software by accident. Look at [this thread]("http://ubuntuforums.org/showthread.php?t=1337564") for advice on how to disable this security measure (**NOT RECCOMENDED!)**.

### Post by JRV on 2013-04-24
Click the icon in the upper right habd corner.
Click on "System Settings".
Click on "User Accounts".
Turn "Automatic Login" on.

This will automatically log you in when the system starts.

To stop the screen from locking after a period of inactivity click on "All Settings"
Click on "Brightness and Lock".
Change "Turn screen off when inactive for" to "Never".
