### Post by Paqman on 2009-09-19
Is it a Linux game? If not, you're probably not going to be able to install it.

If it's a Windows game, there's a chance it'll work through Wine. Check on the [Wine App DB]("http://appdb.winehq.org/") for your game. If it's rated gold or platinum you should be able to get it to work. You'd then need to install wine (Applications > Add/remove > search for "wine" and apply). Then just pop the disk in and click on it when it pops up on your desktop.
