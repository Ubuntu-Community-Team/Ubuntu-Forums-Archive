### Post by Gone fishing on 2009-07-30
Yes you can install SUSE on another partition and it will be able to read and mount the ubuntu partitions. If you install SUSE it will overwrite grub and make SUSE the default os although this is easy to fix.

You can specify the width of the toolbars by right clicking and in properities specifying the width in pixels. You could even go for just one toolbar and / or autohide the toolbars.

### Post by RikoW on 2009-07-30
Hi,

do you want to adjust the icon and text size in OpenOffice especially? If so, in OO go to Tools -> Options and select 'View' under Openoffice.org and adjust the size of the toolbar icons and font size with "Scaling" and "Icon size and style".

Hope that helps,
