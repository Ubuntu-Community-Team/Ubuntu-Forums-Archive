### Post by balbecdaze on 2009-04-16
I wrestled with this yesterday after plugging in new speakers - applications played music (Rythymbox, games) but anything through firefox didn't.

I checked I was up-to date (System-> Administration -> Synaptic Package Manager -> Mark All Upgrades)

Then I opened System -> Preferences -> Sound and tested the various parts, changing the selection from Autodetect if it didn't work)

Finally I opened the volume control and under preferences selected all tracks to be visible and turned them all up quite high.

I think it was the last step that fixed it!
