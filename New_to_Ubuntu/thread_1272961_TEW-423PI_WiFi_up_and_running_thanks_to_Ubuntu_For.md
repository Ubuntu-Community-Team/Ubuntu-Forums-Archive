### Post by LaughingVulcan on 2009-09-22
My first post here... and just a great big Thank You to those who contributed to earlier threads about the TrendNet TEW-423PI cardbus WiFi.

Thanks to searching for the model number I was able to:


[LIST]
[*]Confirm that the card was being seen with "sudo lshw -C network" in Terminal.
[*]Go and get ndiswrapper and ndisgtk from my Live CD (pool->main->n)
[*]From an internet-capable machine download/unzip the MRV8000c driver from Trend Micro's website to a usb stick.
[*](Discover that initially I had downloaded the wrong driver set for the wrong card... and go back and get the actual MRV800c driver...  And also confirm the model number of my card thanks to step 1 of [http://ubuntuforums.org/showthread.php?t=400258&highlight=TEW-423](http://ubuntuforums.org/showthread.php?t=400258&highlight=TEW-423) )
[*]Start ndisgtk and install the driver.
[*]Enable wireless from the desktop.
[*]Connect to my wireless network.
[*]Sign up and make my first Ubuntu Forums post.  :)
[/LIST]

So this is just saying Thanks!!!!  :D
