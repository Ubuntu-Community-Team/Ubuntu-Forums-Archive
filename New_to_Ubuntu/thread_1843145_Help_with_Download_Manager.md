### Post by 3rdalbum on 2011-09-13
I don't really see a need for a download manager. Web browsers can download things just fine, really.

If it's a big file and I'm worried it might cut out during download, I use 'wget' which is a command-line program for downloading files reliably. There are GUI versions of it available in the Ubuntu repositories. They do the same thing, but they work like a proper GUI program.

I have also used DownThemAll on Firefox for downloading a bunch of links from a web page, but this is the only reason I can think of for using it.

Most of my real downloading comes from torrents, which are fairly reliable and fairly quick. Transmission is preinstalled on Ubuntu and works well for torrents.
