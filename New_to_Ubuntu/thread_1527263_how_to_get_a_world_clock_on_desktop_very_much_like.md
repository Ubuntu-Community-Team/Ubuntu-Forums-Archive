### Post by Paqman on 2010-07-09

### Post by Kimm on 2010-07-09
You can also try cairo-clock [http://gnome-look.org/content/show.php?content=34256](http://gnome-look.org/content/show.php?content=34256)

Its a little program that puts a clock on your screen, it wont render anything else though

Install it by doing:
```

sudo apt-get install cairo-clock-hildon

```
