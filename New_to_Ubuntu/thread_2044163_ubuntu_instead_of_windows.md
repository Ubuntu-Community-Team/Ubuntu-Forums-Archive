### Post by Paqman on 2012-08-19
> **amrosama77 said:**
> 
but the problem comes when I talk about visio and there is no alternatives accepted at all as I am going to deliver visio course

so any advice...?


Dual booting will sort you out. If you don't currently have Windows installed you can install it without affecting Ubuntu, but it will overwrite Grub. You can reinstall it using [these instructions]("https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows").

If you've got a reasonably powerful machine then virtualisation is also an option. I recommend Virtualbox. The advantage of this is that you can run applications in "seamless mode" where they appear to be running on your Linux desktop.

Trying to switch 100% to Ubuntu is fine, but an OS is just a tool. Use the right tool for the job. If you really need to use Windows, just use Windows. Linux is a choice, not a religion.
