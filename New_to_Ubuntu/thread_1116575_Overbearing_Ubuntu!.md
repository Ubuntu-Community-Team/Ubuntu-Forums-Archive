### Post by HereInOz on 2009-04-05
Most definitely it is.  If you have a window which is, say 600 pixels wide, or a panel (task bar) which is 25 pixels high, and you increase the number of pixels on the screen (increase the resolution) then the pixels, by definition, will need to be smaller.  

Therefore the 600 pixel wide window, or the 25 pixel high panel will both occupy a smaller area of the screen once you have increased the resolution.

What screen size are you using and what resolution are you set at?
