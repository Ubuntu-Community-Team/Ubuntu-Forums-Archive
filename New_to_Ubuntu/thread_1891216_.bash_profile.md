### Post by nothingspecial on 2011-12-05
> **dep0 said:**
> 
Suppose it is, would it be right to change the owner of the root direction from root to me?

No, that would break your system.

Put your aliases in ~/.bashrc

So type ```
gedit ~/.bashrc
```

If you like to use a lot of aliases you can make a file ~/.bash_aliases to put them in.

Your aliases won't take effect until bash reads your .bashrc again either by opening a new terminal or typing

```
. ~/.bashrc
```
