### Post by FormatSeize on 2011-07-01
> **Jaksuede said:**
> All seems to be well and good. I was wondering if I would get more speed by taking out Windows XP someday? I assume it is running in its own disk space and not running **on top** of Windows? Not sure since it installed thru Windows. :)
No. It's not running "on top of" Windows. It is using it's own disk space, and any available system resources. You won't get any faster by taking out Windows XP, as it's not using any system resources while you are booted into Ubuntu.
 
Are you dual booting? This response is based on the assumption that you are dual booting.
