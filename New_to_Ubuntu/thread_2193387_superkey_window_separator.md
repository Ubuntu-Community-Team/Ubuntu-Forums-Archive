### Post by deadflowr on 2013-12-12
Not by itself.
The super key alone is unity's go to key foor pening the dash, among other default behavior.

But you can use it in a binding with another key using scale.
Install compizconfig-settings-manager and look for scale.
Then look at the bindings section in that, and set to your liking.

Of course in 12.04, scale is somewhat broken.
There is a patch to try to help fix that, somewhat, somewhere.
