### Post by jtclicker on 2010-07-19
Managed to solve this with a lot od wading around in the forums, just thought I'ad another post for anyone else with the same problem.

If you've got all the drivers loaded and you can change the res manually but it won't boot in the correct res...
Create a file called .xprofile  I did by in terminal typing

```
gedit xprofile
```

and then typing
```
xrandr -- addmode VGA 1600x1200
xrandr --output default --mode 1600x1200
```

Saving the file and making it an executable, then setting it to load at startup by going to SYSTEM - PREFERENCES - STARTUP APPLICATIONS
