### Post by coffeecat on 2010-11-06
"My Computer" is a Windows convention. As you were a first-time poster I wanted to make sure you were running Ubuntu. We sometimes get bewildered Windows users wandering in here needing advice for Windows.

Put the CD in the drive. If a file-browser doesn't open automatically, simply double-click on the CD icon that *should* have appeared on your desktop. Then simply drag and drop any files you want to copy just the same way as you would in Windows. For a destination folder, go to the Places menu and open 'Home folder'.

You cannot run Windows programs on Ubuntu. It is an entirely different operating system. For Microsoft Word files, you can use Open Office Writer found in the Applications > Office menu. If you need help finding Linux apps as replacements for your Windows ones just ask, telling us what your requirements are.

As for Avast. Simple answer: you probably don't need AV software.

### Post by Verbeck on 2010-11-06
wine is a program that enables windows executables (.exe's) to run in a linux distribution . you can install wine through the software center. after the installation, right click any .exe and select open with wine

as i said before, dont expect it to run perfectly, or at all. you can list the supported apps at [http://appdb.winehq.org/](http://appdb.winehq.org/) with a platinum/silver/gold rating

also if you're new to linux, try reading the link cofeecat posted

edit : try reading this too [http://www.whylinuxisbetter.net/](http://www.whylinuxisbetter.net/)
it outlines some reasons to choose linux and some to stick with windows
