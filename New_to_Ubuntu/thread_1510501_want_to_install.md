### Post by Saint_ on 2010-06-15
Alright so for starters to put ubuntu on a flash drive, go to: [http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/) and download unetbootin. It's a fairly easy program to use, all very self explanatory. So once you have your live ubuntu usb, just restart your netbook with the flash drive plugged in, go to your boot options and select to boot from the usb. From there, just follow all the instructions and yeah, you can install ubuntu side by side from the USB. That should about do it.
