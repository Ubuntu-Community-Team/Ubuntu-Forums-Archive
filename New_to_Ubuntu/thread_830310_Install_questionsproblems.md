I would check the system hardware against the minimum requirements. If you're OK there, I would look a bad disk perhaps. Maybe try burning another copy at a slower rate. Also, make sure you didn't fingerprint it up before putting it in. I actually did that once and the install didn't work until I cleaned the disk.

Lastly, I'm assuming you downloaded Hardy Heron (8.04). Like Gutsy (7.10), I think there are two options which you can use to install. You probably selected the first option in the menu. Try the second one. I remember for 7.10, I HAD to use the second option to install the OS 'cause the first option always locked up my computer.
