### Post by padams10001 on 2010-08-06
I have run (k)ubuntu for a few years now with 2GB or less of RAM. I have never used up all the RAM i have had. However swap is there for precaution. Make the swap size around 1-1.5x your RAM size and you will be fine unless you are using RAM intensive programs such as video or picture editing programs in which case you will likely need mire RAM.
