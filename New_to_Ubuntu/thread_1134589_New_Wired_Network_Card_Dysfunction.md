### Post by cariboo on 2009-04-24
IF you can access web sites using firefox then your network card is working correctly, you have other problems.

A network card will only transfer data at the speed of the slowest component in your network. If you have a 1Gb nic connected to a 100Mb router or switch, you won't get 1Gb transfer rates.

I would suggest you check /etc/resolv.conf to make sure that you have the correct dns nameserver addresses. Mine looks like this:

```
cat /etc/resolv.conf
# Generated by NetworkManager
domain aplus
search aplus
nameserver 64.59.168.13
nameserver 64.59.168.15
```
