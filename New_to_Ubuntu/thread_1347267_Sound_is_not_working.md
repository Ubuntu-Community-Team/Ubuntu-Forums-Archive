### Post by LuisGMarine on 2009-12-07
Hello and congrats on install Ubuntu 9.10 yay! :popcorn:



Anyhow, down to business.  First open up terminal by going to **Applications > Accessories > Terminal** and typing in the following code:

```
aplay -l
```

If something comes back, then check your audio mixers to make sure you do not have any of them muted.
