### Post by sudodus on 2012-12-31
Welcome to the Ubuntu Forums :-)

If you can see it from the Mac, and you have the same gateway you should also see it from your computer with Ubuntu.

- Can you ping the gateway (or router)?
- Can you ping the Mac?

- Which is the IP address of the NAS?
- Which is the IP address of your computer with Ubuntu?

This information should be safe to post, since they are addresses in a local network behind a router, usually standard addresses like

192.168.0.2
