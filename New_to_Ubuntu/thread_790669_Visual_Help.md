### Post by Fjatle on 2008-05-11
Go here, and download Screenlets: [http://www.getdeb.net/app/Screenlets](http://www.getdeb.net/app/Screenlets)
You can download from Synaptic, but the version in the repos is not as new as this one.

I think you will find it in Programs - Accessories, or in System - Preferences.

And if you want it really neat, you can use a widget-layer.
How?
Install the package compizconfig-settings-manager
System - Preferences - Advanced Desktop Effect Settings.
Find the widget layer, and tick to use it. 
Click on the Icon to change its settings.
Now, in the General Tab, you can decide how to activate the Widget layer. 
E.g you can use a screen edge (top right corner).

Now, back in the Screenlets program:
Find CPUMeter, or the more advanced Sysmonitor.
Tick of both start/stopp and Autostart on login.
And then you click Preferences (in the left menu), and mark 'Widget'
