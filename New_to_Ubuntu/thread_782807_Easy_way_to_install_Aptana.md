Well, sort of.  If you are using 64-bit then you have to install it as a plug in for Eclipse, but apparently you can install Aptana on its own if you have 32-bit Linux.

> **System Requirements:** Aptana Studio standalone requires Windows 32 or 64-bit platforms, Mac OSX 10.4+, or Linux 32-bit with GTK. 64-bit Linux users please install Aptana as a plugin into Eclipse. Full system requirements are listed here: [Aptana System Requirements](http://www.aptana.com/docs/index.php/Aptana_System_Requirements). 

Of course, I didn't remember that I installed the 64-bit version of Ubuntu 8.04 until *after* I had downloaded the 32-bit package and tried to install it.  ;-)
