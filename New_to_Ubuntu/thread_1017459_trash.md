### Post by Elfy on 2008-12-21
If it'd on the desktop you can rename it - Alt+F2 - run gconf-editor

Navigate to /apps/nautilus/desktop

enable trash_icon_visible
double click trash_icon_name - change type to string and enter name in value, Ok
