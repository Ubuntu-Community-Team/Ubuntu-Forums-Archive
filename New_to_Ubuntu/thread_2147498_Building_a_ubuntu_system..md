### Post by richierich1986 on 2013-05-22
AMD motherboards and CPU's work just fine with Linux. WoW works near to flawless, if not that, from what I've heard and read.
There is a list of supported Windows applications over at the Wine HQ website (just search for "Wine HQ"). The website is pretty
much self-explanatory. Support for AMD and Nvidia GPU's is okay.. I myself have a Nvidia GTX560 Ti, which works very well.
This took quite some time though. It's only since Ubuntu 12.10 that the provided proprietary drivers are working well for me. 
Before that, I had to manually download and install the driver from Nvidia's website, or my system wouldn't even start the GUI
after I installed the provided Nvidia driver. It's only in the last three months that drivers from Nvidia really started to work very
well. They have been a little busy. Maybe Steam on Linux has something to do with that. ATI drivers have always worked well
for me personally. My previous GPU was a Radeon HD5870. I think you'll be fine with your card.

Are you planning on making games for Linux then? I'm guessing you are. Creating Windows games on Linux doesn't really work
in your advantage.

Unity is a graphical user interface, not an editor. For that, someone else here might give you some better advice than I'm able
to give.

Good luck! :)
