### Post by deadflowr on 2013-10-19
The default layout is called smart.
Don't know if an app is preinstalled, but if you install the program compizconfig-settings-manager, you can change the layout to center in the section "place windows".
They default to smart because it makes every new window open in a distinct section of the screen, so they don't completely overlap.
Which is what happens when the default is center.
