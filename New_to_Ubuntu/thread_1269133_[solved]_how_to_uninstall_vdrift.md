### Post by etnlIcarus on 2009-09-17
Telling people to run commands to uninstall software, in the Beginner Talk forum, isn't particularly helpful.


For future reference, when you want to remove software, you can:

- Open up, "Add/Remove", in the Applications menu. Use the search box to find the application you want to remove. Un-tick it, then (if I recall correctly), press the, "Apply", button, to remove the programme.

- Open up, "Synaptic Package Manager", in the System>System menu. You can then go through a similar process to the one outlined for using, "Add/Remove".
