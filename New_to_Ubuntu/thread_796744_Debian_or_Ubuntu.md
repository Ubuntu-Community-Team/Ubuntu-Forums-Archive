### Post by tompickles on 2008-05-16
ubuntu is based on debian... but debians focus is very different. debian has a very slow release cycle (ie 4 years ish) whereas ubuntu is 6 months for the normal releases. debains aim is on stability and well tested and security, rather than new. there is a lot of outdated software in debian, but it is as hard as nails.... as a desktop --- personally id go with ubuntu. but, i like pretty desktops. you can always dual boot..... or try one, wipe and try again!

### Post by Monicker on 2008-05-16
I have used Debian for several years now.  Its not as polished as Ubuntu, and some would say its not as user friendly.  Debian's biggest strength is its focus on reliability and stability.  If you don't mind doing more things manually, and taking some time to learn what is behind some of the Ubuntu gui wizards, it will work fine for you.
