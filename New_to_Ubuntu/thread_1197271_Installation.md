### Post by cariboo on 2009-06-25
You can install almost any desktop you want. try

```
sudo ubuntu-desktop
```

This will do the full desktop installation. Other options are [Webmin]("http://www.webmin.com/") and [ebox]("http://ebox-platform.com/"). You may also want to look at this [document]("https://help.ubuntu.com/9.04/serverguide/C/index.html").
