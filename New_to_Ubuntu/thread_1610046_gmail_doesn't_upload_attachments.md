Hmmm, I use Gmail too and have no problem with attachments. You say it works fine when booting into Windows - are you dual-booting the machine and using the same internet connection?

Regarding Word docs opening as PDFs - do you have an office package (e.g. OpenOffice Writer or AbiWord) which can read Word files? If you click on the 'view' option for an attachment within a message in Gmail, Google imports the file into your Google docs folder and then displays it - you might be hitting a problem here - you can bypass this step by choosing the 'download' option instead of the 'view option'.
