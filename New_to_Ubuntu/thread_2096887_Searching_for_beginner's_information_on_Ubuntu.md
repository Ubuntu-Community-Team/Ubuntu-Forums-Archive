Well, the first thing you should know is that Ubuntu doesn't necessarily work well on netbooks. The netbook may not have enough CPU power and/or graphics capability to run the full 12.10 desktop version.  So, you may end up needing to install Xubuntu or Lubuntu.

Another reason is that netbooks, like laptops, often have specialized hardware builtin (to keep the costs down) and Linux, all too often, does not have drivers for that hardware.

Thus, before you install anything, you need to find out if your netbook can boot from CD or from USB.  This is because you really need to make a LiveCD (or LiveUSB), boot from that, and see how well that finds your hardware and installs the proper drivers.
