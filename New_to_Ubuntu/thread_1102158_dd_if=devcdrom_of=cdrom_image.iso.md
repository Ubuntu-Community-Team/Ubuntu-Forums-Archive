Whichever directory you specified in the 'of=' bit of the command. If you copied and pasted the exact command above it will be in /

Go to "Places" -> "Computer" -> "Filesystem".

If I were you, next time just right click the CD on the desktop and choose "Copy disc" and choose "File image" in the dropdown. Much easier if you aren't used to the command line.
