### Post by sandyd on 2013-01-07
I used to have this too - it was the result of a bad profile

I did fix it with Edit -> Preferences -> Content -> Advanced -> Deselect "Allow Pages to choose their own fonts"
you also might want to try Edit -> Preferences -> Content -> Colors -> Deselect "Allow Pages to choose their own colors"
