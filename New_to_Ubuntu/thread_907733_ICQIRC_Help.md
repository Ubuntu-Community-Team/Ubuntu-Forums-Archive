### Post by django0909 on 2008-09-01
IRC is Internet Relay Chat. There are loads of different clients, one of which is mIRC. mIRC is Windows only. Xchat is my favorite client. You can install through synaptic, or do

```
sudo apt-get install xchat
```

There are dozens of networks/channels. The best thing you can do is a Wikipedia search for 'IRC', which will explain the basics. If you want help with Xchat, they have a website at [http://www.xchat.org](http://www.xchat.org)
