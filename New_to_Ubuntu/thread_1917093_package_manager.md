### Post by Carborundum on 2012-01-29
If by package manager you mean Synaptic (which is a GUI front-end for the actual package manager, APT), it is no longer installed per default in 11.10. You can install it by running
```
sudo apt-get install synaptic
```Otherwise you can use the Software Center, which is now the preferred GUI for APT.
