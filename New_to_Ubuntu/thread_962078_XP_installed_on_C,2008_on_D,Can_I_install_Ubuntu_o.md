### Post by OutOfReach on 2008-10-28
Ubuntu and Linux distros in general, don't use drive letters (E:, C:, F:, etc...).
And yes you can install Ubuntu on a different partition (letter).

### Post by bumanie on 2008-10-28
You can format to ext3 prior to installation or during installation as ubuntu has its own partitioner - it may also be worth noting that unlike windows, ubuntu will boot from within a logical partition - some other OSes baulk at that idea.
