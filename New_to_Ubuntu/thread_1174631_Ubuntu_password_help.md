### Post by bodhi.zazen on 2009-06-25
Sounds like you installed ubuntu with wubi.

Boot buntu to recovery mode. This will be a command line.

At the command like simply type

```
passwd your_user_name_goes_here
```

then set a password.

Then:

```
reboot
```
