### Post by spydeyrch on 2010-05-18
In a sense, yes. But it is not true dual booting. If you use wubi, it installs ubuntu from within windows, so that to windows it looks like a normal program. But, it also adds a boot menu entry to allow you to boot into ubuntu instead of windows. If in the end you don't want ubuntu, you can uninstall it via windows. Simple and to the point.

However, I wouldn't recommend wubi if you plan on using ubuntu for a long time and/or heavy usage/primary OS. If you are going to use Ubuntu as your primary OS, then do a true dual boot system or better yet, just use ubuntu! :)

But yes, en fin, wubi will allow you to use either windows or ubuntu with a boot menu to select from. Simple and easy.

### Post by lindsay7 on 2010-05-19
If you do a true dual boot, you can setup separate directories for root (/), and home, and swap. That way when you do an update or upgrade, you will be doing that just to the root directory and all your other files and settings in the home directory will be untouched.
