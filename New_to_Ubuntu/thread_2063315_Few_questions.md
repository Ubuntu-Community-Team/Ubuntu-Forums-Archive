Wubi is like a virtual machine, such as vmware.  The os is real but is installed on the virtual.    If you didn't download the iso by itself, you'll need to download a copy to install on the dvd/cd or flash stick to install to a partition.

I think the safest way to install ubuntu 'normally' is to prepare the partition and install a fresh copy.  Trying to migrate it from wubi when you're not experienced can lead to issues.  Save the personal important stuff somewhere, a dvd or flash stick, and just uninstall from the windows add/remove in control panel.

If you're going to reinstall as in issue number 3,  there's no need to worry about issue #2.  Except that it is a good idea to make sure the ubuntu version works on your computer.
