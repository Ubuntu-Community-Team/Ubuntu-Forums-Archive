### Post by Malkhut on 2011-01-23
If you have a .iso file that you made using DeVeDe, then you can check to see if it actually works the way it's supposed to using VLC.  Once you know that it works, use Brasero -> burn image to burn it to DVD. 

I haven't used Arista, so I'm not sure about your error message, but I use DeVeDe quite a bit and I have found that it works well.  You can even use it to convert other container formats (.avi, .mkv, etc) to DivX, if your dvd player is DivX compatible.  That saves some disc space over making a "real" DVD with menus and all.
