Nice laptop, that. Decent specs.
But...
Way too low on the [COLOR="Red"]RAM[/COLOR].
You MAY be able to install 9.10 with the Alternative CD,
but it would run like a paraplegic slug.

There are other versions of Ubuntu that are lighter,
and there are other distros,
Unless you add more RAM (at least 256m with swap, better 512M with swap)
you will be VERY disappointed in the mainstream Linux Distros
Mandriva, SuSE, Debian, Ubuntu, etc.

Ubuntu 8.04LTS MAY be a decent Ubuntu at 128M.
You would need the Alternative Installer CD,
the LiveCD installer wants at least 192M to run.

I think even Puppy would have trouble with 128M

Any others have opinions?
