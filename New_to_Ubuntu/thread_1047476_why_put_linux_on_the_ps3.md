### Post by mcduck on 2009-01-23
> **insanity99 said:**
> so i can do pretty much anything you do on a desktop pc?

yes, although you won't be able to use any Windows programs, not even with Wine (since PS3 has a different type of processor, Wine doesn't do hardware emulation).

The selection of available software is also a bit smaller than on x86 systems, but as long as the software you want to run is open-source there should be no problems.
