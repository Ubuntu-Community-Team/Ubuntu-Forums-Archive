### Post by mcduck on 2008-08-25
You are not supposed to extract files from the .iso file. It's a disk image, and should be burnt to disk as image.

(this means, do not extract the files and burn to disk, ands do not burn the .iso file itself into the disk, but instead use the "burn disk image" or similiar function in your burning app)
