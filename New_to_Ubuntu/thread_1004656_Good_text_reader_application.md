### Post by mgmiller on 2008-12-07
The one that's built in is called orca.
Go to System > Preferences > Assistive Technologies

Click to Enable Assistive technologies

Finally, open a terminal and type:
```
orca
```Give it a moment to get going and a gui will open up that should start talking to you.  You can also configure many aspects of the orca text to speech synthesizer.

If you find this useful, just add a menu item to the Applications menu so it's available with a mouse click.

Also, take a look at this thread, for other possibilities:
[http://ubuntuforums.org/showthread.php?t=462037](http://ubuntuforums.org/showthread.php?t=462037)

Have fun.
