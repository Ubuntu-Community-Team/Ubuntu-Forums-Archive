### Post by Bucky Ball on 2014-01-03
Welcome. To change the icon dimensions you can right click the desktop>Desktop Settings>Icons tab.

To change the icon sets themselves, Apps>Settings>Settings Manager>Appearance>Icons tab. You might find the one you're looking for in there.

Hope that helps. ;)

PS: You can add other themes and artwork from other buntu flavours. Edubuntu has some nice artwork. To install, Synaptic Package Manager and:

edubuntu-artwork
