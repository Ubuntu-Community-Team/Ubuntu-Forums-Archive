### Post by undecim on 2010-03-10
When you lock, it starts the screen saver. You can customize it (kinda) in System -> Preferences -> Screensaver.

If you want more control over it, there is an old howto here: [http://ubuntuforums.org/showthread.php?t=195557](http://ubuntuforums.org/showthread.php?t=195557). It's a few years old, but AFAIK it still works.
