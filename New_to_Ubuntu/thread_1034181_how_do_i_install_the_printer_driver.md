### Post by porchrat on 2009-01-08
I would recommend that you use CUPS.  To learn a little more about it check out the wikipedia article on it [here]("http://en.wikipedia.org/wiki/CUPS").

Here is a guide to get you started (please read it thouroughly and perform each step in the correct order).

[https://help.ubuntu.com/6.06/ubuntu/serverguide/C/cups.html]("https://help.ubuntu.com/6.06/ubuntu/serverguide/C/cups.html")

You will find that your model of printer is supported by CUPS (most are) and that you don't need the drivers on the CD that came with the printer.

If you have any other questions or problems please let us know, we are all here to help.

EDIT: Now that I look at Peter09's suggestion it does seem like the simplest place to start for you, give his a go first and if unsuccessful then use CUPS.
