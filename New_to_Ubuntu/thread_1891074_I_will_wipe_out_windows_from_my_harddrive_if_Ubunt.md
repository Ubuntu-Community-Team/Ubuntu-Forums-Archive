### Post by 3rdalbum on 2011-12-05
You definitely locked yourself to Windows with that Zune. Almost no Linux users bought them, which means that there's never been much incentive for developers to break the security on them. Better to buy an MP3 player that works drag 'n' drop on all operating systems. I like the Sony Walkman series - they've been drag 'n' drop on all operating systems for ages now.

Are you aware that you can try Ubuntu? Insert the Ubuntu CD and set your BIOS so it boots from CD before trying the hard disk. In a few minutes Ubuntu will boot and you can click "Try Ubuntu" and try it out risk-free from the CD.

For a longer-term trial you can insert the CD while Windows is running and you'll be given the option to install Ubuntu alongside Windows, inside a file. This isn't really appropriate for long-term use as it's dependent on Windows being able to boot and having space, but you can try it out quite well.

Finally, even if you find you're not ready to remove Windows, you can still dual-boot Windows and Ubuntu on the same computer. Defragment the hard disk first, then boot into the Ubuntu CD. Run the installer and follow the instructions for installing Ubuntu side-by-side with Windows. Once installed, you'll have the option on every boot about whether you want to boot into Ubuntu or into Windows.
