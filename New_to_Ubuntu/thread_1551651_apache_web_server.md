### Post by jamisonbryant1993 on 2010-08-12
While I think it would be good for you to try to figure this out on your own (given the links above), I'll give you a quick intro, so you're not completely unprepared.

When you go to [http://www.google.com](http://www.google.com), what you're doing is asking a web server for a file, usually called the *index file*, which gets rendered by your browser into a web page your can surf.

Apache is one such web server.  It takes requests for documents and returns them to the browser, or an error document if they do not exist.  An example of another web server is IIS, but almost every web site uses Apache because 1) it's free and 2) it's open source.

Let me know if you have any further questions. :)
