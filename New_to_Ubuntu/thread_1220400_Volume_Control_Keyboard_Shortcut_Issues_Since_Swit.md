### Post by AndThenWhat on 2009-07-22
You need to go to (System -> Preferences -> Keyboard Shortcuts) and you should see options for 'Volume Up' and 'Volume Down'. If you click in the shortcut column next to those it will wait for you to press the key you want to use for each.
