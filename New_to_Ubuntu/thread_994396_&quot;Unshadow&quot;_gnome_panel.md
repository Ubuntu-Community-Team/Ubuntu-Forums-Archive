### Post by Diabolis on 2008-11-26
That shadow is being drawn by compiz. You can modify compiz settings here:
**System / Preferences / CompizConfig Setting Manager**

If you don't have that entry in your menu, then you have to install the package:
```
sudo apt-get install compizconfig-settings-manager
```

I can't assure you that you will be able to delete the shadow because I have never done it, but thats where I would look.
