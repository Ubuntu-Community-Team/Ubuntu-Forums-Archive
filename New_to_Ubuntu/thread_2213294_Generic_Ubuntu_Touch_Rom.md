### Post by 3rdalbum on 2014-03-26
There is no "generic" ROM that will work on any tablet or phone. Unlike PCs, tablets and smart phones don't have an open architecture, so a generic ROM is not possible.

If nobody is working on an unofficial port, and you can't port Ubuntu to your device yourself, unfortunately you are a bit stuck.
