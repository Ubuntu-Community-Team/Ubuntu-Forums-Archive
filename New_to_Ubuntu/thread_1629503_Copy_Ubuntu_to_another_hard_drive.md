### Post by lmarmisa on 2010-11-23
You can copy the virtual disks of your virtual machines without problems. Create new VMs in your new computer and select the virtual disk files that you have copied.

A full Ubuntu copy is also possible but it requires some technical skills and the procedure is not trivial. Different graphics cards (and other peripherals) could be also a source of problems. Some tools like [Clonezilla]("http://www.clonezilla.org/") could help.
