### Post by acimmarusti on 2009-03-19
Ekiga only works with other ekiga members. Since your friend only has MSN messenger, the only thing you can try is getting amsn.

```

sudo apt-get install amsn

```

Once you get it, run it and configure your webcam. Do you already have a hotmail account? you need one to set up amsn.

Hope this works for ya
