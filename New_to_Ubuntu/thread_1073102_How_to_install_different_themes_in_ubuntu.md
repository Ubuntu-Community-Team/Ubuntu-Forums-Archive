### Post by cariboo on 2009-02-18
The easiest way is to download the theme, then right click on the desktop and select Change Desktop Background, then click the Theme tab, then click the install button, navigate to where you saved the theme, highlight it, and click Open. Nothing to it.

Edit: alright I got some solutions for you.
Firstly with the adena forest theme it seems to be a archive within a archive.
To make it usable you must unzip your 99528-Andena Forest.tar.gz by right clicking on it and selecting
"extract here"
Then with the theme manager make your way into the new folder you just extracted and select the .tar.gz file inside.

Second for the emerald theme, you need to have emerald installed, go up to system> administration and select synaptic package manager.
Once it loads search for emerald and install it, installing things in synaptic is very easy and streightforward.

For conky I cant help with as I dont use it but those other two things I know I can help.
