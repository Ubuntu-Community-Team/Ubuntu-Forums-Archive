### Post by Silent Warrior on 2010-04-16
The safest bet is to only install what's in the repositories - if there's no IDM there, installing it might work fine (sounds like Windows-software - remember to keep Wine around), or it might not work. (No risk of breaking your Ubuntu-installation, though.)
Downloading-speeds generally depend on a number of things. I get the impression that IDM will break downloads up into little chunks and download them through multiple connections (much like Bittorrent), and I know FOR SURE that several websites don't take kindly to that. Maybe it breaks their statistics or summat. *Shrug*
So, honestly, I think you should stick to DTA. I think you can set that to use multiple connections as well, if you really want to.
