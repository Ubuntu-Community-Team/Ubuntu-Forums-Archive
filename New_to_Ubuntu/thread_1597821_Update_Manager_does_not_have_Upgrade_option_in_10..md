### Post by bcbc on 2010-10-15
If you installed 10.04 which is a long term release it probably is set to only notify you when another LTS is released. Go into the Settings (at the bottom of update manager or under System, Administration, Software Sources) and look for that option.

PS I recommend backing up prior to upgrade.
