### Post by Kazade on 2011-05-17
WINE is a compatibility layer that allows Windows programs to run "natively" on Ubuntu, but it's still in development and not all programs run properly.

You can find out how well a Windows application works by searching the AppDB: [http://appdb.winehq.org/](http://appdb.winehq.org/)

MSN Messenger doesn't seem to work very well: [http://appdb.winehq.org/objectManager.php?sClass=application&iId=127](http://appdb.winehq.org/objectManager.php?sClass=application&iId=127)

However, if you just want to connect to MSN chat you can do so with Empathy which is Ubuntu's built in chat client.
