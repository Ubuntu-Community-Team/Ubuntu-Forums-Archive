Good evening everyone, 

Currently through compiz fusion I have Alt-left click set to select any window and move it while holding down the left mouse button. This is great so I don't have to go aiming for the title bars all of the time, but It is detrimental to the functioning of one program. When using Adobe Photoshop (through wine of course) Alt-left click with the zoom tool enabled for instance zooms out... but instead of zooming out, it instead does per its normal routine, and selects the window for movement. I was wondering if there was some way I could get it so it would disable this function ONLY for photoshop... I cant seem to find this functionality in the compiz fusion manager anywhere.

If anyone has any information it would be greatly appreciated.

Thank you all very much in advance.

### Post by CatKiller on 2009-10-12
Unchecking winecfg &#8594; Graphics &#8594; Allow the window manager to control the windows will probably have the effect that you're after, but it may have other consequences that I haven't currently considered.
