You can install a minimal (command-line) installation of Ubuntu, by following [this link]("https://help.ubuntu.com/community/Installation/SystemRequirements#Absolute%20minimum%20installation") then install a GUI such as IceWM, which is an extremely low resource window manager.

I wouldn't say that you'd be comfortable with the performance, but it will work...
