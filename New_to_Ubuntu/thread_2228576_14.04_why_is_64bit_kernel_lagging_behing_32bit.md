### Post by tgalati4 on 2014-06-08
You can research the [changelogs]("http://www.kernel.org") and see the differences.  It's possible that the 32-bit needed some extra love.  Or, many ARM processors (smart phones) run only 32-bit code and some things needed fixing.  Since 3.13 is not showing up in the main kernel source, I would presume it is an Ubuntu-maintained patch.  

I wouldn't worry about it.  Give it a few weeks.
