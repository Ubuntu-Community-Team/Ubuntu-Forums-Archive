### Post by dunkar70 on 2009-11-10
There are several options available. For a full server package, you can try ebox. ebox can be installed from the repositories on an existing installation or you can download an ebox ISO for a fully integrated server installation. [http://www.ebox-platform.com](http://www.ebox-platform.com).

A lighter weight option is webmin. Webmin works quite well, but you will need to download the tarball and install it manually. [http://www.webmin.com](http://www.webmin.com).

Either will work, but you will be better off if you can get used to using the CLI over time.
