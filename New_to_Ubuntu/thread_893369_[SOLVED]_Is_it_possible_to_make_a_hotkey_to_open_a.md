I have had the same problems before. It seems Gnome itself doesn't come with a utility to launch applications using shortcuts, and it has problems with the "Super" key. There are other utilities that you can install ... try searching for "keyboard shortcuts" in the package manager.

But the best solution I have found is to configure shortcuts in the window manager. I don't know about Metacity, but Compiz has very good support for keyboard shortcuts. Also Sawfish, that I used to run before Compiz came along. In fact keyboard shortcuts is the single most important feature on which I choose my window manager. Compiz has many plugins, and keyboard shortcuts are at its very core ... there are user-settable shortcuts for everything ... just puts your entire keyboard on steroids!
