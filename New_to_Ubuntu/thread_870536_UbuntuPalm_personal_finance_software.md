### Post by gobbles414 on 2008-07-28
Go into the *Synaptic Package Manager*, and search for a package called *jpilot-plugins*. Install that package and accept the automatic installation of the main *jpilot* package. Restart your computer, and you should then be able to synchronize your financial data between Ubuntu and your Treo. If you encounter problems syncing, you may need to enable the *Expense* conduit in Ubuntu's *PalmOS Devices* control panel (*System => Preferences => PalmOS Devices*).

I've never used J-Pilot, but it seems like a nice program. For more info, you can consult the J-Pilot website, which is located at [http://www.jpilot.org/index.html]("http://www.jpilot.org/index.html"). Does any of this help?
