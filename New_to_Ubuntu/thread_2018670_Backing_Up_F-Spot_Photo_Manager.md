### Post by sudodus on 2012-07-06
> **daniell59 said:**
> I have so many. I wanted to copy them all at once. I tried to copy then paste all of them, but it did not work.
Identify where the pictures are stored in your file system. My photos in Ubuntu 10.04 (imported with f-spot) are in

```
~/Photos
``` (/home/your-user-name/Photos)

and its subdirectories (year/month/day).

Check the size of the directory by right-clicking the icon in the file browser and select properties.

If space enough on the DVD, you can copy by dragging and dropping ***Photos*** from the file browser window to your DVD burning program for example Brasero. And then wait for the action to finish.
