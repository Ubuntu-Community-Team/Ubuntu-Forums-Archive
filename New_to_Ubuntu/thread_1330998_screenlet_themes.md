### Post by connorh123 on 2009-11-18
Of course :)

If you would, type screenlets into Synaptic Package Manager (System/Administration/Synaptic Package Manager).
Once you type it in, you'll see a box, unchecked. So then check it. It should say mark for installation. Click upgrade, and it should install. Once it's done, look for it under Applications/Accessories/Screenlets. And then you'll find a whole bunch of them you can use :)
