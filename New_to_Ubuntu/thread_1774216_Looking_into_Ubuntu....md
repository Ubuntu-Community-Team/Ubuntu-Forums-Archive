1.  Ubuntu doesnt run Windows programs, but you can use Wine to simulate a windows environment for a lot of things.  I would research individual games that you play because some work better than others in Wine.

2.  Im not a programmer, but I believe that it is in C.  Python is also heavily used.  Bash commands in the command line are different than command line commands in Windows.

3.  Using Ubuntu for the internet and email is simple.  Ubuntu uses icons and menus similar to Windows so that shouldnt be a problem.

4.  You wont be able to run Microsoft .exe programs.  Nearly every program you use on Windows wont work on Linux.  But you can usually find a suitable alternative.  Microsoft Word files and the like will usually work on open source word processors, but complex formatting sometimes isnt perfect.

5.  Standard Ubuntu isnt so lightweight, but some of the spin-off distros (Kubuntu, Lubuntu) are light and will work on a low spec pc.

exe's are Windows Executables, specific to Windows, no other OS on this earth will run them, just as Windows can't run a Linux binary or a Mac .dmg file.

That being said there is a project called wine, which runs on both Linux and Mac which can run Windows programs (like how I contradicted myself there) It's fairly good with the programs you listed in my experience (not sure about css but I play DoD and TFC occasionally)

SC2 runs at a noticeably reduced framerate, Wow runs perfectly, css you'll have to check WineHQ's appdb.


I would suggest as others have suggested, don't commit to just deleting Windows and running Ubuntu full force, especially if your a gamer. Try a dual boot or Wubi. Maybe you'll end up keeping Windows around just for the games that don't work who knows :)
