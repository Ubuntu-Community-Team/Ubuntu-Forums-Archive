### Post by cwsnyder on 2009-03-05
Usually the first thing the CD will ask is the language for the install.  Are you choosing the language properly?

The next thing it will ask is for you to pick the keyboard layout.

If these two questions are answered properly, and your language is supported, you should have few problems installing properly.  If the language is chosen improperly, or your language is not supported, you will have problems as you have reported.
