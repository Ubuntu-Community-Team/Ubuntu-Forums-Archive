### Post by mgranet on 2008-07-22
This code here: ```
Section "Screen"
    Identifier     "Default Screen"
    Device         "Configured Video Device"
    Monitor        "Configured Monitor"
    DefaultDepth    24
    Option         "NoLogo" "True"
    SubSection     "Display"
        Virtual     640 480
        Depth       24
        Modes      "640x480@60"
    EndSubSection

```
is odd. Your modes section only has the low resoultion mode listed. Try editing the "640x480@60" to be "YYYYxYYYY@RR" (YYYY being your desired resolution, and RR being your desired refresh). Also, I don't know what 'virtual 640 480' is. You might try commenting that out.

A few other things. You ran nvidia-xconfig right before posting your xorg. The problem with this is that nvidia-xconfig writes a new xorg config file. So you should only run this command as to reset if tinkering with your xorg manually doesn't work. Also, nvidia-settings needs to be ran as root (sudo) for any settings to take effect.
