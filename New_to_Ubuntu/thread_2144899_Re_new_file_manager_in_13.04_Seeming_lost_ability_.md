### Post by Dennis N on 2013-05-13
I haven't yet seen Ubuntu 13.04's default file manager, but in general, most file managers will create a new folder within the current folder with Shift+Ctrl+N. Try that. If you get some permission denied message, your permissions on the home folder have become incorrect.

I would be surprised there is no option for Create New Folder under the File Menu as there always has been in Nautilus?
