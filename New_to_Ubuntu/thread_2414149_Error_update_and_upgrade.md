### Post by SeijiSensei on 2019-03-06
Looks like lines 52 and 54 of /etc/apt/sources.list are duplicates.  Remove one.  You'll need to use sudo and a text editor.  For instance,

```
sudo nano /etc/apt/sources.list
```
