### Post by ZoiaGuyver on 2016-05-20
[Dropbox]("http://dropbox.com") is about the only one I know of that actually has a "native" Linux application (and is free), [Google Drive]("https://drive.google.com") can be accesed through nautilus (you need to set up your account with the Online Accounts section of system settings) but it seems to have some problems transferring large files (atleast thats what I have found). There is a program called [Insync]("https://www.insynchq.com/") that has a native sync application but it costs $25.

Hope this helps a little
