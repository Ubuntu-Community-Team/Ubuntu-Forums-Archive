### Post by Ghostcrawler on 2013-02-02
Yes, you don't need an AV when running linux. The only time this differs is if you're running a linux server, and sharing files with Windows, in which case an AV would be useful.

An AV on linux is rendered mostly useless since very few linux viruses exist. As long as you don't run commands you don't understand, and as long as you keep your software up to date, you should be safe on linux!
