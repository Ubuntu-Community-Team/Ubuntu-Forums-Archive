### Post by uRock on 2010-08-14
If you want to be able to see what programs are using what resources, then install htop within Ubuntu Software Center or running the command sudo apt-get install htop in a terminal. Then to run htop you just open a terminal and run the command htop. If you want to be able to kill processes, then you will have to start it using sudo htop.
