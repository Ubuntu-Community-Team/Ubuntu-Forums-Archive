### Post by Chronon on 2011-06-10
> **hanzj said:**
> With 5th gen ipod, don't you still need to use iTunes or floola? Rockbox can play stuff, but Rockbox won't be smart enough to understand files imported onto the iPod via simple drag-and-drop, would it?

In fact, Rockbox doesn't care at all about the iTunesDB.  It can create its own database by scanning the metadata of all media added to your player.  You can copy media to any path on your player and Rockbox can find and index it in its database.  If you use sensible directory structure you can also use the file browser to browse your music.

Many iPod users have decided to try Rockbox because you don't need to use iTunes to transfer music.

EDIT: 
Any player that supports MSC will allow the player to act as a UMS device.  If you want to drag-and-drop just stay away from MTP-only players.  Make sure the player's firmware supports MSC.
