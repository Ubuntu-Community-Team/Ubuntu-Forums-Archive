### Post by alpage2 on 2008-09-27
I run xubuntu on an old (192MB RAM) laptop.

Basically, xubuntu has the same access to the entire ubuntu software repository, so you can run a wide range of software. The main difference is the lightweight XFCE 4.0 window manager instead of gnome, but you can still run gnome-centric applications - when you install a gnome-related package, synaptic will automatically install any necessary gnome libraries. The same applies to KDE apps - the necessary KDE libraries will be automatically installed by your package manager.

I find the main limitations are more related to the available RAM and CPU speed.

Hope that helps.
