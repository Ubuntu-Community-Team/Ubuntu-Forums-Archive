### Post by masque7 on 2010-07-14
Assuming this is what you want to do: convert a digital video file into a DVD format, then burn it onto a DVD blank disc to watch on any home DVD player.

To convert the file, you can use either DeVeDe, or Handbrake. The former is found in the software centre. You would then use either the feature in DeVeDe to burn the DVD format files and directories it has created, or use Brasero (comes with ubuntu by default) to burn the DVD data on disc.
