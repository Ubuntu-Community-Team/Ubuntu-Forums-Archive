### Post by kwacka on 2008-05-29
Looking at the attachment its a java downloader for YouTube, unzip it and run it with ```
java -jar filename.jar
``` (you will need java installed of course).

Alternatively check synaptic - there's 3 or 4 apps in there that will do the same thing.
