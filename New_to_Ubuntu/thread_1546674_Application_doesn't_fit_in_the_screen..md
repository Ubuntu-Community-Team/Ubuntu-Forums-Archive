### Post by mcduck on 2010-08-05
> **FZZL said:**
> I'm using a netbook but im using the full ubuntu 10.04 not the remix.
The screen resolution is already at the highest but for some messed up reason i cant click on the buttons on the bottom of the application. is there anyway i can see all the way to the bottom. Ive tried auto-hiding the panels also.

Yes, there is.

Hold down the Alt key and you can move windows around from any point inside the windows (instead of just the titlebar like usually). This will allow you to move it around so that you can access those parts that wouldn't otherwise fit on your screen.

Also note that setting a smaller font size and perhaps using a more compact GTK theme will also save you some screen space.
