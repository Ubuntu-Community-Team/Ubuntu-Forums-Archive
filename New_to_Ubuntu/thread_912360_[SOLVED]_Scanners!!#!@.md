### Post by Zimmer on 2008-09-06
Open GIMP.

File>Acquire>xsane>device dialog should start you off (I have LIDE 30).

If you have a concern, check your scanner on the xsane web site for issues...
[http://www.sane-project.org/sane-mfgs.html#Z-CANON](http://www.sane-project.org/sane-mfgs.html#Z-CANON)

or go Graphics>Xsane Image Scanner which brings up the xsane dialog only.
