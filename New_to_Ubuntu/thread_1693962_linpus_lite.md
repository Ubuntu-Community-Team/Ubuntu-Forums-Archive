### Post by komputes on 2011-02-23
Sounds like you're average netbook. Usually Ubuntu runs pretty well on these types of laptops. If you are interested in installing ubuntu on this. I would recommend booring froma CD on any other computer and once botted into the CD select "Test/Try Ubuntu" once at the desktop, go to System > Administration > Startup Disk Creator. Select the CD as the source, plug in a USB stick and create a bootable stick.

At that point you should be able to boot your netbook from that USB stick. On a Dell Mini, you press 0 to get the boot menu. On an Asus eeepc it's usually ESC. If it doesn't tell you which key to press in the first few seconds after booting the netbook, do some research to find the appropriate key for your model. Once booted from the key you can install Ubuntu over Linpus (all linpus data will be lost).

Hope that helps.
