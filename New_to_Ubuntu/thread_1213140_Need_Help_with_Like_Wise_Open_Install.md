Well, you need to provide more info.
Actually you don't need to install it from  *root* exactly.

You need to :

sudo aptitude install likewise-open (for version4) or
sudo aptitude install likewise-open5 (for version 5)

optionally:
sudo aptitude install likewise-open-gui or
sudo aptitude install likewise-open5-gui
