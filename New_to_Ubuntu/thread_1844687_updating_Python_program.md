### Post by Cheesehead on 2011-09-15
ActiveState Python (or ActivePython) is a commercial product. You would need to purchase it from them.

The 2.6 version installed on your system is generally required for your system to function properly. I advise you to not tinker with that installed version, lest you break parts of your system.

You can install 3.x (but not another 2.x) alongside quite easily using the Software Center or other ordinary package management.
