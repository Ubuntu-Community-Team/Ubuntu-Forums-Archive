### Post by DarinB on 2010-10-06

but it now does not connect with rhythmbox so I will have to play with it each time. maybe leave it as before and only make the change when i want to add pictures or documents.

On Blackberry -
1. Options -> Memory

Media Card support: on
encryption mode: none
Media transfer protocol: on
mass storage mode support: on
Auto enable mass storage mode when connected: yes or prompt (and when you want to mount answer the prompt with 'yes'

<back> and "save"

now connect. It should auto-detect and auto-mount ('yes' if prompted on the mass storage mode)
