### Post by Sceiron on 2009-02-02
Assuming you are using ubuntu and have installed squeezecenter correctly you can start the app by running:

```

/etc/init.d/squeezecenter start 
```

Open your webbrowser and go to [http://youripadress:9000](http://youripadress:9000)
and there you have it

example: [http://192.168.1.2:9000](http://192.168.1.2:9000)

Edit:
However, there is a package for squeezecenter, but you have to add
```
deb http://debian.slimdevices.com stable main
```
to your sources.list, then you can run sudo apt-get install squeezecenter

### Post by nothingspecial on 2009-02-02
Squeezecentre is a web based app. You can launch it by typing in your browser```
http://ip.address.of.server:9000
``` or ```
http://localhost:9000
```

You can also use gnome-do ```
sudo apt-get install gnome-do
``` to control your squeezecenter and players. Look for the squeezecenter plugin under third party plugins.
