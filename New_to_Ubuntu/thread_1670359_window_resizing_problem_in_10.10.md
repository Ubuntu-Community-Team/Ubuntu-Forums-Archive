### Post by coffeecat on 2011-01-19
Hold the alt key down and then left-click on any part of the window (hold the mouse button down) and you will able to move the window above the top edge of the screen until the lower edge of the window is visible. Now use the mouse to resize from the bottom right of window so that the window will fit in your screen. You can then use alt+left-mouse-button to move the window down again so that you can see the top border.

This is a bug in thumbnailme. It simply shouldn't open in a window too big for the screen. I cannot find it in the repositories, so it must be a 3rd party app. Out of interest, where did you get it?
