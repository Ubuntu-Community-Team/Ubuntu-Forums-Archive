### Post by craig10x on 2013-08-21
Unfortunately, Adobe no longer makes new versions of flash for linux with one exception:  they have an agreement with Google to include the latest flash versions for Google Chrome's Pepperflash plug in (only comes with Chrome not Chromium so you need to get the genuine article from the Google Chrome site...just download the deb file, right click the file to have it open into the Ubuntu Software Center and install)...

The flash we get in ubuntu restricted extras is like about 2 years old and only gets security updates now...If you want to have updated flash in ubuntu (or any linux distro) you will have to run Chrome web browser...Firefox does not have it...
Installing that tar file you got from adobe for the linux flash will only give you the same outdated 11.2 version you already got when you downloaded ubuntu restricted extras...
