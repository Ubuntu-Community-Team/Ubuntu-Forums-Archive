### Post by IHeequ5i on 2011-04-09
Ubuntu has a torrent client already:  Transmission. It's under Applications / Internet.

To manually run a command:
Go to Applications / Accessories / Terminal
This is also called "launching a Terminal".

Put in the command you were asked to run. Remember that Linux is case-sensitive, so sudo and SUDO are two different things.

Hit the <enter> key and follow the prompts if any. You'll definitely be asked for your password (sudo does that). 

Best of luck!
