### Post by TuxImpersonator on 2009-02-05
Glad to hear you would like to try out ubuntu and yes you can try it without installing it onto your computer, the best way is to make a live CD: [download]("http://www.ubuntu.com/getubuntu/download") the latest version, burn it to a CD then restart with the CD in the computer.

Your computer should boot off the CD and after a bit of time give you a menu with the first option being to try ubuntu without installing anything to your computer (it runs off the CD)

If you want to use a USB stick there is a guide [here]("https://help.ubuntu.com/community/Installation/FromUSBStick")

Hope this helps

P.s. sorry I forgot to mention, if your computer doesn't load off the CD/USB when you try it your BIOS settings probably need changing (i.e. your computer likes to boot off it's hard drive rather than a CD). This isn't not hard to change and a quick search of the forums should give you a guide.

### Post by eragon100 on 2009-02-05
Both yes. However, the cd you download from the website is also meant to allow you to run the system without installing it, you can check out openoffice.org, browse the internet, look at the gimp and everything without chaning anything on your harddrive. The CD does run MUCH slower than an installed system, especially takes long to load (6 minutes or something), but it will be super fast after install and the performance is enough to test hardware compatibility. In addition, the default installation settings create a dual-boot, i.e. both systems will be installed and you can choose wether to boot ubuntu or windows when you turn on your computer.

If after reading this you still want a bootable ubuntu usb-stick, you can buy one from [www.zareason.com](www.zareason.com), but I think it's a waiste of time to do this just to check it out, because the cd can be used for that and, once you have installed ubuntu, a program is included that allows you to make a bootable usb stick which can even save settings and documents with two mouseclicks. As many sticks as you want, minimum recommended size is just 2 GB's.

Hope this info helps and good luck with ubuntu! :wink:
