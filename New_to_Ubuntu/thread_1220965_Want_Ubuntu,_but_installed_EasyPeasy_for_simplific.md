### Post by mikechant on 2009-07-23
Don't know much about EasyPeasy, but I can recommend UNR (Ubuntu Netbook Remix) 9.04 for eeePC use. I've got it dual booting (via Grub) on my eeePC 1000 and it works damn near perfectly. And the good thing is that it is an official version of Ubuntu so it's got an assured future. It's worth installing to the internal SSD if you can spare the space since it will run faster and free up your SD slot.
