### Post by deadflowr on 2012-09-20
Go to system setting > select appearance > click on the tab marked behavior > turn on the auto-hide feature, and set the reveal sensitivity bar to low.

This can also be achieved through the compiz-config-settings-manager application.
In ccsm click  Ubuntu Unity Plugin > change  hide launcher from never to autohide, and then switch the tab from behavior to experimental and set the launcher reponsiveness.

You would want to set the launchers pressure to be to the point that a train could ram the pointer into the left side and it still wouldn't reveal.

Please note, though, that if you use the super(windows)key to call the unity dash, the launcher will be revealed.
