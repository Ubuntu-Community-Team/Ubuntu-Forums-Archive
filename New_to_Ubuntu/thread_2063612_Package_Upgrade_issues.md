### Post by cogier on 2012-09-27
Welcome to the Forum.

I suggest you try the following: -

Start Terminal by pressing **[Ctrl]+[Alt]+T**

Type **sudo apt-get install -f** 

You will be asked for your password which is the one you login with and/or the one you typed in during installation. You will not see anything happening as you type but if you type your password correctly and press [Enter] the command will try and sort out your problem.

You could then enter the following: -
**sudo apt-get update**

and after that: -

**sudo apt-get upgrade** 

If you still get errors reply to this post.

Hope that helps.
