1. Does using the "grandr" app helps? It's in the software center.

3. Use rythmbox instead, search the tool you can use.

Unity is designed to only have "status icons" on the bar. So no notifications, no minimised apps or anything else. Adding custom things to the panel is against the philosophy.

TLDR: Just tweak the installation so that you like it.
