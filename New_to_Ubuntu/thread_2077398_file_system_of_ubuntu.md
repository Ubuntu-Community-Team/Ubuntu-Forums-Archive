### Post by deadflowr on 2012-10-28
By default you should be using ext4 on Ubuntu(Ubuntu and linux doesn't play so well with ntfs).
To check open up the dash(windows key, superkey) and type disk utility then when it opens find your hard drive listed in the left side and then in the main window click the partition  you think is Ubuntu and read the information provided.
