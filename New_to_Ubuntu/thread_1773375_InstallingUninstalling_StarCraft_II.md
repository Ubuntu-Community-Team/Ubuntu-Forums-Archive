### Post by frankbooth on 2011-06-02
If you're using Gnome:
[img]http://www.liberiangeek.net/wp-content/uploads/2010/07/wine_ubuntu_lucid_15_thumb.png[/img]

If you use Unity you could search for application in the dash,

or launch a terminal and write:
```

wine uninstaller

```
This will bring up an Add/Remove Programs window (similar to the one that exists in Windows). 

Keep in mind that registry files probably don't get removed, so if you need to clean up something there you'll have to launch 'regedit' and do it manually.

### Post by LowSky on 2011-06-02
Go to ubuntu software center.

find and install **playonlinux**

then open playonlinux, then search for starcraft 2, then install

you can also use plaonlinux to remove software installed, or use wine's uninstaller... all pretty simple stuff.
