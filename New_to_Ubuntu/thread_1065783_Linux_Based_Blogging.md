### Post by Cypher on 2009-02-10
If your intention is to start a new Blog based on a Linux machine, then realize that the blogging software seldom care what the OS underneath is. They care more about their programming language avaiable. Most blogging software use PHP which is available in both Linux and Windows. There are also Python based blogs and so on.

There's also ASP/ASP.NET based blogging software that work natively on Windows and can be made to work in Linux through Mono.

If you want to use something like Wordpress to host your own blog, then Linux is perfectly suited for that with Apache, PHP and MySQL. A lot of cheap webhosts online use this scheme..
