### Post by EoflaOE on 2022-01-24
Hello, Innernet!

As you can see in the attached picture, there are a lot of "Software" icons. We all know that, without the expansion of the texts, it can be confusing. However, each one of them have different purpose.

- **Livepatch**: This is a shortcut to the Livepatch tab found in Software & Updates application. This app allows you to configure your machine to get live kernel updates without the need for rebooting your system.
- **Software**: This application comes from GNOME Software, which is an app store for Ubuntu that allows you to download and install software easily. This comes from [COLOR=#ee82ee]gnome-software[/COLOR].
- **Software & Updates**: This app allows you to configure the software sources and system updates, including the addition of the software repositories (PPAs for example). This comes from [COLOR=#ee82ee]software-properties-gtk.[/COLOR]
- **Software Boutique**: This is another app store made by MATE, a desktop environment. It does the same as Software. This comes from [COLOR=#ee82ee]software-boutique[/COLOR] snap.
- **Software Updater**: Allows you to check your system for updates and offers to install all available updates. It also lets you upgrade your Ubuntu system to the next supported version (if any). This comes from [COLOR=#ee82ee]update-manager[/COLOR].

If you don't get it, let me know.
