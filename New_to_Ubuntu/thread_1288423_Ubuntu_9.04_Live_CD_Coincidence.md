### Post by 2Bunoo on 2009-10-11
An Ubuntu newbie, I have been runing Windows XP on a Dell Dimension 4600C for several years.  Last evening I decided to run Ubuntu 9.04 Live CD on my XP machine.  I went on a little Ubuntu adventure.  Everything worked fine, and when I felt content I shut down the system.  After a few minutes, I booted into Windows XP and my NIC no longer functioned.  I tested the Cat 5 cable for life and it was good.  I then accessed the NIC properties...everything was reported working properly.  Using a lifeline, I phoned a friend.  He suggested reinserting the Live CD to see if the NIC would work for Ubuntu.  Sure enough, it worked!  He then suggested going to the manufacturer's site and downoad the latest driver for the NIC.  After uninstalling the old driver and installing the new driver, the NIC works!  I am leaving details about the NIC below.
 
Broadcom 4401 10/100 Integrated Controller
 
Old Driver Version: 3.51 A01
 
New Driver Version: 3.63 A03
