### Post by pytheas22 on 2008-06-22
If you want to install Ubuntu within Windows, try using [wubi]("http://wubi-installer.org/") instead of what comes on the live CD.  It will probably work better.

You could also install Ubuntu to its own partition by shrinking your Windows partition, which will give you slightly better performance than the "install inside Windows" approach.
