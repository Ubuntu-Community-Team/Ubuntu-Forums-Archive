### Post by Cheesemill on 2012-12-26
In that case you've managed to somehow close all of the docks as pierceTN assumed.

Go to Windows > Dockable Dialogs and select the ones you want to see.

If you are unsure which ones you want you could always delete your ~/.gimp-2.8 folder to return Gimp to its default settings.

Did you by any chance close all of the floating dialog boxes before switching to single window mode? If so then this is why they are all closed.
