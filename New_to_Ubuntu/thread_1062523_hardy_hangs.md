### Post by HereInOz on 2009-02-07
To test whether it is the new kernel or not, boot up the computer to the point where GRUB starts to load and where it says to press ESC for options.  

Press ESC and you will then see all your installed kernels and you can select which one you use to boot from.  

If you boot from an earlier kernel, and Hardy behaves itself, then it is clearly the new kernel.
