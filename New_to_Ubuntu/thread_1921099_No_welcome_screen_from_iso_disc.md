### Post by lolpenguin on 2012-02-06
You normally don't get to the Windows Boot Manager when you boot from a LiveCD, so I suggest you open your BIOS's boot menu (Del, Esc, F8, F12?) and select Disk Drive as you first boot option (temporarily) and you should get a welcome screen.

If not then login as "ubuntu" and no password and open "Install Ubuntu xx.xx" . It should be on the desktop or in the launcher.
