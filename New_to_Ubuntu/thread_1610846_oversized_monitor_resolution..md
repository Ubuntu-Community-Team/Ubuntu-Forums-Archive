### Post by sky_ceiling on 2010-11-01
There is such an effect in compiz.
 
go to Ubuntu Software Center, and install Simple CCSM
 
Then go to system > preferences > simple CCSM
 
At the end tab it will allow you to set accessibility options, such as the zoom mode you are looking for
