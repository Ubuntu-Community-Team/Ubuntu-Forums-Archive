This is almost certainly the problem. Alpha versions are ***unstable***. They usually change daily (if not hourly). You read about filing a bug report [here]("https://help.ubuntu.com/community/ReportingBugs"). I advise you to only upgrade to stable "normal" releases in the release cycle unless you like finding and reporting bugs or helping out with the development of the next Ubuntu cycle.
