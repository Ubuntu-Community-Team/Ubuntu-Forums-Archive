### Post by hg21 on 2009-06-15
> **celticbhoy said:**
> I have a directory with roughly 30 text files & want to print them all. Is there an easy way to do this rather than open each one and print separately?

If they are genuinely txt files ( ie ASCII) then, in the directory, lpr * should work. 

If however they are files produced in a word processor (eg open office) you probably need to print them from the application itself.  Most word processors I've used, seem to only be set up to print the open file.
