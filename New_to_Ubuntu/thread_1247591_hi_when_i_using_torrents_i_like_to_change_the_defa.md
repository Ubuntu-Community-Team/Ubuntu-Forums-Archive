### Post by HiImTye on 2009-08-23
if you mean in firefox, go to 'other' or 'browse' depending on the section you're in. find the path:

[LIST=1]
[*]right click on Gnome Panel's menu
[*]select 'Edit Menus'
[*]find 'Vuze' or 'Azureus' or whatever it is called in your menu
[*]take note of the 'command' - you only need the first part (everything before the first spacebar)
[*]go to places > search for files
[*]enter the 'command'
[*]select the drop-down for 'look in folder'
[*]choose 'File System'
[*]click 'find'
[*]right click it and choose 'open folder'
[*]click the button to the left of the path (looks like a notepad and a pencil)
[*]click at the end of the path so you begin typing after the path
[*]add a / to it and type in the command
[*]triple click the path so that you select all of the text
[*]paste that into the command box for firefox
[/LIST]

### Post by freak42 on 2009-08-23
i assume you talk about opening .torrent files in the file system 'explorer' (nautilus):

you can try right clicking on your .torrent file -> open with -> open with other application and then select your preffered application.

If you are talking about the application option upon downloading in firefox:
try in firefox: edit -> preferences -> applications -> search for the torrent file type -> on the drop down menu on the right side (action)  click "use other" and select your preferred application
