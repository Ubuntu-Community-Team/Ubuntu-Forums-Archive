Well do keep in mind that 3.5 is an optional upgrade right now.
Also if you did install 3.5 its actually listed as shiretoko under applications> internet, shiretoko is the coede name of firefox 3.5.
This has a lot to do with the Ubuntu upgrade policy, instead of trying for "latest and greatest" Ubuntu aims for stability even considering using Jaunty the latest version.
If 3.5 was a major security update then it would have been made to overwrite 3.0.12 but since its not Firefox 3.5 is an optional upgrade.
