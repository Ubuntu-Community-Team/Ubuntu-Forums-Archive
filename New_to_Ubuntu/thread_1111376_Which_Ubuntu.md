### Post by kanikilu on 2009-03-30
You can read up on what each distro is in more detail, but the primary difference is the desktop environment for each. Ubuntu uses Gnome, Kubuntu uses KDE, and Xubuntu uses XFCE. Generally speaking, Xubuntu is recommended for lower-end hardware. However, Ubuntu or Kubuntu should work ok with your hardware, though, you'll probably just want to turn off any desktop effects and things like that.

For an even lighter install, you could use a lighter desktop environment like [LXDE](http://www.ubuntugeek.com/lxde-lightweight-x11-desktop-environment-for-ubuntu.html). I haven't used it yet myself, though, so I can't really give much of a recommendation beyond what I've read about it...
