### Post by oldfred on 2013-08-09
You must have a new system as Ubuntu is installed in UEFI mode with an efi partition.

There is no trace of Windows or any other system anywhere. One of of the auto install options is to erase entire drive and install Ubuntu. In [COLOR=#ff0000]RED[/COLOR] it clearly states all your data will be erased.

If you want to preserve existing systems you have to create unallocated space for Ubuntu, manually install with Something else or (not recommended) use installer to resize an existing partition.

If you had data that was vital, the amount of space overwritten by Ubuntu is relatively small. But whatever it overwrote is gone and all the file structure data is gone. You can use tools like photorec to scan a drive for anything that looks like a file and it will copy to another drive. You need lots of room on other drive and then have to sort thru lots of files with just extensions. If music or photos files has metadata you can use to rename. Everything else you have to figure out what it was and rename.

       Testdisk & photorec
[http://www.cgsecurity.org/wiki/PhotoRec](http://www.cgsecurity.org/wiki/PhotoRec)
[http://www.psychocats.net/ubuntucat/recoverdeletedfiles/](http://www.psychocats.net/ubuntucat/recoverdeletedfiles/)
[http://ubuntu4beginners.blogspot.com/2011/01/recover-deleted-data-using-photorec-in.html](http://ubuntu4beginners.blogspot.com/2011/01/recover-deleted-data-using-photorec-in.html)

It also looks like recovery partition is gone. I assume you did not make a backup of recovery partition to reinstall with or a full backup of Windows which we always suggest when making major system reconfiguration. If system was purchased with Windows you may be able to get a new recovery set of DVDs from vendor at a small cost. Otherwise you can buy a new full copy of Windows.
