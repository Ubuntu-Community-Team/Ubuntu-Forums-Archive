### Post by Vladlenin5000 on 2014-10-18
Hi, welcome.

This should get you going: [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)
Please pay special attention to the first part, "Verifying the ISO integrity" because a corrupt file will never work, regardless of the software or OS. It may end up successfully burnt but it won't work.
There are many other programs you can use in each of the OSs discussed in that article but the same principles should be applied. If you need specific help please tell us what ISO are you trying to burn and with which software.
