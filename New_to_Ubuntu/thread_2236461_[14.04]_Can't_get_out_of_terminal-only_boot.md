### Post by coldcritter64 on 2014-07-27
After resetting the "GRUB_CMDLINE_LINUX=" line to  GRUB_CMDLINE_LINUX="", ensure you run the command,
```
sudo update-grub
```
to make the changes permanent. Then when you reboot things should be back to normal. Failing to run that command is _usually_ the reason for the failures.
