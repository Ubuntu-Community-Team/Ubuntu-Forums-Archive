### Post by v3.xx on 2015-06-11
Its asking you to do:
```
sudo apt-get install dkms && sudo /etc/init.d/vboxdrv setup
```
Is that the question?

---

### Post by Quarkrad on 2015-06-11
That did it - thank you very much, all up and running.

---

### Post by v3.xx on 2015-06-11
You may not be done :)

Same thing can happen to guest when updated.  Install dkms to your guest.
```
sudo apt-get install virtualbox-guest-dkms
```
This must be done inside the guest OS.
