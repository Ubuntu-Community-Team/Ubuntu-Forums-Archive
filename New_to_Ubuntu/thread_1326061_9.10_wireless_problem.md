### Post by fadermartin on 2009-11-14
It is very likely that you can fix the problem in 30 seconds. If so, there is no reason to downgrade. Open a terminal, and type
```
gksudo gedit /etc/modprobe.d/blacklist.conf
```Add the following line at the end of the file
```
blacklist rt2800usb
```Save and restart.
