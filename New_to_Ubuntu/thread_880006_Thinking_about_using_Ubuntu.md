### Post by tamoneya on 2008-08-04
when you download ubuntu you get one iso file.  You take this file and burn it to a cd.  This file does not alter your current OS at all so it doesnt matter which account you use.  Then you reboot your computer with the burned CD in the drive and ubuntu will launch.  Ubuntu has a nice feature that allows you to run the entire OS from the cd.  It is significantly slower this way since the CD is slower than a harddrive but it allows you to get a feel for the OS and allows you to see if your hardware is supported.  Running it like this has no effect on your vista install and if you remove the CD vista will work like normal.

### Post by Darkade on 2008-08-04
If you download and install Ubuntu you will be able to install it via Wubi, that allows you to install it as if was a windows app, however the performance won't as good as in a native install. If you DO install it with wubi you can uninstall it anytime you want just as a windows app.

If you choose to do a full install Ubuntu will partition your Hard Drive for you, allowing you to boot both Ubuntu and Windows. Get rid of Ubuntu ain't a problem either.

So, you don't have to worry you can always have windows back.

### Post by csimons on 2008-08-04
If you want to use Ubuntu without installing it as a bootable operating system on your hard disk, you can check out Sun's VirtualBox.  It is a free piece of software that will allow you to run Ubuntu over Windows (or vice-versa) on a "virtual machine."

Basically, you create a file on your hard disk that acts as the complete hard disk for the Ubuntu installation.

[http://www.virtualbox.org/](http://www.virtualbox.org/)

Or, if this seems to be too much hassle, you can burn a "Live CD."  If you burn a Live CD, you can simply put in CD in your CD-ROM drive and reboot your computer to boot into Linux.  Then, to go back to Windows, simply eject the CD and reboot again.

Using the Live CD will not modify your existing hard disk in any way whatsoever, so there is nothing to fear!  Later, if you decide you want to install Ubuntu onto your hard disk, there is a shortcut on the Linux desktop that will walk you through it.

[https://help.ubuntu.com/community/LiveCD](https://help.ubuntu.com/community/LiveCD)

Good luck,


Christopher Simons
