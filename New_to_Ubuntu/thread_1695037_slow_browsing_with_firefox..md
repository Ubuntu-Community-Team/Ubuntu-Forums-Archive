### Post by seawolf167 on 2011-02-25
You can try clearing the cache, cookies, etc. by going to Tools -> Clear Recent History, then select Everything for the time range.

Alternatively, you could also try a different browser like Google Chrome

```
sudo apt-get install chromium-browser
```

or a lightweight browser like 

```
sudo apt-get install midori
```

but I suspect that clearing the stored data in Firefox will help
