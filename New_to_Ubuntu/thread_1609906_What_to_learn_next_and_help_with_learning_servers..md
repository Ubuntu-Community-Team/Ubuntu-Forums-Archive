### Post by papibe on 2010-10-31
Hey Mobidoy.

I think a simple first project would be a file server. Using share folders, you all can share docs, music, movies, etc. Also, you can create a personal folder for every member of the family (password protected if you want), so they can upload files, backup their laptop/desktops, etc.

In general you won't need to have big specs for a home server. You can start as low as a Pentium 4 or even Celeron. For RAM, 512Mb will work just fine (maybe 1Gb could be better). The only thing I would recommend is having a huge hard drive (preferably sata). You neither need a permanent monitor, mouse, or keyboard (just for the installation, then you can admin/configure using ssh).

The install CD uses text mode, but it is very neat. I would recommend checking the installation tutorial videos at the [Ubuntu Screencasts site]("http://screencasts.ubuntu.com/"). Also there's several good videos on youtube. Search for both server install and alternate installation.

I have a server at home, and I'm finding more and more uses for it every day. I really think is very useful.

Good Luck with your projects, and come here often you share your results.
