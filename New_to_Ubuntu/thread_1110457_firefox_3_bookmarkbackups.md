### Post by lhb1142 on 2009-03-29
:KS I have found using the .json file to be a much better backup method than using the HTML backup file.

Every time I add or remove a bookmark, I go into < Bookmarks->Organize Bookmarks->Import and Backup->Backup > and then save the resultant .json file to a folder in Documents I have labeled "His Firefox Backups."

Every couple of days, I back up my entire Documents folder to two external hard drives.

Therefore I always have my bookmarks saved and I can restore them or even put them on another computer running Firefox at any time.

I do the same for my wife's computer using the name "Her Firefox Backups" for the folder in Documents.

When we go on vacation and she's using our portable computer, I put her bookmarks on it; when I take over, I install my bookmarks.

The .json file installs the bookmarks almost instantly. I think it is a real improvement over the backup method used in Firefox 2. :)
