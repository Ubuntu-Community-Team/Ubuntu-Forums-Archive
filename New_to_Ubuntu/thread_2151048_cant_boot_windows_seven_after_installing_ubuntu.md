### Post by Mark Phelps on 2013-06-03
Did you use the "slider" to shrink Win7 to make room for Ubuntu and then install side-by-side? IF so, that's probably what corrupted the boot loader.  Did you make the Win7 Repair CD before you did this install -- so you have a way to repair the boot loader, now?

If boot-repair doesn't fix it, you will need to be prepared to spend some money to buy a Win7 Repair CD image, download that to a PC, and burn it to a CD:  [http://neosmart.net/blog/2009/windows-7-system-repair-discs/]("http://neosmart.net/blog/2009/windows-7-system-repair-discs/")
