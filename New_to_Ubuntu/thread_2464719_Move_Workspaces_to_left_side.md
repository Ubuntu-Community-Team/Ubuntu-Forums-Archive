### Post by monkeybrain20122 on 2021-07-09
I assume you are using gnome shell?

Install gnome-tweak-tool, disable the Ubuntu gnome dock from there, then install the [dash to dock extension]("https://extensions.gnome.org/extension/307/dash-to-dock/") and configure it til you're happy.

### Post by tea for one on 2021-07-10
I'm sure that you cannot easily move the Workspaces to the left side, but you can move the dock back to the left or to the bottom of the screen.

[https://extensions.gnome.org/extension/307/dash-to-dock/](https://extensions.gnome.org/extension/307/dash-to-dock/) as suggested by [COLOR="#0000FF"]monkeybrain20122[/COLOR] is a versatile option.

[https://extensions.gnome.org/extension/1160/dash-to-panel/](https://extensions.gnome.org/extension/1160/dash-to-panel/) is also worth a look.
