### Post by tom.swartz07 on 2009-12-17
Hey-o

Ive had no problems with broadcom wifi cards.

As long as the maker of the computer is fairly mainstream, Dell, Asus, etc. They'll probably use good quality parts.

If youre really worried about compatibility and have the opportunity to physically use the machine before you buy it, I would definitely recommend just taking a LiveUSB ubuntu stick with you and boot into it on the machine. That will give you with 100% certainty what works and what doesnt.

As far as other parts to watch out for in a computer, I would say to stay away from ATI graphics cards- they are notorious for not supporting the cards on anything other than Windows for more than a year or 2. 

Hope this helps!

### Post by anagor on 2009-12-17
I think you would want to look [here]("https://help.ubuntu.com/community/WifiDocs/WirelessCardsSupported"), there is a list of wireless cards/adapters by manufacturer.
If you know what adapter is used in that particular laptop model, you will easily find it there.

Also, almost any laptop with Intel processor uses Intel's chipset, which comes with Intel's wireless adapters, which in turn are all supported out of the box :)

If you can access the exact laptop model you will be buying before you buy it, say in a store, then of course you could just load an Ubuntu live cd and check if it detected the card. It will take 2 minutes, and no harm will be done to the laptop itself. (but you probably knew it already :) )

Any CD with Ubuntu is Live CD.

So if you have CDs from ubuntu, they are live :)
