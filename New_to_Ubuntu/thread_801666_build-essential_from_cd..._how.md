### Post by jimrz on 2008-05-20
> **dagamer said:**
> how can i install the build-essential from the cd and not the internet?

Pop the cd in your cdrom drive, when nautilus opens go to the cdrom drive > pool > main > b > build-essential and install the .deb contained in the "build-essential" folder by double clicking it (system will use GDebi for the install).
