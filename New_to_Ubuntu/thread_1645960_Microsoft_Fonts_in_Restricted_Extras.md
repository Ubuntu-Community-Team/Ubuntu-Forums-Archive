### Post by Ahunt on 2010-12-15
Ubuntu generally comes with free fonts, like the GPL- licenced [FreeSans, FreeSerif and FreeMono series]("http://en.wikipedia.org/wiki/Free_UCS_Outline_Fonts") and also the ["Liberation" series]("http://en.wikipedia.org/wiki/Liberation_fonts"). While setting up my new Lubuntu 10.10 installation the Microsoft Non Free Fonts (package msttcorefonts) in the "restricted extras" package didn't install right and so I decided to just do without them and so I have been using the above free fonts and actually I like them better overall. But still there remains this nagging question - are the Microsoft non-free fonts legal to use. I haven't found a clear answer.

The Ubuntu Software Centre indicates that the Restricted Extras "are restricted by copyright or legal issues in some countries. See [http://www.ubuntu.com/ubuntu/licensing]("http://www.ubuntu.com/ubuntu/licensing")". That URL rediects to [http://www.ubuntu.com/project/about-ubuntu]("http://www.ubuntu.com/project/about-ubuntu") which has no information on the subject. Doing a search of the Ubuntu website I did find that in the Ubuntu community documentation there is [a page]("https://help.ubuntu.com/community/UbuntuEyeCandy") which actually encourages people to install the Microsoft fonts: "To easily install Microsoft fonts and others, consider using the repository. For fonts that are commonly distributed on Windows computers, install the package msttcorefonts using Synaptic or apt-get. For more fonts, browse Synaptic's packages, scrolling down to the packages beginning with ttf-." However the [official Ubuntu documentation has a page]("https://help.ubuntu.com/10.04/add-applications/C/restricted-software.html") which says: "Sometimes it is difficult to determine why a particular software package is restricted. If this is the case, seek advice on the Ubuntu Forums. If you are still unable to determine why a package is restricted, err on the side of caution and do not install it; it may be illegal to use it in your country." Which isn't much help at all.

I have been trying to get to the bottom of the legal restrictions on these fonts, but even after a few days of sifting through a lot of material on the web I don't have a clear answer. Since Canonical says to ask on the Ubuntu Forums I am doing that here.

Here is what I have found out:

1. As one example: [Times New Roman]("http://en.wikipedia.org/wiki/Times_Roman") was actually designed for The Times of London Newspaper in 1931! The font design was sold by The Times to The Monotype Corporation and Microsoft bought a licence for it. I have been through The Monotype Corporation's website and they will offer to sell you fonts, but the licensing is not clear. 

2. [Arial]("http://en.wikipedia.org/wiki/Arial") is a similar situation, designed in 1982 by Monotype Typography, Microsoft had a hand in the design work and also has a licence for it.

3. [This page]("http://www.ascenderfonts.com/info/trademarks.aspx") tells you who owns the non-free fonts and confirms that both Arial and Times New Roman belong to Monotype Typography.

4. [This page]("http://www.ascenderfonts.com/info/font-licensing.aspx") tells you the licensing, sort of. Basically it says each font producer or even each font has its own licencing terms. For instance it says "The following is a brief overview on font licenses: Permitted Uses - You are permitted to install the font onto the number of personal computers for which you have purchased licenses. Most fonts on this website permit up to 5 PCs as part of the base license."

5. In [This blog]("http://embraceubuntu.com/2005/09/09/installing-microsoft-fonts/") (admittedly not at all an official source) one responding poster says: "msttcorefonts has an amusing history; Microsoft licensed the fonts for anyone to freely use (regardless of OS) to help boost the market share of IE. When they won the first browser wars, they removed the files from their site… but the license says you can freely redistribute them so that’s how we can legally use them". I have no idea if that is right or not.

I haven't been able to find out much else.

Does anyone else know whether these are legal to install and use?
