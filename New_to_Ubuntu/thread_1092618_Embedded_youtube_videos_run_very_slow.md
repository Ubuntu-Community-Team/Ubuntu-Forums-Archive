### Post by kansasnoob on 2009-03-10
I've found that adding Flashblock helps a lot. What it does is replace all of the flash objects on a web page with a "button".

That frees up a lot of "puter-power" for actually playing the wanted content.

Now, the repo version of Flashblock is old, so in Firefox just go to Tools > Add-ons > Get Add-ons and search for and install Flashblock 1.5.8.

You'll have to restart Firefox afterwards.
