### Post by diablo75 on 2009-03-15
In Skype, try unchecking the box that says "Allow Skype to automatically adjust my mixer levels"

The other thing you'll want to be sure of is that the sound in device (in this very same Skype config window) is correct.  It might take a little trial and error to find the correct device.  So you'll want to open up your mixer via the sound icon at the top (double-click)... You might click Preferences in there to open up a list of possible additional objects that you can adjust for (check off everything if you want).  Then just be sure that you have your capture for the mic enabled, unmuted, perhaps the 20db boost if available.

Then do that skype test call thing to make sure it's working.
