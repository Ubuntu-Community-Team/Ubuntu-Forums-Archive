### Post by curiousnoob on 2009-01-17
I am trying to set it up so that if I double click on an icon on my desktop it will take me to a certain website.  I know it is an easy thing to do but I can't remember how.

---

### Post by Dirjel on 2009-01-17
Create a launcher using the command [your web browser] [the website you want to go to].

For example:
```
firefox www.google.com
```

### Post by wizard10000 on 2009-01-17

### Post by kukibird1 on 2009-01-17
You can drag the icon in the left corner of the location bar in Firefox 
to the desktop. It will create a desktop shortcut for the url you have entered.
