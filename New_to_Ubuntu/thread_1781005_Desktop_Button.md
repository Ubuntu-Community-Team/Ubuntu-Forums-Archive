### Post by nzjethro on 2011-06-12
Not sure if it will work in Natty, but if you have Compiz installed you can set a mouse movement to the corner to show the desktop by running
```
sudo gconf-editor
```
Then apps > compiz > general > allscreens > options then set showdesktop to your desired corner (I have BottomLeft).
