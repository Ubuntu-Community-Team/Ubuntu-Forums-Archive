### Post by sikander3786 on 2010-09-09
.deb packages can be downloaded from

[http://packages.ubuntu.com](http://packages.ubuntu.com)

But offline installation on Linux is not that much easy as on Windows, because there are lots of dependencies and it is not possible to download all of them. Even dependencies have dependencies. Isn't it problematic?

If you have got an Ubuntu PC somewhere with internet access, you can install the software on it from the repositories and then move it to your home PC using [APTonCD]("http://aptoncd.sourceforge.net/"). No easier method in my mind.
