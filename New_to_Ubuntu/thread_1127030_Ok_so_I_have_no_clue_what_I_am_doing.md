### Post by pbpersson on 2009-04-16
If you downloaded the ISO file for the Live CD, you would go into your CD burning software on Windows and you would say "burn CD from Image" and it would ask you where your ISO file is.

After you have the CD burned, boot from the CD and see if you can run Ubuntu from the CD and if everything works.  Pay close attention to:

Sound - maybe play a music CD
Web - surf some sites and see how they look
Graphics - are you happy with the screen resolution

Then if you are happy with Ubuntu, check back here if you want installation instructions.  Some people click the wrong choice, it blows away their hard drive with all their data, and they are quite unhappy.
