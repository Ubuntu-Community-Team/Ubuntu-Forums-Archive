### Post by petervk on 2009-02-04
Yes.

Just install the edubuntu-desktop package and it will pull everything in.

It is a little messy though as you get the full set of applications from both, and you can choose to boot into KDE or Gnome. If you only need a few of the applications it may be better to install them individually.
