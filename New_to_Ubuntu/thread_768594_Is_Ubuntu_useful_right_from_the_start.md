I would also recommend running the live cd (booting from Ubuntu install cd).

If you're using a **wired** internet connection you will be able to connect & go right away. If you're using a wireless connecting theres a 50/50 chance you might need to fiddle.

For me, Ubuntu definitely needed customisation from the start. But all the essentials are already there (OpenOffice, Firefox-internet etc.)
