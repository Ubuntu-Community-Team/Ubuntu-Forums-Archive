### Post by anewguy on 2008-04-27
I can't believe it - I don't know if Wine has changed or something else has changed, but I installed Family Tree Maker using Wine and it actually appears to be running okay in Hardy!!  I've never had it appear to work right before, so this is REALLY good news for me - no more Windows!!

PLEASE - if you use Family Tree Maker and have updated to Hard (8.04) please try installing Wine and then use Wine to install your Family Tree Maker and see if it appears to work for you, also!

If this holds up, it will be a big plus for Ubuntu!  So far, there really is no program remotely like Family Tree Maker for Linux, and it had been the reason I had a dual-boot with Windows.  Those days may be over!

Thanks to everyone!!   :)

EDIT:  Please be aware that it takes quite a while on my machine before you actually know it is starting up.  Also, on shutdown when it automatically saves the file, you may get a not responding box - just ignore it as it will go away as soon as the file is finished being saved and the program exits.
       Also might be worth noting that I have upgraded my video card since the last attempt and turned on desktop effects and yet everything in FTM seems to be working great!  No more missing text, missing boxes, etc., for me!  I am still trying to figure out printing - it sees my printer but won't let me select it as I have no driver installed.

What VERSION of FTM are you using and HOW MANY individuals are linked therein?

I've used FTM ever since the msDOS days, some of their upgrades caused massive data destruction that had to be rebuilt by hand.  So I'm VERY SLOW on even looking at newer bloated versions.
