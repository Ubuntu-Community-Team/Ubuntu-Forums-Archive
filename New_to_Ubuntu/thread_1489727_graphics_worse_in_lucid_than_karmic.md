### Post by houseworkshy on 2010-05-24
Not a fix but a tip for finding out about your hardware and posting it easily.  First go to System>Administration>software sources ( you will be asked for your password )  and make sure that all but the source code boxes are ticked.  You will be asked to reload the lists, do so.  

At this point you could try "hardware drivers" which is also in administration.  If you are lucky and the right one is in the repostitories then do the obvious and problem solved.  On my system the hardware drivers utility takes a minet to kick in.

If not ( or even if it is as this is a handy utility ) then use synaptic ( which is also under Admin' ) and search for sysinfo and install it.  This should put a link in Applications>system tools.  If you use it the "save file" option will produce a text document which has a summary of your system hardware so you know what to search for.  Posting that with any questions will also help people to help you.
