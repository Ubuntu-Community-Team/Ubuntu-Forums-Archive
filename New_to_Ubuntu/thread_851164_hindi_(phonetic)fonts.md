### Post by victorgreen on 2009-06-19
now i have no idea if this is phonetic or not, but go to system==> administration ==> language support and install hindi language support. then go to system==> preferences and add a keyboard layout for hindi. theres a menu there to set what keys trigger the switch between the english and hindi layouts. i set mine to alt shift, which apparently is what windows does. It worked in open office, but i had already installed a ton of hindi fonts in around a month of fruitless attempts to figure out how to type in open office in hindi
