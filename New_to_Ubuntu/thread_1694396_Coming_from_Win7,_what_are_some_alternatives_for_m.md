### Post by gordintoronto on 2011-02-24
To make DVDs: devede
It creates an ISO, then you use brasero or k3b to actually write the DVD.

Spend a few minutes with devede's help file so you can understand the available options. For example, I made a DVD that immediately starts playing, and when it finishes, it plays again.
