### Post by mikeyphi on 2009-10-02
You might find some answers here: 
[https://help.ubuntu.com/8.04/internet/C/networking.html](https://help.ubuntu.com/8.04/internet/C/networking.html)

Basically, set up both systems to connect via the router.
set up both computers to allow sharing - make sure you remember what names you gave to each computer

linux should be able to see the windows files via the router connection, as long as sharing is enabled on both computers. 

You might need to install/enable Samba (for sharing) in linux
more help here:
[https://help.ubuntu.com/community/SettingUpSamba](https://help.ubuntu.com/community/SettingUpSamba)
