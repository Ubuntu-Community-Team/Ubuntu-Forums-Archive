Hi folks, more beginner questions. When i installed Ubuntu i chose to partition the hd in half. I'm using the other half for backup with the program Quickstart. Is this an ok way to go or if there is a problem i won't be able to restore?

---

### Post by Diabolis on 2008-10-17
It all depends on what you call a problem. You are safe for this situations:

Your system completely crashed
You accidentally deleted some files (in your working partition)

There are risks if you have your back ups in a partition in the **same** hard drive.

The hard drive starts to fail because its old or because it suffered damage
Someone steals your drive
