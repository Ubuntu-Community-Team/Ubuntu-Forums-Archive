### Post by BlakeM on 2010-01-06
With regards to opening MS Word Documents:

Quote from Wikipedia: "[Open Office] Writer is capable of opening and saving documents in a number of formats, including ...**Microsoft Word's DOC, DOCX**..."

*.doc and *.docx are the default file extensions for MS Word 2003 and Word 2007 respectively.

Open Office Writer is the default equivalent of Microsoft Office Word in Ubuntu.

Quote from Wikipedia: "Microsoft has first released support for the OpenDocument Format in Office 2007 SP2."

The OpenDocument Format is the default format used by Open Office Writer.  The file extension of these files is *.odt.

### Post by tom.swartz07 on 2010-01-06
Hi Karen! Welcome to the Ubuntu Community!


You will be pleased to hear that with your computer specs, you should be able to run regular Full on Ubuntu on both. I would suggest that you create whats called a LiveCD, a burned cd with a .iso file on it, to test and see how it performs with your specific hardware. Note that the LiveCD will be really slow compared to your actual install, but the look and feel will be the same. If it turns out that the full install runs sluggish, just email me personally and I will help you get it running faster than a sports car! :)
When they say 'older' they usually refer to something near decades old. Im running Ubuntu on a 3 year old Laptop with no problems at all. The general rule for Ubuntu is, the older the hardware, the better it's supported!

iPods, iPods, iPods... Theres no definitive answer for iPods on Linux. I can tell you for certain if you have an iPod Touch or iPhone, youre out of luck for right now. You could dual-boot and sync with iTunes or even run Windows as a program in Linux with Virtualbox. Every other iPod, though, will sync beautifully with Ubuntu. Songbird is my favorite program- it will even import your iTunes library if you have it.

Ubuntu also comes pre-packaged with a set of programs called OpenOffice. Its basically an open-source version of Microsoft Office. Each can open and edit files from Microsoft Office, as well as save files in formats that are specific to OO. Personally, I even run OpenOffice on my Windows 7 machine because I like it so much.

Here's an EXCELLENT site to help you get started with Ubuntu: [http://www.psychocats.net/ubuntu/](http://www.psychocats.net/ubuntu/)


I hope this helps! If you ever need any help, let us know here! Thats what the forums are for! :D
