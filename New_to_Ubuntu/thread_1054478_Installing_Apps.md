### Post by kspncr on 2009-01-29
.deb's should install easily by double-clicking, similar to .exe in Windows (if the program won't run after that, then that probably means you need to install some dependencies. There is usually a readme to tell what dependencies are needed which you can find in synaptic package manager).

tar.gz is a file archive, and in my experience, installation can vary. You can right-click and then click "extract here" and browse the folder. There is usually a readme here, installation depends on what is in the folder. Note that you also might need to install dependencies with this installation method too. The readme, if it's there, should tell you what, or better yet, give you detailed installation instructions. Look for the dependencies in synaptic package manager. After you install enough programs in this way, you get a feel for the kinds of files you might find and the ways of installing them. Also, [http://ubuntuforums.org/showthread.php?t=246092](http://ubuntuforums.org/showthread.php?t=246092)
