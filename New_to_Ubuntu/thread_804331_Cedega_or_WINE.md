### Post by Ralphie on 2008-05-23
Cedega uses wine, but has better long running support of certain titles. 

I would try it in wine, since it's free, if you can't get it to work, and see that it is supported under cedega, go that rout, because you are likely to get more one on one help there. 

Fifa 08 *should* work without a hitch, though.
