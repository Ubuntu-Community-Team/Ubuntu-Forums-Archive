### Post by juil on 2010-09-04
Since you are completely new, the easiest way to get started is by looking at the software centre. It should be in your Application menu. (I'm using Lucid lynx so not quite sure)

Click the sound and video category and it will give you a list of all the software you can use.

Like 73ckn797 said, visit the links. I've learned ALOT by reading documentation and through forums. Ubuntu is fun to learn, but since Ubuntu is Linux, you'll be learning linux as well, which is awesome to learn for jobs, personal computing etc...

Oh, if you want to read books on Ubuntu, you can download 'ubuntu pocket guide' FREE. Just google it.
If your more serious about it, you can buy 'Beginning Ubuntu' by Keir Thomas from amazon or ebay for about $15 - $20.
