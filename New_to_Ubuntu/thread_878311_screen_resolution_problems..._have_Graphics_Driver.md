### Post by spiderbatdad on 2008-08-02
> **jethier said:**
> Sorry to break up this thread but how does one check their xorg.conf file??

you can navigate to the file graphically Places>>File System>>etc>>X11>>xorg.conf

or through a terminal:```
cat /etc/X11/xorg.conf
```

to edit: ```
gksu gedit /etc/X11/xorg.conf
```

see the man page. ```
man /etc/X11/xorg.conf
```

# Edit this file with caution, and see the /etc/X11/xorg.conf manual page.
