### Post by oldrocker99 on 2016-04-27
I did have few problems scanning with an Epson NX420, which itself is off the market, but scanning worked very well for me with it. This may be a dumb question, but did you install HP scanner drivers? (You probably did). If you're printing wirelessly, you'll need a wireless scanner driver, which Epson has.

I have a Brother color laser printer (which I got cheaply; I'm not rich) which has no scanner, and I keep the Epson hanging around for my infrequent scanner usage.

I'm an old guy myself (67).

Which scanning software have you tried to use? I found xsane to be featureful and easy to use. Unity has its own scanning program, media-scope-mediascanner2. You likely have simplescan already installed.

### Post by him610 on 2016-04-28
A couple of years ago, I purchased a refurbished B&W laser Brother MFC-7360N for ~$99 online from Staples. Original retail price was over $300. It does everything - prints, scans, copies, faxes. I seldom ever use fax anything, but it's there if needed. It works great; never had a problem with it. I am perfectly satisfied with it. Replacement toner cartridges from Amazon are reasonably priced. Linux drivers can be downloaded from Brother support website. If the Brother instructions are followed, the setup in Ubuntu is easy. 

I am sure a newer model can be found with wireless capabilities. The description of this model satisfies all of your specifications.
