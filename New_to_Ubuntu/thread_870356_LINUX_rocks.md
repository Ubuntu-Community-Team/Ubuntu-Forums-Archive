### Post by ad_267 on 2008-07-25
You can download Ubuntu from [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download) and select the "Desktop Edition" and "Standard personal computer."

If your computer is old and doesn't run the live cd you can tick the box for the alternate install cd, but try the live cd first.

This explains how to install Ubuntu and other useful information [https://help.ubuntu.com/8.04/switching/index.html](https://help.ubuntu.com/8.04/switching/index.html)

Have fun! If you need more help just ask in these forums.

### Post by Pasedaddy on 2008-07-25
As said above check to see if it runs form the live CD for starters.  Simple note if XP runs with every feature then Ubuntu should have no problem running. 

System Requirements

Ubuntu is available for PC, 64-Bit PC and Intel based Mac architectures. At least 256 MB of RAM is required to run the alternate install CD (384MB of RAM is required to use the live CD based installer). Install requires at least 4 GB of disk space.

Well in most cases Linux runs great on any hardware. For full Ubuntu I would say you need 512 mbs of ram just because I see my computer uses up about 256 MB. If you don't have that then try Xubuntu. Its the same thing just lighter. 

XUbuntu link is here and specs need are bellow. 

[http://www.ubuntu.com/products/whatisubuntu/xubuntu](http://www.ubuntu.com/products/whatisubuntu/xubuntu)

System Requirements

Xubuntu is available for PC, 64-Bit PC.

CDs require 128MB RAM to run, or 192MB RAM to install. Desktop install requires at least 1.5GB of free space on your hard disk.

You may like KUbuntu better since it is more like Windows (has something similar to start on Windows)

[http://www.ubuntu.com/products/WhatIsUbuntu/kubuntu](http://www.ubuntu.com/products/WhatIsUbuntu/kubuntu)

Finally if you can keep Windows installed on your hard drive. This makes it so when you switch you can use both as you learn Ubuntu. Ubuntu is user friendly but is is not windows and can take some time to get to know. 

Enjoy Ubuntu and Linux!
