### Post by tacantara on 2009-11-10
Have you tried GIMP yet?  It is a photo editing program designed along the lines of Photoshop.  I've used it for some very basic editing, and found that it does quite well.  If you really like Photoshop, you can download an additional package called Gimpshop, which will modify GIMP's menus to more closely resemble Photoshop, for a more familiar user experience.  GIMP may already be installed on your version of Ubuntu, but if it isn't, you can download it through the Synaptic Package Manager.  Gimpshop is available at [http://www.gimpshop.com/download.shtml](http://www.gimpshop.com/download.shtml).
