### Post by Michael.Godawski on 2008-11-28
> **Standard installation**

 To install Ubuntu, you will need *compatible hardware* and an *installation CD*. The  following guides will help you with these steps. 

[LIST=1]
[*]Check if your hardware works with Ubuntu and meets the minimum system requirements. (see [SupportedArchitectures]("https://help.ubuntu.com/community/SupportedArchitectures"), [HardwareSupport]("https://wiki.ubuntu.com/HardwareSupport") and [Installation/SystemRequirements]("https://help.ubuntu.com/community/Installation/SystemRequirements"))
[*]Obtain an installation CD. (see [GettingUbuntu]("https://help.ubuntu.com/community/GettingUbuntu"))
[*]If you downloaded a CD image, burn it to a CD (see [BurningIsoHowto]("https://help.ubuntu.com/community/BurningIsoHowto")) and check to make sure it was burned correctly. (see [Installation/CDIntegrityCheck]("https://help.ubuntu.com/community/Installation/CDIntegrityCheck"))
[*]Install Ubuntu by booting from the installation CD. (see [GraphicalInstall]("https://help.ubuntu.com/community/GraphicalInstall"))
[/LIST]
Perhaps there was something wrong with the burning process?
Did you download the CD iso from the official page?
