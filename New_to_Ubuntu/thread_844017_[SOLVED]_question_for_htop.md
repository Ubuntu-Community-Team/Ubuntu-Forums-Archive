I recently heard about the system monitor bug (showing/using high cpu usage), and that htop is a variant for that program. I was wondering if I can make a little icon on my panel for htop like system monitor has, instead of going into my terminal to check how my computer's running.

### Post by uclalinux on 2008-06-29
**Right click** on the panel where there is no icons, click on "**Add To Panel**"

double click on "**Custom Application Launche**r"

**Type**: Run in Terminal
**Name**: What ever you Want it to be called, this case Htop
**Command**: htop
**Comment**: Whatever you want, it really dose not matter

This should add the a little icon that will launch htop from the panel
