### Post by mapes12 on 2012-07-07
As stated, you need the HPLIP driver. Unfortunately, the version that ships with Ubu and in the repos is out of date. To get the latest supported driver go here and follow both the download and installation instructions:-

[http://hplipopensource.com/hplip-web/index.html](http://hplipopensource.com/hplip-web/index.html)

The download will most likely have the file in your **Download** folder. Drag it to your Desktop then the Terminal commands in the installation guide will work faultlessly.

With the latest version there is no need to install any extra GUI packages. Everything including the GUI comes with the package.
