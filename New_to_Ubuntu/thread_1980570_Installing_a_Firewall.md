### Post by Gone fishing on 2012-05-15
You probably don't need a firewall as all the port are closed and the box is safe. If you are behind a router this also acts as a firewall. You could test using shields up at [www.grc.com](www.grc.com).

If you wish to use a firewall iptables is the Linux firewall and this can be configured easily using firestarter or firewall configuration which can be installed from the software centre.
