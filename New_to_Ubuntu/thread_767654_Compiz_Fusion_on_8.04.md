### Post by adult swim on 2008-04-25
its already on there,  you need a package to configure it though.  open the terminal and type in ```
sudo apt-get install compizconfig-settings-manager
```
once you have that installed, go to system/preferences/advanced desktop effects settings and you can configure your compiz settings
