### Post by Tom Mann on 2009-05-07
Hi Colormoon,

If you switch to the Jaunty repository in Ubuntu, the next update will upgrade you to Jaunty in what is considered an unclean fashion.

With certain applications, you can attempt to download the deb directory from the repository at [http://archive.ubuntu.com/ubuntu/dists/jaunty/]("http://archive.ubuntu.com/ubuntu/dists/jaunty/") and install it, but you will most likely hit dependancy issues, and I strongly advise against this.

Is there any specific component you need from Jaunty?
