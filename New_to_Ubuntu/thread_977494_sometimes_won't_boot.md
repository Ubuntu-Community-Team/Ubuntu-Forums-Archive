### Post by stevebond001 on 2008-11-10
If you have only installed these programs, and it worked fine before you installed them, I would uninstall them one at a time and see if this fixes the problem.  If it does then great.
However, if it doesn't fix it I would suggest backing your data up and performing a fresh install (if you do this, create a separate partition when installing and mount your /home directory there.  This will ensure that your data is separate to your O/S should you wish to reinstall / upgrade in the future).

Hope this helps.
