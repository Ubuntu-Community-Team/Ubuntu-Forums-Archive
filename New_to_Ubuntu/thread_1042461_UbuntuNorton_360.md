### Post by jimmy the saint on 2009-01-17
[http://www.psychocats.net/ubuntu/security](http://www.psychocats.net/ubuntu/security)

Check that out.  The beauty of Ubuntu (and linux in general) is that it is designed to be secure.  Windows in inherently vulnerable and there is a lot of malware out there that targets it.  Linux is more secure and there are no known viruses in the wild (last I checked) that target it.  

If Norton is setup in vista, it should still scan the Ubuntu partition by default (I believe it will scan all drives) but you will not be able to use it on the Ubuntu installation to monitor network traffic or anything.  There are linux tools for that.  Read the article for a much better explanation!

Good Luck
