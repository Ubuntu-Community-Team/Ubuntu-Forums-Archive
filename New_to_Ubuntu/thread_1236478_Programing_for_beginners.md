### Post by binaryburnout on 2009-08-10
I'm not entirely sure what you mean by "programs used for programming". You can simply program in any text editor, then compile and run. If you are referring to IDE's (Integrated Development Environments), there are plenty out there but you might want to be more specific as to what language you wish to learn and more importantly, what you wish to do with your soon-to-develop programming abilities. Be aware that being told what language to learn is like being told which car to buy. It is unimportant what other people like in a new car, what is important is what you like. The following is a small list of languages you MIGHT look into and what they can be used for...
**Games 2D / 3D:**
[LIST]
[*]DarkBasic
[*]DotNet + DirectX
[/LIST]
**Games 2D in websites:**
[LIST]
[*]Flash
[*]Java
[/LIST]
**Office:**
[LIST]
[*]DotNet
[*]Visual Basic
[*]Delphi
[/LIST]
**Utilities & Tools:**
[LIST]
[*]Delphi
[*]Visual Basic
[*]DotNet / Java
[/LIST]
**Business Applications:**
[LIST]

[*]Java
[*]C / C++
[*]Ruby
[/LIST]
**Web Sites:**
[LIST]
[*]Asp(.NET)
[*]PHP
[*]Perl
[*]Javascript
[*]Ruby
[/LIST]
**Graphics/Sound:**
[LIST]
[*]Delphi
[*]C
[*]C++
[/LIST]
**Multimedia:**
[LIST]
[*]Delphi
[/LIST]
**Advanced Tools:**
[LIST]
[*]C
[*]Assembler
[/LIST]

This is a very abridged list and should never be taken as the only languages/purposes you should take a look at.
