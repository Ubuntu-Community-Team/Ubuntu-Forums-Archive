### Post by Quackers on 2010-12-04
Welcome to UF
The .iso image must be burned as an image, not just burned to disc. There are free .iso burning softwares out there for use in Windows (eg Imgburn) which will do the job.
If you have done that already and the computer won't boot from the disc you may need to change the bios settings to boot from cd/dvd drive first.
