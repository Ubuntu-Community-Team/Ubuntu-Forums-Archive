### Post by leunam12 on 2016-05-25
Try this from the terminal (Intenta esto desde la terminal)
```
warzone2100 --fullscreen
```
Once you run it that way, it will always run on full screen.
(Despues que lo empiezas asi la primera vez, siempre arranca en pagina completa).
