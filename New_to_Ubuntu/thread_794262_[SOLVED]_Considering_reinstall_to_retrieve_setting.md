there are also several touchpad management utilities available in the standard Ubuntu repositories that can manage your touchpad. I have to use one to turn my touchpad off so it doesn't function while I am typing. So if you search for touchpad in synaptics by name & description you will find stuff that does this for you.

Also here is my script for backing up some important config files including xorg.conf . It will create a directory on your desktop and save them all in that with some text files with your system information. It does not require the use of sudo and does nothing that can hurt your system as it only runs simple info commands and copys the config files to a directory it will create on your desktop ( works for english desktop for others you will need to change ~/Desktop references to ~/ for your home directory instead )

Just go to the files properties and make it executable - then run it by either doubleclicking and selecting run or entering it into a terminal.
