I think I figured out a way to fix it in Firefox.  

If you go to Tools > Add-ons you can disable some of the languages.  South African was enabled by default and both British English and American English were both enabled.
I disabled South African and British.

If you want, You can install the British English dictionary from [https://addons.mozilla.org/firefox/addon/british-english-dictionary/](https://addons.mozilla.org/firefox/addon/british-english-dictionary/)

That doesn't really solve the Ubuntu Dictionary problem, but it seems to fix Firefox.
