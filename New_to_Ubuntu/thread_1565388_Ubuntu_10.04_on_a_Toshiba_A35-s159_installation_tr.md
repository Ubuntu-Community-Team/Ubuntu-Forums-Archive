### Post by jimrz on 2010-08-31
For that machine you do not want the netbook remix,as it is optimized for intel atom processors. From the specs you listed, it should be able to run the regular ubuntu ok, and certainly a lighter version such as Lubuntu would do well. It sounds as though there may be an issue with your video card, can you post which card it has? and then search on these forums to see what results others get with the card. Should that not turn out to be the issue, you could try the install using the "alternate install" .iso (it is text based installer that frequently succeeds where the "live" cd fails.
