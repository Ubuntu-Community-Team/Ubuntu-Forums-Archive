### Post by ZabiGG on 2008-06-08
Hi :)

I solved that problem (EN-FR) in System > Preferences > Keyboard > Layouts > 

I chose both languages

Then clicked Layout Options

and, in Layout Switching

chose the key combination of my choice (You would choose Caps Lock key changes layout).

Closed everything and was done. I also have SCIM on board.

Hope this helps.
