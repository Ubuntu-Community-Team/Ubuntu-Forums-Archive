### Post by buzzingrobot on 2013-12-27
No.  Windows programs do not run on Ubuntu or any other version of Linux, or vice versa.  Just as they do not run on OS X.

In general, programs written for one operating system run only on that operating system.

Tools -- Wine and PlayOnLinux -- are available that permit a limited number of Windows programs to run on Linux by creating a temporary simulation of the Windows environment that Windows programs require. You'd need to verify a program works with one of these tools before spending your money.
