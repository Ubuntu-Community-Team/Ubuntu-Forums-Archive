### Post by diesch on 2009-11-05
Alt-F1 gets you to the main menu. Use <Left> to go to the System menu. There go to Settings&#8594;Keyboard, hit <Tab> until the label on top of the first tab is selected, use <Left> to go to the "Mouse Keys" tab, <Down> to select "Pointer can be controlled using the keyboard" and <Space> to switch it on. Now you can control the mouse cursor with the numpad keys.

### Post by nothingspecial on 2009-11-05
Install [gnome-do]("http://do.davebsd.com/")

In gnome -

Alt F1 - main menu
Alt F2 - run command
Alt F4 - close window
Alt F5 - un maximise (if you know what I mean)
Alt F7 - move window with arrow keys
Alt F8 - resize window with the arrow keys
Alt F9 - minimise window
Alt F10- maximise window 
Alt Tab - select window

In firefox use Alt B to bring up your bookmarks and Ctrl D to add a bookmark. If you press the > key (as in Shift + . on a uk keyboard) you can start typing the text in a link and when it selects the right one (normally after a couple of keys) just press enter to follow that link.

When you have opened an app (with Alt F2 or gnome-do) any button or menu bar thing (you know.... on the top .... file edit view etc) with a letter underlined means that if you press Alt and that letter its as though you clicked on it ie it will open the menu or do what it does when it`s clicked ....... if you know what I mean.

Those are just a few tips there`s loads more
