### Post by talsemgeest on 2009-03-29
> **vikashkoushik_cric said:**
> Hey trinidad,

 EXACTLY were should i paste on the website?
Ok, if you want it put it in a current website (one that you didn't create), you must save the page to your hdd, then open it with gedit (or your choice of text editor.) Past the code somewhere underneath the <body> tag so that it will appear in the page itself. Then, open the .html file with your web browser, and hopefully the countdown will be on there.

Also, just to avoid any confusion, this timer is for people who run their own websites, and is not meant for the average user. The best way for a normal user to see the timer is just to go to the download page and see it there. :)
