### Post by cszikszoy on 2008-08-17
probably the best is mp3splt.  It allows you to cut a large mp3 into individual tracks.  If you have a .cue file it will also split the tracks, rename and tag them properly as well.

It's a command line app (I believe there is a GUI though), but it's probably exactly what you're looking for.

```
sudo apt-get install mp3splt
```

Check the man pages or howtos online.
