### Post by cmcanulty on 2011-03-29
Very easy go software center and install unetbootin. Empty the flash drive (including hidden folders) then pick ubuntu and version from the dropdown boxes and run it. Then set computer to boot to flash drive usually by hitting f2 or f10 or F12 while booting up. and turn off computer and restart with flash drive already plugged in.
