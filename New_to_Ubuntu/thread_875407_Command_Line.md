### Post by Sydius on 2008-07-30
The biggest tip I have is to use tab to auto-complete filenames and the like.  So it can be faster to type "cp some/long/file/location some/other/long/file/location" if you use tab auto-completion than it would be if you had to go and open those locations in the GUI and click to get to them.  It also depends on how fast you type, of course.

A lot of things can only be done via the shell (command line), too.
