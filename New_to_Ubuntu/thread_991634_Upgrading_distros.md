### Post by jyaan on 2008-11-24
The easiest way to handle this is to always use a separate /home partition. This way you can install a new distro and keep all your data, although you will still need to re-install applications. Anyways, you just need to back up the configuration files and data that you want to keep. There will be hidden folders (they start with a dot) for each application, and that will contain the config settings. Just back those up before you do it and you should be OK.
