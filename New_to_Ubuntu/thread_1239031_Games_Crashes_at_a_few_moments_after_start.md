### Post by Hospadar on 2009-08-13
I think you should try and start them in a terminal.  Often you'll see some helpful (or at least informative) errors.  Without error messages a crash is just a mystery.

So:
Open up a terminal, Applications->Accesories->Terminal
Type in the command for the game - this is generally just the name in all lower case
I suspect
SuperTux: supertux
OpenArena: openarena

If you are unsure or it is non-obvious, you can edit your start menu, find the launcher for that program, edit it, and take a look at the command it's actually running.
