### Post by cariboo on 2008-08-13
Install startup manager this will allow you to change the grub background as well as usplash. to install startup manager in a terminal type:

```
sudo apt-get install startupmanager
```

Startup manager will allow you to change a lot of other things like boot order, screen resolution and more.
