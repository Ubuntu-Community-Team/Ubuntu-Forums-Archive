### Post by newb85 on 2013-06-30
The .../linux-headers-3.2.0-48_3.2.0-48.74_all.deb file your machine downloaded may be corrupt.  Try running

```
sudo apt-get clean
sudo apt-get -f install
```

That should force your machine to re-download it.
