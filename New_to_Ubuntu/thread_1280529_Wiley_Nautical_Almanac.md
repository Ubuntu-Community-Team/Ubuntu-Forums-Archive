### Post by laidback on 2009-10-02
Other readers may be interested to know that I just installed Wiley Nautical Almanac onto my Ubuntu 9.04 system. I had previously tried it with 8.10 and failed.

Method:-

Install Wine & Wine-gecko from the repositories
Navigate to the Wiley web page:-
  [http://www.wileynautical.com/view/0/almanac.html](http://www.wileynautical.com/view/0/almanac.html)
Press the Install Now button....accept conditions..& bingo.

Why does it work this time and not with 8.10? maybe because I also loaded Wine-gecko this time, perhaps Wine has been improved, maybe both.
