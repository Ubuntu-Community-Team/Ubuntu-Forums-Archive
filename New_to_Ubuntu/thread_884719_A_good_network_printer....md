### Post by phidia on 2008-08-09
There are many choices and I recommend you consult the printer search feature of [openprinting.org]("http://www.linuxfoundation.org/en/OpenPrinting") as you shop for a printer. 
In my experience HP makes several good printers that work in linux/ubuntu. Make sure to install the hplip package if you do go with HP.
BTW I have a HP photosmart C5100 which does connect through ethernet (there are other models too) but because that model is no longer in production you can sometimes find them discontinued-new-and inexpensive.
