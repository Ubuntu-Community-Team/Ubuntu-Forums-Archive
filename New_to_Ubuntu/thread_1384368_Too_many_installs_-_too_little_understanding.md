### Post by MarkC502 on 2010-01-18
Your showing pictures of your Wine directories, with a bunch of stuff installed. Wine is a software which allows you to run windows based .exe files from inside linux. So I don't know why you would want to run firefox or any other program that is available in Ubuntu from Wine. Wine is for things you cant get on Ubuntu that you want to run that are only for windows, I am using it to run a filesharing program on a DL box 24/7. So it looks to me that all you did is run amok installing stuff in wine , so what , you should be able to uninstall these as easily as you installed them. Or you could always remove Wine and its configs then re-install it and start over fresh.

If you only use Wine for things you can't replace with either Linux versions or a comparable Linux program you will find you don't have that many things you need it for ( thank god )

Good Luck with Ubuntu
