### Post by Prospero2006 on 2008-06-02
The best thing to do is manually configure your /etc/X11/xorg.conf

Now, it sounds like your xorg.conf has both monitors detected. So, with any luck you'll have an easy time extending the desktop.

```

Section "ServerFlags"
 Option "Xinerama" "ON"
 EndSection


 Section "ServerLayout"
        Identifier      "Default Layout"
        InputDevice     "Generic Keyboard"
        InputDevice     "Configured Mouse"
       Screen "screen0"
       Screen "screen1" RightOf "screen0"
EndSection

```

If the earlier part of your xorg.conf file has a screen0 and screen1, you will be able to simply append that stuff to the end of the file and you should see a difference. 

If your file xorg.conf does not define screens, you'll probably have to go in tell it which monitor belongs to which screen. 

Suggestion: Google for xorg.conf ati configuration.
