### Post by Eisenwinter on 2009-03-24
Go to Pidgin's preferences.

I think it's in Tools -> Preferences, but it may somewhere else, I don't use Pidgin.

There's an option for "Browser", it should say "Default browser", change that, to "Firefox"

---

### Post by cim on 2009-03-24
thank you for your reply, i fixed it by going to preference > network tab > configure browser

firefox wasn't set as my default browser. thanks! it was bugging me for the longest.
