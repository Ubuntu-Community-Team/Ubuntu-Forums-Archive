### Post by ubudog on 2011-06-02
Hi there, welcome to Ubuntu.  The console in Ubuntu is normally referred to as the terminal.  It is probably the most useful thing in Linux.  It is located in Applications>Accessories>Terminal.  It may seem hard to use at first, but you will gradually get the hang of it.

You mentioned essential things to have, well, if you're a gamer, and are used to playing Windows games, I recommend installing PlayOnLinux, a really great program that can help you to install Windows games on Ubuntu.

Welcome to the forums.  The best advise I can offer you is use the operating system and if you find that you need or want something new or different, go to the Ubuntu Software Center and download it from there.
The console is called the terminal and is under accessories.  Using the command sudo gives you root or temporary administrative privileges.
