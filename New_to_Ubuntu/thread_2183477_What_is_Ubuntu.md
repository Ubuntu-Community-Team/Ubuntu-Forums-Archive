### Post by jerome1232 on 2013-10-25
Ubuntu is a Open Source Operating System, other operating systems you may have heard of are Windows, and Mac OSx.

Ubuntu is not a virus, it's not a remote controlled program. It does have utilities (like both Windows and OSx do) which do allow remote access, but it has to explicitly turned on by you.

Universal Access is a suite of programs designed to make using Ubuntu easier for those with disabilities, it can turn on high contrast mode for those who can't see well, it enables magnifiers and a computer generated voice that will read things to you and more.

I'm guessing what you are experiencing is shock of using a new OS, new OS's take some time to pick up.

### Post by slickymaster on 2013-10-25
Ubuntu is a distribution - or distro - of Linux. In Linux, a distribution is a version of the operating system that has the Linux kernel as its foundation. There are literally hundreds of different Linux distros out there. Many are free and have communities of users who provide each other with guidance and support.
But installing a free Linux distro with limited support options can be intimidating to the average computer user. That's where Ubuntu comes in, attempting at leveraging the Linux operating system in a way that's easy and intuitive for the end user.

### Post by newb85 on 2013-10-25
Suspicions of spying aside, no one has asked what release (version) of Ubuntu you have installed.  There's a distinct possibility that he gave you a laptop loaded with a release that is outdated, unsupported, and therefore unsecure.

If you're not going to replace it with Windows, I suggest you reinstall Ubuntu.  That way, your mind will be at ease about it being set up to spy on you, and you will know that you have a current release.  I'm willing to walk you through it, if you're willing to spend a little time on it.  (And for that matter, others will pitch in.)

You will need installation media - either DVD or USB flash drive.  If the laptop has a DVD burner and you have a blank DVD, you can use that.  Otherwise, you'll need to use a USB flash drive that's at least 1GB.
Also, you will need to download Ubuntu - it's around 800MB, so if you pay for internet by how much data you use that could be a problem.  Also, if your connection is slow it will take a while.

Before starting, we should check that your machine is powerful enough to run a current release of Ubuntu.  (If it's not, you'll need to go with a lighter variant.  Still just as secure, and it will probably meet your needs, but most would say not as nice.)  Start by opening a terminal.  Try Ctrl+Alt+T.  A window should pop up with a command prompt ending in a $.  If that works, enter the following.  (You can copy and paste it.)
```
lsb_release -a
```
Post the output.  That will tell us what release is currently installed.

Then enter the following
```
sudo lshw -sanitize
```
It will ask for a password.  Enter the password for your administrator account (I'm assuming you have it).  You won't get any visual feedback as you type it.
And post the output.  That will give us detailed info about your hardware (with the serial numbers, etc. removed).
