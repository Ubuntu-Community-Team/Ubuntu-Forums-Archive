### Post by ajgreeny on 2011-02-09
If you have a USB keyboard try enabling legacy USB support in the computer's BIOS, that may sort out the first problem.

For the lost/forgotten password and username etc etc see [http://www.psychocats.net/ubuntu/resetpassword](http://www.psychocats.net/ubuntu/resetpassword)
In order to boot into recovery mode you will need to sort out the first problem, but once you have done that, the second problem should be straightforward.
