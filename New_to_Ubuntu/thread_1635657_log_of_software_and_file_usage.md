### Post by Zookalicious on 2010-12-02
Try looking up "Activity Journal" in Ubuntu software center. It runs off of Zeitgeist which is essential a daemon that keeps tabs on things like that. It shows when files are accessed and it also shows what programs you open and on what days / times. Keep in mind that not all programs support Zeitgeist integration just yet, but it's picking up speed really quickly.
