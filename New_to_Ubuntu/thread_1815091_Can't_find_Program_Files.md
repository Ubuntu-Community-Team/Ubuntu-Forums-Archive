### Post by anewguy on 2011-07-30
And....if you want to do this with a GUI'd file manager, like Nautilis ("Places"), open the file manager and change the view to show hidden files and directories.  The .Wine folder will then show.  Explore it and you'll see your wine "C" volume there so you can find as you normally would on that drive.
