ATI no longer supports that adapter for modern versions of any Linux distribution.

You will not find a proprietary driver that works, which is why you bored your system.

Stick with the default open source Radeon driver that was installed with Ubuntu originally.

### Post by QIII on 2014-07-28
There is nothing to switch to.

If you see output to your monitor, you are using the default open source driver.  But don't expect it to be a screaming hotrod when gaming.

If you want to stick with ATI and get driver support, you'll need an HD 5000 series or later.
