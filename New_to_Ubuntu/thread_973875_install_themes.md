### Post by _sAm_ on 2008-11-07
1. You can install from Synaptic; search for community-themes, ubuntustudio-theme and bluebuntu-look(if you just search for *theme* you will find more). When done rightclick on desktop > Change Desktop background. Press *Theme* tab and change it. 

2. Go to [http://www.gnome-look.org/](http://www.gnome-look.org/) and download some you like.  When done rightclick on desktop > Change Desktop background. Press Theme tab. and just drag the themes you downloaded on to it to install. When done change to the new theme.
