I think it's because Mozilla tries to adapt each one to the standard "look and feel" of the overall system.

For instance, I dual boot with windows, and there are minor differences in menu layouts and other, mostly gui things, but I have all the same options available. I have friends that use Macs and they report that it is a little different looking there as well.

If you look in your Firefox Extensions, you should find one named **Ubuntu Firefox Modifications** that you could disable. Mainly it ties Firefox in with apt support for downloading and installing extensions and themes. I disable mine because it tends to interfere with several of my most useful extensions.

Still, even with it disabled, the menus, etc. still have that linux "look and feel".
