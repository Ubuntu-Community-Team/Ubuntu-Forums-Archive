### Post by cariboo on 2008-07-09
Install startup manager, it is available in the rpositories and you can use Add/Remove Programs or Synaptic Package Manager. Once you have startup manger installed you can access it from System-->Administration-->Startup Manager.
This program will allow you to set the default os to boot.
