### Post by YannBuntu on 2010-07-05
Hi,

First of all, Skype is closed-source, so that the only software that can do audio&video calls with Skype is... Skype :)

If your interlocutors are ok to try something else, I recommend :
- Ekiga (which exists also on Windows), and that is working pretty well since last version (see [here](http://wiki.ekiga.org/index.php/HowTo_install_Ekiga_packages#10.04_Lucid_Lynx_.28Long_Term_Support.29))
- Empathy, with Telepathy PPA, and the ubuntu-restricted-extras package, and a jabber account (e.g a Gmail address) can do voice+video calls with Windows Gtalk users

Empathy (with its Muji "extension"), can also do voice+video calls with 3 or more Linux users !
