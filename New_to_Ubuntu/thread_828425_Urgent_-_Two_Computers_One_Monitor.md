### Post by flyingtiger on 2008-06-14
I have 2 PC's running all the time. One is MS Windows XP and other Ubuntu Linux. Both use the same mouse, keyboard, and LCD monitor via a KVM 2 port (KVM switch = keyboard-video-mouse switch) switch. Pressing the 'Scroll Lock' switches between the 2 PC's. It works wonderfully, for me.
