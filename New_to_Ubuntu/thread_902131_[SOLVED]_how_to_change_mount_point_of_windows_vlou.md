### Post by jpds on 2008-08-27
> **macvr said:**
> hi, i'm new to ubuntu.[dual booting with ubuntu and XP]

**_1>_** i reinstalled XP recently and changed the previous XP volume labels

*previously* my windows partition was a single volume labeled > WIN XP

*now* i have split the partition in 2 and have labeled them XP and PACKAGES[check attached pic]

but my new partition labeled XP mounts to media/WIN XP

how do i change this to mount as XP?

Linux uses the fstab file to define where partitions are mounted, and with what options.

You'll have to open it with *gksudo gedit /etc/fstab*. And edit the line with "*/media/Win XP*" in it to */media/XP*.

> **macvr said:**
> **_2>_** also my volume XP mounts automatically and is displayed on the desktop, on every restart
when i try to unmount it it says that i dont have permissions to unmount it!
so i unmount it via partition editor! and to mount it back i do it through partition editor again![i dont have this problem with volume PACKAGES]

so how do i prevent it from automatically mounting on start?

Edit /etc/fstab so that the XP line has the <options>: "noauto", this will stop it from being automatically mounted on boot.

> **macvr said:**
> how do i set my privileges to mount and unmount directly?

You need root privileges to mount/unmount partitions on Linux. You can do this by using "sudo".

See: [https://help.ubuntu.com/community/Fstab](https://help.ubuntu.com/community/Fstab) - for more information on fstab.
