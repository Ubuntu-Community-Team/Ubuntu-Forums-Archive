### Post by Lars Noodén on 2011-12-25
1 - I'd say drop Windows and don't look back.  It's a bit harder to get a machine without paying the Windows tax, but sometimes if you call the vendor you can get a different deal than what is offered on their web site.  If nothing else you can document your refusal of the EULA and get a [Windows refund](http://www.google.com/search?q=windows+refund).  The details vary from region to region.
 
2 - My experience is very limited, I've mostly used Audacity.  You could also take a look at [Audacious](http://audacious-media-player.org/)

3 - A lot of applications work well with WINE, but some still don't.  Check the [app database](http://appdb.winehq.org/) to see if or how well an application is supported before buying.  

4 - Try to check the net for success or failure to support various hardware before you buy.  A lot of devices work out of the box.

5 - Yes.  It's just a matter of reformatting it as EXT3 or EXT4 if you plan to only use it with Ubuntu.

6 - See #1.  Vendors are under a lot of pressure from M$ to not sell anything except Windows.  This problem goes back many, many years.  So offers of Linux often last only a short time and you have to grab them while they are available.

### Post by anewguy on 2011-12-25
Yes, it is hard to find a laptop without Windows on it.

If you want a Linux box, I would pick out the PC I want, then check to be sure all the drivers, etc., are available as needed for that hardware in ubuntu.  When you have found that, get the one with Windows but ask for a Windows CD - you'll probably have to contact the vendor for it.  Save the disk.  Be sure the hardware you get is mutli-processor and get as much memory as you can.

Do your thing as far as buying the hardware and installing ubuntu (as others have mentioned, you may want to look at Ubuntu Studio - I assume it's still around). 

Now do as wildmanne39 suggested - install VirtualBox -> download it and the extension pack from the Oracle site.

Once you have VirtualBox installed, make your user part of the vboxusers group.

When you have the Windows disk (hence why to get one with the new PC), install it as a virtual machine in VirtualBox.  Install the extension pack, then be sure to create 1 USB filter - it can just be blank which allows all USB devices.

Why a virtual machine?  Well, you'll be within Ubuntu so you won't have to reboot to get to Windows.  It's far better than Wine, and Wine, the last I checked, did not support USB devices other than the keyboard, mouse, etc..  Graphics will be better in a virtual machine versus Wine, but neither will be as good as running "native" (be it Windows or Ubuntu).  Dedicate one of your processors to the virtual machine and a good amount of memory so you don't slow it down.

Just my 2 cents worth........

Dave ;)

p.s. - I'm a little partial to this just because I have both my desktop and my laptop set up that way.
