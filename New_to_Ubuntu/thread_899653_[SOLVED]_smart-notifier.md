### Post by gmoney on 2008-08-29
Hey Alien,

Are you trying to install smart-notifier?  If so, then do either of the following 2 things:

1)Click on System > Administration > Synaptic Package Manager
Search for smart-monitor, and then click "mark for installation"

2)Type the following into a Terminal:

```
sudo apt-get install smart-notifier
```
