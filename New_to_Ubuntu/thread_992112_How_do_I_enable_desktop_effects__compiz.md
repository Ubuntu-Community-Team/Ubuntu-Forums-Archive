> **Tex786 said:**

kwin is a core part of KDE/Kubuntu.  What matter is what version you have.  If you panel is grey, you will need compiz to get the fancy desktop effects (you are using KDE version 3.5.x).  If your panel is black, you are using KDE 4.1.x and can enable the effects via system settings -> Desktop
