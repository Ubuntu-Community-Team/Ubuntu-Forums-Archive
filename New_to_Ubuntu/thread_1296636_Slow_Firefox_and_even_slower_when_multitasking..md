You're using a testing version of Karmic - that means it's not quite ready for desktops yet! If anything you should have gotten started with Jaunty, rather than the Karmic Beta, but now my suggestion is you wait a week for the final release of Karmic to come out and give that a try.

Also, XP is, despite its persistence, quite an old operating system, and doesn't have very severe requirements. Ubuntu takes slightly more powerful hardware (mainly a little bit more RAM) to run than XP, and my suggestion is to try Xubuntu rather than Ubuntu if it's still not working out for you, since Xubuntu requires far less RAM. You won't have as fancy a desktop as you would on Ubuntu, but you can use all the same programs you like from Ubuntu.
[http://www.xubuntu.org]("http://www.xubuntu.org")

Xubuntu's made by Canonical, just like Ubuntu, so it's supported in much the same way on these forums, should you run into any other issues. Good luck!
