### Post by ajgreeny on 2014-03-23
Perhaps the .iso file you used for the DVD or USB was faulty?

Check your file against the listed values at [https://help.ubuntu.com/community/UbuntuHashes](https://help.ubuntu.com/community/UbuntuHashes) and if they don't match download again.
