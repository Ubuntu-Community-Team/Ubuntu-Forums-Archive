### Post by winh8r on 2012-03-05
To find out what environment you are using, just open a terminal and enter:


```
echo $DESKTOP_SESSION
```

this will tell you what you are using at present.

To choose a different interface at login you can select from the box on the login screen where you enter your password.

There are many file manager software applications available for Ubuntu, you can go to the Ubuntu Software Centre and review each one to find out which one best fits your needs.

When you say "save to library" this can be done in any file manager, just use the "save as" option and you will see a file selection dialog which will allow you to name the file you are saving and also choose where you are saving it to (your chosen library)

I hope this helps you.
