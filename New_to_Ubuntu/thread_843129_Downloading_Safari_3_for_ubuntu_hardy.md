### Post by angry_johnnie on 2008-06-28
Konqueror --KDE's default browser-- runs on the same engine as Safari, if that is what you want.  But there are many other browsers available.

I didn't like FF3 either, so I switched to Seamonkey, and it's very good.  There's also epiphany, galeon, or kazehakase:  all available in the repositories.

You could also download Opera 9.5 from the website.

But, if you really really want Safari, I think wine would be the only way to do it.
