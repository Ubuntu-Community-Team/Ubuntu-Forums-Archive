### Post by VideoRoy on 2012-12-19
I did this successfully on a test laptop with no issues.  However I am using 32bit of both Win 8 and Ubuntu 12.10.

This one case is different for me because I installed Win 8 first then let Ubuntu manage the boot menu / process after it was installed.  Work fine with no issues but my normal process for dual but is to use Neosmart (free version) EasyBCD.  You might want to try it out instead as it has always worked for me and the new version supports Win8.

EDIT: Sorry just notice that you are having trouble booting the install CD not ubuntu after install.  I used an older laptop that did not check for those other options.  Also I understand that there is a Ubuntu 12.10.1 to solve some of the UEFI issues.
