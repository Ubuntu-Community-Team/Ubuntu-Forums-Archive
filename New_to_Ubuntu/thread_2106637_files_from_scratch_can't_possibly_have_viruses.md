### Post by tartalo on 2013-01-19
I don't dare to say that it's absolutely impossible to manipulate a Linux Libreoffice to produce documents that contain malware executable in Windows, but I believe that there's nothing like that on the wild, virus developers have more efficient ways to infect Windows computers. 

If you want to play extra-safe, install clamav and scan the files before sending them.
