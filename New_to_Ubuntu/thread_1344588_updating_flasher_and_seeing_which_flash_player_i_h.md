### Post by hal10000 on 2009-12-03
open firefox, click the "Tools" menu--->Add-ons--->plugins, there you can see which version of flsh is used.

If you want to update to the latest flash version, you don't need to download it from adobe, just make sure your medibuntu repository is activated and install the **flashplugin-nonfree** or **flashplugin-installer** package.
