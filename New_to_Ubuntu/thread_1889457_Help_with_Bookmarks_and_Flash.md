### Post by cortman on 2011-12-01
The best image editing software I've used in Ubuntu is Pinta, hands down. It's really slick and super easy to use. I'd say it's kind of a cross between Gimp and Paint. You can quickly edit an image with the basic tools you'd use in paint, and it also has a number of advanced utilities that you'd find in Gimp as well.
I never put in the time to learn Gimp yet, Pinta does everything I need quite well.
