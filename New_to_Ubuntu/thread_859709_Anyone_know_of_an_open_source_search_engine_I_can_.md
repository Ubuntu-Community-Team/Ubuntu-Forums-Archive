### Post by aktiwers on 2008-07-15
You can add all search engines to firefox with keywords.

Like, try go to google.com
Right-click on the seach field (the area you normally type in keywords)
Pick "Add keyword"
In name type in, lets say Google and in keyword type in g
(yes just "g" without quotes)
Now in your addressbar (where you type URLS :)) type:
g hello 

to search for "hello".

This works on all search engines and can be used to keyword websites too.
