### Post by Crusty Old Fart on 2013-08-25
At the top of your Gparted window, there is a menu.
The right most selection on that menu is: Help
Move your mouse pointer over it and push down one time on your left mouse button (a left click, more often referred to as just a click).
The main help page should open for you.
From there, you'll be able to click on several links that will help you learn how to use the software.
The folks who created the help documentation did a very nice job.

### Post by 3rdalbum on 2013-08-25
Your screenshot definitely does not show the Ubuntu drive. It shows a Windows-formatted disk with only one partition, that is 128 gigabytes.

If you have spare space on your hard disk, you will need to shrink one of the adjoining partitions, and then grow the Ubuntu partition. If you're not sure how to use Gparted for this, consult the Help file or one of the guides online.

I'm assuming that you haven't used the old Windows-based installer; this doesn't create partitions at all; but it's been discontinued so I guess you probably haven't used it.
