### Post by MG&amp;TL on 2011-11-22
This : [http://www.webupd8.org/2011/04/how-to-reset-unity-launcher-icons-or.html]("http://www.webupd8.org/2011/04/how-to-reset-unity-launcher-icons-or.html") worked for me. You can probably login via unity2d to get a launcher back temporarily.

Word of warning about compiz + Unity. Messing via command-line is likely to kill it. Sorry, that's the way it is. If you must tweak, install the package compizconfig-settings-manager and be very careful not to disable the Unity plugin or go anywhere near anything like the desktop cube. Wobbly windows usually works for me though.
