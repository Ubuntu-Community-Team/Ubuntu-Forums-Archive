### Post by Morbius1 on 2012-04-06
**"Personal File Sharing" is not Samba.** 

Personal file sharing uses elements of apache to share one folder ( Public ) in your home folder. It uses avahi to broadcast it's share and is basically only useful if the other machines on your network are running Linux.
