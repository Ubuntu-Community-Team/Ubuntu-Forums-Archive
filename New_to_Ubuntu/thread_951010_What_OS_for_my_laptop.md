### Post by Paqman on 2008-10-17
With only 256MB or RAM i'd go for a Linux distribution with a lightweight desktop, such as Xubuntu. Neither Gnome nor KDE will run well on that spec, but Xubuntu's XFCE desktop should be quite happy. XFCE has some built-in desktop bling like dropshadows and transparency.

### Post by snowpine on 2008-10-17
I have a Dell Latitude Cpx, which is a slightly older version of your laptop. I used Crunchbang on it for a while with good success (Ubuntu + Openbox), so I think Crunchbang is my #1 suggestion for you. It has a really thoughtful selection of default applications, so I had a pretty easy time getting up and running with it. I also tried Ubuntu, Xubuntu, and Fluxbuntu on that computer (all of them worked, though I had to use the Alternate CD for Ubuntu), and Crunchbang was my favorite of those four.

Currently, I've left the Ubuntu family on this particular laptop, and I'm dual booting SliTaz and Debian LXDE. Both are lighter and faster than any of the Ubuntu variants I've tried. 

Don't worry too much about the wireless thing. Ndiswrapper can be installed in every distro I've ever tried, and then once you have diswrapper, the instructions are the same no matter which distro you're using.

Good luck!
