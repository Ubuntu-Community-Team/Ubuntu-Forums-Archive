### Post by Gryllida on 2011-07-02
When you delete an item it is moved to the Trash folder, where it is stored until you empty the trash. Items stored in the Trash folder can be restored to their original location if you decide you need them, or if they were accidentally deleted.

[https://help.ubuntu.com/11.04/ubuntu-help/files-delete.html](https://help.ubuntu.com/11.04/ubuntu-help/files-delete.html)

### Post by Gryllida on 2011-07-02
More accurate link:

If you deleted a file in the File Manager by pressing `Delete` or clicking the `Move to Trash` option in the right-click menu, the file should be in the Trash. This article explains how to recover a file you deleted.

### Post by abybaddi on 2011-07-02
For getting back the network manager you have to add the notification indicator or something like that.:rolleyes: To get the themes you have to add 'em again:
1.: Right click on desktop-> change desk. back.
2.: themes->Select any theme [ I said any;) ].
3.: customize->A new window would open.
4.: Select the appropriate items for your theme. [window's borders, icons, etc.]
5.: Click ok.

If this doesn't work install metacity and gtk2-engines...
