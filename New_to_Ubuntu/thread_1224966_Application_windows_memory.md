### Post by niteshifter on 2009-07-28
Hi,

You may expect that, but others may not. Hence most desktop environments follow a simple rule set with placement behavior embedded in the application (let the DE decide, center, on parent, etc.)

But this can be overridden with compiz and/or by application (if it supports it). In compiz setting manager (System -> Preferences -> Advanced Desktop Effects), that would be under Window Management, Place Windows, Fixed Window Placement tab.
