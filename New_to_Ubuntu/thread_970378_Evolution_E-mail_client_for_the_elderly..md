### Post by candtalan on 2008-11-04
I help a lot of elderly friends (and I am retired....).

Firstly, make sure that she is not using the first user account - the one which was initially set up at installation! It has rights for use of sudo. 

After installation, create another user (mum's name), and make sure it has limited rights, you can choose the facilities. Also notice that menus can be reduced in content and panels (top and bottom I think both) can be locked to prevent accidental 'loss'. Most of this can be done while logged in as the 2nd user, it is not as if you are seeking actual security, just a customisation towards a minimal set of stuff.

I would expect an app such as evolution has similar facilities - menus customised reduced, and maybe much more.
