### Post by Bucky Ball on 2014-07-28
Welcome. You go here:

[http://support.brother.com/g/b/downloadtop.aspx?c=us&lang=en&prod=mfcj4310dw_us](http://support.brother.com/g/b/downloadtop.aspx?c=us&lang=en&prod=mfcj4310dw_us)

... choose 'Linux' and the .deb file. Install. Then go to 'Printing' and 'Add Printer'.

Linux is very well supported by Brother.

* Ninja-ed! As in the previous post. Instructions for install are on the site. If you use the brinstaller it is straightforward. Go to Software Centre and make sure you have 'cups' installed. Should be installed by default, but best to check.

If drivers are not installed by default, not in Software Centre or 'Additional Drivers' always check the manufacturer's site. You might be pleasantly surprised. Not all of them ignore Linux users, and Brother is one of the good ones (to this point).
