### Post by PriceChild on 2008-04-22
Downgrades are explicitly not supported by Ubuntu.

You probably don't need to reinstall, if you paste your xorg.conf here, ie the output from the following command:
```
cat /etc/X11/xorg.conf
```Then we'll probably be able to get it sorted.
