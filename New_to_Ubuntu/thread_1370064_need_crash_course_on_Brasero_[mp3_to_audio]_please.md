Open Brasero. Make an audio cd project. Navigate to folder with mp3s, drag to right most pane. Don't exceed size of cd burning. When satisfied with mix, push burn. Put in cd player. Fin.

### Post by Bodsda on 2010-01-01
Hi,

If you load up a terminal, then type in the following command

```
brasero
```

On the left hand side is a list of options, choose 'Audio project'. On the next screen, click the plus sign in the top left and you will be able to select files to burn. When your done, click the burn button in the bottom right hand side.
