### Post by 3rdalbum on 2009-06-11
Let's clear up some misconceptions that have been thrown at you.

As you are using Ubuntu 8.04, there is no problem with the Intel graphics support. There are problems with Intel graphics on 9.04, but not on 8.10 or 8.04.

The Intel driver is already installed and in use on your computer. If it wasn't, then Tux Racer wouldn't start up.

Are you using Tux Racer, Extreme Tux Racer or PlanetPenguin Racer? I think ETR is the latest version of the game.

Intel graphics are not very powerful at all; if possible you should put a real graphics card in and you'll be much happier.
