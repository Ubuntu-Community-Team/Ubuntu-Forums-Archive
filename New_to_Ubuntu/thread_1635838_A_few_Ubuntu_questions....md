1. Running VB on a netbook with an Atom processor is likely to be so poor in performance that you would not want to use it. i'm not sure you can even get it to run, but others have hopefully tried and either succeeded or failed to confirm.
 
2. Wine is used to be able to run "SOME" Windows software. It is not able to run most Windows software but some applications run well, some run somewhat well and others will not run at all. Online seach or forum search can confirm for the application you want to use.
 
3. Not sure
 
4. iTunes will not run but you can sync your iPod on Ubuntu using another media player.
 
5. There are many messenger programs and pidgin or empathy can be used. But not Windows messenger.
 
6. Intel drivers are built into the kernel. Perhaps searching the forum for other eee posts and graphics will turn up info on getting the best performance from your graphics processor.
 
Since Linux is quite different from Windows and your needs are somewhat Windows based, perhaps just booting into Ubuntu when you need or want to would better allow you to use what works best? A dual boot setup (both operating systems, selecting whichever at startup) would probably serve you better instead of trying to run both simultaneously with a VB setup because of the RAM and processing power required to do both. Just food for thought...and best of luck getting it up and running how you like.
