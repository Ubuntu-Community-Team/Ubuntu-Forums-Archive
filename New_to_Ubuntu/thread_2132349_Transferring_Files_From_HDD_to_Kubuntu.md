### Post by grahammechanical on 2013-04-04
You can access files on the hard disk when using Ubuntu/Kubuntu in a live session. If it is a USB memory stick then you should be able to save the files on the USB stick but test it out first. An alternative that I have used when running UBuntu from a Cd is to create another partition and copy the files into that and them copy them again when you have install Kubuntu.
