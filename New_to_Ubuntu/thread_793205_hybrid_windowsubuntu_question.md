Greetings! Ubuntu occasionally has problems with wireless networking (recognizing and talking politely to the hardware), but I've never seen a situation where wired networking was an issue.

Printers have been an issue in the past, but most any modern printer can be used on Linux with little or no problems.

### Post by Mr A Mouse on 2008-05-13
> **uhhuh said:**
> I dont have my own install key are you saying if i use theirs they wont be able to install it too??

Windows install keys can only be used on one computer. (There are exceptions, but that's a whole different discussion.) Yes, in most situations, if you use your friend's install key, he will not be able to use it.

> also is there a way around the squashfs error w/o redownloading ubuntu??

You can *try* running the live CD with "ide=nodma" (no quotes) as a boot option--i.e. when you see "boot:" on the computer while booting, type that in. 

If that doesn't work, I don't know if there's any cure short of downloading a new copy of Ubuntu.
