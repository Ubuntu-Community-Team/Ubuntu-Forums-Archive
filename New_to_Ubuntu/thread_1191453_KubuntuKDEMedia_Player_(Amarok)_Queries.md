### Post by suffice on 2009-06-19
well in kubuntu there is no links for updates.

basically in the when an update for a program is available it will pop up in the 'system tray' and then enter your password and all installed apps will get updated.

for cool looking things the site to goto is [www.kde-look.org](www.kde-look.org).  things get released everyday.  make sure you are looking for stuff that works in kde4.

for the most part the widgets and things can be directly downloaded from kde-look.org from the add widgets dialogue.  goto the cashew at the top right and the add widgets (or unlock first) and then download new native plasma widgets.

the whole plasma space is still relatively new so there is not a whole lot of great widgets but there are some good ones.  i use cwp weather...its a decent multi day themeing.

basically things should work for them most part when you plug it in...but getting a real broadband connection would be the best thing, as  basically the whole system comunicates with the net on a pretty regular basis.....

get the the old ms program install update process out of your head ( ie www.firefox.com>download exe> double click> next next next next next > start) .....

and get used to the terminal and sudo apt-get install firefox!
