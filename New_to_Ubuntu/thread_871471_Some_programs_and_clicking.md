### Post by silkstone on 2008-07-27
OK, OK! It's sunny and everyone's playing outside!

In Firefox, Edit > Preferences > Advanced > Tick 'Use autoscrolling'. That will get your middle mouse button working in Firefox.

As to the rest, I can't answer your question about gPhoto. I use The GIMP and you can install the dcraw plugin for this, which lets you open RAW files. There's also a ufraw plugin if you prefer - both from Synaptic.

I believe you can install Ubuntu from wubi, but I've not done it this way.
