### Post by Backharlow on 2008-08-03
four things:

1. You could additionally simplify the interface of Firefox via a community developed theme.
2. Ubuntu desktop has an accessibility menu tucked away under System, with a screen reader, and similar I believe.
3. Be sure to set Gnome to show BOTH text and icons for all menus. This will really reinforce associate meanings of things for the unfamiliar.
4. Put application launchers smack in the middle of the desktop and "stretch" the icons out really big.

5. Gnome "workspaces" may confuse the elderly. not totally sure it seems you could loose track of things there if it was all really foreign.
