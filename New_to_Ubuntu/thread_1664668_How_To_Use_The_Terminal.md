### Post by hansolo4949 on 2011-01-11
One quick way to get to the terminal and back is to press ctrl-alt and F1. You will have to enter your account name and password. To get back to the GUI gnome (xfce, lxde whatever interface you're using) press ctrl-alt F7.

To run a command as root ( you can do just about anything to your computer as root) type sudo before the command. to install some software, type sudo apt-get whateverthesoftwarenameis. I hope that helps!

hansolo4949
