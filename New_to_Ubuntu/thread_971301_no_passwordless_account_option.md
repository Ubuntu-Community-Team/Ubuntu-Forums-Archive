### Post by damis648 on 2008-11-05
> **Zopiac said:**
> how so? i want to set up an account for my mom, where she uses facebook and plays mahjong all day. If THAT is risky to a computer, this is a sad point indeed....

If that is all you want, set up an account with a password you can sticky note on her desk or something. Then, go to System>Administration>Login Window and to the Security tab. Check the "Enable Automatic Login" box and select her username from the list. This way, the only time she'll ever need the password is for administrative tasks, not logging in.
