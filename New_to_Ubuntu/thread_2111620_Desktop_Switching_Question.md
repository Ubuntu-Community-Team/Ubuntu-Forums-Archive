### Post by kanikilu on 2013-02-04
Lubuntu/LXDE uses the OpenBox window manager, so you can use the "obconf" program to set the number of virtual desktops. However, in searching how to make 2 "rows" (grid layout), it seems it's not very easy or intuitive:

[http://openbox.org/wiki/Help_talk:FAQ#Additional_Virtual_Desktop_Grid_Information](http://openbox.org/wiki/Help_talk:FAQ#Additional_Virtual_Desktop_Grid_Information)

As for windows spanning over to the next virtual desktop, I'm not sure if that's possible...
