### Post by Old_Grey_Wolf on 2011-08-20
The file is probably in a different location now.

You could use the "Places" > "Search for Files..." menu enter trash-full in the "Name contains" box and change the "Look in folder" to File System.

There will probably be several files and folders listed. You will need to figure out which one to use depending on the theme you are using.
