### Post by buzzingrobot on 2014-11-16
Best to ask about specific models, especially to determine if a vendor-supplied driver is needed and available.

All-in-one devices, and scanners, seem to more frequently be problems on Linux.  Some are Windows-only because they rely on code *within* Windows to actually print.
