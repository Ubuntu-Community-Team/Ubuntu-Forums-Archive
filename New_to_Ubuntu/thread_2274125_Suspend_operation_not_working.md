### Post by leunam12 on 2015-06-11
Sounds like a video card issue.

Possible solutions:
On Ubuntu: see if there is an available additional driver when you go to "Software and Updates" the last tab on the right, if there is, install it and see if the problem persists.
On Windows: go to the laptop's or video card manufacturer's website and see if a later driver or a patch has been issued to solve this problem.
