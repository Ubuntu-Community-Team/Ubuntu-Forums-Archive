### Post by RomeReactor on 2008-07-13
Hi. Look for a README or INSTALL file in the extracted folder, which will tell you how to compile/install the program. If the instructions are to compile it, you'll need to install the necessary tools to do so:
```
sudo aptitude install build-essential
```

The website where you got the program likely has information on how to install the program, along with which dependencies to install. Note that not all programs you get as compressed files are source code; some, like Firefox, are precompiled, and you can run them form the extracted folder.

However, if you don't absolutely need the newest version of the program, you can probably find a .DEB package in [GetDeb]("http://www.getdeb.net/").

### Post by RomeReactor on 2008-07-14
> **coolman121 said:**
> exactly what you quoted there are three variants 2 of them dont work well but the third i cant get to play/show up on ubuntu (it is the best)

**vbaexpress** and **visualboyadvance-gtk** are graphical interfaces to **visualboyadvance**; the entries you see in the 'Games' section of the menu are made by these. **visualboyadvance** is itself a command line program, and doesn't make a menu entry. You can launch it by opening a terminal and writing:
```
VisualBoyAdvance /PATH/TO/FILE.GBA
```
where **/PATH/TO/FILE.GBA** is where you have the game stored.
