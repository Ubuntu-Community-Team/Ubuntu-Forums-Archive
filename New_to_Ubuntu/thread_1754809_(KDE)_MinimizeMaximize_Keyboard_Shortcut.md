### Post by decktrio on 2011-05-10
I'm trying to find the existing keyboard shortcuts (in KDE 4.6) to minimize and maximize the active window. I haven't come across anything that works, as of yet, and the Wikipedia's [Table of Keyboard Shortcuts]("http://en.wikipedia.org/wiki/Table_of_keyboard_shortcuts#Window_management") doesn't have anything in the appropriate boxes. **Does KDE not come with default shortcuts for min/max?**

If I need to create my own custom shortcuts using the KDE Control Module, **does anyone have any pointers/tips?** If I need to make my own shortcut, then ideally I'd have the active window minimize with "Meta + [Down] ", and maximize with "Meta + [Up] ", so that my fingers memorize the same movement that they use with Windows.

Any help from the community would be great. Thanks.

### Post by mutley89 on 2011-05-10
Alt+F2 and type "systemsettings" then click on "Shortcuts and Gestures".  The window management shortcuts are under "Global Keyboard shortcuts", then select "Kwin" from the drop-down menu.  By default maximise is Alt+F11 and minimise is Alt+F10.
