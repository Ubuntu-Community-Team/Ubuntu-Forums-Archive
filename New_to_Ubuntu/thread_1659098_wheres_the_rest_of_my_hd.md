### Post by Vermind on 2011-01-03
Did you install ubuntu from windows under windows?
If so, then the ubuntu disk is limited to the size you chose in the beginning.
Here's how to fix that through Windows: [http://ubuntuforums.org/showthread.php?t=441773](http://ubuntuforums.org/showthread.php?t=441773)
If you installed ubuntu normally, you can reboot with a Ubuntu live cd and use the gparted or "Partition editor" to resize your Ubuntu partition to any size you want. Moving partitions takes very long, growing them to the right is much faster.
