### Post by mikeeve on 2008-10-14
You cannot run the linked (executable) program from windows under linux (unless you use the windows emulator, wine).

You may be able to recompile and link a windows "c" program that uses console in/out without change. If the program uses windows specific routines,  uses a gui interface, then no.
