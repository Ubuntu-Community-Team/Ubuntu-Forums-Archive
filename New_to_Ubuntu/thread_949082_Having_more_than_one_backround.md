### Post by drs305 on 2008-10-15
I use wallpapoz and have a separate wallpaper on each desktop. I have it in my Sessions Startup tab. The command to start it is "daemon_wallpapoz" although to view it's options you can just run 'wallpapoz'.

[http://wallpapoz.akbarhome.com/]("http://wallpapoz.akbarhome.com/")
Added: The developer says he is not going to create a new version but the old one is still available and is the one I'm using.


Here is a location to get the .deb:
[http://www.getdeb.net/app/Wallpapoz]("http://www.getdeb.net/app/Wallpapoz")
