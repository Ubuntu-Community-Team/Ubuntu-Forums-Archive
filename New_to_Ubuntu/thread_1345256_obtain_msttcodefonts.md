### Post by mikewhatever on 2009-12-03
To obtain internet access, you have to pay something to an ISP, or, if it's a notebook, find a hot spot. When that is settled, make sure Multiverse repository is enabled in System->Admin->Software Sources, then open Synaptic and install mssttcodefonts package.
