### Post by jamesrfla on 2008-09-29
First you need to enable remote desktop on Ubuntu. Go to System>preferences>Remote Desktop. Then check some boxes. Then you need to get a VNC viewer for your windows computer. To get a VNC viewer go to [http://www.realvnc.com/products/free/4.1/index.html](http://www.realvnc.com/products/free/4.1/index.html) Hope this helps.
