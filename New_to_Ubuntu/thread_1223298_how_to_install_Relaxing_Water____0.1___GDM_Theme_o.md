### Post by Bigtime_Scrub on 2009-07-26
When you download it is it in tarball format? If it is right click on it and select "Extract here."

After that right click on an open place on the desktop and select "change desktop background."

There should be a tab for themes there, click that.

There is a window with many themes, simply drag the extracted theme into that window and click "apply."



Thats it, enjoy your new theme.

### Post by okey666 on 2009-07-26
This is a theme for the login window. Go:
System>>admin>>login window
click on the local tab and drag the "tar.bz2" file into the box with the other themes in it. Alternatively you can click "install" and navigate to the theme that way.
