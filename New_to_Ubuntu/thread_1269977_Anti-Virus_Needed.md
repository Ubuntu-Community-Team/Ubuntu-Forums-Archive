### Post by Paqman on 2009-09-18
> **carl_76 said:**
> 
Is there a need for an anti-virus program?

Nope, not if your system is kept up to date.

Also, Firestarter isn't really a firewall. It's just a config tool for Ubuntu's built-in firewall. The default configuration that Ubuntu installs in is perfectly safe, so for the vast majority of users Firestarter is a a bit pointless. I haven't used it for years. You'd only get any benefit if you'd installed some software that was a potentially security risk (eg: a server) and needed to check your settings.
