### Post by WRDN on 2011-07-06
I would download and use the 64 bit image (ubuntu-11.04-desktop-amd64.iso).

Your CPU has a 64 bit instruction set, so no problems there.
You could use the 32 bit version as you have less than 4Gb RAM. However, using the 64 bit version, you could add more RAM in the future with no worries about addressable memory.

Further, depending on your computer usage, some applications may perform faster with a 64 bit OS (e.g. compilers).

The only reason I would recommend 32 over 64 bit is extensions such as Adobes Flash, which is currently more stable in the former environment (though 64bit flash stability is improving).
