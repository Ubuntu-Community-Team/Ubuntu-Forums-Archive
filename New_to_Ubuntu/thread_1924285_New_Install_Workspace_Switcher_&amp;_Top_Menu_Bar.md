### Post by grahammechanical on 2012-02-12
This is a guess. You have just installed and you are in Ubuntu 2D and therefore you will not see the effect that you are looking for. I have just checked this out on my system by logging into Ubuntu 2D.

You need to run Additional Drivers and activate a proprietary driver. Then log out and at the log in menu click on the cog icon and select Ubuntu. Then you will be in Ubuntu 3D. And you will see that set of menus in the top left of the top panel.

Unless you have a low specification computer that will not run Ubuntu 3D, then Ubuntu 2D is what you get as the fall-back mode.
