### Post by Holger_Gehrke on 2023-07-01
You don't mention which version and flavour of Ubuntu you're using. On Ubuntu 22.04 with Gnome using Wayland, you should have gestures available. Three finger swipes are used for switching workspaces. This is according to the [official documentation]("https://help.ubuntu.com/lts/ubuntu-help/touchscreen-gestures.html.en"). I can't test it myself since I'm using neither Gnome nor Wayland. There is an extension for the Gnome shell which is supposed to give you more control over what gestures do named 'Gesture Improvements'. You can install it using the Gnome Shell Extension Manager.
