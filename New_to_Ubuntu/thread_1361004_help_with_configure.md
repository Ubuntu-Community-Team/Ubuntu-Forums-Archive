Can someone tell me how to get into the source code directory to run ./configure to compile a program

---

### Post by slughappy1 on 2009-12-21
Most of the time it is just the folder where you downloaded the tar.gz and extracted it to. For example, I have mine download everything to /home/username/Downloads. So open a terminal and type```
cd /home/username/Downloads/FOLDER_YOU_WANT_TO_COMPILE

``` then ```
./configure
```
