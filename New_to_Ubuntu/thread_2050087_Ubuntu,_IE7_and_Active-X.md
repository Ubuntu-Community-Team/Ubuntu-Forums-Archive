### Post by Mark Phelps on 2012-08-30
> **prsills said:**
> ... I just want everything to work in Ubuntu.

Noble idea -- but unrealistic.

Many of the IE-only applications expect to see IE v9 these days, which can't be installed or used in Wine.

IE, unfortunately, is one of the few apps that simply don't work in Wine, except for really old versions.

If you need to rely on using IE, then you're going to be stuck either dual-booting or running MS Windows in a VM.
