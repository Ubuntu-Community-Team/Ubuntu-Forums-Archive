1) If it's available in regular ubuntu it's available in studio edition. It is basically regular ubuntu with packages added.

2) the repos are the same the only difference I believe is that Studio edition has more repos enabled by default. 

3) If you mean can you get enterprise support the answer should be yes because it is an official derivative. If you are asking does it get the same software updates the answer is yes. 

Actually you can install regular ubuntu and install the Studio packages from the repos and the result will be ubuntu studio edition.
