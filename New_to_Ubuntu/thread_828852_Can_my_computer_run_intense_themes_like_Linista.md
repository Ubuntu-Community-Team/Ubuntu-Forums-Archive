### Post by ad_267 on 2008-06-14
Xubuntu uses a different window manager than normal Ubuntu so the window themes will need to be for XFCE. Have a look here: [http://www.xfce-look.org/index.php?xcontentmode=420](http://www.xfce-look.org/index.php?xcontentmode=420)

I found one example here: [http://www.xfce-look.org/content/show.php/Vista+Basic+Xfwm4?content=78371](http://www.xfce-look.org/content/show.php/Vista+Basic+Xfwm4?content=78371)

Both ubuntu and xubuntu use gtk so you can install the same gtk themes and the linsta theme should work ok. Gtk themes can be found here: [http://www.xfce-look.org/index.php?xcontentmode=100](http://www.xfce-look.org/index.php?xcontentmode=100)

You probably won't want be able to run compiz-fusion and emerald with specs like that though.
