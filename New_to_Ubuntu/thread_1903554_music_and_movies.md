### Post by kalfusisagod on 2012-01-02
Depending on which ubuntu version you're using, you can use the built in Bit Torrent client in Ubuntu 11.10 (it's named "Transmission BitTorrent"). Just goto google, type the movie or album + torrent you want and open that name-of-file.torrent file into the bit torrent client. For example, if I wanted to download Revolution OS, I'd type "Revolution OS torrent" into google and find that .torrent file and download that and have the Transmission BitTorrent open that file and download it.
