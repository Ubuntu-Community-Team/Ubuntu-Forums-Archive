### Post by uRock on 2011-07-21
If you are already running 10.04 and running your regular updates, then you do not need to worry about upgrading anything. 

The updated release versions for 10.04 are mostly to prevent people who are installing 10.04 from having to run so many updates. 10.04.3 should have the current kernel and other security patches, so when an install is run there will be minimal packages to be updated.

### Post by AlphaLexman on 2011-07-21
In the 'Update Manager' it is the 'base-files' package that marks your system from 10.04.2 LTS -> 10.04.3 LTS.
```
Changes for the versions:
5.0.0ubuntu20.10.04.3
5.0.0ubuntu20.10.04.4

Version 5.0.0ubuntu20.10.04.4: 

  * /etc/lsb-release, /etc/issue, /etc/issue.net: Bump version number to
    10.04.3 in preparation for the point release.

```
Just install the updates, and you'll be good.

You don't need to upgrade anything. As I posted before you already have 10.04.3 if you have been installing the regular updates. To prove it to yourself type in:

```
lsb_release -a
```

The next LTS release is 12.04 which will come out next April. To get that you will definitely need to do a full upgrade or reinstall. You may find it a bit of a shock to the system as gnome 2 will be completely gone. The default DE will be unity running on gnome 3.
