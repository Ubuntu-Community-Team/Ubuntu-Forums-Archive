### Post by whitethorn on 2011-08-28
This is the ubuntu forums, usually you should try to keep your questions to the proper forum. You can then get better support.

[http://forums.opensuse.org/](http://forums.opensuse.org/)

I usually use unetbootin to get bootable linux usb sticks. Depending on if you have windows or some sort of linux installed you can either download it from whatever Repositories or from this webpage.

[http://sourceforge.net/projects/unetbootin/](http://sourceforge.net/projects/unetbootin/)

Insert the usb stick start the program, choose what distribution and version you want, make sure it uses your usb stick and let it run. Afterwards you should have a bootable usb stick.
