### Post by roger_1960 on 2012-12-14
Hi

qcad is OK for basic stuff but not as refined as autocad - some of the trim commands and offsets etc are not there, but there are long ways round these things.  Printing is completely different to autocads paperspace/modelspace etc. so that will take a bit of learning. It is however free and quite good if just for occasional use but not quite the same if you are used to autocad.
