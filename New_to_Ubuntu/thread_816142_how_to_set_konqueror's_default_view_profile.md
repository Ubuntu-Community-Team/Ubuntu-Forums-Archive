### Post by aysiu on 2008-06-04
Change the launcher command to ```
konqueror --profile filemanagement
```

### Post by aysiu on 2008-06-05
If you're launching it off the panel, right-click the launcher to modify it.

If you're launching it off the KMenu, right-click the KMenu icon to edit the menus.
