### Post by Tony Flury on 2009-02-08
Unlikely - unless all your docs are on a seperate partition (which did not get trashed.

For a start Linux uses a different disk format (ext3 or ext4, compared to NTFS for Windows) - so Linus will reformat the disk (or at least the partition) to get the right format 

If your Windows docs are in a different partition - then you might be in luck - as Linux can read NTFS partitions
