### Post by bab1 on 2013-08-19
> **albert9 said:**
> im ne to the ubuntu world and trying to learn shell any one know were to start

There are plenty of web sites with information for beginning Linux users.  But it might be helpful for you to define what you mean by *"shell"*.  

There are many versions of the command line interpreter that is generically called *the shell*.  There is the original Bourne Shell (sh)  or the Bourne Again Shell (bash) Or C Shell (csh), etc.

Or;  Could you mean how the operating system works using commands and tools via the terminal?  There plenty of web sites for that too.

I don't buy books anymore as most all of the information is on the web.

For example, for the bash shell there is all of this from a [**[COLOR="#0000FF"] _Google search_ [/COLOR]**]("https://www.google.com/search?q=bash+shell&btnG=Go!#bav=on.2,or.r_qf.&fp=fd05ed83d56ff9e4&q=bash+shell") .

I think this is pretty good for a beginner:  [**[COLOR="#0990FF"]_Bash Shell Programming in Linux_[/COLOR]**]("http://arachnoid.com/linux/shell_programming.html")

For Debian/Ubuntu Linux system management there is the Ubuntu [[COLOR="#FF8C00"]**_Using the Terminal_**[/COLOR]]("https://help.ubuntu.com/community/UsingTheTerminal") site.

For a more definitive site there is the [[COLOR="#ff00ff"]**_Debian-Reference_**[/COLOR]]("http://www.debian.org/doc/manuals/debian-reference/").

And of course you can always ask questions here.  ;-)
