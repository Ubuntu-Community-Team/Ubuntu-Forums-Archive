### Post by Ctrl-Alt-F1 on 2010-10-12
I don't think that running a default installation will make your system unbearable.  There are a few conveniences that are installed with a default installation that I find quite nice.  However, you might notice a slight speedup if you use a minimal install.  I have noticed before.  It's not a big deal unless I want to shave a couple seconds off my startup time.

Another option is to just remove offending packages after you install or to make it so that they don't start with Ubuntu.  I always turn off Evolution Alarm for example, since I currently don't use evolution.

I guess what I'm trying to say is, yes you can be faster than the default install, but in my case it's negligible.
