### Post by BDNiner on 2008-12-22
I believe that Evolution can't handle the different accounts they way you want it to work. Evolution will only send from the account that is set to default no matter what settings you use when setting up your multiple accounts. So you have to manually change the default account everytime you want to change the account the mail is coming from.

Thunderbird will send using the correct accounts but then you give up a lot of functionality compared to evolution.
