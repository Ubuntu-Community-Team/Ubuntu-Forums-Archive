### Post by mastablasta on 2010-11-02
just so you know IF you are not in a hurry you could order one CD for free directly from Cannonical. it will take a while for you to get it but it is free.
 
Another thing to consider is that booting from CD can take a while and may sometimes seems it's not working, because the whole system is read from CD. you could try and put it on USB key.
 
also "it just stopped" is not a good problem description. what just stopps? CD stops spinning? system stops loading? does it write anything? any messages? or you get a completelly blank screen?
 
you also left out most important information from your system: the graphics card. it may be that ubuntu doesn't recognise it. there is a way arround that that will help you boot to vesa graphics mode (not so good, but upon propper install it might be possible to get drivers for the card and it will look much nicer).
 
things you might also have a few issues with:
 
-- if the system is notebook or using wireless - it might be necessary to either instal/enable drivers or use a wrapper that will make windows drivers work in linux
-- sound card recognition - could be solved by drivers upgrade
-- graphics card -- might need additional drivers installed to work propperly - possible after propper install i believe.
 
 
EDIT: also Ubuntu updates are weekly i believe and really big in volume. 80 - 200MB
 
hope you will be able to get better connection.
