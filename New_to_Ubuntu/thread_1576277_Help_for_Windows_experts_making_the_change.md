### Post by Sef on 2010-09-17
>  I would look up, for instance "Start, All Programs" and it would tell me how to browse for installed programs in Ubuntu. 

Applications > Ubuntu Software Center > Installed Programs

> Come to think of it, how do I run terminal? If I had the guide I want, I would look up "Start, Run... cmd".

Applications > Accessories > Terminal 

or 

Alt + F2 > gnome-terminal
