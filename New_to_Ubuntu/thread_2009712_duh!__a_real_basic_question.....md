### Post by wildmanne39 on 2012-06-24
Hi, you can find out what version you have installed with:
```
lsb_release -a
```
I recommend that you backup any data you want to keep, reformat the partitions you are going to use and do a fresh install of 12.04 so you do not have any issues from upgrading.
