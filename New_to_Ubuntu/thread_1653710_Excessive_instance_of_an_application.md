### Post by xyepblra on 2010-12-27
Good time of the day to everyone!
I like my desktop wallpapers to be automaticaly changed once in a while, so I installed Cortina. At the same time, I like my laptop to start up with all applictions necessary for my work, so I checked "Remember all working applications on power-off". Cortina gets added to the auto-start-up list on every session, so the system gives me an extra instance of Cortina on each system start-up. Can anyone tell me what should I do to make just one instance of Cortina to be launched without unchecking the "Remember all working applications on power-off"?

---

### Post by XeonBloomfield on 2010-12-27
System > Preferences > Startup Applications

In tab "Options" uncheck "Automatically remember running applications when logging out".

In tab "Startup Programs" check if Cortina is on the list and if not click "Add" button.

In "Add" window you need to write NAME of the item on the list, COMMAND what will be run and COMMENT for this item on the list.
