### Post by mc4man on 2014-12-13
Assuming on recent ubuntu, ie. not 12.04.x
You would disable the option as shown in screen or by below command
```
gsettings set org.gnome.nautilus.preferences enable-interactive-search false
```

The interactive search is very useful while in folders so re-enable if you miss it
