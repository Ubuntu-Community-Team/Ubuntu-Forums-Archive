### Post by matt-fender on 2010-06-22
System > Preferences > Startup Applications

Try disabling the non OpenGL one, and select the one you want. 

OR, 

Run the NON OGL version, and go through the configuration and disable startup

THEN open the OpenGL version and enable startup
