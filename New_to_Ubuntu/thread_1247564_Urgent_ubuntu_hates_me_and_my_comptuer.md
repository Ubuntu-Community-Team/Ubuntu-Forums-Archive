### Post by 3rdalbum on 2009-08-23
Remove Ubuntu from within Windows (it's in the Add/Remove Programs control panel).

Boot up from the Ubuntu CD and choose "Install Ubuntu".

Follow the directions. Make sure that you opt to resize the existing partition. Better yet, install Ubuntu 9.04 instead of 8.04 (8.04 is quite old) and make sure that you **move the slider** on the bottom bar graph on that same screen **to the left** to allocate some space for Ubuntu.

After the installation is done, reboot and you should be able to get into Ubuntu.
