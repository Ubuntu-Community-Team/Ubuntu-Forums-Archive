### Post by jrothwell97 on 2009-08-26
It is not a ZIP file.

WinRAR treats it like a ZIP file, which is not what you want

Right-click on the icon and select "Open in ImgBurn/InfraRecorder/whatever". Insert a blank CD and select burn.

Simply burning the file itself to the CD will not work (i.e. copying it in Windows Explorer) - you have to use that software to burn its contents to the CD.

Good luck.
