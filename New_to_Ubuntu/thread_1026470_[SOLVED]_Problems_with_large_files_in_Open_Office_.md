### Post by mikewhatever on 2008-12-31
You may try increasing the amount of RAM OO is allowed to use, provided there is enough RAM on the system. I don't know if all Dells 640m have much RAM, nor how much, so don't do it if it's less then 512 MB.

Open the Writer, then Tool->Options->Memory and increase the memory for both OO (100+) and object (10+).

### Post by drs305 on 2008-12-31
I routinely open documents which are hundreds of pages long and have never had a problem with any OpenOffice version I have used. It is not a problem with the OO program itself if you have a difficulties opening such files. There is a memory section in Tools, Options, OpenOffice.org for tweaking memory usage.
