### Post by caro on 2009-01-11
This seems to be common lately.  Try this:

When Firefox goes into full screen mode (what you described), press the F11 button twice.  The window should go back to normal.  Then resize the window to make it smaller.  Hit the "maximize" button on the top of the window (between the minimize and close button) and your window should be back like you want it.  Exit Firefox and relaunch.  Your settings should be saved.
