### Post by nickleboyblue on 2011-04-24
OGV is one of the media formats from the Ogg foundation.  Since it is open standard and is not restricted by patents, and because it works so well, it is the default file encoding for Cheese.  You might be able to change the format cheese encodes to when saving a video, but I am not familiar with that.

As for the camera not working elsewhere, that is an unrelated problem.

### Post by oracle2b on 2011-04-24
You can convert the .OGV file to a format you're more familiar with using [WinFF]("http://winff.org/") or Arista - Multimedia Transcoder for GNOME with > sudo apt-get install arista.
