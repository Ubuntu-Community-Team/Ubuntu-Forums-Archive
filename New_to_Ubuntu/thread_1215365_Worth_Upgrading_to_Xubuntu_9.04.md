### Post by Paqman on 2009-07-17
I think it's worth upgrading for Ext4 alone. If you're running Xubuntu you're probably doing so for a bit of extra speed, so a faster boot has to be good. It should be able to get at least 25% off your boot times.

You'll need to partition manually to get Ext4, which means doing a reinstall rather than an upgrade. Burning a LiveCd or USB and installing from it also has the advantage that you can check your hardware on 9.04 before installing.

### Post by 3rdalbum on 2009-07-17
Your graphics are not supported by the ATI/AMD proprietary driver on 9.04; only by the open-source driver which probably won't give you 3D acceleration.

If everything is going alright, stick with 8.04. If you need 3D acceleration but want newer software than what's available on 8.04, upgrade to 8.10.
