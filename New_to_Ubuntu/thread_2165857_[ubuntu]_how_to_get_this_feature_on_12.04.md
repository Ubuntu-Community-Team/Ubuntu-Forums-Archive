### Post by newb85 on 2013-08-06
That image is of the old Gnome 2 desktop environment, which was replaced in Ubuntu with Unity from Natty on.   Gnome 2 is no longer supported, but Mate is, and it does basically the same thing.

If I remember right, there's a good tutorial on installing Mate from the developers' PPA at webupd8.org.  Search for Mate.

Edit: Here's the link to the instructions I mentioned.  [http://www.webupd8.org/2013/04/mate-16-released-install-it-in-ubuntu.html](http://www.webupd8.org/2013/04/mate-16-released-install-it-in-ubuntu.html)
