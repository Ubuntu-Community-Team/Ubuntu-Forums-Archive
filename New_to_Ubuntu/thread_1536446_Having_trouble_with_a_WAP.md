### Post by arubislander on 2010-07-23
Sorry for not having checked on this thread for a day... I've been otherwise occupied.

So now you can access your AP's settings page. Are you doing that wired (do you have a cable running from your computer to the AP) or wireless? If wireless, then it should be broadcasting something, as you can connect to is. In which case all you need to do to connect it to your network is run a cable from your router to your AP. If you are connected to your AP via a wire, then you should rummage in the settings until you se a page that allows you to set the SSID and optionally set it to be broadcast.
