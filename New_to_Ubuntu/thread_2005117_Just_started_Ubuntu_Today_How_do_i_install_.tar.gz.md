### Post by zombifier25 on 2012-06-16
Don't. Ubuntu's repository has all the software you need. Just open up Ubuntu Software Center, search for the apps you want to find and click "Install". .tar.gz files are usually source codes, which must be compiled, and complete newbies should NOT be compiling softwares.
