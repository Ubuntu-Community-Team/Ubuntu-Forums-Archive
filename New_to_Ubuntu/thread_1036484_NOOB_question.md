### Post by HotCupOfJava on 2009-01-10
A very good book is "A Practical Guide to Ubuntu Linux" by Mark G. Sobell. It is extensive and exhaustive, but not very cheap. 

As for running applications, there is usually a Linux equivalent to (almost) everything you would want to run on Windows. Open Office instead of Microsoft Office, GIMP instead of Photoshop, etc. Many come pre-installed (as I am sure you've noticed). Many others are available through the repositories. You can often find what you are looking for by opening Synaptic Package Manager and searching by what type of app you want or by the actual name of its Windows equivalent. Often a Google search will yield reviews of the app you find that may help you decide whether you want to install it or use some other app (reviewers will often say "I thought it was OK but I prefer to use app X"). If you're ever unsure about an application, first check to see if it is in the repositories (because that will be your most pain-free installation option), then check to see what kind of reviews it has gotten.

Hope that helps

### Post by minsf on 2009-01-10
welcome!

for every command there is (almost always) a help page from which you can learn a lot. so if you are curious about some command, type in your terminal
```
 man name_of_the_command
```
these manual pages are sometimes too much for the newbie, but it is always good to peek at them.

to really learn the command line at the newbie level, start from 
[www.linuxcommand.org](www.linuxcommand.org)
then you can continue to the bash guide for beginner and the advanced bash guide in [http://www.tldp.org/guides.html](http://www.tldp.org/guides.html)
by the way, "bash" is the name for the interface that you will normally get in the terminal (bourne again shell). but you will know that after you look at the first link.
 
have fun :)
