### Post by mcduck on 2009-02-22
Not all themes have support for changing colors. Actually the ability to customize colors (without editing the theme's configuration files by hand) at all is a very new feature in Gnome.

The only thing you can do about it is to use some theme that supports this feature, or find a theme that looks nice as it is.
