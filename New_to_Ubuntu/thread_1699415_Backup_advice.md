### Post by tyler9 on 2011-03-03
Hi--I use my Ubuntu home desktop PC for basic tasks like web access, email, misc apps, and mainly writing. So up to now, I've been backing up to CDs--my backed-up files are mostly writing docs, and I have no huge video, music, or image files.

But I'm wondering: for a low-demand user like myself, is using an external hard drive for backup a good idea, or would it be overkill? I'm thinking that keeping a disk image of my main hard drive might be a good idea in case main hd crashes and I need to reinstall; are there any other advantages?
 
I know this is a very basic question, so thx for patience!

### Post by turtle153 on 2011-03-03
Short answer: yes. You can never really have enough space, so if you want to make a disk image, you need a disk as big as all the stuff on your computer. If you just want to back up a few files you won't need a big disk but it might be useful to have one anyway.
