### Post by nrundy on 2012-10-13
There are some settings you can try changing within Firefox to see if it helps the situation.

in Firefox navigate to Edit > Preferences > Content

you'll see Fonts & Colors section. You can experiment with some different fonts and sizes. Also if you click the Advanced button, you can play with enabling/disabling "Allow pages to choose their own fonts, instead of my selections above." 

Some people like using the Ubuntu font on web pages. I generally find it best to let webpages choose their own fonts though. But if there are a lot of proprietary fonts on the page then you might be better off disabling the feature to allow webpages to choose their font because ubuntu probably won't have the proprietary fonts unless you manually added them.
