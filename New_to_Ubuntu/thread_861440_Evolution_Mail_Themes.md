### Post by mc4100 on 2008-07-16
I don't think it's possible to set Evolution to use one specific theme, because Evolution uses the same theme as your other desktop applications.
You can change themes in the Appearance menu, which is found in the "System -> Preferences" menu. 

If I want to find another theme, I go to 
[http://www.gnome-look.org](http://www.gnome-look.org)
Just download the theme, for example, to the Desktop, then open the Appearance manager and click "install". Find the theme file (The one on the Desktop, I think it will have a ".tar.gz" extension)
