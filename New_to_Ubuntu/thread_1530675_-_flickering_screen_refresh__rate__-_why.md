### Post by Zzl1xndd on 2010-07-13
I had a similar issue with one of my monitors and the screen flickering. If you are using an Nvida card just go to System > Administration > Nvidia X Server Settings.  Then Select the Xserver display config. Setting my refresh rate to 60 from auto seemed to correct the problem. Hopefully this helps

Unfortunately Im not sure what might be causing your other issue.
