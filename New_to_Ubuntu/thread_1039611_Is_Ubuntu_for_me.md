### Post by Truefire on 2009-01-14
Hey there newcomer!


I love answering this question, but I'll try not to be too long-winded ;)

Background:
I currently use Vista for gaming, and Ubuntu for everything else,
and I co-admin for a site encouraging people to make the switch to opensource, at [www.easygeek.org](www.easygeek.org) .

I've done some Java programming, and I've learned some Python, so I think I'd be a good advocate for your 'big switch'. 

The Good Stuff:

The first thing to remember when installing a new OS is, well, you are installing a new OS! It's a big deal! You can't expect to install applications like you would in Windows, or update like you would in Windows.

Does this make Ubuntu's installer automatically worse? 
Not if you're willing to try new things - and you seem like a perfect candidate. 

Installing applications in Ubuntu is far <b> easier, </b> and much <i> faster </i> then in Windows, guaranteed. 

For most apps, go to Applications > Add/Remove. Check the apps you want, and hit apply!  

If you want more variety, check out the other Repos (repositories) and them to Software Sources under System> Administration to add a larger library of available applications to Synaptic (like an advanced Add/Remove, also under Administration). 

I suggest you also read up on Apt-get, a super fun+easy terminal app for installing limitless apps with one command.

As for programming, you'll have no trouble there _at all._

Linux/Ubuntu is the basis for the internet, and nearly every programming language out there - Eclipse and Geany are my two favorites, though that other Sun Java one ( I forget the name) is also Linux/Ubuntu compatible.

You will find the expandability/flexibility of Ubuntu/ Linux to be very freeing - I know, it happened to me :D  - and you will realize just how much Microsoft encourages you to overlook, and how Ubuntu compares to Windows in terms of relativity to hardware speed. I'm typing this on a Pentium 3 Thinkpad, brought to life again ( With desktop effects!) by Ubuntu.  

Ubuntu (or Linux in general) will open your eyes to other options that you never thought possible. This realization will further encourage you to find interesting alternatives to the everyday items you take for granted. Everytime you see a DVR, or new piece of electronics ( like the iPhone or Palm Pre') you'll wonder if it does, or can, run Linux (Both already do, in case you're wondering :D ).

It's a different way of doing things - for the better.


PS: Hate the default Ubuntu theme? Welcome to the club. There are hundreds of thousands available for free around places like art.gnome.org .
