### Post by Lars Noodén on 2012-09-25
You could also try Lubuntu or Xubuntu which use the LCDE and XFCE desktops.  You can get them by installing the packages 'lubuntu-desktop' or 'xubuntu-desktop'.  That will add them to your existing system.  If you decide on one and want to get rid of the others, then you can follow these instructions:

[http://www.psychocats.net/ubuntu/purexubuntu](http://www.psychocats.net/ubuntu/purexubuntu)
[http://www.psychocats.net/ubuntu/purelubuntu](http://www.psychocats.net/ubuntu/purelubuntu)

If you really want Windows-like appearances then you could try Sunwah:
[http://en.wikipedia.org/wiki/Sunwah_Linux](http://en.wikipedia.org/wiki/Sunwah_Linux)

I've heard it's very much like Windows XP, but haven't used Windows XP enough to know for sure.  It's usable, but I prefer Lubuntu or Xubuntu.
