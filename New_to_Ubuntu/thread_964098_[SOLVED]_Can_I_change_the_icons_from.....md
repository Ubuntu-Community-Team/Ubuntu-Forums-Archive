### Post by rotwang888 on 2008-10-30
Sure you can.  Right-click on applications and click "edit menus".  From there select internet on the left.  Right-click on Limewire in the list on the right and select "properties".   In the window that comes up click the current icon and you'll be able to browse to the location of whatever you want to replace it with.
