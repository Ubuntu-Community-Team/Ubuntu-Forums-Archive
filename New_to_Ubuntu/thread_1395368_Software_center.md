### Post by keplerspeed on 2010-01-31
Firstly can you confirm what version of ubuntu you are using? Karmic Koala 9.10?

Next ensure you have updated your system. Go to system>administration>update manager, click "check" and install any updates.

Now go to applications>ubuntu software center. Does nothing happen at all? If no window opens at all, open a terminal (applications>accessories>terminal) and enter:

as Psumi recommended. Then paste the output in the terminal here so we can see what is going on.
