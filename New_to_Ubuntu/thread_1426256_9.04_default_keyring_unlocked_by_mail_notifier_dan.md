### Post by 3rdalbum on 2010-03-10
The keyring will remain unlocked for programs that use it. If any new program decided that it wanted to access the keyring, Gnome would ask if you want to allow access.

The keyring daemon does not listen to incoming connections from remote computers. (by default, Ubuntu does not listen to these). Being firewalled from your router would make it doubly-certain.
