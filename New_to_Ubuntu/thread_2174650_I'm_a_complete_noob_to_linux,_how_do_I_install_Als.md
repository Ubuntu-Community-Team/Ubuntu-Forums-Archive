### Post by grahammechanical on 2013-09-15
As far as I know pulseaudio is part of the default installation of Ubuntu. It certainly is in Ubuntu 13.10. Secondly were these Linux drivers or Windows drivers? Windows drivers do not install on a Linux distribution which Ubuntu is.

Did you run a Ubuntu live session from DVD/USB? Did you check that all your hardware was working? Have you tested it now? You might find that things are working better than you think.

According to this link that sound card has been supported by Linux for years. Ubuntu has Linux kernel 3.2; 3.5; 3.8; 3.11 depending on which recent version you are using and not the kernel 2.6 mention in that link. And Ubuntu already uses pulseaudio instead of alsa.

[http://www.linux-hardware-guide.com/2012-11-24-asus-xonar-dg-pci](http://www.linux-hardware-guide.com/2012-11-24-asus-xonar-dg-pci)

Regards.
