### Post by Bölvaður on 2011-04-06
Check System &#8594; Administration &#8594; Synaptic Package Manager
and look for ffmpeg in the small search box on right.

Check if these are installed (green box next to)
gstreamer0.10-ffmpeg
libavcodec52
libavformat52

if you have them all you can do what this guy says

[http://ubuntuforums.org/showthread.php?t=1117283&highlight=dvd+playback](http://ubuntuforums.org/showthread.php?t=1117283&highlight=dvd+playback)
