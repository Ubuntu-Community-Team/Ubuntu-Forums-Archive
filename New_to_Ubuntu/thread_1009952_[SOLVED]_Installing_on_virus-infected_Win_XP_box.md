### Post by Idefix82 on 2008-12-13
You burn the downloaded iso onto a CD and boot from it. There you can choose whether you want to install Ubuntu or try it from the CD first. You might first want to try out whether all the hardware is detected.
When you choose to install, the installer will offer you the option of using the entire hard drive. It will then format it itself, you don't need to do it beforehand.
Hope, this answers your questions.

### Post by Paqman on 2008-12-13
If XP is badly infected and you have your data backed up then you might as well go ahead and format the drive.

Does your friend want to reinstall Windows as well as install Ubuntu? If so, then install Windows first (it's easier). Personally, I would boot into the Ubuntu LiveCD and use the partitioner there to delete the Windows partition, then create new partitions for Ubuntu and Windows. Then do your Windows install, then Ubuntu.

If you're just installing Ubuntu then just dive straight in and tell the installer to use the whole drive, no need to do any preparation ahead of installing. It'll format the drive for you.
