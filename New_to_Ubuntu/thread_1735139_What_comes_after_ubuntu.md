There's nothing in Ubuntu that would limit what you can do with it. Being beginner-friendly doesn't mean it's not suitable for advanced users as well. Of course if you are one of those people who need to be forced to learn anything, then going for a less user-friendly and more demanding distribution might work for you. But in the end you can learn all the same things with Ubuntu, if you just want to.

Go for a minimal Ubuntu install and add the packages and features you need by hand if you feel like you should work for getting a functional setup. :)

(Or if you really want to learn by having to work to get a working system, there's only one choice. And it's not Slackware, Arch, Gentoo or Debian. It's [LFS]("http://www.linuxfromscratch.org/lfs/"), the only distro where you *really* are building and configuring everything yourself. ;))

### Post by k3lt01 on 2011-04-22
In Australia you basically have 3 options if you want to go further with Linux.

1. RedHat.
2. Novell.
3. Debian.

Novell is used by the NSW DET (Department of Education) with Windows. Most universities use either RedHat or Debian for servers and Windows on client machines.

If I were you, because you have already got a start with Ubuntu, I'd have a go at Debian. Once you have a good grounding in Debian and its variants (Ubuntu is a Debian variant) have a go at Suse which is owned by Novell. Then take a look at Fedora (a Free version of RedHat) and move onto Red Hat Enterprise Edition (which you will have to pay for).
