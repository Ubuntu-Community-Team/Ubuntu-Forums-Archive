### Post by LeoSolaris on 2008-05-17
Check the md5sums of the ISO you downloaded. It may be bad. Another point after checking the md5sum is to restart your computer with the CD in (and BIOS set to boot from CD) to test if it will boot to a live desktop.

Another alternative is to boot it from the Live CD and select the USB Hard drive to install it into. Then in BIOS make sure you put boot from USB first. Wubi is still new and may have a few bugs. I do not know, I have never used it.

Hope something in there helps and welcome back!
