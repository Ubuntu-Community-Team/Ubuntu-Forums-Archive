### Post by tom.swartz07 on 2010-01-29
there are some little command line tricks that you could pick up along the way too


```
sudo apt-get <program-name>
``` will install a program without any clicks.

```
sudo apt-get update && sudo apt-get upgrade
``` will update and install any upgrades for your programs


Most of the other beginner stuff is covered in that book.

Have fun!
