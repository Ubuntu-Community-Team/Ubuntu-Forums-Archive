### Post by jimmy the saint on 2009-03-27
Open a terminal.  Go to Edit>Profiles.  Click on New.  select the new profile and click on Edit.  Set it up how you want it and make sure to make it use that profile when it starts using the dropdown box at the bottom right of the profiles dialog.  The transparency is under the effects tab
