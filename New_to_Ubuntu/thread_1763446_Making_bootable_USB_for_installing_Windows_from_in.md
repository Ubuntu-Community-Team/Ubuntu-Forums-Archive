### Post by Mr. Shannon on 2011-05-21
**Running an OEM version of windows in a virtual machine is illegal.**

You need an actual windows disk and matching product key to install windows in a virtual machine.  If you don't have these and you need to run a Windows application you can either buy a Windows disk or try Play On Linux or Wine (windows emulators for Linux).  Actually, Play On Linux is a front end to Wine that makes the installation of software easier.


It's a shame PC manufactures don't give us genuine Windows install disks so we can remove windows and run it in a virtual machine under Linux.
