### Post by Pro-reason on 2008-08-17
> **peterjakey said:**
> how can I access kcontrol in KDE4?
I think you're asking the wrong question.

To access Kcontrol, type "kcontrol" at the command line.  However, bear in mind that it is a KDE 3 program.  If what you want to do is configure KDE 4, the command you want is "system-settings".  It will have "-kde4" suffixed to it if you have the KDE 3 version installed too.
