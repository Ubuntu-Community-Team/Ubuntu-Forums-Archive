### Post by Grenage on 2009-12-01
Welcome :)

Server/workstation is a blurry line; in linux a server is generally the OS without a GUI, installed on server-grade equipment.  Basically whatever you call a server, is a server.

There is a great beginner's guide [here]("http://ubuntuforums.org/showthread.php?t=1052065").  I'd personally recommend just installing things and tinkering about, worry about security once you're done experimenting.  As you're new, you can safely assume that your first install won't last very long; if it doesn't, you weren't experimenting enough. ;)

Install from the CD as normal, you can install extras using *apt-get*.  Any issues you need help with can be posted here as and when needed.

> is it something that would be a little too advanced for someone with no knowledge of servers at all?

Bah, advanced is merely synonymous with a lack of experience; read, learn and ask questions.  It will all fall into place.
