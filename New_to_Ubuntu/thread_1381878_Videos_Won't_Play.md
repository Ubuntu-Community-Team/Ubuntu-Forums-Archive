### Post by llamabr on 2010-01-15
What exactly are you typing?  If you just open a terminal, and type:

```
mplayer
```

you'll only see it for a moment, because it's not playing anything.  If you have a file to play, you want:

```
mplayer filename.mov
```

or whatever.  What are you trying to play?

Likewise when you type

```
vlc player
```

unless you have a file called "player" it's not going to do anything
