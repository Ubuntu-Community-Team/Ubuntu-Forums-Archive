### Post by Frogs Hair on 2012-04-16
I had a situation on 11.10 where compositing would not start the first few times I used classic. This resulted in missing title bars . I used the following command as a work around and the problem fixed its self, so I never looked into it further.  

```
compiz --replace
``` 
After this the add/ remove from panel command changed to Super Key + Alt + Right Click.
