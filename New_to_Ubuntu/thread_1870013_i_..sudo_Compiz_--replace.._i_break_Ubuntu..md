### Post by Seq on 2011-10-26
> **Nixarter said:**
> I took a risk and yanked the battery.  Luckily, it wasn't permanent.

A few points:

[LIST]
[*]Unity is already using compiz, so there is no need to run it manually. You can install compizconfig settings manager to change the settings if you like
[*]Don't run compiz as root. There is no reason to need to elevate it's permissions
[*]Don't use `sudo` with graphical programs, whether compiz or otherwise. If you needed to run a graphical program as root (ex. synaptic), use `gksudo` instead.
[/LIST]
