### Post by Christmas on 2009-08-10
You can try to run the GNOME appearance application (or GNOME mouse properties, whatever it is to change the cursor - I'm using KDE) as root, so:
```
gksudo gnome-mouse-properties
```
Or the equivalent, e.g. **gksudo gnome-appearance-properties** and enter your user password.
