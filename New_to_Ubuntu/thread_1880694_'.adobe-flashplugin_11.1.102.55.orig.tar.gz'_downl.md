Have you tried: 
```
$sudo apt-get update
$sudo apt-get install adobe-flashplugin:i386
```Or Install it using Synaptic.

To Install it directly from [**adobe **]("http://get.adobe.com/flashplayer/")
Download by selection "APT for Ubuntu 10.04+" - Install it with Software Center then run
```
$sudo apt-get update
$sudo apt-get install flash-plugin
```
