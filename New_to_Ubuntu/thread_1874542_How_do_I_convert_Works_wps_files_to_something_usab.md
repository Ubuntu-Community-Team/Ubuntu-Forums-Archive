### Post by Barney-R on 2011-11-03
I'm a comparative newbie, but the only real problem I had was because I hadn't installed the Windows fonts.

Fonts, sizes and formatting were all over the place whenever I opened a .wps file.

If this is the problem, install the restricted extras and you should be ok, then just re-save as .odt or whatever you choose.

You'll lose any graphics, but these can be put back if you've still got the original files.

I realise not everyone would want to do this, preferring to keep a "clean" (open source) system, but it's the easiest way imho.
