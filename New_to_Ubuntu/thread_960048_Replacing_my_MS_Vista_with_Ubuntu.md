### Post by ad_267 on 2008-10-27
I suggest you dual boot with both Ubuntu and Windows first so that you can get to learn Ubuntu before deciding if you want to switch. It can be a big learning curve and cause major issues if you jump straight into using Ubuntu without having Windows as a backup. Once everything is running smoothly then you can delete your Windows partition and go 100% Ubuntu if you want to. I started out with a dual boot myself and soon got rid of Windows, Ubuntu is just better for me in so many ways.

I think everyone else has already answered all of your questions but I'll ad my 2c.

Notepad++
Use gedit with some of the many plugins available. Another option is geany.

OpenOffice is great and version 3 can import docx, xlsx, etc files.

FrostWire is recommended over Limewire.

Rhythmbox, the default music player in Ubuntu, supports ipods automatically, but there can be issues with more recent ipods. Apple does everything they can with each new generation to prevent other applications from being able to work with them.

### Post by estyles on 2008-10-27
Notepad++ - I use this all the time in Windows, but I don't think there's a Linux version.  There are other text editors for Linux, but I don't think there are any as good as Notepad++.  It might run under Wine.  I haven't really tried it, or looked very hard for a replacement, since the stuff I do that is complex enough to require Np++ is done at work on my Work PC, or in a Virtual Machine since I use the VM with WinXP installed to VPN into my work network anyway (for various uninteresting reasons).

iTunes - Try Amarok, I think it's much, much better than iTunes.  If you don't like Amarok, there are some other good alternatives available.  If all else fails, yes, iTunes should work under Wine.

Open Office - I like it personally.  There are some idiosyncracies, though.

I agree with the people that said you should dual boot.  If you've got Vista already, there's not much point in wiping it completely.  Dual-booting is almost easier to set up than removing Vista anyway, and then you always have it if you want to play some Windows-only games or if you need it for any other reason.  There's also the remote possibility that you just can't get Ubuntu to work for you and will want to go back - it happens.... occasionally.
