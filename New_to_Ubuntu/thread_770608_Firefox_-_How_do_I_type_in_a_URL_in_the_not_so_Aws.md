### Post by yanewby on 2008-04-27
I upgraded along with the masses to Hardy Heron and I am enjoying using Firefox 3 (FF3) but one thing is bugging me.

I am in the habit of typing certain URLs without a www.  For instance to access the BBC website I always type bbc.co.uk (as this is how they advertise it) and not [www.bbc.co.uk](www.bbc.co.uk).  If I do this in FF3 it will not allow me to continue to the website until I put either http:// or www. in front of the bbc.co.uk part.  Pressing enter or the "go" arrow do nothing and FF3 just sits there until I add the extra parts to the URL.

How do I stop FF3 from doing this?  Previous versions allowed me to put anything I wanted in the URL bar.

It gets worse when a website URL is something like subdomain.domain.com (and never has www. in front of it).  You then have to type http:// in front of the URL to get the page.  I just want to type subdomain.domain.com and enter to go to the page!

A minor irritation but one that should not be there and there should be a way to override this (if there isn't already).  If you know, please let me know how to override this "feature".

IMO, the AwesomeBar is not as awesome as it would like you to believe!
