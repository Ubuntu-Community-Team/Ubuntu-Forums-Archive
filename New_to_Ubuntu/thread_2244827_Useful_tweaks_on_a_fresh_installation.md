### Post by Rob Sayer on 2014-09-19
The package name is kubuntu-low-fat-settings.

However, while I installed it in kubuntu 12.04, which I mostly use at home, it wasn't available in 13.04 or 13.10 when I tried kubuntu on my netbook, and I don't think it's available in 14.04 either.

It's been a while since I did that but if you search the package name you should be able to find out what it does.

Anyway, to max speed in KDE first you want to turn off nepomuk auto starting.

Also, turn off all visual effects.  And tell it to turn off the compositor when a window is full screen.  This really helps for playing video.  In fact I've tried all 4 ubuntu DEs and KDE definitely plays video best.

Above all, avoid crap blogs that have lots of bad advice, especially involving using untrusted 3rd party ppa's.  You're just asking for problems.
