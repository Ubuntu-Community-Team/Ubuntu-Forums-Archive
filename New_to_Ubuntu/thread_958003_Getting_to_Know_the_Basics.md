### Post by tgalati4 on 2008-10-25
Find some old Linux for Dummies books or browse the discount book stores for obsolete Linux books--older distributions like Red Hat 7.3.  Then use the misinformation to try to figure out what works in the current version of Ubuntu.

For instance in other Linux distributions, user-installed programs can usually be found in /usr/share, but in Ubuntu, many user programs are in both /usr/bin and /usr/share.

Many Linux programs store system-wide configuration files in /etc, but each distribution uses different directories and sometimes different filenames for these configuration files.  It can be a challenge to find the appropriate ones to fix a problem that you are having.

Spend some time studying the family history of Linux:

[http://futurist.se/gldt/](http://futurist.se/gldt/)

That way you can see how the various distributions are connected.  Distributions based on Debian (such as Ubuntu) tend to have the same file system layout and same command line syntax (bash).
