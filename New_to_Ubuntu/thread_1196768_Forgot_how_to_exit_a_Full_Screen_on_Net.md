### Post by aeiah on 2009-06-25
alt+F4 will exit the window. in firefox, alt+q works too (or alt+w if only one tab is open). if your browser is truly in fullscreen mode you should be able to press F11 to get it back, although that might not be the case in this instance.

you should be able to move the window around by holding alt whilst dragging with the left mouse button. perhaps this'll make the browser edge viewable so you can resize it appropriately
