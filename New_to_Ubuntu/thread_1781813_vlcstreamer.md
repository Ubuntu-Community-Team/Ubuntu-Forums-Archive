Just an idea, but if you want the GUI, it might be worth giving WINE ([http://www.winehq.org/](http://www.winehq.org/)) a try.  This will let you run the windows version which isn't beta.
The process for this would be:
```
$sudo apt-get install wine
```
After that, you should be able to install as normal.
