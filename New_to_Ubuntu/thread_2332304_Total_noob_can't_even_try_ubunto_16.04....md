As for actually installing Ubuntu a lot will depend on the decision you make. Erase disk and install Ubuntu is the easiest method. That will get rid of Vista and everything else on the machine.

If you want to dual boot it will get a bit more complicated. If you want advice, ask for it but give us details about the hardware specification of the machine and the partition layout especially if dual booting. 

Different advice is given for dual booting with versions of Windows that came before Windows 8 compared to the advice for dual booting with Windows 8/10.
