### Post by stoogiebuncho on 2010-08-17
Looking through Synaptic, it looks like you might be looking for the sbackup package.  Try installing that instead.  There's more information here:
[https://help.ubuntu.com/community/BackupYourSystem/SimpleBackupSuite]("https://help.ubuntu.com/community/BackupYourSystem/SimpleBackupSuite")

Edit: some packages you install through synaptic are either command-line tools, or else they are parts of other programs and not complete programs in themselves.  These packages will not be added to your menus.  If you want to be sure that programs will be added to your menus, try using the Ubuntu Software Center
