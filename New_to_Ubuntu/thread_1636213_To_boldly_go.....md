### Post by uRock on 2010-12-02
Hello and welcome to the forums.

[Ubuntu Studio]("http://ubuntustudio.org/") is an official version of Ubuntu.

64bit should not give you any compatibilities, unless you use some very custom programs, such as Cisco's PacketTracer.

Some Windows based games play well in Wine. I'd recommend checking [their page]("http://appdb.winehq.org/") to see what games work and how well they work. If this is a big issue, then you may want to stick with dual booting Windows.
