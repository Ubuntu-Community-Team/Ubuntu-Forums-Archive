HOWEVER, to be on the safe side, make sure you back up your current .mozilla file (rename it .mozilla~ ) then copy file you want to your home directory.  Fire up Firefox and see what happens.  The very worse is that Firefox will refuse to start up, in which case, go ahead and delete .mozilla and rename .mozilla~ to .mozilla
