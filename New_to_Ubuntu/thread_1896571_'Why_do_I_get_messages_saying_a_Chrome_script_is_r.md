### Post by lovinglinux on 2011-12-17
Chrome is a generic name to describe the visible graphical interface features of an application. Google simply borrowed that name for their browser.

The alert you received means that the script is running in the application layer and not the web page layer. It could be a script in the application itself or an extension. It has nothing to do with the Chrome browser.
