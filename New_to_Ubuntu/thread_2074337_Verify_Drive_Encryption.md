Is there a way for me to verify that I've encrypted my drive? I'm uncertain whether or not I selected this option during install and I'd like to be 100% certain.  Also, if it turns out I haven't encrypted my drive, what do I do? Is there a way to do so without losing all my files?  Thanks!

### Post by Cheesemill on 2012-10-21
Boot from a Live CD and see if you can access the files in your home folder on the hard drive.

If you can, then they're not encrypted.
