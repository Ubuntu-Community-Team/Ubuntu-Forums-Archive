### Post by amjjawad on 2010-12-09
> **lastelement said:**
> Yea that's what I said earlier. With the one time boot key, I can select Ubuntu and it works, except the Grub screen never pops up to choose. So there's still a problem with Grub independent of my BIOS

Update-grub MUST work. You're probably forgetting to change your BIOS-Boot Sequence.

Reboot your machine
Enter BIOS
Set sda (the HDD where Ubuntu is installed) to be the first drive
Set sdb (the HDD where Windows is installed) to be the second drive
SAVE BIOS Settings
Your system will reboot
Once it logs in to Ubuntu, go to Applications > Accessories > Terminal and run:

```
sudo update-grub

```
You need to enter your root password in order to execute that command.

Now, try to copy the output of the above command and post it here please. Don't forget to wrap up the output with "CODE" tags or "#".

It should work this way but please make sure to follow the above steps correctly.
