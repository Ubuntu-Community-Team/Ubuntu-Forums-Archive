### Post by pedro_orange on 2008-06-11
You can dual boot a number of ways.

The easiest way is to install XP onto a disk. 
Then install Ubuntu on that same disk as a seperate partition.

This will tell you everything you needed to know and more.
[https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)

### Post by Xiong Chiamiov on 2008-06-11
Installing on separate drives is just as easy as separate partitions on the same drive.  You'll have to have your BIOS set up to boot the hard drive you put GRUB on though.
