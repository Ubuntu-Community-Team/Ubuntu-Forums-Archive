### Post by snowpine on 2009-05-28
Fluxbuntu was not just "Ubuntu + Fluxbox." It was a separate distro (not officially supported by Canonical) that used fluxbox, a *very* lightweight suite of applications (kazekahase for browsing, xmms for music, rox for file managing, etc), and stripped out a lot of unneeded services. It was much "lighter" than even Xubuntu. I also really liked its artwork.

I use the past tense because the latest release of Fluxbuntu is 7.10 Gutsy, which is no longer supported. The currently active project that I feel is closest to Fluxbuntu is AntiX (based on mepis/debian). As far as a lightweight Ubuntu derivative, you really can't go wrong with CrunchBang (openbox instead of fluxbox).
