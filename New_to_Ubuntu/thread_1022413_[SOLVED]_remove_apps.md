### Post by balaknair on 2008-12-26
XFCE has an inbuilt menu editor which lets you edit the menus to show only the apps you want. However since it's pretty buggy/limited and I've never heard anything good about it, you'd be better off manually editing the menu settings.

A good howto  can be found here
[http://wiki.xfce.org/howto/customize-menu](http://wiki.xfce.org/howto/customize-menu)

But I think you ought to check this first
