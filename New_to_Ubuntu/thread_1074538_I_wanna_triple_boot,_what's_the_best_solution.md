### Post by notoriousdbp on 2009-02-19
I would install Ubuntu last of the three as this will install the bootloader which includes the other two operating systems and would probably install Windows first of the three.  Yes, you do need a swap partition for Linux.  For Linux I would create root, home and swap partitions as logical partitions within a primary partition (easily done in the Ubuntu installer.)  Have you considered running the other Operating Systems in Virtual Machines if you only need to dip into them occasionaly?
