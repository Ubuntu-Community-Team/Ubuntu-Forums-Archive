### Post by 3rdalbum on 2010-02-15
> **mastablasta said:**
> Ok i will try Cheese. Bu thwat about for TV card? With the card there was some software for Windows, and it sort of works there but i am not sure what to use in Linux. I would probably need to download something i guess...

The "me-tv" program should do the trick nicely. It's in the repositories (Synaptic Package Manager).

If you're planning on using the webcam with Skype, it might be a good idea to install Skype to test it out. Some users report that their cams work fine in Cheese but don't work in Skype. Now, Skype is not a small download because it depends on lots of packages that aren't included in the Ubuntu install, but these will be retrieved for you.
