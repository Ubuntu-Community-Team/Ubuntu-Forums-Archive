### Post by Dylan1473 on 2012-08-05
Ubuntu One is a cloud storage service. You can back things up online or transfer music and bookmarks and such between devices. You don't actually *need* it but it can be quite useful. You need an Ubuntu Single Sign On (or whatever it's called) account to use it, though. This account can be set up right from the Ubuntu one interface I believe.
