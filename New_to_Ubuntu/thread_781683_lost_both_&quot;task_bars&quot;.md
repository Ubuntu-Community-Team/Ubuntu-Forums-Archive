### Post by vambo on 2008-05-04
You might try this:

Bring up Run Program - <alt> <F2>
Run xfce4-panel

Alternatively in terminal run:
xfce4-panel &

This should give you your panels back
For future logins make sure that
"Save Session for Future Logins" is checked in your Exit dialogue.
