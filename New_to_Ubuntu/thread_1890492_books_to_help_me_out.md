The underlying mechanism is pretty similar across Linux Distros so most books are of use.

There are some fundamental changes in 11.10 which wont be covered in there certainly interface related though 11.04 is Unity also.

However if you can wait and want a specific 11.10 book then [http://www.amazon.com/Ubuntu-Unleashed-2012-Covering-11-10/dp/0672335786/ref=sr_1_1?ie=UTF8&qid=1322943482&sr=8-1](http://www.amazon.com/Ubuntu-Unleashed-2012-Covering-11-10/dp/0672335786/ref=sr_1_1?ie=UTF8&qid=1322943482&sr=8-1) is out ina few weeks and there are some others also.

[http://www.amazon.com/Ubuntu-11-10-Server-Administration-Reference/dp/1936280345/ref=sr_1_4?s=books&ie=UTF8&qid=1322943576&sr=1-4](http://www.amazon.com/Ubuntu-11-10-Server-Administration-Reference/dp/1936280345/ref=sr_1_4?s=books&ie=UTF8&qid=1322943576&sr=1-4) this is availabel but covers server (but will be useful)

However most of what you need to know is available online if you really need it or use one of the older book as a base and use online to find the differences etc.
