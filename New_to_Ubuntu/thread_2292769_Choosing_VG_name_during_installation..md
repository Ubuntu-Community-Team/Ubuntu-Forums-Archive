### Post by grahammechanical on 2015-08-31
The technique is to remove the suggested name from the text panel and then type in a name of our choice by which the installation will be recognised on a network. I think that we can change this label after installation by going to About this computer in the power/cog menu or by opening the Details page in System Settings and editing the label in the Device Name text panel. I have never tried this myself. So, I do not know how effective it is but the panel does allow the editing of the label.

And for the record, Canonical is always trying to set new standards - higher standards in design and usability. I do not know what you mean by that comment "like MS did." What has Microsoft got to do with you being unfamiliar with how to install Ubuntu?
