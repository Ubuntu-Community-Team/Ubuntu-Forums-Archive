### Post by howefield on 2010-01-26
> **Glenstr said:**
> I was browsing the downloads, but the 64bit offerings seem to be for certain AMD processors only.

Ignore the AMD bit, it will work perfectly on Intel 64 bit.

It is named thus only for historical reasons, not for the fact it is designed only for AMD.
