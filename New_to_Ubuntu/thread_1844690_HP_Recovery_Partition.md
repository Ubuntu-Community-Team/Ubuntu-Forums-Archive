### Post by Esot-Eric on 2011-09-15
Hey chaps, I've got a HP laptop which after using Ubuntu on my main computer for a bit I've decided I want to install Ubuntu on. It currently has windows 7 and I received no OS disk with the laptop, there is however a separate partition named recovery which from the looks of things will restore the laptop to how it initially was.   How would I go about installing Ubuntu as the sole OS whilst keeping this recovery partition available if I wish to use it and am I right in my belief that I can format everything aside this partition and I'll still be able to return to Windows 7 if I wish?

---

### Post by ibutho on 2011-09-15
In Windows 7, you can make recovery disks by going to Start -> All Programs -> Recovery Disc Creation (only on HP / Compaq computers). If you still want to save the partition, just make sure you do not format it for use when you install Ubuntu.

### Post by coffeecat on 2011-09-15
What to do with a HP comes up a lot because they use up all your quota of four primary partitions. If you delete all but the recovery partition, you should be able to set the machine back to its factory defaults with Windows 7, but you would lose any Ubuntu or other Linux installations you had set up.
