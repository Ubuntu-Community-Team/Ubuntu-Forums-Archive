### Post by donkyhotay on 2009-01-21
Depends what you want, I've found lots of books on ubuntu at my local bookstore... well the big bookstores at least, not the little ones. Although some books will help with ubuntu in general most linux books will deal with specific tasks, and generally the more advanced tasks, like configuring an apache server and stuff like that. Personally if you just want to learn ubuntu in a general way then just use it a bunch, experiment a little. If you run into problems or can't figure something out then ask (ubuntu forums are great BTW). If you really need a book then as I mentioned before it helps to know what you want to focus on so we can give a proper recommendation.
