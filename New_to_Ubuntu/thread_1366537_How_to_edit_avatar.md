Avatars have to be certain file-types, and of a maximum size (pixels width X pixels height) and also in terms of file size.

> Note: The maximum size of your custom image is 90 by 90 pixels or 19.5 KB (whichever is smaller).  If it is bigger, then it may - or may not trim it for you. Best to get a jpg / gif or png within these limits.

Gimp can do this for you, if you need help doing it - post back & I'll pop a link or a quick how to on for you.

### Post by Matt__ on 2009-12-28
ok left click the image you want,
once it opens right click it and 'save image as' (to desktop or wherever)

next open gimp and drag image into window/use 'open' menu

then go to image/scale image and select 90x90 pixels..apply scale then
next go to image/flatten image...apply flatten
next go to file and 'save as' and change .png to .jpg in file name

that should bring down the Kbs and hopefully not reduce the image quality too much.
it will look like this--- [ladybug avatar]("http://img684.imageshack.us/img684/3773/traditionalbubblecaseba.jpg")
It actually scales to 15Kb as a png too---[here it is as png]("http://img237.imageshack.us/img237/835/traditionalbubblecaseba.png") (just do scale to 90x90 and save)
(Just right click that one and save as lol :) )
