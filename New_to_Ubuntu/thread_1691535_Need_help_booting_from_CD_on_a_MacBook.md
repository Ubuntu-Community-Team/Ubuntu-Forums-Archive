### Post by GabrielYYZ on 2011-02-20
check to see if you burned the "image to the CD" and not a "data CD". i know i might seem like captain obvious but it's a common enough mistake. also, check the md5sum on the iso, in ubuntu it's as easy as doing:

```
user@something$md5sum Ubuntu-something.iso
```

on the mac, i don't know... never used one but it might be similar, so i can't comment on the technicalities of booting from CD on one either.
