### Post by CatKiller on 2020-04-22
> **rocket57 said:**
> <for example> I read that Ubuntu 18.04 is an LTS distribution good until April 2023. Does this mean that Ubuntu MATE is also good until April 2023?
Not exactly. 

All the Ubuntu flavours use the same repositories with the same packages. The different flavours come up with their own configurations and packages for their default installation that are different from the ones in the Ubuntu default installation.

Ubuntu will provide the security updates and bug fixes for the Ubuntu packages, but it would be up to the maintainers of the flavours to provide those updates for their flavour. Those maintainers don't typically have the same resources as the main Ubuntu project, so they aren't generally able to commit to doing that for as long. If you check each flavour's webpage they'll say for how long they can commit to supporting a particular release.

So you might find that after 3 years (rather than 5) you wouldn't be getting updates to your window manager, say, if you're using one of the flavours. But you'd still get updates to the things they have in common - like the kernel - because they're the same repositories with the same packages.

### Post by Dennis N on 2020-04-22
Ubuntu 18.04 (Gnome Desktop) and Ubuntu MATE 18.04 (MATE Desktop) share about 75% of their packages - those are all supported until Apr. 2023. The rest are MATE-specific or Gnome-specific. The MATE-specific packages are supported until Apr. 2021.
