### Post by revanb on 2008-09-05
You have to unmount it before unplugging it. Just like you have to stop it on windows before unplugging it. Right-click on the icon and choose unmount. Then the icon will dissappear, only unplug then. It's probably possible to disable the icon coming up in the first place if you just want to unplug it, but I've not wanted to do that.

### Post by NE Key on 2008-09-05
Before you pull the pen drive out of the socket you must "un-mount" it.

(Right click on the icon and select "Unmount").
 

If you do not do this you can end up with corrupted files and problems such as only part of a file being saved to the pen drive.

(You have to do the same thing in windows, or more accurately- You should do the same thing in windows.)
