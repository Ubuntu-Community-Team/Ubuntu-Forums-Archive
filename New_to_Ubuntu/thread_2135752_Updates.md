### Post by grahammechanical on 2013-04-15
Do you have any PPAs installed? If so, remove them. These Personal Package Archives are a good way to get hold of someone's software but if the archive is not maintained then it can cause errors like these. I cannot now remember 11.10. You should have Synaptic Package Manager installed (not installed by default on 12.10). Open that and look for Settings or Repositories or Sources. Best I can do.
