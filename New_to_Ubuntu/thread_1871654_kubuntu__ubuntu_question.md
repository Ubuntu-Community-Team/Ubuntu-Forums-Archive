### Post by tartalo on 2011-10-29
They use different Desktop Environments and have a different default software selection, other than that they are identical, the hardware support should be the same and they use the same repositories so the same software is available even if it isn't installed by default.

The performance is different, a lighter Desktop Environment like XFCE or LXDE will make your computer act faster than Unity or KDE, but it also will have less features, you might or might not need/want those features.

The best is to try them via a Live CD/USB and decide yourself which one suits you.

### Post by Lars Noodén on 2011-10-29
They're identical in that you can add/remove packages to/from one and make it into one of the others.  So you can use packages from Xubuntu in Kubuntu and viceversa.
