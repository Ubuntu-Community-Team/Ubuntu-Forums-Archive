### Post by OutOfReach on 2008-10-14
You can use Startup Manager, install it with:
```
sudo apt-get install startupmanager
```

Then download a splash screen theme (check here for examples: [http://gnome-look.org/index.php?xsortmode=high&page=0&xcontentmode=160](http://gnome-look.org/index.php?xsortmode=high&page=0&xcontentmode=160)) and then startup startupmanager and goto the Appearance tab, then down at the bottom in the Usplash Theme section, select Manage usplash themes and then add your downloaded boot splash and select it to make it default.
