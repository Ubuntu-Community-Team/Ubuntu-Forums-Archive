### Post by presence1960 on 2009-05-13

> Absolute new user.  Had Windows/Internet Explorer.  In explorer.exe, I could see all my documents, pictures, etc. & could delete at that site.  Anything comparable on Ubuntu?  Also having difficulty downloading Yahoo Messenger.

In Ubuntu under Places you can choose to view your data. it will open a file browser window for you.

I may be wrong but I believe Yahoo Messenger won't run in Ubuntu natively, but you can use pidgin which is an IM that supports multiple IM clients such as Yahoo, AIM, GoogleTalk, ICQ to name a few.

Your windows .exe and .msi executables will not run in Linux. Some may run in Wine, but if you must have Windows software you can't get rid of your best bet is to set up a dual boot with Linux/Windows, or Windows in virtualization inside Ubuntu.
