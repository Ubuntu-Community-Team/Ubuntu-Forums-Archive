### Post by kranthi.doss on 2011-06-16
[SIZE=3][FONT=Lucida Sans Unicode]i had the same problem with unity. I was playing around with unity, i lost the taskbar and launcher after changing the profile in unity preferences. I coudn't open terminal to reset unity. i tried resetting unity in safe and classic mode with led no results. Searched the forum and found a way to correct it.
Actually my desktop is clean with no icons so when i inserted a dvd, it showed up on the desktop. opened it and went to the location /usr/bin/ and opened ccsm file. Thats the settings for unity in graphical mode. changed the profile back to unity. Though unity loaded a bit chuncky.. i logged out using 'gnome-session-save logout' command and when i logged in again everything well quite nice.

Im new to Ubuntu. everything is working like charm. ;)
