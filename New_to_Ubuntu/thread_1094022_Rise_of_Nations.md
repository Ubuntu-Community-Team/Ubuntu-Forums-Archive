### Post by Temposs on 2009-03-12
Which version of Rise of Nations do you want to install? Some work a little bit, and some don't. For any program you want to run on WINE, you need to check the appdb:

[http://appdb.winehq.org](http://appdb.winehq.org)

You need to search the AppDB for the programs you want to run. People test programs with different versions of WINE.
