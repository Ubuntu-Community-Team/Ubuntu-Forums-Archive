### Post by themusicalduck on 2009-08-18
You just need to reboot and tell the netbook to boot from the USB disk. It'll load up just like it would if it was booting from a CD.

Remember to install it you'll have to either make a new partition to install it onto or format the entire drive, so it's a really good idea to make backups first. You can tell Ubuntu to install itself alongside windows if you like or to use the entire disk and wipe windows.
