### Post by grahammechanical on 2011-05-10
My understanding of the wubi option is that its purpose is to allow Windows users an opportunity to try out Ubuntu without installing it to the hard disc as you would need to do if you wanted to try out another operating system.

This method has advantages over using a live CD but it has disadvantages when compared with an actual hard disc install. For the long term commitment to Ubuntu a hard disc install is much better for speed, security, and stability.

Regards.

P.S. If you go to the Installation and upgrades section of this forum you will find a sticky thread called wubi megathread. Read that and you will get an idea of the issues with wubi.
