### Post by carlwenrich on 2008-05-23
I'd like to create a script that launches the nautilus file manager, and pass it the directory in which I want it to open. Can I do this? If not, is there another file manager that will allow that?

---

### Post by neurostu on 2008-05-23
yes!
```

nautilus /home

```
open nautilus in the /home dir!

---

### Post by Achetar on 2008-05-23
Yes. You can.

```

nautilus <directory>

```
Example:
```

nautilus /usr/bin

```
would open nautilus in the /usr/bin directory.
