### Post by Bachstelze on 2010-03-30
Updating Ubuntu without an Internet connection can be a tedious task. The best thing to do is probably to not bother with it. Most updates are security fixes, and mostly irrelevant if the machine is not connected to the Internet.

If an update for some package is really needed, you can download the updated package from [http://packages.ubuntu.com](http://packages.ubuntu.com), copy it on an USB stick, and install it on another machine by just double-clicking on it..
