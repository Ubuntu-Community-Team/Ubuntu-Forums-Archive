### Post by ibuclaw on 2009-01-28
DGZDGZ, boot into the Ubuntu live CD environment and run the Partition Editor utilty (**System -> Administration -> Partition Editor**)

Then delete all partitions on your hard drive (Ensure you have the right one in view first though, and make sure that you have backed up all sensitive or sentimental data).
