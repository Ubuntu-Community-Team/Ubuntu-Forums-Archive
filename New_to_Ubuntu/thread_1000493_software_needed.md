### Post by redseventyseven on 2008-12-03
> **seilesh.BE said:**
> hai friend.....how can i able to get this softwares and from which website

Hi, and welcome to Linux!

The way you get software is very different to how you would on Windows or a Mac. A large range of software is available in large software libraries called "Repositories", which are maintained by the Ubuntu team. These can be accessed using the Add/Remove programs tool or (more advanced) using Synaptic Package Manager (in the Administration menu). You can even fetch software from the repositories by opening up a terminal and typing a few commands (even more advanced) but for now it's probably best to stick with the Add/Remove programs tool.

There are also a number of repositories which are available for you, but which aren't enabled by default when you install Ubuntu (predominantly for legal reasons). The one which stores the codecs to play MP3s and DVDs and so on is an example - it's called Medibuntu. If you google Medibuntu you should be able to find some instructions for how to enable that and get your MP3s and DVDs working.

Here's an analogy: getting Windows and Mac software is like going to a bookshop; getting Linux software is like going to the library - but a special library where you never actually have to give the books back.

Have fun!
