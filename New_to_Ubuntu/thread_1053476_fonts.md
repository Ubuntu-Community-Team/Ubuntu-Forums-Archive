### Post by Kellemora on 2009-01-28
Hi TI

Rather than creating a new hidden folder for your font in a place only you see and increase the possibility of ending up with duplicate fonts on your machine.

May I suggest you paste the font (font's don't need installed in Ubuntu) in the folder where it belongs.
Which is: /usr/share/fonts/truetype
here they will be accessible to all programs and users.

Unlike the DOZE, you CAN install fonts into sub-folders to keep font sets together or file them under the typographers names, etc.
You can also rename fonts file name to anything you want and it does not affect how they operate.  A reason you may want to change a fonts file name is because rather than being called what it really is as the file name, it may have been provided under a code name or under a number.  
For example:  The filename may be ML75XR27LA.ttf or 1234567.ttf instead of HumphreyBold.ttf and because of this, you could end up with more than ONE copy of the same font on your machine.

It's also easier to keep track of them if they are all in one place too.  If you run WINE, keep that Font Folder EMPTY also!
