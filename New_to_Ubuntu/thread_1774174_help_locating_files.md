### Post by Mariane on 2011-06-03
Windows cannot read Ubuntu files because they are written in different file systems. Windows uses fat32 or NTFS and Ubuntu uses ext3. 

There is a tool to overcome this:
[http://www.chrysocome.net/explore2fs](http://www.chrysocome.net/explore2fs)
