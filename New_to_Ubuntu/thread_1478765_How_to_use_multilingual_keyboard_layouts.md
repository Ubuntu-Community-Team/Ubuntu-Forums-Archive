### Post by Irihapeti on 2010-05-10
To choose the alternative set of keys, you need to set up "keys to choose 3rd level", in System -> Preferences -> Keyboard. Click on the "layouts" tab and then the "Options" button.

You can choose the key that's most convenient for you. There are quite a few choices available.
