[CENTER][FONT="Arial Black"]Installing Windows 7 taskbar clone for Ubuntu: A guide for the average Ubuntuer[/FONT][/CENTER]

NOTE: if everything goes kablooey, use this guide to reset (should work, tell me if it doesn&#8217;t): 
[http://www.watchingthenet.com/restore-panels-in-ubuntu-back-to-their-default-settings.html](http://www.watchingthenet.com/restore-panels-in-ubuntu-back-to-their-default-settings.html)

So, you just got a fresh install of Ubuntu, but you notice there are 2 taskbars and they&#8217;re pretty sucky, right?  And you also noticed how awesome the new Windows 7 taskbar is, and you wish Ubuntu had it. (If you think it isn&#8217;t you&#8217;re square and you live under a rock).   Here&#8217;s how to get it.

1. Download a program called Talika, link here: [http://linux.softpedia.com/progDownload/Talika-Applet-Download-44824.html](http://linux.softpedia.com/progDownload/Talika-Applet-Download-44824.html)

This is &#8220;GNOME-Applet&#8221; &#8211; aka something that you can put in one of the panels that are on the top and bottom.

2.Next, right click on everything on the top 2 panels, and uncheck the &#8220;lock&#8221; button for all of them.

*This is where we start doing the major work, so be careful*

3.Delete the top bar (click empty space and press remove panel).

4.Remove the taskbar part on the bottom panel that holds all the programs; this is what talika will replace.  Also remove the Applications/System/Places/Ubuntu logo, there is another GNOME-applet that is exactly like the windows logo/start menu in Windows.

5.Right click on empty space and add Talika, Main Menu, Notifications bar, indicator applet Session (so you can shut down your computer), clock.  

6.Move (make sure they are unlocked) and rearrange them so it looks like windows 7.

My setup is from left to right:  Main Menu, Talika, [black space &#8211; not an app] Notifications Bar, Trash   Indicator Applet Session, workplace switcher, clock, show desktop (see screenshot).
	Great! You now have a windows 7 taskbar with window previews on 	mouseover, but whats the point without &#8220;pinning&#8221; programs/applications to 	it?  Easy &#8211; while the program you want to pin is running, right click and press 	pin.  Now when it is closed, you will see it on the left side of the Talika app (	see the seperator bar in between my programs in the screenshot).

It's easy to get back the default setup with decent memory (boot without installing from your live CD if you need your memory refreshed), but (I would think) having this setup is much better.
Note:  to change the panel size, click on blank space and change the pixel sizes, I think mine is about 33ish.
