### Post by zeroseven0183 on 2009-09-15
If you're already using Ubuntu, open **Brasero** in **Applications** > **Sound & Video**.
You have there in the left pane, an option to **Burn image**.

If not, download a free burning tool like [CDBurnerXP]("http://cdburnerxp.se/downloadsetup.exe")

### Post by misfitpierce on 2009-09-15
I'm going to assume your on Jaunty 9.04 release. Let me know if i'm wrong, if i'm correct though you need to hit up Applications-Sound & Video-Brasero CD/DVD Creator. That opens program called Brasero and inside it on left hand side you'll see Burn Image. If you are not on Jaunty and don't have Brasero then open a terminal and enter

[quote="TERMINAL"]sudo apt-get install brasero[/quote]

Or... Hit Add/Remove Programs/Synaptic Package Manager and find Brasero CD/DVD maker and install it. Enjoy.
