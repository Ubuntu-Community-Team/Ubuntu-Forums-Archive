### Post by MG&amp;TL on 2011-10-08
1. Install compiz settings manager. ```
sudo apt-get install compizconfig-settings-manager
```2. Type 'ccsm' in the dash, then click on the compizconfig-settings manager icon.

3. Go to general options, click on the button.

4. Click on the desktop size tab, change there.

Hope it helps.
