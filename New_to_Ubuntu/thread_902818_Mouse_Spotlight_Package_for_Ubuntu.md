### Post by sdennie on 2008-08-27
Though not exactly the same, compiz (Desktop Effects) has something similar.  To enable it install CCSM:

```

sudo apt-get install compizconfig-settings-manager

```

Then go to System->Preferences->Advanced Desktop Effects and enable Show Mouse (It's towards the top).  The default key combo to turn it on is Super-K (the Super key often has a Windows logo on it) and there are a number of options for configuring it.
