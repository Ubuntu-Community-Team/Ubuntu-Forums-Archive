### Post by qyot27 on 2010-03-14
I've been using Ubuntu since 5.10, and every version (with the exception of 7.04-8.04, which I never actually installed to the HDD since I couldn't figure out the display problems I was having...turned out the default was using the wrong Hz frequency for the monitor; I felt like an idiot when I finally figured out *that* was what the problem was) ran/runs fine on my computer. *at the end of December/beginning of January I went on a little upgrading spree and put in a dedicated graphics card and an extra 256MB stick of RAM (so it now has a GeForce 6200 PCI card and 512 MBs of RAM; the graphics card didn't really help all that much on the surface, but it does when it comes to certain tasks - I still can't use visual effects, though).  But I was running Karmic before, during, and after the upgrades, so I know it still works ok.
