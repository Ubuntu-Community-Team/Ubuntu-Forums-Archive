### Post by buzzingrobot on 2015-12-18
> **Mr_Motley said:**
> 
I want to be able to setup Linux for gaming, I want to install wine and other applications. I could really use help on installing PDA net,.. 


Linux isn't Windows and can't be made to function as a direct substitute. You'll just create headaches. The best way to use Windows software is on Windows.

Wine is not intended to be -- and is not -- a tool that allows all Windows apps, for all versions across its long history, to run on Linux. In very simplistic terms, Wine tries to map calls a Windows program makes to equivalent routines in Linux. Because they are very different operating systems, that often can't happen. It especially can't happen when a Windows app ships with proprietary drivers, libraries, etc. that don't exist outside Windows.

Your best bet for a happy Linux experience, then, is to use Linux as Linux, and avoid trying to morph it into Windows. 

If you're seeing consistent problems with Ubuntu right after an install, before you start altering the system, it's very likely related to hardware. To address that, I'd recommend opening a new thread and including your hardware specs in that post.
