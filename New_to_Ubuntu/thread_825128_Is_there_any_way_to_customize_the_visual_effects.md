### Post by Sinkingships7 on 2008-06-10
Compiz is the software that's giving you these "effects" that you don't like, such as wobbly and snapping windows. To disable it, right click on your desktop and click "Change desktop background". Go to the visual effects tab and click "None".

A front-end is a small piece of software used to control the more complicated parts of the software. compizconfig-settings-manager lets you control (turn on and off) certain features that Compiz offers. If you are going to disable Compiz, then that tool will be of no use to you.

### Post by JoshuaRL on 2008-06-10
Compiz is the compositing manager that allows snapping windows and wobbliness.  That is the program you are having with.

The front-end configuration tool is a Graphical User Interface (GUI) that allows configuration of Compiz.  Messing with the myriad of settings there will allow you to change Compiz to something pleasing.

Linux is one of the most customizable OSs out there, but it thinks different than Windows.  You just need to learn where all the places to make changes are.  But that's okay, everyone starts somewhere.
