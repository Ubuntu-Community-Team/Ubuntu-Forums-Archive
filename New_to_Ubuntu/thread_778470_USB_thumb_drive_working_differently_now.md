Another long shot. Go to System-->Preferences-->Session and ensure that Volume Manager is checked on the start-up programs. If it isn't, then check it, reboot, and try the USB thumb drive again. If it (Volume Manager) doesn't show, you will need to add it to the list. I believe it resides in /usr/lib.
