### Post by oldfred on 2011-05-31
You said you did a backup. Is that a complete copy of Vista or just your data? Since the new install overwrite part of the drive, the full Vista partition is almost certainly not recoverable. And if you do recover it, it would not be bootable, and file structures were probably overwritten, so files are not easily recovered.

There is a program photorec that just scans a drive and looks for anything that looks like a file, not just photos. But it takes a lot of space on another drive to copy data to, does not recover full file names, just extensions (which may not always be correct). I have used it to recover some data but it is a lot of work.
