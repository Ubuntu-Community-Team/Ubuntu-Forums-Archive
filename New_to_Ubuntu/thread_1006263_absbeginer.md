### Post by Peter09 on 2008-12-09
The best way is to delete the ubuntu partions using gparted from the LiveCD. When you come to the install process tell it to use the unused disk space, the rest will be done automatically.

Do not delete any FAT32 or NTFS partions unless you want to remove windows.

### Post by theozzlives on 2008-12-09
Live CD will run without installing, but you can't save anything unless you run persistent with a USB drive.

You can install on a USB if you have a big enough one, and your computer will boot USB.
