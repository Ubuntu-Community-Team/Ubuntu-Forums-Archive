First problem: The text that you're referencing was written in 2010. You're using a version of Ubuntu from 2012. The instructions don't carry over, and in some cases it can be dangerous to try and follow instructions for an old version of Ubuntu. ONLY follow instructions for the same version of Ubuntu - maybe one version behind might be okay, but not four versions.

Second problem: Gnome Control Centre comes with Ubuntu now. Click the menu on the top-right corner of the screen and choose System Settings. That's all you have to do.
