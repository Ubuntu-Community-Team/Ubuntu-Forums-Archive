### Post by HotShotDJ on 2008-05-22
> **worldy said:**
> What's a meta package?A metapackage pulls together several or many separate program packages into a single metapackage.  As such, it does not, in and of itself, contain any programs, but simply DEPENDS on all the others.  For example, ubuntu-restricted-extras is a metapackage.  It contains no actual programs.  But it DEPENDS upon things like java, flashplugin-nonfree, some gstreamer plugins, etc.  Think of it as selecting the luxury trim package when you buy a car... it pulls together a bunch of separate items (and probably some stuff that you really didn't want, but comes with the package deal).
