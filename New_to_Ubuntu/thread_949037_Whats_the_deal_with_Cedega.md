### Post by jbrown96 on 2008-10-15
Cadega and many people in the community (Google is a big supporter) create a a free (beer, speech) application (?) called Wine that supports some Windows programs. Cadega takes Wine and makes changes to it to improve performance and compatibility for Windows programs (especially games). You should try WINE or at least check out the supported apps before shelling out for Cadega.
