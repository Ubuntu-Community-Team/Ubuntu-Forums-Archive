### Post by Lars Noodén on 2011-10-27
Both [geany](http://www.geany.org/) and [kate](http://kate-editor.org/) have syntax highlighting and other features useful for HTML editing.  Neither are WYSIWYG.  If you want WYSIWYG, then you might try Bluefish.

I used to use Photshop a lot but find GIMP much better for web work.
