### Post by nerdtron on 2013-09-22
Haven't ran into any problem regarding external hard drives. All seem to work just like they should. Plug and Play.

Just some notes:
Be sure to FAT32 or NTFS as the file system of the external hard drive for compatibility purposes on windows machines. Formatting the drive in windows would be good.
ALWAYS unmount the drive/safely remove the drive before pulling out from the USB port. Sometimes the data is still on the ram/cache and haven't been written on the disk. Unmounting/ejecting/safely removing the drives ensures data is written correctly.
