### Post by slakkie on 2009-10-07
When doing a network upgrade or via an upgrade via the alternate CD, you will need to upgrade first to 9.10 if you run 9.04. The only way to directly upgrade to 10.04 is by running 8.04. Since only LTS versions can directly upgrade to LTS version, other versions cannot. Unless off course you run the version prior the the LTS.

So, you could wait with the upgrade to 9.10 till April and then directly upgrade to 10.04 or you upgrade in October and then upgrade again in April. You need to do the two upgrades.

If you reinstall, then you can directly reinstall 10.04, but that is not an upgrade iyam.
