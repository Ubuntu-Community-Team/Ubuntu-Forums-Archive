### Post by davidmohammed on 2010-09-05
when you install, you can select to install side by side - this is called partitioning.  

Ubuntu will use a filesystem called ext4.  Windows will use either FAT32 or NTFS.  Dont worry, you cant corrupt the other by defragging.
