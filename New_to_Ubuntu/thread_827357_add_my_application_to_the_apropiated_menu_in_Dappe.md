### Post by Vivaldi Gloria on 2008-06-12
> **herbert tamayo said:**
> I've tried the option "Edit menu" right-clicking gnome but it didn't give me all options that I need, so any suggestions are welcome.

All you want can be achieved with edit menu actually.

> I want to move the "a" application from "internet submenu" to "Games submenu"

In the edit menu, you can grab the application a in the internet submenu and move it to games submenu.

> I've installed some application from outsource, not using the repos, so, I created a shortcut but this app are not in the gnome main menu.

You can do this in edit menu by pressing the new item button.
