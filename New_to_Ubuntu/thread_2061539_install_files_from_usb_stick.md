### Post by drmrgd on 2012-09-24
By default, KDE will recognize that something was plugged in, but not necessarily mount it.  Have a look at the USB icon in the lower right (assuming the default desktop setup!), and your device should be listed there.  If it has a hard drive with a little plug icon, then it's mounted.  There will be a diagonal line if not.  Just click the icon to mount it.  See the attached picture for examples.

You can change this setting in System Settings -> Removable Devices.  If you want everything to be automounted, just check the appropriate boxes.
