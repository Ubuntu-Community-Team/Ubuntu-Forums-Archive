### Post by Koselara on 2008-09-27
> I've found exult.cfg but it won't let me edit it.

On the off-chance that it works, try editing it as root.  To do that, you'd call up a console/terminal window or the "run" command from the menu, then type:
[INDENT]*sudo gedit* (if you are in Gnome)
*sudo kedit* (if you are in KDE)
[/INDENT]  Then open the exult.cfg with the text editor, and see what you can do. I just installed it (and was promptly informed that it doesn't know where my games are) so I don't know if it will work. I'm guessing at the answer based on what usually causes "can't edit this" issues. 

If that fails, try asking the [Exult team's forum]("http://exult.sourceforge.net/forum/list.php?f=1") or the fans at [rec.games.computer.ultima.dragons]("http://groups.google.com/group/rec.games.computer.ultima.dragons"). People at both places are usually quite helpful.
