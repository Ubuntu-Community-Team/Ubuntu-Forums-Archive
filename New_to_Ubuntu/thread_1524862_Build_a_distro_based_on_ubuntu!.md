### Post by -humanaut- on 2010-07-06
It depends on alot of things how good of a coder are you? is your intent to create a "branch" sort of like Linux Mint? I think a good place to start is using the Ubuntu minimal image [https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)

With the minimal installation CD you have just the base Ubuntu system which you could tailor to your liking's add new kernels etc... Basically Linux Mint is just Ubuntu base with a heavy amount of customization though it has enough to make it a seperate distro altogether such as Mint Tools,Mint Software Center,Mint Update,etc...

If you really want to learn how to build a distro though I suggest Linux from Scratch or Slackware though.
