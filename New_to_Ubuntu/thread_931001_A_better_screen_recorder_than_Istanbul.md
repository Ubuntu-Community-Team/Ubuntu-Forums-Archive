### Post by starcannon on 2008-09-26
I like gtk-recordMyDesktop very much:
```
sudo apt-get install recordmydesktop gtk-recordmydesktop
```

You'll find the desktop recorder at:
{Applications>Sound & Video>gtk-recordMyDesktop}

gL have fun
