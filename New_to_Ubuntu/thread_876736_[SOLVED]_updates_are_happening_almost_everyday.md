### Post by k3lt01 on 2008-08-01
The updates are a natural part of the OS, my families Windows PCs are updating all the time to lately so its not just Linux.

To limit the size you can select to only update security updates if you wish. Go to System> Administration> Software Sources. Then click the Updates Tab and you see a few options. Now if you want to limited updates I would suggest you have ticks in Important Security Updates and Recommended Updates. The other options are for proposed updates which may come through later and unsupported updates which are not important in the grand scheme of things so I would un-tick them so you are not getting all the extra downloads.

Hope this helps.
