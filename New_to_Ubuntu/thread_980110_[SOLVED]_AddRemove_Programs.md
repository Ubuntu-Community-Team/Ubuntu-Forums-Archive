### Post by bscbrit on 2008-11-12
Well there is no reason that they should just disappear, so we'd better try to find them again:-)

Applications->Accessories->Terminal and type the command 'gksudo synaptic'.  It should ask for your password and then, hopefully, synaptic will burst into life.  Let me know what happens, and what you see if it is not what we expect, please.

### Post by aysiu on 2008-11-12
Right-click on the Applications menu and select *Edit Menus*.

If the launcher for Add/Remove is there but unticked or unchecked, tick or check it.

If it's not there, add it.
