### Post by cariboo on 2010-06-04
I've been using Nvidia video cards for the past 10 years, without having any random crashes. These usually happen when the driver isn't installed properly, and /etc/X11/xorg.conf isn't set up properly. Nvidia/Ubuntu has made it quite a bit easier to setup /etc/X11/xorg.conf, by including a command line applet called **nvidia-xconfig**

I'm currently using the 256.29 drivers on a 9400GT and a GT210 with zero problems.
