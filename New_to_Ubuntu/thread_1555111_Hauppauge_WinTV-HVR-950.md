### Post by kwacka on 2010-08-17
I believe that the drivers are already installed in Ubuntu as part of the kernel.

When you say 'the software' that depends on what you want. If you want a full-blown media centre( a la Windows media centre) check out Mythbuntu, if you just want to watch a few programs on computer install (e.g.) xawtv from the Ubuntu Software centre under 'applications' - just enter 'TV' in the search box
