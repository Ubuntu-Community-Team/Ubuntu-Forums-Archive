### Post by deadflowr on 2013-03-25
No, you can only have one launcher.

But, you can add a dock, if you'd like.

Docky works. Or cairo-dock.

Plus you can customize the launcher so the top applications that you use are all together.

Just right click on the launcher you want to use and make sure it's set as locked to launcher.
Then simply pull it out of the launcher and move it to where you want it set.
