### Post by hyper_ch on 2009-03-28
well, you can just remove the ubuntu-desktop and installed the server kernel and then boot the computer with the server kernel.

Then you should have a cli-only machine. However that's not necessary. You can run a file/print server just like that although if you want to go really server-wise it's recommended to not run a gui.
