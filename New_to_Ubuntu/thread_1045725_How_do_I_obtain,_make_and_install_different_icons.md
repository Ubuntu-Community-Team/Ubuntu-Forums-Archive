### Post by Golem XIV on 2009-01-20
Try [Gnome Look]("http://www.gnome-look.org/") for icons.

They're usually distributed in compressed format (.tar.gz).  Download the one you like to someplace handy, like the desktop.  Installing the icon sets should now be easy, open the theme manager and just drag the .tar.gz file onto it.
