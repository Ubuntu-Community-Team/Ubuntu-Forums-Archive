### Post by s.fox on 2010-01-29
Hello,

I think you mean the icon next to Applications,  Places and System.

To change that image you need to navigate to here:

```
/usr/share/icons/yourtheme/scalable/places/
```You should see a file called distributor-logo.svg

Make a backup of it.

All you need to then is move the linux-mint logo svg file into that directory.  Make sure it is called distributor-logo.svg

Hope this helps you.
