### Post by Austin_KW on 2008-04-22
> **marenkar said:**
> Is there a program that will do this?

In normal operation you cannot hide your ip address. You make a request on the network and the server must know your address to send back the response.

You can use a proxy, who will make the request for you and forward the servers response back to you. The server will only see the proxy's address not yours. Of course the proxy will know who you are.

There is also a program called tor which uses a number of proxies to obscure your identity by forwarding through a few different proxies.
