### Post by 416inversed on 2010-01-25
> **computerguts said:**
> is there a skype app for ubuntu 9.10

Yes, I added the repository
```
deb *http*://*download*.*skype.com/linux*/*repos*/*debian*/ stable non-free
``` to Software Sources, in System, Administration

and then in terminal
```
sudo apt-get update
sudo apt-get install skype
```
