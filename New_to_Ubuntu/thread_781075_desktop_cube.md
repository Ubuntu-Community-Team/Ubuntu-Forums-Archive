### Post by lazyart on 2008-05-04
```
sudo apt-get install gnomecompiz-config-manager
```

This package gives you a new entry in System>Preferences that lets you change a lot of settings.  Go to Desktop Cube>Appearance>Skydome and put a picture there.

The trick is the image has to be a certain ratio.  Best bet is to google *compiz skydome* and find some images that are properly formatted.
