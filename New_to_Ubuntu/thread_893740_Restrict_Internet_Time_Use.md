I think most routers have a tool for that.  Mine, for example, is a D-Link DIR-655 and I can set Schedules under the Tools menu.

"Schedules can be created for use with enforcing rules. For example, if 										you want to restrict web access to Mon-Fri from 3pm to 8pm, you could create a schedule 										selecting Mon, Tue, Wed, Thu, and Fri and enter a Start Time of 3pm and End Time 										of 8pm."

I've never used it, but I understand it works with parental controls, so I would assume you can lock out specific MAC addresses.
