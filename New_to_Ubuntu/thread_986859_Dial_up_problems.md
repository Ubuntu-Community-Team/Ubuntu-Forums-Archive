### Post by jimmy the saint on 2008-11-19
First, go into add/remove and install Gnome PPP.  That is the best dialer to use.  In addition, it will tell you if a modem is even connected.  If it says that there isn't it is possible (likely) that you have a softmodom, which relys on software to do the work. The problem is that most of that software is written for windows, so it is better to get a hardmodem, which usually operate via a serial port.

But first things first, try Gnome PPP and see what it says.
