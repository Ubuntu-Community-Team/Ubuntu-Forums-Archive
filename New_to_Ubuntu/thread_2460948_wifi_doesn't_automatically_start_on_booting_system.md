### Post by grahammechanical on 2021-04-22
Once you have used System Settings>WiFi to set up your WiFi connection to the router open System Settings>WiFi again and you should see a cog icon alongside your network connection. Click on that and a dialog will appear. It will contain a tick box labelled Connect automatically. This is the situation on 20.04. I cannot at the moment check 20.10 but it cannot be very different.

You need to set Wifi to automatically connect to the WiFi network. Not forget to click Apply.
