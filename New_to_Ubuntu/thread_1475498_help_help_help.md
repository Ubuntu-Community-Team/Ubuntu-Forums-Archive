### Post by nathan726 on 2010-05-06
The terminal is the place where you type commands. It is located at Applications > Accessories > Terminal. You can usually also access it by pressing Ctrl+Alt+T (or press Alt+F2 and type gnome-terminal).

Python is usually installed by default, you can run python by typing the word 'python' into the terminal.

You can find many python tutorials  [here]("http://wiki.python.org/moin/BeginnersGuide/NonProgrammers")  and  [here]("http://www.google.com/search?q=python+tutorial+for+beginner"). You can find lots of information about the terminal  [here]("http://beginlinux.com/desktop_training/ubuntu/ubuntu-terminal-guide")  and  [here]("https://help.ubuntu.com/community/UsingTheTerminal") and  lots more  [here]("http://www.google.com/search?q=terminal+tutorial+ubuntu"). There are quite a few Ubuntu user guides [here]("http://www.google.com/search?q=ubuntu+user+guide").
