### Post by Mark Phelps on 2015-05-11
>  I am told it is the "only" way to fly.Nice idea -- but not nearly the case ...

Ubuntu is but one of many (e.g., hundreds) of Linux distributions -- packaging of the Linux OS kernel with a desktop UI and bunch of applications.

Even Ubuntu is a family with several different versions that have the same core but different desktops and (some) different apps.

The best thing for you to do at this point is to try out some versions to see what you like best.  You can download the ISO files from [http://distrowatch.com/]("http://distrowatch.com/")  You download the version you want to a working computer and burn the ISO to a DVD, and then boot from that.
