### Post by cariboo on 2015-11-20
You seem to be using your experiences learned while using Windows, which don't transfer to using Ubuntu. I'd suggest using either the Software Centre, or better yet install Synaptic Package Manager to find either the Ubuntu/Debian version of the programs, or find packages that have similar functionality. 

Many of the web based services can be used, such as:

[LIST]
[*]YouTube
[*]Wikipedia
[*]Facebook
[*]US Truckstop's websites
[/LIST]

with a web browser.

[Bibletime]("http://www.bibletime.info/") looks like it does what you want, it's in the repositories.

GPS satellite imaging, have a look [here]("http://www.linuxlinks.com/article/20110522052357647/GPSTools.html"), several are located in the repositories.

I'm not sure what you mean by CD's and DVD's, but CD's will play almost automatically when inserted, DVD's need a few extra packages, ubuntu-restricted-extras, or VLC.

Personally I use Chromium-browser instead of Chrome, but Chrome for Linux  can be downloaded from [Google]("https://support.google.com/chrome/answer/95346?hl=en"), which will setup PPA's that allow it to keep up-to-date.

To install spotify look [here]("http://http://www.omgubuntu.co.uk/2013/01/how-to-install-spotify-in-ubuntu-12-04-12-10").

As for music downloads, I can't help you there, as we only support downloading open source media. You can probably download it from the same sources you already use.
