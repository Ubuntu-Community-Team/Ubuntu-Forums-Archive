### Post by TheHitman2010 on 2010-05-09
Hey there,

I just thought I would share some links that helped me big time with my first install of Ubuntu 10.04 Lucid Lynx.  They'll help anyone who has the most common issues upon initial install.

It's important to note that after I installed, my first reboot was in Recovery Mode where I booted in Failsafe Graphics Mode, installed the most current NVIDIA driver, and then rebooted normally. The following links helped me after that, and in no particular order. I would recommend, personally, the "Basics" portion of the first link, followed by the Java update/install, the splash screen fix, and finally the remainder of the post install guide.  Now my machine runs flawlessly! Thank you Ubuntu community!

[Post install guide.]("http://blog.thesilentnumber.me/2010/04/ubuntu-1004-post-install-guide-what-to.html")
[Known bugs workaround (including Flash Screen and button location resolutions.)]("http://ubuntuforums.org/showthread.php?t=1469475")
[Installing the most recent Java version (removing Openjdk).]("http://sites.google.com/site/easylinuxtipsproject/java")  DO THIS AFTER INSTALLING UBUNTU-RESTRICTED-EXTRAS.

Enjoy!
