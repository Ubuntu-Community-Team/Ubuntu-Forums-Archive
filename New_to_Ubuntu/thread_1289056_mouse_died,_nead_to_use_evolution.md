### Post by lavinog on 2009-10-12
just found keynav - a keyboard-driven mouse cursor mover
you can install it by pressing alt-f2 and typing:
gnome-terminal

in the terminal type:

```

sudo apt-get install keynav

```

---

### Post by lavinog on 2009-10-12
here is part of the man page:
```

 Run keynav, and activate it by pressing Ctrl+; (semicolon).  You should see a thin frame on the screen with a cross in it.

       Default configuration:

           * h        : select the left half of the region
           * j        : select the bottom half of the region
           * k        : select the top half of the region
           * l        : select the right half of the region
           * Shift+h  : move the region left
           * Shift+j  : move the region down
           * Shift+k  : move the region up
           * Shift+l  : move the region right
           * Semicolon: move the cursor to the center of the selected region
           * Spacebar : move the cursor and left-click
           * Escape   : cancel the move

```

### Post by freak42 on 2009-10-12
system->preferences->assistive technology-> keyboard accessibility (button)->mouse keys(tab)->check pointer can be controlled by keypad


it allows you to move your mouse with the keypad (number block), if on laptop try the Fn (function) key with the numberblock around keys 7-0,u-p,j-.... 

the numbers 2,6,8,4 move the cursor down,right,up,left, 5 is a left-click, don't know how to do a right click yet..
