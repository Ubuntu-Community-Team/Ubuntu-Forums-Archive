### Post by audiomick on 2012-07-30
The tips are likely to all be good. The machine is, according to your post, only about a year old. It should run the latest Ubuntu without too much bother. If you have the feeling that this is too much for it, you might want to try Lubuntu, which is a bit lighter.

As far as optimizing the system goes, I would suggest just using it for a bit to see what you really use it for, and see if it really needs pepping up in any way. I tend to always want the best/ latest/ greatest, and have spent too much money and expended too much effort a number of times on the assumption that I would need more than I really did. If the machine is running well and doesn't have any problem, the best advice I can give you is to just get on with using it, and wait to see if any shortcomings show up. If not, you are already getting the most out of it that you need.
