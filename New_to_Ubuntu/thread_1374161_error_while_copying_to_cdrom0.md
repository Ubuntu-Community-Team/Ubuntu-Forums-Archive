### Post by warfacegod on 2010-01-06
Once you burn to CD-R, that's it no more burning. They can be used only once. What you want is CD-R*_W_*.

That will allow you to erase and add data to the disc.

R means record once

RW means rewritable

### Post by sandyd on 2010-01-06
depends if your using "multisession" burning.
if your using multisession burning, you will be able to add more tracks (for data at least)
if your not using multisession, you only get to burn once.
