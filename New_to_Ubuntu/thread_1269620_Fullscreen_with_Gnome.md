### Post by wynn22 on 2009-09-18
I am running Gnome (default desktop manager) and have two bars on the screen one at the top and one at the bottom.  When I launch anything fullscreen (Guild Wars for example) it initially changes the screen and then positions the launched item within the constraints of the desktop bars. 

I have been trying to get JDarkRoom (full screen editor written in java) working and it states about setting a properties variable to force full screen, which I set. It did run but still within the confines of the two bars on the desktop. 

Any ideas on how I can resolve the issue to allow full screen for apps? Is there a setting in gnome some place to allow full screen?

Using ATI card with their linux drivers installed, disabled the driver to see if that made a difference but it didn't.

---

### Post by wynn22 on 2009-09-18
Resolved the problem, sort of -- System > Preferences > Appearance > Visual Effects, changed the option to 'None' it was set on 'Extra'. I now get full screen.  Visual Effects set on 'Normal' or 'Extra' the bars appear, so I assume their must be some sort of config switch somewhere?

Working at the moment though so I am happy :)

### Post by shae on 2009-09-19
You should check in compiz settings manager, but some programs do have problems running full screen with compiz on.  Unfortunately, you might just have to turn off compiz when you want that feature.
