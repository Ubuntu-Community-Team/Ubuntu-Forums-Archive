### Post by ibuclaw on 2008-07-21
When the PC boots up and appears to be at the login screen (the sound of the drum-roll is a good sign).
Press "**Alt+Ctrl+F1**" and see if it changes to Virtual Terminal 1 and displays screen for it too.

If so, there is something wrong with your xorg file and it needs fixing.
