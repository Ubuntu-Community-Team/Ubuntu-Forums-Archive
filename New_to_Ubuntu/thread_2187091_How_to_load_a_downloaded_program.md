### Post by The Cog on 2013-11-10
The download is a tar.gz file - like a zip file. You need to unpack the archive. On xubuntu I would right-click the file and choose extract here. This creates a folder with the contents in it. In there is an executable file called sjphone. Run that. It may be best to open a commantd-line terminal and run it by command from there, so you can see any error messages it prints.
