### Post by Buuntu on 2010-01-25
For the terminal, you can use:
```
adduser USERNAME
```and to set or change a user's password it's
```
passwd USERNAME
```
(obviously replace USERNAME with the actual name of the user)

For more information:
```
man adduser
man passwd
```
