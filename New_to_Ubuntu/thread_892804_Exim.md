### Post by brian_p on 2008-08-18
> **Badoxide said:**
> 

Please anyone at all what steps do I need to take to fix this before I go out the window here.

There is something wrong with one or more of the configuration files in /etc/exim4. We could investigate, find the problem and fix it but this should also do it:

```
sudo apt-get remove --purge exim4-base
```

```
sudo apt-get install exim4
```

Then stop and restart the service.
