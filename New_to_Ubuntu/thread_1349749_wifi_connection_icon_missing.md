> I currently have the same problem with Ubuntu 10.04, no help from the forum these days.

I'm also using Ubuntu 10.04. You can add the notification area to your panel to view the network connection icon.

1. right click to the panel.
2. Choose "Add to Panel..."
3. Find "Notification Area"; Add then close.
