### Post by lechien73 on 2012-01-08
Hi & welcome to the forums,

To me it sounds like your hard disk is failing. I'd suggest you back up any important data as soon as possible. Then you could try running a disk check on it by booting from a live CD/USB, dropping to a terminal and running fsck.

Don't run fsck against a mounted filesystem or you could cause serious damage.
