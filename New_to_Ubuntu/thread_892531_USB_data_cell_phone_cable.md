### Post by tahina on 2008-08-17
You could look into the [gammu/wammu]("http://wammu.eu/") application(s) (I think gammu is the command line backend whereas wammu is gui frontend).

Here's a link to a list of supported LG-phones: [http://cihar.com/gammu/phonedb/lg/](http://cihar.com/gammu/phonedb/lg/)

You can install it and try if your phone works (even if it's not in the list, maybe the list is not complete(?)), by typing the following in a terminal:

```
sudo apt-get install wammu
```

Wammu will then appear in the Applications>Accessories menu.
