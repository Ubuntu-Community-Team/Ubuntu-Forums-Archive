### Post by warfacegod on 2011-03-30
I've been down this path before and it's not a good one. It's theoretically possible to do what you're asking but the reality is that you'll end up with a mess. If you want an older gdm then use an older Ubuntu. Aside from that, you have a few options.

Ubuntu Tweak [http://ubuntu-tweak.com/](http://ubuntu-tweak.com/) has the ability to change the login background.

GDM2setup. I haven't used this in some time but it was rock solid when I did. Instructions for GDM2 Setup can be found here: [http://openubuntu.com/index.php/topic,182.0.html](http://openubuntu.com/index.php/topic,182.0.html)

There are a few other ways to go about changing the Login screen but these are the most reliable, in my opinion. You might also consider removing the Plymouth logo and text packages. That will help speed boot-up and may make your "look" more seamless.
