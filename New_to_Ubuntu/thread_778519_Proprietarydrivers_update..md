### Post by Nepherte on 2008-05-02
If there would be a newer version of the proprietary drivers you installed in the repositories, they would get updated with the update manager. That doesn't mean every new version of, for example, ati or nvidia vga driver would get in the repositories.

### Post by kpkeerthi on 2008-05-02
If there is a new version of the driver released by nvidia (or any package in the repo), it may never be added to the repository until next ubuntu version is released.
