### Post by 3rdalbum on 2013-03-12
Ubuntu 11.04 and above do not run the old Gnome 2 software. Instead, the desktop environment in use is Unity, and it doesn't have the same menus as Gnome 2.

You can access system settings through the top-right menu (the one that looks a bit like a cog and a power button) and going to System Settings.

If you right-click on the Ubuntu button at the top-left of the screen, and click Applications, you can bring up a list of installed programs. Some of these used to be in the System or Preferences menu in Gnome 2.

### Post by grahammechanical on 2013-03-12
Something in this thread may help you [http://ubuntuforums.org/showthread.php?t=1859961](http://ubuntuforums.org/showthread.php?t=1859961) What you consider "irksome," I consider simply "great." Perhaps Ubuntu is not for you.

The design of the user interface that you prefer was part of the Gnome 2 desktop environment. The Gnome organization are no longer developing Gnome 2 and have switched to Gnome 3 DE and with it Gnome 3 shell which does not have that look at all. What some people consider to be "Gnome Classic" is going to fall by the wayside and there is not much that the Ubuntu developers can do about it, even if they wanted to. And they do not want to do anything about it. The Gnome organization has indicated that they will provide the "classic" look as an extension to Gnome 3 shell.  How that will work out we will have to wait and see. Anyway, you would have to install Gnome 3 shell on Ubuntu and select it as your user interface at the login screen.
