### Post by cariboo on 2017-05-10
I personally don't like using the Software Center, I'd suggest using gdebi to install the package. Just navigate to where the downloaded package is, right click on it and select Open with an other application, choose gdebi from the menu and enter your password when asked.

Gdebi is in the repositories.
