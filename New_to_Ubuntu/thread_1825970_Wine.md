### Post by bolucpap on 2011-08-16
You mean you can't execute windows .exe files? For using programs under WINE, you don't actually execute the executables themselves, but rather you pass them to WINE as a parameter. For example, to run a program called HelloWorld.exe on your Desktop, type: ```
wine ~/Desktop/HelloWorld.exe
```
