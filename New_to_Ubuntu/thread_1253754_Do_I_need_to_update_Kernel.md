### Post by freak42 on 2009-08-30
All Ubunut Versions (except long-term support releases) have support for 18 months, so your version of Ubuntu 8.10 released in October 2008 will get security updates until April 2010. So you don't need to update to 9.04. 

In the future if you're not the type who always wants the newest and greates, but rather a stable system without the need to upgrade often, consider using a LTS Version of Ubuntu (Long-term support) which have support for 3 years.

BTW, 8.04 was a long-term support version, I am not sure which upcoming version will be the next LTS version (probably 10.04 ) (anyone?)

### Post by earthpigg on 2009-08-30
youtube being choppy with 9.04 is merely an example of the ebb and flow of quality that comes with a six month release cycle.

every 6 months, ubuntu tries a bunch of new stuff. for every 1 new thing tried that sucks, hopefully 5 will be an improvement, and hopefully the subsequent release corrects that one thing that sucked -- in addition to trying several new things again (one or two which may suck).

if you want to avoid the 'ebb and flow', stick to the LTS releases.
