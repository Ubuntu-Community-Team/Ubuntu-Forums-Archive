### Post by rbc on 2008-11-19
If it is a U3 partition, go to the web site of the thumb drive's manufacturer. Usually they will have a method for uninstalling that partition. I know Sandisk does. There is a way to prevent it mounting without removing that U3 partition (if that is what it is) but this will prevent ALL removable media from automounting. Post back if that is what you want
