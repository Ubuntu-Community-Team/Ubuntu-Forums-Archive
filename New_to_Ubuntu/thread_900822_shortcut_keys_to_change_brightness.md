### Post by hopkinsjeni on 2008-08-26
The keyboard shortcuts on your laptap wont work until you assign them.
Go to System > Preferences > Keyboard shortcuts.
You should be able to program your shortcuts there, I had to do the same when I changed for volume, brightness, etc. I have a HP.
One other idea is to go to
System > Preferences > Keyboard and make sure you have it set to the correct keyboard. I use Fn + f7/f8 as well and my keyboard is set to Generic 105-Key (intl) PC

hope one of those work for you
