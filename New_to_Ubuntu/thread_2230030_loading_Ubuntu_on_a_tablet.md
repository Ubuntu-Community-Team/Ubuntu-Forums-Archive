### Post by grahammechanical on 2014-06-16
I think that you are trying to do something that Xubuntu cannot do and that Ubuntu is also not able to do. To start with the usual Ubuntu image is built for Intel and AMD CPUs and I think that the Nexus devices use ARM designed CPUs = incompatible code.

You should be looking at the development of Ubuntu for phones and tablets and not desktop Ubuntu and its flavours.

[http://developer.ubuntu.com/start/ubuntu-for-devices/installing-ubuntu-for-devices/](http://developer.ubuntu.com/start/ubuntu-for-devices/installing-ubuntu-for-devices/)

Please understand that all this is still experimental. The code is still under development and is made available primarily for developers to use for building applications to run on Ubuntu pones and tablets. In fact the image for the Nexus 7 2012 is listed as depreciated.

[http://developer.ubuntu.com/start/ubuntu-for-devices/devices/](http://developer.ubuntu.com/start/ubuntu-for-devices/devices/)

That being said there is some evidence that upwards of 10,000 people are using Ubuntu for phones and tablets on their devices.

[http://developer.ubuntu.com/2014/06/10000-users-of-ubuntu-phone/](http://developer.ubuntu.com/2014/06/10000-users-of-ubuntu-phone/)

Regards.
