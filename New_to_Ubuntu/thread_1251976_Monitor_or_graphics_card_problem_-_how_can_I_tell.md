### Post by Bearly Able on 2009-08-28
Hi all,

Sorry if this is a stupid question, but I'd really appreciate some help.

I've been running Intrepid for several months without any trouble, but in the last week or so, I've had intermittent problems with the display, in that the screen becomes "greyed" - not monochrome, but just shifted in colour.  Yesterday, it suddenly shifted to yellow for a few seconds, then corrected itself.  I'm guessing it's a hardware problem, although we dual-boot with XP, and the couple of times I've had to boot into XP, it's appeared normal.  However, we very seldom use XP, so it's probably just sheer luck that the problem hasn't occurred there.

How do I know if the problem is with the monitor or the graphics card?  The monitor is a TFT of uncertain age - we were given it second-hand a couple of years back, and I'm guessing it was several years old then.  The graphics card came with the PC, so is a little over two years old.

I'd be really grateful for any help you can offer.  I can't afford to replace one, only to discover the fault lies with the other.

Thanks in advance,

Lesley

### Post by LewRockwell on 2009-08-28
switch out the monitor and if it still occurs then you'll need to investigate the graphics hardware

if it's on-board(built into the motherboard) then you can probably add a graphics card to fix it

it should be noted, however, that it makes absolutely no sense to spend serious money on a graphics card if you've got a dated machine

we're buying year old machines for $100.00 cash around here(what with Walmart and Newegg selling them new for around $300.00)

whatever you do...don't waste your hard-earned money!
