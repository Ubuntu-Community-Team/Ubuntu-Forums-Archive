### Post by grahammechanical on 2014-04-18
If the CPU is 64 bit then install the 64 bit version of Ubuntu regardless of whether the CPU is made by Intel or AMD. If the CPU is 32 bit then install the 32 bit version of Ubuntu regardless of whether the CPU is made by Intel or AMD.

Ubuntu i386 = 32 bit Ubuntu that can run on any Intel or AMD 32 bit CPU.
Ubuntu amd64 = 64 bit Ubuntu that can run on any Intel or AMD 64 bit CPU.

32 bit Ubuntu will run on a 64 bit CPU because the architecture of the CPU is backwards compatible with 32 bit operating systems. 64 bit Ubuntu will not run on a 32 bit CPU because the architecture of the CPU is not forward compatible with 64 bit operating systems.
