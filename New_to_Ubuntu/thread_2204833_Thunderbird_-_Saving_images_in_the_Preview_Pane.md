### Post by deadflowr on 2014-02-10
Don't know what your screen looks like, but whenever I want to save an image from thunderbird I simply right click the image and select "save as image".
Then it opens a window with a default image name and somewhere in my folders to place it.
It also allows me to save as link.

As for upgrading, Ubuntu can be upgraded two ways, really.
When 14.04 comes out you can do an over the network upgrade, or download a fresh copy.
You'll find a lot of users will recommend downloading a fresh copy, as over the network upgrades have bitten a few users.
I, myself, have had great success doing over the net upgrades.
But I refuse to expect the same for others.

If you do download, backup what you have.

### Post by deadflowr on 2014-02-18
> Regarding over the network upgrading. If you do that, does it import all  the applications I have built up in 13.10...,  or is it backup,  download and then restore the backup..? 

Network upgrade upgrades ALL the packages you have, if they are upgradeable.
(which for the most part, they are)
However some packages may or may not be supported any longer and those get pulled.
(little things like that can happen)
It won't do a backup.
You can backup your own files, but that's on you to do and not something that it tells you to do.
And the typical backup programs don't backup the programs installed.
But there are methods of creating a backup for the apps you have.
(google them, if you want)
