### Post by Toz on 2014-01-06
Hello and welcome to the forums.

You can use xrandr to manually adjust the brightness on the external monitor. There is also an application that you can use that interfaces with xrandr. More information available [here]("http://lordamit.blogspot.ca/2013/03/changing-secondary-external-display.html").
