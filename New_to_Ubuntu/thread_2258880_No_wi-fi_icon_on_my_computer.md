### Post by grahammechanical on 2014-12-31
Ubuntu is under constant development. A new version is released every six months at the end of April and the end of October. Instructions for doing things change over time. There are currently three versions of Ubuntu still being supported. Ubuntu 12.04 (April 2012) and Ubuntu 14.04 (April 2014) are supported for 5 years from the release date. And then there is Ubuntu 14.10 (October 2014). That is only supported for 9 months from released date. So, you see telling us the version of ubuntu that you have installed is important.

There are also several "flavours" of Ubuntu with different user interfaces. I am assuming that you are using Ubuntu with Unity. If I give inaccurate instructions it is because you have given insufficient information.

When we install Ubuntu we are asked to confirm that the machine is connected to the Internet. Was this machine connected to the internet during installation of Ubuntu? If it was and you now have a wired connection (ethernet) to the router then you will see a Network Manager icon at the right side of the top panel. That icon will look like two arrows going in opposite directions. Click on that icon and look at the menu that drops down. Do you see a tick mark against Enable Networking and against Enable WiFi? If WiFi is disabled then you will not see any kind of WiFi icon.

There are a few reasons why WiFi is disabled.

1) The user has not enabled WiFi.
2) The WiFi adapter is not switched on.
3) The operating system has not loaded a driver of the WiFi adapter.

This is where knowing about the hardware of your machine will help. Laptops have a function where WiFi can be switched off at the keyboard to save battery power. If this has happened, then Ubuntu will not see a powered on WiFi adapter.

If Microsoft Windows is installed on the machine and WiFi is switched off in Windows, then Ubuntu will not see a powered on WiFi adapter.

Oh, one more thing. You may simply need instruction on how to set up an automatic WiFi connection to the router because you are unfamiliar with Ubuntu. We do not know what your need is.

Regards.
