### Post by hansolo4949 on 2011-02-03
First of all, how experienced are you with networking? if you are an old timer when it comes to it, and you are fairly experienced with The terminal in linux, you should go with Ubuntu server edition. If the answer to those two is a NO, then just stick to Ubuntu. As a matter of fact, with those specs, I would stick with Xubuntu, if you want everything to run along fairly speedily. If you want a GUI in Ubuntu server, it's just as easy as typing into the terminal: sudo apt-get install ubuntu-desktop. That should be all you need, if you know how to set up sharing.

### Post by TechWiz2100 on 2011-02-03
You an use either but server edition has no GUI meaning that you will have to get very intimate with terminal commands. If you are afraid of terminal, you can grab desktop edition (even though I suggest you embrace the terminal and all its glory) but you might get a little overhead from the GUI as well as "potential" security risks.

IDE is probably fast enough for your purposes but it may become a bottle neck for video streaming and multi access.

Also I think there are also some media suites hovering around in the interwebs like Twonky that are pretty much all in one solutions for your problem.
