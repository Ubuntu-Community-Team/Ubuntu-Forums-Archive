### Post by grahammechanical on 2015-03-31
> [COLOR=#000000]Because Nemo and Cinnamon are compatible with 14.04, does releases make everything backwards compatible, [/COLOR]

Not necessarily. Each release of Ubuntu has different libraries to the previous release. It is up to the developers of Nemo and Cinnamon to keep their software up to date with Ubuntu. There are several Cinnamon related packages in the 15.04 software centre. Nemo is also there. So, that is a good start. But has anybody been testing Cinnamon on 15.04 during the 15.04 development period? Has anybody reported any bugs and have they been fixed?

Personally, I would not upgrade to a new version of Ubuntu until I had dual booted with it and set up the new version the way I like it. I would not try to remove Nautilus because it does more than work as a file manager. The Gnome developers have tightly integrated Nautilus into the Gnome Desktop environment.

> [COLOR=#000000]Is it worth making any of these changes with the release of 15.04 so close as it may not be compatible with Nemo or Cinnamon?[/COLOR]

What changes? Unless you wish to take part in testing 15.04 prior to its release, then it is now best to wait until it is officially released near the end of April. Ubuntu developers focus on developing Ubuntu and every new release needs testing prior to being released. Software such as Cinnamon also needs testing during the development period of Ubuntu.

Why not install the next Ubuntu development release (15.10) in one months time and help the Cinnamon developers produce good software?
