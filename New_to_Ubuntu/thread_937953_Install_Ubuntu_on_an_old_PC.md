### Post by snowpine on 2008-10-04
Crunchbang is another great distro for older hardware. It's similar to Fluxbuntu in many ways but is more up-to-date (based on 8.04 instead of 7.10). Fluxbuntu has the edge in performance, though, because it uses a lightweight suite of applications (Kazekahase browser, Rox file manager, Xmms music player, etc., though you can, of course, install "full" applications like Firefox, the entire Openoffice suite, etc. if you prefer). I would choose either Crunchbang or Fluxbuntu over Xubuntu, personally. The only real advantage to Xubuntu is that as an "offical" distro, it's more widely used, so support resources are easier to come by. Xubuntu is not really as "lightweight" as you might think.

### Post by snowpine on 2008-10-05
From [http://lxde.org/lxde](http://lxde.org/lxde)

> System Requirements

LXDE has been tested under following hardware configurations, and runs well.
CPU 	RAM 	Hard Disk 	Speed
Pentium II 266 MHz 	192 MB 	5400 rpm 	moderate - fast
VIA 400 MHz 	256 MB 	5400 rpm 	moderate - fast
Pentium III 600 MHz 	512 MB 	4500 rpm 	fast
QEMU emulation on AMD Athlon 1.6 (1.4 GHz) 	128 MB 	QEMU disk image 	fast
AMD Athlon 1.6 (1.4 GHz) 	512 MB 	7200 rpm 	very fast

Recommended configuration

    * The hardware requirements of LXDE are similiar to Windows 98 (Maybe a little bit higher). An old Pentium II CPU is enough.
    * After X11 and LXDE are started, the total memory usage is about 45 MB on i386 machines. (This value may be higher or lower according to different system configurations.)
    * Though LXDE itself doesn't require better hardware, other applications under X do need it. For example, Firefox and OpenOffice.org 2 are quite memory-hungry. So it's recommended that you have a RAM of more than 128 MB.


If you go to the page, they have a list of distros that use LXDE, plus instructions on how to add it to a minimal install of Ubuntu, Debian, etc. If you did not like the Fluxbuntu user interface, for example, you might find LXDE has a more familiar/traditional desktop feel. SliTaz uses a modified version of LXDE, and it only uses about 30 out of 256mb of ram on my old pentium 3 laptop (until I open applications like Firefox, of course). My only gripe is I don't like the default LXDE wallpaper, but that's easy enough to change (using PCmanfm)!
