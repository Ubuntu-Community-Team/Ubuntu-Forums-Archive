### Post by Jonathan Precise on 2014-01-03
**[SIZE=5]THE EASY WAY[/SIZE]**

Try:

```
pastebinit || echo $?
```

Wait 10 seconds. If an error appears like:
```
The program pastebinit is not currently installed. To install it type in: <sudo apt-get install pastebinit>
127
```, then go to **ALTERNATIVES**

If nothing happens, press Ctrl+D. Then try:

```
pkexec apt-get install -f -y 2>&1 | pastebinit
```

Post the URL.

**[SIZE=5]ALTERNATIVES[/SIZE]**

Try:

```
touch ~/Desktop/apt-get-install-f-y.txt
pkexec apt-get -f install -y &> ~/Desktop/apt-get-install-f-y.txt
```

When you see again something like:

```
damionem@ubuntu-desktop
```

Then go to the desktop and attach a file called [FONT=Courier New]apt-get-install-f-y.txt[/FONT] that is in your desktop. See the attachment.
