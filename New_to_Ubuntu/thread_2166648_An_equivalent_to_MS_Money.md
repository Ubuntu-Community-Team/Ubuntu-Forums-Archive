### Post by Linux90s on 2013-08-10
skrooge, kmymoney, grisbi
you can find these in synaptic by serching for "money"

gnucash, homebank
you can find these in synaptic by serching for "accounting"

(these appear in 13.04 but should also be there in 12.10)

you will have to field-test each one to see which you prefer, the other option is to use Wine to run MS Money 99, but that will be more complicated.

gd luck!

### Post by PaulW2U on 2013-08-10
The only program that I've found comparable to MSMoney is KMyMoney.

But beware as you're not running Kubuntu and unless you have already installed some KDE programs, by installing KMyMoney you'll probably install a large number of KDE library packages. If you were running Kubuntu then I would recommend KMyMoney 100%. 

Unfortnately, I've not found a comparable GTK/Gnome alternative to MSMoney. :(
