### Post by QIII on 2024-06-15
Hello!  

If you have come from Windows, one of the things that you will find is that you won't want to just download and install software randomly.  Most Linux distributions have what are called repositories with software packages you can install.  If taken from the distribution's "Official" repositories, these packages have generally been tested against the distribution (in this case Ubuntu) to make sure they work.

Linux is not Windows, so there will be some unlearning/relearning to do.  Things will seem unfamiliar and confusing at first.  Just remember that you are "learning a new language", so to speak.  It all makes sense in the Linux context, but you will have to be willing to avoid demanding that your experience should be like Windows.

II would suggest reading and working through an Ubuntu Handbook.  Give me a few minutes to find a reference for you.  I'm sure it will be confusing, frustrating and bizarre for you -- until you learn to use Linux as you learned to use Windows.  Remember that you did not just magically start out knowing how to use Windows, either.

Since you will be confused, we are happy to help you through this.

Hang on for a bit while a review a few choices, and I'll post some links.

### Post by QIII on 2024-06-15
Sorry for getting back to this so late.  The wind kicked up while we were putting bird nets over some of the fruit trees and made a mess!  Sigh.  Farming.

Anyway ...

I wasn't able to find a very current Ubuntu handbook.  It seems as though there used to be a free .pdf that came out shortly after a new release, but I didn't find one that was trustworthy yet.

You can, however, start with some official Ubuntu information [here]("https://help.ubuntu.com/stable/ubuntu-help/").

You may use the Ubuntu standard App Center to find and install new application packages. Click the App Center icon in the Dock, or search for Software in the Activities search bar. 

(I would suggest installing Synaptic eventually.  It is a graphical front end that I think is better than the standard Ubuntu App Center. It is definitely better than Muon for Kubuntu!) 

As suggested above, let us know what sorts of packages you would like to install, and we can provide further help.

But please:  Refrain from downloading and installing packages you find randomly on the web.  You never know what you are getting!
