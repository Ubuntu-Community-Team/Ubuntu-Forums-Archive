### Post by TheFu on 2012-06-26
We'd like to help, but I can't answer all these questions.

I think the main thing to understand is that Ubuntu file names and directory names are case sensitive.

"Documents" is not the same as "documents".
"File.txt" is not the same as "file.txt".

**What is a torrent?** [http://netforbeginners.about.com/od/peersharing/a/torrenthandbook.htm](http://netforbeginners.about.com/od/peersharing/a/torrenthandbook.htm)

**What is an archive manager? ** Is is a tool that helps you manage archives, but that probably isn't too helpful.  An archive manager usually helps you handle ZIP, ARJ, RAR, TGZ, tar.gz, .bz, .Z and other sorts of compressed files. Usually this tool will let you decompress or compress directories and files.  Sometimes an "archive" is an installation package.

As to installing applications, it is important for someone new to Linux to always install programs using the built in package management tools. "**Software Center**" or "Synaptic" is what you need to run. Find the application in that list and install it. 

The** modem issue** will require specific details to troubleshoot, so asking that question alone with lots of details about 
a) the ISP and 
b) the exact modem involved
will be needed.

I wish I could be more help, but I don't run the normal Ubuntu GUI, so I don't have any useful first hand experience with those issues.  

If you could, please **state the exact version of Ubuntu that you are running** and which GUI you are using to get more specific help. Ubuntu isn't like MS-Windows. There are 20+ different GUIs in use by the people here, though most are using Gnome2, Gnome3, Unity, XFCE, LDXE, or KDE.  This information will help get you the best answer for your Ubuntu version.

### Post by ajgreeny on 2012-06-26
Some answers at least, but not all of them.

I can't help with the modem problem other than suggesting you right click on the network icon in the panel and choose "Edit connections".  Go to the mobile broadband tab and double check that the connection showing is set to connect automatically and available for all users.  You will need to give your user (sudo) password to make changes.

For LibreOffice extensions you should open up LO, go to the menu **Tools ->Extension manager**, and there you will see a link "Get more extensions online".  This will open your web browser, and you can then download extensions in .oxt files.  They will download to whatever folder you have set in the browser preferences dialog.  Back in the LO Extension manager, click on the Add button and navigate to the .oxt file and you click on OK or install, I can't remember the details of the buttons.

The Archive manager is most closely related to Winzip in windows, and will open and extract most archive file types that exist, though many of them are probably totally unknown to most windows users, eg **.tar.gz** and **.tar.bz**.  Apart from that it is used in much the same way.

What downloads have you got that do not open?  If you are trying to install applications that way you should forget your old windows ways and use the software-centre instead.  What are the bible applications you have and can not find?

No idea about the defibrillator thing that appears, and I am not sure about your final query regarding your editor's files that appear back in Downloads.  Explain in more detail please.
