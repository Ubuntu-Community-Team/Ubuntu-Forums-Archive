### Post by daniell59 on 2012-01-13
I presently have Ubuntu 10.04. I would like to do a clean install to 11.10. Before doing so, I would like to save my bookmarks on a flash drive. I would also like to save my address book. I am using Thunderbird. Any advice would be appreciated.

Thanks

### Post by lechien73 on 2012-01-13
If you're using Firefox, you can export the bookmarks by going to **Bookmarks -> Show all bookmarks**. Then select the **Import and Backup** and select **Backup**, you can then import the bookmarks into the new computer.

Your Thunderbird addressbook is stored in the ~/.thunderbird directory. There will be a subdirectory with a .default extension, the addressbook is called abook.mab and is in that directory.
