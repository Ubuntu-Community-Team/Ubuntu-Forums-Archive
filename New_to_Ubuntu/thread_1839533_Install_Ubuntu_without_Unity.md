> **allstar1971 said:**
> Unity may be the wave of future...but not for me. i do not like the big icons on the left.

You can make the icons smaller. Install compizconfig-settings-manager, run it, go to the Ubuntu Unity plugin and there's a setting in there for changing the sizes of the icons.

Ubuntu 11.10 is coming out soon; maybe try Unity in that and see if you like it more. The icon sizes can still be changed.

Sitting on Gnome 2 is not really an option for the long-term; it's no longer maintained and pretty soon will be impossible to install on newer distros. You could try Gnome 3 in 11.10, Unity, KDE, LXDE etc.
