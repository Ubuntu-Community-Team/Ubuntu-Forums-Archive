### Post by Moop on 2009-02-21
I'm not sure what you mean by "fixing" a theme but you need admin rights to copy anything into usr/share/themes. Use gksudo nautilus in a terminal to open a nautilus window and navigate to where you want. 

To install a theme all you need to do is drag it into the theme window. System-Preferences-Appearance-Theme.
