### Post by jward3010 on 2009-09-30
It is realistically always required. For your system give at least 2GB to swap, if you're having problems increase to 4GB.

It's primary function is to be a spillover if the Operating System runs out of standard RAM i.e. the 2GB's you have installed. It's inefficient and slow and for that reason noy used that often but if a particular app needs a lot of RAM, rather than the PC crashing from running out of memory it will allow the use of the swap partition to keep going.
