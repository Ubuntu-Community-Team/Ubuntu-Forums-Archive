### Post by Mark Phelps on 2009-06-23
The short response is ... no, at least, not in any simple way.  

Sure, you can probably get folks to upload some graphics of screens for you, but you will need the apps as well, and to run those, you will undoubtedly have to update lots of libraries, and then, you will most probably have to recompile the apps using the new libraries, which means you will need the source as well.

And, let's not forget the kernel changes that are needed to support the new app features ...

So basically, if you're skilled at making new kernels from scratch, as well as building new app executables from scratch, you might be able to do this.  But generally, the answer is -- that's what the upgrade is for.
