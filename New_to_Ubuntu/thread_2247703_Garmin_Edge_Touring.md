### Post by Vladlenin5000 on 2014-10-09
Garmin Edge Touring is a standalone GPS navigator. It doesn't require a computer to work; it may require a computer to update its firmware and maps but if the manufacturer doesn't provide solutions for Linux then there little to nothing anybody else can do.

Use it with a Windows PC for the occasional update you might need. Anther solution would be installing a virtual machine but a Dell Inspiron mini is nowhere near the bare minimum for such task. Plus, 13.10 is EoL and no longer supported either by Canonical or this forum.

### Post by Quarkrad on 2014-10-10
I have a Garmin Edge 500 as used it for some time on 12.04 and 14.04 - I use firefox to upload the data to Garmin (Garmin Connect). Until recently there was a plugin called Garmin Communitcator for fixefox that made the connection automatic but this is broken in the latest release of firefox.  A little inconvenient - I can still upload my rides to Garmin but instead of pressing the UPLOAD button on the Garmin web page I have to upload manually by selecting the latest Activity on my Edge 500.  I don't load maps or reference points from Garmin as such (is this possible?) - I use other web sites to create my routes/maps (sometimes reference points) and load them onto my Edge 500.  I'm always creating new routes and downloading them onto my Garmin - Ubuntu/linux is not really the issue, it all works fine.  You just need to create .gps or .tcx files from the various mapping sites available and then download them onto your Edge - my text above is about loading your rides (the Activity) from your Edge to the Garmin web site (Garmin Connect).
