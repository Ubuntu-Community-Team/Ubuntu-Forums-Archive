### Post by coffeecat on 2011-04-22
Why do you want to erase the hard drive with dban? Unless you want to put any personal files beyond the reach of recovery with data-recovery software, "scrubbing" a drive is just unnecessary ritual. Simply start the Ubuntu installer and select "use whole drive". Or use the manual/advanced option and create a partition layout to your liking.

### Post by coffeecat on 2011-04-22
I don't have experience of that particular model EeePC but the ones I've tried generally work well with Ubuntu. What you can do is to boot up with a live USB and simply try everything out. Not just the default software, but the hardware as well. Then you can see if there are likely to be any significant issues before you commit yourself to a hard drive installation.
