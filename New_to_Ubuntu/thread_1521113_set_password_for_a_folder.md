### Post by Oak37 on 2010-06-30
I'm not aware of a way of enabling permissions that translate across different operating systems.

One way would be to encrypt the directory or the individual files utilising PGP keys and Seahorse, there's a guide [here]("http://ubuntufs.wordpress.com/2007/05/22/encryptingdecrypting-with-gnome/").

Alternatively you might consider [truecrypt]("http://www.truecrypt.org/") or something similar.
