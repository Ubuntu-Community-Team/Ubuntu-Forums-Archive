### Post by sliketymo on 2009-12-27
You could not "access" the full 4 Gigs of memory because your Vista is most likely a 32-bit version.To fully utilize the entire 4 Gigs of memory on your box,you really should be using a 64-bitOS.You are aware that you can set Vista up to function like XP by changing the theme to "Classic". I am a happy camper using a Dell 530 Dual booted with Vista x64, and Karmic.I have my ram maxed out at 4 Gigs.You should really remove the the 512 ,and install 2 1Gig sticks in alternate slots.I have always  heard that you should actually swap out,ie:add ,remove , memory sticks in pairs.As to your question about other hardware,you may want to have a look at System 76,or perhaps TuxMachines.Hope this helps a little.I am aware that there are quite a lot of "Horror" stories it seems regarding Dell hardware,and customer service,support.For myself,I have had no such incidences,and am perfectly satisfied.

### Post by bdaman80 on 2009-12-27
Well if you are wanting MS Office 2007 on an Ubuntu or any Linux machine then you will be disappointed .  I will tell you that OpenOffice can take care of most all the same Word processing and Spreadsheet needs most people have.  

sliketymo is correct about the fact that  a 32 bit machine cannot make use of more than 3GB of ram on any OS.  A 32 bit OS cannot make use of more than 3GB, and a 32 bit processor cannot use more than 3GB either.  Could the ram be the wrong style for the machine?  I have had that problem in the past on some of my boxes.  

As for an Ubuntu machine.....I would suggest downloading a live CD (64Bit if you processor is 64, or 32Bit if otherwise or unknown ), give it a test drive, and if you like it, then look for a machine to home install Ubuntu on.
