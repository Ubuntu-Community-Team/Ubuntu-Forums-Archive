### Post by wildman4god on 2009-10-05
wubi doesn't run windows and ubuntu at the same time, wubi just installs ubuntu on a virtual hard drive so it dual boots without touching your partitions, though hard drive proformance is degraded a little. unfortunatly linux in general doesn't have as good power managment as windows, mainly because the power managment features built into the computer where made in colaboration with microsoft for their windows os, while linux has been trying to reverse engineer the power managment feature. Ubuntu 9.10 is supposed to have better power managment because it is switching from HAL to devicekit, this should help some, also, the reason windows xp gets that much battery life is because it throttles the cpu, in other words it slows the cpu down when the computer doesn't need more power to save power, it speeds it up only when it needs the extra speed. in ubuntu there is a panel app called cpu freq, add that to your panel and set it to power save, this will slow your cpu down and save power. the cpu and the display use more power than any other components in the netbook, so you can also try dimming the display a little, or setting it to dim when your not using it, kinda like an mp3 player or cellphone shutting off it's backlight after 15 seconds of no activity to save power.
 
Edit: Also the next release of ubuntu netbook remix is supposed to have perfect netbook hardware support, so you may want to wait until 9.10 releases and use that.
