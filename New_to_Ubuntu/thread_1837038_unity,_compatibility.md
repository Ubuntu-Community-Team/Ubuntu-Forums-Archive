### Post by LowSky on 2011-09-01
Unity is just a different desktop environment built to work using gnome as a backbone. More info here: [http://unity.ubuntu.com/](http://unity.ubuntu.com/)

As far as I know it isn't available for other distros.  A person could theoretically take the Unity packages from the Ubuntu repositories and modify them to work on another version of Linux. Not something for the faint of heart. Especially when Gnome 3's gnome-shell is a similar experience.
