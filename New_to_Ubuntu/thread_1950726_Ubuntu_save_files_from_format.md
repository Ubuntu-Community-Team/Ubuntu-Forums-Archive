### Post by zombifier25 on 2012-04-01
Yes. Use a Ubuntu liveCD/USB, get in your Windows partition and copy whatever you want to save to an external hard drive.
The instructions for writing a LiveCD/USB is all here: [http://www.ubuntu.com/download/ubuntu/download](http://www.ubuntu.com/download/ubuntu/download)
Scroll down ;)

But if you only want to get some files and NOT want to install Ubuntu, you should use a really lightweight Linux distro instead, like [Puppy Linux](https://en.wikipedia.org/wiki/Puppy_Linux)

### Post by Mark Phelps on 2012-04-02
> **Morningleaf said:**
> So once i do this, is there any way to switch back to windows?

IF you mean WITHOUT reinstalling, the only way would be to get Win7 boot working again.  But, from what you already wrote, it looks like you tried all the repair options.

So, the only option then would be to restore your HP PC to its original state.

Since your PC came with Win7 preinstalled, there should be a way to boot, holding down some key combination, and be booted into an HP recovery feature -- that will restore the original HP Win7 installation.  The HP documentation should tell you what keys to press to do that.

But be aware, if that works, it's likely to erase your ENTIRE drive, reload Win7 (from a compressed HP image on the drive), and return the PC to the state it was in when you first removed it from the box and started it up.

That's why you need to save off any files and settings to an external drive before you do this restore.
