### Post by ~Plue on 2011-04-10
both [WoW]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=1922") and [Rift]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=12482") seems to have good results in the wine appdb, but there could be a little performance difference.. the best option would be to test it yourself

so instead of wiping windows now, try out ubuntu by installing it side by side with windows by either booting>installing from the live cd or installing ubuntu within windows using wubi.exe on the cd
