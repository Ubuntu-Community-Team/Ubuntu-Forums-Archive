### Post by dwasifar on 2011-04-23
I don't think you'll have any problems with that configuration.  Use the 64-bit version.

You will want to run the Restricted Drivers tool after installation to install the nVidia restricted drivers.  It will offer you that soon after your first boot.  And after you do, your splash screen will look kind of ugly, but that's fixable if it bothers you.

Other than that, you should be good to go.
