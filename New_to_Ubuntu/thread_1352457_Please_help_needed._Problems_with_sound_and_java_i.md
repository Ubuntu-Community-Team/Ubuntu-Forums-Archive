### Post by avtolle on 2009-12-11
On java, just a guess, but you may need to install Sun Java. While the open-source java works well in many instances, there are times that the Sun package is looked for by the site. System>Administration>Synaptic Package Manager, then search for java. If Sun is installed (green filled box by name), then there's something else causing the problem. If not installed, mark for installation.
