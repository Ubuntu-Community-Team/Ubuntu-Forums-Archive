Wolvix is a great distro for a netbook, if you don't mind straying away from Ubuntu.

### Post by admiralspark on 2010-03-25
As an alternative to anything Ubuntu (don't get me wrong, I love all things Debian), maybe you can try Knoppix or Puppy Linux. 
Knoppix is built to run form a USB using the LXDE environment, it's lightweight/efficient/Debian-based.

Puppy's the one I'd recommend though: it's 50Mb small, loads completely into memory, and can be installed to the harddrive or run from a USB. It's VERY lightweight, efficient, all of my drivers worked right away (not even Ubuntu can claim that), and it's LOADED with apps too. 

In fact, a special version just for the Atom and Celeron cores is going to be released soon: check [here.]("http://www.puppylinux.org/news/")

If you're set on using Ubuntu, I recommend the normal Ubuntu, either 9.10 or 10.04.
