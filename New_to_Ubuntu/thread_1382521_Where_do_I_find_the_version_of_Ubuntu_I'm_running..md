> **GSam said:**
> I've been running the same version for some time now and the usual "Upgrade Ubuntu" thingie doesn't come up when I run the regular Updates.

You might have it set to ignore anything except LTS updates. Go to System > Admin > Software Sources > Updates and make sure you've got it set to upgrade to non-LTS versions.
