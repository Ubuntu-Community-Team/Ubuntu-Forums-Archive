### Post by MattBD on 2008-08-23
If it won't run in Wine, then it probably won't run at all in Ubuntu, but it does take a bit of work getting something going in Wine. There's a couple of virtualisation programs you could use to run Windows on top of Ubuntu (my favourite is VirtualBox), but you still need a copy of Windows to run in the virtual machine. Or you could use ReactOS in VirtualBox (basically it's an open source clone of Windows that's freely downloadable and will run many Windows apps, but no guarantees that it will run Bejeweled).

### Post by firefeather on 2008-08-23
It appears other people have gotten it to work; have you seen [the AppDB page]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=4300&iTestingId=28510")?

If that doesn't help, I believe you have three options for your problem:

1. Install windows in a virtual machine; for example, VirtualBox has a seamless mode.
2. Try Bejeweled on each version of Wine that comes out, patiently waiting until Bejeweled works
3. Settle for a compromise (You might miss one or two game modes in Geweled, but it is a great alternative!)

In my opinion, the problem is that Bejeweled is closed-source software; people can't fix it on their own.
