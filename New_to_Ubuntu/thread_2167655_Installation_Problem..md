### Post by grahammechanical on 2013-08-14
As you say it is a clean 1TB hard drive. Every OS needs a formatted hard drive and at least one partition. Ubuntu installations format the hard drive as an Ext4 file system and the installer creates 2 primary partitions. The first partition is where Ubuntu is put. The second partition is set up as an Extended partition and in that Extended partition the installer creates on Logical partition that Ubuntu will use as a swap partition.

Now you know why the installer has an option of Use the Entire Disk which does stuff without baffling us with science. :)
