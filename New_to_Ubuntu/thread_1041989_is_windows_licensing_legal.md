Actually, you're not licensing the computer, you're licensing their software, which is why you have to pay (too much!) for the Windows installation CD.  Once the software is licensed, you are allowed to have it installed on ***one computer,*** which would be your computer...you cannot install it on any other computer simultaneously.

So instead of licensing your computer, you are licensing that particular installation disk to install and run on your computer, and *only* that computer.  You want to run Windows on another computer, you must buy a separate installation disk.
