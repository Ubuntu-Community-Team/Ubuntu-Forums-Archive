### Post by Potters Son on 2010-05-11
Sorry, Fire-Roller's hidden in the menus by default. To un-hide:
1. Right-click on the Applications menu, and go down to Edit Menus
2. Click on the Accessories entry in the left-hand pane
3. Locate "Archive Manager" at the top of the list in the right-hand pane and select the checkbox.
4. Hit close.
5. Applications -> Accessories -> Archive Manager. Enjoy!

BTW, if you ever need to start a program by its command-line name and/or you're too lazy to use a mouse, you can bring up the Run Program box with a quick Alt+F2, type "file-roller" (usually, it also has tab completion too! if it shows the rest of the program name, just hit tab.), and press Enter.
