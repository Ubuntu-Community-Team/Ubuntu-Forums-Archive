### Post by grahammechanical on 2014-03-29
open a terminal and run

```
lsb_release -a
```

This is what I get

> No LSB modules are available.
Distributor ID:    Ubuntu
Description:    Ubuntu Trusty Tahr (development branch)
Release:    14.04
Codename:    trusty

On your system saucy will be there instead of trusty. And 13.10 instead of 14.04. Did the wallpapers change? Have you since run an update?
