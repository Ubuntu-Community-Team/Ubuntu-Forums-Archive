### Post by grahammechanical on 2015-05-11
It is not unusual for a live session to only have one icon one the desktop labelled Install, especially if it is a member of the Ubuntu family that uses cascading menus. Ubuntu does it differently. We expect to see more icons with Ubuntu but may be only one icon called Install on the desktop. Others would be in the Launcher and Dash.

This thread will show images from a Lubuntu lives session. Note the image of a Lubuntu live session desktop with one icon but a bottom panel with access to cascading menus.
