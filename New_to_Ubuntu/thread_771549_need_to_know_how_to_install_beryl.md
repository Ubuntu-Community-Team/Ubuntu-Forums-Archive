### Post by scragar on 2008-04-27
beryl has been merged with compiz to form compiz fussion, which is installed by default, you can turn it on or off under:
System>Preferneces>Appearance
and the visual effects tab.

If you can't work out how to use the cube then you could install the manager:
```
sudo aptitude install compizconfig-settings-manager
```
that appears as the option at the bottom of visual effects once you close and reopen it.
