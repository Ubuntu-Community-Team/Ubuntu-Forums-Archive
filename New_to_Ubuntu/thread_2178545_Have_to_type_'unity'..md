### Post by grahammechanical on 2013-10-04
When I see images like that I ask myself, what have you done? If that is the printout when you run unity, then I would guess that there is something wrong with gnome-screenshot. But if you run screenshot through the terminal, then I would not be surprised at those error messages because Unity is not running. May I suggest something?

At the Grub boot menu select Recovery mode. I have to assume that you are using Ubuntu 13.04. So, you will find Recovery mode in the Advanced Options sub-menu. At the Recovery menu select Resume. If that gets you to a working desktop then change videos drivers. System Settings?Software and Updates>Additional Drivers tab.
