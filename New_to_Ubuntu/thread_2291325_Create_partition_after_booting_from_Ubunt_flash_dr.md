### Post by apolo3 on 2015-08-19
I have already installed win7 and I do not want to cause a problem. I just want to install Ubuntu on a different partition which I have not created yet. Is it secure to create the partition after starting the "boot from flash" procedure? I presume that the installation will prompt me where to install Ubuntu, so I will have the option to create a new partition and select the size. I just want to be sure that this will not affect win7.

---

### Post by Bucky Ball on 2015-08-19
You are good to go. Choose 'Something else' and create your partitions in free, unallocated space. If you don't have any and need to resize the Windows partition, go back and reboot into Windows to do that. DO NOT resize the Win partition using Ubuntu.

Rule of thumb: manipulate Win partitions with Win, Linux with Linux.

Bottom line: Do everything you need to do to create the free space needed to install Ubuntu onto using Windows PRIOR to booting the Ubuntu install media so when you get to 'Something Else' you have free space to work with. 

Good luck. :)
