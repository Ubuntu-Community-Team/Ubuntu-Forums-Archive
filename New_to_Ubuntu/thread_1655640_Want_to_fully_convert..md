### Post by bcbc on 2010-12-30
You don't want to use LVPM unless you're on 8.04. 

You can use the script from this guide [http://ubuntuforums.org/showthread.php?t=1519354](http://ubuntuforums.org/showthread.php?t=1519354)

You will need to create a new partition to migrate too - you can't just migrate it over Windows using the default migration. However, once you've migrated and are satisfied that it's successful you could choose to reuse the windows partition.
