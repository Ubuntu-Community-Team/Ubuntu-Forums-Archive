---

### Post by AndyCooll on 2008-10-27
It's greyed out because you cannot resize an active partition. So you're probably going to have to use the Ubuntu Live CD (which has GParted already installed) to do the partitioning.

You can probably resize your main partition and it will be fine, however I'd always make a backup first of any important data you may have ...just in case!
