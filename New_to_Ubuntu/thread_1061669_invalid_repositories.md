### Post by Kevbert on 2009-02-06
It looks like your software sources, where the errors are, may be wrong. If you open a terminal and enter
```
sudo gedit /etc/apt/sources.list
```
you can edit them from there.
Here's my Hardy software sources (I've just included the lines for the actual software sources and no others):
```
deb http://archive.ubuntu.com/ubuntu/ hardy main universe restricted multiverse
deb http://security.ubuntu.com/ubuntu/ hardy-security universe main multiverse restricted
deb http://archive.ubuntu.com/ubuntu/ hardy-updates universe main multiverse restricted
deb http://archive.ubuntu.com/ubuntu/ hardy-backports universe main multiverse restricted
```

### Post by Paqman on 2009-02-06
> **Kevbert said:**
> 
```
sudo gedit /etc/apt/sources.list
```


Please use gksudo for graphical apps. The correct command for gedit would be:

```
gksu gedit /etc/apt/sources.list
```
