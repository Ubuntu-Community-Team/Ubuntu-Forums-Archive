You can start another copy via the application lens (the one with the magnifying glass symbol. You search for the app and start it from there you will get a another instance. This is a real oversight in the design of unity. It's real annoying if you want to have multiple terminal windows open for example. You ought to be able to right click on the unity icon for the running app and select run  or something like that to start a second instance. Unity has some good ideas in it but it has landed half baked IMO.

Edit: Funnily enough for libre office if you right click on the icon and select new document, you do get another instance. Seems it has been partly implemented
