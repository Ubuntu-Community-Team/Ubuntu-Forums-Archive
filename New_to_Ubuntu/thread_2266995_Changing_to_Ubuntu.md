### Post by holysword on 2015-02-26
Hello there,

I decided to come and try Ubuntu now, but there are just so many things I'm missing that I don't think I'll possibly stay. I think the one that annoys me the most is the file browser: it is just the absolute most basic features I see there, but I might be missing things. I am personally used to Konqueror (nops, not Dolphin, Konqueror) which allows me to split the window (recursively) undefinitly both horizontally and vertically; might sound like silly to many, but if you have to be reading and managing and moving files from several different directories all the time, that's much more than a "cool" feature. But by far the most interesting feature of Konqueror (and KDE as a whole with its KIO libraries) is the fish protocol, in which I log in to a remote server using standard ssh, and Konqueror will fetch the files list, and show them to me as if it were local; but more than that, I can simply open the files normally, as if they were local, with a mere double click. KIO will fetch the file, store somewhere in a temporary folder and every time I press "CTRL+S" it will upload the modified file back to the server. Its just **that convenient** if you have to work remotely.

Is there any similar file browser that I am unaware of based on GTK or do move to Kubuntu?

### Post by Rob Sayer on 2015-02-26
I used to use Kubuntu and I don't think Konqueror is installed by default, but if you like it you can certainly install it.

You can install Konqueror (or any other KDE app) in a non KDE DE ... I've used Dolphin in several GTK based DEs and never had a problem running it.  It'll pull in a shedload of KDE dependencies of course but unless your machine has limited resources that shouldn't be any problem.

### Post by ajgreeny on 2015-02-26
There are many dual pane file managers available as well, eg, doublecmd-gtk,  doublecmd-qt, gnome-commander, krusader, mc (midnight-commander), pcmanfm, tuxcmd, and probably others I have missed.  I am not sure about splitting any of them horizontally but they are all either dual pane as default or can be used with two separate panes.
