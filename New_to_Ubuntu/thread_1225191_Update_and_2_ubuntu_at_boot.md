### Post by philinux on 2009-07-28
This is normal. Sometimes a kernel update can cause problems on some hardware. 

Nice to have a fallback.

You can install startupmanager from synaptic and limit how many kernels to display etc.

Or to install easy use firefox address bar.

apt://startupmanager
