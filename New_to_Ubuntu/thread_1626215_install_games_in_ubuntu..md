### Post by sanderd17 on 2010-11-20
If you want it more easy, there is a chance that PlayOnLinux has your game. PlayOnLinux is basically the wine code, completed with some things that make it easy.

When you start PlayOnLinux, you must choose one of the games/apps (ms office is also included), then PlayOnLinux asks you to insert the disk and leads you through a wizard like usually in windows. PlayOnLinux also applies the necessary configurations settings to make the game work in wine.

If your game isn't in the supported PlayOnLinux list, you have a chance it's installable via wine, but that chance is rather small. Of if it can be installed, it probably won't work like it should.
