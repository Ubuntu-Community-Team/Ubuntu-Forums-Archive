### Post by eschatologicalhumor on 2008-10-10
> **Solais said:**
> Simple, very simple. Installing the Zune Program on Ubuntu, lol. I own a Zune 2 80gb, want to install zune program here to manage and stuff, anyone know how? I heard it can't be done. :confused:

Yep, that's about the only solution. Install VMware or Virtualbox, install Windows in that, then the Zune software in Windows.

That's what I do and it works like a charm. I only boot into my Windows partition to play games...once a month.

And just as a reminder, this question has already been answered several times in the forums. There are a LOT of people crying out for Zune support in linux. It just isn't going to happen by MS's hand. The guys over at Rockbox are working on it, but there is firmware encryption on the Zune that just can't seem to be cracked, so for now it's a MS-labeled Toshiba Gigabrick...

Once WINE gets .NET3.0 and 3.5 support, someone will figure out the workaround for using WINE to run the Zune software, but until that time, we're pretty much stuck with using Windows, either native or virtual.
