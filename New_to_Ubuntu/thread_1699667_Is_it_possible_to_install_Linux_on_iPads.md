### Post by mikechant on 2011-03-04
This link does not appear to relate to actually installing Linux on the iPad, it just installs a remote desktop facility to access Linux on some other hardware. Lots of other links are similarly misleading e.g. "Run Windows 7 on your iPad" leads to another virtual desktop solution.

A lot of other link are of the variety "wouldn't it be cool to run linux on the iPad" or "has anyone done this yet?"

I haven't spent a great deal of time on this but I can't see any obvious links to "step by step how I installed Linux on my iPad", and there are quite a few people saying it's really difficult because of the custom undocumented hardware and the difficulties of getting the iPad to even attempt to load a different OS.

I'm 99.9% sure no-one has actually got Linux 'properly' installed and running on the iPad. Yet.

(And Screenshots alone do not count - they *will* be remote desktops, I'll bet)
