### Post by paul101 on 2008-06-24
in de terminal (Application >> Accessories >> Terminal) copy and paste ( control + shift + v to paste in the terminal;) )

```
gedit ~/.mozilla/firefox/whln812w.default/persdict.dat
```


then save the file

restart firefox
