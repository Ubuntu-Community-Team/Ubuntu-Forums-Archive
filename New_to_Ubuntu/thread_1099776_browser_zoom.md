### Post by mcduck on 2009-03-18
Just set the default font sizes in the browser settings to something that you can easily read (and don't forget the minimum font size!).

In Firefox, Edit/Preferences and on the Content-tab click the Advanced-button to access font settings.

For the browser being Beta, just run the normal updates and you should be running Firefox 3.0.7. Firefox 3 wasn't released at the time of Ubuntu 8.10's release so the included package is still the beta version that was available at the time. Ubuntu's repositories of course now have the latest stable release.

..and no, you shouldn't need any driver for your monitor. Computers, in general, don't care what monitor you have connected to them, or if you have any at all. There's not much of communication between the two, if any. The computer simply sends picture at the resolution&frequency it's set to do, without caring what happens after the signal is outside of your computer. ;) You might need drivers for your graphics card, though, but if everything seems to be working and you are able to use the resolution you want even that should be OK.
