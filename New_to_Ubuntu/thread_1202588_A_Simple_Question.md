### Post by WRDN on 2009-07-02
In the examples "player.cpp" and "stats.cpp", they are source files. cpp means C Plus Plus. In the C++/C language, you will find other common files like .h files. These are header files that usually contain the declarations of class', methods, and some data.

On a Linux system, you usually compile everything using GNU Make. The rules of compilation are placed in a Makefile. You then run "make" and it compiles the source/assets.
