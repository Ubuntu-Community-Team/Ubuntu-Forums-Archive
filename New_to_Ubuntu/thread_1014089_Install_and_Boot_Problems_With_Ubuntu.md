### Post by mrbiggbrain on 2008-12-17
Are you useing a flash drive and USB port supporting USB 2.0 instead of USB 1.1, on older computers this can be an issue becuse they dont have USB 2.0 ports and thus have verry limited USB transfer speeds. even on new computers, not all the ports are 2.0, they still ship some with 1.1 ports, and a few 2.0 ports.

try another USB port, if ur useing the front, try the back, if your useing the back, try the front, check if ur computer offers USB 2.0 and which ports, if its plugged into a USB 2.0 port, give us that information and we will continue
