### Post by Lateralis on 2011-05-21
The 1 gig recommendation is probably for Ubuntu.  Ubuntu uses the "Gnome" desktop environment which is pretty heavy on memory useage.  However, unlike Windows, you have a choice of desktop environments!  One of these is the [LxDE]("http://lxde.org/") which is used in [Lubuntu]("http://lubuntu.net/").  I'm not sure what the requirements are for that - I'm not sure anyone does yet! - but it will be considerably less.  So I would check that out first - get a copy of the CD, burn it to disc and try a Live session.  If it works OK then you might well be in luck!  If it doesn't, then you may well be out of luck, unless you can install more physical memory.
