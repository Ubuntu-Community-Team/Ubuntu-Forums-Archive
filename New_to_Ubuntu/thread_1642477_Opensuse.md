### Post by cariboo on 2010-12-10
Ubuntu has an automatic bug reporting system, you can submit a bug by pressing Alt-F2 and typing:

```
ubuntu-bug <packagename>
```

You also need to have an account on [https://launchpad.net]("https://launchpad.net"), in order to file bugs.

How quicky the bug gets fixed depends on how much information you give the devs, and how serious the bug is. Personally, I've had nothing but good experiences, in most cases I've been contacted by the developer involved with the particular package within a day or two, and we've managed to work through to the cause of the problem.

Aside from a few bugs that were sent upstream, that still aren't fixed, most of the bugs I submit are fixed fairly quickly.
