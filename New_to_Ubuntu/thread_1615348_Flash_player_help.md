### Post by themusicalduck on 2010-11-06
It should be in Software Centre under searches "flash" or "flashplugin" (you might need to click to reveal technical items at the bottom of the screen).

Otherwise, just search for Ubuntu Restricted Extras, install that and it will install Flash, Java and extra audio and video codecs.
