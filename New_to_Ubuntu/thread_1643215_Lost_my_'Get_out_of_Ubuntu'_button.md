### Post by qyot27 on 2010-12-16
It's no consolation if you're dead-set on having it always visible in the Panel, but if you *remove* that applet outright (right click, Remove from Panel), then you can access those functions from the System menu.  And this will ensure that you'll always be able to use it, rather than relying on the Panel having correctly loaded the applets each time.
