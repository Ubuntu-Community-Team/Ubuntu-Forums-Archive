### Post by oldsoundguy on 2008-07-12
project.  First, get a router and start with that.

the chain:
Ubuntu computer + Windows computer
plug both into router
plug router into modem
plug modem into cable.

But, since the set up software for most routers is written for WINDOWS, best to hook the Windows machine up first and install the router.  Then simply plug the Ubuntu machine in and it will be on line.

You can NOT have each on it's own modem without paying Comcast EXTRA.  It has to be a router or an old fashioned "head end" set up that very few use now days.

I have a Windows box and an Ubuntu box hard wired into a router/wireless router in my main media room.  I have another Windows box and an Ubuntu box in my guest room for my traveling friends .. both of them are wireless.  And I have an Ubuntu box in the Atrium for those occasions when I want all of that sunshine.  That one is wireless also.
I am on Comcast.

### Post by oldsoundguy on 2008-07-12
project .. you have missed the point entirely.

If you have NO router but your Windows machine hooks directly into your cable modem .. you NEED TO GET A ROUTER (the easiest way) in order to get both machines on line without unplugging one and plugging the other in.

And that makes absolutely NO difference if you go wireless with one of the computers as you STILL have to have a router if the only machine you have now is hooked to the modem by itself.  The Comcast modem does not have TWO inputs for computers and it will not work for a wireless.

Two or more computers on line in the same location using the same internet connection require that you have a router UNLESS you want to go through the hassle of plugging one computer into the other and ATTEMPTING to set the first one up as a head end machine.  That takes a lot more effort than buying a router and plugging it in and setting IT up!
