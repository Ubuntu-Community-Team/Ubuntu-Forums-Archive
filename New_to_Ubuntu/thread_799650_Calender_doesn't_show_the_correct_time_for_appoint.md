### Post by bmac on 2008-05-19
I found a couple of problems using Evolution for my calendar.
Check:
evolution/edit preferences/Calendar and Tasks
1. For some reason it changed my Time Zone from the install - Change if required
2. I live in an area that is adjusted for daylight savings time - Check box as required.
3. My work day begins at 8:00am not 9:00am. All my appoints tasks prior to 9:00am were moved to 9:00AM. Same with day ends....


Hope this helps....
