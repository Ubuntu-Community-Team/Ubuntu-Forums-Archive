### Post by NewRubberSoul on 2009-02-09
I went to 
System>Preferences>Sound
and set all of the defaults to ALSA.

Then I ran these two comands:

$ sudo killall pulseaudio
$ sudo alsa force-reload

The sound works fine and I'm okay with this type of fix, but I don't want to have to run these commands every time I start my computer.  Is there a cleaner way to do this?
