### Post by glln0v on 2013-03-30
> **Yudraciell said:**
> Everyday I check and update. I just find it weird some websites dont let me use its features because of an "incompatable browser"

It's actually more likely that the website is incompatible with a standard's based web-browser. I run into this occasionally. Either the people running the website are not following the standards or they choose to do things their own way. Either way the error is on their part and no fault of Firefox. 

See this article here: [http://arstechnica.com/security/2013/02/exploit-lets-websites-bombard-visitors-pcs-with-gigabytes-of-data/](http://arstechnica.com/security/2013/02/exploit-lets-websites-bombard-visitors-pcs-with-gigabytes-of-data/)

Firefox was the only browser immune to the attack because Firefox was the only browser that implemented the standard that was put forth. All the other browsers were not standards compliant, which caused them to be vulnerable to exploit with this attack.

I've tried all the other browsers. I always ended up coming back to firefox. I think it is pro-community to use Firefox also because of its standards compliance.
