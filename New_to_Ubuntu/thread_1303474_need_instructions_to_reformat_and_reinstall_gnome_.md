### Post by Mighty_Joe on 2009-10-28
[Ubuntu Installation Guide]("https://help.ubuntu.com/9.04/installation-guide/i386/index.html")
The default action for a Ubuntu install is to format the install partition, so if you choose the "guided" partition option, Ubuntu will wipe out everything.
Did you use the default partition option when you installed?
