### Post by smani on 2009-02-26
What you are refering to is the boot-loader, not the BIOS. In any case, no, you will not have any problems concerning the boot-loader, GRUB (the one used by ubuntu) will also detect existing operating systems, and list them in the list of operating systems (which is relevant if you want to set up a dual boot system with linux and windows in parallel).
