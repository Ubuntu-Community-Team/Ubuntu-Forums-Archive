### Post by sonicwind on 2016-02-09
I installed Ubuntu Restricted Extras from the terminal using apt-get install ubuntu-restricted-extras.

It looks like everything went OK but at what appeared to be the end a EULA for ttf-mscorefonts-installer (TrueType core fonts for the Web EULA) has popped up on the screen.

There's no direction as to whether the install is complete or what I'm supposed to do. It says <OK> at the bottom of the EULA but I enter and clicking on it doesn't do anything.

Then I noticed a red circle with a white bar up on the top bar by the time. It says "An error occured, please run Package Manager from the right-click menu or apt-get in a terminal to see what is wrong. 'Error: BrokenCount > 0'. This usually means that your installed packages have unmet dependencies."

Can anyone give me some direction? I don't know which right-click menu it's talking about (Software Center?). It appears I'm still in terminal but the EULA is taking up the whole page and I can't scroll up to where the installation was happening.

Thanks guys.

---

### Post by howefield on 2016-02-09
Use the tab key to highlight the OK button then press enter, left arrow key (or tab key) on the next screen to highlight the YES button and press enter again.
