Based on information I found in a thread for newcomers in this forum, I edited my /home/username/.bash_profile file and added the line

export PATH="$PATH:~/scripts"

to set the path to point to my scripts.

I thought that when I run Terminal, this file would be executed, and the path would be set, but it is not.  If I try to run scripts, after I start Terminal, I get the message file not found, and if I echo the $PATH, at that point, my scripts directory is not added.

"~/.bash_profile" is only read once per login.  So if you log off, and then back on, then your changes should take effect.

It's more convenient to make your changes to "~/.bashrc" - this one is reread every time you open a terminal window.
