### Post by blackbird34 on 2012-01-05
If the included search tool doesn't work well/ to your tastes, you could try Synapse (a search-based application launcher) or Recoll (a comprehensive desktop search tool, based on a user defined index, for finding files and even text from inside files).

Synapse description, installation and PPA: [http://www.webupd8.org/2010/11/syanpse-is-here-new-semantic-launcher.html](http://www.webupd8.org/2010/11/syanpse-is-here-new-semantic-launcher.html)

Recoll is in the Ubuntu Software Center, or you can get it via command line :

sudo apt-get install recoll

good luck!

### Post by mcduck on 2012-01-06
Dash menu is the one that opens when you click the Ubuntu icon in the laucher panel (the one on the left side of the desktop) or if you press the Super key )the one that usually has a Windows logo).

And the file manager's search is in the file manager's toolbar, on far right side. The button with a looking glass icon and text saying "Search". ;)
