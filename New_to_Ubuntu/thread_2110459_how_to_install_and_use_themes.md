### Post by Paradigm Arsenist on 2013-01-30
It depends on the theme. Some will have nice tarballs that you can install with GNOME's appearance configuring tool. Others will be availablein the Ubuntu repository. Others will have other installation methods, such as compiling from source -- often instructions are provided by the theme provider on the gnome-look page, on the download website or in a README file in the archive, these instructions often being too difficult and variable in results for a newuser.

Sometimes, there will be a big tarball with various themes that you can unpack and install with GNOME's appearance configuration tool -- look around the archive and see what's in it. Sometimes there will be archives within archives -- these can be themes in themselves.

Stick with what looks easy to start with. There are some nice themes in the Ubuntu repos -- these will be easiest. Next up will be easilyidentifiable archives that will work with GNOME's default tools.

I recommend that you don't bother with installing from sources or anything that looks complicated -- this will be frustrating for a new user. Be aware that GNOME-look is a bit disorganised and not everything will work in the same unified way and youshould be able to get something nice. :-)

I hope this helps, if not technically then practically.
