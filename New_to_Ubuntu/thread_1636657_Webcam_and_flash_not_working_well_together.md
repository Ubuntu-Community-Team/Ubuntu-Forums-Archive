### Post by Kenjitamura on 2010-12-03
So the problem isn't that my webcam doesn't work, it works.  However, when I use my webcam on a site like [http://testmycam.com/]("http://testmycam.com/") it only shows a small portion of the image Cheese shows.  Rather than resize the image it crops it, this is very frustrating for me if I wanted to use an online chat site like oohya.net.  I got it working last night to display the whole image in flash by playing around with webcam studio but I have no idea what it is I did that started to make it work.  In the adobe settings manager it won't allow me to choose my webcam if I wanted to use the loopback from web studio over the /dev/video0.

Any help is appreciated.  Btw the webcam is a creative, I'm not sure on the product number but it's listed under lsusb as 041e:4053 Creative Technology, Ltd Live! Cam Video IM.

Edit: I got it working again.  My process was open webcam studio, go to tinychat.com and sign in, start the webcam studio output, in tinychat select webcam and choose webcam studio (This shuts down the webcam), go back to webcam studio and exit out of it, go back to tinychat and this time select usb device as webcam.

Anyone have an idea what the problem could be?  I'd hate to have to do this everytime I use flash and a webcam.
