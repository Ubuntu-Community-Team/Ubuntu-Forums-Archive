### Post by lbarnes on 2011-01-12
Open System>Preferences>Appearance and drag the downloaded .tar.gz file over the Appearance window where your installed themes are and drop it in there to install the theme, then simply click on the theme to use it.  Some themes download as a .tar.gz files, others will need to be extracted to reveal the .tar.gz file.

### Post by Frogs Hair on 2011-01-12
If you see themes that are labeled GTK + Emerald they will need to be extracted before you can drag and drop them into Appearance Preferences.

The themes are not always labeled , so if they don't install , it's usually because they contain other packages that need to be extracted .

### Post by Frogs Hair on 2011-01-12
That means that the theme was already installed. To remove the leftovers from the first installation , open your home folder and press Ctrl + H to show hidden folders.

Scroll to the .themes folder and delete the folder from the first installation and install the theme.
