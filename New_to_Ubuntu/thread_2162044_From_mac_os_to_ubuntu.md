### Post by alan9800 on 2013-07-13
for accessing to OS X volumes and for managing them in Ubuntu you may install hfsplus and hfsprogs by running the following command into a Terminal window:```
sudo apt-get install -y hfsplus hfsprogs
```for all your needs about how to run Ubuntu on a Mac you may give also a look to the following subforum: [Apple Users]("http://ubuntuforums.org/forumdisplay.php?f=328")
