### Post by Anuovis on 2009-11-17
I know there are countless other threads about this topic but I have not found one that would be suitable for me.

I want to have my email checked regularly by a stand-alone app or Thunderbird itself, which would start with boot-up and permanently run in the background. It would be nice to have an icon appear in the notification area upon receiving new mail.

So far the closest thing to accomplish this seems to be the Mail Notification app. However, it can not recognize Thunderbird. I can set it to check pop, imap, gmail accounts or integrate with Evolution (which I do not use) but there is no option to select my current mail client. If I click* Autodetect* or *System mailbox* it does not find Thunderbird for some reason. 

Also, if I set Thunderbird to be the default client from its own menu, it appears as *Custom* with a command line beneath in *System>Preferred Applications*.
If I set it as default through *Preferred Applications*, Thunderbird tells my it is not a primary client when I run it.   

I can set Mail Notification to check my email directly via pop but then I have to manually check messages as read in it. In other words, even if I download new message with Thunderbird it still remains marked as unread in the Mail Notification. 

This would be so perfect if it could recognize Thunderbird and I have no idea how to to this. Any ideas would be much appreciated.

Also, if there are other simple ways how to accomplish automatic mail check-ups and notifications, please tell me. I have tried running Thunderbird with All tray and some extensions but all of these were not that good. 

Thank you.

### Post by XCan on 2009-11-17
I am using Thunderbird with the add-on Firetray [https://addons.mozilla.org/en-US/thunderbird/addon/4868](https://addons.mozilla.org/en-US/thunderbird/addon/4868) and it works excellent. When I receive an email, a short notification pops up, and the top panel will show the amount of unread emails currently. I haven't tried it with POP as I use IMAP for all my emails, but it should work for pop as well.
