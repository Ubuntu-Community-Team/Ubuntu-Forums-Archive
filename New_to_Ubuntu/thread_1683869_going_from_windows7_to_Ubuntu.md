### Post by Yougo on 2011-02-08
as johnny163 said, you can install the "restricted extras", it contains the codecs and such to play avi's and a lot more.
they are not includud by default because in some countries, you're not supposed to use that for free.

you can play WoW in Ubuntu with the help of wine. it is in the software center too, if it isn't already installed. 
read all about wine here:
[http://appdb.winehq.org/](http://appdb.winehq.org/)
they have the AppDB, a database on a huge amount of software that can run with wine, and how well it can. here's the info for WoW:
[http://appdb.winehq.org/objectManager.php?sClass=application&iId=1922](http://appdb.winehq.org/objectManager.php?sClass=application&iId=1922)

a program called Playonlinux (again Software Center, or [http://www.playonlinux.com/en/](http://www.playonlinux.com/en/)) can streamline a lot of setting up of programs with wine.

of the rest, google is your friend :)

happy ubuntuing ;)

The terminal is basically the *Linux* shell, which is properly called  ***bash***.    

There are many, many tutorials on ***bash*** available on the web.  Just do a web search and you will find more than you can possibly use.

There are also hard-copy books on ***bash*** that you can purchase if you want to go the text-book route.

Here is a link to the ***bash*** reference manual, which is authoritative but  does make for very dry reading if you're not coming from a Linux/Unix background.   I'd  suggest you try a tutorial or a book first and keep the manual for reference until you get more familiar with the shell.

[http://www.gnu.org/software/bash/manual/bashref.html](http://www.gnu.org/software/bash/manual/bashref.html)

Good luck and welcome to Ubuntu (and to Linux).  I think you will find it a very positive experience.
