### Post by windscape on 2013-01-28
Hi,

The Unity desktop environment in Ubuntu 12.10 requires a video card capable of modern hardware 3D OpenGL acceleration, which the Radeon 9000 is probably incapable of. I suggest trying the options on [this]("http://ubuntuforums.org/showthread.php?t=1743535") sticky thread related to the topic.

If those don't pan out, I suggest using a more lightweight version of Ubuntu, such as Xubuntu or Lubuntu.
