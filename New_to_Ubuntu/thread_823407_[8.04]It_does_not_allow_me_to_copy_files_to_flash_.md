### Post by hp3 on 2008-06-09
Not to be rude, but you have set the card in write mode?
(if the card is write protected, you can not write to it).

if you open a terminal, you can se what permissions linux mounted your 
flash card with:

hp3@pc-unx1:~$ ls -l /media
totalt 32
lrwxrwxrwx  1 root root     6 2007-10-27 19:06 cdrom -> cdrom0
drwxr-xr-x  2 root root  4096 2007-10-27 19:06 cdrom0
drwxr-xr-x  2 root root  4096 2007-10-27 19:06 cdrom1
drwxrwxrwx 10 hp3  hp3   4096 2008-06-02 22:22 disk
drwxrwxrwx 45 root root  4096 2008-06-02 22:22 disk-1
drwx------  7 hp3  root 16384 1970-01-01 01:00 KNUTSIPOD
hp3@pc-unx1:~$ 


abow you can see that disk and disk 1 is owned by my user, and my user (hp3) have write access to them (drwx).

Insert your flash card, and do ls -l /media and post the result.
