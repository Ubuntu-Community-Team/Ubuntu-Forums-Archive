### Post by Kareeser on 2009-04-13
I don't recommend you use the main server. For me, it's always slower.

You can have Ubuntu ping multiple servers and select the fastest one for you.

System -> Administration -> Software Sources -> Download from -> Choose "Other" -> "Select Best Server"
