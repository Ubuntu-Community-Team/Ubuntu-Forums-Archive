### Post by Dennis N on 2017-11-27
I think Lubuntu will need a two-key combination. Alt+F1 is defined to show main menu by default. If instead you want Super Key + another key, you can set that up in Menu > Preferences > Setup Hot Keys. Ubuntu MATE is the one that opens the main menu with Super Key only.
