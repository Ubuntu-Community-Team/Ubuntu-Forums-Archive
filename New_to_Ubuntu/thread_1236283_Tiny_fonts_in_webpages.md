### Post by mcduck on 2009-08-10
Another simple, and every effective, option is to set the minimum font size in Firefox's preferences. This will have the least effect on web page's layouts, while still making sure you'll never have problems with too small text on web pages.

To do that go to Firefox's preferences, Content-tab, and under Fonts & Colors click the Advanced-button. You'll find the minimum font size option under the default font settings.
