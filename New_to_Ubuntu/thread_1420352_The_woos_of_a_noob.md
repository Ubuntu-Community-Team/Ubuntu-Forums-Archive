### Post by NightwishFan on 2010-03-03
I just got done playing Starcraft. It works well, this page here should help: [http://appdb.winehq.org/appview.php?iVersionId=149](http://appdb.winehq.org/appview.php?iVersionId=149)

You need Wine to play Starcraft. Wine lets you run Windows programs at full speed. Some work some dont. You can install using this command:
```
sudo aptitude install wine
```

Just install Starcraft like on Windows.

Battle.net looks odd, but it works. You need to switch desktops sometimes to clear it. I do the cure for slowness on that page but you may not need it. I never needed any of the other tweaks.

As for virtual machines, there are plenty. I like Virtualbox.
[http://en.wikipedia.org/wiki/Virtualbox](http://en.wikipedia.org/wiki/Virtualbox)
