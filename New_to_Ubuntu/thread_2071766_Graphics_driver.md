### Post by grahammechanical on 2012-10-16
As a prerequisite to what?

Ubuntu comes with open source video drivers but we can have a proprietary video driver activated during the installation process by ticking the install third party software box or after installation by using Additional Drivers.

We have to have some kind of video driver that works with our video hardware otherwise we could not test Ubuntu using a Live DVD or install it from the Live DVD.
