### Post by mcduck on 2011-10-16
> **mitk0o0o0 said:**
> Alright, thanks for the replies guys. I fell in love when i saw the effect. Can't wait to install this.

EDIT: I have another question. I will burn this to a CD to do a clean install and i'm wondering if when i get to choose partition, if i choose the partition where my previous ubuntu version is installed, it will install it there again, right? I won't lose any Windows stuff or files?

The installer will give you three options, one is to replace your current Ubuntu install, one is to replace everything and the last one will allow you to define your partitions manually.

Only the "replace everything" would wipe your Windows install as well, both the replace Ubuntu and manual partitioning options allow you to leave Windows alone.
