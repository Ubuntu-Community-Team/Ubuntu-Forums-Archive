### Post by crispy_420 on 2011-06-26
I was think the mumble/murmur setup too but for that you need a dedicated server that I believe. Don't get me wrong, it works well and I have used it as a superior replacement for TeamSpeak. The other thing is that there seems to be more setup as well. Also the users would have to know how to connect.

I just did a quick test setup of ekiga on two separate VMs. As long as you have avahi server enabled and ekiga setup for auto start on login, you have a voip messenger that just works.

Granted I have little experience with the software so setup was simple. Also each client saw other active clients on the LAN and connect to each other with a double click.
