Hi everyone! Couldn't find any previous solved alternatives on this topic, which hadn't been outdated for years. Don't know even, if not in the right section.

Step 1, Connecting to CUPS:
    Open [http://localhost:631/](http://localhost:631/)

Step 2, Finding Printer:
    Select "Administration" -tab, from there under "Printers" -topic, select "Manage Printers"
    From the new page select "Stylus-DX4800"

Step 3, Selecting Printer:
    Open "Modify Printer" from the "Administration" -tab of the printer. Press "Continue" twice.

Step 4, Selecting driver:
    From "Select Another Make/Manufacturer" select Epson. From "Model" -tab select "Epson Stylus DX4850-CUPS+Gutenprint v5.2.8-pre1".
    Then click "Modify Printer"

Step 5, Testing the setup:
    From the "Maintenance" -tab select "Print Test Page"


Possible problems:
    You don't have CUPS or gutenprint installed, check prior retry that you have both. (Usually are there on many distros)
