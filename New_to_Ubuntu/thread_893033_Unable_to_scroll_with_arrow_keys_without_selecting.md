### Post by ViViDboarder on 2009-07-29
I was having the same issue and I just figured it out.  Maybe you did too by now, but in case anyone else stumbles upon this thread, go to Edit>Preferences.  Then click on the tab for Advanced.  Now UNcheck "Always use cursor keys to navigate within pages".

Problem solved!!

The cursor keys will still work to navigate up and down lines in text boxes, don't worry.  You'll just be able to scroll with them again too. :
