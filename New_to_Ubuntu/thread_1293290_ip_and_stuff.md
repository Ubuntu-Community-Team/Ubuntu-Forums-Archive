### Post by camper365 on 2009-10-16
When you're in Windows (or DOS) you're used to typing ```
ipconfig
``` into the command prompt, in Linux (usually universal, not just Ubuntu) you type ```
ifconfig
``` into the terminal (Applications -> Accessories -> Terminal)
note the change from p to f.
