### Post by Paqman on 2009-08-30
According to the [Wine App DB]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=1347"), no version of iTunes runs well under Wine. So you might want to think about using a Virtual Machine or a dual-boot if you really need iTunes.

What exactly do you need iTunes for? Except for syncing with some of the later Apple hardware, everything iTunes does can be done with native Linux apps.
