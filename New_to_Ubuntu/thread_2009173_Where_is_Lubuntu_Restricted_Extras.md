### Post by Fernhill Linux Project on 2012-06-24
You can easily install Lubuntu Restricted Extras from a terminal using 
```
sudo apt-get install lubuntu-restricted-extras
```

I would recommend using Synaptics Package Manager on Lubuntu as Lubuntu software centre is still a bit buggy.
