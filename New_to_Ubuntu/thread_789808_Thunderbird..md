In Ubuntu, we normally install packages/programs from large databases called repositories.  These have programs that are known to work in Ubuntu with your specific hardware architecture.  They tend to be pretty up to date, though not always bleeding edge.  You can read more about installing here: [http://www.psychocats.net/ubuntu/installingsoftware](http://www.psychocats.net/ubuntu/installingsoftware)

Synaptic Package Manager is the preferred GUI for package management.  System->Administration->Synaptic Package Manager.
