### Post by HarrisonNapper on 2009-12-14
VPN client/server software is essentially independent of your ISP. There are a lot of VPN programs, but the viability of each depends on whether you're trying to remote from a *nix machine to a Windows machine, from *nix to *nix, etc. What OS is on each computer? RealVNC is one pretty good program for that sort of stuff as well as Ubuntu's built-in remote desktop client.
