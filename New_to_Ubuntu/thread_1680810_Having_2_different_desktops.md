### Post by fabricator4 on 2011-02-03
The easiest way would be to have two logins, since they can each have their own desktop.  This would require having different home folders, obviously. You'd have to manage shared data by making a group that both logins had access to.

You could search the software repositories to see if there's a theme switcher that will work with Macbuntu, or otherwise emulate what you want to do.

### Post by ronnielsen1 on 2011-02-03
I did this once by having 3 different desktops, one regular, one that looked like xp and one that looked like win7 by using different user names. (Ex: /home/ron  /home/ronxp etc) The problem with using the exact same home is config files will mess you up. It doesn't really matter which desktop you log into, you still have access to all of your files and folders. It's a simple matter to put a link on your desktop to /home/otheruser/music for example
