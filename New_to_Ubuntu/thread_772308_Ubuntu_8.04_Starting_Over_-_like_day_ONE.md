### Post by MattBD on 2008-05-03
I personally never do an upgrade - it can cause problems. It's probably better to do a fresh install each time.
If you do this, it's worth taking the time to set up your /home directory on a separate partition - check out [this link]("http://easierbuntu.blogspot.com/2008/03/setting-up-your-home-directory-on.html") for a guide to setting this up.

### Post by Paqman on 2008-05-03
On the flipside: I updated all the way from Dapper to (I think) Gutsy without any issues at all. Never had anything break. A lot depends on how much you've messed about with your system. For example, compiling software from source generally breaks the dependency system, and means you might have trouble when you upgrade.

fcorourke: There's a few tricks that make reinstalling a lot easier. My favourite is [howto create a list of installed packages](http://ubuntuforums.org/showthread.php?t=261366) that you can automatically reinstall. Between that and a separate /home partition you can thrash out a reinstall in an hour or two, tops.
