### Post by James_Lochhead on 2009-04-16
NBR has the following additional packages:
[LIST=1]
[*]go-home-applet
[*]window-picker-applet
[*]ume-launcher
[*]maximus
[*]human-netbook-theme
[*]ume-config-netbook (only use this if you have an Atom processor)
[/LIST]

Just install these programs normally then add them all (except the applets) to System > Preferences > Sessions.
