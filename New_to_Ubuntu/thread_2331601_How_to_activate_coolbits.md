### Post by QDR06VV9 on 2016-07-23
It works just fine... but I see no xorg.conf file there.
You will need to generate one with
```
gksudo /usr/bin/nvidia-settings

```  
Now Nvidia settings will pop up....Now look for on the left (X Server Display Configuration) click that..Now to the right you will see in the bottom part right a button that will read (Save X Configuration File)
Let it do it's thing and when done Close Nvidia-Settings....Now you follow that guide and modify xorg.conf as root....below is how mine looks...**Do not** copy and paste this to yours just for comparison only.:)
