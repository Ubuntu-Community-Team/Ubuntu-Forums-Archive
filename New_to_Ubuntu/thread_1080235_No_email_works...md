### Post by koperry on 2009-02-25
pop.gmail.com use ssl

smtp.gmail.com use tls

user name needs to be "name@gmail.com" for pop and smtp, if you use google apps then it will be "name@yourdomain.com"

My ISP blocks the standard smtp port so I set up smtp.gmail.com:587, works great.

I hope this helps
