### Post by Thingymebob on 2008-12-06
To stop the update open synaptic, type netatalk in the box, select the package, then from the package menu choose lock version.

For places, navigate to the folder in nautilus then add a bookmark, should now appear in your places.

First question:
Go to System>Administration>Synaptic Package Manager, Select the package you want, then in the Package menu choose 'Lock version'.

Second question:
Though Dj Melik has solved it, another way to do it is browse to the folder you want to bookmark (add to places menu) and press  ctrl+d. To edit bookmarks press ctrl+b when in a nautilus (the gnome file browser) window.
