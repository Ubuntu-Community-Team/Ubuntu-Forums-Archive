### Post by grahammechanical on 2011-10-03
I am using 11.10 and I have got both Unity 3D and 2D installed. I can confirm that in 3D the Compiz Config Settings Manager and the Unity plugin Experimental tab will let you make the top panel completely transparent and the Launcher transparent except for the right hand edge which shows as a vertical line. But in Unity 2D even though Compiz Settings will load and the Unity plugin will show different settings which can be adjusted, they will have no effect. The icons do not change size, the top panel does not become transparent and neither does the Launcher. I have just tested this.

Still, all this is still experimental. I am assuming that Canonical want the same or very similar user experiences in both 3D or 2D Ubuntu. It may come in time.
