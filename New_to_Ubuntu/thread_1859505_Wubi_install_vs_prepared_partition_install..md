### Post by Frogs Hair on 2011-10-14
Wubi has a 30 Gb max size and it can not hibernate . Wubi is also installed in a folder inside the Windows partition  so if the Windows boot loader doesn't work for some reason neither will Ubuntu . As written performance may degrade over time . A  Wubi installation will work the same as a traditional dual boot except for the limitations  above .      [https://wiki.ubuntu.com/WubiGuide](https://wiki.ubuntu.com/WubiGuide)
