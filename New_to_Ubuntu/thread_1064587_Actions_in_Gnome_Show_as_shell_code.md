### Post by yeats on 2009-02-09
Interesting idea. . .  I don't know of an application that does this, but I agree that it would facilitate learning the shell.  You could try some keyword searches in Synaptic.

In the meantime, I would recommend getting a basic UNIX or Linux book to get some of the basics in hand.  Many of the GUI applications do not perform shell commands themselves once they are started, so your idea wouldn't cover, say, playing an MP3 for instance.  If you are looking for a debugging function, you might want to learn how to use the logs in /var/log.  Here's a link to the Ubuntu wiki page on the subject:
[URL="https://help.ubuntu.com/community/LinuxLogFiles"]
https://help.ubuntu.com/community/LinuxLogFiles[/URL]
