### Post by zeroseven0183 on 2010-03-01
I assume you still have the panel on the bottom of your screen. Right?

Please open System Monitor from System > Administration then open any window look at the System Monitor to see the application. (For example, Applications > Internet > Firefox Web Browser). Minimize the Firefox window and observe the System Monitor if firefox-bin is still there. If it is, then windows are only minimized and you lost your Windows List in the panel. If it's not, it's another problem.

Now, see if you can add Window List to the panel. Right-click on the panel then select Add to Panel, scroll down and select Window List then click Add. See if the applications you opened are there.
