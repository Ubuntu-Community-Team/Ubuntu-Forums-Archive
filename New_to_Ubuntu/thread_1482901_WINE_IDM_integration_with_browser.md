### Post by MrNatewood on 2010-05-14
You probably won't be able to integrate native firefox with any wine application. I'd suggest trying some linux-native download managers, like MultiGet or Gwget.

If you just can't do without that specific app, perhaps try running firefox in wine with the plugin you mentionted, that should work. Though kind of spoils the point of using linux in the first place, doesn't it? ;)

If you just need something to download video from streaming media sites(youtube, pr0n, etc.) there are plenty of firefox extensions that would to the trick without an external program. Try DownloadHelper.
