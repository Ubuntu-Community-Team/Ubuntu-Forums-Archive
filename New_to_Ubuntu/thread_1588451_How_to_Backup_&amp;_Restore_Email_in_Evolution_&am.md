### Post by cariboo on 2010-10-05
I'm not sure about evolution, as I haven't run it in years, but all you have to do to backup your Thunderbird email store, is backup the ~/.thunderbird directory if you are running version 3+ and ~/.mozilla-thunderbird if you are running a version before v. 3.

### Post by jtarin on 2010-10-05
Evolution....File>Backup

If you go to the home folder, click on view, show hidden files you will see an Evolution folder. I just copy it to a removable storage device.

### Post by lisati on 2010-10-05
Evolution has options on its "File" menu for "Backup settings" and "Restore Settings".
