### Post by drs305 on 2010-05-25
The information is in the link provided in the previous post, but a more condensed post lists some of the most common tasks and may be a bit easier to absorb:[Grub 2 - 5 Common Tasks]("http://ubuntuforums.org/showthread.php?t=1302743")

An easy-to-use GUI app, Startup-Manager, can also set the default OS. Install "startupmanager" and access it via System, Administration, Startup-Manager.

The above solutions change which OS is the default boot. For changing the order you see on the screen, the easiest way is to rename /etc/grub.d/30_os-prober to 09_os-prober, which will put other OS's before the default Ubuntu installation. If you have more than one partition with Ubuntu installs, the extra Ubuntu installs would still show first, then Windows, then your primary Ubuntu install.

Yet another option is creating a Windows entry in a custom file called /etc/grub.d/06_custom. Here is a link on how to create a custom entry:
[http://sourceforge.net/apps/mediawiki/bootinfoscript/index.php?title=Boot_Problems:Custom_Menu]("http://sourceforge.net/apps/mediawiki/bootinfoscript/index.php?title=Boot_Problems:Custom_Menu")

You can see which version you are running with:
```
grub-install -v
```
or checking the top of the Grub menu when it boots. Grub 0.97 is grub legacy. Grub 1.97 or later is Grub 2.
