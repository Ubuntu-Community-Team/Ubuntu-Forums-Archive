### Post by mick8985 on 2009-05-06
Personally I'd just do a straight up ubuntu install. Chances are you'll want the box to do much more than just run MythTV, in fact you may choose to run XBMC instead. ### Post by fenian on 2009-05-06
You can install the Ubuntu Desktop (Gnome and Ubuntu,open office,gimp,etc...) from the Mythbuntu Control Center in the System Roles Section.Then when you start up, at the log in screen choose sessions then choose Gnome it will log you in to a standard Ubuntu install with all the Mythtv packages.If you are using the computer only for Myth you don't really need all the extras of Gnome and Ubuntu Desktop,in fact some may say you are better off without them as it frees up more resources for Myth.
