### Post by grahammechanical on 2011-10-16
I always create a Live CD (bootable image) whether I intend to fresh install or upgrade over.

You might need to fresh install but do not mark the partition as needing to be formatted. That should leave your /home folder alone. According to some recent posts on this forum. Which you might want to check to see if I am right. It seems that in this way the OS files will be over-written but not the files in /home.
