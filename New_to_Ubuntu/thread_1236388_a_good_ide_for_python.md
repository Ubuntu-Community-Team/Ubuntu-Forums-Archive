### Post by kpkeerthi on 2009-08-10
**_Completely Free:_** [Netbeans + Python]("http://www.netbeans.org/features/python/"). Its still in Early Access (consider it alpha) but has got high potential. I prefer this (although I'm using only its debugger for the time being).

**_Semi Free:_** [Pydev]("http://pydev.sourceforge.net/"). Its a plugin for Eclipse. Although the Java/J2EE Development Tools available for Eclipse are extremely powerful, PyDev is a pity and is no where complete or close to the Java counterparts. I find it buggy and partial done. There is also [PyDev Extensions]("http://fabioz.com/pydev/") that cost you $$$ but Netbeans has everything the paid version of PyDev has. I still don't recommend PyDev or PyDev Extensions. Being a paid software this ought to be much more complete and bug free.

**_Commercial:_** [Wingware]("http://www.wingware.com/"). This one appears to be the best I have tried so far. Its pricey though. Definitely not for me. You may like if you can shell out $180.
