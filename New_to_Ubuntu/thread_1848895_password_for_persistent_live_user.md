### Post by ajgreeny on 2011-09-23
When using a live CD, if you logout and then login again, the username is **ubuntu** for the gnome ubuntu, (and I think also for both xubuntu and kubuntu, but it may be **xubuntu** and **kubuntu** for them).  The password is left totally blank, you just use "Enter" and it logs in successfully.  Perhaps that will also work within VB.
