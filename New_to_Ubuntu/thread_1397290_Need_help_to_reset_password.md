### Post by Puck7 on 2010-02-03
I'm sorry for the confusion:

root password is the sudo password. When you're being asked for the sudo password in terminal, try your username's password, if it is a fresh install of Ubuntu, that is your one and only password.

If you are using a computer, with multiple users, so more people have logins to it, then the sudo/root password can be different.

### Post by Grenage on 2010-02-03
Sudo runs a program with root privileges, on behalf of a user with sudo access.

The default user has sudo access.

Therefore the sudo password will be your normal user password.
