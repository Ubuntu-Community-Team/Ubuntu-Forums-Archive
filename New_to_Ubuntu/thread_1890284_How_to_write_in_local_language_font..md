### Post by grahammechanical on 2011-12-03
You may also need to set a keyboard layout. You find the utility in System Settings>Keyboard Layout. Just click the the Add symbol ( + ) and you will find Oriya in this list. I have just checked this.

Afterwards you will get a keyboard icon in the top panel that you can use to switch between keyboard layouts. I use this to switch between English and Greek. I have also tried this with other languages. You can set this up to work through a keyboard short-cut.

If this does not work you may need to select another keyboard input method such as ibus. You can do this under Language Support>Keyboard Input Method System. I recently used ibus to get Korean characters.

I am trying to find out if you will also need to install a special ibus-oriya package. To get Korean working I had to install ibus-hangul. Instead I found this video:

[http://www.youtube.com/watch?v=LL7icGNhIfI]("http://www.youtube.com/watch?v=LL7icGNhIfI")

From this link I think that you may have to install the ibus-m17n package.

[https://launchpad.net/ubuntu/+source/ibus-m17n]("https://launchpad.net/ubuntu/+source/ibus-m17n")

I have not tested any of this for the language that you want. You are now an experimenter. Tell us if this works, please.

Regards.
