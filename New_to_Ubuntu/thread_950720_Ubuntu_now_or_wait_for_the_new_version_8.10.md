The current version of Ubuntu, 8.04 Hardy Heron, is fantastic! You can install it today. :) You can also try 8.10 Intrepid Ibex right now, if you can tolerate a small risk of bugs/instabilities.

Upgrading to Intrepid Ibex from Hardy is really easy. However, some people are choosing not to upgrade for any number of reasons. Hardy is a "long term support" release, which means it will receive support and upgrades through 2011. 

I would say, if there are specific features you need that are part of Ibex but not Hardy, then by all means start with Ibex today. If stability is a bigger concern, start out in Hardy, and you'll always have the option to upgrade at a later date.

### Post by empty_spaces on 2008-10-17
8.04 has been pretty solid for me after they ironed out the initial bugs.
If you wait a couple of weeks and install 8.10 when it comes out, you will have to deal with the initial bugs. 8.10 should probably get stabilized a month down the line.

If you are new to Ubuntu, I would advise to go with the stable 8.04. It is an LTS version, so it should be supported for 3 more years. But remember that upgrading from a previous version of Ubuntu is an iffy process which may or may not work. Make sure you back up your data if you try it.
