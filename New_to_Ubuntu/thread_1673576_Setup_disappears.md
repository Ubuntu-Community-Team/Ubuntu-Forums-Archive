### Post by Paqman on 2011-01-22
If it's a really old machine, you might not have enough RAM to run both the live session and the installer. However, the fact that it ran the live session at all means you should be fine once you've got it installed. To get around the problem of not being able to run the installer you could download the [Alternate CD]("http://www.ubuntu.com/desktop/get-ubuntu/alternative-download") and install from that. It uses a text-based installer that is a lot lighter.
