### Post by 3rdalbum on 2009-12-14
Your desktop should definitely be faster.

Is there a program that's using the CPU in the background? Try running the "top" command and see if something gets reported as using a lot of CPU time.

Also, try the "iotop" command (will need to be installed).

Kill whatever is burning your CPU.

Also, I think all quad-cores are 64-bit capable; next time you install Ubuntu on your desktop machine you should install the 64-bit version so you can get access to faster 64-bit calculations and more memory addressing.
