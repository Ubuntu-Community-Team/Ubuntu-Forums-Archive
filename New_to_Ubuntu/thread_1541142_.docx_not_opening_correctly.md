### Post by hamstermoon on 2010-07-28
If you have a Windows Live account (or even you just set this up just to use the facilities) there is an online, cutdown version of Microsoft Office you can use even on Ubuntu.

I actually found out about it through the OMG!Ubuntu blog. Here is the link:

[http://www.omgubuntu.co.uk/2010/06/microsoft-have-finally-launched-their.html](http://www.omgubuntu.co.uk/2010/06/microsoft-have-finally-launched-their.html)

With these facilities you should be able to open Word, Powerpoint and Access files with ease. I have used it and it works. It also gives a nice 25 gig of storage space too!

Makes Wine a little reduntant if all you have it on your Ubuntu for is to do is run MS Office (as I do!) ...
