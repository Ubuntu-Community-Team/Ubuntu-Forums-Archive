### Post by Nickjpost on 2010-11-08
When you install using the live disk, It will give you some options: Use entire disk or partition along with current os. At the bottom, there is an "advanced" link that you can select and create partitions to your own partitions. After install, if you've chosen to use the option to run Ubuntu alongside your current operating system, when you boot your PC, there will be a list of OS to choose from.
