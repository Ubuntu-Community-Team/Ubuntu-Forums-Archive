### Post by deadflowr on 2017-04-07
Turn on the Workspaces.
(Open System Settings >> Appearance >> Behavior >> enable Workspaces)
Now you can move between different workspaces, and place program windows in whichever one you want.

Hint: Ubuntu has a layout overview of important keybindings you can use to navigate around workspaces.
To view the overview press and hold the super key on the keyboard.

Mouse dnd(drag and drop) on Ubuntu is rather clumsy, as there is only a basic icon for the workspace switcher.
(desktop environment like gnoem-flashback, mate, kde, xfce, and others allow you to drag windows around in the workspace switcher panel app)
But in Ubuntu to dnd you need first click on the icon to open the expose, which opens an overview of all workspaces, then you drag the window from one workspace to another.
(Like I said, clumsy)

Hope it makes sense
