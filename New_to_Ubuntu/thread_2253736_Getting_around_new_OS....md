### Post by agillator on 2014-11-22
I don't know about smartphones. Someone else will have to help you there. But . . . .

Ubuntu  is not an OS, it is a distribution. Linux is the operating system.  Ubuntu packages that OS with other software to make it useful. You will  find quite a number of distributions available, each a little different. A  user must decide which distribution is most useful to him/her. One of  the advantages of using a distribution over doing all this yourself is  the distribution maintainers make it easy to keep your software (and OS)  current.

One of the things you will find is that Linux gives you  freedom to do things as you want to do them, unlike Windows which  requires you to do things the way MS and software companies want you to  do them. This means that, although you can do things blindly and follow  the herd, Linux probably will make you be more involved and make you  make more choices. 

You will also find that open source software  means a couple of things. First, you can probably trust your OS and  software more since people can and do look at the guts of a piece of  software and if there is a problem or some malware the world finds out  about it quickly which also means it probably gets fixed quickly. It  also means that, if you have the skills, time and desire, you can change  any piece of software to work the way YOU want it to work by  reprogramming it. Of course you still have to watch out for copyright  restrictions but that is what the GPL is all about. This freedom brings  with it a greater responsibility in making your own decisions, of  course, which to put it bluntly is one of the reasons some prefer  Windows over Linux in my opinion.

There is a major difference in  the way of thinking between the Linux and Windows worlds. Windows  generally takes the approach of a program is supposed to do everything.  On the other hand the Linux world generally takes the approach that a  program does one thing and does it very well. It is the user's  responsibility to tie programs together to get things done the way the  USER wants/needs things done.

With that as background, you might  pick up a Linux for Dummies book as a start. The 'For Dummies' series is  a pretty good introduction to any subject. You might also google 'Linux  Tutorials' and look at some of the myriad of tutorials that are out  there. Also ask specific questions here and in other fora. The more  specific the question the more useful the answers normally. Very general  questions like yours are often very difficult to answer although people  will try.

I hope this helps you get started toward finding answers to you questions. Welcome to the world of Linux and Ubuntu.

### Post by QIII on 2014-11-22
Here is what ***I ***would do ...

Install VirtualBox in Windows.  Create a virtual machine.  In that virtual machine, install Ubuntu.

Keep snapshots as you go along, so you can get back to a working Ubuntu system if you break things -- which you *should* because that is the best way to learn.  And you don't risk ruining Windows if you use a virtual machine.

Try things out.  Experiment.  If you get tired of it or get frustrated by it, just shut down the virtual machine and keep using Windows.

I know that many would disagree with me, but I don't think there is any good reason to permanently and completely switch to Linux unless you just can't afford Windows or just decide you would rather do so. If you decide you want to keep up with Linux, dual boot for a while.  That way you can have a natively installed Windows machine AND a natively installed Linux machine.  Get rid of Windows later if you want.  But don't just switch right off the bat.

### Post by grahammechanical on 2014-11-22
> [COLOR=#000000]Is it available for the latest smartphone OS.[/COLOR]

No. Ubuntu is not available for the latest smartphone hardware. We can install Ubuntu on laptops and desktop machines but do not expect to install Ubuntu on phones. Mobile devices use different hardware to laptops and desktop machines and there is a great variety in that hardware that it is not possible to make a version of Ubuntu to be installed on smartphones the way we install Ubuntu on desktops.

For more than a year Ubuntu developers have been working on providing a version of Ubuntu that can be used by manufacturers to install on their phone and tablet devices. Two manufacturers have agreed to release for sale Ubuntu smartphones. When this will happen we do not know. So, do not ask. This is just a user forum. What do we know about these things?

Do not expect a Linux distribution to be a very cheap version of Microsoft Windows. The developers of Linux and the software applications that run on LInux have their own ideas about how a computer operating system should work and should look. We are told that Linux works well with older hardware. And that is true but there is a limit. Some hardware is too old. The specification is too low, even for Linux.

Do not expect to use Windows applications on Linux. Are there any applications that you use in Windows that you cannot do without? Linux has very good alternatives to many Windows applications. But if there is any Windows application that you must use and nothing can replace it for you, then you should stay with Windows. Especially if it has come pre-installed on a machine that you have purchased. You have paid for it, after all.

The best thing for you to do, in my opinion, is to download what we call an ISO image of Ubuntu and burn it to a DVD disk or a USB memory stick and run a Ubuntu live session and see for yourself. The live session does not install anything to your hard disk. It does not change anything on your machine. Ubuntu runs slower in a live session than when it is installed on to the hard drive but the live session does give us the opportunity to see what Ubuntu is like.

[http://www.ubuntu.com/download/desktop/burn-a-dvd-on-windows](http://www.ubuntu.com/download/desktop/burn-a-dvd-on-windows)

[http://www.ubuntu.com/download/desktop/create-a-usb-stick-on-windows](http://www.ubuntu.com/download/desktop/create-a-usb-stick-on-windows)

[http://www.ubuntu.com/download/desktop/create-a-usb-stick-on-windows](http://www.ubuntu.com/download/desktop/create-a-usb-stick-on-windows)

Each version of Ubuntu is provided with security fixes for a certain period of time. Every 2 years a version of Ubuntu is released that is supported for 5 years. During those two years a version of Ubuntu is released every 6 months and those versions are only supported for 9 months.

So, my advice to you is to try Ubuntu 14.04.1 LTS (Long Term Support) and not Ubuntu 14.10. Choose the version that is supported for 5 years starting From April 2014.
