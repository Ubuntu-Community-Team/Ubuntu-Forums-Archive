### Post by chdrappi on 2012-03-18
I am having trouble setting up the Apple Magic Mouse with Ubuntu 11.10 using Bluetooth. I recently set up the Magic Trackpad with no problems at all. But when I tried to pair the mouse, it recognized the device and told me I set it up successfully, but then said it wasn't paired, and the mouse didn't work.

This is what I did:
1) Opened up Bluetooth under User Settings
2) Turned on Magic Mouse
3) Selected the device I want to set up ("Apple Magic Mouse" showed up)
4) Selected PIN settings, and chose "0000" (After this process didn't work, I did the whole process over, trying the automatic PIN and the custom pin, entering "0000")
5) It told me to wait while it finished set up
6) It told me it "successfully set up new device "Apple Magic Mouse"
7) The mouse lists as one of my Bluetooth devices, but shows as "disconnected" and not paired.

What can I do? Before buying the mouse and before posting this I had done thorough searching and it seems no one really has problems with this mouse on Ubuntu 11.10 except me. Thoughts?

---

### Post by chdrappi on 2012-03-18
Resolved. I am so dumb.

The problem was that I didn't enter "0000" using the keyboard when they asked me to enter the PIN. Wow.
