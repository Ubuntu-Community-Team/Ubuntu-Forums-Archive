The second link helped a lot though.

Added these two things to the xorg file:

```
Section "Monitor"
        Identifier   "Monitor1"
        HorizSync    28.0 - 80.0
        VertRefresh  48.0 - 75.0
EndSection
```
```
Section "Device"
   ...
   Option "IgnoreEDID" "true"
EndSection
```After a restart the login prompt looked perfect, but the desktop went back to 1024 x 768. Used the Nvidia Xserver settings to switch back to 1440 x 900 and rebooted but the same thing happened. Every time I logged in the screen resolution would switch back to 1024 x 768. This time I used the Monitor Preferences option to change the resolution. Showed up as 1440 x 900 @ 50 Hz. Things seemed good. Went back to Nvidia Xserver and saved the configuration this time.
