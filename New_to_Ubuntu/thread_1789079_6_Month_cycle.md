NO ONE/QUESTION IS FOOL :)

Yes, every 6 months, a new release will come up. 
Best way to keep an eye on things is here: [http://distrowatch.com/](http://distrowatch.com/)

You're free to upgrade or to keep the current version/release.

There are two types of releases, one of these two is LTS (Long Time Support) which will be supported for 3 years instead of two.

Ubuntu 10.04 for example is supported for 3 years.
Ubuntu 11.04 is supported for 2 years.

Some do an upgrade and some prefer fresh installation. Majority go for Fresh Installation to avoid conflicts and problems.

This is a quick and simple answer :)

### Post by Beacon11 on 2011-06-23
Just a quick addition to the previous answer. It's not an automatic upgrade-- a new button will pop up in Update Manager saying there's a new distribution if you'd like to upgrade. You don't need to click it.

By default, for regular releases this button will show up every time a new release has come out. However, for LTS releases, this button will only show up when the next LTS release has come out.

### Post by nzjethro on 2011-06-23
> I appologise if i sound like a fool. Im a total ubuntu noob

The only foolish question is one that should be asked but isn't. :)

While there are new releases every 6 months, it's not the case that you must upgrade every 6 months. Check out [this.]("http://en.wikipedia.org/wiki/List_of_Ubuntu_releases#Version_timeline") Typically even non-LTS versions are supported for 18 months or so. I have 10.10 installed, and that's supported through until April 2012. If you find a version you like, you can stick with it until it is no longer supported (of course you can even keep using it after then but this is ***NOT RECOMMENDED***, as you don't get important security updates etc.). Basically, find a version you like, and it's likely that by the time it goes OEL, you will have found a new release that you like just as much.
