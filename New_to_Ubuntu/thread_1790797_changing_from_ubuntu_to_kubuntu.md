### Post by josephmills on 2011-06-25
Step one 
Please open up a terminal and type in 
```
sudo add-apt-repository ppa:kubuntu-ppa/backports
```
thenn update and install kde 
```
sudo apt-get update && sudo apt-get install kubuntu-desktop
``` 

this will take an hour or so when you will be givin a blue screen just hi enter a couple of times (or set it up the way you like) 
[img]http://img94.imageshack.us/img94/2585/kdel.png[/img]
next after install is done log off then log back on but switch the session to kde plasma and sign in there you go kde
