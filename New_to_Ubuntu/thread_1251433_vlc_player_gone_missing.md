### Post by jbrown96 on 2009-08-27
There should be a way to edit the menu. System-->Preferences-->Menu Edit. You will want to create a new item. The program is located at /usr/bin/vlc. I think the icon is located some in or near /usr/shre/icons/ 

Another really easy way to launch it is using Alt+F2. This will bring up an application launcher. Just type vlc and press enter.
