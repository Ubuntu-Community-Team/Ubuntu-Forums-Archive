### Post by Tamlynmac on 2010-01-07
Should you use Shoutcast to save the music site - to play on your own player (Rhythmbox), you will need to select Rhythmbox as the default player in Firefox.

If you just to choose to listen through Shoutcast and not download the music site to Rhythmbox this action will be unnecessary.

If Rhythmbox opens and asks to install codecs, make sure you say yes and permit the mp3 codec installation.

You can save the site directly in Rhythmbox and play them without opening your web browser. One thing you might consider is renaming the sites as they appear in Ryhthmbox (right click on site, select properties) and change the name. If not you radio section may get pretty convoluted. You'll see when you start downloading the sites and the long names become a pain. I keep my sectioned out by type for example "Old Talk Radio #1 - PS Radio, Old Talk Radio #2 - AM 600, etc."

As you see I'm a big fan of Old Talk Radio. Shows like the "Six Shooter, Johnny Dollar, The Whistler, X-one" still manage to capature my attention. ;)

Good Luck

### Post by Tamlynmac on 2010-01-07
Most all application executables are located here:

File System/usr/bin
or
File System/usr/sbin

When given the option of which application to use and it doesn't appear on the list, select "other" and go to one of the locations above (usually "bin") - find the executable for the application you wish to use and double click on it.

You can always open your file browser "Nautilus" and locate these executables.

Good Luck - Hope this helps.

### Post by Tamlynmac on 2010-01-08
If you open:
"Places/Home Folder" in your Panel menu, on the left hand side window under "Places" do you see the words "Flie System"? If so, left click on it one time and on the right hand side window - a listing of folders should appear. (Your probably in the icon view default setup. I prefer to use the list view while others may prefer icons. To change from icon view to list view select "List" under the view menu in your file browser).

In the icon view - find the folder (in the right side window) labeled "USR". Double click on "USR" and it should open and show all the subdirectories under "USR" including "Bin" and "SBin". Double click on "Bin" and it will display all the files listed in the "Bin" folder.

In the list view - find the folder (in the right side window) labeled "USR" and left click one time on the arrow just to the left of "USR" and it will display the subdirectories listed under "USR". Then left click one time on the arrow next to the "Bin" folder and it will list all the files under the "Bin" folder.

Hope this helps, I've not installed 9.10 yet (using 9.04) and am not in a hurry to do so. Your running 9.10 and the above instructions may not apply to Nautilus in 9.10. If the above procedure doesn't work, perhaps it's best you start another question and define that your using 9.10 - ask for assistance regarding how to use the Nautilus file browser. It may be slightly different in 9.10, I just don't know.

Were you able to get Rhythmbox working and synced to the online radio stations? Hope so, if not please respond and let me know where your having problems.

Good Luck.
