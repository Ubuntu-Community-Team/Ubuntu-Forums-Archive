### Post by stinkeye on 2009-11-08
GTK 2.x themes from gnome look should just be drag and drop in appearances.
But what I've found is that in the compressed file you download there will
 sometimes be an emerald theme as well.
So if it wont install you have to extract first to get to the gtk theme.
Themes are placed in ~/.themes
If you want to install emerald themes install emerald in synaptic. 
Compiz and metacity are window managers.
Gtk and emerald are window decorators.
To easily switch between different window managers or decorators install
fusion-icon through synaptic.

### Post by 3rdalbum on 2009-11-08
On the site where you downloaded the theme, it should say if it requires a particular theme engine. If it needs one that you don't have (for instance, Nodoka or Murrine) you just go to the Synaptic Package Manager and install the theme engine, then the theme.

Also note that the repositories have themes too.
