### Post by Sarah L on 2008-05-24
Just about any of the more popular music players has the ability to edit the Title, Year, Artist, and Album tags on an MP3 file.

One of the best players out there, Amarok, is native to Linux and has an incredibly easy method of editing the tags - simply right click on the MP3 file and select Edit.

If you don't have it already, install it by typing ```
sudo apt-get install amarok
```
