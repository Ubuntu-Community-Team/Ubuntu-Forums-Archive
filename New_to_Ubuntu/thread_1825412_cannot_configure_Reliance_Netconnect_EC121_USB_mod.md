### Post by varunendra on 2011-08-15
Usually it gets automatically detected as "Mobile Broadband". If not, you may try adding it manually choosing/supplying relevant settings/options.


In another case, if you use an ID, password to get connected (i.e., a dialup connection instead of 'mobile broadband'), you'll have to install wvdial.
