### Post by mkvnmtr on 2008-08-03
8.04.1 is the latest stable release. I think for your Mac the intel disk will work. You just download the iso and burn it as an image in disk utility. You can try Kubuntu or Ubuntu or anything else that you think you might like. I don't have very fast service but it only takes about an hour and a half to download. When you have a disk restart holding down the c key and it should boot from the disk. You then have Linux on your machine. If you like it and everything works you can install from the disk. If you don't like it try another system. If you encounter problems with something you can search the forums for a solucion before you install. It is so easy to make a disk you can try several before you install. The people in the Mac forum are good about helping you when you have a problem.
