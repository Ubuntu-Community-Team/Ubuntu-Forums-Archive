### Post by uRock on 2010-05-02
Go to Synaptic Package Manager and search for and install Ubuntu Netbook Edition. This will install all of the stuff that makes the netbook edition what it is.

### Post by aysiu on 2010-05-02
As someone who's had a netbook for the past two years (first a 7" Eee PC, and then a 10" HP Mini), I haven't found any advantage to the Netbook Edition (or Remix) interface.

I'd recommend using the regular desktop interface and deleting the bottom (or top) panel.

If you do want to install it, just install the *ubuntu-netbook* package. If you don't know how to install software packages, read this:
[http://www.psychocats.net/ubuntu/installingsoftware](http://www.psychocats.net/ubuntu/installingsoftware)

### Post by TriBlox6432 on 2010-05-02
I'm running the normal version on my Dell Mini 10v netbook and it runs perfectly.

I made a few changes to make my own "netbook friendly version"

1, Delete the bottom panel.  2, delete everything on the top panel except the menu bar, network icon, and clock.  3, add to the top panel: Window List.  4, change the menu to autohide, and 21 px.  5, follow all the directions from the link at the bottom of my post.


I did all that, and it works great!  And there is a LOT more screen real estate with these changes, especially in firefox.  It's great!


[http://www.ubuntumini.com/2010/04/customizing-firefox-36-for-netbooks.html](http://www.ubuntumini.com/2010/04/customizing-firefox-36-for-netbooks.html)
