### Post by quadproc on 2010-02-25
Wow!

First, I should say that you are contemplating a huge leap, not to be taken lightly

 If, after you check out the ramifications of such a move and decide to do it, the probably the easiest way is to get a copy of the Ubuntu version (make the 32/64 bit choice, and select the release that you want) on CDROM and boot from that.  It will allow you to run the system under Ubuntu and you can decide if you like it, without changing anything on your disk (unless you ask it to).  If you like it then you can install it from the CDROM and it is a fairly quick and painless process without a lot of knowledge required.

The main issue here is probably "how to get the CDROM".  If you have a suitably fast Internet access then you can download it (700 MB) and burn your own CDROM; see the [http://www.ubuntu.com/getubuntu](http://www.ubuntu.com/getubuntu) site for more ways to get it. The Ubuntu folks will even send you a free CDROM if you are willing to wait weeks for it.

Be sure to get an appropriate version of the software, either 32 or 64 bit.  It isn't obvious but the usual distinction is that the 32 bit versions are known as Intel while the 64 bit versions are known as AMD, regardless of who actually made the CPU that you are using.  (I am right now using a 64 bit processor made by Intel and I am using the 64 bit AMD software.)  The reasons why things are this way are interesting but beyond the scope of this note.

Choosing the release version is more difficult.  I am running 9.04 because it has been out long enough to be fairly stable; most of the serious bugs have been found and many have been fixed.  The latest version is 9.10 but I have seen enough problems with it that I decided to wait a while before making the changeover.

Thanks for considering a good software system, and welcome!

quadproc
