### Post by jucas_lo on 2009-03-01
There ara a few linux based operatin systems for cellphones: One of the most talked about right know is Android, made by Google to conquer the mobile world.

There is also OpenMoko, which atempts to bring the open source world not only to software but also to the design of the hardware used to build a mobile phone.

Also some Motorola phones used linux based operating systems.

And the next version of ubuntu will be compatible with ARM processors which would theoretically mean that Ubuntu could run on a cellphone!

Also some cellphones from HTC are linux based....take a look at [this ]("http://tuxmobil.org/phones_survey_htc.html")site to find more info.
