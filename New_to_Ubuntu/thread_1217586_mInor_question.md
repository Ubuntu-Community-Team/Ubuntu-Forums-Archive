### Post by powel212 on 2009-07-19
If you have not already;

Make sure your system is up to date,

then install compiz config editor;

```
sudo apt-get install compizconfig-settings-manager
```

once installed it will appear under "system/preferences/compiz config editor"

I hope that helps.

### Post by jeppewinther on 2009-07-20
Ubuntu(Gnome, really) is slightly odd in that sometimes there are several ways to change a single setting, and often only one of them is "right", IE one overrules the other(s).

The master control for desktop settings is the Compizconfig-settings-manager.
