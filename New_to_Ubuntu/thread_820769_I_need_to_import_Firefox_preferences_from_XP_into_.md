### Post by Hospadar on 2008-06-06
Easy!
It's slightly different for firefox 2 (I'm using 3) but I'm sure you'll be able to figure it out.

Bookmarks->Organize->Import And Backup->Export HTML

Then in linux, do pretty much the same thing to import.

You could hunt down the link to the bookmark file and make a link to your xp partition I think, but an easier solution would be to use delicious to deal with your bookmarks.  I have a bunch of different systems I use, so I started using delicious to manage my bookmarks a while ago and never looked back.
Here's their main website: [http://del.icio.us/](http://del.icio.us/)
Here's where you can get an addon for firefox 3 to integrate their service seamlessly: [http://blog.delicious.com/blog/2008/04/firefox-3-delicious-and-you.html](http://blog.delicious.com/blog/2008/04/firefox-3-delicious-and-you.html)

Basically, when you add a new bookmark, they store it on their site, so it will get pulled down onto any machine you use with the addon, and if you arn't on a machine with the addon, you can just go to the site, it's le easycakes.  Also the addon will ask if you want to import your bookmarks to delicious when you first install it.
