There are several ways you may be able to get the suite itself running (not in any particular order):

First: dual-boot Ubuntu (or whatever Linux flavor you prefer) with Windoze. It's very easy to do. There's a downside, although not a major one. If you're in Linux, you'll have to restart and choose Windows in the boo screen. 

Second: Install Sun's Virtual Box (the PEUL version, not the OSE). You can then install Windows under Linux. If you have Windows with the Adobe suite installed, you can even set it up to use that installation, as long as it's on the same machine. Visit their website for instructions, downloads and info. That way, you just launch Virtual Box, and up pops your windows installation!

Third: This is a remote one... You can see if Wine (a windows emulator) will support Adobe. It supports a surprising number of games...

Fourth: You can install Ubuntu as an application under Windows (this is kind of like the second option, except in reverse).

I'm sure there are packages similar to each app in the Adobe suite, but it sounds like you want them all collected, hence my suggestions that you find a way to run Adobe.
