### Post by roger_1960 on 2011-04-14
Hi

To run  a command such as you quoted, you need to open the "terminal" from accessories>terminal (main menu)

Then type (or copy and paste the command) > sudo vnstatd -d
and then enter your user password and press return.  The password will not appear as you type it.(security!!)

For more  info on command line instructions you can type > man sudo for information on sudo or say > man ls for information about the list command.  Press Q to leave the information screen.

If you do a general search for > command line instructions linux or similar in google, lots of links appear.
