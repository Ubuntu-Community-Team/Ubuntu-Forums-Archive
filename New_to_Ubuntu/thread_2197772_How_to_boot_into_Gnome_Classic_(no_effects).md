### Post by grahammechanical on 2014-01-05
I think that you will find that the last used User Interface automatically becomes the default user interface. In other words if we want to change user interfaces we have to click the icon again. So, if you set your wife's computer to automatic login and she was using the Ubuntu user interface then remove automatic login and reboot. At the login screen select the "classic" user interface and after she is logged in set the machine to automatic login again and she should automatically load to a "classic" desktop every time she loads Ubuntu.
