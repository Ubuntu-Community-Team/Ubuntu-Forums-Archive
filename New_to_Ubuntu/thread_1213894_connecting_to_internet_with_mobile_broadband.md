### Post by Ji Ruo on 2009-07-15
Mobile broadband works automatically for 8.10 'Intrepid' and onwards, so a simple solution is to upgrade to the 8.10 or 9.04. If you do you can just plug in the dongle and it will work. The only exception I've found is Kubuntu 9.04 where I couldn't seem to get it to work with the new network manager, so you will need some further help if you decided on this version.

You will be able to find a solution for 8.04 by searching the forum, but if you're new to linux then upgrading to or installing one of the newer releases will be the easiest way to get it working by far.
