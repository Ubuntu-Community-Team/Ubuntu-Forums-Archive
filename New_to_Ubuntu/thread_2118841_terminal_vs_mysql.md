### Post by prodigy_ on 2013-02-22
Funny question. :)

Well, terminal (aka [system console](http://en.wikipedia.org/wiki/System_console)) is a tool that allows you to execute commands or scripts and see their output. And [MySQL](http://en.wikipedia.org/wiki/MySQL) is an RDBMS, a client-server software that can manage various databases and - really - is no more like a terminal than a raven is like a writing desk.
