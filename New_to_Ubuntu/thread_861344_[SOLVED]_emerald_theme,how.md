> i downloaded a matrix emerald theme but i dont know how to get it to work.

:confused:rick

Open up "Emerald Theme Manager' (System->Preferences->Emerald Theme Manager)
Hit the "Import" button and browse to your downloaded Emerald theme (should be a *.emerald file).  Double click file or click 'Open' button and you should find your theme listed there.  All you need to do to load it is click it once (to highlight it) and it should automatically apply it... 

If the theme does not apply, you may need to launch emerald - this can be done by running "emerald --replace" (alt+F2 and entering 'emerald --replace' in the run box).

Hope this helps..

-Matt
