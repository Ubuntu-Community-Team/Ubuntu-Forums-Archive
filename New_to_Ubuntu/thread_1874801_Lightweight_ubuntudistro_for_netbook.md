### Post by I2k4 on 2011-11-10
I think you'd do yourself a favor to experiment with Live USB (Persistent) with two or three - and with a netbook, you do well to install and compare the current 11.x with last year's 10.x because the new iteration is more demanding on computer resources.  My Acer was noticeably slower on 11.x any distro, than any 10.10 (except the very sluggish "Netbook Remix" which was an early flier at Unity).

I've recently put Mint 10.10 on a 4GB Live USB and like it in some ways even better than Lubuntu, which I like a lot.  The interface is more configurable and is quite fast when "effects" are turned off, and it comes preloaded with less stuff I want to uninstall and more (Firefox, GIMP) that I have to install. Mint also has a "user friendly" software interface (fewer choices than in Ubuntu's Software Center) but I've gotten used to Synaptic Package Manager from Lubuntu.
