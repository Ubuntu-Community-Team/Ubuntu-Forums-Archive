### Post by Bubba_Jackson on 2013-09-18
Are you asking which virtualisation platform supports Ubuntu? If so, you can run Ubuntu on any virtualisation platform as far as I know (VirtualBox, VMware, Citrix... ). I've only used VMware Workstation and VirtualBox (type 2 hypervisors - they run on top of another operating system, which is what I'm quessing you are looking for), can't say much about the others. Since [VMware ]("http://www.vmware.com/")is a commercial product, you can go with [VirtualBox ]("https://www.virtualbox.org/")(it's free). BTW, I'm guessing you're using windows.
