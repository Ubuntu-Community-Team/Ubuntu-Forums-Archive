### Post by PeterJS on 2008-04-29
I think he means PCMan. But any way, if you look at what the script posted on psycho cats does, it backs up the .desktop files for nautilus, and then downloads a new set that references the file manager you are switching to. If you download one of the scripts and use it and the associated .desktop files as templates you can set your file manager to be whatever you want.
