### Post by longfeltwant on 2009-03-31
I recently began migration to Ubuntu from my Mac. My favorite piece of software for Mac is Launchbar, which is a little utility that allows you to open things by typing some letters from the name of the thing. The thing could be a file, an app, a bookmark, an email contact, or just about anything. There are similar competing products, Butler and Quicksilver, which are also good but I prefer Launchbar. (Spotlight is also similar, but isn't very good at all.)

As soon as I installed Linux I started looking for a utility like Launchbar, and of course I found it built in to Ubuntu: Deskbar. Unfortunately Deskbar seems to be a lot like Spotlight, and is missing some of the most important features. I'm wondering if there might be a way to change the behavior of Deskbar, or if there is another program I should consider.

My specific problems with Deskbar:

* I don't want to type letters all in a row (for instance, I want 'ff' to result in FireFox, or a person's initials to result in an email to that person)
* The software should learn which letters correspond to which eventual item (for instance, 't' might mean Thunderbird to you, but to me it might be Terminal, and the program should learn that)
* It's got to work with Firefox 3 bookmarks
* Searching should happen by selecting where I want to search, and THEN what I want to search (so, if I want to search Google for 'Ubuntu' I would type G for Google, then type 'Ubuntu'; not 'Ubuntu', then scroll down to where the Search Google For Ubuntu item is)
* The item order should be learned, and calculated by appropriateness, not absolutely ordered as it is in Deskbar

I have done a little programming in my time, so the answer to this question might be for me to modify Deskbar, if that is reasonably straightforward.
