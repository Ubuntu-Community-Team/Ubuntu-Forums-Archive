### Post by Paqman on 2010-09-26
I don't know much about the nuts and bolts of what you need for serious hardware hacking, but on the software level Linux will give you full control of what goes on in your system, and you'll have access to a very large, very knowledgeable pool of geeks. Check out the IRC channels and you should be able to find like-minded people.

As for having to re-learn everything, you'll obviously have to do a lot of that if you switch OSes. Expect to have to climb a learning curve. But once you've done that you'll be using a system that has a long pedigree stretching way back to Unix. Many of the concept used will be in common with other systems from the same family tree such as BSD and even OS X.

If you want a very stable system you could go for a distro like Slackware. Personally i'd just use an [LTS release of Ubuntu]("https://wiki.ubuntu.com/LTS"), which has a 3-year lifecycle between upgrades (five years for the server version). You've also got options like being able to very simply build your own custom system that has only the components you need by using the [minimal ISO]("https://help.ubuntu.com/community/Installation/MinimalCD").

Bottom line is that you've got nothing to lose by at least trying Ubuntu. It costs nothing, and it's simple to set it up dual-boot so you don't have to ditch Windows if you don't want to. Uninstalling Ubuntu is just as simple as installing it.

### Post by kwacka on 2010-09-26
I'm not going to try and sell Ubuntu (or any variety of Linux) to you.

See 'Linux is NOT Windows - [http://linux.oneandoneis2.org/LNW.htm](http://linux.oneandoneis2.org/LNW.htm)  (its a few years old now)

If you do want to try Linux download Ubuntu (or any other distribution), burn it as a CD or on a USB stick and boot from that to take a look - if your hardware works, great. 

If there's any problems search the forums/google; you'll almost certainly find an answer.

There are programs that do similar tasks to Windows programs - sometimes better, sometimes without all the extras (e.g. GIMP doesn't do everything Photoshop does, but it will do what 99% of users need and costs 600 dollars less).

Regarding 'command line' - Windows has a command line too, but how many use it? You don't **need** to use command line in Ubuntu as a user but if you have some knowledge it will help (SHOULD you need to install drivers).
