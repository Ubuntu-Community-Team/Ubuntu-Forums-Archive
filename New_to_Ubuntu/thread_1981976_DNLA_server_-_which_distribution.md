### Post by MartyBuntu on 2012-05-28
If it's mainly for file-serving usage and you're more comfortable with a GUI, install Lubuntu or Xubuntu (lighter weight desktops) and run the server headless (accessing the desktop perhaps with a VNC client from another machine).

Try MediaTomb as the DLNA server. It's super easy to set up.
