### Post by Peter09 on 2010-05-24
During the very beginning of booting hold the shift key. This should get you to the Grub menu. Using the down arrow keys you will be able to boot into the recovery mode. There is a menu there which allows you to select the - reconfigure graphics option - not sure of the right words. Try that and then boot normally.
