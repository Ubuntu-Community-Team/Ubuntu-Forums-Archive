### Post by quelx on 2008-05-22
Don't know about Nautilus browsing with the buttons, but in Firefox type **about:config** in the address bar and promise to be careful.

in the filter box type **autoscroll**

double click the **general.autoScroll** to change the value to **true**

that's it.  middle mouse auto scrolling should work.
