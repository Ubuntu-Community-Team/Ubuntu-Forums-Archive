You did not backup her data? Yes, that would = severe panic.

Do you get a Grub menu? You may need to hold down shift if Ubuntu is the only OS on the machine. At the Grub Menu select Recovery mode. If you are on 13.04 you need to select Advanced Options for Ubuntu. I am having to guess what version of Ubuntu that you started off with and what version of Ubuntu you have now.

At the Recovery Mode menu select Grub - update Grub boot loader. See if that works. Have you set the BIOS to boot from DVD?
