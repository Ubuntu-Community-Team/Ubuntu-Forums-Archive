### Post by Joeb454 on 2008-07-22
It's basically software that provides some very fancy and useless (some are useful I'll admit ;)) effects for you're desktop. I *think* you could warrant calling it a Window Manager too, and a compositing manager (to get see-through title bars etc.)

### Post by schnauzer93 on 2008-07-22
```
sudo apt-get install simple-ccsm compizconfig-settings-manager
```

Then go to System > Preferences > Appearance and on the Visual Effects tab, change it to Custom. You can configure Compiz from System > Preferences > Advanced Desktop Effects Settings.
