### Post by Ksiencha on 2014-08-25
Yes, it's not Kubuntu, that's just a KDE desktop environment + Ubuntu.
You see there are different desktop environments (DE): KDE, Unity, Gnome, Cinnamon, Mate, XFCE etc. And there are Linux distros which have one of those preinstalled default DE.

Ubuntu 14.04 uses Unity DE. Kubuntu 14.04 uses KDE environment. Of course you have the option to install the desired DE on any Linux distro, but it's better to use distro with its default preinstalled desktop environment.

I hope this will help you to understand a bit ;)
