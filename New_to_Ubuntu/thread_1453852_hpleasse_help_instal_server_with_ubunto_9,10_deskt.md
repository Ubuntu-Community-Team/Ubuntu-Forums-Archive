To your question, if I understand it correctly, you received 3 CD's and are wondering what they all are.


Server and Desktop you have already identified.  I believe the blue CD may be Kubuntu, the KDE variant of Ubuntu.  The main difference is a different desktop manager, along with some KDE specific default applications.  Some like it better, some prefer Ubuntu with it's Gnome interface.  If it is a liveCD you can boot to it and try both before you decide.

Also, do not forget that you are not locked into one version.  If you install one and decide you do not like it you can change to the other.  You can even run both if you so choose and pick the one you want to use when you boot up.
