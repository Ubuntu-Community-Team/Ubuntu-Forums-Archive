> **anewguy said:**

Try ```
sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 2EBC26B60C5A2783
```

Maybe something here will help with the Synaptic problem? [https://bugs.launchpad.net/ubuntu/+source/synaptic/+bug/839219](https://bugs.launchpad.net/ubuntu/+source/synaptic/+bug/839219)
