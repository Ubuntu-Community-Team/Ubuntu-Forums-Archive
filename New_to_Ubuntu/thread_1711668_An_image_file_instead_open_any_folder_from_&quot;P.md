### Post by or3x on 2011-03-21
Rightclick on o folder in nautilus(file manager) -> Open with -> Other Application.. -> Choose "Open Folder" from the list AND tick the box that says "Remember this...."
The folder may open in a new window, close it, and try some folder from "Places"
