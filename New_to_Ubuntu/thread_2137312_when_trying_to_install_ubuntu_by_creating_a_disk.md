You will need to install an iso burning program to burn Ubuntu to disc. A popular app for doing so is [ImgBurn]("http://www.imgburn.com/")

Also, I recommend that you setup a dual boot, which allows you to choose to boot into Windows or Ubuntu at startup. Many people find that after they install Ubuntu, they are sometimes missing _______ app, which only ran under windows. Having a Windows dual boot allows you to boot into Windows when you need to use Windows-Only software.

### Post by Strongbad on 2013-04-20
In the installation options, it will ask you if you want it to start from scratch or keep windows and dual boot. The latter will resize the hard drive into two partitions, one for windows, and one for Ubuntu. Your windows partition should be accessible from Ubuntu, but not the other way around. I would also recommend at least starting with a dual boot. If you end up not using the windows partition, you can always get rid of it later.
