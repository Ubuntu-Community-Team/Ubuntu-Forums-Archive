### Post by Impavidus on 2018-06-23
Before you installed 17.10, I assume you created the 17.10 live/install disk. You can create the 18.04 disk in the same way. People shouldn't be installing 17.10 any more. When you install 18.04 from the official live disk, you should get everything you need for networking, with the possible exception of some wifi drivers. The trick is to set up a wired connection and then allow the system to find and install the required drivers automatically. The wired network interface is so standardised that it always works.
