### Post by jbaerboc on 2014-05-06
The user interface might be dead but it may have a background process still running which makes your computer think it's still running. The easiest way would be to restart your computer, then try to run it. If that fails you'll need to open the "system monitor" in Ubuntu and go to the processes tab. See if any of the ones on the list says Audacity or anything similar. If you find one you could kill the back ground process and try again.
