### Post by Impavidus on 2013-08-23
Maybe you already found it, but this is the general manual on making bootable USB drives: [https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)

The latest Ubuntu may just run on 1GB memory, but it won't leave much room for applications and after all, you have a computer to run applications, not to run an operating system. It may be better to try Xubuntu. It's the same system under the hood, officially supported, but with a less recource-hungry desktop environment.
