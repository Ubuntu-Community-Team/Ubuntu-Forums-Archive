### Post by 1clue on 2012-02-10
If you're going to upgrade a motherboard then you should just get a new computer.  Whether you build it or buy it is up to you.  I build mine.

In my experience you should use components which are all marketed at the same time.  There is less chance of having the new component sitting there waiting for the old one to finish up, and sometimes the old component was on the verge of dying anyway.  You would be amazed at what you get for your money these days.

With regards to photoshop and such:  Are you using this software in a professional capacity or are you a hobbyist?  If you're a hobbyist or on a severe budget, then you might want to consider Gimp and some other free tools.  I use Gimp when I need to do something like that.  I don't know what most of the features are, but that's OK.

My sister is a professional photographer.  She has the super-jumbo iMac, where it's a 27" flat panel.  She bought it, and didn't have money for the photoshop and associated software, so I showed her Gimp.  I installed it on my laptop first, she played with it there and was surprised that it would do so many things.  Gimp is an Open-Source app which is designed to be competition for Photoshop.  It is capable of doing nearly everything that Photoshop does.

She installed it and a couple other tools.  Then a month or so later I found out she wasn't using it.  The thing is, Photoshop is made for professional applications.  Gimp is made by a bunch of programmers who seem to think more is better.

My sister showed me the difference.  With Gimp, you use a tool and the slider adjusts from 0% to 100%.  On Photoshop, the same equivalent tool might go from 50% to 65% (I'm just making these numbers up) but those values are the range of what are actually used.  So with Gimp, you touch the tool and the values go all over the place because it's so sensitive.  With Photoshop you can fine tune whatever you're doing, which is what it's all about.

Also, Gimp doesn't necessarily group tools conveniently.

So she wound up with Photoshop, and I still use Gimp.

I think that if you use these tools professionally and those tools are what you use your computer for, then you should use whatever you need to make them work natively.  Which probably means Windows.

If on the other hand you're a hobbyist, I would recommend giving Linux a whirl and see if you like it.  Then give Gimp and associated tools a spin as well (they also come for Windows and Mac) and see if you like those before trying to mess with Wine or Crossover Office.

Good luck and have fun.

### Post by Scott Baker on 2012-02-10
Howdy Hoddo. I hope to give you the short and hairy of this quest of yours. First is your own observation of your abilities. While what you want to do is VERY possible within the Linux environment (I have two professional photography and five videographers as clients that have switched to Ubuntu), it will require a period of learning to use not just the Linux software you choose, but the photo packages required to do what you want to do. Recent releases of Ubuntu have made this task much easier, but it's still not Windows. My advice to you is as follows;
1. Buy, don't build the machine that you need to complete your tasks. Building a machine is not impossible, but as the builder, YOU eat the risks. Burn an I/C on a mainboard, there goes $200.00 to $2000.00+ (you get the point, right?) Having the machine built to your specs is a MUCH safer bet.
2. Load it with Windows 7 (professional or ultimate, nothing less if you're serious about the photo work). When you get the Windows 7, make sure to ask your vendor about the upcoming Windows 8 to see if you can qualify for an upgrade when it's time.
3. Try GIMP at [http://www.gimp.org/](http://www.gimp.org/) This is a software package that has been around for a VERY long time, and is quite polished. It allows users to work all kinds of magic on photos. Try it out. It's available for Windows, and a freebie. Beware though, it's a BIG download. It offers numerous plugins that may cover what you are now doing through your paid applications.
One responder stated that you could try your applications through WINE, but this can be hit or miss. While MOST Linux users know how to address these issues, now is NOT the time to learn how to correct these issues (especially if your photo work is school or WORK related). Hope this has given you some serious food for thought. Good luck with your quest.  :-k

### Post by Learning Linux 2011 on 2012-02-11
Hello and welcome!

All of the Mozilla products you mentioned have Linux versions and run very well.  (Firefox is the main browser for Ubuntu anyway).

You can install a product in Linux called **Virtualbox**, which will allow you to run Windows (and Photoshop, Lightroom, etc.)  inside Linux.

I can't tell you about the RAM. If you want more than 4 GB of RAM you will need the 64-bit version of Linux or Windows, whichever you chose.
