### Post by Sub101 on 2010-04-07
ClamAv is in the repositories, with a GUI frontend of Klamav.

If you go to System -> Administration -> Synaptic Package Manager

and search for clamav then you will be able to install.

As to whether you can install Unix programs in Linux then in general Yes, you can. It is not, however, a simple matter of double clicking and installing but requires building the program from source.

### Post by undecim on 2010-04-07
You can install ClamAV in Ubuntu, but there really is no need for it unless you are scanning other drives or files from Windows computers. Linux is built in such a way that it is difficult for a virus to infect it.

Most of the utilities you are used to in Unix have been ported to Linux. Linux is considered a "Unix-like" environment, so if you have any experience with Unix, you wuld do well with Linux.

Ubuntu is based on the Debian branch of linux. The files that are natively installable are .deb files. You can compile source code into programs that will work under Ubuntu, but chances are you won't need to do so.

The Ubuntu repositories have well over 20,000 different packages available, so chances are you'll find everything you need there.

If there is something specific you need, ask in the forums and chances are very good that someone can help you out.
