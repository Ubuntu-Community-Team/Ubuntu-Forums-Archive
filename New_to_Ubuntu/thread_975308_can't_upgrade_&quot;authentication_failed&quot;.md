### Post by w4ett on 2008-11-08
I'm assuming you are trying to upgrade from 6.06....  Feisty Faun has reached the end of it's 18 month support cycle, so the upgrade probably won't work see:  [http://www.ubuntu.com/news/ubuntu-7.04-end-of-life](http://www.ubuntu.com/news/ubuntu-7.04-end-of-life)

I suggest (If you want to use the current LTS version with 3 year support) using 8.04 Hardy Heron.  If you don't currently have a copy for install, look here: [http://www.ubuntu.com/GetUbuntu/download](http://www.ubuntu.com/GetUbuntu/download)

Follow the wiki instructions for burning the ISO to a CD or USB Thumbdrive: [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto) , Install and Enjoy.
