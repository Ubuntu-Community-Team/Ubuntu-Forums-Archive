### Post by perpetualcacophany on 2009-04-09
It would be helpful to know what kinds of programs you are looking for. Are you looking for games or educational content or what?

If you want to have some good educational software on there you could try installing Edubuntu which contains some software aimed toward teaching younger children.

```
sudo apt-get install edubuntu-desktop
```

As far as games go, there are some good games that younger kids will enjoy in the repositories like Frozen Bubble. Just search around.

If you give some more specifics you will probably get better answers.

### Post by knattlhuber on 2009-04-09
Some stuff that our 10-year old enjoys:

tuxmath - math game for kids with Tux
tuxtype - Educational Typing Tutor Game Starring Tux
tuxpaint - A paint program for young children
extremetuxracer - 3D racing game featuring Tux, the Linux penguin
smc - a Jump and Run game like Super Mario World

And the Compiz effects like the rotating cube fascinate him big time...
