### Post by BBQdave on 2012-03-26
> **Pat59 said:**
> I have some older dell inspirions laying around and I thought I might wipe the drives and clean install Ubuntu.

Running Debian 6 on a dell inspiron 1100 (1 gig of ram), stable great distro  :D

Would recommend distro with older kernel (2.6.32) for older hardware. The newer linux kernel and distros tend not to play nice with older hardware. Debian 6 is supported for a good while yet; Ubuntu 10.04 LTS reaches EOL soon.

This old notebook handles Debian 6 (gnome 2) with backports (new applications) well. It's my main machine for production and play.
