### Post by ajgreeny on 2017-07-29
Start any GUI application and an icon for it will appear in the launchbar.  Right click on that icon and select "Lock to launchbar" and it will stay there even when you close the application.

GUI application icons locked to the launchbar should be removed by the system when the application itself is removed, but if not you should be able to reverse the lock with another right click and select "Remove from launchbar"
