### Post by roe_polak on 2008-05-18
If you're serious about making music on Linux I can really recommend using Ubuntu Studio. It can be installed directly from synaptic, no need to download huge DVD-iso's. Also, you should have a decent sound card.

As for programs turning the keyboard into I think that most of the apps in Ubuntu Studio does this. For drums I use Hydrogen. Useful for spontaneous jamming in the appartment :)  I also remember that a program called lmms ([http://lmms.sourceforge.net/](http://lmms.sourceforge.net/)) that lets you do this.

I've never tried GarageBand hence not sure exactly what type of program you are looking for.

The multitrack editor called Ardour does not have this functionality, but the great thing about Ubuntu Studio and its sound server called Jack is that all applications can be linked. You can direct the spund signal from one app to be recorded in another.
