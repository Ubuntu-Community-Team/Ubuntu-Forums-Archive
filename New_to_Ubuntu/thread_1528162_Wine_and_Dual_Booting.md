Wine is a Windows compatibility layer for Linux and runs completely within Linux. It will not affect a separate installation of XP in any way.

A bit more info is that the wine environment creates a hidden folder structure in your Linux home directory into which Windows programs are installed. Dual booting won't be affected by Wine at all. Cheers yetiman64
