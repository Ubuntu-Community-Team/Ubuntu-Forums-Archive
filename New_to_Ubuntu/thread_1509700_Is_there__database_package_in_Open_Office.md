### Post by da burger on 2010-06-14
Go to System>Admin>Synaptic Package Manager.

This is a more advanced way to install software than ubuntu-software center. 

Alternatively if you don't mind typing something into the terminal run ```
sudo apt-get install openoffice.org-base
``` then type in your password. (You won't get any visual feedback while typing the password, don't worry about it just type the password and press enter.)
