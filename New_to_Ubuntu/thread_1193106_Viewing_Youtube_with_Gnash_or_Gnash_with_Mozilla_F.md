### Post by LinuxFox on 2009-06-22
Gnash is just a free software alternative to Flash Player.  It's free as in freedom, but when I tried it, I had problems using some flash sites (then again, that was when I tried Xubuntu).

You could try Flash Player 10.  You can download it [here]("http://get.adobe.com/flashplayer/otherversions/"), by selecting Linux and then choose the .deb package.

Just install it by double clicking it, and see how that works.  If you need to uninstall it, just go to Synaptic Package Manager (under System > Adminstration menu) and search for Flash Player to find it and uninstall it.

I hope this helps.

### Post by jenkinbr on 2009-06-23
The security issues you experianced in Windows have probably been taken care of.  As most malicious third-parties attack Windows machines, you should be relatively safe in Linux.  If you keep regular backups of your data, you should be fine.

Flash Player can also be disabled from within Firefox by going to [Tools >> Add-ons] and clicking the [Plugins] tab then scrolling down, selectig <Shockwave Flash> and clicking [Disable], then restart Firefox.  Repeat the same steps to re-enable it again.
