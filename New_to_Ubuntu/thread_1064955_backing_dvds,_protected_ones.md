### Post by jbrown96 on 2009-02-09
Brasero or K3b can create images of the disc, which is probably what you want. If it fails quickly, then you need an extra package. You may need to get decss2, which is available through the medibuntu repository. Instructions on how to configure it and install decss2 are available [here]("https://help.ubuntu.com/community/Medibuntu")
