### Post by vishzilla on 2008-06-28
Right click on the panel where the menu is present and select 'Edit Menu'. Go to the Internet submenu and click on 'New Item'. Enter the description with name Firefox 3 and in command enter the path leading to firefox ie /home/USERNAME/firefox/firefox
