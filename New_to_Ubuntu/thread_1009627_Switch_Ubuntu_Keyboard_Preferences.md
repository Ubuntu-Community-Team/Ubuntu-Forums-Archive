### Post by Frogbarf on 2008-12-12
Go to System > Preferences > Keyboard > Layouts

Add the keyboard layout you want (probably USA)

mark it as default in the radio box to the right

delete the Canadian layout

close

I'm studying Georgian in a desultory way :popcorn:and regularly enable and disable the Georgian keyboard layout. It's r-e-a-l-l-y to do.
