### Post by TheFu on 2017-08-11
If you google for "ubuntu desktop guide", many user interface things are explained. This will be for a system running "Unity" desktop.  There are about 30 other desktop choices and you will find that Linux people like many different desktops, so it isn't like Windows where asking a GUI question of anyone will get someone who knows about the same GUI you are using.

For example, I haven't used Unity since 2012.

If you want to understand the most about Linux in the least time and have about 20 minutes to jump start it - [http://www.dedoimedo.com/computers/ultimate-linux-guide-for-windows-users.html](http://www.dedoimedo.com/computers/ultimate-linux-guide-for-windows-users.html) is a good start.  Some of the other articles on that site aren't so accurate, but they are professionally written and get the main points across well enough, if you get reading there.
