### Post by drs305 on 2008-06-14
If you open /etc/apt/sources.list you are likely to find the word "web" appended to the end of the entries. Make a backup of sources.list and then edit accordingly (remove "web"). 

```
sudo cp /etc/apt/sources.list /etc/apt/sources.list-bak
gksudo gedit sources.list
```

If you have questions, post the results of:
```
cat /etc/apt/sources.list | grep -v "#"
```
