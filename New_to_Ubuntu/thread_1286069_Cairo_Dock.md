### Post by howefield on 2009-10-08
Sure, add it to your startup applications.

System > Preferences > Startup Applications

Click the add button, and fill in the 3 fields. 

Type whatever you want in the Name & Comment fields, but it needs to be the startup command for cairo-dock in the command field.

Probably cairo-dock but you'd need to check that.
