### Post by bennachie on 2009-04-27
If you let the installer do its own thing, you will be left with about 1.3GB for the your documents and settings. If you set the swap partition size to zero, you'll gain another 256MB (a swap partition of that size is pretty useless anyway). Interestingly, the vanilla Gnome version, which works just fine on my eeePC 4G uses slightly less disk space than UNR (and you don't have to put up with the glacially slow launcher - although I assume that a bug-fix for that problem should soon be forthcoming).

I use an 8GB SD card, mounted as an external disk, for my main document storage, so the 1.5GB available on the internal disk is more than sufficient for the other bits and pieces.
