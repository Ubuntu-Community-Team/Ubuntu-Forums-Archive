### Post by WasMeHere on 2011-09-29
It is called 'Search' in gedit or translated to something meaning the same thing in your own language. So you have it in both gedit and terminal (assuming you use bash).

Search looks for matches in the text that you are viewing/editing, while 'find' in terminal looks for matches in the file system, and I agree with *TeoBigusGeekus*, it is very powerful.
