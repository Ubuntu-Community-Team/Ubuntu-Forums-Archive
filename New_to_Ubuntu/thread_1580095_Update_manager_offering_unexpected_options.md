### Post by e79 on 2010-09-22
It is quite normal as the Update Manager will update w/e software is installed on your computer; and all the softwares you mentioned are part of a basic install. You could decide to 'uncheck' them box in the update manager if you don't want them to be updated but I would not recommend it as sometime security holes are patched through these updates.

### Post by mikewhatever on 2010-09-22
All those programs (and about a hundred more) are installed by default when installing Ubuntu. You don't have to update them, but then, consider turning the Update Manager off altogether.

### Post by lbrty on 2010-09-22
As previous posters have stated, these applications were installed by default when installing Ubuntu. If you do not want non-essential programs installed on your system, simply goto System>Administration>Synaptic Package Manager or Ubuntu Software Center and uninstall the software you do not use or want. Some may be dependencies for programs you do use so choose wisely.
