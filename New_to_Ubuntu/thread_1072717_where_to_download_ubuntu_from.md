Please be aware that us folks are just here helping out - volunteering advice rather than providing paid-for technical support. *Nobody on these forums is contractually obliged to provide you with any assistance at all* - we do it because it's nice to be helpful. Before you go wiping Windows, why don't you consider a dual boot set up. This way you have a safety net of Windows which you are comfortable with, and can then learn Linux at a pace you are comfortable with. This way if you ever get into a jam you can just boot into Windows. Also you may need Windows to run some of those programs you mentioned.

Another option is to use virtualization to run Windows inside of Linux. There is no shame in using Windows either because you like it or are comfortable with it or because you need a certain program which can't run in Linux.

All I am saying is think about your options before you just banish Windows.
