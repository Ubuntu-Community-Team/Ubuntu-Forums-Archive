### Post by SoFl W on 2011-07-27
> **Swagman said:**
> I've also found the search facility on here to be quite.....crappy !!

There is a way to put a ubuntu forms search bar into Firefox's search engines.  It allows multiple words and works quite well. Much better than the forum search.

In a terminal enter
```
sudo nano /usr/lib/firefox-addons/searchplugins/ubuntu-forums.scr
```

enter the following code:
```
<search 
 name="Ubuntu-Forums"
 method="GET"
 action="http://www.google.com/search"
 queryCharset="utf-8"
> 

  <input name="q" user>
  <input name="sitesearch" value="ubuntuforums.org">
</search>

```

Save the file and restart firefox, select "Ubuntu-Forums" in the search pulldown menu and use the search box to query the forums.
