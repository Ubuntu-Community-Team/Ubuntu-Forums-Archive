### Post by Kobalt on 2009-08-15
You can use the tool included in Ubuntu to create a startup USB disk (if you run Ubuntu 9.04) : Ubuntu startup disk creator.
Or you can also use [Unetbootin]("http://unetbootin.sourceforge.net/").

Make sure your "old" computer can boot from USB though... Check out the BIOS to know it.
