### Post by capleton on 2008-05-17
I also had a similar problem.  Also had to re-install windows.  But it is possible to fix without re-installing.  First work on getting the Grub screen back.  Check out [gfx grub]("http://tuxenclave.wordpress.com/2008/01/18/how-to-install-gfx-grub-in-ubuntu/").

Afterward, if still no fix, set the bios to boot from cd, and put in the Windows recovery cd.  At this point, I forget a little, but do some googling on grub and windows and you should be able to figure out what it is you need.
