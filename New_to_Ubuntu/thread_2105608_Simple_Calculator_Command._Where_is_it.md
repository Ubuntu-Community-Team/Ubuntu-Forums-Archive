### Post by mcduck on 2013-01-16
gcalctool automatically stores the previous answer, so you can just type in "1/ans" for the same result.

Also if you want a simple yet really powerful calculator, I'd definitely recommend giving [Qalculate!]("apt:qalculate-gtk") a try. Very easy interface and more features and functions than any one person could ever need. And yes, it supports RPN! :)
