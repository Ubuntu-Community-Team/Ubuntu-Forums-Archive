### Post by QIII on 2012-04-27
Bear in mind that if by "Office" you mean "Microsoft Office", MS Office will neither install nor run on Linux without using a utility/app called Wine, which provides a compatibility layer.  In general, Windows applications do not run natively in Linux.

Wine does not run all MS applications equally well.  In fact, there are some that will not run at all.  [http://www.winehq.org/](http://www.winehq.org/) has a database of what has been found to work to various degrees.
