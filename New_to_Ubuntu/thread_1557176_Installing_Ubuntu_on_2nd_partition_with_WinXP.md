### Post by Zzl1xndd on 2010-08-20
It is possible and its relatively straight forward. If you already have 2 Partitions, you should be able to start the installation and select the second drive as the place to install. Ubuntu should do the rest. 

That being said I would recommend backing everything important up, just in case.

### Post by Thryn on 2010-08-20
[https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements)

According to the page above (as well as what I've heard on the forums and what my own RAM usage indicates) 256 MB RAM is a little skimpy for regular Ubuntu. Lubuntu is a lightweight version, and I liked it, except that I had some minor configuration difficulties with it. Xubuntu is also supposed to be lightweight, but I'll let others weigh in on that because I have yet to try it.

You can still try regular Ubuntu but it will likely run slowly.

That said, [I followed this guide to set up dual-booting on my computer]("https://help.ubuntu.com/community/WindowsDualBoot") (Ubuntu and Windows 7). It was quite easy.
