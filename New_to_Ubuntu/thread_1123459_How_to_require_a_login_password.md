### Post by mmitt on 2009-04-12
When I installed Ubuntu, I set it to not require a login password. Now that I've gotten more comfortable with ubuntu (:)), I don't want people to be able to just take my computer and see everything. Can I require a login password, like in Windows, where it asks you for your password? Thanks!

---

### Post by Paqman on 2009-04-12
Yep, go to System > Admin > Login Window > Security and uncheck the timed login and automatic login options.

If you want a nice face browser login like you get on Windows then go to the Local tab and pick one of the login windows that has a face browser.
