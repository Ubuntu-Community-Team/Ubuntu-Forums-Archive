### Post by Lars Noodén on 2012-11-05
There are several point of sales packages available for Linux.  You might start with looking at them and trying to find out what feedback has been given.  Build on the strengths and no need in making the same mistakes twice.

As far as platform goes you can use the netboot cd image to build a bare-bones system.  You don't want anything extra.  Also, it should be locked down so it can't get changed either on purpose or by accident, so set it up as for a kiosk.  There is a lot of information about how to lock Linux down into a [kiosk mode](http://www.alandmoore.com/blog/2011/11/05/creating-a-kiosk-with-linux-and-x11-2011-edition/).
