### Post by 3rdalbum on 2009-06-27
Your Ubuntu virtual instance will not contract viruses through web surfing, much less pass them on to XP.

However, you must still run either a firewall on your internet gateway / modem or a personal firewall on XP to avoid self-sufficient worms.
