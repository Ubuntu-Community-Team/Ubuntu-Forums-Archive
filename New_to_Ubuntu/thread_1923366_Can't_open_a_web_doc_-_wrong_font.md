### Post by SeijiSensei on 2012-02-10
The simplest solution is to install **ubuntu-restricted-extras**.  Along with things like video codecs, that will offer to install the Microsoft fonts.  You'll see a screen that presents the EULA for the fonts.  Use the Tab key to highlight the <OK> option, then hit Enter to continue.

If you just want the fonts, install the **msttcorefonts** package.
