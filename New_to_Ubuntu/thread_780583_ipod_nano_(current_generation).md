yep all the current generation ipod nano's works with ubuntu through rhythmbox.

but a small problem is there: if your ipod nano is mounted or used on Mac(HFS+ mounted format)  first then it's hard to make it work in ubuntu.

if it installed or used on windows platform (FAT32 format), it will work straight through rhythmbox.
