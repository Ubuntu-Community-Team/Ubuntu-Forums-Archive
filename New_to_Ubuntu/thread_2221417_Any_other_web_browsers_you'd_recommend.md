I'm completely serious.  It is amazing how much faster surfing is with Lynx compared to all other browsers. Less crap gets loaded, so it is much faster.

If you care at all about privacy, do not use Google's Chrome.

Also, running Java in a general purpose web browser is extremely dangerous.  It is the #2 way that people get hacked. Ask any government IT security person.

If you 
a) make money with a java applet (otherwise .. stop using it)
b) setup a special browser ONLY for use with specific websites that run Java Applets (never use it elsewhere)
c) ensure that Java-applet-only browser IS NOT THE DEFAULT for your machine, or your account
then you can be less worried about running Java in a browser. Otherwise it is just too large a risk to consider.

I'm 100% serious. [http://krebsonsecurity.com/2013/09/researchers-oracles-java-security-fails/](http://krebsonsecurity.com/2013/09/researchers-oracles-java-security-fails/) is worth a read to see a few of the warts.
