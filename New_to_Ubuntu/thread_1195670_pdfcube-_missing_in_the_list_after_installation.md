### Post by merlinof2 on 2009-07-09
It does not install itself into the menuing system.  Instead, is should be used as a command in a presentation like environment.

google "ubuntu pdfcube" and you'll find various sites on the program, one of them is a review which oddly enough, is the only place I could find instructions on how to use it.

you can simply open a pdf with the program,  but you'll see none of it's capabilities.  It should be opened as "$ pdfcube filename.pdf (X) (Y) "  where X and Y are transition types and limits.

good luck, it looks to be a cool package for anyone in the business of making multiple presentations, but it won't work for what I want.
