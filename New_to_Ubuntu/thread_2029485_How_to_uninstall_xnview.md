### Post by ty74 on 2012-07-19
I had the same problem with thunderbird. I solved it by installing synapic package manager. I scrolled through the list of programs I installed and found thunderbird files still there so I clicked the box for removal and that solved my problem. I hope it solves yours too. You can install synapic from the ubuntu software center or sudo apt get install synapic package manager.
