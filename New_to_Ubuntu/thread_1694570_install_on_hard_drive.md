### Post by hansolo4949 on 2011-02-24
I do not see why you would want to take it out and install it in another computer to install Ubuntu on it, since you can change the boot order in the bios menu to set the cd drive as the primary boot device and boot from it.:) but to answer your questions, yes, the HDD would be recognized in the other computer, and you can swap one HDD between the two if you want. You also do not have to remove windows from the HDD to use Ubuntu, you can dual boot them just fine, if you have enough space.

1) Yes, when you get to the part where it asks about partitioning choose to use the entire disk.
