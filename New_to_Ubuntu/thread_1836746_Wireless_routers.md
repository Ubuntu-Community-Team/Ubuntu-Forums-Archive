### Post by Bucky Ball on 2011-08-31
Doesn't make much difference with a router. You have a wireless card, it is a wireless router. As long as they are both transmitting and receiving it doesn't make much difference what OS you are using. The router configuration is done using a web browser so you use Firefox. That easy.

Might be better to ask yourself what features you want in a router and start researching for one that has them (unless the one you mention is the one, in which case go for it).
