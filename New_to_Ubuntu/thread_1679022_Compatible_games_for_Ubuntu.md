### Post by mastablasta on 2011-01-31
You can run some windows games via WINE. you will have to check which ones here: [http://appdb.winehq.org/](http://appdb.winehq.org/)

there is also plenty of Games available in Ubuntu software center.
a few more are to be found on [www.playdeb.net](www.playdeb.net)

small games work well. flash based, java based...

there are also some other games that have native linux clients or ports. for example Doom3. they might need a few commands to be copied in order to install them.
