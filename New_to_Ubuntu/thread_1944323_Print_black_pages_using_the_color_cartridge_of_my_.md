### Post by wyliecoyoteuk on 2012-03-21
The setting you need is "composite black", sometimes called "process black".
This uses cyan magenta and yellow to print black (actually a really dark brown)

### Post by anewguy on 2012-03-21
OK, when you ask for something to print, does a box pop up with various tabs on it, starting with "General" where you select your printer?  If so, check the "Color" tab and see if it has any option in the pull-down for "Output Mode" to use color only.  For my wireless HP the only options I get are "Color" which still uses my black cartridge, or gray scales which also use my black cartridge.

Since I have always had HP printers, except for a Brother non-color laser printer, I have no idea if these options change depending on the printer.  You may want to check the other tabs as well and maybe see if there is an "advanced" button somewhere on one of them that may contain that option (again, not on mine - I have an "Advanced" tab but there is nothing on that page.
