### Post by boxcorner on 2010-02-28
Here are some things that I've discovered one can do using the Super key (Windows key, Windows logo key, WinKey, Start key, Flag key or whatever else you may want to call it!)

Super key + E zooms out so both workspaces are displayed, reduced on the screen.
Left-click mouse pointer to drag this window to the other workspace.

Super key + scroll wheel on mouse zooms in and out.

Super key + M inverts screen colours (except desktop).

Super key + N inverts window colours.

This may be nothing new for experienced users, but I thought other newcomers might find it interesting.

Have fun!
