### Post by gamblor01 on 2013-04-27
Typically yes you need to root your phone to be able to tether.  There is one (free) option called azilink that works without rooting the phone though.  The rest I am aware of all require you to pay for them.  There is a great wrapper script for installing everything which you can read about and obtain here:

[http://www.humans-enabled.com/2009/12/how-to-tether-your-verizon-droid-as.html](http://www.humans-enabled.com/2009/12/how-to-tether-your-verizon-droid-as.html)


Pay particular attention to the very first paragraph on the page which has the link to his install script and the readme:

[http://dl.dropboxusercontent.com/u/394584/Droid-Tether-SV-README](http://dl.dropboxusercontent.com/u/394584/Droid-Tether-SV-README)

### Post by MoebusNet on 2013-04-30
Something to consider is whether or not you can mount your Android phone as a USB drive. If you can, great! If your Android version is newer than 4.0.3, and if your computer hardware is capable, one of the improvements to Ubuntu 13.04 is the ability to mount the file system of the newer Android phones, which the older versions of Ubuntu cannot do by default (takes a work-around). I'm using azilink on my Android phones that have the older file system.

Just in case:

[http://www.omgubuntu.co.uk/2011/12/how-to-connect-your-android-ice-cream-sandwich-phone-to-ubuntu-for-file-access](http://www.omgubuntu.co.uk/2011/12/how-to-connect-your-android-ice-cream-sandwich-phone-to-ubuntu-for-file-access)
