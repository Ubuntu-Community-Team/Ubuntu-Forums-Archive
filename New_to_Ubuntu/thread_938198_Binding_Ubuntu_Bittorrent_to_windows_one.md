> Is there any way that Ubuntu's bittorrent recognize those downloads and continue downloading from the point they are on windows??


Yep.

The default torrent client is called Transmission. Just start up a new torrent, making sure to give it the right download location for your partially completed torrents. It should just check the partial file then resume the download.
