### Post by SlidingHorn on 2014-01-21
System Tools > Preferences > Main Menu

Make sure the box is checked for the "Internet" folder.

If the "Main Menu" isn't listed under Preferences, open a terminal:

```
sudo apt-get install alacarte
```
