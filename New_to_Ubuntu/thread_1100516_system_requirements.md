### Post by Kevbert on 2009-03-19
Your display adapter needs to be either ATI, Nvidia or Intel, as these are the only officially supported cards. You typically need 128Mb graphics memory. If you are using Kubuntu, Kwin (the KDE version of Compiz) comes with KDE4.2 as standard.
