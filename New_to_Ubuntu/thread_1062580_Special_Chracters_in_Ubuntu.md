### Post by yther on 2009-02-07
If you often type in several different languages, I think you will become very happy if you set your keyboard layout to one that has a "Compose" key.  Then, instead of having to remember "Alt-xxx" codes ("Normal"?  How bizarre!), you would do something like Compose+o-" and get ö, Compose+a-' for á, and so on.  It's generally more intuitive; for example if you suddenly need to make an ø you would hit Compose+o-/ and there you have it!  :)

Check out this document on how to set that up:  [ComposeKey]("https://help.ubuntu.com/community/ComposeKey")

### Post by mcduck on 2009-02-07
yther already answered that.

You are not going to be able to use Alt-<random number code> in Ubuntu. In Linux special characters are typed either with AltGr key or by using composite key. (Actually many languages use AltGr for this even in Windows).

What keyboard layout are you using? Some layouts use extended character directly by pressing AltGr+letter, others use composite key to combine a letter with character to form the final extended character.

Also if you hae the umlaut/accent keys in your keyboard (usually keys that have ^, ¨, ~, ´ and ` printed on them) you can simply select the umlaut/accent you want by pressing that key, and then press the letter you want to have the umlaut.
