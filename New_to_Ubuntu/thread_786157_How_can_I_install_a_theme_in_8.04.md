I just figured this out myself.

YOu have to unpackage a tar file so first open a terminal.

In terminal:

tar xjvf *file*.tar.bz2

After you unpackage the file save it somewhere. I made an new folder called 'themes' and saved it in my home folder.

Then go to System-> Preferences->Appearence and click install.

Browse for the folder you just created and  click it. Your theme should change a few seconds after that.
