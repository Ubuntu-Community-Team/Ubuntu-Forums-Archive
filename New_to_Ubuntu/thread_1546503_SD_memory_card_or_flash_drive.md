### Post by Macskeeball on 2010-08-05
In a word, convenience.

All flash drives have built-in USB connectors, and many can be attached to a keyring. In other words, you'd always have it when needed and when those situations arise you will be able to use it with whatever computer you happen to encounter in that situation. No need to also carry an SD card reader with you for computers that don't already have one. No need to plan ahead. When you need it, you will have it and be able to use it.

Unfortunately, most flash drives use tiny little keychain rings that break and can't be replaced (not sold separately anywhere as far as I can tell), forcing you to buy a whole new flash drive just to get a new keychain ring. That happened to me a lot, so I ended up getting a [LaCie iamaKey](http://www.lacie.com/us/products/product.htm?pid=11225) which (due to its design) does not have that problem.

### Post by Macskeeball on 2010-08-08
> **ironic.demise said:**
> Aren't Memory sticks also cheaper than Memory cards?
For a larger capacity the price could be quite a jump...

Without looking into it, it seems to me that all things being equal the memory cards should be slightly cheaper (less parts, standardized design). Now, there are different speeds of flash memory available, and higher speed flash memory tends to cost more. You might be comparing the prices of *high speed* flash memory with the prices of USB flash drives using *slower speed* flash memory. 

Flash memory cards are often used in digital cameras. In that situation, the speed of the flash memory matters a lot more because it affects the camera's ability to fire multiple shots in rapid succession. Photographers using slow flash memory cards could potentially miss what would have been a great shut, simply because the camera took too long writing the previous shot to disk.

Again, when I say that you may have been comparing different speeds of flash memory (slow USB flash drive, fast card), I'm just guessing. I haven't taken the time to look into it.
