### Post by ron999 on 2010-05-27
Hi
Look down in that .macromedia folder to find a folder named **#Shared Objects**.
That's the folder to delete. It will create a new folder automatically when needed.
There's also a Firefox addon called **BetterPrivacy**. This lets you control those flash supercookies. You can set it to flush them away each time Firefox closes.
