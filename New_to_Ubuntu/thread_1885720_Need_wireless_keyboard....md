### Post by kurt18947 on 2011-11-23
SWMBO has a Logitech wireless keyboard that works as advertised.  Her wireless keyboard doesn't have numlock & capslock indicators.  If you get a wireless keyboard and it doesn't have a way to indicate key lock state, you can download and install this:
[http://ppa.launchpad.net/tsbarnes/indicator-keylock-daily/ubuntu/pool/main/i/indicator-keylock/](http://ppa.launchpad.net/tsbarnes/indicator-keylock-daily/ubuntu/pool/main/i/indicator-keylock/)
I have my preferences set with "use an application indicator" and "show indicator while lock key is inactive" checked.  You can set a ppa but I just downloaded and installed the .deb file.  It seems to work as expected with Unity, Gnome 3 shell & Xfce-desktop.

Xfce-desktop is imperfect.  It works .... sorta .... but not 100%.  The icon changes from black to gray with changes in capslock or numlock state, the text does not change.  Works as advertized in Unity & Gnome 3-shell.
