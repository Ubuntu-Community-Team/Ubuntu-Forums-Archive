If the automatic configuration does not work, you can try manually editing
/etc/X11/xorg.conf

That's where all your graphics configuration is. Be careful though, I don't claim it is easy. You may want to backup your current copy before you make changes.

If you look inside that file, you should see something like
Section "Screen"
        Identifier      "Default Screen"
        Device          "Default Device"
        Monitor         "Generic Monitor"
        DefaultDepth    16
        SubSection "Display"
                Modes           "1024x768"
        EndSubSection
EndSection

But it probably says something else especially in the Device section.
