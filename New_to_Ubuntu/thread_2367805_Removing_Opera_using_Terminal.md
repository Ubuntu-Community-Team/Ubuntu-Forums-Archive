### Post by zahira+ on 2017-08-03
The instructions included in the previously closed post were not complete. Please see these instructions for current information on Opera Removal.

1. Open Terminal
2. write - sudo apt-get remove Opera <enter> (you will see a list of all of the files to be removed. This does not remove them.)
3. Enter password if applicable <enter>
4. write - sudo apt-get autoremove opera <enter>
