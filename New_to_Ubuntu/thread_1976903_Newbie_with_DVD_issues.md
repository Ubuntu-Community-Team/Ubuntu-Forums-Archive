### Post by jockyburns on 2012-05-09
Go to the start tab (top right of screen) From the drop down menu select System settings. When the next window opens up select Disk Utilities. If your DVD drive is installed correctly it should show up in there. 
Unlike Windows, if there's nothing in your dvd/cd drive, then it won't show, until you insert a disk.
