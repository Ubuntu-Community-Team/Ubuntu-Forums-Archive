### Post by grahammechanical on 2011-03-19
You are still in the Microsoft mental conditioning where every upgrade to the windows operating system requires ever faster, and large hardware.

Linux is not like that. We do not have to wait for years for a new version of Windows or a service pack for vulnerabilities to be fixed. As soon as a vulnerability is discovered work is started on fixing it. And as soon as a fix is put in place we get the opportunity to apply the fix to our machines. We can always be up to date with the core components of the operating system.

It is likewise when programs are updated or improved. We do not have to buy the next release of the program in order to get an improved program.

Ubuntu is distributed by a company called Canonical. It has been the policy of the Ubuntu development team to distribute a revised edition of the distribution every six months. This policy has meant fast development of Ubuntu. I have noticed a big improvement since 2007.

I have upgraded Ubuntu eight times since I first installed Ubuntu. I have not needed to upgrade my hardware at all. I intend to upgrade to the next version in a couple of months. I am confident that my hardware will be sufficient.

There may come a time in the future when my hardware may not be able to run the very latest version of Ubuntu. Just as my old computer could only run Windows 98 and even then it struggled.

Then I may buy a new machine or continue using this machine with the Ubuntu version that works the best on it.

I suggest that you load Update Manager and click on Settings and see what options you have.

Regards.

Simple.

Open update manager. Click on settings bottom left. Enter password.

See the drop down Release upgrade. Choose Never.

You will now continue to get regular security and recommended updates for your current release but it wont offer to upgrade to 11.04 when it comes out.

Just be aware when the support runs out.

Ah, so your question wasn't if you can upgrade, but if you should upgrade.

If you don't need the newest software, you shouldn't upgrade and I think going from LTS to LTS is the best for you because a non-LTS version is only supported for 18 motnths.
