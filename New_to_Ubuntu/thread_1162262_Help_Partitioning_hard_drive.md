### Post by tuxxy on 2009-05-17
Run Ubuntu Live CD and use the graphical partition editor, this will allow you to resize your current partitions where your data is stored. Resizing the partitions will then free up some space which you then will use to create your Ubuntu partition.

[https://help.ubuntu.com/community/HowtoPartition](https://help.ubuntu.com/community/HowtoPartition)
