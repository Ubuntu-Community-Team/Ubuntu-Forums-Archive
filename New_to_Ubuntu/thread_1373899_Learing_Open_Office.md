### Post by Zimmer on 2010-01-06
[http://openoffice.blogs.com/openoffice/calc/](http://openoffice.blogs.com/openoffice/calc/)

Bits and bobs there, but if you want to learn about spreadsheets generally then I would suggest you consider creating a sheet to 'assist' with a minor task and embellish the function of the sheet with 'handy' tools.

For example,a shopping list.
Fill a column with articles you normally buy, foodstuffs etc.
another column with the aisle number they may be located in the supermarket, a column for last known price.. a quantity column, a 'required this week' column.

Make it your plan for the sheet to tell you how much this week's shopping will cost.
Sort the list in aisle order etc..

and maybe a last column that will show the item description only if you have it selected to purchase this week.... or changing the cell colour automatically if it is chosen.

There are more things that can be done, using Data analysis on the resulting sheet, graphs etc.
By having a specific project or need to work on then you have the inspiration to learn what spreadsheets are capable of. 

I learned much by bosses saying to me ' I want a report to show this column changing to green if target x has been achieved and red if it has not....... etc. ' by just reading the Help section... and searching for a solution... not through courses or tutuorials..

### Post by SecretCode on 2010-01-06
> **daniell59 said:**
> an "easy" way for me to learn this. A very basic step by step tutorial might help.

I'm not sure that what you think is "easy" now is really what you need. The thing is, Calc - and Excel and Writer and Word and most office tools - don't do just one thing that you do "step by step". After a very basic tutorial you'll probably still think "so what?"

To learn, you need to have something you want to do. Do you ever need to add up lists of numbers, but would like to keep the list and change some of the numbers? (Monthly expenses for example.) Do you ever need to format a table that includes text and numbers and some calculations? (A quote for a client for example.) Those kinds of things are perfect for spreadsheets.

Pick one and just jump in. Use the help within openoffice. When you're stuck with a specific issue, google it, ask here, ask a the openoffice forums. Pretty soon you'll be comfortable.
