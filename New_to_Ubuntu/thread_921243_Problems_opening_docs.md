### Post by moody_mark on 2008-09-16
The above will work, but may not change the preferred application I dont think

If you are using the gnome desktop, then try this;

1. Right click the file and select "properties"
2. Select the tab "Open With"
3. Select the correct application, or click the "add" button to select the one you would rather use.
4. Make sure the radio button is selected for the correct application in the "open with" list
5. Click "close"
6. Double click the application again to launch it
