Well you are probably better off just going to a torrent site and downloading the torrent, then opening it up in deluge. If you want to do it in deluge though, there is a search bar on the top right corner that should be next to the binoculars image. First you have to click the arrow pointing down and click manage engines. Add the name and url of the torrent site and then click ok. Add as many as you want then click the arrow again and select whichever you want then just type in what you want to search for and press enter. That will bring you to the torrent site search results. So as I said before you are probably better off just going straight to the site.
To uninstall Frostwire open a terminal and type 
```
 sudo apt-get remove frostwire
``` to uninstall anything you just do that and replace the name. I would advise you to keep it because it is very useful to install small files that you don't need too many seeders for unless you are pressed for space. In that case also type ```
 sudo apt-get autoremove 
``` which will uninstall any program frostwire may have installed by searching for anything you computer has that no program currently installed uses.
