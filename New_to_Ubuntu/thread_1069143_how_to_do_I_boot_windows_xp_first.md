### Post by howefield on 2009-02-13
Sorry, from Ubuntu.

Load up Synaptic Package Manager from the System > Administration menu and search for startupmanager, click and select "mark for installation" then apply.

You will find a new entry in your System > Administration menu called Start-Up Manager.

You can use this to edit the grub settings which is what is giving you the choice of system to boot into.
