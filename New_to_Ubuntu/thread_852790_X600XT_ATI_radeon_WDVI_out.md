### Post by Sirron on 2008-07-08
Sorry if you know this but just in case;

The driver provided in ubuntu by default for AMD (ATI) cards is open source, which is good, but it provides no acceleration, which is bad. For acceleration you'll need the proprietary driver, which can easily be installed via the "Hardware Drivers" tool on the Administration menu. Alternatively, you can try [Envy]("http://albertomilone.com/nvidia_scripts1.html") which is a little less straightforward, but is still very easy.

I'm assuming it will work since my laptop uses an x600 card and it works for that - yours can't be that much different right?

EDIT: feel free to call me a patronising time waster ^^ but there's a chance it'll help
