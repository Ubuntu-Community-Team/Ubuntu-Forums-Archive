### Post by semjaza on 2009-07-27
Well. . . sometimes when I can't get stopmotion (the program) to work I use this tcl/tk script I wrote called mystop.  It's very simple and is feature light.

It uses tvtime to take screen shots.  You can either hit the big "take shot" button or use the timed button.

There is also a button that makes a movie using mencoder.

So you'll need to

sudo apt-get install tcl tk mencoder tvtime 

in order to use it.

This is a very early version and it doesn't have much in the way of adjusting setting other than editing the script, but editing is easy since it's text and tcl/tk.

You can get it from here [http://bit.ly/14ykV0](http://bit.ly/14ykV0) 

Let me know if it's useful to you.
