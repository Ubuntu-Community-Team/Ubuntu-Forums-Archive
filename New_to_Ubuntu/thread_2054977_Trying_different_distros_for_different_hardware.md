### Post by The Cog on 2012-09-08
All the different desktop types are based off the same Ubuntu core, so I don't think trying X/K/Lubuntu will make any difference. Going back and forward in time (from the latest to the previous LTS version for example) might make a difference. Trying different distros like Fedora, pclinuxos etc. might get lucky. But I don't hold too much hope - in general the drivers for these things are built into the Linux kernel which all the distros are based on, so they all tend to have pretty-much  the same hardware support.
