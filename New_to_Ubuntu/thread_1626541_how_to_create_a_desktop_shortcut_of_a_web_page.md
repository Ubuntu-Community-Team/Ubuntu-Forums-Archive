### Post by Verbeck on 2010-11-20
you could create a launcher for it in your desktop

Right Click on the desktop, then select 'create launcher' 
select 'Location' as a type from the drop down menu
type the full url as the location eg. [http://ubuntuforums.org](http://ubuntuforums.org)


dunno how its done from within forefox
