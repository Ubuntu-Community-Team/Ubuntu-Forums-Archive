### Post by danbuter on 2008-08-18
Put a . in front of the file name.

---

### Post by alecz20 on 2008-08-18
Note however that any references to that file will have to include the . in front of it.

With Nautilus you can create a file ".hidden" in which you can put the names and folders that you want hidden without adding . in front.

Apps such as gnome-commander will still those files/folders as non-hidden.

---

### Post by john test on 2008-08-18
I think using a period as a prefix to the filename will make it hidden.  as, (mv filename .filename)
