### Post by gtdaqua on 2008-06-24
Axel is a CLI-based download accelerator.

```

sudo apt-get install axel

```

Syntax example:

```

axel -n 10 -a http://servername.extension/downloadfile.prg

```

-n 10: That will throw 10 connections (if your bandwidth is big) to download that file. There are other parameters too.

-a: alternate progress indicator - displays the progress indicator in one line.
