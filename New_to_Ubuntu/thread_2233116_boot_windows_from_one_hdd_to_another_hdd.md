### Post by kansasnoob on 2014-07-06
If you installed Ubuntu on a separate hard drive with the Windows drive disconnected, and then reconnected the Windows drive after installing Ubuntu, you should be able to have Windows appear in the grub menu at boot simply by booting into Ubuntu and  running the command:

```
sudo update-grub
```
