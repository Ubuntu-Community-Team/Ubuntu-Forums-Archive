### Post by Tibuda on 2009-03-20
I use Gedit, with some tweaking. I'm very comfortable with it. With the "external tools", it gets really powerful, with HTML Tidy validation, PHP syntax checker, and version control.

If you want a full IDE, try Aptana. It has plugins for many web languages as PHP and Ruby on Rails.

If you want WYSIWYG, your options are Amaya, Kompozer and Quanta Plus. I can't tell you if they are good, as I prefer to write my own XHTML/CSS for control and flexibility. If you are worried about web standards, try Amaya, because it was created by W3C.
