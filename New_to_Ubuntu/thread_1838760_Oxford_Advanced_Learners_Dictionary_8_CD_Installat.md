This is difficult to answer without knowing what's on the CD. I presume you have read the CD's installation instructions?

Installing programs in Linux takes place in one of two ways. The easy way (which Ubuntu supports) is through a package manager. The hard way is to download the source code and compile it.

Search your CD to see if you can find any file with the extension .deb or .rpm. Let us know what you find, and with luck we can take it from there.
