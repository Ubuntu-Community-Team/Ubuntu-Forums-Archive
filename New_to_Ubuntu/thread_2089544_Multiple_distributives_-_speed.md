If I install a distro, and then from that distro, install another one, will it affect the speed/efficiency of the computer more than if I use the same distro I installed?

### Post by snowpine on 2012-11-29
You can install multiple Desktop Environments (unity, kde, lxde, xfce, etc.) on Ubuntu and switch between them depending on your needs that day. There shouldn't be any performance penalties to doing so, in my experience.

Here is one such tutorial: [http://www.psychocats.net/ubuntu/xfce](http://www.psychocats.net/ubuntu/xfce)
