### Post by Segofam on 2012-09-15
A few people have asked me about HandBrake, which I use everytime I buy a DVD to backup my DVD's, as my Children have damaged so many in the past.

For the Newbies, as I was, put the below lines into Terminal, one at a time,you can open by pushing keys CTRL+ALT+T

```
sudo add-apt-repository ppa:stebbins/handbrake-releases
sudo apt-get update
sudo apt-get install handbrake-gtk
```

For more information:
[https://launchpad.net/~stebbins/+archive/handbrake-releases](https://launchpad.net/~stebbins/+archive/handbrake-releases)

SMPlayer from the repository will play the MKV file, and many others :)
