### Post by lkraemer on 2009-11-25
Balf,
If Dbase Plus is a DOS program, you most likely can run it in DOSBOX.
Just install DOSBOX and give it a test.

If it is a Windows program, then there is a good chance it will install
and run in WINE or Crossover by Codeweavers.  You can check on the various
applications that install, and run at those websites applications databases.
he real test is to just give it a try, because some software runs and is
not in their Database.

If nothing else works you can always install VirtualBox, and then install
Windows in VirtualBox (as a Virtual Disk Image) and run your Dbase software
from there.  You will need 2 Gig RAM and a little time to get VirtualBox
setup, but that doesn't take long.

Next thing you know you will be free of Windows.....

If you have questions.....ask!
