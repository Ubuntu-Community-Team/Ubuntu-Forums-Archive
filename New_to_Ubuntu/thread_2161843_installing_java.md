### Post by claracc on 2013-07-12
Ubuntu 10.04 is an end of life support release. You cannot install anything because the repositories have been moved. It is a security risk to run an eol support release, also, as you can see a problem to install software.

I recommend you install a supported ubuntu as 12.04, 12.10 or 13.04

Anycase, if you want to continue (at your own risk) with 10.04, you can install software in the way pointed in this link:[http://askubuntu.com/questions/91815/how-to-install-software-or-upgrade-from-old-unsupported-release](http://askubuntu.com/questions/91815/how-to-install-software-or-upgrade-from-old-unsupported-release)
