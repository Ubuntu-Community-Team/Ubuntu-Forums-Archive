### Post by cascade9 on 2009-11-28
Linux is the kernel. Ubuntu is a OS, using the linux kernel, and its just one of many 'distributions' (think of them like different version) using the linux kernel. 

As for whats a 'kernel', here a quick version of what that is. With a link in case you want to know more ;) 

> In [computing]("http://en.wikipedia.org/wiki/Computing"), the 'kernel' is the central component of most computer [operating systems]("http://en.wikipedia.org/wiki/Operating_system"); it can be thought of as the bridge between applications and the actual data processing done at the hardware level. ### Post by Some Penguin on 2009-11-28
Linux is a really a set of operating systems based upon various versions of one particular operating system kernel and a number of standards.  Ubuntu is a specific instance or version; it has made various decisions (such as what packages are 'standard' and what others are optional but considered officially supported, how they are packaged, which files go where and what the initial configurations are, how to install it, et cetera) which may be rather similar to but aren't necessarily the same as other Linux distributions.

### Post by wojox on 2009-11-28
Linux is the generic name for operating systems that are based on an open source copy of UNIX which was developed by Bell Labs.
Linux has various "flavors", including the free ubuntu.
While there are many different forms of Linux, If you are new I would recommend Ubuntu, as it is very easy for windows users to learn their way around, also it is very popular right now, so there is a lot of support to be found.
