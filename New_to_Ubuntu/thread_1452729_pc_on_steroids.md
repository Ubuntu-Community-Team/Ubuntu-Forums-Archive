### Post by sydbat on 2010-04-12
I'd go with the wonky keyboard idea too. If you can, try and swap it out. If the problem persists, try loading in a LiveCD and running rkhunter and/or chkrootkit to see if something actually did get into your Ubuntu install.

Alternately, try booting into Windows 7 and see if the problem replicates itself there. If so, first guess is keyboard, second is some kind of malware.
