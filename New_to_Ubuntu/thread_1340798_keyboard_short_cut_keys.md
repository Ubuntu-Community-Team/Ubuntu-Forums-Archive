### Post by stinkeye on 2009-11-29
The plugin is part of the gnome-do package .
xkill is a command which can be run in terminal.
It brings up an x which you can click on the window you wish to kill.
Gnome-do can also run terminal commands.
But if your having that sort of freeze up you probably won't be able to access either.
If you keyboard still works and your using compiz you could bind xkill to a key(eg ctrl+alt+x)
in ccsm under commands.
