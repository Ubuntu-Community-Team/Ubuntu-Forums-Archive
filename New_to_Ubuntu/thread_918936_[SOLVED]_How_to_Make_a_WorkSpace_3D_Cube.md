### Post by Ub1476 on 2008-09-13
Yes, it's very easy :)

First, make sure desktop effects is enabled and working. Then install "advanced desktop effects" from add/remove. Open it from System>Preferences and check the "Cube" plugins. Set the workspace to have 4 desktops horizontal. 

Now you can press ctrl+alt+left/right key to switch desktop. Press ctrl+alt+down key to unfold cube.

Press ctrl+alt+mouse button 1(left) to view it.
