### Post by migs73 on 2011-06-04
If you have the standard unity interface for 11.04 the power management setting can be found by clicking on the power icon in the top right then click on system settings. In the window that opens power management is in the Hardware section.
Here set the computer to go to sleep as 'Never' and set the display to go to sleep as 'Never' as well.
