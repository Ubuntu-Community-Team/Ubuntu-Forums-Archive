### Post by lovinglinux on 2009-03-26
I guess you think that just downloading Ubuntu would affect your Vista installation and that Ubuntu is the culprit for the failure. That's not the case. The problem is inside Vista.

If you have access to another computer, download the Ubuntu iso file again, burn it to a CD and boot from it on the machine with problems. Then, you can use the [Live CD]("https://help.ubuntu.com/community/LiveCD") to run Ubuntu without installing it (it won't affect your system).  If the Live CD works fine, then you can install by clicking the install icon in the desktop.

Here is a guide to how install Ubuntu

[https://help.ubuntu.com/community/GraphicalInstall](https://help.ubuntu.com/community/GraphicalInstall)
