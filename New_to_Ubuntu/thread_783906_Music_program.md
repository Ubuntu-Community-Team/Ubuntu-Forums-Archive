I really like amarok. Install it by...

Its the best music player for linux (arguably). I like it since its simple. Others like it because its fast. Still others like it because it has a music store built in. The other recommendation is rhythmbox. I don't like it as much but it comes by default. The great thing about linux is that it never hurts to try new stuff! Its free and especially on ubuntu its easy to add and remove programs!

### Post by BaffledMollusc on 2008-05-06
I use Amarok. You should be able to find it in the repositories (System->Administration->Synaptic Package Manager, and search for "Amarok").

You can point it at a top-level directory and it will import all music files under that directory recursively, and keep track of any new music you add. The music is indexed by artist, album etc. Very nice.

Edit: Maybe I misunderstood. Are you asking for a music player recommendation or a way to copy music from one computer (Win XP) to a different computer (Ubuntu)?

### Post by mlentink on 2008-05-06
Each to his/her taste, I use Banshee. Try them all out and keep the one you like, that's what I did.
Most of the players in Ubuntu will play all formats. Banshee plays my .wma files perfectly. 

So basically the only thing you need to do is to copy your music files over (cd, dvd, memory-stick, network, whatever), point you music-player to that directory and have fun...


Edit: you may have a problem with DRM-protected music-files though...
