### Post by balaknair on 2009-03-05
The menu bar at the bottom is a dock- Avant Window Navigator, and you can install it via Synaptic(search for avant in synaptic).

The Dark Ice theme you linked the screenshot of above is available as both an Emerald theme and a GTK theme( [http://www.gnome-look.org/content/show.php/Dark+Ice?content=69886](http://www.gnome-look.org/content/show.php/Dark+Ice?content=69886) ).
To use the Emerald theme, you need to install Emerald(again via Synaptic).
