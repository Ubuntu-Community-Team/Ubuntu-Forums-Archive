### Post by ajgreeny on 2012-06-26
I suggest you install **openjdk-7-jre** and the same version **icedtea-plugin** and see how that works on your sites that need java.

In the past I used to install the sun-java (now Oracle-java) packages as some sites did not work with openjdk but that difference has now gone as far as I can see, and I haven't had a problem with openjdk at all on any sites.
