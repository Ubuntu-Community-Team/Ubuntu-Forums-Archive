### Post by svee on 2012-11-14
If I have my HDMI cable connected to TV and enabled dual display; on reboot I don't get login screen. Screen seem to freeze. 

When I disconnect HDMI cable, and hard boot it seem to boot properly. 

Note that after I bring up laptop, I am able to connect HDMI cable and use the other monitor on my TV.


I have Sony VAIO VPCYB15AH laptop with AMD E-350 processor. Using 64 bit version of lubuntu 12.10. 

Curious if anyone else faced this issue ?


Otherwise I am able to bring up lubuntu successfully; but yet to get audio working  (unlike ubuntu 12.10 or 12.04.01 LTS versions in which screen response is poor and screen freezes completely).

### Post by mag1strate on 2012-11-14
Do you have the ATI Proprietary drivers installed? If not, go to the "Additional Drivers" section of the system settings icon to install them. It should work after that.

---

### Post by svee on 2012-11-15
Thank you.. Installing ATI Proprietary drivers solved the problem. 

To install drivers lubuntu however, I had to go through System Tools/Software Updater/Settings/Additional Drivers
