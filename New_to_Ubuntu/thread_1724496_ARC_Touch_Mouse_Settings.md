### Post by pocklecod on 2011-04-08
I'm running a System 76 starling netbook.  This afternoon, I was doing some shopping to spend a gift card and saw this Microsoft ARC Touch mouse.  I hesitated a long time at the thought of buying from the Evil Empire - but the mouse is SUPER portable, very sleek, and is ideal as an external mouse for my netbook.  So I bought it.

The thing is working great right out of the box, running Ubuntu 10.04 (netbook edition).  Only one issue - the mouse makes a goofy artificial clicking noise whenever I scroll (trying to emulate a physical scroll-wheel I guess).  Anyway, I don't like the noise, and I was wondering if anyone had managed to alter the settings for this mouse via ubuntu.  Microsoft has some software for the mouse which I'd assume would allow one to do this - but, naturally, that software only works in Windows.

If anyone has managed to fix this, I'd very much appreciate the help.  I'm not that new to Ubuntu, but I'm not a power user by any stretch either.
