### Post by Suparious on 2010-11-30
If your setup was working on other networks, then more-than most-likely it is not a software/driver/security issue.

Can you ping your wireless router when you are connected?

When you're on your unsecured network, try to do an ```
arp -a
``` and see if you can find the MAC of your wireless router.

This will also help you confirm your own MAC, incase it is incorrect on your wireless router.

I have used all type of wireless, with various cards, routers and setups, and have never encountered this issue. Usually it is the secured networks that I usually have troubles with.

### Post by cariboo on 2010-12-01
It sounds like you have a dns problem. Can you ping Google by name:

```
ping www.google.com
``` 

if not try by ip address:

```
ping 72.14.213.106
```

If you can ping by ip address, but not by name, paste the output of:

```
cat /etc/resolv.conf
```

You may have to manually enter the dns addresses in network-manager.
