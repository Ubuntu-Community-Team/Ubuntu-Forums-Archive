### Post by Rob Sayer on 2014-11-06
AFAIK the only way you can install utorrent in ubuntu is via a 3rd party ppa.  I.e. it's untrusted and not supported.  And I don't think utorrent's linux support is all that great either.

I very rarely install from 3rd party ppas unless it does something I can't do with an app from the ubuntu repos.  I can't remember any examples.

Linux novices in particular should avoid software from untrusted ppa's.  They can cause problems in your system and some of them even contain malware.  Linux is very stable but you can destabilize it, and software from untested ppas is a good way to do it if you don't *really* know what you're doing.

Frankly I never thought utorrent was all that great anyway.  Transmission is definitely better.
