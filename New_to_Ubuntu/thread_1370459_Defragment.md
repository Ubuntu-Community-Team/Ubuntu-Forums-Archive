### Post by iponeverything on 2010-01-02
ext3 and ext4 are very resistant to fragmentation and as a result I don't think defragers even exist for them. When data is written to disk, the system intentionally spaces it out so that adjacent blocks are available to grow a file if needed.  

Indeed fragmentation can occur. But the case where this usually happens is not a common one, so I don't think anything is being done to address it..
