### Post by kellemes on 2008-08-27
For those interested.. I got it working!

I have a x1650 AGP 512
I had to lower my motherboard's AGP Aperture Size to 128 (through BIOS), my motherboard only supports up to 256, so I cannot set it 512.

I added the following line to the "devices section" of my xorg.conf
```
Option      "AGPv3Mask" "0x00000002"
```

Whatever whay I choose to install the ati-driver, it simply works.
