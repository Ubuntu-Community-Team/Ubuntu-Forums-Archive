### Post by Darkwing-Duck on 2009-09-12
I run Aspire One AOD150. I installed UNR for the interface as well. The one thing that I noticed out of the box with it is the resolution fix. When I ran the live CD of 9.04 the resolution was messed up. If you install the UNR all you have to do it go to Prefrences and click on Switch Desktop Mode. 

This way you get the little things that the UNR comes with along with the standard desktop features. 

Hope this helps.
