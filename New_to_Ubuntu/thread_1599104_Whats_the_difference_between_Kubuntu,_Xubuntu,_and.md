### Post by howefield on 2010-10-17
> **Jetso said:**
> Oh... So how come Xubuntu has a picture on their website of hat looks like a Gnome Desktop?

Looks like, but isn't...

> Can you install the environments inside of Ubuntu?

Yes, you can install xubuntu-desktop or kubuntu-desktop or ubuntu-desktop (as well as several others) and choose which to use from your session login screen.

One "disadvantage" might be that you will have mixed menus, eg,  If you install the KDE desktop into an Ubuntu installation, you will have KDE and Gnome applications in the menu, they will function just fine.
