### Post by coffeecat on 2014-07-23
You are wasting your time with old versions of Ubuntu. If it is the 2004 release, it would be 4.10. Early versions did not support NTFS read-write and that model of Compaq has Vista on it, which would be installed on a NTFS filesystem. Even if you got Ubuntu 4.10 to read the NTFS partition, you would not be able to move or delete anything. 

You would be far better off using the live CDs that some antivirus companies offer free. You'd need to do some googling to find out which ones are available currently, but I used some a few years ago for a friend's XP machine. If I remember correctly, they come with a database of virus signatures, but if you could connect to the internet in the live session, they can scan for a more comprehensive set. And, more importantly, they can delete the rogue stuff. It sounds like you won't be able to connect to the internet unless you can connect wirelessly in the live session, but at least with a sample of different AV CDs you might be able to deal with the problem.

**EDIT**: Here's a useful link:

[http://www.livecdlist.com/purpose/windows-antivirus](http://www.livecdlist.com/purpose/windows-antivirus)

Although most of the last release days are somewhat old, if you follow the links you'll end up with more recent versions that the list suggests. At least that seems to be the case for Avira and AVG. I haven't followed any of the others.
