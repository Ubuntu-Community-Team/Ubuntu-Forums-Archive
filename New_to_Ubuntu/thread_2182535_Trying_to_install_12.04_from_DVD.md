### Post by Steve_Cline on 2013-10-21
I downloaded and installed 12.04 from DVD on my kids' desktop to give it a try.  I would like to also install it as a standalone OS on my HP laptop but all I have been able to do is install it along side Windows.  The installation will not automatically boot when I start up the laptop.  Any ideas?

---

### Post by newb85 on 2013-10-21
Your BIOS is probably set to boot from the HDD ahead of the DVD.  You need to get into the BIOS and change the boot order.

### Post by BBQdave on 2013-10-21
> **Steve_Cline said:**
> 12.04...  I would like to also install it as a standalone OS on my HP laptop but all I have been able to do is install it along side Windows.  The installation will not automatically boot when I start up the laptop.  Any ideas?

Depending on hardware, if your HP is 64 bit - newer hardware a couple years old, you will want **Ubuntu Unity 12.04LTS AMD 64**. At start up, you can strike F12 to select Ubuntu disc.

In the install choices, you will want to select *entire disk* for Ubuntu. [COLOR=#ff0000]**This will erase all data on your HP**[/COLOR], so backup data to a seperate drive - that you want to keep.[COLOR=#ff0000][/COLOR][COLOR=#ff0000][/COLOR]

Installing Ubuntu to entire disk of your HP - Ubuntu will be the only system, so should boot up straight away upon start :)
