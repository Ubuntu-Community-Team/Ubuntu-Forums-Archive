### Post by mkstallings1 on 2012-08-15
With 2GB RAM, you should be able to use whatever buntu you want.  As far as performance goes, if your wanting Ubuntu for the eye candy, that can depend a lot on your graphics hardware.  I have a 7 year old laptop with an AMD Turion processor, ATI Radeon graphics card, and 2 GB RAM, and I run Ubuntu 12.04 LTS with no problem.  I have a desktop with an Intel Celeron processor, unsupported graphics card, and 750 MB of RAM and it ran Ubuntu 12.04 LTS with Unity 2D a bit too slow for my taste.  It was very usable, but now I run Peppermint OS which is based off Lubuntu and everything is very snappy.
