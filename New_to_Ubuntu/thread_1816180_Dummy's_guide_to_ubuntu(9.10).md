### Post by 3rdalbum on 2011-08-02
Firstly, you NEED to update to a newer version of Ubuntu. Version 9.10 is End Of Life, you can't get software for it anymore.

Either use 10.04 which will be supported for another two years, or 11.04 which is the latest version (with the latest software) and which will be supported for nearly 18 months.

When you've got a later version of Ubuntu, you need to get out of the Windows mindset a little bit. On Linux you don't look on the Internet for software, and then download it from a web page and double-click it. Instead, you use your Linux distribution's "package manager" which allows you to search, download and install Linux software that it knows about.

On Ubuntu, the easy package manager is called Ubuntu Software Center. Use this to find software. It does contain Wine, and it contains Firefox (which is already installed on Ubuntu anyway). It contains so much Linux software that you'll never even install 10% of it.

When you want to install software that's not in the Software Center, you can usually find a "repository" or "PPA" that contains the software. Once you tell Ubuntu Software Center about the repository, you can download and install the software in it like you would any other software on Linux.

But first, you need a later version of Ubuntu - 9.10 is too old, it's practically ancient in Linux terms. Use AT LEAST 10.04, but preferably 11.04.
