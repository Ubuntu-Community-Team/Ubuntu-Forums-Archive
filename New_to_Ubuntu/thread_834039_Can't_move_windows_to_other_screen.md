### Post by Zenze on 2008-06-19
I believe that in order to be able to drag windows to other desktop you need to have visual effect set to at least normal. To do this go to System > Preferances > Appearance. Then click visual effect and select normal. You may also be able to configure it with the compiz configuration tool.
