### Post by sanderd17 on 2011-11-13
> **Agent26 said:**
> Wow there sure is lots to chose from.
I actually want to get into linux first on my desktop and then when i know it a bit better change my lappy to.
 
do you think if I risk lubunto it might keep freezing, thing is xubuntu was nippy but just keept geting stuck.

The problem is really your memory.

RAM is very fast memory, but when it's full, it's extended with disk space. A HDD is very slow compared to RAM.

So when you want to use something that has moved from RAM to HDD, it will take a while for it to appear (because first the OS has to free some RAM by writing to the HDD, and after that you need to fetch the info from the HDD).

So what you want is an OS that uses a minimal amount of RAM. Off coarse, the less RAM you use, the less functionality it can  have.
