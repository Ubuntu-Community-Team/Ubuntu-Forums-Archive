### Post by Kevbert on 2008-05-10
> **sharks said:**
> I have hardy and can i have gutsy alongside? I want to use both of these. Can i dual boot them?

Yes, I multiboot Hardy with Gutsy and Fedora with no problems. It's just another operating system.  It's probably best to perform manual partitioning when installing the second OS.  Which ever you install second will be the default operating system to boot on power-up.  If you want to boot the second just edit your menu.lst file.  Change 'default 0' to what ever you want (default 3 - second OS counting from 0).
