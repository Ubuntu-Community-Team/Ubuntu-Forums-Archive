### Post by avtolle on 2009-03-24
DillByrne's post makes the most sense, provided the responding poster has exactly the same set of sound card, video card, network card as does the OP.

What he appears to be asking is how can he be sure the "drivers" will work without downoading the Live CD in the first place. If the OP is referring to the Windows drivers, they won't work in linux natively, and may in fact not even be needed. Whether these devices will work within Linux and with the drivers supplied by Ubuntu in particular cannot be definitively anwered without installing the OS, IMO. If the OP has sufficient RAM for the video, etc., drivers to be downloaded and installed in RAM while running from the Live CD, this will give him a good hint that these devices will work once installed; but, from reading many threads on the Forum over time, it is still not an iron clad guarantee these will work "out of the box" after installation. For example: installing the graphics driver, if one is needed. Sometimes, one only needs to log out and back in for the driver to be installed; that will work in a live session. However, if a complete restart of the computer is needed, with the driver in RAM, that will clear out the driver (and everything else in RAM), and the OP will not know if the driver will work pre-installation.

Long winded response, and I regret it. The short answer to the question posed is "one cannot know".
