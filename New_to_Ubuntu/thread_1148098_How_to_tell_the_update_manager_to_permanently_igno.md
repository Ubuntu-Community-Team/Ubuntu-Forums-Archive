### Post by lethalfang on 2009-05-04
I've installed an older version of Kile (a LaTeX editor) over the one that comes with Jaunty due to severe stability issues with the new version. The annoyance now is that the update manager keeps telling me to update Kile.
How do I tell the update manager to ignore just that one source file?

Thanks

---

### Post by iaculallad on 2009-05-04
System->Administration->Synaptics Package Manager. Click on the package then select "Lock Version" on the Package menu.
