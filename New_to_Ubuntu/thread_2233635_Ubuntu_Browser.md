### Post by coffeecat on 2014-07-09
I don't know what you mean by Ubuntu browser, but if in Firefox you mean using the backspace key to go to the previous page, it is enabled by default in Windows versions of Firefox but not Linux ones for reasons which have always mystified me. This is easily fixed. Type "about:config" (without quotes) in the address bar. Dismiss the silly here be dragons warning. Then type "back" in the search bar. About 6th in the list you'll see browser.backspace_action. Double-click on that line and change the 2 to a 0.

### Post by grahammechanical on 2014-07-09
There is a Ubuntu Web Browser simply called Browser. It is part of the default 14.04 install. Its origins are from the Ubuntu phone. So, we can use the mouse to do swipe gestures. Take the mouse to the bottom of the browser window and swipe upwards and a panel will appear. That has a back icon as well as a panel for entering web addresses and a icon for going to the activities page/screen where we can set bookmarks and load previously visited sites.

The developers are working to make Browser more usable  on the desktop. It now has sliders so that we do not have to use the mouse to swipe gesture up or down to see more or less of the page. I got used to doing that. But ever since Browser received those sliders it freezes the desktop. I am using Utopic Unicorn. Over the past few weeks Utopic has received 3 or 4 updates to Browser but the problem still remains. But I live in hope that Browser will progress into a useful if limited application.
