### Post by laidback on 2009-12-12
Well I've found a solution. Moved to Glabels but I suggest that you don't use the version that's in the repositories but get v2.2.5 or 2.2.6 from:-

[http://sourceforge.net/projects/glabels/files/](http://sourceforge.net/projects/glabels/files/)

I'm using 2.2.5 with Ubuntu 9.04 which appears to work for me, certainly far better than the one in the repositories. When you download check that you get gLabels and gLabels-data, you need both. Use the installer on gLabels-data first, then gLabels, otherwisw you'll get a dependency error.

I didn't notice v2.2.6 when I downloaded, but I would have used that if it had been available at the time.

I'm hoping that this info will be useful to someone else.

[http://old.getdeb.net/release/4336](http://old.getdeb.net/release/4336)
