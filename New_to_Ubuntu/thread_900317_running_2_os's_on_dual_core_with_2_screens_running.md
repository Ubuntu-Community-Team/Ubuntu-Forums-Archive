I don't think this is possible. It is possible with Linux and Windows, as they can both be run in Virtual Machines- for this, you could boot into Windows or Ubuntu, then run the other OS on a different physical screen, in a Virtual machine.
Currently there are no VM Applications (that I know of) that can run the Mac OS or osx86.
