### Post by Dennis N on 2023-11-01
There is a gnome-shell extension named "Favourites in AppGrid" which does what you want.  I am using Gnome Shell 43, so you will have to check and see it is also available in Gnome shell 42.

Edit: Extensions Manager says it is available in 42, so you are all set.
