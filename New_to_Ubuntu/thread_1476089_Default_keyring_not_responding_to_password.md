### Post by truker on 2010-05-08
Actually, I found a way to stop it from asking you for the "default" keyring password.

Go to Applications (upper left hand corner of the top panel), Accessories, Passwords and Encryption Keys.
Under the Passwords tab, there should be a "default" keyring. Right click on it and select Delete.

Now it shouldn't ask you for the password to that keyring anymore.
