### Post by zcahfg2 on 2010-10-09
Solves double right clicking issue when go tools-options-under the hood-content settings-java-do not allow anysite.

This also prevents websites like Naver blocking right click using java too.

Edit: If you do this you also disable Gesture extensions T.T
Gesture extension works fine with java disabled on Firefox.
