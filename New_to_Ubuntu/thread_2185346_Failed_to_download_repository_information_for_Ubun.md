### Post by Bucky Ball on 2013-11-02
Welcome. 11.04 is long gone. Support ended over a year ago. 11.10 dead too. I would try a clean install of 12.04 LTS (supported until 2017). You can upgrade directly from LTS>LTS so upgrading to 14.04 LTS will be a one click procedure. You'd be better to post a thread trying to figure why you can't install a supported release. 

Fixing your 11.04 issues could be a bridge too far, but who knows, you might get lucky. You haven't received security updates for that for over a year either so that is a vulnerable machine if online.

Your main issue with no updates is that there are no longer repositories. There may be a way of accessing them via backports but they will not have been updated in ages. Someone might know some magic ...

### Post by Bucky Ball on 2013-11-02
You would do a clean install on the partition you have 11.04 installed on now. To do this, choose 'Something Else' when you get to the partitioning section of the install and direct it to install to the partition currently marked as / (and perhaps with a label '11.04'). You will see it in there.

As suggested, before doing anything, backup any personal data you do not want to lose. Good luck. ;)
