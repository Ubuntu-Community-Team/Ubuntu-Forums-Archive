### Post by James78 on 2011-01-14
I've never seen anything that can affect Linux. And certainly any Windows executable's won't infect your computer, or any automatic spreading and installing things can't infect your computer when they're made for Windows (most/almost all are), unless you have WINE installed and actually make an attempt to install it, but most people are smart enough against that. If you run an exe in WINE, it still can't get at your root installation; it doesn't have permissions as it's running in user mode (DON'T SUDO IT!). If worst comes to worst and it messes up your profile, delete /home/user and restart.

Anyways, I got off topic here. It could be possible, but I've never seen it. Either way, you should install adblock plus (with a bunch of subscriptions), and noscript. There's pretty much no way to infect you this way, especially if you visit a bad site (keep it blocked, which is default). Install WoT too, that'll tell you if the site is good or bad. Usually what I do is I go to each website listed in noscript, see if they're bad, if so, I add them to untrusted and never see them again. :p

This doesn't protect you against phishing sites (Firefox may though with the phishing protection, most times, but this isn't perfect). When it comes down to it, you need a smart enough user that knows how things work, and how to not get infected (someone who's not 'stupid'), then protect your system enough, and you should always be good. I haven't had a virus in 7 years, unless it was on purpose (virtual machine hehe, just messing with virus authors)

You'll see many of the same points (and more in depth ones) in the link given to you earlier. Good luck!
