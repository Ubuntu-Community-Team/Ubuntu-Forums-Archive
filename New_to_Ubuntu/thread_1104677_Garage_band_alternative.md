### Post by stoogiebuncho on 2009-03-24
> **frogdude said:**
> Is there an Unbuntu alternative to garage band/midi maker/ finale notepad?  Or even a freeware version of FL studio (fruity loops)?

Garage Band: Try Audacity.  Jokosher looks and acts more like Garage Band, but it can be buggy.

Midi Maker: Rosegarden

Finale: I haven't found anything as full featured as finale, but you could try MuseScore.  There's also Gnu Denemo and Note Edit (and I'm sure several others).  If you have finale, you could also try running it under WINE. Actually, if you're just looking to print scores, Rosegarden might be enough for you.

### Post by Chemical Imbalance on 2009-03-25
Open up gnome-terminal under Accessories:
```
sudo apt-get install lmms
```
and if you want hydrogen:
```
sudo apt-get install hydrogen
```

Also for graphical installation of programs, I recommend using Synaptic over Add/Remove
