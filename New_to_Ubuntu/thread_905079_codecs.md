### Post by OutOfReach on 2008-08-30
Just goto Applications>Accessories>Terminal and type in:
```
sudo apt-get install ubuntu-restricted-extras
```
That should give you (almost) everything you need. :)
There is also the medibuntu repository for extra codecs.
