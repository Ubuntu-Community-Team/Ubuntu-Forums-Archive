### Post by wannacme16 on 2010-12-05
Congratulations on your move. From your post it appears you have been experimenting with Ubuntu. If Ubuntu recognizes your hard drive than it may install, while have to go through the install process to be sure. Point to consider is that if you don't want to continue using Ubuntu and go back to Windows you'll need to disconnect your hard drive with Windows on it prior to installing Ubuntu. If you don't Ubuntu will install a boot loader that will make Windows potentially unbootable if you decide to un-install Ubuntu later on. Ubuntu is relatively easy to install just follow the prompts and answer a few questions. Hardest being where and how to install Ubuntu. How to install it is only in the advanced options which you may not be comfortable in. Just find your hard drive and click install. If your still unsure there are a lot of tutorials available that walk you through.

I have a 2nd drive on my computer and I use it for all my media and stuff,
I just let Ubuntu install its programmes where it suits best, and I use my second drive for mass storage basically.
do you have your Ubuntu on a separate 'boot' partition?
if not it might be a good idea to do that first, although that would (probably)involve you re-installing Ubuntu and doing a manual partition,
 ie, setting up where you want to put everything, and not just letting Ubuntu run riot on your whole hard drive.
on my primary HDD I have 15G for ' / ' , 2G for ' swap '(1g ram X 2), and the remainder for ' /home ' .

my second drive is simply set up for media, ' /media '
