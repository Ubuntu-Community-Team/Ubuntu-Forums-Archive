### Post by linuxdroid on 2011-05-04
Well, there are some workarounds to add shortcuts to Desktop in Natty, 

1 . Go to /usr/share/applications, select the application shortcut ->Right click -> Copy to Desktop

Or 

2. Login to Ubuntu Classic desktop (old Gnome 2 UI) - while logging in select the Ubuntu classic desktop instead of Ubuntu - then add shortcuts to desktop and login back to Unity.

Hope this will help! :p
