### Post by AndyOpie150 on 2012-09-30
Did you update to 12.04 or install a downloaded version? Sorry for all the 20 Questions, but the more info the better.

EDIT: Try holding down the Alt + Windows keys, then right click on the gear in the upper right corner and select "Remove from Panel". Then hold down Alt + Windows keys again and right click on an empty spot on the tray/task bar the tray/task bar. Select "Add to Panel". Select "Indicator Applet Complete". You should now see the missing applications.
