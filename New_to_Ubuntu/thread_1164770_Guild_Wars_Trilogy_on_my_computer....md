Guild wars doesn't require a disk all your information is saved to the server via your account. Therefore you don't need the disk to install it. All you need is the client which is freely available from the Guild Wars Website. Play On Linux will download the guild wars client from their website and install both it and wine for you. Then you can play.

See the link I posted above to the Play On Linux Website it will give you more information about what it is.


You just install Play on linux then open it up and and click install and from the games category select guild wars, then follow the install instructions. It's very simple.
