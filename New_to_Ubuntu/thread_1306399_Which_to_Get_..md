### Post by shae on 2009-10-30
The larger DVD contains a lot more programs on it that you can install using the DVD as your repository.  It is much more useful if you are installing on a machine without Internet access.  I usually just use the CD.

As for Wine, it is hit or miss with applications and your best place to start is looking up application here: [http://appdb.winehq.org/](http://appdb.winehq.org/).  It will tell you if the application is know to work, what versions of wine it is known to work for, and how to make it work in wine, if it requires special tricks.

### Post by cgb on 2009-10-30
There is no real difference between the medium's other then what your computer can support/larger capacity.  If you can burn and read DVD disks then I would go with the DVD media.  

For Wine I would take a look at their website [www.winehq.org](www.winehq.org) or do a search on this forum for additional information.

shae is correct, after looking at the downloads it appears the DVD disks contain additional language packs and such which add to the file size.  So really depends if you need the extra packages but they can always be added later if you have a good internet connection.
