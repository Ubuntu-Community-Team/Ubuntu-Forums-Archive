Hello, Inese!

You can write a .doc file in OpenOffice

Most probably, OpenOffice is already installed on your system. Here is how you can find it:

If you are using GNOME, go to "Applications" >> "Office" >>  "OpenOffice.org Writer". You can type a document and save it in .doc  format

If you are using KDE, go to the "K-menu" >> "Office" >> "OpenOffice.org Writer"
[http://linux.about.com/od/kubuntu_doc/a/kubudg30t02.htm](http://linux.about.com/od/kubuntu_doc/a/kubudg30t02.htm)

If you are using XFCE, you may prefer to use AbiWord, it is a lighter  program. Just go to "Applications" >> "Office" >> "AbiWord  Word Processor". If you want to use OpenOffice, you may need to install  it.


If OpenOffice is not preinstalled, you need to install it.
In GNOME, go to "Applications" >> "Ubuntu Software Center" >> "Office", and find OpenOffice.org

In KDE, use "K Menu" >> "System" >> "Adept Manager" - "Manage Packages"
[http://www.pseudorant.com/how-to-install-programs-in-kubuntu-linux/](http://www.pseudorant.com/how-to-install-programs-in-kubuntu-linux/)

In XFCE, go to "System" >> "Add/Remove Programs"
