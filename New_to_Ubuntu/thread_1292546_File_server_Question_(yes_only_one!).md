### Post by amingv on 2009-10-16
Many people would recommend against using a testing (beta) release for server purposes, so if you're looking for stability, a LTS edition may be the way to go.

Then again, if it's just a server for personal files which is not vital to any organization, then you'll be fine. Even if it breaks I don't think it will eat your data.
