### Post by diablo75 on 2010-09-15
The two operating systems are not running simultaneously, so the only one that's "awake" or has actually been booted into will check for updates for itself, while the other is "asleep".  So to do updates in Windows, you have to be running Windows.  Same for Ubuntu.
