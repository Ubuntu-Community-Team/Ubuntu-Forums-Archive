### Post by tpjk on 2011-02-19
remove task bar by right clicking on it and choosing 'delete panel'. assuming you're using compiz, keyboard shortcuts for switching between workspaces can be set in the compiz settings manager (CCSM, compiz config settings manager). the default combinations are ctrl+alt+left and ctrl+alt+right.

hope this helps.
