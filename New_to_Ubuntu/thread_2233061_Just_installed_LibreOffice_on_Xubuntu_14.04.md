### Post by coffeecat on 2014-07-06
You need the ttf-mscorefonts-installer package which, apart from webdings, includes Times New Roman, Arial, Verdana and a few others. You can install this alone, but I would recommend you install the metapackage xubuntu-restricted extras which gives you a load of multimedia stuff and flash. Use the Software Centre.

Whichever way you install mscorefonts, you need to agree to a EULA, so stay with your machine while it installs and OK the EULA when it comes up. Threads where the OP has managed to installer the mscorefonts installer without actually installing the fonts by not OK-ing the EULA are common.
