The above Linux software is all in the .tar format, which is rather difficult for someone who "do(es)n't know a single thing about how to actually make a computer work" to install.  If you have someone who can do that for you, or if you can follow directions on-line, from this site or another, then all is well.  If the "Open Office" method works for you, then you don't need the rest of this.

I tried to find software that was distributed as a .deb file (the installation file native to Ubuntu and Debian based distros) but was unable to find anything.  So....

An alternative would be to either dual boot with Windows (I assume that your present computer has Ubuntu only) or run Windows under Linux using a virtualization program, like VirtualBox.  In this manner, it would be easy to continue using your Windows program.

Wine is sometimes problematic, and won't run *all* windows software.  If it will run Knitwear, then that would be fine, but if not, you'll either need to install one of the .tar files above (as I said, rather difficult) or run Windows either as dual boot or under virtualization software under Linux (Ubuntu).

Just throwing some suggestions on the table for consideration.
