### Post by oldfred on 2015-02-26
Have not used Windows since XP. 
But I created a shared NTFS data partition and copied both Thunderbird & Firefox profiles into that partition and updated profile.ini to new paths in both XP & Ubuntu.

[http://kb.mozillazine.org/Moving_your_profile_folder](http://kb.mozillazine.org/Moving_your_profile_folder)
[http://support.mozillamessaging.com/en-US/kb/Profiles](http://support.mozillamessaging.com/en-US/kb/Profiles)
[http://www.mozilla.org/support/thunderbird/profile](http://www.mozilla.org/support/thunderbird/profile)

Best not to mount Windows system partition. Windows for whatever reason does not like that. But if a shared data partition you must permanently mount that so you can always have profile in correct path.

### Post by JeQhdMD on 2015-02-27
And remember Penny, Ubuntu is not Windows . . . there are different ways to do most of the same things.   Most windows apps _will not work_ in Linux, some will work in Wine (install the front end "Play on Linux" first for easier setup).

Overall, I think the best course of action is to find equivalent programs in Ubuntu to do the same kind of things (just search the Ubuntu Software Center).  You'll find dozens of programs to manage email, photos, audio, video, genealogy, office suites, scanning, yada, yada, yada.

No more downloading of programs from cnet, Filehippo, etc. along with the crapware, spamware, etc.   Remember, besides this forum, there are MANY other tutorial sites on the web including just about everything you want to learn at Youtube.    Stick with it and you'll be glad in the long run.
