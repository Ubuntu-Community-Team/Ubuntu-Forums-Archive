### Post by coffeecat on 2010-11-08
> **sikander3786 said:**
> I know about dailybuilds but don't understand its purpose. Can someone enlighten?

Daily builds are for pre-release testing only. A new CD image build is made most days. The apha1, alpha2, alpha3 versions are merely daily builds for that day. There are no daily builds for released versions. The link that cromat posted simply leads (at the moment - if you follow the links in it) to the last Maverick daily build for the live CD (about three days before release) or the Natty Narwhal daily build for the alternate CD, which they've just started building. Natty daily live CDs won't be built for a little while yet.

Another thing to remember is that daily CD images often don't fit on a CD - you have to use a DVD. This is because it is more important to do the build than optimise the size for a CD in the pre-release  phase of development.

Long story short - daily builds are irrelevant to released versions.
