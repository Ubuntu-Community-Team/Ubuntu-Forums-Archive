### Post by westie457 on 2011-10-25
Hello. Ideally Ubuntu Studio should be installed to its own partition because it takes quite a bit of space on a hard drive. If you Wubi install is less than about 8GB it will be very cramped along with your original install.

Having said that and you still want to try Studio open Synaptic Package Manager and search for 'ubuntustudio'. You will only need to mark 4 packages to install. Synaptic will suggest all the others.

The packages to mark are ubuntustudio-audio, ubuntustudio-desktop,ubuntustudio-graphics and ubuntustudio-video.

When all that has installed reboot and at the login screen choose Ubuntu Studio after you have entered your password and before hitting 'Enter' to log in.

Hope this helps and Good Luck.
