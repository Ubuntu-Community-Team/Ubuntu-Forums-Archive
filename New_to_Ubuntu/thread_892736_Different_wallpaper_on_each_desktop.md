### Post by drs305 on 2008-08-17
Agree with what was said in last 2 posts (including 1 second delay). I use wallpapoz to have a separate wallpaper on each screen. Once I had it set up I put it in System, Preferences, Sessions > StartUp Programs with the command being "daemon_wallpapoz".
