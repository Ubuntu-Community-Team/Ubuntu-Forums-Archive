Using Lucid 10.4 and have Wally wallpaper changer installed. But Wally doesn't show up on any menus and I can't figure out how to get it to autostart each time system boots.  Any help would be appreciated.

### Post by SaaTeeVaaGreen on 2010-12-25
try this - go to System > Preferences > Startup Applications and add Wally to the list of apps. this should start it each time you log in.

### Post by wojox on 2010-12-25
Alt+F2 enter 

```
wally
```

It should pop up in your notification area (top right).

Right click and select Settings.
