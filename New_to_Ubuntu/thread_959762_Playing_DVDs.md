### Post by m_duck on 2008-10-26
Have you installed libdvdcss2? It is a package that is required for DVD playback. A good guide is [here]("https://help.ubuntu.com/community/Medibuntu"). Basically, you add medibuntu's software repository and install libdvdcss2 and hopefully all should be well :D.
