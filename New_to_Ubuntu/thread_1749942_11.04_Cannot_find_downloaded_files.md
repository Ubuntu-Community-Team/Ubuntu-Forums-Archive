### Post by tkelito on 2011-05-05
Synaptic installs the things you download.  It isn't like an .exe in your downloads folder you go and install, it did that for you.

To uninstall go into Synaptic or the Software Center and uninstall what you don't want using the remove option or unchecking the box in Synaptic and hitting Apply.

I assume that you're using the new Unity interface in 11.04:

Press the super button(Windows key), then start typing the name of the application. When it shows up, hit enter. Once it is running, you can right-click on its icon and select "Pin to launcher" to make it easier to launch next time.

Next time you want to install an application, do so from the Software Center. It will offer to add it to the launcher for you.

P.S. If you're using the old interface, you will find Xlog in Applications->Accesories->Xlog
