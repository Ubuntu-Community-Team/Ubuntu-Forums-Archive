### Post by Tuxoid on 2008-06-12
The Alt CD provides a DOS-like menu-based installer. It does not load the operating system's desktop unlike the LiveCD which does. If you're planning on dual-booting (I don't know if you will, but just throwing it out there), I don't think the AltCD will set that up for...

The AltCD won't show you if it Linux will work on your system's hardware (that, in terms of drivers), so as long as it does work well enough to use it for a few minutes, use the LiveCD to test your hardware under Ubuntu.
