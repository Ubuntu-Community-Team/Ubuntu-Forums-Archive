### Post by Het Irv on 2009-04-01
The easy way I have found to get stuff to run on startup it to add them under Sessions
System - Prefrences - Sessions.  To get the command, add Avant to one of your panels and then open up the properties, command should be there.

### Post by rburkartjo on 2009-04-01
toe, go to system/preferences/sessions or might be called startup applications (depending on which version of ubuntu you have) click the add button then name it what ever you want then enter or you can enter or paste this in the command box 

avant-window-navigator

then hit the bottom add button and that should make it startup . restart or log out and then log in again and it should work like a charm. let me know how it goes
