### Post by Brandel Valico on 2008-05-08
Okay Orca is in the Universal Access section of the Applications menu.

When you click Applications goto Universal Access and then Orca.

If you don't see Universal Access. Right click Applications. Goto "Edit Menus" Click it. This will open up a window. Make sure "Universal Access" is checked marked. If it's not check mark it you can also use this feature to edit other sections of the menu to show or not show other installed programs.

You should just need to click "Applications" Then goto "Universal Access" Then Click "Orca Screen Reader and Magnifer"

It opens a terminal and has you run through some basic setup stuff like language and various other things. Then asks you to reboot. After which follow the same path to open it up and use it.
