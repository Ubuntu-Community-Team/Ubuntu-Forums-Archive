### Post by j.bell730 on 2009-08-08
> **wormturns said:**
> Do these packages all have to get downloaded or might some be included in the Live CD? Is there a way I can see what is included on the Live CD? Or a way to see or confirm that the CD is correct, complete, safe, uncorrupted?

Some packages are included in the Live CD, such as a web browser, Photoshop-like program, Office suite, a few media players, a package manager (which is a GUI for 'apt-get'), etc..

Also, you can see whether the CD is correct/complete/safe/uncorrupted in the boot menu while the CD is in the drive after a reboot.

And, as others have said, other questions can be answered rather quickly by booting from the live CD.
