Well...

I tinkered with it a bit longer and found that Hardy comes together with emacs 22.1.1 which automatically includes a CUA toggle option (thus the emacs-goodies-el package it outdated in this sense). 

The following site is a good link to read about the package:
[http://olympus.het.brown.edu/cgi-bin/info2www?(emacs-goodies-el)Top](http://olympus.het.brown.edu/cgi-bin/info2www?(emacs-goodies-el)Top)

Additionally, it is possible via the customization of this version of emacs to change the color-themes (I chose the calm forest one).

Just go to: options->customize emacs->color theme->color theme selection.

Hope it helps someone,
Xris
