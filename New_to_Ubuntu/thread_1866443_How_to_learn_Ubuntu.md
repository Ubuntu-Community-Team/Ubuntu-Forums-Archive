### Post by bryncoles on 2011-10-21
I'd say set up a dual boot system (make sure your important data is all backed up before you start.  Most likely nothing will go wrong, but I remember there used to be so many threads about people accidentally wiping all their data!).  

Once you're dual booting, start trying to work in Ubuntu.  At first, things will seem strange and complicated, and straight-forward tasks might seem confusing.  This is just the bedding in period, and it's what we keep windows for.  If you *need* to get something done, and can't be bothered spending five minutes (...hours ...days...) figuring it out, just boot back to Windows, rattle it off then come back to Ubuntu. If you#re like me, you'll find you spend more and more time in Ubuntu, and less and less in Windows, until you can just junk windows altogether.  

Don't worry about the CLI -- the command prompt -- either.  It's quite easy to avoid, if you don't want to use it!
