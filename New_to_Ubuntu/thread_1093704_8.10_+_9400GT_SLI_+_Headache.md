### Post by martrn on 2009-03-12
It maybe that bullet proof X  eg X11 with no config file or incorrect settings in it.  From 8.10 and onwards the Xorg X11 config files have also removed the user input parts to rely more on plug in modules.

[https://wiki.ubuntu.com/BulletProofX](https://wiki.ubuntu.com/BulletProofX)

It causes me problems.

Only the very new nvidia drivers (installed from nvidia after the Xorg latest relase) have proper support for Xorg 7.4.  It could be that you are trying to use an nvidia driver that predates the Xorg files ???

I could be something else....

Have you tried looking in your /var/log directory for logs, there should be some clue there if there is a clue to have.
