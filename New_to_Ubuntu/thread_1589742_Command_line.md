### Post by pizza-is-good on 2010-10-06
The command line in Ubuntu is Bash (Bourne Again Shell). It is standard to most new Linux Operating Systems.
That being said, a good reference list of commands is [this]("http://ss64.com/bash/"). I use it quite often.
If you are looking to learn how to use the command line, I recommend [The Linux Command Line]("http://linuxcommand.org/tlcl.php"), available for free from the link. It is very good and teaches the Command Line step by step. 

~pizza
