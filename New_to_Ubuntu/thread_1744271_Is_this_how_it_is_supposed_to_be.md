### Post by GrouchyGaijin on 2011-04-30
I'm running 10.10 and  *thought*I'd give 11.04 a try.
I made a backup of my current system using Remastersys (a great program) and clicked upgrade in the update manager.

The upgrade went OK.  I didn't have any of the horror stories that others have had with Nvidia cards.

The first thing I noticed was that my conkies all launched on top of the desktop covering places I wanted to click.  I thought OK, I guess I'll need to add a longer sleep time before the conkies launch.

Then I noticed that Firestarter, while it would launch at boot, didn't have any buttons to minimize it.

The same thing with Google chrome - no minimize button.

Then I noticed the three buttons that were now in the upper left hand corner of my screen.  I could minimize with one of those buttons.  Hmmmm I kind of liked the buttons where they had always been.

Then I thought I'd change the theme.  I realized that in addition to no minimize/maximize button on the theme window there was also no scroll bar on the right hand side.

After playing around for a bit, I decided I missed 10.10 as I had it tuned so nicely.

I popped in the Live DVD I had made with Remastersys, rebooted and put everything back the way it had been before 11.04.

So is it supposed to be like that?  No buttons on the windows and no scroll bar on the right hand side of a window?  Or did something go wrong during my upgrade?

### Post by grahammechanical on 2011-04-30
The answer is Yes. This is the way it is designed to be. 

There is a scroll bar. It is very thin. When you move the pointer over it a kind of tab appears that you can use to drag up and down.

To get the max/min/close buttons  to appear click the Ubuntu icon to make the dash appear then click again to get it to disappear and the window's buttons will appear in the top panel and remain there. This kind of behavior happens from time to time. It may be corrected over time.

You will also find that a program's File, Edit, etc, panel will now appear in the top panel at the left when you pass the pointer over it.

This design change is done to create more screen space. It works great on wide screen TV/monitors.

Regards.

### Post by Lateralis on 2011-04-30
Gnome classic will be gone from 11.10.  Unity (the current default desktop environment) will be the only one available in Ubuntu from 11.10.  If you wish to remain on Gnome classic then best bet is to install Lucid - the LTS release - which is supported for the next two years.  

Otherwise, you can try one of the other *buntus, such as Kubuntu, Xubuntu or Lubuntu.  XFCE used in Xubuntu has a Gnome feel to it, whilst Lubuntu uses LXDE which is a lightweight desktop and has shades of KDE about it.  You can install all of these desktops from the main repos: kubuntu-desktop, xubuntu-desktop and lubuntu-desktop respectively.  At login, you can choose which desktop environment to use (Classic Gnome, Unity, KDE, Lubuntu, Xfce, depending on which ones you decide to install).
