### Post by lukjad on 2009-01-06
I suggest creating a folder called Themes and download your selections there for easier reference. After downloading the themes you like open up your themes by going to System->Preferences->Themes near the bottom. Once the themes menu is open click on Install. An install window will open. Browse to the themes folder. Highlight the theme you want to install and click open. This will install the selected theme. You are then given the choice to use that theme or continue using your original theme.

Here is a fairly good [guide]("http://www.msccompcasts.com/2005/05/how-to-install-themes.html").
