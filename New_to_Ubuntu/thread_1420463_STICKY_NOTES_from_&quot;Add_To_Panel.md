### Post by r-senior on 2010-03-03
No guarantees about what might still remain, but the XML file where Sticky Notes stores its stuff is ~/.gnome2/stickynotes_applet.

It's in a hidden directory, so you'll need to View -> Show Hidden Files to find it in Nautilus.
