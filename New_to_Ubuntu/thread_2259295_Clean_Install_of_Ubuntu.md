### Post by Bucky Ball on 2015-01-03
Welcome. Windows viruses mean nothing to Ubuntu. I would go ahead and burn an ISO image to disk. You could wipe and reinstall Win7 if you wanted, but I personally wouldn't. You can also download and burn the Ubuntu ISO on another machine, perhaps a friend's. If you want to purchase pre-made install media, see [HERE]("http://shop.canonical.com/product_info.php?products_id=1146"). 

Good luck. ;)

### Post by Topsiho on 2015-01-03
I guess that if in the virus infected Windows you burn an Ubuntu liveDVD, and have it checked before installing Ubuntu, and it is OK, then you need not have any worry about the safety of the liveDVD. It wouldn't check OK if it is not.
The check option is offered when you start the liveDVD, together with some other options.
If this check is not succesful, you need another computer to burn the liveDVD, as it appears to be tampered with by the malware.

### Post by Mark Phelps on 2015-01-03
You should consider "disinfecting" Win7 by running AV apps from "outside".  If you download the latest Kaspersky AV boot CD and burn in, when you boot from it, you'll discover that it actually is running a version of Linux.  That allows you to disinfect Windows while it is NOT running.
