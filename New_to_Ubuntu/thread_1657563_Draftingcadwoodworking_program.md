### Post by TeoBigusGeekus on 2011-01-01
[Bricscad]("http://www.bricsys.com/en_INTL/bricscad/index.jsp") has a trial version for linux.
It's supposed to last for only 30 days, but if you delete the .bricscad (or something) folder from your home partition it starts counting again.
All in all, it should fit you, it's a fair autocad 14 clone; it should be adequate for drafting.
