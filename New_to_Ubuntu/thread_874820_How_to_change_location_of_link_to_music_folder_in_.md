### Post by kramer65 on 2008-07-30
Hello,


I have an external usb-drive with all my music. In the locations menu there is a link to a Music folder in the home folder. I now want to make it work that when I click that in the locations menu, it goes to another folder I specified (on my external usb drive)..

Is this possible, and if so, how?

---

### Post by Diabolis on 2008-07-30
You can add new folders to the locations menu by bookmarking them. Just as if you were using firefox. I'm guessing that you are using nautilus, so just press <Ctrl>d when you want to add a folder to the bookmarks.

---

### Post by SunnyRabbiera on 2008-07-30
It should be, i have used alternate directories for it myself.
The link to the music folder can be removed by going over to the side pane, right clicking and selecting "remove"
This will remove the default link to the music folder without removing the actual music folder in your system
then when you plug in your external you can just drag and drop the folder you desire into the side pane to create a shortcut.
Just remember it will only work when you have the external plugged in.

---

### Post by saratchandra on 2008-07-30
Go to My computer -> Bookmarks -> Edit Bookmarks and change the location from there.
