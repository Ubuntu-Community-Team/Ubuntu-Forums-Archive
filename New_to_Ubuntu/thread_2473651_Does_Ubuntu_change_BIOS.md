### Post by yancek on 2022-04-09
>  What does that mean and how do I install anything ?  

A web search should give you any number of sites explaining it, as the one below does.  Check your BIOS first for any mention of ACPI then read the info at the link below.

[https://www.compuhoy.com/how-do-i-fix-this-bios-is-not-fully-acpi-compliant/](https://www.compuhoy.com/how-do-i-fix-this-bios-is-not-fully-acpi-compliant/)

Did you install Ubuntu in UEFI mode?  A standard windows 7 will not be UEFI but it can be installed UEFI.  Both systems should be either UEFI or Legacy/CSM.  I'm sure you are aware that windows 7 support ended over one year ago so please don't configure the network on it and do the rest of the internet users a favor.
