### Post by PocketDog on 2008-09-28
Transmission's like a swan, it doesn't look like there's much going on but it's paddling like mad underneath. It's set to open (map) ports by default if it can, so don't worry about that. Initially it may not be as fast as BitComet, because BitComet is extremely aggressive in a torrent swarm, and is a bit naughty  with the protocols. Good for a BitComet user, terrible for everyone else - which is why it's banned from almost all private torrent sites. Have a look at Deluge if Transmission isn't working out for you; it's a little more customisable.
