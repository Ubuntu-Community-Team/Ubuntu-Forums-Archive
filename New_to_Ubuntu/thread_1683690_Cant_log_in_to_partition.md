### Post by Rubi1200 on 2011-02-08
The error looks like something to do with either a failing hard-drive or serious file-system damage (or both).

Did something happen recently to cause this?

You can check the health of your drive using the Disk Utility on the LiveCD

You could try mounting the partition from the LiveCD.

If that does not work, try this to fix the file-system:

```
sudo e2fsck -C0 -p -f -v /dev/sda3
```

if this returns errors, run this:

```
sudo e2fsck -f -y -v /dev/sda3
```

Testdisk is another option to try and restore/recover:
[http://www.cgsecurity.org/wiki/Damaged_Hard_Disk](http://www.cgsecurity.org/wiki/Damaged_Hard_Disk)
[http://www.cgsecurity.org/wiki/TestDisk_Step_By_Step](http://www.cgsecurity.org/wiki/TestDisk_Step_By_Step)
[http://www.cgsecurity.org/wiki/TestDisk](http://www.cgsecurity.org/wiki/TestDisk)
