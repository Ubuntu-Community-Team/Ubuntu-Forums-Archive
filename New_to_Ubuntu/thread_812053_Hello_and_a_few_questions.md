### Post by sdennie on 2008-05-30
I've found the best way to buy hardware for linux is to disregard the fact that you are going to use it for linux.  Find exactly what you want and then do the following google searches:

```

linux hardware_you_want

```

```

ubuntu hardware_you_want

```

```

ubuntu hardy hardware_you_want

```

If you find nothing or that it definitively doesn't work after doing those three searches, cross that off your list and repeat with the next closest thing.

I wish it was like that. But even at Ubuntu Forums we must be honest when talking about Ubuntu.

We tend to think usually things go very well with linux but hardware is the weak point. It's not linux's fault but the problem still is there.  Situation is better all the time.

With linux you can usually do something to get your stuff working - for example ask Ubuntu forums :)
