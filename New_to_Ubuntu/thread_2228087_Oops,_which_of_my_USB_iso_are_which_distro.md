### Post by papibe on 2014-06-05
Hi ross4.

I can't help you with other distributions, but for Ubuntu you can insert the USB, and it will be auto mounted. Then look for a file called:
```
README.diskdefines
```
Among other things, it contains the arch and version.

Hope it helps. Let us know how it goes.
