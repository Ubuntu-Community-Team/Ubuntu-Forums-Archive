### Post by 3rdalbum on 2011-09-01
> **Claude Rebeck said:**
> In Unity how do I change my DVD  tray to be read and write,  now it is no permission for write.

You need to use a DVD burning program in order to write to DVDs (unless you're using DVD-RAM discs). Brasero is the default DVD burning program in Ubuntu, this should work for you.

> Also, I installed "konsole" from the command line. Now how do I get the KDE shell to run??

Konsole is just a program that can run on KDE, it's not the whole KDE shell. You need the "kde-workspace-bin" package, I think, to run KDE. You would find it available on the login screen.
