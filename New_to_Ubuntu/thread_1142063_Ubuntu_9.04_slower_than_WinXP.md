### Post by LindaBW on 2009-08-05
rvgsd ---

I also recently installed Ubuntu 9.04 as a dual boot with Windows XP and found the former to be much slower.  I am new to Ubuntu, but some internet reading made me guess that it was a problem with the interaction between Ubuntu and my graphics card.  I have a Mobility Radeon 9000 graphics card.  I followed the suggestion of the following site:

[https://answers.launchpad.net/ubuntu/+source/xserver-xorg-driver-ati/+question/72258](https://answers.launchpad.net/ubuntu/+source/xserver-xorg-driver-ati/+question/72258)

and added the referenced list of Options to my xorg.conf file.  This greatly improved the speed in the Ubuntu partition -- I can't tell any difference between it and XP now.  I have a Pentium 1.7 Ghz processor and 512 MB of RAM, and I do not feel like I need a faster machine to run Ubuntu 9.04.

It is my impression that Ubuntu 9.04 is having performance issues with a number of graphics cards, so it may be worth reading about Jaunty's interaction with yours.
[]("http://ubuntuforums.org/showthread.php?t=810444")
