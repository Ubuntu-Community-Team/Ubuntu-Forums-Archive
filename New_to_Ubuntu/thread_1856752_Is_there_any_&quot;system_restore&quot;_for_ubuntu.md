### Post by dFlyer on 2011-10-08
If your using a dell and have dell-recovery installed it's very easy, otherwise you will need to use your install disk for what ever version your using. If you can get to a command line and have internet access you can use apt-get to reinstall the programs you removed. You can also try:

unity --reset 

to rebuild your desktop, if your using unity.
