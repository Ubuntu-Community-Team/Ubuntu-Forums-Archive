### Post by cartisdm on 2009-12-30
I cannot, for the life of me, figure out how to add more workspaces.  Usually I would right click on the "Workspace Switcher" --> set "Number of Workspaces"

That's not working.  All I get is the ability to set the number of columns and number of rows.  Changing that changes the "look" of the windows switcher to appear as if there are more workspaces, but there are still only two!  The whole reason I'm doing this is so I can get the rotating cube effect in Compiz but right now all I can do is rotate the flat, 2 dimensional workspaces.

(Yes, I've already enabled Desktop Cube and Cube Rotation in the compiz manager)

---

### Post by warfacegod on 2009-12-30
Set columns to 4 and rows to 1.

To see the cube, put your mouse pointer over your desktop, middle click (left + right mouse buttons) and hold them, move mouse.

Do you have Simple CompizConfig Settings Manager installed?

### Post by lindsay7 on 2009-12-31
Go to the Compiz Conf menu under System/Preferences and then go to general settings/general desktop and set the horizontal setting to four and you will get four workspaces. Setting this number higer will give you more workspaces and also sides on your object to rotate.
