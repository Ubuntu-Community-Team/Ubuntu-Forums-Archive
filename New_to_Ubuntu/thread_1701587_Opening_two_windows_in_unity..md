### Post by coffeecat on 2011-03-06
Is that with Natty or Maverick?

In Natty you don't use the launcher to open a second window. Use the File menu for whichever application: File > New Window. If it's an app that has the menus integrated into the Unity panel, such as File Manager, the File menu will appear in the panel - clearly. But some apps are not integrated yet. Firefox is one and the File menu is in the usual place.

### Post by coffeecat on 2011-03-06
> **CaptainMark said:**
> im trialing natty, can you stop the file menu appearing at the top, id rather it on the window in question

No. It's part of the design philosophy of Unity. It's to save valuable vertical pixels in an age when most people are using widescreen monitors. Remove the menus from the window bar and you save screen real estate. That's why the dock is at the side. (Not many people know that! :wink:)

### Post by EaStErDoM on 2011-05-21
Opening a second window is easy: Just click with the middle mouse button on a starter in unity panel.

If your mouse dont has a middle mouse button but a mouse wheel, you can just click the wheel down while the cursor is over a starter.
