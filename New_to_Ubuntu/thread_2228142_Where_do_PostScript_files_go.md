### Post by deadflowr on 2014-06-05
First, do you know which folder it was suppose to be saved to?

Second, it is possible that the file name was simply .ps, in which case it's actually hidden (because of the dot; dot files are hidden files)
So to see if it is hidden, press ctrl + h, to show hidden files and folders.

If you can't remember which folder it printed to, simply re-run the print again and it'll show you where, as the last place will be listed.

Hope this helps.
