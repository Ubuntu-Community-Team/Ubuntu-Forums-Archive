### Post by drs305 on 2009-01-21
If you really want to limit the downloads, open Software Sources or synaptic (System, Administration ... then Settings > Repositories > Updates tab. As a minimum, have the security option ticked. "Recommended" updates are, as the name says, recommended but not necessary. Hit "Reload" to refresh your preferences.

With a new install, the first time you run the updates there will be a lot but after that the number will decrease if you run updates regularly (set on the same Updates tab).

### Post by drs305 on 2009-01-21
Once you have ticked/unticked what you want in Synaptic or Software Sources and hit "Reload", the system will reset what it shows available for updates based on the new settings. You can then run Update Manager to see the new download list.
