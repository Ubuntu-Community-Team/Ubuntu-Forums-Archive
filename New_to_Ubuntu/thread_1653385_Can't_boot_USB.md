### Post by -kg- on 2010-12-26
How did you install Kubuntu to the Flash drive?  If you used Unetbootin, I've never had any luck with the included downloader (first radio button) installing correctly.

I've had complete successes with downloading the .iso first, then using the .iso to install the LiveUSB version to the Flash drive.  Make sure you have a good download (I usually download via torrent), then use Unetbootin to make the LiveUSB using the "Diskimage" method (2nd radio button).
