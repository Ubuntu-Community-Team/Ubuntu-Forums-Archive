Hi,

If for some reason the window decorator has turned off or not working, try:
```
compiz --replace
```
This bit of code is safe to run even if the decorator is running ok, so you could try that anyway.

Hi, Beelzebufo.  I don't blame you for wanting to throw in the "stupid" comment.  But the message body might have been a better place.  Using the subject field to have a suggestion of the content would be a better contribution to the community for people who have similar problems.

Just something you can consider.  I have lots of problems with Unity.  Most of the responses I find are suggestions to reset Unity back to its defaults every time something goes wrong.  When browsing the forums I usually read the messages that has Unity in the title to help me with some of my current annoyances or problems I haven't gotten to yet.

In my environment I find that when making changes with CCSM I usually have to log off and log back on to bring back many other features that would have stopped working, many times the title bar, or the exit and minimise buttons.

The specific option in CCSM is to check mark Windows Decoration.  If it's already check marked, uncheck it, then check it.  If you still don't get your exit and minimise buttons log off and log back on with the feature checked.
