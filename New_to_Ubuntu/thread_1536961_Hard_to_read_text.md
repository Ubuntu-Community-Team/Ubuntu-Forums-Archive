The text on my monitor is too tiny, can someone teach me how to have large size of text, so that it is easier for me to read?:p

---

### Post by Zeike on 2010-07-22
go to System->Preferences-> Appearance

select the fonts tab

click on the font categories you wish to change, pick a bigger size.

hit ok.

Bam, done.

---

### Post by lisati on 2010-07-22
When you're using FireFox, you can use Ctrl + the scroll wheel on your mouse to change the text size.
