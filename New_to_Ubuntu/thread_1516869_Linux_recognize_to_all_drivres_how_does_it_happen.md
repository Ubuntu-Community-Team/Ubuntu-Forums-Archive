### Post by tarps87 on 2010-06-24
In simple terms the drivers are built into the underling kernel. Instead of having to install drivers as and when they are needed, there are a large amount already built in. There are however some drivers that are not included, mainly these are related to graphic and network cards
This [link]("http://it.toolbox.com/blogs/locutus/the-magic-of-linux-device-drivers-29899") has an explanation of it, again it's only a top level view but it should clear things up a bit.
