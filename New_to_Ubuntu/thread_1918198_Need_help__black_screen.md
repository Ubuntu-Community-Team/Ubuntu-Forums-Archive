### Post by varunendra on 2012-02-02
Default Ubuntu 11.10 with Unity won't run smooth on such a low specs machine anyway. Try [Xubuntu]("http://www.xubuntu.org/getubuntu#oneiric"), or even lighter - [Lubuntu]("http://cdimages.ubuntu.com/lubuntu/releases/11.10/release/") on it. I prefer downloading via torrents for their speed and reliability (automatic integrity check while downloading).

Try in live mode first (via a live cd or usb) to see if your hardware is supported. In case of problems, please try the various [boot options]("https://help.ubuntu.com/community/BootOptions"). If they don't help, post back here.
