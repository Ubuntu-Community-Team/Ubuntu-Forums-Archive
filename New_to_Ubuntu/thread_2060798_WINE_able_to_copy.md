### Post by I8NY on 2012-09-21
Short answer: no

There does appear to be a semi-portable version out there on the web but the dependence will have to be pre-installed on the PC to run it.  If you say have Wine installed on your PC right now you can't copy it to the thumb drive because the install would be looking in the wrong place for it's files.  Easiest thing is just install Wine on another PC when you get there or have the deb package on your thumb drive to install without internet.
