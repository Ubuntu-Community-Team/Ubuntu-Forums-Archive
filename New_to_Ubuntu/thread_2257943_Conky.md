### Post by Frogs Hair on 2014-12-23
Yes, but first you have to install the conky program and then follow any instructions included with the script. 

14.10 ```
sudo apt-get install conky
``` 14.04 ```
sudo apt-get install conky-all
``` Additional programs might be required depending on the script .

You might also want to take a look at the link if running 14.04 which installs all you need to get conky running including some scripts . There was no such program when I started and I learned the old way. 

[http://www.webupd8.org/2014/06/conky-manager-gets-revamped-ui-new.html](http://www.webupd8.org/2014/06/conky-manager-gets-revamped-ui-new.html)
