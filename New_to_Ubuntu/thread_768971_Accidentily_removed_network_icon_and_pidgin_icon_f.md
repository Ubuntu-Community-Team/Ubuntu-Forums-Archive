### Post by PattonPending on 2008-04-26
On the off chance it's pidgin and not the notification area, make sure that you go into pidgin's preferences --> interface and set "Show System Tray Icon" to "Always".

The name of the wireless/network applet is nm-applet.  If it's not showing up in the system tray by itself then you might need to track it down and add it manually.
