### Post by kkgspot on 2009-06-11
I have two computers (both running jaunty) connected by a linksys router. I use the remote desktop viewer on computer A to maintain computer B. The problem I'm having is, I can only remote desktop to one user account (administrator). There are 4 other users on computer B that are very new to linux and it would be nice to use remote desktop to show them how to do things. I've tried everything I can find in the forums, but remote desktop keeps telling me that the connection is refused. Do I need to setup up the remote desktop for each user on computer B?

My other problem is:
when I go to Network>Places on computer B, computer A shows up, and I can share files.
when I got to Network>Places on computer A, computer B does not show up at all.

Both systems are setup almost identically, same jaunty, same updates, same software.

Anybody got any ideas???

---

### Post by kkgspot on 2009-06-11
I figured it out. I needed to run the remote desktop settings for each account on the computer. Everything's working fine now..
