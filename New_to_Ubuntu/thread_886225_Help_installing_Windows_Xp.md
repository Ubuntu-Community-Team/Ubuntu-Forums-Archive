### Post by cozmicharlie on 2008-08-10
Ok - I will assume you are being series since it not April.  You can't just install XP in Ubuntu using your xp install disk- XP is an operating system like Ubuntu.  XP is not a program.  If you want to run XP along with Ubuntu you have 3 options.

[LIST=1]
[*]Dual boot (create a Ubuntu partition and an XP partition on your hard drive and install each OS to the separate partitions).
[*]Install Ubuntu on an XP computer using Wubi.
[*]Install XP via Virtual Box in Ubuntu (this sets up a virtual XP in Linux).
[/LIST]

I would recommend the third option - see the web site for downloads and instructions.  Post back if you need help.
[http://www.virtualbox.org/](http://www.virtualbox.org/)
