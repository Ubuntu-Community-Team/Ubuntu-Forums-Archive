### Post by cottfcfan on 2012-09-01
You can do 3 things.
If you just need to increase the font size in Firefox, then edit/preferences/content is what you need in the Firefox menu.
If you want to zoom the whole desktop in Firefox, the default full level zoom or nosquint addons are good.
 If you want to increase your font size across the board, the Ubuntu Tweak is an excellent tool to do this, you can add the ppa here:
[https://launchpad.net/~tualatrix/+archive/ppa](https://launchpad.net/~tualatrix/+archive/ppa)
Hope this helps.
