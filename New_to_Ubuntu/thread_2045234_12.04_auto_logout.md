### Post by matt_symes on 2012-08-23
Hi

That sounds like X may be crashing dropping you to the lightdm login screen.

Is there anything in the log files. They are the

```
/var/log/xorg.*
```

log files. 

The newest one will be call /var/log/Xorg.0.log. 
The one before that Xorg.1.log.

grepping the log files may help here.

From the terminal...

```
grep "EE" /var/log/Xorg.*
```

The above command is case-sensitive (like all terminal commands) so the uppercase E's are required.

Post the output of that last command back here between code tags.

What video hardware do you have ? 
What drivers are you using ?
Have you noticed a pattern when the crashes occur ? 
What programs are running when the crashes occur ?

Kind regards
