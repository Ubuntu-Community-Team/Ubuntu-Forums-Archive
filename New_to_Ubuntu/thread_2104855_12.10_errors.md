### Post by fantab on 2013-01-14
In another desktop of mine which was running 12.10 with all those crash reports I installed CINNAMON (far better choice that Gnome Classic, personally) on top to Unity. I did not remove anything just added Cinnamon. And surprisingly, it works more or less flawlessly with 12.10... give it try. 

Here is how you can Install Cinnamon from the terminal:
```
$ sudo add-apt-repository ppa:gwendal-lebihan-dev/cinnamon-stable 
$ sudo apt-get update 
$ sudo apt-get install cinnamon
```When it is installed, LOGOUT you can choose it from your login screen (LDM)by clicking the ubunu icon in the Login box... select Cinnamon and hit OK... and Login to Cinnamon.

But I like Unity better... ;)

Edit: Remember if you do install Cinnamon you will have two File Managers, Nautilus (Ubuntu/Unity Default) and Nemo (Cinnamon Default)... You can use both. Just a heads up.
