Ok.

Run sudo fdisk -l

That will tell you which the other drive is called; e.g., /dev/sdb1 and

When you have that info you will edit it into the fstab file.

Example:

/dev/sdb1    /mountpoint    ntfs    defaults    0    0

**/dev/sdb1** is the name of the partition you want to mount
**/mountpoint** is the location where it will be mounted
**ntfs** is the file system
**defaults**
**0**
**0**

### Post by Gannon8 on 2009-10-10
Type this into the terminal
```

mkdir /home/<your username>/sdb1

```

Then edit your /etc/fstab with nano.
Replace "/mount point" with "/home/<your username>/sdb1"

All your stuff will be in your home directory, in the new directory sdb1

If you want a different name for it, change all of the "sdb1"'s up there with whatever you want, but keep the preceding parts. And make sure to replace <your username> with what you might think would go there ;)

For example, lets say the drive contained my music collection. I could type into the terminal "mkdir /home/<my username>/music-collection" and add the line "/dev/sdb1 /home/<my username>/music-collection default,noatime 0 0"
