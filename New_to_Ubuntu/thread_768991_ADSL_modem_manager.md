the error is a dependency error, either a dependency is not met, or in this case, either the python-gnome2-extras is either too low, or too high a version to be compatable with the program you are trying to install.  what is the .deb you are trying to install, we can see what is needed for it.

when I get a dependency error, I first look in synaptic first, for the packages, and install them first, if I can't find them there, I try downloading them from the get command.  you may also not have all the available software channels set.
