### Post by weezyrider on 2011-05-02
I had a major glitch and reinstalled Ubuntu. Now Ubuntu wants a login after the screensaver turns on. It didn't do this before, I have settings checked to not ask for password as I'm the only user on the computer and it is NOT networked. How do I turn off the request for password after the screensaver turns on? Password is OK for admin tasks. 
Thanks

### Post by New Linux Usr on 2011-05-02
Sorry for the late entry, to turn off the requirement of a password once the screensaver kicks in, you have to go to:

System->Prefrences->Screensaver

and un-tick "Lock screen when screensaver is active". That will remove the password prompt.

Kind regards,

New Linux Usr
