### Post by lavinog on 2009-02-09
Does the fan run at all?
How are you determining the CPU temp? (some sensor programs give bad readings on some systems...bios would probably be the best way to read temp.
Could the problem be that your CPU usage is high (common if you are not running accelerated video drivers.) Add the system monitor to your panel and see if cpu usage looks like while using different apps...videos, flash apps really shouldn't use more than 25% to display.
I don't know much about the intel graphics drivers, but I do know on the ATI based laptops, if the drivers aren't installed, system temps will rise just from scrolling a web page.

How old is the laptop? Could the fan require a cleaning.
Side story: I had a friend that gave up on Ubuntu because he thought a bug was causing his fan to make weird noises...it turned out that the fan was full of dust.
