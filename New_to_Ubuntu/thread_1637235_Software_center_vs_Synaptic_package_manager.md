### Post by 3rdalbum on 2010-12-04
Synaptic and Software Center both take their software from the normal Ubuntu repositories, plug any other repositories that you have added. They are "frontends" for the same utility (apt).

Synaptic is more powerful and has more features, but Software Center is easier to use and looks nicer.
