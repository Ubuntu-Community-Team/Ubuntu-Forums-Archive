### Post by jermza on 2010-12-30

So, to work around that, I had to open the folder via Ubuntu's main menu and find the file and then take it to the opened email.

### Post by mcduck on 2010-12-30
Well, the file selection dialog is able to display previews, but the application developer has to tell it to do so.

Most likely Evolution's developers didn't enable that feature because such previews are only really useful for images, while e-mail attachments can be pretty much any types of files.. You'll usually see the preview in applications that *only* handle images.

You can of course send feedback to Evolution's developers and ask them to enable the preview. I kind of doubt they'd do that, though... :)

Places, Home Folder, (up pops a Nautilus view) Edit, Preferences, Preview tab, change your preferences here to see thumbnail previews of kinds of file and file size.
