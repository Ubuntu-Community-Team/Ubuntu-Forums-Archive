### Post by Didius Falco on 2009-05-04
I'm not sure it will work with the alternate install cd, but you can try this:

Put the CD in the drive. 

Go to **System/Administration/Software Sources**, then click on the **Third Party** tab and click the **Add CD-ROM **button.

Of course, you could always install it from the repositories, if that doesn't work. Here is a very good guide to doing just that. Be sure and read it all before you add KDE:

[http://www.psychocats.net/ubuntu/kde](http://www.psychocats.net/ubuntu/kde)
