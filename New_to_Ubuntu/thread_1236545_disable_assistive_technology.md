### Post by Paddy Landau on 2009-08-10
I'm not precisely sure what you are asking for.

Try this:

[LIST]
[*]System > Preferences > Assistive Technologies.
[*]Do not check the box for Enable assistive technologies.
[*]Run though any other options as needed.
[/LIST]

### Post by duanedesign on 2009-10-12
Today I ended up with two Assistive Technology (Universal Access) Icons in my panel. They are slightly different icons.
I got rid of one by doing something similar to the above suggestion.

System > Preferences > Assistive Technology > Keyboard Accesibility > Accessibility features can be toggled with keyboard shortcuts (uncheck)

this can also be found under 

System > Preferences > Keyboard > Accesibility > Accesibility features can be toggled with keyboard shortcuts (uncheck)
