### Post by Kabir_Gandhiok on 2015-08-21
When I right click on the title bar of any program, the menu displayed  is sometimes dark and sometimes white. The selected theme is Ambience. I  did not face this problem until I installed Unity Tweak Tool. For  instance, if I right click on the title bar 5 consecutive times, it  would a dark menu twice and a white menu thrice. I find this annoying  and would appreciate if anyone could help me resolve it.

### Post by Kabir_Gandhiok on 2015-08-22
I think there's some sort of conflict between Unity Tweak Tool and Compiz which was causing me this problem. In unity tweak tool if I turned on auto raise or if I toggled the right click mouse functions, it automatically turned on windows decoration in Compiz. This causes the right clicks menus of all title bars to appear out of sync with the selected theme. I had Ambiance theme selected but the right click menu appeared in the Radiance theme most of the times. Un-checking "windows decoration" in Compiz fixes this problem but if the right click controls are toggled again in Unity Tweak, the problem regenerates. I guess this is some bug that needs to be squashed. Marking this as "solved", though the bug does exist so I'm just not going to use unity tweak any more.
