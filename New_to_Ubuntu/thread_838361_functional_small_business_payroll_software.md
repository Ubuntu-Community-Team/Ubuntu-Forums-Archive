If you find one, be sure to let us know. To my knowledge, there isn't such a software piece available in Linux. Gnucash, in its help files (IIRC) discusses how to create a payroll module using it, but I've not bothered. As a general observation, payroll modules to accounting software are almost always an "extra cost" option, for good reason; payroll is such a specialized function, that it would be darned hard to write a general program that did not need quite a bit of specialization to fit the individual's situation, what with not only the federal side but also the state side. Again, I wish you luck in your search.
