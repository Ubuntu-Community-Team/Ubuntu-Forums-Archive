There's always virtualization. Running Linux as a virtual machine lets you use Linux without the risks. Virtual Box is free and has a version for Windows. It can take a snap shot of the virtual machine also, so it's good for being able to restore to a known good configuration if something goes wrong with the vm. You can take the snap shot first every time you go to use the vm then if something happens while you're using it you could go back to the way it was when you started it that time.
