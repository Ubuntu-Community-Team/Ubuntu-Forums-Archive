### Post by JayKay3000 on 2009-07-27
assuming your running gnome and assuming you have the advanced desktops settings (might be called compizConfig settings) installed then all you need to do is open that up and make sure you have desktop cube and rotate cube checked. 

Go into the general options and under the desktop size tab make sure that the horizontal virtual size is set to 4 or more 3 will give you a triangle and ofc a lot more will turn into a cylinder.

If this does not work then it might be a problem or limitation of your graphics card.

[http://ubuntuforums.org/showthread.php?t=809695](http://ubuntuforums.org/showthread.php?t=809695)

this is a faq(frequently asked questions) link about desktop effects including the cylinder.

Hope this all helps.
