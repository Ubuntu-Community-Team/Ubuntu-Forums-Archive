### Post by Jon Monreal on 2010-04-29
Hello,

Please try pressing Ctrl+H when you're in the folder with the documents and look for files with names like .~lock.mydocument.odt#

Delete these files and try again. When you're done, you can press Ctrl+H again to hide the hidden files.

Thanks.

### Post by Jon Monreal on 2010-04-29
I'm not aware of any methods to get rid of them as they're supposed to allow editing again after you're done with the document. They are created to prevent multiple users from editing the same document at once.
