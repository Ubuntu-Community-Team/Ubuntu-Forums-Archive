How to howto install nvidia GeForce GTX 460 or tutorial for linux or ubuntu. Basically I downloaded nvidia driver 260.19.12 and went through all sorts of contortions to install it- synaptic manager hardware driver additional drivers root archive manager sudo su new user etc etc
Then all i did was download ubuntu 10.10 make a freash install, and it recognised the card, in aditional drivers, I selected that from system administration additional drivers, and low and behold there was my card smiling up at me! all i did was enable it- now i have full 3d.
