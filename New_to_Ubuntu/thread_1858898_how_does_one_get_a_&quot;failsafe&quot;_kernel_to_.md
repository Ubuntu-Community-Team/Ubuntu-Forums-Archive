### Post by Lisiano on 2011-10-13
The "Failsafe" kernel is the normal kernel but with reduced functionality. Just boot into the Recovery Console (Hold Shift while you boot then press Down once so it says "Recovery" on the end of the line). There you can pick either Safe graphics or Failsafe graphics, it might be called X Server instead.
