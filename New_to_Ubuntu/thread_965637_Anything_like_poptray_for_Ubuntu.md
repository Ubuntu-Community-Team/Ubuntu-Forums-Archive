### Post by MrWES on 2008-11-01
I don't know if you're using Thunderbird; but you could run trayed in the notification area with 'Alltray':

alltray thunderbird %u &

Then just add it to your sessions to run when you boot. Built in notifcation, and somewhat cleaner running it from the notification system tray.
