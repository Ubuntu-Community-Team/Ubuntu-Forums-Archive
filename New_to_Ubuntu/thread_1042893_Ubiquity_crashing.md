### Post by yeats on 2009-01-18
It sounds like you have enough RAM to run Xubuntu, but not the installer.  Try using the Alternate installer CD, which runs like Ubiquity, but is all text and no graphics.

[http://mirror.anl.gov/pub/ubuntu-iso/CDs-Xubuntu/8.10/release/](http://mirror.anl.gov/pub/ubuntu-iso/CDs-Xubuntu/8.10/release/)

I think it'll work fine for you.  I haven't used xfce for my main environment, but a friend loves it and says it works well on boxes with lower resources like yours.

Good luck. :-)
