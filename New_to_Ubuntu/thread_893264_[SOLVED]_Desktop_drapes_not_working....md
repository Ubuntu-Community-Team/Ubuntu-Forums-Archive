> **kamitsukai said:**
> OK I tried looking for desktop drapes in the main menu and couldnt see it so I ran the command "drapes" (is that the right command?) but nothing happens:confused:

When desktop-drapes runs, an icon (a computer monitor with drapes hanging on the display) appears on my top panel.  Right-click on this icon and I can set preferences, see Help, etc.  A left-click will cause the desktop background image to change.

Desktop-drapes is set up on my computer to start automatically when the system starts up.  I can't remember if I set that up myself, or if the installation of the app did it.  You should check, under **System > Preferences > Sessions > Start-up Programs**, if drapes is set to run on start-up.  If not, put it in there yourself.  Yes, the correct command is "drapes".
