### Post by northern lights on 2008-08-05
Unfortunately, there is no Linux support for that feature.

Intel's Turbo Memory is not what it promises to be though, anyway. It's hardly more but an integrated flashdrive. Modern flashdrives are warranted for maximum a couple hundred thousand read-write-cycles. It's only advantage are low access times when reading (and when reading only).
I'd like to believe it's a marketing gag for Vista.

If you're lucky, you'll be able to access it via the SATA/PATA controller, but I don't think you can. If so, use it as, I dunno, storage for files you access often.
