I don't use Transmission so I can't help you directly, but if you want to have full remote control over your torrenting, check out rtorrent.  Sure, it's not a gui app, but once you get used to it you'll find it very easy to use.  Add in screen + ssh and you've got a very nice setup.

Check out this page:  [http://kmandla.wordpress.com/2007/05/02/howto-use-rtorrent-like-a-pro/]("http://kmandla.wordpress.com/2007/05/02/howto-use-rtorrent-like-a-pro/")
