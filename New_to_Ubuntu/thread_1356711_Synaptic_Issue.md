### Post by Temposs on 2009-12-16
Whenever you install something with Synaptic, you can find out where it installed things by going to the entry for it in Synaptic, right click, go to Properties, and then click the Installed Files tab. That shows all the installed files for that package. You should be able to find the relevant files there.
