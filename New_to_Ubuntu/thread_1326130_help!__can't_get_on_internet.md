Uhm... have you tried  

System>>>Preferences>>>Network Connections

Or.. 

System>>>Preferences>>>StartUp Applications 

If network manager isnt there the code on mine is

"Network Manager"
Code
"nm-applet --sm-disable"
Comment
"Control your network connections"
