### Post by neurostu on 2008-04-29
If you are new to linux I would agree with Denvercobra, Gnome is much easier to get used to. You can actually have Gnome and KDE on the same computer at the same time, its great b/c you run the same operating system with two differnt GUI's.

I personally haven't used KDE4 but look around and I'm sure you'll find the terminal, if not you can use CNTL+ALT+F1 to open up a virtual terminal. There you can login and install gnome with
```

sudo aptitude install ubuntu-desktop

notice I used aptitude not apt-get, aptitude will grab all the recommended packages that gnome needs, but apt-get won't

```
then go back to the GUI with CNTL+ALT+F7 log out, then on the login screen there should be an option (bottom left corner?) to select your session, chose gnome and then you can use gnome.
