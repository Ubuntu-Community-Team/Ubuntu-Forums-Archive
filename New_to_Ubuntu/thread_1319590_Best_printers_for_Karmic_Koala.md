I would have to agree that HP printers work great. I have an HP Officejet 6500 all in one, that is cheap and works really well. ### Post by HotShotDJ on 2009-11-08

[http://www.openprinting.org/printer_list.cgi](http://www.openprinting.org/printer_list.cgi)+1

That is where I go to check on printers.  I make a list of make/models that I would like, and then narrow that list using the openprinting databae.

I've had good luck, "out-of-the-box," with HP and Brother printers.  I've had the worst experience with Canon printers.  REMEMBER, multi-function machines are a whole different arena.  If your considering one (and some work wonderfully, although I have no personal experience with them) you'll need to do some extra research before purchasing to make sure that all of the functions (printing, scanning, faxing, etc.) will work.

Just to be sure that I'm being very, very, clear... the problem with some of these printers and/or multi-function units is NOT that Linux doesn't support them, it is that THEY don't support Linux.
