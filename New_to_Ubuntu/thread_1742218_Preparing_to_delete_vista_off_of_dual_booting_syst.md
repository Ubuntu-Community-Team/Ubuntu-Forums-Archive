### Post by gstla on 2011-04-28
Vista should be an NTFS partition.  Just so you're clear you should be very specific here so people can help you to the utmost potential.  Are you going to keep running windows, or switch to full-ubuntu, that was unclear to me.  

I've heard that Grub (The Ubuntu Bootloader) and the Windows Boot Loader don't play around together (Windows tends to like being installed first), Ubuntu second.  It's possible to install windows after Ubuntu (if Ubuntu is the "Primary"/First OS, but I suspect it may be a bit difficult for a beginner.

Also, If you're running 11.04 You should experiment with the desktop environments at the login screen (Unity) is the new one, and Gnome is the old one (for 10.04/10.10).  I'm preferential to Gnome, but I'm going to do my best to learn 11.04 Unity.
