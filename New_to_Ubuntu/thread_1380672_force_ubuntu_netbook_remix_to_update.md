### Post by Temposs on 2010-01-14
Go to the System section of the main menu and scroll down to the Administration part. You should find Update Manager program there. If you run that, and click Check, it will find the very latest updates for you, if there are any. Then you can click "Install Updates" to install any updates that appear.
