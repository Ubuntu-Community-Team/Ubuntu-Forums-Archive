### Post by Vaphell on 2011-01-14
no, it's a text editor with syntax highlighting, nothing more. To get the result you need another tool to get the program running (eg compile with gcc/g++ for c/c++, use python for .py files). You can also install some IDE for the language of choice that will have plenty of convenience features and allow you to run programs directly.
