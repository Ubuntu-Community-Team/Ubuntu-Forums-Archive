### Post by warfacegod on 2010-04-13
I have run into "issues" (as in I don't remember specifically what) with my Places menu and having the actual folders on my Desktop. I *think* it was that putting the folders on the Desktop actually created separate folders than the ones that where in the Places menu. So when I would put a file into FolderA on the Desktop it would not be in FolderA in Places. Something like that anyway. I fixed this by using Links. Links can have files Dragged and Dropped into them as though they were the actual folder.
