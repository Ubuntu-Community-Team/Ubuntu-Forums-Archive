Its nice and easy- you could install multiple window managers without bothering your other desktop managers.

I must point out though- if you install another window manager, it will also install programs that are related to it, just like you have gedit in gnome, you will have kate in kde.

If that wont bother you, just install kubuntu-desktop from synaptic.
