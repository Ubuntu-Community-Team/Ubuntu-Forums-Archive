### Post by ankspo71 on 2009-11-20
Hi,
I am using Ubuntu 9.10 and I have the darkroom theme installed by default. Select the theme you are trying to work with and click on the customize button. From there click on the 'Controls' and/or 'Window Border' tabs and select darkroom to see if it will make that message go away. Sometimes you might have to log out / reboot / or reopen the appearance preferences to make those messages go away.

Emerald themes aren't actually a requirement, they are additional window border themes mainly used for 3d effects. I think the required darkroom theme it states is a metacity or gtk theme requirement, which should already be installed on your system.
Hope this helps.
James

Edit, I am a little confused by the original post because the hardy black theme is a gdm theme (the login screen theme).

Be carefull on which guides you follow to install themes, customisations, etc...
If you have Ubuntu 9.10 Installed, only follow guies for Ubuntu 9.10.  Because the operating system changes so rapidly, It is prone to have dependencies fixied in newer versions, which were not available in older versions, thus following an old ubuntu howto could essentially break what you are trying to install.

Ubuntu 9.10 supports GTK 1.X and 2.X, Compiz, GDM and Metacity themes natively.

Try any of those sections out: [http://www.gnome-look.org/](http://www.gnome-look.org/)
