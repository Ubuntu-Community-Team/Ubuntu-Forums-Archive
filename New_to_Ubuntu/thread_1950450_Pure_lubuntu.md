### Post by Peripheral Visionary on 2012-04-01
Both Gnome and LXDE use GTK. So you'll find some "Gnome stuff" in Lubuntu and Xubuntu, simply because the user experience is better even on low-spec machines with some of that "Gnome stuff," or because there is no "non-Gnome equivalent," or because certain applications depend on some "Gnome stuff." 

It's not polluted nor weighed down, it's just there because some applications depend on it and/or work better with it.
