### Post by Geoff918 on 2009-08-01
You can add Firestarter with Applications -> Add/Remove Programs, search box "Firestarter" (make sure all available applications is selected in the drop down) and then check the box to the left of Firestarter. Apply changes, yadda yadda, it's downloaded and installed.

Firestarter will not start everytime you boot, but this isn't necessary anyway it configures the Ubuntu firewall and that will be active on startup if you select that option. I use firestarter and I do start it each time just so I can see active monitoring, but that's not necessary for the protection.

Hope this helps.
