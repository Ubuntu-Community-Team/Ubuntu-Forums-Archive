### Post by limey_rick on 2010-04-27
You can't install inside windows unless you are using a special distrubution called WUBI [http://wubi-installer.org/](http://wubi-installer.org/).

I can't tell from your post, but you may have the Live CD and to install Ubuntu, you need to boot your with the CD in the computer and set the BIOS to boot form the CD/DVD drive first.

You then run Ubuntu just from the CD and try it out. If you want to install, you can reboot and select Install rather than Try, or double click on the Install Icon you get when you Try ubuntu. 

When you install, you will need hard disk space for Ubuntu, so it will either prompt to resize your windows partition or if you have more than one disk, you can select that. Note that you won't want to erase the disk where windows is installed. 

If in doubt, take a full back up first.

When you've complete the install, a new boot loader may now start when you turn on the PC, so if you want windows to start, you will have to select it from the menu.
