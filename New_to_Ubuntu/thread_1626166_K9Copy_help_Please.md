### Post by inkrypted on 2010-11-22
K9 Copy is about as easy as it gets simply choose input DVD and output ISO. To burn the ISO you made with K9 Copy choose K3b and choose more actions then burn image and choose your ISO file that you created with K9 Copy. DVD-r or DVD+R either will work.

### Post by inkrypted on 2010-11-22
If you choose input DVD and output DVD you will get a copy. If you choose input DVD and output MPEG or AVI you will get a file that can be played back on your PC but will require another program to transform it back to a DVD.
