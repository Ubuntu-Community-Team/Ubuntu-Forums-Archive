### Post by grahammechanical on 2012-09-01
We set a password during the installation process even if we are going to set Ubuntu to automatically log in. Why?

It means that only you (if you are the only person who knows that password) can make important changes to system settings.

If anyone tries to make significant changes to the system they will be prompted for a password. You can enter the password and make those changes. Others will not be able to do that.

The power of that password only lasts as long as the task that you are using it for. So, that even though you are listed as having administrator privileges, when you log in you work as a ordinary user and only become the administrator when you enter that password.
