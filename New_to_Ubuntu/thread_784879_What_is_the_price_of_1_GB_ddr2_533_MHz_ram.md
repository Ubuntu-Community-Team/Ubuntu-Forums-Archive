### Post by prshah on 2008-05-07
> **sharks said:**
> What is the price of 1 GB ddr2 533 MHz ram? (in India)

Rs 1100, 667Mhz, which will automatically adjust to 533 if required by the system FSB. Little better quality ram (800 FSB) will be about Rs. 1250, but I have faced no problems with either.

However, 2 x 512Mb (Rs. 1450) is recommended if you have a Core2 Duo, to enable dual channel memory mode.

Dual channel memory mode will allow simultaneous memory access across both clips. It will work only if both clips are the same (FSB, capacity, and in some cases, brand).
