### Post by benj1 on 2009-04-08
assuming you have python installed ( it is by default on ubuntu)

cd to the correct directory and either: 
python script.py

or to make it executable:

chmod +x script.py

### Post by ibuclaw on 2009-04-08
To run it:
```
python **script.py**
```

or

Make the script executable
```
chmod +x **script.py**
./**script.py**
```
