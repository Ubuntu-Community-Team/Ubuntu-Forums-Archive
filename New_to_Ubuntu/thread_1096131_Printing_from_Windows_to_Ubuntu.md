### Post by starcannon on 2009-03-14
In Ubuntu, set the printer's sharing preferences, it will likely add some Samba packages.

Install the printer's windows drivers in VB-windows, then add the printer as per usual for a network printer in windows. I'll do the same on my end if you get hung up, I don't deal with this often enough to have the screens memorized, but am happy to walk through it with you.
