### Post by baumanno on 2012-03-04
It should be as simple as:
[LIST]
[*]Right-click on text-file you want to pen by default in leafpad
[*]Select "Open with..." --> "Open with other program" (or similar, I'm not on an English system ATM)
[*]In the bottom half, you should activate the checkbox that suggests using the selected application for all other files of this type
[*]In the App-list, select Leafpad
[/LIST]

Any help?
