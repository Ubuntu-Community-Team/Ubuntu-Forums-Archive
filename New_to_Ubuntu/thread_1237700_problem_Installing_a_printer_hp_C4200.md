### Post by pedro3005 on 2009-08-11
Root is the system administrator and is allowed to do almost anything. You're probably not logged on as root, and if you're on the command line, if you see an '#' means you're root. $ means you're a normal user. If you're root and want to become a normal user run 'su user' replacing user with your user name.
