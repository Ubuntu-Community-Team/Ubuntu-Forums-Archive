if you are programming and want to run your source codes then you need this package, build-essential is just a meta package to install all necessary programming tools like gc++ and others.
build-essential does not install an IDE, if you want an IDE install from software center maybe Geany or Eclipse or any other found on software center, Geany or Eclipse are the most widely used IDE's on linux for programming and to run your source code directly if those tools from build-essential are installed.
