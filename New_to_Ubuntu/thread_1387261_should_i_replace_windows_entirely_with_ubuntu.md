### Post by TBABill on 2010-01-21
I would keep Windows just in case. There's no reason to get rid of it just to say you did. Having it as a backup could assist if you ever run into issues of incompatibility with Linux, such as a driver issue with some hardware (sound, video, storage), or even with a particular software package you may want to run that Wine doesn't work well with. 

I did wipe Windows from my laptop completely. I keep distro hopping (that's a whole other topic for sure) because I am still seeking that perfect distro for me. However, I have 4 other computers, 3 of them still running Windows only and the other running only Mandriva. I keep Windows on those machines so I have the functionality of either environment. I'm not anti-Microsoft because Windows has its usefulness sometimes, but I prefer Linux by far. It's just faster, easier to run, boots faster and crashes less (not much at all for me).

Good luck.
