### Post by petedct on 2008-08-17
> **rixtr66 said:**
> is it possible to set ubuntu to respond with one click,if so how?
Rick

In Nautilus go to edit > preferences. Select the "Behavior" tab. At the top under "Behavior" select "Single click to open items".
