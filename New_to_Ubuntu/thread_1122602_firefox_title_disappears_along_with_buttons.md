### Post by Flecko on 2009-04-11
I remember this being a problem for me awhile ago. The trick to getting it fixed is this. Hold ALT and click on the firefox window and drag it down so you can see the menu bar. Now, click the un-maximize button and manually size the window down so that it is only taking up a fraction of your screen. Close it, then re-open it and then you can go back to using it maximized.

I remember this being some sort of bug in firefox writing its configuration files upon close. I haven't had it since 8.10, but I see thats what you're using. Shoulda been fixed though.

Best of luck.
