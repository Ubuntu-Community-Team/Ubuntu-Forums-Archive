### Post by 3rdalbum on 2009-11-08
The Nvidia 9600 desktop version works perfectly on Ubuntu, I expect the mobile version will be fine too. Usually Nvidia problems are with older chips, that use archaic drivers.

Other than that, the only way you can know is by trying Ubuntu. The Desktop CD runs a live session from the CD - obviously not everything will work from this CD (for instance, anything requiring a restart). If things work as well as you'd expect from a live CD environment, you can double-click the Install program.
