### Post by newb85 on 2013-11-12
I hope you didn't pay a lot for the CD.  For one, Ubuntu is free.  For another, Ubuntu 13.04 doesn't fit on a CD, unless it's the minimal install (with no graphical environment installed by default).  Moreover, 13.04 is non-LTS, no longer the latest release, and will only be supported until January.  (You should really be using 12.04 LTS or 13.10.)

I second Morgaes' comment.  If that is indeed your machine, you should be running something lighter like Lubuntu.
