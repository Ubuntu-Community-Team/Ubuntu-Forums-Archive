### Post by stoogiebuncho on 2009-03-01
Try going to your home folder and deleting the .vlc folder (this is the folder that stores your personal settings).

It is a hidden folder, so you may need to press Ctrl + H in order to see it.

If that doesn't work, uninstall VLC, check again to make sure the /home/.vlc folder is deleted, and reinstall.
