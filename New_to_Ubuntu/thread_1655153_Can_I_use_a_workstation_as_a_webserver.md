### Post by Penguin Guy on 2010-12-29
You could use a desktop computer as a server, but here are a few things you should think about first: 
[LIST]
[*]Security (if you don't know what you're doing you could open up vulnerabilities to hackers)
[*]Keeping it running (leaving computer on 24/7, backup power supply)
[*]Power (can the desktop computer handle the workload?)
[/LIST]
Running a server is a big deal that requires a lot of work to do well. If you want your browser game to be hosted professionally and reliably, a home-made server will not do.
