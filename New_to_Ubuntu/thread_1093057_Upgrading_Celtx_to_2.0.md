### Post by bolucpap on 2009-03-11
It isn't where you moved it? 

For the record, if you are unfamiliar with Command Line Interface commands, the _mv_ command **m**o**v**es a file or directory. So your directory should be under /opt/celtx now. The _sudo_ command tells the system that you are an administrator and to carry out the move even if the directory/directories in question belong to other users and your user does not have the right to modify them.
