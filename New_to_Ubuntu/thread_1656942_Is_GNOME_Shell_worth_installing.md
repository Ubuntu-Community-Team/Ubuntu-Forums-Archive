### Post by cariboo on 2010-12-31
The version of gnome-shell in the repositories is really out of date, so the only two ways you can install an up-to-date version, is by compiling it yourself by following the guide [here]("http://ubuntuforums.org/showthread.php?t=1603874"), or from the [ricotz testing ppa]("https://launchpad.net/~ricotz/+archive/testing"). The ppa version is not very reliable at this moment, as it crashes if you look at it sideways. :) This is from personal experience, as I have a system specifically setup for it, so far today it's crashed 8 times, twice I've been able to restart it, but the other 6 times I had to reboot. it's a good thing it isn't on my main system, or I'd never get anything done. :)
