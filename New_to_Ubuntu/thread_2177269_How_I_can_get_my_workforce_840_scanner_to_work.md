### Post by gordintoronto on 2013-09-28
Reading this might help you: [https://help.ubuntu.com/community/ScanningHowTo](https://help.ubuntu.com/community/ScanningHowTo)

To the best of my knowledge, you will never be able to initiate a scan by pressing a button on the scanner.

It never hurts to mention what version of Ubuntu you are using; if you don't say, I assume 32-bit Ubuntu 12.04, and might give instructions which don't work on the version you are using.

You might need to install Xsane and/or simple scan.

I also found this interesting page by using Google: [http://www.hamrick.com/vuescan/epson_workforce_840.html](http://www.hamrick.com/vuescan/epson_workforce_840.html)
