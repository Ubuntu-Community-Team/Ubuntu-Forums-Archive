> Is there any way to install this on 64-bit? Sorry if it's a stupid question; I'm a total newbie (refer to my stats at left).

As long as you have installed your cursor theme into ~/.icons (a hidden directory in your home directory called ".icons", the '.' makes it hidden), then you can select your cursor theme through the appearance settings in Gnome.  (assuming that you use Gnome)  Just click the Customize button on the Theme tab.  Then the Pointer tab of the window that pops up.

To install your icon theme into ~/.icons, you usually just need to extract the archive file you downloaded to that directory.
