### Post by zombifier25 on 2013-08-02
You can upgrade your 10.04 to 12.04 via the Update Manager, but beware that the upgrade process may fail. You can also install from scratch, but remember to backup your home folder. It contains all of your personal configs, settings, etc.

Also one thing to note: Newer versions on Ubuntu now use a new GUI, called Unity. It's quite different from the old GUI, so you'll need some getting used to. Tutorial video here: [https://www.youtube.com/watch?hl=en-GB&v=xA9EHaNc2VI](https://www.youtube.com/watch?hl=en-GB&v=xA9EHaNc2VI)

If you don't like the new GUI, you can always install the old one by installing the package gnome-panel.
