### Post by leunam12 on 2015-11-17
When you go to windows press WINDOWSKEY + R and type ncpa.cpl and press enter. Then right-click the wireless network connection icon and click "Diagnose" on the pop-up menu (or "troubleshoot" if available) and go through the diagnostic process. If the problem is not solved this way. Go to the manufacturer's webpage and download the wireless driver for your computer and re-install it. See if that solves your problem. I don't think it is Ubuntu related.
