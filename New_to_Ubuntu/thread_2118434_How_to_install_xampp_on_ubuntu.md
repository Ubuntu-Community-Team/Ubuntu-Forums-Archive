### Post by blackbird34 on 2013-02-21
Ubuntu 9.04 hasn't been supported since october 2010. An upshot of this is that the main repository servers get taken offline.  I suggest you upgrade to (X)ubuntu 12.04 LTS to have a more up to date system. (it is supported for the next 4 years so very little hassle will be involved).

If you really really want to stay on Ubuntu 9.04 for some reason read this: [http://askubuntu.com/questions/91815/how-to-install-software-or-upgrade-from-old-unsupported-release](http://askubuntu.com/questions/91815/how-to-install-software-or-upgrade-from-old-unsupported-release)
