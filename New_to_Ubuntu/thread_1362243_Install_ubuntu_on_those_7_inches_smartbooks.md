### Post by 3rdalbum on 2009-12-22
128MiB of RAM is not enough to run Gnome or KDE. Probably not enough to run XFCE on top of the Ubuntu base either.

Also you'd have to download the ARM version (Armel) of Ubuntu, not the regular i386 version.

And I don't know if that machine has any sort of lock against installing a different operating system.

Invest your money in an Intel Atom netbook or an ARM-based netbook that runs a popular Linux distribution.
