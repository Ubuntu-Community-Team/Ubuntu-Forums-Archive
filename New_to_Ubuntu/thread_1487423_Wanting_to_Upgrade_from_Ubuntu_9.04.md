### Post by Linux_junkie on 2010-05-19
First of all using your LiveCD before attempting to install check that it runs on your laptop by running the LiveCD.  If everything works (but may run slowly) you should be ok to install it on the hard disk and providing you have enough disk space you can duel boot with 9.04.

The problem you highlighted about the screen may mean the graphics card may not be supported by 10.04.

This is why you should always run from a LiveCD first.
