### Post by acrazyplayer on 2011-10-02
I'm assuming that GW is played via Wine, if so then you can try to "emulate a virtual desktop", just go to the wine config in the menu entries and from there choose the "graphics" tab and you will see the check box for virtual desktop. You can change the emulated size of the desktop to be almost full screen. I have used this method for many games and you get used to it after just a few minutes. good luck
