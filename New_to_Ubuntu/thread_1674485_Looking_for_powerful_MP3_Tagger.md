I have tried Ex Falso and EasyTAG, and neither of these does what I am looking for.

My feature wish list is:

1. Ability to embed album art into the file.
2. Convert tags and file names to title case.
3. Rename files based on tag.
4. Easily removed values for tags I want to leave blank.
5. Show only user values for certain tags, i.e. Genre, so the options are only what I have pre-defined.

In Windows, I use a freeware program called MP3Tag ([http://www.mp3tag.de/en/index.html](http://www.mp3tag.de/en/index.html)) which does all of this, but it is seems to be Windows only. I really like the GUI customization options it has.
