### Post by halitech on 2009-04-04
I pop the dvd in the drive, open up K9copy, select the input device as the drive with the dvd in it, select the output as ISO file and then click the button for DVD(4th button from the left at the top) and away it goes. usually takes about 30 minutes to create a 4.3gig "minature" iso of the original. 

Just make sure you go into Settings - configure K9copy and set the dvd size to 4.4 gig on the dvd tab first. You can also decide if you want to auto burn with k3b or not. (I usually don't so I can change the label info before burning)
