### Post by blueridgedog on 2009-04-25
The sensors program should help.  It should be installed, and you can set it up with 
```

sudo sensors-detect
```

Installation and setup are covered here:

[https://help.ubuntu.com/community/SensorInstallHowto](https://help.ubuntu.com/community/SensorInstallHowto)
