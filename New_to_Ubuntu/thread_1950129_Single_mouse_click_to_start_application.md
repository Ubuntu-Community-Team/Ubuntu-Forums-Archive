### Post by ashhab2010 on 2012-03-31
You can control whether you single-click or double-click files, whether folders are opened in new windows, how executable text files are handled, and the trash behavior. In any file manager window, click Edit &#9656; Preferences and select the Behavior tab.

[https://help.ubuntu.com/11.10/ubuntu-help/nautilus-behavior.html]("https://help.ubuntu.com/11.10/ubuntu-help/nautilus-behavior.html")
