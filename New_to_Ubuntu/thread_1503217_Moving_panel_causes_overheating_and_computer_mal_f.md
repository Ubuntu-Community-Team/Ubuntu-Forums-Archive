### Post by ankspo71 on 2010-06-19
Hi,
How is the computer doing now? Is the command bar an applet you added to the panel? Does it happen if there is no command bar on the panels?

If it acts up again, keep an eye on any processes that might be using too high of cpu usage. It's best to do this by running the command 'top' in the terminal. If the cpu usage is pegged at 100% it can cause overheating on certain computers, and if the cpu usage is stuck there it can also cause the desktop to freeze up and get stuck too. I suspect that is what could be happening.
