### Post by cariboo on 2010-09-28
You have to run nvidia-settings as root in order to save your settings from one session to the next. Press Alt-F2 and type:

```
gksu nvidia-settings
```

Make your changes, then click the save button.

### Post by cherryredz on 2010-09-30
Ok think I've found a fix that works (for me anyway)

I've restarted 5 or 6 times and each time monitor resolution has been at the desired 1920 x 1080.

Found solution in another thread, basically went system-> preferences -> monitor, then answered no to the question "it appears that your graphics driver does not support the necessary extensions to use this tool. Do you want to use your graphics driver vendor's tool instead?"

In the screen that appeared I set my preference to my desired resolution (the existing resolution displayed was the one that was present on every restart)

Hope this prove useful to anyone else with a similar problem
