### Post by VH-BIL on 2010-06-24
System > Preferences > Main Menu

You can edit your menu there. Add/Remove Applications is an older Ubuntu isn't it? I can't remember what the application executable was called but Ubuntu Software Center is at "/usr/bin/software-center" and the icon is "/usr/share/icons/hicolor/48x48/apps/softwarecenter.png"

hope this helps.
