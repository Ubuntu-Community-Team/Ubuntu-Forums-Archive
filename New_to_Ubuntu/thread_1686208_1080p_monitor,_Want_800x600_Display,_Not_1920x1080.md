### Post by Hedgehog1 on 2011-02-12
Making this bigger so it is easy for you to read:

[SIZE="3"]If the display you are using is an HDTV, the display may be forcing your card to output at its preferred screen resolution.

HOWEVER - There is a fix.

Menu from System >> preference >> Appearance.

Select the Fonts tab, and make the fonts your system uses larger.

This gives you the full screen detail for photos and video, but allows you to read the text, too.[/SIZE]
