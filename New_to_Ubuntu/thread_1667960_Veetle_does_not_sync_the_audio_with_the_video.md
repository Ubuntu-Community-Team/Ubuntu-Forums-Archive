### Post by koskal on 2011-04-06
Ok guys I think I found a solution :
Don't change anything in veetle_vlc folder. Install padevchooser and run it.
Left click in it's icon on the panel and in the default server choose other and leave it blank.
That's it, now veetle plays as is should  and also this resolve problems beetween skype and pulse audio.

Enjoy.
