### Post by amjjawad on 2011-09-13
As already posted, YES you can install Software Center in Lubuntu (it's not included by default).

IMHO, it's better to learn how to use the Terminal (LXTerminal in Lubuntu) because it's much faster and if that was too complicated then Synaptic is a great choice too.
After all, Synaptic or Software Center are just Graphical Front End and one can do the same with Terminal.

I sometimes use Synaptic when I can't remember the right commands or don't even know them.

Enjoy using Lubuntu, it simply rocks :)
