Yes it can. I think the netbook edition is pretty nice- it places an emphasis on dealing with a small screen, both in launching apps as well as managing them. If you need to use the netbook as a normal desktop for a session, then simply login using Gnome as a session. Netbook edition contains by default the netbook interface, but that doesnt mean you cant use a normal desktop.
