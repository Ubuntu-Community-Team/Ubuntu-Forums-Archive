### Post by mikewhatever on 2010-09-28
Things are explain a bit more in **man ssh-copy-id**. Basically, the command copies the public key from the computer on which it was generated to a 'remote computer' over ssh.
The stuff that goes into ~/.ssh/id_dsa.pub is not code, it's a key, a long string of random characters, if you like, and it is generated by the **ssh-keygen -t dsa** command. You could manually copy the text from ~/.ssh/id_dsa.pub to ~/.ssh/authorized_keys on another computer, but the **ssh-copy-id** command automates the process.
