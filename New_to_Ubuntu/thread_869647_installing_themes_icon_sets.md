### Post by ChompTheMan on 2008-07-25
Well, the icon set included with that suite is only a preview and isn't actually available for download yet.  This isn't exactly just a KDE theme, it's a suite(collection) of what makes up the creators desktop.  The Beryl(Compiz) theme would be opened in Emerald Theme Manager.  The Theme Manager theme(which includes fonts, colors, and the kicker background) would be opened with the Theme Manager, which can be found in Kcontrol(not sure about System Settings, I don't use it).  The K menu theme would be opened with KBFX.  I'm not sure how you apply the shutdown background and Domino config, but I am sure you can figure out the wallpaper.
