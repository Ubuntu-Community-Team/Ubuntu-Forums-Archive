### Post by Achtung on 2008-05-09
You should read this : [https://help.ubuntu.com/community/Installation/Netboot](https://help.ubuntu.com/community/Installation/Netboot)

It's pretty straight foward if your computer can boot from a network. Check your computer's bios boot sequence. If it can't, you can always make a boot floppy, assuming your computer has a floppy drive. Both possibilities are covered in the documentation provided above. If you can't boot from a network and don't have a floppy drive, so long netboot idea.
