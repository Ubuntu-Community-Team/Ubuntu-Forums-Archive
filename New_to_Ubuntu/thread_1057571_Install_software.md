### Post by jerome1232 on 2009-02-01
So it's a shell script, just give it permissions to be executed then execute it.

```
chmod +x /path/to/script.sh
./script.sh
```

Or the gui way

right click the file, select properties, click the permissions tab, check allow exectuting this file as a program.

Then double click the .sh file, tell it to run in terminal, let the magic happen.
