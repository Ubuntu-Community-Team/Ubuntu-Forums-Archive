### Post by bamim2 on 2010-01-21
I'm using Gnome & I can't seem to figure out how to set which application launches a specific file type. Each time I install something new, Ubuntu uses the new app as the default launcher (I'm sure that's not the correct term for it) to start that file type. 

For example, how do I set what app is used to launch a .doc (Word) document?

---

### Post by lotharmat on 2010-01-21
right click on the file then choose properties then select the 'open with' tab and select the program.

That's all there is to it!

---

### Post by sisco311 on 2010-01-21
> **bamim2 said:**
> 
For example, how do I set what app is used to launch a .doc (Word) document?

Right click on a .doc file -> Properties -> Open With tab and select your preferred application to open .doc files.
