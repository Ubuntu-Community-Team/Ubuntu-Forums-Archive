### Post by R Clemons on 2009-05-08
Use a livecd, you should even be able to use your Ubuntu cd.  Open up Gparted, delete the Ubuntu partition and resize ntfs so that it utilizes it.  You may have to use your windows cd afterwards to rebuild the mbr
