Qt is not a software. It is a development library.
It could also be installed by apt-get just like other applications.
But you could install packages from synaptics too which is a graphical interface.
And a new way for installing applications is there which is accessible
from Applications > Ubuntu Software Center menu item.
Anyway you need to type your password (but if you enter password one time, then you could install another package within few seconds since gnome keeps privileges for you).
