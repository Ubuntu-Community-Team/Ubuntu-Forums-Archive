Edit:

I did some more searching around the web and found a solution to the problem. I used UFW instead of Firestarter, and then I used the tut and codes found on this page [http://blog.bodhizazen.net/linux/firewall-ubuntu-desktops/](http://blog.bodhizazen.net/linux/firewall-ubuntu-desktops/)

Next, when you do the code for allowing connections, be sure to add the port: 3074

That is important because that is what the Xbox uses to connect to live. If I knew how to put the code straight right up here in one of those little boxes for you to copy I would. So follow the instructions found on that site and you will be set basically. God bless you all who had the same problem as me with it.
