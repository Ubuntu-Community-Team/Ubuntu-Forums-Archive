### Post by yeats on 2009-03-26
There's really not much that compares, in my experience.  People will suggest Bluefish and others, but there's not a Linux program with all the features that Dreamweaver offers.  If you have the memory/disk space, you might consider running XP in VirtualBox (which I prefer to dual booting).  Just a thought.

### Post by mrjohnston on 2009-03-26
I'll second that.  There are some coding programs and some poorer WYSIWYG editors like NVU, Composer, and OpenOffice if your digging, but nothing comparable.  

I run Virtualbox with XP for it.  Haven't found a better way.  Lets me use Photoshop, Office 2007 and Dreamweaver if the need arises and is pretty darned fast.  If you use programs like Dreamweaver a lot you might want to look into Virtualbox with the Seamless mode enabled to merge them easily.
