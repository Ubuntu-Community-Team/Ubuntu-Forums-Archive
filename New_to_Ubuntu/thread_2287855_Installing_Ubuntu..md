### Post by yancek on 2015-07-22
There are numerous tutorials on this such as the one at the link below.  If this doesn't give you the info you want, just do an online search for dual booting Ubuntu and windows8.  Windows 8 is probably installed using UEFI to boot so pay special attention to that.  Also, if you created a partition from windows, you will still have to format it during the installation and windows can't create a Linux filesystem.

### Post by MGrowl on 2015-07-23
Once you have the image, (.iso) burn in on a cd, or better yet make a USB. Go to you PC's BIOS, makes sure it boots from your cd/usb first. and restart. Just follow the instructions after that. Or you can try Ubuntu without installing by booting with a LIVE image. Make sure you read up first though to avoid any headaches.
