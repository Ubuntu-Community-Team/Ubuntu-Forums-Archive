### Post by CelticWarrior on 2020-05-02
If you're using standard Ubuntu it has no "main menu", it has launcher typically in the left side. If that's what you mean just open Terminal, it will show up in the launcher, right-click and the rest should be obvious.

If you're using any other flavor, Ubuntu variant, then most of them have a menu where Terminal is already there. Some use other terminals, e.g. Kubuntu uses Konsole.
