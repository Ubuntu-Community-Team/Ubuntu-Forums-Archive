### Post by audiomick on 2010-01-05
have you looked at gedit?
it is the standard text editor in ubuntu.
start it from the terminal with
```
gedit /path/to/file
```

or
```
gksu gedit /path/to/file
```
for stuff that needs root priviliges.

Then there's
joe
[http://en.wikipedia.org/wiki/Joe%27s_Own_Editor](http://en.wikipedia.org/wiki/Joe%27s_Own_Editor)

nano
[http://en.wikipedia.org/wiki/Nano_%28text_editor%29](http://en.wikipedia.org/wiki/Nano_%28text_editor%29)

vim
[http://en.wikipedia.org/wiki/Vim_%28text_editor%29](http://en.wikipedia.org/wiki/Vim_%28text_editor%29)

emacs
[http://en.wikipedia.org/wiki/Emacs](http://en.wikipedia.org/wiki/Emacs)

and probably more. They are just the ones I have heard about.
