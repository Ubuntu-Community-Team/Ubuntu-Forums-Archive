### Post by evilkastel on 2008-11-04
Offcourse. For running any jar file as an application you should:

1. Make it executable from, the permissions, also give it full permission.

2. Make a launcher in your panel or desktop ;)

3. in th command section while configurating the launcher you put:
```
java -jar /home/thisistheroute/thisisthefile.jar
```
(in case any one wonders, replace thisistheroute with the route and thisisthefile with your file. Remember, ther isn't such thing as foolprof. ;)

4. Close and try the launcher.

Also I'll like to reccommend Vity's  FreeRapid Downloader at:
[HTML]http://wordrider.net/freerapid/index.html[/HTML]
