### Post by jaddi27 on 2012-02-18
Unfortunately there is no way that I know of to add options to the Global Menu in programs. Each program defines its own menu structure, which then appears in the Global Menu in Ubuntu. Someone might correct me on this though.

I think you would be best off trying to ask Google to add this as an option in Chrome for Ubuntu. I know their implementation on Mac OSX has Bookmarks in the top menu, so they should be able to add it in quite easily if enough people ask for it.

Another way to access Bookmarks is via the bar on the New Tab page, which is fairly easy to use. It might get you out of trouble until an option is added to the Global Menu.
