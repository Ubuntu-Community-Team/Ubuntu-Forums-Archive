### Post by aeiah on 2008-04-25
a repository is as it suggests, its a store, or library, of free software supported by ubuntu. you can search the repository for software that you want, select it, and it will download and install automatically. because of the heavy traffic right now its quite a slow process (although usually it maxes out my 11.5mbit connection). this is also affecting your wubi install.

i dont know which torrent you downloaded, but are you sure it is a rar file? i believe that under windows, if you have certain settings set, then winrar will open .iso files. are you sure that isnt the case? either way, the iso file is an installation cd and is different from wubi. wubi will download ubuntu and install it in a windows environment, and once installed, you will have the choice on boot of whether to boot ubuntu or windows. if you want to make use of your iso file (assuming it was a winrar mixup) you need to burn the iso and boot from it, and follow the install instructions. it would be advisable since you're new to read up on installing ubuntu as part of a dualboot system alongside windows so you dont accidentally mess up windows. if that seems too daunting at this stage then the only thing you can really do is sit tight and wait for the servers to improve their speed so wubi can do its magic for you
