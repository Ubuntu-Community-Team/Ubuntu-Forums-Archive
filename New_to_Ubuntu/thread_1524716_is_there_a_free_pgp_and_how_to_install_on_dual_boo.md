### Post by stmiller on 2010-07-05
Yeah no need for free trials or anything. Linux has many better open source and free tools. 


[gnupg]("http://www.gnupg.org/") (gpg) which is more for email signing and such but can be used to encrypt volumes. 

For full hard drive encryption, you should use [Truecrypt]("http://www.truecrypt.org/").
