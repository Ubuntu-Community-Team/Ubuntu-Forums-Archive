### Post by newb85 on 2013-07-11
Well, you could run 
```
dpkg -l nautilus
```
in the terminal.

Or you could pull it up in the Software Center.  (Beneath the list of add-ons it has the version.)

### Post by BBQdave on 2013-07-11

In top bar of most application windows, in the *Help* sub-menu, under *About* - version information for that application :)
