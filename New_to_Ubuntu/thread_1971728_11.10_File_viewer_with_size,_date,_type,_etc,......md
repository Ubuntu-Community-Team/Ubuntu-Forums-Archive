### Post by billyseth on 2012-05-02
I'm running 12.04, but I expect these instructions to work the same for you.

Run nautilus(the default file manager) and go View->List,

then go Edit->Preferences and choose the List Columns tab, there you can choose which columns will be displayed in the list view.
