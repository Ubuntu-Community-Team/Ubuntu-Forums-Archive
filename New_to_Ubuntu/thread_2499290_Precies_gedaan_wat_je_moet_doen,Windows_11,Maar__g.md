Please understand this is an English speaking site.
If not comfortable with English or Google translate, you need to see what help is available in Holland.
[https://translate.google.com/?hl=en&tab=TT&sl=auto&tl=en&op=translate](https://translate.google.com/?hl=en&tab=TT&sl=auto&tl=en&op=translate)

You mention DVD, but Ubuntu Desktop has not fit on DVD for a few years. Most use USB flashdrives.
I used to "burn" many defective DVDs, so converted to flash drives years ago.
If older system perhaps a light flavor?

[https://ubuntu.com/download/flavours](https://ubuntu.com/download/flavours)
Light weight flavors:
Lubuntu, Xubuntu, Ubuntu MATE, Budgie

I like Kubuntu which is lighter than Ubuntu but seems to have a lot of features.
