### Post by ddrichardson on 2009-07-17
Ubuntu Netbook Remix is a USB image (img).  This is a pain to install in Virtual Box (you need to convert to a vdi then mount and boot from the drive).  I have found that running a live environment from USB is quicker and has one massive advantage in that USB doesn't get smudgy fingerprints so can be kept available for longer.

Installation wise, I've seen no difference.
