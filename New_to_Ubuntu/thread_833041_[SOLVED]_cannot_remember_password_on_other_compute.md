### Post by sisco311 on 2008-06-18
In Hardy, after you boot in recovery mode (single user mode), you need to select the *root Drop to root shell prompt* option from the menu. Change the password with the *passwd username* command, press Ctrl+D and select the *resume resume normal boot *option.
