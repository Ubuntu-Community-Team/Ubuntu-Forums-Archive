### Post by yetiman64 on 2010-07-10
Make it as large as possible on your desktop, use screen capture (print screen button on the keyboard), then use gthumb or gimp to crop to size and save as whatever you need it as. 

Will probably be only good for one page at a time for multi page documents though.

Go to Applications--->Accessories--->Take a screen-shot. Grab the opened .doc file, which can be saved by default in .png format.
Hope this helps you.
