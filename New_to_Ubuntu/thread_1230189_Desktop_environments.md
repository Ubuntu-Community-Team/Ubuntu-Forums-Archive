### Post by Katalog on 2009-08-03
You can install them both and use them on the same Ubuntu install. You just have to choose which environment you want to use when you get to the GDM log in dialog. So why not just install both and form an opinion for yourself? And of couse there are WAY more environments than those two as well, like Openbox, Fluxbox, LXDE, Enlightenment, etc. Everyone's got their own opinion since they are a fan of one DE or another, so I think trying them out yourself is still the best option.

Better yet, why not make a bootable Ubuntu disc and try them out that way? Would probably be less hassle.

### Post by Katalog on 2009-08-03
> **waqar said:**
> How do I do that?

You need to have Compiz installed in GNOME (I think it;s already installed by default anyway), go to Prefrences>Appearance>Visual Effects and adjust your themes and transparancies from there. And you need to use Emerald themes rather than regular Metacity themes to get to the pretty little window borders, then it's just a matter of how well your graphics card will be able to handle all these cutesy little effects.

### Post by t0p on 2009-08-03
Gnome's best.  No, kde.  I mean xfce.  I mean...

Try 'em all.  Assuming you're starting from default ubuntu (gnome), type into a terminal:

```
sudo apt-get install kubuntu-desktop xubuntu-desktop
```

Logout.  Now, from the login page you can select which desktop environment to use by clicking on [b]Sessions/b] and choosing from the list.  Gnome.  KDE. XFCE.  Collect 'em all!  :)
