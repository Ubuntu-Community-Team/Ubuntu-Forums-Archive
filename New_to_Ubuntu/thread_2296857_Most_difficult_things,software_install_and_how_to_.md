### Post by grahammechanical on 2015-09-29
First of all do not expect Linux developers to imitate Microsoft. Linux developers have their own ideas about how an OS should work.

The easy way to install applications on Ubuntu is to only install software that is in the Ubuntu Software Centre. Downloading applications from who knows where (Windows style) brings with it the risk of installing malicious code (Windows style). If we want to download an application from some web site then look for a version that has been packaged for Ubuntu. It will have a .deb file extension. Then use the file manager right click and select to open that file in the Ubuntu Software Centre.

How complicated to you want to make this reply? Do you want information about installing from a Personal Package Archive (PPA)? Or compiling source code?

[https://help.ubuntu.com/community/PPA](https://help.ubuntu.com/community/PPA)

### Post by SeijiSensei on 2015-09-29
99%+ of the software you'll want to be running resides in the Ubuntu repositories.  There is rarely any need to download a program from some third-party site and install it.  Using the Software Center is the correct approach.  Every package in those repositories is vetted by the developers at Canonical or Debian, the distribution on which Ubuntu is based.

### Post by mystics on 2015-09-29
Couple things you can do to make things easy (though they won't make things like Windows):
1. Use the Ubuntu Software Center or Synaptic Package Manager to get software. Both of these offer graphical ways to search for and install software and should be easy by anyone's standards. If you know how to use the Apple App Store, Google Play Store, or Windows Store, you should be able to figure these out as well.

2. Follow instructions. Google (or whatever search engine) how to install the software on Ubuntu. You'll either find an Ubuntu-based site or the developer's site telling you what to write into the terminal. Some of these are even designed for you to just copy/paste into the terminal.
