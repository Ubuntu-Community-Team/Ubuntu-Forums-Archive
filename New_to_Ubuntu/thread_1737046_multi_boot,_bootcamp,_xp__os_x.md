### Post by pi3.1415926535... on 2011-04-23
Assuming that partitioning the drive to allow for an Ubuntu installation would not cause problems with either the OSX or XP install, then there should be no problems. I would not think that Ubuntu can be installed through BootCamp. There is Wubi, though that only runs on Windows, so that would result in an operating system within an operating system within an operating system.
