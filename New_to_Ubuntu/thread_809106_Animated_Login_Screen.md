### Post by gilang on 2008-05-27
you can also change the login picture with your own wallpaper...

find one of ubuntu default login folder (such as circle, etc), 
edit the configuration file (the xml, i i'm not wrong) (you must be administrator of course, just use sudo), 
change the default background file name with your own wallpaper or photo name on that configuration file...

enjoy your new login screen...
