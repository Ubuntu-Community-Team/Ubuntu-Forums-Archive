### Post by leemer3567 on 2017-06-26
Hi there

Just wanted to report and mention that the Dell Inspiron 3567 laptop works great so far for Ubuntu 16.04.02 LTS. It didn't boot correctly when I tried Ubuntu 17.04 the latest release as of now. I bought this laptop from Best Buy in the US and it was on sale for $289.99, with tax it came out to $321.

i3 7100u
6GB ram
1TB seagate HD 5400rpm
1366x768 720p TN 15.6 display

This exact laptop isn't listed on the Ubuntu certified hardware list but I can tell you everything has been working smoothly. Trackpad, wifi, keyboard, booting, display, brightness control.

Will report and update thread when something goes bad.

Hope this post helps anyone in the future searching the web for Dell 3567 linux keywords!
