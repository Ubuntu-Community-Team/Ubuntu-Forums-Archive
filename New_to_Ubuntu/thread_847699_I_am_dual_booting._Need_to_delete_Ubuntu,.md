### Post by kolisikepu on 2008-07-02
You need to do fixmbr to replace master boot record and then in Windows, with admin rights, right click on My Computer icon (desktop or start menu) and go to Manage.  Goto Storage container and click on Disk Management.  There you should see all partitions.

Select the Linux partition and delete.
