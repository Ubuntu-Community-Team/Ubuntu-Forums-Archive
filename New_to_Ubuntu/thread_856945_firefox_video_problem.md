### Post by BGFG on 2008-07-12
The flash players that come as a default in ubuntu don't seem to work very well. that or they need a more advanced touch to get it smooth. I uninstalled the swf stuff and installed adobe flash player using Synaptic Package manager.

system>>administration>>synaptic package manager

also ensure that you have the multiverse and universe enabled in your software sources.

system>>administration>>software sources

Edit - the specific package name is flashplugin-nonfree
