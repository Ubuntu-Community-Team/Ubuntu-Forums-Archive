### Post by jacksaff on 2009-10-10
Pick an image and select 'open with gimp' and it will make a bit more sense to you.
In synaptic search for gimp and doc and you should find the packages that will install the gimp manual for you.
If you just want to tweak photos, I'd suggest digikam or picassa as easier alternatives. 
Gimp has basically been designed for (and by) people with professional photographic needs. It has lots of features for batch processing and scripting that most casual users don't really need. It's extremely powerful, but it's kind of assumed that you will be prepared to put in the time to learn how to use it properly - there isn't much hand-holding.

### Post by NightwishFan on 2009-10-10
I have found gimp fairly simple. You just need to learn the concept of 'layers', which takes time. My advice is to play around a bit and you will find that GIMP is pretty great.

A tip I use that may help you is that I hide the two other docks when I am not using them. So if I am painting, I press TAB to hid all the docks, and then when I need to change tools or colors I press TAB again to show the docks. You need to be focused on an image (center window) to hide the docks. This makes things less cluttered, and gives you more screen space.

Fun GIMP tip: Use fuzzy select (in tool dock or tool menu) on a simple image to highlight an outline or letter, and run the Distortion -> Mosiac plugin. This gives whatever you selected a lizardlike texture.

You can also use GIMP to create logos, screenshots, etc. Try the File -> Create menu.
