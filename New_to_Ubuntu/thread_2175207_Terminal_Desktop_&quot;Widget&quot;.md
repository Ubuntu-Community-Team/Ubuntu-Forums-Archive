I found a solution!  "Tilda" does exactly what I was wanting, and even more.

```
sudo apt-get install tilda
```

### Post by coldraven on 2013-09-18
It's maybe not what you want but I just use guake. It is a drop down terminal, Press F12 and there it is. Can be configured by right click and fullscreen with F11.
[http://guake.org/](http://guake.org/)
```
sudo apt-get install guake
```
