### Post by Tamlynmac on 2011-02-23
I've had a similar problem with a specific file type and found by  following the procedure below, I was able to accomplish the task of  opening a file type in the same predefine app every time.

1. Open the file browser (Nautilus) and right click on one of the specific file types in question.
2. Select properties "open with tab"
3. Select the app to be used and remove any other apps in the properties "open with tab" except leaving only your  preferred choice. By setting one file, Nautilus will automatically alter  all other files of the same type.
4. Then close the properties window and open the file.

After that, every file I opened of that type was opened in the app I  chose. I believe it has something to do with providing Nautilus with the  desired preferred app.

Hope this helps & Good Luck
