### Post by mcduck on 2008-12-31
You shouldn't need to do anything to be able to print to PDF. That should work out-of-the-box.

Just click "File"->"Print" and select "Print To File" as the printer. Set the output format to PDF and select the directory where you want the PDF file to be saved, and click "Print".

Anyway, your home directory is located in /home/*username*/, the easiest way to access it (with Gnome desktop, the default in Ubuntu) is to select "Places"->"Home Folder" from the Gnome's menu bar.
