### Post by Matt__ on 2012-01-12
You could try [Lethe](http://sourceforge.net/projects/lethe/), 
it works in the same way as the proprietary software Deep Freeze by Faronics, in that it creates a separate partition with your original OS image that can't be changed...and is reloaded each boot.
Or you could just remove the HDD and boot from a custom live disc/USB/SD created using [Remastersys](http://www.remastersys.com/repository/).
