### Post by su:bhatta on 2013-07-17
Did you install Ubuntu 13.04 previously?
If you have the 13.04 DVD just boot the computer with the DVD in the drive and you will get the screen where you get to choose to install Ubuntu or restore your Ubuntu OS...
If you want chose to install again, then you will lose all previous files, else you can restore your previously installed Ubuntu OS.

If you go for a fresh install, then at the appropriate screen choose' Install alongside Windows' and you will have a dual boot machine , which gives you a screen at the startup, from where you can choose which Os to boot into...

Please give you computer specifications to guide you.
 if you have Windows 8, chances are you have UEFI, which requires a little extra work to dual boot

In case yoju have UEFI, check this page for further guidance...

[https://help.ubuntu.com/community/UEFI](https://help.ubuntu.com/community/UEFI)
