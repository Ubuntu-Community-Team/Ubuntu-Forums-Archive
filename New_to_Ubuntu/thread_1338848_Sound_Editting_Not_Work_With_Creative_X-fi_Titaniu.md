### Post by Zoot7 on 2009-12-02
Support for the X-Fi's in Ubuntu (and Linux in general) only just arrived a few months ago, so it's got a bit to go yet.
One thing you could try is upgrading Alsa to version 1.0.21, which contains the tried and tested X-Fi driver. There's a good upgrade script that automates the process posted here:
