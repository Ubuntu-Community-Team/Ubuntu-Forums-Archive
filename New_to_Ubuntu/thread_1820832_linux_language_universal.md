### Post by Paqman on 2011-08-08
All Linux distros I'm aware of using the same default "shell", which sets the syntax you use in the terminal. What will differ is that other distros use a different security setup from Ubuntu. Where on Ubuntu you prefix commands with "sudo" to carry out admin tasks, on other distros you'd log in as the root user (so no need for the sudo prefix).

Also, other distros may be set up slightly differently under the hood, or use different versions of the same software. While the commands will generally work, it's impossible to say for sure without knowing exactly what commands you're running.
