### Post by Perfect Storm on 2009-08-30
The "~" means home directory (usually). Eg. "cd ~/Desktop" is the same as "cd /home/USERNAME/Desktop"
The "." (dot) means it's hidden file or folder.

You can with the fil-browser set it to show hidden files. If it doesn't exist, just make a new folder in your home with that name (.icons)
