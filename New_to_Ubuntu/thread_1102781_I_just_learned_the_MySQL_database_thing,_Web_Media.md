### Post by James_Lochhead on 2009-03-22
MySQL is a database application that uses the Structured Query Language (SQL). SQL is a heavy duty tool for IT professionals, it is not really for people unfamiliar with databases (it is not really very hard but there is no reason for the inexperienced to use it).  

As for cool browser applications, try Drupal?

Totem is a very good media player and is installed by default in Ubuntu, just make sure you have the package ubuntu-restricted-extras. It has browser plugins as well.

As for an IM, Pidgin is installed by default as well and is a very good IM.
