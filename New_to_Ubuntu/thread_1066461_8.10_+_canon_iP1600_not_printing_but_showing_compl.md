### Post by Mrent on 2009-02-10
I got the printer to print successfully. I did not have the libxml1 package . After installing using 

-$ sudo aptitude update

-$ sudo aptitude safe-upgrade

-$ sudo aptitude install libxml1

then re-linking using 

-$ sudo ln -s /usr/lib/libtiff.so.4 /usr/lib/libtiff.so.3

-$ sudo ln -s /usr/lib/libpng.so /usr/lib/libpng.so.3

-$ sudo ln -s /usr/lib/libxml2.so.2 /usr/lib/libxml.so.1

-$ sudo ldconfig

finally restarted cups

-$ sudo killall cupsd

-$ sudo cupsd


everything worked!
