### Post by t0p on 2010-01-25
Whoa!  Do not install Warty Warthog!  That's a very old version of Ubuntu (not quite as old as XP, but still very very ancient) and has not been actively supported for years.  Which means it's probably horribly insecure.

Download and install a more recent version.  I advise getting the latest, 9.10 (Karmic Koala).  You can get the disk image [here]("http://www.ubuntu.com/getubuntu/download").

### Post by Jose Catre-Vandis on 2010-01-25
Funny

I installed Warty just the other day to see how things fared on newer hardware. There is still an online repository knocking about for it (you'll have to edit your /etc/apt/sources.list though). It's actually quite quaint, and the installation process is seriously different to what we see now with Ubuntu. Didn't recognise my network card though.

t0p is right - try slitaz or crunchbang, or puppy or a command line install of xubuntu and build openbox on top, on your old machine. I have a circa 1995 Dell laptop that runs well with Xubuntu CLI and openbox.
