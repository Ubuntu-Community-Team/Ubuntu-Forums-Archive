### Post by deadflowr on 2016-07-29
gimp works.
In gimp make a new image > File > New.
Set the size and create the blank image.
Then go to Filter > Render > Pattern > Checkerboard.
Then you can toggle the size settings from the default px to mm (millimeter) from there.
Then if need be, crop it or resize or whatever.
Then save it with export.
