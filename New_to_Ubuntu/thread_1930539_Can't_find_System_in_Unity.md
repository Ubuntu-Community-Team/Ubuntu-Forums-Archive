Hi! **BlueAZ**.

If you really want the System>Preferences/Administration Menus back with Ubuntu 11.10, install the 'ClassicMenuIndicator'.

You can edit it with 'Main Menu' just as you could with previous versions, and it works equally well, whether logged in to Ubuntu/Unity or to Gnome Classic.
