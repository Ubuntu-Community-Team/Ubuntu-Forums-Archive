### Post by nmccrina on 2010-05-03
Well, to find grokking the gimp, open [file:///usr/share/doc/grokking-the-gimp/html/index.html]("file:///usr/share/doc/grokking-the-gimp/html/index.html") in firefox.

BTW, if you right-click an installed package in Synaptic, there is a tab that gives a list of all the files the package installs. ;)

I haven't used GIMP for much beyond basic resizing and stuff, but I used the manual for reference and it was helpful: [http://docs.gimp.org/2.6/en/]("http://docs.gimp.org/2.6/en/")
