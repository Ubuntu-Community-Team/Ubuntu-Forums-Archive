### Post by uStackable on 2010-12-15
First off, as everyone says, make sure your system BIOS is set to boot from CD rather than your hard drive, if that is the case and it still isn't working, then make sure the extension on your burned copy of Ubuntu is a .iso, not a .data, if that is the case, then redownload it, as the .ISO is bad, I've never used CD's to boot from, they always seem to cause problems, I always use USB drives, and they normally work fine. Try that if all else fails, if you have one laying around, just make sure it's 2GB+.
