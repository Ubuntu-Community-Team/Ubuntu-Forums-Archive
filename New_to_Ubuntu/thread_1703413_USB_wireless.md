### Post by Jeffofnh on 2011-03-09
I have a Linksys WUSB54G USB wireless network card configured and working. The only issue is that when I boot the system the wireless does not come up by default. I have to click on the signal icon in the upper right and enable the Linksys wireless signal.
 
The question, is there any configuration changes that I can make to have this enabled by default?
 
Thanks,
 
Jeff

### Post by grahammechanical on 2011-03-09
You need to set the connection to connect automatically. So, the answer to your question is, yes.

Right click the icon. Select Edit connections. Select the wireless tab. Select your wireless connection. Select Edit and then tick the box marked connect automatically.
