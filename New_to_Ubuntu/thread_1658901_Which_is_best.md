Could you be a bit more specific about the audio mixing? What exactly are you doing? Mixing audio tracks (possibly more than two) to a stereo MP3 file?

Ubuntu looks like it can cover your needs. If you want to go all out, you can load the ubuntustudio-desktop from System->Administration->Synaptic Package Manager. That will load all the Ubuntu Studio packages (applications). But you probably don't need to go that far. You can just install the applications you like via Synaptics also. 

Net? Firefox is installed by default.
Email? Evolution is installed by default but I prefer Thunderbird which is also available in Synaptics.

If you go for another email client you can uninstall Evolution also via Synaptics by doing a search for it and marking for removal (click the box next to Evolution).

Good luck and enjoy the learning curve. ;)

### Post by Bucky Ball on 2011-01-03
You will find Filezilla s***s all over the cpanel file manager or 'Web Disk'.

All sounding good and I would suggest Ubuntu Studio is not required. ;)

You can always install separate packages from anywhere as suggested earlier.

Ubuntu Studio does use an rt kernel (real time) BUT you can't use nvidia drivers with the rt kernel. Bit crazy for a distro designed specifically for AV!

Anyways, I have Xubuntu base install on my studio desktop with UStudio AV packages installed in there (yes, you can install just the AV packages, no artwork or desktop), and Gnome desktop environment (regular Ubuntu DE) which I can choose with 'Sessions' at the login screen (along with Openbox, Fluxbox, and a couple of other DEnvironments). It is a real hybrid!

Yes, you will generally find help on the forums. We do our best! You will find before long you will probably be giving some yourself. I considered myself a Windows 'power user!' then got into Ubuntu in 2007 after thinking about it for a few years and the learning curve continues and loving it ... ;)
