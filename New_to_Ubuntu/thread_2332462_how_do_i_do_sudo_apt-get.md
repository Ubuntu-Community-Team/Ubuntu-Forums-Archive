### Post by Bucky Ball on 2016-08-01
And just to throw this in ...

'apt-get' has been overhauled and you can now use simply 'apt' in newer releases. For instance:

```
sudo apt update
```

... instead of 

```
sudo apt-get update
```

... etc. [See here]("https://www.maketecheasier.com/apt-vs-apt-get-ubuntu/"). Both will work. ;)
