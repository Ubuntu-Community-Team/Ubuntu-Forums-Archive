### Post by jou770d on 2009-02-14
I had a similar problem with Arabic using the default text editor (gedit).
Try this:
- Open the text editor.
- Chose open from the file menu
- At the bottom of the open files windows you'll find character encoding, chose add or remove.
- Add the encodings you need.
