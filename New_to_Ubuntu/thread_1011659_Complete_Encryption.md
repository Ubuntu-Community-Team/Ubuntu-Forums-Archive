### Post by stoogiebuncho on 2008-12-15
Ubuntu 8.10 has an encrypted private folder that you can set up with a few simple commands in the terminal.  There's a pretty good tutorial here:

[http://tombuntu.com/index.php/2008/09/23/encrypted-private-directory-in-ubuntu-810/](http://tombuntu.com/index.php/2008/09/23/encrypted-private-directory-in-ubuntu-810/)

Truecrypt is a pretty good program, though, so if you want to use that instead, go to this website:

[http://www.truecrypt.org/downloads.php](http://www.truecrypt.org/downloads.php)

Scroll down to the linux section, and select Ubuntu .deb (you'll have to choose the file depending on whether you have a 32-bit or a 64-bit machine)

Download the .deb file, and then to install it, just double-click it.

To run truecrypt after it is installed, press Alt+F2 and type in 
```
truecrypt
```

You can create a launcher to make it easier to start if you don't like starting it using that method.


One more thing.  If you're talking about wanting to encrypt your entire drive, you'll need to do that during the installation process.  I've never done it, but there are a lot of tutorials out there on Google.
