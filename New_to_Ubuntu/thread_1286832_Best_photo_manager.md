### Post by OldGrey on 2009-10-09
I think you need to try the various options and see which suits you best. FSpot is the default with Ubuntu. DigiKam is the KDE equivalent, but will work in a Gnome environment. I like Picasa for Linux and often use gThumb which is a basic but good viewer/organiser/editor. There are others too. They are all easily installable and of course cost you nothing
