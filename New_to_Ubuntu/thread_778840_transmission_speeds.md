### Post by Monicker on 2008-05-02
Common issue.  Go to Edit -> Preferences.  In the Network, after does its test, does the Incoming port show as open or closed?  Most likely it is closed.  You need to use port forwarding so that the traffic for the port specified there goes to the client machine which is running Transmission.  This will let you get higher download speeds.
