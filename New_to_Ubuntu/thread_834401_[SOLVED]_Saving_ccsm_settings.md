### Post by markbuntu on 2008-06-19
In the compiz settings manager there is a utility called Session Manager or something like that. It links compiz settings to Sessions Manager. It must be enabled for Session Manager to remember your settings. You can then go to System/Preferences/Sessions and set that to remember.
