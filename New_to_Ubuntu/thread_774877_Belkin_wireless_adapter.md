### Post by anewguy on 2008-04-29
Mine is a F5D7050.  It was installed automatically by Linux and uses the zd1211rw driver.  Check your device manager (System/Preferences/Hardware Information) and see if there is a USB device list as USB WLAN.  If there is click on the line labeled "USB Vendor Specific Interface", then move to the right hand side of the screen and click the tab "Advanced".  The Linux driver name should show there.  IF all of that looks ok, then I have to ask if you have the network manager applet running -> is there a "network" icon on your top bar?  If so, click it and see if your wireless network shows.  Post back when you've checked all that.  :)
