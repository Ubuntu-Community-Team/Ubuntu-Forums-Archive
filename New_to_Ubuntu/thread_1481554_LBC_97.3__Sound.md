I had a bit of trouble with it. Make sure you have mplayer installed, you can find it in Synaptic Package Manager.

I still couldn't get the Live pop up player to work, but I got to hear LBC from clicking on this link:

[http://mediaweb.musicradio.com/Show.asx?StreamID=156](http://mediaweb.musicradio.com/Show.asx?StreamID=156)

You could save that link as a Bookmark in Firefox, so it's not a bad solution.
