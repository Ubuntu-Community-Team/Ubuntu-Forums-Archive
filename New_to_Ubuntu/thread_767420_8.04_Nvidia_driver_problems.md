### Post by jimv on 2008-04-25
I don't know about the xfix thing, but there is a release of Envy available for 8.04.  To get it, just type this in a command line:

[INDENT]sudo apt-get install envyng-gtk[/INDENT]

Once it's installed, you can run it from the command line too (in case your graphics aren't working right) with this command:

[INDENT]sudo envyng -t[/INDENT]

You'll get a text menu to install the drivers...I believe that the choice for Nvidia is 1.

It seems like whenever I can't get my Nvidia drivers installed right, Envy can.  Give it a shot.
