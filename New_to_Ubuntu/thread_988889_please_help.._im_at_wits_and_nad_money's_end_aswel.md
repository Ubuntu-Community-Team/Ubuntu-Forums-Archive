### Post by nhasian on 2008-11-21
hello,

your AMD mk34 has 1800 MHz and you have a gig of ram.  that's plenty of power to run the regular Ubuntu.  No need to use xUbuntu.  If your pc is about 2 years old though, your CD Drive may be dirty/defective.  you might want to try to install ubuntu from a USB thumbstick if you have one lying around.  Boot the ubuntu 8.10 liveCD on another system and use System/Administration/Create a USB Startup Disc.  

PS if your going to install from a usb thumbdrive make sure the target pc's bios is setup to boot from USB before the hard disk.
