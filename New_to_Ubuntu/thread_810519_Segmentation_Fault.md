Whilst upgrading my files I got a "Sementation Fault - Core dumped" at least twenty times.


When I upgraded to Ubuntu 8.04LTS - around about half of the files loaded but then the other half (approximately) showed "Segmentation Fault - Core dumped", finally, "Hal" did not load either, and returned the sytem to a jumbled mess. there were quite a few files that did not load with the "Segmentation Fault - Core Dumped - signal 11 not found" label. Just to add, when dowloading the files of Hardy Heron (8.04) on building my system, the Broadband connection did start becoming slower around halfway through downloading the update.


I had to re-build Ubuntu 7.10 Gutsy Gibbon - my previous version) again and install some of my files this afternoon - luckily, I backup all my files to a seperate hard disk and work from them if there is a problem as there was in this case.

Always - Always, backup your files when either upgrading or changing your system in some way!
