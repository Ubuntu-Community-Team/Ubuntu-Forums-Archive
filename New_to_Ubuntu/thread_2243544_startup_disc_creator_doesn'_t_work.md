### Post by yancek on 2014-09-09
When you click on the 'Other' tab, it should open a new windows where you can navigate to wherever the iso file is.  Then you click on the iso file to highlight it and then click 'Open' and you should be back in the window you started from, the one with the 'Other' tab.  You then need to select the flash drive which should be shown on the bottom of the page and should have been plugged in before you open usb-creator.  More details as suggested above would help.
