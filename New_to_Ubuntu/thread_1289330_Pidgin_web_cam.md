### Post by mcduck on 2009-10-12
No, Pidgin doesn't have support for web cams on Yahoo. The Purple libraries already have support for voice & video but currently they are only implemented in Pidgin for the XMPP protocol.

I know aMSN has cam support for MSN protocol, so I suppose there might be some alternative program that has cam support for Yahoo as well.
