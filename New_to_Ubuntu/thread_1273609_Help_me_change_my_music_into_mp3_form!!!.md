### Post by A_Fiachra on 2009-09-23
Unfortunately, iPod uses proprietary formats like mp3 and aa4.

Ogg is an open source format, and most MP3 portable players won't support it (yet).

Evidently Soundconverter will work for your needs.

[http://soundconverter.berlios.de/](http://soundconverter.berlios.de/)

There is a package you can install:

```

[FONT=Arial][COLOR=Black]sudo *apt*-*get* install *soundconverter*[/COLOR][/FONT]

```
