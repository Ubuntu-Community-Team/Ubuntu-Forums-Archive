### Post by 3rdalbum on 2013-10-08
Garageband is a Macintosh program. You can only run it on the MacOS, not on Linux or Windows.

You can try something like Ardour; it's a bit more pro-level than Garageband. Ardour is in the Ubuntu 12.04 repositories so you can just install it from the Software Center.
