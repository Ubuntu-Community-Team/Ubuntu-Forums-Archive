### Post by Lars Noodén on 2012-08-30
In addition to [cd](http://manpages.ubuntu.com/manpages/precise/en/man1/cd.1posix.html) and [ls](http://manpages.ubuntu.com/manpages/precise/en/man1/ls.1posix.html), there are some other programs which are helpful in navigating directories and managing files:

[https://help.ubuntu.com/community/UsingTheTerminal](https://help.ubuntu.com/community/UsingTheTerminal)

One of the most useful ones is [man](http://manpages.ubuntu.com/manpages/precise/en/man1/man.1.html), which is short for manual pages.  It will help you look up the instructions for a program including any options it might take.

```

man ls

```

It's useful to know because it helps you teach yourself.  No one can remember everything but it is possible to remember how to look up the information you need to work.
