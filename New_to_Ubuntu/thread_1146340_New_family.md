### Post by DustyMP on 2009-05-02
If you'd feel safer with some protection I would recommend Firestarter as your firewall and avast! for virus scanning.

If you want to install Firestarter go to System -> Administration -> Synaptic Package Manager and enter your password when prompted. For good measure press the reload button to get the updates and then enter firestarter in the search box. Select the package and Mark for installation. Then hit Apply and it will be installed. After that you can find it in the Sytem -> Administration menu.

Avast! can be downloaded at [http://www.avast.com/eng/download-avast-for-linux-edition.html](http://www.avast.com/eng/download-avast-for-linux-edition.html) and selecting the download labled DEB package(DEB's are the package format Ubuntu uses). You will have to register to get the updated keys but it's still free.

Hi, and Welcome to Ubuntu, and the Forums!! ):P

I think just about everybody (myself included) has started off with the 'Windows Mindset' about security - that we have to have an Anti-Virus program, Anti-Malware, defragment, etc. etc. Ubuntu (thankfully :)) is very different from Windows. In Ubuntu, your firewall is actually a thing called 'ip tables' and they are setup by default with every port closed - so it is a very secure install from the get-go. There are applications which allow you to change and configure the ip-tables as you see fit, however, this is really not necessary for the average user.

As far as viruses go, it is true that you *can* get viruses in Ubuntu, it it probably more likely that you will be crushed to death by a vending machine tipping over on you ;) The main reason people run anti-virus software on Linux is usually if they interact a lot with other Windows computers. The anti-virus programs can detect and deal with the viruses that it finds instead of them getting passed along to those said Windows computers. Personally, unless you're very paranoid, and lock yourself in the basement and blacken the windows when you're on your computer, there's really no great need for an anti-virus program. The same can be said for Malware.

If you're interested in reading up more on Ubuntu Security, there is a great post by a gentleman much more knowledgeable on the subject than I. You can read his post here:
[http://ubuntuforums.org/showthread.php?t=510812](http://ubuntuforums.org/showthread.php?t=510812)

Again, welcome to Ubuntu, and the Forums! And if you have any more questions, please feel free to ask :D

### Post by donkyhotay on 2009-05-02
Many people migrating from windows to linux have the paranoid security mindset thinking that it's impossible for linux to just 'be secure' without any special software since that is the way windows works. Remember you have an OS designed by users for users, although we do many things differently we do that to makes things easy to use. In this case easy means having all security preconfigured and ready to go. Other OS's are designed by corporations for anybody with a checkbook and they try to make things profitable for them.
$200 for the OS
$100 for the antivirus
$50 for the firewall
I think you get the idea. Anyways, welcome to 'the light side of the source' and whatever you do *don't worry*. You are now better protected then most windows (and even mac) users that spend hundreds of dollers in security software. If you have any questions feel free to ask, don't worry about thinking it's stupid and something you should know. Despite what you may think (or even what some people here may claim) we *all* started out as utter linux newbies at one point asking the exact same questions you are right now. Despite all of the posts here telling you don't worry about security some people still insist on installing clamav, firestarter and similar programs just because they can't get past the windows mindset. If despite everything you've read here you're one of those thinking of doing that 'just to be safe' I'd like you to read the following by Eric S. Raymond who is considered to be one of the (unofficial) leaders of the open source movement.


Master Foo and the Nervous Novice
There was a novice who learned much at the Master's feet, but felt something to be missing. After meditating on his doubts for some time, he found the courage to approach Master Foo about his problem.

&#8220;Master Foo,&#8221; he asked &#8220;why do Unix users not employ antivirus programs? And defragmentors? And malware cleaners?&#8221;

Master Foo smiled, and said &#8220;When your house is well constructed, there is no need to add pillars to keep the roof in place.&#8221;

The novice replied &#8220;Would it not be better to use these things anyway, just to be certain?&#8221;

Master Foo reached for a nearby ball of string, and began wrapping it around the novice's feet.

&#8220;What are you doing?&#8221; the novice asked in surprise.

Master Foo replied simply: &#8220;Tying your shoes.&#8221;

Upon hearing this, the novice was enlightened.
