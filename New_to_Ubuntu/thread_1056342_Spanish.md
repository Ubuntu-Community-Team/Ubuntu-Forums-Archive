### Post by mkvnmtr on 2009-01-31
To write in spanish first go to system/pref/keyboard. Choose by country Latin America. Leave as default your native language. Then double click on the panel. Choose to add to panel. Look for keyboard indicator. Add it to the panel. To have full access to spanish open the package manager. Under sections there are two sections for internationalization and localization. Go through both and download all the spanish support you can find. You should find general support and support for open office and more for mozilla and thunderbird. Then if you are serious in the search box write dicitionary and install all the spanish dictionary support you think you need. That about does it.Takes a bit of searching but it is worth it it you use more than one language.
