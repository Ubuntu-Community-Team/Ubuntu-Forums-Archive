### Post by richardjennings on 2008-05-26
I have never used the windows installer, all I know I have read. In My Honest Opinion, you are much better off installing it a more traditional way. By a more traditional way I mean booting from the cd, following instructions & configuring keyboard layout etc as required. IMHO the windows installer is a ploy to get more people interested in linux (i.e making it seem easy). The crazy thing is, installing ubuntu outside of windows IS easy. I suggest you 'add/remove programs ... remove' the ubuntu install in windows, and have a crack at doing it properly. 

The first step in doing this is getting the cd to boot. This invloves pressing a button just after the computer is first turned on, usually one of, F1 F2 Del Escape etc.. there will be a message to tell you.

The next obstacle might be the partitioning of your drive. This is much simpler and less daunting than it might seem.


There may be 1000's of reasons why the windows ubuntu install program is failing. A multitude of these may be due to windows. You are better off loosing the dodgyware overhead windows burdens you with. I've never actually met anyone that has installed linux with a windows interface.
