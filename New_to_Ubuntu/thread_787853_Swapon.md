### Post by Austin_KW on 2008-05-09
Hopefully I remember all the steps;

remake your swapfile
```
sudo mkswap /dev/hda2 
```
This will give a new UUID for the swap partition.

Edit /etc/fstab and /etc/initramfs-tools/conf.d/resume and change the UUID for the swap partition to the one given above.

Try ```
 sudo swapon -a
```
and if that works try a hibernate/resume
