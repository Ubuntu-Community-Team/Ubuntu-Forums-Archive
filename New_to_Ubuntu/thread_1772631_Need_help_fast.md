### Post by smcrossman on 2011-05-31
Were you only in compiz?  My recent issue along this line was because my desktop background was dark The top menu has default black text so it doesn't show up on a dark background.

In Unity the App file menu is merged with the top Unity menu when the app window is maximised.  Try grabbing the top of the window and pulling it down and to either side. That will unmaximise it and your App menu should appear.  Once you have the ability to close the windows, etc....you can go back and give more info on exactly what changes you made.  You can also look and see if you have any keyboard commands to open, close, maximise, unmaximise and minimise windows.  Several applications provide those....my problem is I have so many installed and active I don't know which ones are the correct key bindings!
