### Post by fabricator4 on 2011-03-19
> **xumuk37 said:**
> Hi there! Since few days my numerical part of keyboard works as mouse controller, I mean it controls the mouse movements instead of type numbers. And this happens after log in, because when I put my password at login screen it works as it should! What can it be and how could I fix it? Thnx in advance.

It's an accessibility feature which can be turned off:

```

->System
    ->Preferences
        ->Keyboard

```

Then select the Mouse Keys tab, and uncheck the "pointer can be controlled with using keypad" feature.

Chris.
