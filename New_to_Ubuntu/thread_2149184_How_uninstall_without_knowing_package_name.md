### Post by deadflowr on 2013-05-28
It's rather hard to install or uninstall any package for which the name alludes oneself.

That being said, what is the package?
Is it a program, or a library or something else?
If it's a program does it have a gui or a cli?
If a gui how do you open it? What's the program called.
If a cli how do you execute it. What's the command used.
What does the package do?
Any info on what it does will help figure out the package name.
