### Post by immerohnegott on 2010-04-22
Latest update on Phoronix claims that the author has "sources", who say we may see an announcement about native Steam clientage by June. 

This makes some modicum of sense, considering there are files for download from Steam which contain libs and binaries for a linux variant of the client (though obviously, no game content was available). The client binary does run, though there are authentication errors, presumably due to the closed nature of the project, such that it never escapes the CLI pre-load.

This ain't a server gig, as libraries include steamui.so and vgui2_s.so (which contains a reference to a libX11.so file). 

Regardless, signs appear positive as of right now.
