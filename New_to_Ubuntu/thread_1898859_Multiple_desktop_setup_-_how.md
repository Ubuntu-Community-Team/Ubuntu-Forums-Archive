However, all my machines run vanilla Ubuntu 11.04, and I don't really want to install Kubuntu to the machine and effectively have two systems dual-booting. I want to be able to choose which desktop I use either at boot-up or, if it's feasible, from within a working session.

I'm guessing it's not a grub problem, because I don't want to change the OS, just the desktop.

### Post by qamelian on 2011-12-22
At the login screen, click on the gear icon near the login and you will see a list of desktop environments. Select your KDE desktop from here.
