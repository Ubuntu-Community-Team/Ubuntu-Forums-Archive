### Post by northern lights on 2008-09-30
Running ```
lmms
```from either a terminal or the run dialog (invoke with Alt+F2) will start it.

The binary is located in '/usr/bin/', but you will not need that information.

If you're going to use LMMS often, you can create a launcher in a panel, on the desktop, or in the "Applications" menu via "System > Preferences > Main Menu".
