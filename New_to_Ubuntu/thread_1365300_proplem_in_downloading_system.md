I'm going to guess he meant to ask something more like, "Is it possible to install Linux along side Windows/Is it possible to dual-boot with Linux?  Why would you want to do this?"

Answering these guessed questions:  Yes, it is possible to download and install Linux (Ubuntu, SuSE, Fedora, many many more) along side Windows in a Dual-Boot configuration.  Why would you want to do this?  The biggest advantage for a new user would be the convenience of having a computer/laptop with two operating systems on it, allowing you to get the full experience of Linux without sacrificing your Windows install.
