### Post by fancypiper on 2009-11-17
It's your choice.  There are more hoops to jump through in order to please Microsoft if you decide to dual boot.  It will be easier to install just Ubuntu, but I do suggest making a separate /home partition for ease of upgrading or exploring more than one distribution of Linux.

I usually make these:
/ - from 5 to 15 GB depending on software installed
swap - 2 GB
/home - the rest of the disk.

Welcome to freedom!

### Post by Old *ix Geek on 2009-11-17
It's really up to you.  Personally, the FIRST thing I do when I buy a new computer is wipe windoze off its hard drive and install Linux.  If that's what you want to do, just boot it up with a (K)Ubuntu CD and follow the prompts.  When it gets to the part where it asks about partitioning the hard drive, you can either select the default, which will use the entire disk as--I THINK--a single partition, or you can do the VASTLY superior choice of manually dividing it up yourself.  If that's the route you want to go, post a follow-up for advice.

Note that even if you do the above you can always change your mind and re-do things if you decide you want XP and Ubuntu.  If that's what you decide, post a follow-up!

### Post by Sealbhach on 2009-11-17
Before you do anything, run a Live CD of Ubuntu on it, to see how the hardware works. By Live CD, we mean that the machine boots up from the CD, not the hard drive:

[https://help.ubuntu.com/community/LiveCD](https://help.ubuntu.com/community/LiveCD)

You can also boot up from USB, which is a much faster method, you can do this using Unetbootin: 

[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

Try that first, before you decide anything.
