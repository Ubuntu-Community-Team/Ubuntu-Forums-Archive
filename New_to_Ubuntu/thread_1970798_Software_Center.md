### Post by wmbg on 2012-05-01
I recently installed Ubuntu 12.04. Everything ran properly. Today I ran the Update Manager and all seemed to go well. Ubuntu runs well, but the Software Center has disappeared. It is not present on the launch or in the dash or in programs to be loaded. I searched the forums and saw a somewhat similar problem which suggested running 'sudo -apt-get update' in the terminal. I did this in complete ignorance of what should happen. Still, no Software Center. One other item - had run the Software Center soon after the installation to remove the Keyring program.

---

### Post by kc1di on 2012-05-01
you can try this in a terminal type the following:

```
sudo apt-get install software-center
```

That should restore it for you.  If it does not please post the error messages you get in the terminal.

---

### Post by wmbg on 2012-05-01
Thank you. I reinstalled as suggested. A miracle occured and the Software Center is back. I am most grateful
