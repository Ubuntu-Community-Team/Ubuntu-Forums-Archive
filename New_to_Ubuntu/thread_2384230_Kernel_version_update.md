That is source code you would need to configure and build yourself. As you are posting in "new to ubuntu" I would expect you are a relatively new user of ubuntu and unless you have some particular hardware requirements for running the very latest kernel, I'd advise you to stick with what the Ubuntu maintainers package for you.

To give you a comparison, in the 10 years I've been using Ubuntu I have never come across the need to build a custom kernel. There are some circumstances that it may be necessary but would expect such to happen only very rarely. Some people may also do such tasks if they have a particular interest in learning or are interested in coding and/or kernel hacking.
