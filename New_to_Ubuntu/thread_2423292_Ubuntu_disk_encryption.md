### Post by Dennis N on 2019-07-20
It's an installation option:

To begin the encrypted installation, select &#8220;Erase disk and install Ubuntu,&#8221; and check &#8220;Encrypt the new Ubuntu installation for Security&#8221; box. This will automatically select LVM as well. Both boxes must be checked. After selecting the encryption options, click &#8220;Install Now&#8221; to begin installation.
