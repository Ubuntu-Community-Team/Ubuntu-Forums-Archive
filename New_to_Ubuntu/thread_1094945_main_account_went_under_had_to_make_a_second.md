### Post by sisco311 on 2009-03-13
Boot in [recovery mode]("http://psychocats.net/ubuntu/fixsudo#recoverymode") and use the
```
adduser *username*
```command to add a new  user.

add the new user to the admin group:
```
adduser *username* admin
```
