### Post by buzzingrobot on 2014-04-02
If you happen to be downloading tar.gz files in an effort to install software... don't.

Tar.gz files are simple archives packaging a number of files in one.  Their main use has always been to package source code for use by developers.  They typically contain scripts which can be used to compile and install the source code they contain (often call 'install.sh').  Those scripts often make assumptions about what tools are on a system and where they are located, in the expectation that the person using them is either a developer or someone with similar knowledge and experience.

Applications installed from the Ubuntu repositories, either via the Software Center app or a similar tool, should not be problematic. They should install icons in the Unity overview and that icon should, also, be displayed when the user types the app's name after tapping the Windows key.  E.g., tap the Windows key, start type Okular, and the Okular icon should be displayed.  That's how Unity works.  If that consistently does not happen with apps installed from the Ubuntu repos, something is wrong with your system.
