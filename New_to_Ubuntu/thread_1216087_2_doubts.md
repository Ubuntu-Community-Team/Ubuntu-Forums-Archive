### Post by lisati on 2009-07-17
Each user on a Unix or Linux system has a folder/directory set aside for them on the disk for their own personal use. The "~" symbol refers to your personal "Home" directory.

> 1. What is the use of "~" symbol ~ = $HOME

In other words: These two commands produce the same result:
```
cd ~
cd $HOME
```

... You get returned to your home directory.
