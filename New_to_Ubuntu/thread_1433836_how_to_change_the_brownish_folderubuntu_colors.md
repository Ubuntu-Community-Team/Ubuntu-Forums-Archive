### Post by Jarmen_Deffs on 2010-03-19
Go to gnome-look.org

Here is the link to the icon set that I use: [http://gnome-look.org/content/show.php/ClearlooksOSX?content=69487](http://gnome-look.org/content/show.php/ClearlooksOSX?content=69487)

In your main menu go System > Preferences > Appearance
then click install and select the file you downloaded from gnome-look.org, then click customize in Appearances and you will be able to select the new icons in the icons tab.
