Can anyone tell me how to add programs to the side bar on version 11.10.? I have a number of programs I use quite frequently and it would be nice to have them as quick launch icons.

### Post by Jacobonbuntu on 2011-11-25
to make the icons smaller, install compiz settings manager. after installation open it, go to Desktop > Ubuntu Unity plugin, choose the"experimental" tab > icon size.
Be carefull with the other settings!!

to add a quick launcher: open the Dash, type the name of you application and drag it (the icon) onto the Unity launcher :)
