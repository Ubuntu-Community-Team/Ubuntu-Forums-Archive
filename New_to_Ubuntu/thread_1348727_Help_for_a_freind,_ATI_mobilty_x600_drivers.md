### Post by Terl on 2009-12-07
First place to start is System>Administration>Hardware Drivers.  If the card is listed there, Ubuntu can grab it all for you.  Just follow the directions.

EDIT - I just checked and see it is a legacy card like mine.  I just used the open source driver for mine and all runs well, but I am not running effects or 3d games either so ymmv.
