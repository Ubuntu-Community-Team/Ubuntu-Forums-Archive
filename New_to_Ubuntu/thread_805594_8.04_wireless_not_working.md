### Post by shifty_powers on 2008-05-25
wicd is not just a solution in that situation. at the end of the day, fi ti works for you, then it's a solution. can't say it's useless for you if you haven't tried it ;)

ndiswrapper is a little program that lets you use the windows driver for your wireless card. can be a life saver in the these situations.

use

```
sudo apt-get install ndisgtk
```

you will find it under system>admin>windows wireless drivers.

get the windows driver for your card here

[http://downloadcenter.intel.com/Detail_Desc.aspx?agr=N&Inst=Yes&ProductID=2259&DwnldID=15800&strOSs=44&OSFullName=Windows*%20XP%20Professional&lang=eng](http://downloadcenter.intel.com/Detail_Desc.aspx?agr=N&Inst=Yes&ProductID=2259&DwnldID=15800&strOSs=44&OSFullName=Windows*%20XP%20Professional&lang=eng)

unzip them

then go into the ndiswrapper util and select the appropriate *.inf file from the package you donwloaded...
