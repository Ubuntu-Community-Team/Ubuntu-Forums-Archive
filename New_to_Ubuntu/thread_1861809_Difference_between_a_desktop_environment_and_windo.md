### Post by Lars Noodén on 2011-10-16
You need a window manager to run a desktop environment, but you can run a window manager by itself without needing a desktop environment.  Many run a window manager alone because it can be smaller and faster.

To over simplify, 

The desktop environment provides a user interface based on Desktop Metaphor including Icons, Themes, Toolbars, Folders
and Menus.  

The window manager covers all interaction with the user interface such as window behavior (focus, raising, special effects, etc.), widgets (scroll bars, buttons, borders), virtual desktops and session management.
