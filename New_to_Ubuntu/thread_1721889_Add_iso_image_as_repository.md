### Post by XubuRoxMySox on 2011-04-05
There's a way cool app called **APTONCD** ("apt on CD") available thru Synaptic) that enables a CD to become a source for software. Backup and restore using APTONCD and it makes the upgrade process alot shorter because you don't have to download all those apps from the internet. And you can use APTONCD to install stuff to multiple 'puters if you have a whole classroom or office full to do.
