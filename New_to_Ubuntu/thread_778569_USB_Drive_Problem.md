### Post by monkeymoo on 2008-05-02
When I unmount my USB stick it disappears from wherever it is mounted to. It is then safe to remove. 
The light is still on, but that doesn't matter. When the stick is doing something the light flashes. If it is on continuously it just means it is powered. Don't remove it if it is flashing, but if it's continuously on it should be fine as I have been removing it for years this way. 

Only thing I will say is that if you unmount using the terminal I found I needed to quit the terminal and only then the light stopped flashing.
