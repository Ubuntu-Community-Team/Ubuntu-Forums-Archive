A few possible solutions for you.

Download Ubuntu 10.04LTS (older but still supported)

Burn the .iso image at the slowest speed possible to avoid data corruption issues.

When the disk is burned, put it in the cd drive of the target PC and boot it up.

You will get an option to try Ubuntu or Install Ubuntu,

Select try Ubuntu and give it time to load and run.

Once it is running, have a play around inside and see if everything works for you.

Then just click on the "install" icon on the Desktop and it should begin the install process.

As it installs you can see the progress along the bottom of the window, when it gets to installing language packs, there is a "skip" button above the progress bar. Click this and it will defer installing the language packs.

Another workaround is to disconnect the computer from the internet, then do all the updates after the install is complete. Remember that the first thing you should do aftr installing is open update manager and get all the updates you need (it will be quite a lot, probably about 150-190Mb)

Hope this helps
