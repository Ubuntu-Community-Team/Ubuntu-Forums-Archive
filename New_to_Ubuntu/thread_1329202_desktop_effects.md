### Post by ukripper on 2009-11-17
As emigrant above pointed. You require compizconfig-settings-manager. 

install it via terminal:
```
sudo apt-get install compizconfig-settings-manager
```

then go to System-->Preferences-->compizconfig-settings-manager

### Post by Dougie187 on 2009-11-17
It's Window Management->Scale->Window Picker.

Choose the one that has a picture of a desktop next to it, and you can choose a corner or a side that you can hit with your mouse to initiate it. There are also key combinations and mouse buttons.
