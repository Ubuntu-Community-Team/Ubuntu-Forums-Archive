### Post by 3rag0n on 2009-12-05
Ok, i see your point. You mean that when the updates start downloading, the main window is grayed out, and you can't see the description. Well, you have just made a feature request :) 
You can always click "cancel" while the update packages are being downloaded, tell the resulting dialog box to not install anything till everything is downloaded, and return back to the main download window, and THEN read descriptions to your heart's content :) [ The descriptions aren't exactly informative, though :D ].

You can do this because the downlading will resume from when you cancelled it.
