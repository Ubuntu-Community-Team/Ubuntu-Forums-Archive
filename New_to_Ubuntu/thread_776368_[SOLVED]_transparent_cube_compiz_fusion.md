### Post by Efros on 2008-04-30
Open Compiz manager, go to Desktop Cube and select the appearance tab, you should see a check box for skydome, and a browse button to load an image. 

Skydomes are limited to the maximum texture size supported by your graphics cards driver combo, I use 2048x4096. whatever size you can use you can use Gimp to resize the graphic.
