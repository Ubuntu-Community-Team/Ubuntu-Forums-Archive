### Post by epitaph on 2008-11-09
In the Advanced Appearing Settings manager (compizconfig-settings-manager) go to General and then set the Desktop Size to more than 1 horizontal virtual desktop (4 for a cube).

Also, you referred to scrolling to change desktops. The scrolling while your mouse is over a desktop is part of the Viewport Switcher plugin (Desktop category in the manager).

By default holding ctrl+alt+mouse1 will activate the freely floating cube and ctrl+alt+left or ctrl+alt+right should move to the next workspace.
