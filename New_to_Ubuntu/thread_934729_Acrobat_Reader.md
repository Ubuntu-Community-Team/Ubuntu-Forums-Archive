### Post by flipitj on 2008-10-02
i am a relatively new user for ubuntu as well.  I dont really care to use one app or another to open a .pdf, but i was bound and determined to install adobe reader in ubuntu.  I went to adobe's site and downloaded the tar file.  since i am a little lazy, what i did was go to home directory then my username, made a new folder called adobe, put the tar file in there extracted it in same file. went into the newly extracted folder and ran the install file on the terminal.  once that finished doing itsthing, i discovered that when trying to open a .pdf from link on web, it was barfing up errors leftand right about environment variables and such.  so at this point i got tired, so i went into preferences in firefox on the appications tab, picked pdf document and did the open with choice, drilled down to the adobe reader executable and wahlaa, it works, 

for anyone that this helped, this :popcorn: is for you :)
