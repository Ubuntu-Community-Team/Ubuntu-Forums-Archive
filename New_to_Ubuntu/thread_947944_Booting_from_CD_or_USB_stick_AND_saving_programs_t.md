### Post by rh0000 on 2008-10-14
I received a Ubuntu disc (Feisty Fawn) with a computing magazine a year ago, and had cause to bless it recently, when my laptop was completely unable to work under my usual OS (Windows Vista Home Basic), and I had to reformat my hard drive and reinstall Vista. Ubuntu was a lifeline because it let me recover all my data files and transfer them to an external hard drive before reformatting my laptop's hard drive.

However, using Ubuntu on this occasion reminded me why I have never used it as a serious alternative to Windows. I am reluctant to install Ubuntu on my PC's hard drive, as I have been told that Vista won't allow competing operating systems on the hard drive, and I do want to keep Vista, at least for the time being. For all its faults, Vista tends to do most of the things I want, and there is a lot of software available for it. I don't want to abandon all this before my experiences make me sure that Ubuntu is at least as good.

So I boot into Ubuntu from the CD I hve been given. But when I do this, any settings which I change, or downloads which I make, get lost when I switch off. The result is that I am back to square one the next time I boot into Ubuntu.

My version of Ubuntu has a facility for creating a bootable version of itself onto a USB stick. However, when I try to do this I find that, the next time I switch on, my USB stick is not recognised as being bootable.

What I want to be able to do, is to put Ubuntu onto a bootable USB stick, boot into Ubunto from this USB stick, and then capture any any modifications, software, downloads, etc, onto this USB stick. The result will be that the next time I boot into Ubuntu from my USB stick, I will immediately have the benefit of all the additions I have previously made, and can then of course make some more, and thus progressively build up my copy of Ubuntu so that the various programs are immediately available to me on each occasion.

An alternative would be to boot into Ubuntu from a CD, but have the facility to record modifications, or download software, etc, onto a USB stick, so that these can be recalled each time I boot into Ubuntu from the CD.

A further alternative would be to boot into Ubuntu, and keep any modifications on my hard drive, for recalling each time I use Ubuntu. However, I would prefer to keep any modifications on a USB stick, so that I can use Ubuntu on other PCs, not simply my usual laptop.

At the moment I can't do any of these things, and so I have to start from scratch each time I use Ubuntu, and painfully build up a few items of software, which I then completely lose when I switch off my machine. The result is that I seldom bother to use Ubuntu.

It's irritating to find that Ubuntu is effectively unusable and that I am stuck with Vista, and indeed to find that Vista is the superior operating system for my needs, because it remembers what I do from one session to the next, which Ubuntu doesn't.

I expect that my problem is an extremely simple one, which can be solved in a couple of sentences, but can anyone help?

Thanks in anticipation.

### Post by C.S.Cameron on 2008-10-14
You should probably download a recent version of Ubuntu, say 8.04.1 and make a Live CD. Things have improved since Feisty.
Using the Live CD you can run install and then choose your flash drive when you get to partitioning.
This will build a full version of Ubuntu on your usb stick.
you can modify and upgrade it the same as a hard disk install.

This page explains a lot about flash drive installs:
[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)
