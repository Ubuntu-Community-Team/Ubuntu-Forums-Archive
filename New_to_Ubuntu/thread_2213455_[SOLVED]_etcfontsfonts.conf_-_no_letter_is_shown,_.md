### Post by grahammechanical on 2014-03-26
You can try this. Open the File Manager (Nautilus) and in the View menu select Show Hidden Files. And then browse to /etc/fonts/ and you might find a backup copy. It should have the same name but with the tilde ( ~ ) character added to the end of the name. When we edit system files with Administrator access then the editor (Gedit) makes a back up copy by adding the tilde character. The file then becomes hidden.

Regards.
