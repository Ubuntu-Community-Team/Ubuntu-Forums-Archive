### Post by mcduck on 2014-09-05
Every Ubuntu version has it's own software repositories, so yes, any updates you get are for the Ubuntu version you are running.

However, the updates you get are *not* always the latest versions of programs, the basic rule is that after release, packages for each Ubuntu version will only receive updates for security reasons and to fix any serious bugs. And even then it's often preferred to backport any security patces to current version rather than update to latest version. So if you want to run the latest-and-greatest versions of software, you'll need to find some 3rd-party repositories to get such updates. (There are some exemptions to this rule, for example web browsers are kept at latest versions, but the rules for special cases are quite strict).

The reasoning and rules for the updates are better explained here: [https://wiki.ubuntu.com/StableReleaseUpdates]("https://wiki.ubuntu.com/StableReleaseUpdates")
