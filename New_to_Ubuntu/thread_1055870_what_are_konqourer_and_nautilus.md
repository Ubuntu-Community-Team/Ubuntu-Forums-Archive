### Post by ChildOfMana on 2009-01-31
Nautilus is the default Gnome file manager and Konqueror is the default KDE Web Browser.

I take it you're using Kubuntu? Or have installed KDE at some point?

Someone may correct me here as I haven't used Kubuntu in quite a while, but I believe that in past iterations of Kubuntu Konqueror was both the default Web *and* file browser. Apparently now Konqueror fulfills the role of stand-alone Web Browser and Dolphin the file manager.
