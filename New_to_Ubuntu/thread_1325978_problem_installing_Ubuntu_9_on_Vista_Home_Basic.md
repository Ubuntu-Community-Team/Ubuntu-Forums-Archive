### Post by garvinrick4 on 2009-11-14
You should be able to download a 9.04 or a 9.10 .iso and burn to CD. Place in CD tray
run and it will ask you if you want to run WUBI click yes and off you go. Answer two
questions, size of volume for Ubuntu and your password.

Ubuntu will be in a folder in C: right next to where your User folder hangs out.
