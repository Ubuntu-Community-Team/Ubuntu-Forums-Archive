### Post by Cheesemill on 2012-11-02
You shouldn't extract the .iso file.

This misunderstanding usually happens if you have WinZip or WinRAR installed on your system, it mistakenly presents the .iso file as an extractable archive.
Instead you should just burn the untouched .iso file that you've downloaded as a disk image.

Also bear in mind that the 12.10 image is slightly too large to fit on a standard CD-R, you need to use either an 800MB CD or a blank DVD to burn it on. Alternatively you can create a bootable USB. Instructions for both methods can be found here:

[http://www.ubuntu.com/download/help/burn-a-dvd-on-windows](http://www.ubuntu.com/download/help/burn-a-dvd-on-windows)
[http://www.ubuntu.com/download/help/create-a-usb-stick-on-windows](http://www.ubuntu.com/download/help/create-a-usb-stick-on-windows)

### Post by squakie on 2012-11-02
To make all the above short:

- download the ISO again if you didn't keep it.  Don't extract any files, etc.

- use your CD/DVD burning app and burn as image

Put the CD in your drive, reboot - selecting boot sequence if needed so it will boot from the CD.  Try the "Try Ubuntu" without installing option to see if it works.

Decide which you want - a "wubi" install which is basically linux within Windows (not recommended for much more than a look-see), a dual-boot (linux along with  separately installed OS[s]), or a completely linux-only PC.

When you are ready to actually install, post back with your choice if you have any questions at all so we can be sure you understand exactly how to install.

Also keep this in mind:  the 12.10 download (maybe 12.04 - I used USB so I don't know) requires burning the image to a DVD as it is too large for a CD.
