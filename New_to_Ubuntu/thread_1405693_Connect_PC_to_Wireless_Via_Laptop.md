### Post by yeats on 2010-02-13
> 
Is there anyway to connect it to the internet via the laptop.


Yes.  What OS is the laptop running?  It's easy to do this in Karmic using the default Network Manager.  You dig down into the connection details and select to share the wireless connection via your Ethernet interface.  Then all you need is an Ethernet cable.

This is also possible in Windows, but it's been a while since I did it there.
