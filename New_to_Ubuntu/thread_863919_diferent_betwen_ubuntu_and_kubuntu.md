Yes it is true that Ubuntu uses the Gnome desktop environment and Kubuntu uses KDE.  The difference is that they are different desktop environments, with different look, feel, and a different set of core applications.

If you want to try them both, you can install both desktops.  Suppose you are currently running Ubuntu... you can install the package "kubuntu-desktop".  Then, when you log into your computer, you'll have a choice of sessions - whether to log into Gnome, or KDE. That's what I'm doing at the moment, and it's interesting - the best of both worlds :)
