### Post by HotShotDJ on 2009-01-16
> **boost3d23 said:**
> No its not compizconfig-settings-manager, I have that installed and setup. Its a program that I installed and it was accessed through Applications>System Tools if I'm not mistaken.Hmmmmm.... Perhaps you are talking about "Configuration Editor."  This is installed by default, but is normally "hidden" in the menu.  Right click on "Applications" and select "Edit Menus."  Click on "System Tools."  In the Items box, you should see "Configuration Editor" -- just check it off and it will now appear in Applications --> System Tools.

If, perchance, it is not installed, you can use Synaptic Package Manager and install gconf-editor.
