### Post by David Andersson on 2010-08-19
> **ub9876 said:**
> I have my iso from devede. how do I burn that to dvd  and USB   ??

For DVD/CD: IIRC Brasero should be installed by default in ubuntu. Can you *right-click the iso-file* and select "Open with Brasero", or start Brasero from the *Application > Multimedia*?

(If you start Brasero from the Application menu, select Burn Image when it asks what to do. If burn quality is important, you may want to open Properties and change Burning Speed to something less than Maximum Speed.)
