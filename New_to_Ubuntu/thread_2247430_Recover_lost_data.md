### Post by Mark Phelps on 2014-10-07
First of all, you must stop using the drive because every write risks overwriting previous files and folders.

Second, to truly recover the data, you need to be able to connect this drive to a working PC, and since the filesystem you want to recover is Windows, it would be best to connect this drive to a Windows PC.

Third, you might be able to recover the D: partition in its entirety -- but to do that, you need to be able to run the Minitool Partition Wizard windows app, and then see if it can recover the partition.

If that doesn't work, you then need to consider using Recuva (a Windows data recovery app) to see if it can find the files and folders on the drive.

If recuva doesn't work, then you need to consider using RecoverMyFiles -- which is free to install and try, but you have to go online and buy a license to do the actual file recovery.
