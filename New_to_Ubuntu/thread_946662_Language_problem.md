### Post by Icebear on 2008-10-13
Hi

to type in arabic you will need to install a arabic layout for the keyboard

system>preferences>keyboard

Then hit the layout tab and hit add button

and click the layout button and choose arabic

for changing the keyboard when in the layout tab hit the layout option button

then choose the layout switching and choose what key you want to use

if you want a keyboard indicator on the panel go to a empty space on the panel and click the right mouse button and choose "add to pannel"

then scroll down to keyboard indicator

This will let you type Arabic in all things on ubuntu

If you want you can add Arabic for your OS by going to 
System>administration>Launguage support and then choose Arabic


I think this is what you are asking
