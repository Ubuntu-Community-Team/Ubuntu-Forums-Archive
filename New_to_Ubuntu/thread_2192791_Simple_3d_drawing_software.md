### Post by RayArdia on 2014-04-30
Sounds as though you need only 2D as you say you need only to draw 'shapes'. 
LibreCAD loads from the software centre, is relatively simple to use though you do have to learn by practice.

I have used LibreCAD since it became "free-issue" with Ubuntu - before that I used QCAD (still available) and before that CADvance. In my opinion LibreCAD takes the biscuit for making orthagonal (ie FrontView/Side View/Plan) drawings.

The main advantage of all the CAD drawing programs is that you draw at full size, only scaling down the drawing when you want to print it.

Hope this helps.
