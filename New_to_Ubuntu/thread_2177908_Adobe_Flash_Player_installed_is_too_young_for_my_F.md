### Post by SeijiSensei on 2013-10-01
The current version of Firefox being shipped with Ubuntu is version 24.  I suggest updating your copy of Firefox rather than trying to find an old version of Flashplayer.

You can install the current version of Firefox directly from the Mozilla website.  You'll get a tar.gz file that you can unpack into a directory that resides in your PATH.
