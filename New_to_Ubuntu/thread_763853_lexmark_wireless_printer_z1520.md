### Post by Thelasko on 2008-04-23
That's a pretty new printer and Lexmark is slow to release Linux drivers.  When they do release them they aren't very good.

My best suggestion is to download a z600 series driver, since it seems to be the most common.  It's designed for Redhat so you will have to use a program called Alien to install it.

[http://downloads.lexmark.com/cgi-perl/downloads.cgi?ccs=229:1:0:389:0:0&searchLang=en&os_group=Redhat&target=](http://downloads.lexmark.com/cgi-perl/downloads.cgi?ccs=229:1:0:389:0:0&searchLang=en&os_group=Redhat&target=)
Good Luck
