### Post by fantab on 2013-12-07
It should be resolved once you have installed Ubuntu. 
Whatever fixes you may apply to Ubuntu running from USB... you will lose all those fixes once ubuntu is shutdown. 
If there are any issues after installing then we can look into it.

However, you can use 'nomodeset' option at ubuntu boot and see if it makes a difference.

### Post by Bucky Ball on 2013-12-07
