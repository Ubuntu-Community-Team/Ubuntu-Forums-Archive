### Post by 3rdalbum on 2009-09-18
If you don't mind switching browser, then Opera has this feature built-in. It doesn't use Google, it uses Opera's servers. Opera Link will automatically sync bookmarks, history, speed dial and passwords between all computers logged into your Opera account, including mobile phones running Opera Mini.
