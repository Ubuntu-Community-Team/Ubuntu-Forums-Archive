### Post by studmf on 2010-01-11
If you have all the files you need from windows and want to make your system HD purely Ubuuntu, simply pop the live CD into the PC and boot from CDROM (Esc choose disk to boot from, or go into BIOS and change CDROM to 1st boot order).  Follow the steps and choose to install using entire HD.  When it finishes up, you got your Ubuntu system.
