### Post by HellNoire on 2008-10-13
Sounds like you don't have the GStreamer updated. 

Go to your menu, hit Add/Remove, select all applications at the top.

Type GSteamer and download all that are named that. Nothing more. Hit apply, type your password in, and go. Should be able to open every music/video file now.
