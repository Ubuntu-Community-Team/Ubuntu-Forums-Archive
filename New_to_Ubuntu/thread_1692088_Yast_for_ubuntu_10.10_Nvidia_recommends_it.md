### Post by asmoore82 on 2011-02-20
"System &#8594; Administration &#8594; Hardware Drivers" is the Ubuntu way to get nVidia's drivers.

".exe" is not a format native to computer hardware; it is Windows specific.
Wine can allow you to run some Windows software under Linux, but it can't
install anything such as drivers that affects the real Linux OS.
