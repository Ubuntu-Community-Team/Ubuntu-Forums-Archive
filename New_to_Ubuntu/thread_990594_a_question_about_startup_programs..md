### Post by fr.theo on 2008-11-22
goto system >> Administration >> services here you can select what you want to run at start up. also you can goto Systems >> preferences >> Sessions her also you can modify your sessions and the apps you want to run.

### Post by adult swim on 2008-11-22
system>preferences>sessions
choose add
then for name: write in dock or whatever you want
command: must be avant-window-navigator
and comment can be dock or whatever you want
