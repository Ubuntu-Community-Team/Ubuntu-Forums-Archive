### Post by lovinglinux on 2010-02-14
Yes, they are significant. See my benchmark comparisons below (the higher the value the better):

[IMG]http://lovinglinux.megabyet.net/wp-content/uploads/2010/01/Browser-Benchmark_1264765962620.png[/IMG]

[LIST]
[*]w/ext = with extensions installed and enabled
[*]tests were conducted on a Pentium 4 single core machine, with 2Gb DDR2 RAM and nVidia 7300GT graphics card.
[*]Firefox 3.5 optimized is the official final version from Mozilla, compiled from source with optimization flags for my processor family. PGO (Profile-Guided Optimization) is also used since July 5th tests.
[/LIST]

The optimized versions on those tests are essentially the same as Swiftfox or Swiftweasel.

They are reliable, but Swiftfox is not open source, so I would recommend Swiftweasel.
