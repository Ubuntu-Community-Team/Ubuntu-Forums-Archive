### Post by Mark Phelps on 2011-12-13
IT could be that Belkin includes software with their hub.  If so, that will NOT work in Ubuntu -- as it will be strictly for Windows.

Someone else here mentioned the same device and tried to install the drivers that came with it -- that also can NOT be done, as they are Windows drivers.

USB devices GENERALLY work the same in Windows and Linux, unless they need special drivers -- which hubs generally do not need.
