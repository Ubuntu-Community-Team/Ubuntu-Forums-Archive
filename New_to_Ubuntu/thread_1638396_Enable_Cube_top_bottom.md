### Post by ajgreeny on 2010-12-07
I think you may need the **compiz-fusion-plugins-extra** installed, then in the Effects section there will be a **Cube Reflection and Deformation** plugin.  Using the **Cube Caps->Appearance** tab of that you can set the top and bottom cube pictures as you want them, and also in the **Rotate Cube** plugin you can set to* "Snap to top"* and *"Snap to Bottom"*, if you want that as well.
