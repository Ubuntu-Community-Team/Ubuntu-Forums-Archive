### Post by Paqman on 2008-04-24
> **dboyd1952 said:**
> Can I run Microsoft Office 2007 programs (Word, Excel) with this OS?

No, but Open Office is installed by default. Open Office can handle MS Office file formats like .doc or .xls. It's also cross-platform, so you can also install it on XP and use the same software on both systems. When you're trying a new OS it's nice to have some familiar software.

There's actually quite a lot of software that is cross-platform. Firefox, the Thunderbird mail client, Pidgin instant messaging, etc are all available for both Windows and Ubuntu. In fact the Windows versions are almost all better than the equivalent Microsoft software, so even if you decide not to use Ubuntu it's worth installing them on Windows.
