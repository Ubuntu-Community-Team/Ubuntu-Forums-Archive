### Post by Temposs on 2010-02-28
For many applications, they should run fine on either Gnome or KDE. Sometimes an app has specific dependencies on a specific GNOME or KDE service and so it will have issues if you try to run it on the other platform, but this isn't the case for most apps. You can also see some less serious incompatibility like theme and interface integration.

Usually the only thing that happens the first time you install an app from another platform is you'll see a ton of new stuff has to be installed along with it, that being the base libraries from the other platform needed to run apps on that platform.
