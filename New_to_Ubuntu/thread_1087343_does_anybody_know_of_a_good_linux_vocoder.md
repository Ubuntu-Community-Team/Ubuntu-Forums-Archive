### Post by mikechant on 2009-03-06
There's an 'XMMS2 vocoder plugin' in the Ubuntu repositories if that's what you're looking for. Package name is xmms2-plugin-vocoder.

Do you use the synaptic package manager? If you open synaptic (system->admin->synaptic package manager) and use the search function for 'vocoder' there are a few related packages including the one mentioned. Single click on each package to get its full description.

### Post by transmogrifox on 2010-04-01
I see this is an old post... but for anybody who searches the forums:
We just added a vocoder to the Rakarrack FX software.  As of date you will have to build the most current development version from the git repository to have the vocoder, but it is there for the having.

I also discovered this LADSPA plugin:
[http://www.sirlab.de/linux/descr_vocoder.html](http://www.sirlab.de/linux/descr_vocoder.html)

The sirlab vocoder is a 16-band unit.  In Rakarrack we have implemented 32 bands.  Perhaps the sirlab vocoder is a little less CPU hungry.
