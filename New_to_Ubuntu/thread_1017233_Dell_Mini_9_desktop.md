### Post by earthpigg on 2008-12-21
hello,

im a kind-of experienced ubuntu user who also has a dell mini 9... (but was using ubuntu on my main computer prior)

'dell custom desktop' vs 'classic ubuntu desktop' is something different than themes, do not confuse them ;)

themes: System -> Appearance -> Theme is where you see your themes. if you want to add more, download from gnome-look.org and drag/drop onto that interface... if its a working theme, it will isntall just like that.

one thing i noticed is that when you switch between the two (applications->switch desktop mode) it forgets any changes you made to the other desktop which is kind of annoying.

i, personally, am not a big fan of the 'dell desktop'. no other ubuntu user that i have seen post on the forums likes the 'dell desktop' either.

dell desktop:
-no compiz
-when you install software, it does not update the respective program groups
-... it really doesn't make anything easier, unless your eyes are going bad and you cant read the text in the menus. and if thats the case, just toss an item on the classic desktop or panel.

classic:
-standard, what other folks on the forum will generally assume you are using.
-everything in the 'applications', 'places', and 'system' menu updates automatically... for example, if you install a new game, it will appear in applications->games.
