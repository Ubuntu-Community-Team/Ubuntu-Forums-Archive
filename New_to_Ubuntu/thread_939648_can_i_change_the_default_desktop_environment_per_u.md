### Post by mcduck on 2008-10-06
All you need to do is to click the session-button in the login screen and select what desktop environment (or language) you want to use. After giving the user's password you'll get a message asking if you want to make it the default setting. This setting is stored per user.

(so no need to edit any text files.. ;))
