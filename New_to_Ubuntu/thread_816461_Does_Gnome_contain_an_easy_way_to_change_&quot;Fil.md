### Post by BennyHill on 2008-06-02
In the file browser, right click what ever the file is you want to change, then click properties, then open with and then add to choose your default program for that file format, from then on. hope that helps.

---

### Post by phr0ze on 2008-06-02
Right click on the file you want to change. Choose properties, then choose the Open With tab. Click Add at the bottom and you will be able to select the program you want to open the file with.
