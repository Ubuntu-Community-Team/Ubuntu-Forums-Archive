### Post by Calash on 2010-05-26
While Windows applications "may" run via Wine, you really should invest time in getting familiar with Ubuntu equivalent applications, such as GIMP, and Rhythembox.

Based on the list of applications you may want to dual boot at first.  This way you can go back to Windows as needed until you are more familiar with how things work.

On the subject of "Snap To", Kubuntu has this feature as part of the latest update of KDE.  Gnome does not have this ability now, and looking at the upcoming releases I would not hold my breath.  There may be a setting in Compiz to do this.


Edit:  Looks like Compiz will support a form of Snap-To.  See the following link
[http://ubuntuforums.org/showthread.php?t=1472747](http://ubuntuforums.org/showthread.php?t=1472747)



Reference:
Gnome - Default desktop manager for Ubuntu
KDE   - Default desktop manager for Kubuntu
Compiz - Default compositing in Ubuntu, handles desktop effects such as wobble and transparent windows and has the ability to manage items such as desktop cubes and other decorations.
