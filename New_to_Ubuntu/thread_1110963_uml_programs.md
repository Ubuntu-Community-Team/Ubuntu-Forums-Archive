### Post by fballem on 2009-03-30
There are several, including uml modules for eclipse and uml modules for netbeans.

If you're running 9.04 (which is in beta), then netbeans 6.5 is included in the repository. If you prefer eclipse, then you will have to install from [www.eclipse.org](www.eclipse.org), since the version in the ubuntu repositories is old. If you're running 64-bit ubuntu and want to install eclipse, then please look for instructions, since there are some specific installation instructions that you need to use.

In the commercial world, there are several that will run under Linux. Two of the more reasonably priced products are Enterprise Architect from Sparx Systems ([http://www.sparxsystems.com](http://www.sparxsystems.com)), which apparently runs under Wine and My Eclipse ([http://www.myeclipseide.com/](http://www.myeclipseide.com/)).

I've used Enterprise Architect in Windows for some years now, but haven't yet installed it in Wine - I'm still going through the learning curve and I still have one machine using Windows.

Hope this helps,
