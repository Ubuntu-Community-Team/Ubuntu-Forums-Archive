### Post by Sven6210 on 2010-03-28
You can also consider to use 'gammu'. I use gammu to send and receive SMS with my 'Huawei E160' 3G modem. gammu is a powerful command line tool. Apparently is also offers GUIs, however so far I only used the command line.

The use of gammu requires that you have a mobile phone or 3G modem connected to your computer.
