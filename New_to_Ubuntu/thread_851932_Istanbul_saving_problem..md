### Post by Danielaus on 2008-11-13
to solve Istanbul freeze while saving you can disable "Record Sound" from menu
another good software for screen capture is RecordMyDesktop
```
sudo apt-get install recordmydesktop gtk-recordmydesktop
```there is also xVidCap
```
sudo apt-get install xvidcap
```the second one is better if you want to better manage the output format
