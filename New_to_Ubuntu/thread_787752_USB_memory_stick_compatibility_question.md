### Post by t0p on 2008-05-09
Are there compatibility issues with USB memory sticks and Linux? Reason I ask, I'm looking to buy one, so I was checking out the Maplin website and there were "compatible with Windows" tags on every stick whereas only 1 stick was marked Linux-compatible (and that was the wrong size).

---

### Post by Sirron on 2008-05-09
I've never found one that didn't work, but you'd be wise to steer clear of anything with special security features. If they rely on software installed on the device, it's unlikely that software will work on linux via Wine.

### Post by kwacka on 2008-05-09
I agree with Sirron - I've never found one that didn't work.

I have a couple with the 'crypting for windows' files which are useless under linux, if you want to make it more secure use truecrypt, encfs, etc.
