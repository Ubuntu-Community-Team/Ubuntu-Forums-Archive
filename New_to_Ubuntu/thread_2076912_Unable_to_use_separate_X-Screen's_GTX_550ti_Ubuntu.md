### Post by fooman on 2012-11-04
you need root access to make changes to system files (/etc/X11/xorg.conf)...so if you are opening nvidia-settings from a terminal,  try it like this:

```
gksudo nvidia-settings
```

should be able to save the settings without a problem like that.

---

### Post by fooman on 2012-11-04
re-read your posts and realized it is probably because you do not have a xorg.conf.

to create a blank one....just run this in a terminal:

```
gksudo gedit /etc/X11/xorg.conf
```

make no changes to the file and save it (the file will be blank).  then re-run the nvidia-settings and try to save your changes.  should work for you now the you have a xorg.conf to save it to....hope that helps!
