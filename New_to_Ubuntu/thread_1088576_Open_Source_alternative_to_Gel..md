We call those programs IDEs (Integrated Development Environments) in the programming world. But there are indeed two programs which will work on Ubuntu (and are in the repositories) which do the same thing.

I'd personally recommend NetBeans (because it has a GUI builder integrated on it) but there's also Eclipse.

For both of those programs:

```

sudo apt-get install netbeans

```
for NetBeans, or;

```

sudo apt-get install eclipse

```

for Eclipse.

Hope this helps. :)
