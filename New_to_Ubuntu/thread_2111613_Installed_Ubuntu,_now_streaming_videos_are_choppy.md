### Post by Mark Phelps on 2013-02-02
If you really have a ATI Mobility Radeon 9700, that is a really OLD ATI chipset, for which AMD dropped restricted driver support years ago.

Presuming you're seeing a desktop, you already have the default open-source Radeon graphics installed -- and that is all there is now.

IF you try to force the installation of any other Radeon or AMD drivers, you will only hose up your display and have to do a lot of work to remove those -- and replace them with the same drivers you already have.
