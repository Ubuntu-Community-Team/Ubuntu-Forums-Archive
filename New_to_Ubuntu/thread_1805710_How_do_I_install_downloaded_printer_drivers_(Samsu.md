### Post by relay_man on 2011-07-17
I downloaded the package from the Samsung website to my desktop, and did the following: 

Right click on the folder and choose "extract here" to unzip.

After the unzip is complete you should see a folder called "cd root"

Double click on that folder to open it.  You should see two items.

One is a folder named "Linux", the other is a file (I believe it is actually an install script)
called "auto run".
If you double click on this file, you may be asked if you want to view the file content, or run the file.
Choose "run" to begin installation.

I'm using 10.04 LTS, but this process should be the same for you in 11.04

Hope this helps:)
