### Post by Toshibawarrior on 2009-03-02
You have to go to the GDM, the place where you enter your username and password and select something called "Session" and then select the one that says KDE...

Personally, I think that installing GNOME and KDE on the same system is a mistake. There are some serious difficulties that might arise...I did it a few months ago and I had to format my whole HDD...Now I have two separate partitions one for Ubuntu and one for Kubuntu. (But since I've installed and configured Kubuntu I haven't had a need for Ubuntu, except to sync my iPod Classic which is a mess with Amarok.)

Hope this helps!
