### Post by 3rdalbum on 2010-01-16
The only things you need to unmount or "safely remove" are storage devices. For input devices like webcams and keyboards, and output devices like printers and microphones, you can just unplug them.

Also, you should only use the "Safely Remove" function when you're actually going to unplug the storage device, because Safely Remove will stop it being recognised until you unplug it and plug it back in. If you're not just about to unplug the storage device, then use the "unmount" function in Gnome.
