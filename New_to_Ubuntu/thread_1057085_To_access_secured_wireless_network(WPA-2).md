### Post by jimmy the saint on 2009-02-01
It sounds like the connection issues have to do with the router, since they span all OSs.  I would see if there is a firmware upgrade for the router, and at the very least, write down settings, reset the router and start over from scratch.  I have never used Belkin routers, but with linksys (at least with the default firmware) I found myself troubleshooting issues like this for a year or two until they finally released a firmware that was reliable.
