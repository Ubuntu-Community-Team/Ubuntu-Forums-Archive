### Post by cariboo on 2009-07-12
You install closed source (restricted drivers) by going to System-->Administration-->Hardware Drivers, and enabling the driver there. If you don't have the restricted driver installed it may take a while to download and install it, depending on the speed of the mirror.

I wouldn't suggest you try installing the drivers from Nvidia, if you are a new user, as the process is quite involved and if you don't know what you are doing you could end up with a command line only system.
