### Post by Old_Grey_Wolf on 2010-02-01
> **kendoori said:**
> Getting this error when I do updates. Can't find this key, or determine what its referencing. Usually I've been able to google these for an answer. Right now, no luck

```

W: GPG error: http://ppa.launchpad.net karmic Release: The following signatures couldn't be verified because the public key is not available: NO_PUBKEY 4A1382065BC3E400

```

Try these commands:
```
gpg --keyserver keyserver.ubuntu.com --recv 4A1382065BC3E400
gpg --export --armor 4A1382065BC3E400  | sudo apt-key add -
```
