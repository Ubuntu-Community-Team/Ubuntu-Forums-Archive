### Post by DuckHook on 2019-06-25
It is acceptable to "bump" a thread with no reply once every 12 hours. Please appreciate that forum members live in all time zones and have other cares and duties outside of the forums.

For typical users with mainstream hardware, it is always better to stick with LTS releases because they are more stable and last for 5 years. The last LTS was Bionic (18.04). The next one will not be until 20.04.

Standard releases are less stable and last for 9 months.

### Post by deadflowr on 2019-06-25
> Which release of Ubuntu should I use if I want to install steam for CS GO & Dota 2? 
18.04
For two reasons:
1) It's still unclear what is happening in terms of 32-bit software and Ubuntu.
it looks like it should be fine, but as of now it's an unknown.

2)18.04.2 includes the hardware enablement stack which rolls so you'll eventually get the kernel and graphics stack now included in 19.04
and that too will eventually roll in the stacks provided by 19.10 and 20.04. So 18.04 will eventual get the same hardware stack as 20.04.

What's going on with Ubuntu and 32-bit packages:
[https://community.ubuntu.com/t/statement-on-32-bit-i386-packages-for-ubuntu-19-10-and-20-04-lts/11547]("https://community.ubuntu.com/t/statement-on-32-bit-i386-packages-for-ubuntu-19-10-and-20-04-lts/11547")

What is hardware enablement stack:
[https://wiki.ubuntu.com/Kernel/LTSEnablementStack]("https://wiki.ubuntu.com/Kernel/LTSEnablementStack")


Oh, and a 3rd reason is support for 19.04 ends at the end of the year.
18.04 (.1-5 included) is supported until sometime around April 2023.
