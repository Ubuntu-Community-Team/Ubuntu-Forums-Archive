### Post by Yeti can't ski on 2009-02-23
I personally feel that erasing Vista from my new laptop and switching completely to Ubuntu was fantastic and I will never regret it. However, please note that there is a learning/adaptation cost in such a move. 

Some of the Windows-software will only run in Ubuntu (and in GNU-Linux in general) with special support programs (such as Wine). Many simply won't run at all. 

However, the main question is that in almost all cases a free and open source replacement is readily available to every program preivously used in Windows. 

Instead of IE there is Firefox. You can use MSN through Pidgeon (by means of which in can simultaneously log-in to accounts of Skype and Yahoo and Google messengers). Instead of MS Office, there is OpenOffice, which is less eye-candy but much more logical. 

Conclusion: if you need "functions" you probably won't find any unsolvable problems. If you are over-attached to specific Windows programs, then you might have a hard time. 

On what concerns antivirus and firewall, Ubuntu (GNU-Linux) has a a much different structure from that of Windows. You are not automatic logged-in as a root user (superuser). In practical terms, this means that the whole system is much (soooo much) safer. It is much harder to invade the system and the possible damage resulting from an invasion is usually limited. In a nutshell: there is no antivirus in Ubuntu. Nothing is 100% safe, but in Ubuntu you probably just should pay attention not to be fooled into entering harmful commands (such as "sudo rm...") and then you will be fine.   

In addition to all that, Ubuntu is definitely less hardware-hungry than Vista. There is even a simplified version of Ubuntu (Xubuntu) that would probably run in that old computer that you decided to throw way two years ago! :-)

Well, this is my humble (and absolutely non impartial ;)) opinion. If you decide to try Ubuntu, wellcome aboard! In any event, Good Luck!!!!

### Post by matteojg on 2009-02-23
1) Anti-virus

A lot of people will tell you that viruses are not a concern with Ubuntu because almost all viruses are written for MS Windows, while true in part the real security of a linux OS has to do with how it handles permissions..in short there is no need for anti-virus on Ubuntu unless you are trying to protect a windows system that shares files with your Ubuntu system.

2) Firewall

If you connect to the internet via a router (which most people do these days), then you should be protected by a firewall that is part of the router. Even if this is not the case Ubuntu comes with a firewall of its own. The IP tables might be a bit complicated for a first time user, but a decent graphical user interface called firestarter exists to manage the firewall. You can find it with your synaptic package manager.

3) Defragmenting

Windows and Linux use different file systems. The system used by Linux does not really need defragmenting.

4) Browsers

Ubuntu uses firefox which is better than IE anyways, and Google is expected to release a linux version of Chrome in the near future.

5) Media

VLM will handle most video codecs

6) Other Windows programs

There are several ways of creating a fake/virtual Windows environment on an Ubuntu OS from which to run Windows "only" programs. One of the more popular is called Wine which I use to run a Windows chess client on my Ubuntu laptop.
