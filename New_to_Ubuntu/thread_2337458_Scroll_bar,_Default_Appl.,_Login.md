### Post by deadflowr on 2016-09-19
Per No.2:
Details only lists defaults apps for basic services.
(Email,Music, Video, Calendar, etc.)

Documents don't really all into an area that can utilize a single application for all, like say photos or music can.
Ex: a basic text (.txt) document will have problems opening in evince (aka Document Viewer)
and pdf's will have a hard time opening in gedit.
So,

To set a default document viewer for any file type, you do so with in the file itself.
To set pdf's to open with a specific application, right click on the file and go to properties.
then go to open with and select the application best fit for that type of documents and highlight the select and click on the set as default.
from now on, all documents of that type will open in the application you just set.
