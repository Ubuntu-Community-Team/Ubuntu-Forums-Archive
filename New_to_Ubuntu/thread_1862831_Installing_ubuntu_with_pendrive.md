### Post by mastablasta on 2011-10-17
hmm that netbook edition had issues as i know it...
 
but ok you can use:
linuxliveusb - [http://www.linuxliveusb.com/](http://www.linuxliveusb.com/)
unebootin - [http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)
to help you create boot disk from the downloaded image in windows or.
 
or if you will do it in lubuntu: 
[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)
 
 
no, it is not possible to keep the files. You will have to back them up (e.g. to external USB disk) or create and then move them to a separate partition.
