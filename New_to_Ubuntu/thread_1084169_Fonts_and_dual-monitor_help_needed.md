### Post by RomeReactor on 2009-03-01
Hi. Regarding the fonts in Firefox, you might want to install msttcorefonts; search for them in Synaptic, or to install from the terminal:
```
sudo aptitude install msttcorefonts
```
Make sure you have the 'Multiverse' repository enabled in 'System->Administration->Software Sources'.
