### Post by 3rdalbum on 2011-04-27
> **Solstice Bahai said:**
> 
Q1 I have an Acre Aspire 5740 Notebook, is it worth duel booting Ubuntu with Windows7 (host OS)??

If you want to use both Ubuntu and Windows 7 on this computer, then yes.

> Q2 Ubuntu Netbook Remix, how often does this distro update compared to regular Ubuntu? 

Q3 When is the next version/update?

The next version is 11.04 which will be released in under 48 hours time. The Netbook Edition is being removed, and now the Desktop edition is suitable for desktops and netbooks. You'll like it :-)

When it's released, you can do an in-place update using Update Manager. It will probably go fine. Or, you can do an "Update Install" which is a clean install that preserves your home directory and the software you already have installed (it reinstalls new versions of the software you already have on 10.10). You can do this by booting up the Ubuntu 11.04 CD and running the installer, and selecting "Update" when asked.

The Update Install is pretty cool, and there's another advantage: You can beat the rush and download 11.04 Beta 2, install it now, and then run Update Manager to get those last few packages to make it the final release version.
