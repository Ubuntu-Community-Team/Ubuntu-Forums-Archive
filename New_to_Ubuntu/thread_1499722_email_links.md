### Post by davec64 on 2010-06-02
I suspect your messages are displaying in Plain Text.

Assuming you're using Evolution, go to **Edit** >> **Preferences**.

On the window that opens, select **Mail Preferences** (on the left hand side). On the **HTML Messages** tab at the bottom there's a section called **Plain Text Mode**, from the drop down list select **Show HTML if Present**.

Hope that helps!
