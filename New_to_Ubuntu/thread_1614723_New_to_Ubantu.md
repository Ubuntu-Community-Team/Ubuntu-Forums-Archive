### Post by waynefoutz on 2010-11-06
It's an operating system, much like Windows. But there are some major differences. You, for the most part, can't run Windows programs like Outlook, but there are alternatives, for instance, Ubuntu comes with Evolution as it's email, which in my opinion is far better. For web browsing, it comes with Firefox, and you can install Google Chrome or Opera as well. It also comes with a free, open source version of Office. The main difference between Ubuntu and Windows, is the way you install software. With Windows you either buy it from the store on disk, or download it from the web. Ubuntu has Ubuntu Software Center, which is much like the app store on an iPhone, only everything is free. Another difference is installing drivers. For most people, there are no drivers, everything just works. If drivers are needed, 99% of the time, Ubuntu will download and install them after it asks you for permission to do so. It couldn't be much easier. With Ubuntu, you don't have to worry about viruses, spyware, corrupted registries, or fragmented hard drives. A two year old install runs just as fast as it did when it was fresh. There are a lot of advantages, and some disadvantages. If you have a huge library of Windows software, most of it will be useless. On the other hand, if you name a program, there are most likely half a dozen Linux programs you can install on Ubuntu that does the same thing as well, if not better.  

If you have a problem. just post it on this forum, and you'll usually get an answer within minutes.
