### Post by Crandom on 2009-05-16
Banshee is the player I use the most (good file format support, fast, good sound, iPod support). Get it by typing this into a terminal:
```
sudo apt-get install banshee
```

Songbird is also VERY good (has all the features of banshee and extensions), although is still in "beta" when it is finished it will be amazing (and it's cross platform). It's (literally) like the Firefox of music. You can't get it from the repositories, so you have to go to: [http://www.getdeb.net/app/Songbird](http://www.getdeb.net/app/Songbird)

The project web page is: [http://getsongbird.com](http://getsongbird.com)

### Post by mthei on 2009-05-16
I haven't tried the new version of Audacious, but have used the 1.x series for the last two years as my main music player. If you are looking for something without a library and those extra features, you may want to look elsewhere, but I have had very little complaints about Audacious. It's a nice, lightweight player that comes with a lot of plug-ins as well.
However, if you feel like compiling something yourself, you may want to look into XMMS, as it sounds just as good as Audacious, but is even more lightweight.
If you want a full-featured audio player (with a library, ability to download podcasts, get artwork, etc...), I would recommend Quod Libet, although there's nothing really wrong with Rhythmbox, Exaile or any of the others mentioned.

### Post by wbee on 2009-05-16
Hello,

For compact disks,I like VLC. I have not tested it double blind or anything,but for classical and jazz,it sounds good to me. VLC is in the Synaptic Manager.

For streaming radio,I like Rhythm box,because you can add the stream and simply turn it on.

Rhythm box is installed by default. The first time you try to play a streaming audio signal,it will prompt you to download a plug in. Do so.

If you uninstall a player,remember to check System<Administration<Computer Janitor to take out all the extra files it installed for it to work.

Download a lot of players,find the one or two you like,and then un install the rest.

### Post by swoody on 2009-05-18
Hi chayanmann, and Welcome to Ubuntu, and the Forums ):P

If you are running Ubuntu, I would recommend Exaile. If you are running Kubuntu, then I would recommend Amarok. I find these players offer a great amount of options, and I have really become used to the interfaces. To install them, you can either search for them in Synpatic Package Manager, or run a command from the terminal:
For Exaile:
```
sudo apt-get install exaile
```
For Amarok
```
sudo apt-get install amarok
```

This is obviously just my opinion. I have come to it by using and testing out many other media players out there. As others have said, the best way to find the 'best' program, is to test them out, and find one you like more than the others :D
