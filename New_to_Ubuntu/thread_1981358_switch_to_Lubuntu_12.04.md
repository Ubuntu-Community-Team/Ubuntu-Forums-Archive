### Post by ajgreeny on 2012-05-16
> **Autodave said:**
> Machine will boot and run fine from either USB or CD.  But, when trying ti install, boots out right after entering name, password, etc.  Text on screen is different everytime, but it basically just hangs there.
It sounds as if this is a repeat of a problem many are having installing any of the ubuntu family of live CDs, and that includes ubuntu 12.04, Xubuntu 12.04 and Lubuntu 12.04, maybe aslso kubuntu, but I've not tried that or KDE for a long time.

I think it is probably related to a problem in ubiquity, the installation application on the live CD, and you may find that the alternate installer will allow you to install ubuntu with no problem as it does not use ubiquity, instead relying on a text based installation procedure.

This is exactly the difficulty I have had with both ubuntu and xubuntu, so having downloaded and burned the live CDs of both, I then needed to download the alternate install version of both to get the OS to install.  When the installer crashed it just left the spinning cursor on screen but nothing happened beyond that.

See [http://ubuntuforums.org/showthread.php?t=1965802](http://ubuntuforums.org/showthread.php?t=1965802) for much more discussion of the possible causes and solution.
Good luck!
