### Post by Ampi on 2010-08-19
Make sure you have the program Gparted (Partition Editor) installed. If you have you should be able to see it in System -> Administration. If not you can install it through the Synaptic Package Manager which is in the same menu as just mentioned. 

There there is the possibilty to resize partitions and such, but be aware!!! I strongly recommend a backup of your music and files and inform yourself very well on what exactly you are doing. So read the instruction of Gparted well and read some more in the forum. There have been more users with partition problems (including me :))

Another thing, I do not know what Netflix is but I do know that if you use Vista simply because of iTunes and you use iTunes just because you have an iPod, you don't need Vista for that. I always have used iPods and Ubuntu together without a problem. But that is a whole other discussion ;)
