### Post by squakie on 2012-12-03
And......if you installed it in Wine but are trying to access it outside of Wine, it's not going to work.  Things installed in Wine only apply to Wine - you have to open them as a Wine application.  

So, as already mentioned, best to use a native torrent client.

And remember - if you install a Windows program in Wine, it has to be run in Wine.
