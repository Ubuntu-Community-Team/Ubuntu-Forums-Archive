Hi cannot help with the command in terminal. However you can start 'System Monitor' and look in the processes tab for 'Evolution'. This will show the PID and right-click will give some options including 'Kill Process'. You will get a warning about possibly losing data and harming your system with the options cancel or continue. The choice is yours.

### Post by TeoBigusGeekus on 2011-06-22
By the way, the correct command to find evolution's pid would be
```
ps aux|grep evolution
```
or simply
```
pidof evolution
```
