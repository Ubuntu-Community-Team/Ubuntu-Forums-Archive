### Post by grahammechanical on 2015-08-19
I do not know what you are talking about. I do not have Windows. But I do know that Ubuntu has the Dash. It is a slide out panel that can be resized to full screen and it has a search panel and it will search on the computer and online. The latest versions of Ubuntu come with something like 100 scopes which do the searching. We can filter results and deactivate searching offline and set other privacy features.

You should download a Ubuntu install ISO image and burn it to DVD disc or USB memory stick and then run it in live mode/session and test for yourself to see if the Dash works to your satisfaction or that you can adapt your working practices to fit the way Ubuntu scopes work. 

If we are so inclined we can even write our own scopes. There are tutorials.

As regards applications, well, it all depends of the design of the application. Does it not? You did mean application windows when you said "in the open windows," did you not? All this can be tested in a Ubuntu live session. 

Regards.
