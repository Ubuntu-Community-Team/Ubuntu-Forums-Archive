### Post by Son of William on 2009-11-22
I assume you have access to the tutorial so I will not repeat all the steps.  I don't have your modem so it is impossible for me to do this for myself to see the steps.

Anyway, here is some suggestions to get you going:

You have the .tar file on your desktop - right?  If so, the tutorial says you need to "extract the contents to your preferred directory using the following command"

I suggest that, instead of using the terminal command, you simply double click on the icon for the tar.gz file 

That should open up the file in the archive manager.

Extract the file to a directory of your choice.  You can use the create folder button (comes up after you click on extract) to make a folder.

I suggest making a directory of your home directory called Cricket or something else easy to remember.

After that, resume the tutorial.

Come back here and post if you run into a problem/question re: the tutorial.
