### Post by 3rdalbum on 2011-11-11
Some of the Compaq Presario v3000s come with a 32-bit Intel Core Solo/Duo CPU, some come with an AMD Turion which is 64-bit. If the latter, then you can use the 64-bit edition on it regardless of whether your Windows is 32-bit or 64-bit.

If your Presario has a 32-bit CPU such as the Intel Core, then you will have to install the 32-bit edition, which will run on 32-bit and 64-bit CPUs.
