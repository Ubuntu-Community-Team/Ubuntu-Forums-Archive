First off, try going to System>Administration>Hardware Drivers and see if there is an available driver in there. If there's not one available for your sound card, you may want to go [here]("http://www.realtek.com.tw/downloads/downloadsCheck.aspx?Langid=1&PNid=24&PFid=24&Level=4&Conn=3&DownTypeID=3&GetDown=false") and download the Linux driver manually.
