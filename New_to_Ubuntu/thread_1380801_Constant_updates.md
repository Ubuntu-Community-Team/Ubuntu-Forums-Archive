### Post by audiomick on 2010-01-14
This is pretty normal. Ubuntu works a bit different to MS. MS tends to put out a big update package every few weeks or months, I think. In the linux world, the updates become available as the become ready, i.e. faster than MS. 

My experience is that the updates come in waves a bit; that is to say you will have lots for a while then not so many for a while then lots again.

You should only start to worry if you notice that the system is wanting to install updates that you know you have already installed. Otherwise, just push the button and let it do it's thing.

I did have a thought today; occasionally one sees posts from people who have had something crash after an update. This is not unique to Ubuntu or linux: I have had the same problem in the past with windows updates :)
Anyway, if the updater wants to update anything major like a kernel or something, it might be a good idea to do anything that you really urgently need to do at the time before you let the update run. This should not scare you, it is just a thought...;)
