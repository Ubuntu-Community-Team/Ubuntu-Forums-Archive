### Post by snowpine on 2012-09-16
All of the Ubuntu "flavors" are the same base operating system, with different Desktop Environments or "skins." You can easily install multiple DE's and switch between them depending on your mood that day. As I mentioned in your other thread, you can add Xfce (Xubuntu) to your existing Ubuntu install following this easy how-to with screenshots: [http://www.psychocats.net/ubuntu/xfce](http://www.psychocats.net/ubuntu/xfce)

To install LXDE (Lubuntu) it's the same process except you install **lubuntu-desktop** instead of xubuntu-desktop. 

Give it a try, it will only take a few minutes and you don't need to reinstall. :)

### Post by newb85 on 2012-09-16
Desktop Environments are not "skins".  The term *skin* is more akin to *themes*.

There are more differences between the Ubuntu flavors than just the Desktop Environments.  In the lighter flavors, many other packages have been replaced with lighter alternatives or omitted altogether.  If you simply install lubuntu-desktop or xubuntu-desktop on a Ubuntu setup, you will end up with all packages from both flavors.

Lubuntu is considerably lighter than Xubuntu.

I'm curious, why didn't regular Ubuntu go well?  The specs Toshiba lists for your machine don't look spectacular, but I would expect them to run 12.04 without any issues.  I've got a friend running 12.04 on my old HP G-50, which has basically the same specs.
