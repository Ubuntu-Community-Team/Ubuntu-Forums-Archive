### Post by sdennie on 2008-05-18
You are best off just using synaptic or Add/Remove and searching for the application you want to install.  The repositories contain many thousands of applications.  If you are new to linux, I would not recommend installing applications that are not in synaptic or Add/Remove until you become more familiar with the system.

Really, Synaptic should fulfill 99% (if not more) of your software needs. Go elsewhere only if you have to. A site like getdeb.net can help.
