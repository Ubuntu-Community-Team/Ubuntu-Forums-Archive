Wait a minute! Maybe he installed Ubuntu *in* Vista, using Wubi installer. In this case there is no special Linux partition, and you uninstall it like you uninstall any Windows program (Add/Remove programs).

Wubi was added in 8.04 to make it easier for new users to try Linux, without making a new partition and stuff
