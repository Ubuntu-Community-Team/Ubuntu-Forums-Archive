### Post by dje on 2008-05-22
You could take a look at APTonCD [here]("http://aptoncd.sourceforge.net/"), it backs up selected programs to cd/dvd media
To install it type the following in a terminal:
```
sudo apt-get install aptoncd
```

hope that helps,
