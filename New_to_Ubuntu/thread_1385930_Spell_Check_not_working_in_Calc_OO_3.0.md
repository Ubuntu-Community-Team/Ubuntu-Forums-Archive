### Post by Costas100 on 2010-01-22
> **ubhi said:**
> no, i check the entire sheet, but msg comes, spell check complete with no error.. but error & mistakes are there in that sheet.
I also have OO ver 3.0 and spell check just fine. It underlines with red the errors.
In OO SS you must go to Tools and then select Options and then Language settings / Language. Use the arrows to find and select the language you wish to check and click mark the box "for current document only" if it is only a one time deal (otherwise do not mark). If you cannot find the language you are looking or if it is does not have an ABC mark next to it then you need to install the language from System / Admin / Language support and if it is a second language then you must also add it in System / Preferences / Keyboard / Layout
Good luck to you
