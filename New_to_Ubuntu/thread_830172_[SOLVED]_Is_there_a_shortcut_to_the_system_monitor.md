### Post by waspbr on 2008-06-15
just go to ubuntu "start" menu, go to System>administration>System Monitor, just right click on the icon and that should give you the option to create a short cut where you want. 


You can check the commands involved by right clicking the start menu, and browsing to the program you want to make a short cut of. Once you find it, right click and select properties, that should give you all the necessary details for any launchers you might want to create (manually).
