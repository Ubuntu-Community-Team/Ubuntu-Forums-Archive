### Post by houbysoft.xf.cz on 2008-10-11
Oh and for copying the error message (in terminal, I expect), you should be able to just select the text and copy it using "Edit > Copy" or something like that. Or, you can use the '>' operator to redirect the output to a file, for example
```

program_name -arguments > error.txt

```.
Then just open error.txt and your message should be there.

Hope this helps.
