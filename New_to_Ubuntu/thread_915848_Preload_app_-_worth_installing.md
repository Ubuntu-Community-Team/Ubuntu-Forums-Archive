### Post by kaibob on 2008-09-10

### Post by sdennie on 2008-09-10
I think it's probably useful for most people.  You may not notice any drastic difference but, essentially, the idea is that it learns what applications you use most and tries to keep those applications cached for quicker access the next time you use them.  This is really just an extension of what the kernel does anyway but, it tries to do it in an intelligent and learning manner.  Using it won't cause any harm and won't use extra memory.  On any linux machine the cache will occupy all unused memory after long enough usage regardless of how much RAM you have.  Preload just tries to make sure the cache is full of useful stuff.
