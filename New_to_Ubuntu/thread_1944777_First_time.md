You have probably read the wrong things (outdated information, or information for a particular ill-supported device).

The short answer is, most modems work out-of-the-box, especially if you've got a popular one. It's actually easier to use them in Ubuntu than it is in Windows because you don't need to install extra software.

The good news is that you don't need to read up to see if your modem will work out-of-the-box. Have you got an Ubuntu CD downloaded and burnt already? If so, boot up your computer from it and choose the "Try Ubuntu" option when it appears. Plug in your mobile broadband stick and go to the Network Manager indicator on the bar along to the top of the screen. Go to "Edit Connections" and click on the Mobile Broadband tab. Click Add, follow the prompts. When it's all set up, go back to the Network Manager indicator and choose your new connection.

When you get a notification that your device has connected, open Firefox and see if it works. If it does, then you can double-click the Install Ubuntu icon on the desktop and start your Ubuntu adventure.

### Post by ccrs8 on 2012-03-22
Definitely a perfect use case for a LiveCD test.  Burn the downloadable .iso file to a CD, boot your computer off of it, and you'll have a fully functional Ubuntu environment (although it will run slower than if it was installed on your hard drive since it is running off of a CD or USB stick).  Then you can test out all your hardware to see if any of it won't work.
