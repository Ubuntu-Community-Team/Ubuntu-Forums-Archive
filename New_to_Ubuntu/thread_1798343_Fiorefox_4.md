### Post by jtarin on 2011-07-06
If you can find the 4.0 version you can use these same instructions just adjust them accordingly.I couldn't find 4.0.
These are for reverting to 3.6 but will work for any version.

Uninstall FireFox 4 via Applications -> Ubuntu Software Center

Download FireFox 3.6 from [http://www.mozilla.com/en-US/firefox/all-older.html](http://www.mozilla.com/en-US/firefox/all-older.html)

Unpack taball tar -jxpvf firefox-3.6.17.tar.bz2
Run via terminal command ./firefox/firefox
