### Post by ajgreeny on 2012-04-08
The newest flash update is terribly broken in linux and either does not work at all or shows everything  as a blue colour.

See [http://ubuntuforums.org/showthread.php?t=1948626](http://ubuntuforums.org/showthread.php?t=1948626) which is now a long thread I started when this problem first arose, and gives several ways to overcome the problem.

I recommend you download an older flash version from here:

[http://helpx.adobe.com/flash-player/...-versions.html]("http://helpx.adobe.com/flash-player/kb/archived-flash-player-versions.html")

I downloaded flash player 11.1.102.63 (174MB) which is a zip that  contains all plugins for all 32 and 64 bit systems with windows, mac,  linux, solaris OS's

Extract the appropriate tar.gz file for your system, mine was /flashplayer11_1r102_63_linux.i386.tar.gz, which I then used in the **advanced mode**, custom use of **flash-aid** add-on for firefox. This will also remove any other versions of flash on your system and it will stop any more updates for flash killing the flash ability again.

There is plenty more info on that thread.
