### Post by Impavidus on 2018-09-03
You just installed Ubuntu. I assume you still have the live disk. Use that to boot the computer, then install and run boot repair. Don't worry about 32 or 64 bit.

BTW, you can't run 64 bit Ubuntu on a 32 bit computer. You can't even boot the live disk. But most computers made in the past 15 years are 64 bit.
