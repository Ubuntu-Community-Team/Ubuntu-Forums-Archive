### Post by Ozymandias_117 on 2010-05-18
You can make a persistent USB Key using the "Startup Disk Creator" in System -> Admin - Don't do any updates though, they sometimes break it.

Another option is Puppy Linux, which will ask you if you want to save your changes to a USB key when you shutdown.


Edit: One benefit of Puppy is that it can be loaded to RAM which makes it run fast, unlike Ubuntu.
