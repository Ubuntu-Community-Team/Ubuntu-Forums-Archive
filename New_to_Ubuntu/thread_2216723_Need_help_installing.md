### Post by buzzingrobot on 2014-04-13
If you haven't already, take a look here: [http://www.ubuntu.com/download/desktop](http://www.ubuntu.com/download/desktop).  Midway down the right column are instructions for burning ISO install images on USB sticks and DVD's. (Has to be a DVD, a CD is too small.)

Even though a file manager may display several internal files in an install image, it must be burned, not copied, to the DVD or USB.

When you boot the install DVD/USB, you see a choice to install Ubuntu or to run it in a 'live" seesion. That runs Ubuntu off the DVD/USB without impacting the hard drive.  It's a bit slow on a DVD, but it is a useful way to take a look at Ubuntu.

Also, back up any files on XP you want to preserve, even if you intend to install Ubuntu in dual boot mode with XP. Better safe than sorry.

### Post by jbaerboc on 2014-04-16
As was previously mentioned some Windows files ".exe" may not run properly in Ubuntu [or any Linux distribution]. A similar type of file for Ubuntu would be a .deb file, at least as far as using it for installation of a program is concerned. You can run many Windows programs using wine [i do it for gaming] but most of the time it's best to find a Ubuntu native alternative. For instance if you use MS Office Word you can instead use LibreOffice Writer or something like that. But let us know and we can try to help :-).
