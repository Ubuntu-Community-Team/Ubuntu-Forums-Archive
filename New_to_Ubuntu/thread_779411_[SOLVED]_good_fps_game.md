The zip contained all the binaries for the different architectures. So all you have to do to launch the game is this:
```
./ioUrbanterror.x86_64
```
although if you aren't using a 64-bit OS, this is what you should type:
```
./ioUrbanterror.i386
```
I suggest you find an icon for Urban terror, and make a launcher with Gnome or KDE
