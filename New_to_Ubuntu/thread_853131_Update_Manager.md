### Post by Cypher on 2008-07-08
The updater manager will only install updates for programs that you've already got installed.

Packages are contstantly updated with newer versions of code or security bug fixes. So it's usually a good idea to keep up with the updates.

Majority of the updates will be for applications that don't require a restart, but once in a while a Kernel update will come through and you will have to restart the machine to use the new Kernel..
