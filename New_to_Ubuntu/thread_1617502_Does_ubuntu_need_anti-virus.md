### Post by Nightstrike2009 on 2010-11-09
ClamTk is available in repos if your worried it includeds ClamAV, never suffered a virus on Ubuntu myself though.

Terminal **install command** is:
sudo apt-get install clam-tk

and **for A/V signature updates**:
sudo freshclam

You can also use software manager to install clamtk, but you have to use that terminal command to update the signatures for some weird reason.
