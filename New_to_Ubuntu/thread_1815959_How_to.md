### Post by westie457 on 2011-08-01
Recommendation is go for 10.04.3 LTS.

Download the ISO burn it to CD as an image so it will boot.
Set your bios to boot from CD. At the welcome screen select try. Play around with,see what it can do. Do this for a few weeks so you can get used to it. When you are ready to install 'Google' is your friend as there are many guides on how to install. If you have any doubt come back and ask.

All the while you are just testing from a CD it will run slower than a full install because everything is on a CD. Also remember that Ubuntu is not Windows so while some things are familiar there are a lot that are strange.

It is a learning curve and a very good one. 

Installing Natty will come with own set of problems. Whereas most of the issues have been sorted with Lucid and has support until April 2013.

Welcome to the Forums and the wonderful world of Linux/Ubuntu

### Post by 3rdalbum on 2011-08-01
There's nothing wrong with your CPU or memory for running Ubuntu. Your graphics card sounds a bit weak to use the Unity 3D interface (it might work depending on which one it is) but you can still use the Gnome Classic interface which will automatically come up onto your screen if you can't run Unity 3D.

You could also try installing Unity 2D if you really want the latest GUI environment.

I can't say about your other hardware, whether it's supported or not, but if it's a little bit behind the times and not too unusual it'll probably work okay.

The simplest thing to do is simply try Ubuntu! You can boot it up from the CD, and try it out as much as you want. It won't touch your hard disk until you actually install it.
