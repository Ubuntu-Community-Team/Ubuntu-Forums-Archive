You might try the newer 2.41, but bear in mind it will want to bring some KDE dependencies with it. This is not a problem if your running Kubuntu already, but if not prepare for some additional downloads.

Install amarok 2.4 in Ubuntu 

Open the terminal and run the following commands

    ```
sudo add-apt-repository ppa:kubuntu-ppa/backports
    sudo apt-get update
    sudo apt-get  install amarok

```
