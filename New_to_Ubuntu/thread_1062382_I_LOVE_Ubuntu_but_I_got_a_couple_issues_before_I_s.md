### Post by jimmy the saint on 2009-02-06
Flash will not be problematic.
open a terminal Applications>accessories>terminal
and type in
```
sudo apt-get install ubuntu-restricted-extras
```
That will give you flash, java, better fonts and a few other goodies (like mp3 support)

quick, easy and painless.  The Ubuntu way!
