### Post by Musicmaker384 on 2012-07-06
I believe Rhythmbox works much like iTunes in the respect that it can index all music files on your computer. I've found that it indexes them no matter where they're stored. I've tried I've downloaded a lot of my music through Ubuntu One and it appears in Rhythmbox right away, so you don't have to go through that 'Add File to Library' crap you do on iTunes. The tutorials already posted are probably the best you're going to get, but here's a link to more information about the program and things you can add to it:

[http://projects.gnome.org/rhythmbox/](http://projects.gnome.org/rhythmbox/)

Hope that helps.
