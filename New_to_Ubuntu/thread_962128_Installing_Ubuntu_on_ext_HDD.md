### Post by wolfen69 on 2008-10-28
when you see the final screen (maybe the second last) before the installation begins, click "advanced" and make sure grub is installed to the usb drive. or just disconnect the internal drive before installation.

---

### Post by colinlr on 2008-10-28
What I have done is boot off live cd and add usb drive, then click on install and select the usb as your install location.
On the bios of the computer you want to use make sure usb is in the boot order above the HDD and the box will boot off the usb
