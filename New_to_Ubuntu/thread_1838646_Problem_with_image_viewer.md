### Post by coffeecat on 2011-09-04
I don't know the answer for EoG and Shotwell, etc, but try Gthumb. It's an excellent image app with some basic editing utilities, a very useful renaming tool and will also do what you want. It will sort by date (I think from EXIF), dimensions, file modified date, filename, file path, file size and no sorting. And you can opt for inverse order for all of those.
