### Post by ad_267 on 2009-02-25
Image Magick is a command line program. I think you'd be better off using GIMP which is installed by default. You can just open up the image and resize it by going to Image -> Scale Image then use Save As to save it as a different file type if you need to. If GIMP's a bit too much then maybe try Krita.
