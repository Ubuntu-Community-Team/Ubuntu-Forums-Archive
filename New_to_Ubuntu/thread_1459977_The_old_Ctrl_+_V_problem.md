### Post by RTrev on 2010-04-22
Shift+Ctrl+V might be easier to remember.  I've also lately found that sometimes Ctrl-V doesn't work when I expect it to.  In all cases so far, though, I've found that the simple trick of middle-click works.

For example, highlight some text, do NOT bother to copy it but just leave it highlighted, put the mouse pointer where you want the text and middle-click.

The cool thing about this is that it doesn't over-write what's in the copy/paste buffer.  So I could have something ready to paste, use the highlight and middle-click thing, and still be able to later paste what is still in the buffer.
