### Post by 4Orbs on 2009-09-11
Are you artistic? Probably the most instant gratification can come from customizing the desktop theme... maybe create some new graphics for the panel or wallpaper using Gimp. Try different sets of icons or different colors.

A fun twice-a-month project might be distro hopping. Install a virtual machine, then try out a bunch of different Linux distributions by installing them inside the virtual box. Safe and a bit time consuming. Or create your own custom distro by building up from a minimal Ubuntu installation.

Check out Synaptic or the add/remove Software menu. Science, math, art, office apps, many different web browsers, games. Plenty of things that rival or exceed any comparable Windows application.

Lurking around the Ubuntu forums is great fun and educational. I guess you can do the same things in Windows, but Linux actually encourages you to reach out beyond the mundane.
