### Post by RyanVanDiemen on 2008-07-22
Hey, Calvin B, well, I used to run Gnome for some time and recently switched to Xfce and you can easily change theme by unpacking the tar.gz file and moving the folder to your .theme folder in your home directory. I used to do the same while using Gnome so no changes for me (installing them via install button didn`t used to work very often). You can install icons by moving the unpacked folder to your .icons folder. After that you will see these themes in "User interface" section of your "Settings".
