### Post by Temposs on 2009-02-23
Tomboy actually has a very nice organization. It has "Notebooks", which you can create and then place notes into. So those are sort of like folders. Tomboy is modeled around a wiki, so the main way to create sections is to use the "Link" button to create notes on specific topics. Once you get used to it, it works quite well. It's just different.
