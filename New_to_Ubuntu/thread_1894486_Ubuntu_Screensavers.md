What version of Ubuntu are you using? There are plenty of screensaver packages in Ubuntu's repositories, but if youa re running 11.10 you'll also need to insall xscreensaver and ad it to your startup applications (as Ubuntu doesn't come with screensavers any more due to them being mostly useless on any current display).

Anyway, start by seaeching for xscreensaver, that should bring the xscreensaver itself and alt least a couple of screensaver packages. You might also want the rss-glx -package, which contains some quite pretty OpenGL-based screensavers.

I doubt you'd find many Linux screensavers from the web, they are pretty much always available through each distribution's own sources so there is little need to distribute them on web sites...
