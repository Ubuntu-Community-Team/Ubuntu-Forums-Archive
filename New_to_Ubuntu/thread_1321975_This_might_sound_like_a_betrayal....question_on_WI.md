### Post by Mark Phelps on 2009-11-10
>  ... Can you install any windows software you want on WINE.
... 

Basically, no.  

The link that scragar gave you to the Application Compatibility database is worth using to check out the ratings for individual MS Windows apps.  As you will see when looking at that, ratings vary enormously by product and by version.

In my experience, if you already have a working MS Windows installation, a better solution is to use a multi-boot setup.  That way, you get the benefits of Linux nearly all the time, and can still use the few MS Windows apps you need in native mode.
