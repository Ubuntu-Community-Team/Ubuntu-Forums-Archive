It's my opinion that it really doesn't matter.

The server edition installs a lamp, which you don't need.

The desktop edition installs a gui, which you don't need, although it might be handy from time to time.  If you decide to use the desktop version, then you should use xubuntu since xfce is lighter and will use fewer resources.  Of course, can also just set it up to not start the gui automatically, and then you can start up to a command line and then only start the gui when you want to use it.

### Post by dwasifar on 2010-04-04
You can use either, for what you're doing.  If you prefer a graphical interface to a text interface, it will go easier for you if you use the desktop version.  

If you were making a web server, I'd say start with the server edition and add the GUI if needed, but if it's only a file server you have your choice.
