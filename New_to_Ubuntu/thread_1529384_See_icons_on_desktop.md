Mariane

You can resize it. If you hover over an empty area of the window, a tool bar should appear on one of its sides. Within that tool bar there's a little rectangle with two arrows in it that are pointing to the corners of the rectangle; if you drag that rectangle you'll resize the folder view window. 

To get back to the older view you're used to, first close the folder view window so you have a blank desktop again. Then, right-click your desktop and select Desktop Activity Settings. Then select Activity. Change the Activity from type Desktop to Folder View. (adjust location to Desktop folder, if needed).
