### Post by Steve1961 on 2008-11-02
Basically it's a version of wine for running windows applications such as MS Office.  However, unlike wine, it's sold by codeweavers and they do their  own development to make sure certain applications (e.g. office) run as well as possible.

To install it, if you have the sh file, just drop it in your home directory, open a terminal, and type sh install-crossover-pro-7.1.0.sh  But check that is the name of the file you've downloaded, and if it's different replace the name with the name of the actual file you have.  Then just follow along....

### Post by oldsoundguy on 2008-11-02
If you install it (as they instruct) and go to the help button, you will find a bunch of answers there .. 
Crossover is a third party program that allows the usage of SPECIFIED Windows program to be used within any Linux build (not ALL Windows programs by ANY means)  This means SOME games and a lot of serious and real stuff such as many Adobe programs.  When you install it, you will get a list of the programs it supports and you will get asked if you want to install them.  (of course, you have to HAVE a legit disk of that program to do so!)

WINE, which comes free in the builds, also does some of the same programs, but it also supports some that Crossover does not and visa versa.

Then there is another: Cedega .. a lot of gamers use that one, but I do not.

Plus there is another PAID program that you can install IN WINE to enable it to handle more programs such as Steam based games.  That one is Bordeaux.
