### Post by mcduck on 2011-10-27
> **oserdavid said:**
> Thanks McDuck - is this historical accident or is there a good reason for it?

Yes, there is a good reason. The install CD image, as it was originally released, was tested to work correctly by both the developers and by the community. Since there are quite many Ubuntu versions supported at a time, and of course one being developed, there simply wouldn't be resources to thoroughly test all the the disc images if they were continuously updated to include every security fix.

Pretty much a similar reason to why Ubuntu versions only get security updates and fixes for more serious bugs after release. It's a question balancing between stability & predictability and the quick 6-month release schedule & up-to-date software.
