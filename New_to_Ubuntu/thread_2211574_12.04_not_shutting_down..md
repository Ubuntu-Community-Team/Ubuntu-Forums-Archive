### Post by Hadaka on 2014-03-16
Hi, give this a try...be cautious..copy and paste.

Best to copy and paste these commands.
**FIRST SAVE a copy of the file before changes...do..
```
sudo cp /etc/default/grub /etc/default/grub.old
```
```
sudo gedit /etc/default/grub
```
#This will open your grub file ...go to the line that says.

 GRUB_CMDLINE_LINUX_DEFAULT="quiet splash"  #this is line 11 for me...yours may be different.

change it to..
```
 GRUB_CMDLINE_LINUX_DEFAULT="quiet splash" acpi=force apm=power_off
```
*save and close gedit
then do..
sudo gedit /etc/modules
add one line below  ->  lp  <-
```
apm power=off
```
*save and close gedit
boot.
