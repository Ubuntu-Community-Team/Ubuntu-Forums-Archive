### Post by Bucky Ball on 2012-04-09
When you install 12.04 LTS just immediately install:

```
sudo apt-get install xfce4
```

Log out, choose 'Xfce Session' from the sessions pop-up. You will have the Desktop Environment of Xfce running on Ubuntu 12.04 LTS (with Ubuntu packages, apps, etc). You don't need to necessarily change to Xubuntu as the OS just because you don't like Unity.

I have been using xfce DE for ages and have no interest in Unity for now. On some machines I have what I just described and on other machines I have Xubuntu with a few Ubuntu apps installed. Mix and match ... ;)
