### Post by quixote on 2009-05-13
I'd suggest getting the 64-bit version of 8.10 (Intrepid Ibex).  Jaunty (9.04) can be a bit buggy still on some systems (mine :-( )  The only issue I've had with 64-bit is that initially the multimedia stuff wasn't working too well.  You can now get what you need from synaptic (flashplugin-nonfree, java6 and jre, vlc and vlc-plugin) and things seem to be working.  I have to admit I still get more firefox crashes than I've ever had before.  

If you want to play it safe, just install the 32-bit version.  The only disadvantage is that your programs are running in a more cramped environment, but unless you're a power user (eg lots of animation rendering or big statistics calculations, or heavy photoshop use (in virtualbox, since you're now past Windows :-D) ) you're not likely to notice much difference.
