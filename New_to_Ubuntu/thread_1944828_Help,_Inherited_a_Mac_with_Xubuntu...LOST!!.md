### Post by ccrs8 on 2012-03-22
I'm pretty sure the familiar chime is in the bios/firmware/whatever-it-is-called on Macs.  So hearing that isn't a sign that there is some max osx left over.  Also, do you already have the OSX media?  The G3 is an older machine with PowerPC, so any of the newer versions of OSX won't work (they are for Intel chips only).  OSX 10.5 is the last version to support PowerPC.

Anyway, boot the computer up with the CD inserted and hold down C - that should do the trick.  You'll be able to wipe the drive and install OSX 10.5.
