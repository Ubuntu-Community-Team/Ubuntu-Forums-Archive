### Post by ArthurPeale on 2012-02-28
I have this issue, also.  I've determined that it's not a user issue, as I thought that it once was, but in what order you logon.

As most of you probably know, you can switch between login sessions using <CTRL><ALT><Fxx>.

The session for F7 and F8 appear to be fine.  F9 is the one that has the black box.  Not sure about the others as I don't have other users currently logged in.
