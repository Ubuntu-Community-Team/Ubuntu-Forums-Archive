### Post by oldos2er on 2011-04-21
LTS desktop releases are supported for three years, LTS server version for five years. "Regular" desktop releases are supported for eighteen months. If you install 10.04 and upgrade to 10.10 it would be the same as installing 10.10, in terms of support.
