### Post by leunam12 on 2016-08-11
off-topic: Depending on what version of Word you have you may be able to run it in WINE. I run Word 2003 and 2007 successfully on all my Ubuntu machines.

On topic: Boot Windows and go to Acer's website and see if they have a firmware available for your optical drive; if they do, download it and run it, it may solve your problem. If that doesn't work then my best guess is that you need a new optical drive, either an external or internal.
