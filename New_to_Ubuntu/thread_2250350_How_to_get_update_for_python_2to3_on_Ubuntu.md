### Post by qamelian on 2014-10-28
Ubuntu includes both Python 2.X and 3.X in recent versions. Python 2 is the default for compatibility reasons, as not all the Ubuntu components written in Python have been adapted to Python 3.X yet. You can execute the Python 3 interpreter from the command line by typing:
```
python3
```
