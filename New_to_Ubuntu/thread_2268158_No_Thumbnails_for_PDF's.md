### Post by yetimon_64 on 2015-03-07
> Home > Show Hidden Files >.thumbnails > normalI get the impression you are mistaking icons and thumbnails.

The icon you've posted that you want on pdf files is NOT a thumbnail. 

A thumbnail is an image generated from a files content and used as the icon for that file. Each thumbnail of the same sort (for example if evince-thumbnailer is used for generating pdf thumbnails) will have a different image dependent on its content. You have posted an **icon** if you want the same image on all pdf files.

Icons are set by the theme chosen, not the thumbnailer location you listed. I have researched this issue all day but cannot test/check any possible fixes being on a Debian Wheezy install. 

The first answer at [http://askubuntu.com/questions/52138/how-do-i-change-the-icon-for-a-particular-file-type](http://askubuntu.com/questions/52138/how-do-i-change-the-icon-for-a-particular-file-type), may possibly help. Good luck.
