### Post by SuperSonic4 on 2009-06-04
K9copy is my personal favourite - it can create an ISO of the disc and reburn it via any image burning method.

By default it keeps the menus doing a straight copy but I'll see if it can remove them

edit: if you use the wizard you can remove dvd menus and any titles you don't want.

edit the second: k9copy needs kdebase-runtime and libmpeg2 as deps so it's probably better to wait until other suggestions come in

### Post by tacantara on 2009-06-04
I've been using OGMRip for a few weeks now.  It copies DVDs for me so I can convert them to MP4 (using Handbrake) for uploading to my iPod.  OGMRip is available through the Add/Remove software feature.  If it doesn't do the trick, you can always uncheck the box in Add/Remove Software to uninstall it.
