### Post by martrn on 2009-03-13
> **quarzark said:**
> I don't know WHAT the downloaded ISO image is.

An ISO image is a ISO 9660 file system disk image and its disc image format is supported by a great many software reading and disc burning applications.

See wikipeida page [http://en.wikipedia.org/wiki/ISO_image]("http://en.wikipedia.org/wiki/ISO_image").

Most cd writing/burning applications understand the ISO disk image format, so it is widely used.
