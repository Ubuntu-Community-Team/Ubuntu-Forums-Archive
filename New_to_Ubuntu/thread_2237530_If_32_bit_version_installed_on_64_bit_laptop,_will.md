### Post by whitesmith on 2014-08-02
> **JeQhdMD said:**
> I'm wondering if my full system capabilities are being seen and used from my 32 bit version of Ubuntu?

If you continue to use a 32-bit version of an OS, then you are limiting the system's capabilities. The theoretical maximum amount of RAM that a 32-bit processor can address is 2**32 (4294967296) bytes or 4 GB. From this must be subtracted memory for video and other functions, so 3+ GB is all you can use, regardless of the amount installed. If you update to a 64-bit OS, the memory limitation jumps to 2**64 bytes -- in simpler terms, a stadiumful.
