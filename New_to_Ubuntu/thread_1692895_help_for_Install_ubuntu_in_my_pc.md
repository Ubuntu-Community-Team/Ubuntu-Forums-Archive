### Post by 3rdalbum on 2011-02-22
First, back up all your data (this is a good rule for ANYTHING to do with partitioning).

Secondly, change your BIOS settings so the computer will boot up from CD before anything else. If you don't know how to access your BIOS settings, look on the screen that appears just as your computer starts to boot.

Thirdly, insert the CD. Eventually a screen will appear asking if you want to try Ubuntu or install it. I'd suggest trying it first, making sure that you can connect to your network and hear the sound in the included video files.

When you've verified that these things work, double-click the Install icon on the desktop. The rest is easy. When asked about the partitioning, you can just drag a slider to show how much space you want to give to Ubuntu, and it will downsize your Windows partition to create this space. Very easy. **Just make sure that your username doesn't have spaces and doesn't have capital letters.**
