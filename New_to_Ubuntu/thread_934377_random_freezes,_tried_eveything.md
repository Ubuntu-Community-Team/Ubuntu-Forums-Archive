If it's the kernel, then you can hope that it stops when Intrepid comes out; it has the newest kernel as far as I know, and it might have improved support for whatever seems to be causing the problem for you.

However, if your system works fine without compiz (as your post suggests to me), I, with my layman's knowledge and intuition, think that the problem might be with compiz' support for something (or rather, lack of support). If there's a package that fixes the problem, it's plausible that it will say in the package description -- try searching for compiz in Synaptics (or whatever manager you use) and have a look through the descriptions. You might be lucky. :)
