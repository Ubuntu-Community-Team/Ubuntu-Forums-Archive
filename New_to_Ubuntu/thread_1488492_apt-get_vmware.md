### Post by Paqman on 2010-05-20
You won't find VMWare in the Ubuntu Repositories. You'll need to download it from the VMWare site.

If you want a virtualisation package that is in the repos, try Virtualbox. It's excellent, although the open source version in the repos lacks USB and remote desktop support.
