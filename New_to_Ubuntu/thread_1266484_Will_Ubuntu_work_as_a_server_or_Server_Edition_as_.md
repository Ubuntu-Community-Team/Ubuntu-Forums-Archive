### Post by marshmallow1304 on 2009-09-14
Desktop edition will work fine.  Just install whatever you need (e.g. apache, squid, php, ssh, ftp, cups, samba, etc.)

The [Ubuntu Server Guide]("https://help.ubuntu.com/9.04/serverguide/C/index.html") is very useful for configuring various services.
