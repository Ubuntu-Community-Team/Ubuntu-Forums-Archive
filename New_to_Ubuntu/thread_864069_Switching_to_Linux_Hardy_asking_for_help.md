### Post by coffeecat on 2008-07-19
> **ragzkie said:**
> i may have some peripherals i want to attach though, like epson stylus EPSON Stylus CX2900 printer/scanner and EPSON Stylus C67 printer. I wonder if there is downloadable driver for them as i use them regularly. Thanks a lot in advance!

If a printer is going to work in Linux, there's a fairly good chance that it will work with the drivers that come with a default install of Ubuntu. Hewlett-Packard and Epson are probably the two most Linux-friendly makes. Try attaching your printer and seeing if it is autodetected when you go to System > Administration > Printers.

If you do need further information or if you have to install extra drivers, you'll find [The Linux Foundation OpenPrinting site]("http://www.linuxfoundation.org/en/OpenPrinting") useful.
