### Post by snowpine on 2009-01-29
Ubuntu has pretty good hardware detection. You should be able to boot your USB install on any computer (that supports boot from USB) and it will autodetect the hardware. Not all wireless adaptors, graphics cards, etc. are supported "out-of-the-box", same as if you were using the Ubuntu Live CD.  

Hope that helps answer your question!

### Post by beastrace91 on 2009-01-29
Ubuntu will boot on just about any computer with out having to install drivers, however if you are looking to do 3D on the system you will need to get GFX drivers. (Personally I always use the app. envy-ng, but they are also available in your package manger.)

I've never actually installed the OS to a thumb drive however I have used [unetbootin](http://unetbootin.sourceforge.net/) to create a "livecd" out of a pen drive.
