### Post by alexandremrj on 2008-06-16
Hello,

some programs may not install shortcuts in the main menu but you can always run them from your installation with two ways:

Press <Alt>+F2 for the "Run Command" window and in it insert "freemind" (without the quotes) or in the terminal simply enter "freemind" and press enter.

Both this ways will get the program running, although the second is best because you can see what errors the program may encounter. If there are no problems with them then you can add them yourself to the menu by pressing with the right button on the menu and selecting "edit menu".

In the window that appears simply choose the category where you want you shortcut and press on the right the "new item" button. Name is the name of the program or a name you want, in command is what you entered above in the terminal and description is something to help you. Press ok and you have a menu entry.

Hope that helps.
