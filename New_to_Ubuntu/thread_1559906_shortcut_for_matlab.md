Hi, I have installed latest version of Matlab in my Ubuntu computer. But there is no desktop icon. So to run the matlab I have to fire up a terminal and then go to matlab folder --> go to bin directory and then type ./matlab to start it. How can I create a desktop icon for easy access. ---

### Post by martinr on 2010-08-24
1) Right-click on the task bar and select "add to panel".
2) Select custom application launcher, and fill out the dialog
3) Drag or copy the new icon to the desktop
