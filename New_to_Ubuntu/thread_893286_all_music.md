### Post by rockerphil on 2008-08-18
seeing as it's through Firefox i'm guessing it either runs through Java or Flash, so i'd make sure they're both installed. you can do that with these commands

sudo apt-get install sun-java6-bin sun-java6-jre sun-java6-plugin

and

sudo apt-get install flasplugin-nonfree

then restart the browswer, and that should do the trick. hope it helps,

Phil
