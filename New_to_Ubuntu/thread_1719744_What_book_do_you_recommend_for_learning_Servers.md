### Post by yeats on 2011-04-02
> **papuccino1 said:**
> I'm experienced with Windows Server 2003/2008. Does Ubuntu stack up well against these professional solutions?

Perhaps this is a predictable answer on a Linux/Ubuntu forum, but I think all would agree, that YES, Ubuntu stacks up nicely against Windows server, especially if you factor in the price tag and the availability of thousands of free software solutions for various server needs.

> I'd like to broaden my horizons and learn about servers in the Linux world. Which distro would you recommend and what book should I read to get up to speed with it?

Well, Ubuntu 10.04 LTS is a good choice IMHO, and I can also recommend Debian stable.  Most of the best documentation for both is online, but I would recommend going to a computer-oriented bookstore (or a "big box" store with a decent computer section) and browsing the options (in that case, you'll find more books on Ubuntu than Debian).

There's a 10.04 Server Guide here:

[https://help.ubuntu.com/10.04/serverguide/C/serverguide.pdf](https://help.ubuntu.com/10.04/serverguide/C/serverguide.pdf)

And if your server-related questions get more specific, post them to the [Server Platforms]("http://ubuntuforums.org/forumdisplay.php?f=339") area.

I would also just download Ubuntu Server, Debian (there's no separate server edition, just don't install the GUI and there you go), CentOS, Fedora and others you might want to try.  If you don't have a spare machine, consider using [VirtualBox]("http://www.virtualbox.org") to set up several for testing.
