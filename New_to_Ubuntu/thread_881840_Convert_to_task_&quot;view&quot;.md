### Post by doublewitt on 2008-08-06
Hi,
I recently installed Ubuntu 8.04 on 2 computers - I really enjoy using this OS. On one computer, in Evolution, when I select an email and use the "convert to task" option - all is fine when I click on the "Task" tab - they are in view and I can select the columns to show. On the 2nd computer, when I do the same thing, nothing shows on the "Task" tab - there are no emails in view. I can't figure what I could possibly be overlooking. Would somebody be kind enough to tell what to do?

---

### Post by doublewitt on 2008-08-06
**OK, I found the solution to this**. 
When I installed Ubuntu on the 2nd computer, for some reason, (or maybe, without realizing it, I triggered the setting!), the option to "view preview" in the Task section was already checked. I don't know if the installation set it to that - but anyways, there it is - the solution! So if anybody else wonders...

*Actually, it's because the preview pane (bar) was at the top so the task list could not show. And I know that the install did this. I certainly wouldn't have moved the bar to the top!*
