### Post by Rubi1200 on 2011-02-04
The other thing you need to do after reinstalling Vista is use the Disk Management tool to shrink the Vista partition and leave space for Ubuntu. Don't try and format the space, leave it unallocated.

Then, fire up the Ubuntu CD and use manual partitioning and select the free space. If you use the install alongside option there is a chance things will get messed up again.

For more information, see here:
[http://ubuntuforums.org/showthread.php?t=1622388](http://ubuntuforums.org/showthread.php?t=1622388)
