### Post by jamie0nz on 2009-03-02
Best is a problematic description at the best of times ...

I use mgetty though I do not use the send fax facility. I have it set up to answer as either ppp dial in access or as fax. It served me well for the many years (up until a month ago) on slackware and it seems to work the same on ubuntu server, though the new_fax script caused some difficulties because the dependencies were not completely set.

I like that it can send an incoming email to a couple of email addresses. I have hacked the new_fax script so that it sends me a pdf rather than a png.
