### Post by lovinglinux on 2009-07-03
If you just like KDE interface you can install *kde-core* and *plasma* in Gnome. Check [this post](http://ubuntuforums.org/showpost.php?p=7536130&postcount=5) for a detailed explanation and screenshot of my Ubuntu (Gnome) running the KDE interface over it.
