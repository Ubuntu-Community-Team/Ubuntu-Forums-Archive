### Post by Baldrick_NZ on 2010-09-06
The pre-installed RhythmBox has iPod support, and access to 7digital (an iTunes equivalent). The downside to 7digital via RhythmBox is that for mainstream music it's currently practically useless outside the US, UK, Germany and another country which escapes me..

Another option, which is more like iTunes is Songbird. It too utilises 7digital, and seems to have international copyright licences so you can download practically anything music-wise worldwide. It looks more like iTunes, however I'm not sure whether it supports iPod synching.
The downside to Songbird is that it's creators have decided in their infinate wisdom to discontinue Linux support. You can still get the last stable version here: [http://www.getdeb.net/updates/Ubuntu/10.04/?q=songbird](http://www.getdeb.net/updates/Ubuntu/10.04/?q=songbird) You'll want to follow their install instructions before downloading too.

Hope that helps..
