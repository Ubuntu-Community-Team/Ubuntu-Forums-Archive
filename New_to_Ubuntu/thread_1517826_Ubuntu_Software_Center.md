### Post by Sef on 2010-06-25
The easiest way is with the Ubuntu Software Center.  To get more respositories, open the Software Center (Applications > Ubuntu Software Center) and then Edit >  Software Sources.  Then click on universe and multiverse: the 2nd and 4th from the top.
