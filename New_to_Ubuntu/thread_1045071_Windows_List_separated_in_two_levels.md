### Post by mcduck on 2009-01-20
The height a t which the panel starts using two levels for window list depends on couple of thinks, the font size used in the panel, and the GTK theme you use (as the theme defines how much padding all items have).

Just increase the panel size and at some point the window list will change to two levels instead of one.
