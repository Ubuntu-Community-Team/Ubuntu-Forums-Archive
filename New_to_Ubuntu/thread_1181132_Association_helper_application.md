### Post by jimmy the saint on 2009-06-07
It is easiest just to right-click and save the item to a folder and then use it that way.  The problem with adding the helper application to firefox is that every zip file you encounter will automatically be handled by that application, and you may not want every zip file to be automatically handled in any way.  Once it is saved to a folder, you can do whatever you want with it without the need for firefox to be able to do it.

If you really want to set it up though, you would go into edit>preferences>appications and set up zip archives to use the file-roller application.  I would still advise against it though.

Eidt:  jbruced (below) is 100% correct, but he is dealing with after you get it out of firefox onto your drive.  If you want to automatically open the zip file to see the contents inside, the dialog that pops up when you click the link will allow you to do so.

### Post by jbruced on 2009-06-07
Appearance is for themes, not for dragging in wallpaper, however, if it is a theme and is double zipped, extract it once to any location, them drag the new file onto the appearance window.

You can extract by clicking on the file in nautilus.

To change wallpaper, right click the desktop and follow the menu.

Let us know the name of the file if that doesn't help.
