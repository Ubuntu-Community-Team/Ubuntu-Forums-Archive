### Post by grahammechanical on 2015-12-03
I think that we are entitled to ask, what are you talking about?

Ubuntu is a Linux distribution and applications written to run on Linux do not have setup.exe or install.exe files. In fact any kind of exe file as those kind of executable files are written to run on the Microsoft operating system.

Utilities like Ubuntu Software Centre do all the downloading & installing for us. VLC media player is in Ubuntu Software Centre and all I have to do is click the button labelled "Install" and the Software Centre will install it for me. It is version 2.2.1. Which is the same as will be downloaded from here.

[http://www.videolan.org/vlc/index.en_GB.html](http://www.videolan.org/vlc/index.en_GB.html)

When I click the Download button a small dialog appears called Launch Application and I have the choice to send the file to Ubuntu Software Centre or to choose some other application. I am sure that if I went through with downloading VLC from the official VLC web site that after the download is complete the Ubuntu Software Centre will load and install the application.

And Ubuntu does come with a Ubuntu desktop guide called Ubuntu Help that gives useful information on how to do things.

### Post by coffeecat on 2015-12-03
@gawaine2, first of all, I very much doubt you have to negotiate the Software Centre captcha to install vlc. You should be able to do that just by selecting vlc for installation. Sign-on and captcha in the Software Centre is for a limited subset of restricted applications. I'm 99.9% sure that vlc is not one of them.

Secondly, as deadflowr says, you don't need to bother about the captcha if you already have an Ubuntu One account which, by virtue of the fact that you are posting here, you do have. If you are still having problems with signing on in the Ubuntu Software Centre for software other than vlc, have a look at this sticky post:

[http://ubuntuforums.org/showthread.php?t=2289812](http://ubuntuforums.org/showthread.php?t=2289812)
