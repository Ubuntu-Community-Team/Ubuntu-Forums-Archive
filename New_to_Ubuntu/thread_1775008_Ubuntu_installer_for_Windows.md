### Post by GWBouge on 2011-06-04
Yeah, the Wubi installer will install Ubuntu just like any other software in windows, and can be removed just the same.  It will just ask you what OS you want to use when you start your machine.  Though, I wouldn't use a Wubi install for long term use, or keep anything important on it.  It's there just so you can try out Ubuntu and uninstall easily if you don't like it.

You *can* partition the drive and do a real install, but that's a bit more difficult to put back to 'stock' if you need to get rid of it.  I'd say do the Wubi deal first off, see what it's all about, and once you get more comfortable you might want to put it on there for real.
