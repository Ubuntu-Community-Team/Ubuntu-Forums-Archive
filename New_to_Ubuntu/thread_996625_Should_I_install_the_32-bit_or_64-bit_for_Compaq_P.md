### Post by OutOfReach on 2008-11-29
Yes the 64-bit version will recognize more then 3gb.
But I think you can also get more than 3gb memory in 32-bit by either recompiling your kernel with PAE or installing the server kernel.

People say that 64-bit does improve performance but generally not a big difference.
