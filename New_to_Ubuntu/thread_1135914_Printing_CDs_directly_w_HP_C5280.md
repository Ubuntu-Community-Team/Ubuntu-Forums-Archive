### Post by Mark Phelps on 2009-05-05
I've found that adding GIMP-gutenprint provides a LOT more flexibility when printing directly to CDs.  Main problem I had was that direct printing with GIMP wouldn't let me set the CD outer diameter or the inner hub size -- so I alway had a lot of white space on the outer and inner edges.  Gutenprint not only lets me set those, it also lets me establish printer profiles for different size media.  Strongly suggest installing gimp-gutenprint as well.
