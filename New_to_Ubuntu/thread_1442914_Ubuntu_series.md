### Post by undecim on 2010-03-30
It's an improvement.

When you talk about Windows 7 being the latest version of Windows, you're just talking about a new face. In fact, it wasn't until Windows 7 that Microsoft even really changed the kernel (the core of the operating system) at all.

If you are looking for an equivilent of the XP>Vista>7 progession that Windows has, look for new versions major version of your desktop environment.

For example, KDE 3, which was included in Kubuntu before Hard Heron 8.04 looked something like this:
[http://sidenet.ddo.jp/winetips/files/kde3.2.png](http://sidenet.ddo.jp/winetips/files/kde3.2.png)

Now, KDE 4.3 included in Karmic looks more like this: [http://blog.linuxtracker.org/wp-content/uploads/2009/11/air.png](http://blog.linuxtracker.org/wp-content/uploads/2009/11/air.png)

That's pretty much the same change as XP>Vista or Vista>7

Either way though, the core of the operating system has been improved, made more stable and reliable.
