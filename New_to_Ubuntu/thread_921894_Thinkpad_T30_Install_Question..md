### Post by louieb on 2008-09-16
I've been dual booting Ubuntu and XP on a T30 for almost 2 years. It installed and worked almost without a problem. (Ubuntu tends to make mine run hot - solved that by using a cooling pad).  
How much memory does it have?
How did you install it? wubi? dual boot? Ubuntu only? 
Does the Ubuntu live CD boot to the desktop? 
Black screen on boot up from the hard drive is not good. Try running the memory check option on the live CD. 

If you think you might have a hardware problem the [ThinkPad Forum ]("http://forum.thinkpads.com/") is a great place to get help.
 
Another good place for  thinkpad/Linux stuff is [ThinkWiki - Linux on ThinkPad]("http://thinkwiki.org/wiki/ThinkWiki")
