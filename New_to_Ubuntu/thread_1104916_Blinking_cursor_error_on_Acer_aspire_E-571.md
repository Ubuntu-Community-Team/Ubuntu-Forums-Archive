### Post by rolandrock on 2009-03-24
John,

You should try Ubuntu's text based installer [http://www.ubuntu.com/getubuntu/downloadmirrors#alternate](http://www.ubuntu.com/getubuntu/downloadmirrors#alternate) 
This gives more install options and more feedback on hardware problems.  I had a Dell pc that had a strange hdd connector.  It caused the installer to crash and the only way to fix it was to add the word all_generic_ide to the boot options.   Something you can only do with the alternative text based installer.

Good Luck

> **eoingrisewood said:**

I think you have misundestood the nature of the text based installer.  The disk you currently have is a GUI installer that gives you access to _some_ options on install.  The text based installer offers more control and configuration options for problematic installs.

You need to download the text based installer, burn it and use it to install Ubuntu rather than the GUI installer you are currently using.  Have a look at the link I included in the reply above.
