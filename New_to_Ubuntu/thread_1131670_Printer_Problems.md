### Post by RandomP on 2009-04-21
Here's the deal:

Somehow my printer got set to be Ubuntu's default printer. The problem now is that It is no longer recognized as connected when I am in Windows XP. I even reinstalled the software for it in windows! How can I delete it from Ubuntu?

---

### Post by Mark Phelps on 2009-04-21
System --> Administration --> Printing.

Select the printer on the left side, click the Delete icon in the top of the panel.

---

### Post by RandomP on 2009-04-21
Thanks, my printer now works again with my other OS.
