### Post by Cadmus on 2008-06-26
Depending on your graphics hardware you may be having a problem when ubuntu tries to go to the desktop. Dont worry, this happens all the time. Just download and use the alternate CD. This uses a text-based installer which is much more likely to work.

This has happened to me a few times, and I've never had a problem with the desktop not starting once the system's installed so don't worry about that.
