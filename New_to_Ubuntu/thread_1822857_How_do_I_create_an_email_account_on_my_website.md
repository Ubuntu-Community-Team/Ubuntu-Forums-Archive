### Post by Wim Sturkenboom on 2011-08-12
> **brawnypandora0 said:**
> I plan to create a personal website. How do I create an email account using the domain name? Do I have to have a web login or can there solely be an invisible login using an email client on my computer?
If you plan to use a hosting provider, read the details on their packages. Most (if not all) will provide you with at least 1 address (that you can configure); usually it's more like 5 or 10.

Simply configure your email client to use the correct servers (e.g. pop.yourdomain.org and smtp.yourdomain.org) with some details for the login (e.g. [email]brawnypandora0@yourdomain.org[/email]) and password like you have set it up now for your current email address

Some hosting providers offer webmail functionality as well (possibly depending on the package that you choose).

If you're planning to host yourself (e.g. on a machine at home), things become more complicated as you need to setup a mail server.
