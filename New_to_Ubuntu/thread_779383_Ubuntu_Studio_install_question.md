### Post by zvacet on 2008-05-02
Download [Gparted live Cd]("http://gparted.sourceforge.net/") and with it shrink your Ubuntu ( if it is on all HDD).if you have free space with Gparted make new partition and format it as NTFS.On that partition you will install Windows.After win installation you will have to reinstal grub like [this.]("http://ubuntuforums.org/showthread.php?t=224351&highlight=reinstall+grub")
