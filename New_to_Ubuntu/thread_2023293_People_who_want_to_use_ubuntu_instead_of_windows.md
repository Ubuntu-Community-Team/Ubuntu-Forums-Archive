### Post by Paqman on 2012-07-12
> **AKSoe said:**
> I only have pdaNet Software for window version. 


PdaNet is an application you install on your phone that allows it to tether to a computer. Depending on what phone you have you can also use it to create a wifi hotspot (many phones from the last couple of years can do this without PdaNet).

Ubuntu can tether (AFAIK) to any phone that supports tethering, and can definitely connect over wifi to any phone that can create a wifi hotspot.

The best thing to do is take your phone running PdaNet and see if you can connect your Ubuntu machine to it.
