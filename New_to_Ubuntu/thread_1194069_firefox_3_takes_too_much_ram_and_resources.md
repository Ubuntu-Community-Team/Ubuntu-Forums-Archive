RAM use is relative not absolute. If you have several GB sticks then even a few hundred MB just for firefox is a pittance. If you've only 256, then you might not even want to be running GNOME but instead Xubuntu with xfce, a lighter desktop environment.

epiphany-browser is a package in the repos, it's basically a lighter browser that renders using the gecko engine. Try it if you like. Can easily be installed by synaptics or terminal. There are also numerous other browsers available for Ubuntu, such as opera, chrome/chromium, Konqueror, etc.. .
