### Post by Sorivenul on 2009-01-18

### Post by blackened on 2009-01-18
If you still want to use the icons that you downloaded, but don't like it using the regular human theme for icons that aren't included in the set, then you can always edit the index.theme file of the set and change the inherit settings to a set you prefer. Just find a more complete set, like hydroxygen, and add the inherit setting in glassdark's theme file.

---

### Post by Sorivenul on 2009-01-18
blackened is right, of course. The "hydroxygen" set is one of the most complete I have found. I also recently discovered "GNOME Colors", which is a very nice alternative to the default icon theme. The "Oxygen-Refit" series is also quite nice and comes in a variety of colors, for easy matching to your desktop. Good luck!
