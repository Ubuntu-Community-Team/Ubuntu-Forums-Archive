### Post by Tatty on 2008-05-01
Id advise you to keep windows on a partition as a backup OS - since you have already paid for it, you might as well keep it :)

There is no way to say for *certain* if hardware will work flawlessly with ubuntu, as most hardware is designed with windows in mind.  That said, most hardware does work with ubuntu these days, even if you might have to do a bit of fiddling.

I advise you to install a dual boot setup, as diving in is the only way you are going to know.

1. defrag your windows partition 3/4 times.
2. use vista to shrink your windows partition (i have heard that the ubuntu partitioner doesnt play too nice with vista partitions, but I have never had a vista machine to test)
3. [http://www.psychocats.net/ubuntu/installing](http://www.psychocats.net/ubuntu/installing)
