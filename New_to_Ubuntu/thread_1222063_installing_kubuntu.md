### Post by Bucky Ball on 2009-07-24
It's fun. I have Xfce, KDE, Openbox, ICE and a few others to choose from at login. You know what they say about variety ... 

Xfce is my favourite actually so I usually install that first (Xubuntu), then add the flavours. 

Don't get desktop environments (KDE) mixed up with Linux distributions that use the desktop environment KDE (Kubuntu). Kubuntu is Ubuntu using the KDE desktop environment instead of the Gnome one. Xubuntu uses Xfce, but they are all still Ubuntu. Apologies if you already know all this ... :)
