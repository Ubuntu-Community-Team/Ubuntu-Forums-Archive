### Post by y-lee on 2008-05-09
Try Firefox > Bookmarks > Organize and then in the bookmarks windows File and then export.

btw your bookmarks are stored in a hidden folder in your home directory. If you are using firefox and a default profile the location of this hidden folder will be :

```
/home/<username>/.mozilla/firefox/<random STuff>.default
```


The file is called Bookmarks.html
