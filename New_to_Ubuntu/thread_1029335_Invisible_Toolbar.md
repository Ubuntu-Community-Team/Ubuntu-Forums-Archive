### Post by Dedoimedo on 2009-01-03
You may have removed the top panel. You'll need to recreate it.

Right-click on an existing panel > create new panel.
Position it to top, set its size and color as you see fit.
Once it's positioned, add to it, select the common stuff from the given defaut options, like menu, clock, shutdown button etc.
