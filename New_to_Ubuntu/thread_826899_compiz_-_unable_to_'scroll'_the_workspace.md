### Post by sdennie on 2008-06-12
It sounds like you want the Desktop Wall plugin and possibly Viewport Switcher.  Try the following:

```

sudo apt-get install compizconfig-settings-manager

```

Then go to System->Preferences->Advanced Desktop Effects and make sure that Desktop Wall and Viewport Switcher are enabled.
