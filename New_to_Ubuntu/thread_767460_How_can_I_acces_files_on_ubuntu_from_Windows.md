### Post by Mazza558 on 2008-04-25
Go to places > network and Ubuntu'll pick up any other Windows PCs on the network (and yes, you can copy files to and from them).

Setting Ubuntu up to be accessible from Windows is a bit trickier though - you need to install Samba.
