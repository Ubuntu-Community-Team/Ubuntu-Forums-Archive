### Post by Kartoffelbursche on 2008-10-03
Hi Everyone,

as you can see, i am new to ubuntu, so i would like to say "hello" to the community.... hello!

So, how did i get here?

Let`s say so, i saw an really interesting offer in an electronic store (Acer Aspire One for about $200) and i bought it right away. It was one of these "my eyes go crazy, i need to buy it" thing... anyway, i thought, it could be my way into independence from windows. Dont get me wrong, windows XP is a very good system, and i can handle it really very well. But since xp is old and the support will stop in a few years, i have to look for another system... 

so i will give linux a chance, ubuntu.

So, the first thing i did, was installing ubuntu on my acer aspire one.... everything went fine, it looks good as well, i like it, really i do....and here it come, the big BUT:

my wlan is not working... no problem (i thought) just download a specific driver and that's it... here we go... but i thought wrong, i had to use the terminal and that's where my problems started.... the file system is very complicated. the new commands uff.... i looked so many instructions up in the internet to get the damn thing working, but i failed... i tried 3 days straight to get i running, success less!

but i don't give up, not yet, i will try again and again. 
hopefully i will succeed, otherwise i can always change back to xp, but this would be also a problem, not now but in a few years, when xp will be out of support.

So, hopefully i will get used to linux and that's the point when i will change all my other systems to linux as well...

What i can say now is:

Windows is a real easy to handle system, but soon "out of order"

Linux is a very nice looking and much promising product, but also very complicated to a "new to linux user", especially if the user is not the computer guy (i am thinking of my father; for him linux will never be an alternative, far too complicated) 

Cross your finger for my start with linux, hopefully i will stick to it!

### Post by brad0022 on 2008-10-05
Ok Kartoffelbursche, I have an Aspire Aspire One. The 8 gig SSD one. I found that One Linux Ubuntu works great. It was written specifically for our netbooks. Problem is that it is do longer available for download, sorry. I dont have the iso any more. It could be on a torrent but just have to search for it if its still around.

Solution for me was using Ubuntu eee. It's Ubuntu Netbook Remix for the Asus eeepc but it works perfectly for our computers. Wifi works, sound works, first SD slot (expansion slot) works. Not sure about the second slot yet. Go here: [http://www.aspireoneuser.com/forum/viewtopic.php?f=5&t=4414](http://www.aspireoneuser.com/forum/viewtopic.php?f=5&t=4414)

 After doing that, if you dont like Remix's look, you go here to disable the interface and make it normal Ubuntu.
[http://ubuntuforums.org/showthread.php?t=900084&highlight=Ubuntu+eee](http://ubuntuforums.org/showthread.php?t=900084&highlight=Ubuntu+eee)
