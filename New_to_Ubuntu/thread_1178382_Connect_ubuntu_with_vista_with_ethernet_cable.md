### Post by yaroto98 on 2009-06-04
You can either thumb drive it, or you can install a server on either laptop. For example, install a web server (XAMPP) on the vista laptop, and download them to the ubuntu laptop from that web server. Or the other way, you can install an ftp or ssh server on the ubuntu laptop and just use a windows client (filezilla or winscp) to copy them over. Whatever you want. There's more than one way to transfer a file.
