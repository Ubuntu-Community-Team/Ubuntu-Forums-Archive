### Post by grahammechanical on 2014-01-08
File Manager in Ubuntu 13.10 does not show the partition where Ubuntu is installed as a volume. It should be listing that 9GB Ubuntu partition as "Computer." Can you access the 70GB partiton? Can you load Windows? Is it still working?

If you open the Dash (click the Ubuntu icon at the top of the Launcher (side panel) and type Disks you can load the Disks utility and that will show you the partitions on your hard disk. Be careful. The Disks utility gives us a lot of power, even the power to remove what is on the hard disk including operating systems.
