I don't know [and am too lazy to look up] what PhotoRec is, but I've gleened that it's some kind of file recovery tool I can't help you with. So let's move on.

First the bad news: If you deleted the file without using a method that allows restoration, such as putting it in the trash, it's probably gone for good.

However, do you have backups? You should, you know. :p  If you have a current backup you can pull the file off of it.

Did the file come on a disc?  You can restore it from there.

Did you download it?  Go back and download it again.
