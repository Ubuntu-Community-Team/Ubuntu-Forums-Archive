### Post by grahammechanical on 2014-12-01
We cannot help you unless you give us more information.

Describe the computer hardware. CPU? Memory? Graphic adapter? Other OS installed? What version of Ubuntu?

To install we burn an ISO image to a DVD disk or a USB memory stick and run a live session and install from the live session. At the end of the installation process we are given the option to continue testing Ubuntu in the live session or to re-start.

If we choose to re-start the live session shuts down and it is usual to see some Linux messages. At the end there should be a message that says "Remove the disk from the drive and press Enter." Or something like that. On pressing enter the computer will restart.

What does that "bunch of text" actually say? Are you installing from a DVD disk or USB stick? Do you get a message saying to remove the disk/usb and press enter? If so, what  happens when you remove the installation medium and press enter? Does the computer reboot? Or do you need to power off?

Are you able to load Ubuntu or any other OS that is on the machine? If you can successfully load UBuntu and get to a working desktop, what happens when you shut down Ubuntu? Does the computer shut down? Does it restart if you choose restart?

If Ubuntu is successfully installed and you are now able to load and shut down Ubuntu. Then it does not matter if the live session did not shut down correctly that one time. Does it?
