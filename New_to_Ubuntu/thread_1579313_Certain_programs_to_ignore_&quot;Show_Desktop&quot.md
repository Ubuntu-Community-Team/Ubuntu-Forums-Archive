### Post by Dustin2128 on 2010-09-21
I'm not sure about that, but if you use the terminal often enough to always have a window open, you should install yakuake or tilda. They're multitabbed terminals that drop down from the top of the screen when you press a hotkey and continue running their process when you roll it back up (slides back up with a second button push). You could also put it behind all the other windows as the background of your desktop, which I've done on a number of occasions.
