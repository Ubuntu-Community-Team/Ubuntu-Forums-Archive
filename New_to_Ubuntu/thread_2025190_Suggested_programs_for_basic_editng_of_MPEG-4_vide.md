### Post by m3topaz on 2012-07-14
I've used PiTiVi (which is a GUI editor) - it is OK for stitching tracks and cutting bits and pieces, but a little unreliable for intensive working.

For basic cutting and editing, ffmpeg (command line) is worth reading up on and it's very good at seeking to targeted places in files.

I suggest that if you know exactly where to 'chop', use ffmpeg. If you want to see and edit based on that, use PiTiVi.
