### Post by Temposs on 2009-03-22
Probably the best is a program called conky. It shows any stats you specify on the desktop and doesn't act as a Window. It acts like it's part of your desktop background. So it's pretty cool. It's in the repositories, so just install it and look for a tutorial on setting it up. Look for an example configuration file and just paste it in.
