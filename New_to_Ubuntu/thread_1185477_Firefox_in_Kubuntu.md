### Post by GameManK on 2009-06-12
You can run Firefox in Kubuntu just fine.  The default theme in Jaunty makes it look OK.  My biggest gripe with it that's specific to Kubuntu (there are more universal things I don't like about it...) is that there is no integration for opening files other than web pages.  That is, it won't use any of the file associations that come by default or you have set up and you have to browse for the executable yourself.

Reasons Firefox can't be in Kubuntu by default:
[LIST]
[*]No space on the CD
[*]Doesn't fit in well
[*]Has dependencies that we don't want to install by default, including GTK
[*]Uses a crapload of memory, which is only worse when it's not using any KDE libraries
[*]Fixing the integration problems takes work nobody has stepped up to do so far
[*]Konqueror is there
[/LIST]

If you're looking for webkit, give Arora a try.  It's fairly barebones and still under development, but you can get version 0.7 from Jaunty backports.
