### Post by kevinatkins on 2009-12-29
Yes, the Intel Core 2's are 64 bit - I've got Ubuntu 9.10 64-bit running on this IBM T60 Core 2 Duo right now....

But, having tried 64-bit on this and on my AMD Athlon X2-equipped desktop machine, in reality I haven't found the jump in performance over 32-bit to be great - certainly not enough to warrant buying a new computer, all other things being equal.. In fact, AFAIK, the main benefit to 64 bit is the ability to address more than 3.5G of memory - so yes, if you're using huge apps that chomp memory, 64-bit is a definite advantage... 

I'd see if you can get a refund on your Windows 7 or exchange it for the 32-bit version
