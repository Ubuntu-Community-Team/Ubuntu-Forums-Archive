### Post by Eberhard on 2009-05-15
Go to System -> Preferences -> Keyboard Shortcuts

There should be an action called "Run terminal" - insert whatever shortcut you prefer.

### Post by sgosnell on 2009-05-15
I use tilda.  It's in the repositories, and you can configure it to use whatever hotkey you like.  I use F10, and F10 toggles it on and off.  The first one brings up the terminal, and pressing it again removes the terminal screen.  I run it in startup programs, so it's always available.

I also have a shortcut to terminator in my panel, so I can get to that easily, but that does require one mouse click.
