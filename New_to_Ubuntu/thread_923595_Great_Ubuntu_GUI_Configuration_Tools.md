### Post by drs305 on 2008-09-18
Two that I have found of use are StartUp-Manager and Storage Device Manager (pysdm). 

StartUp-Manager (startuopmanager - System > Admin > StartUp-Manager) is used to tweak the settings of grub's menu.lst You can change how many kernels to display, how long the menu displays before action is taken, change the bootloader colors/image, etc. I would rate StartUp-Manager at least 'very good', especially for beginners. 

Storage Device Manager (pysdm - System > Admin > Storage Device Manager) is a gui-based fstab editor. It does the job, although not elegantly and not without a least a bit amount of knowlege to make sense of it (in which case you could just manually edit fstab). With instructions it's marginally useful.

I've written tutorials on both these apps, found in the tutorials forum. You can visit them via the links in my signature line.

PS: I've also had good success with EnvyNG (envyng-gtk - System, System Tools, EnvyNG) for installing ATI and Nvidia drivers. Others have had mixed results.

### Post by Paqman on 2008-09-18
Two of my favourites:

[NTFS Config](apt:ntfs-config) - no more editting /etc/fstab manually just to mount an NTFS partition.
[Startup Manager](apt:startupmanager) - change Grub settings without editting /boot/grub/menu.lst, and set defaults to keep it organised (no more endless lists of obsolete kernels)
