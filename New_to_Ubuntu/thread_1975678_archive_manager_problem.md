### Post by jerome1232 on 2012-05-07
iTunes is not available for linux, Rhythmbox is the default media player for Ubuntu and has a very Itunes like look, I suggest you try Rhythmbox.

Also Linux is not Windows, exe files are executables meant to run on Windows, not Linux. You generally install applications on Ubuntu from the UBC [Ubuntu Software Center], or download a .deb installer from the software developer's website.

Sometimes it is possible to run applications designed for Windows through a compatibility layer called Wine (installable via UBC), judging by WineHQ's [ENTRY]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=1347") for Itunes, it doesn't seem to run very well so YMMV.
