### Post by newb85 on 2012-09-21
Here's how it works on my machine.  Ctrl+Tab will switch between lenses.  However, the search bar entry isn't common for all lenses, so I have to re-type my search if I switch lenses.  

If I search for something in the home lens, the results are divided into three sections: Recent Apps, Recent Files, and Downloads.  Each section only shows a few entries.  I can switch between sections with Tab.  I can expand a section to show all search results in that section by selecting the section header with Tab and then hitting Enter.  Then I can use the arrow keys to select the specific app/file/download I want and hit Enter again.
