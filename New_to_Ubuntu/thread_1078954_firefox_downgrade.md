### Post by taurus on 2009-02-23
There are many options to do that.  If you are the only user on the system, you can create a ~/bin directory and unpack it in there.  Than, add that path to your PATH in ~/.bashrc so you can run firefox from anywhere by just typing in the name.  Or you can also create a launcher on the top panel for it.  

Or you could use sudo and unpack it in /opt if you wish.
