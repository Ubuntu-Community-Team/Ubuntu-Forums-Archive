### Post by Remten on 2013-10-23
Is there a guide anywhere that explains step-by-step how to install Windows 7 with Truecrypt and Ubuntu 13.10 with dm-crypt-LUKS-LVM for dual-booting?

I was previously able to set up a dual-boot installation of Truecrypted Win 7 and dm-crypt-LUKS-LVM Xubuntu 12.04 LTS after reading quite a number of forums and spending days on trial and error. But the Xubuntu/Ubuntu 12.04 LTS had an alternative installer, which made things easier. As far as I can tell, there is no alternative installer for Ubuntu 13.10, and the desktop installer that is available does not seem to be as user-friendly for setting up LUKS-LVM when a Truecrypted Windows partition is already on the drive.

I have found articles describing how to set up LUKS-LVM for Ubuntu in general, but I haven't seen anything that explains or shows all the steps for doing it when there is a Truecrypt-encrypted Windows installation on the drive already.
