### Post by Vladlenin5000 on 2014-05-11
Hi, welcome.

Regarding your question let me first comment that no, most systems don't allow juwt drag&drop, most system require a CD/DVD/BD burning software. Windows (latest versions) has a limited support natively but yes, it can do that but even in Windows that's not the way to burn optical media, any normal user will tell you that followed by the recommendation to use Nero...

Unlike Windows, Ubuntu comes with a few tools already. In standard Ubuntu you'll find Brasero (other flavors might come with different tools  - Kubuntu comes with K3B by default. Regardless of you choice of software, as soon as you select a data disk project then you'll be able to drag&drop from the file manager to the burning software window, just not inside the file manager as you seem to be used to.

### Post by Vladlenin5000 on 2014-05-11
You're welcome and I'm glad my post was useful is spite of all the typos (it could have been worse... Before I was in a tablet without physical keyboard, in a pub, answering you between two glasses of a 2010 Mencía red wine so yeah...)

Back to the point, what you were used to do in Windows works for plain CD-ROMs but nothing else. Xfburn, albeit a very basic software does the same job much better plus it can produce audio CDs from your audio files nice and easy. The interface isn't bad either and it has many good reviews: [https://apps.ubuntu.com/cat/applications/precise/xfburn/](https://apps.ubuntu.com/cat/applications/precise/xfburn/) . However, my personal recommendation goes to K3b (with all the extras). You can install it like any other software just by searching in Ubuntu Software Center (click more info to get to the page where you can select the extras as well or you can do it later, it's up to you). This program does everything Nero for Windows does and more but its UI and workflow is actually much more similar to an old similar software for Windows (I can't remember its name though... Perhaps I should blame it on the wine) which was very popular in late-90s early 2000s before Nero became the "must have" it is now...
