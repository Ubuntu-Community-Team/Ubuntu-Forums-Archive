### Post by antsu on 2010-07-01
Hi,
I know that this thread is already a month old, and probably was abandoned, but I'm replying anyway, cause it may help someone who search for this in the future.

Well, I have a Dell Latitude D810 with ATI Mobility Radeon x600 (128MB DDR) graphics card. I'm running Ubuntu 10.04 with open source "radeon" driver, under a resolution of 1680x1050@60Hz.
XBMC runs well if Compiz is disabled. Try disabling compiz (if it's enabled) and see if it works (for me it makes a huge difference!). Also, if you have installed XBMC from their PPA, try installing the package "xbmc-standalone" and run it as a session, it will make XBMC run a lot faster. This way I can even watch 1080p videos on this machine.

Sorry for any possible English errors,
