### Post by Perfect Storm on 2010-12-11
Install:

Advanced Desktop Effect Settings

Eventually install Fusion Icon (and set it to startup on every boot) to get easy access to the effects and you can also disable/enable compiz through it - if needed. This can be done through Ubuntu Software Center.


Or it can be done via the terminal:
```
sudo apt-get install fusion-icon compizconfig-settings-manager compiz-fusion-plugins-extra
```

### Post by owise1 on 2010-12-11
Mate - you just need to enable desktop effects - >system>preferences>appearance and click the visual effects tab and select extra.  Then install "advanced desktop efefcts setting" via the software centre.  Once installed >system>preferences>compizconfig settings manager.
