### Post by perspectoff on 2012-04-16
Adding print margins is a CUPS function, not a PDF function. 

The default print margins are, IMO, too small for CUPS.

In the CUPS printer options for your printer there is a "Scale to fit" option that is unticked by default. Try setting that option to see if it works for you.

I often must set the margins on a per-job basis. 

This is one of my biggest gripes about Ubuntu -- the default printing otions and the printing speed is not adequate. CUPS still needs work, IMO.
