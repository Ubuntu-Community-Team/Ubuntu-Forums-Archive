### Post by Mark Phelps on 2011-10-29
> **sunnybrown said:**
> ati hd5770 when run video driver cant get to boot afterwards i am new to linux and i thought driver issues was a thing of the past running 11.10 please help thanks

Not so with Ubuntu. Each new version brings a new Linux kernel, and since proprietary drivers are kernel-specific, new problems as well.

From what I read, 11.10 tries to install Catalyst v11.7 -- which does not work with the new Gnome.  You have to install Catalyst 11.9 by yourself.

Unless you really NEED the ATI drivers for 3D Graphics using hardware acceleration, you would be better off with the default open-source drivers.  I'm using them, with dual monitors, in 11.10 and they work fine.
