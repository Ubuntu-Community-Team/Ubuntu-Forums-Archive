### Post by howefield on 2010-08-30
This appears to be a USB printer so unless your router has a print server, you'll attach it it to your Ubuntu computer.

System > Administration > Printing, click on the printer and then Server > Settings menu.  Place a check beside Publish shared printers connected to this system and set other options as required.

Windows 7 should be able to pick up the printer.

Your Ubuntu computer will need to be switched on for the other to see and use the printer.
