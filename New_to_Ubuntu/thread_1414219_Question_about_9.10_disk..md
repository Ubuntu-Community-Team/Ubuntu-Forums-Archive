### Post by ajgreeny on 2010-02-23
Did you burn it as an image or just as a data file to the CD?  Have a look in your file manager (Windows Explorer?) and if there is just a single ubuntu.iso file showing you will need to burn again, but this time choose "Burn as image" in your CD Burner software.  I no longer have windows, but I used to use a free burner program for all my CD burning needs called CDBurnerXP, which was excellent, but there are many other free ones, I think, and all the commercial programs can do it as well.

### Post by undecim on 2010-02-23
Check the md5sum of the .iso 

From Linux: in a terminal, type "md5sum filename.iso" where filename.iso is the filename of the iso

Or, from windows: [http://www.openoffice.org/dev_docs/using_md5sums.html#win](http://www.openoffice.org/dev_docs/using_md5sums.html#win)
