
You will see Envy appear in your System Administration menu and you can install from there.

### Post by natravis on 2009-09-16
The easiest way to get drivers in Ubuntu is to go to System->Administration-> Hardware Drivers and enable the latest available for your computer from the repository. If you need a newer version of the graphics driver, it is a much more involved process.
