### Post by Bionic Apple on 2008-07-25
-Install Wine (if you don't know how to do this, I will explain)

-Download Steam and run the Msi

-Sign in

-Download one source game and Gmod

-Install Gmod Server -- [http://wiki.garrysmod.com/wiki/?title=Dedicated_Server_Setup]("http://wiki.garrysmod.com/wiki/?title=Dedicated_Server_Setup")

-Forward Port 27015 on your Router



If those basic instructions don't help, tell me and I will help you again.  Oh and Gmod is not going to run well because you require X to run with Wine.  I'm no expert, but I am guessing you will be at 80% performance compared to Windows.
