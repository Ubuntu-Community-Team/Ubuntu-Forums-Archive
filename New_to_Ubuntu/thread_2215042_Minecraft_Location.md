### Post by Double.J on 2014-04-04
Hi there, not familiar with hacking minecraft, but I can right click on the icon to edit texture packs. As for the minecraft folder, it's a hidden folder in your home directory. Click on view in the file browser and 'show hidden folders'

hope it's what you're looking for!

oh and if you are using the terminal, a folder is made hidden by putting a period in front, so it's
/home/username/.minecraft
