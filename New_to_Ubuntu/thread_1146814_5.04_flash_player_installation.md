### Post by Temposs on 2009-05-03
download unetbootin to somewhere you can find on your computer's main HDD. Change the permissions so it's executable like this in terminal:

```
chmod +x ./unetbootin-linux
```

Then run it. Choose Ubuntu 9.04 and choose your USB HDD to make into a bootable USB drive. This will get a LiveCD version of Ubuntu onto your USB HDD(you can remove it later). This will let you boot from your USB HDD so that you can install Ubuntu 9.04 on your main HDD.

If you want to do it without terminal, then just open the file browser to where you downloaded the file to. You can use the Places menu to open the file browser.
