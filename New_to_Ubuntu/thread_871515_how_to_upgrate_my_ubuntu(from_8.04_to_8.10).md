### Post by tamoneya on 2008-07-27
if you dont know how to do this I wouldnt really recommend it.  8.10 is still alpha 3 meaning that it is really buggy.  I recommend that you wait until it is release in october.

Anyways if you still do want to give it a try you can run the following code in terminal (applications -> accessories -> terminal)
```
update-manager -d
```
This can be run at any time between now and when it is released.

> **Lav. said:**
> by the way,is upgrading useful?:confused:

Upgrading right now probably won't be very useful.  Developers are still working on 8.10 and the final release is in October.

If you do choose to upgrade, I suggest having one partition for 8.04 and another for 8.10.  This way if something breaks in 8.10, you have something to go back to.
