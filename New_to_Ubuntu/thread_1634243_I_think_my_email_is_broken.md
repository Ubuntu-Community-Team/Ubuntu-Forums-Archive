### Post by Liverbones on 2010-11-30
Evolution should automatically be set to play a sound whenever mail is delivered to your inbox. It may not literally say "You've got mail," but it should be notifying you via sound when you receive a message. If you want to *change* what sound is played, you should be able to simply go to Edit > Plugins > Mail Notification and select the Configuration tab. From there, you can select a custom sound file to be played.

Let me know if that works for you.
