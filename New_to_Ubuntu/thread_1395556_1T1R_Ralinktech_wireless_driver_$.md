### Post by danicrem on 2010-02-02
Solved.
I have a dual boot with Windows 7. What I did was download the Windiws XP driver form MSI.com from my Windows 7, extract it to a folder in mij root c:. Next start Ubuntu. Copy the folder with the driver over to my Ubuntu desktop. Next go to Administration-> user management and change the permissions of my ow user (make myself connect to wireless internet). Next I started the Ubuntu install CD and installed ndiswrapper ron the cd (no internet needed then). Next used the ndiswrapper (Windows wireless drivers) to install the driver I copied to my dekstop.
