### Post by robalcorn on 2008-05-17
What is the file extension? if it is example "et-linux-2.55.x86.run" you would in a terminal type :

sh et-linux-2.55.x86.run

as far as running it you should find in Applications/Other/EnemyTerritory

### Post by quelx on 2008-05-18
Download the latest from here (right click and **Save Link As**)

[http://ftp.games.skynet.be/pub/wolfenstein/et-linux-2.60.x86.run](http://ftp.games.skynet.be/pub/wolfenstein/et-linux-2.60.x86.run)

in a terminal (**ALT-F2** > **gnome-terminal**)

```
sudo sh Desktop/et-linux-2.60.x86.run
```
