### Post by cdtech on 2008-07-18
If you have two then install two. A lot of users install the min as well as the min for swap space only later wishing they had set it higher. After which you'll have to repartition your drives. If you set it up right the first time you wont have the frustration of going back.

I'm not a real big gamer on my system, although I do use a lot of proc power and mem (running virtual machines and compiz 3D graphics). I'm actually happy I set mine up correctly the first time.

I feel its better to have the overhead even if you don't use it now, in the long run it will save you the headaches later.

### Post by kevmitch on 2008-07-18
Something you may want to consider if you want to reserve the possibility of a future upgrade is the DIMM configuration. If 1GB is really 2x512MB dimms and you laptop only has 2 dimm slots (probably the case), then you wont be able to upgrade to 2GB without throwing away the 1GB you previously purchased. The big price difference may be due the fact that the 2GB pack uses larger 2x1024MB dimm modules.
