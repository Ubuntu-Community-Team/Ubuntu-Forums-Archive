### Post by Kellemora on 2010-08-13
Hi Expat

Not exactly sure what you mean, but I have to use a different Frequency Setting on different monitors so I don't have to adjust the monitor itself, because then that would mess up all the other items using it.

Changing the frequency will shift the video around on the monitor screen, if that is what your problem is.  Nvidia controls also lets you move the image within the screen frame itself.  So if some of the text is cut off do to position within the window, you can shift that part independent of the window on the monitor which is adjusted using the frequency settings.

Probably not the kosher way of doing it, but when you have one monitor and 4 to 8 different machines using it, you have to strike a happy balance somehow.
