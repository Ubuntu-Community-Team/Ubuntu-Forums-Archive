### Post by Vaphell on 2011-01-10
if you are happy with the repo version simply go to software centre/synaptic package manager in system->administration and select thunderbird
alternatively **sudo apt-get install thunderbird** in command line and that's it.

The correct way is to always use synaptic/software center. Other installation methods are useful when you absolutely need bleeding edge stuff.
