### Post by gdonwallace on 2010-01-08
Honestly, if you can do it, I would go with DHCP.  

Let proxy server assign all that information, then you don't have to worry about it.

Otherwise; right click on the connection icon on your panel, choose edit connections.  Find your connections (wired, wireless, etc) and change the server settings.  (Of course this assumes that you have the new proxy settings to enter into Ubuntu)

Good luck.
