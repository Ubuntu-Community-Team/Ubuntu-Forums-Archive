### Post by Dangger on 2009-10-30
System>Preferences>Startup Applications

If anything there is related to pidgin just uncheck it. 

**Alternatively:**

get Ubuntu Tweak and manage startup applications directly through it.

If it's not there perhaps you are saving the session and starting up with a previous state (do you have xubuntu?)
