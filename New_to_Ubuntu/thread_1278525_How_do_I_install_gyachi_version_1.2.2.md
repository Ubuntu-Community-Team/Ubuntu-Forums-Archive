### Post by yeats on 2009-09-29
If you're new to installing software on Linux, you might do better to add the custom repository:

[https://launchpad.net/~loell/+archive/ppa](https://launchpad.net/~loell/+archive/ppa)

Look at the "Adding this PPA to your system" section.  After adding the PPA repo you will be able to download it via the Synaptic Package Manager (System -> Administration -> Synaptic Package Manager).
