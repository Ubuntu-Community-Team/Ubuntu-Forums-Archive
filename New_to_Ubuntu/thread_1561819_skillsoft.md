### Post by Zzl1xndd on 2010-09-04
To install the recommended Java for SkillPort, First ensure the Partner repo's are enabled. Go to System > Administration > Synaptic Package manager. Once the Package manager is open go to settings repositories. Click the Other software tab and ensure that Ubuntu Lucid Partner is checked. Then click close > Then reload. 


Then Close Synaptic and go to Applications > Ubuntu software Centre and search for Sun Java. Install the Java Plugin.
