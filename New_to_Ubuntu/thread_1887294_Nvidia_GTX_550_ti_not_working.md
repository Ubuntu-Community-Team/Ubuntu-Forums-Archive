No, install them from the terminal. I think you can do this using the command:```
sudo apt-get install nvidia-common
```Otherwise, I would say that since it's a newer graphics card you would download them from the nVidia website here: [http://www.nvidia.com/object/linux-display-ia32-270.41.06-driver.html](http://www.nvidia.com/object/linux-display-ia32-270.41.06-driver.html)
If the terminal command doesn't work let us know if you need assistance installing the drivers on the website.
