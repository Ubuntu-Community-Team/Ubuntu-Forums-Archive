### Post by Cypher on 2008-07-08
You would use Wine to install/play the game. So put the CD into your drive and from the Terminal type "wine /media/cdrom/install.exe" or whatever the installation program happens to be. From that point onward, the rest of the installation should look familiar.

Once installed, in the Applications->Wine menu, you should see the option to start Wolfenstein..
