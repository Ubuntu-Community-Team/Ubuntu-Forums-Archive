### Post by Impavidus on 2013-11-09
Gimp can handle pdf, but only as a raster format. Any vector pdf will be converted into raster format. For filling in forms this is quite useless. You can better use a good pdf reader.

### Post by SeijiSensei on 2013-11-09
I've found GIMP's ability to import PDFs quite valuable when I need to [display a data table]("http://www.politicsbythenumbers.org/wp-content/uploads/2012/11/gerrymandering-prospects2.png") in my blog. I create the table in LibreOffice Calc, save it as a PDF, import that into GIMP, then crop the image and save it as a PNG. Works like a charm.
