Hi Old-un,

I gedit is good but for minor tasks but for my web development and for PHP programing I use two of the best I found after giving up Dreamweaver form adobe. The two are listed below and how to install them.

Bluefish Editor very nice for HTML and Web Base Programing.

```
sudo apt-get install bluefish
```

Home page: [bluefish.openoffice.nl]("bluefish.openoffice.nl")

Scribes Editor a breath of fresh air for a programmer who wants to see all the code he writes with out skipping between tabs.

```
sudo add-apt-repository ppa:mystilleef/scribes-daily
sudo apt-get update && sudo apt-get install scribes -y
```

Home page: [http://scribes.sourceforge.net/]("http://scribes.sourceforge.net/")

Hope this helps
