### Post by HermanAB on 2010-02-19
Hmm, it is hard to give details when the machine isn't working, so you should try to boot into single user mode and try to fix it from there.  If all else fails, re-install - if you created a separate /home partition, then re-installing takes only 30 minutes or so and your data will be intact if you *don't* format /home.

The obvious lesson is one I learned many years ago: Once you installed and got the machine to work properly, turn off the automatic screw-ups^Wupdates.  I almost never do updates - I rather re-install once a year or three from scratch.
