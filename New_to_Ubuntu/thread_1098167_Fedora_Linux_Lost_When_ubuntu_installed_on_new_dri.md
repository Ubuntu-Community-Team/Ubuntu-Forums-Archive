### Post by taurus on 2009-03-17
Edit /boot/grub/menu.lst

```
gksudo gedit /boot/grub/menu.lst
```
and near the top, change **timeout** from 3 to 30.  Then, add these lines to the end of the file.

```

title Fedora Core (2.6.22.14-72.fc6)
root (hd0,0)
kernel /vmlinuz-2.6.22.14-72.fc6 ro root=/dev/VolGroup00/LogVol00 rhgb quiet
initrd /initrd-2.6.22.14-72.fc6.img

```
Save it and reboot.  Now, is there an entry for Fedora (last one)?
