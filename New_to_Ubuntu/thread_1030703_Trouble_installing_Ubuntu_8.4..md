### Post by jimmy the saint on 2009-01-04
try to run it in safe graphics mode.  It should be selectable from the f4 options at the initial screen.

If that still fails, download the alternate install cd and try that.  

The issue is that you likely need a restricted driver for your video card so you need to install the OS first, then install that driver.
