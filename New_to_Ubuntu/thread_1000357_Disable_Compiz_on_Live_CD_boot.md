### Post by jeromey on 2008-12-02
i have had similar issues recently

im not sure about any command line options or whatever but i did find a way to disable compiz

after all of the initial boot, when it starts to load up the desktop itself, just hit ctrl-alt-backspace (this will send you back to the login screen)

at the login screen you can change the session to "GNOME failsafe" then just let it auto login to the livecd user

compiz is disabled with "GNOME failsafe" ,although you might run into some other issues im not sure
