### Post by coffeecat on 2013-03-25
Two points:

> **LordChirag said:**
> ubuntu 10.10

Ubuntu 10.10 is end-of-life and unsupported. No security updates. You would be advised to install a supported version of Ubuntu such as 12.04.

> **LordChirag said:**
> 
And also i want to know how to install programs that i just downloaded.

This is a common error made by newcomers to Ubuntu who are used to the Windows way of doing things. Most of the applications you will need you will find in the Software Centre and are automatically downloaded and installed for you, so there is not a need to search for stuff on the internet for most applications. But because Ubuntu 10.10 is obsolete, the software repositories are closed (moved, in fact) and you will not be able to use the Software Centre. Which is another reason for installing a supported version of Ubuntu where you will be able to use the Software Centre.
