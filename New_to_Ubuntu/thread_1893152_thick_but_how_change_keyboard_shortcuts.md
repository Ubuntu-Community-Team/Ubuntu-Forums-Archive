### Post by jadaka on 2011-12-09
To clarify, are you still trying to add ctrl-alt-t as terminal? As mike555 said, it's in settings manager -> keyboard. Then go to the second tab, "application shortcuts," and the "add" button is at the bottom of the window. The button gets cut off if the window is too small.

The " -> @ not being read right sounds like a keyboard model selection issue - you can change that on the "layout" tab in the same place.

Sorry if you've already got it sorted out!
