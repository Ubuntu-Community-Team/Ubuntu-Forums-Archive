### Post by zeroseven0183 on 2009-02-27
Is there any error message on the screen? How about other websites with Flash Player requirement?

I think you need to install Adobe Flash Player.
Go to [Adobe Flash Player download page]("http://get.adobe.com/flashplayer/"), select **.deb for Ubuntu 8.04+** from the dropdown list and click **Agree and Install Now**.

Once the download box appears, select Open With (**GDebi Package Installer (default)**) and click OK.

Open the file and install it. Restart Firefox and you're ready.

I hope that helps.
