### Post by lovinglinux on 2008-09-04
As explained by tuxxy, the Add/Remove manager is the best for searching and installing software. It include all applications from official repositories, but you can enable or add other repositories using System > Administration > Software Sources.

More info about repositories and other ways of installing software [HERE]("https://help.ubuntu.com/8.04/add-applications/C/index.html")
