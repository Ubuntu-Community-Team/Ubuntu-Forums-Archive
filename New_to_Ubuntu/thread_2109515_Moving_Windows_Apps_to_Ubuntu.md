### Post by Bucky Ball on 2013-01-27
You have a Wubi install? Ubuntu inside Windows and not the two on separate partitions?

If you are talking about Windows specific apps, no. They will not run in Linux (unless you use Wine and success varies depending on the app). You need Linux specific apps for Linux. You may find some of your apps have Linux versions and some may be available already in Software Centre. Many torrent clients are ported to Linux as well as Win (and some Mac) and chrome works no problem, but you need to install it in Ubuntu. You can't just dump them in there from the Win install (.exe files won't run, for example).

### Post by DuckHook on 2013-01-28
You have chosen to install a completely different operating system than Windows. Therefore, you cannot expect any of your old Windows apps to run. This is like expecting your fridge parts to run on your microwave oven. Actually, there are ways to run Windows programs on Linux using very complex tricks like WINE and virtual machines, but these are so far beyond the ken of the new Linux user that I feel guilty even mentioning them. If you are unaware that programs are not portable between different operating systems, then you should assume that WINE and virtual machines are far beyond your grasp for now and avoid them until you have learned much more about Linux.

If you seriously explore Linux, then you will be rewarded with some incredible discoveries. You will find a vast array of programs that do most of the things your old Windows programs do, and some that do the same job much better. But they are not Windows programs and cannot run on Windows any more than your Windows programs can run on Linux. Its' not even apples and oranges--more like beefsteak and cupcakes.

BTW, welcome to Linux and to Ubuntu. It's always nice to see new people discovering Linux for the first time.
