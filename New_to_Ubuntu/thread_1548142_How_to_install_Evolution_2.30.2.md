### Post by PhilGil on 2010-08-07
You should be fine installing it from a 3rd party repository as described here: [http://www.webupd8.org/2010/06/install-evolution-230-in-ubuntu-1004.html](http://www.webupd8.org/2010/06/install-evolution-230-in-ubuntu-1004.html)

Using a repository will bring in the correct dependencies and insure that you receive bug fixes and security updates (as long as the package maintainer keeps the repository up to date).  I'm using Evolution 2.30 in Debian testing and there are some significant improvements over 2.28.
