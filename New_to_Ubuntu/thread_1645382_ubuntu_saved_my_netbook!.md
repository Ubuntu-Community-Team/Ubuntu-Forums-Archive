### Post by Daveasceraspireone on 2010-12-14
I just wanted to post a newbie comment. I have been using an acer aspire one for two years wired directly to my router. Why? because the wi-fi capability of this otherwise good little netbook has been seriously suspect. Not only is getting on the wi-fi chancy at best, but after using the acer, I can't get any other computer to get on the wi-fi without rebooting the router.

Yesterday, I created a usb flash drive for ubuntu netbook remix and ran it on trial. Today, I installed the thing and kept the linpus lite as well (what a marvel the install is!, all done for me, flawlessly).
I started connecting to the wi-fi using netgear wg111v3 and then realised that not only does this adapter work "out the box" but the onboard Atheros wi-fi also works.

So, clearly ubuntu has fixed both wireless devices and I'm now free to enjoy the netbook properly for the first time ever. Thank you SO much. The only decision left is whether to dump the linpus completely?

ubuntu has saved my netbook and I'm so happy!

Dave

(looking forward to getting into something new!):P

### Post by kaldor on 2011-01-08
Netbook edition for 10.10 was a horrible mess thanks to the experimental desktop (Unity powered by Mutter).

The netbook edition has the regular (GNOME) desktop available as well. You can use it by logging out, clicking your name, and then at the bottom bar change Desktop session to Desktop instead of Netbook.

In Ubuntu 11.04, the netbook "Unity" desktop is going to be the Default for both netbook and desktop editions. This time, however, it's reworked on different technology and performs a load better. I'm running the development version right now on one PC. Previously, the PC couldn't handle the netbook interface... now, it runs smooth.

Good luck! :)
