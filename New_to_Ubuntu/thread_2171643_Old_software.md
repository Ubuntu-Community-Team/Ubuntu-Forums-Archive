### Post by coldraven on 2013-09-01
There is also VirtualBox which will run various Windows versions in a virtual machine.
You would need to have a reasonably modern computer with plenty of memory and a processor that is capable of virtualisation.
[https://help.ubuntu.com/community/Virtualisation](https://help.ubuntu.com/community/Virtualisation)

I can run easily XP  fullscreen or in seamless mode in a virtual machine.
I once installed 98 but it had problems booting up and I deleted it.

Maybe best to try with Wine and see how you get on as Virtual Machines are a bit more technical for beginners.
