### Post by gn2 on 2008-11-06
Two methods, in a terminal type: ```
sudo apt-get install k3b
```

Or System > Administration > Synaptic Package Manager > Search, and type K3b in the search box and click Search

Then right-click on the K3b entry and select "Mark for installation" then click the Apply button.

Both of these methods will download the application and all the other packages required to make it work and install them for you automatically.
