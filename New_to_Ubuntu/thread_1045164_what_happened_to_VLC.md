### Post by MikeSz on 2009-01-20
I have always found VLC in the package manager though you may also need to ensure all your sources are activated and that you make sure you are searching from all available applications (think the drop down box is in the top right) rather than just the recommended ones.  That way it should show up.  You can always install it from Synaptic as well - that's fairly straight forward and ive never had a problem with any of the programs in there from the default trusted sources.  

Hope that helps.

p.s. you may also need to install DVD codecs though there are plenty of walk-throughs on how to do that.
