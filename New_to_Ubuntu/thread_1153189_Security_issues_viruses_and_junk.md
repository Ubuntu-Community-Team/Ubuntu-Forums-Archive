### Post by kidux on 2009-05-08
Windows virii have no effect on Linux, though they can be transmitted through shared files. No, you don't need to do anything like defragging because the way Linux uses it's file systems takes into account the fragmentation that would occur and keeps it clean as you go, aka journaling.

Temp files are stored in /tmp and are cleaned regularly, IIRC.

### Post by Niniel on 2009-05-08
Nothing is "completely safe".
And you can still get problems when people exploit holes in application software (e.g. phishing is possible) or when you install software from questionable sources (see the news about the MacOS Trojan that's distributed through a cracked copy of iLife).
But it's a lot safer right now. Just maintain some basic vigilance.
