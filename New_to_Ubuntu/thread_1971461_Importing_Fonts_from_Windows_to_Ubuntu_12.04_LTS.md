### Post by tazz4vr on 2012-05-02
I was able to figure it out for myself....

Just in case there is anyone out there scratching their heads in confusion as well, what I did was opened the folder that contained the fonts that I needed.  By right clicking on the icon, I opened the font in the 'font viewer' where it had an option to install.  Ta-da, font installed, verified using LibreOffice Writer, problem solved!
