### Post by lovinglinux on 2010-11-19
Don't need to download it from the web. Vuze and most of the applications you need are available from Ubuntu Software Center (repositories). Just open "Applications >> Software Center", then search for Vuze, select it and click to install.

I recommend reading [https://help.ubuntu.com/10.10/add-applications/C/index.html](https://help.ubuntu.com/10.10/add-applications/C/index.html)

You can also do it via terminal with:

```
sudo apt-get install vuze
```

I would recommend taking a look at qBittorent. Is a much lighter BitTorrent client, with all the features you need to download. Is the fastest client I ever used.

```
sudo apt-get install qbittorrent
```
