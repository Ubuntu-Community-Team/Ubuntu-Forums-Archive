### Post by 3rdalbum on 2009-02-15
No, Control-Escape does nothing. Use the Gnome System Monitor from System > Administration > System Monitor.  There is a HOWTO out there somewhere about how to get the program to launch if you press Control-Alt-Backspace, but most people don't need the system monitor on-call all the time.

I have the "System Monitor" applet on my top panel that shows me CPU and network use. If I want more details, I right-click on it and select "Open System Monitor".
