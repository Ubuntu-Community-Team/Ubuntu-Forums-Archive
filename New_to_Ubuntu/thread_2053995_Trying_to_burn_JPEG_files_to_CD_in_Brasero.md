### Post by Jerry N on 2012-09-06
I don't normally use Brasero but it looks like you need to (1) Create a Data Project (2) add the picture files to the project (3) go to the project tab and select Burn.  This of course assumes that you have a writable CD/DVD in the burner.  There should be no file conversion required to burn the files to disk although if they are in some strange or proprietary format you may need to do something to them before you can use them on another computer.  

I prefer Nero ($$) or K3b for CD/DVD burning in Linux.
