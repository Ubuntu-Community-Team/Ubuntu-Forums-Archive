### Post by Paqman on 2011-06-09
> **brawnypandora0 said:**
> 
1) How come Windows only has one shell interface yet there are so many in Ubuntu? Why would someone choose to use one over another?


Open source tends to generate lots of alternatives for each job, and people can choose which one they like. Because people *can* make different ones, they do.

> 
2) And also how is Unity more user friendly as everyone says?

I think it's pretty user friendly, although it needs a bit of polish. It was originally designed for small, wide screens, which is why the launcher uses horizontal space, why we have the global menu, and why apps run fullscreen. It's designed to maximise the use of vertical space by apps.

Ubuntu had no choice about dropping the old Gnome 2 interface, as Gnome was abandoning it. Ubuntu weren't happy with a lot of what was going on in Gnome 3, so they decided to take Unity (which was a netbook interface) and apply it to other machines too. So it wasn't necessarily designed to be more user friendly than Gnome 2, it's just that they had to use something and didn't like Gnome 3. User-friendliness was an important goal, but not the main driver for change.

### Post by Paqman on 2011-06-09
> **brawnypandora0 said:**
> Does Unity take up less RAM than GNOME?

Well Unity is built on top of Gnome, it just doesn't use Gnome Shell. Compared to Gnome Shell, it's about the same. In terms of actual graphics performance Unity is significantly better than Gnome Shell, because it's built on Compiz, which is relatively mature and plays nicely with graphics drivers. One of the reasons Ubuntu didn't want to go with Gnome Shell is that they didn't like the Mutter technology it uses for this stuff.

### Post by Mark Phelps on 2011-06-09
There actually are different choices in desktop environments in Windows than using the MS default. Stardock has been selling Windowblinds and other such stuff for YEARS.  There are other desktop alternatives that are available as well -- they're just not so widely advertised.

But as to why there are so many in the Linux world, among the reasons are:
1) The different environments are FREE.  You don't have to spend money to buy one -- only to find out afterward that you don't really like it.
2) Linux is, by definition, an "open" environment, meaning, you don't have to deal with Activation and other such irritations to install something NOT supplied by Canonical.  Gives you a LOT more freedom to experiment.
3) Some folks just like changing their desktop from time to time.
