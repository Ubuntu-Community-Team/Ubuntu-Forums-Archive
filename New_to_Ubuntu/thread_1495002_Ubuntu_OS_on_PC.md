### Post by Paddy Landau on 2010-05-27
Ubuntu is not "superior" except in anyone's opinion.

If you're an avid gamer, you may find Windows is best, simply because its popularity leads to most games being written for Windows.

My personal experience (I'm not a gamer) is that Ubuntu has proven much faster and more reliable, not to mention hugely cheaper. Some things are easier; you don't have to do all that housekeeping and antivirus and firewall stuff that Windows needs.

Expect a steep learning curve; as you can expect, Linux is not Windows, so many things are different.

How to get started?

[LIST]
[*]To test and play, the easiest by far for a Windows user is [Wubi]("http://wubi-installer.org/"). Install Wubi, reboot your computer, and play. Reboot to get back to Windows. Warning: Wubi is fantastic for testing and playing, but is **not** good for long-term use, because it gets corrupted easily.
[/LIST]
If you like what you see:

[LIST]
[*]Download and burn [an Ubuntu CD]("http://www.ubuntu.com/") (it's called a Live CD).
[*]Uninstall Wubi.
[*]**Do a full backup of your Windows system!** Although the installation process is very reliable, sometimes a small error can leave you without your system.
[*]Install Ubuntu as a "dual boot". This means that you keep Windows on your system, so that you each time you boot, you choose whether you want to start Windows or Ubuntu.
[/LIST]
