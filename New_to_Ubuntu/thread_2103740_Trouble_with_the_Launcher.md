### Post by krustenBrot on 2013-01-12
Try setting the reveal sensitivity & pressure with an external app.
I am currently using *Unsettings* but there are alternatives out there.
*[COLOR=Blue][Here]("https://launchpad.net/~diesch/+archive/testing")[/COLOR]* is the PPA for Unsettings.
[COLOR=Blue][URL="https://launchpad.net/+help-soyuz/ppa-sources-list.html"]
*Read about installing*[/URL][/COLOR] is pretty good although it does not mention how to install the actual app:

after you completed the previous steps type:
```
sudo apt-get install unsettings
``` into the terminal and hit enter. Unsettings will now install. Launch it through the Dash or Settings screen.
