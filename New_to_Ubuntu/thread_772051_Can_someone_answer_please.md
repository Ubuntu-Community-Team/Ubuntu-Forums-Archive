### Post by natirips on 2008-04-28
> **lilxe07 said:**
> im gonna installit now but i have vista on c i wanna install on d how can i do dual boot

In Linux 'c' is a programme language :). Disk drives are not assigned letters on Linux. They are assigned mount points. Like "/media/sda1", "/media/sdb2", "/media/cdrom0", etc. Or "/mnt" instead of "/media" on some other Linuxes. With the excaption of "/" (root) partition, which acts like 'c:' on dos/windows. Make sure you have at least one such partition on your system ("/") formatted as "ext3". And you might want to make a "swap" partition. If you have loads of RAM (like 2GB), make it small (1GB or less), if you have low RAM (like 512MB or less) make it larger (like 1.5GB). The swap is a part of your disk which is used should your system run out of RAM. Also, if you want to use hybernation, at least 1.5 times RAM is a MUST. Make sure not to delete your windows partition though. You might want to resize your windows partition from windows to make some free/unsued space for Linux to use.

Fat is geneally used by windows 9x and older, also by DOS. Ext3 it what Linux uses. Notice that Linux can read/write fatxx and NTFS, but can only be installed on ext3. Windows however, cannot read ext3, so if you want to share data between windows and Linux, use fat32/ntfs for that, although you probably already have one such where windows is installed. But your root partition (the where linux is installed and is mounted at / ) MUST be ext3.
