Yes, generally, BUT as this is a Pentium 4 (and I think I have the same processor in my desktop) this would represent hyperthreading which impersonates a dual-core but is 32bit, not two physical processors, and NOT a dual core, so will not run on 64bit release. ;)

### Post by chipbuster on 2011-12-31
I believe the factor that limits the use of 64-bit systems is CPU architecture, not number of cores. I've seen several x86_64 systems that run fine on a single core Pentium.

The specification you have is rather generic. As Bucky Ball said, Intel uses something called "Hyperthreading" which nobody seems to really understand (myself included). It can apparently emulate a dual-core processor without actually being dual core. Without more detail, it would be hard to say.
