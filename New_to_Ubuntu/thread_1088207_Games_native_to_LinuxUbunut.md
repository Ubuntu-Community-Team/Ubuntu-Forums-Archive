### Post by ClaytonOT on 2009-03-05
I know you said you've already been directed to Wine, but I'm going to reaffirm that suggestion.

There's enough that's been written about it that you can configure games to run on it pretty well. [http://appdb.winehq.org/](http://appdb.winehq.org/)

You can see which games run well with wine and which don't. 

Getting wine is as easy as > sudo apt-get install wine 

Once you get wine installed you move the games into its Dos folder and execute it through terminal.
