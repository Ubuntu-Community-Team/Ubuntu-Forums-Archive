### Post by plucky on 2012-05-16
You know you can edit **your** posts.

Open a terminal and run ```
sudo rm /var/lib/apt/lists/* -vf
``` which will remove all the .list files in the directory and then run ```
sudo apt-get update
``` to reload them from the repos.

See if you get any errors.

Post the full output if you do get any errors.

Good Luck
