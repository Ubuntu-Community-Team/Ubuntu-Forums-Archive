### Post by sarfarazkazi on 2009-08-03
> **gprateek said:**
> What do these terms mean :warty, hoary, gnome, jaunty, :confused::confused:

These are the names given to each new release version of Ubuntu. Check this wikipedia link to know more - [http://en.wikipedia.org/wiki/Ubuntu_(operating_system)#Releases](http://en.wikipedia.org/wiki/Ubuntu_(operating_system)#Releases)

> **gprateek said:**
> and apart from that , i would like to know how can we install and play games , as they are done in Windows ... 
:(

You can install games and other application via Add/Remove Programs under the Applications menu.

> **gprateek said:**
> apart from that , there are many applications , which do not have the linux substitute  for using them , like Your-Freedom, feegate etc.

For Linux alternatives/replacement of Windows software, check this - [http://www.linuxalt.com/](http://www.linuxalt.com/) and this - [http://linuxappfinder.com/alternatives](http://linuxappfinder.com/alternatives)
