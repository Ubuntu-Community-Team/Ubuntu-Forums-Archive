If you only want to burn (write) the downloaded iso to a cd, follow this short guide: [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)

If you are using windows to burn it, I would suggest to download and use "[free iso burner]("http://www.freeisoburner.com/")". It is tiny, easy to use and since all it does is burn an iso to a disc, there is no chance of common mistakes.

If you want something more advanced, like backing up an installation on cd or creating a custom live cd of your own, try clonezilla or remastersys. Just ask here if you need any detailed help, but ask your question clearly and in detail.
