### Post by newb85 on 2012-11-07
Welcome to Ubuntu and the forums!

First resource should be the man (manual) pages.

To read the man page for a specific command,

```
man *command*
```

To search the man pages for a keyword,

```
apropos *keyword*
```

Of course, you can read more about apropos with

```
man apropos
```
