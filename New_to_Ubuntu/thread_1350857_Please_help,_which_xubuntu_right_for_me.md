### Post by jbrown96 on 2009-12-09
Older computers usually can't boot from USB, so you should try the CD. 

[Lubuntu]("https://wiki.ubuntu.com/Lubuntu") is the lightest Ubuntu that I know of. It's very miminal, and I'm not sure that I recommend it to new users, but you can try a LiveCD without any risk (just the cost of a CD).

Xubuntu is also "light," and is a full-featured *buntu. I was in a debate in a thread a week ago; Xubunutu is only marginally lighter than Ubuntu, with (in my opinion) a worse interface and apps. 

If your computer runs XP, then it should be able to run Ubuntu. Just make sure to create a decent amount of swap space (guided partitioning will do this automatically when you install).

I also highly recommend karmic. The requirements for ubuntu don't increase very much between releases, but the quality is leaps and bounds better.

### Post by halitech on 2009-12-12
The Live (or Desktop) cd is just that, a live cd you can use to run the OS from a cd. You can also use it to install the OS to your computers hard drive. The Alternate Install cd is just that, an install cd only. Both will give the same results but the Live cd does it in a prettier way where the Alternate install cd does it all with text. Normally the Alternate cd will install on systems that the Live cd fails on.
