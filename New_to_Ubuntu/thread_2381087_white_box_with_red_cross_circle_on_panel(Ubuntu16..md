### Post by deadflowr on 2017-12-26
It's a generic icon for any time an app does not properly have one,
or the icon pack used does not have the icon for whatever app it is.

Some apps add an icon to the tray (the name of the panel) when they are running,
such as dropbox, telegram, virt-manager, vlc and several others.

So have you installed anything like those?
