### Post by Impavidus on 2014-05-30
PDF wasn't really meant to be edited. Quite the opposite actually, it doesn't have to contain any information other than what's needed to show or print the document, not even the logical order of the characters on the page (although usually all text in a PDF file is ordered as it is read). If you have the source from which the PDF was generated available, edit the source and create a new PDF.
