Here are some things to check:
1.  Make sure you do not have a firewall between the client and the router. 
2.   If there is no firewall blocking requests then make sure your IP to your admin page has not been changed.
3. your client IP has to be in the network IP range to view the page.  For example the client IP needs to be 198.162.0.XXX to work
