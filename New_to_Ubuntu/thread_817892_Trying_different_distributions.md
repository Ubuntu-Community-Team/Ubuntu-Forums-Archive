### Post by kpkeerthi on 2008-06-04
Try [Zenwalk]("http://www.linux.com/feature/126526") 
- Slackware based.
- XFCE, fast, one application per task, sleek out-of-the-box setup.
- Simple and efficient package management.
- Has GUI tools to configure almost everything.
- Compiling from source is easy. [http://zur.zenwalk.org/](http://zur.zenwalk.org/)
- You can choose to stay on Stable branch (stability) or switch to Snapshot branch (stay bleeding-edge)
- Should appeal to beginners and advanced users as well.
