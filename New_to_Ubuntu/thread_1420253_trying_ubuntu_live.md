### Post by Rex Bouwense on 2010-03-03
> **3948ctyj said:**
> how do I make a USB..?? I have 9,04 on cd but this netbook has no CD  drive.But I still have my Ubuntu pc too.. can I make a usb using that one..??

OK, download the iso from [http://www.ubuntulinux.org/](http://www.ubuntulinux.org/) and then use UNetbootin ([http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)) to make a bootable flash drive.  Set your BIOS to boot from the USB drive and then you choose the option to try Ubuntu without making any changes to your computer.
I too have an EeePC and am currently using 9.04.
I just noticed that you had Ubuntu on another PC.  If that is true, you can create a start up disk by going to System->Administration->USB Startup Disk Creator.  That will get it done too.
