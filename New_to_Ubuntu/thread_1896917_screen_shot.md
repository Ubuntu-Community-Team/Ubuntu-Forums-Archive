### Post by amjjawad on 2011-12-18
> **Matrix01 said:**
> how do u take screen shot on Lubuntu?

Press Print Screen button on your Keyboard then check your Home Folder

OR

```
sudo apt-get update && sudo apt-get install gnome-screenshot
```

Enjoy!

P.S.
If you want to crop the pictures and use some other functions, install "shotwell":

```
sudo apt-get install shotwell

```
