### Post by Firestem4 on 2009-01-26
I downloaded Project Starfighter x86 .deb package and ran the installation. The Installation worked perfectly but I can not find the program in my menu (under games or anywhere). However I can run it from terminal by typing ```
starfighter
```

Can anyone tell me how I can make it so there is an icon or link in my menu?

edit: I'm using Kubuntu 8.10

---

### Post by snova on 2009-01-26
Right click on the menu button, Menu Editor.

Select Games, press New, type in "Starfighter".

Under Command, type "starfighter".

The icon (nice to have) will probably be under /usr/share/pixmaps.
