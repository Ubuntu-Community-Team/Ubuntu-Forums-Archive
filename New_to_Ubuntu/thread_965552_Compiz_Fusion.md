### Post by Sealbhach on 2008-10-31
You need to install 

compizconfig-settings-manager

You'll find it in Synaptic or just install from the terminal:
```

sudo apt-get install compizconfig-settings-manager
```

You'll find the settings tool in the System/Preferences menu.
