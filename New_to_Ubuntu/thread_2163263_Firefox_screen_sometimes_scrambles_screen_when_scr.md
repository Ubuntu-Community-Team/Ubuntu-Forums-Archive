### Post by suicideking on 2013-07-18
Definitely run the system updates. That will update Firefox to v22 and may fix other things causing your problem. 

Run the updates, reboot, run again. Keep doing this until it tells you there are no more updates.

New updates alert will be listed in the *Power Menu* (the right-hand cog icon), although you can also manually check by launching the ‘Update Manager’ from the Dash.
