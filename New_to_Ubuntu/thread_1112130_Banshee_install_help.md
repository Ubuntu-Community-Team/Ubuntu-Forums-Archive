### Post by James_Lochhead on 2009-03-31
Installing programs on Ubuntu is generally not done in the same way as Windows/Mac.

The easiest way to do it is to go to the menu Applications > Add/Remove Programs. Select All available applications from the drop down menu and search for Banshee. Check it and click apply. It should now be in Applications > Sound & Video.

### Post by tarps87 on 2009-03-31
Banshee is in the repositories (a store containing pre compiled software for Ubuntu)
Open up a terminal
Applications -> accessories -> terminal
now type
```
sudo apt-get install banshee
```
now enter you password (you won't see anything, just type your password and hit enter)
It will ask you if you want to install and then install

Or you can do it the gui way

Applications -> add programs
Select all applications and then search for Banshee, select install and now apply
