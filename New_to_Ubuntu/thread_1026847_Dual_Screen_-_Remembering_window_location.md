### Post by fubbleskag on 2008-12-31
to my knowledge, there is no way to do this in the default gnome installation.

however, if you are using compiz you can preset positioning by window matching via the Place Windows plugin. if you are not using compiz, you may want to look at DevilsPie ([http://live.gnome.org/DevilsPie](http://live.gnome.org/DevilsPie))
