### Post by mikewhatever on 2010-02-11
It's System->Administration->Hardware Drivers, to be precise.
Edit: In case you have trouble finding that, run **gksudo jockey-gtk** from Terminal.

### Post by darolu on 2010-02-11
Try what mikewhatever said, if it still doesn't work, [click here]("http://www.nvidia.com/Download/index5.aspx?lang=en-us") and download the driver directly from nvidia.com, to install you just need to run this in a terminal:
```
$ sudo sh NVIDIA-Linux-x86-173.14.22-pkg1.run
```
