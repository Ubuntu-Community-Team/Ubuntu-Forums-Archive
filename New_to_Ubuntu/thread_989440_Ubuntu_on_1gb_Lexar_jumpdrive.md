### Post by bennachie on 2008-11-28
Don't confuse using a USB as a "LiveCD" equivalent (for which 1GB will be just fine) with actually booting and running Ubuntu from a USB, with any changes you make being recorded, and files stored, on the flash drive. 

For that, you will need at least 2Gb, and 4GB would be better.

### Post by 3rdalbum on 2008-11-28
If you don't have success with the Ubuntu live USB creator (System > Administration > Create a USB Startup Disk), you should try UNetbootin; it sometimes works where other programs fail. For instance, the Ubuntu method doesn't boot on an Acer Aspire One. When you use UNetbootin instead, it will boot.
