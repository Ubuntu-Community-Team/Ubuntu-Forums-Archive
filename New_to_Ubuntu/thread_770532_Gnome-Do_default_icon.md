### Post by Ub1476 on 2008-04-27
It's because that there are no higher resolution available for the "search" icon. Gnome-Do doesn't provide this itself if the current icon theme has something it can use. 

I'm using a tango theme generated by [Tango-Generator 3]("http://www.gnome-look.org/content/show.php/Tango+Generator+3?content=72361"), which provides tango-guidelines icons for every Linux app with high resolution available. Just look at my picture below. 

If I change to the human icon theme I get the same icon as yours.
