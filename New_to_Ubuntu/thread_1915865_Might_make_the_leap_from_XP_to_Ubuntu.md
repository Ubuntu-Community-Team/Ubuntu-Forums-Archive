### Post by 3rdalbum on 2012-01-27
For starters, I'd suggest you get to grips with the Linux software wherever possible. Windows software does not integrate with Linux very well, for obvious reasons.

So, forget running Winamp; there are dozens (hundreds?) of Linux-based music players that will work very well. I'm sure you'll find one you like. XMMS is old and I don't even think it's maintained anymore.

I can't think of anything you'd really NEED the command-line for in ordinary use. Installing programs in Wine maybe:

```
wine (drag program to the terminal)
```

It's that easy.

Your Windows installers litter shortcuts all over a Windows desktop and from my experience Wine emulates this too, so you shouldn't have any problems in making it easy for the programs to run once installed.

> I need a computer that runs faster but can still run my loved programs at the same rate.

If this is your main motivation for switching to Ubuntu, you may be disappointed. Ubuntu is an operating system in its own right. It's not "Windows but faster". In fact, Ubuntu may well be slower than Windows XP due to the latter being ancient and obsolete. If you simply want "Windows but faster" then you need a new Windows PC. If you want "Better than Windows" then you should stick with Ubuntu.

### Post by Mark Phelps on 2012-01-27
With one of the older Ubuntu versions (ie 9.x) you might have seen major speed improvement on your PC over XP, but not with the new 11.x vesions.

As to the apps, have to reiterate what others have already told you -- Photoshop, WinAMP, and Illustrator won't run in Ubuntu -- Wine doesn't support them well.

Looks like all you're looking for is a free VERSION of Windows -- and Linux is not that; it's an alternative, including the need to learn to use DIFFERENT apps than in Windows.

My advice, if you really want to try out using Ubuntu, then dual-boot your PC; otherwise, stay with Windows.
