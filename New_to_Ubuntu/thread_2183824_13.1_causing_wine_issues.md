### Post by grahammechanical on 2013-10-26
What makes you think that Ubuntu 13.10 is causing this? It seems that you have three versions of wine installed - wine 1.5, wine 1.6, wine 1.6-i386. The version of wine in the Ubuntu 13.10 Software Centre is wine 1.4. That is why those later versions have to be removed. Have you checked the wine application database about running Netflix under wine?
