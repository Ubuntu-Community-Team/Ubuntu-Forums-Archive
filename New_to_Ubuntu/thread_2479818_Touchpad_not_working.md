### Post by Impavidus on 2022-10-09
By latest version, I assume you mean 22.04. Some people read about Ubuntu on some old blog and get mistaken about the latest version (and then install some older version, believing it's the latest).

Most touchpads work out of the box, but some may need additional drivers. There is an additional drivers utility in Ubuntu, that may be able to find and install such drivers for you. But as this touchpad fails to work in both Ubuntu and Windows, it's less likely that this is a software issue.

### Post by westie457 on 2022-10-09
Further to what @Impavidus wrote, it is possible/probable that the Function button that turns the touchpad on and off has been pressed.
