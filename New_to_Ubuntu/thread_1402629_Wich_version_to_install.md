### Post by ptn107 on 2010-02-09
It all depends on what you want to do with your operating system.  Ubuntu Studio is geared toward professionals needing audio/video software.  If you're just a general computer user I would stick with just regular Ubuntu.  If you need a particular piece of audio/video software you can just install it on regular Ubuntu.
