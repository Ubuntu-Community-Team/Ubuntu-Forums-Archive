### Post by wep940 on 2011-06-10
Here's a couple of things to check for getting the driver:

- go to system/administration/additional drivers and see if a driver is listed there for your video - if so, be sure it is enabled

- if not, connect to the net and do the following:

[LIST]
[*]open a terminal window (applications/accessories/terminal)
[*]type:  sudo apt-get update <press enter> and wait for it to finish
[*]go to system/administration/additional drivers - it will probably run a while searching for any drivers you might need that it can find, and if it finds them it downloads them.  When it has finished check the window to see if a driver for the video shows, and if so, enable it
[/LIST]
- if neither of these 2 options work, then check in synaptic package manager for nvidia - it will return several things.  I'm not familiar with your adapter, so I don't know if one of the ones listed there will work or not - perhaps someone more knowledgeable can help you there.

### Post by beew on 2011-06-10
Well check if your laptop has Optimus. Many new laptops  with Nvidia cards come with Optimus enabled and unless there is a BIOS switch to turn off Optimus they wouldn't work with Linux even though the cards have Linux support (there is an experimental way to get some Optimus laptops work with some success but there is no point getting into that until you have checked)
