### Post by jmill134 on 2013-01-16
sudo vim /etc/network/interfaces

(if you don't have vim installed, you can use 'sudo apt-get install vim' or use 'gedit')

Press 'i' to switch to insert mode and add this line to the bottom of your network configurations

dns-nameservers 8.8.8.8 4.2.2.4  (Or whatever DNS servers you would like to you use)

Then pres the 'Esc' key to return to command mode. While in command mode type ':wq!' and press enter.
That's it

I'm sure most Linux users know how to do this but just in case you are new to it like me, this works!
