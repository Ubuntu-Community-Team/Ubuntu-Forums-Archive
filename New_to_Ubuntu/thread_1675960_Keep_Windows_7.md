Thanks!If you like gaming on your PC, then keep Windows. 

Also, if you already have the dual boot going, and have all your games set up in your Windows partition, then you can install WINE (as Rubi1200 suggested) on Ubuntu and try running your already installed games from your Windows partition. This way, you will know how many work with WINE. If they all do, then you can move those Windows program folders (the ones with the games) into your Ubuntu partition and go from there (I have a "Games" folder in /home and allow WINE to access it).
