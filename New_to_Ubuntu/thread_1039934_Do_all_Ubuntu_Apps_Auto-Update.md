### Post by Circus-Killer on 2009-01-15
everything that is pre-installed on ubuntu will be updated when you check and install updates.

any software you install using apt-get/aptitude/synaptic will update when you check and install updates.

only programs you install manually wont be updated.

### Post by hyper_ch on 2009-01-15
however updates will not occur at the same time that a new version of the software is released. Ubuntu is not a rolling distro but has release cycles of 6 months. Within this period you will (normally) not get any version increase of the software but bugfixes and security fixes.
