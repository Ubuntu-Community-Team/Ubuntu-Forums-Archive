### Post by blueridgedog on 2009-03-02
Though Rupesh_001 is right, the fist question can be as simple as:

```
gksudo gedit /boot/grub/menu.lst
```

But not knowing what you want to do makes it hard to go any further.

As far as .bz2 goes, it is made with bzip2 and you can uncompress it with:

```
bunzip2 /path/to/filename.bz2
```

As far as telling you what to do with what is inside, that requires more information.
