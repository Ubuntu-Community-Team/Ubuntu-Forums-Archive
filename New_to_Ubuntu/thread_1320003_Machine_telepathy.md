### Post by Mariane on 2009-11-12
These currents run in dendrites and axons which connect the neurons together. They don't run like electrons in a wire, they run through chemical ions moving about. 

So it's like a mass of wiring (except it's edible - yuk) the currents run in the wires but the wires are isolated from the external environment so it's a circuit and not one big shortcircuit. 

Sometimes the isolating myeline does decay and when it does you lose it.
