### Post by QIII on 2014-10-26
See [this]("https://sites.google.com/site/easylinuxtipsproject/java").

At the point where it says to download the file, just download it on a machine that has an internet connection and copy it to a USB.  Take it to your other machines, put the file in the directory suggested on each machine and go through the installation process.

Be aware, however, that this will not keep Oracle Java updated when Oracle releases new ones.

Just follow the instructions for removing and reinstalling -- using a newly downloaded file.

Cheers.
