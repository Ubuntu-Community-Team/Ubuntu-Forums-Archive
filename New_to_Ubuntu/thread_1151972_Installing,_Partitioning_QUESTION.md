### Post by argail1980 on 2009-05-07
Hi,

you can shrink the partition during the install process (at least when you install from the alternate CD. But that should work with the desktop-CD too.) It ususally just asks what it should do with the harddisk and to shrink the windows partition is one of the options.

In any case you can boot the LiveCD and start "gparted" in the Live environment. That tool can resize partitions and so on. Easy to use, no text-console ;-)
