### Post by garyed on 2008-09-11
I have a Brother MFC 420CN printer hooked up & working on my Windows 98 Computer. On the same router I have another Ubuntu computer hooked up & can share files with no problem. I've been trying to print from the Ubuntu comptuter unsuccessfully. I've set it up with system>administration>printing   
& it looks like the network printer is installed but when I try to print it just hangs on the print job but doesn't work. I've got the printer set for sharing in 98 so I'm at a loss. Any ideas?

---

### Post by garyed on 2008-09-11
Evidently the problem was with 98 & not with ubuntu.
After reading dozens of network printing tutorials I found the answer was in
running a file called netsetup.exe on my 98 computer. It's found in the windows/system32 directory but in XP & not in 98 so I  transfered the file by floppy. Once I did that Ubuntu recognized the printer & it works fine.
