### Post by ByteJuggler on 2008-04-25
Well, most software that you install on the system will hopefully/should come from the repo's, so the question doesn't even come up.  It's entirely automatic and hidden by the package management system.  

If you do download .deb files manually, then you can put them wherever you like.  I would suggest making a folder in your home folder and putting them there.

### Post by nickpaton on 2008-04-25
Downloaded programs should be saved in your /home/your_name/ folder.

When subsequently running the program in Terminal, by default it will look for the program in this folder.

From Gutsy (or was it Feisty?) Document, Video, Music, dPictures folders are automatically created within your /home/your_name/ folder, so I would suggest saving Word Processing docs in Document, etc, etc.
