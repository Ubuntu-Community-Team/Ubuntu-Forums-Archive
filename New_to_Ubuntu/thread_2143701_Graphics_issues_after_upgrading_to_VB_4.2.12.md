### Post by huubb on 2013-05-09
I run Virtualbox on my Windows 7 PC. In one of the instances I run Ubuntu 12.04.2 (LTS). Till the beginning of this week everything seems to work nice. I did run version Virtualbox 4.2.10. Then I received a message that VB 4.2.12 has been released. OK, installing and going through the upgrade process both for VB 4.2.12 and the Questadditions 4.2.12.

Initially also this combination seemed to work fine. This till I discoved I couldn't change the size of the icons in the launcher. After long research time I noticed that Compiz (3D) didn'd work. Unity did. I also did software update on the Ubuntu end in the last few days. After checking all Ubunte software components it got clear to me that I had to find a solution outside Ubuntu.

The last step was downgrading to Virtualbox 4.2.10 and my problems disappeared! So be careful if you upgrade to VB 4.2.12
