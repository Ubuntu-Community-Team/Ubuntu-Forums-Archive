### Post by vanadium on 2008-12-07
For daily use, you do not need the terminal.

For standard administration task, you normally do not need the terminal.

Using the terminal is by far more precise and fast, so after some learning, you might WANT to use the terminal. But you do not have to.


There are a few possibilities for your transition.

You take the plunge: erase Windows, install Ubuntu instead. You are frustrated the first few weeks because you are missing a lot you did in Windows. You persevere and after a few months you realize that in fact you are doing well without Windows. You are already doing many things in a different way.

You install a dual boot system, either a true dual boot or a wubi install. That might be the safest approach, but it risks to keep you hooked in Windows after all.

With dual boot, you choose the system you load. When choosing Ubuntu, it has full control of your computer, exactly as if it were the only system installed. However, the size of your Windows partition must be reduced and new partitions must be created. The Ubuntu installation program can do that mostly automatically, though.

A Wubi install is also a dual boot system, but this time, Ubuntu is not installed in a separate partition. Instead, it is installed in a file residing on your Windows partition. During startup, a sophisticated system turns that file into a virtual file system for Ubuntu to use. The advantage of Wubi over a separate-partition dual boot is that install is so easy: you install Ubuntu from within Windows just as you would install any other windows application. Ubuntu can be removed with the same ease. Briefly, the advantage of wubi is that it does not change your current system. The disadvantage is that there is a theoretical performance loss because a virtual file system is used.

You have taken the approach "the other side of the spectrum", running Ubuntu in a virtual machine. This has the drawback you stated yourself: Ubuntu does not have all resources and will run slower within the confines of the virtual machine.

So there are several different options for you over running Ubuntu in a virtual machine. Which is best? It is your choice.
