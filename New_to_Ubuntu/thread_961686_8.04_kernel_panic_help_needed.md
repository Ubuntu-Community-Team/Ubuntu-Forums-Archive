### Post by macogw on 2008-10-29
1. Boot from your live cd
2. Mount the disk (Places -> Computer should let you)
3. Applications -> Accessories -> Terminal
4. Type "mount"
5. Type "cd " followed by whatever mountpoint that command told you the disk is on
6. Type "cd boot"
7. Type "ls"
8. What files are listed?  Please post here.
