### Post by Zill on 2010-05-26
Yanno:  Your reference to "MS ppt files" has confused me as I didn't think PPT was used as an equation editor. :confused:

However, I would first check to make sure the OOo equation editor is installed...
```
sudo apt-get install openoffice.org-math
```
If this is installed then you might take a look a the [OpenOffice.org Math FAQ]("http://wiki.services.openoffice.org/wiki/Documentation/FAQ/Formula")

Finally, the [OOo forums]("http://user.services.openoffice.org/en/forum/") might be of more assistance.
