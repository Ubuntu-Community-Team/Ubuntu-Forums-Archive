### Post by zerubbabel on 2013-01-27
Welcome to Linux Land! 

If you watch those auctions with a web browser such as Firefox, there should be no difference from Windows. But if you use a Windows application, it may work under Wine, a Windows compatibility layer that you can install via the Software Center or Synaptics package manager. After you install Wine, then you can install Windows apps just by double-clicking on their Setup.exe or whatever they name their installation executable. Not all Windows apps are "well behaved" enough to run under Wine, but many will run just fine.

### Post by zerubbabel on 2013-01-28
The easiest way to make the transition is to install Ubuntu alongside Windows, and then when you boot Linux, your Windows partition will still be accessible within Linux, and you can copy your files over as you need them. Otherwise, you could copy your user profile (documents, email profile, etc. -- the Windows equivalent of a Linux home directory) onto an external drive before you overwrite your Windows partition with Linux. I'd just get a big enough USB drive to copy over everything you may need from your Windows environment as is, as file folders, not using any kind of backup or compression software, as it will be easier to copy into your Linux home directory. 

I haven't used Windows in years, so I'm probably not the best one to advise you. Perhaps others will add to this.
