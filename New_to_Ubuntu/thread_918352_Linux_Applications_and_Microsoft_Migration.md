### Post by Rolcol on 2008-09-12
Welcome to the community.  A lot of Microsoft programs have a linux alternative to it.

VLC is not installed by default but you can install it from the repositories.  Have you tried using it for DivX and Real Player files?

Of course firefox is the default browser in Ubuntu.  Most of the addons for firefox should work and will work unless they use the Windows API or any of its programs.  For example, IE Tab (or whatever it's called) cannot be installed since it needs Internet Explorer to be installed.

OpenOffice.org is great if you're looking for an alternative to Word and PowerPoint.  If Microsoft was going to sue OOo, they would have done so by now since I'm sure Microsoft knows of it.  The support of .doc is a bit glitchy but it's enough to extract text from it.

You can use Rhythmbox for your iPod which should have as much compatibility as Amarok.  Amarok is a KD3 application so it looks different from the default GNOME environment.

If you need to use windows applications, you can attempt to install them with WINE.  WINE is a Windows to Linux compatibility layer.  It doesn't support all files but it does alright with the most popular.

Ubuntu has an included program that does great at opening Adobe PDF files.

As for Nero, you can use Brasero or K3B (another KDE application that will look different on GNOME).  They can burn files and CD images.

I can't answer all of your questions in one go but hopefully others can help you.

### Post by Zzl1xndd on 2008-09-13
Icyest, Welcome to Linux and jumping in like this is great its how I made the move a few years ago.

as for your question

iTunes/Winamp - I would recommend Amarok it can sync with ipods, has a built in internet radio stations and its own music store. (it might look a little out of place in gnome but its a great program non the less)

VLC - Is available in Linux 

The Divx codec is available but I don't believe their player is

Realplayer is also available for Linux

Firefox - Most extensions work I have only found 1 or 2 that don't 

MS Office - Open Office is fine for this, you can even set it to save in MS formats by default in your settings, I do all of my school work in it without any issues.

AIM - The default IM client for Ubuntu Pidgin can log into AIM, ICQ, MSN, Yahoo, IRC, Groupwise and a few others.

Winrar, I recommend 7zip to fill that void

Nero - Brasero can do most everything Nero can, and if you would like to burn DVD movies I recomend Devede

Adobe reader - Has a Linux Version but there is also a Default PDF reader installed and I prefer it to the Adobe one.

Gimp - Great program although not Photoshop level it is more then enough for a novice to work with.

As for Extra's I like Deluge as a bit torrent client, and I use GTK-record my desktop a lot to show new Linux people how to do things.
