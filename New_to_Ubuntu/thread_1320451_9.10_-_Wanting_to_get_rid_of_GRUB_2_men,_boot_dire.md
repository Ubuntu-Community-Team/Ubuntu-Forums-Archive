### Post by philinux on 2009-11-09
You need to edit this file.

```
gksudo gedit /etc/default/grub
``` and change the GRUB_TIMEOUT="10" to say 0 or 1.

Then run.

```
sudo update-grub
```

Is this a wubi install?
