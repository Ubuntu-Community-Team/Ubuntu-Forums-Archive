### Post by Shmnki on 2013-06-14
So I just installed Ubuntu 12.04 on my Macbook Pro, which was previously running OS X and Windows 7 (through Bootcamp), and I noticed in the installation process that Ubuntu gave me the option to replace OS X entirely with Ubuntu. My question is if I replace OS X with Ubuntu (to save hard drive space), can I still dual boot to Windows even though I'm on an apple product?

---

### Post by fantab on 2013-06-14
It has been noticed that when 'Replace' option is used entire HDD is being formatted. So yes, probably you will lose everything on the HDD. Just be careful and BACK UP all your important DATA.
However if you just want to replace OS X and keep Windows then you must use 'Something Else' option to manually manage Ubuntu installation on the target HDD.

Good Luck...

### Post by Mark Phelps on 2013-06-15
You would think that the Replace option would only overwrite a single OS, or a single partition associated with an OS -- but posts here have indicated that using Replace effectively reformats the entire drive!  So, you did good to come and ask, first.
