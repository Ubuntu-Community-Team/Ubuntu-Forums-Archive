### Post by grandtoubab on 2009-11-10
[http://ubuntuguide.org/wiki/Ubuntu:Karmic](http://ubuntuguide.org/wiki/Ubuntu:Karmic)
** Hardware requirements**

 Ubuntu Karmic Koala runs well with as little as 384 Mb RAM. (The  installer requires a minimum of 256 Mb RAM, while the alternative  installer can run using only 192 Mb RAM.) [Netbooks]("http://ubuntuguide.org/wiki/Ubuntu:Karmic#Netbooks")  can run Ubuntu Karmic Koala. 
The install takes between 3-4 Gb hard drive space, and 8 - 10 Gb  will be needed to run comfortably. 
If you have an older computer with less memory than this,  consider [Xubuntu]("http://www.xubuntu.org/") (if between  256 Mb and 512 Mb RAM) or [PuppyLinux]("http://www.puppylinux.org/")  or [DSL]("http://damnsmalllinux.org/") (if less than  256 Mb RAM). 
Warning: Karmic Koala has a serious flaw in the motherboard/fan  sensor modules (libsensors and lm-sensors). It requires that your  hardware be compatible with existing drivers, or your computer will slow  to a crawl and then freeze. There is no way to fix this problem,  currently. Unless you have a very new computer, you may experience this  problem. I was not able to use 2 of my 5 computers with Karmic because  of this.
