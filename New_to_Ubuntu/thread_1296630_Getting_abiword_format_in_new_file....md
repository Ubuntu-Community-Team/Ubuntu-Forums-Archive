### Post by Mike_IronFist on 2009-10-20
> **Chris Edgell said:**
> I have gone from 8.04 (Hardy) to 9.04 (Jaunty) and now when I go to:
Chris>Songs>(new file), I get what looks like a notepad instead of Abiword format.  What am I doing wrong?

You know the Templates folder within your home folder? That's for creating new files based on files you have already saved in there. You can use it to create new files in certain formats.

In this case, open AbiWord, and save a blank document with the name "AbiWord Document" (without the quotes) in your Templates folder. Then when you right-click to create a new document, there should be an option to create a new AbiWord document.

**EDIT:** I'm not entirely sure if this works identically in Xubuntu (which uses the XFCE desktop). Sorry if I'm mistaken.

### Post by Mike_IronFist on 2009-10-20
Since it's tough for me to describe the process in words, I decided to take some screenshots. It's very, very easy to create templates. So here's a step-by-step (and picture-by-picture) process showing how it's done. You can click any of the thumbnails in this post to see the fullsize version of the image.

[[IMG]http://img32.imageshack.us/img32/9981/howtotemplates01.th.png[/IMG]](http://img32.imageshack.us/i/howtotemplates01.png/)
First we go to our home folder. In this demonstration, the user's home folder is called "Jimmy". Yours is called "Chris", if I'm not mistaken. Make sure that there's a folder in there called "Templates" (without the quotes). If not, make one. Make sure that the capitalization is correct - Linux is case-sensitive.

[[IMG]http://img202.imageshack.us/img202/3869/howtotemplates02.th.png[/IMG]](http://img202.imageshack.us/i/howtotemplates02.png/)
Next, we open AbiWord.

[[IMG]http://img188.imageshack.us/img188/474/howtotemplates03.th.png[/IMG]](http://img188.imageshack.us/i/howtotemplates03.png/)
Now that AbiWord's open, we just need to save a blank AbiWord file. Click "Browse for other folders" so that you can see the folders in your home folder. Double-click on your Templates folder.

[[IMG]http://img26.imageshack.us/img26/8971/howtotemplates04.th.png[/IMG]](http://img26.imageshack.us/i/howtotemplates04.png/)
We save the blank AbiWord document in the Templates folder. I chose the name "AbiWord Document" (without the quotes, of course) to keep things simple, but obviously you can call it whatever you want.

[[IMG]http://img9.imageshack.us/img9/3421/howtotemplates05.th.png[/IMG]](http://img9.imageshack.us/i/howtotemplates05.png/)
Once the file is saved, right-clicking in any open folder should easily allow you to create a copy of that blank document!

And THAT's how you create a template.
