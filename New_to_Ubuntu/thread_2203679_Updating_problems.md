### Post by ajgreeny on 2014-02-04
> How do i install these kind of files on my Ubuntu 12.04 LTS system?
You don't normally update software that way when using a Linux distro; you do it through the repositories and the OS's  own update manager system.

tar.gz files are just an archive like a .zip file, and my advice is not to bother using them for installation of software unless it is the only way available.

Just wait a day or two and I anticipate TB will be updated in the ubuntu repositories and available for you.

### Post by deadflowr on 2014-02-04
Open the update manager and run the check command.
Does an update for thunderbird come up?

Look at the about in thunderbird.
I think it's in Help, or whatever is at the end of the menu bar.
What version does it say?
Right now it should be version 24.something.
If no updates have been listed for thunderbird, it's safe to say the package is up to date.

It's actually one of the few packages on Ubuntu that gets the latest and greatest.
