### Post by linux4life88 on 2010-06-08
I was able to get my mouse back. Here is some detailed information on my laptop:

Brand: Asus
Model: N81vp-c1
Graphics: ATI Radeon HD4650 1GB DDR3

Above I said that I was running the 10.3 version of ATI driver, but I was wrong. I was running the 10.2 driver. I solved my problem my uninstalling the 10.2 driver and installing the 10.5 driver. The mouse is now back once again. I hope this can help if someone runs into a similar problem on Karmic.
