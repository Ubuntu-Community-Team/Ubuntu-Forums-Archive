### Post by yeats on 2009-01-19
If I were you, I would open the Synaptic Package Manager and search "gtkglext", which will show you what packages exist in the repositories, then I would install from there. The package will probably be called "python-gtkglext1" or something like that.

Installing from source is a good thing to learn, but it's an easy way to create dependency conflicts that could result in a broken system.  I speak from experience :-).
