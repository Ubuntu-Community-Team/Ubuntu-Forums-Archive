If it is a network printer did you configure the printer itself with a static IP? If it gets an automatic IP is is possible that changed? When you gave Ubuntu the IP of the printer did it find a driver for it automatically and was it initially able to print?

Other than that, Xerox printers usually do postscript, so they should just work. Many years ago (before cups when printcap was used) I had to print something at a Kinkos Copy Center (now FedEx Office) from my Linux laptop and could not find a Xerox driver. I simply configured it as generic level 2 postscript and it worked to print a 20+ page manual.

sudo restart cups
