### Post by andreazzz on 2013-09-27
My brother has also a Sony Vaio S laptop and he is experiencing the same problems with different Linux distributions. The problem might be caused by not having Catalyst Control Center installed (in which it is possible to adjust the CPU speed). But there you have an Intel processor, I do not really know if it is possible to adjust the CPU speed in CCC (from AMD).

During my Linux Mint period, I had used a nice program: CPUfreq. It is possible to install it in 13.04: [http://itsfoss.com/install-cpufreq-ubuntu-1304/](http://itsfoss.com/install-cpufreq-ubuntu-1304/)

Good luck!
