### Post by MG&amp;TL on 2012-12-05
Yup, my guess is that you need a driver for your video card. Head to the "software sources" application, then to the "Additional drivers" tab. Here you'll see some drivers (hopefully...) which you can choose, download and install.

Then you'll need to reboot, and all should be well. If it's still slow, it's possible you might need some lighter alternatives, such as Lubuntu or Xubuntu. Ubuntu went through some (drastic) interface changes recently and we still haven't quite caught up yet. :)
