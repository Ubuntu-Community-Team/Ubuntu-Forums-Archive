### Post by crlang13 on 2010-02-17
Hey mate, welcome

Have you installed Ubuntu yet?  It comes with quite a few programs.

It comes with the basic open office package (applications > office) which includes a word processor very similar to microsoft word, spreadsheet (excel), and presentation (powerpoint).  All these programs can read, edit, and save in a Microsoft format if you want.

For music, check out rhythm box (applications > sound and video) it's very similar to itunes.

All of the above come default with Ubuntu 9.10.  Firefox also comes of course as well.

Check this out: [http://www.ubuntupocketguide.com/](http://www.ubuntupocketguide.com/) - it's a nice little beginners guide that will help you out with finding more programs.  Everything that you get through the synaptic package manager or ubuntu software center is very easy to install and uninstall so don't worry about trying out a program to see if you like it.
