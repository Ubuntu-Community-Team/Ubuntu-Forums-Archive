### Post by le singe on 2009-01-21
Also, the cursor theme should normally change on the fly as soon as you select it.  If for some reason it doesn't (you select a different cursor and still the default appears) and if you're using compiz effects, you can force the change by going under the general options of CompizConfig Settings Manager and typing in the name of the cursor theme exactly as it's listed.  I say this because I have to do this myself in order for my settings to recognize the change.

And yeah gnome-look.org has a bunch of different themes.
