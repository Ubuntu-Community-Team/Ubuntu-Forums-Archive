### Post by mcduck on 2009-09-12
> **Kaymeerah said:**
> awww.. any other theme managers out there capable of what emerald does?

If "capable of what Emerald does" means transparency, you are not going to get that without running some compositing manager (like Compiz).

Quite many other window managers support transparency, but they all still require some compositing manager to do that. And none of them are built to use transparency to the same extent Compiz/Emerlad does, since they all need to be usable even without compositing. Compiz is different in the sense that it combines compositing manager and window manager into one application (and since Emerald is just a plugin for Compiz it doesn't need to worry about not having a compositing manager available).
