### Post by labinnsw on 2010-02-12
1. Open virtual box
2. Click on the virtual machine to highlight it
3. On the shortcut menu, click the "Settings" icon to configure the machine
4. In the right panel, select Storage
5. Under Storage Tree Add the Windows 7 ISO file.
6. Click to highlight the file
7. Under Attributes set the slot to either IDE Primary Master or IDE Secondary Master.
8. You should now be able to boot up without the error.
