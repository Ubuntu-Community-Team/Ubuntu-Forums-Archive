### Post by 3rdalbum on 2009-10-07
Netbook Remix is ordinary Ubuntu with the "Netbook Launcher" interface.

Moblin Remix is as the above, but with the Moblin interface.

MID edition is a slimmed-down platform designed for running on MIDs and mobile phones, with a number of different UIs that can be used or developed.
