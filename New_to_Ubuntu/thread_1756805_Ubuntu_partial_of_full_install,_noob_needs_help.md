### Post by oldfred on 2011-05-12
How big is your hard drive? How full is your windows partition? If you have room it is best to start with dual booting.

Have your used all 4 primary partitions - most laptops do as windows uses 2 a boot/recovery and the main install and then the vendor adds a partition that is just the image of your hard drive as purchased (not win7 install disk) and then a vendor utility partition. You can and should backup all important data and make the image backups to DVDs that you can do.

Most users starting out find they have some application in windows they cannot live without, but Linux offer many equivalents, some good, some better than windows and some just not as good. Only some applications such as Firefox, Thunderbird, Openoffice or LibreOffice offer versions native to each operating system.

Have you downloaded a Ubuntu liveCD or USB live version to test how well it runs on your PC. The live version is not as fast as a full install and does not let you save info (persistence on USB will let you save some info).
