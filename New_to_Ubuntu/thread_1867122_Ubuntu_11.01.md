#1. Probably, but would be better off running Lubuntu or Xubuntu
More important question is how much RAM is installed?

#2. Probably most any size, but it must be a PATA drive, that board does not have a SATA controller

The bigger problem will be the onboard VIA graphics chip.
Pretty flaky graphics support on Linux, 2d rendering would probably be okay, but I doubt full 3d support.
Which means the Unity desktop most likely will not work.
Hence the suggestion of the other two flavor of Ubuntu above.

If you have enough RAM and can find a decent AGP video card  to install then it would open up the possibilities more.

Wow. Integrated VIA. I'm not sure even Lubuntu would run on that, although there's no reason you can't try a desktop CD and see what happens. Try stuff out and let us know what happens. :D

If you have docking station for a SATA drive, you can hook up a 1TB drive via the USB no problem. Just don't expect to be able to boot off it. (So, put your system on the internal drive and all your big files on the external or something like that.)
