### Post by Steelmourne on 2009-06-27
[FONT=Verdana]If you need RAR support for extracting files, open up terminal from the Accessories menu under Applications and type:

[/FONT][FONT=Verdana]sudo apt-get install unrar
[/FONT][FONT=Verdana]
This will install support for extracting RAR files, but you won't be able to create RAR's.

Note: Make sure the multiverse repositry is enabled under Software Sources.[/FONT]
