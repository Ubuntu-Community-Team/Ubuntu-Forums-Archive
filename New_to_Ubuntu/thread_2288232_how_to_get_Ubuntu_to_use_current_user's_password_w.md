### Post by NoWayWin8 on 2015-07-27
> **stupidquestion said:**
> When I tried to install a software from the Ubuntu Software Center, Ubuntu prompted me to enter my old username's password. I assume this is related to user credentials. How do I get it to ask for my current account's password instead? The first account you used is the Administrator account, and that account's password is needed for administrative ("root") tasks like installing software.. You can change the password for that account in System Settings / User Accounts. You can also make your other account an administrator instead of standard user.

### Post by newb85 on 2015-07-27
I think NoWayWin8 is onto what you want.  If you want another user to have administrative privileges, you have to make their account an administrator account.  

If you're using regular Ubuntu, this can easily be done by accessing User Accounts from the Dash.  Click the user you want to change on the left.  Hit the unlock button in the upper-right.  Then change their account type from Standard to Administrator.
