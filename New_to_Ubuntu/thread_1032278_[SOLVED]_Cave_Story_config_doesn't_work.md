### Post by Dirjel on 2009-01-06
I'm trying to disable the game pad in Cave Story, but every time I close the DoConfig.exe, it resets all the options.  Is there a file I can just manually edit somewhere that will let me disable the controller?

EDIT:  I fixed it myself.  I had to download the Windows version of the game, run THAT DoConfig.exe, then copy the resulting Config.dat file into /usr/share/doukutsu.  That fixed it up.
