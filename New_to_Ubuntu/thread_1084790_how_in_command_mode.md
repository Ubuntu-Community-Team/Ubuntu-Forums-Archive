### Post by Ianprime0509 on 2009-03-02
Well for browsing folders you can use these:

dir - lists all files and folders in a directory
cd <directory> - navigates into the specified directory

If you're connected to the Internet, you can run lynx as a text-only internet browser.

You can also use ./<application> to run the specified application.

To find out more commands, just use:

help

and then run

man <command>

To find out how to use it.
