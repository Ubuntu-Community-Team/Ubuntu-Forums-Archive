### Post by amanchesterman on 2014-04-10
It's possible that your user profile has become corrupted somehow. Quoting from the Libre Office help pages: "[COLOR=#000000][FONT=sans-serif]If you notice any strange behavior on LibreOffice, or simply it fails to start, the first thing to try is to reset the user profile." That advice, and the instructions on how to reset it, are here: [https://wiki.documentfoundation.org/UserProfile](https://wiki.documentfoundation.org/UserProfile)[/FONT][/COLOR]

If that doesn't work or if you need more help I can recommend the unofficial Libre Office help forum here: [http://en.libreofficeforum.org/](http://en.libreofficeforum.org/) They have some very knowledgeable folks there! Remember that although Libre Office is packaged with Ubuntu, it isn't a Ubuntu 'product', so asking for help on the Ubuntu forums isn't always the best strategy.
