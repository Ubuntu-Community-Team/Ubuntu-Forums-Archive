### Post by go2dell on 2009-04-21
You can follow the [Dialup Modem Howto]("https://help.ubuntu.com/community/DialupModemHowto") over on the Community Help site.  Just be forewarned that this can be one of the most complex topics you can encounter in using any GNU/Linux distro, so try to remain calm and take plenty of notes on what you've tried to make your modem work.

Be sure to start with the ScanModem step, since you'll need that information later if you need more help from the forums.

The Ubuntu help site ( [http://help.ubuntu.com](http://help.ubuntu.com) ) can answer lots of questions without having to wait for answers from the forums.

### Post by lkraemer on 2009-04-21
bobin,
Are you using a Desktop, or Laptop? (I know it doesn't make any difference
but it will help us know what you are using!)
What Version of Ubuntu? 8.04, 8.10, or ?.??
Do you have a Winmodem, or an External Serial/USB Modem?

If the answer to Question #3 is Internal Winmodem you must start ay
[www.linmodems.org](www.linmodems.org), by downloading scanmodem and getting with Marvin
to try and make your modem work.

Or you could purchase a USB USRobotics modem from Walmart for $44.00,
or buy a used USRobotics 56K (V.92) serial modem from ebay and go that
route.

This should be of help. -  Post #4
[http://ubuntuforums.org/showthread.php?t=1100310](http://ubuntuforums.org/showthread.php?t=1100310)

I think that you will do fine if you follow the guide, and post back here
if you run into problems. Keep me updated if you find problem areas in
the guide.

Just remember setting up that Winmodem isn't going to be easy, and if you
attempt it you are likely to be at it for several days, so a $44.00
investment might be the cheaper way to go.  I've been there......

Once you have wvdial working, Gnome PPP will be a snap to set up, and it
works good.  (You might want to install it with Synaptic, and give it 
a try, but be warned it might work right until you have other stuff
set up prior to using it.  I have always set up wvdial first.)

Good Luck.
