Should be OK ... as soon as you install it.

More detail: Running from USB should be OK ... as long as you have a swap space for Ubuntu ... which you don't have on an Windows-only machine

So: 
1) install Ubuntu.
2) Remember that it won't as fast as Windows XP on that hardware. You have old hardware, and thus old WinXP (which has less features and thus less requirements than current Windows and Linux versions) is fast
