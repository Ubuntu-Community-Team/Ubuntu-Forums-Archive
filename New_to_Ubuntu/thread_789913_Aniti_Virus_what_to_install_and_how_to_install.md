### Post by Sef on 2008-05-11
An anti-virus is not needed because there are no viruses in the wild for Ubuntu.  The one exception is if you have a mail client, like Evolution or Thunderbird.  If you use an email client, then go to Applications > Add/Remove > Change box (Show) to 'All Available Applications' > Search: Clam > check clam > apply changes > apply > close.
