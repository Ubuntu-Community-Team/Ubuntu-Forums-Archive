### Post by RobGoss on 2017-01-08
Hello and welcome, I'm assuming you're trying to dual boot your system correct? if this is the case there are a number of things to consider. Providing the **specs **for your machine is a big help and will allow others to help you reach your goals using Ubuntu

You mentioned using a tool to burn your** ISO** file to a** DVD** which tool did you use?

Were you able to complete your installation of Ubuntu?

If possible I recommend using a** USB** thumb drive as your bootable device it's faster and works much better DVD's have been known to be problematic 


When dual booting you need to know which **BIOS** setting your Windows is installed in,** example** If Windows is installed in **UEFI **then you need to install Ubuntu in the same way **UEFI**, If Windows is installed in **Legacy** then Ubuntu needs to be installed in Legacy as well 

You will also have to access your BIOS and turn off **Secure boot** and disable **Fast startup,** in order to allow your machine to boot up correctly


Here are a few posts that I think will come in handy Ubuntu requires a lot of reading in order to understand the process of installing any Linux distribution

Dual booting Windows and Ubuntu infomation
[https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)

One of the best threads on Dual booting Windows and Ubuntu UEFI comes from a Member **Old Fred **he breaks it down so you have a idea what's involved
[https://ubuntuforums.org/showthread.php?t=2147295](https://ubuntuforums.org/showthread.php?t=2147295)
