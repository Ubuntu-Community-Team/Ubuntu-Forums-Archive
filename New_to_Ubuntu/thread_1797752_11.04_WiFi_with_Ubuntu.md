### Post by TBABill on 2011-07-05
The CD probably has only a couple utilities for helping you setup your wireless router, but you should be able to do the same from Ubuntu by going, in a browser, to URL 192.168.1.1 and logging in with whatever username and password you setup for your router (or whatever the defaults are if you never set it up). 
 
As far as configuring and using it, your concerns for whether you should and how you should use it would be the same whether running Windows, Mac, Linux, BSD, etc. Choose your comfortable level of security and enjoy.

### Post by grahammechanical on 2011-07-05
Have you ever accessed the router settings and disabled the wireless part of it? If not then to router should still be broadcasting. I am unclear what you are trying to do. Set up the router's connection to your ISP or make a wireless connection to the router from the computer?

If this second thing is what you want to do, then click on the network manager icon and see if the router's wireless network is listed. Is so, then select that network and when you are asked to authenticate put in the router wireless key. You will also need to set the encryption type so that the computer and the router are talking the same language. This information should have been given you when you got the router. It may also be on a sticky label on the bottom of the router.

If your wireless network is not in the list then go to Additional Drivers and see if there is a driver that you need to activate. Do so and then reboot and try the above again.

It would help if you told us what version of Ubuntu you are using. The look and location of menus has changed and you may get the wrong directions because we do not know what version you are using.

Regards.
