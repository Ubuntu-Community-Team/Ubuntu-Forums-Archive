### Post by coffeecat on 2013-11-08
Ubuntu 11.04 passed end-of-life over a year ago. I suggest you install a supported version, either the 12.04 LTS which will be supported until April 2017 or the latest 13.10, which has a shorter life and will become end-of-life in July 2014.

When a system reports an error, you need to quote that in your thread if you want help. Although installing 11.04 is not recommended, it might be useful to know what the error was.
