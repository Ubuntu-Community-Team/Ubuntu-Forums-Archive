### Post by drs305 on 2010-04-20
Yes, you can remove older kernels although it's advised to keep at least one older kernel which you know works.

There are several ways to remove older kernels (which will also remove them from your menu). The easiest is probably the GUI app Ubuntu-Tweak.

Check out the "Removing Older Kernels" section of the following post. It details how to install and run Ubuntu-Tweak, as well as providing other ways to deal with this issue.

[http://ubuntuforums.org/showthread.php?t=818177]("http://ubuntuforums.org/showthread.php?t=818177")
