### Post by Penguinnerd on 2012-05-04
Support for 10.10 has ended. That means that you're vulnerable to unpatched security holes and bugs that are constantly patched/fixed in supported versions. Running an unsupported version is not recommended.

For similar performance reasons, I am still using 10.04, which is still supported for about 1 year. That's one secure option. Another way to get decent performance is to upgrade to the latest release of Xubuntu or Lubuntu. 

Incidentally, upgrading to Xubuntu (or Lubuntu) 12.04 will get you the new software center as well as fast performance.

So that's my suggestion.

### Post by Penguinnerd on 2012-05-05
If you like the old netbook interface, you're out of luck. It got replaced by Unity in 10.10, and then they got rid of the netbook remix altogether for 11.04. 
They actually expect people to run Unity on their atom-powered netbooks. (Lol, just as soon as I install Windows on my TI-84!)

Most people (who still use netbooks) seem to just run a regular desktop on their netbooks. That's what I do. A fancy netbook launcher is a waste of ram/cpu in my opinion. It's faster without it.

My netbook currently has Debian Squeeze with LXDE. It's fast.

Do what you want to do. If you want something from 12.04, then install 12.04. If you don't like 12.04, use something else. You can't have it both ways.
