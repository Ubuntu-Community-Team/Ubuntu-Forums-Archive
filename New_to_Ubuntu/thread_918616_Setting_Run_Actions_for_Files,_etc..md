### Post by Pro-reason on 2008-09-13
OK, firstly, when you want to install software on Ubuntu, the first step is to check in the official repositories, via Synaptic.  Is there any particular reason why you're not using the *zsnes* emulator available in the repositories?

Secondly, if you download files from random websites, you need to tell us the exact address, so that we know what you are talking about.

I don't think that Fluxbuntu (with Fluxbox) is a good choice for a newbie.  It is too bare-bones.  You should be using Xubuntu (with Xfce) or standard Ubuntu (with GNOME).  That way, you could open .tar.gz files simply by clicking on them.

On your system, the best option to deal with that sort of file is to use the command line.

But I'm getting off-topic.  Stick to installing software in Synaptic until you are a very advanced user.
