### Post by 3rdalbum on 2012-06-06
> **raja.genupula said:**
> About Virus , Ubuntu is not completely immune to Virus but its can handle many virus's that are freaking windows

No Windows viruses work on Linux. There are no Linux viruses at the moment. That's as close to "completely immune" as you're going to get in the real world.

If your hardware works, then Ubuntu has detected it and is using it. You can have a look at the "Additional Drivers" program included with Ubuntu to see if there are any other drivers to use your hardware more efficiently, but chances are that everything is going fine as it is.

The Unity desktop included by default in Ubuntu is not especially flexible and customisable, but you can install other desktops using the Ubuntu Software Center program. Other desktops include LXDE, XFCE, Gnome and KDE. Once installed, the other desktops can be logged into at the login screen - click the little "cogwheel" icon on the login screen to choose a different desktop.

Unity is, to my mind, the best desktop available in Ubuntu. Gnome would be second best. Just understand that Unity and Gnome are very, very different to the Windows desktop you're used to. That doesn't make them worse.

### Post by nmyrick on 2012-06-06
I agree with Raja.Genupola, We need to think about malware and trojans.  Thats why I have installed ClamScan.  Once you install and configure it, it will run completely in the background.  Never even shows that it is running, except when it flashes by in about 1/2 second in the terminal before the OS boots up.

If you install the GUI interface called ClamTk, you can easily check the history on the scans, and run a scan from ClamTk. Scans can also be ran manually from the terminal with this command:

sudo clamscan -r -i

I've had this on my Ubuntu system for 4 years, and the only viruses I have ever picked up were executables written for Windows, which will do absolutely nothing in a Linux environment, and they are always in email attachments.  

Think of it this way, viruses look for certain files to infect that are supposed to be located in certain directories. Well, since the files, directories, and software are different in Linux, the viruses written for windows are stopped dead their tracks and do absolutely nothing. The only way you can pass one on to a Windows user is by forwarding an infected email.
