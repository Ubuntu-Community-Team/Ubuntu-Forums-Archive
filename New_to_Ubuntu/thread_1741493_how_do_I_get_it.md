### Post by Mike V on 2011-04-28
you can go here:

[http://releases.ubuntu.com/11.04/](http://releases.ubuntu.com/11.04/)

and download the iso image that best suit your architecture most likely;

[http://releases.ubuntu.com/11.04/ubuntu-11.04-desktop-i386.iso](http://releases.ubuntu.com/11.04/ubuntu-11.04-desktop-i386.iso)

if you are already running ubuntu you can upgrade from within:

```
sudo apt-get install update-manager-core
```

Then all you need to do is run

```
sudo do-release-upgrade
```

Regards
