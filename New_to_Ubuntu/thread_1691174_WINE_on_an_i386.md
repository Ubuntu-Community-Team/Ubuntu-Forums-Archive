While this is technically correct it would probably be more helpful to point
out that iTunes is not going to work in Wine. And even if you did, by some
miracle, get it working, Apple would promptly find a way to break it.

No big loss though, the purpose of iTunes is to make Apple money, the
purpose of Rhythmbox, Banshee, etc. is to manage and playback music libraries.
Sooner or later, the true purpose will manifest itself.

You don't have to use Ubuntu to use Free alternatives to Apps. On the
contrary, you should start your journey to freedom long before you actually
switch to Ubuntu. A user who relies on Internet Explorer, MS Office, iTunes,
etc. daily will not have a good Ubuntu experience.
