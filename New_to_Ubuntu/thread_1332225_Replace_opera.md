### Post by 3rdalbum on 2009-11-20
Is it where the download speed is low, or is it that pages take longer to start to load?

If it's the latter, then it's probably because Ubuntu uses IPv6 by default. Soon (next year, probably) everyone will have to use IPv6. The delay is caused by the system sending out a request with IPv6, finding that a computer or piece of hardware somewhere along the route doesn't support it, and then retrying with IPv4.

It is possible to switch Firefox back to IPv4 which is what Windows uses by default - there are lots of HOWTOs around on the web (google for "disable ipv6 firefox") which will stop the delay.
