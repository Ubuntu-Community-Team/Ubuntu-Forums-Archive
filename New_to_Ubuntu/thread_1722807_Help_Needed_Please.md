### Post by coffeecat on 2011-04-06
Hi

We can certainly help you with using the Ubuntu live CD, but this sounds like a Windows booting problem, even a system corruption. There are plenty of people using Windows here, but not so many with the deep experience of Windows needed to be able diagnose your problem and how to fix it.

The fact that you copied and renamed some files in C:\Windows\System32 before, but that only worked for a little time suggests you need to find out exactly what is wrong. And I don't think anyone here would advise you blind on what to copy or rename. There must be better ways of doing it.

What version of Windows are you using? If Vista or Windows 7, then you can download the ISO for recovery discs here:

[http://neosmart.net/blog/2008/windows-vista-recovery-disc-download/](http://neosmart.net/blog/2008/windows-vista-recovery-disc-download/)

There are utilities on those which are for fixing broken systems.

Do you get an error message? If so, have you googled the exact error message?

And so that we are clear: you are running Ubuntu from the live CD, but do you have Ubuntu installed on that laptop alongside Windows, or do you only have Windows on the laptop?
