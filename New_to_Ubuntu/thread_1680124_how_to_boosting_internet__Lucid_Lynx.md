### Post by Mark Phelps on 2011-02-02
> **Angel gel said:**
> My internet speed so slow... how can i speed it up..?;)

If you're asking if there is any specific to Ubuntu that will vastly increase the speed of your Internet access, then sorry, the answer is NO.

You can experience different real-time performance surfing different websites using different browsers.  For example, Chrome claims to be much faster than Firefox, which is generally much faster than Internet Explorer.

But in terms of overall speed improvement, there is no "miracle cure" in using a Linux-based solution.

If you want really faster connection, you have to purchase one through your ISP.
