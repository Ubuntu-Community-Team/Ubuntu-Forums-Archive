> would like to know if Ubuntu will detect and work properly with my computer hardware. Is there an updated list I can use to see if my hardware are supported?

The easiest way to check your hardware is go ahead and burn a disk. Ubuntu can boot up and run completely from the CD, so you can see for yourself if all the hardware is supported before you commit to installing the system.

> 
I also want to know how does Ubuntu and Windows differ from detecting and running hardware?


Drivers for Linux are generally bundled in with the main system (two notable exceptions are graphics cards and some brands of wifi). If your hardware is supported it will just work, no need to install drivers at all.

If your hardware isn't supported, the situation is pretty much the opposite end of the scale. Getting it to work even partially can be a complete nightmare and involve much fiddling about with the innards of the system.

However, if it's an older machine there's an excellent chance that everything on board will be completely supported and work with no hassle at all. Burn a CD and have a go.
