### Post by halitech on 2010-03-10
Ubuntu and Mint are different distros, not the same :)

Use your Windows install cd and repair the MBR then use Windows Disk Manager and format the partition.

---

### Post by Desert Sailor on 2010-03-10
But if you do as above, any work you did in your Linux Mint partition will be lost.  You can copy documents and music from Linux to Windows if you are set up to mount the NTFS partition and know where the "My Documents" folder is.

### Post by empty_spaces on 2010-03-10
