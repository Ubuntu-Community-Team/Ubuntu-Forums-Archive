### Post by x C0MMAND0 x on 2009-10-08
The only 2 drivers I have every had to install for my system are

1. Driver for my Nvida card.  Now, it had one that was working but I just updated it to a better version for better 3d support

2. Wireless network driver.  I had to install this for my wireless card to work.  It was a 3 click and reboot process.  I just went to System > Administration > Hardware Drivers, clicked on the wireless and clicked install (okay, so 5 clicks).  I had to be hooked in via direct connection for this to work.

Everything else works great.
