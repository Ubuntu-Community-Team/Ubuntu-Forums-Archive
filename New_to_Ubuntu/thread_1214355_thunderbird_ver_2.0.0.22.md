### Post by LewRockwell on 2009-07-21
I don't use an address book myself, but my students sometimes ask about duplicate entries in theirs

I've looked at them and I've noticed that it seems to be generated from the "other end"

in other words, if you allow the automatic entries into your address book and the original sender changes ANYTHING from one email to the next...you're going to end up with a duplicate

of course, it's not a "duplicate" to the software...only a "similar" entry(even if it's only one characters difference)

perhaps that will help
