### Post by mikechant on 2009-06-30
> ...but where can one find more info on how to use UNR specifically?

UNR is basically just a different appearance for Ubuntu to accomodate smaller screens. Almost anything that works on Ubuntu will work on UNR - applications, terminal commands, config file changes. You just get the application/system menus and the task bar presented differently and some other presentation differences like some windows being resized to fit the screen automatically.
