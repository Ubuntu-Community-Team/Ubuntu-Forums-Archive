### Post by zeroseven0183 on 2009-02-26
Hi! I'm also a rookie here.

You can download updates for Clam by:
 1. typing **gksudo clamtk** on the terminal
   Ubuntu will then ask for the root password
 2. once the ClamTk Virus Scanner window opens, click on **Help**
 3. select **Update Signatures**
   Or you can do it with shortcut **CTRL + U**

I hope that helps.
