### Post by mlentink on 2008-05-29
It&#347; a file that Windows uses to present the contents of that folder to you in a certain way. A config-file in a sense. Obviously, the Windows programmers consider such config-files to be way over your head, and thus make them invisable to you lest you wreck your system...

### Post by Shazaam on 2008-05-29
Basically it is a configuration file. And it is a hidden folder in Windows. If you copy/pasted the contents of the Windows folder to a folder in Ubuntu it is safe to delete from the Ubuntu folder.
