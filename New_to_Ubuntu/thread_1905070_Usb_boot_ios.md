### Post by Jlb181 on 2012-01-06
I think you would have to install the OS to the stick.  I've done that.  Unplugged my hard drive so the boot loader was installed onto the the stick, then installed Ubuntu without a swap file (they are hard on memory sticks) from a CD.  After that if you have room you should be able to install any other OS that will run off of a stick.  When you start up the stick the boot loader will ask which OS you want to start.

I only installed Ubuntu with this method, but it worked and I used it every night at work for a long time.
