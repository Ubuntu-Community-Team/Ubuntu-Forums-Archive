I am using Ubuntu 11.04, 32-bit, in a GNOME environment.

It is my understanding that Ubuntu's update manager does not keep track of new releases of installed software not provided by their repository.

Just out of curiosity --other than keeping a manual list of such installed software and periodically scouring the Internet for new releases/patches-- is there an easier way of finding out when a new release/patch is available?

As a matter of comparison, in the Microsoft Windows world, there are programs such as FileHippo's Update Checker or Secunia PSI that automatically keep track of new software releases.  (I concede that they do not keep track of releases for every existing software!)

### Post by westie457 on 2011-09-29
Hi from my understanding of Update Manager and Ubuntu, software in the repositories is updated automatically and only provides updates for your particular machine and its set up. Unofficial software will only be updated when the authors write and release newer versions. Update Manager will only find those updates if the source PPA is in your sources list, otherwise you will have to track them down your self.
