### Post by markbuntu on 2008-07-17
If you are unhappy with the restricted driver from the repository, you can get the latest 8.6 driver from ATI. This one for sure supports your card and has many enhancements and fixes over the 8.47.3 driver.

 Follow these directions carefully, use Method 2:

[http://wiki.cchtml.com/index.php/Ubuntu_Hardy_Installation_Guide](http://wiki.cchtml.com/index.php/Ubuntu_Hardy_Installation_Guide)

If you want to tweak the driver for better performance:

[http://forum.compiz-fusion.org/showthread.php?t=6794](http://forum.compiz-fusion.org/showthread.php?t=6794)

If you want to read the release notes:

[https://a248.e.akamai.net/f/674/9206/0/www2.ati.com/drivers/linux/catalyst_86_linux.html](https://a248.e.akamai.net/f/674/9206/0/www2.ati.com/drivers/linux/catalyst_86_linux.html)
