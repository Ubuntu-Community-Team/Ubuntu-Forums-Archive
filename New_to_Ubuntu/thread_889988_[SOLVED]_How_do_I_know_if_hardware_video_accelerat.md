### Post by dannyboy79 on 2008-08-14
also, make sure you're using the intel video driver. You can find out by issuing this command

cat /etc/X11/xorg.conf | grep Driver

if it returns

Driver         "intel"

you should be using the intel driver which should be giving you decent graphics. Also, there's a way to tell if DRI is enabled but I forget off the top of my head sorry.
