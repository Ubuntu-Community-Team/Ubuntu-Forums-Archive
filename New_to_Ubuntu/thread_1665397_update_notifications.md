Update manager popped up a wee bit ago, said it had some things to install so I went ahead, allowed the download and entered my password. The downloads started and I shifted focus to a different screen.
A little later I went back to update manager it says that it updated the files an hour ago. So is it being literal saying that I'm GTG or is it implying the updates already took place without my authorization?

### Post by Frogs Hair on 2011-01-12
GTG ? You gave Authorization to download and install updates by entering your password.

If the update manager is set to notify only (Update Manager > Settings) it will ask for a password before installation.
