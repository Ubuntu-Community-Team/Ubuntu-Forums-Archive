### Post by kajoky on 2014-01-31
You should have a Downloads folder.  The full path is /home/yourusername/Downloads/
In Nautilus, the file manager, you can add a folder by either right-click in the window and select 'New Folder' or on the keyboard by pressing Shift+Ctrl+n.
Your browser should have Preferences where you can designate the folder to use for downloads.

### Post by novice_penguin on 2014-01-31
If it is an executable file, then when you download from the internet it will give you the option to download where you choose.  If it is a file, it will (by default) download to your user's download directory.  At least that's how it works in Ubuntu 12.04.  Hope this helps.
