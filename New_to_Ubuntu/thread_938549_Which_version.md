### Post by ibuclaw on 2008-10-05
Most proprietary software (most notably games, ie: Uplink) for Linux is compiled for 32bit, so you won't be able to run them unless you hunt down the missing 32bit libraries. Which can be cumbersome if you didn't get the software from an official repository.

Other than that, there is no difference.

I primarily use 64bit machines, so I would encourage you to join us :)
