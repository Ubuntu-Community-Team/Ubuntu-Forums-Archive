### Post by LewRockwell on 2009-10-25
Update:

We have been experimenting with a generic 1GB SD Card which had been goofed up somehow previously

We found no native Windows or Ubuntu tools which would correct this card
(Synaptic lists "gnome-format" as a tool that can format mmc/sdc cards but it didn't work for us)

We did find a utility called "SD Formatter" from Panasonic that worked for us

Here is the link:

[http://panasonic.jp/support/global/cs/sd/download/sd_formatter.html](http://panasonic.jp/support/global/cs/sd/download/sd_formatter.html)

make sure after you download it that you check the exact file size before installing it just to make sure it is correct
