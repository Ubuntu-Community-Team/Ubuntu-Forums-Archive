### Post by Old_Man_Unix on 2010-10-04
The Brother printer drivers are not included in the Ubuntu distribution because of licensing issues.  However, you can install them from the Ubuntu *multiverse* repository.  There are many Brother driver "packages" in that repository and you install the ones you need for your particular printer. 

You really must read up on how to install software in Ubuntu.  This really isn't the place for a tutorial on that.  It isn't difficult at all,  but you do need to know how to do it.

Here is the Wiki entry on the brother drivers.  I would not expect you to understand what it means until you know how to install software.  Once you know that, the Wiki will become clear.
[https://wiki.ubuntu.com/BrotherDriverPackaging](https://wiki.ubuntu.com/BrotherDriverPackaging)
