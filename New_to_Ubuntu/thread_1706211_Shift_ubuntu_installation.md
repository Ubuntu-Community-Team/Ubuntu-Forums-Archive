### Post by bcbc on 2011-03-13
> Requirements
LVPM has been tested on installs created by Wubi 8.04, Wubi 7.10, Wubi 7.04, and Lubi 7.04.
LVPM currently does not work with installs generated by Wubi 10.04 (patches welcome). If you wish to do a standard install of Ubuntu without a CD, please use UNetbootin instead

Try [http://ubuntuforums.org/showthread.php?t=1519354](http://ubuntuforums.org/showthread.php?t=1519354)
