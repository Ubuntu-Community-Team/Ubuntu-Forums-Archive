### Post by jerrrys on 2012-10-16
Go to your local thrift store and pick up a wired usb keyboard for a couple of bucks.  The only problems I have ran across is with specially keys; they usually have to be programed in or just plain won't work.

Wireless keyboards also work, but can give you problems when installing ubuntu.  Some require drivers to be installed, which is much easier to do after the install.  Also wireless keyboards will usually state their linux compatibility.

### Post by grahammechanical on 2012-10-17
You should not have any problems using a USB keyboard. During the installation of Ubuntu you will be asked to select a language and a keyboard layout. A variety of layouts will be on offer, including layouts that fit the Microsoft type keyboard.

You can add and remove keyboard layout after installation by going to System Settings.
