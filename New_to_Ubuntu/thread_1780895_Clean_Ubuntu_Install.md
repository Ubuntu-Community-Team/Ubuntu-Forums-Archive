### Post by duke.tim on 2011-06-12
You should have the option to reformat when you put the ubuntu disc into the computer. When you get to the partitioning options you will need to select advanced options 

(or some similar name, I think they are the same in the different versions of Ubuntu...can't 100% remember)

And select the format check box. The link below has nice pictures :D
