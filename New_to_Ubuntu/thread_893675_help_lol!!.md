### Post by Tomosaur on 2008-08-18
> **Nicholas1212 said:**
> ok well tomorrow im getting ubuntu and i already have windows and first of all would like to know how or if its easy to install ubuntu onto a windows computer?

Ubuntu comes on a Live-CD which means you can try it out without having to install it (this is good for checking whether all of your hardware works under Ubuntu).

Be warned though, the Live-CD is pretty slow to load (Ubuntu, when installed, is actually very fast).

Depending on which version you're getting, you might have the option of using Wubi - which means you can install Ubuntu 'inside Windows' as if it was a Windows program. This is very useful if this is your absolute first experience with Linux and Ubuntu.

Ubuntu can also be installed alongside Windows (this is called a 'dual boot' system). This is what most people do if they still need to use Windows for certain things.

In the future try to give a thread title which lets us know about your problem - lots of people just write 'HELP!'. If you give a thread title which tells us the general area at least of the problem you're having, you'll get a lot more people able to give replies.



EDIT: As for finding programs to use - all you have to do is click the 'Applications' menu, then 'Add / Remove'. This launches the **package manager**. Here you can find new software to install - all you have to do is click the tick-box next to the ones you want, then click 'Apply Changes'. It will then ask you for your password, and once you've typed that in, it'll go and download and install the software for you. Easy!

Pretty much all of those have equivalents in Ubuntu, they are called:

Evolution (Outlook)
OpenOffice (Word/Office)
Pidgin (MSN/YAHOO/ICQ/GoogleTalk/etc)

Programs in general are installed using the command line or graphical user interface (GUI) programs. The command line program is called "aptitude". One GUI program is accessed by going to the Applications menu and selecting "Add/Remove..." and the other by going to the System menu and selecting Administration->Synaptic Package Manager.

More information on all of these programs should already be in the documentation for Ubuntu and scattered all over the forums here.

Welcome to Ubuntu and don't ever hesitate to ask any question! Although use the search function before posting, because most of the time, somebody already asked the same question you're about to ask. Especially as a newly-arrived. :-)
