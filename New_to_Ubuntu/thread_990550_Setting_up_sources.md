### Post by SuperSonic4 on 2008-11-22
since ubuntu and kubuntu both use the same repos you can still open the /etc/apt/sources.list

Open it using ```
kdesu kate /etc/apt/sources.list
``` and add in that like you would in ubuntu.

Adept is kubuntu's equivalent to ubuntu's Synaptic
