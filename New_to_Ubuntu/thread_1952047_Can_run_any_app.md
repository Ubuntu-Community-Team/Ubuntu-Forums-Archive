### Post by forrestcupp on 2012-04-03
Most apps you download are for Windows. Ubuntu is not Windows, and it will not run Windows apps. A lot of apps have versions that are made for Linux/Ubuntu or there may be good alternatives that are native Linux apps. But you can't just download any app on the internet and expect it to run. It's kind of like how Macs won't run Windows apps; they have to run Mac apps. Ubuntu has to run Linux apps.

We do have something called Wine that you can install in Ubuntu that is a Windows compatibility layer. But everything they have done has been reverse engineered, and you can't expect it to be able to run every single thing flawlessly. There is a lot Wine will run perfectly, but there is also a lot it won't run at all.

If you can check out Ubuntu, through a LiveCD or something, have a look at the Software Center for thousands of apps that you can automatically download and install.
