### Post by fluxlizard on 2009-09-22
I have an old one with those specs, but a little less ram. Last year I programmed a website with bluefish, had firefox open to check the changes in bluefish, had gimp open working on graphics for the site, and had tunapie running internet radio in the background all at the same time without slowdown.

I was using pcfluxboxos. If fluxbox isn't your thing, you can install icewm from the repos once you are up and running. I like fluxbox, but it's a little more work- I have a panel with a start menu, quick launcher, desktop switcher and clock at the bottom of the screen and an icon dock with launchers near the top.  It's a full featured os and will run the latest version of popular softwares on p2 no problem.
