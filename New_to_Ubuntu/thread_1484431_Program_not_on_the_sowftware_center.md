### Post by mcduck on 2010-05-15
use Synaptic Package Manager (in the System/Administration-menu).

The software Center only contains a pre-defined selection of packages, while Synpatic is the full-blown package management tool that lists everything available and all the 3rd-party packages you might have installed.
