### Post by locolomo on 2010-09-05
Hi, been using Linux/Ubuntu for about 2 hours now!

This afternoon I installed Ubuntu, setting up my partner as initial user, then set myself up as a user with admin status.  Set up internet via cable then was prompted to install the drivers for wireless, now recognising wireless no problem.  Now I can connect to the internet, chat etc, but my partner can't.  He has wireless connection but when attempted to ping -c3 4.2.2.1 he was told he had no internet connection.

We also have Vista installed - any other info needed?

### Post by redbook4574 on 2010-09-05
right click on your network icon, select edit connections and ensure that you have ticked the box "available to all users" in your wireless connection

---

### Post by gandaran on 2010-09-05
look up the corresponding working setup (wireless or cable)  in network manager and mark the box for all users so every user will have access to internet.
