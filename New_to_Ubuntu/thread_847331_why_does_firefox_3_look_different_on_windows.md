### Post by cariboo on 2008-07-02
Firefox does not natively scan for viruses on download. That is a function of the virus scanner you have installed in Windows. I have never seen a Windows virus that will work in Linux, so there is no need for it to scan for viruses. If your are downloading Windows executables from shady sources and you suspect there might be a chance of a virus, there are many virus scanners for Linux. Search Add/Remove Programs, Synaptic Package Manager or Google for linux virus scanners.
