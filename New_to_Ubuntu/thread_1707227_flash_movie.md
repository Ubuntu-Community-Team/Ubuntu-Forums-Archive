### Post by Dutch70 on 2011-03-15
> **expatCM said:**
> nice idea :)   however, I think the inhibit applet is for laptop users who wish to let the system run on full power rather than automatically go into various power save profiles.

Totem Movie Player does that by default, if you haven't changed the settings on it.
to check it go to >edit>>preferences>>display>> and check "disable screensaver when playing. You can either tick "video" or "video & audio".

You can set Mplayer to do it by going to >edit>>preferences>>interface tab & tick "use X Screen Saver control over Gnome Power Manager"

VLC is supposed to be able to do it also, and I know it used to, but I can't find the setting for it now.
