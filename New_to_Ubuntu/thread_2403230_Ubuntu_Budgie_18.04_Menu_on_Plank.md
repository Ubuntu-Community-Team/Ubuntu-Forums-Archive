### Post by Frogs Hair on 2018-10-09
The applets in Budgie settings are only applicable to the panels. It's possible to create a dock like panel that is not fully extended across the screen. Plank is primarily for frequently used applications. I created the pictured dock using the menu applet, dock mode , transparency , and the icon task list. When applications are open right click and select pin. There will be a dot to indicate running applications.
