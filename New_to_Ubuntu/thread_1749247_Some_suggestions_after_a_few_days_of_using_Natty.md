### Post by ngubk on 2011-05-04
At first! I gotta say, Natty with Unity is much worse than Maverick with Gnome. 
 + Dashboard is worse than Gnome-do
 + Left panel is worse than Docky.
 + It doesn't seem right without gnome-panel ( lacks a lot of Indicator...)
However, it'll definitely the change for better, since Unity is just in its early stage.

After a numerous time of making my system crash, i have some suggestions for anyone who is using Natty:

1/ Compiz setting: i highly recommend that you shouldn't change too many things, it just makes it worse 'cause Unity is not very compatible with Compiz. Some changes could be made are:
  + Opacity
  + Animation
  + Wobbly windows
  + Application switcher (not Shift/Ring switcher)
  + Don't make change on Utilitiy.
  + On OpenGL, disable Sync to Vblank, it could stop some crashs.
  + On Viewport switcher, you can set Button 5/4 to Move Next/Previous
  
2/ Here is a super useful link [http://www.webupd8.org/2011/04/things-to-tweak-fix-after-installing.html](http://www.webupd8.org/2011/04/things-to-tweak-fix-after-installing.html)
that helps you tweak Natty without crashing your system( and easy to do too):
 + Recover Notification Area.
 + Config Left Panel (add Show Destop, remove mounted volume...)
 + Make changes using Dconf-editor (not gconf-editor)
 + etc...

I've been using Ubuntu for 2 years but i'm not familiar with this forum yet so i'm very sorry if i post this to the wrong place.
This is my personal opinion so it may be wrong. Please give me your feedbacks so i can continue to contribute to the forum
Thank you very much.
