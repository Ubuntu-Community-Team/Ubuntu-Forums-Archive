### Post by HotShotDJ on 2008-05-12
> **Zeck said:**
> My question is how can i create .deb package and how to install it ? Of course using synaptic. Does somebody know ?Just open Synaptic Package Manager in the System Menu, search for the application you want, select it, and then install it.
[LIST=1]
[*]System --> Administration --> Synaptic Package Manager
[*]Click on "Search" in the tool bar.
[*]Type in a search term, for example, php5
[*]Scroll through the available choices.
[*]Find the package you want
[*]Click on the square selection box next to the desired package
[*]Verify selection and dependencies then click Mark
[*]Click Apply in the tool bar
[*]Confirm by clicking Apply in the new dialog box
[*]Be amazed as Synaptic installs your package and resolves all dependencies automagically
[/LIST]
