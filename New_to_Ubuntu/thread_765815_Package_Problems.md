### Post by otrojake on 2008-04-24
The servers are getting hammered today with the release of Hardy.  You might want to change to another set of servers.  Go to System-->Administration-->Software Sources.  Under the "Ubuntu Software" tab click on the "Download From" dropdown box and click "other."  In the box that appears click "Select Best Server."  Wait until the tests are done and click "Choose server."  That should help out for the time being.  Afterwards you'll probably want to change it back to the Main Server option.
