### Post by Martin Marshalek on 2009-01-03
If you're trying to install a GDM (the login window) theme do this (this is the first choice): 

Go to System---->Administration---->Login Window then select the "Local" tab and then select "Add." From there, locate your zip file and select the new theme.

If you're trying to install a GTK theme (the theme for your windows and buttons and everything while logged in) do this (this is the second choice):

Go to System---->Preferences---->Appearance and then drag the tar.gz file (without extracting) into the window. Then hit save as [to save the theme (I'm not sure if you have to do this)] and then you're done.

If you're trying to install the icons (the last choice) repeat the above steps except the save as part. If you want to have the icons as part of the theme do this:

Go to System---->Preferences---->Appearance and select "Customize." Then go to the icons tab and select the new icons from the list. Then select "Close" and then select "Save As." Rename the new theme the same as the original theme to overwrite it.
