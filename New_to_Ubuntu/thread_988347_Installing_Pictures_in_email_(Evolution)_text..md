### Post by Tamlynmac on 2008-11-20
Open Evolution left click on "New" in the tool bar.

Anywhere you wish to install the image - place your mouse cursor in that spot and left click on "Insert" up in the tool bar. Then select "Image" from the list. The file browser will open and the only thing you need to do is select the image file you wish to insert. Either double click on the file or highlight the file (single left click) and click on the "Open Button" at the bottom of screen.

Hope this helps you - should you have more questions please feel free to ask. Age has nothing to do with Ubuntu and we will gladly continue to assist you until it works. Everyone can use Ubuntu and get help on the forums.
