### Post by Therion on 2009-04-17
1.  Your CPU supports 64-bit.

2.  Is [64-bit] worth it?  Well, look at it from the reverse:  Why *wouldn't you* use 64-bit if your system supports it?  I've been using 64-bit distros since Gutsy and I've never had a problem.  Zero, zip, nada.  Not an issue.

Some time ago there were workarounds you had to use regarding Flash, but even that has native 64-bit support now.  I can't think of a single reason why anyone who's system supports 64-bit computing would want to use anything else.

### Post by roger_1960 on 2009-04-17
Hi

It is 64 bit capable.  It would seem best to use 64bit to make the most of the chip and RAM. (I think you will only be using it as 2 core anyway).  There are a very few programs which run on 32 bit but not on 64bit (ie Stellarium) but this is unlikely to affect you.  Its best just to check that for any unusual "must have" software by searching the forums.
