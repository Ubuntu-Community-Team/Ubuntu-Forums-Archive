### Post by 3rdalbum on 2010-05-30
1. How are you connecting the TV to the computer? If you connect via a regular TV input such as S-Video, Composite (little round yellow plug) or Component (red, blue and green plugs) you will not get widescreen.

If you have an LCD or plasma, you should use the DVI, HDMI or VGA inputs on the TV hooked up to the same port on your graphics card. Try DVI first, then HDMI etc. You should get widescreen then (you may need to boot up the computer with the TV connected).

2. What graphics card is it? Did you get any error messages when you tried installing the driver? Did you install the driver through the Hardware Drivers program in Ubuntu, or by downloading the installer and running it manually?
