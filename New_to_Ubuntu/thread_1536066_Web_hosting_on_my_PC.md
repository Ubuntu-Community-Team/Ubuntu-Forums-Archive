### Post by Paqman on 2010-07-21
What sort of sites are you making? If it's all just plain HTML/CSS then all you need is a browser to render the pages you make. If you're using PHP and databases you'll need to [install the LAMP stack]("https://help.ubuntu.com/community/ApacheMySQLPHP") (**L**inux, **A**pache, **M**ySQL, **P**HP).
