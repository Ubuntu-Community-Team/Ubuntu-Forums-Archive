### Post by Cheezespread on 2009-09-09
If you are sure the same has been installed , you can add an entry for Virtualbox under applications with the following steps.

1.Right-click on Applications and select Edit Menus
2.select System Tools and then click on New Item from the options available.
3.It will give you a popup " Create launcher ".Add the following details.

Type: Application 
Name: VirtualBox 
Command: VirtualBox

Close it and you will find the same appearing under 
Applications->System tools

Please correct me if i am wrong.
