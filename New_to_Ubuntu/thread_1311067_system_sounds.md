### Post by philinux on 2009-11-02
Until the new gdm has a gui like jaunty then I think you'll have to change the actual file itself.

/usr/share/sounds/ubuntu/stereo/desktop-login.ogg

You'll need to rename the above file as .backup say and then substitute your own sound file.
