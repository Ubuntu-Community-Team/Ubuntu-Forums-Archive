You really do not need any anti-virus with Ubuntu. There is "clamav" in Synaptic that you can down load. I would also recommend "clamtk" which is the graphical interface for it.

If you have installed the 64 bit Ubuntu then you may be limited on other AV programs.

To get to Synaptic Package Manager click on System/Administration/Synaptic Package Manager and select what program(s) you find.
