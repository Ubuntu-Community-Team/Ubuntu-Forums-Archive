### Post by compwiz18 on 2008-05-17
It allows you to run another operating system on your computer at the same time as Ubuntu. For instance, you could run Windows XP inside of Ubuntu so you wouldn't have to reboot to use a Windows application.

### Post by exile on 2008-05-17
It is virtualization software kinda like VMware. You can install different "guests". A Guest can be pretty much any OS these days from any flavor of linux to windows etc.

It is great for making test environments to play with new/different distro's.

see [http://en.wikipedia.org/wiki/Virtualbox](http://en.wikipedia.org/wiki/Virtualbox)
