### Post by fiyer on 2010-09-04
There may be a better way than what I will suggest:


1.) Use the select color tool, in my version of Gimp, this is the 5th icon on the first row (alongside the rectangle and oval select tools).  
2.) Click on the outline color you want.  This selects it. 
3.) Go to the Select menu and click "Invert" which then selects everything but that color. 
4.) Then delete.  You are left with the outlines.

It is somewhat imperfect however, and you'll have to do some fine-tuning with the image.

### Post by fiyer on 2010-09-04
There is the free-select tool, which looks like a lasso. I've done that before, but it takes painstaking effort and very good hand-eye coordination. It doesn't auto-select for you, you have to be spot-on in your selection. Just zoom in like crazy and work on the picture piece by piece.

I'm more of a recreational user myself, so there is likely an easier way to do these things. 

You might try getting [Paint Shop Pro to work in Wine]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=2505") if you can't get the same functionality out of Gimp.
