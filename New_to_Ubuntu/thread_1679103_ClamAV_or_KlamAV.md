### Post by coffeecat on 2011-01-31
@Tuxdroid, are you running Ubuntu with gnome or Kubuntu with kde? clamav by itself does not have a GUI app - it's command line only. klamav is the  GUI frontend for clamav for  Kubuntu/KDE. It (klamav) will run in gnome but from what I remember in earlier versions of Ubuntu, you had to add a gnome menu entry yourself. This may have changed - I don't know.

If you are running Ubuntu/gnome, you would be better off with the clamtk GUI frontend. You can install clamtk with the package manager - it's in the repository.
