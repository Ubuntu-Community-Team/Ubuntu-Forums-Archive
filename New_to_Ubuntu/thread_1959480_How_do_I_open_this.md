### Post by newbie-user on 2012-04-16
Ubuntu uses the Unity interface, so it's now somewhat similar to the Mac interface. There's a dock on the left side of the screen. If you click on the top-most button or hit the super key, it will bring up a search box where you can search for mahjongg. You could also follow the clickable icons in the search window to go through the different categories of apps. The interface is just different. You have to take some time to learn it, just like every other new interface.

As for the tar.bz2, that's a compressed file, just like .zip. You can uncompress it by right-clicking and extracting the contents. A compressed file is not an installation file and you may have to follow some instructions to install whatever app it is.
