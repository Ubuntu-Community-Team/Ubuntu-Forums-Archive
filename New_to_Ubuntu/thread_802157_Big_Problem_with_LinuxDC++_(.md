### Post by Trail on 2008-05-21
Linuxdcpp works properly for me :|

In any case, you can perfectly use the windows client under "emulation" with wine. There should be plenty of documentation if you search how to install dc++ under wine. A short version of this follows:

To install Wine (if you haven't) open a terminal and type:

```
 sudo aptitude install wine 
```

Then download the .exe installer and right-click open it with Wine. It should install normally. Navigate to the installed directory (~/.wine/drive_c/Program Files/DC++), locate the DC++.exe and execute it with Wine again. It should work.

I, myself, use ApexDC++ by the way; it is a quite improved version of dc++, you might be interested. And works perfectly under Wine of course.
