### Post by iaculallad on 2008-07-09
Maybe you have a shared memory setup (w/ internal video card) so it warned you of having a lower memory than the expected 256 memory. Here's Ubuntu Community [page]("https://help.ubuntu.com/community/Installation/SystemRequirements") that shows the minimum system requirement for installing Ubuntu.

With that system RAM available, try installing (x)Ubuntu instead.

If you had the minimum requirement, you're Ubuntu installation would have been successful, directly booting you to the GUI.

BTW, the command **startx** would bring you to the GUI when boot drops you to using terminal.

Suggestion: Try installing the (x)Ubuntu version or try adding more of your RAM.
