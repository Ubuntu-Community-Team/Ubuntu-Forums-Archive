### Post by NCLI on 2010-02-06
Try right-clicking the file, choose "Properties," go to "Permissions," and make sure "Allow executing file as program" is ticked, then try double-clicking the file.

EDIT: Nevermind, just go to the Software Center, make sure Sun Java 6 is installed, then right-click the file and choose "Open with Sun Java 6 Runtime."
