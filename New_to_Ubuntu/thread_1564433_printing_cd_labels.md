### Post by Mark Phelps on 2010-08-30
To create labels, check out GLabels.

Don't use that, so I don't know if it will print directly to CD.

To print to CD with my Epson printer, I do the following:
1) Create a label template using GIMP
2) Add the different layers to the GIMP template as needed
3) Print directly to the CD using GIMP-Gutenprint.

The process of doing this is rather complex, very detailed, and differs by printer make and model.

Sorry, but I don't have the time for writing tutorials, so I'm not up to step-by-step instructions.

If you do some Google searching on printing CDs using GIMP, you will find any number of instructions sets.  You will need to experiment with the instructions, your templates, and your printer settings to get everything working.  A good way to do this (without going through LOTS of CDs) is to cut out circular patterns from paper the size of CDs and print to those.  Believe me, there is a lot of trial and error involved.

It's not a simple process.
