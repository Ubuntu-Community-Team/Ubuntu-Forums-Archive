### Post by ibuclaw on 2008-07-09
OK, I've had a good read through, and looks like it is option 2. (Edit the script to exit graciously)

If the error message you've given is accurate, then this should hopefully fix it

```
sudo sed -i '/atieventsd/s/$?/0/' /var/lib/dpkg/info/xorg-driver-fglrx.postrm
echo "exit 0" | sudo tee -a /var/lib/dpkg/info/xorg-driver-fglrx.postrm
```
Else, apt was lying and the script ended unexpectedly in the prerm script.

Regards
Iain

[EDIT]
After apply, you may "**sudo apt-get remove xorg-driver-fglrx**" again.
