### Post by Mark Phelps on 2012-02-27
Recent Ubuntu versions suffer from a kernel bug that can cause the CPU to run at nearly full power all of the time.  This causes overheating and will drain your battery a lost faster than will Win7.

Info on this is linked below:

[https://bugs.launchpad.net/ubuntu/+bug/901232]("https://bugs.launchpad.net/ubuntu/+bug/901232")

As to video drivers, if you're seeing a desktop, the correct video drivers are already installed.

As to VMWare, most folks here use Virtualbox, so you'll have problems getting answers ... but you need to start a new thread just for your VMWare problem.
