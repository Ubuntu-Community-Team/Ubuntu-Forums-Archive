### Post by anewguy on 2010-05-30
The first 2 options - Gnome and KDE - are for the desktop managers.  By default, Ubuntu installs with the Gnome desktop manager, Kubuntu installs with the KDE desktop manager.  It should be noted you can have multiple desktop managers installed - you just chose which you want to use via "session".  If you haven't tried KDE (or perhaps you installed KDE and haven't tried Gnome), just select either from the menu and try it - you won't hurt anything!

As for the last 2, to be honest I'm not sure what they do.  I would assume xterm opens an x terminal versus a regular desktop, Failsafe Gnome I don't know (but the name implies it will at least be the Gnome desktop).
