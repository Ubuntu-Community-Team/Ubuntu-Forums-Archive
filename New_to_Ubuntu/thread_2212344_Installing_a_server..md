### Post by buzzingrobot on 2014-03-20
Remember, if you want the web site to be publicly accessible -- not just on your own internal network, you will need to host the site on a publicly accessible server.  Probably register a domain, too.  This points you to the world of VPN's.

The music, videos, can all be hosted locally, as can the web testing bit, as long you have no need to get at those files from somewhere else.
