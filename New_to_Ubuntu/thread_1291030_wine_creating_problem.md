### Post by Mark Phelps on 2009-10-14
If you're going to use Wine, you need to check the CodeWeavers application compatibility site linked below before asking questions about why an app doesn't work.  If by "gtalk", you meant Google Talk, had you checked the ratings, you would have seen it's rated "garbage" -- meaning, it's NOT going to work. 

CodeWeavers link:  [http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true]("http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true")
