### Post by thatguruguy on 2011-09-25
You don't need Windows to run Wine.  Wine enables you to run certain Windows programs on Linux (as well as other OSes). But Wine isn't perfect, and can't run every single Windows program flawlessly.

You can get an idea of how well various Windows programs run on Wine [here]("http://appdb.winehq.org/").
