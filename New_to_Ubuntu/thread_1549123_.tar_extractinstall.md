### Post by Bachstelze on 2010-08-10
> **Super_Dork_42 said:**
> 
Also, this thread is also more general about how to install from a downloaded .tar file.

There is no general method. A .tar file, just like a .zip or .rar file, is just an archive that contains other files. How you "install" from it depends on what it contains.

As for the screensaver you want, it was apparently removed in recent versions of xscreensaver.  You can try to:

-> Download the package for the old version, at [http://packages.ubuntu.com/karmic/xscreensaver-data-extra](http://packages.ubuntu.com/karmic/xscreensaver-data-extra)

-> Extract it

```
mkdir foo
dpkg -x *.deb foo
```

-> Copy the relevant files over to your current installation, and pray that it works.  This screensaver was probably removed for a reason, though.

*EDIT:* according to the [changelog]("http://www.jwz.org/xscreensaver/changelog.html") for version 5.09, juggler was merged with juggler3D. juggle3D is in xscreensaver-gl-extras.
