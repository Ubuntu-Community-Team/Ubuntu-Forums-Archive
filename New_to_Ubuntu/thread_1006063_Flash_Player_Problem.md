### Post by DaveyG on 2008-12-09
For me, flashplugin-nonfree (the one from synaptic) goes nuts too..

I don't know if this will solve it, but i downloaded Flash Player fromthe Adobe website: [http://get.adobe.com/flashplayer/](http://get.adobe.com/flashplayer/)

Ubninstall the flashplugin-nonfree from Synaptic Package Manager, Download flash from Adobe.com, Extract and Double-click flashplayer-installer, which should open up in a Terminal Window. Follow those instructions to install it, be sure to close firefox before you install. Otherwise it might go horribly wrong :)
