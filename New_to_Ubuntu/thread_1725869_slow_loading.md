### Post by fabricator4 on 2011-04-10
> **squeakerz said:**
> Ubuntu 10.04.  Everything is generally fast until I restart then Both online and off everything loads slow for about two hours before it  is fast again.  This did not used to happen but may have started after an update a while back.
Anyone know how to fix in beginner talk?

After a reboot open system monitor (in System -> Administration) and tell us what the CPU load and the memory are doing.  Also note if there's any unusual network activity and if there's a lot of hard drive access.

It the CPU is loaded down, open the processes tab in system monitor and sort by CPU percent and tell us if there's a module that seems to be loading the CPU.  This page will also tell you how much memory the modules are using.
