### Post by Mr Tim on 2008-10-16
Installing KDE is fairly easy, just go into synaptic (System, Administration, Synaptic Package manager) and search (Ctrl + F) for kubuntu-desktop. Installing this package will install kubuntu on top of your original Ubuntu desktop (it will be a large download). As for uninstalling gnome, I personally wouldn't bother if you've got enough space, if you do want to uninstall it however you will most probably have to uninstall the various GNOME packages individually, there may be a better way, but I don't know it. 
I myself am a relatively recent user (less than one year) so I don't really know all of the ways of doing things and have minimal experience with other distributions or KDE, apart from installing fedora with KDE (not by default) on my fathers computer (just for fun :D).
Hope this is of some help to you.
