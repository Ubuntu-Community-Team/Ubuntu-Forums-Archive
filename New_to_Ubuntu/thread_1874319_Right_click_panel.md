### Post by 3rdalbum on 2011-11-02
If you are using Gnome Session Fallback (the "classic Gnome" available for Ubuntu 11.10) then you need to hold down Alt as you right-click, to access that option.

If you're using 11.04 or earlier, you need to make sure you're not right-clicking on one of the existing panel elements.
