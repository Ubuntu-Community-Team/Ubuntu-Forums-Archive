### Post by MG&amp;TL on 2012-03-20
This is, I am afraid, a command-line job. Ubuntu doesn't really have setup, more packages. They're kept in a big folder. You can copy them, but often a package will depend on other packages that also need to be installed.

Does the laptop at home have an internet connection? If so, [http://www.ubuntugeek.com/clone-your-ubuntu-installation.html](http://www.ubuntugeek.com/clone-your-ubuntu-installation.html) is the best one I found. You should take the file you made home on a memory stick, or email.

If not, then it is also possible, just not easy, as I recall.
