### Post by dLeon on 2009-02-26
The problem you've got there seems familiar. Here a checklist:

-You're not using third party repository, aren't you? (getdeb, Google deb, etc.)
-Are you install something from sources (ex. tar or sources repo)?
-User manual install sources/software bypass apt check. Sometime this can make apt to get upset if you then decide to use APT's.

I read from Debian docs, Test/propose package/source sometime can do funny stuff too to your system.

If the problem caused by anything mentions above or all, re-installation is the only solution. Of course that if you don't have any system backups.
