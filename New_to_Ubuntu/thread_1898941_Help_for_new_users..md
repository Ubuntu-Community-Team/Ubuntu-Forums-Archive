### Post by irv on 2011-12-22
For new users.

When it comes to using Ubuntu for the first time, and coming to this forum us users that have been using Ubuntu for awhile forget there  are newbies out here and we talk over their heads. This is why I want to try to help out you new users by starting this thread. Some of the old users can jump in here and help out if they want too. Just remember that things we take for granted are not known by new users and must be explained in detail.

First, it is confusing when someone starts talking about Xubuntu, Lubuntu, Kubuntu. And then starts talking about KDE, Gnome, Xfce, etc. Lets start out by clearing the air. When you see the X,L, and the K before ubuntu, all it is saying is you are using ubuntu with a different DE, (Desktop Environment).

A DE is what you see when you come to the graphic environment after booting your computer.

Ubuntu comes with Unity as the default desktop, but you can use whatever you want.
Kubuntu come with KDE Desktop. (see [http://www.kubuntu.org/](http://www.kubuntu.org/) )
Xubuntu come with Xfce Desktop. (see [http://www.xubuntu.org/about](http://www.xubuntu.org/about) )
Lubuntu come with LXDE Desktop. (see [http://desktoplinuxreviews.com/2011/05/12/lubuntu-11-04/](http://desktoplinuxreviews.com/2011/05/12/lubuntu-11-04/) )

There are other buntu's out here also. Like Eubuntu which is use for education (schools) and comes with any desktop you want. What makes it “E” is the software that is packaged with it. There are others but I think you get the point.
 
Ubuntu came with Gnome as the default Desktop before the release of Unity and you can go back to it at the boot screen (fall back). Now Gnome in older releases of Ubuntu was at Gnome 2.x and is now at 3.x. This is the same with Kubuntu. The older releases came with KDE 3.x and is now at KDE 4.x. Things keep changing. 

Other things that could be explained are terminology for new Ubuntu users. But here is a good place to start: [https://help.ubuntu.com/community/Glossary](https://help.ubuntu.com/community/Glossary).  

OK, I think I have said enough about the buntu's and DE's so let's hear from others.

### Post by grahammechanical on 2011-12-22
Ubuntu always installs help documentation. It seems that many new users do not know that fact because a lot of their problems would be solved if they read the Ubuntu Desktop Guide. This is especially true for 11.10 Oneiric Ocelot.

You can find the Desktop Guide by clicking the Ubuntu Icon at the top left of the screen. A panel will open. It is called the Dash. Type help in the search panel and then click the question mark (?) icon that appears. It is white in a blue circle.

This Ubuntu Desktop Guide is also available from this link:

[https://help.ubuntu.com/11.10/ubuntu-help/index.html]("https://help.ubuntu.com/11.10/ubuntu-help/index.html")

Another on-line resource for understanding Ubuntu Linux is this Ubuntu guide:

[http://ubuntuguide.org/wiki/Ubuntu:Oneiric#General_Notes]("http://ubuntuguide.org/wiki/Ubuntu:Oneiric#General_Notes")

The wiki at Ubuntuguide.org has sections for 8.04, 10.04, 10.10, 11.04 as well as 11.10.

Regards.
