### Post by corncob on 2009-08-19
The compose key is not set by default.  Click on System > Preferences > Keyboard > Layouts > Layout Options > Compose Key Position.  I use the Right Windows key but you have other options.
I printed out the compose key sequences from [http://docs.sun.com/app/docs/doc/806-4743/6jdq6q2n7?l=en&a=view&q=Appendix+C+Compose+Key](http://docs.sun.com/app/docs/doc/806-4743/6jdq6q2n7?l=en&a=view&q=Appendix+C+Compose+Key) for reference but seldom look at it as most of what I use are intuitive.  Press the compose key, release it, then press the two keys you want to compose from (the order doesn't matter).  For example [compose key] ~ n gives you ñ.
