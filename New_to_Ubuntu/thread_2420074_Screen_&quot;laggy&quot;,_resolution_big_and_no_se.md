### Post by Autodave on 2019-05-30
You need to go into Settings -> Additional Drivers.  That will search for and probably install the 390 driver which will be fine for the graphics card.  Reboot after it installs.

As several people already have mentioned, PLEASE do not think that things work on Linux the same way that they work on Windows: these are different operating systems altogether.  You will NOT, with very, very few exceptions ever want to download and install a program from the web.  Just about ANYTHING that you will ever need is already available to you in the Software Center.

Start by going into Settings -> Additional Drivers and get the correct driver installed for that Nvidia card and report back.
