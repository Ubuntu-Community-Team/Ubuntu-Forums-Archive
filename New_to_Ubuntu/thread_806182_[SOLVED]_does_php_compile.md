### Post by quelx on 2008-05-24
PHP is an interpreted language, it is executed from the script and requires no compiling.  If you have a script meant for running from the terminal (CLI, command line, console, whathaveyou) open a terminal window (ALT-F2 > gnome-terminal) and type ```
php phpscriptname.php
``` to execute the script
