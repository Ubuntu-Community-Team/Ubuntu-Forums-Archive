### Post by bolucpap on 2009-02-19
wxdfast's latest version has trouble saving downloads from location which require username/password (like easynews, which was a big minus for me). I downloaded and recompiled the previous version, which does not have this problem, but it kept getting segmentation faults and exiting. So I switched to DownThemAll. DownThemAll is being actively developed at the moment, while wxdFast has not had an update since April 2007, so I recommend DTA over wxdFast.
