### Post by Vaedrah on 2009-01-21
I have noted a number of people posting common difficulties with VirtualBox and getting USB devices to connect to the new guest OS as well as file sharing to work between the new "guest OS" and the base "host" OS. Using Virtual Machines seems to have a lot of potential and it would be a pity if some people found that a few unsolvable hassles caused them to dismiss an otherwise interesting technology. 

I have had exactly the same problems with USB support and file sharing using VirtualBox. I have only a very basic knowledge of Linux OS but eventually I managed to piece many different replies together and make an **extremely basic** step by step tutorial for running XP as a guest OS on top of Ubuntu 8.04 as a host. I originally posted this setup procedure as an "example" under the call for topics at the start of the "absolute beginner" threads. If you are interested in an (admittedly pedantic) step by step tutorial that assumes _no prior knowledge_ of Ubuntu and syntax then the instructions can be found at   

[http://ubuntuforums.org/showthread.php?t=984986&page=15](http://ubuntuforums.org/showthread.php?t=984986&page=15)
Reply ID = 147

I expect that setup procedures for other variations could be highly truncated. Once familiarity with one worked procedure is successfully accomplished, then further similar implementation procedures for VirtualBox (or Vmware etc) e.g. running Ubuntu 8.10 on Windows Vista say, could reference previous syntax etc without being quite as pedantic as my worked example. 

However, being new to Linux myself, I have found that the initial hurdles are severe - perhaps a few directed step by step tutorials on certain topics would be useful, leaving the majority of other tutorials to assume more intermediate knowledge. I am presenting the procedure that worked on my two notebooks in the hope that people at a similar "absolute beginner" level can get VirtualBox to work properly (with USB and file sharing). :popcorn:
