### Post by Kafubie on 2010-05-20
Using a usb **DOES NOT** create faster boot times.

If you want to fresh install ubuntu
Do you really have alot of things on your HDD you want save?

If you only have...
[LIST]
[*]Songs
[*]Pictures
[*]Documents
[/LIST]

Just back it up on a USB flash drive.  If you have 2, then fresh install Ubuntu and put the media back in.

If you just have 1 USB and a spare computer, First put the backup'd files on your separate computer, erase disk, put ubuntu on it. (10.04 hopefully?)  Install it, then put the files back on your usb and put it in your FRESH installed distro.

9/10 times, faster everything.  But only slightly.
