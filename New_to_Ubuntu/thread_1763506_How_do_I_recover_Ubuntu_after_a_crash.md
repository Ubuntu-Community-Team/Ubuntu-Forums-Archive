### Post by corrytonapple on 2011-05-20
It may be the warning like what I got after the upgrade.  It might say "Unity could not start because your hardware in not good enough" or something to that affect.  So, if you can see two boxes, maybe "Cancel" and "OK" boxes, try clicking the box on the right, which is probably OK.  Then it will go to classic mode.  If you want to use Unity, you can go two ways.
The GUI way:  Go to **Applications > Ubuntu Software Center** and go to the search box.  Type "Unity 2D" and install it.  You will need an administrative password.
The Command Line Way:  Open the Terminal by going to **Applications > Accessories > Terminal**.  Type:
```
sudo apt-get install unity-2d
```
Good Luck
