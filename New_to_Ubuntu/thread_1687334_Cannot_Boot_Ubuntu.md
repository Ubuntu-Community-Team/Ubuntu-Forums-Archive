### Post by drs305 on 2011-02-13
Unless you had a specific reason to use Grub legacy:

Boot the Ubuntu LiveCD and open a terminal.

```
sudo mount /dev/sda3 /mnt
sudo grub-install --root-directory=/mnt /dev/sda
```
Note you don't use the partition number in the second command.

Reboot. If Ubuntu boots normally, run the following if the menu didn't include a menu entry for Windows:
```
sudo update-grub
```

There are Grub2 files on your computer, but if the above doesn't work they may be corrupted. In that case, you may need to chroot into the partition, purge the Grub and Grub2 files, and reinstall "grub-pc". Use the instructions in the "Chroot" link in my signature line.
