### Post by snowpine on 2010-12-29
Burn a Live CD of each, take them for a test drive (in "live" mode--don't install yet), and decide which you prefer.

If you like, you can install Ubuntu side-by-side with Windows as a "dual boot" configuration. That way you can keep Windows (and all your Windows files) and decide which operating system to use each time you turn on the computer. 

If you think you have a Windows virus, that is really a separate issue. You should install a good anti-virus application and scan your system. Ubuntu is not really succeptible to viruses; however you don't want to accidentally pass the virus on to another Windows user. Also I hope you have a good backup to an external drive of all your important files, documents, and data. That is just common sense (no matter what operating system you use).
