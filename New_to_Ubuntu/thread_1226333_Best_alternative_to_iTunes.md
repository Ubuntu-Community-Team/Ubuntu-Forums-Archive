### Post by Hospadar on 2009-07-29
You could also use itunes in a virtual machine, like virtual box.  If you have a windows cd or cd image it's super easy to set up.  There's a version of virtual box in the repositories, but it doesn't provide USB support.  For that you'll need the main sun version, you can get it here:
[http://www.virtualbox.org/wiki/Linux_Downloads](http://www.virtualbox.org/wiki/Linux_Downloads)
Use the instructions to add their repository and install.
Start it up, make a new vm, follow instructions.
There are plenty of tutorials on getting it all set up, but it's pretty easy if you just read the on-screen directions and keep your wits about you.

Once you have the vm installed, you add the usb device (your ipod) to the VM and away you go (you'll need to install itunes inside the vm of course)!
