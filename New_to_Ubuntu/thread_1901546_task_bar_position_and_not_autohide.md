### Post by fleamour on 2011-12-28
You can summon the Unity task bar by pressing the Super (Windows) Key or mouse hover.

Hitting & holding the Super key results in a numbered corresponding tab which then launches the desired operation instantly, all without the mouse.  A short tap will launch Unity Search.

To change default behaviour:

Alt + F2 ==> about:config

### Post by lolpenguin on 2011-12-28
Welcome to the Ubuntu Community Forums!
It's called the launcher. You can't change the location by default [unless]("http://www.webupd8.org/2011/11/install-ubuntu-unity-bottom-launcher.html").
To prevent it from autohiding, install CompizConfig Settings Manager (search for it in the Ubuntu Software Center).
In CCSM, scroll down to Desktop, the click on Ubuntu Unity Plugin (NOT the checkbox), change "Hide Launcher" to autohide.
