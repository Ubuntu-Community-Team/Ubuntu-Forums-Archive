### Post by coffeecat on 2015-05-31
Is the yellow information line one that tells you that one or more files are selected. If it is, click in a blank area of the main pane of the file manager. That will remove the yellow bar. Then right-click on the file you want to rename to get the context menu, allowing you to rename.
