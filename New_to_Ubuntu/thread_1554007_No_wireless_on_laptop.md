### Post by vertigopher on 2010-08-16
Well, I'm fairly new to ubuntu myself, but this worked on my laptop when I installed it.

The trick is, you'll have to have a wired connection for this to work.  Once you have that, go to System -> Administration -> Hardware Drivers.  See if your driver is listed there and activate it.

Hope this works!

### Post by diablo75 on 2010-08-16
If your laptop has an Ethernet port, use it to connect to the Internet temporarily and download all the latest updates with System>Administration>Update Manager.  About 3/4 of the time I do this, Wireless drivers are revealed within the Hardware Drivers Manager or the linux kernel updates add new support for the chipsets used in the adapter.
