### Post by s3MA00RRNY on 2009-06-09
Don't use Windows partitions for Linux. Move everything to C: and delete F:. Then when you start the Ubuntu install process, manually create a swap partition as large as the amount of RAM you have and create an ext3 partition that takes up the rest of the space.

Explain your stituation a little bit more before you try this. How are you installing Ubuntu?
