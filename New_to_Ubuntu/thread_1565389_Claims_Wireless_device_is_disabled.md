### Post by Rasa1111 on 2010-08-31
Heres a good guide/book/manual..
[http://ubuntu-manual.org/](http://ubuntu-manual.org/)

 Im really not very experienced with wireless connections, 

But had my first Ubuntu wireless connection experience a few days ago. lol

 I installed 10.04 on a laptop with windows 7 on it, 

when in windows, the wireless connected,
when in Ubuntu, it did not. 

All i had to do was click on the "network connection" icon in the top panel, entered the WEP key for the wireless connection when the window asked for it,
(that i wrote down while in windows), and it instantly connected right up. 

So, 
I booted into windows 7,
clicked on the network connection icon in the taskbar,
opened up the connection info, 
checked "show characters" on the WEP key, 
Wrote it down,
then booted into Ubuntu,
opened the network connection window, 
entered the WEP key I wrote down,
and it was done. 
 Hopefully it's as easy for you.
