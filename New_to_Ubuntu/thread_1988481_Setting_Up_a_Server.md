### Post by PhantomTurtle on 2012-05-27
You could set up a NAS(Network Attached Storage). Quidsup did some great tutorial's on this. If you want to build a NAS he has two tutorials on that - [Part 1]("http://www.youtube.com/watch?v=bzBBkywRVbY"), [Part 2]("http://www.youtube.com/watch?v=dxNEhba65SA"). If you already have a computer for that then you can get either [OpenMediaVault]("http://www.openmediavault.org/") or [Ubuntu Server]("http://www.ubuntu.com/download/server"). He did a review on OpenMediaVault which you can check out here -  [http://www.youtube.com/watch?v=fIFM1tH24V0]("http://www.youtube.com/watch?v=fIFM1tH24V0"). OMV is Debian-based. A full tutorial on how to create a NAS with Ubuntu Server, also by Quidsup - [http://www.youtube.com/watch?v=-5Z_-3EBIHE]("http://www.youtube.com/watch?v=-5Z_-3EBIHE"). The iPad and Droid X2 should work with it. I do not have these devices, so I cannot check if they work. However, like I said, it should work. The tutorials are very easy to follow and you shouldn't have any trouble with it. Good Luck with the server :)
