### Post by Paqman on 2011-10-20
A good place to start is learning about package management and repos, as it's one of Linux's main distinguishing features. Install Synaptic Package Manager if you want a tool that exposes a bit more of the nuts and bolts.

Once you've figured out how it works you could do a little project like grabbing the Minimal image and using it to build a customised system. Then scrap it and build one with a completely different desktop environment. The start writing scripts to automate it all. A lot of people write a script that automates all their tweaks and customisations they do post-install. Besides a good learning exercise this can really take a lot of work out of the reinstall process.
