Hello!

You have created what we call a "laundry list" thread.

It is preferred on the forums that each thread deal with one item at a time.  This avoids confusion and allows members of the community with expertise in particular subjects to help with one item without causing a traffic jam for everyone else.

If you would, please start a separate new thread for each of the questions/subjects you would like help with.  You'll get much better advice that way.

As for which threads to find answers in (if there are any already ...), I find that the search functionality provided leaves a great deal to be desired.

When I want to find something, I usually use google and search by doing this:

```
site:ubuntuforums.org my search terms here
```
