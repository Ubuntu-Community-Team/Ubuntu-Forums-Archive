### Post by Buffalo Soldier on 2009-01-09
Hi mikebravo,

Have you tried changing **[SIZE=4]Evolution preferences[/SIZE]**?

> **mikebravo said:**
> I like to strip out names of strangers in the header and snip out text that is unnecessary. I realize that I can do so if I "reply" instead of "forward" but if there are any embedded attachments e.g. photographs, they are lost.

In Evolution toolbar, go to: **Edit -> Preferences -> Composer Preferences -> General -> [SIZE=4]Forward Style[/SIZE]**

By default the forward style is Attachment. Change it to Quote or Inline.

> **mikebravo said:**
> Besides the edit of a "forward" the only other feature I demand is spell checking.

In Evolution toolbar, go to: **Edit -> Preferences -> Composer Preferences -> General -> [SIZE=4]Spell Checking[/SIZE]**
