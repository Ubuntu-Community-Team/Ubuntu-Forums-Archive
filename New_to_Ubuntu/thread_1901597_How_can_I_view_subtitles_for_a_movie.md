> **ACupOfCoffee said:**
> I'm guessing you used BitTorrent to pirate stuff copied from DVDs. If that's right, then you can't because you don't actually have a file with the subtitles in them.

Not everybody pirates movies.. There are a lot of free movies to download.. I.E..  The Internet Archive..  [http://www.archive.org/details/movies]("http://www.archive.org/details/movies")


And you can simply go to Ubuntu Software Center and install subdownloader

SubDownloader is a tool to automatically download/upload subtitles for video files (DIVX, MPEG, AVI, etc.) and DVDs using fast hashing.

After that put the movie and the .srt in one folder, rename both the same title.. Open in VLC. And yer off!
