### Post by sayakb on 2008-09-20
Check out the GDM themes here: [http://www.gnome-look.org/index.php?xcontentmode=150&PHPSESSID=57599e471bc5a8523b6e245998f3f64b](http://www.gnome-look.org/index.php?xcontentmode=150&PHPSESSID=57599e471bc5a8523b6e245998f3f64b)
Goto System->Administration->Login Window->Local
Click Add button and add the downloaded GDM theme
Select it and close the window.
