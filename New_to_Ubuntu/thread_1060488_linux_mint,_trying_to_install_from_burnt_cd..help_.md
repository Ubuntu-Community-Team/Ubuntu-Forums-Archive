### Post by sonu 1807 on 2009-02-04
If you downloaded via torrent then no issues... otherwise you have to check md5 sum

In ubuntu right click the downloaded ISO file and then choose write to disk... choose very slow speed to write, 1x or 2x. Most probably your comp should boot from CD. In my laptop i have to press F9 to change the boot device. 

With unetbootin you can install linux mint on a pen drive and use it to install it on your comp. This way you don't have to burn any CD and also your installation will be quicker. Installation of mint is absolutely the same as that of ubuntu. To be fair, it is not a different distro, just a different look
