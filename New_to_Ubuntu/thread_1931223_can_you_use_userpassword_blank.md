### Post by sudodus on 2012-02-25
Welcome to the Ubuntu Forums!

You need to enter a user name and a password, and you should be able to remember the password, but others should not be able to guess it, so select a good one!

However, you have a choice to log in directly without using the password (but it is still there, and you need it to perform things that need superuser privileges).

*Edit: it is independent of vista, but it might help remember it to use the same.*

Not sure about the blank password. Ubuntu-Live has a blank PW but I have not been able to reproduce that on an installation. There are always issues with ssh-access and other things.

Also I think the installer will not allow an empty password. So for now you have to enter one.

It has nothing to do with your windows-password. You can use a different one if you want.

Usually on a home-pc during install I enter a password, remember it and activate "automatic login". The password it then not needed for starting the PC. You still need it for administering the system, install software, change settings etc.
