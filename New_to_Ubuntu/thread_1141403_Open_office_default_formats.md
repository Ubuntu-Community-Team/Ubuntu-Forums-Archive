### Post by angelsguitar on 2009-04-28
> **Mauler5858 said:**
> I am working to make a desktop solution for my company using Hardy with Open Office3.  In this environment they need to be able to send documents to M$ users.  I need for the file formats that OO saves by default to be the M$ formats.  I found how to make Writer do so, but the other two I am concerned about is Spreadsheet and Impress.  Is there any way to do that?

In OpenOffice, go to Tools - Options, then select from the left Load & Save and finally click on General.  In the lower left part of the opened options screen you'll find a drop down box labeled Document Type that says "Text Document" by default.  You just have to look for the document type you want to change (Text, spreadsheet, presentation, etc) and change it's default format on the drop down box on the right, for each document type.  That way you'll change the default format OpenOffice saves by default.

I'm including a screenshot so you get the idea.  Hope this helps.
