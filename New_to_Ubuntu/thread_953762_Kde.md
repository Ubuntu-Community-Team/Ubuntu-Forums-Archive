### Post by igknighted on 2008-10-20
KDE4 is the latest version of KDE, a nearly complete rewrite from KDE3.  The KDE4 version in Hardy is a very early version and I would avoid it.  If you want to see KDE4, try the Kubuntu Intrepid beta.  Otherwise, use the standard KDE3 in Hardy
