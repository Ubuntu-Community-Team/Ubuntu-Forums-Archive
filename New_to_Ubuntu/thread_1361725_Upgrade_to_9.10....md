### Post by 3rdalbum on 2009-12-22
Get the alternate CD, and burn it. Insert it into the computer and it has a script called "cdromupgrade" on it. Make sure you have an internet connection (as it will also need to update any other packages you have that are not normally on the CD), and then run the script.
