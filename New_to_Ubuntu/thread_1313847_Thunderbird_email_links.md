### Post by ad_267 on 2009-11-04
Go to edit - preferences - attachments, then set your web browser under http and https. If your browser isn't listed, you need to select "use other'. Unfortunately Thunderbird isn't integrated well with Gnome so it won't give you a list of applications but will show the file open dialog. Firefox is located at /usr/bin/firefox so you can enter that in the address bar.

I'm basing this on Thunderbird 3.0 but I think it's the same in Thunderbird 2.
