### Post by hugegreenbug on 2014-08-21
If you are using crouton, you aren't dual booting, but running Ubuntu in a chrooted environment.

If you are using chrubuntu, you are dual booting and running Ubuntu natively.  I have created images for the c720 that has all the issues worked out (suspend, mouse, graphics, etc ...).
