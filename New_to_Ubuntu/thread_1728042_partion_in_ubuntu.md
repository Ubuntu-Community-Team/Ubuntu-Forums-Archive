### Post by mastablasta on 2011-04-13
depends on how you installed it (i am guessign wubi?). If you installed it via Wubi then Linux is actually in a large file inside windows on a partition you chose to install it to. Wubi is good for testing, but to get the real deal you should do a side by side.
 
Linux has it's own paritions. It can be installed on a separate partition. 
 
Also linux doesn't do C:, D: etc, but has a different way of working with drives. They are mounted as directories (called folders these days in windows). Read more about it in the manual.
