### Post by mastablasta on 2011-03-21
These are windows games and they are best run under windows. Only some programmes can run under wine and even then if the games are directX 10 or 11 based they likely won't run as good because wine has Directx9 max. You can check what runs and how to run them here: [http://appdb.winehq.org/](http://appdb.winehq.org/)
 
Oblivon should be able to run, but again not as good as in windows. YOu wont' be able to have extreme graphics (probably). Only the porgrammes and games marked Platinum run flawlessly but that doens't necessarily mean that they run at their highest settins.
