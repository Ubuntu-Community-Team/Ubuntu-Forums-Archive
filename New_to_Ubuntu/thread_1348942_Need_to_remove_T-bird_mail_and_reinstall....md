### Post by D3M0L1$H3® on 2009-12-07
Either:
1. go into Ubuntu Software Center (older versions it will be "Add/Remove Programs" and find Thunderbird and uncheck it and hit apply.
2. Go to Synaptic Package Manager (System  -> Administration) and ctrl+F Thunderbird and do the same

Then go back into whichever you just used and recheck thunderbird and install

### Post by yeats on 2009-12-07
That will still leave your profile folder, which is located in your home folder and is in a hidden file named .mozilla-thunderbird.  If you *really* need to start over, you could rename that folder and open Thunderbird, which will allow you to set your accounts up from scratch.
