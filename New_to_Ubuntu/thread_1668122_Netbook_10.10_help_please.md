### Post by labatts on 2011-01-15
Device manager as in something that will list hardware components (ram, usb ports, etc)?  
There is not one of these installed by default.  However, you can get one easily via Ubuntu Software Center.  There are a couple, but there is one actually called "device manager" that I have used in the past.  It is pretty good.  Install that (if I am right and this is what you are looking for) and you should be good to go.
