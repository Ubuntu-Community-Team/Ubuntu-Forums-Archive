Hi and welcome to the forums. There are many good websites available for ubuntu info and help. [http://ubuntuguide.org/wiki/Ubuntu:Oneiric](http://ubuntuguide.org/wiki/Ubuntu:Oneiric) is one, [https://help.ubuntu.com/](https://help.ubuntu.com/) is another good source. > **sniper8752 said:**
> 
i understand you use the command line a lot, which can seem a little intimidating.
Ubuntu can be used without ever (hardly) using the command line. It may seem intimidating, but realize that command line help is often given because there are various versions of ubuntu that use different desktop environments, and command line advice is given because it is universal between all versions. Most of the time you will only need to copy and paste commands without the need to memorize anything.

Just be patient, as it takes time to learn *anything* new. And most of all, have fun.

### Post by morhin on 2012-01-27
I tried starting with the forums but realized I didn't know which questions to ask and when I did ask I had trouble understanding the answers.
I'm one of those people who for a long time kept looking for the "any" key.

Fortunately I found a book called Unbuntu For Non Geeks (Lucid Lynx 10.04) and it go me rolling. It's still my favorite reference book. 

No Starch Press who puts the book out for authors Rickford Grand and Phil Bull, will be releasing the 5th edition when the next stable unbuntu release comes out. Til then I'll keep running 10.04 while I'm learning what, when, where, why and whatever.

If I had to depend on the forums, as a noobie, I would be frustrated and probably would not have continued. But now that I'm up and running I'm getting more involved with them and find them quite helpful.

 Sometimes I think we noobies don't get enough credit for how much we don't know.

Hang in there and good luck. 

By the way..... the more I use Ubuntu, the more I like it.
