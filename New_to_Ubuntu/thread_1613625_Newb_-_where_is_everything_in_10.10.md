### Post by cariboo on 2010-11-04
It sounds like you installed the netbook version, commonly known as Unity. To use the normal desktop, log out by clicking the power icon in the top right corner and select Log Out. Once at the login screen click you name and in the bottom panel Select Ubuntu Desktop Edition, then enter your password to continue.

Unity is still fairly new and unfinished, it is getting a total revamp during this development cycle. It will also be the default desktop for Natty Narwhal (11.04).

### Post by Vaphell on 2010-11-04
don't be afraid of command line tools - they are more foolproof way to provide a working solution. Gui can be customized so you can't always expect things being in the same place, command line stuff is always the same. Besides if you get a bunch of commands to do, you simply fire up the terminal and copy paste them.
When i have to install anything i prefer to type 'sudo apt-get install <packagename> ', password than to perform the following
1. click system
2. select administration
3. click Synaptic package manager
4. type password
5. find packagename
6. mark packagename for installation
7. click 'Apply'

back to the topic: Assuming you installed the default version of ubuntu, your windows partition should be detected by the system and a shortcut should be created in the 'places' portion of the main menu on the top panel (middle part, between Apps and System), also you should be able to find it in the tree in the file browser (nautilus). Windows boots up just fine?
