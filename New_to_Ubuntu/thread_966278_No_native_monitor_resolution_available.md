### Post by blfgomes on 2008-11-03
Have you tried fiddling with xrandr?
```
xrandr -q
``` will list the supported modes. If 1280x768 is not listed, you can try to add it manually with
```

$ gtf 1280 768 60
  # 1280x768 @ 60.00 Hz (GTF) hsync: 47.70 kHz; pclk: 80.14 MHz
  Modeline "1280x768_60.00"  80.14  1280 1344 1480 1680  768 769 772 795  -HSync +Vsync
$ xrandr --newmode 1280x768 80.14  1280 1344 1480 1680  768 769 772 795  -HSync +Vsync
$ xrandr --addmode default 1280x768
$ xrandr --output default --mode 1280x768

```

Substitute "default" in the snippets above with whatever gets displayed before "connected" after you run "xrandr -q" (second line).
I have to mention that trying to set my resolution to 1280x768 fails (probably because my driver/display doesn't support it) and that I'm no expert here, but hopefully it is something to get you started.
