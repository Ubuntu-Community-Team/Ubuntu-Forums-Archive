### Post by squakie on 2012-11-19
Try going to system settings, clicking on Brightness and Lock, and changing the time out to never.  Sometimes the system thinks that because there has been no input from the keyboard for that amount of time that it should blank the display.  Changing it to never should work for the movie, but if you do want it to blank at other times, you'd probably need to set that time out again.
