### Post by spcwingo on 2012-04-25
You can try the latest release to see if that will get you going.  To do that copy/paste this in a terminal:

```
sudo add-apt-repository ppa:pidgin-developers/ppa && sudo apt-get update && sudo apt-get upgrade
```

After you add that repository you get the latest version of Pidgin whenever a new one come out.  :)
