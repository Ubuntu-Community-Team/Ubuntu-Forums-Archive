### Post by mcduck on 2013-01-25
The sole purpose of "clean up by name" is to rearrange the icons by name. And the file manager's developers have decided to arrange icons on the left side, so that's what you get.

At least on my computer Reload (Ctrl-R, not F5 by the way) doesn't rearrange anything. But I'm running 12.04 so that might be the difference, can't remember how it worked in previous versions as I very rarely need to reload anyway.
