### Post by bangmalley on 2008-10-21
current ubuntu version uses tracker-search-tool
```

sudo apt-get install tracker tracker-utils tracker-search-tool libdeskbar-tracker

```

launch it from Applications->Accessories
or add it to the panel (Deskbar applet).
