### Post by MaxIBoy on 2009-03-12
+1 for shell scripts. Automate stuff!


More things to try:

[LIST]
[*]Writing a small program entirely in awk. (Pretty easy, awk is a full-fledged programming language.)
[*]Writing a small program entirely in sed. (Not so easy, sed is just a stream editor. You'll be pretty much restricted to truth tables.)
[*]Start learning Python. Not exactly restricted to UNIX, but it's a lot of fun and you find uses for it once you know it.
[*]First, get framebuffering to work on your computer so you can have more than 25*80 characters in full textmode. Then, see how long you can spend in full textmode using textmode programs (links as a web browser, midnight commander as a file browser, fbi to look at pictures on the framebuffer, mplayer to view videos on the framebuffer, emacs (and to a lesseer extent, vi) for editing files, pine or mutt for email, and so on.)
[/LIST]
