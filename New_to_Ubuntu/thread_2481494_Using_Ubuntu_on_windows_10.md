Try Ubuntu in your VM and get to know it. Once you learn where things are and learn a bit of what you can do, then make a decision on what you want to do.

Me? I have Windows as a dual boot beside Ubuntu. I live mostly on Ubuntu. I have KVM there as my virtual host, which I find more useful and flexible than Hyper-V. I think the only reason I have Windows loaded on Metal is for some games.

Linux is very flexible in that you can interchange many layers and modules to customize what you want as a Desktop and User UI. That given, it gives you the freedom to break things in a big way, if you are not careful. Sometimes, I think that is how I learn most! LOL. (Be sure to take backups.)
