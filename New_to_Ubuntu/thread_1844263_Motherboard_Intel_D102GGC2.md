### Post by LowSky on 2011-09-14
Hello Welcome to the forums.
The first thing i will say you may not like: Ubuntu 9.10 is not supported any more. You should really use 10.04 LTS or the current 11.04 release.

You can't damage the BIOS unless you try to flash it multiple times over and over, and doing a Ubuntu install does not ever flash the BIOS.
More so I don't see why a Intel motherboard or any motherboard would not work. My guess is you saw a busybox screen which is a problem with some installations, the remedy is to use the Alternative Install CD, that will usually clear up issues.

Without seeing the error you exactly saw it is hard to pinpoint. Can you give us more information on what the screen says when it fails to load?
