### Post by Bosconian on 2008-07-14
I installed screenlets using sudo apt-get install screenlets

It installed fine but it does not show up in the Accessories menu. Also if I type screenlets from a terminal it says command not found.

If I try to reinstall it tells me it is already installed.

How do I fix this?

### Post by Troll_the_Great on 2008-07-14
Try this:right click on "Applications" - "Edit menus" - "Accessories" and see if the "Screenlets" box is checked.If not, check it and they should appear under Applications-Accessories.
Hope this helps.
Cheers!

### Post by prshah on 2008-07-14
> **Bosconian said:**
> 
It installed fine but it does not show up in the Accessories menu. Also if I type screenlets from a terminal it says command not found.


Either look in System-Preferences-screenlets or type ```
screenlets-manager
``` in either a Terminal or Alt+F2
