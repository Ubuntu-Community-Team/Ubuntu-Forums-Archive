### Post by MG&amp;TL on 2012-01-21
> **markbahnman said:**
> So I recently started using Ubuntu and I am trying to understand exactly how customizable Unity is. Can I completely change the color theme, the behaviour of the launcher and system tray, etc?

Depends how much messing you're prepared to do. Install compizconfig settings manager, then click on the Unity plugin, to see some configuration options.

Also, see myunity, and you can also change a lot of things like pictures and textures in /usr/share/unity/

It's becoming a lot more configurable in 12.04, released in April.
