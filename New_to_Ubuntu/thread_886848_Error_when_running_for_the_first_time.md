### Post by WRDN on 2008-08-11
I'm guessing you are using the "Manual" option on the installer from the LiveCD, as that is the only place I have ever seen that error message.
If you are, then select the partition, select "Edit Partition" and then, in the text box (there will only be one), type "/" (root).
