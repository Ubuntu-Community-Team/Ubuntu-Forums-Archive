### Post by anaconda on 2010-03-03
"dir" -command is an old dos command. You should use "ls" in its place.

advantages: ls shows directories with different colours.. It is easier to spot which are files and which are directories etc.

and with different options ls can show quite a lot information. eg. 
ls -l
shows file sizes, and read-write rights, and owner of each file...
