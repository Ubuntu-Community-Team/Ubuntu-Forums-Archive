### Post by decoherence on 2009-01-07
I'm throwing in a vote for Arch Linux, that lovely, simple, 'bare bones' distro. It's a great system for intermediate users, especially if you want to learn more about Linux. It also gives you simple control of how daemons load at boot time. You can easily make certain daemons load in the background, potentially making your boot faster.

Arch is a simple system that makes very few assumptions about what you want. As such, it doesn't come a with GUI but installing one is very simple using the fantastic pacman package manager. Plus, you can be more confident that there isn't some resource sucking process running in the background (eg. tracker)

The documentation is very good and generally simple to understand. If you decide to check out Arch, be sure to make use of their wiki. The folks on the Arch forums aren't quite as gentle as us Ubuntu users when it comes to people asking questions that have documented answers.

Have you detected what the general theme is for Arch?

KISS!
