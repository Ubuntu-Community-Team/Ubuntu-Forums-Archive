Depends on what you're trying to pdf. If you're trying to PDF a website confirmation for example, firefox can do that natively. Go File - > Print select print to file change output format to pdf and enter a file name. You can also change where the PDF you create is saved. Then click print. Hey presto. 

As for an equivalent to PDF writer in OSX or adobe acrobat in windows I'm not to sure as I'm a newbie here (just formatted my mac a week ago and now single boot Ubuntu Jaunty on my macbook!)
