### Post by deadflowr on 2014-01-05
If you wanted to install the unity desktop, then simply install ubuntu-desktop
```
sudo apt-get update && sudo apt-get install ubuntu-desktop
```
when it finishes installing, log out and when you log in look for the icon next to your login name and switch it to ubuntu.
Or if, the login screen has stayed as the lubuntu login screen look for the entry saying default, or lubuntu, or something and switch it to ubuntu. Then login.

If you want to run a pure ubuntu then you would need to follow something like what is in here
[http://www.psychocats.net/ubuntucat/pure-ubuntu-13-04/](http://www.psychocats.net/ubuntucat/pure-ubuntu-13-04/)

However, I would copy the entry for lubuntu, if you go this route, and look over what is being removed to make sure that my version actually has those packages.
