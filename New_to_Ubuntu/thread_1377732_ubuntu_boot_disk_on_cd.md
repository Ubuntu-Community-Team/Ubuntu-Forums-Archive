### Post by cariboo on 2010-01-10
THe Live CD is the same cd that you used to install Ubuntu. Select the Try before installing option form the Menu.

There are several boot floppies available that allow you to boot from a usb drive. Have a look [here]("http://www.pendrivelinux.com/use-a-floppy-to-boot-usb-pendrive-linux/"), you could use this method to create a boot CD that allows you to boot from a usb drive.
