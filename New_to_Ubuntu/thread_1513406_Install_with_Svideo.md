### Post by Dr_Willis on 2010-06-19
Ive found the past that with some nvidia cards the svideo out will work fine for an install. The system will default to using the TV out. But with some other nvidia chipsets it wont default to the svideo out at all. So at least you are seeing somthing.

You Might want to try  the 'alternative installer cd' that will use a text based menu that  should be viewable.  Or plug in a monitor untill you can  use the normal installer..  

Or if you can get  the live cd to the desktop. You might be able to set the proper res/mode so you can see the installer.  

I dont know of any boot time options to set. Ive never had to use them. I normally just use the alternative installer cd in 'problem cases'

" like it is out of sync and it looks like it is black and white only. I can't really see much outside of the Ubuntu logo."

Perhaps this could be a PAL Vs NTSC issue also.  

Good luck.
