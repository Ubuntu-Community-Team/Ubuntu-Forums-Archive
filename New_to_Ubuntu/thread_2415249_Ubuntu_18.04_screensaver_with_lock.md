Hi,

I'm sure I'm missing something obvious, but could someone please tell me how to configure Ubuntu 18.04's built in screensaver so that when the user interacts with the computer after the screensaver has kicked in, the user needs to enter a password (instead of immediately seeing their desktop and whatever applications are open)?

If the built in screensaver doesn't have an option to require a password, then please advise on an alternate screensaver that has this feature that works well with Ubuntu 18.04.

Thank you!

---

### Post by Dennis N on 2019-03-23
In search box on overview screen, type "privacy". In results, click on Settings > Privacy. The screen lock must be turned ON to require password.
