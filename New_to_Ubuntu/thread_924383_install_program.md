### Post by northern lights on 2008-09-19
Offline dependency management is probably the weakest spot in the shared library concept (nonetheless a very good concept).

As long as the applications you'd like to install are in the repositories, you can still use Synaptic to create a download script and then install with GDebi on the offline comp.
