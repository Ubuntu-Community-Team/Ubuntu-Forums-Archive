### Post by kroq-gar78 on 2010-12-29
32-bit ubuntu netbook should work best. Netbooks AFAIK dont really work well with 64-bit or do no work at all.
Are you sure that Ubuntu doesn't have the driver? Once you install it, there should be a prompt to install them. Also, I've only heard of netbooks with graphics accelerators, not actual graphics cards.

To choose between 10.04 Long Term Support and 10.10 (newer), I'm not really sure. If you're not sure, just install through WUBI if you have windoz: [http://www.ubuntu.com/desktop/get-ubuntu/windows-installer](http://www.ubuntu.com/desktop/get-ubuntu/windows-installer)

or just pick 10.04. It's LTS so it'll be supported for 3 years which started April 2010 and is probably more stable (I'm really not sure about stability, but just a logical guess). If needed, you can always upgrade to 10.10.

Ubuntu uses GNOME whole Kubuntu uses KDE. 

[http://en.wikipedia.org/wiki/GNOME](http://en.wikipedia.org/wiki/GNOME)
[http://en.wikipedia.org/wiki/KDE](http://en.wikipedia.org/wiki/KDE)

KDE is more like Windows Desktop Manager
GNOME is different.
There are some differences: [http://www.linuxuser.co.uk/features/gnome-vs-kde/](http://www.linuxuser.co.uk/features/gnome-vs-kde/)

If your machine is 32-bit then you need to download and install Ubuntu 32-bit version. 
If your machine is 64-bit then both should work fine.

Ubuntu has a version for Notebooks like yours. 
[http://www.ubuntu.com/netbook](http://www.ubuntu.com/netbook)

Note that if you'll go for that version, the Desktop Manager is not GNOME, it's Unity.
[http://www.ubuntu.com/netbook/features#unity](http://www.ubuntu.com/netbook/features#unity)
