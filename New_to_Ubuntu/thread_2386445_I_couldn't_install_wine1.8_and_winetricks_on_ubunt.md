### Post by kc1di on 2018-03-06
Hello dangquanghienitc and Welcome to Ubuntu.  

I find that Playonlinux is a good wine frontend and allows you to install many versions of wine.  You can install it by doing this in a terminal```
sudo apt-get install playonlinux
``` Once installed it will be found in the game section of the menu.  after you launch it go to tools tab and manage wine versions and install any wine version from 1.4 to 3.0 it's a nice tool because some programs run better on older versions and some on newer ones.  Also it installs them in virtual files so that if you uninstall it does not affect other installed versions. good luck and enjoy. 
you can read more about playonlinux here: [https://www.playonlinux.com/en/]("https://www.playonlinux.com/en/")
