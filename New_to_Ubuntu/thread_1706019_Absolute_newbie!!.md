Welcome :)

First thing, don't give up!! You might face some troubles but once you get things sorted and become a linux power user it's all smooth sailing :)

> So, it would be a great help to me If someone tells me where exactly to start trying to learn Linux.

First step would be to use it, the more you use it the better you get.

Google everything, although I did this while on windows, there is so much more help available with linux on the web.

Also got the ubuntu manual:

[http://ubuntu-manual.org/](http://ubuntu-manual.org/)

official help articles:

[https://help.ubuntu.com/10.10/index.html](https://help.ubuntu.com/10.10/index.html)

ubuntu wiki you can usually find access to via google, for example if you want to install the nvidia drivers for your graphics card:

[http://www.google.com/search?client=ubuntu&channel=fs&q=nvidia+drivers+ubuntu+wiki&ie=utf-8&oe=utf-8](http://www.google.com/search?client=ubuntu&channel=fs&q=nvidia+drivers+ubuntu+wiki&ie=utf-8&oe=utf-8)

first result is the wiki article on activating the nvidia graphics card drivers.

Almost everything can be found by googling a little :)

> I don't know much of programming.. any to be honest and I decided Linux will be the first Language I'll learn.

Linux technically isn't a language, technically it's not even an Operating system, it's a kernel, Ubuntu is the OS.

[http://en.wikipedia.org/wiki/Linux](http://en.wikipedia.org/wiki/Linux)
[http://en.wikipedia.org/wiki/Ubuntu_%28operating_system%29](http://en.wikipedia.org/wiki/Ubuntu_%28operating_system%29)

There is a programming sub-forum here:

[http://ubuntuforums.org/forumdisplay.php?f=39](http://ubuntuforums.org/forumdisplay.php?f=39)

Anyways feel free to ask any more questions, I'll try to answer them :)
