Here is a previous thread that discusses alternatives to Hazel ([http://ubuntuforums.org/showthread.php?t=738974]("http://ubuntuforums.org/showthread.php?t=738974")). The last post ([http://ubuntuforums.org/showpost.php?p=10880842&postcount=14]("http://ubuntuforums.org/showpost.php?p=10880842&postcount=14")) mentions a program called Maid ([https://github.com/benjaminoakes/maid]("https://github.com/benjaminoakes/maid") & [http://rubygems.org/gems/maid]("http://rubygems.org/gems/maid")) that is based on Hazel.
