### Post by cariboo on 2009-03-26
Once you have activated cool n' quiet in the BIOS you don't have to do anything. CPU speeds will ramp up on demand. I have an MSI K9NGM2 with an AMD 64 X2 3800+, when using cool n' quiet it drops the cpu speed by 50%. When my cpu fan is cleaned out my cores run at 38&#730; C with Firefox running and a cpu load of <2%, the cpu fan runs at 2100RPM.
