### Post by andrewt522 on 2008-07-17
I too am a beginner and picked up a book just last weekend called "Ubuntu Linux Toolbox."  You can find the description here:

[]("http://www.amazon.com/s/ref=nb_ss_gw?url=search-alias%3Daps&field-keywords=ubuntu+linux+toolbox&x=0&y=0")

It's not for the faint of heart as it explains the terminal in depth.  Personally, I was looking for something a little more advanced as I had already figured out how to manipulate most of the GUIs and knew nothing about the terminal, which, I found, is preferred by all the pros on this site.  And, hey, I'd like to be a pro.  Anyway, I saw some other books that were for beginning users.  Depending on your experience level, these may be helpful to you.  Regardless, there are quite a few books out there so take some time flipping through them.  I'm sure you'll find something you're looking for.

### Post by markbuntu on 2008-07-17
The latest Official Ubuntu book is written for 8.04. it is pretty basic but if you a new it is a great resource. I saw it in a bookstore last week. All of the other books are for earlier versions, so far.

The Linux Pocket Guide from O'Reilly is good for a basic terminal command reference. Even though it says it is written for Fedora the commands are pretty much the same. It even has a tiny section at the back about shell scripts.
