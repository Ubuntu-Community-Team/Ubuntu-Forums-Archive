### Post by blackened on 2008-11-12
Imagemagick is a commandline only program, so just typing *imagemagick* into the terminal will do nothing.

Try
```
man imagemagick
```

or begin reading about it on [the website]("http://www.imagemagick.org/script/index.php").

### Post by Paqman on 2008-11-12
It's a terminal app, so you should be able to use it from the terminal.

However, there's a nice plugin for Nautilus that uses imagemagick, it's called nautilus-image-converter and allows you to resize and rotate images from a right-click. Much easier.
