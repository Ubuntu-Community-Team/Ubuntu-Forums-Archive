### Post by Elfy on 2011-05-31
Pick the manual/advanced partition option - it will allow you to pick the existing partition.

Select the correct partition, edit partition - in the new window format and choose / as the mountpoint - save and then you can carry on.

If you've data you want to save - back it up ...

If you've a seperate /home then do the same as above - no format though and /home as the mountpoint.
