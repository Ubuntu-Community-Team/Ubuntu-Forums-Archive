### Post by snowpine on 2012-06-07
Welcome to the forums!

I agree with the advice above to install all available updates. These are bug fixes and security patches to help your Ubuntu system run more stable/secure. Ubuntu is a fast-moving distribution and you should expect frequent updates. (Updating once a week is a good guideline; some people update every day.)

In my experience on these forums, the biggest mistake a lot of new users make is to second-guess: "I don't know what that is, I probably don't need it, must be safe to delete." So I would caution you against falling into that trap. ;)
