### Post by shifty_powers on 2008-11-08
well for a start, an rpm file is the "exe" file for a different linux distribution and will not work on ubuntu (it is for red hat based distro's).

ubuntu is based on debian, as such it uses .deb files as it's "exe" files. that is what you need. you can possibly convert the rpm file to a deb file but it is not recommended.
