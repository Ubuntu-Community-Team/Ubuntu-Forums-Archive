### Post by critin on 2011-10-22
The easiest way is to go to Tools in Firefox, click on Add-Ons and search for 
Web 2PDF Converter 1.12

It will add a button on the browser and you can download as a PDF file which will keep everything in order.

And the button will be on the browser for next time you want to save something.
