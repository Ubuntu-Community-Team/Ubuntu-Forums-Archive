### Post by Allegretto on 2008-05-31
Hello,

I'm just about to attempt installing Ubuntu as a dual boot after getting to know it using Wubi, but before I get started I'd like to know if I've got this right. 

Using PartitionMagic in Windows I've reduced the size of my Windows partition, leaving what I trimmed off as unallocated free space. Will I need to do anything with this free space prior to using the Live CD, or will it recognise it and sort out the new partitions for me?

Thanks very much.

---

### Post by FuturePilot on 2008-05-31
No you shouldn't need to do anything else. When it comes to the partitioning part of the installer, just tell it to use the largest continuous free space.

---

### Post by Joeb454 on 2008-05-31
As long as you choose "Guided - Use Largest Continuous Free Space" during the install - you should be fine :)

Though backup your Windows stuff just in case - something I'd do when installing *any* OS
