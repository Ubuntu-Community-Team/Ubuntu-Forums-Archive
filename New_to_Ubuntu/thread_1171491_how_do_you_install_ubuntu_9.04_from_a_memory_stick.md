### Post by ugm6hr on 2009-05-27
Another option is to install the Netbook Remix.  It can be downloaded as an .img file, which is ready to be written to a FAT-formatted (usual default for most) USB stick of 1GB+.

It is easy enough to then install the rest of the default ubuntu-desktop if you so wish (and remove the netbook launcher).
