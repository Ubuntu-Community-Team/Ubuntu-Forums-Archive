### Post by coffeecat on 2008-07-19
According to a quick Google, .kwd files are those created by Kword. Seems logical.

The only solution I could suggest is to install Kword via Synaptic, and use Kword to convert all your documents to a format understood by whichever word processor you are going to use in Ubuntu. If you are using Ubuntu/Gnome, be aware that Kword will bring in a load of KDE dependencies with it. Not a problem, unless you're using dial-up.

I don't know of a utility to convert .kwd files, but [here's a google search to get you going](http://www.google.com/linux?hl=en&q=.kwd&btnG=Search&meta=). :wink:
