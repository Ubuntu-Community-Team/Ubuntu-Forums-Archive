### Post by papibe on 2010-11-03
In the desktop version you can open as many terminal windows as you wish. Inside a terminal windows you can open a new tab in the same windows by using Shitf-Ctrl-T

You can also get out of graphical interface (Gnome), and go to a old school text mode terminal interface. To do that press Alt-Ctrl-F1

By changing the F key, I believe you can have up to 6 text mode terminal (F1, F2, ..., and F6).

Did this help?

Edit: you can go from the text mode to the GUI by pressing Alt-Ctrl-F7
