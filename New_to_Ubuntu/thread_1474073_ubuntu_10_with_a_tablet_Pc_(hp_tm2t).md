### Post by chadmerkert on 2011-04-20
> Hi chadmerkert! I was just wondering if all the hardware of tm2t are supported in 11.04 (HDMI, touchpad, etc)? And is it hard to setup the hybrid graphics? Sorry I'm pretty new to ubuntu.

5h1ro, I have made a script that automatically sets up the hybrid graphics. If it doesn't work, I've had enough experience with it and I would be happy to help.

As for the HDMI, after the Hybrid Graphics are set up, if you are on the ATI Card, the HDMI out work, but I haven't gotten sound through HDMI to work. I have just been plugging in speakers through the Headphone Jack when using HDMI.

As for the TouchPad, it works partially, but it has to be used a bit different than windows. You can't keep one finger on the left click area, and move the mouse. It causes the cursor to jerk around wildly. Also, the right click doesn't work unless you click in the very far right. But you can enable multi-touch which allows you to right click by tapping with two fingers. The disable button in the upper right doesn't work, but I created a hotkey to disable the touchpad if needed. So basically the touch pad works well enough, it's just a bit of getting used to.

The only other hardware that I haven't gotten working is the Finger Print Scanner. 

Everything else (WebCam, Mic, speakers, touchscreen, VGA out) works perfectly!

You can find the scripts to install the Hybrid Graphics over at this thread: [http://ubuntuforums.org/showthread.php?t=1716403](http://ubuntuforums.org/showthread.php?t=1716403)

I'd be happy to help if run into any problems!
