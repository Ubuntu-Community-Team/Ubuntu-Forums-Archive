### Post by pastalavista on 2009-06-06
You can easily make an Ubuntu (or any other) bootable CD iso file into a bootable USB flash with [Unetbootin]("http://unetbootin.sourceforge.net"). It will even download it for you or you can use an iso you've already downloaded.
