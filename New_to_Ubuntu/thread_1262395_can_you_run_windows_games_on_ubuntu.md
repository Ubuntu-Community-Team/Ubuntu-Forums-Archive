### Post by tuxxy on 2009-09-09
You will not be able to run those gmaes natively in Ubuntu, you will however be able to play some games using WINE and you can check specific games compatibility at [http://www.winehq.org/](http://www.winehq.org/)

Have to warn you though it can take while for new games to work with WINE and even then don't expect the full selection you get with Windows or the same gaming experience you would in Windows either.

### Post by oldsoundguy on 2009-09-09
directly in any Linux build?  NO! (there ARE some games that have been written for Linux .. but I only use the card games and such that come with Ubuntu.)

Using something like installing WINE, yes to some Windows games, but sometimes performance suffers.

And there are PAID emulators that will pick up some others.

Linux is NOT Windows, it does not run Windows programs (even those you PAY for) easily.  The command structure is different as is the file system.
And that is what makes Linux virtually bullet proof as far as malware!

BUT, when you only have 2% of the market, the game developers are not going to pay much attention to you.
