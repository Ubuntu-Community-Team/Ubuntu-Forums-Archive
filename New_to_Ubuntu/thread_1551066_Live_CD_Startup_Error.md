### Post by Ozymandias_117 on 2010-08-11
During boot, press esc during the first purplish screen. Then select your language. Now press f6 and select "noacpi" and attempt to boot like that.

---

### Post by daniel922 on 2010-08-13
Operating system is fully working now, thanks. Putting the acpi=off line into the boot commands did it.
