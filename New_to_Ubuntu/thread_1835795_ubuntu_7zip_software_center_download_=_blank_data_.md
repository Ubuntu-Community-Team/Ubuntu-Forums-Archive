### Post by cariboo on 2011-08-30
The name of the package you want is p7zip-full. To compress a file/directory, just open nautilus, and right-click on the file/directory, and select compress, once the file archiver window opens, choose the type of file compression you want. BTW, does it have to be a zip file? tar.gz is the standard in most Linux distributions, most Windows file compression programs will open a tar.gz archive.
