### Post by frank cox on 2010-03-25
puppylinux.org

Puppy Linux is a Linux distro just  as Ubuntu is. It runs totally in ram so it is about as fast as an operating system gets. It is simple to install to a flash drive and there is no reduction in speed, perhaps a slightly longer boot. Windows or Ubuntu require on virtual memory to function, puppy does not as long as you have twice the ram of the os. With no mods the entire os is 104mg so it will fly with only 250 mb of ram. 

I am not knocking the other sytems although I like Ubuntu over windows for lots of reasons, paticularly the price! Ther are a few things I have to have windows for but probably less than 5% of total . Ubuntu is a powerful system that lends itself to loading up with programs so I use it for testing software and for programs I don't use all the time.
Most of the time ,say 65% I use Puppy because it is blinding fast. The main Puppy build I use  is highly modified but will still run in ram with one and 1/2 gigs of memory.
The only reason it is so big is that I cannot find a program native to English that will replace the E-Sword bible program to my satisfaction and WINE {the Windows emulator that is not an emulator} is a resource hog. An interesting fact is save one rather insignificant feature it runs remakably better in Ubunt or Puppy Linux that its native Windows.

Depending on what you are trying to accomplish you may not even need the stick or to even to install Puppy as it will run just as fast off the install disk.

If all you want is to be able to boot in order to save the files you failed or where unable to back up the disk is all you need. One of the best kept secrets amoung geeks is that Puppy will often  allow you to save data from a corrupted Windows drive as well or better than the most expensive recovery software  and even repair the system sometimes, the techs I know never leave home without it.

If you intend to use it for any length of time as your main os , or you get hooked on the speed :} , you will want to uninstall the default browser SeaMonkey and replace it with whatever you like or at least upgrade it. There is a thread on the Puppy Linux forums you can find by typing "FireDog  stylish faster browser"  in Google and in the first post you will find a link explaining how to uninstall SeaMonkey. It may seem daunting at first but if you back up the Pupsave file which is less than 100 mgs and only takes 2 minutes you can start over easily . Then find the .pet file [basically the same as .deb} for the browser you want and click on it and the install is automatic. My favorite is FireDog which was written just for Pupy but it is based on FF 2.0 so I upgrade it to FF 3.56 and keep the FireDog interface. But there are pets for Opera,FF {use 3.56} ,Gogle Chrome etc . if you prefer. 

If you are forced to use dialup Puppy is much more user friendly and modem freindly than Ubuntu and twice as fast on a dialup  as Windows  You may still need a hardware modem. I have 5 winmodems that work out of the box with Puppy. It has an impressive array of programs and utilities, the interface scanner is cool! It often not only tells you the dev an vendor numbers of the modem,etherling card, video card etc. but the manufacturers name and the brand name of the devive. I use it to find windows drivers.

Anyway download Puppy 431 and burn the image to cd and set the bios to boot from cd.
Answer the questions about time zones etc and when you get to the video I would choose XVesa over Xorg when it asks and set the resolution to a moderate number. Tweaking X can be difficult, not usually thankfully, and XVesa is a no brainer. If you intend to watch movies you will need to set up Xorg for the best picture but why bother if it is basically a rescue disk and if you intend to carry  it around so you can boot into your own operating  system on someone elses machine you won't want to have to configure x everytime. XVesa is not bad and will work on virtually anything. 

To install it go to menu [same as start in Windows} then setup then g-parted. To install to a hard drive create a small ext3 partition  [ .75 to 1.25 gb or so} and also a fat32 partition to exchange files between Linux and Windows. You can share files on NTFS with Ubuntu or Puppy but it can wreck your symlinks. I recommend a frugal install because it is so easy to backup and restore and you don't have to configure the grub but suit yourself.
For a flash drive you can create the first partition as fat32 and may be able to still use the flash to transfer files to Linux and Windows depending on your bios . The second should be ext3. If you intend to boot from Puppy be sure to flag the drive as boot.

On the desktop there is an icon that says install . Click it and follow the directions.

Enjoy!




Good luck!
