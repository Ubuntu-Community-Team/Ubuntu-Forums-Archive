### Post by Kopkins on 2013-02-12
If you don't wish to be able to login to the gnome-shell then you can remove it.

But as the others have said, Unity is essentially a custom gnome 3 shell.

To remove the GNOME-SHELL

```
apt-get remove gnome-shell
```
