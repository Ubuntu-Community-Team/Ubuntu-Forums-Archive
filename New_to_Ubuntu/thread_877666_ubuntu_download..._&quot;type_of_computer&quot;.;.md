### Post by bumanie on 2008-08-02
You can use either version. In the past it was suggested that new users should initially use the 32bit version, until they learn how the OS works. Lately the 64bit version of ubuntu is much more stable than it once was  - but as a first time user, I would err on the side of caution and use the 32bit (x86 architecture) to start with.
