### Post by audiomick on 2012-06-26
Wine allows a Linux OS to run Windows programs by providing a layer between program and OS that makes the program "believe" it is running on Windows, i.e. provides the program with all the things it expects from the OS.

More here

[http://en.wikipedia.org/wiki/Wine_%28software%29]("http://en.wikipedia.org/wiki/Wine_%28software%29")
