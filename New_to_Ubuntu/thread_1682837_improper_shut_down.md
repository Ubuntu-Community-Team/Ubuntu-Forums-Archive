### Post by ajgreeny on 2011-02-06
I'm not sure I follow what happened here, but in any case, rather than do a hard shutdown it is always worth trying holding Alt Gr+Print Screen and then slowly typing REISUB (I find it needs O on my machine at the end to halt/poweroff) to shutdown, even if you can't see anything.  Much better than using the power button, if you can get it to work, but it does not work all the time.
