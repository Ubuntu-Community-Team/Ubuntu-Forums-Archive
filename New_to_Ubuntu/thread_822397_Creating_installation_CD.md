### Post by The Cosmic Hobo on 2008-06-08
> **bigken said:**
> if your using windows use infrarecorder 

[http://infrarecorder.sourceforge.net/](http://infrarecorder.sourceforge.net/)

You could also use [CD Burner XP](http://cdburnerxp.se/), which I've found is great for burning .iso files.
Remember, whichever app you use, there should be an option (often in the file menu) to open or burn an .iso file.

**Whatever you do, do not add the .iso file to the contents of a data CD!** Or you'll just end up with a copy of the file. You need to specifically use the .iso as the source of the disc to make a bootable CD.
