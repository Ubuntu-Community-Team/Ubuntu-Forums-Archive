### Post by dcsoldschool53 on 2012-06-18
Yes, you are right, using the ctrl key and right clicking the mouse will give you the options to delete the panel. On some systems that will not work, they will have to click the ctrl + windows key and right clicking using the mouse. The same technique is used to add to gnome-panel, remove item from the gnome-panel bar, and show the properties of the gnome panel.
