### Post by bobnutfield on 2008-09-17
Hello,

I don't believe this is being caused by your external monitor, but rather Ubuntu is having difficulty with the graphics chipset in your laptop.  I believe there is an option to start Ubuntu in safe graphics mode on the Live CD.  This usually gets you to a desktop by loading the generic Vesa driver.  Check the other boot options as well as there are other options to deal with graphics are difficulties.
