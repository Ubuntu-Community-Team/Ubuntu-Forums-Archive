### Post by jimmy the saint on 2008-12-13
The swf is a shockwave flash animation.  The exe is a windows executable.  you need to concern yourself with the swf.

Open add/remove applications (applications>add/remove)
make sure that the dropdown next to "show" is "all available applications.
search for swf in the search bar
install "Swfdec Flash Player"

Then open the .swf file with the player.
