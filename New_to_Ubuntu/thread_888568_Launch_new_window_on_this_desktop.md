The idea of nc is that you can launch it from the command line on a specific file or files. This works, but if there is an existing window, it launches the new file as a tab in that window, even when it's on a different desktop. 

On my previous dual-head setup, using Xinerama, if I launched nc on a file on the same desktop, it would attach it as a tab, but if it was a different desktop, it would launch in a separate window, which is the behaviour I want. Does anybody have any idea how to fix this? I can't go back to Xinerama because it doesn't seem to work on this machine...
