### Post by MrHippocampus on 2008-04-27
Assuming you are using an nvidia card with the nvidia driver, all you should have to do is plug in the projector, run nvidia-settings, then go to "X Server Display Configuration". From there you should be able to configure the projector (if it doesn't show up try pressing "Detect Displays"). You should also probably disable compiz just to make sure it wont affect anything.
