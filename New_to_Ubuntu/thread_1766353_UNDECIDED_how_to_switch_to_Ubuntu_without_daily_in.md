### Post by Anuovis on 2011-05-24
Peter09 has a point. 

You don't need internet to install Ubuntu and can easily grab a few applications from another computer if you need them. Deb files are easiest, just carry them to your Ubuntu machine and run from there.

As for actual updates, they are useful for two reasons. One is security. Since your Ubuntu will be offline most of the time, it is probably a lesser concern for you. Another is about convenience which comes (but not always) with newer versions. However, you can easily get the things done with slightly outdated applications. I never rush to install the latest versions and don't even bother with every Ubuntu release there is. As long as my machine runs and gets things done, who really cares about that version number.

So, a daily internet connection is not really needed. It is preferable and makes life easier, no argument about that, but not really essential.
