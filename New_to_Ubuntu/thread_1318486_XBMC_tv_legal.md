### Post by matthewbpt on 2009-11-18
I wouldn't aggre that viewing the content through a browser is better, given the poor performance of flash in linux. The content that xbmc gets from these sites has much better performace because xbmc is doing the decompression of the video, instead of the flashplugin embedded in the browser. For example, I can watch BBC iPlayer HD content on XBMC with virtually 0% cpu usage, because xbmc uses vdpau to decode these streams, while in the browser it's nearly unwatchable.
