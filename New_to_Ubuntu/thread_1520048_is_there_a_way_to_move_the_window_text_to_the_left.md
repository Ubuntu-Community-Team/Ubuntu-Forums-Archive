### Post by -kg- on 2010-06-29
OK, you're talking about the Window Title Bar; the bar at the top of the Window which contains the information about what is running in the Window itself.

I *think* that it's a matter of programming.  Default for Lucid (10.04) is with the buttons (the "- [] x" that you posted) to the far left, with the text following it.

Have you changed your themes?  Have you tried changing it and trying others?  The default theme for Lucid (which I have) is "Ambiance".  Click on "System/Preferences/Appearance" and select "Ambiance" (should be the first in the list).

Or if the default is *not* what you want, try some of the others.  I understand your point; at first I was less than pleased with the "- [] x" buttons being to the far left, but I'm used to it now.  One of the other themes in the list should be with the buttons to the right, with the text to the far left.
