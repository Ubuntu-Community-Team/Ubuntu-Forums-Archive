### Post by Locutus_of_Borg on 2008-12-17
well, if everything is frozen, you can safely restart by holding the Alt and SysRq keys, then typing r-e-i-s-u-b (busier backwards to remember it easily)

you could also map gnome-system-monitor to ctrl+alt+del for a more task mnager like experience, but that doesnt include a reboot option

### Post by HotShotDJ on 2008-12-17
I have found that most crashes that I have experience in Linux involve the display manager.  Typically, I can resolve that with a simple Ctrl+Alt+Backspace which restarts Xorg.  Since I so rarely have system crashes that involve the entire system, I've never looked for a recovery method, although I've heard the Alt-SysRq method described above works well.
