### Post by Jeremy23 on 2008-07-28
If all you want to do is talk to a person, I recommend you both use [Ekiga]("http://ekiga.org/") instead.

Ekiga is a free standards-compliant VoIP application that works on both Windows and Linux.

Ubuntu has Ekiga preinstalled, and you can [download the Windows version here]("http://wiki.ekiga.org/index.php/Windows_Users"). The Windows version is considered Beta, but it would be a heck of a lot more stable than a Linux user running Ventrilo under Wine.
