### Post by waspbr on 2008-12-11
right click on the media file you want to play, then click on properties. 

On the properties menu, go to the "open with" tab 

there you should find all the players that have been associated with the file type, click on VLC.

Close the properties menu. 

now ubuntu will always use vlc to open that particular file type.
