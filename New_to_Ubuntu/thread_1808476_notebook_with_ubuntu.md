### Post by Lars Noodén on 2011-07-20
The desktop version will work just fine.  If you want to experiment a bit, you can try putting [Lubuntu](http://lubuntu.net/) on it.  It is lighter weight in regards to system requirements.   Should you change your mind, you can change that to plain Ubuntu later, if you want, by adding the package ubuntu-desktop.
