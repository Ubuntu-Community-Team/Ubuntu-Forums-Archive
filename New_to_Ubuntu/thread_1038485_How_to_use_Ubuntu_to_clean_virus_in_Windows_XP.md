### Post by Svensk023 on 2009-01-12
well you can download a free anti-virus program such as clamTK by searching in

Applications > Add/Remove Applications

you might need to set it to search all available applications. (i think the name will come up as just "Virus Scanner")

after downloading and installing your new anti-virus software open it and under "Options" make sure to check the boxes next to "Scan Hidden Files" "Display All Files" and "No Maximum Size". 

Now under places select your windows partition (or HDD) to be scanned.

You might have to use the Recursive Scan.

It will probably take quite awhile to scan but it should find a virus, if that's what is causing your problem.
