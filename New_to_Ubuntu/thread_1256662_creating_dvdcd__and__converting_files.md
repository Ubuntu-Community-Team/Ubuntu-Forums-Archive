### Post by 3rdalbum on 2009-09-02
> **289531.EXE said:**
> ok, so the thing is:

I want to create a dvd BUT:

1. when i put in a blank dvd for some reason it doesnt come up as blank disc. in other words i put it in and it acts as if nothing was put in.

That's a bit odd. Try a different disc. If your burner only deals with one format (i.e. DVD+r) and you put in an incompatible format, this can happen.

> 2. i just downloaded the you tube downloader. all the videos come out in ".flv" format and in different parts. can anyone suggest the best program(s) to a.)convert the ".flv" to whatever it is that it needs to be so i can watch it on an average dvd player and b.)to edit the film so that i can create a continuous film.

You want a video editor for that, really. Kdenlive can help you splice the videos together and create a DVD.

> I want to burn cd's BUT:

1. every time i burn music on a cd the cd comes out in mp3 format. is there a way i can burn it as aregular cd?

Don't drag the MP3s onto your blank CD, because it will assume you want to create a data disc containing MP3 files (what's known as an "MP3-CD").

Go to Applications > Sound And Video > Brasero Disc Burner and click the Audio Project button. Now drag your MP3s into the Brasero window and they will become part of an ordinary audio CD.

> 2. itried copying a cd but when i tried playing the copied cd it didnt play. I want to be able to copy a cd toa blank one so thatit comes outas the one mentioned in the first part of this "cd" section.

Did you use the "Disc Copy" function of Brasero? Also, some CD and DVD players have trouble with some brands of recordable CD; try using a name brand like Verbatim, TDK or Sony.
