### Post by Terl on 2009-01-09
Hi, and welcome :)  

If you are willing to read and willing to learn, yes, Ubuntu can be for you.  I would recommend reading the installation guides [found here.]("https://help.ubuntu.com/8.10/index.html") and do a little looking about.  Linux is not hard, but, it is different that windows.  As long as you go into it knowing it will be different and still with that willingness to learn, you will have fun.

Don't worry about "coding".  You will be able to do a lot without a lot of commandline stuff.  When you are ready there are many free resources on the internet to learn command line and scripting.

Before I started, I read about Linux in general using the [Rute Users Guide]("http://rute.2038bug.com/index.html.gz") and then some reading on the distribution before I even stuck the disk into the drive.  It helped to get my feet wet.

And, yes, you can set it up on your spare drive and can dual boot.  And don't be afraid about "breaking" Ubuntu; we've all learned from various boo-boos :D

EDIT:  Reference WoW-you can use WINE and run WoW in Linux.  Many windows programs can be run using WINE or other methods.

### Post by ugm6hr on 2009-01-09
For basic computing requirememnts such as yours, Ubuntu is perfect.

Just remember it is different than Windows, but equal (if not more) productivity is possible once you find your way around.

The Start here link below has a few further links with interesting reading.  Nevertheless, the psychocats website is generally a good place to start (previously recommended).

Once installed, I have no doubt you will have no problems with Ubuntu.  However, while installing a dual-boot system is easy if you know how, it is also possible to make a mess of your hard drive.  Make sure you read up on this process first, problem areas include: where to install Grub; installing Ubuntu on the correct HD with a swap partition.

My suggestion is to change your boot priority in BIOS to CD / Ubuntu HD / Windows HD **before** installing Ubuntu, then the default options should work (but ensure you read everything rather than just clicking YES to everything).
