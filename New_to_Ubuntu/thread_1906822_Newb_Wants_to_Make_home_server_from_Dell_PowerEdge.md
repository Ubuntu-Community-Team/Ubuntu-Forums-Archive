### Post by lechien73 on 2012-01-10
Hi & welcome to the forums,

You should certainly be able to install Ubuntu Server on your PowerEdge 1300 (depending on the amount of memory and hard disk space available).

There are plenty of tutorials out there for accomplishing what you want to do, so if you put in the effort to learn, then you should be able to get it working satisfactorily.

You can set up samba shares for your Windows clients in the house, and there are several programs available which offer media streaming.

Probably the best thing to do is to try and read up on each task that you want to accomplish and, if you need help with any specific bit of it, then post back here :)

Have fun!
