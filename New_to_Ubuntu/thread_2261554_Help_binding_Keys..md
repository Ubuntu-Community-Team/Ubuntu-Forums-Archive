### Post by JeQhdMD on 2015-01-20
If using the Unity desktop, it's simple to do key-bindings.

System Settings>Text Entry>Keyboard Settings (lower right corner of Text Entry window)  (or doing a dash search for "keyboard entry" will get you to same place).
