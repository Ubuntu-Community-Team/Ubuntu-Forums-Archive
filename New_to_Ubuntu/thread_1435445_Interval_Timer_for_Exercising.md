### Post by nugget659 on 2010-04-02
I think he means something that can time something like tabata intervals.

Example: 1 tabata interval
20 secs work countdown with 3-2-1 beep warning cue
10 secs rest with 3-2-1 beep warning cue

gymboss.com sells a handheld timer with the following features:
#  One or two different time intervals from 2 seconds to 99 minutes
# Auto mode keeps repeating through intervals
# Set up to 99 rounds
# Manual mode acts as countdown timer
# Stopwatch Function

Replication of these features in software would be nice for those of us who workout close to our pc.
