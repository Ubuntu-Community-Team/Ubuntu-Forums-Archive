### Post by grahammechanical on 2015-01-07
You have something seriously wrong with this machine. How can I say anything else?

I cannot offer any advice without you providing more detail about the machine and the operating systems already installed on it. Were there problems with this machine before you tried to install Ubuuntu? What version of Ubuntu are you trying to install? That machine might not be compatible with the latest version of Ubuntu.

When you first switch on the machine, do you get any Power On Self Test (POST) messages about something being wrong? A computer motherboard will shut down is there is something wrong with the system memory (RAM) or with the CPU or if the machine is overheating.

These are all guesses.
