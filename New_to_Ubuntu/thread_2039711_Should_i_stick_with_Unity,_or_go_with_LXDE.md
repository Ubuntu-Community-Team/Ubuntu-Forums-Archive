### Post by TheFu on 2012-08-09
If PC is slow or has a weak GPU or you just don't want any GUI delay or you prefer a WinXP-like interface, use LXDE.

Otherwise, use Unity.

Why not install both and try them out? It isn't a big deal to do that.  Go with the default install to get Unity, then after that's all done, type in:
```
 sudo apt-get install lxde
``` to get LXDE installed.
On the login screen, there's a tab where you can select the DE you want at login.

Enjoy both for what they are.

### Post by spjackson on 2012-08-09
While I agree that "try it and see" is excellent advice, I would point out that in my experience, Lubuntu is better than Ubuntu+LXDE, and to a lesser extent, Xubuntu is better than Ubuntu+Xfce. It is true that installing Ubuntu and changing the desktop gives you a flavour of the alternative desktop experience. However, there is extra tweaking you have to do to get the smoothness of a dedicated system, for example, keyring integration.

How much RAM do you have? My rule of thumb is that 1GB is good for decent performance with Xubuntu; less than that and Lubuntu is probably a better bet.
