### Post by Temposs on 2010-04-01
You can use a USB stick just like a CD/DVD. Just burn the ISO onto the USB stick with UNetbootin: [http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

You can boot the LiveCD from the USB drive. You just have to set your boot order in your BIOS just like you would with a CD, except you should set your USB drive to boot first.

Since you seem to have a netbook, if your screen size is 10 inches or less, you should try the Ubuntu Netbook Remix. That is the best version for that kind of laptop.

Here's a link to the Ubuntu Netbook Remix: [http://www.ubuntu.com/getubuntu/download-netbook](http://www.ubuntu.com/getubuntu/download-netbook)

It is an official version of Ubuntu. It works quite well.

### Post by Temposs on 2010-04-01
It will be different for every different kind of machine, because each machine has different parts which have different levels of compatibility with Ubuntu. Ubuntu tries to cover everything, but it doesn't all the time. Presumably your machine came with Windows. So, the computer maker designed the system to work with Windows, not Ubuntu. If you want a machine that works entirely with Ubuntu, then you should buy a machine that comes with Ubuntu preloaded, as the system maker will build it with Ubuntu compatibility in mind, and it will work 100%.

Here are some places to get a computer preloaded with Ubuntu:

### Post by debianmike on 2010-04-01
I don't myself prefer the Netbook Remix interface, and I have not found any actual performance improvements (not kernel changes specific to the Atom CPU, not modifications for SSD drives, etc).

So the UNR is really "just" ubuntu with a non-Gnome GUI.

My latest incarnation of my own netbook (Lenovo s10) is Ubuntu, then apt-get install XFCE4.

For some reason XFCE4 doesn't have all the other options I am looking for by default (I forget exactly what was missing now).

I've also turned off icons and toolbars, switched on compiz for Expo and the Widget layer, and added gnome-do.

My small netbook screen is 100% free from "junk" and dedicated completely to application windows; I can start or open anything with Gnome-do; and things like a battery meeter and temp gauge and a clock are all easily accessed via F9 (the widget layer and screenlets).

Have I rambled enough yet?
