System > Preferences > Appearance

select the theme and click customize and then go to the colors tab, it's the window background color you are looking for.

It somewhat looks like you are going for a darker theme, so I might suggest you check out the Ubuntu Studio theme which is configured like this by default, and could be an easier base to start from.  You'll have to download this from synaptic, by installing ubuntustudio-look
