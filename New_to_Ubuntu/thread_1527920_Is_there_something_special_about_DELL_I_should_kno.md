### Post by ST3ALTHPSYCH0 on 2010-07-13
Without looking at your other post, I will say that I had a machine that would not install Windows XP Home (it would install, but then either BSoD or go into an endless login/ out loop) but XP Pro took the first time. I've used both discs (with different COAs) on multiple occasions, so I don't know why this machine was different. Maybe try using UnetBootin to create a bootable thumb drive from your .iso and install from USB? (Yes, I have had machines that would boot from USB that wouldn't from CD and visa-versa. BTW, the same .iso was used to create the CD as the USB drive).
I can also say that I've had machines that wouldn't boot certain versions of Ubuntu that would boot others, now that I think about it. If the USB trick doesn't work, you may have to install a 9.xx release and upgrade to 10.04.
