### Post by 3rdalbum on 2009-05-04
Did you know that Kubuntu is just Ubuntu with a different desktop?

As such, you can just install the "kubuntu-desktop" package inside Ubuntu, and then you can choose at the login screen whether you want to run Gnome or KDE.
