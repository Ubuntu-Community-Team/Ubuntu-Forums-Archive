### Post by dFlyer on 2011-06-21
You may have to add canonical partners. To do so from ubuntu software center start program go to preferences and software sources. make sure that canonical partners is checked under other software. Close the program and run sudo apt-get update from the command line. Restart USC and search adobe flash.
