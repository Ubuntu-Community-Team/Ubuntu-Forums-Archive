### Post by ugm6hr on 2009-07-25
Try plugging in; it should auto-detect.

Then set up the connection with Network Manager.  You may have to right-click and edit the connection if you have already tried setting it up.

Settings:

Number: *99# (should be set by default)
Username: (blank)
Password: (blank)
APN: goto.virginmobile.uk
