### Post by CSNate on 2013-03-19
1. Obtain the source code.
2. Install build dependencies.
3. Extract source code.
4. cd into source directory and configure the build with make menuconfig.
5. run make to build the kernel
6. make install to install the new kernel.  That is a very general process.

See: [https://help.ubuntu.com/community/Kernel/Compile](https://help.ubuntu.com/community/Kernel/Compile)
