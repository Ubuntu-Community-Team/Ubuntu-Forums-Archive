### Post by 3rdalbum on 2013-06-22
You can open these files in Gedit (the text editor) and copy the URL included in them. You can then visit that URL and bookmark it in the web browser.

Alternatively, you can open the Main Menu program and create a new menu item. Set the command to "gnome-open " and then the URL, for instance:

```
gnome-open http://www.google.com
```

The link to the website will now appear in the Dash, and you can drag it to the Launcher.
