### Post by P4man on 2010-10-19
If it is a wubi install, where you installed ubuntu inside the windows partition in c:\ubuntu then the short answer is: you cant. One of many reasons not to use wubi if you dont strictly have to.

If you did a regular dualboot install, then its easy. Boot the live cd, run partition editor (gparted) from the administrition menu, right click the swap partition and select 'swapoff'. THen unmount the other partitions and resize. Beware this can take a very long time.
