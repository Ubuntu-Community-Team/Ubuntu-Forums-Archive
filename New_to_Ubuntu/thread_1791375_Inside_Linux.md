### Post by tcsmith1978 on 2011-06-26
Hi,

Hopefully this will be a good place to put this post. I have created a new wiki site at [http://linux.titan21.co.uk](http://linux.titan21.co.uk) which is an attempt to "map" out the file system in Linux.

This may seem like a strange idea but I have never been one for manuals and following tutorials (unless I'm really stuck!) and so I am hoping that this wiki will show what files are in a Linux distribution and what they are used for. I'd rather get stuck in and have a nose around rather than all the theory. Sometimes the best way to learn is to get stuck in.

I've only done a couple of entries so far but have found it really interesting (check out [http://linux.titan21.co.uk/index.php//bin/busybox](http://linux.titan21.co.uk/index.php//bin/busybox)).

If you think you can contribute to this project please head on over to [http://linux.titan21.co.uk](http://linux.titan21.co.uk) and start writing some entries. If you have questions - please ask them in the discussions and I'm sure someone will get back to you with an answer.

I'm open to new ideas and input so please feel free to contact me on the site (username: Tim)

I hope to see you there.

Kind Regards,

Tim
