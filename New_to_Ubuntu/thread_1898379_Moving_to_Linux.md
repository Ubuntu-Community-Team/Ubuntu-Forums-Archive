### Post by TBABill on 2011-12-21
You shouldn't "need" any programming language to enjoy, tweak, fix, etc. on Ubuntu. It's a common fallacy that someone needs that level of computer experience to manage Linux. If you want to be a developer, that's a different case entirely.
 
As far as what to download, it depends on how you want to run it. If you want Ubuntu as your only operating system or as a dual-boot with Windows, do the download and burn to CD/DVD or USB. If you want to use it within Windows (called a Wubi install), then choose the other. I can't offer insight into using Wubi since I have always installed it to its own partition outside of Windows.
 
Best of luck and welcome to Ubuntu!

Programming knowledge is 100% irrelevant to whether or not you should consider moving to Ubuntu (unless you're a programmer who may need OS-specific tools). However, Linux is a great OS for programmers and has loads of tools available. Repositories make things pretty convenient, too. 

This depends on what you'd rather do. I'd recommend against using the WUBI (standalone app) method because it depends on Windows and is known to be buggy. Instead, I'd recommend using a spare USB drive and using that to install Ubuntu on your PC alongside Windows. While things shouldn't go wrong, you should always do backups beforehand just in case something does. Check the above posts for info on how to create a bootable USB drive and set up a dual boot.
