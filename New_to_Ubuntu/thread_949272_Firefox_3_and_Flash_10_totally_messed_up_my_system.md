### Post by TeaSwigger on 2008-10-16
Perhaps it's a "corrupt" profile. You're using ubuntu? In Nautilus (the file manager) open to your home folder and enable it to show hidden files. You should then be able to see a .mozilla folder which should have your "profile" including your bookmarks (if needed see firefox's site for more info on the profile folder). Just for a test, rename it to mozilla (remove that period from .mozilla, the period at the start makes a folder "hidden"). Install Firefox 3 if it's not already and try to start it. It should start and make a new .mozilla folder while it's at it. If it works you can import your old bookmarks from your old profile folder if the bookmarks survived whatever messed up. Again, Firefox's site will have help on how to do that. I'm offering these suggestions not knowing if you know any or all of this so sorry if it's obvious. At least it might give you some ideas to try. Good luck.
