### Post by RomeReactor on 2008-06-30
Hi. You can dual boot, and have both systems installed; when you power on the PC, it will give you a choice of which OS to boot into. You can read more about [dual booting here]("https://help.ubuntu.com/community/WindowsDualBoot"). Otherwise, if you want to have only Ubuntu installed, you'll need to install those programs in Ubuntu using a compatibility layer called Wine. Note that not all windows programs will run this way. Read more on Wine [here]("https://help.ubuntu.com/community/Wine"). However, it would be worthwhile to search for Linux alternatives to those windows programs, if possible. More on that [here]("https://help.ubuntu.com/community/WindowsApplicationsEquivalents").
