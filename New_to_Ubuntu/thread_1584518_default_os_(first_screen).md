### Post by Mark Phelps on 2010-09-29
Answer depends on HOW you installed it ... which you failed to explain.

Is your first screen a text screen offering Windows and Ubuntu? If so, you most probably installed via Wubi -- and you can't change that screen using Ubuntu.

If you're running Win7 (which I also suspect), your best bet is to go to the NeoSmart Technology forums, download and install EasyBCD.

Run that and it provides an option to select the default OS.
