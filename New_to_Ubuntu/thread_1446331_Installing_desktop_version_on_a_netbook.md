### Post by snowpine on 2010-04-03
The functionality is the same, but the user interface or "look" is totally different. Ubuntu uses a traditional Gnome desktop; Ubuntu Netbook Remix uses a "launcher" interface for small screens.

Compare the screenshots:

[http://images.google.com/images?q=ubuntu+screenshots](http://images.google.com/images?q=ubuntu+screenshots)
[http://images.google.com/images?q=ubuntu+netbook+remix+screenshots](http://images.google.com/images?q=ubuntu+netbook+remix+screenshots)

I personally prefer the desktop version, even on a netbook, but it purely a matter of taste.
