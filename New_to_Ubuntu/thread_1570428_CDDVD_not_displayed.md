### Post by Peter09 on 2010-09-08
This is most probably because the CD will be mounted as the root file system - called / .
 
 
So if you go to the root file system then you are looking at the contents of the CD. 
 
Under normal circumstances the CD would not be involved in running the operating system and would be seen as a normal icon.
