### Post by powel212 on 2009-10-27
This is an FYi

I recently saw a post about getting a new motherboard and is it needed to re-install Ubuntu. One reply stated it was unnecessary. I was surprised.

I tried it. I took the Hard drive with an Ubuntu install out of an old Copermine 700mhz intel box and stuck it in my new socket 775 intel board and booted it.

It booted no problem and is running fine.

So there you go. New motherboard, just plug in your Ubuntu drive and off you go.
