I do understand what you are asking for - you want the Windows 7 system tray that hides icons.  That is what all my Windows 7 systems do as well.  That comes with Windows 7.
 
When you are in Ubuntu - things are not exactly the same, and these add on tools are close; but your are quite correct - they are not Windows 7 exact matches.
 
Ubuntu is built to work differently.  The designers actually dropped the tray out of the Ubuntu release a while back.  The earlier suggstions were to 'ease' your transition.
 
Windows 7 GUI is great. Ubuntu GUI is great. OSX GUI is great.  But they are not the same.
