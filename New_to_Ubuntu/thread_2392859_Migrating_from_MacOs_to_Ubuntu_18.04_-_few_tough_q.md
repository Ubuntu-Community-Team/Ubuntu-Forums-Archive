### Post by yancek on 2018-05-26
Apple of course, is extremely proprietary and is designed to only run their software.  It is possible to install Linux on a Mac although it will be more difficult than a standard PC.  Never having used a Mac myself, I can't suggest anything but I expect "Mr. Google" can help.

Macs are a very small percentage of home computer systems so most if not all Linux systems will not have the tools installed by default to access hfs filesystems but they can be easily installed.
Obviously, you will need to have unallocated space somewhere on which to create a Linux partition with a Linux filesystem to install Ubuntu.
