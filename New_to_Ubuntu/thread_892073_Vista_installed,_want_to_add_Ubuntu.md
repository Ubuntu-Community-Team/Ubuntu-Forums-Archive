### Post by pteri498 on 2008-08-16
I've done the same thing before, and I never had any special problems. Basically, there was a restore partition on the hard drive that I left alone for safety. Mostly, I would make sure to build the restore cd's for your computer before you begin.

When I did this, I shrunk my Vista partition and just install Ubuntu onto a logical partition, if I remember correctly.

Only thing to do then is to make sure that you don't mount that restore partition (do that during installation), and that's about it.
