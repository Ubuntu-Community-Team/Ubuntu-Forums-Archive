### Post by ronnielsen1 on 2011-01-27
[http://lightlinux.blogspot.com/2008/06/top-10-of-lightweight-linux_24.html](http://lightlinux.blogspot.com/2008/06/top-10-of-lightweight-linux_24.html)

The processor is fine but the memory is limiting you. If you upgraded the memory you could run anything but otherwise check out one of the alternatives in the link above. Xubuntu will supposedly run on 256 M but you won't be happy. I'm not a xubuntu fan. You would be better off using an alternative like puppy linux, antiX, or ubuntu with a lighter weight window manager like icewm or fluxbox
