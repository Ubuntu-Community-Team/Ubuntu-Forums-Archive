### Post by buzzingrobot on 2015-11-24
Tapping the Windows key ("Super Key" in Ubuntu parlance) will display a window containing icons in various sections. A row of smaller icons runs along the bottom.  Explore what's there by clicking around with your mouse.  Applications can be launched by clicking their icons.

Once an application is running, its icon appears in the Launcher to the left. Right clicking on that icon will show an option to keep it in the Launcher.

The Launcher is intended to house icons for the tools used most often. It can hold more icons that it appears, at first. 

Searching for something like "Beginner's Guide to Ubuntu Unity" will turn up a lot of results, like this: [http://www.makeuseof.com/tag/ubuntu-an-absolute-beginners-guide/](http://www.makeuseof.com/tag/ubuntu-an-absolute-beginners-guide/)
