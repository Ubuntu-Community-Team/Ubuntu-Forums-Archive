### Post by ra7411 on 2019-01-08
If you're going to install on a recent generation highly capable machine, Ub 18.04 is a good choice. 

If you're going to use an older less capable machine, you might want to post what cpu, ram, HDs and get advice on the best version to try out.

The install usb, or dvd, is also good as a repair disk that you can boot from and do some fixes for the occasional problem - so you want to keep it around.
