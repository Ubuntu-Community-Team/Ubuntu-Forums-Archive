### Post by Denver Dave on 2011-09-30
The ubuntu download was pretty big, even for my Comcast cable connection and when I did download the file, I was not able to successfully create a bootable DVD (my  expertise probably at fault).

I was going to order a ubuntu DVD directly, but was not wild about joining launch pad (might later), so I went to a local computer bookstore and purchased "The Official ubuntu Book" which comes with an ubuntu DVD which we can run without installing on our PCs.   The DVD ran really well and boots from the DVD faster ( 2 min 45 sec.) than my Windows XP boots from the hard drive 

Once I gained some experience with the running the DVD, I created a bootable 16 GB USB stick which I'm using now on a variety of PCs (boots in 1 min 45 sec).  I really like the concept of taking my operating system and applications with me, perhaps with a separate USB stick for the data.

I've also ordered from [www.OSDisc.com](www.OSDisc.com) the 64 bit ubuntu which I can't really see a difference from the disk in the book and also several other flavors of Linux like Knoppix 6.7, Av Linux 5.0, Zorin OS 5.1 32 bit and 64 bit.  I also downloaded Puppy Linux and was able to create a bootable CD, I can't remember if I did this under Windows XP or with Ubuntu, but it is only 132 MB so much easier to download.    

After all of the above, I'm using the 16 GB USB boot drive created while running the ubuntu DVD from the book just fine (with some relatively minor issues).

Since that time I've been "pestering" this forum with questions which you can see if you search for my posts.

This is how I started 20 days ago - start by running ubuntu.

Good luck.
