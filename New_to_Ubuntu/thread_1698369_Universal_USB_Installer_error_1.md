### Post by Mariane on 2011-03-02
Reboot your computer and look into the BIOS. You should have the option to change the boot device order. Put usb keys first. 

Or if you can't see that option at all, maybe you have one of these machines which won't boot from a usb stick. If this is the case use a CD.
