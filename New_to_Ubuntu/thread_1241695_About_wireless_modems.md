### Post by 4Orbs on 2009-08-16
I used a Cricket modem for several months with Ubuntu 9.04 and it worked with no problems. The first time you boot up you'll probably need to configure the Network Manager and tell it to connect automatically (right click on the NM icon on the top panel and select Edit Connections).

### Post by 4Orbs on 2009-08-16
In my case the Cricket modem did not connect automatically the first time it was plugged in, but it was detected by Network Manager. Also, it might be mandatory to install the Cricket software in Windows first in order to activate the account. I was dual booting Windows XP and Ubuntu, and installed the software in XP before using the modem with Ubuntu. If you don't have a Windows machine to do the initial account activation, the modem probably won't work for Ubuntu (but I don't know for certain).
