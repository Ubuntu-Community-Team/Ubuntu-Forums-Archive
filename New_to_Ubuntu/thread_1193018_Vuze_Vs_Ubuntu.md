### Post by taavikko on 2009-06-21
The one in the menu's is indeed the version you installed via synaptic.

If you don't want the older version via synaptic, simply remove it the reverse way you installed it. (through synaptic)

Now, you can add the new version to the menus, by using alacarte
right click on the menu -> edit menus
An add the executive vuze file to the command, 
change the icon of your liking and that's it.

(or should be anyways :) )
