Hi guys,

I currently have ubuntu installed on a drive in the ext3 file format and XP installed on an NTFS drive. I would like to create documents in Ubuntu and download files like music etc. and then be able to read AND edit them when I'm in XP. At the moment, I am not able to view the ext3 drive in Windows XP. Is there a way I can read AND edit the documents in XP?

Thanks

---

### Post by Paqman on 2008-11-04
You'll need to install [ext2ifs](http://www.fs-driver.org/) in XP. It allows you to mount your ext3 partitions as Windows drives.
