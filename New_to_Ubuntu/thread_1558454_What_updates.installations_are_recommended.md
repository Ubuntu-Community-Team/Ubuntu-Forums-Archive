### Post by jmfal on 2010-08-22
There is always going to be a big update after a fresh install, I would do a complete update first off because there may be 200 packages to update, to many to pick and choose.

As far as apps that should be installed thats up to you and how you are going to use ubuntu

Before you update, open software sources: system>administration>software sources
put a check in the first 4 boxes and uncheck the cd box
you need to check the source box if you add a ppa repo or a source repo(repository)
click other soft ware and check all the repositories, close, reload, then update

There are stickies in the multimedia/video section that may help if you want to play dvd`s or youtube
 this should get you going 
good luck
