### Post by Sirron on 2008-08-10
Yahoo Messenger is available for linux, but it looks like it hasn't been updated in many years - so it only supports IM.

Wine's Appdb has a test report for Yahoo Messenger 9.0 Beta [_here_]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=10709"). Unfortunately, it doesn't work :(.

An alternative to using wine, is to run it in a virtual machine. You'll still be able to run ubuntu, but when you want to run Yahoo Messenger, it's a little less straightforward. So I guess it depends on whether you intend to have yahoo messenger running in the background all the time, or only open it for a phone call once every few hours.

An alternative to that is to use Skype or another VOIP service. Skype is available for linux, but they update it regularly, and the current version even supports webcam.
