### Post by tom.swartz07 on 2010-04-11
There are a TON of amazing conky config files. 


The way conky works is thus;

when you launch conky, it looks for a file in your home directory titled .conkyrc (its normally hidden because its a 'dot file', click show hidden files in nautilus file browser to see them)

conky reads this file and outputs information on the desktop or window based on this. 

There are many sites available to show you how to configure your .conkyrc file. 

Here are some:
[http://www.quicktweaks.com/2008/09/27/gmail-weather-beauty-right-on-your-ubuntu-desktop/](http://www.quicktweaks.com/2008/09/27/gmail-weather-beauty-right-on-your-ubuntu-desktop/)

If you want, You could even take my conky setup! Look at this thread here for that: [http://ubuntuforums.org/showthread.php?p=8741617#post8741617](http://ubuntuforums.org/showthread.php?p=8741617#post8741617)
