### Post by KdotJ on 2010-09-18
One thing you can look into is the global menu for GNOME. This makes it function similar to a Mac, where your menus from each program (File, Edit, View....) are taken away from each open window and is put in the gnome panel.

Check out this link [here]("http://www.ubuntugeek.com/how-to-install-globalmenu-in-ubuntu-10-04lucid9-10karmic.html") for instructions and even a video. 

It's good, I use it myself


Note, this application doesn't work with some programs... namely java ones. OpenOffice, VirtualBox etc. But fine for the rest
