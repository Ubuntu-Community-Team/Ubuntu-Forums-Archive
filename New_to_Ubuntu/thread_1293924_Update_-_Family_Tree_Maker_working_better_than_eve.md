### Post by anewguy on 2009-10-17
I thought I would update a post I made quite some time ago regarding running Family Tree Maker (in my case, version 10) in Ubuntu using wine.  At the time of my last post, I was doing okay except for printing.  Since then, I've updated the OS a version or two, and most recently I installed wine 1.1.31 from winehq rather than using the older version that is officially supported in Ubuntu.

I am please to say everything I have tried has worked great, including printing!  I haven't tried the internet link yet, as it does require Internet Explorer by default or having another browser installed in wine as best I can tell (I'll keep playing, as it may be possible to link it to the Ubuntu version of Firefox and not need a browser installed in Wine).

I'm happy!
