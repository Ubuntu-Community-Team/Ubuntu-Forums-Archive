### Post by Locke_99GS on 2011-04-15
If the server will only be a server and you're not afraid of the command line, it'd probably be best to just use a server version, as it doesn't waste resources on things like X and the DE, and the server kernel will probably be built in a way that promotes throughput, at the expense of responsiveness.

I don't know but I'd imagine the PS3 is meant to connect to Windows networks, and as such you would want to setup Samba on the server.
