### Post by lkraemer on 2009-08-01
edward51,
I'd suggest installing File-Roller first, as it will allow you to 
operate on the zipped archives as well as many other formats from the GUI.

SYSTEM -> ADMINISTRATION -> SYNAPTIC PACKAGE MANAGER

Then search for file roller. Click so it shows MARK FOR INSTALL,
then APPLY.

You might want to install and try out DOSBOX, as you should be able to
run your old verion of DOS along with WordPerfect under DOSBOX.
