As far as I can tell, those are .debs so you need to just double click them and they will install with gdebi package installer. When you download most themes from gnomelook ect. they are in a different format than that and you will be able to either drag and drop them into the "Appearances" dialog or use the method that you employed before by clicking install and navigating to them. These are deb files and thay install by double click provided that you have gdebi installed (should be by default)
