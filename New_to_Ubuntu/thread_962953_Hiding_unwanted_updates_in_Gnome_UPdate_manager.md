### Post by mcallenSchmee on 2008-10-29
You can lock the version you have installed in synaptic. Open Synaptic, select the program, and go to Package> Lock Version. No newer versions of that program will be installed.

---

### Post by spiderbatdad on 2008-10-29
You can set the update manager to download only security updates, or
if you want to lock specific versions of packages (and ignore updates to those packages), you can do so by selecting the package in synaptic package manager, then click the package menu, and chose "lock version."
