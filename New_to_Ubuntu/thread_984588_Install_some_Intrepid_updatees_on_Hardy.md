### Post by jimmy the saint on 2008-11-16
It is possible to install updated versions by looking on sites like getdeb or adding the project repositories to your list.  sometimes if the newer version has important new features they will be backported (made available to earlier Ubuntu versions).  If you are a new linux user it HIGHLY recommended to stick with the version in the repositories, as they have been rigorously tested for your configuration and upgrading can sometimes cause issues that are far worse than missing a few minor features.
