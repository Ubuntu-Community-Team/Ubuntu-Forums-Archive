### Post by wep940 on 2011-06-11
Check LIRC and what is needed as a go-between it and the device being remotely controlled.  Be aware that you will still need some sort of connection to your PC to the receiver for the remote control, and then it would probably be line of sight.  I don't know if bluetooth would work in this instance because I don't know if it has the range, plus you'd need a bluetooth enabled remote.  There are sites on the web that show you how to easily build infrared receiver for your PC - these feed to LIRC.  LIRC supports many remotes right now but it is easy to "define" a new one.  LIRC basically interfaces to the receiver and passes key codes. There needs to be another program between LIRC and what you are running (like a web browser if Hulu).  The infrared receiver is very cheap to make and easily built.  LIRC is free, however the software that acts as the go-between isn't usually free, and some can be quite costly.

I mention all of this because if you want to use an existing remote to control your computer you will need something similar.  In addition, as much as you don't want to run cables, keep in mind that a receiver - usch as for a wireless mouse, wireless keyboard and the remote control still has to be hard-wired to your PC.

I'm not associated in any way with the company - just saw in a class at our local MicroCenter - a product called SlingBox might be worth a look as well, but be forwarned - these are not inexpensive - they cost some real dollars.
