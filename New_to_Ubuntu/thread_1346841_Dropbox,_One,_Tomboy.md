### Post by LeifAndersen on 2009-12-05
If it makes you feel any better, I don't use Tomboy notes, and don't really know what it's about either, I just use a text editor (vim or OOo) for note making.

Ubuntu One is basically what DropBox is, you get 2 GB of free online storage which you can sync to your other computers.  I currently use DropBox because it's better.  (Ubuntu One really only works well on Ubuntu 9.04 and later, I have several computers running other distros and other OSes).

### Post by camaron1 on 2009-12-05
Ubuntu one is nowhere so developed as dropbox is. With dropbox you don't need the client installled in the computer to access your files. That means that you only need a browser and internet connection. With ubuntu one you need to have the client installed in a computer to access files (and there is not Windows client).Ubuntu one will probably get there but for now have a look a drop box, it is really good.

Tomboy: well that is really up to you, I don't use personally, If you don't see the point it isn't for you probably...
