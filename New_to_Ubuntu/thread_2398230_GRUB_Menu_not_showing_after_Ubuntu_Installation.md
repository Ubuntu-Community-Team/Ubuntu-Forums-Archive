### Post by yancek on 2018-08-09
It is not necessary to disable Secure Boot to install Ubuntu.

What type hardware?  Are you using an Acer machine?  If not, what do you have?  Check the link below which is the Ubuntu documentation on dual-booting with windows UEFI and compare it to what you did.

When you boot the HP, can you hit the Esc key and then do you see boot/BIOS options with the F9/F10 keys?  In the Boot Options, do you see an Ubuntu option?

You might try getting the boot repair software and running it according to the instructions on the site.  Be sure to use the 2nd option to download and do this from the installation DVD/flash drive.  When you run it, select the option to Create BootInfo Summary and post the link you get here.  That will give enough details on your system so that someone can suggestion a possible solution.

  [https://help.ubuntu.com/community/Boot-Repair](https://help.ubuntu.com/community/Boot-Repair)
