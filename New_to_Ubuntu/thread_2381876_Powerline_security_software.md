### Post by mikewhatever on 2018-01-06
Ethernet devices usually do not require "security software" to configure them. Everything is done through a web browser, you navigate to the gateway address (10.0.0.138 or 192.168.1.1 or similar), and check or uncheck available options, set or change passwords, update firmware, etc. Is there any "security software" for those adapters for any other OS?
