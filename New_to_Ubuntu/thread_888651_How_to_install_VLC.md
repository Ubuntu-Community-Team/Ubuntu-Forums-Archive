### Post by cozmicharlie on 2008-08-13
Welcome to Ubuntu and the forums.

You posted in the right place.  The "Absolute Begginner" section is for any question from people just starting out.

Since you are new I would suggest some reading.  This is a good place to start ([https://help.ubuntu.com/8.04/newtoubuntu/C/index.html](https://help.ubuntu.com/8.04/newtoubuntu/C/index.html)).

As per your questions - yes you can install vlc.  In Ubuntu, programs are called packages and all the packages are available in Synaptic (not all but most you will need).  You get to Synaptic by going to the main menu>system>administration>Synpatic Package Manager.  The first time you do this you should go to the Synaptic menu>settings>repsoitory and it opens to a tab that has a line for the main, universe, restricted and multiverse.  Make sure all these are checked.  Uncheck the cdrom if it is checked.  Then go to the third party software tab and check all the boxes.  Close and a dialog will appear just hit OK then on the menu hit reload.  Now you are ready to install packages.  

Search in Synaptic for vlc.  Then find the package vlc and check it.  A dialog will come up that says "mark for installation (it may have a bunch of other packages listed also - these are what are called Dependencies that is they are needed for the package to work), click OK then in the menu click "Apply" and now vlc will be loaded. Once it is loaded go to the menu>applications>Sound and Video and you should have a start icon for vlc.

Winrar is not available but Hardy comes native with an unrar program already installed.  All you need to do on a file is right click>extract here and it will extract tar, rar, zip files.  If you do want a package with more functionality then you can load 7z ([http://tombuntu.com/index.php/2008/07/21/add-7z-7-zip-file-archive-support-to-ubuntu/](http://tombuntu.com/index.php/2008/07/21/add-7z-7-zip-file-archive-support-to-ubuntu/)).

Enjoy - and don't hesitate to ask questions on the forum if you get stuck.
