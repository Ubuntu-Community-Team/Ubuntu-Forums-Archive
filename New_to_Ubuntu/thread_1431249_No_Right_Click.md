### Post by zeroseven0183 on 2010-03-16
Hi! That happened to me randomly before and was corrected using **gTweakUi**

```
sudo apt-get install gTweakUI
```

After installing, go to **Preferences** and select [B]gTweakUI - Nautilus
[/B] Under the **Desktop** heading, mark the checkbox next to **Use  Nautilus to draw desktop**

Hope that helped
