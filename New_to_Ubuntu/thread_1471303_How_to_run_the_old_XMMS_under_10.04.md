### Post by comsparks on 2010-05-03
This is for the old timers that love the old XMMS and want to use it under Ubuntu 10.04. First go to:
[http://packages.debian.org/search?suite=default&section=all&arch=any&searchon=names&keywords=libglib1](http://packages.debian.org/search?suite=default&section=all&arch=any&searchon=names&keywords=libglib1)

Package Libglib1.2ldbl
Libglib1.22ldbl (Download and Install)
Lenny (stable)

Then search on Libgtk1.2

Package Libgtk1.2 common (Download and Install)
Lenny (Stable)

Package Libgtk1.2 (Download and Install)
Lenny (Stable)

When they are finished than install the XMMS and all your dependencies will be satisfied.

Enjoy!
