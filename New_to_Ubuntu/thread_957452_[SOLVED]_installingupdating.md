### Post by Rocket2DMn on 2008-10-24
If you do this through the package managers like Synaptic, apt-get, aptitude, Add/Remove - then yes, it updates the software to the latest version and removes the old.  Your configurations should remain unchanged.  If you install a program from source (not recommended if the program is available in the repositories), then you may end up with multiple versions, depending on how the source is setup to install.

### Post by OldGrey on 2008-10-24
If you are updating rather than doing a clean install, your data will be preserved. However always back up first as it can go wrong. Also there is a school of thought that a clean install is better as it tends to be less problematic.

### Post by Rocket2DMn on 2008-10-24
If you are upgrading from 8.04 Hardy Heron to 8.10 Intrepid Ibex, your data will remain untouched.  The system software and many appliations that you have installed will be upgraded to the versions that exist in Intrepid.  As always, you should keep complete and up to date backups.

Right now, Intrepid is still technically in the development phase, though I believe the Release Candidate has been made available for testing.  If you are looking to upgrade, I would wait until the official release, and maybe a few days or more afterward so you don't have to fight the traffic *and* you can see what problems are arising when people try to upgrade.
