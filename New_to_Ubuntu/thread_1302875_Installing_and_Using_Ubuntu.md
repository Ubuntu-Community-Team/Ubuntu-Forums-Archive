### Post by martrn on 2009-10-27
Hi C,

Ubuntu is a fully fledged multitasking operating system with the capability or running multi-threded applications simultaneously across many processors or one processor as needed.  It provides the user with a choice of many desktop environments but the most popular being kde, gnome and XFCE.  Ubuntu has gigabytes of software in its repositories and there are many suitable open source applications available for you to use, and there are many free open source applications that will have similar windowzes pripority counterparts and visa-versa.

Its has a Graphical User Interface that is different to windowze, and any new computer user or computer users coming from a different computing environment will have a learning curve.  How easy or difficult it it is to learn is dependent on the users ability to understand that there are differences, and there will be a learning curve.  Linux and more specifically ubuntu doesn't try to imitate windowze and doesn't try or attempt to be a clone of anything.

It is not easy neither is it difficult to install, but does usually require some understanding of partitions, disk-space, disk drivers a small amount of knowledge of your computer.  You can download a cd from ubuntu.com or request a free cd via ship-it.
