### Post by kurt18947 on 2012-09-21
You made a good choice with the WNA 1100.  I have one and it's been trouble free.  What version of Ubuntu?  If 12.04, plug it in, the little blue light should flash and it should work.  You can't get much simpler than that.  Ubuntu versions earlier than 11.04 (I think) require some magic. A few seconds after plugging in the adapter,   You should get a message like "wireless networks are available".  Network Manager is the default  app to configure network connections.  If your preferred Wifi connection is broadcasting its SSID,  it should appear on Network Manager's list of available wireless networks.  Click on the network name, enter the pass phrase and you should be in business.   Please post back if you need further help.
