### Post by drs305 on 2009-12-20
Open Synaptic, go to Settings, Repositories, and in Ubuntu Software tab untick the CD-Rom in the lower panel. If that isn't the correct CD, check for it in the "Other Software" tab. "Reload" to refresh the repository list.
