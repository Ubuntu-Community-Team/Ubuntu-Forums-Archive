### Post by SeanHodges on 2008-11-21
I can't tell you about any work-arounds, but full support for independent virtual desktops is planned for the 4.2 release:

[http://techbase.kde.org/Schedules/KDE4/4.2_Feature_Plan#kdebase-workspace](http://techbase.kde.org/Schedules/KDE4/4.2_Feature_Plan#kdebase-workspace)
-> "Optional Per-Virtual Desktop DesktopView"

This should allow you to put separate backgrounds on each desktop, as well as different plasmoids, panels, etc...
