When I installed it I added those lines in my Sources List
```
deb http://ppa.launchpad.net/chromium-daily/ppa/ubuntu hardy main
deb-src http://ppa.launchpad.net/chromium-daily/ppa/ubuntu hardy main
```

then installed from Synaptic. If you are using Karmic, I think you should replace "hardy" with "karmic" and it would work.
