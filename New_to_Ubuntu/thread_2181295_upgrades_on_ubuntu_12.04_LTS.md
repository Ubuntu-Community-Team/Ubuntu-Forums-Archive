### Post by grahammechanical on 2013-10-17
Ubuntu 12.04 was released in April 2012 and is a Long Term Support (LTS) release. Which means that we get updates/security fixes for 5 years from the date of release. We can set Ubuntu to check for updates every day, every 2 days, weekly, every fortnight or never.

Next year, sometime in April, Ubuntu 14.04 will be released. It will also be an LTS release. You can upgrade from 12.04 direct to 14.04. We can set Ubuntu to notify us of the next LTS release (that is the default setting for 12.04) or to notify us of the next release. Not only will we get notification of a release but also a one click button to run the upgrade.

Stick to 12.04 until 14.04 is released and then make a decision on whether to upgrade to 14.04 or fresh install 14.04 over 12.04.

Ubuntu 13.10 only has 9 months support for updates and security fixes. So, unless you want to be the kind of person who upgrades every six months, like me, then stick with 12.04 until 14.04 is released and run those updates in the mean time.

Regards.
