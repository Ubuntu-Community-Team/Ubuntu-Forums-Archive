### Post by ad_267 on 2008-08-08
Yes you can use virtual box or other virtualisation software.

You can also use WINE which can run windows applications under Linux. Have a look at [http://appdb.winehq.org/](http://appdb.winehq.org/) to see if your application works under wine.

I'm not quite sure what you want to do though. If you want to run Sunbird in Ubuntu it is already available for Linux.
