### Post by The Cog on 2011-11-12
Nautilus used to do it, until someone demonstrated something like windows autorun, where simply plugging a USB stick in made Linux execute some code from the USB stick. It turned out that they had placed malformed PDF files on the stick and were using them to trigger a parsing bug in the PDF viewer that made the thumbnail images. I think the viewer bug has been fixed, but I suspect that's why it doesn't thumbnail PDFs any more.
