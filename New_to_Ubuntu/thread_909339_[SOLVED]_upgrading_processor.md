### Post by oldsoundguy on 2008-09-03
yes, that is provided the motherboard will support a faster processor.
First you will need the make and model of the motherboard.
If she is running Windows, go get Belarc Advisor, download it and run it. 
If she is running Linux go to the terminal and type in: lshw. 
That will tell you more than you ever need to know about the computer.
Once you have the M&M, Google for a manual and find out what type of  processor it supports and what are the processor limits.
Then eBay away for the older processors.

You should also consider adding memory in the upgrade.  

[www.crucial.com](www.crucial.com) has a plug in to tell you what you can put in there (using the make & model of the board once again.)  You do NOT have to buy from them to get the information.

ALL data should remain in tact in the upgrading process.
