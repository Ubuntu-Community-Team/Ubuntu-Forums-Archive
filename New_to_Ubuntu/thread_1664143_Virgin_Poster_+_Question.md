### Post by Jerry N on 2011-01-10
I don't know if you need glasses - how old are you?:P:P  To be a bit more serious, what kind of screen are you using?  If it is an LCD, do you have the resolution set to the native resolution of the screen.  Ubuntu usually figures that out but it doesn't hurt to check.  Also if you right click on the desktop you will bring up the appearance preferences.  Look under the Fonts tab and see what kind of font rendering you are using.  There is a button specifically for LCDs.  Again, Ubuntu usually gets that right.  You can also look under smoothing to see if which looks better to you.
