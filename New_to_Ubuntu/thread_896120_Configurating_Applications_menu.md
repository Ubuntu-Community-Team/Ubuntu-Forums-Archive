### Post by ubuntu1sttimer on 2008-08-20
In 8.04 I have applications that are installed but not in the applications menu. Also some that are no longer installed that appear in the menu. Plus would like to make some changes/additions to the "hover" messages. The Add/Remove option is of limited use for these items. 

Can I edit the configuration files for "Applications" or is there another way to to do this? Also, where do I find the configuration files for the "Applications" drop down?

---

### Post by drs305 on 2008-08-20
You can open the menus for editing by running "alacarte" in a terminal. Once open, highlight the app or folder in the left window and right click on any listing in the right pane and select 'Properties'. 

You can add and delete apps; for adding apps you will just need to know the start command - just as you would if you were adding a shortcut to the panel.

The 'hover' messages are what appear in the 'Comment' section and you can change them to anything you want (or delete it).
