### Post by drs305 on 2010-10-22
In nautilus, right click on any folder. "Open with Other Application". Select File Browser and tick 'Remember this application for "folder" files'

### Post by mc4man on 2010-10-22
Because of the reluctance to fix or even address this bug   - in the future - 

If using the r.click 'open with -> other application' menu on a folder make sure that 'Remember this application for "folder" files'  **is not enabled** - otherwise any app chosen will become the new default folder handler
