If you delete it (I'm not going to tell you how :wink:) it will be re-created next time there's a scheduled filesystem check anyway. It's needed if a filesystem check finds data that needs recovery. Since it's not doing any harm, it's best left alone.

### Post by coffeecat on 2011-11-18
> **LukaHr said:**
> 
3. don't have "Dim display when idle"

ubuntu 11.10

It's there but a little difficult to find.

Click on battery icon -> Power Settings -> All settings -> Screen. Untick "Dim screen to save power".

**Or**

Click on the cogwheel icon top right of screen -> System Settings -> Screen. Untick "Dim screen to save power".
