### Post by ildfroe on 2008-05-29
From:
[http://extensions.services.openoffice.org/resources/user/howto_install](http://extensions.services.openoffice.org/resources/user/howto_install)

You can install an user extension in different ways:

    * By opening Tools &#8211; Extension Manager:
          o Select the top-level node 'My Extensions' as category.
          o Click on the 'Add' button. A dialog opens where you select the extension that you want to install.
