### Post by 3rdalbum on 2010-02-12
You could try installing Beagle or Google Desktop. These are both desktop search engines for your files. Beagle used to ship with Ubuntu.

Desktop search engines go through a quick phase of popularity before everyone stops using them. Mostly, people stop using desktop search engines because they always suck. They suck your CPU power and they suck at finding things when you really need to find them.

If anyone out there reading this has written a desktop search engine, then sorry. But that tends to be the case.
