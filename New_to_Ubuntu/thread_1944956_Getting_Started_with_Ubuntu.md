### Post by TBABill on 2012-03-22
To the OP, in Linux you can always look to see which drives you have "mounted", meaning drives you have "connected to" in a sense. If you have a storage card from the camera with pictures you want to load onto the computer, placing the card in your card reader should show that card in the file manager. You can then open the picture files, edit them, copy them to the HD, etc. 
 
If you are trying to plug a device such as a camera in directly, you will need to find out if Ubuntu supports your model by searching for that info. You could also ask here if you identify your model and see if anyone else has had luck getting it going.
 
I have a short set of items I do on every install to get my Ubuntu ready for full use. Others do different things for sure, but these are my minimum needs. I first fully update my system. Then I install Adobe Flash (if not done during install), Sun/Oracle Java 1.6 or 1.7, install Chromium browser, add items to the launcher (Chromium, terminal, system updater and other highly used apps), install Synaptic package manager, install ubuntu-restricted-extras (for codecs). 
 
Then I just enjoy my system and tweak as I think of other things I want or need. Sites like askubuntu, webupd8 and others have great tutorials on how to do lots of things with Ubuntu as well.

### Post by daslinkard on 2012-03-22
Coming from someone who has been using Ubuntu for 60 days what I would suggest doing would be going to your home folder in Ubuntu.  Next identify your HD and clicking on it.  Depending upon your Windows OS you will either be looking for Documents & Settings (XP) or User (Vista/7) folders.  Once inside this folder you will then see your User Account Folder.

Click on your User Account Folder and then you will be into your Windows System allowing you access to your Desktop, Documents, Pictures, Music, etc.  At this point you should be able to click on your folder/picture, etc. and drag to your desired location.

If I left anything out, someone please add to this...
