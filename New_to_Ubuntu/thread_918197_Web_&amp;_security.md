### Post by WRDN on 2008-09-12
There is a firewall already installed called iptables. For easy configuration, Firestarter can be used. Ubuntu now uses UFW (Uncomplicated Firewall), and there is a GUI application available for this also.
As far as antivirus is concerned, you don't need it, though you may want to scan files if you share files with Windows users/ the Windows installation. This ensures that, if something is downloaded in Ubuntu, it will not affect the Windows installation(s). A good program is Clam-AV which is available in the repositories.
