### Post by meborc on 2009-03-04
when you first install emerald theme manager, there are no themes there... you need to download emerald themes from gnome-look.org for example and install them (add them) in the emerald theme manager... now if compiz is running and you have an emerald theme selected in the emerald theme manager, you should see the window borders using the new theme

i suggest you install a package "fusion-icon" and start it with alt+F2 ... then you see a small icon in the system tray... right-click and set the window manager and whether to use gtk or emerald
