### Post by 73ckn797 on 2011-11-22
I would recommend a clean install of 11.10. Upgrades can mess things up (search the forums and you will see). If you have created a separate /home partition then all of your settings and documents will still be there. You will have to reinstall added programs.

If you do not want notification of upgrades you can change that in System/Administration/Synaptic Package Manager/Settings/Repositories/Updates/Release Upgrade

### Post by peter d on 2011-11-22
It needs to be a clean install.

Upgrades only work between one release and the next. The only exception to this is to upgrade from one LTS (Long Term Support) version and the next LTS. So 10.04 LTS can upgrade to 10.10 or 12.04 LTS when it is released in April.

Apart from a clean install the only other option is to upgrade through the versions 10.04 to 10.10 to 11.04 to 11.10.
