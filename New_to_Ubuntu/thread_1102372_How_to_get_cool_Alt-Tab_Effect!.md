### Post by Nxion on 2009-03-21
That is actually not ALT+TAB. The person in the video has it configured for "hot corners". So this means when he goes to any of the four corners it will do a different action. For example if he went to the top left, it would lock the machine or if he went to the bottom right in this case it shows all the windows on that particular desktop.
