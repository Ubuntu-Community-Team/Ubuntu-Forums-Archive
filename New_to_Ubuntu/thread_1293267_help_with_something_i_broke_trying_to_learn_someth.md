### Post by djbon2112 on 2009-10-16
First of all, what do you have left? Does the OS boot? Do you have a command prompt?

If you have a command prompt, you can probably salvage the system by typing:

```
sudo apt-get install ubuntu-desktop -y
```

...but at this point, you'd probably be best to reinstall it.
