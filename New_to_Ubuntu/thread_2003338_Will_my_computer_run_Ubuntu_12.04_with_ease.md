### Post by Bucky Ball on 2012-06-13
Download the 12.04 LTS ISO, burn it to a CD, boot from the CD and select 'Try Ubuntu' when you get to the selections. Plug in an ethernet cable if you want to get online (wireless should work fine once you have a hard drive install rather than running from the CD).

How did it go? If not good, what happened exactly?

(PS: I have run 10.04 LTS, 10.10, 11.04 on my Toshiba Satellite Pro L510 without a hitch ... but not 12.04 LTS. 10.04 LTS  is rock solid stable and has another year support, as has 11.10. Both options if you have too many problems with the much newer and more problematic (for some) 12.04.)
