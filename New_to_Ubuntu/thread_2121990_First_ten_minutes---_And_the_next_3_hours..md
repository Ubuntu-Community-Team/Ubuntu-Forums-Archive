### Post by Sin@Sin-Sacrifice on 2013-03-03
You can install xfce4 by opening a command terminal and typing "sudo apt-get update" and then "sudo apt-get install xfce4 && sudo apt-get install xfce4-goodies" without quotes. If you want to search for other desktop environments, use "apt-cache search 'name of desktop environment'". Such as "apt-cache search kde". If that gives too many results, just refine the search perameters by trying "kde-desktop" or, as I think Ubuntu does it, "kubuntu-desktop".

### Post by darkcrimson on 2013-03-03
If you're looking at some "laggy" performance and are familiar with Mint, you could try using the Cinnamon desktop as an alternative. However, this would also require adding an additional repository to the list. I've found Cinnamon to be a nice alternative. Clean, fresh, and leaves a lot of room for your desktop. Bear in mind, I love Unity, but sometimes it's nice to have a change.

To install Cinnamon:
```
sudo add-apt-repository ppa:gwendal-lebihan-dev/cinnamon-stable
```
```
sudo apt-get update
```
```
sudo apt-get install cinnamon
```

Once you've installed any other desktop environments, you can access them from the login screen; clicking on the Ubuntu logo next to Username.
