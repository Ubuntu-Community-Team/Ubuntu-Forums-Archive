### Post by sisco311 on 2009-11-12
> **wlraider70 said:**
> When i first start ubuntu and it asks for a username can i put "root"?

What would the PW be?

The root account password is locked, and it's locked for a reason.

You can use sudo/gksu to run applications as root. 

[uhelp]
community/RootSudo[/uhelp]


Logging in as root in the GUI is a very bad idea.
You can run any application as root with sudo/gksu so there is no reason to start a graphical root session. 


Oh, and it's against the forum policy to help to do so.
