### Post by LowSky on 2010-04-28
I found a better and more ubuntu way to get Rawtherapee 
oen a terminal type these commands

```
sudo add-apt-repository ppa:rawtherapee/ppa
```Note this might take a few minutes, dont worry if it hangs for a few its ok, if it hangs for over 10 minutes try again.
 then update the system
```
sudo apt-get update
```
then install rawtherapee
```
sudo apt-get install rawtherapee
```

more info here if needed 
[https://launchpad.net/~rawtherapee/+archive/ppa?field.series_filter=](https://launchpad.net/~rawtherapee/+archive/ppa?field.series_filter=)
