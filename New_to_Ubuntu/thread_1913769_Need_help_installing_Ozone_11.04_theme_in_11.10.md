### Post by Scarbird on 2012-02-21
Not sure that you'll see this message as you posted this 4 weeks ago and probably gave up by now or found the solution yourself, but you can download the theme from:
[https://wiki.ubuntu.com/Lubuntu/Artwork/Incoming/Natty/Ozone#Theme](https://wiki.ubuntu.com/Lubuntu/Artwork/Incoming/Natty/Ozone#Theme)

Once you've downloaded the file you can extract it and install it globally (For all users), or just for yourself.
If you wanna install it just for you, extract the contents of the archive and move the extracted "Ozone" folder to:
/home/***your username***/.themes

If you want it for all users, you'll need to get root user permissions and move the folder to:
/usr/share/themes

then you can select the theme from Customize Look and Feel.
