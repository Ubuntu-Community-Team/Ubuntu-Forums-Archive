### Post by oldos2er on 2008-11-01
> **elleppahc said:**
> How do I install Google earth .bin file please

 Assuming you downloaded it to to your desktop, open a terminal and run these commands one at a time:

 cd Desktop

 chmod a+x GoogleEarthLinux.bin

 sudo ./GoogleEarthLinux.bin

 Once the installer has finished, exit out of it before running GoogleEarth.
