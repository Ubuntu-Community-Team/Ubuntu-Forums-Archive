### Post by ubuntu27 on 2010-06-08
First install Japanese support by going to System/Administration/Language Support.

After installing the language support, go to System/Preference/iBus preference.

Add Japanese.

If it ask you to run iBus, say yes.

(I don't remember the exact steps, but, this is the overall steps_
