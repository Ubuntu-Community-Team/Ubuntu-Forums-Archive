### Post by lovinglinux on 2009-12-20
Rename the swiftweasel profile folder to something else, then select the firefox profile folder and make a link of it. Then rename it to match the original swiftweasel profile folder.

I guess swiftweasel uses ~/.sw/swiftweasel or  ~/.swiftweasel/swiftweasel and firefox uses ~/.mozilla/firefox. Anyway, you should compare their folder structures to determine which folder to link.
