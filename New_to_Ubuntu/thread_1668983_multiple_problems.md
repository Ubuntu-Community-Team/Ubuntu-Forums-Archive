### Post by Wtower on 2011-01-17
Hi, welcome to the Ubuntu community. If you want to resize the partition with which you boot on Ubuntu (ie the partition which mounts as root), then you should better boot from live cd or usb stick. Then you can use any partitioning program like gparted in order to do that.

For the subtitles, take a read at: [http://ubuntuforums.org/showthread.php?t=556267](http://ubuntuforums.org/showthread.php?t=556267)
