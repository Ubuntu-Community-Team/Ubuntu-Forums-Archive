### Post by lisati on 2009-07-23
[http://www.ubuntu.com/getubuntu](http://www.ubuntu.com/getubuntu) will let you download a CD image with all the files needed to install Ubuntu - check [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto) for info on putting it onto CD

Once you have installed Ubuntu, you have options for where you get the software you need - the Applications->Add/Remove menu is a good place to start.

What exactly do you mean essential files?  You can download a net install iso from the main ubuntu download page.  It allows you to install from the internet, which means you have the latest version of all the packages.  It will install Ubuntu with no graphical user interface or extra software.  There is also a server install package which will do almost the same thing.  If you mean that you want to install the full desktop version then just follow lisati's links.  Welcome to the site.
