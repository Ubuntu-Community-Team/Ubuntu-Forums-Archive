---

### Post by Peter Maunder on 2012-09-20
On the Organizer page of the Paragraph Style and Frame Styles is a check box called AutoUpdate. If you manually overide styles, be certain not to tick that box otherwise any change will effect the whole document. Be especially careful if you are creating newspapers etc. with large changes in layout and font sizes.
