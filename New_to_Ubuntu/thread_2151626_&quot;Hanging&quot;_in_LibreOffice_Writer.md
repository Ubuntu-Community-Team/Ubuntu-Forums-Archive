### Post by steeldriver on 2013-06-06
You can set a positive indent value for the paragraph as a whole (e.g. 'Before text: 1.00cm') and then set an equal and opposite negative value for the first line (i.e. 'First line: -1.00cm') - maybe that will work for you?
