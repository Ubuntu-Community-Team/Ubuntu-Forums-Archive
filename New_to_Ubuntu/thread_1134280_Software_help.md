### Post by Paqman on 2009-04-23
All the software on Linux comes with a manual. To read it just put
```

man nameofpackage

```
into the terminal. The trouble with this is that it's written by the developers, not by humans, so can be a little dry, if not downright unintelligible.

Otherwise Google and these forums are your best friends.
