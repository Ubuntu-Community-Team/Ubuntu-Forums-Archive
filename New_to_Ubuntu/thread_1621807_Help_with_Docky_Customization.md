### Post by paranoid_android on 2010-11-15
To install the theme unzip to ~/.local/share/docky/themes(create the 'themes' directory if it's not there). For all users /usr/share/docky/themes.  You may have to restart docky (I can not remember) but the theme should then be an option in the docky settings menu, as for the icons are they specifically for docky or are they system wide icon packs?
