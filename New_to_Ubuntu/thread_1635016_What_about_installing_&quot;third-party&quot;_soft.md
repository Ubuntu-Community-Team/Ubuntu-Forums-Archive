### Post by cariboo on 2010-12-01
You can install any thing that is packaged as a .deb file, with a minimum of trouble, by double-clicking the file. But the thing to keep in mind is where the software is coming from. All the packages in the repositories are signed by the author/maintainer, whereas, if you install anything from a random web site, you don't know where it's coming from
