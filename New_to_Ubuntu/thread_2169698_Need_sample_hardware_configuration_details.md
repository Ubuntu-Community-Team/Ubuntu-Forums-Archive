I think the word you're looking for is "server", not "data center". A data center is a building full of servers.

Ubuntu Server and Ubuntu Desktop system requirements are mentioned here: [https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements)

Please note that Ubuntu Server doesn't contain a GUI, so if you're new to Ubuntu it might be better to install Ubuntu Desktop and then install your server software on top of that. Ubuntu Server is really for people who know what they are doing on the command-line. I struggled to set up Ubuntu Server and I've been using Ubuntu (Desktop) since 2005.
