### Post by PierreDeKat on 2008-11-06
Okay, I'm right there with you on this GIMP thing. I was almost going to turn off desktop effects altogether because it had gotten so annoying.

For me, the biggest problem with the GIMP was that, no matter how many times I hit the "Save Window Positions Now" button  (Edit->Preferences->Window Management) the GIMP kept loading with the main "Toolbox" buried halfway down below the bottom edge of my desktop.

I finally fixed this problem -- like just about 5 minutes ago -- by right-clicking on the window border bar -- the one you might have the minimize/maximize/close buttons on -- selecting "Resize", and then dragging the edges to something I could live with.

Note: you might have to go so far as to temporarily change your monitor resolution to get ahold of the window border bar thing.

But anywho, once you you get the windows resized with this method, go to the GIMP's preferences, click the "Save Window Positions Now" button, and that time, the window position of the main "toolbox" interface will actually stick.

Hopefully this helps.
