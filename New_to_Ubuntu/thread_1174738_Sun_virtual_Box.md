I guess you have installed the open-source version of virtualbox (the one in the ubuntu repository, called 'VirtualBox OSE')? That one has something dodgy going with USB...

Uninstall it and go install the non-free one. Instructions and download here:
[http://www.virtualbox.org/wiki/Linux_Downloads](http://www.virtualbox.org/wiki/Linux_Downloads)

Also, you'll have much more lock if you go to vitrualbox own forum. There are separate forum sections for linux hosts, windows guest etc...
You find it here: [http://forums.virtualbox.org/](http://forums.virtualbox.org/)

Good luck!
