When you log out, it should take you to a screen asking for your username and password.  Somewhere on this screen will be another option, asking for something like 'session'. If you have successfully  installed another desktop environment it can be found on the session menu in the login screen . In Ubuntu there gear symbol on the right side of the greeter box . If you want to use other desktop environments don't use an automatic login because it logs into the default desktop environment , which would be Xubuntu according to the thread tag .
