### Post by LunaticHiatus on 2009-08-08
I'm not sure what it is your saying but if you want to do is rn ubuntu off your usb drive. In which case, just install unetbootin on your windows or linux machine. Use it to install ubuntu, and if you want to run ubuntu from your usb device, just press 'esc' when you first boot up. If you want to always run from usb (at at least, tell your computer to boot from usb first if present) you can go into bios (I think its f2) and reorganize your hard drive priority to usb first and hard drive second.
