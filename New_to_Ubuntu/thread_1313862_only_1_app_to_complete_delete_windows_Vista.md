### Post by AlanR8 on 2009-11-10
This should just work in Karmic.

When you plug your phone in via USB, let's just start there, the networking wizard should open and take you through the setup process. This is very simple and the wizard will apply all the necessary settings like the "phone number" to dial.

If it doesn't automatically open, right click the networking icon and add a new connection. That should get you going.

If that doesn't work, and it should there is a solution I've used before using wvdial. It should work however, my son's just bought a Blackberry Storm 2 and it took about 2 minutes to connect it to the web using my laptop via bluetooth!!!!!
