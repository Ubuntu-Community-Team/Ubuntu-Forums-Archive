### Post by chaosx9 on 2010-07-03
Grub-pc is the Bootloader used when you install ubuntu onto a separate partition. To explain it, imagine installing two windows xp's (jus to educate). On the second install, It will allow you to pick between the first and second installation on startup.

Grub would do that, but i'm not sure if you'll need it for wubi. I would say keep it, but don't upgrade it if it's causing you problems, because it'll work just the same.
