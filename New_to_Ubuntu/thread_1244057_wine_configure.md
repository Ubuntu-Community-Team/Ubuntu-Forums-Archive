If the problem is one of poor game performance, that's most likely a video driver issue and not something you can "fix" in Wine.  If you have an ATI card/chipset and it's not one of the newer "HD" series, you can't even install an improved video driver because none exist that work with Ubuntu 9.04 for the older legacy cards/chipsets.

Furthermore, if you've only installed Wine to run MS Windows games, you're most likely to be very disappointed with the results.
