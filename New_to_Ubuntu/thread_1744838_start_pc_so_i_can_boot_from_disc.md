### Post by beecheese on 2011-04-30
Hi Retro-Rocket,

I'm not sure how universal this is but you should try pressing F2 when the computer is just starting up. This should open up the PC BIOS which will enable you to get the machine to boot from a CD instead of the hard drive. The CD will have to be in the drive.

On my laptop, you have to move to a menu that lets you prioritise how the computer boots up.

If F2 doesn't work, there might be a message as the computer starts that tells you what to press to enter a "Setup" mode or something similar.
