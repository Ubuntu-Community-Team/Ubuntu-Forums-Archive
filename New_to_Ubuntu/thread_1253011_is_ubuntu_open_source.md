### Post by Old_Grey_Wolf on 2009-08-29
Yes, Ubuntu is open source. For example, the source code for Jaunty is at [http://cdimage.ubuntu.com/releases/jaunty/release/source/](http://cdimage.ubuntu.com/releases/jaunty/release/source/). Anyone can examine it, modify it, and compile it themselves.

### Post by perce on 2009-08-29
What is in the CD is open source, however you can download non-free drivers, if you hardware needs them, through the restricted drivers manager, so it is possible that the OS has few non-free components when you finish the set-up.
