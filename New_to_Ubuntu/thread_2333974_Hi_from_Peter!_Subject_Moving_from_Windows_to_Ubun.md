### Post by TheFu on 2016-08-15
Welcome to the forums.

No way to know based on the information provided.

A risk-free method is to try a live-boot off either a CDROM or USB flash drive. If wifi works that way, then it is very likely to work with a full desktop installation. Server installations generally don't support wifi out of the box.  Stay with LTS releases. That would be either 14.04 or 16.04 today. Don't bother with other releases which only have 6-9 months of support.  Also, to prevent confusion, learn that not all 16.04.x releases get the same support times.  [https://wiki.ubuntu.com/Releases](https://wiki.ubuntu.com/Releases) shows that 14.04.4 support ends this month. Only 14.04.1 and 14.04.5 has support until 2019, not the intermediate versions.

I'd also encourage you to learn about the Linux "culture" a little. This is very different from the buyer-relationship that commercial OS users expect.

### Post by grahammechanical on 2016-08-15
Hardware driver modules are often part of the Linux kernel. So, if 2 Linux distributions are using the same Linux kernel then the same driver modules will be available. This is my guess. The differences between the two distributions will be the method used to install proprietary video drivers.

I know nothing about Fedora. But with Ubuntu we get an option to install third party software as part of the installation process. If we tick that box we get some free but proprietary audio & video codecs, a proprietary video driver and possibly a proprietary WiFi driver. I am not sure about the WiFi driver because Linux has always supported my WiFi adapter since I first installed Ubuntu in 2007.

 In Ubuntu if we go to System Settings>Software & Updates>Additional Drivers tab the utility will find proprietary drivers and that will include WiFi drivers. We need to be connected to the internet and allow time for the utility to do its work. And then we can use Additional Drivers to install the driver.

Ubuntu Desktop Guide (Ubuntu Help) is installed by default. It contains a lot of information about matters such as this. And then we have this forum for support.

[https://help.ubuntu.com/16.04/ubuntu-help/](https://help.ubuntu.com/16.04/ubuntu-help/)
