### Post by mvalviar on 2009-03-14
Another great place to hunt for themes is [http://www.gnome-look.org]("http://www.gnome-look.org")

You will usually be given a tar.gz file. Drag and drop that file to the themes tab under "Change desktop background". If it doesn't work you might want to read the documentation regarding the theme you downloaded.
