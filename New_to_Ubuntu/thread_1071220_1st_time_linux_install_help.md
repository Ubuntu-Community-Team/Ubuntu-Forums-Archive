### Post by ugm6hr on 2009-02-15
You will struggle to install Ubuntu on that.

The minimum recommended specs for Ubuntu are much higher.  128MB RAM is not enough (unless you do a "minimal" install manually - which is not recommended for beginners).

There are plenty of alternative Linux options for computers of that era.  My suggestions: Puppy Linux or TinyMe
