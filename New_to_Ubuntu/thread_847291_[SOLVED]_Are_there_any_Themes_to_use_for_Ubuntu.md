### Post by tjwoosta on 2008-07-02
> OK, I just installed the Emeral pack, and downloaded a couple of GTK2 themes from gnome-look.org like you guys suggested... clicked on IMPORT, but none of those files appears on the list to be imported? Should I un-packthem ?.. if so, how ?

Again, I am a little slow with Ubuntu, but learning...

when you click on import you can choose which directory you are looking in 
(the emerald themes should be a file with a .emerald extension , while the gdk themes should be .taz.gz or .tar.bz2)

(with the gdk themes you should just be able to drag and drop the .tar right into the window with the themes and it will automatically install) 

after you install the emerald theme you need to press alt-f2 then in the box type 
```
emerald --replace
```


(if you want emerald to start at every login, go to System-Preferences-Sessions then click add and add the commaand to the box that says command
put whatever you want in the other two boxes)
