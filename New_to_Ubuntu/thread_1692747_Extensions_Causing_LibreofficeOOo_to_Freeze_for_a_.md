### Post by marl30 on 2011-02-21
Very good performance when I assign more memory. I set the amount of memory to:
Libreoffice use 50MB
Memory per object - 12 MB
Cache for inserted objects - 30

With this setting all the freezing seem to have stopped. Libreoffice is starting so fast that I don't even see the start up screen. :D

I guess I can mark this as solved.
