It would help to know the specifics of your hardware.  Many older laptops can't boot off an external cdrom.  You would have to look in the bios to see if that options is available.

Wine is capable of running some windows programs but not all of them.  So wine may work for a few of the programs you want to run but you could check winehq to see if the application you want to run is able to be run with WINE.

Also there are virtual machines to run windows virutally within linux but again with older hardware this might not be possible.
