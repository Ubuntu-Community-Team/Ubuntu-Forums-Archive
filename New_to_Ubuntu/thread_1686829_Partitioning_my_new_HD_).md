Will the new 750gb Hard drive be your new 'booting' drive for both Windows 7 and Ubuntu?

If so, Swap our your current drive (keep it safe so you can put it back in and boot up on it) for the new drive.  Then install Windows 7 taking up the whole drive.

Once Windows is installed and running OK, defrag the disk and then boot Ubuntu and select the side-by-side install option.

As to how big it should be (your real question), if you store your data on the Windows partition so both Win7 and Ubuntu can see it, 100 gigs for Ubuntu is plenty.

### Post by presence1960 on 2011-02-12
Partitioning and size depends on what you want to do with your OSs. Of course there is a recommended minimum size for each OS install for it to work properly. However will you have separate partitions for data? A separate /home? AN NTFS partition for data shared between windows and linux?

You need to be a little more detailed and specific. Also realize there are a multitude of partitioning schemes you could use- there is no one "right" way.
