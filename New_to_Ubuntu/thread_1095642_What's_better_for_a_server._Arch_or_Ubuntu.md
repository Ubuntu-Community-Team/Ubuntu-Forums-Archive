### Post by Kareeser on 2009-03-14
Judgement call.

Arch comes with the bare essentials and you build it up as you need. So when it comes down to it, the Arch install will have more space and be a tad faster.

Ubuntu server is also a slimmed-down install, but installs much of the more common server tools automatically. It also has some differences (down at the way the OS handles processes and IRQ interrupts) when you get reeally into the nitty gritty of it.
