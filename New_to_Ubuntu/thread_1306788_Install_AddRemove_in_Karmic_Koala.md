### Post by Temposs on 2009-10-30
Open a terminal.

```
sudo apt-get install gnome-app-install
```

Or install the gnome-app-install package through synaptic package manager.

---

### Post by Temposs on 2009-10-30
Also, in Ubuntu Software Center you can install more than one app at once. It's actually quite awesome the way they do it. You can click to install something, then go search for another app while the one you're installing is going, and choose another app to install. It gets added to the install queue, or the "In Progress" tab. This is way better than Add/Remove, in my opinion.
