### Post by Favux on 2009-07-06
Hi Essar Allen,

We've figured out how to get Wacom working on Jaunty.

Go to this HOW TO:  [http://ubuntuforums.org/showthread.php?t=1038949](http://ubuntuforums.org/showthread.php?t=1038949)  Read Jaunty Users near the top.  Follow the link in 1) to gali98's HOW TO on post #104 on the same thread.

Since you're new take it slow.  Read things through a few times.  Be sure to follow the instructions exactly.  It's better to copy and paste the commands into a terminal rather than type them.

After you have Wacom working you can go to the HOW TO on the first page and after Section 3 is a link to the Rotation HOW TO.  That will give you screen rotation so you can go into tablet mode.

Good luck!

PS:  For the webcam try installing Cheese through Synaptic Package Manager.
