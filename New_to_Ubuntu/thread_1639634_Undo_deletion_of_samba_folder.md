### Post by RoflHaxBbq on 2010-12-06
Yes.

But it only works well when:

1.The HDD is not fragmented
2.It is a small file you are trying to recover
3.You haven't used the HDD much since the "deletion

You see, when you "Delete" a file, it doesn't actually delete it. It writes into a master file of sorts, saying that the area on the HDD is "Free Space" and ok to overwrite. But if you use the computer since the "Deletion", it may have already overwritten part of the space that this file took up.

I will only advise you try this method with very important files that are of a small size (only a few MB), other wise it just won't work.

There are free programs that can scan your HDD for "Deleted Files" then you can pick which ones you want to attempt to recover. Keep in mind that there is a very high chance that most of these files will be corrupted.

Just search Google with keywords like Undelete and stuff, and you will be sure to find something. Keep in mind that there might be viruses and other nasty stuff hidden inside of these seemingly helpful programs. Scan them with a virus scanner first, and remember, if it looks dodgy, it probably is.
