### Post by WRDN on 2008-07-25
You don't need to worry about getting a virus on your Linux system, but if you share files with Windows users, then it would be a good idea to scan the files before sending them to the Windows users. Clam-AV is a good program, and can be found in the repositories (search for "clamav"). If it doesn't install by default, you should also install "clamtk" so you have a GUI frontend for the program.
