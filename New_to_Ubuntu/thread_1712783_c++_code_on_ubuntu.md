### Post by muteXe on 2011-03-23
After you've installed it and written your code:

Compile with:
```

g++ yourfile.cpp -o <what you want your exe to be called>

```

To run:
```

./<whatever you've called your executable>

```

If you have lots of cpp files, you may want to google 'makefiles'.
