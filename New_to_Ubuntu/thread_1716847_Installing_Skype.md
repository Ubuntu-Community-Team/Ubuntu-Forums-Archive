### Post by thane1 on 2011-03-30
Open up Synaptic.  In Repositories enable Canonical Partners.  Then do a Synaptic search for Skype and you'll find it.  It works fine this way for the latest ubuntu version.  While you're there also download pavucontrol.  If you have problems getting sound on Skype, open up Terminal type in pavucontrol in Terminal, but don't hit the enter key yet.  Open up Skype and start a test call.  Once the female voice from Skype starts, hit <enter> in terminal to start pavucontrol.  Then navigate to pavucontrol's sound functions and set your sound to the desired setting.  Mine is the Microsoft LifeCam sound mic, but yours may be different.  This will only be necessary if you don't have functioning sound though.  Cheers.
