### Post by acrazyplayer on 2011-10-01
This seems more like you need to file a bug to mozilla themselves. Use a different browser to file the bug, in the meantime try to use a different version of firefox. 

[http://portablelinuxapps.org/network/firefox](http://portablelinuxapps.org/network/firefox)

you download what you want to use, right click on it and go to "properties" then "permissions" then check the box "allow executing the file as program" then just normally click on the file and it will run like normal. maybe a little slower to start but see from there if facebook and your threads work. if so then that should help with your bug report.
