### Post by mister_pink on 2008-08-17
If you want to access your files remotely then a webserver isn't a great option. I'd go for using dyndns in conjunction with ssh. A lot of the steps in that guide still apply, just none of the apache stuff. You'll want to install openssh server instead. See [https://help.ubuntu.com/community/SSHHowto](https://help.ubuntu.com/community/SSHHowto) and possibly [https://help.ubuntu.com/community/AdvancedOpenSSH](https://help.ubuntu.com/community/AdvancedOpenSSH)
