### Post by jbrown96 on 2009-09-13
> **Wataru8675 said:**
> I forgot whether I installed the 32 or 64 bit version of 8.04...my computer will run 64 bit, but I may have done 32 to be safe.  Is there a way to check which I have?

In a terminal, ```
uname -a
```
My shows > 2.6.28-15-generic #51-Ubuntu SMP Mon Aug 31 13:39:06 UTC 2009 x86_64 GNU/Linux
 and is 64-bit. 32-bit will say i686.
