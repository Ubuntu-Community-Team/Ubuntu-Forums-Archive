### Post by snowpine on 2009-11-24
All of your apps and everything in your home folder will be available to you, whichever desktkop you choose at login. 

Applications only slow down your computer if you're using them. So long as you only use the "lightweight" applications that are part of Xubuntu, you should theoretically get the full speed benefits.

Good luck!

### Post by coffeecat on 2009-11-24
You can run gnome applications in KDE and vice versa. The only desktop specific apps that would be tied to the particular desktop would be the file managers - Nautilus and Dolphin. And I believe you can run Dolphin in Gnome if you really want to.

Adding Xubuntu won't slow your system down. In fact the magazine Linux Format does a Gnome/KDE/Xfce special of Ubuntu on the cover DVD a month or so after each release. Which means that the Karmic version will probably appear with the Christmas issue. What a nice present!
