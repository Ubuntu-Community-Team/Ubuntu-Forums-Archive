### Post by stracky on 2011-11-28
So what you are asking is: If my computer has nothing at all on it, how do I install Linux?

What your computer will do upon booting is check to see what bootable devices are available.  Generally, the computer can boot form the Hard Drive, CD/DVD Drive, or USB Drive.
The CD drive usually takes precidence over the HDD, but to boot off a USB Drive you will have to hit F2, F12 or something when your computer is in POST (The first thing that shows up when the computer turns on)
The USB will boot just like an Install CD, and prompt you to install Linux Mint.  If you aren't going to put anything else on it, I suggest having it format the HDD and use all of it.

As for as *getting* the bootable USB, the Mint website does not appear, from what i can see, to offer a direct USB maker like Ubuntu.  You will have to download the CD .iso file, and then put it on USB with [Universal USB Installer]("http://www.pendrivelinux.com/universal-usb-installer-easy-as-1-2-3/") (Windows), or Unetbootin (Linux)
You don't need Persistance for an Install CD

Or you could have your Linux-based friend make it for you.:lolflag:
