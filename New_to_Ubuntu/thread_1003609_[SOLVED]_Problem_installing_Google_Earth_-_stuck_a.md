Just followed these steps - 

[I]Now copy and paste the following commands:
Code:

sudo wget [http://www.medibuntu.org/sources.list.d/intrepid.list](http://www.medibuntu.org/sources.list.d/intrepid.list) --output-document=/etc/apt/sources.list.d/medibuntu.list

Code:

sudo apt-get update && sudo apt-get install medibuntu-keyring && sudo apt-get update[/I]

[I]Now that you have the medibuntu repository, you can find Google Earth.

You can install Google Earth with Synaptic Package Manager or with terminal.
The command for the terminal is

Code:

sudo apt-get install googleearth[/I]
