### Post by kellemes on 2009-10-16
I'm not using Gedit myself, using [Geany]("http://www.geany.org/") myself (has this feature neither)..
But as far I know Gedit doesn't offer this..
It does offer "incremental search", maybe an acceptable alternative?

Copy/Paste from some place..
```
For example, if you search for &#8220;the&#8221; you would first type &#8216;t&#8217;,
and gedit would highlight every &#8216;t&#8217; in the document.
You would then type the &#8216;h&#8217;, and gedit would highlight every occurrence
of &#8220;th&#8221; in the document. Finally, when you type the &#8216;e&#8217; gedit would
highlight every occurrence of &#8220;the&#8221; in the document.
```
