### Post by CharlesA on 2012-11-09
> **thenextbigthing said:**
> 
1. what kind of hardware do I need to maintain a busy high traffic server, of course traffic will be low at first but when word gets around hopefully it'll be busy.

Depends on the traffic you will be getting. I've heard Atom boxes work fine, but it all depends on how much traffic your site is getting.
 
> 2. Im using the server software to host my companys website, I don't want to pay people to host for me I host myself, is this server software ideal for this purpose?

Is this for your own company or the company you work for? Personally, I would have it hosted elsewhere, so you don't have to worry about uptime/hardware/backups. I would definitely go this route since you are new to *nix.

Another thing to think about is if you are running this from a home net connection, most ISPs block port 80 and port 25 and have a clause in their ToU that prohibits running servers.
