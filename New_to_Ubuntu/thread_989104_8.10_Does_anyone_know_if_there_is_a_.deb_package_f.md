### Post by .arean on 2008-12-09
I'm sure you've got everything sorted out by now but for future references the standard ubuntu CD has many of the primary packages, including build-essential and all the required dependencies, on the disc. Just pop in your ubuntu CD, edit your software sources (System > Administration > Software Sources) and at the bottom of the first tab is a little window titled "Installable from CD-ROM/DVD". Tick the box next to "Cdrom Ubuntu 8.10 'Intrepid Ibex' (or whatever version you're using), then hit close. You will then get a pop up saying your software sources have changed and need to be updated, just click Reload and let it do it's thing. Since you don't have an internet connection you'll likely get some errors about not being able to update some of the sources. Just ignore these as you don't need them. Now open up Synaptic Package Manager (System > Administration > Synaptic Package Manager) and search for build-essential, double click it. A box will pop up saying it needs to mark some additional packages, just click mark. Now hit apply on the tool bar and synaptic will install build-essential with all the required dependencies from the CD. When you're done just untick the cd-rom box in software sources either through the location I mentioned earlier or  Settings > Repositories in Synaptic.

I'm on dial-up and need build-essential to install my modem drivers before I can have internet access. Before I discovered this method I was running back and forth like you were trying to get all the dependencies needed. Again, you probably got everything sorted out but it's a handy little trick to know and I'm quite surprised nobody mentioned it sooner.
