### Post by lykwydchykyn on 2012-12-25
On that hardware I'd probably steer for something lighter than any Ubuntu derivative.  Slitaz or TinyCore would run reasonably on that hardware.  Puppy would probably run well, but I'm always a little iffy on Puppy because of the whole run-as-root thing.  I've been doing some testing of different OS on Pentium II / Pentium III computers lately, and I can tell you at that level there's a real difference between built-from-scratch lightweight distros and regular-distro-with-a-light-window-manager distros.
