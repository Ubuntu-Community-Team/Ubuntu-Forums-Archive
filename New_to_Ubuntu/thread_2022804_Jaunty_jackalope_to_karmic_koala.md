### Post by kurt18947 on 2012-07-11
You could also try Puppy.  There is a version of Puppy based on 10.04.  I don't know if support on that will extend beyond 10.04's EOL or not.  Puppy also has a Slacko version.  These are supported.  Except for the video card, your system doesn't look all that bad.  I'd also suggest if you haven't tried Lubuntu recently, download and burn a Live USB if your system will boot from a USB-HDD.  USB-HDD is how live USBs are seen, at least on my desktop.  

If you aren't able to boot from USB and your CD/DVD is busted, life might be difficult.  I did pull a hard drive out of a notebook, installed it via an adapter in a desktop, disconnected the normal hard drive and was able to install Lubuntu on the notebook hard drive. Removed the hard drive from the desktop and reinstalled in the notebook.  It booted and away we went.  Try *that* with Windows :P.  The notebook is a 10 year old PIII 1.1 Ghz w/ 512 RAM and crappy Intel video.  I think if you can get Lubuntu installed somehow you might be fine.  It (Lubuntu) has matured rapidly; 12.04 works pretty well.
