### Post by e4uforums on 2010-04-06
The biggest advantage of a 64bit OS is the ability to address more memory.  

With a 32 bit OS you can address 4GB of memory (RAM and Video Memory combined).  A 64 bit OS will address something in the neighborhood of 1TB of memory.  

So, if you want to add more than 4GB of ram, or if you want to use 4GB of ram with a 1GB or more video card, definitely go with the 64bit version.  There may be other performance improvements by using a 64bit OS, but I believe they would be less noticeable.
