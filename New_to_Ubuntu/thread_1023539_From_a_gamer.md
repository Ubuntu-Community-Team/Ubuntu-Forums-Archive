### Post by HavocXphere on 2008-12-28
> **BDubs588 said:**
> is there a surefire solution to gaming seriously with ubuntu?
Not really no. 99% of the games are written for windows, so running them on linux is, by definition going to require work-arounds and emulation etc. You can run stuff with WINE, CrossOver etc but its not surefire.

So you're probably going to have to dual boot for now.
