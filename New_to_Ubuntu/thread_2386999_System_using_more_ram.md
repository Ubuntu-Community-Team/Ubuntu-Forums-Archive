### Post by Autodave on 2018-03-13
Looks normal to me. You are only using 2 gig plus a tiny bit of *swap.  Swap* is space on the hard drive: it is space used when the system runs out of available RAM: it takes some of what is being used by the RAM and writes it to the hard drive so as to free up more RAM for your program(s) to use.

In the future you may want to consider using a lighter version of Linux such as Xubuntu. Xubuntu is the same operating system but uses a lot less RAM for the *desktop* to function. This would allow your programs to run faster.
