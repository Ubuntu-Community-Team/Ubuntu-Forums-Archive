### Post by oldrocker99 on 2016-06-01
I removed the URL in your post.

If you have a blank hard drive, burn the .iso to a thumb drive or DVD and boot with it. The installer will format the drive and install Edubuntu on it. If you want to dual-boot, install Windows **first**, then install Edubuntu.

This is a pretty old laptop, and 512MB isn't a lot of RAM. It's also 32-bit, so get the i386 installer. Here is where to download it:[http://cdimage.ubuntu.com/edubuntu/releases/14.04.2/release/](http://cdimage.ubuntu.com/edubuntu/releases/14.04.2/release/). Download **edubuntu-14.04-dvd-i386.iso** to get started.
