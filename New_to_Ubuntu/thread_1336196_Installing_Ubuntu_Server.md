### Post by harfa on 2009-11-24
just type 'sudo nano /etc/resolv.conf'

the file will be made for you, then just type the DNS server
addresses primary, secondary etc

ie.

xxx.xxx.xxx.xxx
xxx.xxx.xxx.xxx

press ctrl-o to save and crtl-x to exit nano
