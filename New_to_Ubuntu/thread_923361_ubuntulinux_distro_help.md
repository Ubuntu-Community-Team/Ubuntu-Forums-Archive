### Post by Orange_and_Green on 2008-09-18
If by "Applications" you mean graphical/dektop applications like openoffice, pidgin, etc. you could try Ubuntu server edition. You can even choose not to install ANY servers then :)

Basically you would start with a command prompt because the server edition comes without a graphical GUI; if you want one the first thing you should do would be:
```
sudo apt-get install xorg xterm gdm icewm menu firefox gksu synaptic
```.

Have fun, good luck:KS

I guess the minimal install is more on the spot...
