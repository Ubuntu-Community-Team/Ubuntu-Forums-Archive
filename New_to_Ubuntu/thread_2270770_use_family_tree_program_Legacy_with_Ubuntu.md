### Post by grahammechanical on 2015-03-25
A bit more information about this application would have been helpful.

I am assuming that Legacy is a program built to run on a Windows operating system. If that is true then it cannot run on Ubuntu which is a Linux distribution. The developers of Legacy may provide a Linux version But it would need to be packaged for Ubuntu which means the package that contains the program needs to have .deb as the file extension. If it has .rpm then it will not install in Ubuntu unless it has been converted to the deb package format.

Now, we can start to see why the recommendation is to run this program on Linux using a virtual machine (VM) that has Windows installed into it. A possible alternative is to install Wine, which again is not an emulator. The Wine application will let us install Windows programs and run them. But there is often a question mark over the success rate.

See if this program is listed in their database (AppDB) and see what rating of success it has. If any.

[https://www.winehq.org/](https://www.winehq.org/)

What would be the purpose of waiting? Why not search for a Linux alternative?

### Post by Geoffrey_Arndt on 2015-03-25
Legacy family tree will also run in "PlayOnLinux" (Wine) which is in the Ubuntu Software Center.   

Also, there is the "Gramps" genealogy program (native to Linux).   It runs great and is also in the Ubuntu Software Center.   The files from Legacy can be imported into Gramps .gen? or whatever the common format is.

No need to wait if you're interested in genealogy.   [https://gramps-project.org/](https://gramps-project.org/)
