### Post by sTpny on 2008-09-03
Modems are tough in linux, as a general rule. You need to get your computer to recognize and use your modem, which it isn`t doing currently.  If you have a winmodem, I`d suggest a downgrade. Winmodems are a nighmare to set up in linux. Go get an old modem and put that in, and don`t run the disk your isp gives you.. call them and ask for the settings to use.. thats all the install disk is really doing anyhow. the easiest solution is to use a high speed provider, as the setup is automatic for nearly all ethernet cards, but not everywhere has high speed, and not everyone can afford it. Good luck.
