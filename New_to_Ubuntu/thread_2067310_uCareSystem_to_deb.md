### Post by lance bermudez on 2012-10-06
I had to post this so other slow witted people like my self could be at peace with wondering how to get the uCareSystem to work without having to change to the dir ware the ucaresystemcore.sh is every time you want to update the system. I at first thought I would have to convert uCareSystem in tar.gz to .deb. You don't need to do that just download the .tar.gz file and unpackage it. Then open terminal to ware the file is and run sudo ./ucaresystemcore.sh

next time you need to run the program just do 
sudo updatecore

I found this on the web cite after a spent about a hour googleing have to convert uCareSystem in tar.gz to .deb. uCareSystem Core has the same functionality with uCareSystem but without GUI.
Also, the first time you run the uCareSystem Core, it will copy its self to your system so that the next time you don't have to run the executable again. You will need just to run :

sudo updatecore
