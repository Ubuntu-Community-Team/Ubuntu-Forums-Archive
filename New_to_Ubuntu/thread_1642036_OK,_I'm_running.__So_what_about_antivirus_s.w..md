### Post by Ahunt on 2010-12-10
We don't want to be forwarding viruses to our friends who use Windows now, do we?

If you do download a suspicious file and don't have ClamAV installed to scan it with you can also upload to to one or both of these free services:

*[Jotti's Malware scanner]("http://virusscan.jotti.org/en")
*[Virus Total]("http://www.virustotal.com/")

They both scan your file with two dozen or so different scanners (like AVG, Panada, Sophos, Kaspersky, etc) that are kept up to date for virus definitions and then in a minute or so give you a report. These are also very handy for a "second opinion" on a suspicious file if ClamAV says it is clean. Both of these have a 20 MB limit.

### Post by bodhi.zazen on 2010-12-10
> **Ahunt said:**
> We don't want to be forwarding viruses to our friends who use Windows now, do we?

And how would we do that exactly ? Since viruses do not propagate on a Linux system we really can not pass them on. If we receive or download a file we can be a passive conduit, but then you should not be passing untrusted files to your friends from untrusted sources - problem solved.

If you deal in such things, antivirus can help, but it is of little to no use for the vast majority of Linux users due to the sheer numbers of false positives (you are an order of magnitude more likely to find a false positive then a true positive).

You are far better off teaching your Windows brethren to secure their boxen and use antivirus for themselves (teach a man to fish ... )
