### Post by Mr Tim on 2009-04-15
GNOME is more popular in ubuntu, KDE looks pretty but some people may find it less usable. XFCE is the lightest (and my personal favourite). All of them are reasonably stable (as far as I know) and they can all run all the apps in the repository. However some apps are based in different toolkits, so you may find that you will need Qt installed (upon which KDE is based) in order to run some programs in GNOME or XFCE. The converse is true with GTK+ upon which both GNOME and XFCE are based.
In summary I would probably advise GNOME for a new user, unless you have limited system specs in which case I would use XFCE. Remember GNOME can always be themed to look better if you want.
