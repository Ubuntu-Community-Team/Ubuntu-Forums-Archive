### Post by Paqman on 2009-12-30
Most hardware should be automatically detected, you don't need to install anything. However, some devices like graphics cards and wifi will need a driver to be installed. The system will prompt you if this is required.

Some devices however are not compatible with Linux at all. Getting these to work will usually require specific instructions for that device.

Was there something in particular you wanted to get working?
