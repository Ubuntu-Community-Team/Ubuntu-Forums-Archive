### Post by LokiScarlet on 2021-11-24
In theory, Nicehash may run on Linux using WINE, but don't rely on it as CUDA support in WINE can be flaky or sometimes nonexistent. They do have a Linux-based miner called Nicehash OS, but I'm not sure what that entails.
There is also an open-source version of the miner, but it's a Visual Studio project with Microsoft dependencies. If you can cross-compile it onto Linux from a Windows machine, it might work. [https://github.com/nicehash/NiceHashMiner](https://github.com/nicehash/NiceHashMiner)
