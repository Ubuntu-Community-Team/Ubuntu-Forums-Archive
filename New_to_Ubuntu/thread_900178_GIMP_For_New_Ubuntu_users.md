### Post by aimpau on 2008-08-25
Don't really know where to put this but I want to let all of those beginners, specially if you've been using Photoshop CS, that if you're now using Linux and come face to face with GIMP, try the ff below:

Open a terminal and type

```

sudo apt-get install gimp-data-extras gimp-plugin-registry

```

these would install additional brushes and the ever Photoshop important tool: Layer properties that gives you pattern,color,gradient overlay, inner and outer glow and etc.. By default, these were not installed. If I hadn't looked into synaptic, I would have already installed wine+PS7.

Have fun!
