### Post by SeijiSensei on 2018-04-09
I don't understand this problem.  If you attach a PDF to an email on any platform, it will be sent as a binary file without any editing.  I send PDFs from Thunderbird on Linux all the time.  They're never altered when they arrive.

If you get files in .docx, you can open them in LibreOffice Writer and export them to PDF.  You may need to make some minor edits in Writer because of proprietary features in Word.  Compare the original and the PDF version and adjust as needed.
