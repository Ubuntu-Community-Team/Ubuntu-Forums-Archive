### Post by cmnorton on 2009-01-07
As a suggestion, posting a more descriptive post title will go a long way to getting you help. 

This is a good place to start for hardware requirements.

[https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements)

Ubuntu supports many resolutions; it depends on your graphics card.

You can disable using the keyring through Network Settings. 

Given you have an older system, I would recommend installing using a text mode installation, but first have a look at the hardware requirements.
