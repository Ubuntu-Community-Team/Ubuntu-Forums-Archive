### Post by freak42 on 2009-11-12
afaik all external HD's also run with linux, you shouldn't have any problems with it.

One thing to consider is what file system you want to use on that disk.
If it is stationary and always connected to a linux box consider a linux native filesystem like ext3 or 4.

If you also intend to plug it to an osx or windows machine every once in a while you should leave it as fat or maybe ntfs.
