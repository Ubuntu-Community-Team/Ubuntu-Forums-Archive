### Post by PhrankDaChickenGeek on 2009-01-02
> **Paddy Landau said:**
> I don't want to use any wallpaper for my desktop. The background colour does fine for me.

The Appearance Preferences window gives only options to change the wallpaper, not to remove it.

How do I remove the wallpaper?

The upper left "wallpaper" is really just a the background color - Select it, then choose a color

To remove the "saved" wallpapers, delete the file /home/$USER/.gnome2/backgrounds.xml
