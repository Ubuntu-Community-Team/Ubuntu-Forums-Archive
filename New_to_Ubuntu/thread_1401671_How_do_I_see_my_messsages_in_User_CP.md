### Post by audiomick on 2010-02-08
Under "search" at the top of the thread page there is a "show all your posts" option in the popdown.

You can also go to "advanced search" out of the same list and use your user name (or anyone elses) as a search criterion.

### Post by lovinglinux on 2010-02-08
You could also use Google site search with the following search string:

[Alfie O'mega site:ubuntuforums.org]("http://www.google.com.br/search?q=Alfie+O'mega+site%3Aubuntuforums.org")

Just add the keyword you want to find to that string and you will find your message pretty fast. I always do this when I want to find a message to quote that I haven't bookmarked. Is much easier then going through the User CP when you have thousands of posts. In fact, I have a search engine to look for my posts on Ubuntu forums. It makes my life a lot easier.

For example let's say I want to find a post where I recommend a mencoder command to add subtitles. So I search for...

[lovinglinux mencoder command subtitles site:ubuntuforums.org]("http://www.google.com.br/search?q=lovinglinux+mencoder+command+subtitles+site%3Aubuntuforums.org")

...and voilà, the first result has the command I need.
