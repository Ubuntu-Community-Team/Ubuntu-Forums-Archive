### Post by JC Cheloven on 2011-08-08
Hi, some points from an enthusiast of old pc's.

- What you expect to do with the pc is the crucial point, and we don't know that yet! For example, you could browse the web, but flash will be too much. You'll be able to edit text documents, but libreoffice will perform slow (or have problems), you should look into lighter alternatives. Video editing and other heavy tasks are probably out of reach...

- I wouldn't recommend at all an unsupported, 15+ y.o., Microsoft operating system!! If you want something like this which is supported and will run without problems in your pc, have a look at [freeDOS]("http://www.freedos.org/") 

- [Puppy linux]("http://puppylinux.org/main/Overview%20and%20Getting%20Started.htm") will run just fine in your pc, and it is a very nice distro.

- More ram is desirable, sure, but you'll be able to run ubuntu anyway if you want. The procedure is, in short:
[INDENT]-- Use an alternate CD (not a live cd), and install a command-line system only. Don't forget to set some swap space.
-- From here you have options, you can install a bare window manager only, like openbox, and a small set of well choosen applications (midori, pcmanfm...). This approach is very easy on resources, although it will seem a bit strange at the beginning. You will have no panel for example. But Openbox menu is customizable enough to get a functional system.
-- Or you can install the package lubuntu-desktop, wih will give you openbox + lxpanel + a set of cuasi-light applications.[/INDENT]

That should work in your box. If interested, please come back to ask for details if needed.

EDIT: if you decide to go for something really light (and more limited, that's the problem), like slitaz or tinycore, your system will not be slow at all. It will fly! As mentioned: what do you expect to do with this pc? One of these could  be enough, and eventually be a better choice.
