### Post by dark.power on 2010-08-02
you could try installing Compiz Fusion Icon:

```
sudo apt-get install fusion-icon
```

and then when you start it and it appears in the tray, right-click the icon and select "reset window manager"

It might not work, but I've had some similar problems, and this seemed to work for me.
