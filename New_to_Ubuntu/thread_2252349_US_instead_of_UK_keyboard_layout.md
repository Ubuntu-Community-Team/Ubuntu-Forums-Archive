### Post by Dennis N on 2014-11-11
To select another language:

Preferences > Keyboard Input Methods > Input Method Tab > Click "Add" and Select another Language from a list (click the dots to see more than is initially shown).

### Post by Rex Bouwense on 2014-11-11
An alternative would be to right click the US in the LXPanel and choose Keyboard layout handler settings.  Uncheck Keep system layouts and then choose Add from System layouts.  Then select gb for the UK.  Move it to the top so it becomes the default.  Then don't forget to re-check Keep system layouts.  It will take effect upon a reboot or new session.
