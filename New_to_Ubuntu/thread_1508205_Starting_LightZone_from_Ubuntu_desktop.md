### Post by Theo L on 2010-06-12
I think I have a typical beginner question:

Most of the time when I install a program - via Ubuntu software center for instance - I get to see the program in the drop-down menu ' applications'.

Now I have bought a program for Linux called LightZone (a sort of LightRoom but different, LR is not available for Linux) and it works properly.

However...to start this program I have to start the Terminal and type:
cd lightzone
./ LightZone

And while this works, I wouldn't mind being able to add this program to the application menu and/or placing an icon on the desktop. 

How do I achieve this?

Thanks, Theo

---

### Post by Irony on 2010-06-12
System > Preferences > Main Menu

From there create your own button note your command will be;

```
lightzone/LightZone
```

or probably just;

```
LightZone
```

Once you've done that then navigate to your button and right click on it so you can add it to your desktop or panel.
