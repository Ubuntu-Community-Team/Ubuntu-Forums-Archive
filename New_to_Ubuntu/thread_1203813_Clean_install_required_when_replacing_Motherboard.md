### Post by nhasian on 2009-07-04
unlike windows that has a registry and borks if you replace the motherboard or hard disk controller, Linux will autodetect all your goodies on startup.  no need to reinstall.  If you wanted to do a fresh install to separate your /home or /data folder or switch to a faster/better filesystem like EXT4, then you may want to consider a fresh install.  Ubuntu Karmic Koala Alpha 2 has been running great for me, and Alpha 3 is right around the corner.
