### Post by Calmor on 2010-06-14
I have an Athlon XP running at 1.2GHz I believe, with 512MB of RAM.  It's got a stock 8.04 install of Ubuntu on it, and with a decent video card, has all the Compiz effects you could ask for, smoother than my dual-core AMD x64 laptop with 4GB of RAM.  Other things take longer of course, but it's perfectly functional and far faster than XP Home on the other partition.  I currently only use it to scan in documents, since it has an even older HP scanner with 25-pin SCSI interface card hooked up to it.

I don't know how much slower it would be with 10.04 installed.

Otherwise, even Xubuntu (by all accounts, a bloated version of a usually slim XFCE interface comparted to other XFCE distros) should work just fine on that machine.

Or, if you want to use it as an experimental box, you could do a lot of things with it.  Try installing something else on it - different distro, or a command-line only version of Ubuntu.  Use it as a file/print server.  Try installing a build-your-own-system distro like Gentoo on it if you're game.  Install the alpha/beta/RC of 10.10 and report bugs to the dev team.  Use it as a guest computer.  Try Puppy or DSL (Damn Small Linux) to see how fast you can make that old machine feel. 

A P4 with 512MB may not be much today, but it's still plenty of machine, and a sufficient test bed.
