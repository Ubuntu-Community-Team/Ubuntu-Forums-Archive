### Post by sffvba[e0rt on 2011-11-06
Open a terminal (try pressing Ctrl+Alt+t or go to Accessories-> Terminal (all depends on which version of Ubuntu your using)))

Then type ```
sudo apt-get update && sudo apt-get install ubuntu-restricted-extras
```

When requested enter your password (it won't seem like anything is happening but enter the password and hit enter when done)

Please note you have to be connected to the net for this to work ;)

If all goes well your movies and music will work.
