### Post by Jatin_kaushal on 2015-07-02
Haha. I love new people into linux!
Try compiz. For what I do, I always install compiz when I install a new distro.

[B]What is compiz?
[/B]Compiz is one of the most popular and ( in my opinion ) the best window manager!

[B]What is a window manager?

[/B]When you where using windows, and if you are like me, and that windows was win 8.1, then you probably tried to get aero. You can do it, but there is a chance to break the system. So in comparison to linux or Lubuntu in you case, a window manager is the thing ( software layer ) which is responsible for all the effects ( window effects ) and decorations (just like aero!).
[COLOR=#808080][FONT=arial][www.youtube.com/watch?v=nqjG4q59-w8]("http://www.youtube.com/watch?v=nqjG4q59-w8")[/FONT][/COLOR]
This is a video which shows SOME, not all effects compiz can do. But this video is old. to install compiz, open up your terminal or your software installer. (psst, if you want to feel geeky, use the terminal :D)
enter in this to install compiz and it's settings manager:
```
sudo apt-get update && sudo apt-get install compiz compizconfig-settings-manager
```
And this to install some good effects
```
sudo apt-get install compiz-plugins-*
```

[B][I][U]WARNING: USING THE COMPIZ CONFIG SETTINGS MANAGER (CCSM FOR SHORT) CAN BREAK YOUR SYSTEM. USE IT CAREFULLY


[/U][/I][/B][B]Well I installed compiz, how do I run it?
[/B]So the name of your current window manager is probably openbox. It is a really good lightweight window manager, but incredibly boring. So we need to replace it with compiz! to do that, type (in the terminal):
```
 compiz --replace 
```

and it should start. Now tell me if you want to learn more or just have fun looking at your desktop! Either way, if you do want to learn, I'll tell you some awesome 'linux-y' sites. If you don't, well there is always this forum for both purposes!

Oh ya, by the way. [B][U]WELCOME TO LINUX,YOU'RE ALREADY ACCEPTED!
