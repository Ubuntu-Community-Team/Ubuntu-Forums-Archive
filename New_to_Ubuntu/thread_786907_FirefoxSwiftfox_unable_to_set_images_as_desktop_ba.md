### Post by tjwoosta on 2008-05-08
why not just save the picture to the computer then set it as background that way?

just right click the picture and go to "save image as" then choose a folder to put it in that you can remember. After just right click the desktop and go to "change desktop background" then find the folder that you saved the picture to DONE

no real need for firefox or swiftfox to set it to the background
