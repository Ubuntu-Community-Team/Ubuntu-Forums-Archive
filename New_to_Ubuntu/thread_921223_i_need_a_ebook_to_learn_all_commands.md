### Post by scorchgeek on 2008-09-16
Just go to real.com and find the Linux version (they have one). Then you need to add execute permissions, so open a terminal, go to wherevery you have the file, and type chmod +x [filename]. Then type sudo ./[filename]. (Don't add the []'s to the filename.) 

This should install RealPlayer. Hope it helps.
