### Post by cozmicharlie on 2008-07-29
Not sure exactly what the problem - is it they cannot access your site or cannot access the folders on your site?

Most likely you need to set up a DNS server.  It's easy and free.  This is a good one [http://www.dyndns.com/services/dns/dyndns/](http://www.dyndns.com/services/dns/dyndns/).  You most likely have a dynamic ip address.  Dynamic DNS will take care of this and bind a site to your IP even if it is dynamic.  The site I linked to has a lot of good info that explains it.  Basically, you sign up for a free site, bind your IP address to the site and then your friends access your site using the address provided by Dynamic DNS.

You should also check any firewalls.

Also, make sure you have the proper security set up.  The nice thing about xamp is it is easy to set up security.

Enjoy
