### Post by gordintoronto on 2010-12-08
> **Jables2121 said:**
> so that means that once i install the hardware to the computer with the installation disk it will automatically work on both windows 7 and ubuntu 10.04 without have to add drivers or use some program?

No, it will "just work" in Ubuntu, but you'll still need to install drivers for Windows.

In Ubuntu, you need to tell Network Manager what SSID to connect to, the encryption type and the password, but you don't need to install software or use an installation disc. I was astonished when I installed my D-Link DWL-G510.

The same things are also required in Windows.

### Post by gordintoronto on 2010-12-11
> **Jables2121 said:**
> well my question was would i be able to connect to my router with that adapter if the router was not as strong as an 802.11n connection

It should work just fine at a reasonable distance, up to maybe 20 meters. If you have any difficulty, the most likely cause is that a neighbour has a router on the same channel. In some apartment buildings, every channel is in use!

Steel beams or reinforced concrete can also affect connections. Sometimes just moving the router a meter or so can make a difference.
