### Post by icanfly0307 on 2008-11-07
Did you update Ubuntu from a previous release (ie 7.10 to 8.04 etc...)? I would recommend a clean install above all other stuff. When your computer boots and the grub screen comes up with the countdown timer, press "ESC" on your keyboard. Scroll down to the last option that says something like "Ubuntu memTest +86"...

This should tell you if you have faulty RAM or if Ubuntu is detecting your RAM correctly before the system loads...

Let me know what happens and I'll try to help you out further... :)

Good Luck!
