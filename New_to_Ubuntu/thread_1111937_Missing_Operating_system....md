### Post by ibuclaw on 2009-03-31
To make a USB pen drive bootable, I prefer the use of unetbootin.

You can install it from one of the Ubuntu PPA's: [https://launchpad.net/~gezakovacs/+archive/ppa](https://launchpad.net/~gezakovacs/+archive/ppa)

Also, for Netbook Operating Systems, there is: [http://www.geteasypeasy.com/](http://www.geteasypeasy.com/)

And Ubuntu are also going to release their own brand of Netbook Remix: [http://cdimage.ubuntu.com/releases/9.04/beta/](http://cdimage.ubuntu.com/releases/9.04/beta/)
But it is currently in beta, so may not be the most appropriate option to go for.
