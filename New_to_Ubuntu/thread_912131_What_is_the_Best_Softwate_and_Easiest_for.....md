### Post by redseventyseven on 2008-09-06
Don't know about "best" *per se*, but you can do that in Brasero.

Chose your files and folders as though you were going to burn them to a CD. Then click "Burn...". In the dialog box which appears you can select which drive you want to write to - which also includes an option to write to "File image".

Edit: I wouldn't be surprised if you can do a similar thing using other CD burning packages as well.

### Post by Rhubarb on 2008-09-06
As redseventyseven said, you can use Brasero to burn to a file (which means it just gives you an iso file, it doesn't actually burn to a CD).

It's also possible to use nautilus:
Find the files / directories you want to burn, copy them, Places --> CD/DVD Creator, and paste them in there.  Press the "Write to Disc" button, then select change the "Write disc to:" option to "File image", it'll create a nice iso file for you.
