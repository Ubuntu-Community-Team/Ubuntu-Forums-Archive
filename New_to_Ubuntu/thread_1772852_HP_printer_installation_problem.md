### Post by 3rdalbum on 2011-06-01
Ubuntu already comes with hplip, so you should probably just go to the Printing panel (hit the Super/Windows key and type "printing" on Unity, or go to System > Administration > Printing on Gnome) and just tell Ubuntu what sort of printer you have there.

If this is a wireless printer, I think you have to have it connected to your computer for the setup before you can use it wirelessly.

The very-helpful website you linked to says that Ubuntu 10.10 supports your printer out-of-the-box, so you really shouldn't need an additional driver for it.
