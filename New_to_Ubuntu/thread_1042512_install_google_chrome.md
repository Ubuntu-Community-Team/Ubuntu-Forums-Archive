### Post by Muffinabus on 2009-01-17
As far as I know there isn't a native linux port for Google Chrome yet, it is still in development.  You would have to install it under Wine for the time being.

Upon further research, apparently Google is going to try to have Mac OSX and Linux versions out by the first half of '09.  There is apparently a 'test shell' out there for Linux but it probably isn't recommended for normal use and it most likely lacks a lot of features.
