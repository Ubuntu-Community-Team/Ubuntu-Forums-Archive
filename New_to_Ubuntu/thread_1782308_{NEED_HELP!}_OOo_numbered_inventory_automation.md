### Post by Terl on 2011-06-14
What you are looking for is a basic database.  I would recommend you look into installing Libre Office Base.  It is like Microsoft Access (but free ;)).  It is available via the Ubuntu repositories using synaptic package manager, it is the first one listed in Databases category.

A database like this could be used from inventory receipt all the way through to point of sale (if you went big) or could be anything in between.  There is much information to be had on the internet and I would check the Libre Office site and community and search the net for "relational databases".  It is not hard at all, just new maybe :D

To answer also, the front end (the forms the users see) can make it easy for less compter literate folks to use the program
