### Post by Temposs on 2010-02-02
Even if a program is open source, that doesn't mean it will run on both winXP and Linux. A lot of open source programs only run on winXP. Also, this type of program seems rather specialized and not really something the open source linux community would be doing a lot for.

So, I googled a bit, and it doesn't seem like there's much in the way of free software in this area. Here is a list of options I found. None of them are free or open source: [http://www.pearsonassessments.com/NR/rdonlyres/C99D3F42-001D-4DC4-B690-F1EFE387AA79/0/2009ToolstoSupportPositiveBehaviorintheClassroom.pdf](http://www.pearsonassessments.com/NR/rdonlyres/C99D3F42-001D-4DC4-B690-F1EFE387AA79/0/2009ToolstoSupportPositiveBehaviorintheClassroom.pdf)

If your goal is to move your systems to Linux, SWIS will be the better option in the end, because it's web-based, which is more cross-platform, as long as they don't start using ActiveX or Silverlight, which it doesn't look like they do.

EDIT: You could write your own, if you're committed to making a resource like this available.
