### Post by mister_pink on 2009-05-04
I've never lost a connection whilst doing it, but I think you should be ok, it downloads all the packages first, then starts applying them, so I don't think it should end up half done.

As for the amount of stuff to download, the answer is it depends, mostly on how many extra things you have installed (the more programs you have, the more there are to upgrade). 

I think that you can start the upgrade process and it calculates all the changes you need to make, including an estimate of the amount it will need to download, before it starts doing anything*. It will display a summary and ask if you want to continue. 

*It does actually do a couple of things, but it undoes them if you cancel. So don't panic when it says its changing your sources or anything.
