### Post by TheFu on 2018-12-06
I've been using select/paste for decades.  Works in almost any application since it is part of X/Windows - the X/buffer -  and not tied to some "clipboard" addon.

As for how a specific terminal program works with any specific clipboard addon, the exact programs involved need to be known.  There isn't just 1 clipboard tool and there must be 30 different terminal programs. Each has slightly different behaviors and capabilities.  Make sense?

Here's an article about 20 different terminal programs for 14.04 [https://www.tecmint.com/linux-terminal-emulators/](https://www.tecmint.com/linux-terminal-emulators/)
And clipboards: [https://www.tecmint.com/best-clipboard-managers-for-linux/](https://www.tecmint.com/best-clipboard-managers-for-linux/)

Select/paste has met my needs for decades. Just select the text you want using the left-mouse, drag to the end of the selection and release.  Then use the middle mouse button to paste the selection into a text-input location.  Multiple lines or pages of text all work.

But everyone has different ways of working. My method doesn't work for everyone.
