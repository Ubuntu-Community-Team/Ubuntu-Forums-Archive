### Post by NightwishFan on 2010-10-23
The definition of "faster" is open to interpretation as well. I suppose one could say Ubuntu is able to run on lighter hardware.

What you should really say is that the strength of Ubuntu is in freedom (as in rights to use/change the software), cost-free, and flexible. There is more than one very different desktop environment, over 20,000 software packages. If you have an older computer you can run a lighter setup. Etc.
