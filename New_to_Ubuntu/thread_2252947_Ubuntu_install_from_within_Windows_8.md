### Post by coldraven on 2014-11-16
You don't say if it is a laptop or a desktop PC. 
Have you tried cleaning the lens in the DVD drive? If it still does not work replace it, they are not expensive. 
I once replaced a DVD drive on a laptop for about £10. The door of the new drive was slightly too big so I shaved it down with a sharp knife. It was also a different colour but as long as it works who cares!
As for the BIOS, is it password protected or is there no "Boot from USB" option?
If it is password protected I would sell it and buy another one.
