### Post by osamakhn on 2012-05-20
As you are coming from a Windows background. A good place to start will be identifying your software requirements and checking if that software you normally run is available for Ubuntu. If it is not then you will need to find an alternative. One good place to find alternatives is: [http://alternativeto.net/](http://alternativeto.net/)

Most Windows software have Ubuntu versions, be it Skype or Dropbox.
Some windows apps can also run on Ubuntu using the Wine emulator. More information about Wine and the list of programs it supports can he found here: [http://www.winehq.org/](http://www.winehq.org/)

To understand Ubuntu and its file system and other things better you can go through the following links:
1. [http://en.wikibooks.org/wiki/Ubuntu](http://en.wikibooks.org/wiki/Ubuntu)
2. [http://www.ubuntugeek.com/download-free-ubuntu-10-04-lucid-lynx-pdf-guide.html](http://www.ubuntugeek.com/download-free-ubuntu-10-04-lucid-lynx-pdf-guide.html)
3. [http://ubuntu-manual.org/](http://ubuntu-manual.org/)

The versions discussed above might be different from the latest release but it covers most of the general concepts to help you get started.

Goodluck!
