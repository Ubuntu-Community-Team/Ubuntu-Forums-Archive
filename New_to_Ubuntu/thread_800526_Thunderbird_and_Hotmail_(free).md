### Post by quelx on 2008-05-19
I use freepops.  It's in the repos so you just ```
sudo apt-get install freepops
```

There is also a Java based graphical user interface (gui) app 
[http://mrpostman.sourceforge.net/](http://mrpostman.sourceforge.net/)

make sure you have java installed to use it. ```
sudo apt-get install sun-java6-bin
```
