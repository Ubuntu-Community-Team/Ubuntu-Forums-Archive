What kind of system is it? Is it a laptop screen or an LCD Panel. If so, have a look on your keyboard for a function key to adjust the brightness. If not it may be that an incompatible driver has been installed for your display adapter, in which case you should perhaps go to 

  System > Administration > Hardware Drivers

And see if there are any errors there.

Can you post the output of

    $> dmesg

Straight after booting up for me please (press alt+F2 then type terminal in the dialogue that pops up, this will bring up the window to type the above command in)
