I have Ubuntu 14.04 with gnome-session-flashback installed but not Gnome 3 shell. Installing Gnome-flashback does not bring in Gnome 3 Shell. We can install Gnome 3 shell separately but it will be version 3.10. If we then want the "classic" look we should install Gnome shell Classic. It is a Gnome 3 shell extension.

Ubuntu Gnome 14.04 has Gnome 3.10. To get Gnome 3.12 we need to install at least 2 PPAs - gnome3-team/gnome3 and gnome3-team/gnome3-staging. I have just done this and it indeed does bring in gnome 3.12 libraries. I have not yet tried the third PPA - gnome3-team/gnome3-next.
