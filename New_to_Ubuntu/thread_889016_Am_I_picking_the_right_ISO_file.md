### Post by lisati on 2008-08-13
The "desktop" version, also known as a "Live CD", is good for every-day use. (Edit: this includes running on a laptop)

The "server" version is better suited to a more specialised role of having your computer run as a server, and doesn't come with a GUI (graphical user interface) preinstalled (i.e. command line only, but the GUI can be added later if you wish)

I have no direct experience of using the 64-bit version, but as long as your computer is capable of running 64-bit software I don't see why not....
