### Post by earthpigg on 2009-01-17
> **akimatsu123 said:**
> it's about time i start to learn to solve my own problems instead of typing things into the terminal that I don't really understand. 


here is what you should do:

whenever you have a problem, use the methods you already use to solve it...

but before putting that command in your terminal, type "man command" to learn wtf that command is doing.

so, for example, if you have a problem and someone on the forum tells you:

"type 'grep --something ~/home/whatever bla bla bla' in a terminal..."

then before doing it, you should run:

```
man grep
```

in a terminal and see what 'grep' does, look for '--something' to see what that does, and so forth... to understand wtf is happening ;)

man = manual. 'man pages' are manual pages.
