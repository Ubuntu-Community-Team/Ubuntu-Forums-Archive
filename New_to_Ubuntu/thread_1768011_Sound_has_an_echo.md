### Post by I2k4 on 2011-05-27
On my Acer One the Realtek driver comes with a little icon (in Windows XP) that has a lot of fancy settings, including echo effects, etc.  It's called the RealTek HD Audio Manager.  Running Ubuntu seems to use the same settings done in XP - not sure if RealTek provides a similar control for Linux, but worth checking their site and googling for it.  I've had absolutely no problems with netbook output to its speakers, external speakers or headphones from several Ubuntu versions I've tried, including desktop 10.04.
