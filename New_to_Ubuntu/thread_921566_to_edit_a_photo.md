### Post by scorchgeek on 2008-09-16
You need to add another layer on top of the photo. (Layer>New Layer>OK) Then open the Layers dialog if it isn't open (Dialogs>Layers.) Be sure New Layer is on the top, then click the second blank box on the left which appears when moving your mouse over it. It will turn into a wrench. Now draw one of the blocks using any of the tools (such as the pencil, paintbrush, and/or text tool. Now you can repeat this for each block, putting each new layer on top, clicking the wrench for the new one, and clicking off the wrench for the old layer.  The image will build up, but the blocks will be separate. Now remove the eye from one of the New Layers and the background. Save this image, perhaps as a JPEG. Put the eyes back on and repeat.

When you are done, save one with all the blocks for the initial. You will now have five pictures. Also note that if you want to have a background on each block (so you don't have to draw in the part you don't want visible) you can try one of the radio buttons on New Layer.

You will have to come up with a way to get each photo to come up (perhaps Impress, I don't know exactly how you plan to display it). 

Sorry if this is a little complicated, but it's the best way I know, unless you wish to edit the photo live. Hope it helps you.
