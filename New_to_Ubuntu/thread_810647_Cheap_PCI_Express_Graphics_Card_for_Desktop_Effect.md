### Post by diablo75 on 2008-05-28
> **X40nick said:**
> Thank you! Will it work out-of-the-box? 

The Restricted Drivers Manager should pickup on the presence of the card, but you might have to open it up to tell it to install the drivers for you.  Alternatively you could use Envy to install the drivers, but Restricted Drivers Manager handles nVidia cards pretty well.  So getting the drivers installed is as easy as a couple clicks and a reboot.  That's it.

After that, you can enable desktop effects.  I would also recommend you install the Advanced Compiz Config Settings Manager.  You can find it listed in the Add/Remove applet at the bottom of your Applications menu.  Just change the search filter to "All Available Applications" and search for "compiz" and you'll find it.  Check it off, click apply, and you'll be able to use it to enable the Cube, other animations, effects, etc.

Enjoy!
