### Post by pyromanic123boom on 2008-10-13
If the PDF wasn't made from original text, it isn't searchable. Google books are stored in images. 

To try it - open a pdf and do a quick CTRL+F for a search box. 
Or, install xpdf and use the pdftotext tool to convert.
