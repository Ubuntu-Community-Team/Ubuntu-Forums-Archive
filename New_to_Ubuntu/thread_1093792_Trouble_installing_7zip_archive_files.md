### Post by ramjet_1953 on 2009-03-11
What are you trying to install?
Archive files are just that, archives.

If you are trying to install a program from this archive, it is highly likely that it is source code and needs to be compiled.
This really is not something that you should be attempting as a new user, as you need to be conversant with the command line, satisfying dependencies and using the ./compile /make and sudo make install commands.

At this stage you would be far better off just enabling the extra repositories and searching for a program that will suit your needs.

To enable the extra repositories, just navigate to [COLOR="Blue"]Applications>Add/Remove[/COLOR]
When the window opens, just click on the [COLOR="Blue"]Show drop-down menu[/COLOR] and choose [COLOR="Blue"]All Available Applications[/COLOR].
