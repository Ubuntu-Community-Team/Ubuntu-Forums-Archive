### Post by Mark Phelps on 2012-10-31
Since Western Digital already does this with their Passport drive, I would be really concerned if Seagate preloads a Windows "widget" on their drive that does the same thing as WD.  

With the WD drive, they hardware-encrypt the entire drive in advance and you HAVE to use Windows to set/remove the password before you get access to the drive.

If you set or leave the password on in Windows, there is NO WAY to gain access to the WD drive later in Linux.

So, you might have to plug this drive into an MS Windows PC first, to remove any default password that Seagate set on the drive.
