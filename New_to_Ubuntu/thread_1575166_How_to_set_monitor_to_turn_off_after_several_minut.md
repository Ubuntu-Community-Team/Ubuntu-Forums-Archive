### Post by Hisymak on 2010-09-15
Hello. I'm using Xubuntu 9.04 with Xfce 4.4 on notebook HP omnibook xe4100.
I have one problem that monitor never turns off after some time, it ONLY turns into black screen (backlight is still on). I want monitor to TURN COMPLETELY OFF after for example 30 minutes because I often leave the notebook turned on and want to save energy.
There appears to be no possible settings in standard settings menu.
I was already looking for some sollutions but all I found doesn't work properly.

1) I tried to use "xset" command so I typed in terminal "xset dpms 0 0 60". It worked well but it was NOT PERMANENT. After I reboot system, this setting no longer exists.

2) I tried to modify the file "/etc/X11/xorg.conf" - I added there lines:
```
Section "ServerFlags"
    Option         "BlankTime" "0"
    Option         "StandbyTime" "0"
    Option         "SuspendTime" "0"
    Option         "OffTime" "5"
EndSection
