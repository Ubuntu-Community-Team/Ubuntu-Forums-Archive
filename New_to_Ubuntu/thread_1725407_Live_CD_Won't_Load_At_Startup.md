### Post by unknowngal on 2011-04-09
*waves* I'm trying to install either Ubuntu or Xubuntu on my new Windows 7 machine, but neither disk loads at startup. I select the option to test either Ubuntu or Xubuntu out first (then do the full install), then I restart, but instead of the disks loading, Windows loads. I asked a friend about this, and they mentioned that perhaps Windows is reading the hard drive first, then the DVD drive? Is there any way to change that? I hit DEL on startup and it brings up the BIOS and all that, but I have NO CLUE what to do in there, and I'm rather afraid of it. :P But I would like to have either 'buntu system running on here, whichever is the better one, too.

Sorry if the above sounds all confusing. ^^;; Thank you!

---

### Post by Enigmapond on 2011-04-09
Well it's IN the BIOS. You have an option there as to the boot order. Change it to removable disk.

---

### Post by unknowngal on 2011-04-09
Thank you for your reply! :) I changed it to have the CD/DVD drive boot up first instead of the HD booting up first, the disk was recognized, and I was able to install Ubuntu with no trouble. Thanks again! :D
