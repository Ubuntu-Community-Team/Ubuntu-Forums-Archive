### Post by stoogiebuncho on 2008-12-20
The last time you connected your camera to a Windows machine, did you use the "Safely Remove" thing to eject it?  If not, Ubuntu might think it's in use by something else and it won't mount it.

If that's not the case, open up a terminal and type 
```
lsusb
```

Paste the output back here.
