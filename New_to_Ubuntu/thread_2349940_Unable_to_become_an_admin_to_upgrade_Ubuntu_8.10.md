### Post by Impavidus on 2017-01-20
Ubuntu 8.10 was released in 2008 and support ended in 2010. So that computer is probably quite old and I doubt it will happily run the latest Windows version. It may be able to run the latest version of a light Linux, like Lubuntu. So if you want to experiment with Linux, an old Linux computer alongside your regular Windows box for important stuff is a good way to start. If you want Windows, just put in a Windows install dvd and hope it works. If the computer can handle it, the Windows installer will be more than happy to wipe Ubuntu and install Windows.
