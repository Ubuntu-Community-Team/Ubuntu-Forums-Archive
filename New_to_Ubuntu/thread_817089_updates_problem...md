### Post by ruffEdgz on 2008-06-03
Well, tunznath is correct that the fglrx is used for ATI drivers but make sure you are getting the right ATI drivers. 

I found a documentation site which discribes the Prerequisites, Install from Ubuntu repositories and General Troubleshooting.

[https://help.ubuntu.com/community/BinaryDriverHowto/ATI](https://help.ubuntu.com/community/BinaryDriverHowto/ATI)

Also, make sure all the repositories are checked. To do that, you must go to System >> Administration >> Synaptic Package Manager. When the new window pops up, click on Settings >> Repositories and make sure all the check boxes are checked on the first tab titled "Ubuntu Software".

Also, there is an application called Envy ([http://www.albertomilone.com/nvidia_scripts1.html](http://www.albertomilone.com/nvidia_scripts1.html)) and you can try using that to get the newest drivers as well. It worked for me when I was using Ubuntu 7.10 on my PC which has ATI drivers.

Hope this works for you!
