### Post by kgas on 2009-01-02
In Ubuntu 8.10 you can make your desktop icons to visible again using config editor.
Application -> system tools ->config editor. In the left  side panel select apps then go to nautilus see the desktop and select it. In the left side panel click what icons you want to be on the desktop. I assume that your are using gnome desktop.
