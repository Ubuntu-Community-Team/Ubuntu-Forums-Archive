### Post by louieb on 2009-08-01
Put both drives in the same PC. Copy and paste with Gparted.  Can copy the data and resize at the same time.  [Gparted - Copying]("http://gparted.sourceforge.net/larry/move/move.htm")  Use a [SystemRescueCd]("http://www.sysresccd.org/Main_Page")  it has the latest version of Gparted.
