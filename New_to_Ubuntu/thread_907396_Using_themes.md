### Post by swoll1980 on 2008-09-01
hit alt+f2 then type
```
emerald --replace
```
add 
```
emerald --replace 
```
to your start up programs using system>preferences>sessions then they will be there when you log on, or you  could install compiz-icon which will sit in your panel again add compiz -panel to your sessions if you want it to start automatically
