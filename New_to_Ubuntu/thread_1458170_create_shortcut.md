### Post by nhasian on 2010-04-19
> **grobar87 said:**
> How to make shortcuts for folders on desktop (home,music,pictures....)?:confused:

open up the nautilus file manager by going to Places-> Home.  Then navigate to the folder you want to make a shortcut to.  Use the middle mouse button to drag the folder onto the desktop.  this will make a link.  if you only have two buttons on your mouse then hold down the primary and secondary mouse buttons at the same time to drag'n'drop.

If you want to make a link to a program on the desktop, simply use the secondary mouse button while highlighting the program you want from the application menu and select "add this launcher to desktop"
