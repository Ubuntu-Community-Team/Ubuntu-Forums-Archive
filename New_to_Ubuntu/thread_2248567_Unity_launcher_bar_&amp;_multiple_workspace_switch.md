### Post by deadflowr on 2014-10-15
You only have one launcher, regardless of workspaces.
But you can set apps to open only in certain workspaces, by using the settings in compizconfig-settings-manager >> place windows >> fixed window placement.

And using the same tool, you can enable the plugin wallpaper and set different wallpapers per workspace.

Adding: I should note though, that KDE(Kubuntu) can do all that you want, including different launcher/panels per desktop.
fwiw
