### Post by nemesisau on 2008-05-13
Some programs don't have icons, they might be services that start at boot time (webcam-server sounds like it). Check your running processes in System>Administration>System Monitor and see if some of the stuff you installed is actually running.

Also, instead of opening Terminal, you can hit ALT+F2 to bring up the Run dialog and type it in there.
