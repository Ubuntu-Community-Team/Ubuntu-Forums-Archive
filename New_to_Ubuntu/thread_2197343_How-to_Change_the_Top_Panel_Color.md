### Post by Frogs Hair on 2014-01-03

I am going by memory and haven't had 12.04 installed since six months after it came out. The CSS method is for the Gnome Shell and not Fall-back.

### Post by ibjsb4 on 2014-01-03
If you run Fallback with effects, then you are running compiz as a window manager, go here for themes:

[http://gnome-look.org/index.php?xcontentmode=102&PHPSESSID=73582476cdabbc8f42df118b3476e087](http://gnome-look.org/index.php?xcontentmode=102&PHPSESSID=73582476cdabbc8f42df118b3476e087)

Fallback with no-effects uses metacity as a window manager, go here for themes:

[http://gnome-look.org/index.php?xcontentmode=101&PHPSESSID=73582476cdabbc8f42df118b3476e087](http://gnome-look.org/index.php?xcontentmode=101&PHPSESSID=73582476cdabbc8f42df118b3476e087)

There are also themes in the software center.  Search for &#8220;metacity-theme&#8221; and/or &#8220;compiz-theme&#8221;.
