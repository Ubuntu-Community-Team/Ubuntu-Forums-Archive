### Post by sixstringthing on 2014-04-25
Thanks for trying guys.  I found the answer on YouTube:  (It is BEST to download Unetbootin from the Ubuntu Software Center as opposed to direct from the Unetbootin web site).  Go to Dash, at the bottom of the page select the Applications icon (the 2nd icon from the left), then type Unetbootin in the Search Bar.  Select "Install" and it will install.  Follow directions from there or watch a Unetbootin tutorial on YouTube.

Using this method will avoid the huge problems I encountered when trying to download and install from a web site.
