### Post by BBQdave on 2013-12-11
Running Ubuntu Unity 12.04LTS, and screen brightness resets on reboot (might be a default function). Not much of hassle, as my notebook is in sleep mode when not in use (and retains the screen brightness setting). Reboot for updates and notice the screen brightness.

Sound level retains setting between sleep-awake and reboot.

Though not advertised, releases between LTS are still *testing*, shaking bugs out, for the next LTS release.

If you are looking for stability and rock solid performance, I humbly offer hanging with LTS :D

However, if you like the *edge* of releases (non-LTS), file bug reports and join the community :)
