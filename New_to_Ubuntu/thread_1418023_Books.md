### Post by 3rdalbum on 2010-02-28
It might be difficult to recommend a book. Linux and Ubuntu change so fast that book publishers can't really keep up - as soon as a book about Ubuntu 9.04 gets written and published, then 9.10 is out and the old book is somewhat out of date.

I've also found that a lot of Linux books are basically "Here's how to install Ubuntu, and now here's how to use Openoffice and Rhythmbox".

If you have a good local bookstore, I'd suggest taking a look in there and seeing what **recent** Ubuntu-specific books they have. Unless you really need instruction in office suites and e-mail clients, avoid the books that don't focus on how to do stuff in the operating system itself.

You may want to check out the unofficial Ubuntu Guide: [http://ubuntuguide.org/wiki/Ubuntu:Karmic](http://ubuntuguide.org/wiki/Ubuntu:Karmic)

but it often just gives some terminal commands to fix certain things, without explaining what is happening and what each command does. I don't even think it explains how to use the terminal.

the Ubuntu Manual Project may be of some use: [https://launchpad.net/ubuntu-manual](https://launchpad.net/ubuntu-manual)

It is, however, still in development; but it looks to be pretty good.
