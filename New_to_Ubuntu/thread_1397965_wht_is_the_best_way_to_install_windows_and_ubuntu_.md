### Post by merck906 on 2010-02-04
Wubi is mainly designed to "test drive". If you're sure you want Ubuntu, dual boot is the most reliable way to go. Not sure about your hard drive size, but generally, I would recommend a smaller partition for Ubuntu if you use Windows a lot. You can access your Windows data from the Ubuntu partition but not vice versa and you can always resize the partition later.
