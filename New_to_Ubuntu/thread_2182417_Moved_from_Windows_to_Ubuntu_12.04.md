### Post by fantab on 2013-10-21
Ubuntu 12.04 is LTS [Long Term Support, upto 2017] version whereas 13.04 has support for only 9months. It depends on what you like.
For me, I have two partitions dedicated to Ubuntu. One for LTS release and one for Non-LTS releases... LTS version is released every two years, so my LTS gets updated every two years, where as my other Ubuntu is update every six months.
LTS versions are more stable.
