### Post by hockeytux on 2009-09-12
Yes. 

Install Compiz. (Search for 'compiz-gnome' or 'ccsm' in Synaptic)

Then System > Preferences > CompizConfig Settings Manager > General Options > Desktop Size tab

There you can set the number of desktops you like... :D
