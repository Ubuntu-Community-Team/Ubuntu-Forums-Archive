When the package manager hits a snag with one of the listed sources, it freaks out and gives up, telling you in the details where it encountered the problems.

When your problem is a failure to pull translations from a "partner" source, sometimes the source is temporarily unavailable, sometimes it no longer exists, and sometimes the files it would lead to if it were available  are is no longer needed.

One way to bypass this problem, even temporarily, is to remove the offending source location from the "Software Sources". This did not work for you.

Another way to remove it is to make sure all package managers are closed, then open and put a comment mark # in front of the specific source entry in your /etc/apt/sources.list file.

That can only be done as su. From the terminal you could call up the editor as su with:

gksudo gedit
enter password

then open /etc/apt/sources.list from the editor's file menu.

If you first copy the files content and post it here, you can be shown which line(s) to change.

Good luck!
