### Post by dnRoyston on 2009-01-23
Okay, sorry about the quick post before, here's what you should do.

In a terminal window, type this:
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
```
Enter your sudo password, and let it copy.

Next, type this:
```
sudo gedit /etc/X11/xorg.conf
```

When your gedit window pops up, add this in on the "screen" section (Just add a new line after the "Section 'Screen'" line), and add this:
```
	SubSection "Display"
		Depth	24
		Modes		"1280x768"	"1280x960"	"1024x768"	"800x600"	"640x480"
	EndSubSection

```

This should let you use Ubuntu in 1280x768 resolution
