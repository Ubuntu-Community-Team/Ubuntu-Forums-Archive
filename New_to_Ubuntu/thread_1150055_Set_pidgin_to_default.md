### Post by pastalavista on 2009-05-05
There's no such thing as a 'default' IM client because it doesn't produce any files to open. If you want to set it to run at startup, Go to System > Preferrences > Startup Applications and add it to the list. If you click the close window button (the X in the upper right of the title bar) it will minimize to the tray. I don't know if you can make the minimize button do it.

### Post by RealPSL on 2009-05-05
Unless you do Buddies > Quit from the menu or Control + Q Pidgin will continue running. Clicking the X to close just puts it in the notification area.
