### Post by dwasifar on 2009-09-17
> **jose187 said:**
> Hi, please help me with this

Theres a feature in Dream Weaver where you can select lines code, and then minimize it to a small tab. Then you click that tag and the code will expand again.

What you're looking for is called "code folding," and as far as I know gedit doesn't do it.

Try Kate or Geany instead.  They're both in the repos.  From a terminal window:

```
sudo apt-get install kate
sudo apt-get install geany
```

Geany is more like gedit than Kate is, and doesn't bring all the KDE libraries with it, so maybe you would want to try Geany first.
