### Post by EssexEagle on 2010-05-18
> **Newbie2910 said:**
> I can't seem to figure out how to launch it... or do I have to install it separately?

If you open a terminal window and type
```
pidgin
```
what happens?  It should either run pidgin, or just tell you
```
pidgin: command not found
```

If the latter, you'll need to install it:
```
sudo aptitude install pidgin
```
