### Post by antikristian on 2009-04-20
the only real advantage of a 64bit OS is support for more ram (4GB++) Other than that there is no real advantage. With an Intel cpu, your computer will actually run about 2% slower than with 32 bit OS, AMD cpus on the other hand run about 2% faster with 64bit OS.

My experience with 64 bit OSes are that Java still is not 100% compatible with a 32bit java version (earlier you couldn't get native java on a 64bit Ubuntu, you had to use the 32bit java software) For instance I can't use BankID which is a (poorly designed and implemented) security solution for online banking in Norway.

You can run 32bit software on a 64bit Ubuntu, but I haven't felt the need to do this, as almost everything just works out of the box on 64bit these days. 

If you plan to install 64bit Ubuntu though, I would recommend to wait for Jaunty to be released, as I am very happy with it thus far:)
