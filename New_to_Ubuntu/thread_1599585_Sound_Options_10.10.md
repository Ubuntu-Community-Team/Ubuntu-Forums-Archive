### Post by Yarui on 2010-10-18
The sound options used to be much more customizable, but it seems they have decided to make it a little harder on you if you want to change the individual sounds instead of using a theme.  I found this page that explains how to change the startup sound, and I'm guessing any other individual sound will need to be manually changed in some file as well.  Here is the link:

[http://titotheman.wordpress.com/2009/11/06/changing-startup-sound-in-ubuntu-9-10-karmic/](http://titotheman.wordpress.com/2009/11/06/changing-startup-sound-in-ubuntu-9-10-karmic/)

There may be some program out there that you can install to make it easier to change the sounds, but you will probably have to do some searching to figure out how change everything.
