### Post by oldsoundguy on 2008-10-17
Mint IS Ubuntu, but in a different package with a lot of things that Ubuntu itself will not have in THEIR repositories .. things like codec files that MAY have an issue with licensing.  BECAUSE of those license issues, you will not see much about the system.  They do, however have their OWN forum. (Google is your friend).  

It is a nice compact system, and easy to install is not a good enough description.  I have it on one outdated computer (the Elissa version) and I do like it.  It is NOT, however, quite as flexible as Ubuntu itself as there are things that are NOT in the repositories.

IMO, it is the perfect system to be installed for a kid's surfer or an elderly's e-machine that is bullet proof and flexible enough to work.

It also makes for a good home office machine without a lot of stuff that the office does not need on their machine.

With a little tweaking such as dragging icons to the desktop, even a Windows user could use it without having a coronary.

It is definitely worth a shot to investigate it.  Go ahead and install it and take a look in the menu(s) as to what is there.  Think you will be surprised .. I was! (it is NOT a gamer box, but WINE IS installed!)

### Post by gatos on 2008-10-17
Mint is based on Ubuntu. The developers of Mint use Ubuntu as a base and have added various tweaks and proprietary software eg flash, Java. (proprietary software are not "free" (as in liberty) they are "proprietary" (as in commercial- but free as in no cost).

Ubuntu does come with proprietary software in the CD, but you can easily install them using "sudo apt-get install" or Synaptics.

I know what you mean by saying that Mint easier to start with as I had exactly the same dilema as you 6-7 months ago when I first started using Linux. Saying that, I installed and re installed Ubuntu and Mint a few times as I messed up my system a few times. Since then I have learned a lot and seem to be reasonably comfortable with setiing up my system EXACTLY the way i like it and troubleshoot when required. I found that Ubuntu was more to my liking and it is that hard when you get used to things to install Java, Flash or whatever else you want. 

When you are a new user with Linux the Terminal seems to be a daunting experience at first, comparing to the Windowzzz setup, but after a while when you get used to the commands you will find that they are a lot more easy to do things with. I read the same comments when I started with Linux and thought that GUI interface was the only and proper way to do things but have since found out that the command lines are so much better.  

The beauty of Linux is that you can choose whichever distro you like and if you are not happy with one of them move to the next one. I did like Mint but I find Ubuntu just better for me personally. By no means am I saying that Mint is not good, because it is. 

Lets see what you will think in 5-6 months time.

Hope this answers your question

### Post by Paqman on 2008-10-17
Back in the bad old days (like, er, two years ago) setting up all the codecs and such on a fresh Ubuntu install was moderately annoying. That's exactly why Mint sprang up.

Now that we have an automated way to install restricted drivers and cool meta-packages like ubuntu-restricted-extras it's pretty trivial to set up a new system. So Mint's advantage over mainstream Ubuntu has been eroded away to almost nothing.
