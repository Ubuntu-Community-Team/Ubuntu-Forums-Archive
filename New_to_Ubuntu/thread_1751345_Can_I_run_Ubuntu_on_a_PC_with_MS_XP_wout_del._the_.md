### Post by wep940 on 2011-05-06
And.......before you install Ubuntu along side Windows (it's called dual-boot), you want to defrag your drive in Windows a couple of times.  You see, when you install Ubuntu in dual-boot, Ubuntu will want to take part of the hard disk for itself (partition[s]).  This means it will shrink the portion of the disk that Windows is using.  To avoid data loss, it is really important to run the defrag a couple of times right before installing Ubuntu for dual-boot.
 
If, however, you just want to try out Ubuntu to see if you might want to go further with it, you have 2 other choices:
[LIST=1]
[*]LiveCD (or USB) as others have mentioned.  This will not install anything to your hard disk.  There will be some limitations as to what you can do, but it should give you a feel for Ubuntu to see if you want to continue.  Your hard drive, etc., won't be touched.
[*]Another method is called a wubi install.  This actually installs Ubuntu as just a file within your normal Windows disk.  This method is really meant only to try things out, not to use permanently.  Some people experience problems with different things using this method.
[/LIST]
