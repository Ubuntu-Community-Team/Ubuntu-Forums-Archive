### Post by twignation on 2009-03-18
I cracked it!

Basically.. (in case anyone else needs this thread)

I was expecting something like Windows, where you just right click a file and tell it to be shared, and then I can go to my network folders and see the folder I want...

So I figured, I share, then go to Nautilus and it would be in my network.

It isn't like that, there are various choices depending on what you are trying to do; for example from what I have been reading  -samba is for linux to windows connects, whereas NFS is for linux to linux connections.

This ([http://czarism.com/easy-peasy-ubuntu-linux-nfs-file-sharing](http://czarism.com/easy-peasy-ubuntu-linux-nfs-file-sharing)) tutorial worked extremely well and now everything works correctly on my network.

And WOW ubuntu runs video over my network so smoothly!!
