### Post by inameiname on 2010-06-01
First, do you have have some sort of Animated script/program/whatever that will make an animated desktop? 

If not, here is one that many folks seem to like:

[http://gnome-look.org/content/show.php/ORIGINAL+A-DESK+AnimatedDesktopGUI_EN-ES?content=88248](http://gnome-look.org/content/show.php/ORIGINAL+A-DESK+AnimatedDesktopGUI_EN-ES?content=88248)

I think that one will automatically start on startup, but if it doesn't, just add it to the startup applications:

To do so:

Go to System > Preferences > Startup Applications

in the startup tab, 'Add' it using the following parameters:

Name: (whatever you want)
Command: (depends on where the deb installs it's contents to)
Comment: (whatever you want)
