### Post by wrgb2 on 2008-05-08
Open Office writer does save a tmp copy of the document you're working on - the default is every 15 minutes.  You should be able to find it in - /home/username/.openoffice.org2/user/backup (replace "username" with your user name that you logon with).  Also, in order to see the .openoffic.org2 directory, you'll have to click on View>Show Hidden Files in your file browser window.

### Post by wrgb2 on 2008-05-08
Sorry the backup wasn't there.  To ensure backups in the future, click on Tools > Options > Load/Save > General and check "Allways create backup copy" and "Save AutoRecovery information every" x minutes, and set it to 5 minutes or so.
