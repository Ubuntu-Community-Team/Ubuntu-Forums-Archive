### Post by -kg- on 2011-07-29
I'm running Kubuntu 11.04 on an older (say, 2006) Toshiba laptop, and while it has good specs, apparently I'm not able to run some of the latest features under KDE, nor under Gnome 3...it defaults back to "classic Gnome" (2.X, I assume).  I don't know whether it would run Unity properly or not.

If it doesn't, you might want to go with 10.04 (the LTS version) which uses Gnome 2.X.  I have 10.04 loaded on a Compaq from 2000 (originally came with Windows ME), and while it's slow, it runs sufficiently for the uses to which it's being put.

With your specs, you would easily be able to run 10.04 with no problem.  And being Long Term Support, it will be supported beyond the life cycle of 11.04, to boot.

Hey, experiment with it.  If one doesn't work, you can always install another, and you won't even have to ask for your money back. :P

One thing...when you boot to one or the other of the Live CDs (or USBs), check to make everything works before you install.  That can save you a lot of time installing something, then finding it doesn't really work well.

If it works, install it; if it doesn't, try something else.  Just keep in mind that the installed version will run faster than the Live CD/USB version.
