### Post by nimbis on 2008-07-19
Yes, 3rd generation Nano's work on Ubuntu. It depends what you want to do with them. It's easy to transfer songs, although sometimes album cover art is a bit messed up or not there at all. Transferring videos is an easy thing if you use Hardy, but if you are using Gutsy it's a bit of an effort to do. If you want to transfer songs to your iPod, use a program called gtkpod. It seems to be the best of the available ones. You can get it by typing this command into a terminal:```
sudo apt-get install gtkpod-aac
```
