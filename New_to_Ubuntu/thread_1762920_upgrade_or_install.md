### Post by pi.boy.travis on 2011-05-19
I have machines that I've upgraded since 7.10, and machines that I reimage every release. I've never had an upgrade fail, but others aren't so lucky. I'd backup and try the upgrade. If it works, great! If not, just do a reinstall. It's helpful to keep /home on a separate partition, then when you reinstall all you have to do is install all your packages. APTonCD is good for this.
