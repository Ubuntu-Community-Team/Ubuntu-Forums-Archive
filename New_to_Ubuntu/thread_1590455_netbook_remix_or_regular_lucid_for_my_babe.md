### Post by lbrty on 2010-10-07
To my understanding UNE and Ubuntu Desktop are the same with the exception of the GUI interface. As *andrewthomas* stated you can always install that interface at a later date. You can also install kubuntu-desktop (just an example) if you wanted to do that also without having to do a clean install (through Synaptic Package Manager). Ubuntu is flexible.

For example, if you wanted to you could have the UNE, GNOME (Ubuntu Desktop), and KDE (Kubuntu). Just install through the Synaptic Package Manager (as stated above), and then select your interface at the login screen.

### Post by lbrty on 2010-10-07
According to Wikipedia.org, the UNE has an interface of *GNOME + Ubuntu Netbook Edition*
and Ubuntu Desktop is just GNOME. Either way you can change to each one at the login screen. You must first download the packages via Synaptic Package Manager.
