To the OP:

This will help you in future I believe.

When ever you want to install something, terminal is not the only solution as rolnics mentioned is his post. Go to Applications >>> Software Center, type whatever you want to install in the search box, choose the appropriate software and click install. Type the password and viola.... Installation will never be easier than this on any OS.

Alternatively there is Synaptic Package Manager under System >>> Administration menu. Search for the package, right click and select Install this package. But you have to be a little more accurate in it because it will search for the exact package name.

And at last I recommend to newbies that apt-get is used for installation.
