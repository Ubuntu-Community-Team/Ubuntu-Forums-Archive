### Post by Frogs Hair on 2011-07-12
Hi and Welcome

Some Gnome applications will run in KDE and vice versa . The applications will have a Gnome style . You should check the applications you installed and replace them with a KDE equivalent if possible .

Make sure that you did not install the entire gnome desktop. If you install enough of its components the desktop will start looking like Gnome.
