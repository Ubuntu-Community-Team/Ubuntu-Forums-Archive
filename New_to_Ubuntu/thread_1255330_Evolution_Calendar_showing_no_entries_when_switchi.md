### Post by mprince on 2009-09-01
[QUOTE=Matt_K]or is there a way to set Work Week as the default week view?[/QUOTE]
I don't think there is a way to set the Work Week as the default week view.
However, if you highlight a range of days on the calendar with the pointer  it shows the days you've highlighted in a Work Week style view... Select at least 2 consecutive days or you just get the Day view.

If you don't need to know the week numbers you can opt to not show those at all under preferences.
