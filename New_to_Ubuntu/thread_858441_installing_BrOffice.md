### Post by kellemes on 2008-07-13
It's in the universe repository.. [see here]("http://packages.ubuntu.com/nl/hardy/editors/broffice.org").

From a terminal window..
```
sudo apt-get install broffice.org
```
or open synaptic (your graphical package manager) and search for and install broffice.org
