### Post by Jon Monreal on 2010-05-05
Hello,

If you want to try to find out what the failure could be,
```
sudo gedit /etc/cups/cupsd.conf
```
and change "LogLevel warn" to "LogLevel debug".

Restart the cups daemon (or your computer) and then try printing again. Then you should be able to 
```
sudo gedit /var/log/cups/error_log
```
and find the error.

If you want, feel free to post it here. Not sure how much I can help you though.
