### Post by Golem XIV on 2008-05-08
LFS is not so much of a book as it is ia project.  It guides you through building your own Linux system from zero: setting up the computer and disks, obtaining the tools and source code, compiling the kernel, adding libraries and apps, etc.

It takes a fairly long time, but at the end you end up with a Linux system built from the ground up, so you learn a lot about Linux and its inner workings.

I've been meaning to take the plunge but so far I haven't had the required time.  If you have the patience, by all means go for it.
