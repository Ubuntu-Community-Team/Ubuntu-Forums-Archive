### Post by cariboo on 2008-07-13
Sbackup will not make an exact image of your installation. It is made more for making one full backup, then incremental backups, until you tell it to make a full backup again.

The program you are looking for is called **partimage** this will make an exact image of your hardrive as it is now. It uses a ncurses based gui so it isn't to hard to use.

Partimage can bee installed using Add/Remove Programs or Synaptic Package Manager
