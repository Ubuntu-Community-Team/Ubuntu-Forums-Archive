### Post by claracc on 2014-06-11
There is an audio-recorder software package in repository ppa:osmoma/audio-recorder for 14.04:
[https://launchpad.net/~osmoma/+archive/audio-recorder?field.series_filter=trusty](https://launchpad.net/~osmoma/+archive/audio-recorder?field.series_filter=trusty)

In the link provided, there are instructions about howto add this repository to your software sources and install audio-recorder.

Also, there is a useful link here: [http://www.ubuntugeek.com/how-to-install-audio-recorder-on-ubuntu-13-0412-10.html](http://www.ubuntugeek.com/how-to-install-audio-recorder-on-ubuntu-13-0412-10.html), although it is for 13.04 or 12.10, the method is the same, except the ppa added has to be for 14.04.
