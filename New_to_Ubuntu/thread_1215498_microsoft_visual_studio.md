### Post by HereInOz on 2009-07-17
Just to clarify, Windows programs are technically not able to run on Ubuntu, nor are Linux programs technically able to run on Windows. Mac programs are similarly for Mac only.

However the people at the Wine project have created a program (Wine) which runs on your Ubuntu operating system, and Wine will allow some Windows programs to run on Ubuntu.

From the response of the previous poster, Visual Studio is not one of the Windows programs which runs on Ubuntu using Wine.

You are probably going to have to stick with a Windows operating system if you need to run Visual Studio.

### Post by Mornedhel on 2009-07-17
If you use your computer mostly for work in Visual Studio, then there's probably little point in migrating to Linux.

If you use your computer for general programmation, then you should look into what already exists in Linux instead of trying to get Visual Studio to work under Linux.

If you use your computer for general purposes but sometimes do some work under Visual Studio, it depends on whether you need Windows libraries (then stay on Windows or run a Windows VM) or not (then by all means migrate to Linux, and you probably can use any IDE).

There are stickies on the topic of programmation under Linux.
