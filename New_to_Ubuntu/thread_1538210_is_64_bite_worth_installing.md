### Post by howefield on 2010-07-24
Memory is a poor reason (by itself) to choose whether to run 32 or 64 bit Ubuntu. 32 bit can run shed loads of RAM by using a PAE kernel, and 64 bit has advantages that could be felt even with a couple of gig of RAM.

### Post by QIII on 2010-07-24
Amount of RAM need not enter the discussion.  Although you can address an amount of RAM several orders of magnitude greater than you can with a 32 bit system, that should not be the consideration.

If you have a 64 bit processor you should install a 64 bit OS, regardless of your RAM.

Are you old enough to remember those poor souls who stuck with 16 bit OSes when 32 bit became the norm?  Not only were they not using the full capability of the CPUs they had paid good money for, but they were soon left behind in other ways.

Use a 64 bit OS if you have a 64 bit CPU.

The FUD about problems with 64 bit OSes is very, very old news.
