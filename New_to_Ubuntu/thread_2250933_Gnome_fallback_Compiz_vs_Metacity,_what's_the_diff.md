### Post by deadflowr on 2014-10-31
Metacity runs on 2d, compiz runs on 3d.
Metacity can only do so much it terms of added effects.
(It's extra effects is very limited)
Compiz can enable a whole range of extra effects.

By default they both are setup to be run with the basic configurations.
But compiz, via tools like compizconfig-settings-manager, can be tweaked to allow effects such as desktop cube, wobbly windows, exploding windows on close/minimize, and a whole range of other effects.
None of which can be set on a metacity desktop.

Even though those added effects may seem to be nothing more than eye candy, if set right they can actually be quite useful to how you might use your system.
But those added effects run at a cost of system resources, where as metacity's system resources can be quite smaller in comparison. Since metacity does not require/have the ability to run the heavier capabilities of 3d effects.

Hope that helps a little.
