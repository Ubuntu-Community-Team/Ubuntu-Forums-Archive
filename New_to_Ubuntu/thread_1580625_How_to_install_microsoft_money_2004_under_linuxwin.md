### Post by bodhi.zazen on 2010-09-23
FYI wine, IMO, can be tricky, often the version of wine is critical, and some things are fixed and others break between versions.

With something as "mission critical" as finances I highly advise either:

1. Continue to dual boot (or run windows in a VM) and continue to use MS money.

2. Convert to a Linux native alternate. gnucash is cross platform. kmymoney is, IMO, closest to MS money.

3. There are other commercial cross platform options, I have not reviewed those.
