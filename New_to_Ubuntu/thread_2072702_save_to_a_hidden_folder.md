### Post by Frogs Hair on 2012-10-18
I was able to do this in 12.10 but it took a few minutes to figure out using Firefox.
 
1. Set Firefox to ask every time for a download location in preferences and restart.
2. when the save dialog opens select OK.
3. When the nautilus navigation opens select the home icon in the left pane and your hidden folder will be displayed.
4. Select folder and save.

I did not need to enable hidden folders to do this.
