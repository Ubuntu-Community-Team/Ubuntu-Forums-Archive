### Post by newb85 on 2012-10-17
Window placement is handled by Compiz.  Open Compiz Config Settings Manager (not installed in Ubuntu by default, but I believe you have already installed it).  Under "Window Management", there's a feature "Place Windows" that you need to enable.  I recommend the Smart placement mode, but you can experiment with others to see what you like.
