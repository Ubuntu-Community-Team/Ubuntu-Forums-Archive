### Post by Amida on 2010-08-28
Dear Ubuntu users,

I have installed and for some time using Lucid Linx with great pleasure. I am spending time searching for the best application in various fields to make my Ubuntu more complete and functional. Here is a list of these that are not present in the default Ubuntu installation that I found as most contemporary and useful at this time. Please comment and add a possible replacements with better functionality or other advantages.

**Accesories:**
	anyremote - use your mobile phone as a Bluetooth remote control for your PC
DockBarX - Windows 7 like application dock  
**Graphics:**
	Shotwell photo manager
	YAGF - effective OCR tool (also works excellent with Cyrillic)
	ImageMagick - a lot of functional features
**Internet:**
	Filezilla - FTP client
	qBitTorrent - (no firewall problems: work out of the box with wireless router connection)
	Opera web browser
	Thunderbird Mail Client
**Office:**
	PDF Mod - add and remove pages from a pdf documents
**Science:**
	BKchem - draw chemical structures
	Avogadro - 3D molecular design
**Sound & Video:**
	SMPlayer
**System tools:**
	Gmount-iso
	System profiler and benchmark - nice analogue of Everest for Linux
	Ubuntu tweak
	Emerald theme manager

I will be glad to discover here more and better variants of these OpenSource applications!
