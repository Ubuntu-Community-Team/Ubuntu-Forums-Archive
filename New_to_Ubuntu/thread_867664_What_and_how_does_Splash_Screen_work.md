### Post by iaculallad on 2008-07-23
If what you're referring is the Usplash screen w/c shows on Ubuntu startup then try reading Ubuntu Community's [Usplash page]("https://help.ubuntu.com/community/USplash").

You can install the Startup Manager if you want to customize the Usplash screen.

```
sudo apt-get install startupmanager
```
