### Post by robertchahine on 2008-06-17
did you try emerald?

```
sudo apt-get install emerald 
```

to install the software.Then you can find specific themes for emerald

and to install these theme

you open them with emerald and then write in a terminal
```
 emerald --replace 
```

you can get some emerald themes from many webs like [www.gnome-look.org](www.gnome-look.org)
