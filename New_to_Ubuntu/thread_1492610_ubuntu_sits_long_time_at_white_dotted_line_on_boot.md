### Post by SRST Technologies on 2010-05-24
That dotted line is annoying to see for so long.  It's doing that because it's loading up the stuff needed to start up the graphical splash screen.  This is pretty new to Ubuntu and not many people are happy about it.  There is a way to fix it, but it is pretty much dependent on your video card.  If you google Lucid Blank Screen Fix you should find tons of answers.  Find one that matches your hardware and use it.

You can, however, render your entire installation unusable if you mess up.

In the end, it's not that big a deal to see that little blinking dot for a while.  You don't reboot Ubuntu much, it's designed to be turned on and left on forever.
