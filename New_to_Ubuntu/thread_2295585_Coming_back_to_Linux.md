### Post by mystics on 2015-09-19
There is a free book that goes over the Linux command line. You can download it here: [http://linuxcommand.org/tlcl.php](http://linuxcommand.org/tlcl.php)

For a good text editor, I like using Atom, the steps for which to install are here: [http://www.webupd8.org/2014/05/install-atom-text-editor-in-ubuntu-via-ppa.html](http://www.webupd8.org/2014/05/install-atom-text-editor-in-ubuntu-via-ppa.html)

After Atom, Visual Studio Code and Sublime Text, which heavily inspires Atom, are probably the two best. I personally would choose all three over Gedit, though some may not like Sublime Text due to it not being open source or Visual Studio Code due to it being Microsoft. Beyond that, there is Gedit and Mousepad, but in the case of Mousepad, I would personally advise just sticking with nano or learning how to use Emacs or Vim.
