### Post by aphatak on 2011-05-18
You could try xscan (you can install it from the Synaptic Manager if you haven't already).  This program looks for scanners connected to the PC, and if it finds one, will control it for you.  I have been using it with an Epson V300 scanner; this also needs drivers from Epson before the scanner is recognized.  However, once the drivers are in place, xscan has had no problems with it.
