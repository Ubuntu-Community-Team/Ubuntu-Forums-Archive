### Post by tamoneya on 2008-05-05
i dont know what you mean by itunes player but yes you can install gtkpod to interface with an iPod and VLC can play anything you throw at it.  As for adobe flash there is flashplugin-nonfree which can be installed ```
sudo apt-get install flashplugin-nonfree
```This will get flash working in firefox and whatever other web browsers you have.  Flash support is also only going to get better since adobe recently opend up the flash specification.

EDIT: simple answer is YES
