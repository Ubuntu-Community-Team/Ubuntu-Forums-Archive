### Post by wallaroo on 2010-02-26
You can make a desktop shortcut to your Tomboy project note so that it pops up when you need it, and close it when you don't (it still means that Tomboy is always running in the background though, but this is the case with all the notes programs).

For example if you had a tomboy note called 'Project Notes',
Right click on the Desktop and select 'Create Launcher'.

- Select Type as 'Application'
- Put in a meaningful name (e.g 'Working Notes')
- In the command box put 
    tomboy --open-note 'Project Notes'

Then press OK.

You should now have an icon on your desktop to quickly bring up your project notes.
(The first time you click will take a little longer as the tomboy program gets loaded but after that the note will pop-up instantly each time).

### Post by DedMoroz on 2010-03-02
Im using two apps right now. 

-Remember The Milk

and 

-Tomboy

Remember The Milk is a task manager, you can even get it on an iPhone or Blackberry if need be (costs extra). Very handy. The basic website is free, and I use a program you can install called "Tasque" to manage my Remember The Milk tasks (you would need to register for a free RTM account). Works nice and it resides in my system tray at the top. The Tasque app is in the Ubuntu Software Center.

Tomboy is more for note taking allowing you to write more data besides just a task and a deadline. Tomboy is quite nice IMO. No need to put it on the panel. Both of these programs, Tasque and Tomboy I have boot up at startup and they reside up on the system tray. If I need to make a new note or a new tasque, they are right there ready for me. I dont know how much simpler you want it!

To add Tomboy to one of your start up apps, go to:

System > Preferences > Startup Applications

Click "add". The name can be "tomboy" and the command will also be "tomboy".

### Post by Thsyrus on 2010-03-30
I think it depends on what you are looking for. Tomboy has its uses but from an academic note taking point of view it is severely lacking.

Basket has a lot more going for it but still not quite there. Also as a kde application it can look a bit craptacular on a gnome desktop.

Zim seems good but I don't know much about it.



If you are looking for a OneNote replacement (a lot of people seem to be) sadly there isn't a clear solution. Most linux note taking software is generally aimed at the "lighter" side of note taking so basket might be your best bet.

If you are simply wanting to remember to call someone then any of the above choices are fine depending on personal preference.
