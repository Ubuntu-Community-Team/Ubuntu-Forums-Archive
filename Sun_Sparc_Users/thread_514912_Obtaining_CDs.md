### Post by foxholeunder on 2007-08-08
I was looking several weeks ago for ubuntu desktop versions for sparc processors and could not find any either...

However, I did a search for xubuntu sparc desktop and lo and behold their are xubuntu versions for the sparc processor! I found 7.04 version of xubuntu. for sparc processors under alternate installs

Here is the link:
[http://cdimage.ubuntu.com/xubuntu/ports/releases/7.04/release/]("http://cdimage.ubuntu.com/xubuntu/ports/releases/7.04/release/")

NOTE:

During installation when asked to choose your keyboard it will auto highlight the sun type4 .. the other option for sun keyboards is type5

I have both a type 5c and a type6. I tried both sun type5 and type4 with both my type5c and type6 sun keyboards and immediately after choosing them as my keyboard the installation was a  wash as apparently their is no support for English sun keyboards. My keys were typing some different language and made it impossible to finish the installation.

I then retried and chose a general american 105 key keyboard and installation went off without a hitch.

So if you require english you will need to ditch the sun type selctions and go for the general 105 key.

Come to think of it, I am wondering if I have filed that as a bug yet.... I remember starting out to file it as a bug ... anyways.. Except for that one little glitch everything installed and runs as smooth as butter!

The Xfce desktop runs almost as fast on my 128 meg memory 333mhz Sparc Ultra 10 as Gnome does on my 1gig memory 3ghz i686 .. Very nice!

I could not find any other versions for desktop installs for sparc processors other than 7.04 when I did my initial search several weeks ago.

Enjoy your Xubuntu Sparc Desktop!
