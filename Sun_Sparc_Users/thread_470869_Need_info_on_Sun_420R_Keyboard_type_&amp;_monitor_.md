### Post by toastkid on 2007-06-16
I think you paid a fair price. I have 3 of these machines, but you don't really see many of them. The E420 is really a rack mount Ultra 80, and they are just as rare.

I usually run these as headless servers with a serial console, but I can't see any real issues. The graphics card supports a number of VESA compatible modes so the monitor should not get too confused. You can usually pick up a 13W3 monitor pretty cheaply - mine came free with an Ultra 2 :D.

Let me know how you get on - I'll offer any help I can.
