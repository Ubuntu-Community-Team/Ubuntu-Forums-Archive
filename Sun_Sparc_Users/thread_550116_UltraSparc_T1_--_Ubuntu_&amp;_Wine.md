Wine (as of now) runs only on x86:

[http://www.winehq.org/site/myths#only_x86](http://www.winehq.org/site/myths#only_x86)


And since "**W**ine **I**s **N**ot an **E**mulator" it won't emulate a x86 CPU on a Sparc machine. Therefore, you'd need a SPARC version of the Windows applications you intend to run with it. There is no Windows version for SPARC, an very probably will never be - so there'll never be any Windows apps for SPARC.
