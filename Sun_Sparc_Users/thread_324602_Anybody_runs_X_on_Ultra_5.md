### Post by Moulton on 2007-02-02
I run Xubuntu-Desktop on a dual-processor 64-bit Ultra II.  I had to procure a 24-bit UPA Creator graphic adaptor because the X.Org drivers don't work with any of the S-Bus graphic adaptors (e.g. CGsix).  The X.Org drivers install and load, but they fail to actually drive the S-Bus devices properly.  These S-Bus video drivers work fine on the 32-bit Debian Sparc system.

If you don't have a supported graphic adaptor, you can still install X11 and connect via XDMCP from a remote X-Server over the LAN.  You just can't run an X-Server on the console.  If you do connect over the LAN, be sure to disable any animated screensavers as the animations will flood your LAN.

If you have a Sun model that supports the ATI graphic adaptors, you should be in good shape.  Just don't expect to use S-Bus adaptors in X.Org on the 64-bit platform.

I also had trouble with Firefox and Epiphany locking up on the 64-bit Sparc.  I found that Opera Web Server works OK.
