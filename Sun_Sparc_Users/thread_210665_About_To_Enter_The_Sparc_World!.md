### Post by hw-tph on 2006-07-16

As far as I know there is only one [SPARC image](http://releases.ubuntu.com/6.06/ubuntu-6.06-server-sparc.iso). There is quite a lot of information on Linux SPARC support. The [parent project](http://www.debian.org) has a [page about the SPARC support](http://www.debian.org/ports/sparc/) and the same applies for Ubuntu I believe: Userland is 32bit but you can run a 64bit kernel on it which translates between 32bit and 64bit internally.
