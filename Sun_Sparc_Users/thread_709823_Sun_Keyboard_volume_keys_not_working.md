Check GNOME's keyboard shortcut settings. I know that it used to work, but maybe someone forgot to maintain or include these definitions.

In GNOME, go to System -> Preferences -> Keyboard shortcuts, pick the category "Sound", choose the function you want to map to a key to and press the desired key on the keyboard. At least, that's what I used to make a few additional things work.
