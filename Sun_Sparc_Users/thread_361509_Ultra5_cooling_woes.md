### Post by netztier on 2007-02-14
> **mahy said:**
> If i take the fan out and install some passive cooling, will it suffice? Ideas, anyone? TIA

I'd leave the fan in, maybe replace it with some known-to-be-silent 80mm fan and run that one on +7V from a standard CD-ROM/Harddisk power cable instead of the +12V from the mainboard. 

If you pick not too a loud harddisk in addition to that, you can leave it running while sleeping withing hearing distance, as I did many a "emerge night" back in my gentoo days.

see also this [thread]("http://www.ubuntuforums.org/showpost.php?p=1962284&postcount=6").
