I just installed the "ieview" extension to launch IE under Wine (when I need to view a web page that just won't work under Firefox), the standard extension is for Windows, but there are simple instructions on how to make it work in Linux:

[http://ieview.mozdev.org/](http://ieview.mozdev.org/)

Of course you need **wine** and IE installed (**winetools** can make this second part relatively easy), but it does make web browsing just that bit less annoying.......
