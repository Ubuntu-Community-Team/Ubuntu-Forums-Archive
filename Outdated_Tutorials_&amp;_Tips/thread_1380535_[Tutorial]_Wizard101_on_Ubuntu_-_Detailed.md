**[size="3"][COLOR="Black"]The Tutorial[/COLOR][/size]**

Welcome to my first tutorial on Ubuntu forums. In this wonderful tutorial (lol) I will be showing you how to run and install Wizard 101 perfectly, with no problems. Wizard 101 is a Windows based game, not a Linux or Macintosh game. Why? Well programming on a Windows operating system is easier than Linux and/or Mac. Also, the majority of the world is Windows based. However, Linux has a secret weapon called [[COLOR="DimGray"]_Wine_[/COLOR]]("www.winehq.org/")! [[COLOR="DimGray"]_Wine_[/COLOR]]("www.winehq.org/") basically allows you to enjoy Windows applications/games on your Linux operating system, perfectly (sometimes). Lets get started on installing Wine and setting up Wizard101!

> [COLOR="DarkRed"]**Note:**[/COLOR] You will be asked to enter your "root" password, in order to install Wine.



**[size="2"][COLOR="Black"]Installing Wine[/COLOR][/size]**


1. Installing Wine - Install Wine first.

> sudo apt-get install wine wine-utils


2. Configuring Wine - Do this after installing Wine.

> winecfg

[CENTER][IMG]http://i46.tinypic.com/25u416r.png[/IMG]

[SIZE="1"][COLOR="DeepSkyBlue"]Make your configuration of Wine like this.[/COLOR][/SIZE][/CENTER]



**[size="2"][COLOR="Black"]Installing Wizard101[/COLOR][/size]**


1. Download Wizard101 - Need this in order to play & install.

> ...

[SIZE="1"][CENTER][COLOR="DeepSkyBlue"]Use MediaFire if Wizard101.com cuts you off.[/COLOR][/CENTER][/SIZE]


2. Open your newly downloaded .zip file and extract the .exe.


3. Right click on the InstallWizard101.exe and select "Open with Wine Windows Program Loader."


4. Read & follow the instructions when you open the InstallWizard101.exe. Set your settings like the screenshot below. (Hardware Cursor should be unchecked)

[CENTER][IMG]http://i47.tinypic.com/4vsgth.png[/IMG]

[COLOR="DeepSkyBlue"][SIZE="1"]Make sure your settings are somewhat like this.[/SIZE][/COLOR][/CENTER]


5. Wait for patching to finish and finally enter your username and password, then click "Play." Now you're done :D.
