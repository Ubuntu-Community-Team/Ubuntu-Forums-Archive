Greetings, all.

First, a bit of background on me. I run a low-end PC (well, I think it's low-end -- specs at bottom) and I'm always on the lookout for speed optimizations for Xubuntu. At the same time, I try to use software already present that I do not need to install.

Just today I discovered that thumbnails, in combination with gedit (or any editor), can actually be useful outside of the common graphic use.

What I did was make a text file on the desktop (though I don't know which environment it is, since when I right-click, I get all these "Create ..." commands, not the Xfce menu), and name it "ToDo".

Then I double-click to fire up the editor, edit it, save and voila! The thumbnail displays the contents of the text file. To see the whole thing, right-click on the icon, then Stretch Icon and drag the bottom-right corner. :) I can imagine other uses of this, but for now, it's perfect for a ToDo list. What other uses can you think of?
