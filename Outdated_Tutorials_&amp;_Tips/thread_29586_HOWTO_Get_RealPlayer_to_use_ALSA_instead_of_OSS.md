I'm sure most people are aware of the problem of RealPlayer not starting up or only working if no other programs are doing sounds. This is because by default, Real only uses OSS to output audio and no obvious way to change that. Well, here's something I found in the Warty Desktop Support shows you how to change it to ALSA.

[http://ubuntuforums.org/showpost.php?p=80282&postcount=21](http://ubuntuforums.org/showpost.php?p=80282&postcount=21)

I followed it, and now I can get Real to work, with ALSA sound mixing so it'll run with other programs as well!  :D
