### Post by blkno1 on 2006-07-03
Well from reading the posts, I guess K3b can burn .nrg files to disk.  But if thats not an option heres what I did.

```
~$ apt-get install nrg2iso
```

After install run

```
~$ nrg2iso filename.nrg filename.iso
```


All set!  \\:D/
