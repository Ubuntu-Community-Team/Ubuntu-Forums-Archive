### Post by Some_Person on 2007-06-27
This will make KDE apps look better in Ubuntu.

1. Download and extract the attached file.
2. Install "kcontrol" and all its dependencies from the Repositories.
3. Hit Alt+F2 and type in "kcontrol" (without the quotes). Press "Run".
4. Go to "Applications and Themes".
5. Go to "Theme Manager".
6. Select the "Plastik" theme.
7. Click "Apply".
8. Go to "Colors".
9. Click "Import Scheme".
10. Find the attached file you downloaded. Click "Open".
11. Select the "Ubuntu Human" Color Scheme.
12. Click "Apply".
13. Go to "Fonts".
14. Make sure all the fonts, except "Fixed Width", are "Sans Serif 10". Make sure "Fixed Width" is "Monospace 10". Make sure Window Title is bold. If it isn't, fix it.
15. Check the box "Use anti-aliasing" and click "Configure".
16. Check the box "Use sub-pixel hinting".
17. Click "Apply".
18. Go to "Icons".
19. Select the "Human" icon theme.
20. Go to "Peripherals".
21. Go to "Mouse".
22. Go to "Cursor Theme"
23. Select the "Human" Cursor Theme.
24. Click "Apply".
25. Click "OK".
26. Reboot.

Enjoy!
