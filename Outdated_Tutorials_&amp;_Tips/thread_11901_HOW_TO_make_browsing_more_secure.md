1. Go to page [http://www.mvps.org/winhelp2002/hosts.htm](http://www.mvps.org/winhelp2002/hosts.htm) and download hosts.zip
2. Unpack hosts.zip
3. Open your /etc/hosts file and paste there what HOSTS contains

After reboot your network browsing will be more secure, because those entries in /etc/hosts file will be blocked  :D

Remember to update your /etc/hosts file regularly :)
