 [LEFT][FONT=Times New Roman][SIZE=3]**2/ What to Download?**
[/SIZE][/FONT][/LEFT]
 [FONT=Times New Roman][SIZE=3]Let's deal with a couple of points of possible confusion before we go on as there are various releases and versions of Ubuntu available. [/SIZE][/FONT] [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT] [FONT=Times New Roman][SIZE=3]Firstly; [/SIZE][/FONT] [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]* The 'Desktop' version simply means a graphical installer, NOT exclusively for desktop computers; it's suitable for notebooks/netbooks, too.[/SIZE][/FONT][FONT=Times New Roman][SIZE=3] 
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]* The 'Alternate' version is a text based installer, also suitable for both. Try this if you're having trouble installing the Desktop version of whatever you're installing.[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]
* If you are intending to install on a netbook you should go to the [/SIZE][/FONT][FONT=Times New Roman][SIZE=3][Ubuntu Netbook Remix]("http://www.ubuntu.com/getubuntu/download-netbook") (UNR) page. It is designed to run on these machines and would be the one to try. This HOWTO does not deal with UNR (see post #2 for more detail, thanks RH).[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]  
[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]Secondly; [/SIZE][/FONT] [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]* i386 version: For single-core, 32bit processors, Intel OR AMD. Most older machines.[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]* AMD64 version: For dual-core, 64bit processors, Intel OR AMD. [/SIZE][/FONT] [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]Thirdly;[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]* Ubuntu can be run with various 'desktop environments' (DE) or 'desktop managers'. [/SIZE][/FONT] [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]* Three of these distinguish the main members of the Ubuntu family. [/SIZE][/FONT][FONT=Times New Roman][SIZE=3]Ubuntu with three different DEs. [/SIZE][/FONT][FONT=Times New Roman][SIZE=3]Here they are in order of resource usage, most demanding to least:[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]Kubuntu: Kubuntu uses the KDE Desktop Environment; make sure you have plenty of RAM because it has all the bells and whistles[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]Ubuntu: Uses the Gnome DE; not lightweight nor too bloated.[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]Xubuntu: Uses Xfce, a lightweight DE; perfect for low spec/older machines [/SIZE][/FONT] [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]This is general. ALL of these desktop environments and installs can be tweaked to your liking, to run faster or bloated, as you become more familiar with the way things work.[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]NOTE: If you are running a very low spec machine (old CPU, not much RAM) and can't get Xubuntu to run (alternate install, NOT desktop) there are many alternatives. You might like to start with Damn Small Linux or Crunchbang. Look for some at the DistroWatch link given later in the HOWTO. There are many ways to create your own hybrids but for now ...[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]* About Releases: [/SIZE][/FONT] [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]Ubuntu is always moving forward and improving. For this reason, a Ubuntu release comes out every six months. The release dates at the bottom of [/SIZE][/FONT][FONT=Times New Roman][SIZE=3][This Link]("http://www.ubuntu.com/products/whatIsubuntu/releases")[/SIZE][/FONT][FONT=Times New Roman][SIZE=3] are old but give you an idea of the release schedule. [/SIZE][/FONT][FONT=Times New Roman][SIZE=3]Releases are available before the release date and are considered 'testing': [/SIZE][/FONT] [FONT=Times New Roman][SIZE=3]

[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]* Karmic Koala is currently testing.[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]* The current stable Enterprise version is Hardy Heron, 8.04 LTS (Long Term Support).[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]* The current version is Jaunty Jackalope, 9.04 [/SIZE][/FONT] [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]8.04 LTS is a safe place to start, and if you are running production machines a good idea, as it has long term support and is very stable. 9.04 is a little up and down for the moment.

*NOTE: If wanting to run Edubuntu packages see note at end of post. Easier to install on 9.04[/SIZE][/FONT][FONT=Times New Roman][SIZE=3].

**4/ How to Install**
[/SIZE][/FONT]    [FONT=Times New Roman][SIZE=3]You've burned the disk? If it hasn't already ejected, eject it and:[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]* Label the disk, not forgetting to write 'i386' or 'AMD64' along with the release/version and date. This avoids confusion. (Especially if you happen to own a 64 and a 32bit machine!)[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]   [FONT=Times New Roman][SIZE=3]*  Boot your computer and press the appropriate key to get into the BIOS. Probably F1 or DEL. The boot screen when the machine first boots[/SIZE][/FONT][FONT=Times New Roman][SIZE=3] should tell you if unsure
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]*  Once in the BIOS change your boot order to boot from CD[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]*  Open the tray and insert the Ubuntu CD[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]*  Hit 'F10' to 'save changes and exit' [/SIZE][/FONT] [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]... and you're rolling. The computer should restart and boot from CD. (DON'T forget to change the boot order back when you're finished). [/SIZE][/FONT] [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]If you are using the Desktop version you will be greeted with a friendly GUI.[/SIZE][/FONT] [FONT=Times New Roman][SIZE=3]

[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]Safety First: Choose the option 'Check Disk for Defects!'[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT] [FONT=Times New Roman][SIZE=3]Thankyou. It takes a little while. You can then 'Try from CD without making any changes to your system' and have a look around, double check your hardware is compatible (if not things can usually be tweaked), or dive right in and install. [/SIZE][/FONT] [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT] [FONT=Times New Roman][SIZE=3]If you are looking at a GUI it is a promising start for hardware compatibility. Here is a [/SIZE][/FONT][FONT=Times New Roman][SIZE=3][guide to installing]("https://help.ubuntu.com/community/GraphicalInstall")[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]. 

[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]If you have Windows installed and would like to keep it and dual-boot with Ubuntu[/SIZE][/FONT][FONT=Times New Roman][SIZE=3] here is one of the most useful resources around for [/SIZE][/FONT][FONT=Times New Roman][SIZE=3][Windows/Ubuntu dual-booting]("http://members.iinet.net/%7Eherman546/index.html")[/SIZE][/FONT][FONT=Times New Roman][SIZE=3] help, info and install instructions. [/SIZE][/FONT] [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]   [LEFT][FONT=Times New Roman][SIZE=3][URL="http://members.iinet.net/%7Eherman546/index.html"]
[/URL][/SIZE][/FONT][/LEFT]
  [LEFT][FONT=Times New Roman][SIZE=3]* If you are having trouble installing, especially on a lower spec machine, try the ALTERNATE install rather than the Desktop version or try Xubuntu or a more lightweight Linux distribution.
* Lastly, once installed you'll be asked to remove the disk and reboot. This is a good time to reboot to the BIOS and change the boot order back to hard disk. 
* Hit F10 to save changes and the machine should boot from the hard drive
* If you are dual-booting you should see a menu with your operating systems
* If you only have Ubuntu installed you should see a menu with several Ubuntu kernels

[B]5/ I've Installed: Now What?
[/B] [/SIZE][/FONT]    [FONT=Times New Roman][SIZE=3]* Make sure you have an ethernet cable plugged in (even if you are going to be using wireless) and boot in to your new Ubuntu install. The first one on the list and not the 'recovery' kernel; use that if you are having problems booting into your regular kernel.[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]* After not very long you should be notified that you have updates to download.[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]* Click on the icon and get them.[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]* If you are running a video card or a wireless card you may be notified there are 'restricted' drivers available. If you are happy to go the non-free route, accept and follow the instructions. Because they are non-free they are licensed and you need to personally agree to use them (which is why they are not part of the Ubuntu install).[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]

If you are familiar with Windows [HERE]("http://wiki.linuxquestions.org/wiki/Linux_software_equivalent_to_Windows_software") is a fairly comprehensive list of Ubuntu software which approximates Windows software.[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT] [FONT=Times New Roman][SIZE=3]To try out some of the free software from the repositories in your software sources:

* To Add/Remove applications: [/SIZE][/FONT] [LEFT][FONT=Times New Roman][SIZE=3]Applications->Add/Remove: or;[/SIZE][/FONT][/LEFT]
 [LEFT][FONT=Times New Roman][SIZE=3]System->Administration->Synaptic Package Manager[/SIZE][/FONT][/LEFT]
 [FONT=Times New Roman][SIZE=3]
* Getting Help[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [LEFT][FONT=Times New Roman][SIZE=3]If you have questions the forums are the place to look. You can start at 'Absolute Beginner's Talk' (no question too dumb ... well, almost no question!) or post to the right forum for a specific problem area. But before you do that, search the forums or open a search engine and search for your problem as specifically as possible adding 'ubuntu'. For instance, if your Acer monitor were problematic you might try:[/SIZE][/FONT][/LEFT]
 [LEFT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT] [/LEFT]
 [FONT=Times New Roman][SIZE=3]> acer x193hq monitor flickering ubuntu 8.04[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT] [FONT=Times New Roman][SIZE=3]Formula:[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT]  [FONT=Times New Roman][SIZE=3]> (your_problem) ubuntu (your_ubuntu_release_by_number_or_name)[/SIZE][/FONT][FONT=Times New Roman][SIZE=3]

 [LEFT][FONT=Times New Roman][SIZE=3]Few simple rules of thumb when posting that will help you get helped:[/SIZE][/FONT][/LEFT]
 [LEFT][FONT=Times New Roman][SIZE=3]
[/SIZE][/FONT] [/LEFT]
 [LEFT][FONT=Times New Roman][SIZE=3]* Use a descriptive title for your posts, not just 'HELP!' [/SIZE][/FONT] [/LEFT]
