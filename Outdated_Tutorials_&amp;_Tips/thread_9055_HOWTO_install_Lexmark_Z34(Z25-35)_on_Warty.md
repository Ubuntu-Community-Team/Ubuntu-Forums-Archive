### Post by ghostrifle on 2004-12-23
1) Get the drivers from lexmark.com

2) Extract CJLZ35LE-CUPS-2.0-1.TAR.GZ

3) sudo sh lexmarkXXXXXXgz.sh -keep

4) go to 'installer' directory

5) convert both rpms you're finding there with 'alien rpmfile'

6) sudo dpkg --install bothrpms

7) restart, configure your printer with the gnome tools, ready !
