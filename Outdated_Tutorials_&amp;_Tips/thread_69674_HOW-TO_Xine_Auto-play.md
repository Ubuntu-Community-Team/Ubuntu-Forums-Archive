### Post by DoeRayMe on 2005-09-27
To allow xine to auto-play cd/dvd when inserted, do this

system > settings > Removable Drivers And Media *Multimedia Tab*

for cd auto-play

> xine cdda:/1

for dvd

> xine dvd://

sorry if been posted before
