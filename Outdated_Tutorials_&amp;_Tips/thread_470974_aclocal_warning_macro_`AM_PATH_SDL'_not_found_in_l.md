Hi I tryed to install a program called Arianne and needed to use the following command  for the config and got the following arror.

```

warning: macro `AM_PATH_SDL' not found in library

``` Before this I searched the forum and did not find a fix thought a post recommended installing some files which did not fix this.

 This error was solved as found in a google search which had very few results and I found this navigating thought a page at a time forum.

 Installing the following program from synaptic solved this error.

 ```

libsdl1.2-dev

```
