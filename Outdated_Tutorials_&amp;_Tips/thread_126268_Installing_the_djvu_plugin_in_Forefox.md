### Post by Thulemanden on 2006-02-06
The djvu is a competitor to the Adobe PDF format also offering some free tools under Linux. The claim is among others that the files are smaller. At the end you will find some links sources to document files in the djvu-format. To view them you need the plugin for Firefox or the stand alone djview (djvu viewer) which also installs its presence in the KDE menu (I have upgraded my Ubuntu to kUbuntu)

Either download files after activating the Uiverse and apt-get the files
[djview]("http://packages.ubuntu.com/hoary/universe/graphics/djview"), [djvulibre-bin]("http://packages.ubuntu.com/hoary/universe/graphics/djvulibre-bin"), [djvulibre-plugin]("http://packages.ubuntu.com/hoary/universe/web/djvulibre-plugin"), [djvuserve]("http://packages.ubuntu.com/hoary/universe/web/djvuserve"), [libdjvulibre1]("http://packages.ubuntu.com/hoary/universe/libs/libdjvulibre1")
 or download from 
[http://packages.ubuntu.com/cgi-bin/search_packages.pl?searchon=sourcenames&version=all&exact=1&keywords=djvulibre]("http://packages.ubuntu.com/cgi-bin/search_packages.pl?searchon=sourcenames&version=all&exact=1&keywords=djvulibre")

The plugin can be dragged from /usr/lib/mozilla-firefox/plugins using konqueror to the user Firefox plugin dir and inserted as a symbolic link.

Here you can scout for sites offering images in djvu-format:
[http://www.lizardtech.com/products/doc/publicsites.php]("http://www.lizardtech.com/products/doc/publicsites.php")
 [http://www.djvuzone.org/links/]("http://www.djvuzone.org/links/")

Go check it out :-)
