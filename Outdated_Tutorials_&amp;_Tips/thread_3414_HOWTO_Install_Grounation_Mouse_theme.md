[http://www.gnome-look.org/content/show.php?content=14484](http://www.gnome-look.org/content/show.php?content=14484)

If you like the Fedora type of mouse theme, download the GNOME Grounation mouse theme.

Extract the files.

Go to your Home Folder.

View -> Show Hidden Files (or Control-H)

cut /paste Grounation into .icons

rename Grounation to default (not Default, not defaults, 'default').

exit the app.

presto.
