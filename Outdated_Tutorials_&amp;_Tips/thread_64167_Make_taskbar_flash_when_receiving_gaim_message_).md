### Post by topcop on 2005-09-10
grab the three debs here [http://www3.150m.com/ubuntu-warty/](http://www3.150m.com/ubuntu-warty/)  download them to a directory.
 
sudo dpkg -i *.deb

in gaim, enable message notification plugin from Plugins in preferences and check the checkbox next to "Set window manager "URGENT" hint".

killall gnome-panel

now watch the taskbar flash as new messages arrive :-)
