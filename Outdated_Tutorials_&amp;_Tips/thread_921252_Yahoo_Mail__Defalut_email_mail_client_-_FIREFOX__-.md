Nice trick which let you set Yahoo (or enven gmail) as Firefox default e-mail client. Today I want to speak about a similar trick which lets you set Yahoo! Mail in a particular way, so that next time you click on any e-mail address link while surfing websites and blogs, Firefox will prompt you to use such an e-mail client. Here is the procedure to follow:

   1. In Firefox 3 and further versions go to Tools.
   2. Click Options.
   3. Click on the Application tab.
   4. Highlight Mailto and select Yahoo! Mail.
   5. Click OK.
   6. Done!
