### Post by WarrenSH on 2010-12-19
[CENTER][SIZE="3"]**Looking to change the appearance of your GNOME/Ubuntu distro? **[/SIZE][/CENTER]

This is a simple how to install the Bisigi-Project Themes

[CENTER]**Run the following ******mand from your terminal*****[/CENTER]

**Step 1: Add the repository**

sudo add-apt-repository ppa:bisigi/ppa && sudo apt-get update
if the server doesn’t respond, just wait a few minutes and try again

**Step 2:  Install Themes**



sudo apt-get install bisigi-themes or click on apt://bisigi-themes



Want to install only a few themes or just one  theme do this.

sudo aptitude install airlines-theme
sudo aptitude install aquadreams-theme
sudo aptitude install balanzan-theme
sudo aptitude install bamboo-zen-theme
sudo aptitude install eco-theme
sudo aptitude install ellanna-theme
sudo aptitude install exotic-theme
sudo aptitude install infinity-theme
sudo aptitude install orange-theme
sudo aptitude install showtime-theme
sudo aptitude install split-theme
sudo aptitude install step-into-freedom-theme
sudo aptitude install tropical-them
sudo aptitude install wild-shine-theme
sudo aptitude install ubuntu-sunrise-theme

For people using Ubuntu 10.10 you can try this to install the themes. 

A maverick repository is available for bisigi themes.It’s a temporary repository published to track bugs (or request some new features)

[CENTER]**Run the following ******mand from your terminal*****[/CENTER]]

sudo add-apt-repository ppa:bisigi/dev
sudo aptitude update
sudo aptitude install bisigi-themes
