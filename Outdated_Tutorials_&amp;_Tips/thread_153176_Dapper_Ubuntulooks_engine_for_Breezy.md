Hi,

For anyone who's interested in trying out the new Ubuntulooks engine that's in Dapper on their Breezy system, I made a debian package. It's exactly the same as the one in Dapper, I just renamed the Human theme to Human-Dapper so that the current Human theme wouldn't get overwritten.

Link here:
[http://rapidshare.de/files/16876915/gtk2-engines-ubuntulooks_0.9.9-1_i386.deb.html](http://rapidshare.de/files/16876915/gtk2-engines-ubuntulooks_0.9.9-1_i386.deb.html)

I had to put the package up on rapidshare (.deb attachments arent allowed on the forums :confused:)

Installation instructions:
```
sudo dpkg -i gtk2-engines-ubuntulooks_0.9.9-1_i386.deb
```

Let me know if you have any problems. It seems to be working fine on my system though.
