### Post by hakermania on 2011-05-27
It is time consuming to type 'clear' and give enter. It is time consuming to type 'exit' so as to exit the terminal....
Or it is probably too lazy :P

Well, there is a very simple way to execute such commands with one letter.
Edit your ~/.bashrc file and in the last line add:
```

alias c="clear"

```
restart your terminal and type 'c' and hit enter.
It will clear your screen. This can be done with any command and it is very helpful if you execute all these commands frequently.
By default there are aliases for the grep and for the ls commands (l = ls and ll = ls -al)

I found all this very useful as i use the terminal a lot, so I shared it with you
