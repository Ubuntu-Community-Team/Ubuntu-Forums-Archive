### Post by Xgkkp on 2005-04-11
I like using gnome much more than KDE (I prefer the methods and styles), however there are still some KDE tools that I like to use, most notably Kile for latex editing. However I found that it looked ugly and didn't match the rest of the interface well. After looking around for some gnome-based alternatives I discovered it is a relatively simple matter to change the fonts so that it looks exactly the same!

Type this into console to install and run kcontrol
```
sudo apt-get install kcontrol
kcontrol
```

Then browse to the font settings "Appearence and Themes -> Fonts", and assuming that you still have all the ubuntu gnome defaults, change all the fonts so that it matches the settings shown in the attachment.

If you have changes the gnome font settings, simply change them to match the settings whoen in system->Preferences->Font
