### Post by Biltong (Dee) on 2006-08-26
Dual Booting Windows XP and Ubuntu Dapper Drake 6.06LTS

Using XP Installation Disk, Partition drive to 2 equal sized partitions...(Needs to be at LEAST 40GB drive!)

Format NTFS for Partition 1 - Windows XP
Format FAT32 for Partition 2 - Ubuntu

Shutdown and Then...

Install Windows XP
Shutdown XP when installation is complete...

Then..

Restart using 98 startup disk with FDISK on it (or Win98 CDrom) 
Run FDISK and change 'active' partition to partition 2.

Shutdown and Restart with UBUNTU disk in drive.

Then...finally...

Install Ubuntu using DEFAULT settings (1st option)

That's IT!!
Who said it has to be hard to do?? :biggrin:
