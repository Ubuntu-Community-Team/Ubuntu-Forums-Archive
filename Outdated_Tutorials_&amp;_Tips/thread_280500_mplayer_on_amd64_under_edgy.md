### Post by benow on 2006-10-19
I've created a bundle containing 32bit mplayer and required libs.  It is self contained (except for codecs) and should run on all distributions, including edgy.  It is a collection of required gentoo's emul-linux-*-x86 x86 emulation binaries and mplayer-bin (a 32 bit mplayer compile).  Installation instructions are in README, in the tarball:

[http://benow.ca/misc/fat-mplayer-bin-0.1.tar.bz2]("http://benow.ca/misc/fat-mplayer-bin-0.1.tar.bz2") (23M)

It's quite the download, but it works!
