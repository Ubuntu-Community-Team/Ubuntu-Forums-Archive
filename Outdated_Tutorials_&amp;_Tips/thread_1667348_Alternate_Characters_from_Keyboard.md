### Post by aarceng on 2011-01-14
I'm excited to find this although many power users are probably aware of it.

As an engineer I often need to use Greek characters. I have just found that I can set up a second keyboard layout for Greek and switch between them. This will cut down on the need to use the character map. :p

Here's how:
System -> Preferences -> Keyboard, then select Layouts Tab.

Click [Add] -> Country=Greece, Variants=Greece

You will see a diagram showing the new character set. Click [Add] then [Close].

On the main panel (near the date?) you should see an icon showing the current keyboard layout. Click this to change layouts then &#965;&#959;&#952; &#963;&#951;&#959;&#952;&#955;&#948; &#946;&#949; &#945;&#946;&#955;&#949; &#964;&#959; &#964;&#965;&#960;&#949; &#953;&#957; &#947;&#961;&#949;&#949;&#954;. 

You can also change your regular keyboard to use say "USA International (AltGr dead keys)". AltGr=Alternate Graphics=[Right Alt]. Some keys have 4 characters. Access the extra characters by pressing and holding the [Right Alt], then you can type þ & §.
