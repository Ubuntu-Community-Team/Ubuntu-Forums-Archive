### Post by 3rdalbum on 2009-12-20
You could go into Synaptic or Software Centre, do a search for the package you want and then install it. You could open a terminal and type "sudo apt-get install packagename". Or you could take advantage of the apt: URL feature in Gnome and KDE (might require extra packages for KDE).

Simply hit Alt-F2 and type apt: followed by the name of the package you want to install:

```
apt:pidgin
```

You'll be asked if you want to install the package, and you'll be able to see a description of it before you install too.

What a time-saver!
