### Post by Technoviking on 2006-02-15
I have made a **[COLOR="Red"]VERY[/COLOR]** experimental .deb package for Mozilla Thunderbird 1.5. It is working well for me, but is not fully tested and may damage your Breezy install.

I have not updated the language packs or enigmail, so these packages may fail completely.

1. Get Software 
```
wget -c  http://mikesplanet.net/deb/mozilla-thunderbird_1.5-0ubuntu4~breezy_i386.deb
```

2. Install package
```
sudo dpkg -i mozilla-thunderbird_1.5-0ubuntu4~breezy_i386.deb
```

Here are the other debs that got built with Mozilla Thunderbird 1.5, in case you want them.
