### Post by stinger30au on 2008-10-04
IF your like me and own nero 6 and feel insulted that "Ahead" have only made a version 3 available for us to use natively and you dont want to dual boot xp, you can get around it with Virtual Box

[get the full closed version of virtual box here]("http://www.virtualbox.org/wiki/Linux_Downloads")

make an .iso of your windows xp install cd and save it on your hdd and install and run virtual box (i ont go in to it here on how to do this there is many instructions around on how to do this)

once you have xp up and running install nero 6 and of you go.

on my pc it wont detect my dvd burner but im not stressed. i have installed the "guest additions" and have a shared directory for ubuntu/xp to save and share data.
i can now easily create what ever i want with nero 6 then burn it to disc from ubuntu with K3B. you little rippa.


also if you have a USB printer and you find the standard ubunbtu drivers either dont work or give ou poor print quality or wont let yuou clean the print heads. you have a few choices. 

[you can get a copy of linux turbo print 2]("http://www.zedonet.com/en_p_turboprint_info.phtml")   

and i do have a copy of it and its great, but if you dont want to fork out the cash, then install virtual box and xp pro and install your usb printer and make sure it works.

also make sure you install in your virtual box copy of xp a pdf reader
[i suggest you get foxit pdf]("http://www.foxitsoftware.com/pdf/rd_intro.php") as it is much smaller and faster then abobe acrobat reader


and dont forget to install guest additions and share a directory between ubuntu and xp

go to what ever ubuntu app you want and print a file and save it as a pdf file and save it in the shared directory so your virtual xp can read the file

go in to virtual xp and open the pdf file and print it to your usb printer. done. easy as pie.


this wont work if you have a parallel printer as the paralel printer port is not suppoed in the closed source edition of suns virtual box


i hope this may help enlighten a few who may be experiencing problems like this and hopefully help give them the knowledge to move across to ubuntu
