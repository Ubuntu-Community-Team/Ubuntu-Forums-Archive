### Post by cdekter on 2008-04-25
A brief search didn't turn up any how to's on this topic. It seems really obvious now that I know how to do it, and yet there are a lot of posts around these forums about how the repositories are so slow. The solution:

In Synaptic, choose Settings -> Repositories.
In the Download From drop down, choose Other...
Click the Select Best Server button in the resulting dialog.
Wait a minute (or perhaps more on a slow connection).

Once it's finished, close everything and click Reload in the main Synaptic window. You're done!

This simple procedure picks the best mirror for you. Since I started using Ubuntu I never realised how many mirrors there are and I never got more than about 100k/sec downloading packages because I always just used the default. Now it maxes out my ADSL2+ connection :D
