### Post by Potters Son on 2009-11-02
I don't know about the rest of you, but I've personally enjoyed the default Human window border. Not too flashy, not too small; the buttons are just the right size.

However, I was upset when I found that the application icon in the top-left corner of the window was replaced by a nondescript-looking button. No big; I figured out how to change it back to the icon.

Without learning how metacity themes work, I simply kept a copy of the old human theme (/usr/share/themes/Human) on my Desktop when I upgraded and used it to revert the metacity theme in that folder.

If anyone else wants to try it too, I've attached a copy of the old file. Simply extract the contents (with root privileges) into /usr/share/themes/Human/metacity-1/
