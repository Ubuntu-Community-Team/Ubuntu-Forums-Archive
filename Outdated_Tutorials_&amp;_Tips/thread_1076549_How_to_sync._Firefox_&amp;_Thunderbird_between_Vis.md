I don't know if this has been discussed or not but here goes.

I have Vista setup to use my storage partition for my firefox and thunderbird profiles.

So I got to thinking will this work for Ubuntu also?

And sure enough, it did.

If you have a storage partiton, create a folder for you firefox profile and do the same for your thunderbird profile.

Then, go to your firefox and thunderbird profiles that you are currently using and copy them over to the new profile folders you created.

Now, if you have Vista, open the run command and enter
```
 firefox -P 
``````
 thunderbird -P 
```Select "Create Profile"

Select "Choose Folder"

Browse to the new profile folders you created and select it.

NOw, do the same same on Ubuntu.

Wala your done.
