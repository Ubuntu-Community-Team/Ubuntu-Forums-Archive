[CENTER][CENTER]**_[size="5"]EASY GUIDE[/size]_**
[/CENTER]


**_1. Installing Accelerated Graphic Card Drivers_**
- Go to System< Administration< Hardware Drivers. Highlight the latest accelerated graphic card driver and click the "Activate" button.
[COLOR="Red"]NOTE: YOU MUST HAVE ALL UPDATES INSTALLED AND BE CONNECTED TO THE INTERNET BEFORE DOING THIS[/COLOR]
[CENTER][[IMG]http://img133.imageshack.us/img133/8937/screenshothardwaredrive.png[/IMG]](http://img133.imageshack.us/my.php?image=screenshothardwaredrive.png)[/CENTER]

**_2. Installing Compiz-Configuration Manager_**
- Go to Applications< Add/Remove. Next, go to the selection box called "Show". Change it to "All Open Source applications". Go to the search box and type "compiz". Now check the box next to "Advanced Desktop Effects Settings" and click "Apply Changes".
[[IMG]http://img14.imageshack.us/img14/8828/screenshotaddremoveapply.png[/IMG]](http://img14.imageshack.us/my.php?image=screenshotaddremoveapply.png)[/CENTER]

**_[CENTER]3. Configuring and Using Compiz Fusion 3D Cube[/CENTER]_**
[CENTER]- Go to System< Preferences< CompizConfig Settings Manager. Go to the Desktop tab. Check the boxes next to "Desktop Cube" and "Rotate Cube". You may now configure it how ever it meets your needs.

To use the 3D Cube, hold [COLOR="Red"]CTRL+ALT+RIGHT MOUSE BUTTON[/COLOR] and move around with your mouse.
[CENTER][[IMG]http://img154.imageshack.us/img154/5798/screenshotcompizconfigs.png[/IMG]](http://img154.imageshack.us/my.php?image=screenshotcompizconfigs.png)[/CENTER]

Guide Made For Ubuntu 9.04 Jaunty Jackolope.[/SIZE][/CENTER][/U][/B]
