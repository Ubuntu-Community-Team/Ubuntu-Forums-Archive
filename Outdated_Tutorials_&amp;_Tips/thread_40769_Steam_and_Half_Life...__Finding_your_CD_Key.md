### Post by mike998 on 2005-06-10
If you have used Cedega to install Half Life from CD and want to install Steam to play some of the other great games, you can find the key for your game in the user.reg (located at /home/*username*/.transgaming) file in the [Software\\Valve\\Half-Life\\Settings] area in the file key named "key".

Hope this helps someone!
