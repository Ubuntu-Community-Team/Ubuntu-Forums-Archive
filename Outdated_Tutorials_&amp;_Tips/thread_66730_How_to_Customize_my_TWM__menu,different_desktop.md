It looks like you want [Vtwm]("http://www.vtwm.org/"). It has support for virtual desktops. 
For a panel, you'll need a panel program like fspanel or fbpanel. To change the resolution, use xrandr. It changes the X resolution. Try ```
man xrandr
```
for instructions.
