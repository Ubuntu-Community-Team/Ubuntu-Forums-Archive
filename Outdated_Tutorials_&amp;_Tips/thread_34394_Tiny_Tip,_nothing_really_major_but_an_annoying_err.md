### Post by N'Jal on 2005-05-14
Erm i really dunno if this happens to anyone else but sometimes my desktop icons dissapear and logging out was the only way to return them (which is never good). However i have since found that:

```
killall nautilus
``` 

Resets them and open nautilus at the same time. This is the only workaround and as i said i dunno if it happen's to everyone but it happens to me and this is how i solved it.
