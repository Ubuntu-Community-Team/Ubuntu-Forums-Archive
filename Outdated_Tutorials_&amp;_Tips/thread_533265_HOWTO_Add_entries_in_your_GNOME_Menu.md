### Post by SOULRiDER on 2007-08-23
Since ive seen tons of people asking this in the forums already, i thought I'd make a small tutorial with images included :P Hope people find it useful, here it goes...

First off, right click your applications menu (what you can see in the picture) and hit **"Edit Menus"**

[CENTER][img]http://img136.imageshack.us/img136/7118/37917912kc8.png[/img][/CENTER]

The Menu editor will appear (you can see it in the picture). In the left panel select the category like accessibility, Debian, accessories etc. The right panel will update with the entries in that category.

[CENTER][img]http://img53.imageshack.us/img53/822/87629555cp7.png[/img][/CENTER]

After you have selected the category, click on the **New Item Button**

Another dialog will appear, thats where you will make the actual entry.In this case I will add a Firefox entry.

First of all select what **type** of application it is. You can select **application** or ** application in terminal**, both are quite self explanatory, you can also select a file.

Fill **name** with the application name or whatever you want. Fill [b]command[b/] with the applications command, in this case its firefox. If you selected file in the first box instead of application, you can browse for the file or just add the path to it.

[CENTER][img]http://img371.imageshack.us/img371/4016/74191504iv0.png[/img][/CENTER]

**Comment** is rather self explanatory, isn't it?

Now, select an icon by clicking the **button to the left**, in this case since i haven't selected one yet It sais **no Icon**

Hit the **ok** Button and youre all set :) I hope you liked it.
