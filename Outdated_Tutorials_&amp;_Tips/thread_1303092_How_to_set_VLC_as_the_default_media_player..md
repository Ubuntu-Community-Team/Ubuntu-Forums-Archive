### Post by u-slayer on 2009-10-27
I got tired of mplayer giving me 4 popups before crashing every time I opened a video file it didn't understand. You can manually change the preferences for every type of media file, but that becomes tedious quickly.

Here's the shortcut:
```
cp '/usr/share/applications/vlc.desktop' '/usr/share/applications/totem.desktop' 
```

Bam. 1 step.:p
