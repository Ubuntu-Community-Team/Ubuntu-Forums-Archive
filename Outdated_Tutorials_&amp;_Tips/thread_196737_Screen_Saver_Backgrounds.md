### Post by two-sheds on 2006-06-14
The following is a small script to allow users to test their screensavers as background in the root window.  In order for the script to work, zenity and xwinwrap have to be installed.

I am not sure if zenity is default in gnomes installation, but just in case,  just do sudo apt-get install zenity.  In the included zip file, there are two other files, Screensaverbackground (the script) and xwinwrapcvs.deb.  All that needs to be done is to install xwinwrapcvs.deb and copy the script somewhere and run it using ./Screensaversaverbackground or make a launcher for it.  The script will allow any installed screensaver to be tried as a background and will not quit until cancel is pressed.

A sample of the output is included and the needed files.
