### Post by Drizzel on 2008-07-24
In gtk+ right-click an icon, choose "rename" then hold down Ctrl+Shift and type U2008. Now hit "enter" and you have a nameless icon. For each additional icon you will need to do the same key combo x2, x3, x4 etc.. 

You can enter any ascii characters using this method + unicode. U2008 gives you the blank space needed for nameless icons. I didnt see this tip anywhere so I decided to post a quick how-to. Hope its useful. 

:KS

Edit: Doing the key combo x2, x3, x4 etc for each additional icon is not necessary. Once is enough. :)
