_**If you don't know what terminal is then here is an explanation for noobs (command for installing applications without GUI included) :**_

Terminal (gnome-terminal) in ubuntu is a place were you can type commands for example you can type "sudo apt-get install anyapp" and replace anyapp with an application it will download and install it for you without a GUI (Graphical User Interface).

**_Installing applications with GUI:_**

To install apps with a GUI (Graphical User Interface) then click applications at top left of the screen and click the application at the bottom called Ubuntu Software Centre in ubuntu 9.10 and look at a list of apps with categories included. To open any program click applications at the top left of the screen and move your mouse over the category of an application you want to open.

_**Office for ubuntu:**_

You might know this but there is something in ubuntu almost exactly like office 03 in ubuntu called openoffice.org also known as open office. This is also available for windows and mac if you like it and it is free. To open it click applications at the top left of the screen and go to office and select the program you would like to use.
