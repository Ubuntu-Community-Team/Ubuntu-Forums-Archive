### Post by badaveil on 2008-10-27
Ares Galaxy is an excellent user-friendly file sharing application for downloading audio and video popular under Windows mode. It is still usable in Linux mode, here are the steps:

1. Download the ares.exe application from aresgalaxy.sourceforge.net;
2. From add/remove, install wine emulator,when successfully installed, it will appear under the main menu;
3. Double-click the ares.exe application, when successfully installed, it will appear under the main menu;
4. The ares application will ask to install the Wine Gecko installer. Yes, do it;
5. Under main menu>wine>ares, activate ares and you can use it immediately.

If you still have problems transferring songs or whatever, uninstall and reinstall ares, it is possible it got corrupted as I experienced but after a clean installation, everything is prefect. If no sound comes out, then you like me, forgot to install the necessary gstreamers drivers. If so...

6. From main menu>add/remove, under sound, select all the gstreamers applications>apply changes

That should do it.
