### Post by shu on 2005-04-23
As a happy owner of iPod a felt that not using it's pda-like functions is a waste. ;-)

**Evolution-iPod-Sync** 

After a bit of googling I found this small program:

[http://members.iinet.net.au/~wakes/evolution-ipod-sync-0.2.tar.bz2](http://members.iinet.net.au/~wakes/evolution-ipod-sync-0.2.tar.bz2)

It was a gnome bounty attempt, and looks like it's the first and last release of it. Unfortunately it doesn't work with recent gnome/hal releases. I've written small patch and made it work again. You can find it here:

[http://users.vlo.gda.pl/~szuwarek/files/ipod/](http://users.vlo.gda.pl/~szuwarek/files/ipod/)

Just plug ipod (let it mount itself) and launch the program. 

**Tomboy-iPod-Sync**

For a few weeks I've been happy tomboy user and yesterday I thought I should make use of tomboy, ipod and that fat perl book on my shelf. ;-) I've written small script, that takes tomboy notes and puts them into Notes/Tomboy directory on ipod. Text and links between notes are preserved, everything else is stripped. No more no less.

Just plug ipod (let it mount itself) and launch the program.

Again you can find it here:

[http://users.vlo.gda.pl/~szuwarek/files/ipod/](http://users.vlo.gda.pl/~szuwarek/files/ipod/)

Hope it'll be useful.
shu
