### Post by bobgreen5s on 2005-05-15
I made a little plugin where you can search ubuntuforums.org through the search bar in firefox: 

**[http://www.isoquad.com/ubuntusearch.src](http://www.isoquad.com/ubuntusearch.src)**

put this in **/usr/lib/mozilla-firefox/searchplugins**

restart firefox and you can now search ubuntuforums from firefox  :smile:


screenshot here: **[http://www.isoquad.com/ubuntusearch.png](http://www.isoquad.com/ubuntusearch.png)**
