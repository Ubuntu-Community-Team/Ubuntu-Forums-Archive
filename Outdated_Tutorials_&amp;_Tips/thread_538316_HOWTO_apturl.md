apturl is a piece of software that plugs into firefox and allows someone on the internet post a piece of software using the apt:package-name protocol from your repos (if it is available). It is included in gutsy. If you are working on feisty, and would like it, simply download the gutsy package from [here]("http://archive.ubuntu.com/ubuntu/pool/universe/a/apturl/apturl_0.0+bzr20070816_all.deb"). Once that is done, restart firefox and you will be ready to go! To test, click on [this link]("apt:test"). Does a dialog box come up asking if you want to install the package "test"? If so, you are good to go (say no to this dialog box).

To use apturl in your guides, create a link with apt:package-name as the URL (the name of the package is the filename of the package you want the user to install from the repo).

I hope you enjoyed my guide on apturl!
