### Post by termite on 2006-10-28
I just wrote a pretty simple Karamba theme to monitor your Moblock nasty people blocker.  File attached, and here are some basic instructions:
[LIST=1]
[*]Install Superkaramba: ```
sudo apt-get install superkaramba
```
[*]Create yourself a Superkaramba themes directory: ```
cd
mkdir KarambaThemes
```
[*]Download my theme below.
[*]Extract the file: ```
tar xvvf Moblock-1.0.skz.tar.gz
```
[*]Move it to your themes directory: ```
mv Moblock-1.0.skz ~/KarambaThemes/
```
[*]Open Superkaramba.  In KDE it's under Utilities in your K Menu.
[*]Click on Open Local Theme.
[*]Navigate to your KarambaThemes directory and select the Moblock-1.0.skz theme.
[*]Click Add to Desktop and watch your IN Block and OUT Block accumulate!
[/LIST]

Features: this is very simple.  It just lists your outgoing and incoming blocks.  Hopefully I'll improve it over time.  Ideas (and coding) welcome.
