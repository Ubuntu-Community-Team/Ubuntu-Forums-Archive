### Post by stinger30au on 2008-10-18
after much experimenting and reading i have found a free tool to export all your emails including attachments from thunderbird

[http://nic-nac-project.de/~kaosmos/mboximport-en.html](http://nic-nac-project.de/~kaosmos/mboximport-en.html)

this is the file you need to d/l and install in to thunderbird

once installed, export all email including attachments in mbox format, this can be easily imported to evolution


the contacts from thunderbird is a piece of cake, just start up addres book andhit export and save the file. nothing hard there

then fireup evolution and import the file you saved with the email addresses and evolution can handle it automagically

easy peasy

### Post by gyaneshwar on 2009-01-24
There is also a workaround in case if you can not install extension as I could not.

Locate Thunderbird profile:

Usually it is in:

Your_home_directory/.mozilla-thunderbird/o199w1ls.Proba(or some other funny name)/Mail/Local Folders

There will be list of files like

Inbox
Inbox.msf
Sent 
Sent.msf
etc.

Just copy files without .msf somewhere AS they ARE already in mbox format. So start Evolution and import. 
Pity that i took me 1 h to realize that after trying to solve problem with Add on (unnecessary).
