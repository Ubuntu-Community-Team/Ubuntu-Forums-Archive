### Post by calvinpriest on 2006-11-04
Install Avidemux and K3b from Synaptic, if you don't already have them.

1) In Avidemux, from the menu select Auto->VCD, then select a filename.  It will create an mpg, saving to your Home directory by default.
2) In K3b, from the menu select File->New Project->New Video CD Project.  Drag and drop the mpg you created in step 1.  Click Burn.

That's all there is to it.  Or it worked for me anyway :-).
