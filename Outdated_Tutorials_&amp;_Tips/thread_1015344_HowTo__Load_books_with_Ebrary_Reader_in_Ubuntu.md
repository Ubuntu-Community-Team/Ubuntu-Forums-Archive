### Post by hurtstotalktoyou on 2008-12-18
Currently there is no solution to using Ebrary Reader natively in Ubuntu.  It has to be done through wine.  The trick is, though, that you can't use the latest FireFox.  You have to go back to version 2 or earlier (version 3 crashes).  So here are the steps...

1.  If you haven't already installed wine, go to synaptic package manager and do so.  I use version 1.0.

2.  Download and install firefox 2 for windows (wine).  You can get it from [oldversion.com](http://oldversion.com/program.php?n=firefox).  I used, specifically, [version 2.0.0.1](http://oldversion.com/download.php?idlong=ea9fc00cf72d452f0c6ce856d009c8ca).

NOTE:  If you already have firefox 3 installed through wine, you may or may not be able to have both installed simultaneously.  You're welcome to try it, but if you run into problems you'll have to uninstall version 3 like I did.

3.  Using firefox in wine, go to [this page on ebrary.com](http://site.ebrary.com/lib/anysite/Download) and click on the "Download ebrary Reader for Windows now" link.  Follow whatever instructions pop up.

4.  Restart firefox under wine.

That should do it.  Good luck!
