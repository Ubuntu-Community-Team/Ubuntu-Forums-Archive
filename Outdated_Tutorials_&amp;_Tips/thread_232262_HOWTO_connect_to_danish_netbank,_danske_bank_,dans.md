### Post by rune_kg on 2006-08-08
This thread is only relevant for danes who want to use danske netbank in ubuntu. This is intended for the not totally noob ubuntuuser. If you need more specific instructions, ask for it:)

What do I need:

1) wine. If you dont got it allready, follow this howto: [http://www.ubuntuforums.org/showthread.php?t=149585&highlight=wine](http://www.ubuntuforums.org/showthread.php?t=149585&highlight=wine)

2)cabextract. If you dont got it: sudo apt-get install cabextract

OK lets install ie6

1) download IEs4Linux here: [http://www.tatanka.com.br/ies4linux/download.html](http://www.tatanka.com.br/ies4linux/download.html)
2) extract it with tar -zxvf filename
3) cd to folder ies4linux-x.x
4) run: ./ies4linux
5) choose your settings in the installation
6) start internet explorer with: ~/bin/ie6 or /home/<username>/bin/ie6 or by clicking the icon on your desktop
7) have your *.mst file ready
8) browse to danskebank.dk
9) tryk den af 24 timer i døgnet med danskenetbank på ubuntu!!!!
