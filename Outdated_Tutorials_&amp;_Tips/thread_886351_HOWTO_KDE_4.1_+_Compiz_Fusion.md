### Post by fahadsadah on 2008-08-11
Right, I think it's time for my first ever HOWTO. This one is about installing KDE 4.1, optionally with Compiz Fusion.

DISCLAIMER: If you break your box doing this, it's your own fault. You follow my instructions at your own responsibility. These instructions are for Hardy Heron only, and probably won't do anything on Gutsy/worse. Thanks to [http://psychocats.net/ubuntu]("http://psychocats.net/ubuntu") for step 3.

Now that the boring part is over, let's begin :grin:

1: Print these instructions out/write them down/have them open in another computer. This browser will soon disappear. Also, save any work, and allow any downloads to finish.

2: Hit Ctrl-Alt-F1. This texts disappears, you see a black screen with writing. Log on, and type ```
sudo /etc/init.d/gdm stop
```. Kubuntu users, replace gdm with kdm. Xubuntu users just use the GNOME command.

4: The important one: installing KDE 4.1. ```
sudo aptitude install kubuntu-kde4-desktop
```

5: Now let's start using the new desktop: ```
sudo /etc/init.d/kdm-kde4 start
```

6: For session, make sure KDE4 is selected.

7: Log in

8: Use Ctrl-Alt-F1 again

9: Ctrl-D

10: Alt-F7


Congratulations, you have KDE 4.1 installed. Verify it's working, get used to it, then install compiz: (you must already have your graphics drivers installed)

11: ```
sudo aptitude install compiz-kde compizconfig-settings-manager
```

DONE! To configure compiz effects, use KMENU-Applications-Settings-Advanced Desktop Effects Settings.
