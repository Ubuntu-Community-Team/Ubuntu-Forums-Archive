### Post by mattheweast on 2005-10-25
Dear all,

The official documentation resource for the Ubuntu community is on the help website at [https://help.ubuntu.com](https://help.ubuntu.com)

This website contains a community wiki which is open to all to edit, and we invite you to contribute too!

One of the priorities for the official documentation is to use the valuable information on the forum. The forum, apart from being an excellent resource for technical support on Ubuntu, contains a lot of good information that could be made into more permanent documentation. The Ubuntu wiki can benefit from contribution from forum users. In turn, the forum users can benefit from the information available on the wiki, which is a valuable frame of reference for all the support 

So, the Ubuntu Documentation team encourages all forum contributors to get involved with adding their guides to the Ubuntu wiki.

Some people find editing a wiki quite confusing, so we've prepared a simple guide explaining how to create pages, add formatting, and some style recommendations to integrate your guides with the existing material on the wiki. Find the guide at [https://help.ubuntu.com/community/WikiGuide](https://help.ubuntu.com/community/WikiGuide)

There is now an **official team** which is dedicated to integrating material on the forum to the help wiki. See [this wikipage]("https://help.ubuntu.com/community/forum") for more details, and visit [the team's homepage]("https://launchpad.net/people/wikiforumteam") if you'd like to join!

**You can help to make Ubuntu documentation rock!**

You can also find the Documentation Team at #ubuntu-doc on irc on the freenode network, or at their [mailing list]("http://lists.ubuntu.com/mailman/listinfo/ubuntu-doc"). Other links that might be useful are:
 * [WikiTeam]("https://wiki.ubuntu.com/WikiTeam")
 * [DocumentationTeam]("https://wiki.ubuntu.com/DocumentationTeam")

Thanks for reading, and have fun on the forum/wiki!

The Ubuntu Documentation/Wiki Team.
