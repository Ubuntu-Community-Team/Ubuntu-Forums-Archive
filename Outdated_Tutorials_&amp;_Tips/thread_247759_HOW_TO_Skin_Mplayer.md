**New Users Guide to Skinning Mplayer**

I have been using Linux (Suse then Ubuntu as well as trying a buch of other distros) for about a year and half and finally, today figured out how to skin Mplayer. I guess I'm not the smartest – it is the little things that make me happy. So, I thought I would log how I did it step by step for new users and to remind me later. The long version for those that are totally new to Linux and need things really spelt out. This is my first time writing something like this, any comments/criticism is appriciated.

**Short Version**

Download new skins from [http://www.mplayerhq.hu/design7/dload.html](http://www.mplayerhq.hu/design7/dload.html) 
Extract the skin and cut and paste the new skin folder in to the Mplayers skin folder i.e. /usr/share/mplayer/skin or see step 2 below. Open Mplayer and choose “skin browser” from the main menu.

**Long Version**
[B]
Step 1)[/B]
Download skins from the Mplayer site here: [http://www.mplayerhq.hu/design7/dload.html](http://www.mplayerhq.hu/design7/dload.html)
The skins are down the screen a bit. Click on the name to have a look at the skin. If you like the skin download it by clicking on one of the links on the right of the table, in the same row. Make sure you remember where you save the skin, for example the desktop. Unpack the skin using something like Applications>Accessories>Archive Manager. Double clicking on the file should pop open the archive manager. Click the extract button and the skin file should be extracted in to a newly created folder, with same name as the skin, on the desktop (if that was where the skin file was saved!).
[B]
Step 2)[/B]
Copy the new skin folder in to Mplayer's skins folder.
The skins folder for Mplayer for me was located here /usr/share/mplayer/skin I think if you have installed Mplayer with a normal user account rather than the root (sudo) user account it might be under the home folder i.e. ~home/.mplayer/skin (remember to go to view>show hidden files in file explorer or the .mplayer folder will be hidden. Copy (or cut and paste) the new skin folder from the desktop in to Mplayer's skin folder.
[B]
Step 3)[/B]
Open Mplayer. Right click on the Mplayer screen to pop up the menu. Click the “skin browser” option and select the new skin (or one of the others). Mplayer should change to instantly reflect the new skin choice. That is it.
