I just got done writing the first draft to my guide on setting up MythTV with a Leadtek Winfast TV 2000 XP RM card.  I tried to be as complete as possible.  As I state in the guide, there is nothing special about this card.  This guide should easily be adaptable to any BTTV based card (I think).

Here's the link to my guide:  [http://www2.truman.edu/~dat725/htpc_preface.html](http://www2.truman.edu/~dat725/htpc_preface.html)

NOTE:  I DID NOT WRITE A GOOD PORTION OF THIS TUTORIAL.  I "stole" a lot of it from Joe Kallo, [http://www.quietglow.com/docs/ubuntumythtv.html](http://www.quietglow.com/docs/ubuntumythtv.html) .  I just added a section specific for the leadtek card and a few other sections with information that I think is important.

Since this is my first draft, I expect there to be many errors.  Please email me if you find one.  I hope this helps somebody!  At the very least, it will be a useful guide for myself (I tend to forget things quickly if I don't use them).
