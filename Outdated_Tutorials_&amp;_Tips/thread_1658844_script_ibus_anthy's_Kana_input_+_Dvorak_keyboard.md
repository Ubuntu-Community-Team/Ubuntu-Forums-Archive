### Post by Mesqueeb on 2011-01-03
Hi guys!

I wrote an awesome script for those who use the standard USA Dvorak keyboard, and also type Japanese with the infamous "kana-input method"

**first of all:**
End ibus.

open terminal and type ->
```
sudo gedit /usr/share/ibus-anthy/engine/tables.py
```
paste this into the file:
