After trying several attempts to clone and backup my DVD selection, I have discovered one of the better and easier ways to complete the process.

I run a Windows XP db right now, but I am positive that it will run just as well in Wine.

WHAT YOU WILL NEED:

1) Wine or Windows dual-boot

2) DVD-Cloner III (Windows program)

Just run the program. The user interface is the easiest I've seen and it's pretty much a one-click cloner.

I do realize that running Windows XP programs are taboo here, but I must say that after the AcidRip, DVDbackup/DVD Shrink/K3B, or DVD-Decrypter combos (most which run Windows anyways), are not as user friendly. This is about as easy as it comes.
