Do any of you watch How I Met Your Mother?

To celebrate his awesomeness, I have created some fortunes that throw out his quotes. Hmm, apparently I can't upload files to this... just aswell I've a rapidshare account!

Mint users - fortunes are already installed, they show up automatically when you open terminal.

Ubuntu users - Here's how to install


sudo apt-get install fortunes
sudo apt-get install fortune-mod
sudo apt-get install fortune-min 


If you're asked for yer password, bang it in.
Then, to get them to display automatically when you open terminal, you need to add it to your bash file...


echo "fortune" >> ~/.bashrc
 

These files need to be put in

/usr/share/games/fortune

Here's the links:
