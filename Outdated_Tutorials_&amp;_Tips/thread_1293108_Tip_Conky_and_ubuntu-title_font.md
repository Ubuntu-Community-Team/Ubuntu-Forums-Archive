### Post by AlphaLexman on 2009-10-16
If you want to use the official ubuntu font, it's in the repo's... ttf-ubuntu-title, with conky, I found out the hard way you have to add a backslash before the dash.

Like this:
```
${font ubuntu\-title:size=24}
```
