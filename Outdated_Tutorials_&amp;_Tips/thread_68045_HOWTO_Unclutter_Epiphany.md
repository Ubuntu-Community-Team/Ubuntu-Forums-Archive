### Post by Curlydave on 2005-09-21
This is a very basic tutorial that will teach you an interesting trick in Epiphany. If you're like me and find that Epiphany's menus are too cluttered when using the bookmarks toolbar and tabs, there's a simple solution to cut the menu height in half without losing any functionality. 

Open Epiphany and select "Toolbars" from the Edit menu. Select "Icons" only" from the dropdown menu labeled "Toolbar button labels:". This reduces the size of the oversized buttons Now, here's the trick that's less obvious: Drag all of your bookmark toolbar links off the bookmark toolbar and place them next to the menu options. Then uncheck "Bookmarks Bar" under the View menu. You should now have some more window room to play with while browsing the web!
