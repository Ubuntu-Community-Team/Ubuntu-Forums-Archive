### Post by tarek on 2007-07-14
You can use either kchmviewer or gnochm or xchm.

kchmviewer from KDE:
```
sudo aptitude install kchmviewer
```

gnochm from Gnome:
```
sudo aptitude install gnochm
```

xchm:
```
sudo aptitude install xchm
```

See the attachments.
