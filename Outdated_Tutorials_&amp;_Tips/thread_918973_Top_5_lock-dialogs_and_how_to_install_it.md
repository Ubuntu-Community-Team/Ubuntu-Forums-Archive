This post is about top 5 lock-dialogs and how to install it.

Installation

   1. Download [gnome-screensaver]("http://yabblog.com/wp-content/uploads/2008/09/gnome-screensavertar.gz")
      (right click and save link as)
   2. Open nautilus in sudo by running “gksudo nautilus” in console
   3. Unpack files to /usr/share/gnome-screensaver/
   4. Open gconf-editor (Alt+F2, type ‘gconf-editor’ and press enter) change the entry apps/gnome-screensaver/lock_dialog_theme from ‘default’ to ‘newwave’ or ‘curves’ or ‘nsa’ or ‘hack’ or ‘turbolence’
   5. To add you logo open “System>Preferences>About Me” and add a logo there

To view the post along with Snapshots visit
[http://yabblog.com/2008/09/13/show-offs-2/]("http://yabblog.com/2008/09/13/show-offs-2/")
