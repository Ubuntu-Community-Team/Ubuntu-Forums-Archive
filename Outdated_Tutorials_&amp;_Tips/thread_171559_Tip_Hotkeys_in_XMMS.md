### Post by somnambulist on 2006-05-06
Want to use those fancy special keys on your keyboard with XMMS?  Here's how:

1. In System->Preferences->"Keyboard Shortcuts" set the keys to the appropriate action.  If your keyboard is configured correctly, they should set to "XF86AudioPlay", Stop, Prev, Next, etc...
2. Install the "xmms-xf86audio" package.
3. Now start XMMS and enable the XF86Audio Keys plugin.
4. Done!  :cool:
