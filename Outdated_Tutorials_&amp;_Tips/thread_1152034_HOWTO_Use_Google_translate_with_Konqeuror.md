### Post by rkoma on 2009-05-07
Using Google translate service is not possible using Konqueror 4.1.4 on Kubuntu Interpid 8.10. It is possible to set source and destination language, but translate button does not produce any results.

A workaround can be to use some other site, which uses Google translate service in background. One such example is:

[http://www.krstarica.com/dictionary/](http://www.krstarica.com/dictionary/)

On this site you can find two dictionaries, the first one is only usable if you are from Serbia. The other one (scroll below) is just front-end for Google translate dictionary, set by default to Serbian-English translation. You can change translation languages to any of the languages supported by google translate. Translation button works perfectly :)
