i'm only giving the broad overview here. for further detailed instructions please refer to docs / other sources.

To get the feel of a 3D desktop - follow these steps :-

1/ from the repos - install 2 packages [with all deps]
	a) 3ddesktop   & 	b) brightside

2/ launch the Screen Activity applet from the Administrative Desktop options.

3/ within the Screen Activity applet - configure any 2 corners [preferably horizontally opposite corners] - with the custom command '3ddesk'.

that's it.

now whenever you move your mouse near those corners - you'll get very cool 3D desktop switching capability - ENJOY IT !
