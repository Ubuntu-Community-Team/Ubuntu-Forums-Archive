### Post by nikosft on 2006-08-23
When a user who has as location set Greece view the [https://help.ubuntu.com/6.06/](https://help.ubuntu.com/6.06/) all the links that lead to the proper documents are broken. A fast solution for those users to view the documentation is to replace /el/ in the urls with /C/ . That way they will view the English documentation
