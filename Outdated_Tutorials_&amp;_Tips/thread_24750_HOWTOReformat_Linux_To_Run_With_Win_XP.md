### Post by pseudonym on 2005-04-08
You'll need to wipe your Ubuntu installation with a Linux-based partitioning tool such as cfdisk or qtparted. You can get these on bootable Linux 'live' CDs such as Knoppix and probably Ubuntu Live. Better yet, get the SystemRescue CD, which contains a whole bunch of Linux tools - [www.sysresccd.org](www.sysresccd.org) . This is really handy. I'm using it to prepare my HD for installing Hoary which, as I'm typing this, has just finished downloading! Woohoo! :)
