I experienced a problem with rythbox ealier this week, I tried to play a .mp3 track, but it couldn't detect the total time of the track.

So this is a tip for fixing this:

1. Go to "Applications" and use the add/remove
Install sound converter

2. Convert the file to .mp3

I will just add that i did this with files ripped from a cd, I couldn't see how much total time the tracks were on, but then i converted the to .mp3 once again, and it worked.

(I wrote this because i sought a solution to this my self, but i couldn't find anything easy that worked. Good Luck)
