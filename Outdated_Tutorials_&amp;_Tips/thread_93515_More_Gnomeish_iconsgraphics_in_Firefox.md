The Mozilla folks did a pretty good job of coming up with a default theme that fits in most anywhere (more or less).

The [previous]("http://www.quadrone.org/graphics/") default Firefox theme ([Qute]("http://www.quadrone.org/graphics/")) did look much like Windows. The [current theme]("http://www.mozillazine.org/talkback.html?article=4834") (Winstripe/GNOMEstripe) is actually based on a the [default Mac theme]("http://kmgerich.com/archive/000049.html") (***not*** Windows).

The theme you speak of is fine, but only if one is using the default GNOME icons.  Quite a few people don't (as evidenced [here]("http://art.gnome.org/themes/icon/?sort_by=add_timestamp&thumbnails_per_page=1000&view=list&order=DESC") and [here]("http://gnome-look.org/index.php?xsortmode=alpha&page=0")).

Besides, I've never been fond of the default GNOME icons anyway. ;-)
