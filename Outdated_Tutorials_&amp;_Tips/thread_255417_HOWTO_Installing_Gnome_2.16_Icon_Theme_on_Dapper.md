### Post by GreySim on 2006-09-11
**Disclaimer:**
I've been running this for 2-3 days now and haven't noticed any brokeness, but that doesn't mean that there isn't any. But since it's only an icon theme, I imagine it must be pretty safe (else I wouldn't have even tried it!).

1.) Download the gnome-icon-theme package from the [Edgy page](http://packages.ubuntu.com/edgy/gnome/gnome-icon-theme).
2.) Install it (by double-clicking the package to open with Gdebi and hitting Install).
3.) Go to System => Preferences => Theme and change your icon theme to anything else, then change it back.

If you're running Human, Tangerine, Tango, or Gnome itself, you should now have a much more Tango-ified icon theme. Even Human falls back on some Gnome icons, so installing the updated package will fill in random bits and pieces that were massively out of place before.
