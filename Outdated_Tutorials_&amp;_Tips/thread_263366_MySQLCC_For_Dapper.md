### Post by studspud on 2006-09-23
I've never had much success using mysql-query-broswer, so I prefer mysqlcc.

I was unable to find a mysqlcc package for dapper so I build one from mysqlcc src.

It can be download from [http://petrocik.net/mysqlcc_0.9.4-4_i386.deb](http://petrocik.net/mysqlcc_0.9.4-4_i386.deb)

I don't know if it matters but it was built against libmysqlclient15off
