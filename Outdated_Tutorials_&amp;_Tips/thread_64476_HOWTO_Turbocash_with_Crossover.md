Turbocash works very well with wine .Wine is included with ubuntu and easy to install and
free. Once you have wine installed just download Turbocash from their web site and
unzip and use  wine to install.I have been running Turbocash on ubuntu desktop for serveral weeks.
I believe Turbocash is the best FREE accounting software around.

### Post by robertrej on 2007-03-26
I discouvered that my printer that I added in: system/admin./printing/
had to be set to DEFAULT  which of course not the  default setting and
once thats done my windows app.s work fine  running with wine.

So far I have been  able to run  Turbocash and Atrex  accounting programs
on linux 6.10  with the newest version of wine 0.9.33.
