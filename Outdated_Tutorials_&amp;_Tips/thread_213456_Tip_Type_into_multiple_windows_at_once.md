### Post by dolson on 2006-07-11
This might be common knowledge, but I figured I'd post it here anyhow, as it's been quite handy for admining my three machines at home.

```
sudo aptitude install keyboardcast
```

This will install Keyboardcast, your new best friend.

Now run it from **Applications > Accessories > Keyboardcast**.

Select any windows you want to type into at the same time, and then type into the box. It's really self-explanatory, but some people might not know about this awesome application.

It makes running "sudo aptitude update ; sudo aptitude dist-upgrade" easy as pie on as many systems as you have to administer, or other similar things (assuming you are logged into each with SSH in terminal windows or somesuch).

Cheers.
