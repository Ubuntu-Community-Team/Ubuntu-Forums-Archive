### Post by EricZartan on 2009-07-16
I would like to start this new thread in hope of helping people looking for this! After hours of google and ubuntu forum searches the only hits I could find had complicated and laborious solutions e.g. paragraphs of code and linux commands like makedir etc.

Forget that - You can install fonts very simply with FONTY PYTHON!
[LIST=1]
[*]Just use synaptic to install fonty python 
[*]open fonty python 
[*]open folder with your .ttf fonts
[*]add to a "pog" 
[*]then "install pog"
[/LIST]

(check in Abiword, or Gimp etc. that fonts are installed)

Done!

no need for programming cryptic lines of code etc. ; all done with an easy to use GUI.

I hope this helps and that it shows up soon at the top of google / forum hitlists.

Brgs

Eric

p.s. I did not check if this makes the fonts available for all system applications - like browsers and file windows - but it definately works for applications like abiword, gimp etc.
