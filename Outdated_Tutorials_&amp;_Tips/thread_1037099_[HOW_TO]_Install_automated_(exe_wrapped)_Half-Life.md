Hello. This is a quick how-to on installing 3rd party mods for half-life 2/ Counter Strike. More specifically the mods that come packaged as an exe installer (zipped mods just need to be extracted to your SourceMods folder). The exe mods, when run in wine, give a "It seems like you don't have Steam Installed! You need to install Steam in order to play 'modname'." error. Even if steam is running at the time. To get around this error follow the steps below:

1. Place the mod exe in the wine drive_c folder that Steam is installed in. 

2. Open Steam and click the "Add non-Steam game..." button at the bottom of the My games tab. 

3. When the "Chose Application" window opens click "browse" and select the exe you copied over.

4. Next run the installer from the shortcut that is now in your games list. The installer will now run as normal and let you install the mod.

5. Delete the shortcut you made in the games list and restart steam and the mod will now show up as normal.
