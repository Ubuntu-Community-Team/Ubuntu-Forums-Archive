### Post by majikstreet on 2005-10-15
**What is this?**
This is a bookmark file created by zenrox. (I, majikstreet, am going to help him out)
This file contains many links to helpful information for Ubuntuers.
**How can I help?**
You can post some of your favourite links in this thread and we will try to add them.

**How do I download them?**
Please see zenrox's post below with the latest file.

**How do I install them?**
First you want to download them.
Then open a terminal and cd to the directory where the file is saved (Usually ~/Desktop)
Then in the terminal type this:
```

tar xvjf ubuntubookmark.tar.bz2

```
Now open Firefox (or whatever browser you use), go to "Manage Bookmarks", then File >> Import. Then select "From File". Then navigate to where the file is saved and import it!

Now you can just go to "Bookmarks" and see the bookmarks!

**If you have any suggestions, please post them here! :p **

### Post by sailor420 on 2005-10-27
OK, here's a few:

[http://juljas.net/linux/skype/](http://juljas.net/linux/skype/)
Skype for Linux resources. Fixed the "resource is busy" problem I was having with skype, preventing me from making more than one call.

[http://ubuntuforums.org/showthread.php?t=56835](http://ubuntuforums.org/showthread.php?t=56835)
Kernel Compilation for Newbies. Kernel compilation isn't hard, just intimidating. This helps fix that.

[https://wiki.ubuntu.com/FirefoxNewVersion](https://wiki.ubuntu.com/FirefoxNewVersion)
Firefox 1.5 install instructions. The beta 1.5 runs *so* much faster on Linux than any previous installation. The best thing I've done since installing Ubuntu. This walks you though it, since you can't just apt-get it, and there isnt a .deb package.

[http://ubuntuforums.org/showthread.php?t=64629](http://ubuntuforums.org/showthread.php?t=64629)
Easy Ubuntu. Install and configure many often-wanted pieces of software. Helps you get a working desktop install with codecs, media players, etc all up and running with just a few clicks. Awesome.

[http://ubuntuforums.org/showthread.php?t=66563](http://ubuntuforums.org/showthread.php?t=66563)
Automatix. Very similar to the above.

[http://ubuntuforums.org/showthread.php?t=27039](http://ubuntuforums.org/showthread.php?t=27039)
Howto: Multimedia keys. Helped me get some of those multimedia keys on my laptop running like I wanted them to.

[http://ubuntuforums.org/showthread.php?t=1971](http://ubuntuforums.org/showthread.php?t=1971)
Prelinking.

[http://ubuntu.wordpress.com/](http://ubuntu.wordpress.com/)
Ubuntu Blog. A blog of some guy that posts neat Ubuntu tips.

[http://www.thinkwiki.org/wiki/ThinkWiki](http://www.thinkwiki.org/wiki/ThinkWiki)
ThinkWiki. Not Ubuntu specific, but has all sorts of hard-to-find data about IBM Thinkpads, geared towards Linux, but useful for any Thinkpad user. As I only use IBM laptops, this is a huge help for me.
