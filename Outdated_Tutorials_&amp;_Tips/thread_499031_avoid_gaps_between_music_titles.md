Hi folks

This mini-mini-howto is just posted in case that someone out there wondered (like me) for a long time how you can play music without these annoying gaps between titles..
The problem arises when playing a cd where one title seamlessly follows the other. (Which it merely does not, normally, annoyingly)
E.g. absolution from muse.

install xmms
apt-get install xmms
fire it up, listen to a "continuous" cd. -> annoying gaps
install the crossfade plugin
apt-get install xmms-crossfade
restart xmms
listen same cd -> no more annoying gaps.

Cheers
