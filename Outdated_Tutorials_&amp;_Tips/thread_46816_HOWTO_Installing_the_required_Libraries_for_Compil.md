### Post by martinjh99 on 2005-07-06
To be able to install and compile KDE Styles and themes you need the following packages installed:

```
x-dev
libxext-dev
kdebase-dev
build-essential
libice-dev 

``` 

build-essential is the compiler and associated packages which you may have installed already.
