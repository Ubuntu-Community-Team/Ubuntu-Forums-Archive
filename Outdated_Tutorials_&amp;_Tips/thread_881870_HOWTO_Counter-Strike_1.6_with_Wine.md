[COLOR=Sienna][SIZE=5]Installing Wine
[COLOR=Black][SIZE=2]
Goto [B]System->Administration->Software Sources
[/B]Enable all software repositories under the *Ubuntu Software *and *Third Party Software *tabs.
Now, at a terminal, type in:
```
sudo apt-get install wine
```
[COLOR=Sienna][SIZE=5]Install the Wine gecko engine
[COLOR=Black][SIZE=2]

[COLOR=Sienna][SIZE=5][COLOR=Black][SIZE=2][COLOR=Sienna][SIZE=5][COLOR=Black][SIZE=2]```
wine iexplore http://www.google.com
```You will be prompted to install the gecko engine. Proceed with the installation.


[COLOR=Sienna][SIZE=5]Install Counter Strike 1.6[COLOR=Black][SIZE=2]

[/SIZE][/COLOR][/SIZE][/COLOR][/SIZE][/COLOR][/SIZE][/COLOR][/SIZE][/COLOR][/SIZE][/COLOR][ATTACH]80456[/ATTACH][ATTACH]80457[/ATTACH]
[COLOR=Sienna][SIZE=5][COLOR=Black][SIZE=2][COLOR=Sienna][SIZE=5][COLOR=Black][SIZE=2][COLOR=Sienna][SIZE=5][COLOR=Black][SIZE=2]Proceed with the installation of your Counter Strike 1.6 (CS/CZ).
Now goto **Applications->Wine->Browse C:\ Drive**. Open [B]Program Files->Valve->cstrike.
[/B]Now locate the files: **download.htm **and **motd_temp.htm**
[/SIZE][/COLOR][/SIZE][/COLOR]For each of them, right click, Properties->Permissions and **uncheck** write permissions for your username (ie. mark them as read only).

Finally, execute the game, buy an m4 and give some headshots!
