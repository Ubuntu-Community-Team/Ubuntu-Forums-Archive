### Post by durt on 2006-07-07
[Asymptote]("http://asymptote.sourceforge.net/") is a vector graphics language. I haven't learned it yet, but it looks cool. You can install it by "apt-get install asymptote."

To run an session of Asymptote in [TeXmacs]("http://www.texmacs.org/"), download [http://ydirson.free.fr/soft/texmacs/texmacs-asymptote-0.3.tar.gz]("http://ydirson.free.fr/soft/texmacs/texmacs-asymptote-0.3.tar.gz") and extract the contents to ~/.TeXmacs/plugins/. Then just start TeXmacs and go to Insert -> Session -> Asymptote.
