### Post by Mr Egg on 2006-04-20
This guide will teach you how to install the deskbar-applet as seen in Dapper Drake. The Deskbar-applet is the little search-box up the top and it is very useful for finding files on the fly.

Okay, to get started, open Terminal (Applications -> Accessories -> Terminal) and run the following command:

```
sudo apt-get install deskbar-applet
```

Ubuntu will start downloading some packages, and then installing them to your system. After installation of Deskbar-applet is finished, right-click the bar up the top and click "Add to Panel...". This will bring up a window with alot of different icons and text.

Scroll down to the Accessories section of that window and double-click the Deskbar icon (the icon with the magnifying glass). This will then add Deskbar-applet to your screen. To use it just type some text and it will search as you type.

To remove the deskbar-applet from your bar, just right-click the icon on the left and click "Remove From Panel"
