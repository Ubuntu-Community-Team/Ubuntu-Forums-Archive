### Post by notarikon on 2006-10-21
Loved this in OSX, just found out you can do it in Ubuntu as well:

Select the text you would like to save in a web page, and drag it onto the desktop (or open nautilus window) to automatically create a new text file with the selected text inside.

I use this all the time, thought I'd share :)
