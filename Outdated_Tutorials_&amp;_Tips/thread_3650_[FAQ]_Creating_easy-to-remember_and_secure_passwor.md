### Post by jdong on 2004-11-08
Q: How can I make a secure password that I can actually remember??

A: With passgen! Passgen creates secure yet pronouncable passwords, so you can have some way of remembering them. Samples:
tu1ugune
xohep9go
luna5uwa


Now I don't think those are hard to remember!

To get passgen, go to [http://www.linuxbuilt.com/passwords.php](http://www.linuxbuilt.com/passwords.php). The whole untar,make,make install procedure applies here. See the checkinstall howto if you're confused.

Or, if you trust me and want a .deb,  I attached the .deb I created (I don't remember now if I actually created a .deb or checkinstalled it...)
