### Post by haelters on 2007-03-11
I'm running gnome-main-menu, which I find great and Beryl, which is also very cool. However, they did not seem to like each other with the default settings. When I try to open the main menu (clicking on computer), the pop-up window sometimes was on top of my running windows (something I expected), but on other occasions it came below my running windows. This made the menu quite useless. 

The solution is quite simple. There is a setting in Beryl called FSP (Focus Stealing Prevention). When I put the Beryl setting General Options -> Level of Focus Stealing Prevention to Off, the problem goes away and my gnome-main-menu always pops up above my current windows.

I hope this helps someone !
