### Post by Newc on 2008-10-04
This link is broken...however here is a tutorial that I just wrote...


1. Open Synaptic Package Manager and download the Ndiswrapper GUI
2. Re-boot into windows (or linux as long as you can extract .EXE files)
3. Download a program that can extract .EXE Files (FREE example: WinRAR)
4. Download the wireless driver for your specific card
5. Extract the .inf file (within the driver file with .exe extension)
6. Save file to Linux partition, or move to some type of memory device so that you can move the .inf file to linux
7. Reboot into Linux
8. Go to System>Administration>Windows Wireless Drivers
9. Click "Install Driver"
10. Find the .inf file from earlier
11. If "Hardware Present: yes" You are done!

Sorry if not a complete thorough tutorial, however if "Hardware present: NO" you may want to search these forums, as I personally do not have a clue as to why it did not work for you! Good Luck!
