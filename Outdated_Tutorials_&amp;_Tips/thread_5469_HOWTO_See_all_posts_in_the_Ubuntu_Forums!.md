### Post by jdodson on 2004-11-18
Ok the defaults for forum viewing are set to allow you to see only the daily posts and the maximum is small, this makes it look like the posts are gone, when in reality you just cant see them.  

This is how to fix that anoyance.

Click "User CP" then from "Settings and Options" select "Edit Options" a bit down the screen you can set the values to see all posts, here are mine:

Thread Display Mode - Linear Oldest First(default)
Number of Posts to Show Per Page - 20
Default Thread Age Cut Off - Show all threads

I hope this helps!
