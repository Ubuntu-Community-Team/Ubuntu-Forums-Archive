[COLOR="DarkGreen"]Description:
KHomework is an open-source program availible to everyone. It has a user-friendly interface and loads quickly for you "on the go" people. KHomework is basically an HTML file with many helpful search engines for homework that include SparkNotes, Wikipedia, Wiktionary, and Google with Google Calculator.[/COLOR]

KHomework updates:
[LIST]
[*]I am currently pending a review from sourceforge.net to get a homepage
[*]New version: KHomework 0.27 NOW CAN BE INSTALLED IN ANY INTERFACE (EX. GNOME, KDE, XFCE)
[*]KHomework 0.27- Check changelog.html for details
[/LIST]

[COLOR="DimGray"]Download here:[/COLOR] [http://www.freewebs.com/linuxnewbie946/khomework-0.27.tar.gz](http://www.freewebs.com/linuxnewbie946/khomework-0.27.tar.gz)

To Install:
```
tar -zxvf khomework-0.27.tar.gz
```
```
cd khomework*
```
```
./install.sh
```
Now you are done. Type the code below to run the program.```
khomework
```
