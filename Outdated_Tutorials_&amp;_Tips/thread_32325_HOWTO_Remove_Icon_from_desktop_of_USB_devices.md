[QUOTE=nilus]When a USB device is mounted automatically from the system, the icon is shown in the desktop. How to remove It?[/QUOTE]
 Don't ask questions in this forum, please. This is for posting formulated tips and tricks only.
However;
Go to System Tools > Configuration Editor
Follow this tree:
Apps -> Nautilus -> Desktop
Uncheck the option "Volumes Visible"
