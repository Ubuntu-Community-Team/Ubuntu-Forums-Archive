### Post by landotter on 2005-03-10
KDE has a lot to offer it, like a really excellent printing dialogue.

In fact, there is no program within Gnome that will allow me to make borderless prints with my Canon i475 printer.

So I just tell Nautilus to open the photo with "Kprinter" and it's that quick and easy. The Kprint dialog comes up with all the options for my printer ready to be tweaked.

of course you have to apt-get Kprinter first ;)

Now if somebody could show me how to use Kprinter with the excellent Gthumb, I'd be really happy!

You can have other apps use Kprinter simply by using "kprinter" as the printing command instead of "lprXXX".
