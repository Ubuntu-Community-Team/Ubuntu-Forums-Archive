Hi, to install this 3 things.
Fisrt we download xampp from here:
>  
[http://www.apachefriends.org/download.php?xampp-linux-1.7.tar.gz](http://www.apachefriends.org/download.php?xampp-linux-1.7.tar.gz) 

1- Copy the package to /opt use chmod if it is necesary.
2- Open a shell:
```

$ cd /opt
$ sudo tar -zxvf xampp-linux-1.7.tar.gz

```
And its done!
to start xampp:
```

sudo /opt/lampp/lampp start

```
