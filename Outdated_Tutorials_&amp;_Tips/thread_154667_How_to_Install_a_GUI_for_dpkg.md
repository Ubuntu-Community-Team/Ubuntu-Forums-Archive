### Post by Hairy_Palms on 2006-04-03
One complaint many people have about linux is that if you want to install programs downloaded not from synaptic then you have to use the c-line.

so you dont like opening up the terminal to install that program you downloaded? Well this howto will show you how to install a functional gui for dpkg

even if you use dapper this is an alternative choice to using gdebi

1, first download the setup from [http://two.fsphost.com/hairypalms/Dee%20Setup](http://two.fsphost.com/hairypalms/Dee%20Setup)

2,Double click on Dee Setup (please make sure synaptic is closed first)

3, it will ask you for your sudo password

4, go to your .deb file and right click on it and go to 
"Open with other Application"
then scroll down untill you find Dee Installer

5, if you want to make this install packages when you double click on them then right click on a package and go to the "Open with" tab and select Dee installer.

thats it now noone can say windows is easyer :)
