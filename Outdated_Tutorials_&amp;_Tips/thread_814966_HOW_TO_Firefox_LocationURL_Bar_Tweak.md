### Post by steveneddy on 2008-06-01
How many of us use Firefox 3 and want it to stop listing everything it can think of for our "convenience" while we type in a URL address in the location toolbar?

I am one who think this is an annoying "feature".

If you've used FF for any length of time, you will be familiar with

```

about:config

```

Type this into the URL/Location toolbar, as if you are typing in the address of a web site.

Locate the listing

```

browser.urlbar.maxRichResults

```

Double click the listing and in the box enter in the number of results you would like to see.

I hate this feature and changed mine to zero.

I hope this helps some of you out there.

Take care-SE
