### Post by mikesf on 2007-02-15
I needed to upgrade the bios on my toshiba u205-s5034 laptop. There was only a windows executable file on there website. I called toshiba, and they told me that there is NO way to upgrade the bios using linux. I thought they might be wrong. It's true: they're complete idiots. Here's my super easy way to do it:
1) download the latest .exe from their website
2) unzip the file you just downloaded 
3) there will be a bunch of files .... one of them is an *.iso, in my case it was su200v33.iso which is the bios v3.3 for the 200/205 series laptops
4) burn this file to a disk (as a bootable cd!) ... should be automatic using nautilus burner
5) reboot computer with disk in drive ... if you don't have CD before HDD on your boot, then hold down the "c" key when you power up. this will force into cd boot mode.
6) it may prompt you what to do, e.g. flash bios. in any case, it takes about 5 seconds for it flash the bios.
7) you're now done. restart and take out the disk.
