### Post by [slash]root on 2009-03-27
This is my first post/tutorial that will hopefully show you how to read comics (cbr/cbz) natively using a .

1. Download/install [**qcomicbook**](http://packages.ubuntu.com/intrepid/graphics/qcomicbook)

```
sudo apt-get install qcomicbook
```

[IMG]http://i44.tinypic.com/2it3o9i.png[/IMG]

Or use the [S**ynaptic Package Manager**](http://www.nongnu.org/synaptic/).

[IMG]http://i44.tinypic.com/359lmaw.png[/IMG]

[IMG]http://i42.tinypic.com/2qbxg7r.png[/IMG]

2. Download a [**cbr/cbz**](http://www.goldenagecomics.co.uk/) 

3. Now just double click on the cbr/cbz and away you go.

NOTE: The app is located under **Applications** ---> **Graphics** ---> **qcomicbook**

NOTE: There are Public Domain Comics around, Some publishers even release promo comics too.
