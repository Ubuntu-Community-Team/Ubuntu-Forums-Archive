### Post by hounddog32 on 2007-05-10
Red Hat just released new fonts under the GPL - you can get them from [Red Hat]("http://www.redhat.com/promo/fonts/"). I'm not sure if I prefer them to Bitstream Vera fonts for my system, but I'm having a go! I've put a screenshot (link to a much bigger one) below and how to use them in your system. They are the same size as the common Microsoft fonts - Time New Roman, Arial and Courier New - so if you want to replace the fonts in a document they are perfect!

[[IMG]http://hounddog32.com/images/tn_Screenshot.png[/IMG]]("http://hounddog32.com/images/Screenshot.png")

[LIST=1]
[*]Download the "Native TTF" file
[*]Open the .tar.gz file with fileroller and extract the fonts to ~/.fonts (or to /usr/share/fonts/truetype if you want other users to have access to the fonts)
[*]Rebuild your font cache so you can use the fonts
```
sudo fc-cache -f -v
```
System > Preferences > Font to change your system fonts
[/LIST]
