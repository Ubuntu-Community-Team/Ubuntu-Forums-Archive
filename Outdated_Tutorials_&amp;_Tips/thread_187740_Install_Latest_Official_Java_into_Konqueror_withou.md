### Post by PPower on 2006-06-03
This will install the latest version of java for use with konqueror without the need for Firefox. The latest version at the time of wriiting is
 
**JAVA 1.5.0_07 (Update _7_)**
 
**Instructions:**
 
1> Download Java of the [www.java.com]("http://www.java.com") site. Install it somewhere useful like /usr/
2> Ignore the install instructions. No symbolic links for us. When it is installed open Konqueror properties. Go to the plugins tab.
3> Find the empty text box under Netscape plugins. Click the folder next to it and point it to /<wherever java is installed>/plugin/i386/ns7-gcc29.
4> Click add, check for plugins and restart Konqueror
5> If all goes well visit [http://gemal.dk/browserspy/java.html]("http://gemal.dk/browserspy/java.html") and you should see a number of applets.
 
Please correct me if anything I wrote was wrong. Im away from Linux at the moment
