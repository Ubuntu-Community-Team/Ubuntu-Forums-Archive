### Post by kimbuba on 2007-03-25
Hi guys,
we have created a notification system based on logging text events named alertTail.

If you want to be warned or if you want a specified action to be triggered when "some text" is written to a file then you can use AlertTail.

Imagine iptables warning you about intrusion with a popup with geoip localization, imagine a web application having problems and you want to be notified with custom actions i.e. an email.
Maybe you would like to be warned when some kernel events log to /var/og/messages.

Is your favorite friend logged in your IM client? A notify popup would warn you about that.


Alerttail is fully customizable  by a configuration file.
You can insert as many as "warning regex pattern" and let execute many actions on each of it.

If you are interested please go to:

[http://alerttail.sourceforge.net]("http://alerttail.sourceforge.net")


Note that this is in alpha stage so any feed back would be  appreciated.

Kim.
