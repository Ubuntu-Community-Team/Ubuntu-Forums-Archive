VMware are just released the official vmware-player (version 1.0, it has been in beta, till now). Please note that a full Ubuntu 5.10 VM is available at [http://www.vmware.com/vmtn/vm/](http://www.vmware.com/vmtn/vm/)

Also the Browser-Appliance is based upon Breezy.

What good is this?
1) Hoary users can download the Linux VMWare player and run Breezy to see if they want to upgrade.
2) Breezy users can download the Linux VMWare player and the Breezy vm and see the effect of a dist-upgrade to dapper without hurting themselves.
