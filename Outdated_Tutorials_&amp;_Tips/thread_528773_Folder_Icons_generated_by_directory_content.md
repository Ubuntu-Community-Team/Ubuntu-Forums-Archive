### Post by JNik on 2007-08-18
A while ago I made a script to create folder icons from their content _[using mono]("http://ubuntuforums.org/showthread.php?t=338260")_ (a la xp). It was working (most of the times) but I didn't really like how the code "looked".

So yesterday I decided to port the script to Ruby and after hours of googling I made it!

All you need to run this script is Ruby and Image Magick tools (convert & montage).

Instructions
[LIST]
[*] Download the compressed file.
[*] Uzip it
[*] Go to the directory you unzipped it and run
    ```
ruby setfoldericon.rb '/path/to/dir/' '/path/to/dir2' ... '/path/to/dirN'
```
[/LIST]

The script will use "Folder.jpg" if it exists, or will pick 4 random videos and images, as directory icon.
