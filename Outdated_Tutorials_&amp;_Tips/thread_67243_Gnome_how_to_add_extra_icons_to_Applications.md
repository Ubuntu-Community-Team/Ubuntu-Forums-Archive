### Post by matthew on 2005-09-19
There is a great tool to do this called Smeg Menu Editor. You can download and install it using Synaptic as follows:

1. make sure the extra repositories are enabled like this: [http://www.frankandjacq.com/ubuntuguide/5.04/index.html#extrarepositories](http://www.frankandjacq.com/ubuntuguide/5.04/index.html#extrarepositories)

2. Open Synaptic and search for smeg

3. Once installed, you will find Smeg in Applications-->System Tools

Have fun!
