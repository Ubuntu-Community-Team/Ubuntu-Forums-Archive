Hey this is my second tutorial
Here it goes
First you obviously must have listen player and xchat if you dont have them your in the wrong post.
Now download my script
```
http://www.datadl.com/files/106/listen_xchat.py.zip
```
Extract the zip and you should get listen_xchat.py

Now launch xchat and listen player (play what ever u want)
to try it out connect to one of the channels and then go in the tab to
xchat->load plugin or script....
Now browse and select the listen_xchat.py then u should get a small messege on your screen
```
 listenclient_xchat v0.1 loaded... for help type /li help
```

If done right you should be able to type in 
```
/li help
```
get this
```

 Listen Player control. Usage:
 /li play    Prints currently playing on Listen Player
 /li prev    Change to the previous song
 /li next    Change to the next song
 /li toggle  Toggle play/pause mode

```
The rest is for you to decide what to do.

Hope it helps some people :)
