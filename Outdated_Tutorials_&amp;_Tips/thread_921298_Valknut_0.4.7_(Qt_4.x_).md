**PeerWeb emoticons for Valknut**

How to install?
To install this emoticons pack follow this steps:
1) unpack archive
2) copy directory 'PeerWeb/' with content to '/usr/local/share/valknut/icons/emot/'
3) run Valknut and go to Preferences
4) go to GUI / Icons section
5) Chose 'PeerWeb' option from dropdown list for Emoticons
6) Click on 'Save' button
