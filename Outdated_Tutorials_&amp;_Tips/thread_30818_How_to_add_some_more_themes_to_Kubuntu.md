### Post by gunnyman on 2005-04-30
If you've looked at kde-look.org, you have found a few themes for KDE.  The thing is, many of them, like alloy, and knifty, have to be compiled from source.
I found a repo that has a few nice themes in .deb packages.
Just do ```
sudo kate /etc/apt/sources.list
``` 
and add the following line:
deb [http://debian.neo.pl/wfmh](http://debian.neo.pl/wfmh) unstable main contrib non-free


then update your sources with sudo apt-get update.

To find them in synaptic search for kwin.
