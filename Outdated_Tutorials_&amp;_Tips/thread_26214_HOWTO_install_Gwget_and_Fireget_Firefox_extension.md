### Post by SilvioTO on 2005-04-12
Gwget is a very good downloader for gnome. Home page: [http://gnome.org/projects/gwget/index.html](http://gnome.org/projects/gwget/index.html)
Get the latest version here: [http://gnome.org/projects/gwget/download.html;](http://gnome.org/projects/gwget/download.html;)
There's a file specific for ubuntu... download it.

Open terminal and:
dpkg -i gwget_0.94-0ubuntu1_i386.deb
The program automatically added in menu -> application -> internet -- download manager


Install Firefox extension:

go to [http://gnome.org/projects/gwget/download.html;](http://gnome.org/projects/gwget/download.html;)
download file FireGet 0.5;
open Firefox;
clic menu -> file -> open file and select Fireget-0.5.xpi;
follow the instruction for install as a normal extension and restart Firefox.


Use:
in Firefox, right clic on file to download and chose "Download file with Gwget".

I hope this is useful.
