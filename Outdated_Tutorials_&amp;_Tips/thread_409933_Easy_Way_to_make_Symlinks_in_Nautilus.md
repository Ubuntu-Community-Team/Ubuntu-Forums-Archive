Short Version:

To make symlinks in Nautilus, simply hold the [Alt] key when dragging a file to the desired location.

_____
Detailed Version:

Until today I only knew how to make symlinks by using the terminal
```
ln -s TARGET DIRECTORY
```I found recently that you can create links in Nautilus by holding [Alt] when dragging the target file (the one you wish to link to) to the desired link location.
Go ahead and try it. For example, goto your home folder. Then press and hold [Alt], and click and drag a file. You can drop the file in the same directory, or a new one. When you release the mouse button (and [Alt]) you will get a context menu with the options to Move, Copy, or Link here.
