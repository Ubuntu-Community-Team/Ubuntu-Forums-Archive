### Post by onlineapps on 2008-05-29
If you're a Kubuntu user, you may have noticed that some GTK+ apps (like Synaptic) look really pathetic when run as root. Here's how to fix that.

[LIST=1]
[*]Install gtk-chtheme (*sudo apt-get install gtk-chtheme*
[*]Run gtk-chtheme as root (*sudo gtk-chtheme*
[*]Select Qt, Apply, and hit OK
[*]You may have to reboot for changes to occur
[*]Open Synaptic as root
[/LIST]
