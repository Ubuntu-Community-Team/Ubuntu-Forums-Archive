### Post by araz on 2006-11-24
This can be very useful for those who want simulate multiple Internet Explorer or Netscape windows who both differentiate between opening a copy of a window (thus the same session) and a new window (thus a new session).

If you want to have different / multiple sessions in firefox you must run different profiles:[INDENT]how to add/edit/run profiles:```
firefox -ProfileManager
```other useful commands:[LIST]
[*]create profile from command line
```
 firefox -CreateProfile <profile name>
```
[*]run firefox from command line specifying a profile
```
firefox -P <chosen profile name>
```[/INDENT][/LIST]
