### Post by lao_V on 2005-04-07
**For Nero 5.x** do the following:

[list=1]
[*][font=Verdana][size=2]Launch Nero. [/size][/font]    
[*][font=Verdana][size=2]Choose File, Burn Image    and you'll be presented with a dialog box. [/size][/font] 
[*][font=Verdana][size=2]Click on the Files of    type: dropdown box and select All Files (*.*). [/size][/font] 
[*][font=Verdana][size=2]Select the .ISO file you    downloaded then press Open. [/size][/font] 
[*][font=Verdana][size=2]Next, Nero will show    another dialog box of CD burning options.  Accept the defaults: [/size][/font]    
[list]
[*][font=Verdana][size=2]Type of Image: Data      Mode 1[/size][/font][font=Verdana][size=2] [/size][/font]
[*][font=Verdana][size=2]Block size (Bytes):      2048[/size][/font][font=Verdana][size=2] [/size][/font]
[*][font=Verdana][size=2]Image header (Bytes):      0 [/size][/font] 
[*][font=Verdana][size=2]Image trailer (Bytes):      0)[/size][/font][font=Verdana][size=2] [/size][/font]
[/list]
[*][font=Verdana][size=2]Choose OK. [/size][/font] 
[*][font=Verdana][size=2]The Write CD dialog box    should appear next. [/size][/font] 
[*][font=Verdana][size=2]Make sure **Write**,    **Finalize CD**, and **Disc-At-Once** are checked, then choose    Write. [/size][/font]  
[/list]  **[font=Verdana][size=2]Tips:[/size][/font]**

  
[list]
[*]   [font=Verdana][size=2]You must use the “Finalize CD” and “Disc-At-Once” option in Nero, or  	you will get the error: "FATAL ERROR: Installation 
	of side by side  	assemblies failed" during setup.[/size][/font][font=Verdana][size=2] 	[/size][/font] 

[*]   [font=Verdana][size=2]You must specify *.* for the file type  	to see the .ISO file.[/size][/font]

 
[/list] 
[b]For Nero 6.X

[/b] [list=1]
[*][font=Verdana][size=2]Launch Nero. [/size][/font]    
[*][font=Verdana][size=2]Choose Recorder / Burn image[/size][/font]
[*] 	[font=Verdana][size=2]Browse to the location of the ISO file and select the  	ISO file you want to burn to cd/dvd[/size][/font]
[*] 	[font=Verdana][size=2]Check the option "Finalize CD (No further writing  	possible!) [/size][/font]
[*] 	[font=Verdana][size=2]Click on "Burn" now.[/size][/font] 
[/list] (copied from [http://members.home.nl/lsnoek/iso.htm](http://members.home.nl/lsnoek/iso.htm))
