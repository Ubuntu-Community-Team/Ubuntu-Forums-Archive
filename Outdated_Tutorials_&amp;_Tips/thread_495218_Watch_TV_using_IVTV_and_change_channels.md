### Post by tarek on 2007-07-07
Hi,

This is more of a tip than a tutorial. I have a Hauppauge TV card and use IVTV to change channels but using the command line is not convenient so I created a simple application to start the TV player and change channels using Java. It makes use of IVTV of course. You don't have to know Java in order to customize your channels and player.

Just thought it would be useful to somebody, it's hosted on sourceforge.net: [tvremote.sourceforge.net]("http://tvremote.sourceforge.net")

Please post any comments/suggestions you have.

Tarek
