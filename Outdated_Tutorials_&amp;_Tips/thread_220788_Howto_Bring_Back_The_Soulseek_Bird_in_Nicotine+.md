### Post by OffHand on 2006-07-22
1) download the file attached to this message.

2) close Nicotine+ if it is running.

3) replace your IMG folder within the Nicotine+ folder with the one you have      just downloaded. You can save your old one somewhere else if you like.

4) replace file encode_bitmaps.py within your Nicotine+ folder with the one you have downloaded.

5) open a Terminal and navigate to your Nicotine+ folder

6) type: python encode_bitmaps.py

When you open Nicotine+ you will see the old-school Soulseek icons again.
