### Post by masnevets on 2006-10-15
I like Azureus, but it's ugly by default. Namely, it uses KDE icons. I've modified the jar file so that it uses tango icons, and I think this will integrate a lot better for those GNOME users out there. I didn't update all of the icon, but the ones that I didn't touch don't seem to appear anywhere. Anyway, have fun with it. You just need to replace your Azureus2.jar with the  file here:

[http://inst.eecs.berkeley.edu/~ssam/Azureus2.jar](http://inst.eecs.berkeley.edu/~ssam/Azureus2.jar)

It's Azureus 2.5.0.0 stable, by the way. See attached screenshot.
