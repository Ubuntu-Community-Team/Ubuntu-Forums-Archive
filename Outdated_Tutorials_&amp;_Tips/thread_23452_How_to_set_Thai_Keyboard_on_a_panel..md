Dear All

I have found that someone want to know about setting Thai font on ubuntu panel.
This may be useful for you who are looking for it.

First, right-click  on a panel  -> add to panel --> (select) keyboard indicator .
Then, you will see the language indicator on the panel. It is the text symbol (us.).

To make keyboard support thai font, 
right - click on that language indicator and 
select--> Open Keyboard Preferrence --> layouts and 
then add thai-font(Thai-Kadmanee or others) from the avialable layouts list to selected layouts.
Finally, you can enjoy using thai keyboard.

Bye.
