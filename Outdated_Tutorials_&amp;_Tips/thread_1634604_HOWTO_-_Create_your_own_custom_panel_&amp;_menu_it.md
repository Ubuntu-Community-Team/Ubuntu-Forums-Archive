### Post by WinRiddance on 2010-11-30
When I really like a desktop background I'd like to be able to view as much of it as possible on my screen. One way to accomplish this is by creating my own custom menu which only displays what's needed ... but without hampering access to everything if I want to access other items at any given time.

**For the following you can not have complete lockdown of panels enabled in UbuntuTweak. If your panels are locked, unlock them first. You can lock your panels again when you're finished.**

Anyway, if you right-click on the panels for example, you'll be able to make them transparent, shrink them to fit, widen them, and so on. By right-clicking a panel, followed by clicking on **Add to Panel** you can even change your entire menu. You can, for example, remove that big, bulky Applications, Places, and System ... followed by replacing that with a single Ubuntu Symbol which will give you the very same access to all of those functions, but without the clutter. There are plenty of things that you can either add, or take away from your panels.

The attachments below will show you the step by step procedure for going one step further, by providing you the means **to create your own Menu** (such as Accessories, Internet, Office, etc.) in which you can then keep all of the application shortcuts for the items that you use all of the time. Makes using your regular apps faster too, since they're all contained in the same menu. Heck, when you're finished you can even change the individual application and menu symbols to something else if you like.

To eliminate the need of having to hunt down icons I've gone ahead and compressed around 1800 of the nicest icon symbols that I could find into a single tarball (zip type) file. This file can be downloaded right here:

**[Click here to download one file with apx. 1800 icon symbols!]("http://www.ubuntu.steinfein.org/Ubuntu_Icons.tar.gz")**

### Post by WinRiddance on 2010-12-03
I'm using a fairly default HANSO theme with only slightly modified colors and SHIKI-WINE controls. My icons are HUMANITY DARK.

I'm not entirely sure but I think the standard font is sans, size 10, don't remember for sure though since my APPLICATIONS font as well as DOCUMENTS font are set to SANS 11. The others are all set to 10 but the window title font is BOLD which I believe to be standard as well, for that theme anyway.

I do like dark themes but it's really hard to find one that's pleasing to the eyes in every application, especially when dealing with tons of chrome or firefox bookmarks constantly. I like one theme/color set for everything. My favorite color is dark red (but not maroon) and I made the SELECTED ITEMS color (for text) kind of gold since that accents the red very nicely.  ;)

I'm not into zillions of effects that can seriously bog my system down when I have a lot of open applications. Consequently I have ANIMATIONS ACTIVATED in my Visual Effects/Custom settings, as well as the desktop cube and the ring switcher, but that's pretty much all as far as effects are concerned. The TRANSPARENT MENUS are also activated in the COMPIZ SETTINGS of **UbuntuTweak**. My upper panel ist set to 42 which let's me view all of the really important stuff easily & quickly, while my bottom panel is set to the smallest, size 24, since I use that strictly to switch between applications when multi-tasking. Since the terminal can be customized too, I decided to enlarge and bold the font there which then permits me to use a transparent background while still being able to read the text in the terminal.
