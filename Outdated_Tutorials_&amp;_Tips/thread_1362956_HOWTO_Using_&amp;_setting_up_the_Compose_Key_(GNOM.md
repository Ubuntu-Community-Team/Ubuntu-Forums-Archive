### Post by tom66 on 2009-12-23
This is a quick guide on how to set up a useful feature for those of us who require estoric symbols or type often in foreign languages. The Compose Key instructs the X Server to treat the next keys you type as a special character sequence. For example, you can press and release your 'Compose Key', then type AE (in capitals or lower case). You will get "Æ" (without the quotes.) If you typed the letters in lower case you would get "æ".

This is a very useful feature and I only recently discovered it. It is much easier than pressing Shift+Ctrl+U and typing a six-digit number, or on Windows pressing and holding Alt then keying in a two-digit to six-digit number on the numpad (the number row doesn't ever seem to work...) It's also much faster than using Character Map. For a full list of sequences see the following page <http://www.hermit.org/Linux/ComposeKeys.html>.

The Compose Key can however slow you down so it is not suitable for languages with a large number of special characters, like Japanese (in fact there are no sequences for Japanese letters that I know of) or Vietnamese which uses many non-Latin but Latin-like characters. More it is suitable for occasional use. If you are going to be typing a lot in foreign languages you might get a keyboard with multiple symbols on it, and flip between them using a panel applet or something similar.

Some older keyboards and fancier keyboards already have a Compose key, so this tutorial doesn't apply to you. You should be able to use that out of the box. This tutorial only applies to GNOME users, I don't have KDE or any other WM and I don't know how they do it. 

1. System > Preferences > Keyboard
2. Go to 'Layouts' tab.
3. Select your primary/default layout. Most users will have only one; those who deal with multiple languages may have many. 
4. Click 'Layout Options...'
5. Under 'Compose key position', set your desired Compose Key. I didn't need the Menu key much (I mean, who does use that when you can just right-click? Seems somewhat redundant) so I chose that. On a bigger keyboard you might use the Right Logo key or the Caps Lock key if you prefer using Shift. 
6. Press 'Close'. The effects should be immediate. Try it out by using the first example. Press your Compose Key, type ae, and, boom, you've got an *æ* symbol.

This is my first tutorial/guide, I hope people find it helpful. This information is available all over the internet, but I thought it might be nice if it were available in a more simple format and in a familiar place. :)
