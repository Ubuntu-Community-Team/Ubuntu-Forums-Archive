Hi,
This 'guide' will show you  how to make limewire run. I have tested this on Ubuntu 10.7 Gusty 32 bit, but I am sure it will work on all other versions and 64bit. If you are using Compiz Fusion or Beryl with Limewire, i'm sure you will have come across where when you start it up all you get is a blank grey window. Well this is caused by Compiz Fusion or beryl. If you are using Ubuntu 10.7 Gusty, follow these instructions:
Open up:
```
System => Preferences => Appearance

```

Click on:
```
Visual Effects tab
```

Then Select:
```
none
```

Then start LimeWire by going to:
```
Applications => Internet => Limewire
```

Once it has started enable Visual effects by going to:
```
System => Preferences => Appearance
```

Clicking on:
```
Visual Effects tab
```

And selecting:
```
Custom
```

[code]Then click ok on the window that pops up.[code]

If you require more help, feel free to post below.
