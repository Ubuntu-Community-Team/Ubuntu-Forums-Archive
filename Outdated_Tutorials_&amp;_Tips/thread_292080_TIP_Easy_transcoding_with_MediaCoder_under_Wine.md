### Post by dogen2 on 2006-11-03
I was getting frustrated trying to compile a bunch of source code to transcode videos for my ipod. I think I found a better way: [MediaCoder]("http://mediacoder.sourceforge.net/index.htm") under Wine. 

Just install wine and the windows version of Firefox first. (I installed firefox 1.5, since 2.0 kept complaining about not enough disk space during install). Then install MediaCoder from the website. I'm converting an AVI to ipod format. Then I'll try using gtkpod to copy it to my iPod. I hope this tip helps out a lot of frustrated people trying to compile mplayer, ffmpeg or mpeg4ip. 

This program uses a GUI so it should be a lot easier than using the commandline tools
