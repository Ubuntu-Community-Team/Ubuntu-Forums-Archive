### Post by dgorchak on 2006-10-30
Thanks to the **Luke Sandell** (Linuster) for posting this great tool at KDE-apps.org
kio_clipboard provides fast and easy access to clipboard contents: imagine - now you don't need any editor for many routine actions!

So:

1. Go [here](http://www.kde-look.org/content/show.php?content=48046) to get a package (or download the attached one)
2. Just do it :)
```
./configure
make
sudo make install
```

(P.S. I prefer using **sudo checkinstall -D make install** and I recommend you to do the same thing)

3. After restarting KDE you will be able to acess clipboard contents using **clipboard:/** in Konqueror or in file dialog. Attached screenshots are made by the author of this tool.
