### Post by hakermania on 2011-06-26
[SIZE=2]It is quite simple but for firefox funs it will be useful!

STEPS:[/SIZE]

[SIZE=4]1) Make a backup (anywhere handy) of the file /usr/share/applications/firefox.desktop
2) Extract the attached archive and replace the file [/SIZE][SIZE=4]/usr/share/applications/firefox.desktop with the [/SIZE][SIZE=4]generated file (named firefox.desktop as well)
3) **'Repin' firefox***(uncheck "Keep in launcher", launch firefox, check "Keep in launcher")* OR relogin OR restart unity (unity --replace)
4) Enjoy the new firefox feature you added by right clicking on the launcher's firefox icon:
[/SIZE][CENTER][IMG]http://img683.imageshack.us/img683/7435/screenshotmdj.png[/IMG]

[LEFT][SIZE=3]Note that you can also add your own features you would like to by editing the .desktop file and seeing the firefox's available arguments by giving 'man firefox' in a terminal, as the whole thing is quite simple:P[/SIZE]

Enjoy :)
