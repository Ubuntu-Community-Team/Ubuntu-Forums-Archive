hm, well, i don't know how stuff on ppc is supposed to work, but my understanding is that it requires a different binary... ubuntuzilla just downloads the default (and only) i386 binary that is distributed by mozilla, same one you'd get from the mozilla.com website. so if that one doesn't work on ppc, then neither would ubuntuzilla's version, because it's the same one.
sorry. :(

the good news is that there /may/ be a ppc .deb if you contact the mozilla packaging team. see their contact info on [https://wiki.ubuntu.com/MozillaTeam/Contacts](https://wiki.ubuntu.com/MozillaTeam/Contacts)
specifically, the guy most likely to help you would be gnomefreak. (also gnomefreak on these here forums).

good luck!
