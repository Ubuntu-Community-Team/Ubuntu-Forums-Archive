### Post by BigCat4 on 2011-07-02
I had the same issue.  I fixed it by going to System/Preferences/Main Menu, highlighting Internet in the left column, choosing new Item and then filling in the Create Launcher fields.
Type = Application
Name = Thunderbird
Command = thunderbird
Comment = Mail and News Reader
(name and comment can be of your choosing)
