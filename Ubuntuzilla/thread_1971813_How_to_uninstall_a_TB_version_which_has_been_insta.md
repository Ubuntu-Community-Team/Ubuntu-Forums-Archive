### Post by swarup on 2012-05-02
If I install TB using ubuntuzilla, and then need to uninstall TB for any reason, then what is the process for the uninstall? Any time something is to get installed any other way besides the Ubuntu software center, then I like to know in advance how to uninstall since it is sometimes quite a complex task.

---

### Post by nanotube on 2012-05-03
the ubuntuzilla package names are *-mozilla-build. so to uninstall ubuntuzilla-version of tb, just "sudo apt-get remove thunderbird-mozilla-build".
