### Post by nanotube on 2009-01-02
hi
ubuntuzilla lives in /usr/local/bin/ubuntuzilla.py

please read the ubuntuzilla homepage for usage instructions. really briefly, if you want to use it to install firefox, all you need to do is open a terminal, and run command
```
ubuntuzilla.py -a install -p firefox
```
and it will walk you through it.
