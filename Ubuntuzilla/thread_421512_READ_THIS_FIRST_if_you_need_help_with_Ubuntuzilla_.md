### Post by nanotube on 2007-04-24
If you run into a problem with the Ubuntuzilla scripts, before posting please check the [Ubuntuzilla website]("http://ubuntuzilla.sourceforge.net") for usage instructions and other helpful tips. Also make sure to check out the [Help/FAQ section]("http://sourceforge.net/apps/mediawiki/ubuntuzilla/index.php?title=Ubuntuzilla_FAQ") of the website. Chances are, you will find some helpful info there. 

If not (or even if you are just too lazy to check :) ), feel free to start a new thread with your issue in this forum.

Make sure to include the **complete output** that the script generates during your run, including the initial command line that you used to start the script. This will help us troubleshoot your problem.
