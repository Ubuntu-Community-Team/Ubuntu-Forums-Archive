### Post by d0nny2600 on 2007-12-14
Ok im going from memory here so it may be a little off as Im at work with no Ubuntu in front of me. 

Open up Add/Remove. Click preferences and enable all repositories.

Once done the system should update its list of available applications. Search for the restricted drivers module. Install this.

Venture over to System preferences and go to graphics card/screen blah blah.. Search for your nVidia card there. It will work as I had been using a 4400 for a long time with Ubuntu

Menu names are probably wrong but this is pointing you in the right direction!
