### Post by HotShotDJ on 2007-06-14
> **pardesi said:**
> how did you customize ur desktop so welll:pHmmmm....

1)  SuperKaramba for the clock and the calendar (I had to hack the themes to get them exactly how I wanted them -- the calendar still isn't right.  I want "today" to be bold and red rather than the bracket, but I got tired of working on it.)

2) Wallpaper -- I think I got it from National Geographic, but I won't swear to it.  Used GIMP to crop and resize for my desktop.

3) Moved the kicker/pager/system tray from the bottom to the left side -- it just is more visually pleasing (IMHO) and on a wide screen monitor, having it on the bottom wastes desktop real estate.

4) Used KDE's built in transparency options on step 3 elements.  (I'm also using some transparency effects in my KDE menus and pop-ups, but you don't see that in the screenshot)

5) Moved the task-bar to the top and set it to hide.  When I need it, I move my mouse pointer to the top of the screen and it becomes visible.

6) Somebody downloaded the [Oxygen icon theme ]("http://oxygen-icons.org")from KDE 4's SVN and converted them into a KDE 3.* icon theme.  The Oxygen theme isn't complete yet, so there are still some default icons in my KDE.  I found the icon theme as part of something else on kde-look.org.

Probably some other stuff that I forgot to mention. There is SO MUCH you can do with KDE (I'm sure that is true with Gnome as well) as far as icon themes, widow decoration,  widgets, etc.  Go check out [KDE-Look]("http://www.kde-look.org") and see for yourself.  I kept mine pretty simple compared to some!
