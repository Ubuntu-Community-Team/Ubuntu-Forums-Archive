### Post by lamalex on 2007-04-24
open up the GL desktop properties in system > preferences > GL Desktop and there should be an option for how many workspaces (they might be called viewports). Set that to the desired number.

### Post by Spr0k3t on 2007-04-24

Now, restart the desktop effects/beryl and modify the number of workspaces to what you need.  Should work, I helped another out with the same issue just yesterday.
