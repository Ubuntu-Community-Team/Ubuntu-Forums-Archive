For Photoshop, Linux has options GIMP or Krita which deliver everything you need except the highest end options.

I don't know about director MX never used that.

As for Flash, there are a few options [listed here.]("http://linuxappfinder.com/development/flash") I can't vouch for how well any of them are, I don't work with flash creation.

Hope that helps. If you need specific functions from any of these you will need a Virtual Machine like [Virtual Box.]("http://www.virtualbox.org/")  Then you can run Windows XP inside the VM and install any applications you need. Note that you may see a bit of a performance hit, but it should still be manageable.
