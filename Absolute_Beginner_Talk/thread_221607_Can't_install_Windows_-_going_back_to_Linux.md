### Post by HereInOz on 2006-07-23
I have to confess that this is a bit "toungue in cheek" here, but I just couldnt stop myself.

--------------------

As a balance to those who post in this Forum that they are unable to install Ubuntu on their computer, and therefore are forced to go back to Windows, I have a computer which is running Ubuntu quite successfully and with high stability.  AMD Athlon on an Asrock mobo.

I wished, over the weekend, to install Windows 2000 on another HDD on this machine. ( I have a removable drive bay in it) so I removed the Ubuntu HDD and put another in there and ran the install.  All went well until I installed SP4, then it all fell over - blue screen of death, not able to boot into safe mode, nothing.  Board is 5 months old, I changed processor, RAM, and tried again. No luck.

So for all those who reckon Linux is fiddly and Windows "just works", that is not always the case.

I have given up and will run Linux on this machine until it dies of old age.
