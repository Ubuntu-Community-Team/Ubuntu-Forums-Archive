Hello!

You first need to Create an "EXT3" Partition.

Select something like:

"I will manually edit the partition tables"

or:

"I will manually create the Partitions"

Something like that.

Then select "Create a New Partition".
Create an "EXT3" Partition type one & of your preferred size.

Also, in Ubuntu it is good to create a "swap" partition, which should be of size twice of what your PC installed memory is.

Then there is a choice like:

"Done setting up the Partitions"

or:

"Done setting up the Partitions & want to continue"

That should let you move on with your installation.

Hope I helped.

Good Luck.
