### Post by Perfect Storm on 2006-07-29
> **alan yeates said:**
> real player is now available through add/remove if you enable the ubuntu commercial repository. what I don't know is how to associate it with file types!

Right click on the file you want realplayer to open/play and pick "Properties". Go to the "Open with" tab and click the "add" button. Scroll down and pick Realplayer. Now you can choose it from the list. Select and press ok.
