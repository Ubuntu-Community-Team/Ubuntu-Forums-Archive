First let me say welcome. And it didn't boot? Lets go over why. Possibilities:

1. Bunk Live CD. Please don't judge us for a Bunk Live cd. I have a few bunk install and Live CDs myself. It happens when the CD is burned at high speeds, or when it gets a little scratch, or when the media used is really poor quality. Install and Live CD's are WAY more sensitive than an audio CD. Instead of "skipping" over the bad parts, the whole thing just wont work.

2. You may need to do more to boot off a cd. On my GF's toshiba laptop, I have to set the CD as the boot option in the bios menu AND hit F12 THEN select the CD.

3. I might jsut not work. I have an old CD drive that wouldn't boot a Linux CD to save its soul (its in the closet now). My sister's computer JUST WONT WORK with the Ubuntu install CD, the screen goes blank fight off the bat. So if burning a new Ubuntu CD doesn't work....please try another Linux before you give up completely. Which one? you might ask. Either pick Mepis or SUSE, both are excellent OSes.

I hope I was helpful.

Since the liveCD was *factory burned*, it certainly isn't the disk itself.  (in fact, I'm running from such a CD right now.)  However, some notebooks won't boot from a liveCD of any sort (in fact, some won't boot from a CD at all, which makes installing Windows rather a problem on such notebooks). However, the live CD is a good way to *try before you install* Ubuntu (or any other Linux distribution, for that matter).
