### Post by joshrobinson on 2008-03-13
ill teach you a trick...

```
sudo apt-get install apt-file
```
now whenever there is a file you need, say it came up and said libmad.so.0, you would do this
```
sudo apt-file search libmad.so.0
```

it will pop up and tell you what package you need to install to get that file :-p
