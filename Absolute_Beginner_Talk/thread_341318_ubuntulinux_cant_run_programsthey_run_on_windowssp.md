You've **got** to understand that there are some HUGE, ***MAJOR*** differences between Windows and Ubuntu Linux. You (generally) can't run the exact same Windows programs under Linux (the .exe's). BUT:

1) There is a program called WINE that will allow you to use *some* of those programs (mainly focused on games, though), and
2) You can usually find programs that will do the exact same thing (sometimes much better) that will be perfectly compatible with Linux.

Also, you will have some problems getting set up. You'll have to get your hardware working and the software you need installed - sometimes this is easier on some computers than others. But, something you'll have to realize is that after you get everything up and running in Linux, it'll only get better with time (as you tweak everything to your needs). In Windows, it always seemed to me that everything worked at the beginning, but slowly got worse and it got bogged down with malware and unwanted "helper" programs being installed.

As to your problems, make sure you give us as much information as possible. Chances are, there are other people with the same camera, wireless card, etc. as you and can help get everything running smoothly.

As to the wireless, do System -> Administration -> Networking. Do you see "Wireless Connection" anywhere? That's the default program for doing wireless networking, although there are other programs you can use, too.

Also, what version of Ubuntu did you get? Dapper LTS or Edgy Eft?

**Limelight:**
Windows applications will not work on MacOS X
MacOS X applications will not work on Windows

Now the same can be said true by replacing MacOS X with Linux and vice versa.

There is a means around this, but in no way is this means perfect.  If you want to run Windows applications inside of Linux, you have to research to see if the application is compatible through WINE first.
