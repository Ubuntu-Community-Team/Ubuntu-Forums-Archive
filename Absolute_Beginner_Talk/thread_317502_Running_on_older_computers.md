### Post by aysiu on 2006-12-12
Well, yes and no.

Even within a desktop environment itself, there can be a difference in performance, depending on how many services are running. For example, *kde-core* is a lot faster than *kubuntu-desktop*, but they are both metapackages that let you run KDE.

But XFCE is known to be faster than both KDE and Gnome. It's a lightweight desktop environment. And it's fairly easy for an inexperienced Linux user to use.

If you want to get even lighter than XFCE (and I would say at 128 MB, XFCE's advantages are minimal, but it won't really choke until you get below that for RAM), you can use a window manager like IceWM or Fluxbox. Those are bit trickier to configure for a new user, especially Fluxbox.
