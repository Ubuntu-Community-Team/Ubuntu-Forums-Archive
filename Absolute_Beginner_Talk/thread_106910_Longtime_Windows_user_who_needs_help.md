### Post by Evil Whisper on 2005-12-21
I would advise against kazaa... If you must use a p2p client try limewire or frostwire.

I think frostwire is in the repositories.

```

sudo apt-get install frostwire

```

Be sure to enable universe/multiverse.  To find out how to enable extra repositories refer to this link from aysiu's sig: [http://www.psychocats.net/linux/sources.php](http://www.psychocats.net/linux/sources.php)
