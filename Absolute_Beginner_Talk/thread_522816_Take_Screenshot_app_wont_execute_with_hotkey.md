### Post by r3bol on 2007-08-11
I'm trying to take a screenshot whilst accessing a menu in the gimp, but the print screen keyboard button which usually opens the 'Take Screenshot' app wont execute. 
Is there another way to do this or is there an alternative screenshot app that lets me use the hot key facility whatever i'm doing?

Thanks.

---

### Post by macogw on 2007-08-11
You could start up the screenshot app, then set it for a 10-second timer, hit OK, and then open the menu and it'll take it while it's open.
