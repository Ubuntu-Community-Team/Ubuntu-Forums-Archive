### Post by bulldog on 2006-09-20
When you boot the live cd you have only the install icon on your desktop.

But you can use the programs in your Ubuntu menu.

Loading the live cd with only 128 MB RAM, will make it awfull slow,if it will boot at all.
To load the live cd you should at least have to double your RAM,for a better experience.

Try the alternate cd which uses less resources.
