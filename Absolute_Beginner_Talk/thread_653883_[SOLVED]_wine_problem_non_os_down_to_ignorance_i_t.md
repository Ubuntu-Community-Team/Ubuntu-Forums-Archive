You know, its really quite sad you are this impatient. If nothing else a search on running things with wine would have come up with plenty of results.

But for your information:

From the command line (Assuming you are already in the folder the file is in)

```
wine whatever.exe
```

And from GUI, right click on the file, select Open with other application, click the little section that says 'Use a custom command' and put this into the text box it reveals:

```
wine
```
