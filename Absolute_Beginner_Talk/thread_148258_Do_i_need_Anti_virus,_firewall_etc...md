### Post by az on 2006-03-21
[QUOTE=-=Raven=-]When i used to run windows i used to have a firewall, norton anti virus, protowall that sort of thing installed. Do i need to take this precaution with ubuntu?[/QUOTE]
no, and no.

There are security vulnerabilities in Ubuntu, but so long as you keep up-to-date with security updates, you are fine.  Viruses are only a small percentage of vulnerabilities, and linux is not really succeptible to that kind of vulnerability because of the way that linux is designed.  AFAIK, Windows would stop working if all the ways that viruses were closed.

You do not need a firewall since nothing you run by default listens to the network.  Even if you install stuff, a firewall is useless.  It's like opening an umbrealla and then cutting a hole in it.

Just don't install stuff that does things of which you are not certain.
