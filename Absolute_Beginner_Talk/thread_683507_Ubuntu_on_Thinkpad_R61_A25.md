### Post by stoodleysnow on 2008-01-31
Please remember: to install Ubuntu as the sole OS you remove Windows. As such all Windows programs are deleted from the computer. If you dual-boot, Windows and its programs remain as a separate entity on the hard disk, Ubuntu will not be able to use programs installed on the Windows partition. You can, however, try installing your favourite Windows apps through WINE, the Windows compatibility layer - but I would only suggest this if you cannot find a Linux alternative.
Hope this helps:)
