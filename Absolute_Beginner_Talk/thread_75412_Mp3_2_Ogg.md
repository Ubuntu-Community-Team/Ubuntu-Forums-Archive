### Post by drogoh on 2005-10-13
Transcoding is evil, long story short.

You'll create artifacts by transcoding, since MP3 throws bits away when it compresses.  Ogg does too, so you're losing more data and therefore encoding poorer quality files.  You don't really need to transcode anything, just install an MP3 player.  It's unsupported because of licensing, but that doesn't mean YOU can't use it.
