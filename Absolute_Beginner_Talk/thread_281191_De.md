ubuntu-desktop is used to install everything that's installed by default, including Firefox, OpenOffice, etc. gnome-core is the core gnome desktop environment. Everything it installs is also installed by ubuntu-desktop, in Dapper. In Edgy, gnome-core is still at 2.14, for some reason.

### Post by skymt on 2006-10-20
Just install ubuntu-desktop. gnome-core is for people who want a very basic Gnome desktop, but not the other stuff ubuntu-desktop installs. If you have ubuntu-desktop, you have gnome-core.

Yes, it will be labeled "Gnome" (or something including it, like "Gnome desktop") in the session chooser at login.
