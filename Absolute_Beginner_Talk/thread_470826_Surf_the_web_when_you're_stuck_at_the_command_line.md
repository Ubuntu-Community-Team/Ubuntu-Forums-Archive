### Post by lazyart on 2007-06-11
This is just a tip for all users who may find themselves stuck at the command line and need help.```
sudo apt-get install lynx
```

Back when the web was in it's infancy you surfed in text mode and lynx was the way you did it.  If you find yourself without a gui, having lynx installed gives you the chance to get help and perhaps get yourself bailed out.  It's not pretty and tough to figure out at first, but from the command line type```
lynx
```and you are connected.  Use G to enter a web address.  You'll get the text version of the webpage, and use enter or forward arrow to "click" on a link.

No, you won't copy and paste, but you can do crude things like see the command you need to enter, maybe even scribble it down on paper but it may be enough to get you going.  Think of lynx as the donut spare included in your car.  It's not the perfect solution, but it can get you far enough to get more help.

Insure yourself by installing lynx now.  When you need it, it'll be there for you.  Of course you can wait and install it when you are at the command line.... if you remember the command of course.
