### Post by Kvark on 2005-07-10
AMD is a good choice, you'll still want the normal 32bit version of ubuntu in any case, but with that AMD you can take advantage of the 64bit version when it matures. 

What comes to my mind when looking at those specs... Would be good if you can find a CPU with 1MB L2 cache. A Serial-ATA hard drive would be faster. If you want to plug in a nvida card instead of that integrated one later then it would be faster with a PCI-express slot then with the AGP8 slot. If you plan on setting up a cable based lan at home then 1Gbps network cards would be cool.

But all that is just minor details that make only small maybe not even noticeable differences. And so is the intel/amd 32/64bit thing. The only things that make a big difference are CPU speed, graphics card model number, hard disk size, ram size, screen size, speaker size, and internet connetion bandwidth.
