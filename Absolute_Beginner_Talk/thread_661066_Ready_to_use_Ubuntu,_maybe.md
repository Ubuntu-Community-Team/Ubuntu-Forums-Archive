### Post by LowSky on 2008-01-07
DRM protected files dont work under Linux...yet (linspire and novell may have something availible soon do to their partnership with Microsoft)

If your used to MS Office and really need Outlook (try Evolution, but It can only do so much) and Access (Openoffice has its own database program but the two are not compatible so you may need to keep Windows or Apple OS/X)

Game support is lackluster, Windows basically the only option. Linux has many games availible, but nothing close to the newest games like Half Life 2, Bioshock, Call of Duty 4. But we do get Doom 3 and Quake Wars.

Some things like printers/scanners/cameras might not work (if you own a Lexmark printer, forget Linux, if you have HP then you'll be okay)

iTunes has no linux/ubuntu version, and support for the newest ipods is  still being worked out. There are other companies that sell music like Amazon that is great.

Some things like Hibernate and Suspend dont work on many machines, some wireless devices are kinda annoying, and the closest thing to a help desk is this forum, but when compared to Windows phone support I perosnally like this forum better.


If you can live with the quirks, then try it out. If you truely need windows for work or personal obligations to keeping your DRMed music, then stay with windows.

### Post by eternicode on 2008-01-07
You will lose Windows' GUI-centric look-n-feel.  While the Ubuntu team (and the various project dev teams) is trying to make a GUI for everything (from UbuntuTweak to Compiz-Config Settings Manager), there are still some problems that can only be solved at a command line.

You will also lose the configurability restrictions built-in to windows.  Never again will you have to use the uxtheme hack :D  But, if your computer usage mostly consists of using the internet (through FireFox?), you probably won't notice much difference.

For writing, you have OpenOffice.org's writer, which has most of the same functionality of MS Word '03.  It takes some getting used to, but it's definitely workable.

For pictures, you can use the GIMP.  Not nearly as intuitive as Photoshop, but it's free.
