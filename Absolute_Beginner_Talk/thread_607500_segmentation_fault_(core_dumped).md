### Post by yabbadabbadont on 2007-11-09
Segfaults while compiling are usually a symptom of either overheating or bad hardware (memory usually).  Reboot and take the option to run memtest on the grub boot menu.  You might have to hit escape to see the menu if Ubuntu is the only OS on the machine.
