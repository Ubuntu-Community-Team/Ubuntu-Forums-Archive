Add/Remove...

 Multisync
A program to synchronize PIM data  
Synchronize calendars, addressbooks and
other PIM data between programs on your computer and other computers, mobile devices, PDAs or cell phones. Currently it has client plugins for Ximian Evolution and IrMC mobile clients (cell phones such as SonyEricsson T39/T68 and Siemens S45i) via Bluetooth, IR and cable. To get all plugins install libmultisync-plugin-all.
To get all Homepage [http://multisync.sourceforge.net](http://multisync.sourceforge.net)
