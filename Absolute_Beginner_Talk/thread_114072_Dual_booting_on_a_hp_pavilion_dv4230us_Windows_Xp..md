### Post by racin on 2006-01-07
A swap file is used by the two operating systems for normal operation. However you can create a file that both operating systems can see and that you can use to move files from one OS to the other. If you format it as fat32 both operating systems can use it. The reboot time really isn't that big of a deal. It only takes a couple extra minutes if a network is not detected. In the directions it talks about the iso burner tool. If you have nero or another burning program like that they are more than capable of burning an iso image to disc. The directions are pretty decent. I had never done a dual boot with linux before this either.
Good Luck and Take Care
