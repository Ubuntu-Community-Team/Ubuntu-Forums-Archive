### Post by Jimbo2184 on 2007-03-13
Hi guys!

Just dropping in to say hi. Hope this is posted in the right place. 

I've just installed Ubuntu 6.10 for the first time, and I have to say I'm impressed. I was less than happy with Windows (or is that WINDOZE?) XP (and now Vista) for years, and dabbled with SuSE Linux but found SuSE that little bit too big for my liking.

Anyway I've taken the plunge and dual-booted Ubuntu 6.10 along Windows XP without a hitch after recieving my free CD through the post. I'm blown away by how easy this operating system is, it detected my Zip drive, PCMCIA cards, Lexmark z600 printer, Dell Monitor etc first time! I was skeptical at first because I've installed stuff on my Sony Laptop and had a rough time getting components to work...

Its now two weeks on since I first tested Ubuntu and found alternatives to all my windows applications so I've deleted the windows partition for good now and only got Ubuntu as the primary operating system on the laptop. Also love how flexible GRUB is as well ;)

Isn't it nice to have a fast reliable operating system on a PC that consumes less memory? My laptop runs so much cooler too because its not getting so stressed with Microsofts bloated code lol. This is one major gripe with Vista, and it seems Linux had better looking desktops long before Microsoft made Vista too, not to mention Ubuntu is free :D

Anyway before I ramble on, I'd like to say thanks for creating a superb rival to Microsoft's operating systems, I've got three friends now running Linux on a permanent basis - so thats progress! 

I know its late but I was playing Cube - the games are great on Linux so no wonder I'm up so late lol.

Cheers!
