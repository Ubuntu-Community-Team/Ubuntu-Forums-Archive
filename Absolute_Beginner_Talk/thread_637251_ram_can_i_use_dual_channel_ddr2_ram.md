### Post by ByteJuggler on 2007-12-10
To clarify:  Dual channel is something supported by your memory controller.  If you have a dual channel capable system (most modern systems in fact) then what this means is if you have 2 identical sticks of memory then you can run them in "dual channel" thereby increasing memory bandwidth.  There's nothing special about the sticks, and you can also use them seperately/in a single channel system.  

Note, having a system that's dual channel capable is a seperate issue of the type of RAM used by your system, e.g. DDR or DDR2.   You get both DDR dual channel and DDR2 dual channel systems.  

To reiterate: Make sure your system supports DDR2 if you want to buy DDR2 memory.  Using dual-channel memory in a non-dual channel system will work.  I suggest you visit [www.crucial.com](www.crucial.com) website to check what type RAM you need.
