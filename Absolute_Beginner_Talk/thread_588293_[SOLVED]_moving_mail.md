### Post by mikeyphi on 2007-10-23
There might be an easier way - but when I change systems I save each mail folder and import it into the new system.
Open Evolution; highlight required folder; on dropdown menu - select folder/select all messages
(all mails are now highlighted)
now select File/save message as
(choose filename & location)

Open new system and import - mail will probably import into the inbox and you can then move to another folder.
