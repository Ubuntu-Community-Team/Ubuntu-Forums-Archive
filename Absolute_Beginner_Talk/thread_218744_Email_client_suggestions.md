### Post by rentonj7 on 2006-07-19
Thunderbird is a little leaner, focusing on Email, while Evolution is a more complete suite including organizational tools.  I'd been using Thunderbird for a few years, until I switched to Ubuntu from Gentoo (have a lot less time on my hands now) and decided to give Evolution a shot since it's the default.  I've been using it happily for a few weeks now, with no regrets.  I still haven't dug in deep, but it runs nice and quick.  One thing that Thunderbird has going for it is it's cross platform, so if you might use a different OS at work, and can make the choice there, it's a little more familiar.  Also, there is a wide variety of extensions for plugins, many available from [http://www.mozilla.com/addons]("http://www.mozilla.com/addons")
