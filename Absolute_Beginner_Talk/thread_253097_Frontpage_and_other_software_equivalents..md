### Post by skymt on 2006-09-07
There are [several](http://www.novell.com/coolsolutions/feature/11684.html) [lists](http://wiki.linuxquestions.org/wiki/Linux_software_equivalent_to_Windows_software) of [equivalents](http://www.linuxrsp.ru/win-lin-soft/table-eng.html).

Frontpage, however, is a problem. As you know, Linux is used mainly by Geeks. Non-Geeks are using it more and more as it gets easier to use, but Geeks are still the main user base. Unfortunately for Non-Geeks, Geeks like to code their own code, not let something code for them. So, most Linux web designers use a text editor like Vim, Emacs, or gedit, or a web IDE like Quanta, Screem, or Bluefish.

So, nvu is the best you'll get. It's not great, but at least it's graphical and [WYSIWYG](http://en.wikipedia.org/wiki/Wysiwyg).
