### Post by LLRNR on 2006-11-28
Hi Eldar!

First off, **W**ine **I**s **N**ot an **E**mulator ! Really, it's not a joke, it's what they call a "recursive acronym". The developers of WINE wish that their app is regarded as "*a compatibility layer for running Windows programs*". The thing is that they implemented their own way of "understanding" the Windows Application Programming Interface (WinAPI) with their own code, 100% non-Microsoft.

Ok so what can you do with Wine? Basically it allows you to run Windows programs on Linux. Don't ask about limitations and stuff, because it *really* depends on the application. (You can run some games through Wine, too, but as I know it the recommended app for games is Cedega, which is non-free.)

To get started, here are some quick links for you:

- [Wine website](http://winehq.org) ;
- [their Application Database](http://appdb.winehq.org) is a place where you can find specific instructions on installing/running a specific application that you need ;
- [HowTo setup Wine](http://doc.gwos.org/index.php/HowToWine) - a great tutorial.

Good luck !

LLRNR
