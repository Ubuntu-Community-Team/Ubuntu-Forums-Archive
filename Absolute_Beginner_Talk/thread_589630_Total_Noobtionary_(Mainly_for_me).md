### Post by MegaJim on 2007-10-24
There's always Google :)

But at the terminal you have a few options to learn about things.

```
$ apropos grub
```

Gives you a list of files/programs/other things with the information you need to see their man-pages

```
$ man grub
```

Takes you to the manpage to read all about it
