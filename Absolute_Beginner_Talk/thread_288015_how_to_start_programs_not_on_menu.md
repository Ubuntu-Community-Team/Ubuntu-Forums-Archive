### Post by taurus on 2006-10-29
You can start it from a terminal, Applications -> Accessories -> Terminal.  Just type the name of the program in there and it would start or show you the error message if there is any...

### Post by lizzard on 2006-10-29
In general you can start a program with Alt-F2 and then enter the program name. 

Tomboy has to be added to the panel.  Right-klick on the panel then select "add to panel" and from the menu you can choose tomboy.

### Post by d4ndy on 2006-10-29
Also, some of the "minor" programs can get stuck into odd places on the menu. In my Dapper menu, Tomboy is in "Accessories" and also "Debian/Apps/Tools." If you use Alacarte, you can just drag the icons to groups you find more congenial. But, yes, Tomboy is also in the "Add to Panel" options.
