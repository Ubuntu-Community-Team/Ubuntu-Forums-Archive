### Post by john.nicholls on 2006-11-28
The GUI is a Graphical User Interface as opposed to using the command line.
A window manager handles the placement of windows on the screen. The desktop environment includes such things as setting a ackground colour or image, providing a menu when you click on the background, and providing panels, launchers, clocks and so on.

Icewm is probably one of the lightest on resources.

### Post by aysiu on 2006-11-28
As far as I know, a GUI is the ability to see and interact with graphics (and I don't mean ASCII art--"graphics" made up of letters and symbols). It usually involves the ability to point and click.

The GUI can be a window manager, which manages what windows and borders look like.

It can also be a full desktop environment, which usually includes a window manager and some other services (a panel, applets, a desktop with icons, etc.).
