### Post by mabelrxu on 2007-04-22
basically, when you're doing the install from the live cd, either do guided (which sometimes breaks and messes up your mbr, so backup first!) or do manual (if you're more sure ... my experience has been this option messes up less, but it's kinda your own fault if it does) ... then, you can use automatix2 to install a thingy that lets you mount your ntfs and fat32 partitions as read and writable ... they're already readable by default (unless something weird happens)
