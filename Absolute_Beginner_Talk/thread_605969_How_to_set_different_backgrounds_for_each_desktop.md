### Post by aysiu on 2007-11-07
> **eXeCuTeR+ said:**
> what's KDE of gnome?
KDE and Gnome are desktop environments.

Desktop environments are basically what you see when you log in (themed windows, icons, toolbars, menus, etc.). It's the total user interface.

Gnome is the default desktop environment for Ubuntu.
There's another version of Ubuntu called Kubuntu, which uses KDE as the default desktop environment.

More details here:
[http://www.psychocats.net/ubuntu/kdegnome](http://www.psychocats.net/ubuntu/kdegnome)
