The import features function is new and only available for windows and a few linux OS's at the moment (I do not think Ubuntu is one because most Ubuntu users will upgrade rather than reinstall and import).

As for formatting, it only formats what you tell it to.  If you selected a default partition layout then yes, it probably did.  But if you select manual, then you can choose what partitions get formated and exactly what their mountponts are.  If you want to install onto the same partition, then yes, you have to reformat.  You would need to create a new partition in order to save data.

I don't want to sound rude, but many of these issues are because you didn't research before you jumped... please read about what you are doing or ask here to avoid this in the future.
