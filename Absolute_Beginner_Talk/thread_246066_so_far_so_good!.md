### Post by skymt on 2006-08-28
Don't be tempted to install XGl/Compiz. I know, the videos are amazing. Wait until it's stable. You can really mess up a system if you do it wrong, and most of the HOWTOs are extremely messy hacks.

Know your way around Synaptic. Almost any software you will need is somewhere in there.

No, you can't run Windows software seamlessly using Wine. It's not there yet. In fact, I'd say it's only half-way. Give it another ten years. :p 

Beware of anything on the HOWTOs, Tips & Tricks board. A lot of times, people post cutting-edge HOWTOs that are more on the level of "it usually works for me" than "it will always work for everyone."

Despite all that, have fun. Try things. Learn the command line.

:-? What was that? Command line? Yes, command line. It's not hard, and you can do so much more with just a few basics. You say you've used DOS. Imagine that with racing stripes and twin jet engines. Find a good tutorial (Google is your best friend) and give a shot.

Ask for help if you need it. It's great that you're here on the forums. IRC is also good. Install XChat. Speak up, someone will probably answer.

And again, have fun!

### Post by Bloch on 2006-08-28
Check out synaptic 
System -> Administratiuon -> Synaptic management

Read this:
[http://monkeyblog.org/ubuntu/installing/](http://monkeyblog.org/ubuntu/installing/)

Don't go mad installing rpm's or compiling (using make / install etc)
Almost everything you need can be installed from the repositories using synaptic. Remember the best programs are not shoved in your face with advertising like commercial software is. It's all just sitting quietly in the repositories waitng for you to install it.

Learn about automatix (or easyubuntu) to install those grey-area windows formats and propreitory software
