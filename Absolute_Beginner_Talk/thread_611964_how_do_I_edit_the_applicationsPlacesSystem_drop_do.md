### Post by dimbulb1024 on 2007-11-30
For adding folders to Places:
1- open a File Browser
2- make sure the Side Pane is viewable
3- in the Side Pane select Places (as opposed to Tree, History, Information ...)
4- select the folder you wish to add to Places and move it there.

If you get to many in creates a Bookmarks tab that expands.

Hope this is clear enough and what you are looking for :)
