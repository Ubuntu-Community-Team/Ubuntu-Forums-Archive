### Post by g3k0 on 2007-02-12
Hoary is an older Ubuntu distribution.  You are probably using either "Dapper Drake" or "Edgy Eft".
Its just a name of a release when it comes out.  It is similar to Windows 95, Windows ME, XP, Vista etc.

Gnome is a Desktop Environment.  A desktop environment includes the window manager(how titlebars and everything look) and random programs.

### Post by jpeddicord on 2007-02-12
Here are a few to get started:
[LIST]
[*]Ubuntu - The OS you are using :)
[*]Debian - What Ubuntu is based off of
[/LIST]
[LIST]
[*]Warty Warthog - The first release of Ubuntu. Obsolete.
[*]Horay Hedgehog - The second release. Obsolete.
[*]Breezy Badger - The third release. Nearing end of life.
[*]Dapper Drake LTS - A long term support release.
[*]Edfy Eft - The most current version of Ubuntu.
[*]Feisty Fawn - The version of Ubuntu in development. Unstable.
[/LIST]
[LIST]
[*]GNOME - The default desktop environment that comes with Ubuntu
[*]KDE - The default for Kubuntu.
[*]XFCE - The default for Xubuntu.
[/LIST]
[LIST]
[*]Metacity - What controls the windows on your system
[*]X - The basic system that enables you to use a keyboard, mouse tablet, and/or display.
[/LIST]

That should be enough to get you started. :)
