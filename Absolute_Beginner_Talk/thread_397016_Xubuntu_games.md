### Post by sawjew on 2007-03-30
It depends on what sort of games you want.  If you want the traditional style card games then I recommend the ace-of-penguins suite.  It has some great solitaire, mastermind, taipei, etc. games but is really lightweight in keeping with the Xubuntu system.

```
sudo apt-get install ace-of-penguins
```

The only hassle is it does not create menu entries so if you want them in the menu you will have to add them yourself.  You will find the executables in /usr/share/ace-of-penguins I think.
