### Post by x64Jimbo on 2006-12-09
Fair enough. I just figured that it was pretty straightforward from there. I mean, if you can get through an installation, you can probably read a context menu, right? The point of Linux is not customization, though. It's free software. The fact that it is customizable is a byproduct of it being open source, and being customizable makes it more attractive to a lot of people, but that's not the main idea. One of the core ideas behind the Linux community is self-help. There is a lot of info out there about this OS, and it is quite easy to find what you're looking for most of the time. Especially with basic questions like this. So while it's nice that you gave all the details and I'm sure the OP will thank you, we need to encourage our newbies to become more informed about their environment, and giving them a starting point rather than a complete walkthrough will be more fulfilling to them in their learning experience.
