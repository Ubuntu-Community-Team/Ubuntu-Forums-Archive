### Post by tuxcantfly on 2006-11-11
if you don't feel like resizing, you could always use a loopmounted filesystem. check out [http://www.topologilinux.com/](http://www.topologilinux.com/) install it then slowly convert your topologilinux installation to ubuntu. it's kinda hard to do, but the only way to keep all your windows partitions without resizing them. I'd still reccommend resizing, though, because you won't lose any data.
