### Post by derred on 2006-08-23
You could try to install it and use it, in fact that is the easyest way to learn to use any operating system and ubuntu(also kubuntu and xubuntu) has an intuitive interface that you will quickly get the hang of(especially if you've used Windows, MacOS or any other GUI based OS).

Also you should read the ubuntu manual at [help.ubuntu.com/]("https://help.ubuntu.com/ubuntu/desktopguide/C/index.html") as it's really well thought out. 

You may also want to consider some of the unofficial guides that teach you how to add multimedia functionality to ubuntu like for example: [http://www.ubuntuforums.org/showthread.php?t=186792]("http://www.ubuntuforums.org/showthread.php?t=186792") or [http://ubuntuguide.org/wiki/Dapper]("http://ubuntuguide.org/wiki/Dapper")

Hope this helps, and best of luck on your journey to Ubuntu:)
