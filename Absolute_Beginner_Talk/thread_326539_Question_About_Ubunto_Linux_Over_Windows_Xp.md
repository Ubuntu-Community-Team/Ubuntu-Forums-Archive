### Post by wpshooter on 2006-12-27
Ubuntu is just one of many different distributions of Linux.

You can have both M/S windows XP and other flavors of M/S Windows operating systems and Ubuntu and various other Linux operating systems on your computer at the same time (referred to as DUAL booting operating systems) by placing the various O/Ss on different partitions on your hard drive.

However, personally I do not prefer having two different O/Ss on the same computer (please note that many people may differ from me on this).  If I am going to run M/S operating system, I want it on it's own computer and if I am going to run Linux operating system, I want it on it's own computer.

### Post by Sef on 2006-12-27
Before installing either as a single operating system or a dual boot ask yourself some questions:

1) What do I need XP for?

2) What Applications do I need?  (Word Processor, Spreadsheet, etc.)

3) Do these applications that I need on XP have an equal equivalent in Ubuntu?  (Some like OpenOffice word is very close in function; others like Impress (a powerpoint equivalent) is much less an equivalent.)

4) How much time can I afford to spend learning a new operating system?

5) What are the reasons I want to change operating systems for?

6) Can I afford downtime if the install does not go right?

7) If dual booting, can I afford not to be able to access XP if I can only boot into Ubuntu? (till that gets fixed.)  

8) Are other people going to use this computer?

9) If yes, how much time can I afford to train them?

10) How do they feel about switching operating systems?

11) Other questions that you can think of that pertain to you and your situation.
