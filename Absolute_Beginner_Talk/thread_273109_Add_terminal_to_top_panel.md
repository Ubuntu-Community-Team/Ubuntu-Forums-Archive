### Post by kent41 on 2006-10-07
Is there some way to add a terminal icon to the top panel instead of going to Applications>Accessories> Terminal ?

Thanks

---

### Post by aysiu on 2006-10-07
Yes, go to Applications > Accessories > Terminal and right-click on the terminal icon. Then, select *Add Launcher to Panel*.

---

### Post by taurus on 2006-10-07
Place a mouse on the panel and click the right button.

Add to Panel -> Application Launcher... -> Accessories -> Terminal -> Add.
