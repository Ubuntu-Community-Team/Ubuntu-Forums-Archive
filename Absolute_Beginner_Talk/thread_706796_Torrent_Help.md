### Post by richaoj on 2008-02-24
you copy the .torrent file and the actual downloaded (incomplete) onto another machine (or the same machine after re-install or whatever).  just when you open the .torrent file in your bittorent client, select the save directory to be wherever the incomplete file is,and it SHOULD pick up where it left off. good luck.  I am like 90% sure this works, but you may want to get independent verification before continuing.
