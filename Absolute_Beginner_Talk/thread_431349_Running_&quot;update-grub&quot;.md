### Post by louieb on 2007-05-03
Running "update-grub" means the install is finished. The only thing left is to put the boot loader/manager in place.  This should not take but a minute or two. Sounds like there is a problem. Did you run the media check before running the install? Strange as it might sound check the reinstall grub link in my signature. You can follow the instructions there to pick up the install where it died. Or if the media check fails burn the CD again at a slow speed (I usually use 2x or 4x) and try the install again.
