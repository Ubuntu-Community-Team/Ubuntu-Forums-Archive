### Post by hquijanog on 2007-08-09
Hi! I'm new in Ubuntu and Linux and this is my first post.

By default, if you have many windows in your taskbar, Ubuntu groups similar window groups in just one window. My question is how to disable that option. I'm more comfortable using  a single window for each program and I don't care that the window gets to tiny because the are many open windows.

Thanks in advance.

---

### Post by Pragmatist on 2007-08-09
1.Right-click the vertical bar connected to the window list
2. Select "preferences"
3. Under "window grouping" select "never group windows"
