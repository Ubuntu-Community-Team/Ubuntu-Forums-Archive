### Post by libertas on 2008-03-21
you have to go into System>Administration>Network and make sure that wireless connection is selected, then click on it and click on properties. Make sure that 'Enable this connection' is selected if you have wireless working (all correct switches are flicked etc.) and there is wireless around you then the name of the wireless should come up in the ESSID box. if it does the select it and enter the encryption code under network password. Make sure that the correct password type is selected (the most common is WPA personal) if all goes well you should now have wireless working :) if not the specify
