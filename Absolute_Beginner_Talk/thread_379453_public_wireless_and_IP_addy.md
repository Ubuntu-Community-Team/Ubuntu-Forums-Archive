### Post by benfindlay on 2007-03-08
you can either install it through synaptic, or launch a terminal and type ```
sudo apt-get install wifi-radar
```

And its all taken care of for you!

Once its installed it will be listed under System>>Internet>>Wifi-radar

Hope this helps!
