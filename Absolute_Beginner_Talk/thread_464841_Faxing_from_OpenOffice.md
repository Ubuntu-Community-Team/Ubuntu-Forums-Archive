### Post by schwascore on 2007-06-13
I have not tried to do this myself, but here are some links I found regarding your question.

Since e-fax software (as far as I know) is only available for windows and mac, I assume that you are using wine to run the e-fax software.  If this is the case, it will be very difficult to connect OpenOffice.org which is running natively to the e-fax software running through wine.

I would take a look at hylafax, which is located at the link below and also available in the repositories.
[http://www.hylafax.org/content/Main_Page](http://www.hylafax.org/content/Main_Page)

Other links:
[http://documentation.openoffice.org/online_help/htmlhelp/text/shared/guide/fax.html](http://documentation.openoffice.org/online_help/htmlhelp/text/shared/guide/fax.html)
[http://documentation.openoffice.org/files/documents/75/1588/SETUP_GUIDE.pdf](http://documentation.openoffice.org/files/documents/75/1588/SETUP_GUIDE.pdf)  (Look at p.65)
