### Post by Pablo_Escobar on 2005-10-17
Torrents -> Easy way :
1. Install Firestarter Package (not needed, but I prefer it that way :) )
2. Start it, make initial configuration (Click next, and save in the end if not sure)
3. Go to the last tab in the Firestarter, there on the lower part You have something like "Allow policy"
4. Right click it and choose Add rule, choose from ComboBox - Bittorent, OK
5. Click apply rules on the Firestarter main bar. 

Presto, You've just made ports for Bittorent available :)
