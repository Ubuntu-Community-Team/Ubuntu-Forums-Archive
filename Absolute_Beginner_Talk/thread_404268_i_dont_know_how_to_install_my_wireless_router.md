### Post by 3rdalbum on 2008-01-10
I don't want to conform to Linux stereotypes by saying "RTFM", but you really do need to read the manual that came with the router.

Usually setup will involve using your web browser to connect to the router, but the actual IP address you need to connect to will differ according to the router. It's usually 192.168.1.1, but the one I bought uses 192.168.0.1 instead. Read the manual, it will tell you.
