Hi all. I plan to buy a new Dell laptop soon -- either a 6400 or a 1505 -- and I need to run both Windows XP and Ubuntu (it's a gravity issue). Dell already offers XP on its "business" computers, and plans to start offering XP again on some home computers. It also plans to start offering Ubuntu. Given all that, which is the best option?

1. Have XP preinstalled and install Ubuntu over XP in a dual boot set up.

2. Have Ubuntu preinstalled and install XP over Ubuntu in a dual boot set up. (I know it makes no sense pricewise, but assume it for the sake of argument. And is this even possible?)

3. Have XP preinstalled and run Ubuntu from a Live CD.

TIA for your advice.
