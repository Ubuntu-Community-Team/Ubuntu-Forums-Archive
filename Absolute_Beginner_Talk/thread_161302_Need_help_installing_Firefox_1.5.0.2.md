### Post by Jagot on 2006-04-16
Just follow the instructions in that Wiki article.  You will however need to download this file instead:

[http://mozilla.mirrors.easynews.com/mozilla//firefox/releases/1.5.0.2/linux-i686/en-US/firefox-1.5.0.2.tar.gz](http://mozilla.mirrors.easynews.com/mozilla//firefox/releases/1.5.0.2/linux-i686/en-US/firefox-1.5.0.2.tar.gz)

Change any references in the Wiki article from 1.5.0.1 to 1.5.0.2

Extract the file from the command line as in the wiki.
