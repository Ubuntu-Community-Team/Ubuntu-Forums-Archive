### Post by Jeroen Vernooij on 2007-10-05
Hi,
1) Have you also chosen a different Icon theme?
2) an efficient way to analyse ubuntu boot is to install bootchart:
```
sudo apt-get install bootchart
```
The chart images will be located in /var/log/bootchart, so you can see which process stalls.
