### Post by P4man on 2007-11-11
If you open bittorrent, it wants to know what to do (continue downloading or uploading) so it prompts you which .torrent file to use. This is perfectly normal. A torrent client without .torrent file is not very useful :)

If you downloaded a torrent earlier, and you didn't save it, and instead opened in right away in your client, check the /tmp folder, thats probably where the .torrent was saved.  Its a good idea to to save .torrent files in a logical place  before opening them in your bt client.
