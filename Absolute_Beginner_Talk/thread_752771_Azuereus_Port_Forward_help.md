### Post by joshrobinson on 2008-04-12
are you using a router? you would have to login to your router to forward a port, also never use the standard port which are in this range 6881-6999 some internet service providers throttle these ports so you dont get good speeds. so try pick any 4digit port you want

if you are using a router, and it has a Upnp option, turn that on, and azureus should do all the work for you

you can find more information on [http://www.portforward.com](http://www.portforward.com)
