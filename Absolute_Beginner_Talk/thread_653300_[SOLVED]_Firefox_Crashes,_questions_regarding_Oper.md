### Post by RomeReactor on 2007-12-29
Hi. If you're running Ubuntu 32-bit, then the current stable version of Opera has problems detecting Flash; the [latest beta]("http://www.opera.com/download/linux/?ver=9.50b") is stable enough to use, and corrected this bug.

If you're running Ubuntu 64-bit, then Opera 64 bit won't be able to use flash.
