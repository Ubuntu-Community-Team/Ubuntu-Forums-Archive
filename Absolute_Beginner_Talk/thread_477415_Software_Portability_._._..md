Nice to hear from you. Ubuntu, and most Linux distros have the drivers you need, some things like video cards and wireless adapters can cause problems, most can be fixed with a little work. If you have hardware you want to use, you can search the forum for the specifics.

Many Windows programs can run in Wine and such, but I would recommend finding a Linux program. Almost all are free. I do not know about your specific program, perhaps someone else will.

Ubuntu comes with burning software, and has its own programs which are easily obtained. 

Ubuntu is a live disk also, so you can run the OS without installing. If hardware doesn't work properly during a live session, it doesn't mean it won't work if you installed and obtained the necessary drivers.
