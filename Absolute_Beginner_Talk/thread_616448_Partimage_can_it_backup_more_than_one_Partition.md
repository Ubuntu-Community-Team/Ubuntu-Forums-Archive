### Post by mikeyphi on 2007-11-18
Haven't used it yet but did some research......
My take is that it works on a single partition - at a time.
So if you want to back up several partitions you would have to make several images....

On rewriting the partition your system is restored exactly - so yes it would boot if you previously booted from that partition!! 
What you can't do (I think) is back up one partition and then clean your drive and replace only the saved partition.

Remember Booting (normally) relies on the first sector on the drive - probably not in the partition you have set aside for Ubuntu!!
