### Post by kebes on 2006-08-06
On a standard Ubuntu install, the default web browser is, I believe, [Epiphany]("http://www.gnome.org/projects/epiphany/"). If you do a Kubuntu install, the default browser is [Konqueror]("http://www.konqueror.org/") (it is used for both file browsing and web browsing).

In either case it is a simple matter to install [Mozilla Firefox]("http://www.mozilla.com/firefox/"), which runs on Linux, Windows and Mac OS X (in fact it's my preferred browser on all three platforms).

You cannot run Microsoft Internet Explorer in Ubuntu. It's a Windows-only program. However you will find that the other browsers I mentioned are functionally equivalent (and are in fact a bit more up to date as of the writing of this post).

If you're used to using MS Internet Explorer, you should have no trouble adapting to the standard browser in Ubuntu.
