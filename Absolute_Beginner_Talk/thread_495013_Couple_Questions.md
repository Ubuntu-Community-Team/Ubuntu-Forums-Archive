### Post by Happy_Man on 2007-07-07
There is a script somewhere in the repos that allows the keyboard command ctrl-alt-del to bring up the Ubuntu equivalent of Task Manager. 

As for defragging, you don't need to. The way the ext3 filesystem is designed makes the need for defragging almost nil. However, every 30 boots or so, Ubuntu automatically checks and defrags itself for you. Don't worry about it.
