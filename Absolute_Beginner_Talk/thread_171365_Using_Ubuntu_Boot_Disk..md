### Post by confused57 on 2006-05-06
The .iso has to be burned to cd as an "image", not directly burned as data.  Just look for the word "image" in your cd burning software.

Make sure your bios is set to boot first from your cd-rom by pressing F2(should say during bootup what to press), this will get you into the bios setup where there should be an option for boot order.

The folder thing is a mystery to me, unless somehow the iso was extracted or something.
