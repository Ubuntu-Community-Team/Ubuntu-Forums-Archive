### Post by pufuwozu on 2006-09-09
1. There's no such thing as just just having Linux. Linux is a kernel, not an operating system. Ubuntu is an operating system that is built using Linux as the kernel.

2. If you just want Ubuntu as your operating system then you just need to download and burn the Ubuntu CD. Then you can just follow the installation "wizard" and it will present you with the option to clear your hard-drive and setup Ubuntu.

I'm with you all the way on that one.[/QUOTE]
 sO I have two questions I want to talk about in this thread.

1. What is the difference between ubuntu and just regular old Linux?
[/QUOTE]
To a beginner, very little. I was impressed how many things "just worked" on Ubuntu compared to some other distros, but that could just be the timing - all distros are getting better all the time.

I keep reading that the Ubuntu community is better than most others in that they are more tolerant of newbies and newbie questions. It's always seemed a friendly place to me.

The basic Linux is almost identical in all distros. The biggest difference by far is whether a distro chooses Gnome or KDE for the desktop - these have a different appearance and different applications that come with them. It's not that big a deal though, because all apps will run under all desktops, just maybe not keeping the look and feel quite right. Consensus seems to be that Gnome is a bit more Mac-like and KDE is a bit more Windows-like. Ubuntu chooses Gnome, and Kubuntu uses KDE.

Beyond that, it's really down to which applications a distro chooses to bundle in.
[/QUOTE]
2. What is the basic process for removing windows and making Linux ubuntu my operating system? (I don't want it to be a secondary OS. I want it to be the PRIMARY operating system on my laptop. I want a computer completely free of windows and it's programs.)[/QUOTE]

Boot the installer CD, and take the option to erase and use the entire disk, rather than the option to shrink windows to make room. Easy.

Good luck.

P.S. 
Try a live CD first. This runs the operating system off of the CD instead of using the hard disk. It's much slower that way, but will help you know if you hardware is compatible before you replace Windows with a dead OS.
