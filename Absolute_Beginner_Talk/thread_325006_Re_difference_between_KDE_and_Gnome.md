### Post by mojoman on 2006-12-25
> **Fittersman said:**
> so basically the difference is how it looks?

That's one of the differences. That said, with a little tweaking you can get your Ubuntu to look like KDE or XFCE and vice versa.

Another important dfference is the programs that comes with it, there's one set of programs for Kubuntu, another for Ubuntu and yet a third set for Xubuntu. So depending on your choice you'll end up with a different browser, file manager, media player etc. Now, it is of course possible to remove a program that you don't like and install, say, konquerer (the KDE browser/file manager) in Ubuntu or Xubuntu but it might entail installing quite a lot of libraries that KDE uses and Ubuntu don't.

Also, I think the programs in KDE are designed by the philosophy "one program for many tasks" (Konquerer can, I think be used as browser, FTP-client, file manager, and probably then some) whereas the Ubuntu programs are by the philosophy "one program for each task". Xubuntu, being slim, has a fairly poor media support but all can be installed (personally, I really liked Xubuntu and especially XFCE when I used it but I now use Gnome).

If I were to let just one of these differences determine my choice it would be the programs that I wanted rather than looks, unless I were using a machine with modest specs, in which case I would go for Xubuntu.

Hope this helps
/Mojoman
