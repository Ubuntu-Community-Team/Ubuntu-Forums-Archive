### Post by mikesmith36 on 2007-04-15
An easier way to install this is to start synaptic package manager (system - administration), then search for notification, scroll to mail notification (packages are shown in alphabetical order), then select it to install.
This seems to install version 3 when 4 does seem to be available but I looked at the download for 4 and it seems complicated.
One thing to note with 3 which isn't immediately obvious, once installed you can change options by clicking system - preferences - mail notification.
It supports evolution and all mailboxes, and it does work!
Only drawback is it notifies you of ALL mail, there is no option to filter spam out.
