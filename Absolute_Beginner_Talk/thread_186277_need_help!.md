[QUOTE=DillPill]How do I access the actual command line? Im new i dont know what that is! Also, is there any plugin that allows me to play .mp3 and .exe files? Thanks a bunch[/QUOTE]

The command line is reached through any number of terminal emulators, the default one included with ubuntu is gnome-terminal, located on the top toolbar on your computer, under "Applications" and the under "Accessories" and it's called "Terminal"

As far as .exe, that is a windows file type and will not run under Linux, or anything other than windows for that matter. (Without an interpreter/emulator, if you wish to know more about that, read up on a program called Wine)
As far as .mp3, there are plenty of topics on the forum about how to use mp3's on your new linux machine, please use the search function on this website to locate them.
