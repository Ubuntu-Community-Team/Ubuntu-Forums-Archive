### Post by qpieus on 2007-10-11
I've not tried quicken with wine because I get the impression that it doesn't work well. Give it a try though, you've nothing to lose. Since you're running an older version of quicken, maybe it'll work OK for you. If it doesn't work well, I suggest running windows/quicken in a virtual machine. I use virtualbox, it's very good. You install windows in a virtual pc and then install quicken in that virtual machine. This way you won't have to reboot your pc into windows just to tun quicken. You can simply start the  windows virtual machine within ubuntu and use quicken in the virtual machine. Saves time and inconvenience of rebooting into windows for just one application.
