### Post by Inxsible on 2007-11-16
> **jmfa59 said:**
> I tried to install Evince and Bittorrent with and and remove but it is already installed how come I don't see them in the applications menu how to put there.Evince = Document Viewer (Applications>>Office) i think.

Bittorrent is not in the applications menu, but if you try to download a torrent file, it will automatically start

You can however add it to the menu by right clicking on the Applications  and clicking Edit Menu. Then you can find it under Internet. Simply check mark the box beside Bittorrent.


As an aside: Deluge or KTorrent are much better torrent applications. you may want to try them rather than using the default bittorrent app in Ubuntu.```
sudo aptitude install deluge-torrent
```or ```
sudo aptitude install ktorrent
```
