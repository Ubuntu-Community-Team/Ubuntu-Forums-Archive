### Post by gmjs on 2008-01-05
In order to add 'Buddies' using the Yahoo protocol, you have to have a Yahoo! IM account yourself.  You can then add this account in with your Windows Live account in Pidgin and will then be able to select it from the first drop-down list when adding a Buddy.  When you sign into Pidgin, both accounts will be active at once.

This is just the way that Pidgin works.  It connects users of the same protocol together, so you need an account with the same protocol as the Buddy you are connecting with.

Such is life!
