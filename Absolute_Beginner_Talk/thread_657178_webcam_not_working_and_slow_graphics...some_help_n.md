### Post by plucky on 2008-01-04
Flannelmouthed,

I just tried to install your webcam driver on my test system and it seemed to work,but as I don't have your webcam I can't test it out. This is what I did

1) Install **build-essential** in synaptic
2) create folder called **webcam**
3) copy qc-usb-messenger-1.7.tar.gz to webcam
4) Used GUI to extract the file which created a folder in webcam called qc-usb-messenger-1.7
5) Open a terminal
```
cd webcam/qc-usb-messenger-1.7
sudo make install
```
This is what I got
> field@Motspur-home:~/webcam/qc-usb-messenger-1.7$ sudo make install
[sudo] password for field:
/usr/bin/install -c -D -m 644 quickcam.ko        /lib/modules/2.6.22-14-generic/misc/quickcam.ko
/usr/bin/install -c -D -m 755 qcset /usr/local/bin/qcset
/sbin/depmod -a

Hope this helps
