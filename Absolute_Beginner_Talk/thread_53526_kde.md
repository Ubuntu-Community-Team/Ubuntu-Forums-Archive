Go to [www.ubuntuguide.org](www.ubuntuguide.org) follow the adding repositories guide and you should have all available repos for ubuntu. To install kde you can either open synaptic click search and enter kubuntu-desktop and install it or open a terminal and do 
```
sudo apt-get update
``` then
```
sudo apt-get install kubuntu-desktop
```
hope this helps.
