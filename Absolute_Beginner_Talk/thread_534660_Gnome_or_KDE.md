### Post by por100pre1 on 2007-08-25
Gnome tends to be more about simplicity, KDE is more audacious with features. But they are really very similar. If you're using Ubuntu you can install KDE and select it at your login screen. 

```
sudo aptitude install kde-core
```

(That's a minimal KDE install, if you want full kubuntu use kubuntu-desktop instead of kde-core.) :)
