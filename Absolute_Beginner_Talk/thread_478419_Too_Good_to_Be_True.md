### Post by lazyart on 2007-06-19
Spit out your apps and we can tell you what equivalents/alternatives exist.

Don't hold your expectations too high trying to run Windows software in Linux.  Wine can pull off some things but it's far from a perfect solution.  Remember Linux isn't Windows nor does it try to be.

What a lot of people do when they have an app that they simply can only run in Windows is create a virtual machine, install windows to it and visit when they absolutely must.  Dual booting is an option as well but it's a bit clunky to reboot every time you have to use a certain program.  If you are running older hardware or maybe don't have a lot of RAM it's probably the better way to do it.
