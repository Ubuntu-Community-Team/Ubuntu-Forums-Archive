### Post by pbaehr on 2006-04-17
I don't know much about the motherboard compatibility issue. Most should work, I would think, but I'm no expert.

As far as the athlon64, the 64 bit version of ubuntu works, but is less supported. I think this is an issue on the vendor end. You can always run the 32 bit version on a 64 bit chip. That's what I do. Depending on your level of experience you can probably get the 64 bit version working, though.

I think in order take advantage of the dual processors you need a special dual processor kernel. I'm sure someone could help you get that set up.

### Post by steve.horsley on 2006-04-17
I think the mobo IS important. I have seen a few thereads here where people have difficulties with mobo compatibility.

Athlon 64 will be no problem at all, although most people are running 32-bit kernels on them because some things like flash and audio codecs are not available in 4-bit versions. I doubt if an X2 should pose any extra difficulties, but can't speak from experience there.

If it helps, I use an athlon 64 on a Gigayte GA-K8NF-9 with a SATA drive and a Nvidia 66600GT PCI-E. Everyhing seems to work fine - video, sound, USB, Ethernet. The video card was in indulgence, but you should see DOOM3! I run the 32-bit K7 kernel.
