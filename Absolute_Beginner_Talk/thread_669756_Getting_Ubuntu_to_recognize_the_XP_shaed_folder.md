### Post by Speedoo on 2008-01-16
You should be able to see the shared folder when you click on "Places", the "Network" from Ubuntu.  If not, you need to make sure your in the same network group (e.g. "MSHOME"  or "WORKGROUP".)  The 2 computers won't connect if they're in different groups.  I'm somewhat new at this too, so I hope this works for you!
