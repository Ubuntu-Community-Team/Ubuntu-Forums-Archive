A torrent file is more or less a marker or tracking file that lets you download the media file you're looking for from other people who are sharing the file (called Seeders).  The standard Ubuntu install includes a basic bittorrent interface, and you can add improved ones (such as BitTornado and Azureus) through the add/remove programs menu.  (To get Azureus, you'll need to add extra package download repositories.  If you need help doing that, let us know. :) )

Typically, when you download a torrent, you'll be given the option to save it to disk or open it with your bittorrent client.  The best move is to save it, then open the file.  This will allow you to resume downloads later if you have to log off in the middle.

Note 1: generally speaking, the download speed is entirely limited by how many people are hosting the file.  If it's not popular, or if it's old, you might have a hard time getting it.

Note 2: if you're using a wireless network, either (a)connect to a wired network, or (b)don't do anything but download the torrent.  Sometimes bittorrent doesn't play well with others through wireless, and it can cause freezes.

Happy downloading!
