### Post by frodon on 2006-01-17
1- A simple drag and drop might work otherwise just create a symbolic link in your desktop folder, i give you an example of the command to use to create a symbolic link : ```
sudo ln -sf /mnt/storage /home/username/Desktop
```

2- Yes, but the write support is not reliable, if you use it too much you will damage your partition.
