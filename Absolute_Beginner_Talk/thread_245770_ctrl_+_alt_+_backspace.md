### Post by Anonii on 2006-08-28
There is a config file about that, I unfortunately dont remember where it is...

But you need to know, that ctrl + alt + backspace, is a really important "command", because it kills X, that means that if you configured something that otherwise would  need a reboot, you can just kill X and then restart it with "startx" and the new configuration will be applied. I reccomend you to either not change this combo, or change it to something that you will remember.
