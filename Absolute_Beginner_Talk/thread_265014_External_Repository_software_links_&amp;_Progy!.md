 This thread is all about external repositories or rather external Linux software links. The idea is to build a repository of links to external software packages that are not available in Synaptic or more recent versions of software. 

 To start off I am including some links to some popular Linux web sources. Also I would like to have lists of programs sourced from these places with discription and possibly some solutions to some simple problems.

 First:

 Sourcceforge is a huge repository of programs available for linux and is well known by most users.
[http://sourceforge.net/](http://sourceforge.net/)

 Another page which is not as well known is freshmeat, they have some very interesting packages there and that is where the gnome-sensor applet was originally found. Also they have a lot of applets there.
[http://freshmeat.net/](http://freshmeat.net/)

 Gnomfiles was one of the first Linux sources I originally found when looking for Linux programs. I like the page as the program description is a lot better than other pages.
[http://www.gnomefiles.org/app.php?soft_id=810](http://www.gnomefiles.org/app.php?soft_id=810)

 KDE-apps was a recomended page for KDE stuff with packages ranging from themes, Karamba apps, to right click scripts. I found it very usefull for KDE.
[http://www.gnomefiles.org/app.php?soft_id=810](http://www.gnomefiles.org/app.php?soft_id=810)

 Tucows: A very famous win download place which also seems to have some Linux packages but is linited in that.

 Wine the official wine homepage. Also has deb packages for the latest versions.
[http://www.winehq.com/](http://www.winehq.com/)

 The Linux Tome: A very intersting Linux game page with good game descriptions. I found it very usefull for finding Linux games.
[http://www.happypenguin.org/news/](http://www.happypenguin.org/news/)

 Good hunting!
