### Post by ugm6hr on 2008-01-30
Not installed - but it looks like an Ubuntu Russian language pack is available online:
[http://packages.ubuntu.com/gutsy/translations/language-support-ru](http://packages.ubuntu.com/gutsy/translations/language-support-ru)
[http://packages.ubuntu.com/gutsy/translations/language-pack-ru](http://packages.ubuntu.com/gutsy/translations/language-pack-ru)

So, in theory, after installation, you should be able to install these (in Synaptic Package Manager - search for *language* and select:
language-pack-ru
language-support-ru

Russian is also listed in System > Administration > Language Support, so you should be able to select it there after installation in English.
