### Post by Cicatrix on 2007-04-24
You might also want to configure your keyboard to have the linux "compose" key somewhere. Go to system > preferences > keyboard >layout options and place the compose key somewhere, for example the windows key. The compose key might be what you are looking for, as it allows you to combine for example ' with c to give &#263;.

[http://en.wikipedia.org/wiki/Compose_key](http://en.wikipedia.org/wiki/Compose_key)
