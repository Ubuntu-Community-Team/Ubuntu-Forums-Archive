### Post by bodhi.zazen on 2006-09-04
No need to start a new thread. I know you are frustrated but it make it harder to help you in fact. Others have been "waiting" at the old thread.

In a terminal:
```
sudo /etc/init.d/gdm stop
```

This will drop you to a command line interface (be prepared).

Log in if needed.

Back up xorg.conf if you have not already.

Now run
```
sudo sh /home/your_user_name/Desktop/NVIDIA-Linux-x86-1.0-8774-pkg1.run
```

This will install and configure X (xorg.conf)

Now:
```
sudo gdm
```

If that does not work:
copy this new xorg.conf to /ect/X11/xorg.conf.Nvidia
restore xorg.conf from backup

Then:
Exactly what monitor do you have? it is not all that hare to find the specs even on older monitors, I can do it for you but need to know the monitor.

Also, are you single or dual monitoring?
