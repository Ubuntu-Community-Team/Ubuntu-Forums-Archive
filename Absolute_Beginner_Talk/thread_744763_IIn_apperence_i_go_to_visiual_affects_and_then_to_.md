### Post by InfinityCircuit on 2008-04-03
Try adding this to the end of /etc/X11/xorg.conf (type gksudo gedit /etc/X11/xorg.conf)

```
Section "Extensions"
        Option  "Composite"     "on"
EndSection
```

Then restart the X server by logging out and back in again.

### Post by JoshuaRL on 2008-04-05
Okay, looks like you have the proprietary "fglrx" driver installed.  You'll need GLX to get compiz running on that card.  Now, if you decide to do that it will break direct rendering.  So when you try:
```

glxinfo | grep direct

```
It will return "no".  Some 3D games will have problems, since they rely on direct rendering.

But if you know all that and still want Compiz some kinda bad, try:
```

sudo apt-get install xserver-xgl

```
Then reboot and log in.  You should be able to use Compiz.
