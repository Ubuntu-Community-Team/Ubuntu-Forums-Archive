### Post by Nano Geek on 2006-04-25
The icon packs should put new icons on the panel automatic. Try looking through the **Applications** menu, find a program that has a new icon, right-click it, and press **Add this launcher to panel**. See if it still has the new icon. It could be that the icon sets that you picked don't have new icons for the programs on your panel. A good website to find GNOME art is [art.gnome.org]("http://art.gnome.org"), they have some great stuff there. To install a theme, icon set, or a window border, just drag the link into the **Theme Preferences** window.
