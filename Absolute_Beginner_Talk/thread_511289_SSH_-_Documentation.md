### Post by rich.bradshaw on 2007-07-27
It's so easy, you don't even need any!

sudo apt-get install ssh

Will install the server.

Then 

ssh USER@IPADDRESSOFCOMPUTERYOUJUSTINSTALLEDITON!

Where USER is the username on the computer you just installed it on.

Done!
