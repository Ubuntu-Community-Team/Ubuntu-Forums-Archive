### Post by timczer on 2006-07-15
There are several types of wikis that can be setup as "desktop wikis".  They run only on your system, not on an external server.  Take a look here: [http://en.wikipedia.org/wiki/List_of_wiki_software]("http://en.wikipedia.org/wiki/List_of_wiki_software")

Also, there is a project called newton that is a desktop notetaking, wiki-like system that I used on my breezy install.  There is a howto for installing it in dapper: [http://www.ubuntuforums.org/showthread.php?t=185972&highlight=newton]("http://www.ubuntuforums.org/showthread.php?t=185972&highlight=newton")
