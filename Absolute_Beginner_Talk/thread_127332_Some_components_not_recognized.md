### Post by Pragmatist on 2006-02-09
I don't think it should matter.  If you really want to check, you can look in the files in the /proc directory.  Even better, install KDE.  You should easily be able to install it using synaptic.  When you get to your login screen after you bootup, pick KDE from the "sessions" menu.  Then go to the main "K" menu and then to Systems menu and then pick "Info Center".  You probably could also just install KInfoCenter and run it in GNOME if you want.  To find any of these in synaptic, just use synaptic's search feature.
