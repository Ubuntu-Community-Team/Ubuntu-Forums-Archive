### Post by atrus123 on 2006-10-23
Delays give developers and testers time to test the package before it finds its way into the stable repositories.  Ubuntu users know all too well what happens when a package goes live before it has been properly tested.  Firefox 2 could have security holes as well.  In other words, it's too early to trust it entirely.
