### Post by deweese on 2006-10-11
for gnome use: sudo aptitude install gnome desktop
for kde use: sudo aptitude install kde desktop

Also: dual boot implies you have two different operating systems
on a hard drive. Each installation is on a different partition.
Having a choice of xubuntu, kde, or gnome and even fluxbox isn't
considered a dual boot or even a triple or quad because those desktops
have all been booted up from the same kernel, when you choose SESSION.
