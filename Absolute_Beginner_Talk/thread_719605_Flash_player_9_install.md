### Post by aysiu on 2008-03-09
> **Le 3eme Oeil said:**
> I'm using Ubuntu 7.10 gutsy gibbon thats all i know. 

Maybe i was just hitting ALT+F2 at the wrong time, when should i hit it?
With the thumb of your left hand, press the Alt key. Hold it down. Do not let it go right now.

Then, with the index finger of your right hand, press the F2 key (you do not need to hold down this key)

That should do it.

Once the Run dialogue appears, you can let go of the Alt key. Having the Run dialogue isn't necessary for installing Flash (you could use the terminal to run commands), but this could be a symptom of something wrong with your installation if you can't do Alt-F2; and if you can but just don't know how, this is a good thing to know how to do for the future.
