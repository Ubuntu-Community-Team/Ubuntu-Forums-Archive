You may not believe us, but it's not that hard - just **different**. There are many concepts that you think you understand because you're used to a particular way of things, and then when something doesn't work the way that you think it should it can be quite unsettling. Linux is actually **harder** to understand for the more experienced Windows user than for a complete computer newbie, since you have to unlearn quite a lot of stuff before you get used to the differences.

Maybe you'll try a different Linux distribution at some point in the future, and hopefully that will work better for you.

The concept of a Live CD is, IMO, brilliant. It means that you can get your feet wet with a new operating system without worrying too much about hardware, and whether your existing data will be safe, and so on. Perhaps you could try one of those to avoid the installation teething troubles that you've so far experienced?
