### Post by punx45 on 2007-06-24
this is in KDE 3.5   i dont know what the current version is for Kubuntu, but im sure it is fairly similar whatever it is.

right click on the taskbar

in the properties menu go to the taskbar section

uncheck "show windows from all desktops"
