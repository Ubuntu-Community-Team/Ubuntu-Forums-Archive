### Post by robins_web on 2006-09-07
With very rare exceptions, you don't need to defrag Linux, since it doesn't get fragmented to begin with.

Windows, on the other hand, has its file system rooted in DOS, which doesn't handle files in a very intelligent manner; hence the frequent need for defragging.
