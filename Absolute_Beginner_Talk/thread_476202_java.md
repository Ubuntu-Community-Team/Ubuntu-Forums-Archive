### Post by Gwasanaethau on 2007-06-17
Depending on what version you want:
```
sudo apt-get install sun-java6-jre
```
will install version 6, or:
```
sudo apt-get install sun-java5-jre
```
will install version 5.

If you want to install the plugin for firefox too (can be quite useful when surfing the 'net), you can simply add 'sun-java6-plugin' or 'sun-java5-plugin' to the end of the appropriate command above.
