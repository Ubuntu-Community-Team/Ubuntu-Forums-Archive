### Post by cilynx on 2006-07-26
Hey,

GRUB is your bootloader.  It's the first thing you see after your computer's BIOS is done doing it's thing.  If you have an image for GRUB, it will show up behind the menu where you pick which kernel and/or operating system you want to boot into.

You can have an image on the screen as Linux is booting so that you have something to look at besides all that text.  This is generally known as your Boot Splash and less commonly, Console or Kernel Splash.

The screen where you log into your desktop is GDM and you can configure the look of that as well.

The screen with pretty icons and crap right before desktop load is the Gnome Splash Screen.  

Metacity is your window manager.  It's what lets you resize windows and move them around on the screen.  Theme-wise, you are correct, Metacity handles window borders.

GTK is the toolkit used to define application controls.  It goes a lot deeper than that, but for the current discussion, that'll do.

Hope this helps---
