### Post by Peter09 on 2008-03-10
Ok few questions how do I Autostart Berly on boot and Nvidia Settings (for screen resoloution)?
[I]
Screen Resolution is set within xorg.conf and will happen on boot further tweaks can be done from the menus 

 System->preferences->screen resolution[/I]

2nd is it possible to have 1 taskbar on the bottom with all the same stuff?
[I]
You can move the panel from top to bottom by left clicking in empty space on the panel and selecting properties. You can also add additional panels[/I]

3rd I cant write or something like that to some of my HD, so I cant edit the Xorg thingy?
[I]
You most probably are not using the sudo command to go into admin mode.
I suggest you search for more information on setting up Xorg before doing to much more, corrupt that and you will lose your GUI.[/I]
