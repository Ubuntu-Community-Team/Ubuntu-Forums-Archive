### Post by Rocket2DMn on 2008-01-14
Sounds like youre using the wrong keyboard configuration.  Go to System->Preferences->Keyboard and select the Layouts tab and choose one of the Generic keyboard layouts (I have the 105 key (Intl) ).
What types of plugins are you looking for?  Firefox addons?  You would handle those through firefox, but most anything else is just a matter of installing programs from the repositories using Synpatic.
