### Post by smegmaster on 2006-07-03
Is there a way to get Ubuntu to display multiple windows of the same program next to each other on the task bar, like in Windows XP?

---

### Post by Jucato on 2006-07-03
There's a very small, almost invisible, vertical bar at the left end of the task bar/window list, just to the right of the "show desktop" icon. Right-click on that bar and select Preferences. You will have a dialog box where you can choose the settings that you want.
