### Post by CroEragon on 2007-01-21
Wine is automatically assigned to exe and msi windows installation files. So when you try to open them it will use wine. Then you have to install programs with its installation, and then program will be found in applications->wine menu. Also, i suggest to read as many tutorials before using it, rare are programs that work out of box with wine.
