### Post by mac.ryan on 2007-04-10
Well, it depends what you wanted to do with that .exe file...

If you were trying to run a program (like under windows), then I am afraid it won't possible that easily, as GNU/Linux have different executables calle ".bin".

If you can't find alternate solutions to that (i.e. installing the GNU/Linux version of that software, or using an alternate option under GNU/Linux) you might try to install **wine**. This is an interface between the GNU/Linux system and the windows programs that allows various windows programs to work under GNU/Linux.
