### Post by Fully216 on 2006-08-30
This might be a bit more information than you wanted, but here goes ...

The name Alt + Gr stands for Alternate Graphics (or Alternative Grafiken).  The original intended purpose of it was to have an easy way to get to table-based graphical characters that are useful in a console applications, located on the right side of the spacebar (the right Alt key in English keyboards).

Alt + Gr is actively used for keyboards that needed extra keys. This does not apply to the English keyboard hardware, so the equivalent is using the right Alt key, which will actually act as if it were the Alt + Gr key any time you switch to a layout that makes use of this extra shift state.  

I did not know this until someone pointed it out; this extra shift state is also available by hitting Ctrl + Alt, but thats more work to type.  I have changed my keyboard shortcuts so that Ctrl + Alt serves a different purpose, which you may do under your prefrences.  So having a single key to type instead is much more convienent aside from the fact many shortcuts are assigned to Ctrl + Alt.

In any case, I hope this helps! ;)
