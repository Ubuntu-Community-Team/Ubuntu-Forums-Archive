### Post by taurus on 2008-02-12
Install build-essential package first and then run that command again.

```
sudo apt-get update
sudo apt-get install build-essential
./configure
```

### Post by vikrant82 on 2008-02-12
It might still crib for more missing development lbraries. 

The easiest thing is to install, "auto-apt"

and do sthing like 
sudo auto-apt run ./configure

It should install all needed libraries.
