### Post by %hMa@?b&lt;C on 2007-05-26
Ubuntu is an operating using the kernel, Linux.  OS X is an operating system using the kernel, Darwin.  Darwin is based upon the BSD kernel which is based on UNIX.  Linux is also based upon UNIX.  So kernel Linux and kernel Darwin are similar, the operating systems are not that same, but they are similar.  OS X is much more "dumbed down"
Canonical is the company that funds Ubuntu development.  Kernel Linux is just the kernel.  We could just as easy use kernel Solaris or HURD, but Linux is much more advanced (better! :lolflag:) than the others.

Yes the disks should be bootable!
