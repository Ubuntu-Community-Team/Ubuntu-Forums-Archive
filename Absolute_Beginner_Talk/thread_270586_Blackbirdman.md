### Post by skymt on 2006-10-03
I don't think there's any Linux-native program anything like Painter. GIMP is a pretty good Photoshop clone, and Inkscape is a lot like Freehand, but there's no real "painting" program. You may be able to run the Painter version you have through [Wine](http://winehq.org/), though.

### Post by Josh1 on 2006-10-03
Hi,
According to this ([http://appdb.winehq.org/appview.php?iVersionId=1697)](http://appdb.winehq.org/appview.php?iVersionId=1697)), apparently Painter5 works with WINE.
To install wine:
1) Load up terminal
2) Type in "apt-get install wine"
3) When that is all done, just type in "wine /home/yourname/painter5install.exe" or whatever the location is of the installer.

Goodluck,
Josh
