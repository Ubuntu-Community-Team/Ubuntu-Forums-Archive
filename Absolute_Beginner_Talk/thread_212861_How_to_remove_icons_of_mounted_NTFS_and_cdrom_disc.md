### Post by crossedout on 2006-07-10
You can go to Applications --> System Tools --> Configuration Editor
Expand apps --> nautilus --> desktop and uncheck the items you wish to be hidden (not visible on desktop).

If you don't have the System Tools --> Configuration Editor listed, you will have to make it visible by choosing Applications --> Accessories --> Alacarte Menu Editor.  Once in there, choose System Tools then check Configuration Editor.  Exit and it will be available under Applications --> System Tools.
