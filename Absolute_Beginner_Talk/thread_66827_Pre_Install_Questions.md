### Post by bsussman on 2005-09-18
Go for it!

> 1) Will this machine work ok with Ubuntu Linux? 

It will work.  I await your opinion on speed and responsiveness until you build it.

 >  2) If I replace the drive with a new one (without an OS on it) will the Ubuntu install still be able to configure the hardware on the machine? 

Yes, provided the hardware (bios) can see it ubuntu should not have problems.

 >  3) Do I need to format the new drive before the Linux install?  

No - ubuntu should prompt you through this.

 >  4) I do have a Dlink wireless card on the machine. How difficult is it to get this up and running after the install?

I cannot comment - not enough experience.  I hear WIFI cards can be a challenge.

 >  5) I am a big Quicken user. Is there an equivilent or close relative in the Linux world? 

Probably not one you will be thrilled with. Using google 'linux quicken replace' gave me reading to do. GnuCash comes up (free, their hearts are in the right place, doesn't cost money, might do the trick for you, is very cheap and requires some study.

BTW - dual boot is not terrible, except that you will always want to be in ubuntu anyhow :-P
