### Post by Taboo Mirage on 2008-01-18
In case you're referring to which architecture you should download, I believe the CPU chip in that particular notebook is the Intel Core 2 Duo.  This is a 64-bit architecture chip, therefore you can either download the i386 version which will run in 32-bit mode, or the AMD64 version which is the 64-bit edition.

A 64-bit operating system will enhance your performance in some cases and I recommend utilizing that architecture as your chip supports it.  The only downside is that you may experience some minor compatibility issues with some things, although these days 64-bit computing is very widely supported.

Take care.
