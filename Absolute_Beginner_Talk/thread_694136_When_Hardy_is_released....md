### Post by Rocket2DMn on 2008-02-11
There is a bulit in process for upgrading.  You can either do it visually or through the terminal with 
```
sudo apt-get dist-upgrade
```
See [https://help.ubuntu.com/community/SynapticHowto#head-b967bae359f1edea4183799c440b5abe2119426a](https://help.ubuntu.com/community/SynapticHowto#head-b967bae359f1edea4183799c440b5abe2119426a)
