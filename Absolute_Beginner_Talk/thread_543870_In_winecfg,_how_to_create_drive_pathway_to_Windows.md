### Post by swarup on 2007-09-05
When I open winecfg and click "Add" in the drive window, then click on "browse", there in the browser window under /media there is only /media/cdrom and /media/cdrom0. Why doesn't "/media/disk" appear there?

How can I get this /media/disk/ pathway added to the drive in winecfg so I can then access the Windows partition using winefile?

---

### Post by mocoloco on 2007-09-05
Make sure the drive is mounted; if it is you should see it on your desktop or under the places menu. If not go to "Computer" and double click on it.  Then run winecfg again.
