### Post by davmac on 2005-10-02
Make sure your bios boots from CD before the hard-disk, pop in the CD and reboot.

Follow the prompts you're given. The main area to be careful about is the partitioning. MAKE SURE YOU DON'T ALLOW IT TO WIPE THE WHOLE DISK!

I'd suggest that you let the install automatically partition the area you've freed up for it.

Once the install competes, have a look at [http://ubuntuguide.org/]("http://ubuntuguide.org/") for help on getting started with the system.

...and don't forget - there's always someone here to help out!
