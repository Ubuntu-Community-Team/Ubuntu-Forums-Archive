Hello Adam,

Download the .iso and create your installation cd. 

Once you have the installation media created, insert it into your cd drive and boot your system. Be sure to have your bios set to allow booting from your cd drive. When the menu of options appears, select the option to verify your media. This will check to make sure your installation media doesn't have any corrupted file data.

Once you are sure you have a validated installation cd, boot the cd select the first option from the menu that says to start or install Ubuntu.

After a short wait, you will see a live Ubuntu Linux desktop. This is a work version of Ubuntu Linux which is running from the CD. No changes have been made yet to your system.

When you are ready to install Ubuntu Linux on your system, you can double click the icon on the desktop that says "Install".

This will then start the installation process for you. It will ask you a few questions along the way, such as for a user account, and the type of partitioning you want to use.

It is a fairly simply installation process. If you have questions, be sure to ask.
