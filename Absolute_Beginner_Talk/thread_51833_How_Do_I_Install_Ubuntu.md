### Post by Kvark on 2005-07-25
Check the md5 checksum of the .iso file and compare to the md5 checksums found [here](http://releases.ubuntu.com/5.04/MD5SUMS) to make sure you got the correct file.

To check the md5 sum with linux, just type "md5sum file.iso" in a command terminal. To check it with windows, you need a special checksum program, how you do it depends on which program you use.
