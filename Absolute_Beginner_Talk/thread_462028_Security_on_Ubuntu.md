### Post by ruscoe on 2007-06-02
All your ports are closed until they're needed by default on Ubuntu, generally negating the need for a firewall.

There is anti-virus software for Linux, but it is typically used to filter email before passing it on to Windows PCs on a network. It can be handy on a PC to prevent you accidentally emailing a virus to your Windows-using friends, but isn't required for your own protection.

I've heard good things about ClamAV, if you want a virus scanner.

### Post by p_quarles on 2007-06-02
Like ruscoe says, Ubuntu's port policy acts like a built-in firewall. If you want added functionality for this, you can use the Firestarter GUI, available in the repositories.
