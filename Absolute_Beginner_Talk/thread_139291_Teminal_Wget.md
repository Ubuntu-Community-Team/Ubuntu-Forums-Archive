### Post by celloandy on 2006-03-03
Wget saves things in the current directory.  To save them some place other than your home, use the "cd" command to change to a different directory before running wget.

### Post by darth_vector on 2006-03-03
from the wget man page> -P prefix
       --directory-prefix=prefix
           Set directory prefix to prefix.  The directory prefix is the directory where all other files and subdirectories will be saved to, i.e. the top of the retrieval tree.  The default is . (the current directory).
