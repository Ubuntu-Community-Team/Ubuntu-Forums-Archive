### Post by AnRuaRi on 2006-04-30
make sure you're using the most up to date version of open office. I believe this is version 2.0.2

this will save files into MS word format.
there will be changes in what a page looks like if you use complex formatting and inserted objects. this will hapen whenever you change between applications.

Microsoft are refusing to offer mainstream support for Open Format. it's a challenge to their monopoly and wont offer it until they're forced to. Open format is based on XML. the hignest professional levles of office do offer support forXML but not very well.

If you want other people to be able to read your documents and have them display properly, then save them in PDF format. This is essentially a "Print Preview" which can be passed from platform to platform. Unfortunately the recepient wont be able to edit it if that's what you require.

As an alternative, you could save your documents in HTML format which both Word and OpenOffice support. Just dont expect the html to be particularly well written.

### Post by adamkane on 2006-05-27
This thread got way off track fast.

OpenOffice can read .doc files fine. It can also save files in .doc format automatically, but you to enable this is the Options:

OpenOffice Writer -> Options -> Load/Save -> General -> Default File Format

Don't forget to disable Java to speed up OpenOffice:

OpenOffice Writer -> Options -> OpenOffice.org -> Java -> [Uncheck] Use a Java RE
