### Post by ShogunMaster06 on 2007-10-31
I've been doing this for a while with my setup.  I assume that the printer is setup on your desktop.  Go to your printer properties (System -> Administration -> Printers).  Right-click your printer and choose properties.  You didn't say if you were using Gutsy or Feisty (I think it's a little different on Feisty from what I remember.)

In Gutsy, there is a connection tab where you can choose Network Printer and Windows Printer (SMB).  Input your host's IP address, printer name and password.  You should be all set then.  It is very similar in Feisty.
