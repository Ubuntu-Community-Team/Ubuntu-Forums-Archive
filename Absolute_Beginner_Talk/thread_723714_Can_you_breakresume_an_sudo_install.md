### Post by PmDematagoda on 2008-03-13
> **BlizzofOZ said:**
> Currently installing xbuntu-desktop... 

Is there a way to break the install and resume it later?

If it is only downloading the packages then stopping the process would be ok, but if it is installing the packages then do no stop it because if you do then you may be faced with a broken OS afterwards.

But in general it is recommended that you do not interrupt a package management process anyway.
