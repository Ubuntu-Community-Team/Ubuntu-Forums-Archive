### Post by nicedude on 2008-04-19
There are 2 ways to install the newest driver for your Nvidia card.

1. Know how to compile it from the Nvidia source from their website, which is pretty complicated affair

2. Download a free program called envy from the following website which will detect your graphics card and download the driver from Nvidia and then it will compile it and install it. It does all this automatically. You can find it here [http://albertomilone.com/nvidia_scripts1.html](http://albertomilone.com/nvidia_scripts1.html)

Although the Ubuntu staff doesn't support or recommend using envy it has worked just fine for me and gave me a newer driver than the one from Ubuntu and all without having to compile anything.

You just download it to your desktop from the authors site -> then doubleclick to install it -> after installation you can access it on the top toolbar tab Applications -> System Tools -> Envy. Once started just click install Nvidia driver and it will check your Nvidia model then download the driver from Nvidia and finally it will compile and install it.

Hope this works for you as well as it did for me.  :-)
