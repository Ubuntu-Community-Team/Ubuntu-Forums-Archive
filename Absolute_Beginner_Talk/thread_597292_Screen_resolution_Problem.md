### Post by skyjacker on 2007-10-30
Try this procedure:

1) Back up the present xorg.conf
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf-backup

2) Stop gdm
sudo /etc/init.d/gdm stop (or kdm for kde)	

3) Edit xorg.conf
sudo dpkg-reconfigure xserver-xorg

4) Select the resolutions you want available by arrowing down to each desired line and pressing the “space bar” to place an X to the left.

5) Validate your choices with “enter”

6) restart X with Control+Alt+Backspace

7) Choose the desired resolution under the “Systems/Preferences/Screen Resolutions drop down    list

Good Luck!
