### Post by orb9220 on 2006-09-24
You can look into them by renaming the .cbr to .zip because that's all cbr is. And then look to see if the pages are there.

Also right click a cbr file and check properties on which app is the default app to open in.

Another reader is [http://comical.sourceforge.net/](http://comical.sourceforge.net/)

### Post by HerrEkberg on 2006-09-25
A .cbr file is a RAR archive, a .cbz is a Zip archive. You need [unrar](http://packages.ubuntu.com/dapper/utils/unrar) installed in order to view RAR (.cbr) files with Comix.
