### Post by Tomosaur on 2006-08-01
The great thing about linux is that you can learn pretty much anything either from within linux itself or with a cursory glance at the internet. Here's some helpful tips:

In the terminal, double tab the 'tab' key, then press y. This will show you all of the commands available to you.

To find out about a command, type 'man' followed by the command, for example 'man ls' will tell you all about the 'ls' command.

Type 'printenv' to see your environmental variables, most importantly, imo, the 'PATH' variable. These are all the addresses linux looks at when you type a command. This should give you some idea about where programs live.

Here's a list of the built in [bash commands](http://www.ss64.com/bash/).

And here's a crude map of the [Linux directory structure](http://www.comptechdoc.org/os/linux/usersguide/linux_ugfilestruct.html)

That should get you started at least.
