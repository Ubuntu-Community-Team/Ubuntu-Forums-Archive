After you download and burn the iso you just install it new.  It is just as easy to install as Ubuntu.  No special instructions needed.  If you installed ubuntu you can install this one.  You won't be upgrading, you will be doing a new install; you can over write your existing installation if you want.

Remember too, it is not an official release.  It is not version 7.04 either, it is 6.10.  It is just regular Ubuntu with lots of additional programs added and configured for you.
