### Post by red_Marvin on 2008-03-11
This is a known problem with flash, as far as I've heard firefox 3 will circumvent/fix the crashes when it comes out.
In the meantime, if you want to kill flash without killing firefox you can open a terminal and type *killall npviewer.bin*.
