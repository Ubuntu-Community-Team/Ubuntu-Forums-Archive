### Post by Sparkster185 on 2007-05-22
If it's in your start menu, you should be able to right-click it and go to "Edit", or something like that.  That should tell you which icon is used as well as the command used to launch the application (with option flags, etc.).  If the command isn't a full path, you can aways use 'which' to get the full path to an application.
