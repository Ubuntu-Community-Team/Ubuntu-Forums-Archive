### Post by Bloch on 2006-10-14
It's hard to find precise info on system requirements because ubuntu is do customisable.
However your specs are plenty enough for standard ubuntu.
I have the 2.4GHz Celeron with 128MB cache (the cheaper 2.4GHz Celeron), a 64MB Nvidia video card, 40GB hard disk and 512MB ram.

You will notice things run smoother with the extra ram if you are running OpenOffice plus a few other things.

Any other info? Yes. Use Automatix to install all the non-free codecs, MS fonts, realplayer, flash, java.

(personal opinion) Install stuff using the "Add applications" menu iitem. Don't get sucked down the road of compiling stuff. It's not for a beginner.
