### Post by neurostu on 2008-02-29
The reason you screen goes black is b/c you have an nvidia video card. restart your machine with the CD then select the "Start Ubuntu in Safe Graphics Mode" or something like that.  It will then boot with the VESA drivers. The cd isn't big enough to have all the drivers for every video card so it can't properly run your video card, but once you are done installing ubuntu will prompt you to install the nvidia drivers (restricted driver, just say yes)
