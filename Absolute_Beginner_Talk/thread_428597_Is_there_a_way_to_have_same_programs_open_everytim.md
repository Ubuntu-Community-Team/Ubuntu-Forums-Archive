### Post by starcraft.man on 2007-04-30
As long as the program can be triggered via a terminal command, all you need to do is go in:

System > Preferences > Sessions.

Go under startup programs, push New. Type the name of the program in the top, the command should be in the bottom the same as you type in terminal. Then it will start every time you reboot.
