### Post by deadgobby on 2007-08-29
Creative Labs Optical Digital I/O Card 	CT4710 piggy backs to the Sound Blaster Live! Value card

[http://alive.singnet.com.sg/features/hoontech/io1.htm](http://alive.singnet.com.sg/features/hoontech/io1.htm).

In turn SB live is supported by Linux or ALSA.
 
If you want to see if a linux distro reads the card fine and dandy. Run the Linux distro as a live CD . If you get sound. You are good to go.
