### Post by Sef on 2006-05-05
A live cd loses it settings after closing. 

1)  If you want to install Breezy, the current stable, you need to download the install disk.

[http://www.ubuntu.com/download]("http://www.ubuntu.com/download")

2) If you want to install Dapper, the current Beta, download from here.

[http://cdimage.ubuntu.com/daily/current/]("http://cdimage.ubuntu.com/daily/current/")

Note: Beta means it is in testing, and you may experience problems that shouldn't exist in the stable version.  I am using Dapper Beta2, and find it very stable. However, your experience may vary.
