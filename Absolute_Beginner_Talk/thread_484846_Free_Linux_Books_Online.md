to view chm files, I went to google.com and typed the following in the search box:

site:ubuntuforums.org chm  

clicked on search (or hit enter key)                   

and got the following link as top result:
[http://ubuntuforums.org/showthread.php?t=501094](http://ubuntuforums.org/showthread.php?t=501094)

it's very well explained about the packages to view chm. I'm using xchm, I haven't tried
the others yet. 

what I've been doing is make a folder(directory) in my home folder(directory), drag and drop
(move or copy) the compressed file into the folder, and right clicking the file and choose
extract files here. 
some of the stubborn compressed files, I've had to use the terminal (applications > accessories >terminal) cd to the folder(directory) and type:
unrar filename.rar
then the message says to...use e like this
unrar e filename.rar
replacing filename above with the actual filename. I had to install the unrar package on one occassion back aways. Not sure if unrar 
is installed by default on feisty.
