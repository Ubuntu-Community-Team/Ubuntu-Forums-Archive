### Post by Obor on 2007-03-21
Generally all the packages that are in the official repositories can be installed through Synaptic.
Applications->System->Synaptic Package Manager
Just search for screensaver and it should list available packages. Then just right click, mark for installation and apply

If you haven't done so already check out the [Xubuntu Desktop Guide]("https://help.ubuntu.com/xubuntu/desktopguide/C/index.html")
