### Post by Blofeld on 2007-05-06
Hi, there have been a couple of postings by Ubuntu users such as I trying to find a solution to the following problem:
When trying to listen to the BBC radio stream ([http://www.bbc.co.uk/radio/aod/index.shtml](http://www.bbc.co.uk/radio/aod/index.shtml)) in Firefox and with Realplayer installed, they were confronted with this error message:  "could not find the appropriate hxplay or realplay in the system path to use as an embedded system".
When I was then trying to sort out the problem I found that in "Edit > Settings > File types" .ram files were not listed and that I could not add and link them to /usr/bin/realplay
The solution was to simply install the "MIME Edit" add-on to Firefox.
:popcorn:
[https://addons.mozilla.org/de/firefox/addon/4498](https://addons.mozilla.org/de/firefox/addon/4498)
