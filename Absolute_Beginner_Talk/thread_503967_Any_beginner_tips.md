### Post by steve8track on 2007-07-18
I would recommend some of the following things as worth while to get going in linux:

Beryl - Most consider it just to make Ubuntu look great, but you can also set up key shortcuts to just about anything, such as opening a program, or changing a desktop background.  I have the Windows Key + Up maximize the current window, and Windows Key + Down unmaximizes.  I also set up a shortcut to open a internet browser, a text editor, and a console.  Also, I have it set up so I can change desktop backgrounds by running a script with a press of a button combination to go to the next desktop image, or another combination to go back a desktop image.  Not only does beryl look great, but you can use the mouse less through setting up shortcuts.

Epiphany - great web browser as an alterative to Firefox.  If you download one of the plugins for it, you can have tabs open on the left side instead of the top of the browser, so you can see more tabs at once.

Wine - for the games/programs that you can't live without and that you can't seem to find on linux.  I suggest searching first for a similar thing in Ubuntu, but I use Wine for Warcraft 2 :-)

Tilda - a fun terminal that resembles a terminal that a first person shooter uses

Amarok - GREAT music player with great shortcuts to run from anywhere

Nexuiz - A GREAT first person shooter that you can even play with your Windows and Apple friends.

Well, those are the first ones that come to mind.  If you are interested in web development, Ubuntu is great for that too, like php, perl, etc.
