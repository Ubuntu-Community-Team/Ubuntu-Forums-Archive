### Post by phil90 on 2007-10-14
The Root password is randomly created.


If you want to change it and you have access to sudo
 

Open a terminal window and type sudo passwd

and then enter your new password.


Still if you want to go in root account you won't have access


Ok to enable the root access you'll have to go to system administration 

In Security tabs select  Allow local system administrator login


Now you have access to root.



Just to let you know it's might decrease your security level to allow root access
