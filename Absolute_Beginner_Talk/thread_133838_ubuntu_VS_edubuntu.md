### Post by az on 2006-02-21
Install ubuntu on the box.

Use it for a day.

Then install the edubuntu-desktop package.  Edubuntu is built on top of ubuntu, so you can have both at the same time.  It's just a few extra packages and a different look.

To revert back to the ubuntu look, remove the edubuntu artwork package.
