### Post by jordanmthomas on 2006-10-20
Find an exe and right click on it.
Go to Properties.
Go to the Open With tab.
If wine isn't listed here, click Add
Using the custom command box, just put wine as the command.
Click Ok / close in all the open windows

Presto!

Hope this helps
