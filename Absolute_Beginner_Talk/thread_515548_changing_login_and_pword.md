### Post by 505 on 2007-08-02
1. To change your username/password, goto System, Administration, Users and Groups. Enter your password when asked. Select your user account and press edit.

2. First of all, is see no real need for the install CD. I installed Ubuntu once, and since then only did upgraded when new version came out. These are automated, and directly downloaded from the internet. So, you don't need a CD for that. You only need a CD if you want to completely reinstall Ubuntu. That said, if you still want a CD, goto the Ubuntu download page and download the file. This is a ISO file, so you need to burn it as an ISO image. I believe you can just right-click the file and select 'make disc' or 'burn' or something. Just don't use the new disc wizard when you insert a blank cdr.
