### Post by useResa on 2007-05-11
I do not know which version of XFCE you are using.
In XFCE 4.4 the calendar application is Orage of which you can set the preference by using the XFCE Settings Manager. It allows you to set a Calendar Applet in your taskbar.

I know it is not exactly the same as getting the calendar from clicking on the clock, but AFAIK it is the closest you can get.

Good luck and I hope this somewhat answers your question.
