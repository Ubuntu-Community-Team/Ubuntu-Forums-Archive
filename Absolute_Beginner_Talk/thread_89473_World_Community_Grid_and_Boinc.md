### Post by alexanderfrey on 2006-01-04
Hi Kisan,

first you have to download the boinc client [http://einstein.aei.mpg.de/download/boinc/dl/boinc_5.2.13_i686-pc-linux-gnu.sh](http://einstein.aei.mpg.de/download/boinc/dl/boinc_5.2.13_i686-pc-linux-gnu.sh)

from here: [http://boinc.berkeley.edu/download.php](http://boinc.berkeley.edu/download.php)

Probably it needs to be set executable in the terminal with  chmod a+x boinc_5.2.13_i686-pc-linux-gnu.sh

Then start it with ./boinc_5.2.13_i686-pc-linux-gnu.sh
The program will ask you for a key you will get by signing into the wcg at [http://www.worldcommunitygrid.org/](http://www.worldcommunitygrid.org/)
The key will be sent to you by email. Enter it in the boinc client and you're done !

Don't forget to join the ubuntu team ;)

Alexander
