### Post by candtalan on 2007-10-26
Welcome!

Your request does not sound silly, do not worry.

There are several ways of going about installing things into linux distributions, and unless you are looking for a deep technical and interesting experience, the way to choose is the one favoured and tailored for the distribution you are using.

In ubuntu this is 
'Add/Remove programs' I think this is in the Applications menu. 
This will install from choices in the ubuntu repositories. It is expected that the machine is connected to the web.

At a lower level than full applications, there are 'packages' including libraries. These can be easily chosen and installed using the synaptic package manager - in the administration menu I think.

I have rarely needed to do anything else than use these. Be reassured that contrary to what you may have read in the past, distributions such as ubuntu are very easy indeed to manage. Once you are more confident you will find answers to most things with searches, here or on the web of course.

Having said that, this is open source, and there is nothing to stop you starting from source code and compiling and installing in the time honoured way. Some apps are available pre packaged to an extent. The red hat approach uses .rpm (red hat package manager) and the debian approach (Ubuntu uses this) uses .deb packaging.
