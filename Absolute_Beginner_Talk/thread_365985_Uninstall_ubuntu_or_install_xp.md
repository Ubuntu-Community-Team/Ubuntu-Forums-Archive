### Post by BarfBag on 2007-02-20
You have to go into Bios and change the boot order.  Move the CD drive to the front and the hard drive second.  Bios tends to be different on every machine, so I can't give you instructions more specific.

To get into Bios, hold down delete as it's booting.  If that doesn't work, try it over and over again with each of the F keys.  Again, sorry for not being able to be more specific.
