### Post by RudolfMDLT on 2007-12-27
Okay, you can either burn a DATA cd and add the ISO image or you can burn the image. a mistake a lot of people make is that they burn a DATA CD with the ISO on it. An ISO image is file containing the files and the CD structure. In nero select the option that says burn disc image or iso image. Or rightclick on the iso and somewhere there should be an option on writing the image to disc using Nero.
