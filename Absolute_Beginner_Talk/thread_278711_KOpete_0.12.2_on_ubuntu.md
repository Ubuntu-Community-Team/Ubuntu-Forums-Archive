### Post by seijuro on 2006-10-17
if you use apt-get from the terminal it should install the required libraries to run kopete automatically w/o installing any extras that are not needed. Personally if you want Kopete and other Kde goodies I would switch to Kubuntu rather than running kde-desktop on ubuntu mostly because it will reduce the number of updates you need to download.
