### Post by kidders on 2007-04-11
Hi there,

Unless you particularly like kiba-dock's physics engine (personally I find it irritating), I would suggest compiling the CVS version of Avant Window Navigator. It's not exactly a picture of stability either, but (at least for me) it works reasonably well (once you get to know it well enough *not* to do certain things!).

In any case, many of these dock applications are very rough, so don't expect much!

**Edit:** Both applications require Beryl afaik.
