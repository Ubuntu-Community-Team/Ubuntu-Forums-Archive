### Post by zombiepig on 2007-05-23
wohoo... managed to fix it! i'll post my solution here in case anyone else comes across a similar problem :)

basically, i was mucking around with bookmarks in nautilus, and found there was a strange bookmark linking to a folder on my server - when i removed this bookmark, the waiting cursor problem fixed itself!
