### Post by viciouslime on 2007-05-25
ATI cards do have notoriously poor linux support, but you should still be able to get it to install. Have you checked your alternate disc is not corrupt? One of the first steps in the install asks if you would like to "verify" the cd (I think that's what it calls it) basically, it will check the md5sums of all the files against a list of what they should be, takes a while, but will ensure that a corrupt disc isn't the problem.
