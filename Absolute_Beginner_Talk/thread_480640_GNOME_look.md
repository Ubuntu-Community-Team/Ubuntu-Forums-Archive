### Post by qamelian on 2007-06-21
> **dhani99 said:**
> [http://www.gnome-look.org/content/sh...?content=26448](http://www.gnome-look.org/content/sh...?content=26448)

I just want to install this theme, i downloaded that theme as it is in tar.gz format i extracted and from that i dont know what to do!

You don't need to extract it. Just open System > Preferences > Theme.  Click the Install Theme button and navigate to the file you downloaded. Or even easier, just drag and drop it on the Theme Manager window anfter it opens. If it's a valid theme archive it will automatically install.
