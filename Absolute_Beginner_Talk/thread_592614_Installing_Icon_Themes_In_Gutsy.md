### Post by Can+~ on 2007-10-26
"Appearance" is now the general 'look' configuration in gnome, you can install about anything in there.

Open appearance, click on install, select the icon pack, open it, installed. Then you can edit the current skin to change the icons, or just about anything.

Alternative way: Drag and drop the file into the appearance window.
