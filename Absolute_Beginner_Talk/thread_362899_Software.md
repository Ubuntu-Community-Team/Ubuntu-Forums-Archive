### Post by Tomosaur on 2007-02-16
If the software is for Windows, then short answer is 'no'. If the software is for linux, then yes, there should be no problems.

Some windows programs can work on Linux. You need to install software called 'Wine' (it may already be installed). The homepage for Wine is [here](http://www.winehq.com). You can check if the software in question works [here](http://appdb.winehq.org/). If the software is not listed, it doesn't mean that it won't work, it just means nobody has submitted it to the website yet. If your try it out and it works, then by all means submit it to the second website so other people will know.

Generally speaking - it is best to just try and find a Linux equivalent for your Windows software. In cases where there are no acceptable equivalents, you can try Wine, which may or may not work.
