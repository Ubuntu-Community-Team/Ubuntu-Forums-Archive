### Post by dvarsam on 2006-03-10
You can do it with a script, but the best way is to do it this way:

1. Launch a Terminal window.
2. From the Menu, select "Edit\Current Profile"
3. Click on the Tab named "Colors"
4. Deselect the checkbox named "Use colors from System Theme"
5. Under "Background Color", select the color you would like...

Have fun.
