### Post by diplomat.x on 2008-04-05
I am somewhat newbie also...
But i do know there is no need to install a software to install GTK themes...
Just go to gnome-look.org website.

Select GTK2.x from left menu. Dowload any XYZ.tar.gz or XYZ.bz2 file to your desktop. Now right click on ur desktop and select change desktop background. From the tabs on top select THEME.
Now drag that bzr file  into the window and it will be installed...as simple as that, i have installed 100s like that..

Let us know if it solved ur problem. And there is no need to uninstall emerald whatsoever. And use compiz to edit effects. Emerald + what i told = Total looks customization.
