### Post by cogadh on 2007-06-24
Xubuntu uses the XFCE Window Manager while Ubuntu uses Gnome. Gnome is more resource intesive than XFCE, but it also has more administration-type GUI interfaces to manage your PC. Xubuntu is specifically for older or lower-end PCs that don't have the processor, memory or graphics to handle Ubuntu or Kubuntu (which is the same as Ubuntu, but with the KDE window manager).
