### Post by drguerra on 2007-11-01
Okay, so many of you, myself included have been having problems. We get a black screen when we try to install using the Live CD, or if we've already installed using the alternative CD, Ubuntu 7.10 won't boot up, instead the screen may flicker, may get stuck at booting sequence, and/or a black screen.

This is a visual display problem. The solution is quite simple. No need for complicated commands.

Simply download one of the alternative, text based installers. Run that, install it. When you are asked for your resolution, scroll to your resolution and hit the SPACE BAR, NOT ENTER!!!!! Why the space bar selects it and not enter is beyond me, but this is why. Hitting enter will select NO DEFAULT resolution.

So select the res you want using the spacebar, then hit enter. It seems ridiculous, but it's what fixed it for me.

So, goodluck and post here if it works for you, or if it doesn't
