### Post by akiro.yamamoto on 2006-03-27
To enable the trash icon on the desktop... use the Configuration Editor( found under Applications >>System Tools >> Configuration Editor.
Then click apps >> nautilus >> desktop , then put a check for the trash icon(see pic)
To enable the trash icon on the buttom panel, right click on the panel, select Add to Panel and then choose Trash (see pic)..
