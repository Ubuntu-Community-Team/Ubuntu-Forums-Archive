### Post by %hMa@?b&lt;C on 2006-03-12
GAIM should be already installed under APPLICATIONS>INTERNET>GAIM, but if you want to do an new installation, here is the absolute simpliest way.

```
get http://jaist.dl.sourceforge.net/sourceforge/gaim/gaim-1.5.0.x86.package[code]
then run the package that you got in terminal with
[code]sh gaim-1.5.0.x86.package
```
and that should set you up!
