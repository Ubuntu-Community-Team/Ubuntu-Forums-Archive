### Post by damu on 2006-04-18
Ih there is one thing that is not obvious to get in the linux world , it's probably wireless connection. I also have a 3com usb wireless dongle : 3crusb10075.
Some people managed to get it working with Ubuntu... I didn't :-? 

If you put "3crusb10075" in the search of the forum, you will find the threads about it. 

It seems it should work out of the box with the next release of Ubuntu : Ubuntu Dapper. It has inbuilt the drivers wich work for this dongle.
I succeded indeed to get the dongle recognised and activated in an alpha release of Ubuntu Dapper. When I switched to Kubuntu dapper I lost it!
They still have some problems in the alpha releases, but check the forum regularly about that.

You might, like me, email 3Com to suggest them to release their drivers in open source or deliver drivers for linux. After all, Linux in not to be blamed on that...without the drivers which come on the cd, Windows wouldn't be able to manage wireless either...at least, in linux , some genious guys try to find some ways round...
