### Post by ajgreeny on 2006-06-18
To get back your host name go to "Network" in your menus.  In the "General" tab you'll see a place to enter the name you want.  Click OK, and there you are.

As far as connecting to the router is concerned, are you wired or wireless?  If wired you shouldn't need to enter anything, as the computer should find the connection without problems, at any rate it did in my case with DHCP.  I did this all at install, and although it took two attempts to find the network (slow getting the network up, I think), when I tried the second time it was there immediately.
