### Post by macdo on 2006-04-15
System => Administration => Synaptic Package Manager. You'll need your password to launch the application.

Then clisk on search and type in the name of the programme. That should bring up the package. Click on it and select "mark for removal". Whan you've done that, click on apply.

Otherwise, you can open a terminal (Applications => Accessories => Terminal) and type ```
sudo apt-get remove avidemux
``` which will do exactly the same thing.

Welcome to Ubuntu!
