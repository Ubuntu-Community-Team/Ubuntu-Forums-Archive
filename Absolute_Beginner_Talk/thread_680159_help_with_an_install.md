### Post by JoshuaRL on 2008-01-27
The Live CD is the one that you can preview Ubuntu on.  The one that you need to use the Install icon on the desktop to install with.  The Alternate CD installs from a text-based GUI.  It has a blue background and ASCII text.  It uses much less system resources since it doesn't have to load a Live Desktop.  And it includes more, like upgrading from the CD.
