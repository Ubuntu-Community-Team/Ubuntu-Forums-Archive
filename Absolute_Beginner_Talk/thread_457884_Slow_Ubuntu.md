### Post by ryanVickers on 2007-05-30
Your RAM isn't very much, but It should be fine, especially since you said you usually only have one program open.  But that's right, some programs slowly consume more memory over time for no reason, and this could lead to a extreme slow down because it's using swap as ram.  try restarting any programs like this every now and then (to see what they are, go into system monitor and sort by memory - use the View > all processes and turn OFF dependencies (confuses sorting).
