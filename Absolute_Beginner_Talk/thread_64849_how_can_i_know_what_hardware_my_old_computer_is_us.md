### Post by atrus325 on 2005-09-12
I'd use the Gentoo install disk, which comes with a bunch of useful tools pre-loaded.  

Just pop it into the drive, reboot, and wait for the command prompt to come up.  Then type:

# lspci

It'll give you some pretty useful information regarding your hardware.  Just jot it down, pull out the disk and reboot.  Nothing will have been done to your system.

J.

P.s., the Ubuntu live disk might have that tool too.  Gentoo is just the only one I've used.
