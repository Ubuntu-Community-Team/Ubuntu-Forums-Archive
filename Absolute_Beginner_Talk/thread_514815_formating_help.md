### Post by Smu on 2007-08-01
Just install gparted wich is a partition editor for gnome. With gparted you'll be able to format your windows disc.

sudo apt-get install gparted

It'll appear under system -> administration -> GNOME partition editor.

With automatix you can install virtualbox or VMware wich allow you to create virtual machines. I'm using virtualbox and found it quite easy to set up.
