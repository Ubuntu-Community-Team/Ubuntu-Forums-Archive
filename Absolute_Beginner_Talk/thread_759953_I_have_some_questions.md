### Post by AndrewTheArt on 2008-04-19
Windows basically beats Linux in the area of video editing, but there are some decent applications to work with in Linux - such as Cinelerra (a little complicated to learn, however). Kdenlive is also fairly good and simple to use. Honestly, I don't edit videos in Linux. I know there are several programs out there, but I'd be lying if I said that I was assured that they were as intuitive and powerful as the array of Windows video programs out on the market. Some other options are Kino, Avidemux, or simply attempting to run your favorite Windows video editing app in Wine.

I don't see an entry for vEmotion in the Win AppDB [here]("http://appdb.winehq.org/search_results.php?cx=013271970634691685804%3Abc-56dvxydi&cof=FORID%3A11&q=vemotion&sa=Search#204"), but try installing the app with Wine and see what happens. It just might work perfectly.

If you wanted to complete eliminate XP, you'd have to put the movies and pictures on some external media (external HDD, CD, etc). However, you could simply tell the Ubuntu install to resize your Windows partition to the smallest degree possible and then have a dual boot setup. You'd be able to access both XP and Ubuntu in this manner, and get your files fairly easily off the Windows partition using a file explorer in Ubuntu (not hard).

You might consider partitioning in this manner with the Ubuntu installer and following [this](http://ubuntuforums.org/showthread.php?t=380699) guide to virtually run your Windows install right off the disk using VMWARE (kind of like Parallels for Mac, no reboot necessary) 

Depends on the MMORPG. Use the WineAppDB (link above) to search. If you're a serious gamer, you might consider Cedega ([link](http://www.transgaming.com/)) and/or the free PlayOnLinux ([link]("http://www.playonlinux.com/en/")).

Good luck!!!
