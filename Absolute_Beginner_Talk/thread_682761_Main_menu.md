### Post by rubbsdecvik on 2008-01-30
Since Ubuntu is derived from Debian, there is an option to include the "Debian" menu.

In Debian the main menu includes somethings that are different than the one that is used by Ubuntu.  This Debian menu allows for some different programs to be shown and to be separated differently.  It really doesn't do anything other than give you an alternative way to keep your main menu items organized.

### Post by rubbsdecvik on 2008-01-30
two things:

1) The Debian menu would show up as it's own thing.  Usually *under* the accesories menu.
