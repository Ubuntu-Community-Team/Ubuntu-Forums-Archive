### Post by Zzl1xndd on 2007-04-26
Open Office saves in the odt format by default and Word doesnt support it. When saving your documents try saving them as a .doc. This works for me.

### Post by rillip on 2007-04-26
When you go to save a file, the default is an Open Office file, .odt.  If you look at the dropdown in the "Save as" menu there's an option there for Word 97/98/200... etc.  You'll need to use that to save it in a .doc format for Windows.

### Post by canadianwriterman on 2007-04-26
You can also change the default OpenOffice document format in preferences, so that it will always save as a .doc document. This is not the preferred way to do things in the open document world, but it helps if you frequently create documents for those poor unfortunate souls who have to use Word.
