### Post by MenZa on 2007-06-12
While QT applications may not be the prettiest when run in Gnome, they will run fine; but bear in mind that if you attempt to download e.g. Amarok, which by itself is 15 megs, will be 160 megs because it needs to download various KDE and QT libraries.

There are some instructions available [here](https://help.ubuntu.com/community/QtGnome) that may help you; though I don't know if they're too out-dated.
