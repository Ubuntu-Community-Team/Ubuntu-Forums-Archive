### Post by starcraft.man on 2007-05-16
Since you installed your drivers, its probably just not an option listing in the xorg.

Run this command in the terminal, then follow the prompts and respond ok to the default until you get to the screen resolution, then make sure that  you select the size you want.

```
sudo dkpg-reconfigure xserver-xorg
```

That should do it, have fun :).

---

### Post by Kobalt on 2007-05-16
It would be wise to run only ```
sudo dpkg-reconfigure -phigh xserver-xorg
```
It will let you chose which resolution you want as available (select them by hiting the spacebar), and not make you go all the way into X configuration.
