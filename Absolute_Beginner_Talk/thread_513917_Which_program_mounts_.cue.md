### Post by wormser on 2007-07-31
This [blog]("http://techbycolin.com/?p=130") has a solution.  It tells you how to convert the .cue to .iso.

> In Ubuntu, open up a terminal and type sudo apt-get install bchunk. This will install a program called Binchunker.

Binchunker is used in the terminal and is very simple. Navigate to the folder containing the BIN/CUE file and run Binchunker in the following way.

bchunk IMAGE.bin IMAGE.cue IMAGE.iso
