### Post by Taboo Mirage on 2008-01-24
Re-install Windows XP over the entire drive, then start again by partitioning the drive correctly.

From the XP installation disk, you may want to set up the NTFS partition of only a certain size, leaving some free space so that you can easily parttion it from within Ubuntu to avoid having to resize the NTFS partition at a later point.

For example, if you have a 100GB HDD.  Partition XP to only 50GB and install it.  Then you can simply partition the 50GB unused space from within the Ubuntu installer for 10GB /, 39 GB /home and 1GB swap.

Take care.
