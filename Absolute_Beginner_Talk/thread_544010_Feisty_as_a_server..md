### Post by p_quarles on 2007-09-05
It really depends on the kind of server you want. If you just want a file and print server, Ubuntu's a good choice. Other distros, though, such as Debian and CentOS have wizards that will guide you through setting up the appropriate server software (Samba). 

And mirroring two drives would, I guess, be accomplished with a RAID-1 setup, which is definitely do-able with Linux.
