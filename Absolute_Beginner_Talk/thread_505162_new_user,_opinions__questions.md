### Post by ramjet_1953 on 2007-07-20
To get as many workspaces as you want, just right click in the workspace switcher and left click on Preferences. A window will open and you just select the number of workspaces that you want.

NOTE: this only applies if you are NOT using Compiz, or BERYL.

If you are using BERYL, go into Beryl Desktop Manager and select the number of desktops in General Options.
