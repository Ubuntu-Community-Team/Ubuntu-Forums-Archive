### Post by skippy81 on 2006-05-23
AMD64's can work in 32bit mode (like they do in windows) or in 64bit mode.

There are 32 and 64 bit versions of ubuntu and you can choose which you want to install.  64bit version is slightly faster, but it is harder to get software for it (for instance Macromedia Flash is not availiable for 64bit yet).

My advice would be to get an AMD64 though, they are the best processors for your money at the moment.  Just run 32bit linux on it until more 64bit programs are released.  

By the way, I am currently running Dapper Drake 64bit version on an AMD64 3200 and it runs great.   I have also installed 32bit Dapper and can assure you that it runs fine.

### Post by arsenic23 on 2006-05-23
There's really nothing to worry about.  And I most certainly wouldn't want fear of 64bit operation to scare you into buying a P4 if a new AMD based unit is what you want.  It's true that any 64bit OS you install is going to be a little bit harder to get set up correctly, the same is true wether you want to use winXP64 or a 64bit linux distro.  BUT, and though it's already been stated, you DON'T have to install 64bit software on your AMD64 CPU.  It can run the same 32bit applications you'd be running on that P4 with no side affects what-so-ever.

In fact I'm still running 32bit software on all my 64bit CPUs.  I'll more then likely migrate to 64bit stuff in about a year or so, depending on how things go.
