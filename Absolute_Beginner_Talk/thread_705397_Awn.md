### Post by Unstuck on 2008-02-23
Your posts make it sound like you have two issues: You cannot find AWN Manager and AWN itself has disappeared.  

AWN has a habit of disappearing which isn't a big surprise considering the release isn't stable yet.  Try running it through terminal to get it back, and the next time it crashes we will have some kind of output from terminal.
```
avant-window-navigator
```

AWN Manager can be found in System > Preferences

I hope you find this helpful.
