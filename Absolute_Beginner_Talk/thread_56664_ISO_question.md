A .ISO file is data that is meant to be burned to a CD.  You can mount them to the local filesystem (so it looks like a folder).  The following is a link that explains how to do it via the command line:
[http://www.frozenblue.net/tools/howtos/?v=iso-mount](http://www.frozenblue.net/tools/howtos/?v=iso-mount) 

There are probably GUI tools you can use to do the same thing.  Try searching for "mount ISO to filesystem" or something similar.
