### Post by xmastree on 2005-12-01
> how do i install mandrake from xp?
You can't really install linux **from** xp. You need to create some space on your hard disk for it (or even install a second disk) and boot from the install CD which should offer you the option of using the space. That's if you wish to dual boot.

It's not like installing a program which runs under xp. It's an operating system which runs **instead of** xp.

Otherwise you can boot from an install CD and completely overwrite your XP installation.

But I suspect you don't want to do that just yet... :rolleyes:
