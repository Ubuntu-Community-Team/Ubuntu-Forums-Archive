### Post by jken146 on 2007-12-12
A canon 600?  That sounds like a really old model.  If it is, then just plug it in and switch it on.  I'd be surprised if Ubuntu won't set that up for you.

edit:  I know that USB printers are autodetected most of the time.  If it is that old, it's probably on parallel rathere than USB, so you may need to go to System > Preferences > Printing and add it there.  In either case a driver shouldn't be necessary.
