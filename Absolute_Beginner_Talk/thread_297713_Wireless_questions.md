### Post by candtalan on 2006-11-11
Welcome!
An easy approach initially is probably to use the live CD and confirm that you see your wireless card is recognised in ubuntu - (it sounds as if you have not installed it yet?)

It is likely that the PC wireless card can be installed with no problem, although wired ethernet connections to routers are by far the easiest. I like to begin with wired, then move to wireless after the initial phase and I am settled.

Others here may help with any drivers questions.

XP laptop - there will be no conflicts as such. If you use a home netwwork, etc, then you will simply want to use sharing with windows (smb samba protocols) which are all pre arranged in the distro, but you will have to tell it what to do first.

Note that your xp is likely to be in ntfs file system (not FAT) so you can read it from linux but it is not usual to write into it, although it is said to be easier nowadays - using a special package.
