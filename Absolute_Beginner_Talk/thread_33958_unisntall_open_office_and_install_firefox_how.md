### Post by mendicant on 2005-05-12
You would have to uninstall openoffice.org.  Firefox, I thought, was already there.  If not, it is called mozilla-firefox.  You would usually use a package manager frontend, like synaptic.  I think kubuntu has kynaptic, but I've never used it.  If you do have synaptic installed, you can run it, and press the search icon (top right), one at a time.  For openoffice, click the green button under the S, and select one of the removal options.

Similarly, do a search for firefox, click the button, and select "Mark for Installation".  Once that's all done, click the "Apply" button.
