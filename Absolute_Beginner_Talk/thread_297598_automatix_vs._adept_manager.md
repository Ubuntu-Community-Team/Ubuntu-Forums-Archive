### Post by msak007 on 2006-11-11
Comparing those 2 is like comparing apples and oranges, as they don't really serve the same purpose. Adept is a package manager, a front-end for apt just like Synaptic which lets you install anything in the repos. But you have to search for the packages unless you already know the package names. Automatix is a GUI script that simply provides a way for people to install common programs / codecs / etc. from a list all at once - the list is there, you check off what you want to install without having to know the package names. A lot of people use it for automation when setting up a new system so they don't have to do everything manually or remember package names. There's pros and cons, arguments for and against Automatix so the best I can tell you is try it out and see if it suits you. Just be warned that certain programs are not in the repos and the script will install them from 3rd party sources which can cause a lot of problems down the road.
