### Post by Tatty on 2008-04-06
The best advice for learning linux i believe is to simply dive right in...  The best method I have found for learning is to:
Try it -> Break it -> Fix it.

If you set up your own linux machine, then you are more likely to break something than you are to break a closed source OS... but at the same time it is usually much easier to fix.  And in doing so you learn an amazing amout.

Once you run into problems the best place you can go to look for an answer i believe is right here on this forum.  Try searching the forum - almost any problem you run into has been encountered by someone else before, so the chances are the answers are on here somewhere.

Also you will notice at the top of the page you can go to SUpport->Documentation, which contains both the official documentation for each currently supported release, and the semi-official wiki, which is supported by the community.
Start by reading the official documentation to get a basic idea for how things work, then try the forums, then after that try the wiki.  Then try googling, then if you are still struggling try posting on the forum.
I have never run into a problem with Ubuntu which cannot be fixed.

I have never encountered a problem with Ubuntu which could not be solved.


In terms of which distro to use, i would say start with ubuntu, because it is well known for having a good community to support it.  After you get to grips with Ubuntu, then you can start branching off and start using other distros to get a wider linux appreciation.

EDIT: and yes, read the psychocats guide on sef's above post...  they are great for beginners.
