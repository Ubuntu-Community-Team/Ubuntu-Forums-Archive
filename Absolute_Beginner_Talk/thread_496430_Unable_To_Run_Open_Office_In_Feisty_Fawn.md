### Post by p_quarles on 2007-07-10
That's likely more of a compatibility issue with MS's closed document formats. The workaround, in this case, would be to install the MS Powerpoint Viewer for Linux, which is available in Add/Remove Programs. 

About OOo crashing, I can only think of one thing: it requires Java to run, so make sure you have that installed. Apart from that, no clue.
