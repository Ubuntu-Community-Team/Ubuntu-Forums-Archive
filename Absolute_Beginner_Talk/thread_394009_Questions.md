### Post by johnnymac on 2007-03-26
> **ilowtf said:**
> Hi!

I'm new in this forums and in linux's world and before installing ubuntu I have some questions that would help me choose to install it or not.

1) What's the difference between Ubuntu, Kubuntu, Edbuntu and Xubuntu? Do they have different purposes? Which one is better? Which one should I install first as a beginner in Linux?

They are all Ubuntu - but with different Windows managers (Edubuntu is for education).  Kubuntu uses KDE, Ubuntu uses Gnome and Xubuntu uses XFCE.  None are better than the other - just a preference (I use Xubuntu)

2) What are the differences between Linux (Ubuntu, Kubuntu, Ebuntu, Xubuntu), Windows and Mac OS? Does Linux has something that makes it different from the other OS?

Wow...this question would take a long time to answer properly.  Windows is totally different but Mac and Linux are closer alike than they use to be.  You can answer this question by just googling and reading for a very long time.

3) What are its disadvantages? (i.e. compability between common type of documents like .doc, .jpeg etc. and not so common like .java, etc.)

As far as disadvantages - only big one is gaming.  It's not quite "there" yet for Linux

4) Is linux focused to a certain type of user? (i.e. programmer)

It used to be focused for the developer; however, now it's just as easy to use as windows so more and more people are using it for multimedia and just plain "office" type usage.

5) I've read in this forumus terms like feisty fawn, beryl, dapper, grub... What are they? Any links in which I can find some basic terms?

Best would be to do a wiki search on them to get the exact definition.  But at a glance...
 - Feisty Fawn (and Dapper) is a version of Ubuntu (currently under test)
 - Beryl is a type of composite manager for giving you 3-D desktop capabilities
 - Grub is a boot manager (there are two types Grub and Lilo - Grub is common)

6) Is Linux difficult to learn how to use it?

Not too difficult - you'll have some learning issues but the greatest thing is the community forums - use them if you get stuck.

7) Is it hard to install programs?

Nope - not hard at all - just use synaptic package manager

8) What about customization? I've read that you can customize it, but in which aspects?

You can ultimately customize everything....so much you can get lost.

9) Are there any kind of programs or type of documents that just doesn't work with Linux?

Games....unless you do Wine or Cedega
