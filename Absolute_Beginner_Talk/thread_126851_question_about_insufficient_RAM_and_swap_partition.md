### Post by grinchy on 2006-02-07
yea you could.
The general rule is to make your swap partition twice the size of your ram, maybe you should make it 3 or 4 times the size.

I would recomend using xfce as you desktop enviornment instead of gnome as it will use less memory.

try running ubuntu live first to see how well it runs on your pc.
The installed version will be faster but it will be a good guage of speed as well as hardware support
