### Post by taurus on 2007-05-28
Edit /etc/fstab

```
gksudo gedit /etc/fstab
```
and replace the s in the last line with the h.

```
/dev/**h**db5 /media/internal-160 ext3 defaults,errors=remount-ro 0 1 
```
Save it and reboot.
