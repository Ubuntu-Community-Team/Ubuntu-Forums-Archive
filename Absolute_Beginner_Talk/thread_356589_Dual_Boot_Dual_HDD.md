### Post by alienexplorers on 2007-02-08
Is it possible to run Ubuntu on one HDD with its own home partition and Xubuntu or another variant on the second HDD with its own home partition.  I just got a new 80GB drive and want to try out new versions of Linux on the second drive without messing up my Ubuntu stuff.  Can I continue to boot and select the operating system through GRUB.  Do I need a Grub boot disk or something to set up the dual booting.

Don

---

### Post by taurus on 2007-02-08
Yes.  You can install Ubuntu on one harddrive and Xubuntu on a second harddrive.  Just let the second installer to install GRUB on MBR and it should be able to handle booting both versions.
