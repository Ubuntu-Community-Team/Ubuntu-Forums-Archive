### Post by SendDerek on 2007-08-28
If I'm understanding you correctly, you could use keyJnote for a presentation exported as a .pdf.  

You can also export the presentation as jpeg images and then use xslideshow from the screensaver preferences.  In order to point xslideshow to your pictures though, you have to create a file called .xslideshow in your home directory and then add a line that represents the path of your images.  Like so:
imageDirectory: /home/username/Presentation/
