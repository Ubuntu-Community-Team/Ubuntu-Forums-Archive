### Post by kwacka on 2008-03-07
because you are connected to a linux server doesn't mean you have to run linux, run linux or Windows the choice is yours.

If you do want to run linux you can dual boot with Windows so you can choose Windows for gaming (if you wish) and boot linux for everything else.

To check the compatability of the games you wish to run look at the wine database at:[http://appdb.winehq.org/](http://appdb.winehq.org/)  (WoW is included with advice for installation).

Firstly install wine (enable the 'universe' repository) then at the Windows app file right-click and select 'open with'  --> 'wine'
