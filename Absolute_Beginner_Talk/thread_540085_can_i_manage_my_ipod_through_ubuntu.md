### Post by sumguy231 on 2007-09-01
There's gtkpod, which I've never actually used. It only manages the iPod though, it's not a media player. Really I think you might want to look at Amarok - it can't play videos, but it can send them to the iPod. Of course, that is if you happen to have a video file which the iPod will actually play, which is a slim chance in my experience. Amarok is a KDE application, but will run fine in Gnome.
