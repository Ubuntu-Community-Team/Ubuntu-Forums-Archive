### Post by kingofpain on 2008-01-07
Hmm... if you noticed, after installing screenlets, a new menu entry appears in ubuntu menu. The "other" menu entry. There you cand find some screenlets, and there, looking at the Launcher Command, you can see the command which starts the screenlet.
For example, the Clock screenlet starts with "usr/local/share/screenlets/Clock/ClockScreenlet.py > /dev/null" command.
