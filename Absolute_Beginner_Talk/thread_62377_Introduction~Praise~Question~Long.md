### Post by Transmission on 2005-09-04
I've been attempting to break free from the clutches of Windows for some time now. I have tried several Distros such as Mandrake, Knoppix, ect. but I never could get past the overwhelming 'culture shock'. The Documentation I found to be geared more to the already experienced Linux user. The forums were the same, little help, if any, on explanation  to the newcomer.  

A while back I came across Linspire. It made all these promises of being 'user friendly' and the 'easiest on users migrating from Windows'. While this was true for the most part, what they failed to mention was that unlike most other Linux Distros, it had Licencing fees, Subscriptions to the CnR service ( and isn't that just a basic package manager anyway?). Before I found this out I had already downloaded the Live CD version so I figured lets give it a whirl. Since this isnt a Linspire forum I'll just say this, I wasnt impressed.

One of the reasons Ive always ran home to windows was the fact that Im an avid gamer. It was during my Linspire test drive that I noticed Cedega. After doing some looking I seen it was derived/stolen from Wine. Then I noticed Cedega was subscription based as well. So we have Linspire at $50, CnR at $20 a year, and Cedega at $55 a year.....plus the monthly fee I pay for online gamming. Needles to say I didnt find this appealing at all. So back to Windows I go.

I was in the process of writting a couple of guides for a fansite for City if Heroes ( [Gameamp.com](http://cityofheroes.gameamp.com/) ) when I came across Ubuntu. I obtained the Live CD version of 5.0.4. To avoid making this post too long Ill only point out the most noticable aspects of Ubuntu, skipping the part where I immeadiatly fall in love and decide to get the install CD downloaded. The instalation was a breeze, no hang ups or sections way too far over my head to grasp. The default Gnome desktop was attractive and easy to work with. The overall functionality of this package is top notch.

And the most noticable aspect of Ubuntu is....The community. Ive been reading the forums and how to's for a while now. Nothing but helpfull people everywhere. Im amazed at the time and knowledge gone into making the transition  to Ubuntu a very simple task.

So Ive finaly done it. I took the plunge, no dual booting. I am on a 100% Ubuntu setup. 1 hard drive, 1 Operating system. It's nice to be able to say that ^^ . 

As I said before I'm an avid gamer. This is where the question(s) starts. Dont worry I have them all up and running with Cedega no problems there. My Problem is the performance seems to be not all that it could be. From what ive read Gnome is a bit of a resource hog. 

My question is this, Is there another window manager that I can switch to for when I get ingame? Ive already removed the desktop background and changed Gnome's theme to 'Simple' and this has improved gameplay some but not alot. Are there any tips/guides on tweaking Ubuntu / Linux / Gnome for gaming?


All in all I must say I am the newest Ubuntu fanboi. I love it and dont think Ill be running back to windows anymore. Thanks for reading.
