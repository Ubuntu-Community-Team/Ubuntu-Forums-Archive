I'd say that your system should run just fine on standard Ubuntu. With that said go ahead and try xubuntu and kubuntu if you want to check them out. A lot of people end up running systems with a mix of programs from all three flavors of *buntu

As far as installing, my advice is to try things out with the live cd but always install from the alternate CD. In my experience the alternate CD has a lot fewer issues when it comes time to install. The live cd is great to test it out and make sure your hardware is supported.

### Post by cozmicharlie on 2008-04-18
Great you are giving it a shot.  Just a warning though - Linux is not Windows so give it a fair chance and understand you will have to do some learning.  What's great is you millions of people all over the world willing to help on the forums.  

I just set up an old Gateway as a bittorrent server and I tried a number of different Linux Distros so I can pass on a little experience (though  I am hardly am an expert).  I would recommend you spend some time reading here distrowatch.org.  It has nice short descriptions of the various distros and reveiws -a great place to start.  I tried a number of distros that are meant for older PC's and I agree with the above - you can't go wrong with Xubuntu.  A couple others you may want to look into are PClinux - this is a very popular distro and they have a version called MiniMe (you have to do some searching to find it on their web site) that is meant for older PC's.  Even the regular version though should run on your box.  A couple others include Linux Mint and if you want to go real light thier is DSL (Damm Small Linux).  What I do is set up Virtual Box (available free for either Linux or Windows) and play around with various distros in Virtual Box before I install them.  It's real easy and lets you try them out before committing to a full install.  Most have Live CD's also but I find them slow so I like using Virtual Box.  

Good luck and report back and let us know what you decided on.
