### Post by FuturePilot on 2008-02-13
Make sure all the repos are enabled. Go System&#8594;Administration&#8594;Software Sources and check all the boxes in the first tab. Go to the Updates tab and check the first two and optionally the fourth. Hit Close then Reload when prompted. Then check Synaptic again.
