### Post by Stinger on 2006-02-07
The easiest thing to do before trying to install anything you might need is to install [Automatix]("http://ubuntuforums.org/showthread.php?t=66563").
Please read the instructions carefully before installing.

Once you have automatix installed , it enables almost all available repositories , I would think it solves your dependency trouble , I think it would solve the rar file-thing too , haven't tried it myself with Ubuntu yet , but with another distro , and was able to open rar files in fileroller.

Automatix adds a lot of other nice features too , you should try it !
