[QUOTE=erik1397]Hi I'm pretty new to ubuntu so bear with me.

First, you need to enable the *universe* repository. You can find instructions for how to do that here:
[http://www.psychocats.net/linux/sources.php](http://www.psychocats.net/linux/sources.php)

Now you just have to install the **gftp** package from Synaptic, or by typing
```

sudo apt-get install gftp

```
in a terminal.
