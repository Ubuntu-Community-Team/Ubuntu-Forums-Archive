I just want it to work, and I'd like the process to be as painless as possible. Is there another distribution I'd have more luck with? Should I reinstall Feisty Fawn? Should I go back another version to Dapper Drake? 

Please, give me something. I want to keep the faith. I want to recommend this to people I know. I want to [write more stories like this one]("http://scooterdman.newsvine.com/_news/2007/02/13/566477-my-first-linux-experience-overwhelmingly-positive"), but this experience has been so demoralizing.
