### Post by NeoLithium on 2007-04-16
You can install XMMs through the repositories, if you've enabled them all. Check this out:
[http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_add_extra_repositories](http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_add_extra_repositories)

Then you just have to type
```

sudo aptitude install xmms

```

Usually files that are zipped need to be compiled on your system, I think there was a guide somewhere, but I can't find it at the moment.  They usually come with a README file though with instructions as well, inside the archive.
