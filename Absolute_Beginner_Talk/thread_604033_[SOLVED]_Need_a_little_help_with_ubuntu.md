### Post by Maestro23 on 2007-11-05
Need to add the Medibuntu repository for your Distro(version of Ubuntu), using the instrucitons on the following link: [https://help.ubuntu.com/community/Medibuntu?highlight=%28Medibuntu%29](https://help.ubuntu.com/community/Medibuntu?highlight=%28Medibuntu%29)

Then you can use Synaptic to search for Skype.  Mark for installation and Apply.
System>>Admin>>Synaptic

*Welcome to Ubuntu and if you have any questions, dont' be afraid to ask.  Some more links for your reference.

Installing Software in Ubuntu:
Installing Anything: [http://monkeyblog.org/ubuntu/installing/](http://monkeyblog.org/ubuntu/installing/)
