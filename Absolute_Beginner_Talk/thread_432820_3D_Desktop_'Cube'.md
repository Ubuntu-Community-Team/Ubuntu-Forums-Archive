### Post by xpod on 2007-05-04
If you can run beryl then go for that but 3ddesk will do a cube for you,it will even spin the cube just like a screensaver if you leave it to......somehting even beryl cant yet do:) 

```
3ddesk --acquire
```

to acquire your 4 desktops(or how ever many you have set)

then
```
3ddesk --mode=cylinder
```
