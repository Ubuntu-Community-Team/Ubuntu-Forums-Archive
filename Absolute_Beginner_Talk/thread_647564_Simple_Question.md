### Post by dstew on 2007-12-22
[This]("http://www.linux-live.org/#") seems like what you are trying to do. You first get a system installed the way you like it, then make a LiveCD. I think you have be be aware of the restricted software, though. It might be illegal to distribute a LiveCD if your system has restricted drivers in it. To make a disk that can install a system onto a hard drive, [SystemImager]("http://wiki.systemimager.org/index.php/Main_Page") might be what you need. It can be installed from Synaptic.
