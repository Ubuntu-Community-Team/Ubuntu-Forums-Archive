### Post by Kethinov on 2006-04-29
Gabber is a good client, but if you're looking to use a lot of different IM protocols, I highly recommend GAIM. You can install it by going to System > Administration > Synaptic Package Manager and doing a search for "gaim" then installing the gaim package.

Once you've got GAIM installed, you could follow those guides to get Jabber working. GAIM also supports MSN.
