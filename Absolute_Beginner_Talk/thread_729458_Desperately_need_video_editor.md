### Post by Suparious on 2008-03-19
Your best bet would be to find some sort of ubuntu-supported software. A way to do this from your ubuntu right now would be to open up Synaptic Package Manager.

'System' -> 'Administration' -> 'Synaptic Package Manager'

then search for 'video' and look in the descriptions for a program that sounds helpful to your project.

The best advice I could give here, would be to investigate [UbuntuStudio]("http://ubuntustudio.org/").

### Post by nhandler on 2008-03-19
One video editor that I found to work extremely well was Kdenlive. It is a KDE application, but it will run fine on Gnome. It uses ffmpeg to encode the videos, so it supports a wide list of formats. The interface is also extremely easy to use.
