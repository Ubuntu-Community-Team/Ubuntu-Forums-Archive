### Post by Qrk on 2006-09-16
There are a ton of great distros out there. The big three today are Ubuntu, SuSE and Fedora

SuSE has a great GUI for system administration (YAST) which sets it apart if you want an XP like control center. The downside is that the stable version uses Gnome 2.12 not Gnome 2.14, so I found it just a little slow.

Fedora had a great release for FC5. You may want to give it a try, I really liked it, the only problem I had with it is that I'm so used to debian bases systems that I had to relearn some basic commands for configuration. Still, it has a geat package manager, just like ubuntu. Its also quite fast.

I also like Slackware (I installed it to my laptop... which is an older machine) but its a lot of work.



Basically, I'd recommend trying a whole lot of distros out, to see which is the best for you. I usually tell people who are new to linux to stick with one distro for at least a couple of months before trying something new. This forces them to try to solve their problems and think linux-y. Once you've got enough time on one distro, the rest are easy.
