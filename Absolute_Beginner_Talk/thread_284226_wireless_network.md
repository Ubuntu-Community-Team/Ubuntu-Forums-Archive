### Post by Echo35 on 2006-10-25
Network manager should already be there, and ndiswrapper is on the ubuntu CD. Just insert it and when the message box pops up, click "Open Software Manager" and do a search for ndiswrapper, install it, then follow the directions from the wiki. You'll also want to download the Windows version of the driver from whatever PC you are on now. If all else fails, just try an actual wired connection to the internet. They almost always work by default and then you can download what you need that way.
