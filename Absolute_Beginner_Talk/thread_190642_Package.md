### Post by steve.horsley on 2006-06-06
Any official Ubuntu packages will make entries in the menu. Dome unofficial ones do too. So you could check under Games. If not, open synaptic, find the package, right-click and get the properties and the list of installed files. Look for files installed in /usr/bin - these will be the comands to launch it.
