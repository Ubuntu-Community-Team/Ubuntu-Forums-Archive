### Post by silviur on 2007-12-24
If you didn't choose to save your existing partition(s) when you installed Ubuntu, you lost the windows partition. To check, select Places - Computer, then right click "filesystem". If it's size equals your hard disk, that means you installed Ubuntu and lost Windows.
