### Post by vikram on 2007-11-30
I don't think the KDE takbar calendar will show appointments. You can use Kontact, an integrated PIM app with Calendar, Addressbook, email etc.

you can get kontact from the repos.
sudo apt-get install kontact

One use I found for the calendar was to synchronize my clock using NTP. In admin mode choose your timezone (and optionally the ntp server) and click on the synchronize option. hope this helps.
