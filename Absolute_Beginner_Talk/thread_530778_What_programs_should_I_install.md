### Post by gatman3 on 2007-08-21
Wbat exactly do you mean by "video encoder" ?  DeVeDe is a DVD authoring tool.  If you're looking to author DVDs, you might also try qdvdauthor.  if you're looking to edit video, try Kino (begineers) or Cinelerra (advanced).

Kompozer is really good for editing web pages.  It's basically an updated version of the old Mozilla Composer, and unfortunately it's not in the ubuntu repositories.  But I've tried many, and it's the best WYSIWYG HTML editor for linux.

k3b is probably the best program for burning CDs/DVDs.  If you want to rip DVDs, use Automatix to get the proprietary DeCSS DVD decryption stuff.
