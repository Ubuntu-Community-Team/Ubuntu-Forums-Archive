### Post by sumguy231 on 2008-01-25
You can run some software on Ubuntu with a program called Wine. Details on what programs it can run are here:
[http://appdb.winehq.org](http://appdb.winehq.org)

Unfortunately, I'm almost certain that it won't run Photoshop CS3. If you really really need Photoshop CS3 for something, you might want to either stick with Windows or set up a dual boot so that you have both operating systems installed and you can run Windows specifically if you want to run Photoshop. Or you could run Windows in a virtual machine inside of Ubuntu.

There are of course free alternatives for some of those listed applications (search the forum for more):
iTunes - Rythmbox/Amarok/Many more
MSN Messenger - Pidgin/aMSN/Kopete/more
Photoshop - The GIMP (Not for professional work, but good for simple editing)
Steam Gaming Client - None of course, see Wine appdb

> P.S. Will ubuntu keep me away from viruses?
There are no traditional "viruses" in the wild for Ubuntu and it has a far better track record with security, however you should still excercise common sense and caution in what scripts and software you run.
