### Post by orkim on 2007-03-14
If my memory serves me correct, it is the KDE desktop equivalent of .lnk files on Windows.  This is a "short cut" file that will launch a program/applet/etc when clicked on.  If you drop the files in the ~/Desktop/ directory you should be able to test this.

Again, I'm not positive, but I think that is correct.  You might Google for .ink and see what you get.
