### Post by think13 on 2006-09-27
He is talking about using the slider bar for the 'I don't want to mess with partitions myself, do it for me so I don't mess up' part of the installer.  I used the same thing.  It is very unclear which side of the slider will be used for ubuntu and which will be used for Windows.

The left side will be used for windows.  Ubuntu goes on the right side.  So you want to look at the 'amount freed' part for how much space Ubuntu will use.
