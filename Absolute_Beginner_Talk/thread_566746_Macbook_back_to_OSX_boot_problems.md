### Post by mike555 on 2007-10-04
If you installed correctly you should be able to hold the " Option " button while starting , until you get an option of which OS to use .......... if you installed Ubuntu by just clicking the default install you probably wiped your Mac partition ............ in that case if you want the Mac OS back you will need to partition the hard drive into three partitions , one for the Mac , one for the Ubuntu , one smaller one (maybe 1 GB) for a swap , then reinstall the Mac OS , then  then reinstall Ubuntu and be more careful to only install it to the right partition .........
