### Post by HereInOz on 2008-03-07
It could be a dodgy image file, or it could be as simple as that you burned the CD too fast.  When you are burning an iso, try to burn it at a fairly slow speed.  Give that a go before you download the image again.  I usually burn iso files at about 4x for a CD.

Also, if the live CD doesn't work for you, download the alternate CD and install from that.
