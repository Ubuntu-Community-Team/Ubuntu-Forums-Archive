### Post by 3rdalbum on 2006-07-29
Ubuntu doesn't mount Windows-formatted partitions by default, for security reasons. It doesn't mean that it hasn't recognised them.

Does your SATA hard disk show up in the Disks control panel? If so, try looking here: [https://help.ubuntu.com/community/MountingWindowsPartitions]("https://help.ubuntu.com/community/MountingWindowsPartitions")

Look under "For FAT32 Partitions".

If you want to look for another distro, try SUSE, Fedora, or PCLinuxOS. They are not based on the same thing that Ubuntu is based on, so you might have more luck. Ubuntu really shouldn't be locking up at all - I've only ever had lockups when using a buggy ATI driver and occasionally with XGL (beta software).
