### Post by crf on 2008-03-08
I think that is what it is supposed to do, but I personally turned off thumbnailing because of the older computer I'm using. There should be a .thumbnails file in your home directory with thumbnails. There are options for how nautilus thumbnails. (Open a nautilus window and choose edit menu --> preferences). 

Maybe you can set Nautilus to only thumbnail or not thumbnail certain specific folders (personally, I'm not sure how).

*edit*: I see you can choose which file types are thumbnailed using the configuration editor program at the key /desktop/gnome/thumbnailers.
