### Post by spiderbatdad on 2008-01-24
Try plugging in, then starting the computer. There are surely some mount commands that will work if you search a little. Also check out ```
man mount
``` Sorry, I don't know a lot about mounting devices, but I do know hotplugging does not work with all devices in linux. If it mount by coldplugging, once you learn the mount command, someone can help you make a little script to auto mount with by plugging into your running machine and launching the script. This may also be an issue with the file system on the external drive. Is it formatted? Here's a link to more than you may ever want to know about mounting.[https://help.ubuntu.com/community/Mount](https://help.ubuntu.com/community/Mount)
