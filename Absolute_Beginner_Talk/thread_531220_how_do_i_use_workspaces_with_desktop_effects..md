### Post by varun_shrivastava on 2007-08-24
hi

i also face the same problem so i take these steps

1. i disable the "workspace on cube" from desktop effects
2. now increase the number of workspace to desired number by right clicking on the workspace that is visible
3. Now again enable the "workspace on cube" from desktop effects

MOST IMP: Never touch that "Desktop Effects" any more.
If u do that you will again end up with single workspace
so again follow above steps

to switch between workspace: ctrl+Alt+ Arrow key{left and right}
to move a window to another workspace: shift+ctrl+Alt+ Arrow key{left and right}
