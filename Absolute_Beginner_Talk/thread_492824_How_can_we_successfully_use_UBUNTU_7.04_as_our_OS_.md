### Post by coolen on 2007-07-05
You could try getting some native games from the repositories (Application -> Add/Remove... -> Games).
Windows games will require Wine, and no guarantee on how well they'll work.
According to the Wine App;lication Database, Red Alert is silver at the moment...meaning it has a few small problems that, with a little development and a few overrides, should be worked out before long (we hope).
Best thing you can do is try it, I guess.
