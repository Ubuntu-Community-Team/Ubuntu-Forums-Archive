### Post by benerivo on 2007-07-10
I think the kde themes from kde-look should download as .kth files, that are theme files which can be loaded from the 'Control Center'. I use Plastik which is one of the default themes in the kde package, along with a desktop slideshow for the background.

The Control Center is where pretty much everything can be set up as you like it, including web browsing with konqueror. I use konqueror as it is fast and straightforward. There are plenty browser settings you can change, but no extensions as you can get with Firefox. The only ones I used anyway were Adblock and NoScript, both of which can be set up as part of the browser settings in konqueror anyway.

The other kde programs i use are: amarok, kopete, k3b, digi-kam, and kaffeine. I don't know of any QT music player daemon, and i've never tried kmail (the default email client in kde), but going from my experience of the kde suite so far i'm sure it'll be fine for you.
