### Post by Shatrat on 2007-01-17
I just installed a dual boo setup on this laptop this weekend, if I remember correctly it gives you a page partway through the install process with 3 options, format the whole disk, use the largest unallocated space, or repartition manually.
I think those were the options.
I used teh repartition manually option and shrunk the windows partition (its good if youve defragged before doing this) and then installed on the free space created.  Windows is automatically added to the end of the grub boot menu.  It is really pretty easy, don't sweat it too much.
