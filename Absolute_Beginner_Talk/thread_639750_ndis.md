### Post by Outrunner on 2007-12-13
If you can access the 'net without wireless, you can install it using this command

```
sudo apt-get install ndiswrapper-common ndiswrapper-utils-1.9
```

If not you can download the .deb files from [here]("http://packages.ubuntu.com/cgi-bin/search_packages.pl?searchon=names&subword=1&version=gutsy&release=all&keywords=ndiswrapper&sourceid=mozilla-search")
