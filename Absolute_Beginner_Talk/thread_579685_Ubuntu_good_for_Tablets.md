### Post by MontanaMax on 2007-10-18
I've been running Kubuntu on a HP TC1100 tablet for over a year now, and very very happy with it. I haven't tried using any handwriting recognition stuff, but GIMP, Krita, and Inkscape are all great fun with the pen. There are some good articles on this forum about how to setup a virtual keyboard package if you want to log in with pen only.

Definitely burn a live CD and give it a try on your tablet - it's well worth the time to experiment!
