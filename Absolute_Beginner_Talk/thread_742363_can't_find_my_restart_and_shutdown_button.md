### Post by Zorg-it on 2008-04-02
Try this:

1. Click System, mouseover Administration and select Login Window.

2. Input your password when prompted to do so and click OK.

3. When the Login Window Preferences window appears, select the Local tab.

4. In the Menu Bar section, check the Show Actions menu checkbox.

5. Click Close.

Source: [tech-recipes]("http://www.tech-recipes.com/rx/2387/ubuntu_restore_restart_shut_down_log_out_button")
