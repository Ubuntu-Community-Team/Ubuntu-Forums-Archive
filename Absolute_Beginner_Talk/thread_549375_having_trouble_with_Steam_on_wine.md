### Post by Beggar on 2007-09-12
You need to manually install tahoma font now, it has been removed from msttcorefonts (google if you want to know why).

Just do a google search for "filetype:ttf inurl:tahoma"

then download the file and copy it to ~/.wine/drive_c/windows/fonts.
