No...if you are going to use those Debian CDs...just install Debian. Ubuntu is based on Debian Sid (unstable) not the released Debian Sarge (stable) so the two are not compatible.

Some of the programs might work, but in the long run there are some big differences. Debian programs might depend on libraries thats are way onlder than Ubuntu.

For dial up people I often recommend Debian....but I recommend they install it.
