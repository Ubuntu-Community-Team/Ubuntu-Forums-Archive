### Post by Patrick K on 2007-05-22
When you install US, it provides both the standard Feisty generic kernel and a low-latency kernel. The standard kernel is exactly the same as Feisty's. The software packages are different with a heavy emphasis on multimedia production. The "look" is different, being mostly black and gray but you can change that.  US doesn't install the Open Office suite, Ubuntu-Desktop, or Evolution by default, there are some other SW differences. 

The real advantage of US is the low-latency kernel. It helps tremendously with multimedia production. It isn't needed for most any other uses. AFAIK, running the standard generic kernel and adjusting the SW packages would give you exactly the same setup as Feisty generic. AFAIK, You could remove the low-latency kernel. However, you would still have the US upsplash and login theme (black and gray).
