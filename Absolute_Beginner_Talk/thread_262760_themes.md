### Post by CatKiller on 2006-09-22
> **IngSoc said:**
> Well how do I go about installing them?

System -> Preferences -> Theme

Drag-and-drop the file for the new theme onto the Theme Manager window.

Actually, the way I find easiest is the Art Manager (System -> Preferences -> Art Manager. This application will find themes and backgrounds from art.gnome.org and install them for you.
