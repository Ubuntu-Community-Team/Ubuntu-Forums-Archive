### Post by 3rdalbum on 2006-03-22
1. Does the iMac read the CD when you've booted up into OS 9? If it doesn't, try burning to a higher quality CD or ordering a pressed one from ShipIt. If it does, and there's just one file on there (the ISO), then you've used the incorrect settings. You should use the "Disc Image" function in Toast.

2. The Mac won't recognise it as a startup disk in the Startup Disks control panel, so don't try to select it in there. You've got to hold down the "C" key immediately after powering on.
