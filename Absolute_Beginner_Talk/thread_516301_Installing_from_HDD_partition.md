I'd suggest trying xubuntu rather than ubuntu - its significantly faster!  Ubuntu isn't the best linux for optimisation - it seems to aim more for usability (which is good!) but to be honest, just the packages you choose to install make far more difference than a highly optimised, self-compiled system. Basically, keep it small.  Xubuntu defaults to low resource packages, so if you use that you shouldn't even need to think about it much!
