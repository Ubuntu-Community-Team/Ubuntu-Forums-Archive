### Post by aysiu on 2005-12-07
[QUOTE=kane77]I have couple of questions:
1.) Where does package manager keep the downloaded files (those that haven't been installed)?[/quote] I don't know, but you can clear out your package cache in Synaptic. Settings > Preferences > Files

> 
2.) How do I uninstall programs I no longer want to use (not installed from package manager) It depends how you installed them.

> 
3.) I used * update-menus* command and now I have everything in submenu "debian" doubled... It bothers me... How can I fix it (possibly prevent the same thing happening in the future)? Go to Applications > System Tools > Applications Menu Editor. Click on Debian on the left and uncheck the boxes on the right that you don't want to appear.
