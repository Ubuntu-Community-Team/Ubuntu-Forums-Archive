### Post by fiddledd on 2008-01-29
The following are open source, any one will back up your HDD and/or Partitions:

Clonezilla: [http://clonezilla.sourceforge.net/](http://clonezilla.sourceforge.net/)
Partimage: [http://www.partimage.org/Main_Page](http://www.partimage.org/Main_Page)
Ghost 4 Linux: [http://sourceforge.net/projects/g4l](http://sourceforge.net/projects/g4l)

I don't know whether they will backup to DVD though.
I use Acronis True Image (not free), but the above mentioned do the same job.
