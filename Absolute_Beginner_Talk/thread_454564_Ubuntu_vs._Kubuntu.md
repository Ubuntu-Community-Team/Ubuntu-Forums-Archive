### Post by Sparkster185 on 2007-05-25
Many of the applications are specific to the DM you choose.  For example, KDE is an entire suite of applications plus a window manager.  It comes with things like K3b, Konqueror, and amaroK, that GNOME doesn't include by default.  Alternatively, GNOME is the same way.

The layout of the menus is also different.  But you can install KDE apps in GNOME and vice-versa.  There are a bunch of web pages comparing the two DMs, so just do some research and see what you like better.

If you're really new to Linux, I would recommend going with Ubuntu, since that's what most people here use.

### Post by ryanVickers on 2007-05-25
they come with very different applications and tools, but from a apps point of view, you can run either one's programs on the other.  If you must have both, I would recommend getting Ubuntu and then typing:
> sudo apt-get install kubuntu-desktop
this will add KDE, then you'll have KDE and GNOME!  And you don't need to have a separate installation of kubuntu and ubuntu, it can be ubuntu with GNOME and KDE.  Just thought I'd through that command in here because I noticed no one had posted this tip yet.

To use one or the either, before you logon, pick GNOME or KDE from the sessions option dialog.  Oh, one more thing, I would recommend the gdm (gnome logon manager) over the kde one, but that's just me - they both have their advantages.  You can really only choose it once, but then to switch back to the kdm/gdm, type:
> sudo dpkg-reconfigure gdm (gdm can be replaced with kdm)

For me, I need KDE, but very rarely use it.  I would agree with what most people are saying here, use the GNOME because it's more common, and in my (and manys') opinion nicer.

### Post by Dylnuge on 2007-05-25
> **marco123 said:**
> I think because Ubuntu is the main distro more work goes into it.

I disagree. Ubuntu is an open-source project, and Gnome programmers work on Ubuntu, while KDE programmers work on Kubuntu (by programmers, I mean Ubuntu Team Programmers, and by KDE/Gnome programmers, I mean which DM they prefer).

I used to prefer KDE, but now use Gnome. You can install both on the same system-but be aware, it can be a bit glitchy. My recommendation is to download both onto a LiveCD and play around in them, get a feel. Don't be scared off by KDE because it is not the "main" ubuntu, it is just as supported and just as good.

The software for each is interchangeable, but KDE apps look designed for KDE, and Gnome apps look designed for Gnome. It is a semantics issue, and a minor one at that, but you may find yourself prefering one over the other because of what your favorite apps were programmed in. (Many apps are designed for both and therefore look equally as good/bad in both, like OpenOffice).

It seems like pretty crappy advice, to see for yourself, but it is the only way to truly tell. There is a reason there are two (ok, more then two) Desktop Environments, and that is because different people like different ones (and there was a whole open source debate a while back with KDE, but that is over now).

Good luck.

EDIT: Yes, you can only run one at a time. However, you can have both installed, and using the Sessions button in GDM (KDM too?) switch between the two. I know there is a way to do it in KDM, I just forget how.
