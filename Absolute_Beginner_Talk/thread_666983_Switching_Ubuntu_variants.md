### Post by AusIV4 on 2008-01-13
Kubuntu:
```
sudo aptitude install kubuntu-desktop
```

Ubuntu:
```
sudo aptitude install ubuntu-desktop
```

I use Ubuntu on my laptop, Kubuntu on my desktop. I find that Kubuntu is mroe easily configured and has a nicer feel to it, while Gnome tends to have better ACPI and wireless support, which tend to be essential for laptops. Xubuntu is good for systems with fewer resources.

Whichever you choose, they'll be available from GDM or KDM.
