Is there a way to make desktop icons a single click like in KDE. I'm running Gnome.

---

### Post by FuturePilot on 2007-12-01
Both can be accomplished by opening a Nautilus window and going Edit>Preferences
In the first tab you can adjust the default icon size by adjusting the Icon Zoom Level. To make icons one click go to the Behavior tab and select Single Click to Open Items

---

### Post by PointyWombat on 2007-12-01
As far as how to change the behaviour to single-click operations, open up a folder, 

Edit -> Preferences -> Behaviour -> Single click to open items,

HTH
