### Post by tatimmer on 2006-08-26
A modem that works out of the box for Ubuntu.  No driver installation required for linux. It works under Win98 too but is slightly more difficult to set up. 

This is an "external hardware modem" that connects to the 9-pin serial port.

[SIZE="4"]Best Data 56SX-92[/SIZE]

purchased on ebay
I used pppconfig to connect.

My previous experience with the Smartlink soft modem was a little frustrating.  Hope this helps someone.
