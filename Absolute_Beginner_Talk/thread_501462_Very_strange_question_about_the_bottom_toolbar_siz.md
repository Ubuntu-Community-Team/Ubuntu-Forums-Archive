### Post by Roasted on 2007-07-15
Previously I had this computer set up so the bottom toolbar contains my show desktop, window list, workspace selector, system monitor, and trash can, with the desktop and window list being on the left and everything else on the right. Typical setup, right?

Well I set up the toolbar so the minimum was 1 pixel underneath the maximum. I liked this because it kept everything universal, in my mind. If you had 1 window open, it spanned across the entire toolbar. If you had 2, then they cut to half the size and both together spanned across the entire toolbar.

It got messed up because I took my rig and plugged it into an old monitor in the garage, then I used my computer (having all of the songs it does on it) and plugged it into a stereo for a party we had. Pretty sweet deal, just walk over, toss some songs on the playlist, hit random and just go! 

Anyways, now that I'm back on my nice flatscreen inside the settings were a little messed up. The small issue I'm looking at is the toolbar has this space in it that I want to be covered by the window list. So, I naturally keep increasing the size of the window list so it covers it. So really I just have a petty issue that I know can be fixed but yet I can't seem to figure it out. Perhaps in an hour when I'm more awake I could make some magic happen.


EDIT!!!!!!!! - Make that about 30 seconds, not an hour. I was going to delete this since I found the issue but I figured I'd let it here in case other rookies could stumble upon it. I noticed when I right clicked on the blank area that I got different options that I typically would, which was remove from panel, lock, and move. So I knew SOMETHING was there. I guess there was a spacer or something. In fact, there was 2. It just didn't appear to have anything there for some reason... it looked blank... Anyway, I just right clicked and went to remove from panel. They seemed to of been randomly placed in the space though, so I had to click around to find the right click options I wanted to get rid of them. All is good now!! 

The ONLY question I have now, is why did it before take 1375 pixels to get the job done, whereas now I have to set it to 1490 pixels to get the same effect? I guess I must be missing some objects from my toolbar... but I don't think I am... So, it works! But now I'm confused...
