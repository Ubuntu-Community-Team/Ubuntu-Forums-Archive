### Post by jhenager on 2007-03-26
Although this site says "Originally targeting highly experienced webmasters and programmers, it has since matured to a solution that is easier to setup for beginners", it looks like it might be a bit much for an Absolute Newbie.
This isn't as much of an Ubuntu issue as a Linux e-merchant issue. I would suggest looking on their forums for some help. You are more likely to get your specific questions answered there. Perl is a programming language, and you use it to write scripts. ASCII is a standard set of text, and a wrapper is software that usually is like a layer around other software. If you are interested, Google those terms to find out more.
But as a person that formerly did e-commerce support, I can say that setting up an online shopping cart is something that is a little advanced.
