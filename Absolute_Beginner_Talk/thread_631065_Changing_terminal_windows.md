### Post by macogw on 2007-12-04
In Edit > Current Profile in gnome-terminal you can go Effects and set a transparent background on the terminal.  There's also a colors tab for doing text/bg color.

### Post by mikewhatever on 2007-12-04
You can easily get pseudo transparency with the terminal, meaning your desktop background will be showing. In the terminal window, Edit>Profile, click to edit the default profile then go to the effects tab and tick Transparent Background. Move the slider to get more or less transparency.
Update: Actually, I got true transparancy in Gutsy, prbably because Compiz is enaibled.
