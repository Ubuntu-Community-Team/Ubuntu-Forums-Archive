### Post by oakgrove on 2008-03-03
This may seem extreme but one guaranteed way I can think of to keep wine from locking up your computer is to install virtualbox-ose from the repositories (sudo apt-get install virtualbox-ose), go to virtualbox.org and learn everything you can about virtual machines and then install Linux in VirtualBox as a virtual machine and use wine inside of that.  If it locks up your virtual machine, you can easily restart it.  Like I said, extreme but, it would work.

You can also install Windows in a virtual machine and always have it at the ready.  You can even convert your existing Windows installation with all of your documents, settings, and programs intact and run it inside of a virtual machine.
