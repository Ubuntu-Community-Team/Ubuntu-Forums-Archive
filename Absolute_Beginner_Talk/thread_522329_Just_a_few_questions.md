### Post by Cypher on 2007-08-10
Make sure you're downloading the Desktop version of Ubuntu. This is the Live CD version, when you put it into the drive and boot into it, you will get Ubuntu up without any installation.

Now, go through and explore everything, including your internet connection, printer access and your external HDD. If all that works, then you can consider installing it to the HD.

Be aware though, since Windows is already on there, you'll probably have to do some repartitioning to make some space on the HD for Ubuntu. There are many posts/documents that describe this process, so make certain you are completely comfortable with what you're doing before doing it..
