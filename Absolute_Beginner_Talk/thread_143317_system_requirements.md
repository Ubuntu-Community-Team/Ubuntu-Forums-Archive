### Post by Sef on 2006-03-15
If you find Ubuntu a bit slow, then you probably should do a server install and then install Xubuntu: a light weight window manager.

To do a server install, just type server when you first start installing instead of just hitting enter.

To download Xubuntu from a server install, at the prompt type this:

sudo apt-get install Xubuntu-desktop
