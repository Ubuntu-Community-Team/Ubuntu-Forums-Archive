### Post by C.S.Cameron on 2007-08-12
To make a long story short, unplug your hard drives, insert your Flash drive and install Ubuntu from CD as you would to a hard drive.
Plug back in your hard drives. You may need to set bios to boot from USB first.
This install is just like running Linux from a hard drive just a bit slower.
