### Post by mark_g on 2006-11-19
You're right, the new player doesn't have this implemented anymore.
There is a solution to your problem, which is to use Opera. It's a great browser which allows you to disable sound in web pages.
Just uncheck it in "Tools" > "Quick Preferences" > "Enable sound in web pages".
