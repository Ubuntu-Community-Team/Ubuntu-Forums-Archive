### Post by chessercizes on 2007-07-08
Hey!

As long as you have the ntfs-3g drivers installed, ubuntu can read and write to NTFS drives.The ntfs-3g driver is STABLE now too, which is awesome.

Im not 100% sure how to install these manually, since I did it with automatix, so maybe someone else can help you on that front.

But for sure, if you have the ntfs-3g driver installed, ubuntu can read and write to an NTFS drive smoothly.
