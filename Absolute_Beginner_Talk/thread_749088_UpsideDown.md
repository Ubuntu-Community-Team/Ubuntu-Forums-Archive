### Post by aeiah on 2008-04-08
you wont be able to use an nvidia driver because ubuntu isnt seeing an nvidia graphics card, its seeing a virtual box graphics card. you wont be able to access your partitions because virtual box doesnt have extra partitions. you could, however, set up a virtual network and access networked storage (your windows partitions) in ubuntu. all in all its usually easier to check things out with the livecd, although both have their advantages.

you'll be able to install software on your vmware ubuntu and make changes, wheras with the livecd everything is lost upon reboot. the easiest way to instal software is through add/remove on the menu, or via synaptic (navigate to system>administration>synaptic) on the menu.
