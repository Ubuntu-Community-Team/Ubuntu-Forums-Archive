### Post by T700 on 2006-07-29
The best way is what works for you and the particular situation.  Most times, the program can be found in the repositories and then it is easiest to just use Adept, Synaptic, or Add/Remove.  The only exception is if you are installing something huge like a full desktop.  In that case, I'd use aptitude on the command line, as it will make removal much easier, if necessary.
