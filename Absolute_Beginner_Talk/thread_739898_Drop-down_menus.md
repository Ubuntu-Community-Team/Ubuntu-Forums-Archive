### Post by mcduck on 2008-03-30
> **Jongleur said:**
> Can you point me to a website that uses this?
It sounds a very laborious way of achieving what takes me seconds to do now.

Sometimes it takes a bit of work to get a good result ;)

There must be thousands of web pages telling how to do this (each with a bit different way to implement it, while the main idea of having links in a list and using CSS to change the list into a dropdown menu stays the same). Here's one that also has instructions how to do it: [http://www.cssplay.co.uk/menus/final_drop.html]("http://www.cssplay.co.uk/menus/final_drop.html")

The result can of course be styled to look like anything, simply using background images instead of just the element background colors can make a great difference.

Sure, doing it this way might take longer than creating a flash menu with a separate program made for creating flash menus, but if compatibility and usability means anything then it's worth the time it takes. Besides, after you've done it once you can just copypaste the code to every new project you do ;)

Personally, I wouldn't use flash-based menus as I want the pages I create to actually show in Google – making the site navigable for search engines shows important role in search engine optimization. And search engines do not support Flash..
