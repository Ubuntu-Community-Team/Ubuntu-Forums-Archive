### Post by BruisedQuasar07 on 2008-01-30
You can get a professional quality distro on CD from linuxcd.org or
from osdisc.com for under $2.

I would try the newest release of Damn Small Linux (4.2.2) It is 
an incredible little distro, the best of its kind. The project goal is
to keep the distro under 50mb. 

If you get the live cd version of both distros, you can auto install and run them from CD player and RAM without installing either  to your hard drive. Great for checking out distros. I have the last three Puppy
releases in live cd form. I am not impressed with Puppy Linux.

There is a reason Damn Small Linux is far more popular. I'd bet you would love it. The documentation is the best written I've seen in the
Linux world.
