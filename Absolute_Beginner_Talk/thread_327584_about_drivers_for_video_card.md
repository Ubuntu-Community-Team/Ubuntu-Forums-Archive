### Post by batholith on 2006-12-29
If you're running edgy checkout:

[[COLOR="Orange"]http://doc.gwos.org/index.php/Latest_Nvidia_Edgy[/COLOR]]("http://doc.gwos.org/index.php/Latest_Nvidia_Edgy")

if you're running Dapper see:

[[COLOR="Orange"]http://doc.gwos.org/index.php/Latest_Nvidia_Dapper[/COLOR]]("http://doc.gwos.org/index.php/Latest_Nvidia_Dapper")

This are great, step-by-step instructions..

or you can install a script that will automatically download and install Nvidia drivers for you at:

[[COLOR="Orange"]http://albertomilone.com/nvidia_scripts1.html[/COLOR]]("http://albertomilone.com/nvidia_scripts1.html")

If you're unsure about what "running" as root means (basically think of it as "administrator" mode in Windows), or how to install a .deb file, etc... a great place to read-up on things is:

[[COLOR="Orange"]http://ubuntuguide.org/[/COLOR]]("http://ubuntuguide.org/")

it's a long scrolling html page with tons of info, just do a text search for a topic and it'll most likely be on the page!
