### Post by sab0403 on 2007-07-03
In the beryl settings manager, click on the Desktop tab, then on the Cube as desktop option, then the Skydome tab.

Enable skydome (with a checkmark), then click browse to choose the background file you want. I'm not sure if the files have to be of a particular size or type - mine is a 2048x1024 jpg file.
