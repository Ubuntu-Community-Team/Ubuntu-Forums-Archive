### Post by frodon on 2005-10-31
If you're new to ubuntu and linux, I advice you [automatix]("http://ubuntuforums.org/showthread.php?t=80295"). It will install for you most of useful packages and also set the good repositories for that.

The package will which install all the needed features (including gcc) for compilation is build-essential.

Feel free to ask for more details, tell us what you need and we will tell you the packages which fit to your needs.
