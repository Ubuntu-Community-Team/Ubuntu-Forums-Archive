### Post by legend2440 on 2008-03-24
Check if you have the correct repositories selected
Go to System>Administration>Synaptic Package Manager then Settings>Repositories>Ubuntu Software

Make sure all four repositories except the last one (Source Code) are checked
Then Make sure a server to download from is selected and while you are there you can check under the Updates Tab to make sure all are selected except Pre-released updates

Hope this helps
