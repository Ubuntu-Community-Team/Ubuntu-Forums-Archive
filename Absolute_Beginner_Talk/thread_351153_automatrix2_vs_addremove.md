### Post by Extrapolation on 2007-02-01
Automatix2 is more a "combo meal" style menu that lets you set up some essential programs and codecs with a few button clicks.  Add/Remove includes every program that's guaranteed to be safe to install.  

tkjacobsen is right anyway...it's generally better to use Add/Remove or Synaptic.  You can get everything you need without Automatix or EasyUbuntu, and you'll learn Ubuntu a lot more thoroughly.

### Post by old_geekster on 2007-02-01
I find that Add/Remove and Synaptic are far more reliable than Automatix2.  As a newbie, I thought that it would be great to have a program like A2.  However, I found that it didn't do a good job of installing wine; error messages.  Synaptic worked far better.

I have only been using Ubuntu Edgy Eft 6.10 for a week and have already reinstalled it at least 7 times.  This is due to my lack of trouble-shooting skills.  It is easier to install than t-s.  So, I have done many installs using each method and find Synaptic the best option.
