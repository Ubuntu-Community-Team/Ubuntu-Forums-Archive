### Post by chochem on 2008-03-10
It's probably not PC to do wine suggestions in a newbie thread but all the BD clones in the repos are awful. They look like somebody threw up on the games interface. If you have any respect for the classics you'll get Marcin Liwak's [Boulder Remake]("http://home.deds.nl/%7Ebremake/"). It's a Windows version with the classic C64 feel (just updated for deeper colour and resolution), it has user created levels to last you a  lifetime of play and runs very well under [Wine]("https://help.ubuntu.com/community/Wine").
