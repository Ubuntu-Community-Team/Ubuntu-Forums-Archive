### Post by unarmedninja on 2007-12-25
i feel for you. i had some nvidia driver problems when i upgraded to gutsy, it was a pain to figure out. 

If you are using the drivers from the nvidia website you need to remove all nvidia drivers from synaptic. this includes the linux-restricted-modules, nvidia-kernel, nvidia-glx and some others. you dont enable it in the restricted driver manager either.

are you using an lcd or crt monitor? that can effect your resolution if its not configured in xorg.conf properly.
