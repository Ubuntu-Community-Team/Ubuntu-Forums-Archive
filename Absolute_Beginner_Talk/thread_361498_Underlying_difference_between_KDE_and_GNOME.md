### Post by jml on 2007-02-14
I hope this is a real question and not a troll for flame bait, but here is my opinion.  KDE seems to have more things that are able to be customized than Gnome.  Some people have suggested that it is overly complex, but I never found that to be the case.  It seems to be more "Windows-Like" than Gnome which some see as a benefit, others as a disadvantage.  The K-apps seem to be more tightly integrated and have more offerings that Gnome, K-Office for example.

Others feel that Gnome has been dumbed dowm too much, (Linus Torvolds in particular,) in an attempt to make the interface more user friendly.  Again, I don't see it that way.  I actually prefer Gnome over KDE and its not based on any objective data, just my personal preference.  But it might be easier to let the developers speak for themselves.  I've listed the URL's for both the KDE and Gnome home pages.  Enjoy.

[http://www.kde.org/](http://www.kde.org/)

[http://www.gnome.org/](http://www.gnome.org/)
