### Post by eye208 on 2007-11-09
The problem with third party tools such as Envy is that after you have used them, Restricted Drivers Manager may not work properly any more.

I'd recommend doing a clean install and then using Restricted Drivers Manager to enable the Nvidia driver. It will probably save you a lot of headache.
