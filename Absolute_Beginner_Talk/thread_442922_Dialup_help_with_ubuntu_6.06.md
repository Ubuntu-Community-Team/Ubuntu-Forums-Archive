### Post by Quintin Riis on 2007-05-17
"softmodem".  Exactly.  These aren't real modems; instead of having an onboard DSP that work is offloaded to the CPU.  This amkes them cheaper to make, but harder to write drivers for.  It's not that someone couldn't make linux drivers for winmodems, but it's hard when the manufacturers will not release technical specs.  

Might try [http://www.linmodems.org/](http://www.linmodems.org/)

Otherwise you'll need to get a "real" modem that is supported by linux.
