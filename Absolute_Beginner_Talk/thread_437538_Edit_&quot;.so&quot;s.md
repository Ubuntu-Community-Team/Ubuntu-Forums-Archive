### Post by freebird54 on 2007-05-08
Just pick it up from the repositories - it is in there!   

Open System->Administration->Synaptic Package Manager
enter password
click search
enter "hexedit"
when found, click little box and 'mark for installation'
click Apply

use program  (I suggest typing 'man hexedit' to begin with - it is a 'primitive' editor!)

OK?
