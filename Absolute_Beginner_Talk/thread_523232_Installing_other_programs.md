### Post by SOULRiDER on 2007-08-12
Ubuntu is GNU/Linux which means, anything made for Linux runs, things are made for Linux, not ubuntu (except maybe some configuration application)

If you ever want to install a apckage not in the repositories, the ebst idea is to look for a deb file (deb files are that the package manager downloads and installs). If you cannot find a deb file the only  thing to do is to create one by compiling the program and packaging it. Do not worry though, sompiling software is something that you will rarely have to do.

EDIT: Check this wiki entry, it explains about compiling a bit
[https://help.ubuntu.com/community/CompilingSoftware](https://help.ubuntu.com/community/CompilingSoftware) and this one about package management:[https://help.ubuntu.com/community/SoftwareManagement](https://help.ubuntu.com/community/SoftwareManagement)
