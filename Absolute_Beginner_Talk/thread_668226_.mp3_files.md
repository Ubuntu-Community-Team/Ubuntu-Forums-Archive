### Post by zvacet on 2008-01-16
```
gsudo gedit /etc/apt/sources.list
```

Add this line to your source list 
[B]
deb [http://packages.medibuntu.org/](http://packages.medibuntu.org/) gutsy free non-free[/B]

Save and close file.

```
sudo apt-get update
```

After that go in synaptic and find w32(or 64  if you use 64 bit version)codecs and install them.
