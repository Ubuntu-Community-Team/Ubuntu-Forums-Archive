About those directories,  Where are they?  They may be files that you are not supposed to touch unless your a super user.  If they are yours, aka, in your home directory then you can change them with this command...
> pwd
to see where you are and make sure your in /home/your username/ somewhere...
> sudo chown -R username:groupname directory
the sudo gives your superuser rights... chown means "change owner" -R means recursive...(that directory and everything in it) username should be yours... like shaun group in this case should be your username too, all users get there own group. and lastly the directory is the location of the directory to change.

To get the pen working, you just have to download the wacom drivers from synaptic. Do a search for "wacom" in synaptic, and you should see them.

For a windows journal replacement, there are a few options. see this thread ([http://www.ubuntuforums.org/showthread.php?t=170898&highlight=pdf+annotator](http://www.ubuntuforums.org/showthread.php?t=170898&highlight=pdf+annotator) ), and check out jarnal. I havent installed it, so im not sure how good it is. There is also Xournal, [http://xournal.sourceforge.net/](http://xournal.sourceforge.net/) , which I installed and it works great, with a few superior features to journal (including the ability to save as pdf, and annotate pdfs), but also a few shortcomings. 

Unfortunately I haven't found a tablet onscreen keyboard utility, but I know that a few are in the works. And I also don't know how to change the screen layout when you're in tablet mode.
