### Post by atomkarinca on 2007-08-03
> **Dennis Shipman said:**
> Another responder said that, not I. But I do not understand your response. Pls explain, thx.

this is how i understand your situation: you have some files, you downloaded a third-party software and you want to open those files with that software on double-click by default.

i will symbolize these now: i have mp3 files, i want to open them with audacity by default.

i open a folder that contains an mp3 file. for example on my desktop i have an mp3 file and i click places > desktop. then i select that file and right-click it. i select properties and another window pops up. i select the "open with" tab and a list consisting of a bunch of software names appear. then i see that my software (audacity in this case) is not on that list. i click add and audacity is not on that list, either. i click on "use a custom command" and an input box appears. i type
```
audacity
```
in that box. then i select add, then close.

this should cover it.
