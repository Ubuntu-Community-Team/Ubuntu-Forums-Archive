### Post by mssever on 2006-09-01
I don't know of anything for Linux that will work for serious Hebrew study (I wish I did--I'd use it myself; I have a minor in biblical languages). If it doesn't work in crossover office, you might not be able to make it work. You can try wine and Cedega, but I don't hold out a lot of hope. Your best bet, if you want to get away from dual booting, would be to run a virtual machine (using simething like the VMWare player). This allows you to run Windows (the real deal) inside a window on your Linux desktop. And all Windows software should run that way.
