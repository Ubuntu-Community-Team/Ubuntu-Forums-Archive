### Post by Bliepo32 on 2007-10-19
The screen where you have to choose your operating system is called GRUB. Just select the one you want with the arrow keys, and then hit ENTER. If you want to know more about GRUB, you can read [http://ubuntuforums.org/newreply.php?do=newreply&noquote=1&p=3574218](http://ubuntuforums.org/newreply.php?do=newreply&noquote=1&p=3574218)

### Post by Arthur Archnix on 2007-10-19
Strange, since a default install will use Ubuntu as the default. Sounds like Grub may not be installed to the MBR. Simplest answer is just to do the install again, and when it comes to the part asking you whether you want it in the MBR, say yes.

There are other ways. But since you just installed this seems easiest.
