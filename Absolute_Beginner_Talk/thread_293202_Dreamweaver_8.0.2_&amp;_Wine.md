### Post by Rodneyck on 2006-12-17
I finally found a way to update Dreamweaver. You will need a copy running under Windoz, installed and patched. I copied the following three things, but not sure if all are really needed.

1. The c:/program files/macromedia/dreamweaver - copy all the folder
2. c:/program files/common files/macromedia - copied all of this
3. c:/documents and settings/username/application data/macromedia - unhide and copy all the folder

Just copy over using 1 & 2. #3 I had to create the whole path and then copied in the macromedia folder.  It worked, all updated under WINE.
