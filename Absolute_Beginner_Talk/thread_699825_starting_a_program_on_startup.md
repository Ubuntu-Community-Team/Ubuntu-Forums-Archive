### Post by ctswhole on 2008-02-17
> **abcdfv said:**
> i would like to have pidgin start automatically when ubuntu loads.
can anyone tell me how to do this?

I believe you can do this by adding it to your start up programs.  Go to System -> Preferences -> Sessions.  On your Startup Programs tab, click the Add button and enter "Pidgin" in the name box, and "pidgin" (without the quotes!) in the command box.  It should start up for you when you boot up your computer for now on.
