### Post by Metacarpal on 2006-09-15
That is a good initial test, but...

It's also possible to run Ubuntu quite smoothly on a computer that won't run the LiveCD.  Generally speaking, if you have a Pentium II or higher with at least 256MB of RAM, the LiveCD should work.  If you're running an older system, though, it's still quite possible to run Ubuntu.

What are your computer's specs (processor speed, amount of RAM, hard drive size, etc)?
