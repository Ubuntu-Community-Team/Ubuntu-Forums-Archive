### Post by BruisedQuasar07 on 2007-08-14
Have you tried running the live CD version or the newest live CD version Feisty Faun 7.04? 

I find there is a very inexpensive common newbie alterntive in the form of
high quality pre-burned distro live CDs. For a few bucks, you can get Ubuntu
7.04 from linuxcds.org. I prefer to order a few live CDs at the same time, as many as is covered by the minimum $3 priority mail postage charge. Each
CD is under $2...

Just boot the Ubuntu live CD and everything is done for you. Live CD distros
have an excellent auto hardware identifier and ram installer now. My Ubuntu, Xubuntu, Damn Small Linux, Knoppix, Accelerated Knoppix, Archie, etc all
loaded well on different systems from a year 2000 IBM Thinkpad to three
Dell desktops. 

I decide which distro I want, I dual boot installed Xbuntu on my Thinkpad and Ubuntu on my Dells. I had sound in Live CD mode but one of my Dells lost sound, after I installed a video clip player package (not Ubuntu team approved yet) so I knew if I remove the player package my sound comes
back. It is so easy to install Ubuntu 7.04 from the LinuxCds disc I just did
a clean re-install. Sound returned. 

Why not take the easy route? As you gain experience and skills, you can try
fixing a problem with terminal mode then. 

I can't get over how fast and easy, compared to Windows, a person can install the better distros, about ten minutes and a few more minutes to install 150 updates for every program and utility in the Ubuntu 7.04 package. Never, ever possible with Windows!
