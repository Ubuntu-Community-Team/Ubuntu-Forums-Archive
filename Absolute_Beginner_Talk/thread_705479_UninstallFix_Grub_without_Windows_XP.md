### Post by Nirevus on 2008-02-23
A tutorial for solving this problem can be found [here]("http://ubuntuforums.org/showthread.php?t=508927"), although that may require a XP CD.

If it does, but you can still download and burn CDs you'll want to get hold of [SuperGrub]("http://supergrub.forjamari.linex.org/"). SuperGrub is a boot CD for fixing a lot of boot issues. Just use the 'Fix Windows Boot Loader' option and restart when it's finished.
