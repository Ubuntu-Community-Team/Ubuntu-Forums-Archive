### Post by Kobalt on 2006-08-25
Hi,

in a terminal, run this command : 
```
sudo dpkg-reconfigure -phigh xserver-xorg
```

Make sure to select the resolutions you want avaliable with the SPACE BAR and then, restart X by pressing CTRL+ALT+BACKSPACE.

Cheers !

### Post by towsonu2003 on 2006-08-25
try putting in the horizontal and vertical refresh / sync rates. I always find them helpful to make X find what resolution I want. I google to find the values, but a better behavior would be to call the manufacturer... Here is the appropriate part of my xorg.conf: 
```

Section "Monitor"
        Identifier      "Generic Monitor"
        Option          "DPMS"
        HorizSync       31.5-48.5
        VertRefresh     40-70
EndSection

```

[SIZE="6"]**[COLOR="Red"]Be careful, wrong refresh rates will physically break your monitor[/COLOR]**[/SIZE]

sorry for the weird look, but it's dangerous ;)
