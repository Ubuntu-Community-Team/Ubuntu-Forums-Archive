### Post by ray bot on 2007-09-06
I don't think you did anything wrong.  Those files are what you'd normally find on a dvd.  The dvd's root contains folders called audio_ts and video_ts.  Most dvds don't use the audio_ts folder, but the video_ts folder should contain a bunch of files named vts_xx_x.vob, vts_xx_x.ifo, and vts_xx_x.bup.  The .ifo and .bup files contain index data that tells your dvd player where each chapter/scene/etc. is located.  The .vob files contain the actual video.

### Post by bulletxt on 2007-09-06
if your iso is a dvd movie iso, you can play it with AcetoneISO2.

Simply go under "play" tab and select your iso from there. then choose in which player to see it.
