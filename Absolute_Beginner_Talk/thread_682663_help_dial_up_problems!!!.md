### Post by AnonCat on 2008-01-30
If you're using an internal modem which you obviously are there's a 99.9% chance it's a winmodem and you're going to have to find drivers to make it work.  If you're using a Connextant or a Smartlink that won't be a problem even though setting them up sometimes poses difficulties.  Go to the website linked below and download their "scanmodem" utility and read the results of the modemdata.txt file or post them here to find out which drivers you will need.

[http://linmodems.technion.ac.il/](http://linmodems.technion.ac.il/)
