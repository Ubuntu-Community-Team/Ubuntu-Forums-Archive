### Post by justin_c18 on 2007-01-14
Not quite clear on what you mean, but i'll take a stab at it.
You've installed amsn, but you want to change the skin, and install some plug-ins?

First of all, fire up the terminal, then type:

```
gksudo nautilus
```

input your password. (do not tell anyone your password!)
which will bring up your file browser with root privileges navigate to
file system>usr>share>amsn

You'll see two folders, plugins and skins. So uncompress the plugin(s) or skin(s) and put the selected files in the right folders. Then once you've done that successfully, start up amsn, go to;
 Account>Select Skin or Account>Select Plugin and enable either or.

Hope that helped!
Cheers,
Justin
