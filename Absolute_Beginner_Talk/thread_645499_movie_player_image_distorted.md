I cannot fix the problem completely, but I can tell you how I get around this issue.  There is a plugin for firefox called MediaPlayerConnectivity that lets you launch any embedded video to an external player including VLC.  

The link for it is here [https://addons.mozilla.org/en-US/firefox/addon/446]("https://addons.mozilla.org/en-US/firefox/addon/446")

Hopefully someone will be able to give you a more complete solution that integrates VLC into firefox, but this is the best solution I have found.
