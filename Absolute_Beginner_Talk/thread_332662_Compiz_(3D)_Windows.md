### Post by wayward on 2007-09-12
Saša does have a valid point but he takes it too far for my liking.

  I've been using virtual desktops for years now.  I have no need for a taskbar anymore.  In my world, a taskbar is an unnecessary reminder that's just taking up screen real estate. I group my windows on virtual desktops according to the function they perform: desktop one is for my editor, terminals and whatever I'm currently working on; desktop two is for browsing; desktop three is for my e-mail client; desktop four is for one-off things and easy desktop access.

  Many computer users are using the taskbar and I am not trying to make a point *against* using it.  I merely wish to point out that while not all desktop effects are useful, some of them indeed may help increase your productivity.  Compiz-fusion's 'ring switcher', for example, and the ability to group windows are two that I use heavily.  I find them very intuitive and helpful.  For that reason I object to the total dismissal of Compiz and Beryl as being nothing more than "eye candy" -- they *can be made* useful, if you keep an open mind.

  My advice to anyone willing to try the new desktop effects is: don't think of them as being only for show.  Try them one at a time and see which of them, and how, will help you to get your day-to-day work done more easily and more efficiently.
