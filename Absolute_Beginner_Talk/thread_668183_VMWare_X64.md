### Post by descentspb on 2008-01-15
Mate, you can install a 64-bit VMware on 64-bit Ubuntu and you will be able to install 32-bit and 64-bit OSes like Windows on it. You can even install 32-bit VM on 32-bit Ubuntu and install a 64-bit guest operating system there.
VMware is proprietary software and there is a version for 64-bit linux systems. If you want a warez link, that is definitely not the place to search for it.
