### Post by Perfect Storm on 2006-03-31
Found your monitor on google, check if it's correct: [http://www.acer.co.th/product/peripheral/monitor_files/CRT/AC713/index.htm](http://www.acer.co.th/product/peripheral/monitor_files/CRT/AC713/index.htm)

If so:
```
sudo gedit /etc/X11/xorg.conf
```

Change
```
Section "Monitor"
	Identifier	"Acer AC713"
	Option		"DPMS"
EndSection
```

to

```
Section "Monitor"
	Identifier	"Acer AC713"
	Option		"DPMS"
        HorizSync 30 - 72
        VertRefresh 50 - 160
EndSection
```

save and exit
Reboot

In your Xorg config file you need to edit the following to add the resolutions you want;

	[I][COLOR="Navy"]Depth		1
		Modes		"1024x768" "800x600" "640x480"
	EndSubSection
	SubSection "Display"
		Depth		4
		Modes		"1024x768" "800x600" "640x480"
	EndSubSection
	SubSection "Display"
		Depth		8
		Modes		"1024x768" "800x600" "640x480"
	EndSubSection
	SubSection "Display"
		Depth		15
		Modes		"1024x768" "800x600" "640x480"
	EndSubSection
	SubSection "Display"
		Depth		16
		Modes		"1024x768" "800x600" "640x480"
	EndSubSection
	SubSection "Display"
		Depth		24
		Modes		"1024x768" "800x600" "640x480"
	EndSubSection[/COLOR][/I]

As you can see it only lists three resolutions ("1024x768" "800x600" "640x480") for each colour depth so that is why you only have the choice you have.  

IMPORTANT!!!!  Before you edit your Xorg config file make sure you BACK IT UP!  If you make a mistake with your editing you will not be able to get into the GUI.  I know, I have done it!  ](*,)

### Post by Jason_25 on 2006-04-03
Ok this should be easy to fix.

Comment out DDC in the module section and also comment out DPMS in the monitor section.  

Also you may need to add
```

Option "IgnoreEDID" "true"

```
and
```

Option "NoDDC" "true"

```
to the monitor section.
