### Post by zhoux on 2006-06-30
it is all personal preference, if you want a fast XP and full resource, then create a separate partition. However the down-side of it is that you have to reboot everytime you wish to switch between OS'es. 

If you want to avoid rebooting, and you computer is decent enough to run a virtual machine, and you have a windows xp installation disc with you, then i suggest run Win XP inside Ubuntu

The best software to use is VMware, and they have a free VMware Server....there is a thread here somewhere about how to install it.

Hopefully that helps
