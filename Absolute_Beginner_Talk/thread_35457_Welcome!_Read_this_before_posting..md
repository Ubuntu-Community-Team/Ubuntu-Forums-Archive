I'm a recent convert from Mandrake to Ubuntu.

One of the most beatiful things I noticed when installing Ubuntu is that there's actually NO 'root' password.  You cannot login as 'root' since its password field is disabled in /etc/shadow (check it out).  Instead, Ubuntu uses 'sudo' to temporarily give the main user root privileges.  During installation, I was asked to supply an administrative password for _myself_, not for 'root'.  This turned out to be not only simpler (one less question), but also more secure.

Just two days later someone was trying to break into my new Ubuntu box from an IP in Korea. I was watching the system logs as they were trying to guess the root password to no avail.  The box simply wouldn't allow the user called 'root' to get a login session.

After that I secured the box (through /etc/hosts.allow and /etc/hosts.deny) to not allow any ssh connections except from a hadnful of particular IPs I use, but the no password for root was one wonderful feature I happened to discover.

I have been using Linux since 1994 (Yiggdrasil, Slackware, then Red Hat, Then Mandrake) but Ubuntu is by far the best distribution I have ever used. :KS

It looks as if gadwin is a Windows-only program. You might be able to get it running using a utility called WINE, but that's a bit beyond my ken right now. The Ubuntu wiki has some information on WINE [here]("https://wiki.ubuntu.com/Wine").

Alternatively, there's a program called [KSnapshot](http://docs.kde.org/stable/en/kdegraphics/ksnapshot/index.html) that you might find useful. You can install it using the Synaptic Package Manager.

(If you haven't used the Synaptic Package Manager before: Go to System > Administration > Synaptic Package Manager. You'll be asked for a password, which is the password you created when you installed Ubuntu. Search for "KSnapshot". You'll see an entry titled "KSnapshot", along with a brief description. Click the checkbox on the left, select "Mark for Installation" on the menu that appears, then click the "Apply" button. Synaptic does the rest.)

> Also do I still need to install anti virus and firewall for my ubuntu? I told you I was new to Linux so I am flying by the seat of my pants and I am enjoying every second of it.

A lot of opinion indicates (see [this thread](http://www.ubuntuforums.org/showthread.php?t=133220) for example) that a firewall's not needed in Ubuntu, since all the vulnerable ports are turned off by default. If you're a little paranoid like me, there's a pretty good firewall called Firestarter you can install (again using Synaptic) to put your mind at ease.

AFAIK, there are no viruses or malware that run on Linux at this time -- one of Linux's great advantages; the only reason you might want an anti-virus program is to prevent sending on viruses to friends running Windows (just because you can't be infected doesn't mean you can't be a carrier). If you want an AV, there's a program called clamav you can install (again, you can use Synaptic).

Hope this helps. I'm a noob too (about two to three weeks using Ubuntu now). The people on this forum have been great -- patient, and willing to offer a helping hand.

Hi Linds, welcome to the Forum!
It is easy to try Ubuntu because it comes as a 'Live CD' and if you boot the machine from this you get a fully working Ubuntu, running in RAM (a bit slow of course). You can configure it and even install things, but nothing gets onto your hard drive unless you deliberately take special action.

You can install Ubuntu alongside Windows (backup your windows data first though, just in case of trouble)(also defrag Windows well before beginning such an install) and choose Ubuntu or Windows at boot time (Dual booting).

Ubuntu is mostly GUI unless you really want to use command line anyway. Command line is not at all bad if you proceed carefully and maybe gently. It is very powerful, which is why people like it, and also why it is a good idea to go gently at first, anyway. 

Internet: no problem. Note that Firefox is opensource and available also for Windows.
Music:    some codecs such as mp3 will need to be installed after your CD install (for legal reasons)
Office tools: OpenOffice is a good tool, and is available in a Windows version.

Yes I would say Ubuntu would be a good thing for you to use, or certainly try. I am sure you will end up using Linux in the future from what you say.

I see that matlab is available for linux, although from the website it looks like a paid for program
