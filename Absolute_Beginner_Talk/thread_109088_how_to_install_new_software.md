### Post by lrmall01 on 2005-12-27
I'd recommend that you use synaptic to install stuff.

Go to the System menu, then Administration, then Synaptic Package Manager.  It should be fairly straightforward from there.

BTW, synaptic is a graphical front end to apt.

Basically you can use apt-get to install software packages from the Ubuntu repositories..
