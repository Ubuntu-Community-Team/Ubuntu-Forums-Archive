### Post by emarkd on 2008-02-11
As an aside, something you may want to try:

1.  Make a backup copy of your xorg.conf file.  Really, do it.  It's important.
2.  Edit the file and remove any mention of any resolution but the one you want.  For the modelines, remove the entire line.  For the modes under the Screen section, just leave the one you want.
3.  Save the file and log out.
4.  Restart X by pressing <ctrl><alt><backspace>

Of course, you'd never be able to change that resolution because your X server will only know how to handle the one you leave...

If your X server won't start now, press <ctrl><alt>F1, log in to the command line and copy your backup you made over the xorg.conf file you edited.  Then press <ctrl><alt>F7 to get back to your GUI.  Restart X again and see what you've got.
