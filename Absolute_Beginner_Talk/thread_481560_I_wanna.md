### Post by Motoxrdude on 2007-06-22
If you want to set an image as a background,  double click on the image and select 'Image>>Set as Wallpaper'. You can also get more backgrounds and themes from [http://gnome-look.org/]("http://gnome-look.org/").
To apply a theme that you have downloaded, go to System>>Preferences>>Theme.
The click 'Install Theme', locate the theme and press ok. Also, select 'Customize' to view all available themes and if you find one you like just select it and press ok.
