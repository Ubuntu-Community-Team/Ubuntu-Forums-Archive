### Post by matthew on 2007-03-17
From the Gnome menus:

System->Preferences->Keyboard

Select the "Layouts" tab.

Click "Add" and choose whatever layout you would like to have available.


The easiest way to switch that I have found is to right click on a panel, select "Add to Panel," and then install the "Keyboard Indicator" applet.
