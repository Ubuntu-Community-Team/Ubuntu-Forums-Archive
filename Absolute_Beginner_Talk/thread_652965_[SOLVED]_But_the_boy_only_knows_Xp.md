### Post by Crumpets and Jam on 2007-12-29
Scroll down a bit on [this]("https://help.ubuntu.com/community/SwitchingToUbuntu/FromWindows") page and you will find an article on making Ubuntu look more like Windows (e.g. with a task bar and Start Menu and such other things).

[This]("https://help.ubuntu.com/community/UbuntuEyeCandy") page contains details on how you can change Ubuntu's appearence and possibly make it look more interesting for your kid.

Great themes for Ubuntu can be found at [http://www.gnome-look.org/]("http://www.gnome-look.org/").

Good luck.

EDIT: If your son wants to listen to .mp3 files and watch DVD movies, your going to have to install the Restricted Formats package. Ubuntu doesn't come with all the required codex's by default because it is illegal in some countries and Ubuntu wants to be available to everyone and anyone. Installing this package is very easy. You will even be able to use Microsoft fonts such as Times New Roman and Wingdings. There is a page [here]("https://help.ubuntu.com/community/RestrictedFormats") that explains everything.

Sorry for all the links but they are very helpful and all on the same site.
