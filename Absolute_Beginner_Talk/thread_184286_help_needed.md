### Post by tkjacobsen on 2006-05-29
Windows executables can be run using wine ([https://wiki.ubuntu.com/Wine](https://wiki.ubuntu.com/Wine)) but you shouldn't play with that, before you get to know linux a little better. 

Instead try to find a program that fills your needs  with synaptic: system -> administration -> synaptic package manager. Also it might be necessary to add extra package repositroys (form where you can install extra applications etc.): 
[https://wiki.ubuntu.com/AddingRepositoriesHowto](https://wiki.ubuntu.com/AddingRepositoriesHowto)
