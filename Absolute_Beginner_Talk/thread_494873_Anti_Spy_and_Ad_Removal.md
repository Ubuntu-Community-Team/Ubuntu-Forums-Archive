### Post by mlentink on 2007-07-07
Go System>Administration>Systemmonitor

Under Graphs (tab): look at your CPU utilization. Is it fairly high? Look at your memory usage. Is it way up? 

Then take a look under Processes (tab): Any unexpected processes? 

It is highly unlikely that there would be spyware or adware present in your system, because basically the only way they could be there is if you explicitly installed them, which you didn't

### Post by orb9220 on 2007-07-07
> Is there a Program out there for Ubuntu that will scan the entire system and clean out ads and spyware? 

No because there is no spyware adware in linux.

More than likely it is a recent unfriendly app or package installed or you have something not configured right.

You need to remember exactly when things started to slow down.  Was it when I install the video drivers,Compiz,Beryl,Java? some application?

Was it after a major system update?

What is running on startup? Does it slow down only when I am running Firefox,Azures,etc..

Does it return to normal if I turn off Desktop effects,Beryl,etc...?
