### Post by monte48lowes on 2008-02-22
I have had file permissions prevent me from completely deleting files from USB based drives. I use a terminal and manually delete the files using 'sudo rm'. This overrides the file permission issue and deletes the file. The files are most likely located on the USB drive in a .Trash* folder.
