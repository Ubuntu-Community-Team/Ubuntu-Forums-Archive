### Post by jem7v on 2007-04-10
If you want to Install Ubuntu instead of just trying it out, you could use the "Alternate Install" CD which doesn't boot up into a full GUI... it just installs it through a text interface.  It's not that confusing in comparison, but it doesn't look as pretty while you're installing it. :)

However, with the new download page on the Ubuntu site the Alternate install CD is getting increasingly harder to locate.  If you start at the [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download) page, you won't see the option, so you have to follow the link at the bottom of the page to the "complete list of download locations."  From that page, find a suitable server for you and go to their page - for example, perhaps this one:
[http://mirrors.csumb.edu/ubuntu/](http://mirrors.csumb.edu/ubuntu/)
Choose what version you want (i.e., 6.10) and go to THAT download page, where you will be presented with Many types of install CDs to download. Get an Alternate Install CD from the middle of the page, in either the 64 or 32 bit (aka i386) version, depending on your preference, and install from that if you can't get the LiveCD to boot at all.
