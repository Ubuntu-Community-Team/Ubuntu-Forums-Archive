### Post by x1a4 on 2007-03-15
Hi,

Menu -> Settings -> Desktop Settings

Ensure 'Allow Xfce to manage the desktop' is checked

click the 'Behavior' tab

Look under **Desktop Icons**

The drop-down list allows you set set what to show on your desktop: Nothing, Minimized application icons or File/launcher icons. 

Now set the icon and font size and click 'Close'.
