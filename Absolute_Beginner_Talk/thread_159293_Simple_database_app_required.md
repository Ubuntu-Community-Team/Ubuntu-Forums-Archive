### Post by garner_nc on 2006-04-12
MySQL. 

Probably the most commonly used open-source databse. Used to power a large percentage of web-sites. It's not very resource hungry. Easy to install and configure. In my experience,OpenOffice Base is o'kay for simple flat file database design but not much else. MySQL gives you a LOT more flexibility. This should run pretty well on your Presario.

You can set-up Open Office base to connect to your MySQL server and access your various databases and tables. This gives you an easy interface for designing tables and queries. This set-up is not useful for web database design though.
