### Post by nanotube on 2007-07-01

> Every once in a while my computer freezes.  I'm using beryl so that's probably the problem.  Any way to handle this without restarting?  Equivelant to ctrl+alt+del from windows?

ctl-alt-backspace should restart X. though if the window manager is hung up real good, it may not help.

also, ctl-alt-f2 can bring you into a text console, from where you can kill restart X. but again, if it's hung up real good, it may not help. the only solution would be to hit the power button, then...
