### Post by RomeReactor on 2007-01-20
If you're not used to the command line, this will be a heavy amount of reading to do, but if you're up to it type in the terminal:

```
man cdrecord
```

```
man mkisofs
```

```
man growisofs
```

this will invoke the manual pages for those programs. I suggest you scroll to the "EXAMPLES" section in each case. If the man pages don't come up, then you need to install those programs 

```
sudo apt-get install NAME_OF_PROGRAM
```

I don't know if this is what you're looking for, though...
