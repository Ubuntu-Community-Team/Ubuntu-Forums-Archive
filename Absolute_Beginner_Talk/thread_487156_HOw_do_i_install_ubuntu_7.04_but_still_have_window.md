### Post by panther_sn on 2007-06-28
Hi, I have recently done this and, I havent been back to windows since :)
Anyway I will assume you have downloaded and burnt the ISO image, or that you have an ubuntu cd, Well the easy way is load up ubuntu using the CD, Just restart ur computer with the ubuntu cd in the drive.

once the desktop has loaded, Click the install button on the desktop, and let the installer run its course answering the questions it asks as you go.

Now you may have a problem once you get to the part about partitioning, and where you install will depend on whether or not you have a second hard drive, or an already partitioned drive, best thing I can suggest is have a read of [Installing Ubuntu]("https://help.ubuntu.com/community/Installation") and [Graphical Install]("https://help.ubuntu.com/community/GraphicalInstall")  and because you want to dual boot have a look at [Windows Dual Boot Install]("https://help.ubuntu.com/community/WindowsDualBoot")

Hope that helps, and don't hesitate to ask any questions here.
