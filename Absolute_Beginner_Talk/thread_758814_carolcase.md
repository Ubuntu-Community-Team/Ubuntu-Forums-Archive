### Post by racoq on 2008-04-18
Aren't you using evolution? I think it renders HTML emails by default.

EDIT:
You can check it out in Evolution:

Edit > Preferences  > Tab Mail Preferences

Within Mail Preferences you&#8217;ll want to navigate to the second tab, HTML Mail. 


enable show image animations.
enable load images from the internet

After you have these configured you&#8217;ll want to navigate to the next tab, Composer Preferences.
Make sure that Format messages in HTML is turned on.
