### Post by tht00 on 2006-10-07
Hmm... if you don't mind me asking, what's wrong with just customizing Ubuntu?

Automatix can setup Ubuntu to support *almost* every audio and video codecs (mp3, m4a, wma(though why anyone would want that one is beyond me)).  

As for media players, just install some and see what you like.  Personally, I like the default Rhythmbox player better than anything else I've tried (I see it as a mix between Winamp and iTunes).

And for ripping music, what's wrong with Sound Juicer? (default app)  I believe it can be used to rip mp3.  Really, if you don't need portability, OGG is a wonderful format and kicks mp3 around IMHO. [http://www.vorbis.com/faq/#what](http://www.vorbis.com/faq/#what)

If you have the free space to spare and want a loss-less codec, FLAC is a nice format for that.

Oh, and Sound Juicer also downloads tag info from the internet.  I've only had a few CDs not tag correctly (and that was expected, as uncommon as the CDs were).  I think most media players do this anyway.

And if you really want a more 'stripped down' distro, you know, you can remove things you don't want.  Ubuntu ships with a pretty basic install.  If you want to remove things (like Open Office), just uninstall it from Synaptic.

Really, everything you've mentioned can be setup on Ubuntu.  Looking for a different distro with everything customized the way you want it 'out-of-the-box' is unrealistic.  Finding a solid, capable distro to customize is where it's at.
