### Post by aysiu on 2007-02-17
> **adza said:**
> aysiu, how can i bind that gksudo to an icon in the panel then? is that possible, i would prefer to launch apps by clicking on an icon than by [ALT]+F2 etc...
Right-click on the panel. Create a custom launcher. In the *command* line, put the command ```
gksudo nautilus
``` Select whatever icon you think appropriate.

You can also create a keyboard shortcut for it, too (I used to have Control-Shift-N for this):
[http://doc.gwos.org/index.php/GnomeKeyboardShortcut](http://doc.gwos.org/index.php/GnomeKeyboardShortcut)

Pay attention to method B in particular.
