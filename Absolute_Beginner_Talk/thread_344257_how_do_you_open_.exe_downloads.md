### Post by Henry Rayker on 2007-01-22
The absolute best way to install is through the repos (but not everything will be in there). If you ever need to install something, you should check synaptic anyway. (Assuming you're using GNOME) I don't recall exactly where it is in the menus (I don't use GNOME anymore) but it should be in System->Administration->Synaptic Package Manager

Once you open this, use the search function (you can enable more repositories by clicking Settings->Repositories and checking every box on the first page). Most things you'll need are there. These things will be eligible for auto-update and are all free (in terms of money).
