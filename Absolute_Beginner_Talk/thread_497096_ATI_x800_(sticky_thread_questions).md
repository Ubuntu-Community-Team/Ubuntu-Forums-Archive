### Post by Happy_Man on 2007-07-09
Right, I see what they're saying. OK, first..... get the alternate install CD from [http://ubuntu.com/download](http://ubuntu.com/download). Down at the bottom, there's a checkbox that reads "Check here if you need the alternate desktop CD. This CD does not include the Live CD, instead it uses a text-based installer." Check the box, select Ubuntu Feisty, and download, burn to CD, boot, and install. From there, follow the directions given.

The reason you can't use a LiveCD is because the LiveCD wouldn't give you a GUI to work with. So you need the alternate installer to work with. Don't worry, it's every bit as friendly as the GUI. Just older-looking. Don't sweat it!
