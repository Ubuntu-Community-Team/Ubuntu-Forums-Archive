### Post by go2dell on 2007-03-22
If Ubuntu runs fast enough to suit you, then I'd say stick with it.  The main reason to choose Xubuntu is that it contains lighter applications and XFCE isn't as resource-hungry as Gnome.  With Xubuntu you can still install Gnome apps, but you'll end up undoing many of the benefits of having Xubuntu in the first place.

If you're really worried about performance, also take a look at [Fluxbuntu]("http://fluxbuntu.org/").

As for BitTorrent clients, opinions vary far and wide as to the best.  [This thread]("http://ubuntuforums.org/showthread.php?t=387120") might get you started.
