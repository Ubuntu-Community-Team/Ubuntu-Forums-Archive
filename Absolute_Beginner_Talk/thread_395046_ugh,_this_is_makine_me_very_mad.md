### Post by Pikestaff on 2007-03-27
mp3 is a restricted format, if you haven't already enabled restricted formats you won't be able to play mp3s.  I think newer versions of Amarok should automatically detect and download mp3 support when you first try to play an mp3, but older versions might not...

Here's some info on getting restricted formats (like mp3) to work: [https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)
