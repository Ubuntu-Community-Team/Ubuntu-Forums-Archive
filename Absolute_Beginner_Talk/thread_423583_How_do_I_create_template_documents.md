### Post by the8thstar on 2007-04-26
Hello all,

When I right-click on the main desktop, I have the possibility to create a new document. My menu says that I have no templates installed as of now. So my question is: how do I create template documents: pictures, sounds, OO docs, etc...?

---

### Post by guardsman85 on 2007-04-26
Create a folder called Templates in your home directory.  Save any templates you want to use to this directory.  They will show up in the context menu automatically.
