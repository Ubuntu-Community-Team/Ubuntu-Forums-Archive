### Post by qpieus on 2007-10-15
Navigator doesn't have to be installed at all. Inside the tar.gz file is a script, called navigator, that you need to run and the browser will load. So open up a terminal window and type```
./navigator
``` and hit enter.

You could also make a launcher for this script and place it on your desktop or in a menu.

### Post by jfinkels on 2007-10-15
> **sx66gns said:**
> ahh , ok - does this apply to all tar.gz file extensions?

thanks for the reply man.

No. .tar.gz is a compressed package file, like a .zip file which you may have used on Windows.

The .tar comes from the program called 'tar' which organizes the files into a single archive file, and the .gz comes from 'gzip', the compression program.

A file ending in .tar.gz can contain anything (usually, when we talk about .tar.gz files, though, we are talking about source code, which must be compiled before becoming executable).

BTW, you might like Mozilla Firefox if you've never tried it before; Mozilla used the old Netscape code to make firefox, and now Netscape is using the Firefox code to make Navigator!
