### Post by Najand on 2006-09-18
Evolution needs you to setup your Time Zone seperately than default gnome time zone... 
Check if you have the same time zone in:
Evolution Menu: Edit -> Preferences -> Calender and Tasks -> Time Zone
and select the place you live or the same place you set up your Ubuntu's
time.
