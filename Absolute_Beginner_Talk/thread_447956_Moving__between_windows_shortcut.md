### Post by Inxsible on 2007-05-18
If you have Beryl installed, you can switch on the 'Application Switcher' and then use F8 or F9 to switch between windows. This will line up all the open windows in rows and columns, and you can choose which one you want. (I think this is different from the wall plugin that seems to be hot these days in Beryl as I can't move applications to different workspaces using this switcher)
 
Then there is the Alt + Tab switcher in Beryl too. I think this is universally accepted and so Beryl supports it too :)
 
There is another application switcher (ring mode) in Beryl, which creates an Ellipse of all the open windows with the option of windows only from current workspace or all 4(or whatever number of) workspaces. This uses the Super+Tab key as shortcut.
 
And I hope I havent confused you too much !
 
In case I have, message back...and I shall put on some images to explain what I am talking about !

### Post by Inxsible on 2007-05-18
I am sorry I had Application Switcher and Scale confused. I was at work then and had no access to the Ubuntu machine.

There are actually two options.

Application switcher which i detailed before.

and something called Scale. Switch that on as well. Thats where the F8 and F9 keys work. This is called the window picker as opposed to switcher.

You can also change the shortcut bindings if you want !

F8 will allow you to pick applications among all worspaces and F9 only for the current workspace. And no you dont have to type in F and 9.... Just hit the F8 or F9 key
