### Post by lisati on 2007-07-25
One option is to click on "Thread tools" and then "Subscribe to thread". You will then be emailed regularly if the thread has been updated. You'll need to set up an email address @ User CP

EDIT: p.s. Yes, it is possible to bookmark threads. I also remember reading somewhere about threads not being deleted.
