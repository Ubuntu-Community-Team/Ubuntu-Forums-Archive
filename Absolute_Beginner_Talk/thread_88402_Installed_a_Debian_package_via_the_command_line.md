### Post by feathers on 2005-11-10
One good thing to try is to type the first one or two letters and then the TAB key. This shows you a list of commands with these two letters at the front. Imagine you installed kwrite, then type kw TAB and it shows you something like kwrite kwonder, kwater, (I am fantasizing here). 
In you case, you didn't install a bittorrent client, which I assume you wanted to. Try to install one of these packages: bittornado-gui, ktorrent, qtorrent. These programs download files via bittorrent.
Good luck.
