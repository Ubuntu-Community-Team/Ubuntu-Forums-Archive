### Post by chewearn on 2007-02-24
Look to the bottom right of your desktop.  By default, ubuntu already comes with two desktops set up.  (in ubuntu, it is called workspace, btw).

If you right-click on the "boxes", and select Preferences, you can configure for more workspaces.
