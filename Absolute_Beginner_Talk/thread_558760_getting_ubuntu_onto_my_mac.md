### Post by shearn89 on 2007-09-24
1. Make sure you download the .iso for the correct architecture (PPC or Intel - i386).

2. Check the md5sum of the download: on a mac you open a terminal and type "md5 *filename*" and it should print out the md5. Check it on the download page.

3. Burn cd at less than 4x speed to ensure no errors.

4. Insert cd and reboot, holding down c to boot from cd.

5. Follow the onscreen wizard to install...
