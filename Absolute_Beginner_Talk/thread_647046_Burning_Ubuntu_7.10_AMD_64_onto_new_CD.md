### Post by Mud.Knee.Havoc on 2007-12-21
Why can't you install from a DVD?  Maybe you have an error on the disc?

If you just burn an .iso file itself onto a DVD or CD, it's not very useful... it's just a single file sitting there that can't do anything (well, that's not true, but for the Absolute Beginners' Forum, we'll say it is).  You need to specifically tell it that you are burning an image file (programs all call it something different, but it'll be something like "Burn CD image" or "Burn DVD image"), and it'll transform the .iso file into a usable disc structure.
