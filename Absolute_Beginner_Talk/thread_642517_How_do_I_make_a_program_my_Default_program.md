### Post by mcduck on 2007-12-16
Pretty much always the executable file for programs is in /usr/bin. If you want, you can use the 'which'-command to find the executable for a program. For example running 'which firefox' would return '/usr/bin/firefox'.

You can also save the torrent file to desktop, right-click it, go to 'Properties' and set the program you want in the 'Open With'-tab.
