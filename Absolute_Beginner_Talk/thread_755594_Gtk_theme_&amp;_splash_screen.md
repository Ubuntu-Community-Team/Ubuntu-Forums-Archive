### Post by ad_267 on 2008-04-15
What problems are you having exactly?

Assuming you've downloaded a theme from somewhere like gnome-look.org you can install a gtk theme by going to System -> Preferences -> Appearance. Make sure you're in the theme tab and then find your downloaded theme and drag it into this window and it will install. You can then customise a theme and select the new gtk theme.

If by splash screen you mean the screen where you log in, you can download the gdm theme and go to System -> Administration -> Log in window and select the local tab and click add to add your new theme.

This is a good simple guide to customising the look of Ubuntu: [http://justanothertechblog.blogspot.com/2007/03/quick-guide-on-customizing-ubuntu.html](http://justanothertechblog.blogspot.com/2007/03/quick-guide-on-customizing-ubuntu.html)
