This usually suggests that either you have not used "sudo" before the apt-get update.
e.g.```
sudo apt-get update
```Or that you've got synaptic open somewhere else too.

gksudo gedit /etc/apt/sources.list

### Post by Bachstelze on 2006-08-27
Well, the only thing is that if some software you installed with Automatix gets updated and it was on an unofficial repo, apt won't see the updates since the repos are not there anymore.

In short, the sources.list file is the list of all servers (called epositories, or repos for short) where apt will search for packages to install/upgrade.
