### Post by Kasyx on 2008-02-07
Hi,

Good for you for wanting to try out Ubuntu. My suggestion, as for starters, would be to go with Kubuntu as the window manager it uses (basically the user-interface) is more similar to that of Windows. You should be able to find links to download it pretty much everywhere. All you have to do is burn it to CD and boot off it. There are hundreds of tutorials that will show you how to set up a dual boot (so that you can choose between Windows or Kubuntu when your computer starts up).

The sort of programs you will probably use:

Music:  Amarok
Chat (MSN/Googlechat, etc): Pidgin
Web: Firefox is available on Linux, as is Opera

So for those sort of activities you should be set. Gaming on Linux is possible with the help of a program called Cedega, though it takes a bit of effort to set up, though you should be able to play most Windows-based games.

### Post by TheDilettante on 2008-02-08
Kubuntu and windows managers - you know how your computer uses DOS? (if not, go to 'run' in the start menu and enter cmd.exe to see find it.)  DOS is really your computer's operating system, and Windows (c) is the visual interface through which you can control the computer.  In Windows, you never get the operating system without MS' corresponding graphical user interface.  This GUI is actually a windows manager.

Ubuntu is much like Windows, here: it is the graphical/visual interface between you the user and the operating system, which in Ubuntu's case is Linux.  The difference is that in LInux you can choose which program you want to be your GUI - Ubuntu uses Gnome, Kubuntu uses KDE, Xubuntu uses Xfce, to name a few - simply based on what looks good to you.

Here are some pics;

Ubuntu
[http://www.thecodingstudio.com/opensource/linux/screenshots/?linux_distribution_sm=Ubuntu%207.10](http://www.thecodingstudio.com/opensource/linux/screenshots/?linux_distribution_sm=Ubuntu%207.10)

Kubuntu
[http://www.thecodingstudio.com/opensource/linux/screenshots/?linux_distribution_sm=Kubuntu%207.10](http://www.thecodingstudio.com/opensource/linux/screenshots/?linux_distribution_sm=Kubuntu%207.10)

Xubuntu
[http://www.thecodingstudio.com/opensource/linux/screenshots/?linux_distribution_sm=Xubuntu%207.10](http://www.thecodingstudio.com/opensource/linux/screenshots/?linux_distribution_sm=Xubuntu%207.10)

And there are others at that website, too.

Printers, USB, etc - It really depends.  Ubuntu recognizes most USB devices, including the latest iPods (with a little tweaking) and many printers, too.  The best way to find out about yours is simply searching these forums for your devices.
