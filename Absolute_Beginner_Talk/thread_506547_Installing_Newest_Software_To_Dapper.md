### Post by aysiu on 2007-07-21
It sounds as if you're screwed.

You have three options:

1. Suck it up and use the older versions (what's so great about the newer ones?)

2. Use the Alternate CD to *install* Feisty instead of upgrading to Feisty (an upgrade that skips a release isn't likely to work anyway)... [make a separate /home partition](http://www.psychocats.net/ubuntu/separatehome) first.

3. Compile all those programs from source.
