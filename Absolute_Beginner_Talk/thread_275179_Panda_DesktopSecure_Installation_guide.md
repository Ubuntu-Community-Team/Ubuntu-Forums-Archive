### Post by Netsurfer38 on 2006-10-11
This is a quick guide based on my experience of installing Panda DesktopSecure for those that wish to have or try the program.

Firstly I had endless problems trying to install Panda DesktopSecure via Synaptic Package Manager, it would install, but nothing would seem to work (would not even accept the trial Username & Password). Also trying to uninstall it was a problem as well. All attempts came to nothing, even though they were made on a clean updated installation of Ubuntu.

Found that if I downloaded the DesktopSecure installation file directly from Panda's website and then installed it on a **fully updated Ubuntu**, it appears to work flawlessly. Another advantage of downloading it from the Panda site -  you get your trial Username and Password, which are required to be able to update the product.

To install, do the following:

First copy the downloaded file **desktopsecure.sh** to a known location e.g. **/home**

1.Applications
2.Accessories
3.Terminal
4.Enter the following command (if installer requires run permissions)
**chmod a+x /home/desktopsecure.sh**
6.Then enter
**/home/desktopsecure.sh**
7.Enter Administrator Password and follow all the steps through
8.Note: be a little patient at the end – for the **Finish** button to become active

N.B. When registering the program it requires the Root Password 
To create the Root Password one does the following:

1.System
2.Administration
3.Users and Groups
4.Enter Administrator Password if prompted
5.On **Users** tab check **Show all users and groups**
6.highlight **root**
7.double click **root** or select **Properties**
8.Enter root passwords

I am not debating the merits or demerits of having a commercial antivirus program; I'll leave that to others. Other than to say that now that I have installed the program, I do like its User Friendly GUI.
