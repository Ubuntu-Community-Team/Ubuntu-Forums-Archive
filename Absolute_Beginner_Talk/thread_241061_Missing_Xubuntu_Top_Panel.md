### Post by BitTorrentBuddha on 2006-08-21
First, try restarting X by hitting Control+Alt+Backspace and see if that brings it back.([color=red]WARNING: THIS WILL NOT PROMPT YOU TO SAVE THINGS[/color]). 

If that doesn't fix it then try adding a new panel. To do this, click on the existing panel and select new panel, then add everything you had before or everything you wish to have. By default (in Ubuntu) there is: Menu Bar, Application Launchers, Notification Area, Volume Control, and Clock, in that order. I'm not sure about what Xubuntu has.
