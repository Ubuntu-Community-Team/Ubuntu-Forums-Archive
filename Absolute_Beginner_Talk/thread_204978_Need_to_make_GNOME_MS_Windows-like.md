### Post by T700 on 2006-06-27
Personally, I've found that people I've introduced successfully to Linux fall into two groups.

Group One--Wants only to browse, play a few simple games, type some documents, and check email.  This group can happily adapt to most modern, graphical distros with about the same effort as moving from Windows 2000 to XP.  Ubuntu's graphical interface with a very few tweaks is spot on for them.

Group Two--Wants to do all of the above and is interested and motivated to learn the how and why.  Desires to do things on their own.  Likes to experiment.  Not easily frustrated or discouraged.  Ubuntu graphical + command line + these forums = happiness.  They soon forget "making it more like Windows" and just enjoy the ride.

Now, if I could get my "random lockup and crash" script and my spyware simulator to work correctly, Group One will never know it isn't a new version of Windows!  :-)
