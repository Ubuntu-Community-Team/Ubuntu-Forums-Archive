### Post by Ek0nomik on 2007-06-11
First thing first, have you installed your video card drivers?

If so, you should continue with my steps below... (if **not**, then post me your video card brand and #)

I presume you haven't messed around with your xorg.conf file?

If not, let's try to run a command to setup your file with high resolutions.

First, let's back up the file:

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
```

Now let's run a command to set the resolutions.  (Once you run this command, the terminal will become a dialog.  You will want to select your proper video card model on the first prompt, and use the space bar to select items.  (Ex:  Use the space bar to select resolutions 1280x1024 and 1024x768, etc)

```
sudo dpkg-reconfigure -phigh xserver-xorg
```
