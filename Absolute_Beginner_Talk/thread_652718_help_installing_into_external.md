### Post by Erwin Criddle on 2007-12-29
The first thing I would suggest is to not install Ubuntu on a External HDD.
There are many disadvantages on installing Ubuntu on an External HDD. e.g. Very Slow, Bios may not support booting from external HDD, many faults like this one.
If you can, try installing Ubuntu on an Internal HDD and see whether that works.
