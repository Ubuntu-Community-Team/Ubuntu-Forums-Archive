### Post by aysiu on 2007-01-12
There's VMWare, which you can use to run Windows XP within Ubuntu.

There's dual-booting, where you can choose what OS to boot into when you start up your computer.

There are KVM switches that allow you to switch between two computers with one button--sharing the same monitor, keyboard, and mouse--very handy if you have an old spare computer around.

Then there are some helper programs, but none are perfect for *all* Windows programs: Wine, Crossover Office, Cedega.

Your best bet is always to run a native Linux program.
