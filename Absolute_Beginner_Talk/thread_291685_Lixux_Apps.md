### Post by Aranel on 2006-11-02
It's a lot more simplified in Linux - there are these things called repositories that are basically like download.com.

First, enter "gksudo gedit /etc/apt/sources.list" (without the quotation marks) into a terminal. Remove all of the #'s from the beginnings of lines that begin with "deb," save, and close. What you've just done is activated all the major Ubuntu repositories.

Then go to System > Administration > Synaptic Package Manager, search for the application you want, check the box to install it, and hit Apply.
