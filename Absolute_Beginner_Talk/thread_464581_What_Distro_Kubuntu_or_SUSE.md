### Post by daimaru on 2007-06-04
before i switched to ubuntu i used suse. the reason i switched is that with suse (which uses red hat package manager--> RPM) dependencies are not resolved automatically. at least when i was using suse that was the case. so if you try to install a program you get annoying messages telling you that you need this and that library or other thing, and it wont install until you got it all there. in ubuntu (debian based; .deb) you have aptitude package manager or whatever its called. the thing i like about it is that it automatically downloads all the dependencies you need to install a program. so all you have to do is say ummmm install thunderbird and it gets all the stuff it needs automatically and thats it, no hassel everything works ...
EDIT: just in case ur still wondering .... i recommed ubuntu.
