### Post by rsambuca on 2007-11-28
There is NO sun-java plugin for the 64bit browser.

If you are that concerned about 64-bit only, then why don't you use gtk-gnutella instead of Frostwire?  It uses the same gnutella client, and it doesn't require java, so it is much quicker.  It is in the standard repos.

sudo apt-get install gtk-gnutella
