### Post by SactoBob on 2007-04-03
You can run the 32 bit version on the 64 bit machine.  I do that on my own Core-2 duo machines.  As for stability, the older version is generally more stable, but has less support for the latest hardware.  With a Core-2 Duo, you might try Edgy. I find Edgy very stable.
