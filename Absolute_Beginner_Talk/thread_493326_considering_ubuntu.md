### Post by rickycodie on 2007-07-05
yeah you sound ready. ubuntu is a great distro(most popular on desktops) and it looks like your specs are right. with ubuntu you should be able to run beryl although it might be limited in capabilities ( depends on your video card), and i think you would also be fine doing a upgrade to gutsy when it comes out. 

xubuntu uses xfce as a window manager, basically it is a lighter manager that requires less resources, so it's less pretty, but i've seen beryl on it before. so try it if you want. you can install it ontop of ubuntu with this command

sudo apt-get install xfce desktop

uninstalling is a bit harder. but shouldn't take more 1/2 hour.

The main difference between Xubuntu and Ubuntu are the desktop environments (Xfce and Gnome respectively). Xfce is somewhat lighter so you'll be able to squeeze more performance out of your PC but still be able to do everything you mentioned.
