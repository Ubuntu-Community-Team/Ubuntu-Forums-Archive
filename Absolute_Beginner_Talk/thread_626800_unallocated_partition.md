### Post by natehatewindows on 2007-11-29
i would use a gparted live cd, it will be very easy. you will have to make the unallocated space next to the partition you want to "grow" and then just right click the partition you want bigger and select resize (or something like that) and then you can make it bigger. you can also use garted on a live ubuntu disk but i think everyone should have the gparted boot disk...and if you have it then you should use it from time to time :)
