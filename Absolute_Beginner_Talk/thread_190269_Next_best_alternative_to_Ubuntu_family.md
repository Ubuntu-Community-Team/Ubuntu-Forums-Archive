### Post by Kimm on 2006-06-06
If you like KDE I suggest Mepis, its Ubuntu based and uses some of the same repos, but its more KDE centric (and probably better than Kubuntu at its current state)

If you like Gnome... I suggest you try Fedora Core, although some say that it isnt all that nice.... I havn't had any problems with it (besides networking due to the highly paranoid SELinux. Easily fixed though)
Fedora uses the RPM package manager though, sometimes described as "dependency hell" but yum fixes that to some extent.

PS.
I hear its possible to install the standard Ubuntu Gnome desktop using a Mepis system, so dump the Fedora Core idea and go with Mepis, the Gnome desktop is only a "sudo apt-get install ubuntu-desktop" away (+some download and install time...)

[b]But just to make sure: Copy ALL your important files to a safe location (preferably a USB memory stick or a CD) before you atempt to install Linux.
The installation can not remove your windows installation by accident, but its easy to reach the wrong settings and thus formating your windows installation, or rendering it useless

Consider yourself warned[/b]
