### Post by perdubug on 2007-12-24
Try this URL,[http://elboiler.p2pforum.it/server.met.gz](http://elboiler.p2pforum.it/server.met.gz), to update server list. It works on my Ubuntu 7.10 with aMule 2.1.3. Detail steps are:
1. Paste [http://elboiler.p2pforum.it/server.met.gz](http://elboiler.p2pforum.it/server.met.gz) into Network->ED2K->Servers;
2. Click the left button of 'Servers', you would get a server list;
3. Select a server you would like to connect and double click it;
