### Post by ramjet_1953 on 2007-03-03
Why not take a live CD along to the computer store and get them to run it on the system you like?

It will give you a good idea if it works with Ubuntu, or not.

Get them to plug-in the printer, scanner, etc. and it should recognise it all.

Don't forget, though, that screen resolution under the Live CD may not be optimal, as it is set to the default value of 1024 X 768.

If it has an ATI graphics card, you may have problems.
