### Post by chamberlain2006 on 2007-05-15
xubuntu-desktop is whats called a meta-packages.  It installs all of the base applications for Xubuntu.  Similar ones are ubuntu-desktop (for Ubuntu) and kubuntu-desktop (for Kubuntu).  Its ok to not have it installed.

---

### Post by bmk789 on 2007-05-15
Xubuntu-desktop is a meta-package similar to ubuntu-desktop and kubuntu-desktop.  All it does is install all the packages needed to run the default u/k/xubuntu desktop.
