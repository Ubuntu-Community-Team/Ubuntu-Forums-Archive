### Post by Shuja on 2006-11-18
PC-BSD is a more newbie friendly version of FreeBSD.  I use OpenBSD for a few servers but it isn't newbie friendly at all.  I normally use Redhat or Debian systems so I tend to lean towards CentOS for personnel servers.  I have been playing around with an Ubuntu 6.06LTS server install for my home network file sever and so far I haven't had any problems with it.  If you're already an Ubuntu user I don't see why using the server install would be a problem for you.  The support here is excellent for it.
