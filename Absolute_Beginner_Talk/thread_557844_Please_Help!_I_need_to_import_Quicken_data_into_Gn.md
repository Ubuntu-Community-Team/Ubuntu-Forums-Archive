### Post by mdebusk on 2007-09-23
> **Zilbert said:**
> When I open GnuCash It wants a "QIF" file. But my Quicken data seems to be contained in 6 files     
ending in .HCX .IDX .QFD .QEL .QIN and .QPH. all of which were not accepted. Can any one Help?

You need access to a copy of Quicken long enough to open your files and export them to QIF. If I recall correctly, that operation is under Quicken's "file" menu.

The best advice I got with this is as follows:

[list=1]
[*]export everything but the transaction data;
[*]import that into GnuCash;
[*]export the transaction data; and
[*]import the transaction data into GnuCash.
[/list]

Until I did that, I got lots of errors in the account balances.
