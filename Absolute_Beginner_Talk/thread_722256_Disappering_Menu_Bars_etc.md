### Post by glennric on 2008-03-12
I would guess you need to fix your xserver configuration.  Post the output of 
```
cat /etc/X11/xorg.conf
```
Then try 
```
sudo dpkg-reconfigure -phigh xserver-xorg
```
and see if that helps.  If it doesn't we will try to fix your file manually.
