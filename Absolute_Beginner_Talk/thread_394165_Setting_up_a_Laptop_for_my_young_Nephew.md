### Post by tommy1987 on 2007-03-26
I can offer no help on the parental guide.
However, It is very easy to install Ubuntu and Windows and have them dual booting. One peice of advice is to install Windows first, if Windows is installed last then it will overwrite GRUB Bootloader which is installed with Ubuntu (it is this which gives you the choice on boot up).

The GRUB bootloader which will be installed by default with Ubuntu and will automatically detect your Windows installation and offer the choice on startup.

Check this link for lots of screenshots and a very useful guide (although outdated it is much the same as the latest stable release (Edgy)):
