### Post by fedira on 2007-09-24
Your default download location in what? Firefox?

In Firefox, go to Edit > Preferences. 
In the "Main" tab, under "Downloads," change the location where it says "Save files to," or choose "Always ask me to where to save files" if you prefer. 

If you weren't asking about Firefox, please specify what you're asking about.
