### Post by Bufke on 2007-12-30
Ok, you can't install .exe files on ubuntu.  Typically drivers in ubuntu will just work without installation, however the Speedtouch modem may need special attention.  There is a How To here [https://help.ubuntu.com/community/UsbAdslModem/SpeedTouch]("https://help.ubuntu.com/community/UsbAdslModem/SpeedTouch")
Try that and see if it works, good luck.

Well, to begin with, you shouldn't have too install any software to get the modem to work with the mac.  The software that comes  with these modems are usually just setup wizards that aren't necessary. You also will  not be able to run an .exe file on ubuntu, and be thankful for it, as it is a security nightmare.  If I am correct the speedtouch is  a usb modem? maybe this will help.
