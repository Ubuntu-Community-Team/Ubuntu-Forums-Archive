### Post by digitalbenji on 2007-07-25
I think your x server is not starting because of the last line Option "AddARGBGLXVisuals" "True" which is out of place.  Just remove the line for now, and see if x starts.  

To remove the line, boot into recory mode.  When you get a prompt, type:
nano /etc/X11/xorg.conf

and scroll down to that last line, delete it, and then press ctrl+x to save and write the file to disk.  Then type exit, and you should be all set.
