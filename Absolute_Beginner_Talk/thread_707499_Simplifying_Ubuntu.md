### Post by Therion on 2008-02-25
I don't know if you'll like either one but these are two applications I find really helpful:

The first one is called [Gimme](http://dailyapps.net/2007/10/gimme-desktop-organizer-for-linux/); "the desktop organizer for linux".  This just makes my life sooo much easier as a panel applet.  I prefer it over docks (like Kibba or AWN), personally.

The other one is [Gnome Commander](http://www.nongnu.org/gcmd/); once you get used to using a dual-pane file browser you'll wonder how you EVER got along without one.
