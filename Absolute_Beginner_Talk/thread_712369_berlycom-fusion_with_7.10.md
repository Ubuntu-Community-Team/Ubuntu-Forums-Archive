### Post by Miller12 on 2008-03-01
[forlong's how to]("http://forlong.blogage.de/article/2007/8/29/How-to-set-up-Compiz-Fusion#") set up compiz (ie cube, etc)

For your restricted driver issue in your other post I would use Envy. I used it to get my 7600GT going. Download Envy -> right click -> use deb package installer (that might not be exaclty what it is called but I'm not at my Ubuntu machine). It will then be in the 'System tools' run Envy and then let it configure your xorg.conf file.

Turn on restricted driver, if it isn't already. Then use forlong's guide to run all the eye candy.
