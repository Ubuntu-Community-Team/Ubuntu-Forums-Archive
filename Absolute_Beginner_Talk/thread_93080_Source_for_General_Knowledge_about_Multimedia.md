A codec is a file that is used to decode and encode multimedia files. You will need one to play media files. In the case of ubuntu, the codecs for open source multimedia formats are already included, and you will need to see [https://wiki.ubuntu.com/RestrictedFormats](https://wiki.ubuntu.com/RestrictedFormats) for patented formats such as mp3 and dvd.
