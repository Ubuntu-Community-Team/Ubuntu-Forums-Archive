### Post by DSn0wMan on 2007-06-27
I believe that python is installed by default on Ubuntu. If you just need to edit it you can use gedit or kate. If you need to run it you will need to make it executable 

```
chmod u+x somefile.py
```

and run it like so

```
 ./somefile.py 
```
