### Post by jordanmthomas on 2007-01-18
right click on your panel and go to "add to panel"
you'll want to add a "notification area"

For me, at least, sometimes it doesn't work with programs that should show up and are already running:  only newly started programs will show up in it.  Logging out and then back in (or simply restarting any programs that use the tray) will make it work again.

Hope this helps
