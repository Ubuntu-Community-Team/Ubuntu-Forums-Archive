### Post by Happy_Man on 2008-04-20
Well, I'm not sure if you can, but try to install the compiz settings manager using the command : ```
sudo apt-get install compizconfig-settings-manager
```

In there, you can explore, tweak, and discover the various keyboard shortcuts Compiz has to offer.
