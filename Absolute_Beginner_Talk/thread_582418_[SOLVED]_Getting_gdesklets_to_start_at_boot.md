start. I don't know what to type in.

### Post by Hobo Joe on 2007-10-19
Assuming the command to run gdesklets is simply 'gdesklets', you would go to System > Preferences > Sessions. Add a new session, title it what you want, and in the command section type 'gdesklets'. That should do it.
