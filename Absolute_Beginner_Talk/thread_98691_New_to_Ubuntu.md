### Post by aysiu on 2005-12-03
Evolution?

Open up Evolution.
Edit > Preferences > Mail Accounts > Add > Forward > 

Email Address: [email]roostercogburn@gmail.com[/email]

Forward

Server Type: POP
Server: pop.gmail.com
Username: roostercogburn
Use secure connection: Always
Authentication Type: Password

Forward

Leave messages on server

Forward

Server Type: SMTP
Server: smtp.gmail.com
Server requires authentication (check box)

Forward > Forward > Apply
