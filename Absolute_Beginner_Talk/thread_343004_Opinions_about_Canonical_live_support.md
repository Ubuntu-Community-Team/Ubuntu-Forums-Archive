### Post by tonyr1988 on 2007-01-22
The official support is mainly for business solutions. These forums are possibly the best (and free!) resource for you to get started in Ubuntu Linux.

It looks like you got your wireless card working - is that still going alright for you?

Are you having any other hardware problems? Most of your hardware issues will definitely be resolve-able. What kind of software did you typically use under Windows? Depending on what it is, there is usually some sort of Linux alternative (often better than the Windows versions!), but sometimes there will be a learning curve.

I guarantee you that if you stick with it (and ask questions when you need to), you'll get the hang of it in no time.

As for good n00bie guides:

[http://www.ubuntuguide.org/](http://www.ubuntuguide.org/) (helps solve some common issues)
[http://www.psychocats.net/ubuntu/index.php](http://www.psychocats.net/ubuntu/index.php) (several short lessons in Linux from forum member aysiu)

Google and Wikipedia can also be great resources. At least for me, one of the biggest confusions starting out were all the different "root folders": /boot, /dev, /etc, and so on (completely different than C:/WINDOWS, system32, etc....) If you're curious, [this]("http://en.wikipedia.org/wiki/Filesystem_Hierarchy_Standard") is a good read.

Sorry for going so long :). In short: sounds like you've got the basics set up - if you stick with it, it'll only get better from here.

P.S. Once you get your system set up, you may want to make a backup in case you mess something up big time. :)
