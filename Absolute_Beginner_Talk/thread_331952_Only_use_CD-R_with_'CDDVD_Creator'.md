### Post by viciouslime on 2007-01-05
Not being able to do multisessions, mean you can't burn to a cdr more than once, not you can't use a cdrw.

Using a CDRW with CD/DVD creator is fine as you can just erase it if it is already written to, but if say you have a 300mb file you want to burn to a CDR, if you use CD/DVD creator, once you have burned it to the disc, you won't be able to add any more information to that CDR. If you make a multi session disc using something like gnomebaker then you can burn a 300mb file to a CDR and then at a later time/date you could burn another 200mb file and then at yet a later time/date you could burn another 150mb file, etc.

You won't quite get a full 700mb out of your CDR if you do use multi session though, as extra room is needed to create the multisession information.

You could also have a multisession CDRW, but this is a bit pointless since you can just copy its current contents to your hard drive, wipe it then burn the old contents along with any new contents to the CDRW as a single session (using CD/DVD creator if you wish).

Hope that clears things up :D
