### Post by artfwo on 2008-04-01
[LIST=1]
[*]Make sure, that you have "Layers" dialog open (press CTRL+L if not), then paste the 2nd image.
[*]You'll see a "Floating selection" item in the Layers dialog (it contains the 2nd graphic).
[*]Okay, now simply click "New Layer" button in Layers dialog or press CTRL+SHIFT+N.
[*]Now you have two layers one above the other :)
[/LIST]
