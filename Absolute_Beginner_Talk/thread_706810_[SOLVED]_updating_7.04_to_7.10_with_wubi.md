### Post by .Michael on 2008-02-24
I had problems updating, it was giving me wierd HAL errors.

But the way you update a distro is to go System > Admin > Update Manager
it should say 7.10 is availble.

Also on the linux commands, you can find out what you want to do by typing man -k <what you want to do>

ex:
man -k add user
man -k remove user
man -k delete file
man -k network configuration

Man will tell you about any command on your system.
I find linux commands aka shell commands much eaiser than Windows/DOS commands, Llike the / (/usr/bin) seperates directories. In windows you have to reach above your enter key. (C:\Windows\crap)
