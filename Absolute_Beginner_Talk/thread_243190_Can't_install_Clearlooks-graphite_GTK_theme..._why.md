### Post by Metacarpal on 2006-08-25
By default, the theme manager is expecting a theme package to be in .tar.bz format.  Since the file the guy on deviantart is posting is a .zip file, the automated installer can't do anything with it.

Try looking on [http://www.gnome-look.org](http://www.gnome-look.org) and seeing if you can find your desired theme there.  Then it's just a matter of downloading, going to System > Preferences > Themes, clicking Add, and pointing it to the correct .tar.bz file.
