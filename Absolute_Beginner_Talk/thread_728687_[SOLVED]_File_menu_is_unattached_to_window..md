### Post by jordanmthomas on 2008-03-19
I don't use KDE very often, but try this:
1.  Run kcontrol
2.  Go to Desktop --> Behavior
3.  Make sure that the option for "Menu Bar at Top of Screen" is set to "None" and not the one referring to Mac-OS style.

Hope this fixes it for you.
