### Post by Rocket2DMn on 2007-08-01
The best option if the LiveCD doesn't work is to use the alternate install cd.  It doesn't have a like desktop session, but rather a text based install.
[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download) and check the box at the bottom for the alternate desktop cd.
Don't forget to check the md5sum
[HowTo]("https://help.ubuntu.com/community/HowToMD5SUM")
[Hashes]("https://help.ubuntu.com/community/UbuntuHashes") to check against
Then burn at a slow speed, like 4x, to prevent write problems, and use quality media if possible.
It's a regular problem for the LiveCD to not work for people, don't be alarmed or discouraged.
