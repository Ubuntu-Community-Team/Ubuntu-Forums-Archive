### Post by felicity on 2007-08-12
You should just be able to type:
```
sudo apt-get install vlc 
``` from a terminal.

Or if you prefer, you can use the Applications menu to find Add/Remove Programs. Then just search for vlc, check it, and click OK.

If you're using Dapper the directions may be different, I haven't used that version of Ubuntu.
