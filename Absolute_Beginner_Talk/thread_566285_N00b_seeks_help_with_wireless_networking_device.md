### Post by Paul820 on 2007-10-03
The only way to test if your devices will work in ubuntu without installing it is to use the live CD. Nothing will get installed, everything runs from the CD, so just put the disk in and test your devices. If you find you are happy with it, then you can install it by clicking on the icon on the desktop.

Wine works very well, although not everything will work with wine.

I use code::blocks, it's free, has code-completion and integrated debugger, and is very similar to VC.
Check it out here: [http://forums.codeblocks.org/]("http://forums.codeblocks.org/")

You can wait, or you can install 7.04, if you are in a rush then just get 7.04. If you don't want to help with bug testing for gutsy then don't install that just yet, it's still buggy, wait three weeks for it's release.

Ram is very cheap at the moment, a little more never hurts. Ubuntu recommends at least 256Mb. There are distros that will run on lower spec machines.
