### Post by Rocket2DMn on 2007-07-24
You don't install .exe files in Ubuntu, these are for windows.  If you have windows programs that you really can't find linux-compatible programs for, you need to run them through a program called wine, which is like an emulator, but not.
When you decide on certain linux programs to use (or want to test-drive some), use Ubuntu's repositories to download them and have them automatically installed.  You can do this from Add/Remove, Aptitude, or apt-get.  There is plenty of available help for these, or you can ask specifics.
If programs are not in the repositories, you will have to download them and install them manually - ask for specific help on this if you need it.

### Post by Orochi on 2007-07-24
You don't really run exe files on Ubuntu, they're made for Windows machines. You should first check to see if there's an Ubuntu equivalent program in Add/Remove programs that you can run instead. Otherwise you can try installing Wine which is a compatability layer that lets you run some .exe files.

Most programs can be installed automaticaly for you through Add/Remove programs or Synaptic (it's under the Administration menu).
