### Post by DJ_Peng on 2006-11-11
I was one of testers for Firefox 2 (as a Windows user) and although I love seeing it the default browser with Edgy I missed some of the functionality, including the auto update system and how a few extensions worked, not to mention a build date that's almost five months before Firefox 2 was released. (The user string should show something along the lines of Mozilla/5.0 (X11; U; Linux i686; en-US; rv:1.8.1) Gecko/20061010 Firefox/2.0). I ended up getting Firefox 2 directly from [Mozila]("http://www.mozilla.com/en-US/firefox/"), extracting it to my home folder, and changing the link on the toolbar to point to the home/firefox/firefox file. Voila! Firefox 2 as the developers intended it to work.

I don't mean anything against the version of Firefox 2 in the repository, but like I said I have been working with Firefox 2 since it was called Bon Echo and was branded as an alpha version so there were some things that don't work quite the way I expected them to.
