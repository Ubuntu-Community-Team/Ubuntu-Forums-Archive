### Post by mlind on 2006-04-24
If you want to avoid breaking things, see check out [garnome]("http://www.gnome.org/projects/garnome/") utility/script.

Script will build gnome from sources and install it automatically, it won't mess
with your current gnome installation.

gnome session you want to use.
