### Post by eentonig on 2007-07-21
To give you a quick overview.

Linux is the base operating system.
Ubuntu is de distribution. It is the collection of applications and tools that make it easy for you to interact with your computer.
Gnome is de desktop environment that Ubuntu uses and that provides you with the GUI to use your Ubuntu.
Compiz/Beryl/Compiz Fusion is an enhancement you can install to provide some extra functionality/eyecandy for your GUI.

By default, Ubuntu installs a basic compiz setup ('Desktop effects'). But it doesn't use it. It's installed, so the user can try it. But it only provides very basic possibilities.
