### Post by christhemonkey on 2007-04-07
To stop synaptic asking for the cd you can just remove it from the repositories dialog.

Go to synaptic package manager then go Settings > Repositories
Then look for the line referring to the cd and remove it.
Finally close that dialog and click 'Reload'.
Then you should be able to install away! :D
