### Post by TeaSwigger on 2007-10-07
Probably isn't the case but given that you're new it may bear mention; if you update after the install and reboot or start it again later on, you could be seeing the new kernel. That's an updated new core component I suppose we can say for convenience. By default at the login screen it'll show you the old and new kernels with one recovery mode safety option each, just to be extra safe. That way should there be a problem with the new kernel, however unlikely, you can simply boot up with the old one and fix things. You can tell if this is the case by the version numbers.
