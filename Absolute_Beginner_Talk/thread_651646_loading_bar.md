### Post by zuzuzzzip on 2008-01-06
> **themuddaload said:**
> so i have a feisty cd sent from ubuntu, is there any way to do it without using synaptic, thereby avoiding the need of internet?
yes insert your cd, go to Administration > Software Sources
unselect all internet options
select CD/DVD option
go in a terminal and type
Click 'Close' in the bottom

sudo apt-get update
sudo apt-get install ubuntu-desktop

try this, not sure if it works just wrote it down now :)

if you want to be able to update via internet again, don't forget to select the options again!
