### Post by PPAAUULL on 2007-10-19
First you need to calm down. Every one has some frustrating time with Linux. For me it was Graphics card problems and I spent a month trying to get it to work.

Second the reason you can't watch the movies or the music is cause you are missing the codecs. You should be able to search them in synaptic.

If you would like to have more thing enabled like java and such I would suggest checking out Automatix. It is a great program that will install and setup all of those codecs and such for you.

As for your HDD well if it is formatted as NTFS you can only read it unless (and it is included in Automatix) you get a plugin that lets you write aswell to NTFS. if you don't want to try that plugin (I give no garentee that you won't lose data or such) then I suggest you reformat the drive to FAT32.

Hope that helps.
