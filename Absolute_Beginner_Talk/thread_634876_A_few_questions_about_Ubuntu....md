### Post by PmDematagoda on 2007-12-08
1) Install Wine using:-
```
sudo apt-get install wine
```

2) Then do:-
```

winecfg
```
to configure Wine.

3) Then you can execute .exe files by opening them through the file managers using Wine, or you can open them by appending "wine" in front of the .exe's name.
