### Post by paulmilliken on 2006-02-12
Have you unzipped the driver that you downloaded?  If not, do "tar -xzvf Ixm3200-tweaked-20030501.tar.gz" in the directory where the file is located.
Then you should be able to find a file called "readme" or "install" somewhere in the unzipped directory.  The readme or install file should give you some instructions.
