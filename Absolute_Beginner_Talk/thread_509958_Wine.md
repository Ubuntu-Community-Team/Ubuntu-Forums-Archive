### Post by sloggerkhan on 2007-07-26
If you are using feisty with updated wine, when you install a windows application, it shows up in a wine menu under applications, otherwise it doesn't show the menu.

From command line: 
$wine /path/to/program.exe

$winecfg to configure

or just type $wine and then hit tab-tab to see what's there.
