### Post by Seti on 2007-08-22
Sometimes some packages don't include instructions for putting an app in the menu.
Try the whereis command to locate where deluge was installed:
```

whereis deluge

```

That should show you where it is. You will probably have to make a custom launcher for it.

Good luck
