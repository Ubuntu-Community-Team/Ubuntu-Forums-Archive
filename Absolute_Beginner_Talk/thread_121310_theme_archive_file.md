I would imagine the theme archive file *is* the file you have. 

What sort of theme is it?  Icon theme?  Windows control theme?  What file format is the theme too?  .tar.gz? 

Anyway, if you open the theme manager, installing a new theme is easy.   

Go to Applications -> Preferences -> Themes.  Click 'Theme Details' and drag the actual theme file (from a browser or the desktop) into the Themes dialog box.  This should install the theme for you.  After this you should be able to choose the new theme.

The theme is installed into a hidden folder in your home directory.  The folder is .theme.
