### Post by LuisGMarine on 2007-04-27
First find the command to run the program, the script w/e it is to get the program started.

Then head over to System > Preferences > Main Menu and select the menu you want this app  added to.  Once you have that pin pointed go ahead and hit the " New Item " button to add the app.  For the command, you would simply fill that in with the command to run the program, icon, you do the same thing, and so on.  This should add the custom application to your menu.

Good Luck

-LuisGMarine
