### Post by dpeirce on 2007-04-16
Kubuntu 6.10 Edgy (just waiting for Fiesty).

The little Snap search balloons which pop up on some web pages when you hover your cursor over a link just drive me nutty! They are more obtrusive than some advertisements. But there are at least two easy ways to get rid of them for good:

1) As root, edit the /etc/hosts file adding the following line:

127.0.0.1   spa.snap.com

This kills all snap balloons on all sites.

2) Use the Opera browser. From the toolbar, go to Tools -> Quick Preferences -> Edit site preferemces -> Scripting, and disable javascript for that site. Or disable javascript for the whole browser and a LOT of stuff will go away. I'm told Firefox's adblocker can be set to block spa.snap.com also.
