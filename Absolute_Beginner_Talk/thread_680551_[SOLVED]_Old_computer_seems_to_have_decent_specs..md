There is, of course, a loss is quality and speed since it's being run under wine, but even I can deal with it and I'm pretty unforgiving.
To start metacity (disable compiz) under 7.10 w/ Compiz Fusion:
```
metacity --replace
```
To re-enable CF after playing
```
compiz --replace
```
