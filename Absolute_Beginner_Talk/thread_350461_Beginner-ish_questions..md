### Post by tonyr1988 on 2007-01-31
Most people on the forums won't talk about the Add / Remove Programs to help people out because it's not always as powerful as they need it to be. Most will suggest Synaptic (System -> Administration -> Synaptic Package Manager). It's not quite as user-friendly, but it's not bad. You just find what you want, put a checkmark by it, and Update. :) It's also got a search feature to help you out - you may want to check it out. For just installing some programs, Add / Remove (it's called gnome-app-install, I believe) will work, but Synaptic allows you to get libraries, sources, etc.

LTS = Long-Term Support

6.06 (Dapper Drake) is going to be supported longer than most Ubuntu versions. The only updates you'd download are going to be security updates (for example, it doesn't have Firefox 2.0 IIRC). It's mainly for people using Ubuntu for businesses or need to ensure stability as much as possible by not constantly updating programs. For most users, you should be fine with Edgy Eft (6.10) - it's been pretty stable for me.
