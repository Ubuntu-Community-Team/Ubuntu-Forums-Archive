### Post by bscbrit on 2005-12-22
build.sh is a shell script i.e. a series of commands that control, in this case, the building of the program.  Make sure that the file is executable.  In Gnome, right click on it and select properties. Check 'permissions' and set them accordingly.  If you find that you are not the owner, then you will need to use 'sudo' to change owner ship and then change the permissions.  You can run the file from the command line by going to the directory in which it lives, and typing

./build.sh

If you managed to get through the installation guide for firefox, you are probably more than capable of coping with this.

If you have any more problems then come back here (although I am now off to my bed!)
