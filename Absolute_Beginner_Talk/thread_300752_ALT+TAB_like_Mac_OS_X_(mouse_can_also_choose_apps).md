### Post by wieman01 on 2006-11-16
KDE has such a tool and I have been using it for a while. It's called Kompose and you'll find it in the repositories. It should also run in Gnome.
> sudo apt-get install kompose
See screenshot... You can create a shortcut for it (e.g. alt+tab).
