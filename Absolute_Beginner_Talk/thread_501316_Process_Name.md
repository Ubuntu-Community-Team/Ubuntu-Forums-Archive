### Post by felicity on 2007-07-15
You can see a list of running processes by typing ```
top
``` in a terminal. You may be able to infer from there what the name of a specific program is. You can also start ```
gnome-system-monitor
``` if you'd like.

Also if it's a GUI program that you want to terminate, you can type ```
xkill
``` in a terminal then click on the window you want to force quit.
