### Post by Spr0k3t on 2007-08-26
You would be fine either way.  Running the KDE desktop with Gnome apps or the Gnome desktop with KDE apps the memory usage would be the same.  Now if you went pure KDE, you would use a little less memory than using both at the same time.  If you stuck with Gnome only, you would use even less than KDE.  I primarily use a Gnome desktop with mostly Gnome applications but I have a handful of KDE apps installed on the primary desktop.  The systems with dedicated routines run pure configs though.

In a nutshell... if you have more than 512MB in memory, you won't see much difference.
