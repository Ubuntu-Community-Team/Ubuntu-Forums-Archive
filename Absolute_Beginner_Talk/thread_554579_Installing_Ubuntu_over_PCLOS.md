### Post by JawsThemeSwimming428 on 2007-09-19
If you just want to reinstall Ubuntu over PCLOS you can just boot to the Live CD and when asked how to partition, tell it to use the whole disk. This WILL erase both installs and start Ubuntu completely fresh using the entire disk. So make sure you have all of your data if this is the way you are going to go.
