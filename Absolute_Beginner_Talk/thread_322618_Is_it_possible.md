### Post by wpshooter on 2006-12-20
**SUPPOSEDLY**, there are programs like **WINE** and others that can be used to run Windows applications on a Linux machine.  But I think you may be somewhat disappointed with the results.

IMO, the best way to run Windows programs on a Linux system is to wait two to three years and **HOPE** that the programs' owner makes a decision to develop a version of their applications that will run natively on a Linux system.

Good luck.

### Post by aysiu on 2006-12-20
Here are your possibilities, in order of most desirable to least desirable:

1. [Find a native Linux application that does what you want to do](http://www.linuxrsp.ru/win-lin-soft/table-eng.html)
2. Use [Wine](http://www.winehq.com/)
3. Use [Crossover Office](http://www.codeweavers.com/) or [Cedega](http://www.transgaming.com/index.php?module=ContentExpress&func=display&ceid=2&meid=-1)
4. Use [VMWare](http://www.vmware.com/) to emulate Windows
5. Use a KVM switch between a Windows and Ubuntu computer
6. Dual boot
7. Stick with Windows and don't use Ubuntu at all
