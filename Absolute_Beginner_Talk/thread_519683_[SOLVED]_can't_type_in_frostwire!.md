### Post by mwolfe on 2007-08-11
I think i figured this one out. I dont have a good fix but i know how to get it to work. 
The problem is that when you are typing your keystrokes are going into the input field of your chat, even though you may be in the search section. Try it. 
It seems you need to remove focus from that input box. I did so by clicking below the text input in the chat and then i went back to the search and i could type in the search input box.
