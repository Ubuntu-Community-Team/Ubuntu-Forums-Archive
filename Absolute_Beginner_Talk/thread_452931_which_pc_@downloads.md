### Post by netwerx on 2007-05-23
x86 is the standard processor version.  Unless you have a 64 bit processor (ie, AMD 64) then you would most likely need the x86 version.

DT is a desktop version, most users will want this version. 
SVR is the server version, unless your planing on running a server (web, ftp, etc) grab the DT version.
