### Post by strikeback03 on 2007-03-06
> **Cylkk said:**
> Second question - when this does happen, how do I roll back? at the moment I'm reinstalling the whole OS every time & it's getting frustrating on my 8th install :)

backup your working xorg.conf before playing with it:
```
 sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup 
```

then, if you screw up the xorg.conf, you can always do 
```
 sudo cp /etc/X11/xorg.conf_backup /etc/X11/xorg.conf 
```

to restore the previous configuration.  If that does not work, you should be able to do 
```
sudo dpkg-reconfigure xserver-xorg
```

and just give it all the defaults and get a working xorg.conf again.


I got 1920x1200 working properly with my Gateway FPD2485W and 7600GT using the driver from the nVidia site.  I'm using the 32 bit version of the OS though, so dunno if the 64 bit version works or not.
