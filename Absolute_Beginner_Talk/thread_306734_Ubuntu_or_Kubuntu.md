### Post by rev_b on 2006-11-25

> Won't Ubuntu be overwritten by Kubuntu when I try to install over it?

No. Ubuntu and Kubuntu is the same system, the name diferences (a bit confusing IMHO) only reflect the desktop manager used. So when installing ubuntu-desktop from kubuntu, you are only adding a desktop manager, that you can choose when booting. Nothing is deleted, and with both installed, you have Qt and GTK libraries and you can run gnome apps (GTK) from KDE and vice versa.

You don't even need to reboot to change you desktop. Press ctrl+alt+backspace, you'll be logged out and restart X, and then you can choose to log in win KDE or gnome. That takes about 10-20 seconds...

### Post by Average Joe on 2006-11-25
Ubuntu and Kubuntu are the same operating systems, it is just that they have a different desktop environment, and different default applications.

Ubuntu comes with Gnome, and Kubuntu comes with KDE. But after you have installed, let's say, Ubuntu, you can download and install the KDE desktop environment (+ applications) too. This leaves your Ubuntu installation completely intact, you just get an additional desktop environment.

EDIT: To clarify a bit more: The Ubuntu CD has only Gnome on it, and the Kubuntu CD only has KDE on it. So if you install Ubuntu you get Gnome, you have no choice (when using the Desktop/Live CD). After installation, you can download and install KDE.
