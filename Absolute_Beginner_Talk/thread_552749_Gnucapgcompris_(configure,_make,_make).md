### Post by Bachstelze on 2007-09-16
There is a precompiled package of gnucap available for Ubuntu :

[http://packages.ubuntu.com/cgi-bin/search_packages.pl?keywords=gnucap&searchon=names&subword=1&version=all&release=all](http://packages.ubuntu.com/cgi-bin/search_packages.pl?keywords=gnucap&searchon=names&subword=1&version=all&release=all)

And to answer your question, no, you don't need to be in any particular directory to compile.
