### Post by TheThinker on 2007-09-26
If possible, don't install Ubuntu as soon as you reach the menu screen when rebooting. Run the operating system off of the CD in Safe-Graphics Mode for a test-drive, then see if you can install it. 

I don't think the Toshiba computer was below-standards per se. Since it's an older model, it could be that the CD drive has dust in it and needs to be cleaned with a CD cleaner or that it is damaged and needs to be replaced. Many failed CD installations are due to a stupid CD drive, like mine for instance. The same applies for failed Live-CD functions.

If you want to install Ubuntu on a laptop first, you might want to try out Xubuntu instead. It's meant for low-end computers and I hear it's great for battery life. Check it out.
