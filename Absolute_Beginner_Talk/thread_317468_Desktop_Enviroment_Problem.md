### Post by ajgreeny on 2006-12-12
I think what you are seeing is quite normal, even though you think it is confusing.  This situation is a result of linux's ability to use a program from any of the DEs in any other DE that happens to be running.  I think the only answer is to manually edit out the menu items you don't want, though knowing linux, there might just be a way to remove several at once instead of doing it one by one.  Unfortunately if that is so, I don't know how.  Perhaps someone else does.

The boot screen doesn't make any difference.  It is basically a different GUI for the same program.  You only care about being able to log into the desktop you want to use, and I think all of them-there are more than just KDE and Gnome-allow you to log into any of the desktops you have installed on your machine.
