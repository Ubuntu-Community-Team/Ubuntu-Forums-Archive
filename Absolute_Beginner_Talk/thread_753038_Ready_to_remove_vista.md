### Post by R6V2 on 2008-04-12
You should use the VISTA CD to remove VISTA. (It actually removes EVERYTHING). WINE is a Windows API Linux emulator. It allows you to run most programs on Linux, they may be a little bit slower because it's running a program to run your program, but it's the best shot! Type this inside your terminal to get WINE:

```
sudo apt-get install wine
```
