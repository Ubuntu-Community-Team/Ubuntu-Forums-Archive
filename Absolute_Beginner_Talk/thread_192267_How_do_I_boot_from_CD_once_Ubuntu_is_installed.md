### Post by christhemonkey on 2006-06-08
You need to make sure you have selected boot first from CD drive in your BIOS.



If for some reason this doesnt work, you should use the SmartBootManager.
Which allows you to boot off a floppy and then boot from CD drives.

---

### Post by Snakey on 2006-06-08
Go to you bios and select the cd drive as the first device to boot from (check manual of your motherboard for specific things on this). Then save settings and reboot.
