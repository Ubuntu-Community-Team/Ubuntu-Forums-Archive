### Post by w33mhz on 2006-08-14
If you are trying to access a wireless AP via WPA or WPA2, you need to verify that you have downloaded and installed the wpa_supplicant package.  Now with that being said, not all wireless cards are support with wpa_supplicant.  If not connecting via WPA/WPA2 and you are using an open or WEP connection almost any wireless card will work.
