### Post by bobbocanfly on 2007-08-12
To "dual boot" between Whatever you are using now and XFCE use:
```
sudo apt-get install xfce4
```

Or if your wanting to get rid of Gnome and use XFCE instead use:
```
sudo apt-get remove ubuntu-desktop && sudo apt-get install xubuntu-desktop
```
