### Post by jordanmthomas on 2006-10-04
If it says Firefox, then you are running Firefox.  If it says Swiftfox, you are running Swiftfox.  Swiftox is the exact same thing as Firefox, but it is built for specific processor architectures (as opposed to Firefox's general build).  It uses all of Firefox's plugins, bookmarks, and extensions.  

try running swiftfox from a terminal and see if it identifies itself as Swiftfox...mine does in Help --> About.

P.S.  I don't notice that Swiftfox is even that much faster than Firefox, so I use Opera because if you want fast--Opera has it.

### Post by seijuro on 2006-10-05
Most likely you are opening firefox if you look at the window title bar you can tell which one you are running right away as each will display the webpage name followed by "- browser_name" if you go to help and click about in swiftfox it will say firefox ver blah because as mentioned before it is a firefox build optimized for your specific processor. The above mentioned terminal check is a good idea also I would recommend checking that the application launcher properties are indeed using the swiftfox command when I installed swiftfox on both ubuntu and kubuntu swiftfox was not added to the menu automatically. I can't speak for others but on my system swiftfox runs about 2x as fast as the standard firefox and yes I did time it several times with both cold and warm starts. But again that is the results on my own system other systems may be better or worse.
