### Post by dwhitney67 on 2007-06-16
System -> Preferences -> Mouse

Then click on the Motion tab where you can set the acceleration.

Don't expect every MS-capable device to function exactly under Ubuntu or any other linux distro.  Sometimes the vendors of these devices will only provide drivers for MS and Mac, because after all, that is their business base.  There is "hardly" no money to be had with developing Linux drivers.
