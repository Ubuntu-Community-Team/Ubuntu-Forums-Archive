### Post by Shazaam on 2008-03-15
You will need to reinstall XP's mbr using a retail (store bought) XP disk or (if you have them) the Recovery cd's that came with the pc. Once you do that you can delete the Ubuntu partitions. If you want to resize the XP partition to fill up the empty space make sure you defragment Windows first.
