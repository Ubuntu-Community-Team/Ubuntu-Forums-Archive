I have a 7.1 intergrated sound card, and i want to know if this will work with ubuntu. [/quote] I have no idea. The best way to test it is to use the Ubuntu live CD before using the installer CD. If it works in the live CD, it'll probably work in the installer version.

> 
I have heard there are applications that allow you to run windows games. Will this run any windows game, or just ones they have set up support for? One of the applications site says something about CSS, does that mean it runs steam, or just CSS by itself? For gaming, there's something called [Cedega](http://www.transgaming.com/), and for general applications, you can use [Crossover Office](http://www.codeweavers.com/products/cxoffice/) or [Wine](http://www.winehq.com/site/documentation)

> 
Will ubuntu allow me to install any application that is made for linux, or do they have to be special ubuntu supported apps? If they're special Ubuntu supported apps, you'll be able to install them easily (see the second link in my sig). If not, you'll be able to install them, but it may take a few more commands.

> 
Also will i be able to install mythTV? I don't even know what MythTV is, but a simple Google search turned [this](http://www.ubuntuforums.org/showthread.php?t=29510) up.

> 
Will ubuntu work well with my pre configured windows partitions? I currently have 2 partitions, 10gb for system files and 139gb for all other files. [http://www.ubuntuguide.org/#windows](http://www.ubuntuguide.org/#windows)

> I wish to install ubuntu to my system files drive. This has 4.45gb left, will that be enough? and i want it to store documents using my pre-configured folders in my other files drive. Is this all right? Ubuntu needs its own partition. You can't install it on part of a Windows drive. You need to repartition the Windows drive first. Based on this question, I'd highly recommend you read my [Linux Guide for Windows Users](http://www.psychocats.net/essays/linuxguide.php)

> 
If i dont like using ubuntu, will it be easy to completely wipe it from my hd without having to format? Easily with another Linux distro, possible with Windows (but not as easy).
