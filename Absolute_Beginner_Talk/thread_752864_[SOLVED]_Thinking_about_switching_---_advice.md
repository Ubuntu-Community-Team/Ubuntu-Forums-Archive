### Post by nvteighen on 2008-04-12
> **Sanoski said:**
> (...) But what can I expect from the change?
In one phrase: another OS. :)

> I'm assuming there is a different file structure. What is Linux equivalent to an EXE file? And what about file management? How are thing organized, etc?
Well, you can look up for information on these details on [http://www.linux.org/lessons/](http://www.linux.org/lessons/). It's a tutorial to learn how to use the command line and "good manners" at managing any Linux system. I like it a lot because it isn't focused at only one distro, but it gives you a general view on how things are done in most distros. Then, of course, you'll going to "adapt" some things to the specific distro you're going to use (in this case, Ubuntu). Also, I like the fact that it doesn't cover any GUI...

I recommend you to follow that lessons using the Live CD without installing Ubuntu until you feel you can manage it by your own.

As a reference work, you can go to [http://www.linfo.org](http://www.linfo.org), The Linux Information Project, which has also a good absolute beginners section.

> Also, do you have to go through the same process every time a new version comes out? Download the ISO, burn, etc?

No, you can upgrade versions through Update Manager, but I prefer to reinstall each time (but that means you need to have /home in a different partition so you don't lose data and personal configuration).

### Post by pbpersson on 2008-04-12
When a new version comes out you typically just upgrade online.

In theory you only install from CD once.

However, I have heard stories of people having problems that way and they had to re-install from CD but hopefully those episodes are rare.

In Linux......well in Ubuntu Gutsy Gibbon you are so far removed from how things work that you don't worry about .EXE and .COM files.  Everything is done from the menu.

Linux is VERY different from Windows under the hood.  Just don't look under the hood and you won't be freaked out.  ;)

Trying running Ubuntu from the CD first to make sure your hardware is supported.

If you have any problems after installing, post any error messages or questions here - there are lots of knowledgeable people to help :)
