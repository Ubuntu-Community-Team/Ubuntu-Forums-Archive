### Post by cubeist on 2007-08-29
well if you are looking for the folders they were probably installed in /usr/share/   (or somewhere else in /usr).  But you probably just want to run the applications which can be done via the terminal.  Just type the name of the program.  For example, to start your web browser from the terminal just type "firefox".

If you get the program running you can add a launcher by right-clicking on a panel and choosing "Add to Panel" and then "custom application launcher"
