### Post by 3rdalbum on 2008-03-24
128 megabytes of RAM is a bit too lightweight for Xubuntu. You might want to try Puppy Linux or DSL. Both of these will be a bit less user-friendly, but that's the trade-off. You could also try Vector Linux, although I haven't tried it myself.
