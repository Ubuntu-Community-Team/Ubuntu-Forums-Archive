### Post by aysiu on 2006-08-22
The first few methods in there seem a bit convoluted (a floppy disk?).

Boot into recovery mode (it's below what you regularly boot to but above "memtest").

You'll then be root. Type ```
passwd lilxpuser
reboot
```
