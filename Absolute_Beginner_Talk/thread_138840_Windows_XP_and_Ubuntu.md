### Post by Sutekh on 2006-03-02
Yes its perfectly possible.  I would imagine that a very large proportion of people here have exactly the same situation.  Ubuntu uses the GRUB (**GR**and **U**nified **B**ootloader, which can be used to boot Ubuntu, Windows, and anything else you may want to use.

This is a good site, that has installation guides with Screenshots along the way.

[http://users.bigpond.net.au/hermanzone/]("http://users.bigpond.net.au/hermanzone/")

If you scroll down you can find links for installing Ubuntu with Windows on NTFS (or FAT32).  You should have a read through the guide and make sure you understand the process.
