### Post by KIAaze on 2007-05-09
I don't know if there is any GNU/Linux native version or an equivalent software of Macromedia Director.

If not, there are 2 other ways to run it in Ubuntu:

_1)WINE or Cedega:_
WINE and Cedega are some kind of compatibility layer allowing you to run windows programs under GNU/Linux.
However, they are not perfect yet and may not support all windows programs.
Check out these links for more info:
[http://appdb.winehq.org/]("http://appdb.winehq.org/")
[http://frankscorner.org/index.php]("http://frankscorner.org/index.php")
[http://www.transgaming.com/index.php?module=ContentExpress&func=display&ceid=2&meid=-1]("http://www.transgaming.com/index.php?module=ContentExpress&func=display&ceid=2&meid=-1")

Here is a howto for installing WINE in Ubuntu:
[http://www.psychocats.net/ubuntu/wine]("http://www.psychocats.net/ubuntu/wine")

_2)VirtualBox/VMWare:_
Those are programs allowing you to run Windows from within Ubuntu.
Macromedia Director should then almost certainly work "in Ubuntu", since it's the same as running windows.
However, a valid license for windows is required to apply this solution.

[How to run Windows in Ubuntu with VirtualBox (video tutorial)]("http://www.youtube.com/watch?v=ch8X86R6d-g&mode=related&search=")
