### Post by forrestcupp on 2008-03-27
Turbo Tax online doesn't work with Linux even though they support Firefox.  I think it's ridiculous, but there are other trustworthy online options that *do* work with Linux.  H&R Block has an online version of Tax Cut that works.  There is also Tax Slayer and Tax Act.  They all work well, and really, one is about as good as the other.

About the adobe things that don't work.  You didn't give enough information.  Adobe has a lot of products, but from the context, I assume you mean Adobe Reader.  Ubuntu comes with evince which is a capable app that can read any pdf file.  It is also possible to get Adobe Reader for Linux. 

If by chance you meant that you need to see something that uses Adobe Flash, all you have to do is install it.  Just type in a terminal,
```
sudo apt-get install flashplugin-nonfree
```
