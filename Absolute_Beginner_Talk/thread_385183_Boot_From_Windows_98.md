### Post by Lexyboy on 2007-03-15
1) Put CD in drive
2) Reboot computer
3) Wait while Ubuntu loads (NB: it will run slowly in LiveCD mode compared to installed on your HD)

If it doesn't load into ubuntu, I guess either your BIOS is set not to boot from CD (in which case press del key just after rebooting, and set CD-ROM booting priority above HD), or you have a non-liveCD version of ubuntu...
