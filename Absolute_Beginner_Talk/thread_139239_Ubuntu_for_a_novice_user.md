### Post by Q4U on 2006-03-03
If it's only for web and mail, then it is a no-brainer: install Ubuntu!

As for the memory needed, my laptop has 256MB. It is by no means snappy, but it works reasonably well, especially for web and mail. I recommend 512MB upwards if they use OpenOffice. And memory is so cheap these days (you can buy one 512 MB stick for 35 USD) that I see no reason not to upgrade.

My parents have also learnt very quickly how to update the software though Synaptic, it is as easy as upgrading Windows. So you might want to explain this to your relatives as well.

However, I recommend you set the system up so that you can SSH in their box for remote administration, in case of emergencies. Q4U
