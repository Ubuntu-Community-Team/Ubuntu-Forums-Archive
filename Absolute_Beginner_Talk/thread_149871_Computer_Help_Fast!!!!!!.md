### Post by _simon_ on 2006-03-25
I'd recommend you run this online scan first, it's free but pretty good: 

[http://www.pandasoftware.com/products/activescan?NRMODE=Published&NRORIGINALURL=%2factivescan&NRNODEGUID=%7b3B202047-35D4-4DA2-B310-B1DBEC2971F2%7d&NRCACHEHINT=Guest](http://www.pandasoftware.com/products/activescan?NRMODE=Published&NRORIGINALURL=%2factivescan&NRNODEGUID=%7b3B202047-35D4-4DA2-B310-B1DBEC2971F2%7d&NRCACHEHINT=Guest)

You will need to run it from Internet Explorer, it won't run in other browsers.

The fact that you have managed to get a virus indicates you either do not have antivirus software or it's outdated. There are many free ones available e.g.:


AVG [http://www.free-download-center.com/avg/](http://www.free-download-center.com/avg/)
AVAST [http://www.avast.com/eng/avast_4_home.html](http://www.avast.com/eng/avast_4_home.html)

Ok that's antivirus sorted now antispyware.... i've never used Spy Sweeper so don't know how good it is but it would also be worth installing and running:

Spybot Search & Destroy which is free - [http://www.safer-networking.org/](http://www.safer-networking.org/)
If you run a valid version of Windows then also give Windows Defender a go - [http://www.microsoft.com/downloads/details.aspx?FamilyID=435bfce7-da2b-4a6a-afa4-f7f14e605a0d&DisplayLang=en](http://www.microsoft.com/downloads/details.aspx?FamilyID=435bfce7-da2b-4a6a-afa4-f7f14e605a0d&DisplayLang=en)


Now all that's left is a firewall if you haven't already got one... if you use XP then the windows one is not worth using. Try something like Zonealarm (free)

Zonealarm: [http://www.zonelabs.com/store/content/catalog/products/sku_list_za.jsp?dc=12bms&ctry=US&lang=en&lid=dbtopnav_zass](http://www.zonelabs.com/store/content/catalog/products/sku_list_za.jsp?dc=12bms&ctry=US&lang=en&lid=dbtopnav_zass)

I've given you free links so you don't have to go and fork out money, once you are protected then look into what software may be best for you.

And last but certainly not least! Make sure you run Windows Update and get ALL of the updates!

As long as you have the security in place to begin with then viruses and spyware should not be an issue.

Remember: You can only install ONE antivirus program otherwise you will have issues. You can install as many antispyware packages as you want and you should only need one firewall - more than one and you could be causing yourself a headache.

Hope some of the above is of help to you.
