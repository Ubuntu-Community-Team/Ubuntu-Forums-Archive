### Post by taurus on 2007-01-03
Try **oem** as your username and the password that you have created during the installing.  Then from a terminal, run this command to create a regular account that you will use from now on...

```
sudo oem-config-prepare
```

### Post by Efwis on 2007-01-03
you user name was asked at installation. 

it is created right after you put your name in the box during the installation.

for example, it asked for my real name and then the box below it asked for the username I i wished to use, in my case I used my real name in lowercase. 

this is just above where you put your password when installing.
Hope this helps
