### Post by DerArzt on 2007-02-14
To make a drive mount on boot you have to add the drive to fstab. 
To make the drive show on the desktop you have to mount it to /media, so i would first create a folder in /media. Say you want to name the drive ntfs, you would type 
[HTML]sudo mkdir /media/ntfsdrive[/HTML]
Then you would have to edit your fstab
To edit fstab type
[HTML]sudo gedit /etc/fstab[/HTML]. 
to make the drive mount on boot add this line:
[HTML]    /dev/hda1 /meda/ntfsdrive ntfs defaults 0 0[/HTML]

I use ntfs-3g to mount my ntfs drive so that i have both read and write access to it.
If you want more info on ntfs-3g look here [http://www.ntfs-3g.org/](http://www.ntfs-3g.org/) and check out the forums on how to install it. It works pretty well for me. If you want to do that and have trouble with it, feel free to IM me.
