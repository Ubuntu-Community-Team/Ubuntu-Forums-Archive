### Post by expatCM on 2007-10-08
But it can be really hard to get up and running ..............

Finding the server edition is easy.  Installing is easier than the desktop version.  Updating and upgrading is also easy.

What is a real challenge is configuring all the programs which you are going to use for your server.  These programs will vary according to what you are going to use the server to do.   From my experience I have found this to be really not easy at all.  If you know it must be easy but if you are discovering it really can be frustrating.

Someone is sure to post and say use Webmin.  I cannot see the point since Webmin only covers some of the ground and you are left wondering if all has been done.....  Remember to set your firewall or things may not work.
