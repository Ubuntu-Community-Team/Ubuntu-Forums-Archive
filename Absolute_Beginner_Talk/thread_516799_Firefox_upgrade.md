### Post by mgmiller on 2007-08-03
You should not be upgrading Firefox manually.  It was installed with Ubuntu and will update itself along with the rest of your Ubuntu system automatically.  My Feisty install is currently using Firefox 2.0.0.6 which is the newest flavor.  I didn't have to do anything to get it.  It just updates with everything else.

Try running System > Administration > Update Manager and then hit the "Check" Button.

It should be doing this automatically, in the background, every day unless you changed something.
