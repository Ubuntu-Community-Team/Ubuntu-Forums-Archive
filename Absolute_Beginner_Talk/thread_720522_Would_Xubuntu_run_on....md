### Post by lloyd_b on 2008-03-10
Currently typing at you from a P3-500, 256Mb ram, Xubuntu 7.10, 20Gb HD. 

If possible, I would add some memory.  I've run this particular machine with 192Mb without too many problems, but 128 is pretty low - expect to spend a lot of time accessing swap.

You *will* need the alternate install CD - the Live CD is hopeless without 256Mb.
