### Post by Orbital75 on 2007-08-09
I would highly recommend VirtualBox......  
Currently I am Running XP as Host ( Main OS )
and Ubuntu as Guest ( Virtual Machine )

VirtualBox works pretty well and also has USB support.
In fact, I am using it right now to type this reply.

If you want HIGH end grahics or plan to play games a VM is not for you.
I am trying to learn Linux so it fits perfect for what I want to
do.  If you are just wanting to run Windows apps that won't
work in Wine then The VM solution is a good idea.

Also as far as a machine, I would say at least 1gig of ram.
I give my VM 512 and let Windows have the other 512.
Remember your going to be running two OS's at the same time
and both need Ram.....
