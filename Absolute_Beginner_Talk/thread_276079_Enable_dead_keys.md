### Post by hajk on 2006-10-12
System -> Preferences -> Keyboard -> Layout Options, then assign some key (like the Menu key) to be the Compose key. Hitting the Compose key puts the keyboard momentarily in "deadletter" mode, so you can then type ' and e to get the accented letter é, and similar for other accent/letter combinations.
