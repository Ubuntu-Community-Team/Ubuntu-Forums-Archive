### Post by 5-HT on 2006-02-26
DiscoKiller has some good advice.
With the default standard installation, I believe Ubuntu does not ship with any open ports listening on the outside.
If you install certain packages though, like an SSH server...then of course you'd be opening yourself up.

My opinion is, if it doesn't hurt anything, why not install a firewall just for extra comfort (whether or not it is actually helping)?

As DiscoKiller mentioned, Firestarter is a nice, easy to configure firewall.

Note that when installed and configured, it's running all the time.
You do not need to keep its GUI open all the time (a lot of people new to it assume that you do).

Hope that helps.
