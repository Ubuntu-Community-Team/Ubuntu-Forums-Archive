Hello I installed Gutsy x64 on my friend's desktop (C2D E6700, 1GB RAM, i945 onboard) He is migrating from Windows. When he opens any text file, a dialog is displayed asking whether to "Run in Terminal", "Display", or "Run" it.. How to set it to display it by default, since he is somewhat bothered by that dialog..?

---

### Post by Inxsible on 2008-03-28
Right click a text file and then click properties. Go to the Open With tab and select your fav text editor program (like gedit in gnome or kate in KDE or any other for that matter) from the choices. That would become the default program for text files then.

Oh and in case the program doesn't show up in the list, you can click Add and find it in /usr/bin

It's in the Nautilus preferences, under *Behavior*. You can set it to display without asking.
