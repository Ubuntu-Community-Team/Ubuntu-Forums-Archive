I just got the wlan0 interface working for my Dad's Linksys wmp11 v4 wireless card!  Thanks to the Ubuntu ndiswrapper docs [(located here)]("https://help.ubuntu.com/community/WifiDocs/Driver/Ndiswrapper?highlight=%28WifiDocs%2FDriver%29").

Happy to join the Ubuntu ONLINE community!  

I have a home network with a Comcast Netgear router, so if you have a similar setup, try re-setting the router to NO WEP.  That's what worked for me.  

If anyone has gotten this combo to work WITH WEP, please post a note here telling what settings (64 bit, 128 bit, etc.).

We are now a complete NON-WINDOWS household.  I have a Mac running OS X and Dad's running Linux.

Unix (in all it's flavors) ROCKS!
