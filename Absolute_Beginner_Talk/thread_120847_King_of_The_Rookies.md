### Post by Cephyr on 2006-01-23
Ouch! That's a pretty crappy README.

You should go into the terminal (look around for some good tutorials on getting started with the terminal if you haven't already), navigate to the unpacked folder, and look around for a file called MAKE.

Try typing "make" into the terminal run the compiling program. If it whines about make not being installed, use Synaptic to find the package "gcc" and install that.
