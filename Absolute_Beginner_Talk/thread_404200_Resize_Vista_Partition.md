### Post by confused57 on 2007-04-08
With an increasing number of people who will be using Vista, they need to be made aware not to resize the Vista NTFS partition using gparted...but need to use Vista's own partitioner to shrink the partition before installing Ubuntu.
This topic was discussed pretty well in this thread:
[http://ubuntuforums.org/showthread.php?t=391186](http://ubuntuforums.org/showthread.php?t=391186)

I don't know if a warning should be added as a Sticky or not, but I've noticed more people attempting to resize their Vista partition using gparted as the Ubuntu install cd instructs to do.  They then end up with their Vista install being unbootable.

### Post by poohbear1616 on 2007-04-08
And a note about restore disks.
Some of the factory built vista puters do not come with a pre-made restore disk but you are allowed to make a one time copy for safe keeping. So make sure you have a this disk before tinkering with vista partitions.
