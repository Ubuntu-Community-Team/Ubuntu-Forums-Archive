### Post by Perfect Storm on 2007-10-19
ok, try this:

```
sudo aptitude update && sudo aptitude upgrade
sudo aptitude install nvidia-glx
sudo nano /etc/X11/xorg.conf
```

go to the
[b]Section "Device"
Identifier "nVidia Corporation C51 [Geforce 6150 Go]"
Driver "nv"
BusID "PCI:0:5:0"
EndSection[/b]

and change the driver to **nvidia**

[b]Section "Device"
Identifier "nVidia Corporation C51 [Geforce 6150 Go]"
Driver "nvidia"
BusID "PCI:0:5:0"
EndSection
[/b]


Save = [ctrl]+[o]
exit = [ctrl]+x]

restart.

If it fails use

sudo nano /etc/X11/xorg.conf and change the driver back to nv.
and remember that linux is case sensitive.
