### Post by kcox5342 on 2008-04-13
I've never tried Linux with such older hardware, but you might want to consider one of the unofficial Ubuntu derivatives with a minimal window manager, like Fluxbuntu.  Only down side is, Fluxbuntu isn't a live CD, so you'd have to try it by installing it.

I have tried Damn Small Linux, never got it working, but I did enjoy dsl-n ("Damn Small Linux-Not!"), which is a little bigger (the original DSL wants to fit in 50MB, dsl-n is like 100MB), that does run as a live CD.  It's quite a bit different than Ubuntu, but it does fit a lot into such a small package.  I could load the OS into RAM and still have plenty of space left over with only 512MB RAM... so it might even fit into 160.  

Good luck!  Since this is going to be a machine to tinker with, you should have lots of fun.

I'd strongly recommend replacing the 32MB completely. 128MB would be just enough but, if it's got two slots, two of them would be better than adding 128MB to the existing 32MB.
