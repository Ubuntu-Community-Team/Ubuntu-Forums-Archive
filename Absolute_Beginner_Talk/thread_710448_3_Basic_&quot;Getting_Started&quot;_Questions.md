> 1.  I've enabled "all" software provided at each tab in Synaptic (e.g., 3rd party, archives, uni & multiverse, etc).   That generates more "updates". Possible but unlikely. The applications available in these repositories usually don't affect the base system. There are some exceptions (Virtualbox, for instance), but these are generally mature and stable projects themselves.

> 3.   What would be the equivalent apps in gnome for:
> K3b
> kmplayer
> codiene

Gnomebaker or Brasero for optical disk burning.
Mplayer or VLC for video.
sensors-applet for the last item.
