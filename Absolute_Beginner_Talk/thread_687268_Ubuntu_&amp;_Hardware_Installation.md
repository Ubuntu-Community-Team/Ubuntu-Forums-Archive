### Post by bumanie on 2008-02-04
Have you looked at the mythtv site/s, there is heaps of information on set up, especially with regard to video cards - re which work and which don't. The majority of chipsets are supported at kernel level. Also, the most recent kernel has had lots of work done on it to improve wireless connectivity. Look at this
[http://www.mythtv.org/modules.php?name=MythInstall](http://www.mythtv.org/modules.php?name=MythInstall) and this
[http://www.mythtv.org/wiki/index.php/Video_capture_card](http://www.mythtv.org/wiki/index.php/Video_capture_card)
Hope this helps. Best thing for you to do is install the hardware and software and ask questions if you get to a sticking point.
