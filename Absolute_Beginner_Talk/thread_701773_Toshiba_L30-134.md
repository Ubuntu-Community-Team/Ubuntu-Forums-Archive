### Post by genus_001 on 2008-02-19
Every installation of Ubuntu I have installed has Openoffice.org in there by default.  To find the word processor, it should be under Applications>Office>OpenOffice.org Word Processor.   If it is not there simply go to Applications>Add/Remove and type Openoffice.org in the search box.  The openoffice.org applications should show up in the "Application" list.  If it is not installed, there should be no check in the box next to it.  Select the program, and click on "Apply Changes".  After it is installed, it should be in the location listed above.

I hope this helps.
