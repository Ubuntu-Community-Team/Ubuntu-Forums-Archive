### Post by brash on 2006-10-24
You might have to go into BIOS (when you reboot the computer, there is usually a quick message to** hit DEL or F1 or something for SETUP, and that is it)** and under **BOOT options**, tell it to boot off the CD before the hard drive. (Mine is now set to Floppy, then CD, then HDD)

I know on one of my very first disks I made, I didn't understand exactly how to burn an ISO disk, and did my first one wrong (I told it to make a *bootable disk*, which is actually NOT what you want to do; on the program I used the correct way was to just go under the menu FILE option and select to burn *from an ISO*).

Also if you see only a single file instead of many different folders, it means you copied the ISO as a file instead of as an ISO image.

### Post by brash on 2006-10-24
Usually you can tell it several options to pick, if you put "CD" ahead of HDD (harddisk), then it boots off the CD *only if there is a bootable CD in the drive* and if not it proceeds to boot off the hard drive as normal.

So I don't have to change mine back and forth, I just have to remove the LiveCD when I want to boot normally.
