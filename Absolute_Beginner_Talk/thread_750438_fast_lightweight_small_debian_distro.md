### Post by kwacka on 2008-04-09
You are asking two questions:

1. A small distribution to which you can add packages - as has been said DSL or Puppy.

2. Debian-based - go for Debian. DSL was Debian-based, but bears little comparison to it now, don't expect to be able to add current Debian packages to it. Likewise with Slackware/Puppy.

Of course, Ubuntu is Debian based.
