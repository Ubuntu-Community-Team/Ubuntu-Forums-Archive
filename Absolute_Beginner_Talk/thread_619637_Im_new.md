### Post by mdpalow on 2007-11-21
Ensure you partition correctly during install. I'll give you an example to think about:

80 gig hard drive - 

Mount Point, File Type, Size

/,                    EXT3,           10000 megs or 10 gigs primary at the beginning
/home,            EXT3,           6800 megs or 68 gigs primary at the beginning
/swap,            n/a,             2000 megs or 2 gigs put this at the end

So, basically three partitions created...
----------------------------------------------------------------------

Be sure your Laptop has a CAT5 connection while installing Live CD, so install will grab on-line files/drivers, and setup correctly.

----------------------------------------------------------------------

Don't forget to install proper DVD and Codecs files if you want to play a DVD on your laptop. See my signature below.

----------------------------------------------------------------------

Don't forget to click on and enable any restricted drivers after install

----------------------------------------------------------------------

That should help some...
