### Post by pmj on 2005-10-06

### Post by landotter on 2005-10-06
[quote=pmj]Unless you're talking about some other kind of icons, you don't need any special application.

The easiest way is to just drag and drop the file containing the icons on the Theme Preferences window.[/quote]

If that doesn't work, just put the uncompress icon directory into /home/yourname/.icons. To access this directory, open up your file manager to your home, press crtl+l (L) to get the location bar, and enter the directory .icons. The dot indicates it being a hidden folder. 

There is also a directory named .themes where window decorations and themes are stored.
