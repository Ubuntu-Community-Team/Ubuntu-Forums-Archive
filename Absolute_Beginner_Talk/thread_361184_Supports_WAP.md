### Post by rverrips on 2007-02-14
I'd strongly recommend NOT to use wap for browsing a site like ubuntuforums.org as wap is designed for light content browsing, and ubuntuforums.org has lotsa of content, and lotsa text, and chances are your mobile phone will run out memory downloading the pages ... 

However, you can browse any site as wap simply by browsing with wap to 

[http://www.google.com/xhtml](http://www.google.com/xhtml)

Type in the site as search, i.e. ubuntuforums.org and select the first option, which will give you something similar to this link, which is a wap optimised version of ubuntuforums.org

[ubuntuforums.org for wap]("http://www.google.com/gwt/n?mrestrict=xhtml&site=search&q=ubuntuforums.org&source=m&hl=en&ei=gY7TRd-MGL28wQHBt6kM&ct=res&cd=1&rd=1&u=http%3A%2F%2Fwww.ubuntuforums.org%2F")
