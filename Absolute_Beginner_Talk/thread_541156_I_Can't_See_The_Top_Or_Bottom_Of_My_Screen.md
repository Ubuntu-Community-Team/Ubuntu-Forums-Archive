### Post by bobplano on 2007-09-02
try
```
sudo dpkg-reconfigure xserver-xorg
```

if you can't find your resolution there then do
```
gksudo gedit /etc/X11/xorg.conf
```

to open a terminal if you can't get to a menu press Alt+F2 and put in gnome-terminal
