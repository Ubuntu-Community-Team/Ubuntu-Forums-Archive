### Post by ubuntu27 on 2008-04-01
So, you where using Wubi, and then later decided to do a real Ubuntu installation?

It says that there was a error with the disk, so yes, you will have to download the iso again. 
I recommend you to download using a bittorrent client instead of downloading trough http or ftp. Bittorrent compares every bit downloaded with the original, preventing File corruption that is common when you download using http (downloading with the browser normally).

If you are using WIndows, I recommend you to use [µTorrent ]("http://www.utorrent.com/").

The file you have to download ends with torrent. Example: "ubuntu-8.04-beta-alternate-amd64.iso.torrent"
Then open the torrent file using µTorrent (or any other bittorrent client)

[Go to this link]("http://www.ubuntu.com/getubuntu/downloadmirrors")
Select a location near you
Download a file the ends with torrent. pay attention to the names. There are desktop cd (a.k.a Live CD), ALternate CD (text based graphical install). amd64 (for 64bit processors), and i386 (for 32bit processors)

[Downloading and burning an ISO of Ubuntu]("http://www.psychocats.net/ubuntu/iso")

### Post by chewearn on 2008-04-01
Nowadays, I find CD-R to be very iffy in reliability.  The production of these discs has gone so cheap, quality is suspect.  The product label on the discs don't mean anything, as most brand seller source these discs from only a few huge factories, based on contract price.  Disc from two different brand could actually be from the same factory, and of the same quality.

So, how do you go around the problem?  Burn the iso into a DVD+RW.  You use only 700MB of the 4.2GB, you spend a little more $, but you will get a good disc, most of the time.  And the discs can be rewritten for other uses.

To check the quality of the burn, run the "Check CD for detects" option in the livecd boot menu.
