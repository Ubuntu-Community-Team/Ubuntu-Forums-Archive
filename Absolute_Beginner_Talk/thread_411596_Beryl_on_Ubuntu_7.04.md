### Post by joeblow1102 on 2007-04-17
Compiz comes pre-installed with Ubuntu Feisty Fawn.  To enable this, go to System > Preferences > Desktop Effects.  Enable Desktop Effects and put check marks in both of the boxes.

If Compiz works, it's really easy to install Beryl using the repositories.
