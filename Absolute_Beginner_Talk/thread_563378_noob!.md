### Post by lisati on 2007-09-29
Welcome to Ubuntu. 

Don't be scared to ask questions - there's a good chance that someone here on the forums will be able to point you in the right direction. There's also a great wealth of experience available.

If you haven't already done so, you might want to try Ubuntu from a "Live CD". Although it will probably be a little bit slower than installing Ubuntu "for real", it will give you a chance to have a play, and get a feel for Ubuntu. You can get a copy from [http://www.ubuntu.com/getubuntu](http://www.ubuntu.com/getubuntu)

### Post by macogw on 2007-09-29
Stumbling around is the BEST way to learn ANY computer system!  Just don't execute as root any commands that you haven't already explored.  You make a command have root access by putting "sudo" in front of it.  If you try all the commands without "sudo" first to see what they do / how they work, you're far less likely to screw up the system.  If you want to know how a command works, type "man $command".  So if you want to know how "apt-get" works, do "man apt-get" and it'll display a manual page for it.  Manpages can be a bit daunting to read at first, but that's why "man man" exists (no joke).
