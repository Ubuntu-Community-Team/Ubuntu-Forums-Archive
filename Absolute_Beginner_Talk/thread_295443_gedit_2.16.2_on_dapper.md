### Post by Sef on 2006-11-08
Dapper is LTS: Long Term Stable.  That means that packages aren't likely to be upgraded or upgraded very slowly.  If you want the latest packages, then you would have to install Edgy, which is unstable, but the packages are newer.  If you do install Edgy, **back up** your files first.
