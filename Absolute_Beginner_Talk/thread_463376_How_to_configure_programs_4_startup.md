### Post by starcraft.man on 2007-06-03
Easy enough, the generic means is to the System Menu > Preferences > Sessions > New.... 

The name will be what it is called in the list. the command is what program will be opened, it is done by terminal command. So, for the two programs you listed the commands would be "azureus" and "gaim" respectively. Usually it is the same as the name of the program, there are exceptions though.

Thats it, easy enough :).
