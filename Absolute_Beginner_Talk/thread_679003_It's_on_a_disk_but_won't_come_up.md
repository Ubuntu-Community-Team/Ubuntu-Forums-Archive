### Post by Presto123 on 2008-01-26
First thing about when you burn .ISO's is to always make sure that your program is burning it as an "Image" file. By default, some programs, like Nero, burn as an executable file and it can lead to these problems.

With Windows, you can backup with their backup utility, but depending on how much you need to back up, it could take several CD's to do it. You might also find some online places that you can store files on and go back to download it, but I don't always trust in 3rd party file servers. I'll let someone else handle this one.
