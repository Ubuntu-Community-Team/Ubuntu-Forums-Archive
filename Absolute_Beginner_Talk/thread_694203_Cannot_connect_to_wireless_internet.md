### Post by Presto123 on 2008-02-11
In order for wireless to work correctly you really need to have a wired connection first so that you can install the drivers from the internet. To do this, you need to go into System/Administration/Restricted Drivers Manager and see if the card even shows up. Being a USB type, it may not yet.

Let us know what you get, or if you have already done this.
