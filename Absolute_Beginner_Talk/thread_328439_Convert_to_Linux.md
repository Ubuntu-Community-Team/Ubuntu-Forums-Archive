### Post by Frak on 2006-12-30
Almost everything you used in Mac or Windows, there is an alternative in Linux, that is totally free and open source.

Ubuntu has the Linux Kernel, all the programs are based of GNU, so you might sometimes see the name GNU/Linux, Windows has the system32 kernel, in which a kernel tells the computer how to make things work, how the software interacts with the hardware.

Ubuntu was created by Mark Shuttleworth, with help from Mako Hill, it believes in totally open source, even though we will all use a proprietary driver, its a good moral to live off of.

Ubuntu isn't a "release" it is a distrobution, which there is a new one everytime the wind blows. Of course you can play games on Linux, as you can on Windows and Mac OSX, and some of them are stunning!

And if you want to learn more about Open Source and Proprietary apps, just to a search on it, and you'll find your answer, very quickly, its been discussed infinately it seems.

And please if there is more you would like to know, post back, I love answering questions like this.
