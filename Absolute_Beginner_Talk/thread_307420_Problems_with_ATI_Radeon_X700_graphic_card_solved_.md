### Post by 56phil on 2006-11-26
Hello and welcome. I have a X700 in a Gateway M680 laptop. 1024x768 was the best I could get without installing the proprietary driver from ATI. Here are the entries that must be made on the terminal:
```

sudo apt-get install xorg-driver-fglrx
sudo aticonfig --initial
sudo aticonfig --overlay-type=Xv

```
Once that is done You will need to edit your /boot/grub/menu.lst file: and replace each occurrence of 'quiet splash' with 'vga=788'. Actually, you don't need to replace 'quiet splash', but you do need to add 'vga=788' to the end of each line that has quiet splash in it; those lines start with the word kernel.
```
sudo gedit /boot/grub/menu.lst
```
Finally, you need edit the /etc/X11/xorg.conf file to add the resolutions you want for your screen:
```
sudo gedit /etc/X11/xorg.conf
```
It will look something like this:
```

#
# This file was generated by dexconf, the Debian X Configuration tool, using

# if it has not been modified since the last upgrade of the xserver-xorg
```

The only thing I did was add "1680x1050" to each display subsection. Then restart the xserver <ctl><alt><backspace> and you're done.

I hope this is what you're looking for.
