### Post by tubasoldier on 2006-10-25
[http://www.ubuntuguide.org](http://www.ubuntuguide.org) is also a good resource.

My question for you is why do you want anti-virus software? clam works well for mail servers and other server implementations to help keep viruses off the windows computers that connect to us. Dont get me wrong. it is certainly a good idea to install it. But in linux we are not to the point where it is necessary. I think there are only around 20 viruses or so for linux, half of which only exist in a lab. And to get one you have to download it, install it as root, then give it permission to run, and then execute it. Just a thought.
