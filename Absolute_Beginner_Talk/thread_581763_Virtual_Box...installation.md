### Post by Dr Small on 2007-10-19
When using VirtualBox, you create a "virtual hard drive" to install your OS on. This "Virtual hard drive" is nothing more than a file that VirtualBox reads to bootup that OS from.

Installing to a virtual hard drive does not affect your current installation, patitions, or hard drive in any way. It is all on a single file.

Dr Small
