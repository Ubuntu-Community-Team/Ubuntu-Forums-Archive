### Post by dfreer on 2007-11-10
Wow, this thread is confusing. To clear things up, Wine lets you run windows programs on linux, but it does not work with every program. You do not need a version of windows installed for wine to work. These games are not considered to be native, because they require wine to work.

Wine is not an emulator, btw.

Dual-booting is a great idea, if you are a gamer. However, that doesn't mean that you cannot game on Ubuntu. I play Guild Wars, Counter-Strike: Source, Half-Life 2, Doom 3, Unreal Tournament 2004, and console games (NES, SNES, PSX, N64) in ubuntu all the time :)

Some games, like Unreal Tournament 2004 and Doom 3, are windows games, but they also have native linux versions. They do not require wine to run.

PCSX can play playstation games, as well as ePSXe and pSX emulators. Are you having problems getting PCSX to work?
