### Post by silkstone on 2007-06-16
I'm not sure you can do it, unfortunately. Outlook saves everything to a huge .PST file which (as far as I know) only Outlook can read. You can export as other formats, but to do this you have to be able to run Outlook.

My only suggestion is to copy the .PST file to a USB stick and ask someone with access to Windows to run it and convert the files, after backing up his own .PST file first of course!

### Post by hartdg on 2007-08-12
You can find the .pst files by searching for *.pst

Microsoft normally bury them deep in the "Documents and Settings\<user>\Application Data\Microsoft\Outlook" or similarly hidden and obscure locations.

To migrate them to Evolution look at this thread.
[http://ubuntuforums.org/showthread.php?p=3178434#post3178434](http://ubuntuforums.org/showthread.php?p=3178434#post3178434)
(you will need a friend who has Windows and MS-Outlook installed).

The other posts that I've read generally indicate that Linux utilities that can extract data from a .pst file are not being actively developed or can only read the older file versions.
