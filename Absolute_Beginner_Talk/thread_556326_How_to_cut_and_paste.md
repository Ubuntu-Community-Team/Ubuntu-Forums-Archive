Cut copy and paste are also available on the context menu (right-click mouse) of nautilus file manager. 

Openoffice has the very same editing capabilities as MS Word

### Post by mali2297 on 2007-09-21
Highlight the text you want to copy (hold down left mouse button and drag). Then move the cursor to the place where you want to paste, left-click  and middle-click.

### Post by argie on 2007-09-21
AFAIK, the only tool I know that can do that is ksnapshot. But installing that will require the kdelibs and some other stuff an extra 22.4 Mb, so I think if you want to install use this:
```

sudo aptitude install ksnapshot
```
instead of the apt-get you use usually.
