### Post by ugm6hr on 2008-01-09
I would second keyjnote ([http://keyjnote.sourceforge.net/](http://keyjnote.sourceforge.net/))

The transitions are no more impressive than Powerpoint etc, but it has some useful features for presentations, including spotlight, highlight, show all slides on one page and zoom in on selected slide etc.

It can be installed from Synaptic (make sure you also install xpdf-reader and pdftk):
[http://ubuntuforums.org/showpost.php?p=4076058&postcount=3](http://ubuntuforums.org/showpost.php?p=4076058&postcount=3)

To run it, I right-click in Nautilus, and choose a custom command (from Open with Other...): *keyjnote*

Having done that, you can right-click on any pdf and choose keyjnote to run the file in keyjnote.
