### Post by joe.turion64x2 on 2007-06-19

You may search the forum for that, it has been widely covered. Anyway I'll tell you that you need to install libdvdcss, xine, xine-plugins, and more plugins. Totem is not really capable of playing DVDs, try totem-xine instead. And, by the way, don't worry if you don't find all the above packages listed in Synaptic, you have to enable another repo (medibuntu) first.

As I said, search a little, there is plenty information out there.
