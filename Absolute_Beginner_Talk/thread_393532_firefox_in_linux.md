### Post by retaliator666 on 2007-03-25
Is there a way to make firefox on ubuntu act like the one on windows?
What I mean, when clicking in windows in the url space the url is automaticly selected. When clicking in ubuntu in the url space it places its cursor in the text, and you have to doubleclick to fully select the url.
I find this anoying

---

### Post by aysiu on 2007-03-25
There are a few ways to do this:

1. **Change the default behavior**
In a new tab, type ```
about:config
``` Then search for *browser.urlbar.clickSelectsAll* and change it from *false* to *true*

2. **Be More Efficient**
Avoid the mouse altogether and press Control-L or Alt-D to highlight the address bar.

3. **Get used to something else**
If you click the icon to the left of the web address, that will also select the entire address.
