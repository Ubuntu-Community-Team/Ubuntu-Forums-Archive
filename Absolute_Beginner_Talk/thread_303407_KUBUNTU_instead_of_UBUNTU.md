### Post by ReaderRat on 2006-11-20
If you just want to use Kubuntu as your desktop (with apps), this will do it:
```
sudo aptitude update
sudo aptitude install kubuntu-desktop
```
You can then use Kubuntu (KDE) OR Ubuntu (Gnome).

Otherwise, I would do a clean install of Kubuntu (this is only what I would do).

I installed from Kubuntu live CD, after that I tried gnome and I liked it better. KDE is nice but always feels more slugish. Gnome looks better and I prefer Nautilus over Konqueror for the most.
