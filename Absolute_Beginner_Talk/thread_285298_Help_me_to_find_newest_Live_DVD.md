### Post by Fully216 on 2006-10-26
No, your hard disk should be fine if you run the live CD.  If fact, the CD is designed so that you will not use the hard drive at all when you boot from the live CD, that is the beauty of it.  It lets you try out the distro before making any big changes like formating drives and such.

A new release of ubuntu just came out (Edgy 6.10) a couple days ago.  It will be supported for 18 months.  Download links can be found here:

[http://www.ubuntu.com/products/GetUbuntu/download#currentrelease](http://www.ubuntu.com/products/GetUbuntu/download#currentrelease)

A slightly older release (Dapper 6.06) came out a few months ago and offers long term support, 3 years (LTS).  Download links can be found here:

[http://www.ubuntu.com/products/GetUbuntu/download#lts](http://www.ubuntu.com/products/GetUbuntu/download#lts)

I think you will love it once you have tried ubuntu; quite possibly become a convert like me and drop XP for good.  :D
