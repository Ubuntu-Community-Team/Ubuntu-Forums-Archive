### Post by JoshuaRL on 2008-01-07
Yeah.  Pray.

Seriously, find out what the exact kind of video card is and search the forums for any help that you can find.  You'll either need the proprietary "fglrx" or the open source "ati" driver.  You might want to backup your xorg.conf file before upgrading so you have something to refer back to.  But I've heard that Hoary (the one coming out in April and in testing now) works really well with laptops.  So if you don't necessarily need to mess with the version right now, you might think about waiting for Hoary.

But if you have any problem just ask here, we're ready to help.
