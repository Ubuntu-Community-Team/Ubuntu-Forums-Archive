### Post by K.Mandla on 2007-09-05
> **kevindubrow said:**
> I've heard some things about Fluxbuntu being unstable and not very user friendly (the person who will be using this computer is new to Linux). Is that true? I know xBuntu looks a lot similar to Ubuntu, so that may be what I go to if what I have heard about Fluxbuntu is true.

Well anyway, does Xbuntu have any disadvantages over Ubuntu? And is there a version of Ubuntu Studio for Xbuntu or maybe just a similar project? Thanks.
None whatsoever, and after looking at the system specs in the other thread, Xubuntu should run like a champ. Give it a try first, or you could install it with *sudo aptitude install xubuntu-desktop*, and remove it if you don't like it.

Both systems will give you all the software that is shared between the three, and really aside from appearance, there's not much that's different. Each one behaves identically at the core level; it's just the interface that is changed.

Having said that, you might find Xubuntu easier to handle than Fluxbuntu at first. Fluxbuntu isn't more difficult, but it does have a less comprehensive approach than either Ubuntu or Xubuntu. You might save it for a time when you feel more confident adjusting it. 

And to be honest, I'd never heard anything about it being unstable. I use it from time to time, and I like it.

I'm afraid I don't know enough about Ubuntu Studio to answer your last question; perhaps someone else will chime in.
