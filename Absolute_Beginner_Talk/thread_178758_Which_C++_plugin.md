It should be fairly obvious what to run, no? x86 is regular Intel and AMD PC processors, ppc is what goes into some servers and wnt into the Macs before they switched to Intel, x86_64 are the AMD64 processors (and Intels EMT 64) and ia64 are heavy server CPUs.

If you have a PC you probably want x86.

### Post by zerhacke on 2006-05-18
That depends on your processer.  If you're using an Athlon/Duron/Pentium II/Pentium III/Pentium IV, the X86 package is the one you want.

If you're running one of the AMD 64 bit processers, the X86_64 package is the right one.

Running a Mac, the PPC one is the one you want.

Running a Intel 64 bit processer, the IA64 package is the one you want.

It all depends on your processer.
