### Post by Bliepo32 on 2007-10-10
Did you already try rebooting from the terminal. You can do it by the following code:
```
sudo shutdown -r now
```

If that doesn't work, try logging-out and reboot from there. You could also try making a different user, and see if you expierence the same problems with that one. It can also be that something is stopping your computer from rebooting. This can be anything from a broken driver, to a device which has been mounted in a wrong way.

It would be helpful if you posted some extra information. That will make it easier for others to help you.
