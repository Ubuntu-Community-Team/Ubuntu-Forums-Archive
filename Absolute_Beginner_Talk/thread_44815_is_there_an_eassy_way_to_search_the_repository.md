### Post by Kvark on 2005-06-27
There is a short text with info about each package, that should clear up any confusion about what it's for. The cathegories should help you find packages for a specific purpose. If you select "look in: description and name" when searching for.. for example "mp3" then you should find all programs whose descriptions say anything about being able to convert/play/do things with mp3 files.
