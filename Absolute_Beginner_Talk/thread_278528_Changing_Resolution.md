### Post by az on 2006-10-16
> **Nactive said:**
> Hey,
There are only 3 resolutions: "800 x 600", "640 x 480", "400 x 300". So I can't change my resolution to 1024x... or any thing else higher. (btw I have a 17" monitor).

Gz,

Nactive!


Ubuntu uses the highest color depth by default.  On older computers using video cards with little video memory, you need to lower the color depth to use higher resolutions.

Boot into recovery mode and run

dpkg-reconfugre xserver-xorg
and go through the menu.  Just take the default for everything except chosing a lower color depth (16-bit or even 15-bit)

when you are done run

init 2

to get to the desktop.
