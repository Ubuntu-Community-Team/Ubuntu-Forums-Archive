### Post by arctic on 2006-03-23
Basically, .exe's don't work in Linux as the .exe files are built for Windows, a completely different operating system and thus incompatible. Most apps are available in some kind of Linux alternative through synaptic. You can run some .exe files in Linux, but only if you emulate a Windows environment in Linux, e.g. with Wine. However, not all applications can be run in those emulators.
