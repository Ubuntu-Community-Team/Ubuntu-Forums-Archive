### Post by 3rdalbum on 2006-12-17
I assume that you'll be running Ubuntu on a Mac/PowerPC machine? If this is the case, and if you have any questions about installing and booting Ubuntu, make sure you address them to the PowerPC forum - they'll give more useful information :-)

Ubuntu Server edition makes it easy to set up a LAMP (Linux, Apache, MySQL, PHP) server, but that will leave you with just a command-line. If that sounds a bit daunting to you, you could install the regular Ubuntu edition and then add the server software that you want. Once you're familiar with how to set up and use the software, and using the command-line, then you could turn off the GUI.

Good luck!

### Post by hyper_ch on 2006-12-17
Hiya

I see that you want to run a webserver. Generally I recommend using Debian 3.1 Sarge as server software. I'm still a bit sceptical about ubuntu running as server. Debian just has proven it's really stable *ducks and hides behind the couch* ... :)

Anyway, for both - Debian and Ubuntu - there are "Perfect HowTos" written by Falko Timme.

You may want to look at those two:
[http://www.howtoforge.com/perfect_setup_debian_sarge](http://www.howtoforge.com/perfect_setup_debian_sarge)
[http://www.howtoforge.com/perfect_setup_ubuntu_6.10](http://www.howtoforge.com/perfect_setup_ubuntu_6.10)

Both tutorials are aimed at setting up a webserver (and providing ISP functionality). In addition there's also ISPConfig - a server webhost control software - that Falko and Till have developped.

So, my advice is, have a look at the tutorials at [http://www.howtoforge.com](http://www.howtoforge.com)

I have run now my debian server for nearly 18 months without any major hassle - and believe me when I started I had no clue either about linux :)
