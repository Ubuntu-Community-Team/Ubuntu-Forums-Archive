### Post by darkrose on 2007-04-25
Sounds like the disc was not burnt as a bootable image, easy way to check; open the disc in windows and check it's contents, if it has multiple folders and files on it, it's bootable. But if it has only the one file on it which is the .iso you downloaded it is not bootable.

Have a look at [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)
