### Post by gerardo1967 on 2007-06-12
I just want to express my satisfaction with Ubuntu 7.04.
I'm a first time  Linux User. I got Windows VISTA Home Premium, and decided that, if possible, I would try to find an alternative to this waste of money Microsoft C.R.A.P. After a little research, I decided to download and try Ubuntu. 

I started a clean install of both, first VISTA and after that Ubuntu on Saturday 6/9/07. The Clear install was a good idea, because, it took me a couple of times to catch up how the disk partition worked for the Ubuntu Install, so I messed up the partitions the first time and had to start all over :-(.  

Configuring VISTA, with all drivers, software updates etc, took me almost a full day, so I started playing with Ubuntu for a few hours ( 2 or 3 top on Sunday), 4 hours yesterday and 2 hours today, including Research.

Everything working great, with the following equipment.

Dell Dimension 8400
ATI RADEON 300 Video Card (The system put it on Restricted Hardware List, but I enabled it, and it works fine in 1280x1024 @ 60 Hz)
Sound Blaster Live! 24-bit 
1 Gig Intel NetCard
Wireless USB Keyboard and Mouse Set. DELL
HP Officejet 4215 all in one. (Scanner, Fax and Printer working like a charm)
Kodak Easy Share printer dock series 3 and Camera works fine also
Lexmark E234n Network Printer. (This one was a little tricky but it works fine now, I have to change the settings in a different Application)
A No Brand PC Web cam (I still have to find how to adjust settings)
DVD Rom and DVD Rom / CD-RW combo IDE.
SATA HDD.

The only thing I couldn't put to work was a wireless Net Card, so I ran the cable from the Router.
The card is a Netgear WN311T. I'll keep checking to see if in the future there are drivers available, but actually is not urgent since I can work with a cable from the Ethernet card to the Router.

In general, this is more than a pleasant surprise. If I knew it was that good, I would have saved the $ 159.95 I paid for Windows VISTA. At least I feel this will be my LAST TIME; Sorry Bill Gates, you will have to get even richer with somebody's else money! 

I still have to connect the MP3 player, but i feel confident it will work.
Overall, with the info I could read from other postings in the community I was able to get everything working. 

I hope this info could be useful for somebody, and if can help somebody to install any of this hardware please let me know.

Go UBUNTU!
