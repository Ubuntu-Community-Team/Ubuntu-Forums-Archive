### Post by BobSongs on 2005-12-02
Quick Answer

Edubuntu is like Ubuntu running a "Plus! for Kids", if you will. It comes with a few extra goodies that are oriented towards helping kids in their learning.

It features:

* The GNOME desktop (standard in Ubuntu)
* Educational games
* SchoolTool
* A different desktop image.

That's it. If you're considering Edubuntu, you're getting everything Ubuntu has to offer and more.

It's not like Kubuntu where the desktop is different.

Hope it helps.
