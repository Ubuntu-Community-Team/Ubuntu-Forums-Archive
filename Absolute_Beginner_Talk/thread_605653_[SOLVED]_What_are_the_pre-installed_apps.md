### Post by philinux on 2007-11-07
If ever you need a list of what you got installed just do this in terminal

dpkg --get-selections | grep -v deinstall > ubuntu-packages

It creates a file which lists all install apps.

Useful stuff  [https://help.ubuntu.com/community/FeistyUpgradesFreshInstall](https://help.ubuntu.com/community/FeistyUpgradesFreshInstall)
