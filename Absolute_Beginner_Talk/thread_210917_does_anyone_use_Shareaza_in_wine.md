### Post by Stealth on 2006-07-07
I just installed it yesterday. Works flawlessly using the tutorial in the App Database from winehq.com

As for ports, that router side and really has nothing to do with Ubuntu (unless you got a really funky setup). If it works in Windows, then Ubuntu should use it fine (and you change the port it uses in its preferences). Just make sure that port is open on your router. If you have a firewall, I suppose you need to make it allow Shareaza to pass through too.

Amule connects to Edonkey2000 networks aka ED2K (serverbased)
and KAD (serverfree).

I tried Shareazaa but every download I got, came from Emule clients.
So I quickly turned back to Emule. ( Windows )

Amule is Linux version of Emule.
There is also a version called xmule, but I don't think it has KAD.

You have global search option in aMule it's VERY powerful. :p
I have used Edonkey then Emule now Amule for years. From time to time I try something different. But nothing has been able to work as well as the ED2K network. except maybe Torrent, which can be very good in some cases.

Oh BTW
ED2K network can connect even with ports closed, it's just much slower because all requests are handled differently.
