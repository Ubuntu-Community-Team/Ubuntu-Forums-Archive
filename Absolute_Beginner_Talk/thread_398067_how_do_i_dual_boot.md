### Post by BarfBag on 2007-03-31
Dual booting is very possible.  I've run a dual boot setup for years.  It can be risky, though.  Back-up all of your data and defragment your hard drive.  Just to be safe, you should probably run disk check as well.

After that, pop in the Ubuntu CD and boot it.  Go through the installation until you reach the partitioning options.  DO NOT select the option that formats the entire drive.  Use the third option down - the one with the slider.  Slide it to how much you want to shrink your Windows partition.  Then, just continue the installation like normal.

Since you're new to Ubuntu, I recommend running [Automatix]("http://www.getautomatix.com/") after you reboot.  It adds support for MP3/DVD codecs, Flash & Java, etc.
