### Post by linuxwizard on 2008-02-16
Have you tried this

Generic Players
If your player is not in the player list, there is a good chance that you can still control it using FoxyTunes. Choose GenericPlayer list and try one of the playback buttons. If it doesn't perform the desired action, go to the Player > Options submenu and try choosing either "Generic Multimedia Player 1" or "Generic Multimedia Player 2".

I don't use Totem or RealPlayer not sure if this will work. Here is a list of supported players.
[http://www.foxytunes.com/firefox/supportedPlayers.html](http://www.foxytunes.com/firefox/supportedPlayers.html)
