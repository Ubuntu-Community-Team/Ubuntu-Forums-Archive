Start here [http://ubuntuforums.org/showthread.php?t=142716](http://ubuntuforums.org/showthread.php?t=142716) Read everything there first. People here will help but this is not paid support. Noone here works for Ubuntu. You are not entitled to support on this site because you have a copy of Ubuntu. Everyone here has a love for Ubuntu and wants to help as many peopl as they can to get it but, it helps if a person has tried to install or read on installing and has a specific question. Read the stuff in the thread and post a specific question or try to install and if you have a problem post your problem. You should probably try a Live CD first. This is a cd with Ubuntu on it that doesn't need to be installed to your hard drive. You put it in and start your computer. Your computer will run off the CD abd you can start to get the feel of Ubuntu.


If you don't have a copy of Ubuntu's Live CD you can get a Live CD and an install cd here [https://shipit.ubuntu.com/](https://shipit.ubuntu.com/)  They are free.
