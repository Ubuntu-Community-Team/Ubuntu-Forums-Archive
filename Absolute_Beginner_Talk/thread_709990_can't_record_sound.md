### Post by forrestcupp on 2008-02-29
In a terminal, type
```
alsamixer
```
and press F4 to show only the 'Capture' channels.  Use right/left to move to a channel and use the up/down buttons to adjust volumes.  Make sure that whatever channels apply are turned up.
