### Post by DerekVOF on 2007-10-18
Should be able to go to System -> Administration -> Screens & Graphics. Click on the "Model" pull down for Screen1. Select Generic from the manufacturers list, and then scroll down to select "Monitor 1280x1024" -- even if you've an LCD, it'll work fine. Then you should be able to select a higher resolution even with the restricted driver.
