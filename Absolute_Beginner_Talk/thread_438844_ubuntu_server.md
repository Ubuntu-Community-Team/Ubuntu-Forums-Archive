### Post by hyper_ch on 2007-05-10
A server doesn't need a gui :) and there's good reason for that:

(1) The less applicatiosn that are running, the more power is available
(2) The less applications are installed the less security holes there are :)
(3) All configuration on Linux is stored in files. So no gui is needed

In the end the server will have a better performance and is more secure without a gui and you can alter it to your needs without problems.
