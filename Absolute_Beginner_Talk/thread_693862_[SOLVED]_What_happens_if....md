### Post by TheBoat on 2008-02-11
It will allow you to do both.  You can simply boot off of the disc and see how you like it without wiping windows (In the boot menu, you choose to boot from CD/DVD instead of the hard drive). You can also use the disc to fully install Ubuntu.
