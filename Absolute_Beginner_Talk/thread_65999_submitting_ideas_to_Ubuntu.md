### Post by UbuWu on 2005-09-16
There a quite a few places where you can make suggestions to the developers. I will list some here:

[Bugzilla](http://bugzilla.ubuntu.com)  & [Malone](https://launchpad.net/distros/ubuntu/+bugs)
These are used to track bugs. Bugzilla is for programs from main, malone for programs from universe. If you experience any problems with a program the developers should know about, file a bug here. It is also possible to file bug for enhancements.

[Mailing list](http://lists.ubuntu.com/mailman/listinfo/) 
Suggestions with more far reaching consequences can be discussed at the development mailing list.

[UbuntuBelowZero](https://wiki.ubuntu.com/UbuntuBelowZero/BOFs) 
Shortly after breezy is released, the developers will discuss the work to be done for the next release at ubuntubelowzero. At the moment you can make suggestions for things to be discussed there on this wiki page.

[IdeaPool](https://wiki.ubuntu.com/IdeaPool) 
Here you can always make suggestions and put your ideas down.
