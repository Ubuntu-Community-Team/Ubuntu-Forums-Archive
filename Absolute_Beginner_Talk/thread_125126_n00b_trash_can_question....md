### Post by firehead on 2006-02-03
just open konqueror and go to settings -> configure konqueror
there you can set the prompt when deleting files. you can also activate the "show 'Delete' context menu..." to bypass the trashcan. i don't know of any other way to bypass it except by deleting files using shift+delete...

### Post by ardchoille on 2006-02-03
Open nautilus. Choose Edit -> Preferences and go to the "Behavior" tab. Have a look at the last two checkboxes in that tab under "Trash".
