### Post by kyphi on 2007-11-07
Go to System -> Preferences -> Removable Drives and Media.

Go to Multimedia, then Audio Discs.  Where it says 'totem %m' replace with 'rhythmbox' (without the quotation marks).

Leave Video DVD Discs to read 'totem %m'.

Enjoy.

### Post by kyphi on 2007-11-08
Since Xubuntu is organised differently to Ubuntu it seems that I cannot give you direct advice.

You can try this:  If you have a music file saved on your computer.  Right click on this music file and scroll down to Properties.  Then open the tab 'Open With' and select Rhythmbox.  If it is not there you will have to use 'Add'.

After that all the files saved in the original format will play in Rhythmbox.  If you have music files in other formats you will have to follow the same procedure again.

That might help.
