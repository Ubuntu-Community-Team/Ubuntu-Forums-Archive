### Post by into_311 on 2007-07-31
I would recommend Saurbraten

[http://sauerbraten.org/]("http://sauerbraten.org/")

and Nexuiz

[http://alientrap.org/nexuiz/]("http://alientrap.org/nexuiz/")

That video card should work from a latest driver compiled off nvidia's site against your current version of kernel source.  I'm not certain if Ubuntu's package manager (Synaptic) will have a new enough version that includes support for that graphics card.  So you may have to go through using the binary file off of nvidia's site there instead.

[http://www.nvidia.com/object/linux_display_ia32_100.14.11.html]("http://www.nvidia.com/object/linux_display_ia32_100.14.11.html")

For generic instructions on how to install it, read this readme file off their site:

[http://us.download.nvidia.com/XFree86/Linux-x86/100.14.11/README/chapter-04.html]("http://us.download.nvidia.com/XFree86/Linux-x86/100.14.11/README/chapter-04.html")

You can always post an thread here in the forum for help getting the nvidia driver installed also.
