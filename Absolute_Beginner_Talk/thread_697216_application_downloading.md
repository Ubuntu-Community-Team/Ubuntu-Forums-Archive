### Post by Linux_Man on 2008-02-14
Click reload, if that doesn't work fire up a terminal and type "sudo apt-get update" without the quote of course, that will prompt you for your password and will redownload all the lists of applications then try again.

### Post by taurus on 2008-02-14
Click the Preferences and make sure you have a check mark in front of all the repos except the last one, Source code, and the Cdrom in the window below.  Close it and press Refresh.  Now, see if you can find wine this time.
