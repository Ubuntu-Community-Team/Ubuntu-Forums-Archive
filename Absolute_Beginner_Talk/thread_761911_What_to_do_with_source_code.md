### Post by amazingtaters on 2008-04-21
Rather than using that client, why don't you try one that is in the reops, say Deluge or Ktorrent (2 of my favourites). If you really wanna get this client then scroll down towards the bottom and find which version you want (I'd go with the newest, 5.20 at the very bottom) and download the .deb file for that version. After you've done that you can double click the icon for the .deb and you'll get a nice easy GUI installer that does all the work.

### Post by Michael.Godawski on 2008-04-21
An easier way than the source code install is to use the synaptic package manager or the terminal.
If you are using gutsy gibbon bittorent should be installed by default.

```
sudo apt-get install bittorrent
```

A good site for installing guides on ubuntu is:
[http://monkeyblog.org/ubuntu/installing/](http://monkeyblog.org/ubuntu/installing/)
