You can then create a launcher on your desktop on in one of your panel with a right-click, then entering the program's name as the command to execute.

### Post by mcduck on 2007-03-17
> **Majin Zero said:**
> Kismet for one....darn can't recall the names of some others off the top of my head.

I have no issues using the terminal, only thing is I just don't know the commands.

What's a good way to figure out the command for a program?
Usually the command to start a program is just the name of the program. But if you want to check, use the 'which'-command. It will tell you the main executable file of your program. For example 'which firefox' will return '/usr/bin/firefox' (and as /usr/bin/ is in your default path you only need to type 'firefox' to start it)
