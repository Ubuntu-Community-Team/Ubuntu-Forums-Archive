### Post by Ikslewap on 2007-10-09
Addendum:

All of the above posts were extremely helpful and answered this question for me as well. I also just started using Ubuntu and was a little concerned as to why this may be happening.  I never had the problem in MSWin XP Prof. or any Linux distro so I looked into a bit.  A buddy of mine shined the light on this situation. He works heavily with MS doing a lot of MCSE/MCSA work. Yes, its true that you can control the sensitivity and this will fix the problem.  However, MS Windows apparently has a built in mechanism for laptop touch pads that automatically desensitizes the touchpad when you are typing so that it won't act this way.  I only wondered because I had never  adjusted the default sensitivity.   So, in the tradition of open source, lets fix this behind the scenes....
