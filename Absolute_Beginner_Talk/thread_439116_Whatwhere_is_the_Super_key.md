### Post by LaRoza on 2007-05-10
Gnome- Desktop Environment
Nautilus- Like Windows Explorer, shows folders and devices
X- A GUI 
Beryl- A pretty desktop environment
Compiz- A pretty desktop environment, not as stable a Beryl
Feisty- Release of Ubuntu 7.04 Feisty Fawn
Edgy- Previous Release of Ubuntu 6.10 Edgy Eft
Dapper- Ubuntu 6.06 Dapper Drake
Medibuntu- A repository for applications
Super- a super user is root, similar to an "administrator" account in Windows, the super key is typically the Windows key, between the ctrl and Alt keys on the left
gksudo vs. sudo- gksudo = gui sudo, sudo = command line sudo
Grub- Allows you to boot, like the MBR. 
DPKG- a command to install deb packages (dpkg -i file)
aptitude- A program for installing and removing packages
bash- "Bourne again Shell", the terminal, or console
mount- Make a drive usable, to unmount allows you to remove it, partition it, and format it. Mounting enables you to read and write to the disk
synaptic- A program for installing and removing packages
repository- Where you get programs for Ubuntu, you usually use the Internet to get them, but you can get them on disk (like me)

--edit On the next page, Acglaphotis, posted the ones I omitted, I inserted the definitions here to keep it complete and to prevent a hunt for terms.

I may not be technically correct with some definitions, but this should help. As you can see, the terms are not advanced concepts, but are confusing at first (I had no clue what "Edgy" was at first, and GRUB was one of the weirdest terms I had every heard, but I got used to it.)
