### Post by turbojugend_gr on 2006-11-26
1. You can install, cdlabelgen using synaptic.

2. If you are Lucky via Wine.

3. IF you need it, simply open synaptic and search for amarok. then install it. It will also install all th required packages (since amaroK is for KDE).

---> I would strongly suggest you to install [Listen]("http://listengnome.free.fr/"), it's as good as amaroK, has some more features too, and of course fit's much better with the Gnome desktop, since it is built using GTK. On the download section there's a repo, and on [their sourceforge site]("http://sourceforge.net/project/showfiles.php?group_id=161415") there's a deb package.

If you feel like using something awesome try Songbird, use [this simple guide]("http://www.ubuntuguide.org") to install it (click on chapter 1.6.30). It is simply the best audio player ever, watch [this video]("http://www.songbirdnest.com/screencast/") to make sure of it!

Feel free to ask any questions, if you feel like. Cheers, TJ.
