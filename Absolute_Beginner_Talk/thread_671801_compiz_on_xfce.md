### Post by PeterJS on 2008-01-18
Setting up compiz in xfce is pretty easy (I'm assuming it was installed with gnome). The settings manager is under Settings > Advanced Desktop settings. To test it, you can launch it from the run dialog (alt+f2) with:
```

compiz --replace

```
To get it to run at start up in Settings > Autostart, add a new entry you can set the name and description to what ever you like, and for the command use the same command as above to test it.
