### Post by tonyr1988 on 2007-01-10
Two other ones (other than OO.org Impress) that I've heard of are:

-[KPresenter]("http://www.koffice.org/kpresenter/") and
-[MagicPoint]("http://member.wide.ad.jp/wg/mgp/")

I'd try KPresenter first if I were you...MagicPoint isn't WYSIWYG (what you see if what you get) - it's all "scripted," so there will be a bigger learning curve.

I'm sure there are others, but those are the only ones I know of. But OO.org Impress is the only one I've used, so I'm not sure how good they are, but it doesn't hurt to experiment. :)
