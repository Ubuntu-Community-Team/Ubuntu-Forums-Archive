### Post by Scut_Farkus on 2008-02-16
Hmmm...well you could test your RAM with memtest.  Do you have your Ubuntu CD?  Just boot to that and run memtest.  If you want to be really thorough about it, you should run it for at least an hour (or let it do about 3 passes).  The memtest devs say you should let it run overnight to be the most thorough you can, but if you wanna do a quick test, do it for an hour.  If everything comes up okay, then the problem is probably not your RAM.
