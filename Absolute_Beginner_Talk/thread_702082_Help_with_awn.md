### Post by Scunizi on 2008-02-19
If you're not sure if it's actually installed type (sudo apt-get install awn) in a terminal.  If it's installed it will tell you, if not it will install it..

Now go to Applications/Accessories/Avant Window Navigator to start it.

To configure it more go to System/Preferances/Awn Manager
