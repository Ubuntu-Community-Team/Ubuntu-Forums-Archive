### Post by Shatrat on 2007-01-17
You need to install the drivers from nvidia, what you are using now is the generic open source "nv" driver which isnt as fast or as powerful and doesnt do 3D. 
Here is one guide, there are many out there if you dont follow this one and quite a few threads on the subject in the Multimedia forums here.
[http://doc.gwos.org/index.php/Latest_Nvidia_Edgy](http://doc.gwos.org/index.php/Latest_Nvidia_Edgy)
