### Post by Spr0k3t on 2007-08-17
Couple things you can do...

There are several other WYSIWYG editors out there, but none of them are any better than subpar (Visual Web Dev included).

NVU - WYSIWYG [http://nvu.com](http://nvu.com)

Aptana - Code Only [http://www.aptana.com/](http://www.aptana.com/)
BlueFish - Code Only [http://bluefish.openoffice.nl](http://bluefish.openoffice.nl)

Personally I love Aptana... one of the better ones I've used.  A major bonus is that it uses the Eclipse front end.  NVU will give you a good WYSIWYG, but it's buggy and tends to crash here and there.  The best pages are hand coded anyway.  Using an IDE like Eclipse with the Aptana plugin makes developing huge sites real easy.  

If that does not give you what you need, you can also install Dreamweaver via WINE.  Runs quite smooth.  Another WYSIWYG editor that works well under WINE is Homesite (aka Dreamweaver lite).

So you have plenty options... and I'm sure there are more out there.
