### Post by FakeOutdoorsman on 2007-12-07
I've installed Ubuntu on an ancient Toughbook.  I did a command-line install to keep things light and then manually installed things from there.  I ended up using Openbox instead of Gnome (Ubuntu uses Gnome as the windows manager and Xubuntu uses XFCE).  Xubuntu might be a little quicker than Ubuntu, but some people say that Xubuntu has been bulking up and slowing down a bit, but I don't know.  I took a look and it seems that Ubuntu works fine on your Inspiron 8100.  Since you're new to Ubuntu, I recommend a stock Ubuntu or Xubuntu install first.  If the computer isn't fast enough try reinstalling with a command-line install and customizing a bit.  Check these out:

[Installation/LowMemorySystems]("https://help.ubuntu.com/community/Installation/LowMemorySystems") [Ubuntu Wiki]
[Lightweight Gnome]("http://ubuntuforums.org/showthread.php?t=298835") [Ubuntu Forums]
[Looking for a lightweight Gnome alternative]("http://ubuntuforums.org/showthread.php?t=282452") [Ubuntu Forums]
[Dell Inspiron 8100 - Native vs. Non-Native Resolution]("http://ubuntuforums.org/showthread.php?t=15800") [Ubuntu Forums]

### Post by Kobin on 2007-12-09
Ubuntu Gutsy is a nice distribution and it probably should run ok on your computer, but otherwise if you want a fast distro with up to date packages for an old computer I would suggest TinyMe or PCFluxboxOS both built on PCLinusOS.
I have run both on a Toshiba 700 Mhz, 128 Mb ram. And except for Synaptic which of course will run somewhat slower on a machine like that, it is normal desktop experience for me.
That means:
Boottime to a ready desktop was 1 minute on this computer.
All the programs I need for office, internetbrowsing, Multimedia players and messenger.
And of course a nice look in both distros.

### Post by Kobin on 2007-12-09
I forgot to add that I tried 2 times with a cli install from the minimal install CD for Gutsy.
The installation took 1 1/2 hours in comparison with 30 minutes for TinyMe which is not a cli install.
After install boottime was 3 minutes to cli, and TinyMe was 1 minute to a ready desktop.
I think we should start recommending other distros than Ubuntu for people who have older hardware.
I like Ubuntu and have it running on a 1.67 Ghz, 512 Mb ram. It runs fine on that one!
Also if people want packages that are up to date and a system that seems more as complete desktop, we should not recommend Puppy or DSL unless the want to configure a lot themselves, or just have a liveCD to carry around or play with.
And don't get me wrong. I also use Puppy for saving files out of crashed Windows systems, so I like that one as well.
But "there is a place for everything", as they say...

Now if someone think I am completely wrong, I would like to hear about it. If someone could tell me how to make Ubuntu run well on older hardware I would be the first to do just that.:popcorn:
