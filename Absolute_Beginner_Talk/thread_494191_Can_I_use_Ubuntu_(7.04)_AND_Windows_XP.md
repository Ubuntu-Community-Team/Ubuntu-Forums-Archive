### Post by twiceasworn on 2007-07-06
GRUB will be installed along with Ubuntu (that is the program that allows you pick what OS you want to use).  Since you already have XP installed you are going to have to re-partition your hard drive to make room for Ubuntu.  Try a google search for **Dual booting Windows Xp and Ubuntu Linux**.  There is a ton of information about this setup out there.  Grab a walkthrough and start installing :)
