### Post by public_void on 2006-05-03
You can download a .bin file from the website [here](http://www.real.com/linux). This an executable file like an .exe and will install real player. You first have to give it permissions to run. In the terminal do

cd /path/to/file                                     
chmod a+x file.bin                                
./file.bin                                                     

The .bin file will guide you through the installation
