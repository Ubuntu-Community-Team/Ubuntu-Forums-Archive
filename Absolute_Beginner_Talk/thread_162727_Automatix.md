### Post by nalmeth on 2006-04-19
Yes it does do things like that. You were probably told that by someone who had their special config messed up, or lost some special repositories on their sources.list, but these things are being addressed one by one.
If you have basic needs and expectations, and have done a fresh install, automatix will be perfect for you.
Where it might cause problems is when you have tinkered and toyed with your ubuntu in some pretty technical ways, automatix might overwrite/screw up a lot of what you've done.
Just don't be easily scared by the critism it gets.
It's NEVER done anything objectional to me.

### Post by Bloch on 2006-04-19
Automatix worked fine for me. I think if you just 
(i) use it on a reasonably clean install of ubuntu (with no previous messing with repositories)
(ii) use preferably on ubuntu, not kubuntu
(iii) just use it on one or two items at a time, and only on what you need. Don't tick off everything and let it run. You can run the script again at any time.

then you will be fine. It saved me loads of time when I set up ubuntu for a friend. On my own computer however I had already begun installing codecs manually, and when I ran automatix I found rhythmbox didn't work. I don't us it anyway (I use amarok and sometimes xmms), and I can't be sure it was automatix's fault.

I also don't think it's a good idea to install the CD-eject when button pressed option.  
More details on automatix are in the special thread 
[http://www.ubuntuforums.org/forumdisplay.php?f=77](http://www.ubuntuforums.org/forumdisplay.php?f=77)
