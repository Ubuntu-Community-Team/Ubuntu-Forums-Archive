### Post by Kingsley on 2008-02-01
Right click a torrent file, click on Properties, go to the Open With tab, and select Deluge.

---

### Post by linux phreak on 2008-02-01
If you are using firefox  select "open with other" after you click a torrent file .Then navigate to /usr/bin and select Deluge.The torrent will be automatically opened by Deluge afterwards.
