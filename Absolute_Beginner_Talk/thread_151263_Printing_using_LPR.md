Ciao,

hi was able to install my hp710c on asus wl 500g as LPR printer on Ubuntu 7.10.
At first i get the driver of printer from hp and installed it.
I have simply installed lpr printers from add/remove on applications, than ive opened printing from system menu, selected my hp printer chenged "device uri", click on lpd/lpr host and put the ip address of the router (probably 192.168.1.1) , i used as name IP_192.168.1.1 but probably is not important.
