 Ubuntu is completely and legaly free.  It is liscensed under the GPL to ensure that it is free.  All of the software is also free.  Ubuntu can also do pretty much everything Windows can, and plenty of stuff it can't.  Personally, I find it can do everything I want, and I'm quite a demanding user.  It can run alongside Windows - Google for how to, or search the forums.  

The only possible legal problems are the codecs for MP3 and other propriatory Sound/Video formats - although no action I know of has been taken against them.  If you're really worried about that, you can install RealPlayer instead, and use it's codecs.

Ubuntu can cope with old hardware too, and there is a lot of equally legally free software available for Ubuntu.  You can download it off the internet and burn the CD, and order it on CD (it's very cheap) if you don't want to download - it's a 650 Mb download!

Hope that helps!
