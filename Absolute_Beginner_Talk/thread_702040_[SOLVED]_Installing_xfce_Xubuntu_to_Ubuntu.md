### Post by JoshuaRL on 2008-02-19
It's pretty easy to unistall, but you shouldn't need to unless you have little disk space.

```

sudo aptitude xubuntu-desktop

```
And if you want to uninstall then:
```

sudo aptitude remove xubuntu-desktop

```

You would choose it from the Login screen under Sessions as XFCE.

### Post by Scunizi on 2008-02-19
Dont panic.. log out of ubuntu again and when you get to the login screen (user name password entry screen) look for the icon that is labeled sessions or options or something like that.. click and choose sessions then choose xfce instead of ubuntu.. now enter your user name and password.. viola!
