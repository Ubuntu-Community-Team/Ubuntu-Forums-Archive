### Post by daradib on 2007-08-23
The package nautilus-gksu provides some functionality like this (See this image: [http://cache.lifehacker.com/assets/resources/2007/07/Nautilus-Menu.png](http://cache.lifehacker.com/assets/resources/2007/07/Nautilus-Menu.png)). You can right-click on a file and click on "Open as Administrator".
