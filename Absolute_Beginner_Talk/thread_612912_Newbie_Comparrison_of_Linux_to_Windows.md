[www.linuxcommands.org]("http://www.linuxcommands.org") is a good place to start to learn some basic commands. Depending on what distro you use, all of them have a file manager. For instance Ubuntu has Nautilus,  Kubuntu has Dolphin, Xubuntu has Thunar. If you go to the Places>>Computer, it will open the default file browser and show you what "My Computer" shows you.

Installation of software depends on which distro you are using since Linux is quite a generic term. There are multiple distros which do things differently.
