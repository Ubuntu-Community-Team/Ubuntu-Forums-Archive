### Post by reacocard on 2007-02-17
The Ubuntu installer is very well-made. It won't destroy any of your existing data (unless you tell it to). By default, it simply shrinks one of your existing partitions and makes a new one in the space it freed. As for a walkthrough:

1) Download ISO
2) Burn CD
3) Insert and boot from CD
4) Click 'Install'
5) Follow the wizard

It's extremely easy.

### Post by aysiu on 2007-02-17
You're right to be concerned. Unless you have a way to back things up, do **not** install Ubuntu or any operating system.

Since D:\ appears to be your backup partition, it would probably make sense to resize C:\ and create a new Ext3 partition in there for Ubuntu.

Read more about partition planning here:
[http://www.psychocats.net/ubuntu/partitioning](http://www.psychocats.net/ubuntu/partitioning)

Read about the installation process here:
[http://www.psychocats.net/ubuntu/installing](http://www.psychocats.net/ubuntu/installing)

In the meantime, keep playing with the live DVD. The more you practice with that, the more comfortable you'll be when you actually install Ubuntu.
