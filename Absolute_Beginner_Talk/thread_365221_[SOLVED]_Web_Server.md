### Post by louis_nichols on 2007-02-19
Sure. It comes by default. As I remember, both the graphical and the classic installer have options for erver mode installs. I may be wrong, as it's been quite a while since I've had to reinstall my system. Plus, the server install doesn't install the graphical stuff, just command line stuff. However, you can just run the default install and then follow one of these 2 steps:

[LIST=1]
[*]either install the command line version of Uuntu and then follow [these instructions]("https://help.ubuntu.com/community/ApacheMySQLPHP"). These work of course also if you do install the graphical desktop. Or,
[*]in graphical mode there is an even simpler way. You just have to open synaptic, the package manager, then go to File>Mark packages by task (I'm not sure about the exact name, cuz I'm not at my machine, but I can confirm at home). here you have a list of options, one of which is Lamp. This will install all the tools you need.
[/LIST]
