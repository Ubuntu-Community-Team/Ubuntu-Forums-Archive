### Post by whoismilan on 2006-12-18
After having recently (few weeks ago) been testing a large number of text editors (for programming mainly C++ and Python), I decided that the best for me is GVIM.

Some of the reasons I like it is that you can customize it into behaving however you want it to, it feels light-weight, and it is included in more or less every UNIX or UNUX-like system out there, so if you ever need to edit a text file on another distribution, you can count on that there is one which you know how to use. Another plus is that it is not only for UNIX and UNIX-like OSs, it is also available for Windows, which to me means that I am also able to use it e.g. in school.

As a down-side, it is not only a text editor, it is a text editor with a learning curve...

Also see this thread: [http://www.ubuntuforums.org/showthread.php?t=6762](http://www.ubuntuforums.org/showthread.php?t=6762)
