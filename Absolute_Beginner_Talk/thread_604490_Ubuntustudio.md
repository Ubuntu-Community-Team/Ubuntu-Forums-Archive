### Post by nhandler on 2007-11-06
Well, I don't know what Cubase and Reason is, but I can try and provide you with some info on ubuntustudio.

UbuntuStudio is similar to the normal ubuntu. The only main differences are the artwork, and that UbuntuStudio preinstalls various media related programs.

For a list of the programs installed by UbuntuStudio, check out the wiki
[https://wiki.ubuntu.com/UbuntuStudio/PackageList](https://wiki.ubuntu.com/UbuntuStudio/PackageList)

Since those are the only real differences, you can simply install a couple of packages in normal Ubuntu to turn it into UbuntuStudio.

Since I don't know about Cubase and Reason, I don't know what you will be missing.

But by using UbuntuStudio, you get most of the best media related programs installed. These should allow you to do anything that you need to do. If not, post here on the forum, and someone could probably suggest an alternate program.

### Post by Doggles on 2007-11-06
I'm just finding my way around ubuntu studio at the moment - it's pretty good so far. I'm using an old audigy soundcard. I think you might have to google around a bit to see if your hardware will be supported - I know that there are a lot of USB and firewire audio interfaces out there that might not have the full functionality that you have in a windows system. Also, I'm not sure what the situation is with things like VST plugins - so if there are any proprietary effects that you need to use, you'll need to try to find a linux alternative (I think...)

I used Cubasis for a while, and there are a lot of similarities between this and Ardour. One thing to watch out for that isn't immediately apparent - you might have to get "alsamixer" open from the command line in order to un-mute some of your inputs for recording and the like. It could drive you crazy for a while wondering why you can't get a level on your mic! Ardour relies on a background app called "Jack" - which manages all of your audio connections - there are some good help pages for studio, Jack, Ardour and the like so get googling! - see here: [http://www.ubustu.com/](http://www.ubustu.com/)

As with any type of linux, for the absolute novice there will be a certain amount of searching for how-to guides to properly get your system up and running - depends on your attitude, if you like "getting stuck in" and enjoy the challenge, then it can be fun - welcome to the club!

For the non-recording studio side of things, I haven't missed a thing about windows - and I'm not a windows hater by any means - no problem with XP and all that - I just got sick of paying for virus software updates. As for Gutsy Ubu Stu - web, email, office, pictures, video and sound all seem to work fine with only a minimum of tweaking. You'll also find this forum invaluable for help and advice as you learn about Ubuntu - some of the guys on here are fantastic - which is why I try to post whenever I have anything productive to say.

If you're unsure about the switch, dual-booting is a very good option - check it out thoroughly before you do it though - this way you can retain windows until you're sure you want to ditch it!

Best of luck - keep on rockin'
