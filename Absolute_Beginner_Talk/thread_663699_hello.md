### Post by insertAlias on 2008-01-10
The simple answer is that it is an alternative to windows.  Thats actually an over-simplification, but we should start slow.  Ubuntu isn't an application, it is an operating system.  It is (like windows) loaded up before any of your applications are, and it is what allows you to run the applications.  Now, as to whether or not you can/should replace windows, that depends on what you want to do.  Ubuntu can do most anything that windows can do (and some things that windows can't) but it can be very confusing, especially to someone with no experience with this type of thing.  Tell us what kind of computer you have, and what you plan on doing with it, and we can tell you if you should make the switch!

### Post by kleo skywalker on 2008-01-10
> **medo2 said:**
> so does that mean I can get rid of all my windows apps

If you opt for Ubuntu, you will get rid of Windows itself, not just Windows applications (unless you choose to dual boot and have both). 
Like dgray_from_dc suggests, the best option is to download the latest version of Ubuntu (7.10), burn it onto a CD (as .iso image) and boot from the CD. This will not affect your hard disk, and you can see what it's like. (Note that the LiveCD runs much slower than an install on the hard disk.) There are default applications to do everything from browse the internet to listen to music, and so on; and there are many more applications that you can easily install if you have an internet connection.
