### Post by catlett on 2006-07-22
Did you hit the eject button to remove the cd or did you just press the cd drives button to get the cd out?
I ask because if you don't hit eject from the right click menu, the data may not be written. data for floppies and cds are stored in temp and when you eject the data is written. It isn't always written at the time the command or app started the copying, so if you removed it without selecting eject the data may not be on it.
I use gnomebaker for burning isos and anything else that needs writing to a disk.```
sudo apt-get install gnomebaker
```
