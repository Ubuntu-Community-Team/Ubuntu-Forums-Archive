### Post by aysiu on 2006-08-24
Maybe this is the line that's malformed? ```
deb http://antesis.freecontrib.org/mirrors/ubuntu/plf/
``` Notice how the other ones have words after the web address?

Try substituting ```
deb http://packages.freecontrib.org/plf/ dapper free non-free
``` for that line.

To edit the file ```
sudo nano -B /etc/apt/sources.list
``` To save the file: Control-X, Y, Enter.
