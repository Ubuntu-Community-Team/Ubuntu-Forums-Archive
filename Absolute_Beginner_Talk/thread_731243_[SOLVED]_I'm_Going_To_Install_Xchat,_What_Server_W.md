### Post by herbster on 2008-03-21
Run Xchat, at the top menu hit XChat > Network List and under Networks find Freenode or Ubuntu network. Fill in your user name you'd like (the only important one is nick name), and you can hit Edit once you've highlighted the network and add a channel to autojoin. Or hit Connect then type

```
/join #ubuntu
```

To join the Ubuntu channel.

### Post by herbster on 2008-03-21
> **FreewareFan said:**
> Thanks much!  I've connected, and am listening/watching things go by..  How would I go about registering my nickname?  I've tried /help, but register is not listed....

```
/msg nickserv register nickname
```

You'll get instructions on password setting. Then to identify

```
/msg nickserv identify password
```

In that Xchat > Network List dialog you can add the password when you hit Edit on the server so you don't have to identify every connection.
