Ummm, I don't think either sony or toshiba support Ubuntu better... I haven't had a labtop from either so can't really say I'm that interested in recommending either company. I don't think it matters the manufacturer that much, just look for a few things in your computers specs. One, don't buy a computer with the Phoenix "optimized for vista" bios, I've heard bad things about it. Two, make sure it has an nvidia card if you want easy 3d stuff/gaming. ATI drivers still give people nightmares it seems >.>. And lastly, if your interested in getting a wireless card rather than using the built in, don't get linksys... seems broadcom cards cause the most trouble.
