### Post by jasay on 2005-10-19
First guess is the iso wasn't burned as an image.   I'm not familiar with the program you mentioned so I don't know if it's supposed to take care of that for you.  From a screenshot on their page it looks like their may be a checkbox you have to hit, but like I said, I've never used it.  Anyway put the cd in while in windows and take a look at it.  If it only has one file (the iso) then it wasn't burned correctly as an image.

If the cd has a number of files and directories then I might guess a faulty download.  I don't remember if windows has an md5sum checker but they're easy to get.  Run that with your iso and compare to the md5sum with the one listed for your live cd on the page you downloaded from.
