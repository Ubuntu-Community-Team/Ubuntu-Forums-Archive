### Post by aysiu on 2007-03-08
You don't install Ubuntu from *within* Windows.

You reboot your computer with the Ubuntu CD in the drive (and your BIOS set to boot from CD instead of the hard drive).

See [this thread](http://ubuntuforums.org/showthread.php?t=379032) for more details.
