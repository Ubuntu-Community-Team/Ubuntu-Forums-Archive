### Post by teaker1s on 2007-03-16
? streamtuner is in repositories, you may have to uncomment some in your sources file to get it though
```
sudo gedit  /etc/apt/sources.list
```

you will also need xmms and mp3 codecs

```
gksudo synaptic
```

settings>preferences>general
tick box that say show package properties in main window
hit reload tab
you can now search and install
