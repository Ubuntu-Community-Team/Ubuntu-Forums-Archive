### Post by MrHippocampus on 2007-07-23
The projector probably isnt sending out any EDID information, this is needed for the nvidia-driver to set it up automatically. Unfortuantely this means your going to have to edit /etc/X11/xorg.conf in order to get it working right. If you post it up on here somebody should be able to give you some pointers on what to add. Youll also need to find out the resolution of the projector and its Hsync and Vsync values.

### Post by MrHippocampus on 2007-07-23
ok, try using this screen section:

```

Section "Screen"
        Identifier      "Default Screen"
        Device          "nVidia Corporation G72M [GeForce Go 7400]"
        Monitor         "Generic Monitor"

       Option "Twinview" "1"
       Option "TwinViewOrientation" "Clone"
       Option "MetaModes" "1280x800,1024x768;"

        #if the above lines stil have the same problem, try uncommenting these two lines:
        #Option "SecondMonitorHorizSync"     "30-50"
        #Option "SecondMonitorVertRefresh"   "60"

        Defaultdepth    24
        SubSection "Display"
                Depth   1
                Modes           "1280x800"
        EndSubSection
        SubSection "Display"
                Depth   4
                Modes           "1280x800"
        EndSubSection
        SubSection "Display"
                Depth   8
                Modes           "1280x800"
        EndSubSection
        SubSection "Display"
                Depth   15
                Modes           "1280x800"
        EndSubSection
        SubSection "Display"
                Depth   16
                Modes           "1280x800"
        EndSubSection
        SubSection "Display"
                Depth   24
                Modes           "1280x800"
        EndSubSection
EndSection

```

Its a bit of a shot in the dark as I cant replicate your setup at the moment, but try it and see if it gets you anywhere :)

edit: ive assumed that the projector projects at 1024x768 if it needs to be different then some more changes might need to be made
