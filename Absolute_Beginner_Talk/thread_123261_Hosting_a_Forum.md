### Post by az on 2006-01-29
Phpbb runs using php, apache and mysql.  You would have to install and configure those.

How much cpu power it takes depends on the number of people using it.  If a few people are on at a time (less than a dozen, say) you probably won't notice much os a slowdown.  You will hear your cpu fan rev every time someone loads a new page...
