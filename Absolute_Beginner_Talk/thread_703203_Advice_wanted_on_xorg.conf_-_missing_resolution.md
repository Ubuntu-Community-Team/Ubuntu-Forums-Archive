### Post by pedro_orange on 2008-02-21
You can reconfigure xserver with:

```
sudo dpkg-reconfigure xserver-xorg
```

From the command line. This will send you through a series of steps asking you what graphics you have; what resolutions you want available etc. 
This should build your xorg.conf file for you.

[code]
dpkg-reconfigure xserver-xorg
