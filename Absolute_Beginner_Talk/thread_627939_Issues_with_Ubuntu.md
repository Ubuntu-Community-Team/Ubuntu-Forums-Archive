### Post by jfinkels on 2007-11-30
> **IOA said:**
> Well, I do so happen to have another 160gb Hard drive. So is it basically the same as doing it on live CD? Just burn it to that Hard Drive then install? If so, then I can do that for sure.

If you want to install Ubuntu to your hard disk, boot into the LiveCD environment, then double-click on the icon that says "Install". Make sure you don't overwrite any of your personal data. Once it's installed, you get the full benefits of an operating system which boots from a hard disk drive, i.e. data persistence, speed, memory.
