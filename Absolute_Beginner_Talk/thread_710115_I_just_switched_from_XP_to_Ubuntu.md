### Post by jken146 on 2008-02-28
System -> Administration -> Synaptic

Compiz-fusion is a compositing window manager (flash 3-d effects like a rotating cube and painting fire on the screen).  Some minimal options are available in System -> Preferences -> Appearance but to get the whole lot, install the package **compizconfig-settings-manager**.
