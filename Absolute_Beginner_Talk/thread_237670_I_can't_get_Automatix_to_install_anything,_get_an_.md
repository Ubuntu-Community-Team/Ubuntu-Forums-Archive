### Post by mrazster on 2006-08-16
As of the automatix installer, it is safe to remove/delet it.

About the updatemanager I honostly don't know how to turn it of while running in bakground. But I can tell you how to make it not running in the bakround in the future(after next reboot).

Fire up *_synaptic_* klick *_settings_* then *_repositories_* then *_internet updates_*.
Untick all 3 boxes and then *_close_* and lastly close synaptic.

Now it next time you reboot the system it won't run in the bakground...this way you have to run the update manager manually...wich I prefer personally.
