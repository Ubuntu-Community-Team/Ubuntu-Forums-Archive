### Post by Origin415 on 2008-02-19
Make sure that in System -> Preferences -> Removable Drives and Media, that the options for automatically mounting drives are checked.

If that still doesnt work, post the output of

```
ls /dev
``` with and without the hard drive plugged in.

### Post by FuturePilot on 2008-02-19
You could also redirect the command into a text file
```
ls /dev > ls-dev.txt
```
You will find ls-dev.txt in your home folder.
