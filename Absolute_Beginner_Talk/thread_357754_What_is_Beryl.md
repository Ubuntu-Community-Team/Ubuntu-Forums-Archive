I don't bother with graphics add-ons, so I am far from an expert, but a very limited explaination would be that it is a graphics eye candy package similar/related to xgl. If that doesnt ring a bell, think of Vista's "Aero" graphics which are attempting to acheive more or less the same thing.

Imagine transparent windows, things flying about, wobbling, 3D effects and other generally useless, but eye pleasing events.
