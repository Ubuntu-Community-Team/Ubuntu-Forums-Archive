### Post by ChrisTheGeek on 2005-12-07
Hi,

I'm not quite sure why you would want to do this.  You should only need to access the .evolution folder when backing up your email.  If you frequently need to see the hidden folders you can toggle them on and off by pressing Ctrl+H.

In anycase, you won't be able to change where the data is stored just by renaming, you will need to tell Evolution.  I'm not sure if it supports this but you can find out by going through Evolution's Options screen and looking for some kind of 'path to settings folder' option.
