In proper alphabetization. b comes waaaaaaaay before t. It's that simple. Change the filename by removing 'the' if you don't want it alphabetized by the t. Alphabetization is, by definition, done using the first letter. It doesn't sound like Banshee is 'tagging' your files at all. Sounds like it's doing what it's supposed to. I don't use Banshee, so I'm not sure about this next part, but there may be a way to change the method of organization based on something other than simple alphabetization. You'll have to explore banshee a bit to find out, I suppose.

> **matious said:**
>  Is there any way i can configure the tagging so that it reads artists with the infront of them by there actuall band name?

i know im being a bit nit picky, but when it comes to music i seem to have obsessive compulsive disorder :(

anyone of you highly intelligent linux users now how to fix this?;)

Yes, change the filename to "there actuall band name" (your spelling). If 'the' is not a part of "there actuall band name" then why did you name it that? Alternately, you could name them like this: 'Beatles, The' or 'Rolling Stones, The'.

I.E. 'The Rolling Stones' & 'The Beatles' use 'The' as a part of "there actuall band name". In other words, you cannot buy a Beatles or Stones album whose title *doesn't* have 'The' as the first word. So, "there actuall band name" would seem to have nothing to do with your personal preferences. Change your files to do/read as you like.

And yes, this is nit-picky. OCD is a disorder that needs to be treated. If you have it, get treatment and stop the Ipod abuse. ;)

Ummm, I'm a banshee user and I never run into your problem. It's always used the tags the file had, admittedly I did have to edit a few to get it to work right but most were fine.

I'd just recommend using an ID3 tag editor and manually overwriting the artist and whatever other tag you want in it. It reads fine for my files (like Linkin Park, Evanescence no "the" in front).

[Here's a link to install the two editors]("http://ubuntuguide.org/wiki/Ubuntu:Edgy#How_to_install_Music_Organizer_.28Cowbell.29") they are cowbell and easy tag. They haven't been added to the Feisty guide for some reason, commands are the same and they are in the repos. I prefer easy tag... try em both. Then it should be fine.
