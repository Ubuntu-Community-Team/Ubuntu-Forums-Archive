### Post by Bachstelze on 2007-03-21
ubuntu-desktop is a metapackage, meaning that it does nothing by itself. Instead, it depends on all the packes the Ubuntu devs want to include in the "vanilla" Ubuntu desktop, thus providing a simple way to install the complete set. So, if you don't need all the apps, removing ubuntu-desktop is not harmful - but it is recommended to reinstal it before upgrading.
