### Post by Shuja on 2006-11-27
No, LDAP is for user authentication not email.  Easy way is to use IMAP for email and leave the email on the server though that won't help calender syncs.  The more difficult one is to set up a zimbra server and just use that though it's a bit more complicated.  The other is to use a common email client and points to a common directory.
