### Post by pathall on 2006-03-17
Here's what's worked best for me (on 1280x1024 LCD display):

First, Install MS TTF Fonts.  This can easily be done using Automatix

Next, navigate to: System --> Preferences --> Font

     Change 'Desktop Font' to Sans 11

Click 'Details...'

     Change 'Resloution' to 72 dots per inch

Also, I've replaced all icons on my desktop with 32x32 pixel icons available here:
     [http://www.musicsucks.com/linux/icons-32x32/icons-32x32.tar.gz]("http://musicsucks.com/linux/icons-32x32/icons-32x32.tar.gz")

Hope this helps,
