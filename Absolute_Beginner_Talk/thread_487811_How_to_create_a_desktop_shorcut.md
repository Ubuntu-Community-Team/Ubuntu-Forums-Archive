### Post by jfinkels on 2007-06-29
Or right click on the desktop and create a new Launcher. The command to open the file browser is, aptly:```
nautilus
``` and the command to open the file browser at an arbitrary location in the filesystem is, aptly:```
nautilus /path/to/location
```
