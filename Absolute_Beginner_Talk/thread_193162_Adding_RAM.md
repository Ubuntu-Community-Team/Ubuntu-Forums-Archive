### Post by MJN on 2006-06-09
Given it's a branded machine just pop over to [http://www.crucial.com]("http://www.crucial.com") and use their Memory Advisor tool - it'll tell you everything you need to know. Note that you'll also need the machine 'Type' (e.g. 6263, 6265 etc) which you'll find somewhere on a case label.

If you buy it from there too (they're quite competitive in the UK at least) then they'll also guarantee compatibility based on this info. For a machine of this age however you might do better on eBay if cost is the prime concern.

### Post by Warren Watts on 2007-09-24
The [Dohickey Project]("http://www.dohickey-project.com/about.shtml") is an excellent tool for determining system hardware as well, especially useful when the computer is cobbled together from various parts.

> The Dohickey client is a python based GUI software tool which allows the user to list the hardware within his or her machine. Each piece of hardware is given its own id, and this id is then looked up online to see if there is any existing information about this hardware in the database. The GUI then allows the user to report to the server how well the hardware works, edit the information attached to that hardware and send in comments and instructions for the installation of the hardware, which will then be added to the online database. From that point on, anyone who browses the database online will be able to see that information. So, you get to find out about your hardware and help other people, all in one stroke.
