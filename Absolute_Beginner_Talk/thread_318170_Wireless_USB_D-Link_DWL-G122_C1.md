### Post by Dusty545 on 2006-12-13
This one's been covered before.  Take a look here:

[https://help.ubuntu.com/community/WifiDocs/Driver/RalinkRT73](https://help.ubuntu.com/community/WifiDocs/Driver/RalinkRT73)

Just a quick tip for the future...  If you're going to get a wireless device, try as much as possible to get one with an Atheros chipset.  They work brilliantly under Linux.  I found that one out the hard way when I bought a Netgear card before.  I use a cheap Belkin one now with an Atheros chipset and it works perfectly with every distro I've tested it with.

Hope this helps!
