### Post by Metacarpal on 2006-09-11
Don't know if wifi-radar tells you when you got the WEP key wrong, but network-manager doesn't.  Well, not directly... if you have the wrong WEP key, then it takes a looooooooooooong time to get past the "acquiring IP address" progress bar, and the internet doesn't work.  So that's *kinda* like saying it was wrong. :roll: 

But both will save your setup.  TIP: with network-manager, first enter your settings.  Then, at the top where there is a drop-down for Location, choose to create a new location.  If you do it the other way around, the settings don't save.
