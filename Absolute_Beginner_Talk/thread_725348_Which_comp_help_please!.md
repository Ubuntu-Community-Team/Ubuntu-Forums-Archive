### Post by alzie on 2008-03-15
I reread your post.  Ubuntu is an operating system,  it doesn't run on Windows but rather instead of Windows.  You can [[COLOR="DarkOrange"]download a live cd [/COLOR]]("http://www.ubuntu.com/getubuntu/download") (the desktop edition) to try it out without installing.  This way you can see how each computer works with Ubuntu.

The other thought I have is that removing Windows from a brand new machine may have warranty issues.  A dual boot might be in order, having both Windows And Ubuntu on the same machine with the option to chose when you boot up.

I hope this helps :)
