In nautilus, go to Edit -> Background and Emblems

### Post by Brucevdk on 2007-06-25
Like kevinlyfellow said in Nautilus: Edit -> Backgrounds and Emblems -> Add a new emblem

There's no functionality in the user interface to remove these emblems, but any emblems you add through the interface will be stored in *~/.icons/hicolor/48x48/emblems* (the image used and an .icon file describing the image).
