### Post by InuyashaDuelist on 2007-03-31
Welcome to the community, Fowlwing. You're right, we don't mind if people ask silly questions, simply because there's no such thing as a silly question. :) Everyone was a beginner once, and everyone gets past that annoying little 'oh man...' stage. We're here to help you.

> What is Wine?

Wine is a Windows program emulator. For example, by using Wine, you'd be able to play games like Soldat or World of Warcraft. It's not the most working utility, but for some things, it gets its job done.

> Can i run things like Itunes and or Windows Media Player on Ubuntu?

A good replacement for iTunes would be Amarok if you're using Kubuntu, and Exaile if you're using Ubuntu. They have a small learning curve (maybe 5 minutes) to get acquainted with it, but they're alright. As for Windows Media Player, the same goes. If you're wanting to play videos, I recommend VLC Media Player as it plays almost all forms of Video.

> How do i download things like Flash Player?

Flash Player can be installed using Automatix2, which can also give you other useful programs to help you make the switch.

> How do i install programs from cd's such as games, or other applications?

That really depends on the application, if it's Linux compatible, and if not, how well Wine works with it.

If you have any more questions, feel free to ask. :)
