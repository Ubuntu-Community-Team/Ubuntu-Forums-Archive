### Post by d1337 on 2005-09-15
You should be able to boot from the CD (install version) without issue.  One of the great things about Ubuntu, as well as some other Linux distros, is how simple making a dual boot box is.  If you indeed have Windows on the primary hd, Ubuntu should locate Windows and install the Grub boot manager to the master boot record which is on the primary hd (windows drive) without giving you issue.  This, in my opinion, is the easiest/best way to dual boot a box...existing windows install and a secondary harddrive to give completely to Linux.  Just make sure that you partition the right hd, and not the one with ms on it.
