### Post by kebes on 2006-08-07
Yes kino is a video editor for linux. It is free and open source:
[http://www.kinodv.org/](http://www.kinodv.org/)

You can install it like any other piece of software. Either use "Synaptic" or use the command "aptitude" if you are comfortable with the terminal. You can also download the latest code from the link above.

For simple video editing in Linux, you can also use avidemux:
[http://avidemux.sourceforge.net/](http://avidemux.sourceforge.net/)

Cinelerra is another video editor for Linux:
[http://cvs.cinelerra.org/](http://cvs.cinelerra.org/)


You can try them all and stick with the one that matches your needs. There are also lots of command-line tools for encoding video (mplayer, mencoder, etc.).

### Post by evilghost on 2006-08-07
Kino makes for an excellent video editor and is comparable to Windows Movie Maker with more features and more control over titling, transitions, etc.  I use Kino to capture movies from my Sony MiniDV camera (over IEEE1394) and encode them to DVD.

It's free.
