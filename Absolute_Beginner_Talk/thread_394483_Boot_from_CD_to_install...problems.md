### Post by dstew on 2007-03-27
I think you should at least enter the BIOS setup to look at your disk boot order,  and make sure the CD boots before the hard drive. If you have Windows installed, and you put in the XP CD, it may look like it is booting, but maybe it is really starting with the hard drive, and handing off to the CD. I think it's worth a look.

If the Ubuntu CD autoruns, it should also be bootable.
