### Post by zxee on 2006-09-08
Basically all that happens when you install windows is that it overwrites the mbr. So you need to put the ubuntu cd in your drive and setup or reinstall grub. Take a look at the grub howto (page bottom) here: [https://help.ubuntu.com/community/SystemAdministration](https://help.ubuntu.com/community/SystemAdministration)
