### Post by ubername on 2007-05-25
Hi

I guess you are using nautilus as your file manager. If so you can change a number of views and settings by clicking on edit - preferences. To see the address bar click on the notepad thingy icon on the top left of your screen / window (under the 'back' button on the toolbar, if you have a normal config for windows displays).
