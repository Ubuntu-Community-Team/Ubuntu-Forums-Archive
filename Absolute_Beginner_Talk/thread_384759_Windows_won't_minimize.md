### Post by adam.tropics on 2007-03-15
I doubt this is the answer you're hoping for! On a few occasions I have had similar issues, like a new window opening in firefox of a set size. Then closing firefox. Then when I start firefo again the default window has changed to that size.....well that sort of makes sense! The trick is to resize the unmaximized window (manually) to the desired size, then close the program,  I think the default window size (which I imagine would include unmaximized) is a remembered value from its previous instance. So when you restart the application, all should be well with the world once again!
