### Post by Sef on 2007-02-28
> 1. Is there an universal Ubuntu where i can use all these three interface? Is all the interface available in the Synaptic?

Ubuntu, Kubuntu, and Xubuntu are the front ends; the back ends are the same.   Yes, you can install them all if you so desire from synaptic.

> 2. Is there any other difference in these three other than the interfaces?


The backends are the same a mentioned.   Sometimes, they have different apps that do the same thing, but you can install any app on any of them.

> 3. In terms of resources, which type will use the least resources when after a fresh install of OSS? Cause i only have a PIII 550MHz, 512MB SDRam to use.

Xubuntu might work faster cause it requires lower requirements, but they should all work ok because you have 512 ram.   I would recommend Ubuntu, since most people on here use it.

### Post by firefly2442 on 2007-02-28
Gnome and KDE are the two most popular desktop environments for Linux.

[http://en.wikipedia.org/wiki/GNOME](http://en.wikipedia.org/wiki/GNOME)

[http://en.wikipedia.org/wiki/KDE](http://en.wikipedia.org/wiki/KDE)

XFCE is a "lightweight" desktop environment for older computers.  I know for a fact you can install Gnome and then install the KDE desktop environment and switch between them.  I'm not sure about XFCE but I bet you probably could.  Gnome in general is more minimalistic and simple.  It reminds me a little of the OS X interface.  KDE provides more details.  I would suggest trying both if you can.  I've never used XFCE but it is very minimalistic and good for old computers that might not be powerful enough to run all the visual effects of Gnome or KDE.  Based on your system, you have enought RAM to probably run Gnome fairly well.  If that doesn't work, I would suggest trying XFCE.  Hope that helps. :)

Xubuntu is exactly the same as Ubuntu and Kubuntu... It just installs XFCE as the default GUI. There is nothing stopping you from installing GNOME or KDE if you want. But since Xubuntu is designed for older computers, it (by default) takes up less physical space and uses fewer resources. But you could take Ubuntu, remove GNOME and install XFCE if you wanted. The three distros are exactly the same... It's only the default applications that vary.

### Post by firefly2442 on 2007-02-28
Yes, I think you are misunderstanding the idea of the desktop interface.  Ubuntu Linux runs the Linux kernel which is the core and "heart" of the operating system.  These desktop interfaces are just frontends for interacting with Linux.  All your files will be stored in the same way, it's just the way you go about using Linux and interacting with it in the "graphical" sense that is different.
