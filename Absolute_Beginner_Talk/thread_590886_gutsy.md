### Post by mikeyphi on 2007-10-25
I don't know of any 'common' problems - it very much depends on your system hardware - for example some people with ATI graphics cards have problems! I have an average Desktop system and have not had any issues with Gutsy.

From comments in the Forums - most people have found Gutsy an improvement over Feisty.

If you are concerned about making the change: 
1.  Wait a while so that any issues get fixed.
2.  Save personal data
3.  Run the LiveCD to check for any immediate issues
