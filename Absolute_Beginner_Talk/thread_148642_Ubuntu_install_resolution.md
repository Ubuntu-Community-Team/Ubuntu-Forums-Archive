### Post by nanotube on 2006-03-22
not sure at which point this is happening for you, but here is something to try. there are keyboard shortcuts that are available in xwindows, to change screen resolution. it is <Ctrl><Alt><+> and <Ctrl><Alt><->. so when your resolution goes bad, jsut try hitting one of those a couple of times, and see if resolution changes to something decent. :)
