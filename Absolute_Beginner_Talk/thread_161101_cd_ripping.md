### Post by Ian Maxwell on 2006-04-16
Of course, if you have no specific reason not to, you might want to consider using the Ogg format. Not only is it better supported in Linux in general (as you've just found), but it implements VBR much better than MP3, so you'll usually end up with better sound quality for the file size.
