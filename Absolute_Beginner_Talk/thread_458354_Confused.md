### Post by meng on 2007-05-29
There's a firewall on there already (iptables). If you need an easy way to CONFIGURE the firewall settings, then you will want to install firestarter or guarddog or some such.
Basically, you don't need antivirus unless you network with Windows computers, and even then it's only to protect the other computers.

### Post by Ateo on 2007-05-29
The real truth is based on personal experience and I can only share with you that in the past 10 years of linux use, I have never EVER installed a virus scanner NOR firewall on any linux installation which I use for my primary desktop.

I have my mail server using ClamAV for virus scanning as the message is delivered to the user's inbox. The only reason to install a firewall on a desktop PC is when it's connected directly to the internet, with a public IP. If you have a router/gateway, then you should be safe.
