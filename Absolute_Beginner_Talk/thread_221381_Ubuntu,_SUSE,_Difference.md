### Post by GuitarHero on 2006-07-23
Did you install XGL with Ubuntu? If not thats what was causing slowness on SUSE most likely.  SUSE is more bloated than ubuntu to begin with, then piling XGL on top of it makes it a resource hog.  Ubuntu with no xgl is very simple and quick.  Even quicker than that would be Xubuntu, which is a lighter version of Ubuntu per say(basically is uses a different desktop environment).  Kubuntu uses KDE like SUSE and so needs a little more hardware power than Ubuntu which uses GNOME as its desktop environment.  Even so, Kubuntu will run fine on your system.  The sluggishness may have been a video card problem, does your laptop have a video card? Or is it integrated video?
