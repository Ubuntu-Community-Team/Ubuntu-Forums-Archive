My advice: be prepared for a couple of headaches getting it set up.
Most of the time, installation is painless, but every so often, you'll run into trouble. Come back here for help, and be patient. Once you have a working system, you'll love it.
Hopefully you'll have no issues and be amazed with just how easy the install goes.
Oh, and don't mess around with your hard drives before/after the install. That will cause headaches. If you're not planning to touch the hard drives on which Windows resides, you should be fine. I've never backed up before an install anyway (and some of these installs involved resizing NTFS partitions with large amounts of irreplaceable...music). I've never lost a single file.
