### Post by aysiu on 2006-06-18
You may still be better off with the Alternate Install CD.

Installing from the Desktop CD is a bit resource-intensive. It's trying to simultaneously run an entire operating system session *and* a graphical installer on top of that.

The Alternate Install CD doesn't have a live desktop session, and it uses a text-based graphical installer.
