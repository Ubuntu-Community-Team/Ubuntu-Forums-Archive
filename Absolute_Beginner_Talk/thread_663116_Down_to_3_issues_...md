### Post by KhaaL on 2008-01-09
Hiya v.man

1.the problem isn't only restricted to 64-bit ubuntu. It seems that the kernel require certain patches and hassle in order to have a perfectly working sleep/suspend/resume. The proccess is lengthy and a bit tiresome, but if you search for suspend hibernate in the forum, you'll find a couple of threads that'll give you more info.

2. Whats wrong with the default? (mine is serif / sans-serif). Try diffrent sub-pixel hinting settings and see if they help :)

3. Wine can run CoD4 if you patch it, see further details [here]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=10429").
