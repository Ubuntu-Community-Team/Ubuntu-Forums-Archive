### Post by slimdog360 on 2006-10-30
its located at ```
/boot/grub/menu.lst
```
you can open it with ```
sudo gedit /boot/grub/menu.lst
```

but before you start mucking around with it try this ```
sudo cp /boot/grub/menu.lst /boot/grub/menu.lst_backup
```
which makes a backup

and if you stuff anything up you can get back to the original by ```
sudo cp /boot/grub/menu.lst_backup /boot/grub/menu.lst
```
