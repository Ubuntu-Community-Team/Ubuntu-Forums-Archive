### Post by joshrobinson on 2006-04-14
i just found this in the unofficial ubuntu 5.04 starter guide from some googling

Q: How to mount/unmount Image (ISO) files without burning?

   1. Read General Notes
   2. To mount Image (ISO) file

      sudo mkdir /media/iso
      sudo modprobe loop
      sudo mount file.iso /media/iso/ -t iso9660 -o loop

   3. To unmount Image (ISO) file

      sudo umount /media/iso/
