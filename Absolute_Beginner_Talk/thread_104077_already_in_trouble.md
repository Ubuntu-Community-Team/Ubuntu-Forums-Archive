### Post by linbetwin on 2005-12-15
Open System -> Administration -> Synaptic Package Manager

It will prompt you for your user password, the one with which you log in into Ubuntu. Type it and press ENTER.

The click Search and type abiword. ENTER. You will be presented with a list. Look in it for abiword, click it and click install.

Then click Apply.
