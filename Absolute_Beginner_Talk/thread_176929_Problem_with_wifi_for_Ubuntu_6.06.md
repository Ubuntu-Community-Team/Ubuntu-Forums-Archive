Hello,

I've been very succesful lately using the application "wifi-radar", which you can find with the command:

$sudo apt-get install wifi-radar

I'm using this application with a linksys wifi card (which I had to use the nswrapper to configure) and a belkin wireless router.  Wifi-radar allows you to configure for WEP and WPA encryptions, although I've only tested with WEP.
