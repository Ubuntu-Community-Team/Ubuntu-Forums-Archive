### Post by rajeev1204 on 2007-07-19
Hi

just download this ..

Its a script to download and install a program called nspluginwrapper which lets u use the 32 bit flash plugin on a 64 bit browser .


Its what the whole 64 bit community uses .

[http://home.comcast.net/~ubuntume/nsplugin-wrapper-install-0-1.1.tar.gz](http://home.comcast.net/~ubuntume/nsplugin-wrapper-install-0-1.1.tar.gz)  .Thanks a lot to forum member Kilz who made this script,

After u untar it , just double click on the script and select run in terminal. It will automatically download the nspluginwrapper and flash player and install it for u.


If u have any questions please ask



regards

rajeev
