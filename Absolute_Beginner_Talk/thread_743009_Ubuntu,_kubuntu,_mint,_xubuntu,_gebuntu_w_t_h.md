### Post by c-ron on 2008-04-02
Kubuntu is Ubuntu with KDE. Ubuntu uses GNOME. The two are basically interchangeable. Same goes Xubuntu, which uses XFCE. Gobuntu -only- includes open-source non-restricted software. 
Mint is Debian-based, like Ubuntu, but isn't an official Ubuntu distro.

### Post by eriqjaffe on 2008-04-02
To answer your question, yes, they are compatible.  Kubuntu can run GNOME apps and Ubuntu can run KDE apps.  There's a bit of additional system overhead due to the necessary libs that need to load, but if you have a new(ish) system, the impact should be fairly minimal.
