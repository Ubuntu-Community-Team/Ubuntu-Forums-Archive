### Post by GeirG on 2005-08-25
If you are buying it from a local store that has a demo model on display, you could always try to boot it up with a live CD.

I have installed Ubuntu on two laptops, one Asus M5 (or something like that), and on a Dell Latitude D810.

The Asus was no problem at all (except there are no drivers for the MMC reader).
On the D810 I couldn't get Ubuntu to recognize the SATA HD, so I went for the test version of Breezy (due out in october). That worked but is currently under pretty heavy development, so I'm using Fedora Core 4 mostly right now.
As far as I have heard Breezy should have pretty good support for laptops when its ready.

BTW, the Hoary Live CD worked on the D810, even though I couldn't install it. This is of course due to the fact that the Live CD doesn't have to recognice the HD.

Good luck.
