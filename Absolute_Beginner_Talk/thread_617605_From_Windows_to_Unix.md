### Post by mivo on 2007-11-19
BSD is great, and while you can use it for desktop home use, its strength is servers. Drivers usually arrive later for some consumer hardware, and typical desktop applications (Flash. Skype) are not well or not at all supported. Linux is better supported in this regard, so it makes the better Unix-like operating system for you. :) (There are also better resources available to beginners.) BSD is awesome, but probably not the best choice here.

Ubuntu is a great Linux flavour to start with. :)
