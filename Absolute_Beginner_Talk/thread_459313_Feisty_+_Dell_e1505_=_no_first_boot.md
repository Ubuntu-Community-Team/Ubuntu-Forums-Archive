### Post by Ek0nomik on 2007-05-30
Note:  This command will change any previous changes to your xorg file.  I suggest backing it up by issuing this command:

```
cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
```

(If it says permission denied, put *sudo* in front of the command.

Now if you want to get your highest resolutions recognized, issue this command in the terminal:

```
sudo dpkg-reconfigure -phigh xserver-xorg
```

Use the space bar to select items / resolutions.

If that *doesn't* do the trick (but I think it will), paste me your xorg.conf file.

```
cat /etc/X11/xorg.conf
```

copy what the terminal outputs (everything, not just pieces), into a code box on this forum.  :)
