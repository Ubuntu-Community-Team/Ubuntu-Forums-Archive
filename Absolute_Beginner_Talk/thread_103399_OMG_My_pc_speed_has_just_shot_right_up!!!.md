### Post by dgrafix on 2005-12-14
I have to share this with people who havent done it yet:
Im running Ubuntu on an old P3 400Mhz pc. It ran about as slow as XP would if put on the same box. I have been using it like it for months now.
But,
Thanks to some nice people on the forums i did the following...

1>Installed the correct Nvidia drivers from synaptic (not the ones ubuntu selects by default),
2>tuned up firefox using the about:config thing ([http://www.ubuntuforums.org/showthread.php?t=102614](http://www.ubuntuforums.org/showthread.php?t=102614))
3>actually activated my swapfile (for some reason it had vanished from fstab!)

Now it runs as fast for email, office & internet as XP on my 800Mhz Duron :)


is there any more tuning tricks people know of?.....:D
