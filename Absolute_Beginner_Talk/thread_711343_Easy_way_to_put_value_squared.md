### Post by Hospadar on 2008-02-29
Depends on what application you are using, if it's openoffice, you could try doing Insert->Object->Formula or changing the font to "superscript"  if it's on the web somewhere, you might try using html (something like a <div> tag) to set the font to superscript.

I don't think there is any system-wide way to put the font in superscript, but that might not be the case.
