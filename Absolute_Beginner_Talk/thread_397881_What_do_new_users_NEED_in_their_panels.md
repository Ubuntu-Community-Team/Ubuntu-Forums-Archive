### Post by teaker1s on 2007-03-31
my advice is get your base system sorted first
things like installing "alacarte" menu editor help to display more on application lists
think about 
```
gksudo synaptic
```
Settings>Preferences>General
tick show package properties in main window (makes finding package info and location much easier) also uncomment your multiverse etc repositories for more programs.

**Lastly**

install packages you will like and enjoy, remember they are free so you can try several -until you find one you like.:popcorn:

### Post by jem7v on 2007-03-31
Things you Need? Not much. A main menu, I suppose.

For convenience sake I've put a bit of stuff onto my panels. I still have my apps/places/system menu, but I also have program launchers for: Swiftfox, Thunderbird, Abiword, Amarok, Gaim, Calculator, and Sticky Notes, because I find that I use those programs more frequently than anything else and don't feel like scrolling through the menu to run them each time when I could just click once on a panel launcher.

I also have the weather report on my panel because I like being able to see what the temperature is outside.  That's sort of important here up north, when a sunny day during the winter won't be visually distinguishable between -2 and -40.
