### Post by Chuck_W on 2006-04-02
In the hope that my experience will help someone else I have a few thoughts to offer. There have been a number of posts about having Numlocks on (or off) on startup. For those of us still trying to learn the command line the process of changing is a pain. In a future release it would be nice to have a simple way of changing this. Maybe under system preferences or the keyboard section an on/off radio button could be added.
   Setting up my printer was a lot more difficult than it should have been. My Kyocera FS-1010 connects to my home network via a Hawking Technologies print server and has a static IP address. I had to try several configurations, (CUPS, LPD etc), before I got it to work. It seems that changes really didn’t take effect unless I removed the non-working printer and started over. Adding an “Apply” button might help to force the change.
   One thing that made it difficult is that most folks have a printer connected directly to a computer and want to access it from another machine so the information mostly focuses on that. Maybe I’m blazing a new trail! LOL
   Anyway I’m glad this forum is available. It’s good to know there are nice people out there.
