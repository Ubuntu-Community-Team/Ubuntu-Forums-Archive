### Post by MegaJim on 2007-11-18
first backup your xorg.conf

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.atibak
```

open it for editing

```
sudo gedit /etc/X11/xorg.conf
```

change the driver line from "fglrx" to "vesa"

you may need to remove the option lines as well (?) ... save it and restart X, if you aren't able to login again or its completeley messed up or something you can just restore the backup you made earlier.
