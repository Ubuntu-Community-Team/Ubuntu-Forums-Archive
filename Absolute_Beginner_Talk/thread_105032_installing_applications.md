### Post by joshuapurcell on 2005-12-17
There are usually scripts included that you can run by the command **sudo ./nameOfScript.sh**, but it completely depends on the program you are trying to install. The only thing that works every time with every install package is to read the INSTALL or README file and follow the directions that are included in the package.
