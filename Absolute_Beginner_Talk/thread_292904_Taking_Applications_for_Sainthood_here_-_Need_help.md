### Post by gn2 on 2006-11-04
Regarding the non-booting from any Windows CD, have you tried formatting the entire hard drive and deleting all partitons with a partitioning utility eg. gparted live CD first? [http://gparted.sourceforge.net/livecd.php](http://gparted.sourceforge.net/livecd.php)

Is there an option to install a second hard drive: [http://ubuntuforums.org/showthread.php?t=275728](http://ubuntuforums.org/showthread.php?t=275728)

Or if it's a laptop and you want to have Linux on an external USB hard drive, [www.PCLinuxOS.com](www.PCLinuxOS.com) has an automated installer for doing so.
Just remove the internal drive first, then run PCLinuxOS Live CD, then install to USB. Put USB before internal drive in BIOS boot order, and Windows will boot if USB not connected.
