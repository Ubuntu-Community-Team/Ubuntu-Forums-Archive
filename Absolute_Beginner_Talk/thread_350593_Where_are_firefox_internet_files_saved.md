### Post by chavo on 2007-02-07
The cache files for Firefox are not stored as ordinary files on the hard disk. This is supposedly gone for security reasons. There are a couple of ways to view the files though. You can try about:cache in Firefox itself. Try looking here [http://www.securityfocus.com/infocus/1832](http://www.securityfocus.com/infocus/1832) for an in depth look at the way firefox cache works.
