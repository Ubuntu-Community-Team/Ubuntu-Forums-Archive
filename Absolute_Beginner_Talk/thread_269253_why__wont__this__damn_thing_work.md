### Post by jimrz on 2006-10-01
> **kegstand said:**
> ive  tried  booting  it  in safe  mode  but  it  still freezes. i get  to the  2nd  question  and  it  stops.  is  there  another  option  to  boot it  in  text  mode?  and if there  is  can i   get the  normal  version  back up  once  ive  installed it?

download the "alternate" install iso and burn "as image". the installer on this is text based so not as pretty as the graphical installer, but it has more options + less problems on equipment that is older or has limited ram + is quite easy to follow. ALSO it is only the installer that is text only. it will, unless you choose server only install, install complete with whichever GUI comes with the version (Gnome, KDE or Xfce) you chose to download. 


PS - would also recommend upgrading to _>_512 Mb ram, when you are able. until then, Xubuntu would likely give you faster performance with your current ram.
