### Post by greybirds on 2006-08-04
Every application I used under Windows had a free and just as capable replacement under Linux (with one exception), so most likely the suggestions the others made will make you very, very happy.

In my case, I absolutely had to use Adobe's Flash authoring system. I hate it, I don't want to use it, and I wish there were something available (and better) under Linux, but I've got to use it and haven't found an alternative. Flash also doesn't run reliably under Wine. If you have an absolutely-must-run Windows app a solution of last resort is to buy yourself a copy of vmware. It's not the best solution, but it works. It allows you to install a copy of Windows (or many other operating systems) in a file on your linux partition and run the Windows desktop in a window just like a linux app. No partition, no rebooting. You can install your app under *that* Windows version and essentially have it running on your Linux desktop. 

The downsides are (1) vmware isn't free (2) it's a tiny bit slower than normal and (3) you have a big-ole Windows installation sitting in a file somewhere. But it gets the job done. An upside is that you can cut off networking for Windows so you aren't opening a giant hole for every piece of malware out there.

There may be free/open source alternatives to vmware that I'm not aware of. Like everyone's said, in almost all cases you're going to find an equivalent (and usually better) application for Linux, but if not, there's Wine, and if not Wine, there are options beyond that.
