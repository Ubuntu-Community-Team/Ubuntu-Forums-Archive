I don't know if this is the best way to do it, but it should work. Go into a folder you want to back up and select all the messages (i.e. click on the top one, scroll to the bottom one, press shift and left click on it). Then right click on the highlighted messages and choose "save as".

When you save them it's probably best to choose .mbox format (Berkley mailbox format) as this is highly portable. 

This will back up all emails in that folder. You'll probably want to compress the file using some compression utility.

### Post by Fidelio on 2007-04-21
Your emails are by default in a hidden directory in your home folder. Open your home folder, ctrl-h to show hidden files and look for evolution. Just copy them from here.
