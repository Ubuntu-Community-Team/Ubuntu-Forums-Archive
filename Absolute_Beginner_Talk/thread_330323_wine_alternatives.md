### Post by MasterOfDisaster on 2007-01-03
Another option on a different tangent would be virtualization.  Using VMware or Xen and a legal copy of Windows, you can run those programs natively.  If you have a processor that supports hardware virtualization, it runs pretty close to normal speed.

Processor support:
-Pentium 4 6x2, Pentium D 9x0, Xeon 3xxx/5xxx/7xxx, Core Duo and Core 2 Duo processors (excluding the T5500)

-Socket AM2 AMD processors

### Post by snek on 2007-01-05
I don't know what you are coding with dreamweaver, but in case you are programming PHP I strongly suggest checking out Zend Studio Client. It even comes in Linux flavor and seems to run better under Linux than it does under Windows! :D

However, it IS a professional package. Don't know if there are student versions out.

I used to always work in Dreamweaver as well during my study, but since I started working at a company I have completely dropped it. The syntax highlighting and code-completion of Zend are vastly superior imho, and that's not just for PHP but also all other webbased languages basically.

If you need it for CSS/template design and really can't live without WYSIWYG I'm sorry to tell you I can't really help you finding an alternative program.
