### Post by RomeReactor on 2007-12-21
Hi. Is this the only application that gives you this problem? if you have desktop effects running, try disabling them and see if the problem persists. If you have Beryl installed, please remove it as Gutsy now uses Compiz Fusion. If you have an ATI or Intel video card, you may also need to set up the drivers again (or even need to install xserver-xgl, in ATI's case).
