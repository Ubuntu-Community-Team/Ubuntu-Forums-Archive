### Post by petersjm on 2006-10-11
Ubuntu uses [Gnome](http://www.gnome.org/), Kubuntu uses [KDE](http://www.kde.org/). Edubuntu is aimed at educational establishments, I believe, with specific educational programs pre-installed, and Xubuntu uses [XFCE](http://www.xfce.org/).

### Post by anschelsc on 2006-10-11
The main difference is look and feel. If you have a reasonably new computer (say new enough to comfortably run Windows XP) I would go with Ubuntu or Kubuntu, your choice (you can try out either LiveCD before you install). Xubuntu is good for older computers but is not so user-friendly if you don't need it. Edubuntu is mostly for schools. But once they're installed, you can always install any programs or window managers(look and feel of the desktop) with Synaptic or Adept package managers.
