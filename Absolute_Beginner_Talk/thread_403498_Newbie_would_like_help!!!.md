The install cds is also a live cd; you can sample before you install. This will let you know if all the drivers will be ok. For most people everything just about works out of the box; ubuntu is easier to install than windows. You wont get 3d / opengl graphics without installing other drivers; but this is all relativly simple. The other things which probably wont work out of the box is web cams, tvs and wifis. Most of which should be okish to do. Depending upon what you have. Your laptop probably has built in intel wifi? This seems to be quite well suported as there are so many people with this hardware.
