### Post by into_311 on 2007-07-09
One thing to keep in mind is that Wine is not a DIRECT replacement for running windows applications under Linux. It aims to be fully compatible, but it's definetely not there 100%.

Have you looked at using open source alternatives to your windows applications?  That is the best choice to go with, because then you are supporting open source development. I believe, in time, you will find that many open source replacements for windows programs are actually better and free also. I'm going to provide you with a few links to help you in finding a open-source alternative.
