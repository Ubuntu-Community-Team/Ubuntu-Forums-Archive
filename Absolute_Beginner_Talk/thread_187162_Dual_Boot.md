### Post by catlett on 2006-06-02
you will just squeeze the install on 3gb. If you just want to try ubuntu out, install it on the 3gb. Run the install and let the installer partition and install the system automatically. You will just make the install. You won't have room for documents,mp3s etc but if you want to learn its fine. Plus you will be able to access your ntfs NOT WRITE (there are write apps but they are not perfect). The point is you can acces anything in your 200gb ntfs. So put everything in there and access it from ubuntu.
This way you don't have to do anything and you can get the feel of ubuntu. If you like it you can always partition the large drive or get another slave drive. 
Go for it and learn ubuntu. You'll just make the install and you will have access to everything in your ntfs (read access, play mp3s or open documents, you just won't be able to write anything to it or delete files) Good luck. There is a good dual boot guide in my signature plus a dapper guide for when you get the hang of it and want more out of your install
