### Post by neurostu on 2008-02-23
It sounds like someone else might be using your account.  I have Pidgin running on two machines. When I sign into AOL on both, I get the message like "You are already signed in... type 1 to sign out at the other location"  if I type 1 then it signs me out on the other machine and pops up a window that says "Connection reset by peer".  So it isn't really a problem with Pidgin its more of an issue with AOL.  

It is totally possible that someone is using your account without your knowledge.  **I would strongly recommend changing your AOL password** and that should get rid of the problem if someone else is using your account, if that doesn't fix the problem try setting up a 2nd account and see if you get the same problem.
