### Post by cjssmo on 2007-06-27
Hey Tim I don't know how to recovery your bookmarks but in the future use Foxmarks it is a extension for firefox (you can install it from firefox's add-ons page).  What it does is that you sign up for a free account on a foxmarks server and it automatically saves your bookmarks to the server so later if you happen to have a crash or a hd failure when you reload firefox install foxmarks and you can go back in and resync your bookmarks.  It came in very handy with a recent hd failure of my own.  good luck
