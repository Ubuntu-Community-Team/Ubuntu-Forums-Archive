### Post by AndyCooll on 2007-07-24

There are strong opinions on both sides. When I first used Ubuntu I used Automatix and will always be grateful for the help it gave me in easing me into this distro. However, as time has gone on I've found I no longer need it.
Further to this, the Ubuntu team have done alot of work in making codecs (for instance) easier to install. These days the "need" for Automatix has decreased. As others have mentioned installing extras can be done very easily using Ubuntu's own built in apps.
