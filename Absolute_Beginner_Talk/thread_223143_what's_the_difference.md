### Post by Jucato on 2006-07-26
Well, there is a very, very, very slight difference:

1. "KDE in Ubuntu" = Ubuntu base + ubuntu-desktop + kubuntu-desktop

2. "KDE in Kubuntu" = Ubuntu base + kubuntu-desktop

In #1, you basically install kubuntu-desktop (which is Kubuntu's selection of KDE defaults and some of their own customizations) over an existing Ubuntu (GNOME through ubuntu-desktop) installation.

In #2, you get a clean kubuntu-desktop installation.

Other than that. They're exactly the same.
