### Post by T700 on 2006-07-13
Make sure your PC is set to boot from the CD drive (the setting is in BIOS).  Insert the CD.  Follow the prompts.

Go with the Live CD prompts if you want to test things and not destroy your existing data and operating system.  If you actually install Ubuntu to your drive, you **will** lose all your current data.
