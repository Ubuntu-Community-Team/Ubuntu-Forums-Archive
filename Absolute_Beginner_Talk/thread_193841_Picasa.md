### Post by Zyzzyva100 on 2006-06-21
Just to clarify things here, Picasa for Linux does not use WINE per se, it uses the WINE lib.  This is no different than an application using qt or gtk as its "engine".  Google just used the WINE lib so that it would be easier to port.  In the process of porting Picasa alone to Linux Google also contributed over 200 patches to the WINE lib.

So for everyone complaining that its using WINE, then I guess you also shouldn't use anything that uses qt or gtk either (good luck with that one) :-)
