### Post by Mornedhel on 2007-07-28
The default XMMS install (read this as no additional packages) is in fact not very large media library-friendly. XMMS 1 (the one available in the Feisty repos if I'm not mistaken) is not maintained anymore, and XMMS 2 seems to have stalled. Plenty of plugins are available though, thanks to a modular architecture, a community that used to be very large, and, well, age.

Exaile! is more pretty-looking, and also supports last.fm and cover retrieval off Amazon out of the box. It can handle a large media library easily (but sometimes chokes on malformed Unicode tags). But overall, it's just a matter of taste, so you can experiment with various media players and see which one you find better. The other contestants being :

Banshee. May segfault without warning.
Listen. Somewhat clumsy.
Rhythmbox. The preinstalled media player, but one can find it limited.
XMMS. Old and unsupported, however has hundreds of plugins to choose from.
Plenty of others, the names of which I can't remember right now.

And of course, the current champion : Amarok. Yes, it's better than all the others. But it's KDE, so you'll have to downloaded about a hundred additional megabytes worth of packages and have the overhead of running both Qt and GTK applications. Exaile! is supposed to be GNOME's Amarok, but is still lacking in certain areas (notably configuration - well, obviously, since it's Gnome...). Still, I enjoy it pretty much.
