### Post by brownknight on 2007-06-10
What you have downloaded is log-in theme. 

You have to go to >System>Administration>Login Window. Click on Local tab. Drag the file you downloaded  (whatevername.tar.gz) over the Login Window Preferences and it will ask if you want to install it. Click Yes and then find and select this from the list of themes you now have. 

When you logout - will should see this new theme. Hope this helps.
