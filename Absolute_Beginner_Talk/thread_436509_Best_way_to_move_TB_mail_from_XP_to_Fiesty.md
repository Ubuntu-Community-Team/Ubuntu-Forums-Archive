### Post by dpar on 2007-05-07
If Fiesty has the windows partition mounted, then find your TB profile on the windows os and drag and drop it to the .thunderbird directory on Feisty. Then modify the profiles.ini file to point to the new profile.

Edit: All you need to do is transfer the profile, all your mail, addresses etc go with it:)
