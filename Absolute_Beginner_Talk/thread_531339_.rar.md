### Post by oddball on 2007-08-21
It is a command line tool.

Use ```
man unrar
``` to learn how to use it, but as a quick start you extract files like this: ```
unrar x FILENAME.rar
```

### Post by Majorix on 2007-08-21
No its not a command line tool. You can right click a file and say extract here and unrar will be run automatically. Raring goes the same way but you have to say add to archive or something similar if doing so.
