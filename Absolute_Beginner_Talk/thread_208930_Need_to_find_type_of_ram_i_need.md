### Post by Average_Joe on 2006-07-04
Your mobo is the MSI KM4M-L .

You need to buy DDR-333 PC2700 RAM.
This would need to be non-ECC RAM.

Another EXCELLENT tool BEFORE purchasing any RAM
is the memtest86 bootable disk.  I highly suggest it, here:
[http://www.memtest86.com/](http://www.memtest86.com/)

I always use memtest86 before purchasing RAM for any machine.  It tells me everything I need to know so that I can order in confidence.

A good sight to get help with memtest86 is [http://www.commonmancomputing.com](http://www.commonmancomputing.com)

Your motherboard web site is [http://www.msi.com.tw/program/products/mainboard/mbd/pro_mbd_detail.php?UID=475](http://www.msi.com.tw/program/products/mainboard/mbd/pro_mbd_detail.php?UID=475)

Your MSI motherboard:
• Supports four memory banks using two 184-pin DDR DIMM.
• Supports up to 2GB PC1600/PC2100/PC2700 DDR SDRAM.
• Supports 2.5v DDR SDRAM.
