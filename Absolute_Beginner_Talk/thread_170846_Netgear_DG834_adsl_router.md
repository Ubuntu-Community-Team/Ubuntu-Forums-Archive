### Post by jeremy on 2006-05-05
I have the same router, and ubuntu detected the the ethernet card, and the routers DHCP setting automatically, it just worked.

Do you have the router set as 'Use Router as DHCP Server' in the 'LAN IP Setup' page of the routers configuration?

Can you access the router by navigating to 192.168.0.1 using your browser?
