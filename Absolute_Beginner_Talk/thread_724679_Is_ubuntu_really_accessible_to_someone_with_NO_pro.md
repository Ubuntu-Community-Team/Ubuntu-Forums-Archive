### Post by ramjet_1953 on 2008-03-14
A couple of things to consider:

1. Ubuntu Linux is actually much easier to install than a clean install of Windows.
Most Windows users have never actually installed an OS before, as most new PC's come pre-installed with Windows.

If you try a clean install of Windows, especially Vista, you will find that it can be a nightmare getting the correct drivers for your hardware. If you have misplaced the CD's that came with your printer, scanner, even you video card, it can be very time consuming to search the internet for the correct drivers.

2. With a clean install of Ubuntu, most of the drivers for your hardware are already included in the kernel.

Don't get me wrong, here, though, there can be problems if you have hardware such as a video card, or a wireless chipset that is very new.

Many manufacturers do release Linux drivers now, but they generally are lagging by a few months behind the Windows drivers.

Also, some chipset manufacturers in the past have made using Linux very difficult, as they have refused to produce Linux drivers and have also refused to release information, so that others can write a driver.

Examples of this are SIS video cards, some ATI video cards, and Lexmark printers.

This aside, these forums and other Linux forums are very good at overcoming difficulties and offering solutions, or alternatives.

One final word.

Linux is not Windows and doesn't try to be.

You have probably spent many years, gradually learning Windows and it's many quirks. Be prepared to spend some time learning Linux and the way it does things. It is well worth the effort. 

Many have problems with understanding how the file system works, but when you get your head around it, you will see that it is vastly superior and very logical.

Welcome on board, and don't forget to ask all of the "dumb" questions that you like, as there are probably a multitude of us who are to embarrassed to do so.
