### Post by Inxsible on 2007-06-12
A better idea would be to open up Eclipse and go to Help --> Software Updates --> Find and Install.
 
You can put in the web address of your plugin and install it from there and let Eclipse worry about where to put the plugin file.
 
Although, in Windows, simply extracting the plugin in the /eclipse/plugin directory seems to work. I am not 100 % sure whether it will in Ubuntu too.
