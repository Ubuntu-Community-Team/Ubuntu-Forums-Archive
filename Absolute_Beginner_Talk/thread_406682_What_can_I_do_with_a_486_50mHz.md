### Post by brentoboy on 2007-04-12
I'm typing this on ubuntu feisty on an old k6 box, and it runs like greased lightning.

the secret -- well, its acutally running on my "real" computer, but I am using the remote X terminal feature, so it works great over here on this ancient bucket of bolts.

here is the way it works.

Go to debian.org, and get the floppy install disks (I'm assuming you cant boot from CD -- if you can boot from CD get the net install CD.)

boot it up and install the very most minimal version of debian you can manange.

Then, apt-get x-window-system-core

while that's chugging anyway, go over to a nice box on your network, and go into your logon settings, and enable xdmcp (the remote login tab)

now, go back to your paperweight/boat anchor.  (if its ready) and issue this command to run an x session:

X -query 192.168.0.2  (or whatever the ip address is of the computer running xdmcp)

please note that after enabling xdmcp in the login settings, you will need to restart gdm in order for the change to take effect.

if all goes well, you should have a very impressive desktop (considering the circumstances)

this only works if you have a decent lan in place, and if you have a firewall to protect you from outsiders connecting via xdmcp.

its a pretty sweet setup. I can go down to the local thrift shop, and pick up the systems that they cant even sell, and turn them into extensions of my powerbox.

good luck with that, and.. if all else fails, bardstale was a good game from that timeframe.  or skate or die.  man, those were the days.
