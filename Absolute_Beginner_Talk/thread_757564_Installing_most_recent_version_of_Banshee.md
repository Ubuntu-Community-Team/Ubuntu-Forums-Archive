You are probably better off with 0.13.1. Version 0.93 is based on an entirely new architecture and is currently in development. 0.93 is alpha, 0.13 is not. Also 0.93 lacks many features that 0.13 has. See [http://www.banshee-project.org/Releases/0.98.3](http://www.banshee-project.org/Releases/0.98.3)

>  Currently Missing Features

    * This release contains limited hardware support
          o No CD burning support
          o No iPod support
          o No MTP device support 

    * Additionally, a number of plugins have not yet been ported to the 1.0 architecture
          o Podcasting
          o DAAP/iTunes Music Sharing
          o Internet Radio
          o Recommendations
          o Mini Mode
          o Batch Cover Art Fetching 


However, 0.98 can be installed along side 0.13.
