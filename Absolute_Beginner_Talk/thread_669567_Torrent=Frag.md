### Post by SanjoEel on 2008-01-16
The Linux filesystem stores data differently than a Windows fs. Although some fragmentation can occur, it is  much less severe than in a Windows fs, and generally not a concern.  I can say that I have never corrupted a Linux fs by repartitioning without defragging.
