The CD I have here has Windows and Linux on it.

You have to open the Linux folder on the CD and select whether you want 32 or 64 bit, and you will see a bin file of approx 60 mb.

If you drag that file onto the desktop, it will subsequently splurt various things over the desktop, but if you make a folder in your home folder called (say) Personal Accountz, you can drag it into there. Then you have to right click on the file and set the permission to execute.

Now from memory, you double click, select "Run from Terminal" and watch it go. At some point in the process, you will need to key your key.

All this is from memory, however I am due to install the program on my laptop, so if there is a problem, I'll log in the exact steps involved.

It was fairly simple. Distro of Ubuntu is Gutsy 7.10.
