I've never heard of this application. When I googled it, using the word search

TROAMBERS database

I got a grand total of 0 hits, which is saying a lot I think.

An .accdb file is made by Microsoft Access 2007. A .pub file is made by Microsoft Publisher.

As for using them on Ubuntu, you could install Wine (I recommend compiling the latest SVN version, which is complicated.) or looking at CrossOver Games Edition (which sounds odd but I read on the 'net about some people having success with this version of CrossOver and Office 2007).
