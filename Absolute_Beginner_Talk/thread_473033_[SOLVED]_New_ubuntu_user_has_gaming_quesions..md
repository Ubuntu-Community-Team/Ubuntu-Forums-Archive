Open up a terminal, and type
```
sudo aptitude install tremulous tremulous-server tremulous-data tremulous-doc
```

That will fully install Tremulous, which is an awesome FPS based off of the Starcraft world. You can also look for  World of Padman, which is a fun kinda-cartoonish FPS. World of Padman is easiest to get by searching Ubuntuforums for the thread all about it. That sould do it!!

### Post by ironfistchamp on 2007-06-13
Diablo 2 would need Wine or Cedega to run I am afraid. Cedega is a subscription based program that focuses on getting Windows games to run under Linux. Personally I found it to be troublesome and not worth the download let alone the money I paid for it.

Wine is brilliant but it is best to check out [http://appdb.winehq.org](http://appdb.winehq.org) to see what programs are supported.

Big commercial games can be a bit of a pain to get working in Linux as Wine and Cedega are slightly behind the latest technology (which is understandable). Some games have native ports such as UT2004, Doom 3 and Quake 4. These are brilliant games and can be picked up quite cheaply. UT3 should have a Linux port aswell when it finally arrives.

Over in the Gaming and Leisure forum here there is a great guide for getting Oblivion to work under Linux using Wine. Might be worth checking out.

HTH

Ironfistchamp
