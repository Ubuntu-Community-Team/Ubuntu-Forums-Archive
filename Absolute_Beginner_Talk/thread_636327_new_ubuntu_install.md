### Post by oldos2er on 2007-12-09
Type "sudo aptitude install compizconfig-settings-manager" in a terminal. Once it's installed go to System, Preferences, Appearance, Visual Effects; click Custom, then the Preferences tab next to it. You'll find the Desktop Cube and Rotate Cube options there.
