If you are running the 32-bit version (most popular) of Ubuntu, then you are running Intel386 software.  You can run 32-bit software on a 64-bit processor.  

At this point, I reckon most owners of a 64-bit machine only use 32-bit software.

So the short answer is, yes it will run fine.
