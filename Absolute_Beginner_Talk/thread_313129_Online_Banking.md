This is how the web is supposed to work.  A properly written page is supposed to work in any properly written browser.  The only reason this isn't always the case is because proprietary (im sure you can think of at least one) software manufacturers want pages created with their software to work properly only with their browser, and their browser to only properly display pages created by their software.
