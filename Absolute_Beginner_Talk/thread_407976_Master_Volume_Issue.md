### Post by Bias on 2007-04-13
I'm guessing you are using the full alsamixergui (is wider than the screen)? If so you may have that control locked, click on the little set of speakers above the bar controls, they need to be green to enable you to change the setting.

If you are using the cut down version, has only about 3 visible controls, then File menu there is an option to change the device, I have 3 sound devices (apparently :)).

If you don't have the full alsamixergui, it is worth installing from synaptic as it gives you a bit more control.

Hope this helps.
