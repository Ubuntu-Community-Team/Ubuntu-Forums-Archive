### Post by lswest on 2008-02-12
to delete Ubuntu just format the partition it was in using GPARTED off the liveCD and resizing the XP partition to take up the empty space again.  be sure to back up the data you want to keep before doing so.  (if you don't have a liveCD handy, you can download [a gparted liveCD]("http://gparted-livecd.tuxfamily.org/")).  Afterwards you need to use either super grub disk, or an XP CD to restore the boot loader.  (Or install easyBCD into XP and have it restore the bootloader)

*EDIT* I'm assuming here that XP is already installed
