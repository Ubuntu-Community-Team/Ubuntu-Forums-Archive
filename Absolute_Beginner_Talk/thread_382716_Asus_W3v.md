### Post by John.Michael.Kane on 2007-03-12
looking over the ASUS W3V specifications it would seem you should have no issue with installing,however. 

The items listed below stand out as they may not be fully supported. 
Built in Bluetooth
Azalia audio chip
Card Reader

If you want you can download the live cd which is linked here-> [http://releases.ubuntu.com/edgy/ubuntu6.10live.iso]("http://releases.ubuntu.com/edgy/ubuntu-6.10-desktop-i386.iso")
This will allow you run the OS,and see if it fits your needs,and works with your hardware with out erasing your current running OS. 

Should you feel it fits your needs,and works with your hardware the same cd can then be used to install the OS.

included is a guide for getting the ati card working ***after you install the OS***
[Edgy ati installation guide]("http://wiki.cchtml.com/index.php/Ubuntu_Edgy_Installation_Guide")

### Post by TEDNUGNT on 2007-04-05
first, since nobody answered my last post and having figured it out on my own, i will answer it myself.
