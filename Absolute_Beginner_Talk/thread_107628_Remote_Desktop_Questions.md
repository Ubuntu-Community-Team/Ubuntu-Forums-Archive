### Post by cstudent on 2005-12-23
You'll turn on remote desktop on your Ubuntu machine. Set it to not ask for confirmation, but DO set it to ask for a password. You use Remote Desktop in Windows to connect to your Ubuntu pc. You'll need to know the ip address of your home connection. You'll also need to setup port forwarding if you have a router between your Ubuntu pc and your modem. I think it's 5500, but I'm not positive. You can google for vnc ports and should be able to find out. Also, some ISP's block ports, so you may want to test things out before you leave if you can.
