### Post by unarmedninja on 2007-05-20
i dont know of any software that will do it directly for you but you can do it with a few command line tools.
use mencoder or ffmpeg to convert the divx/xvid avi to mpg format then you can use vcdimager to convert the mpg to vcd bin/cue format.
 
you can use any burning software to burn the bin/cue image.  you should know theres going to be some quality loss when going from divx to vcd or svcd.

im assuming you want to play the video on a dvd player since theres no other reason for converting that way.  some dvd players let you play the mpg file as is so you might be able save yourself an extra step of converting to bin/cue.

good luck.
