### Post by bikeboy on 2007-02-28
Not sure about the system beep, but try the Settings Manager.

For F-Spot, since you're new it's probably simpler to keep with the software versions Xubuntu gives you for now, as the next release (Feisty Fawn) is due in April and it has F-Spot 3.4.0 at the moment and may even update further.

[http://packages.ubuntu.com/cgi-bin/search_packages.pl?keywords=f-spot&searchon=names&subword=1&version=all&release=all](http://packages.ubuntu.com/cgi-bin/search_packages.pl?keywords=f-spot&searchon=names&subword=1&version=all&release=all)

Once the new release comes out, you will be able to upgrade the whole operating system with one command over the internet, that way the versions of many programs will be newer.
