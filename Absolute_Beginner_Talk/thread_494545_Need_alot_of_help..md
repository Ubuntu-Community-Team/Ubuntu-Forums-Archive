### Post by eentonig on 2007-07-07
Follow the links in my signature, they'll give you some basic understanding about linux and how to get things done.


Linux is totally different then windows in how it accomplishes certain things. So it is confusing when you first use it.

To answer your questions.

No, there are no .exe or install files to click (well, there are. But you're better of not using them.) To install programs, go to System, Administration, Synaptic Package Manager. This tool allows you to search for an install almost every app you want.
