### Post by Ampersand on 2005-08-14
You can install KDE by getting the kubuntu-desktop package. When you next log in, select KDE from the sessions menu and set it as the default session when prompted. I'm not sure what the best way of removing Gnome is. It's probably best to leave it installed if you don't need the hard drive space too much. You can remove the gnome programs you don't need, though.

### Post by qalimas on 2005-08-15
My Ubuntu CD got scratched, so I had to do the exact opposite using Kubuntu ;)

First, open Synaptic, get kubuntu_desktop, get all the KDE apps you'll want or need, then mark kdm for install, once all that'd done, go in the gnome section, select one, hit ctrl + a, then uninstall them all, make sure you get rid of gdm as well.
