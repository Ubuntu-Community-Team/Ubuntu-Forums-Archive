> **KathyMary said:**
> 
Questions for others : why do some "K" (?) programs work and some don't on Gutsy? My husband said K was another desktop environment  - right term? . He also said things could get messy if we had different desk tops while both of us are learning. 

It is actually called KDE.  There are several desktop environments available for linux, but the "big two" which most people use are Gnome and KDE.  Personally I prefer Gnome, I like its simplicity and i think it looks a bit slicker, less imposing.  But that is just personal preference, there are planty of people who just couldnt go without KDE.

Generally KDE based software should work in Gnome and Gnome based software should work in KDE.  But as with everything in computers, things are never perfect and occasionally one or the other may not work too well.

For KDE software to work in Gnome you need to install some KDE packages, but synaptic should take care of those dependancies for you when you install it, so you shouldnt have to worry if you install things from the repositories - which you should be doing most of the time.

> **KathyMary said:**
> 
He also said things could get messy if we had different desk tops while both of us are learning. 


There is nothing wrong with having both KDE and Gnome installed.  All it means is that you will get the option at the logon screen as to which one you want to log into.

I have never done this myself but i think it is simply a case of installing the kubuntu-desktop package.... But probably best to research that first before you do it and break everything and blame me! lol.

But as you say it might be worth just getting settled with gnome first, as there are so many options and choices with linux that it can become a bit confusing at first, so best to get the basics mastered first.
