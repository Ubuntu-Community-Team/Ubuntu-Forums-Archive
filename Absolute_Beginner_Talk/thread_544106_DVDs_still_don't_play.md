### Post by mocoloco on 2007-09-05
You need to install a package called, libdvdcss2, however it's not included in the repositories anymore.  I think this is because it's legally "iffy" in the USA, even though nothing has been proven in court.  More on that [here]("http://en.wikipedia.org/wiki/Libdvdcss").
Check out [medibuntu]("http://medibuntu.sos-sts.com/repository.php") to add their repository, and then search for and install the package with synaptic.
