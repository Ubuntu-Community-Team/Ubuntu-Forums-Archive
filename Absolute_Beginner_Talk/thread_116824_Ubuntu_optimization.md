### Post by d1337 on 2006-01-13
From my experience, Ubuntu is very user friendly and provides a workable desktop pc very quickly, assuming that your hardware doesn't have Linux compatibility issues.  That said, I use Ubuntu for 2 laptops, 5 desktops and one (non-production/testing) server and have had great success with all of the machines.  I have no experience in clustering and for a standalone hardware firewall, you'd likely be just as well off with monowall or something similar...but Ubuntu, being Debian based, offers a simple apt-get package manager or Synaptic for graphical allowing you to install a variety of firewall and network security tools.
