### Post by voodoonirvana on 2006-11-14
Hey, I have a few things installed in wine and I was wondering if there is any way that you can make a link to that program for your desktop, instead of having to open it with the terminal every time.:confused: Am I missing something?

---

### Post by schrombot on 2006-11-14
If you are using Kubuntu, just right click on the desktop and choose Create New > Link to application. Put the name of the application on the first page and then go to the "Application" tab and type "wine *path to application*". That should do it for you. If you are using Ubuntu, I wouldn't be sure because I don't have any experience with gnome, but I imagine it would be similar.

### Post by CatKiller on 2006-11-14
And if you're using Gnome it's pretty much the same - right-click the desktop and select Create Launcher...

I've found that for programs that make stuff in the pretend Start menu, you sometimes get launchers created in ~/.gnome/apps/Wine. You should be able to drag-n-drop these, if they exist, to your desktop.
