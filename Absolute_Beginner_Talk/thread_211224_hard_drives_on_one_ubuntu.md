### Post by Dr. Nick on 2006-07-08
If you are in gnome then go to System - Administration - DIsk and see if both drives appear. 

If so go the the 2nd drive and click the partitions tab and set it up their.


If you dont use gnome ( can be done even if you use gnome aswell ) then look into editing the file /etc/fstab
