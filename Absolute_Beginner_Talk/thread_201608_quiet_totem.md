### Post by tuxcantfly on 2006-06-22
try playing a different sample file. if you hear the sound, you'll need more codecs to listen to the file you're playing (look under the gstreamer* stuff). if you don't, check if you have alsa, esd, and the rest of the stuff working. just play a simple non-gnome game like supertux, and if you get sound, it's a problem with esd, and if you don't, it's a problem with alsa.
