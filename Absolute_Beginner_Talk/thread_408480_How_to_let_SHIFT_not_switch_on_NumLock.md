### Post by leuce on 2007-04-13
G'day everyone

I'm a Windows user, so this questions may be recognised as "trying to do Windows in Linux", heh-heh.

When NumLock is off, I can use the number pad's arrows and directional keys.  When it is on, those keys type numbers.  But here's a weird thing (in Ubuntu) -- when I press the keys on the number pad while holding down Shift, even if NumLock is off, Ubuntu treats it as if NumLock is on (and consequently types numbers instead of moving the cursor).  Similarly, when I use the number pad while NumLock is off, while holding in Ctrl, the directional keys eg PgUp and Home work fine, but as soon as I press Shift as well, the functionality of the number pad stops (ie, Shift+Ctrl+numpad does nothing).

Any idea how I can get Ubuntu's keyboard to work like the one I'm used to in Windows (with regard to the above)?

Thanks
Samuel

### Post by mac.ryan on 2007-04-13
System -> Preference -> Keyboard -> Layout Options -> Miscellaneous compatibility options -> shift with numpad keys works as in MS windows.

Welcome to ubuntu! :)
