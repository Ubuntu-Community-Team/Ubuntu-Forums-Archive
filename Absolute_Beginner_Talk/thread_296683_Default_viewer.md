### Post by cairnsww on 2006-11-09
When I click on a jpeg file in Nautalis, Eye of Gnome is my default viewer. How can I change that so I get gThumg viewer instead?

---

### Post by bmathis on 2006-11-10
Right click the file > Properties > Open With > Add. Add the program from the list and then make sure the radio button for that program is filled, click close and it should now default to that program.
