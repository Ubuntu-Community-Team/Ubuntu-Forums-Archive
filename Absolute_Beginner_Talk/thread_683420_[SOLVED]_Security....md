### Post by jleaker01z on 2008-01-31
Ubuntus firewall is built in.  Its the major difference in security between Windows and Ubuntu.  Windows security is in the user layer, meaning as a user you install your security.  With Ubuntu, it is built into the Kernel.  You can install Firestarter from the repositories if it makes you feel better tho.

As for anti virus, you can use ClamAV or AVG, tho you will find that a virus on Ubuntu is rare indeed.

Basically, what I am saying, is you dont have to worry about it.  But if it makes you feel more secure you can install the mentioned programs.

### Post by seventhc on 2008-01-31
Firewall is built in, you can install firestarter if you want a graphical display, as for AV, the only reason you would need one would be trading files from windows users that you might forward to another windows user.
I'll try to explain better, if your friend sends you a file and it is infected you should still be safe as it won't run on linux, if you then send it to another friend using windows, they would be infected.
So if you want to protect them then go ahead and install it, but your machine is safe. Most viruses/spyware/etc are written to run on windows since it has such a large user base and isn't secure.
Even if a virus did run on ubuntu it wouldn't be able to do much since it wouldn't have root permissions.
Now you can have a computer that you can enjoy instead of spending countless hours removing malware. :)
