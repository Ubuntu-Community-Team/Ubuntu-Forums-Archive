### Post by stalkingwolf on 2008-02-25
I would suggest that you try running 6.10, 7.04 and 7.10 Live CDs first.  see which works best
on Your System.  I run all three, depending on the system.  When i am preparing to install a system 
I try all 3 to see which runs fastest and supports the system best, taking into account add ons , memory installed, built ins , etc..   You are the only one that can make these decisions.  After YOU have made your decision, any issues can be resolved in these forums.  Asking " which is best" is like asking " what vehicle is best".  I prefer Dodge/Jeep.  But they may not "do what you need " from a vehicle.

IMHO there is no " BEST " version.  there is only the version that does what you need, when you need it.
What works for you today may not be what you need next week.  Things change.
