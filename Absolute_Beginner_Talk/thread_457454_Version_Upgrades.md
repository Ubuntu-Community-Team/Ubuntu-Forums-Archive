### Post by glennph93 on 2007-05-28

### Post by AAM on 2007-05-28
the short answer is **YES**.
the issue of data loss when upgrading can be reduced by having the /home directory on a different partition. In that way you can upgrade or re-install the operating system as required and leave the home directory with their data alone.
you have to work out if this has been done and if not, get it corrected.

just a passing thought, but wouldn't you want them to stay on a release for much longer than 6 months if it's a "work" machine? Go for the H or I version?
