### Post by blendmaster on 2006-10-10
Well, you could install WINE, which for the 64bit version, you'll have to use this guide [http://www.ubuntuforums.org/showthread.php?t=167765](http://www.ubuntuforums.org/showthread.php?t=167765)

Then install the windows version of firefox using WINE.

Then in Firefox, go to a site that requires flash. Click the "install missing plugin" link. This should work.

Any questions on doing this just ask!

Also, this takes a while to do, and the flash is only mediocre, but it's better than nothing.
