### Post by gn2 on 2007-08-21
I would suggest Xubuntu 7.04. It uses the XFCE desktop environment and is suited to such hardware.
I have it running on a P3 500mhz laptop with 192mb RAM, and it's much quicker than Windows 2000 Professional.

My preference is for the "Alternate" CD: [http://www.xubuntu.org/get](http://www.xubuntu.org/get)

Install using these instructions: [https://help.ubuntu.com/community/InstallingXubuntu](https://help.ubuntu.com/community/InstallingXubuntu)

Do it this way and you get a leaner installation.
