### Post by Quillz on 2007-03-24
The LiveCD by default will not see your hard drive, although you can run a command to make it do so. Once it is installed to your hard drive, you will partition it and it should be able to natively see the other partitions, thought you may or may not be able to write to them.

Technically, the latest Live CD is the 7.04 beta, but the latest stable Live CD is of 6.10.
