### Post by an.echte.trilingue on 2006-11-16
They look and feel is different.  To get an idea of the differences, go to [osdir.com]("http://shots.osdir.com/") and take a look at their screenshots.  

Generally, xubuntu is better for older/slower systems because the desktop environment (xfce) is very light weight.

A lot of people consider plain Ubuntu to have the best graphic design of the *buntus, which in fact pretty important for a usable system.

Edubuntu uses the same desktop environment as Ubuntu (gnome), but it also comes with a bunch of educational software for kids.

Kubuntu uses a desktop environment that is generally considered to be the prettiest and most configurable at a GUI level in GNU/Linux, but I find it pretty buggy.

It should be emphasized, though, that as long as you have an internet connection, you can install all the applications from one DE and use them in another, or even install the complete DE and switch between them.

Hope this is helpful!

### Post by hyper_ch on 2006-11-16
The best way to install alternate GUIs is:

Open a terminal and type:

(1) for Gnome (Ubuntu):  sudo aptitude install ubuntu-desktop
(2) for KDE (Kubuntu):  sudo aptitude install kubuntu-desktop
(3) for Xfce (Xubuntu):  sudo aptitude install xubuntu-desktop
(4) for Edu (Edubuntu):  sudo aptitude install edubuntu-desktop
