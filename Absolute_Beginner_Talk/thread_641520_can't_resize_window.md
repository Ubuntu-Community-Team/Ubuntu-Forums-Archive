Ubuntu will support higher resolutions if your display drivers support them, but it seems to me that the card manufacturers put far less effort into the Linux versions of drivers than the Windows ones. 

Sometimes, apart from the Screen Resolution menu item in System/Preferences, there is a utility that comes with the driver (in nVidia's case it is nvidia-settings (run it from the terminal)) but it doesn't work that well for me, so I just stick with 1024x768 on my desktop PC.

I don't know of any utilities that sort this out. I agree though it would be nice if windows that were too big for the desktop had a scrollbar.

Maybe you can set up a large desktop somehow, you know where the whole desktop is larger than the screen res, so it automatically scrolls when the mouse is near the edge of the screen. I've seen this on some setups, but I don't know how to set up Ubuntu to do that.
