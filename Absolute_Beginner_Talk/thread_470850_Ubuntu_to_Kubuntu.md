If you want to check out Kubuntu using:

Will give you what you want without installing too much, and it is easy to remove if you don't want it.

There is a third option you may want to know before you do that. You can install Kubuntu into a VM (kinda like installing a clean copy, you'd have to have a Kubuntu CD or an Ubuntu CD and then install Kubuntu package over it), [Virtual Box]("http://www.virtualbox.org/") for example supports it very well. I'm using VM's myself to try out Fedora and Debian at the moment and see the differences :).

I like VMs because you don't need to modify your system in any way to try it. Only advice I have is you should give at least 256 of ram to the VM, I recommend 512 (or more) for optimal.
