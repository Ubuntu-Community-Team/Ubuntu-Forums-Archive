### Post by rune0077 on 2008-04-18
In Appearances, select "Install Theme" (or something along those lines), then install the files. Afterwards, you should be able to find them in Customize > Icons.

### Post by farsight on 2008-04-18
I'm using the gnome look black and white 2 gloss icon set as far as I know. To install these I created a themes folder in the home directory and extracted the .tar.gz to there. I then clicked system > preferences > appearance followed by install. Direct it to the theme folder in the home directory and click open. It says to use new theme and you give it the thumbs up. Hope thats what you were looking for buddy :)
