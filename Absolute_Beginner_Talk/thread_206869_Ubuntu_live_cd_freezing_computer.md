### Post by catlett on 2006-06-30
You have plenty of processor power but how much ram do you have?
I would recommand the Alternate Install CD. You can download it at the same site. I recomand this because it takes alot of resources to install from the live cd. Your computer has to run a full xserver session from the cd and then format and install a full operating sytem to your hard drive at the same time.
With the Alternate Install CD you only have the install running. My signature has a great how to for the alternate install cd.

Back to ram. For Ubuntu you could probably run on 64mb of ram. 128 would be my recommended minimum and 256mb or more would be optimal.

Good luck.

P.S. xubuntu is top notch as well if you have issues still.
