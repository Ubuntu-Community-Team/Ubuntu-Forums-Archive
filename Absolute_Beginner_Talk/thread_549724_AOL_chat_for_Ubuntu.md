### Post by sstusick on 2007-09-12
You can run AOL 5.0 with WINE to access AOL chats. You can't access Yahoo chats with Pidgin or any other 3rd party client because Yahoo has new security features in place that require you to verify your account before you enter the chat rooms. Until these programs have fixed that issue, only way to access Yahoo chats is to use it on Winblows using the original Yahoo Messenger. 

You might also want to consider IRC for your chatting needs.

### Post by ubuntu27 on 2007-09-13
[Pidgin]("http://www.pidgin.im/") is a multi-protocol instant messenger, meaning it supports many protocols like MSN, Yahoo, Gadu-Gadu, AIM, ICQ, Google Talk, Jabber, etc. 

[Pidgin]("http://www.pidgin.im/") used to be called Gaim before, but it change it's name because AOL made a threat to them since Gaim sounds like AOL's AIM.
The previous version of Gaim includes in Ubuntu, it's in Applications/Internet/Gaim Internet Messenger.

[URL="http://www.getdeb.net/release.php?id=1307"]
You can download the latest version of pidgin here[/URL] if you want.

And if you like some security feature (don't want third or middle person over-hearing your conversation) install [Pidgin-Encryption]("http://pidgin-encrypt.sourceforge.net/")
