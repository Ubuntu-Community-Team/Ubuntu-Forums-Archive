### Post by GoGoGigaton on 2006-11-16
The section on burning a CD (Once you've obtained the CD image, burn it to CD with this guide
[https://help.ubuntu.com/community/BurningIsoHowto)](https://help.ubuntu.com/community/BurningIsoHowto)).  The only reason I post this suggestion is, as a noob to Linux (but not PCs) I ran into some difficulty.

I didnt realize you needed a special burning software for ISO images.  I just used Windows XP burn software.  All this did was transfer the ISO file onto a disk (just the iso intact, ie didnt create a boot CD).

So just a suggestion to maybe explain a little further that the ISO isnt the CD boot file.  And the software suggested ImageBurn or an alternative ISO Boot CD software is required to burn a Boot CD.

I know it might sound like Overkill.  But seems to me the point of Noob Install instructions is to make them relatively detailed and pain free.

So theres my suggestion.  Im sure youll see me posting help requests soon heh.  Im just now running the install CD (Xubuntu Edgy Effy) on my Sony Vaio Laptop.
