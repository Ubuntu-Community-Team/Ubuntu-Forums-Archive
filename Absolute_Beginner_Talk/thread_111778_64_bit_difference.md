### Post by dgbauer on 2006-01-02
I have found that running Ubuntu 64-bit on my eMachine M6805 was quite fast and supported just about everything I could find out there...  However, I do remember having troubles with ndiswrapper for setting up my wireless in 64 bit, and I could never get a working 32-but subsystem set up successfully.

The only technical difference as far as I know with 64-bit vs. 32-bit is that in a single cycle, a 64-bit processor can handle 64-bits of a data where a 32-bit can only handle (you guessed it!) 32.

I have run both the 64 and 32 bit version on my machine only to notice a small difference in speed and stability...  But overall, I have found my compatibility to be much greater running the 32-bit version.
