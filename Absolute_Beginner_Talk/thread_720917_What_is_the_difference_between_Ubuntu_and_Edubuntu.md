### Post by bbqsandwich on 2008-03-10
Edubuntu comes with a lot of educational software and games preinstalled.

Examples:
Gcompris - games for young children to teach reading, colors and numbers
Kalzium - teaches the periodic table & chemical elements

It also has a different icon set & theme.

There is a server edition, which although I haven't used it I believe is aimed at classroom environments.

A little more info is here: [http://en.wikipedia.org/wiki/Edubuntu](http://en.wikipedia.org/wiki/Edubuntu)
