### Post by dondad on 2007-11-01
Most of the commercial DVD's are encrypted. To play them you need to install libdvdcss. Be aware that this library is illegal in some countries. If you do a search on "libdvdcss", there are several tutorials on how to install it.

Look here for info:
[http://ubuntuforums.org/showthread.php?t=588643&highlight=libdvdcss](http://ubuntuforums.org/showthread.php?t=588643&highlight=libdvdcss)
