### Post by taurus on 2007-05-12
You said that you have converted /dev/sdb1 to reiserfs but you still mount it as ntfs in /etc/fstab?

```

# Entry for /dev/sdb1 :
UUID=543039003038EA9E /media/sdb1 ntfs-3g defaults,locale=en_US.UTF-8 0 1 

```
Therefore, you need to edit /etc/fstab and either remove or comment out the line above.  Then, reboot and reformat /dev/sdb1 to reisferfs again.  Then, mount it as a reiserfs filesystem in /etc/fstab.
