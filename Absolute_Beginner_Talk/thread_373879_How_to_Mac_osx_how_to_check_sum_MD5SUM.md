Hi

I've just downloaded the iso twice. Disk utility didn't show MD5SUM as an option and verify said 'no check sum infomation' on the first down load.

While wating for the second down load to complete I had another trawl through and then thought to google for it and found that you can do the MD5SUM check using terminal which is in the utilities folder.

when the terminal window opens simply type 'md5' hit the space bar once then drag the iso into the terminal window (helps if you have the iso on the desk top; but where ever it is) this automaticaly puts the path to the iso in terminal then hit return and wait. Terminal exits with the md5 number which you can verify here for the the DVD iso.

[http://www.mirrorservice.org/sites/cdimage.ubuntu.com/cdimage/releases/dapper/release.1/MD5SUMS](http://www.mirrorservice.org/sites/cdimage.ubuntu.com/cdimage/releases/dapper/release.1/MD5SUMS)

This is the english mirror but should be the same for all 6.06.1 dvd images or go to the down loads page for cds which may be different. the md5sum doc should be listed.

hope that helps some one with the first step. Now to see if it will boot!
