### Post by aysiu on 2006-06-16
[QUOTE=jeffc313]sudo apt-get install kubuntu-desktop 
best of both worlds. no need to re-install.[/QUOTE]
Or better yet ```
sudo aptitude update
sudo aptitude install kubuntu-desktop
``` That way, you can easily get rid of Kubuntu if you later decide you don't like it.

*apt-get* will just leave you with a mess.

### Post by brentoboy on 2006-06-16
if its a fresh install and you dont mind scrapping the whole thing, install kubuntu from its own cd.

if you dont want to loose what you already have, just use kde on ubuntu.  The only thing that is really different is that you have lots of gnome stuff on your PC that you dont need if you dont intend to use gnome.

if you just got started, I always recommend with a clean install to be exactly what you indended in the first place.

by the way, the kernel is no different between the two, its all the little user apps that are different.
