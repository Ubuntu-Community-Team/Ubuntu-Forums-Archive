### Post by steve.horsley on 2006-07-18
Time to learn...

Start the terminal with Accessories->Terminal

Enter the command **cd Desktop**

If you want to install somewhere that all users can share it such as /usr/local/googleearth then your next command needs to be **sudo sh GoogleEarthLinux.bin**, but if you are installing inside your home folder for yourself, just **sh GoogleEarthLinux.bin** will do.

Now follow the installer prompts.
