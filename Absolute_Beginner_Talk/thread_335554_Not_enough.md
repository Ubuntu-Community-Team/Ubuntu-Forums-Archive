### Post by jvc26 on 2007-01-10
Defragmenting XP is done via going to the start menu control panel, if i remember rightly, system -> system tools -> disk defragmenter, and defragment now. Apologies I havent used XP in a while so dont remember it without it in front of me. Once defragged it should be ok. It all depends on what you're going to use it for. If you're using both you may want:
1/ Windows Partition
2/ Ubuntu Partition
3/ Data switching Partition (to move things between the OSs)
4/ Linux Swap
In which case you'd put lots of space into the data switch, a fair bit into windows as it needs lots for its apps and then maybe a 20GB for ubuntu...
