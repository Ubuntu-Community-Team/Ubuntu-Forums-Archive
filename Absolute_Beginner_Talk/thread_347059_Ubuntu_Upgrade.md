### Post by coolen on 2007-01-27
1. You can download codecs for your multimedia files, and as said before, most features of Office documents will work nicely. I noticed that Abiword didn't load some of my images, but I hear OpenOffice is damn near perfect.
2. I agree with dual-booting. Try it out, see if it's for you, and if you decide that it is, you can either just keep using it, or wipe your old Windows partition for extra storage, if you're feeling bold.
3. I had a Canon printer that worked flawlessly out of the box with Ubuntu. Installed easier than it did on Windows, and worked better, too. Again, dual boot just to try out your configuration.
4. If you're connecting to your modem via ethernet, it's incredibly easy. If you're using USB, you'll have to go through some extra steps. The wiki has instructions for this, though, and it should be possible.
5. Absolutely.
6. Ubuntu is Debian based, so you have instant access to a lot of software made specifically for your distribution, and a lot of it is very actively maintained. Installing new software is usually as simple as point, click, download and run. In the unlikely event you need some software that isn't found in the repositories, you may find they provide a .deb package, which is still incredibly easy (although if they maintain a repository it's even easier), or worst case scenario you have to unpack the tarball yourself.
7. Linux is made to be secure. Unlike Windows, where security seems to have been more of an afterthought, Linux was designed from the beginning with security in mind. If you run a server (running a business you probably will), a firewall may be neccesary. There are many good firewall applications out there, some of them easier to use than others. Shouldn't be a problem.
