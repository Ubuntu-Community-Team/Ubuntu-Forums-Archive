### Post by Paqman on 2007-10-09
Cool, welcome to the forum!

One of the biggest things people mess up when they start off is installing software. That's because it's actually much *easier* to install software on Linux than they expect! 

Almost everything you need can be installed through Synaptic, which is a "package manager", or through the "Add/Remove" menu.

To use Synaptic you search for the software by name or category and hit a few buttons. The system will download and install everything automatically. You can even install several packages at once. It gets the packages from an online software library called a repository.

Lot's more info in the [Synaptic HowTo](https://help.ubuntu.com/community/SynapticHowto)

Just to clarify, as OP said he was new to Linux. Root is equivalent to the Windows administrator. Whereas in Windows, it is almost normal practice to run as admin all the time, this is frowned upon in Linux, as root has the capability to literally destroy all the data from a computer, even in other partitions.

However, if you need administrative rights (superuser rights), the system will prompt you to enter your password, and you can then install programs or configure the system, etc.

### Post by strabes on 2007-10-09
Ubuntu doesn't come with any proprietary software (with the exception of a few core drivers) and this is why it does not support proprietary formats out of the box. This includes mp3, avi, playing DVDs, etc. However, they are **very** easy to install. You'll need to add [this repository](http://medibuntu.sos-sts.com/) and install the ubuntu-restricted-extras, w32codecs, and libdvdcss2 packages.

For more information see [this](https://wiki.ubuntu.com/RestrictedFormats) page and [this](http://medibuntu.sos-sts.com/) page.
