### Post by rillip on 2007-06-01
Linux has one of the most powerful firewalls ever built in directly, called iptables.  There are several gui interfaces for it.  I honestly don't use it at home, I do some direct CLI usage at work, so I don't know what the GUI's are called.

As for AV software, it's really neccessary, but there are several good ones.  ClamAV is what I would recommend, but AGV is also highly recommended by some.  I personally never got it to work correctly - it scanned, but would never update.  I wasn't worried about it enough to get it fixed, but I see those two recommended 99% of the time.

### Post by Hendrixski on 2007-06-01
I recommend ClamAV.  I have it set up at work so that it will scan all of my email...that way I won't send viruses to my clients and business partners who still use windows.  Because while Linux can't get viruses it can accidentally forward one to windows.

If you're using this for home use, you absolutely do not need to worry about anti-virus.  There are no viruses that have propagated in Linux, ever.  period.
