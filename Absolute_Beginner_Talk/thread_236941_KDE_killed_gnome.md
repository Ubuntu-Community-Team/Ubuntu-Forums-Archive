### Post by kaffelars on 2006-08-15
Have you tried to reinstall the **ubuntu-desktop**-package? Installing this package will install all the Gnome-packages that are in a default Ubuntu Install.

Oh, and your settings should not be lost, they are stored in hidden folders (such as .gnome2) in your home directory.
