### Post by 5-HT on 2006-03-12
[quote=Ronnie]How do I find out which flavor I am running? [/quote] 
Just as an aside, if you want to know which distro and version you're running there are a couple of commands that can both get the job done:

```
 lsb_release -a 
``` 
```
 cat /etc/issue 
```
