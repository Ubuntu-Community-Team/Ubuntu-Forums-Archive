### Post by jem7v on 2006-10-29
FTP? I haven't had to install an FTP client since I started using Ubuntu. You probably already know this because you seem to know what you're doing, but *just in case* you don't, you can use
Places -> Connect to Server...
to open up a dialogue to connect to several kinds of ftp servers and they'll show up in the left window pane of Nautilus, over where it lists your home folder, file system, floppy drive, and desktop... You can open them like any other folder then. And when you want to git rid of them, just right-click and choose Unmount Volume.
