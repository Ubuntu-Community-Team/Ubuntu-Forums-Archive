### Post by el3ktro on 2005-12-17
[QUOTE=anoncompboy]1) I just recently reformated my HD so I really would not like to do it again. I would like to be able to access my files I already have.[/QUOTE]

It depends on how exactly your HD looks now. If you have a Windows partition with all your data, then you should first install Linux side-by-side with Windows. If you want to share your data between Linux & Windows, the easiest way would be one partition for Linux, one partition for Windows and one large FAT partition for your data.


[QUOTE=anoncompboy]2) I need to be able to play games on it. Will [Steam]("http://www.steampowered.com/") work on Linux?[/QUOTE]

Yes, Steam works, but it's not that easy. Well if you're really into gaming then I suggest you have a dual-boot setup with Windows for gaming, and Linux for everything else. Some major games also have native Linux versions, the most popular ones Unreal, Doom, Quake, AA and a few others. Cedega is a Linux program to make Windows games run within Linux. It often works, but not always.

[QUOTE=anoncompboy]3) Will windows programs not work on linux at all? For example, I have installed OpenOffice in windows, will i need to download the linux version and install that too? Is there some sort of emulator that will emulate and run windows programs on linux?[/QUOTE]

Honestly, if you miss your Windows apps so much, then stick with Windows. If you want to go the Linux way, then find some Linux substitutes for your Windows programs. You can give a list of the programs you're using in Win, and I can tell you some good Linux alternatives. As someone already posted, OpenOffice is THE office application for Linux, and you might also know Firefox & Thunderbird :-) Wine can run some Windows programs under Linux. Cedega is actually Wine+DirectX.


Tom

### Post by el3ktro on 2005-12-23
This SATA thing is a good example of a case where Linux really is superior to Windows. You receive a new Windows version every few years, but you receive a new Linux (=kernel) version every few months, sometimes even it only takes a few weeks. Any new hardware technology that is introduced between Windows' release cycles will not be recognized by Windows itself, you need extra drivers, while every new Linux version catches up with new hardware technology pretty quickly (usually). Try installing Windows XP (the most up-to-date Windows) on a nForce4 mobo - almost nothing will work: no SATA, no network (the worst), no sound, even no USB. You'll have to install several drivers from several CDs, reboot several times, which makes this a really ugly procedure. But when you have a recent Linux distro, all those things will work right out of the box - making it definitely MORE EASY to install for a newbie. Tom
