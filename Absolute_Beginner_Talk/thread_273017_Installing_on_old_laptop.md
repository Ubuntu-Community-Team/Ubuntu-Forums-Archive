### Post by Q4U on 2006-10-07
Hello, 

I am writing to you on a Pentium III 700 MHz with 192 MB memory. It is no speed demon, but it runs without major problems.

Download the ISO image of the install version of Ubuntu, burn it on a CD (make sure to burn it in the correct way - in Linux [http://ubuntuguide.org/wiki/Dapper#How_to_burn_Image_.28ISO.29_files_into_CD.2FDVD](http://ubuntuguide.org/wiki/Dapper#How_to_burn_Image_.28ISO.29_files_into_CD.2FDVD)
 while in Windows - [http://iso.snoekonline.com/iso.htm](http://iso.snoekonline.com/iso.htm))

Everything should be problem-free, unless your lappy is really old and can not boot from the CD (check by accessing the BIOS). If not, then you'll have to make a boot floppy (search the forum). 

Hope this helps. Q
