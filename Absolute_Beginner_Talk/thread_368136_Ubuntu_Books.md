### Post by Mark Gillis on 2007-03-04
I got Beginning Ubuntu and still use it. It is nice for you to start on this bike ride they call using Ubuntu. You will take some falls, but it is not that big a deal if you just keep figuring out what went wrong and are not afraid of rebooting the system. Keir takes some info from other books he has written, but it is still useful because it gives you the basics and a pretty good start on the command line.  I was going to get "Hacks" but I am comfortable with my system and will just wait for an advanced book with the latest version of Ubuntu hits the street.
