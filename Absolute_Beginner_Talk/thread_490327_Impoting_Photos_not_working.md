### Post by Raineer on 2007-07-02
You might try using DigiKam instead, although if it's using the same importing technique the same could happen.

Do you know if your camera is set to share as a HDD or to use PTP (Picture Transfer Protocol)?  If your camera allows you to set it as a USB HDD, then you can just open it like a hard disk and drag the files off it.

DigiKam will support either format, and you can import and delete either way as well. HDD is faster for me than PTP.

### Post by vanadium on 2007-07-02
I have a Canon Powershot which also needs ptp to import pics. It worked perfectly under Ubuntu Edgy, but now under Feisty, I also have issues, though not so severe. In my case, I succeed importing several pictures, then suddenly the importing terminates and the remaining pictures are written as jpg's with length 0.

To work around these issues, you could purchase an USB card reader (not expensive), and use this cardreader instead to import the pictures from the memory card of your camera. This then works just like an external drive.
