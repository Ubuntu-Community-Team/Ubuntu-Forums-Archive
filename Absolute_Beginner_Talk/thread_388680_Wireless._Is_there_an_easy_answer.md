### Post by dstew on 2007-03-19
You could download a debian package under Windows, put it on a partition that can be mounted in Ubuntu, and install it from there.

But we really need to get your wireless working, and we will have to use the terminal. The terminal is very important in using Linux, because it is the one constant behind all the different varieties and distributions.

Enter the command iwconfig and post the output on the forum to get started.
