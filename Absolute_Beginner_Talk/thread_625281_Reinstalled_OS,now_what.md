### Post by jordanmthomas on 2007-11-27
No, you can use all those folders.  For example, if you're not actually USING this as your home directory (it looks like this is just a backup you made, no?), just copy all those files into your home directory that you are using.

Log out and back in, and all your settings will be like they used to be.

For example, I noticed your .vlc folder.  Say you don't have vlc installed (I don't think Ubuntu does by default, right?).  Whenever you do install vlc, your settings will already be there waiting so it will work just like it used to.  If you reinstall kde, kde will already be set up, etc.
