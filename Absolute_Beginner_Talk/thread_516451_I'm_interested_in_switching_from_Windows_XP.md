### Post by Midahed on 2007-08-03
I won't try to address the detail of your questions because I don't know many of the answers.  :)  What I would say is that many gamers run a dual-boot system rather than say bye-bye to their Windows-only games.  For folk like me who are not gamers, but have a couple of important Windows apps that they still need to run, the solution is to run Windows inside Ubuntu using virtualisation software - something like VMware Server or VirtualBox.

If you have enough free disk space you should be able to install Ubuntu and run a dual-boot system.  As far as hardware compatibility is concerned, you don't even have to install Ubuntu to try it out on your PC - just run it from the live CD and see if you like it.
