### Post by madcow72 on 2007-04-24
LaRozza is correct.  It really doesn't matter which one you want (KDE vs Gnome), both are great, but depend on your tastes.  KDE (Kubuntu) will give you a desktop environment similar to the Windows environment.  Gnome (Ubuntu) will give you something slightly different.  Personally, after doing Kubuntu for a while, I also prefer the Gnome desktop, but it's totally based on taste.  

Also, say that you install Ubuntu and want to check out the KDE desktop.  No problem.  Just enter this in a terminal and it will install:
```
sudo aptitude install kde-desktop
```
or vice-versa, you install Kubuntu and want to try Gnome:
```
sudo aptitude install gnome-desktop
```
It's that easy.

[This]("http://www.ubuntu.com/products/whatisubuntu/desktopedition") is a good site explaining the default applications you'll find from the Ubuntu distro.  And [here]("http://kubuntu.org/announcements/7.04-release.php") is a good place to read about the KDE desktop.  And keep in mind that you can almost always install any KDE program on your Gnome system and vice-versa.

You're starting to understand how flexible this whole thing is....

Depends.  Ubuntu seems to do a lot of what I want to do, easiest, but I'm into applications.  

Kubuntu is into more eye candy, more options, more windows and more keystrokes to do the same thing - example, on several KDE systems I've tried, "what is the screen resolution?  What options do I have?"  Easy on Ubuntu, complex on KDE.   I suspect KDE is better for lots of flashy options on the desktop if that's what you're into.

Also, KDE Konqueror doesn't do AOL mail so I have to install Firefox.  But Ubuntu already comes with Firefox.

Neither one will do things like head clean and cartridge change on my Epson Stylus printer.  Also neither one I haven't been able to do ABC News Videos and AOL music videos which are screwed into some proprietary Microsoft code.  Ubuntu & Kubuntu will do BBC videos, CBS videos, Australian ABC videos, NY Times videos, YouTube, all those are flashplayer or realplayer.  I don't have much luck with mplayer.  Totem is fine for my digital camera movie clips.

So it depends on whether you're interested in getting the job done, Ubuntu, or more interested in the tool itself, Kubuntu.  My take anyway.
