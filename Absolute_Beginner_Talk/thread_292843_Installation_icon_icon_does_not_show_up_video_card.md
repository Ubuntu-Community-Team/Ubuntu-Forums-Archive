### Post by bulldog on 2006-11-04
You have to boot from the cd to come to the installer.
Set in your BIOS the option boot from cd as first choice.

When you've done this and boot from cd choose the first option to start the live cd session.

Try this and see if all of your hardware is recognized and working.

If so you can decide to install or not,with the install button on the desktop.

And your nvidia graphics should work well with Ubuntu.
