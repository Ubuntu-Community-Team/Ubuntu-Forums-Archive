### Post by p_quarles on 2007-10-19
Ordinarily, you won't lose any data by making a new partition. There are two precautions to take, however:
1) Back up all crucial data.
2) Defragment your hard drive twice.

For Vista, you should *not* use the Ubuntu partition editor. Instead, use the disk manager in Vista's system tools to create free space on the hard drive. Then, boot the Ubuntu installation disk and install it on the free space. 

You'll be up and running in no time.
