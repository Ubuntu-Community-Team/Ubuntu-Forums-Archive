### Post by apollo2011 on 2005-06-29
You can try WINE, its a program to run Windows apps on Linux.  It is in Alpha...but it works pretty well with Office.  If you add the Universe repository to Synaptic, you should find the wine package.

You could also try Open Office 2, the beta version, which sports better compatibility for Office.  You can get this as well in the Universe repository and it won't interfere with the normal Open Office.
