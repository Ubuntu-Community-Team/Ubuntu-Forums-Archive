### Post by Samurai Penguin on 2008-03-20
Installing IE6 on Ubuntu 7.10

[http://www.alterego7.com/2008/01/internet-explorer-6-on-ubuntu-710-gutsy.html](http://www.alterego7.com/2008/01/internet-explorer-6-on-ubuntu-710-gutsy.html)

Operating System: Ubuntu 7.10 (Gutsy Gibbon)
Wine: 0.9.46 (default repository, no need to modify your sources.list)

1. Get the .deb package from here:

[http://ubuntuforums.org/showpost.php?p=3960527&postcount=5](http://ubuntuforums.org/showpost.php?p=3960527&postcount=5)

it's labeled          ies4linux_2.99.0-1_i386.deb (336.6 KB, 292 views)

2. Double-click the deb package and it will install itself

3. Once that is done, type in Terminal     ie6
    and then press Enter

4. Select what you want to install and proceed. If any other files are
needed you should be prompted - follow the prompts.

This one works and it's easy because I was able to do it. However,
DO NOT use ie6 in Ubuntu except when you absolutely have too
because it doesn't have any security updating
