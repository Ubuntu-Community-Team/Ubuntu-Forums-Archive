### Post by Rocket2DMn on 2008-01-30
There aren't currently any viruses in the wild for linux, and if one were to appear, patches would come out so quickly the virus would be rendered useless.  Most anti-virus clients for linux actually just look for windows viruses - this just prevents you from passing on viruses to windows users. ### Post by mister_pink on 2008-01-30
I feel I should point out something about firestarter as well, since a lot of people are confused about this. Firestarter is not a firewall, iptables is. iptables is installed with ubuntu by default and set to be secure. You only need firestarter if you need to manually customise your firewall settings, for example if you are sharing your internet connection. This is all firestarter does - provide a graphical method to configure the firewall that is already there.
