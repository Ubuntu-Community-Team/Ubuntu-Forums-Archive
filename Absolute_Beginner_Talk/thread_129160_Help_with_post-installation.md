### Post by yota on 2006-02-13
Hi and welcome!

a good start to find resources is:

[http://help.ubuntu.com/](http://help.ubuntu.com/)

more can be found on:

[https://wiki.ubuntu.com/](https://wiki.ubuntu.com/)

Moreover, expecially once you'll get a bit experienced, don't forget that ubuntu is based on Debian, and so many things that apply to Debian are valid for Ubuntu too.
For instance under [http://www.debian.org/doc/](http://www.debian.org/doc/) you can find an howto on APT (advanced packaging tool) under [http://www.debian.org/doc/user-manuals#apt-howto](http://www.debian.org/doc/user-manuals#apt-howto)

The web is full of resources and google is your friend!

A good general linux-user-guide is at [http://www.ibiblio.org/pub/Linux/docs/linux-doc-project/users-guide/](http://www.ibiblio.org/pub/Linux/docs/linux-doc-project/users-guide/)

Ah! don't forget The Linux Documentation Project: [http://www.tldp.org/](http://www.tldp.org/)

and, if you like paper, have a look at [http://eu.wiley.com/WileyCDA/WileyTitle/productCd-0764576445.html](http://eu.wiley.com/WileyCDA/WileyTitle/productCd-0764576445.html)

Hope those help!
