Try a reinstall of xmms

> sudo apt-get purge xmms ; sudo apt-get install xmms

purge will remove the cached .deb file and all xmms configuration settings, so it will be re-downloaded when you do the reinstall. if you do not always have an internet connection, use *remove* instead of purge
