### Post by Shazaam on 2007-10-21
Re your Windows install...
After your hardware change, if you can, boot Windows in safe mode and uninstall ALL of your old motherboard chipset drivers then reboot and see if Windows will rediscover enough of your new hardware to boot. Then install your new motherboard chipset drivers and such. If you had XP or newer you will probably have to authenticate Windows again.
AFAIK Ubuntu should be fine with your new hardware. But as posted previously backup what you have now to be safe.
