### Post by Benjam on 2008-03-06
Today I've gotten some help to fix a problem with my terminal, and it involved being told that terminal is my friend and to use the TAB key.

I had been typing cd/etc/X11 trying to get a process to work.
When I used the TAB key after typing cd /e, and then typing X after that and using the TAB key, I ended up with cd /etc/X11/

My question is, when opening a file/directory in terminal, if it has something inside, do you have to end with a /? Or does it always end with /, no matter what?

---

### Post by taurus on 2008-03-06
You can include the / at the end of you wish.  Either way, it's the same.

```
cd /etc/X11
-same as-
cd /etc/X11/
```

### Post by Origin415 on 2008-03-06
For a *file* you never put a trailing /, but generally it doesnt matter for a directory.
