### Post by tgalati4 on 2007-10-08
If it's a wireless keyboard, then it could be a bluetooth interference problem.  Are there any cell phones, printers, laptops or other devices with bluetooth?

Also, put in fresh batteries in the keyboard and see if that changes its behavior.

Get a wired keyboard and make sure that you don't have a motherboard or USB problem.  If key problems show up with a wired keyboard, then you can be pretty sure you have a bus problem.  If that is the case, go into the BIOS and declock the motherboard.  Run the machine at a lower speed.
