### Post by aysiu on 2006-10-18
> **saintj0n said:**
> Since the printer works in linux, someone had to release a opensource driver for it, right? Linux doesn't allow closed software on its distros from what I was told... That's not exactly true.

Plenty of printers and other hardware work on Linux because the manufacturer produces a closed source Linux driver. Some drivers are closed source and not released by the manufacturer but some Linux developers have managed to reverse-engineer a driver.

And Linux does allow closed source software. Almost everything in the Multiverse repositories is closed source, and many distros (including Linspire, Blag, PCLinuxOS, and Mepis) include proprietary software with the default installation.

Ubuntu does not include proprietary software in the default installation, but it has plenty of closed source in its repositories (Multiverse, as I mentioned earlier).
