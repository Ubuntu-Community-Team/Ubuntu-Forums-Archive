### Post by floke on 2007-01-15
For me the cube is easier to navigate around, and easier to stick documents/windows around than the usual 4-desktop flat-model. With Compiz/Beryl you can, for example, have windows stuck around the corners of a cube, rather than stuck to a flat desktop. This greatly improves functionality, since it makes it far easier to shove windows around. The 'expose' function also makes it far easier to find your open windows too (moving the mouse pointer to top-right of the screen shows you all the windows either on a side of the cube, or the cube as a whole). The 'switcher' function does a similar job as far as functionality goes.

Overall I think its one of those things that you don't necessarily see the benefits from until you start to work on it. I would definitely recommend it if you need improved functionality.

### Post by thewump on 2007-01-15
I installed it for the eyecandy, but keep it installed for the productivity.

CUBE.
Awesome.  I quickly got used to grouping applications on a specific face of the cube.. e.g. email+IM somewhere, browser somewhere else, design tool on a 3rd and music player on the 4th.  I use a standard gnome "panel" on the bottom of the screen made to look like the mac dock, and one great thing is bottom right is always a space of "desktop" so I'm used to dropping my mouse down there and using the scroll wheel to spin the cube.  I also like the fact that each "traditional" desktop has it's own cube, so you can change gnome desktop and have some fun stuff there that you just switch to when it's time for a break.

WINDOW DRESSING
At first I thought all the fade, dream, etc. things ( how your windows appear, act when minimized and maximized etc. ) where just eyecandy, but actually they are useful. For example.. I use the "burn" effect when a window is "closed" and my mind is now used to associating "it's gone" with that rather than something is just minimized. Wobbly windows of course will never be such a thing - but I like it ;-)

BEWARE
Once you are used to it, it's hard to go back.  Windows XP feels as crap to use as it actually is now ;-)
