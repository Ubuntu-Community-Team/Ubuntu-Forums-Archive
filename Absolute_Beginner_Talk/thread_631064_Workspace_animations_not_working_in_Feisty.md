### Post by mikewhatever on 2007-12-04
No such effects are installed by default in Feisty. All there is is wobbly windows. To get what you want, but not all of the effects, install gnome-compiz-manager from synaptic, then look for a new entry under System>Preferences. I found it a little buggy, but nice to try.

### Post by Joeb454 on 2007-12-04
Compiz (the thing with all the fancy effects) isn't installed by default in Fiesty

If you upgrade to Gutsy you'll find there's alot more options available from the start...especially if you install compiz advanced setting manager.

In fact search "Compiz" in Add/Remove in Fiesty (without the quotes) and you should be able to get it and tweak your settings :)
