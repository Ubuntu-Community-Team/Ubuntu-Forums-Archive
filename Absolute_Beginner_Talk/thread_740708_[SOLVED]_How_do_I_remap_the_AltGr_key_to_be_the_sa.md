### Post by potatan on 2008-03-31
Hi,

I can't remember the last time I used the AltGr key for anything. I'm a UK user with a UK keyboard and I don't engage in programming etc.

So.. how can I remap the AltGr key to produce the same results as the Alt key?  This would allow me, for instance, to shutdown or log out one handed while I brush my teeth for instance!  It could also make navigation in Firefox easier (e.g. one-handed alt+left or alt+right for backwards and forwards).

I'm using gutsy gibbon 7.10

I've seen mention of xmodmap and gconf-editor as possible solutions but can't figure it out.

All help appreciated, many thanks.

### Post by chewearn on 2008-03-31
Here in Hardy (there is a same or equivalent dialog in Gutsy):
System > Preferences > Keyboard > Layout > Layout Options

There are many options here you can check out (I'm not sure which one might apply).

---

### Post by potatan on 2008-03-31
Brilliant, thanks - I used the alt/win key behaviour option, and changed it to "Alt and Meta are on the Alt Keys (Default)"

Many thanks.
