### Post by doctorberen on 2006-11-22
I'm currently using Ubuntu 6.10 and I'm loving it. I never thought any of the buttons in my HP laptop will ever work in any Linux distro but I was pleasantly surprised when my SD card reader and volume controls worked.

Before trying Ubuntu 6.10, I was using SuSE. I tried both 9.3 and 10.0. Both were excellent. I also tried 10.1 but gave up because its package manager was seriously buggy.

There are a few things I miss with SuSE though. A lot of people criticize SuSE for taking the kitchen sink approach to an OS but this approach does have a lot of advantages.

Although there seems to be a lot of debate about this, SuSE has its own built-in firewall. It also has a lot of extras like being able to encrypt your files without installing a third party application. It also has the Novell App Armor (although I'm not sure how well it protects).

Also, while the fancy XGL desktop is a feature for the next version of Ubuntu, SuSE has had it for a while. This is same with Virtualization.

I guess people can argue that with Ubuntu, users can do XGL or virtualization by simply installing the necessary packages.

Nothing beats Ubuntu's package manager though and its extensive repositories.

I just wish using the terminal will be minimized. Although I'm comfortable using it, it's something that will prevent the masses from using Linux.
