### Post by Tatty on 2008-04-08
> **StOoZ said:**
> actually , if you want the latest stuff, you usually need to compile it from source.. see pidgin for example, and there are alot more.

True, but that only really applies for the very latest releases of software.  It takes a short time for people to get new code and compile it into a package.  It usually gets done if you are patient...

If you were using a proprietry-oriented operating system where all software usually comes precompiled (naming no particular examples, lol) then you simply wouldn't get access to the latest builds of software.  You would have to wait for a compiled version to become available.

I think its great that the option to get the latest and greatest exists if you want it.

However, personally i tend to just stick to the repos as it is very rare I am after something that cannot be provided by some software available in them.  Or at a stretch look on getdeb.net for a .deb fle.

I have only ever had to compile two pieces of software myself.  One was the driver for my Nova-T 500 TV card.  The other was the latest build of the ScummVM engine, and that was only due to them changing the format of some of the broken sword cutscreens so the one in the repos could not play them.
