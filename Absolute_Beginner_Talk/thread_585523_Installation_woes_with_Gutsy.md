### Post by tnseditor on 2007-10-23
If nothing else works, get the Gutsy iso file and use it to boot into a live CD.  From the live CD you can use a flash drive or CD to back up all your files.  After everything is backed up, install Gutsy over your old installation and make sure you create a separate home partition in the partitioner part of the installation.  By creating a separate home partition your files (in the home folder) will be intact in case you need to reinstall later or even change distributions later.
