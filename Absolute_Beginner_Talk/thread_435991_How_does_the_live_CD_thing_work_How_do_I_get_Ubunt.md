### Post by voided3 on 2007-05-07
Hello. First, go to the download page and select the right version of Ubuntu for your computer (most likely the standard i386 or x86 version). Once the .ISO file has finished downloading, burn the disk image to disk using a program such as Nero or CD Burner XP Pro (make sure to use the "burn cd image" function or you will have an unbootable CD with the ISO file on it unexctracted). Then reboot your computer and boot from the cd; there are a variety of ways to do this and it varies depending on what kind of computer you have. On most desktop computers you just press the "del" key immediately after the computer restarts to access BIOS to select what device the computer should boot from first, but it could also be one of the function keys (F1, F10, F11, F12, etc). If it successfully boots from the disk, you should see the Ubuntu start up screen where you can either wait and let it boot or press enter to skip the wait. After that, it should begin to load your desktop. Have fun!

### Post by teddybairs1 on 2007-05-07
The Desktop Install disk is what's called a livecd installer. What that means is that you can boot the computer directly from the cd without touching the hard disk and run ubuntu without having to install it first. This allows you to test all of your hardware before committing to it. Your XP partition won't be touched in this event. What will happen is you need to download the .iso file appropriate to your machine. Do the Desktop installer, not the Alternative install cd (this is text based and doesn't have livecd capabilities).

Burn the file as a disk image to a disk. It's not the same as just dragging and dropping a file. Burner programs should have a special selection for "Burn ISO File". 

Put the disk in your cd drive and then reboot the machine. You may need to go into the bios to tell the system to boot from cd first. You should come up with the Ubuntu menu options instead of XP. Don't panic, your XP is fine. Nothing has touched your hard drive and it won't unless you tell it to.
