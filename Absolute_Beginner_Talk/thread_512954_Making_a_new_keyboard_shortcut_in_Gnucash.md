### Post by SlithyTove on 2007-07-29
Running Feisty, migrating from Debian.

I'm using GnuCash. It's possible, under Debian, to assign a new keyboard shortcut to an application menu action by simply selecting that menu item with the mouse or keyboard, and typing the shortcut, which then appears on the menu. For example, under Debian I set up Control-B for the GnuCash 'Blank' function. I use this all the time, several times a session.

Doesn't work in Feisty.

I've searched the Ubuntu forums, and the net in general, and can't find a solution, or even an explanation. Same software, same version (2.0.5). Is there a switch somewhere that has to be enabled?

---

### Post by SlithyTove on 2007-12-26
Okay, I found the answer to this one. Yes, there's a switch that has to be enabled.

In Gutsy: System->Preferences->Appearance->Interface, check box 'Enable editable menu shortcut keys'. That does the trick.
