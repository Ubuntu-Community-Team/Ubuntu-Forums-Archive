### Post by dvarsam on 2006-06-21
To get your DVD player to work, all you need is to install the package "totem-xine-firefox-plugin".

Do this, either from "Synaptic" or from the "Terminal", by typing:

```
sudo apt-get install totem-xine-firefox-plugin
```

Good Luck!
