### Post by whizbaby on 2006-09-07
Hit *ctrl-c* while the network 'configures', does it boot through?
If your network connection involves a username and password (or no network is detected) it can't be setup  at the first booting process. When you are able to login you can configure your network and set it to come up at boot time or drop it from boot sequence, whatever you like.
