### Post by Golyadkin on 2007-06-18
Did you try Tools > Clear Private Data and then set it back to "Remember visited pages for 1 day" ?

Also, type about:config in your address bar and check the values of "browser.history_expire_days" and "browser.history_expire_days.mirror"

Also make sure you close down every last instance of Firefox for the changes to take effect. This included the "downloads" window if it is open.
