### Post by jimrz on 2007-09-23
If you want to continue using Gnome then installing Xubuntu will not make your Gnome session blue, only the xfce sessions. Why not just switch to a blue theme? There are several "Clearlooks" "Glossy" "Mist" available in "System" > "Preferences" > "Theme" on the default install and tons more available from places like Gnome Look. No need to make it harder than that.
