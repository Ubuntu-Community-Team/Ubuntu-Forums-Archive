### Post by atraeyu on 2007-03-11
I just got ubuntu installed and running.  I'm using a dell laptop with a wireless card.  

So - when I open the networking utility (through the administration menu) I see three devices:

Wireless Connection (This network interface is not configured)
Wired Connection (Address: DHCP)
Modem Connection (I don't use an internal modem)

So ... Ubuntu automatically configured my wireless card for DHCP and is using it, however the networking tool says it isn't configured.  If I click on "Wireless Connection" then click on properties "Enable this connection" is not checked.  If I check it, then click "OK" and then open it again, "Enable this connection" is not checked again.

So ... even if I set all the correct information to configure my static IP, when I close it and then open it, all the settings are empty again.  Any idea how I can fix this?  :)

---

### Post by atraeyu on 2007-03-11
I just figured it out.  If you don't create a location and set the SSID name of the network you want to configure it for at that location - it won't save the information when you click OK.
