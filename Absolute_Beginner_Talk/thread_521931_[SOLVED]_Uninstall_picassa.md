### Post by RomeReactor on 2007-08-09
Hi. If you installed it through Synaptic (System->Administration->Synaptic Package Manager) you can uninstall it from there; or from the command line:
```
sudo apt-get remove picasa
```

Also make sure you search for **picasa** and not **picassa**; just one "s". Maybe that's why it doesn't show up in Add/Remove.
