### Post by BoneKracker on 2007-04-18
Yes, there is.

I will assume you are using Gnome, and I will assume you are talking about the Window List (which is part of the Gnome Panel).

On the very edge of the Window List, you can see a little series "grab bar" (a little series of of horizontal grooves in a vertical rectangle).  If you right-click on that, and then click "preferences", you can choose options as to how the windows that are displayed in the Window List will be grouped.

The Window List is a "panel app".  All the little things you see on the panel(s) are "panel apps".  You can change preferences on these too.  You can remove the app from the panel.  You can right-click on any item in the menus and add them to the panel.  If you right-click on a blank spot in a panel you can add new apps to it (there are a bunch) and you can add generic things like "launchers" (which will run any command or launch any program), and drawers (which hold multiple launchers).

If you right click on an empty place in a panel you can set preferences for the panel as a whole.  For example, you can make it transparent, change its location or remove it.  If you wanted to only have one panel, you can move the things you need to it and trash the other one.

See the gnome help.

[Edit]: and of course, KDE has similar options.  Other window managers naturally offer complete flexibility in this regard, although the process is a bit more manual.
