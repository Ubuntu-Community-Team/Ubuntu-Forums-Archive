### Post by scxtt on 2006-07-30
yes, subscribe to any thread you've posted to -- you can edit your settings to make it do that by default ...

User CP --> edit options --> Default Thread Subscription Mode

i set mine to "no notification", then anytime there's an update to a thread i've posted in, click on User CP notifies me ... and i can click on "view subscribed threads" to see a listing of them all ...
