### Post by wylfing on 2006-04-22
I think **MHlavsa** misunderstood. The OP is asking about the nForce chipset drivers.

Some -- I repeat *some* -- people have been able to make the sound work better with the nForce drivers. But since they are OSS drivers (not ALSA) there are generally a lot of issues with mixers etc. For almost everyone, the open source sound drivers are just fine.

As for the network driver, there is absolutely no benefit to using nVidia's driver over the open source one that comes with Ubuntu.
