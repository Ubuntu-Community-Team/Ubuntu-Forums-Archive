### Post by sleestak240 on 2006-10-04
Should be at the bottom of your screen on a standard Ubuntu installation.  Right click on one of the window buttons and a context menu appears.

The way I get my transparency (which is virtually the same as in the image you posted) is running the oxygen theme engine.  

Go into the Emerald Themer (Right click Beryl icon in notification area (upper right near date/time) and select Emerald Theme Manager.  Select Edit tab and then set the engine to oxygen.  Then adjust your engine properties however you want, there are all kinds of goodies here to play with.  Default values should match the image pretty closely.
