### Post by qgil on 2005-12-23
When creating new users through System -> Administration -> Users and groups the default setting is that users can't access other users folders. How can I give to certain users access to certain /home folders? *without opening a console*? (very important)

I've tried playing with users, groups and folder permissions via Users and groups and Nautilus, unsuccessfully. Thank you.

---

### Post by qgil on 2005-12-23
Found the solution myself. The problem was that I wasn't ticking the Execute chackbox. I've found somewhere that

> Note that "execute" permission is required in order to read a directory.

And that was. Thanks for reading anyway.  :)
