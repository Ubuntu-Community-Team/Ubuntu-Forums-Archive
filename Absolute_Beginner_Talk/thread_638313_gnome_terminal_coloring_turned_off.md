### Post by asridhar on 2007-12-11
by mistake I turned of the gnome-coloring in my Ubuntu (Gutsy) 7.10 . Now when I open the terminal shell the directories (executables) are not highlighted (they used to be highlighted blue and green respectively). What do I need to change in my profile to get the coloring back.

---

### Post by gary4gar on 2007-12-12
1) Open a terminal via Applications -> Accessories -> Terminal (Gnome) or K-menu -> System -> Konsole (KDE)
2)  choose EDIT>profiles from menu bar
3) choose colors tab & check the check box which says "USE colors from system theme", which is unchecked in screenshot
