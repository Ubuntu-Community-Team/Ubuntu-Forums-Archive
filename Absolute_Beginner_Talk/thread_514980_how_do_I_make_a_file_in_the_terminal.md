### Post by oldos2er on 2007-08-01
Open a terminal, and type "gedit mkdev.sh" without the quote marks.

---

### Post by PurposeOfReason on 2007-08-01
```
sudo gedit /home/YOUR USER NAME/mkdev.sh
```

Of course you can chage the location to anything you want like

```
sudo gedit /homw/user name/.local/mkdev.sh
```

Simply put, to make a file use gedit.
