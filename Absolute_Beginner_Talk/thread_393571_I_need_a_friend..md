### Post by slayerboy on 2007-03-25
Just a word of advice...

I've been using Linux on and off for about 5 years.  Trust me when I tell you, it's never been easier to use Linux, EVER.  Don't worry about what you've heard in the past about Linux.  Using Ubuntu 6.10 Edgy, I haven't compiled anything.  If I did have to make any changes to the setup, I found a lot of help with the ubuntu guide and on these forums.  If I want to install anything, I open up Synaptic, type the name of the program I'm trying to install in the Search box, click to install it, and apply changes.  Most times I don't even have to restart the computer!

Have you installed Ubuntu or any form of Linux yet?  I think you'll find it almost easier than Windows because things make sense in Linux.  If you get an error, you can search for the error message and find out what others did to possibly correct the error.

Have fun with this!  I don't claim to know anything about Linux, frankly because I don't do any coding or developing of software.  Frankly, I don't really have anything to contribute to the community, but I love Linux because it has gotten so much easier to use with all the distros out there.  Ubuntu is one of the easiest distros to use.  If you don't like it, don't worry, you can switch to something else.  I would say if Ubuntu doesn't suit your needs you can try the next best distro, PCLinuxOS.  That was my primary distro for about a year, and Texstar and the gang there do a great job with that distro, I just feel more comfortable and "at home" with Ubuntu.
