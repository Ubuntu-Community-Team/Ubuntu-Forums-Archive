### Post by orkim on 2007-03-14
Since I see that you are an Ubuntu user I will assume that you are familiar with that environment.  There are a different set of default packages that are installed with Edubuntu, which they are all educationally based, but it is very similar.  In fact, they are from the same base and anything you can add/install in Ubuntu you can in Edubuntu.

The only difference?  The packages that come installed by default.  That's it!
