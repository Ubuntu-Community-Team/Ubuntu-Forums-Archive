### Post by mcduck on 2007-03-16
You could try to run 'killall gnome-panel' to restart your panel & menus, and if that doesn't help it means that the person who packaged those games didn't make a menu entry for them. In that case you need to add the entry yourself. (to find out what is the executable file use the 'which' command, for example 'which firefox' would return '/usr/bin/firefox')
