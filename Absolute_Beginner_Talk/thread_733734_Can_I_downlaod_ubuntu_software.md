### Post by ugm6hr on 2008-03-24
Easiest way is to use Synaptic's "Generate Download script", which will create a text script with wget links to all the packages selected, as well as dependencies.

Of course, the Ubuntu computer needs to have updated sources for this to work.

If the Ubuntu computer doesn't have internet at all, then this is useful: [http://nonetdebs.homeip.net/](http://nonetdebs.homeip.net/)
