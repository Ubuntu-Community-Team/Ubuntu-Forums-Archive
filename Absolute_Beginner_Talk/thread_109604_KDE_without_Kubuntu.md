### Post by harisund on 2005-12-29
When you say run KDE, what exactly do you mean? 

If you want to run just KDE apps, you can definitely install them through Synaptic and you will get all the required libraries to run those apps.

However, if you have installed Ubuntu, and want everything that comes with a KDE desktop, you can do
apt-get install kubuntu-desktop 
And you have the freedom / flexibility of both..

### Post by narcolept on 2005-12-29
I know that it was possible before there was a kubuntu, simply be enabling multiverse et al and typing

```
sudo apt-get install kde
```

and it would grab all dependencies, however, ubntu has changed KDE somewhat, so it's not going to be any different than what's installed with kubuntu.  If you mean just adding  kde to use it, running an apt-get install kubuntu-desktop will do the same thing while letting you still use gnome, etc.

So I would recommend just using

```
sudo apt-get install kubuntu-desktop
```
