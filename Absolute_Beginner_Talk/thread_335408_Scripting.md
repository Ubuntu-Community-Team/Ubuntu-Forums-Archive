### Post by Tomosaur on 2007-01-10
Bash scriping basically works by running several different programs as part of the same routine.
Sh is the program which runs shell scripts (it's not necessary if the script is marked as executable, bash will know what to do)
Python is a semi scripting / programming language. It is interpreted, that is - there's a python executable which can read python scripts and work out what to do. However, it is more powerful than shell scripting languages because it can interface with the system and do things which may not be available in bash (which normally just runs other programs which are aready available).
Python is object oriented.
Bash is not.
