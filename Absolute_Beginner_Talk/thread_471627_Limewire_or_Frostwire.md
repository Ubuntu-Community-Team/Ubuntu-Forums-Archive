### Post by NeoLithium on 2007-06-23
I've used both of them on Ubuntu, and I just like frostwire better. Consider it akin to limewire pro, but free and better.  Just ensure that you have java installed, and if you have problems running it, you ensure that you have the Sun-Java Option selected, with this command:

```
sudo update-alternatives --config java
```
Just make sure the little * is next to the java version that has Sun Java :)
