### Post by Trance Gemini on 2007-10-05
Hmm... I suggest you do make a new partiotions for linux first, so you do not have to erase your XP. This will be better, because you are new to linux. That way you can use them both :) To do so, you have to get some hard drive utility that can manage partitions (I liked Partition Magic, but there are others, like Acronis Disc Director), resize your existing partition to make some free space (8Gb will be enough). After that burn the ubuntu CD, reboot, boot from this CD and follow the instructions. There are a lot of guides here how to install ubuntu.

Concerning downloads.. well, if you like to download latest beta of forthcoming Gutsy Gibbon, check the download links [here]("http://www.ubuntu.com/testing/gutsybeta"). You can also read about new features of Gutsy.

Good luck!
