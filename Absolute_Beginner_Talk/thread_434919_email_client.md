### Post by Don Cox on 2007-05-07
Mozilla-thunderbird works perfectly in Feisty to check AOL 9.0 mail. I gave up on 'evolution' for two reasons. First, Help does not work; this is a known bug. Second, there does not appear to be any way to change the outgoing smtp port from the default 25 to 587 which you will require for AOL. M-t has this ability.

AOL Keyword: IMAP provides all the information for settings
