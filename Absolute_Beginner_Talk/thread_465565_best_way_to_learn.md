### Post by Billy McCann on 2007-06-05
You'll want to learn about the core of your system too, and about Linux in general.  The best way is to simply read.  I can't find good Ubuntu documentation, but I've found that the Debian docs are really helpful.  Also, the best way to learn is to just dig in and get your hands dirty.  Set aside a partition with an "experimenting" installation.
