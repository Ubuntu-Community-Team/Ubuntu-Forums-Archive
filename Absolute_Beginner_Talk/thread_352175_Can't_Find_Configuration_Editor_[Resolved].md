In trying to follow some of the tips in that book I tried to activate burn-proof technology for by CD/DVD burner by clicking "Applications>System Tools>Configuration Editor" but I found that menu option to no longer exist.

### Post by mikemn84 on 2007-02-03
The configuration editor you are looking for is hidden, you just need to edit the menu to show it.

Right-click on the menubar (dont have it open) and choose to edit menus.  Then click on System Tools in the left pane then tick the box next to configuration editor.  This will make the item show up in your menu.
