### Post by Mopana on 2007-07-05
click on the network manager icon in the panel in the top left of your screen. Select the name of your network. A new window will open; type the passkey into this window. You need to specify what type of key it is - hexidecimal, ASCII, etc.. If you type the passkey in and it does not work; try again specifying a different key type (eg: try calling it hexidecimal instead of ASCII)
