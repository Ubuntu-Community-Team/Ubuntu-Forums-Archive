### Post by Pobega on 2007-01-18
Compiling source code sometimes gives you faster speeds, so it's usually better to install is manually (Although the boost is barely noticable).

Cons to compiling from source though is that you must meet all of it's dependencies, and everytime you ./configure/make it will check for dependencies. I've tried installing Battle of Wesnoth from source and it took me almost an hour thanks to all of the dependencies I was missing!
