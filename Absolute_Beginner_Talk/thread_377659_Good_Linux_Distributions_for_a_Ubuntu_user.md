### Post by tubasoldier on 2007-03-06
Thats an interesting question. If you like Ubuntu then I reccomend Ubuntu.

But if you are interested in looking how other distributions work then arch linux is pretty sweet.
openSuse, they have some interesting things. Although their configuration tools run slower than a donkey stuck in molasses.
I also really had a lot of fun with Sabayon. The only drawback I saw with it was the whole portage thing. After uninstalling and installing some menial software I was convinced that it took far too long! Portage is a cool idea but I know for sure that it does not take that long to compile software from source.
Slackware was a hoot. It was acutally easier to configure than Ubuntu. IMO. The issue I had with it was the meager amount of available software in the repositories. This isn't fun for someone who just wants to install the software and move on.
Puppy Linux was cool when I had it running on a 120mhz computer with 48mb of ram.
Last but definitely not least - Not a Linux distribution, FreeBSD was cool. It lacks in so many places but I sure learned a lot about Unix from it.
