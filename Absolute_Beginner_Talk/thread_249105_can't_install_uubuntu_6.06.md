### Post by _simon_ on 2006-09-02
I suggest that you defrag your hard drive in windows then again in windows set up another partion for your Ubuntu install and then boot ubuntu CD and try the installation again to the partition you just made.

2. Create a partition (in windows)
3. Boot Ubuntu CD.
4. Install to new partition.
