### Post by kevinlyfellow on 2007-07-26
While I don't recommend not upgrading, sometimes it is the best option for those on slow connections.  Your box will be less secure, but its not necessarily going to catch a virus or anything.  Security updates to linux happen before a problem arises.  If you have a laptop with linux or osx, you can use synaptic to Generate a script for programs you want to update and then run the script at a library or coffee shop to get the packages you need.

DSL uses the 2.4 kernel.  While this is a lower number, it still does get security updates and is maintained.  The 2.4 kernel is still used since it is even more stable than 2.6 and may be the best option for older machines (which is why dsl would use it)

### Post by davidsrsb on 2007-07-26
Different distributions use a variety of releases of the 2.6 kernel. Slackware 12.0 has the latest, unmodified. Ubuntu uses a slightlty older version, with security updates from the newer release backported in. Both are equally secure. You SHOULD update your browser, there are plenty of websites out there trying to use known exploits to break in - usually offering "free wallpaper", ringtones etc
