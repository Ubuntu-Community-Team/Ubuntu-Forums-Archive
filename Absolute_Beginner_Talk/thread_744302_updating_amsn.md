### Post by uberg on 2008-04-03
Run Synaptic and search for TCL.  This should tell you what version you are running.  i am running Gutsy Gibbon right now and it says i am running version tcl 8.4 and that i am running (or would if i had it installed) version 0,97rc1 of amsn.  This seems to be the current version listed on the web site.  Also, every new release a software package has doesn't necessarily show up in Ubuntu repositories as soon as they come out.
