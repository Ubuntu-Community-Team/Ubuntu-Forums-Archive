### Post by pistcivet on 2007-06-24
What I would normally do is create a directory on the volume and take ownership.
(Assuming username=joe, group=joe and we'll call the directory "MyFolder")

```
sudo mkdir /media/hda5/MyFolder/
sudo chown joe:joe /media/hda5/MyFolder
```
You may want (for convenience) to create a sym-link in your own home directory.
```
ln -s /media/hda5/MyFolder/ ~/MyFolder
```
If you really want to allow multiple users read/write permission,
you'll have to create another group, make all the users members of that group,
and change the group owner of the directory you created
(the second "joe" in my example).

If you want user "X" to write to files made by user "Y" - it gets more complicated.
