### Post by Hospadar on 2008-03-13
> do i go into my bios and set the cd as the first boot deviceyes, probably hold down either F2, F10, F12, DEL, or Bakcspace (if you arn't sure, you could always hold them all down=) during boot to get into the bios.

Another thing you might try (other than a re-install) is to test your memory and hard drive.  a lot of times kernel panic is caused by hardware problems.  I really like the [Ultimate Boot CD]("http://www.ultimatebootcd.com/download.html") (pick one of the .iso files and burn it to a cd just like you did with the ubuntu install cd) to do memory tests (you'll want to run "memtest 86+") and hard drive tests (there are a bunch on the cd, just pick the first one, it doesn't really matter).  See if you have any hardware issues.


You may also want to try using a different distro to see if maybe their kernel is build a little different and it makes something work right.
