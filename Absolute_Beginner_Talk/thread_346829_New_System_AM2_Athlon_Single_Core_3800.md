### Post by citizenofnowhere on 2007-01-26
Hi!

I don't see why not, really. The main thing you usually need to worry about is RAM and cpu speed (anything above 128 and above 1.2 respectively). Xubuntu is actually lighter than Ubuntu/Kubuntu, so give it a shot ([http://www.xubuntu.org/get#requirements](http://www.xubuntu.org/get#requirements))

And if you're not sure, download and pop in a xubuntu live-cd. If it boots up and you can open programs without a horrible lag, you're good to go.
