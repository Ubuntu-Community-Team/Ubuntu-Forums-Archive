### Post by Boaslad on 2007-10-14
both Wine and Cedega are aps that allow you to run some Windows programs on Linux. Unfortunately many modern games are too graphically intense to be ran this way. You may need to use something like VMware or Win4Lin to get the performance you are looking for. Good hunting and if you find a good ap that works well, let the rest of us know.
