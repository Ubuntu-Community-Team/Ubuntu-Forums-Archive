### Post by ChrisTheGeek on 2005-12-08
Hi,

First of all, any major OS should run at a decent speed on a machine of that spec so unless you're a real speed junky you're unlikely to notice a massive difference between the distros.

Ubuntu is optimised for usability not raw speed but it is a pretty speedy distro and considerably easier to use than slackware.

Ubuntu is better performing than Windows XP however, it is also about 4 years newer and therefore designed with newer hardware in mind.  You wouldn't expect XP to run optimally on hardware from 1997.  Having said that, my 4 year old 1GHz laptop runs much faster with ubuntu than it did with XP.

As for installing two distros together, you can of course dual boot.  Each distro will have it's own kernel though and would live on it's own partition.

Chris.
