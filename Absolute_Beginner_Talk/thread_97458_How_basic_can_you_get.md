### Post by Brunellus on 2005-12-01
[QUOTE=r4ik]I am a little scared for a command-line only have to learn a lot using that. Might give it a try and learn the hard way. :)Clicked one of the links there is a bit to read out there.[/QUOTE]
you asked for the minimum:  it's always available.  Of course, it means no graphics whatsoever.

Fluxbox/openbox is the best minimal X session out there.  I prefer fluxbox to openbox because of its ability to tab windows together.  

But if you go the *box route, you will need to know how to use the command line a bit, as well as feel comfortable editing config text files.  The *box windowmanagers are generally very lightweight and customizeable, but getting them to exactly how you want may not be what you're used to, if you're coming from GNOME/KDE.

### Post by Kvark on 2005-12-01
The most basic graphical environment must be full screen. No desktop with junk on, no panels with various applets, no menues to navigate to start programs, no windows you can drag around and resize. Only the program you are using in full screen and nothing else.

I use Ratpoison which is just like that, instead of having various widgets to navigate, aim and click on it starts the programs and switches between them with keyboard shortcuts.
