### Post by TheBoat on 2008-02-20
1) There are different versions because about every 6 months changes are made. Ubuntu 7.10 was released in October (10) of 2007 (7), and is called Gusty Gibbon. I recommend using 7.10 (Gusty) because it is the newest stable release. Beryl is completely different. It is a window manager, which adds extra graphical features.

3) For playing windows games, there is an application called wine that allows you to run windows games in linux; however, not every game works. You can check out wine's website here: [http://www.winehq.org/](http://www.winehq.org/)

### Post by seventhc on 2008-02-20
Welcome to Ubuntu :)

1: I use Ubuntu 7.10, when you say ubuntu beryl I think you might be referring to what is now known as compiz. You use compiz to get some extra eye candy, I don't use it myself as imo it doesn't serve any purpose but it is cool. Every so often I'll still mess around with it.

2: It should run fine, if you use the live CD it will boot to the desktop where you can see how your hardware will work on it. (wireless, etc) sort of a test drive.
When using the live cd, obviously it is slower than an installed version since it is running from cd, so don't judge it's speed by the live cd.

3: I don't play any windows only games so not sure.

4:If you are getting rid of windows entirely, then during the install just tell ubuntu to use the entire disk and it will partition everything for you.

Side note. Your laptop should have come with vista cd's or give you an option to make a set. I suggest making them just in case you decide ou want to reinstall it for some strange reason.

Lastly, if you have an problems then ask them here, and most likely will be solved. :)
