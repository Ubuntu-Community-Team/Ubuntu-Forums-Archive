### Post by crispy_420 on 2007-03-30
First you need alsa installed. Then enable the channels, by default only right/left are enabled. This can all be enabled in alsa mixer. Right click the volume control in panel -> open volume control. When new window opens, select edit -> preferences. Now you'll want to enable the channels related to 5.1 sound. So that would be: Front, surround and center. 

Give this a try and report back if it worked or did not.
