### Post by edibledeity on 2007-07-17
Kubuntu.
The K menu is not too big a departure from the start menu and its really easy to customize and make it look however you want.
You can put songs onto an ipod with amarok which just happens to be the best media player around. I'm not so sure if you can do the whole itunes store thing though... I'm sure there's something out there if you want to pay for your music but if not, ktorrent works quite well and azureus is just a click away.
Thunderbird is a quality email program if the family doesn't have email they can access from a browser. 
Use firefox of course, but make sure that they know what konqueror is because it will try to open a website at some point. Also, make sure you install flash and java for them because you have to step into the command line for that.
