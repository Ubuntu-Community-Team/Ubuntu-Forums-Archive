### Post by Rocket2DMn on 2007-07-10
Indeed, be sure to check the md5sum before you start to make sure you don't have a faulty image.
What's that, you might not have heard of this?  Check it out:
[HowTo]("https://help.ubuntu.com/community/HowToMD5SUM")
[Hashes to check against]("https://help.ubuntu.com/community/UbuntuHashes")
Then when you burn, do it at a slow speed, like 4x, this tends to prevent a lot of problems with bad burns.  Use quality media if you can.
Welcome to Ubuntu!
