Yes it is.

If you only want to boot into your new kernel open a terminal and 
```
gksu gedit /boot/grub/menu.lst 
```

Go to the bottom and see which kernel you want to boot into all the time. The first one is 0, the second is 1, etc.

Then go up the page where it says > default		0
and change that to the kernel number you found previously.

When you restart does it do it every time or just sometimes? If just sometimes, Ubuntu is meant to do that to keep your filesystem running good.
