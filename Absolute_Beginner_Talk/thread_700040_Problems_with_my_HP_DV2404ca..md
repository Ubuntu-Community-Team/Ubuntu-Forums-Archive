### Post by mikeyphi on 2008-02-18
Restricted drivers are not legally usably in all countries...thus you have to install them yourself if it's OK in your country!
Unfortunately you need a wired connection first!!
If you can't get a wired connection - ask the specific question about Broadcom drivers again - or search this forum - one of the posters here has a guide for using windows to download the drivers.

The difficulty lies with Ubuntu being 'open' software...not all manufacturers are willing to make their hardware details available so that 'free' drivers  can be issued, and some countries have strict copyright rules!!
