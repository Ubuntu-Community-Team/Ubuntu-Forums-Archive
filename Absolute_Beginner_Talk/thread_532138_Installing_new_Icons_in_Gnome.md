### Post by Dr Small on 2007-08-22
In order to install Icons, I think you need to have the archive, and go to System > Preferences > Theme and "Install Theme" (which really you will be installing the icons) and then edit your theme and choose the icons.

Dr Small

### Post by deadgobby on 2007-08-22
That is very very easy. Go to System>Preferences>Theme>Preferences 
 Click on the install theme, find the file you DL and that is it.
