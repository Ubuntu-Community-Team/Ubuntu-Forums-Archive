### Post by gerscht on 2007-04-27
You don't need to install any drivers. The motherboard has an ATI chipset AFAIK and should be supported out of the box :) .
Installing the apps via wine will most definitely fail, as they won't have access to your hardware directly. If you want hardware monitoring (like temperature etc.), install lm-sensors

Hope this helps,
