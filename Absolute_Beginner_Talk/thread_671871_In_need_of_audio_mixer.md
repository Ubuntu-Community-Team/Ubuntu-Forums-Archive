### Post by nick_h on 2008-01-19
It seems that Gnome does not yet have a graphic equalizer.  You can, however, install applications that have their own equalizer functionality.

Try Exaile, or if you are using KDE, Amorok.

There is a thread on this [here](http://ubuntuforums.org/showthread.php?t=641655).
