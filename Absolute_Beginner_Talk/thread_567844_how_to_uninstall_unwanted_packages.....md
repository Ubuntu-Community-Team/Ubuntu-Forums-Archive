### Post by nowshining on 2007-10-05
do a search for keywords if one comes to mind, also right-click and select either selection with remove next to them, such as remove completely or just remove. When done selecting all u want, and make sure that when you uninstall something to look at what else it will uninstall and if can't then cancel, some things are very picky however be aware that some things may latch on and can't be removed without the whole system going.

Best thing  tho is to backup anything onto external media if need be and do a whole format which will give you what you want. If not then using even synaptic can be a bit dangerous if you don't know what ur doing.
