### Post by nickoli_02 on 2007-02-11
You're going to want to make sure the wireless card you get has supported drivers in linux before you buy. Search the forums, I know there is a list of wireless cards that work well in linux around here somewhere.

If you can find an Intel wireless card, it's a pretty safe bet you can get it working in linux. Intel is pretty good with getting open source drivers to the linux community, from what I know.
