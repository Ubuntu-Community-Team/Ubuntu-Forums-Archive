### Post by parvinder on 2007-09-07
Well if you have downloaded it from the skype website it gives you option to download the Feisty Fawn (7.04) skype to download.

In this case the package installer will start automatically to as soon as the download completes and you don't have to do it. It will also install 2 dependencies.

But if you have the .deb package then type "sudo dpkg -i skype-download-file-name.deb" and it should install or let you know what dependency is required.
