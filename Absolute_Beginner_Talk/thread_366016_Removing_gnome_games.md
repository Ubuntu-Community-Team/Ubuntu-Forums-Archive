### Post by Shatrat on 2007-02-20
ubuntu-desktop is just a meta-package which i sa list of all the default programs.  If you uninstall any of the default gnome desktop programs you will no longer have the full ubuntu-desktop meta-package, but you will still have the packages that are not games.
Not having the full ubuntu-desktop is only a problem when you use the automatic upgrade process, from what I've read.
