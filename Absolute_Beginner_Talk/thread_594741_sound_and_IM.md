### Post by linuxwizard on 2007-10-29
If you are going to use this site
[http://www.emesene.org/trac/wiki/WikiStart](http://www.emesene.org/trac/wiki/WikiStart)
than you want tar.gz (stable, r806, 21 aug 2007) > save to your desktop > right click on it > extract here > now a folder will be there. Right click on download > move to trash. Using this method does not install it that folder stays on your desktop. It will not show up in your Application menu. Click on folder to open it in there is a file emesene > click on it a window will come up asking you to run in terminal or run > choose run > Emesene will open put in your account info




If you are going this way.
[http://www.getdeb.net/search.php?search_distro_id=5&keywords=emesene](http://www.getdeb.net/search.php?search_distro_id=5&keywords=emesene)
You want to use the first one >emesene svn20070927 > click on download > save to your desktop > double click on download > will install for you
