### Post by swisscow on 2007-09-12
Depends...

Reinstall can be a quick fix if you're happy to go back to scratch, i.e. haven't installed loads of stuff, made changes to set up etc.

Alternatively fixing the problems can save the agro if you have made lots of changes etc.

One possibility if you reinstall is to set up a seperate home partition so if everything stuffs up again you can just install the system and leave that home partition alone (save those dvds)

Personally I have reinstalled loads of times until I felt that I had got it sussed................but I'll probably have to do it again in the future. Though there is something nice about a shiny new install.
