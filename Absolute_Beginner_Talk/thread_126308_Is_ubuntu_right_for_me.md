### Post by bonzodog on 2006-02-06
Doom works in Linux, as does quake and UT. We also have Open TTD, Lincity-NG (a simcity like game). Firefox is the standard browser, and it has an internet client for everything. but understand that windows programs do not run as easily in linux due to linux being a completely different OS. It will require a learning curve, and you will need to learn a little bit of the terminal at first to install software. Software installation is completely different from windows, and often requires that you open a terminal and type in an apt-get install command.
OpenOffice will open all MS apps including Powerpoint presentations.

### Post by m.musashi on 2006-02-06
[QUOTE=dave154]is Ubuntu or Linux in general right for the average high school, Microsoft office dependent, internet chatting, web surfing, game playing teen?[/QUOTE]

I'm no expert but I work in a HS and we are starting to experiment with Linux (Ubuntu actually). I think that basic users who just want to surf the web, write papers, and IM will be fine with Ubuntu. In fact, I've read some discussion about whether someone who is computer illiterate will find Linux or Windows more difficult. Some suggest they will find Linux to be easier. Perhaps, I can't really say. However, since most students are using Windows at school that may be a moot point.

Games, however, could pose a problem. I've read other threads (sorry, I don't have a link) where users discussed the difficulty of game play in Linux - especially when they are Windows native games. Emulators like Wine and Cedega may help but in the end I think programs run best on the OS they were designed for.

In short, it will probably come down to your users. Some will probably be fine with Linux but others may require Windows or simply be unwilling to try something new.
