### Post by Phlosten on 2006-10-24
Other users are experiencing the same issues with not being able to see the splash screen on bootup and there are bug reports about it.

You could try the suggestion from this bug report [https://launchpad.net/distros/ubuntu/+source/usplash/+bug/65912](https://launchpad.net/distros/ubuntu/+source/usplash/+bug/65912)

If you customised your boot splash screen then it was most likely replaced through the upgrade process with the normal 6.10 version.

Keep in mind that Ubuntu 6.10 is still in development. If you are a deinfate beginner I would advise waiting for an official release before installing it.
