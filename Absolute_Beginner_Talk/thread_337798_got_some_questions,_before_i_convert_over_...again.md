> So VMWare server and VMWare are both pretty much the same thing?  You can run Windows on it?

VMWare Server is one of the products that are freely available from [VMWare](http://www.vmware.com/products/free_virtualization.html).
Yes you can run Windows on it. Simply put it works as follows:
1. You create a VM
2. You install the OS on it (either Windows or a Linux distro), so you need installation software
3. You run the VM.
