### Post by prosacpils666 on 2007-06-11
I have a dell 1501, You can try 2 things
at boot, when you can choose to install ubuntu in text mode or mem test.. etc. Press F6 and type in the command line

pci=nomsi

then press enter and ubuntu should see your hard drive.

or

You can download ubuntu Feisty fawn, this doesn't need any parameter at boot to recognize the sata disc.
