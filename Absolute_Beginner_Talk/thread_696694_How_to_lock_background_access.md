### Post by rkd on 2008-02-14
If you have your screen saver set up so that it requires your password when you wake up the computer, that will help prevent tampering. You can turn that feature on by going to the System menu, then Preferences, then Screensaver. Down near the bottom of the dialog, make sure you have checked the box beside "Lock screen when screensaver is active".

If you don't want a screen saver, you can leave it at the default black selection.

To prevent a prankster from jumping in before the screensaver activates, get in the habit of typing Ctrl-Alt-L when you step away from your computer. That activates the lock immediately.
