### Post by sancho panza on 2007-10-14
For most of the apps needing low-medium processing power (assuming u have a fairly recent PC), Wine or virtual box should be able to run those in linux. However, for most of the windows based games that u have, dual booting windows with linux would be the best bet. You'd probably get better feedback [here]("http://ubuntuforums.org/forumdisplay.php?f=93").
Some of the apps (not games) would have comparable linux alternatives, and you should be able to get your jobs done with those.
