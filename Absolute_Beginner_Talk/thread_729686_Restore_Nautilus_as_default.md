### Post by Captainm on 2008-03-20
Did you just install thunar or did you change it to the default browser? If so how? You could probably change the commands in /usr/share/applications to the appropriate ones for nautilus.

[EDIT] You'll have to change File Browser, Home Folder, Open Folder, Network Servers, and Computer. It's a bit of a hassle. I think someone on this forum wrote a script to change the default filemanager so I'd search the forum first.
