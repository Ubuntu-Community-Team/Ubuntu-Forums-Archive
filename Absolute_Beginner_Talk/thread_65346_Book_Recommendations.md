"Moving to Linux: Kiss the Blue Screen of Death Goodbye!"
Marcel Gagne
ISBN 0-321-15998-5

This book is a very friendly intro to using various GNU/Linux systems.  It is reminicent of a user manual that you probably received with any other personal computer you may have ever purchased.

"The Linux Cookbook"
Carla Schroder
ISBN 0-596-00640-3

This book has all sorts of practical information on how to accomplish different tasks in a GNU/Linux environment.  Comes with a nice command cheat sheet.
