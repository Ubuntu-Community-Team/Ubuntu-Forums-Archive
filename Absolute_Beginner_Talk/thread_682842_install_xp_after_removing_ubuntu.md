### Post by bumanie on 2008-01-30
Boot with windows cd and ensure that you select to format the entire drive (don't choose 'quick format', do the long format). This in theory should get rid of all traces of grub (and ubuntu in general). If there is any remnants of grub left and the install/booting doesn't work use something like boot and nuke to format the hard drive again and then reinstall with your xp disc.
