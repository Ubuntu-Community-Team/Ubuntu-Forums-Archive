The speed issue in Linux is dependent on Desktop Environment and Kernel, amongst other things.  

For example, DSL - it runs with Fluxbox and Kernel 2.4, and will be ridiculously fast.  But using wifi with it will require manual editing of settings, and, as you say, Flux can be unfamiliar for Windows users.

If Xubuntu 6.06 doesn't satisfy you in speed terms, consider using an IceWM-based distro (e.g. Puppy Linux - which has great wifi support), or use IceWM with Ubuntu minimal install ([http://www.psychocats.net/ubuntu/minimal#barebones](http://www.psychocats.net/ubuntu/minimal#barebones) or just install IceWM on top of Xubuntu).  As for wifi in Xubuntu / IceWM-ubuntu - you could try Wicd (see my signature), although your computer might not have the processor power for it.
