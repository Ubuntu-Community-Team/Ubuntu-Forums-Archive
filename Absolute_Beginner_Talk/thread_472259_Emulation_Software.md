### Post by EndPerform on 2007-06-12
VMWare has a free player and server component.  In order to run Windows, you'd need VMWare server, then you'd need to install Windows using it.  VMWare isn't an emulator, it's a virtual machine.  Wine is more of a compatibility layer than an emulator, but it runs some Windows programs.

### Post by starcraft.man on 2007-06-12
If your looking to make a Virtual Machine. I recommend [Virtual Box.]("http://www.virtualbox.org/") It is very well made and completely free (even part open source, bonus :D). It is far superior to the vmware player (it is based on qemu I think, i like it better I will try qemu more extensively later) and will allow for customization of many things, including shared folders, bidirection clip board (copying into and out of VM) and a few other things. It is a very well written solution IMO and I've been using VMware workstation for a few years. Give it a whirl. You will of course have to install the OS you want into it like any other.

Oh and you should try OpenOffice impress, its a decent product. You may really like it and not even need a VM.
