### Post by psusi on 2006-02-20
If the package has them in its name it means it was pulled directly from the SVN or CVS repository rather than an official numbered release.  In other words, it is bleeding edge stuff.

### Post by knalle on 2006-02-20
SVN and CVS are revision systems for source code, the postfix tells you that the source was maintained by SVN or CVS. if you download binary precompiled packages this doesnt apply for you, just ignore it.

if you download source and just compile, this doesnt apply to you, just ignore it.

if you download source, compile and commit changes back to the source tree this applies for you and you should start reading about both.

happy hacking

### Post by JShadow on 2006-02-20
SVN and CVS are revision control methods. They're used to track changes in source code.

Downloading one of these versions means it is the latest developement version. It is most likely newer than the release in Ubuntu's repository, but you are risking stability for features.
