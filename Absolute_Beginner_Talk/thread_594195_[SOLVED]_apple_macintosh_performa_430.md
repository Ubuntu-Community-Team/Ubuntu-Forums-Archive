### Post by reidster on 2007-10-28
The Performa 430 isn't a PowerPC machine, it uses the 68030 processor.  There are very few Linux distros that support the 68k architecture and Ubuntu is not one of them.  If you are bent on installing Linux, there is a 68k port of Debian.  Having installed it on a newer machine (Quadra 840AV) I wouldn't recommend it as a particularly simple or rewarding project.  BSD is also available, but I would suggest getting Macintosh System 7.5.3 from Apple (a free download now)

[http://download.info.apple.com/Apple_Support_Area/Apple_Software_Updates/English-North_American/Macintosh/System/Older_System/System_7.5_Version_7.5.3/]("http://download.info.apple.com/Apple_Support_Area/Apple_Software_Updates/English-North_American/Macintosh/System/Older_System/System_7.5_Version_7.5.3/")

If you are interested in running Linux, check out [http://www.linux-m68k.org/]("http://www.linux-m68k.org/")
