### Post by aysiu on 2006-05-17
You can find them yourself if you go to System > Preferences > Keyboard Shortcuts

You can *define* them yourself if you go to Applications > System Tools > Configuration Editor > Apps > Metacity > Global Keybindings/Keybinding Commands

Of course, if you're using KDE, it's a different story.

### Post by Wolki on 2006-05-17
Here's some info about default keyboard shortcuts: [http://gnomesupport.org/wiki/index.php/KeyboardShortcuts](http://gnomesupport.org/wiki/index.php/KeyboardShortcuts)

Many keyboard shortcuts for applications are defined in the Gnome HIG. Too lazy to look up the sections right now, but you should be able to find it easily.

If you want to (re)define global shortcuts, System -> Preferences -> Keyboard Shortcuts. You can not change what the shortcuts do here, so music player shortcuts will only apply to rhythmbox.
If you want to (re)define application level shortcuts, System -> Preferences -> Menus & Toolbars -> check Editable Keyboard Shortcuts. You can then press a shortcut while hovering over a menu item, the shortcut will then apply to that function. I reccomend unchecking Editable Keyboard Shortcuts afterwards, to prevent changing them by mistake.

Lastly, you can create global shortcuts for any command with gconf-editor (which is rather complex) or by using Gnome Advanced Preferences ([http://yimports.com/~cpinto/?cat=60](http://yimports.com/~cpinto/?cat=60) ) which is a lot easier but only has support for 12 additional shortcuts. Combined with some commandline switches and tools, this allows for rather powerful keyboard shortcuts.

If you have questions, feel free to ask.
