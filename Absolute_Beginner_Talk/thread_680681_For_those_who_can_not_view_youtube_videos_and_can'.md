I am relatively new to linux and for a while i've been searching the internet for how to view youtube videos in gutsy.......but i couldnt get the appropriate answers.....though there were many,none worked for me....but now i can easily view youtube videos...
    If you have wine installed,like i have,just install firefox web browser(.exe) in wine. Now go to the abobe web site and download the latest flash player(.exe) and install it.Now go to applications>wine>programs>mozilla firefox......now you can easily view youtube videos with sound...just like i did..
     OH yeah....if you wanna copy the videos go to applications>wine>browse c:\ drive>windows>temp>fla.tmp
