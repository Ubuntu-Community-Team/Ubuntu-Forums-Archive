### Post by firestormx37 on 2008-02-26
You need to add the user that will be running VirtualBox (in this case, the account you log in with) to the vboxusers group.  To do this.......
System > Administration > Users and Groups
Click on Manage Groups
Scroll all the way to the bottom and find vboxusers
Select it and click on properties
Mark the check box next to your account and click OK
