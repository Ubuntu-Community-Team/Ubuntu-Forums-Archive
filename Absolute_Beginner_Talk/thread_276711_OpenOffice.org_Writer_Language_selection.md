### Post by TheWizzard on 2006-10-13
ok, here's something that works much better:
- make a table with 2 columns. doesn't matter how large the table is. don't worry about the lines. 
- go with the pointer to the top of the left column. you'll see a downward arrow.click left and you'll see the column selected. 
- go to format -> character and select "English (UK)"
- repeat the above for the right column and select "Thai" (you have the dictionary installed?) 

if you are in the lower right cell and you press "tab", you'll get a new row with the desired languages. 

later you can remove the borders. 


alternative: make a macro and assign a shortcut key. 

does this help?
