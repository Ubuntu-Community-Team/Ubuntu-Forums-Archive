### Post by meng on 2006-11-22
What you need is samba. Samba configuration is not what I consider straightforward, but then again I have never found Windows networking to be straightforward either. There is a way to configure it through GNOME, I think it's System > Admin > Shares (network shares? shared folders?) although that method didn't work for me and I had to edit smb.conf directly. There are excellent resources on the web.
EDIT: Whoops, overlooked that you were using kubuntu. Control center is the place to look - networking/shares.
