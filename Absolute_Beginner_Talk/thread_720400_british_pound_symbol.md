### Post by solitaire on 2008-03-10
It sounds like your keyboard is not set to the correct region.
Go to "System", "Preferences", "Keyboard" you can then change what region / layout you want to use.

Unless you are in a different region than the UK then you can use "applications", "Accessories" "Character map" then it's under "common"

### Post by red_Marvin on 2008-03-10
For me, with a Swedish keyboard layout it is placed at Alt Gr + 3.
You should also be able to insert it by, while holding ctrl + shift, typing "a3" without quotes, regardless of your language settings.

If you want to paste it from the char map (Main menu - Program - accessories)
you can search for it (ctrl + f) and enter "pound". When it finds it, double click on the character and press the copy button and then paste where you want it.

### Post by Vadi on 2008-03-10
