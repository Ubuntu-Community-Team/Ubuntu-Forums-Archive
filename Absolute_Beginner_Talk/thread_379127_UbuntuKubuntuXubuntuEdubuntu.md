### Post by Bachstelze on 2007-03-08
Just to correct an important but common mistake, Gnome, KDE and friends are *desktop environments*, not window managers. The WM is the part of the system responsible - as it's name says - of managing windows - drawing them, making them interact with the user, etc. While Gnome and KDE each have their own WM by default - Gnome's is Metacity and KDE's is KWin - you can use others if you want, for example Beryl or Compiz, without changing your whole DE.

### Post by HereInOz on 2007-03-08
It is a matter of how they look and how you can do various things in there.  Underneath, they are all Ubuntu Linux.

The big difference is the desktops that they use.  Unlike Windows, where you get the Windows desktop and that is it, Ubuntu allows us to use various desktops that offer different capabilities and appearance.

Ubuntu uses the Gnome desktop, which provides a fairly neat interface with good performance and functionality.

Kubuntu uses the KDE desktop, which does offer some very nice stuff that Gnome does not offer.  It is also a little more resource hungry and doesn't run all that well on lower end machines.  It is a more flashy desktop which gives you a different set of default applications and tools than does Gnome.

You can still install KDE applications on a Gnome desktop though - just use Synaptic.

Xubuntu uses yet another desktop, Xfce, and it is specifically designed to provide a functional desktop that will give reasonable performance on a lower end machine.

Edubuntu is specifically geared to educational pursuits, so if you were setting up a machine in a primary school, to teach maths, language and that sort of stuff, it would be the one you would choose.  I am not all that familiar with it, but I think it runs the Gnome desktop (could be wrong) with heavy emphasis on educational applications.  It also contains some KDE applications, particularly the games.

Bear in mind though, that all the applications in Edubuntu are (I think) available to be downloaded and installed even if you are using the normal Ubuntu Gnome desktop.

Its a bit like four flavours of ice cream.  Still the same stuff, but the flavour is different, and you can mix and match together if you want.
