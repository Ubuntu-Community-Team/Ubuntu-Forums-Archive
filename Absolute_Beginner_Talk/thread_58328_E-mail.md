### Post by aysiu on 2005-08-19
First, if you don't use it already, install Thunderbird for Windows. It should give you the option to import Outlook settings and emails.

This will show you where your profile folders exist for Thunderbird:

[http://www.mozilla.org/support/thunderbird/profile#move](http://www.mozilla.org/support/thunderbird/profile#move)

Then copy the profile from Windows to the profile in Linux. I've done it. It works.
