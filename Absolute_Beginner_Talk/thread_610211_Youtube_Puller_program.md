### Post by FuturePilot on 2007-11-11
There's a number of Firefox extensions that will let you save the video to your computer. There's also a program in the repos called youtube-dl. It's very easy to use. All you do is open a terminal and type in
```
youtube-dl URL
```
Replace URL with the URL of the page with the video you want.
```
sudo apt-get install youtube-dl
```
