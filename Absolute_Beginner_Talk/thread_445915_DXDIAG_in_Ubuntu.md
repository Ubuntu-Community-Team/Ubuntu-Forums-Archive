### Post by andoty_jazz on 2008-01-31
you can also use the command **lspci**

If you want a comprehensive one with GUI. Take a look at you **Applications -> System Tools -> Sysinfo**

And if not installed, use this command  **sudo apt-get install sysinfo**. Alternatively, you can use **Synaptic Package Manager** and search for **sysinfo**.

Now you are good to go.
