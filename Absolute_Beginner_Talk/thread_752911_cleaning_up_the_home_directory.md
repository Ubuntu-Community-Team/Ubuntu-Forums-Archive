### Post by wormser on 2008-04-12
Never touch the .bash files.  That would be bad.  If you remove a program then it would be fine to delete the .file associated with it.  Personally I would just learn to live with it and not delete any of them.  IMHO, Ubuntu should put all the config files in a folder like this /home/username/.config/.  It would keep your home directory much cleaner.
