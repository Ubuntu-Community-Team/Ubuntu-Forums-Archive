### Post by MoxJet on 2007-05-11
What? It runs Windows XP? That's insane.

You will have to use the alternative CD to install ubuntu on it - but it won't work to start Gnome or KDE, probably not either Xfce. You might want to try to install a server and then aptitude a lightweight windows manager, like Fluxbox or IceWM... Another good try would be  DamnSmallLinux, [http://www.damnsmalllinux.org/](http://www.damnsmalllinux.org/) good luck.

Two more things:
DSL has a Live CD,
and from my experience, finding ram sticks to get from 64mb to the double or more isn't too difficult... look at a computer dump, second hand stores etc...
