### Post by nikosapi on 2007-05-22
This section:
```
Section "Device"
	Identifier	"Creative Geforce 256 32MB DDR"
	Driver		"vesa"
	BusID		"PCI:1:0:0"
	Option		"UseFBDev"		"true"
EndSection
```

The Driver line should look like this: 
```
Driver		"nvidia"
```
For the binary driver and like this:
```
Driver		"nv"
```
For the open source driver.

Vesa is the default "it'll work with any card" driver but it offers very little in terms of preformance.
