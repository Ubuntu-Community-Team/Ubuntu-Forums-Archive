### Post by angry_johnnie on 2008-04-10
You can still see it because it's still available.  If you don't want to see it, you could go to preferences (in mint update) and uncheck the box.  If I remember correctly, mint uses 5 levels, right?  So, suppose the update you don't want is a level 3.  Just uncheck the level 3 updates.  Refresh the list, and it should be gone.

### Post by angry_johnnie on 2008-04-11
Ok, so you already went back to 1.3.3? 
If you uncheck level 3 updates, you won't be allowing **any** level 3 updates. 
So, what you could do instead, is go to synaptic.
Look for the application that you don't want to update.
Then, in the menu bar, go to **Package**.
Choose **Lock Version**.
Then it won't be updated.
