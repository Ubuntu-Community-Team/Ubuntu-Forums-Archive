### Post by catlett on 2006-08-14
[http://www.albinoblacksheep.com/flash/numa.php](http://www.albinoblacksheep.com/flash/numa.php) If that is what you mean, you don't want to watch that.:D 

But if you really want to, add extra repositories [https://help.ubuntu.com/community/Repositories/Ubuntu](https://help.ubuntu.com/community/Repositories/Ubuntu)

After you add the repositories issue these commands in the terminal 

```
sudo apt-get update
```
```
sudo apt-get install flashplugin-nonfree 
```
```
sudo update-flashplugin
```
