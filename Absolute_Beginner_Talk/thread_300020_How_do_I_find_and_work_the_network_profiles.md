### Post by davmac on 2006-11-15
Have a look at System -> Administration -> Networking. Set this up with what you want for the 1st location, then click the save icon and give it an appropriate location name. Do the same for other locations.

To switch between them, just go back to networking, select the location from the drop down box and click the "Green tick mark" icon.

Hope this helps,

Dave Mac
