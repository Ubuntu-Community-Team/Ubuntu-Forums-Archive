Easiest is getting the Hoary Addon CD. [http://www.mrbass.org/linux/ubuntu/](http://www.mrbass.org/linux/ubuntu/)
It contains a lot of often used software. For other things it's a little more work-intensive, since you'll have to track dependencies yourself. Go to [http://packages.ubuntu.com](http://packages.ubuntu.com), download the file for your version of Ubuntu, search for which dependencies it need that you don't already have, and then get these dependencies in the same way (i.e. checking for which dependencies they have...).
