### Post by u.b.u.n.t.u on 2007-04-25
> **cotiK said:**
> should i?

Ubuntu will automatically scan hardware and install drivers for such. Therefore if you want specific hardware to work when you start Ubuntu the first time, make sure such are connected.

Speakers don't need to be on. Ubuntu will detect the audio chip on the card or mainboard. Have the printer switched on, just the power on.

A mic and an external hard drive don't need drivers installed at first install. Ubuntu picks them up as they are plugged in.

All the best.
