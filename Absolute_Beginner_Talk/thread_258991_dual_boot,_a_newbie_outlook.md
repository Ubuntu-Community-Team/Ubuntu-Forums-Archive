### Post by 3rdalbum on 2006-09-17
You know what? You're absolutely right. People have this idea that computers should only need to have one operating system on them. Many people who want to try Linux erase their Windows partition because they've never heard of "dual-boot".

No operating system is ideal for all purposes, so we should be dual-booting or running one or two inside virtual machines. Here's my case:

1. Mac OS 9 is great for music production and typesetting.
2. Ubuntu is great for general-purpose tasks
3. Windows is great for its large number of games, and because it has DVD authoring software and (3rd-party) support for my video digitiser.

I have two computers and three OSs, and I use them in conjunction with eachother. For example, I can record audio on my Mac, rip existing DVD footage with Ubuntu, and put it onto a DVD with Windows. I can't satisfactorily do all those steps on any one operating system.

Right now I've begun work on a set of tools to make Ubuntu more willing to work alongside Windows, and make the Ubuntu experience a little more feature-filled. I think this is an important thing to do, to change the way people use computers.

One size never fits all!
