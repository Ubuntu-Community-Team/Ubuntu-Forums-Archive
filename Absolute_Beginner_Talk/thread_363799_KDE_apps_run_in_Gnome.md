### Post by Velotix on 2007-02-17
I will warn you though, you may have minor issues but it won't stop the program from running. Example: in GNOME it takes my very fast computer about 30 seconds to load Amarok the first time - in KDE it takes about 3 seconds.

The issue is all down to the fact that the system has to load the KDE library files in when you first run the program because GNOME doesn't normally use them. There's probably a way to have them preloaded on GNOME startup too but I don't know how you'd go about that.
