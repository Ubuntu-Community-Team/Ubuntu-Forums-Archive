### Post by kebes on 2007-02-02
You can search wine application database for tax software:
[http://appdb.winehq.org/search.php?sSearchQuery=tax](http://appdb.winehq.org/search.php?sSearchQuery=tax)

It looks like TurboTax 2005 was working with wine properly:
[http://appdb.winehq.org/appview.php?iVersionId=3831](http://appdb.winehq.org/appview.php?iVersionId=3831)

And TaxCut too:
[http://appdb.winehq.org/appview.php?iAppId=641](http://appdb.winehq.org/appview.php?iAppId=641)


Unfortunately these are older versions of the software running on older versions of wine. So there are no guarantees for newer versions. But it's worth a try if you have the software at hand.
