### Post by bford16 on 2007-09-11
> **KevinM1 said:**
> I'm playing around with different workspaces with my Feisty box, but the panels keep disappearing in all but the 1st/default workspace.  Is there any setting I can select that displays panels in all workspaces?  Or is this a bug?

You should have the same panels in each workspace (one above, one below).  You can configure the bottom panel to only show windows in the active workspace, by right-click, Configure panel.

If you remove a panel from one workspace, it is removed from all.

Have you turned on Desktop Effects, or are you using Beryl, by any chance?
