### Post by maruchan on 2006-09-08
> 1. Am I correct in assuming that when I install Ubuntu from the live CD I got from the site, it will wipe out the tons of MS Word files, Excel files and Wave files that I have stored on my C-drive?

You have the option of installing Ubuntu to any hard disk, or hard disk partition, that you want. That said, if you only have one hard disk and one partition, and you don't want to do any partitioning or install another hard disk, then yes, the install will wipe everything. 

Good question, but it's kind of like asking, "if I unplug my refrigerator to plug my toaster in, will everything in my refridgerator rot?" Yes, but many people just find another outlet for the fridge ;)

> 2. If the answer to #1 is yes, then my next question is, if I back thes files up to a USB drive (or one of those cool USB memory sticks), will the office programs in my new Ubuntu package be able to easily read them? (i.e. is format compatible, etc...)

Should be compatible, but you are responsible for those files, so you might want to give them a try in Ubuntu first (open from the Live CD). Once you install Ubuntu you have many options for running the original software that you created those files with, too, i.e., Crossover Office.

> 3. On a previous Linux live CD that I tried (i.e. not Ubuntu), I could not establish an internet connection with the browser it contained. When I install Ubuntu, will Firefox be able to "just connect" or will I have to deal with a bunch of settings and whatnot? (I have comcast cable internet.)

I doubt you will need to do any configuring if you have cable internet. Try the Ubuntu LiveCD's network connection first, though.

> 4. The previous non-Ubuntu live CD claimed that it supported Wi-Fi. But when it was scanning for signals it said it found none. But I KNOW I have one. I think maybe it just couldn't see my hardware, and I didn't know how to fid the problem. How will I deal with this with Ubuntu.

If it *does* happen in Ubuntu, check the wiki/documentation for help with wireless networks. You may want to make sure your wireless gear is compatible *pre*-install though.
