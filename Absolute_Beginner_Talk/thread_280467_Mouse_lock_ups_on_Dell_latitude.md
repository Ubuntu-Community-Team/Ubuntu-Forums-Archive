### Post by spamcatcher on 2006-10-28
Just trying to fix similar problem on my Dell Latitude C600. Some googling seemed to suggest the problem may lie with power saving features, specifically CPU speed scaling. Opening up KLapTop, I selected the 'performance option under Performance profile, and it seems to have solved my problem with the jittery, jumpy mouse cursor. Still not 100% sure that it's a permanent fix, as I've only made the changes 1/2 hour ago, but it's looking good!
