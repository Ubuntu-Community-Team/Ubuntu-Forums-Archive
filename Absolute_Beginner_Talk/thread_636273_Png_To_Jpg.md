### Post by mcduck on 2007-12-09
Wallpapers can be in almost any format, jpg, png, svg, bmp and so on.

Compiz however only likes png. Easiest way to convert images is to just open them with the default image viewer, then click File/Save As and now you can save the image as png.

Anyway, Compiz also requires that image dimensions are powers of 2, so for example 512x512, 1024x1024 and 2048x1024 all work, but 1000x1000 doesn't. The maximum sizes depend on your graphics card so you'll just have to try what works for you. BEcause of this limit in file sizes you'll probably need to resize the image with Gimp (Applications/Graphics/GIMP Image Editor) and then save it from there as .png file.
