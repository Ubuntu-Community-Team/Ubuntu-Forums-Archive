### Post by stchman on 2008-01-10
> **fester225 said:**
> I'm running Gutsy (7.10) under KDE and should start backing up my files.
What CD/DVD Burner should I use?

If you mean what program I agree with the 2nd poster with K3B.  If you mean what physical burner, nearly all CD/DVD drives are supported by Ubuntu.  TO properly install K3B type this at the command line:

```

sudo apt-get -y install k3b libk3b2-mp3
```

This will install K3B and the MP3 plugin.
