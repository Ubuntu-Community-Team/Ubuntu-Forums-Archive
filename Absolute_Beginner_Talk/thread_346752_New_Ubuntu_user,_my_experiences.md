### Post by Mobby on 2007-01-26
Hi,
  First I would like to say well done to the Ubuntu team as I must say I am very impressed with the progress they have made in making a Linux which is stable and makes a good impression on the new user (minor or no linux experience and trying to make the transition from windows... i.e me :D ). Also Hi! to the community because one of the strongest things about Linux is the support it has from the community and people willing to spend their time helping others in the world of Linux.

Anyway my experiences of Ubuntu...
Although I am posting in the Absolute Beginners Talk forum I have ventured into the world of Linux previously, albeit for a very short period. I have tried Mandrake 8.1 in the past but due to crashes and simply not being able to do most of what I do on Window XP I lost interest quite quickly. I was having to boot back into XP, do what I wanted to do and then boot back in Mandrake. After a while this got annoying :) As for Ubuntu, the only time so far I have had to boot into windows if for my ipod (I have played quickly with the Ubuntu ipod support but it didn't work and caused things to crash and haven't had time to persevere lately).  Using Synaptics to install applications is incredible and I have not had a better experience of getting things to work in Linux, bearing in mind I opted for the amd64 version which I was geared up for having a slew of problems. So all-in-all my impression of Ubuntu is very good and it is definitely something I am going to continue to use, in fact I am trying to always use Ubuntu and only resort back to windows when I really have to. 

I must admit getting Firefox to play nicely with java seems to be a bit of a chore (maybe only on Edgy amd64?) and in the process I think I have a number of different java versions installed and I am bit fussy about only having what is required installed (I don't like excess :D ). So I think I am going to get rid of all my java installations and start again, I would out of preference like to have 1.5 or even 1.6 running with firefox but whether that is possible on amd64 I will find out. I also have to learn about where Ubuntu likes to have things installed (when not using synaptics) just so that everything is in its correct place. An example of what I mean is depending on your Linux installation the JDK sits in various places "/usr/local/", "/java", etc. Again this is most probably me being fussy but I like to keep a tight ship :) So  a lot of things to learn but I have a feeling that Ubuntu will give me chance to solve problems rather than have me recompiling kernels and getting obscure drivers which as a beginner is too daunting ;) ...ok ok so I am a wimp!

hmm I have written loads, sorry ppl :D !!!

Mobby

If you're going to use Java on AMD64, and want the browser plugin, there is no alternative at this time but to use both the 64-bit and 32-bit Java versions, as well as the 32-bit version of your browser.

For your 64-bit Java-needing apps, install 64-bit Sun Java from the Ubuntu repository via Synaptic or sudo apt-get install sun-java-jre (or install the latest version from java.com).

Then for the 32-bit version with Firefox there is this very good [howto]("http://www.ubuntuforums.org/showthread.php?t=202537") in the 64-bit forum. It hasn't been updated in a couple of months, but you can also use [this method]("http://www.ubuntuforums.org/showpost.php?p=1823375&postcount=7") to have the latest 32-bit Firefox (provided you install the relevant 32-bit libraries first, as per the first howto.)
