### Post by vanadium on 2007-08-20
I have seen issues where other users could not play CD's on my desktop system, which was stull Edgy, the version you seem to use. What worked, was to give these users root permissions, then after removing these root permissions again, the issue was fixed. This might not help you, because you probably already have admin privileges, but you might check this anyway.

System - Administration - Users and groups, properties, "User privileges" tab: probably you'll see that "use audio devices" and "Use CD-Rom drives" is checked. I checked the "Administer the system" box, after which these other users could play audio CD's. After removing the Administer checkmark again for these users, it still would work.
