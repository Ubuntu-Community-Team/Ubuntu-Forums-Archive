### Post by timzak on 2007-06-14
If you have an extra hard drive, you could install Ubuntu on it, so that if you end up not liking it, it will simplify uninstallation.  What I did was unplug my Windows hard drive, then install Ubuntu on another hard drive so that the Ubuntu boot menu (called Grub) wouldn't infiltrate my Windows install.  I set the Ubuntu hard drive as my 1st hard drive in the bios, and if I want to boot into Windows, my bios has a boot menu that you can access.  So I hit F11 (in my case) to access the bios boot menu, and select the hard drive that Windows is on.  I have the freedom to go into my bios and set the Windows hard drive as 1st if I want (so it will boot by default) and I could still load Ubuntu by accessing the bios boot menu.  This all depends if your motherboard bios has a boot menu or not.

I recommend you dual boot while you get to know Ubuntu.  If you ever get to the point where Windows is not needed anymore, then you can erase the hard drive it's on and integrate it into your Ubuntu install.
