### Post by Shinbu-Otaku on 2007-12-05
it might just be that ubuntu thinks there are 2 sections to your USB stick, for example, my phone has a memory stick in, and when i connect to the pc it shows 2 icons, 1 being the phone memory, the other being the memory stick.

Also, you might have another USB storage device attached, without you knowing.....

I'd check to see if the USB stick is partitioned, because this might create the 2 USB icons as it would register them as 2 seperate drives.
