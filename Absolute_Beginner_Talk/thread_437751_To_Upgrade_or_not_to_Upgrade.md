### Post by Roytheboytoy on 2007-05-09
...That is the question!

I've been searching around the net on the advantages of upgrading to the newest ubuntu. A few months ago, it was Edgy, and now, it's Feisty. All I've read so far is stuff like "If you're happy with your current version, stick with it" and "You get the latest and the best if you upgrade". 

This sort of doesn't make too much sense considering that all recent ubuntu versions (I started from Dapper) have the software updates functionality. If this update feature works for updating the Kernel itself, why do we need to upgrade at all? It should in principle be able to get ALL the newest libraries, apps etc. that newest version has.

So, either the newest ubuntu version is packaging all this into one nice bundle for the new linux user OR old repositories are not updated with the "latest and the best" (which I don't think is true since this would make the whole software update thing redundant).

All this boils down to 2 simple questions! If I get all the newest software from the software updates in Dapper, Why should I upgrade to Feisty at all? And if I don't get all the latest updates from the old repos.... then why not?
