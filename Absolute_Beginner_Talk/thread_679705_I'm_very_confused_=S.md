### Post by Dr Small on 2008-01-27
Ubuntu supports Linux and Debian software, not Windows, so most likely you wouldn't be able to run Microsoft Word, etc. But there are alternatives, such as OpenOffice.

1.) Yes, you will be able to use Firefox. This comes preinstalled by default on Ubuntu. There are many different applications that come preinstalled to help you with your daily work, such as OpenOffice, Firefox, Evolution (or Thunderbird), Pidgin (a instant messaging application) and much much more.

2.) Yes, absolutely. You can install Ubuntu and it will take up the entire disc, or you can partition it (if you wish) to allow Windows and Ubuntu to run side by side.

3.) I have never heard of Virgin PC guard, but if it is like a antivirus or similar, you will not need it, since Ubuntu is not as prone to viruses as Windows is.
