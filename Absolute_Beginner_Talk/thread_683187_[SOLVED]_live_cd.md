### Post by Rocket2DMn on 2008-01-30
What you do is go to [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download) and download the LiveCD iso image.  Then you open your favorite cd burning program and choose the option to Burn Image to Disc.  This is not like burning data to a disc, it is an image.
Burn at a slow speed like 4x to prevent write errors, and use quality media if you can.
If you need a program to burn with, look here and make sure you use the correct version for your windows:
[https://help.ubuntu.com/community/BurningIsoHowto#head-5f36c46dbbdd2bd773ae1f5d361be66c6553babf](https://help.ubuntu.com/community/BurningIsoHowto#head-5f36c46dbbdd2bd773ae1f5d361be66c6553babf)
