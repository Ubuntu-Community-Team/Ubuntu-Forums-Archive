### Post by bonzodog on 2006-10-02
Go to :
System --> Administration --> Printing, where it will ask for your password.
Click on 'Add new Printer', but make sure printer is switched on first.

If it lists your printer in detected printers, then just click on that and it will set it up for you.

If there is nothing in that box, below it there is a section where you can list the port that it connects. Click on the port, then 'continue'.
It will bring you to a make & model list of Printers. first, find the make, then, if possible, the model. I have always found that often a 'sister' model of printer that is listed works just as well.

The set-up is fairly straightforward. Comeback to us if you have any further problems.
