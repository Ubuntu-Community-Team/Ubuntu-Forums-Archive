What that guide doesn't mention is anything about configuring Wine. On most occasions you won't need to, however occasionally you might ...and this might be one of them.

In the command line type:
```
sudo winecfg
```

To play my Football Manager game, I then go to the Applications tab and add an entry. In it I specify that fm.exe is opened using Windows 98 version.

And couple of other thoughts:
- Did you save Irfanview to your Wine folder?
- And how have you tried opening Irfanview? Did you type in the command line something like:
```
wine /path/to/irfanview
```

### Post by S29K on 2007-07-20
Hmmm....there are two ways to open an image with gThumb.

1. Start Gthumb and double-click on theimage you want to open it.
2. Open the image from Nautilus after setting gThumb as the default app.

Once the image is opened using either method, all you have to do is type Alt-C or select Image--> Crop to crop or Alt-S or select Image-->Resize to Resize.

The order of operations is reversed from Irfanview but IMHO doesn't take any extra clicks to achieve the same result.

At any rate, good luck in finding an app that will work the way you want it to.

I had the same problem after installing WINE could install some windows apps but IrfanView would not install.

I found this on the official Wine Site 
"Installation, once you get a copy of the Win 2000/XP file "mfc42.dll" and copy it to your ".wine/windows/system" folder."
[http://appdb.winehq.org/appview.php?iVersionId=7834](http://appdb.winehq.org/appview.php?iVersionId=7834)

To get the dll I searched a WinXP machine for it and copied it to /home/[ubuntuUserName]/.wine/drive_c/windows/system32

Then installed IrfanView. Right clicked on the Irfanview exe clicked open with other application, custom command typed wine.

I hope this helps you.
