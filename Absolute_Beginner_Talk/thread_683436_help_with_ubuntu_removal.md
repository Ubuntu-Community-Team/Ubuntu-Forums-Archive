### Post by Niniel on 2008-01-31
It is unlikely you can go back to your previous state since it sounds like you installed Ubuntu over XP.
You can, however, reinstall Windows (and delete Ubuntu).
For that, you will have to boot from the Windows installation disk and then take it from there (delete existing partitions, create new partition/s, install Win).

Good luck.

And, oh, yes, this comes up fairly often. :)
