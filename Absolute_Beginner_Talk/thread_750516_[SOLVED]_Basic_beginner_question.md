### Post by TeoBigusGeekus on 2008-04-09
1)Ubuntu uses the GNOME desktop manager and Kubuntu uses the KDE desktop manager. They are the same in their core but they come with different applications and looks. Image google both gnome and kde and see what you like most.. I personally have an allergy to KDE this period...

2)If you have not radically changed your system, then an upgrade will be as smooth as possible. If, on the other hand, you have played a lot with adjustments, altering configuration files, etc. then an upgrade could result in a non-working system. ### Post by nickpaton on 2008-04-09
From [Ubuntu Wiki ]("https://wiki.kubuntu.org/Kubuntu"):

> the Ubuntu CD contains only GNOME as a desktop environment; the Kubuntu CD contains only KDE, this is the primary difference

In general I would suggest only using the fully released version of any distro, or updating to the fully released version.

By their very nature beta versions are not fully working, and whilst you may well find it OK, equally you may find problems which could be somewhat challenging to sort out until the development team gets to it.

I have done both upgrades and reinstalls.  On balance I like to have a fresh install because the distro is clean with no program fragments from old programs which I always seem to end up with over a period of time with an existing version.
