### Post by Maestro23 on 2007-04-01
Installing from source is tricky, run into all kind of dependency issues.  You will need to install "build-essential and make" for starters, maybe some other packages as well.  The following link will get you started.  Read Section 4: Installing from source.  Yeah, you will need to install uudecode as well.  Make sure to READ all instructions for what you are trying to install.

[http://www.psychocats.net/ubuntu/installingsoftware](http://www.psychocats.net/ubuntu/installingsoftware)

Install methods:

GUI(Synaptic): Open up synaptic and search for what you need.  Mark for install and apply.

Command line:

Can use either sudo aptitude or sudo apt-get.

Read the Installing software at that link I provided.  Another link as well: [http://cutlersoftware.com/ubuntuinstall/#where_did_it_go](http://cutlersoftware.com/ubuntuinstall/#where_did_it_go)

Good luck.
