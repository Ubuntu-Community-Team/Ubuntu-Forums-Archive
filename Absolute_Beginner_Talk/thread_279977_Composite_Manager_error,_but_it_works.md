### Post by jordanmthomas on 2006-10-18
Here's your problem:
You don't have proper 3D drivers
```
Section "Device"
	Identifier	"ATI Technologies, Inc. RV370 5B62 [Radeon X600 (PCIE)]"
	Driver		"vesa"
	BusID		"PCI:1:0:0"
	VideoRam	256000
	Option		"UseFBDev"		"true"
EndSection
```

The fact that you are using the vesa driver gives this away.  Look around on the forums for how to set up the ATi drivers and then you should be good to go as far as composite managers go.

Good luck!
