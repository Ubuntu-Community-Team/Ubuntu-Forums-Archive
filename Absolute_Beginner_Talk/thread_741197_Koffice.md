### Post by gunksta on 2008-03-31
If those don't work OR if you really need a presentation program you might also look at Google Docs. The Google engineers are building some clever features into their on-line office applications. So far, they can't replace OOo for me. BUT, they have several useful features that OOo does not have (and will probably never have) that I find very useful. I find myself using Google's tools for specific projects where the collaborative tools are really necessary for me.

NOTE: [COLOR=Silver]Google Docs are _NOT_ available off-line (yet) and are therefore of limited value if you need to use them in a laptop that may not always have access to the web.[/COLOR]

** EDIT:** Google Docs are now available for off-line use now! Combined with ideas like Prism, Google could really start making some headway in the office space with this step forward.

[http://feeds.feedburner.com/~r/blogspot/MKuf/~3/261487288/offline-access-to-google-docs.html]("http://feeds.feedburner.com/%7Er/blogspot/MKuf/%7E3/261487288/offline-access-to-google-docs.html")

Since you are asking about KOffice, I will assume (possibly incorrectly) that you are running Kubuntu. Konqueror does not work with the Google Office, although this may not be true in the KDE 4.x series that will make a serious appearance in Hardy. However, Fire Fox works like a champ with Google Docs.
