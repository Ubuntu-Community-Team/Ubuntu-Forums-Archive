### Post by aysiu on 2006-04-21
[QUOTE=blackant]You would need 'wine' in Linux to run windows program. But it is not fully compatible too.[/quote] Generally (and this isn't true in all cases--I'm just speaking broadly) programs that tend to work in Windows 98 and below work perfectly fine in Wine. Wine is free in every way--open source and cost-free.

If an application requires Windows 2000 or XP, it probably won't work in Wine, and you'll need Crossover Office, which costs money (I can't remember whether it's open source or not). Think iTunes and some of the newer versions of Adobe and Macromedia products (excluding Reader and Flash plugin--which each have native Linux versions).

If you're into gaming, there's a hack of Wine called Cedega that costs money and isn't fully open source, and it can help you play games designed for Windows.

> 
It would be better to find alternative programs which are build for linux. Yes, this is always the first thing you should look for. Use Wine, Crossover Office, Cedega, or a dual boot only if you can't find any satisfactory Linux equivalents.
