### Post by trav1085 on 2006-09-30
What program are you trying to compile? Do you get error messages? If it just won't run, then you need build-essentials, get it by ```
sudo apt-get install build-essential
```. Then type ```
./configure
``` and then wait. Read the last bit of text, it might be ok and say your ready to run make or it will say something like "Checking for sample... Error, sample not installed. It is required to build this program." that means you have to go ```
sudo apt-get install sample
``` or use the Debian Package Search in the firefox search to find the package you need.
