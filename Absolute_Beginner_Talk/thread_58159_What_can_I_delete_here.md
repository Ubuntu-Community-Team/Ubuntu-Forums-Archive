### Post by heimo on 2005-08-19
jre stuff is java install files, lib-something is probably browser plugin for java, essential*bz2 is archive which is probably uncomressed in directory with same name, that png file is image file that's probably created from Firefox by clicking on an image and choosing 'set as wallpaper/background" or something like that, JediSpam folder is probably something you created... Safe way to delete anything from here is to move them to another directory first, create *old *directory and move stuff there (and burn on CD if you're uncertain), after week or so, delete if you didn't need them. Mplayer is probably install files for Mplayer. Only you know if these are needed, if you installed those programs, programs don't anymore use these files.

Hopefully I didn't make things more complex. :D Be careful, but not paranoid.
