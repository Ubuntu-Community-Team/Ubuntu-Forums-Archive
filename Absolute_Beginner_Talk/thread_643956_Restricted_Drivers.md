### Post by dstew on 2007-12-18
"Restricted" refers to the software license agreement. Restricted software remains the property of the maker, and cannot be copied and given out like the "free" (unrestricted) Ubuntu software. There are more subtle legal issues, but that is basically how it works. The warning is true, but it can also be that free software is risky, if it has bugs in it for example, or if it has security problems that are not yet fixed. It is often thought to be less risky than restricted software, because many eyes can examine the code of free software, but only few see the code of restricted software. It is far easier to hide defects and outright malicious code in restricted software.
