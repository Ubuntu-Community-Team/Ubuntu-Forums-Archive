### Post by emarkd on 2008-01-29
Hey, you don't have to apologize.  Lots of people dual-boot.  I've still got a dual-boot laptop that I use to do a little .NET coding on occasion.  VMWare is the way to go there. It's actually quite handy and works very well unless you need some sort of DirectX 3D stuff in Windows.  That's still a no-go in VMWare.

### Post by jonabyte on 2008-01-29
For "learning" lamp, you may be better off using a spare computer(if you have one) or getting a used computer and installing Ubuntu Server. Since its command line default, you will get good experience with the innards of Linux, you can use the Lamp Install Option to make things easier.

This way you can boot to which ever os you desire and still "play" with linux on your server.

I do web development as well, and I have this kind of a set up with a dual boot into xp and Ubuntu, and it has worked quite well for me.
