### Post by ajgreeny on 2006-06-18
.exe files are windows executable, not linux binaries so you can't execute them directly.  You may be able to run some of them, however, if you install and use *wine* though I've never yet found it to be either very dependable nor able to run programs that I used and loved on windows, such as *Mailwasher Free*, *Picasa*, and a few others.

I think personally it is better to just accept that you are now running a different OS and use the alternative programs that will do the same thing, though often in a different manner.  Just my idea; many will disagree with me but that is linux.  Different people think diferently and you are not stuck with what everybody else does, or thinks.
