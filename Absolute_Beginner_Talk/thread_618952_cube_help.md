### Post by smacattack on 2007-11-21
go into compiz-config-manager

go down the the 'utility section' and check the cube caps option. that should put default images on. if you want to add your own go to the 'Desktop Cube' option near the top of the config screen and go to the 'Appearance' tab and add your own image from there.
