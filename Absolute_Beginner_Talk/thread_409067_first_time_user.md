### Post by laidback on 2007-04-14
"so i would be able to swap between the os"

Yes you can. During the installation Ubuntu will install Grub, which controls the boot process and allows dual booting. The installation process will detect XP. You will need some space on your hdd.

### Post by xopher on 2007-04-14
> i know linux isnt generaly famous for its compatability with games. but is it possible?

It's possible, but don't expect too much. Look for wine, it's probably the best way of getting windows games working on linux.

> are there any free programs that will allow me to install stuff like; sims, theme park world, etc

Same as above, obviously, those should work pretty well too.

> can i install this over the top of windows with no hitches?

If you wish to overwrite windows, no problem. If you wish to install it beside windows - and dual boot - still no problem :)

Here's more information about that : [https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)
