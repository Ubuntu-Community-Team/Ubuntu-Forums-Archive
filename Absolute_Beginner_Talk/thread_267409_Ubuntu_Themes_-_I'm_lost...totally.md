### Post by CatKiller on 2006-09-28
The one that you've listed has complaints in the comments section that the downloads don't work. Haven't tried it myself.

With the Art Manager (gnome-art), you need to go to the Art menu and select which type of them object you want to download, then wait for quite a while for it to download the list, but then it will install the ones you want automatically.

### Post by CatKiller on 2006-09-28
The other thing to remember is that each part can be installed individually - the Theme Manager calls them Controls, Window Border and Icons. Unless you have each of them with the same name, they won't show up in the main Theme window, you'll have to select them from the Theme Details window. It allows you to mix-and-match though, so it's not all bad.

Also, one of the three elements is called an "Engine" - I can't remember which off the top of my head - and I think you need the appropriate engine installed before some themes will work. My description is vague because I don't really understand it myself. There are quite a few themes easily installable through Synaptic and the Art Manager though, and those should keep you busy while you work out how the rest of it works. Post back with what you find out - I'm quite curious myself.

Edit: I've noticed that you're looking at GDM themes. I think that these are themes for the login window, rather than themes for the desktop.
