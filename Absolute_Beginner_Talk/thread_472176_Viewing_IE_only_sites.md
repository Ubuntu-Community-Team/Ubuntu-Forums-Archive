### Post by Clay_Banger on 2007-06-12
some websites say they will only work with ie, but if you trick them into thinking that your browser is ie, they will display fine. I use "User Agent Switcher" for firefox. it changes the browsers signature making websites think that you are using ie or opera instead of firefox. this, of course can be turned on and off.

[https://addons.mozilla.org/en-US/firefox/addon/59]("https://addons.mozilla.org/en-US/firefox/addon/59")
