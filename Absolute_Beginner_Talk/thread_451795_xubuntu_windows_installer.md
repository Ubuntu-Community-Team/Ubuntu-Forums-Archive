### Post by irish_flu on 2007-05-22
This might not be exactly what you're looking for, but without being able to boot to anything but the hard drive, it's really hard to install an OS.

Check out "WUBI", the Windows-based Ubuntu Installer:

Forum is here:
[http://ubuntuforums.org/forumdisplay.php?f=234](http://ubuntuforums.org/forumdisplay.php?f=234)

Project homepage is here:
[http://www.cutlersoftware.com/ubuntusetup/wubi/en-US/index.html](http://www.cutlersoftware.com/ubuntusetup/wubi/en-US/index.html)

NOTE: I have never tried this myself, and as far as I know this software is not release-ready.  I'd read through the threads in that forum and check out their website before you decide whether or not it's for you.  Otherwise, you could try VMWare Player or Server, which are both free, or Microsoft's "virtual machine' software.
