### Post by Unknowndeepness on 2006-05-13
xbindkeys and xbindkeys-config, you can download them with apt-get (apt-get install xbindkeys xbindkeys-config) or with synaptic / adept / something else. Then run xbindkeys-config and make some magic! ^^
