### Post by Paqman on 2008-03-09
You'll find Ubuntu is a LOT better than Knoppix. Knoppix is a great recovery/portable distro, but it sucks as a workstation/desktop.

Have you heard of [Ies4Linux](http://www.tatanka.com.br/ies4linux/page/Main_Page)? It'll allow you to do IE browser checks from within Linux.

You're right about the difference between Ubuntu and Kubuntu. The only difference is the default desktop environment. Software designed for one will work fine on the other (although startup times will be slightly longer for aps not within their native environment) There's also Xubuntu which uses the lightweight XFCE desktop.

### Post by mimada on 2008-03-09
May I suggest trying VirtualBox (virtualbox.org)? You can install Ubuntu inside of Vista and check it out before messing with dual boots. Personally, if you have the Vista install discs I would make Ubuntu the host install and load Vista as a guest. Be aware that the 3D effects in a Windows VM suck. If you want to experiment, try creating a Vista VM within Vista to see if it can support the apps you need.
