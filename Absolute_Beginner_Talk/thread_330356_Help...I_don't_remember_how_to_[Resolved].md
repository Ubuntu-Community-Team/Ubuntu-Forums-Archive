### Post by MkfIbK7a on 2007-01-03
you want a different desktop manager?
if so do for each desktop manager respectively,
sudo apt-get install kubuntu-desktop 
(KDE)
sudo apt-get install ubuntu-desktop 
(GNOME)
sudo apt-get install xubuntu-desktop 
(XFCE4)
then log out click the sessions option on the login screen and choose what DM you want
