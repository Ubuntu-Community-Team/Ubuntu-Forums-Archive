### Post by xpod on 2007-06-24
256Mb of ram is just about the bare minimum that will run a live cd,but more often than not it can still be problematic.

You could try the "safe graphics mode" option at the start screen.Worked for me originally.
Of course the first question should have been are you sure it`s a good burn??

Theres a "check cd for defects" option on that start screen too to do that.
If your sure you want to install Ubuntu you could always use an "alternate cd" if the live one wont work or even better you might consider trying Xubuntu which would run a lot better on your 256Mb.
