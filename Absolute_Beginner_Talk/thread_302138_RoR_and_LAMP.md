The server edition and the desktop edition are identical with the exception of the AMP part of LAMP and the desktop graphical environment.  The server edition does not install a graphical environment.  In other words, you can use the desktop version of Ubuntu, install Apache, mySQL, and PHP and you will have a server with a graphical environment.
