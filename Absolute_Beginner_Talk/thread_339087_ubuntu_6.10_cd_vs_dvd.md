### Post by Pobega on 2007-01-15
The DVD is good if you're not on a quick connection, and need a one-time download for (almost) everything you'd ever need (i.e. if you're having a friend download the Ubuntu disc for you). The CD is better for quicker connections because it comes with the essentials, and you can install what you need as you need it.
