### Post by freebird54 on 2007-05-08
Didn't follow the links at all - so not sure what might be confusing you.  However, it is actually pretty easy to set up, and even fairly easy to change around AFTER it's set up too.

Easiest is to just go ahead and install Ubuntu, and let it resize XP - and install in what's left.  That may be all that's required.

To ensure a happy outcome, a little preparation can't hurt...  Turn OFF your page file in XP (under My Computer->Advanced) then run defrag a couple of times minimum.  This will make sure that the maximum room is available for your use.

If you want something fancier, you will have to do a little more work - like creating specific partitions in manual moe when installing  - but that isn't too difficult either.  This site has some interesting things on it that might make it clearer..

[http://www.psychocats.net/ubuntu/index](http://www.psychocats.net/ubuntu/index)

Enjoy - and ask questions anytime.  Remember, you can stay hooked up to the net throughout the install process if you want to - so you can get answers at critical points if necessary!  (slows things down, but makes 'em safer)
