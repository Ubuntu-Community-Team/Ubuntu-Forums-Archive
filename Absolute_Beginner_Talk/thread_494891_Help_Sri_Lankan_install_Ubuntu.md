### Post by Benbow on 2007-07-07
Although it is not mentioned here and not an official download, you may find that Ubuntu Ultimate is a good starter if you have no internet connection. It contains a plethora of alternative programs already installed.
However, you will not have access to this forum which to me is invaluable, nor will you have access to upgrades to ubuntu and programs.
If you download the Ubuntu Ultimate to cd you can boot from it directly and use Ubuntu from the cd as though it were on your hard drive, however, it is just a temporary measure as you will lose anything which you have done as soon as you turn off your pc. It will show you how Ubuntu will accept your current pc hardware though, and.you can later install Ubuntu from the window of the boot cd.
Hope this is of help but your main problem with any system is a lack of net access.
Forgot to add that Ubuntu Ultimate is just under a 2gb download but you can obtain a free installation cd.

### Post by Wim Sturkenboom on 2007-07-09
I will use the following **minimum** szies:

swap is 2x memory with max of 1GB
about 5 GB for root (/)
rest of what you want to spend for /home

This will do for a normal install and some additional space for a media player. You will probably still have a Gigabyte left after that, but logfiles also need some space. I assume that you will not run updates as you don't have internet connectivity so your 5 GB should be safe.
