### Post by Nopposan on 2007-01-01
'Sorry.  Well, I have dealt with a "similar" problem with a U.S. bank but I don't recall exactly what the solution was.  What I do recall is that the solution involved getting more information from the bank about exactly why their online banking system requires IE's security settings be set too low.  Maybe the folks at the bank aren't as friendly as the Ubuntu forum, but I guess they're the ones who have the information you need.

However, it could actually be that they've got something on their website user interface that looks for an IE browser and supposedly requires IE to be there.  I really don't know if this would work, but I'd laugh if it did: try installing Firefox's "User Agent Switcher" extension.  Many websites that say they work only with IE will come through just fine on Firefox if Firefox lies and says, "Sure, I'm IE."  This has worked for my wife's online courses at the local community college and it's also worked for me while using Firefox at work.

Good luck.
