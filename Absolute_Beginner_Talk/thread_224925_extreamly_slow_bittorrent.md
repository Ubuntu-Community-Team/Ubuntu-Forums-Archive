### Post by UltraMathMan on 2006-07-28
I'm not familiar with the stock client and haven't set up bittorrent in ubuntu yet, however I'd recommend using the Bittorrent manager Azureus. I used in when I was still a Windows user but it has a native linux port (you can find it through Synaptic package manager in the universe repository). If you are in a home network with a router or firewall you may need to allow bittorrent access to get optimal speed (if in a corporate/education network you're probably out of luck). I'm not sure if you need to change any settings in ubuntu. From the sound of your situation I'd guess this [ports not forwarded] is the problem. I'd install Azureus and google "port forwarding *your router model*" for more specific instructions.
