### Post by louieb on 2006-09-12
If I remember right from the days of using a 386.
1. The card buss is ISA (long black slots) which means no auto setup of IRQ's or DMA's. 
2. Hard drive space was measured in meg not gig. Forgot what the bios limit was.
3. Probably used a 16 bit data buss. Most modren opperateing system need a machine with a 32 bit buss.
4. limited memory addressing. Needs add on card and drivers to install more that 2 meg of memory. And yes that MEG not GIG.

Conclusion: If you can get a working hadrdrive. You will probally need to find a really old version of linux and compile it yourself. Would be suprised if a GUI for Linux worked at all. Command line only. 
Or you could do what so many of did then. Use DOS 5 or 6. and if you want a GUI then find a copy of WIN 3.11. and be prepaired to see the blue screen of death. 
Is it worth the effort? Not for me.](*,)
