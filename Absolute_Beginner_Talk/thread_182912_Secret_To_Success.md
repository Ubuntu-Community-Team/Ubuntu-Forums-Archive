I'm extremely new to linux/ubuntu (several days), however, despite this I have managed to upgrade firefox, and install numerous applications from the terminal and it was easy.  OK now I'll tell you how I did it.  First, go to psychocats.net/ubuntu/firefox.  Download the script you find there, open the terminal and enter the appropriate command prompt and it's done. Next, go to psychocats.net/ubuntu/sources and you will find everything you need to know about installing applications.  Also check out beggingubuntu.com it is very helpful as well.  Note: using Dell E1705/same as Inspiron 9400 laptop.
