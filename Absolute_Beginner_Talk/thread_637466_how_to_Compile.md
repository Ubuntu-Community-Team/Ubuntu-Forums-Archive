### Post by PmDematagoda on 2007-12-11
First install the necessary prerequisites to compile source codes using:-

```
sudo apt-get install build-essential
```

Then extract the files to a folder. Then navigate to the folder and do the following in their order:-
```

./configure

make

sudo make install
```

This is the basic way of compiling applications.
