### Post by sawjew on 2006-12-21
There is a setting in Nautilus Preferences that governs what information is displayed.

[LIST]
[*]Open Nautilus
[*]Go to 'Edit > Preferences'
[*]Select the 'Display' tab
[/LIST]

You should see a window as shown in the attachment

To turn the size on or off or display anything else just change what is displayed in the first box.  The other boxes won't do anything if the first box says 'None'

hi,
nautilus save settings, like size, position and so on, of each folder on a one-by-one basis.

Even if you don't have general setting defined to show size of files, if you zoom in a folder (pressing ctrl++ or Zoom in from 'View' menu), zoom view will show more details about files like size, eventually date, ownership... if you close the window zoomed it will be opened with that same setting and showing that extra information.

You can go back to default by ctrl+- or Zoom out from menu.
