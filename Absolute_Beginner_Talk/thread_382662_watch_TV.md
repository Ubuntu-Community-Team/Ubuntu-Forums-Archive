### Post by rowdy15 on 2007-03-12
Hey everyone,

this is how I got my tv card working, ps I'm only three weeks into using ubuntu so if 
this post isn't specific for your computer/card, then try investigating on the sites used:


firstly I followed the instructions on this web page

[http://www.linuxtv.org/wiki/index.php/How_to_install_DVB]("http://www.linuxtv.org/wiki/index.php/How_to_install_DVB")

NOTE: before I started anything I typed command:

 sudo -s             (gives continued root access)

and was handy in this case

secondly I followed the instructions on this web page

[http://www.linuxtv.org/wiki/index.php/First_steps_with_a_budget_DVB_card]("http://www.linuxtv.org/wiki/index.php/First_steps_with_a_budget_DVB_card")

NOTE: my setup was slightly different in that I live in Melbourne, Australia and therefore instead of having "uk-Oxford", I used the au-Melbourne channelscan file, which there is a heap of different ones found in:

/usr/share/doc/dvb-utils/examples/scan/dvb-t/

Also where it said i should do the command:

tzap Bloomberg -- Bloomberg

I instead did:

tzap "TEN Digital" -- "TEN Digital"

somehow the talking marks or whatever they're called made it work

also put the channels.conf file in the .xine folder in your home directory

NOTE: the .xine folder is accessible only if you do Ctrl-H in the file browser in your home (of the user) directory

hope this helps someone trying to setup tv on ubuntu

if you are still having problems

you could maybe try deleting some stuff out of the channels.conf file that was put into the  .xine folder
