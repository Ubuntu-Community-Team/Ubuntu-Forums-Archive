### Post by spvo on 2006-09-10
Run the program csm.  Do disable zoom go to 'rotate cube / numberic values' and set zoom to zero.  I did the same thing, it makes the workspace change much faster.  The more features you disable, the faster it will probably go.  Try turning off fade and minimize, that will probably help you windows show up faster.  Also in general it has a refresh rate, you could try turning that down a touch.
