### Post by Strider42 on 2007-03-25
I was wavering for several weeks, should I, shouldn't I drop XP.  In fact everything I had ever done was being relearned.  I can't remember the number of times I re-installed Ubuntu - bloody hell I was going grey overnight...  But then I got to thinking, I'm actually learning something new here, this is interesting - I can even post a few helpful comments based on my own experiences - and that's what makes up this great community.  That doesn't really happen in Bill's world.

Years ago I worked in a software house writing COBOL programs (yes, I know, old fart) in a Unix environment and stuff started coming back to me pretty quickly.  So, I decided to jump - backed up all my data files, emails, contacts, and just recklessly installed Ubuntu over my XP installation formatting the whole drive in the process.

I tried Wine with these, but just couldn't get them running.  Maybe if I have some time I'll try again and I can kiss goodbye to Windows forever.

It's easy to jump on the bandwagon and shout out loud that M$ is crap and everyone should drop it for Linux, but the reality is, it's horses for courses and whatever works best for you.  For me, Linux is the way forward and I'm enjoying it. :biggrin:

### Post by Strider42 on 2007-03-27
Create a folder on your desktop called Test

Now open a terminal window and cd etc/X11 as an example (or any other directory).

Now try to cd  /Desktop/Test - you'll get an error:  No such file or directory

Now try to cd ~/Desktop/Test - you'll end up in /Desktop/Test

Useful for me because all my archives get extracted to the Desktop.
