### Post by avanrens on 2007-04-21
When I first boot the network keeps disconnecting and reconnecting I can rectify this by selecting the wired network connection icon and selecting static configuration. and deslecting   wired connection 1(eth1) and reselecting the same connection. The connection works fine from then on until I reboot again.
