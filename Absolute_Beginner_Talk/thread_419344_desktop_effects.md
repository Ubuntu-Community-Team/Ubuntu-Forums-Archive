### Post by old_geekster on 2007-04-23
First, it would help if you edited your profile to show the distro that you are using.  This will save some questions from those who try to help in the future.  My instructions are based on the presumption that you are using Feisty.

I would suggest installing the nVidia drivers prior to enabling "Desktop Effects".  Here is a link to the best guide that I have found:

[http://ubuntuguide.org/wiki/Ubuntu:Feisty](http://ubuntuguide.org/wiki/Ubuntu:Feisty)

Section 1.12.2.1 will help you install the drivers.

Section 1.9 will help you install one of the 3D desktops or you can attempt using "Desktop Effects".

I prefer using "Beryl".  I have it setup on my rig and it is running perfectly.  All of the plugins are enabled.

Good luck!
