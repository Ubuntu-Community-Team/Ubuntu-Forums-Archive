### Post by sapo on 2005-12-03
I think you should use this oportunity to install the new version of ubuntu 5.10, breezy badger.

but if you want to just change you password, put the ubuntu install cd on the drive, and in the first screen type rescue and press enter.

Chose the partition that you have installed Ubuntu.

You will find yourself in a terminal, in this terminal just type like this:

```

useradd your_user_name
passwd your_user_name

```

it will add a user and in the second command it will ask for the user password.

Now reboot and login with the user and password you have just created.

hope it works =x
