### Post by ncappel1 on 2007-12-11
Can you be more clear about the functionality issues? I use the compose key added from the keyboard layout window (mapped as Right ctrl) on my laptop to type characters like è (ctrl, `, e) ñ (ctrl, ~, n) and ç (ctrl, (comma) c). I had issues early on too because I thought that the keys had to be pressed at the same time, but I discovered that it isn't necessary, one key right after the other will also work. 

Have you tried just configureing another keyboard layout from System-->preferences-->Keyboard? you can have 2 side by side and switch between them.

edit: The following may be overkill, but here is a blog post that will let you tinker with the keyboard layout. you probably won't want to try it until you've tried some other options first: [http://blog.dotkam.com/index.php/2007/06/25/custom-keyboard-layout-in-ubuntu-or-just-linux/](http://blog.dotkam.com/index.php/2007/06/25/custom-keyboard-layout-in-ubuntu-or-just-linux/)
