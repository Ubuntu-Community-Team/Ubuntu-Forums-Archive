### Post by rudeboyskunk on 2007-10-23
Cedega -is- better for games, because Transgaming has done most of the dirty work for you (since you have to pay for it).  However, I can personally say if you are trying to install Half Life 1 (original version, pre-Steam), it will not work in Cedega, but it will work in Wine.

Crossover is meant for office products.  A lot easier to install Photoshop with Crossover (but I'd rather use Gimp anyway).

Wine is great because unlike the other two, it's free.  Plus it helps you learn a lot about editing config files when you use it ;)

I use Cedega now, but I am dropping it because I'm tired of paying for support that I do not get.
