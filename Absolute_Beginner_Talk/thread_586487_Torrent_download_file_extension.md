### Post by jagdave on 2007-10-22
You havent enabled windows to show known file extensions. Open up a windows explorer window, like this computer. Click Folder - Settings something, and change it so that windows does show full file name including known extensions. 
edit. 
you dont have to do what i said to burn the image file though. You only have to burn it. What you do is look for an option in you burning software that is called something like "Burn image",  "Write image file" or "Burn iso" or whatever. If you dont have a program for burning this, search google for iso burner download or something, to find one. Then you just start the burning program after installed and look up the iso file containing ubuntu 7.10 and burn it. Then after its done you can boot from it. But if it doesnt, you may have to change boot sequence in bios, so that it first reads and boots from the cd you burned.
