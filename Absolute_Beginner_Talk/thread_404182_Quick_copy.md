### Post by LaurelLynn on 2007-04-08
Dear paradoc,

That is a Windows technique that works in programs link Firefox, but not in all programs.

The original Unix standard was to highlight text then click the middle mouse put button where you wanted to paste.  That still works, but three buttoned mice are now few and far between. So instead you have to click both buttons at the same time to paste.

That is an X11 standard, so it should work with all programs.
