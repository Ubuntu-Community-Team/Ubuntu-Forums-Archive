### Post by Cre-ve on 2007-02-27
I finally got my dual boot windows XP and Ubuntu to work! Everything works great. The Grub starts up and I can pick whichever I want XP or Ubuntu.

 It was a pain to learn all the details about the harddrive, how to partition it and such but its such a reward to know that i can finally begin my transformation to linux from windows.

If anyone else is having problems with the dual boot here is a little help: Partitioning your HD is the KEY to installing a dual boot linux.

Download [[COLOR="Red"]Linux System Rescue CD[/COLOR]](http://www.sysresccd.org/Main_Page) before you do anything.

Use this CD to partition your hardrive. (if you have windows XP more than likely...you are going to have to reinstall your OS again because XP doesn't let you partition it very effeciently so make sure you SAVE everything) I had to reinstall my XP OS about 2 times before it actually worked...but it was well worth it. THIS IS THE HARDEST PART SO MAKE SURE YOU GET THE PARTITIONING RIGHT.

Follow [[COLOR="Red"]THIS GUIDE[/COLOR]](http://www.matthewjmiller.net/howtos/dual-boot-linux-and-windows/) up to the part that says don't install grub. 

I think grub has been fixed up since the earlier release and it does work with the sytem. So just install that and thats all you really have to do to get linux to work.

Also use [[COLOR="Red"]THIS GUIDE[/COLOR]](http://www.psychocats.net/ubuntu/installing) side by side with the top link. It really helped me out. 

GOOD LUCK to everyone who is trying to install it and having problems.

Also if anyone has any tips on where to go from right after installing ubuntu onto the computer feel free to let me know. :) :guitar:
