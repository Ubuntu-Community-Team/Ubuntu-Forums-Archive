### Post by Canis familiaris on 2007-01-16
I use Ubuntu with the following specs, which by the way has just the minimum specs:
(1) Pentium III (Coppermine) 700 Mhz
(2) 256 MB of RAM*
(3) 4 MB Video RAM (SiS)'' -> 2MB can be used for lower resolutions
(4) An SIS Motherboard
(5) An 80 GB and a 10.2 GB Hard Drive (Atleast a 5 GB Hard Drive Required)

*If you can get hold of an alternate CD, then only 128 MB is required
''Note: ATI Video Cards are poorly supported in Linux.

You can visit here for details:

(1) Pentium III (Coppermine) 700 Mhz


[http://www.ubuntu.com/download/releasenotes/606](http://www.ubuntu.com/download/releasenotes/606)

You can run Ubuntu on a pentium II.  I would not recommend a pentium one.  On a 233 Mhz processor, it will be slow, but it will run fine.

I recommend a 400 MHz processor.

The installation of Ubuntu only requires about two gigs of hard drive space, including the swap space.  Give yourself more than that, so that you can actually use the OS.

It will run on any motherboard.  If you find a montherboard on which it does not run, it's a bug.

### Post by az on 2007-01-16
The installer (in text mode) will need 64 megs at a minimum to run.  As mentioned, the live cd installer requires 256 megs of ram.

You can install using the laternate cd with less ram than 256 megs, but you will probably be frustrated with less than 256 megs of ram using the default gnome desktop.
