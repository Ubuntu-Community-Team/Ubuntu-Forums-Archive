### Post by damontallen on 2007-12-16
I have a problem with my number pad.  When ever I type numbers on my number pad Ubuntu logs out.  I think it might have something to do with a window that came up when I was starting gPodder (or some video program) that said something about mouse gestures and number pad (I think) but I didn't realize It would cause this problem.  There was a message in the window about how to undo the change but I didn't write it down.  Does anyone know how to fix this?

---

### Post by damontallen on 2007-12-16
Actually I think I figured it out.  It seems that the "Mouse Keys" were enabled and somehow this was causing the forced log out.  The problem has gone away since I disabled the "Mouse Keys" under System->Preferences->Keyboard->Accessibility (at the bottom) ->Mouse Keys-> Uncheck Enabled.  I hope this helps someone else.
