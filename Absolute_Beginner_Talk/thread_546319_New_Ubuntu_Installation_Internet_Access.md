### Post by wapfu on 2007-09-08
Hi,
Don't know if this will help but a while back i couldn't get internet after installing ubuntu.
I am using broadband so maybe the same things are at play with windows.
Windows configures the modem to switch off when it shuts down and to switch on when it starts. Ubuntu will see the modem but doesn't know how to start it. In windows, go to device manger search the properties of the modem and there possibly will be a flag that tells the modem to start at boot with power on, not when the operating system tells it to start. Its like having it start itself when you power on.
Not sure if this will shed light.
