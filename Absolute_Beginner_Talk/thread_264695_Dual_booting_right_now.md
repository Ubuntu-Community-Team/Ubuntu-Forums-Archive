### Post by gn2 on 2006-09-25
Be very careful, Ubuntu will install Grub over the mbr whatever you do.

I suggest installing Ubuntu to a separate hard drive, with thwe Windows drive switched off (unplug the power) Then when booting press F8 to select the boot device.

Be vey careful if you are using one harddrive. You can easily lose the lot.
Back up everything before you go any further!
