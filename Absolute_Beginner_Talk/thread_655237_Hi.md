### Post by kestrel1 on 2008-01-01
Just connected my camera. When you click on import photo's it opens Gthumb. If you go to  File-Import Photo's it will open another window that enables you to select the photo's that you want to import & the folder to which you want them copied.
If you want to just copy them you should be able to open up the disk that should show up on the desktop & copy them from there to the folder that you want. By copy I mean, highlight the images you want, right click & select copy from the list or go to Edit-Copy then open the destination folder eg. Pictures & right click & paste or Edit-Paste.
