Animus - If you're using breezy, just click on System--->Preference----->Theme (2nd to the last item)....Window opens and the click on "Install Theme"----->point on the directory where your downloaded theme is and it will say "New Theme is Installed". ### Post by 23meg on 2005-11-01
Gnome theme manager accepts icon, GTK and Metacity themes. GDM themes can be installed in System / Administration / Login Screen Setup. The reason some files seem to be invalid might be that there are actually multiple themes inside the archive; some themes ship with a few variations, such as different colors, rounded edges, etc. Look inside the archive file to see if this is the case.

[This post]("http://www.ubuntuforums.org/showthread.php?p=456910#post456910") should also help you.
