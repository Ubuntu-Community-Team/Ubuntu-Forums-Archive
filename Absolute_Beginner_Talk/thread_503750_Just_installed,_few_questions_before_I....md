### Post by temba on 2007-07-18
Hi there,

tips n tricks are abound! Search for some guides like this one: [http://ubuntuguide.org/wiki/Ubuntu:Feisty](http://ubuntuguide.org/wiki/Ubuntu:Feisty)

I reckon version 7.x is the way to go but its a matter of choice. If you do go for version 6.x, then take Dapper as it is long term supported.

You can use the guide above to install eg media codecs and the like or you can't watch DVDs out the box. It's pretty simple and is a good way to also get to know your system better.

You can also google for Automatix2 as this program installs a lot of software and codecs without a major hassle. Some will not recommend this as you lack a certain degree of control over the programs you install, but it is a matter of taste. It kills/crashes some pcs and works with others. Use it after you have made a clean install so time wasted if anything goes wrong is minimal. The *ubuntu installation doesn't take too much time anyway.

I believe Cedega supprts WoW - check [www.transgaming.com](www.transgaming.com) - yes, it costs money, about USD6 per month?? Alternatively you can fiddle around with WINE - a bit of patience, coffee and logic usually gets this going and oh yes...a few clicks at various forums!

Go for it! If you have other queries, then search the forum first and if no luck, then write back with details and eg what hardware, software you are using. The more details the better, often the better reply.

cheers
