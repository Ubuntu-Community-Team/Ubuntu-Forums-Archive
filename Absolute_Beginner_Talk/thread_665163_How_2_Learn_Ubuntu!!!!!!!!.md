Use partition magic, shrink a partition where you have enough space. Ubuntu Wiki says 2 GB is minimum for swap and user space. I suggest you get somewhat more if you also want to install applications and you sound like you are serious about linux so you should definately get more space than that. During the installation process you can easily select which partition to use and it will also install a boot-loader so you can still boot windows or other OS'es if you wish.
