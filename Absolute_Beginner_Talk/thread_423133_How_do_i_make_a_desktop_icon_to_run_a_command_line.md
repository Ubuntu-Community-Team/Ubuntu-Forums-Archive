### Post by Lord Illidan on 2007-04-25
Very simple.

Right click on the desktop, chose Create Launcher, and chose Type - Application in Terminal.

Then, enter the command in the Command Field, and give the launcher a name, and potentially, an icon for itself. That's all there is to it.
