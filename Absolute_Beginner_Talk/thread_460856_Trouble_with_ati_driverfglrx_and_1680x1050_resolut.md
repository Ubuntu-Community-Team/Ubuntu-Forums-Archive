### Post by InlawBiker on 2007-06-04

It knows what resolution to be in by trying the first resolution listed for your color depth.  If you start up in 24bit color (almost everybody does), and 1680x1050 is your first resolution listed then that's what it'll start in.

```
	SubSection "Display"
		Depth		24
		Modes		"1680x1050" "1600x1200" "1280x1024" "1024x768" "832x624" "800x600" "720x400" "640x480"
	EndSubSection
```

### Post by renzokuken on 2008-02-12
you can either edit your xorg.conf manually by running

```
gksudo gedit /etc/X11/xorg.conf
```

or use the terminal text based wizard by running

```
sudo dpkg-reconfigure xserver-xorg
```

in the end they both do the same thing.

make sure you install the ATI driver first though, preferably through the Restricted drivers Manager.

my advice would be to use the second command after the driver is installed, making sure **fglrx** is selected as the driver, and adding the desired resolutions. any other properties in the wizard can pretty much be left as they are
