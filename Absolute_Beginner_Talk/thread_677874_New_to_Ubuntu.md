### Post by akniss on 2008-01-25
To IM with MSN users, go to Applications > Internet > Pidgin Internet Messenger.  If that doesn't work, there are other options available.  However, MSN IM will probably not be an option on Ubuntu.

There's really no need to install Kaspersky, or any other anti-virus software, as Linux is not  really a major target for virus attacks.  However, if you really want I believe AVG has a linux version.

Ubuntu is not Windows.  Hence many of the programs that you used will not work in Ubuntu.  There are ways of getting some programs to work, but it is often easier to find GNU/Linux compatible alternatives.  I believe CS3 will work under WINE.  Do a google search for "wine" to get an idea of what that is.

Welcome to Ubuntu and good luck!
