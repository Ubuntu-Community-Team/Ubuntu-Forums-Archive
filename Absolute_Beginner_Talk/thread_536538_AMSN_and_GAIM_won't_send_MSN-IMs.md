### Post by Mud.Knee.Havoc on 2007-08-27
$20 says you're using the HTTP method.  Go into your MSN properties (in Pidgin's "Accounts", in the Advanced tab for your MSN account), uncheck that and you're good to go.

I believe you'll need to restart Pidgin for the new setting to take effect.  If you don't have the "Use HTTP method" checked and it still doesn't work, doublecheck that you have necessary ports open.  But I'm 99% sure that you're using the HTTP method...
