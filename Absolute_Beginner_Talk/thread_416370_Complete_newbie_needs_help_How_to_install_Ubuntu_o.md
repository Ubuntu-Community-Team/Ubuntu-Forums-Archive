### Post by avik on 2007-04-21
The 80GB HDD can be safely wiped out, right?  The Windows installation is in another HDD?

In that case, just unplug all the HDDs except the one you want to use for installation.  That way, there's physically no chance of any other HDD except that one being affected.

Then, just use the Live CD, and you'll be good to go!  There is a boot manager, but it will be on that HDD itself, so it won't mess with your Windows boot manager.

Get the system up and running, and then worry about your ATI graphics card (which will require a driver, but as I use Nvidia, I'm not sure of the details).
