### Post by emarkd on 2008-01-22
There are lots of virtual terminals already running.  Press <CTRL><ALT>F1 or F2 or F3 to see what I mean.  Also, if you don't want to switch around to multiple logins, you can use screen.

```
sudo apt-get install screen
```

It's a great virtual terminal application

### Post by Dr Small on 2008-01-22
You can go to your other Virtual Terminals by pressing ALT + F1 thru F6, and you can use screen, and have as many terminals as you need :)
