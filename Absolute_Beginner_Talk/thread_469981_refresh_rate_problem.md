### Post by Pragmatist on 2007-06-10
>  
Section "Monitor"
	Identifier	"Generic Monitor"
	Option		"DPMS"
[B] 	HorizSync	28-96
	VertRefresh	43-60[/B]
EndSection

You want to find out the vertical and horizontal refresh rates for your monitor and then modify the above lines.  Try that first.  If you still have the resolution change on you, try this (In all these cases, BACKUP the file first:

Notice that this section is a list of all the current choices for your screen:
> 
Section "Screen"
	Identifier	"Default Screen"
	Device		"ATI RADEON X850"
	Monitor		"Generic Monitor"
** 	DefaultDepth	24**

Go to the section with depth 24:
>  	
SubSection "Display"
** 		Depth		24**
		Modes		"1280x1024" "1280x960" "1280x854" "1280x800" "1280x768" "1200x800" "1152x864" "1152x768" "1024x768" "800x600" "640x480"
	EndSubSection

And just leave the resolution you want and remove the others:
>  	
SubSection "Display"
** 		Depth		24**
 		Modes		"1280x1024"
 	EndSubSection

### Post by nutz on 2007-06-10
Try this layout and see what you get.


Section "Monitor"
Identifier "Generic Monitor"
Option "DPMS"
EndSection

Section "Screen"
Identifier "Default Screen"
Device "ATI RADEON X850"
Monitor "Generic Monitor"
DefaultDepth 24
SubSection "Display"
Depth 24
Modes "1280x1024"
EndSubSection
EndSection



Just make those two sections look like what I have there and restart. If that still doesn't do the trick then you will need to create a modeline...

[http://en.wikipedia.org/wiki/XFree86_Modeline](http://en.wikipedia.org/wiki/XFree86_Modeline)

How to create and use a modeline...

[http://wiki.caoslinux.org/X_Server_Configuration](http://wiki.caoslinux.org/X_Server_Configuration)

### Post by nutz on 2007-06-10
And unless you are using a tablet you can also get rid of these sections...

Section "InputDevice"
Driver "wacom"
Identifier "stylus"
Option "Device" "/dev/input/wacom"
Option "Type" "stylus"
Option "ForceDevice" "ISDV4" # Tablet PC ONLY
EndSection

Section "InputDevice"
Driver "wacom"
Identifier "eraser"
Option "Device" "/dev/input/wacom"
Option "Type" "eraser"
Option "ForceDevice" "ISDV4" # Tablet PC ONLY
EndSection

Section "InputDevice"
Driver "wacom"
Identifier "cursor"
Option "Device" "/dev/input/wacom"
Option "Type" "cursor"
Option "ForceDevice" "ISDV4" # Tablet PC ONLY
EndSection


But also make sure you delete their load entries like this...

Section "ServerLayout"
Identifier "Default Layout"
Screen "Default Screen"
InputDevice "Generic Keyboard"
InputDevice "Configured Mouse"
[B]InputDevice "stylus" "SendCoreEvents"
InputDevice "cursor" "SendCoreEvents"
InputDevice "eraser" "SendCoreEvents"[/B]
EndSection


Remove the entries in bold...

So it looks like this...

Section "ServerLayout"
Identifier "Default Layout"
Screen "Default Screen"
InputDevice "Generic Keyboard"
InputDevice "Configured Mouse"
EndSection
