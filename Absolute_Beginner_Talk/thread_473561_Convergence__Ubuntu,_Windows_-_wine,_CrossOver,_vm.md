### Post by cwmoser on 2007-06-14
I have all three of these on my Ubuntu PC:
    wine for eSword and Spider Solitaire
    CrossOver running my copy of Quicken and Quickbooks
    vmware running my copy of Quicken and Quickbooks


All three of these technologies provide an environment for "legacy"" Windows applications to run on Ubuntu Linux.  One thing that impresses me most about these is that they far exceeded my expectations of performance.  I had the perception that they were "emulators" but  learned that they are not and that is the reason for the superior performance.  In fact, my copy of Windows XP running in an Ubuntu window with vmware is faster than many actual PC's running Windows XP.

My PC is a 2.8 Ghz PC with Sata hard drives.  My neighbor has a 5000+ AMD dual core CPU and my question is how much faster would wine/CrossOver/vmware run on a PC like my neighbors?  I ask this to get an idea about the potential for "convergence" of Linux and Windows because CPU speed is constantly going up and in a few years ever more powerful computers will be commonplace.

Comments?
