### Post by jasonrandolph on 2007-06-25
I've been dealing with many of the same issues as you, so I'll offer my input.  First off, much of the info I got came out of *Beginning Ubuntu Linux* by Keir Thomas, an outstanding book for those moving from Windows to Linux, or at least setting up a dual boot system.  The Second Edition even covers Vista issues.

I know you can set up your iPod in Linux, but I am keeping iTunes in Windows for that.  It's probably too much to hope that Apple will issue a Linux version of iTunes.  There's another book, *Ubuntu Linux for Non-Geeks* that I do not own which covers iPod use in Linux, so you may want to check it out.  Personally, I don't think there's anything wrong with maintaining Windows for such uses.  I also use Vista to run Photoshop CS2.  While GIMP is good, it's not at the same of CS2...yet.  Once it is, I can't imagine keeping Vista on my hard drive.

Here are some replacements for the programs you asked about:

MP3=>Rythmbox
CD/DVD burning=>Nautilus
Antivirus=>Not really necessary, but Firestarter is supplied

The best part is that all these programs are free.

As for TurboTax, I don't think there's anything in the Linux universe for that, although TurboTax now has an online version that you can use right from Firefox.  I've used it for the last two years to do my taxes, and it works great, without any need to install anything on your computer.

I hope this helps.  I set up my dual boot system last week, and I'm using Windows less and less.  I hope you have as easy of a time setting up your computer as I had with mine!

### Post by TheFoe92 on 2007-06-25
To answer your questions...

> **pancakelizard said:**
> 
Is it true I really don't need a virus scan program? I know spyware and such won't be an issue.


Yes, it is basically true. Although Ubuntu is extremely safe, you might want to download a firewall, named Firestarter. The easiest way to get it is to go to Applications > Add/Remove.  Then in the textbox type in Firestarter and make sure 'All available applications' is shown right beside the textbox under 'Show'. Check the box, click OK. The stuff will download and then install. To set it up, it is under System > Administration.

> **pancakelizard said:**
> 
How do I bring up a task manager? Pressing ctrl + alt + del did nothing


You have to go to System > Preferences > Sessions. Here you can add or remove startup programs. By clicking the Current Session tab, you can see all the programs that are currently running.

> **pancakelizard said:**
> 
What about turbotax?


Go [here]("http://www.winehq.com") and click Get Wine Now under Download. This is a program that allows you to run many Windows programs on Linux. Some programs do not run fully. For example, I installed Photoshop 7.0 on my system. Everything works fine except the text tool. You can go [here]("http://www.frankscorner.org") to see many available Windows programs that can be run on Linux. TurboTax might not work on Ubuntu, though. In that case, make sure you install Ubuntu with Windows (dual-boot). Then, you can run programs that can't be run in Ubuntu, on Windows.
