### Post by FakeOutdoorsman on 2007-12-11
How long has this error been showing up?  Have you ever been able to run Ubuntu? This could be bad ram.  First and easiest thing to try is reseat the ram and insert your Ubuntu installation disc.  When the install menu appears run memtest.  This will check for bad ram.  If the ram seems fine then take a look at these threads:
[URL="http://ubuntuforums.org/showthread.php?t=135920"]
