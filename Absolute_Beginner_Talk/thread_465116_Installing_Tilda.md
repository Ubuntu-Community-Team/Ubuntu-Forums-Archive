### Post by meatpan on 2007-06-05
It looks like there are various versions of tilda available in debian repositories:


[http://packages.debian.org/cgi-bin/search_packages.pl?keywords=tilda&searchon=names&subword=1&version=all&release=all](http://packages.debian.org/cgi-bin/search_packages.pl?keywords=tilda&searchon=names&subword=1&version=all&release=all) debian.org search results

Do you have admin privileges on your machine?  If so, you should be able to use `sudo apt-get install tilda`.  Graphical utilities frequently have a lot of dependencies, so don't be surprised if apt recommends some more packages.
