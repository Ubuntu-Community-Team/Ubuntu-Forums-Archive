### Post by 3rdalbum on 2008-01-09
You can run anything that is compiled for PowerPC architecture. Most things that are source code will compile and run on the PS3. Of course, Gnash works, but not Flash. There's no Skype or Sun Java, but you can try Ekiga and IBM Java.

EDIT: The reason why you can't install those debs is because they have been compiled for the x86 (i386) architecture. The PS3 uses the PowerPC or PPC architecture. Debs that have been compiled for powerpc are difficult to find, so you'll have to rely on the Ubuntu repositories and what you can compile from source code.
