### Post by GS2 on 2007-05-21
> **rich.bradshaw said:**
> Surely it should be their problem, not ours? Why should I waste my system resources running antivirus?

I see this type of comment posted in *nix users on forums across the web, and I am dismayed when I see it, from experienced users to first time ones - and in my opinion security starts and ends with the end-user (and their attitude towards the security of their system).

Indeed it could be argued that you 'wasted' your system resources by making that post, it was certainly not in the spirit of these forums, or helpful towards the original poster's question.

I have installed and run ClamAV, AVG, and Avast on Ubuntu - without problems. I found Clam is more suited to an email server, but AVG and Avast seemed easy to use. My own preference was AVG, I preferred the interface - but that is a personal opinion.

Try both (of course not at the same time) and see which you prefer.

Incidentally here is a lengthy, but informative read on malware - which the 'It isn't my problem' school of thought should read:
[http://www.kernelthread.com/publications/security/index.html](http://www.kernelthread.com/publications/security/index.html)
