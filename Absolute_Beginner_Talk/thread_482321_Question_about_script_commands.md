### Post by freakavoid on 2007-06-23
Sure there is. And it's very powerful in comparison to windows since nearly all the programs have useful command line options. [LinuxCommand.org]("http://www.linuxcommand.org/index.php") is a good place to start. 

Furthermore there are engines for common scripting languages like perl and python preinstalled. You might want to check them out as well.
