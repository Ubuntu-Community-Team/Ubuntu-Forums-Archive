### Post by crav on 2007-04-27
Ubuntu, Kubuntu, and Fiesty Fawn are all essentially the same thing.
Ubuntu is the centralized project. It's currently in version 7.4.
Ubuntu the OS is a linux system with the Gnome environment.
Kubuntu is the same as Ubuntu, only it uses the KDE enviroment.
Fiesty Fawn is the code-name for Ubuntu 7.4 (the current version)

I would recommend getting Ubuntu, simply because the user base is larger, and you'll be able to get a lot of help. Kubuntu (KDE) however is often hailed as being more windows-like.

### Post by trishacupra on 2007-04-28
Welcome. I just converted over myself.

Definitely try out the Live CD first. It doesn't touch your Windows installation. It just runs a bit slower because the operating system is on the CD, not on your hard drive.

Do you know about dual-booting with Windows? I have both Windows XP and Ubuntu Feisty Fawn installed together. You may want to consider doing that until you're used to Ubuntu.

What do you use your computer for? Which programs do you use? If you're not already, start using Firefox to browse the internet instead of Internet Explorer, and if you use Outlook or Outlook Express, try Thunderbird for Windows. If you use Microsoft Word as your word processor, try AbiWord or Open Office Word Processor.

I think you can install these programs (all free) off the Live CD if you put it into your CD drive while using Windows. If you find you can use these programs easily, then moving to Ubuntu will be much easier than if you have to learn both new programs and a new operating system at the same time.

Oh, and make sure you have a high speed internet connection with an ethernet modem. A dial-up user would have some issues even with getting their modem recognised from what I've read, and I had to change from a USB external modem to an Ethernet one because it just wouldn't work. 

If you have any questions, just ask. You can also find a lot information by searching Google.com or these forums. Though sometimes you just have to ask here so you actually know what to look for. :)

All the best!
Trisha
