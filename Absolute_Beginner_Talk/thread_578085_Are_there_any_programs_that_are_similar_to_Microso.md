### Post by rsambuca on 2007-10-16
Go to your top menu bar and select "System -> Administration -> Synaptic Package Manager".  Select Scribus for installation, and then select "apply"
After it has done its thing, you will find the Scribus launcher under "Applications -> Graphics".
