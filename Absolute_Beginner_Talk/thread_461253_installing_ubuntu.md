### Post by Cypher on 2007-06-01
Grab the Live Desktop CD, pop it in, load it up, double-click the Install icon, following the dialogs. Use the default answers if you don't want to mess with anything. Remove the cd, reboot and you're all set.

### Post by drs305 on 2007-06-01
As Cypher said, the live CD will walk you through setting up your system. However, before installing Ubuntu you may wish to explore this forum to get an idea of how you want to set up your hard drive partitions. Do you want a separate partition for your home directory; if so, how large? 

A few minutes of thought before installation may save you time later. The good news is that even if you accept all the defaults you will end up with a fine OS once you get everything to your liking!

A good introduction to setup options can be found here:
[http://www.psychocats.net/ubuntu/index.php](http://www.psychocats.net/ubuntu/index.php)

Welcome to ubuntu.
