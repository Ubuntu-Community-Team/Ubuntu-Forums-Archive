### Post by gn2 on 2008-01-20
As it's a 7.10 version you should be able to run the installer on it's own without booting a live session by pressing F6 at the splash screen and typing ```
only-ubiquity
``` as an additional boot parameter.

Don't know for certain if this works with Fluxbuntu though.....

EDIT: If you use an Ubuntu Alternate CD you will need a working internet connection to install the Fluxbuntu GUI.
