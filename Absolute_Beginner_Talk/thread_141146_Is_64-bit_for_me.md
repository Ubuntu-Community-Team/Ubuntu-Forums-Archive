### Post by el3ktro on 2006-03-07
To keep it short: Unless you don't do any heavy mathematical/computational stuff like simulation of nuclear bombs ;-) or if you _need_ more than 4GB RAM you won't really _need_ 64 bit right now. But, when you buy an Athlon64, you'll probably get the best performance for your money,  and because those 64 bit processors are simply newer than their 32 bit pendants, you'll also have some other advantages which are not related to 64 bit, like general better performance (higher clock speeds), this NX flag stuff, power saving features etc.

Well if it's needed or not, 64 bit will be the future on the desktop, so if you invest a lot of money in a new computer, then better buy one which will likely last longer, and this is 64 bit now (socket 939 is a good choice).

### Post by jjf on 2006-03-08
I had a number of compatability problems with the AMD64 version of Ubuntu.  The two most notable: (1) no Automatix (a very useful utility for easily installing a couple important packages), and (2) no Macromedia Flash player.

I'd bet that with enough gusto and help from the pros you could get everything working, but if you're considering purchasing a computer just for Ubuntu, I would definitely steer clear of 64 bit.  The only advantage I can see to getting 64 bit is  better gaming performance, which is a task Linux is not well suited to, anyway (IMO).  For most daily computing tasks you probably will not notice any difference otherwise.
