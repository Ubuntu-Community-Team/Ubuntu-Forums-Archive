### Post by FleetAdmiral74 on 2007-06-06
This should get you started...

First, in your BIOS look for some boot options that pertain the boot order 1st, 2nd, and so on. Just flip the cd drive over to the top, and the hd under it. walla, should boot from cd first.

so download the iso. once you have it, use your fav. program to burn. if you have nero, there is an option in the smart start thing (backup) to burn as image. If not, try a free program like ImgBurn (free!!!), and just google instructions imgburn burn iso, or something like that. Once it is done, just put it in and reboot, should be good to go with a live desktop, so you can see if things work w/o messing anything up.

get back to us on how it goes!

edit" i think in imgburn, you can just go to the mode tab on top, choose write, open the iso file, and then burn.
