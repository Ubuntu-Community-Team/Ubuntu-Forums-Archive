### Post by mac.ryan on 2007-03-22
First of all, welcome to the world of linux, and of the free(dom) software! :)

Due to a different-than-linux (and I would say debatable) philosophy in managing file permissions, you can never be sure all your configuration and data files have been kept in the same place, under windows. Anyhow, the idea in windows is that all your configurations and settings should stay under "Documents and Settings" folder.

In case you have installed additional fonts, you might wish to backup also "Windows/Fonts".

You should double-check you saved all the data of those programmes that might have been installed outside "program files" (if the software didn't use the standard directory to install itself, there are chances it neither used standard directory to save data and settings...)

I used Ghost some 5 years ago, so things might have dramatically changed, but if they haven't, then you should be able to save a complete image of your HD partition, which should be a 100% safe way of backing up, and the easiest way to recover your system should something goes wrong.

Best luck with your journey into ubuntu!
