### Post by kpkeerthi on 2008-01-29
Ubuntu repositories will be updated only if there are security fixes/critical updates to a package upstream. 
A new version of a package will usually be added to the repository when the 'next' official ubuntu release is available i.e., new versions of softwares are usually made available once in 6 months.
