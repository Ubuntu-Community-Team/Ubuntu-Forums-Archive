### Post by Paul820 on 2007-11-13
To install a program with wine, right click on the .exe file and choose properties->open with and look for wine. It only lets you choose the c drive is because wine has it's own little world :) Everything you need to run windows apps is contained within there and no-where else on the system. When you have the application installed there will be a menu entry for it under Applications->Wine->Programs
