### Post by bone2006 on 2007-12-30
I have dd-wrt on all my routers and I love dd-wrt, but you'll probably get better help for dd-wrt at dd-wrt's forum than at ubuntu's forum.  In all honesty I don't know what a lot of the features do, in which it's probably better just to look for what you need.  If it's security, if it's something specific
For instance I have a dynamic ip address, dd-wrt has a feature that allows me to update no-ip.org or other DNS servers to update my ip to basically a static one so I can run an SFTP server and a web server.  It has other encryption and features that I use that other routers don't.  I actually use dd-wrt has a client as well for a linux box, instead of buying a device and trying to get the drivers to work I hook it up as a client, so that the area I have it, it comes with 4 ports as an ethernet hub and I have wifi access.

Instead of purchasing a wifi pci card or usb card that might not be linux friendly, I use dd-wrt on a cheap 30 dolllar buffalo router and no drivers are required and not only am I using it as a client, but it has other ethernet ports that I can connect to if I have a laptop.

A friend at an apartment uses the scripts in there that you can get a script that will scan your neighborhood for an open wifi signal, it takes the strongest and if it drops it instantly scans for another open up.  The features and capabilities of the scripts are really impressive.
