### Post by Steveway on 2007-08-10
Firestarter is NOT a firewall.
It is a gui-frontend wich helps to configure Iptables.
Iptables is a firewall and it is in the kernel, by default!
The normal Settings are good for a normal Desktop-user and don't really need to be changed.
