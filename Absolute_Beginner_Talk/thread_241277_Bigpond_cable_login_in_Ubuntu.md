### Post by 3rdalbum on 2006-09-03
> **sologeoff said:**
> Help I am Newbie
WHY DO UBUNTU PROGRAMERS MAKE IT SO DARN HARD TO LOG ON WITH BIGPOND CABLE ETHERNET.


You're not going to like the solution.

Go to [http://packages.ubuntu.com](http://packages.ubuntu.com) and download "bpalogin". Transfer it to the Ubuntu machine and install it. If it requires dependancies, download those and install them too.

BPAlogin should give you access to Bigpond.

FYI, Bigpond is the only cable service in the world where your connection is dependant on whether you have a certain program running on your computer. Telstra are going to start phasing this unreliable, substandard technology out, so hopefully your modem will automatically connect some time in the future. I don't think it's fair to blame the Ubuntu developers for not building in something which is only required for one particular service in one particular country.

For more information, see [www.whirlpool.net.au](www.whirlpool.net.au)
