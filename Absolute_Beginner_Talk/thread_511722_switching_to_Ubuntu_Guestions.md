### Post by por100pre1 on 2007-07-28
Start installing Windows. As you said, Windows does not like to be second fiddle in a dual boot system. Then shrink it's partition so you get free space for Ubuntu. If you install Vista shrink its parttition with Vista's own Disk Manager. If you install Xp you can use Gparted in the Ubuntu's Live CD or Partition Magic or similar program if you have one. 

Hope this helps and welcome to the forums.

EDIT: By the way try 7.04, but if you give us your system's specs, we can give you better advice...

### Post by rillip on 2007-07-28
Agreed that Windows should come first, but you can create a small parition for it from its installer rather than having to resize it later.  The reason for this is that the Windows installer overwrites the boot record on each install, and Windows installer isn't designed to easily allow for choosing another OS, whereas Grub is designed for that from the Linux install.

I would install 7.04, and if you ran into problems, fall back on 6.06.  6.10 was a "problem child" for many.
