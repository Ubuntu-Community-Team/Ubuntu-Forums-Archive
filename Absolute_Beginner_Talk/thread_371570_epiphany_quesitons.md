### Post by Quillz on 2007-02-27
For your first question, you are looking for a session saver. This is built in to Firefox 2 and Opera 9, although I'm not sure if Epiphany has something similar, as I don't believe it has a session saver as a stock feature, nor does it appear to support extensions.

As for you second option, this should be possible via the View menu options.

As for drawbacks... Epiphany is based on the Gecko engine, as is Firefox 2, so not having extensions is kind of a downer for me. Also, the way I see it, all that Epiphany really has going for it is seamless integration into the GNOME user interface, although there are many themes for Firefox that look just as good, such as the Ubuntu Human theme.
