The live CD (more properly referred to as the DESKTOP CD) is the installation version that you can boot to and run directly from your CD drive and then when you get to the "DESKTOP" there is an installer on there which allows you to actually install the O/S to your hard drive so you can boot from and run the O/S from your hard drive.

IMO, running the O/S from the CD drive does not give you a very good picture of the capabilites of Ubuntu.  Install it on your hard drive as soon as you are comfortable with it.

I recommend installing it on a machine with ONLY Ubuntu as the O/S and not a dual boot situation where you have both windows and Ubuntu/Linux.  If I am going to run windows, I want it on one machine and Ubuntu on a separate machine.

Good luck.
