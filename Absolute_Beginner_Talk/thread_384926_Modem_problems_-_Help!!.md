Hi, there!

I'm not sure if this will be of help, but it's worth a try.

[http://ubuntuforums.org/showthread.php?t=190728&highlight=install+winmodem+conexant](http://ubuntuforums.org/showthread.php?t=190728&highlight=install+winmodem+conexant)

Most Winmodems are very strictly tied to Windows and usually cause problems for any other OS.

For a long-term solution, I would recommend that you consider subscribing to a DSL service, as you will find that dial-up is painfully slow, if there are some substantial upgrades, or you wish to download packages, using Synaptic.

Remember, though, if you do decide to go for ADSL, make sure that the router you get is a 100 base T, not a USB box, which could cause you grief. Some ISP's supply a router and you may need to pay a little more to get 100 base T.
