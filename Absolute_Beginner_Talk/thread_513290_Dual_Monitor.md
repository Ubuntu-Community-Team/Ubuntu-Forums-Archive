Yea, that is how firefox works ;)

BUT you can :

1. Open a new window (form firefox File -> New window)



2. Create another profie

In a terminal :
```
firefox -P
```

This opens your profile manager

	Make a new profile, say Test

	Then:```
firefox -P Test
```

	You need to download themes and extensions for each profile 

3. Or in a terminal :

```
/usr/bin/firefox -p -no-remote
```


You can make a new entry in your menu for which command you prefer ...
