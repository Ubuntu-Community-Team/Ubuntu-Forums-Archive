### Post by WinterWeaver on 2008-03-06
After downloading a ISO, and before burning it, you should run a md5sum on it to make sure that the iso is ok. Sometimes it might get bad during the dowload.

Some of the Ubuntu versions have a option on the initial boot screen, to check the disk. Choose that option and it verifies the disk for you. If you see any errors, chances are you will run into some problems during your live session, or installation

EDIT: Here is a link to MD5 Sum Howto. Use this do check the ISO before burning it.
[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

Hope it helps,

WW
