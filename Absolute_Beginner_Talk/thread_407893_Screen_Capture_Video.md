### Post by ComplexNumber on 2007-04-12
there are 2 that i know of:
-istanbul (its in the repositories, so you can download it via synaptic or apt-get)
-recordmydesktop ([click]("http://recordmydesktop.sourceforge.net/downloads.php")). download the appropriate deb package for your system. then double click on it to install it. note that it neds recordmydesktop and gtk-recordmydesktop to work. the latter is a GUI frontend for the former.

i've found the latter to be the better of the 2 because istanbul seemed to be temperamental on my system. then again, it depends what works for you.
