### Post by mcduck on 2007-07-30
Mplayer-plugin (install from Synaptic) can handle all of those, and supports embedded videos so you don't always need to open a separate player.

Just install mplayer-plugin, all codecs and, remove totem-plugin (as it seems to 'steal' some file types from mplayer-plugin and is not needed anyway).
