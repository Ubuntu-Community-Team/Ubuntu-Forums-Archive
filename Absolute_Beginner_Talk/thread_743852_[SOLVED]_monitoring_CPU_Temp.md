### Post by spiderbatdad on 2008-04-03
Hi. There are many ways, but for something simple you could easily see during gameplay,
you could install sensors-applet
```
sudo apt-get install sensors-applet
```

Then right click on your panel and select add to panel...find the "hardware sensors monitor" and select it. Now you can right click the new applet and select preferences to set celsius fahrenheit and some other options.
