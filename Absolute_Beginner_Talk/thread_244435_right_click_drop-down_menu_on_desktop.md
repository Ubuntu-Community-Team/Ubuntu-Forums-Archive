### Post by NESFreak on 2006-08-26
you can create nautilus scripts (dumped under rmouse-->scripts)
[https://wiki.ubuntu.com/NautilusScriptsHowto](https://wiki.ubuntu.com/NautilusScriptsHowto)
or you could install the nautilus-open-terminal package, wich gives you a nice open terminal here option.
```
sudo aptitude install nautilus-open-terminal
```
