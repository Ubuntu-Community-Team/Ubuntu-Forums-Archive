### Post by orb9220 on 2007-06-20
I believe that is going to be it for feisty. Since Disto's such as feisty will not do major package upgrades from 2 to 3 versions only minor up grades such as .0.0.3 to .0.0.4 for stability concerns.

Major upgrade of an Application is saved for next distro as in Gutsy which has Pidgin as default install.

The reason for this is Stability for the masses. If you are one of those that need to have the latest greatest version then you will have to:

1) Find a .deb somewhere else where somebody took the time to package it.
2) Compile it yourself

That is the only way I know to have the latest of an application.

The only time I would upgrade a application is to fix a problem I am having or they added a feature that I need.

I have many customer's that have trashed a system because they have that "I need to have the latest upgrade fever" and can't even explain to me why they upgraded.

 Just need my upgrade fix sort of thing.

I am not saying you are of that type.
Just you'll have to do a bit of work if you want the latest.
