### Post by CREEPING DEATH on 2008-04-21
Try the Debian Lenny XFCE PPC disk.  If you want Sudo instead of Root, hit ESC when the installer asks for a Root password and select "no" when it asks about Root.
Man I'm starting to sound like a broken record, but everybody wants to use *buntu when there isn't that much difference!  Xubuntu has gotten rather bloated lately, it isn't officially supported, and PPC isn't supported either.  The last Xubuntu PPC attempt I made was horrible, I ended up installing Debian after hours of frustration and non-functioning.
Keep in mind, there is NO Flash support for PPC Linux!
