### Post by PeterJS on 2008-01-18
I heard mention that there's a new outbreak of something on windows and a couple threads poping up here. Don't worry about it. Most viruses spread one of two ways: 1) Intentional user interaction 2) system flaws. As long as you follow good security practices the first one is a non-issue. As for the second one, the vast majority of the system flaws exploited are in outlook express and IE, neither of which run on Ubuntu.

Supposing the virus some how managed to overcome the above. The virus payload would be designed to run on windows, it would expect windows programs, windows file layouts, and would be in a format executable  on windows, which is different than the format linux uses. Take a look at some of the wine threads, even intentionally with the user trying to get things to work it's hit or miss. Viruses generally rely on really weird  code tricks to get places it shouldn't, and wine either doesn't support those exotic functions or does so in a safe way, and even if the wine method is unsafe it probably unsafe in a radical different way that it's windows counterpart.

That said here's the Ubuntu Security Guide:
[http://ubuntuforums.org/showthread.php?t=510812](http://ubuntuforums.org/showthread.php?t=510812)

Good practices also go a long way. Set cookies to always ask, and look at these FF extensions:
Noscript:
[https://addons.mozilla.org/en-US/firefox/addon/722](https://addons.mozilla.org/en-US/firefox/addon/722)
Adblock Plus:
[https://addons.mozilla.org/en-US/firefox/addon/1865](https://addons.mozilla.org/en-US/firefox/addon/1865)

### Post by charleseddy on 2008-01-18
You're correct that most viruses are targeted to Windows, and that's why PeterJS' post is so relevant--it's nearly impossible for a Windows virus to permeate a Linux system.  Glad you hate Mr. Gates as much as I, and I hope you can enjoy Linux's advantages.  If you have tech problems, you can, of course, personally e-mail me.

There is also another reason viruses don't work well on Linux.  Viruses that are designed for Linux (there are currently, at my count, 23--compared to Windows' 30,000), need to have a root (administrator) password just to *think* of installing them, and even then there are virus scans to catch those 23 threats.  That is the reason that Linux viruses have never been propegated, even though a few exist; if you have to enter a password, it's hard to install them.  It's like saying, "I want to install this virus."

I certainly hope you don't do that!!  Anyways, have no worry, and have a great day.
