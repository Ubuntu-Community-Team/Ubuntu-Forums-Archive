### Post by callan79 on 2007-08-09
> **philinux said:**
> The way I went was to get a second hard drive, I just dont like partitions. (just me) and HD's are so cheap now. Linux is not windows is a phrase that gets a lot of attention. But bottom line is it's a secure OS. There is a learning curve!


I would not consider myself an expert at all, but I've  set up a few Ubuntu machines now for family and friends and had success every time (except dad keeps making his lock up, but that's just him!)

Anyway in my experience you're best to reinstall your whole system if you can, it's cleaner that way, but if reinstalling Windows is not an option :

 - Move all your unnecessary data to removable drive and UNPLUG the drive!
 - Clean up all your temporary files (disk cleanup)
 - Defrag your drive, reboot, disk cleanup, defrag again
 - Use the Live CD to boot Ubuntu, and install. Resize your Windows partition leaving a bit (say, 20GB) of free space
 - During install, create a Primary EXT3 partition for the system files (mount as / )
 - Create partition 1GB for swap
 - Create partition with the balance for /home
 - Install System

When done, boot into Windows, and get the driver from [www.fs-driver.org](www.fs-driver.org) so that Windows can read the contents of the /home area

There are a couple of alternatives, but that set up is basically what I have, plus another 160GB on the drive which provides shared storage space between Windows and Ubuntu.

Regarding learning curve ... yes, there is one. The system is easy to use, but you need to give yourself a good month to get used to it - but be patient, if you just boot Windows because it's easier, you'll never learn!

Good luck.

### Post by lgambett on 2007-08-10
Free space is not the same as not partitioned space. To install Ubuntu you will need not partitioned space; thus you have to shrink your Windows partition. You can do this with the Ubuntu desktop CD, but I personally prefer this process;
1) Defrag the Windows system
2) Chkdsk the Windows system
3) Resize the Windows partition; Vista has its own partition resize utility, while XP has not. You can use a commercial program like Partition Magic or use a program running under Linux. I suggest using Knoppix as explained below.
3a) Download and burn a copy of Knoppix live CD
3b) Start your system with Knoppix live CD
3c) In the Knoppix menu find the Qtparted utility and shrink the Windows partition
4) Now you can start the Ubuntu install process.

### Post by nvteighen on 2007-08-10
Wait a bit, you can resize your Windows partition from your Ubuntu Live CD... in two different ways!

1) go to "System --> Administration --> GNOME Partition Editor" (also known as GParted)

Or

2) Run Ubuntu's installer and when arriving to the partitioning stage, chose "Manual" (the third option). This will open a GParted frontend, but this is not very recomendable because you might get a dumb (non-serious, but disguising) issue with your Windows partition (it will automount itself by default and you'll have to edit /etc/fstab)
