### Post by matthewcraig on 2007-11-11
This is a good question!

What you are seeing is two different operating system "architectures".  Applications written for the Windows XP architecture will not run on the Ubuntu architecture.  Most people will find the functions they need in applications written for the Ubuntu architecture, by searching in the Synaptic Package Manager tool.  It is possible to jury-rig an application for the Windows XP architecture to run on Ubuntu through programs called DOSBOX and WINE, but these are not at all easy to use and may not provide 100% functionality.

Good luck with your Ubuntu experience.  Hope you will find the compatibility limitiations are off-set by the lack of spyware, viruses, and license limitations that are normally in Windows XP operating systems.
