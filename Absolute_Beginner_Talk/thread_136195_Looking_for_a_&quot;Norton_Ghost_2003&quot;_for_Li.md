### Post by dvarsam on 2006-02-25
Hello ALL !!!

For a Long time, I have been using a Norton Ghost 2002" _Floppy_ _Disk_, when I wanted to Backup a Windows OS Partition.

It worked great with NTFS & FAT32 Partitions.
However, it did NOT support EXT3 Partitions.

During that time, I did NOT care about the EXT3 part...

But, now I have moved into Ubuntu Linux !!! ( Yupppiiii !!! \\:D/ )

... and I am very happy about it !!!

I know that Norton Ghost 2003 _Floppy_ _Disk_ is supposed to be compatible with EXT3 Partitions...

For some reasons though, I can not use a Norton Ghost 2003 _Floppy_ _Disk_ nor for Windows OS or Linux OS Backup.
When I run from the Floppy "Ghost.exe", the program freezes up & does NOT let me Backup anything!

And you will probably think tha it is a corrupted Floppy Disk.
Noooo, it is NOT a corrupted Floppy Disk, the Disk is fine!
I have tried it with other Floppies too !!!
It still freezes up !!!

Right now, I am Using Ubuntu _completely_ & I am very happy about it!

However, there are many things I can NOT make work in Ubuntu:

1. Printer color printing is bad...
2. Setting up the Infrared Adaptor to work...
3. APC Backup UPS does NOT have drivers for Linux (that is not so important)
4. Have to setup the USB Camera... (do NOT know how yet)...
5. Nokia software to Backup my Contacts through Infrared...
6. And possibly other stuff too...

And I know it is NOT Ubuntu's fault...
... and it's all the "Silly" companies that SHOULD have created Drivers for the Linux world...

All the above, unwillingly, force me sometimes to go back to (stinky) Windows... bliahhh!!!

So basically, since I am going back&forth from one OS to the other:

I am still in need of some Floppy Disk or CD-Rom that can help me in **Backing Up** Partitions (NTFS, FAT32 & EXT3).

Pleae do NOT suggest to me to install Software in my Ubuntu or Windows.

I dislike installing Software in my PC. (am always scared of the conflicts, etc etc.)

I want to be able to do the Backup Procedure (Clean & Tidy) like this:

A. Shut Down the PC:

1. Plug-in the Hard Drives (Source & Destination)
2. Boot from: Floppy Disk or CD-Rom
3. Run a program (similar like) Norton Ghost 2002
4. Copy from Partition to Partition (Backup)
5. When Finished, to Shutdown

B. Booting again:

1. Remove from the Drive: Floppy or CD-Rom
2. Unplug the (no longer needed - Backup) Hard Drive
3. Boot from my original Hard Drive to whatever I want: Windows OS or Linux OS


Is there something out there you can suggest?

Please Help...
