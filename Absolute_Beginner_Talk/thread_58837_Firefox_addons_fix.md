### Post by matthew on 2005-08-21
Open firefox

In the address bar type "about:config" and hit enter

Scroll down to the line that says "general.useragent.vendorSub" and enter the current version of firefox there. I believe it is 1.0.6 at the moment.

Close firefox completely. The next time you use it it should work for you.

----or----

Use the latest firefox version from the repositories as it works fine.
