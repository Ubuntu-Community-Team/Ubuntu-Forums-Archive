In Linux the user does not work as administrator by default. so virus and other malware will find it difficult to install. User normally uses sudo command to work in root directory.

Also since Windows is more popular at present, more malware / virus r being generated.

Linux being open source, bugs and security vulnerabilities are fixed extra fast.

Linux is generally used by more computer savvy people as compared to Windows.
