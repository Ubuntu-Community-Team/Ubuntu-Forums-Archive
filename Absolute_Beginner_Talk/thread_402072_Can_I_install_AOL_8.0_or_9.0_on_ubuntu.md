There's a sleight possibility that you could make it work through wine, but don't get your hopes too high.

There's an AOL dialer called Pengaol that worked on my old Red Hat system, but it's no longer maintained and I don't think that it works with Ubuntu. Also, I've heard that the Linspire folks have a working AOL dialer, but I don't know how to get it. I think that you can install their CNR tool these days on Ubuntu, which might allow you to get it.
