### Post by Brightbelt on 2007-09-15
A few tips I found helpful: 
- Format the Hard Drive as 'Ext3' and set the mount as '/' - that's right: just a slash. That signals your main drive for Ubuntu.

Ubuntu may want to set up a Swap partition; that's perfectly normal. 

Everything else is fairly easy. The guy who said this is an easier install than MS is telling the absolute truth.

If I had known Linux was this easy to install as an OS, I would have tried it years ago.
