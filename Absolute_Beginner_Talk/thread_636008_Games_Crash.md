### Post by sourcemorph on 2007-12-09
i suppose you're talking about playing games on wine.

first go to the wine configuration editor ("winecfg") and in the graphics tab deselect the "let the window manager manage windows" thing. if u play the games fullscreen u might have already done it..

anyway if this hasn't solved the problem then try getting the right drivers for ur graphics card by using "envy" or trying updating wine (the newest version is 0.9.50).
