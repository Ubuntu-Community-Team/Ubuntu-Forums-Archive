### Post by kvidell on 2005-07-11
[QUOTE=DenniiZ]Does Ubuntu read the files like Word docs, excel, jpg, bmp, .. ?[/QUOTE]There's a sleugh of choices for opening word doc type files and other office formats.
OpenOffice is one choice, and for word documents specifically I think you can also use AbiWord if it suits you better... but I think OpenOffice is most people's choice right now.

As for jpg, bmp, png, gif etc... Of course :) Those are just rendered image files. Any modern OS should have libraries to be able to view those.
You'll want something like GQView to just view them, and The Gimp to edit them.
