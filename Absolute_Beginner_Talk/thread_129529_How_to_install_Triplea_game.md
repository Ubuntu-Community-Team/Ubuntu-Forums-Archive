### Post by Sandlst on 2006-02-14
If you downloaded the zip file (from [http://prdownloads.sourceforge.net/triplea/triplea_0_7_3.zip?download]("http://prdownloads.sourceforge.net/triplea/triplea_0_7_3.zip?download")), no installation (other than unzipping the folder inside) needs to be done-simply open up a terminal (Applications/Accessories/Terminal), cd to the directory you unzipped from the file you downloaded, then type in 'sh triplea_unix.sh' (without the quotes).  If you wish to make a custom shortcut on the desktop or panel, right click on the desktop (panel) select "create launcher" (add to panel) then fill in the information as you like, being sure to put the full pathname under the 'command' section, or select 'custom application launcher' if you prefer to create a panel button.  Post back if you have any problems :)
