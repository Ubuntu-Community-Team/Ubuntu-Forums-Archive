### Post by zK9 on 2006-02-11
Most of the players on linux do not have the codecs installed for playing a large number of media files ( if that makes sense)
anyway  I used to have the same problem, couldn't play half my files until I found a player that could.
Mplayer
Anyway you can get it from [http://www.mplayerhq.hu/](http://www.mplayerhq.hu/)
install it and try though it ic console based so please refer to it's man page after installation.
the easiest way to play a file with it would be in the console type
mplayer <media file>

you can also download a GUI interface for it (though it seems to drop the sound volume).
