### Post by morphodone on 2006-01-31
You have to make sure your computer is setup to boot from the cd.  

This option is in the bios.  
Usually you can access the bios by hitting 'F8' or 'delete' just when you computer is first booting.

Then just select the first boot option as your cd drive.

Reboot with the ubuntu live cd in the drive.
