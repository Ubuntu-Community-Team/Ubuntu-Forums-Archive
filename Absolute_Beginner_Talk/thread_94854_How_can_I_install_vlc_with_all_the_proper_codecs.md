### Post by frodon on 2005-11-25
You can install vlc with synaptic, then all the player will share the same libraries.
This is a collection of codecs, you shouldn't get any problems in the future with those codecs installed : w32codecs, libdvdcss2, gstreamer0.8-plugins, gstreamer0.8-lame, gstreamer0.8-ffmpeg, gstreamer0.8-faac, gstreamer0.8-faad, libxvidcore4, lame, sox, ffmpeg, mjpegtools, vorbis-tools, mpg321.

If you're a newbie, it might be easier to run automatix : [http://ubuntuforums.org/showthread.php?t=66563](http://ubuntuforums.org/showthread.php?t=66563)
