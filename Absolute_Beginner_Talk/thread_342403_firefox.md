### Post by Stephen Howard on 2007-01-20
Nine times out of ten, selecting [FONT="Fixedsys"]'Direct connection to the internet[/FONT]' should be sufficient.  To do that, use the [FONT="Fixedsys"]Edit | Preferences [/FONT]menu in Firefox.  Click on the '[FONT="Fixedsys"]Network[/FONT]' tab and then click the [FONT="Fixedsys"]'settings'[/FONT] button and select the [FONT="Fixedsys"]'Direct connect[/FONT]' option.

If the Firefox option is already set to [FONT="Fixedsys"]'direct connect to the internet'[/FONT], then you'll need to compare it to the equivalent setting in Konqueror.   Start up Konq bring up the [FONT="Fixedsys"]Settings | Configure Konqueror[/FONT] menu.  In the left panel there is an icon labelled [FONT="Fixedsys"]'proxy',[/FONT] click on it then click on the proxy tab.  Write down the settings (presumably it won't be [FONT="Fixedsys"]'direct connection to internet'[/FONT]) and duplicate them in the equivalent fields of firefox Network dialog box.
