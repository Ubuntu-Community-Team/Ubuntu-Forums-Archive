### Post by llamakc on 2007-11-29
> **patchido said:**
> ok.... in a how-to i am asked to use the command su
when i do this asks me for my password i suppose for the root pass but when i put it it says authentication failure..

The failure is what should happen. Why? Because Ubuntu does not enable the root account by default. Instead, use the command "sudo" in place of 'su'. One can also use "sudo -i" to get a root environment for a period of time.

[https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo)
