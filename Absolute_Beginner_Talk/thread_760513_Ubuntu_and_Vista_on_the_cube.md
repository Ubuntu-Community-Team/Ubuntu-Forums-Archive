### Post by sayakb on 2008-04-20
Install vmware or virtualBox. Install Vista on it and move it to another workspace (perhaps, Desk 2). So you would have Vista on Desk 2 while Ubuntu on others.

EDIT: For running Vista on VBox, you need a handsome amount of RAM. (Maybe, atleast 2GB)
