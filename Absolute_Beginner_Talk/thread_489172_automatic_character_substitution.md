### Post by Pizzarth on 2007-07-01
I've found it. Nevermind 

for other users:

System>administration>Language Support, check off the languages you want, and then next time you boot up X (just ctrl+shift+backspace) it'll show up as a small icon in the panel bar :D from there I can choose the language to type
