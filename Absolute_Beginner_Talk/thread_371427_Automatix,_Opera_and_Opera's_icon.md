### Post by shareMenaPeace on 2007-02-26
Rightclick the menu (Applications) -> Choose Edit Menu -> Choose Internet -> Choose Add item 

Here you need to enter the comman dto start and Infos. If you install with automatix, than use automatix to uninstall too. As Synaptic didn't update automatix i think.
