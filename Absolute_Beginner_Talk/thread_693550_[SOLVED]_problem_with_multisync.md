### Post by aBitLater on 2008-02-14
in multisync, edit the sync pair, and on the evolution side of the pair, choose options.  there you should make sure to choose 'personal' in the drop-down box for calendar, addressbook, and tasks.

'Personal' is the default name for those items in evolution.  If you have other calendars in evolution, with different names, they should show up in the drop-down box.  Choose the calendar / addressbook / task file that you want to sync your phone with.
