If I understand this one correctly, here is how I do it. I've made launchers for most used programs to top panel (on menu, right click and select "add this launcher to panel"), then I right click on the launcher, select properties and prefix command line with alltray. Eg: alltray firefox

This will put the program automatically to tray when "closed".
