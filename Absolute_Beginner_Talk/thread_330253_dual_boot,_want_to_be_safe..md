### Post by oilchangeguy on 2007-01-02
yes you will be given the option to set your partition. before you start back up your data, my documents, etc. if you're using xp pro the windows back up utility is installed by default. start, programs, assoc., system, backup. this will allow you to back up some or all of your hardrive. works best if you have another harddrive to back up to. in xp home it's not installed by default. and you'll have to use a windows xp disc (not a factory restore disc) to install it. don't remember the steps, try either [www.pcworld.com](www.pcworld.com) or [www.smartcomputing.com](www.smartcomputing.com) and search their archives. if you have a small amount of data you can use a cd or 2. just (for example) go to my documents and from the edit tab select all and save it to your cr-r/rw drive.

### Post by bruenig on 2007-01-02
Yes the install cd makes you go through partitioning, so you will be able to partition. Also remember that just because you put the cd in that doesn't mean that you are locked into installing. If you ever feel uneasy, so long as you havent pressed that last button to go ahead and install after setting up all your options for the install, it won't do anything. So if you feel uneasy, you can always just shut it down and go back into windows, not that you will.
