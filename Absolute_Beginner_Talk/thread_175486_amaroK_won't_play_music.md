### Post by bluevoodoo1 on 2006-05-13
Make sure there is an engine checked in the 'configure amarok' area. I use the xine engine without any problems and it should have been installed along with amarok. 

You can also check 
```
alsamixer
```

Use the directional arrows and the letter "m" to turn on/off the sliders. Hit escape to exit.

I had this problem when I first installed amaroK--a few sliders were off. Xmms worked, but not AmaroK,  and this solved it.
