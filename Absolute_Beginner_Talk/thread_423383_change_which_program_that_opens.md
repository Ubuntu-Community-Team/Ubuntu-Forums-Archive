### Post by Delkster on 2007-04-25
Right-click on a file of that type, select properties, and open the "Open With" tab. There you can choose which known handler for that file type should be used by default. If the application you want to use isn't listed, you can select "Add" and either select the application from a longer list or use a custom command.
