### Post by benuski on 2006-07-28
At least on windows XP (I can't remember what its like in windows 98), when my BIOS first loads, before the Windows screen pops up or anything, in the top right corner there pops up 2 options, to hit F2 or F10, which leads me to either set-up or to a one time boot menu.  Look up there really quick while your BIOS is booting, and see if you see anything.  If you can get to that boot menu (I think its F10), you should be able to tell it to boot from the CD this time.  But it won't affect your computer permenantely.

### Post by A2A on 2006-07-28
F10, DELETE, and ESC seem to be the three choices BIOS manufactureres use.  Try all of them and one should let you in to your BIOS settings.  Once there, look for Boot Sequence, and make sure you have CDROM set as option 1, and Hard Disk to option 2, (Floppy drive can be Option 3) Once installation's complete, don't forget to make Hard Disk the first option again.

A2A
