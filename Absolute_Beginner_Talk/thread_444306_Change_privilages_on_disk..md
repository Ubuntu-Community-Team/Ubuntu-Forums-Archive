### Post by taurus on 2007-05-15
So, looks like /dev/sda4 is that new ext3 partition.  Now, you need to change the ownership of /media/usr from root to your login name which so happens to be usr!

```
sudo chown -R usr:usr /media/usr
ls -la /media
```
You should be able to save your work to /media/usr now.

And if you want to mount /dev/sda4 to /media/usr each time you boot Ubuntu, you need to add an entry in /etc/fstab for it.

```
gksudo gedit /etc/fstab
```
And add this line to the end of it.

```
/dev/sda4   /media/usr   ext3   defaults   0   1
```
