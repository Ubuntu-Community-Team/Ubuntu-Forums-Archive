### Post by S15_88 on 2007-07-04
First before you start downloading a bunch of programs off the internet look in Synaptic package manager to see if they are available there or the add/remove section.  Also there's a program called automatix that has a bunch of programs too, all of which will install with a click of a button.  if u get a program off the net it may come in a form like .tar.gz or sumthin like that, these are easy to install.  place it in a folder or desktop or wherever, untar it by right clicking on it and extract here, navigate to that directory using : cd /home/blah/blah.  then make, make install.  i'm not 100% sure on this step that i've encountered before some ppl do ./configure but i've done that several times with programs and it returned an error and i proceeded with make, make install and it worked fine.  so i guess after i leave this little tid bit for u if anyone who reads this who has more experience than me who can elaborate on these steps i would greatly appreciate it as i am also very new to ubuntu.
