I still have it. I think it's related to the Snapping Windows plugin. If you're using the Wobbly Windows plugin Snapping Windows will be disabled because Wobbly also provides Snapping. So go into CCSM and go under the settings for the Wobbly Windows and uncheck Snap Inverted. Should get rid of the problem. However you will loose Snapping.
