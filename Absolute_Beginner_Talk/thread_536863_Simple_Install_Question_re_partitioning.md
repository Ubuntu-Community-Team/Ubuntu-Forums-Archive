### Post by marco123 on 2007-08-28
Yes.  That option will resize the windows partition and install ubuntu.  It will automatically create swap file as well.:)

Edit:  The slider will decide how big the windows partition will be so don't push it too far to the left.  You only need about 15-20GB for Ubuntu.
