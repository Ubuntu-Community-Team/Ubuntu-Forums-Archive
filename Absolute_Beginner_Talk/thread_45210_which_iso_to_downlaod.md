### Post by poofyhairguy on 2005-06-29
[/QUOTE]
ubuntu-5.04-install-i386.iso [/QUOTE]

This one installs Ubuntu on your hard disk.

> ubuntu-5.04-live-i386.iso  



This one runs Ubuntu (slowely) off you CD drive. Use this for evaluation before installing.

Look here for more info:

[https://wiki.ubuntu.com/BurningIsoHowto?highlight=%28iso%29](https://wiki.ubuntu.com/BurningIsoHowto?highlight=%28iso%29)
