### Post by Caffeine_Junky on 2007-11-12
Ok, try this:

In a terminal type this: ( or copy'n'paste)

sudo gedit /etc/X11/xorg.conf 

(type your login password if you are asked for it)

...now, scroll down to the section that looks like this:
```
Section "Screen"
    Identifier     "Default Screen"
    Device         "nVidia Corporation C51G [GeForce 6100]"
    Monitor        "Acer AC711"
    DefaultDepth    24
    SubSection     "Display"
        Depth       24
        [COLOR="Red"]Modes      "1024x768"[/COLOR]
    EndSubSection
EndSection
```

see in red?   just keep the resolution you want to use in that line and remove any others that are there...

Save the changes and reboot..

### Post by KentS on 2007-11-12
You need to figure out which graphic card you have first, then which driver you have installed/want to use. Then you choose accordingly. 
Might be easier editing your xorg.conf.
Do 
```
cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup
sudo emacs [OR WHATEVER EDITOR YOU PREFER] /etc/X11/xorg.conf
```

Then find
> Section "Screen"
Change 1024xwhatever to 1280xwhatever2.

Then Ctrl-Alt-backspace.

If it doesn't work (chrashes), do
```
cp /etc/X11/xorg.conf_backup /etc/X11/xorg.conf
```
