### Post by nakedfanatic on 2007-04-11
I never got the second monitor going with the default ATI driver loaded.

I did get it going with the proprietary ATI drivers (FGLRX).

After much frustration and trial and error, the instructions [HERE]("http://wiki.cchtml.com/index.php/Ubuntu_Edgy_Installation_Guide") seem to provide the only reliable way to get the latest driver up and running. Follow all instructions on the page and use the "Method 2" instructions to get the very latest drivers. It is worth the effort (all you are really doing is copy/pasting from the tutorial to the terminal window)

There are automated programs which claim to install and set up the best drivers for you but I have found these to be no good at all and I wouldn't recommend them.

All my 3D apps and the cool screen savers that come with Ubuntu run at lightning speed now, and on 2 monitors!

Good luck.
