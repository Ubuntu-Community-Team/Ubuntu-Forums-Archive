### Post by amaroKer on 2007-04-16
All of those features exist in Ubuntu, I believe, but please be more specific as to what features you mean.  Remote desktop means a few different things, as does streaming media.  
As far as available applications, Windows applications are for Windows, and Linux apps are for Linux.  There are methods such as emulation and virtualization that can allow some apps to run on a non-native platform (Wine and crossover office are examples of this, and each have varying degrees of success), but there is no reason to switch to Linux if you're going to only use Windows applications.  Save yourself the hassle.  If you're willing to step out of your comfort zone and try the fully capable apps designed for Linux, there are thousands that offer equivalent usability and functionality.  Some examples:

Internet Explorer --> Firefox
Outlook --> Evolution
MS Office --> OpenOffice
Nero Suite --> GnomeBaker
Windows Media Player --> Amarok/Totem/MPlayer
Photoshop --> GIMP
LimeWire --> LimeWire/FrostWire/GTK-Gnutella

There is not a lot of great cutting edge games for linux yet, but 'wine' may be able to get some running under linux.  [Check out [http://www.game-warden.com/bsg/](http://www.game-warden.com/bsg/)

So to answer your question, Windows apps cannot be installed under Linux, but in most cases, don't need to be.  Try out ubuntuguide.org for guidance installing software if you cannot figure it out.

Hope this helps

Logan
