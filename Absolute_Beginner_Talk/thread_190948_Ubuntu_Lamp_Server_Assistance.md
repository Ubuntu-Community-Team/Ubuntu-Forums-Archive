### Post by uzi09 on 2006-06-06
for gnome desktop:
sudo aptitude install ubuntu-desktop

for kde:
sudo aptitude install kubuntu-desktop

for xfce:
sudo aptitude install xubuntu-desktop

ps: i'd use aptitude over apt-get since it works a lot better - espcially for uninstalling these types of things.

good luck
