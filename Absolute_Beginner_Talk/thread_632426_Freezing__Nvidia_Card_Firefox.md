### Post by RomeReactor on 2008-01-09
Try this from the terminal:
```
gksudo gedit /etc/X11/xorg.conf
```
that will open your display configuration file in a text editor; now find this section, about halfway down:
```
Section "Device"
Identifier "nVidia Corporation G72 [GeForce 7500 LE]"
Boardname "NVIDIA GeForce 7 Series"
Busid "PCI:2:0:0"
Driver "nvidia"
Screen 0
Vendorname "NVIDIA"
EndSection
```

add the following lines:
```
Option		"AddARGBVisuals"	"True"
Option		"AddARGBGLXVisuals"	"True"
```
so the section looks like this:
```
Section "Device"
Identifier "nVidia Corporation G72 [GeForce 7500 LE]"
Boardname "NVIDIA GeForce 7 Series"
Busid "PCI:2:0:0"
Driver "nvidia"
Screen 0
Vendorname "NVIDIA"
Option		"AddARGBVisuals"	"True"
Option		"AddARGBGLXVisuals"	"True"
EndSection
```
then save the file and restart your display by pressing CTRL+ALT+BACKSPACE, and see if it still gives you the same message.
