### Post by 3rdalbum on 2007-05-12
This isn't really a request for help, so the topic should be moved.

Microsoft does not support drivers other than Mass Storage and Vesa. Device manufacturers support their own drivers for Windows, but Microsoft cannot provide any support for them. Microsoft doesn't even ship any external drivers with Windows, so a "from scratch" Windows installation will require the user to look around the internet for drivers; so user unfriendly.

Ubuntu, though, has built-in support for many hardware devices and components. And when I say support, I mean that if there is a bug and you're a paying customer, a fix will be found for you even though Ubuntu's developers did not create the hardware device.
