### Post by brownknight on 2007-12-29
Have you tried this process? I had that same problem and this is how it worked for me. You can try it:

1. Download adobe flash from their website: ([link here]("http://fpdownload.macromedia.com/get/flashplayer/current/install_flash_player_9_linux.tar.gz"))
2. Save the .tar.gz file to your desktop and wait for the file to download completely.
3. Unpackage the file. A directory called install_flash_player_9_linux will be created.
4. Open a terminal and navigate to this directory and type ./flashplayer-installer to run the installer. Click Enter. The installer will instruct you to shut down your browser(s).
5. Once the installation is complete, the plug-in will be installed in your Mozilla browser. To verify, launch Mozilla and choose Help > About Plug-ins from the browser menu.
