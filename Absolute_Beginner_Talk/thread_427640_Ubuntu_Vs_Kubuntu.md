### Post by jerrylamos on 2007-04-29
Using both, Ubuntu Gnome is pointed more toward application users such as "ordinary desktop computer user", internet, (i.e. KDE Kubuntu Konqueror browser doesn't do AOL Mail), Office, LAN file sharing, has all the function I need and more, and gets it done with fewer menus and keystrokes.

KDE Kubunu is more eye candy, fancy desktop features, and more pointed toward gimmick users who like to play with Linux itself.  You have a lot more choices which means a lot more menus to go through.  Example finding out what the screen resolution choices are on various KDE versions like Kubuntu and Knoppix are a real hunt.  I don't do games and 3D graphics but I assume KDE might be better?

Both are amazingly customizable.  Best way is to download/get CD Live versions of both and try them out before installing.  And then there's Xubuntu with XFCE instead of Ubuntu's Gnome and Kubuntu's KDE.  XFCE is cut down, smaller, faster, and missing some functions like LAN file browsing.
