the only thing is you need to make sure you have a back up of windows first--i didn't, it was almost a crisis when i was interrupted adding fedora core to my box.

look, when your going through the install process remember you can abort at any time before you write to disk. When partitioning, you want to add a partition, not to any small directories, you just want to split the biggest partition and install Ubuntu on the created free space. Not to worry too much as in one of those partitions, FAT32, or is it NFTS--which ever one is like 1gigish--I think, is an ISO image (a fancy name for an installation file that a system can boot from). 
To get to recovery you turn off, unplug, remove battery if applicable, hold the power button down for 30 secs, then boot up while pressing the ctrl key and tapping the F11. You should get recovery in XP.

You can do it. 
Have backups of everything. Of everything including all files the OS currently in use, all applications office, etc. then do it.
