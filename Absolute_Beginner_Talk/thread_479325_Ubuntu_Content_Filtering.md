### Post by edc80 on 2008-02-28
In our case, I decided to keep it simple.  I found one mention of ProCon in one of these forums.  It's an add-on tool for the Firefox browser.  This was very easy to install (60 seconds) and configure (5 minutes).  For simple browser filtering for kids on the internet, I'm hoping this will do the job.  It's also password protected (it feels a lot like the Cyberpatrol we have on our Windows machines).

Check out [https://addons.mozilla.org/en-US/firefox/addon/1803](https://addons.mozilla.org/en-US/firefox/addon/1803) for more information and to download.
