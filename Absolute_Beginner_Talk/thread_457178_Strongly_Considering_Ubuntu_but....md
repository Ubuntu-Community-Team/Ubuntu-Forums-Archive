### Post by SeanHodges on 2007-05-28
*** Took me a while to write this post, so might overlap what some other people have said ;) ***

As far as office applications, your pretty much set as it sounds like your using all the common apps already.

There are alternatives to some of the applications you mentioned, but you'll need to be more specific to get better answers to this.

I haven't used Maya, but I've used 3DS Max and Blender before and found I actually preferred Blender, which is in the install manager. If you prefer to use Maya/3DS Max then Wine/Cedega might hold the answer (explained further down).

Gaming is the tricky one, as people have varying success with it. There are many people who play all their games in Ubuntu and rave about it. However there are others who find their games don't run well, and they often prefer to run Windows dual-boot (so you select the O/S you want to run on start-up) and reboot into Windows to play these games. You can find a run-down of gaming in Ubuntu here: [https://help.ubuntu.com/community/Games](https://help.ubuntu.com/community/Games). You can ask questions about getting things working here: [http://ubuntuforums.org/forumdisplay.php?f=93](http://ubuntuforums.org/forumdisplay.php?f=93)

Enemy territory - you can download a Linux installer from the official site, it runs great, but you may have sound problems, in which case you need to do the fix, which you can find at [https://help.ubuntu.com/community/EnemyTerritory](https://help.ubuntu.com/community/EnemyTerritory).

Quake - Apparently there is a Linux installer on the Quake disc, I haven't tried it but people tell me it works as well as ET. You can also find Open Arena in the install manager, which is very similar to Quake III Arena and is very popular on-line.

As for the other games, you have a couple of options. You can either run them in Ubuntu using a compatibility layer (these are NOT emulators, although many people describe them as emulators...): You can either try Wine (which is free) or Cedega (which costs you money to get the full version, but some people claim to have better success with gamees). Both of these require some patience and technical work, so if you would rather not have the hassle, then dual booting is probably more for you.
