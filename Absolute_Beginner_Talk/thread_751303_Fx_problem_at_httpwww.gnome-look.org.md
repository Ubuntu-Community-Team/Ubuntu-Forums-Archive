### Post by Fate Reconciled on 2008-04-10
Edit -> Preferences -> Content -> Exceptions... next to Load Images Automatically.

Looks like you might have [http://www.gnome-look.org](http://www.gnome-look.org) blocked in there. See if it's in that list of Exceptions. Report back.
