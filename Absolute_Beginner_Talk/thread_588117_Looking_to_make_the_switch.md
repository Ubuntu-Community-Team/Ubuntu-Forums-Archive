### Post by mb01915 on 2007-10-23
> **Nulifax said:**
> Is there an easier way to just upgrade from within the 7.04 version?

You can do that.  You have to install 7.04 and do all the updates and then do the 7.10 updates.  I did that on one desktop and later went back and did a clean 7.10 install.

There are many laptop specific upgrades in 7.10 and I think you will get a better result with a clean install which is what I did on my laptop.

The other choice you have is to leave windows on your hard drive if you have enough space in dual boot mode.  You will be able to access all of your data in the "windows" partition with Ubuntu.  

But as suggested, back up your data first.

Follow the guided partitioning in the install process.

Once you are comfortable, blow the windows partition away and enjoy Ubuntu.

I have the  same basic programs running in windows and Ubuntu - an operating system, email - calendaring and office.  Windows takes up 16gigs of a a 22gig partition with an additonal 10gig windows recovery partition.  Ubunutu takes up 3.8gigs of a 10gig partition.  This is an older ThinkPad T42 with a 40gig drive that is with the local computer doc getting a new 110 gig drive today.

Ubuntu is faster, more reliable, more secure and stable than microsoft.

### Post by Orbital75 on 2007-10-23
1. Your iPod will work with Ubuntu.
You can use either Amarok, Banshee, or Rhythm Box as your media Player.
Windows program will not work in Linux unless you use a program called
Wine. Wine doesn't work with all programs so you'll run into that.

2. You'll be able to keep and play all your Video and Music with Ubuntu.
The trick is, you'll have to download some free codecs that add playback
support to Ubuntu. 

3. As far as 7.10,  I'd wait a month or so to upgrade.  Just like all new software
bugs need to be ironed out.  It's just best to wait and upgrade. Although, if you
really want you can.  Just download the 7.10 iso then burn the Image to a CD.
This is very easy and most all Burning programs will allow you to burn from an
Iso Image. If you have already install 7.04, you can just hit the Upgrade button
and it will upgrade you to 7.10

4. Well, it all depends...  If you have your Disks that came with your computer
you will be able to just put the disks back in the Cd Drive and blast it back to
the way it was when you 1st bought it.  I would Highly recommend you BackUP
all your Data to a DVD or CD now.  Everything will be deleted if you restore it.
What I use is a Program called Ghost to backup my Whole Hard Drive. If something
goes wrong, I can put it back to the way it was when I made my Backup, It's like
a Carbon Copy.  It's on a DVD so if my drive fails I just buy another HD and Blast
the Image to the new drive. Another solution you may like is to Duel Boot. This will
let you choose which Operating System you want to use when you turn your computer
on.  This I think would be a better solution for you. This wouldn't destroy any of your
data and if you had programs you liked in windows that wasn't available in Ubuntu,
you could use windows for that. I've recommend you Duel Boot.  You can do this
just by putting in the Ubuntu Disk and following the directions. It's easy and will
walk you right through it...... :KS

Enjoy..... Your going to really like Ubuntu.....  Everyone here is so friendly and will
help you out when you get stuck .......  :)
