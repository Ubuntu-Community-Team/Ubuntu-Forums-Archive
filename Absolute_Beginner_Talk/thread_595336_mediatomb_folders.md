### Post by hebetude on 2007-10-30
You need to add folders to share, do this by going to localhost:49152 (or whatever port it auto-configures, it will tell you if you run it in a terminal).  Access your filesystem and click the add share button or setup auto-scanning on some folders.  

Not just any old filetypes work on the PS3, you can test with standard mp3s, or convert some movies to MPEG1/2
