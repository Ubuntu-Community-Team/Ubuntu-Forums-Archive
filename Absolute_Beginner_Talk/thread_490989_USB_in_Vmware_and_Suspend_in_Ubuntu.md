### Post by Kosimo on 2007-08-26
Hello guys.

To use some USB device in VMware you need to activate it first from the main window.
When inserting a usb device, vmware "should" recognize it. Then I don't really remember where, but in some option (at the top window) outside the virtual machine you'll find your USB devices. Then just active it. And boom! It will work inside the virtual machine. (You need to have vmware tools intalled on that virtual machine)  

Using this option I could use my webcam to make videoconferences using skype in a virtual windows xp.
