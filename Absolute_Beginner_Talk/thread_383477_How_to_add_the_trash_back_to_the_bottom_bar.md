### Post by compmodder26 on 2007-03-13
Right click the bottom panel and select "Add to Panel".  A new window will appear with applets that you can add.  The trash bin should be in there.

### Post by ClarkePeters on 2007-04-20
> **Oen386 said:**
>  I tried that but I guess I didn't see "Add..". Thanks again.

Make sure you are clicking on a truly empty space on the panel. If you have a lot of programs or browser windows open, then a right click will give you a box for "minimize etc..".   If you right click the workspace, you'll get "preferences etc...."  So you have to get to the underlying panel.

You might need to close some (or all) programs or browser windows to get to the underlying panel.
