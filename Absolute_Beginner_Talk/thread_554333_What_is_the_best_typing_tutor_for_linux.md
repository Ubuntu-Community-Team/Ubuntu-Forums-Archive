### Post by kelnage on 2007-09-18
I've tried a couple, both of which seemed to be pretty decent - one with a GUI and one without. 

The GUI one was Klavaro. It seems to support a lot of keyboards too, which could be handy.

The non-GUI one was gtypist. That used ncurses and starts you off quite slowly, but gets there eventually :)
