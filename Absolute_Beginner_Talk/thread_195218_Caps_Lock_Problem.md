### Post by schmalzette on 2008-01-20
I had the same problem every once in a while. Rebooting helps, but their is a proper solution.

- Go to "Keyboard Preferences", "Layout Options" tab.
- There you'll find the "CapsLock key behavior" option. 
- Choose "CapsLock toggles normal capitalization of alphabetic characters" and things will work fine.

The "Default" option (whatever it is) should work as well, but as we see it sometimes does not. What would then qualify it as a difficult-to-reproduce bug.
