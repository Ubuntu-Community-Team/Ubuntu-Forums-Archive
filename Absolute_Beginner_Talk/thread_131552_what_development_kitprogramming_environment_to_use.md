### Post by moephan on 2006-02-16
I do all my RoR development using gedit, the command line scripts, and Firefox with refresh. I think this is probably typical. 

There is not compiling, linking, etc... with RoR so an IDE is probably overkill for most people. The only things I wish that I had with this set up was a visual debugger and some kind of "go to definition" function that was faster then grep.

Cheers, Rick
