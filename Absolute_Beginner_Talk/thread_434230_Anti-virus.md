### Post by starcraft.man on 2007-05-05
Ummm, just a quick question here but why doesn't your friend simply install [AVG]("http://free.grisoft.com/doc/5390/lng/us/tpl/v5#avg-anti-virus-free") by himself? Its a good free AV that will find almost anything. If his computer is so messed up that he can't install AVG then he may want to consider reformatting after backing up. A PC thats been crippled by malware/rootkits will never likely be "completely" clean or running in prime, especially a windows PC. There are just too many things that can go wrong in the registry, the windows folder or almost anything else. Not to mention that many malicious viruses and malware now latch onto and embed themselves into essential windows program so that when an AV quarantines and deletes them your OS goes poof.

Just trying to give ya advice, a corrupted Windows install is hard if maybe impossible to get running perfectly again >.>
