### Post by SilentCacophony on 2005-09-19
Hello. As Kapre mentioned, you can use a utility called **smeg** to add things to the gnome menu, since it is buggy in Ubuntu Hoary, or you can add launchers to the programs on the top bar on the gnome desktop as well.

In any case, things will generally install into the /usr/bin directory, with thier docs in /usr/share/doc directory. You can also type *info* into a terminal, which will have nodes for things you have installed, usually, or type *man name-of-program* into a terminal for more info on them.
