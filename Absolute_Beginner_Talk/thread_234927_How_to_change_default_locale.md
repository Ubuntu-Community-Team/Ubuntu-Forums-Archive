Today i am set up Ubuntu 6.06.
At installation time i choose language English and location Kiev/Europe.
When i execute 
$locale
I see that default locale on my computer is en_AU.UTF-8
I want to change locale to en_US.UTF-8
How to do this ?
Thank You.

---

### Post by siacs on 2006-08-14
In GNOME - System > Administration > Language Support

In KDE - System Settings > Regional & Accessibility > Country/Region & Language

Choose default language - English United States, then logout
