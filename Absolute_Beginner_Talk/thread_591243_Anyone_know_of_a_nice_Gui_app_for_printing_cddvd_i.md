### Post by ridgeland on 2007-10-25
I use OpenOffice with a frame that is double the size the CD jewel case needs then trim and fold in half.  With OpenOffice I can set fonts insert some images on the front side and include some notes/instructions on the back side (after cut and folded).  Folding makes it thicker and helps it stay in place in the CD case.
Here is an example:
[http://david.dentlinger.net/GIMP/CD_Case.odt](http://david.dentlinger.net/GIMP/CD_Case.odt)

### Post by Matakoo on 2007-10-25
Well, for CD jewel-cases you could always use KoverArtist (it's made for KDE but probably works in Gnome too). Nice and simple.

You can also try glabels. It's a generic "make and print labels" program with some templates for CD inlays. No templates for the kind of box meant for DVD movies though, but can be created if so desired. Otherwise, I'm sure there are templates for Scribus or Inkscape somewhere.
