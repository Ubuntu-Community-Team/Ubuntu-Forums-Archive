### Post by Metacarpal on 2006-09-08
"Sarge" is the Debian stable release, not an Ubuntu release.  If you're looking for Debian Sarge, you can find it [here]("http://www.debian.org/CD/").

If you'd like to download a copy of Ubuntu, [click here]("http://www.ubuntu.com/download").  Choose a mirror from that page.  If you'd rather download Ubuntu 5.10 (Breezy Badger) instead of 6.06 (Dapper Drake), all you have to do is this: after selecting a mirror, delete "6.06" or "dapper" from the end of the URL in the address bar.  All of the US mirrors also have Breezy available.  Didn't check the international mirrors.
