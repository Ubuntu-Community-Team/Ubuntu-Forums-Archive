### Post by ugm6hr on 2007-08-01
> **jonathan21 said:**
> is there a way to set a time in which ubuntu downloads and installs updates.

In "Software Sources" (in System Menu)  the "Updates" tab allows you to specify how often you want to check for updates.  It will also allow automatic installation of updates.  Unfortunately, I can't see that is allows you to specify *when* in the day it would do that (presumably the first time it was turned on that day?).
