### Post by dcstar on 2007-12-08
> **slymi2005 said:**
> I was thinking about getting a new computer and I was just wondering what you guys would recommend regarding the specs? I intend to use either k/ubuntu so how much memory is best? Also for compiz what graphics card is best? I'd rather spend as little as possible.

As others have said, get Linux supporter graphics hardware (like Nvidia), and if you are getting a DDR2 system then make sure you get 2 matched memory modules (instead of just one) as this will take advantage of higher throughput that the DDR2 spec provides for.

You should be able to get a cheap DDR2 motherboard with inbuilt Nvidia graphics, and if you go the AMD path you should be able to get a 64 bit system reasonably inexpensively.

### Post by sloggerkhan on 2007-12-08
So far as memory goes, if you plan on doing graphics for print or high resolution, I'd try 2 gigs of RAM.

I do a moderate amount of work in the gimp and inkscape for fun, and I consistantly use near all my memory (excluding swap, obviously) when editing larger images.

If you don't plan on doing graphics or other intensive tasks, no reason you need more than a gig, IMO.
