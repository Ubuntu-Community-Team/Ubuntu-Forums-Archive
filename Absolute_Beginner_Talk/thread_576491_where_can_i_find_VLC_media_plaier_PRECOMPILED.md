### Post by PacketCollision on 2007-10-15
You should be able to download the official packages check [http://packages.ubuntu.com/cgi-bin/search_packages.pl?keywords=vlc&searchon=names&subword=1&version=feisty&release=all](http://packages.ubuntu.com/cgi-bin/search_packages.pl?keywords=vlc&searchon=names&subword=1&version=feisty&release=all)

Also see [http://www.videolan.org/vlc/download-ubuntu.html](http://www.videolan.org/vlc/download-ubuntu.html) for info.

You're going to need to have VLC's direct dependencies no matter what method you use to get it, but this way you won't need the development packaged and gcc, etc.
