### Post by skompier on 2007-12-02
I run Feisty perfectly with my 7600GS. I had an ATI 9600 that I was using with the usual problems. Before I swapped out cards, I installed Envy. Then when I rebooted Ubuntu with the new Nvidia card, I ran Envy from the command line and it downloaded the latest driver and setup everything for me. When Envy was done, I rebooted and everything worked, including desktop effects.
