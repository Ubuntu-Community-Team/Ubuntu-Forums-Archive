### Post by quandary on 2007-11-30
Um... What are the exact specs of your two computers? 

As far as I know, the 7.x live-CD runs out of RAM easily (memory leak, probably), so in case you don't have very much RAM, RAM will be pretty heavily used and that makes the system very very slow, or even crash. 

You can either download the alternate install CD from the site and burn that. 
It uses a text based installer, hence, it does not use all the resources that the live-CD uses. 
Should you have particularly low RAM you may want to try Xubuntu rather than Ubuntu, as it is less resource heavy and faster.

Or you can download a 6.x version of Ubuntu, and upgrade the distro via apt-get. See
[http://onlyubuntu.blogspot.com/2007/03/upgrade-ubuntu-610-edgy-eft-to-ubuntu.html](http://onlyubuntu.blogspot.com/2007/03/upgrade-ubuntu-610-edgy-eft-to-ubuntu.html)
for details
