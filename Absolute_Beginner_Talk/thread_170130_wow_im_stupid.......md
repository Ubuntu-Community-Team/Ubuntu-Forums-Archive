### Post by yager on 2006-05-04
[QUOTE=stansz]and use the cd key they give me....[/QUOTE]

You need to be careful with the CD key.  Microsoft has now implemented a verification system on their site to get the latest tweaks to Windows.  It verifies that the key is valid before it will allow patches to be installed from their website.  If it detects that the key is invalid, it will ask you some questions in an effort to make you legal, ie pay money, and to find the perpetrator who gave you the bogus key.

You should have a valid key on the machine.  Look for the pretty sticker on the back of the case.  It will have your key value.  Every new box comes with one.
