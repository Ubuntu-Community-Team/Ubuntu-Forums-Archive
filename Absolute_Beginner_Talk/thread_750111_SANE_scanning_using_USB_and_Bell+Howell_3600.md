### Post by Norman Grahams on 2008-04-16
When I first used my packard bell 1200 plus scanner, I was using SUSE which told me to get a .USB file (in my case PS1Dfw.usb) from either the CD or an installed (under windows) version of the driver. Then I had to put it in /usr/share/sane/gt68xx
Now in Ubuntu, each time I reinstall or move to a new computer, I've put that file in that place, and SANE has worked without problem.

Perhaps a similar thing will work for your Bell + Howell? Hopefully someone who knows more about it can suggest a solution.
