### Post by popch on 2007-12-15
The two answers already given are corrrect.

However, you also mentioned that you had many programs in Windows you wanted to keep on using.

In that case you should be aware that an installation of Linux - if properly done - does not damage those programs in any way. However, you can not use those programs while Linux is running.

Since your Windows installation is damaged, you won't be able to use those programs until you fix your Windows.
