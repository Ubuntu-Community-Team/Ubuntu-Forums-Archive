### Post by BuffaloX on 2006-07-18
If you use the default terminal in Ubuntu ( GNOME Terminal )

Select "Edit" in the menu, then select "profiles".
select "Default" profile, then "edit".
Select the tab "colors".

deactivate Use colors from system theme.

You can now select colors of your own liking. :p
