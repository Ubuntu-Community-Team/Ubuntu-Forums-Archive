### Post by BKonkle on 2005-05-31
I was in the exact same boat as you are when I started using Linux.  I had run Windows all my life, and I had been running a server off of my desktop computer to host web pages out of my apartment.  Well, I finally upgraded my computer and built a smaller server computer out of the scraps.  I had been fascinated by all of the people who swore by Linux, and said it was hands down the greatest operating system to run a server from.  My curiosity got the best of me, and so I first tried Fedora Red Hat. . .

It was a disaster!  The graphical installer was sure impressive, but I couldn't get anything after that to work right.  My wireless card wasn't recognized, I couldn't get apache set up right, everything ran so SLOW. . . I was extremely dissapointed because I had heard so many good things about Linux.  Not to mention that everything was so DIFFERENT compared to Windows.

I thought of giving up and just installing a Windows server on the computer, but I started hearing more and more about different distributions of Linux that may have worked out for me better.  Once again, my curiosity led me to try another.  I selected Ubuntu after doing a lot of research, and I've never gone back!

Ubuntu has been so easy for me!  I've loved every minute of learning Linux since I got ahold of my Ubuntu distro!  I would really recommend downloading the install CD, installing a full graphical desktop at first, and just spending a few weeks going over ubuntuguide.org and ubuntuforums.org, just learning the ins and outs of the Linux Operating System.  Then, when you're comfortable enough, reinstall Linux on your server using the "server" paramater on install, and set up your server without any graphical interface at all - just the terminal.  Things run so smoothly and quickly on my system set up that way.  I love Linux so much I'm using it on my desktop too in a dual-boot situation.  I hope to some day leave Windows behind entirely. . .
