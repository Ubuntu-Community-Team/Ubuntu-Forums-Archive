### Post by bsussman on 2006-10-25
> **marianlibrarian said:**
> When I spoke with tech support at BookSystems he told me that it can only be used as a server never a workstation. 

The terms get thrown around alot.  Your library server generally should not be used as a workstation because that implies casual human access to a system that probably should have very limited physical and direct interactive access.

Other screen/kb/cpu systems should be your terminals and workstations.
