Ubuntu was founded by a self proclaimed millionaire, Mark Shuttleworth whose dream was building the best Linux distribution out there, in which he has succeeded. Ubuntu is not just another operating system like the rest; it’s a large distribution of open source code based on anyone and everyone’s input. Knowing this, Ubuntu has been the leading Linux operating system for many years now including present day. Unlike the regular Windows XP or Pro edition in which you have to purchase, this operating system is free and is completely different. This operating system stretches to every corner of the world. It’s also as easy as logging onto [www.Ubuntu.com](www.Ubuntu.com) and emailing a question to their 24 hour support team. I myself have found it even easier posting a question in an online forum like [www.ubuntuforums.org](www.ubuntuforums.org) and getting a reply in minutes. I can honestly say, with my experiences using Ubuntu it has only opened my eyes to a broader perspective of programming language and coding alike.
