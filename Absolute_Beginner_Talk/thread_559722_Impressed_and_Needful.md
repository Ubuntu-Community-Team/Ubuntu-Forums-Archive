### Post by Sethalos on 2007-09-25
Well it's been since the release of Ubuntu Feisty Fawn 7.04 that I have been using a Linux Distro almost exclusively for my day to day computing needs.

Since that time I have managed to learn quite a bit about the command line, in fact, I can't believe that I lived without it for so long. I have learned a lot about networking via Linux (Samba) and much more about Security. The wireless features for me were easy to use and understand, the plethora of games, applications, and system utilities were outstanding.

I use VirtualBox to run a full screen version of Windows XP for certain work apps that I needed that don't have Linux counterparts and it has worked flawlessly for me. There is no greater feeling of satisfaction than having a work mate over and showing him how I can switch between desktops (XP/Linux), or rotate the Compix/Beryl Cube. Their eyes almost bulge out of their heads and my smile widens as they ask, "How the **** did you do that."  Shortly afterwards, "How do I install Ubuntu on my computer."

I have been so spoiled by AmaroK that I can't imagine using another music program EVER. I've also become somewhat adept at using The Gimp.

My only source of frustration is that one major drawback of using Linux or even MacOSX for that matter, no gaming support. Wine is just not there yet as far as the installation and running of most computer games. So, I'm actually forced to install and use a dual booted version of XP to run these games. I hope in the future more developers will jump on board and make Linux versions of games.

I'm writing this basically because I wanted to thank the Ubuntu community for assisting me with all my little technical problems over the past few months, the developers for creating an amazing OS, and Canonical for their vision of what computing should be, Open and Available to all of us.

cheers,

Seth
