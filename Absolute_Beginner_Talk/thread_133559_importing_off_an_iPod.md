### Post by welsh_spud on 2006-02-20

That should connect your iPod to amaoK.

### Post by welsh_spud on 2006-02-20
After a bit of research, you can get konqueror to recognise iPod automatically. In terminal (or Adept if you so wish):

```
sudo apt-get install ipodslave
```

This should get iPods working in amarok and konq.
