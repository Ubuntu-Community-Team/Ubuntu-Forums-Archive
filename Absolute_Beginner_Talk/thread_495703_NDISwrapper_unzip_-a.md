### Post by jimbob on 2007-07-08
I would first move the file to your home directory.  You should be able to open a nautilus window by clicking on places->home folder then drag and drop the file in there.

Open a terminal window and enter *[COLOR="Blue"]unzip -a <name of zipped file>[/COLOR]*

If that fails enter *[COLOR="Blue"]sudo unzip -a <name of zipped file>[/COLOR]*
