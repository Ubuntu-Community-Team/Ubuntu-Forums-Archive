### Post by Metacarpal on 2006-08-28
Well, first up, adept is a package manager, and as such will only install .deb packages, if my understanding is correct.  If you're looking to install a downloaded program from source, [this guide]("http://monkeyblog.org/ubuntu/installing/#source") might prove helpful (though it is written for Gnome, not KDE, the same things apply).

As for your connection issues, are you behind a firewall on your network or router?  If so, you'll probably need to open some ports, or find some that are already open, to use IM and ftp.
