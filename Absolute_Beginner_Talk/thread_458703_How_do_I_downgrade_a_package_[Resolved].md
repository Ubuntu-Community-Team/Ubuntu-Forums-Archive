### Post by joe.turion64x2 on 2007-05-29
If the package is still in the repositories you can use Synaptic to first uninstall the newer version you want to get rid of, and then install the old one.

Another way would be to uninstall the package and then search in Internet for the old version. If it is a .deb package perhaps it will sufice to double click it, if it is a source version you'd have to read its readme file.

### Post by yabbadabbadont on 2007-05-29
Run synaptic (System->Administration->Synaptic Package Manager), then search for and select the package.  Now go to the Package menu and select Force version.  A dialog will pop up that lets you choose the version you want.
