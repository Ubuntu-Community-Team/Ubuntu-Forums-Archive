### Post by machoo02 on 2007-04-09
Go to [http://www.ubuntu.com/getubuntu/downloadmirrors]("http://www.ubuntu.com/getubuntu/downloadmirrors"), and choose the mirror closest to you.  Select the version of ubuntu that you would like: Dapper (6.06.1), Edgy (6.10), or in about a week Feisty (7.04).  Select the alternate install .iso from the list.
