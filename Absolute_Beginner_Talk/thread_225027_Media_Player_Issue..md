### Post by Footissimo on 2006-07-29
Do you mean you access a site and it requires you to log in via a pop-up, but that the pop-up never appears?

If that is the case then it may be that firefox is blocking the relevant pop-up (a grey bar should appear and tell you that though, and give you an option  to see it).

If its a plugin problem (probs flash or java, if it is) then you can check if the relevant plug-in is installed by typing ```
about:plugins
``` into the URL bar.
