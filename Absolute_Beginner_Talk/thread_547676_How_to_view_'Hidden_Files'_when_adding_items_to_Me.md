### Post by Sam Peters on 2007-09-10
Well I don't know how to do that, but here's an alternative I use to launch stuff so you don't have to go in through terminal every time.

This is an example for running a program from CD thru wine.
> 
#/bin/bash!   (this header tells it to run through the terminal)
cd /media/cdrom           (this is my example)
wine Setup.exe             (this is a wine command for running the startup file on the CD i'm using)


I usually just type that into a gedit window, save it to the desktop, and run it from there every time I want to open said program. Make sure when you open it, if it asks, to select 'run'.
