### Post by Skrynesaver on 2007-04-01
Just noticed your post, If you're looking for a front end design tool palate you could do worse than QT3-Designer, KDe-centric but great for rapid front end development.
```

apt-get install qt3-designer

```
Then take a look through the /usr/share/qt3/doc directory
The downside is that it is KDE based, the upside is that it is very useful and if KDE4.0 lives up to it's promise will cross compile to Windows
