### Post by AAM on 2007-05-28
I don't follow either, but you are correct. You reformat the bootable / partition. If you have the /home directory on the same partition, you will lose the entirety of your data also. Back up the /home and /etc directories first. So go back, make sure you have done the backup and then make a bootable root file system.

As you move through the LiveCD, you should chose the **custom** option and have the partitions displayed. If you don't have three partitions at least, then make them now - / (~10GB & bootable), swap (2xRAM) and /home (rest). The next windows will give you opportunity to reformat the pre-existing formats, only check the / partition. The rest should be automatic.
