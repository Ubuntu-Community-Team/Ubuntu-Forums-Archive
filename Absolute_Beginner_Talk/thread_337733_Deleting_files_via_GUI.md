### Post by taurus on 2007-01-13

```
gksudo nautilus
```

### Post by NeoLithium on 2007-01-13
In a terminal, or using alt+f2, type:
```

gksudo nautilus

```
That'll open up a file manager with root permissions that you can browse through your system and delete them.
