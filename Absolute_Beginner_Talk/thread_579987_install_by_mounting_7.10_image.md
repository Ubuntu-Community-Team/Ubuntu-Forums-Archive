### Post by p_quarles on 2007-10-18
Using PowerISO requires you to be running Windows. Linux is a separate operating system, hence you cannot install it from within another operating system.

So, yes, you need to boot from a CD. If you'd rather install it from within Windows, you can set up a virtual machine (VMWare, Virtualbox), and then use the ISO on the hard drive as the installation file.
