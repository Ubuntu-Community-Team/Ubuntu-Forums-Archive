### Post by daynah on 2007-04-12
It's quite possible.

If you're going to do Ubuntu and Windows, make sure you do Windows first and then Ubuntu (Windows will try to eat Ubuntu during installation. It's able to be fixed, but it'll save you trouble if you just do it in order).

Install windows on the harddrive you want Windows on. I'm not sure how to do that, it's been years. Then when you do Ubuntu, ubuntu will ask you which harddrive. Pick the cheak one. :) You may want to peek inside your computer, double and triple check the sizes of all the harddrives and stuff and write down which harddrives are going to have which operating systems.

Ubuntu doesn't name the harddrives C:\ and D:\ so remember the sizes! For Example, I have Ubuntu Feisty on a 60 Gig harddrive (called Lilun) and Ubuntu Breezy on a 200 gig harddrive (called Biggun) you get to name them after Ubuntu is installed. :)

If you want to do Ubuntu/Ubuntu, Just isntall Ubuntu once on one harddrive... then go back and install it again on the second one! :)

Have fun!
