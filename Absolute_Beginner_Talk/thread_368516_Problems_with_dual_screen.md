### Post by caintona on 2007-02-23
The nvidia app may not have the correct permissions to access that directory.  you need to be root to modify /etc/X11 or a member of group.

try editing the xorg.conf file with this.

```
sudo gedit /etc/X11/xorg.conf
```

sudo gedit lets you edit things as root.
