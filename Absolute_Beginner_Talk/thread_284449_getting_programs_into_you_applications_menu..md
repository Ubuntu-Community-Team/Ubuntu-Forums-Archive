### Post by BitTorrentBuddha on 2006-10-25
You can add menu entries with Alacarte menu editor if you're using dapper or edgy. Alacarte can be found under Applications -> Accessories, or by running: ```
alacarte
``` From either a terminal or a run dialog (Alt+F2).
