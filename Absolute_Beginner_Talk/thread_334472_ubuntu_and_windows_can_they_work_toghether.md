### Post by geo_bio on 2007-01-09
Your question is not too clear. Please rephrase.

If you're asking about accessing your files (using ubuntu as a network server), then I believe it is possible... though I don't see the point since it's running two computers. in the same house when you can just have both hard drives in one computer.

If you're asking about running Windows and Ubuntu on the same computer using different partitions, it is easily possible (however, it does not seem to be your question (except implied by title)).

If you have two computers: One is a server running a Windows OS, and you want to use this as backup - that is possible. I can't help too much with that, but I can say it'll take some work if you want it to be automatic (like set it to backup every week). Ubuntu supports NTFS AND FAT32 (for windows) along with other Linux partition types.
