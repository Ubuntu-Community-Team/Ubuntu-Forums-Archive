### Post by lubeck on 2008-01-19
The absolute best method I have found to transfer data, particularly Contacts and Calendar information, from OUTLOOK 2007 to EVOLUTION was using the fabulous tool called OUTPOD from [http://outpod.stoer.de/](http://outpod.stoer.de/) 

I am sure it will work on any version of OUTLOOK.  Really wonderful!
Efficient, fast, and flawless.  Simple too - use is almost self explanatory.  When run in Windows, OUTPOD automatically finds your Outlook files, lists them, allows you to select and then export any one item or whole Mail, Contact or Calendar folders.  Thunderbird IMPORT function then brings them in flawlessly. 

Backstory if interested:

Originally designed to export Outlook stuff to an IPOD, it works great for export to open architecture files that Evolution can digest easily.  Even exports the pics for Contacts and so on.  Loses nothing, as far as I can tell.

Experiences: Tried the recommended Thunderbird export approach touted almost everywhere.  Worked OK for mail, but misplaced lots of the Contact and Calendar data (or lost it) when imported to Thunderbird. OUTPOD lost nothing.  Perfect 1 to 1 transfer.
