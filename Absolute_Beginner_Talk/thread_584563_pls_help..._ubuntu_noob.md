### Post by TeaSwigger on 2007-10-21
Deluge is a torrent client I assume? I don't know as I use Kubuntu's KTorrent. If so, have you configured your router, if applicable, to allow the ports Deluge is wanting to use? Does Deluge have a "UPnP" feature? If so try activating it if it isn't already. It may forward those ports for you (I've always set them by hand but gather the feature is to "automate" it).

If Deluge is broken there are a number of other torrent clients you may easily install freely available for you in the ubuntu repositories including KTorrent.
