### Post by Jimmyfj on 2008-02-27
You'll need to make the rip as an .AVI-file which can be read by most DVD-players. For that you can use either Acid-Rip or DVD::Rip program which you can find in your Add/Remove programs or Synaptic.

### Post by 3rdalbum on 2008-02-27
Thoggen rips DVDs into Ogg Theora/Vorbis file format, which cannot play on your DVD player.

You should download the program "K9Copy", which can rip DVDs to disc image files and then allow you to burn them to ordinary DVDs that will play in normal DVD players.

K9copy is available from the Synaptic Package Manager.
