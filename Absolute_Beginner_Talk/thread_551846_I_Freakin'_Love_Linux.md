### Post by rivalarrival on 2007-09-15
hehehe...  Try setting up various file servers - you'll start getting into the guts of the machine pretty damn quick...

Setup a LAMP server and get some web-apps running (wordpress, dot project, wiki)

Setup a Samba server - without using Swat

Get a SECURE ftp server operating - that one took me a few days as a complete n00b

How about a VPN server? pptpd is still kicking my hind end...

Administer everything from a remote computer via an ssh terminal.

Write some bash scripts - my first significant bash scripting adventure was getting several servers at separate sites to ping (and later port scan) eachother. If one failed to get a reply (but was still connected to the internet) it would try to notify me via email that its partner was down. Oh yeah, and log everything.

I guess what I'm saying is this: Don't try to learn it, try to do it and learning it will just be part of the process.
