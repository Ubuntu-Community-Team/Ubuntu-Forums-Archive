### Post by tgm4883 on 2007-05-01
Depends.

If you plan on adding more tuners later, upgrading your frontend (more powerful or smaller), or already have a decent schedule setup and/or programs recorded, then I would leave the tuner in the desktop (when you say sluggish, is this when watching a program, recording, watching live tv, etc)

If you want to be able to turn off your desktop and still be able to use the mythtv, then I would move it into the frontend.

My current setup has the tuners in the frontend (combined frontend/backend).  I initially set it up this way and have it this way because I do many different cpu intensive things on my desktop and I dont really care about the size of my mythtv box (it sits in a midtower).  If I had the money I would make this my dedicated backend and run a much smaller frontend (probably a via epia)
