### Post by Sutekh on 2006-01-26
Right-click on *any* file that is the type of media you wish to open with VLC.

So say start with .avi.  Right-click on any .avi.  Click on the 'Open With' tab and choose VLC to be the default player for that file type.

Very similar to Windows, but unfortuantely you'll have to do it for each type of media file.
