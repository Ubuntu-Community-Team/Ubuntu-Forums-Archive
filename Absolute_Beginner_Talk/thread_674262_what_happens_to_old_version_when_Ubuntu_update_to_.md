### Post by Cypher on 2008-01-21
The older version of the applications are uninstalled before the new versions are installed, so that the previous version can cleanly stop any services or clean up before a new version comes into play. This is crucial as files might be added or deleted in new versions of a software package and it'd be a pain of every new version tried to remember what the old versions contained..
