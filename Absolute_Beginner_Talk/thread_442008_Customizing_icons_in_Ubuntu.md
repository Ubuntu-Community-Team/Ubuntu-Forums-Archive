### Post by heimo on 2007-05-13
[LIST=1]
[*]Rename the file to Buuf_gnome_1_6_r7_by_djany.tar.gz
[*]Open System->Preferences->Theme
[*]Drag'n'drop the file to Theme Preferences window
[*]Choose Apply New Theme[/LIST]
First step is normally not neccessary, but this filename has wrong extension.
