### Post by eternalsword on 2007-07-02
is your controller detected as a joystick or at all.  with it connected, try running ```
lsusb
``` or ```
lspci
``` from the terminal and see if it's detected.  check out [http://www.linuxlinks.com/Software/Utilities/Keyboard_and_Joystick/](http://www.linuxlinks.com/Software/Utilities/Keyboard_and_Joystick/) for some possible solutions depending on how it's detected.
