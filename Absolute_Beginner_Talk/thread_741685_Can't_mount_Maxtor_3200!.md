### Post by joshrobinson on 2008-04-01
```
sudo gedit /etc/fstab
```

add this line at the bottom

```
/dev/sda1 /media/Maxtor vfat defaults,umask=0000 0 0
```
next
```
cd /media
```
```
sudo mkdir Maxtor
```
and last
```
sudo mount -a
```

in your places menu on your top panel, "Maxtor" should show up
also on your desktop a Maxtor icon should show up also

open it, and create a new folder, name it whatever, and then delete it, just to see if the permissions are correct on the drive
