### Post by bodhi.zazen on 2007-02-07
You should know what Linux native alternate is available.

You should know what program you want to run on wine and how to do it. You will need to follow your how-to exactly, possible to the exact known good version of wine.

Wine is, imo, Beta and constantly changing. Do not be surprised if there are bugs.

See here for more info : [http://doc.gwos.org/index.php/HowToWine](http://doc.gwos.org/index.php/HowToWine)

wine itself if secure, the software your run on it (IE for example) may not be so hot.
