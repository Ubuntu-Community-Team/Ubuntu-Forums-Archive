### Post by sumguy231 on 2006-05-11
You can set up a dual boot, in which Windows is kept on one partition and Ubuntu is kept on the other. This can be set up during install and isn't too difficult, assuming you do have free space on the hard drive. To answer your question, you don't have to erase the whole drive and can set up a dual boot. If you have any specific questions about the partitioning process, you should ask.

### Post by catlett on 2006-05-11
15 gb is enough. The link in the other post is very good. You can have both systems run off of your one hard drive. A partition is just segmenting your drive, it doesn't have to erase your drive.
The easiest way is to restart the computer with the cd in it . If your computer bios are correct  the computer will run off the cd.
Press enter at the prompt. When you get to the screen I have attached choose "resize IDE1 master, partition #1 (hda1) and use free space." [COLOR="Red"]The image didn't upload. Here is the page from the how to with the image[/COLOR] [http://users.bigpond.net.au/hermanzone/p3.htm](http://users.bigpond.net.au/hermanzone/p3.htm)
Once you select this option the installer will automatically create the partitions for Ubuntu and install the system.
Read through the how to but the process when choosing that option is very simple. You can pretty much follow the default choices.
When it is done your computer will start up and go to a menu. From that menu you can choose windows of ubuntu. To switch between them you have to restart your computer and get to that menu again.
