### Post by santiagoward2000 on 2008-02-26
> **yngens said:**
> I have installed Emerald and got an Emerald theme imported, but can't figure out how to apply it to my Desktop. It is not in the list of the themes in Compiz.

Hi!
To get it, first open **ccsm**, go to **Windows Decoration** and make sure that the **Command** line says: ```
emerald --replace
```
Then, open the **Emerald Theme Manager** and click on the theme you want.
Finally, press Alt+F2 and enter: ```
emerald --replace
```
That should restart Emerald and you should see the theme you chose.
Tell us if it works.
Good luck!
