Today is desktop decorating day.

I've downloaded gdesklets and spent some time configuring the starterbar. Somehow I expected to find an option that let's me automatically startup after login. But alas...

When I manually startup gdesklet, I get the starterbar but I do get the gDesklets Shell window aswell.

### Post by nalinde on 2006-10-30
In the ubuntu menu
System > Settings > Sessions > startup progs. (Don't run english version, but it should be something like that)

Just hit Add and type _gdesklets_ and it should be up and running next time you log in. :-)
