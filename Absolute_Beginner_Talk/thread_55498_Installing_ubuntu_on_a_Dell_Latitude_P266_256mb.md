### Post by roland on 2005-08-10
According to [this](http://www.togaware.com/linux/survivor/XF86_Configuration.shtml) website, the horizontal and vertical sync rates are "31.5 - 48.5" and "55-65" respectively.

So the lines in the configfile should be as follows:

```

Section "Monitor"
    Identifier  "Dell SyncMaster 17GLi"
    HorizSync   31.5 - 48.5         # multisync
    VertRefresh 55-65
EndSection

```

Note that the website talks about a server config, which is not applicable here, although it is about the same Dell laptop. Also note that it is about XFree86, which is almost the same as X.org in Ubuntu, especially the configfile.

### Post by Joeak on 2005-11-29
The problem is 1024x768x3 (3 bytes = 24 bits of color depth) = 2,359,296.  The video card only has 2Mb.  Edit the  DefaultDepth    24 down to 16 and it will work well at 1024x768.
