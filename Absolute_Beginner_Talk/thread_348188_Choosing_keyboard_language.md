### Post by bapoumba on 2007-01-28
As-tu un menu en bas à gauche qui s'intitule "K" ? Si oui, tu as KDE, dans ce cas il faudra trouver le menu correspondant au clavier (j'ai GNOME).
Avec GNOME : Système > Préfrérences > Clavier.

Une fois dans le menu du clavier, il y a un onglet correspondant à la touche "compose". J'ai activé la touche sur la touche CTRL (contrôle) de droite, parce que j'ai un portable. Sur un clavier de desktop, il est pratique d'activer la touche "windows" de droite (sinon, tu perds la touche Contrôle droite qui devient "compose").

Avec la touche "compose", tu peux accentuer les lettres :
compose + , (virgule) + c = ç par exemple.

Et voilà ;)

edit : ça vaut pour une installe d'ubuntu sur GNOME et sans doute KDE. Sur xubuntu, je ne sais pas :/
Intaller les language-pack en français avec un clavier qwerty, je ne sais pas si tu auras pour autant les lettres accentuées.

### Post by Pobega on 2007-01-28
Okay, first off you're using Xfce which is Xubuntu. How can anybody miss the "Xubuntu 6.06 Dapper User" title next to his bean count?

Go to Applications -> System -> Lanuage Support. You will be prompted to install other language packs. Choose "Install Now", and it will install all of the language packs. Once they're done installing sort through the list and find French, put a check in the box on the right, and change your default language from English to French using the drop down box above the Cancel button. Click "_A_pply", and then "_O_K", restart your computer and it should be running in French.
