### Post by Brunellus on 2006-09-19
if all you want to do is connect to the ssh server through nautilus:

go to Places>Connect to Server 

then enter the server details there.

Vocabulary note:  Please do not use 'shortcut.'  It's a non-specific term, and has been used by Microsoft (and Windows users) interchangably to refer to a URL, a hyperlink, a program launcher, and something like a symbolic link.  Since it's so ambiguous, I would advise against using it.
