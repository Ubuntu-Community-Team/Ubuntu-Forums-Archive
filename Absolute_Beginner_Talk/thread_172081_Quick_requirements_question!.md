### Post by nalmeth on 2006-05-07
Yes, ubuntu will run on your computer, though with such low ram, it may be slow.
I also think you may need more harddrive space aswell.
When you install, pick the "server" install
when you get the command line type this:
sudo apt-get install xubuntu-desktop
this will install xfce, a much lighter Desktop environment, though less visually impressive.
Can you spare any more HDD space for more swap? Swap is what linux uses when all your RAM is used up. Though less efficiently than real RAM.
