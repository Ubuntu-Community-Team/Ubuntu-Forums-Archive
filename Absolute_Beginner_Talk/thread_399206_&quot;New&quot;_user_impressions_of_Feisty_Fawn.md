### Post by JayRx1981 on 2007-04-02
Ok, so I'm not a new Ubuntu user, strictly speaking. I tried Dapper Drake when it was released back before I bought my Mac Pro but didn't feel like I needed to carry it over onto the Mac pro since Apple's desktop was (in my opinion) better than Dapper Drake was.

However, I just purchased an Acer 5630 laptop for school and it came with Windows Vista Home Premium. The laptop is running a 1.6GHz Core 2 chip with 1GB of ram and Vista just crawled. It literally took almost 2 hours taking it out of the box and turning it on for Vista to finish it's new "setup". After a few days of running Vista, things weren't getting any more tolerable, so I decided to give Ubuntu a try again on my laptop (since linux has historically run much better on "slower" hardware than even XP does). I downloaded Feisty Fawn and quite honestly, I'm loving it.

First, I know this was a trait that dapper drake possessed, but I love the Live CD install process that Ubuntu has working for it. It was easy (for me) to install and I loved being able to surf the web and do things that are USEFUL with my computer while the operating system was installing, unlike either Windows or Mac OS. Feisty Fawn detected my Intel 3954abg from the start and it has worked perfectly with my WPA network since bootup with the live-cd.

And so far, Feisty is handling my hardware very well. It is infinitely more responsive than Vista was, all of my hardware (except for the Acer Orbicam) was detected and installed automatically without any issues (including my wireless access), and I haven't had a single crash yet.

I went ahead and installed Beryl and the eye-candy of Beryl is, quite simply, even cooler to me than Mac OS X's eye-candy. I love the burn and warp out effects (currently my minimize/maximize and close effects respectively), and the expose-like window switcher is working beautifully (albeit the screen goes a wee bit fuzzy when the "expose" screen is up). Overall, I'm very impressed and really do think Feisty Fawn could probably easily stand on it's own against Mac OS and Windows with more exposure to the public.

That said, a few things do bother me. One is that my native desktop resolution (1280x800) was not auto-detected or choosable without downloading a program and then command-lining several steps (I know, 915resolution isn't that hard for me, but for your average user unless they're lucky enough to stumble onto the exact solution, it's likely more a pain). And a second is that for the new user, Ubuntu could really use a good tutorial program that runs on first boot (or at least an option on the desktop to run a tutorial)--a tutorial explaining the basics of adding programs from the repositories, of where to look for help (in support doc form, wiki form, and the forums), etc. And I know those items are certainly easy to find if one really looks, but I think you possibly would see greater adoption if the tutorial was "dummied" to an incredibly simplistic, intuitive level such that even my mom (largely clueless when it comes to most computer stuff) could follow it.

Still, Feisty Fawn has a TON going for it and I salute the Ubuntu devs for their hard work on this fantastic release. While I have a secondary XP partition (to run World of Warcraft), Ubuntu is definitely my primary web-browsing, multi-purpose desktop for everyday use on this laptop and I may even go ahead and install on my Mac pro after seeing how cool it is on this Acer. :)
