### Post by nowshining on 2007-12-21
gdm - login manager (what u see when login)

as for themes to the desktop

u can find them in the follow locations:

open up ur home folder and ctrl + h

for icons = .icons
for themes - .themes
for backgroud - it should be in the same place u chose and be loaded up from there.
for cursors - .icons

edit: note that each users settings are stored in their home folder under their username.
