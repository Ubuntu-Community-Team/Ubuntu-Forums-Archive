### Post by catlett on 2006-06-02
```
sudo aptitude install kubuntu-desktop
``` This will install kde alongside of gnome. Just hit on "session" at the login screen and check off the box for kde instead of gnome,
(the use of aptitude instead of apt-get is just for the purpose of removal. if you ever wanted to remove kubuntu-desktop, aptitude installing it will allow aptitude to remove it entirely. apt-get doesn't always remove everything)

### Post by hotbrainz on 2006-06-02
They are both good desktops. You can load KDE desktop apart from your default gnome and switch between them as u please with each session. :)

In terms of funtionality they are quite similar but i reckon Gnome is more suited for work and KDE for play. Well thats my opinion.. so you can give both atry.

The Ubuntu also comes with the KDE desktop which is called Kubuntu :)

If you dont want to reinstall. Then go to System - Administration -Synaptic Packet manager

There you can search for "KDE desktop Environment" package. Mark all the components and install ( press button "apply" on top ). You are done then. 

Log out and then you can choose session ..

It will have option of KDE or Gnome..

Ask if you want any other clarification.

Hope that helps..
