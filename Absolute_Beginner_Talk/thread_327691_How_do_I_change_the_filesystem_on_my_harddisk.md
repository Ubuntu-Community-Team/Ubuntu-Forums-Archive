OK, so at the moment you have an NTFS formatted hard disk which you are trying to reformat as EXt3 using the windows install CD? This will not work. If you want to install Ubuntu on this disk, just start the installation from the beginning and reformat the disk as part of the process. 

To make it easier, when you get to the disk partitioning section choose the option for 'automatic partitioning' and the Ubuntu CD will partition and format the disk for you. If you don't like the default partition scheme you can always choose the manual option and change it to your preference.
