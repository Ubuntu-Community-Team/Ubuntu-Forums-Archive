### Post by IYY on 2006-08-25
There are two versions, I believe:

sudo apt-get install lxdoom

or

sudo apt-get install prboom

After that's installed, you'll have the engine but not the real data files. So, different graphics and different levels. You can get the classic datafiles from a legal Doom CD, a Windows machine with Doom installed or wherever else you may find them.
