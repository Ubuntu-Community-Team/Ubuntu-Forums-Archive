### Post by eentonig on 2007-06-20
free beer= you don't pay for it.
free in the context you reffer to= You can see and, if you want, change the code of the program. And you can distribute it again (with some restrictions.)
closed source= You don't get to see how the program works, you cannot see the code.
open source= you can see the code and if you want rework it .


Open source doesn't have to be free. I could write a soft, show you guys the source code, but prohibited everybody to redistribute any altered code based on my work.
