### Post by damu on 2006-04-11
Don't forget that to install some softwares, there is even an easier way than Synaptic : Applications/Add applications.

At the end of any section, there is "more programs". The first time, you will click on it, it will ask you if you want to enable universe or multiverse repositeries (something like that). Just say, browse the menu and tick any software that you would like to be installed.
Click Apply, and let Ubuntu download and install everything you asked for!

If you're new to Ubuntu, you will probably want to have some more multimedia funtionnalities. You can also get most of this downloaded and installed for you in a similar easy way with [Automatix]("http://easylinux.info/wiki/Ubuntu#Installing_additional_software_.28Automatix.29").

GO to Applications/Accessories/terminal
Copy and paste in the terminal each of these quotes, on after another :

> sudo apt-get install xterm

> wget [http://beerorkid.com/automatix/automatix_5.7-3_i386.deb](http://beerorkid.com/automatix/automatix_5.7-3_i386.deb)

> sudo dpkg -i automatix_5.7-3_i386.deb

Now, Automatix should be installed. Go to Appliacations/Systems tools/Automatix


Enjoy!
