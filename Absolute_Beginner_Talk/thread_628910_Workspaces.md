### Post by dondad on 2007-12-01
> **jordanae said:**
> I only have two workspaces. How do I get more?
If you have compiz running and have the settings manager, then open the settings manager, go to the general setup at the top and select desktop size. You can set the number of workspaces there. If you don't have an "advanced desktop effects" choice, then go to the synaptic package manager and install compizconfig-settings-manger. Then you will have the settings for Compiz (called advanced desktop effects")in your preferences menu.
