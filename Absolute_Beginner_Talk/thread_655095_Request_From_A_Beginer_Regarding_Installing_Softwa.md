### Post by doorknob60 on 2007-12-31
RAR: My Ubuntu supports it with the default archive manager, but if yours doesn't try ```
sudo apt-get install unrar
```

PDF: Gnome's document viewer displays them jsut fine, but you can download the official Adobe Reader from [www.adobe.com](www.adobe.com) if you need to.

Movies And Audio: Totem and VLC for lovies, RYthmbox and Amarok for Audio
