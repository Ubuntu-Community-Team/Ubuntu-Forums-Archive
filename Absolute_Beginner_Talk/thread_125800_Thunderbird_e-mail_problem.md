Just to let everyone else know how I fixed this I did the following: Go to >System >Preferences >Preferred Applications then where it says web browser I changed it to Firefox. Then for the mail tab I changed it to Thunderbird. Problem solved. Seem's to easy to be true but that was all there was to it.
