### Post by annda on 2007-04-10
repositories are like warehouses for software packages and they have "different profiles". so "main" has the essential, basic software, "universe" has additional packages (still free), "non-free" has packages with licence restrictions, which cannot be redistributed as part of a larger system (like a linux distribution), etc.

i'm not sure about universe and multiverse - what the reasons are for keeping those repositories separate.

anyway, if you enable all the repositories, you will not get multiple packages with different versions. you will simply broaden the base where your system looks for available packages. it takes a few seconds more to check what packages are available but otherwise i'd say it's a good idea: you will simply find more applications and libraries.
