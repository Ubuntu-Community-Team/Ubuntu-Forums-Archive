It is a huge leap forward from Breezy. It does not have a 3D desktop built in. You have to install XGL and Compiz to get the 3D desktop. That only works if your video card supports it.

Driver support is greatly improved as is video. If Breezy installed on your lappy with ease, Dapper should do just as well.

Dapper is still in Alpha, so ther is always the chance something will break during updates. Don't install it if you demand absolute stability. Just be paitient and wait for the final release if stability is an issue.
