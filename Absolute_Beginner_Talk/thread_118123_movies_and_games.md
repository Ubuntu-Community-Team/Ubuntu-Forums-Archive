### Post by AndyCooll on 2006-01-16
To be able to play most dvds the simplest method is to use automatix (see sticky thread at the top). This installs lots of stuff to aid noobies to get up and running and includes certain files for dvd's

For games (assuming they are Windoze versions) you'll probably need to install Cedega or Wine. The apps create a "false" Windoze environment which Winoze stuff can use. There is a Ubuntu gaming forum which might be useful: [http://ubuntuforums.org/forumdisplay.php?f=93]("http://ubuntuforums.org/forumdisplay.php?f=93")

### Post by RaiSuli on 2006-01-16
I recommend Automatix for video players, dvd and divx codecs. After installing them, most movies should play on your system. Getting Windows games to run is more difficult. You will need to install Wine for that or a commercial application called "Cedaga Transgaming". But even with this software, there is no guarantee that the games you like to play will run. Actually games are the only reason I still have a Windows partition on my computer.
