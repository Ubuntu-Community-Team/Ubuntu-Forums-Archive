### Post by Xian on 2005-06-30
[QUOTE=Cerbz]After following the tutorial to install the nvidia drivers on the starter guide, my system monitor is now showing 100% CPU usage... Is this normal???[/QUOTE]
You system monitor? From the main panel:

Applications > System Tools > System Monitor
Click the Processes tab.
In the 'Show' drop-down menu select 'All Processes'

Look down the column marked '% CPU'.
Find the exact name of the process which is using all this CPU.
What is it?
