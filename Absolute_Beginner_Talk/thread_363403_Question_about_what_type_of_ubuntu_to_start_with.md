They all run on the same base. The differences are in the window manager and which programs are installed by default. The various programs (and there are literally thousands available) will all run with whichever WM you choose, so that should not be the determining factor in yoour selection. 
Personally, I would recommend you start with the 6.06 Dapper Drake default install. This will give you the Gnome WM and all of the basic programs that you will need for what you described your desired usage to be. I, also, recommend that you download the "live cd" and try it out on the laptop before attemping to install. This will give you an idea how well things will work on that machine. The one real exception is that, since the "live" cd runs from your cd-rom drive it will be quite a bit slower than your eventual installed versions, so don't let that put you off. If you encounter any problems these forums are a great place to get  help in finding solutions.
I am typing this from my old laptop (ThinkPad 600x - PIII 500Mhz - 384 Mb RAM) using 6.10 Edgy / Gnome and it runs rather nicely. It also ran quite well when I had 6.06 / Gnome and Xfce installed. If, as you say, the one you are getting is from about the time of winME it probably has, except possibly for the amount of RAM, a little bit higher specs than this one. Depending on your video and wifi cards, you should not have any trouble handling Ubuntu. If you do not already have wifi and want to get it, make sure you check the make / model card that you intend to get on these forums to make sure that it wil be compatible (this is one of the most common problems in linux, but easily avoided with a bit of research prior to purchase.
