### Post by Najand on 2007-03-19
If you don't have enough space, installing automatix like softwares is not a good idea, because they waste a lot of free space. Try to install just ubuntu and configure it yourself (of course, if you couldn't do it, ask for help here.) And never remove packages without using apt related packages (apt-get, aptitude, synaptic).
