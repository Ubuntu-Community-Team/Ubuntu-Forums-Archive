### Post by citizenofnowhere on 2007-01-26
This is not a memory test. This is an essential process. On windows, you are recommended to defragment your harddisk and perform other maintenance periodically. What you see every 30 mounts is the equivalent - the program checks your disk for errors, and maximizes your disk space usage.

It shouldn't be disabled, because if your system hangs or crashes the same set of processes will restore necessary data to prevent errors.
