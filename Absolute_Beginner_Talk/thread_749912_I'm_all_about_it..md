```
I am known as "buried" on most sites, I have a year's experience with Ubuntu, 6 years with Windows, and 8 years with Photoshop, now I'm trying to get used to Gimpshop, if I fail, I'm gonna install Photoshop on Ubuntu.
Ubuntu 7.10 is the version that I'm currently using, works fine with Compiz and all that, visit www.ubuntu.com and research.
Let me give you some basic commands if your new to Ubuntu and explain so it's not all complicated.
Now if you're a newb, you'll be wondering "WTH!, how does this work?"
ok Commands:
sudo (a command which places you (your user as root)) (e.g, sudo exec -d "something" (def:Super User and do is (go do something)
apt or aptitude (Downloading Apps, etc.) (e.g sudo aptitude install amarok)
wget or get (to get something, wget is the term used to get a HTTP address and download to install, let's say a .deb package, which will be explained later. (e.g wget http://mitchbuntu.blogspot.com/file.deb or sudo apt-get openarena) this file normally download in to /home/user
```
