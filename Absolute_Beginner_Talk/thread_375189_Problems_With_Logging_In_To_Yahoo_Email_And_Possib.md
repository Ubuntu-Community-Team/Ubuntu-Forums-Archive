### Post by notrublw on 2007-03-03
Today (3/3/2007) I had a problem logging in to yahoo email.  The error message thrown up by firefox said to check to see if personal security manager was installed.

After reading on the mozilla site about what personal security manager does
     [http://www.mozilla.org/projects/security/pki/psm/](http://www.mozilla.org/projects/security/pki/psm/)
I went to 
     Applications
     Add/Remove
    Advanced

and installed the personal secruity manager (it also installed the mozilla browser). 

Now Yahoo Email works fine.

On the windows box that sits next to this one I successfully logged into Yahoo Email
with both the latest version of firefox (2.x) and IE 7, so these applications apparently
have the necessary functionality built in.
