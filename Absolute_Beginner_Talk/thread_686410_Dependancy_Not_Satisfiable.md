### Post by RomeReactor on 2008-02-03
Hi. You need a working internet connection to install software through the package managers--Synaptic, Add/Remove, etc.

If you can't get an internet connection, then you need to download the packages you want [from here]("http://packages.ubuntu.com/") and then transfer them to Ubuntu. However, note that you also need to download the dependencies for what you want to install (marked in red), and the dependencies of those dependencies, and so on.

If possible, it would be best if you could use a friend's internet connection to install the packages through the package managers.
