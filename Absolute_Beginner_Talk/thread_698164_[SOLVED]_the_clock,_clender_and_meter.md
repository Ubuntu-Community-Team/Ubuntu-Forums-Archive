### Post by Presto123 on 2008-02-16
Try Gdesklets in Add/Remove. You pretty much need to build the panel from scratch, but runs well.

And for adding shortcuts, find the program you want in the menus by right-clicking on the menu bar and clicking on "Edit Menus". It lists all of the programs you have and by clicking on a program and then right-clicking it and selecting "Properties" you can navigate to the launcher properties of the program. This will get you the command you need to add it to the launcher. Most likely, you can right click on the AWN bar and there should be an option to add a launcher. 

Type this in the command box and see if it works:
```
audacity
```
