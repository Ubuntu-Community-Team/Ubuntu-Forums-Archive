### Post by byen on 2006-01-19
As paulmilliken suggested It is advisable to use gaim (gnome) or kopete (kubuntu). With the help of those you can not only connect to yahoo but also MSN,AOL,Jabber,Gtalk etc. As for Yahoo Messenger..there is one available for linux ([click here]("http://download.yahoo.com/dl/unix/ymessenger_1.0.4_1_i386.deb")) but I have to warn you...it is a very basic,stripped-down version and has nothing more than simple-text chat options... so try them both and decide which one is best for you.

Note: the package that is in the link will work in ubuntu. Just save it in your home directory and type this in the terminal:
sudo dpkg -i filename.deb

Hope that helps...goodluck!
