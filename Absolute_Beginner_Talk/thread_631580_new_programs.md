### Post by Rinzwind on 2007-12-04
gFTP. Freeloader, Filezilla all have an icon inside the menu after installation. Goes for anything you can install with 'add/remove programs'. So just use 'add/remove' programs' if you desire a menu item. Synaptic has this nifty feature called "properties" that you can use after installation. It shows the files installed and therefor also shows if there's a menu item or if it's a command you need to execute from command line. 

I do not agree on the GUI requirements. GUIs are cool and all that but sometimes it's a lot easier and quicker to do it at the command line, FTP is one of those: command line style FTP is alot better then with a GUI. Same goes for mounting remote systems, SSH and wget. I love command line. Since I use linux for work aswell and I have to connect to SCO, Redhat, Ubuntu, Suse and some other flavors I know 1 thing for sure: the command ssh always works and a GUI for ssh tend to be different on any Unix or Linux.
