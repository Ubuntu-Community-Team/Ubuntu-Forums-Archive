### Post by JetSirus on 2007-05-19
I don't recommend using a virtual machine for loading 3d modeling software in.  If you can get the software to run at all it will run like a dog.  On the other hand if you want to try Linux anyway I would recommend you first give Blender a look.  It's a professional grade 3d modeling studio that does most everything 3D Studio Max does, perhaps even more.  Best of all it's well supported on Linux.  [Here]("http://wiki.blender.org/index.php/BSoD/Introduction_to_Character_Animation") is a good site for learning it. 

Alternatively if you really need to use your windows based software in Linux you might give [Wine]("http://www.winehq.org/") a try.  It's well supported and runs many windows programs well.  Also, you will want an NVidia branded video card as it's way better supported than most others.
