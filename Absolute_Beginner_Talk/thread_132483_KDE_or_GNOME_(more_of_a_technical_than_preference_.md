### Post by nalmeth on 2006-02-18
there is a separate "kubuntu" release, because some people heavily favor it, and don't want to install kde after ubuntu installs gnome by default. The only differences are in the way the two desktop environments are setup, or rather how the ubuntu developers have set them up to interact with ubuntu.

Most of the technical problems you will hear are actually preference problems --> people don't like presets/defaults of either environment.

I have heard about some real tech problems with kubuntu, but I think you'll find it working just fine. If you've got the room, use both! Its what I do, though I'm usually logging into gnome
