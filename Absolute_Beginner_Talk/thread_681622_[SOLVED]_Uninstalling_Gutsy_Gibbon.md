### Post by jpeddicord on 2008-01-29
You can't uninstall Ubuntu - to remove it and start over, just put in the new disk you have and install over top of it (ie, "Use the whole drive" or "Erase this drive" in the installer). That will replace Ubuntu with whatever distribution you want to put on it.

### Post by gn2 on 2008-01-29
Start the Live CD and begin the installation process, select Manual partitioning then instruct the partitioner to format the partitions.
You'll need to set the mount points as well which is / if there's only one partition. 
(or / and /home if you created a separate /home)
If you didn't before, now would be a good time to create a separate /home partition.
