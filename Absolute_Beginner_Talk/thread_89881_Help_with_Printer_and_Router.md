### Post by Herman on 2005-11-13
Hello, 'WhiteWitch', I  have a four port router too, and mine always gets automatically set up during the install providing I have the router plugged in while the install is being performed. I would imagine that would work for most makes and models of router, you should just find it works automatically, by itself.  I don't know how to configure one manually after the install or if it is a rare brand of hardware.
If it was not plugged in during the install, you could repeat the install with it plugged in, (it will only take you about half an hour). 
 As for your printer, Ubuntu comes with drivers for most printers, have you tried 'System', 'Administration', 'Printing'? You can then click on the 'New Printer Icon', and it will pretty much walk you through finding the right printer driver. I'm not sure about the scanner side of things, if it's an 'all-in-one'. Try 'Applications', 'Graphics', 'Xsane Scanning Program', and see if that helps.
Those ideas should work if your hardware is supported, if not, then if you are dual booting with Windows, you can still pass your files from one operating system to the other to make use of the hardware, but it may be less convenient, so I hope all your hardware is able to be configured in Ubuntu. :smile:  (Herman)
