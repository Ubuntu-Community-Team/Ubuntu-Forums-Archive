### Post by RomeReactor on 2008-01-25
Hi. Download [Google Earth]("http://dl.google.com/earth/client/current/GoogleEarthLinux.bin") to your desktop; then open a terminal (Applications->Accessories->Terminal) and change directory to the Desktop:
```
cd Desktop
```
then:
```
chmod +x GoogleEarthLinux.bin
```
and lastly:
```
sudo ./GoogleEarthLinux.bin
```
