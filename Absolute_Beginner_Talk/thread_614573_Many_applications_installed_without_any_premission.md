### Post by Matakoo on 2007-11-16
The extra software (at least those seen on the screenshot) are programs that are a part of KDE. Ark is a KDE-application so they were most likely installed because Ark depends on certain KDE libs and/or apps. The easiest way of not getting those programs in the first place is to make sure to not install KDE-based apps, which sometimes can be easier said than done.

They don't harm anything though, so there's no need to remove all KDE-apps. They just make the menus rather cluttered. I use only KDE myself so I can't say for sure how to do it in Gnome, but I assume there is a way somewhere to edit what you want in the menus? Someone more used to Gnome will have to answer that. The point is that you probably want a way to just remove the menu-entries for the programs you do not use instead of removing all KDE-apps.
