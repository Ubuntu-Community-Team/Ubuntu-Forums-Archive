### Post by comppsyco on 2006-08-28
I don't know mandrake, or linux for that matter, particularly well, but I'm assuming that if you're using a drag and drop tool for burning, it ain't gonna work. What you're probably doing is burning the ISO file as an ISO file, instead of expanding it onto the disc. 

An ISO file is a "disc image." It is one file that contains all the files on a disc, but if you burn the ISO file onto a disc, that's all that your computer will see, is the ISO file. Instead, you need to use a CD burning program that will extract the files within the ISO file and burn them to the CD. On Windows (unfortunately the OS I know best) that would be Nero or Roxio, or, my free favorite, CDBurnerXP Pro. I'm sure that someone can point you towards some good linux CD burning software.
Sam
