### Post by kent41 on 2006-09-23
UPDATE 

After burning a few CD's I found that you can write to a CD-R many times (if Space is available) if you burn the CD in Multisessions. After you close the first session the next time you want to add more data to the CD-R select the continue multisessions options and you can burn the additional data to the CD.
