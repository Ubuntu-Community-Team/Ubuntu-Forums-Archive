It is a just a frontend to a backend server over a network or is it a frontend/backend on the same machine or do you have more than one frontend?

The reason I ask is because there are several different configs with each setup. I would suggest checking out the Mythbuntu manual located here:

[http://www.mythbuntu.org/installation_manual](http://www.mythbuntu.org/installation_manual)
