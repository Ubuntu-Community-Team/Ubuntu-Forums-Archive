### Post by rax_m on 2007-07-15
ok.. wait a minute.. are you talking about the workspace switcher? or the task panel?

The workspace switcher is the on which shows the different virtual desktops
while the task panel shows you which windows are open. 

You can configure the task panel to show you the applications open on all workspaces if you want to. You need to right-click on the task panel and go to preferences. Then select "Show windows from all workspaces".
