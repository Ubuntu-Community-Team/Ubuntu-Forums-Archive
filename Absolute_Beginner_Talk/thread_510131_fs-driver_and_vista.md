### Post by Kobalt on 2007-07-26
I remember using this driver when I tested Vista and it worked fine... Once you installed it, can you see a list of your partitions and select your ext ones ? Do you even see this list or is it empty ? 
Appart from installing the driver you can use [explore2fs]("http://www.chrysocome.net/explore2fs") which is a little programm (to run as Administrator) that allows you to browse ext2/3 drives in Windows. I also tested it in VIsta and it worked.
Concerning ext4, as far as I know it is still in developpment so unless you're a skilled Linux user (considering you might screw up the partition you would format) I don't advise using it appart from testing purposes.
