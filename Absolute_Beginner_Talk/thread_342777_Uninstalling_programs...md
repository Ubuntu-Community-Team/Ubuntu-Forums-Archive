### Post by an.echte.trilingue on 2007-01-20
xubuntu-desktop is a meta-package, meaning that it does not contain anything, but depends on all the elements of xubuntu to make sure that they are installed properly, for example for people who have Ubuntu and want to try xubuntu.  Since thunderbird is part of the xubuntu experience, that is one of the packages it depends on.

You can safely remove it now, though.
