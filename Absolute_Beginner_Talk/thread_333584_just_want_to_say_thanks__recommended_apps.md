### Post by bottleman on 2007-01-07
Hey there, I just thought it might be nice to say thank you to the many Ubuntu developers and people who contribute to the wikis and forums.   For the first time for me, Linux is really working for me.

I've been occasionally installing Linux on miscellaneous PCs for years, always hoping that I'd be able to get to work fast enough to make a Linux desktop actually productive (as opposed to just an exercise in tweaking drivers, etc.).  It was always interesting, but I always ended up going back to Windows to actually get work done.

After a series of bizarre blue screen errors I finally decided to try linux again, this time Ubuntu.  First I installed Dapper on an aging laptop for my mother in law... and changed desktop to Xubuntu.  Then I installed Edgy/ Kubuntu on my own laptop in a dual boot setup (gotta have legacy data & apps).  

To my great surprise, there was really good hardware detection, so a lot of the driver stuff was sorted out for me.  I can't say it was completely smooth but there was a real friendliness and willingness to help a newbie on the support forums.  For the first time I felt supported enough to really try to work in the command line millieu.  But happily that was only for one thing  -- getting the external wireless card on my MIL's laptop to work was a trial.  Still, I did it. :)

I think I've finally made the switch to Linux as a working environment.  Here's some apps that are going to make it work:
- OpenOffice 2 (much better than OpenOffice 1)
- Firefox 
- Zotero for research, bibliography, & notetaking
- R for statistical analysis
- digiKam for photo management.. (wish 0.9 was out in a package right now but building one is still beyond me)

Cheers!
