### Post by PriceChild on 2006-10-24
I reccomend the live cd as you can run it from the cd without changing your hard drive.

You can do this to ensure you do like it, before installing. There is an icon on the desktop of the live cd so you can always install it when you're ready.

However, if you KNOW you want to install it, then i reccomend the alternate cd. I think it is a more stable install, especially when resizing partitions.
