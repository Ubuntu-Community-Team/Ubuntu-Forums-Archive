### Post by felicity on 2007-07-14
Did you try System > Administration > Network Tools? I'm not positive this is how you adjust a wireless card as I've never used one, but you can configure your network interfaces there.

If you don't see that in your menu, go to System > Preferences > Main Menu and under Administration check Network Tools. Then it should appear in your menu.
