### Post by Equinsu Ocha on 2008-03-01
Press ctrl + h, or use View > Show Hidden files from the top menu bar while in your home folder. Any folder that starts with "." is a hidden folder.

---

### Post by macogw on 2008-03-01
Are you looking in the file browser?  Hit Ctrl+H or View->Hidden files.  If you're in the command line, use "ls -A" instead of just "ls".  Any file that stars with a . is a hidden file (part of the UNIX standard).
