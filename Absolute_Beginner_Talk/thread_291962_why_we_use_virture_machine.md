### Post by kidders on 2006-11-03
Hi there,

You may be thinking of virtu*al* machines. The main advantages are:

[LIST]
[*]You can run operating systems intended for one architecture (PC, say) on another (like a Mac).
[*]You can use two OSs simultaneously.
[*]You can install an OS without mucking up your computer, and delete it again very easily.
[/LIST]

It's something I've only ever done with Windoze, because I've never been prepared to put a "real" installation of it onto my PC.

In practical terms, virtual machines often have significant disadvantages though. If you don't have a specific reason for needing to use one, I wouldn't bother.

I hope that helps ... I'm sure others will have more to add to my comments.
