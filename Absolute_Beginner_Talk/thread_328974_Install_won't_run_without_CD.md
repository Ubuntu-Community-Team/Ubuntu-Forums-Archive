### Post by Kazna on 2006-12-31
Get a Live CD: [http://www.ubuntu.com/products/GetUbuntu/download#currentrelease](http://www.ubuntu.com/products/GetUbuntu/download#currentrelease)

Boot from it, and d.click the **install **icon you see on your desktop - go through the wizard, choose your partitioning, your swap space and it'll be installed!

 You need to keep the CD in the drive. As you see there is lots of information on the CD. The CD will stay there till you are prompt to remove the CD after the install is complete. So if you burn the ISO and ISO format CD and burn it at a nice 8x or 10x speed (Faster is not better when burning ISO data) 
 Do not eject the CD before the install is complete. You'll end up with a broken system.
