### Post by Tomosaur on 2007-01-25
You can run Ubuntu alongside Windows if you so wish. This is called 'dual-booting', and you are given the choice of whether to load Ubuntu or Windows when you switch on your computer. You **may** be able to get CS:S working, but I just never bothered after my first failure. Could be worth a try though, but I highly doubt it would work straight away with no fiddling.  Cedega is a 'paid-for' version of Wine which has had extra work in the stuff required for games. [Check here](http://www.transgaming.com/index.php?module=ContentExpress&func=display&ceid=2&meid=-1) and see if CS:S is working correctly using Cedega. 

As far as general performance goes, my Ubuntu install wipes the floor with Windows. It's fast, slick, and I have not once heard my hard drive churning and grinding (probably due to the ext3 filesystem - no defrag required :) ).

Ultimately though, it's up to you :)
