### Post by Klaidas on 2006-08-27
.deb is a package, like a program, that you can install using dpkg.
You can also install using apt-get, aptitude, synaptic, compiling, etc.

[QUOTE=Wikipedia]deb is the extension of the Debian software package format and the most often used name for such binary packages. Like the term Debian, it's named after Debra wife of Debian's founder Ian Murdock.[/QUOTE]
