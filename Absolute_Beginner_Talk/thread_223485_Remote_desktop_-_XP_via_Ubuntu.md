### Post by cilynx on 2006-07-26
Security is in the eye of the beholder.

I do 99% of everything with computers under Linux.  However, for work I have to use Bobby Watchfire, which doesn't work with Wine.  I have a dedicated XP box that does nothing but run Bobby for me.  I have realVNC running on said XP box in the back room and I connect to it with any of the VNC clients that come packaged w/ Ubuntu.  Works like a charm.  

You can also SSH into a Win* box if you have the SSH server installed, but you're not going to get anything more than a DOS prompt.
