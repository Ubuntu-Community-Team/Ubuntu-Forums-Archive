### Post by pseudonym on 2006-12-31
Hi, what you need to do is to stop the gnome display manager (gdm) from loading. See [this thread]("http://www.ubuntuforums.org/showthread.php?t=313241&highlight=startx+gdm") on how to do it (there are others). I found it by entering 'startx gdm' in the search box ;)
