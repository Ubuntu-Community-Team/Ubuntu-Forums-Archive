### Post by arjun.shankar on 2006-08-10
It is just as safe as running the servers on the server install. No difference. The server install is minimal, has no X, and so on.
Obviously, this means that since the desktop install has more programs, it is more vulnerable (more programs always = more vulnerabilities). But that's hardly any reason to bother. It's safe.
