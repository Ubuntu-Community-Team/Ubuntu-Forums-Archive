### Post by Maxtine on 2007-04-26
Information on this site may correct your problems  [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download). Go down to the bottom of the page and select a download site and download the ISO. 

Then goto [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto) and download the installer Infra Recorder it is free follow the directions. If you have any data you want to save just save it, check and make sure all your hardware is attached then do a clean install. Then download and install Automatix2

You be good to go
