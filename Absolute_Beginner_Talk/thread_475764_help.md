### Post by ShareBuntu on 2007-06-16
You can erase everything and solely run Ubuntu or you should have a look at a good dual-boot guide to run both Windows and Ubuntu. You get to choose which one to boot into when you start your computer.

The Ubuntu installer will try to import some information from your Windows account but it's best to make backups. As for MSN Messenger, GAIM (now called Pidgin) comes pre-installed on Ubuntu Feisty. Broadband will get picked up automatically as well as wireless.
