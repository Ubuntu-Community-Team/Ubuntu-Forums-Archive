### Post by Waappu on 2007-02-21
Hi

Open Beryl Setting Manager
Go to Desktop tab
Select Desktop cude
In options tab enable "Slide when changing viewports instead or rotating"

Now if you press Ctrl+Alt+left arrow it slides and
if you press Ctrl+Alt+grap with mouse button 1 you can rotate

Edit
or if you press mouse button 1 + mouse button 2 you can also roate cude
