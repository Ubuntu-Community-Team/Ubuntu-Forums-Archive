### Post by Peepsalot on 2006-11-04
There are programs for linux to do audio editing, video editing, DVD creation, etc.  I haven't personally used any of the programs you listed, so I don't know what they are like, and how the linux programs would compare to them.  

The Ubuntu guide([http://ubuntuguide.org](http://ubuntuguide.org)) has some basic suggestions for some of the more commonly used programs in Linux.  
examples:
[How to install an Audio Editor](http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_install_Audio_Editor_.28Audacity.29) (the guide recommend Audacity for this)
[How to Install a Video Editor](http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_install_Video_Editor_.28Kino.29) (the guide recommends Kino in this case)

Since you are new to linux, and it sounds like you are very dependent on these programs, I would recommend trying a dual boot setup at first to see how it works for you.  If you aren't familiar with dual boot, this means that each time you start/restart your computer, you'll have a menu to chose between going into windows, or going into your linux installation.  There are many tutorial on the net for how to configure a computer for dual booting.

It should be possible to install these same programs, no matter which distro you choose.  Probably the only difference will be in the method(and possibly the difficulty) of program installation.  Of course, I'm partial to Ubuntu ;), and installing programs is usually very simple with it.
