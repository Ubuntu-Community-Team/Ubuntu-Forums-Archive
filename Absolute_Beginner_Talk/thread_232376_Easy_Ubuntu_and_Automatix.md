### Post by Metacarpal on 2006-08-08
Each one installs various codecs and useful applications and libraries.  They have some differences in what they install:

[Automatix]("http://www.ubuntuforums.org/showthread.php?t=177646")
[EasyUbuntu]("http://easyubuntu.freecontrib.org/overview.html")

They each also have their little peculiarities, hangups, bugs... The projects are works in progress (part of the beautiful continual evolution of open-source software).

I'd recommend checking out [their forums ]("http://www.ubuntuforums.org/forumdisplay.php?f=46")and reading up a bit on their capabilities and common issues before making a decision.

But you really only want to use one or the other.

### Post by msak007 on 2006-08-08
I've only used Automatix so I can't comment on EasyUbuntu, as I've always had good luck using Automatix so never really needed a reason to try EasyUbuntu. Automatix provides a ton of options / apps, and the devs are always taking requests to have things added. It seems to me EasyUbuntu is about being simple and only installing the basic codecs and system apps. Installing Automatix is as simple as adding a repository, updating, and doing an apt-get. A big plus for Automatix (for me) is that there's now a KDE / Kubuntu only version, as it was more Gnome-centric in the past and focused only on Ubuntu. It comes down to what you need, trying both, and seeing which one you like better. If you want only the codecs and basics to get your system running, both can do that. If you want more choices of apps you may have never heard of, but may in fact want / need, try Automatix.
