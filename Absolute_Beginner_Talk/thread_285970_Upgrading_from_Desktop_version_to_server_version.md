### Post by CatKiller on 2006-10-27
As everyone has said, you can get either set of functionality from either starting point. You don't exactly have a "Home" and "Corporate" version of Ubuntu like you do with Windows - it's more to do with what you use it for than how much you pay for it with Ubuntu. You may want to get your head around the idea of "packages" and how that works with Ubuntu.

You almost certainly (since you're posting in the Absolute Beginners section) **don't** want the server version. The server install is **entirely** command-line based and comes with pretty much nothing that you'd expect on a generic home computer. Functions great for actual server usage though.
