### Post by Bachstelze on 2007-01-01
Yes, you can download and use the build from Mozilla but it's recommended to not overwrite the default Ubuntu build.

### Post by aysiu on 2007-01-01
> **Quillz said:**
> What would happen if I was to overwrite it? And is it possible to run Firefox 2.0.0.1 seperately, in addition to the bundled 1.5.0.8?
In fact, if you use either the wiki instructions or the aforementioned script, you will have two separate installations--1.5.0.8 and 2.0.0.1.

The command *firefox* will launch 2.0 and the command *firefox.ubuntu* will launch 1.5.
