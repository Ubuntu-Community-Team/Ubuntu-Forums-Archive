### Post by BoyOfDestiny on 2006-02-03
Hi and welcome :)

There is a software called WINE that lets you run many windows apps on top of Linux.

Check out [http://appdb.winehq.org/](http://appdb.winehq.org/) for a list of software they have tested.

There is wine available in the ubuntu repos or you can follow the instructions for debian/ubuntu on the wine home page at [http://www.winehq.org/](http://www.winehq.org/)

### Post by xmastree on 2006-02-03
[QUOTE=holmquistc]Hey all, I am completely new to this whole thing.  I was wondering, can I install regular programs such as Photoshop and Microsoft Office in Ubuntu?[/QUOTE]
Short answer: **No**. Use Gimp and Open Office instead.

Long answer: **Yes**, but it requires running Wine, and it's not that easy for a complete beginner.

Migrating from MS office to Open office is pretty easy. Do you drive? Ever driven a different car from the one you usually drive? Everything's there but in a different place. It's the same thing.

Photoshop to Gimp isn't so easy though. Learning it is fun (for some of us :rolleyes: ) but it's more like moving from the car you're used to on to a motorbike.
