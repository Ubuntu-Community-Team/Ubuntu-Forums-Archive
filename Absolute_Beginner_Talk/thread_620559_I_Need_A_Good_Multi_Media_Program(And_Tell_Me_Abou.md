### Post by tiachopvutru on 2007-11-22
Well, Multi Media Program includes a good number of stuffs, so it'd be best to be more specific

If you're using Ubuntu (and thus nautilus, assuming you haven't change your file manager), to show hidden files press Ctrl + H, or alternatively, on the menu bar, go View > Show Hidden Files

As for multimedia programs, I'll recommend a few

For video player: SMPlayer (it's a frontend of MPlayer so you can try Mplayer only instead, but SMPlayer is far better than MPlayer's GUI imo)
For audio player: Amarok or Exaile, or the pre-installed Rhythmbox
For burner: K3b or Gnomebaker

You can try avidemux for a virtualdub-like video editor. Also, there's a program called mythtv, I don't know what it specifically so you'll need to ask someone else about that.
Try gtk-recordmydesktop if you want a program to record your desktop...

### Post by dips_xe on 2007-11-22
if you're on xubuntu, the file manager is thunar and there you go view >> show hidden files.

for music,
for xubuntu, i use exaile or quod libet. (no kde or gnome libs)
for ubuntu, i'd also use the above or rhythmbox or banshee (although i hear they can crash every so often, but try it out)
for kubuntu, i'd use amarok or the first two.

if you want something very small and customizable like winamp, use beep media player, regardless of os flavor.

for video,
i'd use vlc media player or mplayer. only two worth mentioning, whatever your preference is (i use vlc). os flavor doesn't really matter here.

for burning,
brasero, nero, gnomebaker for xubuntu/ubuntu
k3b for kubuntu.

for torrents,
don't use ktorrent on xubuntu or ubuntu, only for kubuntu. kubuntu can also use below.
deluge (small but nice design, basic features), azureus (powerful and complex, resource heavy, can crash), transmission (very small, very dependable, few features)

for irc
xchat is decent for any flavor.
pidgin can also use irc, not very intuitive, i only use pidgin for chat.

for fun games, try frozen bubble, battle of wesnoth, warsow, xmoto, bloboats.
