### Post by Hospadar on 2008-04-17
I have a machine with one of those little via boards that does all my bittorrent.  I use deluge because it has a really nice webUI you can make use of.  I just run plain old vanilla ubuntu (which is probably better for a server like that, KDE is more demanding on resources).  You could just make the save location inside deluge the external drive.  If you can get it working, it might be nice too, to just mount /home to the external, and put the root (/) on the internal.
