### Post by hk_2999 on 2007-02-03
If your computer crashes often, you might need these tricks to minimize the damage you do every time you force reboot. Type these keys in order, if you cant find a sys-rq key on your keyboard, it's also commonly called the print screen key.

Alt-Sys-r : put the keyboard into raw mode
Alt-Sys-s : instantly sync all hards disks
Alt-Sys-e : send the terminate signal to all process
Alt-Sys-k : kills all process of the acutal virtual console
Alt-Sys-i : stop all process (kill signal)
Alt-Sys-u : put filesystems into 'read only' mode
Alt-Sys-b : Reboot
Alt-Sys-o : Switch off
