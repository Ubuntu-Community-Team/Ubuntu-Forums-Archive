### Post by PmDematagoda on 2007-12-27
You can get a full list of applications that are compatible with Wine from [here]("http://appdb.winehq.org/appbrowse.php").

To run an exe using wine, use a terminal and navigate to where the exe is found, then execute:-
```
wine nameoffile.exe
```

Or, you can just right-click on the file while in the File Manager and then Open With>Wine.
