### Post by Sef on 2006-12-16
1) Don't burn at faster than 4x.  Faster than that can cause the burn not to go well.  

2) When installing with the alternate cd, just go with the default option.

Update:

3) If you find it running slow because of your processor, then consider using [Xubuntu]("http://xubuntu.org").  Same backend but the desktop is made for older computers.
