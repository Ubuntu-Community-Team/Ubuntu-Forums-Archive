### Post by pommattski on 2007-05-01
I had the same problem.... New install of **U**buntu Feisty...

... But it looks as though it's just a problem with "that button" to start the script manager.

If you go to Tools (drop-down menu at the top) and select "Script manager", select "Lyrc" (under "Lyrics"), then click "run", Amarok will fetch lyrics.

I've exited and restarted Amarok, and rebooted, and the Lyrc script is running automatically now.

[SIZE="1"](... and I found [this bug report]("https://bugs.launchpad.net/ubuntu/+source/amarok/+bug/110502") which points out that "Starting the script manager from the menu works")[/SIZE]
