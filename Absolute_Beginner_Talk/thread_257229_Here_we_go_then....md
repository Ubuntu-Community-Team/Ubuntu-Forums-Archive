### Post by clint1010 on 2006-09-14
Welcome to the ubuntu community. 

Have no fear, there are plenty of "switched on" ubuntu users here that can help you out if you have any trouble. Try searching for solutions first if you have a problem, your question may well be answered already.

Do you have the Daper 6.06 live CD? if so, you will find installation a breeze, no comandline stuff at all. Its all done with a graphical user interface (GUI).

There is great support for laptops as well, most users find the install auto detects all hardware straight out of the box.

Most things can be done from the desktop using a GUI, so there is no real need for regular users to worry about using a terminal for comand line type instructions. But have a play around, I find some things easier and quicker using a terminal. Some guids are written to use a comand line in a terminal, and are very easy to follow, so it would pay to familiarise yourself with it.

Have fun
Clint
