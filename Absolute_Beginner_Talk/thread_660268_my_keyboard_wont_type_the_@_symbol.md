The layout of your keyboard depends on where you live. Since the software has no way of telling where you live and what symbols are printed on the keys of your keyboard, you must tell it what kind of keyboard you are using.

On the very first screen which you see when booting the live CD, you can press F3 to set your keyboard and F4 to set the language (or is it the other way 'round?).

On the other hand, you can select the menu System/Settings/Keyboard (or some such; I don't use the English version of Ubuntu).
