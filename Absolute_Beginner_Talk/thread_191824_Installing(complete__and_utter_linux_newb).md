### Post by _simon_ on 2006-06-08
You need to burn the ISO direct to CD. If you look in the menu of whatever program you are using there should be something similar to "burn ISO image"

Once you have it burnt, make sure your CD drive is bootable in the BIOS, stick the CD in and reboot. I assume it's Dapper Drake you have downloaded? In which case make sure it was the Desktop CD you downloaded - when you reboot you will be prompted whether you want to boot from CD or not (which you do) it will then boot into dapper from the CD (kind of a preview) and all you need to do is click the "Install" icon on the desktop and follow the prompts!

### Post by _simon_ on 2006-06-08
In Dapper they are the same thing. 

It used to be that the Live CD would give you a preview so that you could see what hardware worked and if you liked Ubuntu. You would then use the installer CD to install it.

Now the Desktop CD is the Live CD + installer!

The blurb about dapper can be a little confusing as it talks about the desktop cd and live cd, they really need to stick to using one term I think. But you will find any links for the dapper live cd take you to exactly the same place as the links for the dapper desktop cd :)
