### Post by munga_bill on 2006-03-16
Yes, 4.0 GB should be fine, as long as you are not planning on storing too many huge files. It will be enough to try out the operating system and save quite a number of word or spreadsheet files, e-mails and the like. You might run out of room if you try to store too many pictures and music files. The Ubuntu install in my laptop is using 3.72 GiB at the moment complete with automatix installed and all the tweaks and setting I want. I only have a few simple files in it. I have a 40.0 Gb hard drive but I hardly need that much space for what I do. Regards, munga_bill.
