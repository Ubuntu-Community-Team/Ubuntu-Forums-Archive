Hello everyone. I would like Ubuntu to select a default user and login automatically upon startup. How can I do this?

### Post by cowlip on 2007-03-30
System->Administration->Login Screen

There are options for logging in automatically there

"Log in a user automatically at first boot up"

[http://www.redhat.com/docs/manuals/enterprise/RHEL-4-Manual/desktop-guide/s1-ddg-lockdown-other-kiosk-configs.html](http://www.redhat.com/docs/manuals/enterprise/RHEL-4-Manual/desktop-guide/s1-ddg-lockdown-other-kiosk-configs.html)
