### Post by ugm6hr on 2007-05-21
Mediaplayer connectivity plugin isnt a media player.  It merely allows you to decide which embedded file types are handled by which media player.  I use VLC for everything except realmedia (realplayer).

You can download and install it direct from the firefox addons website.  Once installed - restart firefox, and the wizard should appear.  If not: Tools -> media player connectivity -> configure

[https://addons.mozilla.org/en-US/firefox/addon/446](https://addons.mozilla.org/en-US/firefox/addon/446)
then just click install.
