### Post by sumguy231 on 2007-08-12
If nobody's already mentioned this, I recommend setting up a dual boot in which both operating systems are installed. It takes a few more minutes to install that way, but it will save you effort in the long run. I'm sure there's a guide for it floating around, but here's the basics:
1.) You'll want to defrag your XP partition first to make it resizeable
2.) When you do the Ubuntu install, there's some sort of option to resize the XP partition and use the remaining space. Choose this.

> Do you think i should get it connected to the wireless using live boot before i install it couse i am having troubles connecting
As you may know, anything you do to a LiveCD session won't stick, so there's not much point trying to get your wireless card working before you install. Depending on your card, it might be really hard to get it to work, I'll go ahead and warn you. But if you post here with the details once you have the system installed, I'm sure someone will try to help you out.

### Post by gn2 on 2007-08-12
One very simple way of preserving your Windows installation is to pull the hard drive out, put it away in a safe place and fit a new hard drive to install Ubuntu on.
.
When you are happy that Ubuntu meets all your needs and you no longer require Windows, get the old drive out, put it in a USB enclosure and format Windows into oblivion.
.
You can then use the old drive for back-ups/additional storage.
