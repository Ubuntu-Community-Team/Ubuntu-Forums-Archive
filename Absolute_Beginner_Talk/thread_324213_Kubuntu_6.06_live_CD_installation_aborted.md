### Post by jvc26 on 2006-12-23
If you use the live cd again it will just walk you through the install steps it did last time so you can get back to the partitioning section and choose the partitions you want ubuntu to be installed to - make sure you tell it you want to choose the partition tables manually (so as to ensure that you're doing the install to protect your '98 partition. You shouldnt wipe the windows partition unless you select it and tell it to install it over it. Basically just run the live cd again and then when you get to the partitioning section select the partition you want to install ubuntu to and then go for the install.
Hope that helps
