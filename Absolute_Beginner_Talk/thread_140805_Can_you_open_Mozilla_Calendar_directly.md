### Post by EndersGame on 2006-03-30
[QUOTE=twoseids]I was wondering if it is possible to open up Mozilla Calendar without first opening up Thunderbird. I'm trying to use the calendar for reminders and stuff like that, but it doesn't do any reminders if the program isn't open to begin with.![/QUOTE]
In theory you should be able to just add the -calendar argument to thunderbird when running it.  That is: ```
thunderbird -calendar
```  However, some versions of calendar have difficulty with this and require the workaround described here: [http://forums.mozillazine.org/viewtopic.php?t=369617](http://forums.mozillazine.org/viewtopic.php?t=369617)  (scroll down to Rod Whiteley's third post.)
