> **BLTicklemonster said:**

Thats the way I always ran it with windows. Dvd decrypter first and then use dvd shrink on the files outputted by dvd decrypter. I also had dvd shrink set to burn the dvd with Nero. In ubuntu I use dvd::rip to rip the dvd (it seems much slower than dvd decrypter :( ), then I use dvd shrink running in wine to reencode and shrink the files. I have dvd shrink set to output the files as an iso which I then burn with gnomebaker. Hope this helps.
