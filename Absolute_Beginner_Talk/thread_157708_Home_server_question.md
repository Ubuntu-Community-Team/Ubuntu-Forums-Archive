People use a *Server* to provide *Services*, these can include:

Common file storage resources;
Network services (Firewalled Internet access etc.);
E-mail/spam filtering (if you run your own Domain, or want to have all your incoming e-mail pre-processed);
DNS;
Printing;
etc. etc. etc.

If you need something independent of your desktops to provide these services, and you don't mind a bit of hardware being powered up all of the time, then perhaps this is the way to go. If you don't need all of the overhead associated with keeping these things going, then perhaps not.
