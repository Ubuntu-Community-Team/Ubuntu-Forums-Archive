### Post by marshall.robert on 2008-02-11
ill also add, that even with 7.10 the stuff for playng mp3's isnt installed by default. to get all the required software and plugins go:

Applications -> Add/Remove
in the search box type ubuntu restricted, wait a sec, then tick the Ubuntu Restricted Extras package and click apply, that will install all that is needed for playing mp3's, movies, DVD's as well as java and some windows fonts. Basically all the stuff needed to get along.
