### Post by aphirst on 2008-02-20
Might I ask why you feel the need to run ipblock with deluge as well as deluge-s blocklist importer. The deluge plugin is essentially the same as ipblock, but only uses one blocklist. Ipblock can use many.
I've been absolutely fine by running

```
$ sudo ipblock update && sudo ipblock start
``` and then opening deluge.

I take it this is simply for moral reasons, to protect your IP whilst torrenting, and not for protection whilst pirating ;) .
