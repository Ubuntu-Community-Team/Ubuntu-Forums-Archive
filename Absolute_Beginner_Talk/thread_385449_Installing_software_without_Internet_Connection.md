### Post by benfindlay on 2007-03-15
You can use your cd as a repository by enabling it in Synaptic Package Manager. As for repositories. You can navigate via web browser to their locations and save them to disc on another computer.

For exapmle you can go to: [http://archive.ubuntu.com/ubuntu/]("http://archive.ubuntu.com/ubuntu/") or [http://security.ubuntu.com/ubuntu/]("http://security.ubuntu.com/ubuntu/")

Click "Pool" then browse the 4 sub categories and their contents for the files you want!

Hope this helps!
