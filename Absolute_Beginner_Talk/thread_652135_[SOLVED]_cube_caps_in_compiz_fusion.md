### Post by SOULRiDER on 2007-12-28
Install gnome-compiz-manager
```
sudo aptitude install gnome-compiz-manager
``` That willa llow you to change compiz configuration and select images for the cube caps.

---

### Post by FuturePilot on 2007-12-28
Or better yet 
```
sudo apt-get install compizconfig-settings-manager
```

---

### Post by billgoldberg on 2007-12-28
Yes, install that package (can be done using synaptic too).

Then there is an option somewhere called cube caps, activate that and change the cube caps there.

In the "destkop cube" option you can also change the caps, but that doesn't work, it has to be done in "cube caps".
