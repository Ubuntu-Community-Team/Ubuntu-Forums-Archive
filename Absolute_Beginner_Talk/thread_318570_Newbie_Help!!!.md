### Post by deodeep on 2006-12-14
Ahh, An Indian :)

Welcome...

Well, Ubuntu, by default, doesnt come with mp3 playback support.

You'll need to install gstreamer bad/ugly to enable mp3 playback..

[Read this]("https://help.ubuntu.com/community/RestrictedFormats#head-dd75cf9cf440f1782515e6df311684e6f981b4e4") and you'll know what to do..

I'm kinda new to Ubuntu too. But there are very helpful documentations available to support us.

As for video, to play Windows encoded videos, you need to install w32codecs.

Just enable multiverse repositeries from Settings of Synaptic Package manager and you'll be fine.
