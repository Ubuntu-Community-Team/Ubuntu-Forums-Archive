### Post by fatsheep on 2006-07-20
I'm a newbie myself but you can use the add/remove feature under applications to install a lot of things as well as synaptic package manager under system>administration.

However, if you are trying to install a .run file then I BELIEVE (please correct me if I'm wrong) you go to the terminal.  Let's say the file is on your desktop so you'd type in:

> Cd ~/Desktop
Now type:
> chmod +x FILENAME (Including the .run
> ./FILENAME (including the .run extension again

Replace > Filename with the name of your file of course.
