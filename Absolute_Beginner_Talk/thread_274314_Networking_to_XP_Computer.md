Not a problem but a hint rather for newbies, like me.

Before getting frantic and downloading all sorts of stuff to network and print to a XP Computer or a printer on a Microsoft Network, just go straight to Places, Network servers and see if Ubuntu hasn't picked evberything up automatically. It did with me. straight after installation with absolutley no configuration from me Ubuntu had Picked up all the Xp Computers and printers. Musat have done this during installtion all on it's own. I can connect and print with ease, I can honestly say it was easier that from and Xp Comp to a XP Comp. 

Go check it out if this is what you want to do, I sincerely hope your experience is as good as mine was
