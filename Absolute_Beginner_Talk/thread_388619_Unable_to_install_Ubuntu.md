### Post by jem7v on 2007-03-19
For the record, I'm running an old AMD Duron 750 mhz, 384 mb RAM, 32 mb GeForce2 MX, which isn't much faster than yours... and I've had Ubuntu 6.06, Ubuntu 6.10 with Compiz, Xubuntu 7.04 (Herd 5), and DSL-N all running fine on my computer.  Your system stats probably won't be much of a problem.

However, I Have had trouble with the 6.06 partitioner on the LiveCD sometimes.  If you use the Alternate (text) install CD it will probably be fine.  As for trying Xubuntu instead of Ubuntu? You'll definitely notice a performance difference in some respects (such as window render/theme change speed), but I don't know if the LiveCD installer would work any better or not because I'm pretty sure they both use the same partitioner.
