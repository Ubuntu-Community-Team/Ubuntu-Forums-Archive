### Post by igknighted on 2007-03-05
There really isn't that much to do.  Spyware isn't an issue.  Anti-virus can be installed, but is mainly to avoid passing infected docs/emails to Windows users.  The firestarter firewall you installed is really just a control center for the iptables firewall, which is installed and running in Ubuntu by default.  Many recommend using the "noscript" firefox plugin for a little added security, but its all preference.
