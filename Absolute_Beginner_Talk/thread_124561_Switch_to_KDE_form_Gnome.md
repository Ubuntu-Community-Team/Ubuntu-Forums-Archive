### Post by sprok8 on 2006-02-01
It's actually quite easy to switch to KDE:

1. Go to System | Administration | Synaptic Package Manager
2. Enter your password when prompted
3. Select KDE from the list of packages. Synaptic will automatically pick out all related packages and dependencies
4. Click the Apply button, and wait!
5. When done, log out.
6. Select KDE from the Desktop menu, and log back in.

Enjoy!
