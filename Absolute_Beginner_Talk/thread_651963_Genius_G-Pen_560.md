### Post by amyst on 2008-03-13
I just discovered today that Genius G-Pen 560 has plug-in-and-play with Gusty 7.10. I bought it before installing Ubuntu and didn't hope it to work at all, but once I plugged it in ... Gusty recognizes it right away!!

I had no idea why Gusty works wonderfully with this tablet - without installing any driver. From the following site: 

[http://www.manilastandardtoday.com/?page=interactive01_june26_2006](http://www.manilastandardtoday.com/?page=interactive01_june26_2006)

it appears that even the older 560 (the newer ones are blue) probably work with 7.04 too.

This is what is in my System Log file:

input,hiddev96: USB HID v1.10 Mouse [KYE SYSTEM Corp. G-Pen] on usb-0000:00:ld.0-2
input: KYE SYSTEM Corp. G-Pen as /class/input/input9 
usb 3-2: configuration #1 chosen from 1 choice
usb 3-2: new full speed USB device using uhci_hcd and address 4

Maybe the driver is already there?

I hope this helps. Good luck!
