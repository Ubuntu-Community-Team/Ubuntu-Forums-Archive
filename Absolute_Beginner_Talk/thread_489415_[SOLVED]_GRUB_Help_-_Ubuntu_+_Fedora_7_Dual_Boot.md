### Post by bobbocanfly on 2007-07-01
I have an installation of Ubuntu 7.10 and a brand new installation of Fedora 7 on my hard drive (dev/sda). Everything installed fine, except there is one problem : i cant boot into Ubuntu. Fedora installed a new copy of Grub onto the machine which is only showing Fedora. I would like a unified Grub, with Fedora and Ubuntu.

This is my partition table:

dev/sda1 : Ubuntu 50gb
dev/sda2:  Fedora 11gb
dev/sda3:  Swap Partition

Ive done some hacking to my menu.lst but cant get it to work. Here is what i have at the moment


### Post by bodhi.zazen on 2007-07-01
You need to change these lines :

```
title		Ubuntu, kernel 2.6.20-16-generic
root		(hd0,0)
**kernel		/boot/vmlinuz-2.6.20-16-generic root=UUID=96ed5be7-c476-49d5-b0cf-e1119e775f25 ro **quiet splash
**initrd		/boot/initrd.img-2.6.20-16-generic**
**boot**
```

### Post by Bothered on 2007-07-01
I think the problem is likely to be that sda1 can't be accessed by grub as the path /dev/sda1/boot doesn't exist. Try copying the relevant files from /boot on the ubuntu partition to /boot on the fedora partition, and then using those in menu.lst.
