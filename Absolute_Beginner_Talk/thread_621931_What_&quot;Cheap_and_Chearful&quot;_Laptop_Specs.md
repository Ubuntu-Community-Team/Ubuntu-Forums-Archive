### Post by steveneddy on 2007-11-24
I don't think you need a dual core processor to watch DVD's, but a good P4 would be nice.

Around 1.5 Ghz is a good all around spec.

As much RAM as you can afford. 512 minimum. More is better. Run two of the same type of RAM stick. The PC likes it better that way.

You can get good performance out of an Intel video chip, but nVidia looks nice. ATI video is a royal pain, so I've heard.

The webcam is a hit/miss operation. Some onboard work well, some not at all.

Try a Live CD on the lappie you want to buy if possible and look at a lot of them if you can.

Also have a [look here]("http://www.linux-laptop.net/") for Linux compatible laptops.
