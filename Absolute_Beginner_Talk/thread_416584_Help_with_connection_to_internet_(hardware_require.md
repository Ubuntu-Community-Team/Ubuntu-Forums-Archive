### Post by altaaa on 2007-04-21
Yeah, get an ethernet ADSL modem. Make sure you buy the right type, there are different modems for analog lines (annex A) and ISDN lines (annex B).

There are a lot of different versions, so you can choose if you want a built-in switch, wireless, extra security, voip etc.
