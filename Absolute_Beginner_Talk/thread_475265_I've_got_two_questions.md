### Post by candtalan on 2007-06-16
> **roeboat72 said:**
> First, how in the heck to you find and install a virus and firewall program on ubuntu?


The anti virus is not needed in ubuntu. If you are handling windows files and sending them on into a windows environment, then you might want to install in your linux machine and antivirus for windows viruses, but you don't need one in ubuntu.

Ubuntu comes with a default install of having all unused ports closed or something like that. This is what a firewall will do first for you. If you wan to be flexible with ports and such things then a GUI such as firestarter  or similar, will manage the built-in facility of iptables, which is the effective firewall.

Note: the use of a modem router (ethernet cable or wireless to the PC) and not a modem with usb to the PC, will give you the basic firewall functions of a router - in effect a hardware firewall, whichever operating system you use.
