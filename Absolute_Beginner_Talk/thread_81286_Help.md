I can't verify these instructions at the moment. If you don't have anything at the bottom of the screen (panel with buttons to show desktop, show trashcan and so on), you could right-click on the upper panel and choose new panel / create panel (something like that), right click on the new panel (or the existing lower panel) and add new applets, the one which shows running programs (with windows), which is commonly known as taskbar(* (in Windows), is *window list* or something like that. :) Hopefully you at least got an idea howto fix it.

panel = "container" for applets
applet = small application like a application launcher ("icon"), a menu, traschcan, window list, workplace switcher

EDIT: *) Hmm... actually I'm not sure about the terminology here, it could be that panel = task bar, and not: window list = taskbar, but I guess that's irrelevant in this case. :)
