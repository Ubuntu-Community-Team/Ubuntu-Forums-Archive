### Post by Overbyte on 2007-09-22
Hi!

In Linux the file extension is not the basis if it can be executed or not...it depends on something called "file permissions". So you can have a text file with an .exe extension and it doesn't affect anything.

Conversely, most files in linux can execute but don't follow the "exe files must be executable" Windows does (like those package files you download and double-click...they're the "installers" in the linux world :))

As usual, the common stuff :p: don't download files with weird filenames or from places you don't trust. Enable the display of file extensions in Control Panel > Folder Options. And keep those definitions up to date!

Well you can just use Ubuntu to download your stuff if you're a bit used to it. Check the first link in my signature if you need a LimeWire equivalent in Linux.
