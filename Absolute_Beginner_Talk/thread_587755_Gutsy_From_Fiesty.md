### Post by shad0w_walker on 2007-10-23
My advice is this:

If you have a good reason to upgrade, do so but be prepared for unpleasantness. Back up important files (Should be done regardless of expecting issues :)) 

If you don't have a good reason to upgrade then leave it for at least 2 weeks, this gives the developers a decent amount of time to find/fix all the major bugs that have been screwing up systems.

If you just want to upgrade to have the latest but with out a specific reason, probably best just to stick with what you have if it works.

### Post by bigboy_pdb on 2007-10-23
In hindsight, I would have continued using Fiesty over Gutsy.

When I tried to upgrade, the upgrade froze and I ended up having to install Gutsy over Fiesty. The splash screen is using a frequency that my monitor doesn't recognize (and this produces an "out of range" message during system start up). My wireless networking card (which has an rt2500 chip) initially worked and then it later failed. Also, when I adjusted the vga settings in "/boot/grub/menu.lst", I wasn't able to use tty[1-6], and, furthermore, there's a delay when any program starts up.

None of these problems are restricted to myself alone. There are many bug reports on launchpad for problems such as these.

I haven't even followed the procedures that I've written out to restore my system to the state that I'd like it to be in, and I haven't tested my Wacom tablet or my Logitech controller. I'm worried that I'll run into more trouble when I do try to finish setting up my system.

In short, I don't think it's worth it if you don't think that nicer effects are a good trade off for a perfectly working system.
