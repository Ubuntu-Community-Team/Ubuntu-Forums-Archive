### Post by nhandler on 2007-02-20
I say go ahead. Are you planning on dual booting? If so, back up any files/programs you might need to an external media. If you aren't planning on dual booting, make sure you don't need anything stored on the hard drive (it will be DELETED). Pretty much, if you aren't dual booting, you have nothing to worry about. As long as your bios don't get totally messed up (very, very, very, very small chance), you can always reinstall ubuntu again until it installs correct. If you are dual booting, if the cd is faulty, it could wipe your drive. Be sure to back up everything.

All in all, try installing it. As long as you have your data backed up, you should be safe.
