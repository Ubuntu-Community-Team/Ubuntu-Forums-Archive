> **kioshis said:**

To be completely honest, if you want software compatibility, Linux isn't for you.  There's equivalents of MS software in Linux, and they're usually better than their Windows counterparts, but they are NOT the same programs.

> **kioshis said:**
> Most of the applications i run are basically microsoft apps like zune, word, and counter strike.
Toss Zune out the window.  It bites.  Seriously.
That's one part of Linux that will be absolutely jaw-dropping for you.  Try Amarok.  It's not installed by default, but just click "Add/Remove Applications" and it's on the list.
MS Word is replaced entirely by Open Office in Ubuntu, and they do exactly the same thing.
Counter-Strike is, I believe, cross-OS, and there's a Linux version available.  If not, you can run it under WINE or just get Wolfenstein, Nexuiz, or one of the other great FPS's freely available on Linux.

> **kioshis said:**
> I'd like to know if my performance would be better then on a windows machine.
Most certainly.  Linux isn't as RAM-obsessive as Windows, and it doesn't get the nasty registry issues like Windows does, so it will run much trimmer and be far more responsive.

> **kioshis said:**
> Do linux machines get virus'
No.  
There's a more complicated answer, but suffice to say that you will not run into malware or viruses in Linux.

> **kioshis said:**
> i've learned that ubuntu uses code instead of clicking short cuts.
Ubuntu uses shortcuts just fine.  Just drag and drop the application from your applications menu onto the desktop or taskbar.  Bada-bing.  Linux IS more focused on the command line, but you don't have to type code to run a program, it uses a system very similar to Window's start menu.

> **kioshis said:**
> I've also discovered vmware win4lin and the other application wine could i install microsoft office through it and run it from ubuntu or is there some long process that i have to go through.
Well, you COULD install MS Office through WINE or VMware, but you by no means have to.  Like I said earlier, Open Office does everything MS Office does.  If you DO choose to do that for some bizarre reason, you only have to set it up once, and after that it will just work.

> **kioshis said:**

### Post by kodoku on 2007-04-13
Welcome!

You have to understand that linux is COMPLETELY different than Windows.  If you're heavily attached to windows software, and you're planning on using linux like windows, then you're setting yourself up for major headaches.

All operating systems uses "code".  Windows just hides the "code" part, slaps a user interface on everything, and hides the commandline.  The desktop version of Ubuntu is basically the same thing.  You can still make "shortcuts" on your desktop for most common applications, except you'll learn that the commandline is amazing.

As for the the performance, really it depends on what you want it to perform.  Linux manages memory and harddrives (IMO) better than windows, so applications open and close quite fast (given your specs).  Windows games, however, will not perform better on Linux (in fact, most won't run at all).  Aside from games, most windows applications have their linux equivalents (which IMHO, the equivalents are better), and those that don't can often be executed with Wine.  Microsoft Office... can be run with wine (it takes some time to set up, but running it doesn't take much effort at all), but I think OpenOffice is a good alternative that runs natively.

And windows viruses are completely useless on linux.  Linux is nothing like windows, so software designed to run on windows can't do squat on linux.  Unless, of course, you do something silly like run a virus with wine, then.. that's your own fault (in short, you actually have to put in effort to make a virus infect your system lol)

Good luck

### Post by Tsen on 2007-04-13
Amarok, Rythmbox and Banshee are all up-to-date, good music players and media managers.
Amarok is more focused on being feature-filled, and takes the role of a music player but also as a music organizer, and can sync with your mp3 player.  It's somewhat like Windows Media Player, iTunes, or Zune Player (although much, much better than any of them)  Obviously I'm a tad bit biased, because frankly, I LOVE Amarok, but if you try it, I'm sure you'll be impressed.
Rythmbox is just a plain, basic music player.  It can organize and rate your music, transfer it to mp3 players and such, but isn't as feature filled as Amarok, instead going for a simple, easy to use interface.
Banshee, I admittedly don't know too much about, but it has a large fan following and works quite well.

Personally, I'd recommend installing Amarok, but if you boot off of a Live CD, you can give Rythmbox a try without installing anything.
