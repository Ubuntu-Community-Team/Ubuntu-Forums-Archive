### Post by toecutter on 2007-03-19
OK I'm trying to download some torrents. Like XP, Ubuntu has a 'What program do you want to open this with?' window that pops up. Unfortunately I don't want to use BitTorrent or BitTornado to download my torrents, so I installed Ktorrent, a choice which isn't available on the dropdown menu. 

I don't fully understand the file system yet, so I don't know how to find the 'program' (to use the Windows term) for Ktorrent, or what directories to look in. 

How do I choose Ktorrent or Freeloader (another client I'm trying out) instead of the default torrent program?

---

### Post by 23meg on 2007-03-19
Based on your profile info, I'm assuming you're using GNOME, even though KTorrent (apparently) is a KDE app. Right click the file, click "Properties", go to the "Open With" tab, click "Add". If KTorrent isn't in the list that appears, enter "ktorrent" in the "Use a custom command" box.
