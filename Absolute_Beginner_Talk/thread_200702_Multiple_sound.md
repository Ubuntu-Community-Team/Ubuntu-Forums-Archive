### Post by pdobrev on 2006-06-20
If you want to use applications that only support OSS and the solution with aoss won't work (like Skype), you can try using jack server and oss2jack. It's a bit tricky till you get it running because it requires some modules to be compiled, but once you've set it up, you can even use some cool features like system-wide equilazer and stuff. 

For more information: [http://fort.xdas.com/~kor/oss2jack/](http://fort.xdas.com/~kor/oss2jack/)
