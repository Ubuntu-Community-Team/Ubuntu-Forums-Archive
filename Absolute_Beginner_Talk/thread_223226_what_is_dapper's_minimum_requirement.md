### Post by MonkeyNet on 2006-07-26
Desktop Minimum Requirements

256mb RAM, 3 Gb HDD, Processor I would say at least a 500MHz to run properly but will more than likely work on lower spec CPU

Server Minimum Requirements

64mb RAM, 500mb HDD

These specs are from [here]("http://www.ubuntu.com/download/releasenotes/606").
