### Post by postino2 on 2008-04-08
[B][SIZE="4"]How to increase mouse pointer size in Gutsy Gibbon/ How to increase cursor size in Gutsy Gibbon:

From what I see, the ability to select the mouse pointer and pointer size was moved when Feisty Fawn (2007-04) was upgraded to Gusty Gibbon (2007-10).  That functionality was moved from:

SYSTEM / PREFERENCES / MOUSE 

to:

SYSTEM / PREFERENCES / APPEARANCE / CUSTOMIZE / POINTER / SIZE
