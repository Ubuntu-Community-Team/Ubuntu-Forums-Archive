### Post by poofyhairguy on 2006-02-15
[QUOTE=SMF]What is really going on?:eek:[/QUOTE]

You are a kinda early adopter (on the desktop) of an operating system that was designed to replace other operating systems (Unixs) on the most important and expensive computers in the world (called Big Iron). As a result its built with a level of security from the ground up that is far beyond what is normally expected of a desktop operating system (XP). 

So the only viruses for Linux exist in a lab. 

With Ubuntu you give up perfect compatibility with the Windows world- most Windows applications won't work (but for almost all there is a free Linuz equivilent) and some Windows hardware won't work (like some webcams). In return you get thousands of pieces of free software that are easy to install, access to community resources, and an operating system built to run the world safely.

Welcome to the club.
