> **mitchellthegreat said:**

As oldtimetech said, you can have both if you want. Get windows reinstalled first, then load ubuntu in a dual boot setup. That way you can select which OS you want to run at any given time. As far as keeping ubuntu (only), that depends on if you need to use any windows-only applications. If the answer is no, then give ubuntu a shot. I think you'll find it's really nice (and free!). If you need some windows apps still, like some games for instance, then dual boot. You can use windows for the games and windows-only apps and use ubuntu for everything else. For the dual boot, get windows installed first and then install ubuntu second, not the other way around.

### Post by Imamoomoocow on 2007-10-24
I would advise saying to hell with Geek Squad.  They will repartition your disk and install Vista only. I doubt they will even try to touch Ubuntu for you because it's not on their list of things they sell. I would say repartition you disk in this way:
from left to right put One partition for Vista (NTFS primary), then one for ubuntu (etx3 PRIMARY), one fat 32 partition for your media storage (music and such), and finally around 3gb of linux swapspace
I personally use this setup and it works fine for me. The FAT 32 partition helps save you from wasting disk space by putting your music on two partitions. This way you can mount the fat 32 on linux and vista.
