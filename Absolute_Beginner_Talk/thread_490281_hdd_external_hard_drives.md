### Post by Inxsible on 2007-07-02
I think you can find the option under System --> Preferences --> Removable Drives & Media.
 
Change the appropriate command to "nautilus /path/to/the/mount/point"
 
But i think this will work only for one external drive. If you connect some other which is mounted at a different location, it wont :(
