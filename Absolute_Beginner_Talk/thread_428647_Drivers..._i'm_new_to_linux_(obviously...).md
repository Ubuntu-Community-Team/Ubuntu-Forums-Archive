### Post by starcraft.man on 2007-04-30
I still think the best and easiest way to install graphics drivers is the Envy script [here.]("http://www.albertomilone.com/nvidia_scripts1.html")

Just have a quick read up top and install the debian version 0.92.0 should work. Then after you installed it select Envy from the Applications > System Tools > Envy. Click automatically install nvidia drivers and off you go, answer yes to any questions, and before ya know it their in :).
