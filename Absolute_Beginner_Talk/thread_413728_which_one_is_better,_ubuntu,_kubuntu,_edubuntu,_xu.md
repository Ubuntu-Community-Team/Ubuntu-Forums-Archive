### Post by aysiu on 2007-04-19
If you don't know, start with Ubuntu. You can easily add the others later or switch.

Differences between Ubuntu and Kubuntu (mainly a matter of preference):
[http://www.psychocats.net/ubuntu/kdegnome](http://www.psychocats.net/ubuntu/kdegnome)

Xubuntu is a slightly lighter desktop environment. It's great for older systems (128-256 MB of RAM), but some people like it even for more modern computers.

Edubuntu includes software targeted at children (for educational purposes).

### Post by Drunner611 on 2007-04-19
Well, you have to ask yourself what do you need.

Ubuntu - Basic OS, uses GNOME interface
Kubuntu - Basic OS, uses KDE interface
Edubuntu - Basic OS, includes extra features for teaching in classrooms/homes
Xubuntu - Streamlined OS, uses XFCE, but with fewer apps/options, for slower computers.  Still very good.
