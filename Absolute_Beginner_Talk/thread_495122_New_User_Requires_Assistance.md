### Post by Vague on 2007-07-07
Open Synaptic Package Manager.  Go to Settings>Repositories.  Check the box for "Community-maintained Open Source software (Universe)."  You can enable Restricted and Multiverse if you want, too.  Close that and hit reload.  Search for firestarter.
