### Post by actuary286 on 2007-03-01
Unfortunately, printer names can only be set when the printer is created in CUPS (the printing system). If you want to rename a printer you will have to delete it, out of the Printing dialogue in Gnome or directly from CUPS through the web interface, and then create a new entry with the name that you want to give it.
