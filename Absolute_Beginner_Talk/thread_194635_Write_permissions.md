Hello,
I'm guessing you're trying to install flash player. Open a terminal(Applications->Accessories->Terminal). Now type cd (type the directory the file is in here). Then type sudo cp libflashplayer.so wherever you're putting the file. There should also be an install script in the directory. You can run it with sudo ./theinstallscript.
