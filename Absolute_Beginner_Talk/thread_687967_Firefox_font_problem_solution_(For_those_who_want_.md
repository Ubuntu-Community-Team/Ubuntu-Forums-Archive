### Post by xslimmiejimmiex on 2008-02-04
I first was in XP and checked the Firefox preferences there. So this is based on XP defaults-

First install the msttcorefonts via Terminal, or System:Admin:Synaptic

In ubuntu, go to: 
- Edit, Preferences
- Under fonts & colors: 
  Default - Times New Roman        Size: 16

click Advanced:
Fonts for: Western                        Size: 16
Proportional: Serif
Serif: Times New Roman
Sans-serif: Arial
Monospace: Courier (or similar)    Size: 13

Default character encoding: Western (ISO-8859-1)


Check the "Allow pages to choose".


Remember to first download/install :Msttcorefonts in (System: Admin: Synaptic) or through a terminal. Hopefully this will help!:)
