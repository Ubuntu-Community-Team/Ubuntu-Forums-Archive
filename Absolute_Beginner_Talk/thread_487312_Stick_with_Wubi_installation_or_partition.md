dbuss63,

If your intent is to commit part of your HDD to Ubuntu and create a dual boot system, I would uninstall Wubi first, the rationale being that it will free up more HDD space before you install GRUB and Ubuntu on its own partitions. Also clean the litter from Windows using something like CCleaner which can be downloaded at [http://www.ccleaner.com/](http://www.ccleaner.com/), and then defragment the HDD. This will be a big help in getting your Linux in a contiguous area of the HDD.

Wubi itself is unknown to me, but this seems like a good common sense approach. CCleaner can also repair your Windows registry. I've had no problems with it in Windows XP Pro. You get several options before making any registry changes. First, the program shows what changes it deems worthwhile to make, but gives you the option of reviewing them first, and un-marking changes you don't want to be made. Before making the changes, the program then asks if you want to save all the changes so they can be restored should some application fail to work properly that you will know registry changes were made for. It's a very good program. It is considered freeware, even though the developer asks for free-will donations on the website. ;)

DJ
