### Post by jd65pl on 2007-02-07
Yes ubuntu is available as a live cd. Live cd refers to the ability to run something from boot without using a method of making changes persistant. i.e. you can run something but the changes made within the application will not be saved.

edit:
It is done my using your ram as a virtual hard drive, see ram disk

### Post by xpod on 2007-02-07
The live cd is what you want sjmcyyc.

All you have to do is reboot with the cd in your drive and it will load up to a start menu from where you can check the cd`s for defects or load up the live session.

You also have a "safe graphics mode" and a "memtest" option which can be handy at times.
You might need to change the boot order in your bios but other than that give it a go and see for yourself

### Post by mizar001 on 2007-02-07
Eh eh . Simply put the cd in the reader and enable it in the bios setup as first device to read from.

An OS can start from any device source. The important is to have RAM memory to load modules and kernel. Even an installed OS works like this. Even the xp installer is  a minimum OS.

Live version can not deliver you full capabilities to download and install new software or recompile kernel, or something else, but let you work with all applications you see and you can mount your disks too, and that means that you can control your machine entirely from an external loaded OS.
