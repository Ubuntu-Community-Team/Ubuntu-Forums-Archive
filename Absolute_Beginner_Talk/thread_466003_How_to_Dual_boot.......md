### Post by dstew on 2007-06-06
You need to use the Vista disk management tools to shrink the Vista partition to make room for Ubuntu. Once you shrink the Vista partition, it is safe to use the Live CD to install Ubuntu into the new unallocated space. Grub should be able to boot Vista OK, so just let it go into the MBR of the first disk as usual.

rajesh20057: I'm a total newbie to Ubuntu and am in the process of attempting to do what you're doing (create a dual-boot on a Vista machine). I'm still trying to get it working, but based on what I've learned from the super-helpful people on this forum so far I can tell you the following for sure:

Apparently, Vista doesn't like it if you resize partitions with third party tools. So, if you're trying to install Ubuntu on the same drive that Vista is on, you will need to attempt to use Vista's build-in partition resizer first (reach it by typing in diskmgmt.msc in "run"). Search this forum for threads on how to shrink your volume....if you don't find any, I'll help you find it.
