### Post by Pragmatist on 2006-03-18
There is a config tool for windowmaker that is on the desktop by default.  This link explains some more and seems to indicate you can control the position:
[http://main.linuxfocus.org/~georges.t/prefs.html](http://main.linuxfocus.org/~georges.t/prefs.html)

Also, if your mouse can touch any part of a window, if you hold down the ALT key you can drag the window (i.e. you don't need to just grab on the title bar).  However, I can understand why you wouldn't want to have to do this everytime you open a window :)
