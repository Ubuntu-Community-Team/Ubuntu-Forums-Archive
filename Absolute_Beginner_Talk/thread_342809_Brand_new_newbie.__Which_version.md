### Post by aysiu on 2007-01-20
I think you should give Ubuntu a try because you're interested in Ubuntu, not because you're hoping it's a free version of XP.

It is very different from XP and does not--because of its low desktop market share--have the same support from third-party vendors that Windows has. It will not run most Windows applications (since it is not Windows).

For more information, read [Is Ubuntu for You?](http://ubuntuforums.org/showthread.php?t=63315)

If you do decide to give Ubuntu a try, I would recommend you start with 32-bit, though, instead of 64-bit. More applications are available for it.

### Post by Izobalax on 2007-01-20
Ubuntu or Freespire ... Hmmm ... well I've tried Freespire, and I like it. A ***lot***. It's pretty and practically ***everything*** works entirely out of the box, which is a ***massive*** bonus to new Linux users. However, at the mo, Freespire is still quite buggy. For example, installing fonts causes you to be permanently locked out of your system. 

Freespire 2 is currently in production. It will be interesting to see how good it is and how much it's improved. 

However, Ubuntu has several major advantages:

1. It has a ***MASSIVE*** community. Practically ***any*** question you ask will be answered and a solution ***will*** present itself eventually.

2. Ubuntu has been going longer, therefore most rudimentary bugs like hardware detection and so forth have been ironed out. 

3. Ubuntu is based on Debian, one of the oldest Linux distros, and therefore a lot of programs are very compatible with Ubuntu.

I hope this helps you to make your mind up, one way or t'other.
