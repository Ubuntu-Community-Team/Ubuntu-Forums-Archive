### Post by phil66 on 2008-03-24
Hi Tews

I have Cairo-Dock installed on Ubuntu 7.10 Gutsy Gibbons.

In order to have apps minimize the steps I used to set the configure file were:

Right click on the Cairo-Dock toolbar
Click on configure
Click on taskbar
Under interaction section I checked lines to minimize programs
Cick on apply
When I want to minimize a program then I click on the - in the upper right hand corner of the program
The minimized program show up a lighter color in the right hand side of the toolbar

Hope this helps
