### Post by Kheldin on 2007-07-18
Whatever port you set in Azureus you must make sure the same port is set in either your firewall or router (or both). More than likely you don't have any firewall software running so try checking your router settings first (this is assuming you have a router connected to a broadband device).

Log into your router, make sure both TCP and UDP port 17247 are being forwarded to the IP address your network card is assigned (or set to) in Ubuntu. Hope that helps.
