### Post by annda on 2008-03-30
i'd recommend trying fluxbuntu as live CD before you install it. from my experience, it is nearly impossible to 'just uninstall' GNOME, since it will want to take a lot of useful packages with it - i admit, i tried it maybe a year ago and had to note down the names of packages i had to reinstall immediately if i wanted to have a working system with a GUI. ut as i said, that was a long time ago.

### Post by eriqjaffe on 2008-03-30
If you're considering ditching GNOME in favor of Fluxbox, you'd be far best served doing a ground-up installation and just adding the things you need.  It can be a bit time consuming, but in the end you get a system that has far less bloat.
