### Post by mahiyar on 2007-01-22
Your questions have sent me to a journey of linux burning (not literally). It seems that most native burners of linux do not recognize multisession burning. Nautilus will read the CDRW but if you try to add something from Linux it threatens to wipe out the disk. Gnome baker does have multisession. But you have to ask to add the sessions from >project>import sessions. I found even that to be unreliable. Then from the forums I chanced upon Nero Linux. In this particular programme also you have to add the sessions but the result was perfect readability between Linux and Windows (I have tried ot out first hand). The only gripe is that Nero is not a free burning programme. It seems that K3B does all that Nero does.
