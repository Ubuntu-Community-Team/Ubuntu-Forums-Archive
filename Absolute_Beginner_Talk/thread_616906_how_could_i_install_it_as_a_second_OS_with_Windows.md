### Post by aysiu on 2007-11-18
Do you have more than 512 MB of RAM? If so, your safest bet would be to install Ubuntu as a virtual OS inside of Windows:
[http://www.psychocats.net/ubuntu/virtualbox](http://www.psychocats.net/ubuntu/virtualbox)

And, no, Ubuntu doesn't take Windows programs. There's a hack called Wine that allows you to run some Windows programs in Ubuntu, but it doesn't work for every Windows program, and even when it does, it does so with weird glitches sometimes. Your best bet is to find native Linux programs instead of Windows programs.
