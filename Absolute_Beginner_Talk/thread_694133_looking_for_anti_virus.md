### Post by Rocket2DMn on 2008-02-11
There aren't currently any viruses in the wild for linux, and if one were ever released a patch would come so fast it would never really spread.  The only reason to run an anti-virus in linux is to prevent passing viruses on to windows users.  However, most of us believe it is up to windows users to protect their own systems - don't waste your resources on it.
