### Post by twin_57103 on 2008-04-06
If you decide to stay with Ubuntu, it is well worth the cost to purchase a Linux-compatible modem.  I have tried in the past to install a winmodem under Ubuntu and had to give up (tried 3 or 4 modems and worked on it for over a week).  Sorry to be blunt, but I've been there and practically tore out my hair trying to get it to work.

I have found the Ubuntu forums and [http://www.ubuntu.com/]("http://www.ubuntu.com/") more useful than linmodems support sites.  Do you know exactly the model of the modem you're trying to set up?  You can search for that on the forums or post it here.

I agree that Xubuntu is a better choice for your system, but the one time I tried to set up a Winmodem on Linux was on an Ubuntu machine (P3 500 MHz and it runs just fine).  When you do install, you might consider using the alternate CD (text-based installer); I think they suggest 256 Mb ram if you're going to use the live CD for installation.  Fluxbuntu is even lighter, but I think there are a lot more people using Xubuntu, so you'd get better support.

I hope that helps - sorry I can't help more!
