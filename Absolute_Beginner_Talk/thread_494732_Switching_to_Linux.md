### Post by aquavitae on 2007-07-07
> 
I need the linux distro to have an embeded application installer as i'm not quite good with commands !!
Most distributions have a GUI installer - Ubuntu has one of the best I've seen.  Any linux you use will require a certain amount of command line usage.  In theory you don't need it in ubuntu, but invariably when something goes wrong, you'll get instructions on what to do from the command line.  This is fine, because the instructions are usually very detailed, and don't require any knowledge.  In any case, most of the simple commands you'll pick up really quickly.
> 2nd. what is the main differences between ubuntu and kubuntu ??
The only difference is the default desktop manager.  Ubuntu uses gnome and kubuntu uses kde.  There's very little practical difference between them, and anyway its easy to switch if you decide you prefer something else - just install the desktop manager you want from the installer.
> 3rd. I really really can't work out in switching the OS if the new one was unable to run Need For Speed games ( new ones !! )
so is it possible and not hard to do that with linux ??
You're talking about running a windows game in linux?  Its sometimes possible using a program called wine, but not always.  I don't know how easy it is for your specific game, but search the forums and see if anything comes up.  Also check the wine site: [http://www.winehq.org]("http://www.winehq.org") for any known issues with this game.
> 4th. i want it to be able to play any kind of multimidia files 
Ubuntu doesn't come with a lot of codecs preinstalled, but most players can play just about anything with the codecs installed - far more, in my experience, than windows media player.  No problems here!
> 5th. i want to be able to connect my k750 mobile phone to it and synchronize
Not quite sure about this cos I've never done it.  I have no problems plugging in my k800i and using it as a flash drive (or modem!) so you should be able to synchronise it easily enough.
> 6th. and most important I want it to work completely ok with my hardware
That depends entirely on your hardware. Linux identifies more hardware correctly on installation than windows, but some hardware does have issues.  I'd say just search the forums for your hardware and see if anyone else has had problems.

### Post by forrestcupp on 2007-07-07
> **raoufn1984 said:**
> 
I know that many of the questions are not directly about the OS but I don't want to lose any ability I had with XP ( although I really really hate it as an OS )


If you don't want to lose any ability you had with XP, then you need to keep XP.  Ubuntu is not an XP clone, it is a different OS, so you have to assume things will be different.

Wine may or may not run Need for Speed, but Wine will *not* run any and all XP software.  When people buy a Mac, they understand that they have to use Mac software with it.  It's the same with Linux for the most part.

Having said all of that, I think you will be satisfied with Ubuntu with the possible exception of running Need for Speed (as above, check the wine database).

As for the embedded software installation, we have Synaptic which is very easy to use.  It lists all of the software available in the repositories.  You can search for what you want either by looking through the categories or by doing a keyword search.  Then it easily installs it for you.  You can find most software you need here, but sometimes unsupported 3rd party software is harder to install.  But there's plenty of help here for that.

As for the Ubuntu/Kubuntu thing.  You will have to study out the difference between Gnome and KDE.  Personally, I use and like Kubuntu.  But for someone new, I would recommend using Ubuntu because it has some things built into it that make it easier for new people, and some of these things don't work well in Kubuntu.

### Post by aquavitae on 2007-07-08
The folder ~/.mozilla should be created when you first start firefox.  I'm not sure what you mean by "backed up firefox profile", but If at any stage you backed it up/copied it as a different user, or as root, it will change the permissions.  Its easy enough to fix: open a terminal and type:
sudo chown -R <username>:users <name of file/folder you wish to get permissions for>

> also how can i use resolution of 1280X1024 ??
Make sure your graphics card and monitor are set correctly in the system settings, and (assuming yur monitor suppors it) you get have that resolution as an option.  If not, then post the make and model of your graphics card and monitor, and the contents of the file /etc/X11/xorg.conf.
