### Post by ramjet_1953 on 2007-07-15
At this stage, I would recommend sticking with the 32 bit version. The 64 bit version still has some issues and sometimes tales quite a bit of stroking to get it working properly.

Some issues that you may encounter with a laptop install are video card and wireless issues. There are plenty of posts on these forums to get things working properly and if you do it by the numbers, all should go well.
