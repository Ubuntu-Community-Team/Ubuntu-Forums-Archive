At some point shortly after you begin to install, Ubuntu enables several consoles.  There are three or four.  You get between them by pressing alt-f1 alt-f2 alt-f3 alt-f4.  I believe alt-f3 or alt-f4 lists what is going on in more detail.

You can install the 32 bit Ubuntu on a 64 bit machine from what I have read.

You could try Ubuntu 6.06.  

There are two version of Ubuntu for each environment: live and an alt.  

Which version are you trying?  

I'd try Ubuntu 6.06 32 bit alt.
