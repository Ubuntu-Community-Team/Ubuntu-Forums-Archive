### Post by luisromangz on 2008-02-09
You don't say what version driver do you use, so I don't know, but if you are using the ones in the Gutsy repositories, they are many months old, and ATI have been realeasing a new driver each month, based on a new codebase that improves performance. With the new drivers, you don't need to choice between Compiz or games, as you can use Compiz without XGl.

And testing the drivers using Cedega/Wine is a very bad approach, because although Cedega/Wine itself is a miracle (running Windows software in Linux!) you can't count on it as a reliable substitute for all Windows programs.
