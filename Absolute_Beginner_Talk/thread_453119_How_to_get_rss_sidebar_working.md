### Post by vinmar on 2007-05-24
This is a KDE application, and it depends on also having superkaramba installed. So if you're using gnome, which is the ubuntu default, it won't be that easy to use this RSS reader.

I quite like the RSS reader [http://liferea.sourceforge.net/](http://liferea.sourceforge.net/) It stays out of the way, although it doesn't run from a sidebar.

### Post by toasterofirony on 2007-05-24
Getting a really good RSS feed reader was, for me, one of the big challenges after I stopped using Flock (long story...)

I would suggest [Google reader]("http://www.google.com/reader/view/"), if you haven't thought about it. And you don't have to worry about fiddling around with dependancies, either ;)
