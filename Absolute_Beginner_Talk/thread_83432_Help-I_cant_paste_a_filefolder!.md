### Post by Ampersand on 2005-10-28
when you run sudo nautilus, it'll probably start in /root (which is root's home folder). If you press the Up button (or backspace), you'll get to the / directory, from which you can get to your home directory by going to the home folder, followed by the folder with the same name as your username (does that make any sense?).
