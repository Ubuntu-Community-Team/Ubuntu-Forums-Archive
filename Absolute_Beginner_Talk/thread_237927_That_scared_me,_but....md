### Post by msak007 on 2006-08-16
If you're talking about choosing "Run or Install Ubuntu" from the menu that you get when you first boot the CD, you're simply booting into the live environment. Dapper is a single CD for both the live environment and the installer. Once the OS completely loads, there will be a shortcut on the Desktop to install it permanently. If you continued to boot the first time, it would not have erased Windows. Give it another shot, I'm sure you'll end up installing Ubuntu and forgetting about Windows soon enough :)

### Post by msak007 on 2006-08-16
I've never seen the sleeve that the free CD ships in, but yes doing a default installation (meaning that you let the installer automatically partition your drives) WILL format your drive and install Ubuntu. Make sure you when you get to that part that you manually configure the partition table to choose where you want to install the OS (another partition, drive, etc.). Yes the initial load time for the live CD is a little long, but hey what do you expect when you're running a complete OS from a CD ? :)
