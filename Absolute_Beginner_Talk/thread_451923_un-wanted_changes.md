### Post by Martin on 2007-05-22
If you are using regular Ubuntu with Gnome
To change the colour. Right-click on the bar you wish to change, choose properties and then background. You'll see the settings there.

To move a panel just left-click on an exisitng one and while keeping themouse button down drag the panel to where you want it. It will snap to the top, bottom or sides of the screen. If you need a new panel right-click an existing panel and choose the option New Panel. to add stuff ot a new panel right-click on it and choose 'Add to panel' and you will be presented with a whole lot of stuff than can be embedded into a panel

### Post by Inxsible on 2007-05-23
Right Click on the panel and go to Properties and then the Background tab

On it you have 3 radio buttons
1) None (use system theme)
2) Solid Color
3) Background image


Yours is probably set to Solid Color and the Transparent ----Opaque bar is on Transparent
You can either scroll it all the way to Opaque or if you want the default, you can simply select None.


All this assuming you are running Gnome. I am not sure if its the same in KDE
