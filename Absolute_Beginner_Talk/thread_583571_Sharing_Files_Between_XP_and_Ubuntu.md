### Post by Niniel on 2007-10-20
There's nothing to set up if you use 7.10.
I have it, and I can download files onto my NTFS partition, copy/move them into and out of it, read files there of course... So if you have data you want to be able to access from both OS, leave them on the Windows side. Ubuntu can work with them, but XP can't access Linux drives, unless there's a program that adds that functionality.
