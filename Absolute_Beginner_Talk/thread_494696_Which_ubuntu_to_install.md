### Post by JasonL on 2007-07-07
Hello, I would recommend at this time to use the 32-bit version as this is generally more supported and you are more likely to get the system working how you want it, some software will not yet run on 64-bit and you may run into a few problems a long the way.

Because of the issues still being sorted out with 64-bit it is generally better to go with 32-bit as it will work on 64-bit processors and save you a lot of time, especially if you are new to all of this.

Good luck and I hope everything goes well, please do not hesitate to ask any more questions.
