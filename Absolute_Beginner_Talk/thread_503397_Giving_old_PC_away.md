### Post by reckless2k2 on 2007-07-17
you can pop it in and see if it boots from the CD. a lot of computers are set to boot from cd if it's in the tray prior to startup. if not, you can usually hold downthe esc or del key to into the bios and change the boot order. you can install ubuntu and whipe your hard drive or just run it from the cd if you like. running from cd will be slower.
