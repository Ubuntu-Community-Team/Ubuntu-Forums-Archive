### Post by Shay Stephens on 2006-07-26
It's a try and see type thing.  I have apps that are not on the list but run fine.  I also have apps not on the list that almost work.  Then there are the apps that are not on the list and won't even install.

So try it out and see what happens.  After each update, try again to see if it runs.  My big need right now is a program called DVDComposer.  It almost runs in Wine.  It's counterpart, Memories On TV runs just peachy in Wine.  Each time Wine updates, I try out DVDComposer right away to see if there is any change.
