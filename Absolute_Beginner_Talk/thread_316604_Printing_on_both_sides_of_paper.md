### Post by Tzumli_D on 2006-12-11
You will find in the printer settings dialog box (File>printer settings) under properties the option to have duplexing off or on. Turn it on, and then you will have to choose between long edge or short edge.
If this is not there, you may need more help than I can give. I'm trying to set up my Brother duplexing laser printer. I've set that option, but I still can't make anything actually print.
