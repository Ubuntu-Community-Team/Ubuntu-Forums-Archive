### Post by ajmorris on 2007-12-14
Hi there,
There are no key sequences for this compiz fusion plugin. If you start the application CCSM (/usr/bin/ccsm) you can click enable on Trailfocus (under the 'Effects' heading) then change the options for it, from then on, it does the fading effects automatically.

### Post by Krydahl on 2007-12-15
Do you have the advanced desktop effects settings option in your system -> preferences menu?

If so, you want to make sure you have the ring switcher enabled in there. You can then use Super + tab to switch between windows.

If not, you need to install CCSM. Go into synaptic and search for compizconfig-settings-manager.
