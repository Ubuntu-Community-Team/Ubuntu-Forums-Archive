### Post by darkmatter on 2005-10-04
[QUOTE=mit]Hi,
Is it worth installing it? Every time i boot the little red icon comes up if there is anything new...:???:[/QUOTE]

If you are referring to the 'little red icon' that appears in the systray (notification area), there is no need to install Nessus for that.

That icon is a notification from the Update Manager to inform you that there are security updates, bug fixes, etc. for your system. Just click the icon and it will open the Update Manager so that you may review and install the updates.

This is how Ubuntu keeps your system up-to-date and secure.
