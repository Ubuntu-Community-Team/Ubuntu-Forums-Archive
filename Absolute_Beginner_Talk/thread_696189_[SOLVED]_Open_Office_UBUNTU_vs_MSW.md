### Post by macogw on 2008-02-13
Are you using the same fonts?  Do you have msttcorefonts installed?  If you don't it's probably using some other font that may or may not (probably not) be the same size as the Microsoft fonts which would mess up your line height.  You can also just go to (trying to remember, don't have OOo installed) Format -> Paragraph and set the line spacing to something slightly smaller than 2, like 1.8 and see if that results in 25 lines per page.
