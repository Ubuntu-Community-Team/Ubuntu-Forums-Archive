### Post by 5-HT on 2006-12-07
As in open that folder in a graphical file manager? Sure!
```
<executable to filemanager of choice> <path to directory>
```If you're running GNOME with Nautilus for example*, you CD to ~/foo and could type the following into the terminal: ```
nautilus .
``` *If you're using a different file manager just substitute the executable as appropriate (e.g., 'konqueror .' if using konqueror).
