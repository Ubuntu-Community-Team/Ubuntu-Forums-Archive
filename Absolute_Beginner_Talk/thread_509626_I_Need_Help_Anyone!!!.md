### Post by .nedberg on 2007-07-25
Go to ubuntu.com and download the newest version (7.04 Feisty Fawn)
Save that file somewhere on your harddisk
Burn it to a CD (use nero or whatever program you use)
Put the CD in your CD-tray and reboot your computer, make sure it boots from the CD (you might have to change some settings in the BIOS)
Select "Start or intall ubuntu" in the first menu
Select install from the desktop and answer some questions.

Oh, and make sure you have a backup of everything on your computer! The links in the posts abow is a good read!
