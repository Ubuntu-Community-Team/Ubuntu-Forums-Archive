### Post by RichPicker on 2007-05-14
I have used  GAIM with good results. It is in System>Administration>Synaptic Package Manager. Then do a search on GAIM, mark it for installation, and Apply. After installing it will be in Applications, Internet, Gaim.

Gaim is capable of using AIM/ICQ, Yahoo!, MSN, IRC, Jabber, Napster, Zephyr, Gadu-Gadu, Bonjour, Groupwise, Sametime, and SIMPLE att at once.
