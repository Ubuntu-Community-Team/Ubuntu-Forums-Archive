### Post by 5-HT on 2008-01-29
It's called a skydome. As long as both the hight and width of the image are a power of two you shouldn't have any problems (e.g. 1024X1024) with either a .jp{e}g, .svg or .png. Depending on if you're using compiz-fusion or compiz you can configure the skydome in whichever configuration utility you use (e.g., for compiz-fusion you can use the fusion icon -> Settings manager).
