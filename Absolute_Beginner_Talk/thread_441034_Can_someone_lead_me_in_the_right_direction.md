### Post by cmost on 2007-05-12
The term "slipstreaming" was coined in the Windows world and really only applies to merging Windows service packs into the Windows installation CD and then cutting a new installation CD.  What you want to do is create a custom Ubuntu Live-CD that you can install on new systems.  Here's a good place to start:
