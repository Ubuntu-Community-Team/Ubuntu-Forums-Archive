### Post by Leonivek on 2007-10-28
> **jflaker said:**
> Also, make sure you don't burn the cd at a faster rate than it can be read.  I usually burn the cd @ 4X otherwise older systems have trouble reading it.

The speed with which you burn your CD/DVD does not affect how any CD/DVD drive can read it, no matter how slow it can read.  You can burn a CD at x52 and a x4 read drive can still read it.  

HOWEVER, burning a CD too quickly *can* cause errors on the CD which would NOT be readable by any drive.
