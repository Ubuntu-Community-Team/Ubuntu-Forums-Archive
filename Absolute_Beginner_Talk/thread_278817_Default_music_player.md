### Post by thephotoman on 2006-10-17
This process is a bit long, but bear with me.

Open up Nautilus and right click on an MP3 file.  Select "Properties" from the drop down menu.  Go to the Open With tab and select VLC from the list.  Click OK.

Keep doing this with each file type that you want to associate with VLC.
