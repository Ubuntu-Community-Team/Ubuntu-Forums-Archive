### Post by adrianelliott on 2007-08-05
You need to use Kerebos and Samba to join your Linux box to the 2003 domain.  Have a look at this outline presented here. [http://www.enterprisenetworkingplanet.com/netos/article.php/3487081](http://www.enterprisenetworkingplanet.com/netos/article.php/3487081)

I'm sure there will be an ubuntu specific guide around somewhere.

As far as I know, the only benefits you would get by joining your Linux box to the windows domain will be central authentication and file sharing.
