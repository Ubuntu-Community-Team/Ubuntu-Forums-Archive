### Post by Golyadkin on 2007-06-23
If the resolution of the virtual machine is bigger than the resolution of the host (Ubuntu) in this case, it will not fit. Think of it as a putting a 20x30 cm square on top of a 15x20 cm square. It will always overlap. You can try to run the virtual machine fullscreen, which will solve the problem if the resolutions are the same size (both 800x600). You can go fullscreen via an option in the menubar of the virtualmachine.
