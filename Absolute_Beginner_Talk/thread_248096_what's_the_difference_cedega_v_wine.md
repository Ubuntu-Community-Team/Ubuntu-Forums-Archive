### Post by Anonii on 2006-08-31
Well, Wine is free, when Cedega isnt. Thats the main difference.
Cedega is supposed to be easier, with a nice gui, and it is also supposed to be running more games without any or a little configuration. Wine can run the same but most of them require some configuration. Also a big plus of Wine, is that it runs applications too. So if you want (that much) Winamp you should prefer Wine. Cedega is a new name for the old WineX, btw.

Finally, dont install Winamp in Linux. Its a lame project, for elitist retards. It gives nothing more than WMP or any other audio player and I dont even know why its so popular. If you are using KDE, I'd recommend you to give a try to Amarok. If you are GNOME, to Banshee. And if you want something similar to Winamp, be sure to give a look to XMMs. Its open-source, Linux based, and this means that it will run MUCH better to a Linux box, than Winamp :)

Good Luck.
