### Post by tangibleorange on 2008-02-08
When watching a DVD, I think you need to change the subtitles track. In VLC:

File --> Open Disc

And change the subtitles track to '1' and save. If that doesn't work, experiment with the different subtitle tracks.

### Post by nemilar on 2008-02-08
Like the previous posts say, see if the subtitles are in the dropdown menus.  If not, you can also drag the subtitle file (if it is a separate file) into the currently running movie in VLC, which will add it to the menu, and then you can select it.

### Post by paapereira on 2008-03-22
I was having the same problem. In my case I changed the "Subtitles text encoding" to ISO-8859-1. 
This option is available in Input / Codecs > Other codecs > Subtitles.
