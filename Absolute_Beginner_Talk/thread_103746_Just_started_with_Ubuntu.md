### Post by monchichi on 2005-12-15
Howdy.

Cedega-- I actually get *better* performance with some games. It works surprisingly well. Unforunately, lots of games, including Civ4, aren't supported yet. There's been quite a demand for Civ4 in transgaming forums and what not, so it will probably be playable pretty soon. Overall Cedega is very frustrating because it's not GPL. Eventually, Wine will have include directx support and cedega will be be pointless. :razz:  

There's an NTFS driver with write support called captive-ntfs which I've used and it worked fine-- but it's considered unstable and you have to grab some files off your Windows installation to use it. You should be able to read your NTFS partition just fine with the stock kernel driver... But yeah, using a fat32 partition for any shared files is defintiely the best way to go... then you can slowly wean yourself off Windows.

How can you help?
Refer to [http://www.ubuntulinux.org/community/participate]("http://www.ubuntulinux.org/community/participate")
