### Post by Nequeo on 2005-07-12
It sounds like you're after something like [Eclipse](http://www.eclipse.org/). 

However, it's not currently in Ubuntu 5.04 repositories. If you search the forum, you can probably find help on how to download and install it manually. It's probably a little over the top, though, if you're just starting out learning to program.

I learnt to program using Python, which luckily comes pre-installed on Ubuntu. Before you go all out on a visual IDE, I'd reccomend taking a look at some of the Python IDEs that do come in the Python repository. Noteably, 'eric' and 'IDLE'. 

IDLE is written entirely in Python... but it suffers from weird, random lock-ups on both of my Ubuntu machines. Eric is much more fully featured, but it uses the KDE libraries so it may look a little ugly on a Gnome desktop.

There's also a fairly good beginners reference to Python on-line at [http://www.freenetpages.co.uk/hp/alan.gauld/](http://www.freenetpages.co.uk/hp/alan.gauld/) , and an advanced guide at [www.diveintopython.org](www.diveintopython.org). 

Cheers,

### Post by Nequeo on 2005-07-12
IDE stands for 'Integrated Development Environment'.

For example, Microsoft Visual Studio. They do things like check your code syntax as you type, highlight different keywords in different colours, indent your code for you automatically... that sort of thing. 

But if you're just starting to learn to program, it's best to keep it simple. All you need is a simple text editor like 'gedit'. 

Python is probably the best language to start with. I'd reccomend visiting that website I mentioned in my first post. It an on-line version of the book 'Learning to Program', and it covers Python, Javascript and Visualbasic. 

Once you've gone through that, you'll be in a better position to read the more advanced books and pick up harder languages. 

[EDIT]

Found this... [http://www.ubuntuforums.org/showthread.php?t=8682](http://www.ubuntuforums.org/showthread.php?t=8682) 

13 pages of hints and tips about where to start learning to program, Hope that helps.

An IDE is a development environment which can do things like check your code for errors and so forth. There is a wide debate in the programming world about whether they increase/decrease productivity/are a help to learning or a hindrance and so forth. That shouldnt be run into here.

The first idea into what you want to learn would be to ask... what problem do I want to solve, as like with tools, languages and so forth are very suited to what you want the finished application to be able to do.

Then there are many helpful resources on the Internet for each language - there are links above in other people's posts. Have a look round - you may well even want to buy a book at some point as a reference/teaching resource.
