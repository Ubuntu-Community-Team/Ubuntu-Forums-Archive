### Post by TeaSwigger on 2007-08-30
Hello,

Don't feel bad. We all start somewhere eh? 

About firefox, yes there are ways to get the latest version. One way is called ubuntuzilla and it comes with its own updater to keep it updating. Their site:
[http://ubuntuzilla.wiki.sourceforge.net/](http://ubuntuzilla.wiki.sourceforge.net/)
Also can install the latest Thunderbird or SeaMonkey suite if you care for those.

Don't worry too much though. Linux simply doesn't have many of the kinds of issues Windows does so you can just use the version of Firefox it comes with until you're ready to fiddle with updating it.

Hang in there. It's well worth it once you start finding your way around. :)
