### Post by x1a4 on 2007-03-31
Hi,

The Ubuntu logo means the package is supported by Ubuntu.  

The squares (which are clickable) mean the following:  

white = package is not installed
white with arrow = package marked for installation
green = package installed
green with star = package upgradable
green with arrow = package marked for reinstallation
green with red cross = package marked for removal
