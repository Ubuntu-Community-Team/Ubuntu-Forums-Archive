### Post by viciouslime on 2007-01-08
It really is all just a matter of opinion. Kubuntu contains the kubuntu artwork, so you get kubuntu logos instead of ubuntu ones. But the only other difference is that kubuntu-desktop is installed, if you install this package, as well as installing the artowkr, it will also install all the default kde applications, like amarok instead of rhythmbox, etc.

Personally I prefer gnome and its applications, but you just have to try both and make your own mind up. Remember, kde apps run fine on gnome and vice versa, they just don't always look as pretty :mad:

### Post by pcomelade on 2007-01-09
If you are not sure which one you prefer you can install ubuntu (gnome) and later install the kde desktop easily ($ sudo apt-get install kubuntu-desktop).
This way you will have both desktops available and see which one fits better with you.
