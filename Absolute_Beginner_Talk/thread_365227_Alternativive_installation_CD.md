### Post by jvc26 on 2007-02-19
256 should enable you to run the live cd but by the sound of it it is painfully slow. You can install the OS via the alternate cd and it will work faster (as it is running off the HDD rather than from the RAM) but that means you have to install before you can try it out. You could also try Xubuntu which is a slightly different version of Ubuntu designed for PCs with lower specs. Ubuntu should be able to run on 256 though.
