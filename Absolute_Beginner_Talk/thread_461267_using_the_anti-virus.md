### Post by greybirds on 2007-06-01
Once you set up things it's really not necessary, but you can use the command "clamscan" in a terminal (go to Applications > Accessories > Terminal to open a terminal). I'm sure there are graphical front ends available as well.

Here's the documentation from the clamav web site. You might find it helpful. [http://www.clamav.net/doc/latest/html/node1.html]("http://www.clamav.net/doc/latest/html/node1.html").
