### Post by siacs on 2006-08-14
It depends on the browser you're using. 
In Opera goto - Tools > Preferences > Shortcuts > Middle Click Options

There you can set middle click to do various things - open a link, open in a new tab/window, start panning, paste text.

In Firefox it can depend on your installed extensions but generally
goto - Edit > Preferences > Tabbed Browsing > Features or Tab Focus

### Post by gThree on 2006-08-14
@gargoyle:

Sounds like the visual indicator for a grabbed link (result of a left-click + drag action).  Possible you're accidentally performing that action every now and then when aiming for middle button.  Possible middle button default is left-click behavior so any motion while clicking the link with the middle button is producing the same result.
