### Post by oedipuss on 2006-09-16
Why not use the ubuntu package of swiftfox from [here]("http://www.getswiftfox.com/ubuntu.htm") ? 

The zipped file you downloaded doesn't need compiling (that's what ./configure && make do ) . You just type ./swiftfox to run it. Eventually you'd want to make links from your panels or desktop that run swiftfox instead of firefox. 

Still I'd strongly recommend using the package for ubuntu, it's better policy. Packages can be better managed, and I'd bet that it will create those links automatically for you.
