I edited the /etc/x11/xorg.conf file to turn off the touch pad button click on my laptop. Now the system won't boot. I can boot into the sysetm recovery and i get a command line. what do i use to  edit the file from here to undo what i did before?

---

### Post by ahaslam on 2006-07-31
**sudo nano /etc/X11/xorg.conf**

ctrl & X to exit 
Y to save 
Enter to use existing name
