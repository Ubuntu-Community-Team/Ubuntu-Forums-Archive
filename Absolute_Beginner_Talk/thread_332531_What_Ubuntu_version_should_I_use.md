### Post by Zdravko on 2007-01-06
O, fellow Ubuntu users! I am thinking of setting a dual-boot system, where WindowsXP and Ubuntu should function together ;)
But I have problems choosing the right Ubuntu version for my needs.
My main concerns are:
[LIST=1]
[*]Stability and reliability: I want to work on Ubuntu for at least 5-6 years. Without any re-installations. No matter what applications I install. No system crashes or data-losses are allowed.
[*]Security and performance: No anti virus applications, nor firewalls should be needed. Personal data should be locked and stored in secure place. No matter how long the up-time is, I want my Ubuntu to work flawlessly and without interruptions.
[*]Innovation and productivity: latest releases of applications are not a must. But **working** versions are needed. I don't want to loose all my date, just because of a bug somewhere in an unfinished application.
[*]Simplicity and outlook - no fancy stuff please! The computer machine will get older with the time. No need to bloat additional glassy mess. It will only make my work "mission impossible" :)[/LIST]Now, we can go on to the possible setups:
[LIST=1]
[*]Ubuntu Dapper Drake 6.06.1 (Long term support)
[*]Ubuntu Edgy Eft 6.10 (unstable release)
[*]Ubuntu Feisty Fawn 7.04 (not yet released, but I keep it in mind, however)[/LIST]Notice, that I am not going to make 'dist-upgrade'. I will stick only to the security-related upgrades. This way, I reckon, my system will remain stable and untouched from bugs.
What Ubuntu version should I use? Maybe Dapper is the best choice for me?
