### Post by njparton on 2008-01-04
Search for what you require in Administration > Synaptic Package Mananger (in the gnome menus at the top of the screen).

It's a GUI program so it the best place to start. Right click on a program to see the installation/uninstallation options.

It's also possible to install programs from .deb files and compile them from source code, but I would stick with Synaptic for now.
