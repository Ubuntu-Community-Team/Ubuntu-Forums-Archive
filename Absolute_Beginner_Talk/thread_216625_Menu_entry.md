### Post by catlett on 2006-07-15
If you're saying, you have made an entry in the menu that works but you want a shortcut somewhere so you don't have to open the menu, then do this.
Go to the entry in the menu but right-click on it. A menu will come up with options. Select "add this panel" to put a shortcut up at the top next to firefox and the menu or select "add this launcher to desktop" to put an icon shortcut on the desktop.
You can make the shortcut without an entry in the menu if you want. The same method is used. Either right click on the desktop or the panel (top or bottom panel, depending on where you want it)Then select "create launcher". The process is pretty much the same as making a menu entry except it becomes a shortcut (launcher) instead of a menu entry.
