### Post by doddo on 2008-02-09
hello!  I'm glad you like ubuntu.

if you open a file browser window then go under Preferences, click on the tab "Behaviour" you can set an option to prompt before a file is deleted

And The files you delete, mostly end up in a hidden folder in your home named .Trash

if you do
```

ls -l  ~/.Trash

``` From a terminal you can see what stuff is in there
