### Post by Sturmeh on 2007-10-24
On the Windows Box, be sure to share the "External HDD" like you would normally on a Windows Network, by rightclicking the drive and sharing, give it a name and read/write access... ( Remember the name. )

Next, unless you have problems with samba... ( Reply then? )

Go to Network under places, then browse the windows network for the new share.
