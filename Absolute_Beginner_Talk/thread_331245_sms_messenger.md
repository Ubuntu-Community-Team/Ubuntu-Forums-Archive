### Post by Jussi Kukkonen on 2007-01-04
Sending, yes -- you'll need access to a SMS gateway of course (something that actually sends the messages to the phone network). There were some free services last I checked, but you may need to jump through some hoops or be a customer of a certain service provider. 

This will show some tools available in Ubuntu:
```
aptitude search sms gsm
```

check [http://en.wikipedia.org/wiki/SMS_gateways](http://en.wikipedia.org/wiki/SMS_gateways) for more info,
