### Post by Sef on 2008-01-12
> I would really like to run Ubuntu, I tried the live CD and I got a black screen that kept flickering.

1) Run it in safe graphics mode and see if that works.

2) Did you [md5sum]("https://help.ubuntu.com/community/HowToMD5SUM") the download? (Checks to make sure download is good.)

3) Did you burn the iso at 4x or less? (Faster can corrupt the burn.)

4) Instead of start/install, go down to 'Check Disk for Errors' and see if it is ok.
