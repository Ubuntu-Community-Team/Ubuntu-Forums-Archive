Depends on what setup you have. For example, if you don't have a Printer you probably don't need **cups **running. If you don't use file sharing over the LAN etc you can probably uncheck **Samba** and maybe **NFS**.

You'll know if you disabled the wrong thing though, as something won't work:)

In general if you are not sure leave it runnning, or search the forums.
