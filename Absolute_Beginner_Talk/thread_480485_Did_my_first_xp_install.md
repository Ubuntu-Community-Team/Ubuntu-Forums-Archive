### Post by vexorian on 2007-06-21
Installing windows XP on vexorian's machine.

I am a victim of trauma since I had to do this 10 times and it is always a terrible process.
- Boot the CD see it expend 10 minutes loading some gibberish in text mode install.
- Select reinstall, since restore would obvious not work.
- Type serial key.
- Wait 40 minutes while text mode installer copies a bunch of files.
- Reboot, enter the graphical installer (finally)
- Click next repeteadly just make sure not to skip the regional version that you must configure manually else your keyboard will suck.
- Begin installing , it says 38 minutes left, but it will take 3 hours.
- When the system reboots notice how there is no sound in the first time presentation and that the resolution of the screen is 640x400 . 
- Spend the next hour making sure you got all the required CDs to make the system barely usable:
    motherboard CD (else sound won't work), nvidia CD, HP printer CD, HP scanner CD, MSoffice CD, photoshop CD, warcraft III CD, warcraft III expansion CD, 3dsmax CD.
- Use the next 6 hours in installing all the required drivers and software...

- Open internet explorer and rush to mozilla's page, install firefox.

Done!
