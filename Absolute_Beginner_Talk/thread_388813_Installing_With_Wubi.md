### Post by Efwis on 2007-03-20
here is how i see it, If you are looking to extend your battery life and operating efficiency, then why would you run Ubuntu Linux through windows??? You will not see an improvement over these since this is still running using Windows as the base. This is just creating a virtual desktop in your windows install, which means you are still running windows with Ubuntu on top of it. 

based on this line of logic, you won't increase efficiency or battery life because windows is what is killing it in the first place.
