### Post by njknight on 2007-07-23
Hi and welcome to Ubuntu.  If you need a program to remind you at 'boot time' of up-coming events I think that there is a program in the synaptic Package Manager called 'Birthdays' - for everything else Diary Calendar you have Evolution or Thunderbird.  Also worth a mention is reminderfox [http://reminderfox.mozdev.org/](http://reminderfox.mozdev.org/) this add on for firefox brings up reminders whenever you start firefox and can indicate birthdays/events and to-do's.  For notes I use the 'tomboy' notes program, it's easy to use, it doesn't forget, you can 'search' for items and have as many sections as you like.
