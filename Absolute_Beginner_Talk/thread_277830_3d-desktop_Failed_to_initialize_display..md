### Post by taurus on 2006-10-15
If you want to log into root from a terminal and bring up any graphic, you need to set xhost and setenv.  Otherwise, just use the sudo command from a terminal as a regular user...

Try these two commands and see what happens.

From a terminal as a regular user:

```
sudo gedit
```
Log in as root and:

```
gedit
```
