### Post by Sonic Alpha on 2006-07-17
On the list of CDs to download you have "Desktop CD", "Server CD" and "Alternate Install CD", in that order.

You want the "Alternate Install CD", near the bottom of the page.

If you have a Pentium based computer pick 
"PC (Intel x86) alternate install CD"

If you have a Mac, pick "Mac (PowerPC) alternate install CD"

And if you have a 64-bit processor, pick "64-bit PC (AMD64) alternate install CD"

Update: If you're still having issues trying to install, you may have to choose "[Xubuntu]("http://www.xubuntu.com/")" instead, as it is designed to run on lower spec machines.

Hope that helps :)
