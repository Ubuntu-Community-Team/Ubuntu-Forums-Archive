I d/led and installed the updates which included a Kernel update.  Now on  my start up menu I have two different versions I can boot to (thank God).  How do I take the new kernel version off and leave the old one on?  With the updated kernel, I can't connect to the internet. ### Post by confused57 on 2006-08-02
A temporary fix would be to go into your /boot/grub/menu.lst and comment out(place a # in front of) the new kernel entries...it wouldn't show up in your grub menu at startup, just make sure you comment the latest kernel that you don't want.  It can be uninstalled using Synaptic Package Manager, but you may want to keep it for the time being.
