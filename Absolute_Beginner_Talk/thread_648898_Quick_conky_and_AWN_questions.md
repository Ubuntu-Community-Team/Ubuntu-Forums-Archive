### Post by kpkeerthi on 2007-12-24
> 
1- If I want to install AWN, do I have to install the package 'xgl' first? (I'm on Gutsy)

Not really. You just need a compositing manager. If you have "Desktop Effects" enabled and working , you are ready to install AWN.

> 
2- How can I make conky runs when I turn on my PC (like startup in Windows)? Do I add it in inittab file?

Under System -> Preferences -> Session, add **conky** for the command
