### Post by irish_flu on 2007-08-27
If you're connecting to a Linux machine then VNC is indeed what you're looking for.  If you're connecting to a Windows box, you can either set up VNC or use "tsclient " (which connects using Microsoft Terminal Server/ Remote Desktop).

Yes, there are VNC servers and clients available in the repository; I'm using "vnc4server".  Do a search for that package name here on the forum, and you'll find instructions for setting it up.
