You can use Network-Manager, which can be found in Synaptic.

[QUOTE=Cbl12886]2.) I use MS Visual Studio 2003 for most of my programming and I was wondering how I install that.  I have a CD with the file that needs to be unzipped to run. You can't use Windows apps directly on Ubuntu, you'd have to try using wine. However, an easier way would be to use Eclipse, which is a very capable IDE which is available on Ubuntu. You just have to enable the Universe repositories in Synaptic.

[Synaptic Howto.]("https://wiki.ubuntu.com/SynapticHowto?highlight=%28synaptic%29")
