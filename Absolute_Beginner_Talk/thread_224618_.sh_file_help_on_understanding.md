### Post by cilynx on 2006-07-28
You need to install the 'build-essential' package before you have 'make', 'gcc', etc available to you.  It's a holy battle as per whether or not 'build-essential' should be installed by default or not, so please don't go there.  Install it and everything should work for you.

Good Luck

### Post by Lord Illidan on 2006-07-28
Specifically, you have to do this:

```
sudo apt-get install build-essential
```

The wiki [http://wiki.ubuntu.com/](http://wiki.ubuntu.com/) will provide more useful information and other hints too.
