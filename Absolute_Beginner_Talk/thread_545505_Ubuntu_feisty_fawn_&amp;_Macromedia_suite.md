### Post by mlentink on 2007-09-07
Just as many ICT companies, Ubuntu uses codenames for specific releases. Version 6.06 was nicknamed "Dapper Drake", version 6.10 was called 'Edgy Eft", version 7.04 "Feisty Fawn".

If it ran on Dapper (=6.06) there's a fairly high probability it will run on 7.04 (Feisty) as well, and the install-procedure will be tretty much the same.
