### Post by Zuph on 2007-04-20
A fresh install should always be cleaner, but you probably won't have an issues with an upgrade.  Most of the time upgrades fail because of the various hacks that you put in place to make the previous version work.

If you don't have anything to lose, why not do the upgrade?  If it works, Great!  If not, you haven't lost anything by doing a completely clean reinstall anyway.
