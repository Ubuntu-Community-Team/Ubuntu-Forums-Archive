### Post by jdriessen on 2007-06-05
if you go into your home folder and to .themes and in there are the gnome gtk2 themes. open the one you're using, find a file called "gtkrc". this is the configuration file of that theme. you can change the colours in this file. be carefull to only edit the lines you want to and don't make mistakes, you could screw the way your GUI loooks.

in anycase make a backup of it to restore it later if you mess it up. have fun.
