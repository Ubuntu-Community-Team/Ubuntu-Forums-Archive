### Post by Golyadkin on 2007-06-15
Well, what I think seems to be the problem is that you expect that you install software through Firefox? 
Maybe you are used to downloading installers in Windows, using your webbrowser. In Ubuntu you don't necessarily have to do that. It is possible to download a .deb file which kinda works as a setup.exe in Windows. But the easier and nicer way is to simply browse the handy lists of available software, check what you want to have and hit apply! It has never been this easy, ever. 

Also, there is the Synaptic package manager. You can find it by clicking in the bar at the top on "System  >   Administration  >   Synaptic Package Manager".
