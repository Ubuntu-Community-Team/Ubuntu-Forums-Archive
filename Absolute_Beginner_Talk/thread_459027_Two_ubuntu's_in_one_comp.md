### Post by elpichi on 2007-05-30
i almost had this same problem, but what i did was i used norton partition to completely delete the old ubuntu partition, but since you don't have windows anymore, on the ubuntu installation it asks you if you want to use the whole drive for ubuntu(is one of the options they give you to partition your disk). if you don't feel like reinstalling ubuntu you can use the Gnome Partition manager from the repositories on which you could just erase the second partition on your hard drive.
 I"m not sure if your Grub will automaticly update on this.
