### Post by AndyCooll on 2006-12-01
I always say start off with Ubuntu, for as Sef says if you run into any issues you will get a faster response since that is the default install. When you begin to feel more comfortable with Ubuntu then by all means have a look at Kubuntu.

You can also have the best of both worlds by typing into the command line the following:

```
sudo aptitude install kubuntu-desktop
```

Once that is installed logout, and on the login screen there is an option to choose your "session", you'll now see KDE as an additional choice.

### Post by lumpki on 2006-12-01
*I some how set Kubuntu as default and wish to change it back to Ubuntu*

"gdm" is the gnome display/login manager, the default in Ubuntu. "kdm" is the login manager for KDE. You can select which one you want to use with
```
sudo dpkg-reconfigure gdm
```
