*EDIT* : I've had a quick search of the forums, and found something that might be of use.

Try copy/pasting this command into terminal (Applications>Accessories>Terminal):

```

```

```

  Does read CD-R media

  Does read CD-RW media
  Does write CD-RW media
  Does read DVD-ROM media
  Does read DVD-R media
  Does not write DVD-R media
  Does not read DVD-RAM media
  Does not write DVD-RAM media
  Does support test writing

```

Plus a bunch of other stuff. I don't understand it all, but it does seem to give indication of the media your drive should support- and it should show whether the drive itself is being detected right.

### Post by agustin.g on 2006-12-01
Put your Ubuntu live-cd in the drive, boot from it, make a new partition (big enough to hold all the important data) (with the Gnome Partition Editor: GParted), format it in FAT32, preferably (so you can access it from Windows). (If you are using EXT3 (the Ubuntu default Filesystem), you need not worry about data loss, EXT2 and EXT3 can be easily resized without any problems). Reboot your computer, take out the CD, boot normal Ubuntu, mount the new partition, copy the important data on it. You are now free to install Windoze on the other partition, or try reinstalling Ubuntu, changing versions (from Edgy to Dapper, or vice versa).

I know this is not a solution to your actual problem but it's a try to help you not-lose the data you need to back up, for the moment at least.

IF there's an error in my instructions or if they're incomprehensible, please notify me and i'll gladly correct/clear up the instructions.
