### Post by kevinatkins on 2008-04-01
NVu / Kompozer are probably the closest thing to WYSIWYG web development you're going to get in Linux.. incidentally, Kompozer is just a 'continuation' of NVu, development of which seems to have stalled somewhat...

I'd suggest getting to know Kompozer a bit better - it's quite a good application, if not completely bug-free.

Failing that, if you're more comfortable using MS Publisher, you could run that using Crossover Office (or Wine, but for ease I'd recommend Crossover!) - Crossover isn't free, but it's good value..
