### Post by eentonig on 2007-07-31
in gross terms.

Debian ==> stability
Ubuntu ==> Bleeding edge, latest of the latest apps.

Debian 'Etch' is the official curent Debian release and can be compared with Feisty Fawn for Ubuntu. But for debian, a release can only be considered ready for releasing, when they are 99,9999% sure it's bugfree. Which makes that they aren't very eager to include new apps or features in it.

If you want the latest of the latest, you can install Debian Sid or Gutsy Gibbon. But be aware that this comes with bugs, crashes, incompatibility, etc...
