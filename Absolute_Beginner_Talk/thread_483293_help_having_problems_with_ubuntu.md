You see where it says section 'Screen' (the first one, I don't have a clue why there is a second one with a # afterwards)?  Well under there it says modes, and the only one that you have listed is 640x480 with 60 hertz refresh rate.  Go to a terminal and do sudo gedit etc/X11/xorg.conf , then try adding 800x600 and 1024x768 after that, so that the line looks like this:

modes "640x480@60" "800x600" "1024x768"

Make sure you leave 640x480 as the first mode in the list, this way it should stay the default if things don't work.  Save the file.
Once you've changed that line you need to restart X, by pressing ctrl+alt+backspace .  Then go to display settings and try the new resolutions.  If there are no new resolutions, try adding the modes that I said under the screen # section, but only as a last resort as I don't know what that section is for.

**If by any chance this messes up and you can't get back into X you need to go into the recovery console (on the boot menu), type nano etc/X11/xorg.conf and undo the change that you made**
