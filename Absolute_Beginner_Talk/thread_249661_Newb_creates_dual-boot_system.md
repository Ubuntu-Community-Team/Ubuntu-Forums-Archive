### Post by bobmorris on 2006-09-02
I just completed a dual-boot install, my first.

Other might find my experience helpful. The install options I got were somewhat different than in the help files online.

The first option on mine was 
"resize IDE1 master partition #1 (hda1) and use freed space)"

There was a slider bar showing 25gb free, which I knew was the free amount on what Windows calls C:. I moved the slider back to 17.5gb so C: would still have 7.5 gb free.

I clicked next. It started the process, then said
"the following will be formatted
partition #3 of /dev/hda as ext3
partition #8 of /dev/hda as swap" 

While I wasn't sure why it named them #3 and #8, this looked to be what it should be doing, so I clicked next. It formatted that free space, did the install, and when I rebooted, there was a text menu to choose which OS to use.

So, if you have ample free space on Windows C:, you can do an install easily, at least I did.
