### Post by lyceum on 2007-08-20
When you start up Ubuntu by the the CD, called a "Live CD" you chose the first option, run or install Ubuntu. This starts your computer and runs it from the CD, hence a "Live CD" as you are running the OS before you install. If you like it and want to install, click the install button. Any changes  you make while using the CD will not save, as it is a CD and you cannot save to a CD. It will run slower than your installed Ubuntu. Hope that helps.

Enjoy.
