### Post by RomeReactor on 2007-08-02
Hi. [These]("http://ubuntuforums.org/showthread.php?t=489201") [threads]("http://ubuntuforums.org/showthread.php?t=445764") may help you start looking for games in the repositories; [these]("http://ubuntuforums.org/showthread.php?t=426673") [other]("http://ubuntuforums.org/showthread.php?t=359896") threads cover some games that you can install on your own by downloading packages or installers. [This thread]("http://ubuntuforums.org/showthread.php?t=427205") in particular is very comprehensive.

There's also the [Ubuntu Documentation]("http://doc.gwos.org/index.php/Native_Games") page of native Linux games, with installation and setup instructions.

Then there's [Ubuntu Gamers Arena]("http://gaming.gwos.org/news.php") and [The Linux Game Tome]("http://happypenguin.org/") sites, and the [Gaming & Leisure]("http://ubuntuforums.org/forumdisplay.php?f=93") section of these forums. Hope this helps.
