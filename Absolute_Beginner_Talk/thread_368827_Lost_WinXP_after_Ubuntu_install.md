### Post by MasterOfDisaster on 2007-02-23
On startup, there should appear a message on the monitor saying something like "Press Esc to enter Grub:" and some numbers counting down.  It should appear before the Ubuntu start-up screen.  Press escape and you should see a menu with one of the options mentioning Windows.  Select it with the up and down keys, then voila.
