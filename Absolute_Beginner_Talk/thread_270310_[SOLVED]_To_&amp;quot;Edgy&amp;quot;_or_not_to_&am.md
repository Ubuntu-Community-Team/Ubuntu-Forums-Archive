### Post by _duncan_ on 2006-10-03
If there's room in your hard drive for another partition, it's also possible to multi-boot Dapper and Edgy on the same computer. This way, you get to test Edgy without having to wipe out Dapper. If for some reason Edgy doesn't agree with your system, you can always fall back to using Dapper. The 2 can even share the same swap partition.

BTW, I'm also still using dial-up. Downloading iso images of linux distros  using a slow connection isn't as tough as it looks. All you need is a good download manager and some patience.

My download speed ranges from 4.8 - 5.3 kBytes/sec. So a typical 650 MB download takes a little less than 40 hours. Before going to sleep I start the download manager, observe for a couple of minutes if the download is stable, switch off the monitor and leave the cpu running overnight. The next morning I'll pause the download and shut off the PC before going about my daily routine.

After 5 nights I have a brand-new linux version to play with. This is a lot faster than ordering CDs from shipit, which takes 4-6 weeks. Of course you'll have to do the usual md5 check, burn the iso image at low speed to a CD, etc.

I'm assuming you're paying a flat rate for unlimited dial-up (24 hours) connection like me. If your subscription is based on consumption, this process may not be feasible for you.

### Post by _duncan_ on 2006-10-04
Yes, torrent programs are commonly used to download iso images. It's actually the preferred mode so that the Ubuntu server doesn't get slowed down with download requests. Although it's also possible to download directly using programs like Free Download Manager in Windows and wget in linux.

The basic feature you're looking for is the ability to pause and resume downloads from where you left off. There are a lot of free programs available on the net.

Re. burning iso: Just to add to what petri said, if you're more comfortable working in Windowx, you can use programs like Nero instead. The basic requirement though is you have a write-capable CD or DVD drive.
