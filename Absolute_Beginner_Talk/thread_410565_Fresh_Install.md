### Post by Nythain on 2007-04-15
the first thing i would do is check the checksum of the iso and make sure its good, then i would try burning another copy at the slowest possible speed you can to ensure and accurate error free burn... i didnt put much stock in these recomendations untill i encounterd it myself... 40x equaled fried disk 2x equall perfect install using same iso...

if you still encounter problems... try the alternate install iso... no flashy livecd gui and much more stable for other reasons i dont know off the top of my head.

if you still encounter problems, maybe possible hard drive damage or something beyond my troubleshooting capabilities
