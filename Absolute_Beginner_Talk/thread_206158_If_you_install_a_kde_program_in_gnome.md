### Post by Jagot on 2006-06-29
The interface of a KDE app in GNOME (and indeed vice-versa) may appear slightly different from your other apps, but other than that, not really.  When you install a KDE app it will download the KDE libs that are required to run it, so there is no real difference - GNOME apps use the preinstalled GNOME libs.

### Post by pbaehr on 2006-06-29
Your memory will take a minor hit because your computer needs to load the KDE files required by your KDE application. If you have a reasonable amount of memory, this isn't really an issue, though.
