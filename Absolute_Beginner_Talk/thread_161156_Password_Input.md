### Post by aysiu on 2006-04-16
The notes? Which notes?
Most Ubuntu notes won't ask you to type *su*.
Take, for example, [these notes](https://wiki.ubuntu.com/RestrictedFormats#head-68565ae07a003332e82c9f23706638777396c249).

Oh, and when you're in the terminal, nothing will show up when you type your password--your password is still being accepted. Just press Enter when you're done typing.

For more on *su* versus *sudo*:
[https://wiki.ubuntu.com/RootSudo](https://wiki.ubuntu.com/RootSudo)
