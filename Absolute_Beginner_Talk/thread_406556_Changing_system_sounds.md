### Post by detroit/zero on 2007-04-11
Hi,

Just wondering how to change system sounds.

I realize that there's the sound selection app in System>Preferences>Sounds, but there's a couple I'd like to change that aren't included in the choices in that app. For example, when you change the login sound, it changes the .wav that plays *after *you've logged in, but nowhere is there a choice to change the sound that plays when the login window pops up (stuck on *question.wav* bongos).

I guess what I'm asking is, where is the file located that points to all the sounds? How can I change sounds that the GUI frontend doesn't allow? At the very least, how can I change that sound when the login window appears from those awful bongos to the sound of my choosing?

Thanks for any help.

---

### Post by Jussi Kukkonen on 2007-04-11
System --> Administration --> Login Window

It's in a different place since it's not a user preference, but a system wide setting.
