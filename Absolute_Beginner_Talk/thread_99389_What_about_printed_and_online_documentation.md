### Post by SilentCacophony on 2005-12-05
Hi.

I don't know if there are any printed books on ubuntu, but there are some ubuntu-specific sites with documentation. Also, be sure to check **System -> Help** from the Ubuntu desktop menus. I trust you've seen the official wiki on the main ubuntu site. Some that I currently have in mind:

A collaborative effort to organize doc info:

[http://help.ubuntu.com/](http://help.ubuntu.com/)

Variations of the Unofficial Starter Guide:

[http://makuchaku.info/amnesty/](http://makuchaku.info/amnesty/)
[http://ubuntuguide.squarecows.com/](http://ubuntuguide.squarecows.com/)

A work-in-progress pdf guide:

[http://www.psychocats.net/linux/ubuntu_user_guide_05.11.pdf](http://www.psychocats.net/linux/ubuntu_user_guide_05.11.pdf)

There is more scattered around elsewhere. Lots of documentation, if you can find it. ;)

As for the last question, from what I've seen, ubuntu users come from many distros, so it's hard to say. Many come from M$-only OS backgrounds. I'd say most *advanced* users probably simply come from some previous unix-like OS.
