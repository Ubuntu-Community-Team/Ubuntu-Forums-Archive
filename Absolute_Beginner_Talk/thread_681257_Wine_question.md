### Post by Rhubarb on 2008-01-28
Well, considering it hasn't been released yet, we can't tell you if it works can we?

The best way is to find out for yourself, is to grab the latest version of wine here:
[http://www.winehq.org/site/download-deb](http://www.winehq.org/site/download-deb)

Then just run it with wine.  You may have to tweak wine a little, run:
```
winecfg
```
And adjust some options there that may help.

This is the current qtrax ubuntu thread:
[http://ubuntuforums.org/showthread.php?t=678084](http://ubuntuforums.org/showthread.php?t=678084)

Please be mindful that qtrax is riddled with DRM, so if you are able to play music in the qtrax application in wine, it'd be heavily DRMed, so any music you have in qtrax, will only be able to be played in qtrax.
