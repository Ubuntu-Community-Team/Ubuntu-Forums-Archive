### Post by alphaniner on 2008-03-10
If vista is anything like other versions of windows, press F8 during the load screen.  In XP that's the black screen with the windows logo and the little scrolling bars beneath, but I've never used vista so I dunno if it has anything similar.

edit:
As for the Ubuntu side of it, I would unload the restricted driver if it is being used, at least.  Beyond that I'm not sure, except maybe set your resolution to something small like 800x600 just in case.
