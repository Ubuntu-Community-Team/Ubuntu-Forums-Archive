### Post by schorsch on 2007-08-21
Add the following line to the file /etc/fstab
```
UUID=587507bd-d038-4d4a-ae55-0b743bb416ae /media/sdb1 ext3 defaults 0 1
```
and you should be done. Change "/media/sdb1" to whatever you want but be sure to create the directory first! It is an ext3 filesystem, isn't it?
