If you want to do it via the GUI, just right click the folder you want, then select 'create archive'. Change the archive type to .tar.gz, then click ok. Easy :P

If you want to do it via the terminal, type 'man tar' for the user manual, which should explain everything you need to know.

### Post by srt5 on 2007-05-01
First of all welcome to the world of Ubuntu and the Ubuntu forums :)

There are several archiving applications around that will allow you to create zip/tar/rar/jar and many other format archives for storing files.

The easiest way (in my opinion) is to use Archive Manager which is shipped with Ubuntu. It should be under the

Applications Menu >> Accessories >> Archive Manager

If it does not show up under that menu, then try right clicking and editing the menu (it may just be hidden) like it was for me. No doubt that several other users will mention other programs that you could use.

Hope this helps, and use the forums whenever you get stuck, the support from the Ubuntu community is very good. :)
