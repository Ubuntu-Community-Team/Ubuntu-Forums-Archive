I would try commenting this section out (use #):

Section "Device"
	Identifier  "aticonfig-Device[0]"
	Driver      "fglrx"
	Option	    "VideoOverlay" "on"
	Option	    "OpenGLOverlay" "off"
	Option "AIGLX" "true"
EndSection

Also, do a full reboot before running beryl-manager again.

### Post by igknighted on 2006-12-16
What ATI card exactly do you have?  If it is an x850 or prior, then try this.  Check this website out: [http://help.ubuntu.com/community/RadeonDriver](http://help.ubuntu.com/community/RadeonDriver) , as it will help you set up the open source driver to enable AIGLX.  If you have, say, an x1300 (or other x<four digit> card) you will need to use the fglrx driver, which is a pain and doesn't support AIGLX or the newest X.org yet.
