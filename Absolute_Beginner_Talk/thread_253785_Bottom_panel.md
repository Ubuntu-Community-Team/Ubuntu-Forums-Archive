### Post by Sweet Spot on 2006-09-08
What he said^^  And if you just want application launchers like I did, on the bottom, right click on the panel, select add to panel then choose custom application launcher. Then browse : "file system>usr>bin>"name of application"  For instance if you want a Gimp launcher, you'd go : file sys, usr, bin, gimp. And then double click on gimp. Then, click on where it says no icon to actually find an icon you want to use. 

I personally have downloaded some icon theme packs and like to experiment. Here's a shot of my desktop to show the launcher on the bottom. I made it transparent, and have it set to autohide. Really easy to do.
