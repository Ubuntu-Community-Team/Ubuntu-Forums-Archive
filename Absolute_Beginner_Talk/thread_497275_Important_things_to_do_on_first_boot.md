### Post by mdsmedia on 2007-07-10
> **eentonig said:**
> Concerning the first boot thing. Normally, during installation, all updates where already downloaded for ALL software on your system. And normally, you'll get a pop-up to give you a heads up when anything on your system gets updated (as long as it was installed from the repo's)

Concerning the mouse issue. I can't help you.As eentonig says, updates should be loaded during installation, depending on your method of installation.

Once installed, if there are updates available you will get a RED indicator in the top panel, including a popup advising that updates are available for your software.

I installed FireStarter as a front end to the inbuilt firewall (IP Tables) but this is not necessary unless you want to configure the firewall which is already there.

I also installed ClamAV (anti-virus) simply in case you receive a Windows virus via email and forward that email on to friends using Windows, the virus should be detected and won't be passed on.

You may want to install Automatix2 (although there are some who would disagree) as a simple and quick way to install multimedia codecs and other programs all in one go. I don't remember the page or which repositories are required but there are many posts on the forums regarding Automatix.

After that it really depends on what you want to do with your system. OpenOffice is already installed and ready to go. XChat is already installed for chatting. I'm not sure if GAIM was already installed for instant messaging. Evolution is pre-installed for email, Firefox is pre-installed for browsing.

You might like to install NoScript and AdBlock addons for Firefox.

It really is up to you what you want your system to be able to help you do. For example, I'm still using Dapper, although when I have time I'll be installing Feisty, to see what all the fuss is about ;)

As for your mouse, I'd be doing a Google search for the particular mouse AND Linux, to see if anybody has created drivers for that mouse and its extended button set.
