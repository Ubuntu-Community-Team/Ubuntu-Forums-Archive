### Post by ske1fr on 2006-10-24
> **mendingo84 said:**
> i burned a data cd using k3b but when i try to read from windows it simply doesn't "see" it, not even physically.
[list=]
[*]Is the Windows computer the same as the Ubuntu one? In other words, are you dual booting?  If so, we can discount the possibility of physical incompatibility between one DVD drive and another.  This is what stopped me from reading a disk made in Kubuntu:  a DVD-ROM drive in a Windows machine that only recognised DVD-R  and DVD-RW media.  Once I burned DVD-R I could read the disk in the Windows machine.
[*]Did you configure K3B to use Joliet?  Joliet file structures make the disk compatible with Windows.
[/list]
