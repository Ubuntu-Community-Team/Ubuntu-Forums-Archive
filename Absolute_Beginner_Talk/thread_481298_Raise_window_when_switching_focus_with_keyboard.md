I configured the keyboard shortcuts in a way that allows me to switch between windows with Alt+Tab. What I want to achieve is that the currently selected window gets highlighted immediately while I am pressing Alt+Tab several time without releasing the Alt key. In Windowmaker this functionality is called "Raise window when switching focus with keyboard". In gnome the window is raised when I release the Alt-key.

### Post by thelinuxguy on 2007-06-22
I think you are looking for "Move between windows immediately" under System >> Preferences >> Keyboard shortcuts. Default is ALT + ESC.
