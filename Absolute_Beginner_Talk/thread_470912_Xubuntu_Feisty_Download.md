### Post by dptxp on 2007-06-11
Boot with CD. Carry out Integrity check.

Your file may be bad or you got a bad burning.



Download iso using Bittorrent.

Download and install Bittorrent first.
Next click on the corresponding torrent file of XUBUNTU (say 20 to 30 KB)
Open in Bittorrent. You can point destination as your old downloaded file.
Then only bad files shall be replaced. Fast job.


After download burn iso into CD using InfraREcorder (download and install it first).
Use "Burn Image " command in ACTION menu. Select yourdownloaded iso file.
DO nothing else to any setting, not even speed.
It burns at dynamic speed that varies.

Boot with CD. Carry out Integrity check. If OK. Click Live/Install.
