### Post by jblebrun on 2007-01-03
Yes, it's possible. I've done it before. I basically copied the files from the KMail mail directory into Thunderbird's mail directory. It wasn't completely straightforward, but it is possible. You may have to rename some directories and get rid of all of Kmail's index files. Thunderbird will rebuild indexes for you, which will take a few minutes if you have a lot of mail. 

The best thing to do is probably to follow the procedure described on [the Thunderbird website]("http://www.mozilla.org/support/thunderbird/faq#kmail") for each folder in Kmail.
