### Post by Zikes on 2006-06-26
lol no problem.  The command would be about anything you could type into a terminal/console command line.  Most programs can be started just by typing their name.  The name's usually all lower-case.  Most everything in Linux is case-sensitive and most programs go with lower-case names to make it easier.  If you wanted to run XMMS (X Multimedia System, an mp3 player) the command would simply be "xmms".  For Firefox the command would be "firefox".  Command line options are available as well, just like at a terminal.

If you have trouble tracking down the proper command for starting a program you can usually find it in Synaptic Package Manager.  Just open Synaptic and locate the program's package, right-click on it, select Properties, then go to the Installed Files tab.  A full list of all the files installed by the package will be there, and usually near the top will be "/usr/bin/*programname*".  That "programname" part at the end will be what you need to type in to run it.
