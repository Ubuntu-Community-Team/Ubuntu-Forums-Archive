### Post by Marsolin on 2006-10-08
I would think any good [video editor]("http://linuxappfinder.com/multimedia/videoeditors") could do that, but I'm not aware of anything that is designed specifically for that purpose. There is a program called [Subtitle Editor]("http://linuxappfinder.com/package/subtitleeditor") that should be good for any text on the screen, but you probably need something like [Avidemux]("http://linuxappfinder.com/package/avidemux") or [Cinelerra]("http://linuxappfinder.com/package/cinelerra") to edit the actual audio.
