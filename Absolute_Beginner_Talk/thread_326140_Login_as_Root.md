### Post by macogw on 2006-12-27
preface the command with "sudo" for a temporary (15 min, I think) switch into root permissions.  Ubuntu doesn't let you run as root by default (bad for security).  Instead you use sudo.  I've heard of some sysadmins liking sudo because if something goes wrong, they can see which user messed it up.  When it just says "root" everyone can say "not me!"
