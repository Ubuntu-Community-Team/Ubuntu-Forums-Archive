### Post by aysiu on 2006-11-17
I've had very few problems with Edgy, but I know a lot of others have.

To update Firefox in Dapper, use this script:
[http://www.psychocats.net/ubuntu/firefox](http://www.psychocats.net/ubuntu/firefox)

If you're planning to upgrade to Edgy, I'd make a backup of your Dapper installation first, just in case anything goes wrong.

### Post by bodhi.zazen on 2006-11-17
Edgy works well for some, not so well for others. See the [sticky at the top of this page](s) for a reasonable discussion of the issues.

I would at least run on a Live CD first to see what it looks like on your system. Consider a fresh install. If you do upgrade, back up you data first.

[Firefox 2](https://help.ubuntu.com/community/FirefoxNewVersion) is easier. It is distributed as a binary. Download and try it. If you like you can either remove it of make it permanent.
