### Post by PmDematagoda on 2008-01-31
If you want a Ubuntu that has improved within one month, then you would have to go for a development version such as Hardy Heron. There are some good improvements on Hardy, especially concerning the user experience and new systems like PulseAudio. But as is normal with any development version, Hardy Heron still has quite a number of bugs and caveats, so if you face one of those that have been reported and being worked on, then you will have to live with it until it is solved.
