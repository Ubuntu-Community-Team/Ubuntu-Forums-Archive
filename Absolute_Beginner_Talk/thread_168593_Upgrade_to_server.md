### Post by Nequeo on 2006-04-30
[QUOTE=zxcvbnm]I was wondering if you could upgrade to server without reinstalling Ubuntu...Because i don't want to lose my data
Thanks[/QUOTE]

You really don't need to... A 'server' install is just the absolute bare bones Ubuntu system, with no extra programs or even a graphical desktop on top. 

Nothing is stopping you from installing server packages on your desktop installation. Just open Synaptic and search for 'Apache', 'MySQL', 'Postgresql', or whatever it was you were interested in serving up.

You might also want to do a search in synaptic for 'linux' by name only (not description), and hunt down the server kernal image. But this isn't really necessary.
