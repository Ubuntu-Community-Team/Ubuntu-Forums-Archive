### Post by Inxsible on 2007-06-05
There are lots of other neat little tricks.

You might know of a few already but try:

1) Super + Tab = Ring Application Switcher
2) Super +Ctrl+ Tab = Ring App Switcher across all desktops
3) Alt +Tab = App Switcher
4) Alt +Ctrl + Tab = App Switcher across all desktops
5) Super + F6 = Fade to Desktop
6) Ctrl + Alt + next = rotate desktop right
7) Super + n = toggle window negative
8) Super +m = toggle background negative

All of the above assumes that you have the appropriate settings switched on in Beryl. There are a lot more tricks, but these are all i can rbr right now. Play around with it and you will possibly find a lot more
