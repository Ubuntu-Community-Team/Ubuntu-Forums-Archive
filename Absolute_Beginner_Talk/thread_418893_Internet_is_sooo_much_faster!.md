### Post by drbob07 on 2007-04-22
The reason it's faster on Linux is because Linux does not use a "QoS" system.

Windows reserves about 20% of your bandwidth, so it can download updates without you really noticing it. However it reserves this 20% *all the time*, so you're bandwidth can become drastically limited. However it can be disabled by changing a service's settings. In Linux it just downloads updates full speed.
