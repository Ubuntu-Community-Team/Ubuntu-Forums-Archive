### Post by hezral on 2006-04-19
hi. i've been playing warcraft for a while but it uses a different resolution that my display resolution. so after i quit warcraft the resolution sometimes stays the same as when i played the game. is there a command i could put in a script or launcher so i could just double click it to change the resolution. i dont mind going to the systems menu but if there is an easier way, please tell. thanks

---

### Post by revdev on 2006-05-17
try this:

```
xrandr -s 1024x768
```

replace 1024x768 with whatever your desired resolution is.
