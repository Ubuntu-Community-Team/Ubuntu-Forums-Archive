### Post by Paqman on 2008-01-09
It's up to you. 64-bit processors can run in 32-bit mode no problem. If it's a Core 2 Duo you've got then it can run 64-bit.

A 64-bit reinstall would give you a small speed increase (rising to a large increase for some types of task) Personally i'd wait until you had another reason to reinstall (ie: Hardy) and then switch.
