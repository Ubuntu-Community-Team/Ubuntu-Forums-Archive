### Post by FuturePilot on 2007-11-03
You can install Wine a few different ways. You can open a terminal Applications>Accessories>Terminal and copy this into it
```
sudo apt-get install wine
```
Or you can go System>Administration>Synaptic Package Manager and search for Wine
Or since your using Gutsy you can also just click this link to install it
[apt://wine]("apt://wine")
Pretty cool eh?;)

For future reference here's a nice explanation of installing software in Ubuntu.
[http://www.psychocats.net/ubuntu/installingsoftware]("http://www.psychocats.net/ubuntu/installingsoftware")
