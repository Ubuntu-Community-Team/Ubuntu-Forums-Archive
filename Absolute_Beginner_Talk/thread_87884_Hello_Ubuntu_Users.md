### Post by parktownprawn on 2005-11-09
The short answer is use ssh. 

You have to set up an ssh server on at least on of the machines so that it will accept ssh connections.

Check out [https://wiki.ubuntu.com/SSHHowto]("https://wiki.ubuntu.com/SSHHowto")
for instructions on how to set it up and use it
