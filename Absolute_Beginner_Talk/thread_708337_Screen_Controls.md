Okay, now do you use Compiz? (ie, advanced desktop effects, like fade in or out windows when closed etc.) If yes, just press the Windows (Super) key and scroll up to zoom a specific area.

Now to change any specific Window to high contrast, just press Windows Key (Super) + N

Super + N toggles betweek high contrast and normal back again.

Hope this helps.
