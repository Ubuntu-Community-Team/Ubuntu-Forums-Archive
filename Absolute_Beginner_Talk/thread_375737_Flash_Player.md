The version of Flash you requires 32bit Ubuntu. Download the file directly from the Adobe site itself to your desktop. Extract the folder using the file roller. Once the folder is present, open a terminal and go root. Move to the Desktop directory, and then to the folder containing the installation file. Follow the directions Adobe has so thoughtfully provided in a text document contained within the folder and you should be fine. Remember that the browser will have to be restarted in order for the changes to take effect.
