### Post by gobbles414 on 2007-08-27
Hi glc,

Make sure that your system meets the following system requirements if you are trying to install Ubuntu 7.04:

*256 MB of RAM is required to run the desktop install CD
*The installation requires at least 4 GB of disk space.

If your laptop does not meet these requirements, you might try Xubuntu. Xubuntu 7.04 has the following system requirements:

*To run the Desktop CD (LiveCD + Install CD), you need 128 MB RAM
*192 MB of RAM to install (the Alternate Install CD only requires you to have 64 MB of RAM).
*To install Xubuntu, you need 1.5 GB of free space on your hard disk.
*Once installed, Xubuntu can run with 64 MB RAM (128 MB of RAM strongly recommended).
