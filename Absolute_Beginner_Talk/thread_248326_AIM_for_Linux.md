### Post by IYY on 2006-08-31
Yes, of course there are more apps, but they are sometimes not as easy to install. Makes sure to enable the Universe and Multiverse repositories if you haven't done so already to get far more apps in Synaptic.

If you find software in .deb form, it will usually be very easy to install, especially if it states that it's compiled for Ubuntu.

If it's in .tar, or soemething like it, it's source code that needs to be compiled. It's not really difficult, but not nearly as simple to install as Synaptic apps and debs.
