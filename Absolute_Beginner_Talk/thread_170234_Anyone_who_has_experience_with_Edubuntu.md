### Post by mips on 2006-05-04
1. I don't know.
2. Yes. The installer is a simple text based menu system which requires just a few inputs from the user and a couple of presses of the enter key. Very simple.
3. The hardware support for Edubuntu is exactly the same as for all the other Ubuntu distributions (Ubuntu/Kubuntu/Xubuntu). The easiest way to test hardware support is with one of the Ubuntu LiveCD's.

What is the spec of the PC, CPU, Mem, HD, GFX, Network etc. ? I'm asking this as Edubuntu uses Gnome Desktop Manager which has certain system requirements.

Another option is to simply install any of the Ubuntu versions and simply add the education packages that would be usefull.
