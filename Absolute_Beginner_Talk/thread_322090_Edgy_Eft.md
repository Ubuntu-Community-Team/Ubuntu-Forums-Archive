### Post by sawjew on 2006-12-20
You can upgrade to Edgy Eft at any time using the following command ```
gksu "update-manager -c" 
```

For more information on upgrading and other methods of upgrading see this page [https://help.ubuntu.com/community/EdgyUpgrades]("https://help.ubuntu.com/community/EdgyUpgrades")

This should have no effect on your Windows installation and should be bootable into Windows from the Grub boot menu as before the upgrade.
