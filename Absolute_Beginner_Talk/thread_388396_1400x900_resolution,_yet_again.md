### Post by Kobalt on 2007-03-19
Try to run the follwing command line 
```
sudo dpkg-reconfigure -phigh xserver-xorg
```
Then select the resolutions you want as avaliable with the spacebar, and press Enter to validate.
Finally restart X with Ctrl+Alt+Backspace, and the resolutions should be avaliable from System > Pref. > Screen Resolution.
