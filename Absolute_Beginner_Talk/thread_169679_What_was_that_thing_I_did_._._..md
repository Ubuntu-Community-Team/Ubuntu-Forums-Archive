### Post by hw-tph on 2006-05-03
Ubuntu and Kubuntu is the same distribution, only Kubuntu comes with a KDE desktop as per default. You can install the full KDE desktop by typing **sudo aptitude install kubuntu-desktop**. Then you can choose KDE or Gnome from the Session menu when logging in.
