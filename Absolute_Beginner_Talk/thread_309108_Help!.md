### Post by Scorpuk on 2006-11-29
You could also try:

```
gksudo nautilus
```

This will open the nautilus browser with superuser privilages.


Please be careful as to what you do with the superuser privilages as you can delete  / modify core files and make the computer not work anymore. :mrgreen:
