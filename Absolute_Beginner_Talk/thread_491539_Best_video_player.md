### Post by ggaaron on 2007-07-03
None, why? Because everyone likes different things, if everyone would prefer one video player, there would be only one.

But back to what you want to hear, I prefer mplayer (and it's gui gmplayer) because it plays many video types plus it is a great plugin for firefox. It has limited dvd playback (no menu), so I use xine-ui for dvds, but you can use kaffeine, totem and many others, as they use xine library and are only different guis, every one of them should play the same video types.

### Post by FoolsGold_MKII on 2007-07-04
I'd also recommend mplayer.

Prior to that my favorite was VLC, but I found out that VLC had some interesting wmv decoding issues, as well as some other issues which meant some stuff simply played better in totem than VLC (and vice versa), so I had to basically mix and match the two players. totem also didn't properly stream videos in Firefox, which was annoying.

That's the way things went until I discovered mplayer. mplayer plays everything perfectly, without corruption, with full and working plugin support for firefox. I also prefer the gui and the mouse-wheel control.
