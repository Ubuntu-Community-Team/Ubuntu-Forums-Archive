### Post by oilchangeguy on 2007-08-29
> **twin1961 said:**
> I have a HP laptop with WindowsXP and I want to add Ubuntu but I don't know how to partition the hard drive.

first back up your data. then defrag the hard drive at least twice. then install ubuntu. when prompted on how to partition the drive, it's usually easiest to let ubuntu use the largest free space. do not allow it to use the entire drive, or windows will be gone.
