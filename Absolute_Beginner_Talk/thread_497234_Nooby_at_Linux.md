### Post by st0nes on 2007-07-10
I recommend "The Official Ubuntu Book" by Benjamin Mako Hill, Jono Bacon, Corey Burger, Jonathan Jesse, Ivan Krstic (Prentice Hall, Print ISBN-10: 0-13-243594-2).  It gives an excellent overview of the OS, as well as background on the open source movement and the Ubuntu ethos.  It uses 6.06 (Edgy) for examples, but there are not too many changes if you're using Feisty.  Most important: it is available as a free downloadable ebook from various sites! I can't remember where I got it, but try googling the title.

Hope this helps.
