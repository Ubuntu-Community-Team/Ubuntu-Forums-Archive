### Post by jISh on 2006-09-18
A quick forum search would have yielded you [this]("http://www.ubuntuforums.org/showthread.php?t=183936&highlight=avi+dvd").

In the end, it's best to learn the ffmpeg command line and do it from there. You need MPEG-2 video with MP2 or AC3 audio. 

You can then use an application such as Varsha to build yourself an ISO image to be burned with Nautilus/Gnomebaker/K3b/whatever. 

Look into the program AVIDemux as well.
