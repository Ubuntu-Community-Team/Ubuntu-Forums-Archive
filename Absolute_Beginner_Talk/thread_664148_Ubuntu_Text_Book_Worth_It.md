### Post by eolson on 2008-01-10
I like having a couple books laying around for quick reference.  Right now I have
 Beginning Ubuntu Linux
 Hacking Ubuntu
and
 The Official Ubuntu Book
All three are good,  with the third probably the best for starters.

If you want to get down and dirty with the command line "Linux in a Nutshell" is good,  but not specific to Ubuntu,  but covers a ton of Linux stuff.  If you've ever entertained the idea of learning all the commands,  it devotes 369 pages to just the commands and the Ubuntu/Debian specific ones aren't included.  The book is close to 800 pages.  I'd say for serious users only.
