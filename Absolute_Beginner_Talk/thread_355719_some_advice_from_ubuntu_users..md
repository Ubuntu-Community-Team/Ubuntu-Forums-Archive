### Post by Shatrat on 2007-02-07
64 bit presents a lot of problems getting binary-only stuff like flash working, since nobody can just recompile it for 64 bit.  There are workarounds for most everything,b ut its really only worth the trouble if youre the type who will do it just because you can.  The performance increase is not really noticeable.

ubuntu-xubuntu-kubuntu are all the same thing with a different default window manager.  You can aways install kde or xfce on ubuntu and have all thee if you really want.  Dont lose sleep choosing which one to download.
Edubuntu is more of a server install for administering schools and running thin clients off a fat server.

Also, unless you have ubuntu on a really slow drive now, theres really no reason not to just leave it on the IDE drive.  2 drives are better than one.  I still have a maxtor 40 gig in my box thats almost 7 years old.
