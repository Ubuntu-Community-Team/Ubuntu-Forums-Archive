### Post by mahiyar on 2007-02-23
This is from where I installed my sensor temperatures package [https://help.ubuntu.com/community/SensorInstallHowto](https://help.ubuntu.com/community/SensorInstallHowto) . You need not go in to all the mkdev thing lm-sensor is standard package and you can get it from synaptic. However other than the install part other instructions apply. As I understand the success is motherboard chip dependent. As given in the instructions running >sensors in the CLI will give you temperatures. ANother nice graphic interface sensor programme is Xsensors, you could also get that from synaptic. You might need to restart before the sensor starts sensing.
