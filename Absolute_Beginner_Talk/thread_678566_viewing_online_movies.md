### Post by spiderbatdad on 2008-01-26
[http://www.adobe.com/shockwave/download/download.cgi?P1_Prod_Version=ShockwaveFlash](http://www.adobe.com/shockwave/download/download.cgi?P1_Prod_Version=ShockwaveFlash)

download the .tar.gz  scroll down a little and copy the instructions for installing the tar.gz.
download the package to your desktop
right click on the package and select "extract here"
open your terminal and enter```
cd Desktop
``` with capital "D" to get to the package on your desktop...the ```
cd install_flash_player_9_linux 
``` if thats the exact file name. (tip: if you just begin typing the first few letters in the name, the tab key will auto complete the name for you.)
Next ```
./flashplayer-installer
```
select "y" when prompted
when all done, you can throw the package and installer file away

watch video clips!
