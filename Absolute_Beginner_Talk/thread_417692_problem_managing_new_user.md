### Post by leibowitz on 2007-04-21
Please go to the System menu > Administration > Users and Groups.

In the "Users and groups" window, do you have a checkbox in the bottom left of it called "Show all users and groups". It should be just above the Help button (I have an old Ubuntu version so it might differ a little bit)

If yes, just check this box to see every users and groups on your system.


Anyway you should not delete application users, because they are used by the applications itself. Unless you deleted that specific application.
