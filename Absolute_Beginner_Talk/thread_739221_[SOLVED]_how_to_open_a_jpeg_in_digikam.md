### Post by c-ron on 2008-03-29
From the chapter 3 of the Digikam documentation: [http://www.digikam.org/handbooks/en/interface.html]("http://www.digikam.org/handbooks/en/interface.html")

Album->New Album (Ctrl-N)

    Create a new album folder

Album->Add Images (Ctrl-I)

    Select images to be added to the currently selected Album

Hope this helps.

### Post by Sbarton on 2008-03-29
For a jpeg on desktop try this:
Open digikam then .../Album/import/add images
you should then have a dialogue box in which you can select where the picture you want to edit is, then open it.
