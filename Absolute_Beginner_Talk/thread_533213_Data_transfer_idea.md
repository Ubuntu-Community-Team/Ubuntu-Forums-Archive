### Post by Inxsible on 2007-08-23
No.

The LiveCD home folder will lose all the data once you remove the CD from the drive. Remember, the LiveCD does not write anything to disk. Everything is in the physical memory, which is lost when you shutdown (which you will need to during/after the installation to boot into the installed version). 

Another issue I see is that you will be limited by the memory that you have on the machine, and so you wont be able to copy a lot of data, because you will need some memory to run the installation too.

### Post by Jimmyfj on 2007-08-23
This would not work. The Live Cd home folder is removed the second you reboot your system and with that your data. The content of the Live Cd is running in your System RAM. 

If you have a cd burner in your computer backup your data to Cd or DVD. Or to an external drive.
