### Post by quirks on 2008-04-18
Hi stubblepoo,

most Gnome applications use the proxy settings you enter under System -> Preferences -> Proxy Settings. Firefox doesn't respect these settings, so you have to configure it manually. In Firefox, go to Edit -> Preferences ..., select the Advanced category and switch to the Network tab. Then, click the Settings ... button.
