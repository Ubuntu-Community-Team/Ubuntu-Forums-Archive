### Post by forrestcupp on 2007-07-07
Open up the Restricted Drivers Manager and see if you need to install proprietary drivers.  Sometimes the open source drivers don't do opengl very well.

Also, are you running Beryl or Compiz?  Sometimes opengl games are sporadic when Beryl/Compiz is turned on.  If so, try turning it off while you are running games.
