### Post by erfahren on 2007-12-16
from the command line the only thing I can suggest is doing:
```

sudo dpkg-reconfigure xserver-xorg

```
sudo (for root permissions) may not be necessary there. If you boot into recovery mode you get prompted to enter the root password (enter yours) and you'll get the root command prompt ( # ).

that command will take you through some steps to reconfigure your hardware for the /etc/X11/xorg.conf file.  Most of the steps you can take the defaults. When it comes to selecting the video driver you really want "fglrx", the driver named "ati" is the open-source driver for ATI cards ("vesa" is the default basic driver).

I don't know of any good documentation for that.
