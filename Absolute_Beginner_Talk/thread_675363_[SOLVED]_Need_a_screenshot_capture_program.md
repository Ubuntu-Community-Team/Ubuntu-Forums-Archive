### Post by noynac on 2008-01-22
I use scrot, a commandline utility that's in the repo's. It's not fancy, but it is fast, easy on resources, and allows you to select an area of the screen to be saved.

To try it out, install scrot from the repo's. Then open a terminal window and type:

scrot --select

Finally, select some section of the screen with your mouse. After you release the mouse key, the screenshot will be saved to your home directory in PNG format. There are many options, which are detailed in scrot's manual page. If you decide to use scrot, it's easy to assign it to a key or create a custom launcher.
