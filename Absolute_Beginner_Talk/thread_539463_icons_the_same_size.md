I believe that there's a better way to do it, but, if I recall correctly, for each application in Desktop there's a file called **<application name>.desktop** in ~/Desktop. They're just plain-text files, that you can edit using your preferred text editor. Take a look at them and you'll see that one of the lines specify the image file to be used as the icon. You can change that to anything you want.

I know, it would be boring to do it for each application. That's why I think there's a better way. :)
