### Post by nu2this on 2006-08-18
Quick answer no.
What'll happen is this:
Say ur HD is 60G with just Windows You check by doing Start>My computer>C disk> properties.
You'llsee a graphic of a pie chart & it'll have somthing like used space 10G& freespace 50G (your's will vary)
If you do the Ubuntu autoinstall the Partition program in it will will thake that 50G of free space divide up by say 1/2 give windows 25G then put your Ubuntu on the remaining 25G.
So that if you do the window proceedure above it'll read used space 10G freespace 25G,that 25G is where anything you write to windows will go. Becase this is ur Windows Partition.
Your Ubuntu Partition will in this case be 25G compltely separate from ur windows partition.
Hope that helps!
