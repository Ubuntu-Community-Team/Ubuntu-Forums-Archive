### Post by podunk on 2006-10-17
Reading on the net – it seems this is a Debian error? Do you have Debian repositories listed in etc/apt/sources.list
?

Researching – every time I saw this error (first 6 results I checked in fact) there were Debian repositories mixed in with the native distribution repositories. I guess mixing repositories is a no-no.

Why don't you save your present sources.list file by
```
 sudo cp /etc/apt/sources.list /etc/apt/sources.list_messed_up
```

Then you edit the file and remove the Debian repositories
```
gksudo gedit /etc/apt/sources.list
```
