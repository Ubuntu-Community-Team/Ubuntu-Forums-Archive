I learned a lot from [http://www.psychocats.net/ubuntu/index.php](http://www.psychocats.net/ubuntu/index.php) You`ll find answers to a lot of the basic stuff there.

"Support" is AFAIK bugfixes and security patches for the OS and applications. It usually lasts 18 months for regular versions of Ubuntu, but has much longer support for LTS releases. ### Post by Obor on 2007-08-30
Ubuntu guides:
1. Ubuntu wiki [https://wiki.ubuntu.com/](https://wiki.ubuntu.com/)
2. Ubuntu Guide [http://ubuntuguide.org](http://ubuntuguide.org)
3. Psychocats site [http://www.psychocats.net/ubuntu/](http://www.psychocats.net/ubuntu/)
4. Ubuntu document storage [http://doc.gwos.org/](http://doc.gwos.org/) (currently down)
5. Ubuntu forums, which you've already found

Support period: starts with ubuntu release, which is every 6 months.

Ubuntu 6.06 was a long term support version (3 years on desktop 5 on servers) and so will be 8.04 (released in April '08 )
The non LTS versions are supported for shorter time but if you upgrade to the latest release that should not matter. The support period means that you will be getting security updates.

Hope it makes sense
