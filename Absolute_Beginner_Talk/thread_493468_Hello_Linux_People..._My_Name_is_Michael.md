### Post by TheTaylorEffect on 2007-07-05
Hello Ubuntu community,

My name is Michael. 

I decided to install 7.04 on a Windows laptop less then 2 hours ago... and I am writing this now from the comfort of a very sexy looking Gnome desktop.

This has been my first successful attempt to implement Linux!

I am impressed.

I've been reading a lot of RSS feed entries and Tech Blogs  about your strange and mysterious operating system of quite some time now. I've seen the news regarding Dell and its new love affair with Ubuntu... so I decided to see what all the hub-ub was about.

I am a Microsoft Certified Windows Administrator by trade - and I've been using Windows based systems for as long as I've owned a computer.

I've dabbled in Linux here and there over the last few years - using distros like Fedora and Mandrake (usually on Frankenstein computers made from old and obsolete and parts).

Typically I'd get frustrated with things like screen resolution problems, or networking issues and give up quickly. I'd feel bad for doing so (especially after investing so much time and bandwidth into procuring the correct ISO files from a remote FTP server and having failed in my attempts on account of my own impatience and ignorance).

But, Ubuntu 7.04 has changed my views on Linux forever. 

I wasn't 100% willing to part ways with my beloved WindowsXP Professional. A quick Google search turned up a number of How To's on creating a dual boot system.

I found [Matthew Millers]("http://www.matthewjmiller.net/howtos/dual-boot-linux-and-windows/") guide particularly helpful... and I followed his advice    in using the Linux System Rescue CD to correctly petition my drive.

I was pleasantly surprised when I was able to pull down the ISO of feisty Fawn in less then 10 minutes.

I was even more surprised when the installation seemed to take on on the first attempt...

I was even more surprised when the installation completed before the first inning of the Red Sox game (which had coincidentally started at the exact moment Ubuntu began to load)

And when Ubuntu correctly identified my wireless card, and connected to my network my jaw dropped.

I tracked down QuickSynergy on the Synaptic Package Manager, and it installed with ease (using my Windows XP desktop as the server).

Installing this operating system has been nothing less then a pleasure. Already the internet is noticeably faster than it had been on this computer while running XP.

Although I'm sure many Linux purists will loath my adventure, and despise this seemingly "watered down" distribution I have to  admit, this experience has made me proud to be a new member of the Linux community.

I hope you'll have me!
