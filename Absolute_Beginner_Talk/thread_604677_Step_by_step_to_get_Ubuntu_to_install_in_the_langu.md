### Post by miriam on 2007-11-06
My hard disk is blank, unpartitioned, completely raw.

I want to install ubuntu in my native language.

I boot off the Gutsy Gibbon 7.10 i386 CD, and ubuntu Live loads up.

I go to System -> Administration -> Language Support

I select the language I want, put a tick on it, and then hit apply. It successfully downloads 10 files and installs software. It then says "Successfully applied all changes".

Question 1: Since my HD is blank, not even a file system on it, where the hell is it downloading these language files to?

Anyway, I click "close" on the Success dialog. I then select my "Default Language" as the relevant language and again hit Apply. That goes fine. I then hit OK. Now it tells me I need to Restart, so I restart.

It boots back up again off the Live CD. It freezes and displays weird lines on the screen.

So I reboot again, and it loads in... wait for it... English.

Tried it a few times and this happens every time.

So how exactly do I install or use ubuntu in my native language?

Any ideas?

### Post by popch on 2007-11-06
Since you are first booting off the live CD, you should be able to choose your language even before the live session starts.

Immediately after booting the CD, you should see a menu giving different options. Also, you should see a timer counting down from 30 seconds.

Before the counter expires, hit the f2 key to select your language, then the f3 key to select your keyboard layout.

This worked every time for me. It should work for you, too, unless you happen to need a language which is not listed in that menu. Also, not all programs might have been translated so far.
