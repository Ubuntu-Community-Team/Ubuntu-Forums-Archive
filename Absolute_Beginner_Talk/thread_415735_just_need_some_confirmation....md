### Post by vanadium on 2007-04-20
You are correct. The partition shrinks, but the data remains intact. First defragment the drive of your friend  before proceding with the install (and repartitioning). 

Obviously make sure all the data (only the user data like text files, etc, not the applications) are backed up. There is always a possibility that things go wrong and that data are lost.
