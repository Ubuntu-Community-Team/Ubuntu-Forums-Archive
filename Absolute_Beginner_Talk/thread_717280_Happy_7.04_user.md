### Post by p_quarles on 2008-03-06
Normal releases are supported for 18 months following the release date. So, support for 7.04 will officially end in October of this year. 

You have a number of options at that point, though. Find a distro which continues to use the same version of GNOME present in Ubuntu 7.04, or switch to a lighter environment.
