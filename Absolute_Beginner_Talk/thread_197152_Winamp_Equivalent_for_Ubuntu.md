### Post by IYY on 2006-06-15
```
How to add skins to Xmms, the ones I've found are .zip files,the one that's in the skins folder is a tar.gz.
Plus, how do I get permmisions to write to /usr/share/xmms/skins folder? Thanks
```

Zip format is fine. You don't even need to extract it, just place the zip file in the skins directory. The theme doesn't need to be an XMMS skin, any winamp classic skin will work (not the modern ones). To get access to that directory you'd have to launch nautilus with gksudo. However, you can also place skins in your own home folder in the directory ~/.xmms/Skins. You have access to it already.
