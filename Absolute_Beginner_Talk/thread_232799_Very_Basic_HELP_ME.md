### Post by whein on 2006-08-09
Does the CD work in any other computers?  I would try to narrow down and isolate the cause of the problem.  If the CD does not work in another machine, try the suggestion of burning another ISO at VERY low speed with maybe better media.  Run diagnostic programs to make sure it burned correctly.
If the CD works fine, make sure that you have the option to boot from CD enabled on your target system.  Not all old computers even allow this, and many of them have it turned off by default.  Sometimes you have to change some settings in the BIOS to set the boot order (floppy, then CD, then HD1, HD2, etc...), and sometimes there is a special key you can press during or right after the PowerOnSelfTest(POST) that will let you choose the media to boot from.  Watch the messages that flash on the screen to see if your system has this feature.  Usually it's a function key or ESC or DEL.
Best of luck!
-Will
