### Post by az on 2007-02-05
> **bandie said:**
> you need as least 192mb to get the live CD to work.  

And you need 256 megs of ram to use the installer on the live cd.  The alternate cd should work fine, but Ubuntu will be very slow with so little ram.

For Ubuntu, you need 256 megs of ram to be happy.  For Xubuntu, you can do fairly well with just 128 megs of ram.
