A little off subject I know.

First, in my opinion XFCE is not for older or slower systems. It is lighter then KDE or Gnome yet has enough features to qualify for a desktop manager (window manager + applications like file manager, etc) and I agree of the three is the fastest. I would note that XFCE has been somewhat buggy over the last year.

If you are interested in speed, consider a lighter desktop manager or a window manager. Optinos include IceWM, Fluxbox, Balckbox, and Openbox. My preference is Fluxbox ( I like transparency), but IceWM is also nice. Openbox is very light weight but has no panel (although you can run a panel if you like, fbpanel, perlpanel, pypanel, etc).

With the window managers you pick and choose with pieces of KDE/Gnome/XFCE or run light weight applications.

For example, I prefer to burn with K3b, no problem.

There are several replacements for Konqueror/Nautilus like xfe or rox. Just as functional (in my opinion) but run faster.

Result: More power for applications, faster time from GDM/KDM to desktop.

If you are interested in optimizing speed but still want a nice WM, and are willing to try something new, look at Fluxbox or IceWM.
