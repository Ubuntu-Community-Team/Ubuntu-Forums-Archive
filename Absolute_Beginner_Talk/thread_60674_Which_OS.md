### Post by az on 2005-08-28
Read up on debian.

Install Ubuntu to see what Debian is like when it is well configured for the desktop.  If you find it too limiting, in that you feel you want to do things on your own, install debian.

Install slackware once you have been running Ubuntu for a week.  (just shrink a partition on your hard drive and install it there.  You will see whan a more "barebones, nitty-gritty" distribution is like.  Again, you can have that with debian, if you like.

I think you will be pleased with Ubuntu.  It has all the same things debian has to offer, just with less effort.

GCC is the C compiler in the linux world.  Look into python, perl or any other language you like.
