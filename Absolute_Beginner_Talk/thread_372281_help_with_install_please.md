### Post by old_geekster on 2007-02-28
Hey samster, welcome to the wonderful world of Linux/Ubuntu!

First, your HDD is going to present you with some challenges.  I have an 80GB HDD and if you eliminate the 3GB swap file, I am using 15GB's of disk space.  This is with simply the default programs installed.

Which version of Ubuntu did you burn, 6.06 or 6.10?

I am using the desktop version of U Edgy Eft 6.10, 32 bit and it works great.

Did you try to run Ubuntu from the CD?  This will allow you to use it without doing a complete clean install.

One more thought, did you do the disk integrity check?  This will test your CD to assure that there are no problems with the burn.  You do this before attempting to do an install.  It can save you allot of time if there is a problem.  Sometimes ISO's can be a pain.

I hope this will help!

Good luck!

p.s. After giving it some thought, since you are using an older PC, you probably should be using Ubuntu 6.06.  I believe it would do better with your components.  U 6.10 is the newest version and likes higher end components.
