### Post by Tatty on 2008-03-02
Yes that is very easy to do, i think most people probably have this kind of setup... Basically it gives it you as an option when you are installing. But do a little research before you go ahead with this as there is also the option to wipe everything, and you dont want to do that by mistake.

Basically you will need to partition your hard drive so windows and Ubuntu can both have their own partitions.

Make sure you defrag windows before you partition, to prevent accidental data loss.  And also back up anything which you absolutely could not risk losing - its rare, but things can go wrong.

Also if you have windows vista, it is advised to partition the drive with the vista partition tool, as it does not play nicely with the Ubuntu partition tool...

Try starting here for info and guidance [https://help.ubuntu.com/community/WindowsDualBoot?highlight=%28dual%29%7C%28boot%29]("https://help.ubuntu.com/community/WindowsDualBoot?highlight=%28dual%29%7C%28boot%29")

### Post by justin whitaker on 2008-03-02
> **RVApunkROCK said:**
> is there anyway I could install ubuntu as a 2nd OS and still keep XP on my system and be able to use either whenever I wanted?

Yes, you can dual boot Ubuntu and XP. Defrag your XP install, and use the installer for Ubuntu to resize the XP install to make room for Ubuntu. Then use the boot loader from Ubuntu (grub) to allow you to boot into either OS.
