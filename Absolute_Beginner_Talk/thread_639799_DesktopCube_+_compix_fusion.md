### Post by jonnywombat on 2007-12-13
To change the ends of the cube then first install the advance desktop effects manager if you have not already done so... To do this fire up add/remove.. select the "other" section and then find and select the package called advanced desktop effects manager..

Once you have the manager installed go to Menu>>System>>preferance and fire up the desktop effects manager

Then from the manager find "cube caps" in the utility section and enable it.. Click on the cube caps to bring up the options screen.

From within the options you can select any image you want to appear on top and bottom of your cube, or select colours and levels of transparency...

The link below takes you to a picture of what you are looking for [http://img144.imageshack.us/img144/8211/screenshotfs1.png](http://img144.imageshack.us/img144/8211/screenshotfs1.png)
