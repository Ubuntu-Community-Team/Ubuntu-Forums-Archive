### Post by catlett on 2006-10-24
> **Picco said:**
> The instructions explains Ubuntu setup.  Is there a big difference between the Kubuntu and Ubuntu setup?  I'n going to do Ubuntu.

The install procedure will be the same except the splash screens will say Kubuntu. Just make sure to download the Kubuntu iso. As the previous posted already said, the systems themselves are different. Ubuntu uses gnome for the desktop and kubuntu uses kde. This is where linux is really different from windows. In windows you only have explorer for your desktop/windows manager, whereas in linux you can have any one of a dozen desktops. KDE and Gnome are the 2 most popular but there is also xfce, fluxbox, fvwm, icewm etc. The system underneath will be the same but the way you interact with it will be different. This is Aysiu's page about KDE and Gnome [http://www.psychocats.net/ubuntu/kdegnome](http://www.psychocats.net/ubuntu/kdegnome)
After you install KDE, you may want to refer to this guide [http://kudos.berlios.de/kf/kf1.html](http://kudos.berlios.de/kf/kf1.html)
