### Post by TheAL76 on 2007-10-22
Do a hard shutdown. Then, turn it back on. When the BIOS starts up (it'll flash the brand name of the laptop most times), it'll say press F2 or F-something to get to the BIOS options. Hit that button -- be quick though, you only have a limited window to do it. Then poke around in there for boot order and change it to CD-ROM, so that you'll boot from CD-ROM next time.
