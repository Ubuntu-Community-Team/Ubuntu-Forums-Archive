### Post by mikeyphi on 2007-10-20
Is your 'wired' card recognised? - look in System/Preferences/Hardware information.

For screen size - try System/Administration/Screens & Graphics - see if you can find your screen or something similar - if so you should then be able to get the correct resolution.

```

```

Or System > Administration > Network
