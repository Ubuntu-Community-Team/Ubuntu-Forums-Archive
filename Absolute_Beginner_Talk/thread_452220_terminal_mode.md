### Post by Celegorm on 2007-05-23
A lot of people give directions using terminal commands because it is easier to give directions by saying to copy and paste than by describing how to get to menu that takes 5 clicks. It's quite possible to use linux with no real knowledge of the command line, and there are many tutorials available if you want to learn how to use it. The Ubuntu installation process itself requires absolutely no use of the command line, and is just about the easiest and least painful operating system installation I have ever done.

The best thing to do is to boot up the live cd and see for yourself what the interface is like.
