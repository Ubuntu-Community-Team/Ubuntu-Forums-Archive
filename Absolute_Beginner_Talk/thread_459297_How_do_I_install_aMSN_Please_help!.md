### Post by Bakerman on 2007-05-30
Hi, if you're not yet familiar with the Terminal, you can open add/remove programmes in the main menu, under the internet section you'll find aMSN. Just mark the checkbox next to it and click apply. It should then be installed for you, and a shortcut to it should appear in the internet folder of your main menu.
