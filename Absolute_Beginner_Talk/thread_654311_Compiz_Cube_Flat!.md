### Post by RomeReactor on 2007-12-30
Hi. In "Advanced Desktop Effects Settings" make sure you disable Desktop Wall and enable Desktop Cube and Rotate Cube. If you changed the number of desktops to four using the Workspace Switcher applet and you still get two, try going to "General Options" in Advanced Desktop Effects Settings and on the second tab ("Desktop Size") increase the Horizontal Virtual Size to four.
