### Post by Martin on 2007-05-10
On version 2.0 (and I'm pretty sure version 1.5) of Thunderbird extensions are added under the Tools menu. Tools - Add ons and then Get Extensions in the bottom right corner. That said you usually have to right click and download the add-on to get it to install. Right click agree to the dialog to save it locally then go Tools  Add ons and choose the install button, bottom left and point to the file yuo just downloaded.
