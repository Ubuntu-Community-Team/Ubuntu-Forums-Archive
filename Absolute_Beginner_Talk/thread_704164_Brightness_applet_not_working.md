### Post by neurostu on 2008-02-22
I have the exact same problem on my lenovo T61, Whenever I use the brightness controlls the applet acts like the brightness should be changing but it doesn't.  What I have figured out that works is i switch to a virtual terminal "CNTL+ALT+F1 (F1-F6" and then I can change the brightness there. Then I switch back to X by using "CNTL+ALT+F7".

I know it isn't a perfect fix but it works for now...
