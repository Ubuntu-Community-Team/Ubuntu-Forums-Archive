### Post by lorenzo on 2007-01-04
Which one is "Better" is mostly arguable.

Thunderbird seems to be more efficient as e-mail client, while I find that evolution is more buggy and slow but...

Evolution address book is integrated in gnome (you can add contact's birthday and have them shown in the clock panel applet).
Evolution is more a PIM (personal information manager). It has the calendar, the task list ecc...
Evolution manage meeting requests from Lotus Notes or Exchange and puts them in the calendar.

So, If you need those things (like me) you better use evolution even if sometimes crashes or it is slow.
If you just need an e-mail client, probably thunderbird is better (e.g. Evolution has problems working offline with imap).

### Post by TLE on 2007-01-04
Well for starters. Thunderbird is an email program (The best one in my opinion, but that's a matter of taste) and it has a very good spamfilter. Evolution also handles mails, but it also has a calender and a TODO list and so it can sync with a PDA.

But which is best for you really depends, both on what functions you need. And how you prefer to handle emails.

[EDIT] Actually I use Thunderbird for emails and Evolution for calender and todo list
