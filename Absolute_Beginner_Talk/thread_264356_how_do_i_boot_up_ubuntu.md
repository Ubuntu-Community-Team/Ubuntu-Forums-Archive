### Post by Klaidas on 2006-09-24
You should change boot device order in BIOS - when your computer boots, press one of these keys: Escape, F1, F2, F12, or Delete.
Then the process is pretty simple.
After you set your computer to boot from CD, insert ubuntu cd and reboot.

### Post by comppsyco on 2006-09-24
You will have to change the BIOS settings. This can normally be done by pressing F12 or ESC at bootup. One you're there, there should be an option to change the primary boot device to CD-ROM. Do that, reboot, and you should be home free.
