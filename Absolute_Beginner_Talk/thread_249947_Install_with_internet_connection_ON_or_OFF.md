### Post by Blur-king on 2006-09-03
Hi I notice there are instances where when newbies (me included) started out to install Ubuntu, a number of us have problem with the internet connection. From my own experience when I first install Ubuntu on my first box I had problem with my internet connection as I install with the internet turn OFF. When I subsequently reinstall but with  my internet connection turn ON, ubuntu was able to autodetect everything and it was just a breeze. Repeated it with my second box when I converted it to ubuntu and the same thing happens. Has anyone had the same experience? If so, maybe it might be prudent that we let others keen to try ubuntu know it may be a good idea to leave the internet connections ON during setup so that ubuntu can try to autodetect the internet connections during installation.
