### Post by gh0st on 2007-04-06
You could use Automatix to install all the codecs you need easily. Check it out: [www.getautomatix.com](www.getautomatix.com)

Easy Ubuntu is another option - [http://easyubuntu.freecontrib.org/](http://easyubuntu.freecontrib.org/)

It's a similar thing, it's designed to make installing all the basic multimedia and restricted stuff you need as simple as possible. It worked well for me, hopefully it will do the same for you.

Good luck :)

### Post by gh0st on 2007-04-08
I think it's totally up to the user what they want to do, thats why I suggested Automatix and Easy Ubuntu. I know Automatix is not officially supported in Ubuntu and a lot of people don't like it but I've been using it for a long time and never had a problem. There's no right or wrong answer to this debate, it's all about personal choice. Whatever works for you is what's best :)
