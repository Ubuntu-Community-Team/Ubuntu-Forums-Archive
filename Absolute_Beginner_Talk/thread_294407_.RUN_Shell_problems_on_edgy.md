### Post by po0f on 2006-11-06
PPAAUULL,

Ok I just took the two seconds required to confirm that dash is indeed the default shell interpreter.  The problem is that the symlink **/bin/sh** points to /bin/dash.  You can remedy the situation by temporarily replacing the symlink with your own:
```
$ sudo mv /bin/sh /bin/sh.bak
$ sudo ln -s /bin/bash /bin/sh
$ sudo /path/to/file.run
$ sudo rm /bin/sh
$ sudo mv /bin/sh.bak /bin/sh
```
