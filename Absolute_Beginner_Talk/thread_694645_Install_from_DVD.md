### Post by taurus on 2008-02-12
Your machine only has 256MB of RAM!  It could be that it doesn't have enough RAM to run the LiveCD session from the DVD.  To make sure it is the case, see if you can boot/run from the DVD with another computer, maybe with more RAM.

---

### Post by bumanie on 2008-02-12
It is highly likely that you will need to install with the alternate cd and will probably have to look at xubuntu or (?) fluxbox (lighter desktop environments). 256mb ram is not adequate to run a live cd as they are ram intensive and arguably, even if you managed to get ubuntu installed, 256mb ram is going to cause the OS to struggle.
