### Post by PmDematagoda on 2007-12-05
Activate the repositories you need on Software Sources in System>Administration>Software Sources. After you select the repositories you want, reload the sources list when told to and then try Add/Remove.

### Post by philinux on 2007-12-05
You need to add more sources to your list.

[https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)

If you go system>admin>synaptic it will ask for password. At the top is the settings tab. Clcik repositories to activate the ones you want.
