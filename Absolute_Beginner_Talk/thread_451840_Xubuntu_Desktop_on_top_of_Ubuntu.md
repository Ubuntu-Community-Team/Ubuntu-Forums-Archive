### Post by Kobalt on 2007-05-22
If you want to install Xubuntu just run this command : ```
sudo aptitude install xubuntu-desktop
```
It will install Xubuntu aside your present Ubuntu installation and you'll be able to chose between your desktop environments at the login screen (Options > Session : Gnome/XFCE).
