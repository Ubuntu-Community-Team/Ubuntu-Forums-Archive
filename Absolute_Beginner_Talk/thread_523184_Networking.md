Ummm, excuse me but networking is one thing that Linux does great (at least from everything I've read and seen/done), better than Windows I'd say (probably equal to Apple, their Bonjour is pretty good). Samba is an incredibly powerful networking tool for both *nix and Windows machines. 

If you want to share folders, go System > Administration > Shared Folders. Install the services required. Then fill out the information needed (if any) and share a folder on either a Windows netowrk or a *nix network.

If you want to share an internet connection, my recommendation is to get a hardware router (you can get a cheap one for 20 dollars, good investment. I dunno why you don't have one.) that will both let you share your connection among 4 computers (or more) and provide a hardware firewall and a lot of control on security of your network. A router is a must for all home users in my mind. Not to mention, software connection sharing isn't a very good solution, that's me though, it requires the main computer be on all the time and has other limitations.

[Oh and Samba wiki. ]("https://help.ubuntu.com/community/SettingUpSamba")

Oh and for the record I have my Ubuntu box networked with an XP labtop wirelessly and an XP desktop via my cheap linksys router and it took me all of 5 minutes.
