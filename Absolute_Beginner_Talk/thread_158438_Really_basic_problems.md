### Post by steve.horsley on 2006-04-11
IN synaptic, you can search for a package by name, or by a word/phrase that is likely to be the description of the package - use the search button at the top.

You may want to add additional repositories to get more packages available for download - the multiverse and universe repositories are not oficially supported, but contain loads of quality stuff. In Synaptic, use Settings->Repositories, Add and add the multiverse/universe ones.

For a command line, pull down Accessories->Terminal.
