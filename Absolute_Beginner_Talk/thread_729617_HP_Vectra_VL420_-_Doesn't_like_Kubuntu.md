### Post by Zokbuntu on 2008-05-25
Okay... I never got a response, so I figured something out myself.    In case anyone else has this problem, what I did was load an old version of Ubuntu that worked (6.10) and then had to upgrade to 7.04 and then to 7.10!  That was a pain in the you-know-what, but it worked.  Now I can update to 8.04LTS when I feel like it.  I loaded the Kubuntu desktop and it's working fine.  I still don't know why Ubuntu 7.04 and higher versions didn't want to load from a optical media...
