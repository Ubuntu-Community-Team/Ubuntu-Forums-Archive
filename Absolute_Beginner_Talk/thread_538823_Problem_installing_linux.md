### Post by igknighted on 2007-08-30
(1) Ubuntu 7.10 is alpha software.  If you are new to linux I would strongly recommend you choose the current stable version, 7.04.

(2) My guess as to why you are getting that error is that you burned the CD too fast and there are errors on the CD.  There should be a disk check on the boot menu, run that and see if the disk is OK.

EDIT: looking at your join date I am assuming that you indeed do want to test 7.10 and I was mistaken at first... try the disk check first and see if that says the CD is bad.  If its good can you tell us a bit about your HW (especially post the /etc/fstab file from your current install)?
