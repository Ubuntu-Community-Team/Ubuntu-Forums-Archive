### Post by meng on 2007-02-17
[http://ubuntu-releases.cs.umn.edu//6.06/](http://ubuntu-releases.cs.umn.edu//6.06/)
states you need at least 192MB RAM to install Ubuntu from the Live CD

Many folks here suggest you need 256 MB RAM to be more certain of a successful install.

You can install from the Alternate CD with less RAM, but you should still have at least 128 MB for Xubuntu (Xfce), and 192 MB RAM for the others (GNOME/KDE versions)

You can get by on a server installation (no graphical desktop) with 64 MB.
