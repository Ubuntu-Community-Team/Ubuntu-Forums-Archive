### Post by tgoose on 2007-06-12
For beryl, just go System -> Administration -> Synaptic

Search for and install beryl-manager and heliodor.

That installs all the stuff you need, and puts a little diamond in the top right hand corner. From there you can select to use Beryl in Select Window Manager.

However, the default theme probably isn't very nice (or at least very Ubuntu-like), so you need to right-click on the diamond again and select Helidor in Select Window Decorator. If that's already selected then don't worry about it!

For Limewire I'm pretty sure there will be some equivalent (better) software but I've no idea what it's called. I'm sure someone else will know, though!

Uh are you asking someone to remotely control your computer and install everything? Where the fun in that? :p

Seriously though, I wouldn't want that... ya need to learn how to maintain Ubuntu and you do that by reading, doing, making mistakes and learning :). I'm gonna list some sites for info, read up on them and if you have any questions at all that aren't covered please feel free to ask.

#1 most important thing is how to open the terminal. The easy GUI way is to go Applications > Accessories > Terminal. The alternate way is through the run command prompt (like run in windows, push Alt + F2 to get it), then type in "gnome-terminal".

Now for the guides:

[Ubuntuguide for basic installing of common apps and configuration of common things.]("http://ubuntuguide.org/wiki/Ubuntu:Feisty")
[Psychocats for more graphical explanations of basic things.]("http://www.psychocats.net/ubuntu/")
[LinuxCommand for learning the command line and its intricacies.]("http://www.linuxcommand.org/")
[Linuxappfinder for finding applications that meet all your needs including windows alternatives.]("http://linuxappfinder.com/")
[Ubuntu Wiki for specialized questions, search the key word and there should be a wiki for it.]("https://wiki.ubuntu.com/")
[URL="http://users.bigpond.net.au/hermanzone/"]
Oh and hermanzone has an assortment of other guides especially about GRUB and a few other key features of Ubuntu.[/URL]

Enjoy the reading, you will be a pro in no time. Ubuntuguide is a great stop to get started installing.

Please, I recommend against heading straight for Beryl. If you don't know what your doing, you will break your system. You do need drivers for your card in any case, for that I recommend [Envy.]("http://albertomilone.com/nvidia_scripts1.html") Download, install and follow the instructions at top of page.

Good luck, feel free to ask questions if you don't understand. Do feel free to learn about Ubuntu by reading :D.
