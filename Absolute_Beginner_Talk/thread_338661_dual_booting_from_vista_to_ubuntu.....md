### Post by marx2k on 2007-01-14
> **smitty423 said:**
> when Ubuntu boot's up and after i log in the updater comes's  up just as fast it tell's me what to do but seems to take forever to finish. and for the other problem in the vista partition im running bell south's internet security suite and when i shut down and switch over to Ubuntu do i need another security setup?:confused:

When you see the Ubuntu Update notification, you can choose not to install the updates.  It's advised that you do since they are there to secure your system and each update (unless they are updates to major parts of Ubuntu like Xorg or video drivers or something) are right around or under 1 megabyte.  But again, this is optional.  I believe Windows does this in the background and then asks if you want to install whatever it downloaded and it has an option to let you see what its about to install.  Ubuntu, on the other hand, first tells you what it's about to download and then install.  I find this to be much, much better for the user.  But again, this is optional so you don't HAVE to do this. But it is strongly recommended.  My update takes about 45 seconds on average after hitting 'update' (And this includes entering my password to do so)

If you choose to (NOT RECOMMENDED) you can disable this VALUABLE feature by going to System Menu -> Administration -> Software Sources - Click on the 2nd tab (Internet Updates) and uncheck what you feel needs unchecking. (NOT RECOMMENDED)

You do not need another security set-up in Ubuntu.  If you can see this post from Ubuntu, consider your internet connection setup complete and there is no need for a virus protection suite for Ubuntu and your firewall setup is already running (IPtables)
(type 'man iptables' in a terminal) and if you want to play with the built in firewall, install the front end for it (type 'sudo aptitude install firestarter' in a terminal or Search for and install "Firestarter" from synaptic or Add/Remove programs)

Firestarter Description: gtk program for managing and observing your firewall
 Firestarter is a complete firewall tool for Linux machines. It features an easy
 to use firewall wizard to quickly create a firewall. Using the program you can
 then open and close ports with a few clicks, or stealth your machine giving
 access only to a select few. The real-time hit monitor shows attackers probing
 your machine.

This, also, is optional and most users do not have to deal with it.
