### Post by cilynx on 2006-04-17
The package name depends on what version of Ubuntu you're running.  If you do a search for "flash" in Synaptic, you should find the right one.  Keep in mind, there is a free version and a non-free version.  Also, you may have to manually remove the plugin from your ~/.mozilla/plugins directory as some of the packages just help you install the commercial version.  All Synaptic can do in remove the installer, not the plugin itself.
