### Post by rubbsdecvik on 2007-11-13
I'm not *exactly* sure what's going on.  But I think it's something wrong with the file system.

I haven't tried wubi in a while, but I know what it's about.  It installs a virtual hard drive on your existing NTFS partition. 

My guess is something got corrupted somewhere in the installation.  Here is my suggestion.  Boot into windows and uninstall wubi.  

Then on the C:/ drive make sure you have all of the contents of the wubi and wubi-save folders deleted.  I think those are the only ones.  They will contain the installation files as well as the virtual hard drive.  

then try reinstalling.

I know that it kinda sucks to do it, but it might fix your problem.  I don't know how to fix what's going on directly.
