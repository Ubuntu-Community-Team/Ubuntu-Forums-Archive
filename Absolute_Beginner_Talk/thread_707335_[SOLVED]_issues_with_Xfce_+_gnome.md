### Post by Syndacate on 2008-02-25
You'll be fine, man, nothing will happen.

Here, check it out.

I used to love KDE, so I installed Kubuntu.

Then after actually using KDE I hated it, also, all the "how-to's" are written in GNome, which made it harder to set up.

So I installed the Ubuntu DE, by typing:

```

sudo apt-get update
sudo apt-get install ubuntu-desktop

```

Then I switched to GNome.

Now I have all the KDE applications, and all the GNome applications.

Unless I was to type:
```

sudo apt-get remove kubuntu-desktop

```

I'll keep all my KDE applications.

It's the same for you, just with different environments.

I don't necessarily agree it'd be quicker to do a fresh install.  Just swap the desktop environments, nothing lost, and if you like it, leave it, if not, you can revert back in <30 seconds.

You have to remember that the menus will all be different because the DEs are different, but everything is still there (somewhere) and will still run.

Install 'em both, ****, I might install XFCE just to grab a few more programs.  No it won't hurt anything, everything will still work.
