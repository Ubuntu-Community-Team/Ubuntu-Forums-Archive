### Post by thinkbuddha on 2007-02-28
Hello,
I recently got myself a new (old) dell latitude C400 laptop. A nice little machine. Seeing that it had a very creaky version of windows 2000 installed, bored by XP and unimpressed by Vista's price tag, I decided to give Ubuntu a whirl.

I'm running Egdy and using a speedtouch modem to connect to the internet. It was a bit fiddly, but I followed the instructions at [https://help.ubuntu.com/community/UsbAdslModem/SpeedTouch](https://help.ubuntu.com/community/UsbAdslModem/SpeedTouch) and, although it doesn't connect automatically when switching on, running```
 sudo pppd call speedtch 
``` in the terminal does the trick very nicely and gets me connected.

So this is just me saying hello. I'm glad to be on board!

Best wishes,
