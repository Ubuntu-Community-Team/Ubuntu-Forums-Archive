### Post by thefoolisme on 2007-03-21
I know you said the checksum was fine on the CD, but did you actually run the disk checker utility on the CD?  You need to burn the ISO as slowly as you can, or else you will get errors that will stop the installation.  I had burned a Live CD, and it worked fine for the live sessions, so I thought, I'll just use it to install.  It failed.  I had to create another CD at a much slower burning speed to ensure a good disk.
