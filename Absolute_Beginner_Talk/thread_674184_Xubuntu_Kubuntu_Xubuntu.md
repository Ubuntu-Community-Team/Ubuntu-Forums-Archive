### Post by taurus on 2008-01-21
You can install both Xubuntu (Xfce4) and Kubuntu (KDE) on your machine with

Applications -> Accessories -> Terminal
```
sudo apt-get update
sudo apt-get install xubuntu-desktop
-or-
sudo apt-get install kubuntu-desktop
```
Then, you can choose which window manager you want to use at the Options (bottom left corner) at the GUI login screen.
