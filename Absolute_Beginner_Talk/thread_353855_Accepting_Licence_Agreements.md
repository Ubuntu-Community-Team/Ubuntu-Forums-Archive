### Post by rwfnetworking on 2007-12-04
To sum this up, if you are installing any application that uses a non-graphical license agreement such as Sun Java use the "TAB" key and then press the Enter key. Also mentioned in one of the last posts you may be able to use the left arrow key and then press the Enter. At least for the Sun Jave LIcense agreement, the spcebar does not work for this as well as pressing the Y key....
