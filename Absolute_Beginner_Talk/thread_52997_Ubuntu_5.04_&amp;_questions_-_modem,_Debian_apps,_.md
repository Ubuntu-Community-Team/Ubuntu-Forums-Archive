To answer number two, go to 'systen->administration -> synaptic package manager.  Using the information available here: [http://www.ubuntulinux.org/wiki/AddingRepositoriesHowto](http://www.ubuntulinux.org/wiki/AddingRepositoriesHowto) , you can add the Universe and Multiverse repositories, and then install your Ham applications.  I'm pretty sure that Ubuntu has all of the ones Debian has for that area.  It's recommended you just use the software available through synaptic; it has over 16,000 available packages, which should cover all your needs :)
