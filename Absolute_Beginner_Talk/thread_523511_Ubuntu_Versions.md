### Post by aysiu on 2007-08-12
Ubuntu 7.04 is the most recent version of Ubuntu (7 = 2007, 04 = April)
Ubuntu 6.06 is an older version of Ubuntu (6 = 2006, 06 = June)

More details here:
[http://www.psychocats.net/ubuntu/whichbuntu](http://www.psychocats.net/ubuntu/whichbuntu)

If one can be installed to and run off a USB drive, so can the other. But I don't believe there is an *easy* way to set up a USB Ubuntu. You may be able to track down a complicated HowTo somewhere.

**Edit**: Here's one of the complicated HowTos... [HowTo: Install Feisty on a bootable USB Flash drive](http://ubuntuforums.org/showthread.php?t=476302)

### Post by aysiu on 2007-08-12
> **SOULRiDER said:**
> Also, 4.07 is UNSUPPORTED. The LTS in 6.06 means Long Term Support.
Um, that's not *exactly* true.

All Ubuntu releases are supported. Regular releases get 18 months of support. Long-Term Support releases get 3 years of support on the desktop and 5 years of support on the server.

"Support" here means bug fixes and security updates.

And, there is **no** Ubuntu 4.07. The only Ubuntu releases are these:
4.10 (October 2004)
5.04 (April 2005)
5:10 (October 2005)
6.06 (June 2006)
6.10 (October 2006)
7.04 (April 2007)
