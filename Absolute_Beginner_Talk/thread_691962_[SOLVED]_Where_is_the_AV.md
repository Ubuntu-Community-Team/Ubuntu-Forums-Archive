### Post by seventhc on 2008-02-09
There are a bunch to choose from, go to *Applications>Add/Remove* and you should see a lot. Limewire runs on Ubuntu but I would suggest Frostwire instead, it is basically limewire without the annoying pops up to buy it. [http://www.frostwire.com/](http://www.frostwire.com/) You will need to have java installed.

### Post by Moop on 2008-02-09
Applications-Add/Remove-Sound and Video will show you the available applications. Make sure you put "all available applications" in the drop down box. 

If you're looking for a specific application you can search for it in Synaptic Package Manager.
