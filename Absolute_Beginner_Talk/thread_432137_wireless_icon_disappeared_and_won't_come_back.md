### Post by benanzo on 2007-05-03
I think what you're talking about is the NetworkManager applet.  It is not an applet in the same sense as other applets.  It does not appear in the Add to panel menu.  It should start automatically when you log in if you use NetworkManager.  However, you might be talking about the network-admin applet.  If that's the case then it is called Network Monitor under the System & Hardware section of the Add to Panel menu.

The NetworkManager applet (nm-applet) doesn't have a "remove from panel" option in it's menu, so you must be talking about the network-admin applet.
