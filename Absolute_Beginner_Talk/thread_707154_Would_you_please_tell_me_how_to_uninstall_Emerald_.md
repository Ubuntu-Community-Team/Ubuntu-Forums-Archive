### Post by Trail on 2008-02-25
If you use Emerald, you need to look for emerald(/beryl/compiz) themes. The themes from Appearance Preferences are intended for the normal Metacity theme manager.

### Post by stoodleysnow on 2008-02-25
Emerald is a replacement theme manager. If you want decent themes, use Emerald in System > Preferences > Emerald Theme Manager. If you have problems it should uninstall through Synaptic. Just bear in mind that Icon themes and window backgrounds will still be controlled from the Appearances window.
