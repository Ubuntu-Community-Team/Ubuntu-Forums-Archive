### Post by jaytek13 on 2008-02-13
What do you mean by nothing happens? It just boots regularly and doesn't recognize the cd? If that's the case, you can first make sure that you burnt the ISO image to the cd rather than just burning the actual ISO file itself, then go on to make sure cdrom is an enabled boot device in the BIOS and that it is listed as a bootable device before the hard drive.
