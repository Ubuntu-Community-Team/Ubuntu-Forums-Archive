### Post by bailout on 2006-11-22
I imagine that Konqueror is probably in shock from your colour scheme and this explains its odd behaviour ;)

Konqueror loads a default profile set by the kubuntu developers. You can set it up how you like and then go to 'Settings > save view profile'. However this will save everything about konqueror. You can choose a new name for the profile so you create a new file rather than save over the kubuntu default. Kubuntu sets konq to open with their profile so after you restart your new settings won't show up. Go to 'Settings > load profile' and choose the one you saved. You can edit the shortcut or button you use to open konq so that it starts with your profile as well.

Not sure on your second problem. What do you mean by 'selecting a control function'?

### Post by bailout on 2006-11-22
Once you have konq set as you want you then save it through the 'Settings > save profile' menu. Either choose a new name or just save with the same name as the default kubuntu profile to overwrite it. If you overwrite the current profile then it will open your altered version whenever you start konq.
