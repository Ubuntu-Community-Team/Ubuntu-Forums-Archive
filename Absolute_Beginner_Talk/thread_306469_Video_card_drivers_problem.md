### Post by Voxxi on 2006-11-25
Ok then, I have a Radeon 9550 card and it worked perfectly, maybe your xorg.conf is awry? Xorg.conf is the configuration file for the graphical interface you know and love.

To access xorg.conf:

1) Go into Applications -> Accessories -> Terminal

2) Type in:
```
sudo gedit /etc/X11/xorg.conf
```

Enter your password when prompted.

3) Post the contents of that file here.

I think the problem might be an incorrect driver in xorg.conf for your card. If its that, it should be pretty easy to sort out. ;) 

As for the 30 gigs lost, can you post your partition layout?
