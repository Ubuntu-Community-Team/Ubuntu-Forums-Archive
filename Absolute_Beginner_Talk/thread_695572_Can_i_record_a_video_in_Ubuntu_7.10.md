### Post by billgoldberg on 2008-02-13
in synaptic or add/remove download:

gtk-recordmydesktop

If you have trouble with the audio part (I had), you can use avidemux (also in add/remove) to add audio to the capture.

Note:

The video format will be .ogg theora so you'll need to convert it to something else because 99.9% of the sites don't allow that to be uploaded.

I use the excellent and very easy to use winff.

You have to download the ubuntu .deb [here.]("http://biggmatt.com/winff/downloads/winff-version-0.33.html")

To install a .deb just double click it.
