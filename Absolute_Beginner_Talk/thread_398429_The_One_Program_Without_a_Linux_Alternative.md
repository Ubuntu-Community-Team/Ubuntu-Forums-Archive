[Imageshack Quickshot](http://reg.imageshack.us/content.php?page=quickshot) is probably the only program that there isn't an alternative for on Linux. Wine doesn't work for it (it works to the point where you get it installed, but that part of the program doesn't actually work).

For those of you that don't know, Imageshack Quickshot is a Windows program that lets you take and upload screenshots with a hotkey. So, for example, I could take a screenshot of this page right now and have it uploaded within 10 seconds by pressing Ctrl+Home.

I've looked everywhere, but haven't found anything. I'm sure someone, somewhere has found/made an alternative to this program, but where to find it is the question I have.
