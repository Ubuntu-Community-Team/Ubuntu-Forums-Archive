### Post by rolnics on 2008-02-16
You can get to rotate / change to another work space / desktop using the mouse. 

If you goto System -> Preferences -> Advance Desktop Effects Settings. Then under Desktop you'll find Rotate Cube, click on that and at the top of the list will says "Edge Flip Pointer" and "Edge Flip Move" tick them you there you go!
