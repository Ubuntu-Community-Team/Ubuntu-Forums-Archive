### Post by heatpumpcontrol on 2007-12-21
try in terminal:

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.old

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf

What you are doing is making a back up of the current file just in case. Then you are restoring your original xorg.conf file if one was made.

### Post by PmDematagoda on 2007-12-22
I used this solution when I had a similar problem on Gutsy, I do not know if it might work for you but it might be worth a try. Open the modules file for editing using:-

```
sudo gedit /etc/default/linux-restricted-modules-common

```
and add the nv module to the the disabled modules to make it look like this:-
```
DISABLED_MODULES="nv"
```

Then reinstall the new Nvidia driver and making sure that it works, reboot the computer again.
