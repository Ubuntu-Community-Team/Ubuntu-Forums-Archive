### Post by Kvark on 2005-07-09
To make sure your download is not currupt. Check the md5 checksum of the .iso image you burned the CD from and compare to the md5 sum on the ubuntu download site.

Unfortunately I can't remember how to check the md5 sum of a file in windows. In linux you open a command terminal and type "md5sum filename".

Make sure you burned the CD in the right way. You need to "burn from image", "burn image" or something along those lines, depending on which program you use. If you use another method then it won't work.

When you click on the CD in windows it should open as a folder and you should be able to browse around among the various files on the CD. If you can't do this then the CD doesn't work.
