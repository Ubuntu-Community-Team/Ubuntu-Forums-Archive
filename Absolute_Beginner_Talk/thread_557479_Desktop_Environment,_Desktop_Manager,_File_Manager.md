### Post by Happy_Man on 2007-09-22
A Desktop Environment controls the themes, and (usually) wallpaper. 

A file manager lets you manage your files. So you could use Konqueror (KDE filemanager) with GNOME, or Nautilus in XFCE. It's just a program like any other.
