### Post by snakyjake on 2006-10-07
Between all the different "Linux managers", it is important to ask yourself what you want your desktop/window manager to do.

The next question would be *how* you want the dm/wm to do it.

For me, I wanted my operating system to do only OS related activity, and I only wanted the applications to have the rich features and eye candy.  Some flavors will have the dm/wm doing more and consuming more resources.

Between Xfce and Fluxbox, Fluxbox consumes less resources and does everything that Xfce was doing for me.  Does that mean it is faster?  I'd say some things are a bit quicker on my PII, but not anything to be bragging about.  The #1 reason why I choose Fluxbox over Xfce is because I like the way Fluxbox works.  It is so simple, very easy, no waste, and the slightly minimalst way of the dm/wm doing things is very appealing to me.  And I can use the same richness as all the gnome/kde, etc applications that I choose.

The lightweight distributions are not for everyone.  The best part of Linux is that you can find a flavor that best fits your desires.

Giving the LiveCD's is a good way to become aquainted.
