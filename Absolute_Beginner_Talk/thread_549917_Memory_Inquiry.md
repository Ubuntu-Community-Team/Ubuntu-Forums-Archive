### Post by AtrejuT on 2007-09-13
I'd guess you'll run into problems near 4 GBs or so. 2 should be good.

as to the performance difference: That depends on what you're doing on the computer. Working with big images it will probably make quite a difference. The same is true for scientific calculations and such. For browsing the web the difference should not be very big.

You probably won't see any marked improvement unless you tend to tax your system already.  I have 1gb of memory  and run compiz-fusion with gnome and usually a handful of average apps open and I never hit the point where memory is a bottleneck.  That said, your results may vary.
