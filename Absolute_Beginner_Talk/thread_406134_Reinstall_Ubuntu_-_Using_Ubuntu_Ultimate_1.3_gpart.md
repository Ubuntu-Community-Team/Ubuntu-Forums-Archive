### Post by bigee1212 on 2007-04-10
You can either manually partition your hard drive, which I don't suggest if you don't know what you are doing, or you can just choose the automatic resize option. It will ask you how big you want to make your Windows partition, the leftover space is for ubuntu. It will install in the empty space that was created by resizing the hard drive. 

Make sure you defragment Windows before proceeding with the installtion.
