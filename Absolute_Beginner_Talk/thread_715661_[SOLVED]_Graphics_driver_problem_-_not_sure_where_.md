### Post by PmDematagoda on 2008-03-05
If the driver still does not work after the reinstall, the do this:-

1) Open a modules file for editing using:-
sudo gedit /etc/default/linux-restricted-modules-common

2) Edit the DISABLED_MODULES line to make it look like this:-
```
DISABLED_MODULES="nv nvidia_new"
```
and save the file, then reconfigure the X-Server and reboot the PC to see if the problem is now solved.
