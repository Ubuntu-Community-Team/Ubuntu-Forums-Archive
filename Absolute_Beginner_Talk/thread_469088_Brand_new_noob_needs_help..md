### Post by bigfox on 2007-06-09
Deep Burner can burn ISO files.

[http://www.deepburner.com/?r=download](http://www.deepburner.com/?r=download)

Download the free version.

The ISO file is an image of a CD.  You don't burn the ISO file itself to a CD.  Deep Burner can burn the contents of the ISO file to a CD.

The Ubuntu CD's are bootable once burned correctly.
