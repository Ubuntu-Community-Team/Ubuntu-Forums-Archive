### Post by 3rdalbum on 2006-04-03
From what you say, it sounds like the install has finished and Ubuntu Live is starting up.

Syncing with the time server is not an important step. If it fails, Ubuntu just uses the normal system time. However, it takes quite a while before Ubuntu stops trying to contact the time server. Just wait, and eventually it will finish.

On a 300 MHz computer, Ubuntu will run dreadfully slowly, especially on the Live CD. That's probably why it "takes forever". Otherwise, part of your hardware might be crashing the OS. Does the CD-ROM drive still access the disc?

On such an old machine, you might want to try Puppy Linux or DSL. These are Live CDs as well, and are very suitable for ancient computers like yours and mine :-) Puppy can even be installed to your hard disk.
