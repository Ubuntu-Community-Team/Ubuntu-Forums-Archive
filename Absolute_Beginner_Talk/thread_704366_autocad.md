### Post by Golem XIV on 2008-02-22
Check the [Applications Database on WineHQ]("http://appdb.winehq.org/"), they may have something.  However, I wouldn't hold my breath - AutoCAD is an extremely complex piece of software that relies on way too many components to run properly on Wine.  If you must use AutoCAD I would recommend that you set up a dual-boot system.
