### Post by Najand on 2006-09-11
Nothing is impossible in Linux. When you install windows after installing linux, Windows Boot Loader will overwrite the MBR of your system which was written by Grub... It is a bit long proccess but if you can do it step by step as written in [https://wiki.ubuntu.com/RecoveringUbuntuAfterInstallingWindows]("https://wiki.ubuntu.com/RecoveringUbuntuAfterInstallingWindows") it is possible.
