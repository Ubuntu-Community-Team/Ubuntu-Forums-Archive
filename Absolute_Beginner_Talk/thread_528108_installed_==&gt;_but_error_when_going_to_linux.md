### Post by aJayRoo on 2007-08-17
When I installed Ubuntu 7.04 on both of my machines it did a file system check on first boot and always finds the same error as you. If you let it reboot it will restart and re-do the file system check and will fix errors and you will eventually get into your Linux system. At least that is the way it worked on my machine. This only happens on first boot up after installation, regular file system checks do not usually fail in this way, so I don't think there is anything for you to worry about here :) hope that helps!
