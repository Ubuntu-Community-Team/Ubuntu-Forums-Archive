### Post by surhudm on 2008-01-01
I installed jabbin on gutsy. The package was the one provided on the sourceforge website.
It had a libssl0.9.7 and libcrypto0.9.7 dependency, but you can just soft link the libssl0.9.8 library to libssl0.9.7 and similarly libcrypto0.9.8 to libcrypto0.9.7. Then Jabbin does the logging in and text chat properly with windows gtalk client. The voice chat works one way. I can hear the person on windows talk but not vice versa. When the other person installed jabbin on windows, the voice call worked from jabbin to jabbin.
