### Post by xXx 0wn3d xXx on 2007-03-25

EX:

sudo mkdir /usr/lib/moo

### Post by johnc4510 on 2007-03-25
In a terminal (Applications>Accessories>Terminal) type this:

gksudo nautilus

Navigate to where you want the folder, right click and create it.

This command gives you root privleges for the root system
