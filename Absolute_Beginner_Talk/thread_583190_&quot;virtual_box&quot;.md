### Post by JurB on 2007-10-20
Innotek Virtualbox let's you install an OS in a virtual machine, it's like having a second PC running inside Ubuntu.
You can find out more about it [here.]("http://www.virtualbox.org/")
However, you need a decent spec'd pc to run a second OS upon the first.
AND: 3d games don't work yet, 3d is not supported.
