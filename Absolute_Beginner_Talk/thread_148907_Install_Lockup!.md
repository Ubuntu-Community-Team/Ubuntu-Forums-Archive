### Post by Qrk on 2006-03-22
If you downloaded it yourself, check the md5 sum of your .iso file (you'll have to get a md5 sum checker if you are in windows, I haven't used any so I can't help).

If the md5 sum of the .iso file matches the one on the ubuntu download page, the problem may be the media. Make sure you burn the cd slowly, 4x max. 

If you are still having trouble, and you have an internet connection, try doing a server install by typing "server" at the boot prompt. Once that installation is complete you can type:

sudo apt-get install kubuntu desktop
