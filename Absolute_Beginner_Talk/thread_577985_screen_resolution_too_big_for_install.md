### Post by NT4usB on 2007-10-16
When you run the install disk, F4 (iirc, read the on screen) will let you select a different resolution. Pick something like 640x480.
When the installer starts, use the auto setup function of your monitor to center it up. On a Viewsonic: button 1, scroll to auto image adjust, then button 2.
