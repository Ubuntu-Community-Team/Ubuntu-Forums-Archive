### Post by Dr Small on 2008-02-28
Booting from the LiveCD will load the contents of the CD into the RAM, which will not harm your current OS. But if you install any applications while on the LiveCD, it will be lost when the system powers down / reboots, because the RAM will be emptied.

Dr Small
