### Post by Taliskerd on 2007-07-01
One way to get further along is to stay at these forums, trying to find solutions for people.
The upside is that you have to think for yourself and so learn faster, and even if someone beats you to a solution, you still have gotten your mind exercised ;)
The downside is, um.. well it does take time from whatever you'd be doing instead -but that's the price we pay to improve.

But before you go ahead and start learning, figure out What it is you want to learn more about.
Administration? programming? There are no one way to master a subject, so I'd recommend you make up your mind on what area you want to improve first, then scan the forum for that kind of questions (and anything that look interesting too) and see if you can help out.

Good luck!

I've always found that the best way to learn something new is to come up with a project of some kind, something you want to be able to do.  I wanted to learn about writing web pages, so I checked around, got myself a hosted site, and began playing.

Is there some goal you can define, so that you'll know it when you reach it?  If you want to learn about file systems, for example, you might get yourself a cheap extra disk and play with getting it partitioned, having it mount the way you want in fstab, begin storing certain things on it, etc.
