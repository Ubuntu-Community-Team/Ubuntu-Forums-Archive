> Hi!
I can only read from my external ntfs disk.

Have you installed ntfs-3g which is required to have write access to NTFS drives ?

You can install it by Applications--> Add/Remove. Search for "ntfs" without quotes. Install NTFS Configuration Tool.
After installation, go to Applications-->System Tools --> NTFS Configuration Tool. Enable write support for internal and external drives.

That should do it
