### Post by dfreer on 2007-06-11
Skype works pretty well, although I can't say whether the webcam part works. As for the streaming website, it may be that you do not have all the codecs you need installed. If you can post a link to I can try it out, to see if I can get it working. If you'd rather not or can't, try installing all of the multimedia codecs:
([http://ubuntuguide.org/wiki/Ubuntu:Feisty#How_to_install_Multimedia_Codecs](http://ubuntuguide.org/wiki/Ubuntu:Feisty#How_to_install_Multimedia_Codecs) ).

Another thing is to try a different media player. A few good ones are Mplayer, VLC, xine, and the default totem with the xine plugin. You can install the Media player connectivity plugin for firefox, and use it to pick which player you want to use for your streaming video. You can find out how to install those programs with ubuntuguide as well. Hope this helps!
