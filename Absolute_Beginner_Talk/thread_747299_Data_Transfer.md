### Post by boardfoot on 2008-04-06
As with windows -> windows networking... you need to log into the location as a user that has permissions to access that data. Meaning... go into windows, setup a share, pick a user/password. Then log into the drive with that user/pass. 

Or... easiest way is to sync your username/passwords on both machines. Then it should be seamless.

I'm relatively new to ubuntu but have had no issues accessing my data on the winsux machine from my ubuntu machine. So long as the usernames/passwords are synced.

I do however stand to be corrected. I have not attempted a split install on the same machine. Since discovering and using Ubuntu, my household is now 100% linux. I game on my xbox, not my computers.

Hope that helps or sends you in the right direction.
