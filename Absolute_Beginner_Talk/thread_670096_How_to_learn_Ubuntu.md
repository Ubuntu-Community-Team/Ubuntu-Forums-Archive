### Post by perfecttao on 2008-01-17
I've always found that the best way to learn about an operating system is to set specific goals about what you want to do/learn.  For example, you probably want to set up internet access, customise your desktop and install some applications for a specific use (desktop publishing, Skype, Pidgin etc)

The best way is by finding specific tutorials for these functions first (Ubuntugeek has some good tutorials).  This will no doubt open up a whole load more questions which you can then attack in turn.

Be clear about what you want to learn before you start though, or you may run the risk of drifting and getting bored... 

Have fun
