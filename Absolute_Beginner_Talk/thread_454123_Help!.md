By extracting the files from the ISO image you no longer have a bootable CD image.  You need to burn the image, as an image, directly to the CD.  If you don't know how to burn a cd image you will fins several methods of doing so here [http://www.petri.co.il/how_to_write_iso_files_to_cd.htm]("http://www.petri.co.il/how_to_write_iso_files_to_cd.htm")
