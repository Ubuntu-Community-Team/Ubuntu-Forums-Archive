### Post by KMW on 2008-02-04
After a few months of fooling around with this and asking questions I've finally got this working. Seems that the trouble is with the Onboard video, the drivers and the widescreen monitor.

What I've ended up doing was to install a dedicated video card (eVGA GeForce 7300 gs)  and just like FM it all went away and is now running perfectly. I just wish now that I'd have done it sooner! Now I can get on with learning how to use this and start moving forward w/o having to have 2 computers running. :)
