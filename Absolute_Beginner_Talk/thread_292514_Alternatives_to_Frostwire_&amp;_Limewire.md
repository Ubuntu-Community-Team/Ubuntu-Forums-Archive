### Post by lloyd_b on 2006-11-03
Try GTK-Gnutella.  It's (as far as I know) the only native Linux/Unix Gnutella client that's still being actively developed.  Unlike Frost/Limewire, which use interpreted Java, GtkG is written entirely in C, and so uses much less CPU power for a given load.

Current stable release is 0.96.2, which is available at sourceforge.net.  A 0.96.1 package is currently available from the Ubuntu "universe" servers (via Synaptic, apt-get, etc).

Lloyd B.

### Post by lloyd_b on 2006-11-04
> can i get this GTK-gnutella from synaptic?
forgive me im new to linux but that program sounds gr8 where might i obtain it from?

Version 0.96.1 is available from the repositories.  Start up Synaptic, then search for "gtk-gnutella" as the "name".

If it doesn't show:

"Settings" from the top menu
"Repositories"

Select the "deb" and "deb-src" repositories for "http://us.archive.ubuntu.com/ubuntu" that have "main universe restricted" to the right of them.  This will enable the "universe" range of packages.  Then do the search again, and "gtk-gnutella" should show up.

Lloyd B.
