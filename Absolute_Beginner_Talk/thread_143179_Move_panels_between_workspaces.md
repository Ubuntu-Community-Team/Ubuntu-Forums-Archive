### Post by Wolki on 2006-03-12
I'm not sure I understand what you mean?

You can't move panels between workspaces. Panels are always on each workspace. 

You can, however move windows between workspaces. Try with the workspace switcher, right-clicking the titlebar/window list entry or shift-ctrl-alt-left/right. A fourth soultion would be installing brightside from the repositories and enabling edge flipping (somewhere System -> Preferences -> screen edges, if i remember correctly). It only allows on or off though, not "just while dragging windows".
