When you say "start stop the server"; if you mean the VMware application it's self; then it's normal to leave it alone; it runs as a back ground service. If you mean any virtual machines you have running; then they start and stop through their normal interface.

You add a new virtual machine by downloading and installing the vmware server client software onto your machine and connecting to "local host". 

You also use the client to configure, start and stop virtual machines.

If you've installed vmware server properly you'll find the client under the start menu under applications/system tools/vmware server console
