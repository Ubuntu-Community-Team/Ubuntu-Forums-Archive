### Post by djsroknrol on 2006-05-14
Sorry...my bad...

In a Terminal type:
```
sudo dpkg-reconfigure xserver-xorg
```

that will start a program in the terminal window that will, with a little input from you, will set up your new Xorg.conf file which you'll find in:

Places>Filesystem>/etc/X11/Xorg.conf

The link to my HOW-TO gives an idea as to formating your Xorg.conf file via cut & paste or just plain typing what you need....remember, my Xorg.conf was only to study...it could be used in it's entirty, but edit your monitors, as mine are both CRT's and change the CRT2Position to "Leftof" or "Rightof" depending where your other monitors is.

save the file after you edit it (And back it up before you start...mucho important), and them reboot for the change to take place...simple as that...
