### Post by hardyn on 2007-11-17
Im guessing it is a router problem.  Linux does not yet have a upnp device, you must open and forward the ports manually using your routers firmware config tool.

The only P2P device that i use is bittorrents, and the forwarding it required to get any real speed out if it... BUT... have a look to see if an open-source/free software will suit your needs before running to P2P
