### Post by taurus on 2006-03-19
Regular install comes with the basic system and Gnome window manager.  So, the regular or default install will take up most space.  For server install, it comes with the basic system and some server utilities.  Remember, server doesn't come with X so you need to install it if you plan to run fluxbox or any window manager but "sudo apt-get install fluxbox" will take care of installing Xorg for you.
