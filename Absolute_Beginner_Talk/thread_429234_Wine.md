### Post by useResa on 2007-05-01
There is information on how to install Wine in Ubuntu in the Community Documentation.
You can find this information [here]("https://help.ubuntu.com/community/Wine").

Please note that Wine is an open source implementation of the Windows API and thus NOT windows. Therefore it may not run all programs due to lacking dll's.
For more information on it's capabilities, please visit [Wine HQ]("http://www.winehq.com/").

If you would like to have a full install of Windows you could consider using a Virtual Machine or implement a dual boot.
